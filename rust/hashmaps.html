Introducción a los HashMaps

Los HashMaps son una de las estructuras de datos más comúnmente utilizadas en programación. Permiten almacenar datos en pares clave-valor, proporcionando una recuperación de datos eficiente basada en claves. En Rust, el HashMap es parte de la biblioteca estándar y ofrece una forma robusta de gestionar colecciones de elementos.

Conceptos Clave

  • Pares Clave-Valor: Cada entrada en un HashMap consiste en una clave y un valor.
  • Hashing: Las claves se hashéan para determinar dónde almacenar el valor en memoria.
  • Propiedad: Las reglas de propiedad de Rust se aplican a las claves y valores en un HashMap.
  • Rendimiento: Los HashMaps proporcionan una complejidad de tiempo promedio O(1) para inserciones, eliminaciones y búsquedas.

Creación y Uso de HashMaps

Creación de un HashMap

Para crear un HashMap, necesitas importar el tipo HashMap del módulo std::collections.

use std::collections::HashMap;

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);
    
    println!("{:?}", scores);
}

Acceso a Valores

Puedes acceder a los valores en un HashMap utilizando el método get, que devuelve una Option<&V>.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    let team_name = String::from("Blue");
    match scores.get(&team_name) {
        Some(score) => println!("El puntaje para {} es {}", team_name, score),
        None => println!("No se encontró puntaje para {}", team_name),
    }
}

Iteración Sobre un HashMap

Puedes iterar sobre los pares clave-valor en un HashMap usando un bucle for.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    for (key, value) in &scores {
        println!("{}: {}", key, value);
    }
}

Actualización de Valores

Puedes actualizar los valores en un HashMap insertando un nuevo valor para una clave existente.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    scores.insert(String::from("Blue"), 25); // Actualiza el valor para "Blue"

    println!("{:?}", scores);
}

Eliminación de Valores

Para eliminar un valor de un HashMap, usa el método remove.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    scores.remove(&String::from("Blue"));

    println!("{:?}", scores);
}

Uso Avanzado de HashMap

Manejo de Propiedad y Préstamo

Las reglas de propiedad de Rust se aplican a los HashMaps. Cuando insertas un par clave-valor, el HashMap toma posesión de la clave y el valor.

fn main() {
    let field_name = String::from("Favorite color");
    let field_value = String::from("Blue");

    let mut map = HashMap::new();
    map.insert(field_name, field_value);

    // field_name y field_value ya no son válidos aquí
}

Uso de HashMap con Tipos Personalizados

Puedes usar tipos personalizados como claves o valores en un HashMap, pero el tipo de clave debe implementar los rasgos Eq y Hash.

use std::collections::HashMap;

#[derive(Hash, Eq, PartialEq, Debug)]
struct Person {
    name: String,
    age: u8,
}

fn main() {
    let mut people = HashMap::new();
    let person = Person { name: String::from("Alice"), age: 30 };

    people.insert(person, "Engineer");

    for (key, value) in &people {
        println!("{:?} es un(a) {}", key, value);
    }
}

Métodos de HashMap

Aquí hay algunos métodos útiles proporcionados por el tipo HashMap:

| Método | Descripción | |-----------------|-----------------------------------------------------| | new | Crea un HashMap vacío. | | insert | Inserta un par clave-valor en el HashMap. | | get | Devuelve una referencia al valor correspondiente a la clave. | | remove | Elimina un par clave-valor del HashMap. | | contains_key | Verifica si el HashMap contiene una clave. | | entry | Proporciona acceso mutable en el lugar a una entrada. |

Conclusión

Los HashMaps son una estructura de datos poderosa y flexible en Rust, adecuada para varios casos de uso donde necesitas almacenar y recuperar datos de manera eficiente. Entender cómo crear, acceder, actualizar y gestionar HashMaps mejorará significativamente tu capacidad para escribir programas efectivos en Rust. A medida que avances, descubrirás que dominar los HashMaps es crucial para manejar estructuras de datos y algoritmos más complejos.
