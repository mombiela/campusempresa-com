Introducción a las Funciones

Las funciones son bloques de construcción fundamentales en Rust, permitiéndote encapsular código en unidades reutilizables. Ayudan a organizar el código, mejorar la legibilidad y reducir la redundancia.

Sintaxis Básica

Una función en Rust se define usando la palabra clave fn, seguida del nombre de la función, parámetros y el cuerpo de la función.

fn main() {
    println!("¡Hola, mundo!");
}

Definiendo Funciones

Para definir una función, especificas el nombre, los parámetros (si los hay) y el tipo de retorno (si lo hay).

fn greet(name: &str) {
    println!("¡Hola, {}!", name);
}

fn main() {
    greet("Alice");
}
  • fn: Palabra clave para definir una función.
  • greet: Nombre de la función.
  • name: &str: Parámetro con tipo &str (rebanada de cadena).
  • println!: Macro para imprimir en la consola.

Parámetros y Tipos de Retorno

Las funciones pueden tomar múltiples parámetros y devolver un valor. El tipo de retorno se especifica después de una flecha (->).

fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    let sum = add(5, 3);
    println!("Suma: {}", sum);
}
  • a: i32, b: i32: Parámetros de tipo i32.
  • -> i32: Tipo de retorno i32.
  • a + b: Expresión para devolver la suma de a y b.

Conceptos Avanzados de Funciones

Sobrecarga de Funciones (No Soportada)

Rust no soporta la sobrecarga de funciones directamente. En su lugar, puedes usar traits o diferentes nombres de funciones para lograr una funcionalidad similar.

Closures

Los closures son funciones anónimas que pueden capturar variables de su entorno.

fn main() {
    let x = 10;
    let add_x = |y: i32| x + y;
    println!("Resultado: {}", add_x(5));
}
  • |y: i32| x + y: Closure capturando x del entorno.
  • add_x(5): Llamando al closure con el argumento 5.

Funciones de Orden Superior

Las funciones que toman otras funciones como parámetros o devuelven funciones se llaman funciones de orden superior.

fn apply<F>(f: F, x: i32) -> i32
where
    F: Fn(i32) -> i32,
{
    f(x)
}

fn main() {
    let double = |x: i32| x * 2;
    println!("Resultado: {}", apply(double, 5));
}
  • F: Fn(i32) -> i32: Parámetro genérico F restringido a ser una función que toma i32 y devuelve i32.
  • apply(double, 5): Aplicando el closure double a 5.

Punteros a Funciones

Los punteros a funciones te permiten pasar funciones como argumentos a otras funciones.

fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn apply_function(f: fn(i32, i32) -> i32, a: i32, b: i32) -> i32 {
    f(a, b)
}

fn main() {
    println!("Resultado: {}", apply_function(add, 2, 3));
}
  • fn(i32, i32) -> i32: Tipo del puntero a la función.
  • apply_function(add, 2, 3): Pasando la función add como argumento.

Resumen

Las funciones en Rust son herramientas versátiles que ayudan a organizar y reutilizar el código. Desde definiciones básicas de funciones hasta conceptos avanzados como closures y funciones de orden superior, Rust proporciona características poderosas para trabajar con funciones de manera efectiva. Entender estos conceptos es crucial para escribir código en Rust que sea limpio, eficiente y mantenible.
