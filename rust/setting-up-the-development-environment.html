Configurar el entorno de desarrollo es un primer paso crucial para aprender Rust. Esta sección te guiará a través del proceso de instalación de Rust, configuración de un editor de código y configuración de tu entorno para un desarrollo eficiente.

Instalación de Rust

Para empezar a programar en Rust, necesitas instalar el lenguaje de programación Rust en tu máquina. Rust proporciona una herramienta llamada rustup para gestionar las versiones de Rust y las herramientas asociadas.

Pasos para Instalar Rust

  1. Instalar Rustup:

    • Abre tu terminal.

    • Ejecuta el siguiente comando para descargar e instalar rustup:

      curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

  2. Sigue las Instrucciones en Pantalla:

    • El instalador te guiará a través del proceso de instalación.
    • Por defecto, instalará la última versión estable de Rust.

  3. Verifica la Instalación:

    • Después de la instalación, puedes verificar que Rust está instalado correctamente ejecutando:

      rustc --version

      Este comando debería mostrar la versión de Rust que tienes instalada.

Configuración de un Editor de Código

Elegir el editor de código adecuado puede mejorar significativamente tu productividad. Aquí, configuraremos Visual Studio Code (VS Code), un editor popular y potente para el desarrollo en Rust.

Instalación de Visual Studio Code

  1. Descargar VS Code:

    • Ve al sitio web de Visual Studio Code.
    • Descarga el instalador para tu sistema operativo (Windows, macOS o Linux).

  2. Instalar VS Code:

    • Ejecuta el instalador descargado y sigue las instrucciones de instalación.

Configuración de VS Code para Rust

  1. Instalar la Extensión de Rust:

    • Abre VS Code.
    • Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividad en el lado de la ventana o presionando Ctrl+Shift+X.
    • Busca "Rust" e instala la extensión "Rust (rls)" del equipo de Rust.

  2. Instalar Extensiones Adicionales (Opcional pero Recomendado):

    • CodeLLDB: Proporciona soporte para depuración.
    • Crates: Ayuda a gestionar dependencias en tu archivo Cargo.toml.
    • Rust Analyzer: Una alternativa a RLS, que ofrece más características y mejor rendimiento.

  3. Configurar la Extensión de Rust:

    • Abre la Paleta de Comandos (Ctrl+Shift+P).

    • Escribe "Preferences: Open Settings (JSON)" y selecciónalo.

    • Añade la siguiente configuración a tus ajustes:

      {
  "rust-analyzer.cargo.runBuildScripts": true,
  "rust-analyzer.procMacro.enable": true
}

Creación de tu Primer Proyecto en Rust

Ahora que tienes Rust y VS Code configurados, vamos a crear tu primer proyecto en Rust usando Cargo, el gestor de paquetes y sistema de construcción de Rust.

Pasos para Crear un Nuevo Proyecto

  1. Abre tu Terminal:

    • Navega al directorio donde quieres crear tu proyecto.

  2. Crear un Nuevo Proyecto Cargo:

    • Ejecuta el siguiente comando:

      cargo new hello_world

    • Este comando crea un nuevo directorio llamado hello_world con la siguiente estructura:

      hello_world
├── Cargo.toml
└── src
    └── main.rs

  3. Navega al Directorio del Proyecto:

    • Cambia al directorio del proyecto:

      cd hello_world

  4. Construye y Ejecuta tu Proyecto:

    • Construye el proyecto ejecutando:

      cargo build

    • Ejecuta el proyecto ejecutando:

      cargo run

    • Deberías ver la salida:

      Hello, world!

Conclusión

Configurar el entorno de desarrollo para Rust implica instalar Rust usando rustup, configurar un editor de código como Visual Studio Code y configurarlo con las extensiones necesarias. Con tu entorno listo, puedes crear y ejecutar tu primer proyecto en Rust usando Cargo. Esta configuración proporcionará una base sólida para tu viaje de programación en Rust, permitiéndote escribir, construir y depurar aplicaciones en Rust de manera eficiente.
