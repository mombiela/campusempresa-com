Introducción a los Slices

Los slices son una vista de una secuencia contigua de elementos en una colección en lugar de la colección en sí. Son una referencia a una porción de un array o un vector, lo que te permite trabajar con un subconjunto de la colección sin copiar datos.

Conceptos Clave

  • Slices: Una referencia a una secuencia contigua de elementos en una colección.
  • Array Slices: Slices que referencian una porción de un array.
  • Vector Slices: Slices que referencian una porción de un vector.
  • Inmutabilidad: Los slices son inmutables por defecto, pero se pueden crear slices mutables.

Creación y Uso de Slices

Array Slices

Para crear un slice de un array, usas un rango dentro de corchetes.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[1..4]; // Slicing desde el índice 1 hasta el 3 (4 es exclusivo)
    
    println!("Slice: {:?}", slice); // Salida: Slice: [2, 3, 4]
}

Vector Slices

De manera similar, puedes crear un slice de un vector.

fn main() {
    let vector = vec![10, 20, 30, 40, 50];
    let slice = &vector[2..]; // Slicing desde el índice 2 hasta el final
    
    println!("Slice: {:?}", slice); // Salida: Slice: [30, 40, 50]
}

Slices Mutables

Los slices mutables te permiten modificar los elementos del slice.

fn main() {
    let mut array = [1, 2, 3, 4, 5];
    let slice = &mut array[1..4];
    
    slice[0] = 10; // Modificando el primer elemento del slice
    println!("Array modificado: {:?}", array); // Salida: Array modificado: [1, 10, 3, 4, 5]
}

Slices vs. Arrays y Vectores

| Característica | Array | Vector | Slice | |----------------|----------------|----------------|----------------| | Tamaño Fijo | Sí | No | No | | Tamaño Dinámico| No | Sí | No | | Propiedad | Sí | Sí | No | | Mutabilidad | Sí | Sí | Sí (si es slice mutable) | | Sintaxis | [T; N] | Vec<T> | &[T] o &mut [T] |

Operaciones Comunes con Slices

Iterando Sobre Slices

Puedes iterar sobre slices usando un bucle for.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[..];
    
    for element in slice {
        println!("{}", element);
    }
}

Slicing Strings

Los slices también se pueden usar con strings, pero ten cuidado con los límites de caracteres UTF-8.

fn main() {
    let s = String::from("hola");
    let slice = &s[0..2]; // Slicing los primeros dos bytes
    
    println!("Slice: {}", slice); // Salida: Slice: ho
}

Conclusión

Los slices en Rust proporcionan una forma poderosa de referenciar y trabajar con una porción de una colección sin copiar datos. Son versátiles, permitiendo referencias tanto inmutables como mutables, y se pueden usar con arrays, vectores y strings. Entender los slices es crucial para una manipulación de datos eficiente y una gestión de memoria en Rust.
