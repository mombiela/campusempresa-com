Gestión de Directorios
Permisos y Propiedad de Archivos ►

La gestión de directorios es una habilidad fundamental cuando se trabaja con Bash. Este tema cubrirá comandos y técnicas esenciales para navegar y manipular directorios en un sistema operativo similar a Unix.

Navegación de Directorios

Navegar a través de directorios es una de las primeras tareas que necesitarás dominar en Bash.

`pwd` - Imprimir Directorio de Trabajo

El comando pwd muestra el directorio actual en el que te encuentras.

pwd

Ejemplo de salida:

/home/user

`cd` - Cambiar Directorio

El comando cd se utiliza para cambiar el directorio actual.

cd /ruta/al/directorio

Ejemplo:

cd /home/user/Documents

`ls` - Listar Contenidos del Directorio

El comando ls lista los contenidos de un directorio.

ls

Ejemplo:

ls /home/user/Documents

Opciones Comunes de `ls`

  • -l: Formato de listado largo
  • -a: Incluir archivos ocultos
  • -h: Tamaños de archivo legibles por humanos

Ejemplo:

ls -lah /home/user/Documents

Creación y Eliminación de Directorios

`mkdir` - Crear Directorio

El comando mkdir crea un nuevo directorio.

mkdir nuevo_directorio

Ejemplo:

mkdir /home/user/nueva_carpeta

`rmdir` - Eliminar Directorio

El comando rmdir elimina un directorio vacío.

rmdir directorio_vacío

Ejemplo:

rmdir /home/user/carpeta_vieja

`rm -r` - Eliminar Directorio y Contenidos

El comando rm -r elimina un directorio y sus contenidos de manera recursiva.

rm -r directorio_con_contenidos

Ejemplo:

rm -r /home/user/carpeta_a_eliminar

Mover y Renombrar Directorios

`mv` - Mover o Renombrar

El comando mv se utiliza para mover o renombrar directorios.

mv directorio_viejo directorio_nuevo

Ejemplo:

mv /home/user/carpeta_vieja /home/user/carpeta_nueva

Copiar Directorios

`cp -r` - Copiar Directorio y Contenidos

El comando cp -r copia un directorio y sus contenidos de manera recursiva.

cp -r directorio_fuente directorio_destino

Ejemplo:

cp -r /home/user/carpeta_fuente /home/user/carpeta_destino

Visualización de la Estructura de Directorios

`tree` - Vista de Árbol de Directorios

El comando tree muestra directorios y archivos en un formato similar a un árbol. Nota que tree podría necesitar ser instalado en tu sistema.

tree

Ejemplo:

tree /home/user/Documents

Resumen

En este tema, cubrimos los comandos esenciales para la gestión de directorios en Bash:

  • pwd para imprimir el directorio de trabajo actual.
  • cd para cambiar de directorios.
  • ls para listar los contenidos del directorio.
  • mkdir para crear nuevos directorios.
  • rmdir y rm -r para eliminar directorios.
  • mv para mover o renombrar directorios.
  • cp -r para copiar directorios.
  • tree para ver estructuras de directorios.

Dominar estos comandos mejorará significativamente tu capacidad para navegar y gestionar el sistema de archivos de manera eficiente. Practica estos comandos para sentirte cómodo con la gestión de directorios en Bash.
