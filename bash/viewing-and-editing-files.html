Introducción

En esta sección, cubriremos cómo ver y editar archivos usando Bash. Estas habilidades son esenciales para gestionar y manipular archivos directamente desde la línea de comandos. Comenzaremos con comandos básicos y avanzaremos hacia técnicas más avanzadas.

Visualización de Archivos

Comando `cat`

El comando cat se utiliza para concatenar y mostrar el contenido de los archivos.

  • Sintaxis: cat [opciones] nombre_del_archivo
  • Ejemplo: 
    cat ejemplo.txt
    Este comando mostrará el contenido de ejemplo.txt.

Comando `less`

El comando less te permite ver el contenido de un archivo una pantalla a la vez, lo cual es útil para archivos grandes.

  • Sintaxis: less nombre_del_archivo
  • Ejemplo: 
    less ejemplo.txt
    Usa las teclas de flecha para navegar por el archivo. Presiona q para salir.

Comando `more`

Similar a less, el comando more también te permite ver archivos una pantalla a la vez pero con menos características.

  • Sintaxis: more nombre_del_archivo
  • Ejemplo: 
    more ejemplo.txt
    Usa la barra espaciadora para moverte a la siguiente página y q para salir.

Comandos `head` y `tail`

Los comandos head y tail se utilizan para mostrar el principio y el final de un archivo, respectivamente.

  • Sintaxis:
    • head -n número_de_líneas nombre_del_archivo
    • tail -n número_de_líneas nombre_del_archivo
  • Ejemplo: 
    head -n 10 ejemplo.txt
tail -n 10 ejemplo.txt
    Estos comandos mostrarán las primeras y últimas 10 líneas de ejemplo.txt.

Edición de Archivos

Editor `nano`

nano es un editor de texto simple y fácil de usar.

  • Sintaxis: nano nombre_del_archivo
  • Ejemplo: 
    nano ejemplo.txt
    Usa el teclado para navegar y editar el archivo. Presiona Ctrl + X para salir, Y para guardar los cambios, y N para descartar los cambios.

Editor `vim`

vim es un editor de texto poderoso con una curva de aprendizaje más pronunciada pero ofrece más características.

  • Sintaxis: vim nombre_del_archivo
  • Ejemplo: 
    vim ejemplo.txt
    • Presiona i para entrar en modo de inserción y comenzar a editar.
    • Presiona Esc para salir del modo de inserción.
    • Escribe :wq para guardar y salir, o :q! para salir sin guardar.

Comando `sed`

sed es un editor de flujo utilizado para analizar y transformar texto en archivos.

  • Sintaxis: sed 's/texto_viejo/texto_nuevo/g' nombre_del_archivo
  • Ejemplo: 
    sed 's/hola/mundo/g' ejemplo.txt
    Este comando reemplaza todas las instancias de "hola" con "mundo" en ejemplo.txt.

Comando `awk`

awk es un lenguaje de programación poderoso para el escaneo y procesamiento de patrones.

  • Sintaxis: awk 'patrón {acción}' nombre_del_archivo
  • Ejemplo: 
    awk '/patrón_de_búsqueda/ {print $0}' ejemplo.txt
    Este comando imprime las líneas en ejemplo.txt que coinciden con patrón_de_búsqueda.

Conclusión

En esta sección, cubrimos varios comandos y herramientas para ver y editar archivos en Bash. Desde comandos básicos como cat y less hasta herramientas más avanzadas como vim, sed y awk, ahora tienes un conjunto completo de herramientas para gestionar archivos directamente desde la línea de comandos. Practica estos comandos para volverte competente en la manipulación de archivos usando Bash.
