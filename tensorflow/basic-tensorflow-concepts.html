Introducción a TensorFlow

TensorFlow es un marco de aprendizaje automático de código abierto desarrollado por Google. Es ampliamente utilizado para construir y desplegar modelos de aprendizaje automático, particularmente modelos de aprendizaje profundo. TensorFlow proporciona un ecosistema integral de herramientas, bibliotecas y recursos comunitarios que permiten a los desarrolladores crear y desplegar aplicaciones de aprendizaje automático de manera eficiente.

Conceptos Clave

Tensores

  • Definición: Los tensores son las estructuras de datos fundamentales en TensorFlow. Son arreglos multidimensionales que pueden contener varios tipos de datos.
  • Tipos de Tensores:
    • Escalares (tensores 0-D)
    • Vectores (tensores 1-D)
    • Matrices (tensores 2-D)
    • Tensores de mayor dimensión (3-D, 4-D, etc.)
import tensorflow as tf

# Escalar
escalar = tf.constant(3)
print(escalar)

# Vector
vector = tf.constant([1, 2, 3])
print(vector)

# Matriz
matriz = tf.constant([[1, 2], [3, 4]])
print(matriz)

# Tensor 3-D
tensor_3d = tf.constant([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print(tensor_3d)

Grafos Computacionales

  • Definición: Un grafo computacional es una serie de operaciones de TensorFlow organizadas en un grafo de nodos. Cada nodo representa una operación, y los bordes representan los datos (tensores) que fluyen entre estas operaciones.
  • Ejecución Eager: TensorFlow 2.x utiliza la ejecución eager por defecto, lo que permite que las operaciones se ejecuten inmediatamente a medida que se llaman dentro de Python.
# Ejemplo de ejecución eager
a = tf.constant(2)
b = tf.constant(3)
c = a + b
print(c)  # Salida: 5

Variables

  • Definición: Las variables son tensores especiales que pueden ser actualizados durante la ejecución de un programa. Se utilizan para almacenar y actualizar parámetros en un modelo de aprendizaje automático.
  • Uso: Las variables se crean usando tf.Variable.
# Creando una variable
var = tf.Variable(5)
print(var)

# Actualizando una variable
var.assign(10)
print(var)

Operaciones

  • Definición: Las operaciones (u ops) son funciones que toman uno o más tensores como entrada y producen uno o más tensores como salida. Las operaciones comunes incluyen suma, multiplicación y multiplicación de matrices.
# Operaciones básicas
a = tf.constant(2)
b = tf.constant(3)

add = tf.add(a, b)
mul = tf.multiply(a, b)
matmul = tf.matmul([[1, 2]], [[3], [4]])

print("Suma:", add)
print("Multiplicación:", mul)
print("Multiplicación de Matrices:", matmul)

Sesiones (TensorFlow 1.x)

  • Definición: En TensorFlow 1.x, las sesiones se utilizan para ejecutar operaciones en el grafo computacional. Sin embargo, en TensorFlow 2.x, la ejecución eager elimina la necesidad de sesiones.
# Ejemplo de TensorFlow 1.x (para referencia)
import tensorflow.compat.v1 as tf
tf.disable_v2_behavior()

a = tf.constant(2)
b = tf.constant(3)
c = a + b

with tf.Session() as sess:
    result = sess.run(c)
    print(result)

Resumen

En esta sección, cubrimos los conceptos básicos de TensorFlow, incluyendo tensores, grafos computacionales, variables y operaciones. También tocamos las diferencias entre TensorFlow 1.x y 2.x, particularmente el uso de la ejecución eager en TensorFlow 2.x. Entender estos conceptos fundamentales es crucial para construir y desplegar modelos de aprendizaje automático usando TensorFlow. En las próximas secciones, profundizaremos en temas más avanzados y aplicaciones prácticas de TensorFlow.
