Introducción a los Registros

Los registros en Ada son una forma de agrupar datos relacionados. Son similares a las estructuras en C o a los objetos en otros lenguajes de programación. Los registros te permiten definir un tipo de datos compuesto que puede contener múltiples elementos, cada uno con su propio tipo.

Conceptos Clave

  • Tipo de Datos Compuesto: Un registro es un tipo de datos compuesto que puede contener múltiples elementos.
  • Campos: Los elementos individuales dentro de un registro se llaman campos.
  • Declaración de Tipo: Necesitas declarar un tipo de registro antes de poder usarlo.

Definiendo un Registro

Para definir un registro en Ada, usas la palabra clave type seguida del nombre del registro y la palabra clave record. Cada campo dentro del registro se define con su nombre y tipo.

Ejemplo

type Person is record
    Name  : String(1..50);
    Age   : Integer;
    Email : String(1..100);
end record;

En este ejemplo:

  • Person es el nombre del tipo de registro.
  • Name, Age y Email son campos dentro del registro.
  • Cada campo tiene un tipo específico (String o Integer).

Creando y Usando Registros

Una vez que has definido un tipo de registro, puedes crear variables de ese tipo y acceder a sus campos.

Ejemplo

procedure Main is
    John : Person;
begin
    John.Name  := "John Doe";
    John.Age   := 30;
    John.Email := "john.doe@example.com";
    
    Ada.Text_IO.Put_Line("Name: " & John.Name);
    Ada.Text_IO.Put_Line("Age: " & Integer'Image(John.Age));
    Ada.Text_IO.Put_Line("Email: " & John.Email);
end Main;

En este ejemplo:

  • Se crea una variable John de tipo Person.
  • Se asignan valores a los campos de John.
  • Los valores de los campos se imprimen en la consola.

Registros Anidados

Los registros pueden contener otros registros como campos, permitiéndote crear estructuras de datos complejas.

Ejemplo

type Address is record
    Street : String(1..100);
    City   : String(1..50);
    Zip    : String(1..10);
end record;

type Employee is record
    Name    : String(1..50);
    Age     : Integer;
    Contact : Address;
end record;

procedure Main is
    Jane : Employee;
begin
    Jane.Name := "Jane Smith";
    Jane.Age  := 28;
    Jane.Contact.Street := "123 Main St";
    Jane.Contact.City   := "Anytown";
    Jane.Contact.Zip    := "12345";
    
    Ada.Text_IO.Put_Line("Name: " & Jane.Name);
    Ada.Text_IO.Put_Line("Age: " & Integer'Image(Jane.Age));
    Ada.Text_IO.Put_Line("Address: " & Jane.Contact.Street & ", " & Jane.Contact.City & " " & Jane.Contact.Zip);
end Main;

En este ejemplo:

  • Address es un tipo de registro que contiene campos para calle, ciudad y código postal.
  • Employee es un tipo de registro que contiene un campo de tipo Address.
  • Se crea la variable Jane de tipo Employee y se le asignan valores a sus campos, incluyendo el registro anidado Contact.

Operaciones con Registros

Asignando Registros

Puedes asignar un registro a otro si son del mismo tipo.

John := Jane;

Comparando Registros

Puedes comparar registros para igualdad o desigualdad.

if John = Jane then
    Ada.Text_IO.Put_Line("John and Jane are the same.");
else
    Ada.Text_IO.Put_Line("John and Jane are different.");
end if;

Actualizando Campos

Puedes actualizar campos individuales de un registro.

John.Age := 31;

Conclusión

Los registros en Ada proporcionan una forma poderosa de agrupar datos relacionados. Son flexibles y pueden ser anidados para crear estructuras de datos complejas. Entender cómo definir, crear y manipular registros es esencial para una programación efectiva en Ada. Practica creando y usando registros para familiarizarte con esta importante característica del lenguaje.
