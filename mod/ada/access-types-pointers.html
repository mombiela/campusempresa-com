Introducción a los Tipos de Acceso

Los tipos de acceso en Ada, comúnmente conocidos como punteros en otros lenguajes de programación, se utilizan para referenciar objetos asignados dinámicamente. Proporcionan una forma de crear y manipular estructuras de datos como listas enlazadas, árboles y grafos.

Conceptos Clave

  • Declaración de Tipo de Acceso: Define un tipo que puede contener referencias a objetos de un tipo especificado.
  • Asignación Dinámica: Asigna memoria en tiempo de ejecución usando la palabra clave new.
  • Desreferenciación: Accede al valor almacenado en la ubicación apuntada por un tipo de acceso.
  • Acceso Nulo: Representa un tipo de acceso que no apunta a ningún objeto.

Declaración de Tipos de Acceso

Para declarar un tipo de acceso, se utiliza la palabra clave access seguida del tipo de objeto al que apuntará.

type Int_Ptr is access Integer;

En este ejemplo, Int_Ptr es un tipo de acceso que puede apuntar a un Integer.

Asignación Dinámica

La asignación dinámica en Ada se realiza usando la palabra clave new. Esto asigna memoria para un objeto y devuelve un valor de acceso que apunta a él.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   -- Ptr ahora apunta a un Integer con valor 10
end;

Desreferenciación de Tipos de Acceso

Para acceder al valor almacenado en la ubicación apuntada por un tipo de acceso, se utiliza la notación .all.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   Put_Line(Integer'Image(Ptr.all)); -- Salida: 10
   Ptr.all := 20;
   Put_Line(Integer'Image(Ptr.all)); -- Salida: 20
end;

Acceso Nulo

Un tipo de acceso puede establecerse explícitamente en null, indicando que no apunta a ningún objeto.

declare
   Ptr : Int_Ptr := null;
begin
   if Ptr = null then
      Put_Line("El puntero es nulo");
   end if;
end;

Ejemplo: Lista Enlazada

Vamos a crear una lista enlazada simple usando tipos de acceso.

with Ada.Text_IO; use Ada.Text_IO;

procedure Linked_List_Example is
   type Node;
   type Node_Ptr is access Node;
   
   type Node is
      record
         Value : Integer;
         Next  : Node_Ptr;
      end record;
   
   Head : Node_Ptr := null;
   
   procedure Add_Node(Value : Integer) is
      New_Node : Node_Ptr := new Node'(Value, null);
   begin
      if Head = null then
         Head := New_Node;
      else
         New_Node.Next := Head;
         Head := New_Node;
      end if;
   end Add_Node;
   
   procedure Print_List is
      Current : Node_Ptr := Head;
   begin
      while Current /= null loop
         Put_Line(Integer'Image(Current.Value));
         Current := Current.Next;
      end loop;
   end Print_List;
   
begin
   Add_Node(10);
   Add_Node(20);
   Add_Node(30);
   
   Print_List; -- Salida: 30 20 10
end Linked_List_Example;

Resumen

Los tipos de acceso en Ada son herramientas poderosas para gestionar memoria dinámica y crear estructuras de datos complejas. Puntos clave a recordar:

  • Los tipos de acceso se declaran usando la palabra clave access.
  • La asignación de memoria dinámica se realiza usando la palabra clave new.
  • La desreferenciación se realiza usando la notación .all.
  • Los tipos de acceso pueden establecerse en null para indicar que no apuntan a ningún objeto.

Entender y usar los tipos de acceso de manera efectiva te permite construir programas eficientes y flexibles en Ada.
