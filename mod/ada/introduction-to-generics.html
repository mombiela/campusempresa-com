Introducción a los Genéricos
Procedimientos y Funciones Genéricos ►

Los genéricos en Ada proporcionan una forma poderosa de crear componentes de código flexibles y reutilizables. Permiten escribir algoritmos y estructuras de datos que pueden operar sobre cualquier tipo de dato, haciendo que tu código sea más modular y fácil de mantener.

Conceptos Clave

  • Unidades Genéricas: Son plantillas para crear subprogramas o paquetes que pueden operar sobre diferentes tipos de datos.
  • Parámetros Genéricos: Son los marcadores de posición para los tipos o valores que se especificarán cuando se instancie la unidad genérica.
  • Instanciación: Es el proceso de crear una instancia específica de una unidad genérica proporcionando parámetros reales.

Unidades Genéricas

En Ada, puedes crear procedimientos, funciones y paquetes genéricos. Aquí hay un ejemplo básico de un procedimiento genérico:

generic
   type Item_Type is private;
procedure Swap (X, Y : in out Item_Type);
  • La palabra clave generic indica el inicio de una unidad genérica.
  • type Item_Type is private; declara un parámetro de tipo genérico.
  • procedure Swap es el procedimiento genérico que operará sobre el Item_Type.

Instanciación de Genéricos

Para usar una unidad genérica, necesitas instanciarla con parámetros reales. Aquí se muestra cómo instanciar el procedimiento Swap para enteros:

procedure Swap_Integer is new Swap (Item_Type => Integer);

Ahora puedes usar Swap_Integer para intercambiar valores enteros:

declare
   A, B : Integer := 10;
begin
   Swap_Integer (A, B);
   -- Ahora A = 10 y B = 10
end;

Paquetes Genéricos

Los paquetes genéricos te permiten crear componentes reutilizables que pueden ser personalizados con diferentes tipos o valores. Aquí hay un ejemplo de un paquete genérico para una pila:

generic
   type Element_Type is private;
package Generic_Stack is
   procedure Push (Item : in Element_Type);
   procedure Pop (Item : out Element_Type);
end Generic_Stack;

Para instanciar este paquete para enteros:

package Integer_Stack is new Generic_Stack (Element_Type => Integer);

Ahora puedes usar Integer_Stack para crear y manipular pilas de enteros.

Ejemplo: Algoritmo de Ordenación Genérico

Vamos a crear un paquete genérico para un algoritmo de ordenación:

generic
   type Element_Type is private;
   with function "<" (Left, Right : Element_Type) return Boolean;
package Generic_Sort is
   procedure Sort (Array : in out array (Positive range <>) of Element_Type);
end Generic_Sort;

Para instanciar y usar este paquete para ordenar un array de enteros:

package Integer_Sort is new Generic_Sort (Element_Type => Integer, "<" => "<");

declare
   Arr : array (1 .. 5) of Integer := (5, 3, 4, 1, 2);
begin
   Integer_Sort.Sort (Arr);
   -- Ahora Arr está ordenado: (1, 2, 3, 4, 5)
end;

Conclusión

Los genéricos en Ada proporcionan un mecanismo robusto para crear componentes de código flexibles y reutilizables. Al entender cómo definir e instanciar unidades genéricas, puedes escribir código más modular y mantenible. Practica creando y usando genéricos con diferentes tipos y observa cómo pueden simplificar tus tareas de programación.
