Introducción a Ada

Esta sección introduce el lenguaje de programación Ada, su historia y sus usos principales. Establece la base para entender por qué Ada es un lenguaje poderoso para ciertas aplicaciones.

  • Historia y Antecedentes
    • Origen de Ada
    • Características clave y objetivos de diseño
  • Instalación y Configuración
    • Instalación de GNAT (GNU NYU Ada Translator)
    • Configuración del entorno de desarrollo

Ejemplo: Hola Mundo en Ada

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("¡Hola, Mundo!");
end Hello;

Explicación:

  • with Ada.Text_IO; use Ada.Text_IO;: Importando el paquete Ada Text_IO para operaciones de entrada/salida.
  • procedure Hello is: Declarando un procedimiento llamado Hello.
  • begin ... end Hello;: El cuerpo del procedimiento, donde Put_Line imprime "¡Hola, Mundo!" en la consola.

Sintaxis y Constructos Básicos

Esta sección cubre la sintaxis y los constructos fundamentales de Ada, esenciales para escribir programas básicos.

  • Tipos de Datos y Variables
    • Tipos escalares: Integer, Float, Boolean, Character
    • Tipos compuestos: Arrays, Records
  • Estructuras de Control
    • Declaraciones condicionales: if, case
    • Bucles: for, while
  • Subprogramas
    • Procedimientos
    • Funciones

Ejemplo: Declaración Condicional

procedure Check_Number is
   Number : Integer := 10;
begin
   if Number > 0 then
      Put_Line("El número es positivo");
   elsif Number < 0 then
      Put_Line("El número es negativo");
   else
      Put_Line("El número es cero");
   end if;
end Check_Number;

Explicación:

  • if Number > 0 then ... end if;: Verifica si Number es positivo, negativo o cero e imprime el mensaje correspondiente.

Conceptos Intermedios

Basándose en los conceptos básicos, esta sección profundiza en temas más complejos como la programación modular y el manejo de excepciones.

  • Paquetes y Programación Modular
    • Creación y uso de paquetes
    • Encapsulación y ocultación de información
  • Manejo de Excepciones
    • Declaración y lanzamiento de excepciones
    • Manejo de excepciones con begin ... exception ... end

Ejemplo: Manejo de Excepciones

procedure Divide is
   A, B, Result : Integer;
begin
   A := 10;
   B := 0;
   begin
      Result := A / B;
   exception
      when Constraint_Error =>
         Put_Line("No se permite la división por cero");
   end;
end Divide;

Explicación:

  • begin ... exception ... end;: Intenta dividir A por B y captura un Constraint_Error si B es cero.

Temas Avanzados

Esta sección cubre características avanzadas de Ada como tareas, genéricos y la interfaz con otros lenguajes.

  • Tareas y Concurrencia
    • Tipos de tareas y objetos de tareas
    • Sincronización con objetos protegidos
  • Genéricos
    • Definición e instanciación de paquetes y subprogramas genéricos
  • Interfaz con Otros Lenguajes
    • Interfaz con C y otros lenguajes

Ejemplo: Tareas

task type Worker is
end Worker;

task body Worker is
begin
   Put_Line("La tarea Worker está en ejecución");
end Worker;

procedure Main is
   W : Worker;
begin
   null;
end Main;

Explicación:

  • task type Worker is ... end Worker;: Declara un tipo de tarea llamado Worker.
  • task body Worker is ... end Worker;: Define el comportamiento de la tarea Worker.
  • W : Worker;: Instancia una tarea Worker en el procedimiento Main.

Conclusión

Al dominar estos conceptos, los estudiantes están bien equipados para abordar problemas del mundo real utilizando Ada. Las habilidades adquiridas aquí serán valiosas para desarrollar sistemas de software confiables, mantenibles y eficientes.
