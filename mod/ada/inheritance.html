Herencia en Ada
Polimorfismo ►

La herencia es un concepto fundamental en la programación orientada a objetos (OOP) que permite que una nueva clase herede propiedades y comportamientos (métodos) de una clase existente. En Ada, la herencia se logra mediante el uso de tipos etiquetados y extensiones de tipo.

Conceptos Clave

  • Tipos Etiquetados: Son tipos que pueden ser extendidos para crear nuevos tipos.
  • Extensión de Tipo: El mecanismo por el cual un nuevo tipo hereda propiedades y comportamientos de un tipo existente.
  • Sobrescritura: La capacidad de redefinir métodos en el tipo derivado.

Tipos Etiquetados

Los tipos etiquetados son la base de la herencia en Ada. Un tipo etiquetado es un tipo que puede ser extendido para crear nuevos tipos.

Ejemplo

package Shapes is
    type Shape is tagged record
        X, Y : Float;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Float);
end Shapes;

En este ejemplo, Shape es un tipo etiquetado con dos campos, X y Y. El procedimiento Move es un método que puede aplicarse a cualquier Shape.

Extensión de Tipo

La extensión de tipo te permite crear un nuevo tipo que hereda las propiedades y comportamientos de un tipo existente.

Ejemplo

package Shapes is
    type Circle is new Shape with record
        Radius : Float;
    end record;

    procedure Move(S : in out Circle; New_X, New_Y : Float);
end Shapes;

Aquí, Circle es un nuevo tipo que extiende Shape añadiendo un campo Radius. El procedimiento Move se sobrescribe para manejar específicamente Circle.

Sobrescritura de Métodos

Cuando extiendes un tipo, puedes sobrescribir sus métodos para proporcionar un nuevo comportamiento.

Ejemplo

package body Shapes is
    procedure Move(S : in out Shape; New_X, New_Y : Float) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Move(S : in out Circle; New_X, New_Y : Float) is
    begin
        S.X := New_X;
        S.Y := New_Y;
        -- Se puede añadir comportamiento adicional para Circle aquí
    end Move;
end Shapes;

En este ejemplo, el procedimiento Move se sobrescribe para el tipo Circle para proporcionar un comportamiento específico para los círculos.

Polimorfismo

El polimorfismo te permite usar un tipo derivado donde se espera el tipo base. Esto es útil para escribir código genérico que puede operar sobre diferentes tipos de objetos.

Ejemplo

procedure Draw(Shape : in Shape'Class) is
begin
    -- Código para dibujar la forma
end Draw;

procedure Draw_All(Shapes : in Shape'Class) is
begin
    for S of Shapes loop
        Draw(S);
    end loop;
end Draw_All;

En este ejemplo, Draw es un procedimiento que puede operar sobre cualquier tipo derivado de Shape. El procedimiento Draw_All demuestra cómo usar el polimorfismo para dibujar una colección de formas.

Conclusión

La herencia en Ada es una característica poderosa que te permite crear nuevos tipos basados en los existentes, promoviendo la reutilización y extensibilidad del código. Al entender los tipos etiquetados, la extensión de tipo, la sobrescritura de métodos y el polimorfismo, puedes usar efectivamente la herencia para construir programas en Ada robustos y mantenibles.
