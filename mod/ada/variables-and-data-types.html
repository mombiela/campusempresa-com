Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y tipos de datos en Ada. Comprender estos conceptos es crucial para escribir programas en Ada de manera efectiva y eficiente. Cubriremos los siguientes temas:

  • ¿Qué son las variables?
  • Declaración de variables en Ada
  • Tipos de datos básicos en Ada
  • Conversión de tipos
  • Tipos de datos avanzados

¿Qué son las Variables?

Las variables son ubicaciones de almacenamiento nombradas en la memoria que contienen datos que pueden ser modificados durante la ejecución del programa. Son esenciales para almacenar y manipular información en un programa.

Declaración de Variables en Ada

En Ada, las variables deben ser declaradas antes de que puedan ser usadas. Una declaración de variable especifica el nombre de la variable y su tipo de dato.

Sintaxis

nombre_variable : tipo_dato;

Ejemplo

-- Declarando una variable entera
edad : Integer;

-- Declarando una variable de punto flotante
temperatura : Float;

-- Declarando una variable de cadena
nombre : String(1..50);

Tipos de Datos Básicos en Ada

Ada proporciona varios tipos de datos básicos que se utilizan para definir variables. Estos incluyen:

  • Integer
  • Float
  • Boolean
  • Character
  • String

Integer

Los enteros son números enteros sin una parte fraccionaria.

-- Declarando e inicializando una variable entera
contador : Integer := 10;

Float

Los números de punto flotante se utilizan para representar números reales con una parte fraccionaria.

-- Declarando e inicializando una variable de punto flotante
pi : Float := 3.14159;

Boolean

Los tipos de datos booleanos se utilizan para representar valores de verdad: True o False.

-- Declarando e inicializando una variable booleana
es_valido : Boolean := True;

Character

Los tipos de datos de carácter se utilizan para representar caracteres individuales.

-- Declarando e inicializando una variable de carácter
inicial : Character := 'A';

String

Las cadenas son secuencias de caracteres.

-- Declarando e inicializando una variable de cadena
saludo : String(1..5) := "Hello";

Conversión de Tipos

La conversión de tipos es el proceso de convertir un valor de un tipo de dato a otro. Ada proporciona funciones integradas para la conversión de tipos.

Ejemplo

-- Convirtiendo un entero a un flotante
valor_entero : Integer := 42;
valor_flotante : Float := Float(valor_entero);

-- Convirtiendo un flotante a un entero
valor_flotante : Float := 3.99;
valor_entero : Integer := Integer(valor_flotante);

Tipos de Datos Avanzados

Ada también soporta tipos de datos más complejos como arreglos, registros y enumeraciones.

Arreglos

Los arreglos se utilizan para almacenar múltiples valores del mismo tipo.

-- Declarando e inicializando un arreglo de enteros
numeros : array(1..5) of Integer := (1, 2, 3, 4, 5);

Registros

Los registros se utilizan para agrupar datos relacionados.

-- Declarando un tipo de registro
type Persona is record
    Nombre : String(1..50);
    Edad   : Integer;
end record;

-- Declarando e inicializando una variable de registro
juan : Persona := (Nombre => "Juan Pérez", Edad => 30);

Enumeraciones

Las enumeraciones se utilizan para definir un conjunto de valores nombrados.

-- Declarando un tipo de enumeración
type Dia is (Lunes, Martes, Miércoles, Jueves, Viernes, Sábado, Domingo);

-- Declarando e inicializando una variable de enumeración
hoy : Dia := Viernes;

Resumen

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en Ada. Aprendimos cómo declarar variables, los tipos de datos básicos disponibles en Ada y cómo realizar conversiones de tipos. También tocamos tipos de datos avanzados como arreglos, registros y enumeraciones. Comprender estos conceptos es fundamental para escribir programas efectivos en Ada.
