Introducción a los Paquetes Genéricos

Los paquetes genéricos en Ada te permiten escribir código flexible y reutilizable al parametrizar paquetes con tipos, objetos o subprogramas. Esta característica es particularmente útil para crear estructuras de datos y algoritmos que pueden operar sobre diferentes tipos sin reescribir el código.

Conceptos Clave

  • Genéricos: Un mecanismo para crear plantillas para paquetes, subprogramas o tipos.
  • Parámetros Genéricos: Tipos, objetos o subprogramas que se especifican cuando el paquete genérico se instancia.
  • Instanciación: El proceso de crear una instancia específica de un paquete genérico con parámetros concretos.

Definiendo un Paquete Genérico

Para definir un paquete genérico, se utiliza la palabra clave generic seguida de los parámetros formales. Aquí hay un ejemplo simple:

generic
   type Element_Type is private;
package Generic_Stack is
   procedure Push(Item : in Element_Type);
   procedure Pop(Item : out Element_Type);
   function Is_Empty return Boolean;
end Generic_Stack;

Explicación

  • generic: Indica el inicio de una definición de paquete genérico.
  • type Element_Type is private;: Declara un parámetro de tipo formal llamado Element_Type que es privado, lo que significa que puede ser cualquier tipo.
  • Generic_Stack: El nombre del paquete genérico.
  • Push, Pop, Is_Empty: Subprogramas que operan en la pila.

Instanciando un Paquete Genérico

Para usar un paquete genérico, necesitas instanciarlo con tipos o valores específicos. Aquí se muestra cómo puedes instanciar el paquete Generic_Stack para un tipo Integer:

package Integer_Stack is new Generic_Stack(Element_Type => Integer);

Explicación

  • Integer_Stack: El nombre de la nueva instancia del paquete.
  • new Generic_Stack(Element_Type => Integer);: Instancia el paquete Generic_Stack con Element_Type establecido en Integer.

Usando el Paquete Instanciado

Una vez instanciado, puedes usar el paquete como cualquier otro paquete. Aquí hay un ejemplo de cómo usar el paquete Integer_Stack:

procedure Use_Stack is
   package Integer_Stack is new Generic_Stack(Element_Type => Integer);
   My_Stack : Integer_Stack;
   Item : Integer;
begin
   Integer_Stack.Push(My_Stack, 10);
   Integer_Stack.Push(My_Stack, 20);
   Integer_Stack.Pop(My_Stack, Item);
   Ada.Text_IO.Put_Line("Popped item: " & Integer'Image(Item));
end Use_Stack;

Explicación

  • My_Stack: Una instancia de Integer_Stack.
  • Push, Pop: Operaciones en la pila.
  • Ada.Text_IO.Put_Line: Usado para imprimir el elemento extraído.

Uso Avanzado de Paquetes Genéricos

Múltiples Parámetros

Puedes definir un paquete genérico con múltiples parámetros, incluyendo tipos, objetos y subprogramas. Aquí hay un ejemplo:

generic
   type Key_Type is private;
   type Value_Type is private;
   with function Hash(Key : Key_Type) return Integer;
package Generic_Hash_Table is
   procedure Insert(Key : in Key_Type; Value : in Value_Type);
   function Retrieve(Key : in Key_Type) return Value_Type;
end Generic_Hash_Table;

Explicación

  • type Key_Type is private;: Un parámetro de tipo formal para la clave.
  • type Value_Type is private;: Un parámetro de tipo formal para el valor.
  • with function Hash(Key : Key_Type) return Integer;: Un parámetro de subprograma formal para la función hash.

Instanciando con Múltiples Parámetros

function Integer_Hash(Key : Integer) return Integer is
begin
   return Key mod 101;
end Integer_Hash;

package Integer_String_Hash_Table is new Generic_Hash_Table(
   Key_Type => Integer,
   Value_Type => String,
   Hash => Integer_Hash
);

Explicación

  • Integer_Hash: Una función hash para enteros.
  • Integer_String_Hash_Table: Una instancia de Generic_Hash_Table con Integer como el tipo de clave y String como el tipo de valor.

Conclusión

Los paquetes genéricos en Ada proporcionan una manera poderosa de crear código reutilizable y flexible. Al parametrizar paquetes con tipos, objetos o subprogramas, puedes escribir algoritmos y estructuras de datos genéricos que funcionen con cualquier tipo. Entender cómo definir, instanciar y usar paquetes genéricos es esencial para escribir programas en Ada eficientes y mantenibles.
