Introducción

Redis CLI (Interfaz de Línea de Comandos) es una herramienta poderosa que te permite interactuar con tu servidor Redis directamente desde la línea de comandos. Esta sección cubrirá los conceptos básicos del uso de Redis CLI, incluyendo cómo conectarse a un servidor Redis, ejecutar comandos y entender la salida.

Conectarse al Servidor Redis

Para empezar a usar Redis CLI, primero necesitas conectarte a tu servidor Redis. Esto se hace típicamente usando el comando redis-cli seguido de la dirección y el puerto del servidor.

Ejemplo

redis-cli -h 127.0.0.1 -p 6379
  • -h: Especifica el host (por defecto es 127.0.0.1).
  • -p: Especifica el puerto (por defecto es 6379).

Comandos Básicos

Una vez conectado, puedes empezar a ejecutar comandos de Redis. Aquí tienes algunos de los comandos más comúnmente usados:

PING

El comando PING verifica si el servidor está en funcionamiento.

PING
  • Salida esperada: PONG

SET y GET

El comando SET almacena un par clave-valor, y el comando GET recupera el valor de una clave.

SET mykey "Hello, Redis!"
GET mykey
  • Salida esperada para GET mykey: "Hello, Redis!"

DEL

El comando DEL elimina una clave.

DEL mykey
  • Salida esperada: (integer) 1 (si la clave existía y fue eliminada)

Trabajando con Tipos de Datos

Redis soporta varios tipos de datos, incluyendo cadenas, listas, conjuntos, hashes y conjuntos ordenados. Aquí tienes algunos comandos básicos para cada tipo:

Cadenas

SET key1 "value1"
GET key1

Listas

LPUSH mylist "item1"
LPUSH mylist "item2"
LRANGE mylist 0 -1
  • LPUSH: Añade un elemento a la lista.
  • LRANGE: Recupera elementos de la lista.

Conjuntos

SADD myset "member1"
SADD myset "member2"
SMEMBERS myset
  • SADD: Añade un miembro al conjunto.
  • SMEMBERS: Recupera todos los miembros del conjunto.

Hashes

HSET myhash field1 "value1"
HGET myhash field1
  • HSET: Establece el valor de un campo en un hash.
  • HGET: Recupera el valor de un campo en un hash.

Conjuntos Ordenados

ZADD myzset 1 "member1"
ZADD myzset 2 "member2"
ZRANGE myzset 0 -1 WITHSCORES
  • ZADD: Añade un miembro con una puntuación al conjunto ordenado.
  • ZRANGE: Recupera miembros en un rango especificado.

Opciones Útiles de CLI

Redis CLI ofrece varias opciones para mejorar tu experiencia:

Modo Interactivo

Inicia Redis CLI en modo interactivo simplemente ejecutando:

redis-cli
  • Este modo te permite ingresar comandos uno a la vez y ver los resultados inmediatamente.

Modo No Interactivo

También puedes ejecutar comandos directamente desde la terminal sin entrar en modo interactivo:

redis-cli SET key "value"
  • Esto es útil para scripting y automatización.

Pipelining

Para ejecutar múltiples comandos en una sola solicitud, usa pipelining:

redis-cli --pipe
  • Esto puede mejorar significativamente el rendimiento para operaciones en lote.

Conclusión

Redis CLI es una herramienta esencial para gestionar e interactuar con tu servidor Redis. Al entender los comandos básicos y las opciones, puedes realizar diversas operaciones de manera eficiente y solucionar problemas. Practica usando estos comandos para familiarizarte más con Redis CLI, y estarás bien preparado para abordar temas más avanzados en Redis.
