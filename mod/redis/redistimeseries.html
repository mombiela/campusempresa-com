RedisTimeSeries es un módulo de Redis diseñado para manejar datos de series temporales de manera eficiente. Extiende Redis con un conjunto de comandos y estructuras de datos específicamente optimizados para datos de series temporales, facilitando el almacenamiento, consulta y análisis de datos de series temporales.

Introducción a RedisTimeSeries

RedisTimeSeries proporciona una estructura de datos especializada para datos de series temporales, que son datos indexados por tiempo. Este módulo es particularmente útil para aplicaciones que requieren una ingesta y consulta de datos de series temporales de alto rendimiento, como sistemas de monitoreo, aplicaciones IoT y análisis de datos del mercado financiero.

Características Clave

  • Almacenamiento Eficiente de Datos: Optimizado para almacenar grandes volúmenes de datos de series temporales.
  • Alto Rendimiento: Capaz de manejar un alto rendimiento de escritura y lectura.
  • Agregación y Muestreo: Soporte incorporado para la agregación y muestreo de datos.
  • Compactación: Compactación automática de datos para reducir los requisitos de almacenamiento.
  • Consulta: Capacidades avanzadas de consulta, incluyendo consultas de rango y filtrado.

Instalación de RedisTimeSeries

Para usar RedisTimeSeries, necesitas instalar el módulo. Puedes hacerlo usando la CLI del módulo Redis o incluyéndolo en tu configuración de Redis.

Ejemplo de Instalación

# Descargar e instalar RedisTimeSeries
git clone https://github.com/RedisTimeSeries/RedisTimeSeries.git
cd RedisTimeSeries
make

# Cargar el módulo en Redis
redis-server --loadmodule ./redistimeseries.so

Comandos Básicos

Creación de una Serie Temporal

Para crear una serie temporal, usa el comando TS.CREATE. Este comando inicializa una nueva clave de serie temporal.

TS.CREATE temperature:room1

Añadir Puntos de Datos

Para añadir puntos de datos a una serie temporal, usa el comando TS.ADD. Este comando requiere una marca de tiempo y un valor.

TS.ADD temperature:room1 1633024800 22.5
TS.ADD temperature:room1 1633028400 23.0

Consultar Datos

Para consultar datos de una serie temporal, usa el comando TS.RANGE. Este comando recupera puntos de datos dentro de un rango de tiempo especificado.

TS.RANGE temperature:room1 1633024800 1633028400

Agregación y Muestreo

RedisTimeSeries soporta la agregación y muestreo para resumir datos en intervalos especificados.

TS.RANGE temperature:room1 1633024800 1633028400 AGGREGATION avg 3600

Características Avanzadas

Reglas de Compactación

Las reglas de compactación te permiten muestrear datos automáticamente a medida que se ingieren. Esto es útil para reducir los requisitos de almacenamiento mientras se retiene información importante.

TS.CREATERULE temperature:room1 avg_temperature:room1 AVG 3600

Consultas de Rango Múltiple

Las consultas de rango múltiple te permiten consultar múltiples series temporales a la vez. Esto es útil para comparar datos entre diferentes series.

TS.MRANGE 1633024800 1633028400 FILTER room=room1

Etiquetado y Filtrado

Puedes añadir etiquetas a las series temporales para categorizarlas y filtrarlas más fácilmente.

TS.CREATE temperature:room1 LABELS room room1
TS.CREATE temperature:room2 LABELS room room2

# Consultar todas las series temporales con la etiqueta 'room=room1'
TS.MRANGE 1633024800 1633028400 FILTER room=room1

Ejemplo Práctico

Vamos a recorrer un ejemplo práctico de uso de RedisTimeSeries para monitorear datos de temperatura de múltiples habitaciones.

Ejemplo Paso a Paso

  1. Crear Series Temporales para Cada Habitación

    TS.CREATE temperature:room1 LABELS room room1
TS.CREATE temperature:room2 LABELS room room2

  2. Añadir Puntos de Datos

    TS.ADD temperature:room1 1633024800 22.5
TS.ADD temperature:room1 1633028400 23.0
TS.ADD temperature:room2 1633024800 21.0
TS.ADD temperature:room2 1633028400 21.5

  3. Consultar Datos para una Habitación Específica

    TS.RANGE temperature:room1 1633024800 1633028400

  4. Agregar Datos

    TS.RANGE temperature:room1 1633024800 1633028400 AGGREGATION avg 3600

  5. Filtrar Datos por Etiquetas

    TS.MRANGE 1633024800 1633028400 FILTER room=room1

Conclusión

RedisTimeSeries es un módulo poderoso para gestionar datos de series temporales en Redis. Proporciona almacenamiento eficiente, alto rendimiento y capacidades avanzadas de consulta, haciéndolo ideal para aplicaciones que requieren análisis en tiempo real de datos de series temporales. Aprovechando características como agregación, compactación y etiquetado, puedes construir soluciones robustas y escalables de datos de series temporales con RedisTimeSeries.
