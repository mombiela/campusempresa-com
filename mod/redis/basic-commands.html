Introducción a los Comandos de Redis

Redis es una tienda de estructuras de datos en memoria, utilizada como base de datos, caché y corredor de mensajes. Para interactuar con Redis, utilizas un conjunto de comandos que realizan varias operaciones en los datos almacenados en Redis. Esta sección cubrirá los comandos básicos que necesitas para comenzar con Redis.

Conectándose a Redis

Antes de que puedas ejecutar cualquier comando, necesitas conectarte a tu servidor Redis. Puedes hacerlo usando la CLI de Redis (Interfaz de Línea de Comandos).

redis-cli

Una vez conectado, puedes comenzar a emitir comandos.

Tipos de Datos Básicos y Comandos

Cadenas

Las cadenas son el tipo de valor más básico que puedes almacenar en Redis. Son seguras para binarios, lo que significa que pueden contener cualquier tipo de datos, como una imagen JPEG o un objeto Ruby serializado.

  • SET: Establece el valor de una clave.

    SET key "value"

    Ejemplo:

    SET name "John"

  • GET: Obtiene el valor de una clave.

    GET key

    Ejemplo:

    GET name

  • DEL: Elimina una clave.

    DEL key

    Ejemplo:

    DEL name

Hashes

Los hashes son mapas entre campos de cadena y valores de cadena, lo que los hace perfectos para representar objetos.

  • HSET: Establece el valor de un campo en un hash.

    HSET key field value

    Ejemplo:

    HSET user:1000 name "John"

  • HGET: Obtiene el valor de un campo en un hash.

    HGET key field

    Ejemplo:

    HGET user:1000 name

  • HDEL: Elimina un campo en un hash.

    HDEL key field

    Ejemplo:

    HDEL user:1000 name

Listas

Las listas son colecciones ordenadas de cadenas. Puedes agregar elementos a la cabeza o a la cola de la lista.

  • LPUSH: Agrega uno o más elementos a la cabeza de una lista.

    LPUSH key value [value ...]

    Ejemplo:

    LPUSH mylist "world"
LPUSH mylist "hello"

  • LRANGE: Obtiene un rango de elementos de una lista.

    LRANGE key start stop

    Ejemplo:

    LRANGE mylist 0 -1

  • LPOP: Elimina y obtiene el primer elemento en una lista.

    LPOP key

    Ejemplo:

    LPOP mylist

Conjuntos

Los conjuntos son colecciones desordenadas de cadenas únicas.

  • SADD: Agrega uno o más miembros a un conjunto.

    SADD key member [member ...]

    Ejemplo:

    SADD myset "hello"
SADD myset "world"

  • SMEMBERS: Obtiene todos los miembros en un conjunto.

    SMEMBERS key

    Ejemplo:

    SMEMBERS myset

  • SREM: Elimina uno o más miembros de un conjunto.

    SREM key member [member ...]

    Ejemplo:

    SREM myset "hello"

Conjuntos Ordenados

Los conjuntos ordenados son similares a los conjuntos pero donde cada miembro está asociado con una puntuación. Los elementos están ordenados por sus puntuaciones.

  • ZADD: Agrega uno o más miembros a un conjunto ordenado, o actualiza su puntuación si ya existe.

    ZADD key score member [score member ...]

    Ejemplo:

    ZADD myzset 1 "one"
ZADD myzset 2 "two"

  • ZRANGE: Devuelve un rango de miembros en un conjunto ordenado, por índice.

    ZRANGE key start stop [WITHSCORES]

    Ejemplo:

    ZRANGE myzset 0 -1

  • ZREM: Elimina uno o más miembros de un conjunto ordenado.

    ZREM key member [member ...]

    Ejemplo:

    ZREM myzset "one"

Conclusión

En esta sección, cubrimos los comandos básicos para interactuar con Redis. Exploramos cómo conectarse a Redis y realizar operaciones en diferentes tipos de datos como cadenas, hashes, listas, conjuntos y conjuntos ordenados. Entender estos comandos básicos es esencial para usar Redis de manera efectiva en tus aplicaciones. A medida que progreses, aprenderás comandos y técnicas más avanzadas para aprovechar todo el poder de Redis.
