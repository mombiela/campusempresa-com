Instalación de Redis
Tipos de Datos Básicos de Redis ►

En esta sección, cubriremos los pasos necesarios para instalar Redis en diferentes sistemas operativos. Redis es una tienda de estructuras de datos en memoria, utilizada como base de datos, caché y corredor de mensajes. La instalación adecuada es el primer paso para utilizar sus potentes características.

Instalación de Redis en Linux

Usando el Gestor de Paquetes (APT para Ubuntu/Debian)

  1. Actualiza tu lista de paquetes: 
    sudo apt update
  2. Instala Redis: 
    sudo apt install redis-server
  3. Inicia el servicio de Redis: 
    sudo systemctl start redis
  4. Habilita Redis para que se inicie al arrancar: 
    sudo systemctl enable redis

Usando el Gestor de Paquetes (YUM para CentOS/RHEL)

  1. Añade el repositorio EPEL: 
    sudo yum install epel-release
  2. Instala Redis: 
    sudo yum install redis
  3. Inicia el servicio de Redis: 
    sudo systemctl start redis
  4. Habilita Redis para que se inicie al arrancar: 
    sudo systemctl enable redis

Compilando desde el Código Fuente

  1. Instala las dependencias de compilación: 
    sudo apt update
sudo apt install build-essential tcl
  2. Descarga la última versión estable de Redis: 
    curl -O http://download.redis.io/redis-stable.tar.gz
  3. Extrae el archivo tar: 
    tar xzvf redis-stable.tar.gz
  4. Compila Redis: 
    cd redis-stable
make
  5. Ejecuta el conjunto de pruebas de Redis: 
    make test
  6. Instala Redis: 
    sudo make install

Instalación de Redis en macOS

Usando Homebrew

  1. Instala Homebrew si aún no lo has hecho: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instala Redis: 
    brew install redis
  3. Inicia el servicio de Redis: 
    brew services start redis

Instalación de Redis en Windows

Usando WSL (Subsistema de Windows para Linux)

  1. Habilita WSL e instala una distribución de Linux (por ejemplo, Ubuntu) desde la Microsoft Store.
  2. Abre la terminal de WSL y sigue los pasos de instalación de Linux mencionados anteriormente.

Usando Redis para Windows

  1. Descarga el último archivo zip de Redis desde el archivo de Microsoft.
  2. Extrae el archivo zip en un directorio de tu elección.
  3. Ejecuta el servidor de Redis: 
    redis-server.exe

Verificando la Instalación

Después de instalar Redis, puedes verificar la instalación ejecutando el CLI de Redis y ejecutando un comando simple.

  1. Abre tu terminal o símbolo del sistema.
  2. Ejecuta el CLI de Redis: 
    redis-cli
  3. Ejecuta un comando simple: 
    ping
    • Deberías ver la respuesta: 
      PONG

Conclusión

En esta sección, cubrimos los pasos para instalar Redis en varios sistemas operativos, incluyendo Linux, macOS y Windows. Ya sea que uses un gestor de paquetes, compiles desde el código fuente o uses un método específico para Windows, el proceso es sencillo y asegura que tengas una instancia de Redis en funcionamiento lista para una mayor exploración y desarrollo. En las siguientes secciones, profundizaremos en la configuración y uso efectivo de Redis.
