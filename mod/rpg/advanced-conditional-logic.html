Introducción

En esta sección, profundizaremos en la lógica condicional avanzada en RPG (Report Program Generator). La lógica condicional es un aspecto fundamental de la programación que te permite controlar el flujo de tu programa basado en ciertas condiciones. Dominar la lógica condicional avanzada te permitirá escribir programas RPG más eficientes y poderosos.

Conceptos Clave

  • Condicionales Anidados: Usar condicionales dentro de otros condicionales.
  • Condiciones Complejas: Combinar múltiples condiciones usando operadores lógicos.
  • Estructuras de Caso: Usar SELECT y WHEN para ramificaciones múltiples.
  • Evaluación de Cortocircuito: Entender cómo los operadores lógicos evalúan condiciones.
  • Expresiones Condicionales: Usar expresiones para simplificar la lógica condicional.

Condicionales Anidados

Los condicionales anidados son condicionales colocados dentro de otros condicionales. Esto permite procesos de toma de decisiones más complejos.

Ejemplo

IF customerType = 'Premium';
   IF orderAmount > 1000;
      discount = 0.20;
   ELSE;
      discount = 0.10;
   ENDIF;
ELSE;
   discount = 0.05;
ENDIF;

En este ejemplo, el programa verifica si el cliente es un cliente premium. Si es verdadero, verifica además el monto del pedido para determinar la tasa de descuento.

Condiciones Complejas

Las condiciones complejas implican combinar múltiples condiciones usando operadores lógicos como AND, OR y NOT.

Ejemplo

IF (customerType = 'Premium' AND orderAmount > 1000) OR (customerType = 'Regular' AND orderAmount > 2000);
   discount = 0.15;
ELSE;
   discount = 0.05;
ENDIF;

Aquí, el descuento se establece en 15% si cualquiera de las condiciones combinadas es verdadera.

Estructuras de Caso

Las declaraciones SELECT y WHEN proporcionan una forma más limpia de manejar múltiples condiciones en comparación con las declaraciones IF anidadas.

Ejemplo

SELECT;
   WHEN customerType = 'Premium';
      discount = 0.20;
   WHEN customerType = 'Regular';
      discount = 0.10;
   OTHER;
      discount = 0.05;
ENDSL;

Este ejemplo usa SELECT y WHEN para manejar diferentes tipos de clientes y establecer el descuento apropiado.

Evaluación de Cortocircuito

La evaluación de cortocircuito significa que la evaluación de una expresión lógica se detiene tan pronto como se determina el resultado.

Ejemplo

IF (isMember AND hasDiscountCard);
   discount = 0.25;
ENDIF;

Si isMember es falso, hasDiscountCard no se evalúa, ahorrando tiempo de procesamiento.

Expresiones Condicionales

Las expresiones condicionales te permiten simplificar tu lógica condicional usando la expresión IF.

Ejemplo

discount = IF customerType = 'Premium' AND orderAmount > 1000 THEN 0.20 ELSE 0.10;

Esta sola línea reemplaza una declaración IF anidada más compleja, haciendo el código más limpio y fácil de leer.

Resumen

La lógica condicional avanzada en RPG te permite escribir programas más sofisticados y eficientes. Al dominar los condicionales anidados, las condiciones complejas, las estructuras de caso, la evaluación de cortocircuito y las expresiones condicionales, puedes manejar una amplia gama de escenarios en tus programas RPG. Practica estos conceptos con ejemplos del mundo real para solidificar tu comprensión y mejorar tus habilidades de programación.
