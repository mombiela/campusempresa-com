Introducción

Las funciones integradas en RPG (Report Program Generator) son funciones predefinidas proporcionadas por el lenguaje para realizar tareas comunes. Estas funciones pueden simplificar tu código, hacerlo más legible y reducir la probabilidad de errores. En esta sección, exploraremos varias funciones integradas, entenderemos su uso y veremos cómo pueden aplicarse en escenarios del mundo real.

Conceptos Clave

  • Definición: Las funciones integradas son rutinas predefinidas disponibles en RPG que realizan tareas específicas.
  • Categorías: Estas funciones pueden categorizarse en manipulación de cadenas, operaciones matemáticas, manejo de fechas y horas, y más.
  • Sintaxis: Entender la sintaxis y los parámetros de cada función es crucial para su uso efectivo.

Funciones Integradas Comunes

Funciones de Manipulación de Cadenas

Las funciones de manipulación de cadenas ayudan a manejar y modificar datos de cadena.

  • %TRIM: Elimina los espacios en blanco al principio y al final de una cadena.

    Dcl-S myString Char(20) Inz('  Hello World  ');
Dcl-S trimmedString Char(20);

trimmedString = %Trim(myString);
// trimmedString ahora contiene 'Hello World'

  • %SUBST: Extrae una subcadena de una cadena.

    Dcl-S myString Char(20) Inz('Hello World');
Dcl-S subString Char(5);

subString = %Subst(myString: 1: 5);
// subString ahora contiene 'Hello'

  • %SCAN: Busca una subcadena dentro de una cadena y devuelve su posición.

    Dcl-S myString Char(20) Inz('Hello World');
Dcl-S position Int(10);

position = %Scan('World': myString);
// position ahora contiene 7

Funciones Matemáticas

Las funciones matemáticas realizan operaciones aritméticas y otras tareas matemáticas.

  • %ABS: Devuelve el valor absoluto de un número.

    Dcl-S myNumber Int(10) Inz(-5);
Dcl-S absValue Int(10);

absValue = %Abs(myNumber);
// absValue ahora contiene 5

  • %REM: Devuelve el resto de una operación de división.

    Dcl-S dividend Int(10) Inz(10);
Dcl-S divisor Int(10) Inz(3);
Dcl-S remainder Int(10);

remainder = %Rem(dividend: divisor);
// remainder ahora contiene 1

Funciones de Fecha y Hora

Las funciones de fecha y hora ayudan a manejar y manipular valores de fecha y hora.

  • %DATE: Convierte una cadena en una fecha.

    Dcl-S dateString Char(10) Inz('2023-10-01');
Dcl-S myDate Date;

myDate = %Date(dateString);
// myDate ahora contiene la fecha 2023-10-01

  • %DIFF: Calcula la diferencia entre dos fechas.

    Dcl-S startDate Date Inz(D'2023-01-01');
Dcl-S endDate Date Inz(D'2023-10-01');
Dcl-S dateDiff Int(10);

dateDiff = %Diff(endDate: startDate: *D);
// dateDiff ahora contiene 273 (días)

Ejemplos y Ejercicios

Ejemplo 1: Uso de Funciones de Cadenas

Dcl-S fullName Char(50) Inz('  John Doe  ');
Dcl-S firstName Char(20);
Dcl-S lastName Char(20);
Dcl-S position Int(10);

// Recortar el nombre completo
fullName = %Trim(fullName);

// Encontrar la posición del espacio
position = %Scan(' ': fullName);

// Extraer nombres y apellidos
firstName = %Subst(fullName: 1: position - 1);
lastName = %Subst(fullName: position + 1);

// Salida: firstName = 'John', lastName = 'Doe'

Ejercicio 1: Calcular Edad

Escribe un programa que calcule la edad de una persona dada su fecha de nacimiento.

Dcl-S birthDate Date Inz(D'1990-05-15');
Dcl-S currentDate Date Inz(%Date());
Dcl-S age Int(10);

age = %Diff(currentDate: birthDate: *Y);
// Salida: age contendrá el número de años desde birthDate

Conclusión

Las funciones integradas en RPG son herramientas poderosas que pueden mejorar enormemente tu eficiencia en la programación. Al entender y utilizar estas funciones, puedes escribir código más limpio y eficiente. Practica el uso de estas funciones en varios escenarios para volverte competente en su aplicación.
