Manejar la entrada del usuario es un aspecto fundamental del desarrollo de aplicaciones interactivas. En esta sección, exploraremos cómo gestionar la entrada del usuario en Xcode, desde campos de texto básicos hasta reconocimientos de gestos avanzados.

Entrada de Texto Básica

UITextField

UITextField es un componente de UI básico que permite a los usuarios ingresar texto. Se utiliza comúnmente para formularios y barras de búsqueda.

  • Crear un UITextField: Puedes agregar un UITextField a tu storyboard o crearlo programáticamente.
  • Configurar UITextField: Personaliza propiedades como el texto de marcador de posición, el tipo de teclado y el tipo de tecla de retorno.

Ejemplo: Crear un UITextField Programáticamente

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Ingresa tu nombre"
        textField.borderStyle = .roundedRect
        self.view.addSubview(textField)
    }
}

Manejo de la Entrada de Texto

Para manejar la entrada de texto, puedes usar el protocolo UITextFieldDelegate.

  • Implementar UITextFieldDelegate: Conformarse al protocolo UITextFieldDelegate e implementar sus métodos.
  • Métodos Comunes:
    • textFieldShouldReturn(_:): Se llama cuando se presiona el botón de retorno.
    • textField(_:shouldChangeCharactersIn:replacementString:): Se llama cuando el usuario escribe o elimina un carácter.

Ejemplo: Manejo de la Tecla de Retorno

import UIKit

class ViewController: UIViewController, UITextFieldDelegate {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Ingresa tu nombre"
        textField.borderStyle = .roundedRect
        textField.delegate = self
        self.view.addSubview(textField)
    }
    
    func textFieldShouldReturn(_ textField: UITextField) -> Bool {
        textField.resignFirstResponder() // Ocultar el teclado
        print("Usuario ingresó: \(textField.text ?? "")")
        return true
    }
}

Manejo Avanzado de Entrada

Reconocedores de Gestos

Los reconocedores de gestos te permiten detectar y responder a eventos táctiles como toques, deslizamientos y pellizcos.

  • Tipos de Reconocedores de Gestos:
    • UITapGestureRecognizer
    • UIPinchGestureRecognizer
    • UISwipeGestureRecognizer
    • UILongPressGestureRecognizer

Ejemplo: Agregar un Reconocedor de Toque

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let tapGesture = UITapGestureRecognizer(target: self, action: #selector(handleTap(_:)))
        self.view.addGestureRecognizer(tapGesture)
    }
    
    @objc func handleTap(_ sender: UITapGestureRecognizer) {
        print("Vista tocada")
    }
}

Vistas de Entrada Personalizadas

Para escenarios de entrada más complejos, puedes crear vistas de entrada personalizadas.

  • Crear Vistas de Entrada Personalizadas: Diseña una vista personalizada que pueda ser utilizada como una vista de entrada.
  • Asignar Vistas de Entrada Personalizadas: Asigna la vista personalizada a la propiedad inputView de un responder, como un UITextField.

Ejemplo: Selector de Fecha Personalizado

import UIKit

class ViewController: UIViewController, UITextFieldDelegate {
    let datePicker = UIDatePicker()
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Selecciona una fecha"
        textField.borderStyle = .roundedRect
        textField.delegate = self
        textField.inputView = datePicker
        self.view.addSubview(textField)
        
        datePicker.datePickerMode = .date
        datePicker.addTarget(self, action: #selector(dateChanged(_:)), for: .valueChanged)
    }
    
    @objc func dateChanged(_ sender: UIDatePicker) {
        print("Fecha seleccionada: \(sender.date)")
    }
}

Conclusión

Manejar la entrada del usuario es una habilidad crucial para cualquier desarrollador de iOS. Comenzando con campos de texto básicos y progresando a reconocedores de gestos avanzados y vistas de entrada personalizadas, puedes crear aplicaciones interactivas y fáciles de usar. Practica estos conceptos y explora las diversas técnicas de manejo de entrada para mejorar tus habilidades de desarrollo de aplicaciones en Xcode.
