Introducción

En esta sección, profundizaremos en las estructuras de datos avanzadas en Swift. Comprender estas estructuras te ayudará a escribir código más eficiente y optimizado. Cubriremos:

  • Listas Enlazadas
  • Árboles
  • Grafos
  • Tablas Hash
  • Montículos

Listas Enlazadas

Una lista enlazada es una estructura de datos lineal donde cada elemento es un objeto separado, conocido como nodo. Cada nodo contiene datos y una referencia (o enlace) al siguiente nodo en la secuencia.

Conceptos Clave

  • Nodo: La unidad básica de una lista enlazada, que contiene datos y una referencia al siguiente nodo.
  • Cabeza: El primer nodo en la lista enlazada.
  • Cola: El último nodo en la lista enlazada, que apunta a nil.

Ejemplo de Código

class Node<T> {
    var value: T
    var next: Node?

    init(value: T) {
        self.value = value
    }
}

class LinkedList<T> {
    var head: Node<T>?

    func append(value: T) {
        let newNode = Node(value: value)
        if let tailNode = head {
            var currentNode = tailNode
            while let nextNode = currentNode.next {
                currentNode = nextNode
            }
            currentNode.next = newNode
        } else {
            head = newNode
        }
    }
}

Explicación

  • Clase Node: Define un nodo con un valor y una referencia al siguiente nodo.
  • Clase LinkedList: Gestiona la lista enlazada, con métodos para agregar nuevos nodos.

Árboles

Un árbol es una estructura de datos jerárquica que consiste en nodos, con un solo nodo como la raíz y sub-nodos como hijos.

Conceptos Clave

  • Raíz: El nodo superior del árbol.
  • Hoja: Un nodo sin hijos.
  • Árbol Binario: Un árbol donde cada nodo tiene como máximo dos hijos.

Ejemplo de Código

class TreeNode<T> {
    var value: T
    var left: TreeNode?
    var right: TreeNode?

    init(value: T) {
        self.value = value
    }
}

class BinaryTree<T> {
    var root: TreeNode<T>?

    func insert(value: T) {
        let newNode = TreeNode(value: value)
        if let rootNode = root {
            self.insertNode(rootNode, newNode)
        } else {
            root = newNode
        }
    }

    private func insertNode(_ node: TreeNode<T>, _ newNode: TreeNode<T>) {
        if newNode.value < node.value {
            if let leftNode = node.left {
                insertNode(leftNode, newNode)
            } else {
                node.left = newNode
            }
        } else {
            if let rightNode = node.right {
                insertNode(rightNode, newNode)
            } else {
                node.right = newNode
            }
        }
    }
}

Explicación

  • Clase TreeNode: Define un nodo con un valor y referencias a los hijos izquierdo y derecho.
  • Clase BinaryTree: Gestiona el árbol binario, con métodos para insertar nuevos nodos.

Grafos

Un grafo es una colección de nodos (vértices) y aristas que los conectan. Los grafos pueden ser dirigidos o no dirigidos.

Conceptos Clave

  • Vértice: Un nodo en el grafo.
  • Arista: Una conexión entre dos vértices.
  • Grafo Dirigido: Las aristas tienen una dirección.
  • Grafo No Dirigido: Las aristas no tienen una dirección.

Ejemplo de Código

class Vertex<T> {
    var value: T
    var edges: [Edge<T>] = []

    init(value: T) {
        self.value = value
    }
}

class Edge<T> {
    var from: Vertex<T>
    var to: Vertex<T>

    init(from: Vertex<T>, to: Vertex<T>) {
        self.from = from
        self.to = to
    }
}

class Graph<T> {
    var vertices: [Vertex<T>] = []

    func addVertex(value: T) -> Vertex<T> {
        let vertex = Vertex(value: value)
        vertices.append(vertex)
        return vertex
    }

    func addEdge(from: Vertex<T>, to: Vertex<T>) {
        let edge = Edge(from: from, to: to)
        from.edges.append(edge)
    }
}

Explicación

  • Clase Vertex: Define un vértice con un valor y una lista de aristas.
  • Clase Edge: Define una arista con referencias a los vértices de inicio y fin.
  • Clase Graph: Gestiona el grafo, con métodos para agregar vértices y aristas.

Tablas Hash

Una tabla hash es una estructura de datos que asigna claves a valores para una búsqueda eficiente.

Conceptos Clave

  • Función Hash: Convierte una clave en un índice en la tabla hash.
  • Colisión: Cuando dos claves se hash a la misma índice.
  • Encadenamiento: Un método para manejar colisiones almacenando múltiples elementos en el mismo índice.

Ejemplo de Código

class HashTable<Key: Hashable, Value> {
    private var buckets: [[(key: Key, value: Value)]]
    private let capacity: Int

    init(capacity: Int) {
        self.capacity = capacity
        self.buckets = Array(repeating: [], count: capacity)
    }

    func insert(key: Key, value: Value) {
        let index = self.index(for: key)
        buckets[index].append((key: key, value: value))
    }

    func value(for key: Key) -> Value? {
        let index = self.index(for: key)
        for element in buckets[index] {
            if element.key == key {
                return element.value
            }
        }
        return nil
    }

    private func index(for key: Key) -> Int {
        return abs(key.hashValue) % capacity
    }
}

Explicación

  • Clase HashTable: Gestiona la tabla hash con métodos para insertar y recuperar valores.
  • Método index(for:): Calcula el índice para una clave dada usando una función hash.

Montículos

Un montículo es una estructura de datos especializada basada en árboles que satisface la propiedad de montículo.

Conceptos Clave

  • Montículo Máximo: El nodo padre es siempre mayor o igual que los nodos hijos.
  • Montículo Mínimo: El nodo padre es siempre menor o igual que los nodos hijos.

Ejemplo de Código

class Heap<T: Comparable> {
    var elements: [T] = []

    func insert(_ value: T) {
        elements.append(value)
        siftUp(from: elements.count - 1)
    }

    private func siftUp(from index: Int) {
        var childIndex = index
        let childValue = elements[childIndex]
        var parentIndex = (childIndex - 1) / 2

        while childIndex > 0 && childValue > elements[parentIndex] {
            elements[childIndex] = elements[parentIndex]
            childIndex = parentIndex
            parentIndex = (childIndex - 1) / 2
        }
        elements[childIndex] = childValue
    }
}

Explicación

  • Clase Heap: Gestiona el montículo con métodos para insertar elementos.
  • Método siftUp(from:): Asegura que se mantenga la propiedad del montículo después de la inserción.

Conclusión

Comprender estas estructuras te permitirá escribir código más eficiente y optimizado. Practica implementando estas estructuras de datos y usándolas en tus proyectos para solidificar tu comprensión.
