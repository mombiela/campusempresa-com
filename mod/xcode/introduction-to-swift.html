Introducción a Swift
Escribiendo Tu Primer Programa en Swift ►

Swift es un lenguaje de programación poderoso e intuitivo desarrollado por Apple para construir aplicaciones para iOS, macOS, watchOS y tvOS. Esta sección te introducirá a los conceptos básicos de Swift, estableciendo una base sólida para desarrollar aplicaciones en Xcode.

¿Qué es Swift?

Swift es un lenguaje de programación moderno que es:

  • Seguro: Swift elimina clases enteras de código inseguro.
  • Rápido: Swift está diseñado para ser rápido y eficiente.
  • Expresivo: Swift tiene una sintaxis limpia que hace que el código sea fácil de leer y escribir.

Configuración de tu Entorno

Antes de comenzar a programar en Swift, necesitas configurar tu entorno de desarrollo:

  1. Instalar Xcode: Xcode es el entorno de desarrollo integrado (IDE) para macOS que contiene un conjunto de herramientas de desarrollo de software.
  2. Crear un Nuevo Proyecto: Abre Xcode y crea un nuevo proyecto para comenzar a escribir código en Swift.

Sintaxis Básica

Variables y Constantes

En Swift, usas var para declarar variables y let para declarar constantes.

var myVariable = 42
myVariable = 50
let myConstant = 42
  • Variables: Pueden cambiar después de ser establecidas.
  • Constantes: No pueden cambiar una vez establecidas.

Tipos de Datos

Swift es un lenguaje seguro en cuanto a tipos, lo que significa que te ayuda a ser claro sobre los tipos de valores con los que tu código puede trabajar.

let explicitDouble: Double = 70
let explicitFloat: Float = 4.0
let explicitInt: Int = 42

Cadenas y Caracteres

Las cadenas se usan para almacenar texto. Puedes crear una cadena encerrando el texto entre comillas dobles.

let greeting = "¡Hola, Mundo!"

Arreglos y Diccionarios

Los arreglos y diccionarios se usan para almacenar colecciones de valores.

var shoppingList = ["Huevos", "Leche"]
var occupations = [
    "Malcolm": "Capitán",
    "Kaylee": "Mecánica",
]

Control de Flujo

Swift proporciona todas las estructuras estándar de control de flujo como for, while y if.

let individualScores = [75, 43, 103, 87, 12]
var teamScore = 0
for score in individualScores {
    if score > 50 {
        teamScore += 3
    } else {
        teamScore += 1
    }
}
print(teamScore)

Funciones

Las funciones son bloques de código autónomos que realizan una tarea específica.

func greet(persona: String, día: String) -> String {
    return "Hola \(persona), hoy es \(día)."
}
print(greet(persona: "Juan", día: "Miércoles"))
  • Parámetros: Entradas para la función.
  • Tipo de Retorno: El tipo de valor que la función devuelve.

Clases y Estructuras

Swift proporciona constructos poderosos para crear tipos de datos personalizados usando clases y estructuras.

Clases

Las clases soportan herencia, lo que significa que una clase puede heredar las características de otra.

class Shape {
    var numberOfSides = 0
    func simpleDescription() -> String {
        return "Una forma con \(numberOfSides) lados."
    }
}

Estructuras

Las estructuras son tipos de valor y no soportan herencia.

struct Card {
    var rank: Int
    var suit: String
    func simpleDescription() -> String {
        return "El \(rank) de \(suit)"
    }
}

Opcionales

Los opcionales se usan para manejar la ausencia de un valor.

var optionalString: String? = "Hola"
print(optionalString == nil)

var optionalName: String? = "Juan Appleseed"
var greeting = "¡Hola!"
if let name = optionalName {
    greeting = "Hola, \(name)"
}

Conclusión

En esta introducción a Swift, cubrimos:

  • Los conceptos básicos de la sintaxis de Swift.
  • Cómo declarar variables y constantes.
  • Tipos de datos básicos y colecciones.
  • Estructuras de control de flujo.
  • Funciones, clases y estructuras.
  • Opcionales y su uso.

Entender estos fundamentos te preparará para temas más avanzados en Swift y desarrollo en Xcode. Continúa practicando estos conceptos para volverte competente en programación con Swift.
