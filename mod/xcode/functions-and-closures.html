Introducción a las Funciones

Las funciones son fragmentos de código autónomos que realizan una tarea específica. En Swift, las funciones son ciudadanos de primera clase, lo que significa que pueden ser pasadas y usadas en tu código como cualquier otro valor.

Definiendo y Llamando Funciones

Para definir una función en Swift, usas la palabra clave func seguida del nombre de la función, un par de paréntesis y un conjunto de llaves que contienen el código de la función.

func greet(persona: String) -> String {
    return "¡Hola, \(persona)!"
}

let saludo = greet(persona: "Alice")
print(saludo) // Salida: ¡Hola, Alice!
  • func: Palabra clave para definir una función.
  • greet(persona: String) -> String: Firma de la función con un parámetro y un tipo de retorno.
  • return "¡Hola, \\(persona)!": El cuerpo de la función que devuelve un mensaje de saludo.

Parámetros de Función y Valores de Retorno

Las funciones pueden tomar múltiples parámetros y devolver múltiples valores usando tuplas.

func add(a: Int, b: Int) -> Int {
    return a + b
}

let suma = add(a: 3, b: 5)
print(suma) // Salida: 8
  • add(a: Int, b: Int) -> Int: Función con dos parámetros y un tipo de retorno.

Valores Predeterminados de Parámetros

Puedes proporcionar valores predeterminados para cualquier parámetro en una función asignando un valor al parámetro después de su tipo.

func greet(persona: String, desde ciudad: String = "Cupertino") -> String {
    return "¡Hola \(persona)! Me alegra que hayas podido visitar desde \(ciudad)."
}

print(greet(persona: "Alice")) // Salida: ¡Hola Alice! Me alegra que hayas podido visitar desde Cupertino.
print(greet(persona: "Bob", desde: "Nueva York")) // Salida: ¡Hola Bob! Me alegra que hayas podido visitar desde Nueva York.

Conceptos Avanzados de Funciones

Parámetros In-Out

Los parámetros in-out permiten que una función modifique una variable pasada desde fuera de la función.

func swapTwoInts(_ a: inout Int, _ b: inout Int) {
    let temporaryA = a
    a = b
    b = temporaryA
}

var x = 3
var y = 5
swapTwoInts(&x, &y)
print("x: \(x), y: \(y)") // Salida: x: 5, y: 3
  • inout: Palabra clave para indicar un parámetro in-out.
  • &: Prefijo usado al pasar una variable como un parámetro in-out.

Tipos de Función

Cada función tiene un tipo de función específico, compuesto por los tipos de parámetros y el tipo de retorno.

func multiply(a: Int, b: Int) -> Int {
    return a * b
}

let mathFunction: (Int, Int) -> Int = multiply
print(mathFunction(2, 3)) // Salida: 6
  • (Int, Int) -> Int: Tipo de función para una función que toma dos parámetros Int y devuelve un Int.

Introducción a los Closures

Los closures son bloques de funcionalidad autónomos que pueden ser pasados y usados en tu código. Los closures en Swift son similares a los bloques en C y Objective-C y a las lambdas en otros lenguajes de programación.

Definiendo y Usando Closures

Los closures pueden capturar y almacenar referencias a cualquier constante y variable del contexto en el que se definen.

let nombres = ["Chris", "Alex", "Ewa", "Barry", "Daniella"]

let nombresOrdenados = nombres.sorted(by: { (s1: String, s2: String) -> Bool in
    return s1 < s2
})

print(nombresOrdenados) // Salida: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]
  • { (s1: String, s2: String) -> Bool in return s1 < s2 }: Sintaxis de expresión de closure.

Nombres de Argumentos Abreviados

Swift proporciona nombres de argumentos abreviados para hacer los closures más concisos.

let nombresOrdenados = nombres.sorted(by: { $0 < $1 })
print(nombresOrdenados) // Salida: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]
  • $0, $1: Nombres de argumentos abreviados para el primer y segundo parámetro.

Closures de Clausura

Si un closure es el último argumento de una función, puedes usar la sintaxis de closure de clausura.

let nombresOrdenados = nombres.sorted { $0 < $1 }
print(nombresOrdenados) // Salida: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]

Capturando Valores

Los closures pueden capturar constantes y variables del contexto circundante en el que se definen.

func makeIncrementer(forIncrement amount: Int) -> () -> Int {
    var total = 0
    let incrementer: () -> Int = {
        total += amount
        return total
    }
    return incrementer
}

let incrementByTwo = makeIncrementer(forIncrement: 2)
print(incrementByTwo()) // Salida: 2
print(incrementByTwo()) // Salida: 4
  • total += amount: Capturando y modificando la variable total del contexto circundante.

Conclusión

Las funciones y los closures son conceptos fundamentales en la programación en Swift. Las funciones te permiten encapsular código para su reutilización y claridad, mientras que los closures proporcionan una forma poderosa de capturar y manipular valores de su contexto circundante. Dominar estos conceptos te permitirá escribir código más modular, flexible y conciso en Swift.
