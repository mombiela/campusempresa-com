Introducción

En Vue.js, los enlaces de clase y estilo son características poderosas que te permiten aplicar dinámicamente clases CSS y estilos en línea a tus componentes. Este tema cubrirá cómo usar estos enlaces de manera efectiva, desde el uso básico hasta el avanzado.

Enlaces de Clase

Enlace de Clase Básico

Vue.js te permite enlazar clases a elementos usando la directiva v-bind:class o su abreviatura :class.

Ejemplo:

<div :class="{'active': isActive}">¡Hola, Vue!</div>

En este ejemplo, la clase active se aplicará al elemento div si isActive es true.

Sintaxis de Objeto

Puedes usar un objeto para aplicar condicionalmente múltiples clases.

Ejemplo:

<div :class="{'active': isActive, 'disabled': isDisabled}">¡Hola, Vue!</div>

Aquí, la clase active se aplicará si isActive es true, y la clase disabled se aplicará si isDisabled es true.

Sintaxis de Array

También puedes usar un array para aplicar múltiples clases.

Ejemplo:

<div :class="[classA, classB]">¡Hola, Vue!</div>

En este caso, tanto classA como classB se aplicarán al elemento div.

Clases Computadas

Puedes usar propiedades computadas para generar dinámicamente nombres de clase.

Ejemplo:

<div :class="computedClass">¡Hola, Vue!</div>

<script>
export default {
  data() {
    return {
      isActive: true,
      isDisabled: false
    };
  },
  computed: {
    computedClass() {
      return {
        active: this.isActive,
        disabled: this.isDisabled
      };
    }
  }
};
</script>

Este enfoque mantiene tu plantilla limpia y hace que tu lógica de clases sea más fácil de gestionar.

Enlaces de Estilo

Enlace de Estilo Básico

Vue.js te permite enlazar estilos en línea usando la directiva v-bind:style o su abreviatura :style.

Ejemplo:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">¡Hola, Vue!</div>

En este ejemplo, el elemento div tendrá sus estilos color y fontSize configurados dinámicamente según los valores de activeColor y fontSize.

Sintaxis de Objeto

Puedes usar un objeto para aplicar múltiples estilos.

Ejemplo:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">¡Hola, Vue!</div>

Aquí, tanto los estilos color como fontSize se aplican al elemento div.

Sintaxis de Array

También puedes usar un array para aplicar múltiples objetos de estilo.

Ejemplo:

<div :style="[baseStyles, overridingStyles]">¡Hola, Vue!</div>

En este caso, los estilos de baseStyles y overridingStyles se aplicarán al elemento div. Si hay estilos en conflicto, el último en el array tendrá prioridad.

Estilos Computados

Puedes usar propiedades computadas para generar dinámicamente estilos.

Ejemplo:

<div :style="computedStyles">¡Hola, Vue!</div>

<script>
export default {
  data() {
    return {
      activeColor: 'red',
      fontSize: 14
    };
  },
  computed: {
    computedStyles() {
      return {
        color: this.activeColor,
        fontSize: this.fontSize + 'px'
      };
    }
  }
};
</script>

Este enfoque mantiene tu plantilla limpia y hace que tu lógica de estilos sea más fácil de gestionar.

Resumen

Los enlaces de clase y estilo en Vue.js proporcionan una forma flexible y poderosa de aplicar dinámicamente clases CSS y estilos en línea a tus componentes. Al entender y utilizar la sintaxis de objeto y array, así como las propiedades computadas, puedes crear aplicaciones Vue.js más dinámicas y mantenibles.
