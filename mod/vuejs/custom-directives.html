Introducción a las Directivas Personalizadas

Las directivas personalizadas en Vue.js te permiten extender la funcionalidad de los elementos HTML. Aunque Vue proporciona algunas directivas integradas como v-if, v-for y v-bind, las directivas personalizadas te dan la flexibilidad de crear tus propios comportamientos reutilizables.

Conceptos Clave

  • Directivas: Tokens especiales en el marcado que le dicen a la biblioteca que haga algo con un elemento del DOM.
  • Funciones Hook: Hooks de ciclo de vida para directivas, tales como bind, inserted, update, componentUpdated y unbind.

Creando una Directiva Personalizada Básica

Para crear una directiva personalizada, usas el método Vue.directive. Aquí tienes un ejemplo simple de una directiva personalizada que cambia el color de fondo de un elemento.

Vue.directive('highlight', {
  bind(el, binding) {
    el.style.backgroundColor = binding.value;
  }
});

Explicación

  • Vue.directive('highlight', {...}): Registra una nueva directiva llamada highlight.
  • bind(el, binding): El hook bind se llama solo una vez, cuando la directiva se enlaza por primera vez al elemento. Aquí, el es el elemento al que está enlazada la directiva, y binding contiene información sobre la directiva, como su valor.

Uso

<div v-highlight="'yellow'">Esto tendrá un fondo amarillo.</div>

Directivas Personalizadas Avanzadas

Las directivas personalizadas pueden ser más complejas y pueden usar múltiples hooks de ciclo de vida para gestionar el DOM.

Ejemplo: Directiva de Tooltip

Vamos a crear una directiva de tooltip que muestra un tooltip cuando el usuario pasa el ratón sobre un elemento.

Vue.directive('tooltip', {
  bind(el, binding) {
    let tooltip = document.createElement('div');
    tooltip.className = 'tooltip';
    tooltip.innerText = binding.value;
    el.tooltip = tooltip;
    el.addEventListener('mouseenter', () => {
      document.body.appendChild(tooltip);
      let rect = el.getBoundingClientRect();
      tooltip.style.left = `${rect.left + window.scrollX}px`;
      tooltip.style.top = `${rect.top + window.scrollY - tooltip.offsetHeight}px`;
    });
    el.addEventListener('mouseleave', () => {
      document.body.removeChild(tooltip);
    });
  },
  unbind(el) {
    if (el.tooltip) {
      el.removeEventListener('mouseenter');
      el.removeEventListener('mouseleave');
    }
  }
});

Explicación

  • bind: Crea el elemento tooltip y configura los event listeners para mouseenter y mouseleave.
  • unbind: Limpia los event listeners cuando la directiva se desliga del elemento.

Uso

<button v-tooltip="'Este es un tooltip'">Pasa el ratón sobre mí</button>

Funciones Hook de Directivas

Vue.js proporciona varias funciones hook para directivas personalizadas:

Función Hook Descripción
bind Llamada solo una vez, cuando la directiva se enlaza por primera vez al elemento.
inserted Llamada cuando el elemento enlazado ha sido insertado en su nodo padre.
update Llamada cuando el VNode del componente se actualiza, pero posiblemente antes de que sus hijos se actualicen.
componentUpdated Llamada después de que el VNode del componente y sus hijos se hayan actualizado.
unbind Llamada solo una vez, cuando la directiva se desliga del elemento.

Resumen

Las directivas personalizadas en Vue.js son una forma poderosa de extender la funcionalidad de tus aplicaciones. Te permiten encapsular comportamientos reutilizables y aplicarlos a cualquier elemento del DOM. Al comprender y utilizar los hooks de ciclo de vida proporcionados por Vue, puedes crear directivas personalizadas complejas e interactivas.

Puntos Clave

  • Las directivas personalizadas mejoran los elementos HTML con comportamientos reutilizables.
  • Usa Vue.directive para registrar una directiva personalizada.
  • Los hooks de ciclo de vida (bind, inserted, update, componentUpdated, unbind) gestionan el comportamiento de la directiva.
  • Las directivas personalizadas pueden variar desde cambios de estilo simples hasta interacciones complejas como tooltips.

Al dominar las directivas personalizadas, puedes mejorar significativamente la modularidad y reutilización de tus aplicaciones Vue.js.
