En este proyecto, construiremos una aplicación de chat en tiempo real usando Vue.js. Este proyecto te ayudará a entender cómo manejar datos en tiempo real, gestionar el estado e integrarte con servicios de backend. Al final de este proyecto, tendrás una comprensión sólida de cómo crear una aplicación web dinámica e interactiva.

Conceptos Clave

  • Configurar un proyecto Vue.js
  • Crear y gestionar componentes
  • Manejar el estado con Vuex
  • Integrarse con un backend usando Firebase
  • Implementar actualizaciones en tiempo real con Firebase Realtime Database
  • Gestionar la autenticación de usuarios

Configuración del Proyecto

Primero, configuremos un nuevo proyecto Vue.js usando Vue CLI.

# Instalar Vue CLI si aún no lo has hecho
npm install -g @vue/cli

# Crear un nuevo proyecto
vue create real-time-chat

# Navegar al directorio del proyecto
cd real-time-chat

Crearemos varios componentes para nuestra aplicación de chat:

  • ChatApp.vue: El componente principal que contendrá la interfaz de chat.

MessageInput.vue

<template>
  <div class="message-input">
    <input v-model="newMessage" @keyup.enter="handleSend" placeholder="Escribe un mensaje" />
    <button @click="handleSend">Enviar</button>
  </div>
</template>

<script>
export default {
  name: 'MessageInput',
  data() {
    return {
      newMessage: ''
    };
  },
  methods: {
    handleSend() {
      if (this.newMessage.trim() !== '') {
        this.$emit('sendMessage', this.newMessage);
        this.newMessage = '';
      }
    }
  }
};
</script>

Manejo del Estado con Vuex

Para gestionar el estado de nuestra aplicación, usaremos Vuex. Instala Vuex y configúralo en tu proyecto.

npm install vuex

store/index.js

import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    messages: []
  },
  mutations: {
    addMessage(state, message) {
      state.messages.push(message);
    }
  },
  actions: {
    sendMessage({ commit }, message) {
      commit('addMessage', message);
    }
  },
  getters: {
    messages: state => state.messages
  }
});

Integración con Firebase

Usaremos Firebase para nuestro backend para manejar actualizaciones en tiempo real y autenticación de usuarios. Primero, configura Firebase en tu proyecto.

npm install firebase

firebaseConfig.js

import firebase from 'firebase/app';
import 'firebase/database';
import 'firebase/auth';

const firebaseConfig = {
  apiKey: "TU_API_KEY",
  authDomain: "TU_AUTH_DOMAIN",
  databaseURL: "TU_DATABASE_URL",
  projectId: "TU_PROJECT_ID",
  storageBucket: "TU_STORAGE_BUCKET",
  messagingSenderId: "TU_MESSAGING_SENDER_ID",
  appId: "TU_APP_ID"
};

firebase.initializeApp(firebaseConfig);

const database = firebase.database();
const auth = firebase.auth();

export { database, auth };

Actualización de la Tienda Vuex para Firebase

Actualiza la tienda Vuex para manejar actualizaciones en tiempo real desde Firebase.

import Vue from 'vue';
import Vuex from 'vuex';
import { database } from '../firebaseConfig';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    messages: []
  },
  mutations: {
    setMessages(state, messages) {
      state.messages = messages;
    },
    addMessage(state, message) {
      state.messages.push(message);
    }
  },
  actions: {
    fetchMessages({ commit }) {
      database.ref('messages').on('value', snapshot => {
        commit('setMessages', snapshot.val() || []);
      });
    },
    sendMessage({ commit }, message) {
      database.ref('messages').push(message);
      commit('addMessage', message);
    }
  },
  getters: {
    messages: state => state.messages
  }
});

Gestión de la Autenticación de Usuarios

Usaremos Firebase Authentication para gestionar el inicio de sesión y el registro de usuarios.

Auth.vue

<template>
  <div class="auth">
    <input v-model="email" placeholder="Correo electrónico" />
    <input v-model="password" type="password" placeholder="Contraseña" />
    <button @click="login">Iniciar Sesión</button>
    <button @click="register">Registrarse</button>
  </div>
</template>

<script>
import { auth } from '../firebaseConfig';

export default {
  name: 'Auth',
  data() {
    return {
      email: '',
      password: ''
    };
  },
  methods: {
    async login() {
      try {
        await auth.signInWithEmailAndPassword(this.email, this.password);
        this.$router.push('/chat');
      } catch (error) {
        console.error(error);
      }
    },
    async register() {
      try {
        await auth.createUserWithEmailAndPassword(this.email, this.password);
        this.$router.push('/chat');
      } catch (error) {
        console.error(error);
      }
    }
  }
};
</script>
