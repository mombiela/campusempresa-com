Introducción

En esta sección, exploraremos los componentes fundamentales en React Native: View, Text e Image. Estos componentes son los bloques de construcción para crear interfaces de usuario en aplicaciones React Native. Entender cómo usar estos componentes de manera efectiva es crucial para cualquier desarrollador de React Native, desde principiantes hasta usuarios avanzados.

Componente View

El componente View es un contenedor que soporta diseño con Flexbox, estilo, manejo de toques y controles de accesibilidad. Es análogo a un div en HTML.

Conceptos Clave

  • Contenedor: Se usa para envolver otros componentes.
  • Estilización: Puede ser estilizado usando la prop style.
  • Flexbox: Soporta diseño Flexbox para organizar componentes hijos.

Ejemplo

import React from 'react';
import { View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <View style={styles.box1} />
      <View style={styles.box2} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'row',
    justifyContent: 'space-around',
    alignItems: 'center',
  },
  box1: {
    width: 50,
    height: 50,
    backgroundColor: 'red',
  },
  box2: {
    width: 50,
    height: 50,
    backgroundColor: 'blue',
  },
});

export default App;

Explicación

  • Contenedor: El View exterior actúa como un contenedor para los componentes View internos.
  • Flexbox: Las propiedades flexDirection, justifyContent y alignItems se usan para organizar los componentes hijos.

Componente Text

El componente Text se usa para mostrar texto en tu aplicación. Soporta anidamiento, estilización y manejo de toques.

Conceptos Clave

  • Anidamiento: Los componentes de texto pueden ser anidados unos dentro de otros.
  • Estilización: Puede ser estilizado usando la prop style.
  • Manejo de Toques: Puede manejar eventos de toque.

Ejemplo

import React from 'react';
import { Text, StyleSheet } from 'react-native';

const App = () => {
  return (
    <Text style={styles.text}>
      Hola, <Text style={styles.bold}>React Native</Text>!
    </Text>
  );
};

const styles = StyleSheet.create({
  text: {
    fontSize: 20,
    color: 'black',
  },
  bold: {
    fontWeight: 'bold',
  },
});

export default App;

Explicación

  • Anidamiento: El componente Text con el estilo bold está anidado dentro del componente Text padre.
  • Estilización: El estilo text establece el tamaño de la fuente y el color, mientras que el estilo bold establece el peso de la fuente.

Componente Image

El componente Image se usa para mostrar imágenes en tu aplicación. Soporta varios formatos de imagen y puede ser estilizado y redimensionado.

Conceptos Clave

  • Fuente: La prop source especifica la imagen a mostrar.
  • Estilización: Puede ser estilizado usando la prop style.
  • Redimensionamiento: Soporta modos de redimensionamiento como cover, contain, etc.

Ejemplo

import React from 'react';
import { Image, StyleSheet, View } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Image
        style={styles.image}
        source={{ uri: 'https://reactnative.dev/img/tiny_logo.png' }}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  image: {
    width: 100,
    height: 100,
  },
});

export default App;

Explicación

  • Fuente: La prop source se usa para especificar la URL de la imagen.
  • Estilización: El estilo image establece el ancho y la altura de la imagen.
  • Redimensionamiento: La imagen será redimensionada para ajustarse a las dimensiones especificadas.

Conclusión

En esta sección, cubrimos los componentes básicos en React Native: View, Text e Image. Estos componentes son esenciales para construir interfaces de usuario. Aprendimos cómo usarlos, estilizarlos y manejar sus propiedades. Dominar estos componentes proporcionará una base sólida para crear aplicaciones más complejas e interactivas en React Native.
