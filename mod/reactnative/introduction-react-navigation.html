Introducción a React Navigation
Stack Navigator ►

React Navigation es una biblioteca popular para el enrutamiento y la navegación en aplicaciones React Native. Proporciona una forma de navegar entre diferentes pantallas y gestionar el historial de navegación. Este tema cubrirá los conceptos básicos de React Navigation, incluyendo su configuración, la creación de navegadores y el manejo de acciones de navegación.

Configuración de React Navigation

Para comenzar con React Navigation, necesitas instalar los paquetes necesarios. Sigue estos pasos:

  1. Instala React Navigation y sus dependencias:

    npm install @react-navigation/native
npm install @react-navigation/native-stack
npm install react-native-screens react-native-safe-area-context

  2. Envuelve tu aplicación con NavigationContainer:

    import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';

export default function App() {
  return (
    <NavigationContainer>
      {/* Resto del código de tu aplicación */}
    </NavigationContainer>
  );
}

Creación de Navegadores

React Navigation proporciona varios tipos de navegadores para manejar diferentes patrones de navegación. Los navegadores más comúnmente utilizados son:

  • Stack Navigator
  • Tab Navigator
  • Drawer Navigator

Stack Navigator

Un Stack Navigator te permite navegar entre pantallas de una manera similar a una pila, donde cada nueva pantalla se coloca encima de la anterior.

  1. Instala el stack navigator:

    npm install @react-navigation/stack

  2. Crea un Stack Navigator:

    import { createNativeStackNavigator } from '@react-navigation/native-stack';

const Stack = createNativeStackNavigator();

function MyStack() {
  return (
    <Stack.Navigator>
      <Stack.Screen name="Home" component={HomeScreen} />
      <Stack.Screen name="Details" component={DetailsScreen} />
    </Stack.Navigator>
  );
}

  3. Usa el Stack Navigator en tu aplicación:

    export default function App() {
  return (
    <NavigationContainer>
      <MyStack />
    </NavigationContainer>
  );
}

Tab Navigator

Un Tab Navigator te permite crear un sistema de navegación basado en pestañas.

  1. Instala el tab navigator:

    npm install @react-navigation/bottom-tabs

  2. Crea un Tab Navigator:

    import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';

const Tab = createBottomTabNavigator();

function MyTabs() {
  return (
    <Tab.Navigator>
      <Tab.Screen name="Home" component={HomeScreen} />
      <Tab.Screen name="Settings" component={SettingsScreen} />
    </Tab.Navigator>
  );
}

  3. Usa el Tab Navigator en tu aplicación:

    export default function App() {
  return (
    <NavigationContainer>
      <MyTabs />
    </NavigationContainer>
  );
}

Drawer Navigator

Un Drawer Navigator te permite crear un cajón de navegación que se desliza desde el lado de la pantalla.

  1. Instala el drawer navigator:

    npm install @react-navigation/drawer

  2. Crea un Drawer Navigator:

    import { createDrawerNavigator } from '@react-navigation/drawer';

const Drawer = createDrawerNavigator();

function MyDrawer() {
  return (
    <Drawer.Navigator>
      <Drawer.Screen name="Home" component={HomeScreen} />
      <Drawer.Screen name="Profile" component={ProfileScreen} />
    </Drawer.Navigator>
  );
}

  3. Usa el Drawer Navigator en tu aplicación:

    export default function App() {
  return (
    <NavigationContainer>
      <MyDrawer />
    </NavigationContainer>
  );
}

Manejo de Acciones de Navegación

React Navigation proporciona varios métodos para manejar acciones de navegación, como navegar a una nueva pantalla, volver a la pantalla anterior y restablecer el estado de navegación.

  • Navegar a una nueva pantalla:

    function HomeScreen({ navigation }) {
  return (
    <Button
      title="Go to Details"
      onPress={() => navigation.navigate('Details')}
    />
  );
}

  • Volver a la pantalla anterior:

    function DetailsScreen({ navigation }) {
  return (
    <Button
      title="Go back"
      onPress={() => navigation.goBack()}
    />
  );
}

  • Restablecer el estado de navegación:

    function HomeScreen({ navigation }) {
  return (
    <Button
      title="Reset to Home"
      onPress={() => navigation.reset({
        index: 0,
        routes: [{ name: 'Home' }],
      })}
    />
  );
}

Conclusión

React Navigation es una biblioteca poderosa y flexible para manejar la navegación en aplicaciones React Native. Al entender cómo configurar y usar diferentes tipos de navegadores, puedes crear una experiencia de navegación fluida e intuitiva para tus usuarios. Practica usando los navegadores Stack, Tab y Drawer, y experimenta con las acciones de navegación para volverte competente en la gestión de la navegación en tus aplicaciones React Native.
