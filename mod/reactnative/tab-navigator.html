Introducción al Navegador de Pestañas

La navegación por pestañas es un patrón común en las aplicaciones móviles donde los usuarios pueden cambiar entre diferentes pantallas o vistas tocando las pestañas. En React Native, la biblioteca react-navigation proporciona una forma poderosa y flexible de implementar la navegación por pestañas.

Configuración del Entorno

Antes de sumergirnos en la creación de un Navegador de Pestañas, asegúrate de tener instaladas las dependencias necesarias.

  1. Instalar React Navigation:

    npm install @react-navigation/native

  2. Instalar las dependencias requeridas:

    npm install @react-navigation/bottom-tabs
npm install react-native-screens react-native-safe-area-context

  3. Envuelve tu aplicación en NavigationContainer:

    import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';

export default function App() {
  return (
    <NavigationContainer>
      {/* Resto del código de tu aplicación */}
    </NavigationContainer>
  );
}

Creando un Navegador de Pestañas Básico

Para crear un navegador de pestañas básico, sigue estos pasos:

  1. Importar los módulos necesarios:

    import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import HomeScreen from './screens/HomeScreen';
import SettingsScreen from './screens/SettingsScreen';

  2. Crear el Navegador de Pestañas:

    const Tab = createBottomTabNavigator();

function MyTabs() {
  return (
    <Tab.Navigator>
      <Tab.Screen name="Home" component={HomeScreen} />
      <Tab.Screen name="Settings" component={SettingsScreen} />
    </Tab.Navigator>
  );
}

  3. Usar el Navegador de Pestañas en tu Aplicación:

    export default function App() {
  return (
    <NavigationContainer>
      <MyTabs />
    </NavigationContainer>
  );
}

Personalizando el Navegador de Pestañas

Puedes personalizar la apariencia y el comportamiento del Navegador de Pestañas usando varias opciones.

Opciones de la Barra de Pestañas

  • tabBarOptions: Personaliza la apariencia de la barra de pestañas. 
    <Tab.Navigator
  tabBarOptions={{
    activeTintColor: 'tomato',
    inactiveTintColor: 'gray',
  }}
>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Opciones de Pantalla

  • tabBarIcon: Añade iconos a las pestañas. 
    import { Ionicons } from '@expo/vector-icons';

<Tab.Navigator
  screenOptions={({ route }) => ({
    tabBarIcon: ({ focused, color, size }) => {
      let iconName;

      if (route.name === 'Home') {
        iconName = focused ? 'home' : 'home-outline';
      } else if (route.name === 'Settings') {
        iconName = focused ? 'settings' : 'settings-outline';
      }

      return <Ionicons name={iconName} size={size} color={color} />;
    },
  })}
  tabBarOptions={{
    activeTintColor: 'tomato',
    inactiveTintColor: 'gray',
  }}
>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Personalización Avanzada

Para una personalización más avanzada, puedes usar componentes personalizados para la barra de pestañas.

Componente Personalizado de la Barra de Pestañas

  • Creando una barra de pestañas personalizada: 
    import { View, Text, TouchableOpacity } from 'react-native';

function MyTabBar({ state, descriptors, navigation }) {
  return (
    <View style={{ flexDirection: 'row' }}>
      {state.routes.map((route, index) => {
        const { options } = descriptors[route.key];
        const label =
          options.tabBarLabel !== undefined
            ? options.tabBarLabel
            : options.title !== undefined
            ? options.title
            : route.name;

        const isFocused = state.index === index;

        const onPress = () => {
          const event = navigation.emit({
            type: 'tabPress',
            target: route.key,
          });

          if (!isFocused && !event.defaultPrevented) {
            navigation.navigate(route.name);
          }
        };

        const onLongPress = () => {
          navigation.emit({
            type: 'tabLongPress',
            target: route.key,
          });
        };

        return (
          <TouchableOpacity
            key={index}
            accessibilityRole="button"
            accessibilityStates={isFocused ? ['selected'] : []}
            accessibilityLabel={options.tabBarAccessibilityLabel}
            testID={options.tabBarTestID}
            onPress={onPress}
            onLongPress={onLongPress}
            style={{ flex: 1 }}
          >
            <Text style={{ color: isFocused ? '#673ab7' : '#222' }}>
              {label}
            </Text>
          </TouchableOpacity>
        );
      })}
    </View>
  );
}

<Tab.Navigator tabBar={props => <MyTabBar {...props} />}>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Conclusión

La navegación por pestañas es una parte esencial de muchas aplicaciones móviles, proporcionando una forma fácil de usar para cambiar entre diferentes pantallas. El Navegador de Pestañas de React Navigation ofrece una solución flexible y personalizable para implementar la navegación por pestañas en tus aplicaciones React Native. Al comprender los conceptos básicos y explorar las opciones de personalización avanzada, puedes crear una experiencia de navegación fluida e intuitiva para tus usuarios.
