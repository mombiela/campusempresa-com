Tema 1-4: Variables y Tipos de Datos

Introducción

En este tema, aprenderemos sobre las variables y los tipos de datos en Python. Las variables son fundamentales en cualquier lenguaje de programación, ya que permiten almacenar y manipular datos. Python es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable antes de usarla.

1. Variables en Python

Las variables en Python se utilizan para almacenar datos que pueden ser utilizados y manipulados a lo largo del programa. Aquí hay algunas reglas básicas para nombrar variables:

  • Los nombres de las variables deben comenzar con una letra (a-z, A-Z) o un guion bajo (_).
  • Los nombres de las variables pueden contener letras, números y guiones bajos.
  • Los nombres de las variables son sensibles a mayúsculas y minúsculas (por ejemplo, edad y Edad son diferentes).

Ejemplo:

# Asignación de variables
nombre = "Juan"
edad = 25
altura = 1.75

# Imprimir variables
print(nombre)  # Salida: Juan
print(edad)    # Salida: 25
print(altura)  # Salida: 1.75

2. Tipos de Datos Básicos

Python tiene varios tipos de datos incorporados. Los más comunes son:

  • Enteros (int): Números enteros, positivos o negativos, sin decimales.
  • Flotantes (float): Números reales con punto decimal.
  • Cadenas (str): Secuencias de caracteres.
  • Booleanos (bool): Valores de verdad (True o False).

Ejemplo:

# Entero
numero_entero = 10
print(type(numero_entero))  # Salida: <class 'int'>

# Flotante
numero_flotante = 10.5
print(type(numero_flotante))  # Salida: <class 'float'>

# Cadena
cadena_texto = "Hola, Mundo"
print(type(cadena_texto))  # Salida: <class 'str'>

# Booleano
es_mayor = True
print(type(es_mayor))  # Salida: <class 'bool'>

3. Conversión de Tipos

En Python, es posible convertir entre diferentes tipos de datos utilizando funciones de conversión de tipo.

  • int(): Convierte a entero.
  • float(): Convierte a flotante.
  • str(): Convierte a cadena.
  • bool(): Convierte a booleano.

Ejemplo:

# Conversión de tipos
numero = "123"
numero_entero = int(numero)
print(type(numero_entero))  # Salida: <class 'int'>

decimal = 12.34
decimal_cadena = str(decimal)
print(type(decimal_cadena))  # Salida: <class 'str'>

verdad = 1
verdad_booleano = bool(verdad)
print(type(verdad_booleano))  # Salida: <class 'bool'>

4. Ejemplos Prácticos

Ejemplo 1: Operaciones con Variables

# Definir variables
a = 10
b = 20

# Sumar variables
suma = a + b
print("Suma:", suma)  # Salida: Suma: 30

# Restar variables
resta = a - b
print("Resta:", resta)  # Salida: Resta: -10

# Multiplicar variables
multiplicacion = a * b
print("Multiplicación:", multiplicacion)  # Salida: Multiplicación: 200

# Dividir variables
division = a / b
print("División:", division)  # Salida: División: 0.5

Ejemplo 2: Concatenación de Cadenas

# Definir cadenas
saludo = "Hola"
nombre = "Juan"

# Concatenar cadenas
mensaje = saludo + ", " + nombre + "!"
print(mensaje)  # Salida: Hola, Juan!

5. Ejercicios

Ejercicio 1: Asignación y Operaciones Básicas

  1. Crea una variable llamada x y asígnale el valor 10.
  2. Crea una variable llamada y y asígnale el valor 5.
  3. Crea una variable llamada suma que almacene la suma de x y y.
  4. Imprime el valor de suma.

Ejercicio 2: Conversión de Tipos

  1. Crea una variable llamada cadena_numero y asígnale el valor "123".
  2. Convierte cadena_numero a un entero y almacénalo en una variable llamada numero_entero.
  3. Imprime el tipo de numero_entero.

Soluciones

Solución Ejercicio 1

# Ejercicio 1
x = 10
y = 5
suma = x + y
print(suma)  # Salida: 15

Solución Ejercicio 2

# Ejercicio 2
cadena_numero = "123"
numero_entero = int(cadena_numero)
print(type(numero_entero))  # Salida: <class 'int'>

Conclusión

En este tema, hemos aprendido sobre las variables y los tipos de datos en Python. Hemos visto cómo asignar valores a las variables, los tipos de datos básicos y cómo convertir entre diferentes tipos de datos. Estos conceptos son fundamentales para cualquier programador y serán la base para los temas más avanzados que veremos en los siguientes módulos.
