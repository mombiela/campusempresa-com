Introducción

En Python, las variables y constantes son conceptos fundamentales que te permiten almacenar y manipular datos. Entender cómo usarlas de manera efectiva es crucial para escribir código eficiente y legible.

Variables

¿Qué es una Variable?

  • Una variable es una ubicación nombrada utilizada para almacenar datos en la memoria.
  • El valor almacenado en una variable puede cambiar durante la ejecución del programa.

Declaración de Variables

En Python, no necesitas declarar explícitamente el tipo de una variable. Simplemente asignas un valor a un nombre de variable.

# Ejemplo de declaración de variable
x = 10
nombre = "Alicia"
es_estudiante = True

Reglas para Nombrar Variables

  • Los nombres de las variables deben comenzar con una letra o un guion bajo (_).
  • El resto del nombre puede contener letras, números o guiones bajos.
  • Los nombres de las variables son sensibles a mayúsculas y minúsculas (edad y Edad son variables diferentes).

Ejemplos de Nombres de Variables Válidos e Inválidos

Nombres Válidos Nombres Inválidos
edad 1edad
_nombre nombre!
suma_total suma-total

Actualización de Variables

Puedes actualizar el valor de una variable asignándole un nuevo valor.

# Actualización de una variable
x = 10
x = 20  # x ahora tiene el valor 20

Tipos de Variables

Python es de tipado dinámico, lo que significa que el tipo de una variable se determina en tiempo de ejecución. Los tipos comunes incluyen:

  • int para enteros
  • float para números de punto flotante
  • str para cadenas de texto
  • bool para valores booleanos
# Ejemplos de diferentes tipos
edad = 25          # int
altura = 5.9       # float
nombre = "Alicia"  # str
es_estudiante = True # bool

Constantes

¿Qué es una Constante?

  • Una constante es un tipo de variable cuyo valor no puede cambiar una vez que se ha asignado.
  • Python no tiene tipos de constantes incorporados, pero por convención, las constantes se escriben en letras mayúsculas.

Declaración de Constantes

Aunque Python no impone valores constantes, puedes indicar que una variable debe ser tratada como una constante usando letras mayúsculas.

# Ejemplo de declaración de constante
PI = 3.14159
MAX_USUARIOS = 100

Uso de Constantes

Las constantes se utilizan para definir valores que no deben cambiar a lo largo del programa, como valores de configuración o parámetros fijos.

# Uso de constantes en un programa
PI = 3.14159
radio = 5
area = PI * (radio ** 2)
print("Área del círculo:", area)

Mejores Prácticas

Convenciones de Nombres

  • Usa nombres descriptivos para las variables para hacer tu código más legible.
  • Usa letras mayúsculas para las constantes para distinguirlas de las variables regulares.

Evita Números Mágicos

  • Evita usar valores codificados directamente en tu código (números mágicos). En su lugar, usa constantes para darles nombres significativos.
# Evita números mágicos
descuento = 0.1

# Mejor enfoque usando constantes
TASA_DE_DESCUENTO = 0.1

Conclusión

Entender las variables y constantes es esencial para cualquier programador de Python. Las variables te permiten almacenar y manipular datos, mientras que las constantes te ayudan a definir valores fijos que no deben cambiar. Siguiendo las mejores prácticas y usando nombres significativos, puedes escribir código más legible y mantenible.
