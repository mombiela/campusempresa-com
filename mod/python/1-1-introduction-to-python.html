Tema 1-1: Introducción a Python

Objetivos del Tema

  • Comprender qué es Python y sus características principales.
  • Conocer las aplicaciones y ventajas de usar Python.
  • Familiarizarse con la historia y evolución de Python.

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Python se destaca por su sintaxis clara y legible, lo que facilita su aprendizaje y uso.

Características Principales de Python

  1. Sintaxis Clara y Legible:

    • Python utiliza una sintaxis que permite a los programadores expresar conceptos en menos líneas de código en comparación con otros lenguajes como C++ o Java.
    • Ejemplo: 
      # Ejemplo de código Python
print("Hola, Mundo!")

  2. Interpretado:

    • Python es un lenguaje interpretado, lo que significa que el código se ejecuta línea por línea, facilitando la depuración y el desarrollo interactivo.

  3. Multiparadigma:

    • Python soporta múltiples paradigmas de programación, incluyendo la programación orientada a objetos, la programación imperativa y la programación funcional.

  4. Bibliotecas Estándar Amplias:

    • Python viene con una extensa biblioteca estándar que incluye módulos y paquetes para realizar diversas tareas, desde manipulación de archivos hasta desarrollo web.

  5. Portabilidad:

    • El código Python puede ejecutarse en diferentes plataformas sin necesidad de modificaciones, lo que lo hace altamente portátil.

  6. Comunidad Activa:

    • Python cuenta con una comunidad activa y creciente, lo que significa que hay una gran cantidad de recursos, tutoriales y soporte disponibles.

Aplicaciones de Python

Python se utiliza en una amplia variedad de aplicaciones, incluyendo:

  • Desarrollo Web: Frameworks como Django y Flask.
  • Ciencia de Datos y Análisis: Bibliotecas como NumPy, Pandas y Matplotlib.
  • Inteligencia Artificial y Aprendizaje Automático: Herramientas como TensorFlow y Scikit-learn.
  • Automatización y Scripting: Scripts para automatizar tareas repetitivas.
  • Desarrollo de Juegos: Motores de juegos como Pygame.
  • Desarrollo de Software: Aplicaciones de escritorio y herramientas de desarrollo.

Historia y Evolución de Python

  • 1980s: Guido van Rossum comenzó a trabajar en Python como un sucesor del lenguaje ABC.
  • 1991: Se lanzó la primera versión de Python (0.9.0).
  • 2000: Se lanzó Python 2.0, que introdujo nuevas características como la recolección de basura y la comprensión de listas.
  • 2008: Se lanzó Python 3.0, una versión mayor que no es completamente compatible con Python 2.x, pero que introdujo mejoras significativas en la sintaxis y la funcionalidad del lenguaje.
  • Presente: Python sigue evolucionando con nuevas versiones y mejoras, y es uno de los lenguajes de programación más populares y utilizados en el mundo.

Ejercicio Práctico

Ejercicio 1: Instalación de Python y Ejecución de un Script Simple

  1. Instalación de Python:

    • Visita el sitio web oficial de Python (https://www.python.org/) y descarga la última versión de Python.
    • Sigue las instrucciones de instalación para tu sistema operativo (Windows, macOS, Linux).

  2. Ejecución de un Script Simple:

    • Abre tu editor de texto favorito (por ejemplo, Visual Studio Code, Sublime Text, o el editor integrado IDLE).
    • Escribe el siguiente código en un nuevo archivo y guárdalo con la extensión .py (por ejemplo, hola_mundo.py): 
      print("Hola, Mundo!")
    • Abre una terminal o línea de comandos.
    • Navega hasta el directorio donde guardaste el archivo hola_mundo.py.
    • Ejecuta el script usando el comando: 
      python hola_mundo.py
    • Deberías ver la salida Hola, Mundo! en la terminal.

Conclusión

En este tema, hemos introducido Python, destacando sus características principales, aplicaciones y evolución histórica. Ahora que tienes una comprensión básica de qué es Python y por qué es tan popular, estás listo para configurar tu entorno de desarrollo y comenzar a escribir tus primeros programas en Python.
