9-4: Manejo de Formularios en Flask
Integración de Bases de Datos ►

En esta sección, aprenderemos cómo manejar formularios en Flask, una parte esencial del desarrollo web. Los formularios permiten a los usuarios enviar datos a la aplicación web, que luego pueden ser procesados y almacenados.

Objetivos de Aprendizaje

  • Entender cómo crear y renderizar formularios en Flask.
  • Aprender a manejar las solicitudes POST y GET.
  • Validar y procesar datos de formularios.
  • Mostrar mensajes de error y éxito.

Contenido

  1. Creación de Formularios en HTML
  2. Configuración de Rutas para Manejar Formularios
  3. Procesamiento de Datos del Formulario

1. Creación de Formularios en HTML

Primero, necesitamos crear un formulario en HTML. Aquí hay un ejemplo básico de un formulario que solicita el nombre y el correo electrónico del usuario:

<!-- templates/form.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Formulario de Ejemplo</title>
</head>
<body>
    <h1>Formulario de Ejemplo</h1>
    <form action="/submit" method="POST">
        <label for="name">Nombre:</label>
        <input type="text" id="name" name="name" required>
        <br>
        <label for="email">Correo Electrónico:</label>
        <input type="email" id="email" name="email" required>
        <br>
        <input type="submit" value="Enviar">
    </form>
</body>
</html>

2. Configuración de Rutas para Manejar Formularios

En Flask, configuramos rutas para manejar las solicitudes GET y POST. La solicitud GET se utiliza para mostrar el formulario, y la solicitud POST se utiliza para procesar los datos enviados.

# app.py
from flask import Flask, render_template, request, redirect, url_for

app = Flask(__name__)

@app.route('/form')
def form():
    return render_template('form.html')

@app.route('/submit', methods=['POST'])
def submit():
    name = request.form['name']
    email = request.form['email']
    # Procesar los datos del formulario aquí
    return f"Nombre: {name}, Correo Electrónico: {email}"

if __name__ == '__main__':
    app.run(debug=True)

3. Procesamiento de Datos del Formulario

En la función submit, utilizamos request.form para acceder a los datos enviados por el formulario. Luego, podemos procesar estos datos según sea necesario.

4. Validación de Formularios

Es importante validar los datos del formulario antes de procesarlos. Aquí hay un ejemplo de cómo hacerlo:

@app.route('/submit', methods=['POST'])
def submit():
    name = request.form['name']
    email = request.form['email']
    
    if not name or not email:
        error = "Por favor, complete todos los campos."
        return render_template('form.html', error=error)
    
    # Procesar los datos del formulario aquí
    return f"Nombre: {name}, Correo Electrónico: {email}"

6. Ejercicio Práctico

Ejercicio: Crear un formulario de registro

  1. Crea un formulario en HTML que solicite el nombre de usuario, la contraseña y la confirmación de la contraseña.
  2. Configura las rutas en Flask para manejar la solicitud GET y POST.
  3. Valida que la contraseña y la confirmación de la contraseña coincidan.
  4. Muestra un mensaje de error si las contraseñas no coinciden.

Solución:

# app.py
from flask import Flask, render_template, request

app = Flask(__name__)

@app.route('/register')
def register():
    return render_template('register.html')

@app.route('/register', methods=['POST'])
def register_post():
    username = request.form['username']
    password = request.form['password']
    confirm_password = request.form['confirm_password']
    
    if not username or not password or not confirm_password:
        error = "Por favor, complete todos los campos."
        return render_template('register.html', error=error)
    
    if password != confirm_password:
        error = "Las contraseñas no coinciden."
        return render_template('register.html', error=error)
    
    return f"Usuario registrado: {username}"

if __name__ == '__main__':
    app.run(debug=True)
<!-- templates/register.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Registro</title>
</head>
<body>
    <h1>Formulario de Registro</h1>
    {% if error %}
        <p style="color: red;">{{ error }}</p>
    {% endif %}
    <form action="/register" method="POST">
        <label for="username">Nombre de Usuario:</label>
        <input type="text" id="username" name="username" required>
        <br>
        <label for="password">Contraseña:</label>
        <input type="password" id="password" name="password" required>
        <br>
        <label for="confirm_password">Confirmar Contraseña:</label>
        <input type="password" id="confirm_password" name="confirm_password" required>
        <br>
        <input type="submit" value="Registrar">
    </form>
</body>
</html>
