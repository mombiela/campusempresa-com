¿Qué es Python?

Python es un lenguaje de programación interpretado de alto nivel conocido por su simplicidad y legibilidad. Se utiliza ampliamente en diversos dominios como el desarrollo web, análisis de datos, inteligencia artificial, computación científica y más.

  • Alto nivel: Python abstrae la mayoría de los detalles complejos del hardware del ordenador.
  • Interpretado: El código de Python se ejecuta línea por línea, lo que facilita la depuración.
  • Legibilidad: La sintaxis de Python está diseñada para ser fácil de leer y escribir.

¿Por qué aprender Python?

  • Facilidad de aprendizaje: Python tiene una sintaxis simple que imita el lenguaje natural.
  • Versatilidad: Python se puede utilizar para desarrollo web, ciencia de datos, automatización y más.
  • Soporte comunitario: Python tiene una comunidad grande y activa, lo que significa que hay muchos recursos y bibliotecas disponibles.

Configuración de Python

Instalación de Python

  1. Descargar Python: Ve al sitio web oficial de Python y descarga la última versión.
  2. Instalar Python: Sigue las instrucciones de instalación para tu sistema operativo.

Configuración de un IDE

Un Entorno de Desarrollo Integrado (IDE) puede facilitar la codificación en Python. Las opciones populares incluyen:

  • PyCharm
  • VS Code
  • Jupyter Notebook

Escribiendo tu primer programa en Python

Comencemos con un programa simple que imprime "¡Hola, Mundo!" en la pantalla.

print("¡Hola, Mundo!")
  • Explicación: La función print muestra texto en la consola. El texto a imprimir se encierra entre comillas.

Sintaxis básica

Variables y tipos de datos

Python admite varios tipos de datos, incluidos enteros, flotantes, cadenas y booleanos.

# Entero
edad = 25

# Flotante
altura = 5.9

# Cadena
nombre = "Alicia"

# Booleano
es_estudiante = True
  • Explicación: Las variables se utilizan para almacenar datos. El tipo de dato de una variable se determina por el valor asignado a ella.

Operadores básicos

Python admite operadores aritméticos, de comparación y lógicos.

# Operadores aritméticos
suma = 5 + 3
resta = 5 - 3
multiplicacion = 5 * 3
division = 5 / 3

# Operadores de comparación
igual = (5 == 3)
diferente = (5 != 3)
mayor_que = (5 > 3)
menor_que = (5 < 3)

# Operadores lógicos
operador_y = (5 > 3) and (3 > 1)
operador_o = (5 > 3) or (3 < 1)
operador_no = not(5 > 3)
  • Explicación: Los operadores se utilizan para realizar operaciones sobre variables y valores.

Estructuras de control

Sentencias If

edad = 18
if edad >= 18:
    print("Eres un adulto.")
else:
    print("Eres un menor.")
  • Explicación: La sentencia if te permite ejecutar código basado en una condición.

Bucles

Bucle For
for i in range(5):
    print(i)
  • Explicación: El bucle for itera sobre una secuencia (en este caso, números del 0 al 4).
Bucle While
cuenta = 0
while cuenta < 5:
    print(cuenta)
    cuenta += 1
  • Explicación: El bucle while continúa ejecutándose mientras la condición sea verdadera.

Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica.

def saludar(nombre):
    print(f"¡Hola, {nombre}!")

saludar("Alicia")
  • Explicación: La palabra clave def se utiliza para definir una función. La función saludar toma un parámetro nombre e imprime un saludo.

Conclusión

En esta introducción, cubrimos los conceptos básicos de Python, incluida su sintaxis, variables, operadores, estructuras de control y funciones. La simplicidad y legibilidad de Python lo convierten en una excelente opción para principiantes. A medida que avancemos en el curso, profundizaremos en temas y aplicaciones más avanzadas de Python.
