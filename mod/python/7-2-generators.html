7-2: Generadores

Introducción a los Generadores

Los generadores son una herramienta poderosa en Python que permite crear iteradores de una manera más sencilla y eficiente. A diferencia de las funciones normales que devuelven un valor y terminan, los generadores pueden pausar su ejecución y devolver un valor, y luego continuar desde donde se quedaron.

Conceptos Clave

  • Iterador: Un objeto que permite recorrer una colección de elementos uno a uno.
  • Generador: Una función que devuelve un iterador y que puede pausar su ejecución y reanudarla posteriormente.
  • yield: La palabra clave utilizada en los generadores para pausar la ejecución y devolver un valor.

Creación de un Generador

Un generador se define como una función normal, pero en lugar de usar return para devolver un valor, se usa yield.

def contador(maximo):
    contador = 0
    while contador < maximo:
        yield contador
        contador += 1

En este ejemplo, contador es un generador que produce una secuencia de números desde 0 hasta maximo - 1.

Uso de Generadores

Para utilizar un generador, se puede iterar sobre él utilizando un bucle for o la función next().

# Usando un bucle for
for numero in contador(5):
    print(numero)

# Usando la función next()
gen = contador(5)
print(next(gen))  # Salida: 0
print(next(gen))  # Salida: 1
print(next(gen))  # Salida: 2

Ventajas de los Generadores

  1. Eficiencia de Memoria: Los generadores no almacenan toda la secuencia en memoria; generan cada valor sobre la marcha.
  2. Pausabilidad: Los generadores pueden pausar su ejecución y reanudarla, lo que permite manejar flujos de datos grandes o infinitos.
  3. Simplicidad: Los generadores pueden simplificar el código que necesita producir una secuencia de valores.

Ejemplo Práctico: Generador de Números Fibonacci

El siguiente ejemplo muestra cómo crear un generador para la secuencia de Fibonacci.

def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Usando el generador de Fibonacci
for numero in fibonacci(10):
    print(numero)

Ejercicio Práctico

Ejercicio 1: Crea un generador llamado cuadrados que produzca los cuadrados de los números desde 0 hasta un número dado n.

def cuadrados(n):
    for i in range(n):
        yield i ** 2

# Prueba tu generador
for numero in cuadrados(5):
    print(numero)

Solución:

def cuadrados(n):
    for i in range(n):
        yield i ** 2

# Prueba tu generador
for numero in cuadrados(5):
    print(numero)

Salida Esperada:

0
1
4
9
16

Errores Comunes y Consejos

  • Olvidar yield: Asegúrate de usar yield en lugar de return dentro de tu generador.
  • No manejar el final del iterador: Cuando se usa next(), se debe manejar la excepción StopIteration que se lanza cuando el generador se agota.
  • Confundir generadores con listas: Recuerda que los generadores no almacenan todos los valores en memoria, por lo que no puedes acceder a sus elementos por índice.

Conclusión

Los generadores son una herramienta esencial en Python para manejar secuencias de datos de manera eficiente y elegante. Al comprender cómo funcionan y cómo utilizarlos, puedes escribir código más eficiente y fácil de mantener. En el próximo tema, exploraremos los administradores de contexto, otra característica avanzada de Python que te ayudará a manejar recursos de manera segura y eficiente.
