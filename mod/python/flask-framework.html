Introducción a Flask

Flask es un framework ligero de aplicaciones web WSGI en Python. Está diseñado con simplicidad y flexibilidad en mente, lo que lo convierte en una opción popular tanto para principiantes como para desarrolladores avanzados.

  • WSGI: Interfaz de pasarela de servidor web, una especificación para una interfaz simple y universal entre servidores web y aplicaciones web.
  • Ligero: Flask no requiere herramientas o bibliotecas particulares.
  • Flexibilidad: Flask permite a los desarrolladores elegir sus herramientas y bibliotecas.

Configuración de Flask

Antes de que puedas comenzar a usar Flask, necesitas instalarlo.

pip install Flask

Creando tu primera aplicación Flask

Aquí hay un ejemplo simple de una aplicación Flask:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return '¡Hola, Mundo!'

if __name__ == '__main__':
    app.run(debug=True)
  • Explicación:
    • from flask import Flask: Importa la clase Flask.
    • app = Flask(__name__): Crea una instancia de la clase Flask.
    • @app.route('/'): Define la ruta para la página de inicio.
    • def hello_world(): Define la función de vista para la página de inicio.
    • app.run(debug=True): Ejecuta la aplicación en modo de depuración.

Enrutamiento en Flask

El enrutamiento se utiliza para vincular una URL a una función.

  • Ruta básica:

    @app.route('/about')
def about():
    return 'Página Acerca de'

  • Ruta dinámica:

    @app.route('/user/<username>')
def show_user_profile(username):
    return f'Usuario {username}'

  • Explicación:

    • <username>: Un componente dinámico que puede capturar parte de la URL y pasarlo a la función como un argumento.

Plantillas en Flask

Flask utiliza el motor de plantillas Jinja2 para renderizar HTML.

Creando una plantilla

  1. Crea una carpeta llamada templates.
  2. Dentro de la carpeta templates, crea un archivo HTML, por ejemplo, index.html.
<!doctype html>
<html>
  <head>
    <title>{{ title }}</title>
  </head>
  <body>
    <h1>{{ heading }}</h1>
    <p>{{ content }}</p>
  </body>
</html>

Renderizando una plantilla

from flask import render_template

@app.route('/')
def home():
    return render_template('index.html', title='Página de Inicio', heading='Bienvenido', content='Esta es la página de inicio.')
  • Explicación:
    • render_template('index.html', ...): Renderiza la plantilla index.html y pasa variables a ella.

Formularios e ingreso de usuario

Manejar formularios e ingreso de usuario es una tarea común en aplicaciones web.

Creando un formulario

<form method="POST" action="/submit">
  <input type="text" name="username">
  <input type="submit" value="Enviar">
</form>

Manejo de envío de formularios

from flask import request

@app.route('/submit', methods=['POST'])
def submit():
    username = request.form['username']
    return f'¡Hola, {username}!'
  • Explicación:
    • request.form['username']: Accede a los datos del formulario enviados a través de una solicitud POST.

Integración con bases de datos

Flask puede integrarse con varias bases de datos usando extensiones como Flask-SQLAlchemy.

Configuración de Flask-SQLAlchemy

  1. Instala Flask-SQLAlchemy:

    pip install Flask-SQLAlchemy

  2. Configura e inicializa la base de datos:

    from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///site.db'
db = SQLAlchemy(app)

Creando un modelo

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'
  • Explicación:
    • db.Model: Clase base para todos los modelos.
    • db.Column: Define una columna en la base de datos.
    • db.Integer, db.String: Tipos de datos para las columnas.
    • primary_key=True: Establece la columna como clave primaria.
    • unique=True: Asegura que los valores de la columna sean únicos.
    • nullable=False: Asegura que la columna no pueda ser nula.

Temas avanzados

Blueprints

Los Blueprints ayudan a organizar tu aplicación en módulos.

from flask import Blueprint

admin = Blueprint('admin', __name__)

@admin.route('/admin')
def admin_home():
    return 'Inicio de Admin'

Manejo de errores

Se pueden crear páginas de error personalizadas para manejar errores.

@app.errorhandler(404)
def page_not_found(e):
    return render_template('404.html'), 404

Pruebas

Flask proporciona herramientas para probar tu aplicación.

import unittest

class FlaskTestCase(unittest.TestCase):
    def setUp(self):
        app.config['TESTING'] = True
        self.app = app.test_client()

    def test_home(self):
        response = self.app.get('/')
        self.assertEqual(response.status_code, 200)

if __name__ == '__main__':
    unittest.main()

Conclusión

Flask es un framework poderoso y flexible para construir aplicaciones web en Python. Es adecuado tanto para principiantes como para desarrolladores avanzados debido a su simplicidad y extensibilidad. Al dominar Flask, puedes crear aplicaciones web robustas e integrarlas con varias herramientas y bibliotecas.
