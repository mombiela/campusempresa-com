5-2: Atributos y Métodos

Introducción

En esta sección, aprenderemos sobre los atributos y métodos en la programación orientada a objetos (POO) en Python. Los atributos son variables que pertenecen a una clase o a una instancia de una clase, mientras que los métodos son funciones que pertenecen a una clase.

Atributos

Atributos de Instancia

Los atributos de instancia son variables que pertenecen a una instancia específica de una clase. Se definen dentro del método __init__ y se accede a ellos utilizando self.

class Perro:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

# Crear una instancia de la clase Perro
mi_perro = Perro("Fido", 2)

# Acceder a los atributos de instancia
print(mi_perro.nombre)  # Salida: Fido
print(mi_perro.edad)    # Salida: 2

Atributos de Clase

Los atributos de clase son variables que pertenecen a la clase en sí y son compartidos por todas las instancias de la clase.

class Perro:
    especie = "Canis lupus familiaris"  # Atributo de clase

    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

# Crear dos instancias de la clase Perro
perro1 = Perro("Fido", 2)
perro2 = Perro("Buddy", 3)

# Acceder al atributo de clase
print(perro1.especie)  # Salida: Canis lupus familiaris
print(perro2.especie)  # Salida: Canis lupus familiaris

Métodos

Métodos de Instancia

Los métodos de instancia son funciones que operan en una instancia específica de una clase. Se definen con self como su primer parámetro.

class Perro:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def ladrar(self):
        return f"{self.nombre} dice guau!"

# Crear una instancia de la clase Perro
mi_perro = Perro("Fido", 2)

# Llamar al método de instancia
print(mi_perro.ladrar())  # Salida: Fido dice guau!

Métodos de Clase

Los métodos de clase operan en la clase en sí, en lugar de en una instancia específica. Se definen utilizando el decorador @classmethod y cls como su primer parámetro.

class Perro:
    especie = "Canis lupus familiaris"

    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    @classmethod
    def obtener_especie(cls):
        return cls.especie

# Llamar al método de clase
print(Perro.obtener_especie())  # Salida: Canis lupus familiaris

Métodos Estáticos

Los métodos estáticos no operan ni en la instancia ni en la clase. Se definen utilizando el decorador @staticmethod.

class Perro:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    @staticmethod
    def es_cachorro(edad):
        return edad < 1

# Llamar al método estático
print(Perro.es_cachorro(0.5))  # Salida: True
print(Perro.es_cachorro(2))    # Salida: False

Ejercicios Prácticos

Ejercicio 1: Crear una Clase con Atributos y Métodos

  1. Define una clase Coche con los atributos de instancia marca, modelo y año.
  2. Añade un método de instancia descripcion que devuelva una cadena con la descripción del coche.
  3. Añade un método de clase es_moderno que devuelva True si el coche es del año 2000 o posterior.
class Coche:
    def __init__(self, marca, modelo, año):
        self.marca = marca
        self.modelo = modelo
        self.año = año

    def descripcion(self):
        return f"{self.marca} {self.modelo} del año {self.año}"

    @classmethod
    def es_moderno(cls, año):
        return año >= 2000

# Crear una instancia de la clase Coche
mi_coche = Coche("Toyota", "Corolla", 2021)

# Llamar al método de instancia
print(mi_coche.descripcion())  # Salida: Toyota Corolla del año 2021

# Llamar al método de clase
print(Coche.es_moderno(2021))  # Salida: True
print(Coche.es_moderno(1999))  # Salida: False

Resumen

En esta sección, hemos aprendido sobre los atributos y métodos en Python:

  • Atributos de Instancia: Variables específicas de una instancia.
  • Atributos de Clase: Variables compartidas por todas las instancias de la clase.
  • Métodos de Instancia: Funciones que operan en una instancia específica.
  • Métodos de Clase: Funciones que operan en la clase en sí.
  • Métodos Estáticos: Funciones que no operan ni en la instancia ni en la clase.

Estos conceptos son fundamentales para comprender y utilizar la programación orientada a objetos en Python. En la próxima sección, exploraremos la herencia, un concepto clave que permite crear nuevas clases basadas en clases existentes.
