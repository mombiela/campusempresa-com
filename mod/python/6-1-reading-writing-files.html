6-1: Lectura y Escritura de Archivos

Introducción

En este tema, aprenderemos cómo manejar archivos en Python. La capacidad de leer y escribir archivos es fundamental para muchas aplicaciones, desde el procesamiento de datos hasta la creación de informes y la gestión de configuraciones.

Conceptos Clave

  1. Apertura de Archivos: Cómo abrir archivos en diferentes modos (lectura, escritura, etc.).
  2. Lectura de Archivos: Métodos para leer el contenido de un archivo.
  3. Escritura de Archivos: Métodos para escribir datos en un archivo.
  4. Cierre de Archivos: La importancia de cerrar archivos después de usarlos.
  5. Manejo de Errores: Cómo manejar errores comunes al trabajar con archivos.

Apertura de Archivos

Para trabajar con archivos en Python, utilizamos la función open(). Esta función toma dos argumentos principales: el nombre del archivo y el modo en el que queremos abrirlo.

# Sintaxis básica para abrir un archivo
archivo = open('nombre_del_archivo.txt', 'modo')

Modos de Apertura

Modo Descripción
'r' Lectura (por defecto). El archivo debe existir.
'w' Escritura. Crea un archivo nuevo o sobrescribe uno existente.
'a' Añadir. Escribe al final del archivo si existe.
'b' Modo binario. Se usa junto con otros modos ('rb', 'wb', etc.).
't' Modo texto (por defecto). Se usa junto con otros modos ('rt', 'wt', etc.).
'x' Creación. Crea un archivo nuevo y falla si ya existe.

Lectura de Archivos

Existen varios métodos para leer el contenido de un archivo:

  1. read(): Lee todo el contenido del archivo.
  2. readline(): Lee una sola línea del archivo.
  3. readlines(): Lee todas las líneas del archivo y las devuelve como una lista.
# Ejemplo de lectura de un archivo completo
with open('ejemplo.txt', 'r') as archivo:
    contenido = archivo.read()
    print(contenido)

# Ejemplo de lectura línea por línea
with open('ejemplo.txt', 'r') as archivo:
    for linea in archivo:
        print(linea, end='')

# Ejemplo de uso de readline()
with open('ejemplo.txt', 'r') as archivo:
    linea = archivo.readline()
    while linea:
        print(linea, end='')
        linea = archivo.readline()

Escritura de Archivos

Para escribir en un archivo, utilizamos los métodos write() y writelines().

# Ejemplo de escritura en un archivo
with open('salida.txt', 'w') as archivo:
    archivo.write('Hola, mundo!\n')
    archivo.write('Esta es una segunda línea.\n')

# Ejemplo de uso de writelines()
lineas = ['Primera línea\n', 'Segunda línea\n', 'Tercera línea\n']
with open('salida.txt', 'w') as archivo:
    archivo.writelines(lineas)

Cierre de Archivos

Es importante cerrar los archivos después de usarlos para liberar recursos. Esto se hace automáticamente cuando se usa la declaración with, pero también se puede hacer manualmente con el método close().

archivo = open('ejemplo.txt', 'r')
contenido = archivo.read()
archivo.close()

Manejo de Errores

Es común que ocurran errores al trabajar con archivos, como intentar abrir un archivo que no existe. Podemos manejar estos errores utilizando bloques try-except.

try:
    with open('archivo_inexistente.txt', 'r') as archivo:
        contenido = archivo.read()
except FileNotFoundError:
    print("El archivo no existe.")
except IOError:
    print("Ocurrió un error al leer el archivo.")

Ejercicio Práctico

Ejercicio 1:

  1. Crea un archivo llamado mi_archivo.txt y escribe las siguientes líneas en él:

    Línea 1: Hola, mundo!
Línea 2: Bienvenidos al curso de Python.
Línea 3: Esta es la tercera línea.

  2. Lee el contenido del archivo y muéstralo en la consola.

Solución:

# Escritura en el archivo
with open('mi_archivo.txt', 'w') as archivo:
    archivo.write('Línea 1: Hola, mundo!\n')
    archivo.write('Línea 2: Bienvenidos al curso de Python.\n')
    archivo.write('Línea 3: Esta es la tercera línea.\n')

# Lectura del archivo
with open('mi_archivo.txt', 'r') as archivo:
    contenido = archivo.read()
    print(contenido)

Resumen

En esta sección, hemos aprendido cómo abrir, leer, escribir y cerrar archivos en Python. También hemos visto cómo manejar errores comunes al trabajar con archivos. Estos conceptos son fundamentales para el manejo de datos y la automatización de tareas en Python.

En la próxima sección, exploraremos cómo trabajar con archivos CSV, que son muy comunes en el análisis de datos.
