Tema 8-2: Pandas para Análisis de Datos

Introducción a Pandas

Pandas es una biblioteca de Python que proporciona estructuras de datos y herramientas de análisis de datos de alto rendimiento y fáciles de usar. Es especialmente útil para trabajar con datos tabulares, como los que se encuentran en hojas de cálculo o bases de datos.

Instalación de Pandas

Para instalar Pandas, puedes usar pip:

pip install pandas

Conceptos Clave

  1. Series: Una Serie es una matriz unidimensional que puede contener cualquier tipo de datos.
  2. DataFrame: Un DataFrame es una estructura de datos bidimensional con etiquetas en los ejes (filas y columnas).

Creación de Series y DataFrames

Series

import pandas as pd

# Crear una Serie a partir de una lista
data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

DataFrames

import pandas as pd

# Crear un DataFrame a partir de un diccionario
data = {
    'Nombre': ['Ana', 'Luis', 'Carlos'],
    'Edad': [23, 45, 34],
    'Ciudad': ['Madrid', 'Barcelona', 'Valencia']
}
df = pd.DataFrame(data)
print(df)

Operaciones Básicas con DataFrames

Visualización de Datos

# Mostrar las primeras filas del DataFrame
print(df.head())

# Mostrar las últimas filas del DataFrame
print(df.tail())

Selección de Datos

# Seleccionar una columna
print(df['Nombre'])

# Seleccionar múltiples columnas
print(df[['Nombre', 'Ciudad']])

# Seleccionar filas por índice
print(df.iloc[0])  # Primera fila
print(df.iloc[0:2])  # Primeras dos filas

# Seleccionar filas por condición
print(df[df['Edad'] > 30])

Modificación de Datos

# Añadir una nueva columna
df['Salario'] = [3000, 4000, 3500]
print(df)

# Modificar valores
df.at[0, 'Edad'] = 24
print(df)

# Eliminar una columna
df.drop('Salario', axis=1, inplace=True)
print(df)

Operaciones Avanzadas

Agrupación y Agregación

# Agrupar por una columna y calcular la media
grouped = df.groupby('Ciudad').mean()
print(grouped)

Unión de DataFrames

# Crear otro DataFrame
data2 = {
    'Nombre': ['Ana', 'Luis', 'Carlos'],
    'Salario': [3000, 4000, 3500]
}
df2 = pd.DataFrame(data2)

# Unir DataFrames
merged_df = pd.merge(df, df2, on='Nombre')
print(merged_df)

Ejercicio Práctico

Ejercicio 1: Análisis de Datos de Ventas

  1. Descripción: Tienes un archivo CSV con datos de ventas. Debes cargar los datos, realizar algunas operaciones básicas y responder algunas preguntas.
  2. Archivo CSV: ventas.csv
Fecha,Producto,Precio,Cantidad
2023-01-01,Producto A,100,2
2023-01-02,Producto B,150,1
2023-01-03,Producto A,100,1
2023-01-04,Producto C,200,3
2023-01-05,Producto B,150,2

Pasos:

  1. Cargar los datos:
import pandas as pd

# Cargar el archivo CSV
df = pd.read_csv('ventas.csv')
print(df)
  1. Calcular el total de ventas por producto:
# Añadir una columna de Total
df['Total'] = df['Precio'] * df['Cantidad']

# Agrupar por Producto y sumar el Total
total_ventas = df.groupby('Producto')['Total'].sum()
print(total_ventas)
  1. Encontrar el producto más vendido:
# Agrupar por Producto y sumar la Cantidad
cantidad_ventas = df.groupby('Producto')['Cantidad'].sum()

# Encontrar el producto con la mayor cantidad vendida
producto_mas_vendido = cantidad_ventas.idxmax()
print(f'El producto más vendido es: {producto_mas_vendido}')

Soluciones

  1. Cargar los datos:
