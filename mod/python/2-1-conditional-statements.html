Tema 2-1: Sentencias Condicionales

Introducción

Las sentencias condicionales permiten que tu programa tome decisiones y ejecute diferentes bloques de código en función de ciertas condiciones. En Python, las sentencias condicionales más comunes son if, elif y else.

Conceptos Clave

  1. Sentencia if: Evalúa una condición y ejecuta un bloque de código si la condición es verdadera.
  2. Sentencia elif: (abreviatura de "else if") Evalúa una nueva condición si la condición anterior es falsa.
  3. Sentencia else: Ejecuta un bloque de código si todas las condiciones anteriores son falsas.
  4. Indentación: La indentación es crucial en Python para definir bloques de código.

Sintaxis Básica

if condición:
    # Bloque de código si la condición es verdadera
elif otra_condición:
    # Bloque de código si la otra_condición es verdadera
else:
    # Bloque de código si ninguna de las condiciones anteriores es verdadera

Ejemplo Práctico

Vamos a ver un ejemplo simple que determina si un número es positivo, negativo o cero.

numero = int(input("Introduce un número: "))

if numero > 0:
    print("El número es positivo.")
elif numero < 0:
    print("El número es negativo.")
else:
    print("El número es cero.")

Explicación del Código:

  1. Entrada del Usuario: numero = int(input("Introduce un número: "))
    • Se solicita al usuario que introduzca un número y se convierte a un entero.
  2. Condición if: if numero > 0:
    • Si el número es mayor que 0, se imprime "El número es positivo."
  3. Condición elif: elif numero < 0:
    • Si el número es menor que 0, se imprime "El número es negativo."
  4. Condición else: else:
    • Si ninguna de las condiciones anteriores es verdadera, se imprime "El número es cero."

Ejercicios Prácticos

Ejercicio 1: Determinar si un Número es Par o Impar

Escribe un programa que determine si un número introducido por el usuario es par o impar.

numero = int(input("Introduce un número: "))

if numero % 2 == 0:
    print("El número es par.")
else:
    print("El número es impar.")

Ejercicio 2: Calificación de un Estudiante

Escribe un programa que asigne una calificación basada en la puntuación de un estudiante.

  • 90-100: A
  • 80-89: B
  • 70-79: C
  • 60-69: D
  • 0-59: F
puntuacion = int(input("Introduce la puntuación del estudiante: "))

if puntuacion >= 90:
    print("Calificación: A")
elif puntuacion >= 80:
    print("Calificación: B")
elif puntuacion >= 70:
    print("Calificación: C")
elif puntuacion >= 60:
    print("Calificación: D")
else:
    print("Calificación: F")

Soluciones

Solución al Ejercicio 1

numero = int(input("Introduce un número: "))

if numero % 2 == 0:
    print("El número es par.")
else:
    print("El número es impar.")

Solución al Ejercicio 2

puntuacion = int(input("Introduce la puntuación del estudiante: "))

if puntuacion >= 90:
    print("Calificación: A")
elif puntuacion >= 80:
    print("Calificación: B")
elif puntuacion >= 70:
    print("Calificación: C")
elif puntuacion >= 60:
    print("Calificación: D")
else:
    print("Calificación: F")

Errores Comunes y Consejos

  1. Indentación Incorrecta: Asegúrate de que los bloques de código dentro de if, elif y else estén correctamente indentados.
  2. Condiciones No Excluyentes: Asegúrate de que las condiciones en elif sean mutuamente excluyentes para evitar resultados inesperados.
  3. Uso de elif en Lugar de if: Usa elif en lugar de if para condiciones adicionales que dependen de la condición anterior.

Conclusión

Las sentencias condicionales son fundamentales para controlar el flujo de un programa en Python. Con if, elif y else, puedes hacer que tu programa tome decisiones basadas en diferentes condiciones. Practica con los ejercicios proporcionados para reforzar tu comprensión y estarás listo para avanzar al siguiente tema sobre bucles.

En el próximo tema, exploraremos los bucles for y while, que te permitirán ejecutar bloques de código repetidamente bajo ciertas condiciones. ¡Sigue adelante!
