Tema 2-2: Bucles: for y while

Introducción

En este tema, aprenderás sobre los bucles for y while en Python. Los bucles son estructuras de control que permiten ejecutar un bloque de código repetidamente mientras se cumple una condición. Son fundamentales para automatizar tareas repetitivas y manejar colecciones de datos.

Bucles `for`

El bucle for se utiliza para iterar sobre una secuencia (como una lista, tupla, diccionario, conjunto o cadena). La sintaxis básica es:

for elemento in secuencia:
    # Bloque de código a ejecutar

Ejemplo 1: Iterar sobre una lista

frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
    print(fruta)

Explicación:

  • frutas es una lista que contiene tres elementos.
  • El bucle for itera sobre cada elemento de la lista y lo asigna a la variable fruta.
  • El bloque de código dentro del bucle imprime cada fruta.

Ejemplo 2: Usar range en un bucle for

for i in range(5):
    print(i)

Explicación:

  • range(5) genera una secuencia de números del 0 al 4.
  • El bucle for itera sobre cada número en la secuencia y lo asigna a la variable i.
  • El bloque de código dentro del bucle imprime cada número.

Bucles `while`

El bucle while se utiliza para repetir un bloque de código mientras una condición sea verdadera. La sintaxis básica es:

while condición:
    # Bloque de código a ejecutar

Ejemplo 1: Contar hasta 5

contador = 1
while contador <= 5:
    print(contador)
    contador += 1

Explicación:

  • contador se inicializa en 1.
  • El bucle while continúa ejecutándose mientras contador sea menor o igual a 5.
  • En cada iteración, se imprime el valor de contador y luego se incrementa en 1.

Ejemplo 2: Bucle infinito (con condición de salida)

while True:
    respuesta = input("¿Quieres salir? (sí/no): ")
    if respuesta == "sí":
        break

Explicación:

  • El bucle while se ejecuta indefinidamente debido a la condición True.
  • Dentro del bucle, se solicita al usuario una respuesta.
  • Si la respuesta es "sí", se utiliza break para salir del bucle.

Comparación de `for` y `while`

Característica for while
Uso principal Iterar sobre una secuencia Repetir mientras una condición sea verdadera
Sintaxis for elemento in secuencia: while condición:
Control de iteración Automático (a través de la secuencia) Manual (a través de la condición)

Ejercicios Prácticos

Ejercicio 1: Sumar números en una lista

Enunciado: Escribe un programa que sume todos los números en la lista [1, 2, 3, 4, 5] usando un bucle for.

Solución:

numeros = [1, 2, 3, 4, 5]
suma = 0
for numero in numeros:
    suma += numero
print("La suma es:", suma)

Ejercicio 2: Encontrar el primer número par en una lista

Enunciado: Escribe un programa que encuentre el primer número par en la lista [1, 3, 5, 6, 7, 8] usando un bucle while.

Solución:

numeros = [1, 3, 5, 6, 7, 8]
i = 0
while i < len(numeros):
    if numeros[i] % 2 == 0:
        print("El primer número par es:", numeros[i])
        break
    i += 1

Retroalimentación y Consejos

  • Error común: Olvidar incrementar el contador en un bucle while, lo que puede causar un bucle infinito.
  • Consejo: Siempre asegúrate de que la condición del bucle while eventualmente se vuelva falsa.
  • Error común: Modificar la secuencia sobre la que se está iterando en un bucle for, lo que puede causar comportamientos inesperados.
  • Consejo: Si necesitas modificar la secuencia, considera iterar sobre una copia de la misma.

Conclusión

En este tema, has aprendido cómo utilizar los bucles for y while en Python para iterar sobre secuencias y repetir bloques de código mientras se cumple una condición. Estos conceptos son fundamentales para manejar tareas repetitivas y trabajar con colecciones de datos de manera eficiente. En el próximo tema, exploraremos el flujo de control con las sentencias break, continue y pass.
