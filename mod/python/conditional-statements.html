Introducción a las Declaraciones Condicionales

Las declaraciones condicionales te permiten ejecutar ciertas piezas de código basadas en si una condición es verdadera o falsa. Este es un concepto fundamental en la programación que permite la toma de decisiones en tu código.

La Declaración `if`

La declaración if es la forma más básica de declaración condicional en Python.

Sintaxis

if condition:
    # código a ejecutar si la condición es verdadera

Ejemplo

age = 18
if age >= 18:
    print("Eres un adulto.")

En este ejemplo, el mensaje "Eres un adulto." se imprimirá porque la condición age >= 18 es verdadera.

La Declaración `else`

La declaración else se puede usar para ejecutar un bloque de código si la condición en la declaración if es falsa.

Sintaxis

if condition:
    # código a ejecutar si la condición es verdadera
else:
    # código a ejecutar si la condición es falsa

Ejemplo

age = 16
if age >= 18:
    print("Eres un adulto.")
else:
    print("No eres un adulto.")

En este ejemplo, el mensaje "No eres un adulto." se imprimirá porque la condición age >= 18 es falsa.

La Declaración `elif`

La declaración elif (abreviatura de else if) te permite verificar múltiples condiciones.

Sintaxis

if condition1:
    # código a ejecutar si la condición1 es verdadera
elif condition2:
    # código a ejecutar si la condición2 es verdadera
else:
    # código a ejecutar si tanto la condición1 como la condición2 son falsas

Ejemplo

age = 20
if age < 13:
    print("Eres un niño.")
elif age < 18:
    print("Eres un adolescente.")
else:
    print("Eres un adulto.")

En este ejemplo, el mensaje "Eres un adulto." se imprimirá porque la condición age < 18 es falsa, pero la condición age >= 18 es verdadera.

Declaraciones Condicionales Anidadas

Puedes anidar declaraciones condicionales dentro de otras para verificar múltiples condiciones.

Ejemplo

age = 20
if age >= 18:
    if age < 21:
        print("Eres un adulto pero no lo suficientemente mayor para beber alcohol en los EE.UU.")
    else:
        print("Eres un adulto y lo suficientemente mayor para beber alcohol en los EE.UU.")
else:
    print("No eres un adulto.")

En este ejemplo, el mensaje "Eres un adulto pero no lo suficientemente mayor para beber alcohol en los EE.UU." se imprimirá porque la condición age >= 18 es verdadera y age < 21 también es verdadera.

Operadores Lógicos

Los operadores lógicos se pueden usar para combinar múltiples condiciones en una sola declaración if.

Operadores

  • and: Verdadero si ambas condiciones son verdaderas.
  • or: Verdadero si al menos una condición es verdadera.
  • not: Verdadero si la condición es falsa.

Ejemplo

age = 20
has_id = True

if age >= 18 and has_id:
    print("Se te permite entrar al club.")
else:
    print("No se te permite entrar al club.")

En este ejemplo, el mensaje "Se te permite entrar al club." se imprimirá porque ambas condiciones age >= 18 y has_id son verdaderas.

Expresiones Condicionales (Operador Ternario)

Python admite una sintaxis más corta para las declaraciones condicionales conocida como el operador ternario.

Sintaxis

value_if_true if condition else value_if_false

Ejemplo

age = 20
status = "adulto" si age >= 18 else "menor"
print(status)

En este ejemplo, la variable status se establecerá en "adulto" porque la condición age >= 18 es verdadera.

Resumen

  • Las declaraciones condicionales (if, else, elif) te permiten ejecutar código basado en condiciones.
  • Las declaraciones condicionales anidadas permiten una toma de decisiones más compleja.
  • Los operadores lógicos (and, or, not) pueden combinar múltiples condiciones.
  • El operador ternario proporciona una forma concisa de escribir declaraciones condicionales simples.

Comprender y usar las declaraciones condicionales de manera efectiva te permitirá escribir programas en Python más dinámicos y receptivos. Practica usando estos conceptos en varios escenarios para volverte competente.
