9-5: Integración de Bases de Datos

Introducción

En esta sección, aprenderemos cómo integrar bases de datos en nuestras aplicaciones web utilizando Flask. Trabajaremos con SQLite, una base de datos ligera y fácil de usar que es ideal para aplicaciones pequeñas y medianas. También veremos cómo utilizar SQLAlchemy, un ORM (Object-Relational Mapping) que facilita la interacción con bases de datos relacionales.

Objetivos

  • Entender cómo configurar una base de datos en Flask.
  • Aprender a utilizar SQLAlchemy para interactuar con la base de datos.
  • Realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en la base de datos.
  • Integrar la base de datos con nuestra aplicación Flask.

Configuración del Entorno

Antes de comenzar, asegúrate de tener Flask y SQLAlchemy instalados. Puedes instalarlos utilizando pip:

pip install Flask SQLAlchemy

Configuración de la Base de Datos

  1. Configurar la Aplicación Flask:

    Primero, necesitamos configurar nuestra aplicación Flask para utilizar SQLAlchemy. Crea un archivo llamado app.py y añade el siguiente código:

    from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///site.db'
db = SQLAlchemy(app)

if __name__ == '__main__':
    app.run(debug=True)

    Aquí, hemos configurado la URI de la base de datos para utilizar SQLite y hemos inicializado SQLAlchemy con nuestra aplicación Flask.

  2. Definir Modelos:

    Los modelos son clases que representan tablas en la base de datos. Vamos a definir un modelo simple para un usuario.

    class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f"User('{self.username}', '{self.email}')"

    En este modelo, hemos definido una tabla User con tres columnas: id, username y email.

  3. Crear la Base de Datos:

    Para crear la base de datos y las tablas, abre una terminal y ejecuta el siguiente comando en el intérprete de Python:

    from app import db
db.create_all()

    Esto creará un archivo site.db con la tabla User.

Operaciones CRUD

Ahora que tenemos nuestra base de datos configurada, vamos a realizar operaciones CRUD.

  1. Crear un Usuario:

    from app import db, User

new_user = User(username='john_doe', email='john@example.com')
db.session.add(new_user)
db.session.commit()

  2. Leer Usuarios:

    users = User.query.all()
print(users)

user = User.query.filter_by(username='john_doe').first()
print(user)

  3. Actualizar un Usuario:

    user = User.query.filter_by(username='john_doe').first()
user.email = 'john_doe@example.com'
db.session.commit()

  4. Eliminar un Usuario:

    user = User.query.filter_by(username='john_doe').first()
db.session.delete(user)
db.session.commit()

Integración con la Aplicación Flask

Vamos a integrar estas operaciones CRUD en nuestra aplicación Flask utilizando rutas.

  1. Crear Rutas:

    from flask import render_template, url_for, flash, redirect
from app import app, db, User

@app.route('/')
def home():
    users = User.query.all()
    return render_template('home.html', users=users)

@app.route('/add_user', methods=['POST'])
def add_user():
    username = request.form['username']
    email = request.form['email']
    new_user = User(username=username, email=email)
    db.session.add(new_user)
    db.session.commit()
    flash('User added successfully!', 'success')
    return redirect(url_for('home'))

  2. Crear Plantillas:

    Crea un archivo home.html en una carpeta llamada templates:

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Home</title>
</head>
<body>
    <h1>Users</h1>
    <ul>
        {% for user in users %}
            <li>{{ user.username }} - {{ user.email }}</li>
        {% endfor %}
    </ul>
    <form action="{{ url_for('add_user') }}" method="POST">
        <input type="text" name="username" placeholder="Username">
        <input type="email" name="email" placeholder="Email">
        <button type="submit">Add User</button>
    </form>
</body>
</html>

Ejercicio Práctico

Ejercicio:

  1. Añade una nueva ruta para actualizar un usuario.
  2. Añade una nueva ruta para eliminar un usuario.
  3. Modifica la plantilla home.html para incluir botones de actualización y eliminación junto a cada usuario.

Solución:

  1. Ruta para Actualizar un Usuario:

    @app.route('/update_user/<int:id>', methods=['POST'])
def update_user(id):
    user = User.query.get_or_404(id)
    user.username = request.form['username']
    user.email = request.form['email']
    db.session.commit()
    flash('User updated successfully!', 'success')
    return redirect(url_for('home'))

  2. Ruta para Eliminar un Usuario:

    @app.route('/delete_user/<int:id>', methods=['POST'])
def delete_user(id):
    user = User.query.get_or_404(id)
    db.session.delete(user)
    db.session.commit()
    flash('User deleted successfully!', 'success')
    return redirect(url_for('home'))

  3. Modificar home.html:

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Home</title>
</head>
<body>
    <h1>Users</h1>
    <ul>
        {% for user in users %}
            <li>
                {{ user.username }} - {{ user.email }}
                <form action="{{ url_for('update_user', id=user.id) }}" method="POST">
                    <input type="text" name="username" value="{{ user.username }}">
                    <input type="email" name="email" value="{{ user.email }}">
                    <button type="submit">Update</button>
                </form>
                <form action="{{ url_for('delete_user', id=user.id) }}" method="POST">
                    <button type="submit">Delete</button>
                </form>
            </li>
        {% endfor %}
    </ul>
    <form action="{{ url_for('add_user') }}" method="POST">
        <input type="text" name="username" placeholder="Username">
        <input type="email" name="email" placeholder="Email">
        <button type="submit">Add User</button>
    </form>
</body>
</html>

Conclusión

En esta sección, hemos aprendido cómo integrar una base de datos en una aplicación Flask utilizando SQLAlchemy. Hemos cubierto la configuración de la base de datos, la definición de modelos, y cómo realizar operaciones CRUD. Además, hemos integrado estas operaciones en nuestra aplicación Flask mediante rutas y plantillas.
