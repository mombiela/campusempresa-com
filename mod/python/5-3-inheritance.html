5-3: Herencia

Introducción a la Herencia

La herencia es un concepto fundamental en la programación orientada a objetos (POO) que permite crear nuevas clases basadas en clases existentes. La clase nueva se llama clase derivada o subclase, y la clase existente se llama clase base o superclase. La herencia permite reutilizar código, mejorar la organización y facilitar el mantenimiento del software.

Conceptos Clave

  • Clase Base (Superclase): La clase de la cual se heredan atributos y métodos.
  • Clase Derivada (Subclase): La clase que hereda atributos y métodos de la clase base.
  • Herencia Simple: Una subclase hereda de una única superclase.
  • Herencia Múltiple: Una subclase hereda de más de una superclase (Python soporta herencia múltiple).

Ejemplo Práctico

Vamos a crear un ejemplo simple para ilustrar la herencia. Imaginemos que tenemos una clase Animal y queremos crear una subclase Perro que herede de Animal.

# Clase base
class Animal:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    
    def hacer_sonido(self):
        print("El animal hace un sonido")

# Clase derivada
class Perro(Animal):
    def __init__(self, nombre, edad, raza):
        super().__init__(nombre, edad)  # Llamada al constructor de la clase base
        self.raza = raza
    
    def hacer_sonido(self):
        print("El perro ladra")

# Crear una instancia de Perro
mi_perro = Perro("Fido", 3, "Labrador")
print(f"Nombre: {mi_perro.nombre}, Edad: {mi_perro.edad}, Raza: {mi_perro.raza}")
mi_perro.hacer_sonido()

Explicación del Código

  1. Clase Base Animal:

    • El constructor __init__ inicializa los atributos nombre y edad.
    • El método hacer_sonido imprime un mensaje genérico.

  2. Clase Derivada Perro:

    • El constructor __init__ de Perro llama al constructor de Animal usando super().__init__(nombre, edad).
    • Se añade un nuevo atributo raza.
    • El método hacer_sonido se sobrescribe para proporcionar un comportamiento específico para Perro.

Ejercicio Práctico

Ejercicio 1: Crear una Subclase

Crea una clase base Vehiculo con atributos marca y modelo, y un método arrancar. Luego, crea una subclase Coche que herede de Vehiculo y añade un atributo num_puertas. Sobrescribe el método arrancar para que imprima un mensaje específico para Coche.

# Solución
class Vehiculo:
    def __init__(self, marca, modelo):
        self.marca = marca
        self.modelo = modelo
    
    def arrancar(self):
        print("El vehículo está arrancando")

class Coche(Vehiculo):
    def __init__(self, marca, modelo, num_puertas):
        super().__init__(marca, modelo)
        self.num_puertas = num_puertas
    
    def arrancar(self):
        print(f"El coche {self.marca} {self.modelo} está arrancando")

# Crear una instancia de Coche
mi_coche = Coche("Toyota", "Corolla", 4)
print(f"Marca: {mi_coche.marca}, Modelo: {mi_coche.modelo}, Número de puertas: {mi_coche.num_puertas}")
mi_coche.arrancar()

Retroalimentación sobre Errores Comunes

  • Olvidar llamar al constructor de la clase base: Asegúrate de usar super().__init__(...) para inicializar los atributos heredados.
  • No sobrescribir métodos correctamente: Si sobrescribes un método, asegúrate de que la firma del método (nombre y parámetros) coincida con la de la clase base.
  • Confusión con herencia múltiple: En Python, la herencia múltiple puede ser compleja. Usa con precaución y asegúrate de entender el orden de resolución de métodos (MRO).

Conclusión

La herencia es una herramienta poderosa en la programación orientada a objetos que permite reutilizar y extender el código de manera eficiente. En este módulo, hemos aprendido cómo crear clases base y derivadas, cómo sobrescribir métodos y cómo utilizar super() para llamar a métodos de la clase base. Con estos conocimientos, estarás mejor preparado para diseñar sistemas más complejos y organizados.
