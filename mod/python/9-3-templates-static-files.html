Tema 9-3: Plantillas y Archivos Estáticos

Introducción

En este tema, aprenderás cómo utilizar plantillas y archivos estáticos en Flask para crear aplicaciones web dinámicas y atractivas. Las plantillas permiten separar la lógica de la presentación, mientras que los archivos estáticos (como CSS, JavaScript e imágenes) mejoran la apariencia y la funcionalidad de tu aplicación.

1. Plantillas en Flask

Introducción a Jinja2

Flask utiliza Jinja2 como motor de plantillas. Jinja2 permite incluir variables y lógica de control de flujo dentro de archivos HTML, facilitando la creación de contenido dinámico.

Sintaxis Básica de Jinja2

  • Variables: Se utilizan dobles llaves {{ }} para insertar variables en la plantilla.
  • Control de Flujo: Se utilizan etiquetas {% %} para estructuras de control como bucles y condicionales.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>{{ message }}</p>
</body>
</html>

Variables y Control de Flujo en Plantillas

  • Variables: Puedes pasar variables desde tu aplicación Flask a la plantilla.
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', title='Home Page', heading='Welcome!', message='Hello, Flask!')

if __name__ == '__main__':
    app.run(debug=True)
  • Condicionales: Utiliza {% if %} y {% endif %} para condicionales.
{% if user %}
    <p>Welcome, {{ user }}!</p>
{% else %}
    <p>Welcome, Guest!</p>
{% endif %}
  • Bucles: Utiliza {% for %} y {% endfor %} para iterar sobre listas.
<ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

2. Archivos Estáticos

Estructura de Directorios para Archivos Estáticos

Flask espera que los archivos estáticos se encuentren en un directorio llamado static dentro de tu proyecto.

/project
    /static
        /css
            styles.css
        /js
            scripts.js
        /images
            logo.png
    /templates
        index.html
    app.py

Servir Archivos Estáticos en Flask

Para incluir archivos estáticos en tus plantillas, utiliza la función url_for.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>{{ message }}</p>
    <script src="{{ url_for('static', filename='js/scripts.js') }}"></script>
</body>
</html>

3. Ejemplo Práctico

Crear una Plantilla HTML

Crea un archivo index.html en el directorio templates.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>{{ message }}</p>
    <script src="{{ url_for('static', filename='js/scripts.js') }}"></script>

</html>

Integrar CSS y JavaScript

Crea un archivo styles.css en el directorio static/css.

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}
h1 {
    color: #007BFF;
}

Crea un archivo scripts.js en el directorio static/js.

document.addEventListener('DOMContentLoaded', function() {
    console.log('JavaScript is working!');
});

4. Ejercicio Práctico

Crear una Página Web con Plantillas y Archivos Estáticos

Objetivo: Crear una página web que muestre una lista de elementos y permita agregar nuevos elementos mediante un formulario.

  1. Crear la plantilla HTML:

    • Crea un archivo index.html en el directorio templates.
    • Incluye un formulario para agregar nuevos elementos.
    • Muestra la lista de elementos utilizando un bucle.

  2. Crear el archivo CSS:

    • Crea un archivo styles.css en el directorio static/css.
    • Aplica estilos básicos al formulario y la lista.

  3. Crear el archivo JavaScript:

    • Crea un archivo scripts.js en el directorio static/js.
    • Añade funcionalidad para manejar el envío del formulario.

  4. Actualizar la aplicación Flask:

    • Modifica app.py para manejar la lógica de agregar elementos y renderizar la plantilla.

Solución:

index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <h1>{{ heading }}</h1>
    <form action="/" method="post">
        <input type="text" name="item" placeholder="Add new item">
        <button type="submit">Add</button>
    </form>
    <ul>
        {% for item in items %}
            <li>{{ item }}</li>
        {% endfor %}
    </ul>
    <script src="{{ url_for('static', filename='js/scripts.js') }}"></script>
</body>
</html>

styles.css:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}
h1 {
    color: #007BFF;
}
form {
    margin-bottom: 20px;
}
ul {
    list-style-type: none;
    padding: 0;
}
li {
    background-color: #fff;
    margin: 5px 0;
    padding: 10px;
    border: 1px solid #ddd;
}

scripts.js:

document.addEventListener('DOMContentLoaded', function() {
    console.log('JavaScript is working!');
});

app.py:

from flask import Flask, render_template, request, redirect, url_for

app = Flask(__name__)

items = []

@app.route('/', methods=['GET', 'POST'])
def home():
    if request.method == 'POST':
        new_item = request.form['item']
        items.append(new_item)
        return redirect(url_for('home'))
    return render_template('index.html', title='To-Do List', heading='My To-Do List', items=items)

if __name__ == '__main__':
    app.run(debug=True)
