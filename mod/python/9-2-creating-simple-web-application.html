Tema 9-2: Creación de una Aplicación Web Simple

Introducción

En este tema, aprenderemos a crear una aplicación web simple utilizando Flask, un microframework de Python. Flask es conocido por su simplicidad y flexibilidad, lo que lo hace ideal para proyectos pequeños y medianos. Vamos a construir una aplicación web básica que muestre un mensaje en la página principal.

Objetivos

  • Entender la estructura básica de una aplicación Flask.
  • Crear y ejecutar una aplicación Flask simple.
  • Manejar rutas y vistas en Flask.

Requisitos Previos

  • Conocimientos básicos de Python.
  • Haber completado el tema anterior sobre la introducción a Flask.

1. Instalación de Flask

Primero, necesitamos instalar Flask. Puedes hacerlo utilizando pip, el gestor de paquetes de Python.

pip install Flask

2. Estructura Básica de una Aplicación Flask

Una aplicación Flask básica consta de un archivo principal (por lo general, app.py) que contiene el código de la aplicación. Aquí está la estructura básica:

my_flask_app/
│
├── app.py
└── templates/

3. Creación de la Aplicación

Vamos a crear un archivo llamado app.py y escribir el siguiente código:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Mundo!"

if __name__ == '__main__':
    app.run(debug=True)

Explicación del Código

  • from flask import Flask: Importamos la clase Flask del módulo flask.
  • app = Flask(__name__): Creamos una instancia de la clase Flask.
  • @app.route('/'): Utilizamos un decorador para definir la ruta de la URL. En este caso, la ruta es la página principal (/).
  • def home(): Definimos una función llamada home que se ejecutará cuando alguien visite la página principal.
  • return "¡Hola, Mundo!": La función home devuelve un mensaje que se mostrará en el navegador.
  • if __name__ == '__main__':: Verificamos si el script se está ejecutando directamente.
  • app.run(debug=True): Ejecutamos la aplicación en modo de depuración.

4. Ejecutar la Aplicación

Para ejecutar la aplicación, abre una terminal, navega hasta el directorio donde se encuentra app.py y ejecuta:

python app.py

Deberías ver una salida similar a esta:

 * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
 * Restarting with stat
 * Debugger is active!
 * Debugger PIN: 123-456-789

Abre tu navegador y visita http://127.0.0.1:5000/. Deberías ver el mensaje "¡Hola, Mundo!".

5. Manejo de Rutas y Vistas

Flask permite manejar múltiples rutas y vistas. Vamos a agregar una nueva ruta para una página "Acerca de".

@app.route('/about')
def about():
    return "Esta es la página Acerca de."

Ahora, el archivo app.py completo se verá así:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Mundo!"

@app.route('/about')
def about():
    return "Esta es la página Acerca de."

if __name__ == '__main__':
    app.run(debug=True)

6. Ejercicio Práctico

Ejercicio:

  1. Crea una nueva ruta /contact que devuelva el mensaje "Esta es la página de Contacto".
  2. Ejecuta la aplicación y verifica que las tres rutas (/, /about, /contact) funcionen correctamente.

Solución:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Mundo!"

@app.route('/about')
def about():
    return "Esta es la página Acerca de."

@app.route('/contact')
def contact():
    return "Esta es la página de Contacto."

if __name__ == '__main__':
    app.run(debug=True)

7. Conclusión

En este tema, hemos aprendido a crear una aplicación web simple utilizando Flask. Hemos cubierto la instalación de Flask, la estructura básica de una aplicación, la creación y ejecución de la aplicación, y el manejo de rutas y vistas. En el próximo tema, exploraremos cómo trabajar con plantillas y archivos estáticos en Flask.
