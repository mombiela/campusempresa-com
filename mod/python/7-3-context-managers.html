7-3: Administradores de Contexto

Introducción

Los administradores de contexto en Python son una característica poderosa que permite gestionar recursos de manera eficiente y segura. Son especialmente útiles para manejar recursos que necesitan ser adquiridos y liberados, como archivos, conexiones de red, y bloqueos.

Conceptos Clave

  • Administrador de Contexto: Un objeto que define los métodos __enter__ y __exit__ para gestionar la configuración y limpieza de recursos.
  • Declaración with: Una declaración que simplifica el uso de administradores de contexto, asegurando que los recursos se liberen correctamente.

Sintaxis Básica

La declaración with se utiliza para envolver la ejecución de un bloque de código con métodos definidos por un administrador de contexto.

with open('archivo.txt', 'r') as archivo:
    contenido = archivo.read()
    print(contenido)

En este ejemplo, open es un administrador de contexto que abre un archivo y garantiza que se cierre correctamente después de que el bloque de código se ejecute.

Implementación de un Administrador de Contexto

Puedes crear tus propios administradores de contexto definiendo los métodos __enter__ y __exit__.

Ejemplo Práctico

class AdministradorDeArchivo:
    def __init__(self, nombre_archivo, modo):
        self.nombre_archivo = nombre_archivo
        self.modo = modo
        self.archivo = None

    def __enter__(self):
        self.archivo = open(self.nombre_archivo, self.modo)
        return self.archivo

    def __exit__(self, tipo, valor, traza):
        if self.archivo:
            self.archivo.close()

# Uso del administrador de contexto personalizado
with AdministradorDeArchivo('archivo.txt', 'w') as archivo:
    archivo.write('Hola, Mundo!')

Explicación del Código

  1. Clase AdministradorDeArchivo:

    • __init__: Inicializa el nombre del archivo y el modo de apertura.
    • __enter__: Abre el archivo y lo devuelve.
    • __exit__: Cierra el archivo, asegurando que se liberen los recursos.

  2. Uso del Administrador de Contexto:

    • La declaración with utiliza el administrador de contexto AdministradorDeArchivo para abrir y escribir en el archivo de manera segura.

Ejercicio Práctico

Ejercicio 1

Crea un administrador de contexto que mida el tiempo de ejecución de un bloque de código.

import time

class MedidorDeTiempo:
    def __enter__(self):
        self.inicio = time.time()
        return self

    def __exit__(self, tipo, valor, traza):
        self.fin = time.time()
        self.tiempo_total = self.fin - self.inicio
        print(f'Tiempo de ejecución: {self.tiempo_total:.4f} segundos')

# Uso del administrador de contexto MedidorDeTiempo
with MedidorDeTiempo() as medidor:
    suma = sum(range(1000000))
    print(f'Suma: {suma}')

Solución

  1. Clase MedidorDeTiempo:

    • __enter__: Registra el tiempo de inicio.
    • __exit__: Calcula y muestra el tiempo total de ejecución.

  2. Uso del Administrador de Contexto:

    • La declaración with utiliza MedidorDeTiempo para medir el tiempo de ejecución del bloque de código que suma los números del 0 al 999999.

Resumen

En esta sección, hemos aprendido sobre los administradores de contexto y cómo se utilizan para gestionar recursos de manera eficiente y segura en Python. Hemos visto cómo implementar un administrador de contexto personalizado y cómo utilizar la declaración with para simplificar la gestión de recursos. Además, hemos practicado creando un administrador de contexto que mide el tiempo de ejecución de un bloque de código.
