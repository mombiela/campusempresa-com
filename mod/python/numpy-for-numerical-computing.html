Introducción a NumPy

NumPy (Numerical Python) es una poderosa biblioteca para la computación numérica en Python. Proporciona soporte para arrays, matrices y muchas funciones matemáticas para operar sobre estas estructuras de datos.

  • Conceptos Clave:
    • Arrays y su importancia en la computación numérica.
    • Diferencias entre listas de Python y arrays de NumPy.
    • Operaciones básicas con arrays de NumPy.
import numpy as np

# Creando un array de NumPy
array = np.array([1, 2, 3, 4, 5])
print(array)

Creación y Manipulación de Arrays

Aprende cómo crear y manipular arrays usando NumPy.

  • Creación de Arrays:
    • Usando np.array()
    • Usando funciones como np.zeros(), np.ones(), np.arange(), y np.linspace()
# Creando arrays usando diferentes métodos
zeros_array = np.zeros((3, 3))
ones_array = np.ones((2, 2))
range_array = np.arange(10)
linspace_array = np.linspace(0, 1, 5)

print("Array de Ceros:\n", zeros_array)
print("Array de Unos:\n", ones_array)
print("Array de Rango:\n", range_array)
print("Array de Linspace:\n", linspace_array)
  • Manipulación de Arrays:
    • Remodelar arrays con reshape()
    • Aplanar arrays con flatten()
    • Concatenar arrays con concatenate()
# Remodelar y aplanar arrays
reshaped_array = range_array.reshape((2, 5))
flattened_array = reshaped_array.flatten()

print("Array Remodelado:\n", reshaped_array)
print("Array Aplanado:\n", flattened_array)

# Concatenar arrays
array1 = np.array([1, 2, 3])
array2 = np.array([4, 5, 6])
concatenated_array = np.concatenate((array1, array2))

print("Array Concatenado:\n", concatenated_array)

Operaciones con Arrays

Realiza varias operaciones en arrays, incluyendo operaciones aritméticas, operaciones estadísticas y más.

  • Operaciones Aritméticas:
    • Operaciones elemento a elemento
    • Broadcasting
# Operaciones elemento a elemento
array1 = np.array([1, 2, 3])
array2 = np.array([4, 5, 6])
sum_array = array1 + array2
product_array = array1 * array2

print("Array Suma:\n", sum_array)
print("Array Producto:\n", product_array)

# Broadcasting
escalar = 2
broadcasted_array = array1 * escalar

print("Array con Broadcasting:\n", broadcasted_array)
  • Operaciones Estadísticas:
    • Media, mediana, desviación estándar, etc.
# Operaciones estadísticas
data = np.array([1, 2, 3, 4, 5])
mean = np.mean(data)
median = np.median(data)
std_dev = np.std(data)

print("Media:", mean)
print("Mediana:", median)
print("Desviación Estándar:", std_dev)

Manipulaciones Avanzadas de Arrays

Explora técnicas más avanzadas para trabajar con arrays.

  • Slicing e Indexación:
    • Slicing básico
    • Indexación booleana
    • Indexación avanzada
# Slicing básico
array = np.array([1, 2, 3, 4, 5])
slice_array = array[1:4]

print("Array Slicing:", slice_array)

# Indexación booleana
bool_array = array[array > 2]

print("Array con Indexación Booleana:", bool_array)

# Indexación avanzada
indices = [0, 2, 4]
fancy_indexed_array = array[indices]

print("Array con Indexación Avanzada:", fancy_indexed_array)
  • Matemáticas con Arrays:
    • Operaciones de álgebra lineal
    • Generación de números aleatorios
# Operaciones de álgebra lineal
matrix1 = np.array([[1, 2], [3, 4]])
matrix2 = np.array([[5, 6], [7, 8]])
dot_product = np.dot(matrix1, matrix2)

print("Producto Punto:\n", dot_product)

# Generación de números aleatorios
random_array = np.random.rand(3, 3)

print("Array Aleatorio:\n", random_array)

Conclusión

NumPy es una biblioteca esencial para la computación numérica en Python. Proporciona herramientas poderosas para crear y manipular arrays, realizar operaciones matemáticas y manejar grandes conjuntos de datos de manera eficiente. Dominar NumPy mejorará significativamente tu capacidad para realizar cálculos científicos y numéricos en Python.

  • Resumen:
    • Los arrays de NumPy son más eficientes y convenientes que las listas de Python para operaciones numéricas.
    • Puedes crear arrays usando varias funciones de NumPy y manipularlos con remodelación, slicing e indexación.
    • NumPy soporta una amplia gama de operaciones matemáticas, incluyendo operaciones aritméticas, estadísticas y de álgebra lineal.
    • Técnicas avanzadas como broadcasting y generación de números aleatorios también están disponibles en NumPy.

Al entender y utilizar NumPy, puedes realizar cálculos numéricos complejos con facilidad y eficiencia.
