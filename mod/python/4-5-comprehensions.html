4.5 Comprensiones

Introducción

Las comprensiones en Python son una forma concisa y eficiente de crear listas, diccionarios y conjuntos. Permiten generar nuevas colecciones a partir de iterables existentes de una manera más legible y compacta que utilizando bucles tradicionales.

Comprensiones de Listas

Las comprensiones de listas son la forma más común de comprensión en Python. La sintaxis básica es:

[nuevo_elemento for elemento in iterable if condición]

Ejemplo 1: Crear una lista de cuadrados

# Usando un bucle for
cuadrados = []
for x in range(10):
    cuadrados.append(x**2)
print(cuadrados)

# Usando una comprensión de lista
cuadrados = [x**2 for x in range(10)]
print(cuadrados)

Ejemplo 2: Filtrar elementos

# Usando un bucle for
pares = []
for x in range(10):
    if x % 2 == 0:
        pares.append(x)
print(pares)

# Usando una comprensión de lista
pares = [x for x in range(10) if x % 2 == 0]
print(pares)

Comprensiones de Diccionarios

Las comprensiones de diccionarios permiten crear diccionarios de manera similar a las comprensiones de listas.

{clave: valor for elemento in iterable if condición}

Ejemplo 3: Crear un diccionario de cuadrados

# Usando un bucle for
cuadrados_dict = {}
for x in range(10):
    cuadrados_dict[x] = x**2
print(cuadrados_dict)

# Usando una comprensión de diccionario
cuadrados_dict = {x: x**2 for x in range(10)}
print(cuadrados_dict)

Comprensiones de Conjuntos

Las comprensiones de conjuntos permiten crear conjuntos de manera similar a las comprensiones de listas.

{nuevo_elemento for elemento in iterable if condición}

Ejemplo 4: Crear un conjunto de cuadrados

# Usando un bucle for
cuadrados_set = set()
for x in range(10):
    cuadrados_set.add(x**2)
print(cuadrados_set)

# Usando una comprensión de conjunto
cuadrados_set = {x**2 for x in range(10)}
print(cuadrados_set)

Ejercicios Prácticos

Ejercicio 1: Crear una lista de cubos

Crea una lista que contenga los cubos de los números del 0 al 9.

# Solución
cubos = [x**3 for x in range(10)]
print(cubos)

Ejercicio 2: Filtrar palabras

Dada una lista de palabras, crea una nueva lista que contenga solo las palabras que tienen más de 3 letras.

# Solución
palabras = ["hola", "a", "Python", "es", "genial"]
palabras_filtradas = [palabra for palabra in palabras if len(palabra) > 3]
print(palabras_filtradas)

Ejercicio 3: Crear un diccionario de longitudes

Dada una lista de palabras, crea un diccionario donde las claves sean las palabras y los valores sean sus longitudes.

# Solución
palabras = ["hola", "Python", "es", "genial"]
longitudes = {palabra: len(palabra) for palabra in palabras}
print(longitudes)

Resumen

En esta sección, hemos aprendido sobre las comprensiones en Python, una herramienta poderosa para crear listas, diccionarios y conjuntos de manera concisa y eficiente. Las comprensiones no solo hacen que el código sea más legible, sino que también pueden mejorar el rendimiento en comparación con los bucles tradicionales. Ahora estás listo para aplicar comprensiones en tus propios proyectos y simplificar tu código.
