4-3: Diccionarios

Introducción a los Diccionarios

Los diccionarios en Python son estructuras de datos que permiten almacenar pares de clave-valor. Son muy útiles cuando necesitas asociar una colección de valores con claves únicas. A diferencia de las listas, que están indexadas por una secuencia de números, los diccionarios están indexados por claves, que pueden ser de cualquier tipo inmutable (como cadenas, números o tuplas).

Conceptos Clave

  • Clave (Key): Un identificador único para cada valor en el diccionario.
  • Valor (Value): El dato asociado a una clave en el diccionario.
  • Par Clave-Valor: La unidad básica de un diccionario, compuesta por una clave y su valor asociado.

Creación de Diccionarios

Puedes crear un diccionario utilizando llaves {} y separando las claves y valores con dos puntos :.

# Creación de un diccionario vacío
mi_diccionario = {}

# Creación de un diccionario con algunos pares clave-valor
mi_diccionario = {
    "nombre": "Juan",
    "edad": 30,
    "ciudad": "Madrid"
}

Acceso a los Valores

Para acceder a los valores en un diccionario, utilizas las claves entre corchetes [].

print(mi_diccionario["nombre"])  # Salida: Juan
print(mi_diccionario["edad"])    # Salida: 30

Modificación de Valores

Puedes modificar el valor asociado a una clave existente o añadir un nuevo par clave-valor.

# Modificar un valor existente
mi_diccionario["edad"] = 31

# Añadir un nuevo par clave-valor
mi_diccionario["profesión"] = "Ingeniero"

Eliminación de Elementos

Puedes eliminar elementos utilizando la palabra clave del o el método pop().

# Usando del
del mi_diccionario["ciudad"]

# Usando pop()
mi_diccionario.pop("profesión")

Métodos Útiles

Los diccionarios tienen varios métodos útiles:

  • keys(): Devuelve una vista de las claves del diccionario.
  • values(): Devuelve una vista de los valores del diccionario.
  • items(): Devuelve una vista de los pares clave-valor del diccionario.
# Ejemplo de uso de métodos
claves = mi_diccionario.keys()
valores = mi_diccionario.values()
pares = mi_diccionario.items()

print(claves)   # Salida: dict_keys(['nombre', 'edad'])
print(valores)  # Salida: dict_values(['Juan', 31])
print(pares)    # Salida: dict_items([('nombre', 'Juan'), ('edad', 31)])

Ejemplo Práctico

Vamos a crear un diccionario que almacene información sobre estudiantes y sus calificaciones.

# Creación del diccionario
calificaciones = {
    "Ana": 95,
    "Luis": 85,
    "Carlos": 90,
    "Marta": 88
}

# Acceso a las calificaciones
print(calificaciones["Ana"])  # Salida: 95

# Modificación de una calificación
calificaciones["Luis"] = 89

# Añadir una nueva calificación
calificaciones["Pedro"] = 92

# Eliminar una calificación
del calificaciones["Carlos"]

# Mostrar todas las calificaciones
for estudiante, calificacion in calificaciones.items():
    print(f"{estudiante}: {calificacion}")

Ejercicio Práctico

Ejercicio 1:

Crea un diccionario que almacene los nombres de varios países y sus capitales. Luego, realiza las siguientes operaciones:

  1. Añade un nuevo país y su capital.
  2. Modifica la capital de un país existente.
  3. Elimina un país del diccionario.
  4. Imprime todas las claves y valores del diccionario.

Solución:

# Creación del diccionario
paises_capitales = {
    "España": "Madrid",
    "Francia": "París",
    "Italia": "Roma",
    "Alemania": "Berlín"
}

# Añadir un nuevo país y su capital
paises_capitales["Portugal"] = "Lisboa"

# Modificar la capital de un país existente
paises_capitales["Italia"] = "Milán"

# Eliminar un país del diccionario
del paises_capitales["Alemania"]

# Imprimir todas las claves y valores del diccionario
for pais, capital in paises_capitales.items():
    print(f"{pais}: {capital}")

Conclusión

En esta sección, hemos aprendido sobre los diccionarios en Python, una estructura de datos poderosa y flexible para almacenar pares clave-valor. Hemos cubierto cómo crear, acceder, modificar y eliminar elementos en un diccionario, así como algunos métodos útiles para trabajar con ellos. Con estos conocimientos, estás listo para utilizar diccionarios en tus proyectos de Python y manejar datos de manera más eficiente.
