Introducción al Desarrollo Web
Framework Flask ►

El desarrollo web es un término amplio que abarca todas las actividades involucradas en la creación de un sitio web o aplicación web. Este tema te introducirá a los conceptos básicos del desarrollo web utilizando Python, cubriendo conceptos esenciales, herramientas y frameworks.

¿Qué es el Desarrollo Web?

El desarrollo web implica construir y mantener sitios web. Se puede dividir en tres categorías principales:

  • Desarrollo Front-end: Se ocupa de los aspectos visuales de un sitio web con los que los usuarios interactúan.
  • Desarrollo Back-end: Se enfoca en la lógica del lado del servidor, bases de datos y funcionalidad de la aplicación.
  • Desarrollo Full-stack: Combina tanto el desarrollo front-end como el back-end.

¿Por qué Usar Python para el Desarrollo Web?

Python es un lenguaje de programación versátil y poderoso que se usa ampliamente en el desarrollo web debido a:

  • Legibilidad y Simplicidad: La sintaxis de Python es limpia y fácil de entender.
  • Bibliotecas y Frameworks Extensos: Python ofrece bibliotecas y frameworks robustos como Django y Flask.
  • Soporte de la Comunidad: Una gran comunidad de desarrolladores proporciona recursos y soporte extensos.

Configurando Tu Entorno

Antes de sumergirte en el desarrollo web con Python, necesitas configurar tu entorno de desarrollo.

Instalando Python

Asegúrate de tener Python instalado en tu sistema. Puedes descargarlo desde python.org.

Instalando un Editor de Código

Elige un editor de código o IDE (Entorno de Desarrollo Integrado) como:

  • VS Code
  • PyCharm
  • Sublime Text

Instalando un Entorno Virtual

Un entorno virtual ayuda a gestionar las dependencias de tus proyectos. Instálalo usando pip:

pip install virtualenv

Crea un entorno virtual:

virtualenv myenv

Activa el entorno virtual:

  • En Windows: 
    myenv\Scripts\activate
  • En macOS/Linux: 
    source myenv/bin/activate

Introducción a Flask

Flask es un framework web ligero para Python. Es fácil de configurar y perfecto para principiantes.

Instalando Flask

Instala Flask usando pip:

pip install Flask

Creando una Aplicación Simple con Flask

Crea un archivo llamado app.py y añade el siguiente código:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Mundo!"

if __name__ == '__main__':
    app.run(debug=True)

Explicación:

  • Importar Flask: Importa la clase Flask del módulo flask.
  • Crear una Instancia: Crea una instancia de la clase Flask.
  • Definir una Ruta: Usa el decorador @app.route para definir la ruta URL para la página de inicio.
  • Ejecutar la Aplicación: Usa app.run() para ejecutar la aplicación.

Ejecuta la aplicación:

python app.py

Visita http://127.0.0.1:5000/ en tu navegador web para ver el resultado.

Introducción a Django

Django es un framework web de alto nivel para Python que fomenta el desarrollo rápido y un diseño limpio y pragmático.

Instalando Django

Instala Django usando pip:

pip install django

Creando un Proyecto Django

Crea un nuevo proyecto Django:

django-admin startproject myproject

Navega al directorio del proyecto y ejecuta el servidor de desarrollo:

cd myproject
python manage.py runserver

Visita http://127.0.0.1:8000/ para ver la página de bienvenida predeterminada de Django.

Creando una Aplicación Django

Crea una nueva aplicación dentro de tu proyecto:

python manage.py startapp myapp

Añade la aplicación a la configuración de tu proyecto en myproject/settings.py:

INSTALLED_APPS = [
    ...
    'myapp',
]

Define una vista en myapp/views.py:

from django.http import HttpResponse

def home(request):
    return HttpResponse("¡Hola, Mundo!")

Mapea la vista a una URL en myapp/urls.py:

from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]

Incluye las URLs de la aplicación en el urls.py del proyecto:

from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

Ejecuta el servidor nuevamente y visita http://127.0.0.1:8000/ para ver el resultado.

Conclusión

En este tema, has aprendido los conceptos básicos del desarrollo web utilizando Python. Cubrimos:

  • Los diferentes aspectos del desarrollo web.
  • Configuración de tu entorno de desarrollo.
  • Creación de aplicaciones web simples usando Flask y Django.

El desarrollo web es un campo vasto, y esta introducción es solo el comienzo. A medida que avances, explorarás temas más avanzados y construirás aplicaciones más complejas. ¡Feliz codificación!
