Tema 10-3: Uso de pdb para Depuración

Introducción

El depurador pdb (Python Debugger) es una herramienta poderosa que permite a los desarrolladores inspeccionar y controlar la ejecución de un programa Python. Con pdb, puedes establecer puntos de interrupción, inspeccionar variables, y ejecutar código paso a paso para identificar y corregir errores.

1. Introducción a pdb

pdb es el depurador estándar de Python y se incluye en la biblioteca estándar. Permite a los desarrolladores:

  • Pausar la ejecución del programa en puntos específicos.
  • Inspeccionar el estado del programa.
  • Ejecutar el código línea por línea.
  • Modificar variables en tiempo de ejecución.

2. Comandos Básicos de pdb

A continuación, se presentan algunos de los comandos más utilizados en pdb:

Comando Descripción
break o b Establece un punto de interrupción en una línea específica.
continue o c Reanuda la ejecución hasta el siguiente punto de interrupción.
step o s Ejecuta la siguiente línea de código, entrando en funciones.
next o n Ejecuta la siguiente línea de código, sin entrar en funciones.
list o l Muestra el código fuente alrededor de la línea actual.
print o p Imprime el valor de una expresión.
quit o q Sale del depurador y termina el programa.

3. Uso de pdb en un Script

Para utilizar pdb en un script Python, puedes importar el módulo pdb y llamar a pdb.set_trace() en el punto donde deseas iniciar la depuración.

import pdb

def suma(a, b):
    resultado = a + b
    pdb.set_trace()  # Inicia el depurador aquí
    return resultado

print(suma(3, 4))

4. Ejemplo Práctico

Vamos a depurar un script que contiene un error lógico.

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Supongamos que queremos inspeccionar cómo se calcula el factorial. Podemos añadir pdb.set_trace() dentro de la función factorial.

import pdb

def factorial(n):
    pdb.set_trace()  # Inicia el depurador aquí
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))

Al ejecutar el script, el depurador se activará en la línea donde se llama a pdb.set_trace(). Puedes usar los comandos de pdb para inspeccionar y controlar la ejecución.

5. Ejercicios Prácticos

Ejercicio 1: Depuración de un Error de Suma

def suma(a, b):
    resultado = a - b  # Error intencional
    return resultado

print(suma(10, 5))

Instrucciones:

  1. Añade pdb.set_trace() dentro de la función suma.
  2. Utiliza pdb para identificar y corregir el error.

Ejercicio 2: Depuración de una Función de Búsqueda

def buscar_elemento(lista, elemento):
    for i in range(len(lista)):
        if lista[i] == elemento:
            return i
    return -1

mi_lista = [1, 2, 3, 4, 5]
print(buscar_elemento(mi_lista, 3))

Instrucciones:

  1. Añade pdb.set_trace() dentro de la función buscar_elemento.
  2. Utiliza pdb para seguir la ejecución y verificar que la función funciona correctamente.

6. Conclusión

El uso de pdb es esencial para cualquier desarrollador que desee depurar y entender mejor su código. Con pdb, puedes pausar la ejecución del programa, inspeccionar variables y ejecutar el código paso a paso, lo que facilita la identificación y corrección de errores.

Resumen

  • pdb es el depurador estándar de Python.
  • Puedes establecer puntos de interrupción, inspeccionar variables y ejecutar código paso a paso.
  • Los comandos básicos de pdb incluyen break, continue, step, next, list, print y quit.
  • pdb.set_trace() se utiliza para iniciar el depurador en un punto específico del código.
  • La práctica con pdb ayuda a mejorar tus habilidades de depuración y comprensión del código.
