4-2: Tuplas

Introducción a las Tuplas

Las tuplas son una estructura de datos en Python que permite almacenar múltiples elementos en una sola variable. A diferencia de las listas, las tuplas son inmutables, lo que significa que una vez creadas, sus elementos no pueden ser modificados. Las tuplas son útiles cuando se necesita una colección de elementos que no deben cambiar a lo largo del tiempo.

Características de las Tuplas

  • Inmutables: No se pueden modificar después de su creación.
  • Ordenadas: Mantienen el orden de los elementos.
  • Permiten duplicados: Pueden contener elementos duplicados.
  • Heterogéneas: Pueden contener diferentes tipos de datos (enteros, cadenas, listas, etc.).

Creación de Tuplas

Las tuplas se pueden crear de varias maneras:

  1. Usando paréntesis:

    mi_tupla = (1, 2, 3)

  2. Sin paréntesis (empaquetado de tuplas):

    mi_tupla = 1, 2, 3

  3. Usando la función tuple():

    mi_tupla = tuple([1, 2, 3])

Acceso a Elementos

Puedes acceder a los elementos de una tupla utilizando índices, de la misma manera que con las listas:

mi_tupla = (10, 20, 30, 40)
print(mi_tupla[0])  # Salida: 10
print(mi_tupla[2])  # Salida: 30

Desempaquetado de Tuplas

El desempaquetado de tuplas permite asignar los elementos de una tupla a variables individuales:

mi_tupla = (1, 2, 3)
a, b, c = mi_tupla
print(a)  # Salida: 1
print(b)  # Salida: 2
print(c)  # Salida: 3

Operaciones con Tuplas

  • Concatenación:

    tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla3 = tupla1 + tupla2
print(tupla3)  # Salida: (1, 2, 3, 4, 5, 6)

  • Repetición:

    mi_tupla = (1, 2, 3)
mi_tupla_repetida = mi_tupla * 2
print(mi_tupla_repetida)  # Salida: (1, 2, 3, 1, 2, 3)

  • Verificación de membresía:

    mi_tupla = (1, 2, 3)
print(2 in mi_tupla)  # Salida: True
print(4 in mi_tupla)  # Salida: False

Métodos de Tuplas

Las tuplas tienen métodos limitados debido a su inmutabilidad:

  • count(): Devuelve el número de veces que un valor aparece en la tupla.

    mi_tupla = (1, 2, 2, 3)
print(mi_tupla.count(2))  # Salida: 2

  • index(): Devuelve el índice del primer elemento con el valor especificado.

    mi_tupla = (1, 2, 3)
print(mi_tupla.index(2))  # Salida: 1

Ejercicios Prácticos

  1. Crear y Acceder a una Tupla:

    # Crea una tupla con los números del 1 al 5
numeros = (1, 2, 3, 4, 5)

# Accede y muestra el tercer elemento de la tupla
print(numeros[2])  # Salida: 3

  2. Desempaquetado de Tuplas:

    # Crea una tupla con tres elementos
coordenadas = (10.0, 20.0, 30.0)

# Desempaqueta los valores en variables individuales
x, y, z = coordenadas

# Muestra los valores desempaquetados
print(f"x: {x}, y: {y}, z: {z}")  # Salida: x: 10.0, y: 20.0, z: 30.0

  3. Operaciones con Tuplas:

    # Crea dos tuplas
tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)

# Concatenación de tuplas
tupla_concatenada = tupla1 + tupla2
print(tupla_concatenada)  # Salida: (1, 2, 3, 4, 5, 6)

# Repetición de tuplas
tupla_repetida = tupla1 * 2
print(tupla_repetida)  # Salida: (1, 2, 3, 1, 2, 3)

Soluciones a los Ejercicios

  2. Desempaquetado de Tuplas:

    coordenadas = (10.0, 20.0, 30.0)
x, y, z = coordenadas
print(f"x: {x}, y: {y}, z: {z}")  # Salida: x: 10.0, y: 20.0, z: 30.0

  3. Operaciones con Tuplas:

    tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla_concatenada = tupla1 + tupla2
print(tupla_concatenada)  # Salida: (1, 2, 3, 4, 5, 6)

tupla_repetida = tupla1 * 2
print(tupla_repetida)  # Salida: (1, 2, 3, 1, 2, 3)

Conclusión

Las tuplas son una estructura de datos fundamental en Python que permite almacenar colecciones de elementos inmutables. Son útiles cuando se necesita garantizar que los datos no cambien a lo largo del tiempo. En esta sección, hemos aprendido cómo crear, acceder y manipular tuplas, así como algunas de sus operaciones y métodos más comunes. Con estos conocimientos, estás listo para utilizar tuplas en tus programas de Python y aprovechar su inmutabilidad para escribir código más seguro y confiable.
