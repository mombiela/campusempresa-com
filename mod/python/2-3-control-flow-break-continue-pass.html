Tema 2-3: Flujo de Control: `break`, `continue` y `pass`

En este tema, aprenderemos sobre tres importantes declaraciones de control de flujo en Python: break, continue y pass. Estas declaraciones se utilizan para alterar el comportamiento de los bucles y otras estructuras de control de flujo.

`break`

La declaración break se utiliza para salir de un bucle de forma prematura. Cuando se ejecuta break, el control del programa se transfiere inmediatamente a la primera línea de código después del bucle.

Ejemplo:

for i in range(10):
    if i == 5:
        break
    print(i)

Explicación:

  • El bucle for itera desde 0 hasta 9.
  • Cuando i es igual a 5, la declaración break se ejecuta y el bucle se termina.
  • La salida del programa será: 
    0
1
2
3
4

`continue`

La declaración continue se utiliza para omitir el resto del código dentro de un bucle en la iteración actual y pasar a la siguiente iteración del bucle.

Ejemplo:

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Explicación:

  • El bucle for itera desde 0 hasta 9.
  • Si i es un número par (i % 2 == 0), la declaración continue se ejecuta y el resto del código en el bucle se omite para esa iteración.
  • La salida del programa será: 
    1
3
5
7
9

`pass`

La declaración pass no hace nada. Se utiliza como un marcador de posición en el código donde se requiere una declaración sintácticamente, pero no se necesita ninguna acción.

Ejemplo:

for i in range(5):
    if i == 3:
        pass
    print(i)

Explicación:

  • El bucle for itera desde 0 hasta 4.
  • Cuando i es igual a 3, la declaración pass se ejecuta, pero no hace nada y el bucle continúa.
  • La salida del programa será: 
    0
1
2
3
4

Ejercicios Prácticos

Ejercicio 1: Uso de break

Escribe un programa que encuentre el primer número divisible por 7 en un rango de 1 a 100 y termine el bucle una vez que lo encuentre.

for i in range(1, 101):
    if i % 7 == 0:
        print(f"El primer número divisible por 7 es: {i}")
        break

Ejercicio 2: Uso de continue

Escribe un programa que imprima todos los números del 1 al 10, excepto el 5.

for i in range(1, 11):
    if i == 5:
        continue
    print(i)

Ejercicio 3: Uso de pass

Escribe un programa que itere sobre una lista de números y use pass para los números negativos, imprimiendo solo los números positivos.

numbers = [10, -3, 5, -1, 7, -6]

for num in numbers:
    if num < 0:
        pass
    else:
        print(num)

Resumen

En este tema, hemos aprendido sobre las declaraciones de control de flujo break, continue y pass en Python. Estas declaraciones son herramientas poderosas para controlar el comportamiento de los bucles y otras estructuras de control de flujo. Asegúrate de practicar con los ejemplos y ejercicios proporcionados para consolidar tu comprensión de estos conceptos.
