1-2: Configuración del Entorno de Desarrollo

En esta lección, aprenderás cómo configurar tu entorno de desarrollo para comenzar a programar en Python. Cubriremos la instalación de Python, la configuración de un editor de código y la instalación de paquetes necesarios.

Objetivos de la Lección

  • Instalar Python en tu sistema operativo.
  • Configurar un editor de código.
  • Instalar paquetes necesarios usando pip.

Instalación de Python

Windows

  1. Descargar el Instalador:

    • Ve a la página oficial de Python: python.org.
    • Descarga el instalador de la última versión estable de Python.

  2. Ejecutar el Instalador:

    • Abre el archivo descargado.
    • Asegúrate de marcar la opción "Add Python to PATH".
    • Haz clic en "Install Now".

  3. Verificar la Instalación:

    • Abre la terminal de comandos (CMD).
    • Escribe python --version y presiona Enter.
    • Deberías ver la versión de Python instalada.

macOS

  1. Usar Homebrew (Recomendado):

    • Abre la Terminal.
    • Instala Homebrew si no lo tienes: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)".
    • Instala Python: brew install python.

  2. Verificar la Instalación:

    • Escribe python3 --version en la Terminal y presiona Enter.
    • Deberías ver la versión de Python instalada.

Linux

  1. Usar el Gestor de Paquetes:

    • Abre la Terminal.
    • Para distribuciones basadas en Debian (como Ubuntu): sudo apt-get update y luego sudo apt-get install python3.
    • Para distribuciones basadas en Red Hat: sudo yum install python3.

  2. Verificar la Instalación:

    • Escribe python3 --version en la Terminal y presiona Enter.
    • Deberías ver la versión de Python instalada.

Configuración de un Editor de Código

Visual Studio Code (VS Code)

  1. Descargar e Instalar:

    • Ve a la página oficial de VS Code: code.visualstudio.com.
    • Descarga e instala la versión correspondiente a tu sistema operativo.

  2. Instalar la Extensión de Python:

    • Abre VS Code.
    • Ve a la pestaña de Extensiones (icono de cuadrado en la barra lateral izquierda).
    • Busca "Python" y selecciona la extensión de Microsoft.
    • Haz clic en "Install".

Otros Editores Populares

  • PyCharm: Ideal para proyectos grandes y complejos.
  • Sublime Text: Ligero y rápido.
  • Atom: Altamente personalizable.

Instalación de Paquetes con `pip`

pip es el gestor de paquetes de Python que te permite instalar y gestionar bibliotecas adicionales que no están incluidas en la biblioteca estándar de Python.

Instalación de un Paquete

  1. Abrir la Terminal o CMD.
  2. Instalar un Paquete:
    • Escribe pip install nombre_del_paquete y presiona Enter.
    • Ejemplo: pip install requests.

Verificar la Instalación de un Paquete

  1. Abrir la Terminal o CMD.
  2. Verificar la Instalación:
    • Escribe pip show nombre_del_paquete y presiona Enter.
    • Ejemplo: pip show requests.

Ejemplo Práctico

Vamos a instalar el paquete requests y hacer una solicitud HTTP simple.

  1. Instalar requests:

    pip install requests

  2. Crear un Archivo Python:

    • Abre tu editor de código.
    • Crea un nuevo archivo llamado example.py.

  3. Escribir el Código:

    import requests

response = requests.get('https://api.github.com')
print(response.status_code)
print(response.json())

  4. Ejecutar el Archivo:

    • Abre la Terminal o CMD.
    • Navega hasta la carpeta donde guardaste example.py.
    • Escribe python example.py y presiona Enter.

Ejercicio Práctico

Objetivo: Instalar el paquete numpy y usarlo para realizar una operación matemática básica.

  1. Instalar numpy:

    pip install numpy

  2. Crear un Archivo Python:

    • Abre tu editor de código.
    • Crea un nuevo archivo llamado numpy_example.py.

  3. Escribir el Código:

    import numpy as np

array = np.array([1, 2, 3, 4, 5])
print("Array:", array)
print("Suma de todos los elementos:", np.sum(array))

  4. Ejecutar el Archivo:

    • Abre la Terminal o CMD.
    • Navega hasta la carpeta donde guardaste numpy_example.py.
    • Escribe python numpy_example.py y presiona Enter.

Solución del Ejercicio

El código debería imprimir:

Array: [1 2 3 4 5]
Suma de todos los elementos: 15

Conclusión

En esta lección, has aprendido a instalar Python, configurar un editor de código y usar pip para instalar paquetes. Ahora estás listo para comenzar a escribir y ejecutar tus propios programas en Python. En la próxima lección, exploraremos la sintaxis básica de Python y crearemos nuestro primer programa "Hola Mundo".
