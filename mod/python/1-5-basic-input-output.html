Tema 1.5: Entrada y Salida Básica

Introducción

En este tema, aprenderás cómo manejar la entrada y salida básica en Python. La entrada se refiere a cómo obtener datos del usuario, mientras que la salida se refiere a cómo mostrar datos al usuario. Estos son conceptos fundamentales que te permitirán interactuar con los usuarios de tus programas.

Contenido

  1. Salida Básica con print()
  2. Entrada Básica con input()
  3. Ejemplos Prácticos
  4. Ejercicios Prácticos

1. Salida Básica con `print()`

La función print() se utiliza para mostrar información en la pantalla. Es una de las funciones más utilizadas en Python y es muy sencilla de usar.

Sintaxis

print(objeto1, objeto2, ..., sep=' ', end='\n', file=sys.stdout, flush=False)
  • objeto1, objeto2, ...: Los objetos que deseas imprimir.
  • sep: El separador entre los objetos (por defecto es un espacio).
  • end: Lo que se añade al final de la salida (por defecto es una nueva línea).
  • file: El archivo donde se enviará la salida (por defecto es la pantalla).
  • flush: Si se debe forzar el vaciado del búfer de salida (por defecto es False).

Ejemplos

# Ejemplo básico
print("Hola, Mundo!")

# Imprimir múltiples objetos
print("Hola", "Mundo", sep=", ")

# Cambiar el final de la línea
print("Hola, Mundo!", end=" FIN\n")

2. Entrada Básica con `input()`

La función input() se utiliza para obtener datos del usuario. Siempre devuelve los datos como una cadena de texto (string).

Sintaxis

variable = input(prompt)
  • prompt: El mensaje que se muestra al usuario (opcional).

Ejemplos

# Solicitar al usuario que ingrese su nombre
nombre = input("¿Cuál es tu nombre? ")
print("Hola, " + nombre + "!")

# Solicitar al usuario que ingrese su edad y convertirla a un número entero
edad = int(input("¿Cuántos años tienes? "))
print("Tienes " + str(edad) + " años.")

3. Ejemplos Prácticos

Ejemplo 1: Calculadora de Suma

# Solicitar dos números al usuario
numero1 = float(input("Ingresa el primer número: "))
numero2 = float(input("Ingresa el segundo número: "))

# Calcular la suma
suma = numero1 + numero2

# Mostrar el resultado
print("La suma de", numero1, "y", numero2, "es", suma)

Ejemplo 2: Conversión de Temperatura

# Solicitar la temperatura en grados Celsius
celsius = float(input("Ingresa la temperatura en grados Celsius: "))

# Convertir a Fahrenheit
fahrenheit = (celsius * 9/5) + 32

# Mostrar el resultado
print(celsius, "grados Celsius son", fahrenheit, "grados Fahrenheit")

4. Ejercicios Prácticos

Ejercicio 1: Calculadora de Promedio

Escribe un programa que solicite tres números al usuario y calcule su promedio.

# Solicitar tres números al usuario
numero1 = float(input("Ingresa el primer número: "))
numero2 = float(input("Ingresa el segundo número: "))
numero3 = float(input("Ingresa el tercer número: "))

# Calcular el promedio
promedio = (numero1 + numero2 + numero3) / 3

# Mostrar el resultado
print("El promedio de", numero1, ",", numero2, "y", numero3, "es", promedio)

Ejercicio 2: Calculadora de Área de un Círculo

Escribe un programa que solicite el radio de un círculo al usuario y calcule su área. Usa la fórmula: área = π * radio^2.

import math

# Solicitar el radio al usuario
radio = float(input("Ingresa el radio del círculo: "))

# Calcular el área
area = math.pi * (radio ** 2)

# Mostrar el resultado
print("El área del círculo con radio", radio, "es", area)

Conclusión

En este tema, has aprendido cómo manejar la entrada y salida básica en Python utilizando las funciones print() e input(). Estos conceptos son fundamentales para cualquier programa que necesite interactuar con el usuario. Asegúrate de practicar con los ejemplos y ejercicios proporcionados para reforzar tu comprensión.
