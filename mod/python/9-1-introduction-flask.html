9-1: Introducción a Flask

Objetivos del Tema

  • Comprender qué es Flask y sus ventajas.
  • Configurar un entorno de desarrollo para Flask.
  • Crear una aplicación web básica con Flask.

¿Qué es Flask?

Flask es un microframework para Python que permite desarrollar aplicaciones web de manera sencilla y rápida. A diferencia de otros frameworks más grandes como Django, Flask es minimalista y flexible, permitiendo al desarrollador elegir las herramientas y bibliotecas que mejor se adapten a sus necesidades.

Ventajas de Flask:

  • Ligero y Flexible: No impone una estructura rígida, lo que permite mayor libertad en el diseño de la aplicación.
  • Extensible: Puedes añadir extensiones para funcionalidades adicionales como autenticación, bases de datos, etc.
  • Documentación y Comunidad: Amplia documentación y una comunidad activa que facilita la resolución de problemas.

Configuración del Entorno de Desarrollo

Para comenzar a trabajar con Flask, primero necesitamos configurar nuestro entorno de desarrollo.

  1. Instalar Python: Asegúrate de tener Python instalado en tu sistema. Puedes descargarlo desde python.org.

  2. Crear un Entorno Virtual: Es una buena práctica utilizar un entorno virtual para gestionar las dependencias de tu proyecto.

    python -m venv myenv
source myenv/bin/activate  # En Windows usa `myenv\Scripts\activate`

  3. Instalar Flask: Una vez activado el entorno virtual, instala Flask usando pip.

    pip install Flask

Crear una Aplicación Web Básica

Vamos a crear una aplicación web simple que muestre "Hola, Mundo!" en el navegador.

  1. Estructura del Proyecto: Crea una carpeta para tu proyecto y dentro de ella un archivo llamado app.py.

    my_flask_app/
└── app.py

  2. Código de la Aplicación: Abre app.py y escribe el siguiente código:

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hola, Mundo!'

if __name__ == '__main__':
    app.run(debug=True)

    Explicación del Código:

    • from flask import Flask: Importamos la clase Flask del módulo flask.
    • app = Flask(__name__): Creamos una instancia de la clase Flask.
    • @app.route('/'): Definimos una ruta para la URL raíz ('/').
    • def hello_world(): Definimos una función que se ejecutará cuando se acceda a la URL raíz.
    • return 'Hola, Mundo!': La función devuelve el texto "Hola, Mundo!".
    • if __name__ == '__main__':: Verificamos si el script se está ejecutando directamente.
    • app.run(debug=True): Ejecutamos la aplicación en modo de depuración.

  3. Ejecutar la Aplicación: En la terminal, navega a la carpeta de tu proyecto y ejecuta el archivo app.py.

    python app.py

    Deberías ver una salida similar a esta:

    * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
* Restarting with stat
* Debugger is active!
* Debugger PIN: 123-456-789

  4. Verificar en el Navegador: Abre tu navegador web y navega a http://127.0.0.1:5000/. Deberías ver el mensaje "Hola, Mundo!".

Ejercicio Práctico

Ejercicio: Modifica la aplicación para que tenga una nueva ruta /saludo que devuelva el mensaje "¡Bienvenido a Flask!".

Solución:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hola, Mundo!'

@app.route('/saludo')
def saludo():
    return '¡Bienvenido a Flask!'

if __name__ == '__main__':
    app.run(debug=True)

Resumen

En esta lección, hemos aprendido qué es Flask y sus ventajas, cómo configurar un entorno de desarrollo para Flask y cómo crear una aplicación web básica. Ahora estamos listos para profundizar en la creación de aplicaciones web más complejas utilizando Flask.

En la próxima lección, aprenderemos a crear una aplicación web simple con más funcionalidades y a manejar rutas y vistas en Flask.
