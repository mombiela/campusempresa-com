Introducción

En esta sección, profundizaremos en estructuras de datos avanzadas en Python. Comprender estas estructuras te ayudará a escribir código más eficiente y optimizado. Cubriremos los siguientes conceptos clave:

  • Montículos
  • Trie
  • Grafos
  • Árboles de Segmento
  • Conjunto Disjunto (Union-Find)

Montículos

Los montículos son estructuras de datos especializadas basadas en árboles que satisfacen la propiedad del montículo. Se utilizan comúnmente para implementar colas de prioridad.

Conceptos Clave

  • Min-Montículo: La clave en la raíz debe ser la mínima entre todas las claves presentes en el Montículo Binario.
  • Max-Montículo: La clave en la raíz debe ser la máxima entre todas las claves presentes en el Montículo Binario.

Ejemplo de Código

import heapq

# Min-Montículo
min_heap = []
heapq.heappush(min_heap, 10)
heapq.heappush(min_heap, 20)
heapq.heappush(min_heap, 5)

print("Min-Montículo:", min_heap)
print("Elemento más pequeño:", heapq.heappop(min_heap))

# Max-Montículo
max_heap = []
heapq.heappush(max_heap, -10)
heapq.heappush(max_heap, -20)
heapq.heappush(max_heap, -5)

print("Max-Montículo:", [-x for x in max_heap])
print("Elemento más grande:", -heapq.heappop(max_heap))

Explicación

  • heapq.heappush añade un elemento al montículo.
  • heapq.heappop elimina y devuelve el elemento más pequeño del montículo.
  • Para un max-montículo, insertamos valores negativos para simular el comportamiento.

Trie

Un Trie (pronunciado como "try") es un tipo especial de árbol utilizado para almacenar estructuras de datos asociativas. Una aplicación común de un Trie es almacenar un diccionario de texto predictivo o autocompletado.

Conceptos Clave

  • Nodos: Cada nodo representa un carácter de las cadenas almacenadas.
  • Aristas: Las aristas conectan nodos y representan la secuencia de caracteres.

Ejemplo de Código

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

# Uso
trie = Trie()
trie.insert("hello")
print(trie.search("hello"))  # Salida: True
print(trie.search("hell"))   # Salida: False

Explicación

  • El método insert añade una palabra al Trie.
  • El método search verifica si una palabra está presente en el Trie.

Grafos

Los grafos se utilizan para representar redes de nodos conectados. Son ampliamente utilizados en diversas aplicaciones como redes sociales, algoritmos de enrutamiento, etc.

Conceptos Clave

  • Vértices (Nodos): Unidades fundamentales del grafo.
  • Aristas: Conexiones entre los vértices.

Ejemplo de Código

class Graph:
    def __init__(self):
        self.graph = {}

    def add_edge(self, u, v):
        if u not in self.graph:
            self.graph[u] = []
        self.graph[u].append(v)

    def bfs(self, start):
        visited = set()
        queue = [start]
        while queue:
            vertex = queue.pop(0)
            if vertex not in visited:
                print(vertex, end=" ")
                visited.add(vertex)
                queue.extend(set(self.graph[vertex]) - visited)

# Uso
g = Graph()
g.add_edge(0, 1)
g.add_edge(0, 2)
g.add_edge(1, 2)
g.add_edge(2, 0)
g.add_edge(2, 3)
g.add_edge(3, 3)

print("Búsqueda en Anchura comenzando desde el vértice 2:")
g.bfs(2)  # Salida: 2 0 3 1

Explicación

  • El método add_edge añade una arista entre dos vértices.
  • El método bfs realiza una Búsqueda en Anchura comenzando desde un vértice dado.

Árboles de Segmento

Los Árboles de Segmento se utilizan para almacenar intervalos o segmentos. Permiten consultar qué segmentos almacenados contienen un punto dado de manera eficiente.

Conceptos Clave

  • Construcción: Construir el árbol a partir de un array.
  • Consulta: Encontrar la suma, mínimo o máximo en un rango dado.

Ejemplo de Código

class SegmentTree:
    def __init__(self, data):
        self.n = len(data)
        self.tree = [0] * (2 * self.n)
        self.build(data)

    def build(self, data):
        for i in range(self.n):
            self.tree[self.n + i] = data[i]
        for i in range(self.n - 1, 0, -1):
            self.tree[i] = self.tree[i * 2] + self.tree[i * 2 + 1]

    def update(self, pos, value):
        pos += self.n
        self.tree[pos] = value
        while pos > 1:
            pos //= 2
            self.tree[pos] = self.tree[2 * pos] + self.tree[2 * pos + 1]

    def query(self, left, right):
        left += self.n
        right += self.n
        result = 0
        while left < right:
            if left % 2:
                result += self.tree[left]
                left += 1
            if right % 2:
                right -= 1
                result += self.tree[right]
            left //= 2
            right //= 2
        return result

# Uso
data = [1, 3, 5, 7, 9, 11]
seg_tree = SegmentTree(data)
print("Suma de valores en el rango [1, 5):", seg_tree.query(1, 5))  # Salida: 24
seg_tree.update(1, 10)
print("Suma de valores en el rango [1, 5) después de la actualización:", seg_tree.query(1, 5))  # Salida: 31

Explicación

  • El método build construye el árbol de segmento.
  • El método update actualiza el valor en una posición específica.
  • El método query devuelve la suma de valores en un rango dado.

Conjunto Disjunto (Union-Find)

El Conjunto Disjunto, también conocido como Union-Find, es una estructura de datos que realiza un seguimiento de una partición de un conjunto en subconjuntos disjuntos (no superpuestos).

Conceptos Clave

  • Union: Fusionar dos subconjuntos en un único subconjunto.
  • Find: Determinar a qué subconjunto pertenece un elemento en particular.

Ejemplo de Código

class DisjointSet:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, u):
        if self.parent[u] != u:
            self.parent[u] = self.find(self.parent[u])
        return self.parent[u]

    def union(self, u, v):
        root_u = self.find(u)
        root_v = self.find(v)
        if root_u != root_v:
            if self.rank[root_u] > self.rank[root_v]:
                self.parent[root_v] = root_u
            elif self.rank[root_u] < self.rank[root_v]:
                self.parent[root_u] = root_v
            else:
                self.parent[root_v] = root_u
                self.rank[root_u] += 1

# Uso
ds = DisjointSet(5)
ds.union(0, 2)
ds.union(4, 2)
ds.union(3, 1)
print(ds.find(4))  # Salida: 0
print(ds.find(3))  # Salida: 1

Explicación

  • El método find devuelve la raíz del conjunto que contiene el elemento.
  • El método union fusiona dos conjuntos.

Conclusión

En esta sección, exploramos varias estructuras de datos avanzadas en Python, incluyendo Montículos, Trie, Grafos, Árboles de Segmento y Conjuntos Disjuntos. Cada una de estas estructuras tiene propiedades y casos de uso únicos que pueden ayudarte a resolver problemas complejos de manera más eficiente. Comprender e implementar estas estructuras de datos mejorará significativamente tus habilidades de programación y te permitirá abordar temas y desafíos más avanzados.
