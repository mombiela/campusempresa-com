4-4: Conjuntos (Sets)

Introducción a los Conjuntos

Los conjuntos (sets) en Python son una colección desordenada de elementos únicos. Esto significa que no pueden contener elementos duplicados y no mantienen ningún orden específico. Los conjuntos son útiles cuando necesitas almacenar elementos únicos y realizar operaciones matemáticas como unión, intersección y diferencia.

Creación de Conjuntos

Puedes crear un conjunto utilizando la función set() o utilizando llaves {}.

# Creación de un conjunto vacío
conjunto_vacio = set()

# Creación de un conjunto con elementos
conjunto = {1, 2, 3, 4, 5}

# Creación de un conjunto a partir de una lista
lista = [1, 2, 2, 3, 4, 4, 5]
conjunto_desde_lista = set(lista)

print(conjunto_vacio)  # Output: set()
print(conjunto)        # Output: {1, 2, 3, 4, 5}
print(conjunto_desde_lista)  # Output: {1, 2, 3, 4, 5}

Operaciones Básicas con Conjuntos

Añadir y Eliminar Elementos

Puedes añadir elementos a un conjunto utilizando el método add() y eliminarlos utilizando remove() o discard().

conjunto = {1, 2, 3}
conjunto.add(4)
print(conjunto)  # Output: {1, 2, 3, 4}

conjunto.remove(2)
print(conjunto)  # Output: {1, 3, 4}

# El método discard() no genera un error si el elemento no está presente
conjunto.discard(5)
print(conjunto)  # Output: {1, 3, 4}

Verificar Pertenencia

Puedes verificar si un elemento está en un conjunto utilizando el operador in.

conjunto = {1, 2, 3, 4, 5}
print(3 in conjunto)  # Output: True
print(6 in conjunto)  # Output: False

Operaciones de Conjuntos

Unión

La unión de dos conjuntos contiene todos los elementos de ambos conjuntos. Puedes utilizar el operador | o el método union().

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
union = conjunto1 | conjunto2
print(union)  # Output: {1, 2, 3, 4, 5}

union = conjunto1.union(conjunto2)
print(union)  # Output: {1, 2, 3, 4, 5}

Intersección

La intersección de dos conjuntos contiene solo los elementos que están presentes en ambos conjuntos. Puedes utilizar el operador & o el método intersection().

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
interseccion = conjunto1 & conjunto2
print(interseccion)  # Output: {3}

interseccion = conjunto1.intersection(conjunto2)
print(interseccion)  # Output: {3}

Diferencia

La diferencia de dos conjuntos contiene los elementos que están en el primer conjunto pero no en el segundo. Puedes utilizar el operador - o el método difference().

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
diferencia = conjunto1 - conjunto2
print(diferencia)  # Output: {1, 2}

diferencia = conjunto1.difference(conjunto2)
print(diferencia)  # Output: {1, 2}

Diferencia Simétrica

La diferencia simétrica de dos conjuntos contiene los elementos que están en uno de los conjuntos, pero no en ambos. Puedes utilizar el operador ^ o el método symmetric_difference().

conjunto1 = {1, 2, 3}
conjunto2 = {3, 4, 5}
diferencia_simetrica = conjunto1 ^ conjunto2
print(diferencia_simetrica)  # Output: {1, 2, 4, 5}

diferencia_simetrica = conjunto1.symmetric_difference(conjunto2)
print(diferencia_simetrica)  # Output: {1, 2, 4, 5}

Ejercicios Prácticos

Ejercicio 1: Creación y Manipulación de Conjuntos

  1. Crea un conjunto con los números del 1 al 5.
  2. Añade el número 6 al conjunto.
  3. Elimina el número 3 del conjunto.
  4. Verifica si el número 4 está en el conjunto.
# Solución
conjunto = {1, 2, 3, 4, 5}
conjunto.add(6)
conjunto.remove(3)
print(4 in conjunto)  # Output: True

Ejercicio 2: Operaciones de Conjuntos

  1. Crea dos conjuntos: conjuntoA con los números del 1 al 5 y conjuntoB con los números del 4 al 8.
  2. Encuentra la unión de conjuntoA y conjuntoB.
  3. Encuentra la intersección de conjuntoA y conjuntoB.
  4. Encuentra la diferencia de conjuntoA y conjuntoB.
  5. Encuentra la diferencia simétrica de conjuntoA y conjuntoB.
# Solución
conjuntoA = {1, 2, 3, 4, 5}
conjuntoB = {4, 5, 6, 7, 8}

union = conjuntoA | conjuntoB
print(union)  # Output: {1, 2, 3, 4, 5, 6, 7, 8}

interseccion = conjuntoA & conjuntoB
print(interseccion)  # Output: {4, 5}

diferencia = conjuntoA - conjuntoB
print(diferencia)  # Output: {1, 2, 3}

diferencia_simetrica = conjuntoA ^ conjuntoB
print(diferencia_simetrica)  # Output: {1, 2, 3, 6, 7, 8}

Conclusión

En esta sección, hemos aprendido sobre los conjuntos en Python, cómo crearlos y manipularlos, y cómo realizar operaciones comunes como unión, intersección, diferencia y diferencia simétrica. Los conjuntos son una herramienta poderosa cuando necesitas trabajar con colecciones de elementos únicos y realizar operaciones matemáticas.
