Pandas para la Manipulación de Datos
Matplotlib para Visualización de Datos ►

Pandas es una biblioteca de Python poderosa y flexible utilizada para la manipulación y análisis de datos. Proporciona estructuras de datos y funciones necesarias para manipular datos estructurados sin problemas. Este tema te guiará desde las características básicas hasta las avanzadas de Pandas, asegurándote de que puedas manejar diversas tareas de manipulación de datos de manera eficiente.

Introducción a Pandas

  • ¿Qué es Pandas?

    • Pandas es una biblioteca de análisis y manipulación de datos de código abierto para Python.
    • Proporciona estructuras de datos como Series y DataFrame.

  • Instalando Pandas

    !pip install pandas

  • Importando Pandas

    import pandas as pd

Estructuras de Datos de Pandas

Series

  • Creando una Serie

    import pandas as pd

data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

  • Serie con Índice Personalizado

    data = [1, 2, 3, 4, 5]
index = ['a', 'b', 'c', 'd', 'e']
series = pd.Series(data, index=index)
print(series)

DataFrame

  • Creando un DataFrame

    data = {
    'Nombre': ['Alice', 'Bob', 'Charlie'],
    'Edad': [25, 30, 35],
    'Ciudad': ['Nueva York', 'Los Ángeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)

  • Leyendo Datos desde un Archivo CSV

    df = pd.read_csv('data.csv')
print(df.head())

Manipulación de Datos con Pandas

Selección de Datos

  • Seleccionando Columnas

    # Columna única
print(df['Nombre'])

# Múltiples columnas
print(df[['Nombre', 'Edad']])

  • Seleccionando Filas

    # Por índice
print(df.iloc[0])

# Por etiqueta
print(df.loc[0])

Filtrado de Datos

  • Usando Condiciones 
    # Filtrar filas donde Edad > 30
filtered_df = df[df['Edad'] > 30]
print(filtered_df)

Añadiendo y Modificando Columnas

  • Añadiendo una Nueva Columna

    df['Salario'] = [50000, 60000, 70000]
print(df)

  • Modificando una Columna Existente

    df['Edad'] = df['Edad'] + 1
print(df)

Manejo de Datos Faltantes

  • Comprobando Datos Faltantes

    print(df.isnull())
print(df.isnull().sum())

  • Rellenando Datos Faltantes

    df['Edad'].fillna(df['Edad'].mean(), inplace=True)
print(df)

  • Eliminando Datos Faltantes

    df.dropna(inplace=True)
print(df)

Agrupación y Agregación de Datos

  • Agrupando Datos

    grouped = df.groupby('Ciudad')
print(grouped.mean())

  • Agregando Datos

    agg_df = df.groupby('Ciudad').agg({'Edad': 'mean', 'Salario': 'sum'})
print(agg_df)

Fusionando y Uniendo DataFrames

  • Fusionando DataFrames

    df1 = pd.DataFrame({
    'Nombre': ['Alice', 'Bob'],
    'Edad': [25, 30]
})

df2 = pd.DataFrame({
    'Nombre': ['Alice', 'Bob'],
    'Salario': [50000, 60000]
})

merged_df = pd.merge(df1, df2, on='Nombre')
print(merged_df)

  • Uniendo DataFrames

    df1 = df1.set_index('Nombre')
df2 = df2.set_index('Nombre')

joined_df = df1.join(df2)
print(joined_df)

Técnicas Avanzadas de Pandas

Tablas Dinámicas

  • Creando una Tabla Dinámica 
    pivot_table = df.pivot_table(values='Salario', index='Ciudad', columns='Edad', aggfunc='mean')
print(pivot_table)

Análisis de Series Temporales

  • Manejo de Datos de Fecha y Hora

    df['Fecha'] = pd.to_datetime(df['Fecha'])
print(df)

  • Re-muestreo de Datos de Series Temporales

    ts = df.set_index('Fecha')
resampled_ts = ts.resample('M').mean()
print(resampled_ts)

Conclusión

Pandas es una herramienta esencial para la manipulación de datos en Python. Proporciona estructuras de datos y funciones poderosas que hacen que las tareas de análisis de datos sean sencillas y eficientes. Al dominar Pandas, puedes manejar una amplia gama de tareas de manipulación de datos, desde la limpieza simple de datos hasta transformaciones y agregaciones complejas de datos. Sigue practicando con conjuntos de datos reales para mejorar tus habilidades y convertirte en un experto en el uso de Pandas para la manipulación de datos.
