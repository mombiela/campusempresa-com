Tema 7-1: Decoradores

Introducción a los Decoradores

Los decoradores en Python son una herramienta poderosa que permite modificar el comportamiento de una función o método. Los decoradores son funciones que envuelven otra función, permitiendo ejecutar código antes y después de la función decorada sin modificar su estructura.

Conceptos Clave

  1. Función Decoradora: Una función que toma otra función como argumento y devuelve una nueva función.
  2. Función Decorada: La función original que es modificada por el decorador.
  3. Sintaxis de Decoradores: Utiliza el símbolo @ seguido del nombre del decorador antes de la definición de la función a decorar.

Ejemplo Básico de Decorador

Vamos a crear un decorador simple que imprime un mensaje antes y después de ejecutar una función.

def mi_decorador(func):
    def envoltura():
        print("Algo se está haciendo antes de la función.")
        func()
        print("Algo se está haciendo después de la función.")
    return envoltura

@mi_decorador
def saludar():
    print("¡Hola!")

saludar()

Explicación del Código

  1. Definición del Decorador:

    def mi_decorador(func):
    def envoltura():
        print("Algo se está haciendo antes de la función.")
        func()
        print("Algo se está haciendo después de la función.")
    return envoltura
    • mi_decorador es la función decoradora que toma una función func como argumento.
    • envoltura es una función interna que envuelve la función func.
    • print se usa para mostrar mensajes antes y después de llamar a func.

  2. Uso del Decorador:

    @mi_decorador
def saludar():
    print("¡Hola!")
    • @mi_decorador aplica el decorador mi_decorador a la función saludar.

  3. Llamada a la Función Decorada:

    saludar()
    • Al llamar a saludar(), se ejecuta la función envoltura del decorador, que a su vez llama a saludar.

Decoradores con Argumentos

Los decoradores también pueden aceptar argumentos. Para ello, se necesita una función adicional que acepte los argumentos del decorador y devuelva la función decoradora.

def decorador_con_argumentos(arg1, arg2):
    def mi_decorador(func):
        def envoltura(*args, **kwargs):
            print(f"Argumentos del decorador: {arg1}, {arg2}")
            return func(*args, **kwargs)
        return envoltura
    return mi_decorador

@decorador_con_argumentos("Hola", "Mundo")
def saludar(nombre):
    print(f"¡Hola, {nombre}!")

saludar("Python")

Explicación del Código

  1. Definición del Decorador con Argumentos:

    def decorador_con_argumentos(arg1, arg2):
    def mi_decorador(func):
        def envoltura(*args, **kwargs):
            print(f"Argumentos del decorador: {arg1}, {arg2}")
            return func(*args, **kwargs)
        return envoltura
    return mi_decorador
    • decorador_con_argumentos es una función que acepta argumentos arg1 y arg2.
    • Devuelve mi_decorador, que es el decorador real.
    • envoltura es la función interna que envuelve la función func y puede aceptar cualquier número de argumentos (*args, **kwargs).

  2. Uso del Decorador con Argumentos:

    @decorador_con_argumentos("Hola", "Mundo")
def saludar(nombre):
    print(f"¡Hola, {nombre}!")
    • @decorador_con_argumentos("Hola", "Mundo") aplica el decorador con los argumentos especificados a la función saludar.

  3. Llamada a la Función Decorada:

    saludar("Python")
    • Al llamar a saludar("Python"), se ejecuta la función envoltura del decorador, que a su vez llama a saludar.

Ejercicio Práctico

Ejercicio: Crea un decorador llamado tiempo_ejecucion que mida y muestre el tiempo que tarda en ejecutarse una función.

import time

def tiempo_ejecucion(func):
    def envoltura(*args, **kwargs):
        inicio = time.time()
        resultado = func(*args, **kwargs)
        fin = time.time()
        print(f"Tiempo de ejecución: {fin - inicio} segundos")
        return resultado
    return envoltura

@tiempo_ejecucion
def suma(a, b):
    time.sleep(1)  # Simula una operación que tarda 1 segundo
    return a + b

print(suma(5, 3))

Solución

  1. Definición del Decorador:

    import time

def tiempo_ejecucion(func):
    def envoltura(*args, **kwargs):
        inicio = time.time()
        resultado = func(*args, **kwargs)
        fin = time.time()
        print(f"Tiempo de ejecución: {fin - inicio} segundos")
        return resultado
    return envoltura

  2. Uso del Decorador:

    @tiempo_ejecucion
def suma(a, b):
    time.sleep(1)  # Simula una operación que tarda 1 segundo
    return a + b

  3. Llamada a la Función Decorada:

    print(suma(5, 3))

Resumen

En esta sección, hemos aprendido sobre los decoradores en Python, cómo se definen y utilizan, y cómo pueden aceptar argumentos. Los decoradores son una herramienta poderosa para modificar el comportamiento de las funciones de manera limpia y reutilizable. En el siguiente tema, exploraremos los generadores, otra característica avanzada de Python que permite trabajar con secuencias de datos de manera eficiente.
