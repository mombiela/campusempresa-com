Introducción a los Decoradores

Los decoradores son una herramienta poderosa y útil en Python que te permite modificar el comportamiento de una función o una clase. A menudo se utilizan para agregar funcionalidad al código existente de una manera limpia y legible.

  • Definición: Un decorador es una función que toma otra función y extiende su comportamiento sin modificarla explícitamente.
  • Sintaxis: Los decoradores generalmente se definen con la sintaxis @nombre_del_decorador encima de la función a decorar.

Ejemplo Básico de Decorador

Comencemos con un ejemplo simple para entender cómo funcionan los decoradores.

def mi_decorador(func):
    def envoltura():
        print("Algo está sucediendo antes de que se llame a la función.")
        func()
        print("Algo está sucediendo después de que se llame a la función.")
    return envoltura

@mi_decorador
def decir_hola():
    print("¡Hola!")

decir_hola()

Explicación

  • mi_decorador: Esta es la función decoradora. Toma otra función func como argumento.
  • envoltura: Esta es una función interna que agrega algún comportamiento antes y después de llamar a func.
  • @mi_decorador: Esta sintaxis aplica el decorador a la función decir_hola.
  • decir_hola(): Cuando se llama a esta función, en realidad llama a la función envoltura dentro del decorador.

Uso de Decoradores con Argumentos

Los decoradores también pueden aceptar argumentos. Esto es útil cuando necesitas pasar alguna configuración al decorador.

def repetir(num_veces):
    def decorador_repetir(func):
        def envoltura(*args, **kwargs):
            for _ in range(num_veces):
                func(*args, **kwargs)
        return envoltura
    return decorador_repetir

@repetir(num_veces=3)
def saludar(nombre):
    print(f"¡Hola, {nombre}!")

saludar("Alicia")

Explicación

  • repetir(num_veces): Esta es una fábrica de decoradores que toma un argumento y devuelve un decorador.
  • decorador_repetir(func): Este es el decorador real.
  • **envoltura(*args, kwargs): Esta función interna llama a la función original num_veces veces.
  • @repetir(num_veces=3): Esto aplica el decorador con el argumento 3 a la función saludar.

Decoradores Incorporados

Python proporciona varios decoradores incorporados, como @staticmethod, @classmethod y @property.

@staticmethod y @classmethod

Estos decoradores se utilizan para definir métodos dentro de una clase que no están conectados a una instancia particular de la clase.

class MiClase:
    @staticmethod
    def metodo_estatico():
        print("Este es un método estático.")

    @classmethod
    def metodo_de_clase(cls):
        print(f"Este es un método de clase de {cls}.")

MiClase.metodo_estatico()
MiClase.metodo_de_clase()

@property

Este decorador se utiliza para definir un método como una propiedad, que puede ser accedida como un atributo.

class Circulo:
    def __init__(self, radio):
        self._radio = radio

    @property
    def radio(self):
        return self._radio

    @radio.setter
    def radio(self, valor):
        if valor < 0:
            raise ValueError("El radio no puede ser negativo")
        self._radio = valor

c = Circulo(5)
print(c.radio)  # Accediendo a la propiedad
c.radio = 10    # Estableciendo la propiedad

Decoradores Avanzados

Los decoradores también pueden ser utilizados para decorar clases y métodos dentro de clases.

Decoradores de Clase

Los decoradores de clase son similares a los decoradores de función pero se aplican a clases.

def agregar_repr(cls):
    cls.__repr__ = lambda self: f"{self.__class__.__name__}({self.__dict__})"
    return cls

@agregar_repr
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

p = Persona("Alicia", 30)
print(p)

Decoradores de Método

También puedes usar decoradores para modificar métodos dentro de una clase.

def registrar_llamada_metodo(metodo):
    def envoltura(self, *args, **kwargs):
        print(f"Llamando al método {metodo.__name__}")
        return metodo(self, *args, **kwargs)
    return envoltura

class MiClase:
    @registrar_llamada_metodo
    def mi_metodo(self):
        print("Se ha llamado al método")

obj = MiClase()
obj.mi_metodo()

Conclusión

Los decoradores son una característica versátil y poderosa en Python que te permite modificar el comportamiento de funciones y clases de una manera limpia y legible. Pueden ser utilizados para una variedad de propósitos, como registro, control de acceso, instrumentación, almacenamiento en caché y más. Entender cómo crear y usar decoradores de manera efectiva puede mejorar enormemente tus habilidades de programación en Python.
