3-1: Definición y Llamada de Funciones

Introducción

Las funciones son bloques de código reutilizables que realizan una tarea específica. En Python, las funciones se definen usando la palabra clave def, seguida del nombre de la función y paréntesis (). Las funciones pueden aceptar parámetros y devolver valores.

Objetivos

  • Comprender cómo definir y llamar funciones en Python.
  • Aprender a pasar argumentos a las funciones.
  • Conocer la importancia de las funciones para la modularidad y la reutilización del código.

1. Definición de Funciones

Para definir una función en Python, se utiliza la palabra clave def seguida del nombre de la función, paréntesis () y dos puntos :. El cuerpo de la función se indenta.

def nombre_de_la_funcion():
    # Cuerpo de la función
    print("Hola desde la función")

2. Llamada de Funciones

Para llamar a una función, simplemente se escribe su nombre seguido de paréntesis ().

nombre_de_la_funcion()  # Salida: Hola desde la función

3. Parámetros y Argumentos

Las funciones pueden aceptar parámetros, que son variables que se pasan a la función. Los valores que se pasan a estos parámetros se llaman argumentos.

def saludar(nombre):
    print(f"Hola, {nombre}")

saludar("Juan")  # Salida: Hola, Juan

4. Valores de Retorno

Las funciones pueden devolver valores usando la palabra clave return.

def sumar(a, b):
    return a + b

resultado = sumar(3, 5)
print(resultado)  # Salida: 8

5. Ejemplos Prácticos

Ejemplo 1: Función sin Parámetros

def mostrar_mensaje():
    print("Este es un mensaje desde una función sin parámetros")

mostrar_mensaje()  # Salida: Este es un mensaje desde una función sin parámetros

Ejemplo 2: Función con Parámetros

def multiplicar(x, y):
    return x * y

producto = multiplicar(4, 5)
print(producto)  # Salida: 20

Ejemplo 3: Función con Valor de Retorno

def es_par(numero):
    if numero % 2 == 0:
        return True
    else:
        return False

print(es_par(4))  # Salida: True
print(es_par(7))  # Salida: False

6. Ejercicios Prácticos

Ejercicio 1: Función de Suma

Instrucciones: Define una función llamada suma que tome dos números como parámetros y devuelva su suma.

def suma(a, b):
    return a + b

# Prueba tu función
print(suma(10, 5))  # Salida esperada: 15
print(suma(-3, 7))  # Salida esperada: 4

Ejercicio 2: Función de Factorial

Instrucciones: Define una función llamada factorial que tome un número como parámetro y devuelva su factorial.

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

# Prueba tu función
print(factorial(5))  # Salida esperada: 120
print(factorial(0))  # Salida esperada: 1

Ejercicio 3: Función de Conversión de Temperatura

Instrucciones: Define una función llamada celsius_a_fahrenheit que convierta una temperatura de grados Celsius a Fahrenheit.

def celsius_a_fahrenheit(celsius):
    return (celsius * 9/5) + 32

# Prueba tu función
print(celsius_a_fahrenheit(0))    # Salida esperada: 32.0
print(celsius_a_fahrenheit(100))  # Salida esperada: 212.0

Retroalimentación sobre Errores Comunes

  • Indentación Incorrecta: Asegúrate de que el cuerpo de la función esté correctamente indentado.
  • Olvidar los Paréntesis al Llamar a la Función: Recuerda siempre usar paréntesis () al llamar a una función.
  • No Devolver un Valor: Si tu función debe devolver un valor, asegúrate de usar la palabra clave return.

Conclusión

En esta sección, hemos aprendido cómo definir y llamar funciones en Python, cómo pasar parámetros y argumentos, y cómo devolver valores desde una función. Las funciones son fundamentales para la modularidad y la reutilización del código, y son una herramienta esencial en cualquier lenguaje de programación.
