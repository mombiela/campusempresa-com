Tema 10-2: Técnicas de Depuración

Introducción

La depuración es una parte esencial del desarrollo de software. Consiste en identificar y corregir errores en el código. En este tema, exploraremos diversas técnicas de depuración que te ayudarán a encontrar y solucionar problemas en tus programas de Python.

Contenido

  2. Uso de Herramientas de Depuración Integradas
  3. Depuración con pdb

1. Impresión de Mensajes de Depuración

Una técnica básica pero efectiva es usar la función print() para mostrar el valor de variables en diferentes puntos del programa.

Ejemplo:

def suma(a, b):
    print(f"Valores recibidos: a={a}, b={b}")
    resultado = a + b
    print(f"Resultado de la suma: {resultado}")
    return resultado

suma(3, 5)

Explicación:

  • print(f"Valores recibidos: a={a}, b={b}"): Muestra los valores de a y b antes de realizar la suma.
  • print(f"Resultado de la suma: {resultado}"): Muestra el resultado de la suma antes de devolverlo.

2. Uso de Herramientas de Depuración Integradas

Python incluye varias herramientas integradas que pueden ayudarte a depurar tu código, como assert y logging.

Uso de assert:

def dividir(a, b):
    assert b != 0, "El divisor no puede ser cero"
    return a / b

dividir(10, 0)

Explicación:

  • assert b != 0, "El divisor no puede ser cero": Verifica que b no sea cero antes de realizar la división. Si b es cero, se lanza una excepción con el mensaje "El divisor no puede ser cero".

Uso de logging:

import logging

logging.basicConfig(level=logging.DEBUG)

def multiplicar(a, b):
    logging.debug(f"Multiplicando {a} por {b}")
    resultado = a * b
    logging.debug(f"Resultado: {resultado}")
    return resultado

multiplicar(4, 5)

Explicación:

  • logging.basicConfig(level=logging.DEBUG): Configura el nivel de registro para mostrar mensajes de depuración.
  • logging.debug(f"Multiplicando {a} por {b}"): Registra un mensaje de depuración antes de realizar la multiplicación.
  • logging.debug(f"Resultado: {resultado}"): Registra el resultado de la multiplicación.

3. Depuración con pdb

El módulo pdb es el depurador interactivo de Python. Permite pausar la ejecución del programa y examinar el estado del programa.

Ejemplo:

import pdb

def resta(a, b):
    pdb.set_trace()
    resultado = a - b
    return resultado

resta(10, 5)

Explicación:

  • pdb.set_trace(): Pausa la ejecución del programa y abre una sesión interactiva de depuración en la línea donde se llama.

4. Depuración en Entornos de Desarrollo Integrados (IDEs)

Los IDEs modernos como PyCharm, VSCode y otros, ofrecen herramientas de depuración avanzadas que permiten:

  • Puntos de interrupción (breakpoints): Pausar la ejecución del programa en una línea específica.
  • Inspección de variables: Ver el valor de las variables en el momento de la pausa.
  • Ejecución paso a paso: Ejecutar el programa línea por línea para observar el flujo de ejecución.

Ejemplo en PyCharm:

  1. Establecer un punto de interrupción: Haz clic en el margen izquierdo junto a la línea de código donde deseas pausar la ejecución.
  2. Iniciar la depuración: Haz clic en el botón de depuración (icono de insecto) en la barra de herramientas.
  3. Inspeccionar variables: Usa la ventana de variables para ver los valores actuales.
  4. Ejecución paso a paso: Usa los botones de "Step Over", "Step Into" y "Step Out" para controlar la ejecución.

5. Errores Comunes y Cómo Evitarlos

Errores Comunes:

  • Errores de sintaxis: Faltan paréntesis, comillas, dos puntos, etc.
  • Errores de tipo: Operar con tipos de datos incompatibles.
  • Errores de lógica: El programa se ejecuta sin errores, pero no produce el resultado esperado.

Cómo Evitarlos:

  • Revisar el código: Lee tu código cuidadosamente para detectar errores de sintaxis.
  • Pruebas unitarias: Escribe pruebas para verificar que cada parte de tu código funciona correctamente.
  • Comentarios y documentación: Documenta tu código para que sea más fácil de entender y mantener.

Ejercicio Práctico

Ejercicio:

Corrige el siguiente código que contiene errores:

def calcular_area_rectangulo(base, altura):
    area = base * altura
    return area

print(calcular_area_rectangulo(5, '10'))

Solución:

def calcular_area_rectangulo(base, altura):
    assert isinstance(base, (int, float)), "La base debe ser un número"
    assert isinstance(altura, (int, float)), "La altura debe ser un número"
    area = base * altura
    return area

print(calcular_area_rectangulo(5, 10))

Explicación:

  • assert isinstance(base, (int, float)), "La base debe ser un número": Verifica que base sea un número.
  • assert isinstance(altura, (int, float)), "La altura debe ser un número": Verifica que altura sea un número.

Conclusión

En este tema, hemos explorado diversas técnicas de depuración que puedes usar para identificar y corregir errores en tus programas de Python. Desde la impresión de mensajes de depuración hasta el uso de herramientas avanzadas en IDEs, estas técnicas te ayudarán a mejorar la calidad de tu código y a resolver problemas de manera más eficiente.
