3-4: Módulos y Paquetes

Introducción

En Python, los módulos y paquetes son fundamentales para organizar y reutilizar el código. Un módulo es simplemente un archivo que contiene definiciones y declaraciones de Python, mientras que un paquete es una colección de módulos organizados en un directorio con una estructura jerárquica.

Objetivos

  • Comprender qué es un módulo y cómo se utiliza.
  • Aprender a crear y usar paquetes.
  • Conocer las diferencias entre módulos y paquetes.
  • Aprender a importar módulos y paquetes en un programa Python.

1. Módulos

Definición de un Módulo

Un módulo es un archivo que contiene definiciones y declaraciones de Python. El nombre del archivo es el nombre del módulo con la extensión .py.

Creación de un Módulo

Para crear un módulo, simplemente crea un archivo .py con las funciones y variables que deseas incluir.

# archivo: mi_modulo.py

def saludar(nombre):
    return f"Hola, {nombre}!"

PI = 3.14159

Importación de Módulos

Para usar un módulo en otro archivo, utilizamos la palabra clave import.

# archivo: main.py

import mi_modulo

print(mi_modulo.saludar("Mundo"))
print(f"El valor de PI es: {mi_modulo.PI}")

Uso de Funciones y Variables de un Módulo

Una vez importado el módulo, puedes acceder a sus funciones y variables utilizando la notación de punto (.).

# archivo: main.py

import mi_modulo

mensaje = mi_modulo.saludar("Estudiante")
print(mensaje)

area = mi_modulo.PI * (5 ** 2)
print(f"El área de un círculo con radio 5 es: {area}")

2. Paquetes

Definición de un Paquete

Un paquete es una colección de módulos organizados en un directorio con una estructura jerárquica. Los paquetes permiten organizar los módulos de manera lógica y estructurada.

Estructura de un Paquete

Un paquete es un directorio que contiene un archivo especial __init__.py y uno o más módulos.

mi_paquete/
    __init__.py
    modulo1.py
    modulo2.py

Creación de un Paquete

  1. Crea un directorio para el paquete.
  2. Añade un archivo __init__.py (puede estar vacío).
  3. Añade los módulos al directorio del paquete.
# archivo: mi_paquete/__init__.py
# Este archivo puede estar vacío o contener código de inicialización del paquete.

# archivo: mi_paquete/modulo1.py

def funcion_modulo1():
    return "Función del módulo 1"

# archivo: mi_paquete/modulo2.py

def funcion_modulo2():
    return "Función del módulo 2"

Importación de Paquetes

Para usar un paquete, importamos los módulos que contiene.

# archivo: main.py

from mi_paquete import modulo1, modulo2

print(modulo1.funcion_modulo1())
print(modulo2.funcion_modulo2())

3. Importación Avanzada

Importación Relativa y Absoluta

  • Importación Absoluta: Importa un módulo utilizando su ruta completa desde el directorio raíz del proyecto.
  • Importación Relativa: Importa un módulo utilizando una ruta relativa al módulo actual.
# archivo: mi_paquete/modulo1.py

from . import modulo2  # Importación relativa

def funcion_modulo1():
    return modulo2.funcion_modulo2()

Importación Selectiva

Puedes importar solo partes específicas de un módulo.

# archivo: main.py

from mi_modulo import saludar, PI

print(saludar("Mundo"))
print(f"El valor de PI es: {PI}")

Uso de __init__.py

El archivo __init__.py se ejecuta cuando se importa el paquete y puede contener código de inicialización del paquete.

# archivo: mi_paquete/__init__.py

print("Inicializando el paquete mi_paquete")

from .modulo1 import funcion_modulo1
from .modulo2 import funcion_modulo2

Ejercicio Práctico

Ejercicio 1: Crear y Usar un Módulo

  1. Crea un archivo llamado matematica.py que contenga las siguientes funciones:

    • suma(a, b): Devuelve la suma de a y b.
    • resta(a, b): Devuelve la resta de a y b.

  2. Crea un archivo llamado main.py que importe el módulo matematica y use las funciones suma y resta.

Solución:

# archivo: matematica.py

def suma(a, b):
    return a + b

def resta(a, b):
    return a - b
# archivo: main.py

import matematica

resultado_suma = matematica.suma(10, 5)
resultado_resta = matematica.resta(10, 5)

print(f"Suma: {resultado_suma}")
print(f"Resta: {resultado_resta}")

Conclusión
