Tema 3-2: Argumentos de Función y Valores de Retorno

Introducción

En este tema, aprenderemos sobre los argumentos de función y los valores de retorno en Python. Estos conceptos son fundamentales para escribir funciones reutilizables y modulares. Exploraremos cómo pasar datos a las funciones y cómo devolver resultados desde ellas.

1. Argumentos de Función

Argumentos Posicionales

Los argumentos posicionales son aquellos que se pasan a la función en el orden en que se definen.

def greet(name, age):
    print(f"Hola, mi nombre es {name} y tengo {age} años.")

greet("Ana", 25)

Argumentos por Nombre (Keyword Arguments)

Los argumentos por nombre se pasan a la función utilizando el nombre del parámetro, lo que permite especificar los argumentos en cualquier orden.

def greet(name, age):
    print(f"Hola, mi nombre es {name} y tengo {age} años.")

greet(age=25, name="Ana")

Argumentos por Defecto

Los argumentos por defecto permiten definir valores predeterminados para los parámetros. Si no se proporciona un valor, se utiliza el valor por defecto.

def greet(name, age=30):
    print(f"Hola, mi nombre es {name} y tengo {age} años.")

greet("Ana")
greet("Carlos", 25)

Argumentos Arbitrarios

Python permite definir funciones que aceptan un número arbitrario de argumentos utilizando *args y **kwargs.

  • *args para argumentos posicionales arbitrarios.
  • **kwargs para argumentos por nombre arbitrarios.
def greet(*names):
    for name in names:
        print(f"Hola, {name}!")

greet("Ana", "Carlos", "Luis")

def greet(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

greet(name="Ana", age=25, city="Madrid")

2. Valores de Retorno

Uso de la Palabra Clave return

La palabra clave return se utiliza para devolver un valor desde una función.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Salida: 8

Devolución de Múltiples Valores

Python permite devolver múltiples valores desde una función utilizando tuplas.

def get_person_info():
    name = "Ana"
    age = 25
    return name, age

name, age = get_person_info()
print(name)  # Salida: Ana
print(age)   # Salida: 25

3. Ejemplos Prácticos

Ejemplo 1: Función con Argumentos y Valor de Retorno

def calculate_area(length, width):
    area = length * width
    return area

length = 5
width = 3
area = calculate_area(length, width)
print(f"El área es {area}")  # Salida: El área es 15

Ejemplo 2: Función con Argumentos por Defecto y Múltiples Valores de Retorno

def get_full_name(first_name, last_name="Doe"):
    full_name = f"{first_name} {last_name}"
    return full_name, len(full_name)

name, length = get_full_name("Ana")
print(f"Nombre completo: {name}, Longitud: {length}")  # Salida: Nombre completo: Ana Doe, Longitud: 7

Ejercicio 1: Función de Suma con Argumentos Arbitrarios

Escribe una función sum_all que acepte un número arbitrario de argumentos y devuelva la suma de todos ellos.

def sum_all(*args):
    # Tu código aquí
    pass

# Prueba tu función
print(sum_all(1, 2, 3, 4))  # Salida esperada: 10
print(sum_all(5, 10, 15))   # Salida esperada: 30

Ejercicio 2: Función con Argumentos por Defecto

Escribe una función introduce que acepte un nombre y una edad (con un valor por defecto de 30) y devuelva una cadena de presentación.

def introduce(name, age=30):
    # Tu código aquí
    pass

# Prueba tu función
print(introduce("Ana"))         # Salida esperada: Hola, mi nombre es Ana y tengo 30 años.

5. Resumen

En este tema, hemos aprendido sobre los diferentes tipos de argumentos de función en Python, incluyendo argumentos posicionales, por nombre, por defecto y arbitrarios. También hemos explorado cómo devolver valores desde una función, incluyendo la devolución de múltiples valores. Estos conceptos son esenciales para escribir funciones flexibles y reutilizables.

Soluciones a los Ejercicios

Ejercicio 1: Solución

def sum_all(*args):
    return sum(args)

# Prueba tu función
print(sum_all(1, 2, 3, 4))  # Salida: 10
print(sum_all(5, 10, 15))   # Salida: 30

Ejercicio 2: Solución

def introduce(name, age=30):
    return f"Hola, mi nombre es {name} y tengo {age} años."

# Prueba tu función
print(introduce("Ana"))         # Salida: Hola, mi nombre es Ana y tengo 30 años.
print(introduce("Carlos", 25))  # Salida: Hola, mi nombre es Carlos y tengo 25 años.
