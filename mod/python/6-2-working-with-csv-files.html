6-2: Trabajo con Archivos CSV

Introducción

Los archivos CSV (Comma Separated Values) son un formato común para almacenar datos tabulares. Python proporciona varias formas de trabajar con archivos CSV, siendo la biblioteca csv una de las más utilizadas. En esta lección, aprenderemos cómo leer y escribir archivos CSV utilizando esta biblioteca.

Objetivos

  • Comprender el formato de los archivos CSV.
  • Leer archivos CSV utilizando la biblioteca csv.
  • Escribir datos en archivos CSV.
  • Manejar diferentes delimitadores y encabezados en archivos CSV.

1. Formato de Archivos CSV

Un archivo CSV es un archivo de texto plano que utiliza un delimitador (por defecto, una coma) para separar los valores. Cada línea del archivo representa una fila de datos, y cada valor separado por el delimitador representa una columna.

Ejemplo de archivo CSV:

nombre,edad,ciudad
Juan,28,Madrid
Ana,22,Barcelona
Luis,35,Valencia

2. Lectura de Archivos CSV

Para leer archivos CSV en Python, utilizamos la función csv.reader de la biblioteca csv.

Ejemplo de lectura de un archivo CSV:

import csv

# Abrir el archivo CSV
with open('datos.csv', mode='r', newline='') as archivo:
    lector_csv = csv.reader(archivo)
    
    # Leer cada fila del archivo CSV
    for fila in lector_csv:
        print(fila)

Explicación:

  • open('datos.csv', mode='r', newline=''): Abre el archivo datos.csv en modo lectura ('r').
  • csv.reader(archivo): Crea un objeto lector CSV.
  • for fila in lector_csv: Itera sobre cada fila del archivo CSV.

3. Escritura de Archivos CSV

Para escribir datos en un archivo CSV, utilizamos la función csv.writer.

Ejemplo de escritura en un archivo CSV:

import csv

# Datos a escribir en el archivo CSV
datos = [
    ['nombre', 'edad', 'ciudad'],
    ['Juan', 28, 'Madrid'],
    ['Ana', 22, 'Barcelona'],
    ['Luis', 35, 'Valencia']
]

# Abrir el archivo CSV en modo escritura
with open('datos.csv', mode='w', newline='') as archivo:
    escritor_csv = csv.writer(archivo)
    
    # Escribir cada fila en el archivo CSV
    escritor_csv.writerows(datos)

Explicación:

  • open('datos.csv', mode='w', newline=''): Abre el archivo datos.csv en modo escritura ('w').
  • csv.writer(archivo): Crea un objeto escritor CSV.
  • escritor_csv.writerows(datos): Escribe todas las filas en el archivo CSV.

4. Manejo de Delimitadores y Encabezados

Delimitadores

El delimitador por defecto en un archivo CSV es la coma, pero podemos especificar otros delimitadores como el punto y coma (;).

Ejemplo de lectura con un delimitador diferente:

import csv

# Abrir el archivo CSV con punto y coma como delimitador
with open('datos.csv', mode='r', newline='') as archivo:
    lector_csv = csv.reader(archivo, delimiter=';')
    
    for fila in lector_csv:
        print(fila)

Encabezados

A menudo, los archivos CSV contienen una fila de encabezado. Podemos utilizar la clase csv.DictReader para leer archivos CSV en forma de diccionarios, donde las claves son los encabezados.

Ejemplo de lectura de un archivo CSV con encabezados:

import csv

# Abrir el archivo CSV
with open('datos.csv', mode='r', newline='') as archivo:
    lector_csv = csv.DictReader(archivo)
    
    for fila in lector_csv:
        print(fila)

Explicación:

  • csv.DictReader(archivo): Crea un objeto lector CSV que interpreta la primera fila como encabezados.

5. Ejercicios Prácticos

Ejercicio 1: Lectura de un Archivo CSV

Instrucciones:

  1. Crea un archivo CSV llamado empleados.csv con el siguiente contenido:

    nombre,puesto,salario
Carlos,Ingeniero,50000
Marta,Analista,45000
Pedro,Gerente,60000

  2. Escribe un programa en Python que lea el archivo empleados.csv y muestre cada fila en la consola.

Solución:

import csv

with open('empleados.csv', mode='r', newline='') as archivo:
    lector_csv = csv.reader(archivo)
    
    for fila in lector_csv:
        print(fila)

Ejercicio 2: Escritura en un Archivo CSV

Instrucciones:

  1. Crea una lista de listas con los siguientes datos:

    datos = [
    ['producto', 'precio', 'cantidad'],
    ['Manzanas', 1.2, 10],
    ['Naranjas', 0.8, 20],
    ['Plátanos', 1.1, 15]
]

  2. Escribe un programa en Python que escriba estos datos en un archivo CSV llamado inventario.csv.

Solución:

import csv

datos = [
    ['producto', 'precio', 'cantidad'],
    ['Manzanas', 1.2, 10],
    ['Naranjas', 0.8, 20],
    ['Plátanos', 1.1, 15]
]

with open('inventario.csv', mode='w', newline='') as archivo:
    escritor_csv = csv.writer(archivo)
    escritor_csv.writerows(datos)

Conclusión

En esta lección, hemos aprendido cómo trabajar con archivos CSV en Python utilizando la biblioteca csv. Hemos cubierto cómo leer y escribir archivos CSV, así como cómo manejar diferentes delimitadores y encabezados. Estos conocimientos son fundamentales para trabajar con datos tabulares en Python.

En la próxima lección, exploraremos el manejo de excepciones en Python, lo que nos permitirá escribir programas más robustos y manejables.
