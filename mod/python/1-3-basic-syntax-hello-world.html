Tema 1-3: Sintaxis Básica y Hola Mundo

Objetivos del Tema

  • Comprender la sintaxis básica de Python.
  • Escribir y ejecutar un programa simple en Python.
  • Familiarizarse con los comentarios y la indentación en Python.

Contenido

1. Sintaxis Básica de Python

Python es conocido por su sintaxis clara y legible. A continuación, se presentan algunos conceptos clave de la sintaxis básica de Python:

  1. Comentarios:

    • Los comentarios en Python comienzan con el símbolo # y se extienden hasta el final de la línea.
    • Los comentarios no son ejecutados por el intérprete de Python y se utilizan para explicar el código.
    # Este es un comentario en Python

  2. Indentación:

    • Python utiliza la indentación para definir bloques de código.
    • La indentación debe ser consistente dentro de un bloque de código.
    if True:
    print("Esto está dentro del bloque if")

  3. Declaración de Variables:

    • Las variables en Python se crean al asignarles un valor.
    • No es necesario declarar el tipo de variable explícitamente.
    x = 5
y = "Hola"

  4. Tipos de Datos Básicos:

    • Enteros (int): Números enteros.
    • Flotantes (float): Números decimales.
    • Cadenas (str): Secuencias de caracteres.
    • Booleanos (bool): Verdadero (True) o Falso (False).
    entero = 10
flotante = 10.5
cadena = "Hola, Mundo"
booleano = True

2. Escribiendo y Ejecutando un Programa Simple

Vamos a escribir nuestro primer programa en Python: "Hola Mundo".

  1. Escribiendo el Código:

    • Abre tu editor de texto o entorno de desarrollo integrado (IDE) preferido.
    • Escribe el siguiente código:
    # Este es nuestro primer programa en Python
print("Hola, Mundo")

  2. Ejecutando el Código:

    • Guarda el archivo con una extensión .py, por ejemplo, hola_mundo.py.
    • Abre una terminal o línea de comandos.
    • Navega hasta el directorio donde guardaste el archivo.
    • Ejecuta el programa usando el comando python seguido del nombre del archivo:
    python hola_mundo.py
    • Deberías ver la salida Hola, Mundo en la terminal.

3. Ejemplo Práctico

Vamos a ver un ejemplo más complejo que incluye comentarios, variables y la función print.

# Este programa calcula la suma de dos números

# Declaración de variables
numero1 = 10
numero2 = 20

# Calculando la suma
suma = numero1 + numero2

# Imprimiendo el resultado
print("La suma de", numero1, "y", numero2, "es", suma)

Explicación del Código:

  • Primero, declaramos dos variables numero1 y numero2 y les asignamos los valores 10 y 20, respectivamente.
  • Luego, calculamos la suma de estas dos variables y almacenamos el resultado en la variable suma.
  • Finalmente, usamos la función print para mostrar el resultado en la pantalla.

4. Ejercicio Práctico

Ejercicio: Escribe un programa en Python que haga lo siguiente:

  1. Declare dos variables a y b y les asigne los valores 15 y 30, respectivamente.
  2. Calcule la diferencia entre a y b.
  3. Imprima el resultado en la pantalla.

Solución:

# Declaración de variables
a = 15
b = 30

# Calculando la diferencia
diferencia = b - a

# Imprimiendo el resultado
print("La diferencia entre", b, "y", a, "es", diferencia)

Conclusión

En este tema, hemos aprendido sobre la sintaxis básica de Python, cómo escribir y ejecutar un programa simple, y cómo utilizar comentarios y la indentación. También hemos visto ejemplos prácticos y realizado un ejercicio para reforzar los conceptos aprendidos. En el próximo tema, exploraremos las variables y los tipos de datos en mayor profundidad.
