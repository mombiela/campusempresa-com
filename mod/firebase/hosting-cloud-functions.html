Introducción a Firebase Hosting

Firebase Hosting es una plataforma de alojamiento de contenido web de nivel de producción para desarrolladores. Te permite desplegar aplicaciones web y contenido estático de manera rápida y segura.

Conceptos Clave

  • Contenido Estático y Dinámico: Firebase Hosting soporta tanto archivos estáticos (HTML, CSS, JavaScript) como contenido dinámico a través de Cloud Functions.
  • SSL por Defecto: Todo el contenido se sirve a través de HTTPS, asegurando la seguridad.
  • CDN Global: Firebase Hosting utiliza una red de entrega de contenido global (CDN) para asegurar una entrega rápida del contenido.

Empezando con Firebase Hosting

  1. Instalar Firebase CLI: 
    npm install -g firebase-tools
  2. Inicializar tu Proyecto: 
    firebase init hosting
    • Sigue las indicaciones para configurar tu proyecto.
  3. Desplegar tu Sitio: 
    firebase deploy

Ejemplo: Desplegando una Página HTML Simple

  1. Crea un archivo index.html en el directorio de tu proyecto: 
    <!DOCTYPE html>
<html>
<head>
    <title>Mi Sitio Alojado en Firebase</title>
</head>
<body>
    <h1>¡Hola, Firebase Hosting!</h1>
</body>
</html>
  2. Inicializa Firebase Hosting: 
    firebase init hosting
  3. Despliega tu sitio: 
    firebase deploy

Introducción a Firebase Cloud Functions

Firebase Cloud Functions te permite ejecutar código backend en respuesta a eventos desencadenados por características de Firebase y solicitudes HTTPS.

Conceptos Clave

  • Funciones Basadas en Eventos: Funciones que se desencadenan por eventos de Firebase (por ejemplo, cambios en la base de datos, eventos de autenticación).
  • Funciones HTTPS: Funciones que pueden ser desencadenadas a través de solicitudes HTTP.
  • Escalabilidad: Escala automáticamente hacia arriba y hacia abajo según el número de solicitudes entrantes.

Configuración de Cloud Functions

  1. Inicializar Cloud Functions: 
    firebase init functions
    • Sigue las indicaciones para configurar tu proyecto.
  2. Escribe tu Primera Función: 
    const functions = require('firebase-functions');

// Crear y Desplegar tu Primera Cloud Function
exports.helloWorld = functions.https.onRequest((request, response) => {
    response.send("¡Hola desde Firebase!");
});
  3. Despliega tus Funciones: 
    firebase deploy --only functions

Ejemplo: Creando una Función HTTPS

  1. Inicializa Cloud Functions: 
    firebase init functions
  2. Escribe una simple función HTTPS en functions/index.js: 
    const functions = require('firebase-functions');

exports.helloWorld = functions.https.onRequest((request, response) => {
    response.send("¡Hola desde Firebase!");
});
  3. Despliega tu función: 
    firebase deploy --only functions

Temas Avanzados en Hosting y Cloud Functions

Dominios Personalizados

  • Configuración de Dominios Personalizados: Firebase Hosting te permite conectar tu dominio personalizado a tu proyecto de Firebase.
    • Ve a la Consola de Firebase > Hosting > Conectar Dominio.
    • Sigue las instrucciones para verificar tu dominio y actualizar los registros DNS.

Usando Cloud Functions con Firestore

  • Desencadenando Funciones en Eventos de Firestore: 
    const functions = require('firebase-functions');
const admin = require('firebase-admin');
admin.initializeApp();

exports.createUserProfile = functions.firestore
    .document('users/{userId}')
    .onCreate((snap, context) => {
        const newValue = snap.data();
        // Realizar operaciones con newValue
        return admin.firestore().collection('profiles').doc(context.params.userId).set(newValue);
    });

Combinando Hosting y Cloud Functions

  • Sirviendo Contenido Dinámico: Usa Cloud Functions para servir contenido dinámico junto con archivos estáticos. 
    const functions = require('firebase-functions');
const express = require('express');
const app = express();

app.get('/timestamp', (req, res) => {
    res.send(`Timestamp: ${Date.now()}`);
});

exports.app = functions.https.onRequest(app);

Conclusión

Firebase Hosting y Cloud Functions proporcionan una combinación poderosa para desplegar tanto aplicaciones web estáticas como dinámicas. Con Firebase Hosting, puedes desplegar contenido estático seguro y de carga rápida rápidamente, mientras que Cloud Functions te permite agregar lógica backend que escala automáticamente. Al dominar estas herramientas, puedes construir aplicaciones web robustas y escalables con facilidad.
