Firebase Test Lab proporciona una infraestructura basada en la nube para probar aplicaciones Android e iOS. Te permite ejecutar tu aplicación en una amplia gama de dispositivos y configuraciones, asegurando que tu aplicación funcione bien para todos los usuarios. Esta sección te guiará a través de la configuración de Firebase Test Lab, desde lo básico hasta configuraciones más avanzadas.

Introducción a Firebase Test Lab

  • ¿Qué es Firebase Test Lab?

    • Una infraestructura de pruebas de aplicaciones basada en la nube.
    • Soporta tanto plataformas Android como iOS.
    • Permite probar en una variedad de dispositivos y configuraciones.

  • ¿Por qué usar Firebase Test Lab?

    • Asegura la compatibilidad de la aplicación en diferentes dispositivos.
    • Identifica errores y problemas de rendimiento temprano.
    • Ahorra tiempo y recursos en comparación con las pruebas manuales.

Configuración de Firebase Test Lab

Requisitos previos

  • Proyecto de Firebase

    • Asegúrate de tener un proyecto de Firebase configurado. Si no, crea uno en el Firebase Console.

  • Proyecto de Google Cloud

    • Firebase Test Lab requiere un proyecto de Google Cloud. Vincula tu proyecto de Firebase a un proyecto de Google Cloud si aún no está vinculado.

Habilitando Firebase Test Lab

  1. Navegar a Firebase Console

    • Ve al Firebase Console.
    • Selecciona tu proyecto.

  2. Acceder a Test Lab

    • En el menú de la izquierda, haz clic en "Test Lab".
    • Si se te solicita, habilita la API de Test Lab.

Ejecutando tu primera prueba

  • Sube tu aplicación

    • Para Android: Sube tu archivo APK.
    • Para iOS: Sube tu archivo IPA.

  • Selecciona el tipo de prueba

    • Robo Test: Explora automáticamente tu aplicación.
    • Instrumentation Test: Ejecuta tus scripts de prueba personalizados.

  • Elige dispositivos

    • Selecciona los dispositivos y configuraciones en los que deseas probar. Firebase Test Lab proporciona una amplia gama de dispositivos con diferentes versiones de SO y tamaños de pantalla.

  • Ejecuta la prueba

    • Haz clic en "Run" para iniciar la prueba. Firebase Test Lab ejecutará las pruebas en los dispositivos seleccionados.

Visualización de resultados de pruebas

  • Accediendo a los resultados

    • Una vez que las pruebas estén completas, puedes ver los resultados en el Firebase Console bajo la sección "Test Lab".

  • Interpretando los resultados

    • Resumen de la prueba: Proporciona una visión general de los resultados de la prueba.
    • Registros y capturas de pantalla: Registros detallados y capturas de pantalla de las pruebas.
    • Videos: Grabaciones de video de las pruebas.

Ejemplo: Ejecutando una prueba Robo

// Ejemplo de una aplicación Android simple para ser probada
public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }
}
  1. Sube el APK: Sube el archivo APK de la aplicación a Firebase Test Lab.
  2. Selecciona Robo Test: Elige la opción Robo Test.
  3. Elige dispositivos: Selecciona una gama de dispositivos para probar.
  4. Ejecuta la prueba: Haz clic en "Run" y espera los resultados.

Configuraciones avanzadas

Pruebas de instrumentación personalizadas

  • Escribiendo pruebas de instrumentación

    • Usa frameworks como Espresso para Android o XCTest para iOS para escribir scripts de prueba personalizados.

  • Subiendo scripts de prueba

    • Sube tus scripts de prueba junto con tu aplicación a Firebase Test Lab.

Pruebas en paralelo

  • Ejecutando pruebas en paralelo
    • Firebase Test Lab te permite ejecutar pruebas en paralelo en múltiples dispositivos, reduciendo el tiempo total de prueba.

Integración con CI/CD

  • Integración Continua/Despliegue Continuo

    • Integra Firebase Test Lab con pipelines de CI/CD usando herramientas como Jenkins, CircleCI o GitHub Actions.

  • Ejemplo: Flujo de trabajo de GitHub Actions

name: Firebase Test Lab

on: [push]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
    - name: Checkout code
      uses: actions/checkout@v2
    - name: Set up JDK
      uses: actions/setup-java@v1
      with:
        java-version: '11'
    - name: Build APK
      run: ./gradlew assembleDebug
    - name: Run tests on Firebase Test Lab
      run: |
        gcloud firebase test android run \
          --type instrumentation \
          --app app/build/outputs/apk/debug/app-debug.apk \
          --test app/build/outputs/apk/androidTest/debug/app-debug-androidTest.apk
      env:
        GOOGLE_APPLICATION_CREDENTIALS: ${{ secrets.GOOGLE_APPLICATION_CREDENTIALS }}

Conclusión

Firebase Test Lab es una herramienta poderosa para asegurar que tu aplicación funcione sin problemas en una variedad de dispositivos y configuraciones. Siguiendo esta guía, puedes configurar y ejecutar pruebas en tu aplicación, ver resultados detallados e integrar las pruebas en tu pipeline de CI/CD. Esto te ayudará a detectar errores temprano, mejorar el rendimiento de la aplicación y ofrecer una mejor experiencia a tus usuarios.
