Introducción a Firebase Analytics

Firebase Analytics es una herramienta poderosa que te permite rastrear el comportamiento de los usuarios y obtener información sobre cómo interactúan con tu aplicación. Proporciona informes detallados y paneles para ayudarte a entender el compromiso del usuario, la retención y otras métricas críticas.

Configuración de Firebase Analytics

Antes de sumergirte en análisis avanzados, asegúrate de que Firebase Analytics esté correctamente configurado en tu proyecto.

Pasos para Configurar Firebase Analytics:

  1. Crear un Proyecto de Firebase:

    • Ve a la Consola de Firebase.
    • Haz clic en "Agregar proyecto" y sigue las instrucciones de configuración.

  2. Agregar Firebase a Tu Aplicación:

    • Para Android: Agrega el archivo google-services.json a tu proyecto.
    • Para iOS: Agrega el archivo GoogleService-Info.plist a tu proyecto.

  3. Agregar el SDK de Firebase:

    • Para Android: 
      dependencies {
    implementation platform('com.google.firebase:firebase-bom:28.4.1')
    implementation 'com.google.firebase:firebase-analytics'
}
    • Para iOS: 
      pod 'Firebase/Analytics'

  4. Inicializar Firebase:

    • Para Android: 
      import com.google.firebase.analytics.FirebaseAnalytics;

public class MainActivity extends AppCompatActivity {
    private FirebaseAnalytics mFirebaseAnalytics;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);
    }
}
    • Para iOS: 
      import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()
        return true
    }
}

Eventos y Parámetros Personalizados

Los eventos y parámetros personalizados te permiten rastrear acciones y atributos específicos que son relevantes para tu aplicación.

Registro de Eventos Personalizados

  • Android:

    Bundle bundle = new Bundle();
bundle.putString(FirebaseAnalytics.Param.ITEM_ID, "id123");
bundle.putString(FirebaseAnalytics.Param.ITEM_NAME, "name123");
bundle.putString(FirebaseAnalytics.Param.CONTENT_TYPE, "image");
mFirebaseAnalytics.logEvent(FirebaseAnalytics.Event.SELECT_CONTENT, bundle);

  • iOS:

    Analytics.logEvent(AnalyticsEventSelectContent, parameters: [
  AnalyticsParameterItemID: "id123" as NSObject,
  AnalyticsParameterItemName: "name123" as NSObject,
  AnalyticsParameterContentType: "image" as NSObject
])

Uso de Parámetros Personalizados

Los parámetros personalizados proporcionan contexto adicional a tus eventos.

  • Android:

    Bundle bundle = new Bundle();
bundle.putString("custom_param", "value123");
mFirebaseAnalytics.logEvent("custom_event", bundle);

  • iOS:

    Analytics.logEvent("custom_event", parameters: [
  "custom_param": "value123" as NSObject
])

Propiedades del Usuario

Las propiedades del usuario son atributos que defines para describir segmentos de tu base de usuarios, como la preferencia de idioma o la ubicación geográfica.

Configuración de Propiedades del Usuario

  • Android:

    mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

  • iOS:

    Analytics.setUserProperty("favorite_food", forName: "pizza")

Técnicas Avanzadas de Análisis

Las técnicas avanzadas implican el uso de Firebase Analytics junto con otros servicios de Firebase y herramientas externas.

Análisis de Embudo

El análisis de embudo te ayuda a entender los pasos que los usuarios toman para completar un objetivo en tu aplicación.

Análisis de Cohorte

El análisis de cohorte agrupa a los usuarios en función de características compartidas para analizar su comportamiento a lo largo del tiempo.

Integración con BigQuery

BigQuery te permite realizar consultas complejas sobre tus datos de Firebase Analytics.

Exportación de Datos a BigQuery

  1. Habilitar la Integración con BigQuery:

    • Ve a la Consola de Firebase.
    • Navega a la sección "Integraciones".
    • Vincula tu proyecto de Firebase a BigQuery.

  2. Consulta de Datos:

    SELECT
  event_name,
  COUNT(*) as event_count
FROM
  `your_project_id.analytics_1234567890.events_*`
WHERE
  _TABLE_SUFFIX BETWEEN '20210101' AND '20210131'
GROUP BY
  event_name
ORDER BY
  event_count DESC

Conclusión

Firebase Analytics proporciona una plataforma robusta para rastrear y analizar el comportamiento de los usuarios en tu aplicación. Al configurar eventos personalizados, parámetros y propiedades del usuario, puedes obtener información profunda sobre las interacciones de los usuarios. Las técnicas avanzadas como el análisis de embudo y de cohorte, junto con la integración de BigQuery, te permiten realizar análisis detallados y complejos, ayudándote a tomar decisiones basadas en datos para mejorar tu aplicación.
