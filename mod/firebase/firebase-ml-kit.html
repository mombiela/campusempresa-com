Firebase ML Kit
Firebase Remote Config ►

Firebase ML Kit es un SDK móvil que trae la experiencia en aprendizaje automático de Google a las aplicaciones de Android e iOS en un paquete potente y fácil de usar. Ofrece tanto APIs en el dispositivo como basadas en la nube para ayudarte a implementar funciones de aprendizaje automático en tu aplicación.

Introducción a Firebase ML Kit

  • ¿Qué es Firebase ML Kit?
    • Un conjunto de APIs proporcionadas por Firebase para integrar el aprendizaje automático en aplicaciones móviles.
    • Soporta modelos de aprendizaje automático tanto en el dispositivo como en la nube.
  • ¿Por qué usar Firebase ML Kit?
    • Simplifica la integración de funciones de aprendizaje automático.
    • Proporciona modelos preentrenados para casos de uso comunes.
    • Ofrece la capacidad de desplegar modelos personalizados.

Empezando con Firebase ML Kit

  • Configurando Firebase en tu proyecto
    • Añade Firebase a tu proyecto de Android o iOS.
    • Habilita ML Kit en la consola de Firebase.
  • Añadiendo dependencias
    • Para Android: 
      dependencies {
    implementation 'com.google.firebase:firebase-ml-vision:24.0.3'
}
    • Para iOS: 
      pod 'Firebase/MLVision'

APIs en el Dispositivo

  • Reconocimiento de Texto
    • Reconoce texto en imágenes.
    • Ejemplo: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionTextRecognizer detector = FirebaseVision.getInstance().getOnDeviceTextRecognizer();

detector.processImage(image)
    .addOnSuccessListener(new OnSuccessListener<FirebaseVisionText>() {
        @Override
        public void onSuccess(FirebaseVisionText texts) {
            // Tarea completada con éxito
            for (FirebaseVisionText.TextBlock block : texts.getTextBlocks()) {
                String text = block.getText();
                Log.d("TextRecognition", text);
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // La tarea falló con una excepción
            Log.e("TextRecognition", e.getMessage());
        }
    });
  • Detección de Rostros
    • Detecta rostros en imágenes y proporciona información como posición, puntos de referencia y clasificación.
    • Ejemplo: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionFaceDetectorOptions options =
    new FirebaseVisionFaceDetectorOptions.Builder()
        .setModeType(FirebaseVisionFaceDetectorOptions.FAST)
        .setLandmarkType(FirebaseVisionFaceDetectorOptions.ALL_LANDMARKS)
        .setClassificationType(FirebaseVisionFaceDetectorOptions.ALL_CLASSIFICATIONS)
        .build();
FirebaseVisionFaceDetector detector = FirebaseVision.getInstance().getVisionFaceDetector(options);

detector.detectInImage(image)
    .addOnSuccessListener(new OnSuccessListener<List<FirebaseVisionFace>>() {
        @Override
        public void onSuccess(List<FirebaseVisionFace> faces) {
            // Tarea completada con éxito
            for (FirebaseVisionFace face : faces) {
                Rect bounds = face.getBoundingBox();
                Log.d("FaceDetection", "Límites del rostro: " + bounds.toString());
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // La tarea falló con una excepción
            Log.e("FaceDetection", e.getMessage());
        }
    });

APIs Basadas en la Nube

  • Reconocimiento de Texto en la Nube
    • Reconoce texto en imágenes usando modelos basados en la nube.
    • Ejemplo: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionCloudTextRecognizerOptions options =
    new FirebaseVisionCloudTextRecognizerOptions.Builder()
        .setLanguageHints(Arrays.asList("en", "fr"))
        .build();
FirebaseVisionTextRecognizer detector = FirebaseVision.getInstance().getCloudTextRecognizer(options);

detector.processImage(image)
    .addOnSuccessListener(new OnSuccessListener<FirebaseVisionText>() {
        @Override
        public void onSuccess(FirebaseVisionText texts) {
            // Tarea completada con éxito
            for (FirebaseVisionText.TextBlock block : texts.getTextBlocks()) {
                String text = block.getText();
                Log.d("CloudTextRecognition", text);
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // La tarea falló con una excepción
            Log.e("CloudTextRecognition", e.getMessage());
        }
    });

Modelos Personalizados

  • Desplegando Modelos Personalizados
    • Sube tus modelos personalizados de TensorFlow Lite a Firebase.
    • Ejemplo: 
      FirebaseModelDownloadConditions conditions = new FirebaseModelDownloadConditions.Builder()
    .requireWifi()
    .build();
FirebaseCustomRemoteModel remoteModel = new FirebaseCustomRemoteModel.Builder("my_model")
    .build();
FirebaseModelManager.getInstance().download(remoteModel, conditions)
    .addOnSuccessListener(new OnSuccessListener<Void>() {
        @Override
        public void onSuccess(Void aVoid) {
            // Modelo descargado con éxito
            Log.d("CustomModel", "Modelo descargado");
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // La tarea falló con una excepción
            Log.e("CustomModel", e.getMessage());
        }
    });

Conclusión

Firebase ML Kit proporciona un conjunto completo de herramientas para integrar el aprendizaje automático en tus aplicaciones móviles. Ya sea que estés utilizando modelos preentrenados para tareas comunes como el reconocimiento de texto y la detección de rostros o desplegando tus modelos personalizados, Firebase ML Kit simplifica el proceso y te permite centrarte en construir grandes experiencias de usuario. Aprovechando tanto las capacidades en el dispositivo como en la nube, puedes crear aplicaciones potentes y eficientes que utilicen los últimos avances en aprendizaje automático.
