Introducción a las Notificaciones de Firebase

Firebase Notifications es un servicio que te permite enviar notificaciones a tus usuarios en diferentes plataformas como Android, iOS y web. Utiliza Firebase Cloud Messaging (FCM) para entregar mensajes de manera confiable y eficiente.

Conceptos Clave

  • Firebase Cloud Messaging (FCM): Una solución de mensajería multiplataforma que te permite enviar mensajes de manera confiable y sin costo.
  • Mensajes de Notificación: Mensajes que son mostrados al usuario por el sistema.
  • Mensajes de Datos: Mensajes que son manejados por la aplicación cliente.

Configuración de Notificaciones de Firebase

Antes de poder enviar notificaciones, necesitas configurar Firebase en tu proyecto.

Pasos para Configurar Firebase

  1. Crear un Proyecto de Firebase:

    • Ve a la Consola de Firebase.
    • Haz clic en "Agregar proyecto" y sigue el asistente de configuración.

  2. Agregar Firebase a Tu Aplicación:

    • Para Android, descarga el archivo google-services.json y colócalo en el directorio app de tu aplicación.
    • Para iOS, descarga el archivo GoogleService-Info.plist y agrégalo a tu proyecto de Xcode.

  3. Agregar el SDK de Firebase:

    • Para Android, agrega las siguientes dependencias en tu archivo build.gradle: 
      implementation 'com.google.firebase:firebase-messaging:23.0.0'
    • Para iOS, agrega lo siguiente a tu Podfile: 
      pod 'Firebase/Messaging'

Envío de Notificaciones

Hay múltiples maneras de enviar notificaciones usando Firebase.

Usando la Consola de Firebase

  1. Navegar a la Consola de Firebase:

    • Selecciona tu proyecto.
    • Ve a la sección "Cloud Messaging".

  2. Redactar un Mensaje:

    • Haz clic en "Enviar tu primer mensaje".
    • Ingresa el título y texto de la notificación.
    • Selecciona el objetivo (por ejemplo, aplicación, tema o segmento de usuarios).

  3. Enviar el Mensaje:

    • Haz clic en "Enviar".

Usando el SDK de Firebase Admin

También puedes enviar notificaciones programáticamente usando el SDK de Firebase Admin.

Ejemplo: Enviar una Notificación desde un Servidor Node.js

  1. Instalar el SDK de Firebase Admin:

    npm install firebase-admin --save

  2. Inicializar el SDK:

    const admin = require('firebase-admin');
const serviceAccount = require('path/to/serviceAccountKey.json');

admin.initializeApp({
  credential: admin.credential.cert(serviceAccount)
});

  3. Enviar una Notificación:

    const message = {
  notification: {
    title: 'Hola Mundo',
    body: '¡Esta es una notificación de Firebase!'
  },
  token: 'user-device-token'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Mensaje enviado exitosamente:', response);
  })
  .catch((error) => {
    console.log('Error al enviar el mensaje:', error);
  });

Manejo de Notificaciones en la Aplicación

Para manejar notificaciones en tu aplicación, necesitas configurar escuchadores.

Android

  1. Crear un Servicio para Manejar Mensajes:

    public class MyFirebaseMessagingService extends FirebaseMessagingService {
  @Override
  public void onMessageReceived(RemoteMessage remoteMessage) {
    // Manejar el mensaje
    if (remoteMessage.getNotification() != null) {
      showNotification(remoteMessage.getNotification().getTitle(), remoteMessage.getNotification().getBody());
    }
  }

  private void showNotification(String title, String body) {
    NotificationCompat.Builder builder = new NotificationCompat.Builder(this, "CHANNEL_ID")
        .setSmallIcon(R.drawable.notification_icon)
        .setContentTitle(title)
        .setContentText(body)
        .setPriority(NotificationCompat.PRIORITY_DEFAULT);

    NotificationManagerCompat notificationManager = NotificationManagerCompat.from(this);
    notificationManager.notify(0, builder.build());
  }
}

  2. Registrar el Servicio en AndroidManifest.xml:

    <service android:name=".MyFirebaseMessagingService">
  <intent-filter>
    <action android:name="com.google.firebase.MESSAGING_EVENT"/>
  </intent-filter>
</service>

iOS

  1. Solicitar Permisos de Notificación:

    import UserNotifications

UNUserNotificationCenter.current().requestAuthorization(options: [.alert, .sound, .badge]) { granted, error in
  // Manejar permiso concedido o error
}

  2. Manejar Notificaciones Entrantes:

    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate, UNUserNotificationCenterDelegate, MessagingDelegate {
  func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    FirebaseApp.configure()
    UNUserNotificationCenter.current().delegate = self
    Messaging.messaging().delegate = self
    return true
  }

  func messaging(_ messaging: Messaging, didReceive remoteMessage: MessagingRemoteMessage) {
    // Manejar el mensaje
  }

  func userNotificationCenter(_ center: UNUserNotificationCenter, willPresent notification: UNNotification, withCompletionHandler completionHandler: @escaping (UNNotificationPresentationOptions) -> Void) {
    completionHandler([.alert, .sound, .badge])
  }
}

Temas Avanzados

Envío de Mensajes de Datos

Los mensajes de datos te permiten enviar pares clave-valor personalizados a tu aplicación.

Ejemplo: Enviar un Mensaje de Datos

const message = {
  data: {
    key1: 'value1',
    key2: 'value2'
  },
  token: 'user-device-token'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Mensaje enviado exitosamente:', response);
  })
  .catch((error) => {
    console.log('Error al enviar el mensaje:', error);
  });

Mensajería por Temas

La mensajería por temas te permite enviar mensajes a múltiples dispositivos que se han suscrito a un tema en particular.

Ejemplo: Suscribirse a un Tema

const topic = 'news';

admin.messaging().subscribeToTopic(['user-device-token'], topic)
  .then((response) => {
    console.log('Suscripción al tema exitosa:', response);
  })
  .catch((error) => {
    console.log('Error al suscribirse al tema:', error);
  });

Ejemplo: Enviar un Mensaje a un Tema

const message = {
  notification: {
    title: 'Noticias de Última Hora',
    body: '¡Revisa las últimas noticias!'
  },
  topic: 'news'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Mensaje enviado exitosamente:', response);
  })
  .catch((error) => {
    console.log('Error al enviar el mensaje:', error);
  });

Conclusión

Las Notificaciones de Firebase proporcionan una manera poderosa y flexible de interactuar con tus usuarios. Ya sea que estés enviando mensajes de notificación simples o mensajes de datos complejos, Firebase facilita la entrega confiable de tu contenido. Al entender los conceptos básicos y explorar temas avanzados como los mensajes de datos y la mensajería por temas, puedes crear un sistema de notificaciones robusto para tu aplicación.
