Firebase Cloud Messaging (FCM) es una solución de mensajería multiplataforma que te permite enviar mensajes de manera confiable y sin costo. Este tema te guiará a través de la configuración de FCM desde un nivel principiante hasta un nivel avanzado.

Introducción a Firebase Cloud Messaging

  • ¿Qué es FCM?
    • Un servicio para enviar notificaciones y mensajes a usuarios en diferentes plataformas.
    • Soporta tanto mensajes directos como mensajes basados en temas.
  • ¿Por qué usar FCM?
    • Gratis y escalable.
    • Fácil integración con los servicios de Firebase.
    • Soporta varios tipos de mensajes (notificación, datos, etc.).

Prerrequisitos

  • Comprensión básica de Firebase.
  • Proyecto de Firebase configurado en la Consola de Firebase.
  • Conocimientos básicos de desarrollo Android/iOS.

Configuración de FCM para Android

Paso 1: Agregar Firebase a Tu Proyecto Android

  1. Crea un proyecto de Firebase en la Consola de Firebase.
  2. Registra tu aplicación con Firebase:
    • Agrega el nombre del paquete de tu aplicación Android.
    • Descarga el archivo google-services.json.
  3. Agrega el archivo google-services.json a tu proyecto:
    • Colócalo en el directorio app.

Paso 2: Agregar el SDK de Firebase a Tu Proyecto

  1. Abre tu archivo build.gradle (nivel de proyecto) y agrega lo siguiente: 
    buildscript {
    dependencies {
        // Agrega esta línea
        classpath 'com.google.gms:google-services:4.3.10'
    }
}
  2. Abre tu archivo build.gradle (nivel de aplicación) y agrega lo siguiente: 
    dependencies {
    // Agrega el SDK de Firebase para FCM
    implementation 'com.google.firebase:firebase-messaging:23.0.0'
}

// Agrega esta línea al final del archivo
apply plugin: 'com.google.gms.google-services'

Paso 3: Configurar FCM en Tu Aplicación

  1. Inicializa Firebase en tu MainActivity.java: 
    import com.google.firebase.messaging.FirebaseMessaging;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Inicializa Firebase
        FirebaseMessaging.getInstance().subscribeToTopic("news")
            .addOnCompleteListener(task -> {
                String msg = "Suscrito al tema de noticias";
                if (!task.isSuccessful()) {
                    msg = "Suscripción fallida";
                }
                Log.d("FCM", msg);
            });
    }
}

Configuración de FCM para iOS

Paso 1: Agregar Firebase a Tu Proyecto iOS

  1. Crea un proyecto de Firebase en la Consola de Firebase.
  2. Registra tu aplicación con Firebase:
    • Agrega el ID del paquete de tu aplicación iOS.
    • Descarga el archivo GoogleService-Info.plist.
  3. Agrega el archivo GoogleService-Info.plist a tu proyecto:
    • Arrástralo y suéltalo en tu proyecto de Xcode.

Paso 2: Agregar el SDK de Firebase a Tu Proyecto

  1. Abre tu Podfile y agrega lo siguiente: 
    pod 'Firebase/Messaging'
  2. Ejecuta pod install para instalar el SDK de Firebase.

Paso 3: Configurar FCM en Tu Aplicación

  1. Inicializa Firebase en tu AppDelegate.swift: 
    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

    var window: UIWindow?

    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicializa Firebase
        FirebaseApp.configure()

        // Suscribirse a un tema
        Messaging.messaging().subscribe(toTopic: "news") { error in
            print("Suscrito al tema de noticias")
        }

        return true
    }
}

Envío de Mensajes con FCM

Envío de un Mensaje de Notificación

  • A través de la Consola de Firebase:
    • Navega a la Consola de Firebase.
    • Selecciona tu proyecto y ve a la sección de Cloud Messaging.
    • Haz clic en "Enviar tu primer mensaje".
    • Rellena los detalles del mensaje y el objetivo (por ejemplo, tema "news").
    • Haz clic en "Enviar".

Envío de un Mensaje de Datos

  • A través del SDK de Administración de Firebase: 
    const admin = require('firebase-admin');
admin.initializeApp();

const message = {
  data: {
    score: '850',
    time: '2:45'
  },
  topic: 'news'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Mensaje enviado exitosamente:', response);
  })
  .catch((error) => {
    console.log('Error al enviar el mensaje:', error);
  });

Temas Avanzados

Manejo de Cargas Útiles de Mensajes

  • Mensajes de Notificación: Mostrados automáticamente por el sistema.
  • Mensajes de Datos: Manejado por la aplicación en segundo plano o en primer plano.

Personalización de Notificaciones

  • Sonido Personalizado: Especifica un archivo de sonido personalizado en la carga útil de la notificación.
  • Acciones Personalizadas: Agrega botones de acción a las notificaciones.

Manejo de Errores

  • Errores Comunes:
    • Token de registro inválido.
    • Límite de tasa de mensajes excedido.
  • Consejos de Depuración:
    • Revisa los registros de la Consola de Firebase.
    • Usa Firebase DebugView.

Conclusión

Configurar Firebase Cloud Messaging implica integrar el SDK de Firebase en tu proyecto Android o iOS, configurar tu aplicación para recibir mensajes y enviar mensajes a través de la Consola de Firebase o el SDK de Administración. Con FCM, puedes enviar tanto mensajes de notificación como de datos, personalizar notificaciones y manejar errores de manera efectiva. Esta configuración proporciona una base robusta para implementar mensajería en tiempo real en tus aplicaciones.
