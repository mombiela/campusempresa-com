Firebase Analytics es una herramienta poderosa que te ayuda a entender cómo los usuarios interactúan con tu aplicación. Esta sección te guiará a través de la configuración de Firebase Analytics, desde lo básico hasta configuraciones más avanzadas.

Introducción a Firebase Analytics

Firebase Analytics proporciona información sobre el comportamiento y la participación de los usuarios. Captura automáticamente una amplia gama de eventos y propiedades de usuario, y puedes registrar eventos personalizados para medir acciones específicas.

Conceptos Clave

  • Eventos: Acciones que los usuarios realizan en tu aplicación, como vistas de pantalla, clics en botones o compras.
  • Propiedades de Usuario: Atributos que defines para describir segmentos de tu base de usuarios, como la preferencia de idioma o la ubicación geográfica.
  • Tableros: Representaciones visuales de tus datos, disponibles en la consola de Firebase.

Configuración de Firebase Analytics

Paso 1: Crear un Proyecto de Firebase

  1. Ve a la Consola de Firebase.
  2. Haz clic en "Agregar proyecto" y sigue las indicaciones para crear un nuevo proyecto.

Paso 2: Agregar Firebase a Tu Aplicación

Para Android

  1. En la consola de Firebase, selecciona tu proyecto.
  2. Haz clic en el ícono de Android para agregar una aplicación de Android.
  3. Registra tu aplicación con el nombre del paquete y sigue las instrucciones para descargar el archivo google-services.json.
  4. Coloca el archivo google-services.json en el directorio app de tu proyecto de Android.
  5. Agrega el SDK de Firebase a tus archivos build.gradle: 
    // build.gradle a nivel de proyecto
buildscript {
    dependencies {
        // Agrega esta línea
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// build.gradle a nivel de aplicación
apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services'

dependencies {
    // Agrega esta línea
    implementation 'com.google.firebase:firebase-analytics:19.0.0'
}

Para iOS

  1. En la consola de Firebase, selecciona tu proyecto.
  2. Haz clic en el ícono de iOS para agregar una aplicación de iOS.
  3. Registra tu aplicación con el ID del paquete y sigue las instrucciones para descargar el archivo GoogleService-Info.plist.
  4. Agrega el archivo GoogleService-Info.plist a tu proyecto de Xcode.
  5. Agrega el SDK de Firebase a tu Podfile: 
    # Descomenta la siguiente línea para definir una plataforma global para tu proyecto
# platform :ios, '9.0'

target 'YourApp' do
  # Agrega esta línea
  pod 'Firebase/Analytics'
end
  6. Ejecuta pod install para instalar el SDK.

Paso 3: Inicializar Firebase en Tu Aplicación

Para Android

En tu archivo MainActivity.java o MainActivity.kt, inicializa Firebase:

import com.google.firebase.analytics.FirebaseAnalytics;

public class MainActivity extends AppCompatActivity {
    private FirebaseAnalytics mFirebaseAnalytics;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Inicializa Firebase Analytics
        mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);
    }
}

Para iOS

En tu archivo AppDelegate.swift, inicializa Firebase:

import UIKit
import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicializa Firebase
        FirebaseApp.configure()
        return true
    }
}

Registro de Eventos

Firebase Analytics registra automáticamente algunos eventos, pero puedes registrar eventos personalizados para rastrear acciones específicas.

Ejemplo: Registro de un Evento Personalizado

Para Android

Bundle bundle = new Bundle();
bundle.putString(FirebaseAnalytics.Param.ITEM_ID, "id123");
bundle.putString(FirebaseAnalytics.Param.ITEM_NAME, "example_item");
bundle.putString(FirebaseAnalytics.Param.CONTENT_TYPE, "text");
mFirebaseAnalytics.logEvent(FirebaseAnalytics.Event.SELECT_CONTENT, bundle);

Para iOS

Analytics.logEvent(AnalyticsEventSelectContent, parameters: [
    AnalyticsParameterItemID: "id123" as NSObject,
    AnalyticsParameterItemName: "example_item" as NSObject,
    AnalyticsParameterContentType: "text" as NSObject
])

Visualización de Datos de Analytics

  1. Ve a la Consola de Firebase.
  2. Selecciona tu proyecto.
  3. Navega a la sección "Analytics" para ver tableros e informes.

Configuraciones Avanzadas

Propiedades de Usuario

Define propiedades de usuario para segmentar tu audiencia:

Para Android

mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

Para iOS

Analytics.setUserProperty("pizza", forName: "favorite_food")

Depuración de Analytics

Usa la Vista de Depuración en la Consola de Firebase para ver eventos en tiempo real:

  1. Para Android, habilita el modo de depuración ejecutando: 
    adb shell setprop debug.firebase.analytics.app <your_package_name>
  2. Para iOS, habilita el modo de depuración configurando el siguiente argumento en Xcode: 
    -FIRDebugEnabled

Conclusión

Configurar Firebase Analytics implica crear un proyecto de Firebase, agregar Firebase a tu aplicación, inicializar Firebase y registrar eventos. Al aprovechar Firebase Analytics, puedes obtener información valiosa sobre el comportamiento de los usuarios y tomar decisiones basadas en datos para mejorar tu aplicación.
