Introducción

Las propiedades del usuario en Firebase son atributos que puedes definir para describir segmentos de tu base de usuarios. Estas propiedades te ayudan a entender mejor a tus usuarios y pueden ser utilizadas para dirigir audiencias específicas para análisis, notificaciones y más.

Conceptos Clave

  • Propiedades del Usuario: Atributos personalizados que defines para describir segmentos de tu base de usuarios.
  • Propiedades del Usuario Predefinidas: Firebase proporciona algunas propiedades del usuario predefinidas como first_open_time, last_sign_in_time, etc.
  • Propiedades del Usuario Personalizadas: Puedes definir hasta 25 propiedades del usuario personalizadas por proyecto.

Configuración de Propiedades del Usuario

Usando la Consola de Firebase

  1. Navega a la Consola de Firebase.
  2. Selecciona tu proyecto.
  3. Ve a la sección "Analytics".
  4. Haz clic en "User Properties".
  5. Añade una nueva propiedad del usuario definiendo su nombre y descripción.

Usando el SDK de Firebase

Android (Java)

// Importar Firebase Analytics
import com.google.firebase.analytics.FirebaseAnalytics;

// Obtener instancia de Firebase Analytics
FirebaseAnalytics mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);

// Configurar propiedad del usuario
mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

iOS (Swift)

// Importar Firebase
import Firebase

// Obtener instancia de Firebase Analytics
let analytics = Analytics.self

// Configurar propiedad del usuario
analytics.setUserProperty("favorite_food", forName: "pizza")

Web (JavaScript)

// Importar Firebase
import firebase from "firebase/app";
import "firebase/analytics";

// Inicializar Firebase
firebase.initializeApp({
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID",
  measurementId: "YOUR_MEASUREMENT_ID"
});

// Obtener instancia de Firebase Analytics
const analytics = firebase.analytics();

// Configurar propiedad del usuario
analytics.setUserProperties({ favorite_food: "pizza" });

Uso de Propiedades del Usuario en Analytics

Creación de Audiencias

  1. Ve a la Consola de Firebase.
  2. Selecciona tu proyecto.
  3. Navega a la sección "Analytics".
  4. Haz clic en "Audiences".
  5. Crea una nueva audiencia definiendo condiciones basadas en propiedades del usuario.

Filtrado de Informes

  • Puedes filtrar tus informes de análisis basados en propiedades del usuario para obtener información sobre segmentos específicos de tu base de usuarios.

Mejores Prácticas

  • Manténlo Simple: Usa nombres claros y concisos para tus propiedades del usuario.
  • Consistencia: Asegura consistencia en las convenciones de nombres a través de diferentes plataformas.
  • Privacidad: Evita usar información sensible como propiedades del usuario.

Casos de Uso Ejemplares

Dirigir Notificaciones

Puedes usar propiedades del usuario para dirigir notificaciones a segmentos específicos de usuarios. Por ejemplo, si tienes una propiedad del usuario favorite_food, puedes enviar una notificación sobre un descuento en pizza a los usuarios que hayan configurado favorite_food como "pizza".

Personalización de la Experiencia del Usuario

Usa propiedades del usuario para personalizar la experiencia del usuario. Por ejemplo, si tienes una propiedad del usuario preferred_language, puedes mostrar contenido en el idioma preferido del usuario.

Conclusión

Las propiedades del usuario en Firebase son herramientas poderosas para entender y segmentar tu base de usuarios. Al configurar y usar propiedades del usuario de manera efectiva, puedes obtener información valiosa y crear experiencias más personalizadas para tus usuarios. Recuerda seguir las mejores prácticas para asegurar consistencia y privacidad.
