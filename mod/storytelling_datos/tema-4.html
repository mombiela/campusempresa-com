Tipos de Datos y su Relevancia
Herramientas y Técnicas de Análisis de Datos ►

En el contexto del storytelling con datos, entender los tipos de datos y su relevancia es fundamental para comunicar de manera efectiva. Los datos pueden ser categorizados en diferentes tipos, cada uno con sus propias características y usos específicos. Este conocimiento te permitirá seleccionar las visualizaciones adecuadas y contar historias más convincentes.

Tipos de Datos

Existen varios tipos de datos que puedes encontrar al trabajar con datasets. Los principales son:

  • Datos Categóricos: Representan categorías o grupos. No tienen un orden inherente.
    • Ejemplo: Colores (rojo, azul, verde), tipos de frutas (manzana, naranja, plátano).
  • Datos Ordinales: Similar a los datos categóricos, pero con un orden específico.
    • Ejemplo: Clasificaciones (primero, segundo, tercero), niveles de satisfacción (bajo, medio, alto).
  • Datos Numéricos: Representan cantidades y pueden ser continuos o discretos.
    • Datos Discretos: Valores contables, no fraccionables.
      • Ejemplo: Número de hijos, cantidad de productos vendidos.
    • Datos Continuos: Valores medibles, pueden tomar cualquier valor dentro de un rango.
      • Ejemplo: Altura, peso, temperatura.

Relevancia de los Tipos de Datos

La relevancia de los tipos de datos radica en cómo afectan la elección de las visualizaciones y la interpretación de la información. A continuación, se presentan algunas consideraciones clave:

  • Datos Categóricos:

    • Visualizaciones comunes: Gráficos de barras, gráficos de pastel.
    • Uso: Comparar diferentes categorías, mostrar proporciones.
    • Ejemplo: 
      import matplotlib.pyplot as plt

categorias = ['Manzanas', 'Naranjas', 'Plátanos']
cantidades = [10, 15, 7]

plt.bar(categorias, cantidades)
plt.xlabel('Frutas')
plt.ylabel('Cantidad')
plt.title('Cantidad de Frutas')
plt.show()

  • Datos Ordinales:

    • Visualizaciones comunes: Gráficos de barras ordenados, gráficos de líneas.
    • Uso: Mostrar tendencias, comparar rangos.
    • Ejemplo: 
      import matplotlib.pyplot as plt

niveles = ['Bajo', 'Medio', 'Alto']
cantidades = [5, 20, 15]

plt.bar(niveles, cantidades)
plt.xlabel('Niveles de Satisfacción')
plt.ylabel('Cantidad de Respuestas')
plt.title('Encuesta de Satisfacción')
plt.show()

  • Datos Numéricos:

    • Visualizaciones comunes: Histogramas, gráficos de dispersión, gráficos de líneas.
    • Uso: Mostrar distribuciones, relaciones entre variables.
    • Ejemplo: 
      import matplotlib.pyplot as plt

edades = [22, 25, 30, 35, 40, 45, 50, 55, 60, 65]
salarios = [30000, 32000, 35000, 37000, 40000, 42000, 45000, 47000, 50000, 52000]

plt.scatter(edades, salarios)
plt.xlabel('Edad')
plt.ylabel('Salario')
plt.title('Relación entre Edad y Salario')
plt.show()

Comparación de Tipos de Datos

Tipo de Datos Ejemplo Visualizaciones Comunes Uso Principal
Categóricos Tipos de frutas Gráficos de barras, gráficos de pastel Comparar categorías, mostrar proporciones
Ordinales Niveles de satisfacción Gráficos de barras ordenados, gráficos de líneas Mostrar tendencias, comparar rangos
Numéricos Edad, salario Histogramas, gráficos de dispersión, gráficos de líneas Mostrar distribuciones, relaciones entre variables

Conclusión

Comprender los tipos de datos y su relevancia es esencial para el storytelling con datos. Cada tipo de dato tiene sus propias características y aplicaciones, lo que influye en la elección de las visualizaciones y en cómo se interpreta la información. Al dominar estos conceptos, podrás contar historias más efectivas y persuasivas con tus datos.
