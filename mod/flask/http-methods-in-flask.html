Introducción a los Métodos HTTP

Los métodos HTTP son una parte fundamental del desarrollo web, permitiendo a los clientes comunicarse con los servidores. En Flask, entender y utilizar estos métodos es crucial para crear aplicaciones web robustas.

Métodos HTTP Clave

  • GET: Recuperar datos del servidor.
  • POST: Enviar datos al servidor para crear un nuevo recurso.
  • PUT: Actualizar un recurso existente en el servidor.
  • DELETE: Eliminar un recurso del servidor.

Usando Métodos HTTP en Flask

Configuración de Flask

Primero, asegúrate de tener Flask instalado. Puedes instalarlo usando pip:

pip install Flask

Aplicación Básica de Flask

Comencemos con una aplicación básica de Flask para demostrar cómo manejar diferentes métodos HTTP.

from flask import Flask, request

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Bienvenido al tutorial de Métodos HTTP en Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Manejo de Solicitudes GET

El método GET se utiliza para recuperar datos del servidor. Por defecto, las rutas de Flask manejan solicitudes GET.

@app.route('/get-example', methods=['GET'])
def get_example():
    return "Esta es una solicitud GET"

Manejo de Solicitudes POST

El método POST se utiliza para enviar datos al servidor. Esto se usa comúnmente para envíos de formularios.

@app.route('/post-example', methods=['POST'])
def post_example():
    data = request.form['data']
    return f"Datos POST recibidos: {data}"

Manejo de Solicitudes PUT

El método PUT se utiliza para actualizar recursos existentes. En Flask, puedes manejar solicitudes PUT de manera similar a las solicitudes POST.

@app.route('/put-example', methods=['PUT'])
def put_example():
    data = request.form['data']
    return f"Datos PUT recibidos: {data}"

Manejo de Solicitudes DELETE

El método DELETE se utiliza para eliminar recursos del servidor.

@app.route('/delete-example', methods=['DELETE'])
def delete_example():
    data = request.form['data']
    return f"Solicitud DELETE recibida para los datos: {data}"

Aplicación de Ejemplo

Vamos a crear una aplicación simple que demuestre los cuatro métodos HTTP.

from flask import Flask, request, jsonify

app = Flask(__name__)

data_store = {}

@app.route('/data', methods=['GET', 'POST', 'PUT', 'DELETE'])
def data_handler():
    if request.method == 'GET':
        return jsonify(data_store)
    
    elif request.method == 'POST':
        new_data = request.json
        data_store.update(new_data)
        return jsonify({"message": "Datos añadidos", "data": new_data})
    
    elif request.method == 'PUT':
        update_data = request.json
        data_store.update(update_data)
        return jsonify({"message": "Datos actualizados", "data": update_data})
    
    elif request.method == 'DELETE':
        delete_data = request.json
        for key in delete_data.keys():
            data_store.pop(key, None)
        return jsonify({"message": "Datos eliminados", "data": delete_data})

if __name__ == '__main__':
    app.run(debug=True)

Resumen

Entender y usar los métodos HTTP en Flask es esencial para construir aplicaciones web dinámicas. Este tema cubrió los conceptos básicos de los métodos GET, POST, PUT y DELETE, proporcionando ejemplos y una aplicación simple para ilustrar su uso. Al dominar estos métodos, puedes crear aplicaciones web más interactivas y funcionales con Flask.
