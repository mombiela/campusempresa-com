Introducción a Websockets

  • Definición: Los Websockets proporcionan un canal de comunicación bidireccional completo sobre una única conexión TCP.
  • Casos de Uso: Aplicaciones en tiempo real como aplicaciones de chat, notificaciones en vivo, juegos, etc.
  • Ventajas: Baja latencia, menor sobrecarga en comparación con el sondeo HTTP.

Configuración de Flask-SocketIO

  • Instalación: Instala Flask-SocketIO usando pip. 
    pip install flask-socketio
  • Configuración Básica: Crea una aplicación simple de Flask con SocketIO. 
    from flask import Flask, render_template
from flask_socketio import SocketIO

app = Flask(__name__)
socketio = SocketIO(app)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    socketio.run(app)

Creando una Aplicación de Chat Simple

  • Plantilla HTML: Crea un archivo index.html para el lado del cliente.

    <!DOCTYPE html>
<html>
<head>
    <title>Chat</title>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/socket.io/4.0.0/socket.io.js"></script>
    <script type="text/javascript" charset="utf-8">
        document.addEventListener('DOMContentLoaded', (event) => {
            var socket = io.connect('http://' + document.domain + ':' + location.port);
            socket.on('connect', function() {
                socket.send('¡El usuario se ha conectado!');
            });
            socket.on('message', function(msg) {
                var p = document.createElement('p');
                p.innerHTML = msg;
                document.getElementById('messages').appendChild(p);
            });
            document.getElementById('sendbutton').onclick = function() {
                var message = document.getElementById('myMessage').value;
                socket.send(message);
            };
        });
    </script>
</head>
<body>
    <h1>Sala de Chat</h1>
    <div id="messages"></div>
    <input type="text" id="myMessage">
    <button id="sendbutton">Enviar</button>
</body>
</html>

  • Manejo del Lado del Servidor: Actualiza la aplicación Flask para manejar mensajes.

    from flask import Flask, render_template
from flask_socketio import SocketIO, send

app = Flask(__name__)
socketio = SocketIO(app)

@app.route('/')
def index():
    return render_template('index.html')

@socketio.on('message')
def handleMessage(msg):
    print('Mensaje: ' + msg)
    send(msg, broadcast=True)

if __name__ == '__main__':
    socketio.run(app, debug=True)

Transmisión de Mensajes

  • Transmisión: Asegurando que los mensajes se envíen a todos los clientes conectados.
    • En el ejemplo anterior, send(msg, broadcast=True) asegura que el mensaje se envíe a todos los clientes.

Espacios de Nombres y Salas

  • Espacios de Nombres: Permiten la separación de preocupaciones dentro de la misma aplicación.

    @socketio.on('message', namespace='/chat')
def handle_chat_message(msg):
    send(msg, broadcast=True, namespace='/chat')

  • Salas: Permiten agrupar clientes.

    @socketio.on('join')
def on_join(data):
    username = data['username']
    room = data['room']
    join_room(room)
    send(username + ' ha entrado en la sala.', to=room)

@socketio.on('leave')
def on_leave(data):
    username = data['username']
    room = data['room']
    leave_room(room)
    send(username + ' ha salido de la sala.', to=room)

Manejo de Errores

  • Manejo de Desconexiones: Detectar y manejar desconexiones de clientes. 
    @socketio.on('disconnect')
def test_disconnect():
    print('Cliente desconectado')
