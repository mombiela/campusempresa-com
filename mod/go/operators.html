Introducción

Los operadores son bloques de construcción fundamentales en cualquier lenguaje de programación. Permiten realizar varias operaciones en variables y valores. En Go, los operadores se categorizan en varios tipos, incluyendo operadores aritméticos, relacionales, lógicos, bit a bit y de asignación. Esta sección cubrirá cada tipo de operador, proporcionando explicaciones claras y ejemplos para ayudarte a entender cómo usarlos de manera efectiva.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas como suma, resta, multiplicación, división y módulo.

  • + : Suma
  • - : Resta
  • * : Multiplicación
  • / : División
  • % : Módulo

Ejemplo

package main

import "fmt"

func main() {
    a := 10
    b := 3

    fmt.Println("Suma:", a + b)        // Salida: 13
    fmt.Println("Resta:", a - b)       // Salida: 7
    fmt.Println("Multiplicación:", a * b)  // Salida: 30
    fmt.Println("División:", a / b)    // Salida: 3
    fmt.Println("Módulo:", a % b)      // Salida: 1
}

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores. Devuelven un resultado booleano (verdadero o falso).

  • == : Igual a
  • != : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a

Ejemplo

package main

import "fmt"

func main() {
    a := 10
    b := 20

    fmt.Println("a == b:", a == b)  // Salida: falso
    fmt.Println("a != b:", a != b)  // Salida: verdadero
    fmt.Println("a > b:", a > b)    // Salida: falso
    fmt.Println("a < b:", a < b)    // Salida: verdadero
    fmt.Println("a >= b:", a >= b)  // Salida: falso
    fmt.Println("a <= b:", a <= b)  // Salida: verdadero
}

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones o valores booleanos.

  • && : AND lógico
  • || : OR lógico
  • ! : NOT lógico

Ejemplo

package main

import "fmt"

func main() {
    a := verdadero
    b := falso

    fmt.Println("a && b:", a && b)  // Salida: falso
    fmt.Println("a || b:", a || b)  // Salida: verdadero
    fmt.Println("!a:", !a)          // Salida: falso
}

Operadores Bit a Bit

Los operadores bit a bit realizan operaciones a nivel de bits en tipos enteros.

  • & : AND bit a bit
  • | : OR bit a bit
  • ^ : XOR bit a bit
  • &^ : Limpieza de bits (AND NOT)
  • << : Desplazamiento a la izquierda
  • >> : Desplazamiento a la derecha

Ejemplo

package main

import "fmt"

func main() {
    a := 10  // 1010 en binario
    b := 3   // 0011 en binario

    fmt.Println("a & b:", a & b)    // Salida: 2  (0010 en binario)
    fmt.Println("a | b:", a & b)    // Salida: 11 (1011 en binario)
    fmt.Println("a ^ b:", a ^ b)    // Salida: 9  (1001 en binario)
    fmt.Println("a &^ b:", a &^ b)  // Salida: 8  (1000 en binario)
    fmt.Println("a << 1:", a << 1)  // Salida: 20 (10100 en binario)
    fmt.Println("a >> 1:", a >> 1)  // Salida: 5  (0101 en binario)
}

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. Incluyen el operador de asignación estándar = y los operadores de asignación compuesta.

  • = : Asignar
  • += : Sumar y asignar
  • -= : Restar y asignar
  • *= : Multiplicar y asignar
  • /= : Dividir y asignar
  • %= : Módulo y asignar
  • &= : AND bit a bit y asignar
  • |= : OR bit a bit y asignar
  • ^= : XOR bit a bit y asignar
  • <<=: Desplazamiento a la izquierda y asignar
  • >>=: Desplazamiento a la derecha y asignar

Ejemplo

package main

import "fmt"

func main() {
    a := 10
    b := 3

    a += b  // Equivalente a a = a + b
    fmt.Println("a += b:", a)  // Salida: 13

    a -= b  // Equivalente a a = a - b
    fmt.Println("a -= b:", a)  // Salida: 10

    a *= b  // Equivalente a a = a * b
    fmt.Println("a *= b:", a)  // Salida: 30

    a /= b  // Equivalente a a = a / b
    fmt.Println("a /= b:", a)  // Salida: 10

    a %= b  // Equivalente a a = a % b
    fmt.Println("a %= b:", a)  // Salida: 1
}

Conclusión

Entender los operadores es crucial para realizar varias operaciones en Go. Esta sección cubrió operadores aritméticos, relacionales, lógicos, bit a bit y de asignación, proporcionando ejemplos para ilustrar su uso. Dominar estos operadores te permitirá escribir código Go más eficiente y efectivo. A medida que progreses, encontrarás estos operadores indispensables para resolver problemas complejos e implementar algoritmos.
