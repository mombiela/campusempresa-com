Introducción

Go, también conocido como Golang, es un lenguaje de programación compilado y de tipado estático diseñado para la simplicidad y eficiencia. Entender la sintaxis y la estructura de Go es crucial para escribir programas efectivos y eficientes. Esta sección cubrirá los conceptos básicos de la sintaxis y estructura de Go, proporcionando una base para temas más avanzados.

Sintaxis Básica

Hola Mundo

El punto de partida tradicional para aprender cualquier lenguaje de programación es el programa "¡Hola, Mundo!". Así es como se ve en Go:

package main

import "fmt"

func main() {
    fmt.Println("¡Hola, Mundo!")
}
  • package main: Define el nombre del paquete. El paquete main es especial porque define un programa ejecutable independiente.
  • import "fmt": Importa el paquete fmt, que contiene funciones para E/S formateada.
  • func main() { ... }: La función main es el punto de entrada del programa.

Variables y Constantes

Las variables en Go se declaran usando la palabra clave var, mientras que las constantes se declaran usando la palabra clave const.

var nombre string = "Juan"
var edad int = 30
const pi float64 = 3.14159
  • var nombre string = "Juan": Declara una variable nombre de tipo string y la inicializa con "Juan".
  • const pi float64 = 3.14159: Declara una constante pi de tipo float64.

Declaraciones Cortas de Variables

Go también soporta declaraciones cortas de variables usando la sintaxis :=.

nombre := "Juan"
edad := 30
  • nombre := "Juan": Declara e inicializa nombre con "Juan". El tipo se infiere.

Tipos de Datos Básicos

Go tiene varios tipos de datos básicos:

  • int, float64, string, bool
  • Ejemplo:
var a int = 10
var b float64 = 3.14
var c string = "Hola"
var d bool = true

Estructuras de Control

Sentencias If-Else

Las sentencias condicionales en Go son similares a otros lenguajes.

if edad > 18 {
    fmt.Println("Adulto")
} else {
    fmt.Println("Menor")
}

Bucles For

Go usa la palabra clave for para los bucles.

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

Sentencias Switch

Las sentencias switch proporcionan una forma de ejecutar diferentes códigos basados en el valor de una variable.

switch dia := "Lunes"; dia {
case "Lunes":
    fmt.Println("Inicio de la semana")
case "Viernes":
    fmt.Println("Fin de la semana")
default:
    fmt.Println("Mitad de semana")
}

Funciones

Declaración Básica de Funciones

Las funciones en Go se declaran usando la palabra clave func.

func sumar(a int, b int) int {
    return a + b
}
  • func sumar(a int, b int) int: Declara una función sumar que toma dos parámetros int y devuelve un int.

Valores de Retorno Múltiples

Las funciones en Go pueden devolver múltiples valores.

func intercambiar(x, y string) (string, string) {
    return y, x
}
  • func intercambiar(x, y string) (string, string): Declara una función intercambiar que toma dos parámetros string y devuelve dos valores string.

Estructuras y Métodos

Estructuras

Las estructuras se usan para agrupar datos relacionados.

type Persona struct {
    Nombre string
    Edad   int
}
  • type Persona struct { ... }: Define un nuevo tipo de estructura Persona.

Métodos

Los métodos son funciones con un argumento receptor.

func (p Persona) saludar() string {
    return "Hola, mi nombre es " + p.Nombre
}
  • func (p Persona) saludar() string: Declara un método saludar en el tipo Persona.

Paquetes y Módulos

Creación de un Paquete

Los paquetes se usan para organizar el código en componentes reutilizables.

// archivo: mathutil/mathutil.go
package mathutil

func Sumar(a, b int) int {
    return a + b
}

Uso de un Paquete

Para usar un paquete, necesitas importarlo.

package main

import (
    "fmt"
    "path/to/mathutil"
)

func main() {
    resultado := mathutil.Sumar(1, 2)
    fmt.Println(resultado)
}

Conclusión

Entender la sintaxis y estructura de Go es esencial para escribir código eficiente y legible. Esta sección cubrió los conceptos básicos, incluyendo declaraciones de variables, estructuras de control, funciones, estructuras y paquetes. Con esta base, estás bien equipado para explorar temas más avanzados en la programación con Go.
