Configuración del Proyecto

Configurar un proyecto de Go correctamente es crucial para un desarrollo y mantenimiento eficientes. Esta sección te guiará a través del proceso de configurar un proyecto de Go desde cero, asegurándote de tener una base sólida para tu viaje de codificación.

Instalación de Go

Antes de que puedas comenzar un proyecto de Go, necesitas tener Go instalado en tu máquina.

  1. Descargar Go: Visita el sitio web oficial de Go y descarga el instalador para tu sistema operativo.

  2. Instalar Go: Sigue las instrucciones de instalación proporcionadas para tu sistema operativo.

  3. Verificar la Instalación: Abre una terminal y ejecuta el siguiente comando para verificar que Go esté instalado correctamente:

    go version

    Deberías ver una salida similar a go version go1.17.1 darwin/amd64, indicando la versión de Go instalada.

Configuración de tu Espacio de Trabajo

Go utiliza una estructura de espacio de trabajo específica para gestionar el código. El espacio de trabajo es una jerarquía de directorios con tres directorios en su raíz:

  • src: Contiene archivos fuente de Go.
  • pkg: Contiene objetos de paquetes.
  • bin: Contiene comandos ejecutables.

Creación del Espacio de Trabajo

  1. Crear un Directorio: Crea un directorio para tu espacio de trabajo de Go. Por ejemplo:

    mkdir -p $HOME/go/{bin,pkg,src}

  2. Configurar Variables de Entorno: Configura la variable de entorno GOPATH para que apunte a tu directorio de espacio de trabajo. Añade las siguientes líneas a tu archivo de configuración del shell (por ejemplo, .bashrc, .zshrc):

    export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

  3. Recargar Configuración: Recarga la configuración de tu shell para aplicar los cambios:

    source ~/.bashrc

Creación de un Nuevo Proyecto

Ahora que tu espacio de trabajo está configurado, puedes crear un nuevo proyecto de Go.

  1. Navegar al Espacio de Trabajo: Cambia al directorio src en tu espacio de trabajo:

    cd $GOPATH/src

  2. Crear Directorio del Proyecto: Crea un directorio para tu nuevo proyecto. Por ejemplo, si tu proyecto se llama hello, ejecutarías:

    mkdir -p hello
cd hello

  3. Inicializar el Módulo: Inicializa un nuevo módulo de Go en tu directorio de proyecto. Este paso crea un archivo go.mod, que rastrea las dependencias de tu proyecto:

    go mod init hello

    El archivo go.mod se verá algo así:

    module hello

go 1.17

Escribiendo tu Primer Programa en Go

Con tu proyecto configurado, ahora puedes escribir tu primer programa en Go.

  1. Crear un Archivo Principal: Crea un archivo llamado main.go en tu directorio de proyecto:

    touch main.go

  2. Escribir el Código: Abre main.go en tu editor de texto favorito y añade el siguiente código:

    package main

import "fmt"

func main() {
    fmt.Println("¡Hola, Mundo!")
}

  3. Ejecutar el Programa: Guarda el archivo y ejecuta tu programa usando el comando go run:

    go run main.go

    Deberías ver la salida:

    ¡Hola, Mundo!

Resumen

En esta sección, aprendiste cómo configurar un proyecto de Go desde cero. Instalaste Go, configuraste tu espacio de trabajo, creaste un nuevo proyecto y escribiste tu primer programa en Go. Esta configuración fundamental te permitirá gestionar y desarrollar tus proyectos de Go de manera eficiente a medida que avances en el curso.
