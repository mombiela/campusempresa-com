Introducción a HTTP y Servidores Web

  • HTTP (Protocolo de Transferencia de Hipertexto): La base de cualquier intercambio de datos en la Web, y es un protocolo utilizado para transmitir solicitudes de hipertexto e información entre servidores y clientes.
  • Servidor Web: Un servidor que utiliza HTTP para servir archivos que forman páginas web a los usuarios en respuesta a sus solicitudes, que son enviadas por los clientes HTTP de sus computadoras.

Configuración de un Servidor Web Básico en Go

  • Go tiene una biblioteca estándar poderosa que incluye el paquete net/http, lo que facilita la creación de servidores web.

Ejemplo: Servidor HTTP Básico

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "¡Hola, Mundo!")
}

func main() {
    http.HandleFunc("/", handler)
    fmt.Println("Iniciando servidor en el puerto 8080")
    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • http.HandleFunc("/", handler): Registra la función manejadora para la ruta URL raíz.
    • http.ListenAndServe(":8080", nil): Inicia el servidor en el puerto 8080.

Manejo de Diferentes Rutas

  • Puedes manejar diferentes rutas registrando múltiples funciones manejadoras.

Ejemplo: Manejo de Múltiples Rutas

package main

import (
    "fmt"
    "net/http"
)

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "¡Hola, Mundo!")
}

func aboutHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Página Acerca de")
}

func main() {
    http.HandleFunc("/", helloHandler)
    http.HandleFunc("/about", aboutHandler)
    fmt.Println("Iniciando servidor en el puerto 8080")
    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • http.HandleFunc("/about", aboutHandler): Registra la función aboutHandler para la ruta URL /about.

Servir Archivos Estáticos

  • Servir archivos estáticos como HTML, CSS y JavaScript es un requisito común para los servidores web.

Ejemplo: Servir Archivos Estáticos

package main

import (
    "net/http"
)

func main() {
    fs := http.FileServer(http.Dir("static"))
    http.Handle("/", fs)

    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • http.FileServer(http.Dir("static")): Crea un servidor de archivos que sirve archivos desde el directorio static.
    • http.Handle("/", fs): Registra el servidor de archivos para manejar solicitudes a la ruta URL raíz.

Manejo de Formularios y Parámetros de Consulta

  • Manejar la entrada del usuario a través de formularios y parámetros de consulta es esencial para aplicaciones web dinámicas.

Ejemplo: Manejo de Formularios

package main

import (
    "fmt"
    "net/http"
)

func formHandler(w http.ResponseWriter, r *http.Request) {
    if r.Method == "POST" {
        name := r.FormValue("name")
        fmt.Fprintf(w, "¡Hola, %s!", name)
    } else {
        http.ServeFile(w, r, "form.html")
    }
}

func main() {
    http.HandleFunc("/form", formHandler)
    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • r.Method == "POST": Verifica si el método de la solicitud es POST.
    • r.FormValue("name"): Recupera el valor del campo del formulario llamado "name".
    • http.ServeFile(w, r, "form.html"): Sirve el archivo HTML del formulario para solicitudes GET.

Ejemplo: Manejo de Parámetros de Consulta

package main

import (
    "fmt"
    "net/http"
)

func queryHandler(w http.ResponseWriter, r *http.Request) {
    name := r.URL.Query().Get("name")
    fmt.Fprintf(w, "¡Hola, %s!", name)
}

func main() {
    http.HandleFunc("/query", queryHandler)
    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • r.URL.Query().Get("name"): Recupera el valor del parámetro de consulta llamado "name".

Middleware en Go

  • Middleware es una forma de procesar solicitudes antes de que lleguen al manejador final.

Ejemplo: Middleware Simple

package main

import (
    "fmt"
    "net/http"
)

func loggingMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        fmt.Println("Solicitud recibida")
        next.ServeHTTP(w, r)
    })
}

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "¡Hola, Mundo!")
}

func main() {
    hello := http.HandlerFunc(helloHandler)
    http.Handle("/", loggingMiddleware(hello))
    http.ListenAndServe(":8080", nil)
}
  • Explicación:
    • loggingMiddleware(next http.Handler) http.Handler: Define una función middleware que registra solicitudes.
    • next.ServeHTTP(w, r): Llama al siguiente manejador en la cadena.
