Trabajando con Datos Externos

Introducción

En este módulo, exploraremos cómo trabajar con datos externos en VBA (Visual Basic for Applications). Esto incluye leer y escribir en archivos externos como archivos de texto, archivos CSV y bases de datos. Al final de este módulo, podrás automatizar tareas de importación/exportación de datos e interactuar con diversas fuentes de datos de manera eficiente.

Leyendo desde Archivos Externos

Leyendo desde Archivos de Texto

Leer desde archivos de texto es una tarea común en VBA. El siguiente ejemplo demuestra cómo leer datos de un archivo de texto línea por línea.

Sub ReadFromTextFile()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim line As String
    
    filePath = "C:\path\to\your\file.txt"
    fileNumber = FreeFile
    
    Open filePath For Input As #fileNumber
    
    Do While Not EOF(fileNumber)
        Line Input #fileNumber, line
        Debug.Print line
    Loop
    
    Close #fileNumber
End Sub
  • filePath: La ruta al archivo de texto.
  • fileNumber: Un número único asignado al archivo.
  • Line Input: Lee una línea del archivo.

Leyendo desde Archivos CSV

Los archivos CSV se utilizan comúnmente para el intercambio de datos. El siguiente ejemplo muestra cómo leer datos de un archivo CSV y poblarlos en una hoja de cálculo de Excel.

Sub ReadFromCSV()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim line As String
    Dim data As Variant
    Dim row As Integer
    
    filePath = "C:\path\to\your\file.csv"
    fileNumber = FreeFile
    row = 1
    
    Open filePath For Input As #fileNumber
    
    Do While Not EOF(fileNumber)
        Line Input #fileNumber, line
        data = Split(line, ",")
        For col = 0 To UBound(data)
            Cells(row, col + 1).Value = data(col)
        Next col
        row = row + 1
    Loop
    
    Close #fileNumber
End Sub
  • Split: Divide la línea en un array basado en el delimitador de coma.

Escribiendo en Archivos Externos

Escribiendo en Archivos de Texto

Escribir datos en un archivo de texto puede ser útil para registrar o exportar datos. Aquí hay un ejemplo de cómo escribir datos en un archivo de texto.

Sub WriteToTextFile()
    Dim filePath As String
    Dim fileNumber As Integer
    
    filePath = "C:\path\to\your\output.txt"
    fileNumber = FreeFile
    
    Open filePath For Output As #fileNumber
    
    Print #fileNumber, "This is a sample text."
    Print #fileNumber, "Writing to a text file using VBA."
    
    Close #fileNumber
End Sub
  • Print: Escribe una línea en el archivo.

Escribiendo en Archivos CSV

Exportar datos a un archivo CSV es sencillo. El siguiente ejemplo demuestra cómo escribir datos de una hoja de cálculo de Excel a un archivo CSV.

Sub WriteToCSV()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim row As Integer
    Dim col As Integer
    Dim line As String
    
    filePath = "C:\path\to\your\output.csv"
    fileNumber = FreeFile
    
    Open filePath For Output As #fileNumber
    
    row = 1
    Do While Cells(row, 1).Value <> ""
        line = ""
        col = 1
        Do While Cells(row, col).Value <> ""
            If col > 1 Then line = line & ","
            line = line & Cells(row, col).Value
            col = col + 1
        Loop
        Print #fileNumber, line
        row = row + 1
    Loop
    
    Close #fileNumber
End Sub

Interactuando con Bases de Datos

Conectándose a una Base de Datos

Conectarse a una base de datos te permite leer y escribir datos directamente. El siguiente ejemplo demuestra cómo conectarse a una base de datos de Access usando ADO (ActiveX Data Objects).

Sub ConnectToDatabase()
    Dim conn As Object
    Dim rs As Object
    Dim connStr As String
    Dim sql As String
    
    connStr = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\path\to\your\database.accdb;"
    sql = "SELECT * FROM TableName"
    
    Set conn = CreateObject("ADODB.Connection")
    conn.Open connStr
    
    Set rs = CreateObject("ADODB.Recordset")
    rs.Open sql, conn
    
    Do While Not rs.EOF
        Debug.Print rs.Fields("FieldName").Value
        rs.MoveNext
    Loop
    
    rs.Close
    conn.Close
End Sub
  • connStr: Cadena de conexión a la base de datos.
  • sql: Consulta SQL para ejecutar.
  • ADODB.Connection: Objeto para gestionar la conexión.
  • ADODB.Recordset: Objeto para contener los resultados de la consulta.

Ejecutando Comandos SQL

Ejecutar comandos SQL te permite manipular datos en la base de datos. El siguiente ejemplo muestra cómo insertar un nuevo registro en una tabla.

Sub InsertIntoDatabase()
    Dim conn As Object
    Dim connStr As String
    Dim sql As String
    
    connStr = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\path\to\your\database.accdb;"
    sql = "INSERT INTO TableName (FieldName) VALUES ('SampleData')"
    
    Set conn = CreateObject("ADODB.Connection")
    conn.Open connStr
    
    conn.Execute sql
    
    conn.Close
End Sub
  • INSERT INTO: Comando SQL para insertar datos.

Conclusión

En este módulo, cubrimos varios métodos para trabajar con datos externos en VBA, incluyendo leer y escribir en archivos de texto y CSV, así como interactuar con bases de datos. Estas habilidades son esenciales para automatizar tareas de procesamiento de datos e integrar Excel con otras fuentes de datos. Practica estos ejemplos e intenta implementarlos en tus proyectos para obtener una comprensión más profunda.
