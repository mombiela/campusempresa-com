Introducción

Los anuncios de colección en Facebook Ads son una poderosa herramienta para mostrar múltiples productos en un solo anuncio. Este formato es especialmente útil para tiendas en línea y marcas que desean destacar varios artículos a la vez, proporcionando una experiencia de compra inmersiva y fluida para los usuarios.

¿Qué son los Anuncios de Colección?

Los anuncios de colección combinan una imagen o video principal con una serie de productos relacionados que los usuarios pueden explorar. Al hacer clic en el anuncio, los usuarios son llevados a una experiencia de pantalla completa en Facebook, donde pueden ver más detalles sobre los productos y realizar compras directamente desde la plataforma.

Características Clave:

  • Imagen o Video Principal: Un elemento visual destacado que capta la atención del usuario.
  • Productos Relacionados: Una serie de productos que se muestran debajo del elemento principal.
  • Experiencia de Pantalla Completa: Al hacer clic, los usuarios acceden a una experiencia inmersiva dentro de Facebook.

Ventajas de los Anuncios de Colección

  1. Experiencia de Usuario Mejorada: Proporciona una experiencia de compra fluida y atractiva.
  2. Mayor Engagement: Los usuarios pueden interactuar con varios productos en un solo anuncio.
  3. Optimización para Móviles: Diseñados específicamente para dispositivos móviles, donde la mayoría de los usuarios de Facebook interactúan.
  4. Facilidad de Compra: Permite a los usuarios comprar productos directamente desde el anuncio.

Ejemplo de Anuncio de Colección

A continuación, se muestra un ejemplo de cómo se podría estructurar un anuncio de colección para una tienda de ropa:

{
  "name": "Colección de Verano",
  "ad_format": "collection",
  "main_media": {
    "type": "video",
    "url": "https://example.com/video.mp4"
  },
  "products": [
    {
      "title": "Vestido Floral",
      "image_url": "https://example.com/vestido-floral.jpg",
      "price": "$49.99",
      "product_url": "https://example.com/vestido-floral"
    },
    {
      "title": "Sombrero de Paja",
      "image_url": "https://example.com/sombrero-paja.jpg",
      "price": "$19.99",
      "product_url": "https://example.com/sombrero-paja"
    },
    {
      "title": "Sandalias de Cuero",
      "image_url": "https://example.com/sandalias-cuero.jpg",
      "price": "$29.99",
      "product_url": "https://example.com/sandalias-cuero"
    }
  ]
}

Explicación del Código

  • name: Nombre de la campaña.
  • ad_format: Tipo de anuncio, en este caso, "collection".
  • main_media: Elemento visual principal, puede ser una imagen o un video.
  • products: Lista de productos que se mostrarán en el anuncio, cada uno con su título, URL de la imagen, precio y URL del producto.

Creación de un Anuncio de Colección en Facebook Ads Manager

Paso a Paso:

  1. Accede al Administrador de Anuncios: Inicia sesión en tu cuenta de Facebook Business y abre el Administrador de Anuncios.
  2. Crea una Nueva Campaña: Selecciona "Crear" y elige el objetivo de tu campaña, como "Ventas del Catálogo" o "Tráfico".
  3. Configura el Conjunto de Anuncios: Define tu audiencia, presupuesto y calendario.
  4. Selecciona el Formato de Anuncio: Elige "Colección" como formato de anuncio.
  5. Añade el Contenido Principal: Sube la imagen o video principal que deseas usar.
  6. Añade los Productos: Selecciona los productos que deseas mostrar en tu anuncio de colección. Puedes hacerlo manualmente o importar desde tu catálogo de productos.
  7. Revisa y Publica: Revisa todos los detalles de tu anuncio y haz clic en "Publicar".

Ejercicio Práctico

Ejercicio: Crear un Anuncio de Colección

Objetivo: Crear un anuncio de colección para una tienda de electrónica que destaque varios productos.

  1. Imagen o Video Principal: Selecciona una imagen o video que represente la colección de productos.
  2. Productos: Elige al menos 3 productos para incluir en el anuncio.
  3. Configura el Anuncio: Utiliza el Administrador de Anuncios de Facebook para configurar y publicar tu anuncio de colección.

Solución:

  1. Imagen o Video Principal: Video promocional de la nueva colección de gadgets.
  2. Productos:
    • Smartphone XYZ
    • Tablet ABC
    • Auriculares DEF
{
  "name": "Colección de Gadgets",
  "ad_format": "collection",
  "main_media": {
    "type": "video",
    "url": "https://example.com/gadgets-video.mp4"
  },
  "products": [
    {
      "title": "Smartphone XYZ",
      "image_url": "https://example.com/smartphone-xyz.jpg",
      "price": "$699.99",
      "product_url": "https://example.com/smartphone-xyz"
    },
    {
      "title": "Tablet ABC",
      "image_url": "https://example.com/tablet-abc.jpg",
      "price": "$499.99",
      "product_url": "https://example.com/tablet-abc"
    },
    {
      "title": "Auriculares DEF",
      "image_url": "https://example.com/auriculares-def.jpg",
      "price": "$199.99",
      "product_url": "https://example.com/auriculares-def"
    }
  ]
}

Conclusión

Los anuncios de colección son una excelente manera de mostrar múltiples productos y proporcionar una experiencia de compra inmersiva para los usuarios. Al dominar este formato, puedes aumentar significativamente el engagement y las conversiones de tus campañas publicitarias en Facebook. Asegúrate de seguir experimentando y optimizando tus anuncios para obtener los mejores resultados posibles.
