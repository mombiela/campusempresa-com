Introducción

Las operaciones de entrada y salida (I/O) son fundamentales en cualquier lenguaje de programación, incluido Fortran. Permiten que los programas interactúen con los usuarios y otros sistemas leyendo datos de y escribiendo datos en diversas fuentes como la consola, archivos y más.

Entrada y Salida Básica

Lectura desde la Consola

Fortran utiliza la instrucción READ para tomar entrada del usuario. La forma más simple lee desde la entrada estándar (usualmente el teclado).

PROGRAM ReadExample
  IMPLICIT NONE
  INTEGER :: num

  PRINT *, 'Ingrese un número:'
  READ *, num
  PRINT *, 'Usted ingresó:', num
END PROGRAM ReadExample
  • PRINT * se usa para mostrar un mensaje en la consola.
  • READ * lee la entrada de la consola y la almacena en la variable num.

Escritura en la Consola

La instrucción PRINT se utiliza para enviar datos a la consola.

PROGRAM PrintExample
  IMPLICIT NONE
  INTEGER :: num

  num = 10
  PRINT *, 'El número es:', num
END PROGRAM PrintExample
  • PRINT *, es seguido por el mensaje y las variables a imprimir.

Entrada y Salida Avanzada

Entrada y Salida Formateada

Para tener más control sobre el formato de la entrada y salida, Fortran proporciona I/O formateada utilizando las instrucciones READ y WRITE con especificadores de formato.

PROGRAM FormattedIO
  IMPLICIT NONE
  INTEGER :: num
  REAL :: value

  PRINT *, 'Ingrese un número entero y un número real:'
  READ '(I5, F10.2)', num, value
  PRINT '(A, I5, A, F10.2)', 'Entero:', num, ' Real:', value
END PROGRAM FormattedIO
  • READ '(I5, F10.2)' especifica el formato para la lectura: un entero de hasta 5 dígitos y un número real de hasta 10 dígitos y 2 decimales.
  • PRINT '(A, I5, A, F10.2)' especifica el formato para la impresión: una cadena, un entero de hasta 5 dígitos, otra cadena y un número real de hasta 10 dígitos y 2 decimales.

Entrada y Salida de Archivos

Fortran permite leer de y escribir en archivos utilizando las instrucciones OPEN, READ, WRITE y CLOSE.

PROGRAM FileIO
  IMPLICIT NONE
  INTEGER :: num
  CHARACTER(LEN=100) :: line

  ! Abrir un archivo para escritura
  OPEN(UNIT=10, FILE='output.txt', STATUS='REPLACE')
  WRITE(10, *) 'Esta es una línea de texto.'
  WRITE(10, *) 12345
  CLOSE(10)

  ! Abrir un archivo para lectura
  OPEN(UNIT=10, FILE='output.txt', STATUS='OLD')
  READ(10, '(A)') line
  PRINT *, 'Primera línea:', line
  READ(10, *) num
  PRINT *, 'Segunda línea (número):', num
  CLOSE(10)
END PROGRAM FileIO
  • OPEN abre un archivo con un número de unidad especificado.
  • WRITE escribe datos en el archivo asociado con el número de unidad.
  • READ lee datos del archivo asociado con el número de unidad.
  • CLOSE cierra el archivo asociado con el número de unidad.

Manejo de Errores en I/O

Fortran proporciona mecanismos para manejar errores durante las operaciones de I/O utilizando los especificadores IOSTAT y ERR.

PROGRAM IOErrorHandling
  IMPLICIT NONE
  INTEGER :: num, io_status

  OPEN(UNIT=10, FILE='nonexistent.txt', STATUS='OLD', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error al abrir el archivo. IOSTAT =', io_status
  ELSE
    READ(10, *, IOSTAT=io_status) num
    IF (io_status /= 0) THEN
      PRINT *, 'Error al leer el archivo. IOSTAT =', io_status
    ELSE
      PRINT *, 'Número leído del archivo:', num
    END IF
    CLOSE(10)
  END IF
END PROGRAM IOErrorHandling
  • IOSTAT captura el estado de la operación de I/O.
  • ERR se puede usar para especificar una etiqueta a la que saltar en caso de error.

Resumen

En esta sección, cubrimos los aspectos básicos y avanzados de la entrada y salida en Fortran. Aprendimos cómo:

  • Leer de y escribir en la consola.
  • Usar I/O formateada para un mejor control sobre la representación de datos.
  • Realizar operaciones de I/O con archivos.
  • Manejar errores durante las operaciones de I/O.

Entender estos conceptos es crucial para desarrollar programas robustos en Fortran que puedan interactuar efectivamente con los usuarios y otros sistemas.
