¿Qué es Fortran?
Configuración del Entorno de Desarrollo ►

Fortran, abreviatura de "Formula Translation" (Traducción de Fórmulas), es un lenguaje de programación de alto nivel que fue desarrollado en la década de 1950 por IBM. Es uno de los lenguajes de programación más antiguos y está particularmente bien adaptado para el cálculo numérico y la computación científica.

Historia de Fortran

  • Años 1950: Desarrollado por IBM, Fortran fue el primer lenguaje de programación de alto nivel.
  • Años 1960-1970: Se introdujeron Fortran II, Fortran IV y Fortran 66, añadiendo más características y mejorando el lenguaje.
  • Años 1980: Fortran 77 se convirtió en el estándar, introduciendo constructos de programación estructurada.
  • Años 1990: Fortran 90 y Fortran 95 añadieron características modernas de programación como la programación de arrays y la modularidad.
  • Años 2000-Presente: Fortran 2003, Fortran 2008 y Fortran 2018 han continuado evolucionando, añadiendo programación orientada a objetos y otras características avanzadas.

Características Clave de Fortran

  • Cálculo Numérico: Optimizado para cálculos matemáticos y científicos.
  • Manejo de Arrays: Manipulación eficiente de arrays.
  • Portabilidad: El código puede ejecutarse en diferentes tipos de sistemas informáticos con cambios mínimos.
  • Rendimiento: Alto rendimiento para tareas computacionales.
  • Código Legado: Amplias bibliotecas y bases de código que se han desarrollado durante décadas.

Sintaxis y Estructura Básica

El código Fortran se organiza en programas y subprogramas. Aquí hay un ejemplo simple de un programa Fortran:

program HelloWorld
  print *, 'Hello, World!'
end program HelloWorld

Explicación:

  • program HelloWorld: Declara el inicio de un programa llamado HelloWorld.
  • print *, 'Hello, World!': Imprime la cadena 'Hello, World!' en la consola.
  • end program HelloWorld: Marca el final del programa.

Tipos de Datos

Fortran soporta varios tipos de datos, incluyendo:

  • Integer: Números enteros.
  • Real: Números de punto flotante.
  • Complex: Números complejos.
  • Character: Cadenas de texto.
  • Logical: Valores booleanos (verdadero o falso).

Ejemplo:

program DataTypes
  integer :: i = 10
  real :: x = 3.14
  complex :: z = (1.0, 2.0)
  character(len=20) :: str = 'Fortran'
  logical :: flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Complex:', z
  print *, 'Character:', str
  print *, 'Logical:', flag
end program DataTypes

Estructuras de Control

Fortran incluye estructuras de control estándar como:

  • If Statements: Ejecución condicional.
  • Loops: Repetición de bloques de código (por ejemplo, bucles do).

Ejemplo:

program ControlStructures
  integer :: i

  ! If statement
  if (i > 0) then
    print *, 'i es positivo'
  else
    print *, 'i no es positivo'
  end if

  ! Do loop
  do i = 1, 5
    print *, 'Iteración:', i
  end do
end program ControlStructures

Arrays

Fortran sobresale en el manejo de arrays, permitiendo operaciones eficientes y concisas sobre arrays.

Ejemplo:

program Arrays
  integer, dimension(5) :: arr = [1, 2, 3, 4, 5]
  integer :: i

  ! Imprimir elementos del array
  do i = 1, 5
    print *, 'Elemento', i, ':', arr(i)
  end do
end program Arrays

Conclusión

Fortran es un lenguaje poderoso para la computación numérica y científica, con una rica historia y un fuerte enfoque en el rendimiento y el manejo de arrays. Comprender su sintaxis básica, tipos de datos, estructuras de control y operaciones con arrays proporciona una base sólida para una exploración más profunda y programación avanzada en Fortran.
