Escritura en Archivos en Fortran
Posicionamiento de Archivos ►

Escribir en archivos es una habilidad fundamental en la programación en Fortran, permitiéndote guardar datos para uso posterior, compartir resultados o registrar información. Este tema te guiará desde las técnicas básicas hasta las avanzadas de escritura en archivos en Fortran.

Escritura Básica en Archivos

Abrir un Archivo

Para escribir en un archivo, primero necesitas abrirlo usando la instrucción OPEN. Esta instrucción asocia un número de unidad con un archivo.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
END PROGRAM WriteToFile
  • UNIT: Un identificador único para el archivo.
  • FILE: El nombre del archivo.
  • STATUS: Especifica el estado del archivo (NEW, OLD, REPLACE, etc.).

Escribir Datos

Usa la instrucción WRITE para escribir datos en el archivo.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteToFile
  • WRITE(unit_number, *): Escribe en el archivo asociado con unit_number.
  • CLOSE(unit_number): Cierra el archivo.

Ejemplo: Escribir Múltiples Líneas

PROGRAM WriteMultipleLines
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Line 1: Hello, Fortran!'
  WRITE(unit_number, *) 'Line 2: Writing to files is easy.'
  CLOSE(unit_number)
END PROGRAM WriteMultipleLines

Escritura Intermedia en Archivos

Formatear la Salida

Puedes formatear la salida usando especificadores de formato.

PROGRAM WriteFormatted
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') 'Value: ', 42
  CLOSE(unit_number)
END PROGRAM WriteFormatted
  • '(A, I3)': Especificador de formato donde A es para cadena y I3 es para un entero con un ancho de 3.

Escribir Arreglos

También puedes escribir arreglos en archivos.

PROGRAM WriteArray
  INTEGER :: unit_number
  INTEGER, DIMENSION(5) :: array = (/1, 2, 3, 4, 5/)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(5I3)') array
  CLOSE(unit_number)
END PROGRAM WriteArray
  • '(5I3)': Escribe 5 enteros, cada uno con un ancho de 3.

Escritura Avanzada en Archivos

Manejo de Errores

Usa el parámetro IOSTAT para manejar errores durante las operaciones de archivo.

PROGRAM WriteWithErrorHandling
  INTEGER :: unit_number, io_status
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file!'
    STOP
  END IF
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteWithErrorHandling
  • IOSTAT=io_status: Almacena el estado de la operación de E/S.

Añadir a Archivos

Para añadir datos a un archivo existente, usa las opciones STATUS='OLD' y POSITION='APPEND'.

PROGRAM AppendToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='OLD', POSITION='APPEND')
  WRITE(unit_number, *) 'Appending this line.'
  CLOSE(unit_number)
END PROGRAM AppendToFile

Escribir Estructuras de Datos Complejas

Para estructuras de datos más complejas, como tipos derivados, necesitas escribir cada componente individualmente.

MODULE DataModule
  TYPE :: Person
    CHARACTER(LEN=20) :: name
    INTEGER :: age
  END TYPE Person
END MODULE DataModule

PROGRAM WriteComplexData
  USE DataModule
  INTEGER :: unit_number
  TYPE(Person) :: person
  person = Person('John Doe', 30)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') person%name, person%age
  CLOSE(unit_number)
END PROGRAM WriteComplexData

Conclusión

Escribir en archivos en Fortran implica varios pasos: abrir el archivo, escribir datos y cerrar el archivo. Puedes formatear la salida, manejar errores, añadir a archivos y escribir estructuras de datos complejas. Dominar estas técnicas te permitirá gestionar eficazmente la salida de datos en tus programas de Fortran.
