Introducción a los Métodos Numéricos

Los métodos numéricos son técnicas utilizadas para resolver problemas matemáticos numéricamente, es decir, utilizando números y algoritmos en lugar de manipulaciones simbólicas. Esta sección te introducirá a los conceptos básicos de los métodos numéricos y cómo pueden ser implementados en Fortran.

  • Definición: Los métodos numéricos implican algoritmos para resolver problemas matemáticos.
  • Aplicaciones: Ingeniería, física, finanzas y otros campos que requieren soluciones numéricas.

Sintaxis y Estructuras Básicas de Fortran

Antes de profundizar en los métodos numéricos, es esencial entender la sintaxis y las estructuras básicas de Fortran.

  • Variables y Tipos de Datos: INTEGER, REAL, DOUBLE PRECISION, COMPLEX, etc.
  • Estructuras de Control: Sentencias IF, bucles DO, etc.
  • Funciones y Subrutinas: Modularizar el código para su reutilización.

Ejemplo: Programa Básico en Fortran

PROGRAM HelloWorld
  PRINT *, '¡Hola, Mundo!'
END PROGRAM HelloWorld

Métodos de Búsqueda de Raíces

Los métodos de búsqueda de raíces se utilizan para encontrar las raíces de ecuaciones. Los métodos comunes incluyen el Método de Bisección y el Método de Newton-Raphson.

Método de Bisección

El Método de Bisección es un método simple y robusto para encontrar raíces de una función.

  • Algoritmo:
    1. Elegir dos puntos \(a\) y \(b\) tales que \(f(a)\) y \(f(b)\) tengan signos opuestos.
    2. Calcular el punto medio \(c = (a + b) / 2\).
    3. Verificar el signo de \(f(c)\).
    4. Reemplazar \(a\) o \(b\) con \(c\) según el signo.
    5. Repetir hasta que se encuentre la raíz dentro de una tolerancia deseada.

Ejemplo: Método de Bisección en Fortran

PROGRAM BisectionMethod
  IMPLICIT NONE
  REAL :: a, b, c, tol
  INTEGER :: max_iter, iter

  a = 1.0
  b = 2.0
  tol = 1.0e-6
  max_iter = 100

  DO iter = 1, max_iter
    c = (a + b) / 2.0
    IF (ABS(f(c)) < tol) EXIT
    IF (f(a) * f(c) < 0.0) THEN
      b = c
    ELSE
      a = c
    END IF
  END DO

  PRINT *, 'La raíz es aproximadamente: ', c

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2 - 2.0
  END FUNCTION f
END PROGRAM BisectionMethod

Integración Numérica

La integración numérica implica aproximar la integral de una función. Los métodos comunes incluyen la Regla del Trapecio y la Regla de Simpson.

Regla del Trapecio

La Regla del Trapecio aproxima la integral dividiendo el área bajo la curva en trapecios.

  • Algoritmo:
    1. Dividir el intervalo \([a, b]\) en \(n\) subintervalos.
    2. Calcular la suma de las áreas de los trapecios.

Ejemplo: Regla del Trapecio en Fortran

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i * h)
  END DO

  integral = integral * h
  PRINT *, 'Integral aproximada: ', integral

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2
  END FUNCTION f
END PROGRAM TrapezoidalRule

Resolución de Sistemas de Ecuaciones Lineales

Resolver sistemas de ecuaciones lineales es un problema fundamental en los métodos numéricos. Las técnicas incluyen Eliminación Gaussiana y Descomposición LU.

Eliminación Gaussiana

La Eliminación Gaussiana es un método para resolver sistemas lineales transformando la matriz del sistema en una forma triangular superior.

  • Algoritmo:
    1. Eliminación hacia adelante para formar una matriz triangular superior.
    2. Sustitución hacia atrás para resolver las variables.

Ejemplo: Eliminación Gaussiana en Fortran

PROGRAM GaussianElimination
  IMPLICIT NONE
  REAL, DIMENSION(3,3) :: A
  REAL, DIMENSION(3) :: b, x
  INTEGER :: i, j, k

  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [1.0, 0.0, 4.0]

  CALL gauss_elimination(A, b, x)

  PRINT *, 'Solución: ', x

CONTAINS
  SUBROUTINE gauss_elimination(A, b, x)
    REAL, DIMENSION(:,:), INTENT(INOUT) :: A
    REAL, DIMENSION(:), INTENT(INOUT) :: b
    REAL, DIMENSION(:), INTENT(OUT) :: x
    INTEGER :: n, i, j, k
    REAL :: factor

    n = SIZE(b)
    DO k = 1, n-1
      DO i = k+1, n
        factor = A(i,k) / A(k,k)
        A(i,k:n) = A(i,k:n) - factor * A(k,k:n)
        b(i) = b(i) - factor * b(k)
      END DO
    END DO

    x(n) = b(n) / A(n,n)
    DO i = n-1, 1, -1
      x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
    END DO
  END SUBROUTINE gauss_elimination
END PROGRAM GaussianElimination

Conclusión

Los métodos numéricos son herramientas esenciales para resolver una amplia gama de problemas matemáticos. En este tema, cubrimos:

  • Sintaxis y estructuras básicas de Fortran.
  • Métodos de búsqueda de raíces como el Método de Bisección.
  • Integración numérica usando la Regla del Trapecio.
  • Resolución de sistemas de ecuaciones lineales con Eliminación Gaussiana.
