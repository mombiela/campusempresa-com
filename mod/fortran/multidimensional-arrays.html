Arrays multidimensionales en Fortran
Manejo de Cadenas ►

Los arrays multidimensionales son un concepto fundamental en Fortran, que te permite almacenar y manipular datos en más de una dimensión. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de arrays multidimensionales en Fortran.

Introducción a los Arrays Multidimensionales

Los arrays multidimensionales en Fortran son arrays con más de un índice. Se utilizan para representar datos en forma de cuadrícula o matriz, lo cual es común en cálculos científicos e ingenieriles.

Conceptos Clave

  • Dimensiones: El número de índices necesarios para acceder a un elemento en el array.
  • Rango: El número de dimensiones que tiene un array.
  • Forma: El tamaño de cada dimensión.

Declaración de Arrays Multidimensionales

Para declarar un array multidimensional en Fortran, especificas las dimensiones entre paréntesis después del nombre del array.

Ejemplo: Declaración de un Array 2D

PROGRAM TwoDArray
  IMPLICIT NONE
  INTEGER, DIMENSION(3, 4) :: matrix
END PROGRAM TwoDArray

En este ejemplo, matrix es un array 2D con 3 filas y 4 columnas.

Ejemplo: Declaración de un Array 3D

PROGRAM ThreeDArray
  IMPLICIT NONE
  REAL, DIMENSION(2, 3, 4) :: tensor
END PROGRAM ThreeDArray

Aquí, tensor es un array 3D con dimensiones 2x3x4.

Inicialización de Arrays Multidimensionales

Puedes inicializar arrays multidimensionales en el momento de la declaración o más tarde en el programa.

Ejemplo: Inicialización en la Declaración

PROGRAM InitArray
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArray

En este ejemplo, la función RESHAPE se utiliza para inicializar el matrix con valores.

Ejemplo: Inicialización en el Programa

PROGRAM InitArrayInProgram
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArrayInProgram

Acceso a Elementos en Arrays Multidimensionales

Los elementos en un array multidimensional se acceden utilizando sus índices.

Ejemplo: Acceso a Elementos

PROGRAM AccessElements
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
  PRINT *, 'Elemento en (1,1): ', matrix(1, 1)
  PRINT *, 'Elemento en (2,2): ', matrix(2, 2)
END PROGRAM AccessElements

Este programa imprime los elementos en las posiciones (1,1) y (2,2) del matrix.

Operaciones en Arrays Multidimensionales

Puedes realizar varias operaciones en arrays multidimensionales, como adición, sustracción y multiplicación.

Ejemplo: Adición de Arrays

PROGRAM ArrayAddition
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix1, matrix2, result
  matrix1 = RESHAPE([1, 2, 3, 4], SHAPE(matrix1))
  matrix2 = RESHAPE([5, 6, 7, 8], SHAPE(matrix2))
  result = matrix1 + matrix2
  PRINT *, 'Resultado de la adición:'
  PRINT *, result
END PROGRAM ArrayAddition

Este programa suma dos arrays 2D e imprime el resultado.

Temas Avanzados

Asignación Dinámica

Fortran permite la asignación dinámica de arrays, lo cual es útil cuando el tamaño del array no se conoce en tiempo de compilación.

Ejemplo: Asignación Dinámica

PROGRAM DynamicAllocation
  IMPLICIT NONE
  INTEGER, ALLOCATABLE, DIMENSION(:,:) :: matrix
  INTEGER :: rows, cols
  PRINT *, 'Introduce el número de filas y columnas:'
  READ *, rows, cols
  ALLOCATE(matrix(rows, cols))
  matrix = 0  ! Inicializar todos los elementos a cero
  PRINT *, 'Matriz después de la asignación:'
  PRINT *, matrix
  DEALLOCATE(matrix)
END PROGRAM DynamicAllocation

Este programa asigna dinámicamente un array 2D basado en la entrada del usuario.

Secciones de Arrays

Puedes trabajar con secciones de arrays, lo cual es útil para manipular subarrays.

Ejemplo: Secciones de Arrays

PROGRAM ArraySections
  IMPLICIT NONE
  INTEGER, DIMENSION(4, 4) :: matrix
  INTEGER :: i, j
  ! Inicializar la matriz
  DO i = 1, 4
    DO j = 1, 4
      matrix(i, j) = i * 10 + j
    END DO
  END DO
  PRINT *, 'Matriz original:'
  PRINT *, matrix
  PRINT *, 'Submatriz (2:3, 2:3):'
  PRINT *, matrix(2:3, 2:3)
END PROGRAM ArraySections

Este programa demuestra cómo acceder e imprimir una submatriz.

Conclusión

Los arrays multidimensionales son una característica poderosa en Fortran, que permite la manipulación y almacenamiento de datos complejos. Desde declaraciones e inicializaciones básicas hasta temas avanzados como la asignación dinámica y las secciones de arrays, entender los arrays multidimensionales es crucial para una programación efectiva en Fortran. Practica con los ejemplos y ejercicios proporcionados para solidificar tu comprensión.
