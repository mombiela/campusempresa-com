Configuración del Entorno de Desarrollo
Sintaxis y Estructura Básica ►

Configurar el entorno de desarrollo es un primer paso crucial para aprender Fortran. Esta sección te guiará a través del proceso de instalación de las herramientas necesarias y la configuración de tu sistema para escribir, compilar y ejecutar programas en Fortran.

1. Instalación de un Compilador de Fortran

Un compilador de Fortran es esencial para convertir tu código Fortran en programas ejecutables. Hay varios compiladores de Fortran populares disponibles:

  • GNU Fortran (gfortran): Parte de la Colección de Compiladores GNU (GCC), es de código abierto y ampliamente utilizado.
  • Intel Fortran Compiler (ifort): Conocido por sus optimizaciones de rendimiento, especialmente en hardware Intel.
  • NAG Fortran Compiler: Conocido por su fuerte cumplimiento de estándares y características de depuración.

Instalación de gfortran

En Windows

  1. Descargar e Instalar MinGW-w64:

    • Visita la página de descarga de MinGW-w64.
    • Elige el instalador apropiado para tu sistema (32-bit o 64-bit).
    • Sigue las instrucciones de instalación, asegurándote de seleccionar el compilador de Fortran durante la configuración.

  2. Verificar la Instalación:

    gfortran --version

    Este comando debería mostrar la versión de gfortran instalada.

En macOS

  1. Instalar Homebrew (si no está ya instalado):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instalar gfortran:

    brew install gcc

    Esto instalará gfortran junto con otros compiladores de GCC.

  3. Verificar la Instalación:

    gfortran --version

En Linux

  1. Instalar gfortran usando tu gestor de paquetes:

    • Debian/Ubuntu: 
      sudo apt-get update
sudo apt-get install gfortran
    • Fedora: 
      sudo dnf install gcc-gfortran

  2. Verificar la Instalación:

    gfortran --version

2. Configuración de un Entorno de Desarrollo Integrado (IDE)

Un IDE puede mejorar significativamente tu productividad al proporcionar características como resaltado de sintaxis, autocompletado de código y herramientas de depuración. Algunos IDEs populares para Fortran incluyen:

  • Visual Studio Code (VS Code): Un editor altamente personalizable y extensible.
  • Eclipse con Photran: Un poderoso IDE con un plugin dedicado a Fortran.
  • Code::Blocks: Un IDE gratuito y de código abierto que soporta Fortran.

Configuración de Visual Studio Code para Fortran

  1. Instalar Visual Studio Code:

    • Descarga e instala VS Code desde el sitio web oficial.

  2. Instalar Extensiones para Fortran:

    • Abre VS Code y ve a la vista de Extensiones haciendo clic en el icono de Extensiones en la Barra de Actividades en el lado de la ventana.
    • Busca "Modern Fortran" e instálalo.

  3. Configurar el Compilador de Fortran:

    • Abre la Paleta de Comandos (Ctrl+Shift+P) y escribe "Preferences: Open Settings (JSON)".
    • Añade la siguiente configuración para especificar la ruta a tu compilador gfortran: 
      {
  "fortran.compilerPath": "/path/to/gfortran"
}

  4. Crear y Ejecutar un Programa Simple en Fortran:

    • Crea un nuevo archivo con una extensión .f90 (por ejemplo, hello_world.f90).
    • Escribe un programa simple en Fortran: 
      program hello
  print *, "¡Hola, Fortran!"
end program hello
    • Guarda el archivo y abre la terminal en VS Code.
    • Compila el programa: 
      gfortran -o hello_world hello_world.f90
    • Ejecuta el ejecutable: 
      ./hello_world

3. Configuración de Variables de Entorno

Configurar variables de entorno puede ayudar a agilizar el proceso de desarrollo haciendo que las herramientas y bibliotecas sean fácilmente accesibles.

Añadir gfortran al PATH

En Windows

  1. Abrir Propiedades del Sistema:

    • Haz clic derecho en "Este PC" o "Equipo" en el escritorio o en el Explorador de Archivos.
    • Selecciona "Propiedades" y luego "Configuración avanzada del sistema".

  2. Editar Variables de Entorno:

    • Haz clic en "Variables de entorno".
    • Bajo "Variables del sistema", encuentra la variable Path y haz clic en "Editar".
    • Añade la ruta al directorio que contiene gfortran.exe (por ejemplo, C:\MinGW\bin).

  3. Verificar:

    • Abre un nuevo Símbolo del sistema y escribe: 
      gfortran --version

En macOS y Linux

  1. Editar el Archivo de Configuración del Shell:

    • Abre ~/.bashrc (o ~/.zshrc si usas Zsh) en un editor de texto.
    • Añade la siguiente línea: 
      export PATH="/path/to/gfortran:$PATH"

  2. Aplicar los Cambios:

    source ~/.bashrc

  3. Verificar:

    gfortran --version

Conclusión

Configurar tu entorno de desarrollo es el primer paso hacia convertirte en un programador competente en Fortran. Al instalar un compilador de Fortran, configurar un IDE y establecer variables de entorno, creas una base robusta para escribir, compilar y ejecutar programas en Fortran de manera eficiente. Con tu entorno listo, ahora puedes enfocarte en aprender el lenguaje y desarrollar tus habilidades de programación.
