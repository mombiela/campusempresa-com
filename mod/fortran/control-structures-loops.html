Introducción a los Bucles en Fortran

Los bucles son estructuras de control fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. En Fortran, hay varios tipos de bucles, cada uno adecuado para diferentes escenarios. Esta sección cubrirá las construcciones básicas de bucles en Fortran, incluyendo los bucles DO, DO WHILE y FORALL.

Bucle DO

El bucle DO es el bucle más común en Fortran. Repite un bloque de código un número especificado de veces.

Sintaxis

DO variable = start, end, step
    ! Código a ejecutar
END DO
  • variable: La variable de control del bucle.
  • start: El valor inicial de la variable de control del bucle.
  • end: El valor final de la variable de control del bucle.
  • step: El incremento (opcional, por defecto es 1).

Ejemplo

PROGRAM DoLoopExample
    INTEGER :: i
    DO i = 1, 5
        PRINT *, "Iteración: ", i
    END DO
END PROGRAM DoLoopExample

En este ejemplo, el bucle imprimirá el número de iteración del 1 al 5.

Bucles DO Anidados

Puedes anidar bucles DO para manejar iteraciones más complejas.

Ejemplo

PROGRAM NestedDoLoopExample
    INTEGER :: i, j
    DO i = 1, 3
        DO j = 1, 2
            PRINT *, "i = ", i, " j = ", j
        END DO
    END DO
END PROGRAM NestedDoLoopExample

Este ejemplo demuestra un bucle anidado donde el bucle interno se ejecuta dos veces por cada iteración del bucle externo.

Bucle DO WHILE

El bucle DO WHILE continúa ejecutándose mientras una condición especificada sea verdadera.

Sintaxis

DO WHILE (condition)
    ! Código a ejecutar
END DO

Ejemplo

PROGRAM DoWhileExample
    INTEGER :: i
    i = 1
    DO WHILE (i <= 5)
        PRINT *, "Iteración: ", i
        i = i + 1
    END DO
END PROGRAM DoWhileExample

En este ejemplo, el bucle se ejecutará mientras i sea menor o igual a 5.

Bucle FORALL

El bucle FORALL se utiliza para operaciones con arrays y puede ser más eficiente para ciertos tipos de cálculos.

Sintaxis

FORALL (index = start:end:step, condition)
    array(index) = expression
END FORALL

Ejemplo

PROGRAM ForallExample
    INTEGER, DIMENSION(5) :: array
    FORALL (i = 1:5)
        array(i) = i * i
    END FORALL
    PRINT *, array
END PROGRAM ForallExample

Este ejemplo inicializa un array con los cuadrados de sus índices.

Resumen

En esta sección, cubrimos las construcciones básicas de bucles en Fortran:

  • Bucle DO: Repite un bloque de código un número especificado de veces.
  • Bucles DO Anidados: Permite iteraciones más complejas.
  • Bucle DO WHILE: Continúa ejecutándose mientras una condición sea verdadera.
  • Bucle FORALL: Utilizado para operaciones eficientes con arrays.

Entender estos bucles te ayudará a escribir programas en Fortran más eficientes y legibles. Practica usando estos bucles con diferentes condiciones y rangos para familiarizarte más con su sintaxis y comportamiento.
