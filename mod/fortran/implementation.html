Introducción a la Implementación en Fortran

En esta sección, cubriremos cómo implementar programas en Fortran, desde escribir scripts simples hasta desarrollar aplicaciones más complejas. Comenzaremos con conceptos básicos y gradualmente avanzaremos a temas más avanzados, asegurándonos de que entiendas cada paso del camino.

Escribiendo Tu Primer Programa en Fortran

Comencemos con un simple programa de "Hola, Mundo!" para que te familiarices con la estructura básica de un programa en Fortran.

program hello
    print *, 'Hello, World!'
end program hello
  • program: Esta palabra clave inicia el programa.
  • print: Esta instrucción muestra texto en la pantalla.
  • end program: Esta palabra clave finaliza el programa.

Ejercicio: Modificar el Programa

Intenta modificar el programa para que imprima tu nombre en lugar de "Hello, World!".

program hello
    print *, 'Hello, [Tu Nombre]!'
end program hello

Variables y Tipos de Datos

Fortran soporta varios tipos de datos, incluyendo enteros, números reales y caracteres. Veamos cómo declarar y usar estas variables.

program variables
    integer :: i
    real :: x
    character(len=20) :: name

    i = 10
    x = 3.14
    name = 'Fortran'

    print *, 'Integer:', i
    print *, 'Real:', x
    print *, 'Character:', name
end program variables
  • integer: Declara una variable entera.
  • real: Declara una variable de punto flotante.
  • character: Declara una variable de cadena con una longitud especificada.

Ejercicio: Declarar e Imprimir Variables

Declara variables de diferentes tipos e imprime sus valores.

program my_variables
    integer :: age
    real :: height
    character(len=30) :: hobby

    age = 25
    height = 5.9
    hobby = 'Programming'

    print *, 'Age:', age
    print *, 'Height:', height
    print *, 'Hobby:', hobby
end program my_variables

Estructuras de Control

Las estructuras de control como bucles y condicionales son esenciales para implementar lógica en tus programas.

Instrucciones If

program if_example
    integer :: num

    num = 10

    if (num > 5) then
        print *, 'Number is greater than 5'
    else
        print *, 'Number is 5 or less'
    end if
end program if_example

Bucles Do

program loop_example
    integer :: i

    do i = 1, 5
        print *, 'Iteration:', i
    end do
end program loop_example

Ejercicio: Implementar Estructuras de Control

Escribe un programa que use tanto instrucciones if como bucles do.

program control_structures
    integer :: i

    do i = 1, 10
        if (mod(i, 2) == 0) then
            print *, i, 'is even'
        else
            print *, i, 'is odd'
        end if
    end do
end program control_structures

Funciones y Subrutinas

Las funciones y subrutinas te permiten modularizar tu código, haciéndolo más legible y reutilizable.

Funciones

program function_example
    real :: result

    result = square(4.0)
    print *, 'Square of 4.0 is', result

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square
end program function_example

Subrutinas

program subroutine_example
    real :: a, b, result

    a = 3.0
    b = 4.0
    call add(a, b, result)
    print *, 'Sum of', a, 'and', b, 'is', result

contains
    subroutine add(x, y, sum)
        real, intent(in) :: x, y
        real, intent(out) :: sum
        sum = x + y
    end subroutine add
end program subroutine_example

Ejercicio: Crear Funciones y Subrutinas

Crea un programa que incluya tanto una función como una subrutina.

program func_sub_example
    real :: num, sq, sum

    num = 5.0
    sq = square(num)
    call add(num, sq, sum)

    print *, 'Square of', num, 'is', sq
    print *, 'Sum of', num, 'and its square is', sum

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square

    subroutine add(x, y, result)
        real, intent(in) :: x, y
        real, intent(out) :: result
        result = x + y
    end subroutine add
end program func_sub_example

Temas Avanzados

Módulos

Los módulos se utilizan para agrupar procedimientos y datos relacionados.

module math_operations
    contains
        real function square(x)
            real, intent(in) :: x
            square = x * x
        end function square

        subroutine add(x, y, sum)
            real, intent(in) :: x, y
            real, intent(out) :: sum
            sum = x + y
        end subroutine add
end module math_operations

program use_module
    use math_operations
    real :: a, b, result

    a = 2.0
    b = 3.0
    result = square(a)
    print *, 'Square of', a, 'is', result

    call add(a, b, result)
    print *, 'Sum of', a, 'and', b, 'is', result
end program use_module

Ejercicio: Crear y Usar Módulos

Crea un módulo con al menos dos funciones y úsalo en un programa.

module my_module
    contains
        real function cube(x)
            real, intent(in) :: x
            cube = x * x * x
        end function cube

        subroutine multiply(x, y, product)
            real, intent(in) :: x, y
            real, intent(out) :: product
            product = x * y
        end subroutine multiply
end module my_module

program use_my_module
    use my_module
    real :: num, cubed, product

    num = 3.0
    cubed = cube(num)
    call multiply(num, cubed, product)

    print *, 'Cube of', num, 'is', cubed
    print *, 'Product of', num, 'and its cube is', product
end program use_my_module

Conclusión

En esta sección, cubrimos los conceptos básicos de la implementación de programas en Fortran, incluyendo la escritura de programas simples, el uso de variables y tipos de datos, la implementación de estructuras de control y la creación de funciones y subrutinas. También tocamos temas avanzados como los módulos. Practicando estos conceptos, estarás en buen camino para convertirte en un programador competente en Fortran.
