Operaciones con Archivos Binarios en Fortran
Módulos y Paquetes ►

Las operaciones con archivos binarios son cruciales para manejar eficientemente grandes conjuntos de datos y realizar manipulaciones de datos a bajo nivel. Esta sección te guiará desde las técnicas básicas hasta las avanzadas de operaciones con archivos binarios en Fortran.

Introducción a los Archivos Binarios

  • Archivos Binarios: A diferencia de los archivos de texto, los archivos binarios almacenan datos en un formato que no es legible por humanos. Se utilizan para almacenar datos como imágenes, audio y programas compilados.
  • Ventajas:
    • Operaciones de lectura/escritura más rápidas.
    • Almacenamiento más compacto.
    • Acceso directo a los datos.

Apertura y Cierre de Archivos Binarios

Para trabajar con archivos binarios en Fortran, necesitas abrir el archivo en un modo específico y cerrarlo después de completar las operaciones.

Apertura de un Archivo Binario

Usa la declaración OPEN para abrir un archivo binario.

OPEN(UNIT=10, FILE='data.bin', STATUS='UNKNOWN', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)
  • UNIT: Número de unidad lógica.
  • FILE: Nombre del archivo.
  • STATUS: Estado del archivo ('NEW', 'OLD', 'UNKNOWN').
  • ACCESS: Modo de acceso ('SEQUENTIAL', 'DIRECT').
  • FORM: Formato del archivo ('FORMATTED', 'UNFORMATTED').
  • RECL: Longitud del registro (en bytes).

Cierre de un Archivo Binario

Usa la declaración CLOSE para cerrar el archivo.

CLOSE(UNIT=10)

Escritura en un Archivo Binario

Para escribir datos en un archivo binario, usa la declaración WRITE con el número de unidad apropiado.

Ejemplo: Escritura de Enteros en un Archivo Binario

PROGRAM WriteBinary
  INTEGER :: i, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    WRITE(UNIT=unit_num, REC=i) i
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM WriteBinary
  • REC: Número de registro para acceso directo.

Lectura de un Archivo Binario

Para leer datos de un archivo binario, usa la declaración READ con el número de unidad apropiado.

Ejemplo: Lectura de Enteros de un Archivo Binario

PROGRAM ReadBinary
  INTEGER :: i, value, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i) value
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ReadBinary

Operaciones Avanzadas con Archivos Binarios

Manejo de Diferentes Tipos de Datos

Fortran te permite escribir y leer varios tipos de datos en/desde archivos binarios.

Ejemplo: Escritura y Lectura de Números Reales

PROGRAM WriteReadReal
  REAL :: x, y
  INTEGER :: unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  x = 3.14
  WRITE(UNIT=unit_num, REC=1) x

  CLOSE(UNIT=unit_num)

  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  READ(UNIT=unit_num, REC=1) y
  PRINT *, 'Read value:', y

  CLOSE(UNIT=unit_num)
END PROGRAM WriteReadReal

Manejo de Errores

El manejo adecuado de errores asegura que tu programa pueda manejar de manera elegante problemas como archivo no encontrado o errores de lectura/escritura.

Ejemplo: Uso de IOSTAT para Manejo de Errores

PROGRAM ErrorHandling
  INTEGER :: i, unit_num, io_status

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4, IOSTAT=io_status)

  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file. IOSTAT =', io_status
    STOP
  END IF

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i, IOSTAT=io_status) value
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading record', i, '. IOSTAT =', io_status
      EXIT
    END IF
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ErrorHandling

Resumen

Las operaciones con archivos binarios en Fortran son esenciales para el manejo eficiente de datos. Esta sección cubrió:

  • Conceptos básicos de archivos binarios.
  • Apertura y cierre de archivos.
  • Escritura y lectura de diferentes tipos de datos.
  • Técnicas avanzadas y manejo de errores.

Al dominar estas operaciones, puedes mejorar significativamente el rendimiento y las capacidades de tus programas en Fortran.
