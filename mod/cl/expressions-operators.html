Introducción

Las expresiones y los operadores son componentes fundamentales en la programación del Lenguaje de Control (CL). Te permiten realizar cálculos, tomar decisiones y manipular datos. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de expresiones y operadores en CL.

Conceptos Básicos

¿Qué es una Expresión?

  • Una expresión es una combinación de variables, constantes, operadores y funciones que se evalúan para producir un valor.
  • Ejemplo: 3 + 5 es una expresión que se evalúa como 8.

Tipos de Operadores

Los operadores son símbolos especiales o palabras clave que se utilizan para realizar operaciones sobre variables y valores. Se pueden categorizar en:

  • Operadores Aritméticos
  • Operadores de Comparación
  • Operadores Lógicos
  • Operadores de Asignación

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

| Operador | Descripción | Ejemplo | Resultado | |----------|-------------|---------|-----------| | + | Adición | 5 + 3 | 8 | | - | Sustracción | 5 - 3 | 2 | | * | Multiplicación | 5 * 3 | 15 | | / | División | 6 / 3 | 2 | | % | Módulo | 5 % 3 | 2 |

Ejemplo

DCL VAR(&RESULT) TYPE(*DEC) LEN(5 2)
CHGVAR VAR(&RESULT) VALUE(5 + 3)

En este ejemplo, &RESULT se establecerá en 8.

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores.

| Operador | Descripción | Ejemplo | Resultado | |----------|-------------|---------|-----------| | = | Igual a | 5 = 3 | Falso | | <> | No igual a | 5 <> 3| Verdadero| | > | Mayor que | 5 > 3 | Verdadero| | < | Menor que | 5 < 3 | Falso | | >= | Mayor o igual que | 5 >= 3 | Verdadero | | <= | Menor o igual que | 5 <= 3 | Falso |

Ejemplo

DCL VAR(&IS_EQUAL) TYPE(*LGL)
CHGVAR VAR(&IS_EQUAL) VALUE(5 = 3)

En este ejemplo, &IS_EQUAL se establecerá en *FALSE.

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

| Operador | Descripción | Ejemplo | Resultado | |----------|-------------|---------|-----------| | & | Y | *TRUE & *FALSE | *FALSE | | | | O | *TRUE | *FALSE | *TRUE | | ! | NO | ! *TRUE | *FALSE |

Ejemplo

DCL VAR(&RESULT) TYPE(*LGL)
CHGVAR VAR(&RESULT) VALUE((*TRUE & *FALSE) | *TRUE)

En este ejemplo, &RESULT se establecerá en *TRUE.

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

| Operador | Descripción | Ejemplo | Resultado | |----------|-------------|---------|-----------| | = | Asignar | VAR = 5 | VAR se establece en 5 |

Ejemplo

DCL VAR(&NUMBER) TYPE(*DEC) LEN(5 0)
CHGVAR VAR(&NUMBER) VALUE(10)

En este ejemplo, &NUMBER se establecerá en 10.

Uso Avanzado

Combinando Operadores

Puedes combinar diferentes tipos de operadores en una sola expresión para realizar cálculos y evaluaciones complejas.

Ejemplo

DCL VAR(&RESULT) TYPE(*DEC) LEN(5 2)
CHGVAR VAR(&RESULT) VALUE((5 + 3) * 2 / (4 - 2))

En este ejemplo, &RESULT se establecerá en 8.00.

Usando Funciones en Expresiones

También puedes usar funciones integradas dentro de expresiones para realizar operaciones más avanzadas.

Ejemplo

DCL VAR(&DATE) TYPE(*CHAR) LEN(8)
CHGVAR VAR(&DATE) VALUE(%DATE())

En este ejemplo, &DATE se establecerá en la fecha actual del sistema.

Conclusión

Las expresiones y los operadores son herramientas esenciales en la programación de CL. Te permiten realizar cálculos, tomar decisiones y manipular datos de manera efectiva. Al dominar estos conceptos, puedes escribir programas CL más eficientes y potentes. Practica combinando diferentes operadores y usando funciones dentro de tus expresiones para volverte competente en la programación de CL.
