Introducción

Las declaraciones condicionales son fundamentales en la programación ya que permiten la ejecución de código basado en condiciones específicas. En CL (Control Language), las declaraciones condicionales ayudan a controlar el flujo de un programa tomando decisiones. Esta sección cubrirá desde los conceptos básicos hasta el uso avanzado de las declaraciones condicionales en CL.

Declaraciones condicionales básicas

Declaración IF

La declaración IF se utiliza para ejecutar un bloque de código si una condición especificada es verdadera.

Sintaxis

IF COND(logical-expression) THEN(DO)
    /* Código a ejecutar si la condición es verdadera */
ENDDO

Ejemplo

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO

En este ejemplo, se envía el mensaje "You are an adult." si la variable &AGE es 18 o mayor.

Declaración ELSE

La declaración ELSE se utiliza para ejecutar un bloque de código si la condición en la declaración IF es falsa.

Sintaxis

IF COND(logical-expression) THEN(DO)
    /* Código a ejecutar si la condición es verdadera */
ENDDO
ELSE DO
    /* Código a ejecutar si la condición es falsa */
ENDDO

Ejemplo

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(16)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are a minor.')
ENDDO

En este ejemplo, se envía el mensaje "You are a minor." si la variable &AGE es menor de 18.

Declaraciones condicionales avanzadas

Declaración ELSEIF

La declaración ELSEIF permite verificar múltiples condiciones en secuencia.

Sintaxis

IF COND(logical-expression1) THEN(DO)
    /* Código a ejecutar si la condición1 es verdadera */
ENDDO
ELSEIF COND(logical-expression2) THEN(DO)
    /* Código a ejecutar si la condición2 es verdadera */
ENDDO
ELSE DO
    /* Código a ejecutar si ninguna de las condiciones anteriores es verdadera */
ENDDO

Ejemplo

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE < 13) THEN(DO)
    SNDPGMMSG MSG('You are a child.')
ENDDO
ELSEIF COND(&AGE < 18) THEN(DO)
    SNDPGMMSG MSG('You are a teenager.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are an adult.')
ENDDO

En este ejemplo, se envía el mensaje "You are an adult." si la variable &AGE es 18 o mayor.

Declaraciones IF anidadas

Las declaraciones IF anidadas son declaraciones IF colocadas dentro de otra declaración IF para verificar múltiples condiciones.

Sintaxis

IF COND(logical-expression1) THEN(DO)
    IF COND(logical-expression2) THEN(DO)
        /* Código a ejecutar si ambas condiciones son verdaderas */
    ENDDO
ENDDO

Ejemplo

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
DCL VAR(&MEMBER) TYPE(*CHAR) LEN(1)
CHGVAR VAR(&AGE) VALUE(20)
CHGVAR VAR(&MEMBER) VALUE('Y')

IF COND(&AGE >= 18) THEN(DO)
    IF COND(&MEMBER = 'Y') THEN(DO)
        SNDPGMMSG MSG('You are an adult member.')
    ENDDO
ENDDO

En este ejemplo, se envía el mensaje "You are an adult member." si la variable &AGE es 18 o mayor y &MEMBER es 'Y'.

Resumen

Las declaraciones condicionales en CL son cruciales para controlar el flujo de un programa. Comenzando con las declaraciones básicas IF y ELSE, puedes crear estructuras de toma de decisiones complejas utilizando ELSEIF y declaraciones IF anidadas. Comprender y dominar estos constructos mejorará significativamente tu capacidad para escribir programas CL eficientes y efectivos.
