Introducción a Prolog

Prolog (Programación en Lógica) es un lenguaje de programación de alto nivel asociado con la inteligencia artificial y la lingüística computacional. Prolog se basa en la lógica formal y permite la programación declarativa, donde describes el problema en lugar de la solución.

Conceptos Clave

  • Hechos: Afirmaciones básicas sobre el mundo.
  • Reglas: Declaraciones lógicas que definen relaciones entre hechos.
  • Consultas: Preguntas hechas al sistema Prolog para recuperar información basada en hechos y reglas.

Escribiendo Hechos

Los hechos son la forma más simple de declaraciones en Prolog. Declaran información que es incondicionalmente verdadera.

Ejemplo

parent(tom, bob).
parent(bob, alice).

En este ejemplo:

  • parent(tom, bob). indica que Tom es padre de Bob.
  • parent(bob, alice). indica que Bob es padre de Alice.

Escribiendo Reglas

Las reglas definen relaciones entre hechos usando implicaciones lógicas. Típicamente consisten en una cabeza y un cuerpo, separados por :-.

Ejemplo

grandparent(X, Y) :- parent(X, Z), parent(Z, Y).

Esta regla indica que X es abuelo de Y si X es padre de Z y Z es padre de Y.

Haciendo Consultas

Las consultas se utilizan para hacer preguntas sobre la información almacenada en la base de datos de Prolog.

Ejemplo

?- parent(tom, bob).

Esta consulta pregunta si Tom es padre de Bob. Prolog responderá con true.

Ejemplo con Variables

?- parent(X, alice).

Esta consulta pregunta quién es padre de Alice. Prolog responderá con X = bob.

Ejemplos Prácticos

Ejemplo 1: Relaciones Familiares

% Hechos
parent(tom, bob).
parent(bob, alice).
parent(alice, john).

% Reglas
grandparent(X, Y) :- parent(X, Z), parent(Z, Y).
ancestor(X, Y) :- parent(X, Y).
ancestor(X, Y) :- parent(X, Z), ancestor(Z, Y).

% Consultas
?- grandparent(tom, alice).
% Salida esperada: true

?- ancestor(tom, john).
% Salida esperada: true

Ejemplo 2: Base de Datos Simple

% Hechos
book('The Hobbit', tolkien).
book('1984', orwell).
book('Brave New World', huxley).

% Consultas
?- book('1984', Author).
% Salida esperada: Author = orwell.

?- book(Title, tolkien).
% Salida esperada: Title = 'The Hobbit'.

Resumen

En esta sección, cubrimos los conceptos básicos de Prolog, incluyendo:

  • Hechos: Afirmaciones simples sobre el mundo.
  • Reglas: Declaraciones lógicas que definen relaciones.
  • Consultas: Preguntas para recuperar información.

Al entender estos conceptos fundamentales, puedes comenzar a escribir programas simples en Prolog y a realizar consultas de información. En las próximas secciones, profundizaremos en temas más avanzados y en la resolución de problemas complejos utilizando Prolog.
