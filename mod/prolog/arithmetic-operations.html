Operaciones aritméticas en Prolog
Estructuras de Control ►

Introducción

Las operaciones aritméticas son fundamentales en cualquier lenguaje de programación, incluido Prolog. Esta sección cubrirá los conceptos básicos de las operaciones aritméticas en Prolog, incluyendo cómo realizar cálculos, usar operadores aritméticos y manejar expresiones aritméticas.

Operadores aritméticos básicos

Prolog soporta una variedad de operadores aritméticos. Aquí están los más comúnmente utilizados:

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División (punto flotante)
  • // : División entera
  • mod : Módulo (resto)
  • ^ : Exponenciación

Ejemplos

Veamos algunos ejemplos básicos de operaciones aritméticas en Prolog:

% Adición
Resultado es 3 + 2. % Resultado = 5

% Sustracción
Resultado es 5 - 3. % Resultado = 2

% Multiplicación
Resultado es 4 * 2. % Resultado = 8

% División (punto flotante)
Resultado es 7 / 2. % Resultado = 3.5

% División entera
Resultado es 7 // 2. % Resultado = 3

% Módulo
Resultado es 7 mod 2. % Resultado = 1

% Exponenciación
Resultado es 2 ^ 3. % Resultado = 8

Uso de expresiones aritméticas

En Prolog, las expresiones aritméticas se evalúan utilizando el operador is. El operador is se usa para asignar el resultado de una expresión aritmética a una variable.

Ejemplo

% Asignar el resultado de una expresión aritmética a una variable
X es 5 + 3. % X = 8

Nota importante

El operador is requiere que el lado derecho de la expresión esté completamente instanciado (es decir, todas las variables deben tener valores) antes de la evaluación.

% Esto funcionará
A = 5, B = 3, X es A + B. % X = 8

% Esto no funcionará y resultará en un error
X es A + B, A = 5, B = 3. % Error: Los argumentos no están suficientemente instanciados

Operadores de comparación

Prolog también proporciona operadores de comparación para expresiones aritméticas:

  • =:= : Igual a
  • =\= : No igual a
  • < : Menor que
  • > : Mayor que
  • =< : Menor o igual a
  • >= : Mayor o igual a

Ejemplos

% Igual a
5 =:= 5. % true

% No igual a
5 =\= 3. % true

% Menor que
3 < 5. % true

% Mayor que
5 > 3. % true

% Menor o igual a
3 =< 5. % true

% Mayor o igual a
5 >= 3. % true

Ejercicios prácticos

Aquí hay algunos ejercicios para practicar operaciones aritméticas en Prolog:

  1. Escribe un predicado de Prolog sum/3 que tome dos números y devuelva su suma.

    sum(A, B, Sum) :-
    Sum is A + B.

  2. Escribe un predicado de Prolog factorial/2 que calcule el factorial de un número dado.

    factorial(0, 1).
factorial(N, Resultado) :-
    N > 0,
    N1 es N - 1,
    factorial(N1, R1),
    Resultado es N * R1.

  3. Escribe un predicado de Prolog max/3 que tome tres números y devuelva el máximo de los tres.

    max(A, B, C, Max) :-
    Max1 es max(A, B),
    Max es max(Max1, C).

Conclusión

Las operaciones aritméticas en Prolog son sencillas una vez que entiendes los operadores básicos y cómo usar el operador is para evaluar expresiones. Recuerda asegurarte de que todas las variables en una expresión aritmética estén instanciadas antes de la evaluación. Practica con los ejercicios proporcionados para solidificar tu comprensión de las operaciones aritméticas en Prolog.
