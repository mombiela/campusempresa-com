Introducción

Prolog, un lenguaje de programación lógica, viene con un rico conjunto de predicados incorporados que facilitan varias operaciones como aritmética, manipulación de listas, entrada/salida y más. Entender estos predicados incorporados es crucial para escribir programas Prolog eficientes y efectivos.

Conceptos Clave

¿Qué son los Predicados Incorporados?

  • Definición: Los predicados incorporados son funciones predefinidas en Prolog que realizan operaciones específicas.
  • Propósito: Simplifican tareas comunes y mejoran la funcionalidad del lenguaje sin requerir que los usuarios definan estas operaciones desde cero.

Categorías de Predicados Incorporados

  • Predicados Aritméticos: Realizan operaciones matemáticas.
  • Predicados de Manipulación de Listas: Manejan operaciones en listas.
  • Predicados de Entrada/Salida: Gestionan la lectura y escritura en flujos.
  • Predicados de Control: Controlan el flujo del programa.
  • Predicados de Comparación: Comparan términos y valores.

Predicados Aritméticos

Los predicados aritméticos te permiten realizar operaciones matemáticas. Aquí hay algunos comunes:

  • is/2: Evalúa una expresión aritmética.
  • =/2: Verifica si dos términos son iguales.
  • >/2, >=/2, </2, =</2: Operadores de comparación.

Ejemplos

% Ejemplo de `is/2`
X is 3 + 4. % X = 7

% Ejemplo de comparación
7 > 5. % true
3 =< 4. % true

Predicados de Manipulación de Listas

Prolog proporciona varios predicados para trabajar con listas:

  • member/2: Verifica si un elemento es miembro de una lista.
  • append/3: Concatenar dos listas.
  • length/2: Encuentra la longitud de una lista.
  • reverse/2: Invierte una lista.

Ejemplos

% Ejemplo de `member/2`
member(3, [1, 2, 3, 4]). % true

% Ejemplo de `append/3`
append([1, 2], [3, 4], X). % X = [1, 2, 3, 4]

% Ejemplo de `length/2`
length([1, 2, 3, 4], L). % L = 4

% Ejemplo de `reverse/2`
reverse([1, 2, 3, 4], R). % R = [4, 3, 2, 1]

Predicados de Entrada/Salida

Estos predicados manejan la lectura y escritura en flujos:

  • write/1: Escribe un término en el flujo de salida actual.
  • read/1: Lee un término del flujo de entrada actual.
  • nl/0: Escribe un carácter de nueva línea.

Ejemplos

% Ejemplo de `write/1` y `nl/0`
write('¡Hola, Mundo!'), nl. % ¡Hola, Mundo!

% Ejemplo de `read/1`
% Suponiendo que el usuario ingresa: hola.
read(X). % X = hola

Predicados de Control

Los predicados de control gestionan el flujo del programa:

  • fail/0: Siempre falla.
  • true/0: Siempre tiene éxito.
  • repeat/0: Siempre tiene éxito y puede usarse para crear bucles.
  • cut (!)/0: Poda puntos de elección para controlar el retroceso.

Ejemplos

% Ejemplo de `fail/0` y `true/0`
true. % true
fail. % false

% Ejemplo de `repeat/0`
repeat, write('Hola'), nl, fail. % Bucle infinito imprimiendo 'Hola'

% Ejemplo de `cut (!)/0`
member(X, [1, 2, 3]), !, write(X). % Imprime el primer elemento y detiene el retroceso

Predicados de Comparación

Los predicados de comparación se utilizan para comparar términos:

  • ==/2: Verifica si dos términos son idénticos.
  • \==/2: Verifica si dos términos no son idénticos.
  • @</2, @>/2, @=</2, @>=/2: Comparación lexicográfica de términos.

Ejemplos

% Ejemplo de `==/2` y `\==/2`
1 == 1. % true
1 \== 2. % true

% Ejemplo de comparación lexicográfica
manzana @< banana. % true
banana @> manzana. % true

Conclusión

Los predicados incorporados en Prolog son herramientas poderosas que simplifican muchas tareas comunes de programación. Al dominar estos predicados, puedes escribir código Prolog más eficiente y legible. Este tema cubrió los predicados incorporados esenciales en varias categorías, proporcionando ejemplos para ilustrar su uso. Practica usando estos predicados en diferentes escenarios para volverte competente en la programación en Prolog.
