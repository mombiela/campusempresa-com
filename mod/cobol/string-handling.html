Manejo de Cadenas en COBOL
Manejo de Fechas y Tiempos ►

El manejo de cadenas es un aspecto fundamental de la programación en COBOL, al igual que en muchos otros lenguajes. Este tema cubrirá los conceptos básicos del manejo de cadenas, incluyendo cómo definir, manipular y realizar operaciones con cadenas en COBOL. Al final de esta sección, tendrás una comprensión sólida de cómo trabajar con cadenas en COBOL, desde niveles principiantes hasta avanzados.

Introducción a las Cadenas en COBOL

  • Definición: En COBOL, las cadenas se denominan elementos de datos alfanuméricos.
  • Uso: Las cadenas se utilizan para almacenar y manipular datos de texto.

Definición de Cadenas

En COBOL, las cadenas se definen utilizando la cláusula PIC (Picture) en la División de Datos.

01 CUSTOMER-NAME PIC A(30).
01 ADDRESS       PIC X(50).
  • PIC A(n): Define una cadena alfanumérica de longitud n.
  • PIC X(n): Define una cadena que puede contener cualquier carácter, incluyendo espacios.

Inicialización de Cadenas

Las cadenas pueden inicializarse utilizando la cláusula VALUE.

01 GREETING PIC X(20) VALUE 'Hello, World!'.

Operaciones Básicas con Cadenas

Concatenación

La concatenación en COBOL se puede realizar utilizando la instrucción STRING.

STRING FIRST-NAME DELIMITED BY SPACE
       LAST-NAME DELIMITED BY SPACE
       INTO FULL-NAME.
  • Explicación: Este código concatena FIRST-NAME y LAST-NAME en FULL-NAME, con un delimitador de espacio.

Extracción de Subcadena

La extracción de una subcadena se puede realizar utilizando el REFERENCE MODIFIER.

MOVE CUSTOMER-NAME(1:5) TO SHORT-NAME.
  • Explicación: Este código extrae los primeros 5 caracteres de CUSTOMER-NAME y los mueve a SHORT-NAME.

Longitud de la Cadena

La longitud de una cadena se puede determinar utilizando la FUNCTION LENGTH.

COMPUTE NAME-LENGTH = FUNCTION LENGTH(CUSTOMER-NAME).
  • Explicación: Este código calcula la longitud de CUSTOMER-NAME y la almacena en NAME-LENGTH.

Operaciones Avanzadas con Cadenas

Búsqueda en Cadenas

La búsqueda de una subcadena dentro de una cadena se puede realizar utilizando la instrucción INSPECT.

INSPECT FULL-NAME TALLYING COUNT-OF-A FOR ALL 'A'.
  • Explicación: Este código cuenta el número de ocurrencias del carácter 'A' en FULL-NAME y almacena el conteo en COUNT-OF-A.

Reemplazo de Cadenas

El reemplazo de partes de una cadena se puede realizar utilizando la instrucción INSPECT con la cláusula REPLACING.

INSPECT FULL-NAME REPLACING ALL 'A' BY 'E'.
  • Explicación: Este código reemplaza todas las ocurrencias de 'A' en FULL-NAME con 'E'.

Comparación de Cadenas

La comparación de cadenas se puede realizar utilizando operadores relacionales.

IF CUSTOMER-NAME = 'JOHN DOE'
   DISPLAY 'Match Found'.
  • Explicación: Este código verifica si CUSTOMER-NAME es igual a 'JOHN DOE' y muestra un mensaje si lo es.

Ejemplos Prácticos

Ejemplo 1: Concatenación de Nombre y Apellido

IDENTIFICATION DIVISION.
PROGRAM-ID. CONCAT-NAME.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 FIRST-NAME PIC X(10) VALUE 'John'.
01 LAST-NAME  PIC X(10) VALUE 'Doe'.
01 FULL-NAME  PIC X(21).

PROCEDURE DIVISION.
    STRING FIRST-NAME DELIMITED BY SPACE
           LAST-NAME DELIMITED BY SPACE
           INTO FULL-NAME
    END-STRING.
    DISPLAY 'Full Name: ' FULL-NAME.
    STOP RUN.

Ejemplo 2: Contando Caracteres en una Cadena

IDENTIFICATION DIVISION.
PROGRAM-ID. COUNT-CHAR.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 FULL-NAME  PIC X(20) VALUE 'Johnathan Doe'.
01 COUNT-OF-A PIC 9(2).

PROCEDURE DIVISION.
    INSPECT FULL-NAME TALLYING COUNT-OF-A FOR ALL 'A'.
    DISPLAY 'Number of A's: ' COUNT-OF-A.
    STOP RUN.

Conclusión

El manejo de cadenas en COBOL implica definir, inicializar y realizar varias operaciones con cadenas. Desde operaciones básicas como la concatenación y la extracción de subcadenas hasta operaciones avanzadas como la búsqueda y el reemplazo, COBOL proporciona herramientas robustas para gestionar datos de texto. Al dominar estas técnicas de manejo de cadenas, estarás bien equipado para manejar una amplia gama de tareas de procesamiento de texto en tus programas COBOL.
