Procesamiento de Archivos Relativos en COBOL
División de Procedimientos ►

Introducción

El procesamiento de archivos relativos en COBOL implica manejar archivos donde los registros se acceden basándose en su posición relativa dentro del archivo. Este método es útil para aplicaciones que requieren acceso rápido a registros sin leer secuencialmente todo el archivo.

Conceptos Clave

  • Archivo Relativo: Un archivo donde los registros se identifican por su número de registro relativo (RRN).
  • RRN (Número de Registro Relativo): Un número único que representa la posición de un registro dentro de un archivo relativo.
  • Modos de Acceso: Métodos para acceder a registros en un archivo relativo, incluyendo acceso secuencial y aleatorio.

Definiendo un Archivo Relativo

Para trabajar con archivos relativos en COBOL, necesitas definir el archivo en la SECCIÓN DE ARCHIVOS de la DIVISIÓN DE DATOS y especificar su organización en la DIVISIÓN DE ENTORNO.

Ejemplo

IDENTIFICATION DIVISION.
PROGRAM-ID. RelativeFileExample.

ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
    SELECT REL-FILE ASSIGN TO 'relative.dat'
    ORGANIZATION IS RELATIVE
    ACCESS MODE IS RANDOM
    RELATIVE KEY IS REL-KEY.

DATA DIVISION.
FILE SECTION.
FD  REL-FILE.
01  REL-RECORD.
    05  REL-KEY        PIC 9(4).
    05  REL-DATA       PIC X(20).

WORKING-STORAGE SECTION.
01  WS-REL-KEY         PIC 9(4).
01  WS-REL-DATA        PIC X(20).

PROCEDURE DIVISION.

Abrir y Cerrar Archivos Relativos

Los archivos relativos deben abrirse antes de que puedan ser leídos o escritos, y cerrarse después de que las operaciones estén completas.

Ejemplo

OPEN INPUT REL-FILE.
CLOSE REL-FILE.

Leyendo Registros

Para leer un registro de un archivo relativo, necesitas establecer la clave relativa y usar la instrucción READ.

Ejemplo

MOVE 1 TO WS-REL-KEY.
READ REL-FILE
    INVALID KEY DISPLAY 'Record not found.'
    NOT INVALID KEY DISPLAY 'Record found: ' REL-DATA.

Escribiendo Registros

Para escribir un registro en un archivo relativo, necesitas establecer la clave relativa y los datos, luego usar la instrucción WRITE.

Ejemplo

MOVE 2 TO WS-REL-KEY.
MOVE 'Sample Data' TO WS-REL-DATA.
WRITE REL-RECORD INVALID KEY DISPLAY 'Error writing record.'

Actualizando Registros

Para actualizar un registro, léelo primero, modifica los datos y luego reescríbelo.

Ejemplo

MOVE 1 TO WS-REL-KEY.
READ REL-FILE
    INVALID KEY DISPLAY 'Record not found.'
    NOT INVALID KEY
        MOVE 'Updated Data' TO REL-DATA
        REWRITE REL-RECORD INVALID KEY DISPLAY 'Error updating record.'

Borrando Registros

Para borrar un registro, usa la instrucción DELETE después de establecer la clave relativa.

Ejemplo

MOVE 1 TO WS-REL-KEY.
DELETE REL-FILE INVALID KEY DISPLAY 'Error deleting record.'

Resumen

El procesamiento de archivos relativos en COBOL permite un acceso eficiente a los registros basándose en sus posiciones relativas. Las operaciones clave incluyen leer, escribir, actualizar y borrar registros usando el número de registro relativo. Entender cómo definir, abrir, cerrar y manipular archivos relativos es esencial para una programación efectiva en COBOL.

Al dominar estos conceptos, puedes manejar datos de manera más eficiente en aplicaciones que requieren acceso rápido y directo a los registros.
