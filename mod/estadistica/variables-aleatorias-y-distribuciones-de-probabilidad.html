Variables Aleatorias y Distribuciones de Probabilidad

Introducción

En estadística, las variables aleatorias y las distribuciones de probabilidad son conceptos fundamentales que permiten modelar y analizar fenómenos inciertos. Este tema cubre los conceptos básicos, tipos de variables aleatorias, y las distribuciones de probabilidad más comunes.

Variables Aleatorias

Una variable aleatoria es una función que asigna un valor numérico a cada resultado posible de un experimento aleatorio.

Tipos de Variables Aleatorias

  • Variable Aleatoria Discreta: Toma un número finito o contable de valores. Ejemplo: Número de caras al lanzar una moneda tres veces.
  • Variable Aleatoria Continua: Toma un número infinito de valores dentro de un intervalo. Ejemplo: Tiempo de espera en una fila.

Ejemplo de Variable Aleatoria Discreta

Supongamos que lanzamos un dado justo. La variable aleatoria \(X\) representa el resultado del lanzamiento del dado.

  • Posibles valores de \(X\): {1, 2, 3, 4, 5, 6}

Ejemplo de Variable Aleatoria Continua

Consideremos el tiempo \(T\) que un cliente espera en una fila. Esta variable puede tomar cualquier valor positivo.

  • Posibles valores de \(T\): {0, 0.1, 0.2, ..., ∞}

Distribuciones de Probabilidad

Una distribución de probabilidad describe cómo se distribuyen los valores de una variable aleatoria.

Distribución de Probabilidad para Variables Discretas

  • Función de Probabilidad (PMF): Asigna una probabilidad a cada valor posible de la variable aleatoria discreta.

    Ejemplo: Lanzamiento de un dado justo. \[ P(X = x) = \frac{1}{6} \quad \text{para } x \in {1, 2, 3, 4, 5, 6} \]

Distribución de Probabilidad para Variables Continuas

  • Función de Densidad de Probabilidad (PDF): Describe la probabilidad de que una variable aleatoria continua tome un valor dentro de un intervalo.

    Ejemplo: Distribución normal. \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \] donde \(\mu\) es la media y \(\sigma\) es la desviación estándar.

Distribuciones de Probabilidad Comunes

Distribución Binomial

  • Descripción: Modela el número de éxitos en una serie de ensayos independientes con probabilidad constante de éxito.
  • Parámetros: \(n\) (número de ensayos), \(p\) (probabilidad de éxito).
  • PMF: \[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \] donde \(\binom{n}{k}\) es el coeficiente binomial.

Distribución Normal

  • Descripción: Modela fenómenos naturales y es simétrica alrededor de la media.
  • Parámetros: \(\mu\) (media), \(\sigma\) (desviación estándar).
  • PDF: \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

Distribución Poisson

  • Descripción: Modela el número de eventos que ocurren en un intervalo de tiempo o espacio.
  • Parámetros: \(\lambda\) (tasa de eventos).
  • PMF: \[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \]

Ejemplos de Código

Ejemplo en Python: Distribución Binomial

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom

# Parámetros
n = 10  # número de ensayos
p = 0.5  # probabilidad de éxito

# Valores posibles de X
x = np.arange(0, n+1)

# PMF
pmf = binom.pmf(x, n, p)

# Gráfica
plt.bar(x, pmf)
plt.xlabel('Número de éxitos')
plt.ylabel('Probabilidad')
plt.title('Distribución Binomial (n=10, p=0.5)')
plt.show()

Ejemplo en Python: Distribución Normal

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# Parámetros
mu = 0  # media
sigma = 1  # desviación estándar

# Valores posibles de X
x = np.linspace(mu - 3*sigma, mu + 3*sigma, 100)

# PDF
pdf = norm.pdf(x, mu, sigma)

# Gráfica
plt.plot(x, pdf)
plt.xlabel('Valor')
plt.ylabel('Densidad de probabilidad')
plt.title('Distribución Normal (μ=0, σ=1)')
plt.show()

Conclusión

Las variables aleatorias y las distribuciones de probabilidad son herramientas esenciales en estadística para modelar y analizar datos. Comprender estos conceptos permite a los profesionales tomar decisiones informadas basadas en datos y manejar la incertidumbre de manera efectiva. En este tema, hemos cubierto los tipos de variables aleatorias, las distribuciones de probabilidad más comunes y proporcionado ejemplos prácticos para ilustrar estos conceptos.
