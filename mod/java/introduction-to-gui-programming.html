Introducción a la Programación de Interfaces Gráficas de Usuario (GUI)
Conceptos Básicos de Swing ►

Visión General

La programación de Interfaces Gráficas de Usuario (GUI) permite a los desarrolladores crear aplicaciones visualmente interactivas. En Java, la programación GUI se realiza principalmente utilizando las bibliotecas Swing y JavaFX. Esta sección te introducirá a los conceptos básicos de la programación GUI en Java, cubriendo conceptos esenciales, componentes y ejemplos prácticos.

Conceptos Clave

  • Programación Orientada a Eventos: Las aplicaciones GUI son orientadas a eventos, lo que significa que responden a acciones del usuario como clics, pulsaciones de teclas, etc.
  • Componentes: Bloques básicos de una GUI, como botones, etiquetas, campos de texto, etc.
  • Contenedores: Contienen y organizan componentes, como paneles, marcos y ventanas.
  • Gestores de Diseño: Definen cómo se organizan los componentes dentro de los contenedores.

Empezando con Swing

Swing es parte de las Clases Fundamentales de Java (JFC) y proporciona un conjunto rico de componentes GUI.

Creando una Ventana Simple

Para crear una ventana básica en Swing, necesitas usar la clase JFrame.

import javax.swing.JFrame;

public class SimpleWindow {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Simple Window");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicación:
    • JFrame: Un contenedor de nivel superior que representa una ventana.
    • setSize(): Establece el tamaño de la ventana.
    • setDefaultCloseOperation(): Define la operación predeterminada cuando se cierra la ventana.
    • setVisible(): Hace visible la ventana.

Añadiendo Componentes

Vamos a añadir un botón a la ventana.

import javax.swing.JButton;
import javax.swing.JFrame;

public class ButtonExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Example");
        JButton button = new JButton("Click Me");
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(button); // Añadiendo el botón al marco
        frame.setVisible(true);
    }
}
  • Explicación:
    • JButton: Un componente que representa un botón.
    • add(): Añade el botón al marco.

Manejo de Eventos

Para manejar los clics de botones, necesitas implementar un ActionListener.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonClickExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Click Example");
        JButton button = new JButton("Click Me");
        
        button.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                System.out.println("Button was clicked!");
            }
        });
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(button);
        frame.setVisible(true);
    }
}
  • Explicación:
    • ActionListener: Una interfaz para recibir eventos de acción.
    • actionPerformed(): Método que se llama cuando se hace clic en el botón.

Gestores de Diseño

Los gestores de diseño controlan la posición y el tamaño de los componentes dentro de un contenedor.

BorderLayout

BorderLayout divide el contenedor en cinco regiones: Norte, Sur, Este, Oeste y Centro.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.BorderLayout;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setLayout(new BorderLayout());
        
        frame.add(new JButton("North"), BorderLayout.NORTH);
        frame.add(new JButton("South"), BorderLayout.SOUTH);
        frame.add(new JButton("East"), BorderLayout.EAST);
        frame.add(new JButton("West"), BorderLayout.WEST);
        frame.add(new JButton("Center"), BorderLayout.CENTER);
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicación:
    • BorderLayout: Un gestor de diseño que organiza componentes en cinco regiones.

GridLayout

GridLayout organiza los componentes en una cuadrícula de celdas, cada una de igual tamaño.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.GridLayout;

public class GridLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("GridLayout Example");
        frame.setLayout(new GridLayout(2, 2)); // 2 filas, 2 columnas
        
        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));
        frame.add(new JButton("Button 4"));
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicación:
    • GridLayout: Un gestor de diseño que organiza componentes en una cuadrícula.

Introducción a JavaFX

JavaFX es un moderno toolkit GUI para Java, que proporciona APIs ricas para desarrollar aplicaciones de escritorio.

Creando una Aplicación JavaFX Simple

Para crear una aplicación básica de JavaFX, necesitas extender la clase Application y sobrescribir el método start().

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.stage.Stage;

public class SimpleJavaFXApp extends Application {
    @Override
    public void start(Stage primaryStage) {
        Button button = new Button("Click Me");
        button.setOnAction(e -> System.out.println("Button was clicked!"));
        
        Scene scene = new Scene(button, 400, 300);
        primaryStage.setTitle("Simple JavaFX Application");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}
  • Explicación:
    • Application: La clase base para aplicaciones JavaFX.
    • start(): El punto de entrada principal para aplicaciones JavaFX.
    • Stage: La ventana principal.
    • Scene: El contenedor para todo el contenido en un gráfico de escena.
    • Button: Un componente de botón de JavaFX.

Conclusión

En este tema, introdujimos los conceptos básicos de la programación GUI en Java utilizando Swing y JavaFX. Cubrimos conceptos clave, creamos ventanas simples, añadimos componentes, manejamos eventos y exploramos gestores de diseño. Entender estos fundamentos te permitirá construir aplicaciones Java más complejas e interactivas. A medida que avances, puedes explorar temas avanzados como componentes personalizados, animaciones e integración con otras tecnologías.
