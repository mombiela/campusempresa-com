Introducción

Leer y escribir archivos es una habilidad fundamental para cualquier programador Java. Este tema cubrirá los conceptos básicos del manejo de archivos en Java, incluyendo la lectura de archivos, la escritura de archivos y el manejo de excepciones. Al final de esta sección, serás capaz de leer datos de archivos y escribir datos en archivos usando Java.

Conceptos Clave

  • Clase File: Representa una ruta de archivo o directorio.
  • FileReader y FileWriter: Clases para leer y escribir archivos de texto.
  • BufferedReader y BufferedWriter: Clases para una lectura y escritura eficientes.
  • FileInputStream y FileOutputStream: Clases para leer y escribir archivos binarios.
  • Manejo de Excepciones: Gestión de errores que ocurren durante las operaciones de archivo.

Trabajando con la Clase File

La clase File en Java se usa para representar nombres de ruta de archivos y directorios.

import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");

        // Verificar si el archivo existe
        if (file.exists()) {
            System.out.println("El archivo existe.");
        } else {
            System.out.println("El archivo no existe.");
        }
    }
}

Leyendo Archivos

Usando FileReader y BufferedReader

FileReader y BufferedReader se usan para leer archivos de texto de manera eficiente.

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Usando FileInputStream

FileInputStream se usa para leer archivos binarios.

import java.io.FileInputStream;
import java.io.IOException;

public class ReadBinaryFileExample {
    public static void main(String[] args) {
        try (FileInputStream fis = new FileInputStream("example.bin")) {
            int content;
            while ((content = fis.read()) != -1) {
                System.out.print((char) content);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Escribiendo Archivos

Usando FileWriter y BufferedWriter

FileWriter y BufferedWriter se usan para escribir archivos de texto de manera eficiente.

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("example.txt"))) {
            bw.write("¡Hola, Mundo!");
            bw.newLine();
            bw.write("Este es un archivo de texto.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Usando FileOutputStream

FileOutputStream se usa para escribir archivos binarios.

import java.io.FileOutputStream;
import java.io.IOException;

public class WriteBinaryFileExample {
    public static void main(String[] args) {
        try (FileOutputStream fos = new FileOutputStream("example.bin")) {
            String content = "¡Hola, Mundo Binario!";
            fos.write(content.getBytes());
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Manejo de Excepciones

Manejar excepciones es crucial cuando se trabaja con archivos para asegurar que el programa pueda manejar errores de manera elegante.

import java.io.File;
import java.io.FileNotFoundException;
import java.io.FileReader;
import java.io.IOException;

public class ExceptionHandlingExample {
    public static void main(String[] args) {
        File file = new File("example.txt");
        try (FileReader fr = new FileReader(file)) {
            int content;
            while ((content = fr.read()) != -1) {
                System.out.print((char) content);
            }
        } catch (FileNotFoundException e) {
            System.out.println("Archivo no encontrado: " + e.getMessage());
        } catch (IOException e) {
            System.out.println("Error de E/S: " + e.getMessage());
        }
    }
}

Resumen

En esta sección, cubrimos:

  • La clase File para representar rutas de archivos.
  • Leer archivos de texto usando FileReader y BufferedReader.
  • Leer archivos binarios usando FileInputStream.
  • Escribir archivos de texto usando FileWriter y BufferedWriter.
  • Escribir archivos binarios usando FileOutputStream.
  • Manejar excepciones para gestionar errores durante las operaciones de archivo.

Entender estos conceptos te permitirá gestionar efectivamente la entrada y salida de archivos en tus aplicaciones Java. Practica estos ejemplos e intenta crear tus propias operaciones de manejo de archivos para solidificar tu comprensión.
