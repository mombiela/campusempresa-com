Introducción

En esta sección, profundizaremos en los componentes avanzados de Swing en Java. Swing es una parte de las Clases Fundamentales de Java (JFC) utilizada para crear aplicaciones basadas en ventanas. Mientras que los componentes básicos como JButton, JLabel y JTextField son comúnmente utilizados, Swing también ofrece componentes más avanzados para crear interfaces de usuario sofisticadas.

Componentes Avanzados Clave de Swing

  • JTable
  • JTree
  • JTabbedPane
  • JToolBar
  • JInternalFrame

JTable

JTable se utiliza para mostrar y editar tablas bidimensionales regulares de celdas.

Ejemplo:

import javax.swing.*;
import javax.swing.table.DefaultTableModel;

public class JTableExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTable Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        String[] columns = {"ID", "Name", "Age"};
        Object[][] data = {
            {1, "John Doe", 25},
            {2, "Jane Smith", 30},
            {3, "Kathy Green", 22}
        };

        DefaultTableModel model = new DefaultTableModel(data, columns);
        JTable table = new JTable(model);
        JScrollPane scrollPane = new JScrollPane(table);

        frame.add(scrollPane);
        frame.setVisible(true);
    }
}
  • Explicación: Este ejemplo crea una tabla simple con tres columnas: ID, Nombre y Edad. Se utiliza DefaultTableModel para gestionar los datos.

JTree

JTree se utiliza para mostrar datos jerárquicos en una estructura de árbol.

Ejemplo:

import javax.swing.*;
import javax.swing.tree.DefaultMutableTreeNode;

public class JTreeExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTree Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        DefaultMutableTreeNode root = new DefaultMutableTreeNode("Root");
        DefaultMutableTreeNode child1 = new DefaultMutableTreeNode("Child 1");
        DefaultMutableTreeNode child2 = new DefaultMutableTreeNode("Child 2");

        root.add(child1);
        root.add(child2);

        JTree tree = new JTree(root);
        JScrollPane scrollPane = new JScrollPane(tree);

        frame.add(scrollPane);
        frame.setVisible(true);
    }
}
  • Explicación: Este ejemplo crea un árbol simple con un nodo raíz y dos nodos hijos. Se utiliza DefaultMutableTreeNode para crear los nodos.

JTabbedPane

JTabbedPane se utiliza para crear una interfaz de paneles con pestañas, donde se pueden agregar múltiples pestañas.

Ejemplo:

import javax.swing.*;

public class JTabbedPaneExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTabbedPane Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        JTabbedPane tabbedPane = new JTabbedPane();

        JPanel panel1 = new JPanel();
        panel1.add(new JLabel("This is Tab 1"));
        tabbedPane.addTab("Tab 1", panel1);

        JPanel panel2 = new JPanel();
        panel2.add(new JLabel("This is Tab 2"));
        tabbedPane.addTab("Tab 2", panel2);

        frame.add(tabbedPane);
        frame.setVisible(true);
    }
}
  • Explicación: Este ejemplo crea un panel con pestañas con dos pestañas, cada una conteniendo una etiqueta simple.

JToolBar

JToolBar se utiliza para crear una barra de herramientas que puede contener botones, etiquetas y otros componentes.

Ejemplo:

import javax.swing.*;

public class JToolBarExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JToolBar Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        JToolBar toolBar = new JToolBar();
        JButton button1 = new JButton("Button 1");
        JButton button2 = new JButton("Button 2");

        toolBar.add(button1);
        toolBar.add(button2);

        frame.add(toolBar, "North");
        frame.setVisible(true);
    }
}
  • Explicación: Este ejemplo crea una barra de herramientas con dos botones y la agrega en la parte superior (Norte) del marco.

JInternalFrame

JInternalFrame se utiliza para crear marcos internos (sub-ventanas) dentro de una ventana principal de la aplicación.

Ejemplo:

import javax.swing.*;

public class JInternalFrameExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JInternalFrame Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(600, 400);

        JDesktopPane desktopPane = new JDesktopPane();

        JInternalFrame internalFrame = new JInternalFrame("Internal Frame", true, true, true, true);
        internalFrame.setSize(200, 150);
        internalFrame.setVisible(true);

        desktopPane.add(internalFrame);
        frame.add(desktopPane);
        frame.setVisible(true);
    }
}
  • Explicación: Este ejemplo crea un marco interno dentro de un panel de escritorio. El marco interno puede ser movido, redimensionado, cerrado y maximizado.

Conclusión

Los componentes avanzados de Swing como JTable, JTree, JTabbedPane, JToolBar y JInternalFrame proporcionan herramientas poderosas para crear aplicaciones Java complejas y ricas en funcionalidades. Comprender y utilizar estos componentes puede mejorar significativamente la funcionalidad y la experiencia del usuario de tus aplicaciones. Practica utilizando estos componentes en diferentes escenarios para volverte competente en la construcción de interfaces de usuario avanzadas basadas en Swing.
