Introducción

La entrada/salida de archivos (File I/O) es un concepto fundamental en la programación en Java que permite a los programas leer y escribir archivos. Esta sección cubrirá los conceptos básicos del manejo de archivos en Java, incluyendo la lectura de archivos, la escritura de archivos y el manejo de excepciones.

Conceptos Clave

  • Clase File: Representa una ruta de archivo o directorio.
  • FileReader y FileWriter: Clases para leer y escribir archivos de caracteres.
  • BufferedReader y BufferedWriter: Clases para una lectura y escritura eficientes.
  • FileInputStream y FileOutputStream: Clases para leer y escribir archivos binarios.
  • Manejo de Excepciones: Gestión de errores que ocurren durante las operaciones de archivo.

Trabajando con la Clase File

La clase File en Java se utiliza para representar nombres de ruta de archivos y directorios. Proporciona métodos para crear, eliminar y verificar propiedades de archivos y directorios.

Ejemplo: Creando un Objeto File

import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");
        
        if (file.exists()) {
            System.out.println("El archivo existe.");
        } else {
            System.out.println("El archivo no existe.");
        }
    }
}

En este ejemplo, creamos un objeto File que representa un archivo llamado "example.txt" y verificamos si existe.

Leyendo de un Archivo

Para leer de un archivo, puedes usar FileReader y BufferedReader para archivos de caracteres, o FileInputStream para archivos binarios.

Ejemplo: Leyendo un Archivo de Texto

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

En este ejemplo, se utiliza BufferedReader para leer el archivo línea por línea. La declaración try-with-resources asegura que el archivo se cierre automáticamente.

Escribiendo en un Archivo

Para escribir en un archivo, puedes usar FileWriter y BufferedWriter para archivos de caracteres, o FileOutputStream para archivos binarios.

Ejemplo: Escribiendo en un Archivo de Texto

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("example.txt"))) {
            bw.write("¡Hola, Mundo!");
            bw.newLine();
            bw.write("Esto es una prueba.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

En este ejemplo, se utiliza BufferedWriter para escribir texto en el archivo. La declaración try-with-resources asegura que el archivo se cierre automáticamente.

Manejo de Excepciones

Las operaciones de entrada/salida de archivos pueden lanzar varias excepciones, como FileNotFoundException y IOException. El manejo adecuado de excepciones es crucial para asegurar que tu programa pueda manejar errores de manera elegante.

Ejemplo: Manejo de Excepciones

import java.io.File;
import java.io.FileNotFoundException;
import java.io.FileReader;
import java.io.IOException;

public class ExceptionHandlingExample {
    public static void main(String[] args) {
        File file = new File("nonexistent.txt");
        try (FileReader fr = new FileReader(file)) {
            // Leer contenido del archivo
        } catch (FileNotFoundException e) {
            System.out.println("Archivo no encontrado: " + file.getName());
        } catch (IOException e) {
            System.out.println("Ocurrió un error de E/S: " + e.getMessage());
        }
    }
}

En este ejemplo, manejamos FileNotFoundException y IOException por separado para proporcionar mensajes de error específicos.

Resumen

En esta sección, cubrimos los conceptos básicos de la entrada/salida de archivos en Java, incluyendo:

  • Creación y uso de la clase File.
  • Lectura y escritura de archivos de texto usando FileReader, FileWriter, BufferedReader y BufferedWriter.
  • Manejo de excepciones que pueden ocurrir durante las operaciones de archivo.

Entender estos conceptos es esencial para trabajar con archivos en Java, y forman la base para técnicas más avanzadas de manejo de archivos.
