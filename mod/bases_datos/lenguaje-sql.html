Introducción al Lenguaje SQL

El Lenguaje de Consulta Estructurado (SQL, por sus siglas en inglés) es un lenguaje estándar utilizado para gestionar y manipular bases de datos relacionales. SQL permite realizar diversas operaciones como la creación de bases de datos y tablas, la inserción de datos, la actualización y eliminación de registros, y la consulta de información almacenada.

Conceptos Clave

  • Base de Datos: Conjunto organizado de datos almacenados y gestionados electrónicamente.
  • Tabla: Estructura dentro de una base de datos que organiza los datos en filas y columnas.
  • Registro: Fila en una tabla que representa una única entrada de datos.
  • Campo: Columna en una tabla que representa un atributo específico de los datos.

Operaciones Básicas en SQL

Creación de Bases de Datos y Tablas

Para comenzar a trabajar con SQL, primero necesitamos crear una base de datos y luego definir las tablas que almacenarán nuestros datos.

-- Crear una base de datos
CREATE DATABASE MiBaseDeDatos;

-- Seleccionar la base de datos
USE MiBaseDeDatos;

-- Crear una tabla
CREATE TABLE Empleados (
    ID INT PRIMARY KEY,
    Nombre VARCHAR(50),
    Apellido VARCHAR(50),
    Edad INT,
    Departamento VARCHAR(50)
);

Inserción de Datos

Una vez que tenemos nuestras tablas, podemos insertar datos en ellas.

-- Insertar un registro en la tabla Empleados
INSERT INTO Empleados (ID, Nombre, Apellido, Edad, Departamento)
VALUES (1, 'Juan', 'Pérez', 30, 'Ventas');

Consulta de Datos

La consulta de datos es una de las operaciones más comunes en SQL. Utilizamos la cláusula SELECT para recuperar información de una o más tablas.

-- Consultar todos los registros de la tabla Empleados
SELECT * FROM Empleados;

-- Consultar registros específicos
SELECT Nombre, Apellido FROM Empleados WHERE Edad > 25;

Actualización de Datos

Podemos actualizar los datos existentes en una tabla utilizando la cláusula UPDATE.

-- Actualizar el departamento de un empleado
UPDATE Empleados
SET Departamento = 'Marketing'
WHERE ID = 1;

Eliminación de Datos

Para eliminar registros de una tabla, utilizamos la cláusula DELETE.

-- Eliminar un registro de la tabla Empleados
DELETE FROM Empleados WHERE ID = 1;

Funciones y Operadores en SQL

Funciones Agregadas

SQL proporciona varias funciones agregadas que permiten realizar cálculos sobre un conjunto de valores.

  • COUNT: Cuenta el número de filas.
  • SUM: Suma los valores de una columna.
  • AVG: Calcula el promedio de los valores de una columna.
  • MAX: Encuentra el valor máximo.
  • MIN: Encuentra el valor mínimo.
-- Contar el número de empleados
SELECT COUNT(*) FROM Empleados;

-- Calcular la edad promedio de los empleados
SELECT AVG(Edad) FROM Empleados;

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores.

  • =: Igual a
  • <>: Diferente de
  • >: Mayor que
  • <: Menor que
  • >=: Mayor o igual que
  • <=: Menor o igual que
-- Seleccionar empleados mayores de 25 años
SELECT * FROM Empleados WHERE Edad > 25;

Ejercicios Prácticos

  1. Crear una tabla de Productos:

    • Definir una tabla llamada Productos con las columnas ID, Nombre, Precio, y Cantidad.

  2. Insertar registros en la tabla Productos:

    • Insertar al menos tres registros en la tabla Productos.

  3. Consultar datos específicos:

    • Seleccionar todos los productos cuyo precio sea mayor a 50.

  4. Actualizar registros:

    • Actualizar la cantidad de un producto específico.

  5. Eliminar registros:

    • Eliminar un producto de la tabla Productos.

Conclusión

El lenguaje SQL es una herramienta poderosa y esencial para cualquier profesional que trabaje con bases de datos. A través de este tema, hemos cubierto las operaciones básicas de SQL, incluyendo la creación de bases de datos y tablas, la inserción, consulta, actualización y eliminación de datos. Además, hemos explorado funciones agregadas y operadores de comparación que permiten realizar consultas más complejas y obtener información valiosa de nuestras bases de datos. Practicar estos conceptos es fundamental para dominar SQL y aplicarlo eficazmente en el manejo de datos.
