Introducción

Antes de sumergirte en BigQuery, es esencial configurar tu cuenta de Google Cloud. Esta sección te guiará a través del proceso, asegurándote de que tienes todo lo necesario para comenzar a trabajar con BigQuery.

Creación de una cuenta de Google Cloud

Para usar BigQuery, primero necesitas una cuenta de Google Cloud. Sigue estos pasos para crear una:

  1. Visita el sitio web de Google Cloud: Ve a Google Cloud.
  2. Regístrate: Haz clic en el botón "Comenzar gratis".
  3. Cuenta de Google: Si ya tienes una cuenta de Google, inicia sesión. Si no, crea una nueva cuenta de Google.
  4. Información de facturación: Ingresa tu información de facturación. Google Cloud ofrece un nivel gratuito con $300 en créditos para nuevos usuarios, que puedes usar para explorar BigQuery y otros servicios.

Configuración de un proyecto

En Google Cloud, los recursos se organizan en proyectos. Necesitas crear un proyecto para usar BigQuery.

  1. Navega a la consola: Ve a la Consola de Google Cloud.
  2. Crea un nuevo proyecto:
    • Haz clic en el menú desplegable de proyectos en la parte superior de la página.
    • Selecciona "Nuevo proyecto".
    • Ingresa un nombre para el proyecto y selecciona tu cuenta de facturación.
    • Haz clic en "Crear".

Habilitar la API de BigQuery

Para usar BigQuery, necesitas habilitar la API de BigQuery para tu proyecto.

  1. API y servicios: En la Consola de Google Cloud, navega a "API y servicios" > "Biblioteca".
  2. Busca BigQuery: Escribe "BigQuery" en la barra de búsqueda.
  3. Habilitar API: Haz clic en "API de BigQuery" y luego haz clic en "Habilitar".

Configuración de la facturación

BigQuery cobra por el almacenamiento de datos, consultas y procesamiento de datos. Asegúrate de que tu facturación esté configurada correctamente.

  1. Sección de facturación: En la Consola de Google Cloud, ve a "Facturación".
  2. Vincular cuenta de facturación: Vincula tu proyecto a una cuenta de facturación. Si no tienes una, necesitarás crearla.

Creación de una cuenta de servicio

Las cuentas de servicio se utilizan para autenticar y autorizar aplicaciones para acceder a los recursos de Google Cloud.

  1. IAM y administración: En la Consola de Google Cloud, navega a "IAM y administración" > "Cuentas de servicio".
  2. Crear cuenta de servicio:
    • Haz clic en "Crear cuenta de servicio".
    • Ingresa un nombre y una descripción.
    • Haz clic en "Crear".
  3. Asignar roles: Asigna los roles necesarios a la cuenta de servicio. Para BigQuery, podrías necesitar roles como "Usuario de BigQuery" o "Administrador de BigQuery".
  4. Crear clave: Crea una clave para la cuenta de servicio. Elige el formato JSON y descarga el archivo de clave. Este archivo se utilizará para autenticar tus aplicaciones.

Acceso a BigQuery

Puedes acceder a BigQuery a través de varios métodos:

  • Consola de BigQuery: La interfaz web disponible en la Consola de Google Cloud.
  • Herramienta de línea de comandos bq: Una herramienta de línea de comandos para interactuar con BigQuery.
  • Bibliotecas cliente: Bibliotecas disponibles para varios lenguajes de programación como Python, Java y Node.js.

Ejemplo: Acceso a BigQuery mediante Python

Para acceder a BigQuery usando Python, necesitas instalar la biblioteca cliente de Google Cloud BigQuery y autenticarte usando la clave de la cuenta de servicio.

  1. Instalar la biblioteca cliente:

    pip install google-cloud-bigquery

  2. Autenticar y consultar BigQuery:

    from google.cloud import bigquery

# Configurar la autenticación
client = bigquery.Client.from_service_account_json('path/to/service_account_key.json')

# Definir una consulta
query = """
SELECT name, SUM(number) as total
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
GROUP BY name
ORDER BY total DESC
LIMIT 10
"""

# Ejecutar la consulta
query_job = client.query(query)

# Imprimir los resultados
for row in query_job:
    print(f"{row.name}: {row.total}")

Conclusión

Configurar tu cuenta de Google Cloud es el primer paso para usar BigQuery. Siguiendo los pasos descritos anteriormente, tendrás un proyecto listo con BigQuery habilitado, facturación configurada y una cuenta de servicio creada para la autenticación. Con esta base, puedes comenzar a explorar y consultar datos en BigQuery.
