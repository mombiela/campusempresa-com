Introducción

Google BigQuery es un almacén de datos completamente gestionado y sin servidor que permite consultas SQL súper rápidas utilizando la potencia de procesamiento de la infraestructura de Google. La consola de BigQuery es la interfaz web para interactuar con BigQuery. Esta sección proporcionará una descripción general de la consola de BigQuery, guiándote a través de sus principales características y funcionalidades.

Características clave de la consola de BigQuery

  • Gestión de proyectos y conjuntos de datos: Organiza tus datos en proyectos y conjuntos de datos.
  • Editor de consultas: Escribe y ejecuta consultas SQL.
  • Monitoreo de trabajos: Rastrea el estado de tus consultas y otros trabajos.
  • Gestión de tablas: Crea, visualiza y gestiona tablas.
  • Importación/Exportación de datos: Carga datos en BigQuery y exporta resultados.

Navegando por la consola de BigQuery

Gestión de proyectos y conjuntos de datos

Cuando abres por primera vez la consola de BigQuery, verás una lista de tus proyectos en el lado izquierdo. Cada proyecto puede contener múltiples conjuntos de datos, y cada conjunto de datos puede contener múltiples tablas.

  • Proyectos: El contenedor de nivel superior para tus recursos de BigQuery.
  • Conjuntos de datos: Contenedores dentro de los proyectos que organizan las tablas.
  • Tablas: Las unidades de almacenamiento de datos reales dentro de los conjuntos de datos.

Editor de consultas

El editor de consultas es donde escribes y ejecutas consultas SQL. Proporciona resaltado de sintaxis, autocompletado y verificación de errores para ayudarte a escribir consultas de manera más eficiente.

-- Consulta de ejemplo: Seleccionando datos de una tabla
SELECT name, age
FROM `my_project.my_dataset.my_table`
WHERE age > 30;
  • Resaltado de sintaxis: Las diferentes partes de la sintaxis SQL están coloreadas de manera diferente para hacer que el código sea más fácil de leer.
  • Autocompletado: A medida que escribes, el editor sugiere posibles completaciones para nombres de tablas, nombres de columnas y palabras clave de SQL.
  • Verificación de errores: El editor resalta errores de sintaxis y proporciona sugerencias para corregirlos.

Monitoreo de trabajos

Después de ejecutar una consulta, puedes monitorear su progreso en la sección de Monitoreo de trabajos. Esta sección muestra una lista de todos tus trabajos recientes, incluidas consultas, cargas de datos y exportaciones.

  • ID de trabajo: Un identificador único para cada trabajo.
  • Estado: Indica si el trabajo está en ejecución, completado o fallido.
  • Hora de inicio: Cuándo comenzó el trabajo.
  • Duración: Cuánto tiempo tomó completar el trabajo.

Gestión de tablas

La sección de Gestión de tablas te permite crear nuevas tablas, ver tablas existentes y gestionar esquemas de tablas.

  • Creación de tablas: Puedes crear una nueva tabla especificando el nombre de la tabla, el esquema y otras propiedades.
  • Visualización de tablas: Haz clic en una tabla para ver su esquema, previsualizar sus datos y ver metadatos como la hora de creación y el tamaño.
  • Gestión de esquemas: Puedes agregar, eliminar o modificar columnas en el esquema de una tabla.

Importación/Exportación de datos

BigQuery admite varios métodos para importar y exportar datos.

  • Importación de datos: Carga datos desde Google Cloud Storage, Google Drive o archivos locales.
  • Exportación de datos: Exporta resultados de consultas a Google Cloud Storage o descárgalos como archivos CSV.
-- Ejemplo: Cargando datos desde Google Cloud Storage
LOAD DATA INTO `my_project.my_dataset.my_table`
FROM 'gs://my_bucket/my_file.csv'
WITH
  FORMAT = CSV,
  FIELD_DELIMITER = ',',
  SKIP_LEADING_ROWS = 1;

Conclusión

La consola de BigQuery es una herramienta poderosa para gestionar tu almacén de datos, escribir y ejecutar consultas, y monitorear trabajos. Al comprender sus características y funcionalidades clave, puedes navegar y utilizar BigQuery de manera eficiente para satisfacer tus necesidades de análisis de datos. En las siguientes secciones, profundizaremos en cada una de estas características, proporcionando ejemplos detallados y ejercicios para ayudarte a convertirte en un experto en el uso de BigQuery.
