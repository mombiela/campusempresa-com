Introducción

Las funciones de agregación son herramientas esenciales en SQL y BigQuery para resumir y analizar datos. Permiten realizar cálculos en múltiples filas de datos y devolver un solo valor. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de las funciones de agregación en BigQuery.

Funciones de Agregación Clave

  • COUNT(): Cuenta el número de filas.
  • SUM(): Calcula la suma total de una columna numérica.
  • AVG(): Calcula el valor promedio de una columna numérica.
  • MIN(): Encuentra el valor mínimo en una columna.
  • MAX(): Encuentra el valor máximo en una columna.

Uso Básico

COUNT()

La función COUNT() devuelve el número de filas que coinciden con una condición especificada.

SELECT COUNT(*) AS total_rows
FROM `project.dataset.table`;

SUM()

La función SUM() suma todos los valores en una columna numérica.

SELECT SUM(sales) AS total_sales
FROM `project.dataset.table`;

AVG()

La función AVG() calcula el valor promedio de una columna numérica.

SELECT AVG(sales) AS average_sales
FROM `project.dataset.table`;

MIN() y MAX()

Las funciones MIN() y MAX() devuelven los valores más pequeños y más grandes en una columna, respectivamente.

SELECT MIN(sales) AS minimum_sales, MAX(sales) AS maximum_sales
FROM `project.dataset.table`;

Agrupando Datos con Funciones de Agregación

Cláusula GROUP BY

Para aplicar funciones de agregación a subconjuntos de datos, usa la cláusula GROUP BY.

SELECT region, SUM(sales) AS total_sales
FROM `project.dataset.table`
GROUP BY region;

Cláusula HAVING

La cláusula HAVING filtra grupos basados en los resultados de la función de agregación.

SELECT region, SUM(sales) AS total_sales
FROM `project.dataset.table`
GROUP BY region
HAVING total_sales > 10000;

Uso Avanzado

Combinando Múltiples Funciones de Agregación

Puedes combinar múltiples funciones de agregación en una sola consulta.

SELECT region, 
       COUNT(*) AS total_orders, 
       SUM(sales) AS total_sales, 
       AVG(sales) AS average_sales
FROM `project.dataset.table`
GROUP BY region;

Funciones de Ventana

Las funciones de ventana te permiten realizar cálculos de agregación a través de un conjunto de filas de la tabla que están de alguna manera relacionadas con la fila actual.

SELECT region, 
       sales, 
       SUM(sales) OVER (PARTITION BY region) AS total_sales_per_region
FROM `project.dataset.table`;

Manejo de Valores NULL

Las funciones de agregación generalmente ignoran los valores NULL. Sin embargo, puedes usar COALESCE() para manejar los valores NULL.

SELECT region, 
       SUM(COALESCE(sales, 0)) AS total_sales
FROM `project.dataset.table`
GROUP BY region;

Resumen

Las funciones de agregación en BigQuery son herramientas poderosas para resumir y analizar datos. Al dominar funciones como COUNT(), SUM(), AVG(), MIN() y MAX(), y entender cómo usarlas con las cláusulas GROUP BY y HAVING, puedes realizar análisis de datos complejos de manera eficiente. Las técnicas avanzadas como las funciones de ventana y el manejo de valores NULL mejoran aún más tu capacidad para derivar conocimientos de tus datos.
