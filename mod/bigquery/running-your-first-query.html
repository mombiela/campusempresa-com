Introducción

BigQuery es un almacén de datos completamente gestionado y sin servidor que permite un análisis escalable sobre petabytes de datos. En esta sección, te guiaremos a través de la ejecución de tu primera consulta en BigQuery. Esto incluirá entender la interfaz, escribir una consulta SQL básica e interpretar los resultados.

Configuración de tu entorno

Antes de ejecutar tu primera consulta, asegúrate de tener acceso a Google Cloud Platform (GCP) y BigQuery. Sigue estos pasos:

  1. Crea una cuenta de Google Cloud: Si no tienes una, regístrate en Google Cloud.
  2. Habilita la API de BigQuery: Navega a la biblioteca de API en la Consola de GCP y habilita la API de BigQuery.
  3. Crea un proyecto: En la Consola de GCP, crea un nuevo proyecto o selecciona uno existente.
  4. Accede a BigQuery: Ve a la sección de BigQuery en la Consola de GCP.

Entendiendo la interfaz de BigQuery

La interfaz de BigQuery consta de varios componentes clave:

  • Panel de navegación: Ubicado a la izquierda, muestra tus proyectos y conjuntos de datos.
  • Editor de consultas: El área central donde escribes y ejecutas consultas SQL.
  • Panel de resultados: Muestra los resultados de tus consultas.
  • Historial de trabajos: Muestra el historial de tus consultas ejecutadas.

Escribiendo tu primera consulta

Vamos a escribir una consulta simple para recuperar datos de un conjunto de datos público. Usaremos el proyecto bigquery-public-data, que contiene varios conjuntos de datos para practicar.

Consulta de ejemplo

Consultaremos el conjunto de datos samples y la tabla shakespeare para encontrar las 10 palabras más frecuentes en las obras de Shakespeare.

SELECT
  word,
  word_count
FROM
  `bigquery-public-data.samples.shakespeare`
ORDER BY
  word_count DESC
LIMIT
  10;

Explicación

  • SELECT: Especifica las columnas a recuperar (word y word_count).
  • FROM: Indica el conjunto de datos y la tabla (bigquery-public-data.samples.shakespeare).
  • ORDER BY: Ordena los resultados por word_count en orden descendente.
  • LIMIT: Restringe los resultados a las 10 filas superiores.

Ejecutando la consulta

  1. Copia la consulta SQL anterior.
  2. Pégala en el Editor de Consultas en la interfaz de BigQuery.
  3. Haz clic en el botón "Run".

Interpretando los resultados

Después de ejecutar la consulta, el Panel de Resultados mostrará las 10 palabras más frecuentes en las obras de Shakespeare junto con sus conteos. Los resultados deberían verse algo así:

| palabra | conteo_palabra | |---------|----------------| | the | 27,378 | | and | 24,850 | | I | 22,228 | | to | 20,108 | | of | 18,971 | | a | 14,701 | | my | 14,598 | | in | 11,348 | | you | 10,759 | | that | 10,151 |

Resumen

En esta sección, aprendiste cómo configurar tu entorno, navegar por la interfaz de BigQuery, escribir una consulta SQL básica e interpretar los resultados. Ejecutar consultas en BigQuery es una forma poderosa de analizar grandes conjuntos de datos de manera rápida y eficiente. A medida que avances, aprenderás técnicas de consulta más avanzadas y optimizaciones.
