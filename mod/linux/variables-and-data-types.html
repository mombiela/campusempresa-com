Introducción

En esta sección, cubriremos los conceptos fundamentales de variables y tipos de datos en el contexto de la programación en Linux. Entender estos conceptos es crucial para escribir scripts y programas eficientes y efectivos.

Variables

¿Qué es una Variable?

  • Una variable es un nombre simbólico asociado con un valor y cuyo valor asociado puede ser cambiado.
  • En Linux, las variables se usan en scripts de shell para almacenar datos que pueden ser usados y manipulados a lo largo del script.

Declaración de Variables

  • Las variables en scripts de shell de Linux no requieren una declaración explícita de sus tipos de datos.
  • Para declarar una variable, simplemente asigna un valor a ella usando el operador =.
#!/bin/bash
# Declarando una variable
mi_variable="¡Hola, Mundo!"
echo $mi_variable

Reglas de Nombres de Variables

  • Los nombres de las variables pueden contener letras (a-z, A-Z), números (0-9) y guiones bajos (_).
  • Los nombres de las variables deben comenzar con una letra o un guion bajo.
  • Los nombres de las variables son sensibles a mayúsculas y minúsculas.

Uso de Variables

  • Para usar el valor de una variable, antepone el nombre de la variable con un símbolo $.
#!/bin/bash
# Usando una variable
saludo="Hola"
nombre="Alicia"
echo "$saludo, $nombre!"

Tipos de Datos

Visión General

  • A diferencia de muchos lenguajes de programación, los scripts de shell en Linux no requieren declaraciones explícitas de tipos de datos.
  • El shell interpreta el tipo de datos basado en el contexto en el que se usa la variable.

Tipos de Datos Comunes

  1. Cadenas
    • Una secuencia de caracteres.
    • Encerradas en comillas dobles (") o comillas simples (').
#!/bin/bash
# Variable de cadena
cadena_var="Esto es una cadena"
echo $cadena_var
  1. Enteros
    • Números enteros sin componente fraccional.
    • Usados para operaciones aritméticas.
#!/bin/bash
# Variable entera
num1=5
num2=10
suma=$((num1 + num2))
echo "Suma: $suma"
  1. Arreglos
    • Una colección de valores indexados por números.
    • Declarados usando paréntesis ().
#!/bin/bash
# Variable de arreglo
mi_arreglo=(manzana banana cereza)
echo "Primer elemento: ${mi_arreglo[0]}"
echo "Todos los elementos: ${mi_arreglo[@]}"

Conversión de Tipos

  • Los scripts de shell no soportan la conversión explícita de tipos.
  • El shell convierte automáticamente entre tipos basado en el contexto.

Comparación de Tipos de Datos

| Tipo de Dato | Descripción | Ejemplo | |--------------|---------------------------------|--------------------------| | Cadena | Secuencia de caracteres | nombre="Juan Pérez" | | Entero | Números enteros | edad=30 | | Arreglo | Colección indexada de valores | frutas=(manzana banana)|

Ejemplos y Ejercicios

Ejemplo 1: Manipulación de Cadenas

#!/bin/bash
# Concatenando cadenas
primer_nombre="Juan"
apellido="Pérez"
nombre_completo="$primer_nombre $apellido"
echo "Nombre Completo: $nombre_completo"

Ejemplo 2: Operaciones Aritméticas

#!/bin/bash
# Realizando operaciones aritméticas
a=15
b=5
suma=$((a + b))
diferencia=$((a - b))
producto=$((a * b))
cociente=$((a / b))
echo "Suma: $suma"
echo "Diferencia: $diferencia"
echo "Producto: $producto"
echo "Cociente: $cociente"

Ejercicio 1: Crear y Usar Variables

  • Crea un script que declare tres variables: nombre, edad y ciudad.
  • Asigna valores apropiados a estas variables.
  • Imprime una oración usando estas variables, por ejemplo, "Mi nombre es Juan, tengo 25 años y vivo en Nueva York."

Ejercicio 2: Manipulación de Arreglos

  • Crea un script que declare un arreglo de tus frutas favoritas.
  • Imprime la primera fruta en el arreglo.
  • Imprime todas las frutas en el arreglo.

Conclusión

Entender las variables y los tipos de datos es esencial para la escritura efectiva de scripts de shell en Linux. Las variables te permiten almacenar y manipular datos, mientras que los tipos de datos te ayudan a entender cómo trabajar con diferentes tipos de datos. Practica usando variables y diferentes tipos de datos para volverte competente en la escritura de scripts de shell.
