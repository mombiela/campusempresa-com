Introducción a la Programación de Scripts en Shell
Variables y Tipos de Datos ►

Resumen

La programación de scripts en shell es una forma poderosa de automatizar tareas en un entorno Linux. Este tema te guiará a través de los conceptos básicos de la programación de scripts en shell, desde entender qué es un shell hasta escribir tu primer script y más allá.

¿Qué es un Shell?

  • Definición: Un shell es un intérprete de línea de comandos que proporciona una interfaz de usuario para el sistema operativo Unix/Linux.
  • Tipos de Shells:
    • Bourne Shell (sh)
    • Bourne Again Shell (bash)
    • C Shell (csh)
    • Korn Shell (ksh)
    • Z Shell (zsh)

¿Por qué Usar Scripts en Shell?

  • Automatización: Automatiza tareas repetitivas.
  • Eficiencia: Realiza tareas complejas con comandos simples.
  • Personalización: Adapta tu entorno a tus necesidades.
  • Integración: Combina múltiples comandos y utilidades.

Comandos Básicos del Shell

  • Navegación: cd, ls, pwd
  • Operaciones de Archivos: cp, mv, rm, touch
  • Visualización de Archivos: cat, more, less, head, tail
  • Permisos: chmod, chown

Escribiendo Tu Primer Script en Shell

  1. Crear un Archivo de Script: 
    touch myscript.sh
  2. Hacer el Script Ejecutable: 
    chmod +x myscript.sh
  3. Editar el Script: 
    nano myscript.sh
  4. Agregar la Línea Shebang: La línea shebang le dice al sistema qué intérprete usar. 
    #!/bin/bash
  5. Agregar Comandos: 
    #!/bin/bash
echo "¡Hola, Mundo!"
  6. Ejecutar el Script: 
    ./myscript.sh

Variables

  • Definición: Las variables almacenan datos que pueden ser usados y manipulados dentro de un script.
  • Sintaxis: 
    NOMBRE_VARIABLE="valor"
echo $NOMBRE_VARIABLE

Declaraciones Condicionales

  • If-Else: 
    #!/bin/bash
if [ $1 -gt 10 ]; then
    echo "El número es mayor que 10"
else
    echo "El número es 10 o menos"
fi
  • Case: 
    #!/bin/bash
case $1 in
    start)
        echo "Iniciando..."
        ;;
    stop)
        echo "Deteniendo..."
        ;;
    *)
        echo "Uso: $0 {start|stop}"
        ;;
esac

Bucles

  • Bucle For: 
    #!/bin/bash
for i in {1..5}; do
    echo "Bienvenido $i veces"
done
  • Bucle While: 
    #!/bin/bash
CONTADOR=0
while [ $CONTADOR -lt 5 ]; do
    echo "Contador: $CONTADOR"
    CONTADOR=$((CONTADOR + 1))
done

Funciones

  • Definición: Las funciones son bloques de código reutilizables.
  • Sintaxis: 
    #!/bin/bash
mi_funcion() {
    echo "Esto es una función"
}
mi_funcion

Entrada y Salida

  • Leer Entrada del Usuario: 
    #!/bin/bash
echo "Introduce tu nombre:"
read NOMBRE
echo "Hola, $NOMBRE"
  • Redirigir Salida: 
    #!/bin/bash
echo "Esto es una prueba" > salida.txt

Manejo de Errores

  • Estado de Salida: Cada comando devuelve un estado de salida (0 para éxito, distinto de cero para fallo). 
    #!/bin/bash
cp archivo1.txt archivo2.txt
if [ $? -eq 0 ]; then
    echo "Copia exitosa"
else
    echo "Copia fallida"
fi

Temas Avanzados

  • Arrays: 
    #!/bin/bash
NOMBRES=("Alicia" "Bob" "Carlos")
echo "Primer nombre: ${NOMBRES[0]}"
  • Expresiones Regulares: 
    #!/bin/bash
if [[ "hola" =~ ^h.*o$ ]]; then
    echo "El patrón coincide"
else
    echo "El patrón no coincide"
fi
  • Depuración: Usa set -x para habilitar la depuración. 
    #!/bin/bash
set -x
echo "Depuración habilitada"

Conclusión

La programación de scripts en shell es una habilidad esencial para cualquiera que trabaje en un entorno Linux. Te permite automatizar tareas, gestionar operaciones del sistema y personalizar tu flujo de trabajo. Al entender los conceptos básicos y practicar con ejemplos, puedes volverte competente en escribir y usar scripts en shell para mejorar tu productividad.
