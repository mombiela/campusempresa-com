Estructuras de Control
Funciones y Bibliotecas ►

Las estructuras de control son conceptos fundamentales en la programación que te permiten controlar el flujo de ejecución en tus scripts. En Linux, las estructuras de control son esenciales para escribir scripts de shell eficientes y efectivos. Este tema cubrirá los conceptos básicos de las estructuras de control, incluyendo condicionales, bucles y declaraciones case, progresando desde niveles principiantes hasta avanzados.

Condicionales

Los condicionales te permiten ejecutar ciertas partes de tu código basándote en condiciones específicas.

Declaraciones if

La declaración if se utiliza para probar una condición y ejecutar un bloque de código si la condición es verdadera.

Sintaxis:

if [ condition ]; then
    # Código a ejecutar si la condición es verdadera
fi

Ejemplo:

#!/bin/bash

number=10

if [ $number -gt 5 ]; then
    echo "El número es mayor que 5"
fi

Declaraciones if-else

La declaración if-else te permite ejecutar un bloque de código si la condición es verdadera y otro bloque si la condición es falsa.

Sintaxis:

if [ condition ]; then
    # Código a ejecutar si la condición es verdadera
else
    # Código a ejecutar si la condición es falsa
fi

Ejemplo:

#!/bin/bash

number=3

if [ $number -gt 5 ]; then
    echo "El número es mayor que 5"
else
    echo "El número no es mayor que 5"
fi

Declaraciones if-elif-else

La declaración if-elif-else te permite probar múltiples condiciones.

Sintaxis:

if [ condition1 ]; then
    # Código a ejecutar si condition1 es verdadera
elif [ condition2 ]; then
    # Código a ejecutar si condition2 es verdadera
else
    # Código a ejecutar si ninguna de las condiciones es verdadera
fi

Ejemplo:

#!/bin/bash

number=7

if [ $number -gt 10 ]; then
    echo "El número es mayor que 10"
elif [ $number -gt 5 ]; then
    echo "El número es mayor que 5 pero menor o igual a 10"
else
    echo "El número es 5 o menos"
fi

Bucles

Los bucles te permiten ejecutar un bloque de código múltiples veces.

Bucle for

El bucle for itera sobre una lista de elementos y ejecuta un bloque de código para cada elemento.

Sintaxis:

for item in list; do
    # Código a ejecutar para cada elemento
done

Ejemplo:

#!/bin/bash

for i in 1 2 3 4 5; do
    echo "Número: $i"
done

Bucle while

El bucle while ejecuta un bloque de código mientras una condición sea verdadera.

Sintaxis:

while [ condition ]; do
    # Código a ejecutar mientras la condición sea verdadera
done

Ejemplo:

#!/bin/bash

counter=1

while [ $counter -le 5 ]; do
    echo "Contador: $counter"
    ((counter++))
done

Bucle until

El bucle until ejecuta un bloque de código hasta que una condición se vuelva verdadera.

Sintaxis:

until [ condition ]; do
    # Código a ejecutar hasta que la condición sea verdadera
done

Ejemplo:

#!/bin/bash

counter=1

until [ $counter -gt 5 ]; do
    echo "Contador: $counter"
    ((counter++))
done

Declaraciones Case

La declaración case te permite ejecutar diferentes bloques de código basándote en el valor de una variable.

Sintaxis:

case $variable in
    pattern1)
        # Código a ejecutar si la variable coincide con pattern1
        ;;
    pattern2)
        # Código a ejecutar si la variable coincide con pattern2
        ;;
    *)
        # Código a ejecutar si la variable no coincide con ningún patrón
        ;;
esac

Ejemplo:

#!/bin/bash

day="Lunes"

case $day in
    "Lunes")
        echo "Inicio de la semana laboral"
        ;;
    "Viernes")
        echo "Fin de la semana laboral"
        ;;
    "Sábado" | "Domingo")
        echo "¡Fin de semana!"
        ;;
    *)
        echo "Día entre semana"
        ;;
esac

Conclusión

Las estructuras de control son esenciales para escribir scripts de shell efectivos en Linux. Al dominar los condicionales, bucles y declaraciones case, puedes crear scripts que tomen decisiones, repitan tareas y manejen múltiples escenarios de manera eficiente. Practica estos conceptos con varios ejemplos para volverte competente en la escritura de scripts de shell robustos.
