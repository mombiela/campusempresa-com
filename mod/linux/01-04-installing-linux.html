Instalando Linux
Siguiente ►

Instalar Linux puede parecer una tarea intimidante para los principiantes, pero con la guía adecuada, es un proceso bastante sencillo. En esta sección, cubriremos los pasos necesarios para instalar una distribución de Linux en tu computadora. Nos enfocaremos en Ubuntu, una de las distribuciones más populares y amigables para los nuevos usuarios.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una computadora con al menos 2 GB de RAM y 25 GB de espacio en disco.
  • Una unidad USB de al menos 4 GB.
  • Una conexión a Internet para descargar la imagen ISO de Ubuntu.

Paso 1: Descargar la Imagen ISO de Ubuntu

  1. Visita el sitio web oficial de Ubuntu: Ubuntu Downloads
  2. Selecciona la versión de Ubuntu que deseas instalar. Para principiantes, se recomienda la versión LTS (Long Term Support) por su estabilidad y soporte a largo plazo.
  3. Descarga la imagen ISO haciendo clic en el botón de descarga.

Paso 2: Crear un USB de Arranque

Para crear un USB de arranque, necesitarás una herramienta como Rufus (para Windows) o Etcher (para macOS y Linux).

Usando Rufus (Windows)

  1. Descarga e instala Rufus desde Rufus.
  2. Inserta la unidad USB en tu computadora.
  3. Abre Rufus y selecciona tu unidad USB en el campo "Dispositivo".
  4. Selecciona la imagen ISO de Ubuntu que descargaste en el campo "Selección de arranque".
  5. Haz clic en "Empezar" y sigue las instrucciones para crear el USB de arranque.

Usando Etcher (macOS y Linux)

  1. Descarga e instala Etcher desde Etcher.
  2. Inserta la unidad USB en tu computadora.
  3. Abre Etcher y selecciona la imagen ISO de Ubuntu.
  4. Selecciona tu unidad USB como destino.
  5. Haz clic en "Flash!" para crear el USB de arranque.

Paso 3: Arrancar desde el USB

  1. Reinicia tu computadora y entra en el menú de arranque. Esto generalmente se hace presionando una tecla como F12, F2, Esc o Del durante el arranque. Consulta el manual de tu computadora para obtener instrucciones específicas.
  2. Selecciona la unidad USB desde el menú de arranque para iniciar el instalador de Ubuntu.

Paso 4: Instalar Ubuntu

  1. Selecciona el idioma y haz clic en "Instalar Ubuntu".
  2. Conéctate a una red Wi-Fi si es necesario.
  3. Selecciona las opciones de instalación:
    • Actualizaciones y software de terceros: Se recomienda marcar estas opciones para obtener la mejor experiencia.
  4. Selecciona el tipo de instalación:
    • Borrar disco e instalar Ubuntu: Esta opción eliminará todos los datos en el disco y es la más sencilla para nuevos usuarios.
    • Instalación manual: Para usuarios avanzados que desean particionar el disco manualmente.
  5. Configura tu zona horaria y haz clic en "Continuar".
  6. Crea tu cuenta de usuario:
    • Nombre: Tu nombre completo.
    • Nombre de tu computadora: Un nombre para identificar tu computadora en la red.
    • Nombre de usuario: El nombre que usarás para iniciar sesión.
    • Contraseña: Una contraseña segura para tu cuenta.
  7. Haz clic en "Continuar" y espera a que se complete la instalación.

Paso 5: Reiniciar y Configurar

  1. Reinicia tu computadora cuando se te indique y retira la unidad USB.
  2. Inicia sesión con tu nombre de usuario y contraseña.
  3. Configura las actualizaciones y el software adicional según sea necesario.

Conclusión

¡Felicidades! Has instalado Ubuntu en tu computadora. Ahora estás listo para explorar el sistema operativo y comenzar a aprender más sobre Linux. En el próximo módulo, nos adentraremos en los comandos básicos de Linux, que te permitirán navegar y operar tu nuevo sistema de manera eficiente.

Resumen del Módulo:

  • Descargamos la imagen ISO de Ubuntu.
  • Creamos un USB de arranque usando herramientas como Rufus o Etcher.
  • Arrancamos desde el USB y seguimos los pasos del instalador de Ubuntu.
  • Configuramos nuestra cuenta de usuario y completamos la instalación.

En el siguiente módulo, aprenderemos sobre los comandos básicos de Linux, que son esenciales para interactuar con el sistema operativo desde la línea de comandos.
