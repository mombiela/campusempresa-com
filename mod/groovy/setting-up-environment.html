Configurar el entorno de desarrollo es un primer paso crucial para aprender Groovy. Esta sección te guiará a través del proceso de instalación de Groovy, configuración de un Entorno de Desarrollo Integrado (IDE) y verificación de tu configuración.

Instalación de Groovy

Para comenzar a programar en Groovy, necesitas instalar el lenguaje Groovy en tu máquina. Sigue estos pasos para instalar Groovy:

  1. Descargar Groovy:

    • Visita el sitio web oficial de Groovy.
    • Descarga la última versión estable adecuada para tu sistema operativo.

  2. Instalar Groovy:

    • Windows: Ejecuta el instalador descargado y sigue las instrucciones en pantalla.
    • macOS: Usa Homebrew para instalar Groovy ejecutando el siguiente comando en tu terminal: 
      brew install groovy
    • Linux: Usa SDKMAN para instalar Groovy ejecutando los siguientes comandos: 
      curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install groovy

  3. Verificar la Instalación:

    • Abre una terminal o símbolo del sistema.
    • Escribe groovy -version y presiona Enter.
    • Deberías ver la versión de Groovy instalada.

Configuración de un Entorno de Desarrollo Integrado (IDE)

Aunque puedes escribir código Groovy en cualquier editor de texto, usar un IDE puede mejorar significativamente tu productividad. Aquí hay algunos IDEs populares que soportan Groovy:

  • IntelliJ IDEA:

    • Descarga e instala IntelliJ IDEA desde el sitio web oficial.
    • Abre IntelliJ IDEA y configúralo para soportar Groovy instalando el plugin de Groovy desde el repositorio de plugins.

  • Eclipse:

    • Descarga e instala Eclipse desde el sitio web oficial.
    • Instala el plugin Groovy Development Tools (GDT) desde el Eclipse Marketplace.

  • Visual Studio Code:

    • Descarga e instala Visual Studio Code desde el sitio web oficial.
    • Instala la extensión de Groovy desde el Marketplace de Extensiones.

Ejemplo: Configuración de Groovy en IntelliJ IDEA

  1. Instalar IntelliJ IDEA:

    • Descarga e instala IntelliJ IDEA desde el sitio web oficial.

  2. Configurar Groovy SDK:

    • Abre IntelliJ IDEA.
    • Ve a File > Project Structure > SDKs.
    • Haz clic en el botón + y selecciona Groovy SDK.
    • Navega al directorio donde está instalado Groovy y selecciónalo.

  3. Crear un Proyecto Groovy:

    • Ve a File > New > Project.
    • Selecciona Groovy de la lista de tipos de proyectos.
    • Sigue las indicaciones para crear un nuevo proyecto Groovy.

  4. Escribir y Ejecutar un Script Groovy:

    • Crea un nuevo archivo Groovy haciendo clic derecho en la carpeta src y seleccionando New > Groovy Class.
    • Nombra el archivo HelloWorld.
    • Agrega el siguiente código al archivo: 
      class HelloWorld {
    static void main(String[] args) {
        println 'Hello, Groovy!'
    }
}
    • Ejecuta el script haciendo clic derecho en el archivo y seleccionando Run 'HelloWorld.main()'.

Verificación de tu Configuración

Para asegurarte de que todo está configurado correctamente, crea y ejecuta un script Groovy simple:

  1. Crear un Script Groovy:

    • Abre tu IDE o editor de texto.
    • Crea un nuevo archivo llamado HelloGroovy.groovy.

  2. Escribir el Script:

    • Agrega el siguiente código al archivo: 
      println 'Hello, Groovy!'

  3. Ejecutar el Script:

    • Abre una terminal o símbolo del sistema.
    • Navega al directorio donde está guardado el script.
    • Ejecuta el script usando el siguiente comando: 
      groovy HelloGroovy.groovy
    • Deberías ver la salida Hello, Groovy!.

Conclusión

Configurar el entorno de desarrollo para Groovy implica instalar el lenguaje Groovy, elegir y configurar un IDE, y verificar la configuración con un script simple. Siguiendo estos pasos, te aseguras de estar listo para comenzar a programar en Groovy de manera eficiente. En las próximas secciones, profundizaremos en la sintaxis y características de Groovy, construyendo sobre esta sólida base.
