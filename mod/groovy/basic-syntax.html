Introducción

Groovy es un lenguaje de programación orientado a objetos para la plataforma Java. Se compila dinámicamente a bytecode de Java e integra perfectamente con Java. Entender la sintaxis básica de Groovy es esencial para escribir código efectivo y eficiente. Esta sección cubrirá los elementos fundamentales de la sintaxis de Groovy, incluyendo variables, tipos de datos, operadores y estructuras de control.

Variables y Tipos de Datos

Groovy es tipado dinámicamente, lo que significa que no tienes que declarar explícitamente el tipo de una variable. Sin embargo, aún puedes usar tipado estático si lo prefieres.

Declaración de Variables

  • Tipado Dinámico: Puedes declarar una variable sin especificar su tipo.
  • Tipado Estático: Puedes declarar una variable con un tipo específico.
// Tipado Dinámico
def name = "Groovy"
def age = 25

// Tipado Estático
String language = "Groovy"
int year = 2023

Tipos de Datos

Groovy soporta varios tipos de datos, incluyendo:

  • Números: Integer, Float, Double, BigDecimal
  • Cadenas: Comillas simples, Comillas dobles, Comillas triples
  • Booleanos: true, false
  • Colecciones: Listas, Mapas, Rangos
// Números
int num = 10
float pi = 3.14
BigDecimal bigNum = 12345678901234567890.12345678901234567890

// Cadenas
String singleQuote = 'Hello'
String doubleQuote = "World"
String tripleQuote = """Esto es una
cadena de múltiples líneas"""

// Booleanos
boolean isGroovy = true

// Colecciones
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ['Alice': 30, 'Bob': 25]
Range<Integer> range = 1..5

Operadores

Groovy soporta una amplia gama de operadores, incluyendo aritméticos, relacionales, lógicos y de bits.

Operadores Aritméticos

  • Adición (+)
  • Sustracción (-)
  • Multiplicación (*)
  • División (/)
  • Módulo (%)
int a = 10
int b = 5

println a + b  // 15
println a - b  // 5
println a * b  // 50
println a / b  // 2
println a % b  // 0

Operadores Relacionales

  • Igual a (==)
  • No igual a (!=)
  • Mayor que (>)
  • Menor que (<)
  • Mayor o igual a (>=)
  • Menor o igual a (<=)
int x = 10
int y = 20

println x == y  // false
println x != y  // true
println x > y   // false
println x < y   // true
println x >= y  // false
println x <= y  // true

Operadores Lógicos

  • Y (&&)
  • O (||)
  • NO (!)
boolean isTrue = true
boolean isFalse = false

println isTrue && isFalse  // false
println isTrue || isFalse  // true
println !isTrue            // false

Estructuras de Control

Groovy proporciona varias estructuras de control para gestionar el flujo del programa, incluyendo declaraciones condicionales y bucles.

Declaraciones Condicionales

  • if-else: Ejecuta un bloque de código si una condición es verdadera, de lo contrario ejecuta otro bloque de código.
int score = 85

if (score >= 90) {
    println "A"
} else if (score >= 80) {
    println "B"
} else {
    println "C"
}

Bucles

  • bucle for: Itera sobre un rango, lista o mapa.
  • bucle while: Repite un bloque de código mientras una condición sea verdadera.
// bucle for
for (int i = 1; i <= 5; i++) {
    println i
}

// bucle while
int count = 1
while (count <= 5) {
    println count
    count++
}

Conclusión

Entender la sintaxis básica de Groovy es crucial para escribir código efectivo. Esta sección cubrió la declaración de variables, tipos de datos, operadores y estructuras de control. Con estos fundamentos, puedes comenzar a escribir programas simples en Groovy y gradualmente avanzar a tareas más complejas. En las siguientes secciones, profundizaremos en características y capacidades más avanzadas de Groovy.
