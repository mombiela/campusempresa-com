Groovy es un lenguaje poderoso, opcionalmente tipado y dinámico, con capacidades de tipado estático y compilación estática, para la plataforma Java, destinado a mejorar la productividad del desarrollador. Puede ser utilizado tanto como un lenguaje de scripting como para escribir aplicaciones completas. Esta sección se centrará en el uso de Groovy para scripting, cubriendo desde lo básico hasta técnicas de scripting más avanzadas.

Introducción al Scripting con Groovy

Los scripts de Groovy se utilizan típicamente para automatizar tareas, manipular archivos y realizar cálculos rápidos. A menudo se utilizan en herramientas de automatización de compilación como Gradle.

Conceptos Clave

  • Tipado Dinámico: Groovy te permite escribir scripts sin definir explícitamente los tipos.
  • Azúcar Sintáctico: Groovy proporciona una sintaxis más concisa y legible en comparación con Java.
  • Interoperabilidad: Groovy puede integrarse sin problemas con bibliotecas y frameworks de Java.

Script de Hola Mundo

Comencemos con un simple script de "Hola Mundo" en Groovy.

println '¡Hola, Mundo!'

Este script demuestra la simplicidad de la sintaxis de Groovy. La función println imprime la cadena en la consola.

Variables y Tipos de Datos

Groovy soporta varios tipos de datos y permite el tipado dinámico, lo que lo hace flexible para scripting.

Declaración de Variables

Puedes declarar variables sin especificar su tipo.

def name = 'Groovy'
def age = 10
println "Nombre: $name, Edad: $age"

Tipos de Datos Comunes

  • String: Datos textuales.
  • Integer: Números enteros.
  • Float/Double: Números decimales.
  • Boolean: Valores verdaderos o falsos.

Ejemplo: Tipos de Datos

def message = '¡Hola, Groovy!'
def number = 42
def price = 19.99
def isAvailable = true

println "Mensaje: $message"
println "Número: $number"
println "Precio: $price"
println "Disponible: $isAvailable"

Estructuras de Control

Groovy soporta estructuras de control estándar como bucles y condicionales.

Sentencias If-Else

def number = 10

if (number > 0) {
    println 'Número positivo'
} else {
    println 'Número no positivo'
}

Bucles For

for (i in 1..5) {
    println "Iteración: $i"
}

Bucles While

def count = 5

while (count > 0) {
    println "Cuenta: $count"
    count--
}

Funciones y Closures

Las funciones y closures son esenciales para escribir código reutilizable en scripts de Groovy.

Definición de Funciones

def greet(name) {
    return "¡Hola, $name!"
}

println greet('Groovy')

Uso de Closures

Los closures son bloques de código anónimos que pueden ser asignados a variables y pasados alrededor.

def square = { number -> number * number }

println square(4)  // Salida: 16

Operaciones con Archivos

Groovy facilita el trabajo con archivos, lo cual es una tarea común en scripting.

Lectura de Archivos

def file = new File('example.txt')
file.eachLine { line ->
    println line
}

Escritura de Archivos

def file = new File('output.txt')
file.withWriter { writer ->
    writer.writeLine('¡Hola, Groovy!')
}

Técnicas Avanzadas de Scripting

A medida que te sientas más cómodo con el scripting en Groovy, puedes explorar técnicas más avanzadas.

Uso de Groovy Shell

El Groovy Shell (groovysh) es una shell interactiva de línea de comandos para evaluar expresiones Groovy.

groovy:000> println '¡Hola desde Groovy Shell!'

Integración con Bibliotecas Java

Groovy puede utilizar fácilmente bibliotecas Java, lo que lo convierte en un lenguaje de scripting versátil.

import java.util.Date

def now = new Date()
println "Fecha y hora actual: $now"

Conclusión

Groovy es un lenguaje de scripting versátil y poderoso que puede mejorar significativamente tu productividad. Su tipado dinámico, sintaxis concisa e integración sin problemas con Java lo convierten en una excelente opción para diversas tareas de scripting. Al dominar lo básico y explorar técnicas avanzadas, puedes aprovechar Groovy para automatizar tareas, manipular archivos e integrarte eficazmente con bibliotecas Java.
