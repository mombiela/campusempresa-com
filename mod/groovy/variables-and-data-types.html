Variables y Tipos de Datos en Groovy
Operadores ►

Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en Groovy. Entender estos conceptos es crucial ya que forman la base de cualquier lenguaje de programación. Comenzaremos con lo básico y gradualmente avanzaremos a temas más avanzados.

Variables

Las variables se utilizan para almacenar datos que pueden ser manipulados y recuperados a lo largo de tu programa.

Declaración de Variables

  • En Groovy, puedes declarar una variable usando la palabra clave def o especificando el tipo explícitamente.
  • Groovy es un lenguaje de tipado dinámico, lo que significa que no siempre necesitas especificar el tipo de una variable.

Usando la Palabra Clave def

def name = "John Doe"
def age = 30
def isStudent = true

Especificando el Tipo

String name = "John Doe"
int age = 30
boolean isStudent = true

Reglas de Nomenclatura de Variables

  • Los nombres de las variables deben comenzar con una letra, un signo de dólar $, o un guion bajo _.
  • Los caracteres subsiguientes pueden ser letras, dígitos, signos de dólar o guiones bajos.
  • Los nombres de las variables son sensibles a mayúsculas y minúsculas.

Tipos de Datos

Groovy soporta varios tipos de datos, que pueden ser categorizados en tipos primitivos y tipos de objeto.

Tipos de Datos Primitivos

  • byte: Entero con signo de 8 bits
  • short: Entero con signo de 16 bits
  • int: Entero con signo de 32 bits
  • long: Entero con signo de 64 bits
  • float: Número de punto flotante de 32 bits
  • double: Número de punto flotante de 64 bits
  • char: Carácter Unicode de 16 bits
  • boolean: verdadero o falso

Ejemplos

byte b = 10
short s = 100
int i = 1000
long l = 10000L
float f = 10.5F
double d = 20.5
char c = 'A'
boolean bool = true

Tipos de Datos de Objeto

  • String: Una secuencia de caracteres
  • List: Una colección ordenada de elementos
  • Map: Una colección de pares clave-valor
  • Range: Una secuencia de números o caracteres

Ejemplos

String str = "¡Hola, Groovy!"
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ["John": 30, "Jane": 25]
Range range = 1..10

Inferencia de Tipos

Groovy puede inferir el tipo de una variable basado en el valor asignado a ella. Esto es particularmente útil para simplificar el código.

Ejemplo

def message = "¡Hola, Mundo!"  // Groovy infiere esto como un String
def count = 42                // Groovy infiere esto como un Integer
def price = 19.99             // Groovy infiere esto como un Double

Conversión de Tipos

Groovy permite tanto conversiones de tipo implícitas como explícitas.

Conversión de Tipo Implícita

Groovy puede convertir automáticamente entre tipos compatibles.

Ejemplo

int num = 10
double result = num + 5.5  // num se convierte implícitamente a double

Conversión de Tipo Explícita

Puedes convertir tipos explícitamente usando casting de tipos.

Ejemplo

double num = 10.5
int result = (int) num  // Convertir explícitamente double a int

Resumen

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en Groovy. Aprendimos cómo declarar variables usando la palabra clave def y especificando tipos explícitamente. También exploramos los varios tipos de datos primitivos y de objeto soportados por Groovy, junto con ejemplos. Adicionalmente, discutimos la inferencia de tipos y la conversión de tipos, tanto implícita como explícita.

Entender estos conceptos fundamentales es esencial ya que forman los bloques de construcción para temas de programación más avanzados. Asegúrate de practicar escribiendo tus propios scripts en Groovy para reforzar estos conceptos.
