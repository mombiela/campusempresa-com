Introducción a JSON

  • JSON (JavaScript Object Notation) es un formato de intercambio de datos ligero.
  • Es fácil de leer y escribir para los humanos y fácil de analizar y generar para las máquinas.
  • JSON se usa a menudo para transmitir datos en aplicaciones web.

Ejemplo de JSON

{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}

Trabajando con JSON en Groovy

Groovy proporciona soporte incorporado para JSON a través del paquete groovy.json. Este paquete incluye clases como JsonSlurper y JsonOutput para analizar y generar datos JSON.

Analizando JSON con JsonSlurper

  • JsonSlurper se usa para analizar cadenas JSON en objetos Groovy.

Ejemplo: Analizando JSON

import groovy.json.JsonSlurper

def jsonString = '''
{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.name  // Salida: John Doe
println jsonObject.age   // Salida: 30
println jsonObject.isStudent  // Salida: false
println jsonObject.courses  // Salida: [Math, Science, History]

Generando JSON con JsonOutput

  • JsonOutput se usa para convertir objetos Groovy en cadenas JSON.

Ejemplo: Generando JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
println jsonString

Impresión Bonita de JSON

  • JsonOutput.prettyPrint se puede usar para formatear cadenas JSON para una mejor legibilidad.

Ejemplo: Impresión Bonita de JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
def prettyJsonString = JsonOutput.prettyPrint(jsonString)
println prettyJsonString

Manejo de Estructuras JSON Complejas

  • Groovy puede manejar estructuras JSON anidadas y complejas con facilidad.

Ejemplo: Analizando JSON Complejo

import groovy.json.JsonSlurper

def jsonString = '''
{
  "person": {
    "name": "John Doe",
    "age": 30,
    "address": {
      "street": "123 Main St",
      "city": "Springfield",
      "zip": "12345"
    },
    "courses": ["Math", "Science", "History"]
  }
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.person.name  // Salida: John Doe
println jsonObject.person.address.city  // Salida: Springfield
println jsonObject.person.courses  // Salida: [Math, Science, History]

Resumen

  • JSON es un formato ampliamente utilizado para el intercambio de datos.
  • Groovy proporciona un soporte robusto para JSON a través del paquete groovy.json.
  • JsonSlurper se usa para analizar cadenas JSON en objetos Groovy.
  • JsonOutput se usa para generar cadenas JSON a partir de objetos Groovy.
  • Groovy puede manejar estructuras JSON tanto simples como complejas de manera eficiente.

Al dominar estas herramientas, puedes trabajar eficazmente con datos JSON en tus aplicaciones Groovy, haciendo que el intercambio de datos sea fluido y eficiente.
