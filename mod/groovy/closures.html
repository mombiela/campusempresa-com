Introducción a las Closures

Las closures son una de las características más poderosas en Groovy. Son bloques de código que pueden ser asignados a variables, pasados como parámetros y ejecutados en un momento posterior. Las closures pueden capturar variables de su contexto circundante, haciéndolas altamente flexibles y útiles para diversas tareas de programación.

Conceptos Clave

  • Definición: Una closure es una instancia de la clase groovy.lang.Closure.
  • Sintaxis: Las closures se definen usando llaves {}.
  • Parámetros: Las closures pueden tomar parámetros, que se definen dentro de las llaves.
  • Valor de Retorno: La última expresión en una closure es el valor de retorno.

Sintaxis Básica y Uso

Definiendo una Closure

Una closure simple puede definirse de la siguiente manera:

def myClosure = { println "¡Hola, Groovy!" }
myClosure() // Salida: ¡Hola, Groovy!

Closures con Parámetros

Las closures pueden aceptar parámetros, que se especifican antes del símbolo ->:

def greet = { name -> println "¡Hola, $name!" }
greet("Alice") // Salida: ¡Hola, Alice!

Parámetro Implícito `it`

Si una closure toma un solo parámetro, Groovy proporciona un parámetro implícito llamado it:

def printIt = { println it }
printIt("Ejemplo de parámetro implícito") // Salida: Ejemplo de parámetro implícito

Características Avanzadas

Capturando Variables

Las closures pueden capturar variables de su contexto circundante, también conocidas como "variables libres":

def greeting = "Hola"
def greeter = { name -> println "$greeting, $name!" }
greeter("Bob") // Salida: ¡Hola, Bob!

Retornando Valores

La última expresión en una closure es el valor de retorno:

def square = { it * it }
println square(4) // Salida: 16

Closures como Parámetros de Métodos

Las closures pueden ser pasadas como parámetros a métodos, permitiendo funciones de orden superior:

def processList(list, closure) {
    list.each { closure(it) }
}

def numbers = [1, 2, 3, 4, 5]
processList(numbers, { println it * 2 }) // Salida: 2, 4, 6, 8, 10

Closures con Múltiples Parámetros

Las closures pueden tomar múltiples parámetros, separados por comas:

def sum = { a, b -> a + b }
println sum(3, 7) // Salida: 10

Ejemplos Prácticos

Filtrando una Lista

Usando closures para filtrar una lista:

def numbers = [1, 2, 3, 4, 5, 6]
def evenNumbers = numbers.findAll { it % 2 == 0 }
println evenNumbers // Salida: [2, 4, 6]

Transformando una Lista

Usando closures para transformar una lista:

def numbers = [1, 2, 3, 4, 5]
def squares = numbers.collect { it * it }
println squares // Salida: [1, 4, 9, 16, 25]

Ordenando una Lista

Usando closures para ordenar una lista:

def names = ["Charlie", "Alice", "Bob"]
names.sort { a, b -> a <=> b }
println names // Salida: [Alice, Bob, Charlie]

Conclusión

Las closures en Groovy proporcionan una forma poderosa y flexible de manejar bloques de código. Pueden capturar variables de su contexto circundante, haciéndolas altamente útiles para tareas como filtrar, transformar y ordenar colecciones. Entender las closures es esencial para escribir código Groovy conciso y expresivo. Al dominar las closures, puedes aprovechar todo el potencial de Groovy para crear programas más legibles y mantenibles.
