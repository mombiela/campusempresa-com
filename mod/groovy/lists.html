Introducción a las Listas

Las listas son una estructura de datos fundamental en Groovy, utilizada para almacenar colecciones ordenadas de elementos. Son dinámicas, lo que significa que su tamaño puede cambiar, y pueden contener elementos de diferentes tipos.

Conceptos Clave

  • Tamaño Dinámico: Las listas pueden crecer y encogerse según sea necesario.
  • Colección Ordenada: Los elementos en una lista mantienen su orden.
  • Elementos Heterogéneos: Las listas pueden contener elementos de diferentes tipos.

Creando Listas

Crear listas en Groovy es sencillo. Puedes usar corchetes [] para definir una lista.

def emptyList = []
def numberList = [1, 2, 3, 4, 5]
def mixedList = [1, "Groovy", true, 3.14]

Explicación

  • emptyList es una lista vacía.
  • numberList contiene enteros.
  • mixedList contiene un entero, una cadena, un booleano y un número de punto flotante.

Accediendo a Elementos de la Lista

Puedes acceder a los elementos en una lista usando su índice, comenzando desde 0.

def numberList = [1, 2, 3, 4, 5]
println(numberList[0])  // Salida: 1
println(numberList[2])  // Salida: 3

Explicación

  • numberList[0] accede al primer elemento.
  • numberList[2] accede al tercer elemento.

Modificando Listas

Las listas en Groovy son mutables, lo que significa que puedes cambiar su contenido.

Añadiendo Elementos

Puedes añadir elementos usando el operador << o el método add().

def numberList = [1, 2, 3]
numberList << 4
numberList.add(5)
println(numberList)  // Salida: [1, 2, 3, 4, 5]

Eliminando Elementos

Puedes eliminar elementos usando el método remove() o especificando el índice.

def numberList = [1, 2, 3, 4, 5]
numberList.remove(2)  // Elimina el elemento en el índice 2
println(numberList)   // Salida: [1, 2, 4, 5]
numberList.remove((Integer) 4)  // Elimina el elemento con valor 4
println(numberList)   // Salida: [1, 2, 5]

Actualizando Elementos

Puedes actualizar elementos asignando un nuevo valor a un índice específico.

def numberList = [1, 2, 3, 4, 5]
numberList[1] = 10
println(numberList)  // Salida: [1, 10, 3, 4, 5]

Iterando Sobre Listas

Puedes iterar sobre listas usando bucles o el método each de Groovy.

Usando un Bucle `for`

def numberList = [1, 2, 3, 4, 5]
for (num in numberList) {
    println(num)
}

Usando el Método `each`

def numberList = [1, 2, 3, 4, 5]
numberList.each { num ->
    println(num)
}

Operaciones Avanzadas con Listas

Groovy proporciona varias operaciones avanzadas para listas, como filtrado, mapeo y reducción.

Filtrado

Puedes filtrar listas usando el método findAll.

def numberList = [1, 2, 3, 4, 5]
def evenNumbers = numberList.findAll { it % 2 == 0 }
println(evenNumbers)  // Salida: [2, 4]

Mapeo

Puedes transformar listas usando el método collect.

def numberList = [1, 2, 3, 4, 5]
def squaredNumbers = numberList.collect { it * it }
println(squaredNumbers)  // Salida: [1, 4, 9, 16, 25]

Reducción

Puedes reducir listas a un solo valor usando el método inject.

def numberList = [1, 2, 3, 4, 5]
def sum = numberList.inject(0) { acc, num -> acc + num }
println(sum)  // Salida: 15

Conclusión

Las listas son una estructura de datos versátil y poderosa en Groovy. Te permiten almacenar, acceder, modificar e iterar sobre colecciones de elementos de manera eficiente. Al comprender los conceptos básicos y las operaciones avanzadas de las listas, puedes aprovechar todo su potencial en tus programas Groovy.
