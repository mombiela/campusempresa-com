Introducción

En este curso, construiremos una aplicación de blog completamente funcional utilizando Django, un framework web de alto nivel para Python. Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado, cubriendo todos los conceptos y características esenciales de Django.

Configuración del Entorno

Antes de comenzar a construir nuestra aplicación de blog, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Python: Asegúrate de tener Python instalado en tu máquina. Puedes descargarlo desde python.org.
  • Instalar Django: Usa pip para instalar Django. 
    pip install django
  • Crear un Proyecto Django: Inicia un nuevo proyecto Django. 
    django-admin startproject myblog
cd myblog

Creando la Aplicación de Blog

A continuación, crearemos una nueva aplicación Django dentro de nuestro proyecto.

  • Crear la Aplicación de Blog: 
    python manage.py startapp blog
  • Agregar la Aplicación de Blog a las Aplicaciones Instaladas: Abre myblog/settings.py y agrega 'blog' a la lista INSTALLED_APPS.

Modelos

Definiremos la estructura de datos para nuestro blog utilizando modelos de Django.

  • Crear Modelos de Blog: Abre blog/models.py y define los modelos para nuestro blog.

    from django.db import models
from django.utils import timezone

class Post(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    created_at = models.DateTimeField(default=timezone.now)
    updated_at = models.DateTimeField(auto_now=True)

    def __str__(self):
        return self.title

  • Migrar los Modelos: Crea y aplica migraciones para actualizar el esquema de la base de datos.

    python manage.py makemigrations
python manage.py migrate

Interfaz de Administración

Django proporciona una interfaz de administración incorporada para gestionar nuestros modelos.

  • Registrar Modelos en la Administración: Abre blog/admin.py y registra el modelo Post. 
    from django.contrib import admin
from .models import Post

admin.site.register(Post)

Vistas y URLs

Crearemos vistas para manejar la lógica de mostrar publicaciones del blog y configuraremos URLs para enrutar solicitudes a estas vistas.

  • Crear Vistas: Abre blog/views.py y define vistas para listar y mostrar publicaciones del blog.

    from django.shortcuts import render
from .models import Post

def post_list(request):
    posts = Post.objects.all()
    return render(request, 'blog/post_list.html', {'posts': posts})

def post_detail(request, pk):
    post = Post.objects.get(pk=pk)
    return render(request, 'blog/post_detail.html', {'post': post})

  • Configurar URLs: Crea un archivo blog/urls.py y configura los patrones de URL.

    from django.urls import path
from . import views

urlpatterns = [
    path('', views.post_list, name='post_list'),
    path('post/<int:pk>/', views.post_detail, name='post_detail'),
]

    Actualiza el urls.py principal para incluir las URLs del blog.

    from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('blog.urls')),
]

Plantillas

Crearemos plantillas HTML para renderizar nuestras publicaciones del blog.

  • Crear Directorio de Plantillas: Crea un directorio templates dentro de la aplicación blog y agrega subdirectorios para cada vista.

    blog/
└── templates/
    └── blog/
        ├── post_list.html
        └── post_detail.html

  • Plantilla de Lista de Publicaciones: Crea post_list.html.

    <!DOCTYPE html>
<html>
<head>
    <title>Blog</title>
</head>
<body>
    <h1>Publicaciones del Blog</h1>
    <ul>
        {% for post in posts %}
            <li>
                <a href="{% url 'post_detail' post.pk %}">{{ post.title }}</a>
                <p>{{ post.content|truncatewords:30 }}</p>
            </li>
        {% endfor %}
    </ul>
</body>
</html>

  • Plantilla de Detalle de Publicación: Crea post_detail.html.

    <!DOCTYPE html>
<html>
<head>
    <title>{{ post.title }}</title>
</head>
<body>
    <h1>{{ post.title }}</h1>
    <p>{{ post.content }}</p>
    <p><a href="{% url 'post_list' %}">Volver al Blog</a></p>
</body>
</html>

Formularios e Input de Usuario

Agregaremos funcionalidad para crear y editar publicaciones del blog utilizando formularios de Django.

  • Crear Formularios: Abre blog/forms.py y define un formulario para el modelo Post.

    from django import forms
from .models import Post

class PostForm(forms.ModelForm):
    class Meta:
        model = Post
        fields = ['title', 'content']

  • Crear y Editar Vistas: Actualiza blog/views.py para incluir vistas para crear y editar publicaciones.

    from django.shortcuts import redirect
from .forms import PostForm

def post_new(request):
    if request.method == "POST":
        form = PostForm(request.POST)
        if form.is_valid():
            post = form.save(commit=False)
            post.save()
            return redirect('post_detail', pk=post.pk)
    else:
        form = PostForm()
    return render(request, 'blog/post_edit.html', {'form': form})

def post_edit(request, pk):
    post = Post.objects.get(pk=pk)
    if request.method == "POST":
        form = PostForm(request.POST, instance=post)
        if form.is_valid():
            post = form.save(commit=False)
            post.save()
            return redirect('post_detail', pk=post.pk)
    else:
        form = PostForm(instance=post)
    return render(request, 'blog/post_edit.html', {'form': form})

  • Actualizar URLs: Agrega patrones de URL para las nuevas vistas en blog/urls.py.

    urlpatterns = [
    path('', views.post_list, name='post_list'),
    path('post/<int:pk>/', views.post_detail, name='post_detail'),
    path('post/new/', views.post_new, name='post_new'),
    path('post/<int:pk>/edit/', views.post_edit, name='post_edit'),
]

  • Crear Plantilla de Formulario: Crea post_edit.html.

    <!DOCTYPE html>
<html>
<head>
    <title>Nueva Publicación</title>
</head>
<body>
    <h1>Nueva Publicación</h1>
    <form method="POST">
        {% csrf_token %}
        {{ form.as_p }}
        <button type="submit">Guardar</button>
    </form>
</body>
</html>

Características Avanzadas

Agregaremos algunas características avanzadas a nuestra aplicación de blog.

  • Autenticación de Usuarios: Agrega autenticación de usuarios para permitir que los usuarios inicien sesión y gestionen sus publicaciones.
  • Comentarios: Agrega un sistema de comentarios para permitir que los usuarios comenten en las publicaciones.
  • Etiquetas y Categorías: Organiza las publicaciones utilizando etiquetas y categorías.
  • Funcionalidad de Búsqueda: Implementa funcionalidad de búsqueda para permitir que los usuarios busquen publicaciones.

Conclusión

En este curso, hemos construido una aplicación de blog completa utilizando Django. Comenzamos configurando el entorno, creando modelos, vistas y plantillas, y luego agregamos formularios e input de usuario. Finalmente, exploramos algunas características avanzadas para mejorar nuestra aplicación de blog. Al final de este curso, deberías tener una comprensión sólida de Django y ser capaz de construir tus propias aplicaciones web.
