Arrays Asociativos en REXX
Estructuras de Datos ►

Los arrays asociativos, también conocidos como variables stem en REXX, son una característica poderosa que te permite crear y manipular colecciones de datos indexados por cadenas. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de arrays asociativos en REXX.

Introducción a los Arrays Asociativos

Los arrays asociativos en REXX se implementan usando variables stem. Una variable stem es una colección de variables que comparten un prefijo común, conocido como el nombre del stem. Cada elemento en la colección se accede usando un índice único.

Conceptos Clave

  • Variable Stem: Una variable que actúa como prefijo para una colección de variables relacionadas.
  • Índice: Una cadena o número utilizado para acceder a un elemento específico en la variable stem.
  • Elemento: Un ítem individual en el array asociativo, accedido usando el nombre del stem y un índice.

Sintaxis Básica

Para definir y usar una variable stem en REXX, sigues esta sintaxis:

stem.index = value

Aquí, stem es el nombre del stem, index es el identificador único para el elemento, y value es el dato que quieres almacenar.

Ejemplo

/* Definir una variable stem */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Acceder a los elementos */
say students.1  /* Salida: Alice */
say students.2  /* Salida: Bob */
say students.3  /* Salida: Charlie */

Uso Avanzado de Arrays Asociativos

A medida que te sientas más cómodo con las variables stem, puedes comenzar a usar características y técnicas más avanzadas.

Indexación Dinámica

Puedes usar valores dinámicos para el índice, permitiendo estructuras de datos más flexibles y poderosas.

/* Usando indexación dinámica */
do i = 1 to 3
    students.i = "Student" || i
end

/* Acceder a los elementos dinámicamente */
do i = 1 to 3
    say students.i  /* Salida: Student1, Student2, Student3 */
end

Arrays Asociativos Multidimensionales

REXX te permite crear arrays asociativos multidimensionales usando múltiples índices.

/* Definir un array asociativo multidimensional */
grades.Alice.Math = 95
grades.Alice.English = 88
grades.Bob.Math = 78
grades.Bob.English = 85

/* Acceder a los elementos */
say grades.Alice.Math     /* Salida: 95 */
say grades.Bob.English    /* Salida: 85 */

Iterar Sobre Arrays Asociativos

Puedes iterar sobre todos los elementos en una variable stem usando el bucle do y el prefijo stem..

/* Definir una variable stem */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Iterar sobre los elementos */
i = 1
do while students.i \= ""
    say students.i
    i = i + 1
end

Comprobar Existencia

Puedes comprobar si un elemento existe en una variable stem usando la función symbol.

/* Definir una variable stem */
students.1 = "Alice"

/* Comprobar existencia */
if symbol("students.2") = "VAR" then
    say "Student 2 exists"
else
    say "Student 2 does not exist"

Conclusión

Los arrays asociativos, o variables stem, en REXX proporcionan una forma flexible y poderosa de gestionar colecciones de datos. Desde el uso básico hasta técnicas avanzadas como la indexación dinámica y los arrays multidimensionales, las variables stem pueden mejorar enormemente tus capacidades de programación en REXX. Practica usando estas características para volverte competente en el manejo de estructuras de datos complejas en REXX.
