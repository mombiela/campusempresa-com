Introducción a REXX

REXX (Restructured Extended Executor) es un lenguaje de programación de alto nivel desarrollado por IBM. Está diseñado para ser fácil de aprender y leer, lo que lo convierte en una excelente opción tanto para principiantes como para programadores experimentados. REXX se utiliza ampliamente para scripting, procesamiento de datos y automatización de tareas en mainframes de IBM, así como en otras plataformas.

Características clave de REXX

  • Facilidad de uso: La sintaxis de REXX es sencilla y fácil de leer, lo que reduce la curva de aprendizaje para los nuevos programadores.
  • Portabilidad: REXX está disponible en múltiples plataformas, incluidos mainframes de IBM, Windows, Unix y Linux.
  • Interoperabilidad: REXX puede interactuar con otros lenguajes de programación y sistemas, lo que lo hace versátil para diversas aplicaciones.
  • Funciones integradas: REXX incluye un conjunto rico de funciones integradas para manipulación de cadenas, operaciones aritméticas y más.
  • Extensibilidad: Los usuarios pueden extender REXX escribiendo sus propias funciones y procedimientos.

Sintaxis básica de REXX

Veamos un programa simple en REXX para entender su sintaxis básica:

/* Este es un comentario en REXX */
say "¡Hola, Mundo!"
  • Comentarios: Los comentarios en REXX se encierran dentro de /* y */.
  • Salida: La instrucción say se usa para imprimir texto en la pantalla.

Variables y tipos de datos

En REXX, las variables son de tipo dinámico, lo que significa que no necesitas declarar su tipo explícitamente. Aquí hay un ejemplo:

name = "John Doe"
age = 30
say "Nombre:" name
say "Edad:" age
  • Variables de cadena: Variables como name pueden contener valores de cadena.
  • Variables numéricas: Variables como age pueden contener valores numéricos.

Estructuras de control

REXX admite estructuras de control estándar como bucles y declaraciones condicionales. Aquí hay algunos ejemplos:

Declaraciones condicionales

age = 20
if age >= 18 then
    say "Eres un adulto."
else
    say "Eres un menor."

Bucles

do i = 1 to 5
    say "Iteración:" i
end

Funciones integradas

REXX proporciona una variedad de funciones integradas para realizar tareas comunes. Aquí hay algunos ejemplos:

  • Manipulación de cadenas: length, substr, pos
  • Operaciones aritméticas: abs, max, min
  • Fecha y hora: date, time

Ejemplo de uso de funciones integradas:

text = "¡Hola, Mundo!"
say "Longitud del texto:" length(text)
say "Subcadena desde la posición 8:" substr(text, 8)

Conclusión

REXX es un lenguaje de programación poderoso pero fácil de aprender, ideal para scripting y automatización de tareas. Su simplicidad, portabilidad y rico conjunto de funciones integradas lo convierten en una herramienta versátil para programadores de todos los niveles. En las próximas secciones de este curso, profundizaremos en las características avanzadas de REXX y exploraremos aplicaciones prácticas para mejorar tus habilidades de programación.
