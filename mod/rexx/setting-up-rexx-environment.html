Configuración del Entorno REXX
Hola Mundo en REXX ►

Configurar el entorno REXX es el primer paso para dominar este poderoso lenguaje de scripting. Esta sección te guiará a través del proceso de instalación y configuración de REXX en tu sistema, asegurándote de tener todo lo necesario para comenzar a escribir y ejecutar scripts REXX.

Instalando REXX

Antes de que puedas comenzar a escribir scripts REXX, necesitas instalar un intérprete REXX. Hay varias opciones disponibles, dependiendo de tu sistema operativo.

Windows

Para los usuarios de Windows, Regina REXX es una opción popular. Sigue estos pasos para instalarlo:

  1. Descargar Regina REXX:

    • Visita el sitio web de Regina REXX.
    • Descarga el instalador apropiado para tu versión de Windows.

  2. Instalar Regina REXX:

    • Ejecuta el instalador descargado.
    • Sigue las instrucciones en pantalla para completar la instalación.

  3. Verificar la Instalación:

    • Abre el Símbolo del sistema.
    • Escribe rexx y presiona Enter. Si está instalado correctamente, deberías ver el prompt de REXX.

macOS

Para los usuarios de macOS, puedes usar el gestor de paquetes Homebrew para instalar Regina REXX:

  1. Instalar Homebrew (si no está ya instalado):

    • Abre Terminal.
    • Ejecuta el comando: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instalar Regina REXX:

    • Ejecuta el comando: 
      brew install regina-rexx

  3. Verificar la Instalación:

    • Escribe rexx en Terminal y presiona Enter. Deberías ver el prompt de REXX.

Linux

Para los usuarios de Linux, Regina REXX se puede instalar usando el gestor de paquetes de tu distribución:

  1. Debian/Ubuntu:

    • Abre Terminal.
    • Ejecuta el comando: 
      sudo apt-get install regina-rexx

  2. Fedora:

    • Abre Terminal.
    • Ejecuta el comando: 
      sudo dnf install regina-rexx

  3. Verificar la Instalación:

    • Escribe rexx en Terminal y presiona Enter. Deberías ver el prompt de REXX.

Configurando el Entorno

Después de instalar REXX, puede que necesites configurar tu entorno para facilitar la ejecución de scripts REXX.

Configuración de Variables de Entorno

Configurar variables de entorno puede ayudarte a ejecutar scripts REXX desde cualquier directorio sin especificar la ruta completa al intérprete.

Windows

  1. Abrir Variables de Entorno:

    • Haz clic derecho en 'Este PC' o 'Equipo' en el escritorio o en el Explorador de archivos.
    • Selecciona 'Propiedades'.
    • Haz clic en 'Configuración avanzada del sistema'.
    • Haz clic en 'Variables de entorno'.

  2. Agregar REXX a PATH:

    • En la sección 'Variables del sistema', encuentra y selecciona la variable 'Path', luego haz clic en 'Editar'.
    • Haz clic en 'Nuevo' y agrega la ruta al directorio donde está instalado Regina REXX (por ejemplo, C:\Program Files\Regina).

  3. Aplicar Cambios:

    • Haz clic en 'Aceptar' para cerrar todos los cuadros de diálogo.

macOS y Linux

  1. Abrir Terminal.

  2. Editar Archivo de Configuración del Shell:

    • Para Bash, abre .bashrc o .bash_profile: 
      nano ~/.bashrc
    • Para Zsh, abre .zshrc: 
      nano ~/.zshrc

  3. Agregar REXX a PATH:

    • Agrega la siguiente línea al archivo: 
      export PATH=$PATH:/usr/local/bin

  4. Aplicar Cambios:

    • Guarda el archivo y ejecuta: 
      source ~/.bashrc
    • O para Zsh: 
      source ~/.zshrc

Escribiendo Tu Primer Script REXX

Ahora que tu entorno está configurado, escribamos un simple script REXX para asegurarnos de que todo funciona correctamente.

  1. Crear un Nuevo Archivo:

    • Abre tu editor de texto preferido.
    • Crea un nuevo archivo llamado hello.rexx.

  2. Escribir el Script:

    • Agrega el siguiente código al archivo: 
      /* Hola Mundo en REXX */
say "¡Hola, Mundo!"

  3. Guardar el Archivo.

  4. Ejecutar el Script:

    • Abre el Símbolo del sistema (Windows) o Terminal (macOS/Linux).
    • Navega al directorio donde guardaste hello.rexx.
    • Ejecuta el script escribiendo: 
      rexx hello.rexx
    • Deberías ver la salida: 
      ¡Hola, Mundo!

Conclusión

Configurar el entorno REXX implica instalar el intérprete y configurar tu sistema para ejecutar fácilmente scripts REXX. Siguiendo los pasos descritos en esta sección, ahora deberías tener un entorno REXX completamente funcional. También escribiste y ejecutaste tu primer script REXX, confirmando que todo está configurado correctamente. En las siguientes secciones, profundizaremos en la sintaxis y las características del lenguaje REXX.
