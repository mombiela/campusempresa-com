Introducción

Las estructuras de control son fundamentales en cualquier lenguaje de programación, ya que te permiten controlar el flujo de ejecución basado en ciertas condiciones. En REXX, las principales estructuras de control incluyen IF, DO y SELECT. Esta sección te guiará a través de estas estructuras desde un nivel principiante hasta un nivel avanzado.

Sentencias IF

La sentencia IF en REXX se utiliza para ejecutar un bloque de código condicionalmente.

Sentencia IF Básica

La forma más simple de una sentencia IF verifica una única condición.

/* Sentencia IF básica */
age = 20
if age >= 18 then
  say "Eres un adulto."

Sentencia IF-ELSE

Puedes extender la sentencia IF con una cláusula ELSE para manejar condiciones alternativas.

/* Sentencia IF-ELSE */
age = 16
if age >= 18 then
  say "Eres un adulto."
else
  say "Eres un menor."

Sentencias IF Anidadas

Las sentencias IF pueden anidarse para verificar múltiples condiciones.

/* Sentencias IF anidadas */
age = 16
if age >= 18 then
  say "Eres un adulto."
else
  if age >= 13 then
    say "Eres un adolescente."
  else
    say "Eres un niño."

Bucles DO

El bucle DO en REXX se utiliza para repetir un bloque de código múltiples veces.

Bucle DO Básico

Un simple bucle DO se ejecuta un número fijo de veces.

/* Bucle DO básico */
do i = 1 to 5
  say "Iteración" i
end

Bucle DO WHILE

Un bucle DO WHILE se ejecuta mientras una condición sea verdadera.

/* Bucle DO WHILE */
i = 1
do while i <= 5
  say "Iteración" i
  i = i + 1
end

Bucle DO UNTIL

Un bucle DO UNTIL se ejecuta hasta que una condición se vuelva verdadera.

/* Bucle DO UNTIL */
i = 1
do until i > 5
  say "Iteración" i
  i = i + 1
end

Bucle DO FOREVER

Un bucle DO FOREVER se ejecuta indefinidamente hasta que se termine explícitamente.

/* Bucle DO FOREVER */
i = 1
do forever
  say "Iteración" i
  i = i + 1
  if i > 5 then leave
end

SELECT Case

La sentencia SELECT en REXX se utiliza para manejar múltiples condiciones de manera más eficiente que las sentencias IF anidadas.

Sentencia SELECT Básica

Una sentencia SELECT evalúa múltiples condiciones y ejecuta el bloque de código correspondiente.

/* Sentencia SELECT básica */
day = "Monday"
select
  when day = "Monday" then
    say "Inicio de la semana laboral."
  when day = "Friday" then
    say "Fin de la semana laboral."
  otherwise
    say "Día de mitad de semana."
end

Sentencias SELECT Anidadas

Las sentencias SELECT también pueden anidarse para una toma de decisiones más compleja.

/* Sentencias SELECT anidadas */
day = "Monday"
time = "Morning"
select
  when day = "Monday" then
    select
      when time = "Morning" then
        say "Inicio de la semana laboral en la mañana."
      otherwise
        say "Inicio de la semana laboral en la tarde."
    end
  when day = "Friday" then
    say "Fin de la semana laboral."
  otherwise
    say "Día de mitad de semana."
end

Conclusión

Las estructuras de control como IF, DO y SELECT son esenciales para gestionar el flujo de tus programas en REXX. Comprender estos constructos te permite escribir código más eficiente y legible. Practica estos ejemplos e intenta crear tus propios escenarios para profundizar tu comprensión.
