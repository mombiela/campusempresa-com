Operadores y Expresiones en REXX
Entrada y Salida ►

Introducción

En esta sección, exploraremos los diversos operadores y expresiones en REXX. Los operadores son símbolos especiales que realizan operaciones sobre variables y valores. Las expresiones son combinaciones de variables, valores y operadores que REXX evalúa para producir otro valor. Comprender los operadores y las expresiones es fundamental para escribir programas efectivos en REXX.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

  • Adición (+): Suma dos números.
  • Sustracción (-): Resta el segundo número del primero.
  • Multiplicación (*): Multiplica dos números.
  • División (/): Divide el primer número por el segundo.
  • División Entera (%): Divide el primer número por el segundo y devuelve la parte entera del cociente.
  • Módulo (//): Divide el primer número por el segundo y devuelve el resto.

Ejemplos

/* Adición */
result = 5 + 3
say result  /* Salida: 8 */

/* Sustracción */
result = 10 - 4
say result  /* Salida: 6 */

/* Multiplicación */
result = 7 * 2
say result  /* Salida: 14 */

/* División */
result = 20 / 4
say result  /* Salida: 5 */

/* División Entera */
result = 20 % 3
say result  /* Salida: 6 */

/* Módulo */
result = 20 // 3
say result  /* Salida: 2 */

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores. Devuelven un valor booleano (1 para verdadero, 0 para falso).

  • Igual (= o ==): Verifica si dos valores son iguales.
  • No Igual (\= o <>): Verifica si dos valores no son iguales.
  • Menor Que (<): Verifica si el primer valor es menor que el segundo.
  • Mayor Que (>): Verifica si el primer valor es mayor que el segundo.
  • Menor o Igual Que (<=): Verifica si el primer valor es menor o igual que el segundo.
  • Mayor o Igual Que (>=): Verifica si el primer valor es mayor o igual que el segundo.

Ejemplos

/* Igual */
result = 5 = 5
say result  /* Salida: 1 */

/* No Igual */
result = 5 \= 3
say result  /* Salida: 1 */

/* Menor Que */
result = 3 < 5
say result  /* Salida: 1 */

/* Mayor Que */
result = 7 > 4
say result  /* Salida: 1 */

/* Menor o Igual Que */
result = 5 <= 5
say result  /* Salida: 1 */

/* Mayor o Igual Que */
result = 6 >= 3
say result  /* Salida: 1 */

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

  • AND (&): Devuelve verdadero si ambas condiciones son verdaderas.
  • OR (|): Devuelve verdadero si al menos una condición es verdadera.
  • NOT (\): Devuelve lo opuesto de la condición.

Ejemplos

/* AND */
result = (5 = 5) & (3 < 4)
say result  /* Salida: 1 */

/* OR */
result = (5 = 5) | (3 > 4)
say result  /* Salida: 1 */

/* NOT */
result = \ (5 = 3)
say result  /* Salida: 1 */

Operadores de Cadenas

Los operadores de cadenas se utilizan para manipular cadenas.

  • Concatenación (||): Une dos cadenas juntas.
  • Subcadena (substr): Extrae una subcadena de una cadena.

Ejemplos

/* Concatenación */
result = "Hello" || " " || "World"
say result  /* Salida: Hello World */

/* Subcadena */
result = substr("Hello World", 1, 5)
say result  /* Salida: Hello */

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación Simple (=): Asigna el valor de la derecha a la variable de la izquierda.
  • Asignación Compuesta (+=, -=, *=, /=, %=, //=): Combina una operación aritmética con la asignación.

Ejemplos

/* Asignación Simple */
a = 10
say a  /* Salida: 10 */

/* Asignación Compuesta */
a += 5
say a  /* Salida: 15 */

a -= 3
say a  /* Salida: 12 */

a *= 2
say a  /* Salida: 24 */

a /= 4
say a  /* Salida: 6 */

a %= 2
say a  /* Salida: 3 */

a //= 2
say a  /* Salida: 1 */

Resumen

En esta sección, cubrimos los diversos operadores y expresiones en REXX, incluyendo operadores aritméticos, de comparación, lógicos, de cadenas y de asignación. Comprender estos operadores y cómo usarlos en expresiones es crucial para escribir programas efectivos en REXX. Practica usando estos operadores en diferentes escenarios para familiarizarte más con su uso.
