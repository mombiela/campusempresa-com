Introducción

REXX (Restructured Extended Executor) es un lenguaje de programación que es particularmente fuerte en la manipulación de cadenas. Esta sección cubrirá varias funciones de cadenas en REXX, desde básicas hasta avanzadas, proporcionando explicaciones claras, ejemplos y ejercicios.

Funciones Básicas de Cadenas

LENGTH

  • Propósito: Devuelve la longitud de una cadena.
  • Sintaxis: LENGTH(string)
/* Ejemplo */
string = "Hello, World!"
say LENGTH(string)  /* Salida: 13 */

SUBSTR

  • Propósito: Extrae una subcadena de una cadena.
  • Sintaxis: SUBSTR(string, start, length)
/* Ejemplo */
string = "Hello, World!"
say SUBSTR(string, 8, 5)  /* Salida: World */

POS

  • Propósito: Encuentra la posición de una subcadena dentro de una cadena.
  • Sintaxis: POS(needle, haystack, start)
/* Ejemplo */
string = "Hello, World!"
say POS("World", string)  /* Salida: 8 */

LEFT y RIGHT

  • Propósito: Extrae un número especificado de caracteres desde el lado izquierdo o derecho de una cadena.
  • Sintaxis: LEFT(string, length) y RIGHT(string, length)
/* Ejemplo */
string = "Hello, World!"
say LEFT(string, 5)   /* Salida: Hello */
say RIGHT(string, 6)  /* Salida: World! */

Funciones Intermedias de Cadenas

STRIP

  • Propósito: Elimina espacios en blanco al inicio y/o al final de una cadena.
  • Sintaxis: STRIP(string, option, char)
/* Ejemplo */
string = "   Hello, World!   "
say STRIP(string)  /* Salida: "Hello, World!" */

OVERLAY

  • Propósito: Superpone una subcadena sobre una cadena.
  • Sintaxis: OVERLAY(new, old, start, length)
/* Ejemplo */
string = "Hello, World!"
say OVERLAY("REXX", string, 8, 4)  /* Salida: Hello, REXX! */

TRANSLATE

  • Propósito: Traduce caracteres en una cadena.
  • Sintaxis: TRANSLATE(string, to, from)
/* Ejemplo */
string = "Hello, World!"
say TRANSLATE(string, "123", "Hlo")  /* Salida: 1e22, W2r2d! */

VERIFY

  • Propósito: Verifica caracteres en una cadena contra un conjunto de caracteres permitidos.
  • Sintaxis: VERIFY(string, reference, option)
/* Ejemplo */
string = "Hello, World!"
say VERIFY(string, "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz, !")  /* Salida: 0 */

Funciones Avanzadas de Cadenas

WORD

  • Propósito: Extrae una palabra específica de una cadena.
  • Sintaxis: WORD(string, n)
/* Ejemplo */
string = "Hello, World! Welcome to REXX."
say WORD(string, 3)  /* Salida: Welcome */

WORDPOS

  • Propósito: Encuentra la posición de una palabra dentro de una cadena.
  • Sintaxis: WORDPOS(word, string)
/* Ejemplo */
string = "Hello, World! Welcome to REXX."
say WORDPOS("Welcome", string)  /* Salida: 3 */

WORDS

  • Propósito: Cuenta el número de palabras en una cadena.
  • Sintaxis: WORDS(string)
/* Ejemplo */
string = "Hello, World! Welcome to REXX."
say WORDS(string)  /* Salida: 5 */

DELSTR

  • Propósito: Elimina una subcadena de una cadena.
  • Sintaxis: DELSTR(string, start, length)
/* Ejemplo */
string = "Hello, World!"
say DELSTR(string, 8, 6)  /* Salida: Hello, */

INSERT

  • Propósito: Inserta una subcadena en una cadena.
  • Sintaxis: INSERT(new, old, start, length, pad)
/* Ejemplo */
string = "Hello, World!"
say INSERT("REXX", string, 8)  /* Salida: Hello, REXXXWorld! */

Resumen

En esta sección, cubrimos una gama de funciones de cadenas en REXX, desde básicas hasta avanzadas. Estas funciones son esenciales para la manipulación efectiva de cadenas, que es una tarea común en muchos escenarios de programación. Entender y dominar estas funciones mejorará significativamente tu capacidad para trabajar con datos de texto en REXX.

Ejercicios

  1. Escribe un script en REXX para extraer el nombre de dominio de una dirección de correo electrónico.
  2. Crea una función que capitalice la primera letra de cada palabra en una cadena.
  3. Escribe un script para contar el número de vocales en una cadena dada.
  4. Desarrolla una función que invierta las palabras en una oración manteniendo las palabras en su orden original.

Al practicar estos ejercicios, reforzarás tu comprensión de las funciones de cadenas en REXX y mejorarás tus habilidades para resolver problemas.
