Introducción

En esta sección, exploraremos cómo implementar varias funcionalidades en Dart. Esto incluirá entender los conceptos básicos de la sintaxis de Dart, crear funciones, manejar excepciones y trabajar con programación asíncrona. Al final de esta sección, serás capaz de implementar funcionalidades de manera efectiva en tus aplicaciones Dart.

Sintaxis y Estructura Básica

Antes de sumergirnos en la implementación de funcionalidades, es esencial entender la sintaxis y estructura básica de Dart.

Ejemplo de Hola Mundo

void main() {
  print('¡Hola, Mundo!');
}
  • void main() {}: El punto de entrada de una aplicación Dart.
  • print('¡Hola, Mundo!');: Imprime texto en la consola.

Variables y Tipos de Datos

void main() {
  int edad = 25;
  double altura = 5.9;
  String nombre = 'John Doe';
  bool esEstudiante = true;

  print('Nombre: $nombre, Edad: $edad, Altura: $altura, Es Estudiante: $esEstudiante');
}
  • int, double, String, bool: Tipos de datos básicos en Dart.
  • Las variables se declaran usando var, final o tipos específicos.

Funciones

Las funciones son esenciales para implementar funcionalidades ya que permiten la reutilización de código y la modularidad.

Definir y Llamar Funciones

void saludar(String nombre) {
  print('¡Hola, $nombre!');
}

void main() {
  saludar('Alice');
}
  • void saludar(String nombre) {}: Define una función llamada saludar que toma un parámetro String.
  • saludar('Alice');: Llama a la función saludar con el argumento 'Alice'.

Devolver Valores desde Funciones

int sumar(int a, int b) {
  return a + b;
}

void main() {
  int resultado = sumar(3, 4);
  print('Suma: $resultado');
}
  • int sumar(int a, int b) {}: Define una función que devuelve un int.
  • return a + b;: Devuelve la suma de a y b.

Manejo de Excepciones

Manejar excepciones es crucial para construir aplicaciones robustas.

Bloques Try-Catch

void main() {
  try {
    int resultado = 12 ~/ 0;
    print(resultado);
  } catch (e) {
    print('Error: $e');
  }
}
  • try {}: Código que podría lanzar una excepción.
  • catch (e) {}: Maneja la excepción.

Lanzar Excepciones

void verificarEdad(int edad) {
  if (edad < 18) {
    throw Exception('La edad debe ser al menos 18.');
  }
}

void main() {
  try {
    verificarEdad(16);
  } catch (e) {
    print('Error: $e');
  }
}
  • throw Exception('mensaje');: Lanza una excepción con un mensaje personalizado.

Programación Asíncrona

Dart soporta la programación asíncrona usando Future y async/await.

Usando Future

Future<String> obtenerOrdenUsuario() {
  return Future.delayed(Duration(seconds: 2), () => 'Latte Grande');
}

void main() {
  obtenerOrdenUsuario().then((orden) {
    print('Orden: $orden');
  });
}
  • Future.delayed(Duration(seconds: 2), () => 'Latte Grande');: Simula una operación retardada.
  • .then((orden) {}: Maneja el resultado del Future.

Async y Await

Future<String> obtenerOrdenUsuario() async {
  return await Future.delayed(Duration(seconds: 2), () => 'Latte Grande');
}

void main() async {
  String orden = await obtenerOrdenUsuario();
  print('Orden: $orden');
}
  • async: Marca una función como asíncrona.
  • await: Espera a que el Future se complete.

Conclusión

En esta sección, cubrimos los conceptos básicos de la implementación de funcionalidades en Dart, incluyendo entender la sintaxis, crear y usar funciones, manejar excepciones y trabajar con programación asíncrona. Al dominar estos conceptos, estarás bien equipado para construir aplicaciones Dart robustas y eficientes. Continúa practicando estos conceptos para solidificar tu comprensión y mejorar tus habilidades de programación.
