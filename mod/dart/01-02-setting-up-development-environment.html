Configuración del Entorno de Desarrollo
Siguiente ►

En esta sección, aprenderás a configurar tu entorno de desarrollo para programar en Dart. Este proceso incluye la instalación del SDK de Dart, la configuración de un editor de texto o un IDE, y la verificación de que todo esté funcionando correctamente.

  1. Instalación del SDK de Dart

El SDK de Dart es el kit de desarrollo de software que contiene todas las herramientas necesarias para desarrollar aplicaciones en Dart.

Pasos para instalar el SDK de Dart:

  1. Visita el sitio web oficial de Dart:

    • URL: https://dart.dev/get-dart

  2. Selecciona tu sistema operativo:

    • Windows
    • macOS
    • Linux

  3. Sigue las instrucciones específicas para tu sistema operativo:

    Windows:

    • Descarga el instalador de Dart para Windows.
    • Ejecuta el instalador y sigue las instrucciones en pantalla.
    • Asegúrate de que la opción "Add Dart to PATH" esté seleccionada durante la instalación.

    macOS:

    • Abre la terminal.
    • Ejecuta el siguiente comando para instalar Dart usando Homebrew: 
      brew tap dart-lang/dart
brew install dart

    Linux:

    • Abre la terminal.
    • Ejecuta los siguientes comandos para instalar Dart: 
      sudo apt update
sudo apt install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt update
sudo apt install dart

  4. Verifica la instalación:

    • Abre una terminal o símbolo del sistema.
    • Ejecuta el siguiente comando: 
      dart --version
    • Deberías ver la versión de Dart instalada en tu sistema.

  1. Configuración del Editor de Texto o IDE

Para escribir y ejecutar código Dart, puedes usar cualquier editor de texto o IDE. Sin embargo, algunos editores ofrecen mejor soporte y características específicas para Dart.

Recomendaciones de editores:

  1. Visual Studio Code (VS Code):

    • Descarga e instala Visual Studio Code desde https://code.visualstudio.com/.
    • Abre VS Code y ve a la sección de extensiones (ícono de cuadrado en la barra lateral izquierda).
    • Busca e instala la extensión "Dart" de Dart Code.

  2. IntelliJ IDEA:

    • Descarga e instala IntelliJ IDEA desde https://www.jetbrains.com/idea/.
    • Abre IntelliJ IDEA y ve a "File" > "Settings" > "Plugins".
    • Busca e instala el plugin "Dart".

  3. Android Studio:

    • Descarga e instala Android Studio desde https://developer.android.com/studio.
    • Abre Android Studio y ve a "File" > "Settings" > "Plugins".
    • Busca e instala el plugin "Dart".

  1. Verificación del Entorno de Desarrollo

Una vez que hayas instalado el SDK de Dart y configurado tu editor de texto o IDE, es importante verificar que todo esté funcionando correctamente.

Crear y ejecutar un programa de prueba:

  1. Abre tu editor de texto o IDE.
  2. Crea un nuevo archivo llamado main.dart.
  3. Escribe el siguiente código en main.dart: 
    void main() {
  print('¡Hola, mundo!');
}
  4. Guarda el archivo.
  5. Abre una terminal o el terminal integrado de tu editor.
  6. Navega al directorio donde guardaste main.dart.
  7. Ejecuta el programa con el siguiente comando: 
    dart main.dart
  8. Deberías ver la salida: 
    ¡Hola, mundo!

Conclusión

En esta sección, has aprendido a instalar el SDK de Dart, configurar un editor de texto o IDE, y verificar que tu entorno de desarrollo esté funcionando correctamente. Ahora estás listo para comenzar a escribir y ejecutar programas en Dart. En la siguiente sección, escribirás tu primer programa en Dart y explorarás la sintaxis básica del lenguaje.
