Tu Primer Programa en Dart
Siguiente ►

En esta lección, aprenderás a escribir y ejecutar tu primer programa en Dart. Este es un paso fundamental para familiarizarte con la sintaxis básica y el flujo de trabajo de desarrollo en Dart.

Objetivos de Aprendizaje

  • Escribir un programa simple en Dart.
  • Ejecutar el programa en tu entorno de desarrollo.
  • Comprender la estructura básica de un programa en Dart.

Requisitos Previos

Antes de comenzar, asegúrate de haber completado las siguientes lecciones:

  • Introducción a Dart
  • Configuración del Entorno de Desarrollo

Escribiendo tu Primer Programa

Paso 1: Crear un Archivo Dart

Primero, crea un archivo con la extensión .dart. Puedes nombrarlo hello_world.dart.

Paso 2: Escribir el Código

Abre el archivo hello_world.dart en tu editor de texto o IDE y escribe el siguiente código:

void main() {
  print('Hello, World!');
}

Explicación del Código

  • void main() { ... }: Esta es la función principal de un programa en Dart. Es el punto de entrada donde la ejecución del programa comienza.
  • print('Hello, World!');: Esta línea imprime el texto Hello, World! en la consola.

Paso 3: Ejecutar el Programa

Para ejecutar tu programa, abre una terminal y navega hasta el directorio donde guardaste hello_world.dart. Luego, ejecuta el siguiente comando:

dart hello_world.dart

Deberías ver la salida:

Hello, World!

Ejercicio Práctico

Ejercicio 1: Modificar el Mensaje

Modifica el programa para que imprima un mensaje diferente, por ejemplo, Hello, Dart!.

Solución

void main() {
  print('Hello, Dart!');
}

Ejercicio 2: Añadir Más Mensajes

Añade más líneas de print para que el programa imprima tres mensajes diferentes.

Solución

void main() {
  print('Hello, Dart!');
  print('Welcome to the Dart programming language.');
  print('Let\'s start coding!');
}

Errores Comunes y Consejos

Error: Archivo no encontrado

Si recibes un error indicando que el archivo no se encuentra, asegúrate de que estás en el directorio correcto y que el nombre del archivo es correcto.

Error: Sintaxis Incorrecta

Si hay un error de sintaxis, revisa cuidadosamente el código para asegurarte de que todas las llaves {}, paréntesis () y comillas '' están correctamente emparejados.

Consejo: Uso de Comentarios

Usa comentarios para hacer tu código más legible. Los comentarios en Dart se escriben con // para comentarios de una sola línea y /* ... */ para comentarios de múltiples líneas.

void main() {
  // Este es un comentario de una sola línea
  print('Hello, Dart!'); // Imprime un mensaje en la consola

  /*
   * Este es un comentario
   * de múltiples líneas
   */
  print('Welcome to the Dart programming language.');
}

Conclusión

En esta lección, has aprendido a escribir y ejecutar tu primer programa en Dart. Ahora estás familiarizado con la estructura básica de un programa y cómo imprimir mensajes en la consola. En la próxima lección, profundizaremos en la sintaxis y estructura básica de Dart.

¡Felicidades por dar tu primer paso en la programación con Dart!
