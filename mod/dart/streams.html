Streams en Dart
Generadores ►

Los streams son una parte fundamental de la programación asíncrona en Dart. Te permiten trabajar con secuencias de eventos asíncronos, como datos recibidos de un servidor web o la entrada del usuario.

¿Qué son los Streams?

  • Definición: Un stream es una secuencia de eventos asíncronos. Cada evento es un evento de datos o un evento de error.
  • Casos de Uso: Los streams son útiles para manejar múltiples eventos asíncronos a lo largo del tiempo, tales como:
    • Recibir datos de un servidor web.
    • Manejar la entrada del usuario.
    • Escuchar actualizaciones en tiempo real.

Tipos de Streams

Dart proporciona dos tipos principales de streams:

  • Streams de Suscripción Única: Estos streams solo pueden ser escuchados una vez. Son adecuados para eventos que ocurren solo una vez o que necesitan ser procesados en un orden específico.
  • Streams de Difusión: Estos streams pueden ser escuchados múltiples veces. Son adecuados para eventos que pueden ser procesados por múltiples oyentes de manera independiente.

Creación de Streams

Stream de Suscripción Única

Puedes crear un stream de suscripción única usando la clase Stream.

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() async {
  await for (var value in countStream(5)) {
    print(value); // Imprime 1, 2, 3, 4, 5
  }
}

Stream de Difusión

Puedes crear un stream de difusión usando la clase StreamController.

import 'dart:async';

void main() {
  final controller = StreamController<int>.broadcast();

  controller.stream.listen((data) {
    print('Oyente 1: $data');
  });

  controller.stream.listen((data) {
    print('Oyente 2: $data');
  });

  for (int i = 1; i <= 3; i++) {
    controller.add(i); // Añade datos al stream
  }

  controller.close();
}

Escuchar Streams

  • await for: Usa esta sintaxis para escuchar un stream de manera asíncrona.
  • listen(): Usa este método para manejar eventos del stream con callbacks.

Usando `await for`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() async {
  await for (var value in countStream(5)) {
    print(value); // Imprime 1, 2, 3, 4, 5
  }
}

Usando `listen()`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() {
  countStream(5).listen((value) {
    print(value); // Imprime 1, 2, 3, 4, 5
  });
}

Transformar Streams

Los streams pueden ser transformados usando varios métodos como map(), where(), expand(), y reduce().

Ejemplo: Usando `map()`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() {
  countStream(5).map((value) => value * 2).listen((value) {
    print(value); // Imprime 2, 4, 6, 8, 10
  });
}

Manejo de Errores en Streams

Los streams pueden emitir eventos de error. Puedes manejar estos errores usando el parámetro onError en el método listen().

Stream<int> errorStream() async* {
  yield 1;
  throw Exception('Ocurrió un error');
  yield 2;
}

void main() {
  errorStream().listen(
    (value) {
      print(value); // Imprime 1
    },
    onError: (error) {
      print('Error: $error'); // Imprime 'Error: Exception: Ocurrió un error'
    },
  );
}

Resumen

Los streams en Dart son una herramienta poderosa para manejar secuencias asíncronas de eventos. Vienen en dos tipos principales: streams de suscripción única y streams de difusión. Puedes crear streams usando la clase Stream o StreamController. Escuchar streams puede hacerse usando await for o listen(). Los streams pueden ser transformados y los errores pueden ser manejados de manera elegante. Entender los streams es esencial para una programación asíncrona efectiva en Dart.
