Introducción

Dart es un lenguaje de programación versátil que se puede usar tanto para el desarrollo del lado del cliente como del lado del servidor. Una de sus características más poderosas es la capacidad de interactuar con HTML, lo que lo convierte en una excelente opción para el desarrollo web. En esta sección, exploraremos cómo usar Dart con HTML, desde conceptos básicos hasta técnicas avanzadas.

Configuración de Dart para el Desarrollo Web

Antes de sumergirnos en el uso de Dart con HTML, configuremos nuestro entorno.

  • Instalar Dart SDK: Asegúrese de tener el Dart SDK instalado en su máquina.
  • Instalar Stagehand: Una herramienta de creación de proyectos Dart. 
    pub global activate stagehand
  • Crear un nuevo proyecto web de Dart: 
    stagehand web-simple
cd <your_project_name>
pub get

Integración Básica con HTML

Comencemos con un ejemplo simple de integración de Dart con HTML.

Estructura HTML

Cree un archivo index.html con el siguiente contenido:

<!DOCTYPE html>
<html>
<head>
  <title>Dart con HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <h1 id="title">¡Hola, Dart!</h1>
</body>
</html>

Código Dart

Cree un archivo main.dart con el siguiente contenido:

import 'dart:html';

void main() {
  querySelector('#title')?.text = '¡Hola, Dart y HTML!';
}

Explicación

  • HTML: El archivo HTML incluye una etiqueta de script que apunta al código Dart compilado (main.dart.js).
  • Dart: El código Dart usa querySelector para seleccionar el elemento HTML con el id title y cambia su contenido de texto.

Interacción con Elementos HTML

Dart proporciona un conjunto rico de APIs para interactuar con elementos HTML.

Selección de Elementos

  • Por ID: 
    var element = querySelector('#elementId');
  • Por Clase: 
    var elements = querySelectorAll('.className');

Modificación de Elementos

  • Cambio de Texto: 
    querySelector('#elementId')?.text = 'Nuevo Texto';
  • Cambio de Atributos: 
    querySelector('#elementId')?.setAttribute('attributeName', 'value');

Ejemplo

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = '¡Botón clickeado!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart con HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <button id="myButton">Haz clic en mí</button>
  <div id="output"></div>
</body>
</html>

Manejo de Eventos

Manejar eventos en Dart es sencillo y similar a JavaScript.

Ejemplo: Evento de Clic

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = '¡Botón clickeado!';
  });
}

Explicación

  • Listener de Evento: El método onClick.listen adjunta un listener de evento al botón, que actualiza el texto del div output cuando se hace clic en el botón.

Técnicas Avanzadas

Usando Dart con Formularios

Manejar formularios es una tarea común en el desarrollo web.

Ejemplo: Envío de Formulario

import 'dart:html';

void main() {
  var form = querySelector('#myForm');
  form?.onSubmit.listen((event) {
    event.preventDefault(); // Prevenir el envío predeterminado del formulario
    var input = (querySelector('#name') as InputElement).value;
    querySelector('#output')?.text = '¡Hola, $input!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart con HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <form id="myForm">
    <input type="text" id="name" placeholder="Introduce tu nombre">
    <button type="submit">Enviar</button>
  </form>
  <div id="output"></div>
</body>
</html>

Explicación

  • Manejo de Formularios: El método onSubmit.listen adjunta un listener de evento al formulario, que previene el envío predeterminado del formulario y actualiza el div output con el valor del input.

Conclusión

Usar Dart con HTML te permite crear aplicaciones web dinámicas e interactivas. Al entender cómo seleccionar y manipular elementos HTML, manejar eventos y trabajar con formularios, puedes aprovechar las capacidades de Dart para construir aplicaciones web robustas. A medida que progreses, puedes explorar características y bibliotecas más avanzadas para mejorar tus habilidades de desarrollo web con Dart.
