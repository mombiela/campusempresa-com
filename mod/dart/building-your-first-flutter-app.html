Introducción a Flutter

Flutter es un kit de herramientas de desarrollo de software de interfaz de usuario de código abierto creado por Google. Se utiliza para desarrollar aplicaciones para Android, iOS, Linux, Mac, Windows, Google Fuchsia y la web desde una única base de código.

  • ¿Por qué Flutter?
    • Una única base de código para múltiples plataformas.
    • Conjunto rico de widgets pre-diseñados.
    • Alto rendimiento debido a la compilación directa a código nativo.
    • Comunidad fuerte y documentación extensa.

Configurando tu Entorno

Antes de comenzar a construir tu primera aplicación Flutter, necesitas configurar tu entorno de desarrollo.

  • Instalar Flutter SDK

    • Descarga el SDK de Flutter desde el sitio web oficial.
    • Extrae el archivo zip y añade el directorio flutter/bin al PATH de tu sistema.

  • Instalar un IDE

    • Recomendado: Visual Studio Code o Android Studio.
    • Instala los plugins de Flutter y Dart para el IDE que elijas.

  • Verificar la Instalación

    • Ejecuta flutter doctor en tu terminal para verificar si hay dependencias faltantes.

Creando un Nuevo Proyecto Flutter

Una vez que tu entorno esté configurado, puedes crear un nuevo proyecto Flutter.

  • Usando la Línea de Comandos

    flutter create my_first_app
cd my_first_app

  • Usando un IDE

    • Abre tu IDE y selecciona "Create New Flutter Project".
    • Sigue las indicaciones para configurar tu proyecto.

Entendiendo la Estructura del Proyecto

Un proyecto Flutter tiene una estructura específica que debes entender.

  • Directorios y Archivos Clave
    • lib/: Contiene el código principal en Dart de tu aplicación.
    • pubspec.yaml: Administra las dependencias del proyecto.
    • android/ y ios/: Código específico de la plataforma para Android e iOS.
    • test/: Contiene pruebas unitarias y de widgets.

Escribiendo tu Primera Aplicación Flutter

Comencemos escribiendo una simple aplicación Flutter que muestre "Hello, World!".

  • main.dart

    import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Hello, World!'),
        ),
        body: Center(
          child: Text('Hello, World!'),
        ),
      ),
    );
  }
}

  • Explicación

    • main(): El punto de entrada de la aplicación.
    • runApp(): Infla el widget dado y lo adjunta a la pantalla.
    • MyApp: Un widget sin estado que construye la interfaz de usuario de la aplicación.
    • MaterialApp: Un widget de conveniencia que envuelve varios widgets comúnmente requeridos para aplicaciones de diseño material.
    • Scaffold: Implementa la estructura visual básica del diseño material.
    • AppBar: Una barra de aplicaciones de diseño material.
    • Center: Centra su widget hijo.
    • Text: Un widget que muestra una cadena de texto.

Ejecutando tu Aplicación

Puedes ejecutar tu aplicación Flutter en un emulador o en un dispositivo físico.

  • Usando la Línea de Comandos

    flutter run

  • Usando un IDE

    • Haz clic en el botón "Run" en tu IDE.

Añadiendo Interactividad

Añadamos un botón que cambie el texto cuando se presione.

  • main.dart Actualizado

    import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  String _displayText = 'Hello, World!';

  void _updateText() {
    setState(() {
      _displayText = 'You pressed the button!';
    });
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Interactive App'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              Text(_displayText),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _updateText,
                child: Text('Press me'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

  • Explicación

    • StatefulWidget: Un widget que tiene estado mutable.
    • _MyAppState: La clase de estado para MyApp.
    • _displayText: Una cadena que contiene el texto a mostrar.
    • _updateText(): Un método que actualiza _displayText y llama a setState() para reconstruir la interfaz de usuario.
    • Column: Un widget que organiza sus hijos en una columna vertical.
    • ElevatedButton: Un botón elevado de diseño material.

Conclusión

En este tema, aprendiste cómo configurar tu entorno de desarrollo Flutter, crear un nuevo proyecto Flutter, entender la estructura del proyecto, escribir una simple aplicación Flutter, ejecutar la aplicación y añadir interactividad. Este conocimiento fundamental te ayudará a medida que continúes explorando y construyendo aplicaciones Flutter más complejas.
