Introducción

Iterar a través de colecciones es un concepto fundamental en la programación. En Dart, las colecciones como listas, conjuntos y mapas se usan comúnmente, y saber cómo iterar a través de ellas de manera eficiente es crucial. Esta sección cubrirá varias formas de iterar a través de colecciones en Dart, desde bucles básicos hasta técnicas más avanzadas.

Iteración Básica con el Bucle `for`

El bucle for es una forma básica y directa de iterar a través de colecciones.

Ejemplo: Iterando a través de una Lista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  
  for (int i = 0; i < numbers.length; i++) {
    print(numbers[i]);
  }
}
  • Explicación: Este ejemplo usa un bucle for tradicional para iterar a través de una lista de enteros. El bucle se ejecuta desde 0 hasta numbers.length - 1, imprimiendo cada elemento.

Iteración con el Bucle `for-in`

El bucle for-in proporciona una forma más concisa de iterar a través de colecciones.

Ejemplo: Iterando a través de una Lista

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  
  for (var fruit in fruits) {
    print(fruit);
  }
}
  • Explicación: El bucle for-in itera directamente sobre los elementos de la colección, haciendo que el código sea más limpio y fácil de leer.

Iterando a través de Conjuntos

Los conjuntos en Dart son colecciones desordenadas de elementos únicos. Puedes iterar a través de ellos de manera similar a las listas.

Ejemplo: Iterando a través de un Conjunto

void main() {
  Set<String> animals = {'Dog', 'Cat', 'Bird'};
  
  for (var animal in animals) {
    print(animal);
  }
}
  • Explicación: El bucle for-in funciona perfectamente con conjuntos, iterando sobre cada elemento único.

Iterando a través de Mapas

Los mapas son colecciones de pares clave-valor. Iterar a través de mapas se puede hacer de varias maneras, dependiendo de si necesitas claves, valores o ambos.

Ejemplo: Iterando a través de Entradas de un Mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var entry in scores.entries) {
    print('${entry.key}: ${entry.value}');
  }
}
  • Explicación: Usando entries, puedes acceder tanto a las claves como a los valores en cada iteración.

Ejemplo: Iterando a través de Claves de un Mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var key in scores.keys) {
    print('Key: $key');
  }
}
  • Explicación: Usando keys, puedes iterar a través de todas las claves en el mapa.

Ejemplo: Iterando a través de Valores de un Mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var value in scores.values) {
    print('Value: $value');
  }
}
  • Explicación: Usando values, puedes iterar a través de todos los valores en el mapa.

Iteración Avanzada con `forEach`

El método forEach proporciona un enfoque funcional para iterar a través de colecciones.

Ejemplo: Usando `forEach` con una Lista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  
  numbers.forEach((number) {
    print(number);
  });
}
  • Explicación: El método forEach toma una función como argumento y la aplica a cada elemento en la lista.

Ejemplo: Usando `forEach` con un Mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  scores.forEach((key, value) {
    print('$key: $value');
  });
}
  • Explicación: El método forEach para mapas toma una función con dos parámetros (clave y valor) y la aplica a cada entrada.

Conclusión

Iterar a través de colecciones es una habilidad versátil en la programación con Dart. Ya sea usando bucles básicos, el bucle for-in o el método forEach, entender estas técnicas te ayudará a manejar colecciones de manera eficiente. Practica estos métodos para volverte competente en iterar a través de diferentes tipos de colecciones en Dart.
