Introducción

Las pruebas y la depuración son aspectos cruciales del desarrollo de software. Aseguran que tus aplicaciones Dart funcionen sin problemas y estén libres de errores. Esta sección cubrirá los conceptos básicos de las pruebas y la depuración en Dart, progresando desde niveles principiantes hasta avanzados.

Pruebas en Dart

Por Qué las Pruebas son Importantes

  • Aseguran la fiabilidad y corrección del código.
  • Facilitan el mantenimiento y la refactorización del código.
  • Ayudan a detectar errores temprano en el proceso de desarrollo.

Tipos de Pruebas

  • Pruebas Unitarias: Prueban unidades individuales de código (funciones, métodos).
  • Pruebas de Widgets: Prueban los componentes de la interfaz de usuario en aplicaciones Flutter.
  • Pruebas de Integración: Prueban una aplicación completa o una gran parte de ella.

Configuración de un Entorno de Pruebas

Para comenzar a probar en Dart, necesitas agregar el paquete test a tu archivo pubspec.yaml:

dev_dependencies:
  test: ^1.16.0

Escribiendo Tu Primera Prueba Unitaria

Aquí hay un ejemplo simple de una prueba unitaria en Dart:

import 'package:test/test.dart';

int add(int a, int b) {
  return a + b;
}

void main() {
  test('suma dos números', () {
    expect(add(1, 2), 3);
  });
}
  • Explicación:
    • import 'package:test/test.dart';: Importa el paquete de pruebas.
    • int add(int a, int b): Una función simple para sumar dos números.
    • test('suma dos números', () { ... }): Define un caso de prueba.
    • expect(add(1, 2), 3): Verifica si la función add devuelve 3 cuando se le da 1 y 2.

Ejecutando Pruebas

Para ejecutar tus pruebas, usa el siguiente comando en tu terminal:

dart test

Pruebas de Widgets en Flutter

Para aplicaciones Flutter, las pruebas de widgets son esenciales para asegurar que los componentes de la interfaz de usuario se comporten como se espera.

import 'package:flutter_test/flutter_test.dart';
import 'package:my_app/my_widget.dart';

void main() {
  testWidgets('MyWidget tiene un título y un mensaje', (WidgetTester tester) async {
    await tester.pumpWidget(MyWidget(title: 'T', message: 'M'));

    final titleFinder = find.text('T');
    final messageFinder = find.text('M');

    expect(titleFinder, findsOneWidget);
    expect(messageFinder, findsOneWidget);
  });
}
  • Explicación:
    • import 'package:flutter_test/flutter_test.dart';: Importa el paquete de pruebas de Flutter.
    • testWidgets('MyWidget tiene un título y un mensaje', (WidgetTester tester) async { ... }): Define una prueba de widget.
    • await tester.pumpWidget(MyWidget(title: 'T', message: 'M'));: Construye el widget.
    • final titleFinder = find.text('T');: Encuentra el texto del título.
    • expect(titleFinder, findsOneWidget);: Verifica que el título esté presente.

Depuración en Dart

Técnicas Comunes de Depuración

  • Declaraciones Print: Simples pero efectivas para problemas pequeños.
  • Depurador: Usa IDEs como VSCode o IntelliJ que tienen depuradores integrados.
  • Registro: Usa bibliotecas de registro para aplicaciones más complejas.

Uso de Declaraciones Print

void main() {
  int a = 5;
  int b = 10;
  print('a: $a, b: $b'); // Salida de depuración
  int sum = a + b;
  print('Suma: $sum'); // Salida de depuración
}
  • Explicación:
    • print('a: $a, b: $b');: Muestra los valores de a y b.
    • print('Suma: $sum');: Muestra el resultado de la suma.

Uso del Depurador

  • Establecer Puntos de Interrupción: Haz clic en el margen junto al número de línea en tu IDE.
  • Paso a Paso por el Código: Usa las funciones de paso sobre, paso dentro y paso fuera para navegar por tu código.
  • Inspección de Variables: Pasa el cursor sobre las variables para ver sus valores actuales.

Registro

Agrega el paquete logging a tu archivo pubspec.yaml:

dependencies:
  logging: ^0.11.4

Ejemplo de uso del paquete de registro:

import 'package:logging/logging.dart';

void main() {
  final Logger logger = Logger('MyLogger');
  logger.onRecord.listen((record) {
    print('${record.level.name}: ${record.time}: ${record.message}');
  });

  logger.info('Este es un mensaje informativo');
  logger.warning('Este es un mensaje de advertencia');
  logger.severe('Este es un mensaje severo');
}
  • Explicación:
    • import 'package:logging/logging.dart';: Importa el paquete de registro.
    • Logger logger = Logger('MyLogger');: Crea una instancia de logger.
    • logger.onRecord.listen((record) { ... });: Escucha los registros y los imprime.
    • logger.info('Este es un mensaje informativo');: Registra un mensaje informativo.

Conclusión

Las pruebas y la depuración son habilidades esenciales para cualquier desarrollador de Dart. Al escribir pruebas, aseguras que tu código sea fiable y mantenible. La depuración te ayuda a identificar y solucionar problemas de manera eficiente. A medida que progreses de niveles principiantes a avanzados, estas prácticas se volverán naturales, mejorando significativamente tu flujo de trabajo de desarrollo.
