Introducción al Sistema de Tipos de Dart

Dart es un lenguaje tipado estáticamente, lo que significa que el tipo de una variable se conoce en tiempo de compilación. Esto ayuda a detectar errores temprano y mejora la calidad y el rendimiento del código.

Conceptos Clave

  • Tipado Estático: Los tipos se verifican en tiempo de compilación.
  • Inferencia de Tipos: Dart puede inferir tipos, por lo que no siempre necesitas declararlos explícitamente.
  • Tipo Dinámico: Dart también soporta tipado dinámico usando la palabra clave dynamic.

Tipos Básicos

Dart proporciona varios tipos incorporados que son comúnmente utilizados en programación.

Tipos Comunes

  • int: Representa valores enteros.
  • double: Representa valores de punto flotante.
  • String: Representa una secuencia de caracteres.
  • bool: Representa valores booleanos (true o false).
  • List: Representa una colección de objetos ordenados.
  • Map: Representa una colección de pares clave-valor.

Ejemplos de Código

void main() {
  int age = 25;
  double height = 5.9;
  String name = 'Alice';
  bool isStudent = true;
  List<int> numbers = [1, 2, 3, 4, 5];
  Map<String, String> capitals = {
    'USA': 'Washington, D.C.',
    'France': 'París',
    'Japan': 'Tokio'
  };

  print('Nombre: $name, Edad: $age, Altura: $height, Estudiante: $isStudent');
  print('Números: $numbers');
  print('Capitales: $capitals');
}

Inferencia de Tipos

Dart puede inferir el tipo de una variable basado en el valor asignado, lo que hace que el código sea más limpio y fácil de leer.

Ejemplos de Código

void main() {
  var age = 25; // Dart infiere que age es de tipo int
  var height = 5.9; // Dart infiere que height es de tipo double
  var name = 'Alice'; // Dart infiere que name es de tipo String
  var isStudent = true; // Dart infiere que isStudent es de tipo bool

  print('Nombre: $name, Edad: $age, Altura: $height, Estudiante: $isStudent');
}

Tipo Dinámico

La palabra clave dynamic permite que una variable contenga valores de cualquier tipo. Esto es útil cuando el tipo de la variable no se conoce en tiempo de compilación.

Ejemplos de Código

void main() {
  dynamic variable = 25;
  print('Variable: $variable, Tipo: ${variable.runtimeType}');

  variable = 'Hola';
  print('Variable: $variable, Tipo: ${variable.runtimeType}');

  variable = true;
  print('Variable: $variable, Tipo: ${variable.runtimeType}');
}

Seguridad de Tipos

Dart asegura la seguridad de tipos al prevenir operaciones que no están permitidas en ciertos tipos.

Ejemplos de Código

void main() {
  int age = 25;
  // age = 'veinticinco'; // Esto causará un error en tiempo de compilación

  String name = 'Alice';
  // name = 123; // Esto causará un error en tiempo de compilación

  print('Nombre: $name, Edad: $age');
}

Conversión de Tipos

A veces, puede que necesites convertir una variable de un tipo a otro. Dart proporciona varias formas de realizar conversiones de tipos.

Ejemplos de Código

void main() {
  // String a int
  String ageStr = '25';
  int age = int.parse(ageStr);
  print('Edad: $age');

  // int a String
  int year = 2023;
  String yearStr = year.toString();
  print('Año: $yearStr');

  // double a int
  double height = 5.9;
  int roundedHeight = height.toInt();
  print('Altura Redondeada: $roundedHeight');
}

Seguridad de Nulos

Dart tiene una característica de seguridad de nulos que ayuda a prevenir errores de referencia nula. Por defecto, las variables no pueden ser nulas a menos que se declaren explícitamente como anulables.

Ejemplos de Código

void main() {
  int? nullableAge;
  nullableAge = null; // Esto está permitido

  int nonNullableAge = 25;
  // nonNullableAge = null; // Esto causará un error en tiempo de compilación

  print('Edad Anulable: $nullableAge, Edad No Anulable: $nonNullableAge');
}

Resumen

En esta sección, cubrimos lo esencial del sistema de tipos de Dart, incluyendo:

  • Tipos básicos y su uso.
  • Inferencia de tipos para un código más limpio.
  • El tipo dynamic para flexibilidad.
  • Seguridad de tipos para prevenir errores.
  • Conversión de tipos para convertir entre tipos.
  • Seguridad de nulos para evitar errores de referencia nula.

Entender el sistema de tipos de Dart es crucial para escribir código robusto y libre de errores. A medida que progreses, encontrarás que estos conceptos forman la base para técnicas de programación en Dart más avanzadas.
