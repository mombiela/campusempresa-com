Introducción

¡Bienvenido a tu primera lección de programación en Dart! Dart es un lenguaje optimizado para clientes que permite crear aplicaciones rápidas en cualquier plataforma. En esta sección, aprenderás a escribir, ejecutar y entender un programa básico en Dart. Al final de esta lección, tendrás una base sólida sobre la cual construir a medida que avances en el curso.

Configuración de Tu Entorno

Antes de escribir tu primer programa en Dart, necesitas configurar tu entorno de desarrollo. Sigue estos pasos:

  1. Instalar Dart SDK: Descarga e instala el SDK de Dart desde el sitio web oficial de Dart.
  2. Elige un IDE: Puedes usar cualquier editor de texto, pero IDEs como Visual Studio Code o IntelliJ IDEA ofrecen un gran soporte para Dart.
  3. Instalar el Plugin de Dart: Si estás usando un IDE, asegúrate de instalar el plugin de Dart para resaltar la sintaxis y otras características.

Escribiendo Tu Primer Programa en Dart

Vamos a escribir un programa simple en Dart que imprima "¡Hola, Mundo!" en la consola.

Ejemplo de Código

Crea un nuevo archivo llamado main.dart y añade el siguiente código:

void main() {
  print('¡Hola, Mundo!');
}

Explicación

  • void main() { ... }: Esta es la función principal, el punto de entrada de cada aplicación en Dart. La palabra clave void indica que la función no devuelve un valor.
  • print('¡Hola, Mundo!');: Esta línea imprime la cadena '¡Hola, Mundo!' en la consola. La función print es una función incorporada en Dart utilizada para la salida.

Ejecutando Tu Programa en Dart

Para ejecutar tu programa en Dart, abre una terminal o símbolo del sistema y navega al directorio donde se encuentra tu archivo main.dart. Luego, ejecuta el siguiente comando:

dart run main.dart

Deberías ver la salida:

¡Hola, Mundo!

Conceptos Clave

  • Función Principal: El punto de inicio de un programa en Dart.
  • Función Print: Utilizada para mostrar texto en la consola.
  • Comentarios: Usa // para comentarios de una sola línea y /* ... */ para comentarios de varias líneas.

Ejemplo con Comentarios

// Este es un comentario de una sola línea

/*
 Este es un comentario de varias líneas
 que abarca múltiples líneas.
*/

void main() {
  // Imprime un saludo en la consola
  print('¡Hola, Mundo!');
}

Resumen

En esta lección, has aprendido a:

  • Configurar tu entorno de desarrollo en Dart.
  • Escribir un programa simple en Dart.
  • Entender la estructura de un programa básico en Dart.
  • Ejecutar tu programa en Dart y ver la salida.

¡Felicitaciones por escribir tu primer programa en Dart! Este conocimiento fundamental será crucial a medida que avances hacia temas más complejos. Sigue practicando, y pronto estarás construyendo aplicaciones más avanzadas con Dart.
