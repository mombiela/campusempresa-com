Exámenes Simulados
Visión General del Proyecto ►

Los exámenes simulados son una parte esencial del proceso de aprendizaje, especialmente para materias complejas como Hadoop. Te ayudan a evaluar tu comprensión, identificar áreas de mejora y acostumbrarte al formato y tipos de preguntas que podrías encontrar en exámenes reales o entrevistas de trabajo.

Nivel Principiante

En el nivel principiante, los exámenes simulados se centrarán en conceptos fundamentales y operaciones básicas en Hadoop.

Conceptos Clave

  • Comprensión del Ecosistema Hadoop
  • Comandos básicos de HDFS
  • Introducción a MapReduce
  • Consultas básicas de Hive

Preguntas de Ejemplo

Opción Múltiple

  1. ¿Cuál es la función principal de HDFS en Hadoop?

    • A. Procesamiento de Datos
    • B. Almacenamiento de Datos
    • C. Análisis de Datos
    • D. Visualización de Datos

    Respuesta: B. Almacenamiento de Datos

  2. ¿Cuál de los siguientes es un componente central de Hadoop?

    • A. Apache Spark
    • B. Apache Kafka
    • C. HDFS
    • D. Apache Flink

    Respuesta: C. HDFS

Tareas Prácticas

  1. Lista todos los archivos en el directorio raíz de HDFS.

    hdfs dfs -ls /

  2. Escribe un trabajo simple de MapReduce para contar el número de ocurrencias de cada palabra en un archivo de texto.

    import java.io.IOException;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class WordCount {
    public static class TokenizerMapper extends Mapper<Object, Text, Text, IntWritable>{
        private final static IntWritable one = new IntWritable(1);
        private Text word = new Text();

        public void map(Object key, Text value, Context context) throws IOException, InterruptedException {
            String[] words = value.toString().split("\\s+");
            for (String str : words) {
                word.set(str);
                context.write(word, one);
            }
        }
    }

    public static class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
        private IntWritable result = new IntWritable();

        public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            result.set(sum);
            context.write(key, result);
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "word count");
        job.setJarByClass(WordCount.class);
        job.setMapperClass(TokenizerMapper.class);
        job.setCombinerClass(IntSumReducer.class);
        job.setReducerClass(IntSumReducer.class);
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);
        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));
        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

Nivel Intermedio

En el nivel intermedio, los exámenes simulados cubrirán temas más avanzados y aplicaciones prácticas.

Conceptos Clave

  • Operaciones avanzadas de HDFS
  • Trabajos complejos de MapReduce
  • Introducción a YARN
  • Consultas avanzadas de Hive y Pig

Preguntas de Ejemplo

Opción Múltiple

  1. ¿Cuál es el papel de YARN en Hadoop?

    • A. Almacenamiento de Datos
    • B. Gestión de Recursos
    • C. Análisis de Datos
    • D. Visualización de Datos

    Respuesta: B. Gestión de Recursos

  2. ¿Qué comando se usa para mover archivos en HDFS?

    • A. hdfs dfs -mv
    • B. hdfs dfs -cp
    • C. hdfs dfs -rm
    • D. hdfs dfs -put

    Respuesta: A. hdfs dfs -mv

Tareas Prácticas

  1. Escribe una consulta de Hive para encontrar los 5 empleados mejor pagados de una tabla de empleados.

    SELECT name, salary 
FROM employee 
ORDER BY salary DESC 
LIMIT 5;

  2. Crea un script de Pig para cargar datos desde un archivo CSV y filtrar registros donde la edad sea mayor a 30.

    data = LOAD 'employee.csv' USING PigStorage(',') AS (name:chararray, age:int, salary:float);
filtered_data = FILTER data BY age > 30;
DUMP filtered_data;

Nivel Avanzado

En el nivel avanzado, los exámenes simulados evaluarán tu capacidad para manejar escenarios complejos y optimizar operaciones de Hadoop.

Conceptos Clave

  • Gestión de Clústeres Hadoop
  • Optimización del Rendimiento
  • Configuraciones avanzadas de YARN
  • Integración con otras herramientas de Big Data

Preguntas de Ejemplo

Opción Múltiple

  1. ¿Cuál de los siguientes es un método para mejorar el rendimiento de un clúster Hadoop?

    • A. Aumentar el número de mappers
    • B. Disminuir el tamaño del bloque
    • C. Usar compresión
    • D. Reducir el número de reducers

    Respuesta: C. Usar compresión

  2. ¿Cuál es el propósito del Secondary NameNode en Hadoop?

    • A. Reemplazar el NameNode en caso de falla
    • B. Realizar recolección de basura
    • C. Fusionar el registro de ediciones con la imagen del sistema
    • D. Gestionar la replicación de datos

    Respuesta: C. Fusionar el registro de ediciones con la imagen del sistema

Tareas Prácticas

  1. Configura YARN para asignar más memoria a ApplicationMaster.

    <property>
    <name>yarn.app.mapreduce.am.resource.mb</name>
    <value>2048</value>
</property>

  2. Escribe un trabajo de Spark para leer datos desde HDFS y realizar un conteo de palabras.

    from pyspark import SparkContext, SparkConf

conf = SparkConf().setAppName("WordCount")
sc = SparkContext(conf=conf)

text_file = sc.textFile("hdfs:///user/hadoop/input.txt")
counts = text_file.flatMap(lambda line: line.split(" ")) 
                         .map(lambda word: (word, 1)) 
                         .reduceByKey(lambda a, b: a + b)

counts.saveAsTextFile("hdfs:///user/hadoop/output")
