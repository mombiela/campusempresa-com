En esta sección, cubriremos los aspectos prácticos de la implementación de un proyecto utilizando Hadoop. Esto incluirá la configuración del entorno, la escritura y ejecución de trabajos MapReduce, y el uso de otras herramientas del ecosistema Hadoop como Hive y Pig. Al final de esta sección, deberías ser capaz de implementar una canalización completa de procesamiento de datos utilizando Hadoop.

Configuración del Entorno

Antes de sumergirnos en la escritura de código, necesitamos configurar nuestro entorno Hadoop. Esto implica instalar Hadoop y configurarlo tanto para clústeres de un solo nodo como de múltiples nodos.

Configuración de un Solo Nodo

Una configuración de un solo nodo es ideal para propósitos de desarrollo y prueba.

  1. Descargar Hadoop: Descarga la última versión estable de Hadoop desde el sitio web oficial.
  2. Extraer Hadoop: Extrae el archivo tar descargado en un directorio de tu elección.
  3. Configurar Hadoop: Edita los archivos de configuración ubicados en el directorio etc/hadoop.
    • core-site.xml
    • hdfs-site.xml
    • mapred-site.xml
    • yarn-site.xml
  4. Iniciar Hadoop: Usa los siguientes comandos para iniciar los servicios de Hadoop: 
    start-dfs.sh
start-yarn.sh

Configuración de Múltiples Nodos

Una configuración de múltiples nodos se utiliza para entornos de producción.

  1. Requisitos Previos: Asegúrate de que SSH esté instalado y configurado para el inicio de sesión sin contraseña entre nodos.
  2. Instalar Hadoop: Instala Hadoop en todos los nodos.
  3. Configurar Nodos Maestro y Esclavo: Edita los archivos masters y slaves para especificar los nodos maestro y esclavo.
  4. Distribuir Configuración: Copia los archivos de configuración desde el nodo maestro a todos los nodos esclavos.
  5. Iniciar Hadoop: Inicia los servicios de Hadoop en todos los nodos utilizando los mismos comandos que en la configuración de un solo nodo.

Escritura y Ejecución de Trabajos MapReduce

MapReduce es el componente central de Hadoop para procesar grandes conjuntos de datos.

Escritura de un Trabajo MapReduce

Un trabajo MapReduce consiste en un Mapper y un Reducer.

  • Mapper: Procesa datos de entrada y produce pares clave-valor intermedios.
  • Reducer: Procesa pares clave-valor intermedios y produce la salida final.

Aquí hay un ejemplo simple de un trabajo MapReduce en Java que cuenta el número de ocurrencias de cada palabra en un archivo de texto:

import java.io.IOException;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class WordCount {

    public static class TokenizerMapper extends Mapper<Object, Text, Text, IntWritable> {
        private final static IntWritable one = new IntWritable(1);
        private Text word = new Text();

        public void map(Object key, Text value, Context context) throws IOException, InterruptedException {
            String[] tokens = value.toString().split("\\s+");
            for (String token : tokens) {
                word.set(token);
                context.write(word, one);
            }
        }
    }

    public static class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
        public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            context.write(key, new IntWritable(sum));
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "word count");
        job.setJarByClass(WordCount.class);
        job.setMapperClass(TokenizerMapper.class);
        job.setCombinerClass(IntSumReducer.class);
        job.setReducerClass(IntSumReducer.class);
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);
        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));
        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

Ejecución de un Trabajo MapReduce

  1. Compilar el Código: Compila el código Java utilizando javac. 
    javac -classpath `hadoop classpath` -d wordcount_classes WordCount.java
  2. Crear un Archivo JAR: Empaqueta las clases compiladas en un archivo JAR. 
    jar -cvf wordcount.jar -C wordcount_classes/ .
  3. Ejecutar el Trabajo: Envía el trabajo al clúster Hadoop. 
    hadoop jar wordcount.jar WordCount /input /output

Uso de Hive y Pig

Hive y Pig son herramientas de alto nivel en el ecosistema Hadoop que simplifican el proceso de escribir trabajos de procesamiento de datos.

Hive

Hive proporciona una interfaz similar a SQL para consultar datos almacenados en Hadoop.

  • Crear una Tabla: 
    CREATE TABLE word_counts (word STRING, count INT) ROW FORMAT DELIMITED FIELDS TERMINATED BY '\t';
  • Cargar Datos: 
    LOAD DATA INPATH '/output' INTO TABLE word_counts;
  • Consultar Datos: 
    SELECT word, count FROM word_counts WHERE count > 10;

Pig

Pig utiliza un lenguaje de scripting llamado Pig Latin para procesar datos.

  • Cargar Datos: 
    word_counts = LOAD '/output' USING PigStorage('\t') AS (word:chararray, count:int);
  • Filtrar Datos: 
    filtered_word_counts = FILTER word_counts BY count > 10;
  • Almacenar Datos: 
    STORE filtered_word_counts INTO '/filtered_output';

Conclusión

En esta sección, cubrimos los pasos esenciales para implementar un proyecto utilizando Hadoop. Comenzamos con la configuración del entorno, luego pasamos a la escritura y ejecución de trabajos MapReduce, y finalmente exploramos herramientas de alto nivel como Hive y Pig. Con este conocimiento, deberías estar bien equipado para abordar tareas de procesamiento de datos del mundo real utilizando Hadoop.
