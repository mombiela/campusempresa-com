Introducción a Hadoop

Hadoop es un marco de trabajo de código abierto que permite el procesamiento distribuido de grandes conjuntos de datos a través de clústeres de computadoras utilizando modelos de programación simples. Está diseñado para escalar desde servidores individuales hasta miles de máquinas, cada una ofreciendo computación y almacenamiento local.

Conceptos Clave

  • Almacenamiento Distribuido: Hadoop utiliza el Sistema de Archivos Distribuidos de Hadoop (HDFS) para almacenar datos en múltiples máquinas.
  • Procesamiento Distribuido: Hadoop procesa datos utilizando el modelo de programación MapReduce.
  • Escalabilidad: Hadoop puede escalar fácilmente desde un solo nodo hasta miles de nodos.
  • Tolerancia a Fallos: Hadoop maneja automáticamente fallos de hardware y asegura la integridad de los datos.

Componentes de Hadoop

Hadoop consta de varios componentes principales que trabajan juntos para proporcionar un entorno de procesamiento de datos robusto y escalable.

Sistema de Archivos Distribuidos de Hadoop (HDFS)

HDFS está diseñado para almacenar conjuntos de datos muy grandes de manera confiable y para transmitir esos conjuntos de datos a alta velocidad a las aplicaciones de los usuarios.

  • NameNode: Gestiona los metadatos y el espacio de nombres del sistema de archivos.
  • DataNode: Almacena los bloques de datos reales.

MapReduce

MapReduce es un modelo de programación y una implementación asociada para procesar y generar grandes conjuntos de datos.

  • Función Map: Procesa datos de entrada y produce un conjunto de pares clave/valor intermedios.
  • Función Reduce: Fusiona todos los valores intermedios asociados con la misma clave intermedia.

YARN (Yet Another Resource Negotiator)

YARN es el sistema de gestión de recursos de clúster de Hadoop.

  • ResourceManager: Gestiona recursos y programa aplicaciones.
  • NodeManager: Gestiona recursos en un solo nodo.

Ecosistema de Hadoop

El ecosistema de Hadoop incluye una variedad de herramientas y tecnologías que mejoran las capacidades de los componentes principales de Hadoop.

  • Hive: Almacenamiento de datos y lenguaje de consulta similar a SQL.
  • Pig: Plataforma de alto nivel para crear programas MapReduce.
  • HBase: Almacén de datos grande, distribuido y escalable.
  • Spark: Sistema de computación en clúster rápido y de propósito general.
  • Sqoop: Herramienta para transferir datos entre Hadoop y bases de datos relacionales.
  • Flume: Servicio para recopilar, agregar y mover grandes cantidades de datos de registro de manera eficiente.

Ejemplo: Programa de Conteo de Palabras

Para ilustrar cómo funciona Hadoop, veamos un ejemplo simple: un programa de conteo de palabras utilizando MapReduce.

Código del Mapper

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

public class TokenizerMapper extends Mapper<LongWritable, Text, Text, IntWritable> {
    private final static IntWritable one = new IntWritable(1);
    private Text word = new Text();

    public void map(LongWritable key, Text value, Context context) throws IOException, InterruptedException {
        String[] tokens = value.toString().split("\\s+");
        for (String token : tokens) {
            word.set(token);
            context.write(word, one);
        }
    }
}

Código del Reducer

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Reducer;

public class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
    public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
        int sum = 0;
        for (IntWritable val : values) {
            sum += val.get();
        }
        context.write(key, new IntWritable(sum));
    }
}

Explicación

  • Mapper: La clase TokenizerMapper toma una línea de texto como entrada y la divide en palabras. Cada palabra se emite como una clave con un valor de 1.
  • Reducer: La clase IntSumReducer toma la lista de valores asociados con una palabra (clave) y los suma para obtener el conteo total de esa palabra.

Conclusión

En esta sección, proporcionamos una visión general de Hadoop, sus componentes principales y el ecosistema más amplio. También revisamos un ejemplo simple de un programa de conteo de palabras para ilustrar cómo funciona el modelo MapReduce de Hadoop. Esta comprensión fundamental será crucial a medida que profundicemos en temas más avanzados y aplicaciones prácticas de Hadoop en las secciones posteriores de este curso.
