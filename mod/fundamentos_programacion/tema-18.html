Introducción

En este tema, exploraremos dos estructuras de datos fundamentales en la programación: las pilas y las colas. Estas estructuras son esenciales para la gestión de datos en muchos algoritmos y aplicaciones. Aprenderemos qué son, cómo funcionan, y cómo implementarlas en código.

Pilas

Definición

Una pila es una estructura de datos que sigue el principio LIFO (Last In, First Out), es decir, el último elemento en entrar es el primero en salir.

Operaciones Básicas

  • Push: Añadir un elemento al tope de la pila.
  • Pop: Eliminar el elemento del tope de la pila.
  • Peek: Obtener el valor del elemento en el tope sin eliminarlo.
  • isEmpty: Verificar si la pila está vacía.

Ejemplo de Código en Python

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return None

    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        else:
            return None

    def is_empty(self):
        return len(self.items) == 0

# Ejemplo de uso
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.peek())  # Salida: 2
print(pila.pop())   # Salida: 2
print(pila.pop())   # Salida: 1
print(pila.is_empty())  # Salida: True

Aplicaciones Comunes

  • Evaluación de expresiones: Utilizadas en la evaluación de expresiones aritméticas y lógicas.
  • Recursión: Las pilas son fundamentales para la implementación de llamadas recursivas.
  • Deshacer/rehacer: Funcionalidades en editores de texto y otros programas.

Colas

Definición

Una cola es una estructura de datos que sigue el principio FIFO (First In, First Out), es decir, el primer elemento en entrar es el primero en salir.

Operaciones Básicas

  • Enqueue: Añadir un elemento al final de la cola.
  • Dequeue: Eliminar el primer elemento de la cola.
  • Front: Obtener el valor del primer elemento sin eliminarlo.
  • isEmpty: Verificar si la cola está vacía.

Ejemplo de Código en Python

class Cola:
    def __init__(self):
        self.items = []

    def enqueue(self, item):
        self.items.append(item)

    def dequeue(self):
        if not self.is_empty():
            return self.items.pop(0)
        else:
            return None

    def front(self):
        if not self.is_empty():
            return self.items[0]
        else:
            return None

    def is_empty(self):
        return len(self.items) == 0

# Ejemplo de uso
cola = Cola()
cola.enqueue(1)
cola.enqueue(2)
print(cola.front())  # Salida: 1
print(cola.dequeue())  # Salida: 1
print(cola.dequeue())  # Salida: 2
print(cola.is_empty())  # Salida: True

Aplicaciones Comunes

  • Gestión de tareas: Utilizadas en sistemas operativos para gestionar procesos.
  • Sistemas de impresión: Manejo de trabajos de impresión en una impresora.
  • Redes: Gestión de paquetes de datos en redes de comunicación.

Comparación entre Pilas y Colas

| Característica | Pila (LIFO) | Cola (FIFO) | |----------------|-------------|-------------| | Orden de acceso | Último en entrar, primero en salir | Primero en entrar, primero en salir | | Operaciones principales | Push, Pop, Peek, isEmpty | Enqueue, Dequeue, Front, isEmpty | | Ejemplos de uso | Evaluación de expresiones, Recursión, Deshacer/rehacer | Gestión de tareas, Sistemas de impresión, Redes |

Conclusión

Las pilas y las colas son estructuras de datos fundamentales que todo programador debe conocer. Entender cómo funcionan y cómo implementarlas es crucial para resolver una variedad de problemas en la programación. Practicar con ejemplos y ejercicios ayudará a consolidar estos conceptos y a aplicarlos en situaciones reales.
