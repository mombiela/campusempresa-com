Declaraciones de Control de Flujo en Flutter
Funciones y Métodos ►

Las declaraciones de control de flujo son fundamentales en cualquier lenguaje de programación, incluido Dart, que se utiliza en Flutter. Estas declaraciones te permiten controlar el flujo de ejecución de tu código, haciendo posible realizar diferentes acciones basadas en condiciones, repetir acciones y manejar excepciones.

Declaraciones Condicionales

Las declaraciones condicionales te permiten ejecutar ciertas piezas de código basadas en si una condición es verdadera o falsa.

if y else

La declaración if se usa para ejecutar un bloque de código si una condición es verdadera. La declaración else se puede usar para ejecutar un bloque de código si la condición es falsa.

void main() {
  int number = 10;

  if (number > 0) {
    print('El número es positivo.');
  } else {
    print('El número no es positivo.');
  }
}

else if

La declaración else if te permite verificar múltiples condiciones.

void main() {
  int number = 0;

  if (number > 0) {
    print('El número es positivo.');
  } else if (number < 0) {
    print('El número es negativo.');
  } else {
    print('El número es cero.');
  }
}

Declaración Switch

La declaración switch te permite ejecutar un bloque de código entre muchos basados en el valor de una variable.

void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print('¡Excelente!');
      break;
    case 'B':
      print('¡Bien!');
      break;
    case 'C':
      print('¡Regular!');
      break;
    case 'D':
      print('¡Pobre!');
      break;
    default:
      print('Calificación inválida.');
  }
}

Declaraciones de Bucle

Las declaraciones de bucle te permiten ejecutar un bloque de código múltiples veces.

Bucle for

El bucle for se usa para ejecutar un bloque de código un número específico de veces.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteración $i');
  }
}

Bucle while

El bucle while se usa para ejecutar un bloque de código mientras una condición sea verdadera.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteración $i');
    i++;
  }
}

Bucle do-while

El bucle do-while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez.

void main() {
  int i = 0;

  do {
    print('Iteración $i');
    i++;
  } while (i < 5);
}

Manejo de Excepciones

El manejo de excepciones te permite gestionar errores y otros eventos excepcionales en tu código.

try y catch

El bloque try te permite probar un bloque de código en busca de errores, mientras que el bloque catch te permite manejar el error.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('Ocurrió un error: $e');
  }
}

finally

El bloque finally te permite ejecutar código, independientemente de si se lanzó una excepción o no.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('Ocurrió un error: $e');
  } finally {
    print('Este código se ejecuta sin importar qué.');
  }
}

Conclusión

Las declaraciones de control de flujo son esenciales para gestionar la ejecución de tu código. Al comprender y usar estas declaraciones de manera efectiva, puedes crear aplicaciones Flutter más dinámicas y receptivas. Practica usando estas declaraciones en diferentes escenarios para volverte competente en el control del flujo de tu código Dart.
