Introducción

En esta sección, exploraremos cómo manejar redes y solicitudes HTTP en Flutter. Esto es crucial para construir aplicaciones que necesiten comunicarse con servicios web o APIs. Cubriremos los conceptos básicos de hacer solicitudes HTTP, manejar respuestas y analizar datos JSON. Al final de esta sección, serás capaz de integrar llamadas de red en tus aplicaciones Flutter.

Conceptos Clave

  • Métodos HTTP: GET, POST, PUT, DELETE
  • Programación Asíncrona: Futures, async, await
  • Análisis de JSON: Convertir datos JSON a objetos Dart y viceversa
  • Manejo de Errores: Gestionar errores de red y excepciones

Haciendo Solicitudes HTTP

Usando el paquete `http`

Flutter proporciona el paquete http para manejar solicitudes HTTP. Primero, agrega el paquete http a tu archivo pubspec.yaml:

dependencies:
  http: ^0.13.3

Luego, importa el paquete en tu archivo Dart:

import 'package:http/http.dart' as http;

Solicitud GET

Una solicitud GET se usa para recuperar datos de un servidor. Aquí hay un ejemplo de cómo hacer una solicitud GET:

import 'dart:convert';
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

  if (response.statusCode == 200) {
    List<dynamic> data = jsonDecode(response.body);
    print(data);
  } else {
    throw Exception('Failed to load data');
  }
}

Solicitud POST

Una solicitud POST se usa para enviar datos a un servidor. Aquí hay un ejemplo de cómo hacer una solicitud POST:

import 'dart:convert';
import 'package:http/http.dart' as http;

Future<void> sendData() async {
  final response = await http.post(
    Uri.parse('https://jsonplaceholder.typicode.com/posts'),
    headers: <String, String>{
      'Content-Type': 'application/json; charset=UTF-8',
    },
    body: jsonEncode(<String, String>{
      'title': 'foo',
      'body': 'bar',
      'userId': '1',
    }),
  );

  if (response.statusCode == 201) {
    print('Data sent successfully');
  } else {
    throw Exception('Failed to send data');
  }
}

Análisis de Datos JSON

Cuando recibes datos de un servidor, a menudo están en formato JSON. Necesitas analizar estos datos JSON en objetos Dart.

Ejemplo de Datos JSON

[
  {
    "userId": 1,
    "id": 1,
    "title": "Sample Title",
    "body": "Sample Body"
  }
]

Analizando JSON en Dart

Primero, crea una clase Dart para representar los datos:

class Post {
  final int userId;
  final int id;
  final String title;
  final String body;

  Post({required this.userId, required this.id, required this.title, required this.body});

  factory Post.fromJson(Map<String, dynamic> json) {
    return Post(
      userId: json['userId'],
      id: json['id'],
      title: json['title'],
      body: json['body'],
    );
  }
}

Luego, analiza los datos JSON:

Future<List<Post>> fetchPosts() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

  if (response.statusCode == 200) {
    List<dynamic> jsonData = jsonDecode(response.body);
    return jsonData.map((json) => Post.fromJson(json)).toList();
  } else {
    throw Exception('Failed to load posts');
  }
}

Manejo de Errores

Manejar errores es crucial para una experiencia de usuario fluida. Aquí hay un ejemplo de cómo manejar errores en tus solicitudes de red:

Future<void> fetchDataWithErrorHandling() async {
  try {
    final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

    if (response.statusCode == 200) {
      List<dynamic> data = jsonDecode(response.body);
      print(data);
    } else {
      print('Server error: ${response.statusCode}');
    }
  } catch (e) {
    print('Network error: $e');
  }
}

Conclusión

En esta sección, cubrimos los conceptos básicos de redes y solicitudes HTTP en Flutter. Aprendiste cómo hacer solicitudes GET y POST, analizar datos JSON y manejar errores. Estas habilidades son esenciales para construir aplicaciones Flutter robustas que interactúan con servicios web. Practica estos conceptos integrándolos en tus proyectos para solidificar tu comprensión.
