Publicar tu aplicación Flutter es el paso final en el proceso de desarrollo. Esto implica preparar tu aplicación para su lanzamiento, configurarla para diferentes plataformas y finalmente enviarla a las tiendas de aplicaciones. Esta guía te llevará a través de los pasos necesarios para publicar tu aplicación Flutter tanto para las plataformas Android como iOS.

Preparación para el lanzamiento

Antes de poder publicar tu aplicación, necesitas prepararla para el lanzamiento. Esto incluye tareas como actualizar la versión de la aplicación, configurar los iconos de la aplicación y configurar los permisos de la aplicación.

Actualizar la versión de la aplicación

  • Android: Actualiza el versionCode y versionName en el archivo android/app/build.gradle.
  • iOS: Actualiza el CFBundleShortVersionString y CFBundleVersion en el archivo ios/Runner/Info.plist.
// android/app/build.gradle
android {
    ...
    defaultConfig {
        ...
        versionCode 2
        versionName "1.1"
    }
}
<!-- ios/Runner/Info.plist -->
<key>CFBundleShortVersionString</key>
<string>1.1</string>
<key>CFBundleVersion</key>
<string>2</string>

Configurar los iconos de la aplicación

  • Android: Coloca tus iconos de aplicación en el directorio android/app/src/main/res.
  • iOS: Coloca tus iconos de aplicación en el directorio ios/Runner/Assets.xcassets/AppIcon.appiconset.

Configurar los permisos de la aplicación

  • Android: Actualiza el archivo AndroidManifest.xml para incluir los permisos necesarios.
  • iOS: Actualiza el archivo Info.plist para incluir los permisos necesarios.
<!-- android/app/src/main/AndroidManifest.xml -->
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.myapp">
    <uses-permission android:name="android.permission.INTERNET"/>
    ...
</manifest>
<!-- ios/Runner/Info.plist -->
<key>NSCameraUsageDescription</key>
<string>Necesitamos acceso a tu cámara para escanear códigos QR.</string>

Construir la aplicación

Una vez que tu aplicación esté preparada para el lanzamiento, necesitas construirla para las respectivas plataformas.

Construir para Android

  1. Abre una terminal y navega al directorio de tu proyecto Flutter.
  2. Ejecuta el siguiente comando para construir el APK:
flutter build apk --release
  1. El archivo APK generado estará ubicado en el directorio build/app/outputs/flutter-apk/.

Construir para iOS

  1. Abre una terminal y navega al directorio de tu proyecto Flutter.
  2. Ejecuta el siguiente comando para construir la aplicación iOS:
flutter build ios --release
  1. Abre el proyecto Xcode generado en ios/Runner.xcworkspace.
  2. En Xcode, selecciona un dispositivo de destino y haz clic en el menú "Product", luego elige "Archive".
  3. Sigue las indicaciones para crear un archivo y subirlo a la App Store.

Enviar a las tiendas de aplicaciones

Después de construir tu aplicación, el siguiente paso es enviarla a las respectivas tiendas de aplicaciones.

Enviar a Google Play Store

  1. Inicia sesión en tu Google Play Console.
  2. Crea una nueva aplicación y completa los detalles requeridos.
  3. Sube el archivo APK generado.
  4. Completa el cuestionario de clasificación de contenido.
  5. Configura los precios y la distribución.
  6. Revisa y publica tu aplicación.

Enviar a Apple App Store

  1. Inicia sesión en tu App Store Connect.
  2. Crea una nueva aplicación y completa los detalles requeridos.
  3. Sube la compilación usando Xcode o Transporter.
  4. Completa la información y los metadatos de la aplicación.
  5. Envía tu aplicación para revisión.
  6. Una vez aprobada, tu aplicación estará disponible en la App Store.

Conclusión

Publicar tu aplicación Flutter implica varios pasos, desde preparar tu aplicación para el lanzamiento hasta enviarla a las tiendas de aplicaciones. Siguiendo esta guía, puedes asegurarte de que tu aplicación esté correctamente configurada y lista para su distribución. Recuerda probar exhaustivamente tu aplicación antes de publicarla para ofrecer la mejor experiencia de usuario. ¡Feliz codificación y buena suerte con el lanzamiento de tu aplicación!
