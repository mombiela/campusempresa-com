¿Qué es Dart?

Dart es un lenguaje de programación optimizado para clientes que permite crear aplicaciones rápidas en cualquier plataforma. Es desarrollado por Google y se utiliza para construir aplicaciones móviles, de escritorio, servidor y web. Dart es el lenguaje detrás de Flutter, un popular framework para construir aplicaciones móviles multiplataforma.

Características Clave de Dart

  • Optimizado para UI: Dart está diseñado para crear interfaces de usuario fluidas y responsivas.
  • Desarrollo Productivo: Funciones como la recarga en caliente facilitan ver los cambios en tiempo real.
  • Rendimiento Rápido: Dart se compila a código máquina ARM y x64 para móviles, escritorio y backend.
  • Herramientas Flexibles: Dart soporta una amplia gama de herramientas y bibliotecas.

Configuración de Dart

Antes de comenzar a programar en Dart, necesitas configurar tu entorno de desarrollo.

Instalando Dart

  1. Descargar Dart SDK: Visita la página de Dart SDK y descarga el SDK para tu sistema operativo.
  2. Instalar Dart: Sigue las instrucciones de instalación para tu sistema operativo específico.
  3. Verificar la Instalación: Abre tu terminal o símbolo del sistema y escribe: 
    dart --version
    Deberías ver la versión de Dart instalada en tu máquina.

Sintaxis y Estructura Básica

Entender la sintaxis y estructura básica de Dart es crucial para escribir código efectivo.

Ejemplo de Hola Mundo

Comencemos con un simple programa de "Hola, Mundo!" para familiarizarnos con la sintaxis de Dart.

void main() {
  print('¡Hola, Mundo!');
}
  • void main(): El punto de entrada de cada aplicación Dart.
  • print(): Una función para mostrar texto en la consola.

Variables y Tipos de Datos

Dart es un lenguaje de tipado estático, lo que significa que necesitas especificar el tipo de una variable cuando la declaras.

Declarando Variables

void main() {
  int edad = 25;
  double altura = 5.9;
  String nombre = 'John Doe';
  bool esEstudiante = true;
}
  • int: Valores enteros.
  • double: Números de punto flotante.
  • String: Datos textuales.
  • bool: Valores booleanos (verdadero o falso).

Sentencias de Control de Flujo

Las sentencias de control de flujo te permiten gestionar el flujo de tu programa.

Sentencia If-Else

void main() {
  int puntuación = 85;
  if (puntuación >= 90) {
    print('Calificación: A');
  } else if (puntuación >= 80) {
    print('Calificación: B');
  } else {
    print('Calificación: C');
  }
}

Bucle For

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteración $i');
  }
}

Funciones

Las funciones son una parte central de Dart, permitiéndote encapsular código para su reutilización.

Definiendo una Función

void saludar(String nombre) {
  print('¡Hola, $nombre!');
}

void main() {
  saludar('Alice');
}
  • void: Indica que la función no devuelve un valor.
  • saludar(String nombre): Una función llamada saludar que toma un parámetro String.

Programación Orientada a Objetos en Dart

Dart es un lenguaje orientado a objetos, lo que significa que soporta conceptos como clases y objetos.

Clases y Objetos

Definiendo una Clase

class Persona {
  String nombre;
  int edad;

  Persona(this.nombre, this.edad);

  void presentarse() {
    print('Hola, soy $nombre y tengo $edad años.');
  }
}

void main() {
  Persona persona = Persona('Alice', 30);
  persona.presentarse();
}
  • class Persona: Define una clase llamada Persona.
  • Persona(this.nombre, this.edad): Un constructor que inicializa nombre y edad.
  • presentarse(): Un método que imprime un mensaje.

Conclusión

En esta introducción a Dart, hemos cubierto los conceptos básicos del lenguaje, incluyendo sus características clave, cómo configurar tu entorno, sintaxis básica, sentencias de control de flujo, funciones y conceptos de programación orientada a objetos. Entender estos fundamentos te proporcionará una base sólida mientras avanzas en temas más avanzados en el desarrollo con Dart y Flutter.
