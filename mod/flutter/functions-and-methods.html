Introducción

Las funciones y métodos son bloques de construcción fundamentales en cualquier lenguaje de programación, y Flutter no es una excepción. Entender cómo definir, usar y manipular funciones y métodos es crucial para construir aplicaciones Flutter eficientes y mantenibles.

Funciones en Dart

Las funciones en Dart son objetos de primera clase, lo que significa que pueden ser asignadas a variables, pasadas como argumentos y devueltas desde otras funciones.

Definiendo Funciones

Una función en Dart se define usando la palabra clave void si no devuelve un valor, o un tipo de retorno específico si lo hace.

// Función que devuelve una cadena
String greet(String name) {
  return 'Hello, $name!';
}

// Función que no devuelve un valor
void printGreeting(String name) {
  print('Hello, $name!');
}

Llamando a Funciones

Para llamar a una función, simplemente usa su nombre seguido de paréntesis.

void main() {
  String message = greet('Alice');
  print(message); // Salida: Hello, Alice!

  printGreeting('Bob'); // Salida: Hello, Bob!
}

Parámetros Opcionales

Dart soporta parámetros opcionales, que pueden ser posicionales o nombrados.

  • Parámetros Opcionales Posicionales: Encerrados entre corchetes [].
String greet([String name = 'Guest']) {
  return 'Hello, $name!';
}

void main() {
  print(greet()); // Salida: Hello, Guest!
  print(greet('Alice')); // Salida: Hello, Alice!
}
  • Parámetros Opcionales Nombrados: Encerrados entre llaves {} y pueden ser marcados como requeridos usando la anotación @required.
String greet({String name = 'Guest'}) {
  return 'Hello, $name!';
}

void main() {
  print(greet()); // Salida: Hello, Guest!
  print(greet(name: 'Alice')); // Salida: Hello, Alice!
}

Métodos en Dart

Los métodos son funciones que están asociadas con un objeto. En Dart, los métodos se definen dentro de clases.

Definiendo Métodos

Los métodos se definen de manera similar a las funciones pero dentro del contexto de una clase.

class Greeter {
  String name;

  Greeter(this.name);

  String greet() {
    return 'Hello, $name!';
  }

  void printGreeting() {
    print('Hello, $name!');
  }
}

Llamando a Métodos

Para llamar a un método, necesitas crear una instancia de la clase y luego usar la notación de punto.

void main() {
  Greeter greeter = Greeter('Alice');
  print(greeter.greet()); // Salida: Hello, Alice!
  greeter.printGreeting(); // Salida: Hello, Alice!
}

Métodos Estáticos

Los métodos estáticos pertenecen a la clase en sí misma en lugar de a una instancia de la clase. Se definen usando la palabra clave static.

class MathUtils {
  static int add(int a, int b) {
    return a + b;
  }
}

void main() {
  print(MathUtils.add(2, 3)); // Salida: 5
}

Funciones de Orden Superior

Dart soporta funciones de orden superior, que son funciones que toman otras funciones como parámetros o las devuelven.

void printResult(int a, int b, int Function(int, int) operation) {
  print(operation(a, b));
}

int add(int a, int b) => a + b;
int multiply(int a, int b) => a * b;

void main() {
  printResult(2, 3, add); // Salida: 5
  printResult(2, 3, multiply); // Salida: 6
}

Conclusión

Las funciones y métodos son esenciales para estructurar tu código y hacerlo reutilizable y modular. En Flutter, entender cómo definir y usar funciones y métodos te permitirá construir aplicaciones más complejas y eficientes. Ya sea que estés trabajando con funciones simples o métodos más complejos dentro de clases, dominar estos conceptos es crucial para cualquier desarrollador de Flutter.
