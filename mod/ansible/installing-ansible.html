Introducción

Ansible es una herramienta de automatización de código abierto utilizada para la gestión de configuraciones, despliegue de aplicaciones y automatización de tareas. Esta sección te guiará a través del proceso de instalación de Ansible en varios sistemas operativos. Al final de esta sección, tendrás Ansible instalado y listo para usar.

Requisitos previos

Antes de instalar Ansible, asegúrate de tener lo siguiente:

  • Un sistema operativo compatible (Linux, macOS o Windows Subsystem for Linux).
  • Python 3.8 o posterior instalado.
  • Privilegios administrativos para instalar software.

Instalación de Ansible en Linux

Usando el gestor de paquetes (Recomendado)

Ansible se puede instalar utilizando el gestor de paquetes específico de tu distribución de Linux.

Ubuntu/Debian

  1. Actualiza tu índice de paquetes: 
    sudo apt update
  2. Instala el paquete software-properties-common: 
    sudo apt install software-properties-common
  3. Añade el PPA (Personal Package Archive) de Ansible: 
    sudo add-apt-repository --yes --update ppa:ansible/ansible
  4. Instala Ansible: 
    sudo apt install ansible

CentOS/RHEL

  1. Instala el repositorio EPEL: 
    sudo yum install epel-release
  2. Instala Ansible: 
    sudo yum install ansible

Usando Pip (Instalador de paquetes de Python)

Si prefieres usar pip, sigue estos pasos:

  1. Asegúrate de que pip esté instalado: 
    sudo apt install python3-pip  # Para Ubuntu/Debian
sudo yum install python3-pip  # Para CentOS/RHEL
  2. Instala Ansible usando pip: 
    pip3 install ansible

Instalación de Ansible en macOS

Puedes instalar Ansible en macOS usando Homebrew, un gestor de paquetes popular para macOS.

  1. Instala Homebrew si aún no lo has hecho: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instala Ansible: 
    brew install ansible

Instalación de Ansible en Windows

Ansible no es compatible de forma nativa con Windows, pero puedes usar el Windows Subsystem for Linux (WSL) para ejecutar una distribución de Linux e instalar Ansible.

  1. Habilita WSL e instala una distribución de Linux desde la Microsoft Store (por ejemplo, Ubuntu).
  2. Abre la distribución de Linux instalada y actualiza el índice de paquetes: 
    sudo apt update
  3. Instala Ansible usando los mismos pasos que para Ubuntu/Debian: 
    sudo apt install software-properties-common
sudo add-apt-repository --yes --update ppa:ansible/ansible
sudo apt install ansible

Verificación de la instalación

Después de instalar Ansible, verifica la instalación comprobando la versión:

ansible --version

Deberías ver una salida similar a:

ansible [core 2.10.7]
  config file = /etc/ansible/ansible.cfg
  configured module search path = ['/home/user/.ansible/plugins/modules', '/usr/share/ansible/plugins/modules']
  ansible python module location = /usr/local/lib/python3.8/dist-packages/ansible
  executable location = /usr/local/bin/ansible
  python version = 3.8.5 (default, Jul 28 2020, 12:59:40) [GCC 9.3.0]

Conclusión

En esta sección, aprendiste cómo instalar Ansible en varios sistemas operativos, incluyendo Linux, macOS y Windows (a través de WSL). Asegurarte de que Ansible esté correctamente instalado es el primer paso para aprovechar sus poderosas capacidades de automatización. En las siguientes secciones, exploraremos cómo configurar y usar Ansible para diferentes tareas de automatización.
