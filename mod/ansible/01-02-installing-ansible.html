Instalando Ansible
Siguiente ►

En esta sección, aprenderás cómo instalar Ansible en tu sistema. Ansible es una herramienta de automatización de TI que se puede instalar en varias plataformas. A continuación, cubriremos los pasos para instalar Ansible en sistemas basados en Linux, macOS y Windows.

Requisitos Previos

Antes de comenzar con la instalación, asegúrate de tener lo siguiente:

  • Acceso a una terminal o línea de comandos.
  • Privilegios de superusuario (root) o acceso a sudo.
  • Conexión a Internet para descargar los paquetes necesarios.

Instalación en Sistemas Basados en Linux

Instalación en Ubuntu/Debian

  1. Actualizar el sistema:

    sudo apt update
sudo apt upgrade

  2. Instalar Ansible:

    sudo apt install ansible

  3. Verificar la instalación:

    ansible --version

Instalación en CentOS/RHEL

  1. Actualizar el sistema:

    sudo yum update

  2. Agregar el repositorio EPEL:

    sudo yum install epel-release

  3. Instalar Ansible:

    sudo yum install ansible

  4. Verificar la instalación:

    ansible --version

Instalación en Fedora

  1. Actualizar el sistema:

    sudo dnf update

  2. Instalar Ansible:

    sudo dnf install ansible

  3. Verificar la instalación:

    ansible --version

Instalación en macOS

Usando Homebrew

  1. Instalar Homebrew (si no está instalado):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Actualizar Homebrew:

    brew update

  3. Instalar Ansible:

    brew install ansible

  4. Verificar la instalación:

    ansible --version

Instalación en Windows

Usando WSL (Windows Subsystem for Linux)

  1. Instalar WSL:

    • Abre PowerShell como administrador y ejecuta: 
      wsl --install

  2. **Instalar una distribución de Linux (por ejemplo, Ubuntu) desde Microsoft Store.

  3. Abrir la terminal de WSL y actualizar el sistema:

    sudo apt update
sudo apt upgrade

  4. Instalar Ansible:

    sudo apt install ansible

  5. Verificar la instalación:

    ansible --version

Usando Cygwin

  1. Descargar e instalar Cygwin desde cygwin.com.

  2. Durante la instalación, selecciona los paquetes necesarios:

    • python3
    • python3-pip
    • openssh

  3. Instalar Ansible usando pip:

    pip3 install ansible

  4. Verificar la instalación:

    ansible --version

Solución de Problemas Comunes

Error: "ansible: command not found"

  • Asegúrate de que Ansible esté instalado correctamente.
  • Verifica que el directorio de binarios de Ansible esté en tu PATH.

Error: "Permission denied"

  • Asegúrate de tener los permisos necesarios para ejecutar los comandos.
  • Usa sudo si es necesario.

Conclusión

En esta sección, hemos cubierto cómo instalar Ansible en diferentes sistemas operativos. Ahora que tienes Ansible instalado, estás listo para comenzar a explorar sus capacidades. En la siguiente sección, aprenderemos sobre la arquitectura de Ansible y cómo funciona.

¡Felicidades por completar esta sección! Asegúrate de verificar la instalación ejecutando ansible --version y prepárate para sumergirte en la arquitectura de Ansible en el próximo módulo.
