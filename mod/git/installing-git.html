Instalando Git
Terminología Básica de Git ►

Git es un poderoso sistema de control de versiones que es esencial para el desarrollo de software moderno. Esta sección te guiará a través del proceso de instalación de Git en varios sistemas operativos. Al final de esta sección, tendrás Git instalado y listo para usar en tu máquina.

Instalando Git en Windows

  1. Descargar el Instalador de Git:

    • Visita el sitio web oficial de Git: Git para Windows.
    • Haz clic en el botón "Download" para obtener la última versión del instalador de Git.

  2. Ejecutar el Instalador:

    • Localiza el archivo .exe descargado y haz doble clic para ejecutarlo.
    • Sigue los pasos del asistente de instalación. La configuración predeterminada es generalmente adecuada para la mayoría de los usuarios.

  3. Configurar Git:

    • Después de la instalación, abre la terminal de Git Bash desde el menú de Inicio.
    • Configura tu nombre de usuario y correo electrónico: 
      git config --global user.name "Tu Nombre"
git config --global user.email "tu.correo@example.com"

Instalando Git en macOS

  1. Usando Homebrew (Recomendado):

    • Si no tienes Homebrew instalado, puedes instalarlo ejecutando el siguiente comando en tu terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Una vez que Homebrew esté instalado, instala Git ejecutando: 
      brew install git

  2. Usando el Instalador de Git:

    • Visita el sitio web oficial de Git: Git para macOS.
    • Descarga y ejecuta el paquete del instalador.

  3. Configurar Git:

    • Abre la aplicación Terminal.
    • Configura tu nombre de usuario y correo electrónico: 
      git config --global user.name "Tu Nombre"
git config --global user.email "tu.correo@example.com"

Instalando Git en Linux

  1. Usando el Gestor de Paquetes:

    • Para distribuciones basadas en Debian (por ejemplo, Ubuntu): 
      sudo apt update
sudo apt install git
    • Para distribuciones basadas en RPM (por ejemplo, Fedora): 
      sudo dnf install git

  2. Configurar Git:

    • Abre tu terminal.
    • Configura tu nombre de usuario y correo electrónico: 
      git config --global user.name "Tu Nombre"
git config --global user.email "tu.correo@example.com"

Verificando la Instalación

Después de instalar Git, deberías verificar que se haya instalado correctamente. Abre tu terminal o Git Bash y ejecuta el siguiente comando:

git --version

Deberías ver una salida similar a:

git version 2.x.x

Esto confirma que Git está instalado y listo para usar.

Resumen

En esta sección, aprendiste cómo instalar Git en Windows, macOS y Linux. También configuraste Git con tu nombre de usuario y correo electrónico, lo cual es esencial para rastrear cambios y contribuciones. Ahora que Git está instalado, estás listo para comenzar a usarlo para el control de versiones en tus proyectos.
