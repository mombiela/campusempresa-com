Validación de Datos con Reglas Personalizadas
Siguiente ►

La validación de datos en Excel es una herramienta poderosa que permite controlar el tipo de datos o los valores que los usuarios pueden ingresar en una celda. En esta lección, aprenderemos a crear reglas de validación de datos personalizadas para asegurar la integridad y precisión de nuestros datos.

Objetivos de Aprendizaje

  • Comprender la importancia de la validación de datos.
  • Aprender a crear reglas de validación de datos personalizadas.
  • Aplicar fórmulas para validar datos de manera avanzada.
  • Gestionar mensajes de entrada y alertas de error.

Conceptos Clave

¿Qué es la Validación de Datos?

La validación de datos es una característica de Excel que permite definir restricciones sobre los datos que se pueden ingresar en una celda. Esto ayuda a mantener la integridad de los datos y a evitar errores comunes.

Tipos de Validación de Datos

  • Predefinida: Números enteros, decimales, listas, fechas, horas, longitud de texto.
  • Personalizada: Utiliza fórmulas para definir reglas específicas.

Creando Reglas de Validación de Datos Personalizadas

Paso 1: Seleccionar las Celdas

Selecciona las celdas donde deseas aplicar la validación de datos.

Paso 2: Abrir la Herramienta de Validación de Datos

  1. Ve a la pestaña Datos en la cinta de opciones.
  2. Haz clic en Validación de datos en el grupo Herramientas de datos.

Paso 3: Configurar la Validación de Datos

  1. En el cuadro de diálogo Validación de datos, selecciona la pestaña Configuración.
  2. En el campo Permitir, selecciona Personalizada.
  3. En el campo Fórmula, ingresa la fórmula que define tu regla de validación.

Ejemplo Práctico: Validar que una Celda Contenga un Valor Mayor a 10

= A1 > 10

Esta fórmula asegura que el valor ingresado en la celda A1 sea mayor a 10.

Ejemplo Práctico: Validar que una Celda Contenga un Texto Específico

= A1 = "Aprobado"

Esta fórmula asegura que el valor ingresado en la celda A1 sea exactamente "Aprobado".

Ejemplo Práctico: Validar que una Fecha Sea Posterior a Hoy

= A1 > HOY()

Esta fórmula asegura que la fecha ingresada en la celda A1 sea posterior a la fecha actual.

Mensajes de Entrada y Alertas de Error

Configurar Mensajes de Entrada

  1. En el cuadro de diálogo Validación de datos, selecciona la pestaña Mensaje de entrada.
  2. Marca la casilla Mostrar mensaje de entrada al seleccionar la celda.
  3. Ingresa un título y un mensaje que se mostrará cuando el usuario seleccione la celda.

Configurar Alertas de Error

  1. En el cuadro de diálogo Validación de datos, selecciona la pestaña Alerta de error.
  2. Marca la casilla Mostrar alerta de error después de ingresar datos no válidos.
  3. Selecciona el estilo de alerta (Detener, Advertencia, Información).
  4. Ingresa un título y un mensaje que se mostrará cuando el usuario ingrese datos no válidos.

Ejercicio Práctico

Ejercicio 1: Validar Números en un Rango Específico

Crea una regla de validación de datos que solo permita números entre 1 y 100 en la columna B.

Solución:

  1. Selecciona la columna B.
  2. Abre la herramienta de Validación de datos.
  3. En Permitir, selecciona Personalizada.
  4. En Fórmula, ingresa: 
    = Y(B1 >= 1, B1 <= 100)
  5. Configura un mensaje de entrada y una alerta de error si lo deseas.

Ejercicio 2: Validar Correos Electrónicos

Crea una regla de validación de datos que solo permita direcciones de correo electrónico válidas en la columna C.

Solución:

  1. Selecciona la columna C.
  2. Abre la herramienta de Validación de datos.
  3. En Permitir, selecciona Personalizada.
  4. En Fórmula, ingresa: 
    = Y(ESNUMERO(HALLAR("@", C1)), ESNUMERO(HALLAR(".", C1)))
  5. Configura un mensaje de entrada y una alerta de error si lo deseas.

Resumen

En esta lección, hemos aprendido a utilizar la validación de datos con reglas personalizadas en Excel. Esta herramienta es esencial para mantener la integridad de los datos y evitar errores comunes. Hemos visto cómo aplicar fórmulas para crear reglas específicas y cómo configurar mensajes de entrada y alertas de error para guiar a los usuarios.
