Validación de Datos

Siguiente ►
►

La validación de datos en Excel es una herramienta poderosa que permite controlar el tipo de datos que los usuarios pueden ingresar en una celda. Esto es especialmente útil para mantener la integridad de los datos y evitar errores comunes. En esta lección, aprenderás cómo configurar reglas de validación de datos y cómo utilizarlas para mejorar la calidad de tus hojas de cálculo.

Conceptos Clave

  1. Validación de Datos: Proceso de definir reglas que los datos deben cumplir para ser ingresados en una celda.
  2. Tipos de Validación: Números enteros, decimales, listas, fechas, horas, longitud de texto, y fórmulas personalizadas.
  3. Mensajes de Entrada y de Error: Mensajes que guían al usuario sobre qué tipo de datos ingresar y qué hacer si se ingresa un dato incorrecto.

Configuración de la Validación de Datos

Paso 1: Seleccionar las Celdas

Primero, selecciona las celdas donde deseas aplicar la validación de datos.

Paso 2: Abrir la Herramienta de Validación de Datos

  1. Ve a la pestaña Datos en la cinta de opciones.
  2. Haz clic en Validación de datos en el grupo Herramientas de datos.

Paso 3: Configurar las Reglas de Validación

En el cuadro de diálogo de Validación de Datos, puedes configurar las reglas según el tipo de datos que deseas permitir.

Ejemplo: Validación de Números Enteros

  1. En la pestaña Configuración, selecciona Número entero en el menú desplegable Permitir.
  2. Define las condiciones, por ejemplo, entre 1 y 100.
  3. Opcionalmente, puedes agregar un Mensaje de entrada y un Mensaje de error en las pestañas correspondientes.
Configuración:
Permitir: Número entero
Datos: entre
Mínimo: 1
Máximo: 100

Mensaje de entrada:
Título: Ingrese un número
Mensaje: Por favor, ingrese un número entre 1 y 100.

Mensaje de error:
Título: Error de entrada
Mensaje: El valor ingresado no es un número válido entre 1 y 100.

Paso 4: Probar la Validación

Intenta ingresar diferentes valores en las celdas seleccionadas para asegurarte de que la validación funciona correctamente.

Ejemplos Prácticos

Ejemplo 1: Validación de Lista

Supongamos que deseas que los usuarios solo puedan seleccionar un valor de una lista predefinida.

  1. Selecciona las celdas donde deseas aplicar la validación.
  2. Abre la herramienta de Validación de Datos.
  3. En la pestaña Configuración, selecciona Lista en el menú desplegable Permitir.
  4. En el campo Origen, ingresa los valores de la lista separados por comas (por ejemplo, "Sí,No,Quizás").
Configuración:
Permitir: Lista
Origen: Sí,No,Quizás

Mensaje de entrada:
Título: Seleccione una opción
Mensaje: Por favor, seleccione una opción de la lista.

Mensaje de error:
Título: Error de entrada
Mensaje: El valor ingresado no es una opción válida.

Ejemplo 2: Validación de Fecha

Para asegurarte de que los usuarios ingresen una fecha dentro de un rango específico:

  1. Selecciona las celdas donde deseas aplicar la validación.
  2. Abre la herramienta de Validación de Datos.
  3. En la pestaña Configuración, selecciona Fecha en el menú desplegable Permitir.
  4. Define las condiciones, por ejemplo, entre 01/01/2023 y 31/12/2023.
Configuración:
Permitir: Fecha
Datos: entre
Fecha de inicio: 01/01/2023
Fecha de finalización: 31/12/2023

Mensaje de entrada:
Título: Ingrese una fecha
Mensaje: Por favor, ingrese una fecha entre el 01/01/2023 y el 31/12/2023.

Mensaje de error:
Título: Error de entrada
Mensaje: La fecha ingresada no está dentro del rango permitido.

Ejercicio Práctico

Ejercicio 1: Validación de Números Enteros

  1. Crea una nueva hoja de cálculo.
  2. Selecciona el rango de celdas A1:A10.
  3. Configura una validación de datos que solo permita números enteros entre 10 y 50.
  4. Agrega un mensaje de entrada y un mensaje de error apropiados.

Solución

Configuración:
Permitir: Número entero
Datos: entre
Mínimo: 10
Máximo: 50

Mensaje de entrada:
Título: Ingrese un número
Mensaje: Por favor, ingrese un número entre 10 y 50.

Mensaje de error:
Título: Error de entrada
Mensaje: El valor ingresado no es un número válido entre 10 y 50.

Ejercicio 2: Validación de Lista

  1. Selecciona el rango de celdas B1:B10.
  2. Configura una validación de datos que solo permita seleccionar entre "Aprobado", "Reprobado" y "Pendiente".
  3. Agrega un mensaje de entrada y un mensaje de error apropiados.

Solución

Configuración:
Permitir: Lista
Origen: Aprobado,Reprobado,Pendiente

Mensaje de entrada:
Título: Seleccione una opción
Mensaje: Por favor, seleccione una opción de la lista.

Mensaje de error:
Título: Error de entrada
Mensaje: El valor ingresado no es una opción válida.

Conclusión

La validación de datos es una herramienta esencial para mantener la integridad y la calidad de los datos en tus hojas de cálculo de Excel. Al aprender a configurar y utilizar reglas de validación, puedes evitar errores comunes y asegurarte de que los datos ingresados cumplan con los criterios específicos que has definido. En el próximo módulo, exploraremos cómo trabajar con múltiples hojas de trabajo para organizar y gestionar mejor tus datos.
