Análisis de Suposiciones (Administrador de Escenarios, Buscar Objetivo)
Siguiente ►

El análisis de suposiciones en Excel es una herramienta poderosa que permite a los usuarios explorar diferentes escenarios y determinar cómo los cambios en los datos pueden afectar los resultados. En esta sección, aprenderemos a utilizar dos herramientas clave para el análisis de suposiciones: el Administrador de Escenarios y la función Buscar Objetivo.

  1. Administrador de Escenarios

El Administrador de Escenarios permite crear y guardar diferentes conjuntos de valores de entrada y luego comparar los resultados. Esto es útil para realizar análisis de "qué pasaría si" y tomar decisiones informadas.

Pasos para usar el Administrador de Escenarios:

  1. Abrir el Administrador de Escenarios:

    • Ve a la pestaña Datos.
    • En el grupo Herramientas de datos, haz clic en Análisis de hipótesis y selecciona Administrador de escenarios.

  2. Crear un nuevo escenario:

    • En el cuadro de diálogo del Administrador de Escenarios, haz clic en Agregar.
    • Asigna un nombre al escenario.
    • Selecciona las celdas que cambiarán en este escenario (celdas de entrada).
    • Introduce los valores para estas celdas en el nuevo escenario.
    • Haz clic en Aceptar y luego en Agregar para crear más escenarios, o en Cerrar si has terminado.

  3. Mostrar un escenario:

    • En el Administrador de Escenarios, selecciona el escenario que deseas ver y haz clic en Mostrar.

Ejemplo Práctico:

Supongamos que tienes una hoja de cálculo que calcula las ganancias basadas en el precio de venta y el número de unidades vendidas. Puedes crear diferentes escenarios para ver cómo cambiarían las ganancias si cambias el precio de venta o el número de unidades vendidas.

| A            | B       | C       |
|--------------|---------|---------|
| Precio Venta | 10      |         |
| Unidades     | 100     |         |
| Ganancias    | =A2*B2  |         |
  1. Abre el Administrador de Escenarios y crea un escenario llamado "Escenario Optimista" con un precio de venta de 12 y 120 unidades.
  2. Crea otro escenario llamado "Escenario Pesimista" con un precio de venta de 8 y 80 unidades.
  3. Muestra cada escenario para ver cómo cambian las ganancias.

  1. Buscar Objetivo

La función Buscar Objetivo permite encontrar el valor de entrada necesario para alcanzar un resultado específico. Es útil cuando sabes el resultado que deseas obtener, pero no estás seguro de qué valor de entrada lo producirá.

Pasos para usar Buscar Objetivo:

  1. Abrir Buscar Objetivo:

    • Ve a la pestaña Datos.
    • En el grupo Herramientas de datos, haz clic en Análisis de hipótesis y selecciona Buscar objetivo.

  2. Configurar Buscar Objetivo:

    • En el cuadro de diálogo Buscar Objetivo, selecciona la celda que contiene la fórmula cuyo resultado deseas cambiar.
    • Introduce el valor objetivo que deseas alcanzar.
    • Selecciona la celda de entrada que Excel debe ajustar para alcanzar el valor objetivo.
    • Haz clic en Aceptar.

Ejemplo Práctico:

Supongamos que deseas saber cuántas unidades necesitas vender para alcanzar una ganancia de $1500, dado un precio de venta de $10 por unidad.

| A            | B       | C       |
|--------------|---------|---------|
| Precio Venta | 10      |         |
| Unidades     | 100     |         |
| Ganancias    | =A2*B2  |         |
  1. Abre Buscar Objetivo y selecciona la celda C3 (Ganancias).
  2. Introduce 1500 como el valor objetivo.
  3. Selecciona la celda B2 (Unidades) como la celda de entrada.
  4. Haz clic en Aceptar para ver cuántas unidades necesitas vender para alcanzar la ganancia deseada.

Ejercicios Prácticos

Ejercicio 1: Administrador de Escenarios

  1. Crea una hoja de cálculo con los siguientes datos:

    • Precio de venta: $15
    • Unidades vendidas: 200
    • Ganancias: =A2*B2

  2. Usa el Administrador de Escenarios para crear dos escenarios:

    • Escenario A: Precio de venta $18, Unidades vendidas 220.
    • Escenario B: Precio de venta $12, Unidades vendidas 180.

  3. Muestra cada escenario y observa cómo cambian las ganancias.

Ejercicio 2: Buscar Objetivo

  1. Usa la misma hoja de cálculo del ejercicio anterior.
  2. Utiliza Buscar Objetivo para determinar cuántas unidades necesitas vender para alcanzar una ganancia de $3000, dado un precio de venta de $15.

Soluciones

Solución Ejercicio 1:

  1. Escenario A:

    • Precio de venta: $18
    • Unidades vendidas: 220
    • Ganancias: $3960

  2. Escenario B:

    • Precio de venta: $12
    • Unidades vendidas: 180
    • Ganancias: $2160

Solución Ejercicio 2:

  • Para alcanzar una ganancia de $3000 con un precio de venta de $15, necesitas vender 200 unidades.

Conclusión

El análisis de suposiciones con el Administrador de Escenarios y Buscar Objetivo en Excel es una herramienta esencial para la toma de decisiones informadas. Permite explorar diferentes escenarios y determinar los valores de entrada necesarios para alcanzar objetivos específicos. Con la práctica, estas herramientas pueden mejorar significativamente tu capacidad para analizar y planificar en Excel.
