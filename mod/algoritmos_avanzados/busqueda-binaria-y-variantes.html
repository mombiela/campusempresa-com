Introducción a la Búsqueda Binaria

La búsqueda binaria es un algoritmo eficiente para encontrar un elemento en una lista ordenada. Su eficiencia radica en dividir repetidamente el rango de búsqueda a la mitad, lo que reduce significativamente el número de comparaciones necesarias.

  • Propósito: Encontrar la posición de un elemento en una lista ordenada.
  • Complejidad Temporal: O(log n), donde n es el número de elementos en la lista.
  • Requisitos: La lista debe estar ordenada previamente.

Algoritmo Básico de Búsqueda Binaria

El algoritmo de búsqueda binaria sigue estos pasos:

  1. Inicializar dos índices, low y high, que representan los límites del rango de búsqueda.
  2. Calcular el índice medio (mid) del rango actual.
  3. Comparar el elemento en mid con el valor buscado (target).
  4. Si el target es igual al elemento en mid, retornar mid.
  5. Si el target es menor que el elemento en mid, ajustar high a mid - 1.
  6. Si el target es mayor que el elemento en mid, ajustar low a mid + 1.
  7. Repetir los pasos 2-6 hasta que low sea mayor que high.

Ejemplo de Código en Python

def busqueda_binaria(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1  # Elemento no encontrado

# Ejemplo de uso
arr = [1, 3, 5, 7, 9, 11]
target = 7
resultado = busqueda_binaria(arr, target)
print(f'Elemento encontrado en el índice: {resultado}')

Variantes de la Búsqueda Binaria

Búsqueda Binaria Recursiva

La búsqueda binaria también puede implementarse de manera recursiva. La lógica es similar a la iterativa, pero se utiliza una función que se llama a sí misma con parámetros actualizados.

Ejemplo de Código en Python

def busqueda_binaria_recursiva(arr, target, low, high):
    if low > high:
        return -1  # Elemento no encontrado

    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return busqueda_binaria_recursiva(arr, target, mid + 1, high)
    else:
        return busqueda_binaria_recursiva(arr, target, low, mid - 1)

# Ejemplo de uso
arr = [1, 3, 5, 7, 9, 11]
target = 7
resultado = busqueda_binaria_recursiva(arr, target, 0, len(arr) - 1)
print(f'Elemento encontrado en el índice: {resultado}')

Búsqueda Binaria para el Primer y Último Elemento

En algunos casos, es necesario encontrar la primera o la última aparición de un elemento en una lista ordenada que puede contener elementos duplicados.

Búsqueda del Primer Elemento

def busqueda_binaria_primero(arr, target):
    low = 0
    high = len(arr) - 1
    resultado = -1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            resultado = mid
            high = mid - 1  # Seguir buscando en la mitad izquierda
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return resultado

# Ejemplo de uso
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
resultado = busqueda_binaria_primero(arr, target)
print(f'Primera aparición del elemento en el índice: {resultado}')

Búsqueda del Último Elemento

def busqueda_binaria_ultimo(arr, target):
    low = 0
    high = len(arr) - 1
    resultado = -1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            resultado = mid
            low = mid + 1  # Seguir buscando en la mitad derecha
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return resultado

# Ejemplo de uso
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
resultado = busqueda_binaria_ultimo(arr, target)
print(f'Última aparición del elemento en el índice: {resultado}')

Conclusión

La búsqueda binaria es un algoritmo fundamental en la informática debido a su eficiencia y simplicidad. Comprender tanto la implementación iterativa como la recursiva, así como sus variantes para encontrar el primer y último elemento, es crucial para resolver problemas de búsqueda en listas ordenadas. Practicar estos algoritmos con diferentes tipos de datos y escenarios ayudará a consolidar su comprensión y aplicación en situaciones del mundo real.
