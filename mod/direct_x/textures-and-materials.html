Introducción

En esta sección, exploraremos cómo usar texturas y materiales en DirectX para mejorar la calidad visual de tus aplicaciones 3D. Las texturas añaden detalle a las superficies de tus modelos 3D, mientras que los materiales definen cómo estas superficies interactúan con la luz.

Texturas

Las texturas son imágenes aplicadas a las superficies de los modelos 3D para darles más detalle y realismo.

Cargando Texturas

Para cargar una textura en DirectX, típicamente usas la DirectX Tool Kit o la biblioteca DirectXTex. Aquí hay un ejemplo básico de cómo cargar una textura usando la DirectX Tool Kit:

#include "DDSTextureLoader.h"

// Asumimos que device es un puntero válido a ID3D11Device
ID3D11ShaderResourceView* textureView = nullptr;
HRESULT hr = DirectX::CreateDDSTextureFromFile(device, L"texture.dds", nullptr, &textureView);
if (FAILED(hr)) {
    // Manejar error
}

Aplicando Texturas

Una vez cargada, necesitas enlazar la textura al pipeline. Esto se hace configurando la vista de recurso del shader a la etapa del pixel shader.

// Asumimos que context es un puntero válido a ID3D11DeviceContext
context->PSSetShaderResources(0, 1, &textureView);

Coordenadas de Textura

Las coordenadas de textura (coordenadas UV) mapean la textura al modelo 3D. Van desde (0,0) hasta (1,1).

struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT2 texCoord; // Coordenadas de textura
};

Materiales

Los materiales definen cómo las superficies interactúan con la luz, incluyendo propiedades como color, brillo y reflectividad.

Propiedades Básicas del Material

  • Color Difuso: El color base del material.
  • Color Especular: El color de los reflejos.
  • Brillo: Controla el tamaño de los reflejos especulares.

Definiendo Materiales

Los materiales típicamente se definen en una estructura y se pasan a los shaders.

struct Material {
    DirectX::XMFLOAT4 diffuseColor;
    DirectX::XMFLOAT4 specularColor;
    float shininess;
};

Usando Materiales en Shaders

En tu shader HLSL, puedes definir una estructura de material y usarla para calcular el color final de un píxel.

cbuffer MaterialBuffer : register(b0) {
    float4 diffuseColor;
    float4 specularColor;
    float shininess;
};

float4 PSMain(VertexOut input) : SV_TARGET {
    // Calcular iluminación aquí
    float4 color = diffuseColor; // Ejemplo simplificado
    return color;
}

Combinando Texturas y Materiales

Para lograr una renderización realista, a menudo combinas texturas y materiales. Por ejemplo, podrías usar una textura para el color difuso y un material para las propiedades especulares.

Ejemplo: Superficie Texturizada y Brillante

Aquí hay un ejemplo de cómo combinar una textura con propiedades de material en un pixel shader:

Texture2D diffuseTexture : register(t0);
SamplerState samplerState : register(s0);

cbuffer MaterialBuffer : register(b0) {
    float4 specularColor;
    float shininess;
};

float4 PSMain(VertexOut input) : SV_TARGET {
    float4 diffuseColor = diffuseTexture.Sample(samplerState, input.texCoord);
    // Calcular iluminación aquí
    float4 finalColor = diffuseColor; // Ejemplo simplificado
    return finalColor;
}

Conclusión

En esta sección, cubrimos los conceptos básicos de texturas y materiales en DirectX. Aprendiste cómo cargar y aplicar texturas, definir propiedades de materiales y combinarlas para crear superficies realistas. Dominar estos conceptos mejorará significativamente la calidad visual de tus aplicaciones 3D. Continúa practicando experimentando con diferentes texturas y propiedades de materiales para ver sus efectos en tus modelos.
