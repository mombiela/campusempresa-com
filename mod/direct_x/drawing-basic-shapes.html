Introducción

En esta sección, exploraremos cómo dibujar formas básicas usando DirectX. Esto es fundamental para cualquiera que desee crear aplicaciones gráficas o juegos. Comenzaremos con formas simples como líneas y rectángulos y gradualmente avanzaremos hacia formas más complejas.

Configuración de DirectX

Antes de comenzar a dibujar formas, asegúrate de tener DirectX configurado en tu entorno de desarrollo. Si aún no lo has hecho, consulta la guía de configuración proporcionada en la introducción del curso.

Dibujo de una Línea

Dibujar una línea es una de las tareas más simples en DirectX. Usaremos el método ID3D11DeviceContext::Draw para renderizar una línea.

Conceptos Clave

  • Buffer de Vértices: Almacena los vértices de la forma.
  • Disposición de Entrada: Describe el formato del vértice a la GPU.
  • Shaders: Programas que se ejecutan en la GPU para procesar vértices y píxeles.

Ejemplo de Código

// Definir los vértices para una línea
struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT4 color;
};

Vertex vertices[] = {
    { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(1.0f, 1.0f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) }
};

// Crear el buffer de vértices
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Establecer el buffer de vértices
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Dibujar la línea
context->Draw(2, 0);

Explicación

  • Estructura del Vértice: Define la posición y el color de cada vértice.
  • Creación del Buffer de Vértices: Asigna memoria en la GPU y copia los datos del vértice.
  • Establecimiento del Buffer de Vértices: Vincula el buffer de vértices a la etapa del ensamblador de entrada.
  • Dibujo: Emite una llamada de dibujo para renderizar la línea.

Dibujo de un Rectángulo

Dibujar un rectángulo implica definir cuatro vértices y dos triángulos.

Conceptos Clave

  • Buffer de Índices: Especifica el orden en que se utilizan los vértices para formar triángulos.

Ejemplo de Código

// Definir los vértices para un rectángulo
Vertex vertices[] = {
    { DirectX::XMFLOAT3(-0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(-0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 0.0f, 1.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 0.0f, 1.0f) }
};

// Definir los índices para el rectángulo
unsigned short indices[] = {
    0, 1, 2,
    0, 2, 3
};

// Crear el buffer de vértices
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Crear el buffer de índices
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(indices);
bufferDesc.BindFlags = D3D11_BIND_INDEX_BUFFER;

initData.pSysMem = indices;

ID3D11Buffer* indexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &indexBuffer);

// Establecer el buffer de vértices
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Establecer el buffer de índices
context->IASetIndexBuffer(indexBuffer, DXGI_FORMAT_R16_UINT, 0);

// Dibujar el rectángulo
context->DrawIndexed(6, 0, 0);

Explicación

  • Vértices e Índices: Define la forma del rectángulo y cómo conectar los vértices.
  • Creación del Buffer de Índices: Asigna memoria en la GPU y copia los datos del índice.
  • Establecimiento de Buffers: Vincula los buffers de vértices e índices a la etapa del ensamblador de entrada.
  • Dibujo: Emite una llamada de dibujo para renderizar el rectángulo usando los índices.

Dibujo de un Círculo

Dibujar un círculo es más complejo e implica aproximar el círculo con un polígono.

Conceptos Clave

  • Aproximación del Círculo: Usar un polígono con muchos lados para aproximar un círculo.

Ejemplo de Código

#include <vector>
#include <cmath>

// Definir el número de segmentos
const int numSegments = 36;
const float radius = 1.0f;
std::vector<Vertex> vertices(numSegments + 1);

// Vértice central
vertices[0] = { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 1.0f, 1.0f) };

// Definir los vértices para el círculo
for (int i = 1; i <= numSegments; ++i) {
    float angle = i * (2.0f * DirectX::XM_PI / numSegments);
    vertices[i] = { DirectX::XMFLOAT3(radius * cosf(angle), radius * sinf(angle), 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) };
}

// Crear el buffer de vértices
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(Vertex) * vertices.size();
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices.data();

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Establecer el buffer de vértices
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Dibujar el círculo
context->Draw(vertices.size(), 0);

Explicación

  • Vértices del Círculo: Calcula los vértices alrededor de la circunferencia del círculo.
  • Creación del Buffer de Vértices: Asigna memoria en la GPU y copia los datos del vértice.
  • Establecimiento del Buffer de Vértices: Vincula el buffer de vértices a la etapa del ensamblador de entrada.
  • Dibujo: Emite una llamada de dibujo para renderizar el círculo.

Conclusión

Dibujar formas básicas en DirectX implica entender los buffers de vértices, los buffers de índices y cómo emitir llamadas de dibujo. Al dominar estos conceptos, puedes crear formas más complejas y eventualmente construir aplicaciones gráficas sofisticadas. Practica experimentando con diferentes formas y configuraciones para profundizar tu comprensión.
