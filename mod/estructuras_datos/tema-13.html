Introducción a las Pilas

Las pilas son una estructura de datos fundamental en informática que sigue el principio de LIFO (Last In, First Out). Esto significa que el último elemento en ser añadido a la pila será el primero en ser removido. Las operaciones básicas de una pila son:

  • push: Añadir un elemento al tope de la pila.
  • pop: Remover el elemento del tope de la pila.
  • peek: Ver el elemento en el tope de la pila sin removerlo.
  • isEmpty: Verificar si la pila está vacía.

Ejercicio 1: Implementación Básica de una Pila

Objetivo

Implementar una pila utilizando una lista en Python y realizar las operaciones básicas.

Instrucciones

  1. Crear una clase Pila.
  2. Implementar los métodos push, pop, peek e isEmpty.
  3. Probar la funcionalidad de la pila con algunos ejemplos.

Ejemplo de Código

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.isEmpty():
            return self.items.pop()
        else:
            return "La pila está vacía"

    def peek(self):
        if not self.isEmpty():
            return self.items[-1]
        else:
            return "La pila está vacía"

    def isEmpty(self):
        return len(self.items) == 0

# Pruebas
pila = Pila()
pila.push(1)
pila.push(2)
pila.push(3)
print(pila.peek())  # Salida: 3
print(pila.pop())   # Salida: 3
print(pila.pop())   # Salida: 2
print(pila.isEmpty())  # Salida: False
print(pila.pop())   # Salida: 1
print(pila.isEmpty())  # Salida: True

Explicación

  • push: Añade un elemento al final de la lista.
  • pop: Remueve y retorna el último elemento de la lista.
  • peek: Retorna el último elemento de la lista sin removerlo.
  • isEmpty: Verifica si la lista está vacía.

Ejercicio 2: Verificación de Paréntesis Balanceados

Objetivo

Utilizar una pila para verificar si los paréntesis en una expresión están balanceados.

Instrucciones

  1. Crear una función verificar_parentesis que tome una cadena como entrada.
  2. Utilizar una pila para verificar si los paréntesis están balanceados.

Ejemplo de Código

def verificar_parentesis(expresion):
    pila = Pila()
    for char in expresion:
        if char == '(':
            pila.push(char)
        elif char == ')':
            if pila.isEmpty():
                return False
            pila.pop()
    return pila.isEmpty()

# Pruebas
print(verificar_parentesis("(())"))  # Salida: True
print(verificar_parentesis("(()"))   # Salida: False
print(verificar_parentesis(")("))    # Salida: False

Explicación

  • Recorre cada carácter de la expresión.
  • Si el carácter es un '(', se añade a la pila.
  • Si el carácter es un ')', se remueve un elemento de la pila.
  • Al final, la pila debe estar vacía para que los paréntesis estén balanceados.

Ejercicio 3: Conversión de Notación Infix a Postfix

Objetivo

Convertir una expresión en notación infix (convencional) a notación postfix (notación polaca inversa) utilizando una pila.

Instrucciones

  1. Crear una función infix_a_postfix que tome una expresión infix como entrada.
  2. Utilizar una pila para manejar los operadores y paréntesis.

Ejemplo de Código

def infix_a_postfix(expresion):
    precedencia = {'+': 1, '-': 1, '*': 2, '/': 2, '^': 3}
    pila = Pila()
    salida = []

    for char in expresion:
        if char.isalnum():
            salida.append(char)
        elif char == '(':
            pila.push(char)
        elif char == ')':
            while not pila.isEmpty() and pila.peek() != '(':
                salida.append(pila.pop())
            pila.pop()
        else:
            while (not pila.isEmpty() and pila.peek() != '(' and
                   precedencia[char] <= precedencia[pila.peek()]):
                salida.append(pila.pop())
            pila.push(char)

    while not pila.isEmpty():
        salida.append(pila.pop())

    return ''.join(salida)

# Pruebas
print(infix_a_postfix("a+b*(c^d-e)^(f+g*h)-i"))  # Salida: abcd^e-fgh*+^*+i-

Explicación

  • Utiliza una pila para almacenar operadores y paréntesis.
  • Los operandos se añaden directamente a la salida.
  • Los operadores se manejan según su precedencia y se añaden a la salida en el orden correcto.

Conclusión

Las pilas son una estructura de datos versátil y esencial en la informática. A través de estos ejercicios, hemos aprendido a implementar una pila básica, verificar paréntesis balanceados y convertir expresiones infix a postfix. Estos ejercicios no solo refuerzan el entendimiento de las pilas, sino que también muestran su aplicabilidad en problemas del mundo real.
