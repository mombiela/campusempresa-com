Introducción a las Pilas

Una pila es una estructura de datos lineal que sigue el principio LIFO (Last In, First Out), lo que significa que el último elemento en entrar es el primero en salir. Las pilas son fundamentales en la informática y se utilizan en diversas aplicaciones, como la gestión de memoria, la evaluación de expresiones y la implementación de algoritmos de retroceso.

Operaciones Fundamentales

Las operaciones básicas que se pueden realizar en una pila son:

  • Push: Añadir un elemento a la pila.
  • Pop: Eliminar el elemento superior de la pila.
  • Peek (o Top): Obtener el elemento superior de la pila sin eliminarlo.
  • isEmpty: Verificar si la pila está vacía.
  • Size: Obtener el número de elementos en la pila.

Push

La operación push añade un nuevo elemento a la parte superior de la pila.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

# Ejemplo de uso
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.items)  # Salida: [1, 2]

Pop

La operación pop elimina y devuelve el elemento superior de la pila.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "La pila está vacía"

    def is_empty(self):
        return len(self.items) == 0

# Ejemplo de uso
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.pop())  # Salida: 2
print(pila.items)  # Salida: [1]

Peek (o Top)

La operación peek devuelve el elemento superior de la pila sin eliminarlo.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "La pila está vacía"

    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        else:
            return "La pila está vacía"

    def is_empty(self):
        return len(self.items) == 0

# Ejemplo de uso
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.peek())  # Salida: 2
print(pila.items)   # Salida: [1, 2]

isEmpty

La operación isEmpty verifica si la pila está vacía.

class Pila:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

# Ejemplo de uso
pila = Pila()
print(pila.is_empty())  # Salida: True
pila.push(1)
print(pila.is_empty())  # Salida: False

Size

La operación size devuelve el número de elementos en la pila.

class Pila:
    def __init__(self):
        self.items = []

    def size(self):
        return len(self.items)

# Ejemplo de uso
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.size())  # Salida: 2

Comparación de Operaciones

Operación Descripción Complejidad Temporal
Push Añadir un elemento a la pila O(1)
Pop Eliminar el elemento superior O(1)
Peek Obtener el elemento superior O(1)
isEmpty Verificar si la pila está vacía O(1)
Size Obtener el número de elementos O(1)

Conclusión

Las pilas son una estructura de datos fundamental en la informática, y comprender sus operaciones básicas es crucial para cualquier profesional de TI. Las operaciones push, pop, peek, isEmpty y size son esenciales para manipular pilas de manera eficiente. Practicar con ejemplos de código y ejercicios ayudará a solidificar estos conceptos y prepararse para aplicaciones más avanzadas.
