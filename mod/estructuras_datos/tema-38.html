Proyectos Finales

Introducción

En esta sección, los estudiantes aplicarán los conocimientos adquiridos sobre estructuras de datos en proyectos prácticos. Estos proyectos están diseñados para consolidar el aprendizaje y proporcionar experiencia práctica en la implementación y manipulación de diversas estructuras de datos.

Objetivos de los Proyectos Finales

  • Aplicar conceptos teóricos en escenarios prácticos.
  • Desarrollar habilidades de resolución de problemas utilizando estructuras de datos.
  • Mejorar la capacidad de escribir código eficiente y bien estructurado.
  • Fomentar el trabajo en equipo y la colaboración (si los proyectos son grupales).

Descripción de los Proyectos

Proyecto 1: Sistema de Gestión de Biblioteca

Descripción: Desarrollar un sistema de gestión de biblioteca que permita realizar operaciones básicas como agregar, eliminar, buscar y listar libros. Utilizar una estructura de datos adecuada para cada operación.

Requisitos:

  • Implementar una lista enlazada para almacenar los libros.
  • Cada libro debe tener atributos como título, autor, ISBN y año de publicación.
  • Funcionalidades:
    • Agregar un nuevo libro.
    • Eliminar un libro por ISBN.
    • Buscar un libro por título o autor.
    • Listar todos los libros en la biblioteca.

Ejemplo de Código:

class Libro:
    def __init__(self, titulo, autor, isbn, ano):
        self.titulo = titulo
        self.autor = autor
        self.isbn = isbn
        self.ano = ano
        self.siguiente = None

class Biblioteca:
    def __init__(self):
        self.cabeza = None

    def agregar_libro(self, libro):
        if not self.cabeza:
            self.cabeza = libro
        else:
            actual = self.cabeza
            while actual.siguiente:
                actual = actual.siguiente
            actual.siguiente = libro

    def eliminar_libro(self, isbn):
        actual = self.cabeza
        anterior = None
        while actual and actual.isbn != isbn:
            anterior = actual
            actual = actual.siguiente
        if anterior is None:
            self.cabeza = actual.siguiente
        elif actual:
            anterior.siguiente = actual.siguiente

    def buscar_libro(self, titulo):
        actual = self.cabeza
        while actual:
            if actual.titulo == titulo:
                return actual
            actual = actual.siguiente
        return None

    def listar_libros(self):
        actual = self.cabeza
        while actual:
            print(f'Título: {actual.titulo}, Autor: {actual.autor}, ISBN: {actual.isbn}, Año: {actual.ano}')
            actual = actual.siguiente

Proyecto 2: Sistema de Gestión de Tareas

Descripción: Crear un sistema de gestión de tareas que permita a los usuarios agregar, eliminar, buscar y listar tareas. Utilizar una estructura de datos adecuada para manejar las tareas.

Requisitos:

  • Implementar una pila (stack) para gestionar las tareas.
  • Cada tarea debe tener atributos como descripción, fecha de vencimiento y prioridad.
  • Funcionalidades:
    • Agregar una nueva tarea.
    • Eliminar la tarea más reciente.
    • Buscar una tarea por descripción.
    • Listar todas las tareas pendientes.

Ejemplo de Código:

class Tarea:
    def __init__(self, descripcion, fecha_vencimiento, prioridad):
        self.descripcion = descripcion
        self.fecha_vencimiento = fecha_vencimiento
        self.prioridad = prioridad

class GestionTareas:
    def __init__(self):
        self.pila = []

    def agregar_tarea(self, tarea):
        self.pila.append(tarea)

    def eliminar_tarea(self):
        if self.pila:
            return self.pila.pop()
        return None

    def buscar_tarea(self, descripcion):
        for tarea in self.pila:
            if tarea.descripcion == descripcion:
                return tarea
        return None

    def listar_tareas(self):
        for tarea in reversed(self.pila):
            print(f'Descripción: {tarea.descripcion}, Fecha de Vencimiento: {tarea.fecha_vencimiento}, Prioridad: {tarea.prioridad}')

Proyecto 3: Sistema de Gestión de Contactos

Descripción: Desarrollar un sistema de gestión de contactos que permita agregar, eliminar, buscar y listar contactos. Utilizar una estructura de datos adecuada para cada operación.

Requisitos:

  • Implementar un árbol binario de búsqueda (BST) para almacenar los contactos.
  • Cada contacto debe tener atributos como nombre, número de teléfono y correo electrónico.
  • Funcionalidades:
    • Agregar un nuevo contacto.
    • Eliminar un contacto por nombre.
    • Buscar un contacto por nombre.
    • Listar todos los contactos en orden alfabético.

Ejemplo de Código:

class Contacto:
    def __init__(self, nombre, telefono, correo):
        self.nombre = nombre
        self.telefono = telefono
        self.correo = correo
        self.izquierda = None
        self.derecha = None

class GestionContactos:
    def __init__(self):
        self.raiz = None

    def agregar_contacto(self, contacto):
        if not self.raiz:
            self.raiz = contacto
        else:
            self._agregar(self.raiz, contacto)

    def _agregar(self, nodo, contacto):
        if contacto.nombre < nodo.nombre:
            if nodo.izquierda is None:
                nodo.izquierda = contacto
            else:
                self._agregar(nodo.izquierda, contacto)
        else:
            if nodo.derecha is None:
                nodo.derecha = contacto
            else:
                self._agregar(nodo.derecha, contacto)

    def eliminar_contacto(self, nombre):
        self.raiz = self._eliminar(self.raiz, nombre)

    def _eliminar(self, nodo, nombre):
        if not nodo:
            return nodo
        if nombre < nodo.nombre:
            nodo.izquierda = self._eliminar(nodo.izquierda, nombre)
        elif nombre > nodo.nombre:
            nodo.derecha = self._eliminar(nodo.derecha, nombre)
        else:
            if not nodo.izquierda:
                return nodo.derecha
            elif not nodo.derecha:
                return nodo.izquierda
            temp = self._min_valor_nodo(nodo.derecha)
            nodo.nombre = temp.nombre
            nodo.telefono = temp.telefono
            nodo.correo = temp.correo
            nodo.derecha = self._eliminar(nodo.derecha, temp.nombre)
        return nodo

    def _min_valor_nodo(self, nodo):
        actual = nodo
        while actual.izquierda:
            actual = actual.izquierda
        return actual

    def buscar_contacto(self, nombre):
        return self._buscar(self.raiz, nombre)

    def _buscar(self, nodo, nombre):
        if not nodo or nodo.nombre == nombre:
            return nodo
        if nombre < nodo.nombre:
            return self._buscar(nodo.izquierda, nombre)
        return self._buscar(nodo.derecha, nombre)

    def listar_contactos(self):
        self._inorden(self.raiz)

    def _inorden(self, nodo):
        if nodo:
            self._inorden(nodo.izquierda)
            print(f'Nombre: {nodo.nombre}, Teléfono: {nodo.telefono}, Correo: {nodo.correo}')
            self._inorden(nodo.derecha)
