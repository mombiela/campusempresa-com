Introducción a las Colas

Las colas son estructuras de datos fundamentales que siguen el principio FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir. Son ampliamente utilizadas en escenarios donde se necesita gestionar tareas en orden secuencial, como en sistemas de impresión, manejo de procesos en sistemas operativos, y en algoritmos de búsqueda en grafos.

Conceptos Clave

  • FIFO (First In, First Out): El primer elemento que se inserta en la cola es el primero en ser removido.
  • Operaciones Básicas:
    • enqueue: Añadir un elemento al final de la cola.
    • dequeue: Remover el primer elemento de la cola.
    • peek o front: Obtener el primer elemento de la cola sin removerlo.
    • isEmpty: Verificar si la cola está vacía.
  • Implementaciones Comunes:
    • Colas usando listas enlazadas.
    • Colas usando arrays.

Ejemplos de Código

Implementación de una Cola usando Arrays

class Queue:
    def __init__(self):
        self.queue = []

    def enqueue(self, item):
        self.queue.append(item)
        print(f"Enqueued: {item}")

    def dequeue(self):
        if not self.isEmpty():
            item = self.queue.pop(0)
            print(f"Dequeued: {item}")
            return item
        else:
            print("Queue is empty")
            return None

    def peek(self):
        if not self.isEmpty():
            return self.queue[0]
        else:
            print("Queue is empty")
            return None

    def isEmpty(self):
        return len(self.queue) == 0

# Ejemplo de uso
q = Queue()
q.enqueue(1)
q.enqueue(2)
q.enqueue(3)
print(f"Front element: {q.peek()}")
q.dequeue()
q.dequeue()
q.dequeue()
q.dequeue()  # Intentar dequeuing cuando la cola está vacía

Implementación de una Cola usando Listas Enlazadas

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class LinkedListQueue:
    def __init__(self):
        self.front = None
        self.rear = None

    def enqueue(self, item):
        new_node = Node(item)
        if self.rear is None:
            self.front = self.rear = new_node
        else:
            self.rear.next = new_node
            self.rear = new_node
        print(f"Enqueued: {item}")

    def dequeue(self):
        if self.isEmpty():
            print("Queue is empty")
            return None
        temp = self.front
        self.front = temp.next
        if self.front is None:
            self.rear = None
        print(f"Dequeued: {temp.value}")
        return temp.value

    def peek(self):
        if not self.isEmpty():
            return self.front.value
        else:
            print("Queue is empty")
            return None

    def isEmpty(self):
        return self.front is None

# Ejemplo de uso
llq = LinkedListQueue()
llq.enqueue(1)
llq.enqueue(2)
llq.enqueue(3)
print(f"Front element: {llq.peek()}")
llq.dequeue()
llq.dequeue()
llq.dequeue()
llq.dequeue()  # Intentar dequeuing cuando la cola está vacía

Ejercicios Prácticos

  1. Implementar una Cola Circular:

    • Descripción: Implementa una cola circular usando arrays. Asegúrate de manejar correctamente los casos en los que la cola esté llena o vacía.
    • Puntos clave:
      • Uso de punteros front y rear.
      • Manejo del wrap-around en el array.

  2. Simulación de un Sistema de Impresión:

    • Descripción: Simula un sistema de impresión donde las tareas de impresión se encolan y se procesan en orden.
    • Puntos clave:
      • Cada tarea de impresión tiene un identificador único y una duración.
      • Implementa una función para añadir tareas y otra para procesarlas.

  3. Cola de Prioridad Básica:

    • Descripción: Implementa una cola de prioridad básica donde cada elemento tiene una prioridad asociada y los elementos con mayor prioridad se procesan primero.
    • Puntos clave:
      • Uso de una estructura de datos adecuada para manejar las prioridades (puede ser un heap o una lista ordenada).
      • Implementación de las operaciones enqueue y dequeue con prioridades.

Conclusión

Las colas son una estructura de datos esencial en la informática, con aplicaciones prácticas en diversos campos. Comprender cómo implementar y utilizar colas es fundamental para cualquier profesional de TI. A través de los ejemplos y ejercicios proporcionados, los estudiantes pueden fortalecer su comprensión y habilidades en el manejo de colas, preparándose mejor para enfrentar problemas del mundo real que requieran esta estructura de datos.
