Introducción

En esta sección, exploraremos cómo manejar CSS y SASS en un proyecto de Webpack. Comenzaremos con lo básico de cargar archivos CSS y luego pasaremos a temas más avanzados como el uso de SASS, la optimización de CSS para producción y el aprovechamiento de los módulos CSS.

Cargando Archivos CSS

Para cargar archivos CSS en un proyecto de Webpack, necesitas usar css-loader y style-loader.

Pasos para Cargar Archivos CSS

  1. Instalar los cargadores necesarios: 
    npm install css-loader style-loader --save-dev
  2. Actualizar la configuración de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader'],
      },
    ],
  },
};
  3. Importar CSS en tus archivos JavaScript: 
    // src/index.js
import './styles.css';

Explicación

  • css-loader: Este cargador interpreta @import y url() como import/require() y los resolverá.
  • style-loader: Este cargador inyecta CSS en el DOM.

Usando SASS

SASS es un potente preprocesador de CSS que te permite usar variables, reglas anidadas, mixins y más.

Pasos para Cargar Archivos SASS

  1. Instalar los cargadores necesarios: 
    npm install sass-loader sass css-loader style-loader --save-dev
  2. Actualizar la configuración de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: ['style-loader', 'css-loader', 'sass-loader'],
      },
    ],
  },
};
  3. Importar SASS en tus archivos JavaScript: 
    // src/index.js
import './styles.scss';

Explicación

  • sass-loader: Este cargador compila SASS a CSS.
  • El orden de los cargadores es importante: style-loader -> css-loader -> sass-loader.

Optimizando CSS para Producción

Cuando construyes para producción, quieres minimizar y optimizar tus archivos CSS.

Pasos para Optimizar CSS

  1. Instalar los plugins necesarios: 
    npm install mini-css-extract-plugin css-minimizer-webpack-plugin --save-dev
  2. Actualizar la configuración de Webpack: 
    // webpack.config.js
const MiniCssExtractPlugin = require('mini-css-extract-plugin');
const CssMinimizerPlugin = require('css-minimizer-webpack-plugin');

module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
      {
        test: /\.scss$/,
        use: [MiniCssExtractPlugin.loader, 'css-loader', 'sass-loader'],
      },
    ],
  },
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  optimization: {
    minimizer: [
      `...`,
      new CssMinimizerPlugin(),
    ],
  },
};

Explicación

  • MiniCssExtractPlugin: Este plugin extrae CSS en archivos separados. Crea un archivo CSS por cada archivo JS que contiene CSS.
  • CssMinimizerPlugin: Este plugin optimiza y minimiza el CSS.

Módulos CSS

Los módulos CSS te permiten delimitar el CSS creando automáticamente un nombre de clase único.

Pasos para Usar Módulos CSS

  1. Actualizar la configuración de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.module\.css$/,
        use: [
          'style-loader',
          {
            loader: 'css-loader',
            options: {
              modules: true,
            },
          },
        ],
      },
    ],
  },
};
  2. Importar Módulos CSS en tus archivos JavaScript: 
    // src/index.js
import styles from './styles.module.css';

const element = document.createElement('div');
element.className = styles.myClass;
document.body.appendChild(element);

Explicación

  • La opción modules: true en css-loader habilita los módulos CSS.
  • Los módulos CSS delimitan automáticamente el CSS generando nombres de clase únicos.

Conclusión

En esta sección, cubrimos cómo manejar CSS y SASS en un proyecto de Webpack. Comenzamos con lo básico de cargar archivos CSS, pasamos al uso de SASS, discutimos la optimización de CSS para producción y finalmente exploramos los módulos CSS. Siguiendo estos pasos, puedes gestionar y optimizar eficientemente tus estilos en un proyecto de Webpack.
