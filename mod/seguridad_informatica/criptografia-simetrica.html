Introducción

La criptografía simétrica es una técnica de cifrado en la que la misma clave se utiliza tanto para cifrar como para descifrar la información. Es uno de los métodos más antiguos y ampliamente utilizados para proteger datos sensibles.

Conceptos Clave

  • Clave Secreta: La misma clave se utiliza para cifrar y descifrar los datos.
  • Algoritmo de Cifrado: Procedimiento matemático que transforma el texto claro en texto cifrado.
  • Texto Claro: Información original que se desea proteger.
  • Texto Cifrado: Información encriptada que resulta del proceso de cifrado.
  • Desempeño: Generalmente, los algoritmos simétricos son más rápidos que los asimétricos.

Algoritmos Comunes

DES (Data Encryption Standard)

  • Longitud de la Clave: 56 bits.
  • Estructura: Algoritmo de cifrado en bloque.
  • Seguridad: Considerado inseguro debido a su corta longitud de clave.

AES (Advanced Encryption Standard)

  • Longitud de la Clave: 128, 192 o 256 bits.
  • Estructura: Algoritmo de cifrado en bloque.
  • Seguridad: Actualmente considerado seguro y ampliamente utilizado.

RC4

  • Longitud de la Clave: Variable, típicamente entre 40 y 2048 bits.
  • Estructura: Algoritmo de cifrado en flujo.
  • Seguridad: Vulnerable a ciertos tipos de ataques, pero aún en uso en algunas aplicaciones.

Ejemplo de Código

A continuación, se presenta un ejemplo de cifrado y descifrado utilizando el algoritmo AES en Python con la biblioteca pycryptodome.

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
import base64

# Generar una clave de 16 bytes (128 bits)
key = get_random_bytes(16)

# Crear un objeto de cifrado AES
cipher = AES.new(key, AES.MODE_EAX)

# Texto claro
data = b"Este es un mensaje secreto"

# Cifrar el texto claro
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(data)

# Mostrar el texto cifrado en base64
print("Texto cifrado:", base64.b64encode(ciphertext).decode('utf-8'))

# Crear un nuevo objeto de cifrado AES para descifrar
cipher_dec = AES.new(key, AES.MODE_EAX, nonce=nonce)

# Descifrar el texto cifrado
plaintext = cipher_dec.decrypt(ciphertext)

# Mostrar el texto claro
print("Texto descifrado:", plaintext.decode('utf-8'))

Explicación del Código

  1. Generación de Clave: Se genera una clave de 16 bytes utilizando get_random_bytes.
  2. Creación del Objeto de Cifrado: Se crea un objeto de cifrado AES en modo EAX.
  3. Cifrado del Texto Claro: Se cifra el texto claro y se obtiene el texto cifrado junto con un tag de autenticación.
  4. Descifrado del Texto Cifrado: Se crea un nuevo objeto de cifrado AES con la misma clave y nonce para descifrar el texto cifrado.

Comparación de Algoritmos

Algoritmo Longitud de Clave Estructura Seguridad
DES 56 bits Bloque Inseguro
AES 128, 192, 256 bits Bloque Seguro
RC4 Variable Flujo Vulnerable

Conclusión

La criptografía simétrica es una técnica fundamental en la seguridad informática, utilizada para proteger datos sensibles mediante el uso de una clave secreta compartida. Aunque algunos algoritmos como DES han sido superados por razones de seguridad, otros como AES siguen siendo estándares robustos y confiables. Es crucial entender tanto las fortalezas como las debilidades de estos algoritmos para aplicarlos adecuadamente en diferentes contextos de seguridad.
