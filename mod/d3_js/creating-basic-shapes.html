En esta sección, exploraremos cómo crear formas básicas usando D3.js. Esta es una habilidad fundamental para cualquiera que desee crear visualizaciones de datos, ya que las formas constituyen los bloques de construcción de gráficos más complejos.

Introducción a D3.js

D3.js (Documentos Impulsados por Datos) es una biblioteca de JavaScript para producir visualizaciones de datos dinámicas e interactivas en navegadores web. Utiliza estándares HTML, SVG y CSS.

Conceptos Clave

  • SVG (Gráficos Vectoriales Escalables): Un lenguaje para describir gráficos 2D en XML.
  • Selecciones D3: Métodos para seleccionar y manipular elementos del DOM.
  • Vinculación de Datos: Asociar datos con elementos del DOM.

Configurando Tu Entorno

Antes de comenzar a crear formas, asegúrate de tener un archivo HTML básico configurado e incluir la biblioteca D3.js.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>D3.js Formas Básicas</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <svg width="500" height="500"></svg>
    <script>
        // Tu código D3.js irá aquí
    </script>
</body>
</html>

Creando un Círculo

Para crear un círculo en D3.js, usarás el elemento SVG circle.

Ejemplo

const svg = d3.select("svg");

svg.append("circle")
    .attr("cx", 250) // coordenada x del centro
    .attr("cy", 250) // coordenada y del centro
    .attr("r", 50)   // radio
    .attr("fill", "blue"); // color de relleno

Explicación

  • d3.select("svg"): Selecciona el elemento SVG.
  • append("circle"): Añade un elemento círculo al SVG.
  • attr("cx", 250): Establece la coordenada x del centro del círculo en 250.
  • attr("cy", 250): Establece la coordenada y del centro del círculo en 250.
  • attr("r", 50): Establece el radio del círculo en 50.
  • attr("fill", "blue"): Establece el color de relleno del círculo en azul.

Creando un Rectángulo

Para crear un rectángulo, usa el elemento SVG rect.

Ejemplo

svg.append("rect")
    .attr("x", 100) // coordenada x de la esquina superior izquierda
    .attr("y", 100) // coordenada y de la esquina superior izquierda
    .attr("width", 200) // ancho del rectángulo
    .attr("height", 100) // altura del rectángulo
    .attr("fill", "green"); // color de relleno

Explicación

  • attr("x", 100): Establece la coordenada x de la esquina superior izquierda del rectángulo en 100.
  • attr("y", 100): Establece la coordenada y de la esquina superior izquierda del rectángulo en 100.
  • attr("width", 200): Establece el ancho del rectángulo en 200.
  • attr("height", 100): Establece la altura del rectángulo en 100.
  • attr("fill", "green"): Establece el color de relleno del rectángulo en verde.

Creando una Línea

Para crear una línea, usa el elemento SVG line.

Ejemplo

svg.append("line")
    .attr("x1", 50) // coordenada x del punto de inicio
    .attr("y1", 50) // coordenada y del punto de inicio
    .attr("x2", 450) // coordenada x del punto final
    .attr("y2", 450) // coordenada y del punto final
    .attr("stroke", "red") // color del trazo
    .attr("stroke-width", 2); // ancho del trazo

Explicación

  • attr("x1", 50): Establece la coordenada x del punto de inicio de la línea en 50.
  • attr("y1", 50): Establece la coordenada y del punto de inicio de la línea en 50.
  • attr("x2", 450): Establece la coordenada x del punto final de la línea en 450.
  • attr("y2", 450): Establece la coordenada y del punto final de la línea en 450.
  • attr("stroke", "red"): Establece el color del trazo de la línea en rojo.
  • attr("stroke-width", 2): Establece el ancho del trazo de la línea en 2.

Creando una Elipse

Para crear una elipse, usa el elemento SVG ellipse.

Ejemplo

svg.append("ellipse")
    .attr("cx", 250) // coordenada x del centro
    .attr("cy", 250) // coordenada y del centro
    .attr("rx", 100) // radio x
    .attr("ry", 50) // radio y
    .attr("fill", "purple"); // color de relleno

Explicación

  • attr("cx", 250): Establece la coordenada x del centro de la elipse en 250.
  • attr("cy", 250): Establece la coordenada y del centro de la elipse en 250.
  • attr("rx", 100): Establece el radio x de la elipse en 100.
  • attr("ry", 50): Establece el radio y de la elipse en 50.
  • attr("fill", "purple"): Establece el color de relleno de la elipse en púrpura.

Conclusión

En esta sección, aprendiste cómo crear formas básicas como círculos, rectángulos, líneas y elipses usando D3.js. Estas formas son la base para construir visualizaciones más complejas. Practica creando estas formas y experimenta con diferentes atributos para obtener una mejor comprensión de cómo D3.js trabaja con elementos SVG.
