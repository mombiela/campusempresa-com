Introducción

Antes de sumergirte en D3.js, es esencial configurar correctamente tu entorno de desarrollo. Esto asegura que tengas todas las herramientas y bibliotecas necesarias para crear y visualizar datos de manera efectiva. Esta sección te guiará a través de la instalación del software requerido y la configuración de una estructura básica de proyecto.

Prerrequisitos

Antes de configurar D3.js, asegúrate de tener lo siguiente:

  • Un navegador web moderno (por ejemplo, Chrome, Firefox, Safari)
  • Un editor de código (por ejemplo, Visual Studio Code, Sublime Text, Atom)
  • Conocimientos básicos de HTML, CSS y JavaScript

Instalación de Node.js y npm

Node.js y npm (Node Package Manager) son herramientas esenciales para gestionar bibliotecas y dependencias de JavaScript.

  1. Descargar e Instalar Node.js:

    • Visita el sitio web de Node.js.
    • Descarga la versión LTS (Long Term Support) para tu sistema operativo.
    • Sigue las instrucciones de instalación.

  2. Verificar la Instalación:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta los siguientes comandos para verificar las versiones instaladas: 
      node -v
npm -v
    • Deberías ver los números de versión tanto de Node.js como de npm.

Configuración de un Nuevo Proyecto

  1. Crear un Directorio de Proyecto:

    • Abre tu terminal o símbolo del sistema.
    • Navega al directorio donde deseas crear tu proyecto.
    • Ejecuta el siguiente comando para crear un nuevo directorio: 
      mkdir d3-project
cd d3-project

  2. Inicializar un Nuevo Proyecto npm:

    • Ejecuta el siguiente comando para inicializar un nuevo proyecto npm: 
      npm init -y
    • Esto creará un archivo package.json con configuraciones predeterminadas.

Instalación de D3.js

  1. Instalar D3.js a través de npm:

    • Ejecuta el siguiente comando para instalar D3.js: 
      npm install d3

  2. Verificar la Instalación:

    • Revisa el directorio node_modules para asegurarte de que D3.js ha sido instalado.
    • El archivo package.json también debería listar D3.js como una dependencia.

Configuración de una Plantilla HTML Básica

Crea un archivo HTML básico para empezar a usar D3.js.

  1. Crear un Archivo index.html:

    • En tu directorio de proyecto, crea un archivo llamado index.html.
    • Añade la siguiente estructura básica de HTML: 
      <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Proyecto D3.js</title>
</head>
<body>
    <h1>Bienvenido a D3.js</h1>
    <script src="node_modules/d3/dist/d3.min.js"></script>
    <script src="app.js"></script>
</body>
</html>

  2. Crear un Archivo app.js:

    • En tu directorio de proyecto, crea un archivo llamado app.js.
    • Añade el siguiente código para verificar que D3.js está funcionando: 
      // Selecciona el elemento body y añade un párrafo
d3.select("body").append("p").text("¡D3.js está funcionando!");

  3. Abrir index.html en un Navegador:

    • Abre el archivo index.html en tu navegador web.
    • Deberías ver el mensaje "¡D3.js está funcionando!" mostrado en la página.

Uso de un Servidor de Desarrollo Local

Para una experiencia de desarrollo más fluida, utiliza un servidor de desarrollo local para servir tus archivos.

  1. Instalar live-server:

    • Ejecuta el siguiente comando para instalar live-server globalmente: 
      npm install -g live-server

  2. Iniciar el Servidor:

    • En tu directorio de proyecto, ejecuta el siguiente comando: 
      live-server
    • Esto iniciará un servidor local y abrirá tu proyecto en el navegador web predeterminado. Cualquier cambio que realices en tus archivos actualizará automáticamente el navegador.

Conclusión

Configurar tu entorno es el primer paso hacia el dominio de D3.js. Al asegurarte de tener Node.js, npm y un servidor de desarrollo local, puedes gestionar eficientemente las dependencias de tu proyecto y agilizar tu proceso de desarrollo. Con tu entorno listo, ahora puedes centrarte en aprender y crear impresionantes visualizaciones de datos con D3.js.
