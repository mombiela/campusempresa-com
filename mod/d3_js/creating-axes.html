Creando Ejes
Personalización de Ejes ►

En esta sección, exploraremos cómo crear y personalizar ejes en D3.js. Los ejes son esenciales para visualizar datos de manera significativa, proporcionando contexto y escala a los puntos de datos mostrados en gráficos y diagramas.

Introducción a los Ejes

  • Ejes: Los ejes son líneas que definen el sistema de coordenadas para un gráfico. Ayudan a mapear valores de datos a posiciones en el gráfico.
  • Ejes en D3.js: D3.js proporciona funciones integradas para crear y personalizar ejes fácilmente.

Configuración del Entorno

Antes de sumergirnos en la creación de ejes, asegúrate de tener la siguiente configuración:

  • Archivo HTML: Crea un archivo HTML para incluir tu script de D3.js.
  • Biblioteca D3.js: Incluye la biblioteca D3.js en tu archivo HTML.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Ejes en D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <script src="script.js"></script>
</body>
</html>

Creando un Eje Básico

  1. Definir el Contenedor SVG: Crea un elemento SVG donde se dibujará el eje.
const svg = d3.select("body")
    .append("svg")
    .attr("width", 500)
    .attr("height", 300);
  1. Crear Escalas: Las escalas mapean valores de datos a valores de píxeles. Para los ejes, típicamente usamos escalas lineales.
const xScale = d3.scaleLinear()
    .domain([0, 100])  // Dominio de entrada
    .range([0, 480]);  // Rango de salida

const yScale = d3.scaleLinear()
    .domain([0, 100])
    .range([280, 0]);
  1. Generar Ejes: Usa los generadores de ejes de D3 para crear los ejes.
const xAxis = d3.axisBottom(xScale);
const yAxis = d3.axisLeft(yScale);
  1. Renderizar Ejes: Añade los ejes al contenedor SVG.
svg.append("g")
    .attr("transform", "translate(10, 280)")  // Posicionar el eje x
    .call(xAxis);

svg.append("g")
    .attr("transform", "translate(10, 0)")  // Posicionar el eje y
    .call(yAxis);

Personalizando Ejes

Ticks y Formato de Ticks

  • Ticks: Controla el número de ticks en el eje.
  • Formato de Ticks: Personaliza el formato de las etiquetas de los ticks.
const xAxis = d3.axisBottom(xScale)
    .ticks(5)  // Establecer el número de ticks
    .tickFormat(d3.format(".0f"));  // Formatear ticks como enteros

const yAxis = d3.axisLeft(yScale)
    .ticks(5)
    .tickFormat(d3.format(".0%"));  // Formatear ticks como porcentajes

Estilizando Ejes

  • CSS: Usa CSS para estilizar los ejes.
<style>
    .axis path,
    .axis line {
        fill: none;
        stroke: black;
        shape-rendering: crispEdges;
    }

    .axis text {
        font-family: sans-serif;
        font-size: 12px;
    }
</style>
  • Aplicando Clases CSS: Aplica clases CSS a los ejes.
svg.append("g")
    .attr("class", "x axis")
    .attr("transform", "translate(10, 280)")
    .call(xAxis);

svg.append("g")
    .attr("class", "y axis")
    .attr("transform", "translate(10, 0)")
    .call(yAxis);

Personalizaciones Avanzadas

Líneas de Cuadrícula

  • Líneas de Cuadrícula: Añade líneas de cuadrícula al gráfico para una mejor legibilidad.
const xGridlines = d3.axisBottom(xScale)
    .tickSize(-280)
    .tickFormat("");

const yGridlines = d3.axisLeft(yScale)
    .tickSize(-480)
    .tickFormat("");

svg.append("g")
    .attr("class", "grid")
    .attr("transform", "translate(10, 280)")
    .call(xGridlines);

svg.append("g")
    .attr("class", "grid")
    .attr("transform", "translate(10, 0)")
    .call(yGridlines);

Rotando Etiquetas de Ticks

  • Rotando Etiquetas: Rota las etiquetas de los ticks para una mejor legibilidad cuando son demasiado largas.
svg.append("g")
    .attr("class", "x axis")
    .attr("transform", "translate(10, 280)")
    .call(xAxis)
    .selectAll("text")
    .attr("transform", "rotate(-45)")
    .style("text-anchor", "end");

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación y personalización de ejes en D3.js. Aprendimos cómo:

  • Configurar el entorno y crear un eje básico.
  • Personalizar ticks y formatos de ticks.
  • Estilizar ejes usando CSS.
  • Añadir personalizaciones avanzadas como líneas de cuadrícula y rotación de etiquetas de ticks.

Entender cómo crear y personalizar ejes es crucial para construir visualizaciones de datos efectivas. Practica estos conceptos para volverte competente en el uso de D3.js para tus proyectos de visualización de datos.
