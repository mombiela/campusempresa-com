Introducción a D3.js

D3.js (Documentos Basados en Datos) es una poderosa biblioteca de JavaScript para crear visualizaciones de datos dinámicas e interactivas en navegadores web. Permite a los desarrolladores vincular datos a un Modelo de Objeto de Documento (DOM) y aplicar transformaciones basadas en datos al documento.

Conceptos Clave

  • Selecciones: Seleccionar elementos del DOM.
  • Vinculación de Datos: Asociar datos con elementos del DOM.
  • Entrar, Actualizar, Salir: Manejar cambios basados en datos en el DOM.
  • Escalas: Mapear valores de datos a propiedades visuales.
  • Ejes: Crear y personalizar ejes para visualizaciones.

Configuración de Tu Entorno

Antes de sumergirte en D3.js, necesitas configurar tu entorno de desarrollo.

Pasos para Configurar

  1. Instalar un Editor de Código: Usa editores como Visual Studio Code, Sublime Text o Atom.
  2. Incluir D3.js en Tu Proyecto:
    • Vía CDN: 
      <script src="https://d3js.org/d3.v7.min.js"></script>
    • Vía npm: 
      npm install d3

Selecciones Básicas y Vinculación de Datos

Las selecciones y la vinculación de datos son fundamentales para trabajar con D3.js.

Seleccionar Elementos

Usa D3.js para seleccionar elementos del DOM:

// Seleccionar un solo elemento
d3.select("body").style("background-color", "lightblue");

// Seleccionar múltiples elementos
d3.selectAll("p").style("color", "blue");

Vincular Datos a Elementos

Vincula datos a elementos del DOM para crear visualizaciones dinámicas:

// Datos de ejemplo
const data = [10, 20, 30, 40, 50];

// Seleccionar elementos y vincular datos
d3.select("body").selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "teal")
  .style("margin", "5px");

Patrón de Entrar, Actualizar y Salir

El patrón de entrar, actualizar y salir es crucial para manejar datos dinámicos.

Selección de Entrar

Manejar nuevos puntos de datos:

const data = [10, 20, 30];

const divs = d3.select("body").selectAll("div")
  .data(data);

divs.enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "teal")
  .style("margin", "5px");

Selección de Actualizar

Actualizar elementos existentes:

const updatedData = [15, 25, 35];

const divs = d3.select("body").selectAll("div")
  .data(updatedData);

divs.style("width", d => `${d * 10}px`);

Selección de Salir

Eliminar elementos que ya no tienen datos correspondientes:

const reducedData = [15, 25];

const divs = d3.select("body").selectAll("div")
  .data(reducedData);

divs.exit().remove();

Creación de Visualizaciones Básicas

Aprende a crear visualizaciones básicas como gráficos de barras y diagramas de dispersión.

Ejemplo de Gráfico de Barras

const data = [10, 20, 30, 40, 50];

const svg = d3.select("body").append("svg")
  .attr("width", 500)
  .attr("height", 100);

svg.selectAll("rect")
  .data(data)
  .enter()
  .append("rect")
  .attr("x", (d, i) => i * 30)
  .attr("y", d => 100 - d)
  .attr("width", 25)
  .attr("height", d => d)
  .attr("fill", "teal");

Ejemplo de Diagrama de Dispersión

const data = [
  { x: 30, y: 20 },
  { x: 50, y: 80 },
  { x: 90, y: 50 },
  { x: 120, y: 70 },
  { x: 150, y: 30 }
];

const svg = d3.select("body").append("svg")
  .attr("width", 200)
  .attr("height", 100);

svg.selectAll("circle")
  .data(data)
  .enter()
  .append("circle")
  .attr("cx", d => d.x)
  .attr("cy", d => d.y)
  .attr("r", 5)
  .attr("fill", "teal");

Conclusión

En esta sección, aprendiste los conceptos básicos de D3.js, incluyendo la configuración de tu entorno, hacer selecciones, vincular datos y usar el patrón de entrar, actualizar y salir. También creaste visualizaciones básicas como gráficos de barras y diagramas de dispersión. Estas habilidades fundamentales te ayudarán a construir visualizaciones de datos más complejas e interactivas a medida que avances en el curso.
