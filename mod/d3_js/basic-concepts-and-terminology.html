Introducción a D3.js

D3.js (Documentos Impulsados por Datos) es una poderosa biblioteca de JavaScript para crear visualizaciones de datos dinámicas e interactivas en navegadores web. Aprovecha los estándares web como HTML, SVG y CSS para dar vida a los datos.

Conceptos Clave

Selecciones

  • Selecciones: D3 usa selecciones para vincular datos a elementos del DOM. Una selección es un grupo de elementos sobre los que puedes operar como una unidad única. 
    d3.select("body") // Selecciona el primer elemento <body>
d3.selectAll("p") // Selecciona todos los elementos <p>

Vinculación de Datos

  • Vinculación de Datos: Este es el proceso de conectar datos a elementos del DOM. D3 proporciona métodos como .data() y .enter() para facilitar esto. 
    const data = [10, 20, 30];
d3.selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .text(d => d);

Escalas

  • Escalas: Las escalas son funciones que mapean desde un dominio de entrada a un rango de salida. Son esenciales para traducir valores de datos en representaciones visuales. 
    const scale = d3.scaleLinear()
  .domain([0, 100])
  .range([0, 500]);
console.log(scale(50)); // Salida: 250

Ejes

  • Ejes: D3 proporciona funciones integradas para crear y personalizar ejes, que son esenciales para muchos tipos de visualizaciones. 
    const xAxis = d3.axisBottom(scale);
d3.select("svg")
  .append("g")
  .attr("transform", "translate(0, 500)")
  .call(xAxis);

Elementos SVG

  • Elementos SVG: D3 usa Gráficos Vectoriales Escalables (SVG) para crear formas como círculos, rectángulos y líneas. 
    d3.select("svg")
  .append("circle")
  .attr("cx", 50)
  .attr("cy", 50)
  .attr("r", 40)
  .style("fill", "blue");

Transiciones

  • Transiciones: D3 te permite animar cambios en tus visualizaciones con transiciones. 
    d3.select("circle")
  .transition()
  .duration(1000)
  .attr("r", 20);

Ejemplo: Creando un Gráfico de Barras Simple

Vamos a juntar estos conceptos para crear un gráfico de barras simple.

  1. Configuración HTML:

    <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Gráfico de Barras Simple</title>
  <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
  <svg width="500" height="300"></svg>
</body>
</html>

  2. Código JavaScript:

    const data = [30, 86, 168, 281, 303, 365];
const svg = d3.select("svg");
const width = +svg.attr("width");
const height = +svg.attr("height");

const xScale = d3.scaleBand()
  .domain(d3.range(data.length))
  .range([0, width])
  .padding(0.1);

const yScale = d3.scaleLinear()
  .domain([0, d3.max(data)])
  .range([height, 0]);

svg.selectAll("rect")
  .data(data)
  .enter()
  .append("rect")
  .attr("x", (d, i) => xScale(i))
  .attr("y", d => yScale(d))
  .attr("width", xScale.bandwidth())
  .attr("height", d => height - yScale(d))
  .attr("fill", "steelblue");

Conclusión

Entender los conceptos básicos y la terminología de D3.js es crucial para crear visualizaciones de datos efectivas. Al dominar las selecciones, la vinculación de datos, las escalas, los ejes, los elementos SVG y las transiciones, puedes construir una amplia gama de visualizaciones. En las próximas secciones, profundizaremos en cada uno de estos conceptos y exploraremos características más avanzadas de D3.js.
