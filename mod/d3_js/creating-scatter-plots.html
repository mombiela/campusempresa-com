Introducción

Los gráficos de dispersión son una poderosa herramienta de visualización utilizada para mostrar la relación entre dos variables. En esta sección, aprenderemos a crear gráficos de dispersión usando D3.js, una biblioteca de JavaScript para producir visualizaciones de datos dinámicas e interactivas en navegadores web.

Configuración del Entorno

Antes de comenzar a crear gráficos de dispersión, asegúrate de tener lo siguiente:

  • Un conocimiento básico de HTML, CSS y JavaScript.
  • La biblioteca D3.js incluida en tu proyecto.

Incluir D3.js

Puedes incluir D3.js en tu archivo HTML añadiendo la siguiente etiqueta de script en la sección <head>:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Gráfico de Dispersión con D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <!-- Tu contenido irá aquí -->
</body>
</html>

Creación de un Gráfico de Dispersión Básico

Comencemos creando un gráfico de dispersión simple.

Estructura HTML

Primero, configura la estructura básica de HTML:

<body>
    <div id="scatter-plot"></div>
    <script src="scatter-plot.js"></script>
</body>

Código JavaScript

A continuación, crea un archivo scatter-plot.js y añade el siguiente código:

// Establece las dimensiones y márgenes del gráfico
const margin = {top: 20, right: 30, bottom: 40, left: 50},
      width = 460 - margin.left - margin.right,
      height = 400 - margin.top - margin.bottom;

// Añade el objeto svg al cuerpo de la página
const svg = d3.select("#scatter-plot")
  .append("svg")
    .attr("width", width + margin.left + margin.right)
    .attr("height", height + margin.top + margin.bottom)
  .append("g")
    .attr("transform", `translate(${margin.left},${margin.top})`);

// Datos de ejemplo
const data = [
  {x: 30, y: 20}, {x: 50, y: 90}, {x: 70, y: 50},
  {x: 90, y: 120}, {x: 110, y: 80}, {x: 130, y: 150}
];

// Añadir eje X
const x = d3.scaleLinear()
  .domain([0, d3.max(data, d => d.x)])
  .range([ 0, width ]);
svg.append("g")
  .attr("transform", `translate(0,${height})`)
  .call(d3.axisBottom(x));

// Añadir eje Y
const y = d3.scaleLinear()
  .domain([0, d3.max(data, d => d.y)])
  .range([ height, 0]);
svg.append("g")
  .call(d3.axisLeft(y));

// Añadir puntos
svg.append('g')
  .selectAll("dot")
  .data(data)
  .enter()
  .append("circle")
    .attr("cx", d => x(d.x) )
    .attr("cy", d => y(d.y) )
    .attr("r", 5)
    .style("fill", "#69b3a2");

Explicación

  • Márgenes y Dimensiones: Define los márgenes y dimensiones del contenedor SVG.
  • Contenedor SVG: Añade un elemento SVG al div #scatter-plot y establece sus dimensiones.
  • Datos de Ejemplo: Usa un array de objetos para representar los puntos de datos.
  • Escalas: Crea escalas lineales para los ejes X e Y basadas en los datos.
  • Ejes: Añade los ejes X e Y al SVG.
  • Puntos: Añade círculos para representar los puntos de datos.

Mejorando el Gráfico de Dispersión

Ahora que tenemos un gráfico de dispersión básico, vamos a mejorarlo con características adicionales.

Añadir Etiquetas y Títulos

// Añadir etiqueta del eje X
svg.append("text")
  .attr("text-anchor", "end")
  .attr("x", width)
  .attr("y", height + margin.top + 20)
  .text("Etiqueta del Eje X");

// Añadir etiqueta del eje Y
svg.append("text")
  .attr("text-anchor", "end")
  .attr("transform", "rotate(-90)")
  .attr("y", -margin.left + 20)
  .attr("x", -margin.top)
  .text("Etiqueta del Eje Y");

// Añadir título
svg.append("text")
  .attr("x", width / 2)
  .attr("y", -margin.top / 2)
  .attr("text-anchor", "middle")
  .style("font-size", "16px")
  .style("text-decoration", "underline")
  .text("Título del Gráfico de Dispersión");

Añadir Tooltips

// Crear un div de tooltip que está oculto por defecto
const tooltip = d3.select("body").append("div")
  .style("opacity", 0)
  .attr("class", "tooltip")
  .style("background-color", "white")
  .style("border", "solid")
  .style("border-width", "1px")
  .style("border-radius", "5px")
  .style("padding", "10px");

// Añadir event listeners para tooltips
svg.selectAll("circle")
  .on("mouseover", (event, d) => {
    tooltip.transition()
      .duration(200)
      .style("opacity", .9);
    tooltip.html(`X: ${d.x}<br>Y: ${d.y}`)
      .style("left", (event.pageX + 5) + "px")
      .style("top", (event.pageY - 28) + "px");
  })
  .on("mouseout", () => {
    tooltip.transition()
      .duration(500)
      .style("opacity", 0);
  });

Explicación

  • Etiquetas y Títulos: Añade elementos de texto para etiquetar los ejes y proporcionar un título para el gráfico de dispersión.
  • Tooltips: Crea un div de tooltip y añade event listeners para mostrar el tooltip al pasar el ratón sobre los puntos de datos.

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación de gráficos de dispersión usando D3.js. Comenzamos con un gráfico de dispersión simple y lo mejoramos añadiendo etiquetas, títulos y tooltips. Los gráficos de dispersión son una herramienta versátil para visualizar datos, y D3.js proporciona capacidades poderosas para crear visualizaciones interactivas y dinámicas.

Al dominar estas técnicas, puedes crear gráficos de dispersión más complejos e informativos para comprender y presentar mejor tus datos.
