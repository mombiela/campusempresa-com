Creando Tu Primer Proyecto
Entendiendo los Objetos del Juego y Componentes ►

¡Bienvenido a tu viaje en Unity! En esta sección, te guiaremos a través del proceso de crear tu primer proyecto en Unity. Este es un paso esencial que sentará las bases para tu trabajo de desarrollo futuro.

Configurando Unity

Antes de sumergirnos en la creación de un proyecto, asegúrate de tener Unity instalado en tu computadora. Puedes descargarlo desde el sitio web de Unity.

Unity Hub

Unity Hub es una herramienta de gestión que te ayuda a manejar múltiples instalaciones y proyectos de Unity. Asegúrate de tener Unity Hub instalado también.

  1. Descargar e Instalar Unity Hub: Sigue las instrucciones en el sitio web de Unity para descargar e instalar Unity Hub.
  2. Instalar Unity Editor: Abre Unity Hub, ve a la pestaña "Installs" y haz clic en "Add" para instalar la última versión del Unity Editor.

Creando un Nuevo Proyecto

Una vez que Unity Hub y Unity Editor estén instalados, puedes crear tu primer proyecto.

  1. Abrir Unity Hub: Lanza Unity Hub desde tus aplicaciones.
  2. Crear un Nuevo Proyecto:
    • Haz clic en el botón "New".
    • Elige una plantilla. Para principiantes, recomendamos comenzar con la plantilla "3D".
    • Nombra tu proyecto (por ejemplo, "MiPrimerProyectoUnity").
    • Elige una ubicación para guardar tu proyecto.
    • Haz clic en "Create".

Entendiendo la Interfaz de Unity

Cuando tu proyecto se abra, verás la interfaz del Unity Editor. Aquí están los componentes clave:

  • Vista de Escena: Aquí es donde construyes y organizas tus objetos de juego.
  • Vista de Juego: Esto muestra cómo se verá tu juego cuando se ejecute.
  • Jerarquía: Esto lista todos los objetos de juego en tu escena.
  • Inspector: Esto muestra propiedades y configuraciones para el objeto de juego seleccionado.
  • Proyecto: Aquí es donde se almacenan todos tus recursos (modelos, scripts, texturas, etc.).
  • Consola: Esto muestra mensajes, advertencias y errores.

Añadiendo un Objeto de Juego

Vamos a añadir un objeto de juego simple a tu escena.

  1. Crear un Cubo:

    • Haz clic derecho en la ventana de Jerarquía.
    • Selecciona 3D Object > Cube.
    • Un cubo aparecerá en tu vista de Escena y será listado en la Jerarquía.

  2. Posicionar el Cubo:

    • Selecciona el cubo en la Jerarquía.
    • En el Inspector, establece la Posición en (0, 0, 0).

Escribiendo tu Primer Script

Ahora, vamos a añadir algo de interactividad con un script simple.

  1. Crear un Script:

    • En la ventana de Proyecto, haz clic derecho y selecciona Create > C# Script.
    • Nombra el script MoverCubo.

  2. Editar el Script:

    • Haz doble clic en el script MoverCubo para abrirlo en tu editor de código (por ejemplo, Visual Studio).
    • Reemplaza el código predeterminado con el siguiente:
using UnityEngine;

public class MoverCubo : MonoBehaviour
{
    public float speed = 5f;

    void Update()
    {
        float move = Input.GetAxis("Horizontal") * speed * Time.deltaTime;
        transform.Translate(move, 0, 0);
    }
}
  1. Adjuntar el Script al Cubo:
    • Arrastra el script MoverCubo desde la ventana de Proyecto hasta el cubo en la Jerarquía.

Ejecutando tu Proyecto

  1. Guardar tu Escena:

    • Ve a File > Save Scene.
    • Nombra tu escena (por ejemplo, "EscenaPrincipal").

  2. Jugar el Juego:

    • Haz clic en el botón "Play" en la parte superior del Unity Editor.
    • Usa las teclas de flecha izquierda y derecha para mover el cubo.

Resumen

¡Felicidades! Has creado tu primer proyecto en Unity, añadido un objeto de juego y escrito un script para hacerlo interactivo. Aquí tienes un resumen rápido de lo que hemos cubierto:

  • Configuración de Unity y Unity Hub.
  • Creación de un nuevo proyecto.
  • Entendiendo la interfaz de Unity.
  • Añadiendo y posicionando un objeto de juego.
  • Escribiendo y adjuntando un script simple.
  • Ejecutando tu proyecto.

Este conocimiento fundamental será crucial a medida que continúes explorando y desarrollando proyectos más complejos en Unity. ¡Feliz programación!
