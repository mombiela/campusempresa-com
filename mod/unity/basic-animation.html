Introducción a la Animación en Unity

Unity es un motor de desarrollo de juegos poderoso que permite a los desarrolladores crear animaciones impresionantes. Entender los conceptos básicos de la animación es crucial para dar vida a tus personajes y entornos de juego.

Conceptos Clave

  • Clip de Animación: Un contenedor para datos de animación que se pueden aplicar a GameObjects.
  • Controlador de Animación: Gestiona múltiples clips de animación y las transiciones entre ellos.
  • Fotogramas Clave: Puntos específicos en el tiempo donde defines el valor de una propiedad.
  • Línea de Tiempo: Una representación visual de la secuencia de animación.

Creando Tu Primera Animación

Configurando la Escena

  1. Crea un nuevo proyecto de Unity.
  2. Añade un GameObject (por ejemplo, un cubo) a la escena.
  3. Abre la ventana de Animación (Ventana > Animación > Animación).

Creando un Clip de Animación

  1. Selecciona el GameObject.
  2. Haz clic en "Crear" en la ventana de Animación para crear un nuevo Clip de Animación.
  3. Guarda el Clip de Animación con un nombre descriptivo (por ejemplo, "CubeBounce").

Añadiendo Fotogramas Clave

  1. Mueve el cursor de la línea de tiempo al inicio (0:00).
  2. Modifica las propiedades del GameObject (por ejemplo, posición, rotación, escala).
  3. Haz clic en el botón "Añadir Fotograma Clave" para registrar los valores de las propiedades.
  4. Mueve el cursor de la línea de tiempo a una nueva posición (por ejemplo, 1:00).
  5. Cambia las propiedades del GameObject nuevamente.
  6. Añade otro fotograma clave.

Código de Ejemplo: Moviendo un Cubo Arriba y Abajo

using UnityEngine;

public class CubeBounce : MonoBehaviour
{
    void Update()
    {
        float y = Mathf.PingPong(Time.time, 2);
        transform.position = new Vector3(transform.position.x, y, transform.position.z);
    }
}
  • Explicación:
    • Mathf.PingPong(Time.time, 2) crea un valor que oscila entre 0 y 2.
    • Este valor se asigna a la coordenada y de la posición del cubo, haciéndolo moverse arriba y abajo.

Usando el Controlador de Animación

Creando un Controlador de Animación

  1. Haz clic derecho en la ventana del Proyecto y selecciona Crear > Controlador de Animación.
  2. Nombra el Controlador de Animación (por ejemplo, "CubeAnimator").
  3. Selecciona el GameObject y asigna el Controlador de Animación en el componente Animator.

Añadiendo Clips de Animación al Controlador de Animación

  1. Abre la ventana del Animator (Ventana > Animación > Animator).
  2. Arrastra y suelta los Clips de Animación en la ventana del Animator.
  3. Crea transiciones entre los estados de animación.

Ejemplo: Transicionando Entre Dos Animaciones

using UnityEngine;

public class AnimationSwitcher : MonoBehaviour
{
    private Animator animator;

    void Start()
    {
        animator = GetComponent<Animator>();
    }

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            animator.SetTrigger("Jump");
        }
    }
}
  • Explicación:
    • animator.SetTrigger("Jump") activa una transición a la animación "Jump" cuando se presiona la barra espaciadora.

Técnicas Avanzadas de Animación

Árboles de Mezcla

  • Árboles de Mezcla permiten transiciones suaves entre múltiples animaciones basadas en parámetros.
  • Ejemplo: Mezclando entre animaciones de caminar y correr basadas en la velocidad del personaje.

Eventos de Animación

  • Eventos de Animación pueden llamar funciones en puntos específicos de una animación.
  • Ejemplo: Activar un efecto de sonido cuando el pie de un personaje toca el suelo.

Cinemática Inversa (IK)

  • IK permite animaciones más realistas ajustando automáticamente las posiciones de las extremidades.
  • Ejemplo: Hacer que la mano de un personaje alcance un objeto específico.

Resumen

En este tema, cubrimos los conceptos básicos de la animación en Unity, incluyendo la creación y gestión de Clips de Animación, el uso del Controlador de Animación, e implementación de técnicas avanzadas como Árboles de Mezcla, Eventos de Animación y Cinemática Inversa. Con estas habilidades, puedes crear animaciones dinámicas y atractivas para tus proyectos en Unity.
