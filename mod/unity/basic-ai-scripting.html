Introducción

En esta sección, cubriremos los conceptos básicos de la creación de scripts de IA en Unity. Aprenderás a crear comportamientos de IA simples utilizando las herramientas integradas de Unity y la programación en C#. Al final de esta sección, serás capaz de crear IA básica para personajes en tus proyectos de Unity.

Configuración de tu proyecto

Antes de sumergirnos en la creación de scripts de IA, configuremos un nuevo proyecto de Unity.

  1. Abre Unity Hub y crea un nuevo proyecto 3D.
  2. Nombra tu proyecto "BasicAIScripting" y elige una ubicación para guardarlo.
  3. Haz clic en "Crear" para configurar tu nuevo proyecto.

Conceptos básicos de IA

Entender algunos conceptos fundamentales de IA es crucial antes de comenzar a programar.

  • Agente de IA: Una entidad en el juego que exhibe un comportamiento inteligente.
  • Máquina de estados: Un sistema que gestiona los estados y transiciones de un agente de IA.
  • Malla de navegación (NavMesh): Una malla que define las áreas transitables para los agentes de IA.

Creación de un agente de IA simple

Vamos a crear un agente de IA simple que pueda moverse hacia un objetivo.

Paso 1: Configuración de la escena

  1. Crea un plano que sirva como el suelo.
  2. Añade un objeto 3D (por ejemplo, un cubo) para que actúe como el agente de IA.
  3. Añade otro objeto 3D (por ejemplo, una esfera) para que actúe como el objetivo.

Paso 2: Añadir componentes de NavMesh

  1. Selecciona el plano y ve a Ventana > IA > Navegación.
  2. En la ventana de Navegación, haz clic en la pestaña Hornear y luego haz clic en Hornear para generar la NavMesh.

Paso 3: Escribir el script de IA

Crea un nuevo script en C# llamado SimpleAI y adjúntalo al agente de IA (cubo).

using UnityEngine;
using UnityEngine.AI;

public class SimpleAI : MonoBehaviour
{
    public Transform target; // El objetivo hacia el cual se moverá la IA
    private NavMeshAgent agent;

    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
    }

    void Update()
    {
        if (target != null)
        {
            agent.SetDestination(target.position);
        }
    }
}

Paso 4: Configuración del agente de IA

  1. Selecciona el agente de IA (cubo) y añade un componente NavMeshAgent.
  2. Asigna el objetivo (esfera) al campo target en el script SimpleAI.

Añadiendo una máquina de estados básica

Para hacer nuestra IA más sofisticada, podemos implementar una máquina de estados básica.

Paso 1: Definir estados

Crea un enum para definir diferentes estados.

public enum AIState
{
    Idle,
    Chasing,
    Attacking
}

Paso 2: Implementar la máquina de estados

Actualiza el script SimpleAI para incluir la gestión de estados.

public class SimpleAI : MonoBehaviour
{
    public Transform target;
    private NavMeshAgent agent;
    private AIState currentState;

    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        currentState = AIState.Idle;
    }

    void Update()
    {
        switch (currentState)
        {
            case AIState.Idle:
                // Comportamiento en reposo
                if (Vector3.Distance(transform.position, target.position) < 10f)
                {
                    currentState = AIState.Chasing;
                }
                break;

            case AIState.Chasing:
                // Comportamiento de persecución
                agent.SetDestination(target.position);
                if (Vector3.Distance(transform.position, target.position) < 2f)
                {
                    currentState = AIState.Attacking;
                }
                break;

            case AIState.Attacking:
                // Comportamiento de ataque
                Debug.Log("¡Atacando al objetivo!");
                break;
        }
    }
}

Comportamientos avanzados de IA

Una vez que te sientas cómodo con la creación de scripts de IA básicos, puedes explorar comportamientos más avanzados como:

  • Búsqueda de caminos: Usar la NavMesh de Unity para crear rutas de navegación complejas.
  • Árboles de decisión: Implementar procesos de toma de decisiones para agentes de IA.
  • Árboles de comportamiento: Crear comportamientos de IA más modulares y reutilizables.

Conclusión

En este tema, cubrimos los conceptos básicos de la creación de scripts de IA en Unity. Aprendiste a configurar un agente de IA simple, implementar una máquina de estados básica y explorar algunos comportamientos avanzados de IA. Con estas habilidades, puedes comenzar a crear IA más compleja e inteligente para tus proyectos de Unity. ¡Sigue experimentando y ampliando tu conocimiento para dominar la creación de scripts de IA en Unity!
