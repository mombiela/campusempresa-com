Configuración del Entorno de Desarrollo
Tu Primer Script en PHP ►

Configurar el entorno de desarrollo es un primer paso crucial para aprender PHP. Esta sección te guiará a través del proceso de instalación y configuración de las herramientas necesarias para comenzar a programar en PHP.

Requisitos Previos

Antes de configurar tu entorno de desarrollo PHP, asegúrate de tener lo siguiente:

  • Una computadora con un sistema operativo (Windows, macOS o Linux).
  • Una conexión a internet para descargar el software necesario.

Instalación de PHP

Windows

  1. Descargar PHP:

    • Visita el sitio web oficial de PHP y descarga la última versión de PHP para Windows.

  2. Instalar PHP:

    • Extrae el archivo ZIP descargado en un directorio de tu elección (por ejemplo, C:\php).
    • Agrega el directorio de PHP a la variable de entorno PATH de tu sistema:
      • Abre el Menú de Inicio, busca "Variables de Entorno" y selecciona "Editar las variables de entorno del sistema."
      • Haz clic en "Variables de Entorno" y encuentra la variable "Path" bajo Variables del sistema.
      • Haz clic en "Editar," luego en "Nuevo," y agrega la ruta a tu directorio de PHP (por ejemplo, C:\php).
    • Abre el Símbolo del sistema y escribe php -v para verificar la instalación.

macOS

  1. Usando Homebrew:
    • Abre Terminal e instala Homebrew si aún no lo has hecho: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instala PHP usando Homebrew: 
      brew install php
    • Verifica la instalación escribiendo php -v en Terminal.

Linux

  1. Usando el Gestor de Paquetes:
    • Abre Terminal y actualiza tu lista de paquetes: 
      sudo apt update
    • Instala PHP: 
      sudo apt install php
    • Verifica la instalación escribiendo php -v en Terminal.

Instalación de un Servidor Web

PHP funciona mejor con un servidor web. La elección más común es Apache, pero también puedes usar Nginx u otros servidores. Para simplificar, usaremos XAMPP, que incluye Apache, MySQL y PHP.

Instalación de XAMPP

  1. Descargar XAMPP:

    • Visita el sitio web de XAMPP y descarga la versión adecuada para tu sistema operativo.

  2. Instalar XAMPP:

    • Sigue las instrucciones de instalación para tu sistema operativo.
    • Inicia el Panel de Control de XAMPP y arranca los módulos Apache y MySQL.

  3. Verificar la Instalación:

    • Abre tu navegador web y ve a http://localhost. Deberías ver la página de bienvenida de XAMPP.

Configuración de un Editor de Código

Un buen editor de código puede mejorar significativamente tu productividad. Algunas opciones populares para el desarrollo en PHP son:

  • Visual Studio Code:

    • Descárgalo desde Visual Studio Code.
    • Instala extensiones de PHP como "PHP Intelephense" para una mejor autocompletación y análisis de código.

  • PHPStorm:

    • Un potente IDE diseñado específicamente para el desarrollo en PHP.
    • Descárgalo desde JetBrains.

  • Sublime Text:

    • Descárgalo desde Sublime Text.
    • Instala paquetes de PHP a través de Package Control para una funcionalidad mejorada.

Configuración del Entorno de Desarrollo

Configuración de PHP

  1. Archivo php.ini:
    • Localiza tu archivo php.ini (normalmente en el directorio de PHP o en el directorio de instalación de XAMPP).
    • Ábrelo en tu editor de código y configura ajustes como:
      • error_reporting = E_ALL
      • display_errors = On
      • max_execution_time = 30

Configuración de Apache

  1. Archivo httpd.conf:
    • Localiza el archivo httpd.conf en el directorio de Apache (por ejemplo, C:\xampp\apache\conf).
    • Ábrelo en tu editor de código y asegúrate de que las siguientes líneas estén descomentadas: 
      LoadModule php_module "C:/php/php7apache2_4.dll"
AddType application/x-httpd-php .php
PHPIniDir "C:/php"

Prueba de la Configuración

  1. Crear un Archivo PHP:
    • En el directorio raíz de tu servidor web (por ejemplo, C:\xampp\htdocs), crea un archivo llamado info.php.
    • Agrega el siguiente código: 
      <?php
phpinfo();
?>
    • Guarda el archivo y abre tu navegador web. Navega a http://localhost/info.php. Deberías ver la página de información de PHP.

Conclusión

Configurar tu entorno de desarrollo PHP implica instalar PHP, un servidor web y un editor de código. Una configuración adecuada asegura que puedas escribir, probar y depurar tu código PHP de manera eficiente. Con tu entorno listo, ahora estás preparado para sumergirte en la programación en PHP y comenzar a construir aplicaciones web dinámicas.
