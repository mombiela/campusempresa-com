Conectarse a una Base de Datos MySQL
Realización de Operaciones CRUD ►

Conectarse a una base de datos MySQL es una habilidad fundamental para cualquier desarrollador de PHP. Este tema te guiará a través del proceso, desde configurar tu entorno hasta ejecutar consultas básicas.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una instalación funcional de PHP.
  • Un servidor MySQL en funcionamiento y accesible.
  • Conocimientos básicos de la sintaxis de PHP.

Configuración del Entorno

Para conectarte a una base de datos MySQL usando PHP, necesitas tener habilitada la extensión mysqli. Esta extensión está incluida en PHP por defecto, pero debes verificar que esté habilitada en tu archivo de configuración php.ini.

; Asegúrate de que la siguiente línea esté descomentada en php.ini
extension=mysqli

Estableciendo una Conexión

Usando `mysqli`

La extensión mysqli proporciona una manera simple de conectarse a una base de datos MySQL. Así es como puedes hacerlo:

  1. Crear una Conexión: Usa la función mysqli_connect para establecer una conexión.
  2. Verificar la Conexión: Verifica si la conexión fue exitosa.
<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "test_db";

// Crear conexión
$conn = mysqli_connect($servername, $username, $password, $dbname);

// Verificar conexión
if (!$conn) {
    die("Conexión fallida: " . mysqli_connect_error());
}
echo "Conectado exitosamente";
?>

Usando PDO (PHP Data Objects)

PDO es una forma más flexible y segura de conectarse a una base de datos. Soporta múltiples tipos de bases de datos y proporciona una interfaz consistente.

  1. Crear una Instancia PDO: Usa el constructor new PDO.
  2. Manejar Excepciones: Usa bloques try-catch para manejar errores de conexión.
<?php
$dsn = 'mysql:host=localhost;dbname=test_db';
$username = 'root';
$password = '';

try {
    $pdo = new PDO($dsn, $username, $password);
    // Establecer el modo de error de PDO a excepción
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Conectado exitosamente";
} catch (PDOException $e) {
    echo "Conexión fallida: " . $e->getMessage();
}
?>

Ejecutando Consultas

Usando `mysqli`

Una vez conectado, puedes ejecutar consultas SQL usando la función mysqli_query.

<?php
$sql = "SELECT id, firstname, lastname FROM Users";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    // Salida de datos de cada fila
    while($row = mysqli_fetch_assoc($result)) {
        echo "id: " . $row["id"]. " - Nombre: " . $row["firstname"]. " " . $row["lastname"]. "<br>";
    }
} else {
    echo "0 resultados";
}
mysqli_close($conn);
?>

Usando PDO

Con PDO, puedes preparar y ejecutar sentencias, lo cual es más seguro y eficiente.

<?php
$sql = 'SELECT id, firstname, lastname FROM Users';
$stmt = $pdo->query($sql);

while ($row = $stmt->fetch(PDO::FETCH_ASSOC)) {
    echo "id: " . $row['id'] . " - Nombre: " . $row['firstname'] . " " . $row['lastname'] . "<br>";
}
?>

Manejo de Errores

`mysqli`

Verifica errores después de ejecutar consultas.

<?php
if (!mysqli_query($conn, $sql)) {
    echo "Error: " . $sql . "<br>" . mysqli_error($conn);
}
?>

PDO

Usa excepciones para manejar errores.

<?php
try {
    $stmt = $pdo->query($sql);
} catch (PDOException $e) {
    echo "Consulta fallida: " . $e->getMessage();
}
?>

Cerrando la Conexión

`mysqli`

Siempre cierra la conexión cuando termines.

<?php
mysqli_close($conn);
?>

PDO

Las conexiones PDO se cierran automáticamente cuando el objeto es destruido, pero puedes establecer explícitamente el objeto PDO a null.

<?php
$pdo = null;
?>

Resumen

Conectarse a una base de datos MySQL en PHP se puede hacer usando la extensión mysqli o PDO. Mientras que mysqli es directo y fácil de usar, PDO ofrece más flexibilidad y seguridad, especialmente para proyectos más grandes. Siempre asegúrate de manejar los errores de manera adecuada y cerrar tus conexiones cuando ya no sean necesarias.
