Introducción

En esta sección, exploraremos cómo implementar varias funcionalidades en PHP, desde niveles básicos hasta avanzados. Al final de esta sección, tendrás una comprensión sólida de cómo crear aplicaciones PHP funcionales y dinámicas.

Funcionalidades Básicas

1. Manejo de Formularios

El manejo de formularios es un aspecto fundamental del desarrollo web. PHP facilita la recopilación, validación y procesamiento de datos de formularios.

Ejemplo: Formulario de Contacto Simple

<!DOCTYPE html>
<html>
<head>
    <title>Formulario de Contacto</title>
</head>
<body>
    <form method="post" action="process_form.php">
        Nombre: <input type="text" name="name"><br>
        Correo Electrónico: <input type="email" name="email"><br>
        Mensaje: <textarea name="message"></textarea><br>
        <input type="submit" value="Enviar">
    </form>
</body>
</html>

Procesamiento de los Datos del Formulario

<?php
// process_form.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = htmlspecialchars($_POST['name']);
    $email = htmlspecialchars($_POST['email']);
    $message = htmlspecialchars($_POST['message']);

    // Validación básica
    if (!empty($name) && !empty($email) && !empty($message)) {
        echo "Gracias, $name. Hemos recibido tu mensaje.";
    } else {
        echo "Todos los campos son obligatorios.";
    }
}
?>

2. Subida de Archivos

Subir archivos es otra funcionalidad común en aplicaciones web. PHP proporciona funciones integradas para manejar la subida de archivos de manera segura.

Ejemplo: Formulario de Subida de Archivos

<!DOCTYPE html>
<html>
<head>
    <title>Subida de Archivos</title>
</head>
<body>
    <form method="post" enctype="multipart/form-data" action="upload.php">
        Seleccionar archivo: <input type="file" name="fileToUpload"><br>
        <input type="submit" value="Subir Archivo">
    </form>
</body>
</html>

Procesamiento de la Subida de Archivos

<?php
// upload.php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$fileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

// Verificar si el archivo ya existe
if (file_exists($target_file)) {
    echo "Lo siento, el archivo ya existe.";
    $uploadOk = 0;
}

// Verificar el tamaño del archivo
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Lo siento, tu archivo es demasiado grande.";
    $uploadOk = 0;
}

// Permitir ciertos formatos de archivo
if ($fileType != "jpg" && $fileType != "png" && $fileType != "jpeg" && $fileType != "gif") {
    echo "Lo siento, solo se permiten archivos JPG, JPEG, PNG y GIF.";
    $uploadOk = 0;
}

// Verificar si $uploadOk está configurado a 0 por un error
if ($uploadOk == 0) {
    echo "Lo siento, tu archivo no fue subido.";
} else {
    if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
        echo "El archivo ". htmlspecialchars(basename($_FILES["fileToUpload"]["name"])). " ha sido subido.";
    } else {
        echo "Lo siento, hubo un error subiendo tu archivo.";
    }
}
?>

Funcionalidades Intermedias

1. Autenticación de Usuarios

La autenticación de usuarios es crucial para asegurar aplicaciones web. PHP puede ser utilizado para crear un sistema de inicio de sesión simple.

Ejemplo: Formulario de Inicio de Sesión

<!DOCTYPE html>
<html>
<head>
    <title>Inicio de Sesión</title>
</head>
<body>
    <form method="post" action="login.php">
        Nombre de Usuario: <input type="text" name="username"><br>
        Contraseña: <input type="password" name="password"><br>
        <input type="submit" value="Iniciar Sesión">
    </form>
</body>
</html>

Procesamiento del Inicio de Sesión

<?php
// login.php
session_start();
$users = [
    'admin' => 'password123',
    'user' => 'userpass'
];

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    $password = $_POST['password'];

    if (isset($users[$username]) && $users[$username] == $password) {
        $_SESSION['username'] = $username;
        echo "Bienvenido, $username!";
    } else {
        echo "Nombre de usuario o contraseña inválidos.";
    }
}
?>

2. Interacción con Bases de Datos

Interactuar con una base de datos es esencial para aplicaciones web dinámicas. PHP proporciona varias maneras de conectar e interactuar con bases de datos, como MySQL.

Ejemplo: Conectando a una Base de Datos MySQL

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}
echo "Conectado exitosamente";
?>

Ejemplo: Insertando Datos en una Base de Datos

<?php
// Asumiendo que la conexión está establecida como se muestra arriba
$sql = "INSERT INTO Users (username, email, password) VALUES ('john_doe', 'john@example.com', 'securepassword')";

if ($conn->query($sql) === TRUE) {
    echo "Nuevo registro creado exitosamente";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>

Funcionalidades Avanzadas

1. Implementación de APIs RESTful

Las APIs RESTful permiten que diferentes aplicaciones se comuniquen entre sí. PHP puede ser utilizado para crear y consumir APIs RESTful.

Ejemplo: API RESTful Simple

<?php
header("Content-Type: application/json");

$method = $_SERVER['REQUEST_METHOD'];

switch ($method) {
    case 'GET':
        echo json_encode(["message" => "Solicitud GET recibida"]);
        break;
    case 'POST':
        echo json_encode(["message" => "Solicitud POST recibida"]);
        break;
    default:
        echo json_encode(["message" => "Método de solicitud no soportado"]);
        break;
}
?>

2. Uso de Frameworks

Frameworks de PHP como Laravel y Symfony pueden acelerar significativamente el desarrollo y fomentar las mejores prácticas.

Ejemplo: Ruta Básica en Laravel

// routes/web.php
Route::get('/', function () {
    return view('welcome');
});

Ejemplo: Controlador Básico en Laravel

// app/Http/Controllers/ExampleController.php
namespace App\Http\Controllers;

use Illuminate\Http\Request;

class ExampleController extends Controller
{
    public function index()
    {
        return view('example');
    }
}

Conclusión

Implementar funcionalidades en PHP abarca desde manejar formularios simples hasta crear complejas APIs RESTful y usar frameworks. Al dominar estas funcionalidades, puedes construir aplicaciones web robustas y dinámicas. Practica cada ejemplo y experimenta con variaciones para profundizar tu comprensión.
