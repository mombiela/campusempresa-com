Realización de Operaciones CRUD
Uso de Sentencias Preparadas ►

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las cuatro operaciones básicas para gestionar datos en una base de datos. En esta sección, aprenderemos cómo realizar estas operaciones usando PHP y MySQL.

Prerrequisitos

Antes de sumergirnos en las operaciones CRUD, asegúrate de tener:

  • Un entendimiento básico de la sintaxis de PHP.
  • Un entorno de servidor local (por ejemplo, XAMPP, WAMP o MAMP).
  • MySQL instalado y en funcionamiento.
  • Una base de datos y una tabla con las que trabajar.

Configuración de la Base de Datos

Primero, vamos a crear una base de datos y una tabla para realizar nuestras operaciones CRUD.

CREATE DATABASE crud_example;
USE crud_example;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Conectando a la Base de Datos

Para interactuar con la base de datos, necesitamos establecer una conexión usando PHP.

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "crud_example";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}
echo "Conectado exitosamente";
?>

Operación de Creación

Para insertar datos en la base de datos, usamos la declaración SQL INSERT INTO.

<?php
$name = "John Doe";
$email = "john.doe@example.com";

$sql = "INSERT INTO users (name, email) VALUES ('$name', '$email')";

if ($conn->query($sql) === TRUE) {
    echo "Nuevo registro creado exitosamente";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}
?>

Explicación

  • $name y $email son los datos que queremos insertar.
  • La declaración INSERT INTO especifica la tabla (users) y las columnas (name, email) donde se insertarán los datos.
  • $conn->query($sql) ejecuta la declaración SQL.

Operación de Lectura

Para recuperar datos de la base de datos, usamos la declaración SQL SELECT.

<?php
$sql = "SELECT id, name, email, created_at FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    // Salida de datos de cada fila
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Nombre: " . $row["name"]. " - Email: " . $row["email"]. " - Creado en: " . $row["created_at"]. "<br>";
    }
} else {
    echo "0 resultados";
}
?>

Explicación

  • La declaración SELECT especifica las columnas que queremos recuperar.
  • $result = $conn->query($sql) ejecuta la declaración SQL.
  • $result->num_rows verifica si hay filas devueltas.
  • while($row = $result->fetch_assoc()) recorre el conjunto de resultados y obtiene cada fila como un array asociativo.

Operación de Actualización

Para actualizar datos existentes, usamos la declaración SQL UPDATE.

<?php
$id = 1;
$newEmail = "new.email@example.com";

$sql = "UPDATE users SET email='$newEmail' WHERE id=$id";

if ($conn->query($sql) === TRUE) {
    echo "Registro actualizado exitosamente";
} else {
    echo "Error actualizando el registro: " . $conn->error;
}
?>

Explicación

  • $id es el identificador del registro que queremos actualizar.
  • $newEmail es el nuevo valor para la columna de email.
  • La declaración UPDATE especifica la tabla (users), el nuevo valor (email='$newEmail') y la condición (WHERE id=$id).

Operación de Eliminación

Para eliminar datos de la base de datos, usamos la declaración SQL DELETE.

<?php
$id = 1;

$sql = "DELETE FROM users WHERE id=$id";

if ($conn->query($sql) === TRUE) {
    echo "Registro eliminado exitosamente";
} else {
    echo "Error eliminando el registro: " . $conn->error;
}
?>

Explicación

  • $id es el identificador del registro que queremos eliminar.
  • La declaración DELETE especifica la tabla (users) y la condición (WHERE id=$id).

Conclusión

En esta sección, cubrimos los conceptos básicos de realizar operaciones CRUD usando PHP y MySQL:

  • Crear: Insertar nuevos registros en la base de datos.
  • Leer: Recuperar registros de la base de datos.
  • Actualizar: Modificar registros existentes en la base de datos.
  • Eliminar: Eliminar registros de la base de datos.

Entender estas operaciones es fundamental para gestionar datos en cualquier aplicación web. Practica estos ejemplos y experimenta con diferentes consultas para profundizar tu comprensión.
