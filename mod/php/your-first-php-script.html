Introducción a PHP

PHP (Hypertext Preprocessor) es un popular lenguaje de scripting del lado del servidor diseñado para el desarrollo web, pero también se utiliza como un lenguaje de programación de propósito general. En esta sección, te guiaremos a través de la escritura de tu primer script en PHP.

Configurando Tu Entorno

Antes de escribir tu primer script en PHP, asegúrate de tener lo siguiente:

  • Un servidor web (por ejemplo, Apache, Nginx)
  • PHP instalado en tu servidor
  • Un editor de texto (por ejemplo, VS Code, Sublime Text)

Instalando PHP

Si aún no has instalado PHP, puedes descargarlo desde el sitio web oficial de PHP. Sigue las instrucciones de instalación para tu sistema operativo.

Escribiendo Tu Primer Script en PHP

Vamos a sumergirnos en la escritura de un simple script en PHP.

Creando un Archivo PHP

  1. Abre tu editor de texto.
  2. Crea un nuevo archivo y guárdalo como index.php.

Sintaxis Básica de PHP

El código PHP se incrusta dentro de HTML usando las etiquetas <?php ... ?>.

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primer Script en PHP</title>
</head>
<body>
    <?php
        echo "¡Hola, Mundo!";
    ?>
</body>
</html>

Explicación

  • <!DOCTYPE html>: Declara el tipo de documento como HTML5.
  • <html>, <head>, <title>, <body>: Etiquetas estándar de HTML.
  • <?php ... ?>: Bloque de código PHP.
  • echo "¡Hola, Mundo!";: Declaración de PHP para mostrar texto.

Ejecutando Tu Script en PHP

  1. Coloca index.php en el directorio raíz de tu servidor web (por ejemplo, htdocs para XAMPP, www para WAMP).
  2. Inicia tu servidor web.
  3. Abre un navegador web y navega a http://localhost/index.php.

Deberías ver "¡Hola, Mundo!" mostrado en la página.

Conceptos Clave

  • Etiquetas PHP: El código PHP se escribe dentro de las etiquetas <?php ... ?>.
  • Declaración Echo: Se usa para mostrar texto o variables.
  • Comentarios: Usa // para comentarios de una sola línea y /* ... */ para comentarios de varias líneas.

Ejemplo con Comentarios

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primer Script en PHP con Comentarios</title>
</head>
<body>
    <?php
        // Este es un comentario de una sola línea
        echo "¡Hola, Mundo!"; // Esto mostrará ¡Hola, Mundo!
        
        /* Este es un comentario de varias líneas
           Puede abarcar varias líneas */
        echo "¡Aprender PHP es divertido!";
    ?>
</body>
</html>

Errores Comunes y Solución de Problemas

  • Errores de Sintaxis: Asegúrate de que todas las declaraciones de PHP terminen con un punto y coma (;).
  • Configuración del Servidor: Asegúrate de que tu servidor web esté en funcionamiento y PHP esté correctamente instalado.
  • Ubicación del Archivo: Asegúrate de que tu archivo PHP esté en el directorio correcto para tu servidor web.

Resumen

En esta sección, aprendiste a:

  • Configurar tu entorno para el desarrollo en PHP.
  • Escribir y ejecutar un script básico en PHP.
  • Entender la sintaxis básica de PHP y conceptos clave.

Al dominar estos conceptos básicos, estás en buen camino para convertirte en un experto en PHP. A continuación, exploraremos temas y funcionalidades más avanzadas en PHP.
