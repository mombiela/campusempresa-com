Introducción a las Funciones de Cadenas

Las funciones de cadenas en PHP son herramientas esenciales para manipular y gestionar texto. Te permiten realizar una variedad de operaciones como buscar, reemplazar, formatear y dividir cadenas. Esta sección te guiará a través de las funciones de cadenas más comúnmente usadas en PHP, desde niveles básicos hasta avanzados.

Funciones Básicas de Cadenas

strlen()

  • Descripción: Devuelve la longitud de una cadena.
  • Sintaxis: strlen(string $string): int
  • Ejemplo: 
    $text = "¡Hola, Mundo!";
echo strlen($text); // Salida: 13

strpos()

  • Descripción: Encuentra la posición de la primera ocurrencia de una subcadena en una cadena.
  • Sintaxis: strpos(string $haystack, string $needle, int $offset = 0): int|false
  • Ejemplo: 
    $text = "¡Hola, Mundo!";
echo strpos($text, "Mundo"); // Salida: 7

substr()

  • Descripción: Devuelve una parte de una cadena.
  • Sintaxis: substr(string $string, int $start, int $length = null): string
  • Ejemplo: 
    $text = "¡Hola, Mundo!";
echo substr($text, 7, 5); // Salida: Mundo

strtoupper() y strtolower()

  • Descripción: Convierte una cadena a mayúsculas o minúsculas.
  • Sintaxis:
    • strtoupper(string $string): string
    • strtolower(string $string): string
  • Ejemplo: 
    $text = "¡Hola, Mundo!";
echo strtoupper($text); // Salida: ¡HOLA, MUNDO!
echo strtolower($text); // Salida: ¡hola, mundo!

Funciones Intermedias de Cadenas

str_replace()

  • Descripción: Reemplaza todas las ocurrencias de la cadena de búsqueda con la cadena de reemplazo.
  • Sintaxis: str_replace(mixed $search, mixed $replace, mixed $subject, int &$count = null): mixed
  • Ejemplo: 
    $text = "¡Hola, Mundo!";
echo str_replace("Mundo", "PHP", $text); // Salida: ¡Hola, PHP!

explode() y implode()

  • Descripción:
    • explode(): Divide una cadena por un delimitador especificado en un array.
    • implode(): Une elementos de un array en una cadena con un delimitador especificado.
  • Sintaxis:
    • explode(string $delimiter, string $string, int $limit = PHP_INT_MAX): array
    • implode(string $glue, array $pieces): string
  • Ejemplo: 
    $text = "manzana,plátano,cereza";
$array = explode(",", $text);
print_r($array); // Salida: Array ( [0] => manzana [1] => plátano [2] => cereza )

$array = ["manzana", "plátano", "cereza"];
echo implode(", ", $array); // Salida: manzana, plátano, cereza

trim()

  • Descripción: Elimina espacios en blanco (u otros caracteres) del inicio y final de una cadena.
  • Sintaxis: trim(string $string, string $characters = " \n\r\t\v\x00"): string
  • Ejemplo: 
    $text = "  ¡Hola, Mundo!  ";
echo trim($text); // Salida: ¡Hola, Mundo!

Funciones Avanzadas de Cadenas

preg_match() y preg_replace()

  • Descripción:
    • preg_match(): Realiza una coincidencia de expresión regular.
    • preg_replace(): Realiza una búsqueda y reemplazo de expresión regular.
  • Sintaxis:
    • preg_match(string $pattern, string $subject, array &$matches = null, int $flags = 0, int $offset = 0): int
    • preg_replace(mixed $pattern, mixed $replacement, mixed $subject, int $limit = -1, int &$count = null): mixed
  • Ejemplo: 
    $text = "El rápido zorro marrón salta sobre el perro perezoso.";
if (preg_match("/rápido/", $text)) {
    echo "¡Coincidencia encontrada!"; // Salida: ¡Coincidencia encontrada!
}

$text = "El rápido zorro marrón salta sobre el perro perezoso.";
echo preg_replace("/rápido/", "lento", $text); // Salida: El lento zorro marrón salta sobre el perro perezoso.

sprintf()

  • Descripción: Devuelve una cadena formateada.
  • Sintaxis: sprintf(string $format, mixed ...$values): string
  • Ejemplo: 
    $number = 9;
$text = "Beetlejuice";
echo sprintf("Hay %d películas de %s.", $number, $text); // Salida: Hay 9 películas de Beetlejuice.

htmlentities() y htmlspecialchars()

  • Descripción: Convierte caracteres a entidades HTML.
  • Sintaxis:
    • htmlentities(string $string, int $flags = ENT_COMPAT, string $encoding = ini_get("default_charset"), bool $double_encode = true): string
    • htmlspecialchars(string $string, int $flags = ENT_COMPAT, string $encoding = ini_get("default_charset"), bool $double_encode = true): string
  • Ejemplo: 
    $text = "<a href='test'>Test</a>";
echo htmlentities($text); // Salida: &lt;a href=&#039;test&#039;&gt;Test&lt;/a&gt;
echo htmlspecialchars($text); // Salida: &lt;a href='test'&gt;Test&lt;/a&gt;

Conclusión

Las funciones de cadenas en PHP son herramientas poderosas que te permiten manipular y gestionar texto de manera eficiente. Desde funciones básicas como strlen() y strpos() hasta funciones más avanzadas como preg_match() y sprintf(), entender estas funciones te permitirá manejar cadenas de manera efectiva en tus aplicaciones PHP. Practica usando estas funciones con diferentes ejemplos para volverte competente en la manipulación de cadenas en PHP.
