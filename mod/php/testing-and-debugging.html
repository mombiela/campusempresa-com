Pruebas y Depuración en PHP
Control de Versiones con Git ►

Las pruebas y la depuración son habilidades cruciales para cualquier desarrollador de PHP. Esta sección te guiará a través de los conceptos básicos de las pruebas y la depuración, desde niveles principiantes hasta avanzados.

Introducción a las Pruebas

Las pruebas son el proceso de evaluar tu código para asegurarte de que se comporte como se espera. Ayuda a identificar errores y verificar que tu aplicación cumpla con sus requisitos.

Conceptos Clave en las Pruebas

  • Pruebas Unitarias: Pruebas de unidades o componentes individuales de un software.
  • Pruebas de Integración: Pruebas de la interacción entre diferentes unidades o componentes.
  • Pruebas Funcionales: Pruebas de la funcionalidad completa de una aplicación.
  • Desarrollo Guiado por Pruebas (TDD): Escribir pruebas antes de escribir el código real.

Por Qué las Pruebas son Importantes

  • Asegura la fiabilidad y calidad del código.
  • Ayuda a identificar errores temprano.
  • Facilita el mantenimiento y la refactorización del código.

Introducción a la Depuración

La depuración es el proceso de identificar, analizar y eliminar errores de tu código. Es una habilidad esencial para cualquier desarrollador para asegurar el funcionamiento correcto de una aplicación.

Conceptos Clave en la Depuración

  • Reporte de Errores: Mecanismo incorporado de PHP para reportar errores.
  • Registro: Registro de mensajes de error y otra información en un archivo de registro.
  • Puntos de Interrupción: Pausar la ejecución del código en un punto específico para inspeccionar el estado.
  • Ejecución Paso a Paso: Ejecutar el código línea por línea para entender su comportamiento.

Nivel Principiante

Pruebas Básicas con Aserciones

PHP proporciona formas simples de realizar pruebas básicas usando aserciones.

<?php
// Aserción básica
assert(1 + 1 == 2);

// Aserción personalizada
function test_addition($a, $b, $expected) {
    assert($a + $b == $expected);
}

test_addition(2, 3, 5);
?>

Depuración Básica con `var_dump` y `print_r`

Usar var_dump y print_r para inspeccionar variables.

<?php
$variable = array("foo" => "bar", 12 => true);
var_dump($variable);
print_r($variable);
?>

Reporte de Errores

Habilitar el reporte de errores para ver errores en tu código PHP.

<?php
error_reporting(E_ALL);
ini_set('display_errors', 1);

// Error intencional
echo $undefined_variable;
?>

Nivel Intermedio

Pruebas Unitarias con PHPUnit

PHPUnit es un marco de pruebas popular para PHP.

Instalación

composer require --dev phpunit/phpunit

Caso de Prueba Básico

<?php
use PHPUnit\Framework\TestCase;

class MathTest extends TestCase {
    public function testAddition() {
        $this->assertEquals(4, 2 + 2);
    }
}
?>

Depuración con Xdebug

Xdebug es una herramienta poderosa para depurar aplicaciones PHP.

Instalación

pecl install xdebug

Configuración

Añade las siguientes líneas a tu archivo php.ini:

zend_extension=xdebug.so
xdebug.remote_enable=1
xdebug.remote_host=127.0.0.1
xdebug.remote_port=9000

Uso de Puntos de Interrupción

Establece puntos de interrupción en tu IDE (por ejemplo, PhpStorm) y ejecuta tu aplicación. La ejecución se pausará en los puntos de interrupción, permitiéndote inspeccionar variables y la pila de llamadas.

Nivel Avanzado

Desarrollo Guiado por Pruebas (TDD)

TDD es un proceso de desarrollo donde escribes pruebas antes de escribir el código real.

Ejemplo de Flujo de Trabajo

  1. Escribe una prueba que falle.
  2. Escribe el código mínimo para pasar la prueba.
  3. Refactoriza el código asegurándote de que la prueba aún pase.
<?php
use PHPUnit\Framework\TestCase;

class CalculatorTest extends TestCase {
    public function testAddition() {
        $calculator = new Calculator();
        $this->assertEquals(5, $calculator->add(2, 3));
    }
}

class Calculator {
    public function add($a, $b) {
        return $a + $b;
    }
}
?>

Técnicas Avanzadas de Depuración

Perfilado con Xdebug

El perfilado ayuda a identificar cuellos de botella de rendimiento en tu aplicación.

xdebug.profiler_enable=1
xdebug.profiler_output_dir=/tmp

Depuración Remota

Configura Xdebug para la depuración remota para depurar aplicaciones que se ejecutan en un servidor.

xdebug.remote_enable=1
xdebug.remote_host=your.remote.host
xdebug.remote_port=9000

Conclusión

Las pruebas y la depuración son habilidades esenciales para los desarrolladores de PHP. Al dominar estas técnicas, puedes asegurarte de que tu código sea fiable, mantenible y tenga un buen rendimiento. Comienza con pruebas y depuración básicas, y gradualmente avanza hacia técnicas más avanzadas como TDD y perfilado para convertirte en un desarrollador de PHP competente.
