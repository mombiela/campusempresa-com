Introducción a la Gestión de Estado
Uso de la API de Contexto para la Gestión de Estado ►

La gestión de estado es un concepto crucial en el desarrollo con React. Te permite gestionar y sincronizar el estado de tu aplicación de manera eficiente. Este tema te guiará a través de los conceptos básicos de la gestión de estado, desde entender qué es el estado hasta implementar técnicas avanzadas de gestión de estado.

¿Qué es el Estado?

El estado se refiere a los datos que un componente mantiene. Puede cambiar con el tiempo y afectar cómo se renderiza y se comporta el componente.

  • Estado Local: Gestionado dentro de un solo componente.
  • Estado Global: Compartido entre múltiples componentes.

Gestión de Estado en React

Usando el Hook useState

El hook useState es la forma más simple de gestionar el estado local en un componente funcional.

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Has hecho clic {count} veces</p>
      <button onClick={() => setCount(count + 1)}>
        Haz clic en mí
      </button>
    </div>
  );
}
  • useState: Inicializa el estado.
  • setCount: Actualiza el estado.

Usando el Hook useReducer

Para lógica de estado más compleja, useReducer puede ser una mejor opción.

import React, { useReducer } from 'react';

const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}

function Counter() {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Cuenta: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        +
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        -
      </button>
    </div>
  );
}
  • useReducer: Gestiona lógica de estado compleja.
  • dispatch: Envía acciones a la función reductora.

Gestión de Estado Global

API de Contexto

La API de Contexto te permite compartir estado entre múltiples componentes sin pasar props manualmente en cada nivel.

import React, { createContext, useContext, useState } from 'react';

const CountContext = createContext();

function CounterProvider({ children }) {
  const [count, setCount] = useState(0);

  return (
    <CountContext.Provider value={{ count, setCount }}>
      {children}
    </CountContext.Provider>
  );
}

function Counter() {
  const { count, setCount } = useContext(CountContext);

  return (
    <div>
      <p>Cuenta: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Incrementar
      </button>
    </div>
  );
}

function App() {
  return (
    <CounterProvider>
      <Counter />
    </CounterProvider>
  );
}
  • createContext: Crea un objeto de contexto.
  • useContext: Consume el valor del contexto.

Redux

Redux es una biblioteca popular para gestionar el estado global en aplicaciones más grandes.

// store.js
import { createStore } from 'redux';

const initialState = { count: 0 };

function counterReducer(state = initialState, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

const store = createStore(counterReducer);

export default store;

// App.js
import React from 'react';
import { Provider, useDispatch, useSelector } from 'react-redux';
import store from './store';

function Counter() {
  const count = useSelector(state => state.count);
  const dispatch = useDispatch();

  return (
    <div>
      <p>Cuenta: {count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        Incrementar
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        Decrementar
      </button>
    </div>
  );
}

function App() {
  return (
    <Provider store={store}>
      <Counter />
    </Provider>
  );
}

export default App;
  • createStore: Crea una tienda de Redux.
  • Provider: Hace que la tienda de Redux esté disponible para la aplicación.
  • useSelector: Accede al estado desde la tienda de Redux.
  • useDispatch: Envía acciones a la tienda de Redux.

Conclusión

La gestión de estado es un aspecto fundamental del desarrollo con React. Entender cómo gestionar tanto el estado local como el global de manera eficiente puede mejorar enormemente el rendimiento y la mantenibilidad de tus aplicaciones. Comienza con los conceptos básicos usando useState y useReducer, y luego explora técnicas más avanzadas como la API de Contexto y Redux para aplicaciones más grandes.
