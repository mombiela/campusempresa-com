Desplegando Aplicaciones React
Mejores Prácticas y Estándares de Codificación ►

Desplegar una aplicación React implica preparar tu aplicación para producción, asegurarte de que funcione eficientemente y hacerla accesible para los usuarios. Este tema te guiará a través de los pasos necesarios para desplegar una aplicación React, desde la configuración básica hasta estrategias avanzadas de despliegue.

Preparando Tu Aplicación React para el Despliegue

Antes de desplegar, necesitas asegurarte de que tu aplicación React esté lista para producción.

Construir Tu Aplicación

  • Crear una construcción de producción: React proporciona un comando incorporado para crear una construcción optimizada de producción de tu aplicación.
npm run build
  • Entender la carpeta de construcción: La carpeta build contiene los archivos estáticos que pueden ser servidos por un servidor web. Esto incluye:
    • index.html
    • Archivos JavaScript minificados
    • Imágenes optimizadas

Variables de Entorno

  • Configurar variables de entorno: Usa archivos .env para gestionar variables específicas del entorno.
REACT_APP_API_URL=https://api.example.com
  • Acceder a las variables de entorno en tu código:
const apiUrl = process.env.REACT_APP_API_URL;

Desplegando en Hosts de Sitios Estáticos

GitHub Pages

  • Instalar el paquete gh-pages:
npm install --save gh-pages
  • Agregar scripts de despliegue a package.json:
"homepage": "https://tuusuario.github.io/tu-repo-nombre",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d build"
}
  • Desplegar tu aplicación:
npm run deploy

Netlify

  • Conectar tu repositorio: Vincula tu repositorio de GitHub a Netlify.
  • Configurar ajustes de construcción:
    • Comando de construcción: npm run build
    • Directorio de publicación: build
  • Desplegar: Netlify construirá y desplegará automáticamente tu sitio.

Vercel

  • Instalar Vercel CLI:
npm install -g vercel
  • Desplegar tu aplicación:
vercel
  • Configurar ajustes: Sigue las indicaciones para configurar tu proyecto.

Desplegando en Proveedores de Nube

AWS S3 y CloudFront

  • Crear un bucket S3: Asegúrate de que el bucket sea público.
  • Subir la carpeta de construcción: Usa la CLI de AWS o la consola de S3.
  • Configurar CloudFront: Configura una distribución de CloudFront para servir tu contenido de S3.

Google Cloud Platform (GCP)

  • Instalar el SDK de Google Cloud: Sigue las instrucciones en el sitio web de GCP.

  • Desplegar en Firebase Hosting:

    • Instalar Firebase CLI:
    npm install -g firebase-tools
    • Inicializar Firebase:
    firebase init
    • Desplegar tu aplicación:
    firebase deploy

Estrategias Avanzadas de Despliegue

Integración Continua/Despliegue Continuo (CI/CD)

  • Configurar pipelines de CI/CD: Usa herramientas como GitHub Actions, GitLab CI, o CircleCI para automatizar el proceso de despliegue.
  • Ejemplo de flujo de trabajo de GitHub Actions:
name: Deploy React App

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v2

      - name: Install dependencies
        run: npm install

      - name: Build project
        run: npm run build

      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./build

Renderizado del Lado del Servidor (SSR) con Next.js

  • Instalar Next.js:
npx create-next-app
  • Desplegar en Vercel: Vercel proporciona una integración perfecta con Next.js para SSR.

Conclusión

Desplegar una aplicación React implica varios pasos, desde crear una construcción de producción hasta elegir el servicio de alojamiento adecuado. Ya sea que estés desplegando en un host de sitio estático como GitHub Pages, un proveedor de nube como AWS, o usando pipelines avanzados de CI/CD, entender estos procesos asegura que tu aplicación sea accesible y funcione bien en un entorno de producción. Con estas herramientas y técnicas, puedes desplegar tus aplicaciones React con confianza y mantenerlas funcionando sin problemas.
