Hola Mundo en React
JSX: JavaScript XML ►

Introducción

React es una biblioteca de JavaScript popular para construir interfaces de usuario, particularmente aplicaciones de una sola página. Permite a los desarrolladores crear componentes de UI reutilizables. En esta sección, cubriremos los conceptos básicos para crear una aplicación de "Hola Mundo" en React.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Conocimientos básicos de JavaScript y HTML.
  • Node.js y npm instalados en tu máquina.

Configuración del Entorno

Para crear una aplicación de React, utilizaremos Create React App, una herramienta que configura un nuevo proyecto de React con una configuración predeterminada sensata.

  1. Instalar Create React App: Abre tu terminal y ejecuta:

    npx create-react-app hello-world

    Este comando creará un nuevo directorio llamado hello-world con todos los archivos y dependencias necesarios.

  2. Navegar al Directorio del Proyecto:

    cd hello-world

  3. Iniciar el Servidor de Desarrollo:

    npm start

    Esto iniciará el servidor de desarrollo y abrirá tu nueva aplicación de React en el navegador.

Entendiendo la Estructura del Proyecto

Cuando creas una nueva aplicación de React, la estructura del proyecto se ve así:

hello-world/
├── node_modules/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── App.css
│   ├── App.js
│   ├── App.test.js
│   ├── index.css
│   ├── index.js
│   └── ...
├── .gitignore
├── package.json
├── README.md
└── ...
  • public/index.html: El único archivo HTML que sirve tu aplicación de React.
  • src/index.js: El punto de entrada de JavaScript de tu aplicación.
  • src/App.js: Un componente de React de muestra.

Creando el Componente "Hola Mundo"

Vamos a modificar el componente predeterminado de React para mostrar "Hola Mundo".

  1. Abre src/App.js:

    import React from 'react';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>Hola Mundo</h1>
      </header>
    </div>
  );
}

export default App;

  2. Explicación:

    • Importando React: La declaración import React from 'react'; es necesaria para usar JSX, una extensión de sintaxis para JavaScript que se parece al HTML.
    • Componente Funcional: App es un componente funcional que devuelve JSX. JSX nos permite escribir código similar a HTML dentro de JavaScript.
    • JSX: La declaración return contiene JSX que renderiza un div con un header y un elemento h1 que muestra "Hola Mundo".

Ejecutando la Aplicación

Después de guardar los cambios, tu servidor de desarrollo debería recargarse automáticamente y deberías ver "Hola Mundo" en tu navegador.

Conclusión

En esta sección, cubrimos los conceptos básicos para configurar un entorno de React y crear una simple aplicación de "Hola Mundo". Este conocimiento fundamental es crucial a medida que avanzamos hacia temas más complejos en React. Entender cómo configurar y modificar componentes será esencial para construir aplicaciones más avanzadas.
