Introducción

En Objective-C, los operadores son símbolos especiales que realizan operaciones sobre variables y valores. Las expresiones combinan variables, constantes y operadores para producir un nuevo valor. Entender los operadores y las expresiones es fundamental para escribir código Objective-C efectivo y eficiente.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas como suma, resta, multiplicación y división.

  • Suma (+): Suma dos operandos.
  • Resta (-): Resta el segundo operando del primero.
  • Multiplicación (*): Multiplica dos operandos.
  • División (/): Divide el primer operando por el segundo.
  • Módulo (%): Devuelve el resto de una operación de división.

Ejemplo

int a = 10;
int b = 3;

int sum = a + b;        // sum es 13
int difference = a - b; // difference es 7
int product = a * b;    // product es 30
int quotient = a / b;   // quotient es 3
int remainder = a % b;  // remainder es 1

Operadores Relacionales

Los operadores relacionales comparan dos valores y devuelven un resultado Booleano (YES o NO en Objective-C).

  • Igual a (==): Comprueba si dos operandos son iguales.
  • No igual a (!=): Comprueba si dos operandos no son iguales.
  • Mayor que (>): Comprueba si el primer operando es mayor que el segundo.
  • Menor que (<): Comprueba si el primer operando es menor que el segundo.
  • Mayor o igual a (>=): Comprueba si el primer operando es mayor o igual al segundo.
  • Menor o igual a (<=): Comprueba si el primer operando es menor o igual al segundo.

Ejemplo

int a = 10;
int b = 20;

BOOL isEqual = (a == b);       // isEqual es NO
BOOL isNotEqual = (a != b);    // isNotEqual es YES
BOOL isGreater = (a > b);      // isGreater es NO
BOOL isLess = (a < b);         // isLess es YES
BOOL isGreaterOrEqual = (a >= b); // isGreaterOrEqual es NO
BOOL isLessOrEqual = (a <= b);    // isLessOrEqual es YES

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones o valores Booleanos.

  • AND lógico (&&): Devuelve YES si ambos operandos son YES.
  • OR lógico (||): Devuelve YES si al menos un operando es YES.
  • NOT lógico (!): Devuelve YES si el operando es NO.

Ejemplo

BOOL a = YES;
BOOL b = NO;

BOOL resultAND = (a && b); // resultAND es NO
BOOL resultOR = (a || b);  // resultOR es YES
BOOL resultNOT = !a;       // resultNOT es NO

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación simple (=): Asigna el operando del lado derecho al operando del lado izquierdo.
  • Suma y asigna (+=): Suma el operando del lado derecho al operando del lado izquierdo y asigna el resultado al operando del lado izquierdo.
  • Resta y asigna (-=): Resta el operando del lado derecho del operando del lado izquierdo y asigna el resultado al operando del lado izquierdo.
  • Multiplica y asigna (*=): Multiplica el operando del lado derecho con el operando del lado izquierdo y asigna el resultado al operando del lado izquierdo.
  • Divide y asigna (/=): Divide el operando del lado izquierdo por el operando del lado derecho y asigna el resultado al operando del lado izquierdo.
  • Módulo y asigna (%=): Toma el módulo usando el operando del lado izquierdo y el operando del lado derecho y asigna el resultado al operando del lado izquierdo.

Ejemplo

int a = 10;
a += 5;  // a es ahora 15
a -= 3;  // a es ahora 12
a *= 2;  // a es ahora 24
a /= 4;  // a es ahora 6
a %= 4;  // a es ahora 2

Operador Condicional (Ternario)

El operador condicional es una forma abreviada del if-else. Toma tres operandos.

  • Sintaxis: condition ? expression1 : expression2

Ejemplo

int a = 10;
int b = 20;

int max = (a > b) ? a : b; // max es 20

Operadores a Nivel de Bits

Los operadores a nivel de bits realizan operaciones sobre las representaciones binarias de los enteros.

  • AND (&): Realiza una operación AND a nivel de bits.
  • OR (|): Realiza una operación OR a nivel de bits.
  • XOR (^): Realiza una operación XOR a nivel de bits.
  • NOT (~): Realiza una operación NOT a nivel de bits.
  • Desplazamiento a la izquierda (<<): Desplaza bits a la izquierda.
  • Desplazamiento a la derecha (>>): Desplaza bits a la derecha.

Ejemplo

int a = 5;  // Binario: 0101
int b = 3;  // Binario: 0011

int andResult = a & b; // andResult es 1 (Binario: 0001)
int orResult = a | b;  // orResult es 7 (Binario: 0111)
int xorResult = a ^ b; // xorResult es 6 (Binario: 0110)
int notResult = ~a;    // notResult es -6 (Binario: 1010 en complemento a 2)
int leftShift = a << 1; // leftShift es 10 (Binario: 1010)
int rightShift = a >> 1; // rightShift es 2 (Binario: 0010)

Resumen

Entender los operadores y las expresiones es crucial para manipular datos y controlar el flujo de tus programas en Objective-C. Este tema cubrió:

  • Operadores aritméticos para operaciones matemáticas básicas.
  • Operadores relacionales para comparar valores.
  • Operadores lógicos para combinar expresiones Booleanas.
  • Operadores de asignación para asignar y actualizar valores de variables.
  • El operador condicional (ternario) para expresiones condicionales concisas.
  • Operadores a nivel de bits para realizar operaciones sobre representaciones binarias.

Al dominar estos operadores, podrás escribir código Objective-C más eficiente y legible.
