Introducción

En Objective-C, las colecciones son estructuras de datos fundamentales que te permiten almacenar y gestionar grupos de objetos. Los tipos principales de colecciones son Arrays, Diccionarios y Sets. Cada uno de estos tiene características y casos de uso únicos.

Arrays

Los Arrays en Objective-C son colecciones ordenadas de objetos. Pueden almacenar múltiples objetos de cualquier tipo, pero todos los objetos deben ser instancias de NSObject o sus subclases.

Creación e Inicialización de Arrays

Puedes crear e inicializar arrays usando varios métodos:

// Creando un array vacío
NSArray *emptyArray = [NSArray array];

// Creando un array con objetos
NSArray *fruits = [NSArray arrayWithObjects:@"Apple", @"Banana", @"Cherry", nil];

// Usando literales modernos de Objective-C
NSArray *colors = @[@"Red", @"Green", @"Blue"];

Acceso a Elementos del Array

Puedes acceder a los elementos de un array usando su índice:

NSString *firstFruit = fruits[0]; // Accediendo al primer elemento
NSLog(@"First fruit: %@", firstFruit);

Iteración sobre Arrays

Puedes iterar sobre arrays usando un bucle for o enumeración rápida:

// Usando un bucle for
for (int i = 0; i < [fruits count]; i++) {
    NSLog(@"Fruit: %@", fruits[i]);
}

// Usando enumeración rápida
for (NSString *fruit in fruits) {
    NSLog(@"Fruit: %@", fruit);
}

Diccionarios

Los Diccionarios en Objective-C son colecciones que almacenan pares clave-valor. Cada clave debe ser única y está asociada con un valor.

Creación e Inicialización de Diccionarios

Puedes crear e inicializar diccionarios usando varios métodos:

// Creando un diccionario vacío
NSDictionary *emptyDict = [NSDictionary dictionary];

// Creando un diccionario con pares clave-valor
NSDictionary *person = [NSDictionary dictionaryWithObjectsAndKeys:
                        @"John", @"firstName",
                        @"Doe", @"lastName",
                        @30, @"age", nil];

// Usando literales modernos de Objective-C
NSDictionary *car = @{@"make": @"Toyota", @"model": @"Camry", @"year": @2020};

Acceso a Valores del Diccionario

Puedes acceder a los valores en un diccionario usando sus claves:

NSString *firstName = person[@"firstName"];
NSLog(@"First name: %@", firstName);

Iteración sobre Diccionarios

Puedes iterar sobre diccionarios usando enumeración rápida:

for (NSString *key in person) {
    NSLog(@"%@: %@", key, person[key]);
}

Sets

Los Sets en Objective-C son colecciones desordenadas de objetos únicos. Son útiles cuando necesitas asegurarte de que no haya duplicados.

Creación e Inicialización de Sets

Puedes crear e inicializar sets usando varios métodos:

// Creando un set vacío
NSSet *emptySet = [NSSet set];

// Creando un set con objetos
NSSet *uniqueNumbers = [NSSet setWithObjects:@1, @2, @3, @4, nil];

// Usando literales modernos de Objective-C
NSSet *letters = [NSSet setWithArray:@[@"A", @"B", @"C"]];

Acceso a Elementos del Set

Dado que los sets son desordenados, no puedes acceder a los elementos por índice. En su lugar, puedes verificar la pertenencia:

if ([uniqueNumbers containsObject:@2]) {
    NSLog(@"The set contains the number 2");
}

Iteración sobre Sets

Puedes iterar sobre sets usando enumeración rápida:

for (NSNumber *number in uniqueNumbers) {
    NSLog(@"Number: %@", number);
}

Resumen

En este tema, cubrimos los tres tipos principales de colecciones en Objective-C: Arrays, Diccionarios y Sets. Los Arrays son colecciones ordenadas, los Diccionarios almacenan pares clave-valor, y los Sets son colecciones desordenadas de objetos únicos. Entender estas colecciones y cómo usarlas efectivamente es crucial para gestionar datos en tus programas de Objective-C. Practica creando, accediendo e iterando sobre estas colecciones para volverte competente en su uso.
