Pseudoclases y Pseudoelementos
Transiciones y animaciones ►

Introducción

Las pseudoclases y los pseudoelementos son herramientas poderosas en CSS que te permiten estilizar elementos basados en su estado o posición en el árbol del documento, sin necesidad de agregar clases o IDs adicionales a tu HTML. Esta sección cubrirá los conceptos básicos y el uso avanzado de las pseudoclases y los pseudoelementos.

Pseudoclases

Las pseudoclases son palabras clave añadidas a los selectores que especifican un estado especial de los elementos seleccionados. Se utilizan para estilizar un elemento cuando está en un cierto estado, como cuando se pasa el ratón sobre él, o cuando es el primer hijo de su padre.

Pseudoclases comunes

  • :hover: Se aplica cuando el usuario designa un elemento (con algún dispositivo señalador), pero no lo activa.
  • :active: Se aplica mientras un elemento está siendo activado por el usuario.
  • :focus: Se aplica mientras un elemento tiene el foco (acepta eventos de teclado o ratón, u otras formas de entrada).
  • :first-child: Se aplica al primer hijo de un elemento especificado.
  • :last-child: Se aplica al último hijo de un elemento especificado.
  • :nth-child(n): Se aplica al enésimo hijo de un elemento especificado.

Ejemplo

/* Cambia el color de fondo de un botón cuando se pasa el ratón sobre él */
button:hover {
  background-color: lightblue;
}

/* Cambia el color del primer párrafo en un contenedor */
.container p:first-child {
  color: red;
}

/* Estiliza cada segundo elemento de la lista */
ul li:nth-child(2n) {
  background-color: #f2f2f2;
}

Explicación

  • El selector button:hover cambia el color de fondo de un botón cuando el puntero del ratón está sobre él.
  • El selector .container p:first-child cambia el color del texto del primer párrafo dentro de un contenedor con la clase container.
  • El selector ul li:nth-child(2n) cambia el color de fondo de cada segundo elemento de la lista en una lista desordenada.

Pseudoelementos

Los pseudoelementos son palabras clave añadidas a los selectores que te permiten estilizar partes específicas de un elemento. Se utilizan para estilizar una parte de un elemento, como la primera letra o línea de un párrafo, o para insertar contenido antes o después del contenido de un elemento.

Pseudoelementos comunes

  • ::before: Inserta contenido antes del contenido de un elemento.
  • ::after: Inserta contenido después del contenido de un elemento.
  • ::first-letter: Estiliza la primera letra de la primera línea de un elemento de nivel de bloque.
  • ::first-line: Estiliza la primera línea de un elemento de nivel de bloque.

Ejemplo

/* Inserta contenido antes de un párrafo */
p::before {
  content: "Nota: ";
  font-weight: bold;
}

/* Inserta contenido después de un párrafo */
p::after {
  content: " [Leer más]";
  color: blue;
}

/* Estiliza la primera letra de un párrafo */
p::first-letter {
  font-size: 2em;
  color: red;
}

/* Estiliza la primera línea de un párrafo */
p::first-line {
  font-weight: bold;
}

Explicación

  • El selector p::before inserta el texto "Nota: " antes del contenido de cada párrafo y lo pone en negrita.
  • El selector p::after inserta el texto " [Leer más]" después del contenido de cada párrafo y lo colorea de azul.
  • El selector p::first-letter aumenta el tamaño de la fuente y cambia el color de la primera letra de cada párrafo.
  • El selector p::first-line pone en negrita la primera línea de cada párrafo.

Combinando Pseudoclases y Pseudoelementos

Puedes combinar pseudoclases y pseudoelementos para crear estilos más complejos.

Ejemplo

/* Estiliza la primera letra de un párrafo cuando se pasa el ratón sobre él */
p:hover::first-letter {
  color: green;
  font-size: 3em;
}

Explicación

  • El selector p:hover::first-letter cambia el color y el tamaño de la fuente de la primera letra de un párrafo cuando se pasa el ratón sobre el párrafo.

Conclusión

Las pseudoclases y los pseudoelementos son herramientas esenciales en CSS que te permiten aplicar estilos basados en el estado o partes específicas de los elementos. Entender cómo usarlos efectivamente puede mejorar enormemente la interactividad y el atractivo visual de tus páginas web. Practica usando estos selectores en diferentes escenarios para volverte competente en su uso.
