Fuentes y Tipografía
Modelo de caja ►

La tipografía es un aspecto crucial del diseño web, y CSS proporciona una variedad de propiedades para controlar la apariencia del texto en tus páginas web. Este tema te guiará desde lo básico hasta técnicas avanzadas en tipografía CSS.

Propiedades Básicas de Fuentes

Familia de Fuentes

  • Definición: La propiedad font-family especifica la fuente para un elemento.
  • Uso: Puedes listar múltiples fuentes como un sistema de respaldo.
body {
    font-family: "Arial", "Helvetica", sans-serif;
}

Tamaño de Fuente

  • Definición: La propiedad font-size establece el tamaño del texto.
  • Unidades: Las unidades comunes incluyen px, em, rem, %, y vw.
p {
    font-size: 16px;
}

Peso de Fuente

  • Definición: La propiedad font-weight establece el grosor de la fuente.
  • Valores: Los valores comunes incluyen normal, bold, bolder, lighter, y valores numéricos (100-900).
h1 {
    font-weight: bold;
}

Estilo de Fuente

  • Definición: La propiedad font-style especifica el estilo de la fuente.
  • Valores: normal, italic, y oblique.
em {
    font-style: italic;
}

Propiedades Avanzadas de Fuentes

Altura de Línea

  • Definición: La propiedad line-height establece el espacio entre líneas de texto.
  • Uso: Puede establecerse usando números sin unidades, porcentajes o valores de longitud.
p {
    line-height: 1.5;
}

Transformación de Texto

  • Definición: La propiedad text-transform controla la capitalización del texto.
  • Valores: none, capitalize, uppercase, y lowercase.
h2 {
    text-transform: uppercase;
}

Decoración de Texto

  • Definición: La propiedad text-decoration añade decoración al texto.
  • Valores: none, underline, overline, line-through, y combinaciones.
a {
    text-decoration: underline;
}

Espaciado entre Letras

  • Definición: La propiedad letter-spacing controla el espacio entre caracteres.
  • Uso: Puede establecerse usando valores de longitud.
h1 {
    letter-spacing: 2px;
}

Espaciado entre Palabras

  • Definición: La propiedad word-spacing establece el espacio entre palabras.
  • Uso: Puede establecerse usando valores de longitud.
p {
    word-spacing: 4px;
}

Fuentes Web

Uso de Google Fonts

  • Definición: Google Fonts es una biblioteca de familias de fuentes con licencia gratuita.
  • Uso: Importa fuentes a través de una etiqueta <link> en HTML o usando @import en CSS.
<!-- HTML -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
/* CSS */
body {
    font-family: 'Roboto', sans-serif;
}

Regla @font-face

  • Definición: La regla @font-face te permite definir fuentes personalizadas.
  • Uso: Especifica el nombre de la fuente y los archivos de origen.
@font-face {
    font-family: 'MyCustomFont';
    src: url('mycustomfont.woff2') format('woff2'),
         url('mycustomfont.woff') format('woff');
}

body {
    font-family: 'MyCustomFont', sans-serif;
}

Tipografía Responsiva

Tipografía Fluida

  • Definición: La tipografía fluida se escala suavemente entre diferentes tamaños de ventana.
  • Uso: Usa funciones CSS como calc() y unidades de ventana.
h1 {
    font-size: calc(1.5rem + 2vw);
}

Consultas de Medios

  • Definición: Las consultas de medios te permiten aplicar estilos basados en el tamaño de la ventana.
  • Uso: Ajusta los tamaños de fuente para diferentes dispositivos.
@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

@media (min-width: 601px) {
    body {
        font-size: 16px;
    }
}

Conclusión

La tipografía en CSS es una herramienta poderosa que puede mejorar enormemente la legibilidad y la estética de tus páginas web. Desde propiedades básicas como font-family y font-size hasta técnicas avanzadas como @font-face y tipografía responsiva, dominar estas propiedades te permitirá crear diseños web visualmente atractivos y accesibles. Practica usando estas propiedades en varias combinaciones para ver sus efectos y desarrollar un buen ojo para la buena tipografía.
