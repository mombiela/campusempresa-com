Bootstrap es un poderoso framework de front-end para un desarrollo web más rápido y fácil. Incluye plantillas de diseño basadas en HTML y CSS para tipografía, formularios, botones, tablas, navegación, modales, carruseles de imágenes y muchos otros componentes, así como plugins opcionales de JavaScript.

Introducción a Bootstrap

  • ¿Qué es Bootstrap?
    • Un framework CSS gratuito y de código abierto dirigido al desarrollo web front-end, primero móvil y responsivo.
    • Contiene plantillas de diseño basadas en CSS y JavaScript.
  • ¿Por qué usar Bootstrap?
    • Acelera el desarrollo.
    • Garantiza consistencia en diferentes navegadores y dispositivos.
    • Proporciona un sistema de cuadrícula responsivo.
    • Ofrece componentes pre-estilizados extensos.

Empezando con Bootstrap

  • Incluir Bootstrap en tu Proyecto
    • Usando CDN (Red de Entrega de Contenidos): 
      <!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo de Bootstrap</title>
  <link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <h1>¡Hola, Bootstrap!</h1>
  <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.5.4/dist/umd/popper.min.js"></script>
  <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
</body>
</html>
    • Descargando Bootstrap:
      • Descarga desde getbootstrap.com.
      • Incluye los archivos descargados en el directorio de tu proyecto.

Sistema de Cuadrícula de Bootstrap

  • Entendiendo el Sistema de Cuadrícula
    • El sistema de cuadrícula de Bootstrap utiliza una serie de contenedores, filas y columnas para diseñar y alinear contenido.
    • Está construido con flexbox y es completamente responsivo.
  • Estructura Básica: 
    <div class="container">
  <div class="row">
    <div class="col">
      Columna 1
    </div>
    <div class="col">
      Columna 2
    </div>
    <div class="col">
      Columna 3
    </div>
  </div>
</div>
  • Columnas Responsivas: 
    <div class="container">
  <div class="row">
    <div class="col-sm">
      Columna Pequeña
    </div>
    <div class="col-md">
      Columna Mediana
    </div>
    <div class="col-lg">
      Columna Grande
    </div>
  </div>
</div>

Componentes de Bootstrap

  • Botones: 
    <button type="button" class="btn btn-primary">Botón Primario</button>
<button type="button" class="btn btn-secondary">Botón Secundario</button>
  • Barras de Navegación: 
    <nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#">Inicio <span class="sr-only">(actual)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Características</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Precios</a>
      </li>
    </ul>
  </div>
</nav>
  • Tarjetas: 
    <div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Título de la Tarjeta</h5>
    <p class="card-text">Algún texto de ejemplo rápido para construir sobre el título de la tarjeta y formar la mayor parte del contenido de la tarjeta.</p>
    <a href="#" class="btn btn-primary">Ir a algún lugar</a>
  </div>
</div>

Utilidades de Bootstrap

  • Espaciado: 
    <div class="mt-3">Margen superior 3</div>
<div class="mb-3">Margen inferior 3</div>
<div class="pt-3">Relleno superior 3</div>
<div class="pb-3">Relleno inferior 3</div>
  • Alineación de Texto: 
    <div class="text-left">Texto alineado a la izquierda</div>
<div class="text-center">Texto alineado al centro</div>
<div class="text-right">Texto alineado a la derecha</div>

Conclusión

Bootstrap es una herramienta esencial para el desarrollo web moderno, proporcionando un robusto framework que simplifica la creación de sitios web responsivos y visualmente atractivos. Al dominar Bootstrap, puedes acelerar significativamente tu proceso de desarrollo y asegurar una apariencia consistente en diferentes dispositivos y navegadores. Comienza con los conceptos básicos, practica con los componentes y explora las utilidades para aprovechar al máximo el poder de Bootstrap en tus proyectos.
