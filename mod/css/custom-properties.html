Introducción a las Propiedades Personalizadas

Las propiedades personalizadas, también conocidas como variables CSS, te permiten almacenar valores que puedes reutilizar a lo largo de tu hoja de estilos. Esto hace que tu CSS sea más mantenible y fácil de leer.

  • Definición: Las propiedades personalizadas son entidades definidas por los autores de CSS que contienen valores específicos para ser reutilizados a lo largo de un documento.
  • Sintaxis: Las propiedades personalizadas se definen usando el prefijo -- y se accede a ellas usando la función var().

Sintaxis Básica

Para definir una propiedad personalizada, utilizas el prefijo -- seguido del nombre de la propiedad. Luego puedes usar la función var() para acceder al valor de la propiedad personalizada.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
}

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}
  • :root: Esta es una pseudo-clase que coincide con el elemento raíz del documento. A menudo se usa para definir propiedades personalizadas globales.
  • --main-bg-color: Esta es una propiedad personalizada para el color de fondo principal.
  • var(--main-bg-color): Esta función recupera el valor de la propiedad personalizada.

Ventajas de Usar Propiedades Personalizadas

  • Reutilización: Define un valor una vez y reutilízalo a lo largo de tu hoja de estilos.
  • Mantenibilidad: Cambia el valor en un solo lugar y se actualizará en todas partes donde se use.
  • Cambios Dinámicos: Las propiedades personalizadas pueden actualizarse dinámicamente usando JavaScript.

Ejemplo: Reutilización y Mantenibilidad

:root {
  --primary-color: #e74c3c;
  --secondary-color: #2ecc71;
}

.header {
  background-color: var(--primary-color);
}

.footer {
  background-color: var(--secondary-color);
}

.button {
  background-color: var(--primary-color);
  color: var(--secondary-color);
}

En este ejemplo, cambiar --primary-color o --secondary-color en el :root actualizará el esquema de colores en todos los elementos que usan estas propiedades personalizadas.

Uso Avanzado

Valores de Respaldo

Puedes proporcionar un valor de respaldo en caso de que la propiedad personalizada no esté definida.

.button {
  background-color: var(--nonexistent-color, #333);
}

En este caso, si --nonexistent-color no está definida, el color de fondo del botón será por defecto #333.

Consultas de Medios y Propiedades Personalizadas

Las propiedades personalizadas pueden usarse dentro de consultas de medios para crear diseños responsivos.

:root {
  --font-size: 16px;
}

@media (min-width: 768px) {
  :root {
    --font-size: 18px;
  }
}

body {
  font-size: var(--font-size);
}

Aquí, el tamaño de la fuente cambia según el ancho de la ventana, demostrando cómo las propiedades personalizadas pueden adaptarse a diferentes tamaños de pantalla.

JavaScript y Propiedades Personalizadas

Puedes manipular propiedades personalizadas usando JavaScript para crear estilos dinámicos.

<!DOCTYPE html>
<html lang="es">
<head>
  <style>
    :root {
      --dynamic-color: #3498db;
    }
    .dynamic {
      color: var(--dynamic-color);
    }
  </style>
</head>
<body>
  <p class="dynamic">Este texto cambiará de color dinámicamente.</p>
  <button onclick="changeColor()">Cambiar Color</button>

  <script>
    function changeColor() {
      document.documentElement.style.setProperty('--dynamic-color', '#e74c3c');
    }
  </script>
</body>
</html>

En este ejemplo, al hacer clic en el botón se cambia el color del texto actualizando la propiedad personalizada --dynamic-color.

Conclusión

Las propiedades personalizadas en CSS proporcionan una forma poderosa de gestionar y reutilizar valores a lo largo de tus hojas de estilo. Mejoran la mantenibilidad, la reutilización y permiten cambios dinámicos, convirtiéndolas en una herramienta esencial para el desarrollo web moderno. Al comprender y utilizar las propiedades personalizadas, puedes crear hojas de estilo más eficientes y adaptables.
