Introducción a la sintaxis de CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. Comprender la sintaxis básica de CSS es crucial para estilizar páginas web de manera efectiva.

Sintaxis básica de CSS

  • Selector: Indica a qué elementos HTML se aplica el estilo.
  • Bloque de declaración: Contiene una o más declaraciones separadas por punto y coma.
  • Declaración: Consiste en una propiedad y un valor, separados por dos puntos.
selector {
  property: value;
}

Ejemplo

p {
  color: blue;
  font-size: 16px;
}

En este ejemplo:

  • p es el selector, que apunta a todos los elementos <p>.
  • color y font-size son propiedades.
  • blue y 16px son los valores correspondientes.

Tipos de selectores de CSS

Los selectores de CSS se utilizan para seleccionar los elementos HTML que deseas estilizar. Hay varios tipos de selectores, cada uno con su propio caso de uso.

Selectores básicos

  • Selector universal (*): Selecciona todos los elementos.
  • Selector de tipo: Selecciona todos los elementos de un tipo dado.
  • Selector de clase (.classname): Selecciona todos los elementos con una clase específica.
  • Selector de ID (#idname): Selecciona un solo elemento con un ID específico.
  • Selector de atributo: Selecciona elementos basados en un atributo o valor de atributo.

Ejemplos

* {
  margin: 0;
  padding: 0;
}

h1 {
  color: red;
}

.container {
  width: 100%;
}

#header {
  background-color: #f1f1f1;
}

input[type="text"] {
  border: 1px solid #ccc;
}

Selectores de combinador

  • Selector descendiente (ancestor descendant): Selecciona todos los descendientes de un elemento especificado.
  • Selector hijo (parent > child): Selecciona todos los hijos directos de un elemento especificado.
  • Selector de hermano adyacente (element + adjacent): Selecciona un elemento que es precedido inmediatamente por un elemento especificado.
  • Selector de hermano general (element ~ siblings): Selecciona todos los hermanos de un elemento especificado.

Ejemplos

div p {
  color: green;
}

ul > li {
  list-style-type: none;
}

h1 + p {
  margin-top: 0;
}

h1 ~ p {
  color: gray;
}

Pseudo-clases y pseudo-elementos

  • Pseudo-clases: Se utilizan para definir un estado especial de un elemento.
  • Pseudo-elementos: Se utilizan para estilizar partes específicas de un elemento.

Ejemplos

/* Pseudo-clases */
a:hover {
  color: orange;
}

input:focus {
  border-color: blue;
}

/* Pseudo-elementos */
p::first-line {
  font-weight: bold;
}

div::before {
  content: "Nota: ";
  color: red;
}

Agrupación y anidación de selectores

  • Agrupación de selectores: Varios selectores pueden compartir los mismos estilos separándolos con comas.
  • Anidación de selectores: CSS no soporta anidación verdadera, pero puedes lograr resultados similares usando combinadores.

Ejemplos

/* Agrupación de selectores */
h1, h2, h3 {
  font-family: Arial, sans-serif;
}

/* Anidación de selectores */
nav ul {
  list-style-type: none;
}

nav ul li {
  display: inline;
}

Especificidad y herencia

  • Especificidad: Determina qué estilos se aplican cuando múltiples reglas podrían aplicarse al mismo elemento.
  • Herencia: Algunas propiedades de CSS se heredan de elementos padres a hijos, mientras que otras no.

Cálculo de especificidad

  • Los estilos en línea tienen la mayor especificidad.
  • Los selectores de ID son más específicos que los selectores de clase.
  • Los selectores de clase son más específicos que los selectores de tipo.

Ejemplo

/* Ejemplo de especificidad */
#unique {
  color: red; /* Mayor especificidad */
}

.container {
  color: blue; /* Especificidad media */
}

p {
  color: green; /* Menor especificidad */
}

Si un elemento <p> está dentro de un elemento con la clase container y tiene el ID unique, será rojo.

Conclusión

Comprender la sintaxis y los selectores de CSS es fundamental para crear páginas web bien estilizadas. Al dominar diferentes tipos de selectores y su especificidad, puedes aplicar estilos de manera precisa y eficiente. Practica usando estos selectores en varios escenarios para volverte competente en CSS.

Resumen

  • La sintaxis de CSS consiste en selectores y bloques de declaración.
  • Los selectores básicos incluyen universal, tipo, clase, ID y selectores de atributo.
  • Los selectores de combinador ayudan a apuntar a elementos basados en sus relaciones.
  • Las pseudo-clases y pseudo-elementos proporcionan capacidades de estilización adicionales.
  • La agrupación y anidación de selectores pueden simplificar tu CSS.
  • La especificidad y herencia determinan cómo se aplican los estilos.

Al comprender estos conceptos, estarás en buen camino para volverte competente en CSS.
