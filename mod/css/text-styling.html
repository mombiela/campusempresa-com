Introducción

La estilización de texto es un aspecto fundamental del diseño web que mejora la legibilidad y el atractivo visual de tu contenido. En esta sección, cubriremos varias propiedades de CSS que te permiten estilizar el texto de manera efectiva, desde técnicas básicas hasta avanzadas.

Estilización Básica de Texto

Familia de Fuentes

  • La propiedad font-family especifica la fuente para un elemento.
  • Puedes listar múltiples fuentes como un sistema de respaldo.
p {
  font-family: Arial, Helvetica, sans-serif;
}

Tamaño de Fuente

  • La propiedad font-size establece el tamaño del texto.
  • Las unidades comunes incluyen px, em, rem y %.
h1 {
  font-size: 24px;
}

Grosor de Fuente

  • La propiedad font-weight establece el grosor del texto.
  • Los valores varían de 100 (delgado) a 900 (negrita), o puedes usar palabras clave como normal y bold.
strong {
  font-weight: bold;
}

Estilo de Fuente

  • La propiedad font-style te permite hacer el texto en cursiva o oblicuo.
em {
  font-style: italic;
}

Color del Texto

  • La propiedad color establece el color del texto.
  • Puedes usar nombres de colores, valores HEX, RGB o HSL.
a {
  color: #3498db;
}

Estilización Intermedia de Texto

Alineación de Texto

  • La propiedad text-align alinea el texto dentro de su contenedor.
  • Los valores incluyen left, right, center y justify.
p {
  text-align: justify;
}

Decoración de Texto

  • La propiedad text-decoration agrega decoraciones como subrayados, líneas sobre el texto y líneas a través del texto.
a {
  text-decoration: underline;
}

Transformación de Texto

  • La propiedad text-transform controla la capitalización del texto.
  • Los valores incluyen uppercase, lowercase y capitalize.
h2 {
  text-transform: uppercase;
}

Altura de Línea

  • La propiedad line-height establece el espacio entre líneas de texto.
  • Puede establecerse usando números sin unidad, porcentajes o valores de longitud.
p {
  line-height: 1.5;
}

Espaciado entre Letras

  • La propiedad letter-spacing ajusta el espacio entre caracteres.
h1 {
  letter-spacing: 2px;
}

Espaciado entre Palabras

  • La propiedad word-spacing ajusta el espacio entre palabras.
p {
  word-spacing: 4px;
}

Estilización Avanzada de Texto

Sombra de Texto

  • La propiedad text-shadow agrega sombra al texto.
  • Sintaxis: text-shadow: h-shadow v-shadow blur-radius color;
h1 {
  text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

Fuentes Web

  • Usando @font-face para incluir fuentes personalizadas.
  • Ejemplo de importación de una fuente de Google.
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}

Efectos Avanzados de Texto

  • Usando background-clip y color para texto con degradado.
h1 {
  background: linear-gradient(to right, #30CFD0 0%, #330867 100%);
  -webkit-background-clip: text;
  color: transparent;
}

Texto Responsivo

  • Usando unidades vw (ancho de la ventana gráfica) para tamaños de texto responsivos.
h1 {
  font-size: 5vw;
}

Conclusión

La estilización de texto en CSS abarca desde propiedades básicas como el tamaño y el color de la fuente hasta técnicas avanzadas como sombras de texto y texto responsivo. Dominar estas propiedades te permite crear páginas web visualmente atractivas y legibles. Practica usando estas propiedades en diferentes combinaciones para ver sus efectos y comprender mejor su comportamiento.
