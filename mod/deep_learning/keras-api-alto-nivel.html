Introducción a Keras

Keras es una API de alto nivel para construir y entrenar modelos de aprendizaje profundo. Fue desarrollada con el objetivo de permitir una experimentación rápida y fácil, y es conocida por su simplicidad y facilidad de uso. Keras se puede ejecutar sobre varias bibliotecas de backend, como TensorFlow, Theano y Microsoft Cognitive Toolkit (CNTK).

Características Principales de Keras

  • Simplicidad y facilidad de uso: Keras está diseñado para ser intuitivo y fácil de usar, lo que permite a los desarrolladores construir modelos de manera rápida y eficiente.
  • Modularidad: Los modelos en Keras están construidos a partir de componentes modulares que se pueden combinar de diversas maneras.
  • Extensibilidad: Keras es altamente extensible, lo que permite a los investigadores implementar nuevas capas, funciones de pérdida y métricas personalizadas.
  • Compatibilidad con múltiples backends: Keras puede ejecutarse sobre TensorFlow, Theano y CNTK, lo que proporciona flexibilidad en la elección del backend.

Construcción de Modelos en Keras

Keras proporciona dos formas principales de construir modelos: el modelo secuencial y el modelo funcional.

Modelo Secuencial

El modelo secuencial es una pila lineal de capas. Es adecuado para la mayoría de los problemas de aprendizaje profundo.

Ejemplo de Código: Modelo Secuencial

from keras.models import Sequential
from keras.layers import Dense

# Crear un modelo secuencial
model = Sequential()

# Añadir capas al modelo
model.add(Dense(32, input_shape=(784,), activation='relu'))
model.add(Dense(10, activation='softmax'))

# Compilar el modelo
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resumen del modelo
model.summary()

En este ejemplo:

  • Se crea un modelo secuencial.
  • Se añaden dos capas densas (fully connected).
  • Se compila el modelo especificando el optimizador, la función de pérdida y las métricas.
  • Se imprime un resumen del modelo.

Modelo Funcional

El modelo funcional es más flexible y permite la construcción de modelos más complejos, como redes neuronales con múltiples entradas y salidas.

Ejemplo de Código: Modelo Funcional

from keras.models import Model
from keras.layers import Input, Dense

# Definir las entradas
inputs = Input(shape=(784,))

# Añadir capas
x = Dense(32, activation='relu')(inputs)
outputs = Dense(10, activation='softmax')(x)

# Crear el modelo
model = Model(inputs=inputs, outputs=outputs)

# Compilar el modelo
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resumen del modelo
model.summary()

En este ejemplo:

  • Se define una entrada usando Input.
  • Se añaden capas usando la sintaxis funcional.
  • Se crea el modelo especificando las entradas y salidas.
  • Se compila el modelo y se imprime un resumen.

Entrenamiento y Evaluación de Modelos

Una vez que el modelo está construido y compilado, se puede entrenar y evaluar utilizando los métodos fit y evaluate.

Entrenamiento del Modelo

# Entrenar el modelo
model.fit(x_train, y_train, epochs=10, batch_size=32, validation_split=0.2)

En este ejemplo:

  • x_train y y_train son los datos de entrenamiento.
  • epochs es el número de veces que el modelo verá el conjunto de datos completo.
  • batch_size es el número de muestras por actualización de gradiente.
  • validation_split es la fracción de datos de entrenamiento que se utilizará como datos de validación.

Evaluación del Modelo

# Evaluar el modelo
loss, accuracy = model.evaluate(x_test, y_test)
print(f'Loss: {loss}, Accuracy: {accuracy}')

En este ejemplo:

  • x_test y y_test son los datos de prueba.
  • El método evaluate devuelve la pérdida y la precisión del modelo en los datos de prueba.

Conclusión

Keras es una herramienta poderosa y fácil de usar para construir y entrenar modelos de aprendizaje profundo. Su simplicidad y modularidad la hacen ideal tanto para principiantes como para investigadores avanzados. Con Keras, puedes construir modelos secuenciales simples o modelos funcionales complejos, y entrenarlos y evaluarlos con facilidad.
