Introducción a las Redes Generativas Antagónicas (GAN)

Las Redes Generativas Antagónicas (GAN) son un tipo de modelo de aprendizaje profundo que puede generar datos nuevos y realistas a partir de un conjunto de datos de entrenamiento. Fueron introducidas por Ian Goodfellow en 2014 y han revolucionado la generación de imágenes, videos y otros tipos de datos.

Conceptos Clave

  • Generador: Una red neuronal que genera datos falsos a partir de ruido aleatorio.
  • Discriminador: Una red neuronal que intenta distinguir entre datos reales y falsos.
  • Juego Minimax: El generador y el discriminador compiten entre sí en un juego de suma cero.
  • Función de Pérdida: Mide qué tan bien el generador y el discriminador están desempeñándose.

Arquitectura de una GAN

Una GAN consta de dos componentes principales:

  1. Generador (G):

    • Toma un vector de ruido como entrada.
    • Produce una imagen falsa que intenta parecerse a las imágenes reales del conjunto de datos.

  2. Discriminador (D):

    • Toma una imagen como entrada.
    • Clasifica la imagen como real (proveniente del conjunto de datos) o falsa (generada por el generador).

Ejemplo de Código

A continuación, se muestra un ejemplo básico de cómo implementar una GAN utilizando TensorFlow y Keras.

import tensorflow as tf
from tensorflow.keras import layers

# Definir el generador
def build_generator():
    model = tf.keras.Sequential()
    model.add(layers.Dense(256, input_dim=100, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(512, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(1024, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(28 * 28 * 1, activation='tanh'))
    model.add(layers.Reshape((28, 28, 1)))
    return model

# Definir el discriminador
def build_discriminator():
    model = tf.keras.Sequential()
    model.add(layers.Flatten(input_shape=(28, 28, 1)))
    model.add(layers.Dense(512, activation='relu'))
    model.add(layers.Dense(256, activation='relu'))
    model.add(layers.Dense(1, activation='sigmoid'))
    return model

# Crear el generador y el discriminador
generator = build_generator()
discriminator = build_discriminator()

# Compilar el discriminador
discriminator.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# Crear la GAN combinada
discriminator.trainable = False
gan_input = layers.Input(shape=(100,))
generated_image = generator(gan_input)
gan_output = discriminator(generated_image)
gan = tf.keras.Model(gan_input, gan_output)

# Compilar la GAN
gan.compile(optimizer='adam', loss='binary_crossentropy')

Entrenamiento de la GAN

El entrenamiento de una GAN implica entrenar tanto el generador como el discriminador en un bucle alternado.

Pasos de Entrenamiento

  1. Entrenar el Discriminador:

    • Usar un lote de imágenes reales y un lote de imágenes generadas.
    • Calcular la pérdida y actualizar los pesos del discriminador.

  2. Entrenar el Generador:

    • Generar un lote de imágenes falsas.
    • Pasar estas imágenes al discriminador.
    • Calcular la pérdida basada en la capacidad del discriminador para distinguir las imágenes falsas.
    • Actualizar los pesos del generador para engañar mejor al discriminador.

Ejemplo de Código de Entrenamiento

import numpy as np

# Parámetros de entrenamiento
epochs = 10000
batch_size = 64
sample_interval = 1000

# Cargar y preprocesar el conjunto de datos MNIST
(X_train, _), (_, _) = tf.keras.datasets.mnist.load_data()
X_train = X_train / 127.5 - 1.0
X_train = np.expand_dims(X_train, axis=3)

# Etiquetas para imágenes reales y falsas
real = np.ones((batch_size, 1))
fake = np.zeros((batch_size, 1))

for epoch in range(epochs):
    # Entrenar el discriminador
    idx = np.random.randint(0, X_train.shape[0], batch_size)
    real_images = X_train[idx]

    noise = np.random.normal(0, 1, (batch_size, 100))
    generated_images = generator.predict(noise)

    d_loss_real = discriminator.train_on_batch(real_images, real)
    d_loss_fake = discriminator.train_on_batch(generated_images, fake)
    d_loss = 0.5 * np.add(d_loss_real, d_loss_fake)

    # Entrenar el generador
    noise = np.random.normal(0, 1, (batch_size, 100))
    g_loss = gan.train_on_batch(noise, real)

    # Imprimir el progreso
    if epoch % sample_interval == 0:
        print(f"{epoch} [D loss: {d_loss[0]}, acc.: {100*d_loss[1]}%] [G loss: {g_loss}]")

Conclusión

En este proyecto, hemos explorado cómo crear y entrenar una GAN para la generación de imágenes. Las GAN son una herramienta poderosa en el campo del aprendizaje profundo, con aplicaciones que van desde la generación de imágenes y videos hasta la creación de datos sintéticos para el entrenamiento de otros modelos.
