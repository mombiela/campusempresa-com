Introducción

En JavaScript, entender los prototipos y la herencia es crucial para dominar la programación orientada a objetos. Esta sección cubrirá los conceptos básicos de los prototipos, cómo funciona la herencia en JavaScript y cómo usar estos conceptos para crear código más eficiente y organizado.

Prototipos

¿Qué es un Prototipo?

  • Cada objeto en JavaScript tiene un prototipo.
  • Un prototipo también es un objeto.
  • Los prototipos son el mecanismo por el cual los objetos de JavaScript heredan características unos de otros.

Cadena de Prototipos

  • Los objetos pueden tener un prototipo, que a su vez puede tener su propio prototipo, formando una cadena.
  • La cadena termina cuando un objeto tiene null como su prototipo.

Ejemplo de Cadena de Prototipos

let animal = {
    eats: true
};

let rabbit = {
    jumps: true
};

rabbit.__proto__ = animal; // estableciendo animal como el prototipo de rabbit

console.log(rabbit.eats); // true
console.log(rabbit.jumps); // true

Accediendo a los Prototipos

  • Usa Object.getPrototypeOf(obj) para obtener el prototipo de un objeto.
  • Usa Object.setPrototypeOf(obj, prototype) para establecer el prototipo de un objeto.

Ejemplo

let animal = {
    eats: true
};

let rabbit = Object.create(animal); // rabbit hereda de animal

console.log(Object.getPrototypeOf(rabbit) === animal); // true

Herencia

¿Qué es la Herencia?

  • La herencia es una característica que permite a un objeto heredar propiedades y métodos de otro objeto.
  • Esto ayuda a reutilizar código y crear una estructura jerárquica.

Herencia Clásica vs Prototípica

  • La herencia clásica involucra clases y es común en lenguajes como Java y C++.
  • La herencia prototípica involucra objetos y es más flexible.

Creando una Cadena de Herencia

  • Usa Object.create() para crear un nuevo objeto con un prototipo especificado.

Ejemplo

let animal = {
    eats: true,
    walk() {
        console.log("Animal walks");
    }
};

let rabbit = Object.create(animal);
rabbit.jumps = true;

rabbit.walk(); // Animal walks
console.log(rabbit.eats); // true
console.log(rabbit.jumps); // true

Funciones Constructoras y Prototipos

Funciones Constructoras

  • Las funciones constructoras se usan para crear múltiples objetos similares.
  • Usa la palabra clave new para crear una instancia de un objeto.

Ejemplo

function Animal(name) {
    this.name = name;
    this.eats = true;
}

Animal.prototype.walk = function() {
    console.log(this.name + " walks");
};

let rabbit = new Animal("Rabbit");
rabbit.walk(); // Rabbit walks

Propiedad Prototype

  • Cada función en JavaScript tiene una propiedad prototype.
  • Esta propiedad se usa para adjuntar métodos y propiedades que deben ser heredados por las instancias de la función constructora.

Ejemplo

function Animal(name) {
    this.name = name;
}

Animal.prototype.eats = true;

let rabbit = new Animal("Rabbit");
console.log(rabbit.eats); // true

Clases de ES6

Introducción a las Clases

  • ES6 introdujo clases para facilitar la escritura y comprensión de la herencia en JavaScript.
  • Las clases son azúcar sintáctico sobre la herencia basada en prototipos existente en JavaScript.

Ejemplo

class Animal {
    constructor(name) {
        this.name = name;
        this.eats = true;
    }

    walk() {
        console.log(this.name + " walks");
    }
}

let rabbit = new Animal("Rabbit");
rabbit.walk(); // Rabbit walks

Herencia con Clases

  • Usa la palabra clave extends para crear una clase que hereda de otra clase.

Ejemplo

class Animal {
    constructor(name) {
        this.name = name;
    }

    walk() {
        console.log(this.name + " walks");
    }
}

class Rabbit extends Animal {
    constructor(name, color) {
        super(name); // llama al constructor del padre
        this.color = color;
    }

    jump() {
        console.log(this.name + " jumps");
    }
}

let rabbit = new Rabbit("Rabbit", "white");
rabbit.walk(); // Rabbit walks
rabbit.jump(); // Rabbit jumps

Resumen

  • Los prototipos son fundamentales para entender el modelo de herencia de JavaScript.
  • Los objetos heredan propiedades y métodos de sus prototipos.
  • Las funciones constructoras y la palabra clave new se usan para crear instancias con propiedades y métodos compartidos.
  • Las clases de ES6 proporcionan una sintaxis más intuitiva para crear y gestionar la herencia en JavaScript.
  • Entender estos conceptos permite un código más eficiente y organizado, aprovechando el poder de la programación orientada a objetos en JavaScript.
