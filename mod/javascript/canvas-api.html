API de Canvas
Prototipos y Herencia ►

La API de Canvas es una característica poderosa en HTML5 que te permite dibujar gráficos usando JavaScript. Este tema cubrirá los conceptos básicos de la API de Canvas, progresando hacia técnicas más avanzadas.

Introducción a la API de Canvas

  • ¿Qué es la API de Canvas?

    • Una API de JavaScript para dibujar gráficos 2D.
    • Permite la renderización dinámica y programable de formas e imágenes.

  • Configuración del Canvas

    • Elemento HTML <canvas>.
    • Acceso al contexto del canvas en JavaScript.

Ejemplo: Configuración Básica del Canvas

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Canvas</title>
</head>
<body>
    <canvas id="myCanvas" width="500" height="500" style="border:1px solid #000000;"></canvas>
    <script>
        const canvas = document.getElementById('myCanvas');
        const ctx = canvas.getContext('2d');
    </script>
</body>
</html>

Dibujo de Formas

  • Rectángulos
    • fillRect(x, y, width, height)
    • strokeRect(x, y, width, height)
    • clearRect(x, y, width, height)

Ejemplo: Dibujo de Rectángulos

ctx.fillStyle = 'green';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'blue';
ctx.strokeRect(130, 10, 100, 100);

ctx.clearRect(50, 50, 60, 60);
  • Rutas
    • beginPath()
    • moveTo(x, y)
    • lineTo(x, y)
    • closePath()
    • stroke()
    • fill()

Ejemplo: Dibujo de Rutas

ctx.beginPath();
ctx.moveTo(50, 50);
ctx.lineTo(200, 50);
ctx.lineTo(200, 200);
ctx.closePath();
ctx.stroke();

Trabajo con Colores y Estilos

  • Estilos de Relleno y Trazo
    • fillStyle
    • strokeStyle
    • globalAlpha

Ejemplo: Uso de Colores y Estilos

ctx.fillStyle = 'rgba(255, 0, 0, 0.5)';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'rgba(0, 0, 255, 0.5)';
ctx.strokeRect(130, 10, 100, 100);

Dibujo de Texto

  • Métodos de Texto
    • fillText(text, x, y)
    • strokeText(text, x, y)
    • font
    • textAlign
    • textBaseline

Ejemplo: Dibujo de Texto

ctx.font = '30px Arial';
ctx.fillText('Hola Canvas', 10, 50);

ctx.strokeText('Hola Canvas', 10, 100);

Transformaciones

  • Escalado

    • scale(x, y)

  • Rotación

    • rotate(angle)

  • Traslación

    • translate(x, y)

Ejemplo: Aplicación de Transformaciones

ctx.translate(100, 100);
ctx.rotate(Math.PI / 4);
ctx.fillStyle = 'purple';
ctx.fillRect(0, 0, 100, 100);

Técnicas Avanzadas

  • Gradientes
    • createLinearGradient(x0, y0, x1, y1)
    • createRadialGradient(x0, y0, r0, x1, y1, r1)

Ejemplo: Uso de Gradientes

const gradient = ctx.createLinearGradient(0, 0, 200, 0);
gradient.addColorStop(0, 'red');
gradient.addColorStop(1, 'blue');

ctx.fillStyle = gradient;
ctx.fillRect(10, 10, 200, 100);
  • Patrones
    • createPattern(image, repetition)

Ejemplo: Uso de Patrones

const img = new Image();
img.src = 'path/to/image.jpg';
img.onload = function() {
    const pattern = ctx.createPattern(img, 'repeat');
    ctx.fillStyle = pattern;
    ctx.fillRect(0, 0, 500, 500);
};

Animaciones

  • Bucle de Animación Básico
    • requestAnimationFrame(callback)

Ejemplo: Animación Básica

let x = 0;
function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    ctx.fillRect(x, 50, 50, 50);
    x += 1;
    requestAnimationFrame(draw);
}
draw();

Conclusión

La API de Canvas es una herramienta versátil para crear gráficos dinámicos en aplicaciones web. Al comprender los conceptos básicos del dibujo de formas, trabajar con colores, aplicar transformaciones y crear animaciones, puedes construir visualizaciones complejas e interactivas. Practica estas técnicas para volverte competente en el uso de la API de Canvas para tus proyectos.
