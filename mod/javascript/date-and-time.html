Fecha y Hora en JavaScript
Expresiones de Función y Funciones Flecha ►

Introducción

Manejar fechas y horas es un requisito común en muchas aplicaciones de JavaScript. JavaScript proporciona un objeto Date incorporado para trabajar con fechas y horas. Esta sección cubrirá los conceptos básicos del objeto Date, cómo manipular fechas y horas, y algunas técnicas avanzadas para trabajar con fechas y horas en JavaScript.

Creación de Fechas

El objeto Date de JavaScript se puede crear de varias maneras:

  • Fecha y Hora Actual:

    const now = new Date();
console.log(now); // Muestra la fecha y hora actuales

  • Fecha y Hora Específica:

    const specificDate = new Date('2023-10-01T10:20:30Z');
console.log(specificDate); // Muestra: 2023-10-01T10:20:30.000Z

  • Componentes de Fecha:

    const dateComponents = new Date(2023, 9, 1, 10, 20, 30);
console.log(dateComponents); // Muestra: Sun Oct 01 2023 10:20:30 GMT+0000 (Tiempo Universal Coordinado)

Obtener Componentes de Fecha y Hora

Puedes extraer varios componentes de un objeto Date usando diferentes métodos:

  • Año, Mes, Día:

    const date = new Date();
console.log(date.getFullYear()); // Muestra el año
console.log(date.getMonth()); // Muestra el mes (0-11)
console.log(date.getDate()); // Muestra el día del mes (1-31)

  • Horas, Minutos, Segundos:

    const time = new Date();
console.log(time.getHours()); // Muestra las horas (0-23)
console.log(time.getMinutes()); // Muestra los minutos (0-59)
console.log(time.getSeconds()); // Muestra los segundos (0-59)

Establecer Componentes de Fecha y Hora

También puedes establecer varios componentes de un objeto Date:

  • Establecer Año, Mes, Día:

    const date = new Date();
date.setFullYear(2025);
date.setMonth(11); // Diciembre
date.setDate(25);
console.log(date); // Muestra: Thu Dec 25 2025

  • Establecer Horas, Minutos, Segundos:

    const time = new Date();
time.setHours(15);
time.setMinutes(45);
time.setSeconds(30);
console.log(time); // Muestra: fecha actual con hora establecida a 15:45:30

Formatear Fechas

JavaScript proporciona varios métodos para formatear fechas:

  • toLocaleDateString():

    const date = new Date();
console.log(date.toLocaleDateString()); // Muestra la fecha en formato local

  • toLocaleTimeString():

    const time = new Date();
console.log(time.toLocaleTimeString()); // Muestra la hora en formato local

  • toISOString():

    const isoDate = new Date();
console.log(isoDate.toISOString()); // Muestra la fecha en formato ISO 8601

Aritmética de Fechas

Puedes realizar operaciones aritméticas en fechas:

  • Agregar Días:

    const date = new Date();
date.setDate(date.getDate() + 5);
console.log(date); // Muestra la fecha 5 días a partir de ahora

  • Restar Días:

    const date = new Date();
date.setDate(date.getDate() - 5);
console.log(date); // Muestra la fecha de hace 5 días

Comparar Fechas

Para comparar fechas, puedes usar operadores de comparación estándar:

  • Igualdad:

    const date1 = new Date('2023-10-01');
const date2 = new Date('2023-10-01');
console.log(date1.getTime() === date2.getTime()); // Muestra: true

  • Mayor/Menor Que:

    const date1 = new Date('2023-10-01');
const date2 = new Date('2023-10-02');
console.log(date1 < date2); // Muestra: true

Trabajar con Zonas Horarias

Los objetos Date de JavaScript se basan en la zona horaria local del navegador, pero puedes trabajar con diferentes zonas horarias usando bibliotecas como moment-timezone:

  • Usando moment-timezone: 
    // Incluir la biblioteca moment-timezone
const moment = require('moment-timezone');
const date = moment.tz('2023-10-01 10:20:30', 'America/New_York');
console.log(date.format()); // Muestra: 2023-10-01T10:20:30-04:00

Conclusión

Trabajar con fechas y horas en JavaScript puede ser complejo debido a varios factores como zonas horarias y diferentes formatos de fecha. Sin embargo, el objeto Date incorporado proporciona un conjunto robusto de métodos para manejar la mayoría de las tareas comunes. Para una manipulación más avanzada de fechas y horas, bibliotecas como moment.js o date-fns pueden ser muy útiles. Entender estos conceptos básicos te permitirá gestionar eficazmente fechas y horas en tus aplicaciones de JavaScript.
