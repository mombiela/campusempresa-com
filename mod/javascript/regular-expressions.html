Introducción a las Expresiones Regulares

Las expresiones regulares (regex) son patrones utilizados para coincidir combinaciones de caracteres en cadenas. En JavaScript, las expresiones regulares son objetos que se pueden usar para realizar funciones de coincidencia de patrones y "buscar y reemplazar" en texto.

  • Sintaxis Básica: Las expresiones regulares se crean de dos maneras:
    • Notación literal: Encerradas entre barras (/patrón/).
    • Función constructora: Creada con el constructor RegExp (new RegExp('patrón')).

Ejemplo:

// Notación literal
const regex1 = /hello/;

// Función constructora
const regex2 = new RegExp('hello');

Patrones Básicos

  • Caracteres Literales: Coincide con los caracteres exactos en el patrón.
  • Metacaracteres: Caracteres especiales que tienen un significado único en regex.

Metacaracteres Comunes:

  • .: Coincide con cualquier carácter excepto nueva línea.
  • \d: Coincide con cualquier dígito (equivalente a [0-9]).
  • \w: Coincide con cualquier carácter de palabra (alfanumérico + guion bajo).
  • \s: Coincide con cualquier carácter de espacio en blanco (espacios, tabulaciones, saltos de línea).
  • \b: Coincide con un límite de palabra.
  • ^: Coincide con el inicio de una cadena.
  • $: Coincide con el final de una cadena.

Ejemplo:

const regex = /\d\w\s/;
const str = "3a ";
console.log(regex.test(str)); // true

Clases de Caracteres y Rangos

  • Clases de Caracteres: Definidas usando corchetes [] para coincidir con cualquiera de los caracteres incluidos.
  • Rangos: Definidos usando un guion - dentro de corchetes para especificar un rango de caracteres.

Ejemplo:

const regex = /[a-z]/; // Coincide con cualquier letra minúscula
const str = "hello";
console.log(regex.test(str)); // true

Cuantificadores

Los cuantificadores especifican cuántas veces debe coincidir un carácter o grupo.

Cuantificadores Comunes:

  • *: Coincide 0 o más veces.
  • +: Coincide 1 o más veces.
  • ?: Coincide 0 o 1 vez.
  • {n}: Coincide exactamente n veces.
  • {n,}: Coincide n o más veces.
  • {n,m}: Coincide entre n y m veces.

Ejemplo:

const regex = /a{2,4}/; // Coincide con 'a' entre 2 y 4 veces
const str = "aaa";
console.log(regex.test(str)); // true

Grupos y Alternancia

  • Grupos: Los paréntesis () se usan para crear grupos.
  • Alternancia: La barra vertical | se usa para coincidir con una cosa u otra.

Ejemplo:

const regex = /(cat|dog)/; // Coincide con 'cat' o 'dog'
const str = "I have a cat";
console.log(regex.test(str)); // true

Banderas

Las banderas son parámetros opcionales que modifican el comportamiento de la regex.

Banderas Comunes:

  • g: Búsqueda global.
  • i: Búsqueda insensible a mayúsculas.
  • m: Búsqueda multilínea.

Ejemplo:

const regex = /hello/i; // Búsqueda insensible a mayúsculas
const str = "Hello";
console.log(regex.test(str)); // true

Métodos para Expresiones Regulares

  • test(): Prueba una coincidencia en una cadena. Devuelve true o false.
  • exec(): Ejecuta una búsqueda de una coincidencia en una cadena. Devuelve un array de información o null.

Ejemplo:

const regex = /hello/;
const str = "hello world";
console.log(regex.test(str)); // true
console.log(regex.exec(str)); // ["hello"]

Métodos de Cadena con Expresiones Regulares

  • match(): Recupera las coincidencias al comparar una cadena con una regex.
  • replace(): Reemplaza subcadenas coincidentes con una nueva subcadena.
  • search(): Prueba una coincidencia en una cadena y devuelve el índice de la coincidencia.
  • split(): Divide una cadena en un array de subcadenas usando una regex.

Ejemplo:

const str = "hello world";
const regex = /o/;

console.log(str.match(regex)); // ["o"]
console.log(str.replace(regex, 'a')); // "hella world"
console.log(str.search(regex)); // 4
console.log(str.split(regex)); // ["hell", " w", "rld"]

Patrones Avanzados

  • Lookaheads y Lookbehinds: Aserciones que coinciden con un grupo antes o después de un patrón principal sin incluirlo en el resultado.
    • Lookahead Positivo: (?=...)
    • Lookahead Negativo: (?!...)
    • Lookbehind Positivo: (?<=...)
    • Lookbehind Negativo: (?<!...)

Ejemplo:

const regex = /foo(?=bar)/; // Coincide con 'foo' solo si es seguido por 'bar'
const str = "foobar";
console.log(regex.test(str)); // true

Conclusión

Las expresiones regulares son una herramienta poderosa para la coincidencia de patrones y la manipulación de texto en JavaScript. Al comprender la sintaxis básica, las clases de caracteres, los cuantificadores, los grupos, la alternancia, las banderas y los métodos, puedes realizar tareas complejas de procesamiento de texto de manera eficiente. Practica usando expresiones regulares con diferentes ejemplos para volverte competente en su uso.
