Configuración de tu Entorno de Desarrollo
Tu Primer Programa en JavaScript ►

Configurar tu entorno de desarrollo es el primer paso para volverte competente en JavaScript. Esta sección te guiará a través de las herramientas y configuraciones necesarias para comenzar, ya seas un principiante o estés avanzando en tus habilidades.

1. Elegir un Editor de Código

Un editor de código es donde escribirás tu código JavaScript. Aquí hay algunas opciones populares:

  • Visual Studio Code (VS Code)
  • Sublime Text
  • Atom
  • WebStorm

Visual Studio Code (VS Code)

VS Code es altamente recomendado debido a sus extensas características y soporte de la comunidad. Vamos a configurarlo:

  1. Descargar e Instalar VS Code:

    • Ve al sitio web de VS Code.
    • Descarga el instalador para tu sistema operativo (Windows, macOS o Linux).
    • Sigue las instrucciones de instalación.

  2. Instalar Extensiones:

    • Abre VS Code.
    • Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana o presionando Ctrl+Shift+X.
    • Busca e instala las siguientes extensiones:
      • ESLint: Herramienta de linting para identificar y corregir problemas en tu código JavaScript.
      • Prettier: Formateador de código para asegurar un estilo de código consistente.
      • Live Server: Lanza un servidor de desarrollo local con función de recarga en vivo para páginas estáticas y dinámicas.

2. Configuración de Node.js y npm

Node.js es un entorno de ejecución de JavaScript que te permite ejecutar JavaScript en el lado del servidor. npm (Node Package Manager) es un gestor de paquetes para JavaScript, que te permite instalar bibliotecas y herramientas.

  1. Descargar e Instalar Node.js:

    • Ve al sitio web de Node.js.
    • Descarga la versión LTS (Long Term Support) para tu sistema operativo.
    • Sigue las instrucciones de instalación.

  2. Verificar la Instalación:

    • Abre tu terminal (Command Prompt, PowerShell o Terminal).
    • Ejecuta los siguientes comandos para verificar la instalación: 
      node -v
npm -v
    • Deberías ver los números de versión de Node.js y npm.

3. Configuración de Git y GitHub

El control de versiones es esencial para gestionar tu base de código. Git es un sistema de control de versiones, y GitHub es una plataforma para alojar y colaborar en repositorios de Git.

  1. Descargar e Instalar Git:

    • Ve al sitio web de Git.
    • Descarga el instalador para tu sistema operativo.
    • Sigue las instrucciones de instalación.

  2. Configurar Git:

    • Abre tu terminal.
    • Configura tu nombre de usuario y correo electrónico: 
      git config --global user.name "Tu Nombre"
git config --global user.email "tu.email@ejemplo.com"

  3. Crear una Cuenta de GitHub:

    • Ve al sitio web de GitHub.
    • Regístrate para una cuenta gratuita.

  4. Conectar Git a GitHub:

    • Genera una clave SSH: 
      ssh-keygen -t rsa -b 4096 -C "tu.email@ejemplo.com"
    • Agrega la clave SSH a tu cuenta de GitHub copiando la clave de ~/.ssh/id_rsa.pub y pegándola en la sección de claves SSH de la configuración de tu cuenta de GitHub.

4. Crear tu Primer Proyecto en JavaScript

Ahora que tu entorno está configurado, vamos a crear un proyecto simple en JavaScript.

  1. Crear un Directorio de Proyecto:

    • Abre tu terminal.
    • Crea un nuevo directorio y navega dentro de él: 
      mkdir mi-proyecto-js
cd mi-proyecto-js

  2. Inicializar un Repositorio Git:

    • Inicializa un nuevo repositorio Git: 
      git init

  3. Crear un Archivo JavaScript:

    • Abre VS Code.
    • Crea un nuevo archivo llamado app.js en tu directorio de proyecto.
    • Agrega el siguiente código a app.js: 
      console.log('¡Hola, Mundo!');

  4. Ejecutar tu Archivo JavaScript:

    • Abre tu terminal.
    • Ejecuta el archivo JavaScript usando Node.js: 
      node app.js
    • Deberías ver ¡Hola, Mundo! impreso en la terminal.

Conclusión

Configurar tu entorno de desarrollo es un paso crucial en tu camino para convertirte en un desarrollador competente en JavaScript. Al elegir las herramientas adecuadas y configurarlas correctamente, puedes optimizar tu flujo de trabajo y enfocarte en escribir buen código. En esta sección, hemos cubierto:

  • Elegir y configurar un editor de código (VS Code).
  • Instalar Node.js y npm.
  • Configurar Git y GitHub para el control de versiones.
  • Crear y ejecutar tu primer proyecto en JavaScript.

Con tu entorno listo, ahora estás preparado para profundizar en la programación en JavaScript. ¡Feliz codificación!
