Creación y Eliminación de Elementos
API de Almacenamiento Web ►

En esta sección, exploraremos cómo crear y eliminar elementos en el DOM (Modelo de Objeto del Documento) usando JavaScript. Esta es una habilidad fundamental para manipular páginas web dinámicamente.

Creación de Elementos

Crear elementos en JavaScript implica usar el método document.createElement. Este método te permite crear nuevos elementos HTML que pueden ser añadidos al DOM.

Conceptos Clave

  • document.createElement(tagName): Crea un nuevo elemento con el nombre de etiqueta especificado.
  • element.appendChild(childElement): Añade un elemento hijo al elemento padre especificado.
  • element.innerHTML: Establece o obtiene el contenido HTML dentro de un elemento.

Ejemplo: Creando un Nuevo Párrafo

// Crear un nuevo elemento de párrafo
const newParagraph = document.createElement('p');

// Establecer el contenido de texto del párrafo
newParagraph.textContent = 'Este es un nuevo párrafo creado con JavaScript.';

// Encontrar el elemento padre donde se añadirá el nuevo párrafo
const parentElement = document.getElementById('content');

// Añadir el nuevo párrafo al elemento padre
parentElement.appendChild(newParagraph);

En este ejemplo:

  1. Creamos un nuevo elemento <p>.
  2. Establecemos su contenido de texto.
  3. Encontramos el elemento padre con el ID content.
  4. Añadimos el nuevo párrafo al elemento padre.

Ejemplo: Creando un Nuevo Elemento de Lista

// Crear un nuevo elemento de lista
const newItem = document.createElement('li');

// Establecer el contenido de texto del elemento de lista
newItem.textContent = 'Nuevo elemento de lista';

// Encontrar el elemento padre (una lista desordenada)
const list = document.getElementById('myList');

// Añadir el nuevo elemento de lista a la lista desordenada
list.appendChild(newItem);

Eliminación de Elementos

Eliminar elementos del DOM se puede hacer usando el método removeChild o el método remove.

Conceptos Clave

  • parentElement.removeChild(childElement): Elimina un elemento hijo especificado del elemento padre.
  • element.remove(): Elimina el elemento del DOM.

Ejemplo: Eliminando un Elemento Hijo

// Encontrar el elemento padre
const parentElement = document.getElementById('content');

// Encontrar el elemento hijo que se va a eliminar
const childElement = document.getElementById('paragraphToRemove');

// Eliminar el elemento hijo del padre
parentElement.removeChild(childElement);

Ejemplo: Eliminando un Elemento Directamente

// Encontrar el elemento que se va a eliminar
const elementToRemove = document.getElementById('itemToRemove');

// Eliminar el elemento del DOM
elementToRemove.remove();

Resumen

En esta sección, cubrimos los conceptos básicos de crear y eliminar elementos en el DOM usando JavaScript. Aprendimos cómo:

  • Crear nuevos elementos con document.createElement.
  • Añadir nuevos elementos al DOM con appendChild.
  • Eliminar elementos usando removeChild y remove.

Estas habilidades son esenciales para manipular páginas web dinámicamente, permitiéndote crear experiencias de usuario interactivas y responsivas. Practica estas técnicas con diferentes elementos HTML para familiarizarte más con la manipulación del DOM.
