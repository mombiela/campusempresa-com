Introducción a los Condicionales

Los condicionales son bloques fundamentales en la programación que te permiten ejecutar diferentes códigos basados en ciertas condiciones. En JavaScript, los condicionales te ayudan a tomar decisiones en tu código.

Declaración `if`

La declaración if es la forma más básica de condicional. Ejecuta un bloque de código si una condición especificada es verdadera.

Sintaxis

if (condition) {
  // código a ejecutar si la condición es verdadera
}

Ejemplo

let age = 18;

if (age >= 18) {
  console.log("Eres un adulto.");
}

En este ejemplo, el mensaje "Eres un adulto." se imprimirá en la consola porque la condición age >= 18 es verdadera.

Declaración `if...else`

La declaración if...else te permite ejecutar un bloque de código si la condición es verdadera, y otro bloque si la condición es falsa.

Sintaxis

if (condition) {
  // código a ejecutar si la condición es verdadera
} else {
  // código a ejecutar si la condición es falsa
}

Ejemplo

let age = 16;

if (age >= 18) {
  console.log("Eres un adulto.");
} else {
  console.log("Eres un menor.");
}

En este ejemplo, el mensaje "Eres un menor." se imprimirá porque la condición age >= 18 es falsa.

Declaración `if...else if...else`

La declaración if...else if...else se usa para probar múltiples condiciones.

Sintaxis

if (condition1) {
  // código a ejecutar si la condition1 es verdadera
} else if (condition2) {
  // código a ejecutar si la condition2 es verdadera
} else {
  // código a ejecutar si tanto condition1 como condition2 son falsas
}

Ejemplo

let score = 85;

if (score >= 90) {
  console.log("Calificación: A");
} else if (score >= 80) {
  console.log("Calificación: B");
} else if (score >= 70) {
  console.log("Calificación: C");
} else {
  console.log("Calificación: F");
}

En este ejemplo, el mensaje "Calificación: B" se imprimirá porque la condición score >= 80 es verdadera.

Operador Ternario

El operador ternario es una forma abreviada de la declaración if...else. Es útil para condiciones simples.

Sintaxis

condition ? expressionIfTrue : expressionIfFalse;

Ejemplo

let age = 20;
let message = age >= 18 ? "Eres un adulto." : "Eres un menor.";
console.log(message);

En este ejemplo, el mensaje "Eres un adulto." se imprimirá porque la condición age >= 18 es verdadera.

Declaración `switch`

La declaración switch se usa para realizar diferentes acciones basadas en diferentes condiciones. Es una alternativa a usar múltiples declaraciones else if.

Sintaxis

switch (expression) {
  case value1:
    // código a ejecutar si expression === value1
    break;
  case value2:
    // código a ejecutar si expression === value2
    break;
  // más casos...
  default:
    // código a ejecutar si expression no coincide con ningún caso
}

Ejemplo

let day = 3;
let dayName;

switch (day) {
  case 1:
    dayName = "Lunes";
    break;
  case 2:
    dayName = "Martes";
    break;
  case 3:
    dayName = "Miércoles";
    break;
  case 4:
    dayName = "Jueves";
    break;
  case 5:
    dayName = "Viernes";
    break;
  case 6:
    dayName = "Sábado";
    break;
  case 7:
    dayName = "Domingo";
    break;
  default:
    dayName = "Día inválido";
}

console.log(dayName);

En este ejemplo, el mensaje "Miércoles" se imprimirá porque el valor de day es 3.

Resumen

  • Declaración if: Ejecuta un bloque de código si una condición es verdadera.
  • Declaración if...else: Ejecuta un bloque de código si una condición es verdadera, y otro bloque si es falsa.
  • Declaración if...else if...else: Prueba múltiples condiciones.
  • Operador Ternario: Una forma abreviada de las declaraciones if...else.
  • Declaración switch: Una alternativa a múltiples declaraciones else if para realizar diferentes acciones basadas en diferentes condiciones.

Entender y usar estas estructuras de control de manera efectiva te permitirá escribir código JavaScript más dinámico y flexible.
