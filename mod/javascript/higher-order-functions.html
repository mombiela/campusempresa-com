Funciones de Orden Superior
Closures ►

Las funciones de orden superior son un concepto fundamental en JavaScript y en la programación funcional. Son funciones que pueden tomar otras funciones como argumentos o devolver funciones como resultado. Este tema te guiará desde los conceptos básicos hasta los usos más avanzados de las funciones de orden superior.

¿Qué son las Funciones de Orden Superior?

  • Definición: Una función de orden superior es una función que toma una o más funciones como argumentos o devuelve una función como resultado.
  • Importancia: Permiten un código más abstracto y flexible, habilitando patrones poderosos como callbacks, composición de funciones y más.

Ejemplos de Funciones de Orden Superior

Ejemplo Básico

function higherOrderFunction(callback) {
    return callback();
}

function sayHello() {
    return "¡Hola, Mundo!";
}

console.log(higherOrderFunction(sayHello)); // Salida: ¡Hola, Mundo!

En este ejemplo:

  • higherOrderFunction es una función de orden superior porque toma otra función (callback) como argumento.
  • sayHello es una función simple que devuelve una cadena.
  • Cuando higherOrderFunction se llama con sayHello como argumento, ejecuta sayHello y devuelve su resultado.

Funciones de Orden Superior Comunes en JavaScript

Métodos de Array

JavaScript proporciona varias funciones de orden superior integradas para arrays, como map, filter y reduce.

map

  • Propósito: Crea un nuevo array poblado con los resultados de llamar a una función proporcionada en cada elemento del array que llama.
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Salida: [2, 4, 6, 8]

filter

  • Propósito: Crea un nuevo array con todos los elementos que pasan la prueba implementada por la función proporcionada.
const numbers = [1, 2, 3, 4];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // Salida: [2, 4]

reduce

  • Propósito: Ejecuta una función reductora (que proporcionas) en cada elemento del array, resultando en un solo valor de salida.
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum); // Salida: 10

Uso Avanzado de Funciones de Orden Superior

Composición de Funciones

  • Definición: Combinar dos o más funciones para producir una nueva función.
const add = x => y => x + y;
const multiply = x => y => x * y;

const addAndMultiply = (a, b, c) => multiply(add(a)(b))(c);

console.log(addAndMultiply(1, 2, 3)); // Salida: 9

Currying

  • Definición: El proceso de transformar una función que toma múltiples argumentos en una secuencia de funciones que toman un solo argumento cada una.
function curry(f) {
    return function(a) {
        return function(b) {
            return f(a, b);
        };
    };
}

function sum(a, b) {
    return a + b;
}

const curriedSum = curry(sum);
console.log(curriedSum(1)(2)); // Salida: 3

Ejercicios

  1. Ejercicio Básico: Escribe una función de orden superior que tome una función y un número, y aplique la función al número tres veces.
function applyThrice(func, num) {
    return func(func(func(num)));
}

function increment(x) {
    return x + 1;
}

console.log(applyThrice(increment, 5)); // Salida: 8
  1. Ejercicio Intermedio: Usa map para crear un nuevo array que contenga las longitudes de cada cadena en un array de cadenas.
const strings = ["hola", "mundo", "JavaScript"];
const lengths = strings.map(str => str.length);
console.log(lengths); // Salida: [4, 5, 10]
  1. Ejercicio Avanzado: Escribe una función compose que tome dos funciones y devuelva una nueva función que sea su composición.
function compose(f, g) {
    return function(x) {
        return f(g(x));
    };
}

const add1 = x => x + 1;
const double = x => x * 2;

const add1ThenDouble = compose(double, add1);
console.log(add1ThenDouble(5)); // Salida: 12

Conclusión

Las funciones de orden superior son una característica poderosa en JavaScript que permiten un código más abstracto y flexible. Al entender y utilizar funciones de orden superior, puedes escribir código más conciso y expresivo. Practica usando funciones de orden superior integradas como map, filter y reduce, y trata de crear tus propias funciones de orden superior para profundizar tu comprensión.
