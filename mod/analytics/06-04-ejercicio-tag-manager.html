Ejercicio 2: Implementación de Google Tag Manager en un Sitio Web
Siguiente ►

En este ejercicio, aprenderás a implementar Google Tag Manager (GTM) en un sitio web. Google Tag Manager es una herramienta gratuita que permite gestionar y desplegar etiquetas de marketing (tags) en tu sitio web sin necesidad de modificar el código directamente. Esto facilita la implementación y el seguimiento de diversas métricas y eventos.

Objetivos del Ejercicio

  1. Crear una cuenta y un contenedor en Google Tag Manager.
  2. Instalar el código de Google Tag Manager en un sitio web.
  3. Configurar una etiqueta básica (por ejemplo, Google Analytics).
  4. Verificar que la etiqueta esté funcionando correctamente.

Paso 1: Crear una Cuenta y un Contenedor en Google Tag Manager

Instrucciones:

  1. Accede a Google Tag Manager:

    • Ve a Google Tag Manager.
    • Inicia sesión con tu cuenta de Google.

  2. Crear una Nueva Cuenta:

    • Haz clic en "Crear cuenta".
    • Rellena los campos necesarios:
      • Nombre de la cuenta: El nombre de tu empresa o proyecto.
      • País: Selecciona tu país.
    • Haz clic en "Siguiente".

  3. Crear un Contenedor:

    • Rellena los campos necesarios:
      • Nombre del contenedor: El nombre de tu sitio web.
      • Plataforma de destino: Selecciona "Web".
    • Haz clic en "Crear".
    • Acepta los términos de servicio.

Resultado Esperado:

Deberías tener una cuenta y un contenedor creados en Google Tag Manager. GTM te proporcionará dos fragmentos de código para instalar en tu sitio web.

Paso 2: Instalar el Código de Google Tag Manager en un Sitio Web

Instrucciones:

  1. Copiar el Código de GTM:
    • En la pantalla de configuración del contenedor, verás dos fragmentos de código.
    • Copia el primer fragmento de código y pégalo inmediatamente después de la etiqueta <head> en todas las páginas de tu sitio web.
    • Copia el segundo fragmento de código y pégalo inmediatamente después de la etiqueta <body> en todas las páginas de tu sitio web.

Ejemplo de Código HTML:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Sitio Web</title>
    <!-- Google Tag Manager -->
    <script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
    new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
    j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
    'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
    })(window,document,'script','dataLayer','GTM-XXXX');</script>
    <!-- Fin Google Tag Manager -->
</head>
<body>
    <!-- Google Tag Manager (noscript) -->
    <noscript><iframe src="https://www.googletagmanager.com/ns.html?id=GTM-XXXX"
    height="0" width="0" style="display:none;visibility:hidden"></iframe></noscript>
    <!-- Fin Google Tag Manager (noscript) -->
    <h1>Bienvenido a Mi Sitio Web</h1>
</body>
</html>

Resultado Esperado:

El código de Google Tag Manager debería estar instalado en tu sitio web. Puedes verificarlo utilizando la herramienta "Tag Assistant" de Google Chrome.

Paso 3: Configurar una Etiqueta Básica (Google Analytics)

Instrucciones:

  1. Accede a tu Contenedor en Google Tag Manager:

    • Ve a tu contenedor en Google Tag Manager.
    • Haz clic en "Añadir una nueva etiqueta".

  2. Configurar la Etiqueta:

    • Haz clic en "Configuración de la etiqueta".
    • Selecciona "Google Analytics: Universal Analytics".
    • Configura los siguientes campos:
      • Tipo de seguimiento: Página vista.
      • ID de seguimiento: Introduce tu ID de seguimiento de Google Analytics (por ejemplo, UA-XXXXXX-X).

  3. Configurar el Activador:

    • Haz clic en "Activadores".
    • Selecciona "All Pages" (Todas las páginas).

  4. Guardar y Publicar:

    • Haz clic en "Guardar".
    • Asigna un nombre a tu etiqueta (por ejemplo, "Google Analytics - Página Vista").
    • Haz clic en "Enviar" y luego en "Publicar".

Resultado Esperado:

Deberías tener una etiqueta de Google Analytics configurada y activada en todas las páginas de tu sitio web.

Paso 4: Verificar que la Etiqueta esté Funcionando Correctamente

Instrucciones:

  1. Utilizar Tag Assistant:

    • Instala la extensión "Tag Assistant" de Google Chrome.
    • Activa la extensión y recarga tu sitio web.
    • Verifica que la etiqueta de Google Analytics esté activa y funcionando correctamente.

  2. Verificar en Google Analytics:

    • Accede a tu cuenta de Google Analytics.
    • Ve a "Tiempo real" > "Visión general".
    • Navega por tu sitio web y verifica que las visitas se registren en tiempo real.

Resultado Esperado:

Deberías ver los datos de tu sitio web reflejados en Google Analytics en tiempo real, confirmando que la etiqueta está funcionando correctamente.

Conclusión

En este ejercicio, has aprendido a implementar Google Tag Manager en un sitio web, configurar una etiqueta básica de Google Analytics y verificar su funcionamiento. Esta habilidad es fundamental para gestionar y desplegar etiquetas de manera eficiente, permitiéndote recopilar datos valiosos sin necesidad de modificar el código de tu sitio web directamente.

Resumen de Conceptos Aprendidos:

  • Crear una cuenta y un contenedor en Google Tag Manager.
  • Instalar el código de Google Tag Manager en un sitio web.
  • Configurar una etiqueta básica de Google Analytics.
  • Verificar el funcionamiento de las etiquetas utilizando herramientas como Tag Assistant y Google Analytics.

Próximos Pasos:

  • Experimenta con la creación de etiquetas adicionales para eventos específicos (clics en botones, envíos de formularios, etc.).
  • Explora las capacidades avanzadas de Google Tag Manager, como las variables y los activadores personalizados.

¡Felicidades por completar este ejercicio! Ahora estás listo para utilizar Google Tag Manager para mejorar la analítica de tu sitio web.
