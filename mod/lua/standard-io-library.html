Biblioteca de E/S Estándar
Técnicas de Depuración ►

La Biblioteca de E/S Estándar en Lua proporciona funciones para operaciones de entrada y salida, permitiéndote leer y escribir archivos, así como interactuar con la consola. Este tema cubrirá los conceptos básicos del uso de la Biblioteca de E/S Estándar, incluyendo la lectura y escritura de archivos, y el manejo de la entrada y salida estándar.

Conceptos Clave

  • Entrada Estándar (stdin): Leer entrada del usuario u otra fuente de entrada.
  • Salida Estándar (stdout): Escribir salida en la consola u otro destino de salida.
  • Manejo de Archivos: Abrir, leer, escribir y cerrar archivos.

Leyendo desde la Entrada Estándar

Lua proporciona varias funciones para leer la entrada del usuario. La función más comúnmente utilizada es io.read.

Ejemplo: Leyendo una Línea de Entrada

print("Introduce tu nombre:")
local name = io.read()
print("¡Hola, " .. name .. "!")

En este ejemplo:

  • io.read() lee una línea de entrada del usuario.
  • La entrada se almacena en la variable name.
  • El programa luego imprime un saludo usando la entrada.

Escribiendo en la Salida Estándar

Para escribir salida en la consola, Lua usa las funciones print y io.write.

Ejemplo: Escribiendo Salida

print("¡Hola, Mundo!")
io.write("Esta es una línea sin un carácter de nueva línea.")
io.write("Esto continúa en la misma línea.")

En este ejemplo:

  • print imprime texto seguido de un carácter de nueva línea.
  • io.write imprime texto sin agregar un carácter de nueva línea.

Manejo de Archivos

La Biblioteca de E/S Estándar de Lua proporciona funciones para abrir, leer, escribir y cerrar archivos.

Abriendo un Archivo

Para abrir un archivo, usa la función io.open. Esta función devuelve un manejador de archivo que puedes usar para realizar operaciones en el archivo.

local file = io.open("example.txt", "r")

En este ejemplo:

  • "example.txt" es el nombre del archivo.
  • "r" es el modo en el que se abre el archivo. Los modos comunes incluyen:
    • "r": Modo de lectura (por defecto).
    • "w": Modo de escritura (crea un nuevo archivo o trunca un archivo existente).
    • "a": Modo de anexado (escribe datos al final del archivo).

Leyendo desde un Archivo

Puedes leer desde un archivo usando el método read del manejador de archivo.

local file = io.open("example.txt", "r")
local content = file:read("*all")
print(content)
file:close()

En este ejemplo:

  • file:read("*all") lee todo el contenido del archivo.
  • file:close() cierra el archivo.

Escribiendo en un Archivo

Para escribir en un archivo, usa el método write del manejador de archivo.

local file = io.open("example.txt", "w")
file:write("¡Hola, archivo!")
file:close()

En este ejemplo:

  • file:write("¡Hola, archivo!") escribe la cadena en el archivo.
  • file:close() cierra el archivo.

Manejo de Errores

Es importante manejar errores al trabajar con archivos. Lua proporciona la función pcall para manejar errores de manera elegante.

Ejemplo: Manejo de Errores de Archivo

local function readFile(filename)
    local file, err = io.open(filename, "r")
    if not file then
        print("Error al abrir el archivo: " .. err)
        return
    end
    local content = file:read("*all")
    file:close()
    return content
end

local content = readFile("nonexistent.txt")
if content then
    print(content)
end

En este ejemplo:

  • io.open devuelve nil y un mensaje de error si el archivo no puede ser abierto.
  • El mensaje de error se imprime si el archivo no puede ser abierto.

Resumen

La Biblioteca de E/S Estándar en Lua es una herramienta poderosa para manejar operaciones de entrada y salida. Las funciones clave incluyen:

  • io.read para leer entrada.
  • print y io.write para escribir salida.
  • io.open, file:read, file:write y file:close para el manejo de archivos.

Al entender estas funciones, puedes gestionar eficazmente la entrada y salida en tus programas Lua.
