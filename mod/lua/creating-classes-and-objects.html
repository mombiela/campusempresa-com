Introducción a la Programación Orientada a Objetos en Lua

La programación orientada a objetos (OOP) es un paradigma de programación que utiliza "objetos" para diseñar aplicaciones y programas informáticos. Lua, aunque no es un lenguaje puramente orientado a objetos, proporciona mecanismos para implementar conceptos de OOP.

  • Clase: Un plano para crear objetos (instancias).
  • Objeto: Una instancia de una clase.
  • Método: Una función que opera sobre un objeto.

Definiendo una Clase Básica en Lua

En Lua, las clases pueden ser creadas usando tablas y metatablas. Aquí hay una guía paso a paso para crear una clase básica.

Creando una Clase

  1. Define una tabla para representar la clase.
  2. Define una función constructora para inicializar objetos.
  3. Establece la metatabla para la clase para soportar la herencia.
-- Define una clase llamada 'Person'
Person = {}
Person.__index = Person

-- Constructor para la clase Person
function Person:new(name, age)
    local instance = setmetatable({}, Person)
    instance.name = name
    instance.age = age
    return instance
end

-- Método para mostrar los detalles de la persona
function Person:display()
    print("Name: " .. self.name .. ", Age: " .. self.age)
end

-- Crea un objeto de la clase Person
local person1 = Person:new("Alice", 30)
person1:display()  -- Salida: Name: Alice, Age: 30

Explicación

  • Person = {}: Crea una tabla para representar la clase.
  • Person.__index = Person: Establece el campo __index de la metatabla a la clase misma, habilitando la herencia.
  • Person:new(name, age): Función constructora para inicializar nuevos objetos.
  • setmetatable({}, Person): Establece la metatabla del nuevo objeto a la clase, permitiendo llamadas a métodos.
  • self: Se refiere a la instancia del objeto dentro de los métodos.

Añadiendo Métodos a una Clase

Los métodos son funciones que operan sobre instancias de la clase. Se definen dentro de la tabla de la clase.

-- Añadir un método para actualizar la edad
function Person:setAge(newAge)
    self.age = newAge
end

-- Actualiza la edad de person1
person1:setAge(31)
person1:display()  -- Salida: Name: Alice, Age: 31

Herencia en Lua

La herencia permite que una clase herede propiedades y métodos de otra clase. Esto se logra usando metatablas.

Creando una Subclase

  1. Define una nueva tabla para la subclase.
  2. Establece la metatabla de la subclase a la clase padre.
  3. Sobrescribe o añade nuevos métodos según sea necesario.
-- Define una subclase llamada 'Employee' que hereda de 'Person'
Employee = setmetatable({}, {__index = Person})
Employee.__index = Employee

-- Constructor para la clase Employee
function Employee:new(name, age, position)
    local instance = Person.new(self, name, age)
    instance.position = position
    return instance
end

-- Sobrescribe el método display para incluir la posición
function Employee:display()
    print("Name: " .. self.name .. ", Age: " .. self.age .. ", Position: " .. self.position)
end

-- Crea un objeto de la clase Employee
local employee1 = Employee:new("Bob", 25, "Developer")
employee1:display()  -- Salida: Name: Bob, Age: 25, Position: Developer

Explicación

  • Employee = setmetatable({}, {__index = Person}): Establece la metatabla de Employee a Person, habilitando la herencia.
  • Employee:new(name, age, position): Constructor para la subclase, llamando al constructor de la clase padre.
  • Employee:display(): Sobrescribe el método display para incluir el atributo position.

Temas Avanzados

Miembros Privados y Protegidos

Lua no tiene soporte incorporado para miembros privados o protegidos, pero puedes simularlos usando variables locales y cierres.

-- Define una clase con miembros privados
Person = {}
Person.__index = Person

function Person:new(name, age)
    local instance = setmetatable({}, Person)
    local privateAge = age  -- Miembro privado

    function instance:getAge()
        return privateAge
    end

    function instance:setAge(newAge)
        privateAge = newAge
    end

    instance.name = name
    return instance
end

local person2 = Person:new("Charlie", 40)
print(person2:getAge())  -- Salida: 40
person2:setAge(41)
print(person2:getAge())  -- Salida: 41

Polimorfismo

El polimorfismo permite que los métodos se usen indistintamente entre diferentes clases. Esto se logra a través de la sobrescritura de métodos.

-- Define una nueva clase 'Manager' que hereda de 'Employee'
Manager = setmetatable({}, {__index = Employee})
Manager.__index = Manager

-- Sobrescribe el método display para incluir el departamento
function Manager:display()
    print("Name: " .. self.name .. ", Age: " .. self.age .. ", Position: " .. self.position .. ", Department: " .. self.department)
end

-- Crea un objeto de la clase Manager
local manager1 = Manager:new("Dana", 35, "Manager")
manager1.department = "IT"
manager1:display()  -- Salida: Name: Dana, Age: 35, Position: Manager, Department: IT

Conclusión

En este tema, cubrimos los conceptos básicos de la creación de clases y objetos en Lua, incluyendo la definición de clases, la adición de métodos y la implementación de la herencia. También tocamos conceptos avanzados como miembros privados y polimorfismo. Al entender estos conceptos, puedes utilizar efectivamente la programación orientada a objetos en Lua para crear código robusto y mantenible.
