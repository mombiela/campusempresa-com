Tablas en Lua
Arreglos ►

Las tablas son una parte fundamental de Lua, sirviendo como la estructura de datos principal. Pueden ser usadas como arreglos, diccionarios, o incluso objetos. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de las tablas en Lua.

Introducción a las Tablas

Las tablas en Lua son versátiles y pueden almacenar diferentes tipos de valores, incluyendo números, cadenas, funciones, e incluso otras tablas.

Creando una Tabla

Para crear una tabla, usas la sintaxis {}.

local miTabla = {}

Añadiendo Elementos a una Tabla

Puedes añadir elementos a una tabla usando claves numéricas o de cadena.

-- Usando claves numéricas
miTabla[1] = "Lua"
miTabla[2] = "Programación"

-- Usando claves de cadena
miTabla["lenguaje"] = "Lua"
miTabla["tipo"] = "Scripting"

Accediendo a Elementos de la Tabla

Accede a los elementos usando sus claves.

print(miTabla[1])          -- Salida: Lua
print(miTabla["lenguaje"]) -- Salida: Lua

Tablas como Arreglos

En Lua, las tablas pueden ser usadas como arreglos, donde se usan índices numéricos para almacenar y acceder a los elementos.

Ejemplo: Arreglo Simple

local frutas = {"Manzana", "Banana", "Cereza"}

for i = 1, #frutas do
    print(frutas[i])
end

Explicación

  • #frutas da la longitud del arreglo.
  • El bucle itera a través de cada elemento, imprimiéndolo.

Tablas como Diccionarios

Las tablas también pueden funcionar como diccionarios (o mapas), donde se usan claves de cadena.

Ejemplo: Diccionario Simple

local capitales = {
    Francia = "París",
    Alemania = "Berlín",
    Italia = "Roma"
}

print(capitales["Francia"]) -- Salida: París

Explicación

  • Las claves son nombres de países, y los valores son sus capitales.
  • Acceder al valor usando capitales["Francia"] devuelve "París".

Tablas Anidadas

Las tablas pueden contener otras tablas, permitiendo estructuras de datos complejas.

Ejemplo: Tabla Anidada

local persona = {
    nombre = "Juan",
    edad = 30,
    direccion = {
        calle = "123 Lua St",
        ciudad = "Ciudad Lua"
    }
}

print(persona.nombre)          -- Salida: Juan
print(persona.direccion.ciudad)  -- Salida: Ciudad Lua

Explicación

  • La tabla persona contiene otra tabla direccion.
  • Accede a los elementos anidados usando notación de punto.

Funciones de Tabla

Lua proporciona varias funciones integradas para manipular tablas.

Funciones Comunes de Tabla

  • table.insert(table, [pos,] value): Inserta un valor en la tabla en la posición especificada.
  • table.remove(table, [pos]): Elimina el elemento en la posición especificada.
  • table.sort(table, [comp]): Ordena los elementos de la tabla.

Ejemplo: Usando Funciones de Tabla

local numeros = {5, 3, 8, 1}

table.insert(numeros, 2, 7) -- Inserta 7 en la posición 2
table.sort(numeros)         -- Ordena la tabla

for i, v in ipairs(numeros) do
    print(v)
end

Explicación

  • table.insert(numeros, 2, 7) inserta 7 en la segunda posición.
  • table.sort(numeros) ordena la tabla en orden ascendente.
  • ipairs se usa para iterar sobre la tabla ordenada.

Metatablas y Metamétodos

Las metatablas te permiten cambiar el comportamiento de las tablas. Los metamétodos son funciones especiales que pueden ser definidas en una metatabla para sobrescribir comportamientos predeterminados.

Ejemplo: Metatabla con __index

local porDefecto = {x = 0, y = 0}
local punto = setmetatable({}, {__index = porDefecto})

print(punto.x) -- Salida: 0

Explicación

  • La tabla porDefecto proporciona valores predeterminados.
  • setmetatable establece la metatabla para punto con __index apuntando a porDefecto.
  • Acceder a punto.x devuelve 0 porque punto hereda de porDefecto.

Conclusión

Las tablas en Lua son poderosas y flexibles, sirviendo como arreglos, diccionarios, e incluso objetos. Entender cómo crear, manipular y extender tablas es crucial para una programación efectiva en Lua. Practica usando tablas en diferentes escenarios para volverte competente en su uso.
