Introducción a los Bucles

Los bucles son constructos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. Lua proporciona varios tipos de bucles para manejar diferentes escenarios.

Tipos de Bucles en Lua

Bucle `while`

El bucle while repite un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

while condition do
    -- código a ejecutar
end

Ejemplo

local count = 1
while count <= 5 do
    print("El conteo es: " .. count)
    count = count + 1
end
  • Explicación: Este bucle imprimirá los números del 1 al 5. El bucle continúa mientras count sea menor o igual a 5.

Bucle `for`

El bucle for se utiliza para iterar sobre un rango de números o a través de elementos de una tabla.

Bucle for Numérico

El bucle for numérico itera sobre un rango de números.

Sintaxis

for variable = start, stop, step do
    -- código a ejecutar
end

Ejemplo

for i = 1, 5 do
    print("Número: " .. i)
end
  • Explicación: Este bucle imprimirá los números del 1 al 5. La variable del bucle i comienza en 1 y se incrementa en 1 hasta que alcanza 5.

Bucle for Genérico

El bucle for genérico se utiliza para iterar sobre elementos de una tabla.

Sintaxis

for key, value in pairs(table) do
    -- código a ejecutar
end

Ejemplo

local fruits = {apple = "red", banana = "yellow", grape = "purple"}
for fruit, color in pairs(fruits) do
    print(fruit .. " es " .. color)
end
  • Explicación: Este bucle itera sobre la tabla fruits e imprime cada fruta y su color.

Bucle `repeat`

El bucle repeat es similar al bucle while pero verifica la condición después de ejecutar el bloque de código.

Sintaxis

repeat
    -- código a ejecutar
until condition

Ejemplo

local count = 1
repeat
    print("El conteo es: " .. count)
    count = count + 1
until count > 5
  • Explicación: Este bucle imprimirá los números del 1 al 5. El bucle continúa hasta que count sea mayor que 5.

Comparación de Tipos de Bucle

| Tipo de Bucle | Verificación de Condición | Caso de Uso | |----------------|---------------------------|--------------------------------------------| | while | Antes del bucle | Cuando el número de iteraciones es desconocido| | for Numérico | Antes del bucle | Cuando se itera sobre un rango de números | | for Genérico | Antes del bucle | Cuando se itera sobre elementos de una tabla | | repeat | Después del bucle | Cuando el bloque de código necesita ejecutarse al menos una vez |

Ejercicios

Ejercicio 1: Usando Bucle `while`

Escribe un bucle while que imprima los primeros 10 números pares.

local num = 2
while num <= 20 do
    print(num)
    num = num + 2
end

Ejercicio 2: Usando Bucle `for` Numérico

Escribe un bucle for numérico que imprima los cuadrados de los números del 1 al 5.

for i = 1, 5 do
    print(i * i)
end

Ejercicio 3: Usando Bucle `for` Genérico

Escribe un bucle for genérico que imprima las claves y valores de una tabla que contenga tres de tus películas favoritas y sus años de lanzamiento.

local movies = {Inception = 2010, Matrix = 1999, Interstellar = 2014}
for movie, year in pairs(movies) do
    print(movie .. " fue lanzada en " .. year)
end

Ejercicio 4: Usando Bucle `repeat`

Escribe un bucle repeat que siga pidiendo al usuario que ingrese un número hasta que ingrese un número mayor que 10.

local num
repeat
    print("Ingresa un número:")
    num = tonumber(io.read())
until num > 10
print("¡Ingresaste un número mayor que 10!")

Conclusión

Los bucles son esenciales para realizar tareas repetitivas de manera eficiente. Lua proporciona varios tipos de bucles (while, for y repeat) para manejar diferentes escenarios. Entender cómo y cuándo usar cada tipo de bucle es crucial para escribir programas efectivos en Lua. Practica usando estos bucles con varios ejemplos y ejercicios para solidificar tu comprensión.
