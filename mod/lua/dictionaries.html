Introducción a los Diccionarios

En Lua, los diccionarios se conocen como tablas. Las tablas son una estructura de datos versátil que se puede usar para representar arreglos, listas, conjuntos y diccionarios (también conocidos como arreglos asociativos o mapas). Te permiten almacenar pares clave-valor, donde cada clave es única y está asociada con un valor.

Creando un Diccionario

Para crear un diccionario en Lua, usas el constructor de tablas {} y defines pares clave-valor dentro de él.

Ejemplo:

-- Creando un diccionario con algunos pares clave-valor
local person = {
    name = "John",
    age = 30,
    occupation = "Engineer"
}

En este ejemplo:

  • name, age y occupation son claves.
  • "John", 30 y "Engineer" son los valores correspondientes.

Accediendo a Valores

Puedes acceder a los valores en un diccionario usando las claves.

Ejemplo:

print(person.name)       -- Salida: John
print(person["age"])     -- Salida: 30
print(person.occupation) -- Salida: Engineer
  • person.name y person["name"] ambos acceden al valor asociado con la clave name.
  • De manera similar, person.age y person["age"] acceden al valor asociado con la clave age.

Modificando Valores

Puedes modificar los valores en un diccionario asignando nuevos valores a las claves.

Ejemplo:

person.age = 31
person["occupation"] = "Senior Engineer"

print(person.age)          -- Salida: 31
print(person.occupation)   -- Salida: Senior Engineer

Añadiendo y Eliminando Pares Clave-Valor

Puedes añadir nuevos pares clave-valor a un diccionario o eliminar los existentes.

Añadiendo:

person.gender = "Male"
person["country"] = "USA"

print(person.gender)   -- Salida: Male
print(person.country)  -- Salida: USA

Eliminando:

Para eliminar un par clave-valor, estableces la clave en nil.

person.gender = nil
person["country"] = nil

print(person.gender)   -- Salida: nil
print(person.country)  -- Salida: nil

Iterando Sobre un Diccionario

Puedes iterar sobre todos los pares clave-valor en un diccionario usando la función pairs.

Ejemplo:

for key, value in pairs(person) do
    print(key, value)
end

Esto producirá:

name    John
age     31
occupation  Senior Engineer

Diccionarios Anidados

Las tablas de Lua pueden contener otras tablas, lo que te permite crear diccionarios anidados.

Ejemplo:

local company = {
    name = "TechCorp",
    employees = {
        {name = "Alice", position = "Developer"},
        {name = "Bob", position = "Designer"}
    }
}

print(company.employees[1].name)  -- Salida: Alice
print(company.employees[2].position)  -- Salida: Designer

Uso Avanzado

Metatablas y Metamétodos

Las tablas de Lua pueden tener metatablas, lo que te permite cambiar el comportamiento de las tablas. Los metamétodos son funciones especiales que se pueden definir en una metatabla para personalizar las operaciones en las tablas.

Ejemplo:

local mt = {
    __index = function(table, key)
        return "Key not found"
    end
}

local t = setmetatable({}, mt)
print(t.someKey)  -- Salida: Key not found

En este ejemplo, si no se encuentra una clave en la tabla t, se llama al metamétodo __index, devolviendo "Key not found".

Resumen

  • Las tablas de Lua se pueden usar como diccionarios para almacenar pares clave-valor.
  • Puedes crear, acceder, modificar, añadir y eliminar pares clave-valor en un diccionario.
  • La iteración sobre diccionarios se realiza usando la función pairs.
  • Las tablas se pueden anidar para crear estructuras de datos complejas.
  • Las metatablas y los metamétodos proporcionan una personalización avanzada del comportamiento de las tablas.

Entender los diccionarios en Lua es crucial para gestionar y manipular datos de manera eficiente. Con estos conceptos básicos y avanzados, puedes aprovechar las tablas de Lua para construir programas robustos y flexibles.
