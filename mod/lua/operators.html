Introducción

Los operadores son fundamentales en cualquier lenguaje de programación, y Lua no es una excepción. Te permiten realizar varias operaciones sobre los datos, como cálculos aritméticos, comparaciones y operaciones lógicas. Esta sección cubrirá los diferentes tipos de operadores disponibles en Lua, desde la aritmética básica hasta operadores lógicos y relacionales más avanzados.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas.

  • + : Suma
  • - : Resta
  • * : Multiplicación
  • / : División
  • % : Módulo (resto de la división)
  • ^ : Exponenciación
  • - : Negación unaria

Ejemplos

local a = 10
local b = 3

print(a + b)  -- Salida: 13
print(a - b)  -- Salida: 7
print(a * b)  -- Salida: 30
print(a / b)  -- Salida: 3.3333333333333
print(a % b)  -- Salida: 1
print(a ^ b)  -- Salida: 1000
print(-a)     -- Salida: -10

Operadores Relacionales

Los operadores relacionales comparan dos valores y devuelven un resultado booleano (true o false).

  • == : Igual a
  • ~= : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a

Ejemplos

local x = 5
local y = 10

print(x == y)  -- Salida: false
print(x ~= y)  -- Salida: true
print(x > y)   -- Salida: false
print(x < y)   -- Salida: true
print(x >= y)  -- Salida: false
print(x <= y)  -- Salida: true

Operadores Lógicos

Los operadores lógicos se utilizan para combinar o invertir valores booleanos.

  • and : Y lógico
  • or : O lógico
  • not : NO lógico

Ejemplos

local a = true
local b = false

print(a and b)  -- Salida: false
print(a or b)   -- Salida: true
print(not a)    -- Salida: false

Operador de Concatenación

El operador de concatenación (..) se utiliza para unir dos cadenas de texto.

Ejemplos

local str1 = "Hola"
local str2 = "Mundo"

print(str1 .. " " .. str2)  -- Salida: Hola Mundo

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. Lua soporta el operador de asignación básico =.

Ejemplos

local a = 10
local b = 20

a = a + b  -- a es ahora 30
print(a)   -- Salida: 30

Operadores de Asignación Compuesta

Lua no tiene operadores de asignación compuesta como += o -=, pero puedes lograr el mismo efecto utilizando el operador de asignación básico.

Ejemplos

local a = 10

a = a + 5  -- Equivalente a a += 5 en otros lenguajes
print(a)   -- Salida: 15

Precedencia de Operadores

La precedencia de operadores determina el orden en el que se realizan las operaciones. Lua sigue un orden específico de precedencia, que se resume en la tabla a continuación:

Operador Descripción
^ Exponenciación
not NO lógico
*, /, % Multiplicación, División, Módulo
+, - Suma, Resta
.. Concatenación
<, >, <=, >=, ==, ~= Operadores relacionales
and Y lógico
or O lógico

Ejemplos

local result = 5 + 2 * 3  -- La multiplicación se realiza primero
print(result)  -- Salida: 11

local result2 = (5 + 2) * 3  -- Los paréntesis cambian el orden de las operaciones
print(result2)  -- Salida: 21

Conclusión

Los operadores son herramientas esenciales en Lua que te permiten realizar una amplia gama de operaciones sobre los datos. Entender cómo usar operadores aritméticos, relacionales, lógicos y de concatenación, así como conocer el orden de precedencia, te permitirá escribir código Lua más eficiente y efectivo. Practica usando estos operadores en diferentes escenarios para familiarizarte con su funcionalidad.
