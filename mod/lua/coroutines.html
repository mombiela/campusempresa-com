Coroutines en Lua
Módulos y Paquetes ►

Las coroutines son una característica poderosa en Lua que te permiten realizar multitarea cooperativa. Te permiten pausar y reanudar funciones, facilitando el manejo de tareas asincrónicas, la gestión de estados y la creación de flujos de control complejos.

Introducción a las Coroutines

  • Definición: Las coroutines son funciones que pueden ser pausadas y reanudadas, permitiendo la multitarea cooperativa.
  • Casos de Uso: Útiles en escenarios como programación asincrónica, máquinas de estados y flujos de control complejos.

Conceptos Básicos

  • Estados de las Coroutines: Las coroutines pueden estar en uno de varios estados: suspendido, en ejecución, muerto.
  • Funciones de Coroutines: coroutine.create, coroutine.resume, coroutine.yield, coroutine.status.

Creación y Ejecución de Coroutines

Para crear una coroutine, usa coroutine.create. Esta función toma otra función como argumento y devuelve un objeto coroutine.

function myCoroutine()
    print("¡Hola desde la coroutine!")
end

co = coroutine.create(myCoroutine)
print(co)  -- Salida: thread: 0x...

Para iniciar o reanudar una coroutine, usa coroutine.resume.

coroutine.resume(co)  -- Salida: ¡Hola desde la coroutine!

Pausar y Reanudar

Las coroutines pueden ceder la ejecución de vuelta al llamador usando coroutine.yield.

function myCoroutine()
    for i = 1, 3 do
        print("Iteración de coroutine: " .. i)
        coroutine.yield()
    end
end

co = coroutine.create(myCoroutine)

coroutine.resume(co)  -- Salida: Iteración de coroutine: 1
coroutine.resume(co)  -- Salida: Iteración de coroutine: 2
coroutine.resume(co)  -- Salida: Iteración de coroutine: 3

Estados de las Coroutines

Puedes verificar el estado de una coroutine usando coroutine.status.

print(coroutine.status(co))  -- Salida: suspendido
coroutine.resume(co)
print(coroutine.status(co))  -- Salida: muerto

Uso Avanzado

Pasar Argumentos

Puedes pasar argumentos a coroutine.resume y recibirlos en la función coroutine.

function myCoroutine(a, b)
    print("Coroutine recibió: " .. a .. ", " .. b)
end

co = coroutine.create(myCoroutine)
coroutine.resume(co, 10, 20)  -- Salida: Coroutine recibió: 10, 20

Devolver Valores

Las coroutines pueden devolver valores usando coroutine.yield.

function myCoroutine()
    return coroutine.yield(10, 20)
end

co = coroutine.create(myCoroutine)
_, a, b = coroutine.resume(co)
print(a, b)  -- Salida: 10, 20

Envoltorios de Coroutines

Puedes usar coroutine.wrap para simplificar el uso de coroutines. Devuelve una función que reanuda la coroutine cada vez que se llama.

function myCoroutine()
    for i = 1, 3 do
        print("Iteración de coroutine: " .. i)
        coroutine.yield()
    end
end

co = coroutine.wrap(myCoroutine)
co()  -- Salida: Iteración de coroutine: 1
co()  -- Salida: Iteración de coroutine: 2
co()  -- Salida: Iteración de coroutine: 3

Resumen

  • Las coroutines en Lua permiten la multitarea cooperativa al pausar y reanudar funciones.
  • Las funciones clave incluyen coroutine.create, coroutine.resume, coroutine.yield y coroutine.status.
  • Las coroutines pueden ceder y reanudar con argumentos y valores de retorno.
  • coroutine.wrap proporciona una interfaz simplificada para trabajar con coroutines.

Entender y utilizar las coroutines puede mejorar enormemente tu capacidad para gestionar flujos de control complejos y tareas asincrónicas en Lua.
