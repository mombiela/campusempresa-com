Introducción

La Biblioteca de Depuración en Lua es una herramienta poderosa que permite a los desarrolladores inspeccionar y manipular la ejecución de programas Lua. Proporciona funciones para acceder y modificar variables, establecer puntos de interrupción y rastrear la ejecución del programa. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de la Biblioteca de Depuración.

Conceptos Clave

  • Depuración: El proceso de identificar y eliminar errores de los programas de computadora.
  • Puntos de Interrupción: Puntos en el programa donde la ejecución se detendrá, permitiendo la inspección del estado del programa.
  • Rastreo de Pila: Un informe de los marcos de pila activos en un cierto punto en el tiempo durante la ejecución de un programa.

Uso Básico

Cargando la Biblioteca de Depuración

Para usar la Biblioteca de Depuración, necesitas requerirla en tu script Lua:

local debug = require("debug")

Inspeccionando Variables

Puedes usar la función debug.getlocal para inspeccionar variables locales dentro de una función.

Ejemplo:

function exampleFunction()
    local a = 10
    local b = 20
    debug.debug()  -- Entrar en modo interactivo
end

exampleFunction()

Cuando se llama a la función debug.debug(), Lua entra en un modo interactivo donde puedes inspeccionar y modificar variables.

Estableciendo Puntos de Interrupción

Los puntos de interrupción pueden establecerse usando la función debug.sethook.

Ejemplo:

local function hook()
    print("¡Punto de interrupción alcanzado!")
    debug.debug()  -- Entrar en modo interactivo
end

debug.sethook(hook, "l")  -- Establecer un gancho de línea

function exampleFunction()
    local a = 10
    local b = 20
    print(a + b)
end

exampleFunction()

En este ejemplo, la función de gancho se llama antes de cada línea de código en exampleFunction, permitiéndote inspeccionar el estado del programa.

Uso Avanzado

Rastreo de Pila

Puedes obtener un rastreo de pila usando la función debug.traceback.

Ejemplo:

function exampleFunction()
    local a = 10
    local b = 20
    print(debug.traceback("Rastreo de pila:"))
end

exampleFunction()

Esto imprimirá el rastreo de pila en el punto donde se llama a debug.traceback.

Modificando Variables

Puedes modificar variables locales usando la función debug.setlocal.

Ejemplo:

function exampleFunction()
    local a = 10
    local b = 20
    print("Antes de la modificación: a = " .. a)
    debug.setlocal(1, 1, 30)  -- Modificar la primera variable local en el primer nivel de pila
    print("Después de la modificación: a = " .. a)
end

exampleFunction()

En este ejemplo, el valor de a se modifica de 10 a 30.

Perfilado

Puedes usar la Biblioteca de Depuración para perfilar tu código estableciendo ganchos para contar llamadas a funciones y tiempo de ejecución.

Ejemplo:

local call_count = 0

local function hook(event)
    if event == "call" then
        call_count = call_count + 1
    end
end

debug.sethook(hook, "cr")  -- Establecer ganchos para eventos de llamada y retorno

function exampleFunction()
    local a = 10
    local b = 20
    return a + b
end

exampleFunction()
exampleFunction()

print("Conteo de llamadas a funciones: " .. call_count)

Este ejemplo cuenta el número de llamadas a funciones realizadas durante la ejecución del script.

Conclusión

La Biblioteca de Depuración en Lua es una herramienta versátil que puede ayudarte a inspeccionar, modificar y perfilar tus programas Lua. Al entender cómo usar sus funciones, puedes depurar y optimizar tu código de manera efectiva. Recuerda usar estas herramientas sabiamente, ya que el uso excesivo de las características de depuración puede afectar el rendimiento de tu programa.
