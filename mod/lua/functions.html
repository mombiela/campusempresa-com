Introducción a las Funciones

Las funciones son un bloque de construcción fundamental en Lua, permitiéndote encapsular código en bloques reutilizables. Ayudan a organizar el código, evitar la repetición y hacer que los programas sean más modulares.

Definiendo una Función

En Lua, las funciones se definen usando la palabra clave function seguida del nombre de la función y un par de paréntesis.

function sayHello()
    print("¡Hola, Mundo!")
end
  • function: Palabra clave para definir una función.
  • sayHello: Nombre de la función.
  • print("¡Hola, Mundo!"): Cuerpo de la función, que imprime "¡Hola, Mundo!" en la consola.

Llamando a una Función

Para ejecutar el código dentro de una función, necesitas llamarla por su nombre seguido de paréntesis.

sayHello()  -- Salida: ¡Hola, Mundo!

Parámetros y Argumentos de Función

Las funciones pueden tomar parámetros, que son variables pasadas a la función para personalizar su comportamiento.

Parámetro Único

function greet(name)
    print("¡Hola, " .. name .. "!")
end

greet("Alice")  -- Salida: ¡Hola, Alice!

Múltiples Parámetros

function add(a, b)
    return a + b
end

print(add(3, 4))  -- Salida: 7
  • return: Palabra clave usada para devolver un valor desde la función.

Alcance de Variables

Las variables definidas dentro de una función son locales a esa función y no pueden ser accedidas fuera de ella.

function example()
    local x = 10  -- Variable local
    print(x)
end

example()  -- Salida: 10
print(x)  -- Error: x no está definida

Funciones Anónimas

Las funciones en Lua pueden ser anónimas, lo que significa que no tienen un nombre. Estas se usan a menudo como argumentos para otras funciones o se asignan a variables.

local square = function(x)
    return x * x
end

print(square(5))  -- Salida: 25

Funciones de Orden Superior

Lua soporta funciones de orden superior, que son funciones que pueden tomar otras funciones como argumentos o devolverlas como resultados.

Ejemplo: Usando una Función como Argumento

function applyFunction(func, value)
    return func(value)
end

local double = function(x)
    return x * 2
end

print(applyFunction(double, 5))  -- Salida: 10

Clausuras

Una clausura es una función que captura las variables locales de su ámbito envolvente.

function makeCounter()
    local count = 0
    return function()
        count = count + 1
        return count
    end
end

local counter = makeCounter()
print(counter())  -- Salida: 1
print(counter())  -- Salida: 2

Funciones Recursivas

Una función recursiva es aquella que se llama a sí misma. Esto es útil para tareas que pueden descomponerse en subtareas similares.

Ejemplo: Función Factorial

function factorial(n)
    if n == 0 then
        return 1
    else
        return n * factorial(n - 1)
    end
end

print(factorial(5))  -- Salida: 120

Optimización de Llamadas en Cola

Lua soporta la optimización de llamadas en cola, lo que permite que ciertos tipos de funciones recursivas se ejecuten sin aumentar la pila de llamadas.

Ejemplo: Función Recursiva en Cola

function tailFactorial(n, acc)
    acc = acc or 1
    if n == 0 then
        return acc
    else
        return tailFactorial(n - 1, n * acc)
    end
end

print(tailFactorial(5))  -- Salida: 120

Resumen

  • Las funciones en Lua se definen usando la palabra clave function.
  • Las funciones pueden tomar parámetros y devolver valores.
  • Las variables dentro de las funciones son locales a esa función.
  • Lua soporta funciones anónimas, funciones de orden superior y clausuras.
  • Las funciones recursivas pueden llamarse a sí mismas, y Lua soporta la optimización de llamadas en cola para una recursión eficiente.

Entender las funciones es crucial para escribir código modular y reutilizable en Lua. Practica definiendo, llamando y manipulando funciones para volverte competente en su uso.
