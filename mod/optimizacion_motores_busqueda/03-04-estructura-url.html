Estructura de URL

Siguiente ►

La estructura de las URLs (Uniform Resource Locators) es un componente crucial del SEO On-Page. Una URL bien estructurada no solo mejora la experiencia del usuario, sino que también facilita a los motores de búsqueda la indexación y comprensión del contenido de tu sitio web. En esta sección, aprenderás cómo optimizar las URLs para mejorar el posicionamiento en los motores de búsqueda.

Conceptos Clave

  1. URL Amigable: Una URL que es fácil de leer y entender tanto para los usuarios como para los motores de búsqueda.
  2. Palabras Clave en la URL: Incluir palabras clave relevantes en la URL puede mejorar el SEO.
  3. Estructura Jerárquica: La estructura de la URL debe reflejar la jerarquía del contenido del sitio.
  4. Longitud de la URL: Las URLs más cortas tienden a ser más efectivas.
  5. Uso de Guiones: Los guiones (-) son preferibles a los guiones bajos (_) para separar palabras en una URL.

Buenas Prácticas para la Estructura de URL

  1. Crear URLs Amigables

Las URLs deben ser fáciles de leer y entender. Evita el uso de caracteres especiales, números y letras sin sentido.

Ejemplo:

Mala URL: www.ejemplo.com/p=12345
Buena URL: www.ejemplo.com/productos/zapatos-deportivos

  1. Incluir Palabras Clave

Incluir palabras clave relevantes en la URL puede ayudar a los motores de búsqueda a entender de qué trata la página.

Ejemplo:

Mala URL: www.ejemplo.com/categoría1/producto2
Buena URL: www.ejemplo.com/zapatos/deportivos

  1. Mantener una Estructura Jerárquica

La estructura de la URL debe reflejar la jerarquía del contenido del sitio, lo que facilita la navegación tanto para los usuarios como para los motores de búsqueda.

Ejemplo:

www.ejemplo.com/categoría/subcategoría/producto

  1. Mantener las URLs Cortas

Las URLs más cortas son más fáciles de recordar y compartir. Además, los motores de búsqueda prefieren URLs concisas.

Ejemplo:

Mala URL: www.ejemplo.com/categoría/subcategoría/producto-con-muchas-palabras
Buena URL: www.ejemplo.com/categoría/producto

  1. Usar Guiones para Separar Palabras

Los guiones (-) son preferibles a los guiones bajos (_) para separar palabras en una URL, ya que los motores de búsqueda los interpretan mejor.

Ejemplo:

Mala URL: www.ejemplo.com/zapatos_deportivos
Buena URL: www.ejemplo.com/zapatos-deportivos

Ejercicio Práctico

Ejercicio 1: Optimización de URLs

A continuación, se presentan algunas URLs. Optimízalas siguiendo las buenas prácticas mencionadas anteriormente.

  1. Original URL: www.ejemplo.com/p=98765 Optimizada URL: ___________________________

  2. Original URL: www.ejemplo.com/categoría1/producto2 Optimizada URL: ___________________________

  3. Original URL: www.ejemplo.com/categoría/subcategoría/producto-con-muchas-palabras Optimizada URL: ___________________________

  4. Original URL: www.ejemplo.com/zapatos_deportivos Optimizada URL: ___________________________

Soluciones

  1. Original URL: www.ejemplo.com/p=98765 Optimizada URL: www.ejemplo.com/productos/zapatos-deportivos

  2. Original URL: www.ejemplo.com/categoría1/producto2 Optimizada URL: www.ejemplo.com/zapatos/deportivos

  3. Original URL: www.ejemplo.com/categoría/subcategoría/producto-con-muchas-palabras Optimizada URL: www.ejemplo.com/categoría/producto

  4. Original URL: www.ejemplo.com/zapatos_deportivos Optimizada URL: www.ejemplo.com/zapatos-deportivos

Resumen

Optimizar la estructura de las URLs es una parte esencial del SEO On-Page. Asegúrate de crear URLs amigables, incluir palabras clave relevantes, mantener una estructura jerárquica, usar guiones para separar palabras y mantener las URLs cortas. Estas prácticas no solo mejorarán la experiencia del usuario, sino que también facilitarán a los motores de búsqueda la indexación y comprensión de tu contenido.
