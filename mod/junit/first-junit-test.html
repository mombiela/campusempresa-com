Primer Test de JUnit
Entendiendo las Aserciones ►

En esta sección, cubriremos los conceptos básicos para escribir tu primer test de JUnit. JUnit es un popular marco de pruebas para Java que ayuda a los desarrolladores a escribir y ejecutar pruebas repetibles. Al final de esta sección, podrás crear un caso de prueba simple de JUnit y comprender sus componentes básicos.

Conceptos Clave

  • Marco de JUnit: Un marco de pruebas para Java.
  • Caso de Prueba: Una unidad única de prueba.
  • Aserciones: Declaraciones que verifican si una condición es verdadera.
  • Anotaciones: Marcadores especiales en el código que proporcionan metadatos sobre los métodos de prueba.

Configuración de JUnit

Antes de escribir tu primer test de JUnit, necesitas configurar JUnit en tu proyecto. Esto se puede hacer usando una herramienta de construcción como Maven o Gradle, o añadiendo manualmente la biblioteca de JUnit a tu proyecto.

Usando Maven

Añade la siguiente dependencia a tu archivo pom.xml:

<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>

Usando Gradle

Añade lo siguiente a tu archivo build.gradle:

dependencies {
    testImplementation 'junit:junit:4.13.2'
}

Escribiendo Tu Primer Caso de Prueba

Vamos a escribir un caso de prueba simple para entender la estructura básica de un test de JUnit.

Código de Ejemplo

import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {

    @Test
    public void testAddition() {
        Calculator calculator = new Calculator();
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

Explicación

  1. Declaraciones de Importación:

    • import org.junit.Test; - Importa la anotación @Test.
    • import static org.junit.Assert.assertEquals; - Importa el método assertEquals para las aserciones.

  2. Clase de Prueba:

    • public class CalculatorTest - Define una clase de prueba llamada CalculatorTest.

  3. Método de Prueba:

    • @Test - Marca el método testAddition como un método de prueba.
    • public void testAddition() - Define un método de prueba llamado testAddition.

  4. Aserciones:

    • assertEquals(5, result); - Verifica si el resultado de la suma es igual a 5.

Clase Calculator

Para que la prueba anterior funcione, necesitas una clase Calculator simple:

public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

Ejecutando la Prueba

Puedes ejecutar la prueba usando el ejecutor de pruebas integrado de tu IDE o desde la línea de comandos usando Maven o Gradle.

Usando IDE

  • Haz clic derecho en la clase de prueba y selecciona "Run" o "Run as JUnit Test".

Usando Maven

Ejecuta el siguiente comando en tu terminal:

mvn test

Usando Gradle

Ejecuta el siguiente comando en tu terminal:

gradle test

Conclusión

En esta sección, aprendiste cómo configurar JUnit en tu proyecto y escribir tu primer caso de prueba simple. También aprendiste sobre los componentes básicos de un test de JUnit, incluidos los métodos de prueba, las aserciones y las anotaciones. Escribir pruebas es una habilidad esencial para cualquier desarrollador, y dominar JUnit te ayudará a asegurar que tu código sea confiable y esté libre de errores.
