Usando marcas y tarjetas
Siguiente ►

En este tema, aprenderemos a utilizar marcas y tarjetas en Tableau para personalizar y mejorar nuestras visualizaciones. Las marcas y tarjetas son herramientas poderosas que nos permiten controlar la apariencia y el comportamiento de los gráficos. Vamos a desglosar los conceptos clave y proporcionar ejemplos prácticos para que puedas aplicarlos en tus propios proyectos.

Conceptos Clave

1. Marcas

Las marcas son elementos visuales que representan datos en una visualización. Pueden ser puntos, líneas, barras, áreas, entre otros. Las marcas se configuran en la tarjeta de marcas (Marks card).

2. Tarjetas

Las tarjetas son paneles en Tableau que permiten controlar diferentes aspectos de las visualizaciones, como el color, el tamaño, la forma y la etiqueta de las marcas. Las tarjetas más comunes son:

  • Color
  • Tamaño
  • Etiqueta
  • Detalle
  • Forma

Desglose de las Tarjetas

Tarjeta de Color

Permite asignar colores a las marcas en función de los valores de un campo. Esto es útil para diferenciar categorías o rangos de datos.

Tarjeta de Tamaño

Permite ajustar el tamaño de las marcas en función de los valores de un campo. Esto es útil para representar la magnitud de los datos.

Tarjeta de Etiqueta

Permite mostrar etiquetas de texto en las marcas. Las etiquetas pueden ser valores de campos o cálculos personalizados.

Tarjeta de Detalle

Permite agregar más información a las marcas sin cambiar su apariencia visual. Esto es útil para desglosar datos en niveles más detallados.

Tarjeta de Forma

Permite cambiar la forma de las marcas. Esto es útil para diferenciar categorías de datos de manera visual.

Ejemplo Práctico

Vamos a crear una visualización simple utilizando marcas y tarjetas. Supongamos que tenemos un conjunto de datos de ventas con las siguientes columnas: Producto, Categoría, Ventas, y Fecha.

Paso 1: Crear una Visualización Básica

  1. Arrastra el campo Categoría al estante de filas.
  2. Arrastra el campo Ventas al estante de columnas.
  3. Tableau creará un gráfico de barras básico.

Paso 2: Usar la Tarjeta de Color

  1. Arrastra el campo Producto a la tarjeta de Color.
  2. Observa cómo las barras se colorean según el producto, permitiendo una diferenciación visual.

Paso 3: Usar la Tarjeta de Tamaño

  1. Arrastra el campo Ventas a la tarjeta de Tamaño.
  2. Observa cómo el tamaño de las barras cambia en función de las ventas, proporcionando una representación visual de la magnitud.

Paso 4: Usar la Tarjeta de Etiqueta

  1. Arrastra el campo Ventas a la tarjeta de Etiqueta.
  2. Observa cómo se muestran las etiquetas de ventas en cada barra, proporcionando información adicional.

Paso 5: Usar la Tarjeta de Detalle

  1. Arrastra el campo Fecha a la tarjeta de Detalle.
  2. Observa cómo se desglosan los datos de ventas por fecha, sin cambiar la apariencia visual de las barras.

Paso 6: Usar la Tarjeta de Forma

  1. Arrastra el campo Categoría a la tarjeta de Forma.
  2. Cambia la forma de las marcas para diferenciar las categorías de productos.

Ejercicio Práctico

Ejercicio 1: Personalización de una Visualización

  1. Crea una visualización de dispersión (scatter plot) utilizando los campos Ventas y Fecha.
  2. Usa la tarjeta de Color para diferenciar los puntos por Categoría.
  3. Usa la tarjeta de Tamaño para ajustar el tamaño de los puntos según Ventas.
  4. Usa la tarjeta de Etiqueta para mostrar las etiquetas de Producto en cada punto.
  5. Usa la tarjeta de Detalle para desglosar los datos por Fecha.

Solución del Ejercicio 1

  1. Arrastra Ventas al estante de columnas y Fecha al estante de filas.
  2. Cambia el tipo de gráfico a dispersión (scatter plot).
  3. Arrastra Categoría a la tarjeta de Color.
  4. Arrastra Ventas a la tarjeta de Tamaño.
  5. Arrastra Producto a la tarjeta de Etiqueta.
  6. Arrastra Fecha a la tarjeta de Detalle.

Resumen

En esta lección, hemos aprendido a utilizar marcas y tarjetas en Tableau para personalizar nuestras visualizaciones. Las marcas nos permiten representar datos de diferentes maneras, mientras que las tarjetas nos permiten controlar aspectos visuales como el color, el tamaño, la forma y las etiquetas de las marcas. Practicar con estos elementos te ayudará a crear visualizaciones más informativas y atractivas.
