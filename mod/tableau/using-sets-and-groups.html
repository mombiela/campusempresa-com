Introducción

En esta sección, exploraremos cómo usar Conjuntos y Grupos en Tableau para mejorar tus capacidades de análisis y visualización de datos. Los Conjuntos y Grupos son características poderosas que te permiten segmentar tus datos de maneras significativas, facilitando la obtención de insights y la creación de visualizaciones más dinámicas.

¿Qué son los Conjuntos y Grupos?

Conjuntos

  • Definición: Los conjuntos son campos personalizados que definen un subconjunto de datos basado en algunas condiciones.
  • Tipos:
    • Conjuntos Dinámicos: Se actualizan automáticamente según los datos.
    • Conjuntos Fijos: Permanecen estáticos a menos que se actualicen manualmente.
  • Casos de Uso:
    • Resaltar productos de mejor rendimiento.
    • Comparar segmentos específicos de datos.

Grupos

  • Definición: Los grupos se utilizan para combinar múltiples miembros de una dimensión en categorías de nivel superior.
  • Casos de Uso:
    • Simplificar datos complejos agrupando elementos similares.
    • Agregar datos para un mejor análisis.

Creación y Uso de Conjuntos

Creación de un Conjunto

  1. Selección Manual:
    • Selecciona las marcas que deseas incluir en el conjunto.
    • Haz clic derecho y elige "Crear Conjunto".
  2. Basado en Condiciones:
    • Haz clic derecho en una dimensión.
    • Elige "Crear" > "Conjunto".
    • Define las condiciones según las cuales se creará el conjunto.

Uso de Conjuntos en Visualizaciones

  • Filtrado de Datos: Arrastra el conjunto al estante de Filtros para filtrar tus datos.
  • Resaltado de Datos: Usa conjuntos para resaltar puntos de datos específicos en tus visualizaciones.
  • Comparación de Datos: Crea campos calculados para comparar datos dentro y fuera del conjunto.
Ejemplo:
1. Crea un conjunto para los 10 principales clientes por ventas.
2. Usa este conjunto para filtrar un gráfico de barras para mostrar solo estos principales clientes.

Creación y Uso de Grupos

Creación de un Grupo

  1. Agrupación Manual:
    • Selecciona los elementos que deseas agrupar.
    • Haz clic derecho y elige "Agrupar".
  2. Agrupación Automática:
    • Haz clic derecho en una dimensión.
    • Elige "Crear" > "Grupo".
    • Usa opciones como "Agrupar por Prefijo" para agrupar automáticamente elementos similares.

Uso de Grupos en Visualizaciones

  • Simplificación de Datos: Reemplaza una dimensión detallada con una dimensión agrupada para simplificar tu visualización.
  • Agregación de Datos: Usa grupos para agregar datos para un análisis de nivel superior.
Ejemplo:
1. Agrupa diferentes categorías de productos en categorías más amplias como "Electrónica", "Muebles", etc.
2. Usa esta dimensión agrupada en un gráfico de pastel para mostrar la distribución de ventas entre estas categorías más amplias.

Ejercicios Prácticos

Ejercicio 1: Creación de un Conjunto

  1. Abre Tableau y conéctate al conjunto de datos Sample Superstore.
  2. Crea un conjunto para los 10 principales clientes por ventas.
  3. Usa este conjunto para filtrar un gráfico de barras que muestre ventas por cliente.

Ejercicio 2: Creación de un Grupo

  1. Abre Tableau y conéctate al conjunto de datos Sample Superstore.
  2. Agrupa las categorías de productos en categorías más amplias.
  3. Usa esta dimensión agrupada en un gráfico de pastel para mostrar la distribución de ventas.

Conclusión

Los Conjuntos y Grupos son herramientas esenciales en Tableau que te ayudan a gestionar y analizar tus datos de manera más efectiva. Al entender cómo crearlos y usarlos, puedes mejorar tus visualizaciones y obtener insights más profundos en tus datos. Practica creando conjuntos y grupos con diferentes conjuntos de datos para volverte competente en el uso de estas características.
