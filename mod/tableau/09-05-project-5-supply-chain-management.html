Proyecto 5: Gestión de la cadena de suministro

Introducción

En este proyecto, aplicaremos los conocimientos adquiridos a lo largo del curso para crear un tablero de gestión de la cadena de suministro. Este tablero ayudará a visualizar y analizar datos clave relacionados con la cadena de suministro, como inventarios, tiempos de entrega, costos y eficiencia operativa.

Objetivos del Proyecto

  1. Visualizar el inventario actual y su distribución geográfica.
  2. Analizar los tiempos de entrega y su variabilidad.
  3. Evaluar los costos asociados a la cadena de suministro.
  4. Identificar áreas de mejora en la eficiencia operativa.

Paso 1: Conectar a la Fuente de Datos

Descripción de la Fuente de Datos

Para este proyecto, utilizaremos un conjunto de datos ficticio que incluye información sobre inventarios, tiempos de entrega, costos y ubicaciones de almacenes.

Conexión a la Fuente de Datos

  1. Abrir Tableau y seleccionar "Conectar a un archivo".
  2. Seleccionar el archivo CSV o Excel que contiene los datos de la cadena de suministro.
  3. Verificar que los datos se hayan importado correctamente.

Paso 2: Visualizar el Inventario Actual

Crear un Mapa de Inventario

  1. Arrastrar el campo "Ubicación del Almacén" a la vista.
  2. Arrastrar el campo "Cantidad de Inventario" a la tarjeta de tamaño.
  3. Seleccionar el tipo de gráfico "Mapa".
  4. Ajustar los colores y tamaños para mejorar la visualización.
// Ejemplo de código para crear un mapa de inventario
MAP [Ubicación del Almacén] SIZE [Cantidad de Inventario]

Filtrar por Tipo de Producto

  1. Arrastrar el campo "Tipo de Producto" a la tarjeta de filtros.
  2. Seleccionar los tipos de productos que se desean visualizar.
// Ejemplo de código para filtrar por tipo de producto
FILTER [Tipo de Producto]

Paso 3: Analizar los Tiempos de Entrega

Crear un Gráfico de Barras de Tiempos de Entrega

  1. Arrastrar el campo "Fecha de Entrega" al eje X.
  2. Arrastrar el campo "Tiempo de Entrega" al eje Y.
  3. Seleccionar el tipo de gráfico "Barras".
// Ejemplo de código para crear un gráfico de barras de tiempos de entrega
BAR [Fecha de Entrega] [Tiempo de Entrega]

Agregar una Línea de Tendencia

  1. Hacer clic derecho en el gráfico y seleccionar "Agregar Línea de Tendencia".
  2. Configurar la línea de tendencia según sea necesario.
// Ejemplo de código para agregar una línea de tendencia
TRENDLINE [Tiempo de Entrega]

Paso 4: Evaluar los Costos de la Cadena de Suministro

Crear un Gráfico de Líneas de Costos

  1. Arrastrar el campo "Fecha" al eje X.
  2. Arrastrar el campo "Costo Total" al eje Y.
  3. Seleccionar el tipo de gráfico "Líneas".
// Ejemplo de código para crear un gráfico de líneas de costos
LINE [Fecha] [Costo Total]

Filtrar por Centro de Costo

  1. Arrastrar el campo "Centro de Costo" a la tarjeta de filtros.
  2. Seleccionar los centros de costo que se desean analizar.
// Ejemplo de código para filtrar por centro de costo
FILTER [Centro de Costo]

Paso 5: Identificar Áreas de Mejora en la Eficiencia Operativa

Crear un Gráfico de Dispersión de Eficiencia

  1. Arrastrar el campo "Eficiencia Operativa" al eje X.
  2. Arrastrar el campo "Costo Total" al eje Y.
  3. Seleccionar el tipo de gráfico "Dispersión".
// Ejemplo de código para crear un gráfico de dispersión de eficiencia
SCATTER [Eficiencia Operativa] [Costo Total]

Agregar un Filtro de Fecha

  1. Arrastrar el campo "Fecha" a la tarjeta de filtros.
  2. Seleccionar el rango de fechas que se desea analizar.
// Ejemplo de código para agregar un filtro de fecha
FILTER [Fecha]

Conclusión

En este proyecto, hemos creado un tablero de gestión de la cadena de suministro que permite visualizar y analizar datos clave. Hemos cubierto la visualización del inventario, el análisis de los tiempos de entrega, la evaluación de los costos y la identificación de áreas de mejora en la eficiencia operativa. Este tablero puede ser utilizado por los gerentes de la cadena de suministro para tomar decisiones informadas y mejorar la eficiencia general.

Resumen de Conceptos Aprendidos

  • Conexión a fuentes de datos: Importar y verificar datos.
  • Visualización de datos: Crear mapas, gráficos de barras, líneas y dispersión.
  • Análisis de datos: Filtrar y agregar líneas de tendencia.
  • Evaluación de costos y eficiencia: Identificar áreas de mejora.
