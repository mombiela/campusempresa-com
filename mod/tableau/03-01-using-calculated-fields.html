Usando campos calculados

Siguiente ►

Introducción

En este módulo, aprenderás a utilizar campos calculados en Tableau. Los campos calculados son una herramienta poderosa que te permite crear nuevas dimensiones y medidas a partir de los datos existentes. Esto es esencial para realizar análisis más profundos y personalizados.

¿Qué es un campo calculado?

Un campo calculado es una nueva columna que se crea en tu fuente de datos mediante una fórmula o expresión. Estos campos pueden ser utilizados en tus visualizaciones de la misma manera que cualquier otro campo de datos.

Ventajas de los campos calculados:

  • Personalización: Permiten crear métricas y dimensiones personalizadas.
  • Flexibilidad: Facilitan la manipulación de datos sin necesidad de modificar la fuente de datos original.
  • Automatización: Ayudan a automatizar cálculos repetitivos y complejos.

Creando un campo calculado

Paso 1: Abrir el editor de campos calculados

  1. En Tableau, abre tu fuente de datos y selecciona la hoja de trabajo donde deseas crear el campo calculado.
  2. En el panel de datos, haz clic derecho en cualquier lugar y selecciona "Crear campo calculado".

Paso 2: Definir la fórmula

  1. Se abrirá una ventana emergente donde puedes definir tu fórmula.
  2. Asigna un nombre a tu campo calculado en la parte superior de la ventana.
  3. Escribe la fórmula en el área de texto proporcionada.

Ejemplo básico

Supongamos que tienes una columna llamada Ventas y otra llamada Costos. Quieres crear un campo calculado para el Beneficio.

[Beneficio] = [Ventas] - [Costos]

Ejemplo avanzado

Ahora, supongamos que quieres calcular un campo que clasifique las ventas en "Alta", "Media" y "Baja" según ciertos umbrales.

IF [Ventas] > 10000 THEN "Alta"
ELSEIF [Ventas] > 5000 THEN "Media"
ELSE "Baja"
END

Funciones comunes en campos calculados

Función Descripción
SUM() Suma los valores de un campo.
AVG() Calcula el promedio de los valores de un campo.
IF Evalúa una condición y devuelve un valor si es verdadera y otro si es falsa.
DATEADD() Añade un intervalo de tiempo a una fecha.
DATEDIFF() Calcula la diferencia entre dos fechas.
CONTAINS() Verifica si una cadena contiene una subcadena específica.

Ejercicio práctico

Ejercicio 1: Crear un campo calculado para el margen de beneficio

  1. Abre Tableau y carga una fuente de datos que contenga las columnas Ventas y Costos.
  2. Crea un nuevo campo calculado llamado Margen de Beneficio con la siguiente fórmula:
[Margen de Beneficio] = ([Ventas] - [Costos]) / [Ventas]
  1. Arrastra el nuevo campo Margen de Beneficio a la vista para visualizarlo.

Solución

[Margen de Beneficio] = ([Ventas] - [Costos]) / [Ventas]

Ejercicio 2: Clasificar ventas

  1. Crea un nuevo campo calculado llamado Clasificación de Ventas con la siguiente fórmula:
IF [Ventas] > 10000 THEN "Alta"
ELSEIF [Ventas] > 5000 THEN "Media"
ELSE "Baja"
END
  1. Arrastra el campo Clasificación de Ventas a la vista para visualizarlo.

Solución

IF [Ventas] > 10000 THEN "Alta"
ELSEIF [Ventas] > 5000 THEN "Media"
ELSE "Baja"
END

Errores comunes y consejos

Errores comunes

  1. Errores de sintaxis: Asegúrate de que todas las funciones y operadores estén correctamente escritos.
  2. Referencias incorrectas: Verifica que los nombres de los campos sean correctos y existan en tu fuente de datos.
  3. División por cero: Ten cuidado al realizar divisiones; asegúrate de que el denominador no sea cero.

Consejos

  • Prueba tus fórmulas: Utiliza la función de validación en el editor de campos calculados para verificar que tu fórmula sea correcta.
  • Documenta tus cálculos: Asigna nombres descriptivos a tus campos calculados para facilitar su comprensión y mantenimiento.
  • Utiliza comentarios: Puedes añadir comentarios en tus fórmulas utilizando // para mejorar la legibilidad.

Conclusión

En esta sección, has aprendido a crear y utilizar campos calculados en Tableau. Esta habilidad te permitirá realizar análisis más complejos y personalizados, mejorando la calidad de tus visualizaciones. En el próximo módulo, exploraremos cómo crear gráficos de doble eje para visualizar múltiples métricas en una sola vista.
