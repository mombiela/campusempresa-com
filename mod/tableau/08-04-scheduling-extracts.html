Programando extractos
Siguiente ►

En este tema, aprenderás cómo programar extractos en Tableau Server y Tableau Online. La programación de extractos es una funcionalidad crucial para mantener tus datos actualizados sin necesidad de intervención manual constante. A continuación, desglosaremos los conceptos clave, proporcionaremos ejemplos prácticos y ejercicios para reforzar tu comprensión.

¿Qué es un extracto?

Un extracto en Tableau es una copia de los datos que se almacena en un formato optimizado para el rendimiento. Los extractos permiten trabajar con grandes volúmenes de datos de manera más eficiente y rápida.

Ventajas de usar extractos:

  • Rendimiento mejorado: Los extractos están optimizados para consultas rápidas.
  • Trabajo sin conexión: Puedes trabajar con los datos sin necesidad de estar conectado a la fuente de datos original.
  • Reducción de carga en la base de datos: Al usar extractos, reduces la carga en la base de datos original.

Programando extractos en Tableau Server

Paso 1: Publicar el libro de trabajo con un extracto

  1. Crear un extracto:

    • Abre tu libro de trabajo en Tableau Desktop.
    • Conéctate a tu fuente de datos.
    • En el panel de datos, haz clic derecho en la fuente de datos y selecciona "Extraer datos...".
    • Configura las opciones de extracción según tus necesidades y haz clic en "Extraer".

  2. Publicar el libro de trabajo:

    • Ve a "Archivo" > "Publicar en Tableau Server".
    • Selecciona el proyecto y la ubicación donde deseas publicar el libro de trabajo.
    • Asegúrate de que la opción "Incluir extracto de datos" esté seleccionada.
    • Haz clic en "Publicar".

Paso 2: Programar la actualización del extracto

  1. Acceder a Tableau Server:

    • Inicia sesión en Tableau Server con tus credenciales.

  2. Navegar al libro de trabajo:

    • Encuentra el libro de trabajo que publicaste en el proyecto correspondiente.

  3. Configurar la programación:

    • Haz clic en el menú de opciones (tres puntos) junto al libro de trabajo y selecciona "Programar actualización de extracto".
    • Selecciona una programación existente o crea una nueva.
    • Configura la frecuencia y el horario de la actualización según tus necesidades.
    • Haz clic en "Guardar".

Ejemplo práctico

Supongamos que tienes un libro de trabajo con datos de ventas que se actualizan diariamente. Quieres que el extracto se actualice cada noche a la medianoche.

  1. Crear y publicar el extracto:

    - Conéctate a la base de datos de ventas.
- Extrae los datos y publica el libro de trabajo en Tableau Server.

  2. Configurar la programación:

    - Inicia sesión en Tableau Server.
- Encuentra el libro de trabajo en el proyecto "Ventas".
- Selecciona "Programar actualización de extracto".
- Configura la programación para que se ejecute diariamente a las 12:00 AM.
- Guarda la configuración.

Ejercicio práctico

Ejercicio 1: Programar un extracto diario

  1. Objetivo:

    • Publicar un libro de trabajo con datos de inventario y programar la actualización del extracto para que se ejecute diariamente a las 6:00 AM.

  2. Pasos:

    • Crea un extracto de los datos de inventario en Tableau Desktop.
    • Publica el libro de trabajo en Tableau Server o Tableau Online.
    • Configura la programación para que se ejecute diariamente a las 6:00 AM.

Solución:

  1. Crear y publicar el extracto:

    - Conéctate a la base de datos de inventario.
- Extrae los datos y publica el libro de trabajo en Tableau Server o Tableau Online.

  2. Configurar la programación:

    - Inicia sesión en Tableau Server o Tableau Online.
- Encuentra el libro de trabajo en el proyecto "Inventario".
- Selecciona "Programar actualización de extracto".
- Configura la programación para que se ejecute diariamente a las 6:00 AM.
- Guarda la configuración.

Conclusión

En esta sección, has aprendido cómo programar extractos en Tableau Server y Tableau Online. La programación de extractos es una herramienta poderosa para mantener tus datos actualizados y optimizar el rendimiento de tus visualizaciones. Asegúrate de practicar estos pasos con tus propios datos para familiarizarte con el proceso. En el próximo tema, exploraremos cómo colaborar con Tableau Online.
