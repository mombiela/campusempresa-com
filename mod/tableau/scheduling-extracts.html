Programación de Extractos
Usando Tableau Online ►

En esta sección, cubriremos el concepto de programación de extractos en Tableau, que es esencial para automatizar las actualizaciones de datos y asegurar que tus visualizaciones estén siempre actualizadas. Comenzaremos con lo básico y gradualmente avanzaremos a temas más avanzados.

Introducción a los Extractos

Los extractos son instantáneas de datos optimizadas para la agregación y cargadas en la memoria del sistema para ser rápidamente recuperadas para la visualización. Pueden mejorar significativamente el rendimiento, especialmente con conjuntos de datos grandes.

  • Beneficios de los Extractos:
    • Rendimiento mejorado.
    • Acceso a datos sin conexión.
    • Reducción de la carga en la fuente de datos original.

Creación de Extractos

Antes de programar extractos, necesitas crearlos. Aquí te mostramos cómo puedes crear un extracto en Tableau:

  1. Conéctate a tu fuente de datos.
  2. Selecciona los datos que quieres extraer.
  3. Haz clic en el menú Datos y selecciona Extraer Datos.
  4. Configura las opciones del extracto (por ejemplo, filtros, agregación).
  5. Haz clic en Extraer.
# Ejemplo: Creación de un Extracto
1. Abre Tableau y conéctate a tu fuente de datos.
2. Ve al menú Datos y selecciona 'Extraer Datos'.
3. Elige los campos y filtros que quieres incluir en el extracto.
4. Haz clic en 'Extraer' para crear el archivo de extracto.

Programación de Extractos en Tableau Desktop

Programar extractos en Tableau Desktop implica configurar un horario para actualizar los datos en intervalos especificados.

  • Pasos para Programar Extractos:
    1. Crea un extracto como se describió anteriormente.
    2. Publica el libro de trabajo en Tableau Server o Tableau Online.
    3. Configura el horario de actualización del extracto durante el proceso de publicación.
# Ejemplo: Programación de un Extracto en Tableau Desktop
1. Crea y guarda tu extracto.
2. Ve a 'Servidor' -> 'Publicar Libro de Trabajo'.
3. En el cuadro de diálogo 'Publicar Libro de Trabajo', selecciona 'Programación y Autenticación'.
4. Elige un horario existente o crea uno nuevo.
5. Publica el libro de trabajo.

Programación de Extractos en Tableau Server

Tableau Server proporciona opciones más robustas para programar y gestionar extractos.

  • Pasos para Programar Extractos:
    1. Navega a la pestaña de Fuentes de Datos en Tableau Server.
    2. Selecciona la fuente de datos que quieres programar.
    3. Haz clic en la pestaña de Actualizaciones de Extractos.
    4. Haz clic en Nueva Actualización de Extracto.
    5. Configura el horario (por ejemplo, frecuencia, hora).
# Ejemplo: Programación de un Extracto en Tableau Server
1. Inicia sesión en Tableau Server y ve a la pestaña 'Fuentes de Datos'.
2. Selecciona la fuente de datos que quieres programar.
3. Haz clic en 'Actualizaciones de Extractos' y luego en 'Nueva Actualización de Extracto'.
4. Configura la frecuencia (por ejemplo, diaria, semanal) y la hora para la actualización.
5. Guarda el horario.

Opciones Avanzadas de Programación

Para usuarios avanzados, Tableau proporciona opciones adicionales como actualizaciones incrementales y actualizaciones condicionales.

  • Actualización Incremental: Solo se añaden al extracto los datos nuevos o actualizados.
  • Actualización Condicional: Las actualizaciones se desencadenan en función de condiciones o eventos específicos.
# Ejemplo: Actualización Incremental
1. Al crear el extracto, selecciona 'Actualización incremental'.
2. Especifica la columna para rastrear cambios (por ejemplo, una columna de marca de tiempo).
3. Programa el extracto como de costumbre.

# Ejemplo: Actualización Condicional
1. Usa la API REST de Tableau para configurar actualizaciones condicionales.
2. Escribe un script para desencadenar la actualización en función de condiciones específicas.
3. Programa el script para que se ejecute en intervalos deseados.

Conclusión

Programar extractos en Tableau es una característica poderosa que asegura que tus visualizaciones de datos estén siempre actualizadas sin intervención manual. Al entender cómo crear, programar y gestionar extractos, puedes mejorar significativamente la eficiencia y la fiabilidad de tus paneles de Tableau.

  • Puntos Clave:
    • Los extractos mejoran el rendimiento y proporcionan acceso sin conexión.
    • Programar extractos automatiza las actualizaciones de datos.
    • Tableau Server ofrece opciones de programación más avanzadas.

Al dominar estos conceptos, puedes asegurar que tus visualizaciones de Tableau sean tanto eficientes como precisas, proporcionando información en tiempo real a tus interesados.
