Introducción

Los gráficos de doble eje, también conocidos como gráficos combinados, te permiten comparar dos medidas diferentes en una sola visualización. Esto puede ser particularmente útil para identificar relaciones entre variables, comparar tendencias y mejorar la narración de datos.

Conceptos clave

  • Gráfico de doble eje: Un gráfico que utiliza dos ejes para trazar dos medidas diferentes.
  • Eje primario: El eje principal utilizado para la primera medida.
  • Eje secundario: El eje secundario utilizado para la segunda medida.
  • Sincronización: Alinear las escalas de los ejes primario y secundario para una mejor comparación.

Creación de un gráfico de doble eje

Guía paso a paso

  1. Conectar a la fuente de datos: Abre Tableau y conéctate a tu fuente de datos.
  2. Arrastrar medidas a Filas: Arrastra la primera medida al estante de Filas.
  3. Arrastrar medidas a Columnas: Arrastra la segunda medida al estante de Columnas.
  4. Crear doble eje: Haz clic derecho en la segunda medida en el estante de Filas y selecciona "Doble eje".
  5. Sincronizar ejes: Haz clic derecho en el eje secundario y selecciona "Sincronizar eje" para alinear las escalas.

Ejemplo

Vamos a crear un gráfico de doble eje para comparar Ventas y Beneficio a lo largo del tiempo.

  1. Conectar a datos de muestra: Usa el conjunto de datos Sample - Superstore de Tableau.
  2. Arrastrar Fecha de pedido a Columnas: Arrastra Fecha de pedido al estante de Columnas.
  3. Arrastrar Ventas a Filas: Arrastra Ventas al estante de Filas.
  4. Arrastrar Beneficio a Filas: Arrastra Beneficio al estante de Filas, junto a Ventas.
  5. Crear doble eje: Haz clic derecho en la pastilla de Beneficio en el estante de Filas y selecciona "Doble eje".
  6. Sincronizar ejes: Haz clic derecho en el eje secundario (Beneficio) y selecciona "Sincronizar eje".
  7. Personalizar marcas: Cambia el tipo de marca para cada medida para visualizar mejor los datos (por ejemplo, línea para Ventas y barra para Beneficio).
Columnas: Fecha de pedido
Filas: Ventas, Beneficio
Clic derecho en Beneficio -> Doble eje
Clic derecho en eje secundario -> Sincronizar eje

Personalización de gráficos de doble eje

Cambio de tipos de marca

  • Combinación de línea y barra: Usa una línea para una medida y barras para otra para resaltar tendencias y diferencias.
  • Forma y color: Personaliza formas y colores para diferenciar entre las dos medidas.

Adición de líneas de referencia

  • Líneas de referencia: Añade líneas de referencia para resaltar valores o tendencias específicos.
  • Anotaciones: Usa anotaciones para proporcionar contexto adicional o ideas.

Formateo de ejes

  • Títulos de ejes: Personaliza los títulos de los ejes para indicar claramente qué representa cada eje.
  • Marcas de graduación: Ajusta las marcas de graduación para una mejor legibilidad.

Mejores prácticas

  • Sincronización: Siempre sincroniza los ejes para asegurar una comparación precisa.
  • Claridad: Usa colores y formas claros y distintos para diferenciar las medidas.
  • Contexto: Proporciona contexto con anotaciones y líneas de referencia para mejorar la comprensión.

Casos de uso comunes

  • Ventas vs. Beneficio: Compara ventas y beneficio a lo largo del tiempo para identificar tendencias y correlaciones.
  • Temperatura vs. Precipitación: Compara temperatura y precipitación para estudiar patrones climáticos.
  • Precios de acciones vs. Volumen: Analiza precios de acciones y volumen de negociación para entender el comportamiento del mercado.

Conclusión

Los gráficos de doble eje en Tableau son herramientas poderosas para comparar dos medidas en una sola visualización. Siguiendo las mejores prácticas y personalizando tus gráficos, puedes crear visualizaciones atractivas y perspicaces que mejoren tu narración de datos. Recuerda siempre sincronizar tus ejes y usar distinciones claras entre las medidas para asegurar que tu audiencia pueda interpretar fácilmente los datos.
