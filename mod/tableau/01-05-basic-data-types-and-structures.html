Tipos y estructuras de datos básicos
Siguiente ►

En este tema, aprenderemos sobre los tipos y estructuras de datos básicos en Tableau. Comprender estos conceptos es fundamental para trabajar eficazmente con datos y crear visualizaciones precisas y significativas.

Tipos de Datos en Tableau

Tableau reconoce varios tipos de datos que se pueden utilizar en las visualizaciones. Los tipos de datos más comunes son:

  1. Númerico (Number)

    • Entero (Integer): Números sin decimales.
    • Decimal (Float): Números con decimales.

  2. Cadena de texto (String)

    • Cualquier secuencia de caracteres, como nombres, direcciones, etc.

  3. Fecha y hora (Date & Time)

    • Fecha (Date): Solo la fecha (e.g., 2023-10-01).
    • Fecha y hora (Date & Time): Fecha y hora combinadas (e.g., 2023-10-01 14:30:00).

  4. Booleano (Boolean)

    • Valores de verdadero o falso (True/False).

  5. Geográfico (Geographic)

    • Datos que representan ubicaciones geográficas, como países, ciudades, códigos postales, etc.

Ejemplo de Tipos de Datos

| Nombre     | Edad | Fecha de Nacimiento | Empleado Activo | Ciudad       |
|------------|------|---------------------|-----------------|--------------|
| Juan Pérez | 30   | 1993-05-15          | True            | Madrid       |
| Ana López  | 25   | 1998-08-22          | False           | Barcelona    |
| Luis Gómez | 40   | 1983-01-10          | True            | Valencia     |

En este ejemplo:

  • "Nombre" es una cadena de texto.
  • "Edad" es un número entero.
  • "Fecha de Nacimiento" es una fecha.
  • "Empleado Activo" es un booleano.
  • "Ciudad" es una cadena de texto.

Estructuras de Datos en Tableau

Tableau puede conectarse a diversas fuentes de datos y manejar diferentes estructuras de datos. Las estructuras de datos más comunes son:

  1. Tablas

    • Datos organizados en filas y columnas, similar a una hoja de cálculo.

  2. Jerarquías

    • Estructuras que permiten agrupar datos en niveles, como Año > Mes > Día.

  3. Campos Calculados

    • Campos creados mediante fórmulas para realizar cálculos sobre los datos existentes.

  4. Uniones (Joins)

    • Combinación de datos de diferentes tablas basadas en un campo común.

  5. Blending de Datos

    • Combinación de datos de diferentes fuentes sin necesidad de una unión directa.

Ejemplo de Estructuras de Datos

Tabla Simple

| Producto   | Ventas | Fecha de Venta |
|------------|--------|----------------|
| Producto A | 100    | 2023-01-01     |
| Producto B | 150    | 2023-01-02     |
| Producto C | 200    | 2023-01-03     |

Jerarquía

Año > Mes > Día
2023 > Enero > 01
2023 > Enero > 02
2023 > Enero > 03

Campo Calculado

Ventas con IVA = Ventas * 1.21

Unión de Tablas

Tabla 1: Productos
| ProductoID | Nombre     |
|------------|------------|
| 1          | Producto A |
| 2          | Producto B |

Tabla 2: Ventas
| ProductoID | Ventas | Fecha de Venta |
|------------|--------|----------------|
| 1          | 100    | 2023-01-01     |
| 2          | 150    | 2023-01-02     |

Ejercicio Práctico

Ejercicio: Identificar Tipos de Datos

Dado el siguiente conjunto de datos, identifica el tipo de dato para cada columna:

| Nombre del Cliente | Edad | Fecha de Registro | Activo | País       |
|--------------------|------|-------------------|--------|------------|
| María García       | 28   | 2022-03-15        | True   | España     |
| Pedro Martínez     | 35   | 2021-07-22        | False  | México     |
| Laura Sánchez      | 22   | 2023-01-10        | True   | Argentina  |

Solución:

  • "Nombre del Cliente" es una cadena de texto.
  • "Edad" es un número entero.
  • "Fecha de Registro" es una fecha.
  • "Activo" es un booleano.
  • "País" es una cadena de texto.
