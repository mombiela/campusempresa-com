Conectando a Fuentes de Datos
Preparación Básica de Datos ►

Conectar a fuentes de datos es una habilidad fundamental en Tableau, que te permite visualizar y analizar datos de varios orígenes. Este tema te guiará a través del proceso de conexión a diferentes fuentes de datos, desde niveles básicos hasta avanzados.

Introducción a las Conexiones de Datos

Antes de profundizar en los detalles, es esencial entender qué son las conexiones de datos y por qué son cruciales en Tableau.

  • Conexiones de Datos: Los enlaces entre Tableau y las fuentes de datos que deseas analizar.
  • Importancia: Te permite extraer datos en Tableau para visualización y análisis.

Tipos de Fuentes de Datos

Tableau soporta una amplia gama de fuentes de datos. Aquí hay algunos tipos comunes:

  • Basadas en Archivos: Excel, CSV, JSON, PDF.
  • Servidores de Bases de Datos: MySQL, PostgreSQL, SQL Server, Oracle.
  • Basadas en la Nube: Google Sheets, Salesforce, Amazon Redshift.
  • Conectores de Datos Web: APIs, conectores personalizados.

Conectando a Fuentes de Datos Basadas en Archivos

Excel

  1. Abre Tableau.
  2. Haz clic en "Microsoft Excel" bajo el panel "Conectar".
  3. Selecciona el archivo Excel al que deseas conectarte.
  4. Arrastra y suelta las hojas desde el panel izquierdo al lienzo.
# Ejemplo: Conectando a un archivo Excel
1. Abre Tableau.
2. Haz clic en "Microsoft Excel".
3. Selecciona "SalesData.xlsx".
4. Arrastra "Sheet1" al lienzo.

CSV

  1. Abre Tableau.
  2. Haz clic en "Archivo de Texto" bajo el panel "Conectar".
  3. Selecciona el archivo CSV al que deseas conectarte.
  4. Arrastra y suelta el archivo al lienzo.
# Ejemplo: Conectando a un archivo CSV
1. Abre Tableau.
2. Haz clic en "Archivo de Texto".
3. Selecciona "SalesData.csv".
4. Arrastra el archivo al lienzo.

Conectando a Servidores de Bases de Datos

MySQL

  1. Abre Tableau.
  2. Haz clic en "MySQL" bajo el panel "Conectar".
  3. Ingresa los detalles del servidor (Servidor, Puerto, Base de Datos, Usuario, Contraseña).
  4. Haz clic en "Iniciar Sesión".
# Ejemplo: Conectando a una base de datos MySQL
1. Abre Tableau.
2. Haz clic en "MySQL".
3. Ingresa los detalles del servidor:
   - Servidor: `localhost`
   - Puerto: `3306`
   - Base de Datos: `sales_db`
   - Usuario: `root`
   - Contraseña: `password`
4. Haz clic en "Iniciar Sesión".

PostgreSQL

  1. Abre Tableau.
  2. Haz clic en "PostgreSQL" bajo el panel "Conectar".
  3. Ingresa los detalles del servidor (Servidor, Puerto, Base de Datos, Usuario, Contraseña).
  4. Haz clic en "Iniciar Sesión".
# Ejemplo: Conectando a una base de datos PostgreSQL
1. Abre Tableau.
2. Haz clic en "PostgreSQL".
3. Ingresa los detalles del servidor:
   - Servidor: `localhost`
   - Puerto: `5432`
   - Base de Datos: `sales_db`
   - Usuario: `postgres`
   - Contraseña: `password`
4. Haz clic en "Iniciar Sesión".

Conexiones de Datos Avanzadas

Conectores de Datos Web

  1. Abre Tableau.
  2. Haz clic en "Conector de Datos Web" bajo el panel "Conectar".
  3. Ingresa la URL del conector de datos web.
  4. Sigue las indicaciones para conectarte a los datos.
# Ejemplo: Conectando a un Conector de Datos Web
1. Abre Tableau.
2. Haz clic en "Conector de Datos Web".
3. Ingresa la URL: `https://example.com/webdataconnector`.
4. Sigue las indicaciones para conectarte a los datos.

SQL Personalizado

  1. Abre Tableau.
  2. Conéctate a una base de datos (por ejemplo, MySQL, PostgreSQL).
  3. Haz clic en "Nuevo SQL Personalizado".
  4. Ingresa tu consulta SQL.
-- Ejemplo: Consulta SQL Personalizada
SELECT 
    customer_id,
    customer_name,
    total_sales
FROM 
    sales_data
WHERE 
    total_sales > 1000;

Conclusión

Conectar a fuentes de datos en Tableau es un paso crítico en el proceso de visualización de datos. Ya sea que estés trabajando con datos basados en archivos, bases de datos, servicios en la nube o conectores web personalizados, Tableau proporciona un conjunto robusto de herramientas para ayudarte a llevar tus datos a la plataforma. Dominar estas conexiones te permitirá aprovechar todo el poder de Tableau para tus necesidades de análisis de datos.
