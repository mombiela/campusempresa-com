Combinación de datos
Siguiente ►

La combinación de datos (Data Blending) en Tableau es una técnica poderosa que permite integrar datos de diferentes fuentes en una sola vista. Esto es especialmente útil cuando los datos no están en la misma base de datos o cuando no se pueden unir directamente debido a limitaciones de la estructura de los datos.

Conceptos clave

  1. Fuente de datos primaria y secundaria:

    • Fuente de datos primaria: La fuente de datos principal que se utiliza en la visualización.
    • Fuente de datos secundaria: La fuente de datos adicional que se combina con la fuente primaria.

  2. Campos de combinación:

    • Los campos que se utilizan para combinar las fuentes de datos. Estos campos deben tener valores coincidentes para que la combinación sea efectiva.

  3. Relaciones de combinación:

    • Las relaciones que se establecen entre los campos de las fuentes de datos primaria y secundaria.

Pasos para combinar datos en Tableau

  1. Conectar a las fuentes de datos

Primero, conecta a las fuentes de datos que deseas combinar. Puedes hacerlo desde la pantalla de inicio de Tableau o desde el menú de datos.

1. Haz clic en "Conectar" en la pantalla de inicio de Tableau.
2. Selecciona las fuentes de datos que deseas utilizar (por ejemplo, una base de datos SQL y un archivo Excel).
3. Carga las fuentes de datos en Tableau.

  1. Establecer la fuente de datos primaria

Selecciona una de las fuentes de datos como la fuente primaria. Esta será la fuente principal que se utilizará en la visualización.

1. Arrastra un campo de la fuente de datos primaria a la vista.
2. Tableau automáticamente establece esta fuente como la fuente primaria.

  1. Agregar la fuente de datos secundaria

Agrega la fuente de datos secundaria a la vista. Tableau reconocerá automáticamente que estás intentando combinar datos y te permitirá establecer relaciones de combinación.

1. Arrastra un campo de la fuente de datos secundaria a la vista.
2. Tableau mostrará un icono de cadena junto al campo, indicando que se está utilizando una combinación.

  1. Establecer relaciones de combinación

Define las relaciones de combinación entre los campos de las fuentes de datos primaria y secundaria.

1. Haz clic en el icono de cadena junto al campo de la fuente secundaria.
2. En el cuadro de diálogo "Editar relaciones de combinación", selecciona los campos que deseas utilizar para la combinación.
3. Asegúrate de que los campos seleccionados tengan valores coincidentes.

  1. Crear la visualización

Ahora puedes crear visualizaciones utilizando datos de ambas fuentes. Tableau combinará los datos en función de las relaciones que has establecido.

1. Arrastra campos de ambas fuentes de datos a la vista.
2. Crea gráficos, tablas y otras visualizaciones utilizando los datos combinados.

Ejemplo práctico

Supongamos que tienes dos fuentes de datos: una base de datos SQL con información de ventas y un archivo Excel con datos de objetivos de ventas. Quieres combinar estos datos para comparar las ventas reales con los objetivos.

Paso 1: Conectar a las fuentes de datos

1. Conecta a la base de datos SQL y carga la tabla de ventas.
2. Conecta al archivo Excel y carga la hoja de objetivos de ventas.

Paso 2: Establecer la fuente de datos primaria

1. Arrastra el campo "Fecha de venta" de la tabla de ventas a la vista.
2. Tableau establece la tabla de ventas como la fuente primaria.

Paso 3: Agregar la fuente de datos secundaria

1. Arrastra el campo "Fecha de objetivo" de la hoja de objetivos de ventas a la vista.
2. Tableau muestra un icono de cadena junto al campo "Fecha de objetivo".

Paso 4: Establecer relaciones de combinación

1. Haz clic en el icono de cadena junto al campo "Fecha de objetivo".
2. En el cuadro de diálogo "Editar relaciones de combinación", selecciona "Fecha de venta" y "Fecha de objetivo" como los campos de combinación.
3. Asegúrate de que los campos tengan valores coincidentes.

Paso 5: Crear la visualización

1. Arrastra el campo "Ventas reales" de la tabla de ventas a la vista.
2. Arrastra el campo "Objetivo de ventas" de la hoja de objetivos de ventas a la vista.
3. Crea un gráfico de barras para comparar las ventas reales con los objetivos.

Ejercicio práctico

Ejercicio 1: Combinación de datos de ventas y marketing

Objetivo: Combinar datos de ventas y marketing para analizar el impacto de las campañas de marketing en las ventas.

Instrucciones:

  1. Conecta a una base de datos SQL con información de ventas.
  2. Conecta a un archivo Excel con datos de campañas de marketing.
  3. Establece la tabla de ventas como la fuente de datos primaria.
  4. Agrega la hoja de campañas de marketing como la fuente de datos secundaria.
  5. Establece relaciones de combinación entre los campos "Fecha de venta" y "Fecha de campaña".
  6. Crea una visualización que muestre las ventas reales y las campañas de marketing en un gráfico de líneas.

Solución:

1. Conecta a la base de datos SQL y carga la tabla de ventas.
2. Conecta al archivo Excel y carga la hoja de campañas de marketing.
3. Arrastra el campo "Fecha de venta" de la tabla de ventas a la vista.
4. Arrastra el campo "Fecha de campaña" de la hoja de campañas de marketing a la vista.
5. Haz clic en el icono de cadena junto al campo "Fecha de campaña".
6. En el cuadro de diálogo "Editar relaciones de combinación", selecciona "Fecha de venta" y "Fecha de campaña" como los campos de combinación.
7. Arrastra el campo "Ventas reales" de la tabla de ventas a la vista.
8. Arrastra el campo "Campaña de marketing" de la hoja de campañas de marketing a la vista.
9. Crea un gráfico de líneas para mostrar las ventas reales y las campañas de marketing.

Conclusión

La combinación de datos en Tableau es una técnica esencial para integrar datos de diferentes fuentes y obtener una visión más completa de tus análisis. Al dominar esta técnica, podrás crear visualizaciones más ricas y significativas que te ayudarán a tomar decisiones informadas basadas en datos integrados. En el próximo tema, exploraremos las uniones de datos, otra técnica importante para combinar datos en Tableau.
