Introducción

Tableau es una poderosa herramienta de visualización de datos utilizada para transformar datos en bruto en un formato comprensible. Su interfaz fácil de usar permite a los usuarios crear una amplia variedad de visualizaciones sin necesidad de conocimientos extensivos de programación. Esta sección proporcionará una descripción general de la interfaz de Tableau, cubriendo sus principales componentes y funcionalidades.

Componentes Clave de la Interfaz de Tableau

1. La Página de Inicio

Cuando abres Tableau por primera vez, te recibe la Página de Inicio. Esta página proporciona acceso rápido a libros de trabajo recientes, fuentes de datos y varios recursos de aprendizaje.

  • Panel de Conexión: Te permite conectarte a diferentes fuentes de datos.
  • Abrir: Proporciona acceso a libros de trabajo abiertos recientemente.
  • Libros de Trabajo de Muestra: Libros de trabajo preconstruidos que pueden ayudarte a entender las capacidades de Tableau.
  • Descubrir: Enlaces a recursos de Tableau como tutoriales, foros y blogs.

2. El Panel de Datos

El Panel de Datos es donde gestionas tus fuentes de datos y campos. Está ubicado en el lado izquierdo de la interfaz.

  • Pestaña de Fuente de Datos: Muestra las fuentes de datos conectadas y te permite gestionarlas.
  • Dimensiones y Medidas: Los campos de tu fuente de datos se categorizan en dimensiones (datos cualitativos) y medidas (datos cuantitativos).
  • Campos Calculados: Crea nuevos campos basados en datos existentes usando cálculos.

3. El Espacio de Trabajo

El Espacio de Trabajo es el área central donde construyes tus visualizaciones.

  • Hojas: Pestañas individuales donde creas diferentes visualizaciones.
  • Tableros: Combina múltiples hojas en una sola vista.
  • Historias: Secuencia de visualizaciones que trabajan juntas para transmitir información.

4. La Barra de Herramientas

La Barra de Herramientas proporciona acceso rápido a funciones y herramientas comunes.

  • Deshacer/Rehacer: Revertir o repetir acciones fácilmente.
  • Guardar: Guarda tu libro de trabajo.
  • Show Me: Sugiere diferentes tipos de visualizaciones basadas en los datos seleccionados.
  • Filtros: Aplica filtros a tus datos para enfocarte en información específica.

5. Las Estanterías

Las Estanterías son áreas donde arrastras y sueltas campos para construir tus visualizaciones.

  • Estanterías de Columnas y Filas: Definen la estructura de tu visualización.
  • Estantería de Filtros: Aplica filtros a tus datos.
  • Estantería de Marcas: Personaliza la apariencia de tus puntos de datos (por ejemplo, color, tamaño, forma).

6. La Barra de Estado

La Barra de Estado en la parte inferior de la interfaz proporciona información sobre el estado actual de tu libro de trabajo, como el número de marcas (puntos de datos) en tu visualización.

Ejemplo: Creación de un Gráfico de Barras Simple

Vamos a recorrer la creación de un gráfico de barras simple para ilustrar cómo usar la interfaz de Tableau.

  1. Conectar a Datos: Abre Tableau y conéctate a una fuente de datos de muestra (por ejemplo, "Sample - Superstore").
  2. Crear una Nueva Hoja: Haz clic en la pestaña "Hoja 1" para crear una nueva hoja.
  3. Arrastrar Campos a las Estanterías:
    • Arrastra el campo "Categoría" a la estantería de Columnas.
    • Arrastra el campo "Ventas" a la estantería de Filas.
  4. Personalizar la Visualización:
    • Haz clic en el panel "Show Me" y selecciona el icono de gráfico de barras.
    • Arrastra el campo "Región" a la estantería de Color para codificar las barras por región.
  5. Aplicar Filtros:
    • Arrastra el campo "Fecha de Pedido" a la estantería de Filtros.
    • Selecciona un rango específico de fechas para filtrar los datos.

1. Conéctate a la fuente de datos "Sample - Superstore".
2. Crea una nueva hoja.
3. Arrastra "Categoría" a la estantería de Columnas.

6. Arrastra "Región" a la estantería de Color.
7. Arrastra "Fecha de Pedido" a la estantería de Filtros y selecciona el rango de fechas.

Conclusión

Entender la interfaz de Tableau es crucial para crear visualizaciones perspicaces de manera eficiente. Familiarizándote con la Página de Inicio, el Panel de Datos, el Espacio de Trabajo, la Barra de Herramientas, las Estanterías y la Barra de Estado, podrás navegar Tableau con confianza. Practica creando visualizaciones simples, como el ejemplo del gráfico de barras, para reforzar tu comprensión de estos componentes. A medida que avances en el curso, construirás sobre este conocimiento fundamental para crear visualizaciones más complejas e interactivas.
