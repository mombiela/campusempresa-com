Usando cálculos de tabla
Siguiente ►

Introducción

Los cálculos de tabla en Tableau son una poderosa herramienta que permite realizar análisis complejos y detallados directamente en tus visualizaciones. Estos cálculos se aplican a los datos que ya están en la vista y pueden ser utilizados para crear métricas avanzadas, como acumulados, diferencias porcentuales, y más.

Conceptos Clave

  1. Cálculos de tabla básicos: Operaciones matemáticas simples aplicadas a los datos en la vista.
  2. Dirección de cálculo: La forma en que Tableau recorre los datos para aplicar el cálculo (por fila, por columna, etc.).
  3. Ámbito de cálculo: El conjunto de datos sobre el cual se aplica el cálculo (tabla completa, panel, celda, etc.).

Tipos de Cálculos de Tabla

  1. Diferencia: Calcula la diferencia entre valores consecutivos.
  2. Porcentaje de diferencia: Calcula el cambio porcentual entre valores consecutivos.
  3. Total acumulado: Suma acumulativa de los valores.
  4. Promedio móvil: Promedio de un conjunto de valores en una ventana móvil.
  5. Rango: Diferencia entre el valor máximo y mínimo en un conjunto de datos.

Ejemplo Práctico

Paso 1: Crear una Visualización Básica

  1. Conectar a una fuente de datos: Para este ejemplo, utilizaremos el conjunto de datos "Sample - Superstore" que viene con Tableau.
  2. Arrastrar campos a la vista:
    • Arrastra Order Date a la columna.
    • Arrastra Sales a la fila.
+----------------+----------------+
| Order Date     | Sales          |
+----------------+----------------+
| 2018-01-01     | 1000           |
| 2018-02-01     | 1500           |
| 2018-03-01     | 2000           |
+----------------+----------------+

Paso 2: Aplicar un Cálculo de Tabla

  1. Agregar un cálculo de tabla:
    • Haz clic derecho en el campo SUM(Sales) en la vista.
    • Selecciona Quick Table Calculation (Cálculo rápido de tabla).
    • Elige Running Total (Total acumulado).
+----------------+----------------+----------------+
| Order Date     | Sales          | Running Total  |
+----------------+----------------+----------------+
| 2018-01-01     | 1000           | 1000           |
| 2018-02-01     | 1500           | 2500           |
| 2018-03-01     | 2000           | 4500           |
+----------------+----------------+----------------+

Paso 3: Personalizar el Cálculo de Tabla

  1. Editar el cálculo de tabla:
    • Haz clic derecho en SUM(Sales) y selecciona Edit Table Calculation (Editar cálculo de tabla).
    • En la ventana de edición, puedes cambiar la dirección del cálculo (por ejemplo, de Table (Across) a Table (Down)).

Ejercicio Práctico

Ejercicio 1: Crear un Promedio Móvil

  1. Conectar a la fuente de datos "Sample - Superstore".
  2. Crear una visualización básica:
    • Arrastra Order Date a la columna.
    • Arrastra Sales a la fila.
  3. Aplicar un cálculo de tabla:
    • Haz clic derecho en SUM(Sales).
    • Selecciona Quick Table Calculation.
    • Elige Moving Average (Promedio móvil).
  4. Personalizar el cálculo:
    • Edita el cálculo para que el promedio móvil sea de 3 meses.

Solución:

+----------------+----------------+----------------+
| Order Date     | Sales          | Moving Average |
+----------------+----------------+----------------+
| 2018-01-01     | 1000           | -              |
| 2018-02-01     | 1500           | -              |
| 2018-03-01     | 2000           | 1500           |
| 2018-04-01     | 2500           | 2000           |
| 2018-05-01     | 3000           | 2500           |
+----------------+----------------+----------------+

Errores Comunes y Consejos

  1. Dirección de cálculo incorrecta: Asegúrate de que la dirección del cálculo (por fila, por columna) sea la correcta para tu análisis.
  2. Ámbito de cálculo incorrecto: Verifica que el ámbito del cálculo (tabla completa, panel, celda) esté configurado correctamente.
  3. Datos faltantes: Los cálculos de tabla pueden verse afectados por datos faltantes. Asegúrate de que tus datos estén completos y limpios.

Conclusión

Los cálculos de tabla en Tableau son una herramienta esencial para realizar análisis avanzados y detallados. Al comprender cómo aplicar y personalizar estos cálculos, puedes obtener insights más profundos y precisos de tus datos. En el próximo módulo, exploraremos técnicas avanzadas de visualización que te permitirán llevar tus análisis al siguiente nivel.
