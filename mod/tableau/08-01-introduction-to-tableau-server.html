¿Qué es Tableau Server?

Tableau Server es una plataforma de análisis empresarial que permite a las organizaciones compartir, distribuir y colaborar en visualizaciones de datos y dashboards creados en Tableau Desktop. Proporciona una infraestructura segura y escalable para el análisis de datos, permitiendo a los usuarios acceder a sus datos y visualizaciones desde cualquier lugar y en cualquier momento.

Características clave de Tableau Server:

  • Colaboración y Compartición: Permite a los usuarios compartir dashboards y visualizaciones con otros miembros de la organización.
  • Seguridad y Gobernanza: Ofrece controles de seguridad robustos para proteger los datos y gestionar el acceso de los usuarios.
  • Escalabilidad: Puede manejar grandes volúmenes de datos y usuarios, adaptándose a las necesidades de la organización.
  • Acceso Móvil: Los usuarios pueden acceder a sus visualizaciones desde dispositivos móviles.
  • Automatización: Permite la programación de actualizaciones de datos y la automatización de tareas repetitivas.

Instalación de Tableau Server

Requisitos del Sistema

Antes de instalar Tableau Server, asegúrate de que tu sistema cumple con los siguientes requisitos mínimos:

Requisito Detalle
Sistema Operativo Windows Server 2016 o posterior, Linux (Ubuntu, RHEL, CentOS)
CPU Mínimo 8 núcleos
Memoria RAM Mínimo 32 GB
Espacio en Disco Mínimo 50 GB de espacio libre
Navegador Chrome, Firefox, Safari, Edge

Pasos de Instalación

  1. Descargar el Instalador:

    • Visita el sitio web de Tableau y descarga el instalador de Tableau Server para tu sistema operativo.

  2. Ejecutar el Instalador:

    • Ejecuta el archivo descargado y sigue las instrucciones del asistente de instalación.

  3. Configurar Tableau Server:

    • Durante la instalación, se te pedirá que configures varios parámetros, como el puerto de acceso, la ubicación de los archivos de datos, y las credenciales de administrador.

  4. Activar Tableau Server:

    • Una vez instalado, deberás activar Tableau Server utilizando una clave de licencia proporcionada por Tableau.

  5. Iniciar Tableau Server:

    • Después de la activación, inicia Tableau Server y accede a la interfaz de administración a través de un navegador web.

Descripción General de la Interfaz de Tableau Server

Componentes Principales

  1. Inicio:

    • La página de inicio muestra una visión general de los proyectos, vistas y fuentes de datos disponibles.

  2. Explorar:

    • Permite navegar por los proyectos, libros de trabajo y vistas disponibles en el servidor.

  3. Favoritos:

    • Los usuarios pueden marcar sus visualizaciones y dashboards favoritos para un acceso rápido.

  4. Proyectos:

    • Los proyectos son contenedores que organizan libros de trabajo y fuentes de datos. Pueden ser públicos o privados.

  5. Libros de Trabajo:

    • Los libros de trabajo contienen una o más visualizaciones y dashboards creados en Tableau Desktop.

  6. Fuentes de Datos:

    • Las fuentes de datos son conexiones a bases de datos y otros orígenes de datos que se utilizan para crear visualizaciones.

Navegación Básica

  • Buscar:

    • Utiliza la barra de búsqueda para encontrar rápidamente proyectos, libros de trabajo, vistas y fuentes de datos.

  • Filtros:

    • Aplica filtros para refinar los resultados de búsqueda y encontrar exactamente lo que necesitas.

  • Acciones:

    • Realiza acciones como compartir, descargar, o suscribirse a actualizaciones de visualizaciones y dashboards.

Conclusión

En esta lección, hemos cubierto los conceptos básicos de Tableau Server, incluyendo sus características clave, los requisitos del sistema, los pasos de instalación y una descripción general de la interfaz. Tableau Server es una herramienta poderosa para compartir y colaborar en análisis de datos dentro de una organización, proporcionando una plataforma segura y escalable para el acceso a datos y visualizaciones.
