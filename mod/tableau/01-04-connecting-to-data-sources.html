Conectando a fuentes de datos
Siguiente ►

En esta sección, aprenderás cómo conectar Tableau a diversas fuentes de datos. Este es un paso crucial, ya que la calidad y estructura de los datos que importes influirán directamente en la efectividad de tus visualizaciones.

Objetivos de Aprendizaje

  • Comprender los tipos de fuentes de datos que Tableau puede conectar.
  • Aprender a conectar Tableau a una fuente de datos.
  • Conocer las opciones de conexión y configuración disponibles.

Tipos de Fuentes de Datos

Tableau puede conectarse a una amplia variedad de fuentes de datos, incluyendo:

  1. Archivos:

    • Excel
    • Texto (CSV)
    • JSON
    • PDF
    • Access

  2. Bases de Datos:

    • MySQL
    • PostgreSQL
    • SQL Server
    • Oracle
    • Amazon Redshift

  3. Servicios en la Nube:

    • Google Sheets
    • Google BigQuery
    • Salesforce
    • Amazon Athena

  4. Conectores Web:

    • APIs
    • Conectores personalizados

Pasos para Conectar a una Fuente de Datos

  1. Abrir Tableau y Seleccionar una Fuente de Datos

  1. Abre Tableau.
  2. En la pantalla de inicio, verás una sección titulada "Conectar". Aquí puedes seleccionar el tipo de fuente de datos a la que deseas conectarte.

  1. Conectar a un Archivo

Ejemplo: Conectar a un archivo Excel

  1. Haz clic en "Microsoft Excel" en la sección "Conectar".
  2. Navega hasta el archivo Excel en tu computadora y selecciónalo.
  3. Tableau abrirá una nueva ventana mostrando las hojas disponibles en el archivo Excel.

  1. Conectar a una Base de Datos

Ejemplo: Conectar a una base de datos MySQL

  1. Haz clic en "MySQL" en la sección "Conectar".
  2. Ingresa la información de conexión requerida:
    • Servidor: La dirección del servidor MySQL.
    • Puerto: El puerto de conexión (por defecto es 3306).
    • Nombre de usuario: Tu nombre de usuario de MySQL.
    • Contraseña: Tu contraseña de MySQL.
  3. Haz clic en "Conectar".
  4. Selecciona la base de datos a la que deseas conectarte.

  1. Configurar la Conexión

Una vez que te hayas conectado a una fuente de datos, puedes configurar la conexión:

  • Seleccionar Tablas: Arrastra las tablas que deseas usar al área de trabajo.
  • Unir Tablas: Si necesitas combinar datos de múltiples tablas, puedes crear uniones (joins) entre ellas.
  • Filtrar Datos: Puedes aplicar filtros para limitar los datos que se importan a Tableau.

Ejemplo Práctico: Conectar a un Archivo CSV

  1. Haz clic en "Texto" en la sección "Conectar".
  2. Selecciona el archivo CSV en tu computadora.
  3. Tableau mostrará una vista previa de los datos. Aquí puedes:
    • Renombrar Columnas: Haz clic en el nombre de una columna para cambiar su nombre.
    • Cambiar Tipos de Datos: Haz clic en el icono de tipo de datos para cambiar el tipo (por ejemplo, de texto a número).

Ejercicio Práctico

Objetivo: Conectar Tableau a un archivo Excel y crear una visualización básica.

  1. Conectar a Excel:

    • Descarga el archivo de ejemplo ventas.xlsx.
    • Abre Tableau y selecciona "Microsoft Excel".
    • Navega hasta el archivo descargado y selecciónalo.

  2. Configurar la Conexión:

    • Arrastra la hoja "Ventas" al área de trabajo.
    • Asegúrate de que los datos se muestren correctamente.

  3. Crear una Visualización:

    • Arrastra la columna "Producto" al estante de filas.
    • Arrastra la columna "Ventas" al estante de columnas.
    • Selecciona un gráfico de barras desde la tarjeta de marcas.

Solución del Ejercicio

1. Conectar a Excel:
   - Abre Tableau y selecciona "Microsoft Excel".
   - Selecciona el archivo "ventas.xlsx".

2. Configurar la Conexión:
   - Arrastra la hoja "Ventas" al área de trabajo.

3. Crear una Visualización:
   - Arrastra "Producto" al estante de filas.
   - Arrastra "Ventas" al estante de columnas.
   - Selecciona el gráfico de barras.

Conclusión

Conectar Tableau a diversas fuentes de datos es un paso fundamental para cualquier análisis. En esta sección, aprendiste a conectar Tableau a archivos y bases de datos, así como a configurar estas conexiones. En el próximo módulo, exploraremos cómo crear visualizaciones básicas utilizando los datos importados.
