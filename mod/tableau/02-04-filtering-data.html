Filtrando datos
Siguiente ►

En este tema, aprenderemos cómo filtrar datos en Tableau para enfocarnos en la información más relevante y útil. Los filtros son una herramienta esencial para cualquier análisis de datos, ya que permiten reducir el volumen de datos y centrarse en los aspectos más importantes.

Conceptos clave

  1. Filtros de Dimensión: Permiten filtrar datos basados en dimensiones específicas, como categorías o nombres.
  2. Filtros de Medida: Permiten filtrar datos basados en medidas, como valores numéricos.
  3. Filtros de Fecha: Permiten filtrar datos basados en fechas o rangos de fechas.
  4. Filtros de Contexto: Permiten establecer un filtro como contexto para otros filtros.
  5. Filtros de Tabla: Permiten filtrar datos a nivel de tabla, afectando solo la visualización actual.

Tipos de Filtros

Filtros de Dimensión

Los filtros de dimensión se utilizan para incluir o excluir categorías específicas de datos. Por ejemplo, si tienes una lista de productos, puedes filtrar para mostrar solo ciertos productos.

Ejemplo:

Categoría de Producto: [Electrónica, Muebles, Ropa]

Filtros de Medida

Los filtros de medida se utilizan para incluir o excluir datos basados en valores numéricos. Por ejemplo, puedes filtrar para mostrar solo las ventas que superan un cierto umbral.

Ejemplo:

Ventas > 1000

Filtros de Fecha

Los filtros de fecha permiten filtrar datos basados en fechas específicas o rangos de fechas. Esto es útil para análisis temporales.

Ejemplo:

Fecha de Venta: [01/01/2022 - 31/12/2022]

Filtros de Contexto

Los filtros de contexto se utilizan para establecer un filtro como contexto para otros filtros. Esto puede mejorar el rendimiento y la lógica de filtrado.

Ejemplo:

Categoría de Producto: [Electrónica]

Filtros de Tabla

Los filtros de tabla se aplican a nivel de tabla y afectan solo la visualización actual. Son útiles para análisis específicos dentro de una visualización.

Ejemplo:

Mostrar solo las 10 principales ventas

Cómo aplicar filtros en Tableau

Paso 1: Seleccionar el campo a filtrar

  1. En el panel de datos, selecciona la dimensión o medida que deseas filtrar.
  2. Arrastra el campo seleccionado al estante de filtros.

Paso 2: Configurar el filtro

  1. Una vez que el campo esté en el estante de filtros, aparecerá un cuadro de diálogo.
  2. Configura las opciones del filtro según tus necesidades (por ejemplo, seleccionar categorías específicas, establecer rangos de fechas, etc.).

Paso 3: Aplicar el filtro

  1. Haz clic en "Aceptar" para aplicar el filtro.
  2. La visualización se actualizará para reflejar los datos filtrados.

Ejemplo práctico

Supongamos que tenemos un conjunto de datos de ventas y queremos filtrar para mostrar solo las ventas de productos electrónicos en el año 2022.

Paso 1: Seleccionar el campo a filtrar

  1. En el panel de datos, selecciona la dimensión "Categoría de Producto".
  2. Arrastra "Categoría de Producto" al estante de filtros.

Paso 2: Configurar el filtro de categoría

  1. En el cuadro de diálogo de filtro, selecciona "Electrónica".
  2. Haz clic en "Aceptar".

Paso 3: Seleccionar el campo de fecha

  1. En el panel de datos, selecciona la dimensión "Fecha de Venta".
  2. Arrastra "Fecha de Venta" al estante de filtros.

Paso 4: Configurar el filtro de fecha

  1. En el cuadro de diálogo de filtro, selecciona el rango de fechas "01/01/2022 - 31/12/2022".
  2. Haz clic en "Aceptar".

Resultado

La visualización ahora mostrará solo las ventas de productos electrónicos en el año 2022.

Ejercicio práctico

Ejercicio:

  1. Conecta a una fuente de datos que contenga información de ventas.
  2. Crea una visualización de barras que muestre las ventas por categoría de producto.
  3. Aplica un filtro para mostrar solo las ventas de la categoría "Muebles".
  4. Aplica un filtro de fecha para mostrar solo las ventas del año 2021.

Solución:

  1. Conecta a la fuente de datos.
  2. Arrastra "Categoría de Producto" al estante de filas.
  3. Arrastra "Ventas" al estante de columnas.
  4. Arrastra "Categoría de Producto" al estante de filtros y selecciona "Muebles".
  5. Arrastra "Fecha de Venta" al estante de filtros y selecciona el rango de fechas "01/01/2021 - 31/12/2021".

Conclusión

Filtrar datos en Tableau es una habilidad fundamental que te permite enfocarte en la información más relevante para tu análisis. Al dominar los diferentes tipos de filtros y cómo aplicarlos, podrás crear visualizaciones más precisas y útiles. En el próximo tema, aprenderemos sobre cómo ordenar y agrupar datos para mejorar aún más nuestras visualizaciones.
