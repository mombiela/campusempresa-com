Instalando Tableau
Siguiente ►

En esta sección, aprenderás cómo instalar Tableau en tu computadora. Tableau ofrece varias versiones de su software, incluyendo Tableau Desktop, Tableau Public, Tableau Online y Tableau Server. Para este curso, nos enfocaremos en Tableau Desktop, que es la versión más completa y adecuada para la mayoría de los usuarios.

Requisitos del Sistema

Antes de comenzar con la instalación, asegúrate de que tu sistema cumpla con los siguientes requisitos mínimos:

Windows

  • Sistema Operativo: Windows 7 o superior (64 bits)
  • Procesador: Intel Pentium 4 o AMD Opteron con soporte SSE2
  • Memoria RAM: 4 GB (se recomiendan 8 GB o más)
  • Espacio en Disco: 1.5 GB de espacio libre
  • Resolución de Pantalla: 1366 x 768 (se recomienda 1920 x 1080 o superior)

Mac

  • Sistema Operativo: macOS 10.14 o superior
  • Procesador: Intel Core i3 o superior
  • Memoria RAM: 4 GB (se recomiendan 8 GB o más)
  • Espacio en Disco: 1.5 GB de espacio libre
  • Resolución de Pantalla: 1366 x 768 (se recomienda 1920 x 1080 o superior)

Pasos para la Instalación

Paso 1: Descargar Tableau Desktop

  1. Visita el sitio web de Tableau:

    • Ve a Tableau Desktop Download.

  2. Selecciona tu sistema operativo:

    • Elige entre Windows o Mac según tu sistema operativo.

  3. Completa el formulario de registro:

    • Ingresa tu información personal y profesional para obtener una versión de prueba gratuita de 14 días.

  4. Descarga el instalador:

    • Haz clic en el botón de descarga para obtener el archivo de instalación.

Paso 2: Instalar Tableau Desktop

En Windows

  1. Ejecuta el instalador:

    • Haz doble clic en el archivo descargado (TableauDesktop-<version>.exe).

  2. Acepta los términos y condiciones:

    • Lee y acepta el acuerdo de licencia.

  3. Selecciona la ubicación de instalación:

    • Puedes dejar la ubicación predeterminada o elegir una carpeta diferente.

  4. Completa la instalación:

    • Haz clic en "Instalar" y espera a que el proceso termine.

  5. Inicia Tableau Desktop:

    • Una vez completada la instalación, abre Tableau Desktop desde el menú de inicio o el acceso directo en el escritorio.

En Mac

  1. Ejecuta el instalador:

    • Haz doble clic en el archivo descargado (TableauDesktop-<version>.dmg).

  2. Arrastra Tableau a la carpeta de Aplicaciones:

    • En la ventana que se abre, arrastra el icono de Tableau a la carpeta de Aplicaciones.

  3. Inicia Tableau Desktop:

    • Abre la carpeta de Aplicaciones y haz doble clic en Tableau Desktop para iniciarlo.

Paso 3: Activar Tableau Desktop

  1. Inicia Tableau Desktop:

    • Al abrir Tableau por primera vez, se te pedirá que actives el software.

  2. Ingresa tu clave de producto:

    • Si tienes una clave de producto, ingrésala en el campo correspondiente.
    • Si estás utilizando la versión de prueba, selecciona "Iniciar prueba gratuita".

  3. Completa la activación:

    • Sigue las instrucciones en pantalla para completar el proceso de activación.

Solución de Problemas Comunes

Problema: Error durante la instalación

  • Solución: Asegúrate de que tu sistema cumple con los requisitos mínimos. Verifica que tienes suficiente espacio en disco y que tu sistema operativo está actualizado.

Problema: Tableau no se inicia después de la instalación

  • Solución: Reinicia tu computadora y vuelve a intentar abrir Tableau. Si el problema persiste, desinstala y reinstala el software.

Problema: No puedo activar Tableau

  • Solución: Verifica que estás ingresando la clave de producto correcta. Si estás utilizando la versión de prueba, asegúrate de que no has agotado el período de prueba en el pasado.

Conclusión

Ahora que has instalado Tableau Desktop, estás listo para comenzar a explorar sus poderosas capacidades de visualización de datos. En la siguiente sección, aprenderás sobre la interfaz de Tableau y cómo navegar por ella para aprovechar al máximo sus funcionalidades.
