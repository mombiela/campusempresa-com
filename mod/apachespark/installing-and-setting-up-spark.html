Instalación y Configuración de Spark
Arquitectura de Spark ►

Apache Spark es un poderoso sistema de computación distribuida de código abierto que proporciona una interfaz para programar clústeres completos con paralelismo de datos implícito y tolerancia a fallos. Esta sección te guiará a través del proceso de instalación y configuración de Apache Spark en tu máquina local.

Requisitos Previos

Antes de comenzar con la instalación, asegúrate de tener los siguientes requisitos previos:

  • Java Development Kit (JDK) 8 o posterior: Spark se ejecuta en Java, por lo que necesitas tener JDK instalado.
  • Scala (opcional): Si planeas escribir aplicaciones Spark en Scala, necesitas tener Scala instalado.
  • Python (opcional): Si planeas escribir aplicaciones Spark en Python, necesitas tener Python instalado.
  • Apache Hadoop (opcional): Si planeas usar Spark con Hadoop, necesitas tener Hadoop instalado.

Guía de Instalación Paso a Paso

Paso 1: Instalar Java Development Kit (JDK)

  1. Descargar JDK: Ve a la página de descarga de Oracle JDK y descarga la versión apropiada para tu sistema operativo.
  2. Instalar JDK: Sigue las instrucciones de instalación proporcionadas en la página de descarga.

Paso 2: Descargar Apache Spark

  1. Visita la Página de Descarga de Spark: Ve a la página de descarga de Apache Spark.
  2. Seleccionar Versión: Elige la versión de Spark que deseas descargar. Se recomienda descargar la última versión estable.
  3. Seleccionar Tipo de Paquete: Elige un paquete preconstruido para Hadoop. Si no estás usando Hadoop, selecciona "Pre-built for Apache Hadoop 2.7 and later".
  4. Descargar: Haz clic en el enlace de descarga para descargar el paquete de Spark.

Paso 3: Instalar Apache Spark

  1. Extraer el Paquete: Extrae el paquete de Spark descargado en un directorio de tu elección. Por ejemplo, en sistemas basados en Unix, puedes usar el siguiente comando: 
    tar -xvf spark-<version>-bin-hadoop2.7.tgz
  2. Configurar Variables de Entorno: Añade Spark a la variable de entorno PATH de tu sistema. Por ejemplo, en sistemas basados en Unix, añade las siguientes líneas a tu archivo .bashrc o .zshrc: 
    export SPARK_HOME=/path/to/spark-<version>-bin-hadoop2.7
export PATH=$SPARK_HOME/bin:$PATH

Paso 4: Verificar la Instalación

  1. Abrir una Terminal: Abre una terminal o símbolo del sistema.
  2. Ejecutar Spark Shell: Escribe el siguiente comando para iniciar el shell de Spark: 
    spark-shell
  3. Verificar Salida: Si Spark está instalado correctamente, deberías ver el prompt del shell de Spark: 
    Spark context Web UI available at http://<your-ip>:4040
Spark context available as 'sc' (master = local[*], app id = local-<timestamp>).
Spark session available as 'spark'.

Paso 5: Instalar Componentes Opcionales

  • Scala: Si planeas escribir aplicaciones Spark en Scala, descarga e instala Scala desde la página de descarga de Scala.
  • Python: Si planeas escribir aplicaciones Spark en Python, descarga e instala Python desde la página de descarga de Python.

Ejecutar una Aplicación Simple de Spark

Para asegurarte de que todo está configurado correctamente, ejecutemos una aplicación simple de Spark.

  1. Crear un Script en Python: Crea un archivo llamado simple_app.py con el siguiente contenido:

    from pyspark.sql import SparkSession

# Inicializar sesión de Spark
spark = SparkSession.builder.appName("SimpleApp").getOrCreate()

# Crear un DataFrame simple
data = [("Alice", 34), ("Bob", 45), ("Cathy", 29)]
df = spark.createDataFrame(data, ["Name", "Age"])

# Mostrar el DataFrame
df.show()

# Detener la sesión de Spark
spark.stop()

  2. Ejecutar el Script: Ejecuta el script usando el siguiente comando:

    spark-submit simple_app.py

  3. Verificar Salida: Deberías ver la siguiente salida:

    +-----+---+
| Name|Age|
+-----+---+
|Alice| 34|
|  Bob| 45|
|Cathy| 29|
+-----+---+

Conclusión

En esta sección, cubrimos los pasos para instalar y configurar Apache Spark en tu máquina local. También ejecutamos una aplicación simple de Spark para verificar la instalación. Con Spark instalado, ahora estás listo para profundizar en sus poderosas capacidades de procesamiento de datos.
