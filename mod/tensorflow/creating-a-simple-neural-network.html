Introducción

En esta sección, cubriremos los conceptos básicos para crear una red neuronal simple usando TensorFlow. Esto incluirá configurar el entorno, entender los conceptos clave y construir un modelo básico de red neuronal.

Configuración del Entorno

Antes de empezar a codificar, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar TensorFlow: Asegúrate de tener Python instalado, luego instala TensorFlow usando pip. 
    pip install tensorflow
  • Verificar la Instalación: Comprueba si TensorFlow está instalado correctamente. 
    import tensorflow as tf
print(tf.__version__)

Conceptos Clave

Antes de sumergirnos en el código, entendamos algunos conceptos clave:

  • Red Neuronal: Una serie de algoritmos que intentan reconocer relaciones subyacentes en un conjunto de datos a través de un proceso que imita la forma en que opera el cerebro humano.
  • Capas: Los bloques de construcción de una red neuronal. Cada capa consta de nodos (neuronas) que realizan cálculos.
  • Función de Activación: Una función aplicada a la salida de cada nodo para introducir no linealidad en el modelo.
  • Función de Pérdida: Un método para medir qué tan bien está funcionando la red neuronal.
  • Optimizador: Un algoritmo para ajustar los pesos de la red para minimizar la función de pérdida.

Construyendo una Red Neuronal Simple

Construyamos una red neuronal simple para clasificar dígitos escritos a mano del conjunto de datos MNIST.

Paso 1: Importar Librerías

Primero, importa las librerías necesarias.

import tensorflow as tf
from tensorflow.keras import layers, models
import numpy as np

Paso 2: Cargar y Preprocesar Datos

Carga el conjunto de datos MNIST y preprocésalo.

# Cargar el conjunto de datos
mnist = tf.keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

# Normalizar los datos
x_train, x_test = x_train / 255.0, x_test / 255.0

Paso 3: Definir el Modelo

Crea un modelo de red neuronal simple.

model = models.Sequential([
    layers.Flatten(input_shape=(28, 28)),  # Aplanar la entrada
    layers.Dense(128, activation='relu'),  # Capa oculta con activación ReLU
    layers.Dense(10, activation='softmax') # Capa de salida con activación softmax
])

Paso 4: Compilar el Modelo

Compila el modelo con un optimizador, función de pérdida y métricas.

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

Paso 5: Entrenar el Modelo

Entrena el modelo usando los datos de entrenamiento.

model.fit(x_train, y_train, epochs=5)

Paso 6: Evaluar el Modelo

Evalúa el modelo usando los datos de prueba.

test_loss, test_acc = model.evaluate(x_test, y_test)
print(f'Precisión en prueba: {test_acc}')

Conclusión

En esta sección, cubrimos los conceptos básicos para crear una red neuronal simple usando TensorFlow. Comenzamos configurando el entorno, entendiendo los conceptos clave, y luego construimos, compilamos, entrenamos y evaluamos un modelo básico de red neuronal. Este conocimiento fundamental será crucial a medida que avancemos a temas más avanzados en redes neuronales y TensorFlow.
