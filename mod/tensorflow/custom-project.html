Introducción

En esta sección, te guiaremos a través de la creación de un proyecto personalizado utilizando TensorFlow. Este proyecto consolidará tu comprensión de los conceptos de TensorFlow y te dará experiencia práctica en la construcción y despliegue de un modelo de aprendizaje automático.

Descripción del Proyecto

  • Objetivo: Construir una red neuronal para clasificar imágenes del conjunto de datos CIFAR-10.
  • Conjunto de datos: CIFAR-10, que consta de 60,000 imágenes en color de 32x32 en 10 clases.
  • Herramientas: TensorFlow, Keras y otras bibliotecas esenciales.

Guía Paso a Paso

Paso 1: Configuración del Entorno

Antes de comenzar, asegúrate de tener instaladas TensorFlow y otras bibliotecas necesarias.

pip install tensorflow numpy matplotlib

Paso 2: Importación de Bibliotecas

Comienza importando las bibliotecas requeridas.

import tensorflow as tf
from tensorflow.keras import datasets, layers, models
import matplotlib.pyplot as plt

Paso 3: Carga y Preprocesamiento de Datos

Carga el conjunto de datos CIFAR-10 y prepáralo.

# Cargar conjunto de datos
(train_images, train_labels), (test_images, test_labels) = datasets.cifar10.load_data()

# Normalizar valores de píxeles para que estén entre 0 y 1
train_images, test_images = train_images / 255.0, test_images / 255.0

Paso 4: Visualización de Datos

Visualiza algunas imágenes del conjunto de datos para entender con qué estás trabajando.

class_names = ['airplane', 'automobile', 'bird', 'cat', 'deer', 'dog', 'frog', 'horse', 'ship', 'truck']

plt.figure(figsize=(10,10))
for i in range(25):
    plt.subplot(5,5,i+1)
    plt.xticks([])
    plt.yticks([])
    plt.grid(False)
    plt.imshow(train_images[i])
    plt.xlabel(class_names[train_labels[i][0]])
plt.show()

Paso 5: Construcción del Modelo

Crea un modelo de Red Neuronal Convolucional (CNN).

model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10))

Paso 6: Compilación del Modelo

Compila el modelo con la función de pérdida, el optimizador y las métricas apropiadas.

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

Paso 7: Entrenamiento del Modelo

Entrena el modelo utilizando los datos de entrenamiento.

history = model.fit(train_images, train_labels, epochs=10, 
                    validation_data=(test_images, test_labels))

Paso 8: Evaluación del Modelo

Evalúa el rendimiento del modelo en el conjunto de datos de prueba.

test_loss, test_acc = model.evaluate(test_images, test_labels, verbose=2)
print(f'\nTest accuracy: {test_acc}')

Paso 9: Realización de Predicciones

Utiliza el modelo entrenado para hacer predicciones sobre nuevos datos.

probability_model = tf.keras.Sequential([model, 
                                         tf.keras.layers.Softmax()])

predictions = probability_model.predict(test_images)
print(predictions[0])
print(f'Predicted label: {class_names[np.argmax(predictions[0])]}')

Paso 10: Visualización de Predicciones

Visualiza las predicciones para entender el rendimiento del modelo.

def plot_image(i, predictions_array, true_label, img):
    true_label, img = true_label[i], img[i]
    plt.grid(False)
    plt.xticks([])
    plt.yticks([])
    
    plt.imshow(img, cmap=plt.cm.binary)
    
    predicted_label = np.argmax(predictions_array)
    if predicted_label == true_label:
        color = 'blue'
    else:
        color = 'red'
    
    plt.xlabel(f"{class_names[predicted_label]} {100*np.max(predictions_array):2.0f}% ({class_names[true_label]})", color=color)

def plot_value_array(i, predictions_array, true_label):
    true_label = true_label[i]
    plt.grid(False)
    plt.xticks(range(10))
    plt.yticks([])
    thisplot = plt.bar(range(10), predictions_array, color="#777777")
    plt.ylim([0, 1])
    predicted_label = np.argmax(predictions_array)
    
    thisplot[predicted_label].set_color('red')
    thisplot[true_label].set_color('blue')

i = 0
plt.figure(figsize=(6,3))
plt.subplot(1,2,1)
plot_image(i, predictions[i], test_labels, test_images)
plt.subplot(1,2,2)
plot_value_array(i, predictions[i], test_labels)
plt.show()
