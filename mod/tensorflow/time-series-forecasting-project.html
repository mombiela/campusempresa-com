Introducción

El pronóstico de series temporales es un aspecto crucial de muchas aplicaciones del mundo real, como la predicción del mercado de valores, la previsión meteorológica y la planificación de la demanda. En esta sección, exploraremos cómo usar TensorFlow para construir y entrenar modelos para el pronóstico de series temporales. Este tema te guiará desde los conceptos básicos hasta técnicas avanzadas, asegurando una comprensión integral del tema.

Conceptos Clave

  • Datos de Series Temporales: Una secuencia de puntos de datos recopilados o registrados en intervalos de tiempo específicos.
  • Pronóstico: El proceso de hacer predicciones sobre puntos de datos futuros basados en datos históricos.
  • TensorFlow: Un marco de aprendizaje automático de código abierto desarrollado por Google.

Configuración del Entorno

Antes de comenzar, asegúrate de tener TensorFlow instalado. Puedes instalarlo usando pip:

pip install tensorflow

Preparación de los Datos

Cargando el Conjunto de Datos

Para este proyecto, utilizaremos un conjunto de datos de muestra que contiene lecturas diarias de temperatura.

import pandas as pd

# Cargar el conjunto de datos
url = 'https://example.com/daily-temperature.csv'
data = pd.read_csv(url, parse_dates=['date'], index_col='date')

# Mostrar las primeras filas
print(data.head())

Preprocesamiento de Datos

  • Normalización: Escalar los datos a un rango adecuado para redes neuronales.
  • División de Entrenamiento y Prueba: Dividir los datos en conjuntos de entrenamiento y prueba.
from sklearn.preprocessing import MinMaxScaler

# Normalizar los datos
scaler = MinMaxScaler()
data['temperature'] = scaler.fit_transform(data[['temperature']])

# Dividir los datos
train_size = int(len(data) * 0.8)
train_data = data[:train_size]
test_data = data[train_size:]

Construcción del Modelo

Creación del Conjunto de Datos de Series Temporales

Necesitamos crear secuencias de datos para que el modelo aprenda.

import numpy as np

def create_dataset(data, time_step=1):
    X, y = [], []
    for i in range(len(data) - time_step - 1):
        X.append(data[i:(i + time_step), 0])
        y.append(data[i + time_step, 0])
    return np.array(X), np.array(y)

time_step = 10
X_train, y_train = create_dataset(train_data.values, time_step)
X_test, y_test = create_dataset(test_data.values, time_step)

# Redimensionar la entrada a [muestras, pasos de tiempo, características]
X_train = X_train.reshape(X_train.shape[0], time_step, 1)
X_test = X_test.reshape(X_test.shape[0], time_step, 1)

Definición del Modelo

Usaremos una red LSTM (Long Short-Term Memory) simple para esta tarea.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense

model = Sequential()
model.add(LSTM(50, return_sequences=True, input_shape=(time_step, 1)))
model.add(LSTM(50, return_sequences=False))
model.add(Dense(1))
model.compile(optimizer='adam', loss='mean_squared_error')

model.summary()

Entrenamiento del Modelo

Entrena el modelo usando los datos de entrenamiento.

history = model.fit(X_train, y_train, validation_data=(X_test, y_test), epochs=50, batch_size=32, verbose=1)

Evaluación del Modelo

Visualización de la Pérdida

Grafica la pérdida de entrenamiento y validación para verificar si hay sobreajuste.

import matplotlib.pyplot as plt

plt.plot(history.history['loss'], label='Pérdida de Entrenamiento')
plt.plot(history.history['val_loss'], label='Pérdida de Validación')
plt.legend()
plt.show()

Realización de Predicciones

Usa el modelo entrenado para hacer predicciones en los datos de prueba.

train_predict = model.predict(X_train)
test_predict = model.predict(X_test)

# Transformar inversamente para obtener valores reales
train_predict = scaler.inverse_transform(train_predict)
test_predict = scaler.inverse_transform(test_predict)

Graficando los Resultados

Compara los valores predichos con los valores reales.

# Desplazar las predicciones de entrenamiento para graficar
train_predict_plot = np.empty_like(data.values)
train_predict_plot[:, :] = np.nan
train_predict_plot[time_step:len(train_predict) + time_step, :] = train_predict

# Desplazar las predicciones de prueba para graficar
test_predict_plot = np.empty_like(data.values)
test_predict_plot[:, :] = np.nan
test_predict_plot[len(train_predict) + (time_step * 2) + 1:len(data) - 1, :] = test_predict

# Graficar línea base y predicciones
plt.plot(scaler.inverse_transform(data.values), label='Datos Reales')
plt.plot(train_predict_plot, label='Predicción de Entrenamiento')
plt.plot(test_predict_plot, label='Predicción de Prueba')
plt.legend()
plt.show()

Conclusión

En este tema, cubrimos lo esencial del pronóstico de series temporales usando TensorFlow. Comenzamos con la preparación de datos, pasamos a la construcción y entrenamiento de un modelo LSTM, y finalmente evaluamos el rendimiento del modelo. Siguiendo estos pasos, ahora deberías tener una base sólida para abordar proyectos de pronóstico de series temporales más complejos.
