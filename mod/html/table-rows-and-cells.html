Filas y Celdas de Tabla
Atributos de Tabla ►

En esta sección, exploraremos cómo crear y manipular filas y celdas de tabla en HTML. Las tablas son una parte fundamental del desarrollo web, permitiéndote organizar datos en un formato estructurado. Cubriremos las etiquetas básicas utilizadas para crear tablas, filas y celdas, y proporcionaremos ejemplos y ejercicios para solidificar tu comprensión.

Conceptos Clave

  • Tabla (<table>): El elemento contenedor para una tabla.
  • Fila de Tabla (<tr>): Define una fila en una tabla.
  • Encabezado de Tabla (<th>): Define una celda de encabezado en una tabla.
  • Datos de Tabla (<td>): Define una celda estándar en una tabla.

Creando una Tabla Básica

Para crear una tabla, necesitas usar el elemento <table>. Dentro de este elemento, puedes definir filas usando el elemento <tr>. Cada fila puede contener celdas de encabezado (<th>) y celdas de datos (<td>).

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tabla Básica</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
            <th>Encabezado 3</th>
        </tr>
        <tr>
            <td>Dato 1</td>
            <td>Dato 2</td>
            <td>Dato 3</td>
        </tr>
        <tr>
            <td>Dato 4</td>
            <td>Dato 5</td>
            <td>Dato 6</td>
        </tr>
    </table>
</body>
</html>

Explicación

  • <table border="1">: Crea una tabla con un borde.
  • <tr>: Define una fila de tabla.
  • <th>: Define una celda de encabezado, que es negrita y centrada por defecto.
  • <td>: Define una celda estándar.

Añadiendo Más Filas y Celdas

Puedes añadir tantas filas y celdas como necesites. Cada fila (<tr>) puede contener múltiples celdas de encabezado (<th>) o de datos (<td>).

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tabla Extendida</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
            <th>Encabezado 3</th>
        </tr>
        <tr>
            <td>Dato 1</td>
            <td>Dato 2</td>
            <td>Dato 3</td>
        </tr>
        <tr>
            <td>Dato 4</td>
            <td>Dato 5</td>
            <td>Dato 6</td>
        </tr>
        <tr>
            <td>Dato 7</td>
            <td>Dato 8</td>
            <td>Dato 9</td>
        </tr>
    </table>
</body>
</html>

Fusionando Celdas

Puedes fusionar celdas horizontal o verticalmente usando los atributos colspan y rowspan.

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Celdas Fusionadas</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
            <th>Encabezado 3</th>
        </tr>
        <tr>
            <td colspan="2">Celda Fusionada 1</td>
            <td>Dato 3</td>
        </tr>
        <tr>
            <td>Dato 4</td>
            <td rowspan="2">Celda Fusionada 2</td>
            <td>Dato 6</td>
        </tr>
        <tr>
            <td>Dato 7</td>
            <td>Dato 9</td>
        </tr>
    </table>
</body>
</html>

Explicación

  • colspan="2": Fusiona dos celdas horizontalmente.
  • rowspan="2": Fusiona dos celdas verticalmente.

Estilizando Tablas

Las tablas pueden ser estilizadas usando CSS para mejorar su apariencia.

Ejemplo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tabla Estilizada</title>
    <style>
        table {
            width: 100%;
            border-collapse: collapse;
        }
        th, td {
            border: 1px solid black;
            padding: 8px;
            text-align: left;
        }
        th {
            background-color: #f2f2f2;
        }
    </style>
</head>
<body>
    <table>
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
            <th>Encabezado 3</th>
        </tr>
        <tr>
            <td>Dato 1</td>
            <td>Dato 2</td>
            <td>Dato 3</td>
        </tr>
        <tr>
            <td>Dato 4</td>
            <td>Dato 5</td>
            <td>Dato 6</td>
        </tr>
    </table>
</body>
</html>

Explicación

  • border-collapse: collapse;: Colapsa los bordes en un solo borde.
  • padding: 8px;: Añade relleno dentro de las celdas.
  • background-color: #f2f2f2;: Establece el color de fondo de las celdas de encabezado.

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación de tablas en HTML, incluyendo cómo definir filas y celdas, fusionar celdas y aplicar estilos básicos. Las tablas son una herramienta poderosa para organizar datos, y entender cómo usarlas efectivamente es esencial para cualquier desarrollador web. Practica creando tus propias tablas y experimenta con diferentes estilos para sentirte más cómodo con estos conceptos.
