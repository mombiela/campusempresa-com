Introducción

¡Bienvenido a tu viaje en HTML! En esta sección, te guiaremos a través del proceso de crear tu primera página HTML. Al final de esta lección, tendrás una comprensión básica de la estructura de HTML y serás capaz de crear una página web simple.

Conceptos Clave

  • HTML (HyperText Markup Language): El lenguaje estándar para crear páginas web.
  • Etiquetas: HTML usa etiquetas para estructurar el contenido. Las etiquetas están encerradas en corchetes angulares, por ejemplo, <tagname>.
  • Elementos: Un elemento consiste en una etiqueta de inicio, contenido y una etiqueta de cierre, por ejemplo, <p>Este es un párrafo.</p>.
  • Atributos: Las etiquetas pueden tener atributos que proporcionan información adicional sobre los elementos, por ejemplo, <a href="https://example.com">Enlace</a>.

Estructura Básica de un Documento HTML

Cada documento HTML tiene una estructura básica que incluye los siguientes elementos:

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primera Página HTML</title>
</head>
<body>
    <h1>Bienvenido a Mi Primera Página HTML</h1>
    <p>Este es un párrafo de texto en mi primera página HTML.</p>
</body>
</html>

Explicación del Código

  • <!DOCTYPE html>: Esta declaración define el tipo de documento y la versión de HTML. Ayuda al navegador a entender cómo renderizar la página.
  • <html>: El elemento raíz que contiene todos los demás elementos HTML.
  • <head>: Contiene meta-información sobre el documento, como el título y enlaces a hojas de estilo.
  • <title>: Establece el título de la página web, que aparece en la pestaña del navegador.
  • <body>: Contiene el contenido del documento HTML, como texto, imágenes y enlaces.
  • <h1>: Una etiqueta de encabezado. Hay seis niveles de encabezados, desde <h1> (más importante) hasta <h6> (menos importante).
  • <p>: Una etiqueta de párrafo utilizada para definir un bloque de texto.

Creando Tu Primera Página HTML

Sigue estos pasos para crear tu primera página HTML:

  1. Abre un Editor de Texto:

    • Puedes usar cualquier editor de texto, como Notepad (Windows), TextEdit (Mac), o un editor de código como Visual Studio Code o Sublime Text.

  2. Escribe el Código HTML:

    • Copia la estructura básica proporcionada arriba en tu editor de texto.

  3. Guarda el Archivo:

    • Guarda el archivo con una extensión .html, por ejemplo, index.html.

  4. Abre el Archivo HTML en un Navegador:

    • Haz doble clic en el archivo HTML guardado o ábrelo en un navegador web (Chrome, Firefox, Safari, etc.).

Ejemplo: Añadiendo Más Contenido

Vamos a añadir más contenido a nuestra página HTML para hacerla más interesante.

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primera Página HTML</title>
</head>
<body>
    <h1>Bienvenido a Mi Primera Página HTML</h1>
    <p>Este es un párrafo de texto en mi primera página HTML.</p>
    <h2>Sobre Mí</h2>
    <p>Mi nombre es John Doe, y estoy aprendiendo HTML.</p>
    <ul>
        <li>Me encanta programar.</li>
        <li>Disfruto aprendiendo nuevas tecnologías.</li>
        <li>Estoy emocionado de construir mi primer sitio web.</li>
    </ul>
    <a href="https://example.com">Visita Example.com</a>
</body>
</html>

Explicación del Código Adicional

  • <h2>: Un encabezado de segundo nivel.
  • <ul>: Una lista desordenada.
  • <li>: Un elemento de lista dentro de una lista desordenada.
  • <a href="URL">: Una etiqueta de anclaje utilizada para crear hipervínculos. El atributo href especifica la URL de la página a la que va el enlace.

Conclusión

¡Felicidades! Has creado tu primera página HTML. Ahora entiendes la estructura básica de un documento HTML y cómo añadir encabezados, párrafos, listas y enlaces. Practica creando más páginas HTML para familiarizarte más con estos conceptos. En las próximas lecciones, profundizaremos en los elementos y atributos de HTML para mejorar aún más tus páginas web. ¡Feliz codificación!
