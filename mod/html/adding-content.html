Añadiendo Contenido
Estilizando con CSS ►

Añadir contenido es uno de los aspectos fundamentales de la creación de páginas web con HTML. Este tema te guiará a través de los conceptos básicos de añadir texto, imágenes, enlaces y otros elementos multimedia a tus páginas web, progresando desde niveles principiantes hasta avanzados.

Contenido de Texto

El texto es la forma más básica de contenido que puedes añadir a una página web. HTML proporciona varias etiquetas para estructurar y formatear el texto.

Párrafos y Encabezados

  • Párrafos: Usa la etiqueta <p> para definir un párrafo.
  • Encabezados: Usa las etiquetas <h1> a <h6> para definir encabezados, siendo <h1> el nivel más alto y <h6> el más bajo.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Contenido de Texto</title>
</head>
<body>
    <h1>Este es un Encabezado H1</h1>
    <p>Este es un párrafo de texto.</p>
    <h2>Este es un Encabezado H2</h2>
    <p>Otro párrafo de texto.</p>
</body>
</html>

Listas

  • Listas Ordenadas: Usa la etiqueta <ol> para listas ordenadas (numeradas).
  • Listas Desordenadas: Usa la etiqueta <ul> para listas desordenadas (con viñetas).
  • Elementos de Lista: Usa la etiqueta <li> para definir elementos de lista.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Listas</title>
</head>
<body>
    <h2>Lista Ordenada</h2>
    <ol>
        <li>Primer ítem</li>
        <li>Segundo ítem</li>
        <li>Tercer ítem</li>
    </ol>
    <h2>Lista Desordenada</h2>
    <ul>
        <li>Primer ítem</li>
        <li>Segundo ítem</li>
        <li>Tercer ítem</li>
    </ul>
</body>
</html>

Imágenes

Las imágenes pueden añadirse a tus páginas web usando la etiqueta <img>. El atributo src especifica la ruta de la imagen, y el atributo alt proporciona texto alternativo para lectores de pantalla.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Imágenes</title>
</head>
<body>
    <h2>Añadiendo una Imagen</h2>
    <img src="ruta/a/la/imagen.jpg" alt="Descripción de la imagen">
</body>
</html>

Enlaces

Los enlaces se crean usando la etiqueta <a>. El atributo href especifica la URL de la página a la que lleva el enlace.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Enlaces</title>
</head>
<body>
    <h2>Añadiendo un Enlace</h2>
    <a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
</body>
</html>

Contenido Multimedia

Audio

Puedes añadir audio a tus páginas web usando la etiqueta <audio>. El atributo controls añade controles de reproducción, pausa y volumen.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Audio</title>
</head>
<body>
    <h2>Añadiendo Audio</h2>
    <audio controls>
        <source src="ruta/al/audio.mp3" type="audio/mpeg">
        Tu navegador no soporta el elemento de audio.
    </audio>
</body>
</html>

Video

La etiqueta <video> se usa para incrustar contenido de video. Similar a la etiqueta <audio>, puede incluir controls para la interacción del usuario.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Video</title>
</head>
<body>
    <h2>Añadiendo Video</h2>
    <video controls>
        <source src="ruta/al/video.mp4" type="video/mp4">
        Tu navegador no soporta la etiqueta de video.
    </video>
</body>
</html>

Tablas

Las tablas se usan para mostrar datos en un formato tabular. Usa la etiqueta <table> para crear una tabla, <tr> para filas de tabla, <th> para encabezados de tabla y <td> para celdas de datos de tabla.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Tablas</title>
</head>
<body>
    <h2>Añadiendo una Tabla</h2>
    <table border="1">
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
        </tr>
        <tr>
            <td>Dato 1</td>
            <td>Dato 2</td>
        </tr>
        <tr>
            <td>Dato 3</td>
            <td>Dato 4</td>
        </tr>
    </table>
</body>
</html>

Conclusión

En este tema, has aprendido cómo añadir varios tipos de contenido a tus páginas web usando HTML. Desde texto básico y listas hasta imágenes, enlaces, multimedia y tablas, estos elementos forman los bloques de construcción del contenido web. Practica usando estas etiquetas para familiarizarte con la adición y estructuración de contenido en tus páginas web.
