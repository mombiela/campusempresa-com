Introducción a los Elementos HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Los elementos HTML son los bloques de construcción de las páginas HTML. Están representados por etiquetas y pueden contener texto, atributos, otros elementos, o ser auto-cerrados.

Conceptos Clave

  • Etiquetas: Los elementos HTML se definen mediante etiquetas, que están encerradas en corchetes angulares (< >).
  • Atributos: Las etiquetas pueden tener atributos que proporcionan información adicional sobre el elemento.
  • Anidamiento: Los elementos HTML pueden estar anidados dentro de otros elementos.
  • Etiquetas auto-cerradas: Algunos elementos HTML no tienen una etiqueta de cierre.

Elementos HTML Básicos

Párrafos y Encabezados

  • Párrafo (<p>): Se usa para definir un bloque de texto.
  • Encabezados (<h1> a <h6>): Se usan para definir encabezados, siendo <h1> el nivel más alto y <h6> el más bajo.
<h1>Este es un Encabezado 1</h1>
<p>Este es un párrafo.</p>
<h2>Este es un Encabezado 2</h2>
<p>Este es otro párrafo.</p>

Listas

  • Lista Ordenada (<ol>): Crea una lista numerada.
  • Lista Desordenada (<ul>): Crea una lista con viñetas.
  • Elemento de Lista (<li>): Define un elemento en una lista.
<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
</ol>
<ul>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
</ul>

Enlaces e Imágenes

  • Ancla (<a>): Crea un hipervínculo.
  • Imagen (<img>): Inserta una imagen.
<a href="https://www.example.com">Visitar Example.com</a>
<img src="image.jpg" alt="Descripción de la imagen">

Tablas

Las tablas se usan para mostrar datos en un formato tabular.

  • Tabla (<table>): Define una tabla.
  • Fila de Tabla (<tr>): Define una fila en una tabla.
  • Encabezado de Tabla (<th>): Define una celda de encabezado en una tabla.
  • Dato de Tabla (<td>): Define una celda estándar en una tabla.
<table>
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

Elementos HTML Intermedios

Formularios

Los formularios se usan para recopilar datos del usuario.

  • Formulario (<form>): Define un formulario.
  • Entrada (<input>): Define un campo de entrada.
  • Etiqueta (<label>): Define una etiqueta para un elemento de entrada.
  • Botón (<button>): Define un botón clicable.
<form action="/submit" method="post">
  <label for="name">Nombre:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Enviar</button>
</form>

Elementos Semánticos

Los elementos semánticos proporcionan significado a la estructura de la página web.

  • Encabezado (<header>): Define un encabezado para un documento o sección.
  • Pie de página (<footer>): Define un pie de página para un documento o sección.
  • Artículo (<article>): Define un contenido independiente y autónomo.
  • Sección (<section>): Define una sección en un documento.
  • Navegación (<nav>): Define enlaces de navegación.
<header>
  <h1>Título del Sitio Web</h1>
</header>
<nav>
  <a href="#home">Inicio</a>
  <a href="#about">Acerca de</a>
</nav>
<section>
  <article>
    <h2>Título del Artículo</h2>
    <p>El contenido del artículo va aquí.</p>
  </article>
</section>
<footer>
  <p>El contenido del pie de página va aquí.</p>
</footer>

Elementos HTML Avanzados

Elementos Multimedia

  • Audio (<audio>): Inserta contenido de audio.
  • Video (<video>): Inserta contenido de video.
<audio controls>
  <source src="audiofile.mp3" type="audio/mpeg">
  Tu navegador no soporta el elemento de audio.
</audio>
<video width="320" height="240" controls>
  <source src="videofile.mp4" type="video/mp4">
  Tu navegador no soporta la etiqueta de video.
</video>

Canvas y SVG

  • Canvas (<canvas>): Se usa para dibujar gráficos mediante scripting (usualmente JavaScript).
  • SVG (<svg>): Se usa para definir gráficos basados en vectores.
<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;">
  Tu navegador no soporta la etiqueta canvas de HTML.
</canvas>
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Conclusión

Los elementos HTML son los componentes fundamentales de las páginas web. Entender cómo usar estos elementos de manera efectiva es crucial para crear páginas web bien estructuradas y funcionales. Desde elementos básicos como párrafos y encabezados hasta elementos avanzados como multimedia y gráficos, dominar los elementos HTML es el primer paso para convertirse en un desarrollador web competente.
