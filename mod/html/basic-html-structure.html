Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Entender la estructura básica de un documento HTML es esencial para cualquiera que desee construir sitios web. En esta sección, cubriremos los elementos fundamentales que componen un documento HTML.

Conceptos Clave

  • Estructura del Documento HTML: El esqueleto básico de un documento HTML.
  • Etiquetas y Elementos: Los bloques de construcción de HTML.
  • Atributos: Información adicional proporcionada dentro de las etiquetas.
  • Anidamiento: Colocar elementos dentro de otros elementos.

Estructura del Documento HTML

Un documento HTML está compuesto por varios componentes clave. Aquí hay un ejemplo simple de un documento HTML básico:

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primera Página HTML</title>
</head>
<body>
    <h1>Bienvenido a Mi Sitio Web</h1>
    <p>Este es un párrafo de texto en mi primera página HTML.</p>
</body>
</html>

Explicación

  • <!DOCTYPE html>: Esta declaración define el tipo de documento y la versión de HTML. Ayuda al navegador a entender cómo renderizar el contenido.
  • <html>: El elemento raíz que envuelve todo el contenido del documento HTML.
  • <head>: Contiene meta-información sobre el documento, como el título y enlaces a hojas de estilo.
  • <title>: Establece el título de la página web, que aparece en la pestaña del navegador.
  • <body>: Contiene el contenido del documento HTML que es visible para el usuario, como texto, imágenes y enlaces.

Etiquetas y Elementos

HTML usa etiquetas para crear elementos. Las etiquetas están encerradas en corchetes angulares, y la mayoría de los elementos tienen una etiqueta de apertura y una de cierre.

Etiquetas Comunes

  • <h1> a <h6>: Etiquetas de encabezado, con <h1> siendo el nivel más alto y <h6> el más bajo.
  • <p>: Etiqueta de párrafo para bloques de texto.
  • <a>: Etiqueta de ancla para hipervínculos.
  • <img>: Etiqueta de imagen para mostrar imágenes.
  • <ul>, <ol>, <li>: Etiquetas para crear listas (desordenadas y ordenadas).

Ejemplo

<h1>Este es un Encabezado</h1>
<p>Este es un párrafo.</p>
<a href="https://www.example.com">Este es un enlace</a>
<img src="image.jpg" alt="Descripción de la imagen">
<ul>
    <li>Primer ítem</li>
    <li>Segundo ítem</li>
</ul>

Atributos

Los atributos proporcionan información adicional sobre los elementos HTML. Siempre se incluyen en la etiqueta de apertura y generalmente vienen en pares de nombre/valor como nombre="valor".

Atributos Comunes

  • href: Usado con etiquetas <a> para especificar la URL del enlace.
  • src: Usado con etiquetas <img> para especificar la ruta de la imagen.
  • alt: Usado con etiquetas <img> para proporcionar texto alternativo para la imagen.
  • class: Usado para definir una clase para un elemento, que puede ser usada para estilizar con CSS.
  • id: Usado para identificar de manera única un elemento.

Ejemplo

<a href="https://www.example.com" target="_blank">Visitar Ejemplo</a>
<img src="image.jpg" alt="Un paisaje hermoso" width="500" height="300">
<p class="intro">Este es un párrafo introductorio.</p>
<div id="contenido-principal">Esta es el área de contenido principal.</div>

Anidamiento

El anidamiento se refiere a colocar elementos HTML dentro de otros elementos. El anidamiento adecuado es crucial para crear documentos HTML bien estructurados y legibles.

Ejemplo

<div>
    <h2>Título de la Sección</h2>
    <p>Este es un párrafo dentro de un div.</p>
    <ul>
        <li>Ítem de lista anidado 1</li>
        <li>Ítem de lista anidado 2</li>
    </ul>
</div>

Resumen

Entender la estructura básica de un documento HTML es el primer paso en el desarrollo web. Al dominar el uso de etiquetas, elementos, atributos y anidamiento, puedes crear páginas web bien estructuradas y significativas. A medida que progreses, estas habilidades fundamentales serán esenciales para temas más avanzados de HTML y desarrollo web.
