Introducción al Almacenamiento Web

El Almacenamiento Web es una característica en HTML5 que permite a las aplicaciones web almacenar datos localmente dentro del navegador del usuario. Esto es útil para guardar preferencias del usuario, mantener información de sesión y más, sin la necesidad de almacenamiento en el servidor.

Conceptos Clave

  • Almacenamiento Local: Almacena datos sin fecha de expiración. Los datos persisten incluso cuando el navegador se cierra y se vuelve a abrir.
  • Almacenamiento de Sesión: Almacena datos durante la duración de la sesión de la página. Los datos se eliminan cuando la sesión de la página termina (es decir, cuando se cierra la pestaña del navegador).

Almacenamiento Local

El Almacenamiento Local proporciona una forma de almacenar pares clave-valor en un navegador web sin fecha de expiración.

Operaciones Básicas

  • Set Item: Almacenar datos.
  • Get Item: Recuperar datos.
  • Remove Item: Eliminar datos.
  • Clear: Limpiar todos los datos almacenados.

Ejemplos de Código

Estableciendo un Ítem

// Almacenar datos en el almacenamiento local
localStorage.setItem('username', 'JohnDoe');

Obteniendo un Ítem

// Recuperar datos del almacenamiento local
let username = localStorage.getItem('username');
console.log(username); // Output: JohnDoe

Eliminando un Ítem

// Eliminar datos del almacenamiento local
localStorage.removeItem('username');

Limpiando Todos los Ítems

// Limpiar todos los datos del almacenamiento local
localStorage.clear();

Ejemplo Práctico

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Almacenamiento Local</title>
</head>
<body>
    <input type="text" id="nameInput" placeholder="Introduce tu nombre">
    <button onclick="saveName()">Guardar Nombre</button>
    <button onclick="showName()">Mostrar Nombre</button>
    <p id="displayName"></p>

    <script>
        function saveName() {
            let name = document.getElementById('nameInput').value;
            localStorage.setItem('name', name);
        }

        function showName() {
            let name = localStorage.getItem('name');
            document.getElementById('displayName').innerText = name ? `Hola, ${name}!` : 'No se encontró ningún nombre.';
        }
    </script>
</body>
</html>

Almacenamiento de Sesión

El Almacenamiento de Sesión funciona de manera similar al Almacenamiento Local pero con una vida útil más corta. Los datos solo están disponibles durante la duración de la sesión de la página.

Operaciones Básicas

  • Set Item: Almacenar datos.
  • Get Item: Recuperar datos.
  • Remove Item: Eliminar datos.
  • Clear: Limpiar todos los datos almacenados.

Ejemplos de Código

Estableciendo un Ítem

// Almacenar datos en el almacenamiento de sesión
sessionStorage.setItem('sessionUser', 'JaneDoe');

Obteniendo un Ítem

// Recuperar datos del almacenamiento de sesión
let sessionUser = sessionStorage.getItem('sessionUser');
console.log(sessionUser); // Output: JaneDoe

Eliminando un Ítem

// Eliminar datos del almacenamiento de sesión
sessionStorage.removeItem('sessionUser');

Limpiando Todos los Ítems

// Limpiar todos los datos del almacenamiento de sesión
sessionStorage.clear();

Ejemplo Práctico

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo de Almacenamiento de Sesión</title>
</head>
<body>
    <input type="text" id="sessionInput" placeholder="Introduce datos de sesión">
    <button onclick="saveSessionData()">Guardar Datos de Sesión</button>
    <button onclick="showSessionData()">Mostrar Datos de Sesión</button>
    <p id="displaySessionData"></p>

    <script>
        function saveSessionData() {
            let data = document.getElementById('sessionInput').value;
            sessionStorage.setItem('sessionData', data);
        }

        function showSessionData() {
            let data = sessionStorage.getItem('sessionData');
            document.getElementById('displaySessionData').innerText = data ? `Datos de Sesión: ${data}` : 'No se encontraron datos de sesión.';
        }
    </script>
</body>
</html>

Comparación entre Almacenamiento Local y Almacenamiento de Sesión

Característica Almacenamiento Local Almacenamiento de Sesión
Vida Útil Sin fecha de expiración Duración de la sesión de la página
Persistencia de Datos Persistente entre sesiones del navegador Se elimina cuando se cierra la pestaña
Capacidad de Almacenamiento Típicamente 5-10 MB por origen Típicamente 5-10 MB por origen

Conclusión

El Almacenamiento Web proporciona una forma poderosa de almacenar datos en el lado del cliente, ofreciendo tanto Almacenamiento Local para datos persistentes como Almacenamiento de Sesión para datos temporales. Comprender cómo usar estos mecanismos de almacenamiento puede mejorar en gran medida la funcionalidad y la experiencia del usuario en las aplicaciones web.
