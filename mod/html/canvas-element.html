Introducción al Elemento Canvas

El elemento <canvas> es una característica poderosa en HTML5 que permite la renderización dinámica y scriptable de formas 2D e imágenes de mapa de bits. Es ampliamente utilizado para crear gráficos, animaciones, juegos y visualizaciones de datos.

Conceptos Clave

  • Elemento: El propio elemento <canvas>, que actúa como un contenedor para gráficos.
  • Contexto: El contexto de dibujo, que proporciona los métodos y propiedades para dibujar en el canvas.
  • Renderización: El proceso de dibujar formas, texto, imágenes y otros objetos en el canvas.

Uso Básico

Para usar el elemento <canvas>, necesitas:

  1. Definir el elemento <canvas> en tu HTML.
  2. Acceder al canvas a través de JavaScript para dibujar en él.

Estructura HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo de Canvas</title>
</head>
<body>
    <canvas id="myCanvas" width="500" height="500" style="border:1px solid #000000;"></canvas>
</body>
</html>
  • El atributo id se usa para referenciar el canvas en JavaScript.
  • Los atributos width y height definen el tamaño del canvas.

Accediendo al Canvas en JavaScript

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
  • El método getContext('2d') devuelve un contexto de dibujo en el canvas.

Dibujando Formas

Dibujando Rectángulos

// Dibujar un rectángulo relleno
ctx.fillStyle = 'green';
ctx.fillRect(10, 10, 150, 100);

// Dibujar un rectángulo con borde
ctx.strokeStyle = 'blue';
ctx.strokeRect(200, 10, 150, 100);
  • fillStyle establece el color para rellenar el rectángulo.
  • fillRect(x, y, width, height) dibuja un rectángulo relleno.
  • strokeStyle establece el color para el borde del rectángulo.
  • strokeRect(x, y, width, height) dibuja el contorno de un rectángulo.

Dibujando Rutas

// Comenzar una nueva ruta
ctx.beginPath();
ctx.moveTo(50, 50); // Moverse al punto de inicio
ctx.lineTo(200, 50); // Dibujar una línea hasta el siguiente punto
ctx.lineTo(200, 200); // Dibujar otra línea
ctx.closePath(); // Cerrar la ruta
ctx.stroke(); // Contornear la ruta
  • beginPath() inicia una nueva ruta.
  • moveTo(x, y) mueve el punto de inicio.
  • lineTo(x, y) dibuja una línea hasta el punto especificado.
  • closePath() cierra la ruta.
  • stroke() contornea la ruta.

Técnicas Avanzadas

Dibujando Círculos y Arcos

ctx.beginPath();
ctx.arc(150, 150, 75, 0, 2 * Math.PI); // Dibujar un círculo
ctx.fillStyle = 'red';
ctx.fill();
ctx.stroke();
  • arc(x, y, radius, startAngle, endAngle) dibuja un arco o círculo.

Dibujando Texto

ctx.font = '30px Arial';
ctx.fillStyle = 'purple';
ctx.fillText('Hola Canvas', 50, 250);
ctx.strokeText('Hola Canvas', 50, 300);
  • font establece las propiedades de la fuente.
  • fillText(text, x, y) dibuja texto relleno.
  • strokeText(text, x, y) dibuja el contorno del texto.

Trabajando con Imágenes

const img = new Image();
img.onload = function() {
    ctx.drawImage(img, 0, 0, 300, 200);
};
img.src = 'path/to/image.jpg';
  • drawImage(image, x, y, width, height) dibuja una imagen en el canvas.

Conclusión

El elemento <canvas> es una herramienta versátil en HTML5 para crear gráficos dinámicos. Al entender los conceptos básicos del elemento canvas, dibujar formas, rutas, texto e imágenes, puedes crear una amplia gama de contenido visual. A medida que avances, puedes explorar características más complejas como animaciones, transformaciones y gráficos interactivos para mejorar tus aplicaciones web.
