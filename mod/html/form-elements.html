Introducción

Los formularios son una parte crucial del desarrollo web, permitiendo a los usuarios interactuar con los sitios web enviando datos. En esta sección, exploraremos varios elementos de formulario HTML, sus atributos y cómo usarlos de manera efectiva.

Estructura Básica del Formulario

Un formulario en HTML se define usando la etiqueta <form>. Aquí hay un ejemplo simple:

<form action="/submit" method="post">
  <!-- Los elementos del formulario van aquí -->
</form>
  • action: Especifica la URL a la que se deben enviar los datos del formulario.
  • method: Especifica el método HTTP a usar al enviar los datos del formulario (GET o POST).

Elementos Comunes del Formulario

Entrada de Texto

El elemento <input> con type="text" se usa para la entrada de texto de una sola línea.

<form action="/submit" method="post">
  <label for="name">Nombre:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Enviar">
</form>
  • label: Asocia una etiqueta de texto con un elemento del formulario.
  • id: Identificador único para el elemento del formulario.
  • name: Nombre del elemento del formulario, usado para identificar los datos cuando se envían.
  • type: Especifica el tipo de entrada (text en este caso).

Entrada de Contraseña

El elemento <input> con type="password" se usa para campos de contraseña.

<form action="/submit" method="post">
  <label for="password">Contraseña:</label>
  <input type="password" id="password" name="password">
  <input type="submit" value="Enviar">
</form>

Botones de Radio

Los botones de radio permiten a los usuarios seleccionar una opción de un conjunto.

<form action="/submit" method="post">
  <label for="gender">Género:</label>
  <input type="radio" id="male" name="gender" value="male">
  <label for="male">Hombre</label>
  <input type="radio" id="female" name="gender" value="female">
  <label for="female">Mujer</label>
  <input type="submit" value="Enviar">
</form>

Casillas de Verificación

Las casillas de verificación permiten a los usuarios seleccionar múltiples opciones.

<form action="/submit" method="post">
  <label for="hobbies">Aficiones:</label>
  <input type="checkbox" id="reading" name="hobbies" value="reading">
  <label for="reading">Leer</label>
  <input type="checkbox" id="traveling" name="hobbies" value="traveling">
  <label for="traveling">Viajar</label>
  <input type="submit" value="Enviar">
</form>

Lista Desplegable

El elemento <select> crea una lista desplegable.

<form action="/submit" method="post">
  <label for="country">País:</label>
  <select id="country" name="country">
    <option value="usa">EE.UU.</option>
    <option value="canada">Canadá</option>
    <option value="mexico">México</option>
  </select>
  <input type="submit" value="Enviar">
</form>

Área de Texto

El elemento <textarea> se usa para la entrada de texto de varias líneas.

<form action="/submit" method="post">
  <label for="message">Mensaje:</label>
  <textarea id="message" name="message" rows="4" cols="50"></textarea>
  <input type="submit" value="Enviar">
</form>

Botones

Los botones pueden ser creados usando el elemento <button> o el elemento <input> con type="submit", type="reset" o type="button".

<form action="/submit" method="post">
  <button type="submit">Enviar</button>
  <button type="reset">Restablecer</button>
</form>

Elementos Avanzados del Formulario

Entrada de Archivo

El elemento <input> con type="file" permite a los usuarios subir archivos.

<form action="/upload" method="post" enctype="multipart/form-data">
  <label for="file">Subir un archivo:</label>
  <input type="file" id="file" name="file">
  <input type="submit" value="Subir">
</form>
  • enctype: Especifica cómo deben codificarse los datos del formulario al enviarlos al servidor. Para subir archivos, use multipart/form-data.

Entrada de Fecha y Hora

HTML5 introdujo nuevos tipos de entrada para fecha y hora.

<form action="/submit" method="post">
  <label for="birthday">Cumpleaños:</label>
  <input type="date" id="birthday" name="birthday">
  <label for="appointment">Cita:</label>
  <input type="datetime-local" id="appointment" name="appointment">
  <input type="submit" value="Enviar">
</form>

Entrada de Rango

El elemento <input> con type="range" crea un control deslizante.

<form action="/submit" method="post">
  <label for="volume">Volumen:</label>
  <input type="range" id="volume" name="volume" min="0" max="100">
  <input type="submit" value="Enviar">
</form>

Validación de Formularios

HTML5 proporciona atributos de validación de formularios integrados.

  • required: Especifica que un campo de entrada debe ser completado antes de enviar el formulario.
  • pattern: Especifica una expresión regular que el valor del campo de entrada debe coincidir.
  • min y max: Especifican los valores mínimo y máximo para campos de entrada numéricos.
<form action="/submit" method="post">
  <label for="email">Correo Electrónico:</label>
  <input type="email" id="email" name="email" required>
  <label for="age">Edad:</label>
  <input type="number" id="age" name="age" min="18" max="99">
  <input type="submit" value="Enviar">
</form>

Conclusión

Entender y usar los elementos de formulario HTML de manera efectiva es esencial para crear aplicaciones web interactivas y fáciles de usar. Desde entradas de texto básicas hasta subidas de archivos avanzadas y validación integrada, los formularios HTML proporcionan una amplia gama de funcionalidades para recopilar y procesar datos de los usuarios. Practique el uso de estos elementos en varios escenarios para volverse competente en el manejo de formularios.
