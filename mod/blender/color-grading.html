Corrección de Color en Blender
Añadiendo Efectos ►

La corrección de color es una parte esencial del proceso de postproducción en el renderizado y la animación 3D. Involucra ajustar los colores y tonos de tus imágenes o animaciones renderizadas para lograr un aspecto o ambiente deseado. Este tema te guiará desde las técnicas básicas hasta las avanzadas de corrección de color en Blender.

Introducción a la Corrección de Color

La corrección de color puede cambiar dramáticamente la sensación de tu proyecto. Puede hacer que tus escenas se vean más cinematográficas, resaltar elementos importantes o crear una atmósfera específica.

Conceptos Clave

  • Corrección de Color: Ajustar los colores para que se vean naturales y consistentes.
  • Corrección de Color: Estilizar los colores para lograr un efecto artístico específico.
  • LUTs (Tablas de Consulta): Preajustes de corrección de color que se pueden aplicar a tus renders.
  • Scopes: Herramientas como histogramas, formas de onda y vectorescopios que ayudan a analizar la información del color.

Técnicas Básicas de Corrección de Color

Usando el Nodo de Balance de Color

El nodo de Balance de Color en Blender te permite ajustar el lift, gamma y gain de tu imagen, que corresponden a sombras, tonos medios y luces, respectivamente.

import bpy

# Crear un nuevo árbol de nodos para la composición
bpy.context.scene.use_nodes = True
tree = bpy.context.scene.node_tree

# Limpiar los nodos predeterminados
for node in tree.nodes:
    tree.nodes.remove(node)

# Añadir nodo de Capas de Render
render_layers = tree.nodes.new(type='CompositorNodeRLayers')

# Añadir nodo de Balance de Color
color_balance = tree.nodes.new(type='CompositorNodeColorBalance')

# Añadir nodo de Composición
composite = tree.nodes.new(type='CompositorNodeComposite')

# Enlazar nodos
tree.links.new(render_layers.outputs['Image'], color_balance.inputs['Image'])
tree.links.new(color_balance.outputs['Image'], composite.inputs['Image'])

# Ajustar configuraciones de Balance de Color
color_balance.correction_method = 'LIFT_GAMMA_GAIN'
color_balance.lift = [0.8, 0.8, 0.8]
color_balance.gamma = [1.2, 1.2, 1.2]
color_balance.gain = [1.1, 1.1, 1.1]

Usando el Nodo de Curvas RGB

El nodo de Curvas RGB proporciona un control más preciso sobre los canales de color y el brillo general.

import bpy

# Añadir nodo de Curvas RGB
rgb_curves = tree.nodes.new(type='CompositorNodeCurveRGB')

# Enlazar nodos
tree.links.new(render_layers.outputs['Image'], rgb_curves.inputs['Image'])
tree.links.new(rgb_curves.outputs['Image'], composite.inputs['Image'])

# Ajustar configuraciones de Curvas RGB
curve = rgb_curves.mapping.curves[3]  # Curva maestra
curve.points.new(0.3, 0.2)
curve.points.new(0.7, 0.8)

Técnicas Intermedias de Corrección de Color

Usando LUTs (Tablas de Consulta)

Las LUTs se pueden usar para aplicar preajustes de corrección de color complejos a tus renders.

import bpy

# Añadir un nodo LUT
lut_node = tree.nodes.new(type='CompositorNodeLUT')

# Enlazar nodos
tree.links.new(render_layers.outputs['Image'], lut_node.inputs['Image'])
tree.links.new(lut_node.outputs['Image'], composite.inputs['Image'])

# Cargar archivo LUT
lut_node.lut.filepath = '/ruta/a/tu/lut.cube'

Usando Scopes para Análisis

Los scopes te ayudan a analizar la información del color en tu imagen para hacer ajustes más informados.

  • Histograma: Muestra la distribución del brillo de los píxeles.
  • Forma de Onda: Muestra los niveles de brillo a lo largo de la imagen.
  • Vectorescopio: Visualiza la información del color.

Técnicas Avanzadas de Corrección de Color

Combinando Múltiples Nodos

Para una corrección de color avanzada, puedes combinar múltiples nodos para lograr efectos complejos.

import bpy

# Añadir múltiples nodos
hue_saturation = tree.nodes.new(type='CompositorNodeHueSat')
brightness_contrast = tree.nodes.new(type='CompositorNodeBrightContrast')

# Enlazar nodos
tree.links.new(render_layers.outputs['Image'], hue_saturation.inputs['Image'])
tree.links.new(hue_saturation.outputs['Image'], brightness_contrast.inputs['Image'])
tree.links.new(brightness_contrast.outputs['Image'], composite.inputs['Image'])

# Ajustar configuraciones
hue_saturation.color_saturation = 1.2
brightness_contrast.inputs['Bright'].default_value = 0.1
brightness_contrast.inputs['Contrast'].default_value = 1.5

Conclusión

La corrección de color es una herramienta poderosa en Blender que puede mejorar significativamente la calidad visual de tus proyectos. Al entender y utilizar los diversos nodos y técnicas disponibles, puedes lograr resultados de nivel profesional. Comienza con ajustes básicos usando los nodos de Balance de Color y Curvas RGB, explora las LUTs para preajustes rápidos y sumérgete en técnicas avanzadas combinando nodos y usando scopes para un control preciso. ¡Feliz corrección de color!
