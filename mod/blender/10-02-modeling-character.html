Modelando un Personaje

Siguiente ►

En este tema, aprenderás a modelar un personaje desde cero utilizando Blender. Este proceso implica varias etapas, desde la creación de una malla base hasta la adición de detalles y refinamientos. A lo largo de este módulo, te guiaremos paso a paso para que puedas crear tu propio personaje 3D.

Objetivos del Tema

  • Crear una malla base para el personaje.
  • Refinar la malla utilizando técnicas de modelado.
  • Añadir detalles y características al personaje.
  • Preparar el modelo para texturizado y animación.

  1. Creación de la Malla Base

1.1. Configuración Inicial

Antes de comenzar, asegúrate de tener una referencia visual de tu personaje. Puede ser un dibujo o una imagen que te ayude a mantener las proporciones y detalles correctos.

  1. Abrir Blender: Inicia Blender y crea un nuevo proyecto.
  2. Añadir una Primitiva: Añade una primitiva básica (por ejemplo, un cubo) que servirá como punto de partida para tu personaje. 
    bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))

1.2. Esculpido de la Forma Básica

Utiliza las herramientas de transformación para dar forma a la malla base.

  1. Modo de Edición: Cambia al modo de edición presionando Tab.
  2. Escalar y Mover: Usa las herramientas de escala (S) y mover (G) para ajustar la forma básica del cuerpo. 
    bpy.ops.transform.resize(value=(1, 0.5, 2))

1.3. Añadir Detalles Básicos

Añade más primitivas para crear las extremidades y la cabeza del personaje.

  1. Añadir una Esfera para la Cabeza: 
    bpy.ops.mesh.primitive_uv_sphere_add(radius=1, location=(0, 0, 2.5))
  2. Añadir Cilindros para los Brazos y Piernas: 
    bpy.ops.mesh.primitive_cylinder_add(radius=0.3, depth=2, location=(1, 0, 1))
bpy.ops.mesh.primitive_cylinder_add(radius=0.3, depth=2, location=(-1, 0, 1))
bpy.ops.mesh.primitive_cylinder_add(radius=0.3, depth=2, location=(0.5, 0, -1))
bpy.ops.mesh.primitive_cylinder_add(radius=0.3, depth=2, location=(-0.5, 0, -1))

  1. Refinamiento de la Malla

2.1. Unir las Primitivas

Une todas las primitivas en una sola malla para facilitar el modelado.

  1. Seleccionar Todas las Primitivas: En el modo de objeto, selecciona todas las primitivas (A).
  2. Unir Primitivas: Presiona Ctrl + J para unirlas en una sola malla.

2.2. Uso de Modificadores

Aplica modificadores para suavizar y refinar la malla.

  1. Añadir Modificador de Subdivisión de Superficie: 
    bpy.ops.object.modifier_add(type='SUBSURF')
bpy.context.object.modifiers["Subdivision"].levels = 2
bpy.context.object.modifiers["Subdivision"].render_levels = 2

2.3. Esculpido de Detalles

Utiliza las herramientas de esculpido para añadir detalles finos al personaje.

  1. Modo de Esculpido: Cambia al modo de esculpido (Ctrl + Tab y selecciona Sculpt Mode).
  2. Herramientas de Esculpido: Usa herramientas como Draw, Smooth, y Crease para esculpir detalles como músculos, facciones del rostro, etc.

  1. Preparación para Texturizado y Animación

3.1. Mapeo UV

Prepara el modelo para el texturizado mediante el mapeo UV.

  1. Modo de Edición: Cambia al modo de edición (Tab).
  2. Desplegar UVs: Selecciona todas las caras (A) y despliega las UVs (U > Unwrap).

3.2. Aplicación de Materiales Básicos

Aplica materiales básicos para visualizar mejor el modelo.

  1. Añadir Material: En el panel de propiedades, añade un nuevo material.
  2. Asignar Color Base: Asigna un color base al material para diferenciar las partes del personaje.

Ejercicio Práctico

Ejercicio 1: Modelar un Personaje Simple

  1. Objetivo: Crear un personaje simple utilizando las técnicas aprendidas.
  2. Instrucciones:
    • Crea una malla base utilizando primitivas.
    • Une las primitivas y aplica modificadores.
    • Esculpe detalles básicos.
    • Prepara el modelo para texturizado mediante el mapeo UV.

Solución del Ejercicio

  1. Crear Malla Base: Añade un cubo para el cuerpo, una esfera para la cabeza y cilindros para las extremidades.
  2. Unir y Refinar: Une las primitivas y aplica un modificador de subdivisión de superficie.
  3. Esculpir Detalles: Utiliza herramientas de esculpido para añadir detalles.
  4. Mapeo UV: Despliega las UVs para preparar el modelo para texturizado.

Conclusión

En este tema, has aprendido a modelar un personaje desde cero utilizando Blender. Has pasado por las etapas de creación de una malla base, refinamiento de la malla, y preparación para texturizado y animación. Con estas habilidades, estás listo para crear personajes más complejos y detallados en tus futuros proyectos. En el próximo tema, aprenderás a animar un clip corto utilizando el personaje que has modelado.
