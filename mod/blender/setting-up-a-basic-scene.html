Configuración de una Escena Básica
Introducción al Renderizado ►

En esta sección, cubriremos los pasos fundamentales para configurar una escena básica en Blender. Esto incluirá entender la interfaz de Blender, agregar y manipular objetos, y configurar iluminación y ángulos de cámara básicos. Al final de esta sección, tendrás una escena simple lista para un desarrollo posterior.

Descripción General de la Interfaz de Blender

Antes de sumergirnos en la creación de escenas, es esencial familiarizarse con la interfaz de Blender.

  • Vista 3D: El área principal donde interactúas con tus modelos 3D.
  • Outliner: Muestra una vista jerárquica de todos los objetos en tu escena.
  • Panel de Propiedades: Contiene configuraciones para el objeto y la escena seleccionados.
  • Línea de Tiempo: Usada para animación, mostrando fotogramas clave y controles de reproducción.

Agregar Objetos

Para crear una escena, necesitas agregar objetos. Blender proporciona varias formas primitivas para comenzar.

Agregar un Objeto de Malla

  1. Abre Blender y asegúrate de estar en la Vista 3D.
  2. Presiona Shift + A para abrir el menú Añadir.
  3. Navega a Malla y selecciona una forma primitiva, como Cubo.
# Ejemplo de agregar un cubo usando scripting en Python en Blender
import bpy

# Crear un nuevo cubo
bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))

Manipulación de Objetos

Una vez que se agrega un objeto, puedes moverlo, rotarlo y escalarlo.

  • Mover: Presiona G y mueve el ratón o escribe coordenadas.
  • Rotar: Presiona R y mueve el ratón o escribe ángulos.
  • Escalar: Presiona S y mueve el ratón o escribe factores de escala.

Configuración de la Iluminación

La iluminación es crucial para renderizar una escena. Blender proporciona diferentes tipos de luces.

Agregar una Fuente de Luz

  1. Presiona Shift + A para abrir el menú Añadir.
  2. Navega a Luz y selecciona un tipo, como Luz Puntual.
# Ejemplo de agregar una luz puntual usando scripting en Python en Blender
import bpy

# Crear una nueva luz puntual
bpy.ops.object.light_add(type='POINT', location=(5, 5, 5))

Ajuste de Propiedades de la Luz

Selecciona la luz y ve al Panel de Propiedades para ajustar sus configuraciones, como intensidad y color.

Configuración de la Cámara

La cámara determina el punto de vista para renderizar la escena.

Posicionamiento de la Cámara

  1. Selecciona la cámara en el Outliner.
  2. Usa las herramientas de Mover (G), Rotar (R) y Escalar (S) para posicionar la cámara.
# Ejemplo de posicionar la cámara usando scripting en Python en Blender
import bpy

# Establecer ubicación y rotación de la cámara
camera = bpy.data.objects['Camera']
camera.location = (7, -7, 7)
camera.rotation_euler = (1.1, 0, 0.8)

Vista de la Cámara

Para ver a través de la cámara, presiona 0 en el teclado numérico.

Resumen

En esta sección, cubrimos los conceptos básicos de la configuración de una escena en Blender:

  • Nos familiarizamos con la interfaz de Blender.
  • Agregamos y manipulamos objetos básicos.
  • Configuramos iluminación básica.
  • Posicionamos la cámara para renderizar.

Estas habilidades fundamentales son esenciales para crear escenas y animaciones más complejas. Practica estos pasos para sentirte cómodo con el entorno de Blender y prepárate para temas más avanzados.
