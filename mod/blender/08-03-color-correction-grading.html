Corrección y Gradación de Color
Siguiente ►

La corrección y gradación de color son procesos esenciales en la post-producción de cualquier proyecto visual. En Blender, estas técnicas se utilizan para ajustar y mejorar la apariencia de las imágenes y animaciones, asegurando que los colores sean precisos y estéticamente agradables.

Objetivos de Aprendizaje

Al final de esta lección, deberías ser capaz de:

  1. Entender la diferencia entre corrección y gradación de color.
  2. Utilizar las herramientas de corrección de color en Blender.
  3. Aplicar técnicas de gradación de color para mejorar la estética de tus proyectos.

1. Diferencia entre Corrección y Gradación de Color

Corrección de Color

La corrección de color se centra en ajustar los colores para que sean precisos y consistentes. Esto incluye:

  • Ajustar el balance de blancos.
  • Corregir la exposición.
  • Asegurar que los colores sean naturales y realistas.

Gradación de Color

La gradación de color, por otro lado, se enfoca en la estética y el estilo visual. Esto incluye:

  • Crear un ambiente o estado de ánimo específico.
  • Aplicar looks estilizados.
  • Ajustar los colores para que se alineen con la visión artística del proyecto.

2. Herramientas de Corrección de Color en Blender

Blender ofrece varias herramientas para la corrección de color, accesibles a través del Compositor. Aquí te mostramos cómo utilizarlas:

Paso 1: Configurar el Compositor

  1. Abre Blender y cambia al Editor de Composición.
  2. Activa la opción Use Nodes.
  3. Añade un nodo Render Layers y un nodo Composite.

Paso 2: Añadir un Nodo de Corrección de Color

  1. Presiona Shift + A para abrir el menú de adición.
  2. Navega a Color > RGB Curves y añade el nodo.
  3. Conecta el nodo Render Layers al nodo RGB Curves y luego al nodo Composite.

Paso 3: Ajustar los Colores

  1. Selecciona el nodo RGB Curves.
  2. Ajusta las curvas para corregir el balance de blancos, la exposición y otros aspectos del color.

3. Técnicas de Gradación de Color

Uso de Nodos de Gradación

Blender también ofrece nodos específicos para la gradación de color, como el nodo Color Balance.

Paso 1: Añadir un Nodo de Color Balance

  1. Presiona Shift + A para abrir el menú de adición.
  2. Navega a Color > Color Balance y añade el nodo.
  3. Conecta el nodo RGB Curves al nodo Color Balance y luego al nodo Composite.

Paso 2: Ajustar los Colores para Gradación

  1. Selecciona el nodo Color Balance.
  2. Ajusta los controles de Lift, Gamma y Gain para crear el look deseado.

Ejemplo Práctico

Vamos a aplicar una gradación de color para crear un ambiente cálido y acogedor en una escena.

  1. Ajuste de Curvas RGB:

    • Levanta ligeramente la curva de los tonos medios para aumentar la exposición.
    • Ajusta la curva de los tonos rojos para añadir calidez.

  2. Ajuste de Color Balance:

    • En Lift, añade un toque de rojo para los tonos oscuros.
    • En Gamma, ajusta hacia un tono más cálido.
    • En Gain, añade un poco de amarillo para los tonos claros.

Ejercicio Práctico

Ejercicio 1: Corrección de Color

  1. Abre una escena en Blender y configura el Compositor.
  2. Añade un nodo RGB Curves y ajusta las curvas para corregir el balance de blancos y la exposición.

Ejercicio 2: Gradación de Color

  1. Añade un nodo Color Balance a la configuración anterior.
  2. Ajusta los controles de Lift, Gamma y Gain para crear un look estilizado.

Soluciones

  • Corrección de Color: Asegúrate de que los colores sean naturales y la exposición sea correcta.
  • Gradación de Color: Experimenta con diferentes ajustes para encontrar el look que mejor se adapte a tu visión artística.

Conclusión

En esta lección, hemos aprendido la diferencia entre corrección y gradación de color, y cómo utilizar las herramientas de Blender para aplicar estas técnicas. La corrección de color asegura que los colores sean precisos y consistentes, mientras que la gradación de color permite crear un estilo visual único. Practica estas técnicas para mejorar la calidad visual de tus proyectos en Blender.
