Introducción a los Arrays

Los arrays son una estructura de datos fundamental en C# que te permite almacenar múltiples valores del mismo tipo en una sola variable. Son útiles para gestionar colecciones de datos de manera eficiente.

Conceptos Clave

  • Definición: Un array es una colección de variables que se acceden con un número de índice.
  • Tamaño Fijo: Los arrays tienen un tamaño fijo que se define cuando se crean.
  • Indexación Basada en Cero: El primer elemento de un array está en el índice 0.

Creación de Arrays

Para crear un array en C#, necesitas especificar el tipo de elementos que contendrá y su tamaño.

// Declarando e inicializando un array de enteros con 5 elementos
int[] numbers = new int[5];

// Declarando e inicializando un array con valores específicos
int[] predefinedNumbers = { 1, 2, 3, 4, 5 };

Acceso a Elementos del Array

Puedes acceder a los elementos en un array usando su índice.

int firstNumber = numbers[0]; // Accediendo al primer elemento
numbers[1] = 10; // Estableciendo el segundo elemento a 10

Iteración Sobre Arrays

Puedes usar bucles para iterar sobre arrays.

// Usando un bucle for
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}

// Usando un bucle foreach
foreach (int number in predefinedNumbers)
{
    Console.WriteLine(number);
}

Conceptos Avanzados de Arrays

Arrays Multidimensionales

C# soporta arrays multidimensionales, como los arrays 2D.

// Declarando un array 2D
int[,] matrix = new int[3, 3];

// Inicializando un array 2D
int[,] predefinedMatrix = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

// Accediendo a elementos en un array 2D
int element = matrix[1, 2]; // Accediendo al elemento en la fila 1, columna 2

Arrays Irregulares

Los arrays irregulares son arrays de arrays, donde cada array "interno" puede tener una longitud diferente.

// Declarando un array irregular
int[][] jaggedArray = new int[3][];

// Inicializando el array irregular
jaggedArray[0] = new int[] { 1, 2 };
jaggedArray[1] = new int[] { 3, 4, 5 };
jaggedArray[2] = new int[] { 6, 7, 8, 9 };

// Accediendo a elementos en un array irregular
int jaggedElement = jaggedArray[1][2]; // Accediendo al tercer elemento del segundo array

Métodos de Array

C# proporciona varios métodos integrados para trabajar con arrays, como Array.Sort, Array.Reverse y Array.Copy.

int[] numbers = { 5, 3, 8, 1, 2 };

// Ordenando el array
Array.Sort(numbers);

// Invirtiendo el array
Array.Reverse(numbers);

// Copiando el array
int[] copiedNumbers = new int[numbers.Length];
Array.Copy(numbers, copiedNumbers, numbers.Length);

Conclusión

Los arrays son una parte poderosa y esencial de la programación en C#. Te permiten almacenar y gestionar colecciones de datos de manera eficiente. Entender cómo crear, acceder y manipular arrays es crucial para cualquier programador de C#. A medida que avances, encontrarás estructuras de arrays más complejas como los arrays multidimensionales y los arrays irregulares, que ofrecen aún más flexibilidad para manejar datos.
