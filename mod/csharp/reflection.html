Reflexión en C#
Programación Dinámica ►

La reflexión es una característica poderosa en C# que te permite inspeccionar e interactuar con los metadatos de los tipos en tiempo de ejecución. Esto puede ser particularmente útil para tareas como la creación dinámica de tipos, la invocación de métodos y el acceso a atributos.

Conceptos Clave de la Reflexión

  • Metadatos: Información sobre los tipos definidos en un programa, como clases, interfaces y métodos.
  • Ensamblado: Una biblioteca de código compilado utilizada por aplicaciones .NET. La reflexión se puede usar para inspeccionar el contenido de un ensamblado.
  • Tipo: Representa declaraciones de tipos: tipos de clase, tipos de interfaz, tipos de matriz, tipos de valor, tipos de enumeración, parámetros de tipo y tipos de puntero.
  • MemberInfo: Proporciona información sobre los miembros de una clase, como métodos, propiedades, campos y eventos.

Uso de la Reflexión

Cargando un Ensamblado

Para usar la reflexión, a menudo comienzas cargando un ensamblado. Esto se puede hacer usando la clase Assembly.

using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        // Cargar el ensamblado actual
        Assembly assembly = Assembly.GetExecutingAssembly();
        Console.WriteLine("Nombre Completo del Ensamblado: " + assembly.FullName);
    }
}

Obteniendo Tipos de un Ensamblado

Una vez que tienes un ensamblado, puedes obtener los tipos definidos en él.

using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Assembly assembly = Assembly.GetExecutingAssembly();
        Type[] types = assembly.GetTypes();

        foreach (Type type in types)
        {
            Console.WriteLine("Tipo: " + type.Name);
        }
    }
}

Inspeccionando Miembros de Tipo

Puedes inspeccionar los miembros de un tipo, como sus métodos, propiedades y campos.

using System;
using System.Reflection;

class SampleClass
{
    public int SampleProperty { get; set; }
    public void SampleMethod() { }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);

        // Obtener propiedades
        PropertyInfo[] properties = type.GetProperties();
        foreach (PropertyInfo property in properties)
        {
            Console.WriteLine("Propiedad: " + property.Name);
        }

        // Obtener métodos
        MethodInfo[] methods = type.GetMethods();
        foreach (MethodInfo method in methods)
        {
            Console.WriteLine("Método: " + method.Name);
        }
    }
}

Invocando Métodos Dinámicamente

La reflexión te permite invocar métodos en tiempo de ejecución sin conocer sus nombres en tiempo de compilación.

using System;
using System.Reflection;

class SampleClass
{
    public void SampleMethod()
    {
        Console.WriteLine("¡SampleMethod invocado!");
    }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);
        MethodInfo method = type.GetMethod("SampleMethod");

        if (method != null)
        {
            object instance = Activator.CreateInstance(type);
            method.Invoke(instance, null);
        }
    }
}

Trabajando con Atributos

La reflexión se puede usar para leer atributos personalizados aplicados a tipos y miembros.

using System;
using System.Reflection;

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class SampleAttribute : Attribute
{
    public string Description { get; set; }
}

[Sample(Description = "Esta es una clase de ejemplo.")]
class SampleClass
{
    [Sample(Description = "Este es un método de ejemplo.")]
    public void SampleMethod() { }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);
        object[] classAttributes = type.GetCustomAttributes(typeof(SampleAttribute), false);

        foreach (SampleAttribute attr in classAttributes)
        {
            Console.WriteLine("Descripción del Atributo de la Clase: " + attr.Description);
        }

        MethodInfo method = type.GetMethod("SampleMethod");
        object[] methodAttributes = method.GetCustomAttributes(typeof(SampleAttribute), false);

        foreach (SampleAttribute attr in methodAttributes)
        {
            Console.WriteLine("Descripción del Atributo del Método: " + attr.Description);
        }
    }
}

Conclusión

La reflexión en C# es una herramienta versátil que te permite inspeccionar y manipular los metadatos de los tipos en tiempo de ejecución. Se puede usar para una variedad de tareas, incluida la creación dinámica de tipos, la invocación de métodos y la inspección de atributos. Aunque poderosa, la reflexión debe usarse con prudencia debido a su posible impacto en el rendimiento y la seguridad. Comprender los conceptos básicos de la reflexión puede mejorar enormemente tu capacidad para escribir aplicaciones C# flexibles y dinámicas.
