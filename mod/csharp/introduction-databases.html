Introducción a Bases de Datos
Conexión a una Base de Datos ►

Resumen

Las bases de datos son esenciales para almacenar, recuperar y gestionar datos en aplicaciones. Esta sección te introducirá a las bases de datos, cómo interactuar con ellas usando C#, y los conceptos básicos de SQL (Lenguaje de Consulta Estructurado).

Conceptos Clave

  • Base de Datos: Una colección organizada de datos, generalmente almacenados y accesados electrónicamente desde un sistema informático.
  • SQL: Un lenguaje estándar para acceder y manipular bases de datos.
  • RDBMS: Sistema de Gestión de Bases de Datos Relacionales, un tipo de sistema de gestión de bases de datos que almacena datos en tablas y permite relaciones entre tablas.
  • Operaciones CRUD: Las cuatro operaciones básicas para gestionar datos - Crear, Leer, Actualizar, Eliminar.

Configuración

Antes de sumergirte en el código, asegúrate de tener lo siguiente:

  • SQL Server: Instala SQL Server o usa un servicio basado en la nube como Azure SQL Database.
  • SQL Server Management Studio (SSMS): Una herramienta para gestionar bases de datos SQL Server.
  • Visual Studio: IDE para escribir código en C#.

Conectando a una Base de Datos

Para interactuar con una base de datos en C#, necesitas establecer una conexión usando la clase SqlConnection.

Ejemplo

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            try
            {
                connection.Open();
                Console.WriteLine("¡Conexión exitosa!");
            }
            catch (Exception ex)
            {
                Console.WriteLine("Conexión fallida: " + ex.Message);
            }
        }
    }
}

Explicación:

  • connectionString: Contiene los detalles necesarios para conectarse a la base de datos.
  • SqlConnection: Representa una conexión a una base de datos SQL Server.
  • Open(): Abre la conexión a la base de datos.
  • try-catch: Maneja cualquier excepción que ocurra durante el intento de conexión.

Operaciones CRUD

Crear

Para insertar datos en una base de datos, usa la declaración SQL INSERT INTO.

Ejemplo

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        string query = "INSERT INTO Users (Name, Age) VALUES (@Name, @Age)";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 30);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Filas afectadas: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicación:

  • SqlCommand: Representa una declaración SQL para ejecutar contra una base de datos SQL Server.
  • Parámetros: Usados para prevenir inyección SQL mediante la parametrización de la consulta.

Leer

Para recuperar datos, usa la declaración SQL SELECT.

Ejemplo

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        string query = "SELECT Name, Age FROM Users";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);

            try
            {
                connection.Open();
                SqlDataReader reader = command.ExecuteReader();
                while (reader.Read())
                {
                    Console.WriteLine($"Nombre: {reader["Name"]}, Edad: {reader["Age"]}");
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicación:

  • SqlDataReader: Proporciona una forma de leer una secuencia de filas de solo avance desde una base de datos SQL Server.

Actualizar

Para modificar datos existentes, usa la declaración SQL UPDATE.

Ejemplo

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        string query = "UPDATE Users SET Age = @Age WHERE Name = @Name";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 31);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Filas afectadas: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicación:

  • UPDATE: Modifica registros existentes en una tabla.

Eliminar

Para eliminar datos, usa la declaración SQL DELETE.

Ejemplo

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Filas afectadas: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicación:

  • DELETE: Elimina registros de una tabla.

Conclusión

En esta sección, aprendiste los conceptos básicos de bases de datos, cómo conectarte a una base de datos usando C#, y cómo realizar operaciones CRUD. Estas son habilidades fundamentales para cualquier desarrollador que trabaje con aplicaciones basadas en datos. Practica estos ejemplos e intenta crear tus propias interacciones con bases de datos para solidificar tu comprensión.
