Introducción a las Listas

Las listas en C# son parte del espacio de nombres System.Collections.Generic y proporcionan una forma de almacenar colecciones de objetos. A diferencia de los arrays, las listas pueden crecer y reducirse dinámicamente en tamaño.

  • Espacio de nombres: System.Collections.Generic
  • Clase: List<T>

Conceptos Clave

  • Tamaño Dinámico: Las listas pueden redimensionarse automáticamente a medida que se agregan o eliminan elementos.
  • Seguridad de Tipo: Las listas son fuertemente tipadas, lo que significa que solo puedes almacenar elementos de un tipo especificado.
  • Indexación Basada en Cero: Al igual que los arrays, las listas están indexadas desde cero.

Creación e Inicialización de Listas

Para crear una lista, necesitas especificar el tipo de elementos que almacenará.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creando una lista de enteros
        List<int> numbers = new List<int>();

        // Inicializando una lista con valores
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Imprimir la lista
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}

Explicación

  • List<int> numbers = new List<int>();: Crea una lista vacía de enteros.
  • List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };: Crea e inicializa una lista de cadenas con tres elementos.

Agregar y Eliminar Elementos

Puedes agregar elementos a una lista usando el método Add y eliminarlos usando los métodos Remove o RemoveAt.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 1, 2, 3 };

        // Agregar elementos
        numbers.Add(4);
        numbers.Add(5);

        // Eliminar elementos
        numbers.Remove(2); // Elimina la primera ocurrencia de 2
        numbers.RemoveAt(0); // Elimina el elemento en el índice 0

        // Imprimir la lista
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explicación

  • numbers.Add(4);: Agrega el elemento 4 al final de la lista.
  • numbers.Remove(2);: Elimina la primera ocurrencia del elemento 2.
  • numbers.RemoveAt(0);: Elimina el elemento en el índice 0.

Acceso a Elementos

Puedes acceder a los elementos en una lista usando el indexador.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Acceso a elementos
        string firstFruit = fruits[0];
        string secondFruit = fruits[1];

        Console.WriteLine(firstFruit); // Salida: Apple
        Console.WriteLine(secondFruit); // Salida: Banana
    }
}

Explicación

  • fruits[0]: Accede al primer elemento en la lista.
  • fruits[1]: Accede al segundo elemento en la lista.

Iterar Sobre una Lista

Puedes iterar sobre una lista usando un bucle foreach o un bucle for.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Usando bucle foreach
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }

        // Usando bucle for
        for (int i = 0; i < fruits.Count; i++)
        {
            Console.WriteLine(fruits[i]);
        }
    }
}

Explicación

  • foreach (string fruit in fruits): Itera sobre cada elemento en la lista.
  • for (int i = 0; i < fruits.Count; i++): Itera sobre la lista usando un índice.

Operaciones Avanzadas

Ordenar una Lista

Puedes ordenar una lista usando el método Sort.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 3, 1, 4, 1, 5, 9 };

        // Ordenar la lista
        numbers.Sort();

        // Imprimir la lista ordenada
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explicación

  • numbers.Sort();: Ordena la lista en orden ascendente.

Encontrar Elementos

Puedes encontrar elementos en una lista usando métodos como Contains, Find y FindAll.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Verificar si la lista contiene un elemento específico
        bool hasApple = fruits.Contains("Apple");

        // Encontrar el primer elemento que cumpla una condición
        string fruitWithA = fruits.Find(f => f.Contains("a"));

        // Encontrar todos los elementos que cumplan una condición
        List<string> fruitsWithA = fruits.FindAll(f => f.Contains("a"));

        Console.WriteLine(hasApple); // Salida: True
        Console.WriteLine(fruitWithA); // Salida: Banana
        foreach (string fruit in fruitsWithA)
        {
            Console.WriteLine(fruit); // Salida: Banana, Cherry
        }
    }
}

Explicación

  • fruits.Contains("Apple");: Verifica si "Apple" está en la lista.
  • fruits.Find(f => f.Contains("a"));: Encuentra el primer elemento que contiene la letra "a".
  • fruits.FindAll(f => f.Contains("a"));: Encuentra todos los elementos que contienen la letra "a".

Conclusión

Las listas en C# son una forma versátil y poderosa de gestionar colecciones de objetos. Ofrecen tamaño dinámico, seguridad de tipo y una variedad de métodos para agregar, eliminar, acceder y manipular elementos. Al comprender y utilizar listas, puedes escribir programas en C# más eficientes y flexibles.
