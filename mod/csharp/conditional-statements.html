Declaraciones condicionales en C#
Bucles ►

Las declaraciones condicionales son fundamentales en la programación, ya que te permiten tomar decisiones basadas en ciertas condiciones. En C#, las declaraciones condicionales incluyen if, else if, else, switch y operadores ternarios. Esta sección te guiará desde lo básico hasta el uso más avanzado de estas declaraciones.

Declaraciones If

La declaración if es la forma más básica de declaración condicional. Evalúa una expresión booleana y ejecuta un bloque de código si la expresión es verdadera.

Sintaxis

if (condition)
{
    // Código a ejecutar si la condición es verdadera
}

Ejemplo

int number = 10;
if (number > 5)
{
    Console.WriteLine("El número es mayor que 5.");
}

Explicación

  • condition: Una expresión booleana que se evalúa como true o false.
  • Si number > 5 es verdadero, se imprimirá el mensaje "El número es mayor que 5."

Declaraciones If-Else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

Sintaxis

if (condition)
{
    // Código a ejecutar si la condición es verdadera
}
else
{
    // Código a ejecutar si la condición es falsa
}

Ejemplo

int number = 3;
if (number > 5)
{
    Console.WriteLine("El número es mayor que 5.");
}
else
{
    Console.WriteLine("El número no es mayor que 5.");
}

Explicación

  • Si number > 5 es falso, se imprimirá el mensaje "El número no es mayor que 5."

Declaraciones Else If

La declaración else if te permite verificar múltiples condiciones.

Sintaxis

if (condition1)
{
    // Código a ejecutar si condition1 es verdadera
}
else if (condition2)
{
    // Código a ejecutar si condition2 es verdadera
}
else
{
    // Código a ejecutar si ninguna de las condiciones anteriores es verdadera
}

Ejemplo

int number = 7;
if (number > 10)
{
    Console.WriteLine("El número es mayor que 10.");
}
else if (number > 5)
{
    Console.WriteLine("El número es mayor que 5 pero menor o igual a 10.");
}
else
{
    Console.WriteLine("El número es 5 o menos.");
}

Explicación

  • Las condiciones se verifican en orden. Se ejecuta el bloque de código de la primera condición verdadera.
  • Si number > 10 es falso pero number > 5 es verdadero, se imprimirá el mensaje "El número es mayor que 5 pero menor o igual a 10."

Declaraciones Switch

La declaración switch se usa para seleccionar uno de varios bloques de código a ejecutar.

Sintaxis

switch (expression)
{
    case value1:
        // Código a ejecutar si expression == value1
        break;
    case value2:
        // Código a ejecutar si expression == value2
        break;
    // Más casos...
    default:
        // Código a ejecutar si ninguno de los casos coincide
        break;
}

Ejemplo

int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Lunes");
        break;
    case 2:
        Console.WriteLine("Martes");
        break;
    case 3:
        Console.WriteLine("Miércoles");
        break;
    default:
        Console.WriteLine("Otro día");
        break;
}

Explicación

  • La declaración switch evalúa la expression y ejecuta el bloque case correspondiente.
  • Si day es 3, se imprimirá el mensaje "Miércoles."

Operador Ternario

El operador ternario es una forma abreviada de las declaraciones if-else.

Sintaxis

condition ? expression1 : expression2;

Ejemplo

int number = 8;
string result = (number > 5) ? "Mayor que 5" : "5 o menos";
Console.WriteLine(result);

Explicación

  • Si number > 5 es verdadero, result será "Mayor que 5".
  • De lo contrario, result será "5 o menos".

Conclusión

Las declaraciones condicionales son esenciales para controlar el flujo de un programa basado en diferentes condiciones. Entender y usar if, else if, else, switch y el operador ternario te permitirá escribir código más dinámico y receptivo. Practica estos conceptos con varios ejemplos para volverte competente en tomar decisiones dentro de tus programas en C#.
