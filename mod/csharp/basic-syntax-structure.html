Introducción

C# es un lenguaje de programación moderno y orientado a objetos desarrollado por Microsoft. Es ampliamente utilizado para desarrollar aplicaciones de escritorio, aplicaciones web y juegos. Entender la sintaxis y estructura básica de C# es esencial para escribir código eficiente y sin errores.

Conceptos Clave

1. Programa Hola Mundo

La mejor manera de comenzar a aprender cualquier lenguaje de programación es escribiendo un simple programa "¡Hola, Mundo!". Este programa te introducirá a la estructura básica de un programa en C#.

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("¡Hola, Mundo!");
        }
    }
}
  • using System;: Esta línea importa el espacio de nombres System, que contiene clases fundamentales para operaciones básicas de entrada y salida.
  • namespace HelloWorld: Los espacios de nombres se utilizan para organizar el código y prevenir conflictos de nombres.
  • class Program: Las clases son los bloques de construcción de los programas en C#.
  • static void Main(string[] args): El método Main es el punto de entrada de una aplicación en C#. Es donde el programa comienza su ejecución.
  • Console.WriteLine("¡Hola, Mundo!");: Esta línea imprime "¡Hola, Mundo!" en la consola.

2. Variables y Tipos de Datos

Las variables se utilizan para almacenar datos, y los tipos de datos especifican el tipo de datos que se pueden almacenar en una variable.

Tipos de Datos Comunes

| Tipo de Dato | Descripción | Ejemplo | |--------------|-----------------------------|---------------| | int | Números enteros | 1, 2, 3 | | double | Números de punto flotante | 1.23, 4.56 | | char | Carácter único | 'a', 'b' | | string | Secuencia de caracteres | "Hola" | | bool | Valores booleanos (true/false) | true, false |

Ejemplo

int edad = 25;
double salario = 50000.50;
char calificación = 'A';
string nombre = "Juan";
bool estáEmpleado = true;

3. Operadores

Los operadores se utilizan para realizar operaciones en variables y valores.

Operadores Comunes

  • Operadores Aritméticos: +, -, *, /, %
  • Operadores de Comparación: ==, !=, >, <, >=, <=
  • Operadores Lógicos: &&, ||, !

Ejemplo

int a = 10;
int b = 20;
int suma = a + b; // 30
bool esIgual = (a == b); // false
bool esMayor = (a > b); // false
bool lógicoY = (a < b && b > 15); // true

4. Estructuras de Control

Las estructuras de control se utilizan para controlar el flujo de ejecución en un programa.

Sentencia If-Else

int número = 10;
if (número > 0)
{
    Console.WriteLine("Número positivo");
}
else
{
    Console.WriteLine("Número no positivo");
}

Sentencia Switch

int día = 3;
switch (día)
{
    case 1:
        Console.WriteLine("Lunes");
        break;
    case 2:
        Console.WriteLine("Martes");
        break;
    case 3:
        Console.WriteLine("Miércoles");
        break;
    default:
        Console.WriteLine("Otro día");
        break;
}

Bucles

  • Bucle For
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}
  • Bucle While
int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}
  • Bucle Do-While
int i = 0;
do
{
    Console.WriteLine(i);
    i++;
} while (i < 5);

5. Métodos

Los métodos son bloques de código que realizan una tarea específica y pueden ser llamados cuando se necesiten.

Ejemplo

class Program
{
    static void Main(string[] args)
    {
        Saludar("Juan");
    }

    static void Saludar(string nombre)
    {
        Console.WriteLine("Hola, " + nombre);
    }
}

Conclusión

Entender la sintaxis y estructura básica de C# es crucial para cualquier principiante. Esto incluye saber cómo escribir un programa simple, declarar variables, usar operadores, controlar el flujo del programa con estructuras de control y definir métodos. El dominio de estos conceptos básicos proporcionará una base sólida para temas más avanzados en C#.
