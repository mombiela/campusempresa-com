Introducción a ASP.NET
Construcción de una Aplicación Web Simple ►

ASP.NET es un potente framework para construir aplicaciones web utilizando C# y el ecosistema .NET. Esta sección te introducirá a los conceptos básicos de ASP.NET, sus conceptos centrales y cómo empezar a construir tu primera aplicación web.

¿Qué es ASP.NET?

ASP.NET es un framework de aplicaciones web del lado del servidor, de código abierto, diseñado para el desarrollo web para producir páginas web dinámicas. Fue desarrollado por Microsoft para permitir a los programadores construir sitios web, aplicaciones y servicios dinámicos.

Características Clave de ASP.NET

  • Multiplataforma: ASP.NET Core es multiplataforma, lo que significa que se ejecuta en Windows, macOS y Linux.
  • Alto Rendimiento: Optimizado para el rendimiento, lo que lo hace adecuado para aplicaciones de alto tráfico.
  • MVC y Web API Unificados: Combina las características de MVC (Model-View-Controller) y Web API en un solo framework.
  • Inyección de Dependencias: Soporte incorporado para la inyección de dependencias, lo que facilita la gestión de dependencias.
  • Razor Pages: Simplifica el desarrollo de escenarios enfocados en páginas utilizando un modelo de codificación basado en páginas.

Configurando Tu Entorno de Desarrollo

Antes de empezar a desarrollar con ASP.NET, necesitas configurar tu entorno de desarrollo.

Prerrequisitos

  • Visual Studio: Descarga e instala Visual Studio (Community, Professional o Enterprise).
  • .NET SDK: Asegúrate de tener instalado el .NET SDK.

Creando Tu Primer Proyecto ASP.NET

  1. Abre Visual Studio.
  2. Crea un nuevo proyecto:
    • Selecciona "Create a new project".
    • Elige "ASP.NET Core Web Application" y haz clic en "Next".
  3. Configura tu nuevo proyecto:
    • Nombra tu proyecto (por ejemplo, "MyFirstAspNetApp").
    • Elige una ubicación para guardar tu proyecto.
    • Haz clic en "Create".
  4. Selecciona una plantilla:
    • Elige "Web Application (Model-View-Controller)".
    • Haz clic en "Create".

Entendiendo la Estructura del Proyecto

Cuando creas un nuevo proyecto ASP.NET, Visual Studio genera una estructura de proyecto predeterminada. Aquí están los componentes clave:

  • Controllers: Manejan las solicitudes HTTP entrantes y envían respuestas de vuelta al cliente.
  • Views: Definen la interfaz de usuario (UI) utilizando la sintaxis Razor.
  • Models: Representan los datos y la lógica de negocio.
  • wwwroot: Contiene archivos estáticos como CSS, JavaScript e imágenes.
  • appsettings.json: Archivo de configuración para los ajustes de la aplicación.

Ejemplo: Controlador Básico

using Microsoft.AspNetCore.Mvc;

namespace MyFirstAspNetApp.Controllers
{
    public class HomeController : Controller
    {
        public IActionResult Index()
        {
            return View();
        }
    }
}

En este ejemplo, la clase HomeController hereda de Controller y define una acción Index que devuelve una vista.

Ejemplo: Vista Básica

Crea un nuevo archivo llamado Index.cshtml en la carpeta Views/Home:

@{
    ViewData["Title"] = "Home Page";
}

<h1>Welcome to ASP.NET!</h1>
<p>This is your first ASP.NET application.</p>

Esta vista Razor muestra un mensaje de bienvenida simple.

Ejecutando Tu Aplicación

Para ejecutar tu aplicación:

  1. Presiona F5 o haz clic en el botón "Run" en Visual Studio.
  2. Tu navegador web se abrirá y deberías ver el mensaje de bienvenida definido en tu vista Index.cshtml.

Resumen

En esta introducción a ASP.NET, aprendiste:

  • Qué es ASP.NET y sus características clave.
  • Cómo configurar tu entorno de desarrollo.
  • Cómo crear un nuevo proyecto ASP.NET en Visual Studio.
  • La estructura básica de un proyecto ASP.NET.
  • Cómo crear un controlador y una vista simples.

Al entender estos conceptos fundamentales, ahora estás listo para profundizar en ASP.NET y construir aplicaciones web más complejas.
