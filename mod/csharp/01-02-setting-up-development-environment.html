Configuración del Entorno de Desarrollo
Siguiente ►

En esta sección, aprenderás a configurar tu entorno de desarrollo para programar en C#. Este es un paso crucial para comenzar a escribir y ejecutar tus programas. Vamos a utilizar Visual Studio, que es uno de los entornos de desarrollo integrado (IDE) más populares para C#. También exploraremos Visual Studio Code como una alternativa ligera.

1. Instalación de Visual Studio

Paso 1: Descargar Visual Studio

  1. Visita el sitio web oficial de Visual Studio: Visual Studio.
  2. Haz clic en el botón "Descargar Visual Studio" y selecciona la versión "Community", que es gratuita.

Paso 2: Instalar Visual Studio

  1. Ejecuta el instalador descargado.
  2. En el instalador de Visual Studio, selecciona la carga de trabajo "Desarrollo de escritorio con .NET". Esta opción incluye todas las herramientas necesarias para desarrollar aplicaciones en C#.
  3. Haz clic en "Instalar" y espera a que se complete la instalación.

Paso 3: Configurar Visual Studio

  1. Una vez instalado, abre Visual Studio.
  2. Inicia sesión con tu cuenta de Microsoft (puedes crear una cuenta gratuita si no tienes una).
  3. Selecciona un tema de color para tu IDE y haz clic en "Iniciar Visual Studio".

2. Instalación de Visual Studio Code (Alternativa)

Paso 1: Descargar Visual Studio Code

  1. Visita el sitio web oficial de Visual Studio Code: Visual Studio Code.
  2. Haz clic en el botón "Download" y selecciona la versión adecuada para tu sistema operativo.

Paso 2: Instalar Visual Studio Code

  1. Ejecuta el instalador descargado y sigue las instrucciones en pantalla para completar la instalación.

Paso 3: Configurar Visual Studio Code

  1. Abre Visual Studio Code.
  2. Instala la extensión "C# for Visual Studio Code" desde el Marketplace de extensiones:
    • Haz clic en el icono de extensiones en la barra lateral izquierda.
    • Busca "C#" y selecciona "C# for Visual Studio Code".
    • Haz clic en "Instalar".

3. Instalación del SDK de .NET

Paso 1: Descargar el SDK de .NET

  1. Visita el sitio web oficial de .NET: .NET.
  2. Descarga el SDK de .NET más reciente.

Paso 2: Instalar el SDK de .NET

  1. Ejecuta el instalador descargado y sigue las instrucciones en pantalla para completar la instalación.

Paso 3: Verificar la Instalación

  1. Abre una terminal o línea de comandos.
  2. Escribe el siguiente comando para verificar que el SDK de .NET se ha instalado correctamente: 
    dotnet --version
    Deberías ver la versión del SDK de .NET instalada.

4. Primer Proyecto en Visual Studio

Crear un Nuevo Proyecto

  1. Abre Visual Studio.
  2. Haz clic en "Crear un nuevo proyecto".
  3. Selecciona "Aplicación de consola (.NET Core)" y haz clic en "Siguiente".
  4. Asigna un nombre a tu proyecto y selecciona una ubicación para guardarlo.
  5. Haz clic en "Crear".

Ejecutar el Proyecto

  1. Una vez creado el proyecto, Visual Studio abrirá el archivo Program.cs por defecto.
  2. Haz clic en el botón "Iniciar" (o presiona F5) para compilar y ejecutar el proyecto.
  3. Deberías ver la salida "Hello World!" en la consola.

5. Primer Proyecto en Visual Studio Code

Crear un Nuevo Proyecto

  1. Abre Visual Studio Code.
  2. Abre una terminal integrada (Ctrl + ).
  3. Navega a la carpeta donde deseas crear tu proyecto.
  4. Ejecuta el siguiente comando para crear un nuevo proyecto de consola: 
    dotnet new console -o MiPrimerProyecto
  5. Abre la carpeta del proyecto en Visual Studio Code: 
    code MiPrimerProyecto

Ejecutar el Proyecto

  1. En la terminal integrada, navega a la carpeta del proyecto: 
    cd MiPrimerProyecto
  2. Ejecuta el siguiente comando para compilar y ejecutar el proyecto: 
    dotnet run
  3. Deberías ver la salida "Hello World!" en la consola.

Conclusión

En esta sección, has aprendido a configurar tu entorno de desarrollo para programar en C#. Has instalado Visual Studio y Visual Studio Code, configurado el SDK de .NET y creado tu primer proyecto de consola. Ahora estás listo para comenzar a escribir y ejecutar programas en C#. En la siguiente sección, exploraremos la estructura básica de un programa en C# y escribiremos nuestro primer programa "Hola Mundo".
