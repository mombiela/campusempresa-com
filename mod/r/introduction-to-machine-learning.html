Resumen

El Aprendizaje Automático (ML) es una subcategoría de la inteligencia artificial que implica el uso de algoritmos y modelos estadísticos para permitir que las computadoras realicen tareas sin instrucciones explícitas. Este curso te guiará a través de los conceptos básicos del aprendizaje automático utilizando R, un lenguaje poderoso para la computación estadística y gráficos.

¿Qué es el Aprendizaje Automático?

  • Definición: El Aprendizaje Automático es el estudio de algoritmos que mejoran automáticamente a través de la experiencia.
  • Tipos de Aprendizaje Automático:
    • Aprendizaje Supervisado: Aprender a partir de datos etiquetados (por ejemplo, clasificación, regresión).
    • Aprendizaje No Supervisado: Aprender a partir de datos no etiquetados (por ejemplo, agrupamiento, asociación).
    • Aprendizaje por Refuerzo: Aprender interactuando con un entorno para maximizar alguna noción de recompensa acumulativa.

Configuración de R para el Aprendizaje Automático

Antes de sumergirte en el aprendizaje automático, asegúrate de tener R y RStudio instalados. También necesitarás instalar algunos paquetes esenciales.

Instalación de R y RStudio

  1. Descarga e instala R desde CRAN.
  2. Descarga e instala RStudio desde RStudio.

Instalación de Paquetes Esenciales

install.packages(c("tidyverse", "caret", "randomForest", "e1071"))

Aprendizaje Supervisado

El aprendizaje supervisado implica entrenar un modelo con datos etiquetados. Cubriremos dos tipos principales: clasificación y regresión.

Clasificación

Ejemplo: Predicción de Especies de Flores de Iris

# Cargar el conjunto de datos
data(iris)

# Dividir los datos en conjuntos de entrenamiento y prueba
set.seed(123)
training_indices <- sample(1:nrow(iris), 0.7 * nrow(iris))
train_data <- iris[training_indices, ]
test_data <- iris[-training_indices, ]

# Entrenar un modelo de bosque aleatorio
library(randomForest)
model <- randomForest(Species ~ ., data = train_data)

# Hacer predicciones
predictions <- predict(model, test_data)

# Evaluar el modelo
confusion_matrix <- table(predictions, test_data$Species)
print(confusion_matrix)

Regresión

Ejemplo: Predicción de Precios de Casas

# Cargar el conjunto de datos
data <- read.csv("path/to/house_prices.csv")

# Dividir los datos en conjuntos de entrenamiento y prueba
set.seed(123)
training_indices <- sample(1:nrow(data), 0.7 * nrow(data))
train_data <- data[training_indices, ]
test_data <- data[-training_indices, ]

# Entrenar un modelo de regresión lineal
model <- lm(price ~ ., data = train_data)

# Hacer predicciones
predictions <- predict(model, test_data)

# Evaluar el modelo
mse <- mean((predictions - test_data$price)^2)
print(mse)

Aprendizaje No Supervisado

El aprendizaje no supervisado implica encontrar patrones ocultos en datos no etiquetados. Cubriremos agrupamiento y reducción de dimensionalidad.

Agrupamiento

Ejemplo: Agrupamiento K-means

# Cargar el conjunto de datos
data <- read.csv("path/to/your/dataset.csv")

# Realizar agrupamiento K-means
set.seed(123)
clusters <- kmeans(data, centers = 3)

# Agregar asignaciones de clúster a los datos
data$cluster <- clusters$cluster

# Visualizar los clústeres
library(ggplot2)
ggplot(data, aes(x = feature1, y = feature2, color = as.factor(cluster))) +
  geom_point() +
  labs(color = "Clúster")

Reducción de Dimensionalidad

Ejemplo: Análisis de Componentes Principales (PCA)

# Cargar el conjunto de datos
data <- read.csv("path/to/your/dataset.csv")

# Realizar PCA
pca <- prcomp(data, scale. = TRUE)

# Visualizar los dos primeros componentes principales
pca_data <- data.frame(pca$x)
ggplot(pca_data, aes(x = PC1, y = PC2)) +
  geom_point()
