Introducción a las APIs y JSON

Las APIs (Interfaces de Programación de Aplicaciones) permiten que diferentes aplicaciones de software se comuniquen entre sí. JSON (Notación de Objetos de JavaScript) es un formato ligero de intercambio de datos que es fácil de leer y escribir para los humanos, y fácil de analizar y generar para las máquinas.

  • API: Un conjunto de reglas y protocolos para construir e interactuar con aplicaciones de software.
  • JSON: Un formato para estructurar datos, a menudo utilizado en APIs para intercambiar información.

Realizando Solicitudes API en R

Para interactuar con APIs en R, normalmente se utiliza el paquete httr, que proporciona funciones para enviar solicitudes HTTP y manejar respuestas.

Instalación y Carga del Paquete `httr`

install.packages("httr")
library(httr)

Realizando una Solicitud GET

Una solicitud GET recupera datos de un punto final de API.

# Ejemplo: Obteniendo datos de una API pública
url <- "https://api.example.com/data"
response <- GET(url)

# Verificar el estado de la respuesta
status_code(response)

Manejo de la Respuesta

Después de realizar una solicitud, necesitas manejar la respuesta, que a menudo está en formato JSON.

# Convertir la respuesta a texto
response_text <- content(response, "text")

# Analizar los datos JSON
library(jsonlite)
data <- fromJSON(response_text)

# Ver los datos
print(data)

Trabajando con Datos JSON

El paquete jsonlite en R se utiliza comúnmente para analizar datos JSON.

Instalación y Carga del Paquete `jsonlite`

install.packages("jsonlite")
library(jsonlite)

Analizando Datos JSON

# Datos JSON de ejemplo
json_data <- '{"name": "John", "age": 30, "city": "New York"}'

# Analizar JSON a una lista
parsed_data <- fromJSON(json_data)

# Accediendo a elementos
name <- parsed_data$name
age <- parsed_data$age
city <- parsed_data$city

print(paste("Nombre:", name))
print(paste("Edad:", age))
print(paste("Ciudad:", city))

Convirtiendo Objetos R a JSON

También puedes convertir objetos R a formato JSON usando la función toJSON.

# Crear una lista en R
r_list <- list(name = "Jane", age = 25, city = "Los Angeles")

# Convertir la lista a JSON
json_output <- toJSON(r_list, pretty = TRUE)

print(json_output)

Interacciones Avanzadas con APIs

Para interacciones más complejas, como solicitudes POST o manejo de autenticación, se requieren pasos adicionales.

Realizando una Solicitud POST

Una solicitud POST envía datos a un punto final de API.

# Ejemplo: Enviando datos a una API
url <- "https://api.example.com/submit"
body <- list(name = "Jane", age = 25, city = "Los Angeles")

response <- POST(url, body = body, encode = "json")

# Verificar el estado de la respuesta
status_code(response)

Manejo de Autenticación

Algunas APIs requieren autenticación, que puede manejarse usando encabezados.

# Ejemplo: Realizando una solicitud autenticada
url <- "https://api.example.com/protected"
api_key <- "your_api_key"

response <- GET(url, add_headers(Authorization = paste("Bearer", api_key)))

# Verificar el estado de la respuesta
status_code(response)

Resumen

En este tema, cubrimos los conceptos básicos de la interacción con APIs y el manejo de datos JSON en R. Aprendiste cómo:

  • Realizar solicitudes GET y POST usando el paquete httr.
  • Analizar y generar datos JSON usando el paquete jsonlite.
  • Manejar la autenticación para solicitudes API.

Entender estos conceptos te permite integrar R con varios servicios web y APIs, permitiéndote obtener y manipular datos de fuentes externas de manera eficiente.
