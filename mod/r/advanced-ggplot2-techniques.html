Introducción

En esta sección, profundizaremos en técnicas avanzadas para crear gráficos sofisticados y visualmente atractivos utilizando ggplot2 en R. Esto se basará en tu conocimiento fundamental y te ayudará a crear visualizaciones más complejas y personalizadas.

Personalización de la apariencia del gráfico

Temas

  • Temas te permiten controlar los elementos no relacionados con los datos de tus gráficos, como el color de fondo, las líneas de la cuadrícula y el texto.
  • ggplot2 viene con varios temas integrados como theme_minimal(), theme_classic() y theme_dark().
library(ggplot2)

# Gráfico básico
p <- ggplot(mpg, aes(x=displ, y=hwy)) + geom_point()

# Aplicando un tema
p + theme_minimal()

Temas personalizados

  • Puedes crear tu propio tema modificando los existentes o comenzando desde cero usando theme().
custom_theme <- theme(
  panel.background = element_rect(fill = "lightblue"),
  panel.grid.major = element_line(color = "white"),
  axis.text = element_text(size = 12, color = "blue"),
  axis.title = element_text(size = 14, face = "bold")
)

p + custom_theme

Geometrías avanzadas

Geom_Smooth

  • geom_smooth() agrega una media condicional suavizada, que es útil para visualizar tendencias.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE, color = "red")

Geom_Text y Geom_Label

  • geom_text() y geom_label() te permiten agregar anotaciones de texto a tus gráficos.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  geom_text(aes(label=manufacturer), vjust=-1, size=3)

Facetado

Facet Wrap

  • facet_wrap() crea paneles separados para cada nivel de un factor.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  facet_wrap(~ class)

Facet Grid

  • facet_grid() permite un facetado más complejo especificando filas y columnas.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  facet_grid(drv ~ cyl)

Sistemas de coordenadas

Coord_Flip

  • coord_flip() invierte los ejes x e y, lo cual es útil para diagramas de caja o gráficos de barras horizontales.
ggplot(mpg, aes(x=class, y=hwy)) +
  geom_boxplot() +
  coord_flip()

Coord_Polar

  • coord_polar() transforma el gráfico en un sistema de coordenadas polares, útil para crear gráficos de pastel.
ggplot(mpg, aes(x=factor(1), fill=class)) +
  geom_bar(width=1) +
  coord_polar(theta="y")

Anotaciones avanzadas

Agregar anotaciones personalizadas

  • Puedes agregar anotaciones personalizadas usando annotate().
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  annotate("text", x=6, y=40, label="Alta Eficiencia", color="red", size=5, angle=45)

Combinando múltiples gráficos

Usando Patchwork

  • El paquete patchwork te permite combinar múltiples gráficos de ggplot2 en uno solo.
library(patchwork)

p1 <- ggplot(mpg, aes(x=displ, y=hwy)) + geom_point()
p2 <- ggplot(mpg, aes(x=class, fill=class)) + geom_bar()

p1 + p2

Conclusión

En esta sección, exploramos técnicas avanzadas de ggplot2 para mejorar tus habilidades de visualización de datos en R. Cubrimos la personalización de la apariencia del gráfico, el uso de geometrías avanzadas, el facetado, los sistemas de coordenadas, la adición de anotaciones y la combinación de múltiples gráficos. Estas técnicas te ayudarán a crear gráficos más informativos y visualmente atractivos, haciendo tu análisis de datos más efectivo y atractivo.
