Introducción a las Listas

Las listas en R son una estructura de datos versátil y poderosa que puede contener elementos de diferentes tipos y longitudes. A diferencia de los vectores, que solo pueden contener elementos del mismo tipo, las listas pueden contener una mezcla de números, cadenas, vectores, otras listas y más.

Conceptos Clave

  • Definición: Una lista es una colección ordenada de elementos, donde cada elemento puede ser de un tipo diferente.
  • Creación: Las listas se crean usando la función list().
  • Acceso a Elementos: Los elementos en una lista pueden ser accedidos usando dobles corchetes [[ ]] o el operador $.
  • Modificación de Listas: Las listas pueden ser modificadas asignando nuevos valores a elementos específicos.
  • Listas Anidadas: Las listas pueden contener otras listas, lo que las convierte en una herramienta poderosa para estructuras de datos complejas.

Creación de Listas

Para crear una lista en R, usa la función list(). Aquí hay un ejemplo simple:

# Creando una lista con diferentes tipos de elementos
mi_lista <- list(nombre = "Juan", edad = 30, puntuaciones = c(85, 90, 88))

# Imprimiendo la lista
print(mi_lista)

Explicación

  • nombre = "Juan": Una cadena de caracteres.
  • edad = 30: Un valor numérico.
  • puntuaciones = c(85, 90, 88): Un vector numérico.

Acceso a Elementos de la Lista

Puedes acceder a los elementos en una lista usando dobles corchetes [[ ]] o el operador $.

Usando Dobles Corchetes

# Accediendo al elemento 'nombre'
nombre <- mi_lista[[1]]
print(nombre)

# Accediendo al elemento 'puntuaciones'
puntuaciones <- mi_lista[[3]]
print(puntuaciones)

Usando el Operador `$`

# Accediendo al elemento 'nombre'
nombre <- mi_lista$nombre
print(nombre)

# Accediendo al elemento 'puntuaciones'
puntuaciones <- mi_lista$puntuaciones
print(puntuaciones)

Modificación de Listas

Puedes modificar elementos en una lista asignando nuevos valores a elementos específicos.

# Modificando el elemento 'edad'
mi_lista$edad <- 31

# Añadiendo un nuevo elemento
mi_lista$ciudad <- "Nueva York"

# Imprimiendo la lista modificada
print(mi_lista)

Listas Anidadas

Las listas pueden contener otras listas, lo que permite la creación de estructuras de datos complejas.

# Creando una lista anidada
lista_anidada <- list(persona = mi_lista, trabajo = list(título = "Científico de Datos", salario = 90000))

# Accediendo a elementos en una lista anidada
título_trabajo <- lista_anidada$trabajo$título
print(título_trabajo)

Resumen

Las listas en R son una estructura de datos flexible y poderosa que puede contener elementos de diferentes tipos y longitudes. Se crean usando la función list() y pueden ser accedidas y modificadas usando dobles corchetes [[ ]] o el operador $. Las listas también pueden ser anidadas, permitiendo la creación de estructuras de datos complejas.

Al dominar las listas, podrás manejar una amplia variedad de tipos y estructuras de datos en R, haciendo tus tareas de análisis de datos y programación más eficientes y efectivas.
