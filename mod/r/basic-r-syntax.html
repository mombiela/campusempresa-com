Introducción

R es un lenguaje poderoso para la computación estadística y gráficos. Entender la sintaxis básica es crucial para escribir código R eficiente y legible. Esta sección cubrirá los elementos fundamentales de la sintaxis de R, incluyendo variables, tipos de datos, operadores y funciones básicas.

Variables y Asignación

Las variables en R se utilizan para almacenar datos que pueden ser referenciados y manipulados. El operador de asignación <- se usa comúnmente para asignar valores a las variables.

# Asignando un valor a una variable
x <- 10
y <- 5
  • Operador de Asignación: <- es preferido, pero = también puede ser usado.
  • Nombres de Variables: Deben comenzar con una letra y pueden contener letras, números y guiones bajos.

Tipos de Datos

R soporta varios tipos de datos, incluyendo numérico, entero, carácter, lógico y complejo.

# Numérico
num <- 3.14

# Entero
int <- 42L

# Carácter
char <- "¡Hola, R!"

# Lógico
bool <- TRUE

# Complejo
comp <- 2 + 3i

| Tipo de Dato | Ejemplo | Descripción | |--------------|---------------|-------------------------------| | Numérico | 3.14 | Números decimales | | Entero | 42L | Números enteros con sufijo L| | Carácter | "¡Hola, R!" | Cadenas de texto | | Lógico | TRUE | Valores booleanos (TRUE, FALSE) | | Complejo | 2 + 3i | Números complejos |

Operadores

Los operadores en R se utilizan para realizar operaciones aritméticas, de comparación y lógicas.

Operadores Aritméticos

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • ^ : Exponenciación
# Operaciones aritméticas
sum <- x + y
diff <- x - y
prod <- x * y
quot <- x / y
exp <- x ^ y

Operadores de Comparación

  • == : Igual a
  • != : No igual a
  • < : Menor que
  • > : Mayor que
  • <= : Menor o igual a
  • >= : Mayor o igual a
# Operaciones de comparación
is_equal <- x == y
is_not_equal <- x != y
is_less <- x < y
is_greater <- x > y

Operadores Lógicos

  • & : Y
  • | : O
  • ! : NO
# Operaciones lógicas
and_op <- (x > 5) & (y < 10)
or_op <- (x > 5) | (y < 10)
not_op <- !(x > 5)

Funciones Básicas

Las funciones se utilizan para realizar tareas específicas. R tiene muchas funciones integradas, y también puedes crear las tuyas propias.

Funciones Integradas

  • print(): Imprime salida en la consola.
  • sqrt(): Calcula la raíz cuadrada.
  • abs(): Calcula el valor absoluto.
# Usando funciones integradas
print("¡Hola, R!")
sqrt_val <- sqrt(16)
abs_val <- abs(-5)

Funciones Definidas por el Usuario

Puedes crear tus propias funciones usando la palabra clave function.

# Definiendo una función
add_numbers <- function(a, b) {
  result <- a + b
  return(result)
}

# Llamando a la función
sum_result <- add_numbers(3, 4)
print(sum_result)

Conclusión

Entender la sintaxis básica de R es el primer paso hacia el dominio del lenguaje. Esta sección cubrió lo esencial, incluyendo variables, tipos de datos, operadores y funciones básicas. Con estos fundamentos, puedes comenzar a escribir scripts simples en R y gradualmente avanzar hacia tareas más complejas. Practica estos conceptos con ejercicios para reforzar tu aprendizaje y construir una base sólida en la programación en R.
