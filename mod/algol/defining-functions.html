Introducción

En esta sección, exploraremos cómo definir y usar funciones en ALGOL, un lenguaje de programación de alto nivel que fue ampliamente utilizado en las décadas de 1960 y 1970. Entender las funciones es crucial ya que te permiten encapsular código en bloques reutilizables, haciendo tus programas más modulares y fáciles de gestionar.

Conceptos Clave

  • Definición de Función: Cómo declarar y definir una función en ALGOL.
  • Parámetros de Función: Pasar argumentos a las funciones.
  • Valores de Retorno: Cómo las funciones devuelven valores.
  • Ámbito y Duración: Entender el ámbito y la duración de las variables dentro de las funciones.

Definición de Función

En ALGOL, las funciones se definen usando la palabra clave procedure. Aquí está la sintaxis básica para definir una función:

procedure NombreDeLaFunción(parámetros);
begin
    // cuerpo de la función
end;

Ejemplo

Vamos a definir una función simple que suma dos números y devuelve el resultado:

procedure SumarNumeros(a, b);
    integer a, b;
begin
    SumarNumeros := a + b;
end;
  • procedure SumarNumeros(a, b); declara una función llamada SumarNumeros con dos parámetros a y b.
  • integer a, b; especifica que ambos parámetros son de tipo entero.
  • SumarNumeros := a + b; asigna la suma de a y b al nombre de la función, que es como ALGOL devuelve valores.

Parámetros de Función

Las funciones en ALGOL pueden tomar múltiples parámetros. Estos parámetros pueden ser de diferentes tipos, como enteros, números reales o arreglos.

Ejemplo

Aquí hay una función que calcula el promedio de tres números:

procedure Promedio(x, y, z);
    real x, y, z;
begin
    Promedio := (x + y + z) / 3;
end;
  • real x, y, z; especifica que los parámetros x, y y z son números reales (números de punto flotante).
  • Promedio := (x + y + z) / 3; calcula el promedio y lo asigna al nombre de la función.

Valores de Retorno

En ALGOL, el valor de retorno de una función se asigna al nombre de la función dentro del cuerpo de la función. Esto es diferente de muchos lenguajes modernos donde se usa la palabra clave return.

Ejemplo

Considera una función que devuelve el máximo de dos números:

procedure Maximo(a, b);
    integer a, b;
begin
    if a > b then
        Maximo := a
    else
        Maximo := b;
end;
  • La declaración if-else verifica cuál número es mayor y lo asigna a Maximo.

Ámbito y Duración

Las variables declaradas dentro de una función son locales a esa función. Se crean cuando la función es llamada y se destruyen cuando la función termina.

Ejemplo

procedure EjemploAmbitoLocal();
    integer varLocal;
begin
    varLocal := 10;
    // varLocal solo es accesible dentro de esta función
end;
  • varLocal es una variable local y no puede ser accedida fuera de EjemploAmbitoLocal.

Conclusión

Definir funciones en ALGOL implica entender la sintaxis para declarar funciones, pasar parámetros y devolver valores. Las funciones ayudan a crear código modular y reutilizable, lo cual es esencial para escribir programas eficientes. Al dominar estos conceptos, puedes escribir programas en ALGOL más organizados y mantenibles.
