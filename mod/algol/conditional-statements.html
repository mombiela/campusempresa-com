Introducción

Las declaraciones condicionales son fundamentales en la programación ya que permiten la ejecución de código basado en ciertas condiciones. En ALGOL, las declaraciones condicionales ayudan a controlar el flujo del programa, haciendo posible ejecutar diferentes bloques de código basados en criterios específicos.

Conceptos clave

  • Declaraciones condicionales: Instrucciones que ejecutan código solo si una condición especificada es verdadera.
  • Expresiones booleanas: Expresiones que se evalúan como verdaderas o falsas.
  • Declaraciones If: La principal declaración condicional en ALGOL.
  • Declaraciones Else: Usadas para ejecutar código cuando la condición if es falsa.
  • Declaraciones Else If: Usadas para verificar múltiples condiciones en secuencia.

Declaraciones If

La declaración if se usa para ejecutar un bloque de código si una condición especificada es verdadera.

Sintaxis

if condition then
    statement;

Ejemplo

begin
    integer x;
    x := 10;
    if x > 5 then
        print("x es mayor que 5");
end

En este ejemplo, el programa verifica si x es mayor que 5. Dado que x es 10, la condición es verdadera y se imprime el mensaje "x es mayor que 5".

Declaraciones If-Else

La declaración if-else permite ejecutar un bloque de código si la condición es verdadera y otro bloque si la condición es falsa.

Sintaxis

if condition then
    statement1
else
    statement2;

Ejemplo

begin
    integer x;
    x := 3;
    if x > 5 then
        print("x es mayor que 5")
    else
        print("x no es mayor que 5");
end

En este ejemplo, dado que x es 3, la condición x > 5 es falsa, por lo que se imprime el mensaje "x no es mayor que 5".

Declaraciones Else If

La declaración else if se usa para verificar múltiples condiciones en secuencia.

Sintaxis

if condition1 then
    statement1
else if condition2 then
    statement2
else
    statement3;

Ejemplo

begin
    integer x;
    x := 7;
    if x > 10 then
        print("x es mayor que 10")
    else if x > 5 then
        print("x es mayor que 5 pero menor o igual a 10")
    else
        print("x es 5 o menos");
end

En este ejemplo, dado que x es 7, la primera condición x > 10 es falsa, pero la segunda condición x > 5 es verdadera, por lo que se imprime el mensaje "x es mayor que 5 pero menor o igual a 10".

Declaraciones condicionales anidadas

Las declaraciones condicionales pueden anidarse entre sí para manejar lógica más compleja.

Ejemplo

begin
    integer x, y;
    x := 8;
    y := 3;
    if x > 5 then
        if y < 5 then
            print("x es mayor que 5 y y es menor que 5")
        else
            print("x es mayor que 5 y y es 5 o más")
    else
        print("x es 5 o menos");
end

En este ejemplo, el programa primero verifica si x es mayor que 5. Dado que x es 8, luego verifica si y es menor que 5. Ambas condiciones son verdaderas, por lo que se imprime el mensaje "x es mayor que 5 y y es menor que 5".

Conclusión

Las declaraciones condicionales en ALGOL son esenciales para controlar el flujo de un programa basado en condiciones específicas. Entender cómo usar las declaraciones if, else y else if, así como cómo anidarlas, te permite escribir programas más complejos y dinámicos. Practica estos conceptos con varias condiciones para volverte competente en el uso de declaraciones condicionales en ALGOL.
