Introducción a las Declaraciones Switch Case

Las declaraciones switch case en ALGOL se utilizan para ejecutar un bloque de código entre muchos basados en el valor de una expresión. Proporcionan una forma más legible y organizada de manejar múltiples ramas condicionales en comparación con el uso de múltiples declaraciones if-else.

Sintaxis Básica

La sintaxis básica de una declaración switch case en ALGOL es la siguiente:

switch expression
begin
    case value1: 
        // bloque de código para value1
    case value2: 
        // bloque de código para value2
    ...
    default: 
        // bloque de código por defecto si no coincide ningún caso
end switch
  • expression: El valor que se está evaluando.
  • case value: El valor específico para coincidir con la expresión.
  • default: El bloque de código a ejecutar si ningún caso coincide con la expresión.

Ejemplo: Switch Case Simple

Aquí hay un ejemplo simple para demostrar la declaración switch case en ALGOL:

begin
    integer x;
    x := 2;
    
    switch x
    begin
        case 1: 
            print("x es 1");
        case 2: 
            print("x es 2");
        case 3: 
            print("x es 3");
        default: 
            print("x no es 1, 2, o 3");
    end switch
end

Explicación

  • La variable x se asigna el valor 2.
  • La declaración switch evalúa el valor de x.
  • Dado que x es 2, se ejecuta el bloque de código bajo case 2, imprimiendo "x es 2".

Uso Avanzado

Las declaraciones switch case también pueden manejar escenarios más complejos, como casos switch anidados o el uso de expresiones en los casos.

Ejemplo de Switch Case Anidado

begin
    integer x, y;
    x := 1;
    y := 2;
    
    switch x
    begin
        case 1: 
            switch y
            begin
                case 1: 
                    print("x es 1 y y es 1");
                case 2: 
                    print("x es 1 y y es 2");
                default: 
                    print("x es 1 y y es otra cosa");
            end switch
        case 2: 
            print("x es 2");
        default: 
            print("x no es 1 o 2");
    end switch
end

Explicación

  • El switch externo evalúa x.
  • Dado que x es 1, entra en el switch anidado evaluando y.
  • Dado que y es 2, imprime "x es 1 y y es 2".

Comparación con Declaraciones If-Else

Las declaraciones switch case a menudo pueden ser más legibles y eficientes que múltiples declaraciones if-else. Aquí hay una comparación:

Ejemplo If-Else

begin
    integer x;
    x := 2;
    
    if x = 1 then
        print("x es 1")
    else if x = 2 then
        print("x es 2")
    else if x = 3 then
        print("x es 3")
    else
        print("x no es 1, 2, o 3")
end

Ejemplo Switch Case

begin
    integer x;
    x := 2;
    
    switch x
    begin
        case 1: 
            print("x es 1");
        case 2: 
            print("x es 2");
        case 3: 
            print("x es 3");
        default: 
            print("x no es 1, 2, o 3");
    end switch
end

Tabla Comparativa

| Característica | Declaraciones If-Else | Declaraciones Switch Case | |--------------------|-------------------------------------|-----------------------------------| | Legibilidad | Puede volverse desordenado con muchos casos | Más legible con múltiples casos | | Eficiencia | Puede evaluar múltiples condiciones | Evalúa la expresión una vez | | Caso de Uso | Condiciones simples o pocas | Múltiples condiciones discretas |

Conclusión

Las declaraciones switch case en ALGOL proporcionan una forma estructurada y legible de manejar múltiples ramas condicionales. Son particularmente útiles cuando se trata de múltiples valores discretos y pueden mejorar tanto la legibilidad como la eficiencia de tu código en comparación con el uso de múltiples declaraciones if-else. Al dominar las declaraciones switch case, puedes escribir código más organizado y mantenible.
