Introducción

Los métodos numéricos son técnicas utilizadas para resolver problemas numéricos, a menudo involucrando cálculos matemáticos complejos. Esta sección te guiará desde los conceptos básicos hasta los métodos numéricos avanzados utilizando ALGOL, un lenguaje de programación de alto nivel que fue ampliamente utilizado en las décadas de 1960 y 1970 para cálculos científicos.

Conceptos Básicos

¿Qué son los Métodos Numéricos?

  • Definición: Los métodos numéricos son algoritmos utilizados para resolver problemas matemáticos numéricamente en lugar de analíticamente.
  • Aplicaciones: Se utilizan en ingeniería, física, finanzas y otros campos que requieren cálculos precisos.

¿Por qué Usar ALGOL para Métodos Numéricos?

  • Significado Histórico: ALGOL (Lenguaje Algorítmico) fue diseñado para cálculos científicos y ha influido en muchos lenguajes de programación modernos.
  • Características: ALGOL soporta operaciones aritméticas complejas, programación estructurada y recursión, lo que lo hace adecuado para métodos numéricos.

Empezando con ALGOL

Sintaxis Básica

Antes de profundizar en los métodos numéricos, entendamos algo de la sintaxis básica de ALGOL.

begin
  integer i;
  real x;
  x := 10.5;
  for i := 1 step 1 until 10 do
    begin
      x := x + i;
      print(x);
    end;
end;
  • begin...end: Delimitadores de bloque.
  • integer, real: Tipos de datos.
  • :=: Operador de asignación.
  • for...do: Estructura de bucle.

Operaciones Aritméticas Simples

begin
  real a, b, c;
  a := 5.0;
  b := 3.0;
  c := a + b;
  print(c);  // Salida: 8.0
end;

Métodos Numéricos Intermedios

Algoritmos de Búsqueda de Raíces

Método de Bisección

El método de bisección es un algoritmo de búsqueda de raíces simple y robusto.

begin
  real a, b, c, fa, fb, fc;
  procedure f(x); real x; real f;
  begin
    f := x * x - 4.0;  // Función de ejemplo: f(x) = x^2 - 4
  end;
  
  a := 0.0;
  b := 3.0;
  fa := f(a);
  fb := f(b);
  
  while (b - a) > 0.001 do
    begin
      c := (a + b) / 2.0;
      fc := f(c);
      if fc = 0.0 then
        exit;
      if fa * fc < 0.0 then
        b := c
      else
        a := c;
    end;
  print(c);  // Salida: Raíz aproximada
end;

Integración Numérica

Regla del Trapecio

La regla del trapecio es un método simple para la integración numérica.

begin
  real a, b, h, sum;
  integer n, i;
  procedure f(x); real x; real f;
  begin
    f := x * x;  // Función de ejemplo: f(x) = x^2
  end;
  
  a := 0.0;
  b := 1.0;
  n := 100;
  h := (b - a) / n;
  sum := (f(a) + f(b)) / 2.0;
  
  for i := 1 step 1 until n - 1 do
    sum := sum + f(a + i * h);
  
  sum := sum * h;
  print(sum);  // Salida: Integral aproximada
end;

Métodos Numéricos Avanzados

Resolución de Ecuaciones Diferenciales

Método de Euler

El método de Euler es una técnica sencilla para resolver ecuaciones diferenciales ordinarias (EDOs).

begin
  real x, y, h;
  integer n, i;
  procedure f(x, y); real x, y; real f;
  begin
    f := y - x;  // EDO de ejemplo: dy/dx = y - x
  end;
  
  x := 0.0;
  y := 1.0;
  h := 0.1;
  n := 10;
  
  for i := 1 step 1 until n do
    begin
      y := y + h * f(x, y);
      x := x + h;
      print(x, y);
    end;
end;

Operaciones con Matrices

Eliminación Gaussiana

La eliminación gaussiana es un método para resolver sistemas de ecuaciones lineales.

begin
  real a[1:3, 1:4];
  integer i, j, k;
  
  // Matriz de ejemplo: matriz aumentada 3x4 para sistema de ecuaciones
  a[1,1] := 2.0; a[1,2] := 1.0; a[1,3] := -1.0; a[1,4] := 8.0;
  a[2,1] := -3.0; a[2,2] := -1.0; a[2,3] := 2.0; a[2,4] := -11.0;
  a[3,1] := -2.0; a[3,2] := 1.0; a[3,3] := 2.0; a[3,4] := -3.0;
  
  for i := 1 step 1 until 3 do
    begin
      for j := i + 1 step 1 until 3 do
        begin
          real factor;
          factor := a[j,i] / a[i,i];
          for k := i step 1 until 4 do
            a[j,k] := a[j,k] - factor * a[i,k];
        end;
    end;
  
  // Sustitución hacia atrás
  real x[1:3];
  for i := 3 step -1 until 1 do
    begin
      x[i] := a[i,4];
      for j := i + 1 step 1 until 3 do
        x[i] := x[i] - a[i,j] * x[j];
      x[i] := x[i] / a[i,i];
    end;
  
  for i := 1 step 1 until 3 do
    print(x[i]);  // Salida: Solución del sistema
end;

Conclusión

Los métodos numéricos son herramientas esenciales para resolver problemas matemáticos complejos, y ALGOL proporciona una plataforma robusta para implementar estos métodos. Desde operaciones aritméticas básicas hasta técnicas avanzadas como la resolución de ecuaciones diferenciales y sistemas de ecuaciones lineales, entender y aplicar métodos numéricos en ALGOL puede mejorar enormemente tus habilidades computacionales. Practica estos ejemplos y explora más para dominar los métodos numéricos en ALGOL.
