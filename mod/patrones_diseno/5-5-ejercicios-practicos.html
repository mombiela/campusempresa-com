Ejercicios Prácticos

En este módulo, pondremos en práctica los conocimientos adquiridos sobre patrones de diseño a través de una serie de ejercicios prácticos. Estos ejercicios están diseñados para reforzar tu comprensión y habilidad para aplicar los patrones de diseño en situaciones del mundo real.

Ejercicio 1: Implementación del Patrón Singleton

Descripción: Implementa el patrón Singleton en un sistema de registro de usuarios. El sistema debe asegurarse de que solo exista una instancia del registro de usuarios en toda la aplicación.

Requisitos:

  1. Crear una clase UserRegistry que siga el patrón Singleton.
  2. La clase debe tener un método addUser para agregar usuarios y un método getUser para obtener un usuario por su ID.

Ejemplo de Código:

class UserRegistry:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super(UserRegistry, cls).__new__(cls)
            cls._instance.users = {}
        return cls._instance

    def addUser(self, user_id, user_name):
        self.users[user_id] = user_name

    def getUser(self, user_id):
        return self.users.get(user_id)

# Uso del patrón Singleton
registry1 = UserRegistry()
registry2 = UserRegistry()

registry1.addUser(1, "Alice")
print(registry2.getUser(1))  # Output: Alice

print(registry1 is registry2)  # Output: True

Ejercicio 2: Uso del Patrón Factory Method

Descripción: Implementa el patrón Factory Method para crear diferentes tipos de documentos (por ejemplo, PDF, Word).

Requisitos:

  1. Crear una clase base Document con un método create.
  2. Crear clases derivadas PDFDocument y WordDocument que implementen el método create.
  3. Crear una clase DocumentFactory con un método getDocument que devuelva una instancia del tipo de documento solicitado.

Ejemplo de Código:

from abc import ABC, abstractmethod

class Document(ABC):
    @abstractmethod
    def create(self):
        pass

class PDFDocument(Document):
    def create(self):
        return "PDF Document Created"

class WordDocument(Document):
    def create(self):
        return "Word Document Created"

class DocumentFactory:
    def getDocument(self, doc_type):
        if doc_type == "PDF":
            return PDFDocument()
        elif doc_type == "Word":
            return WordDocument()
        else:
            return None

# Uso del patrón Factory Method
factory = DocumentFactory()

pdf_doc = factory.getDocument("PDF")
print(pdf_doc.create())  # Output: PDF Document Created

word_doc = factory.getDocument("Word")
print(word_doc.create())  # Output: Word Document Created

Ejercicio 3: Aplicación del Patrón Observer

Descripción: Implementa el patrón Observer en un sistema de notificaciones. Los usuarios deben poder suscribirse y recibir notificaciones cuando se publique un nuevo mensaje.

Requisitos:

  1. Crear una clase Subject que mantenga una lista de observadores y notifique a todos cuando se publique un nuevo mensaje.
  2. Crear una clase Observer con un método update que se llame cuando haya una nueva notificación.
  3. Implementar las clases User y NotificationSystem que sigan el patrón Observer.

Ejemplo de Código:

class Subject:
    def __init__(self):
        self._observers = []

    def attach(self, observer):
        self._observers.append(observer)

    def detach(self, observer):
        self._observers.remove(observer)

    def notify(self, message):
        for observer in self._observers:
            observer.update(message)

class Observer:
    def update(self, message):
        pass

class User(Observer):
    def __init__(self, name):
        self.name = name

    def update(self, message):
        print(f"{self.name} received message: {message}")

class NotificationSystem(Subject):
    def publish_message(self, message):
        self.notify(message)

# Uso del patrón Observer
notification_system = NotificationSystem()

user1 = User("Alice")
user2 = User("Bob")

notification_system.attach(user1)
notification_system.attach(user2)

notification_system.publish_message("New message available!")  
# Output: 
# Alice received message: New message available!
# Bob received message: New message available!

Ejercicio 4: Implementación del Patrón Decorator

Descripción: Implementa el patrón Decorator para añadir funcionalidades adicionales a un objeto de manera dinámica. En este caso, añadir diferentes tipos de bordes a una ventana gráfica.

Requisitos:

  1. Crear una clase base Window con un método draw.
  2. Crear una clase SimpleWindow que implemente Window.
  3. Crear una clase WindowDecorator que también implemente Window y que sirva de base para otros decoradores.
  4. Crear decoradores específicos como BorderDecorator y ScrollDecorator.

Ejemplo de Código:

class Window:
    def draw(self):
        pass

class SimpleWindow(Window):
    def draw(self):
        return "Drawing simple window"

class WindowDecorator(Window):
    def __init__(self, window):
        self._window = window

    def draw(self):
        return self._window.draw()

class BorderDecorator(WindowDecorator):
    def draw(self):
        return f"{self._window.draw()} with border"

class ScrollDecorator(WindowDecorator):
    def draw(self):
        return f"{self._window.draw()} with scroll"

# Uso del patrón Decorator
simple_window = SimpleWindow()
decorated_window = BorderDecorator(ScrollDecorator(simple_window))

print(decorated_window.draw())  
# Output: Drawing simple window with scroll with border
