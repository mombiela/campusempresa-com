Introducción

Los operadores de comparación en Ruby se utilizan para comparar dos valores. Devuelven un valor booleano (true o false) basado en la comparación. Entender estos operadores es crucial para controlar el flujo de tus programas a través de declaraciones condicionales y bucles.

Principales Operadores de Comparación

Operadores de Igualdad

  • ==: Verifica si dos valores son iguales.
  • !=: Verifica si dos valores no son iguales.
a = 5
b = 5
c = 10

puts a == b  # Salida: true
puts a != c  # Salida: true

Operadores Relacionales

  • >: Mayor que.
  • <: Menor que.
  • >=: Mayor o igual que.
  • <=: Menor o igual que.
x = 7
y = 10

puts x > y   # Salida: false
puts x < y   # Salida: true
puts x >= 7  # Salida: true
puts y <= 10 # Salida: true

Operador de Comparación Combinada

  • <=>: También conocido como el operador de nave espacial, devuelve:
    • -1 si el operando izquierdo es menor que el operando derecho.
    • 0 si el operando izquierdo es igual al operando derecho.
    • 1 si el operando izquierdo es mayor que el operando derecho.
puts 5 <=> 10  # Salida: -1
puts 10 <=> 10 # Salida: 0
puts 15 <=> 10 # Salida: 1

Operador de Igualdad de Caso

  • ===: Usado en declaraciones case para probar la igualdad.
range = (1..10)

puts range === 5  # Salida: true
puts range === 15 # Salida: false

Identidad de Objeto

  • equal?: Verifica si dos objetos son el mismo objeto.
str1 = "hello"
str2 = "hello"
str3 = str1

puts str1.equal?(str2) # Salida: false
puts str1.equal?(str3) # Salida: true

Ejemplos Prácticos

Usando Operadores de Comparación en Declaraciones Condicionales

age = 18

if age >= 18
  puts "Eres elegible para votar."
else
  puts "No eres elegible para votar."
end

Usando el Operador de Nave Espacial para Ordenar

numbers = [5, 2, 9, 3, 8]

sorted_numbers = numbers.sort { |a, b| a <=> b }
puts sorted_numbers.inspect  # Salida: [2, 3, 5, 8, 9]

Usando Igualdad de Caso en Declaraciones Case

grade = 'B'

case grade
when 'A'
  puts "¡Excelente!"
when 'B'
  puts "¡Buen trabajo!"
when 'C'
  puts "¡Bien hecho!"
else
  puts "¡Sigue intentando!"
end

Resumen

Los operadores de comparación son fundamentales en Ruby para tomar decisiones y controlar el flujo de tus programas. Te permiten comparar valores para igualdad, orden relacional e identidad de objeto. Entender cómo usar estos operadores efectivamente te permitirá escribir código más dinámico y receptivo.

Al dominar estos operadores, puedes manejar una amplia gama de tareas de programación, desde simples verificaciones condicionales hasta algoritmos de ordenación complejos. Practica usando estos operadores en diferentes escenarios para volverte competente en su aplicación.
