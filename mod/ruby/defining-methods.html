Introducción a los Métodos

Los métodos en Ruby son una forma de agrupar código reutilizable en una sola unidad. Ayudan a que el código sea más modular, legible y mantenible. Un método en Ruby se define usando la palabra clave def seguida del nombre del método y parámetros opcionales.

Conceptos Clave

  • Definición de Método: La sintaxis para definir un método.
  • Invocación de Método: Cómo llamar o invocar un método.
  • Parámetros y Argumentos: Pasar datos a los métodos.
  • Valores de Retorno: Lo que un método devuelve después de la ejecución.
  • Alcance: Entender el alcance de las variables dentro de los métodos.

Definición de Método

Para definir un método en Ruby, se usa la palabra clave def seguida del nombre del método. Aquí está la sintaxis básica:

def nombre_del_metodo
  # código a ejecutar
end

Ejemplo

def decir_hola
  puts "¡Hola, Mundo!"
end

En este ejemplo, decir_hola es un método que imprime "¡Hola, Mundo!" en la consola.

Invocación de Método

Para invocar o llamar a un método, simplemente se usa el nombre del método seguido de paréntesis (opcional si no se pasan argumentos).

Ejemplo

decir_hola

Esto imprimirá:

¡Hola, Mundo!

Parámetros y Argumentos

Los métodos pueden tomar parámetros, que son variables que contienen los valores pasados al método cuando se llama.

Ejemplo

def saludar(nombre)
  puts "¡Hola, #{nombre}!"
end

saludar("Alicia")

Esto imprimirá:

¡Hola, Alicia!

Múltiples Parámetros

Puedes definir métodos con múltiples parámetros.

def sumar(a, b)
  a + b
end

puts sumar(2, 3)

Esto imprimirá:

5

Valores de Retorno

Por defecto, un método en Ruby devuelve el valor de la última expresión evaluada. También puedes usar la palabra clave return para devolver explícitamente un valor.

Ejemplo

def multiplicar(a, b)
  return a * b
end

puts multiplicar(4, 5)

Esto imprimirá:

20

Alcance

Las variables definidas dentro de un método no son accesibles fuera de él. Esto se conoce como el alcance de las variables.

Ejemplo

def metodo_ejemplo
  var_local = "Estoy dentro del método"
  puts var_local
end

metodo_ejemplo
# puts var_local # Esto generará un error

Esto imprimirá:

Estoy dentro del método

Intentar acceder a var_local fuera del método resultará en un error.

Conceptos Avanzados de Métodos

Parámetros por Defecto

Puedes asignar valores por defecto a los parámetros. Si no se pasa ningún argumento, se usa el valor por defecto.

def saludar(nombre="Mundo")
  puts "¡Hola, #{nombre}!"
end

saludar
saludar("Alicia")

Esto imprimirá:

¡Hola, Mundo!
¡Hola, Alicia!

Número Variable de Argumentos

Ruby permite que los métodos acepten un número variable de argumentos usando el operador splat (*).

def sumar(*numeros)
  numeros.inject(0) { |suma, numero| suma + numero }
end

puts sumar(1, 2, 3, 4)

Esto imprimirá:

10

Argumentos de Palabra Clave

Ruby también soporta argumentos de palabra clave, que son una forma de pasar argumentos usando pares clave-valor.

def imprimir_direccion(ciudad:, estado:)
  puts "Ciudad: #{ciudad}, Estado: #{estado}"
end

imprimir_direccion(ciudad: "San Francisco", estado: "CA")

Esto imprimirá:

Ciudad: San Francisco, Estado: CA

Conclusión

Definir métodos en Ruby es una habilidad fundamental que ayuda a escribir código limpio, reutilizable y mantenible. Al entender cómo definir métodos, pasar parámetros, manejar valores de retorno y gestionar el alcance, puedes crear programas Ruby más complejos y eficientes. A medida que avances, también encontrarás características más sofisticadas como parámetros por defecto, argumentos variables y argumentos de palabra clave, que proporcionan flexibilidad y potencia adicionales en las definiciones de métodos.
