Configurar Rails es un paso esencial para cualquier desarrollador de Ruby que busque construir aplicaciones web. Esta sección te guiará a través del proceso de instalación y configuración de Rails en tu sistema, asegurando que tengas una base sólida para comenzar a desarrollar tus aplicaciones.

Requisitos previos

Antes de configurar Rails, asegúrate de tener los siguientes requisitos previos instalados en tu sistema:

  • Ruby: Rails es un framework de Ruby, por lo que necesitas tener Ruby instalado. Preferiblemente, usa un gestor de versiones como RVM o rbenv.
  • Node.js: Rails usa Node.js para gestionar dependencias de JavaScript.
  • Yarn: Un gestor de paquetes para JavaScript, usado por Rails para gestionar dependencias de front-end.
  • SQLite3: Una base de datos ligera usada por defecto en Rails.

Instalación de Ruby

Primero, instala Ruby usando un gestor de versiones. Aquí, usaremos rbenv:

  1. Instala rbenv y ruby-build:

    $ brew install rbenv
$ brew install ruby-build

  2. Configura rbenv en tu shell:

    $ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile
$ source ~/.bash_profile

  3. Instala Ruby:

    $ rbenv install 3.0.0
$ rbenv global 3.0.0

  4. Verifica la instalación:

    $ ruby -v

Instalación de Node.js y Yarn

A continuación, instala Node.js y Yarn:

  1. Instala Node.js:

    $ brew install node

  2. Instala Yarn:

    $ brew install yarn

Instalación de SQLite3

SQLite3 es la base de datos predeterminada para el desarrollo de Rails. Instálala usando Homebrew:

$ brew install sqlite3

Instalación de Rails

Con Ruby, Node.js, Yarn y SQLite3 instalados, ahora puedes instalar Rails:

  1. Instala Rails:

    $ gem install rails

  2. Verifica la instalación:

    $ rails -v

Creación de una nueva aplicación Rails

Una vez que Rails esté instalado, puedes crear una nueva aplicación Rails:

  1. Crea una nueva aplicación Rails:

    $ rails new myapp

  2. Navega a tu directorio de aplicación:

    $ cd myapp

  3. Inicia el servidor de Rails:

    $ rails server

  4. Abre tu navegador y navega a http://localhost:3000 para ver tu nueva aplicación Rails en funcionamiento.

Conclusión

Configurar Rails implica instalar varias dependencias y configurar tu entorno de desarrollo. Siguiendo los pasos descritos en esta sección, deberías tener una configuración de Rails completamente funcional lista para el desarrollo. Esta base te permitirá comenzar a construir potentes aplicaciones web utilizando el framework Rails.
