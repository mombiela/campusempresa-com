Introducción a las Constantes

En Ruby, las constantes se utilizan para almacenar valores que no deben cambiar durante la ejecución de un programa. Son una parte fundamental de la programación en Ruby y ayudan a mantener la integridad de los datos que deben permanecer constantes.

  • Definición: Las constantes son variables cuyos valores están destinados a permanecer sin cambios.
  • Convención de Nombres: Las constantes se escriben en letras mayúsculas, con palabras separadas por guiones bajos (_).

Declaración de Constantes

Para declarar una constante en Ruby, simplemente asigna un valor a un nombre de variable que comience con una letra mayúscula.

PI = 3.14159
MAX_USERS = 100
WELCOME_MESSAGE = "¡Bienvenido a la Programación en Ruby!"
  • PI, MAX_USERS y WELCOME_MESSAGE son constantes.
  • Por convención, las constantes se escriben en letras mayúsculas.

Acceso a las Constantes

Las constantes se pueden acceder directamente por sus nombres dentro del mismo ámbito.

puts PI  # Salida: 3.14159
puts MAX_USERS  # Salida: 100
puts WELCOME_MESSAGE  # Salida: ¡Bienvenido a la Programación en Ruby!

Constantes en Clases y Módulos

Las constantes también se pueden definir dentro de clases y módulos. Se acceden usando el operador de resolución de ámbito ::.

class Circle
  PI = 3.14159
end

module Config
  MAX_USERS = 100
end

puts Circle::PI  # Salida: 3.14159
puts Config::MAX_USERS  # Salida: 100
  • Circle::PI accede a la constante PI dentro de la clase Circle.
  • Config::MAX_USERS accede a la constante MAX_USERS dentro del módulo Config.

Modificación de Constantes

Aunque las constantes están destinadas a permanecer sin cambios, Ruby permite modificarlas. Sin embargo, hacerlo generará una advertencia.

PI = 3.14159
PI = 3.14  # Advertencia: constante PI ya inicializada
  • Modificar una constante generalmente se desaconseja ya que va en contra del propósito de usar constantes.

Constantes e Herencia

Cuando una clase hereda de otra clase, también hereda sus constantes.

class Parent
  CONSTANT = "Constante del Padre"
end

class Child < Parent
end

puts Child::CONSTANT  # Salida: Constante del Padre
  • La clase Child hereda la constante CONSTANT de la clase Parent.

Ejercicios

  1. Declarar y Acceder a Constantes: Crea una constante GRAVITY con un valor de 9.8 e imprímela.
  2. Constantes en Clases: Define una clase MathConstants con una constante E (número de Euler) establecida en 2.718. Accede e imprime esta constante.
  3. Modificar una Constante: Declara una constante SPEED_OF_LIGHT con un valor de 299792458. Intenta cambiar su valor y observa la advertencia.

Soluciones de Ejemplo

  1. Declarar y Acceder a Constantes

    GRAVITY = 9.8
puts GRAVITY  # Salida: 9.8

  2. Constantes en Clases

    class MathConstants
  E = 2.718
end

puts MathConstants::E  # Salida: 2.718

  3. Modificar una Constante

    SPEED_OF_LIGHT = 299792458
SPEED_OF_LIGHT = 300000000  # Advertencia: constante SPEED_OF_LIGHT ya inicializada

Conclusión

Las constantes en Ruby son una característica poderosa que ayuda a mantener la integridad de los valores que no deben cambiar durante la ejecución de un programa. Siguiendo las convenciones y entendiendo el ámbito y la herencia de las constantes, puedes escribir código Ruby más robusto y mantenible. Recuerda, aunque Ruby permite modificar constantes, es una buena práctica evitar hacerlo para preservar la inmutabilidad prevista de estos valores.
