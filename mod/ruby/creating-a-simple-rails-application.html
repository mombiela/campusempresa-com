Creando una Aplicación Simple de Rails
Descripción del Proyecto ►

Introducción

En esta sección, recorreremos el proceso de creación de una aplicación simple de Rails. Rails es un poderoso framework de aplicaciones web escrito en Ruby, y sigue la arquitectura Modelo-Vista-Controlador (MVC). Al final de esta sección, tendrás una comprensión básica de cómo configurar un proyecto de Rails, crear modelos, vistas y controladores, y ejecutar tu aplicación.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente instalado:

  • Ruby (versión 2.5 o superior)
  • Rails (versión 6 o superior)
  • SQLite3 (u otro sistema de bases de datos)

Configurando una Nueva Aplicación de Rails

  1. Instalar Rails: Si aún no has instalado Rails, puedes hacerlo ejecutando:

    gem install rails

  2. Crear una Nueva Aplicación de Rails: Usa la herramienta de línea de comandos de Rails para crear una nueva aplicación:

    rails new my_first_app

    Este comando generará un nuevo proyecto de Rails en un directorio llamado my_first_app.

  3. Navegar a Tu Directorio de Aplicación:

    cd my_first_app

  4. Iniciar el Servidor de Rails: Para ver tu aplicación en acción, inicia el servidor de Rails:

    rails server

    Abre tu navegador web y navega a http://localhost:3000. Deberías ver la página de bienvenida de Rails.

Creando un Modelo

Los modelos en Rails se utilizan para interactuar con la base de datos. Vamos a crear un modelo simple para un Post.

  1. Generar el Modelo:

    rails generate model Post title:string body:text

    Este comando crea un archivo de migración, un archivo de modelo y archivos de prueba para el modelo Post.

  2. Ejecutar la Migración:

    rails db:migrate

    Este comando aplica la migración, creando la tabla posts en la base de datos.

Creando un Controlador

Los controladores manejan las solicitudes y respuestas web. Vamos a crear un controlador para nuestro modelo Post.

  1. Generar el Controlador:

    rails generate controller Posts

    Este comando crea un archivo de controlador, plantillas de vista y archivos de prueba para el controlador Posts.

  2. Definir Acciones en el Controlador: Abre app/controllers/posts_controller.rb y define acciones para index, show, new y create:

    class PostsController < ApplicationController
  def index
    @posts = Post.all
  end

  def show
    @post = Post.find(params[:id])
  end

  def new
    @post = Post.new
  end

  def create
    @post = Post.new(post_params)
    if @post.save
      redirect_to @post
    else
      render 'new'
    end
  end

  private

  def post_params
    params.require(:post).permit(:title, :body)
  end
end

Creando Vistas

Las vistas se utilizan para mostrar los datos al usuario. Vamos a crear vistas para nuestras acciones de Post.

  1. Crear la Vista index: Crea un archivo app/views/posts/index.html.erb:

    <h1>All Posts</h1>
<ul>
  <% @posts.each do |post| %>
    <li>
      <%= link_to post.title, post %>
    </li>
  <% end %>
</ul>
<%= link_to 'New Post', new_post_path %>

  2. Crear la Vista show: Crea un archivo app/views/posts/show.html.erb:

    <h1><%= @post.title %></h1>
<p><%= @post.body %></p>
<%= link_to 'Back', posts_path %>

  3. Crear la Vista new: Crea un archivo app/views/posts/new.html.erb:

    <h1>New Post</h1>
<%= form_with model: @post, local: true do |form| %>
  <div>
    <%= form.label :title %>
    <%= form.text_field :title %>
  </div>
  <div>
    <%= form.label :body %>
    <%= form.text_area :body %>
  </div>
  <div>
    <%= form.submit %>
  </div>
<% end %>
<%= link_to 'Back', posts_path %>

Enrutamiento

Para conectar las acciones del controlador a las URLs, necesitamos definir rutas.

  1. Definir Rutas de Recursos: Abre config/routes.rb y añade: 
    Rails.application.routes.draw do
  resources :posts
  root 'posts#index'
end

Ejecutando la Aplicación

  1. Iniciar el Servidor de Rails: 
    rails server
  2. Acceder a la Aplicación: Abre tu navegador web y navega a http://localhost:3000. Deberías ver la lista de publicaciones y poder crear nuevas publicaciones.

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación de una aplicación simple de Rails. Configuramos un nuevo proyecto de Rails, creamos un modelo, un controlador y vistas, y definimos rutas para conectar todo. Esto es solo el comienzo; Rails ofrece muchas más características y herramientas para construir aplicaciones web complejas. Continúa explorando Rails para profundizar tu comprensión y construir aplicaciones más sofisticadas.
