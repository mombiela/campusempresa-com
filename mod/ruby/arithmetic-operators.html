Introducción

Los operadores aritméticos son fundamentales en cualquier lenguaje de programación, incluido Ruby. Permiten realizar operaciones matemáticas básicas como suma, resta, multiplicación, división y más. Entender estos operadores es crucial para manipular datos numéricos y realizar cálculos en tus programas.

Operadores Aritméticos Básicos

Ruby soporta los siguientes operadores aritméticos básicos:

  • Suma (+)
  • Resta (-)
  • Multiplicación (*)
  • División (/)
  • Módulo (%)
  • Exponenciación (**)

Suma (`+`)

El operador de suma añade dos números juntos.

a = 5
b = 3
sum = a + b
puts sum  # Output: 8

Resta (`-`)

El operador de resta resta el segundo número del primero.

a = 5
b = 3
difference = a - b
puts difference  # Output: 2

Multiplicación (`*`)

El operador de multiplicación multiplica dos números.

a = 5
b = 3
product = a * b
puts product  # Output: 15

División (`/`)

El operador de división divide el primer número por el segundo. Nota que en Ruby, dividir dos enteros resultará en un entero.

a = 5
b = 3
quotient = a / b
puts quotient  # Output: 1

Para obtener un resultado de punto flotante, al menos uno de los números debe ser un flotante.

a = 5.0
b = 3
quotient = a / b
puts quotient  # Output: 1.6666666666666667

Módulo (`%`)

El operador de módulo devuelve el resto de la división del primer número por el segundo.

a = 5
b = 3
remainder = a % b
puts remainder  # Output: 2

Exponenciación (`**`)

El operador de exponenciación eleva el primer número a la potencia del segundo número.

a = 5
b = 3
power = a ** b
puts power  # Output: 125

Uso Avanzado

Combinando Operadores

Puedes combinar operadores aritméticos para realizar cálculos más complejos.

a = 5
b = 3
c = 2
result = (a + b) * c - (a / b)
puts result  # Output: 15

Precedencia de Operadores

Ruby sigue las reglas estándar de precedencia matemática (PEMDAS/BODMAS). Los paréntesis pueden ser usados para definir explícitamente el orden de las operaciones.

a = 5
b = 3
c = 2
result = a + b * c  # Multiplicación antes de la suma
puts result  # Output: 11

result = (a + b) * c  # Los paréntesis cambian el orden
puts result  # Output: 16

Usando Operadores Aritméticos con Variables

Los operadores aritméticos pueden ser usados directamente con variables para actualizar sus valores.

a = 5
a += 3  # Equivalente a a = a + 3
puts a  # Output: 8

a -= 2  # Equivalente a a = a - 2
puts a  # Output: 6

a *= 2  # Equivalente a a = a * 2
puts a  # Output: 12

a /= 4  # Equivalente a a = a / 4
puts a  # Output: 3

a %= 2  # Equivalente a a = a % 2
puts a  # Output: 1

a **= 3  # Equivalente a a = a ** 3
puts a  # Output: 1

Conclusión

Los operadores aritméticos son herramientas esenciales en Ruby para realizar operaciones matemáticas. Desde sumas y restas básicas hasta expresiones más complejas que involucran múltiples operadores, entender cómo usar estos operadores efectivamente es clave para escribir código Ruby eficiente y preciso. Practica usando estos operadores en diferentes escenarios para familiarizarte más con su comportamiento y precedencia.
