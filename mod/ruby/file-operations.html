Introducción

Las operaciones con archivos son esenciales para muchas aplicaciones, permitiéndote leer y escribir archivos. Esta sección cubrirá los conceptos básicos del manejo de archivos en Ruby, avanzando desde operaciones simples hasta técnicas más avanzadas.

Operaciones Básicas con Archivos

Abrir y Cerrar Archivos

  • Abrir un Archivo: Usa el método File.open para abrir un archivo. Este método puede tomar un bloque, lo que asegura que el archivo se cierre correctamente después de que el bloque se ejecute.
  • Cerrar un Archivo: Si no usas un bloque, debes cerrar explícitamente el archivo usando el método close.
# Abrir un archivo con un bloque
File.open('example.txt', 'w') do |file|
  file.puts '¡Hola, mundo!'
end
# El archivo se cierra automáticamente después del bloque

# Abrir un archivo sin un bloque
file = File.open('example.txt', 'w')
file.puts '¡Hola, mundo!'
file.close

Leer Archivos

  • Leer Todo el Archivo: Usa File.read para leer todo el contenido de un archivo.
  • Leer Línea por Línea: Usa File.foreach o IO.readlines para leer un archivo línea por línea.
# Leer todo el archivo
content = File.read('example.txt')
puts content

# Leer línea por línea
File.foreach('example.txt') do |line|
  puts line
end

Escribir en Archivos

  • Modos de Escritura: Usa diferentes modos como 'w' para escribir, 'a' para añadir, y 'r+' para leer/escribir.
  • Escribir Datos: Usa métodos como puts, write, y print para escribir datos en un archivo.
# Escribir en un archivo
File.open('example.txt', 'w') do |file|
  file.puts '¡Hola, mundo!'
end

# Añadir a un archivo
File.open('example.txt', 'a') do |file|
  file.puts 'Añadiendo esta línea.'
end

Operaciones Intermedias con Archivos

Modos y Permisos de Archivos

  • Modos de Archivo: Entiende los diferentes modos de archivo como 'r', 'w', 'a', 'r+', 'w+', y 'a+'.
  • Permisos: Usa File.chmod para cambiar los permisos de un archivo.
# Cambiar permisos de archivo
File.chmod(0644, 'example.txt')

Posicionamiento de Archivos

  • Seek y Tell: Usa seek para mover el puntero del archivo y tell para obtener la posición actual.
File.open('example.txt', 'r') do |file|
  file.seek(5, IO::SEEK_SET) # Mover al quinto byte
  puts file.tell # Salida: 5
end

Trabajar con Archivos Binarios

  • Modo Binario: Abre archivos en modo binario usando 'rb', 'wb', o 'ab'.
# Leer un archivo binario
File.open('example.bin', 'rb') do |file|
  data = file.read
  puts data
end

Operaciones Avanzadas con Archivos

Manipulación de Archivos

  • Renombrar y Eliminar Archivos: Usa File.rename y File.delete para renombrar y eliminar archivos.
# Renombrar un archivo
File.rename('example.txt', 'new_example.txt')

# Eliminar un archivo
File.delete('new_example.txt')

Operaciones con Directorios

  • Crear y Eliminar Directorios: Usa Dir.mkdir y Dir.rmdir para crear y eliminar directorios.
  • Listar Contenidos del Directorio: Usa Dir.entries o Dir.glob para listar los contenidos del directorio.
# Crear un directorio
Dir.mkdir('new_directory')

# Listar contenidos del directorio
puts Dir.entries('.')

# Eliminar un directorio
Dir.rmdir('new_directory')

Información de Archivos y Directorios

  • Información de Archivos: Usa métodos como File.size, File.exist?, y File.mtime para obtener información sobre archivos.
  • Información de Directorios: Usa Dir.pwd para obtener el directorio actual y Dir.chdir para cambiar el directorio actual.
# Obtener información del archivo
puts File.size('example.txt')
puts File.exist?('example.txt')
puts File.mtime('example.txt')

# Obtener y cambiar el directorio actual
puts Dir.pwd
Dir.chdir('..')
puts Dir.pwd

Conclusión

Las operaciones con archivos en Ruby son poderosas y versátiles, permitiéndote manejar archivos y directorios de manera eficiente. Desde la lectura y escritura básicas hasta la manipulación avanzada de archivos y operaciones con directorios, dominar estas habilidades te permitirá construir aplicaciones robustas que interactúen con el sistema de archivos. Practica estos conceptos con varios ejercicios para solidificar tu comprensión y volverte competente en el manejo de archivos con Ruby.
