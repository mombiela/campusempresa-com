Tu Primer Programa en Ruby
Sintaxis Básica ►

¡Bienvenido a tu viaje en la programación con Ruby! En esta sección, te guiaremos a través de la escritura de tu primer programa en Ruby. Al final de esta lección, tendrás una comprensión básica de cómo escribir y ejecutar un script simple en Ruby.

Configurando Tu Entorno

Antes de comenzar a codificar, asegúrate de tener Ruby instalado en tu computadora.

  1. Verificar si Ruby está instalado:

    • Abre tu terminal (Command Prompt en Windows, Terminal en macOS/Linux).
    • Escribe ruby -v y presiona Enter.
    • Si Ruby está instalado, verás el número de versión. Si no, necesitas instalarlo.

  2. Instalando Ruby:

    • Visita el sitio web oficial de Ruby para obtener instrucciones de instalación específicas para tu sistema operativo.
    • Sigue las instrucciones para descargar e instalar Ruby.

Escribiendo Tu Primer Programa en Ruby

Vamos a escribir un programa simple que imprima "¡Hola, Mundo!" en la pantalla.

  1. Crear un nuevo archivo:

    • Abre tu editor de texto (por ejemplo, VSCode, Sublime Text, o incluso Notepad).
    • Crea un nuevo archivo y guárdalo como hello_world.rb.

  2. Escribir el código:

    • En tu nuevo archivo, escribe el siguiente código:
    # Esto es un comentario
puts "¡Hola, Mundo!"

  3. Explicación:

    • # Esto es un comentario: Cualquier cosa que siga al símbolo # es un comentario y es ignorado por el intérprete de Ruby. Los comentarios son útiles para explicar tu código.
    • puts "¡Hola, Mundo!": El método puts se usa para imprimir texto en la pantalla. En este caso, imprime ¡Hola, Mundo!.

Ejecutando Tu Programa en Ruby

Ahora que has escrito tu primer programa en Ruby, vamos a ejecutarlo.

  1. Abre tu terminal.
  2. Navega al directorio donde guardaste hello_world.rb usando el comando cd.
  3. Ejecuta el programa escribiendo ruby hello_world.rb y presionando Enter.

Deberías ver la salida:

¡Hola, Mundo!

Conceptos Clave

  • Intérprete de Ruby: El programa que lee y ejecuta el código Ruby.
  • Comentarios: Líneas que comienzan con # y son ignoradas por el intérprete. Se usan para documentación.
  • Método puts: Un método incorporado en Ruby para mostrar texto en la consola.

Ejercicios

  1. Modificar el Programa:

    • Cambia el texto dentro del método puts para imprimir tu nombre en lugar de "¡Hola, Mundo!".
    puts "¡Hola, [Tu Nombre]!"

  2. Agregar Más Salida:

    • Agrega otra declaración puts para imprimir una segunda línea de texto.
    puts "¡Hola, [Tu Nombre]!"
puts "¡Bienvenido a la programación con Ruby!"

  3. Experimentar con Comentarios:

    • Agrega comentarios a tu código para describir lo que hace cada línea.
    # Esta línea imprime un saludo
puts "¡Hola, [Tu Nombre]!"
# Esta línea imprime un mensaje de bienvenida
puts "¡Bienvenido a la programación con Ruby!"

Resumen

¡Felicidades! Has escrito y ejecutado tu primer programa en Ruby. En esta lección, aprendiste a:

  • Configurar tu entorno Ruby.
  • Escribir un script simple en Ruby.
  • Usar el método puts para imprimir texto.
  • Agregar comentarios a tu código.

Estas habilidades fundamentales son los pilares para tu viaje en la programación con Ruby. Sigue experimentando y practicando para sentirte más cómodo con la sintaxis y los conceptos de Ruby. En la próxima lección, profundizaremos en los tipos de datos y variables de Ruby. ¡Feliz codificación!
