Pasar Datos Entre Actividades
Introducción a los Fragmentos ►

Pasar datos entre actividades es un concepto fundamental en el desarrollo de Android. Este tema te guiará desde los conceptos básicos hasta técnicas avanzadas para compartir datos entre actividades en una aplicación de Android.

Introducción a los Intents

Los intents son el método principal para iniciar nuevas actividades y pasar datos entre ellas.

  • Intent: Una descripción abstracta de una operación a realizar.
  • Intent Explícito: Especifica la actividad exacta a iniciar.
  • Intent Implícito: Declara una acción general a realizar, permitiendo al sistema encontrar una actividad apropiada.

Crear un Intent

Para iniciar una nueva actividad, creas un objeto Intent y llamas a startActivity().

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
startActivity(intent);

Pasar Datos con Intents

Puedes pasar datos entre actividades usando el método putExtra de la clase Intent.

Tipos de Datos Básicos

Puedes pasar tipos de datos básicos como int, String, boolean, etc.

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MESSAGE", "¡Hola, Mundo!");
intent.putExtra("EXTRA_NUMBER", 123);
startActivity(intent);

Recuperar Datos en la Actividad Destino

En la actividad destino, recuperas los datos usando el método getIntent() y los métodos get apropiados.

Intent intent = getIntent();
String message = intent.getStringExtra("EXTRA_MESSAGE");
int number = intent.getIntExtra("EXTRA_NUMBER", 0);

Pasar Tipos de Datos Complejos

Para tipos de datos más complejos, como objetos personalizados, necesitas implementar la interfaz Parcelable o Serializable.

Usar Parcelable

Parcelable es el método preferido para pasar tipos de datos complejos porque es más eficiente que Serializable.

Crear una Clase Parcelable

public class MyObject implements Parcelable {
    private int id;
    private String name;

    // Constructor
    public MyObject(int id, String name) {
        this.id = id;
        this.name = name;
    }

    // Implementación de Parcelable
    protected MyObject(Parcel in) {
        id = in.readInt();
        name = in.readString();
    }

    public static final Creator<MyObject> CREATOR = new Creator<MyObject>() {
        @Override
        public MyObject createFromParcel(Parcel in) {
            return new MyObject(in);
        }

        @Override
        public MyObject[] newArray(int size) {
            return new MyObject[size];
        }
    };

    @Override
    public int describeContents() {
        return 0;
    }

    @Override
    public void writeToParcel(Parcel dest, int flags) {
        dest.writeInt(id);
        dest.writeString(name);
    }
}

Pasar un Objeto Parcelable

MyObject myObject = new MyObject(1, "Ejemplo");
Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MY_OBJECT", myObject);
startActivity(intent);

Recuperar un Objeto Parcelable

Intent intent = getIntent();
MyObject myObject = intent.getParcelableExtra("EXTRA_MY_OBJECT");

Técnicas Avanzadas

Usar Bundles

Los bundles son útiles para pasar múltiples piezas de datos juntas.

Bundle bundle = new Bundle();
bundle.putString("EXTRA_MESSAGE", "¡Hola, Mundo!");
bundle.putInt("EXTRA_NUMBER", 123);

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtras(bundle);
startActivity(intent);

Usar Shared Preferences

Para datos persistentes que necesitan ser accesibles en múltiples actividades, considera usar Shared Preferences.

// Guardar datos
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("EXTRA_MESSAGE", "¡Hola, Mundo!");
editor.apply();

// Recuperar datos
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
String message = sharedPreferences.getString("EXTRA_MESSAGE", "Valor Predeterminado");

Conclusión

Pasar datos entre actividades es una habilidad crucial en el desarrollo de Android. Al dominar intents, bundles y shared preferences, puedes compartir datos de manera eficiente y crear una experiencia de usuario fluida. Practica estas técnicas para volverte competente en la gestión del flujo de datos en tus aplicaciones de Android.
