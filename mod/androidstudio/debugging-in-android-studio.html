Depuración en Android Studio
Pruebas Unitarias ►

La depuración es una habilidad crucial para cualquier desarrollador, y Android Studio proporciona un conjunto robusto de herramientas para ayudarte a identificar y solucionar problemas en tu código. Esta sección te guiará a través del proceso de depuración en Android Studio, desde niveles principiantes hasta avanzados.

Introducción a la Depuración

La depuración es el proceso de identificar y eliminar errores de tu código. En Android Studio, las herramientas de depuración te ayudan a inspeccionar tu código, monitorear los valores de las variables y controlar el flujo de ejecución para encontrar y corregir errores.

Conceptos Clave

  • Puntos de interrupción: Marcadores que puedes establecer en tu código para pausar la ejecución en una línea específica.
  • Puntos de observación: Puntos de interrupción especiales que pausan la ejecución cuando el valor de una variable cambia.
  • Logcat: Una herramienta que proporciona mensajes del sistema, incluyendo trazas de pila cuando la aplicación lanza un error.
  • Depurador: Una herramienta que te permite ejecutar tu aplicación paso a paso para inspeccionar su comportamiento.

Configuración de Puntos de Interrupción

Los puntos de interrupción son esenciales para pausar la ejecución de tu aplicación y así inspeccionar el estado de tu aplicación en un punto específico.

Cómo Establecer un Punto de Interrupción

  1. Abre tu proyecto en Android Studio.
  2. Navega a la línea de código donde deseas establecer el punto de interrupción.
  3. Haz clic en el margen izquierdo junto al número de línea, o presiona Ctrl+F8 (Windows/Linux) o Cmd+F8 (Mac).
public void onClick(View view) {
    int a = 5;
    int b = 10;
    int sum = a + b; // Establece un punto de interrupción aquí
    Log.d("Debugging", "Sum: " + sum);
}

Uso de Logcat

Logcat es una herramienta poderosa para ver mensajes del sistema, incluyendo registros de errores y trazas de pila.

Filtrando la Salida de Logcat

  1. Abre Logcat haciendo clic en View > Tool Windows > Logcat.
  2. Usa la barra de búsqueda para filtrar mensajes por etiqueta o palabra clave.
  3. También puedes usar los menús desplegables para filtrar por nivel de registro (por ejemplo, Debug, Info, Warn, Error).
Log.d("MainActivity", "This is a debug message");
Log.e("MainActivity", "This is an error message");

Depuración con el Depurador

El depurador te permite controlar el flujo de ejecución de tu aplicación e inspeccionar variables.

Iniciando una Sesión de Depuración

  1. Establece puntos de interrupción en tu código.
  2. Haz clic en el botón Debug (un icono de un insecto) o presiona Shift+F9.
  3. La aplicación se iniciará en modo de depuración y se pausará en el primer punto de interrupción.

Inspeccionando Variables

  • Cuando la aplicación se pausa en un punto de interrupción, puedes pasar el cursor sobre las variables para ver sus valores.
  • Usa el panel Variables para ver y modificar los valores de las variables.
public void calculateSum() {
    int x = 3;
    int y = 7;
    int result = x + y; // Inspecciona variables aquí
    Log.d("Debugging", "Result: " + result);
}

Técnicas Avanzadas de Depuración

Puntos de Interrupción Condicionales

Los puntos de interrupción condicionales te permiten pausar la ejecución solo cuando se cumple una condición específica.

  1. Haz clic derecho en un punto de interrupción.
  2. Selecciona Breakpoint Properties.
  3. Ingresa una condición (por ejemplo, x > 5).

Puntos de Observación

Los puntos de observación pausan la ejecución cuando el valor de una variable cambia.

  1. Haz clic derecho en una variable en el panel Variables.
  2. Selecciona Add Watchpoint.

Análisis de Trazas de Pila

Las trazas de pila proporcionan una instantánea de la pila de llamadas en el momento en que ocurre una excepción.

  • Usa Logcat para ver las trazas de pila.
  • Haz clic en una traza de pila para navegar a la línea de código correspondiente.
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    Log.e("MainActivity", "Error: " + e.getMessage());
}

Conclusión

La depuración es una habilidad esencial para cualquier desarrollador de Android. Al dominar las herramientas y técnicas proporcionadas por Android Studio, puedes identificar y solucionar problemas en tu código de manera eficiente. Comienza con puntos de interrupción básicos y Logcat, y gradualmente explora características avanzadas como puntos de interrupción condicionales y puntos de observación para mejorar tus capacidades de depuración.
