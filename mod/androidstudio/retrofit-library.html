Introducción a Retrofit

Retrofit es un cliente HTTP seguro para Android y Java desarrollado por Square. Simplifica el proceso de consumir servicios web RESTful al convertir la API HTTP en una interfaz Java.

  • Seguro: Asegura que los tipos de datos utilizados en las llamadas a la API sean consistentes y correctos.
  • Cliente HTTP: Maneja solicitudes y respuestas HTTP.
  • Servicios web RESTful: Servicios web que siguen los principios de Transferencia de Estado Representacional (REST).

Configuración de Retrofit en Android Studio

Agregar Dependencias

Para usar Retrofit en tu proyecto de Android, necesitas agregar las dependencias necesarias en tu archivo build.gradle.

dependencies {
    implementation 'com.squareup.retrofit2:retrofit:2.9.0'
    implementation 'com.squareup.retrofit2:converter-gson:2.9.0'
}
  • retrofit: La biblioteca principal de Retrofit.
  • converter-gson: Una biblioteca de conversión para analizar datos JSON usando Gson.

Creando una Instancia de Retrofit

Para crear una instancia de Retrofit, necesitas especificar la URL base y la fábrica de convertidores.

Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("https://api.example.com/")
    .addConverterFactory(GsonConverterFactory.create())
    .build();
  • baseUrl: La URL base para la API.
  • addConverterFactory: Especifica el convertidor a usar para analizar los datos de respuesta.

Definiendo Endpoints de la API

Creando una Interfaz

Define los endpoints de la API en una interfaz usando anotaciones de Retrofit.

public interface ApiService {
    @GET("users/{user}/repos")
    Call<List<Repo>> listRepos(@Path("user") String user);
}
  • @GET: Especifica el método HTTP GET.
  • @Path: Reemplaza el marcador {user} con el parámetro de usuario real.

Realizando Llamadas a la API

Crea una instancia del servicio API y realiza llamadas a la API.

ApiService service = retrofit.create(ApiService.class);
Call<List<Repo>> repos = service.listRepos("octocat");

repos.enqueue(new Callback<List<Repo>>() {
    @Override
    public void onResponse(Call<List<Repo>> call, Response<List<Repo>> response) {
        if (response.isSuccessful()) {
            List<Repo> repoList = response.body();
            // Manejar la respuesta
        }
    }

    @Override
    public void onFailure(Call<List<Repo>> call, Throwable t) {
        // Manejar el error
    }
});
  • create: Crea una implementación de los endpoints de la API definidos por la interfaz.
  • enqueue: Envía la solicitud de manera asíncrona y notifica al callback de su respuesta o si ocurrió un error.

Uso Avanzado

Agregar Interceptores

Los interceptores pueden ser usados para modificar solicitudes y respuestas.

OkHttpClient client = new OkHttpClient.Builder()
    .addInterceptor(new Interceptor() {
        @Override
        public Response intercept(Chain chain) throws IOException {
            Request request = chain.request().newBuilder()
                .addHeader("Authorization", "Bearer token")
                .build();
            return chain.proceed(request);
        }
    })
    .build();

Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("https://api.example.com/")
    .client(client)
    .addConverterFactory(GsonConverterFactory.create())
    .build();
  • OkHttpClient: Un cliente HTTP personalizable.
  • addInterceptor: Agrega un interceptor al cliente.

Manejo de Diferentes Formatos de Respuesta

Retrofit puede manejar diferentes formatos de respuesta usando diferentes convertidores.

dependencies {
    implementation 'com.squareup.retrofit2:converter-moshi:2.9.0'
}
Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("https://api.example.com/")
    .addConverterFactory(MoshiConverterFactory.create())
    .build();
  • converter-moshi: Una biblioteca de conversión para analizar datos JSON usando Moshi.

Conclusión

Retrofit es una biblioteca poderosa y flexible para realizar solicitudes HTTP en aplicaciones Android. Al entender su configuración, definir endpoints de la API y manejar respuestas, puedes integrar eficientemente servicios web en tus proyectos de Android. Las características avanzadas como interceptores y diferentes convertidores mejoran aún más sus capacidades, haciendo de Retrofit una herramienta esencial para los desarrolladores de Android.
