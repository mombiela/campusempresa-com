Tipos Mapeados en TypeScript
Estilo de Código y Formateo ►

Los tipos mapeados son una característica poderosa en TypeScript que te permiten crear nuevos tipos transformando los existentes. Son particularmente útiles para crear tipos que son variaciones de otros tipos, como hacer que todas las propiedades sean opcionales o de solo lectura.

Conceptos Clave

  • Tipos Mapeados: Tipos que transforman propiedades de un tipo existente.
  • Tipos de Utilidad de TypeScript: Tipos mapeados predefinidos proporcionados por TypeScript, como Partial, Readonly, Record y Pick.

Tipos Mapeados Básicos

Los tipos mapeados usan el operador keyof y corchetes para iterar sobre las propiedades de un tipo existente y transformarlas.

Ejemplo: Hacer Todas las Propiedades Opcionales

type Person = {
  name: string;
  age: number;
};

type OptionalPerson = {
  [P in keyof Person]?: Person[P];
};

// Equivalente a:
// type OptionalPerson = {
//   name?: string;
//   age?: number;
// };

En este ejemplo, OptionalPerson es un tipo mapeado que hace que todas las propiedades de Person sean opcionales.

Tipos de Utilidad de TypeScript

TypeScript proporciona varios tipos de utilidad que están construidos usando tipos mapeados. Estos tipos de utilidad simplifican las transformaciones comunes de tipos.

Partial

El tipo de utilidad Partial hace que todas las propiedades de un tipo sean opcionales.

type Person = {
  name: string;
  age: number;
};

type PartialPerson = Partial<Person>;

// Equivalente a:
// type PartialPerson = {
//   name?: string;
//   age?: number;
// };

Readonly

El tipo de utilidad Readonly hace que todas las propiedades de un tipo sean de solo lectura.

type Person = {
  name: string;
  age: number;
};

type ReadonlyPerson = Readonly<Person>;

// Equivalente a:
// type ReadonlyPerson = {
//   readonly name: string;
//   readonly age: number;
// };

Record

El tipo de utilidad Record construye un tipo con un conjunto de propiedades K del tipo T.

type PageInfo = {
  title: string;
};

type Page = "home" | "about" | "contact";

type Pages = Record<Page, PageInfo>;

// Equivalente a:
// type Pages = {
//   home: PageInfo;
//   about: PageInfo;
//   contact: PageInfo;
// };

Pick

El tipo de utilidad Pick construye un tipo seleccionando un conjunto de propiedades K del tipo T.

type Person = {
  name: string;
  age: number;
  address: string;
};

type PersonNameAndAge = Pick<Person, "name" | "age">;

// Equivalente a:
// type PersonNameAndAge = {
//   name: string;
//   age: number;
// };

Tipos Mapeados Personalizados

Puedes crear tus propios tipos mapeados personalizados para satisfacer necesidades específicas.

Ejemplo: Hacer Todas las Propiedades Nulas

type Person = {
  name: string;
  age: number;
};

type Nullable<T> = {
  [P in keyof T]: T[P] | null;
};

type NullablePerson = Nullable<Person>;

// Equivalente a:
// type NullablePerson = {
//   name: string | null;
//   age: number | null;
// };

Conclusión

Los tipos mapeados en TypeScript proporcionan una forma flexible de transformar tipos existentes en nuevos. Al entender y utilizar tipos mapeados, puedes crear definiciones de tipos más dinámicas y reutilizables. Los tipos de utilidad integrados de TypeScript como Partial, Readonly, Record y Pick son ejemplos prácticos de cómo los tipos mapeados pueden simplificar las transformaciones de tipos. Los tipos mapeados personalizados amplían aún más esta capacidad, permitiéndote adaptar las transformaciones de tipos a tus necesidades específicas.
