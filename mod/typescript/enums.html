Introducción a las Enumeraciones

Las enumeraciones (abreviadas como Enums) son una característica en TypeScript que te permite definir un conjunto de constantes con nombre. Facilitan el trabajo con conjuntos de valores relacionados y mejoran la legibilidad y el mantenimiento del código.

Conceptos Clave

  • Enumeración: Una forma de definir una colección de valores relacionados.
  • Constantes con Nombre: Cada valor en una enumeración es una constante con un nombre.
  • Seguridad de Tipos: Las enumeraciones proporcionan seguridad de tipos al asegurar que solo se puedan usar valores predefinidos.

Definiendo Enumeraciones

En TypeScript, puedes definir una enumeración usando la palabra clave enum.

Ejemplo

enum Dirección {
    Arriba,
    Abajo,
    Izquierda,
    Derecha
}

En este ejemplo, Dirección es una enumeración con cuatro valores posibles: Arriba, Abajo, Izquierda y Derecha.

Explicación

  • Por defecto, los valores de los miembros de la enumeración se autoincrementan comenzando desde 0.
  • Arriba es 0, Abajo es 1, Izquierda es 2, y Derecha es 3.

Usando Enumeraciones

Puedes usar enumeraciones para declarar variables y asignarles valores de la enumeración.

Ejemplo

let movimiento: Dirección = Dirección.Arriba;
console.log(movimiento); // Salida: 0

Explicación

  • La variable movimiento es del tipo Dirección.
  • Se le asigna el valor Dirección.Arriba, que corresponde a 0.

Personalizando Valores de Enumeración

Puedes asignar valores personalizados a los miembros de la enumeración.

Ejemplo

enum Estado {
    Éxito = 1,
    Fallo = -1,
    Pendiente = 0
}

Explicación

  • Éxito se asigna el valor 1.
  • Fallo se asigna el valor -1.
  • Pendiente se asigna el valor 0.

Enumeraciones de Cadenas

Las enumeraciones también pueden tener valores de cadena.

Ejemplo

enum Color {
    Rojo = "ROJO",
    Verde = "VERDE",
    Azul = "AZUL"
}

Explicación

  • Cada miembro de la enumeración Color se le asigna un valor de cadena.

Enumeraciones Heterogéneas

Las enumeraciones pueden tener tanto valores numéricos como de cadena, aunque esto es menos común.

Ejemplo

enum Mixto {
    No = 0,
    Sí = "SÍ"
}

Explicación

  • No se asigna el valor numérico 0.
  • Sí se asigna el valor de cadena "SÍ".

Tipos de Miembros de Enumeración

Los miembros de la enumeración pueden ser constantes o calculados.

Miembros Constantes

Estos son miembros con valores que pueden ser calculados en tiempo de compilación.

Ejemplo

enum EnumeraciónConstante {
    A = 1,
    B = A + 1
}

Explicación

  • A es 1.
  • B es A + 1, que es 2.

Miembros Calculados

Estos son miembros con valores que se calculan en tiempo de ejecución.

Ejemplo

enum EnumeraciónCalculada {
    X = "X".length,
    Y = 10 * 2
}

Explicación

  • X es la longitud de la cadena "X", que es 1.
  • Y es el resultado de la expresión 10 * 2, que es 20.

Mapeo Inverso de Enumeraciones

TypeScript proporciona mapeo inverso para enumeraciones numéricas.

Ejemplo

enum Días {
    Lunes,
    Martes,
    Miércoles
}

console.log(Días.Lunes); // Salida: 0
console.log(Días[0]); // Salida: "Lunes"

Explicación

  • Días.Lunes da el valor numérico 0.
  • Días[0] da la cadena "Lunes".

Enumeraciones como Tipos

Las enumeraciones pueden ser usadas como tipos para asegurar que las variables solo puedan tomar los valores definidos en la enumeración.

Ejemplo

enum Tamaño {
    Pequeño,
    Mediano,
    Grande
}

function seleccionarTamaño(tamaño: Tamaño): void {
    console.log(`Tamaño seleccionado: ${Tamaño[tamaño]}`);
}

seleccionarTamaño(Tamaño.Mediano); // Salida: Tamaño seleccionado: Mediano

Explicación

  • La función seleccionarTamaño acepta un parámetro del tipo Tamaño.
  • Esto asegura que solo valores válidos de Tamaño puedan ser pasados a la función.

Conclusión

Las enumeraciones en TypeScript proporcionan una forma poderosa de definir y usar conjuntos de constantes relacionadas. Mejoran la legibilidad del código, el mantenimiento y la seguridad de tipos. Al entender cómo definir, usar y personalizar enumeraciones, puedes aprovechar todo su potencial en tus proyectos de TypeScript.
