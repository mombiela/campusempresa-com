Introducción a TypeScript con Node.js

  • Descripción general de TypeScript y sus beneficios.
  • ¿Por qué usar TypeScript con Node.js?
  • Configuración de un proyecto TypeScript con Node.js.

Configuración del Entorno

  • Instalación de Node.js y npm.
  • Instalación global de TypeScript: 
    npm install -g typescript
  • Inicialización de un nuevo proyecto Node.js: 
    mkdir my-typescript-node-project
cd my-typescript-node-project
npm init -y
  • Añadir TypeScript al proyecto: 
    npm install typescript --save-dev
  • Creación de un archivo tsconfig.json: 
    npx tsc --init

Conceptos Básicos de TypeScript

  • Anotaciones de Tipo: 
    let message: string = "¡Hola, TypeScript con Node.js!";
console.log(message);
  • Interfaces: 
    interface User {
  name: string;
  age: number;
}

const user: User = {
  name: "John Doe",
  age: 30
};
console.log(user);
  • Clases: 
    class Person {
  constructor(public name: string, public age: number) {}

  greet() {
    console.log(`Hola, mi nombre es ${this.name} y tengo ${this.age} años.`);
  }
}

const person = new Person("Jane Doe", 25);
person.greet();

Integración de TypeScript con Node.js

  • Creación de un servidor simple con Express: 
    npm install express @types/express --save

    import express, { Request, Response } from 'express';

const app = express();
const port = 3000;

app.get('/', (req: Request, res: Response) => {
  res.send('¡Hola, TypeScript con Node.js!');
});

app.listen(port, () => {
  console.log(`El servidor está corriendo en http://localhost:${port}`);
});
  • Compilación de TypeScript a JavaScript: 
    npx tsc
  • Ejecución del JavaScript compilado: 
    node dist/index.js

Características Avanzadas de TypeScript

  • Genéricos: 
    function identity<T>(arg: T): T {
  return arg;
}

let output = identity<string>("myString");
console.log(output);
  • Decoradores: 
    function log(target: any, key: string) {
  let value = target[key];

  const getter = () => {
    console.log(`Get: ${key} => ${value}`);
    return value;
  };

  const setter = (newVal: any) => {
    console.log(`Set: ${key} => ${newVal}`);
    value = newVal;
  };

  Object.defineProperty(target, key, {
    get: getter,
    set: setter,
    enumerable: true,
    configurable: true
  });
}

class Example {
  @log
  public name: string;

  constructor(name: string) {
    this.name = name;
  }
}

const example = new Example("TypeScript");
example.name = "Node.js";
console.log(example.name);

Pruebas de TypeScript con Node.js

  • Configuración de Jest para pruebas: 
    npm install jest ts-jest @types/jest --save-dev
  • Configuración de Jest: 
    npx ts-jest config:init
  • Escritura de una prueba simple: 
    // sum.ts
export const sum = (a: number, b: number): number => {
  return a + b;
};

// sum.test.ts
import { sum } from './sum';

test('suma 1 + 2 para igualar 3', () => {
  expect(sum(1, 2)).toBe(3);
});
  • Ejecución de las pruebas: 
    npx jest
