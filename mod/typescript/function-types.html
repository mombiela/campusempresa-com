Introducción

En TypeScript, las funciones son un bloque de construcción fundamental de cualquier aplicación. Entender cómo definir y usar tipos de funciones es crucial para escribir código robusto y seguro en términos de tipos. Esta sección cubrirá los conceptos básicos de los tipos de funciones, incluyendo cómo definirlos, usarlos y algunos conceptos avanzados.

Definiendo Tipos de Funciones

En TypeScript, puedes definir el tipo de una función especificando los tipos de sus parámetros y el tipo de su valor de retorno.

Sintaxis Básica

Aquí está la sintaxis básica para definir un tipo de función:

type GreetFunction = (name: string) => string;

En este ejemplo:

  • GreetFunction es un alias de tipo para una función que toma un parámetro string y devuelve un string.

Ejemplo

Vamos a crear una función que coincida con este tipo:

const greet: GreetFunction = (name: string) => {
    return `¡Hola, ${name}!`;
};

console.log(greet("Alice")); // Salida: ¡Hola, Alice!

Parámetros Opcionales y por Defecto

TypeScript te permite definir parámetros opcionales y por defecto en las funciones.

Parámetros Opcionales

Los parámetros opcionales se denotan con un ? después del nombre del parámetro.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age) => {
    if (age) {
        return `Hola, ${name}. Tienes ${age} años.`;
    }
    return `¡Hola, ${name}!`;
};

console.log(greet("Alice")); // Salida: ¡Hola, Alice!
console.log(greet("Bob", 30)); // Salida: Hola, Bob. Tienes 30 años.

Parámetros por Defecto

Los parámetros por defecto se asignan un valor por defecto en la firma de la función.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age = 25) => {
    return `Hola, ${name}. Tienes ${age} años.`;
};

console.log(greet("Alice")); // Salida: Hola, Alice. Tienes 25 años.
console.log(greet("Bob", 30)); // Salida: Hola, Bob. Tienes 30 años.

Tipos de Funciones en Interfaces

También puedes definir tipos de funciones dentro de interfaces.

Ejemplo

interface MathOperations {
    add: (a: number, b: number) => number;
    subtract: (a: number, b: number) => number;
}

const operations: MathOperations = {
    add: (a, b) => a + b,
    subtract: (a, b) => a - b,
};

console.log(operations.add(5, 3)); // Salida: 8
console.log(operations.subtract(5, 3)); // Salida: 2

Sobrecargas de Funciones

Las sobrecargas de funciones te permiten definir múltiples firmas para una sola función.

Ejemplo

function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
    return a + b;
}

console.log(add(5, 3)); // Salida: 8
console.log(add("Hola, ", "mundo!")); // Salida: Hola, mundo!

Conclusión

Entender los tipos de funciones en TypeScript es esencial para escribir código seguro en términos de tipos y mantenible. Al definir tipos de funciones, usar parámetros opcionales y por defecto, aprovechar las interfaces y utilizar sobrecargas de funciones, puedes crear funciones más robustas y flexibles. Practica estos conceptos con diferentes ejemplos para solidificar tu comprensión y volverte competente en los tipos de funciones de TypeScript.
