Introducción

Configurar el entorno de TypeScript es el primer paso para comenzar a desarrollar aplicaciones usando TypeScript. Esta sección te guiará a través del proceso de instalación de TypeScript, configuración de tu entorno de desarrollo y escritura de tu primer programa en TypeScript.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una computadora con un sistema operativo (Windows, macOS o Linux).
  • Node.js instalado en tu máquina. Puedes descargarlo desde nodejs.org.

Instalando TypeScript

Para instalar TypeScript, necesitas usar npm (Node Package Manager), que viene con Node.js. Abre tu terminal o símbolo del sistema y ejecuta el siguiente comando:

npm install -g typescript

Este comando instala TypeScript globalmente en tu sistema, haciendo que el comando tsc (TypeScript Compiler) esté disponible desde cualquier directorio.

Verificando la Instalación

Para verificar que TypeScript se ha instalado correctamente, ejecuta el siguiente comando en tu terminal:

tsc --version

Deberías ver el número de versión de TypeScript impreso en la pantalla, lo que indica que la instalación fue exitosa.

Configurando un Proyecto de TypeScript

  1. Crear un Directorio de Proyecto: Comienza creando un nuevo directorio para tu proyecto de TypeScript.

    mkdir my-typescript-project
cd my-typescript-project

  2. Inicializar un Proyecto de Node.js: Inicializa un nuevo proyecto de Node.js ejecutando:

    npm init -y

    Este comando crea un archivo package.json con configuraciones predeterminadas.

  3. Instalar TypeScript Localmente: Aunque instalamos TypeScript globalmente, es una buena práctica instalarlo localmente en tu proyecto también.

    npm install typescript --save-dev

  4. Crear un Archivo tsconfig.json: El archivo tsconfig.json se usa para configurar las opciones del compilador de TypeScript. Crea este archivo en la raíz de tu directorio de proyecto:

    tsc --init

    Este comando genera un archivo tsconfig.json con configuraciones predeterminadas.

Configurando tsconfig.json

El archivo tsconfig.json contiene varias opciones para personalizar el compilador de TypeScript. Aquí hay algunas opciones clave:

  • target: Especifica la versión de JavaScript a la que se compilará el código TypeScript.
  • module: Define el sistema de módulos a usar (por ejemplo, CommonJS, ES6).
  • outDir: Especifica el directorio de salida para los archivos JavaScript compilados.
  • rootDir: Especifica el directorio raíz de los archivos fuente de TypeScript.

Ejemplo de tsconfig.json:

{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true
  }
}

Escribiendo Tu Primer Programa en TypeScript

  1. Crear un Directorio de Fuente: Crea un directorio llamado src para almacenar tus archivos TypeScript.

    mkdir src

  2. Crear un Archivo TypeScript: Dentro del directorio src, crea un archivo llamado index.ts.

    // src/index.ts
let message: string = "Hello, TypeScript!";
console.log(message);

  3. Compilar el Archivo TypeScript: Usa el compilador de TypeScript para compilar tu código TypeScript a JavaScript.

    tsc

    Este comando lee el archivo tsconfig.json y compila los archivos TypeScript en el directorio src, generando los archivos JavaScript en el directorio dist.

  4. Ejecutar el JavaScript Compilado: Usa Node.js para ejecutar el archivo JavaScript compilado.

    node dist/index.js

    Deberías ver el mensaje "Hello, TypeScript!" impreso en la pantalla.

Conclusión

Configurar el entorno de TypeScript implica instalar TypeScript, configurar las opciones del compilador y escribir tu primer programa en TypeScript. Siguiendo estos pasos, has sentado las bases para desarrollar aplicaciones usando TypeScript. En las siguientes secciones, profundizaremos en las características y capacidades de TypeScript, construyendo sobre esta configuración inicial.
