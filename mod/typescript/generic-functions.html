Los genéricos en TypeScript te permiten crear componentes reutilizables que funcionan con una variedad de tipos en lugar de uno solo. Este tema cubrirá los conceptos básicos de las funciones genéricas, su sintaxis y ejemplos prácticos para ayudarte a entender cómo usarlas de manera efectiva.

Introducción a los Genéricos

Los genéricos proporcionan una forma de crear componentes flexibles y reutilizables. Te permiten definir funciones, clases e interfaces que funcionan con una variedad de tipos sin sacrificar la seguridad de tipos.

Conceptos Clave

  • Variables de Tipo: Marcadores de posición para los tipos que puedes especificar cuando usas la función genérica.
  • Inferencia de Tipos: TypeScript a menudo puede inferir los argumentos de tipo que pasas a una función genérica.
  • Restricciones: Puedes restringir los tipos que se pueden usar con los genéricos.

Sintaxis Básica de las Funciones Genéricas

Una función genérica usa variables de tipo para definir los tipos con los que puede trabajar. Estas variables de tipo se especifican dentro de corchetes angulares (<>) después del nombre de la función.

function identity<T>(arg: T): T {
    return arg;
}

En este ejemplo:

  • T es una variable de tipo.
  • identity es una función genérica que toma un argumento arg de tipo T y devuelve un valor de tipo T.

Uso de Funciones Genéricas

Puedes llamar a una función genérica con diferentes tipos:

let output1 = identity<string>("¡Hola, TypeScript!");
let output2 = identity<number>(42);

TypeScript también puede inferir el argumento de tipo:

let output3 = identity("¡Hola, TypeScript!"); // inferido como string
let output4 = identity(42); // inferido como number

Funciones Genéricas con Múltiples Variables de Tipo

Puedes definir funciones genéricas con múltiples variables de tipo:

function pair<T, U>(first: T, second: U): [T, U] {
    return [first, second];
}

let pair1 = pair<string, number>("Edad", 30);
let pair2 = pair<boolean, string>(true, "Éxito");

Restricciones en los Genéricos

A veces, necesitas restringir los tipos que se pueden usar con una función genérica. Puedes lograr esto usando restricciones.

interface Lengthwise {
    length: number;
}

function logLength<T extends Lengthwise>(arg: T): T {
    console.log(arg.length);
    return arg;
}

logLength("¡Hola, TypeScript!"); // Funciona, string tiene una propiedad length
logLength([1, 2, 3]); // Funciona, array tiene una propiedad length
// logLength(42); // Error: number no tiene una propiedad length

Ejemplos Prácticos

Función Genérica para Operaciones con Arrays

Un caso de uso común para los genéricos es crear funciones utilitarias para arrays.

function getFirstElement<T>(arr: T[]): T | undefined {
    return arr[0];
}

let firstString = getFirstElement<string>(["manzana", "banana", "cereza"]);
let firstNumber = getFirstElement<number>([1, 2, 3, 4, 5]);

Función Genérica con Tipo Predeterminado

Puedes proporcionar un tipo predeterminado para una función genérica.

function createArray<T = string>(length: number, value: T): T[] {
    return Array(length).fill(value);
}

let stringArray = createArray(3, "predeterminado"); // Inferido como string[]
let numberArray = createArray<number>(3, 42); // Explícitamente number[]

Conclusión

Las funciones genéricas en TypeScript son una característica poderosa que te permite escribir código flexible y reutilizable. Al entender la sintaxis básica, cómo usar múltiples variables de tipo y cómo aplicar restricciones, puedes crear funciones robustas que funcionen con una variedad de tipos. Practica usando genéricos en diferentes escenarios para sentirte más cómodo con este concepto y mejorar tus habilidades de programación en TypeScript.
