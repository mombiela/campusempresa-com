Funciones Flecha
Clases e Interfaces ►

Las funciones flecha son una forma concisa de escribir funciones en TypeScript. Son sintácticamente más cortas que las expresiones de función tradicionales y no enlazan su propio this, arguments, super o new.target. Esto las hace particularmente útiles en ciertos contextos, como dentro de métodos de arrays o cuando necesitas preservar el contexto de this.

Conceptos Clave

  • Sintaxis: Las funciones flecha usan la sintaxis =>.
  • this Léxico: Las funciones flecha no tienen su propio contexto this; heredan this del ámbito padre.
  • Retorno Implícito: Si el cuerpo de la función contiene solo una expresión, puedes omitir las llaves y la palabra clave return.

Sintaxis Básica

La sintaxis básica de una función flecha es:

(param1, param2, ..., paramN) => expresión

Si la función tiene solo un parámetro, puedes omitir los paréntesis:

param => expresión

Para funciones sin parámetros, debes usar paréntesis vacíos:

() => expresión

Ejemplos

Función Flecha Simple

Una función flecha simple que suma dos números:

const add = (a: number, b: number): number => a + b;

console.log(add(2, 3)); // Salida: 5

Función Flecha con Cuerpo de Bloque

Si necesitas más de una sola expresión, puedes usar un cuerpo de bloque:

const multiply = (a: number, b: number): number => {
    return a * b;
};

console.log(multiply(2, 3)); // Salida: 6

Función Flecha sin Parámetros

Una función flecha que devuelve un valor constante:

const getFive = (): number => 5;

console.log(getFive()); // Salida: 5

`this` Léxico

Las funciones flecha no tienen su propio contexto this. En su lugar, heredan this del ámbito padre. Esto es particularmente útil en situaciones donde quieres preservar el contexto de this.

Ejemplo con `this` Léxico

Considera el siguiente ejemplo:

class Counter {
    count: number = 0;

    increment() {
        setTimeout(() => {
            this.count++;
            console.log(this.count);
        }, 1000);
    }
}

const counter = new Counter();
counter.increment(); // Salida después de 1 segundo: 1

En este ejemplo, la función flecha dentro de setTimeout hereda this del método increment, que se refiere a la instancia de Counter.

Comparación con Funciones Tradicionales

| Característica | Función Flecha | Función Tradicional | |---------------------|---------------------------------|---------------------------------| | Sintaxis | param => expresión | function(param) { return expresión; } | | Contexto this | Léxico (hereda del padre) | Dinámico (depende de cómo se llame) | | Objeto arguments | No disponible | Disponible | | Constructor | No puede ser usado como constructor | Puede ser usado como constructor |

Ejercicios

  1. Convertir a Función Flecha: Convierte la siguiente función tradicional a una función flecha.

    function greet(name: string): string {
    return `Hello, ${name}!`;
}

  2. this Léxico: Crea una clase con un método que use una función flecha para registrar una propiedad de la clase después de un retraso.

  3. Métodos de Array: Usa una función flecha con el método map para crear un nuevo array donde cada elemento esté duplicado.

    const numbers = [1, 2, 3, 4];
// Tu código aquí

Conclusión

Las funciones flecha proporcionan una forma concisa y poderosa de escribir funciones en TypeScript. Son particularmente útiles para preservar el contexto de this y para escribir funciones cortas de una sola expresión. Entender cómo y cuándo usar funciones flecha puede ayudarte a escribir código más limpio y mantenible.
