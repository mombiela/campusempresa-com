Introducción a los Hooks

Los hooks son funciones que te permiten "engancharte" a las características de estado y ciclo de vida de React desde componentes funcionales. Fueron introducidos en React 16.8 y se han convertido en una parte fundamental del desarrollo moderno de React. Al usar TypeScript, puedes aprovechar su sistema de tipos para hacer tus hooks más robustos y mantenibles.

Hooks Básicos con TypeScript

useState

El hook useState te permite agregar estado a tus componentes funcionales. Aquí tienes cómo usarlo con TypeScript:

import React, { useState } from 'react';

const Counter: React.FC = () => {
  const [count, setCount] = useState<number>(0);

  return (
    <div>
      <p>Has hecho clic {count} veces</p>
      <button onClick={() => setCount(count + 1)}>Haz clic en mí</button>
    </div>
  );
};

export default Counter;
  • Conceptos Clave:
    • useState toma un estado inicial y devuelve un array con el estado actual y una función para actualizarlo.
    • TypeScript puede inferir el tipo a partir del estado inicial, pero también puedes definirlo explícitamente.

useEffect

El hook useEffect te permite realizar efectos secundarios en componentes funcionales. Aquí tienes un ejemplo con TypeScript:

import React, { useState, useEffect } from 'react';

const Timer: React.FC = () => {
  const [seconds, setSeconds] = useState<number>(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setSeconds(prevSeconds => prevSeconds + 1);
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return <div>Segundos: {seconds}</div>;
};

export default Timer;
  • Conceptos Clave:
    • useEffect se ejecuta después del renderizado y puede opcionalmente limpiarse a sí mismo.
    • El segundo argumento de useEffect es un array de dependencias que controla cuándo se ejecuta el efecto.

Hooks Avanzados con TypeScript

useReducer

El hook useReducer es útil para manejar lógica de estado compleja. Aquí tienes un ejemplo:

import React, { useReducer } from 'react';

interface State {
  count: number;
}

type Action = { type: 'increment' } | { type: 'decrement' };

const initialState: State = { count: 0 };

function reducer(state: State, action: Action): State {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}

const Counter: React.FC = () => {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Cuenta: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
    </div>
  );
};

export default Counter;
  • Conceptos Clave:
    • useReducer es similar a useState pero es más adecuado para lógica de estado compleja.
    • La función reductora toma el estado actual y una acción, y devuelve un nuevo estado.

Hooks Personalizados

Los hooks personalizados te permiten extraer y reutilizar lógica en múltiples componentes. Aquí tienes un ejemplo:

import { useState, useEffect } from 'react';

function useFetch<T>(url: string): { data: T | null, loading: boolean, error: Error | null } {
  const [data, setData] = useState<T | null>(null);
  const [loading, setLoading] = useState<boolean>(true);
  const [error, setError] = useState<Error | null>(null);

  useEffect(() => {
    fetch(url)
      .then(response => response.json())
      .then(data => {
        setData(data);
        setLoading(false);
      })
      .catch(error => {
        setError(error);
        setLoading(false);
      });
  }, [url]);

  return { data, loading, error };
}

export default useFetch;
  • Conceptos Clave:
    • Los hooks personalizados son funciones de JavaScript cuyos nombres comienzan con "use".
    • Pueden llamar a otros hooks y encapsular lógica reutilizable.

Conclusión

Los hooks son una característica poderosa en React que te permiten usar estado y otras características de React sin escribir una clase. Cuando se combinan con TypeScript, se vuelven aún más poderosos al proporcionar seguridad de tipos y mejor soporte de herramientas. Al entender y usar hooks como useState, useEffect, useReducer y hooks personalizados, puedes escribir aplicaciones de React más mantenibles y robustas.
