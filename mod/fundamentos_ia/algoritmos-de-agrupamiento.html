Introducción

El agrupamiento o clustering es una técnica de aprendizaje no supervisado utilizada en inteligencia artificial y aprendizaje automático para agrupar un conjunto de objetos de tal manera que los objetos en el mismo grupo (o clúster) sean más similares entre sí que los objetos en otros grupos. Esta técnica es fundamental para el análisis exploratorio de datos y tiene aplicaciones en diversas áreas como la segmentación de clientes, la detección de anomalías y la compresión de datos.

Conceptos Clave

  • Agrupamiento (Clustering): Proceso de dividir un conjunto de datos en grupos o clústeres, donde los datos dentro de cada grupo son más similares entre sí que a los datos de otros grupos.
  • Distancia o Similitud: Medida utilizada para determinar qué tan similares o diferentes son dos objetos. Las métricas comunes incluyen la distancia euclidiana, la distancia de Manhattan y la similitud del coseno.
  • Centroides: Puntos que representan el centro de un clúster en algoritmos como K-means.
  • Algoritmos de Agrupamiento: Métodos específicos utilizados para realizar el agrupamiento, como K-means, DBSCAN y el agrupamiento jerárquico.

Algoritmos de Agrupamiento Comunes

K-means

K-means es uno de los algoritmos de agrupamiento más populares y fáciles de entender. El objetivo es dividir un conjunto de datos en K clústeres, donde cada clúster está representado por el centroide de los puntos en ese clúster.

Pasos del algoritmo K-means:

  1. Seleccionar K puntos iniciales como centroides.
  2. Asignar cada punto de datos al clúster cuyo centroide esté más cercano.
  3. Recalcular los centroides como el promedio de los puntos asignados a cada clúster.
  4. Repetir los pasos 2 y 3 hasta que los centroides no cambien significativamente.
from sklearn.cluster import KMeans
import numpy as np

# Datos de ejemplo
X = np.array([[1, 2], [1, 4], [1, 0],
              [4, 2], [4, 4], [4, 0]])

# Crear el modelo K-means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Predecir los clústeres
labels = kmeans.labels_

# Obtener los centroides
centroids = kmeans.cluster_centers_

print("Labels:", labels)
print("Centroids:", centroids)

DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

DBSCAN es un algoritmo de agrupamiento basado en densidad que puede encontrar clústeres de forma arbitraria y manejar ruido (puntos que no pertenecen a ningún clúster).

Pasos del algoritmo DBSCAN:

  1. Seleccionar un punto aleatorio no visitado.
  2. Recuperar todos los puntos densamente conectados a ese punto.
  3. Si el punto es un punto central, formar un clúster.
  4. Si el punto es un punto de borde, marcarlo como ruido.
  5. Repetir hasta que todos los puntos hayan sido visitados.
from sklearn.cluster import DBSCAN

# Datos de ejemplo
X = np.array([[1, 2], [2, 2], [2, 3],
              [8, 7], [8, 8], [25, 80]])

# Crear el modelo DBSCAN
dbscan = DBSCAN(eps=3, min_samples=2).fit(X)

# Predecir los clústeres
labels = dbscan.labels_

print("Labels:", labels)

Agrupamiento Jerárquico

El agrupamiento jerárquico construye una jerarquía de clústeres mediante un enfoque aglomerativo (de abajo hacia arriba) o divisivo (de arriba hacia abajo).

Pasos del algoritmo aglomerativo:

  1. Comenzar con cada punto como un clúster individual.
  2. Combinar los dos clústeres más cercanos en un solo clúster.
  3. Repetir el paso 2 hasta que todos los puntos estén en un solo clúster.
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# Datos de ejemplo
X = np.array([[1, 2], [2, 2], [2, 3],
              [8, 7], [8, 8], [25, 80]])

# Crear el modelo de agrupamiento jerárquico
Z = linkage(X, 'ward')

# Dibujar el dendrograma
plt.figure()
dendrogram(Z)
plt.show()

Comparación de Algoritmos

| Algoritmo | Ventajas | Desventajas | |-----------------|--------------------------------------------------|--------------------------------------------------| | K-means | Simple, rápido, fácil de entender | Sensible a la selección de K, no maneja bien el ruido | | DBSCAN | Maneja bien el ruido, encuentra clústeres arbitrarios | Sensible a los parámetros eps y min_samples | | Jerárquico | No requiere predefinir el número de clústeres | Computacionalmente costoso para grandes datasets |

Conclusión

Los algoritmos de agrupamiento son herramientas poderosas para el análisis exploratorio de datos en inteligencia artificial. Cada algoritmo tiene sus propias ventajas y desventajas, y la elección del algoritmo adecuado depende de la naturaleza de los datos y el problema específico a resolver. Es fundamental comprender las características y limitaciones de cada método para aplicarlos de manera efectiva en la práctica.
