Configuración de Fuentes de Datos
Creación y Uso de Repositorios ►

En esta sección, exploraremos cómo configurar fuentes de datos en Spring Boot. Este es un aspecto crucial de cualquier aplicación que interactúa con una base de datos. Cubriremos los conceptos básicos de la configuración de una fuente de datos, la configuración de múltiples fuentes de datos y configuraciones avanzadas.

Introducción a las Fuentes de Datos

Una fuente de datos es una fábrica de conexiones a una fuente de datos física, como una base de datos. En Spring Boot, configurar una fuente de datos es sencillo gracias a sus capacidades de auto-configuración.

Conceptos Clave

  • DataSource: Una interfaz que proporciona una forma estándar de conectarse a una base de datos.
  • JdbcTemplate: Una utilidad de Spring que simplifica las operaciones JDBC.
  • EntityManager: Una interfaz de JPA utilizada para interactuar con el contexto de persistencia.

Configuración Básica de la Fuente de Datos

Spring Boot simplifica las configuraciones de base de datos proporcionando configuraciones y propiedades predeterminadas. Comencemos con una configuración básica.

Propiedades de la Aplicación

Spring Boot usa application.properties o application.yml para configurar la fuente de datos. Aquí hay un ejemplo usando application.properties:

spring.datasource.url=jdbc:mysql://localhost:3306/mydb
spring.datasource.username=root
spring.datasource.password=secret
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

Ejemplo de Código

También puedes configurar la fuente de datos programáticamente en una clase @Configuration:

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.boot.jdbc.DataSourceBuilder;
import javax.sql.DataSource;

@Configuration
public class DataSourceConfig {

    @Bean
    public DataSource dataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/mydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }
}

Configuración de Múltiples Fuentes de Datos

En algunas aplicaciones, es posible que necesites conectarte a múltiples bases de datos. Spring Boot te permite configurar múltiples fuentes de datos fácilmente.

Propiedades de la Aplicación

Define múltiples fuentes de datos en application.properties:

# Fuente de Datos Primaria
spring.datasource.primary.url=jdbc:mysql://localhost:3306/primarydb
spring.datasource.primary.username=root
spring.datasource.primary.password=secret
spring.datasource.primary.driver-class-name=com.mysql.cj.jdbc.Driver

# Fuente de Datos Secundaria
spring.datasource.secondary.url=jdbc:mysql://localhost:3306/secondarydb
spring.datasource.secondary.username=root
spring.datasource.secondary.password=secret
spring.datasource.secondary.driver-class-name=com.mysql.cj.jdbc.Driver

Ejemplo de Código

Crea una clase de configuración para definir múltiples fuentes de datos:

import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.boot.jdbc.DataSourceBuilder;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.Primary;

import javax.sql.DataSource;

@Configuration
public class MultiDataSourceConfig {

    @Primary
    @Bean(name = "primaryDataSource")
    public DataSource primaryDataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/primarydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }

    @Bean(name = "secondaryDataSource")
    public DataSource secondaryDataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/secondarydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }
}

Configuración Avanzada de la Fuente de Datos

Para configuraciones avanzadas, es posible que necesites personalizar el agrupamiento de conexiones, la gestión de transacciones o usar JPA.

Agrupamiento de Conexiones

Spring Boot soporta varias bibliotecas de agrupamiento de conexiones como HikariCP, Tomcat y DBCP2. HikariCP es el predeterminado.

Ejemplo con HikariCP

spring.datasource.hikari.maximum-pool-size=10
spring.datasource.hikari.minimum-idle=5
spring.datasource.hikari.idle-timeout=30000
spring.datasource.hikari.pool-name=MyHikariCP

Configuración de JPA

Si estás usando JPA, necesitas configurar el EntityManagerFactory y el TransactionManager.

Ejemplo de Configuración

import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.boot.orm.jpa.EntityManagerFactoryBuilder;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.data.jpa.repository.config.EnableJpaRepositories;
import org.springframework.orm.jpa.JpaTransactionManager;
import org.springframework.orm.jpa.LocalContainerEntityManagerFactoryBean;
import org.springframework.transaction.PlatformTransactionManager;

import javax.persistence.EntityManagerFactory;
import javax.sql.DataSource;

@Configuration
@EnableJpaRepositories(
        basePackages = "com.example.repository",
        entityManagerFactoryRef = "entityManagerFactory",
        transactionManagerRef = "transactionManager"
)
public class JpaConfig {

    @Bean(name = "entityManagerFactory")
    public LocalContainerEntityManagerFactoryBean entityManagerFactory(
            EntityManagerFactoryBuilder builder, @Qualifier("primaryDataSource") DataSource dataSource) {
        return builder
                .dataSource(dataSource)
                .packages("com.example.model")
                .persistenceUnit("primary")
                .build();
    }

    @Bean(name = "transactionManager")
    public PlatformTransactionManager transactionManager(
            @Qualifier("entityManagerFactory") EntityManagerFactory entityManagerFactory) {
        return new JpaTransactionManager(entityManagerFactory);
    }
}

Conclusión

Configurar fuentes de datos en Spring Boot es una habilidad fundamental para cualquier desarrollador que trabaje con bases de datos. Cubrimos:

  • Configuración básica de la fuente de datos usando application.properties y configuración programática.
  • Configuración de múltiples fuentes de datos.
  • Configuraciones avanzadas como el agrupamiento de conexiones y la configuración de JPA.

Entender estas configuraciones te permitirá construir aplicaciones Spring Boot robustas y escalables que interactúen eficientemente con bases de datos.
