Configuración de TCP/IP
Siguiente ►

Introducción

La configuración de TCP/IP en OpenVMS es esencial para habilitar la comunicación en red y permitir que el sistema interactúe con otros dispositivos y servicios en una red TCP/IP. Este módulo cubrirá los conceptos básicos, los pasos de configuración y algunos comandos útiles para gestionar TCP/IP en OpenVMS.

Conceptos Clave

Antes de comenzar con la configuración, es importante entender algunos conceptos clave:

  • TCP/IP (Transmission Control Protocol/Internet Protocol): Es un conjunto de protocolos de comunicación utilizados para interconectar dispositivos en una red.
  • Dirección IP: Una dirección única asignada a cada dispositivo en una red.
  • Máscara de Subred: Utilizada para dividir una red IP en subredes más pequeñas.
  • Puerta de Enlace (Gateway): Un dispositivo que actúa como un punto de acceso a otra red.
  • DNS (Domain Name System): Un sistema que traduce nombres de dominio en direcciones IP.

Pasos de Configuración

1. Verificar la Instalación de TCP/IP Services

Primero, asegúrate de que los servicios de TCP/IP estén instalados en tu sistema OpenVMS. Puedes verificar esto con el siguiente comando:

$ @SYS$MANAGER:TCPIP$CONFIG

Si los servicios no están instalados, deberás instalarlos desde el medio de instalación de OpenVMS.

2. Configurar la Dirección IP y la Máscara de Subred

Para configurar la dirección IP y la máscara de subred, sigue estos pasos:

  1. Ejecuta el comando de configuración de TCP/IP:

    $ @SYS$MANAGER:TCPIP$CONFIG

  2. Selecciona la opción para configurar interfaces de red:

    TCPIP> INTERFACE

  3. Selecciona la interfaz de red que deseas configurar (por ejemplo, WE0):

    TCPIP> SET INTERFACE WE0 /HOST=192.168.1.10 /NETWORK_MASK=255.255.255.0

3. Configurar la Puerta de Enlace

Para configurar la puerta de enlace predeterminada, utiliza el siguiente comando:

$ @SYS$MANAGER:TCPIP$CONFIG

Luego, selecciona la opción para configurar la puerta de enlace:

TCPIP> GATEWAY

Y establece la puerta de enlace:

TCPIP> SET GATEWAY 192.168.1.1

4. Configurar el Servidor DNS

Para configurar el servidor DNS, sigue estos pasos:

  1. Ejecuta el comando de configuración de TCP/IP:

    $ @SYS$MANAGER:TCPIP$CONFIG

  2. Selecciona la opción para configurar el cliente DNS:

    TCPIP> CLIENT

  3. Establece el servidor DNS:

    TCPIP> SET NAME_SERVICE /DOMAIN=example.com /SERVERS=8.8.8.8,8.8.4.4

5. Guardar y Aplicar la Configuración

Después de realizar todas las configuraciones, guarda y aplica los cambios:

TCPIP> EXIT

Comandos Útiles

Aquí hay algunos comandos útiles para gestionar y verificar la configuración de TCP/IP en OpenVMS:

  • Mostrar la configuración de la interfaz de red:

    $ TCPIP SHOW INTERFACE

  • Mostrar la configuración de la puerta de enlace:

    $ TCPIP SHOW GATEWAY

  • Mostrar la configuración del servidor DNS:

    $ TCPIP SHOW NAME_SERVICE

  • Reiniciar los servicios de TCP/IP:

    $ @SYS$STARTUP:TCPIP$RESTART

Ejercicio Práctico

Ejercicio 1: Configuración Básica de TCP/IP

  1. Verifica si los servicios de TCP/IP están instalados en tu sistema.
  2. Configura la dirección IP 192.168.1.20 y la máscara de subred 255.255.255.0 para la interfaz WE0.
  3. Establece la puerta de enlace predeterminada en 192.168.1.1.
  4. Configura los servidores DNS 8.8.8.8 y 8.8.4.4.
  5. Verifica la configuración utilizando los comandos de muestra.

Solución

  1. Verificar los servicios de TCP/IP:

    $ @SYS$MANAGER:TCPIP$CONFIG

  2. Configurar la dirección IP y la máscara de subred:

    $ @SYS$MANAGER:TCPIP$CONFIG
TCPIP> INTERFACE
TCPIP> SET INTERFACE WE0 /HOST=192.168.1.20 /NETWORK_MASK=255.255.255.0

  3. Configurar la puerta de enlace:

    $ @SYS$MANAGER:TCPIP$CONFIG
TCPIP> GATEWAY
TCPIP> SET GATEWAY 192.168.1.1

  4. Configurar el servidor DNS:

    $ @SYS$MANAGER:TCPIP$CONFIG
TCPIP> CLIENT
TCPIP> SET NAME_SERVICE /DOMAIN=example.com /SERVERS=8.8.8.8,8.8.4.4

  5. Verificar la configuración:

    $ TCPIP SHOW INTERFACE
$ TCPIP SHOW GATEWAY
$ TCPIP SHOW NAME_SERVICE

Conclusión

En esta sección, hemos cubierto los pasos básicos para configurar TCP/IP en OpenVMS, incluyendo la configuración de la dirección IP, la máscara de subred, la puerta de enlace y los servidores DNS. También hemos proporcionado algunos comandos útiles para gestionar y verificar la configuración. Con esta base, estarás preparado para configurar y gestionar la conectividad de red en tu sistema OpenVMS.
