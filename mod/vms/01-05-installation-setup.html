Instalación y Configuración
Siguiente ►

En esta sección, aprenderás cómo instalar y configurar OpenVMS en tu sistema. Este proceso es fundamental para poder trabajar con OpenVMS y aprovechar todas sus funcionalidades. A continuación, se detallan los pasos necesarios para llevar a cabo una instalación exitosa.

Requisitos Previos

Antes de comenzar con la instalación, asegúrate de cumplir con los siguientes requisitos:

  1. Hardware Compatible: Verifica que tu hardware sea compatible con OpenVMS. Consulta la documentación oficial para obtener una lista de hardware soportado.
  2. Medios de Instalación: Necesitarás los medios de instalación de OpenVMS, que pueden ser en formato de CD/DVD o una imagen ISO.
  3. Acceso a la Consola del Sistema: Asegúrate de tener acceso a la consola del sistema donde se realizará la instalación.

Pasos de Instalación

  1. Preparación del Medio de Instalación

Si tienes una imagen ISO, deberás grabarla en un CD/DVD o crear una unidad USB de arranque. Aquí te mostramos cómo hacerlo en ambos casos:

Crear un CD/DVD de Arranque

  1. Windows:

    • Inserta un CD/DVD en blanco en tu unidad de disco.
    • Usa un software de grabación de discos como ImgBurn o CDBurnerXP.
    • Selecciona la opción para grabar una imagen ISO y elige la imagen de OpenVMS.

  2. Linux:

    • Inserta un CD/DVD en blanco en tu unidad de disco.
    • Abre una terminal y usa el comando wodim: 
      wodim -v -eject openvms.iso

Crear una Unidad USB de Arranque

  1. Windows:

    • Usa una herramienta como Rufus.
    • Selecciona tu unidad USB y la imagen ISO de OpenVMS.
    • Configura el esquema de partición y el sistema de archivos según las recomendaciones de OpenVMS.

  2. Linux:

    • Inserta tu unidad USB.
    • Abre una terminal y usa el comando dd: 
      sudo dd if=openvms.iso of=/dev/sdX bs=4M status=progress && sync
    • Reemplaza /dev/sdX con la ruta de tu unidad USB.

  1. Iniciar el Proceso de Instalación

  1. Arrancar desde el Medio de Instalación:

    • Inserta el CD/DVD o conecta la unidad USB en el sistema donde deseas instalar OpenVMS.
    • Reinicia el sistema y accede al menú de arranque (generalmente presionando una tecla como F12, Esc, o Del durante el inicio).
    • Selecciona el medio de instalación como dispositivo de arranque.

  2. Pantalla de Bienvenida:

    • Una vez que el sistema arranque desde el medio de instalación, verás la pantalla de bienvenida de OpenVMS.
    • Sigue las instrucciones en pantalla para iniciar el proceso de instalación.

  1. Configuración Inicial

  1. Selección del Idioma y Teclado:

    • Selecciona el idioma y la configuración del teclado que prefieras.

  2. Configuración de la Red:

    • Configura la red si es necesario. Esto puede incluir la configuración de una dirección IP estática o el uso de DHCP.

  3. Particionado del Disco:

    • Selecciona el disco donde deseas instalar OpenVMS.
    • Crea las particiones necesarias. OpenVMS generalmente requiere una partición de sistema y una partición de datos.

  1. Instalación del Sistema

  1. Copia de Archivos:

    • El instalador comenzará a copiar los archivos necesarios al disco seleccionado. Este proceso puede tardar varios minutos.

  2. Configuración del Sistema:

    • Configura el nombre del sistema y otros parámetros básicos.
    • Establece una contraseña para el usuario administrador (SYSTEM).

  1. Finalización de la Instalación

  1. Reinicio del Sistema:

    • Una vez que la instalación haya finalizado, se te pedirá que reinicies el sistema.
    • Retira el medio de instalación antes de reiniciar.

  2. Primer Inicio:

    • Después del reinicio, el sistema arrancará en OpenVMS.
    • Inicia sesión con el usuario administrador (SYSTEM) y la contraseña que configuraste.

Configuración Post-Instalación

  1. Actualización del Sistema

Es recomendable actualizar el sistema para asegurarte de tener las últimas mejoras y correcciones de seguridad.

$ @SYS$UPDATE:VMSINSTAL

  1. Configuración de Usuarios

Crea usuarios adicionales según sea necesario.

$ SET DEFAULT SYS$SYSTEM
$ RUN AUTHORIZE
UAF> ADD username /PASSWORD=password /DEVICE=device /DIRECTORY=directory
UAF> EXIT

  1. Configuración de Servicios

Configura los servicios que necesites, como servidores web, bases de datos, etc.

$ @SYS$STARTUP:TCPIP$STARTUP

Ejercicio Práctico

Ejercicio 1: Instalación de OpenVMS en una Máquina Virtual

  1. Objetivo: Instalar OpenVMS en una máquina virtual utilizando VirtualBox.
  2. Instrucciones:
    • Descarga e instala VirtualBox.
    • Crea una nueva máquina virtual con las siguientes especificaciones:
      • Tipo: Other
      • Versión: Other/Unknown (64-bit)
      • Memoria: 2048 MB
      • Disco duro: 20 GB
    • Monta la imagen ISO de OpenVMS como unidad óptica.
    • Sigue los pasos de instalación descritos anteriormente.

Solución

  1. Crear Máquina Virtual:

    • Abre VirtualBox y haz clic en "Nueva".
    • Configura el nombre y el tipo de sistema operativo.
    • Asigna la memoria y crea un disco duro virtual.

  2. Montar ISO y Arrancar:

    • Ve a la configuración de la máquina virtual y monta la imagen ISO en la unidad óptica.
    • Inicia la máquina virtual y sigue los pasos de instalación.

Conclusión

En esta sección, hemos cubierto los pasos necesarios para instalar y configurar OpenVMS en tu sistema. Desde la preparación del medio de instalación hasta la configuración post-instalación, ahora deberías estar listo para comenzar a trabajar con OpenVMS. En el próximo módulo, exploraremos los comandos básicos de OpenVMS y cómo utilizarlos para gestionar archivos, procesos y el sistema en general.
