Introducción

Las funciones y métodos son bloques de construcción fundamentales en F#. Permiten encapsular lógica, promover la reutilización de código y mejorar la legibilidad. Esta sección cubrirá los conceptos básicos de la definición y uso de funciones, así como temas más avanzados como funciones de orden superior, recursión y métodos en programación orientada a objetos.

Funciones Básicas

Las funciones en F# son ciudadanos de primera clase, lo que significa que pueden ser pasadas como argumentos, devueltas de otras funciones y asignadas a variables.

Definiendo una Función

Una función simple puede ser definida usando la palabra clave let.

let add x y = x + y
  • let: Palabra clave usada para definir una función.
  • add: Nombre de la función.
  • x y y: Parámetros de la función.
  • x + y: Cuerpo de la función.

Llamando a una Función

Puedes llamar a una función usando su nombre seguido de argumentos.

let result = add 3 5
printfn "El resultado es %d" result

Función con Múltiples Líneas

Si una función tiene múltiples líneas, usa la indentación para definir el alcance.

let greet name =
    let message = "Hola, " + name
    printfn "%s" message

greet "Alice"

Funciones de Orden Superior

Las funciones de orden superior son funciones que toman otras funciones como parámetros o devuelven funciones como resultados.

Ejemplo: Función `map`

La función List.map aplica una función dada a cada elemento de una lista.

let square x = x * x
let numbers = [1; 2; 3; 4]
let squaredNumbers = List.map square numbers
printfn "%A" squaredNumbers

Recursión

La recursión es una técnica donde una función se llama a sí misma para resolver un problema.

Ejemplo: Función Factorial

Un ejemplo clásico de recursión es la función factorial.

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "El factorial de 5 es %d" (factorial 5)
  • rec: Palabra clave usada para definir una función recursiva.
  • factorial: Nombre de la función.
  • n: Parámetro de la función.
  • if n <= 1 then 1: Caso base.
  • else n * factorial (n - 1): Caso recursivo.

Métodos en Programación Orientada a Objetos

F# soporta la programación orientada a objetos, y los métodos son funciones definidas dentro de una clase u objeto.

Definiendo una Clase con Métodos

Aquí hay un ejemplo de una clase simple con métodos.

type Person(name: string, age: int) =
    member this.Name = name
    member this.Age = age
    member this.Greet() = printfn "Hola, mi nombre es %s y tengo %d años." this.Name this.Age

let alice = Person("Alice", 30)
alice.Greet()
  • type: Palabra clave usada para definir una clase.
  • Person: Nombre de la clase.
  • name y age: Parámetros del constructor.
  • member: Palabra clave usada para definir un método o propiedad.
  • this: Referencia a la instancia actual de la clase.

Resumen

En esta sección, cubrimos los conceptos básicos de funciones y métodos en F#. Comenzamos con definiciones y llamadas de funciones simples, pasamos a funciones de orden superior y recursión, y concluimos con métodos en programación orientada a objetos. Entender estos conceptos es crucial para escribir código eficiente y reutilizable en F#. Practica estos conceptos con varios ejemplos para solidificar tu comprensión.
