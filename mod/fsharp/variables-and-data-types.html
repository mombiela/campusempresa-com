Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y tipos de datos en F#. Comprender estos conceptos básicos es crucial para escribir programas F# efectivos y eficientes. Cubriremos:

  • Declaración de variables
  • Variables inmutables y mutables
  • Tipos de datos básicos
  • Inferencia de tipos
  • Tipos de datos personalizados

Declaración de Variables

En F#, las variables se declaran usando la palabra clave let. Por defecto, las variables en F# son inmutables, lo que significa que sus valores no pueden cambiar una vez asignados.

Ejemplo:

let x = 10
let y = "Hola, F#"

En el ejemplo anterior:

  • x es una variable entera con un valor de 10.
  • y es una variable de cadena con un valor de "Hola, F#".

Variables Inmutables y Mutables

Variables Inmutables

Como se mencionó, las variables declaradas con let son inmutables.

Ejemplo:

let a = 5
// a <- 10  // Esto causará un error porque 'a' es inmutable

Variables Mutables

Si necesitas una variable cuyo valor pueda cambiar, puedes usar la palabra clave mutable.

Ejemplo:

let mutable b = 5
b <- 10  // Esto está permitido porque 'b' es mutable

Tipos de Datos Básicos

F# soporta varios tipos de datos básicos. Aquí hay algunos de los más comúnmente usados:

Tipo de Dato Descripción Ejemplo
int Entero let num = 42
float Número de punto flotante let pi = 3.14
bool Booleano let isTrue = true
string Cadena let greeting = "Hola"
char Carácter let letter = 'A'

Ejemplo:

let age: int = 25
let price: float = 19.99
let isActive: bool = true
let name: string = "Alicia"
let initial: char = 'A'

Inferencia de Tipos

F# tiene un sistema de inferencia de tipos poderoso que a menudo puede deducir el tipo de una variable a partir de su valor, por lo que no siempre necesitas especificar el tipo explícitamente.

Ejemplo:

let inferredInt = 100  // F# infiere esto como int
let inferredString = "Inferencia de Tipos"  // F# infiere esto como string

Tipos de Datos Personalizados

F# te permite definir tus propios tipos de datos usando la palabra clave type. Puedes crear tipos de registros, tipos de unión y más.

Tipos de Registros

Los tipos de registros se usan para agrupar valores relacionados.

Ejemplo:

type Person = {
    Name: string
    Age: int
}

let person1 = { Name = "Juan"; Age = 30 }

Tipos de Unión

Los tipos de unión se usan para representar un valor que puede ser uno de varios tipos diferentes.

Ejemplo:

type Shape =
    | Circle of float
    | Rectangle of float * float

let circle = Circle 5.0
let rectangle = Rectangle (4.0, 6.0)

Conclusión

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en F#. Aprendimos cómo declarar variables, la diferencia entre variables inmutables y mutables, y exploramos tipos de datos básicos y personalizados. Comprender estos conceptos es esencial para escribir programas robustos en F#. En la próxima sección, profundizaremos en las funciones y cómo usarlas efectivamente en F#.
