Listas y Arrays en F#

Introducción

En F#, las listas y los arrays son estructuras de datos fundamentales que te permiten almacenar colecciones de elementos. Entender cómo usar estas estructuras de manera efectiva es crucial para cualquier programador de F#. Esta sección cubrirá los conceptos básicos de las listas y los arrays, sus diferencias y cómo manipularlos.

Listas

Las listas en F# son colecciones inmutables y ordenadas de elementos del mismo tipo. Se definen usando corchetes y puntos y comas para separar los elementos.

Creación de Listas

Para crear una lista, puedes usar la siguiente sintaxis:

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Operaciones Comunes con Listas

  • Acceder a Elementos: Usa la función List.item o la sintaxis .[index].
  • Agregar Elementos: Usa el operador @ para concatenar listas.
  • Mapeo: Usa la función List.map para aplicar una función a cada elemento.
let firstElement = List.item 0 numbers
let secondElement = numbers.[1]
let combinedList = numbers @ [6; 7; 8]
let squaredNumbers = List.map (fun x -> x * x) numbers

Coincidencia de Patrones con Listas

La coincidencia de patrones es una característica poderosa en F# que te permite descomponer listas fácilmente.

let rec sumList lst =
    match lst with
    | [] -> 0
    | head :: tail -> head + sumList tail

let total = sumList numbers

Arrays

Los arrays en F# son colecciones mutables y de tamaño fijo de elementos del mismo tipo. Se definen usando corchetes y barras verticales.

Creación de Arrays

Para crear un array, puedes usar la siguiente sintaxis:

let numbersArray = [| 1; 2; 3; 4; 5 |]
let fruitsArray = [| "apple"; "banana"; "cherry" |]

Operaciones Comunes con Arrays

  • Acceder a Elementos: Usa la sintaxis .[index].
  • Modificar Elementos: Asigna directamente un nuevo valor a un índice.
  • Mapeo: Usa la función Array.map para aplicar una función a cada elemento.
let firstElementArray = numbersArray.[0]
numbersArray.[1] <- 10
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray

Array vs Lista

| Característica | Lista (Inmutable) | Array (Mutable) | |-----------------|-------------------|-----------------| | Mutabilidad | Inmutable | Mutable | | Tamaño | Dinámico | Fijo | | Rendimiento | Más lento para actualizaciones | Más rápido para actualizaciones | | Sintaxis | [1; 2; 3] | [| 1; 2; 3 |] |

Temas Avanzados

Funciones de Orden Superior

Tanto las listas como los arrays soportan funciones de orden superior como map, filter y fold.

let evenNumbers = List.filter (fun x -> x % 2 == 0) numbers
let sumOfArray = Array.fold (+) 0 numbersArray

Procesamiento Paralelo con Arrays

Los arrays pueden ser procesados en paralelo usando el módulo Array.Parallel para mejoras en el rendimiento.

let squaredNumbersParallel = Array.Parallel.map (fun x -> x * x) numbersArray

Conclusión

Las listas y los arrays son estructuras de datos esenciales en F#. Las listas son inmutables y de tamaño dinámico, lo que las hace adecuadas para paradigmas de programación funcional. Los arrays son mutables y de tamaño fijo, ofreciendo un mejor rendimiento para ciertas operaciones. Entender cuándo y cómo usar estas estructuras mejorará enormemente tu capacidad para escribir código F# eficiente y efectivo.
