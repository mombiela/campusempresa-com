Introducción

F# es un lenguaje de programación funcional-primero que también soporta programación orientada a objetos e imperativa. Entender la sintaxis y estructura básica de F# es crucial para escribir código eficiente y legible. Esta sección cubrirá los elementos fundamentales de la sintaxis y estructura de F#, proporcionando una base sólida para un aprendizaje más profundo.

Conceptos Clave

Comentarios

  • Comentarios de una sola línea: Usa // para comentar una sola línea.
  • Comentarios de múltiples líneas: Usa (* ... *) para comentar múltiples líneas.
// Este es un comentario de una sola línea

(* Este es un 
   comentario de múltiples líneas *)

Variables y Valores

  • Valores inmutables: Definidos usando la palabra clave let.
  • Valores mutables: Definidos usando la palabra clave let mutable.
let x = 10  // Valor inmutable
let mutable y = 20  // Valor mutable
y <- 30  // Cambiando el valor de y

Tipos de Datos Básicos

  • Entero: int
  • Punto flotante: float
  • Booleano: bool
  • Cadena: string
let a: int = 5
let b: float = 3.14
let c: bool = true
let d: string = "Hola, F#"

Funciones

  • Definidas usando la palabra clave let.
  • Pueden ser anidadas y soportan recursión.
let add x y = x + y  // Función simple

let rec factorial n =  // Función recursiva
    if n <= 1 then 1
    else n * factorial (n - 1)

Expresiones Condicionales

  • Usa if ... then ... else para lógica condicional.
let isEven n =
    if n % 2 = 0 then
        true
    else
        false

Coincidencia de Patrones

  • Característica poderosa para descomponer tipos de datos.
let describeNumber n =
    match n with
    | 0 -> "Cero"
    | 1 -> "Uno"
    | _ -> "Otro"

Listas

  • Colecciones inmutables de elementos.
  • Creado usando corchetes [].
let numbers = [1; 2; 3; 4; 5]
let firstNumber = List.head numbers
let remainingNumbers = List.tail numbers

Tuplas

  • Agrupan múltiples valores en un solo valor compuesto.
  • Creado usando paréntesis ().
let person = ("Juan", 30)
let name, age = person

Registros

  • Definen tipos de datos personalizados con campos nombrados.
type Person = { Name: string; Age: int }
let john = { Name = "Juan"; Age = 30 }

Uniones Discriminadas

  • Definen un tipo que puede ser uno de varios casos nombrados.
type Shape =
    | Circle of float
    | Rectangle of float * float

let area shape =
    match shape with
    | Circle radius -> 3.14 * radius * radius
    | Rectangle (width, height) -> width * height

Conclusión

Entender la sintaxis y estructura básica de F# es esencial para escribir código efectivo. Esta sección cubrió los elementos fundamentales, incluyendo comentarios, variables, tipos de datos, funciones, expresiones condicionales, coincidencia de patrones, listas, tuplas, registros y uniones discriminadas. Con estos conceptos básicos, estás bien equipado para profundizar en características y paradigmas más avanzados de la programación en F#.
