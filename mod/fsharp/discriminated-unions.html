Uniones Discriminadas en F#
Tipos Opción y Manejo de Errores ►

Las Uniones Discriminadas (DUs) son una característica poderosa en F# que te permiten definir un tipo que puede contener uno de varios tipos de valores diferentes, pero fijos. Son particularmente útiles para modelar datos que pueden tomar múltiples formas. En este tema, exploraremos qué son las Uniones Discriminadas, cómo definirlas y usarlas, y algunas técnicas avanzadas.

¿Qué son las Uniones Discriminadas?

Las Uniones Discriminadas son una forma de definir un tipo que puede ser uno de varios casos nombrados, cada uno potencialmente con diferentes tipos y valores. Son similares a los enums en otros lenguajes, pero son más poderosas porque cada caso puede llevar datos adicionales.

Conceptos Clave

  • Caso: Cada valor posible del tipo de unión.
  • Etiqueta: El nombre del caso.
  • Carga Útil: Los datos asociados con un caso (opcional).

Definiendo Uniones Discriminadas

Para definir una Unión Discriminada en F#, usas la palabra clave type seguida del nombre de la unión y los casos.

type Shape =
    | Circle of float
    | Rectangle of float * float
    | Triangle of float * float * float

En este ejemplo:

  • Shape es el tipo de unión.
  • Circle, Rectangle y Triangle son los casos.
  • Cada caso tiene datos asociados (por ejemplo, Circle tiene un float que representa el radio).

Usando Uniones Discriminadas

Una vez definidas, puedes crear instancias de una Unión Discriminada y usar coincidencia de patrones para trabajar con ellas.

Creando Instancias

let myCircle = Circle(5.0)
let myRectangle = Rectangle(4.0, 6.0)
let myTriangle = Triangle(3.0, 4.0, 5.0)

Coincidencia de Patrones

La coincidencia de patrones te permite descomponer y trabajar con los valores de una Unión Discriminada.

let describeShape shape =
    match shape with
    | Circle(radius) -> sprintf "Un círculo con radio %f" radius
    | Rectangle(width, height) -> sprintf "Un rectángulo con ancho %f y alto %f" width height
    | Triangle(a, b, c) -> sprintf "Un triángulo con lados %f, %f, %f" a b c

let description = describeShape myCircle
printfn "%s" description

Técnicas Avanzadas

Uniones Discriminadas Anidadas

Puedes anidar Uniones Discriminadas dentro de otras Uniones Discriminadas para modelar estructuras de datos más complejas.

type Point = { X: float; Y: float }

type Shape =
    | Circle of float
    | Rectangle of float * float
    | Triangle of float * float * float
    | Polygon of Point list

Usando Tipos Opción

Las Uniones Discriminadas a menudo se usan con el tipo Option para representar valores que pueden o no estar presentes.

type Result =
    | Success of string
    | Error of string

let processResult result =
    match result with
    | Success(message) -> printfn "Éxito: %s" message
    | Error(errorMessage) -> printfn "Error: %s" errorMessage

let result = Success("Operación completada con éxito")
processResult result

Uniones Discriminadas Recursivas

Las Uniones Discriminadas pueden ser recursivas, lo cual es útil para representar estructuras tipo árbol.

type BinaryTree =
    | Leaf of int
    | Node of BinaryTree * int * BinaryTree

let rec sumTree tree =
    match tree with
    | Leaf(value) -> value
    | Node(left, value, right) -> sumTree(left) + value + sumTree(right)

let myTree = Node(Node(Leaf(1), 2, Leaf(3)), 4, Leaf(5))
let total = sumTree myTree
printfn "Suma del árbol: %d" total

Conclusión

Las Uniones Discriminadas son una característica versátil y poderosa en F# que te permiten definir tipos con múltiples casos distintos. Son útiles para modelar datos que pueden tomar diferentes formas y para implementar coincidencia de patrones. Al entender y utilizar las Uniones Discriminadas, puedes escribir código F# más expresivo y robusto.
