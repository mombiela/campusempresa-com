Introducción

En F#, las secuencias y colecciones son estructuras de datos fundamentales que te permiten almacenar y manipular grupos de datos. Esta sección cubrirá los conceptos básicos de secuencias, listas, arreglos y otros tipos de colecciones, proporcionándote el conocimiento para usarlos eficazmente en tus programas.

Listas

Las listas son colecciones inmutables de elementos del mismo tipo. Son uno de los tipos de colecciones más comúnmente usados en F#.

Creación de Listas

Puedes crear listas usando corchetes y separando los elementos con punto y coma.

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Operaciones Básicas

  • Acceso a Elementos: Usa la función List.item o el operador ..
  • Agregar Elementos: Usa el operador :: para anteponer elementos.
  • Concatenar Listas: Usa el operador @.
let firstElement = List.item 0 numbers
let newList = 0 :: numbers
let combinedList = numbers @ [6; 7; 8]

Funciones Comunes

  • List.map: Aplica una función a cada elemento.
  • List.filter: Filtra elementos basado en un predicado.
  • List.fold: Agrega elementos usando una función.
let squaredNumbers = List.map (fun x -> x * x) numbers
let evenNumbers = List.filter (fun x -> x % 2 = 0) numbers
let sum = List.fold (+) 0 numbers

Arreglos

Los arreglos son colecciones mutables de elementos del mismo tipo, permitiendo acceso y actualizaciones aleatorias eficientes.

Creación de Arreglos

Puedes crear arreglos usando corchetes y separando los elementos con punto y coma, o usando la función Array.init.

let numbersArray = [|1; 2; 3; 4; 5|]
let initializedArray = Array.init 5 (fun i -> i * 2)

Operaciones Básicas

  • Acceso a Elementos: Usa el operador . con un índice.
  • Actualización de Elementos: Usa el operador <- con un índice.
let firstElementArray = numbersArray.[0]
numbersArray.[0] <- 10

Funciones Comunes

  • Array.map: Aplica una función a cada elemento.
  • Array.filter: Filtra elementos basado en un predicado.
  • Array.fold: Agrega elementos usando una función.
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray
let evenNumbersArray = Array.filter (fun x -> x % 2 = 0) numbersArray
let sumArray = Array.fold (+) 0 numbersArray

Secuencias

Las secuencias son colecciones perezosas que se evalúan bajo demanda, haciéndolas adecuadas para trabajar con conjuntos de datos grandes o infinitos.

Creación de Secuencias

Puedes crear secuencias usando la palabra clave seq o usando expresiones de secuencia.

let numbersSeq = seq {1 .. 5}
let infiniteSeq = Seq.initInfinite (fun i -> i * 2)

Operaciones Básicas

  • Acceso a Elementos: Usa la función Seq.item.
  • Conversión a Otras Colecciones: Usa funciones como Seq.toList o Seq.toArray.
let firstElementSeq = Seq.item 0 numbersSeq
let numbersList = Seq.toList numbersSeq
let numbersArrayFromSeq = Seq.toArray numbersSeq

Funciones Comunes

  • Seq.map: Aplica una función a cada elemento.
  • Seq.filter: Filtra elementos basado en un predicado.
  • Seq.fold: Agrega elementos usando una función.
let squaredNumbersSeq = Seq.map (fun x -> x * x) numbersSeq
let evenNumbersSeq = Seq.filter (fun x -> x % 2 = 0) numbersSeq
let sumSeq = Seq.fold (+) 0 numbersSeq

Otras Colecciones

F# también proporciona otros tipos de colecciones como conjuntos y mapas, que ofrecen propiedades y casos de uso únicos.

Conjuntos

Los conjuntos son colecciones de elementos únicos.

let numbersSet = Set.ofList [1; 2; 3; 4; 5; 5]
let isMember = Set.contains 3 numbersSet

Mapas

Los mapas son colecciones de pares clave-valor.

let numbersMap = Map.ofList [(1, "one"); (2, "two"); (3, "three")]
let value = Map.find 2 numbersMap

Conclusión

En esta sección, exploramos varias secuencias y colecciones en F#, incluyendo listas, arreglos, secuencias, conjuntos y mapas. Entender estas estructuras de datos y sus operaciones es crucial para una programación efectiva en F#. Practica usando estas colecciones en diferentes escenarios para volverte competente en la gestión y manipulación de datos en tus aplicaciones F#.
