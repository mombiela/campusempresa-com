Introducción

Las pruebas y la depuración son aspectos cruciales del desarrollo de software. Aseguran que tu código funcione como se espera y ayudan a identificar y corregir problemas. En esta sección, cubriremos los conceptos básicos de las pruebas y la depuración en F#, desde niveles principiantes hasta avanzados.

Pruebas Unitarias en F#

¿Qué son las Pruebas Unitarias?

  • Las pruebas unitarias implican probar componentes o funciones individuales de tu código para asegurarse de que funcionen correctamente.
  • Ayudan a detectar errores temprano y hacen que tu código sea más mantenible.

Configuración de un Entorno de Pruebas

  • Usa el marco xUnit para pruebas unitarias en F#.
  • Instala los paquetes NuGet xUnit y xUnit.runner.visualstudio.

Escribiendo tu Primera Prueba

open Xunit

module MyTests =

    [<Fact>]
    let ``Addition Test`` () =
        let result = 2 + 2
        Assert.Equal(4, result)
  • open Xunit importa la biblioteca xUnit.
  • El atributo [<Fact>] marca el método como una prueba.
  • Assert.Equal verifica si el resultado esperado coincide con el resultado real.

Ejecutando Pruebas

  • Usa Visual Studio Test Explorer o el comando dotnet test para ejecutar tus pruebas.

Pruebas Unitarias Avanzadas

Pruebas Parametrizadas

  • Usa los atributos [<Theory>] y [<InlineData>] para pruebas parametrizadas.
open Xunit

module MyTests =

    [<Theory>]
    [<InlineData(2, 2, 4)>]
    [<InlineData(3, 3, 6)>]
    let ``Addition Test`` (a: int, b: int, expected: int) =
        let result = a + b
        Assert.Equal(expected, result)

Simulación de Dependencias

  • Usa la biblioteca Foq para simular dependencias.
  • Instala el paquete NuGet Foq.
open Foq
open Xunit

type ICalculator =
    abstract member Add: int -> int -> int

module MyTests =

    [<Fact>]
    let ``Mocking Test`` () =
        let mockCalculator = Mock<ICalculator>()
                              .Setup(fun x -> <@ x.Add(2, 2) @>)
                              .Returns(4)
                              .Create()
        let result = mockCalculator.Add(2, 2)
        Assert.Equal(4, result)

Depuración en F#

Uso del Depurador de Visual Studio

  • Establece puntos de interrupción haciendo clic en el margen junto al número de línea.
  • Usa F5 para iniciar la depuración y F10/F11 para avanzar sobre/entrar en el código.

Técnicas Comunes de Depuración

  • Declaraciones de Impresión: Usa printfn para imprimir valores de variables.
  • Ventana de Observación: Monitorea valores de variables en tiempo real.
  • Ventana Inmediata: Ejecuta fragmentos de código durante la depuración.

Manejo de Excepciones

  • Usa bloques try...with para manejar excepciones.
try
    // Código que podría lanzar una excepción
    let result = 10 / 0
    printfn "Result: %d" result
with
| :? System.DivideByZeroException as ex ->
    printfn "Exception: %s" ex.Message

Conclusión

Las pruebas y la depuración son habilidades esenciales para cualquier desarrollador de F#. Las pruebas unitarias aseguran que tu código funcione como se espera, mientras que la depuración te ayuda a identificar y corregir problemas. Al dominar estas técnicas, puedes escribir código más confiable y mantenible.
