Introducción al Desarrollo Web con F#

En esta sección, introduciremos los conceptos básicos del desarrollo web utilizando F#. Cubriremos los conceptos y herramientas fundamentales necesarios para comenzar.

  • ¿Qué es F#?

    • Un lenguaje de programación funcional-principal que también soporta programación orientada a objetos e imperativa.
    • Parte del ecosistema .NET.

  • ¿Por qué usar F# para el Desarrollo Web?

    • Sistema de tipos fuerte e inmutabilidad por defecto.
    • Sintaxis concisa y expresiva.
    • Interoperabilidad con otros lenguajes y bibliotecas de .NET.

Configuración de Tu Entorno

Antes de sumergirnos en el desarrollo web, necesitamos configurar nuestro entorno de desarrollo.

  • Instalación del SDK de .NET

    • Descarga e instala el SDK de .NET desde el sitio web oficial de Microsoft.

  • Configuración de un proyecto F#

    dotnet new console -lang F# -o MyWebApp
cd MyWebApp
dotnet run

  • Instalación de herramientas necesarias

    • Visual Studio Code o Visual Studio.
    • Extensión Ionide para soporte de F# en Visual Studio Code.

Construyendo un Servidor Web Simple

Comenzaremos construyendo un servidor web simple usando Suave, una biblioteca de servidor web ligero para F#.

  • Instalación de Suave

    dotnet add package Suave

  • Creación de un servidor web básico

    open Suave
open Suave.Successful

let app = OK "Hello, World!"

startWebServer defaultConfig app

  • Ejecución del servidor

    dotnet run

Manejo de Rutas y Solicitudes

En esta sección, aprenderemos cómo manejar diferentes rutas y solicitudes en nuestra aplicación web.

  • Definición de rutas

    open Suave
open Suave.Operators
open Suave.Filters
open Suave.Successful

let app =
  choose [
    GET >=> path "/" >=> OK "Home Page"
    GET >=> path "/about" >=> OK "About Page"
  ]

startWebServer defaultConfig app

  • Manejo de parámetros de consulta

    open Suave
open Suave.Filters
open Suave.Operators
open Suave.Successful

let app =
  choose [
    GET >=> pathScan "/hello/%s" (fun name -> OK (sprintf "Hello, %s!" name))
  ]

startWebServer defaultConfig app

Trabajando con Plantillas

Las plantillas son esenciales para renderizar contenido HTML dinámico. Usaremos la biblioteca Giraffe para este propósito.

  • Instalación de Giraffe

    dotnet add package Giraffe

  • Creación de una plantilla simple

    open Giraffe
open Giraffe.ViewEngine

let layout (content: XmlNode list) =
  html [] [
    head [] [ title [] [ str "My Web App" ] ]
    body [] content
  ]

let indexView =
  layout [
    h1 [] [ str "Welcome to My Web App" ]
    p [] [ str "This is a simple web application using F# and Giraffe." ]
  ]

let app = choose [ GET >=> route "/" >=> htmlView indexView ]

startWebServer defaultConfig app

Interactuando con Bases de Datos

En esta sección, cubriremos cómo interactuar con bases de datos usando F#.

  • Instalación de SQLProvider

    dotnet add package SQLProvider

  • Conexión a una base de datos

    open FSharp.Data.Sql

type sql = SqlDataProvider< 
              Common.DatabaseProviderTypes.MSSQLSERVER, 
              "Server=myServerAddress;Database=myDataBase;User Id=myUsername;Password=myPassword;" >

let ctx = sql.GetDataContext()

let customers = ctx.Dbo.Customers |> Seq.toList

  • Realización de operaciones CRUD

    // Crear
let newCustomer = ctx.Dbo.Customers.Create()
newCustomer.Name <- "John Doe"
ctx.SubmitUpdates()

// Leer
let customer = ctx.Dbo.Customers |> Seq.head

// Actualizar
customer.Name <- "Jane Doe"
ctx.SubmitUpdates()

// Eliminar
ctx.Dbo.Customers.DeleteOnSubmit(customer)
ctx.SubmitUpdates()

Temas Avanzados

En esta sección, cubriremos temas más avanzados como autenticación, middleware y despliegue de aplicaciones web F#.

  • Autenticación y Autorización

    • Uso de OAuth o JWT para autenticación segura.
    • Implementación de control de acceso basado en roles.

  • Middleware

    • Creación de middleware personalizado para registro, manejo de errores, etc.

  • Despliegue de Aplicaciones Web F#

    • Despliegue en Azure, AWS u otras plataformas en la nube.
    • Contenerización de tu aplicación usando Docker.

Conclusión

En este curso, cubrimos lo esencial del desarrollo web con F#. Comenzamos con la configuración del entorno, construyendo un servidor web simple, manejando rutas, trabajando con plantillas, interactuando con bases de datos y finalmente, temas avanzados como autenticación y despliegue. Con estas habilidades, ahora estás equipado para construir aplicaciones web robustas usando F#. ¡Feliz codificación!
