Introducción

Crear informes optimizados para móviles en Power BI asegura que tus datos sean accesibles y legibles en dispositivos móviles. Esta sección te guiará a través del proceso desde niveles principiantes hasta avanzados, cubriendo conceptos clave, ejemplos prácticos y ejercicios para solidificar tu comprensión.

Nivel Principiante

Entendiendo la Vista Móvil

  • Vista Móvil: Una característica en Power BI que te permite diseñar informes específicamente para dispositivos móviles.
  • Importancia: Asegura que los informes sean fáciles de usar y accesibles en pantallas más pequeñas.

Configurando la Vista Móvil

  1. Abre Power BI Desktop.
  2. Crea un Informe: Diseña tu informe como de costumbre.
  3. Cambia a la Disposición Móvil:
    • Ve a la pestaña Vista.
    • Haz clic en Disposición Móvil.

Diseñando para Móviles

  • Arrastrar y Soltar Visuales: Organiza los visuales en la disposición móvil arrastrándolos desde el panel de la derecha.
  • Redimensionar Visuales: Ajusta el tamaño de los visuales para que se adapten a la pantalla del móvil.
  • Optimizar Visuales: Asegúrate de que la información clave sea visible sin necesidad de desplazarse excesivamente.

Ejemplo

1. Crea un informe simple con un gráfico de barras y una tabla.
2. Cambia a la Disposición Móvil.
3. Arrastra el gráfico de barras y la tabla a la disposición móvil.
4. Redimensiónalos para que se ajusten a la pantalla.

Ejercicio

  1. Crea un informe con al menos tres visuales diferentes.
  2. Cambia a la Disposición Móvil y organiza los visuales para crear un informe móvil coherente.

Nivel Intermedio

Técnicas Avanzadas de Disposición Móvil

  • Marcadores: Usa marcadores para crear diferentes vistas para usuarios móviles.
  • Visuales Personalizados: Implementa visuales personalizados que estén optimizados para dispositivos móviles.

Uso de Marcadores

  1. Crear Marcadores:
    • Ve a la pestaña Vista.
    • Haz clic en Panel de Marcadores.
    • Añade un nuevo marcador para cada vista que quieras crear.
  2. Vincular Marcadores a Botones:
    • Inserta un botón.
    • Configura la acción para navegar al marcador deseado.

Ejemplo

1. Crea un informe con múltiples páginas.
2. Añade marcadores para cada página.
3. Inserta botones y vincúlalos a los marcadores.

Ejercicio

  1. Crea un informe de varias páginas.
  2. Usa marcadores para crear un sistema de navegación amigable para móviles.

Nivel Avanzado

Optimización del Rendimiento

  • Reducción de Datos: Reduce la cantidad de datos cargados para mejorar el rendimiento.
  • Consultas DAX Optimizadas: Escribe consultas DAX eficientes para acelerar la recuperación de datos.

Técnicas de Reducción de Datos

  • Agregaciones: Usa datos agregados para reducir el volumen de datos.
  • Filtros: Aplica filtros para limitar los datos mostrados en los visuales.

Ejemplo

1. Crea un informe con un gran conjunto de datos.
2. Aplica filtros para mostrar solo los datos necesarios.
3. Usa medidas agregadas en lugar de datos detallados.

Ejercicio

  1. Crea un informe con un gran conjunto de datos.
  2. Optimiza el informe para móviles reduciendo datos y mejorando el rendimiento.

Conclusión

Crear informes optimizados para móviles en Power BI implica entender la vista móvil, diseñar disposiciones específicamente para dispositivos móviles y optimizar el rendimiento. Siguiendo los pasos y ejercicios proporcionados, puedes asegurarte de que tus informes sean accesibles y eficientes en dispositivos móviles.
