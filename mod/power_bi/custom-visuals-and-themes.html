Introducción

En esta sección, exploraremos cómo mejorar tus informes de Power BI utilizando visuales y temas personalizados. Los visuales personalizados te permiten crear visualizaciones de datos únicas e interactivas, mientras que los temas te permiten aplicar un estilo consistente en tus informes.

Visuales Personalizados

¿Qué son los Visuales Personalizados?

Los visuales personalizados son tipos adicionales de visualización que puedes importar en Power BI para extender sus capacidades predeterminadas. Estos visuales pueden ser creados por la comunidad de Power BI o por desarrolladores utilizando el SDK de Visuales de Power BI.

Importar Visuales Personalizados

Para importar visuales personalizados en tu informe de Power BI:

  1. Ve al panel de Visualizaciones.
  2. Haz clic en los puntos suspensivos (...) y selecciona "Importar desde el marketplace" o "Importar desde archivo".
  3. Elige el visual personalizado deseado y agrégalo a tu informe.

Ejemplo: Usar un Visual Personalizado

Vamos a importar y usar un visual personalizado llamado "Nube de Palabras".

  1. Importar el Visual de Nube de Palabras:

    - Ve al panel de Visualizaciones.
- Haz clic en los puntos suspensivos (`...`) y selecciona "Importar desde el marketplace".
- Busca "Nube de Palabras" y haz clic en "Agregar".

  2. Agregar Datos a la Nube de Palabras:

    - Arrastra un campo de texto (por ejemplo, Nombres de Productos) al visual de Nube de Palabras.
- Ajusta la configuración para personalizar la apariencia.

Crear Visuales Personalizados

Para usuarios avanzados, crear visuales personalizados implica:

  • Configurar un entorno de desarrollo con Node.js y el SDK de Visuales de Power BI.
  • Escribir código TypeScript para definir el comportamiento del visual.
  • Empaquetar el visual e importarlo en Power BI.

Ejemplo: Código Básico de un Visual Personalizado

Aquí tienes un ejemplo simple de un visual personalizado usando TypeScript:

module powerbi.extensibility.visual {
    export class SimpleVisual implements IVisual {
        private target: HTMLElement;

        constructor(options: VisualConstructorOptions) {
            this.target = options.element;
        }

        public update(options: VisualUpdateOptions) {
            this.target.innerHTML = `<div style="color:blue;">¡Hola, Power BI!</div>`;
        }
    }
}
  • Explicación:
    • SimpleVisual es una clase que implementa la interfaz IVisual.
    • El constructor inicializa el visual.
    • El método update actualiza el visual con nuevos datos o configuraciones.

Temas

¿Qué son los Temas?

Los temas en Power BI te permiten aplicar una apariencia consistente a tus informes definiendo colores, fuentes y otros elementos visuales.

Aplicar un Tema

Para aplicar un tema a tu informe de Power BI:

  1. Ve a la pestaña Inicio.
  2. Haz clic en "Cambiar tema".
  3. Elige entre los temas disponibles o importa un archivo de tema personalizado.

Crear Temas Personalizados

Los temas personalizados se definen utilizando archivos JSON. Aquí tienes un ejemplo de un tema personalizado simple:

{
  "name": "Tema Personalizado",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#FFFFFF",
  "foreground": "#000000",
  "tableAccent": "#FF5733"
}
  • Explicación:
    • dataColors: Define los colores utilizados en los gráficos.
    • background: Establece el color de fondo del informe.
    • foreground: Establece el color del texto.
    • tableAccent: Define el color de acento para las tablas.

Ejemplo: Aplicar un Tema Personalizado

  1. Crear un archivo JSON con la definición del tema:

    {
  "name": "Mi Tema Personalizado",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#F0F0F0",
  "foreground": "#333333",
  "tableAccent": "#FF5733"
}

  2. Importar el Tema Personalizado:

    - Ve a la pestaña Inicio.
- Haz clic en "Cambiar tema" y selecciona "Importar tema".
- Elige el archivo JSON que creaste.

Conclusión

Los visuales y temas personalizados son herramientas poderosas en Power BI que te permiten crear informes más atractivos y visualmente impactantes. Al importar o crear visuales personalizados, puedes extender las capacidades de visualización predeterminadas de Power BI. Los temas ayudan a asegurar una apariencia consistente y profesional en tus informes. Experimenta con estas características para mejorar tus proyectos de Power BI y ofrecer insights más impactantes.
