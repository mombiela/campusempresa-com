Introducción

En esta sección, profundizaremos en las funciones avanzadas de DAX (Data Analysis Expressions) utilizadas en Power BI. Estas funciones te permiten realizar cálculos complejos y manipulaciones de datos, permitiéndote crear informes más perspicaces y dinámicos.

Conceptos Clave

  • CALCULATE: Modifica el contexto de un cálculo.
  • FILTER: Devuelve una tabla que representa un subconjunto de otra tabla.
  • ALL: Elimina filtros de una tabla o columna.
  • RELATED: Recupera valores de tablas relacionadas.
  • SUMX: Itera sobre una tabla para sumar expresiones.
  • EARLIER: Se refiere a un contexto de fila anterior.
  • RANKX: Clasifica elementos en una tabla basada en una expresión.

Función CALCULATE

La función CALCULATE es una de las funciones DAX más poderosas. Evalúa una expresión en un contexto de filtro modificado.

Sintaxis

CALCULATE(<expression>, <filter1>, <filter2>, ...)

Ejemplo

Total Sales = CALCULATE(SUM(Sales[Amount]), Sales[Region] = "North America")

Este ejemplo calcula el monto total de ventas para la región de América del Norte.

Función FILTER

La función FILTER devuelve una tabla que representa un subconjunto de otra tabla.

Sintaxis

FILTER(<table>, <filter_expression>)

Ejemplo

FilteredSales = FILTER(Sales, Sales[Amount] > 1000)

Este ejemplo filtra la tabla Sales para incluir solo filas donde el monto de ventas es mayor a 1000.

Función ALL

La función ALL elimina filtros de una tabla o columna.

Sintaxis

ALL(<table_or_column>)

Ejemplo

AllRegions = CALCULATE(SUM(Sales[Amount]), ALL(Sales[Region]))

Este ejemplo calcula el monto total de ventas para todas las regiones, ignorando cualquier filtro existente en la columna Region.

Función RELATED

La función RELATED recupera valores de tablas relacionadas.

Sintaxis

RELATED(<column>)

Ejemplo

ProductCategory = RELATED(Product[Category])

Este ejemplo recupera la categoría de un producto de la tabla relacionada Product.

Función SUMX

La función SUMX itera sobre una tabla para sumar expresiones.

Sintaxis

SUMX(<table>, <expression>)

Ejemplo

TotalProfit = SUMX(Sales, Sales[Amount] - Sales[Cost])

Este ejemplo calcula el beneficio total iterando sobre la tabla Sales y sumando la diferencia entre el monto y el costo.

Función EARLIER

La función EARLIER se refiere a un contexto de fila anterior.

Sintaxis

EARLIER(<column>, <number_of_outer_evaluations>)

Ejemplo

RunningTotal = 
CALCULATE(
    SUM(Sales[Amount]),
    FILTER(
        Sales,
        Sales[Date] <= EARLIER(Sales[Date])
    )
)

Este ejemplo calcula un total acumulado de los montos de ventas hasta la fecha de la fila actual.

Función RANKX

La función RANKX clasifica elementos en una tabla basada en una expresión.

Sintaxis

RANKX(<table>, <expression>, [<value>, <order>, <ties>])

Ejemplo

SalesRank = RANKX(ALL(Sales), Sales[Amount], , DESC, Dense)

Este ejemplo clasifica los montos de ventas en orden descendente, utilizando un método de clasificación densa.

Conclusión

Las funciones avanzadas de DAX como CALCULATE, FILTER, ALL, RELATED, SUMX, EARLIER y RANKX proporcionan herramientas poderosas para realizar manipulaciones y cálculos de datos complejos en Power BI. Dominar estas funciones te permitirá crear informes más dinámicos y perspicaces, mejorando tus capacidades de análisis de datos. Practica usando estas funciones con datos del mundo real para solidificar tu comprensión y mejorar tu competencia en Power BI.
