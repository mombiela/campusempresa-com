Introducción

Los filtros y segmentaciones son herramientas esenciales en Power BI que permiten a los usuarios refinar y controlar los datos mostrados en sus informes. Este tema te guiará desde lo básico hasta el uso avanzado de filtros y segmentaciones, asegurando que puedas manipular datos de manera efectiva para obtener conocimientos más profundos.

Filtros en Power BI

Tipos de Filtros

Power BI ofrece varios tipos de filtros:

  • Filtros a nivel de visualización: Se aplican a visualizaciones individuales.
  • Filtros a nivel de página: Se aplican a todas las visualizaciones en una sola página del informe.
  • Filtros a nivel de informe: Se aplican a todas las visualizaciones en todo el informe.

Aplicando Filtros

Para aplicar un filtro:

  1. Selecciona la visualización que deseas filtrar.
  2. En el panel de Filtros, arrastra el campo por el que deseas filtrar al área de filtro correspondiente (Visual, Página o Informe).
  3. Configura los ajustes del filtro (por ejemplo, selecciona valores específicos, establece rangos).

Ejemplo

Supongamos que tienes un informe de ventas y deseas filtrar los datos para mostrar solo las ventas del año 2022.

  1. Arrastra el campo Año al área de filtro a nivel de visualización.
  2. Selecciona 2022 de la lista de años.

Filtrado Avanzado

El filtrado avanzado permite condiciones más complejas:

  • Filtrado Básico: Selecciona valores específicos.
  • Filtrado Avanzado: Usa condiciones como "es mayor que", "es menor que", "contiene", etc.
  • Filtrado Top N: Muestra los N elementos principales basados en una medida.

Ejemplo de Filtrado Avanzado

Para filtrar los datos de ventas y mostrar solo transacciones mayores a $1000:

  1. Arrastra el campo Monto de Ventas al área de filtro.
  2. Elige Filtrado Avanzado.
  3. Establece la condición a es mayor que e ingresa 1000.
Filtros en esta visualización
------------------------------
Monto de Ventas
  - es mayor que 1000

Segmentaciones en Power BI

¿Qué son las Segmentaciones?

Las segmentaciones son herramientas visuales que permiten a los usuarios seleccionar y filtrar datos de manera interactiva. Proporcionan una forma fácil de usar para filtrar datos en la página del informe.

Agregando una Segmentación

  1. Ve al panel de Visualizaciones y selecciona el ícono de Segmentación.
  2. Arrastra un campo (por ejemplo, Categoría) a la segmentación.
  3. La segmentación aparecerá en el lienzo del informe, permitiendo a los usuarios seleccionar categorías para filtrar los datos.

Ejemplo

Para agregar una segmentación para categorías de productos:

  1. Selecciona el ícono de Segmentación en el panel de Visualizaciones.
  2. Arrastra el campo Categoría a la segmentación.
Segmentación de Categoría
--------------------------
[ ] Electrónica
[ ] Muebles
[ ] Ropa
[ ] Alimentos

Personalizando Segmentaciones

Las segmentaciones se pueden personalizar de varias maneras:

  • Orientación: Vertical, horizontal o desplegable.
  • Formato: Cambiar colores, fuentes y tamaños.
  • Controles de Selección: Selección única, selección múltiple y opciones de seleccionar todo.

Ejemplo de Personalización de una Segmentación

Para crear una segmentación horizontal con selección múltiple habilitada:

  1. Selecciona la segmentación.
  2. En el panel de Formato, establece la Orientación a Horizontal.
  3. Habilita Selección múltiple con CTRL.
Segmentación de Categoría (Horizontal)
---------------------------------------
[ ] Electrónica  [ ] Muebles  [ ] Ropa  [ ] Alimentos

Combinando Filtros y Segmentaciones

Los filtros y segmentaciones se pueden usar juntos para crear experiencias poderosas de exploración de datos. Por ejemplo, puedes usar una segmentación para filtrar categorías y un filtro a nivel de visualización para refinar aún más los datos por fecha.

Ejemplo

  1. Agrega una segmentación para Categoría.
  2. Aplica un filtro a nivel de visualización para mostrar datos solo del año 2022.

Conclusión

Los filtros y segmentaciones son herramientas poderosas en Power BI que permiten a los usuarios interactuar y refinar sus datos. Al comprender los diferentes tipos de filtros y cómo aplicarlos, así como cómo usar y personalizar efectivamente las segmentaciones, puedes crear informes dinámicos y perspicaces. Practica usando estas herramientas para volverte competente en la manipulación y visualización de datos en Power BI.
