Conectando a fuentes de datos

Siguiente ►
►

En este tema, aprenderás cómo conectar Power BI a diversas fuentes de datos. La capacidad de conectarse a múltiples fuentes de datos es una de las características más poderosas de Power BI, permitiéndote importar, transformar y visualizar datos de una amplia variedad de orígenes.

Objetivos de Aprendizaje

  • Comprender los tipos de fuentes de datos compatibles con Power BI.
  • Conectar Power BI a diferentes fuentes de datos.
  • Configurar las conexiones de datos y gestionar credenciales.

Tipos de Fuentes de Datos Compatibles

Power BI puede conectarse a una amplia gama de fuentes de datos, incluyendo:

  1. Archivos:

    • Excel
    • CSV
    • XML
    • JSON

  2. Bases de Datos:

    • SQL Server
    • Oracle
    • MySQL
    • PostgreSQL

  3. Servicios en la Nube:

    • Azure SQL Database
    • Google BigQuery
    • Amazon Redshift

  4. Servicios en Línea:

    • SharePoint Online
    • Dynamics 365
    • Google Analytics

  5. Otros:

    • Web (HTML, JSON)
    • OData Feed
    • R Scripts

Conectando a una Fuente de Datos

Paso 1: Abrir Power BI Desktop

  1. Inicia Power BI Desktop.
  2. En la pantalla de inicio, selecciona Obtener datos.

Paso 2: Seleccionar la Fuente de Datos

  1. En la ventana Obtener datos, verás una lista de categorías de fuentes de datos.
  2. Selecciona la categoría adecuada. Por ejemplo, para conectar a un archivo Excel, selecciona Archivo y luego Excel.

Paso 3: Configurar la Conexión

  1. Navega hasta la ubicación del archivo o ingresa la URL de la fuente de datos.
  2. Selecciona el archivo o ingresa las credenciales necesarias para acceder a la fuente de datos.
  3. Haz clic en Conectar.

Paso 4: Seleccionar los Datos

  1. Una vez establecida la conexión, Power BI mostrará una vista previa de los datos disponibles.
  2. Selecciona las tablas o elementos que deseas importar.
  3. Haz clic en Cargar para importar los datos directamente o en Transformar datos para abrir el Editor de Power Query y realizar transformaciones antes de cargar los datos.

Ejemplo Práctico: Conectando a un Archivo Excel

Paso a Paso

  1. Abrir Power BI Desktop:

    • Inicia Power BI Desktop.
    • Selecciona Obtener datos en la pantalla de inicio.

  2. Seleccionar la Fuente de Datos:

    • En la ventana Obtener datos, selecciona Archivo > Excel.

  3. Configurar la Conexión:

    • Navega hasta la ubicación del archivo Excel en tu computadora.
    • Selecciona el archivo y haz clic en Abrir.

  4. Seleccionar los Datos:

    • En la ventana Navegador, selecciona las hojas o tablas que deseas importar.
    • Haz clic en Cargar para importar los datos directamente o en Transformar datos para abrir el Editor de Power Query.

Ejercicio Práctico

Ejercicio 1: Conectar a un Archivo CSV

  1. Abre Power BI Desktop.
  2. Selecciona Obtener datos > Archivo > Texto/CSV.
  3. Navega hasta la ubicación de un archivo CSV en tu computadora y selecciónalo.
  4. Carga los datos y realiza cualquier transformación necesaria en el Editor de Power Query.

Solución

  1. Abrir Power BI Desktop:

    • Inicia Power BI Desktop.
    • Selecciona Obtener datos en la pantalla de inicio.

  2. Seleccionar la Fuente de Datos:

    • En la ventana Obtener datos, selecciona Archivo > Texto/CSV.

  3. Configurar la Conexión:

    • Navega hasta la ubicación del archivo CSV en tu computadora.
    • Selecciona el archivo y haz clic en Abrir.

  4. Seleccionar los Datos:

    • Revisa la vista previa de los datos.
    • Haz clic en Cargar para importar los datos directamente o en Transformar datos para abrir el Editor de Power Query.

Resumen

En esta lección, aprendiste cómo conectar Power BI a diversas fuentes de datos, incluyendo archivos, bases de datos y servicios en la nube. También exploraste un ejemplo práctico de conexión a un archivo Excel y realizaste un ejercicio práctico para conectar a un archivo CSV. Con estas habilidades, estás listo para importar y trabajar con datos en Power BI, lo que te permitirá crear informes y visualizaciones poderosas.

En el próximo módulo, profundizaremos en la transformación y modelado de datos utilizando Power Query.
