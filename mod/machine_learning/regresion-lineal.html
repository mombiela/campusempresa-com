Introducción a la Regresión Lineal

La regresión lineal es uno de los algoritmos más simples y ampliamente utilizados en Machine Learning. Se utiliza para modelar la relación entre una variable dependiente y una o más variables independientes. En este tema, aprenderemos los conceptos básicos de la regresión lineal, cómo implementarla y cómo evaluar su rendimiento.

Conceptos Clave

  • Variable Dependiente (Y): La variable que estamos tratando de predecir o explicar.
  • Variable Independiente (X): Las variables que utilizamos para hacer predicciones sobre la variable dependiente.
  • Coeficientes (β0, β1, ..., βn): Los parámetros que el modelo aprende durante el entrenamiento.
  • Error (ε): La diferencia entre el valor real y el valor predicho por el modelo.

Fórmula de la Regresión Lineal

La fórmula general de la regresión lineal es:

\[ Y = β0 + β1X1 + β2X2 + ... + βnXn + ε \]

Donde:

  • \( Y \) es la variable dependiente.
  • \( X1, X2, ..., Xn \) son las variables independientes.
  • \( β0 \) es el intercepto.
  • \( β1, β2, ..., βn \) son los coeficientes de las variables independientes.
  • \( ε \) es el error.

Implementación en Python

A continuación, se muestra un ejemplo de cómo implementar una regresión lineal simple utilizando Python y la biblioteca scikit-learn.

Ejemplo de Código

import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Datos de ejemplo
X = np.array([[1], [2], [3], [4], [5]])
Y = np.array([1, 3, 2, 5, 4])

# Crear el modelo de regresión lineal
model = LinearRegression()

# Entrenar el modelo
model.fit(X, Y)

# Hacer predicciones
Y_pred = model.predict(X)

# Visualizar los resultados
plt.scatter(X, Y, color='blue')  # Datos reales
plt.plot(X, Y_pred, color='red')  # Línea de regresión
plt.xlabel('Variable Independiente (X)')
plt.ylabel('Variable Dependiente (Y)')
plt.title('Regresión Lineal Simple')
plt.show()

Explicación del Código

  1. Importación de Bibliotecas: Importamos numpy para manejar los datos y matplotlib para la visualización. scikit-learn se utiliza para crear y entrenar el modelo de regresión lineal.
  2. Datos de Ejemplo: Creamos un conjunto de datos de ejemplo con una variable independiente X y una variable dependiente Y.
  3. Creación del Modelo: Utilizamos LinearRegression de scikit-learn para crear el modelo de regresión lineal.
  4. Entrenamiento del Modelo: Ajustamos el modelo a los datos utilizando el método fit.
  5. Predicciones: Utilizamos el modelo entrenado para hacer predicciones sobre los datos de entrada.
  6. Visualización: Graficamos los datos reales y la línea de regresión para visualizar el ajuste del modelo.

Evaluación del Modelo

Para evaluar el rendimiento de un modelo de regresión lineal, podemos utilizar varias métricas, como:

  • Coeficiente de Determinación (R²): Indica la proporción de la varianza en la variable dependiente que es predecible a partir de las variables independientes.
  • Error Cuadrático Medio (MSE): La media de los cuadrados de los errores, que mide la magnitud promedio de los errores.

Ejemplo de Evaluación

from sklearn.metrics import mean_squared_error, r2_score

# Calcular el MSE
mse = mean_squared_error(Y, Y_pred)

# Calcular el R²
r2 = r2_score(Y, Y_pred)

print(f"Mean Squared Error: {mse}")
print(f"R² Score: {r2}")

Conclusión

La regresión lineal es una técnica fundamental en Machine Learning que permite modelar relaciones lineales entre variables. A través de este tema, hemos aprendido los conceptos básicos, cómo implementar una regresión lineal en Python y cómo evaluar su rendimiento. Con esta base, estarás preparado para explorar modelos más complejos y técnicas avanzadas en Machine Learning.
