Introducción

La transformación de datos es un paso crucial en el proceso de Machine Learning. Consiste en convertir datos brutos en un formato adecuado para el análisis y la modelización. Este proceso puede incluir la limpieza de datos, la normalización, la codificación de variables categóricas, entre otros.

Importancia de la Transformación de Datos

  • Mejora la calidad de los datos: Elimina inconsistencias y errores.
  • Facilita la modelización: Transforma los datos en un formato que los algoritmos de Machine Learning pueden procesar eficientemente.
  • Aumenta la precisión del modelo: Datos bien transformados pueden mejorar significativamente el rendimiento del modelo.

Tipos de Transformaciones de Datos

Normalización y Estandarización

  • Normalización: Escala los datos para que estén en un rango específico, generalmente [0, 1].
  • Estandarización: Transforma los datos para que tengan una media de 0 y una desviación estándar de 1.
from sklearn.preprocessing import MinMaxScaler, StandardScaler

# Normalización
scaler = MinMaxScaler()
data_normalized = scaler.fit_transform(data)

# Estandarización
scaler = StandardScaler()
data_standardized = scaler.fit_transform(data)

Codificación de Variables Categóricas

  • One-Hot Encoding: Convierte variables categóricas en una serie de variables binarias.
  • Label Encoding: Asigna un número entero a cada categoría.
from sklearn.preprocessing import OneHotEncoder, LabelEncoder

# One-Hot Encoding
encoder = OneHotEncoder()
data_encoded = encoder.fit_transform(data_categorical)

# Label Encoding
encoder = LabelEncoder()
data_encoded = encoder.fit_transform(data_categorical)

Manejo de Valores Faltantes

  • Imputación: Rellena los valores faltantes con la media, mediana o moda.
  • Eliminación: Elimina filas o columnas con valores faltantes.
from sklearn.impute import SimpleImputer

# Imputación con la media
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data_with_missing_values)

Transformaciones Logarítmicas y de Potencia

  • Transformación Logarítmica: Utilizada para reducir la asimetría de la distribución de los datos.
  • Transformación de Potencia: Utilizada para estabilizar la varianza y hacer que los datos se asemejen más a una distribución normal.
import numpy as np

# Transformación Logarítmica
data_log_transformed = np.log(data)

# Transformación de Potencia
from sklearn.preprocessing import PowerTransformer
pt = PowerTransformer()
data_power_transformed = pt.fit_transform(data)

Ejemplo Completo de Transformación de Datos

import pandas as pd
from sklearn.preprocessing import MinMaxScaler, OneHotEncoder, SimpleImputer

# Cargar datos
data = pd.read_csv('data.csv')

# Manejo de valores faltantes
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data)

# Normalización
scaler = MinMaxScaler()
data_normalized = scaler.fit_transform(data_imputed)

# Codificación de variables categóricas
encoder = OneHotEncoder()
data_encoded = encoder.fit_transform(data_normalized)

# Convertir a DataFrame para facilitar la visualización
data_transformed = pd.DataFrame(data_encoded.toarray())
print(data_transformed.head())

Conclusión

La transformación de datos es un paso esencial en el pipeline de Machine Learning. Asegura que los datos estén en un formato adecuado para el análisis y la modelización, mejorando así la calidad y la precisión de los modelos. Al dominar las técnicas de transformación de datos, los profesionales pueden construir modelos más robustos y eficientes.
