Introducción

La asociación de reglas es una técnica de minería de datos que se utiliza para descubrir relaciones interesantes entre variables en grandes bases de datos. Es especialmente útil en el análisis de cestas de mercado, donde se busca identificar productos que se compran juntos con frecuencia.

Conceptos Clave

  • Itemset: Un conjunto de uno o más ítems.
  • Frecuencia de un Itemset: El número de transacciones en las que aparece un itemset.
  • Soporte (Support): La proporción de transacciones en la base de datos que contienen el itemset.
  • Confianza (Confidence): La proporción de transacciones que contienen el itemset A que también contienen el itemset B.
  • Lift: La razón entre la confianza observada y la confianza esperada si A y B fueran independientes.

Ejemplo de Asociación de Reglas

Para ilustrar estos conceptos, consideremos una base de datos de transacciones en un supermercado:

| Transacción | Ítems Comprados | |-------------|-----------------------------| | 1 | Leche, Pan, Mantequilla | | 2 | Leche, Pan | | 3 | Leche, Mantequilla | | 4 | Pan, Mantequilla | | 5 | Leche, Pan, Mantequilla, Queso |

Cálculo del Soporte

El soporte de un itemset se calcula como:

\[ \text{Soporte}(A) = \frac{\text{Número de transacciones que contienen A}}{\text{Número total de transacciones}} \]

Por ejemplo, el soporte del itemset {Leche, Pan} es:

\[ \text{Soporte}({Leche, Pan}) = \frac{3}{5} = 0.6 \]

Cálculo de la Confianza

La confianza de una regla A → B se calcula como:

\[ \text{Confianza}(A \rightarrow B) = \frac{\text{Número de transacciones que contienen A y B}}{\text{Número de transacciones que contienen A}} \]

Por ejemplo, la confianza de la regla {Leche} → {Pan} es:

\[ \text{Confianza}({Leche} \rightarrow {Pan}) = \frac{3}{4} = 0.75 \]

Cálculo del Lift

El lift de una regla A → B se calcula como:

\[ \text{Lift}(A \rightarrow B) = \frac{\text{Confianza}(A \rightarrow B)}{\text{Soporte}(B)} \]

Por ejemplo, el lift de la regla {Leche} → {Pan} es:

\[ \text{Lift}({Leche} \rightarrow {Pan}) = \frac{0.75}{0.8} = 0.9375 \]

Implementación en Python

A continuación, se muestra un ejemplo de cómo implementar la asociación de reglas utilizando la biblioteca mlxtend en Python.

# Importar las bibliotecas necesarias
import pandas as pd
from mlxtend.frequent_patterns import apriori, association_rules

# Crear un DataFrame con las transacciones
data = {'Leche': [1, 1, 1, 0, 1],
        'Pan': [1, 1, 0, 1, 1],
        'Mantequilla': [1, 0, 1, 1, 1],
        'Queso': [0, 0, 0, 0, 1]}
df = pd.DataFrame(data)

# Aplicar el algoritmo Apriori para encontrar itemsets frecuentes
frequent_itemsets = apriori(df, min_support=0.6, use_colnames=True)

# Generar las reglas de asociación
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)

# Mostrar las reglas
print(rules)

Explicación del Código

  1. Importación de Bibliotecas: Se importan pandas para la manipulación de datos y mlxtend para la implementación del algoritmo Apriori.
  2. Creación del DataFrame: Se crea un DataFrame con las transacciones, donde cada columna representa un ítem y cada fila una transacción.
  3. Algoritmo Apriori: Se aplica el algoritmo Apriori para encontrar itemsets frecuentes con un soporte mínimo de 0.6.
  4. Generación de Reglas: Se generan las reglas de asociación con una confianza mínima de 0.7.
  5. Mostrar Reglas: Se imprimen las reglas generadas.

Conclusión

La asociación de reglas es una técnica poderosa para descubrir relaciones ocultas en grandes conjuntos de datos. A través de conceptos como soporte, confianza y lift, podemos identificar patrones significativos que pueden ser utilizados para tomar decisiones informadas en diversas aplicaciones, como la venta al por menor y el marketing. La implementación en Python con bibliotecas como mlxtend facilita el proceso de descubrimiento de estas reglas, permitiendo a los profesionales del sector aplicar estas técnicas de manera eficiente.
