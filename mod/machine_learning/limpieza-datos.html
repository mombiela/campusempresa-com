Introducción

La limpieza de datos es una etapa crucial en el proceso de Machine Learning. Consiste en identificar y corregir errores, inconsistencias y valores faltantes en los datos para mejorar la calidad y la precisión de los modelos predictivos. Este proceso asegura que los datos sean adecuados para el análisis y la construcción de modelos.

Importancia de la Limpieza de Datos

  • Mejora la calidad del modelo: Datos limpios y precisos conducen a modelos más fiables y precisos.
  • Reducción de errores: Minimiza la presencia de valores atípicos y errores que pueden distorsionar los resultados.
  • Eficiencia: Facilita el proceso de análisis y modelado, ahorrando tiempo y recursos.

Pasos en la Limpieza de Datos

1. Identificación de Valores Faltantes

  • Detección: Identificar las columnas y filas que contienen valores faltantes.
  • Tratamiento:
    • Eliminación: Quitar filas o columnas con valores faltantes.
    • Imputación: Rellenar valores faltantes con la media, mediana, moda o mediante técnicas más avanzadas como KNN.
import pandas as pd
from sklearn.impute import SimpleImputer

# Cargar datos
data = pd.read_csv('data.csv')

# Identificar valores faltantes
print(data.isnull().sum())

# Imputación con la media
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data)

2. Detección y Tratamiento de Valores Atípicos

  • Detección: Utilizar métodos estadísticos como el rango intercuartílico (IQR) o visualizaciones como boxplots.
  • Tratamiento:
    • Eliminación: Quitar los valores atípicos.
    • Transformación: Aplicar transformaciones para reducir el impacto de los valores atípicos.
import numpy as np

# Detección de valores atípicos usando IQR
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1

# Filtrar valores atípicos
data_filtered = data[~((data < (Q1 - 1.5 * IQR)) | (data > (Q3 + 1.5 * IQR))).any(axis=1)]

3. Normalización y Escalado de Datos

  • Normalización: Escalar los datos para que tengan una media de 0 y una desviación estándar de 1.
  • Escalado: Ajustar los datos para que estén en un rango específico, como [0, 1].
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# Normalización
scaler = StandardScaler()
data_normalized = scaler.fit_transform(data)

# Escalado
scaler = MinMaxScaler()
data_scaled = scaler.fit_transform(data)

4. Manejo de Datos Duplicados

  • Detección: Identificar filas duplicadas en el conjunto de datos.
  • Eliminación: Quitar las filas duplicadas para evitar redundancias.
# Detección de duplicados
duplicates = data.duplicated()

# Eliminación de duplicados
data_cleaned = data.drop_duplicates()

Herramientas y Librerías Comunes

  • Pandas: Para manipulación y análisis de datos.
  • NumPy: Para operaciones numéricas y manejo de matrices.
  • Scikit-learn: Para preprocesamiento y modelado de datos.

Ejemplo Completo de Limpieza de Datos

import pandas as pd
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import StandardScaler

# Cargar datos
data = pd.read_csv('data.csv')

# Identificar y tratar valores faltantes
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data)

# Detección y tratamiento de valores atípicos
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1
data_filtered = data[~((data < (Q1 - 1.5 * IQR)) | (data > (Q3 + 1.5 * IQR))).any(axis=1)]

# Normalización
scaler = StandardScaler()
data_normalized = scaler.fit_transform(data_filtered)

# Manejo de duplicados
data_cleaned = pd.DataFrame(data_normalized).drop_duplicates()

Conclusión

La limpieza de datos es un paso esencial en el proceso de Machine Learning que no debe ser subestimado. A través de la identificación y corrección de valores faltantes, atípicos y duplicados, así como la normalización y escalado de datos, podemos asegurar que nuestros modelos sean más precisos y eficientes. Utilizar herramientas y librerías adecuadas facilita este proceso y mejora la calidad de nuestros análisis y predicciones.
