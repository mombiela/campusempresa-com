Introducción a SQL

SQL (Structured Query Language) es un lenguaje de programación utilizado para gestionar y manipular bases de datos relacionales. Es una herramienta esencial para los analistas de datos, ya que permite extraer, filtrar y transformar datos de manera eficiente.

Conceptos Básicos de SQL

  • Bases de Datos Relacionales: Conjunto de tablas relacionadas entre sí.
  • Tablas: Estructuras que almacenan datos en filas y columnas.
  • Filas: Cada fila representa un registro único.
  • Columnas: Cada columna representa un atributo del registro.

Operaciones Básicas en SQL

SELECT

La instrucción SELECT se utiliza para consultar datos de una o más tablas.

SELECT columna1, columna2
FROM nombre_tabla;

WHERE

La cláusula WHERE se utiliza para filtrar registros que cumplen una condición específica.

SELECT columna1, columna2
FROM nombre_tabla
WHERE condicion;

Ejemplo

SELECT nombre, edad
FROM empleados
WHERE edad > 30;

Este ejemplo selecciona los nombres y edades de los empleados que tienen más de 30 años.

Funciones de Agregación

Las funciones de agregación permiten realizar cálculos sobre un conjunto de valores y devolver un solo valor.

  • COUNT: Cuenta el número de filas.
  • SUM: Suma los valores de una columna.
  • AVG: Calcula el promedio de los valores de una columna.
  • MAX: Encuentra el valor máximo.
  • MIN: Encuentra el valor mínimo.

Ejemplo

SELECT COUNT(*), AVG(salario), MAX(salario), MIN(salario)
FROM empleados;

Este ejemplo cuenta el número de empleados y calcula el salario promedio, máximo y mínimo.

Joins

Los JOINs se utilizan para combinar filas de dos o más tablas basadas en una columna relacionada entre ellas.

  • INNER JOIN: Devuelve filas cuando hay una coincidencia en ambas tablas.
  • LEFT JOIN: Devuelve todas las filas de la tabla izquierda y las filas coincidentes de la tabla derecha.
  • RIGHT JOIN: Devuelve todas las filas de la tabla derecha y las filas coincidentes de la tabla izquierda.
  • FULL JOIN: Devuelve filas cuando hay una coincidencia en una de las tablas.

Ejemplo

SELECT empleados.nombre, departamentos.nombre
FROM empleados
INNER JOIN departamentos ON empleados.departamento_id = departamentos.id;

Este ejemplo selecciona los nombres de los empleados y los nombres de los departamentos donde trabajan, mostrando solo las coincidencias.

Subconsultas

Las subconsultas son consultas anidadas dentro de otra consulta.

Ejemplo

SELECT nombre, salario
FROM empleados
WHERE salario > (SELECT AVG(salario) FROM empleados);

Este ejemplo selecciona los nombres y salarios de los empleados que ganan más que el salario promedio.

Agrupación de Datos

La cláusula GROUP BY se utiliza para agrupar filas que tienen los mismos valores en columnas especificadas.

Ejemplo

SELECT departamento_id, COUNT(*)
FROM empleados
GROUP BY departamento_id;

Este ejemplo cuenta el número de empleados en cada departamento.

Ordenación de Datos

La cláusula ORDER BY se utiliza para ordenar los resultados de una consulta.

Ejemplo

SELECT nombre, salario
FROM empleados
ORDER BY salario DESC;

Este ejemplo selecciona los nombres y salarios de los empleados, ordenándolos de mayor a menor salario.

Conclusión

SQL es una herramienta poderosa para el análisis de datos, permitiendo a los analistas extraer, filtrar y transformar datos de manera eficiente. Comprender las operaciones básicas, funciones de agregación, joins, subconsultas, agrupación y ordenación de datos es fundamental para cualquier profesional del análisis de datos. Practicar con ejemplos y ejercicios es crucial para dominar estas habilidades.
