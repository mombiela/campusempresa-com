Introducción a Azure Machine Learning

Azure Machine Learning es un servicio basado en la nube proporcionado por Microsoft que permite a los desarrolladores y científicos de datos construir, entrenar y desplegar modelos de aprendizaje automático. Proporciona un conjunto completo de herramientas y servicios para facilitar todo el ciclo de vida del aprendizaje automático.

Conceptos Clave

  • Workspace: Un lugar centralizado para gestionar todos los recursos necesarios para el aprendizaje automático.
  • Datasets: Colecciones de datos utilizadas para entrenar y probar modelos de aprendizaje automático.
  • Experiments: Ejecuciones de un script o pipeline de aprendizaje automático.
  • Compute Targets: Recursos donde puedes ejecutar tus experimentos de aprendizaje automático, como máquinas locales, máquinas virtuales o clústeres.
  • Models: Modelos de aprendizaje automático entrenados que pueden ser desplegados para inferencia.
  • Endpoints: Servicios RESTful que alojan tus modelos y realizan predicciones.

Configuración de Tu Entorno

Antes de comenzar a usar Azure Machine Learning, necesitas configurar tu entorno.

Pasos para Configurar

  1. Crear una Cuenta de Azure: Si no tienes una, regístrate para obtener una cuenta gratuita.
  2. Crear un Workspace: Usa el portal de Azure para crear un nuevo workspace de Azure Machine Learning.
  3. Instalar Azure CLI: Instala la Interfaz de Línea de Comandos de Azure (CLI) para interactuar con los servicios de Azure desde tu terminal.
  4. Instalar Azure Machine Learning SDK: Usa pip para instalar el SDK de Azure ML para Python.
pip install azureml-sdk

Creación y Gestión de Datasets

Los datasets son cruciales para entrenar modelos de aprendizaje automático. Azure Machine Learning soporta varios tipos de datasets, incluyendo datasets tabulares y de archivos.

Creación de un Dataset

Puedes crear un dataset desde un archivo local o desde datos almacenados en Azure Blob Storage.

from azureml.core import Workspace, Dataset

# Conectar al workspace
ws = Workspace.from_config()

# Crear un dataset desde un archivo local
dataset = Dataset.Tabular.from_delimited_files(path='data.csv')

# Registrar el dataset
dataset = dataset.register(workspace=ws, name='my_dataset')

Gestión de Datasets

Puedes listar, actualizar y eliminar datasets usando el SDK de Azure ML.

# Listar todos los datasets en el workspace
datasets = ws.datasets
for name, dataset in datasets.items():
    print(name, dataset.id)

# Eliminar un dataset
dataset_to_delete = datasets['my_dataset']
dataset_to_delete.unregister_all_versions()

Construcción y Entrenamiento de Modelos

La construcción y entrenamiento de modelos implica crear experimentos, definir objetivos de cómputo y ejecutar scripts de entrenamiento.

Creación de un Experimento

Un experimento es una colección de pruebas (ejecuciones individuales) que representan intentos de resolver un problema de aprendizaje automático.

from azureml.core import Experiment

# Crear un experimento
experiment = Experiment(workspace=ws, name='my_experiment')

Definición de Objetivos de Cómputo

Los objetivos de cómputo son los recursos donde ejecutas tus scripts de entrenamiento.

from azureml.core.compute import ComputeTarget, AmlCompute
from azureml.core.compute_target import ComputeTargetException

# Definir la configuración de cómputo
compute_config = AmlCompute.provisioning_configuration(vm_size='STANDARD_D2_V2', max_nodes=4)

# Crear el objetivo de cómputo
compute_target = ComputeTarget.create(ws, 'my-compute-cluster', compute_config)
compute_target.wait_for_completion(show_output=True)

Ejecución de Scripts de Entrenamiento

Puedes ejecutar tus scripts de entrenamiento en los objetivos de cómputo definidos.

from azureml.core import ScriptRunConfig

# Definir la configuración de ejecución del script
src = ScriptRunConfig(source_directory='.', script='train.py', compute_target=compute_target)

# Enviar el experimento
run = experiment.submit(src)
run.wait_for_completion(show_output=True)

Despliegue de Modelos

Una vez que has entrenado un modelo, el siguiente paso es desplegarlo para que pueda ser usado para inferencia.

Registro del Modelo

Registra el modelo entrenado en tu workspace.

from azureml.core import Model

# Registrar el modelo
model = Model.register(workspace=ws, model_name='my_model', model_path='outputs/model.pkl')

Creación de una Configuración de Inferencia

Define el entorno y el script de scoring para tu modelo.

from azureml.core.environment import Environment
from azureml.core.model import InferenceConfig

# Definir el entorno
env = Environment.from_conda_specification(name='myenv', file_path='environment.yml')

# Definir la configuración de inferencia
inference_config = InferenceConfig(entry_script='score.py', environment=env)

Despliegue del Modelo

Despliega el modelo en una Instancia de Contenedor de Azure (ACI) o en el Servicio de Kubernetes de Azure (AKS).

from azureml.core.webservice import AciWebservice, Webservice

# Definir la configuración de despliegue
deployment_config = AciWebservice.deploy_configuration(cpu_cores=1, memory_gb=1)

# Desplegar el modelo
service = Model.deploy(workspace=ws, name='my-service', models=[model], inference_config=inference_config, deployment_config=deployment_config)
service.wait_for_deployment(show_output=True)

Conclusión

Azure Machine Learning proporciona una plataforma robusta para construir, entrenar y desplegar modelos de aprendizaje automático. Al entender los conceptos clave y seguir los pasos descritos en este curso, puedes aprovechar efectivamente Azure Machine Learning para resolver problemas complejos y desplegar soluciones de aprendizaje automático escalables.
