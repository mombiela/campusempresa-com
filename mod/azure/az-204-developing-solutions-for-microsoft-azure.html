Introducción

Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado en el desarrollo de soluciones para Microsoft Azure. Aprenderás a implementar soluciones de cómputo en Azure, crear Azure Functions, desarrollar para el almacenamiento en Azure, implementar seguridad en Azure, monitorear, solucionar problemas y optimizar soluciones en Azure, y conectarte y consumir servicios de Azure y servicios de terceros.

  • Implementación de Azure Blob Storage
    • Creación y gestión de Blob storage
    • Carga, descarga y eliminación de blobs
using Azure.Storage.Blobs;
using System;
using System.Threading.Tasks;

public class BlobStorageExample
{
    private const string connectionString = "<Your_Connection_String>";
    private const string containerName = "example-container";
    private const string blobName = "example-blob.txt";

    public static async Task Main(string[] args)
    {
        BlobServiceClient blobServiceClient = new BlobServiceClient(connectionString);
        BlobContainerClient containerClient = blobServiceClient.GetBlobContainerClient(containerName);
        await containerClient.CreateIfNotExistsAsync();

        BlobClient blobClient = containerClient.GetBlobClient(blobName);
        await blobClient.UploadAsync("path/to/local/file.txt", true);

        Console.WriteLine("Blob cargado exitosamente.");
    }
}

Continúa practicando y explorando la amplia oferta de Azure para seguir mejorando tu experiencia.
