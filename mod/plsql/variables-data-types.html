Variables y Tipos de Datos en PL/SQL
Estructuras de Control: Sentencias IF ►

Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y tipos de datos en PL/SQL. Comprender estos conceptos es crucial para escribir código PL/SQL eficiente y efectivo.

Variables en PL/SQL

Las variables en PL/SQL se utilizan para almacenar datos que pueden ser manipulados durante la ejecución de un bloque de código. Son esenciales para realizar cálculos, almacenar resultados intermedios y gestionar el flujo de datos.

Declaración de Variables

Para declarar una variable en PL/SQL, necesitas especificar su nombre y tipo de dato. Opcionalmente, también puedes inicializarla con un valor.

DECLARE
    variable_name data_type [:= initial_value];
BEGIN
    -- Bloque PL/SQL
END;
  • variable_name: El nombre de la variable.
  • data_type: El tipo de dato que la variable contendrá.
  • initial_value: Un valor inicial opcional para la variable.

Ejemplo

DECLARE
    employee_id NUMBER := 1001;
    employee_name VARCHAR2(50) := 'John Doe';
    hire_date DATE := SYSDATE;
BEGIN
    DBMS_OUTPUT.PUT_LINE('Employee ID: ' || employee_id);
    DBMS_OUTPUT.PUT_LINE('Employee Name: ' || employee_name);
    DBMS_OUTPUT.PUT_LINE('Hire Date: ' || hire_date);
END;

En este ejemplo:

  • employee_id es una variable numérica inicializada a 1001.
  • employee_name es una variable de cadena inicializada a 'John Doe'.
  • hire_date es una variable de fecha inicializada a la fecha actual.

Tipos de Datos en PL/SQL

PL/SQL soporta varios tipos de datos para manejar diferentes tipos de datos. Estos tipos de datos pueden ser categorizados en tipos de datos escalares, compuestos, de referencia y LOB (Objeto Grande).

Tipos de Datos Escalares

Los tipos de datos escalares contienen un solo valor. Incluyen:

  • NUMBER: Para valores numéricos.
  • VARCHAR2: Para cadenas de caracteres de longitud variable.
  • CHAR: Para cadenas de caracteres de longitud fija.
  • DATE: Para valores de fecha y hora.
  • BOOLEAN: Para valores lógicos (TRUE, FALSE, NULL).

Ejemplo

DECLARE
    age NUMBER := 30;
    first_name VARCHAR2(20) := 'Alice';
    is_active BOOLEAN := TRUE;
    birth_date DATE := TO_DATE('1990-01-01', 'YYYY-MM-DD');
BEGIN
    DBMS_OUTPUT.PUT_LINE('Age: ' || age);
    DBMS_OUTPUT.PUT_LINE('First Name: ' || first_name);
    DBMS_OUTPUT.PUT_LINE('Is Active: ' || is_active);
    DBMS_OUTPUT.PUT_LINE('Birth Date: ' || birth_date);
END;

Tipos de Datos Compuestos

Los tipos de datos compuestos pueden contener múltiples valores. Incluyen:

  • RECORD: Un grupo de elementos de datos relacionados.
  • TABLE: Una colección de elementos de datos del mismo tipo.

Ejemplo

DECLARE
    TYPE EmployeeRec IS RECORD (
        id NUMBER,
        name VARCHAR2(50),
        salary NUMBER
    );
    employee EmployeeRec;
BEGIN
    employee.id := 1001;
    employee.name := 'John Doe';
    employee.salary := 50000;
    DBMS_OUTPUT.PUT_LINE('Employee ID: ' || employee.id);
    DBMS_OUTPUT.PUT_LINE('Employee Name: ' || employee.name);
    DBMS_OUTPUT.PUT_LINE('Employee Salary: ' || employee.salary);
END;

Tipos de Datos de Referencia

Los tipos de datos de referencia contienen punteros a otros elementos de datos. Incluyen:

  • REF CURSOR: Un puntero a un cursor que puede ser usado para obtener filas de un conjunto de resultados de una consulta.

Ejemplo

DECLARE
    TYPE EmpCurTyp IS REF CURSOR;
    emp_cursor EmpCurTyp;
    emp_record employees%ROWTYPE;
BEGIN
    OPEN emp_cursor FOR SELECT * FROM employees;
    LOOP
        FETCH emp_cursor INTO emp_record;
        EXIT WHEN emp_cursor%NOTFOUND;
        DBMS_OUTPUT.PUT_LINE('Employee ID: ' || emp_record.employee_id);
    END LOOP;
    CLOSE emp_cursor;
END;

Tipos de Datos LOB

Los tipos de datos LOB se utilizan para almacenar objetos grandes como texto, imágenes y datos binarios. Incluyen:

  • BLOB: Objeto Binario Grande.
  • CLOB: Objeto de Caracteres Grande.
  • NCLOB: Objeto de Caracteres Nacional Grande.
  • BFILE: Archivo Binario.

Ejemplo

DECLARE
    image BLOB;
    text CLOB;
BEGIN
    -- Asumiendo que image y text ya están poblados
    DBMS_OUTPUT.PUT_LINE('Image size: ' || DBMS_LOB.GETLENGTH(image));
    DBMS_OUTPUT.PUT_LINE('Text size: ' || DBMS_LOB.GETLENGTH(text));
END;

Resumen

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en PL/SQL. Aprendimos cómo declarar e inicializar variables, y exploramos varios tipos de datos incluyendo escalares, compuestos, de referencia y LOB. Comprender estos conceptos es fundamental para escribir programas PL/SQL robustos.
