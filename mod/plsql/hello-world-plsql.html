Introducción

PL/SQL (Procedural Language/Structured Query Language) es la extensión procedural de Oracle Corporation para SQL y la base de datos relacional Oracle. Combina el poder de manipulación de datos de SQL con el poder de procesamiento de los lenguajes procedurales. En esta sección, aprenderemos cómo escribir un simple programa "Hola Mundo" en PL/SQL.

Conceptos Clave

  • Estructura de Bloques PL/SQL: Los programas PL/SQL están compuestos por bloques, que pueden ser anónimos o nombrados.
  • Bloque Anónimo: Un bloque PL/SQL que no se almacena en la base de datos y se ejecuta inmediatamente.
  • Paquete DBMS_OUTPUT: Un paquete que te permite mostrar la salida de bloques PL/SQL, subprogramas, paquetes y disparadores.

Estructura de Bloques PL/SQL

Un bloque PL/SQL consta de tres secciones:

  1. Sección de Declaración: Opcional. Se usa para declarar variables, constantes, cursores, etc.
  2. Sección de Ejecución: Obligatoria. Contiene el código que realiza las operaciones deseadas.
  3. Sección de Manejo de Excepciones: Opcional. Se usa para manejar errores en tiempo de ejecución.

Estructura Básica de un Bloque PL/SQL

DECLARE
   -- Sección de declaración (opcional)
BEGIN
   -- Sección de ejecución (obligatoria)
EXCEPTION
   -- Sección de manejo de excepciones (opcional)
END;
/

Escribiendo "Hola Mundo" en PL/SQL

Para escribir un programa "Hola Mundo" en PL/SQL, usaremos el procedimiento DBMS_OUTPUT.PUT_LINE para imprimir el mensaje.

Código de Ejemplo

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hola, Mundo!');
END;
/

Explicación

  • BEGIN: Marca el inicio de la sección de ejecución.
  • DBMS_OUTPUT.PUT_LINE: Un procedimiento que muestra una línea de texto.
  • 'Hola, Mundo!': La cadena a imprimir.
  • END;: Marca el final del bloque PL/SQL.
  • /: Ejecuta el bloque PL/SQL en SQL*Plus o SQL Developer.

Habilitando la Salida en SQL*Plus o SQL Developer

Antes de ejecutar el bloque PL/SQL, necesitas habilitar la salida del servidor para ver el mensaje impreso.

Habilitando la Salida del Servidor

SET SERVEROUTPUT ON;

Ejemplo Completo con Salida del Servidor

SET SERVEROUTPUT ON;

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hola, Mundo!');
END;
/

Conclusión

En esta sección, aprendimos cómo escribir un simple programa "Hola Mundo" en PL/SQL. Cubrimos la estructura básica de un bloque PL/SQL y cómo usar el procedimiento DBMS_OUTPUT.PUT_LINE para imprimir la salida. Este conocimiento fundamental te ayudará a medida que avances hacia tareas de programación PL/SQL más complejas.
