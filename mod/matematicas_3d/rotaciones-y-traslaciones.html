Introducción

En el estudio de las matemáticas 3D, las rotaciones y traslaciones son transformaciones fundamentales que permiten manipular objetos en el espacio tridimensional. Estas transformaciones son esenciales en campos como la computación gráfica, la robótica y la simulación física.

Traslaciones

La traslación es una transformación que desplaza cada punto de un objeto una distancia constante en una dirección específica.

  • Definición: Una traslación en 3D se define por un vector de traslación \(\mathbf{t} = (t_x, t_y, t_z)\).
  • Fórmula: Si un punto \(\mathbf{P} = (x, y, z)\) se traslada por \(\mathbf{t}\), el nuevo punto \(\mathbf{P'}\) será: \[ \mathbf{P'} = \mathbf{P} + \mathbf{t} = (x + t_x, y + t_y, z + t_z) \]

Ejemplo de Traslación

Supongamos que tenemos un punto \(\mathbf{P} = (1, 2, 3)\) y queremos trasladarlo por el vector \(\mathbf{t} = (4, -2, 1)\).

\[ \mathbf{P'} = (1 + 4, 2 - 2, 3 + 1) = (5, 0, 4) \]

En código Python, esto se puede realizar de la siguiente manera:

import numpy as np

# Punto original
P = np.array([1, 2, 3])

# Vector de traslación
t = np.array([4, -2, 1])

# Nuevo punto después de la traslación
P_prime = P + t

print("Punto original:", P)
print("Vector de traslación:", t)
print("Nuevo punto:", P_prime)

Rotaciones

La rotación es una transformación que gira un objeto alrededor de un eje fijo.

  • Definición: Una rotación en 3D se define por un ángulo de rotación \(\theta\) y un eje de rotación \(\mathbf{u} = (u_x, u_y, u_z)\).
  • Matriz de Rotación: La matriz de rotación \(R\) para un ángulo \(\theta\) alrededor del eje \(\mathbf{u}\) se puede expresar utilizando la fórmula de Rodrigues: \[ R = I + \sin(\theta)K + (1 - \cos(\theta))K^2 \] donde \(I\) es la matriz identidad y \(K\) es la matriz de componentes cruzados del vector \(\mathbf{u}\).

Ejemplo de Rotación

Supongamos que queremos rotar un punto \(\mathbf{P} = (1, 0, 0)\) 90 grados (\(\pi/2\) radianes) alrededor del eje \(z\).

La matriz de rotación \(R_z\) para un ángulo \(\theta\) alrededor del eje \(z\) es:

\[ R_z = \begin{bmatrix} \cos(\theta) & -\sin(\theta) & 0
\sin(\theta) & \cos(\theta) & 0
0 & 0 & 1 \end{bmatrix} \]

Para \(\theta = \pi/2\):

\[ R_z = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \]

Aplicando esta matriz al punto \(\mathbf{P}\):

\[ \mathbf{P'} = R_z \mathbf{P} = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \begin{bmatrix} 1
0
0 \end{bmatrix} = \begin{bmatrix} 0
1
0 \end{bmatrix} \]

En código Python, esto se puede realizar de la siguiente manera:

import numpy as np

# Punto original
P = np.array([1, 0, 0])

# Ángulo de rotación (90 grados en radianes)
theta = np.pi / 2

# Matriz de rotación alrededor del eje z
R_z = np.array([
    [np.cos(theta), -np.sin(theta), 0],
    [np.sin(theta), np.cos(theta), 0],
    [0, 0, 1]
])

# Nuevo punto después de la rotación
P_prime = R_z.dot(P)

print("Punto original:", P)
print("Matriz de rotación R_z:\n", R_z)
print("Nuevo punto:", P_prime)

Conclusión

Las rotaciones y traslaciones son herramientas esenciales para manipular objetos en el espacio tridimensional. Comprender cómo aplicar estas transformaciones es crucial para cualquier profesional que trabaje en campos relacionados con las matemáticas 3D. A través de ejemplos y ejercicios prácticos, se puede adquirir una comprensión sólida de estos conceptos fundamentales.
