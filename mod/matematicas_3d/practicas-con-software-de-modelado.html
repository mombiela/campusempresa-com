Prácticas con Software de Modelado
Definición del Proyecto ►

Introducción

En esta sección del curso sobre Matemáticas 3D, exploraremos cómo utilizar software de modelado para aplicar conceptos matemáticos tridimensionales. El objetivo es que los estudiantes se familiaricen con herramientas prácticas que les permitan visualizar y manipular objetos en un espacio tridimensional.

Software de Modelado Recomendado

Para las prácticas de este curso, recomendamos el uso de los siguientes programas:

  • Blender: Un software de código abierto muy popular para modelado 3D, animación y renderizado.
  • Autodesk Maya: Una herramienta profesional ampliamente utilizada en la industria del cine y los videojuegos.
  • SketchUp: Ideal para principiantes y muy utilizado en arquitectura y diseño.

Instalación y Configuración

Blender

  1. Descarga: Visita blender.org y descarga la última versión.
  2. Instalación: Sigue las instrucciones del instalador para tu sistema operativo.
  3. Configuración Inicial: Abre Blender y ajusta las preferencias según tus necesidades (por ejemplo, atajos de teclado, tema de la interfaz).

Autodesk Maya

  1. Descarga: Visita autodesk.com y descarga la versión de prueba o educativa.
  2. Instalación: Sigue las instrucciones del instalador.
  3. Configuración Inicial: Configura las preferencias iniciales y asegúrate de tener los plugins necesarios activados.

SketchUp

  1. Descarga: Visita sketchup.com y descarga la versión gratuita o de prueba.
  2. Instalación: Sigue las instrucciones del instalador.
  3. Configuración Inicial: Ajusta las preferencias según tus necesidades.

Conceptos Clave

  • Vertices, Edges y Faces: Los componentes básicos de cualquier modelo 3D.
  • Transformaciones: Operaciones como traslación, rotación y escalado.
  • Modificadores: Herramientas que permiten deformar y modificar objetos de manera no destructiva.
  • Texturizado: Aplicación de imágenes a la superficie de un modelo 3D para darle apariencia realista.

Ejemplos Prácticos

Crear un Cubo en Blender

  1. Agregar un Cubo:

    import bpy

bpy.ops.mesh.primitive_cube_add(size=2, enter_editmode=False, align='WORLD', location=(0, 0, 0))

  2. Escalar el Cubo:

    bpy.ops.transform.resize(value=(2, 2, 2))

  3. Rotar el Cubo:

    bpy.ops.transform.rotate(value=1.5708, orient_axis='Z')

Crear una Esfera en Maya

  1. Agregar una Esfera:

    polySphere -r 1 -sx 20 -sy 20;

  2. Escalar la Esfera:

    scale -r 2 2 2;

  3. Rotar la Esfera:

    rotate -r 45 45 0;

Crear un Cilindro en SketchUp

  1. Agregar un Cilindro:

    • Selecciona la herramienta "Círculo" y dibuja un círculo en el plano.
    • Usa la herramienta "Empujar/Tirar" para extruir el círculo y crear un cilindro.

  2. Escalar el Cilindro:

    • Selecciona el cilindro y usa la herramienta "Escalar" para cambiar su tamaño.

  3. Rotar el Cilindro:

    • Selecciona el cilindro y usa la herramienta "Rotar" para cambiar su orientación.

Ejercicios

  1. Blender: Crea una escena con un cubo, una esfera y un cilindro. Aplica transformaciones (traslación, rotación, escalado) a cada objeto.
  2. Maya: Modela una simple casa utilizando primitivas básicas (cubos, pirámides, etc.). Aplica texturas a las superficies.
  3. SketchUp: Diseña un mueble sencillo (por ejemplo, una mesa) y aplica materiales para darle un aspecto realista.

Conclusión

El uso de software de modelado 3D es esencial para visualizar y aplicar conceptos matemáticos tridimensionales. A través de estas prácticas, los estudiantes no solo aprenderán a manejar herramientas de modelado, sino que también comprenderán mejor cómo se representan y manipulan los objetos en un espacio tridimensional.
