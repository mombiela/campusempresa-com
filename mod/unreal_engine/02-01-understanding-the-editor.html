Entendiendo el Editor
Siguiente ►

En esta sección, aprenderás a navegar y comprender las diferentes partes del editor de Unreal Engine. Este conocimiento es fundamental para trabajar de manera eficiente y efectiva en tus proyectos.

1. Visión General del Editor

El editor de Unreal Engine es una herramienta poderosa que te permite crear y gestionar todos los aspectos de tu juego o aplicación. A continuación, se describen las principales áreas del editor:

1.1. Barra de Herramientas

La barra de herramientas se encuentra en la parte superior del editor y contiene accesos directos a funciones comunes, como guardar tu proyecto, deshacer/rehacer acciones, y ejecutar el juego.

1.2. Vista del Mundo (Viewport)

La vista del mundo es donde puedes ver y manipular los objetos en tu nivel. Puedes moverte por la vista del mundo usando las siguientes combinaciones de teclas y ratón:

  • WASD: Moverse en el espacio 3D.
  • Botón derecho del ratón: Rotar la cámara.
  • Rueda del ratón: Acercar y alejar.

1.3. Outliner

El Outliner muestra una lista jerárquica de todos los actores en tu nivel. Puedes seleccionar y organizar actores desde aquí.

1.4. Detalles

El panel de detalles muestra las propiedades del actor seleccionado. Aquí puedes ajustar parámetros como la posición, rotación, escala, y otras propiedades específicas del actor.

1.5. Content Browser

El Content Browser es donde puedes gestionar todos los activos de tu proyecto, como modelos 3D, texturas, sonidos, y Blueprints. Puedes arrastrar y soltar activos desde el Content Browser a la vista del mundo.

1.6. Modes

El panel de Modes te permite cambiar entre diferentes modos de edición, como el modo de selección, el modo de pintura de terreno, y el modo de edición de malla.

2. Navegando por el Editor

2.1. Moviéndose en la Vista del Mundo

Para moverte en la vista del mundo, puedes usar las siguientes combinaciones de teclas y ratón:

  • Alt + Botón izquierdo del ratón: Orbitar alrededor del punto de interés.
  • Alt + Botón derecho del ratón: Acercar y alejar.
  • Alt + Botón central del ratón: Desplazar la vista.

2.2. Seleccionando y Manipulando Actores

Para seleccionar un actor, simplemente haz clic en él en la vista del mundo o en el Outliner. Una vez seleccionado, puedes manipularlo usando las siguientes herramientas:

  • W: Mover.
  • E: Rotar.
  • R: Escalar.

2.3. Usando el Content Browser

El Content Browser es esencial para gestionar tus activos. Puedes buscar activos usando la barra de búsqueda, organizar activos en carpetas, y previsualizar activos antes de usarlos.

3. Ejemplo Práctico: Creando un Escenario Simple

Vamos a crear un escenario simple para familiarizarnos con el editor.

Paso 1: Crear un Nuevo Nivel

  1. Abre el menú File y selecciona New Level.
  2. Elige la opción Default para crear un nivel básico con un suelo y una luz.

Paso 2: Añadir un Actor

  1. Abre el Content Browser y busca un modelo 3D, como una mesa o una silla.
  2. Arrastra el modelo desde el Content Browser a la vista del mundo.

Paso 3: Manipular el Actor

  1. Selecciona el actor en la vista del mundo.
  2. Usa las teclas W, E, y R para mover, rotar, y escalar el actor hasta que esté en la posición deseada.

Paso 4: Guardar el Nivel

  1. Abre el menú File y selecciona Save Current Level As.
  2. Elige un nombre y una ubicación para guardar tu nivel.

4. Ejercicio Práctico

Ejercicio 1: Crear una Escena Simple

  1. Crea un nuevo nivel vacío.
  2. Añade al menos tres actores diferentes desde el Content Browser.
  3. Organiza los actores en una disposición lógica (por ejemplo, una mesa con sillas alrededor).
  4. Guarda tu nivel.

Solución del Ejercicio

  1. Abre el menú File y selecciona New Level.
  2. Elige la opción Empty Level.
  3. Abre el Content Browser y busca tres modelos 3D diferentes.
  4. Arrastra los modelos a la vista del mundo y usa las herramientas de manipulación para organizarlos.
  5. Guarda el nivel con un nombre descriptivo.

5. Conclusión

En esta sección, hemos cubierto los aspectos básicos del editor de Unreal Engine, incluyendo la barra de herramientas, la vista del mundo, el Outliner, el panel de detalles, el Content Browser, y el panel de Modes. También hemos practicado creando y manipulando actores en un nivel simple. Con este conocimiento, estarás bien preparado para avanzar a temas más complejos en Unreal Engine.
