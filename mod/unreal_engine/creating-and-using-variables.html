Introducción

Las variables son fundamentales en la programación ya que almacenan datos que pueden ser utilizados y manipulados a lo largo de tu código. En Unreal Engine, las variables se utilizan dentro de Blueprints y C++ para controlar la lógica del juego, almacenar datos del jugador y gestionar estados del juego. Esta sección te guiará a través de los conceptos básicos de la creación y uso de variables en Unreal Engine, progresando desde conceptos para principiantes hasta avanzados.

Nivel Principiante

¿Qué es una Variable?

  • Una variable es una ubicación de almacenamiento identificada por un nombre que contiene datos que pueden ser modificados durante la ejecución del programa.
  • Las variables tienen tipos, que definen el tipo de datos que pueden contener (por ejemplo, enteros, flotantes, cadenas).

Creación de Variables en Blueprints

  1. Abrir una Clase Blueprint: Comienza abriendo cualquier clase Blueprint, como un Actor o un Personaje.
  2. Agregar una Variable:
    • En el panel "My Blueprint", haz clic en el botón "+" junto a "Variables".
    • Nombra tu variable (por ejemplo, Health).
    • Elige el tipo de variable del menú desplegable (por ejemplo, Float para puntos de salud).
+-------------------+
| My Blueprint      |
|-------------------|
| Variables         |
| + Health (Float)  |
+-------------------+

Uso de Variables en Blueprints

  • Nodos Get y Set: Usa los nodos "Get" y "Set" para recuperar y modificar el valor de una variable.
  • Ejemplo: Establecer la salud del jugador en 100 cuando el juego comienza.
Event BeginPlay
    |
    V
Set Health (100)

Ejemplo de Código Blueprint

Event BeginPlay
    |
    V
Set Health (100)

Creación de Variables en C++

  1. Abrir tu Clase C++: Abre cualquier clase C++, como una clase ACharacter personalizada.
  2. Declarar una Variable:
    • En el archivo de encabezado (.h), declara tu variable.
    • Ejemplo: float Health;
// MyCharacter.h
public:
    float Health;
  1. Inicializar la Variable:
    • En el archivo fuente (.cpp), inicializa tu variable en el constructor u otro método apropiado.
// MyCharacter.cpp
AMyCharacter::AMyCharacter()
{
    Health = 100.0f;
}

Uso de Variables en C++

  • Acceder y Modificar: Usa la variable directamente dentro de los métodos de tu clase.
  • Ejemplo: Imprimir la salud del jugador en la consola.
UE_LOG(LogTemp, Warning, TEXT("Player Health: %f"), Health);

Nivel Intermedio

Alcance y Duración de las Variables

  • Variables Locales: Declaradas dentro de una función y solo accesibles dentro de esa función.
  • Variables Miembro: Declaradas en una clase y accesibles por todos los métodos de la clase.
  • Variables Globales: Declaradas fuera de cualquier clase o función y accesibles desde cualquier parte del código (usarlas con moderación).

Arreglos y Colecciones

  • Arreglos: Almacenan múltiples elementos del mismo tipo.
  • Ejemplo en Blueprints: Crear un arreglo de enemigos.
+-------------------+
| My Blueprint      |
|-------------------|
| Variables         |
| + Enemies (Array) |
+-------------------+
  • Ejemplo en C++:
TArray<AEnemy*> Enemies;

Estructuras

  • Definición: Una estructura es un tipo de dato personalizado que agrupa múltiples variables.
  • Ejemplo en Blueprints: Crear una estructura para estadísticas del jugador.
+-------------------+
| My Blueprint      |
|-------------------|
| Structs           |
| + PlayerStats     |
+-------------------+
  • Ejemplo en C++:
USTRUCT(BlueprintType)
struct FPlayerStats
{
    GENERATED_BODY()

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float Health;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float Stamina;
};

Nivel Avanzado

Estructuras de Datos Avanzadas

  • Mapas: Almacenan pares clave-valor.
  • Conjuntos: Almacenan elementos únicos.
  • Ejemplo en C++:
TMap<FString, int32> PlayerScores;
TSet<int32> UniqueIDs;

Enums

  • Definición: Los Enums son tipos definidos por el usuario que consisten en valores nombrados.
  • Ejemplo en Blueprints: Crear un enum para tipos de armas.
+-------------------+
| My Blueprint      |
|-------------------|
| Enums             |
| + WeaponType      |
+-------------------+
  • Ejemplo en C++:
UENUM(BlueprintType)
enum class EWeaponType : uint8
{
    EWT_Sword UMETA(DisplayName = "Sword"),
    EWT_Bow UMETA(DisplayName = "Bow"),
    EWT_Axe UMETA(DisplayName = "Axe")
};

Uso de Punteros y Referencias

  • Punteros: Variables que almacenan direcciones de memoria.
  • Referencias: Alias para variables existentes.
  • Ejemplo en C++:
AEnemy* EnemyPointer;
AEnemy& EnemyReference = *EnemyPointer;

Conclusión

Entender y usar efectivamente las variables es crucial para el desarrollo de juegos en Unreal Engine. Comenzando desde la creación básica y manipulación de variables en Blueprints y C++, puedes progresar a estructuras de datos más complejas y conceptos avanzados. El dominio de estos temas te permitirá gestionar los datos del juego de manera eficiente y crear experiencias de juego más dinámicas e interactivas.
