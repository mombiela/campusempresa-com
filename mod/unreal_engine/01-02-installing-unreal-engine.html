Instalando Unreal Engine

Siguiente ►

En esta sección, aprenderás cómo instalar Unreal Engine en tu computadora. Unreal Engine es una poderosa herramienta de desarrollo de juegos y aplicaciones 3D, y su instalación es el primer paso para comenzar a crear tus propios proyectos. A continuación, se detallan los pasos necesarios para instalar Unreal Engine.

Requisitos del Sistema

Antes de comenzar con la instalación, asegúrate de que tu sistema cumpla con los requisitos mínimos:

Requisitos Mínimos

  • Sistema Operativo: Windows 7/8/10 64-bit, macOS 10.14.6 o superior
  • Procesador: Quad-core Intel o AMD, 2.5 GHz o más rápido
  • Memoria: 8 GB de RAM
  • Tarjeta Gráfica: DirectX 11 o DirectX 12 compatible
  • Almacenamiento: 20 GB de espacio libre en disco

Requisitos Recomendados

  • Sistema Operativo: Windows 10 64-bit, macOS 10.15.6 o superior
  • Procesador: Quad-core Intel o AMD, 3 GHz o más rápido
  • Memoria: 16 GB de RAM
  • Tarjeta Gráfica: NVIDIA GeForce GTX 970 o AMD R9 290X o superior
  • Almacenamiento: SSD con 100 GB de espacio libre

Paso 1: Crear una Cuenta de Epic Games

Para descargar Unreal Engine, primero necesitas una cuenta de Epic Games.

  1. Visita el sitio web de Epic Games.
  2. Haz clic en "Sign In" en la esquina superior derecha.
  3. Selecciona "Sign Up" para crear una nueva cuenta.
  4. Completa el formulario de registro con tu información personal.
  5. Verifica tu cuenta a través del correo electrónico que recibirás.

Paso 2: Descargar el Epic Games Launcher

El Epic Games Launcher es la aplicación que te permitirá descargar e instalar Unreal Engine.

  1. Una vez que hayas iniciado sesión en tu cuenta de Epic Games, dirígete a la sección de Unreal Engine.
  2. Haz clic en "Get Started" y luego en "Download".
  3. El archivo de instalación del Epic Games Launcher se descargará automáticamente.

Paso 3: Instalar el Epic Games Launcher

  1. Abre el archivo de instalación que descargaste.
  2. Sigue las instrucciones del asistente de instalación.
  3. Una vez instalado, abre el Epic Games Launcher e inicia sesión con tu cuenta de Epic Games.

Paso 4: Descargar e Instalar Unreal Engine

  1. En el Epic Games Launcher, selecciona la pestaña "Unreal Engine" en el lado izquierdo.
  2. Haz clic en "Library" y luego en "Install Engine".
  3. Selecciona la versión de Unreal Engine que deseas instalar. Se recomienda instalar la última versión estable.
  4. Elige la ubicación de instalación y haz clic en "Install".

Paso 5: Configuración Inicial

Una vez que la instalación esté completa, puedes abrir Unreal Engine desde el Epic Games Launcher.

  1. Haz clic en "Launch" en la pestaña "Unreal Engine".
  2. La primera vez que abras Unreal Engine, se te pedirá que configures algunos ajustes iniciales, como la calidad gráfica y la ubicación de los proyectos.
  3. Completa la configuración según tus preferencias.

Ejercicio Práctico

Para asegurarte de que has instalado Unreal Engine correctamente, realiza el siguiente ejercicio:

  1. Abre Unreal Engine desde el Epic Games Launcher.
  2. Crea un nuevo proyecto seleccionando "Games" y luego "Blank".
  3. Configura el proyecto con las siguientes opciones:
    • Blueprint: Seleccionado
    • Maximum Quality: Seleccionado
    • No Starter Content: Seleccionado
    • Raytracing: Desactivado (a menos que tu hardware lo soporte)
  4. Haz clic en "Create" y espera a que se cargue el proyecto.

Si puedes ver la interfaz del editor de Unreal Engine y moverte por el entorno, ¡felicidades! Has instalado Unreal Engine correctamente.

Conclusión

En esta sección, has aprendido cómo instalar Unreal Engine en tu computadora. Desde la creación de una cuenta de Epic Games hasta la configuración inicial del motor, ahora estás listo para comenzar a explorar y crear tus propios proyectos en Unreal Engine. En la próxima sección, aprenderás a navegar por la interfaz de Unreal Engine, lo cual es fundamental para trabajar de manera eficiente en tus proyectos.
