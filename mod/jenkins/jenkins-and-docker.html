Introducción a Jenkins y Docker

En esta sección, exploraremos los conceptos básicos de Jenkins y Docker, y cómo pueden usarse juntos para optimizar tu pipeline de CI/CD.

¿Qué es Jenkins?

  • Jenkins es un servidor de automatización de código abierto.
  • Ayuda a automatizar partes del proceso de desarrollo de software, incluyendo la construcción, prueba y despliegue de código.
  • Jenkins soporta la integración continua (CI) y la entrega continua (CD).

¿Qué es Docker?

  • Docker es una plataforma para desarrollar, enviar y ejecutar aplicaciones dentro de contenedores.
  • Los contenedores son ligeros, portátiles y aseguran consistencia a través de diferentes entornos.

Configuración de Jenkins con Docker

En esta sección, cubriremos cómo configurar Jenkins usando Docker.

Prerrequisitos

  • Comprensión básica de Docker y Jenkins.
  • Docker instalado en tu máquina.

Ejecutando Jenkins en un Contenedor Docker

  1. Descargar la Imagen Docker de Jenkins: 
    docker pull jenkins/jenkins:lts
  2. Ejecutar el Contenedor Jenkins: 
    docker run -d -p 8080:8080 -p 50000:50000 --name jenkins jenkins/jenkins:lts
    • -d: Ejecutar el contenedor en modo desacoplado.
    • -p 8080:8080: Mapear el puerto 8080 del host al puerto 8080 del contenedor.
    • -p 50000:50000: Mapear el puerto 50000 del host al puerto 50000 del contenedor.
    • --name jenkins: Nombrar el contenedor "jenkins".

Accediendo a Jenkins

  • Abre un navegador web y navega a http://localhost:8080.
  • Sigue las instrucciones en pantalla para completar la configuración.

Integrando Docker con Jenkins

En esta sección, aprenderemos cómo integrar Docker con Jenkins para construir y desplegar aplicaciones.

Instalando el Plugin Docker en Jenkins

  1. Navegar al Tablero de Jenkins:
    • Ve a Manage Jenkins > Manage Plugins.
  2. Instalar el Plugin Docker:
    • Busca "Docker" e instala el plugin Docker.

Configurando Docker en Jenkins

  1. Navegar al Tablero de Jenkins:
    • Ve a Manage Jenkins > Configure System.
  2. Agregar Host Docker:
    • Bajo Cloud, haz clic en Add a new cloud y selecciona Docker.
    • Configura los ajustes del host Docker (por ejemplo, URL de Docker, credenciales).

Creando un Pipeline de Jenkins con Docker

  1. Crear un Nuevo Trabajo de Pipeline:
    • Ve a New Item, ingresa un nombre y selecciona Pipeline.
  2. Definir el Script del Pipeline:
    • Usa el siguiente script de ejemplo para construir y ejecutar un contenedor Docker: 
      pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                script {
                    docker.build('my-app-image')
                }
            }
        }
        stage('Run') {
            steps {
                script {
                    docker.image('my-app-image').inside {
                        sh 'echo "Running inside Docker container"'
                    }
                }
            }
        }
    }
}

Uso Avanzado de Jenkins y Docker

En esta sección, cubriremos temas más avanzados, como usar Docker Compose con Jenkins y configurar aplicaciones de múltiples contenedores.

Usando Docker Compose en Pipelines de Jenkins

  1. Instalar Docker Compose:
    • Asegúrate de que Docker Compose esté instalado en tu servidor Jenkins.
  2. Definir el Archivo Docker Compose:
    • Crea un archivo docker-compose.yml para tu aplicación. 
      version: '3'
servicios:
  web:
    image: nginx
    ports:
      - "80:80"
  db:
    image: mysql
    environment:
      MYSQL_ROOT_PASSWORD: example
  3. Actualizar el Script del Pipeline de Jenkins:
    • Usa el siguiente script de ejemplo para ejecutar Docker Compose: 
      pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                script {
                    sh 'docker-compose build'
                }
            }
        }
        stage('Run') {
            steps {
                script {
                    sh 'docker-compose up -d'
                }
            }
        }
    }
}

Conclusión

En este tema, cubrimos la integración de Jenkins y Docker, desde la configuración básica hasta el uso avanzado. Al aprovechar Docker dentro de Jenkins, puedes crear un pipeline de CI/CD robusto y flexible que asegura consistencia a través de diferentes entornos. Esta integración permite una construcción, prueba y despliegue eficientes de aplicaciones, acelerando el proceso de desarrollo y mejorando la calidad del software.
