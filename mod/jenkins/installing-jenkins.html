Introducción

Jenkins es un servidor de automatización de código abierto que ayuda a automatizar las partes del desarrollo de software relacionadas con la construcción, prueba y despliegue, facilitando la integración continua y la entrega continua (CI/CD). Esta sección te guiará a través del proceso de instalación de Jenkins en diferentes sistemas operativos.

Requisitos previos

Antes de instalar Jenkins, asegúrate de tener los siguientes requisitos previos:

  • Un sistema que ejecute un sistema operativo compatible (Windows, macOS o Linux).
  • Kit de Desarrollo de Java (JDK) instalado (Jenkins requiere Java para funcionar).

Instalación de Jenkins en Windows

Paso 1: Descargar Jenkins

  1. Visita la página de descarga de Jenkins.
  2. Haz clic en la pestaña "Windows".
  3. Descarga el paquete instalador de Windows (archivo .msi).

Paso 2: Instalar Jenkins

  1. Ejecuta el instalador .msi descargado.
  2. Sigue el asistente de instalación:
    • Acepta el acuerdo de licencia.
    • Elige el directorio de instalación.
    • Selecciona los componentes a instalar (se recomiendan las opciones predeterminadas).
    • Haz clic en "Instalar" para comenzar la instalación.

Paso 3: Iniciar Jenkins

  1. Después de la instalación, Jenkins se iniciará automáticamente.
  2. Abre un navegador web y navega a http://localhost:8080.
  3. Sigue las instrucciones en pantalla para completar la configuración.

Instalación de Jenkins en macOS

Paso 1: Instalar Homebrew

Si no tienes Homebrew instalado, abre Terminal y ejecuta:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Paso 2: Instalar Jenkins

  1. Abre Terminal.
  2. Ejecuta el siguiente comando para instalar Jenkins: 
    brew install jenkins-lts

Paso 3: Iniciar Jenkins

  1. Inicia Jenkins ejecutando: 
    brew services start jenkins-lts
  2. Abre un navegador web y navega a http://localhost:8080.
  3. Sigue las instrucciones en pantalla para completar la configuración.

Instalación de Jenkins en Linux

Paso 1: Agregar el repositorio de Jenkins

  1. Abre Terminal.
  2. Agrega el repositorio y la clave de Jenkins: 
    wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -
sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'

Paso 2: Instalar Jenkins

  1. Actualiza tu lista de paquetes: 
    sudo apt-get update
  2. Instala Jenkins: 
    sudo apt-get install jenkins

Paso 3: Iniciar Jenkins

  1. Inicia el servicio de Jenkins: 
    sudo systemctl start jenkins
  2. Habilita Jenkins para que se inicie al arrancar: 
    sudo systemctl enable jenkins
  3. Abre un navegador web y navega a http://localhost:8080.
  4. Sigue las instrucciones en pantalla para completar la configuración.

Configuración posterior a la instalación

Después de instalar Jenkins, necesitas realizar algunos pasos de configuración inicial:

  1. Desbloquear Jenkins:
    • Localiza la contraseña inicial del administrador en el archivo especificado en la pantalla de configuración (por ejemplo, /var/lib/jenkins/secrets/initialAdminPassword).
    • Ingresa la contraseña en el asistente de configuración.
  2. Personalizar Jenkins:
    • Instala los complementos sugeridos o selecciona complementos específicos según tus necesidades.
    • Crea el primer usuario administrador.
    • Configura la URL de la instancia.

Conclusión

Instalar Jenkins es un proceso sencillo que varía ligeramente según el sistema operativo. Siguiendo los pasos descritos anteriormente, puedes configurar Jenkins en Windows, macOS o Linux. Una vez instalado, Jenkins proporciona una plataforma poderosa para automatizar tus procesos de desarrollo de software, permitiendo la integración continua y la entrega continua.
