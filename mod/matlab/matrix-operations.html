Introducción a las Matrices

Las matrices son fundamentales para MATLAB, que significa "Matrix Laboratory" (Laboratorio de Matrices). Entender cómo crear, manipular y realizar operaciones con matrices es esencial para cualquier programador de MATLAB.

Creación de Matrices

  • Vector Fila: Una matriz de 1xN. 
    row_vector = [1, 2, 3, 4];
  • Vector Columna: Una matriz de Nx1. 
    column_vector = [1; 2; 3; 4];
  • Matriz 2D: Una matriz de MxN. 
    matrix_2d = [1, 2, 3; 4, 5, 6; 7, 8, 9];

Operaciones Básicas con Matrices

  • Suma y Resta: Operaciones elemento a elemento. 
    A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Suma
D = A - B; % Resta
  • Multiplicación por un Escalar: Multiplicar cada elemento por un escalar. 
    scalar = 2;
E = scalar * A;

Multiplicación de Matrices

La multiplicación de matrices sigue las reglas del álgebra lineal.

  • Multiplicación Elemento a Elemento: Usar el operador .*. 
    F = A .* B;
  • Producto de Matrices: Usar el operador *. 
    G = A * B;

Transposición de una Matriz

  • Transposición: Voltear una matriz sobre su diagonal. 
    H = A';

Operaciones Avanzadas con Matrices

Inversa y Determinante

  • Inversa: Solo para matrices cuadradas. 
    I = inv(A);
  • Determinante: Un valor escalar que es una función de las entradas de una matriz cuadrada. 
    det_A = det(A);

Valores y Vectores Propios

  • Valores y Vectores Propios: Importantes en muchas aplicaciones como el análisis de estabilidad. 
    [V, D] = eig(A);
    • V contiene los vectores propios.
    • D es una matriz diagonal con los valores propios.

Resolución de Sistemas Lineales

  • Sistema Lineal: Ax = b 
    A = [1, 2; 3, 4];
b = [5; 6];
x = A \ b; % Resuelve para x

Matrices Especiales

  • Matriz Identidad: Una matriz cuadrada con unos en la diagonal y ceros en otros lugares. 
    I = eye(3); % Matriz identidad de 3x3
  • Matriz de Ceros: Una matriz llena de ceros. 
    Z = zeros(3, 2); % Matriz de 3x2 de ceros
  • Matriz de Unos: Una matriz llena de unos. 
    O = ones(2, 3); % Matriz de 2x3 de unos

Ejemplos Prácticos

Ejemplo 1: Suma y Multiplicación de Matrices

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Suma de matrices
D = A * B; % Multiplicación de matrices

Ejemplo 2: Resolución de un Sistema Lineal

A = [2, -1; -3, 4];
b = [1; 7];
x = A \ b; % Resolviendo Ax = b

Ejemplo 3: Valores y Vectores Propios

A = [4, -2; 1, 1];
[V, D] = eig(A);

Resumen

Entender las operaciones con matrices en MATLAB es crucial para realizar una amplia gama de tareas matemáticas e ingenieriles. Desde operaciones básicas como la suma y la multiplicación hasta conceptos avanzados como los valores propios y la resolución de sistemas lineales, dominar estas habilidades mejorará significativamente tu capacidad para usar MATLAB de manera efectiva. Practica estas operaciones con diferentes matrices para volverte competente en el manejo de problemas relacionados con matrices.
