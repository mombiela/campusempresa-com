Declaraciones condicionales en MATLAB
Bucles: for y while ►

Las declaraciones condicionales son fundamentales en la programación, ya que permiten ejecutar diferentes bloques de código según ciertas condiciones. En MATLAB, las declaraciones condicionales incluyen if, else, elseif, switch y case. Esta sección te guiará desde lo básico hasta el uso más avanzado de estas declaraciones.

Introducción a las declaraciones condicionales

Las declaraciones condicionales controlan el flujo de ejecución en un programa. Permiten que el programa tome decisiones y ejecute bloques de código específicos basados en la evaluación de condiciones.

Conceptos clave

  • Condición: Una expresión que se evalúa como verdadera o falsa.
  • Bloque de código: Un conjunto de declaraciones que se ejecutan si la condición es verdadera.
  • Flujo de control: El orden en el que se ejecutan o evalúan declaraciones individuales, instrucciones o llamadas a funciones.

Declaraciones `if`, `else` y `elseif`

La declaración if es la forma más básica de declaración condicional. Ejecuta un bloque de código si una condición especificada es verdadera. Las declaraciones else y elseif proporcionan control adicional al permitir que se ejecuten bloques de código alternativos si la condición inicial es falsa.

Sintaxis

if condition
    % Código a ejecutar si la condición es verdadera
elseif another_condition
    % Código a ejecutar si otra condición es verdadera
else
    % Código a ejecutar si ninguna de las condiciones anteriores es verdadera
end

Ejemplo

x = 10;

if x > 0
    disp('x es positivo');
elseif x < 0
    disp('x es negativo');
else
    disp('x es cero');
end

En este ejemplo:

  • Si x es mayor que 0, muestra "x es positivo".
  • Si x es menor que 0, muestra "x es negativo".
  • Si x es 0, muestra "x es cero".

Declaraciones `if` anidadas

Puedes anidar declaraciones if dentro de otras para verificar múltiples condiciones de manera jerárquica.

Ejemplo

x = 5;
y = 10;

if x > 0
    if y > 0
        disp('Ambos x e y son positivos');
    else
        disp('x es positivo, pero y no es positivo');
    end
else
    disp('x no es positivo');
end

En este ejemplo:

  • El if externo verifica si x es positivo.
  • El if interno verifica si y es positivo si la condición externa es verdadera.

Declaraciones `switch` y `case`

La declaración switch se utiliza para ejecutar uno de varios grupos de declaraciones. Es una alternativa a usar múltiples declaraciones if y elseif.

Sintaxis

switch variable
    case value1
        % Código a ejecutar si la variable es igual a value1
    case value2
        % Código a ejecutar si la variable es igual a value2
    otherwise
        % Código a ejecutar si la variable no coincide con ningún caso
end

Ejemplo

day = 'Monday';

switch day
    case 'Monday'
        disp('Inicio de la semana laboral');
    case 'Friday'
        disp('Fin de la semana laboral');
    otherwise
        disp('Mitad de semana');
end

En este ejemplo:

  • Si day es 'Monday', muestra "Inicio de la semana laboral".
  • Si day es 'Friday', muestra "Fin de la semana laboral".
  • Para cualquier otro valor de day, muestra "Mitad de semana".

Comparación de `if` y `switch`

| Característica | Declaración if | Declaración switch | |------------------|-----------------------------------------------------|---------------------------------------------------| | Caso de uso | Múltiples condiciones con diferentes expresiones | Variable única con múltiples valores posibles | | Legibilidad | Puede volverse complejo con muchas condiciones elseif | Más legible para verificar una sola variable | | Flexibilidad | Más flexible con condiciones complejas | Menos flexible, mejor para valores discretos |

Uso avanzado

Operadores lógicos

Los operadores lógicos se pueden usar para combinar múltiples condiciones en declaraciones if.

  • AND (&&): Verdadero si ambas condiciones son verdaderas.
  • OR (||): Verdadero si al menos una condición es verdadera.
  • NOT (~): Verdadero si la condición es falsa.

Ejemplo

a = 5;
b = 10;

if a > 0 && b > 0
    disp('Ambos a y b son positivos');
elseif a > 0 || b > 0
    disp('Al menos uno de a o b es positivo');
else
    disp('Ninguno de a o b es positivo');
end

Conclusión

Las declaraciones condicionales son esenciales para controlar el flujo de un programa basado en diferentes condiciones. En MATLAB, las declaraciones if, else, elseif, switch y case proporcionan herramientas poderosas para la toma de decisiones. Comprender y usar efectivamente estas declaraciones te permitirá escribir código más dinámico y receptivo.

Resumen

  • if, else, elseif: Se utilizan para verificar múltiples condiciones.
  • if anidado: Permite la verificación jerárquica de condiciones.
  • switch, case: Simplifica la verificación de una variable única contra múltiples valores.
  • Operadores lógicos: Combinan múltiples condiciones dentro de declaraciones if.

Al dominar estos conceptos, estarás bien equipado para manejar procesos de toma de decisiones complejos en tus programas de MATLAB.
