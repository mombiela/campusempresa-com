Introducción a las Tablas

Las tablas son un tipo de dato poderoso en MATLAB que te permite almacenar datos de diferentes tipos y tamaños en un solo contenedor. Son particularmente útiles para organizar y analizar conjuntos de datos que incluyen tanto datos numéricos como no numéricos.

Conceptos Clave

  • Creación de Tablas: Cómo crear tablas a partir de matrices, arreglos de celdas o importando datos.
  • Propiedades de la Tabla: Entender las dimensiones de la tabla, nombres de variables y nombres de filas.
  • Acceso a Datos: Métodos para acceder y modificar datos dentro de las tablas.
  • Operaciones con Tablas: Operaciones comunes como ordenar, filtrar y unir tablas.

Creación de Tablas

Puedes crear tablas en MATLAB usando la función table. Aquí hay algunas formas básicas de crear tablas:

A partir de Matrices

% Crear matrices
Age = [25; 30; 35];
Height = [5.5; 6.0; 5.8];
Weight = [150; 180; 165];

% Crear tabla
T = table(Age, Height, Weight);
disp(T);

Este código crea una tabla T con tres columnas: Age, Height y Weight.

A partir de Arreglos de Celdas

% Crear arreglos de celdas
Names = {'John'; 'Jane'; 'Jim'};
Scores = {85, 90, 88};

% Crear tabla
T = table(Names, Scores);
disp(T);

Este código crea una tabla T con dos columnas: Names y Scores.

Importando Datos

También puedes crear tablas importando datos de archivos como CSV, Excel o archivos de texto.

% Importar datos de un archivo CSV
T = readtable('data.csv');
disp(T);

Propiedades de la Tabla

Las tablas tienen propiedades que definen su estructura y metadatos.

Dimensiones

Puedes obtener el tamaño de una tabla usando la función size.

% Obtener dimensiones de la tabla
[rows, cols] = size(T);
disp(['Filas: ', num2str(rows), ', Columnas: ', num2str(cols)]);

Nombres de Variables

Los nombres de las variables pueden ser accedidos y modificados usando la propiedad Properties.VariableNames.

% Obtener nombres de variables
varNames = T.Properties.VariableNames;
disp(varNames);

% Establecer nuevos nombres de variables
T.Properties.VariableNames = {'Age', 'Height_in', 'Weight_lb'};
disp(T);

Nombres de Filas

De manera similar, los nombres de las filas pueden ser accedidos y modificados.

% Establecer nombres de filas
T.Properties.RowNames = {'Person1', 'Person2', 'Person3'};
disp(T);

Acceso y Modificación de Datos

Puedes acceder y modificar datos de la tabla usando varios métodos de indexación.

Por Nombre de Variable

% Acceder a datos por nombre de variable
heights = T.Height_in;
disp(heights);

% Modificar datos por nombre de variable
T.Height_in = T.Height_in * 2.54; % Convertir pulgadas a cm
disp(T);

Por Fila y Nombre de Variable

% Acceder a elemento específico
height_person1 = T{'Person1', 'Height_in'};
disp(height_person1);

% Modificar elemento específico
T{'Person1', 'Height_in'} = 170;
disp(T);

Operaciones con Tablas

MATLAB proporciona varias funciones para realizar operaciones en tablas.

Ordenar

% Ordenar tabla por Age
T = sortrows(T, 'Age');
disp(T);

Filtrar

% Filtrar filas donde Age es mayor que 30
T_filtered = T(T.Age > 30, :);
disp(T_filtered);

Unir Tablas

Puedes unir dos tablas usando la función join.

% Crear otra tabla
Gender = {'M'; 'F'; 'M'};
T2 = table(Gender, 'RowNames', {'Person1', 'Person2', 'Person3'});

% Unir tablas
T_joined = join(T, T2);
disp(T_joined);

Conclusión

Las tablas en MATLAB son versátiles y poderosas para gestionar datos heterogéneos. Al entender cómo crear, manipular y realizar operaciones en tablas, puedes organizar y analizar eficazmente conjuntos de datos complejos. Practica creando tablas a partir de diferentes fuentes de datos y realizando varias operaciones para volverte competente en el uso de tablas en MATLAB.
