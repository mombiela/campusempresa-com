Introducción

En esta sección, profundizaremos en técnicas avanzadas de gráficos en MATLAB. Estas técnicas te ayudarán a crear visualizaciones más sofisticadas e informativas. Cubriremos temas como la personalización de gráficos, la creación de gráficos 3D y el uso de funciones de gráficos especializadas.

Personalización de Gráficos

Personalizar gráficos te permite hacer tus visualizaciones más informativas y estéticamente agradables.

Cambiar Estilos y Colores de Líneas

  • Estilos de Líneas: Puedes cambiar el estilo de las líneas en tus gráficos usando la propiedad LineStyle.
  • Colores: MATLAB te permite especificar colores usando la propiedad Color.
x = 0:0.1:10;
y = sin(x);

figure;
plot(x, y, 'LineStyle', '--', 'Color', 'r'); % Línea roja discontinua
title('Onda Senoidal');
xlabel('Eje X');
ylabel('Eje Y');

Añadir Anotaciones

  • Anotaciones de Texto: Usa la función text para añadir anotaciones de texto a tus gráficos.
  • Flechas y Formas: Usa la función annotation para añadir flechas, rectángulos y otras formas.
figure;
plot(x, y);
title('Onda Senoidal con Anotación');
xlabel('Eje X');
ylabel('Eje Y');
text(5, 0, 'Punto Medio', 'FontSize', 12, 'Color', 'blue');
annotation('arrow', [0.5, 0.7], [0.5, 0.7]);

Subgráficos

  • Crear Subgráficos: Usa la función subplot para crear múltiples gráficos en una sola figura.
figure;
subplot(2, 1, 1); % 2 filas, 1 columna, primer subgráfico
plot(x, y);
title('Onda Senoidal');

subplot(2, 1, 2); % 2 filas, 1 columna, segundo subgráfico
plot(x, cos(x));
title('Onda Cosenoidal');

Gráficos 3D

Los gráficos 3D proporcionan una forma de visualizar datos en tres dimensiones.

Gráficos 3D Básicos

  • Gráficos de Malla: Usa la función mesh para crear gráficos de malla 3D.
  • Gráficos de Superficie: Usa la función surf para gráficos de superficie 3D.
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

figure;
mesh(X, Y, Z);
title('Gráfico de Malla 3D');
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');

Personalización de Gráficos 3D

  • Mapas de Color: Usa la función colormap para cambiar el esquema de colores de tus gráficos 3D.
  • Iluminación y Sombreado: Usa las funciones lighting y shading para mejorar la apariencia de los gráficos 3D.
figure;
surf(X, Y, Z);
colormap('jet');
shading interp;
lighting phong;
title('Gráfico de Superficie 3D con Personalización');
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');

Funciones de Gráficos Especializadas

MATLAB proporciona funciones de gráficos especializadas para tipos específicos de datos.

Gráficos de Contorno

  • Gráficos de Contorno: Usa la función contour para crear gráficos de contorno.
figure;
contour(X, Y, Z);
title('Gráfico de Contorno');
xlabel('Eje X');
ylabel('Eje Y');

Gráficos Polares

  • Gráficos Polares: Usa la función polarplot para datos en coordenadas polares.
theta = linspace(0, 2*pi, 100);
rho = abs(sin(2*theta) .* cos(2*theta));

figure;
polarplot(theta, rho);
title('Gráfico Polar');

Conclusión

En esta sección, exploramos técnicas avanzadas de gráficos en MATLAB, incluyendo la personalización de gráficos, la creación de gráficos 3D y el uso de funciones de gráficos especializadas. Estas técnicas te ayudarán a crear visualizaciones más informativas y visualmente atractivas. Practica estas técnicas con tus propios datos para volverte competente en gráficos avanzados con MATLAB.
