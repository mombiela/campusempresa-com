Introducción

En esta sección, profundizaremos en las funciones avanzadas de arreglos en MATLAB. Estas funciones son esenciales para la manipulación y el análisis eficiente de datos. Al final de esta sección, podrás realizar operaciones complejas en arreglos con facilidad.

Conceptos Clave

  • Manipulación de Arreglos: Técnicas para cambiar la forma, ordenar y filtrar arreglos.
  • Operaciones Matemáticas: Funciones matemáticas avanzadas que operan en arreglos.
  • Operaciones Lógicas: Uso de indexación y operaciones lógicas para manipular arreglos.
  • Funciones Especializadas: Funciones diseñadas para tareas específicas como encontrar elementos únicos, interpolar datos y más.

Manipulación de Arreglos

Cambiar la Forma de los Arreglos

Cambiar la forma de los arreglos te permite modificar las dimensiones de un arreglo sin cambiar sus datos.

A = 1:12; % Un arreglo 1x12
B = reshape(A, [3, 4]); % Cambiar la forma de A a un arreglo 3x4
disp(B);

Salida:

     1     4     7    10
     2     5     8    11
     3     6     9    12

Ordenar Arreglos

Ordenar arreglos ayuda a organizar los datos en orden ascendente o descendente.

C = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5];
sortedC = sort(C); % Ordenar en orden ascendente
disp(sortedC);

Salida:

     1     1     2     3     3     4     5     5     5     6     9

Filtrar Arreglos

Filtrar te permite extraer elementos que cumplen ciertos criterios.

D = [10, 20, 30, 40, 50];
filteredD = D(D > 25); % Extraer elementos mayores que 25
disp(filteredD);

Salida:

    30    40    50

Operaciones Matemáticas

Operaciones Elemento por Elemento

Realizar operaciones en cada elemento del arreglo individualmente.

E = [1, 2, 3, 4];
F = [5, 6, 7, 8];
G = E .* F; % Multiplicación elemento por elemento
disp(G);

Salida:

     5    12    21    32

Operaciones de Matrices

Realizar operaciones en arreglos como matrices.

H = [1, 2; 3, 4];
I = [5, 6; 7, 8];
J = H * I; % Multiplicación de matrices
disp(J);

Salida:

    19    22
    43    50

Operaciones Lógicas

Indexación Lógica

Usar condiciones lógicas para indexar arreglos.

K = [10, 20, 30, 40, 50];
logicalIndex = K > 25;
disp(K(logicalIndex)); % Elementos mayores que 25

Salida:

    30    40    50

Funciones Lógicas

Usar funciones lógicas para realizar operaciones.

L = [1, 0, 1, 0, 1];
M = [0, 1, 0, 1, 0];
N = L & M; % AND lógico
disp(N);

Salida:

     0     0     0     0     0

Funciones Especializadas

Elementos Únicos

Encontrar elementos únicos en un arreglo.

O = [1, 2, 2, 3, 4, 4, 4, 5];
uniqueO = unique(O);
disp(uniqueO);

Salida:

     1     2     3     4     5

Interpolación

Interpolar puntos de datos dentro de un arreglo.

x = 1:10;
y = sin(x);
xi = 1:0.5:10;
yi = interp1(x, y, xi, 'linear');
disp(yi);

Salida:

    0.8415    0.9093    0.1411   -0.7568   -0.9589   -0.2794    0.6570    0.9894    0.4121   -0.5440   -0.9999   -0.5366    0.4202    0.9906    0.6503   -0.2879   -0.9614   -0.7539    0.1499    0.9129

Conclusión

En esta sección, exploramos funciones avanzadas de arreglos en MATLAB, incluyendo manipulación de arreglos, operaciones matemáticas, operaciones lógicas y funciones especializadas. Dominar estas funciones mejorará significativamente tu capacidad para manejar y analizar datos de manera eficiente en MATLAB. Practica estos conceptos con diferentes conjuntos de datos para volverte competente en el uso de funciones avanzadas de arreglos.
