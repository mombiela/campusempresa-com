Introducción

En esta sección, cubriremos las operaciones y funciones fundamentales en MATLAB. Esto incluye operaciones aritméticas básicas, manipulaciones de matrices y la creación y uso de funciones. Al final de esta sección, serás capaz de realizar cálculos esenciales y escribir funciones simples en MATLAB.

Operaciones Aritméticas Básicas

MATLAB está diseñado para cálculos numéricos, y entender las operaciones aritméticas básicas es crucial.

  • Adición (+)
  • Sustracción (-)
  • Multiplicación (*)
  • División (/ y \)
  • Exponenciación (^)

Ejemplos

a = 5;
b = 3;

% Adición
c = a + b; % c = 8

% Sustracción
d = a - b; % d = 2

% Multiplicación
e = a * b; % e = 15

% División
f = a / b; % f ≈ 1.6667
g = a \ b; % g = 0.6 (equivalente a b / a)

% Exponenciación
h = a ^ b; % h = 125

Operaciones con Matrices

MATLAB sobresale en operaciones con matrices, las cuales son fundamentales en muchas aplicaciones científicas y de ingeniería.

  • Adición y Sustracción de Matrices
  • Multiplicación de Matrices
  • Operaciones Elemento a Elemento
  • Transposición

Ejemplos

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];

% Adición de Matrices
C = A + B; % C = [6, 8; 10, 12]

% Sustracción de Matrices
D = A - B; % D = [-4, -4; -4, -4]

% Multiplicación de Matrices
E = A * B; % E = [19, 22; 43, 50]

% Multiplicación Elemento a Elemento
F = A .* B; % F = [5, 12; 21, 32]

% Transposición
G = A'; % G = [1, 3; 2, 4]

Funciones Integradas

MATLAB proporciona una vasta biblioteca de funciones integradas para diversas tareas, incluyendo cálculos matemáticos, análisis de datos y visualización.

  • Funciones Matemáticas: sum, mean, max, min, sqrt, abs
  • Funciones Trigonométricas: sin, cos, tan, asin, acos, atan
  • Funciones Estadísticas: median, std, var

Ejemplos

% Funciones Matemáticas
x = [1, 2, 3, 4, 5];

total = sum(x); % total = 15
promedio = mean(x); % promedio = 3
máximo = max(x); % máximo = 5
mínimo = min(x); % mínimo = 1
raíz_cuadrada = sqrt(x); % raíz_cuadrada = [1, 1.4142, 1.7321, 2, 2.2361]
absoluto = abs([-1, -2, -3]); % absoluto = [1, 2, 3]

% Funciones Trigonométricas
ángulo = pi / 4; % 45 grados

seno = sin(ángulo); % seno ≈ 0.7071
coseno = cos(ángulo); % coseno ≈ 0.7071
tangente = tan(ángulo); % tangente = 1

Creación de Funciones

Crear funciones personalizadas en MATLAB te permite encapsular código para su reutilización y mejor organización.

Sintaxis

function [salida1, salida2, ...] = nombreFuncion(entrada1, entrada2, ...)
    % Descripción de la función
    % Explicación detallada aquí

    % Código de la función
    salida1 = ...;
    salida2 = ...;
end

Ejemplo

function [área, circunferencia] = propiedadesCírculo(radio)
    % propiedadesCírculo calcula el área y la circunferencia de un círculo
    % dado su radio.

    % Área del círculo
    área = pi * radio^2;

    % Circunferencia del círculo
    circunferencia = 2 * pi * radio;
end

% Uso
[áreaRadio, circunferenciaRadio] = propiedadesCírculo(5);
% áreaRadio = 78.5398
% circunferenciaRadio = 31.4159

Conclusión

En esta sección, cubrimos los conceptos básicos de las operaciones y funciones en MATLAB. Aprendiste cómo realizar operaciones aritméticas y con matrices, usar funciones integradas y crear funciones personalizadas. Estas habilidades fundamentales son esenciales para temas más avanzados en programación con MATLAB. Practica estos conceptos con varios ejemplos para solidificar tu comprensión y prepararte para tareas más complejas.
