Arrays de Celdas en MATLAB
Estructuras ►

Los arrays de celdas son un tipo de dato versátil en MATLAB que te permite almacenar arrays de diferentes tamaños y tipos. Esto los hace particularmente útiles para manejar datos heterogéneos. En esta sección, cubriremos los conceptos básicos de los arrays de celdas, cómo crearlos y manipularlos, y algunas técnicas avanzadas.

Introducción a los Arrays de Celdas

  • Definición: Un array de celdas es un tipo de dato con contenedores de datos indexados llamados celdas, donde cada celda puede contener cualquier tipo de dato.
  • Casos de Uso: Útil para almacenar una colección de diferentes tipos de datos, como cadenas, números y otros arrays.

Creación de Arrays de Celdas

Usando la Función `cell`

La función cell crea un array de celdas vacío de tamaño especificado.

C = cell(3, 2); % Crea un array de celdas de 3x2

Asignación Directa

Puedes asignar valores directamente a las celdas usando llaves {}.

C{1, 1} = 42;          % Asigna un número
C{1, 2} = 'Hola';      % Asigna una cadena
C{2, 1} = [1, 2, 3];   % Asigna un array numérico
C{2, 2} = {1, 'a', 3}; % Asigna un array de celdas

Combinando Diferentes Tipos de Datos

C = {42, 'Hola'; [1, 2, 3], {1, 'a', 3}};

Acceso a Datos en Arrays de Celdas

  • Usando Llaves {}: Para acceder al contenido de una celda.
  • Usando Paréntesis (): Para acceder a la celda en sí.

Ejemplos

% Accediendo al contenido
value = C{1, 1}; % Devuelve 42

% Accediendo a la celda
cellContent = C(1, 1); % Devuelve {42}

Manipulación de Arrays de Celdas

Añadiendo Elementos

Puedes añadir elementos a un array de celdas asignando un valor a una nueva celda.

C{3, 1} = 'Nuevo Elemento';

Eliminando Elementos

Puedes eliminar elementos asignando un array vacío [] a la celda.

C{1, 1} = [];

Redimensionando Arrays de Celdas

Los arrays de celdas pueden redimensionarse dinámicamente asignando valores fuera de las dimensiones actuales.

C{4, 3} = 'Otro Elemento'; % Expande el array de celdas a 4x3

Técnicas Avanzadas

Funciones de Celdas

MATLAB proporciona varias funciones para trabajar con arrays de celdas:

  • cellfun: Aplica una función a cada celda.
  • celldisp: Muestra el contenido de un array de celdas.
  • cell2mat: Convierte un array de celdas de matrices en una sola matriz.

Ejemplo: Usando `cellfun`

C = {1, 2, 3; 4, 5, 6};
result = cellfun(@(x) x^2, C); % Eleva al cuadrado cada elemento en el array de celdas

Ejemplo: Usando `cell2mat`

C = {1, 2; 3, 4};
matrix = cell2mat(C); % Convierte a una matriz de 2x2

Resumen

Los arrays de celdas en MATLAB son una herramienta poderosa para manejar datos heterogéneos. Te permiten almacenar diferentes tipos y tamaños de datos en un solo contenedor. Entender cómo crear, acceder y manipular arrays de celdas es esencial para el manejo avanzado de datos en MATLAB. Al dominar los arrays de celdas, puedes escribir código MATLAB más flexible y eficiente.
