Introducción

En este proyecto, exploraremos cómo usar MATLAB para aplicaciones de procesamiento de señales. El procesamiento de señales es un aspecto crucial de muchas aplicaciones de ingeniería y científicas, incluyendo el procesamiento de audio, comunicaciones e ingeniería biomédica. Al final de este proyecto, tendrás una comprensión sólida de cómo manipular y analizar señales usando MATLAB.

Conceptos Clave

  • Representación de Señales: Comprender cómo se representan las señales en MATLAB.
  • Filtrado: Aplicar diferentes tipos de filtros a las señales.
  • Transformada de Fourier: Usar la Transformada de Fourier para analizar los componentes de frecuencia de las señales.
  • Visualización de Señales: Técnicas para visualizar señales tanto en el dominio del tiempo como en el de la frecuencia.

Representación de Señales

Las señales en MATLAB se representan típicamente como vectores o matrices. Comencemos con un ejemplo simple de creación y trazado de una onda sinusoidal.

Ejemplo: Creación y Trazado de una Onda Sinusoidal

% Definir parámetros
fs = 1000; % Frecuencia de muestreo en Hz
t = 0:1/fs:1-1/fs; % Vector de tiempo de 0 a 1 segundo
f = 5; % Frecuencia de la onda sinusoidal en Hz

% Crear onda sinusoidal
x = sin(2*pi*f*t);

% Trazar la onda sinusoidal
figure;
plot(t, x);
title('Onda Sinusoidal');
xlabel('Tiempo (s)');
ylabel('Amplitud');

En este ejemplo:

  • fs es la frecuencia de muestreo.
  • t es el vector de tiempo.
  • f es la frecuencia de la onda sinusoidal.
  • x es la señal de la onda sinusoidal.

Filtrado

El filtrado se usa para eliminar componentes no deseados de una señal. MATLAB proporciona varias funciones integradas para el filtrado.

Ejemplo: Aplicación de un Filtro Pasa-Bajos

% Crear una señal ruidosa
x_noisy = x + 0.5*randn(size(x));

% Diseñar un filtro pasa-bajos
fc = 10; % Frecuencia de corte en Hz
[b, a] = butter(6, fc/(fs/2)); % Filtro Butterworth de 6º orden

% Aplicar el filtro
x_filtered = filter(b, a, x_noisy);

% Trazar las señales original y filtrada
figure;
subplot(2, 1, 1);
plot(t, x_noisy);
title('Señal Ruidosa');
xlabel('Tiempo (s)');
ylabel('Amplitud');

subplot(2, 1, 2);
plot(t, x_filtered);
title('Señal Filtrada');
xlabel('Tiempo (s)');
ylabel('Amplitud');

En este ejemplo:

  • x_noisy es la onda sinusoidal original con ruido añadido.
  • butter diseña un filtro pasa-bajos Butterworth.
  • filter aplica el filtro diseñado a la señal ruidosa.

Transformada de Fourier

La Transformada de Fourier se usa para analizar los componentes de frecuencia de una señal. La función fft de MATLAB calcula la Transformada Rápida de Fourier.

Ejemplo: Cálculo de la Transformada de Fourier

% Calcular la Transformada de Fourier
X = fft(x);

% Calcular el vector de frecuencia
N = length(x);
f = (0:N-1)*(fs/N);

% Trazar la magnitud de la Transformada de Fourier
figure;
plot(f, abs(X));
title('Magnitud de la Transformada de Fourier');
xlabel('Frecuencia (Hz)');
ylabel('Magnitud');

En este ejemplo:

  • fft calcula la Transformada de Fourier de la señal.
  • f es el vector de frecuencia.
  • abs(X) da la magnitud de la Transformada de Fourier.

Visualización de Señales

Visualizar señales ayuda a comprender sus características. MATLAB proporciona varias funciones de trazado para este propósito.

Ejemplo: Visualización de una Señal en los Dominios del Tiempo y la Frecuencia

% Trazar la señal en el dominio del tiempo
figure;
subplot(2, 1, 1);
plot(t, x);
title('Señal en el Dominio del Tiempo');
xlabel('Tiempo (s)');
ylabel('Amplitud');

% Calcular y trazar la Transformada de Fourier
X = fft(x);
f = (0:N-1)*(fs/N);

subplot(2, 1, 2);
plot(f, abs(X));
title('Señal en el Dominio de la Frecuencia');
xlabel('Frecuencia (Hz)');
ylabel('Magnitud');

En este ejemplo:

  • La señal se traza tanto en los dominios del tiempo como de la frecuencia para una mejor comprensión.

Conclusión

En este proyecto, cubrimos los conceptos básicos del procesamiento de señales usando MATLAB. Aprendimos cómo representar señales, aplicar filtros, calcular la Transformada de Fourier y visualizar señales. Estas habilidades son fundamentales para cualquiera que trabaje con aplicaciones de procesamiento de señales. Continúa practicando con diferentes tipos de señales y filtros para profundizar tu comprensión.
