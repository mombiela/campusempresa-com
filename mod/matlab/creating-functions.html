Introducción a las Funciones

Las funciones son un bloque de construcción fundamental en la programación de MATLAB. Permiten encapsular el código en bloques reutilizables, haciendo que tus programas sean más modulares, legibles y mantenibles.

Conceptos Clave

  • Definición de Función: Cómo definir una función en MATLAB.
  • Argumentos de Entrada y Salida: Manejo de entradas y salidas en funciones.
  • Ámbito y Duración: Comprender el ámbito de las variables dentro de las funciones.
  • Funciones Anónimas: Crear funciones simples y sin nombre.

Definiendo una Función

En MATLAB, una función se define en un archivo separado con el mismo nombre que la función. El archivo debe tener una extensión .m.

Estructura Básica

function [output1, output2, ...] = functionName(input1, input2, ...)
    % Descripción de la función
    % Explicación detallada va aquí
    
    % Código de la función
    output1 = ...;
    output2 = ...;
end

Ejemplo

Vamos a crear una función simple que sume dos números.

function result = addNumbers(a, b)
    % addNumbers suma dos números a y b
    result = a + b;
end

Para usar esta función, guárdala en un archivo llamado addNumbers.m y llámala desde la ventana de comandos de MATLAB u otro script:

sum = addNumbers(3, 5);
disp(sum);  % Salida: 8

Argumentos de Entrada y Salida

Las funciones pueden tener múltiples argumentos de entrada y salida. El número de entradas y salidas debe coincidir con la definición de la función al llamarla.

Ejemplo con Múltiples Salidas

function [sum, difference] = addAndSubtract(a, b)
    % addAndSubtract devuelve la suma y la diferencia de a y b
    sum = a + b;
    difference = a - b;
end

Llamando a la función:

[sum, diff] = addAndSubtract(10, 4);
disp(sum);  % Salida: 14
disp(diff); % Salida: 6

Ámbito y Duración

Las variables definidas dentro de una función son locales a esa función y no pueden ser accedidas desde fuera. Esto se conoce como el ámbito de la variable.

Ejemplo

function result = multiplyNumbers(a, b)
    % multiplyNumbers multiplica dos números a y b
    product = a * b;  % 'product' es local a esta función
    result = product;
end

Intentar acceder a product fuera de la función resultará en un error.

Funciones Anónimas

Las funciones anónimas son funciones simples de una línea que no requieren un archivo separado. Son útiles para operaciones cortas.

Ejemplo

square = @(x) x^2;
result = square(5);
disp(result);  % Salida: 25

Conclusión

Crear funciones en MATLAB es una forma poderosa de organizar y reutilizar código. Al comprender cómo definir funciones, manejar argumentos de entrada y salida, y gestionar el ámbito de las variables, puedes escribir programas de MATLAB más eficientes y legibles. Además, las funciones anónimas proporcionan una forma rápida de definir operaciones simples sin la necesidad de archivos separados.
