Descripción General

El examen integral está diseñado para evaluar tu comprensión y competencia en MATLAB, cubriendo temas desde niveles principiantes hasta avanzados. Este examen probará tu conocimiento en varios aspectos de MATLAB, incluyendo sintaxis básica, manipulación de datos, funciones avanzadas y desarrollo de aplicaciones.

Estructura del Examen

El examen está dividido en tres secciones:

  1. Nivel Básico
  2. Nivel Intermedio
  3. Nivel Avanzado

Cada sección contendrá preguntas de opción múltiple, preguntas de respuesta corta y ejercicios prácticos de codificación.

Nivel Básico

Conceptos Clave

  • Entorno MATLAB
    • Ventana de Comandos
    • Espacio de Trabajo
    • Editor
  • Sintaxis Básica
    • Variables y Tipos de Datos
    • Operaciones Aritméticas
    • Funciones Básicas
  • Estructuras de Control
    • Sentencias If-Else
    • Bucles For
    • Bucles While

Preguntas de Ejemplo

  1. Opción Múltiple:

    • ¿Cuál es el comando para borrar todas las variables del espacio de trabajo?
      • a) clear all
      • b) clc
      • c) clearvars
      • d) reset

  2. Respuesta Corta:

    • Explica la diferencia entre = y == en MATLAB.

  3. Ejercicio de Codificación:

    • Escribe un script que calcule el factorial de un número dado usando un bucle for.
% Cálculo del Factorial
n = 5; % Número de ejemplo
factorial = 1;
for i = 1:n
    factorial = factorial * i;
end
disp(['El factorial de ', num2str(n), ' es ', num2str(factorial)]);

Preguntas de Ejemplo

  1. Opción Múltiple:

    • ¿Qué función se usa para leer datos de un archivo de texto?
      • a) fopen
      • b) fread
      • c) textread
      • d) readtable

  2. Respuesta Corta:

    • Describe cómo crear una matriz identidad de 3x3 en MATLAB.

  3. Ejercicio de Codificación:

    • Escribe un script para graficar las funciones seno y coseno en el mismo gráfico con diferentes colores y agrega una leyenda.
% Graficando Seno y Coseno
x = linspace(0, 2*pi, 100);
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'r', 'DisplayName', 'Seno');
hold on;
plot(x, y2, 'b', 'DisplayName', 'Coseno');
legend show;
title('Funciones Seno y Coseno');
xlabel('x');
ylabel('y');
hold off;

Preguntas de Ejemplo

  1. Opción Múltiple:

    • ¿Qué función se usa para encontrar el mínimo de una función en MATLAB?
      • a) fminsearch
      • b) min
      • c) fminbnd
      • d) minimize

  2. Respuesta Corta:

    • Explica el uso de manejadores de funciones en MATLAB.

  3. Ejercicio de Codificación:

    • Crea una GUI simple que permita al usuario ingresar un número y muestre su cuadrado.
% GUI Simple para Calcular el Cuadrado de un Número
function simple_gui
    fig = uifigure('Position', [100 100 300 200]);
    lbl = uilabel(fig, 'Position', [100 150 100 22], 'Text', 'Ingresa un número:');
    edt = uieditfield(fig, 'numeric', 'Position', [100 120 100 22]);
    btn = uibutton(fig, 'Position', [100 80 100 22], 'Text', 'Calcular', ...
        'ButtonPushedFcn', @(btn,event) calculateSquare(edt.Value));
    lblResult = uilabel(fig, 'Position', [100 50 100 22], 'Text', '');
    
    function calculateSquare(num)
        result = num^2;
        lblResult.Text = ['Cuadrado: ', num2str(result)];
    end
end
