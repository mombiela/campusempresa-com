Introducción

Personalizar gráficas en MATLAB es esencial para crear gráficos claros, informativos y visualmente atractivos. Esta sección te guiará a través de varias técnicas de personalización, desde niveles básicos hasta avanzados.

Personalización Básica de Gráficas

Cambiar Estilos y Colores de Líneas

  • Estilos de Línea: MATLAB ofrece diferentes estilos de línea como sólido (-), discontinuo (--), punteado (:) y guion-punto (-.).
  • Colores: Puedes especificar colores usando nombres cortos ('r' para rojo, 'g' para verde, etc.), tripletes RGB o códigos de color hexadecimal.
x = 0:0.1:10;
y = sin(x);

figure;
plot(x, y, 'r--'); % Línea roja discontinua
title('Onda Senoidal');
xlabel('x');
ylabel('sin(x)');

Añadir Títulos y Etiquetas

  • Título: Usa la función title para añadir un título a tu gráfica.
  • Etiquetas de Ejes: Usa xlabel y ylabel para etiquetar el eje x y el eje y, respectivamente.
title('Onda Senoidal');
xlabel('x');
ylabel('sin(x)');

Leyendas

  • Leyenda: Usa la función legend para añadir una leyenda a tu gráfica.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'b-', x, y2, 'r--');
legend('sin(x)', 'cos(x)');

Personalización Intermedia de Gráficas

Cuadrícula y Caja

  • Cuadrícula: Usa los comandos grid on y grid off para activar o desactivar la cuadrícula.
  • Caja: Usa los comandos box on y box off para activar o desactivar la caja alrededor de la gráfica.
grid on;
box on;

Límites y Marcas de Ejes

  • Límites de Ejes: Usa las funciones xlim y ylim para establecer los límites del eje x y del eje y.
  • Marcas: Usa las funciones xticks y yticks para personalizar las marcas de los ejes.
xlim([0 10]);
ylim([-1 1]);
xticks(0:2:10);
yticks(-1:0.5:1);

Múltiples Gráficas

  • Hold On/Off: Usa hold on para superponer múltiples gráficas y hold off para dejar de superponer.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'b-');
hold on;
plot(x, y2, 'r--');
hold off;

Personalización Avanzada de Gráficas

Subgráficas

  • Subgráfica: Usa la función subplot para crear múltiples gráficas en una sola figura.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
subplot(2, 1, 1); % 2 filas, 1 columna, 1ª gráfica
plot(x, y1);
title('Onda Senoidal');

subplot(2, 1, 2); % 2 filas, 1 columna, 2ª gráfica
plot(x, y2);
title('Onda Cosenoidal');

Anotaciones

  • Anotaciones de Texto: Usa la función text para añadir anotaciones de texto a tu gráfica.
  • Flechas y Formas: Usa la función annotation para añadir flechas y formas.
x = 0:0.1:10;
y = sin(x);

figure;
plot(x, y);
text(5, 0, 'Punto Medio');
annotation('arrow', [0.5, 0.7], [0.5, 0.7]);

Personalización de la Apariencia de la Gráfica

  • Propiedades de la Figura: Usa la función set para personalizar propiedades de la figura como el color de fondo, tamaño, etc.
  • Propiedades de los Ejes: Personaliza las propiedades de los ejes usando la función set.
figure;
set(gcf, 'Color', [0.8 0.8 0.8]); % Establecer color de fondo de la figura

axes;
set(gca, 'FontSize', 14, 'LineWidth', 2); % Establecer propiedades de los ejes

Conclusión

Personalizar gráficas en MATLAB te permite crear gráficos más informativos y visualmente atractivos. Al dominar estas técnicas, puedes comunicar efectivamente tus datos y resultados. Comienza con personalizaciones básicas y gradualmente explora características más avanzadas para mejorar tus gráficas.
