Introducción

La indexación de arrays y matrices es un concepto fundamental en MATLAB que te permite acceder y manipular elementos dentro de arrays y matrices. Este tema cubre los conceptos básicos de la indexación, técnicas avanzadas de indexación y ejemplos prácticos para ayudarte a ser competente en el manejo de arrays y matrices en MATLAB.

Indexación Básica

  • Arrays Unidimensionales (Vectores):
    • Accede a los elementos usando paréntesis ().
    • La indexación comienza en 1 (no en 0 como en muchos otros lenguajes de programación).
% Crear un vector fila
row_vector = [10, 20, 30, 40, 50];

% Acceder al tercer elemento
third_element = row_vector(3); % Salida: 30
  • Arrays Bidimensionales (Matrices):
    • Usa índices de fila y columna para acceder a los elementos.
    • Sintaxis: matrix(row, column).
% Crear una matriz 3x3
matrix = [1, 2, 3; 4, 5, 6; 7, 8, 9];

% Acceder al elemento en la segunda fila, tercera columna
element = matrix(2, 3); % Salida: 6

Técnicas Avanzadas de Indexación

  • Operador de Dos Puntos (:):
    • Se usa para acceder a un rango de elementos o filas/columnas enteras.
% Acceder a todos los elementos en la segunda fila
second_row = matrix(2, :); % Salida: [4, 5, 6]

% Acceder a todos los elementos en la tercera columna
third_column = matrix(:, 3); % Salida: [3; 6; 9]
  • Indexación Lógica:
    • Usa condiciones lógicas para indexar arrays.
% Crear un array lógico donde los elementos > 4
logical_index = matrix > 4;

% Usar el array lógico para indexar la matriz
filtered_elements = matrix(logical_index); % Salida: [5, 6, 7, 8, 9]
  • Indexación Lineal:
    • Trata la matriz como un vector columna único.
% Acceder al quinto elemento en orden de columna mayor
linear_element = matrix(5); % Salida: 5

Modificación de Elementos

  • Modificación de un Solo Elemento:
% Modificar el elemento en la primera fila, segunda columna
matrix(1, 2) = 20; % la matriz se convierte en [1, 20, 3; 4, 5, 6; 7, 8, 9]
  • Modificación de Múltiples Elementos:
% Modificar una fila entera
matrix(2, :) = [40, 50, 60]; % la matriz se convierte en [1, 20, 3; 40, 50, 60; 7, 8, 9]

Funciones Especiales de Indexación

  • Palabra Clave end:
    • Representa el último índice en una dimensión.
% Acceder al último elemento en el vector
last_element = row_vector(end); % Salida: 50

% Acceder al último elemento en la primera fila
last_element_row = matrix(1, end); % Salida: 3
  • sub2ind y ind2sub:
    • Convertir entre índices de subíndice y lineales.
% Convertir subíndice a índice lineal
linear_index = sub2ind(size(matrix), 3, 2); % Salida: 8

% Convertir índice lineal a subíndice
[row, col] = ind2sub(size(matrix), 8); % Salida: row = 3, col = 2

Ejemplos Prácticos

  • Ejemplo 1: Extracción de una Submatriz
% Extraer una submatriz 2x2 desde la esquina superior izquierda
submatrix = matrix(1:2, 1:2); % Salida: [1, 20; 40, 50]
  • Ejemplo 2: Reemplazo de Elementos Basado en una Condición
% Reemplazar todos los elementos mayores que 5 con 0
matrix(matrix > 5) = 0; % la matriz se convierte en [1, 20, 3; 40, 50, 0; 7, 8, 9]

Conclusión

Entender la indexación de arrays y matrices en MATLAB es crucial para una manipulación y análisis de datos eficientes. Al dominar las técnicas básicas y avanzadas de indexación, puedes realizar operaciones complejas con facilidad. Practica estos conceptos con varios ejemplos para solidificar tu comprensión y volverte competente en la programación en MATLAB.
