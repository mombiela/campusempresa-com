Introducción a los Arrays

Los arrays en MUMPS (también conocido como M) son una estructura de datos fundamental utilizada para almacenar y manipular colecciones de datos. A diferencia de los arrays en muchos otros lenguajes de programación, los arrays en MUMPS son asociativos, lo que significa que están indexados por claves que pueden ser cadenas o números.

Conceptos Clave

  • Arrays Asociativos: Arrays indexados por claves, que pueden ser cadenas o números.
  • Subíndices: Las claves utilizadas para indexar arrays.
  • Arrays Globales: Arrays que se almacenan en disco y persisten entre sesiones.
  • Arrays Locales: Arrays que existen solo durante la ejecución de un programa.

Trabajando con Arrays Locales

Los arrays locales son temporales y existen solo durante la ejecución de un programa. Son útiles para almacenar datos que no necesitan persistir.

Creación y Acceso a Arrays Locales

Para crear y acceder a arrays locales, usa la siguiente sintaxis:

SET array("key1") = "value1"
SET array("key2") = "value2"
WRITE array("key1"), !
WRITE array("key2"), !

Ejemplo

SET fruits("apple") = 10
SET fruits("banana") = 20
WRITE "Apple count: ", fruits("apple"), !
WRITE "Banana count: ", fruits("banana"), !

Iterando sobre Arrays Locales

Puedes iterar sobre arrays locales usando el bucle FOR:

SET fruits("apple") = 10
SET fruits("banana") = 20
SET fruits("cherry") = 30

SET key = ""
FOR  SET key=$ORDER(fruits(key)) QUIT:key=""  DO
. WRITE key, ": ", fruits(key), !

Trabajando con Arrays Globales

Los arrays globales se almacenan en disco y persisten entre sesiones. Se identifican por un prefijo de caret (^).

Creación y Acceso a Arrays Globales

Para crear y acceder a arrays globales, usa la siguiente sintaxis:

SET ^globalArray("key1") = "value1"
SET ^globalArray("key2") = "value2"
WRITE ^globalArray("key1"), !
WRITE ^globalArray("key2"), !

Ejemplo

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
WRITE "Apple inventory: ", ^inventory("apple"), !
WRITE "Banana inventory: ", ^inventory("banana"), !

Iterando sobre Arrays Globales

Puedes iterar sobre arrays globales de manera similar a los arrays locales:

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
SET ^inventory("cherry") = 300

SET key = ""
FOR  SET key=$ORDER(^inventory(key)) QUIT:key=""  DO
. WRITE key, ": ", ^inventory(key), !

Temas Avanzados

Arrays Multidimensionales

MUMPS soporta arrays multidimensionales, que son arrays de arrays.

Ejemplo

SET matrix(1,1) = 10
SET matrix(1,2) = 20
SET matrix(2,1) = 30
SET matrix(2,2) = 40

WRITE "Matrix(1,1): ", matrix(1,1), !
WRITE "Matrix(1,2): ", matrix(1,2), !
WRITE "Matrix(2,1): ", matrix(2,1), !
WRITE "Matrix(2,2): ", matrix(2,2), !

Subíndices Dinámicos

Los subíndices en MUMPS pueden ser generados dinámicamente y utilizados para acceder a elementos del array.

Ejemplo

SET i = 1
SET j = 2
SET matrix(i,j) = 50
WRITE "Matrix(", i, ",", j, "): ", matrix(i,j), !

Conclusión

Los arrays y subíndices son características poderosas en MUMPS que permiten un almacenamiento y manipulación de datos flexibles y eficientes. Entender cómo trabajar con arrays locales y globales, así como conceptos avanzados como arrays multidimensionales y subíndices dinámicos, es esencial para una programación efectiva en MUMPS. Practica creando, accediendo e iterando sobre arrays para volverte competente en el uso de estas estructuras.
