Introducción

MUMPS (Massachusetts General Hospital Utility Multi-Programming System), también conocido como M, es un lenguaje de programación diseñado para la industria de la salud. Es conocido por su sintaxis única y sus potentes capacidades de base de datos. Esta sección te introducirá a la sintaxis y estructura básica de MUMPS, proporcionando una base para temas más avanzados.

Conceptos Clave

  • Líneas y Comandos: El código MUMPS se escribe en líneas, cada una contiene uno o más comandos.
  • Variables: Las variables en MUMPS pueden ser locales o globales.
  • Tipos de Datos: MUMPS usa principalmente cadenas y números.
  • Funciones: Las funciones integradas realizan varias operaciones.
  • Estructuras de Control: Las declaraciones condicionales y los bucles controlan el flujo del programa.

Líneas y Comandos

En MUMPS, cada línea de código puede contener uno o más comandos. Los comandos están separados por espacios, y cada comando puede tener argumentos.

Ejemplo:

WRITE "¡Hola, Mundo!",!
SET X=10
  • WRITE "¡Hola, Mundo!",! imprime "¡Hola, Mundo!" seguido de una nueva línea.
  • SET X=10 asigna el valor 10 a la variable X.

Variables

MUMPS soporta dos tipos de variables: locales y globales.

  • Variables Locales: Estas son temporales y existen solo durante la ejecución de un programa.
  • Variables Globales: Estas se almacenan en la base de datos y persisten entre ejecuciones del programa.

Ejemplo:

SET localVar="Soy local"
SET ^globalVar="Soy global"
  • localVar es una variable local.
  • ^globalVar es una variable global, indicada por el símbolo de caret (^).

Tipos de Datos

MUMPS usa principalmente dos tipos de datos: cadenas y números. No es necesario declarar el tipo explícitamente.

Ejemplo:

SET str="Hola"
SET num=123
  • str es una variable de cadena.
  • num es una variable numérica.

Funciones

MUMPS tiene varias funciones integradas para diversas operaciones, como manipulación de cadenas, cálculos matemáticos y manejo de fechas/horas.

Ejemplo:

SET length=$LENGTH("Hola") ; Devuelve la longitud de la cadena "Hola"
SET upper=$ZCONVERT("hola","U") ; Convierte "hola" a mayúsculas
  • $LENGTH devuelve la longitud de una cadena.
  • $ZCONVERT convierte una cadena a mayúsculas.

Estructuras de Control

MUMPS proporciona estructuras de control como declaraciones condicionales y bucles para controlar el flujo del programa.

Declaraciones Condicionales

IF X=10 WRITE "X es 10",!
ELSE  WRITE "X no es 10",!
  • IF verifica si X es igual a 10.
  • ELSE se ejecuta si la condición es falsa.

Bucles

SET i=1
FOR  SET i=$INCREMENT(i) QUIT:i>5  WRITE i,!
  • El bucle FOR incrementa i e imprime su valor hasta que i sea mayor que 5.

Conclusión

Entender la sintaxis y estructura básica de MUMPS es crucial para escribir programas efectivos. Esta sección cubrió lo esencial, incluyendo líneas y comandos, variables, tipos de datos, funciones y estructuras de control. Con esta base, ahora estás preparado para profundizar en temas más avanzados en la programación MUMPS.
