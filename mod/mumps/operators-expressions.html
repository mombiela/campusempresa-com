Introducción

En esta sección, exploraremos los diversos operadores y expresiones utilizados en MUMPS (M). Comprender estos conceptos fundamentales es crucial para escribir código MUMPS efectivo y eficiente. Cubriremos operadores aritméticos, lógicos y relacionales, así como cómo construir y evaluar expresiones.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos. Los operadores aritméticos comunes en MUMPS incluyen:

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • # : División Entera
  • ** : Exponenciación

Ejemplos

SET A=10, B=3
WRITE "Adición: ", A+B, !
WRITE "Sustracción: ", A-B, !
WRITE "Multiplicación: ", A*B, !
WRITE "División: ", A/B, !
WRITE "División Entera: ", A#B, !
WRITE "Exponenciación: ", A**B, !

Salida:

Adición: 13
Sustracción: 7
Multiplicación: 30
División: 3.333333
División Entera: 3
Exponenciación: 1000

Operadores Lógicos

Los operadores lógicos se utilizan para realizar operaciones lógicas. Los operadores lógicos comunes en MUMPS incluyen:

  • & : Y Lógico
  • ! : O Lógico
  • ' : NO Lógico

Ejemplos

SET X=1, Y=0
WRITE "Y Lógico: ", X&Y, !
WRITE "O Lógico: ", X!Y, !
WRITE "NO Lógico: ", 'X, !

Salida:

Y Lógico: 0
O Lógico: 1
NO Lógico: 0

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores. Los operadores relacionales comunes en MUMPS incluyen:

  • = : Igual a
  • `'= : No igual a
  • < : Menor que
  • > : Mayor que
  • <= : Menor o igual a
  • >= : Mayor o igual a

Ejemplos

SET A=5, B=10
WRITE "Igual a: ", A=B, !
WRITE "No igual a: ", A'=B, !
WRITE "Menor que: ", A<B, !
WRITE "Mayor que: ", A>B, !
WRITE "Menor o igual a: ", A<=B, !
WRITE "Mayor o igual a: ", A>=B, !

Salida:

Igual a: 0
No igual a: 1
Menor que: 1
Mayor que: 0
Menor o igual a: 1
Mayor o igual a: 0

Combinando Operadores en Expresiones

Los operadores se pueden combinar para formar expresiones complejas. El orden de las operaciones (precedencia) en MUMPS sigue las reglas matemáticas estándar:

  1. Paréntesis ()
  2. Exponenciación **
  3. Multiplicación * y División /
  4. Adición + y Sustracción -
  5. Operadores relacionales
  6. Operadores lógicos

Ejemplos

SET A=5, B=10, C=2
WRITE "Expresión 1: ", (A+B)*C, !
WRITE "Expresión 2: ", A+B*C, !
WRITE "Expresión 3: ", A>B&(B>C), !
WRITE "Expresión 4: ", (A<B)!('C), !

Salida:

Expresión 1: 30
Expresión 2: 25
Expresión 3: 0
Expresión 4: 1

Conclusión

Comprender y utilizar eficazmente los operadores y expresiones es esencial para programar en MUMPS. Esta sección cubrió los operadores aritméticos, lógicos y relacionales básicos, y cómo combinarlos para formar expresiones complejas. Practica usando estos operadores en diferentes escenarios para volverte competente en la escritura de código MUMPS.
