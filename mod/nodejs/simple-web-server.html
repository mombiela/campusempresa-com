Creando un Servidor Web Simple
Enrutamiento ►

Introducción

En esta sección, aprenderemos cómo crear un servidor web simple usando Node.js. Esto cubrirá los conceptos básicos de la configuración de un servidor, el manejo de solicitudes y respuestas, y la comprensión de los módulos principales involucrados.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Node.js instalado en tu máquina.
  • Una comprensión básica de JavaScript.

Configurando Tu Entorno

  1. Instalar Node.js: Descarga e instala Node.js desde el sitio web oficial.
  2. Crear un Directorio de Proyecto: Abre tu terminal y crea un nuevo directorio para tu proyecto. 
    mkdir simple-web-server
cd simple-web-server
  3. Inicializar un Proyecto Node.js: Ejecuta el siguiente comando para crear un archivo package.json. 
    npm init -y

Creando el Servidor

  1. Crear el Archivo del Servidor: Crea un archivo llamado server.js en tu directorio de proyecto.

  2. Importar el Módulo HTTP: Node.js tiene un módulo incorporado llamado http que te permite crear un servidor.

    const http = require('http');

  3. Crear el Servidor: Usa el método http.createServer para crear un servidor.

    const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('¡Hola, Mundo!\n');
});

  4. Escuchar en un Puerto: Haz que el servidor escuche en un puerto específico.

    const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Servidor corriendo en http://localhost:${PORT}/`);
});

Ejemplo de Código Completo

const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('¡Hola, Mundo!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Servidor corriendo en http://localhost:${PORT}/`);
});

Ejecutando el Servidor

  1. Iniciar el Servidor: Ejecuta el siguiente comando en tu terminal. 
    node server.js
  2. Acceder al Servidor: Abre tu navegador web y navega a http://localhost:3000. Deberías ver "¡Hola, Mundo!" mostrado.

Manejo de Diferentes Rutas

Para manejar diferentes rutas, puedes inspeccionar la propiedad req.url.

Ejemplo: Manejo de Múltiples Rutas

const http = require('http');

const server = http.createServer((req, res) => {
    if (req.url === '/') {
        res.statusCode = 200;
        res.setHeader('Content-Type', 'text/plain');
        res.end('¡Bienvenido a la Página Principal!\n');
    } else if (req.url === '/about') {
        res.statusCode = 200;
        res.setHeader('Content-Type', 'text/plain');
        res.end('Sobre Nosotros\n');
    } else {
        res.statusCode = 404;
        res.setHeader('Content-Type', 'text/plain');
        res.end('Página No Encontrada\n');
    }
});

const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Servidor corriendo en http://localhost:${PORT}/`);
});

Conceptos Avanzados

Usando Express.js

Para un enrutamiento y middleware más avanzado, puedes usar el framework Express.js.

  1. Instalar Express:

    npm install express

  2. Crear un Servidor Express:

    const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
    res.send('¡Bienvenido a la Página Principal!');
});

app.get('/about', (req, res) => {
    res.send('Sobre Nosotros');
});

app.use((req, res) => {
    res.status(404).send('Página No Encontrada');
});

app.listen(PORT, () => {
    console.log(`Servidor corriendo en http://localhost:${PORT}/`);
});

Comparación: Módulo HTTP vs. Express.js

Característica Módulo HTTP Express.js
Enrutamiento Manejo manual con req.url Enrutamiento incorporado con app.get, etc.
Middleware Implementación manual Soporte incorporado
Simplicidad Básico y mínimo Abstracciones de alto nivel
Rendimiento Ligeramente más rápido para tareas simples Ligeramente más lento debido a características adicionales

Conclusión

En este tema, cubrimos los conceptos básicos de la creación de un servidor web simple usando Node.js. Comenzamos con el módulo http incorporado y luego exploramos un enrutamiento más avanzado con Express.js. Al comprender estos fundamentos, puedes construir servidores web más complejos y ricos en características.
