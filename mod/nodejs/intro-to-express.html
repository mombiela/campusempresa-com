Introducción a Express.js
Configuración de una Aplicación Express ►

Express.js es un framework minimalista y flexible para aplicaciones web en Node.js que proporciona un conjunto robusto de características para desarrollar aplicaciones web y móviles. Facilita el desarrollo rápido de aplicaciones web basadas en Node.

¿Qué es Express.js?

Express.js es un framework de aplicaciones web para Node.js, diseñado para construir aplicaciones web y APIs. Es el estándar de facto para frameworks de servidores en Node.js.

  • Minimalista: Express es un framework minimalista que proporciona una capa delgada de características fundamentales para aplicaciones web.
  • Flexible: Es altamente flexible y permite a los desarrolladores estructurar sus aplicaciones como lo consideren adecuado.
  • Robusto: Express proporciona un conjunto robusto de características para aplicaciones web y móviles.

Configuración de Express.js

Para empezar a usar Express.js, necesitas tener Node.js instalado en tu máquina. Puedes instalar Express.js usando npm (Node Package Manager).

Pasos de Instalación

  1. Inicializar un nuevo proyecto Node.js:

    mkdir myapp
cd myapp
npm init -y

  2. Instalar Express.js:

    npm install express

Creando una Aplicación Básica con Express.js

Vamos a crear una simple aplicación con Express.js para entender su estructura básica.

Estructura Básica de la Aplicación

  1. Crear un archivo llamado app.js:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

app.listen(port, () => {
  console.log(`Aplicación de ejemplo escuchando en http://localhost:${port}`);
});

  2. Ejecutar la aplicación:

    node app.js

  3. Acceder a la aplicación: Abre tu navegador y navega a http://localhost:3000. Deberías ver "¡Hola Mundo!" en pantalla.

Explicación del Código

  • Importando Express:

    const express = require('express');

    Esta línea importa el módulo Express.

  • Creando una aplicación Express:

    const app = express();

    Esta línea inicializa una nueva aplicación Express.

  • Definiendo una ruta:

    app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

    Esto define una ruta para la URL raíz (/). Cuando se hace una solicitud GET a la URL raíz, el servidor responde con "¡Hola Mundo!".

  • Iniciando el servidor:

    app.listen(port, () => {
  console.log(`Aplicación de ejemplo escuchando en http://localhost:${port}`);
});

    Esto inicia el servidor y escucha en el puerto 3000. Cuando el servidor se inicia, registra un mensaje en la consola.

Características Clave de Express.js

  • Middleware: Funciones que se ejecutan durante el ciclo de vida de una solicitud al servidor.
  • Enrutamiento: Define cómo una aplicación responde a una solicitud del cliente a un endpoint particular.
  • Motores de Plantillas: Permite usar archivos de plantillas estáticas en tu aplicación.

Middleware

Las funciones middleware son funciones que tienen acceso al objeto de solicitud (req), al objeto de respuesta (res), y a la siguiente función middleware en el ciclo de solicitud-respuesta de la aplicación.

Ejemplo de un middleware simple:

app.use((req, res, next) => {
  console.log('Hora:', Date.now());
  next();
});

Enrutamiento

El enrutamiento se refiere a cómo los endpoints (URIs) de una aplicación responden a las solicitudes del cliente.

Ejemplo de definición de rutas:

app.get('/about', (req, res) => {
  res.send('Página Acerca de');
});

app.post('/submit', (req, res) => {
  res.send('Formulario Enviado');
});

Motores de Plantillas

Express soporta varios motores de plantillas como Pug, EJS, etc., para generar HTML dinámicamente.

Ejemplo usando Pug:

  1. Instalar Pug:

    npm install pug

  2. Configurar Pug en Express:

    app.set('view engine', 'pug');
app.set('views', './views');

  3. Crear una plantilla Pug (views/index.pug):

    html
  head
    title= title
  body
    h1= message

  4. Renderizar la plantilla Pug:

    app.get('/', (req, res) => {
  res.render('index', { title: 'Hey', message: '¡Hola!' });
});

Conclusión

Express.js es un framework poderoso y flexible para construir aplicaciones web con Node.js. Proporciona un conjunto minimalista pero robusto de características que facilitan el desarrollo de aplicaciones web y móviles. Al entender los conceptos básicos de la configuración de una aplicación Express, la definición de rutas, el uso de middleware y la integración de motores de plantillas, puedes empezar a construir tus propias aplicaciones web de manera eficiente.
