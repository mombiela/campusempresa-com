Configuración del Entorno de Desarrollo
Tu Primera Aplicación Node.js ►

Configurar tu entorno de desarrollo es el primer paso para volverte competente en Node.js. Esta sección te guiará a través del proceso de instalación de Node.js, configuración de un editor de código y verificación de tu instalación.

Instalación de Node.js

Node.js es un entorno de ejecución de JavaScript construido sobre el motor de JavaScript V8 de Chrome. Para comenzar a desarrollar con Node.js, necesitas instalarlo en tu máquina.

Pasos para Instalar Node.js

  1. Descargar Node.js:

    • Ve al sitio web oficial de Node.js.
    • Descarga la versión LTS (Long Term Support) para tu sistema operativo (Windows, macOS o Linux).

  2. Instalar Node.js:

    • Ejecuta el instalador y sigue las instrucciones en pantalla.
    • El instalador también instalará npm (Node Package Manager), que es esencial para gestionar paquetes de Node.js.

  3. Verificar la Instalación:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta los siguientes comandos para verificar si Node.js y npm están instalados correctamente: 
      node -v
npm -v
    • Deberías ver los números de versión de Node.js y npm.

Configuración de un Editor de Código

Un buen editor de código puede mejorar significativamente tu productividad. Para el desarrollo de Node.js, se recomienda encarecidamente Visual Studio Code (VS Code) debido a sus potentes características y extensiones.

Instalación de Visual Studio Code

  1. Descargar VS Code:

    • Visita el sitio web de Visual Studio Code.
    • Descarga el instalador para tu sistema operativo.

  2. Instalar VS Code:

    • Ejecuta el instalador y sigue las instrucciones en pantalla.

  3. Instalar Extensiones:

    • Abre VS Code.
    • Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana o presionando Ctrl+Shift+X.
    • Busca e instala las siguientes extensiones:
      • Node.js Extension Pack: Proporciona una colección de extensiones para el desarrollo de Node.js.
      • ESLint: Ayuda a identificar y corregir problemas en tu código JavaScript.
      • Prettier - Code formatter: Asegura un formato de código consistente.

Creación de tu Primer Proyecto Node.js

Ahora que tienes Node.js y un editor de código configurados, vamos a crear un proyecto simple de Node.js para asegurarnos de que todo esté funcionando correctamente.

Pasos para Crear un Proyecto Node.js

  1. Crear un Directorio de Proyecto:

    • Abre tu terminal o símbolo del sistema.
    • Crea un nuevo directorio para tu proyecto y navega dentro de él: 
      mkdir my-node-project
cd my-node-project

  2. Inicializar el Proyecto:

    • Ejecuta el siguiente comando para crear un archivo package.json, que gestionará las dependencias de tu proyecto: 
      npm init -y

  3. Crear un Archivo de Punto de Entrada:

    • En tu directorio de proyecto, crea un archivo llamado index.js.
    • Abre index.js en tu editor de código y añade el siguiente código: 
      console.log('¡Hola, Node.js!');

  4. Ejecutar tu Aplicación Node.js:

    • En tu terminal, ejecuta el siguiente comando: 
      node index.js
    • Deberías ver la salida ¡Hola, Node.js!.

Resumen

En esta sección, has aprendido cómo configurar tu entorno de desarrollo para Node.js. Instalaste Node.js y npm, configuraste Visual Studio Code con extensiones esenciales y creaste un proyecto simple de Node.js para verificar tu configuración. Con tu entorno listo, ahora estás preparado para profundizar en el desarrollo de Node.js.
