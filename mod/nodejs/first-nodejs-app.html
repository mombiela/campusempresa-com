Introducción

¡Bienvenido a tu primera aplicación Node.js! En esta sección, te guiaremos a través de la configuración de un entorno básico de Node.js y la creación de una aplicación simple. Al final de esta lección, tendrás una base sólida para construir aplicaciones Node.js más complejas.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Node.js instalado en tu computadora. Puedes descargarlo desde nodejs.org.
  • Un editor de código como Visual Studio Code, Sublime Text, o cualquier otro de tu elección.

Configurando Tu Entorno

  1. Instalar Node.js: Sigue las instrucciones en el sitio web de Node.js para descargar e instalar Node.js.
  2. Verificar la Instalación: Abre tu terminal o símbolo del sistema y escribe: 
    node -v
    Este comando debería mostrar la versión de Node.js instalada en tu sistema.

Creando Tu Primera Aplicación Node.js

Vamos a crear una simple aplicación "¡Hola, Mundo!" para comenzar.

Paso 1: Crear un Directorio de Proyecto

Primero, crea un nuevo directorio para tu proyecto y navega dentro de él:

mkdir my-first-node-app
cd my-first-node-app

Paso 2: Inicializar un Proyecto Node.js

Inicializa tu proyecto con npm (Node Package Manager):

npm init -y

Este comando creará un archivo package.json con configuraciones predeterminadas.

Paso 3: Crear el Archivo de la Aplicación

Crea un nuevo archivo llamado app.js en tu directorio de proyecto. Este será el archivo principal de tu aplicación.

Paso 4: Escribir Tu Primer Código Node.js

Abre app.js en tu editor de código y agrega el siguiente código:

// app.js
console.log('¡Hola, Mundo!');

Paso 5: Ejecutar Tu Aplicación

Vuelve a tu terminal y ejecuta la aplicación usando el siguiente comando:

node app.js

Deberías ver la salida:

¡Hola, Mundo!

Entendiendo el Código

  • console.log('¡Hola, Mundo!');: Esta línea de código imprime la cadena '¡Hola, Mundo!' en la consola. El objeto console en Node.js proporciona una consola de depuración simple que es similar a la consola de JavaScript en los navegadores web.

Añadiendo Más Funcionalidad

Vamos a extender nuestra aplicación para crear un servidor HTTP simple que responda con "¡Hola, Mundo!" cuando se acceda a través de un navegador web.

Paso 1: Importar el Módulo HTTP

Actualiza tu archivo app.js para incluir el módulo HTTP:

const http = require('http');

Paso 2: Crear el Servidor

Agrega el siguiente código para crear un servidor HTTP:

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('¡Hola, Mundo!\n');
});

Paso 3: Escuchar en un Puerto

Agrega el siguiente código para hacer que el servidor escuche en el puerto 3000:

const port = 3000;
server.listen(port, () => {
  console.log(`Servidor corriendo en http://localhost:${port}/`);
});

Ejemplo Completo de Código

Aquí está el archivo app.js completo:

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('¡Hola, Mundo!\n');
});

const port = 3000;
server.listen(port, () => {
  console.log(`Servidor corriendo en http://localhost:${port}/`);
});

Ejecutando el Servidor

Ejecuta tu aplicación nuevamente:

node app.js

Abre tu navegador web y navega a http://localhost:3000/. Deberías ver "¡Hola, Mundo!" mostrado en tu navegador.

Resumen

¡Felicidades! Has creado tu primera aplicación Node.js. En esta lección, aprendiste a:

  • Configurar un proyecto Node.js.
  • Escribir una simple aplicación "¡Hola, Mundo!".
  • Crear un servidor HTTP usando Node.js.

Este conocimiento fundamental será crucial a medida que avances hacia temas más avanzados en Node.js. ¡Sigue experimentando y construyendo sobre lo que has aprendido!
