Introducción

En esta sección, cubriremos los pasos esenciales para configurar un proyecto Node.js. Esto incluye instalar Node.js, inicializar un proyecto y comprender la estructura básica de una aplicación Node.js. Al final de esta sección, tendrás un entorno de Node.js funcionando y una estructura básica de proyecto sobre la cual construir.

Conceptos clave

  • Node.js: Un entorno de ejecución de JavaScript construido sobre el motor de JavaScript V8 de Chrome.
  • npm: Node Package Manager, utilizado para gestionar las dependencias del proyecto.
  • package.json: Un archivo que contiene varios metadatos relevantes para el proyecto y se utiliza para gestionar las dependencias, scripts, versión y más del proyecto.

Instalación de Node.js

Para comenzar a trabajar con Node.js, necesitas instalarlo en tu máquina. Sigue estos pasos:

  1. Descargar Node.js: Visita el sitio web oficial de Node.js y descarga la versión LTS (Long Term Support).
  2. Instalar Node.js: Ejecuta el instalador y sigue las instrucciones. Esto también instalará npm.

Verificar la instalación

Abre tu terminal o símbolo del sistema y ejecuta los siguientes comandos para verificar la instalación:

node -v

Esto debería mostrar la versión de Node.js instalada.

npm -v

Esto debería mostrar la versión de npm instalada.

Inicialización de un proyecto Node.js

Una vez que Node.js está instalado, puedes inicializar un nuevo proyecto. Sigue estos pasos:

  1. Crear un directorio de proyecto: Navega al directorio donde quieres crear tu proyecto y crea una nueva carpeta.
mkdir my-node-project
cd my-node-project
  1. Inicializar el proyecto: Usa npm para inicializar el proyecto. Esto creará un archivo package.json.
npm init

Se te pedirá que ingreses varios detalles sobre tu proyecto. Puedes presionar Enter para aceptar los valores predeterminados o proporcionar los tuyos.

Ejemplo de `package.json`

Aquí tienes un ejemplo de un archivo package.json básico:

{
  "name": "my-node-project",
  "version": "1.0.0",
  "description": "Un proyecto simple de Node.js",
  "main": "index.js",
  "scripts": {
    "start": "node index.js"
  },
  "author": "Tu Nombre",
  "license": "ISC"
}

Estructura del proyecto

Una estructura típica de proyecto Node.js podría verse así:

my-node-project/
├── node_modules/
├── package.json
├── index.js
└── README.md
  • node_modules/: Esta carpeta contiene todas las dependencias instaladas a través de npm.
  • package.json: Este archivo contiene metadatos sobre el proyecto y sus dependencias.
  • index.js: El punto de entrada principal de la aplicación.
  • README.md: Un archivo markdown que proporciona una descripción general del proyecto.

Creación de `index.js`

Crea un archivo index.js en tu directorio de proyecto. Este será el punto de entrada de tu aplicación.

// index.js
console.log("¡Hola, Node.js!");

Ejecución de la aplicación

Para ejecutar tu aplicación Node.js, usa el siguiente comando:

npm start

Esto ejecutará el script start definido en tu package.json, que ejecuta node index.js.

Conclusión

En esta sección, cubrimos los conceptos básicos para configurar un proyecto Node.js. Aprendiste cómo instalar Node.js, inicializar un proyecto con npm y crear una estructura básica de proyecto. Con esta base, ahora estás listo para comenzar a construir aplicaciones Node.js más complejas. En las próximas secciones, profundizaremos en las características y mejores prácticas de Node.js.
