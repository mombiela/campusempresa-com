Módulo HTTP en Node.js
Creación de un Servidor Web Simple ►

El módulo HTTP en Node.js te permite crear servidores web y manejar solicitudes y respuestas HTTP. Este módulo es fundamental para construir aplicaciones web y APIs en Node.js.

Introducción al Módulo HTTP

  • El módulo HTTP está integrado en Node.js, por lo que no se requiere instalación adicional.
  • Proporciona funcionalidades para crear tanto servidores HTTP como clientes.

Servidor HTTP Básico

Para crear un servidor HTTP básico, necesitas importar el módulo HTTP y usar el método createServer.

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('¡Hola, Mundo!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en http://localhost:${PORT}/`);
});
  • http.createServer(callback): Crea un nuevo servidor HTTP.
  • req: Representa el objeto de solicitud.
  • res: Representa el objeto de respuesta.
  • res.statusCode: Establece el código de estado HTTP.
  • res.setHeader(): Establece los encabezados de la respuesta.
  • res.end(): Finaliza el proceso de respuesta.

Manejo de Diferentes Métodos HTTP

Métodos HTTP como GET, POST, PUT, DELETE, etc., se utilizan para realizar diferentes acciones en el servidor.

Ejemplo: Manejo de Solicitudes GET y POST

const http = require('http');

const server = http.createServer((req, res) => {
  if (req.method === 'GET') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Recibida una solicitud GET\n');
  } else if (req.method === 'POST') {
    let body = '';
    req.on('data', chunk => {
      body += chunk.toString();
    });
    req.on('end', () => {
      res.statusCode = 200;
      res.setHeader('Content-Type', 'application/json');
      res.end(`Recibida una solicitud POST con datos: ${body}\n`);
    });
  } else {
    res.statusCode = 405;
    res.end('Método No Permitido\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en http://localhost:${PORT}/`);
});
  • req.method: Verifica el método HTTP de la solicitud.
  • req.on('data', callback): Escucha eventos de datos cuando se reciben fragmentos de datos.
  • req.on('end', callback): Escucha el final de la transmisión de datos.

Trabajando con Parámetros de Consulta y Enrutamiento de URL

Análisis de Parámetros de Consulta

Para analizar parámetros de consulta, puedes usar el módulo url.

const http = require('http');
const url = require('url');

const server = http.createServer((req, res) => {
  const parsedUrl = url.parse(req.url, true);
  const query = parsedUrl.query;

  res.statusCode = 200;
  res.setHeader('Content-Type', 'application/json');
  res.end(JSON.stringify(query));
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en http://localhost:${PORT}/`);
});
  • url.parse(urlString, true): Analiza la URL y devuelve un objeto con parámetros de consulta.

Enrutamiento Básico de URL

El enrutamiento te permite manejar diferentes puntos finales.

const http = require('http');

const server = http.createServer((req, res) => {
  const url = req.url;

  if (url === '/') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Página de Inicio\n');
  } else if (url === '/about') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Página Acerca de\n');
  } else {
    res.statusCode = 404;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Página No Encontrada\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en http://localhost:${PORT}/`);
});

Temas Avanzados

Manejo de Datos JSON

Cuando trabajas con APIs, a menudo necesitas manejar datos JSON.

const http = require('http');

const server = http.createServer((req, res) => {
  if (req.method === 'POST' && req.headers['content-type'] === 'application/json') {
    let body = '';
    req.on('data', chunk => {
      body += chunk.toString();
    });
    req.on('end', () => {
      const data = JSON.parse(body);
      res.statusCode = 200;
      res.setHeader('Content-Type', 'application/json');
      res.end(JSON.stringify({ message: 'Datos recibidos', data }));
    });
  } else {
    res.statusCode = 400;
    res.end('Solicitud Incorrecta\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en http://localhost:${PORT}/`);
});

Servidor HTTPS

Para comunicación segura, puedes crear un servidor HTTPS.

const https = require('https');
const fs = require('fs');

const options = {
  key: fs.readFileSync('path/to/private-key.pem'),
  cert: fs.readFileSync('path/to/certificate.pem')
};

const server = https.createServer(options, (req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('¡Hola, Mundo Seguro!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Servidor ejecutándose en https://localhost:${PORT}/`);
});
  • https.createServer(options, callback): Crea un servidor HTTPS con las opciones proporcionadas.

Conclusión

El módulo HTTP en Node.js es una herramienta poderosa para crear servidores web y manejar solicitudes y respuestas HTTP. Al entender las características básicas y avanzadas, puedes construir aplicaciones web y APIs robustas. Practica creando diferentes tipos de servidores y manejando varios métodos HTTP y formatos de datos para solidificar tu comprensión.
