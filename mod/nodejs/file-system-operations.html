Introducción

Las operaciones del sistema de archivos son una parte crucial de muchas aplicaciones, permitiéndote leer, escribir y manipular archivos y directorios. Node.js proporciona un módulo incorporado llamado fs para manejar estas operaciones de manera eficiente.

Operaciones Básicas del Sistema de Archivos

Lectura de Archivos

La lectura de archivos es una de las operaciones más comunes. Node.js proporciona métodos tanto sincrónicos como asincrónicos para leer archivos.

Lectura Asincrónica

Los métodos asincrónicos no bloquean, lo que significa que el programa puede continuar ejecutándose mientras se lee el archivo.

const fs = require('fs');

fs.readFile('example.txt', 'utf8', (err, data) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log(data);
});

Lectura Sincrónica

Los métodos sincrónicos bloquean la ejecución hasta que la operación se completa.

const fs = require('fs');

try {
  const data = fs.readFileSync('example.txt', 'utf8');
  console.log(data);
} catch (err) {
  console.error(err);
}

Escritura de Archivos

La escritura de archivos también puede hacerse de manera asincrónica o sincrónica.

Escritura Asincrónica

const fs = require('fs');

const content = 'Este es algún contenido para escribir en el archivo.';

fs.writeFile('example.txt', content, 'utf8', (err) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log('El archivo ha sido escrito');
});

Escritura Sincrónica

const fs = require('fs');

const content = 'Este es algún contenido para escribir en el archivo.';

try {
  fs.writeFileSync('example.txt', content, 'utf8');
  console.log('El archivo ha sido escrito');
} catch (err) {
  console.error(err);
}

Añadir a Archivos

Añadir datos a un archivo existente puede hacerse usando fs.appendFile y fs.appendFileSync.

Añadir Asincrónicamente

const fs = require('fs');

const content = '\nEste es contenido adicional.';

fs.appendFile('example.txt', content, 'utf8', (err) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log('El contenido ha sido añadido');
});

Añadir Sincrónicamente

const fs = require('fs');

const content = '\nEste es contenido adicional.';

try {
  fs.appendFileSync('example.txt', content, 'utf8');
  console.log('El contenido ha sido añadido');
} catch (err) {
  console.error(err);
}

Borrar Archivos

Borrar archivos puede hacerse usando fs.unlink y fs.unlinkSync.

Borrar Asincrónicamente

const fs = require('fs');

fs.unlink('example.txt', (err) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log('El archivo ha sido borrado');
});

Borrar Sincrónicamente

const fs = require('fs');

try {
  fs.unlinkSync('example.txt');
  console.log('El archivo ha sido borrado');
} catch (err) {
  console.error(err);
}

Operaciones Avanzadas del Sistema de Archivos

Trabajando con Directorios

Node.js te permite crear, leer y borrar directorios.

Creación de Directorios

const fs = require('fs');

// Asincrónico
fs.mkdir('newDir', (err) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log('Directorio creado');
});

// Sincrónico
try {
  fs.mkdirSync('newDir');
  console.log('Directorio creado');
} catch (err) {
  console.error(err);
}

Lectura de Directorios

const fs = require('fs');

// Asincrónico
fs.readdir('.', (err, files) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log(files);
});

// Sincrónico
try {
  const files = fs.readdirSync('.');
  console.log(files);
} catch (err) {
  console.error(err);
}

Borrar Directorios

const fs = require('fs');

// Asincrónico
fs.rmdir('newDir', (err) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log('Directorio borrado');
});

// Sincrónico
try {
  fs.rmdirSync('newDir');
  console.log('Directorio borrado');
} catch (err) {
  console.error(err);
}

Observando Archivos y Directorios

Node.js te permite observar cambios en archivos y directorios usando fs.watch.

const fs = require('fs');

fs.watch('example.txt', (eventType, filename) => {
  if (filename) {
    console.log(`${filename} archivo cambiado`);
  }
});

Conclusión

Entender las operaciones del sistema de archivos en Node.js es esencial para desarrollar aplicaciones robustas. Ya sea que estés leyendo, escribiendo, añadiendo o borrando archivos, Node.js proporciona métodos tanto sincrónicos como asincrónicos para satisfacer tus necesidades. Además, trabajar con directorios y observar cambios puede ayudarte a construir aplicaciones más dinámicas y receptivas.
