¿Qué es Node.js?

Introducción

Node.js es un entorno de ejecución potente, de código abierto y multiplataforma que permite a los desarrolladores ejecutar código JavaScript fuera de un navegador web. Está construido sobre el motor JavaScript V8 de Chrome y está diseñado para construir aplicaciones de red escalables.

Conceptos Clave

  • Entorno de Ejecución de JavaScript: Node.js permite que JavaScript se ejecute en el lado del servidor, no solo en el navegador.
  • I/O No Bloqueante: Node.js utiliza un modelo de I/O no bloqueante y dirigido por eventos, lo que lo hace ligero y eficiente.
  • Bucle de Eventos de un Solo Hilo: Node.js opera en un bucle de eventos de un solo hilo, que puede manejar múltiples conexiones simultáneamente.
  • Gestor de Paquetes (npm): Node.js viene con npm, un gestor de paquetes que permite a los desarrolladores compartir y reutilizar código.

Características de Node.js

  • Asíncrono y Dirigido por Eventos: Todas las APIs de la biblioteca de Node.js son asíncronas, lo que significa que no bloquean.
  • Ejecución Rápida: Construido sobre el motor JavaScript V8, Node.js ejecuta el código rápidamente.
  • Un Solo Lenguaje de Programación: Los desarrolladores pueden escribir código tanto del lado del cliente como del servidor en JavaScript.
  • Comunidad y Ecosistema: Una comunidad grande y activa contribuye a un rico ecosistema de módulos y herramientas.

Ejemplo de Código: Servidor HTTP Básico

Vamos a crear un servidor HTTP simple usando Node.js para entender su funcionalidad básica.

// Cargar el módulo http para crear un servidor HTTP.
const http = require('http');

// Configurar el servidor HTTP para responder con "Hello, World!" a todas las solicitudes.
const server = http.createServer((req, res) => {
  res.statusCode = 200; // Código de estado HTTP: 200 (OK)
  res.setHeader('Content-Type', 'text/plain'); // Establecer el encabezado de la respuesta
  res.end('Hello, World!\n'); // Enviar el cuerpo de la respuesta
});

// Escuchar en el puerto 3000, la IP por defecto es 127.0.0.1
server.listen(3000, '127.0.0.1', () => {
  console.log('Servidor ejecutándose en http://127.0.0.1:3000/');
});

Explicación

  1. Cargar el Módulo HTTP: Usamos require('http') para cargar el módulo HTTP, que es parte de los módulos principales de Node.js.
  2. Crear el Servidor: El método http.createServer() crea un nuevo servidor HTTP y configura una función de devolución de llamada para manejar las solicitudes entrantes.
  3. Manejo de Solicitudes: La función de devolución de llamada toma dos argumentos, req (solicitud) y res (respuesta). Establecemos el código de estado a 200 (OK), establecemos el tipo de contenido a text/plain, y enviamos el cuerpo de la respuesta "Hello, World!".
  4. Escuchar en el Puerto 3000: El servidor escucha en el puerto 3000, y cuando se inicia, registra un mensaje en la consola.

Resumen

Node.js es un entorno de ejecución versátil y eficiente para ejecutar JavaScript en el lado del servidor. Su modelo de I/O no bloqueante, bucle de eventos de un solo hilo y ejecución rápida lo hacen ideal para construir aplicaciones de red escalables. Con npm, los desarrolladores tienen acceso a un vasto repositorio de módulos reutilizables, mejorando la productividad y la colaboración. El ejemplo básico del servidor HTTP demuestra lo fácil que es comenzar con Node.js y destaca sus características principales.
