Introducción a los Cmdlets de Active Directory
Gestionando Usuarios y Grupos ►

Descripción general

Active Directory (AD) es un servicio de directorio desarrollado por Microsoft para redes de dominio de Windows. PowerShell proporciona un conjunto de cmdlets específicamente diseñados para interactuar con Active Directory. Esta sección te introducirá a estos cmdlets, explicará su uso y proporcionará ejemplos prácticos para ayudarte a entender cómo gestionar objetos de AD usando PowerShell.

Conceptos clave

  • Active Directory: Un servicio de directorio para redes de dominio de Windows.
  • Cmdlets: Comandos especializados en PowerShell que realizan operaciones específicas.
  • Módulo AD: Un módulo de PowerShell que contiene cmdlets para gestionar Active Directory.

Requisitos previos

Antes de sumergirte en los cmdlets de Active Directory, asegúrate de tener:

  • Conocimientos básicos de la sintaxis y comandos de PowerShell.
  • Acceso a un servidor Windows con Active Directory instalado.
  • Privilegios administrativos para realizar operaciones en AD.

Instalación del módulo de Active Directory

Para usar los cmdlets de AD, necesitas instalar el módulo de Active Directory. Este módulo está disponible en las Herramientas de Administración de Servidor Remoto (RSAT) para Windows.

Pasos para instalar RSAT

  1. Abre Configuración.
  2. Ve a Aplicaciones > Características opcionales.
  3. Haz clic en Agregar una característica.
  4. Busca RSAT: Servicios de dominio de Active Directory y Herramientas de directorio ligero.
  5. Instala la característica.

Cmdlets comunes de Active Directory

Aquí tienes algunos de los cmdlets de Active Directory más comúnmente usados:

| Cmdlet | Descripción | |-----------------------|--------------------------------------------------| | Get-ADUser | Recupera objetos de usuario de AD. | | New-ADUser | Crea un nuevo usuario en AD. | | Set-ADUser | Modifica propiedades de un usuario existente. | | Remove-ADUser | Elimina un usuario de AD. | | Get-ADGroup | Recupera objetos de grupo de AD. | | New-ADGroup | Crea un nuevo grupo en AD. | | Add-ADGroupMember | Añade un usuario a un grupo. | | Remove-ADGroupMember| Elimina un usuario de un grupo. |

Uso de los cmdlets de Active Directory

Recuperar información de usuario

Para recuperar información sobre un usuario, puedes usar el cmdlet Get-ADUser.

Get-ADUser -Identity "jdoe"

Explicación:

  • Get-ADUser: El cmdlet para recuperar información de usuario.
  • -Identity "jdoe": Especifica el usuario a recuperar por su nombre de usuario.

Crear un nuevo usuario

Para crear un nuevo usuario en Active Directory, usa el cmdlet New-ADUser.

New-ADUser -Name "John Doe" -GivenName "John" -Surname "Doe" -SamAccountName "jdoe" -UserPrincipalName "jdoe@example.com" -Path "OU=Users,DC=example,DC=com" -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force) -Enabled $true

Explicación:

  • New-ADUser: El cmdlet para crear un nuevo usuario.
  • -Name "John Doe": El nombre completo del usuario.
  • -GivenName "John": El nombre de pila del usuario.
  • -Surname "Doe": El apellido del usuario.
  • -SamAccountName "jdoe": El nombre de cuenta SAM.
  • -UserPrincipalName "jdoe@example.com": El UPN del usuario.
  • -Path "OU=Users,DC=example,DC=com": La unidad organizativa donde se creará el usuario.
  • -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force): Establece la contraseña de la cuenta.
  • -Enabled $true: Habilita la cuenta de usuario.

Modificar propiedades de usuario

Para modificar propiedades de un usuario existente, usa el cmdlet Set-ADUser.

Set-ADUser -Identity "jdoe" -Title "Senior Developer" -Department "IT"

Explicación:

  • Set-ADUser: El cmdlet para modificar propiedades de usuario.
  • -Identity "jdoe": Especifica el usuario a modificar.
  • -Title "Senior Developer": Establece el título del usuario.
  • -Department "IT": Establece el departamento del usuario.

Eliminar un usuario

Para eliminar un usuario de Active Directory, usa el cmdlet Remove-ADUser.

Remove-ADUser -Identity "jdoe"

Explicación:

  • Remove-ADUser: El cmdlet para eliminar un usuario.
  • -Identity "jdoe": Especifica el usuario a eliminar.

Conclusión

En esta sección, aprendiste sobre los conceptos básicos de los cmdlets de Active Directory en PowerShell. Ahora sabes cómo instalar el módulo de AD, recuperar información de usuario, crear nuevos usuarios, modificar propiedades de usuario y eliminar usuarios. Estos cmdlets son herramientas poderosas para gestionar objetos de Active Directory de manera eficiente. Practica usando estos cmdlets para familiarizarte más con la gestión de AD a través de PowerShell.
