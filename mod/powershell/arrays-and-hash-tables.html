Arrays y Tablas Hash en PowerShell
Entrada y Salida Básica ►

Introducción

En esta sección, exploraremos dos estructuras de datos fundamentales en PowerShell: Arrays y Tablas Hash. Comprender estas estructuras es crucial para la manipulación y almacenamiento eficiente de datos en tus scripts.

Arrays

¿Qué es un Array?

Un array es una colección de elementos almacenados en ubicaciones de memoria contiguas. En PowerShell, los arrays pueden almacenar múltiples elementos de diferentes tipos.

Creando Arrays

Puedes crear un array encerrando los elementos en @().

# Creando un array de enteros
$numbers = @(1, 2, 3, 4, 5)

# Creando un array de cadenas
$fruits = @("Apple", "Banana", "Cherry")

# Creando un array mixto
$mixedArray = @(1, "Hello", 3.14, $true)

Accediendo a Elementos del Array

Los elementos del array se acceden usando su índice, comenzando desde 0.

# Accediendo al primer elemento
$firstFruit = $fruits[0]  # Salida: Apple

# Accediendo al tercer elemento
$thirdNumber = $numbers[2]  # Salida: 3

Modificando Arrays

Puedes modificar los elementos del array asignando nuevos valores a índices específicos.

# Cambiando la segunda fruta
$fruits[1] = "Blueberry"  # $fruits ahora contiene "Apple", "Blueberry", "Cherry"

Métodos del Array

Los arrays en PowerShell vienen con varios métodos útiles.

# Añadiendo un elemento al array
$numbers += 6  # $numbers ahora contiene 1, 2, 3, 4, 5, 6

# Eliminando un elemento del array
$numbers = $numbers - 3  # $numbers ahora contiene 1, 2, 4, 5, 6

Iterando Sobre Arrays

Puedes usar bucles para iterar sobre los elementos del array.

# Usando un bucle foreach
foreach ($fruit in $fruits) {
    Write-Output $fruit
}

Tablas Hash

¿Qué es una Tabla Hash?

Una tabla hash es una colección de pares clave-valor, donde cada clave es única. Las tablas hash son útiles para almacenar datos que necesitan ser recuperados rápidamente usando una clave.

Creando Tablas Hash

Puedes crear una tabla hash usando @{}.

# Creando una tabla hash
$person = @{
    Name = "John"
    Age = 30
    Occupation = "Engineer"
}

Accediendo a Elementos de la Tabla Hash

Los elementos de la tabla hash se acceden usando sus claves.

# Accediendo al valor asociado con la clave 'Name'
$name = $person["Name"]  # Salida: John

# Accediendo al valor usando notación de punto
$age = $person.Age  # Salida: 30

Modificando Tablas Hash

Puedes añadir o modificar elementos en una tabla hash asignando valores a claves.

# Añadiendo un nuevo par clave-valor
$person["City"] = "New York"

# Modificando un valor existente
$person.Age = 31

Métodos de la Tabla Hash

Las tablas hash vienen con varios métodos útiles.

# Comprobando si una clave existe
$exists = $person.ContainsKey("Name")  # Salida: True

# Eliminando un par clave-valor
$person.Remove("Occupation")

Iterando Sobre Tablas Hash

Puedes usar bucles para iterar sobre los elementos de la tabla hash.

# Usando un bucle foreach para iterar sobre las claves
foreach ($key in $person.Keys) {
    Write-Output "$key: $($person[$key])"
}

Conclusión

Los arrays y las tablas hash son estructuras de datos poderosas en PowerShell que te permiten almacenar y manipular colecciones de datos de manera eficiente. Los arrays son ideales para colecciones ordenadas, mientras que las tablas hash son perfectas para pares clave-valor. Comprender cómo crear, acceder, modificar e iterar sobre estas estructuras mejorará significativamente tus capacidades de scripting en PowerShell.
