Operadores en PowerShell
Trabajando con Cadenas ►

Introducción

Los operadores en PowerShell son caracteres especiales o palabras clave que se utilizan para realizar operaciones en variables y valores. Son esenciales para manipular datos y controlar el flujo de los scripts. Esta sección cubrirá los diversos tipos de operadores disponibles en PowerShell, desde aritmética básica hasta operadores avanzados de comparación y lógicos.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • % : Módulo (resto)

Ejemplos

# Adición
$sum = 5 + 3
Write-Output $sum  # Salida: 8

# Sustracción
$difference = 10 - 4
Write-Output $difference  # Salida: 6

# Multiplicación
$product = 7 * 6
Write-Output $product  # Salida: 42

# División
$quotient = 20 / 4
Write-Output $quotient  # Salida: 5

# Módulo
$remainder = 10 % 3
Write-Output $remainder  # Salida: 1

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • = : Asignación simple
  • += : Sumar y asignar
  • -= : Restar y asignar
  • *= : Multiplicar y asignar
  • /= : Dividir y asignar
  • %= : Módulo y asignar

Ejemplos

# Asignación simple
$a = 5

# Sumar y asignar
$a += 3  # Equivalente a $a = $a + 3
Write-Output $a  # Salida: 8

# Restar y asignar
$a -= 2  # Equivalente a $a = $a - 2
Write-Output $a  # Salida: 6

# Multiplicar y asignar
$a *= 4  # Equivalente a $a = $a * 4
Write-Output $a  # Salida: 24

# Dividir y asignar
$a /= 3  # Equivalente a $a = $a / 3
Write-Output $a  # Salida: 8

# Módulo y asignar
$a %= 5  # Equivalente a $a = $a % 5
Write-Output $a  # Salida: 3

Operadores de Comparación

Los operadores de comparación se utilizan para comparar valores y devolver un resultado booleano ($true o $false).

  • -eq : Igual a
  • -ne : No igual a
  • -gt : Mayor que
  • -ge : Mayor o igual que
  • -lt : Menor que
  • -le : Menor o igual que

Ejemplos

# Igual a
Write-Output (5 -eq 5)  # Salida: True

# No igual a
Write-Output (5 -ne 3)  # Salida: True

# Mayor que
Write-Output (7 -gt 4)  # Salida: True

# Mayor o igual que
Write-Output (7 -ge 7)  # Salida: True

# Menor que
Write-Output (3 -lt 5)  # Salida: True

# Menor o igual que
Write-Output (3 -le 3)  # Salida: True

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

  • -and : AND lógico
  • -or : OR lógico
  • -not : NOT lógico

Ejemplos

# AND lógico
Write-Output ((5 -eq 5) -and (3 -lt 4))  # Salida: True

# OR lógico
Write-Output ((5 -eq 5) -or (3 -gt 4))  # Salida: True

# NOT lógico
Write-Output (-not (5 -eq 3))  # Salida: True

Operadores Especiales

PowerShell también incluye algunos operadores especiales para tareas específicas.

  • .. : Operador de rango
  • -contains : Operador de contiene
  • -match : Coincidencia de expresión regular
  • -replace : Operador de reemplazo

Ejemplos

# Operador de rango
$range = 1..5
Write-Output $range  # Salida: 1 2 3 4 5

# Operador de contiene
$array = @(1, 2, 3, 4, 5)
Write-Output ($array -contains 3)  # Salida: True

# Coincidencia de expresión regular
$string = "PowerShell"
Write-Output ($string -match "Power")  # Salida: True

# Operador de reemplazo
$string = "Hello World"
$newString = $string -replace "World", "PowerShell"
Write-Output $newString  # Salida: Hello PowerShell

Conclusión

Los operadores en PowerShell son herramientas fundamentales que te permiten realizar una amplia gama de operaciones, desde aritmética simple hasta evaluaciones lógicas complejas. Entender cómo usar estos operadores de manera efectiva mejorará enormemente tu capacidad para escribir scripts de PowerShell poderosos y eficientes. Practica usando estos operadores en diferentes escenarios para volverte más competente en la escritura de scripts en PowerShell.
