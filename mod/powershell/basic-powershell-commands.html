Introducción

PowerShell es un marco de automatización de tareas y gestión de configuración de Microsoft, que consta de una consola de línea de comandos y el lenguaje de scripting asociado. Esta sección te presentará algunos de los comandos básicos de PowerShell que son esenciales para principiantes.

Conceptos clave

  • Cmdlets: Los comandos de PowerShell se conocen como cmdlets. Siguen una convención de nombres Verbo-Sustantivo.
  • Pipelines: PowerShell utiliza pipelines para pasar la salida de un cmdlet como entrada a otro.
  • Objetos: A diferencia de los shells tradicionales que trabajan con texto, PowerShell trabaja con objetos.

Cmdlets básicos

Get-Command

El cmdlet Get-Command recupera todos los comandos disponibles en tu sesión de PowerShell.

Get-Command
  • Explicación: Este cmdlet lista todos los cmdlets, funciones, alias y scripts disponibles.

Get-Help

El cmdlet Get-Help proporciona información detallada sobre los cmdlets y conceptos de PowerShell.

Get-Help Get-Command
  • Explicación: Este comando mostrará información de ayuda sobre el cmdlet Get-Command.

Get-Process

El cmdlet Get-Process recupera información sobre los procesos que se ejecutan en tu sistema.

Get-Process
  • Explicación: Este cmdlet lista todos los procesos que se están ejecutando actualmente en tu máquina.

Get-Service

El cmdlet Get-Service recupera el estado de los servicios en una máquina local o remota.

Get-Service
  • Explicación: Este cmdlet lista todos los servicios y sus estados.

Set-Location

El cmdlet Set-Location cambia el directorio actual.

Set-Location -Path "C:\Users"
  • Explicación: Este comando cambia el directorio actual a C:\Users.

Get-ChildItem

El cmdlet Get-ChildItem recupera los archivos y directorios en una ubicación especificada.

Get-ChildItem -Path "C:\Users"
  • Explicación: Este cmdlet lista todos los archivos y directorios dentro de C:\Users.

Copy-Item

El cmdlet Copy-Item copia un elemento de una ubicación a otra.

Copy-Item -Path "C:\Users\example.txt" -Destination "C:\Backup\example.txt"
  • Explicación: Este comando copia example.txt de C:\Users a C:\Backup.

Remove-Item

El cmdlet Remove-Item elimina un elemento.

Remove-Item -Path "C:\Users\example.txt"
  • Explicación: Este comando elimina example.txt de C:\Users.

New-Item

El cmdlet New-Item crea un nuevo elemento, como un archivo o directorio.

New-Item -Path "C:\Users\newfile.txt" -ItemType "File"
  • Explicación: Este comando crea un nuevo archivo llamado newfile.txt en C:\Users.

Rename-Item

El cmdlet Rename-Item renombra un elemento.

Rename-Item -Path "C:\Users\newfile.txt" -NewName "renamedfile.txt"
  • Explicación: Este comando renombra newfile.txt a renamedfile.txt.

Resumen

En esta sección, cubrimos algunos de los comandos básicos de PowerShell que son esenciales para principiantes. Estos comandos incluyen Get-Command, Get-Help, Get-Process, Get-Service, Set-Location, Get-ChildItem, Copy-Item, Remove-Item, New-Item y Rename-Item. Entender estos cmdlets básicos proporcionará una base sólida para tareas más avanzadas de scripting y automatización en PowerShell.
