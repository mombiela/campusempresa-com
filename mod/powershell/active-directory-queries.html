Introducción

Active Directory (AD) es un servicio de directorio desarrollado por Microsoft para redes de dominio de Windows. PowerShell proporciona una forma poderosa de interactuar con AD, permitiendo a los administradores consultar y gestionar objetos de AD de manera eficiente. Esta sección te guiará a través de la consulta de Active Directory usando PowerShell, desde técnicas básicas hasta avanzadas.

Requisitos previos

Antes de sumergirte en las consultas de Active Directory, asegúrate de tener:

  • Conocimientos básicos de PowerShell.
  • Módulo de Active Directory instalado (RSAT-AD-PowerShell).

Consultas básicas

Conectarse a Active Directory

Para realizar cualquier consulta, necesitas importar el módulo de Active Directory:

Import-Module ActiveDirectory

Recuperar un solo usuario

Para recuperar un solo usuario por su nombre de usuario (sAMAccountName):

Get-ADUser -Identity "username"
  • -Identity: Especifica el usuario a recuperar.

Recuperar múltiples usuarios

Para recuperar todos los usuarios en el dominio:

Get-ADUser -Filter *
  • -Filter *: Recupera todos los usuarios.

Filtrar usuarios

Para filtrar usuarios basados en criterios específicos, como usuarios en un departamento específico:

Get-ADUser -Filter {Department -eq "Sales"}
  • -Filter {Department -eq "Sales"}: Recupera usuarios en el departamento de Ventas.

Seleccionar propiedades específicas

Por defecto, Get-ADUser devuelve un conjunto limitado de propiedades. Para seleccionar propiedades específicas:

Get-ADUser -Identity "username" -Properties DisplayName, EmailAddress
  • -Properties: Especifica propiedades adicionales a recuperar.

Consultas intermedias

Recuperar grupos

Para recuperar todos los grupos en el dominio:

Get-ADGroup -Filter *

Filtrar grupos

Para filtrar grupos basados en criterios específicos, como grupos que comienzan con "Admin":

Get-ADGroup -Filter {Name -like "Admin*"}

Recuperar miembros de un grupo

Para recuperar miembros de un grupo específico:

Get-ADGroupMember -Identity "GroupName"

Combinando filtros y propiedades

Para recuperar usuarios con propiedades específicas y criterios de filtro:

Get-ADUser -Filter {Title -eq "Manager"} -Properties DisplayName, EmailAddress

Consultas avanzadas

Usando filtros LDAP

Los filtros LDAP proporcionan un control más granular sobre las consultas. Para recuperar usuarios con un filtro LDAP:

Get-ADUser -LDAPFilter "(objectClass=user)"
  • -LDAPFilter: Especifica un filtro LDAP.

Consultar Unidades Organizativas (OUs)

Para recuperar todas las OUs en el dominio:

Get-ADOrganizationalUnit -Filter *

Consultas recursivas

Para recuperar todos los usuarios en una OU específica y sus OUs hijas:

Get-ADUser -Filter * -SearchBase "OU=Sales,DC=example,DC=com" -SearchScope Subtree
  • -SearchBase: Especifica la base de la búsqueda.
  • -SearchScope Subtree: Busca en la OU especificada y todas sus OUs hijas.

Atributos personalizados

Para recuperar usuarios con atributos personalizados:

Get-ADUser -Filter * -Properties CustomAttribute1

Exportar resultados de consultas

Para exportar los resultados de la consulta a un archivo CSV:

Get-ADUser -Filter * | Select-Object Name, EmailAddress | Export-Csv -Path "C:\Users.csv" -NoTypeInformation
  • Select-Object: Selecciona propiedades específicas.
  • Export-Csv: Exporta los resultados a un archivo CSV.

Conclusión

PowerShell proporciona un conjunto robusto de cmdlets para consultar Active Directory, permitiendo a los administradores gestionar y recuperar objetos de AD de manera eficiente. Al dominar estas consultas, puedes automatizar y agilizar muchas tareas administrativas, haciendo tu flujo de trabajo más eficiente y efectivo. Practica estas consultas y explora las vastas capacidades de PowerShell para volverte competente en la gestión de Active Directory.
