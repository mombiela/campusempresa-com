Construyendo una Aplicación de Tareas con Ionic
Creando una Aplicación del Clima ►

Introducción

En este tema, te guiaremos a través de la construcción de una aplicación de tareas utilizando Ionic. Este proyecto te ayudará a comprender los conceptos básicos de Ionic, incluidos los componentes de la interfaz de usuario, la navegación, la gestión del estado y el almacenamiento de datos. Al final de este tema, tendrás una aplicación de tareas funcional y una comprensión sólida de cómo construir aplicaciones móviles con Ionic.

Configuración del Entorno

Antes de comenzar a construir la aplicación, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Node.js y npm: Asegúrate de tener Node.js y npm instalados en tu sistema.
  • Instalar Ionic CLI: Ejecuta el siguiente comando para instalar Ionic CLI globalmente: 
    npm install -g @ionic/cli
  • Crear un Nuevo Proyecto Ionic: Usa Ionic CLI para crear un nuevo proyecto: 
    ionic start todo-app blank
  • Navegar al Directorio del Proyecto: Muévete al directorio del proyecto: 
    cd todo-app

Construyendo la Interfaz de Usuario

Comenzaremos construyendo la interfaz de usuario para nuestra aplicación de tareas.

Página de Inicio

La página de inicio mostrará la lista de tareas y un botón para agregar nuevos elementos.

  • Crear Página de Inicio: Genera una nueva página usando Ionic CLI:

    ionic generate page home

  • Actualizar app-routing.module.ts: Establece la página de inicio como la ruta predeterminada.

    const routes: Routes = [
  { path: '', redirectTo: 'home', pathMatch: 'full' },
  { path: 'home', loadChildren: () => import('./home/home.module').then(m => m.HomePageModule) }
];

  • Diseñar la Página de Inicio: Modifica home.page.html para incluir una lista y un botón.

    <ion-header>
  <ion-toolbar>
    <ion-title>Lista de Tareas</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-list>
    <ion-item *ngFor="let item of todoItems">
      {{ item }}
    </ion-item>
  </ion-list>
  <ion-fab vertical="bottom" horizontal="end" slot="fixed">
    <ion-fab-button (click)="addItem()">
      <ion-icon name="add"></ion-icon>
    </ion-fab-button>
  </ion-fab>
</ion-content>

Agregar Elementos

Necesitamos implementar la funcionalidad para agregar nuevos elementos a la lista.

  • Actualizar home.page.ts: Agrega un método para manejar la adición de nuevos elementos. 
    import { Component } from '@angular/core';
import { AlertController } from '@ionic/angular';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  todoItems: string[] = [];

  constructor(private alertController: AlertController) {}

  async addItem() {
    const alert = await this.alertController.create({
      header: 'Nuevo Elemento',
      inputs: [
        {
          name: 'item',
          type: 'text',
          placeholder: 'Ingresa tu tarea'
        }
      ],
      buttons: [
        {
          text: 'Cancelar',
          role: 'cancel',
          cssClass: 'secondary'
        },
        {
          text: 'Agregar',
          handler: (data) => {
            if (data.item) {
              this.todoItems.push(data.item);
            }
          }
        }
      ]
    });

    await alert.present();
  }
}

Gestión del Estado

Para gestionar el estado de nuestra aplicación, utilizaremos un servicio.

  • Crear un Servicio de Tareas: Genera un nuevo servicio usando Ionic CLI:

    ionic generate service todo

  • Implementar el Servicio: Actualiza todo.service.ts para gestionar la lista de tareas.

    import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class TodoService {
  private todoItems: string[] = [];

  constructor() {}

  getItems(): string[] {
    return this.todoItems;
  }

  addItem(item: string) {
    this.todoItems.push(item);
  }
}

  • Actualizar la Página de Inicio para Usar el Servicio: Modifica home.page.ts para usar el TodoService.

    import { Component } from '@angular/core';
import { AlertController } from '@ionic/angular';
import { TodoService } from '../services/todo.service';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  todoItems: string[];

  constructor(private alertController: AlertController, private todoService: TodoService) {
    this.todoItems = this.todoService.getItems();
  }

  async addItem() {
    const alert = await this.alertController.create({
      header: 'Nuevo Elemento',
      inputs: [
        {
          name: 'item',
          type: 'text',
          placeholder: 'Ingresa tu tarea'
        }
      ],
      buttons: [
        {
          text: 'Cancelar',
          role: 'cancel',
          cssClass: 'secondary'
        },
        {
          text: 'Agregar',
          handler: (data) => {
            if (data.item) {
              this.todoService.addItem(data.item);
              this.todoItems = this.todoService.getItems();
            }
          }
        }
      ]
    });

    await alert.present();
  }
}

Almacenamiento de Datos

Para persistir los elementos de la lista de tareas, utilizaremos Ionic Storage.

  • Instalar Ionic Storage: Agrega los paquetes necesarios.

    npm install @ionic/storage-angular
npm install @ionic/storage

  • Configurar Storage: Actualiza app.module.ts para configurar Ionic Storage.

    import { IonicStorageModule } from '@ionic/storage-angular';

@NgModule({
  ...
  imports: [
    ...
    IonicStorageModule.forRoot()
  ],
  ...
})
export class AppModule {}

  • Actualizar el Servicio de Tareas: Modifica todo.service.ts para usar Ionic Storage.

    import { Injectable } from '@angular/core';
import { Storage } from '@ionic/storage-angular';

@Injectable({
  providedIn: 'root'
})
export class TodoService {
  private todoItems: string[] = [];

  constructor(private storage: Storage) {
    this.init();
  }

  async init() {
    await this.storage.create();
    this.todoItems = (await this.storage.get('todoItems')) || [];
  }

  getItems(): string[] {
    return this.todoItems;
  }

  async addItem(item: string) {
    this.todoItems.push(item);
    await this.storage.set('todoItems', this.todoItems);
  }
}

Conclusión

En este tema, hemos construido una simple aplicación de tareas utilizando Ionic. Cubrimos la configuración del entorno, la construcción de la interfaz de usuario, la gestión del estado con un servicio y la persistencia de datos utilizando Ionic Storage. Este proyecto te ha proporcionado una comprensión fundamental de Ionic, que puedes ampliar para crear aplicaciones más complejas.
