Introducción

En esta sección, te guiaremos a través del proceso de creación de tu primera aplicación Ionic. Ionic es un marco popular para construir aplicaciones móviles multiplataforma utilizando tecnologías web como HTML, CSS y JavaScript. Al final de esta sección, tendrás una aplicación Ionic básica ejecutándose en tu máquina local.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente instalado en tu computadora:

  • Node.js y npm (Node Package Manager)
  • Ionic CLI (Command Line Interface)
  • Un editor de código (por ejemplo, Visual Studio Code)

Configurando Tu Entorno

  1. Instalar Node.js y npm: Descarga e instala Node.js desde nodejs.org. Esto también instalará npm.
  2. Instalar Ionic CLI: Abre tu terminal o símbolo del sistema y ejecuta el siguiente comando: 
    npm install -g @ionic/cli
    Este comando instala el Ionic CLI globalmente en tu sistema.

Creando un Nuevo Proyecto Ionic

  1. Abre tu terminal y navega al directorio donde deseas crear tu proyecto.

  2. Ejecuta el siguiente comando para crear un nuevo proyecto Ionic:

    ionic start myFirstApp blank
    • myFirstApp es el nombre de tu proyecto.
    • blank es la plantilla de inicio. Puedes elegir otras plantillas como tabs o sidemenu.

  3. Navega a tu directorio de proyecto:

    cd myFirstApp

Ejecutando Tu Aplicación Ionic

  1. Sirve la aplicación localmente ejecutando: 
    ionic serve
    Este comando compilará tu aplicación y la abrirá en tu navegador web predeterminado. Deberías ver una aplicación Ionic en blanco ejecutándose.

Entendiendo la Estructura del Proyecto

Aquí tienes una breve descripción de los archivos y directorios clave en tu nuevo proyecto Ionic:

  • src/: Contiene el código fuente de tu aplicación.
    • app/: Contiene el módulo principal de la aplicación y el componente raíz.
    • assets/: Contiene activos estáticos como imágenes y fuentes.
    • environments/: Contiene archivos de configuración específicos del entorno.
    • theme/: Contiene estilos y variables globales.
    • index.html: El archivo HTML principal.
    • main.ts: El punto de entrada principal para la aplicación.
  • www/: Contiene la salida compilada de tu aplicación.
  • ionic.config.json: Archivo de configuración para el Ionic CLI.

Añadiendo un Componente Simple

Vamos a añadir un componente simple a tu aplicación para mostrar un mensaje de bienvenida.

  1. Genera un nuevo componente:

    ionic generate component Welcome

  2. Modifica el componente:

    • Abre src/app/welcome/welcome.component.html y añade el siguiente código: 
      <div>
  <h1>¡Bienvenido a Mi Primera Aplicación Ionic!</h1>
</div>
    • Abre src/app/welcome/welcome.component.ts y asegúrate de que se vea así: 
      import { Component } from '@angular/core';

@Component({
  selector: 'app-welcome',
  templateUrl: './welcome.component.html',
  styleUrls: ['./welcome.component.scss']
})
export class WelcomeComponent {}

  3. Usa el componente en tu aplicación:

    • Abre src/app/app.component.html y añade la siguiente línea: 
      <app-welcome></app-welcome>

  4. Sirve la aplicación nuevamente:

    ionic serve

    Ahora deberías ver el mensaje de bienvenida mostrado en tu aplicación.

Conclusión

¡Felicidades! Has creado con éxito tu primera aplicación Ionic y añadido un componente simple. En esta sección, aprendiste cómo configurar tu entorno de desarrollo, crear un nuevo proyecto Ionic, entender la estructura del proyecto y añadir un componente básico. Este conocimiento fundamental te ayudará a medida que avances en temas más avanzados en el desarrollo con Ionic.
