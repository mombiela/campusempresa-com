Introducción a la Navegación en Ionic
Usando el Router de Ionic ►

Resumen

En esta sección, exploraremos los fundamentos de la navegación en aplicaciones Ionic. La navegación es un aspecto crucial de cualquier aplicación móvil, ya que define cómo los usuarios se mueven entre diferentes páginas o vistas dentro de la aplicación. Cubriremos los conceptos básicos, tipos de navegación y proporcionaremos ejemplos prácticos para ayudarte a entender cómo implementar la navegación en tus proyectos Ionic.

Conceptos Clave

  • Controlador de Navegación: Gestiona la pila de navegación y el historial.
  • Páginas: Vistas o pantallas individuales en la aplicación.
  • Enrutamiento: Define los caminos y componentes asociados con cada ruta.
  • NavController: Un servicio que proporciona métodos para navegar entre páginas.

Tipos de Navegación

Ionic soporta varios tipos de navegación, incluyendo:

  1. Navegación Push: Añadir una nueva página a la pila de navegación.
  2. Navegación Pop: Eliminar la página actual de la pila de navegación.
  3. Navegación Root: Establecer una nueva página raíz, limpiando la pila de navegación.
  4. Navegación por Pestañas: Usar pestañas para cambiar entre diferentes vistas.

Configuración de la Navegación

Instalación de Ionic CLI

Antes de comenzar, asegúrate de tener instalado el Ionic CLI. Si no, puedes instalarlo usando npm:

npm install -g @ionic/cli

Creación de un Nuevo Proyecto Ionic

Crea un nuevo proyecto Ionic usando el siguiente comando:

ionic start myApp blank

Navega al directorio del proyecto:

cd myApp

Añadiendo Páginas

Para añadir una nueva página, usa el Ionic CLI:

ionic generate page Home
ionic generate page About

Configuración de Rutas

Abre src/app/app-routing.module.ts y configura las rutas:

import { NgModule } from '@angular/core';
import { PreloadAllModules, RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: '',
    redirectTo: 'home',
    pathMatch: 'full'
  },
  {
    path: 'home',
    loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
  },
  {
    path: 'about',
    loadChildren: () => import('./about/about.module').then(m => m.AboutPageModule)
  }
];

@NgModule({
  imports: [
    RouterModule.forRoot(routes, { preloadingStrategy: PreloadAllModules })
  ],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Uso de NavController para la Navegación

Para navegar entre páginas, puedes usar el servicio NavController. Aquí tienes un ejemplo de navegación desde la página Home a la página About:

Página Home (home.page.ts)

import { Component } from '@angular/core';
import { NavController } from '@ionic/angular';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  constructor(private navCtrl: NavController) {}

  navigateToAbout() {
    this.navCtrl.navigateForward('/about');
  }
}

Plantilla de la Página Home (home.page.html)

<ion-header>
  <ion-toolbar>
    <ion-title>
      Home
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-button (click)="navigateToAbout()">Ir a About</ion-button>
</ion-content>

También demostramos cómo usar el servicio NavController para navegar entre páginas. Entender estos conceptos es esencial para construir aplicaciones móviles intuitivas y amigables con el usuario usando Ionic.
