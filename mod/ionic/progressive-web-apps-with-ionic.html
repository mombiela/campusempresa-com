Introducción a las Aplicaciones Web Progresivas (PWAs)

  • Definición de Aplicaciones Web Progresivas
  • Beneficios de las PWAs
    • Capacidades sin conexión
    • Rendimiento mejorado
    • Experiencia similar a una aplicación
    • Compatibilidad multiplataforma

Configurando tu Entorno Ionic

  • Instalación de Node.js y npm
  • Instalación de Ionic CLI 
    npm install -g @ionic/cli
  • Creación de un nuevo proyecto Ionic 
    ionic start myPWA blank --type=angular

Construyendo tu Primera PWA con Ionic

  • Descripción general de la estructura del proyecto
  • Ejecutando la aplicación Ionic 
    cd myPWA
ionic serve

Añadiendo Funcionalidades PWA

  • Habilitando Service Workers
    • Explicación de los Service Workers
    • Añadiendo Service Worker al proyecto 
      ng add @angular/pwa --project app
  • Creando un archivo de manifiesto
    • Propósito del archivo de manifiesto
    • Ejemplo de un archivo de manifiesto 
      {
  "name": "My PWA",
  "short_name": "PWA",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#4A90E2",
  "icons": [
    {
      "src": "assets/icons/icon-192x192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "assets/icons/icon-512x512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}

Mejorando la Experiencia del Usuario

  • Añadiendo una pantalla de bienvenida
  • Personalizando el tema de la aplicación
    • Usando las capacidades de tematización de Ionic
    • Ejemplo de personalización del tema en variables.scss 
      $colors: (
  primary: #4A90E2,
  secondary: #32db64,
  tertiary: #f4a142,
  success: #10dc60,
  warning: #ffce00,
  danger: #f04141,
  dark: #222428,
  medium: #989aa2,
  light: #f4f5f8
);

Capacidades sin Conexión

  • Estrategias de caché
    • Explicación de las estrategias de caché (por ejemplo, Caché Primero, Red Primero)
    • Implementando una estrategia de Caché Primero 
      self.addEventListener('fetch', function(event) {
  event.respondWith(
    caches.match(event.request).then(function(response) {
      return response || fetch(event.request);
    })
  );
});

Funcionalidades Avanzadas de PWA

  • Sincronización en segundo plano
    • Explicación y casos de uso
    • Ejemplo de implementación 
      self.addEventListener('sync', function(event) {
  if (event.tag == 'myFirstSync') {
    event.waitUntil(doSomeStuff());
  }
});
  • Notificaciones push
    • Configuración de notificaciones push
    • Ejemplo de envío de una notificación push 
      self.registration.showNotification('Hello World!', {
  body: 'This is a push notification.',
  icon: 'assets/icons/icon-192x192.png'
});
