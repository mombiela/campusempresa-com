Introducción a las Expresiones Regulares

Las expresiones regulares (regex) son herramientas poderosas para la coincidencia de patrones y la manipulación de texto. En Perl, regex está integrado directamente en el lenguaje, lo que lo convierte en una herramienta versátil para el procesamiento de cadenas.

Conceptos Clave

  • Coincidencia de Patrones: Encontrar patrones específicos dentro de cadenas.
  • Sustitución: Reemplazar partes de una cadena que coinciden con un patrón.
  • Modificadores: Cambiar el comportamiento de las operaciones de regex.

Sintaxis Básica

Perl usa el operador =~ para aplicar una regex a una cadena.

Ejemplo

my $string = "¡Hola, mundo!";
if ($string =~ /mundo/) {
    print "¡Coincidencia encontrada!\n";
}
  • /mundo/ es el patrón regex.
  • =~ aplica el patrón a $string.

Metacaracteres Comunes

  • .: Coincide con cualquier carácter excepto nueva línea.
  • *: Coincide con 0 o más del elemento anterior.
  • +: Coincide con 1 o más del elemento anterior.
  • ?: Coincide con 0 o 1 del elemento anterior.
  • \d: Coincide con cualquier dígito.
  • \w: Coincide con cualquier carácter de palabra (alfanumérico más guion bajo).
  • \s: Coincide con cualquier carácter de espacio en blanco.

Clases de Caracteres y Cuantificadores

Las clases de caracteres te permiten coincidir con cualquiera de varios caracteres.

Ejemplo

my $string = "abc123";
if ($string =~ /[a-z]/) {
    print "Contiene una letra minúscula.\n";
}
  • [a-z] coincide con cualquier letra minúscula.

Cuantificadores

  • {n}: Coincide exactamente con n ocurrencias.
  • {n,}: Coincide con n o más ocurrencias.
  • {n,m}: Coincide entre n y m ocurrencias.

Ejemplo

my $string = "aaaab";
if ($string =~ /a{3,5}/) {
    print "Coincide entre 3 y 5 'a's.\n";
}

Agrupación y Captura

Los paréntesis () se usan para agrupar partes de una regex y capturar subcadenas coincidentes.

Ejemplo

my $string = "¡Hola, mundo!";
if ($string =~ /(Hola), (mundo)/) {
    print "Primera parte: $1\n";
    print "Segunda parte: $2\n";
}
  • $1 y $2 almacenan las subcadenas coincidentes.

Sustitución

El operador s/// se usa para la sustitución.

Ejemplo

my $string = "¡Hola, mundo!";
$string =~ s/mundo/Perl/;
print "$string\n";  # Salida: ¡Hola, Perl!
  • s/mundo/Perl/ reemplaza "mundo" con "Perl".

Temas Avanzados

Lookahead y Lookbehind

Las afirmaciones lookahead y lookbehind te permiten coincidir con un patrón solo si está (o no está) seguido o precedido por otro patrón.

Ejemplo

my $string = "abc123";
if ($string =~ /abc(?=\d)/) {
    print "abc está seguido por un dígito.\n";
}
  • (?=\d) es un lookahead positivo.

Grupos No Capturantes

Los grupos no capturantes (?:...) se usan para agrupar partes de una regex sin capturar las subcadenas coincidentes.

Ejemplo

my $string = "¡Hola, mundo!";
if ($string =~ /(?:Hola), (mundo)/) {
    print "Coincidencia: $1\n";  # Salida: Coincidencia: mundo
}

Modificadores

Los modificadores cambian el comportamiento de las operaciones de regex.

Modificadores Comunes

  • i: Coincidencia insensible a mayúsculas y minúsculas.
  • m: Trata la cadena como múltiples líneas.
  • s: Trata la cadena como una sola línea.
  • g: Coincidencia global.

Ejemplo

my $string = "¡Hola, mundo!";
if ($string =~ /HOLA/i) {
    print "¡Coincidencia insensible a mayúsculas y minúsculas encontrada!\n";
}
  • /HOLA/i coincide con "HOLA" sin importar mayúsculas y minúsculas.

Ejercicios Prácticos

Ejercicio 1: Coincidencia Básica

Escribe un script en Perl para verificar si una cadena contiene la palabra "Perl".

Ejercicio 2: Sustitución

Escribe un script en Perl para reemplazar todas las ocurrencias de "foo" con "bar" en una cadena dada.

Ejercicio 3: Grupos de Captura

Escribe un script en Perl para extraer el nombre de dominio de una dirección de correo electrónico.

Conclusión

Las expresiones regulares en Perl son una herramienta poderosa para el procesamiento de texto. Al comprender la sintaxis básica, las clases de caracteres, los cuantificadores, la agrupación y las características avanzadas como lookahead y lookbehind, puedes realizar tareas complejas de coincidencia de patrones y manipulación de texto de manera eficiente. Practica con los ejercicios proporcionados para solidificar tu comprensión y convertirte en un experto en el uso de regex en Perl.
