Manejo de Archivos en Perl
Subrutinas ►

El manejo de archivos es un aspecto crucial de la programación, permitiéndote leer y escribir en archivos. En Perl, el manejo de archivos es sencillo pero poderoso. Esta sección te guiará desde lo básico hasta técnicas más avanzadas de manejo de archivos.

Introducción al Manejo de Archivos

  • Operaciones de Archivos: Abrir, leer, escribir y cerrar archivos.
  • Modos de Archivo: Diferentes modos para abrir archivos (leer, escribir, añadir).

Abrir y Cerrar Archivos

Para trabajar con archivos, primero necesitas abrirlos. Perl usa la función open para este propósito.

# Abrir un archivo para lectura
open(my $fh, '<', 'filename.txt') or die "No se pudo abrir el archivo: $!";

# Abrir un archivo para escritura
open(my $fh, '>', 'filename.txt') or die "No se pudo abrir el archivo: $!";

# Abrir un archivo para añadir
open(my $fh, '>>', 'filename.txt') or die "No se pudo abrir el archivo: $!";

# Siempre cierra el manejador de archivo cuando termines
close($fh);
  • <: Modo de lectura.
  • >: Modo de escritura (crea un nuevo archivo o trunca un archivo existente).
  • >>: Modo de añadir (crea un nuevo archivo si no existe).

Leer de Archivos

Hay varias formas de leer de un archivo en Perl:

Leer Línea por Línea

open(my $fh, '<', 'filename.txt') or die "No se pudo abrir el archivo: $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

Leer Todo el Archivo de Una Vez

open(my $fh, '<', 'filename.txt') or die "No se pudo abrir el archivo: $!";
my @lines = <$fh>;
close($fh);

foreach my $line (@lines) {
    print $line;
}

Escribir en Archivos

Escribir en archivos es similar a leer pero usa diferentes modos.

Escribir Línea por Línea

open(my $fh, '>', 'filename.txt') or die "No se pudo abrir el archivo: $!";
print $fh "Esta es una línea de texto.\n";
print $fh "Esta es otra línea de texto.\n";
close($fh);

Añadir a un Archivo

open(my $fh, '>>', 'filename.txt') or die "No se pudo abrir el archivo: $!";
print $fh "Esta línea será añadida.\n";
close($fh);

Manejo de Errores

Siempre maneja errores cuando trabajes con archivos para evitar problemas inesperados.

open(my $fh, '<', 'nonexistent.txt') or die "No se pudo abrir el archivo: $!";

La función die terminará el programa e imprimirá el mensaje de error si el archivo no puede ser abierto.

Manejo Avanzado de Archivos

Pruebas de Archivos

Perl proporciona varios operadores de prueba de archivos para verificar varios atributos de los archivos.

  • -e: Verificar si un archivo existe.
  • -r: Verificar si un archivo es legible.
  • -w: Verificar si un archivo es escribible.
  • -x: Verificar si un archivo es ejecutable.
my $filename = 'filename.txt';

if (-e $filename) {
    print "El archivo existe.\n";
}

if (-r $filename) {
    print "El archivo es legible.\n";
}

if (-w $filename) {
    print "El archivo es escribible.\n";
}

if (-x $filename) {
    print "El archivo es ejecutable.\n";
}

Trabajar con Rutas de Archivos

El módulo File::Spec de Perl ayuda a manejar rutas de archivos de una manera independiente de la plataforma.

use File::Spec;

my $path = File::Spec->catfile('directory', 'filename.txt');
print "La ruta del archivo es: $path\n";

Leer y Escribir Archivos Binarios

Para archivos binarios, usa la función binmode para asegurar un manejo adecuado.

# Leer un archivo binario
open(my $fh, '<:raw', 'binaryfile.bin') or die "No se pudo abrir el archivo: $!";
binmode($fh);
my $data;
while (read($fh, my $buffer, 1024)) {
    $data .= $buffer;
}
close($fh);

# Escribir en un archivo binario
open(my $fh, '>:raw', 'binaryfile.bin') or die "No se pudo abrir el archivo: $!";
binmode($fh);
print $fh $data;
close($fh);

Conclusión

El manejo de archivos en Perl es una habilidad fundamental que te permite leer y escribir en archivos de manera eficiente. Al dominar lo básico y entender técnicas avanzadas, puedes manejar varias operaciones de archivos en tus scripts de Perl. Siempre recuerda manejar errores de manera adecuada y usar los modos de archivo y funciones apropiadas para tus necesidades específicas.
