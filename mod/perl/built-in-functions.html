Introducción

Perl, un lenguaje de programación altamente capaz y rico en características, viene con una vasta gama de funciones integradas. Estas funciones están diseñadas para realizar una amplia variedad de tareas, desde la manipulación básica de cadenas hasta el procesamiento de datos complejo. Entender estas funciones integradas es crucial para escribir código Perl eficiente y efectivo.

Funciones Integradas Básicas

Print y Say

  • print: Imprime una cadena o una lista de cadenas en la salida estándar.
  • say: Similar a print, pero añade automáticamente una nueva línea al final de la salida (disponible en Perl 5.10 y posteriores).
print "¡Hola, Mundo!";
say "¡Hola, Mundo!";  # Imprime "¡Hola, Mundo!" seguido de una nueva línea

Funciones Escalares

  • chomp: Elimina el carácter de nueva línea del final de una cadena.
  • length: Devuelve la longitud de una cadena.
my $string = "¡Hola, Mundo!\n";
chomp($string);  # $string ahora es "¡Hola, Mundo!"
my $length = length($string);  # $length es 12

Funciones Numéricas

  • int: Devuelve la porción entera de un número.
  • rand: Genera un número aleatorio entre 0 y el valor especificado.
my $number = 3.14;
my $integer = int($number);  # $integer es 3
my $random = rand(10);  # $random es un número entre 0 y 10

Funciones Integradas Intermedias

Funciones de Lista

  • push: Añade elementos al final de un array.
  • pop: Elimina el último elemento de un array.
  • shift: Elimina el primer elemento de un array.
  • unshift: Añade elementos al principio de un array.
my @array = (1, 2, 3);
push(@array, 4);  # @array ahora es (1, 2, 3, 4)
pop(@array);  # @array ahora es (1, 2, 3)
shift(@array);  # @array ahora es (2, 3)
unshift(@array, 1);  # @array ahora es (1, 2, 3)

Funciones de Cadena

  • substr: Extrae una subcadena de una cadena.
  • index: Encuentra la posición de una subcadena dentro de una cadena.
  • split: Divide una cadena en una lista de cadenas basándose en un delimitador.
my $string = "¡Hola, Mundo!";
my $substring = substr($string, 0, 5);  # $substring es "¡Hola"
my $position = index($string, "Mundo");  # $position es 7
my @words = split(/, /, $string);  # @words es ("¡Hola", "Mundo!")

Funciones de Hash

  • keys: Devuelve una lista de claves de un hash.
  • values: Devuelve una lista de valores de un hash.
  • each: Itera sobre pares clave-valor en un hash.
my %hash = (foo => 1, bar => 2);
my @keys = keys %hash;  # @keys es ("foo", "bar")
my @values = values %hash;  # @values es (1, 2)
while (my ($key, $value) = each %hash) {
    print "$key: $value\n";
}

Funciones Integradas Avanzadas

Funciones de Manejo de Archivos

  • open: Abre un archivo para lectura, escritura o adición.
  • close: Cierra un manejador de archivo abierto.
  • read: Lee datos de un manejador de archivo.
  • write: Escribe datos en un manejador de archivo.
open(my $fh, '<', 'file.txt') or die "No se puede abrir el archivo: $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

Funciones de Interacción con el Sistema

  • system: Ejecuta un comando externo y espera a que termine.
  • exec: Reemplaza el proceso actual con un comando externo.
system("ls -l");  # Ejecuta el comando 'ls -l'
exec("perl script.pl");  # Reemplaza el proceso actual con 'perl script.pl'

Funciones de Expresiones Regulares

  • m//: Coincide una cadena contra una expresión regular.
  • s///: Sustituye una parte de una cadena que coincide con una expresión regular.
  • tr///: Translitera caracteres en una cadena.
my $string = "¡Hola, Mundo!";
if ($string =~ m/Mundo/) {
    print "¡Coincidencia encontrada!\n";
}
$string =~ s/Mundo/Perl/;  # $string ahora es "¡Hola, Perl!"
$string =~ tr/A-Z/a-z/;  # $string ahora es "¡hola, perl!"

Conclusión

Las funciones integradas de Perl son herramientas poderosas que pueden simplificar muchas tareas de programación. Desde operaciones básicas de entrada/salida hasta manipulación compleja de datos, estas funciones proporcionan una base sólida para escribir código Perl eficiente. Al dominar estas funciones integradas, puedes mejorar significativamente tus habilidades de programación en Perl y abordar una amplia gama de problemas con facilidad.
