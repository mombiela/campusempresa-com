Estructuras de Control en Perl
Bucles ►

Las estructuras de control son fundamentales en cualquier lenguaje de programación, y Perl no es una excepción. Te permiten controlar el flujo de tu programa, tomar decisiones, recorrer datos y manejar excepciones. Este tema cubrirá las estructuras de control esenciales en Perl, desde niveles básicos hasta avanzados.

Declaraciones Condicionales

Las declaraciones condicionales te permiten ejecutar código basado en ciertas condiciones.

Declaración if

La declaración if se usa para ejecutar un bloque de código si una condición especificada es verdadera.

my $age = 18;

if ($age >= 18) {
    print "Eres un adulto.\n";
}

Declaración if-else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

my $age = 16;

if ($age >= 18) {
    print "Eres un adulto.\n";
} else {
    print "Eres un menor.\n";
}

Declaración if-elsif-else

La declaración if-elsif-else permite que se verifiquen múltiples condiciones en secuencia.

my $age = 20;

if ($age < 13) {
    print "Eres un niño.\n";
} elsif ($age < 18) {
    print "Eres un adolescente.\n";
} else {
    print "Eres un adulto.\n";
}

Estructuras de Bucle

Las estructuras de bucle te permiten ejecutar un bloque de código múltiples veces.

Bucle while

El bucle while ejecuta un bloque de código mientras la condición sea verdadera.

my $count = 0;

while ($count < 5) {
    print "El conteo es $count\n";
    $count++;
}

Bucle for

El bucle for se usa para iterar sobre un rango de valores.

for (my $i = 0; $i < 5; $i++) {
    print "Iteración $i\n";
}

Bucle foreach

El bucle foreach se usa para iterar sobre una lista de valores.

my @colors = ('rojo', 'azul', 'verde');

foreach my $color (@colors) {
    print "Color: $color\n";
}

Bucle until

El bucle until es lo opuesto al bucle while; ejecuta un bloque de código mientras la condición sea falsa.

my $count = 0;

until ($count == 5) {
    print "El conteo es $count\n";
    $count++;
}

Modificadores de Flujo de Control

Los modificadores de flujo de control se pueden usar para alterar el comportamiento de los bucles y las declaraciones condicionales.

last

La declaración last se usa para salir de un bucle inmediatamente.

for (my $i = 0; $i < 10; $i++) {
    if ($i == 5) {
        last;
    }
    print "Iteración $i\n";
}

next

La declaración next se usa para saltar el resto de la iteración actual y pasar a la siguiente iteración.

for (my $i = 0; $i < 10; $i++) {
    if ($i % 2 == 0) {
        next;
    }
    print "Número impar: $i\n";
}

redo

La declaración redo reinicia la iteración actual sin evaluar nuevamente la condición del bucle.

my $count = 0;

while ($count < 5) {
    $count++;
    if ($count == 3) {
        redo;
    }
    print "El conteo es $count\n";
}

Estructuras de Control Avanzadas

given-when

La estructura given-when es similar a la declaración switch-case que se encuentra en otros lenguajes.

use feature 'switch';

my $fruit = 'manzana';

given ($fruit) {
    when ('manzana') {
        print "Es una manzana.\n";
    }
    when ('plátano') {
        print "Es un plátano.\n";
    }
    default {
        print "Fruta desconocida.\n";
    }
}

Conclusión

Las estructuras de control son esenciales para dirigir el flujo de tus programas en Perl. Desde declaraciones condicionales básicas como if e if-else hasta estructuras de bucle más complejas como while, for y foreach, entender estos constructos te permitirá escribir código más eficiente y legible. Además, los modificadores de flujo de control como last, next y redo proporcionan un mayor control sobre tus bucles, mientras que las estructuras avanzadas como given-when ofrecen formas más expresivas de manejar múltiples condiciones. Practica estos constructos para volverte competente en la gestión del flujo de tus programas en Perl.
