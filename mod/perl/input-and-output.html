Entrada y Salida en Perl
Arrays ►

Introducción

Las operaciones de entrada y salida (I/O) son fundamentales en cualquier lenguaje de programación. En Perl, las operaciones de I/O te permiten interactuar con usuarios, archivos y otros programas. Esta sección cubrirá los conceptos básicos de lectura y escritura en la consola y archivos, y progresará hacia temas más avanzados como el manejo de archivos binarios y el uso de módulos para tareas de I/O más complejas.

Entrada y Salida en la Consola

Imprimir en la Consola

La función print se utiliza para mostrar texto en la consola.

print "¡Hola, Mundo!\n";
  • Explicación: La función print muestra la cadena "¡Hola, Mundo!" seguida de un carácter de nueva línea \n.

Leer desde la Consola

El manejador de archivos <STDIN> se utiliza para leer la entrada desde la consola.

print "Introduce tu nombre: ";
my $name = <STDIN>;
chomp($name);  # Elimina el carácter de nueva línea
print "¡Hola, $name!\n";
  • Explicación:
    • print "Introduce tu nombre: "; solicita al usuario.
    • my $name = <STDIN>; lee la entrada del usuario.
    • chomp($name); elimina el carácter de nueva línea de la entrada.
    • print "¡Hola, $name!\n"; muestra un saludo con la entrada del usuario.

Entrada y Salida en Archivos

Abrir y Cerrar Archivos

Para leer o escribir en un archivo, primero necesitas abrirlo.

# Abrir un archivo para lectura
open(my $fh, '<', 'input.txt') or die "No se pudo abrir el archivo 'input.txt' $!";

# Abrir un archivo para escritura
open(my $fh, '>', 'output.txt') or die "No se pudo abrir el archivo 'output.txt' $!";

# Cerrar un archivo
close($fh);
  • Explicación:
    • open(my $fh, '<', 'input.txt') abre input.txt para lectura.
    • open(my $fh, '>', 'output.txt') abre output.txt para escritura.
    • or die "No se pudo abrir el archivo 'filename' $!"; maneja errores si el archivo no se puede abrir.
    • close($fh); cierra el manejador de archivos.

Leer desde un Archivo

Puedes leer desde un archivo línea por línea o todo de una vez.

# Leer línea por línea
open(my $fh, '<', 'input.txt') or die "No se pudo abrir el archivo 'input.txt' $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

# Leer todo el archivo
open(my $fh, '<', 'input.txt') or die "No se pudo abrir el archivo 'input.txt' $!";
my @lines = <$fh>;
close($fh);
print @lines;
  • Explicación:
    • El primer ejemplo lee el archivo línea por línea usando un bucle while.
    • El segundo ejemplo lee todo el archivo en un array @lines.

Escribir en un Archivo

Puedes escribir en un archivo usando la función print.

open(my $fh, '>', 'output.txt') or die "No se pudo abrir el archivo 'output.txt' $!";
print $fh "Esta es una línea de texto.\n";
close($fh);
  • Explicación:
    • print $fh "Esta es una línea de texto.\n"; escribe la cadena en el archivo.

Temas Avanzados

Manejo de Archivos Binarios

Los archivos binarios requieren un manejo especial para evitar la corrupción de datos.

# Leer un archivo binario
open(my $fh, '<:raw', 'binaryfile.bin') or die "No se pudo abrir el archivo 'binaryfile.bin' $!";
my $data;
read($fh, $data, -s $fh);
close($fh);

# Escribir en un archivo binario
open(my $fh, '>:raw', 'binaryfile.bin') or die "No se pudo abrir el archivo 'binaryfile.bin' $!";
print $fh $data;
close($fh);
  • Explicación:
    • Los modos '<:raw' y '>:raw' se utilizan para manejar archivos binarios.
    • read($fh, $data, -s $fh); lee todo el archivo binario en $data.

Uso de Módulos para I/O Complejo

Perl tiene varios módulos que simplifican las operaciones de I/O complejas. Uno de esos módulos es File::Slurp.

use File::Slurp;

# Leer un archivo
my $content = read_file('input.txt');

# Escribir en un archivo
write_file('output.txt', $content);
  • Explicación:
    • read_file('input.txt'); lee todo el archivo en $content.
    • write_file('output.txt', $content); escribe $content en el archivo.

Conclusión

Entender las operaciones de entrada y salida en Perl es esencial para una programación efectiva. Esta sección cubrió los conceptos básicos de I/O en la consola y archivos, así como temas más avanzados como el manejo de archivos binarios y el uso de módulos para tareas complejas. Dominar estos conceptos te permitirá construir aplicaciones Perl más interactivas y basadas en datos.
