Introducción

En esta sección, exploraremos estructuras de datos complejas en Perl, tales como arreglos de arreglos, hashes de arreglos, arreglos de hashes y hashes de hashes. Entender estas estructuras es crucial para manejar datos más sofisticados en tus programas.

Arreglos de Arreglos

Los arreglos de arreglos, también conocidos como arreglos multidimensionales, te permiten crear una estructura similar a una matriz.

Creando un Arreglo de Arreglos

my @matrix = (
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
);

Accediendo a Elementos

Para acceder a elementos en un arreglo de arreglos:

print $matrix[0][1];  # Imprime 2
print $matrix[2][0];  # Imprime 7

Iterando Sobre un Arreglo de Arreglos

foreach my $row (@matrix) {
    foreach my $element (@$row) {
        print "$element ";
    }
    print "\n";
}

Hashes de Arreglos

Los hashes de arreglos te permiten asociar múltiples valores con una sola clave.

Creando un Hash de Arreglos

my %hash_of_arrays = (
    fruits => ['apple', 'banana', 'cherry'],
    vegetables => ['carrot', 'pea', 'potato']
);

Accediendo a Elementos

Para acceder a elementos en un hash de arreglos:

print $hash_of_arrays{fruits}[1];  # Imprime 'banana'
print $hash_of_arrays{vegetables}[2];  # Imprime 'potato'

Iterando Sobre un Hash de Arreglos

foreach my $key (keys %hash_of_arrays) {
    print "$key: ";
    foreach my $item (@{$hash_of_arrays{$key}}) {
        print "$item ";
    }
    print "\n";
}

Arreglos de Hashes

Los arreglos de hashes te permiten almacenar una lista de referencias de hash, útil para representar registros.

Creando un Arreglo de Hashes

my @array_of_hashes = (
    { name => 'Alice', age => 30 },
    { name => 'Bob', age => 25 },
    { name => 'Charlie', age => 35 }
);

Accediendo a Elementos

Para acceder a elementos en un arreglo de hashes:

print $array_of_hashes[0]{name};  # Imprime 'Alice'
print $array_of_hashes[1]{age};  # Imprime 25

Iterando Sobre un Arreglo de Hashes

foreach my $record (@array_of_hashes) {
    print "Name: $record->{name}, Age: $record->{age}\n";
}

Hashes de Hashes

Los hashes de hashes te permiten crear una estructura de hash anidada, útil para representar relaciones complejas.

Creando un Hash de Hashes

my %hash_of_hashes = (
    person1 => { name => 'Alice', age => 30 },
    person2 => { name => 'Bob', age => 25 },
    person3 => { name => 'Charlie', age => 35 }
);

Accediendo a Elementos

Para acceder a elementos en un hash de hashes:

print $hash_of_hashes{person1}{name};  # Imprime 'Alice'
print $hash_of_hashes{person2}{age};  # Imprime 25

Iterando Sobre un Hash de Hashes

foreach my $person (keys %hash_of_hashes) {
    print "Name: $hash_of_hashes{$person}{name}, Age: $hash_of_hashes{$person}{age}\n";
}

Resumen

En esta sección, cubrimos:

  • Arreglos de arreglos para datos multidimensionales.
  • Hashes de arreglos para asociar múltiples valores con una sola clave.
  • Arreglos de hashes para almacenar listas de registros.
  • Hashes de hashes para estructuras de hash anidadas.

Estas estructuras de datos complejas son esenciales para manejar y organizar datos en Perl, permitiéndote escribir código más eficiente y legible. Practica creando y manipulando estas estructuras para volverte competente en el manejo de datos complejos en tus programas Perl.
