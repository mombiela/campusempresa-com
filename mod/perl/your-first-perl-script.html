Introducción

¡Bienvenido a tu primer script en Perl! Perl es un lenguaje de scripting versátil y poderoso utilizado para una variedad de tareas, desde el procesamiento de texto hasta el desarrollo web. En esta sección, te guiaremos a través de la escritura y ejecución de tu primer script en Perl. Al final de esta lección, tendrás una comprensión básica de la sintaxis de Perl y cómo ejecutar scripts en Perl.

Configurando Tu Entorno

Antes de comenzar a escribir nuestro primer script en Perl, asegúrate de tener Perl instalado en tu sistema. Puedes verificar si Perl está instalado ejecutando el siguiente comando en tu terminal o símbolo del sistema:

perl -v

Si Perl está instalado, verás la información de la versión. Si no, puedes descargar e instalar Perl desde perl.org.

Escribiendo Tu Primer Script en Perl

Vamos a sumergirnos en la escritura de un simple script en Perl. Crearemos un script que imprime "¡Hola, Mundo!" en la consola.

Guía Paso a Paso

  1. Crear un Nuevo Archivo: Abre tu editor de texto favorito y crea un nuevo archivo llamado hello_world.pl.

  2. Escribir el Script: Agrega el siguiente código a tu archivo hello_world.pl:

    #!/usr/bin/perl
use strict;
use warnings;

print "¡Hola, Mundo!\n";
    • #!/usr/bin/perl: Esta es la línea shebang que le dice al sistema que use el intérprete de Perl para ejecutar el script.
    • use strict; y use warnings;: Estas líneas habilitan los modos strict y warnings, que ayudan a detectar errores potenciales y a imponer buenas prácticas de codificación.
    • print "¡Hola, Mundo!\n";: Esta línea imprime "¡Hola, Mundo!" seguido de un carácter de nueva línea en la consola.

  3. Guardar el Archivo: Guarda el archivo y cierra el editor de texto.

Ejecutando Tu Script en Perl

Ahora que has escrito tu primer script en Perl, es hora de ejecutarlo.

  1. Abrir Terminal o Símbolo del Sistema: Navega al directorio donde guardaste hello_world.pl.

  2. Ejecutar el Script: Ejecuta el script escribiendo el siguiente comando:

    perl hello_world.pl

    Deberías ver la salida:

    ¡Hola, Mundo!

Conceptos Clave

  • Línea Shebang (#!/usr/bin/perl):
    • Le dice al sistema operativo que use el intérprete de Perl para ejecutar el script.
  • Modos Strict y Warnings:
    • use strict;: Impone reglas estrictas de declaración de variables.
    • use warnings;: Proporciona advertencias para posibles problemas en el código.
  • Función Print:
    • print "¡Hola, Mundo!\n";: Muestra texto en la consola.

Errores Comunes y Solución de Problemas

  • Permiso Denegado:
    • Si encuentras un error de "Permiso denegado", asegúrate de que el script tenga permisos de ejecución. Puedes agregar permisos de ejecución con el siguiente comando: 
      chmod +x hello_world.pl
  • Comando No Encontrado:
    • Si ves un error de "comando no encontrado", asegúrate de que Perl esté instalado y que la línea shebang apunte a la ruta correcta del intérprete de Perl.

Conclusión

¡Felicidades! Has escrito y ejecutado tu primer script en Perl. Este simple programa de "¡Hola, Mundo!" te introduce a la estructura básica de un script en Perl, incluyendo la línea shebang, los modos strict y warnings, y la función print. A medida que avances en este curso, construirás sobre estos fundamentos para crear scripts en Perl más complejos y poderosos. ¡Feliz codificación!
