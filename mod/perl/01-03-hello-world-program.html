Programa Hola Mundo
Siguiente ►

En esta lección, aprenderemos a escribir nuestro primer programa en Perl: el clásico "Hola Mundo". Este programa es una excelente manera de familiarizarse con la sintaxis básica de Perl y con el proceso de ejecución de un script Perl.

Objetivos de la Lección

  • Entender la estructura básica de un script Perl.
  • Aprender a imprimir texto en la consola.
  • Ejecutar un script Perl desde la línea de comandos.

Estructura Básica de un Script Perl

Un script Perl es simplemente un archivo de texto que contiene código Perl. La extensión comúnmente utilizada para los archivos Perl es .pl. A continuación, se muestra la estructura básica de un script Perl:

#!/usr/bin/perl
# Este es un comentario en Perl

print "Hola Mundo\n";

Desglose del Código

  1. Shebang (#!/usr/bin/perl):

    • La primera línea #!/usr/bin/perl se conoce como "shebang" y le indica al sistema operativo que use el intérprete de Perl ubicado en /usr/bin/perl para ejecutar el script. Esta línea es opcional en algunos sistemas, pero es una buena práctica incluirla.

  2. Comentarios:

    • Los comentarios en Perl comienzan con el símbolo #. Todo lo que sigue al # en esa línea es ignorado por el intérprete de Perl. Los comentarios son útiles para documentar el código.

  3. Impresión de Texto:

    • La función print se utiliza para imprimir texto en la consola. En este caso, print "Hola Mundo\n"; imprime la cadena "Hola Mundo" seguida de un salto de línea (\n).

Ejemplo Práctico

Vamos a crear y ejecutar nuestro primer script Perl.

Paso 1: Crear el Archivo del Script

  1. Abre tu editor de texto favorito (por ejemplo, Notepad++, Sublime Text, Visual Studio Code).
  2. Escribe el siguiente código:
#!/usr/bin/perl
# Este es un comentario en Perl

print "Hola Mundo\n";
  1. Guarda el archivo con el nombre hola_mundo.pl.

Paso 2: Ejecutar el Script

Para ejecutar el script, sigue estos pasos:

  1. Abre una terminal o línea de comandos.
  2. Navega al directorio donde guardaste el archivo hola_mundo.pl.
  3. Ejecuta el script escribiendo el siguiente comando:
perl hola_mundo.pl

Resultado Esperado

Al ejecutar el script, deberías ver la siguiente salida en la consola:

Hola Mundo

Ejercicio Práctico

Ejercicio 1: Modificar el Mensaje

Modifica el script hola_mundo.pl para que imprima un mensaje diferente, por ejemplo, "¡Bienvenidos a Perl!".

Solución

#!/usr/bin/perl
# Este es un comentario en Perl

print "¡Bienvenidos a Perl!\n";

Ejercicio 2: Añadir Más Líneas de Texto

Añade más líneas de texto para que el script imprima tres mensajes diferentes, uno debajo del otro.

Solución

#!/usr/bin/perl
# Este es un comentario en Perl

print "Hola Mundo\n";
print "¡Bienvenidos a Perl!\n";
print "Este es nuestro primer script.\n";

Conclusión

En esta lección, hemos aprendido a escribir y ejecutar un simple programa "Hola Mundo" en Perl. Hemos cubierto la estructura básica de un script Perl, cómo imprimir texto en la consola y cómo ejecutar un script desde la línea de comandos. Estos son los primeros pasos fundamentales para cualquier programador que quiera aprender Perl.

En la próxima lección, profundizaremos en la sintaxis y estructura básica de Perl, lo que nos permitirá escribir scripts más complejos y útiles. ¡Sigue practicando y nos vemos en la siguiente lección!
