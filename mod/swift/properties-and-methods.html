Introducción

En Swift, las propiedades y los métodos son conceptos fundamentales que te permiten almacenar y manipular datos dentro de tus clases, estructuras y enumeraciones. Comprender estos conceptos es crucial para escribir código Swift eficiente y efectivo.

Propiedades

Las propiedades asocian valores con una clase, estructura o enumeración en particular. Pueden categorizarse en propiedades almacenadas, propiedades calculadas y observadores de propiedades.

Propiedades Almacenadas

Las propiedades almacenadas guardan valores constantes o variables como parte de una instancia. Pueden ser propiedades variables (declaradas con var) o propiedades constantes (declaradas con let).

struct Person {
    var name: String
    let birthYear: Int
}

var person = Person(name: "Alice", birthYear: 1990)
person.name = "Bob" // Permitido porque 'name' es una propiedad variable
// person.birthYear = 1991 // Error: No se puede asignar a 'birthYear' porque es una constante 'let'

Propiedades Calculadas

Las propiedades calculadas no almacenan un valor directamente. En su lugar, proporcionan un getter y un setter opcional para recuperar y establecer otras propiedades o valores indirectamente.

struct Rectangle {
    var width: Double
    var height: Double
    
    var area: Double {
        return width * height
    }
}

let rect = Rectangle(width: 5.0, height: 10.0)
print("Área: \(rect.area)") // Salida: Área: 50.0

Observadores de Propiedades

Los observadores de propiedades observan y responden a los cambios en el valor de una propiedad. Pueden añadirse a propiedades almacenadas (excepto a las propiedades lazy) y se llaman cada vez que se establece el valor de una propiedad.

class StepCounter {
    var totalSteps: Int = 0 {
        willSet(newTotalSteps) {
            print("A punto de establecer totalSteps a \(newTotalSteps)")
        }
        didSet {
            if totalSteps > oldValue {
                print("Añadidos \(totalSteps - oldValue) pasos")
            }
        }
    }
}

let stepCounter = StepCounter()
stepCounter.totalSteps = 200
// Salida:
// A punto de establecer totalSteps a 200
// Añadidos 200 pasos

Métodos

Los métodos son funciones que están asociadas con un tipo particular. Pueden ser métodos de instancia o métodos de tipo.

Métodos de Instancia

Los métodos de instancia son funciones que pertenecen a instancias de una clase, estructura o enumeración en particular.

class Counter {
    var count = 0
    
    func increment() {
        count += 1
    }
    
    func increment(by amount: Int) {
        count += amount
    }
    
    func reset() {
        count = 0
    }
}

let counter = Counter()
counter.increment()
print(counter.count) // Salida: 1
counter.increment(by: 5)
print(counter.count) // Salida: 6
counter.reset()
print(counter.count) // Salida: 0

Métodos de Tipo

Los métodos de tipo son métodos que se llaman en el tipo en sí, en lugar de en una instancia del tipo. Se definen con la palabra clave static para estructuras y enumeraciones, y con la palabra clave class para clases para permitir la sobrescritura de métodos.

class SomeClass {
    class func someTypeMethod() {
        print("Método de tipo llamado")
    }
}

SomeClass.someTypeMethod() // Salida: Método de tipo llamado

Conclusión

Las propiedades y los métodos son bloques de construcción esenciales en la programación Swift. Las propiedades te permiten almacenar y calcular valores, mientras que los métodos te permiten definir comportamientos para tus tipos. Al dominar estos conceptos, puedes crear aplicaciones Swift más robustas y mantenibles. Practica usando propiedades almacenadas, propiedades calculadas, observadores de propiedades, métodos de instancia y métodos de tipo para profundizar tu comprensión y mejorar tus habilidades de codificación.
