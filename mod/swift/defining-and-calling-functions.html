Definiendo y Llamando Funciones en Swift

Introducción

Las funciones son bloques fundamentales en la programación con Swift. Permiten encapsular un bloque de código que realiza una tarea específica, que luego puede ser reutilizado a lo largo de tu programa. Esta sección cubrirá cómo definir y llamar funciones, desde conceptos básicos hasta avanzados.

Definiendo Funciones

Definición Básica de Función

Una función en Swift se define usando la palabra clave func, seguida del nombre de la función, un par de paréntesis y un conjunto de llaves que contienen el código de la función.

func greet() {
    print("¡Hola, Mundo!")
}
  • func: Palabra clave para definir una función.
  • greet: Nombre de la función.
  • (): Paréntesis que pueden contener parámetros.
  • { ... }: Llaves que contienen el código de la función.

Función con Parámetros

Las funciones pueden tomar parámetros para hacerlas más flexibles y reutilizables.

func greet(name: String) {
    print("¡Hola, \(name)!")
}
  • name: String: Parámetro llamado name de tipo String.

Función con Valor de Retorno

Las funciones también pueden devolver valores. El tipo de retorno se especifica después de la lista de parámetros.

func add(a: Int, b: Int) -> Int {
    return a + b
}
  • -> Int: Indica que la función devuelve un Int.

Llamando Funciones

Llamando una Función sin Parámetros

Para llamar una función, simplemente usa su nombre seguido de paréntesis.

greet()

Llamando una Función con Parámetros

Cuando llamas una función con parámetros, proporciona los argumentos dentro de los paréntesis.

greet(name: "Alice")

Usando el Valor de Retorno de una Función

Puedes capturar el valor de retorno de una función en una variable o usarlo directamente.

let sum = add(a: 5, b: 3)
print(sum) // Salida: 8

Conceptos Avanzados de Funciones

Valores Predeterminados de Parámetros

Puedes proporcionar valores predeterminados para los parámetros, haciéndolos opcionales al llamar la función.

func greet(name: String = "Mundo") {
    print("¡Hola, \(name)!")
}

greet() // Salida: ¡Hola, Mundo!
greet(name: "Alice") // Salida: ¡Hola, Alice!

Parámetros Variádicos

Un parámetro variádico acepta cero o más valores de un tipo especificado.

func sum(numbers: Int...) -> Int {
    var total = 0
    for number in numbers {
        total += number
    }
    return total
}

let totalSum = sum(numbers: 1, 2, 3, 4, 5)
print(totalSum) // Salida: 15

Parámetros In-Out

Los parámetros in-out permiten que una función modifique el valor de un parámetro.

func increment(value: inout Int) {
    value += 1
}

var number = 10
increment(value: &number)
print(number) // Salida: 11

Tipos de Funciones

Las funciones tienen tipos, que pueden ser usados como parámetros o tipos de retorno para otras funciones.

func multiply(a: Int, b: Int) -> Int {
    return a * b
}

let operation: (Int, Int) -> Int = multiply
let result = operation(3, 4)
print(result) // Salida: 12

Funciones Anidadas

Las funciones pueden ser definidas dentro de otras funciones, proporcionando una manera de encapsular funciones auxiliares.

func outerFunction() {
    func innerFunction() {
        print("Función interna")
    }
    innerFunction()
}

outerFunction() // Salida: Función interna

Conclusión

Las funciones en Swift son herramientas versátiles y poderosas que te ayudan a escribir código limpio, reutilizable y mantenible. Desde definiciones y llamadas básicas hasta conceptos avanzados como parámetros in-out y funciones anidadas, dominar las funciones mejorará significativamente tus habilidades de programación en Swift. Practica definiendo y llamando funciones con diferentes parámetros y tipos de retorno para volverte competente en su uso efectivo.
