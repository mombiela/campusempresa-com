Introducción

Enviar una aplicación a la App Store es un paso crítico en el ciclo de vida del desarrollo de aplicaciones. Esta sección te guiará a través del proceso de preparar tu aplicación Swift para su envío, asegurando que cumpla con las directrices de Apple y enviándola exitosamente a la App Store.

Requisitos previos

Antes de comenzar el proceso de envío, asegúrate de tener:

  • Una cuenta de desarrollador de Apple.
  • Una aplicación completa que cumpla con las Directrices de Revisión de la App Store de Apple.
  • Metadatos de la aplicación (nombre, descripción, palabras clave, capturas de pantalla, etc.).

Preparando tu aplicación para el envío

Directrices de Revisión de la App Store

  • Funcionalidad: Asegúrate de que tu aplicación esté completa y totalmente funcional.
  • Rendimiento: Prueba tu aplicación a fondo para evitar bloqueos y errores.
  • Diseño: Sigue las Directrices de Interfaz Humana de Apple para una experiencia de usuario coherente.
  • Legal: Cumple con todas las leyes y regulaciones relevantes.

Metadatos de la aplicación

  • Nombre de la aplicación: Elige un nombre único y descriptivo.
  • Descripción: Escribe una descripción clara y concisa de las características y funcionalidades de tu aplicación.
  • Palabras clave: Selecciona palabras clave relevantes para mejorar la visibilidad de tu aplicación.
  • Capturas de pantalla: Proporciona capturas de pantalla de alta calidad que muestren las principales características de tu aplicación.
  • Icono de la aplicación: Diseña un icono de aplicación atractivo y reconocible.

Creando un perfil de aprovisionamiento

Pasos para crear un perfil de aprovisionamiento

  1. Inicia sesión en el Portal de Desarrolladores de Apple.
  2. Navega a Certificados, Identificadores y Perfiles.
  3. Crea un nuevo ID de aplicación:
    • Selecciona App IDs bajo Identifiers.
    • Haz clic en el botón + para crear un nuevo ID de aplicación.
    • Rellena los campos requeridos y registra tu ID de aplicación.
  4. Genera un perfil de aprovisionamiento:
    • Selecciona Provisioning Profiles bajo Profiles.
    • Haz clic en el botón + para crear un nuevo perfil.
    • Elige el tipo de perfil (por ejemplo, App Store, Ad Hoc).
    • Selecciona el ID de aplicación que creaste.
    • Elige los certificados y dispositivos (si aplica).
    • Descarga e instala el perfil.

Archivando tu aplicación

Pasos para archivar tu aplicación en Xcode

  1. Abre tu proyecto en Xcode.
  2. Selecciona un Dispositivo iOS Genérico como el dispositivo objetivo.
  3. Ve a Producto > Archivo.
  4. Espera a que se complete el proceso de archivado.
  5. Se abrirá la ventana del Organizador con tu aplicación archivada.

Subiendo tu aplicación a App Store Connect

Pasos para subir tu aplicación

  1. Abre la ventana del Organizador en Xcode.
  2. Selecciona tu aplicación archivada.
  3. Haz clic en el botón Distribuir Aplicación.
  4. Elige el método de distribución App Store Connect.
  5. Sigue las indicaciones para validar y subir tu aplicación.

Configurando tu aplicación en App Store Connect

Pasos para configurar tu aplicación

  1. Inicia sesión en App Store Connect.
  2. Navega a Mis Apps.
  3. Selecciona tu aplicación o crea una nueva.
  4. Rellena la información requerida de la aplicación:
    • Nombre de la aplicación
    • Idioma principal
    • ID del paquete
    • SKU
  5. Sube la versión de tu aplicación:
    • Selecciona la versión que subiste desde Xcode.
  6. Completa los metadatos de la aplicación:
    • Descripción
    • Palabras clave
    • URL de soporte
    • URL de marketing
  7. Añade capturas de pantalla y el icono de la aplicación.
  8. Establece el precio y la disponibilidad de la aplicación.

Enviando tu aplicación para revisión

Pasos para enviar tu aplicación

  1. Navega a la sección de Revisión de la Aplicación en App Store Connect.
  2. Selecciona la versión que deseas enviar.
  3. Responde las preguntas requeridas sobre cumplimiento de exportación.
  4. Envía tu aplicación para revisión.

Conclusión

Enviar una aplicación a la App Store implica varios pasos, desde preparar tu aplicación y crear un perfil de aprovisionamiento hasta subir tu aplicación y configurarla en App Store Connect. Siguiendo este proceso estructurado, puedes asegurarte de que tu aplicación cumpla con las directrices de Apple y esté lista para su revisión. Recuerda probar a fondo tu aplicación y proporcionar toda la información necesaria para evitar retrasos en el proceso de revisión. ¡Buena suerte con el envío de tu aplicación!
