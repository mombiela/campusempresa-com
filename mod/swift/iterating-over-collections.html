Iterando Sobre Colecciones
Introducción a Clases y Estructuras ►

En esta sección, exploraremos cómo iterar sobre colecciones en Swift. Las colecciones en Swift incluyen arrays, diccionarios, conjuntos y rangos. Entender cómo iterar eficientemente sobre estas colecciones es fundamental para cualquier programador de Swift.

Conceptos Clave

  • Colecciones en Swift: Arrays, diccionarios, conjuntos y rangos.
  • Bucle For-in: El bucle principal utilizado para iterar sobre colecciones.
  • Método Enumerated: Utilizado para obtener tanto el índice como el valor.
  • Funciones de Orden Superior: forEach, map, filter, reduce.

Iterando Sobre Arrays

Los arrays son colecciones ordenadas de elementos. Comencemos con lo básico de iterar sobre arrays.

Usando Bucle For-in

El bucle for-in es la forma más común de iterar sobre un array.

let fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits {
    print(fruit)
}

Explicación:

  • fruits es un array de cadenas.
  • El bucle for-in itera sobre cada elemento en el array y lo imprime.

Usando el Método Enumerated

El método enumerated proporciona tanto el índice como el valor de cada elemento en el array.

let fruits = ["Apple", "Banana", "Cherry"]

for (index, fruit) in fruits.enumerated() {
    print("Fruit \(index + 1): \(fruit)")
}

Explicación:

  • enumerated devuelve una secuencia de pares (índice, elemento).
  • El bucle imprime tanto el índice como el valor.

Iterando Sobre Diccionarios

Los diccionarios son colecciones de pares clave-valor. Iterar sobre diccionarios es ligeramente diferente de los arrays.

Usando Bucle For-in

let capitals = ["France": "Paris", "Japan": "Tokyo", "USA": "Washington D.C."]

for (country, capital) in capitals {
    print("The capital of \(country) is \(capital).")
}

Explicación:

  • capitals es un diccionario con nombres de países como claves y ciudades capitales como valores.
  • El bucle for-in itera sobre cada par clave-valor.

Iterando Sobre Conjuntos

Los conjuntos son colecciones desordenadas de elementos únicos. Iterar sobre conjuntos es similar a los arrays pero sin orden garantizado.

Usando Bucle For-in

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Explicación:

  • uniqueNumbers es un conjunto de enteros.
  • El bucle for-in itera sobre cada elemento en el conjunto.

Iterando Sobre Rangos

Los rangos representan una secuencia de valores. A menudo se utilizan en bucles.

Usando Bucle For-in

for number in 1...5 {
    print(number)
}

Explicación:

  • 1...5 es un rango cerrado que incluye números del 1 al 5.
  • El bucle for-in itera sobre cada número en el rango.

Funciones de Orden Superior

Swift proporciona varias funciones de orden superior que pueden ser utilizadas para iterar sobre colecciones de una manera más funcional.

forEach

El método forEach es una alternativa al bucle for-in.

let fruits = ["Apple", "Banana", "Cherry"]

fruits.forEach { fruit in
    print(fruit)
}

Explicación:

  • forEach toma un closure que se ejecuta para cada elemento en el array.

map

El método map transforma cada elemento en la colección.

let numbers = [1, 2, 3, 4, 5]
let squaredNumbers = numbers.map { $0 * $0 }

print(squaredNumbers)

Explicación:

  • map aplica el closure a cada elemento y devuelve un nuevo array con los elementos transformados.

filter

El método filter devuelve una nueva colección que contiene elementos que satisfacen una condición dada.

let numbers = [1, 2, 3, 4, 5]
let evenNumbers = numbers.filter { $0 % 2 == 0 }

print(evenNumbers)

Explicación:

  • filter aplica el closure a cada elemento y devuelve un nuevo array con elementos que cumplen la condición.

reduce

El método reduce combina todos los elementos de la colección en un solo valor.

let numbers = [1, 2, 3, 4, 5]
let sum = numbers.reduce(0, +)

print(sum)

Explicación:

  • reduce toma un valor inicial y un closure que combina el valor inicial con cada elemento del array.

Conclusión

Iterar sobre colecciones es una habilidad fundamental en la programación en Swift. Ya sea que estés trabajando con arrays, diccionarios, conjuntos o rangos, entender los diversos métodos y técnicas para la iteración te ayudará a escribir código más eficiente y legible. Además, aprovechar las funciones de orden superior como forEach, map, filter y reduce puede llevar a un código más expresivo y conciso. Practica estos conceptos para volverte competente en el manejo de colecciones en Swift.
