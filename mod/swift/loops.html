Bucles en Swift
Sentencias de Transferencia de Control ►

Los bucles son constructos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. Swift proporciona varios tipos de bucles para manejar diferentes escenarios. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de los bucles en Swift.

Tipos de Bucles

Swift admite tres tipos principales de bucles:

  • Bucles for-in
  • Bucles while
  • Bucles repeat-while

Bucles `for-in`

El bucle for-in se utiliza para iterar sobre una secuencia, como elementos en un array, rangos de números o caracteres en una cadena.

Sintaxis

for item in collection {
    // Código a ejecutar
}

Ejemplo

let fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits {
    print(fruit)
}

En este ejemplo, el bucle itera sobre cada elemento en el array fruits y lo imprime.

Bucles `while`

El bucle while continúa ejecutando un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

while condition {
    // Código a ejecutar
}

Ejemplo

var count = 5

while count > 0 {
    print("Count is \(count)")
    count -= 1
}

Este bucle imprimirá el valor de count y lo decrementará hasta que count ya no sea mayor que 0.

Bucles `repeat-while`

El bucle repeat-while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez antes de evaluar la condición.

Sintaxis

repeat {
    // Código a ejecutar
} while condition

Ejemplo

var count = 5

repeat {
    print("Count is \(count)")
    count -= 1
} while count > 0

Este bucle se comporta de manera similar al ejemplo del bucle while pero asegura que el bloque de código se ejecute al menos una vez.

Declaraciones de Control de Bucles

Swift proporciona declaraciones de control para alterar el flujo de los bucles:

  • break
  • continue

Declaración `break`

La declaración break termina el bucle inmediatamente.

Ejemplo

for number in 1...5 {
    if number == 3 {
        break
    }
    print(number)
}

Este bucle imprimirá 1 y 2, luego terminará cuando number sea igual a 3.

Declaración `continue`

La declaración continue omite la iteración actual y procede a la siguiente iteración del bucle.

Ejemplo

for number in 1...5 {
    if number == 3 {
        continue
    }
    print(number)
}

Este bucle imprimirá 1, 2, 4 y 5, omitiendo la iteración donde number es igual a 3.

Bucles Anidados

Los bucles pueden anidarse dentro de otros bucles para manejar escenarios más complejos.

Ejemplo

for i in 1...3 {
    for j in 1...3 {
        print("i = \(i), j = \(j)")
    }
}

Este ejemplo demuestra un bucle anidado donde el bucle exterior se ejecuta tres veces, y para cada iteración del bucle exterior, el bucle interior también se ejecuta tres veces.

Iterando a Través de Colecciones

El bucle for-in de Swift es particularmente útil para iterar a través de colecciones como arrays, diccionarios y conjuntos.

Iterando a Través de un Array

let numbers = [1, 2, 3, 4, 5]

for number in numbers {
    print(number)
}

Iterando a Través de un Diccionario

let studentGrades = ["John": "A", "Jane": "B", "Jim": "C"]

for (student, grade) in studentGrades {
    print("\(student) received a grade of \(grade)")
}

Iterando a Través de un Conjunto

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Técnicas Avanzadas de Bucle

Usando `stride` para Incrementos Personalizados

La función stride te permite crear rangos con incrementos personalizados.

Ejemplo

for number in stride(from: 0, to: 10, by: 2) {
    print(number)
}

Este bucle imprimirá 0, 2, 4, 6 y 8.

Bucle con Secuencias Enumeradas

El método enumerated() devuelve una secuencia de pares (índice, elemento), útil cuando necesitas tanto el índice como el valor.

Ejemplo

let animals = ["Dog", "Cat", "Bird"]

for (index, animal) in animals.enumerated() {
    print("Animal \(index + 1): \(animal)")
}

Este bucle imprimirá:

Animal 1: Dog
Animal 2: Cat
Animal 3: Bird

Conclusión

Los bucles son una herramienta poderosa en Swift que te permiten realizar tareas repetitivas de manera eficiente. Comprender los diferentes tipos de bucles y sus declaraciones de control te permitirá escribir código más efectivo y conciso. Practica usando bucles con varias colecciones y condiciones para volverte competente en su uso.
