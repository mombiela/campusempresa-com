Sentencias condicionales en Swift
Bucles ►

Las sentencias condicionales son fundamentales en la programación ya que permiten ejecutar diferentes códigos basados en ciertas condiciones. En Swift, las principales sentencias condicionales son if, else if, else y switch.

Conceptos clave

  • Sentencia If: Ejecuta un bloque de código si una condición especificada es verdadera.
  • Sentencia Else If: Proporciona una condición adicional para verificar si la condición inicial if es falsa.
  • Sentencia Else: Ejecuta un bloque de código si ninguna de las condiciones anteriores es verdadera.
  • Sentencia Switch: Evalúa un valor y ejecuta bloques de código correspondientes basados en casos coincidentes.

Sentencia If

La sentencia if se utiliza para ejecutar un bloque de código si una condición es verdadera.

let temperature = 30

if temperature > 25 {
    print("¡Es un día caluroso!")
}

En este ejemplo, se imprimirá el mensaje "¡Es un día caluroso!" porque la condición temperature > 25 es verdadera.

Sentencia Else If

La sentencia else if te permite verificar múltiples condiciones.

let temperature = 20

if temperature > 25 {
    print("¡Es un día caluroso!")
} else if temperature > 15 {
    print("¡Es un día cálido!")
}

Aquí, se imprimirá el mensaje "¡Es un día cálido!" porque la primera condición es falsa, pero la segunda condición temperature > 15 es verdadera.

Sentencia Else

La sentencia else se utiliza para ejecutar un bloque de código si ninguna de las condiciones anteriores es verdadera.

let temperature = 10

if temperature > 25 {
    print("¡Es un día caluroso!")
} else if temperature > 15 {
    print("¡Es un día cálido!")
} else {
    print("¡Es un día frío!")
}

En este caso, se imprimirá el mensaje "¡Es un día frío!" porque ambas condiciones anteriores son falsas.

Sentencia Switch

La sentencia switch evalúa un valor y ejecuta el bloque de caso correspondiente.

let dayOfWeek = 3

switch dayOfWeek {
case 1:
    print("Lunes")
case 2:
    print("Martes")
case 3:
    print("Miércoles")
case 4:
    print("Jueves")
case 5:
    print("Viernes")
case 6:
    print("Sábado")
case 7:
    print("Domingo")
default:
    print("Día inválido")
}

En este ejemplo, se imprimirá el mensaje "Miércoles" porque dayOfWeek es 3.

Sentencia Switch con rangos y tuplas

La sentencia switch de Swift es poderosa y soporta rangos y tuplas.

let score = 85

switch score {
case 0..<50:
    print("Reprobado")
case 50..<75:
    print("Aprobado")
case 75...100:
    print("Excelente")
default:
    print("Puntuación inválida")
}

Aquí, se imprimirá el mensaje "Excelente" porque score cae dentro del rango de 75 a 100.

Resumen

  • Sentencia If: Ejecuta código si una condición es verdadera.
  • Sentencia Else If: Verifica condiciones adicionales si las anteriores son falsas.
  • Sentencia Else: Ejecuta código si ninguna de las condiciones es verdadera.
  • Sentencia Switch: Evalúa un valor y lo compara con múltiples casos.

Entender y usar las sentencias condicionales de manera efectiva te permite controlar el flujo de tus programas en Swift, haciéndolos más dinámicos y receptivos a diferentes entradas y condiciones.
