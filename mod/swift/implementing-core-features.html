Introducción

En esta sección, exploraremos cómo implementar funcionalidades básicas en Swift. Esto incluye entender los bloques de construcción fundamentales del lenguaje, tales como variables, constantes, funciones y flujo de control. Al final de esta sección, serás capaz de crear programas básicos en Swift y entender cómo estructurar tu código de manera efectiva.

Variables y Constantes

Las variables y constantes son las unidades básicas de almacenamiento en Swift. Te permiten almacenar y manipular datos.

  • Variables: Usa var para declarar una variable que puede cambiar.
  • Constantes: Usa let para declarar una constante que no puede cambiar.

Ejemplo:

var myVariable = 42
myVariable = 50
print(myVariable) // Salida: 50

let myConstant = 42
// myConstant = 50 // Esto causará un error porque myConstant es una constante
print(myConstant) // Salida: 42

Tipos de Datos

Swift es un lenguaje seguro en cuanto a tipos, lo que significa que te anima a ser claro sobre los tipos de valores con los que tu código puede trabajar.

  • Tipos de Datos Comunes:
    • Int: Números enteros
    • Double y Float: Números de punto flotante
    • Bool: Valores booleanos
    • String: Datos textuales

Ejemplo:

let integer: Int = 10
let floatingPoint: Double = 10.5
let boolean: Bool = true
let text: String = "¡Hola, Swift!"

Funciones

Las funciones son bloques de código autónomos que realizan una tarea específica.

  • Definir una Función: Usa la palabra clave func.
  • Llamar a una Función: Usa el nombre de la función seguido de paréntesis.

Ejemplo:

func greet(name: String) -> String {
    return "¡Hola, \(name)!"
}

let greeting = greet(name: "Alice")
print(greeting) // Salida: ¡Hola, Alice!

Flujo de Control

Las declaraciones de flujo de control te permiten controlar la ejecución de tu código basado en ciertas condiciones.

  • If Statements: Ejecuta código basado en una condición.
  • Switch Statements: Elige entre múltiples valores posibles.
  • Loops: Repite un bloque de código múltiples veces.

Ejemplo:

// If Statement
let condition = true
if condition {
    print("La condición es verdadera")
} else {
    print("La condición es falsa")
}

// Switch Statement
let number = 3
switch number {
case 1:
    print("Uno")
case 2:
    print("Dos")
case 3:
    print("Tres")
default:
    print("Otro número")
}

// For Loop
for i in 1...5 {
    print(i)
}

// While Loop
var count = 5
while count > 0 {
    print(count)
    count -= 1
}

Colecciones

Swift proporciona tipos de colecciones poderosos como arrays, sets y diccionarios para almacenar grupos de valores.

  • Arrays: Colecciones ordenadas de valores.
  • Sets: Colecciones desordenadas de valores únicos.
  • Diccionarios: Colecciones desordenadas de pares clave-valor.

Ejemplo:

// Array
var fruits: [String] = ["Apple", "Banana", "Cherry"]
fruits.append("Date")
print(fruits) // Salida: ["Apple", "Banana", "Cherry", "Date"]

// Set
var uniqueNumbers: Set<Int> = [1, 2, 3, 3, 4]
print(uniqueNumbers) // Salida: [2, 3, 1, 4]

// Dictionary
var capitals: [String: String] = ["France": "Paris", "Japan": "Tokyo"]
capitals["Italy"] = "Rome"
print(capitals) // Salida: ["France": "Paris", "Japan": "Tokyo", "Italy": "Rome"]

Conclusión

En esta sección, cubrimos las funcionalidades básicas de Swift, incluyendo variables, constantes, tipos de datos, funciones, flujo de control y colecciones. Estos bloques de construcción son esenciales para escribir cualquier programa en Swift. Entender y dominar estos conceptos proporcionará una base sólida para temas más avanzados en la programación con Swift.
