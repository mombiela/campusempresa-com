Introducción

En Swift, las variables y constantes son bloques fundamentales para almacenar y manipular datos. Entender cómo declararlas y usarlas es esencial para cualquier programador de Swift. Esta sección cubrirá los conceptos básicos de variables y constantes, incluyendo su declaración, uso y diferencias.

Variables

Las variables se utilizan para almacenar valores que pueden cambiar con el tiempo. En Swift, se declara una variable usando la palabra clave var.

Declaración de Variables

Para declarar una variable, se usa la palabra clave var seguida del nombre de la variable y una anotación de tipo opcional.

var nombreVariable: Tipo = valorInicial

Ejemplo

var edad: Int = 25
var nombre: String = "John"

Conceptos Clave

  • Inferencia de Tipo: Swift a menudo puede inferir el tipo de una variable a partir del valor inicial, por lo que puedes omitir la anotación de tipo.
  • Mutabilidad: Las variables pueden cambiar después de ser declaradas.
var edad = 25 // Swift infiere que edad es de tipo Int
edad = 26 // Cambiando el valor de edad

Constantes

Las constantes se utilizan para almacenar valores que no cambian una vez que se establecen. En Swift, se declara una constante usando la palabra clave let.

Declaración de Constantes

Para declarar una constante, se usa la palabra clave let seguida del nombre de la constante y una anotación de tipo opcional.

let nombreConstante: Tipo = valorInicial

Ejemplo

let añoNacimiento: Int = 1995
let pi: Double = 3.14159

Conceptos Clave

  • Inmutabilidad: Las constantes no pueden cambiar después de ser declaradas.
  • Inferencia de Tipo: Al igual que las variables, Swift puede inferir el tipo de una constante a partir del valor inicial.
let añoNacimiento = 1995 // Swift infiere que añoNacimiento es de tipo Int
// añoNacimiento = 1996 // Esto causará un error porque añoNacimiento es una constante

Comparación entre Variables y Constantes

Característica Variables (var) Constantes (let)
Mutabilidad Pueden cambiar No pueden cambiar
Declaración Palabra clave var Palabra clave let
Caso de Uso Valores que cambian Valores que permanecen constantes

Ejemplos de Código

Uso de Variables

var saludo = "¡Hola, Mundo!"
print(saludo) // Salida: ¡Hola, Mundo!
saludo = "¡Hola, Swift!"
print(saludo) // Salida: ¡Hola, Swift!

Uso de Constantes

let númeroMáximoDeIntentosDeInicioDeSesión = 3
var intentoDeInicioDeSesiónActual = 0

// Descomentar la siguiente línea causará un error
// númeroMáximoDeIntentosDeInicioDeSesión = 4

Conclusión

Entender la diferencia entre variables y constantes es crucial para escribir código efectivo en Swift. Las variables (var) se usan para valores que pueden cambiar, mientras que las constantes (let) se usan para valores que permanecen iguales. Al usarlas apropiadamente, puedes hacer tu código más predecible y fácil de entender. Recuerda aprovechar la inferencia de tipo de Swift para escribir código más limpio y conciso.
