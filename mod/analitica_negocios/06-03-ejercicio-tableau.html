Ejercicio 1: Creación de Dashboards en Tableau
Siguiente ►

Objetivo del Ejercicio

En este ejercicio, aprenderás a crear un dashboard interactivo en Tableau utilizando un conjunto de datos de ventas. Al finalizar, deberás ser capaz de:

  • Importar y preparar datos en Tableau.
  • Crear visualizaciones básicas como gráficos de barras, gráficos de líneas y mapas.
  • Combinar estas visualizaciones en un dashboard interactivo.
  • Añadir filtros y acciones para mejorar la interactividad del dashboard.

Paso 1: Preparación de los Datos

1.1 Descarga del Conjunto de Datos

Para este ejercicio, utilizaremos un conjunto de datos de ventas ficticio. Puedes descargar el archivo CSV desde el siguiente enlace: Datos de Ventas.

1.2 Importación de Datos en Tableau

  1. Abre Tableau y selecciona "Conectar" en la pantalla de inicio.
  2. Selecciona "Archivo de texto" y navega hasta el archivo CSV descargado.
  3. Tableau importará los datos y mostrará una vista previa. Asegúrate de que los datos se vean correctos.

Paso 2: Creación de Visualizaciones Básicas

2.1 Gráfico de Barras de Ventas por Categoría

  1. Arrastra el campo Categoría al estante de filas.
  2. Arrastra el campo Ventas al estante de columnas.
  3. Tableau creará automáticamente un gráfico de barras que muestra las ventas por categoría.

2.2 Gráfico de Líneas de Ventas por Mes

  1. Arrastra el campo Fecha al estante de columnas y selecciona "Mes" en la opción de fecha.
  2. Arrastra el campo Ventas al estante de filas.
  3. Tableau creará un gráfico de líneas que muestra las ventas por mes.

2.3 Mapa de Ventas por Región

  1. Arrastra el campo Región al estante de filas.
  2. Arrastra el campo Ventas al estante de columnas.
  3. Cambia el tipo de gráfico a "Mapa" desde la barra de herramientas de Tableau.
  4. Tableau creará un mapa que muestra las ventas por región.

Paso 3: Creación del Dashboard

3.1 Combinación de Visualizaciones

  1. Ve a la pestaña "Dashboard" y selecciona "Nuevo Dashboard".
  2. Arrastra las hojas de trabajo creadas anteriormente (gráfico de barras, gráfico de líneas y mapa) al área del dashboard.
  3. Ajusta el tamaño y la disposición de las visualizaciones según sea necesario.

3.2 Añadir Filtros Interactivos

  1. En el dashboard, selecciona una de las visualizaciones y haz clic en el icono de filtro en la esquina superior derecha.
  2. Repite este paso para las otras visualizaciones.
  3. Ahora, los filtros aplicados a una visualización afectarán a todas las visualizaciones en el dashboard.

3.3 Añadir Acciones

  1. En el menú superior, selecciona "Dashboard" > "Acciones".
  2. Haz clic en "Añadir Acción" y selecciona "Filtro".
  3. Configura la acción para que al hacer clic en una visualización, se filtren las otras visualizaciones en el dashboard.
  4. Haz clic en "Aceptar" para guardar la acción.

Paso 4: Publicación del Dashboard

  1. Una vez que estés satisfecho con el dashboard, selecciona "Archivo" > "Guardar como" y guarda el archivo de Tableau.
  2. Para compartir el dashboard, selecciona "Servidor" > "Publicar en Tableau Public" o "Publicar en Tableau Server" si tienes acceso a un servidor de Tableau.

Ejercicio Práctico

Tarea

Crea un dashboard en Tableau utilizando los pasos anteriores y responde las siguientes preguntas:

  1. ¿Cuál es la categoría con mayores ventas?
  2. ¿Cómo varían las ventas a lo largo del año?
  3. ¿Qué región tiene las ventas más altas?

Solución

  1. La categoría con mayores ventas se puede identificar observando el gráfico de barras.
  2. La variación de las ventas a lo largo del año se puede analizar mediante el gráfico de líneas.
  3. La región con las ventas más altas se puede determinar a partir del mapa.

Conclusión

En este ejercicio, has aprendido a crear un dashboard interactivo en Tableau, combinando diferentes tipos de visualizaciones y añadiendo filtros y acciones para mejorar la interactividad. Estas habilidades son fundamentales para analizar y presentar datos de manera efectiva en un entorno de negocios.

Resumen

  • Importación y preparación de datos en Tableau.
  • Creación de gráficos de barras, líneas y mapas.
  • Combinación de visualizaciones en un dashboard.
  • Añadir filtros y acciones para interactividad.
  • Publicación del dashboard.

Con estas habilidades, estás listo para explorar más funcionalidades avanzadas de Tableau y aplicar la analítica de datos en tus proyectos de negocios.
