Filtrando Datos con WHERE

Introducción

En SQL, la cláusula WHERE se utiliza para filtrar registros que cumplen ciertos criterios. Esta cláusula es esencial para consultar datos específicos de una base de datos, convirtiéndola en un concepto fundamental tanto para principiantes como para usuarios avanzados.

Uso Básico de WHERE

La cláusula WHERE se utiliza en una declaración SELECT para especificar condiciones que los registros deben cumplir para ser incluidos en el conjunto de resultados.

Sintaxis

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Ejemplo

Considera una tabla llamada employees con las siguientes columnas: id, name, age, y department.

SELECT name, age
FROM employees
WHERE department = 'Sales';

Esta consulta recupera el name y age de los empleados que trabajan en el departamento de Ventas.

Operadores Comunes en la Cláusula WHERE

La cláusula WHERE admite varios operadores para definir condiciones.

  • Operadores de Comparación: =, !=, >, <, >=, <=
  • Operadores Lógicos: AND, OR, NOT
  • Operadores de Rango: BETWEEN
  • Operadores de Conjunto: IN
  • Coincidencia de Patrones: LIKE
  • Verificación de Nulos: IS NULL, IS NOT NULL

Ejemplos

Operadores de Comparación

SELECT name, age
FROM employees
WHERE age > 30;

Esta consulta recupera el name y age de los empleados que tienen más de 30 años.

Operadores Lógicos

SELECT name, department
FROM employees
WHERE department = 'Sales' AND age < 40;

Esta consulta recupera el name y department de los empleados que trabajan en Ventas y tienen menos de 40 años.

Operadores de Rango

SELECT name, age
FROM employees
WHERE age BETWEEN 25 AND 35;

Esta consulta recupera el name y age de los empleados cuya edad está entre 25 y 35 años.

Operadores de Conjunto

SELECT name, department
FROM employees
WHERE department IN ('Sales', 'Marketing');

Esta consulta recupera el name y department de los empleados que trabajan en Ventas o Marketing.

Coincidencia de Patrones

SELECT name
FROM employees
WHERE name LIKE 'J%';

Esta consulta recupera el name de los empleados cuyo nombre comienza con 'J'.

Verificación de Nulos

SELECT name, department
FROM employees
WHERE department IS NULL;

Esta consulta recupera el name y department de los empleados que no tienen un departamento asignado.

Técnicas Avanzadas de Filtrado

Combinación de Múltiples Condiciones

Puedes combinar múltiples condiciones usando operadores lógicos para crear consultas complejas.

SELECT name, age, department
FROM employees
WHERE (department = 'Sales' OR department = 'Marketing') AND age > 30;

Esta consulta recupera el name, age, y department de los empleados que trabajan en Ventas o Marketing y tienen más de 30 años.

Subconsultas en la Cláusula WHERE

Se puede usar una subconsulta dentro de una cláusula WHERE para filtrar datos basados en el resultado de otra consulta.

SELECT name, age
FROM employees
WHERE department = (SELECT department FROM departments WHERE manager_id = 5);

Esta consulta recupera el name y age de los empleados que trabajan en el departamento gestionado por el gerente con id 5.

Conclusión

La cláusula WHERE es una herramienta poderosa en SQL para filtrar datos basados en condiciones específicas. Al dominar el uso de varios operadores y combinar múltiples condiciones, puedes crear consultas precisas y eficientes para extraer los datos que necesitas. Practica usando la cláusula WHERE con diferentes operadores y condiciones para volverte competente en el filtrado de datos en SQL.
