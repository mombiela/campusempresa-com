Introducción a los Registros

Los registros en Delphi/Object Pascal son una forma de agrupar datos relacionados. Son similares a las estructuras en C/C++ y pueden contener campos de diferentes tipos. Los registros son útiles para organizar datos que pertenecen juntos y pueden ayudar a que tu código sea más legible y mantenible.

Conceptos Clave

  • Definición: Un registro es un tipo de datos compuesto que agrupa diferentes tipos de datos.
  • Campos: Los elementos individuales dentro de un registro.
  • Acceso: Los campos dentro de un registro se acceden usando el operador punto (.).

Definiendo Registros

Para definir un registro en Delphi/Object Pascal, se usa la palabra clave record. Aquí hay un ejemplo básico:

type
  TPerson = record
    FirstName: string;
    LastName: string;
    Age: Integer;
  end;

En este ejemplo:

  • TPerson es el nombre del tipo de registro.
  • FirstName, LastName y Age son campos dentro del registro.

Creando y Usando Registros

Una vez que has definido un tipo de registro, puedes crear variables de ese tipo y acceder a sus campos.

Ejemplo

var
  Person: TPerson;
begin
  Person.FirstName := 'John';
  Person.LastName := 'Doe';
  Person.Age := 30;

  WriteLn('Name: ', Person.FirstName, ' ', Person.LastName);
  WriteLn('Age: ', Person.Age);
end;

En este ejemplo:

  • Se crea una variable Person del tipo TPerson.
  • Los campos FirstName, LastName y Age se les asignan valores.
  • Los valores se imprimen en la consola.

Características Avanzadas de los Registros

Métodos en Registros

Los registros en Delphi/Object Pascal también pueden contener métodos, que son funciones o procedimientos que operan sobre los datos del registro.

Ejemplo

type
  TPerson = record
    FirstName: string;
    LastName: string;
    Age: Integer;
    procedure Display;
  end;

procedure TPerson.Display;
begin
  WriteLn('Name: ', FirstName, ' ', LastName);
  WriteLn('Age: ', Age);
end;

var
  Person: TPerson;
begin
  Person.FirstName := 'Jane';
  Person.LastName := 'Smith';
  Person.Age := 25;
  Person.Display;
end;

En este ejemplo:

  • Se añade un método Display al registro TPerson.
  • El método Display imprime los campos del registro en la consola.

Registros con Constructores

Los registros también pueden tener constructores, que son métodos especiales usados para inicializar los campos del registro.

Ejemplo

type
  TPerson = record
    FirstName: string;
    LastName: string;
    Age: Integer;
    constructor Create(AFirstName, ALastName: string; AAge: Integer);
  end;

constructor TPerson.Create(AFirstName, ALastName: string; AAge: Integer);
begin
  FirstName := AFirstName;
  LastName := ALastName;
  Age := AAge;
end;

var
  Person: TPerson;
begin
  Person := TPerson.Create('Alice', 'Johnson', 28);
  Person.Display;
end;

En este ejemplo:

  • Se define un constructor Create para inicializar los campos de TPerson.
  • Se usa el constructor Create para crear e inicializar un registro Person.

Comparación con Clases

Aunque los registros son útiles para agrupar datos relacionados, tienen algunas limitaciones en comparación con las clases. Aquí hay una comparación:

| Característica | Registros | Clases | |------------------|---------------------------------|----------------------------------| | Asignación de Memoria | Pila (usualmente) | Montón | | Herencia | No | Sí | | Métodos | Sí | Sí | | Constructores | Sí | Sí | | Destructores | No | Sí |

Conclusión

Los registros en Delphi/Object Pascal son una forma poderosa de agrupar datos relacionados. Pueden contener campos de diferentes tipos, métodos y constructores. Aunque no son tan ricos en características como las clases, son más simples y pueden ser más eficientes para ciertas tareas. Entender cuándo y cómo usar registros te ayudará a escribir código más organizado y mantenible.
