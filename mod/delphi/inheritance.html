Herencia en Delphi/Object Pascal
Polimorfismo ►

La herencia es un concepto fundamental en la Programación Orientada a Objetos (OOP) que permite a una clase heredar propiedades y métodos de otra clase. Esto ayuda en la reutilización del código y en el establecimiento de una relación jerárquica entre las clases.

Introducción a la Herencia

  • Definición: La herencia permite que una nueva clase (clase derivada) herede atributos y métodos de una clase existente (clase base).
  • Propósito: Promueve la reutilización del código, reduce la redundancia y establece una jerarquía natural.

Conceptos Básicos

Clase Base y Clase Derivada

  • Clase Base: La clase cuyas propiedades y métodos son heredados.
  • Clase Derivada: La clase que hereda de la clase base.

Sintaxis

En Delphi/Object Pascal, la herencia se implementa usando la palabra clave class seguida de la clase base TObject o cualquier otra clase definida por el usuario.

type
  TBaseClass = class
    procedure ShowMessage; virtual;
  end;

  TDerivedClass = class(TBaseClass)
    procedure ShowMessage; override;
  end;

Métodos Virtuales y Override

  • Método Virtual: Un método en la clase base que puede ser sobrescrito en una clase derivada.
  • Método Override: Un método en la clase derivada que sobrescribe un método virtual en la clase base.
procedure TBaseClass.ShowMessage;
begin
  Writeln('Mensaje de la Clase Base');
end;

procedure TDerivedClass.ShowMessage;
begin
  Writeln('Mensaje de la Clase Derivada');
end;

Implementación de la Herencia

Creando una Clase Base

Definir una clase base con algunas propiedades y métodos.

type
  TAnimal = class
  public
    procedure Speak; virtual;
  end;

procedure TAnimal.Speak;
begin
  Writeln('El animal hace un sonido');
end;

Creando una Clase Derivada

Definir una clase derivada que herede de la clase base y sobrescriba sus métodos.

type
  TDog = class(TAnimal)
  public
    procedure Speak; override;
  end;

procedure TDog.Speak;
begin
  Writeln('El perro ladra');
end;

Usando las Clases

Instanciar las clases y llamar a sus métodos para ver la herencia en acción.

var
  Animal: TAnimal;
  Dog: TDog;
begin
  Animal := TAnimal.Create;
  Dog := TDog.Create;

  Animal.Speak; // Salida: El animal hace un sonido
  Dog.Speak;    // Salida: El perro ladra

  Animal.Free;
  Dog.Free;
end;

Conceptos Avanzados

Clases y Métodos Abstractos

  • Clase Abstracta: Una clase que no puede ser instanciada y está destinada a ser una clase base para otras clases.
  • Método Abstracto: Un método que debe ser sobrescrito en una clase derivada.
type
  TShape = class
  public
    procedure Draw; virtual; abstract;
  end;

  TCircle = class(TShape)
  public
    procedure Draw; override;
  end;

procedure TCircle.Draw;
begin
  Writeln('Dibujando un círculo');
end;

Herencia y Constructores

Los constructores en las clases derivadas deben llamar al constructor de la clase base para asegurar una inicialización adecuada.

type
  TPerson = class
  private
    FName: string;
  public
    constructor Create(AName: string); virtual;
  end;

  TEmployee = class(TPerson)
  private
    FEmployeeID: Integer;
  public
    constructor Create(AName: string; AEmployeeID: Integer); reintroduce;
  end;

constructor TPerson.Create(AName: string);
begin
  FName := AName;
end;

constructor TEmployee.Create(AName: string; AEmployeeID: Integer);
begin
  inherited Create(AName);
  FEmployeeID := AEmployeeID;
end;

Conclusión

La herencia en Delphi/Object Pascal es una característica poderosa que promueve la reutilización del código y establece una jerarquía clara entre las clases. Al comprender y utilizar la herencia, puedes crear código más organizado y mantenible. Recuerda usar métodos virtuales y override para personalizar el comportamiento en las clases derivadas y considera las clases y métodos abstractos para diseños más avanzados.
