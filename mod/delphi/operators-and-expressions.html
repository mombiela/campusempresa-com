Introducción

En Delphi/Object Pascal, los operadores y las expresiones son conceptos fundamentales que te permiten realizar cálculos, tomar decisiones y manipular datos. Esta sección cubrirá los diversos tipos de operadores disponibles en Delphi/Object Pascal, cómo usarlos y cómo formar expresiones.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

  • Adición (+): Suma dos operandos.
  • Sustracción (-): Resta el segundo operando del primero.
  • Multiplicación (*): Multiplica dos operandos.
  • División (/): Divide el primer operando por el segundo, resultando en un número de punto flotante.
  • División Entera (div): Divide el primer operando por el segundo, resultando en un entero.
  • Módulo (mod): Devuelve el resto de la división del primer operando por el segundo.

Ejemplo

var
  a, b: Integer;
  c: Real;
begin
  a := 10;
  b := 3;
  c := a / b;  // c será 3.3333
  a := a div b;  // a será 3
  b := a mod b;  // b será 1
end;

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores.

  • Igual a (=): Verifica si dos operandos son iguales.
  • No igual a (<>): Verifica si dos operandos no son iguales.
  • Mayor que (>): Verifica si el primer operando es mayor que el segundo.
  • Menor que (<): Verifica si el primer operando es menor que el segundo.
  • Mayor o igual a (>=): Verifica si el primer operando es mayor o igual al segundo.
  • Menor o igual a (<=): Verifica si el primer operando es menor o igual al segundo.

Ejemplo

var
  a, b: Integer;
  result: Boolean;
begin
  a := 10;
  b := 5;
  result := a > b;  // result será True
  result := a = b;  // result será False
end;

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

  • AND (and): Devuelve True si ambos operandos son True.
  • OR (or): Devuelve True si al menos uno de los operandos es True.
  • NOT (not): Invierte el estado lógico de su operando.

Ejemplo

var
  a, b: Boolean;
  result: Boolean;
begin
  a := True;
  b := False;
  result := a and b;  // result será False
  result := a or b;   // result será True
  result := not a;    // result será False
end;

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación Simple (:=): Asigna el valor a la derecha a la variable a la izquierda.
  • Asignación de Adición (+=): Suma el valor a la derecha a la variable a la izquierda y asigna el resultado a la variable a la izquierda.
  • Asignación de Sustracción (-=): Resta el valor a la derecha de la variable a la izquierda y asigna el resultado a la variable a la izquierda.
  • Asignación de Multiplicación (*=): Multiplica la variable a la izquierda por el valor a la derecha y asigna el resultado a la variable a la izquierda.
  • Asignación de División (/=): Divide la variable a la izquierda por el valor a la derecha y asigna el resultado a la variable a la izquierda.

Ejemplo

var
  a: Integer;
begin
  a := 10;
  a += 5;  // a será 15
  a -= 3;  // a será 12
  a *= 2;  // a será 24
  a /= 4;  // a será 6
end;

Operadores a Nivel de Bit

Los operadores a nivel de bit se utilizan para realizar operaciones en las representaciones binarias de los enteros.

  • AND (and): Realiza una operación AND a nivel de bit.
  • OR (or): Realiza una operación OR a nivel de bit.
  • XOR (xor): Realiza una operación XOR a nivel de bit.
  • NOT (not): Realiza una operación NOT a nivel de bit.
  • Desplazamiento a la Izquierda (shl): Desplaza los bits a la izquierda.
  • Desplazamiento a la Derecha (shr): Desplaza los bits a la derecha.

Ejemplo

var
  a, b: Integer;
begin
  a := 5;  // binario: 0101
  b := 3;  // binario: 0011
  a := a and b;  // a será 1 (binario: 0001)
  a := a or b;   // a será 3 (binario: 0011)
  a := a xor b;  // a será 0 (binario: 0000)
  a := not a;    // a será -1 (binario: 1111...1111)
  a := b shl 1;  // a será 6 (binario: 0110)
  a := b shr 1;  // a será 1 (binario: 0001)
end;

Expresiones

Las expresiones son combinaciones de variables, constantes, operadores y llamadas a funciones que se evalúan para producir un valor.

Ejemplo

var
  a, b, c: Integer;
  result: Boolean;
begin
  a := 10;
  b := 5;
  c := 2;
  result := (a > b) and (b > c);  // result será True
end;

Conclusión

Entender los operadores y las expresiones es crucial para escribir código efectivo y eficiente en Delphi/Object Pascal. Al dominar los operadores aritméticos, relacionales, lógicos, de asignación y a nivel de bit, puedes realizar una amplia gama de operaciones y crear expresiones complejas para resolver diversos problemas de programación. Practica el uso de estos operadores en diferentes escenarios para volverte competente en su uso.
