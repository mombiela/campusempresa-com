Tipos Enumerados en Delphi/Object Pascal
Cadenas y Manejo de Cadenas ►

Los tipos enumerados, o enums, son una característica poderosa en Delphi/Object Pascal que te permiten definir un conjunto de valores nombrados. Estos valores son constantes y pueden ser utilizados para hacer tu código más legible y mantenible. Este tema te guiará desde lo básico hasta el uso avanzado de tipos enumerados en Delphi/Object Pascal.

Introducción a los Tipos Enumerados

Los tipos enumerados son tipos de datos definidos por el usuario que consisten en un conjunto de valores nombrados llamados elementos. Cada elemento en un tipo enumerado está asociado con un valor entero, comenzando desde 0 por defecto.

Definiendo Tipos Enumerados

Para definir un tipo enumerado, utilizas la palabra clave type seguida del nombre del tipo y la lista de valores posibles encerrados entre paréntesis.

type
  TColor = (Red, Green, Blue);

En este ejemplo, TColor es un tipo enumerado con tres valores posibles: Red, Green, y Blue.

Usando Tipos Enumerados

Una vez que has definido un tipo enumerado, puedes declarar variables de ese tipo y asignarles cualquiera de los valores definidos en la enumeración.

var
  MyColor: TColor;
begin
  MyColor := Red;
  if MyColor = Green then
    WriteLn('El color es verde.')
  else
    WriteLn('El color no es verde.');
end;

Tipos Enumerados Avanzados

Asignando Valores Específicos

Por defecto, el primer elemento en un tipo enumerado se le asigna el valor 0, al segundo elemento el valor 1, y así sucesivamente. Sin embargo, puedes asignar valores enteros específicos a los elementos.

type
  TStatus = (Pending = 1, Approved = 5, Rejected = 10);

En este ejemplo, Pending se le asigna el valor 1, Approved el valor 5, y Rejected el valor 10.

Rangos de Tipos Enumerados

Puedes usar tipos enumerados para definir rangos, lo cual puede ser útil en bucles e indexación de arrays.

type
  TDay = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);

var
  Day: TDay;
begin
  for Day := Mon to Fri do
    WriteLn(Day);
end;

Tipos Enumerados y Conjuntos

Los tipos enumerados pueden ser utilizados para definir conjuntos, que son colecciones de valores del tipo enumerado.

type
  TDays = set of TDay;

var
  Weekend: TDays;
begin
  Weekend := [Sat, Sun];
  if Sun in Weekend then
    WriteLn('El domingo es un día de fin de semana.');
end;

Funciones de Tipos Enumerados

Delphi proporciona varias funciones integradas para trabajar con tipos enumerados, tales como Ord, Pred, Succ, y High.

  • Ord: Devuelve el valor ordinal de un elemento del tipo enumerado.
  • Pred: Devuelve el predecesor de un elemento del tipo enumerado.
  • Succ: Devuelve el sucesor de un elemento del tipo enumerado.
  • High: Devuelve el valor más alto en un tipo enumerado.
begin
  WriteLn(Ord(Mon));  // Salida: 0
  WriteLn(Pred(Wed)); // Salida: Tue
  WriteLn(Succ(Tue)); // Salida: Wed
  WriteLn(High(TDay)); // Salida: Sun
end;

Conclusión

Los tipos enumerados en Delphi/Object Pascal proporcionan una manera de definir un conjunto de constantes nombradas, haciendo tu código más legible y mantenible. Puedes asignar valores específicos a los elementos, usarlos en rangos y conjuntos, y aprovechar las funciones integradas para manipularlos. Entender y utilizar los tipos enumerados de manera efectiva puede mejorar en gran medida la claridad y funcionalidad de tus programas en Delphi/Object Pascal.
