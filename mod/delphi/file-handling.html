Manejo de Archivos en Delphi/Object Pascal
Acceso a Bases de Datos con Delphi ►

El manejo de archivos es una habilidad esencial para cualquier programador, ya que te permite leer y escribir en archivos, habilitando la persistencia y manipulación de datos. Este tema te guiará desde los conceptos básicos hasta los avanzados del manejo de archivos en Delphi/Object Pascal.

Introducción al Manejo de Archivos

El manejo de archivos en Delphi implica el uso de varias clases y procedimientos para crear, leer, escribir y manipular archivos. Delphi proporciona un conjunto rico de funciones y clases en las unidades System.SysUtils y System.Classes para trabajar con archivos.

Conceptos Clave

  • Tipos de Archivos: Archivos de texto, archivos binarios.
  • Operaciones de Archivos: Crear, abrir, leer, escribir, cerrar y eliminar archivos.
  • Modos de Archivo: Leer, escribir, anexar.

Operaciones Básicas de Archivos

Crear y Escribir en un Archivo de Texto

Para crear y escribir en un archivo de texto, puedes usar el tipo TextFile y los procedimientos asociados.

program WriteToFile;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  AssignFile(MyFile, 'example.txt');
  Rewrite(MyFile);  // Crear el archivo para escribir
  
  Text := 'Hello, Delphi!';
  Writeln(MyFile, Text);  // Escribir texto en el archivo
  
  CloseFile(MyFile);  // Cerrar el archivo
end.

Leer de un Archivo de Texto

Leer de un archivo de texto implica abrir el archivo en modo de lectura y usar Readln para leer líneas.

program ReadFromFile;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  AssignFile(MyFile, 'example.txt');
  Reset(MyFile);  // Abrir el archivo para leer
  
  while not Eof(MyFile) do
  begin
    Readln(MyFile, Text);  // Leer una línea del archivo
    Writeln(Text);  // Salida de la línea a la consola
  end;
  
  CloseFile(MyFile);  // Cerrar el archivo
end.

Manejo Avanzado de Archivos

Usar TFileStream para Archivos Binarios

Para operaciones de archivo más avanzadas, como el manejo de archivos binarios, puedes usar la clase TFileStream.

Escribir en un Archivo Binario

program WriteBinaryFile;

uses
  SysUtils, Classes;

var
  FileStream: TFileStream;
  Data: array[1..5] of Byte;
  I: Integer;
begin
  FileStream := TFileStream.Create('binary.dat', fmCreate);
  try
    for I := 1 to 5 do
      Data[I] := I * 10;  // Inicializar el arreglo de datos
    
    FileStream.WriteBuffer(Data, SizeOf(Data));  // Escribir datos en el archivo
  finally
    FileStream.Free;
  end;
end.

Leer de un Archivo Binario

program ReadBinaryFile;

uses
  SysUtils, Classes;

var
  FileStream: TFileStream;
  Data: array[1..5] of Byte;
  I: Integer;
begin
  FileStream := TFileStream.Create('binary.dat', fmOpenRead);
  try
    FileStream.ReadBuffer(Data, SizeOf(Data));  // Leer datos del archivo
    
    for I := 1 to 5 do
      Writeln('Data[', I, '] = ', Data[I]);  // Salida de los datos
  finally
    FileStream.Free;
  end;
end.

Manejo de Excepciones

Las operaciones de archivo pueden fallar por varias razones, como archivos faltantes o problemas de permisos. Usa bloques try..except para manejar excepciones de manera elegante.

program SafeFileHandling;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  try
    AssignFile(MyFile, 'example.txt');
    Reset(MyFile);  // Intentar abrir el archivo para leer
    
    while not Eof(MyFile) do
    begin
      Readln(MyFile, Text);  // Leer una línea del archivo
      Writeln(Text);  // Salida de la línea a la consola
    end;
    
    CloseFile(MyFile);  // Cerrar el archivo
  except
    on E: EInOutError do
      Writeln('Ocurrió un error de manejo de archivos: ', E.Message);
  end;
end.

Resumen

En este tema, cubrimos los conceptos básicos y avanzados del manejo de archivos en Delphi/Object Pascal. Aprendiste cómo:

  • Crear, escribir y leer archivos de texto usando TextFile.
  • Manejar archivos binarios usando TFileStream.
  • Gestionar excepciones durante las operaciones de archivo.

El manejo de archivos es una habilidad fundamental que te permitirá construir aplicaciones más robustas y ricas en funcionalidades. Practica estos conceptos para volverte competente en la gestión de archivos en Delphi.
