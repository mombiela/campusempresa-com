Introducción a Delphi/Object Pascal

Delphi, también conocido como Object Pascal, es un lenguaje de alto nivel, compilado y fuertemente tipado que soporta programación estructurada y orientada a objetos. Es conocido por su legibilidad y facilidad de uso, lo que lo convierte en una excelente opción tanto para principiantes como para programadores experimentados.

Conceptos Clave

  • Sensibilidad a Mayúsculas y Minúsculas: Delphi no distingue entre mayúsculas y minúsculas.
  • Comentarios: Los comentarios de una sola línea usan //, y los comentarios de varias líneas usan {} o (* *).
  • Identificadores: Nombres para variables, tipos, funciones, etc., que deben comenzar con una letra o guion bajo y pueden ser seguidos por letras, dígitos o guiones bajos.
  • Palabras Reservadas: Palabras clave que tienen un significado especial en Delphi y no pueden ser usadas como identificadores.

Estructura Básica del Programa

Un programa simple en Delphi consta de las siguientes partes:

  1. Encabezado del Programa: Declara el nombre del programa.
  2. Cláusula Uses: Lista las unidades que el programa utiliza.
  3. Declaraciones de Variables: Declara variables.
  4. Bloque Begin-End: Contiene el código principal.

Ejemplo

program HelloWorld;

uses
  SysUtils;

begin
  WriteLn('¡Hola, Mundo!');
end.

Explicación

  • program HelloWorld;: Declara el nombre del programa.
  • uses SysUtils;: Incluye la unidad SysUtils, que proporciona varias funciones de utilidad.
  • begin ... end.: El bloque principal de código donde la ejecución del programa comienza y termina.
  • WriteLn('¡Hola, Mundo!');: Muestra el texto "¡Hola, Mundo!" en la consola.

Variables y Tipos de Datos

Declaración de Variables

Las variables en Delphi se declaran usando la palabra clave var seguida del nombre de la variable y el tipo.

var
  age: Integer;
  name: String;

Tipos de Datos Comunes

| Tipo de Dato | Descripción | |--------------|---------------------------------------| | Integer | Números enteros | | Real | Números de punto flotante | | Char | Carácter único | | String | Secuencia de caracteres | | Boolean | Valores Verdadero o Falso |

Ejemplo

var
  age: Integer;
  price: Real;
  initial: Char;
  name: String;
  isStudent: Boolean;
begin
  age := 25;
  price := 19.99;
  initial := 'A';
  name := 'Alice';
  isStudent := True;
end.

Explicación

  • age := 25;: Asigna el valor 25 a la variable age.
  • price := 19.99;: Asigna el valor 19.99 a la variable price.
  • initial := 'A';: Asigna el carácter 'A' a la variable initial.
  • name := 'Alice';: Asigna la cadena "Alice" a la variable name.
  • isStudent := True;: Asigna el valor booleano True a la variable isStudent.

Estructuras de Control

Sentencias Condicionales

Delphi soporta las sentencias if, if-else y case para la ejecución condicional.

Ejemplo: if-else

var
  age: Integer;
begin
  age := 18;
  if age >= 18 then
    WriteLn('Adulto')
  else
    WriteLn('Menor');
end.

Sentencias de Bucle

Delphi soporta los bucles for, while y repeat-until.

Ejemplo: bucle for

var
  i: Integer;
begin
  for i := 1 to 10 do
    WriteLn('Número: ', i);
end.

Explicación

  • for i := 1 to 10 do: Itera de 1 a 10.
  • WriteLn('Número: ', i);: Muestra el valor actual de i.

Procedimientos y Funciones

Procedimientos

Los procedimientos son bloques de código que realizan una tarea pero no devuelven un valor.

procedure Greet;
begin
  WriteLn('¡Hola!');
end;

begin
  Greet;
end.

Funciones

Las funciones son similares a los procedimientos pero devuelven un valor.

function Add(a, b: Integer): Integer;
begin
  Add := a + b;
end;

begin
  WriteLn('Suma: ', Add(5, 3));
end.

Explicación

  • procedure Greet;: Declara un procedimiento llamado Greet.
  • function Add(a, b: Integer): Integer;: Declara una función llamada Add que toma dos enteros y devuelve un entero.
  • Add := a + b;: Devuelve la suma de a y b.

Conclusión

Entender la sintaxis y estructura básica de Delphi/Object Pascal es crucial para escribir programas efectivos. Esto incluye saber cómo declarar variables, usar estructuras de control y definir procedimientos y funciones. Con estos fundamentos, puedes comenzar a construir aplicaciones más complejas y explorar características avanzadas del lenguaje.
