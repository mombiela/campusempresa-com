Introducción a STL
Contenedores ►

La Biblioteca de Plantillas Estándar (STL) es un conjunto poderoso de clases plantilla de C++ que proporciona clases y funciones de propósito general con plantillas que implementan muchos algoritmos y estructuras de datos populares y comúnmente utilizados, como vectores, listas, colas y pilas. Esta sección te introducirá a la STL, sus componentes y cómo usarlos de manera efectiva.

¿Qué es STL?

La STL es una colección de clases plantilla de C++ que proporcionan estructuras de datos y algoritmos. Incluye:

  • Contenedores: Objetos que almacenan datos.
  • Algoritmos: Procedimientos para procesar datos.
  • Iteradores: Objetos que apuntan a elementos dentro de los contenedores.

¿Por qué usar STL?

  • Eficiencia: STL está optimizada para el rendimiento.
  • Reusabilidad: Proporciona componentes de código reutilizables.
  • Consistencia: Interfaz estandarizada a través de diferentes contenedores y algoritmos.

Componentes clave de STL

Contenedores

Los contenedores se utilizan para gestionar colecciones de objetos. Los contenedores más comúnmente utilizados son:

  • Contenedores de Secuencia: Almacenan datos de manera lineal.
    • vector
    • deque
    • list
  • Contenedores Asociativos: Almacenan datos en un orden clasificado.
    • set
    • map
    • multiset
    • multimap
  • Contenedores Desordenados: Almacenan datos de manera desordenada.
    • unordered_set
    • unordered_map
    • unordered_multiset
    • unordered_multimap

Iteradores

Los iteradores se utilizan para apuntar a los elementos de un contenedor. Son similares a los punteros y se pueden usar para recorrer los elementos de un contenedor.

  • Iteradores de Entrada: Leer desde el contenedor.
  • Iteradores de Salida: Escribir en el contenedor.
  • Iteradores de Avance: Leer y escribir, pueden avanzar.
  • Iteradores Bidireccionales: Leer y escribir, pueden avanzar y retroceder.
  • Iteradores de Acceso Aleatorio: Leer y escribir, pueden moverse a cualquier elemento directamente.

Algoritmos

Los algoritmos son un conjunto de procedimientos para procesar los elementos de los contenedores. Algunos algoritmos comunes incluyen:

  • Búsqueda: find, binary_search
  • Ordenación: sort, stable_sort
  • Modificación: copy, swap
  • Operaciones de Conjunto: set_union, set_intersection

Ejemplos

Uso de un Vector

Los vectores son arreglos dinámicos que pueden cambiar de tamaño.

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers;

    // Añadiendo elementos al vector
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    // Accediendo a los elementos
    for (int i = 0; i < numbers.size(); ++i) {
        std::cout << numbers[i] << " ";
    }
    std::cout << std::endl;

    // Usando un iterador
    for (std::vector<int>::iterator it = numbers.begin(); it != numbers.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    return 0;
}

Uso de un Map

Los mapas almacenan pares clave-valor y permiten una recuperación rápida basada en claves.

#include <iostream>
#include <map>

int main() {
    std::map<std::string, int> ageMap;

    // Añadiendo elementos al mapa
    ageMap["Alice"] = 30;
    ageMap["Bob"] = 25;
    ageMap["Charlie"] = 35;

    // Accediendo a los elementos
    std::cout << "Edad de Alice: " << ageMap["Alice"] << std::endl;

    // Usando un iterador
    for (std::map<std::string, int>::iterator it = ageMap.begin(); it != ageMap.end(); ++it) {
        std::cout << it->first << ": " << it->second << std::endl;
    }

    return 0;
}

Conclusión

La STL es una parte esencial de la programación moderna en C++, proporcionando un conjunto rico de herramientas para gestionar datos y realizar operaciones comunes de manera eficiente. Al dominar los contenedores, iteradores y algoritmos de STL, puedes escribir código más eficiente, legible y mantenible. Practica usando diferentes componentes de la STL para volverte competente en aprovechar todo su potencial.
