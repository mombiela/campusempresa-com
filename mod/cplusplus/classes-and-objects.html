Introducción a Clases y Objetos

En C++, las clases y los objetos son conceptos fundamentales de la Programación Orientada a Objetos (OOP). Ayudan a organizar el código y a crear componentes reutilizables.

Conceptos Clave

  • Clase: Un plano para crear objetos. Define un tipo de dato agrupando datos y métodos que trabajan sobre los datos.
  • Objeto: Una instancia de una clase. Se crea usando la definición de la clase.

Ejemplo

#include <iostream>
using namespace std;

class Car {
public:
    string brand;
    string model;
    int year;

    void displayInfo() {
        cout << "Marca: " << brand << ", Modelo: " << model << ", Año: " << year << endl;
    }
};

int main() {
    Car car1;
    car1.brand = "Toyota";
    car1.model = "Corolla";
    car1.year = 2020;

    car1.displayInfo();

    return 0;
}

En este ejemplo:

  • Car es una clase con tres atributos públicos: brand, model y year.
  • displayInfo es un método que imprime los detalles del coche.
  • car1 es un objeto de la clase Car.

Definiendo Clases

Una clase se define usando la palabra clave class seguida del nombre de la clase y un par de llaves {}.

Sintaxis

class ClassName {
public:
    // Miembros públicos
private:
    // Miembros privados
protected:
    // Miembros protegidos
};

Especificadores de Acceso

  • public: Los miembros son accesibles desde fuera de la clase.
  • private: Los miembros son accesibles solo dentro de la clase.
  • protected: Los miembros son accesibles dentro de la clase y por clases derivadas.

Ejemplo

class Person {
public:
    string name;
    int age;

    void introduce() {
        cout << "Hola, soy " << name << " y tengo " << age << " años." << endl;
    }
};

Creando Objetos

Los objetos son instancias de una clase. Se crean declarando una variable del tipo de la clase.

Ejemplo

Person person1;
person1.name = "Alice";
person1.age = 30;
person1.introduce();

Constructores y Destructores

Los constructores y destructores son funciones miembro especiales de una clase.

Constructores

  • Constructor: Una función especial que se llama automáticamente cuando se crea un objeto. Inicializa el objeto.
  • Sintaxis: El constructor tiene el mismo nombre que la clase y no tiene tipo de retorno.

Ejemplo

class Person {
public:
    string name;
    int age;

    // Constructor
    Person(string n, int a) {
        name = n;
        age = a;
    }

    void introduce() {
        cout << "Hola, soy " << name << " y tengo " << age << " años." << endl;
    }
};

int main() {
    Person person1("Alice", 30);
    person1.introduce();

    return 0;
}

Destructores

  • Destructor: Una función especial que se llama automáticamente cuando un objeto es destruido. Limpia los recursos.
  • Sintaxis: El destructor tiene el mismo nombre que la clase precedido por una tilde ~ y no tiene tipo de retorno.

Ejemplo

class Person {
public:
    string name;
    int age;

    // Constructor
    Person(string n, int a) {
        name = n;
        age = a;
    }

    // Destructor
    ~Person() {
        cout << "Destructor llamado para " << name << endl;
    }

    void introduce() {
        cout << "Hola, soy " << name << " y tengo " << age << " años." << endl;
    }
};

int main() {
    Person person1("Alice", 30);
    person1.introduce();

    return 0;
}

Temas Avanzados

Herencia

La herencia permite que una clase herede propiedades y métodos de otra clase.

Ejemplo

class Animal {
public:
    void eat() {
        cout << "Este animal está comiendo." << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "El perro está ladrando." << endl;
    }
};

int main() {
    Dog dog1;
    dog1.eat();  // Método heredado
    dog1.bark(); // Método propio del perro

    return 0;
}

Polimorfismo

El polimorfismo permite que los métodos hagan diferentes cosas basándose en el objeto sobre el que actúan.

Ejemplo

class Animal {
public:
    virtual void sound() {
        cout << "Este animal hace un sonido." << endl;
    }
};

class Dog : public Animal {
public:
    void sound() override {
        cout << "El perro ladra." << endl;
    }
};

class Cat : public Animal {
public:
    void sound() override {
        cout << "El gato maúlla." << endl;
    }
};

int main() {
    Animal* animal1 = new Dog();
    Animal* animal2 = new Cat();

    animal1->sound(); // Salida: El perro ladra.
    animal2->sound(); // Salida: El gato maúlla.

    delete animal1;
    delete animal2;

    return 0;
}

Resumen

  • Clases y Objetos: Conceptos fundamentales en la OOP de C++.
  • Clase: Plano para objetos, que contiene datos y métodos.
  • Objeto: Instancia de una clase.
  • Constructores y Destructores: Funciones especiales para inicializar y limpiar objetos.
  • Herencia: Mecanismo para crear una nueva clase a partir de una clase existente.
  • Polimorfismo: Capacidad de llamar al mismo método en diferentes objetos y que cada uno responda a su manera.

Entender estos conceptos es crucial para dominar C++ y escribir código eficiente y reutilizable.
