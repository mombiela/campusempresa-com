Punteros Inteligentes en C++
Expresiones Lambda ►

Los punteros inteligentes son una característica poderosa en C++ que ayudan a gestionar la memoria dinámica automáticamente. Aseguran que la memoria se desaloje correctamente cuando ya no es necesaria, previniendo fugas de memoria y punteros colgantes.

Introducción a los Punteros Inteligentes

  • ¿Qué son los Punteros Inteligentes?

    • Los punteros inteligentes son objetos que actúan como punteros pero proporcionan gestión automática de memoria.
    • Son parte de la Biblioteca Estándar de C++, introducida en C++11.

  • ¿Por qué Usar Punteros Inteligentes?

    • Para evitar fugas de memoria.
    • Para asegurar la seguridad ante excepciones.
    • Para simplificar la gestión de memoria.

Tipos de Punteros Inteligentes

C++ proporciona varios tipos de punteros inteligentes, cada uno con diferentes propósitos:

  1. std::unique_ptr
  2. std::shared_ptr
  3. std::weak_ptr

`std::unique_ptr`

  • Características:

    • Posee un recurso de manera exclusiva.
    • No puede ser copiado, solo movido.
    • Elimina automáticamente el recurso cuando sale del ámbito.

  • Ejemplo de Uso:

#include <iostream>
#include <memory>

void uniquePtrExample() {
    std::unique_ptr<int> ptr1 = std::make_unique<int>(10);
    std::cout << "Valor: " << *ptr1 << std::endl;

    // std::unique_ptr<int> ptr2 = ptr1; // Error: no se puede copiar unique_ptr
    std::unique_ptr<int> ptr2 = std::move(ptr1); // Transferir propiedad

    if (!ptr1) {
        std::cout << "ptr1 ahora es nulo." << std::endl;
    }
    std::cout << "Valor: " << *ptr2 << std::endl;
}

int main() {
    uniquePtrExample();
    return 0;
}

`std::shared_ptr`

  • Características:

    • Propiedad compartida de un recurso.
    • Mecanismo de conteo de referencias para rastrear el número de propietarios.
    • El recurso se elimina cuando el último shared_ptr que lo posee es destruido.

  • Ejemplo de Uso:

#include <iostream>
#include <memory>

void sharedPtrExample() {
    std::shared_ptr<int> ptr1 = std::make_shared<int>(20);
    std::cout << "Valor: " << *ptr1 << std::endl;
    std::cout << "Conteo de uso: " << ptr1.use_count() << std::endl;

    {
        std::shared_ptr<int> ptr2 = ptr1;
        std::cout << "Conteo de uso después de la copia: " << ptr1.use_count() << std::endl;
    }

    std::cout << "Conteo de uso después de que ptr2 sale del ámbito: " << ptr1.use_count() << std::endl;
}

int main() {
    sharedPtrExample();
    return 0;
}

`std::weak_ptr`

  • Características:

    • No posee el recurso.
    • Se usa para romper referencias circulares.
    • Puede convertirse en std::shared_ptr si es necesario.

  • Ejemplo de Uso:

#include <iostream>
#include <memory>

void weakPtrExample() {
    std::shared_ptr<int> sharedPtr = std::make_shared<int>(30);
    std::weak_ptr<int> weakPtr = sharedPtr;

    std::cout << "Conteo de uso: " << sharedPtr.use_count() << std::endl;

    if (auto tempPtr = weakPtr.lock()) {
        std::cout << "Valor: " << *tempPtr << std::endl;
    } else {
        std::cout << "El recurso ya no existe." << std::endl;
    }
}

int main() {
    weakPtrExample();
    return 0;
}

Comparación de Punteros Inteligentes

| Característica | std::unique_ptr | std::shared_ptr | std::weak_ptr | |-----------------------|-------------------|-------------------|-----------------| | Propiedad | Exclusiva | Compartida | No posee | | Copiable | No | Sí | No | | Movible | Sí | Sí | No | | Conteo de Referencias | No | Sí | Sí | | Caso de Uso | Propiedad única | Propiedad compartida | Romper ciclos |

Conclusión

Los punteros inteligentes en C++ proporcionan una forma robusta de gestionar la memoria dinámica, reduciendo el riesgo de fugas de memoria y punteros colgantes. Al entender y usar std::unique_ptr, std::shared_ptr y std::weak_ptr, puedes escribir código C++ más seguro y eficiente. Siempre elige el puntero inteligente apropiado basado en tus semánticas de propiedad y requisitos del caso de uso.
