Manejo de Cadenas en C++
Punteros y Arreglos ►

Introducción

El manejo de cadenas es un aspecto fundamental de la programación en C++. Este tema cubrirá los conceptos básicos de las cadenas, varias operaciones que se pueden realizar en ellas y técnicas avanzadas de manejo de cadenas. Al final de esta sección, tendrás una comprensión sólida de cómo manipular y utilizar cadenas en tus programas en C++.

Conceptos Básicos de las Cadenas

En C++, las cadenas se pueden manejar utilizando dos métodos principales: cadenas estilo C y la clase std::string de la Biblioteca Estándar.

Cadenas Estilo C

  • Las cadenas estilo C son arreglos de caracteres terminados por un carácter nulo ('\0').
  • Se definen utilizando arreglos de caracteres.
#include <iostream>

int main() {
    char greeting[] = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

Clase `std::string`

  • La clase std::string proporciona una forma más flexible y poderosa de manejar cadenas.
  • Es parte de la Biblioteca Estándar de C++ y requiere la inclusión del encabezado <string>.
#include <iostream>
#include <string>

int main() {
    std::string greeting = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

Operaciones con Cadenas

Concatenación

  • La concatenación es el proceso de unir dos cadenas.

Usando Cadenas Estilo C

#include <iostream>
#include <cstring>

int main() {
    char str1[20] = "Hello, ";
    char str2[] = "World!";
    strcat(str1, str2); // Concatenar str2 a str1
    std::cout << str1 << std::endl;
    return 0;
}

Usando std::string

#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello, ";
    std::string str2 = "World!";
    std::string result = str1 + str2; // Concatenar str1 y str2
    std::cout << result << std::endl;
    return 0;
}

Longitud de una Cadena

  • Determinar la longitud de una cadena es una operación común.

Usando Cadenas Estilo C

#include <iostream>
#include <cstring>

int main() {
    char str[] = "Hello, World!";
    std::cout << "Longitud de la cadena: " << strlen(str) << std::endl;
    return 0;
}

Usando std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::cout << "Longitud de la cadena: " << str.length() << std::endl;
    return 0;
}

Acceso a Caracteres

  • Los caracteres individuales en una cadena se pueden acceder utilizando el índice de un arreglo.

Usando Cadenas Estilo C

#include <iostream>

int main() {
    char str[] = "Hello";
    std::cout << "Primer carácter: " << str[0] << std::endl;
    return 0;
}

Usando std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello";
    std::cout << "Primer carácter: " << str[0] << std::endl;
    return 0;
}

Subcadenas

  • Extraer una subcadena de una cadena es un requisito común.

Usando std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::string substr = str.substr(7, 5); // Extrae "World"
    std::cout << "Subcadena: " << substr << std::endl;
    return 0;
}

Manejo Avanzado de Cadenas

Flujos de Cadenas

  • Los flujos de cadenas te permiten realizar operaciones de entrada y salida en cadenas.
#include <iostream>
#include <sstream>
#include <string>

int main() {
    std::string str = "123 456 789";
    std::istringstream iss(str);
    int a, b, c;
    iss >> a >> b >> c;
    std::cout << "Números extraídos: " << a << ", " << b << ", " << c << std::endl;
    return 0;
}

Expresiones Regulares

  • Las expresiones regulares proporcionan una forma poderosa de buscar y manipular cadenas.
#include <iostream>
#include <regex>
#include <string>

int main() {
    std::string str = "The quick brown fox jumps over the lazy dog.";
    std::regex pattern("(\\w+)"); // Coincide con palabras
    std::sregex_iterator begin(str.begin(), str.end(), pattern);
    std::sregex_iterator end;
    for (std::sregex_iterator i = begin; i != end; ++i) {
        std::smatch match = *i;
        std::cout << match.str() << std::endl;
    }
    return 0;
}

Conclusión

El manejo de cadenas en C++ abarca desde operaciones básicas con cadenas estilo C hasta manipulaciones más avanzadas utilizando la clase std::string y otras utilidades como flujos de cadenas y expresiones regulares. Comprender estos conceptos es crucial para un procesamiento y manipulación de texto efectivos en tus programas. Practica estas técnicas para volverte competente en el manejo de cadenas en C++.
