Herencia en C++
Polimorfismo ►

La herencia es un concepto fundamental en la programación orientada a objetos (OOP) que permite a una clase heredar propiedades y comportamientos (métodos) de otra clase. Esto promueve la reutilización del código y establece una jerarquía natural entre las clases.

¿Qué es la Herencia?

La herencia permite que una nueva clase (clase derivada) herede atributos y métodos de una clase existente (clase base). Esto permite que la clase derivada reutilice el código de la clase base y extienda o modifique su comportamiento.

Conceptos Clave

  • Clase Base: La clase cuyas propiedades y métodos son heredados.
  • Clase Derivada: La clase que hereda propiedades y métodos de la clase base.
  • Especificadores de Acceso: Palabras clave que definen la accesibilidad de los miembros de la clase base en la clase derivada (public, protected, private).
  • Sobrescritura: Redefinir un método de la clase base en la clase derivada.
  • Polimorfismo: La capacidad de llamar a métodos de la clase derivada a través de una referencia de la clase base.

Sintaxis Básica

Definiendo una Clase Base

class Base {
public:
    int baseValue;
    void display() {
        std::cout << "Función display de la clase base" << std::endl;
    }
};

Definiendo una Clase Derivada

class Derived : public Base {
public:
    int derivedValue;
    void show() {
        std::cout << "Función show de la clase derivada" << std::endl;
    }
};

Creando Objetos y Accediendo a Miembros

int main() {
    Derived obj;
    obj.baseValue = 10; // Accediendo al miembro de la clase base
    obj.derivedValue = 20; // Accediendo al miembro de la clase derivada
    obj.display(); // Llamando al método de la clase base
    obj.show(); // Llamando al método de la clase derivada
    return 0;
}

Especificadores de Acceso en la Herencia

Los especificadores de acceso controlan la accesibilidad de los miembros de la clase base en la clase derivada.

| Especificador de Acceso | Acceso a Miembros de la Clase Base en la Clase Derivada | Acceso Externo | |-------------------------|---------------------------------------------------------|----------------| | public | Público | Público | | protected | Protegido | Privado | | private | Privado | Privado |

Ejemplo con Especificadores de Acceso

class Base {
protected:
    int protectedValue;
public:
    int publicValue;
};

class Derived : public Base {
public:
    void setValues() {
        protectedValue = 10; // Accesible
        publicValue = 20; // Accesible
    }
};

Sobrescritura de Métodos

Una clase derivada puede sobrescribir un método de la clase base para proporcionar una implementación específica.

Ejemplo de Sobrescritura de Métodos

class Base {
public:
    virtual void display() {
        std::cout << "Función display de la clase base" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Función display de la clase derivada" << std::endl;
    }
};

int main() {
    Base* basePtr;
    Derived derivedObj;
    basePtr = &derivedObj;
    basePtr->display(); // Llama a la función display de la clase derivada
    return 0;
}

Polimorfismo

El polimorfismo permite que la misma interfaz sea utilizada para diferentes tipos de datos. En C++, esto se logra a menudo mediante la sobrescritura de funciones y punteros o referencias a tipos de clase base.

Ejemplo de Polimorfismo

class Base {
public:
    virtual void display() {
        std::cout << "Función display de la clase base" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Función display de la clase derivada" << std::endl;
    }
};

void showDisplay(Base* basePtr) {
    basePtr->display();
}

int main() {
    Base baseObj;
    Derived derivedObj;
    showDisplay(&baseObj); // Llama a la función display de la clase base
    showDisplay(&derivedObj); // Llama a la función display de la clase derivada
    return 0;
}

Conclusión

La herencia es una característica poderosa en C++ que promueve la reutilización del código y establece una jerarquía natural entre las clases. Al comprender y utilizar la herencia, puedes crear código más modular, mantenible y escalable. Conceptos clave como los especificadores de acceso, la sobrescritura de métodos y el polimorfismo son esenciales para dominar la herencia en C++. Practica estos conceptos con varios ejemplos para solidificar tu comprensión.
