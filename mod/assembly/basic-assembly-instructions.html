Introducción al Lenguaje de Ensamblado

El lenguaje de ensamblado es un lenguaje de programación de bajo nivel que está estrechamente relacionado con el código máquina. Proporciona una forma de escribir programas que son ejecutados directamente por la CPU. Entender las instrucciones básicas de ensamblado es crucial para escribir código de ensamblado eficiente y efectivo.

Conceptos Clave

  • Arquitectura del Conjunto de Instrucciones (ISA): El conjunto de instrucciones que una CPU en particular puede ejecutar.
  • Registros: Pequeñas y rápidas ubicaciones de almacenamiento dentro de la CPU utilizadas para contener datos y direcciones.
  • Operandos: Los datos o direcciones sobre los que operan las instrucciones.
  • Mnemónicos: Representaciones legibles por humanos de las instrucciones de máquina.

Instrucciones Básicas de Ensamblado

Instrucciones de Movimiento de Datos

Estas instrucciones se utilizan para mover datos entre registros, memoria y puertos de E/S.

  • MOV: Mover datos de una ubicación a otra.

    MOV AX, BX  ; Mueve el contenido del registro BX al registro AX
MOV [1234H], AX  ; Mueve el contenido del registro AX a la ubicación de memoria 1234H

  • PUSH/POP: Empujar datos en la pila y sacar datos de la pila.

    PUSH AX  ; Empuja el contenido de AX en la pila
POP BX   ; Saca la parte superior de la pila en BX

Instrucciones Aritméticas

Estas instrucciones realizan operaciones aritméticas como suma, resta, multiplicación y división.

  • ADD: Sumar dos operandos.

    ADD AX, BX  ; Suma el contenido de BX a AX y almacena el resultado en AX

  • SUB: Restar un operando de otro.

    SUB AX, BX  ; Resta el contenido de BX de AX y almacena el resultado en AX

  • MUL: Multiplicar dos operandos.

    MUL BX  ; Multiplica AX por BX, el resultado se almacena en AX (para operandos de 16 bits)

  • DIV: Dividir un operando por otro.

    DIV BX  ; Divide AX por BX, el cociente en AX, el resto en DX (para operandos de 16 bits)

Instrucciones Lógicas

Estas instrucciones realizan operaciones lógicas a nivel de bits.

  • AND: Realizar una operación AND a nivel de bits.

    AND AX, BX  ; Realiza un AND a nivel de bits en AX y BX, almacena el resultado en AX

  • OR: Realizar una operación OR a nivel de bits.

    OR AX, BX  ; Realiza un OR a nivel de bits en AX y BX, almacena el resultado en AX

  • XOR: Realizar una operación XOR a nivel de bits.

    XOR AX, BX  ; Realiza un XOR a nivel de bits en AX y BX, almacena el resultado en AX

  • NOT: Realizar una operación NOT a nivel de bits.

    NOT AX  ; Realiza un NOT a nivel de bits en AX, almacena el resultado en AX

Instrucciones de Control de Flujo

Estas instrucciones alteran el flujo de ejecución en un programa.

  • JMP: Salto incondicional a una dirección especificada.

    JMP 1234H  ; Salta a la ubicación de memoria 1234H

  • Saltos Condicionales: Salta a una dirección especificada si se cumple una condición.

    JE 1234H  ; Salta a 1234H si el flag de cero (ZF) está establecido (igual)
JNE 1234H ; Salta a 1234H si el flag de cero (ZF) no está establecido (no igual)

  • CALL/RET: Llamar a un procedimiento y retornar de un procedimiento.

    CALL 1234H  ; Llama al procedimiento en la ubicación de memoria 1234H
RET         ; Retorna del procedimiento

Programa de Ejemplo

Aquí hay un programa de ejemplo simple que demuestra algunas de las instrucciones básicas de ensamblado discutidas anteriormente:

section .data
    num1 dw 10
    num2 dw 20
    result dw 0

section .text
    global _start

_start:
    ; Cargar num1 y num2 en registros
    MOV AX, [num1]
    MOV BX, [num2]

    ; Sumar num1 y num2
    ADD AX, BX

    ; Almacenar el resultado
    MOV [result], AX

    ; Salir del programa
    MOV AX, 1
    INT 0x80

Conclusión

Entender las instrucciones básicas de ensamblado es la base para escribir programas de ensamblado efectivos. Al dominar las instrucciones de movimiento de datos, aritméticas, lógicas y de control de flujo, puedes comenzar a escribir código más complejo y eficiente. Practica estas instrucciones con programas simples para construir una comprensión sólida antes de pasar a temas más avanzados.
