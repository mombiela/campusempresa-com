Introducción al Ensamblador x86

  • Descripción general del lenguaje ensamblador
  • Importancia de aprender ensamblador
  • Conceptos básicos: registros, memoria e instrucciones

Ejemplo de código: Hola Mundo en ensamblador x86

section .data
    hello db '¡Hola, Mundo!', 0

section .text
    global _start

_start:
    ; Escribir la cadena en stdout
    mov eax, 4          ; número de syscall para sys_write
    mov ebx, 1          ; descriptor de archivo 1 es stdout
    mov ecx, hello      ; puntero a la cadena
    mov edx, 13         ; longitud de la cadena
    int 0x80            ; llamar al kernel

    ; Salir del programa
    mov eax, 1          ; número de syscall para sys_exit
    xor ebx, ebx        ; código de salida 0
    int 0x80            ; llamar al kernel
  • Explicación:
    • section .data: Define una sección de datos donde se declaran las variables.
    • section .text: Define una sección de código donde reside el código ejecutable.
    • global _start: Punto de entrada para el programa.
    • Instrucción mov: Mueve datos de un lugar a otro.
    • int 0x80: Interrupción para llamar al kernel.

Instrucciones básicas del ensamblador x86

  • Instrucciones de movimiento de datos: mov, push, pop
  • Instrucciones aritméticas: add, sub, mul, div
  • Instrucciones lógicas: and, or, xor, not
  • Instrucciones de control de flujo: jmp, je, jne, call, ret

Ejemplo de código: Operaciones aritméticas básicas

section .text
    global _start

_start:
    mov eax, 5          ; Cargar 5 en eax
    add eax, 3          ; Sumar 3 a eax (eax = 8)
    sub eax, 2          ; Restar 2 de eax (eax = 6)
    ; Salir del programa
    mov eax, 1          ; número de syscall para sys_exit
    xor ebx, ebx        ; código de salida 0
    int 0x80            ; llamar al kernel
  • Explicación:
    • add eax, 3: Suma 3 al valor en el registro eax.
    • sub eax, 2: Resta 2 del valor en el registro eax.

Trabajando con memoria

  • Comprender los modos de direccionamiento: inmediato, directo, indirecto, indexado
  • Uso de punteros y matrices
  • Operaciones de pila: push, pop, call, ret

Ejemplo de código: Uso de punteros

section .data
    num1 dd 10          ; Definir una palabra doble con valor 10
    num2 dd 20          ; Definir una palabra doble con valor 20

section .text
    global _start

_start:
    mov eax, [num1]     ; Cargar el valor de num1 en eax
    add eax, [num2]     ; Sumar el valor de num2 a eax (eax = 30)
    ; Salir del programa
    mov eax, 1          ; número de syscall para sys_exit
    xor ebx, ebx        ; código de salida 0
    int 0x80            ; llamar al kernel
  • Explicación:
    • [num1]: Accede al valor almacenado en la ubicación de memoria etiquetada como num1.
    • [num2]: Accede al valor almacenado en la ubicación de memoria etiquetada como num2.

Control de flujo y procedimientos

  • Saltos condicionales: je, jne, jg, jl
  • Construcciones de bucles: loop, loopz, loopnz
  • Definición y llamada de procedimientos

Ejemplo de código: Bucle simple

section .data
    counter dd 5        ; Definir un contador con valor inicial 5

section .text
    global _start

_start:
    mov ecx, [counter]  ; Cargar el valor del contador en ecx

loop_start:
    dec ecx             ; Decrementar ecx en 1
    cmp ecx, 0          ; Comparar ecx con 0
    jne loop_start      ; Saltar a loop_start si ecx no es cero

    ; Salir del programa
    mov eax, 1          ; número de syscall para sys_exit
    xor ebx, ebx        ; código de salida 0
    int 0x80            ; llamar al kernel
  • Explicación:
    • dec ecx: Decrementa el valor en el registro ecx en 1.
    • cmp ecx, 0: Compara el valor en el registro ecx con 0.
    • jne loop_start: Salta a loop_start si la comparación no es igual.

Temas avanzados

  • Macros y ensamblaje condicional
  • Ensamblaje en línea en lenguajes de alto nivel
  • Optimización del código ensamblador
  • Interfaz con programas en C/C++

Ejemplo de código: Ensamblaje en línea en C

#include <stdio.h>

int main() {
    int a = 5, b = 3, result;

    __asm__ (
        "movl %1, %%eax;"
        "addl %2, %%eax;"
        "movl %%eax, %0;"
        : "=r" (result)
        : "r" (a), "r" (b)
        : "%eax"
    );

    printf("Resultado: %d\n", result);
    return 0;
}
  • Explicación:
    • __asm__: Bloque de ensamblaje en línea en C.
    • "movl %1, %%eax;": Mueve el valor de a al registro eax.
    • "addl %2, %%eax;": Suma el valor de b al registro eax.
    • "movl %%eax, %0;": Mueve el resultado del registro eax a result.
