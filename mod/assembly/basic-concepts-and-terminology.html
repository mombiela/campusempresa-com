Introducción

El lenguaje ensamblador es un lenguaje de programación de bajo nivel que está estrechamente relacionado con el código máquina. Proporciona una forma de escribir programas que son ejecutados directamente por la CPU del ordenador. Entender los conceptos y la terminología básica es crucial para cualquiera que comience con la programación en lenguaje ensamblador.

Conceptos Clave

Código Máquina

  • Definición: El conjunto de instrucciones binarias que la CPU de un ordenador ejecuta directamente.
  • Ejemplo: 10110000 01100001 (Esto podría ser una instrucción en código máquina para mover el valor 97 a un registro).

Lenguaje Ensamblador

  • Definición: Una representación legible por humanos del código máquina usando mnemónicos.
  • Ejemplo: MOV AL, 61h (Esta es una instrucción en ensamblador equivalente al ejemplo de código máquina anterior).

Ensamblador

  • Definición: Una herramienta que convierte el código en lenguaje ensamblador en código máquina.
  • Ejemplo: NASM (Netwide Assembler), MASM (Microsoft Macro Assembler).

Registros

  • Definición: Pequeñas y rápidas ubicaciones de almacenamiento dentro de la CPU utilizadas para mantener datos temporalmente.
  • Registros Comunes:
    • AX, BX, CX, DX: Registros de propósito general.
    • SP: Puntero de Pila.
    • BP: Puntero Base.
    • SI, DI: Índice de Fuente, Índice de Destino.

Mnemónicos

  • Definición: Instrucciones legibles por humanos en lenguaje ensamblador que corresponden a instrucciones en código máquina.
  • Ejemplo: ADD, SUB, MOV, JMP.

Opcode

  • Definición: La parte de una instrucción en lenguaje máquina que especifica la operación a realizar.
  • Ejemplo: En la instrucción MOV AL, 61h, MOV es el mnemónico, y su opcode correspondiente en código máquina podría ser B0.

Operandos

  • Definición: Los datos sobre los que se realiza la operación.
  • Ejemplo: En MOV AL, 61h, AL y 61h son operandos.

Conjunto de Instrucciones

  • Definición: El conjunto completo de instrucciones que una CPU puede ejecutar.
  • Ejemplo: x86, ARM.

Instrucciones Básicas en Ensamblador

Movimiento de Datos

  • MOV: Transfiere datos de una ubicación a otra. 
    MOV AX, BX  ; Mueve el contenido de BX a AX
MOV AL, 61h ; Mueve el valor hexadecimal 61 a AL

Operaciones Aritméticas

  • ADD: Suma dos operandos. 
    ADD AX, BX  ; Suma el contenido de BX a AX
  • SUB: Resta el segundo operando del primero. 
    SUB AX, BX  ; Resta el contenido de BX de AX

Flujo de Control

  • JMP: Salta a una dirección especificada. 
    JMP 0040h   ; Salta a la dirección 0040h

Comparación y Saltos Condicionales

  • CMP: Compara dos operandos. 
    CMP AX, BX  ; Compara AX con BX
  • JE: Salta si son iguales. 
    JE 0040h    ; Salta a 0040h si la comparación fue igual

Programa de Ejemplo

Aquí hay un programa simple en ensamblador que suma dos números y almacena el resultado en un registro:

section .data
    num1 db 10h  ; Define un byte con valor 10h
    num2 db 20h  ; Define un byte con valor 20h

section .text
    global _start

_start:
    mov al, [num1]  ; Mueve el valor de num1 a AL
    add al, [num2]  ; Suma el valor de num2 a AL
    ; AL ahora contiene el resultado (30h)
    
    ; Salir del programa (específico de Linux)
    mov eax, 1      ; Número de llamada al sistema (sys_exit)
    int 0x80        ; Llamada al kernel

Conclusión

Entender los conceptos y la terminología básica del lenguaje ensamblador es el primer paso hacia el dominio de este poderoso lenguaje de programación. Conceptos clave como el código máquina, los mnemónicos, los registros y el conjunto de instrucciones forman la base de la programación en ensamblador. Familiarizándote con estos términos y practicando instrucciones simples, estarás bien preparado para profundizar en temas más complejos de programación en lenguaje ensamblador.
