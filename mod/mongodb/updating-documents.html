Introducción

Actualizar documentos en MongoDB es una habilidad fundamental para gestionar y manipular datos. Esta sección te guiará a través de los conceptos básicos de la actualización de documentos, varios operadores de actualización y técnicas avanzadas para actualizaciones más complejas.

Actualizaciones Básicas

MongoDB proporciona varios métodos para actualizar documentos en una colección. Los métodos más comúnmente utilizados son updateOne(), updateMany() y replaceOne().

`updateOne()`

El método updateOne() actualiza un único documento que coincide con un filtro especificado.

db.collection.updateOne(
  { _id: 1 }, // Filtro
  { $set: { name: "John Doe" } } // Operación de actualización
);
  • Filtro: Especifica los criterios para coincidir con los documentos.
  • Operación de actualización: Define las modificaciones a aplicar.

`updateMany()`

El método updateMany() actualiza todos los documentos que coinciden con un filtro especificado.

db.collection.updateMany(
  { status: "active" }, // Filtro
  { $set: { status: "inactive" } } // Operación de actualización
);

`replaceOne()`

El método replaceOne() reemplaza un único documento que coincide con un filtro especificado con un nuevo documento.

db.collection.replaceOne(
  { _id: 1 }, // Filtro
  { name: "Jane Doe", age: 30 } // Nuevo documento
);

Operadores de Actualización

MongoDB proporciona una variedad de operadores de actualización para modificar documentos. Aquí hay algunos de los operadores más comúnmente utilizados:

  • $set: Establece el valor de un campo.
  • $unset: Elimina un campo de un documento.
  • $inc: Incrementa el valor de un campo en una cantidad especificada.
  • $push: Añade un elemento a un array.
  • $pull: Elimina un elemento de un array.

Ejemplos

$set

db.collection.updateOne(
  { _id: 1 },
  { $set: { age: 25 } }
);

$unset

db.collection.updateOne(
  { _id: 1 },
  { $unset: { age: "" } }
);

$inc

db.collection.updateOne(
  { _id: 1 },
  { $inc: { age: 1 } }
);

$push

db.collection.updateOne(
  { _id: 1 },
  { $push: { hobbies: "reading" } }
);

$pull

db.collection.updateOne(
  { _id: 1 },
  { $pull: { hobbies: "reading" } }
);

Upserts

Una operación upsert actualiza un documento si existe, o inserta un nuevo documento si no existe. Esto se logra configurando la opción upsert a true.

db.collection.updateOne(
  { _id: 2 },
  { $set: { name: "Alice" } },
  { upsert: true }
);

Actualizaciones de Arrays

Actualizar arrays en MongoDB puede ser más complejo. Aquí hay algunas operaciones comunes:

Actualización de Elementos Específicos del Array

Para actualizar un elemento específico en un array, usa el operador posicional $.

db.collection.updateOne(
  { _id: 1, "grades.grade": 85 },
  { $set: { "grades.$.grade": 90 } }
);

Añadir Elementos a Arrays

Usa el operador $addToSet para añadir un elemento a un array solo si no existe ya.

db.collection.updateOne(
  { _id: 1 },
  { $addToSet: { hobbies: "swimming" } }
);

Conclusión

Actualizar documentos en MongoDB es una característica poderosa que te permite modificar tus datos de manera eficiente. Al entender los métodos básicos de actualización, varios operadores de actualización y técnicas avanzadas como upserts y actualizaciones de arrays, puedes gestionar tus colecciones de MongoDB de manera efectiva. Practica estos conceptos con diferentes escenarios para volverte competente en la actualización de documentos en MongoDB.
