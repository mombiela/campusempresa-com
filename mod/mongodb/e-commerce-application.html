Introducción

En este tema, exploraremos cómo construir una aplicación de comercio electrónico utilizando MongoDB. Cubriremos todo, desde la configuración de la base de datos hasta la implementación de funciones avanzadas como indexación y agregación. Al final de este tema, tendrás una comprensión sólida de cómo usar MongoDB para gestionar y consultar datos para una aplicación de comercio electrónico.

Configuración de MongoDB

  • Instalación: Guía sobre la instalación de MongoDB en varias plataformas.
  • Conexión a MongoDB: Uso de la shell de MongoDB y controladores para diferentes lenguajes de programación.
# Iniciando el servidor de MongoDB
mongod --dbpath /path/to/your/db
// Conectando a MongoDB usando Node.js
const { MongoClient } = require('mongodb');
const uri = "mongodb://localhost:27017";
const client = new MongoClient(uri, { useNewUrlParser: true, useUnifiedTopology: true });

async function run() {
    try {
        await client.connect();
        console.log("Conectado a MongoDB");
    } finally {
        await client.close();
    }
}
run().catch(console.dir);

Operaciones CRUD

  • Crear: Insertar documentos en colecciones.
  • Leer: Consultar documentos de colecciones.
  • Actualizar: Modificar documentos existentes.
  • Eliminar: Eliminar documentos de colecciones.
// Insertando un documento en la colección de Usuarios
const user = {
    _id: "user123",
    name: "John Doe",
    email: "john.doe@example.com",
    password: "hashed_password",
    address: {
        street: "123 Main St",
        city: "Anytown",
        state: "CA",
        zip: "12345"
    }
};

await client.db("ecommerce").collection("users").insertOne(user);
// Consultando documentos de la colección de Productos
const products = await client.db("ecommerce").collection("products").find({ category: "Electrónica" }).toArray();
console.log(products);
// Actualizando un documento en la colección de Pedidos
await client.db("ecommerce").collection("orders").updateOne(
    { _id: "order123" },
    { $set: { status: "Shipped" } }
);
// Eliminando un documento de la colección de Usuarios
await client.db("ecommerce").collection("users").deleteOne({ _id: "user123" });

Indexación

  • Propósito de los Índices: Mejorar el rendimiento de las consultas.
  • Creación de Índices: Cómo crear índices en MongoDB.
  • Tipos de Índices: Índices de campo único, compuestos, de texto y geoespaciales.
// Creando un índice en el campo email en la colección de Usuarios
await client.db("ecommerce").collection("users").createIndex({ email: 1 });

Marco de Agregación

  • Pipeline de Agregación: Uso de etapas para transformar y analizar datos.
  • Etapas Comunes: $match, $group, $sort, $project.
// Agregando ventas totales por categoría de producto
const salesByCategory = await client.db("ecommerce").collection("orders").aggregate([
    { $unwind: "$products" },
    { $lookup: {
        from: "products",
        localField: "products.product_id",
        foreignField: "_id",
        as: "product_details"
    }},
    { $unwind: "$product_details" },
    { $group: {
        _id: "$product_details.category",
        total_sales: { $sum: "$products.quantity" }
    }},
    { $sort: { total_sales: -1 } }
]).toArray();
console.log(salesByCategory);

Temas Avanzados

  • Transacciones: Asegurando la consistencia de los datos con transacciones multi-documento.
  • Sharding: Distribuyendo datos a través de múltiples servidores para escalabilidad.
  • Replicación: Asegurando alta disponibilidad con conjuntos de réplicas.
// Usando una transacción para asegurar atomicidad
const session = client.startSession();
session.startTransaction();
try {
    await client.db("ecommerce").collection("users").updateOne(
        { _id: "user123" },
        { $set: { email: "new.email@example.com" } },
        { session }
    );
    await client.db("ecommerce").collection("orders").updateOne(
        { user_id: "user123" },
        { $set: { status: "Updated" } },
        { session }
    );
    await session.commitTransaction();
} catch (error) {
    await session.abortTransaction();
    throw error;
} finally {
    session.endSession();
}
