Instalando MongoDB

Siguiente ►
►

En esta sección, aprenderás cómo instalar MongoDB en diferentes sistemas operativos. MongoDB es una base de datos NoSQL de código abierto que se utiliza para almacenar datos en un formato de documento similar a JSON. La instalación de MongoDB es un proceso sencillo, pero varía ligeramente dependiendo del sistema operativo que estés utilizando.

Contenido

  1. Instalación en Windows
  2. Instalación en macOS
  3. Instalación en Linux

Instalación en Windows

Paso 1: Descargar el Instalador

  1. Visita la página oficial de descargas de MongoDB: MongoDB Download Center.
  2. Selecciona la versión de MongoDB adecuada para tu sistema operativo Windows.
  3. Descarga el archivo .msi.

Paso 2: Ejecutar el Instalador

  1. Abre el archivo .msi descargado.
  2. Sigue las instrucciones del asistente de instalación.
  3. Asegúrate de seleccionar la opción "Complete" para una instalación completa.
  4. Durante la instalación, marca la opción "Install MongoDB as a Service" para que MongoDB se ejecute como un servicio de Windows.

Paso 3: Configurar las Variables de Entorno

  1. Abre el Panel de Control y navega a Sistema y Seguridad > Sistema > Configuración avanzada del sistema.
  2. En la pestaña "Opciones avanzadas", haz clic en "Variables de entorno".
  3. En "Variables del sistema", selecciona la variable Path y haz clic en "Editar".
  4. Añade la ruta del directorio bin de MongoDB (por ejemplo, C:\Program Files\MongoDB\Server\4.4\bin).

Paso 4: Crear Directorios de Datos y Logs

  1. Abre una ventana de comandos (cmd) como administrador.
  2. Ejecuta los siguientes comandos para crear los directorios necesarios: 
    mkdir C:\data\db
mkdir C:\data\log

Instalación en macOS

Paso 1: Usar Homebrew

  1. Si no tienes Homebrew instalado, abre una terminal y ejecuta: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Una vez que Homebrew esté instalado, ejecuta: 
    brew tap mongodb/brew
brew install mongodb-community@4.4

Paso 2: Iniciar MongoDB

  1. Inicia MongoDB como un servicio: 
    brew services start mongodb/brew/mongodb-community

Paso 3: Verificar la Instalación

  1. Para verificar que MongoDB se está ejecutando, puedes usar: 
    ps aux | grep -v grep | grep mongod

Instalación en Linux

Paso 1: Importar la Clave Pública

  1. Abre una terminal y ejecuta: 
    wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -

Paso 2: Crear el Archivo de Lista para MongoDB

  1. Crea el archivo de lista para MongoDB: 
    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list

Paso 3: Actualizar el Índice de Paquetes

  1. Actualiza el índice de paquetes: 
    sudo apt-get update

Paso 4: Instalar MongoDB

  1. Instala MongoDB: 
    sudo apt-get install -y mongodb-org

Paso 5: Iniciar MongoDB

  1. Inicia el servicio de MongoDB: 
    sudo systemctl start mongod

Paso 6: Habilitar MongoDB para que se Inicie al Arrancar

  1. Habilita MongoDB para que se inicie automáticamente al arrancar: 
    sudo systemctl enable mongod

Verificación de la Instalación

Paso 1: Verificar el Estado del Servicio

  1. Para verificar que MongoDB se está ejecutando, usa el siguiente comando: 
    sudo systemctl status mongod

Paso 2: Conectar a MongoDB

  1. Abre una terminal o una ventana de comandos y ejecuta: 
    mongo
  2. Si ves el prompt de MongoDB (>) significa que te has conectado exitosamente a la base de datos.

Conclusión

En esta sección, hemos cubierto cómo instalar MongoDB en Windows, macOS y Linux. También hemos visto cómo verificar que la instalación se haya realizado correctamente. Ahora que tienes MongoDB instalado, estás listo para comenzar a explorar sus características y funcionalidades. En el próximo módulo, aprenderemos sobre las operaciones CRUD en MongoDB, que son fundamentales para interactuar con la base de datos.
