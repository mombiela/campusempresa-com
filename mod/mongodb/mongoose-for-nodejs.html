Introducción a Mongoose

Mongoose es una biblioteca de Modelado de Datos de Objetos (ODM) para MongoDB y Node.js. Proporciona una solución sencilla basada en esquemas para modelar los datos de tu aplicación. Incluye conversión de tipos incorporada, validación, construcción de consultas, hooks de lógica de negocio y más.

Conceptos Clave

  • ODM (Modelado de Datos de Objetos): Mongoose te permite definir esquemas para tus datos, que luego pueden ser utilizados para crear modelos.
  • Esquema: Un esquema define la estructura de los documentos dentro de una colección.
  • Modelo: Un modelo es una versión compilada del esquema, que proporciona una interfaz para interactuar con la base de datos.

Configuración de Mongoose

Para comenzar con Mongoose, necesitas instalarlo a través de npm y conectarlo a tu base de datos MongoDB.

Instalación

npm install mongoose

Conectando a MongoDB

const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/mydatabase', {
  useNewUrlParser: true,
  useUnifiedTopology: true
})
.then(() => console.log('MongoDB connected'))
.catch(err => console.log(err));

Definiendo Esquemas y Modelos

Los esquemas y modelos son el núcleo de Mongoose. Definen la estructura y el comportamiento de los datos en tu aplicación.

Creando un Esquema

const mongoose = require('mongoose');
const Schema = mongoose.Schema;

const userSchema = new Schema({
  name: String,
  age: Number,
  email: String,
  createdAt: {
    type: Date,
    default: Date.now
  }
});

Creando un Modelo

const User = mongoose.model('User', userSchema);

Operaciones CRUD

Mongoose facilita la realización de operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

Crear

const newUser = new User({
  name: 'John Doe',
  age: 30,
  email: 'john.doe@example.com'
});

newUser.save()
  .then(user => console.log(user))
  .catch(err => console.log(err));

Leer

User.find()
  .then(users => console.log(users))
  .catch(err => console.log(err));

Actualizar

User.findOneAndUpdate({ name: 'John Doe' }, { age: 31 }, { new: true })
  .then(user => console.log(user))
  .catch(err => console.log(err));

Eliminar

User.findOneAndDelete({ name: 'John Doe' })
  .then(user => console.log(user))
  .catch(err => console.log(err));

Características Avanzadas

Mongoose ofrece varias características avanzadas para mejorar tu aplicación.

Validación

Mongoose proporciona validadores incorporados y la capacidad de crear lógica de validación personalizada.

const userSchema = new Schema({
  name: {
    type: String,
    required: true
  },
  age: {
    type: Number,
    min: 0,
    max: 120
  },
  email: {
    type: String,
    match: /.+\@.+\..+/
  }
});

Middleware (Hooks)

El middleware de Mongoose (también conocido como hooks) son funciones que se ejecutan en etapas específicas del ciclo de vida de un modelo.

userSchema.pre('save', function(next) {
  this.createdAt = Date.now();
  next();
});

Población

La población es el proceso de reemplazar automáticamente las rutas especificadas en el documento con documento(s) de otra(s) colección(es).

const postSchema = new Schema({
  title: String,
  author: { type: Schema.Types.ObjectId, ref: 'User' }
});

const Post = mongoose.model('Post', postSchema);

Post.find()
  .populate('author')
  .then(posts => console.log(posts))
  .catch(err => console.log(err));

Conclusión

Mongoose es una herramienta poderosa para trabajar con MongoDB en Node.js. Simplifica el proceso de definir esquemas, realizar operaciones CRUD e implementar características avanzadas como validación, middleware y población. Al dominar Mongoose, puedes gestionar eficientemente los datos de tu aplicación y asegurar un manejo robusto e integridad de los datos.
