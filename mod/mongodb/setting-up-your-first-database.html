Introducción

En esta sección, te guiaremos a través del proceso de configuración de tu primera base de datos MongoDB. MongoDB es una base de datos NoSQL popular conocida por su flexibilidad y escalabilidad. Al final de esta sección, tendrás una base de datos MongoDB en funcionamiento y comprenderás los conceptos básicos para interactuar con ella.

Requisitos previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una computadora con acceso a internet
  • Conocimientos básicos de la interfaz de línea de comandos (CLI)
  • MongoDB instalado en tu máquina (cubrirémos la instalación brevemente)

Instalación de MongoDB

En Windows

  1. Descargar MongoDB: Visita el Centro de Descargas de MongoDB y descarga el instalador para Windows.
  2. Ejecutar el instalador: Sigue las indicaciones de instalación. Asegúrate de seleccionar la opción para instalar MongoDB como un servicio.
  3. Verificar la instalación: Abre el Símbolo del sistema y escribe: 
    mongod --version
    Deberías ver la información de la versión si MongoDB está instalado correctamente.

En macOS

  1. Instalar Homebrew: Si no tienes Homebrew instalado, abre Terminal y ejecuta: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instalar MongoDB: Usa Homebrew para instalar MongoDB: 
    brew tap mongodb/brew
brew install mongodb-community
  3. Iniciar MongoDB: Inicia el servicio de MongoDB: 
    brew services start mongodb/brew/mongodb-community
  4. Verificar la instalación: Verifica la instalación ejecutando: 
    mongod --version

En Linux

  1. Importar la clave pública: Abre Terminal y ejecuta: 
    wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -
  2. Crear un archivo de lista: Crea el archivo /etc/apt/sources.list.d/mongodb-org-4.4.list para Ubuntu: 
    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list
  3. Instalar MongoDB: Actualiza la lista de paquetes e instala MongoDB: 
    sudo apt-get update
sudo apt-get install -y mongodb-org
  4. Iniciar MongoDB: Inicia el servicio de MongoDB: 
    sudo systemctl start mongod
  5. Verificar la instalación: Verifica la instalación ejecutando: 
    mongod --version

Creando tu primera base de datos

Iniciando la Shell de MongoDB

Para interactuar con MongoDB, usarás la shell de MongoDB. Abre tu terminal o símbolo del sistema y escribe:

mongo

Este comando inicia la shell de MongoDB, donde puedes ejecutar comandos para interactuar con tu base de datos.

Creando una base de datos

En MongoDB, las bases de datos se crean implícitamente cuando almacenas datos en ellas por primera vez. Para crear una base de datos, cambia a un nuevo contexto de base de datos:

use myFirstDatabase

Este comando cambia a myFirstDatabase. Si no existe, MongoDB la creará cuando almacenes datos por primera vez.

Insertando datos

Para insertar datos en tu nueva base de datos, usa el método insertOne:

db.myCollection.insertOne({ name: "John Doe", age: 30, profession: "Software Engineer" })

Este comando crea una colección llamada myCollection e inserta un documento en ella.

Consultando datos

Para recuperar datos de tu colección, usa el método find:

db.myCollection.find()

Este comando devuelve todos los documentos en la colección myCollection.

Actualizando datos

Para actualizar datos en tu colección, usa el método updateOne:

db.myCollection.updateOne({ name: "John Doe" }, { $set: { age: 31 } })

Este comando actualiza el campo age del documento donde name es "John Doe".

Eliminando datos

Para eliminar datos de tu colección, usa el método deleteOne:

db.myCollection.deleteOne({ name: "John Doe" })

Este comando elimina el documento donde name es "John Doe".

Resumen

En esta sección, aprendiste cómo configurar MongoDB en diferentes sistemas operativos, crear tu primera base de datos y realizar operaciones básicas de CRUD (Crear, Leer, Actualizar, Eliminar). Estas habilidades fundamentales son esenciales para trabajar con MongoDB y servirán como base para temas más avanzados en este curso.
