En esta sección, exploraremos cómo implementar varias funcionalidades en MongoDB, que van desde funcionalidades básicas hasta avanzadas. Al final de este tema, serás capaz de crear, leer, actualizar y eliminar documentos, así como implementar características más complejas como indexación, agregación y transacciones.

Operaciones CRUD Básicas

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las operaciones básicas que realizarás en tus colecciones de MongoDB.

Crear

Para insertar un documento en una colección, utilizas los métodos insertOne o insertMany.

// Insertar un solo documento
db.collection('users').insertOne({
  name: "John Doe",
  age: 30,
  email: "john.doe@example.com"
});

// Insertar múltiples documentos
db.collection('users').insertMany([
  { name: "Jane Doe", age: 25, email: "jane.doe@example.com" },
  { name: "Alice Smith", age: 28, email: "alice.smith@example.com" }
]);

Leer

Para recuperar documentos de una colección, utilizas el método find.

// Encontrar todos los documentos en la colección
db.collection('users').find({});

// Encontrar documentos con una condición específica
db.collection('users').find({ age: { $gt: 25 } });

Actualizar

Para modificar documentos existentes, utilizas los métodos updateOne o updateMany.

// Actualizar un solo documento
db.collection('users').updateOne(
  { name: "John Doe" },
  { $set: { age: 31 } }
);

// Actualizar múltiples documentos
db.collection('users').updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "active" } }
);

Eliminar

Para eliminar documentos de una colección, utilizas los métodos deleteOne o deleteMany.

// Eliminar un solo documento
db.collection('users').deleteOne({ name: "John Doe" });

// Eliminar múltiples documentos
db.collection('users').deleteMany({ age: { $lt: 25 } });

Indexación

Los índices mejoran el rendimiento de las consultas de búsqueda. MongoDB soporta varios tipos de índices.

  • Índice de Campo Único: Indexa un solo campo en un documento.
  • Índice Compuesto: Indexa múltiples campos en un documento.
  • Índice Multikey: Indexa campos que contienen arrays.
  • Índice de Texto: Indexa contenido de texto para búsqueda de texto.
  • Índice Geoespacial: Indexa para consultas basadas en ubicación.

Crear un Índice

// Crear un índice de campo único
db.collection('users').createIndex({ email: 1 });

// Crear un índice compuesto
db.collection('users').createIndex({ name: 1, age: -1 });

Usar Índices

Los índices son utilizados automáticamente por MongoDB para optimizar el rendimiento de las consultas.

// Consulta que se beneficia de un índice
db.collection('users').find({ email: "john.doe@example.com" });

Agregación

Las operaciones de agregación procesan registros de datos y devuelven resultados calculados. MongoDB proporciona el marco de agregación, que es una herramienta poderosa para el análisis de datos.

Pipeline de Agregación Básica

db.collection('orders').aggregate([
  { $match: { status: "completed" } },
  { $group: { _id: "$customerId", totalAmount: { $sum: "$amount" } } },
  { $sort: { totalAmount: -1 } }
]);
  • $match: Filtra los documentos.
  • $group: Agrupa documentos por un identificador especificado.
  • $sort: Ordena los documentos.

Transacciones

Las transacciones te permiten ejecutar múltiples operaciones en aislamiento y asegurar la consistencia de los datos.

Usar Transacciones

const session = client.startSession();

session.startTransaction();

try {
  db.collection('accounts').updateOne(
    { accountId: 1 },
    { $inc: { balance: -100 } },
    { session }
  );

  db.collection('accounts').updateOne(
    { accountId: 2 },
    { $inc: { balance: 100 } },
    { session }
  );

  session.commitTransaction();
} catch (error) {
  session.abortTransaction();
  throw error;
} finally {
  session.endSession();
}

Conclusión

En este tema, cubrimos las características esenciales que necesitas implementar en MongoDB, desde operaciones CRUD básicas hasta funcionalidades más avanzadas como indexación, agregación y transacciones. Entender estas características te ayudará a construir aplicaciones eficientes y escalables usando MongoDB. Practica estos conceptos con datos del mundo real para solidificar tu comprensión y volverte competente en MongoDB.
