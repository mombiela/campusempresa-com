Introducción a las Consultas Geoespaciales

Las consultas geoespaciales te permiten realizar búsquedas y operaciones basadas en la ubicación en tus datos. MongoDB soporta consultas geoespaciales mediante el uso de índices geoespaciales y operadores de consulta específicos.

Conceptos Clave

  • Tipos de Datos Geoespaciales: MongoDB soporta dos tipos principales de datos geoespaciales: Point y Polygon.
  • Índices Geoespaciales: Índices que permiten consultas eficientes de datos geoespaciales.
  • Operadores de Consulta Geoespaciales: Operadores que te permiten realizar varias consultas geoespaciales.

Tipos de Datos Geoespaciales

MongoDB soporta objetos GeoJSON y pares de coordenadas heredados para datos geoespaciales.

Objetos GeoJSON

GeoJSON es un formato para codificar una variedad de estructuras de datos geográficos.

  • Point: Representa una única ubicación. 
    { "type": "Point", "coordinates": [longitud, latitud] }
  • Polygon: Representa un área encerrada por una serie de puntos. 
    { 
  "type": "Polygon", 
  "coordinates": [
    [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ]
  ] 
}

Pares de Coordenadas Heredados

Los pares de coordenadas heredados son simples arrays de [longitud, latitud].

Índices Geoespaciales

Para realizar consultas geoespaciales de manera eficiente, necesitas crear índices geoespaciales.

Índice 2dsphere

El índice 2dsphere soporta consultas que interpretan geometrías en una esfera similar a la Tierra.

db.collection.createIndex({ location: "2dsphere" })

Índice 2d

El índice 2d soporta consultas en un plano cartesiano plano.

db.collection.createIndex({ location: "2d" })

Operadores de Consulta Geoespaciales

MongoDB proporciona varios operadores para realizar consultas geoespaciales.

$near

Encuentra documentos que están cerca de un punto especificado.

db.collection.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [longitud, latitud] },
      $maxDistance: distancia_en_metros
    }
  }
})

$geoWithin

Encuentra documentos dentro de una geometría especificada.

db.collection.find({
  location: {
    $geoWithin: {
      $geometry: {
        type: "Polygon",
        coordinates: [ [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ] ]
      }
    }
  }
})

$geoIntersects

Encuentra documentos que intersectan con una geometría especificada.

db.collection.find({
  location: {
    $geoIntersects: {
      $geometry: {
        type: "Polygon",
        coordinates: [ [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ] ]
      }
    }
  }
})

Ejemplos y Ejercicios

Ejemplo 1: Creando un Índice Geoespacial

Crea un índice 2dsphere en una colección llamada places.

db.places.createIndex({ location: "2dsphere" })

Ejemplo 2: Encontrando Ubicaciones Cercanas

Encuentra todos los lugares dentro de 500 metros de un punto específico.

db.places.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [-73.97, 40.77] },
      $maxDistance: 500
    }
  }
})

Ejercicio 1: Creando y Consultando Datos Geoespaciales

  1. Inserta algunos documentos con datos geoespaciales en una colección llamada restaurants.
  2. Crea un índice 2dsphere en el campo location.
  3. Escribe una consulta para encontrar todos los restaurantes dentro de 1 kilómetro de un punto dado.
// Paso 1: Insertar documentos
db.restaurants.insertMany([
  { name: "Restaurant A", location: { type: "Point", coordinates: [-73.97, 40.77] } },
  { name: "Restaurant B", location: { type: "Point", coordinates: [-73.98, 40.78] } },
  { name: "Restaurant C", location: { type: "Point", coordinates: [-73.99, 40.79] } }
])

// Paso 2: Crear un índice 2dsphere
db.restaurants.createIndex({ location: "2dsphere" })

// Paso 3: Consultar restaurantes cercanos
db.restaurants.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [-73.97, 40.77] },
      $maxDistance: 1000
    }
  }
})

Conclusión

Las consultas geoespaciales en MongoDB te permiten realizar potentes búsquedas y operaciones basadas en la ubicación. Al entender los tipos de datos geoespaciales, índices y operadores de consulta, puedes gestionar y consultar datos geoespaciales de manera eficiente en tus aplicaciones. Practica creando índices y ejecutando consultas para volverte competente en el manejo de datos geoespaciales con MongoDB.
