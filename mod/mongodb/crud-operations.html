Operaciones CRUD en MongoDB

Introducción

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las cuatro operaciones básicas que se pueden realizar en una base de datos. En esta sección, exploraremos cómo realizar operaciones CRUD utilizando MongoDB, una popular base de datos NoSQL.

Requisitos previos

Antes de sumergirse en las operaciones CRUD, asegúrese de tener:

  • MongoDB instalado en su máquina.
  • MongoDB Compass u otro cliente de MongoDB.
  • Comprensión básica de las colecciones y documentos de MongoDB.

Operaciones de creación

Crear datos en MongoDB implica insertar documentos en una colección.

Insertar un solo documento

Para insertar un solo documento en una colección, use el método insertOne.

db.collectionName.insertOne({
  name: "John Doe",
  age: 29,
  email: "john.doe@example.com"
});

Insertar múltiples documentos

Para insertar varios documentos a la vez, use el método insertMany.

db.collectionName.insertMany([
  { name: "Jane Doe", age: 25, email: "jane.doe@example.com" },
  { name: "Alice Smith", age: 30, email: "alice.smith@example.com" }
]);

Operaciones de lectura

Leer datos de MongoDB implica consultar la base de datos para recuperar documentos.

Encontrar un solo documento

Para encontrar un solo documento que coincida con una consulta, use el método findOne.

db.collectionName.findOne({ name: "John Doe" });

Encontrar múltiples documentos

Para encontrar varios documentos que coincidan con una consulta, use el método find.

db.collectionName.find({ age: { $gt: 25 } });

Proyección

Para limitar los campos devueltos por una consulta, use la proyección.

db.collectionName.find(
  { age: { $gt: 25 } },
  { name: 1, email: 1, _id: 0 }
);

Operaciones de actualización

Actualizar datos en MongoDB implica modificar documentos existentes.

Actualizar un solo documento

Para actualizar un solo documento, use el método updateOne.

db.collectionName.updateOne(
  { name: "John Doe" },
  { $set: { age: 30 } }
);

Actualizar múltiples documentos

Para actualizar varios documentos, use el método updateMany.

db.collectionName.updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "young" } }
);

Reemplazar un documento

Para reemplazar un documento completo, use el método replaceOne.

db.collectionName.replaceOne(
  { name: "John Doe" },
  { name: "John Doe", age: 30, email: "john.doe@newdomain.com" }
);

Operaciones de eliminación

Eliminar datos en MongoDB implica eliminar documentos de una colección.

Eliminar un solo documento

Para eliminar un solo documento, use el método deleteOne.

db.collectionName.deleteOne({ name: "John Doe" });

Eliminar múltiples documentos

Para eliminar varios documentos, use el método deleteMany.

db.collectionName.deleteMany({ age: { $lt: 25 } });

Conclusión

En esta sección, cubrimos las operaciones CRUD básicas en MongoDB:

  • Crear: Insertar uno o varios documentos.
  • Leer: Consultar uno o varios documentos con proyecciones opcionales.
  • Actualizar: Modificar uno o varios documentos, o reemplazar documentos completos.
  • Eliminar: Eliminar uno o varios documentos.

Entender estas operaciones es fundamental para trabajar con MongoDB, ya que forman la base de la interacción con la base de datos. Practique estas operaciones para volverse competente en la gestión de colecciones y documentos de MongoDB.
