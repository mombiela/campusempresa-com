Configurar tu entorno es el primer paso para comenzar a trabajar con PyTorch. Esta sección te guiará a través del proceso de instalación de PyTorch y la configuración de tu entorno de desarrollo para asegurar una experiencia fluida.

Instalando PyTorch

Para comenzar con PyTorch, necesitas instalarlo en tu máquina. PyTorch puede ser instalado usando pip o conda, dependiendo de tu preferencia y el gestor de paquetes que uses.

Usando pip

  1. Abre tu terminal o símbolo del sistema.

  2. Ejecuta el siguiente comando:

    pip install torch torchvision torchaudio
    • torch: La biblioteca principal de PyTorch.
    • torchvision: Contiene conjuntos de datos populares, arquitecturas de modelos y transformaciones de imágenes para visión por computadora.
    • torchaudio: Proporciona E/S de audio y transformaciones.

Usando conda

  1. Abre tu terminal o símbolo del sistema.

  2. Ejecuta el siguiente comando:

    conda install pytorch torchvision torchaudio -c pytorch
    • -c pytorch: Especifica el canal de PyTorch para asegurar que obtengas la última versión.

Verificando la Instalación

Después de instalar PyTorch, es importante verificar que la instalación fue exitosa.

  1. Abre tu intérprete de Python o crea un nuevo script de Python.

  2. Ejecuta el siguiente código:

    import torch
print(torch.__version__)
    • Esto debería imprimir la versión de PyTorch que tienes instalada.

  3. Verifica el soporte para CUDA (si tienes una GPU compatible):

    print(torch.cuda.is_available())
    • Esto debería devolver True si CUDA está disponible y configurado correctamente.

Configurando Tu IDE

Elegir el Entorno de Desarrollo Integrado (IDE) adecuado puede mejorar significativamente tu productividad. Aquí hay algunas opciones populares:

  • PyCharm: Un potente IDE con muchas características diseñadas para el desarrollo en Python.
  • Visual Studio Code (VS Code): Un editor ligero y altamente personalizable con un rico ecosistema de extensiones.
  • Jupyter Notebook: Un entorno interactivo ideal para análisis de datos y experimentación.

Configurando PyCharm

  1. Instala PyCharm desde el sitio web oficial.

  2. Crea un nuevo proyecto y configura un entorno virtual:

    • Ve a File > New Project.
    • Elige una ubicación y selecciona New environment using: Virtualenv.
    • Haz clic en Create.

  3. Instala PyTorch en el entorno virtual:

    • Abre la terminal dentro de PyCharm.
    • Ejecuta el comando de instalación de pip mencionado anteriormente.

Configurando Visual Studio Code

  1. Instala VS Code desde el sitio web oficial.

  2. Instala la extensión de Python:

    • Ve a la vista de Extensiones haciendo clic en el icono de Extensiones en la Barra de Actividad en el lado de la ventana.
    • Busca Python e instala la extensión proporcionada por Microsoft.

  3. Configura un entorno virtual:

    • Abre la terminal en VS Code.

    • Ejecuta los siguientes comandos:

      python -m venv venv
source venv/bin/activate  # En Windows usa `venv\Scripts\activate`

  4. Instala PyTorch en el entorno virtual:

    • Ejecuta el comando de instalación de pip mencionado anteriormente.

Usando Jupyter Notebook

  1. Instala Jupyter Notebook:

    pip install notebook

  2. Inicia Jupyter Notebook:

    jupyter notebook

  3. Crea un nuevo notebook e instala PyTorch:

    • Abre un nuevo notebook.

    • Ejecuta el siguiente código en una celda:

      !pip install torch torchvision torchaudio

Conclusión

Configurar tu entorno correctamente es crucial para una experiencia de desarrollo fluida con PyTorch. Ya sea que elijas usar pip o conda para la instalación, y ya sea que prefieras PyCharm, VS Code o Jupyter Notebook, la clave es asegurar que PyTorch esté instalado y configurado correctamente. Siguiendo los pasos descritos en esta sección, deberías estar listo para comenzar tu viaje con PyTorch.
