Introducción a TorchVision

TorchVision es una biblioteca crucial en el ecosistema de PyTorch, específicamente diseñada para tareas de visión por computadora. Proporciona utilidades para el procesamiento de imágenes, conjuntos de datos y modelos preentrenados, lo que facilita el trabajo con datos de imágenes.

Conceptos Clave

  • Datasets: Conjuntos de datos predefinidos como CIFAR-10, ImageNet y MNIST.
  • Transforms: Transformaciones de imágenes para preprocesamiento y aumento de datos.
  • Models: Modelos preentrenados para tareas como clasificación de imágenes, detección de objetos y segmentación.

Instalando TorchVision

Para comenzar con TorchVision, necesitas instalarlo junto con PyTorch. Puedes hacerlo usando pip:

pip install torch torchvision

Trabajando con Datasets

TorchVision proporciona varios conjuntos de datos populares que pueden cargarse y usarse fácilmente.

Cargando un Dataset

Aquí tienes un ejemplo de cómo cargar el conjunto de datos CIFAR-10:

import torchvision
import torchvision.transforms as transforms

# Define una transformación para normalizar los datos
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))
])

# Carga los conjuntos de datos de entrenamiento y prueba
trainset = torchvision.datasets.CIFAR10(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=4, shuffle=True, num_workers=2)

testset = torchvision.datasets.CIFAR10(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4, shuffle=False, num_workers=2)

Entendiendo las Transformaciones

Las transformaciones se utilizan para preprocesar y aumentar los datos. Las transformaciones comunes incluyen cambiar el tamaño, recortar, normalizar y convertir imágenes en tensores.

Ejemplo de Transformaciones Comunes

transform = transforms.Compose([
    transforms.Resize((256, 256)),
    transforms.RandomCrop(224),
    transforms.RandomHorizontalFlip(),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

Usando Modelos Preentrenados

TorchVision proporciona varios modelos preentrenados que pueden usarse para diversas tareas. Estos modelos están entrenados en grandes conjuntos de datos y pueden ajustarse para tareas específicas.

Cargando un Modelo Preentrenado

Aquí tienes un ejemplo de cómo cargar y usar un modelo ResNet preentrenado para la clasificación de imágenes:

import torch
import torchvision.models as models

# Carga un modelo ResNet preentrenado
model = models.resnet18(pretrained=True)

# Configura el modelo en modo de evaluación
model.eval()

# Ejemplo de uso del modelo para inferencia
from PIL import Image
from torchvision import transforms

# Carga una imagen
img = Image.open("path_to_image.jpg")

# Define la transformación para cumplir con los requisitos de entrada del modelo
preprocess = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]),
])

# Preprocesa la imagen
img_t = preprocess(img)
batch_t = torch.unsqueeze(img_t, 0)

# Realiza la inferencia
with torch.no_grad():
    out = model(batch_t)

# Imprime las 5 clases predichas más probables
_, indices = torch.topk(out, 5)
print(indices)

Técnicas de Aumento de Datos

El aumento de datos es una técnica utilizada para aumentar la diversidad de tus datos de entrenamiento sin recopilar nuevos datos. TorchVision proporciona varias técnicas de aumento.

Ejemplo de Aumento de Datos

transform = transforms.Compose([
    transforms.RandomResizedCrop(224),
    transforms.RandomHorizontalFlip(),
    transforms.RandomRotation(10),
    transforms.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2, hue=0.2),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

Conclusión

TorchVision es una biblioteca poderosa que simplifica muchos aspectos de las tareas de visión por computadora en PyTorch. Proporciona acceso fácil a conjuntos de datos populares, una variedad de transformaciones de imágenes para preprocesamiento y aumento, y modelos preentrenados para un despliegue rápido y ajuste fino. Al dominar TorchVision, puedes mejorar significativamente tus proyectos y flujos de trabajo de visión por computadora.
