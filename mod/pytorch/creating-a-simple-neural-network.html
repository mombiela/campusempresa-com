Creando una Red Neuronal Simple
Funciones de Activación ►

En esta sección, aprenderemos a crear una red neuronal simple usando PyTorch. Esto incluirá configurar el entorno, entender los bloques de construcción básicos de las redes neuronales e implementar una red neuronal simple desde cero.

Configuración del Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno para usar PyTorch.

  1. Instalar PyTorch: Sigue las instrucciones en el sitio web oficial de PyTorch para instalar PyTorch.

  2. Verificar la Instalación: Abre un intérprete de Python y ejecuta los siguientes comandos para asegurarte de que PyTorch esté instalado correctamente:

    import torch
print(torch.__version__)

Entendiendo los Bloques de Construcción Básicos

Antes de sumergirnos en el código, entendamos los componentes básicos de una red neuronal en PyTorch.

  • Tensores: El bloque de construcción fundamental en PyTorch, similar a los arrays de NumPy pero con soporte para GPU.
  • Autograd: La biblioteca de diferenciación automática de PyTorch que ayuda a calcular gradientes.
  • Módulo de Red Neuronal (torch.nn): Contiene clases y funciones para construir redes neuronales.
  • Optimizadores (torch.optim): Se utilizan para actualizar los pesos de la red basándose en los gradientes calculados.
  • Funciones de Pérdida: Miden qué tan bien está funcionando la red neuronal.

Implementando una Red Neuronal Simple

Implementemos una red neuronal simple para clasificar dígitos escritos a mano del conjunto de datos MNIST.

Paso 1: Importar Bibliotecas

Primero, importa las bibliotecas necesarias.

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Paso 2: Cargar y Preprocesar los Datos

Carga el conjunto de datos MNIST y aplica las transformaciones necesarias.

transform = transforms.Compose([transforms.ToTensor(), transforms.Normalize((0.5,), (0.5,))])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=64, shuffle=True)

testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=64, shuffle=False)

Paso 3: Definir la Red Neuronal

Define una red neuronal simple con una capa oculta.

class SimpleNN(nn.Module):
    def __init__(self):
        super(SimpleNN, self).__init__()
        self.fc1 = nn.Linear(28 * 28, 128)  # Capa de entrada a capa oculta
        self.fc2 = nn.Linear(128, 10)       # Capa oculta a capa de salida

    def forward(self, x):
        x = x.view(-1, 28 * 28)  # Aplanar la entrada
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

net = SimpleNN()
print(net)

Paso 4: Definir la Función de Pérdida y el Optimizador

Usa la Pérdida de Entropía Cruzada y el optimizador de Descenso de Gradiente Estocástico (SGD).

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.01, momentum=0.9)

Paso 5: Entrenar la Red Neuronal

Entrena la red por unos pocos epochs.

for epoch in range(5):  # Recorrer el conjunto de datos varias veces
    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        inputs, labels = data

        optimizer.zero_grad()  # Poner a cero los gradientes de los parámetros

        outputs = net(inputs)  # Paso hacia adelante
        loss = criterion(outputs, labels)  # Calcular la pérdida
        loss.backward()  # Paso hacia atrás
        optimizer.step()  # Optimizar los pesos

        running_loss += loss.item()
        if i % 100 == 99:  # Imprimir cada 100 mini-lotes
            print(f'[Epoch {epoch + 1}, Batch {i + 1}] loss: {running_loss / 100:.3f}')
            running_loss = 0.0

print('Finished Training')

Paso 6: Evaluar la Red Neuronal

Evalúa el rendimiento de la red entrenada en el conjunto de datos de prueba.

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Accuracy of the network on the 10000 test images: {100 * correct / total:.2f}%')

Resumen

En esta sección, hemos cubierto los conceptos básicos para crear una red neuronal simple usando PyTorch. Comenzamos configurando el entorno, entendiendo los componentes básicos, y luego implementamos una red neuronal simple para clasificar dígitos escritos a mano del conjunto de datos MNIST. También entrenamos y evaluamos la red para medir su rendimiento. Este conocimiento fundamental será crucial a medida que avancemos hacia temas más avanzados en PyTorch.
