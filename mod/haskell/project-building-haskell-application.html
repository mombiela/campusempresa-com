Proyecto: Construcción de una Aplicación en Haskell
Cabal y Stack ►

Introducción

En esta sección, nos embarcaremos en un viaje para construir una aplicación completa en Haskell desde cero. Este proyecto te ayudará a consolidar tus conocimientos de Haskell y te dará experiencia práctica en el desarrollo de una aplicación del mundo real.

Resumen del Proyecto

  • Objetivo: Construir una aplicación de lista de tareas simple en la línea de comandos.
  • Características:
    • Añadir una nueva tarea.
    • Listar todas las tareas.
    • Marcar una tarea como completada.
    • Eliminar una tarea.
  • Herramientas: GHC (Glasgow Haskell Compiler), Stack (herramienta de construcción de Haskell).

Configuración del Entorno

Antes de comenzar a codificar, configuremos nuestro entorno de desarrollo.

Instalación de GHC y Stack

  1. GHC: El Glasgow Haskell Compiler es el compilador de Haskell más utilizado.
  2. Stack: Un programa multiplataforma para desarrollar proyectos en Haskell.

Pasos de Instalación

  • Windows: 
    choco install haskell-stack
  • macOS: 
    brew install haskell-stack
  • Linux: 
    curl -sSL https://get.haskellstack.org/ | sh

Creación de un Nuevo Proyecto

Usa Stack para crear un nuevo proyecto en Haskell:

stack new todo-app simple
cd todo-app
stack setup

Diseño de la Aplicación

Vamos a delinear la estructura de nuestra aplicación.

Modelo de Datos

Definiremos un tipo de datos Task para representar cada elemento de la lista de tareas.

data Task = Task {
    taskId :: Int,
    taskDescription :: String,
    taskCompleted :: Bool
} deriving (Show, Read)

Funciones Principales

  • Añadir Tarea: Añadir una nueva tarea a la lista.
  • Listar Tareas: Mostrar todas las tareas.
  • Completar Tarea: Marcar una tarea como completada.
  • Eliminar Tarea: Eliminar una tarea de la lista.

Implementación de la Aplicación

Módulo Principal

Crea un archivo Main.hs en el directorio src.

Importación de Módulos Necesarios

import System.IO
import Data.List
import Control.Monad

Función Principal

main :: IO ()
main = do
    putStrLn "¡Bienvenido a la Aplicación de Lista de Tareas en Haskell!"
    mainMenu

Menú Principal

mainMenu :: IO ()
mainMenu = do
    putStrLn "1. Añadir Tarea"
    putStrLn "2. Listar Tareas"
    putStrLn "3. Completar Tarea"
    putStrLn "4. Eliminar Tarea"
    putStrLn "5. Salir"
    putStr "Elige una opción: "
    hFlush stdout
    option <- getLine
    handleOption option

Manejo de la Entrada del Usuario

handleOption :: String -> IO ()
handleOption option = case option of
    "1" -> addTask
    "2" -> listTasks
    "3" -> completeTask
    "4" -> deleteTask
    "5" -> putStrLn "¡Adiós!"
    _   -> putStrLn "¡Opción inválida!" >> mainMenu

Añadir una Tarea

addTask :: IO ()
addTask = do
    putStr "Introduce la descripción de la tarea: "
    hFlush stdout
    description <- getLine
    -- Aquí normalmente guardarías la tarea en un archivo o base de datos
    putStrLn $ "Tarea añadida: " ++ description
    mainMenu

Conclusión

En este proyecto, construimos una aplicación de lista de tareas simple en la línea de comandos en Haskell. Cubrimos:

  • Configuración del entorno de desarrollo en Haskell.
