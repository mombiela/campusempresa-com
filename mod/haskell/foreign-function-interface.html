Introducción a FFI

La Interfaz de Funciones Extranjeras (FFI) permite a los programas Haskell llamar a funciones escritas en otros lenguajes de programación, como C. Esto es particularmente útil para aprovechar bibliotecas existentes, optimizar secciones críticas de rendimiento o interactuar con APIs a nivel de sistema.

Conceptos Clave

  • Importaciones Extranjeras: Declarar funciones externas en Haskell.
  • Exportaciones Extranjeras: Hacer funciones Haskell disponibles para otros lenguajes.
  • Marshalling: Convertir datos entre Haskell y el lenguaje extranjero.

Importaciones Extranjeras

Declarar Funciones Externas

Para llamar a una función escrita en C desde Haskell, necesitas declararla usando la palabra clave foreign import.

Ejemplo: Importando una Función en C

Supongamos que tienes una función en C int add(int a, int b):

// add.c
#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

Puedes importar esta función en Haskell de la siguiente manera:

-- Add.hs
{-# LANGUAGE ForeignFunctionInterface #-}

module Add where

foreign import ccall "add" c_add :: Int -> Int -> Int

main :: IO ()
main = do
    let result = c_add 5 3
    putStrLn $ "El resultado es: " ++ show(result)

Explicación

  • foreign import ccall "add": Indica que estamos importando una función en C llamada add.
  • c_add :: Int -> Int -> Int: Especifica la firma de tipo en Haskell para la función importada.

Exportaciones Extranjeras

Hacer Funciones Haskell Disponibles para C

También puedes hacer que las funciones Haskell sean llamables desde C usando la palabra clave foreign export.

Ejemplo: Exportando una Función Haskell

Supongamos que tienes una función Haskell multiply:

-- Multiply.hs
{-# LANGUAGE ForeignFunctionInterface #-}

module Multiply where

foreign export ccall multiply :: Int -> Int -> Int

multiply :: Int -> Int -> Int
multiply a b = a * b

Explicación

  • foreign export ccall multiply: Indica que estamos exportando la función Haskell multiply para que sea llamable desde C.
  • multiply :: Int -> Int -> Int: Especifica la función Haskell a exportar.

Marshalling de Datos

Convertir Tipos de Datos

Al interactuar con funciones extranjeras, a menudo necesitas convertir datos entre Haskell y el lenguaje extranjero. Este proceso se conoce como marshalling.

Ejemplo: Marshalling de Cadenas

Considera una función en C que toma una cadena:

// greet.c
#include <stdio.h>

void greet(const char* name) {
    printf("¡Hola, %s!\n", name);
}

Puedes llamar a esta función desde Haskell haciendo marshalling de una String de Haskell a una cadena C (CString):

-- Greet.hs
{-# LANGUAGE ForeignFunctionInterface #-}

module Greet where

import Foreign.C.String (CString, withCString)

foreign import ccall "greet" c_greet :: CString -> IO ()

greet :: String -> IO ()
greet name = withCString name c_greet

main :: IO ()
main = greet "Haskell"

Explicación

  • import Foreign.C.String (CString, withCString): Importa las funciones necesarias para el marshalling de cadenas.
  • withCString name c_greet: Convierte la String de Haskell a una CString y llama a la función en C.

Conclusión

La Interfaz de Funciones Extranjeras (FFI) en Haskell es una característica poderosa que te permite llamar a funciones escritas en otros lenguajes y hacer funciones Haskell disponibles para esos lenguajes. Al entender cómo declarar importaciones y exportaciones extranjeras, y cómo hacer marshalling de datos entre Haskell y el lenguaje extranjero, puedes extender las capacidades de tus programas Haskell e integrarlos con una amplia gama de bibliotecas y sistemas externos.
