Variables y Constantes en Haskell
Tipos de Datos Básicos ►

Introducción

En Haskell, las variables y constantes son conceptos fundamentales que ayudan a almacenar y manipular datos. A diferencia de los lenguajes imperativos, Haskell trata las variables de manera diferente debido a su naturaleza funcional. Esta sección cubrirá los conceptos básicos de variables y constantes, sus diferencias y cómo usarlas efectivamente en Haskell.

Variables en Haskell

En Haskell, las variables son inmutables, lo que significa que una vez que se asigna un valor a una variable, no puede cambiarse. Esta es una característica clave de la programación funcional.

Conceptos Clave

  • Inmutabilidad: Una vez que se asigna un valor a una variable, no puede cambiarse.
  • Vinculación: El proceso de asociar una variable con un valor.
  • Ámbito: La región del código donde la variable es accesible.

Ejemplo

-- Vinculación de variable
x = 10

-- Usando la variable en una función
addTen :: Int -> Int
addTen y = x + y

-- Ejemplo de uso
main = print (addTen 5)  -- Salida: 15

Explicación:

  • x = 10 vincula el valor 10 a la variable x.
  • addTen es una función que toma un entero y y le suma x.
  • En main, addTen 5 resulta en 15 porque x es 10 y y es 5.

Constantes en Haskell

Las constantes en Haskell son similares a las variables pero se usan para representar valores fijos que no cambian a lo largo del programa.

Conceptos Clave

  • Valores literales: Valores escritos directamente como 42, 3.14 o "Hola".
  • Constantes nombradas: Constantes a las que se les da un nombre para una mejor legibilidad y mantenibilidad.

Ejemplo

-- Constante nombrada
piValue = 3.14159

-- Usando la constante en una función
circumference :: Float -> Float
circumference radius = 2 * piValue * radius

-- Ejemplo de uso
main = print (circumference 5)  -- Salida: 31.4159

Explicación:

  • piValue = 3.14159 asigna el valor de Pi a la constante piValue.
  • circumference es una función que calcula la circunferencia de un círculo usando el radio y piValue.
  • En main, circumference 5 resulta en 31.4159 porque el radio es 5 y piValue es 3.14159.

Diferencias Entre Variables y Constantes

Característica Variables Constantes
Mutabilidad Inmutable Inmutable
Propósito Almacenar valores temporales Representar valores fijos
Uso Puede usarse en funciones Usado para valores fijos
Ejemplo x = 10 piValue = 3.14159

Ejercicios

  1. Vinculación Básica de Variables: Define una variable a con el valor 20 y crea una función double que duplique el valor de a.

    a = 20

double :: Int -> Int
double y = a * 2

main = print (double 0)  -- La salida debería ser 40

  2. Usando Constantes: Define una constante gravity con el valor 9.8 y crea una función fallTime que calcule el tiempo que tarda un objeto en caer desde una altura dada.

    gravity = 9.8

fallTime :: Float -> Float
fallTime height = sqrt (2 * height / gravity)

main = print (fallTime 20)  -- La salida debería ser aproximadamente 2.02

Conclusión

Entender las variables y constantes en Haskell es crucial para escribir código efectivo y eficiente. Las variables en Haskell son inmutables, lo que se alinea con el paradigma de programación funcional, y las constantes se usan para representar valores fijos. Al dominar estos conceptos, puedes escribir programas en Haskell más predecibles y mantenibles.
