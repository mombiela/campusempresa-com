Introducción

Haskell, un lenguaje de programación puramente funcional y con tipado estático, ofrece una variedad de tipos de datos básicos que son esenciales para cualquier programador de Haskell. Esta sección cubrirá los tipos de datos fundamentales en Haskell, proporcionando explicaciones claras, ejemplos y ejercicios para solidificar tu comprensión.

Tipos de Datos Clave

Enteros

  • Int: Un tipo de entero de precisión fija.
  • Integer: Un tipo de entero de precisión arbitraria.
-- Ejemplo de Int
let x :: Int
x = 42

-- Ejemplo de Integer
let y :: Integer
y = 12345678901234567890

Números de Punto Flotante

  • Float: Un número de punto flotante de precisión simple.
  • Double: Un número de punto flotante de doble precisión.
-- Ejemplo de Float
let a :: Float
a = 3.14

-- Ejemplo de Double
let b :: Double
b = 2.718281828459045

Caracteres y Cadenas

  • Char: Representa un solo carácter.
  • String: Una lista de caracteres, esencialmente [Char].
-- Ejemplo de Char
let c :: Char
c = 'H'

-- Ejemplo de String
let s :: String
s = "¡Hola, Haskell!"

Booleano

  • Bool: Representa un valor booleano, que puede ser True o False.
-- Ejemplo de Bool
let isHaskellFun :: Bool
isHaskellFun = True

Tuplas

  • Las tuplas son colecciones de tamaño fijo de elementos, que pueden ser de diferentes tipos.
-- Ejemplo de una tupla
let person :: (String, Int)
person = ("Alice", 30)

Listas

  • Las listas son colecciones de elementos del mismo tipo.
-- Ejemplo de una lista
let numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

Comparación de Tipos de Datos

| Tipo de Dato | Descripción | Ejemplo | |--------------|-------------|---------| | Int | Entero de precisión fija | 42 | | Integer | Entero de precisión arbitraria | 12345678901234567890 | | Float | Punto flotante de precisión simple | 3.14 | | Double | Punto flotante de doble precisión | 2.718281828459045 | | Char | Carácter único | 'H' | | String | Lista de caracteres | "¡Hola, Haskell!" | | Bool | Valor booleano | True | | Tuple | Colección de tamaño fijo de elementos | ("Alice", 30) | | List | Colección de elementos del mismo tipo | [1, 2, 3, 4, 5] |

Ejercicios

  1. Definir Variables:

    • Define una variable Int con el valor 10.
    • Define una variable Double con el valor 3.14159.
    • Define una variable String con el valor "Haskell".

  2. Crear Tuplas y Listas:

    • Crea una tupla que represente un libro con un título (String) y el número de páginas (Int).
    • Crea una lista de valores Bool que representen los resultados de algunas condiciones: [True, False, True].

  3. Manipular Datos:

    • Escribe una función que tome una lista de enteros y devuelva la suma de los elementos.
    • Escribe una función que tome una tupla de dos enteros y devuelva su producto.
-- Ejercicio 1
let myInt :: Int
myInt = 10

let myDouble :: Double
myDouble = 3.14159

let myString :: String
myString = "Haskell"

-- Ejercicio 2
let book :: (String, Int)
book = ("Learn You a Haskell", 400)

let results :: [Bool]
results = [True, False, True]

-- Ejercicio 3
sumList :: [Int] -> Int
sumList xs = sum xs

productTuple :: (Int, Int) -> Int
productTuple (a, b) = a * b

Conclusión

Entender los tipos de datos básicos de Haskell es crucial para escribir código efectivo y eficiente. Esta sección te ha introducido a los principales tipos de datos, incluyendo enteros, números de punto flotante, caracteres, cadenas, booleanos, tuplas y listas. Al practicar los ejercicios proporcionados, obtendrás una base sólida en el manejo de estos tipos de datos en Haskell.
