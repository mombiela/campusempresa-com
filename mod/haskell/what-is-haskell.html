Introducción a Haskell

Haskell es un lenguaje de programación puramente funcional con tipado estático fuerte y evaluación perezosa. Se llama así en honor al lógico Haskell Curry. Haskell es conocido por su sistema de tipos expresivo, abstracciones de alto nivel y sintaxis concisa, lo que lo convierte en una herramienta poderosa tanto para la investigación académica como para el desarrollo de software práctico.

Características Clave de Haskell

  • Puramente Funcional: Las funciones en Haskell son puras, lo que significa que no tienen efectos secundarios. Esto conduce a un código más predecible y mantenible.
  • Tipado Estático Fuerte: Haskell utiliza un sistema de tipos robusto para detectar errores en tiempo de compilación, reduciendo los errores en tiempo de ejecución.
  • Evaluación Perezosa: Haskell evalúa expresiones solo cuando sus valores son necesarios, lo que puede llevar a mejoras en el rendimiento y la capacidad de trabajar con estructuras de datos infinitas.
  • Inmutabilidad: Los datos en Haskell son inmutables por defecto, lo que simplifica el razonamiento sobre el código y mejora la concurrencia.

Sintaxis y Conceptos Básicos de Haskell

Ejemplo de Hola Mundo

Comencemos con un simple programa de "¡Hola, Mundo!" en Haskell:

main :: IO ()
main = putStrLn "¡Hola, Mundo!"
  • main es el punto de entrada de un programa en Haskell.
  • putStrLn es una función que imprime una cadena seguida de una nueva línea.
  • IO () indica que main realiza una acción de entrada/salida y devuelve un tipo unitario ().

Tipos de Datos Básicos

Haskell tiene varios tipos de datos básicos:

  • Int: Entero de precisión fija.
  • Integer: Entero de precisión arbitraria.
  • Float: Número de punto flotante de precisión simple.
  • Double: Número de punto flotante de doble precisión.
  • Char: Un solo carácter.
  • Bool: Tipo booleano con valores True o False.
  • String: Una lista de caracteres.

Ejemplo:

miEntero :: Int
miEntero = 42

miFlotante :: Float
miFlotante = 3.14

miCaracter :: Char
miCaracter = 'a'

miBooleano :: Bool
miBooleano = True

miCadena :: String
miCadena = "¡Hola, Haskell!"

Funciones

Las funciones son ciudadanos de primera clase en Haskell. Así es como defines y usas funciones:

-- Función para sumar dos números
sumar :: Int -> Int -> Int
sumar x y = x + y

-- Usando la función
resultado = sumar 5 3  -- resultado es 8
  • sumar es una función que toma dos argumentos Int y devuelve un Int.
  • La firma de tipo Int -> Int -> Int indica que sumar toma dos valores Int y devuelve un Int.

Listas

Las listas son una estructura de datos fundamental en Haskell. Son homogéneas, lo que significa que todos los elementos deben ser del mismo tipo.

-- Una lista de enteros
numeros :: [Int]
numeros = [1, 2, 3, 4, 5]

-- Una lista de caracteres (una cadena)
caracteres :: [Char]
caracteres = ['H', 'a', 's', 'k', 'e', 'l', 'l']

Coincidencia de Patrones

La coincidencia de patrones es una característica poderosa en Haskell que te permite desestructurar tipos de datos.

-- Función para obtener la cabeza de una lista
cabezaDeLista :: [a] -> a
cabezaDeLista (x:_) = x
cabezaDeLista [] = error "Lista vacía"

-- Usando la función
primerElemento = cabezaDeLista [1, 2, 3]  -- primerElemento es 1
  • cabezaDeLista usa coincidencia de patrones para extraer la cabeza de una lista.
  • (x:_) coincide con una lista no vacía, vinculando x a la cabeza.
  • [] coincide con una lista vacía y lanza un error.

Conclusión

Haskell es un lenguaje poderoso y expresivo que enfatiza los principios de la programación funcional. Su fuerte sistema de tipos, evaluación perezosa e inmutabilidad lo convierten en una herramienta única y valiosa tanto para aplicaciones académicas como prácticas. Al comprender los conceptos básicos de Haskell, como su sintaxis, tipos de datos y características principales, puedes comenzar a apreciar la elegancia y el poder de la programación funcional.
