Introducción

En esta sección del curso de Analítica Digital, nos enfocaremos en ejercicios prácticos de análisis de datos. Estos ejercicios están diseñados para ayudarte a aplicar los conceptos teóricos aprendidos en situaciones del mundo real. Trabajaremos con diferentes tipos de datos y utilizaremos diversas herramientas y técnicas para analizarlos.

Objetivos

  • Aplicar técnicas de análisis de datos en situaciones prácticas.
  • Utilizar herramientas de software para el análisis de datos.
  • Interpretar y comunicar los resultados del análisis de datos.

Herramientas Necesarias

  • Python: Lenguaje de programación utilizado para la manipulación y análisis de datos.
  • Pandas: Biblioteca de Python para la manipulación de datos.
  • Matplotlib/Seaborn: Bibliotecas de Python para la visualización de datos.
  • Jupyter Notebook: Entorno interactivo para la programación en Python.

Ejercicio 1: Análisis Exploratorio de Datos (EDA)

Descripción

Realizaremos un análisis exploratorio de un conjunto de datos para entender su estructura, identificar patrones y detectar anomalías.

Pasos

  1. Cargar el conjunto de datos:

    • Utilizaremos un conjunto de datos de ejemplo, como el conjunto de datos de iris.

  2. Inspeccionar los datos:

    • Verificar las primeras filas del conjunto de datos.
    • Obtener un resumen estadístico de las variables.

  3. Visualizar los datos:

    • Crear gráficos de dispersión para visualizar las relaciones entre las variables.
    • Crear histogramas para entender la distribución de las variables.

Código de Ejemplo

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Cargar el conjunto de datos
df = sns.load_dataset('iris')

# Inspeccionar las primeras filas
print(df.head())

# Resumen estadístico
print(df.describe())

# Gráfico de dispersión
sns.pairplot(df, hue='species')
plt.show()

# Histogramas
df.hist(bins=20, figsize=(10, 8))
plt.show()

Explicación

  • Cargar el conjunto de datos: Utilizamos la función load_dataset de Seaborn para cargar el conjunto de datos de iris.
  • Inspeccionar las primeras filas: Utilizamos head() para ver las primeras filas del DataFrame.
  • Resumen estadístico: Utilizamos describe() para obtener un resumen estadístico de las variables.
  • Gráfico de dispersión: Utilizamos pairplot de Seaborn para crear gráficos de dispersión entre todas las variables.
  • Histogramas: Utilizamos hist de Pandas para crear histogramas de las variables.

Ejercicio 2: Limpieza de Datos

Descripción

Limpiaremos un conjunto de datos eliminando valores nulos y duplicados, y transformando datos categóricos.

Pasos

  1. Cargar el conjunto de datos:

    • Utilizaremos un conjunto de datos con valores nulos y duplicados.

  2. Eliminar valores nulos:

    • Identificar y eliminar filas con valores nulos.

  3. Eliminar duplicados:

    • Identificar y eliminar filas duplicadas.

  4. Transformar datos categóricos:

    • Convertir variables categóricas en variables numéricas.

Código de Ejemplo

# Cargar el conjunto de datos
df = pd.read_csv('data.csv')

# Eliminar valores nulos
df = df.dropna()

# Eliminar duplicados
df = df.drop_duplicates()

# Transformar datos categóricos
df['category'] = df['category'].astype('category').cat.codes

print(df.head())

Explicación

  • Cargar el conjunto de datos: Utilizamos read_csv de Pandas para cargar el conjunto de datos.
  • Eliminar valores nulos: Utilizamos dropna() para eliminar filas con valores nulos.
  • Eliminar duplicados: Utilizamos drop_duplicates() para eliminar filas duplicadas.
  • Transformar datos categóricos: Convertimos la columna categórica category en códigos numéricos utilizando astype('category').cat.codes.

Ejercicio 3: Análisis de Correlación

Descripción

Analizaremos la correlación entre variables en un conjunto de datos para identificar relaciones significativas.

Pasos

  1. Cargar el conjunto de datos:

    • Utilizaremos un conjunto de datos de ejemplo.

  2. Calcular la matriz de correlación:

    • Utilizar la función corr() de Pandas.

  3. Visualizar la matriz de correlación:

    • Crear un mapa de calor para visualizar las correlaciones.

Código de Ejemplo

# Cargar el conjunto de datos
df = pd.read_csv('data.csv')

# Calcular la matriz de correlación
correlation_matrix = df.corr()

# Visualizar la matriz de correlación
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm')
plt.show()

Explicación

  • Cargar el conjunto de datos: Utilizamos read_csv de Pandas para cargar el conjunto de datos.
  • Calcular la matriz de correlación: Utilizamos corr() para calcular la matriz de correlación.
  • Visualizar la matriz de correlación: Utilizamos heatmap de Seaborn para crear un mapa de calor de la matriz de correlación.
