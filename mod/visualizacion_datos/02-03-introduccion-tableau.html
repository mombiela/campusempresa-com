Introducción a Tableau
Siguiente ►

Objetivos del Módulo

En este módulo, aprenderás los conceptos básicos de Tableau, una de las herramientas más populares para la visualización de datos. Al finalizar, serás capaz de:

  • Comprender qué es Tableau y sus componentes principales.
  • Conocer las ventajas de usar Tableau para la visualización de datos.
  • Crear visualizaciones básicas utilizando Tableau.

¿Qué es Tableau?

Tableau es una herramienta de visualización de datos que permite a los usuarios crear gráficos interactivos y dashboards de manera sencilla y eficiente. Es ampliamente utilizada en la industria debido a su capacidad para manejar grandes volúmenes de datos y su facilidad de uso.

Componentes Principales de Tableau

  1. Tableau Desktop: Herramienta principal para la creación de visualizaciones y dashboards.
  2. Tableau Server: Plataforma para compartir visualizaciones y dashboards en una red interna.
  3. Tableau Online: Versión en la nube de Tableau Server.
  4. Tableau Public: Versión gratuita de Tableau Desktop con capacidades limitadas, ideal para compartir visualizaciones públicamente.
  5. Tableau Prep: Herramienta para la preparación y limpieza de datos antes de su visualización.

Ventajas de Usar Tableau

  • Interactividad: Permite crear visualizaciones interactivas que facilitan la exploración de datos.
  • Facilidad de Uso: Interfaz intuitiva que no requiere conocimientos avanzados de programación.
  • Integración de Datos: Capacidad para conectarse a múltiples fuentes de datos.
  • Rendimiento: Maneja grandes volúmenes de datos de manera eficiente.
  • Comunidad y Soporte: Amplia comunidad de usuarios y recursos de soporte disponibles.

Primeros Pasos con Tableau

Instalación de Tableau Desktop

  1. Descargar Tableau Desktop: Visita el sitio web oficial de Tableau y descarga la versión de prueba de Tableau Desktop.
  2. Instalación: Sigue las instrucciones del instalador para completar la instalación en tu computadora.

Interfaz de Tableau Desktop

Al abrir Tableau Desktop, encontrarás una interfaz dividida en varias secciones:

  • Panel de Datos: Aquí puedes conectar y gestionar tus fuentes de datos.
  • Hoja de Trabajo (Worksheet): Área principal donde crearás tus visualizaciones.
  • Panel de Marcas: Permite personalizar los elementos visuales de tus gráficos.
  • Panel de Filtros: Utilizado para aplicar filtros a tus datos.
  • Panel de Páginas: Permite crear animaciones y visualizaciones por etapas.

Ejemplo Práctico: Creación de una Visualización Básica

Paso 1: Conectar una Fuente de Datos

  1. Abrir Tableau Desktop.
  2. Conectar a una Fuente de Datos: En el panel de datos, selecciona una fuente de datos (por ejemplo, un archivo Excel).
Archivo Excel -> Seleccionar archivo -> Hoja de trabajo

Paso 2: Crear una Visualización

  1. Arrastrar Campos: Arrastra los campos desde el panel de datos hasta las estanterías de filas y columnas.

    Ejemplo: Arrastra "Ventas" a la estantería de columnas y "Fecha" a la estantería de filas.

  2. Seleccionar Tipo de Gráfico: En el panel de marcas, selecciona el tipo de gráfico que deseas crear (por ejemplo, gráfico de líneas).

Paso 3: Personalizar la Visualización

  1. Agregar Filtros: Arrastra campos al panel de filtros para refinar los datos mostrados.
  2. Modificar Colores y Etiquetas: Utiliza el panel de marcas para cambiar colores, agregar etiquetas y ajustar otros elementos visuales.

Paso 4: Guardar y Compartir

  1. Guardar el Proyecto: Guarda tu trabajo en un archivo .twbx (Tableau Workbook).
  2. Compartir: Publica tu visualización en Tableau Public o Tableau Server para compartirla con otros.

Ejercicio Práctico

Ejercicio 1: Crear un Gráfico de Barras

  1. Conectar a una Fuente de Datos: Utiliza un archivo Excel con datos de ventas.
  2. Crear un Gráfico de Barras:
    • Arrastra "Categoría de Producto" a la estantería de filas.
    • Arrastra "Ventas" a la estantería de columnas.
    • Selecciona el tipo de gráfico de barras en el panel de marcas.
  3. Personalizar:
    • Agrega etiquetas de datos para mostrar los valores de ventas.
    • Cambia los colores de las barras según la categoría de producto.
  4. Guardar y Compartir: Guarda tu visualización y publícala en Tableau Public.

Solución del Ejercicio

1. Conectar a una Fuente de Datos:
   - Archivo Excel -> Seleccionar archivo -> Hoja de trabajo

2. Crear un Gráfico de Barras:
   - Arrastra "Categoría de Producto" a la estantería de filas.
   - Arrastra "Ventas" a la estantería de columnas.
   - En el panel de marcas, selecciona "Gráfico de Barras".

3. Personalizar:
   - En el panel de marcas, arrastra "Ventas" a "Etiqueta" para mostrar los valores.
   - En el panel de marcas, arrastra "Categoría de Producto" a "Color" para cambiar los colores de las barras.

4. Guardar y Compartir:
   - Archivo -> Guardar como -> Seleccionar ubicación y nombre.
   - Archivo -> Publicar en Tableau Public -> Iniciar sesión y seguir las instrucciones.

Conclusión

En este módulo, hemos explorado los conceptos básicos de Tableau, sus componentes principales y las ventajas de utilizar esta herramienta para la visualización de datos. También hemos aprendido a crear una visualización básica y a personalizarla. En los próximos módulos, profundizaremos en técnicas avanzadas y otras herramientas de visualización de datos.
