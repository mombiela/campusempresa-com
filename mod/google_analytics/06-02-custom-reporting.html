Informes Personalizados
Siguiente ►

Introducción

Los informes personalizados en Google Analytics te permiten crear informes específicos que se ajusten a las necesidades particulares de tu negocio. A diferencia de los informes predeterminados, los informes personalizados te ofrecen la flexibilidad de elegir las dimensiones y métricas que deseas analizar, así como la forma en que se presentan los datos.

Conceptos Clave

Antes de profundizar en la creación de informes personalizados, es importante entender algunos conceptos clave:

  • Dimensiones: Atributos de tus datos. Ejemplos incluyen "Ciudad", "Navegador", "Página de destino".
  • Métricas: Medidas cuantitativas. Ejemplos incluyen "Usuarios", "Sesiones", "Tasa de rebote".
  • Filtros: Permiten incluir o excluir datos específicos en tu informe.
  • Segmentos: Subconjuntos de tus datos que puedes aplicar a tus informes para un análisis más detallado.

Creación de un Informe Personalizado

Paso 1: Acceder a la Sección de Informes Personalizados

  1. Inicia sesión en tu cuenta de Google Analytics.
  2. En el menú de la izquierda, selecciona Personalización y luego Informes personalizados.

Paso 2: Crear un Nuevo Informe

  1. Haz clic en el botón + Nuevo informe personalizado.
  2. Asigna un nombre a tu informe.
  3. Define las pestañas del informe. Cada pestaña puede tener diferentes dimensiones y métricas.

Paso 3: Configurar las Dimensiones y Métricas

  1. Dimensiones: Selecciona las dimensiones que deseas analizar. Por ejemplo, "Ciudad" y "Navegador".
  2. Métricas: Selecciona las métricas que deseas medir. Por ejemplo, "Usuarios" y "Tasa de rebote".
Dimensiones: Ciudad, Navegador
Métricas: Usuarios, Tasa de rebote

Paso 4: Aplicar Filtros y Segmentos

  1. Filtros: Puedes aplicar filtros para incluir o excluir datos específicos. Por ejemplo, puedes filtrar para incluir solo datos de una ciudad específica.
  2. Segmentos: Aplica segmentos para analizar subconjuntos de tus datos. Por ejemplo, puedes aplicar un segmento para analizar solo usuarios nuevos.

Paso 5: Guardar y Visualizar el Informe

  1. Haz clic en Guardar para crear tu informe personalizado.
  2. Una vez guardado, podrás visualizar el informe y analizar los datos según las dimensiones y métricas seleccionadas.

Código de Ejemplo

Aunque Google Analytics no requiere código para crear informes personalizados, aquí hay un ejemplo de cómo podrías usar la API de Google Analytics para obtener datos similares:

from googleapiclient.discovery import build
from oauth2client.service_account import ServiceAccountCredentials

# Autenticación
SCOPES = ['https://www.googleapis.com/auth/analytics.readonly']
KEY_FILE_LOCATION = 'path/to/your-service-account-key.json'
VIEW_ID = 'your-view-id'

credentials = ServiceAccountCredentials.from_json_keyfile_name(KEY_FILE_LOCATION, SCOPES)
analytics = build('analyticsreporting', 'v4', credentials=credentials)

# Solicitud de datos
response = analytics.reports().batchGet(
    body={
        'reportRequests': [
            {
                'viewId': VIEW_ID,
                'dateRanges': [{'startDate': '30daysAgo', 'endDate': 'today'}],
                'metrics': [{'expression': 'ga:users'}, {'expression': 'ga:bounceRate'}],
                'dimensions': [{'name': 'ga:city'}, {'name': 'ga:browser'}],
                'dimensionFilterClauses': [{
                    'filters': [{
                        'dimensionName': 'ga:country',
                        'operator': 'EXACT',
                        'expressions': ['United States']
                    }]
                }],
                'segments': [{
                    'segmentId': 'gaid::-3'  # Segmento de usuarios nuevos
                }]
            }]
    }
).execute()

# Procesar y mostrar los datos
for report in response.get('reports', []):
    for row in report.get('data', {}).get('rows', []):
        dimensions = row.get('dimensions', [])
        metrics = row.get('metrics', [])[0].get('values', [])
        print(f"Ciudad: {dimensions[0]}, Navegador: {dimensions[1]}, Usuarios: {metrics[0]}, Tasa de rebote: {metrics[1]}")

Conclusión

Los informes personalizados en Google Analytics te permiten adaptar los análisis a las necesidades específicas de tu negocio. Al comprender cómo configurar dimensiones, métricas, filtros y segmentos, puedes obtener insights más profundos y accionables. Practica creando diferentes tipos de informes personalizados para familiarizarte con las diversas opciones y capacidades que ofrece Google Analytics.
