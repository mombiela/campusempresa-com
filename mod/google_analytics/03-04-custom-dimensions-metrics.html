Dimensiones y Métricas Personalizadas
Siguiente ►

En este tema, aprenderás a configurar y utilizar dimensiones y métricas personalizadas en Google Analytics. Estas herramientas te permiten recopilar y analizar datos específicos que no están disponibles en los informes estándar de Google Analytics.

¿Qué son las Dimensiones y Métricas Personalizadas?

Dimensiones Personalizadas

Las dimensiones personalizadas son atributos adicionales que puedes definir para tus datos. Por ejemplo, si tienes un sitio web de comercio electrónico, podrías crear una dimensión personalizada para rastrear el tipo de usuario (por ejemplo, "Cliente Nuevo" o "Cliente Recurrente").

Métricas Personalizadas

Las métricas personalizadas son valores cuantitativos adicionales que puedes definir. Por ejemplo, podrías crear una métrica personalizada para rastrear el número de productos en el carrito de compras.

Diferencias Clave

Característica Dimensiones Personalizadas Métricas Personalizadas
Tipo de Datos Atributos cualitativos Valores cuantitativos
Ejemplos Tipo de Usuario, Categoría de Producto Número de Productos en el Carrito, Valor de Descuento

Configuración de Dimensiones y Métricas Personalizadas

Paso 1: Acceder a la Configuración de Propiedad

  1. Inicia sesión en tu cuenta de Google Analytics.
  2. Navega a la propiedad para la que deseas configurar dimensiones y métricas personalizadas.
  3. Ve a Administrador y selecciona la propiedad adecuada.

Paso 2: Crear una Dimensión Personalizada

  1. En la columna Propiedad, haz clic en Definiciones Personalizadas y luego en Dimensiones Personalizadas.
  2. Haz clic en + Nueva Dimensión Personalizada.
  3. Rellena los campos necesarios:
    • Nombre: Un nombre descriptivo para la dimensión.
    • Ámbito: Selecciona el ámbito adecuado (Usuario, Sesión, Hit, Producto).
    • Estado Activo: Asegúrate de que esté activado.
  4. Haz clic en Crear.

Paso 3: Crear una Métrica Personalizada

  1. En la columna Propiedad, haz clic en Definiciones Personalizadas y luego en Métricas Personalizadas.
  2. Haz clic en + Nueva Métrica Personalizada.
  3. Rellena los campos necesarios:
    • Nombre: Un nombre descriptivo para la métrica.
    • Ámbito: Selecciona el ámbito adecuado (Usuario, Sesión, Hit, Producto).
    • Formato: Selecciona el formato adecuado (Entero, Decimal, Tiempo).
    • Estado Activo: Asegúrate de que esté activado.
  4. Haz clic en Crear.

Implementación del Código de Seguimiento

Para que las dimensiones y métricas personalizadas funcionen, debes modificar el código de seguimiento de tu sitio web.

Ejemplo de Código de Seguimiento para Dimensiones Personalizadas

// Suponiendo que estás usando gtag.js
gtag('config', 'GA_TRACKING_ID', {
  'custom_map': {'dimension1': 'user_type'}
});

gtag('event', 'login', {'user_type': 'Cliente Recurrente'});

Ejemplo de Código de Seguimiento para Métricas Personalizadas

// Suponiendo que estás usando gtag.js
gtag('config', 'GA_TRACKING_ID', {
  'custom_map': {'metric1': 'cart_value'}
});

gtag('event', 'add_to_cart', {'cart_value': 100});

Ejercicio Práctico

Ejercicio 1: Crear y Configurar una Dimensión Personalizada

  1. Crea una dimensión personalizada llamada "Tipo de Usuario" con el ámbito "Usuario".
  2. Modifica el código de seguimiento de tu sitio web para enviar el valor "Cliente Nuevo" o "Cliente Recurrente" según corresponda.

Ejercicio 2: Crear y Configurar una Métrica Personalizada

  1. Crea una métrica personalizada llamada "Valor del Carrito" con el ámbito "Hit" y formato "Decimal".
  2. Modifica el código de seguimiento de tu sitio web para enviar el valor del carrito cada vez que un usuario añade un producto al carrito.

Soluciones

Solución al Ejercicio 1

  1. Crear la dimensión personalizada:

    • Nombre: Tipo de Usuario
    • Ámbito: Usuario
    • Estado Activo: Sí

  2. Código de seguimiento:

gtag('config', 'GA_TRACKING_ID', {
  'custom_map': {'dimension1': 'user_type'}
});

gtag('event', 'login', {'user_type': 'Cliente Nuevo'});

Solución al Ejercicio 2

  1. Crear la métrica personalizada:

    • Nombre: Valor del Carrito
    • Ámbito: Hit
    • Formato: Decimal
    • Estado Activo: Sí

  2. Código de seguimiento:

gtag('config', 'GA_TRACKING_ID', {
  'custom_map': {'metric1': 'cart_value'}
});

gtag('event', 'add_to_cart', {'cart_value': 100});

Conclusión

En esta sección, has aprendido a configurar y utilizar dimensiones y métricas personalizadas en Google Analytics. Estas herramientas te permiten recopilar datos específicos y detallados que no están disponibles en los informes estándar, lo que te proporciona una visión más profunda y personalizada de tus usuarios y su comportamiento. En el próximo tema, exploraremos cómo utilizar filtros para refinar y segmentar tus datos de manera más efectiva.
