Configuración del Código de Seguimiento
Entendiendo la recopilación de datos ►

Configurar el código de seguimiento es un paso fundamental para usar Google Analytics y monitorear y analizar el tráfico y el comportamiento de los usuarios en tu sitio web. Este tema te guiará a través del proceso desde un nivel principiante hasta uno avanzado.

Introducción al Código de Seguimiento de Google Analytics

  • ¿Qué es el Código de Seguimiento de Google Analytics?

    • Un fragmento de código JavaScript que recopila datos sobre las interacciones de los usuarios en tu sitio web.
    • Envía estos datos a Google Analytics para su procesamiento y generación de informes.

  • ¿Por qué es importante?

    • Te permite rastrear el comportamiento de los usuarios, medir la efectividad de tus esfuerzos de marketing y tomar decisiones basadas en datos.

Comenzando con el Código de Seguimiento de Google Analytics

Creando una Cuenta de Google Analytics

  1. Regístrate en Google Analytics:

    • Visita el sitio web de Google Analytics.
    • Inicia sesión con tu cuenta de Google o crea una nueva.
    • Sigue las indicaciones para configurar tu cuenta, propiedad y vista.

  2. Obtén el ID de Seguimiento:

    • Después de configurar tu propiedad, recibirás un ID de Seguimiento único (por ejemplo, UA-XXXXXXXXX-X).

Añadiendo el Código de Seguimiento a tu Sitio Web

  1. Localiza el Código de Seguimiento:

    • En tu cuenta de Google Analytics, navega a Admin > Property > Tracking Info > Tracking Code.
    • Copia el fragmento de código JavaScript proporcionado.

  2. Inserta el Código de Seguimiento:

    • Pega el fragmento de código de seguimiento en la sección <head> de cada página que desees rastrear.
<!DOCTYPE html>
<html>
<head>
  <title>Mi Sitio Web</title>
  <!-- Etiqueta global del sitio (gtag.js) - Google Analytics -->
  <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
  <script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'UA-XXXXXXXXX-X');
  </script>
</head>
<body>
  <!-- Contenido de tu sitio web -->
</body>
</html>

Verificando la Instalación

  • Informes en Tiempo Real:
    • Revisa los informes en Tiempo Real en Google Analytics para ver si tu código de seguimiento está funcionando correctamente.
    • Visita tu sitio web y busca tu visita en la vista general en Tiempo Real.

Personalización Avanzada del Código de Seguimiento

Rastreo de Múltiples Dominios

  • Rastreo de Dominio Cruzado:
    • Útil si tienes múltiples dominios y deseas rastrear las interacciones de los usuarios entre ellos.
    • Modifica el código de seguimiento para incluir el plugin linker.
<script>
  gtag('config', 'UA-XXXXXXXXX-X', {
    'linker': {
      'domains': ['example1.com', 'example2.com']
    }
  });
</script>

Seguimiento Mejorado de Comercio Electrónico

  • Seguimiento de Comercio Electrónico:
    • Rastrea transacciones, vistas de productos y otras interacciones de comercio electrónico.
    • Requiere código adicional que debe ser añadido a tus páginas de comercio electrónico.
<script>
  gtag('event', 'purchase', {
    "transaction_id": "24.031608523954162",
    "affiliation": "Tienda en Línea",
    "value": 23.07,
    "currency": "USD",
    "tax": 1.24,
    "shipping": 0,
    "items": [
      {
        "id": "P12345",
        "name": "Camiseta Android Warhol",
        "category": "Ropa/Camisetas",
        "quantity": 1,
        "price": 23.07
      }
    ]
  });
</script>

Dimensiones y Métricas Personalizadas

  • Dimensiones y Métricas Personalizadas:
    • Te permiten recopilar datos que Google Analytics no rastrea automáticamente.
    • Define dimensiones y métricas personalizadas en la configuración de tu propiedad de Google Analytics.
<script>
  gtag('event', 'custom_event', {
    'custom_dimension': 'dimension_value',
    'custom_metric': 123
  });
</script>

Conclusión

Configurar el código de seguimiento de Google Analytics es un paso crucial para aprovechar el poder de Google Analytics. Siguiendo los pasos descritos en este tema, puedes comenzar a recopilar datos valiosos sobre el rendimiento de tu sitio web y el comportamiento de los usuarios. A medida que te sientas más cómodo con los conceptos básicos, puedes explorar técnicas de seguimiento avanzadas para obtener conocimientos más profundos y tomar decisiones más informadas.
