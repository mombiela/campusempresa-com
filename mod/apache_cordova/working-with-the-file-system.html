Introducción

Apache Cordova te permite construir aplicaciones móviles usando HTML, CSS y JavaScript. Una de las características esenciales de cualquier aplicación móvil es la capacidad de interactuar con el sistema de archivos. Este tema te guiará a través de los conceptos básicos de trabajar con el sistema de archivos en Cordova, desde niveles principiantes hasta avanzados.

Configuración

Antes de comenzar a trabajar con el sistema de archivos, asegúrate de tener el entorno de Cordova configurado. Si no es así, sigue estos pasos:

  1. Instala Node.js y npm.
  2. Instala Cordova globalmente usando npm: 
    npm install -g cordova
  3. Crea un nuevo proyecto de Cordova: 
    cordova create myApp
cd myApp
  4. Añade una plataforma (por ejemplo, Android): 
    cordova platform add android

Instalando el Plugin de Archivos

Para trabajar con el sistema de archivos, necesitas instalar el plugin de Archivos de Cordova:

cordova plugin add cordova-plugin-file

Conceptos Básicos

  • Sistema de Archivos: La estructura que almacena archivos y directorios.
  • Archivo: Una pieza individual de datos almacenada en el sistema de archivos.
  • Directorio: Un contenedor que guarda archivos y otros directorios.

Accediendo al Sistema de Archivos

Para acceder al sistema de archivos, necesitas usar el objeto cordova.file, que proporciona varias propiedades que representan diferentes ubicaciones del sistema de archivos.

Ejemplo: Accediendo al Directorio Raíz

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        console.log("Directorio raíz:", directoryEntry);
    }, function(error) {
        console.error("Error al acceder al sistema de archivos:", error);
    });
});
  • cordova.file.dataDirectory: Representa el directorio de datos de la aplicación.
  • resolveLocalFileSystemURL: Resuelve una URL del sistema de archivos a una entrada de directorio.

Leyendo y Escribiendo Archivos

Escribiendo un Archivo

Para escribir un archivo, necesitas crear o obtener una entrada de archivo y luego escribir datos en él.

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("example.txt", { create: true, exclusive: false }, function(fileEntry) {
            fileEntry.createWriter(function(fileWriter) {
                var dataObj = new Blob(['¡Hola, mundo!'], { type: 'text/plain' });

                fileWriter.write(dataObj);

                fileWriter.onwriteend = function() {
                    console.log("Archivo escrito exitosamente.");
                };

                fileWriter.onerror = function(e) {
                    console.error("Error al escribir el archivo:", e);
                };
            });
        });
    });
});
  • getFile: Obtiene una entrada de archivo, creándola si no existe.
  • createWriter: Crea un escritor de archivos para escribir datos en el archivo.

Leyendo un Archivo

Para leer un archivo, necesitas obtener una entrada de archivo y luego leer su contenido.

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("example.txt", { create: false }, function(fileEntry) {
            fileEntry.file(function(file) {
                var reader = new FileReader();

                reader.onloadend = function() {
                    console.log("Contenido del archivo:", this.result);
                };

                reader.readAsText(file);
            });
        });
    });
});
  • FileReader: Lee el contenido de un archivo.

Temas Avanzados

Gestión de Directorios

Puedes crear, leer y eliminar directorios de manera similar a los archivos.

Creando un Directorio

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getDirectory("newDir", { create: true }, function(newDirEntry) {
            console.log("Directorio creado:", newDirEntry);
        });
    });
});

Eliminando un Directorio

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getDirectory("newDir", { create: false }, function(newDirEntry) {
            newDirEntry.removeRecursively(function() {
                console.log("Directorio eliminado.");
            });
        });
    });
});

Manejo de Errores

Siempre maneja los errores para asegurar que tu aplicación pueda recuperarse de manera elegante de los problemas.

function onError(error) {
    console.error("Código de error:", error.code);
}

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("nonexistent.txt", { create: false }, null, onError);
    });
});

Resumen

Trabajar con el sistema de archivos en Apache Cordova implica entender cómo acceder a diferentes ubicaciones del sistema de archivos, leer y escribir archivos, y gestionar directorios. Al dominar estos conceptos, puedes crear aplicaciones móviles robustas que manejen eficientemente las operaciones de archivos.

  • Puntos Clave:
    • Instala el plugin de Archivos de Cordova.
    • Usa cordova.file para acceder a diferentes ubicaciones del sistema de archivos.
    • Lee y escribe archivos usando FileReader y FileWriter.
    • Gestiona directorios creando, leyendo y eliminándolos.
    • Siempre maneja los errores para asegurar una experiencia de usuario fluida.

Siguiendo estas pautas y ejemplos, deberías estar bien equipado para manejar operaciones del sistema de archivos en tus aplicaciones de Cordova.
