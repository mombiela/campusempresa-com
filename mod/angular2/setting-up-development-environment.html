Configuración del Entorno de Desarrollo
Tu Primera Aplicación Angular ►

Configurar el entorno de desarrollo es el primer paso crucial en tu camino para dominar Angular 2+. Esta sección te guiará a través de las herramientas y configuraciones necesarias para comenzar con el desarrollo en Angular.

Requisitos Previos

Antes de configurar Angular, asegúrate de tener los siguientes requisitos previos instalados en tu sistema:

  • Node.js y npm: Angular requiere Node.js y npm (Node Package Manager). Puedes descargarlos e instalarlos desde el sitio web oficial de Node.js.

Verificación de la Instalación de Node.js y npm

Para verificar si Node.js y npm están instalados, abre tu terminal o símbolo del sistema y ejecuta los siguientes comandos:

node -v
npm -v

Deberías ver los números de versión de Node.js y npm, respectivamente.

Instalación de Angular CLI

Angular CLI (Command Line Interface) es una herramienta poderosa que simplifica el proceso de creación y gestión de aplicaciones Angular. Para instalar Angular CLI globalmente en tu sistema, usa el siguiente comando:

npm install -g @angular/cli

Verificación de la Instalación de Angular CLI

Después de la instalación, verifica que Angular CLI esté instalado correctamente ejecutando:

ng version

Este comando mostrará la versión de Angular CLI y otros paquetes relacionados.

Creación de un Nuevo Proyecto Angular

Con Angular CLI instalado, ahora puedes crear un nuevo proyecto Angular. Usa el siguiente comando para crear un nuevo proyecto llamado my-angular-app:

ng new my-angular-app

Opciones de Configuración del Proyecto

Durante el proceso de creación del proyecto, Angular CLI te pedirá que elijas algunas opciones de configuración:

  • ¿Te gustaría agregar enrutamiento de Angular?: Elige Sí o No dependiendo de si necesitas enrutamiento en tu aplicación.
  • ¿Qué formato de hoja de estilos te gustaría usar?: Selecciona tu formato de hoja de estilos preferido (CSS, SCSS, SASS, LESS, Stylus).

Ejecución de la Aplicación Angular

Navega a tu directorio de proyecto y comienza el servidor de desarrollo usando los siguientes comandos:

cd my-angular-app
ng serve

Acceso a la Aplicación

Una vez que el servidor esté en funcionamiento, abre tu navegador web y navega a http://localhost:4200/. Deberías ver la página de bienvenida predeterminada de Angular.

Herramientas y Extensiones de Desarrollo

Para mejorar tu experiencia de desarrollo, considera instalar las siguientes herramientas y extensiones:

  • Visual Studio Code (VS Code): Un editor de código popular con excelente soporte para Angular.
  • Angular Language Service: Una extensión para VS Code que proporciona edición enriquecida e IntelliSense para plantillas Angular.

Instalación de Angular Language Service en VS Code

  1. Abre VS Code.
  2. Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana.
  3. Busca "Angular Language Service" y haz clic en Instalar.

Resumen

En esta sección, has aprendido cómo configurar el entorno de desarrollo para Angular 2+. Instalaste Node.js y npm, configuraste Angular CLI, creaste un nuevo proyecto Angular y ejecutaste el servidor de desarrollo. Además, exploraste algunas herramientas y extensiones útiles para mejorar tu experiencia de codificación.

Al asegurarte de que tu entorno de desarrollo esté correctamente configurado, ahora estás listo para profundizar en Angular y comenzar a construir aplicaciones web poderosas.
