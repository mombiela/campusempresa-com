Carga Perezosa de Módulos en Angular 2+
Angular Universal (Renderizado del Lado del Servidor) ►

La carga perezosa es un patrón de diseño comúnmente utilizado en aplicaciones Angular para cargar componentes de JavaScript de manera asincrónica cuando se activa una ruta específica. Esto puede mejorar significativamente el rendimiento de tu aplicación al reducir el tiempo de carga inicial.

Conceptos Clave

  • Carga Perezosa: Cargar módulos solo cuando se necesitan, en lugar de al inicio de la aplicación.
  • Módulos: Los módulos de Angular (NgModule) se utilizan para agrupar componentes, directivas, pipes y servicios relacionados.
  • Rutas: Definen los caminos de navegación en tu aplicación. La carga perezosa a menudo se implementa utilizando el enrutador de Angular.

¿Por Qué Usar Carga Perezosa?

  • Rendimiento: Reduce el tiempo de carga inicial de la aplicación al cargar solo los módulos necesarios.
  • Escalabilidad: Ayuda a gestionar aplicaciones grandes dividiéndolas en piezas más pequeñas y manejables.
  • Optimización de Recursos: Carga recursos solo cuando se necesitan, lo que puede ahorrar ancho de banda y mejorar la experiencia del usuario.

Configuración de la Carga Perezosa

Paso 1: Crear un Módulo de Funcionalidad

Primero, crea un nuevo módulo de funcionalidad que quieras cargar de manera perezosa.

ng generate module feature --route feature --module app.module

Este comando creará un nuevo módulo y configurará el enrutador para cargarlo de manera perezosa.

Paso 2: Configurar Rutas para la Carga Perezosa

En tu app-routing.module.ts, configura las rutas para cargar de manera perezosa el módulo de funcionalidad.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: 'feature',
    loadChildren: () => import('./feature/feature.module').then(m => m.FeatureModule)
  }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Paso 3: Definir Rutas en el Módulo de Funcionalidad

En tu feature-routing.module.ts, define las rutas para el módulo de funcionalidad.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { FeatureComponent } from './feature.component';

const routes: Routes = [
  { path: '', component: FeatureComponent }
];

@NgModule({
  imports: [RouterModule.forChild(routes)],
  exports: [RouterModule]
})
export class FeatureRoutingModule { }

Paso 4: Crear el Componente de Funcionalidad

Crea el componente que será parte del módulo de funcionalidad.

ng generate component feature/feature

Paso 5: Actualizar el Módulo de Funcionalidad

Actualiza el feature.module.ts para incluir el nuevo componente y el módulo de enrutamiento.

import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { FeatureComponent } from './feature.component';
import { FeatureRoutingModule } from './feature-routing.module';

@NgModule({
  declarations: [FeatureComponent],
  imports: [
    CommonModule,
    FeatureRoutingModule
  ]
})
export class FeatureModule { }

Ejemplo: Carga Perezosa de un Módulo de Panel de Control

Vamos a crear un ejemplo más concreto cargando de manera perezosa un DashboardModule.

Paso 1: Generar el Módulo de Panel de Control

ng generate module dashboard --route dashboard --module app.module

Paso 2: Configurar Rutas para la Carga Perezosa

Actualiza app-routing.module.ts:

const routes: Routes = [
  {
    path: 'dashboard',
    loadChildren: () => import('./dashboard/dashboard.module').then(m => m.DashboardModule)
  }
];

Paso 3: Definir Rutas en el Módulo de Panel de Control

Actualiza dashboard-routing.module.ts:

const routes: Routes = [
  { path: '', component: DashboardComponent }
];

Paso 4: Crear el Componente de Panel de Control

ng generate component dashboard/dashboard

Paso 5: Actualizar el Módulo de Panel de Control

Actualiza dashboard.module.ts:

@NgModule({
  declarations: [DashboardComponent],
  imports: [
    CommonModule,
    DashboardRoutingModule
  ]
})
export class DashboardModule { }

Conclusión

La carga perezosa es una característica poderosa en Angular que puede ayudar a mejorar el rendimiento y la escalabilidad de tu aplicación. Al cargar módulos solo cuando se necesitan, puedes reducir el tiempo de carga inicial y optimizar el uso de recursos. Esta guía proporcionó un enfoque paso a paso para implementar la carga perezosa en Angular, desde la creación de un módulo de funcionalidad hasta la configuración de rutas y la actualización de módulos. Con estas técnicas, puedes construir aplicaciones Angular más eficientes y receptivas.
