¿Qué es TypeScript?

TypeScript es un superconjunto de JavaScript que añade tipos estáticos. Es desarrollado y mantenido por Microsoft y ha ganado una popularidad significativa en la comunidad de desarrollo, especialmente para aplicaciones a gran escala. TypeScript ayuda a detectar errores temprano a través de un sistema de tipos y hace que el desarrollo en JavaScript sea más robusto y mantenible.

Características Clave de TypeScript

  • Tipado Estático: Te permite definir tipos para variables, parámetros de funciones y valores de retorno.
  • Inferencia de Tipos: Infere automáticamente tipos cuando no están definidos explícitamente.
  • Interfaces: Definen contratos dentro de tu código.
  • Clases y Herencia: Soporte para programación orientada a objetos.
  • Módulos: Organizan el código en componentes reutilizables.
  • Herramientas: Soporte mejorado en IDEs para navegación de código, refactorización y depuración.

Configuración de TypeScript

Para comenzar a usar TypeScript, necesitas instalarlo a través de npm (Node Package Manager):

npm install -g typescript

Luego puedes compilar archivos TypeScript a JavaScript usando el comando tsc:

tsc filename.ts

Sintaxis Básica de TypeScript

Variables y Tipos

TypeScript te permite definir tipos para variables:

let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];

Funciones

Puedes especificar tipos para parámetros de funciones y valores de retorno:

function add(x: number, y: number): number {
    return x + y;
}

Interfaces

Las interfaces definen la estructura de un objeto:

interface Person {
    firstName: string;
    lastName: string;
}

function greet(person: Person) {
    return "Hello, " + person.firstName + " " + person.lastName;
}

let user = { firstName: "John", lastName: "Doe" };
console.log(greet(user));

Clases

TypeScript soporta clases y herencia:

class Animal {
    name: string;
    constructor(name: string) {
        this.name = name;
    }
    move(distance: number = 0) {
        console.log(`${this.name} moved ${distance}m.`);
    }
}

class Dog extends Animal {
    bark() {
        console.log('Woof! Woof!');
    }
}

let dog = new Dog("Rex");
dog.bark();
dog.move(10);

Módulos

Los módulos ayudan a organizar el código:

// math.ts
export function add(x: number, y: number): number {
    return x + y;
}

// app.ts
import { add } from './math';
console.log(add(5, 3));

TypeScript en Angular

Angular está construido con TypeScript, y entender TypeScript es crucial para desarrollar aplicaciones Angular. Aquí hay algunos puntos clave donde se usa TypeScript en Angular:

  • Componentes: Definir clases de componentes con TypeScript.
  • Servicios: Usar TypeScript para definir servicios y sus métodos.
  • Inyección de Dependencias: TypeScript ayuda a definir tipos para dependencias.
  • Enrutamiento: Definir rutas y sus parámetros usando TypeScript.

Ejemplo: Componente Angular con TypeScript

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `<h1>{{title}}</h1>`,
})
export class AppComponent {
  title: string = 'Hello, Angular!';
}

Conclusión

TypeScript es una herramienta poderosa que mejora JavaScript añadiendo tipos estáticos, interfaces, clases y módulos. Es esencial para desarrollar aplicaciones Angular robustas y mantenibles. Al entender los conceptos básicos de TypeScript, estarás mejor preparado para trabajar con Angular y aprovechar su máximo potencial.
