¿Qué es la Programación?

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora cómo realizar una tarea. Estas instrucciones se escriben en un lenguaje de programación. C es uno de los lenguajes de programación más utilizados debido a su eficiencia y control sobre los recursos del sistema.

¿Por qué Aprender C?

  • Fundamento para Otros Lenguajes: Muchos lenguajes modernos como C++, Java y Python tienen sus raíces en C.
  • Programación a Nivel de Sistema: C se utiliza para la programación de sistemas, incluidos sistemas operativos y sistemas embebidos.
  • Rendimiento: C proporciona acceso de bajo nivel a la memoria y a los procesos del sistema, lo que lo hace altamente eficiente.

Conceptos Básicos en C

Variables y Tipos de Datos

Las variables se utilizan para almacenar datos que pueden ser manipulados por el programa. Cada variable en C debe ser declarada con un tipo de dato específico.

  • int: Números enteros
  • float: Números de punto flotante
  • char: Caracteres individuales
#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char initial = 'A';

    printf("Edad: %d\n", age);
    printf("Altura: %.1f\n", height);
    printf("Inicial: %c\n", initial);

    return 0;
}

Operadores

Los operadores son símbolos que realizan operaciones sobre variables y valores.

  • Operadores Aritméticos: +, -, *, /, %
  • Operadores Relacionales: ==, !=, >, <, >=, <=
  • Operadores Lógicos: &&, ||, !
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int sum = a + b;
    int isEqual = (a == b);

    printf("Suma: %d\n", sum);
    printf("¿Son iguales? %d\n", isEqual);

    return 0;
}

Estructuras de Control

Las estructuras de control se utilizan para controlar el flujo de ejecución en un programa.

  • if-else: Declaraciones condicionales
  • for: Bucle con un número conocido de iteraciones
  • while: Bucle con un número desconocido de iteraciones
  • do-while: Bucle que se ejecuta al menos una vez
#include <stdio.h>

int main() {
    int number = 10;

    // Declaración if-else
    if (number > 0) {
        printf("El número es positivo\n");
    } else {
        printf("El número no es positivo\n");
    }

    // Bucle for
    for (int i = 0; i < 5; i++) {
        printf("i: %d\n", i);
    }

    // Bucle while
    int count = 0;
    while (count < 3) {
        printf("Cuenta: %d\n", count);
        count++;
    }

    // Bucle do-while
    int j = 0;
    do {
        printf("j: %d\n", j);
        j++;
    } while (j < 2);

    return 0;
}

Funciones

Las funciones son bloques de código que realizan una tarea específica y pueden ser reutilizadas.

  • Declaración: Especifica el nombre de la función, el tipo de retorno y los parámetros.
  • Definición: Contiene el código real que se va a ejecutar.
  • Llamada: Ejecuta la función.
#include <stdio.h>

// Declaración de la función
void greet();

// Función principal
int main() {
    greet(); // Llamada a la función
    return 0;
}

// Definición de la función
void greet() {
    printf("¡Hola, Mundo!\n");
}

Conclusión

En este tema introductorio, hemos cubierto los conceptos básicos de la programación en C, incluyendo variables, tipos de datos, operadores, estructuras de control y funciones. Estos conceptos fundamentales son los bloques de construcción para temas más avanzados en la programación en C. Comprender estos conceptos básicos proporcionará una base sólida para tu viaje hacia técnicas de programación más complejas y poderosas.
