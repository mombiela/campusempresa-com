Operadores en C

Introducción

Los operadores son símbolos especiales en C que realizan operaciones sobre variables y valores. Son fundamentales para manipular datos y realizar cálculos en un programa. Esta sección cubrirá los diversos tipos de operadores disponibles en C, su uso y ejemplos para ilustrar su funcionalidad.

Tipos de Operadores

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • Adición (+): Suma dos operandos.
  • Sustracción (-): Resta el segundo operando del primero.
  • Multiplicación (*): Multiplica dos operandos.
  • División (/): Divide el primer operando por el segundo.
  • Módulo (%): Devuelve el resto de una operación de división.
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("Adición: %d\n", a + b);       // Salida: 13
    printf("Sustracción: %d\n", a - b);    // Salida: 7
    printf("Multiplicación: %d\n", a * b); // Salida: 30
    printf("División: %d\n", a / b);       // Salida: 3
    printf("Módulo: %d\n", a % b);        // Salida: 1
    return 0;
}

Operadores Relacionales

Los operadores relacionales comparan dos valores y devuelven un resultado booleano (verdadero o falso).

  • Igual a (==): Verifica si dos operandos son iguales.
  • No igual a (!=): Verifica si dos operandos no son iguales.
  • Mayor que (>): Verifica si el primer operando es mayor que el segundo.
  • Menor que (<): Verifica si el primer operando es menor que el segundo.
  • Mayor o igual a (>=): Verifica si el primer operando es mayor o igual al segundo.
  • Menor o igual a (<=): Verifica si el primer operando es menor o igual al segundo.
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    printf("a == b: %d\n", a == b); // Salida: 0 (falso)
    printf("a != b: %d\n", a != b); // Salida: 1 (verdadero)
    printf("a > b: %d\n", a > b);   // Salida: 0 (falso)
    printf("a < b: %d\n", a < b);   // Salida: 1 (verdadero)
    printf("a >= b: %d\n", a >= b); // Salida: 0 (falso)
    printf("a <= b: %d\n", a <= b); // Salida: 1 (verdadero)
    return 0;
}

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones o para invertir la lógica de una condición.

  • AND lógico (&&): Devuelve verdadero si ambos operandos son verdaderos.
  • OR lógico (||): Devuelve verdadero si al menos uno de los operandos es verdadero.
  • NOT lógico (!): Invierte el valor booleano del operando.
#include <stdio.h>

int main() {
    int a = 1, b = 0;
    printf("a && b: %d\n", a && b); // Salida: 0 (falso)
    printf("a || b: %d\n", a || b); // Salida: 1 (verdadero)
    printf("!a: %d\n", !a);         // Salida: 0 (falso)
    printf("!b: %d\n", !b);         // Salida: 1 (verdadero)
    return 0;
}

Operadores a Nivel de Bits

Los operadores a nivel de bits realizan operaciones sobre las representaciones binarias de los enteros.

  • AND (&): Realiza una operación AND a nivel de bits.
  • OR (|): Realiza una operación OR a nivel de bits.
  • XOR (^): Realiza una operación XOR a nivel de bits.
  • NOT (~): Realiza una operación NOT a nivel de bits.
  • Desplazamiento a la izquierda (<<): Desplaza bits a la izquierda.
  • Desplazamiento a la derecha (>>): Desplaza bits a la derecha.
#include <stdio.h>

int main() {
    int a = 5;  // Binario: 0101
    int b = 3;  // Binario: 0011
    printf("a & b: %d\n", a & b); // Salida: 1 (0001)
    printf("a | b: %d\n", a | b); // Salida: 7 (0111)
    printf("a ^ b: %d\n", a ^ b); // Salida: 6 (0110)
    printf("~a: %d\n", ~a);       // Salida: -6 (invierte todos los bits)
    printf("a << 1: %d\n", a << 1); // Salida: 10 (1010)
    printf("a >> 1: %d\n", a >> 1); // Salida: 2 (0010)
    return 0;
}

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. También pueden realizar operaciones y asignar el resultado a la variable.

  • Asignación (=): Asigna el operando derecho al operando izquierdo.
  • Suma y asigna (+=): Suma el operando derecho al operando izquierdo y asigna el resultado al operando izquierdo.
  • Resta y asigna (-=): Resta el operando derecho del operando izquierdo y asigna el resultado al operando izquierdo.
  • Multiplica y asigna (*=): Multiplica el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.
  • Divide y asigna (/=): Divide el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.
  • Módulo y asigna (%=): Toma el módulo usando el operando izquierdo y el operando derecho y asigna el resultado al operando izquierdo.
#include <stdio.h>

int main() {
    int a = 10;
    a += 5; // Equivalente a a = a + 5
    printf("a += 5: %d\n", a); // Salida: 15
    a -= 3; // Equivalente a a = a - 3
    printf("a -= 3: %d\n", a); // Salida: 12
    a *= 2; // Equivalente a a = a * 2
    printf("a *= 2: %d\n", a); // Salida: 24
    a /= 4; // Equivalente a a = a / 4
    printf("a /= 4: %d\n", a); // Salida: 6
    a %= 5; // Equivalente a a = a % 5
    printf("a %%= 5: %d\n", a); // Salida: 1
    return 0;
}

Operadores de Incremento y Decremento

Estos operadores se utilizan para aumentar o disminuir el valor de una variable en uno.

  • Incremento (++): Aumenta el valor en uno.
  • Decremento (--): Disminuye el valor en uno.
#include <stdio.h>

int main() {
    int a = 10;
    a++; // Equivalente a a = a + 1
    printf("a++: %d\n", a); // Salida: 11
    a--; // Equivalente a a = a - 1
    printf("a--: %d\n", a); // Salida: 10
    return 0;
}

Operador Condicional (Ternario)

El operador condicional es una forma abreviada de una declaración if-else. Toma tres operandos.

  • Sintaxis: condición ? expresión_si_verdadero : expresión_si_falso
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int max = (a > b) ? a : b; // Si a > b, max = a; de lo contrario, max = b
    printf("Valor máximo: %d\n", max); // Salida: 20
    return 0;
}

Conclusión

Los operadores en C son herramientas esenciales para realizar una amplia gama de operaciones, desde aritmética básica hasta manipulaciones lógicas y a nivel de bits complejas. Entender cómo usar estos operadores de manera efectiva es crucial para escribir programas en C eficientes y funcionales. Practica usando estos operadores en diferentes escenarios para volverte competente en su aplicación.
