Introducción

Entender la sintaxis y estructura básica de C es crucial para escribir programas eficientes y sin errores. Esta sección cubrirá los elementos fundamentales de la programación en C, incluyendo la estructura de un programa en C, tipos de datos, variables y operaciones básicas de entrada/salida.

Estructura de un Programa en C

Un programa en C típicamente consiste en las siguientes partes:

  • Comandos del Preprocesador
  • Funciones
  • Variables
  • Declaraciones y Expresiones
  • Comentarios

Ejemplo de un Programa Simple en C

#include <stdio.h> // Comando del Preprocesador

// Función Principal
int main() {
    // Declaración de Variable
    int number;

    // Declaración de Impresión
    printf("Enter a number: ");

    // Declaración de Entrada
    scanf("%d", &number);

    // Declaración de Salida
    printf("You entered: %d\n", number);

    return 0; // Declaración de Retorno
}

Explicación:

  • #include <stdio.h>: Este es un comando del preprocesador que le dice al compilador que incluya la biblioteca estándar de entrada-salida antes de compilar el programa.
  • int main() { ... }: Esta es la función principal donde comienza la ejecución del programa.
  • int number;: Esto declara una variable entera llamada number.
  • printf("Enter a number: ");: Esto imprime el mensaje "Enter a number:" en la consola.
  • scanf("%d", &number);: Esto lee una entrada entera del usuario y la almacena en la variable number.
  • printf("You entered: %d\n", number);: Esto imprime el mensaje "You entered:" seguido del valor de number.
  • return 0;: Esta declaración termina la función principal y devuelve el valor 0 al proceso que llama.

Tipos de Datos y Variables

En C, los tipos de datos especifican el tipo de datos que una variable puede contener. Los tipos de datos comunes incluyen:

  • int: Tipo entero
  • float: Tipo de punto flotante
  • char: Tipo de carácter
  • double: Tipo de punto flotante de doble precisión

Declaración de Variables

Las variables deben ser declaradas antes de ser usadas. La sintaxis para declarar una variable es:

data_type variable_name;

Ejemplo:

int age;
float salary;
char grade;
double distance;

Operaciones Básicas de Entrada/Salida

C proporciona funciones estándar para operaciones de entrada y salida, principalmente a través de la biblioteca stdio.h.

Entrada usando `scanf`

La función scanf se usa para leer entrada formateada desde la entrada estándar (teclado).

int number;
scanf("%d", &number); // Lee un entero y lo almacena en la variable 'number'

Salida usando `printf`

La función printf se usa para imprimir salida formateada en la salida estándar (consola).

int number = 10;
printf("The number is: %d\n", number); // Imprime "The number is: 10"

Comentarios

Los comentarios se usan para explicar el código y son ignorados por el compilador. Hay dos tipos de comentarios en C:

  • Comentarios de una sola línea: Comienzan con //
  • Comentarios de múltiples líneas: Se encierran dentro de /* ... */

Ejemplo:

// Este es un comentario de una sola línea

/*
Este es un
comentario de múltiples líneas
*/

Resumen

En esta sección, cubrimos la sintaxis y estructura básica de un programa en C. Aprendimos sobre los componentes esenciales como comandos del preprocesador, funciones, variables y declaraciones. También exploramos operaciones básicas de entrada/salida usando scanf y printf, y el uso de comentarios para hacer el código más legible. Entender estos fundamentos es el primer paso hacia convertirse en un programador competente en C.
