Introducción

Los argumentos de línea de comandos son una forma de proporcionar entrada a un programa desde la línea de comandos. Permiten a los usuarios especificar parámetros u opciones al ejecutar un programa, haciéndolo más flexible y dinámico.

Conceptos Clave

  • Interfaz de Línea de Comandos (CLI): Una interfaz basada en texto utilizada para interactuar con software y sistemas operativos.
  • Argumentos: Valores proporcionados al programa en tiempo de ejecución a través de la línea de comandos.
  • argc (Conteo de Argumentos): Un entero que representa el número de argumentos de línea de comandos.
  • argv (Vector de Argumentos): Un arreglo de punteros de caracteres que enumera todos los argumentos.

Entendiendo los Argumentos de Línea de Comandos

En C, los argumentos de línea de comandos se manejan usando dos parámetros de la función main:

int main(int argc, char *argv[]) {
    // Tu código aquí
}
  • argc: Este parámetro (conteo de argumentos) contiene el número de argumentos pasados al programa, incluyendo el nombre del programa.
  • argv: Este parámetro (vector de argumentos) es un arreglo de cadenas (punteros de caracteres) que representan los argumentos.

Ejemplo: Uso Básico

Veamos un ejemplo simple para entender cómo funcionan los argumentos de línea de comandos:

#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Número de argumentos: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("Argumento %d: %s\n", i, argv[i]);
    }
    return 0;
}

Explicación

  • El programa imprime el número de argumentos (argc).
  • Luego itera a través del arreglo argv e imprime cada argumento.

Ejecutando el Programa

Para ejecutar el programa con argumentos de línea de comandos, compílalo y ejecútalo desde la terminal:

gcc main.c -o main
./main arg1 arg2 arg3

Salida

Número de argumentos: 4
Argumento 0: ./main
Argumento 1: arg1
Argumento 2: arg2
Argumento 3: arg3

Uso Avanzado

Analizando Argumentos

En programas más complejos, podrías necesitar analizar y validar los argumentos de línea de comandos. Aquí hay un ejemplo que espera dos argumentos numéricos y calcula su suma:

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    if (argc != 3) {
        printf("Uso: %s <num1> <num2>\n", argv[0]);
        return 1;
    }

    int num1 = atoi(argv[1]);
    int num2 = atoi(argv[2]);
    int suma = num1 + num2;

    printf("Suma: %d\n", suma);
    return 0;
}

Explicación

  • El programa verifica si el número de argumentos es exactamente 3 (incluyendo el nombre del programa).
  • Convierte los argumentos de cadenas a enteros usando atoi.
  • Calcula la suma y la imprime.

Ejecutando el Programa

gcc main.c -o main
./main 5 10

Salida

Suma: 15

Errores Comunes

  • Conteo de Argumentos Incorrecto: Siempre valida el número de argumentos para evitar comportamientos inesperados.
  • Tipos de Datos Inválidos: Asegúrate de que los argumentos sean del tipo esperado (por ejemplo, convertir cadenas a enteros).

Conclusión

Los argumentos de línea de comandos son una característica poderosa en C que te permiten hacer tus programas más flexibles e interactivos. Al entender cómo usar argc y argv, puedes crear programas que acepten entrada del usuario directamente desde la línea de comandos, mejorando su usabilidad y funcionalidad. Practica escribiendo programas que tomen diferentes tipos de entrada y realicen varias operaciones basadas en esas entradas.
