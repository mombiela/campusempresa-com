Introducción

Los tipos de datos en C son fundamentales para entender cómo almacenar, manipular y usar datos en tus programas. Esta sección cubrirá los tipos de datos básicos y avanzados en C, su uso y cómo elegir el tipo de dato apropiado para tus necesidades.

Tipos de Datos Básicos

C proporciona varios tipos de datos básicos que se utilizan para definir variables y el tipo de datos que pueden contener.

Tipos Enteros

  • int: Se utiliza para almacenar enteros (números enteros).
  • short: Una versión más corta de int, que típicamente usa menos memoria.
  • long: Una versión más larga de int, que típicamente usa más memoria.
  • long long: Una versión aún más larga de int para números muy grandes.
int a = 10;
short b = 20;
long c = 30000;
long long d = 4000000000;

Tipos de Punto Flotante

  • float: Se utiliza para almacenar números de punto flotante de precisión simple.
  • double: Se utiliza para almacenar números de punto flotante de doble precisión.
  • long double: Se utiliza para números de punto flotante de precisión extendida.
float e = 3.14f;
double f = 3.141592653589793;
long double g = 3.14159265358979323846L;

Tipo de Carácter

  • char: Se utiliza para almacenar caracteres individuales. También se puede usar para almacenar enteros pequeños.
char h = 'A';
char i = 65; // Valor ASCII de 'A'

Tipo Booleano

  • _Bool: Se utiliza para almacenar valores booleanos (verdadero o falso).
_Bool j = 1; // verdadero
_Bool k = 0; // falso

Tipos de Datos Derivados

Los tipos de datos derivados se construyen a partir de los tipos de datos básicos e incluyen arreglos, punteros, estructuras y uniones.

Arreglos

Los arreglos son colecciones de elementos del mismo tipo.

int arr[5] = {1, 2, 3, 4, 5};
char str[] = "Hola";

Punteros

Los punteros almacenan la dirección de memoria de otra variable.

int l = 10;
int *ptr = &l;

Estructuras

Las estructuras se utilizan para agrupar diferentes tipos de variables bajo un solo nombre.

struct Persona {
    char nombre[50];
    int edad;
};

struct Persona persona1;
persona1.edad = 30;

Uniones

Las uniones son similares a las estructuras pero usan la misma ubicación de memoria para todos sus miembros.

union Datos {
    int intVal;
    float floatVal;
    char charVal;
};

union Datos datos;
datos.intVal = 10;

Tipos Enumerados

Los tipos enumerados (enums) son tipos definidos por el usuario que consisten en constantes enteras con nombre.

enum DiaSemana {Domingo, Lunes, Martes, Miércoles, Jueves, Viernes, Sábado};
enum DiaSemana hoy = Miércoles;

Calificadores de Tipo

Los calificadores de tipo añaden propiedades especiales a las variables.

  • const: Indica que el valor de la variable no puede cambiar.
  • volatile: Indica que el valor de la variable puede cambiar en cualquier momento sin ninguna acción por parte del código.
  • restrict: Un calificador de puntero que indica que el puntero es la única referencia a esa ubicación de memoria.
const int m = 100;
volatile int n = 200;
int * restrict p = &n;

Resumen

Entender los tipos de datos en C es crucial para una programación eficiente. Los tipos de datos básicos incluyen enteros, números de punto flotante, caracteres y booleanos. Los tipos de datos derivados incluyen arreglos, punteros, estructuras y uniones, mientras que los tipos enumerados permiten constantes con nombre definidas por el usuario. Los calificadores de tipo añaden propiedades adicionales a las variables, mejorando el control sobre cómo se usan y almacenan los datos. Al dominar estos conceptos, puedes escribir programas en C más eficientes y efectivos.
