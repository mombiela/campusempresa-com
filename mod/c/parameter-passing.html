Paso de Parámetros en C
Recursión ►

Introducción

El paso de parámetros es un concepto fundamental en la programación en C que permite a las funciones recibir valores de entrada y devolver resultados. Entender cómo se pasan los parámetros a las funciones es crucial para escribir código en C eficiente y efectivo.

Tipos de Paso de Parámetros

En C, hay dos formas principales de pasar parámetros a las funciones:

  1. Paso por Valor
  2. Paso por Referencia

Paso por Valor

Cuando los parámetros se pasan por valor, se hace una copia del valor real del parámetro en la memoria. Esto significa que los cambios realizados en el parámetro dentro de la función no afectan el valor original.

Ejemplo:

#include <stdio.h>

void addTen(int num) {
    num = num + 10;
    printf("Valor dentro de la función: %d\n", num);
}

int main() {
    int number = 5;
    addTen(number);
    printf("Valor fuera de la función: %d\n", number);
    return 0;
}

Explicación:

  • number se pasa a addTen por valor.
  • Dentro de addTen, num es una copia de number.
  • Los cambios en num no afectan a number.

Paso por Referencia

Cuando los parámetros se pasan por referencia, la función recibe una referencia (o puntero) al parámetro real. Esto permite que la función modifique el valor original.

Ejemplo:

#include <stdio.h>

void addTen(int *num) {
    *num = *num + 10;
    printf("Valor dentro de la función: %d\n", *num);
}

int main() {
    int number = 5;
    addTen(&number);
    printf("Valor fuera de la función: %d\n", number);
    return 0;
}

Explicación:

  • number se pasa a addTen por referencia usando su dirección (&number).
  • Dentro de addTen, num es un puntero a number.
  • Los cambios en *num afectan a number.

Comparación del Paso por Valor y el Paso por Referencia

| Característica | Paso por Valor | Paso por Referencia | |------------------------|-----------------------------------|-------------------------------------| | Tipo de Parámetro | Copia del valor real | Dirección del valor real | | Uso de Memoria | Más (debido a la copia) | Menos (sin copia, solo referencia) | | Efecto en el Original | Sin efecto | Puede modificar el valor original | | Caso de Uso | Cuando los datos originales deben permanecer sin cambios | Cuando los datos originales necesitan ser modificados |

Ejemplos Prácticos

Intercambio de Valores

Paso por Valor (Incorrecto):

#include <stdio.h>

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(x, y);
    printf("x: %d, y: %d\n", x, y); // La salida será x: 10, y: 20
    return 0;
}

Paso por Referencia (Correcto):

#include <stdio.h>

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);
    printf("x: %d, y: %d\n", x, y); // La salida será x: 20, y: 10
    return 0;
}

Conclusión

Entender el paso de parámetros es esencial para una programación en C efectiva. Al dominar tanto el paso por valor como el paso por referencia, puedes controlar cómo se manipulan los datos dentro de tus funciones, lo que lleva a un código más eficiente y predecible. Recuerda:

  • Usa el paso por valor cuando no quieras que la función modifique los datos originales.
  • Usa el paso por referencia cuando necesites que la función modifique los datos originales.
