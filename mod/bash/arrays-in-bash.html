Introducción a los Arrays

Los arrays en Bash son una característica poderosa que te permite almacenar múltiples valores en una sola variable. Esto puede ser particularmente útil para gestionar listas de datos, como nombres de archivos, entradas de usuario u opciones de configuración.

Creación de Arrays

  • Arrays Indexados: Estos arrays usan índices numéricos para acceder a sus elementos.
  • Arrays Asociativos: Estos arrays usan cadenas como índices.

Arrays Indexados

Para crear un array indexado, puedes usar la siguiente sintaxis:

# Método 1: Usando paréntesis
fruits=("Apple" "Banana" "Cherry")

# Método 2: Usando asignaciones individuales
fruits[0]="Apple"
fruits[1]="Banana"
fruits[2]="Cherry"

Arrays Asociativos

Para crear un array asociativo, necesitas declararlo primero y luego asignar valores:

# Declarar un array asociativo
declare -A colors

# Asignar valores
colors[apple]="red"
colors[banana]="yellow"
colors[cherry]="red"

Acceso a Elementos del Array

  • Arrays Indexados: Accede a los elementos usando su índice numérico.
  • Arrays Asociativos: Accede a los elementos usando su clave de cadena.

Arrays Indexados

# Acceso a elementos
echo ${fruits[0]}  # Salida: Apple
echo ${fruits[1]}  # Salida: Banana
echo ${fruits[2]}  # Salida: Cherry

Arrays Asociativos

# Acceso a elementos
echo ${colors[apple]}  # Salida: red
echo ${colors[banana]} # Salida: yellow
echo ${colors[cherry]} # Salida: red

Modificación de Arrays

  • Agregar Elementos: Puedes agregar elementos asignando valores a nuevos índices o claves.
  • Eliminar Elementos: Usa el comando unset para eliminar elementos.

Agregar Elementos

# Array Indexado
fruits[3]="Date"

# Array Asociativo
colors[date]="brown"

Eliminar Elementos

# Array Indexado
unset fruits[1]  # Elimina "Banana"

# Array Asociativo
unset colors[banana]  # Elimina la clave "banana"

Iterar a Través de Arrays

Puedes usar bucles para iterar a través de arrays y procesar sus elementos.

Arrays Indexados

# Iterar a través de un array indexado
for fruit in "${fruits[@]}"; do
  echo $fruit
done

Arrays Asociativos

# Iterar a través de un array asociativo
for key in "${!colors[@]}"; do
  echo "$key is ${colors[$key]}"
done

Longitud del Array

Puedes obtener la longitud de un array usando la sintaxis ${#array[@]}.

Arrays Indexados

# Obtener la longitud de un array indexado
echo ${#fruits[@]}  # Salida: 4

Arrays Asociativos

# Obtener la longitud de un array asociativo
echo ${#colors[@]}  # Salida: 3

Resumen

Los arrays en Bash proporcionan una forma flexible de manejar múltiples valores dentro de una sola variable. Pueden ser indexados o asociativos, permitiendo tanto la indexación numérica como la basada en cadenas. Las operaciones clave incluyen la creación, acceso, modificación e iteración a través de arrays. Entender estos conceptos es esencial para una eficiente escritura de scripts en Bash y gestión de datos.

Al dominar los arrays, puedes escribir scripts en Bash más eficientes y legibles, haciendo tu código más poderoso y versátil.
