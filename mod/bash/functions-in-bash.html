Introducción a las Funciones

Las funciones en Bash son una forma de agrupar comandos para reutilizarlos. Ayudan a organizar el código, hacer los scripts más legibles y reducir la redundancia.

Conceptos Clave

  • Definición: Una función se define usando la palabra clave function o simplemente escribiendo el nombre de la función seguido de paréntesis.
  • Invocación: Las funciones se llaman por su nombre seguido de paréntesis.
  • Ámbito: Las variables dentro de las funciones son locales por defecto a menos que se declaren globales.
  • Valores de Retorno: Las funciones pueden devolver valores usando la palabra clave return o haciendo eco del resultado.

Definiendo Funciones

Hay dos formas principales de definir una función en Bash:

Usando la palabra clave `function`

function greet {
    echo "¡Hola, $1!"
}

Sin la palabra clave `function`

greet() {
    echo "¡Hola, $1!"
}

Ambos métodos son equivalentes y pueden usarse indistintamente.

Ejemplo: Función Simple

greet() {
    echo "¡Hola, $1!"
}

# Llamando a la función
greet "Alice"

Salida:

¡Hola, Alice!

Parámetros de Función

Las funciones pueden aceptar parámetros, que se acceden usando $1, $2, etc.

Ejemplo: Función con Parámetros

add() {
    local sum=$(( $1 + $2 ))
    echo "Suma: $sum"
}

# Llamando a la función con argumentos
add 5 3

Salida:

Suma: 8

Devolviendo Valores

Las funciones pueden devolver valores usando la palabra clave return o haciendo eco del resultado.

Usando `return`

multiply() {
    local product=$(( $1 * $2 ))
    return $product
}

# Llamando a la función
multiply 4 5
result=$?
echo "Producto: $result"

Nota: La palabra clave return solo puede devolver un entero entre 0 y 255.

Usando `echo`

multiply() {
    local product=$(( $1 * $2 ))
    echo $product
}

# Llamando a la función
result=$(multiply 4 5)
echo "Producto: $result"

Variables Locales y Globales

Las variables dentro de las funciones son locales por defecto cuando se declaran con local.

Ejemplo: Variables Locales

calculate() {
    local result=$(( $1 + $2 ))
    echo $result
}

# Llamando a la función
sum=$(calculate 10 20)
echo "Suma: $sum"

Ejemplo: Variables Globales

result=0

calculate() {
    result=$(( $1 + $2 ))
}

# Llamando a la función
calculate 10 20
echo "Suma: $result"

Características Avanzadas de las Funciones

Funciones Recursivas

Las funciones pueden llamarse a sí mismas recursivamente.

Ejemplo: Función Recursiva

factorial() {
    if [ $1 -le 1 ]; then
        echo 1
    else
        local temp=$(( $1 - 1 ))
        local result=$(factorial $temp)
        echo $(( $1 * result ))
    fi
}

# Llamando a la función
result=$(factorial 5)
echo "Factorial: $result"

Bibliotecas de Funciones

Las funciones pueden almacenarse en archivos separados y ser incluidas en scripts.

Ejemplo: Biblioteca de Funciones

Crea un archivo mathlib.sh:

add() {
    echo $(( $1 + $2 ))
}

subtract() {
    echo $(( $1 - $2 ))
}

Incluye la biblioteca en tu script:

#!/bin/bash
source mathlib.sh

result=$(add 10 5)
echo "Adición: $result"

result=$(subtract 10 5)
echo "Sustracción: $result"

Conclusión

Las funciones en Bash son herramientas poderosas para organizar y reutilizar código. Pueden tomar parámetros, devolver valores e incluso llamarse a sí mismas recursivamente. Al dominar las funciones, puedes escribir scripts Bash más eficientes y mantenibles.
