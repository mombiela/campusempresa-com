Introducción

Los bucles son constructos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. En la programación de scripts en Bash, hay tres tipos principales de bucles: for, while y until. Esta sección cubrirá cada tipo de bucle, proporcionará ejemplos e incluirá ejercicios para reforzar tu comprensión.

Bucle For

El bucle for itera sobre una lista de elementos y ejecuta un bloque de código para cada elemento.

Sintaxis

for item in list
do
    # Comandos a ejecutar
done

Ejemplo

#!/bin/bash

# Lista de elementos
for fruit in apple banana cherry
do
    echo "Me gusta $fruit"
done

Explicación:

  • El bucle itera sobre la lista apple banana cherry.
  • Para cada elemento en la lista, imprime "Me gusta [fruit]".

Ejercicio

Escribe un script que imprima números del 1 al 5 usando un bucle for.

Bucle While

El bucle while ejecuta un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

while [ condition ]
do
    # Comandos a ejecutar
done

Ejemplo

#!/bin/bash

count=1

while [ $count -le 5 ]
do
    echo "El conteo es $count"
    ((count++))
done

Explicación:

  • El bucle continúa mientras count sea menor o igual a 5.
  • Imprime el valor actual de count y luego incrementa count en 1.

Ejercicio

Escribe un script que imprima números pares del 2 al 10 usando un bucle while.

Bucle Until

El bucle until es similar al bucle while pero ejecuta un bloque de código mientras una condición especificada sea falsa.

Sintaxis

until [ condition ]
do
    # Comandos a ejecutar
done

Ejemplo

#!/bin/bash

count=1

until [ $count -gt 5 ]
do
    echo "El conteo es $count"
    ((count++))
done

Explicación:

  • El bucle continúa hasta que count sea mayor que 5.
  • Imprime el valor actual de count y luego incrementa count en 1.

Ejercicio

Escribe un script que imprima números del 5 al 1 usando un bucle until.

Comparación de Bucles

| Tipo de Bucle | Verificación de Condición | Caso de Uso | |---------------|---------------------------|-------------| | for | Itera sobre una lista de elementos | Cuando conoces el número de iteraciones | | while | Ejecuta mientras la condición sea verdadera | Cuando el número de iteraciones no se conoce de antemano | | until | Ejecuta mientras la condición sea falsa | Similar a while, pero con la condición invertida |

Conclusión

Los bucles son herramientas poderosas en la programación de scripts en Bash que te permiten automatizar tareas repetitivas. El bucle for es ideal para iterar sobre una lista predefinida, mientras que los bucles while y until son más adecuados para situaciones donde el número de iteraciones depende de condiciones dinámicas. Practica usando estos bucles con los ejercicios proporcionados para solidificar tu comprensión.
