Presentación Final y Revisión

Resumen

En esta sección final del curso, consolidaremos todo lo que has aprendido sobre OpenGL. Esto implicará preparar una presentación final de tu proyecto, revisar conceptos clave y asegurarte de tener una comprensión sólida de OpenGL desde niveles principiantes hasta avanzados.

Presentación Final

Requisitos del Proyecto

  • Objetivo: Crear un proyecto integral de OpenGL que demuestre tu comprensión de los conceptos cubiertos en el curso.
  • Componentes:
    • Renderizado Básico: Incluir formas básicas y transformaciones.
    • Shaders: Implementar shaders de vértices y fragmentos.
    • Texturas: Aplicar texturas a los objetos.
    • Iluminación: Incorporar al menos un tipo de iluminación (ambiente, difusa, especular).
    • Características Avanzadas: Implementar al menos una característica avanzada como mapeo de sombras, efectos de post-procesamiento o animaciones complejas.

Estructura de la Presentación

  • Introducción:
    • Presenta brevemente tu proyecto.
    • Explica el objetivo y las características clave.
  • Detalles Técnicos:
    • Discute los conceptos de OpenGL utilizados.
    • Explica los shaders y cómo fueron implementados.
    • Describe las técnicas de mapeo de texturas e iluminación.
  • Demostración:
    • Muestra una demostración en vivo de tu proyecto.
    • Destaca las características avanzadas.
  • Desafíos y Soluciones:
    • Discute los desafíos enfrentados durante el desarrollo.
    • Explica cómo superaste estos desafíos.
  • Conclusión:
    • Resume el proyecto.
    • Reflexiona sobre lo que has aprendido.

Revisión de Conceptos Clave

Renderizado Básico

  • Vértices y Buffers:
    • Usa Objetos de Buffer de Vértices (VBOs) y Objetos de Array de Vértices (VAOs) para gestionar datos de vértices.
    // Ejemplo: Configuración de un VBO y VAO
GLuint VBO, VAO;
glGenBuffers(1, &VBO);
glGenVertexArrays(1, &VAO);

glBindVertexArray(VAO);
glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);

Shaders

  • Shaders de Vértices y Fragmentos:
    • Escribe y compila shaders para controlar el pipeline de renderizado.
    // Ejemplo: Shader de vértices simple
const char* vertexShaderSource = "#version 330 core\n"
                                 "layout (location = 0) in vec3 aPos;\n"
                                 "void main()\n"
                                 "{\n"
                                 "   gl_Position = vec4(aPos, 1.0);\n"
                                 "}\0";

GLuint vertexShader = glCreateShader(GL_VERTEX_SHADER);
glShaderSource(vertexShader, 1, &vertexShaderSource, NULL);
glCompileShader(vertexShader);

Texturas

  • Cargar y Aplicar Texturas:
    • Usa bibliotecas como SOIL o stb_image para cargar texturas.
    // Ejemplo: Cargar una textura usando stb_image
int width, height, nrChannels;
unsigned char *data = stbi_load("path/to/texture.jpg", &width, &height, &nrChannels, 0);
if (data)
{
    glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, width, height, 0, GL_RGB, GL_UNSIGNED_BYTE, data);
    glGenerateMipmap(GL_TEXTURE_2D);
}
stbi_image_free(data);

Iluminación

  • Tipos de Iluminación:
    • Implementa iluminación ambiente, difusa y especular.
    // Ejemplo: Modelo de iluminación Phong en shader de fragmentos
vec3 ambient = light.ambient * vec3(texture(material.diffuse, TexCoords));
vec3 norm = normalize(Normal);
vec3 lightDir = normalize(light.position - FragPos);
float diff = max(dot(norm, lightDir), 0.0);
vec3 diffuse = light.diffuse * diff * vec3(texture(material.diffuse, TexCoords));
vec3 viewDir = normalize(viewPos - FragPos);
vec3 reflectDir = reflect(-lightDir, norm);
float spec = pow(max(dot(viewDir, reflectDir), 0.0), material.shininess);
vec3 specular = light.specular * spec * vec3(texture(material.specular, TexCoords));
vec3 result = ambient + diffuse + specular;
FragColor = vec4(result, 1.0);

Características Avanzadas

  • Mapeo de Sombras:
    • Implementa mapeo de sombras para añadir sombras realistas a tu escena.
    // Ejemplo: Configuración de mapeo de sombras
glGenFramebuffers(1, &depthMapFBO);
glGenTextures(1, &depthMap);
glBindTexture(GL_TEXTURE_2D, depthMap);
glTexImage2D(GL_TEXTURE_2D, 0, GL_DEPTH_COMPONENT, SHADOW_WIDTH, SHADOW_HEIGHT, 0, GL_DEPTH_COMPONENT, GL_FLOAT, NULL);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_NEAREST);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_NEAREST);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_CLAMP_TO_BORDER);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_CLAMP_TO_BORDER);
float borderColor[] = { 1.0, 1.0, 1.0, 1.0 };
glTexParameterfv(GL_TEXTURE_2D, GL_TEXTURE_BORDER_COLOR, borderColor);
glBindFramebuffer(GL_FRAMEBUFFER, depthMapFBO);
glFramebufferTexture2D(GL_FRAMEBUFFER, GL_DEPTH_ATTACHMENT, GL_TEXTURE_2D, depthMap, 0);
glDrawBuffer(GL_NONE);
glReadBuffer(GL_NONE);
glBindFramebuffer(GL_FRAMEBUFFER, 0);
