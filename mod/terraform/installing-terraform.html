Introducción

Terraform es una herramienta de código abierto de infraestructura como código (IaC) que te permite definir y aprovisionar la infraestructura del centro de datos utilizando un lenguaje de configuración de alto nivel. Antes de que puedas comenzar a usar Terraform, necesitas instalarlo en tu máquina. Esta sección te guiará a través del proceso de instalación en diferentes sistemas operativos.

Prerrequisitos

Antes de instalar Terraform, asegúrate de tener lo siguiente:

  • Una computadora con acceso a internet.
  • Conocimientos básicos de interfaces de línea de comandos.

Pasos de Instalación

Windows

  1. Descargar Terraform:

    • Visita la página de descargas de Terraform.
    • Descarga la versión apropiada para Windows.

  2. Extraer el archivo ZIP:

    • Extrae el archivo ZIP descargado en un directorio de tu elección.

  3. Agregar Terraform a tu PATH:

    • Abre la búsqueda de Inicio, escribe "env" y selecciona "Editar las variables de entorno del sistema".
    • En la ventana de Propiedades del Sistema, haz clic en el botón "Variables de entorno".
    • En la ventana de Variables de Entorno, encuentra la variable "Path" en la sección "Variables del sistema" y haz clic en "Editar".
    • Haz clic en "Nuevo" y agrega la ruta al directorio donde extrajiste Terraform.
    • Haz clic en "Aceptar" para cerrar todas las ventanas.

  4. Verificar la instalación:

    • Abre el Símbolo del sistema y escribe: 
      terraform -v
    • Deberías ver la información de la versión de Terraform.

macOS

  1. Usando Homebrew:

    • Si no tienes Homebrew instalado, instálalo ejecutando: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instala Terraform usando Homebrew: 
      brew tap hashicorp/tap
brew install hashicorp/tap/terraform

  2. Verificar la instalación:

    • Abre Terminal y escribe: 
      terraform -v
    • Deberías ver la información de la versión de Terraform.

Linux

  1. Descargar Terraform:

    • Visita la página de descargas de Terraform.
    • Descarga la versión apropiada para Linux.

  2. Extraer el archivo ZIP:

    • Abre Terminal y navega al directorio donde descargaste el archivo ZIP.
    • Extrae el archivo usando: 
      unzip terraform_<VERSION>_linux_amd64.zip

  3. Mover el binario a un directorio en tu PATH:

    • Mueve el binario extraído a /usr/local/bin: 
      sudo mv terraform /usr/local/bin/

  4. Verificar la instalación:

    • Abre Terminal y escribe: 
      terraform -v
    • Deberías ver la información de la versión de Terraform.

Conclusión

Instalar Terraform es un proceso sencillo que implica descargar el binario apropiado para tu sistema operativo, extraerlo y agregarlo al PATH de tu sistema. Una vez instalado, puedes verificar la instalación comprobando la versión de Terraform. Con Terraform instalado, ahora estás listo para comenzar a definir y aprovisionar tu infraestructura como código.
