Introducción

En esta sección del curso, nos enfocaremos en ejercicios prácticos para la configuración de redes. Estos ejercicios están diseñados para proporcionar una comprensión profunda de cómo configurar y gestionar redes en un entorno profesional. Los ejercicios cubrirán desde configuraciones básicas hasta avanzadas, utilizando tanto interfaces gráficas como líneas de comando.

Objetivos de Aprendizaje

  • Comprender los conceptos básicos de configuración de redes.
  • Configurar dispositivos de red como routers y switches.
  • Implementar y gestionar subredes.
  • Configurar servicios de red esenciales como DHCP y DNS.
  • Realizar configuraciones avanzadas de seguridad en redes.

Ejercicio 1: Configuración Básica de un Router

Descripción

En este ejercicio, configuraremos un router básico para permitir la comunicación entre dos redes diferentes.

Pasos

  1. Conectar el hardware:

    • Conectar el router a dos redes diferentes utilizando cables Ethernet.
    • Asegurarse de que las interfaces del router estén correctamente conectadas.

  2. Acceder al router:

    • Utilizar una consola o una interfaz web para acceder al router.
    • Iniciar sesión con las credenciales proporcionadas.

  3. Configurar las interfaces:

    • Asignar direcciones IP a las interfaces del router.
    Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit

  4. Guardar la configuración:

    • Asegurarse de que la configuración se guarda correctamente.
    Router# copy running-config startup-config

Verificación

  • Verificar la conectividad entre las dos redes utilizando comandos de ping. 
    Router# ping 192.168.2.1

Ejercicio 2: Configuración de un Switch

Descripción

En este ejercicio, configuraremos un switch para segmentar una red en múltiples VLANs.

Pasos

  1. Acceder al switch:

    • Utilizar una consola o una interfaz web para acceder al switch.
    • Iniciar sesión con las credenciales proporcionadas.

  2. Crear VLANs:

    • Crear dos VLANs en el switch.
    Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Sales
Switch(config-vlan)# exit
Switch(config)# vlan 20
Switch(config-vlan)# name Engineering
Switch(config-vlan)# exit

  3. Asignar puertos a las VLANs:

    • Asignar puertos específicos a cada VLAN.
    Switch(config)# interface range GigabitEthernet0/1 - 2
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10
Switch(config-if-range)# exit
Switch(config)# interface range GigabitEthernet0/3 - 4
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 20
Switch(config-if-range)# exit

  4. Guardar la configuración:

    • Asegurarse de que la configuración se guarda correctamente.
    Switch# copy running-config startup-config

Verificación

  • Verificar la configuración de las VLANs. 
    Switch# show vlan brief

Ejercicio 3: Configuración de DHCP

Descripción

En este ejercicio, configuraremos un servidor DHCP en un router para asignar direcciones IP automáticamente a los dispositivos en la red.

Pasos

  1. Acceder al router:

    • Utilizar una consola o una interfaz web para acceder al router.
    • Iniciar sesión con las credenciales proporcionadas.

  2. Configurar el pool DHCP:

    • Configurar un pool de direcciones IP para la red.
    Router> enable
Router# configure terminal
Router(config)# ip dhcp pool LAN
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# exit

  3. Excluir direcciones IP:

    • Excluir direcciones IP específicas del pool DHCP.
    Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10

  4. Guardar la configuración:

    • Asegurarse de que la configuración se guarda correctamente.
    Router# copy running-config startup-config

Verificación

  • Verificar que los dispositivos en la red reciben direcciones IP automáticamente. 
    Router# show ip dhcp binding

Conclusión

Estos ejercicios proporcionan una base sólida para la configuración de redes. A través de la práctica, los estudiantes aprenderán a configurar routers, switches y servicios de red esenciales como DHCP. La comprensión y la habilidad para realizar estas configuraciones son fundamentales para cualquier profesional de TI que trabaje con redes.
