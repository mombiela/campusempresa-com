Añadir filtros y controles interactivos

Siguiente ►
►

En este tema, aprenderás cómo añadir filtros y controles interactivos a tus informes en Google Data Studio. Los filtros y controles interactivos permiten a los usuarios personalizar la visualización de los datos según sus necesidades, mejorando la interactividad y la utilidad de los informes.

Conceptos Clave

  1. Filtros: Permiten restringir los datos que se muestran en un informe según criterios específicos.
  2. Controles interactivos: Elementos de la interfaz que permiten a los usuarios ajustar los filtros en tiempo real, como listas desplegables, controles de fecha, y controles de rango.

Tipos de Controles Interactivos

  1. Control de Fecha: Permite a los usuarios seleccionar un rango de fechas.
  2. Lista Desplegable: Permite seleccionar uno o varios valores de una lista.
  3. Control de Rango: Permite seleccionar un rango de valores numéricos.
  4. Cuadro de Búsqueda: Permite buscar y filtrar datos basados en texto.

Pasos para Añadir Filtros y Controles Interactivos

  1. Añadir un Control de Fecha

  1. Selecciona el informe donde deseas añadir el control de fecha.
  2. Haz clic en "Añadir un control" en la barra de herramientas.
  3. Selecciona "Control de fecha".
  4. Coloca el control de fecha en el área del informe donde desees que aparezca.
  5. Configura el control de fecha en el panel de propiedades, seleccionando el campo de fecha correspondiente y ajustando las opciones de formato.

  1. Añadir una Lista Desplegable

  1. Selecciona el informe donde deseas añadir la lista desplegable.
  2. Haz clic en "Añadir un control" en la barra de herramientas.
  3. Selecciona "Lista desplegable".
  4. Coloca la lista desplegable en el área del informe donde desees que aparezca.
  5. Configura la lista desplegable en el panel de propiedades, seleccionando el campo que deseas utilizar para los valores de la lista.

  1. Añadir un Control de Rango

  1. Selecciona el informe donde deseas añadir el control de rango.
  2. Haz clic en "Añadir un control" en la barra de herramientas.
  3. Selecciona "Control de rango".
  4. Coloca el control de rango en el área del informe donde desees que aparezca.
  5. Configura el control de rango en el panel de propiedades, seleccionando el campo numérico correspondiente y ajustando las opciones de rango.

  1. Añadir un Cuadro de Búsqueda

  1. Selecciona el informe donde deseas añadir el cuadro de búsqueda.
  2. Haz clic en "Añadir un control" en la barra de herramientas.
  3. Selecciona "Cuadro de búsqueda".
  4. Coloca el cuadro de búsqueda en el área del informe donde desees que aparezca.
  5. Configura el cuadro de búsqueda en el panel de propiedades, seleccionando el campo de texto correspondiente.

Ejemplo Práctico

Vamos a crear un informe simple que incluya un control de fecha y una lista desplegable para filtrar los datos de ventas por fecha y por producto.

Paso 1: Crear el Informe

  1. Abre Google Data Studio y crea un nuevo informe.
  2. Conecta una fuente de datos que contenga información de ventas, incluyendo campos de fecha y producto.

Paso 2: Añadir un Control de Fecha

  1. Haz clic en "Añadir un control" y selecciona "Control de fecha".
  2. Coloca el control de fecha en la parte superior del informe.
  3. Configura el control de fecha seleccionando el campo de fecha de tu fuente de datos.

Paso 3: Añadir una Lista Desplegable

  1. Haz clic en "Añadir un control" y selecciona "Lista desplegable".
  2. Coloca la lista desplegable debajo del control de fecha.
  3. Configura la lista desplegable seleccionando el campo de producto de tu fuente de datos.

Paso 4: Añadir un Gráfico de Barras

  1. Haz clic en "Añadir un gráfico" y selecciona "Gráfico de barras".
  2. Coloca el gráfico de barras en el área principal del informe.
  3. Configura el gráfico de barras seleccionando el campo de producto como dimensión y el campo de ventas como métrica.

Paso 5: Probar los Controles Interactivos

  1. Publica el informe y abre la vista de usuario.
  2. Utiliza el control de fecha para seleccionar un rango de fechas.
  3. Utiliza la lista desplegable para seleccionar uno o varios productos.
  4. Observa cómo el gráfico de barras se actualiza en tiempo real según los filtros aplicados.

Ejercicio Práctico

Ejercicio 1: Añadir Filtros y Controles Interactivos

  1. Crea un nuevo informe en Google Data Studio.
  2. Conecta una fuente de datos que contenga información de ventas, incluyendo campos de fecha, producto y región.
  3. Añade un control de fecha para filtrar los datos por rango de fechas.
  4. Añade una lista desplegable para filtrar los datos por producto.
  5. Añade un control de rango para filtrar los datos por cantidad de ventas.
  6. Añade un gráfico de barras que muestre las ventas por región.
  7. Configura los controles interactivos para que afecten al gráfico de barras.
  8. Publica el informe y prueba los controles interactivos.

Solución del Ejercicio 1

  1. Crear el informe y conectar la fuente de datos.
  2. Añadir un control de fecha:
    • Haz clic en "Añadir un control" y selecciona "Control de fecha".
    • Coloca el control de fecha en la parte superior del informe.
    • Configura el control de fecha seleccionando el campo de fecha de la fuente de datos.
  3. Añadir una lista desplegable:
    • Haz clic en "Añadir un control" y selecciona "Lista desplegable".
    • Coloca la lista desplegable debajo del control de fecha.
    • Configura la lista desplegable seleccionando el campo de producto de la fuente de datos.
  4. Añadir un control de rango:
    • Haz clic en "Añadir un control" y selecciona "Control de rango".
    • Coloca el control de rango debajo de la lista desplegable.
    • Configura el control de rango seleccionando el campo de cantidad de ventas de la fuente de datos.
  5. Añadir un gráfico de barras:
    • Haz clic en "Añadir un gráfico" y selecciona "Gráfico de barras".
    • Coloca el gráfico de barras en el área principal del informe.
    • Configura el gráfico de barras seleccionando el campo de región como dimensión y el campo de ventas como métrica.
  6. Configurar los controles interactivos:
    • Asegúrate de que los controles interactivos están configurados para afectar al gráfico de barras.
  7. Publicar el informe y probar los controles interactivos.

Conclusión

Añadir filtros y controles interactivos a tus informes en Google Data Studio mejora significativamente la interactividad y la personalización de los datos. Los usuarios pueden ajustar los filtros en tiempo real para visualizar la información que más les interesa. Practica añadiendo diferentes tipos de controles interactivos a tus informes para familiarizarte con sus configuraciones y opciones. En el próximo módulo, exploraremos funcionalidades avanzadas como el uso de campos calculados y la creación de segmentos y métricas personalizadas.
