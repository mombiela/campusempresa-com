Uso de campos calculados
Siguiente ►

Introducción

En Google Data Studio, los campos calculados son una herramienta poderosa que permite crear nuevas métricas y dimensiones basadas en los datos existentes. Estos campos se pueden utilizar para realizar cálculos personalizados, transformar datos y crear nuevas perspectivas que no están disponibles directamente en las fuentes de datos.

Conceptos Clave

  1. Campos Calculados: Son campos personalizados que se crean utilizando expresiones y funciones matemáticas, lógicas o textuales.
  2. Expresiones: Fórmulas que definen cómo se debe calcular el valor de un campo.
  3. Funciones: Operaciones predefinidas que se pueden utilizar dentro de las expresiones para realizar cálculos específicos.

Creación de un Campo Calculado

Paso 1: Acceder a la Configuración de la Fuente de Datos

  1. Abre Google Data Studio y selecciona el informe en el que deseas trabajar.
  2. Haz clic en el icono de lápiz junto a la fuente de datos que deseas modificar.

Paso 2: Añadir un Campo Calculado

  1. En la configuración de la fuente de datos, haz clic en "Añadir un campo".
  2. Se abrirá un editor donde puedes definir el nombre y la expresión del nuevo campo calculado.

Paso 3: Definir la Expresión

  1. Nombre del Campo: Asigna un nombre descriptivo al nuevo campo calculado.
  2. Expresión: Escribe la fórmula que define cómo se calculará el valor del campo. Puedes utilizar operadores matemáticos, funciones y referencias a otros campos.

Ejemplo de Expresión

Supongamos que tienes un campo llamado Revenue (Ingresos) y otro campo llamado Cost (Costo). Puedes crear un campo calculado para calcular el Profit (Beneficio) utilizando la siguiente expresión:

Profit = Revenue - Cost

Paso 4: Guardar y Aplicar

  1. Después de definir la expresión, haz clic en "Guardar".
  2. El nuevo campo calculado aparecerá en la lista de campos disponibles y podrás utilizarlo en tus informes.

Ejemplos Prácticos

Ejemplo 1: Calcular el Beneficio

Campos Originales:

  • Revenue (Ingresos)
  • Cost (Costo)

Campo Calculado:

  • Profit (Beneficio)

Expresión:

Profit = Revenue - Cost

Ejemplo 2: Calcular el Margen de Beneficio

Campos Originales:

  • Revenue (Ingresos)
  • Profit (Beneficio)

Campo Calculado:

  • Profit Margin (Margen de Beneficio)

Expresión:

Profit Margin = (Profit / Revenue) * 100

Ejemplo 3: Concatenar Textos

Campos Originales:

  • First Name (Nombre)
  • Last Name (Apellido)

Campo Calculado:

  • Full Name (Nombre Completo)

Expresión:

Full Name = CONCAT(First Name, " ", Last Name)

Ejercicio Práctico

Ejercicio 1: Crear un Campo Calculado para el Beneficio

Instrucciones:

  1. Abre un informe en Google Data Studio.
  2. Accede a la configuración de la fuente de datos.
  3. Añade un nuevo campo calculado llamado Profit.
  4. Define la expresión para calcular el beneficio como Revenue - Cost.
  5. Guarda el campo calculado y utilízalo en un gráfico de barras para visualizar el beneficio por mes.

Solución

  1. Abre Google Data Studio y selecciona tu informe.
  2. Haz clic en el icono de lápiz junto a la fuente de datos.
  3. Haz clic en "Añadir un campo".
  4. Asigna el nombre Profit.
  5. Escribe la expresión Revenue - Cost.
  6. Haz clic en "Guardar".
  7. Añade un gráfico de barras al informe y selecciona Profit como la métrica a visualizar.

Conclusión

Los campos calculados en Google Data Studio son una herramienta esencial para personalizar y enriquecer tus informes. Permiten realizar cálculos avanzados y crear nuevas métricas y dimensiones que proporcionan una visión más profunda de tus datos. Practicar la creación y uso de campos calculados te ayudará a aprovechar al máximo las capacidades de Google Data Studio y a crear informes más informativos y útiles.

En el próximo tema, exploraremos cómo crear segmentos y métricas personalizadas para una mayor personalización y análisis de tus datos.
