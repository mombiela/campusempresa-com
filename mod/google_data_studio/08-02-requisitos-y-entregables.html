Requisitos y entregables
Siguiente ►

En esta sección, detallaremos los requisitos y entregables del proyecto final del curso de Google Data Studio. Este proyecto tiene como objetivo consolidar todos los conocimientos adquiridos a lo largo del curso, permitiendo a los estudiantes aplicar las habilidades aprendidas en un contexto práctico y realista.

Requisitos del Proyecto Final

1. Selección de una Fuente de Datos

  • Descripción: Elige una fuente de datos relevante para tu proyecto. Puede ser una hoja de cálculo de Google Sheets, una base de datos SQL, datos de Google Analytics, entre otros.
  • Ejemplo: Si trabajas en marketing, podrías usar datos de campañas publicitarias de Google Ads.

2. Conexión de la Fuente de Datos a Google Data Studio

  • Descripción: Conecta la fuente de datos seleccionada a Google Data Studio siguiendo los pasos aprendidos en el Módulo 2.
  • Ejemplo: Conectar una hoja de cálculo de Google Sheets que contiene datos de ventas mensuales.

3. Creación de un Informe desde Cero

  • Descripción: Crea un informe nuevo en Google Data Studio y configura la estructura básica del informe.
  • Ejemplo: Un informe de ventas que incluya secciones para ventas por región, ventas por producto y tendencias de ventas mensuales.

4. Añadir y Configurar Gráficos Básicos

  • Descripción: Incluye al menos tres tipos de gráficos básicos (por ejemplo, gráficos de barras, gráficos de líneas, tablas) que representen los datos de manera efectiva.
  • Ejemplo: Un gráfico de barras para mostrar ventas por región, un gráfico de líneas para mostrar tendencias de ventas mensuales y una tabla para listar ventas por producto.

5. Personalización del Informe

  • Descripción: Aplica temas y estilos personalizados para mejorar la apariencia del informe. Personaliza los gráficos y tablas para que sean coherentes con el tema del informe.
  • Ejemplo: Usar colores corporativos y fuentes específicas para mantener la identidad visual de la empresa.

6. Añadir Filtros y Controles Interactivos

  • Descripción: Añade filtros y controles interactivos que permitan a los usuarios del informe explorar los datos de manera dinámica.
  • Ejemplo: Un filtro de fecha para seleccionar el rango de tiempo y un control de selección para elegir diferentes regiones.

7. Uso de Funcionalidades Avanzadas

  • Descripción: Implementa al menos una funcionalidad avanzada, como campos calculados, segmentos personalizados o integración con otras herramientas de Google.
  • Ejemplo: Crear un campo calculado para calcular el crecimiento porcentual de las ventas mes a mes.

8. Compartir y Colaborar

  • Descripción: Configura las opciones de compartir para permitir la colaboración en tiempo real con otros usuarios. Asegúrate de que el informe sea accesible para las partes interesadas.
  • Ejemplo: Compartir el informe con el equipo de ventas y permitirles editar ciertos aspectos del informe.

Entregables del Proyecto Final

1. Informe de Google Data Studio

  • Descripción: El informe final debe estar completo y accesible a través de un enlace compartido. Debe incluir todas las secciones y funcionalidades requeridas.
  • Formato: Enlace a Google Data Studio.

2. Documentación del Proyecto

  • Descripción: Un documento que explique el proceso seguido para crear el informe, incluyendo la selección de la fuente de datos, la configuración de gráficos, la personalización aplicada y las funcionalidades avanzadas utilizadas.
  • Formato: Documento en Google Docs o PDF.
  • Contenido Sugerido:
    • Introducción y objetivo del informe.
    • Descripción de la fuente de datos.
    • Pasos para conectar la fuente de datos.
    • Detalles sobre la creación y configuración de gráficos.
    • Explicación de la personalización aplicada.
    • Descripción de los filtros y controles interactivos añadidos.
    • Detalles sobre las funcionalidades avanzadas implementadas.
    • Proceso de compartir y colaborar en el informe.

3. Video de Presentación (Opcional)

  • Descripción: Un video corto (5-10 minutos) en el que presentes el informe, expliques las decisiones tomadas y demuestres las funcionalidades interactivas.
  • Formato: Enlace a un video en YouTube, Google Drive o cualquier otra plataforma de video.

Evaluación del Proyecto Final

Criterios de Evaluación

  • Completitud: El informe incluye todas las secciones y funcionalidades requeridas.
  • Calidad de los Datos: Los datos están correctamente conectados y representados.
  • Personalización: El informe está visualmente atractivo y personalizado de acuerdo con las directrices del curso.
  • Interactividad: Los filtros y controles interactivos funcionan correctamente.
  • Funcionalidades Avanzadas: Se han implementado correctamente las funcionalidades avanzadas.
  • Documentación: La documentación es clara, detallada y explica adecuadamente el proceso seguido.
  • Presentación (Opcional): El video de presentación es claro, conciso y bien estructurado.

Retroalimentación

  • Descripción: Los instructores proporcionarán retroalimentación detallada sobre cada aspecto del proyecto, destacando los puntos fuertes y ofreciendo sugerencias para mejorar.
  • Formato: Comentarios en el documento de Google Docs o PDF, y comentarios en el informe de Google Data Studio.

Conclusión

El proyecto final es una oportunidad para demostrar tu dominio de Google Data Studio y aplicar las habilidades adquiridas en un contexto práctico. Asegúrate de seguir todos los requisitos y entregar todos los elementos solicitados para obtener la mejor evaluación posible. ¡Buena suerte!
