Opciones de compartir informes
Configuración de permisos y acceso ►

En esta sección del curso sobre Google Data Studio, exploraremos las diversas opciones disponibles para compartir informes. Aprenderás cómo colaborar con otros usuarios, controlar los permisos de acceso y utilizar enlaces compartidos para distribuir tus informes de manera efectiva.

Opciones de Compartir

Google Data Studio ofrece varias formas de compartir tus informes con otros usuarios. Aquí están las principales opciones:

  • Compartir con usuarios específicos: Puedes invitar a personas específicas a ver o editar tu informe.
  • Enlace compartido: Genera un enlace que puedes compartir con cualquier persona, permitiéndoles ver o editar el informe según los permisos que configures.
  • Incorporar informes: Inserta tus informes en sitios web o blogs utilizando código HTML.
  • Descargar como PDF: Exporta tu informe a un archivo PDF para compartirlo fuera de Google Data Studio.

Compartir con Usuarios Específicos

Para compartir un informe con usuarios específicos, sigue estos pasos:

  1. Abre el informe que deseas compartir.
  2. Haz clic en el botón "Compartir" en la esquina superior derecha.
  3. Ingresa las direcciones de correo electrónico de las personas con las que deseas compartir el informe.
  4. Selecciona los permisos adecuados (ver o editar).
  5. Haz clic en "Enviar".

Ejemplo

1. Abre el informe "Ventas Mensuales".
2. Haz clic en "Compartir".
3. Ingresa el correo electrónico "ejemplo@dominio.com".
4. Selecciona "Puede ver".
5. Haz clic en "Enviar".

Enlace Compartido

Para generar un enlace compartido:

  1. Abre el informe.
  2. Haz clic en "Compartir".
  3. En la sección "Obtener enlace", haz clic en "Copiar enlace".
  4. Configura los permisos del enlace (ver o editar).
  5. Comparte el enlace con las personas deseadas.

Ejemplo

1. Abre el informe "Análisis de Tráfico Web".
2. Haz clic en "Compartir".
3. En "Obtener enlace", haz clic en "Copiar enlace".
4. Configura los permisos como "Cualquiera con el enlace puede ver".
5. Comparte el enlace: https://datastudio.google.com/reporting/ejemplo

Incorporar Informes

Para incorporar un informe en un sitio web:

  1. Abre el informe.
  2. Haz clic en "Archivo" > "Incorporar informe".
  3. Copia el código HTML proporcionado.
  4. Pega el código en el HTML de tu sitio web o blog.

Ejemplo

<iframe width="600" height="450" src="https://datastudio.google.com/reporting/ejemplo" frameborder="0" style="border:0" allowfullscreen></iframe>

Descargar como PDF

Para descargar un informe como PDF:

  1. Abre el informe.
  2. Haz clic en "Archivo" > "Descargar como PDF".
  3. Configura las opciones de descarga (páginas, orientación, etc.).
  4. Haz clic en "Descargar".

Ejemplo

1. Abre el informe "Resumen Financiero".
2. Haz clic en "Archivo" > "Descargar como PDF".
3. Selecciona "Todas las páginas".
4. Haz clic en "Descargar".

Comparación de Opciones

Opción Permisos Disponibles Ideal Para
Usuarios Específicos Ver, Editar Colaboración directa con compañeros de equipo
Enlace Compartido Ver, Editar Distribución amplia sin necesidad de cuentas
Incorporar Informes Ver Publicación en sitios web o blogs
Descargar como PDF N/A Compartir fuera de Google Data Studio

Conclusión

Compartir informes en Google Data Studio es una funcionalidad poderosa que facilita la colaboración y distribución de datos. Ya sea que necesites colaborar con colegas, distribuir informes a una audiencia más amplia o incorporar tus datos en un sitio web, Google Data Studio ofrece opciones flexibles para satisfacer tus necesidades. Asegúrate de configurar los permisos adecuadamente para mantener el control sobre quién puede ver o editar tus informes.
