Configuración de tu cuenta de AWS

Siguiente ►
►

En esta sección, aprenderás cómo configurar tu cuenta de AWS, un paso esencial para comenzar a utilizar los servicios de Amazon Web Services. Este proceso incluye la creación de una cuenta, la configuración de la autenticación multifactor (MFA) y la creación de un usuario administrador.

  1. Creación de una cuenta de AWS

Paso 1: Registro en AWS

  1. Visita el sitio web de AWS: Abre tu navegador y dirígete a aws.amazon.com.
  2. Haz clic en "Crear una cuenta de AWS": Este botón generalmente se encuentra en la esquina superior derecha de la página.
  3. Introduce tu dirección de correo electrónico: Asegúrate de usar una dirección de correo electrónico válida, ya que recibirás un correo de confirmación.
  4. Elige una contraseña: Crea una contraseña segura para tu cuenta.
  5. Introduce tu nombre de cuenta de AWS: Este puede ser tu nombre personal o el nombre de tu empresa.

Paso 2: Información de contacto

  1. Selecciona el tipo de cuenta: Personal o Profesional.
  2. Introduce tu información de contacto: Nombre, dirección, número de teléfono, etc.
  3. Acepta los términos y condiciones: Lee y acepta los términos de servicio de AWS.

Paso 3: Información de pago

  1. Introduce los detalles de tu tarjeta de crédito o débito: AWS requiere una tarjeta de crédito válida para verificar tu identidad y para los cargos futuros.
  2. Verificación de identidad: AWS puede realizar una verificación telefónica o enviar un código de verificación a tu correo electrónico.

Paso 4: Selección de plan de soporte

  1. Elige un plan de soporte: AWS ofrece varios planes de soporte, desde el plan gratuito hasta planes empresariales. Para comenzar, puedes seleccionar el plan gratuito.

Paso 5: Confirmación

  1. Revisa y confirma tu información: Asegúrate de que toda la información proporcionada sea correcta.
  2. Haz clic en "Crear cuenta y continuar": Recibirás un correo electrónico de confirmación.

  1. Configuración de la autenticación multifactor (MFA)

La autenticación multifactor (MFA) añade una capa adicional de seguridad a tu cuenta de AWS. Aquí te mostramos cómo configurarla:

Paso 1: Accede a la consola de administración de AWS

  1. Inicia sesión en tu cuenta de AWS: Usa las credenciales que creaste durante el registro.
  2. Navega a IAM (Identity and Access Management): Puedes encontrar IAM en el menú de servicios.

Paso 2: Configuración de MFA

  1. Selecciona "Usuarios" en el panel de navegación: Luego, haz clic en tu nombre de usuario.
  2. Haz clic en la pestaña "Seguridad": Aquí encontrarás la opción para configurar MFA.
  3. Haz clic en "Asignar MFA": Sigue las instrucciones para configurar MFA usando una aplicación de autenticación como Google Authenticator o Authy.

Paso 3: Verificación de MFA

  1. Escanea el código QR: Usa tu aplicación de autenticación para escanear el código QR proporcionado por AWS.
  2. Introduce los códigos de verificación: La aplicación generará códigos de verificación que deberás introducir en la consola de AWS para completar la configuración.

  1. Creación de un usuario administrador

Para seguir las mejores prácticas de seguridad, es recomendable crear un usuario administrador en lugar de usar la cuenta raíz para las tareas diarias.

Paso 1: Accede a IAM

  1. Inicia sesión en la consola de administración de AWS: Usa las credenciales de tu cuenta raíz.
  2. Navega a IAM: Encuentra IAM en el menú de servicios.

Paso 2: Crear un grupo de usuarios administradores

  1. Selecciona "Grupos" en el panel de navegación: Luego, haz clic en "Crear nuevo grupo".
  2. Introduce un nombre para el grupo: Por ejemplo, "Administradores".
  3. Adjunta políticas de permisos: Selecciona la política "AdministratorAccess" para otorgar permisos administrativos completos.
  4. Haz clic en "Crear grupo": El grupo de usuarios administradores se creará con los permisos seleccionados.

Paso 3: Crear un usuario administrador

  1. Selecciona "Usuarios" en el panel de navegación: Luego, haz clic en "Agregar usuario".
  2. Introduce un nombre de usuario: Por ejemplo, "admin-user".
  3. Selecciona el tipo de acceso: Marca la casilla "Acceso programático" y "Acceso a la consola de administración de AWS".
  4. Configura los permisos: Añade el usuario al grupo de administradores que creaste anteriormente.
  5. Revisa y crea el usuario: AWS generará credenciales de inicio de sesión para el nuevo usuario.

Paso 4: Configuración de MFA para el usuario administrador

  1. Selecciona el usuario administrador: Navega a la pestaña "Seguridad".
  2. Configura MFA: Sigue los mismos pasos que utilizaste para configurar MFA en la cuenta raíz.

Conclusión

En esta sección, has aprendido cómo crear y configurar tu cuenta de AWS, incluyendo la configuración de la autenticación multifactor (MFA) y la creación de un usuario administrador. Estos pasos son fundamentales para asegurar tu cuenta y prepararte para utilizar los servicios de AWS de manera segura y eficiente. En el próximo módulo, exploraremos la infraestructura global de AWS y cómo utilizar la consola de administración de AWS para gestionar tus recursos.
