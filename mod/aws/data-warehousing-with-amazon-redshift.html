Configuración de Amazon Redshift

  • Requisitos Previos
    • Cuenta de AWS
    • Roles y Permisos de IAM
  • Creación de un Clúster Redshift
    • Guía Paso a Paso
    • Opciones de Configuración
    • Ejemplo: 
      import boto3

redshift = boto3.client('redshift')

response = redshift.create_cluster(
    ClusterIdentifier='mi-cluster-redshift',
    NodeType='dc2.large',
    MasterUsername='usuarioaws',
    MasterUserPassword='TuContraseña123',
    ClusterType='multi-node',
    NumberOfNodes=2
)

print(response)
  • Conexión al Clúster Redshift
    • Usando Clientes SQL (por ejemplo, SQL Workbench/J)
    • Ejemplo de Cadena de Conexión:

Consulta de Datos en Amazon Redshift

  • Consultas SQL Básicas
    • SELECT, INSERT, UPDATE, DELETE
    • Ejemplo: 
      SELECT * FROM mi_tabla WHERE columna1 = 'valor';
  • Consultas SQL Avanzadas
    • Joins, Agregaciones, Funciones de Ventana
    • Ejemplo: 
      SELECT columna1, COUNT(*)
FROM mi_tabla
GROUP BY columna1
HAVING COUNT(*) > 1;
