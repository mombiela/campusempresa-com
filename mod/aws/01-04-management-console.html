Consola de administración de AWS
Siguiente ►

La Consola de administración de AWS es una interfaz web que permite a los usuarios interactuar con los servicios de Amazon Web Services (AWS). A través de esta consola, puedes gestionar y monitorear tus recursos de AWS de manera intuitiva y eficiente. En esta sección, aprenderás a navegar por la consola, entender sus componentes principales y realizar tareas básicas.

Objetivos de Aprendizaje

Al finalizar esta sección, deberías ser capaz de:

  • Navegar por la Consola de administración de AWS.
  • Identificar y utilizar los componentes principales de la consola.
  • Realizar tareas básicas como la creación de instancias y la gestión de recursos.

1. Navegación por la Consola de administración de AWS

1.1 Acceso a la Consola

Para acceder a la Consola de administración de AWS:

  1. Abre tu navegador web y dirígete a https://aws.amazon.com/.
  2. Haz clic en "Sign In to the Console" en la esquina superior derecha.
  3. Ingresa tus credenciales de AWS (ID de usuario y contraseña).

1.2 Componentes Principales de la Consola

Una vez que hayas iniciado sesión, verás la página principal de la consola. Los componentes principales incluyen:

  • Barra de navegación superior: Contiene enlaces a servicios, la barra de búsqueda, notificaciones, y el menú de cuenta.
  • Barra de búsqueda: Permite buscar servicios rápidamente.
  • Panel de servicios: Muestra una lista de todos los servicios de AWS categorizados.
  • Panel de recursos recientes: Muestra los recursos que has utilizado recientemente.
  • Panel de favoritos: Puedes agregar servicios que usas frecuentemente para un acceso rápido.

1.3 Barra de Navegación Superior

La barra de navegación superior es una herramienta esencial para moverse por la consola. Incluye:

  • Servicios: Un menú desplegable que lista todos los servicios de AWS.
  • Recursos recientes: Acceso rápido a los recursos que has utilizado recientemente.
  • Notificaciones: Alertas y notificaciones sobre el estado de tus recursos.
  • Cuenta: Acceso a la configuración de tu cuenta, facturación y soporte.

2. Uso de la Consola de administración de AWS

2.1 Creación de una Instancia EC2

Vamos a realizar una tarea básica: la creación de una instancia EC2.

  1. Accede al servicio EC2:

    • En la barra de búsqueda, escribe "EC2" y selecciona "EC2" de la lista de resultados.

  2. Lanza una nueva instancia:

    • Haz clic en "Launch Instance".
    • Selecciona una Amazon Machine Image (AMI). Para este ejemplo, selecciona "Amazon Linux 2 AMI".
    • Elige un tipo de instancia. Para este ejemplo, selecciona "t2.micro" (que es elegible para el nivel gratuito).
    • Configura los detalles de la instancia. Puedes dejar las configuraciones predeterminadas.
    • Agrega almacenamiento. La configuración predeterminada suele ser suficiente.
    • Agrega etiquetas si lo deseas.
    • Configura el grupo de seguridad. Puedes crear uno nuevo o usar uno existente.
    • Revisa y lanza la instancia. Se te pedirá que selecciones o crees un par de claves para acceder a la instancia.

  3. Accede a la instancia:

    • Una vez que la instancia esté en estado "running", selecciona la instancia y haz clic en "Connect".
    • Sigue las instrucciones para conectarte a la instancia usando SSH.

2.2 Gestión de Recursos

La consola también permite gestionar y monitorear tus recursos de manera eficiente. Algunas tareas comunes incluyen:

  • Detener/Iniciar instancias: Selecciona la instancia y usa los botones "Stop" o "Start".
  • Monitoreo: Usa Amazon CloudWatch para monitorear el rendimiento de tus recursos.
  • Etiquetado: Agrega etiquetas a tus recursos para una mejor organización y gestión.

3. Ejercicio Práctico

Ejercicio 1: Creación y Gestión de una Instancia EC2

  1. Accede a la Consola de administración de AWS.
  2. Navega al servicio EC2.
  3. Lanza una nueva instancia EC2 siguiendo los pasos descritos anteriormente.
  4. Una vez que la instancia esté en estado "running", detén la instancia.
  5. Reinicia la instancia y verifica que esté en estado "running" nuevamente.

Solución del Ejercicio

  1. Accede a la Consola de administración de AWS y navega al servicio EC2.
  2. Haz clic en "Launch Instance" y sigue los pasos para configurar y lanzar una nueva instancia.
  3. Una vez que la instancia esté en estado "running", selecciónala y haz clic en "Stop".
  4. Espera a que la instancia se detenga completamente, luego selecciónala y haz clic en "Start".
  5. Verifica que la instancia esté en estado "running".

Conclusión

En esta sección, has aprendido a navegar por la Consola de administración de AWS, identificar sus componentes principales y realizar tareas básicas como la creación y gestión de instancias EC2. La consola es una herramienta poderosa que te permite interactuar con los servicios de AWS de manera intuitiva y eficiente. En el próximo módulo, exploraremos los servicios principales de AWS, comenzando con Amazon EC2.
