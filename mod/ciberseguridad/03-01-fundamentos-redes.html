Fundamentos de Redes

Siguiente ►

Introducción

En este módulo, exploraremos los conceptos fundamentales de las redes de computadoras, que son esenciales para comprender cómo se comunican los dispositivos y cómo proteger esta comunicación. Las redes son la columna vertebral de la infraestructura de TI y, por lo tanto, su seguridad es crucial para la ciberseguridad.

Objetivos del Módulo

  • Comprender los conceptos básicos de las redes de computadoras.
  • Conocer los diferentes tipos de redes y sus características.
  • Entender los componentes y dispositivos de red.
  • Familiarizarse con los modelos de referencia OSI y TCP/IP.
  • Aprender sobre las direcciones IP y el enrutamiento.

Conceptos Básicos de Redes

¿Qué es una Red de Computadoras?

Una red de computadoras es un conjunto de dispositivos interconectados que pueden comunicarse y compartir recursos entre sí. Los dispositivos en una red pueden incluir computadoras, servidores, impresoras, y otros dispositivos de hardware.

Tipos de Redes

Las redes se pueden clasificar según su tamaño, alcance y propósito:

  • Red de Área Personal (PAN): Conecta dispositivos en un área muy pequeña, como una habitación.
  • Red de Área Local (LAN): Conecta dispositivos en un área limitada, como una oficina o un edificio.
  • Red de Área Metropolitana (MAN): Cubre un área geográfica más grande, como una ciudad.
  • Red de Área Amplia (WAN): Cubre grandes distancias, como países o continentes.

Componentes de una Red

  • Dispositivos de Red: Incluyen computadoras, servidores, impresoras, etc.
  • Medios de Transmisión: Cables (cobre, fibra óptica) o inalámbricos (Wi-Fi, Bluetooth).
  • Dispositivos de Interconexión: Routers, switches, hubs, etc.
  • Protocolos de Comunicación: Reglas y estándares que permiten la comunicación entre dispositivos.

Modelos de Referencia

Modelo OSI (Open Systems Interconnection)

El modelo OSI es un marco conceptual que estandariza las funciones de una red de telecomunicaciones o de computadoras en siete capas distintas:

  1. Capa Física: Transmisión y recepción de datos en forma de señales eléctricas o ópticas.
  2. Capa de Enlace de Datos: Establecimiento y terminación de enlaces de datos entre dos nodos.
  3. Capa de Red: Enrutamiento de paquetes de datos entre dispositivos.
  4. Capa de Transporte: Transferencia fiable de datos entre sistemas finales.
  5. Capa de Sesión: Gestión de sesiones de comunicación.
  6. Capa de Presentación: Traducción de datos entre el formato de red y el formato de aplicación.
  7. Capa de Aplicación: Servicios de red para aplicaciones de usuario final.

Modelo TCP/IP

El modelo TCP/IP es un conjunto de protocolos de comunicación utilizados en Internet y redes similares. Tiene cuatro capas:

  1. Capa de Acceso a la Red: Equivalente a las capas Física y de Enlace de Datos del modelo OSI.
  2. Capa de Internet: Equivalente a la Capa de Red del modelo OSI.
  3. Capa de Transporte: Equivalente a la Capa de Transporte del modelo OSI.
  4. Capa de Aplicación: Combina las capas de Sesión, Presentación y Aplicación del modelo OSI.

Direcciones IP y Enrutamiento

Direcciones IP

Una dirección IP es una etiqueta numérica asignada a cada dispositivo conectado a una red que utiliza el Protocolo de Internet para la comunicación. Existen dos versiones principales:

  • IPv4: Utiliza direcciones de 32 bits, expresadas en formato decimal (por ejemplo, 192.168.1.1).
  • IPv6: Utiliza direcciones de 128 bits, expresadas en formato hexadecimal (por ejemplo, 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

Enrutamiento

El enrutamiento es el proceso de seleccionar caminos en una red para enviar datos desde el origen hasta el destino. Los routers son dispositivos que realizan esta función, utilizando tablas de enrutamiento y protocolos de enrutamiento como OSPF, BGP, y RIP.

Ejemplo Práctico: Configuración Básica de una Red

Configuración de una Red LAN

  1. Conectar los Dispositivos:

    • Conectar computadoras y otros dispositivos a un switch utilizando cables Ethernet.
    • Conectar el switch a un router que proporciona acceso a Internet.

  2. Asignar Direcciones IP:

    • Configurar el router para asignar direcciones IP automáticamente mediante DHCP.
    • Alternativamente, asignar direcciones IP estáticas a cada dispositivo.

  3. Configurar el Router:

    • Acceder a la interfaz de administración del router a través de un navegador web.
    • Configurar la conexión a Internet (PPPoE, DHCP, IP estática).
    • Configurar la red Wi-Fi (SSID, contraseña).

Ejemplo de Configuración de IP Estática en Windows

1. Abrir el Panel de Control.
2. Ir a "Centro de redes y recursos compartidos".
3. Hacer clic en "Cambiar configuración del adaptador".
4. Hacer clic derecho en la conexión de red y seleccionar "Propiedades".
5. Seleccionar "Protocolo de Internet versión 4 (TCP/IPv4)" y hacer clic en "Propiedades".
6. Seleccionar "Usar la siguiente dirección IP" e ingresar la dirección IP, máscara de subred y puerta de enlace predeterminada.
7. Ingresar las direcciones de los servidores DNS.
8. Hacer clic en "Aceptar" para guardar los cambios.

Ejercicios Prácticos

Ejercicio 1: Clasificación de Redes

Instrucciones: Clasifica las siguientes redes según su tipo (PAN, LAN, MAN, WAN):

  1. Una red que conecta dispositivos en una casa.
  2. Una red que conecta varias oficinas en una ciudad.
  3. Una red que conecta computadoras en una oficina.
  4. Una red que conecta sucursales de una empresa en diferentes países.

Solución:

  1. PAN
  2. MAN
  3. LAN
  4. WAN

Ejercicio 2: Identificación de Capas del Modelo OSI

Instrucciones: Asocia las siguientes funciones con la capa correspondiente del modelo OSI:

  1. Enrutamiento de paquetes de datos.
  2. Traducción de datos entre el formato de red y el formato de aplicación.
  3. Transmisión de datos en forma de señales eléctricas.
  4. Transferencia fiable de datos entre sistemas finales.

Solución:

  1. Capa de Red
  2. Capa de Presentación
  3. Capa Física
  4. Capa de Transporte

Conclusión

En este módulo, hemos cubierto los fundamentos de las redes de computadoras, incluyendo los tipos de redes, componentes, modelos de referencia OSI y TCP/IP, direcciones IP y enrutamiento. Estos conceptos son esenciales para comprender cómo se comunican los dispositivos en una red y cómo proteger esta comunicación. En el próximo módulo, exploraremos los protocolos de seguridad en redes para profundizar en cómo proteger estas infraestructuras críticas.
