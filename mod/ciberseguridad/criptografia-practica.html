Introducción a la Criptografía

La criptografía es una disciplina fundamental en la ciberseguridad que se encarga de proteger la información mediante técnicas de codificación. En este tema, exploraremos cómo se aplica la criptografía en la práctica, sus principales algoritmos y cómo implementarlos en el desarrollo de software.

Conceptos Clave

  • Cifrado y Descifrado: Proceso de convertir información legible en un formato codificado y viceversa.
  • Clave Criptográfica: Información utilizada para cifrar y descifrar datos.
  • Algoritmos Simétricos: Utilizan la misma clave para cifrar y descifrar.
  • Algoritmos Asimétricos: Utilizan un par de claves (pública y privada) para cifrar y descifrar.
  • Hashing: Transformación de datos en una cadena de longitud fija que representa los datos originales.

Algoritmos de Cifrado Simétrico

Los algoritmos simétricos son rápidos y eficientes para cifrar grandes cantidades de datos. Algunos de los más utilizados son:

  • AES (Advanced Encryption Standard): Algoritmo de cifrado por bloques que utiliza claves de 128, 192 o 256 bits.
  • DES (Data Encryption Standard): Algoritmo más antiguo y menos seguro que AES, utiliza una clave de 56 bits.
  • 3DES (Triple DES): Mejora de DES que aplica el cifrado tres veces para mayor seguridad.

Ejemplo de Cifrado Simétrico con AES en Python

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clave de 16 bytes (128 bits)
key = get_random_bytes(16)

# Crear un objeto de cifrado AES
cipher = AES.new(key, AES.MODE_EAX)

# Datos a cifrar
data = b'Este es un mensaje secreto'

# Cifrar los datos
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(data)

print(f'Cifrado: {ciphertext}')

En este ejemplo, utilizamos la biblioteca pycryptodome para cifrar un mensaje con AES.

Algoritmos de Cifrado Asimétrico

Los algoritmos asimétricos son más lentos pero permiten la distribución segura de claves. Los más comunes son:

  • RSA (Rivest-Shamir-Adleman): Utiliza claves de longitud variable, comúnmente 2048 o 4096 bits.
  • ECC (Elliptic Curve Cryptography): Ofrece la misma seguridad que RSA pero con claves más cortas.

Ejemplo de Cifrado Asimétrico con RSA en Python

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP
from Crypto.Random import get_random_bytes

# Generar un par de claves RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objeto de cifrado RSA
cipher = PKCS1_OAEP.new(public_key)

# Datos a cifrar
data = b'Este es un mensaje secreto'

# Cifrar los datos
ciphertext = cipher.encrypt(data)

print(f'Cifrado: {ciphertext}')

En este ejemplo, utilizamos la biblioteca pycryptodome para cifrar un mensaje con RSA.

Hashing

El hashing es una técnica utilizada para verificar la integridad de los datos. Los algoritmos de hashing más comunes son:

  • SHA-256 (Secure Hash Algorithm 256 bits): Produce un hash de 256 bits.
  • MD5 (Message Digest Algorithm 5): Produce un hash de 128 bits, pero es menos seguro que SHA-256.

Ejemplo de Hashing con SHA-256 en Python

import hashlib

# Datos a hashear
data = b'Este es un mensaje secreto'

# Crear un objeto de hashing SHA-256
hash_object = hashlib.sha256(data)

# Obtener el hash en formato hexadecimal
hash_hex = hash_object.hexdigest()

print(f'Hash: {hash_hex}')

En este ejemplo, utilizamos la biblioteca hashlib para generar un hash SHA-256 de un mensaje.

Conclusión

La criptografía es esencial para proteger la información en el ámbito de la ciberseguridad. Conocer y aplicar correctamente los diferentes algoritmos de cifrado y hashing permite asegurar la confidencialidad, integridad y autenticidad de los datos. En este tema, hemos explorado los conceptos clave, los algoritmos más utilizados y cómo implementarlos en Python, proporcionando una base sólida para su aplicación práctica en el desarrollo de software seguro.
