Introducción

En esta sección, te guiaremos a través del proceso de crear una aplicación Android simple usando Kotlin. Este curso está diseñado para principiantes y te llevará gradualmente a un nivel avanzado. Al final de este tema, tendrás una comprensión sólida de cómo desarrollar una aplicación Android básica.

Configurando Tu Entorno de Desarrollo

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Android Studio: Descarga e instala la última versión de Android Studio desde el sitio web oficial.
  • Configurar Android Studio: Sigue el asistente de configuración para configurar Android Studio e instalar los SDK necesarios.
  • Crear un Nuevo Proyecto: Abre Android Studio y crea un nuevo proyecto con una actividad vacía.

Entendiendo la Estructura del Proyecto

Un proyecto Android consiste en varios componentes clave:

  • Archivo Manifest: AndroidManifest.xml contiene información esencial sobre tu aplicación.
  • Archivos Java/Kotlin: Contiene el código fuente de tu aplicación.
  • Recursos: Incluye diseños, cadenas, imágenes y otros recursos.

Ejemplo de Estructura del Proyecto

Creando Tu Primera Actividad

Una actividad representa una sola pantalla con una interfaz de usuario.

Paso 1: Definir el Diseño

Crea un diseño simple para tu actividad en activity_main.xml.

<!-- res/layout/activity_main.xml -->
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <TextView
        android:id="@+id/helloTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:layout_centerInParent="true"/>
</RelativeLayout>

Paso 2: Implementar la Actividad

Escribe el código Kotlin para tu actividad en MainActivity.kt.

// MainActivity.kt
package com.example.myapplication

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}

Ejecutando Tu Aplicación

  • Conectar un Dispositivo: Conecta un dispositivo Android vía USB o usa un emulador.
  • Ejecutar la Aplicación: Haz clic en el botón "Run" en Android Studio para construir y desplegar tu aplicación.

Añadiendo Interactividad

Vamos a añadir un botón para cambiar el texto del TextView cuando se haga clic.

Paso 1: Actualizar el Diseño

Añade un botón a activity_main.xml.

<!-- res/layout/activity_main.xml -->
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <TextView
        android:id="@+id/helloTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:layout_centerInParent="true"/>

    <Button
        android:id="@+id/changeTextButton"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Change Text"
        android:layout_below="@id/helloTextView"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="20dp"/>
</RelativeLayout>

Paso 2: Actualizar la Actividad

Modifica MainActivity.kt para manejar los clics del botón.

// MainActivity.kt
package com.example.myapplication

import android.os.Bundle
import android.widget.Button
import android.widget.TextView
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        val helloTextView: TextView = findViewById(R.id.helloTextView)
        val changeTextButton: Button = findViewById(R.id.changeTextButton)

        changeTextButton.setOnClickListener {
            helloTextView.text = "Text Changed!"
        }
    }
}

Conclusión

En este tema, cubrimos los conceptos básicos de la creación de una aplicación Android simple usando Kotlin. Configuramos el entorno de desarrollo, entendimos la estructura del proyecto, creamos una actividad y añadimos interactividad. Este conocimiento fundamental te ayudará a medida que avances a temas más avanzados en el desarrollo de Android. ¡Sigue experimentando y construyendo aplicaciones más complejas para mejorar tus habilidades!
