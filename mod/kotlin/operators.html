Introducción

Los operadores son símbolos especiales o palabras clave utilizadas para realizar operaciones en variables y valores. En Kotlin, los operadores se categorizan en varios tipos, como aritméticos, relacionales, lógicos y más. Entender cómo usar estos operadores es fundamental para escribir código Kotlin efectivo.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • % : Módulo (resto de la división)

Ejemplo

fun main() {
    val a = 10
    val b = 5

    println("Adición: ${a + b}")        // Salida: 15
    println("Sustracción: ${a - b}")    // Salida: 5
    println("Multiplicación: ${a * b}") // Salida: 50
    println("División: ${a / b}")       // Salida: 2
    println("Módulo: ${a % b}")         // Salida: 0
}

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores. Devuelven un resultado Booleano.

  • == : Igual a
  • != : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a

Ejemplo

fun main() {
    val a = 10
    val b = 5

    println("Igual a: ${a == b}")       // Salida: false
    println("No igual a: ${a != b}")    // Salida: true
    println("Mayor que: ${a > b}")      // Salida: true
    println("Menor que: ${a < b}")      // Salida: false
    println("Mayor o igual: ${a >= b}") // Salida: true
    println("Menor o igual: ${a <= b}") // Salida: false
}

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones Booleanas.

  • && : AND lógico
  • || : OR lógico
  • ! : NOT lógico

Ejemplo

fun main() {
    val a = true
    val b = false

    println("AND lógico: ${a && b}")    // Salida: false
    println("OR lógico: ${a || b}")     // Salida: true
    println("NOT lógico: ${!a}")        // Salida: false
}

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. También pueden combinarse con operadores aritméticos.

  • = : Asignación simple
  • += : Sumar y asignar
  • -= : Restar y asignar
  • *= : Multiplicar y asignar
  • /= : Dividir y asignar
  • %= : Módulo y asignar

Ejemplo

fun main() {
    var a = 10

    a += 5
    println("Sumar y asignar: $a")        // Salida: 15

    a -= 3
    println("Restar y asignar: $a")       // Salida: 12

    a *= 2
    println("Multiplicar y asignar: $a")  // Salida: 24

    a /= 4
    println("Dividir y asignar: $a")      // Salida: 6

    a %= 5
    println("Módulo y asignar: $a")       // Salida: 1
}

Operadores Unarios

Los operadores unarios se utilizan con solo un operando.

  • + : Unario más
  • - : Unario menos
  • ++ : Incremento
  • -- : Decremento
  • ! : NOT lógico

Ejemplo

fun main() {
    var a = 5

    println("Unario más: ${+a}")         // Salida: 5
    println("Unario menos: ${-a}")       // Salida: -5

    a++
    println("Incremento: $a")            // Salida: 6

    a--
    println("Decremento: $a")            // Salida: 5

    val b = false
    println("NOT lógico: ${!b}")         // Salida: true
}

Operadores a Nivel de Bits

Los operadores a nivel de bits se utilizan para realizar operaciones a nivel de bits.

  • shl : Desplazamiento a la izquierda con signo
  • shr : Desplazamiento a la derecha con signo
  • ushr : Desplazamiento a la derecha sin signo
  • and : AND a nivel de bits
  • or : OR a nivel de bits
  • xor : XOR a nivel de bits
  • inv : Inversión a nivel de bits

Ejemplo

fun main() {
    val a = 5  // 0101 en binario
    val b = 3  // 0011 en binario

    println("AND a nivel de bits: ${a and b}")   // Salida: 1 (0001 en binario)
    println("OR a nivel de bits: ${a or b}")     // Salida: 7 (0111 en binario)
    println("XOR a nivel de bits: ${a xor b}")   // Salida: 6 (0110 en binario)
    println("Inversión a nivel de bits: ${a.inv()}") // Salida: -6 (bits invertidos de 5)
    println("Desplazamiento a la izquierda con signo: ${a shl 1}") // Salida: 10 (1010 en binario)
    println("Desplazamiento a la derecha con signo: ${a shr 1}") // Salida: 2 (0010 en binario)
    println("Desplazamiento a la derecha sin signo: ${a ushr 1}") // Salida: 2 (0010 en binario)
}

Conclusión

Los operadores son una parte fundamental de la programación en Kotlin, permitiéndote realizar una amplia gama de operaciones en variables y valores. Desde aritmética básica hasta operaciones complejas a nivel de bits, entender cómo usar estos operadores de manera efectiva mejorará tu capacidad para escribir código Kotlin eficiente y legible. Practica usando estos operadores en diferentes escenarios para sentirte más cómodo con su funcionalidad.
