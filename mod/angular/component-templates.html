Introducción a las Plantillas de Componentes

Las plantillas de componentes en Angular definen la vista para un componente. Están escritas en HTML y pueden incluir sintaxis específica de Angular para enlazar datos y manejar interacciones del usuario.

  • Componente: Un bloque de construcción de aplicaciones Angular, que consiste en una clase TypeScript, una plantilla HTML y estilos CSS opcionales.
  • Plantilla: La parte HTML de un componente que define lo que el usuario ve.

Sintaxis Básica de Plantillas

Las plantillas de Angular usan una combinación de HTML estándar y sintaxis de plantillas de Angular para crear vistas dinámicas.

Interpolación

La interpolación te permite incrustar expresiones en tu HTML.

<p>{{ title }}</p>

En este ejemplo, {{ title }} será reemplazado con el valor de la propiedad title de la clase del componente.

Enlace de Propiedades

El enlace de propiedades te permite establecer propiedades de elementos HTML o directivas.

<img [src]="imageUrl">

Aquí, [src] enlaza el atributo src del elemento img a la propiedad imageUrl en el componente.

Enlace de Eventos

El enlace de eventos te permite escuchar y responder a acciones del usuario.

<button (click)="onClick()">Click me</button>

En este ejemplo, (click) enlaza el evento de clic del botón al método onClick en el componente.

Directivas Estructurales

Las directivas estructurales alteran el diseño añadiendo o eliminando elementos del DOM.

*ngIf

La directiva *ngIf incluye condicionalmente una plantilla basada en el valor de una expresión.

<p *ngIf="isVisible">Este párrafo es visible.</p>

El párrafo solo se mostrará si isVisible es true.

*ngFor

La directiva *ngFor repite una plantilla para cada elemento en una lista.

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

Esto creará un elemento de lista por cada elemento en el array items.

Directivas de Atributo

Las directivas de atributo cambian la apariencia o el comportamiento de un elemento.

ngClass

La directiva ngClass añade y elimina clases CSS.

<div [ngClass]="{ 'active': isActive }">Contenido</div>

La clase active se añadirá al div si isActive es true.

ngStyle

La directiva ngStyle añade y elimina estilos en línea.

<div [ngStyle]="{ 'color': textColor }">Texto estilizado</div>

El estilo color se establecerá al valor de textColor.

Variables de Referencia de Plantilla

Las variables de referencia de plantilla te permiten referenciar un elemento del DOM dentro de una plantilla.

<input #inputElement type="text">
<button (click)="logValue(inputElement.value)">Log Value</button>

En este ejemplo, #inputElement es una variable de referencia de plantilla que se refiere al elemento input. El método logValue puede acceder al valor del input.

Conclusión

Las plantillas de componentes son una parte fundamental de Angular, permitiéndote crear interfaces de usuario dinámicas e interactivas. Al entender y utilizar la interpolación, el enlace de propiedades, el enlace de eventos, las directivas estructurales, las directivas de atributo y las variables de referencia de plantilla, puedes construir aplicaciones Angular poderosas y responsivas.
