Formularios Dinámicos en Angular
Introducción a la Gestión de Estado ►

Los formularios dinámicos en Angular te permiten crear formularios que pueden cambiar según la interacción del usuario u otras condiciones. Esta es una característica poderosa que puede hacer que tus aplicaciones sean más flexibles y fáciles de usar.

Conceptos Clave

  • Formularios Reactivos: Un enfoque impulsado por modelos para manejar entradas de formularios cuyos valores cambian con el tiempo.
  • FormBuilder: Un servicio que proporciona métodos convenientes para crear controles de formularios.
  • FormGroup y FormControl: Clases que representan un grupo de controles de formularios y controles de formularios individuales, respectivamente.
  • FormArray: Una clase que representa un array de controles de formularios, útil para formularios dinámicos.
  • Validadores: Funciones que verifican la validez de los controles de formularios.

Configuración de Formularios Reactivos

Para comenzar a usar formularios reactivos, necesitas importar los módulos necesarios en tu aplicación Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // otros imports
  ],
  // otras configuraciones
})
export class AppModule { }

Creando un Formulario Reactivo Básico

Vamos a crear un formulario simple con un campo de nombre y correo electrónico.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-basic-form',
  template: `
    <form [formGroup]="basicForm" (ngSubmit)="onSubmit()">
      <label for="name">Nombre:</label>
      <input id="name" formControlName="name">
      <div *ngIf="basicForm.get('name').invalid && basicForm.get('name').touched">
        El nombre es obligatorio.
      </div>

      <label for="email">Correo Electrónico:</label>
      <input id="email" formControlName="email">
      <div *ngIf="basicForm.get('email').invalid && basicForm.get('email').touched">
        Introduce un correo electrónico válido.
      </div>

      <button type="submit" [disabled]="basicForm.invalid">Enviar</button>
    </form>
  `
})
export class BasicFormComponent {
  basicForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.basicForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.basicForm.valid) {
      console.log(this.basicForm.value);
    }
  }
}

Añadiendo Controles Dinámicos

Para añadir controles dinámicos, puedes usar FormArray. Vamos a extender nuestro formulario para incluir una lista de números de teléfono.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, FormArray, Validators } from '@angular/forms';

@Component({
  selector: 'app-dynamic-form',
  template: `
    <form [formGroup]="dynamicForm" (ngSubmit)="onSubmit()">
      <label for="name">Nombre:</label>
      <input id="name" formControlName="name">
      <div *ngIf="dynamicForm.get('name').invalid && dynamicForm.get('name').touched">
        El nombre es obligatorio.
      </div>

      <label for="email">Correo Electrónico:</label>
      <input id="email" formControlName="email">
      <div *ngIf="dynamicForm.get('email').invalid && dynamicForm.get('email').touched">
        Introduce un correo electrónico válido.
      </div>

      <div formArrayName="phones">
        <div *ngFor="let phone of phones.controls; let i=index">
          <label for="phone{{i}}">Teléfono {{i + 1}}:</label>
          <input [id]="'phone' + i" [formControlName]="i">
          <button type="button" (click)="removePhone(i)">Eliminar</button>
        </div>
      </div>
      <button type="button" (click)="addPhone()">Añadir Teléfono</button>

      <button type="submit" [disabled]="dynamicForm.invalid">Enviar</button>
    </form>
  `
})
export class DynamicFormComponent {
  dynamicForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.dynamicForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]],
      phones: this.fb.array([])
    });
  }

  get phones() {
    return this.dynamicForm.get('phones') as FormArray;
  }

  addPhone() {
    this.phones.push(this.fb.control('', Validators.required));
  }

  removePhone(index: number) {
    this.phones.removeAt(index);
  }

  onSubmit() {
    if (this.dynamicForm.valid) {
      console.log(this.dynamicForm.value);
    }
  }
}

Técnicas Avanzadas

Validación Condicional

Puedes añadir o eliminar validadores basándote en ciertas condiciones.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-conditional-validation',
  template: `
    <form [formGroup]="conditionalForm" (ngSubmit)="onSubmit()">
      <label for="username">Nombre de Usuario:</label>
      <input id="username" formControlName="username">
      <div *ngIf="conditionalForm.get('username').invalid && conditionalForm.get('username').touched">
        El nombre de usuario es obligatorio.
      </div>

      <label for="password">Contraseña:</label>
      <input id="password" formControlName="password" type="password">
      <div *ngIf="conditionalForm.get('password').invalid && conditionalForm.get('password').touched">
        La contraseña es obligatoria.
      </div>

      <label for="confirmPassword">Confirmar Contraseña:</label>
      <input id="confirmPassword" formControlName="confirmPassword" type="password">
      <div *ngIf="conditionalForm.get('confirmPassword').invalid && conditionalForm.get('confirmPassword').touched">
        Las contraseñas deben coincidir.
      </div>

      <button type="submit" [disabled]="conditionalForm.invalid">Enviar</button>
    </form>
  `
})
export class ConditionalValidationComponent {
  conditionalForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.conditionalForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', Validators.required],
      confirmPassword: ['']
    });

    this.conditionalForm.get('confirmPassword').setValidators([
      Validators.required,
      this.passwordMatchValidator.bind(this)
    ]);
  }

  passwordMatchValidator(control: FormControl): { [key: string]: boolean } | null {
    if (this.conditionalForm) {
      return control.value === this.conditionalForm.get('password').value ? null : { mismatch: true };
    }
    return null;
  }

  onSubmit() {
    if (this.conditionalForm.valid) {
      console.log(this.conditionalForm.value);
    }
  }
}

Resumen

Los formularios dinámicos en Angular proporcionan una forma poderosa de crear formularios flexibles y fáciles de usar. Al aprovechar los formularios reactivos, FormBuilder, FormGroup, FormControl y FormArray, puedes crear formularios que se adapten a la entrada del usuario y otras condiciones. Las técnicas avanzadas como la validación condicional mejoran aún más las capacidades de tus formularios. Comprender y utilizar estos conceptos te permitirá construir formularios robustos y dinámicos en tus aplicaciones Angular.
