Configuración del Entorno de Desarrollo
Arquitectura de Angular ►

Configurar el entorno de desarrollo es el primer paso para comenzar a trabajar con Angular. Esta sección te guiará a través de las herramientas y configuraciones necesarias para poner en marcha tus proyectos de Angular.

Requisitos Previos

Antes de configurar Angular, asegúrate de tener los siguientes requisitos previos instalados en tu máquina:

  • Node.js y npm: Angular requiere Node.js y npm (Node Package Manager) para gestionar dependencias y ejecutar scripts.
  • Editor de Texto o IDE: Un buen editor de texto o Entorno de Desarrollo Integrado (IDE) como Visual Studio Code, WebStorm o Sublime Text.

Instalación de Node.js y npm

  1. Descargar e Instalar Node.js:

    • Visita el sitio web oficial de Node.js.
    • Descarga la versión LTS (Long Term Support) para tu sistema operativo.
    • Sigue las instrucciones de instalación.

  2. Verificar la Instalación:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta los siguientes comandos para verificar las versiones instaladas: 
      node -v
npm -v
    • Deberías ver los números de versión de Node.js y npm.

Instalación de Angular CLI

El Angular CLI (Command Line Interface) es una herramienta poderosa que te ayuda a inicializar, desarrollar, estructurar y mantener aplicaciones Angular.

  1. Instalar Angular CLI Globalmente:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta el siguiente comando para instalar Angular CLI globalmente: 
      npm install -g @angular/cli
    • Este comando instala Angular CLI globalmente, haciéndolo accesible desde cualquier directorio en tu máquina.

  2. Verificar la Instalación:

    • Ejecuta el siguiente comando para verificar la versión instalada de Angular CLI: 
      ng version
    • Deberías ver los detalles de la versión de Angular CLI y sus dependencias.

Creación de un Nuevo Proyecto Angular

Una vez que Angular CLI está instalado, puedes crear un nuevo proyecto Angular usando un comando simple.

  1. Crear un Nuevo Proyecto:

    • Navega al directorio donde deseas crear tu proyecto.
    • Ejecuta el siguiente comando: 
      ng new my-angular-app
    • Reemplaza my-angular-app con el nombre de proyecto que desees.
    • El CLI te pedirá algunas opciones de configuración. Puedes elegir las opciones predeterminadas presionando Enter.

  2. Navegar al Directorio del Proyecto:

    • Después de que el proyecto se haya creado, navega al directorio del proyecto: 
      cd my-angular-app

  3. Servir la Aplicación:

    • Ejecuta el siguiente comando para servir la aplicación: 
      ng serve
    • Abre tu navegador y navega a http://localhost:4200/. Deberías ver la página de bienvenida predeterminada de Angular.

Configuración del Entorno de Desarrollo

Para mejorar tu experiencia de desarrollo, considera configurar tu editor de texto o IDE con las siguientes herramientas y extensiones:

  • Extensiones de Visual Studio Code:

    • Angular Language Service: Proporciona capacidades ricas de edición y navegación.
    • TSLint: Ayuda a identificar y corregir problemas de código TypeScript.
    • Prettier: Formatea automáticamente tu código para mantener la consistencia.

  • Plugins de WebStorm:

    • Angular and AngularJS: Proporciona soporte para proyectos Angular.
    • TypeScript: Mejora el desarrollo con TypeScript.

Ejemplo de Configuración para Visual Studio Code

  1. Instalar Extensiones:

    • Abre Visual Studio Code.
    • Ve a la vista de Extensiones haciendo clic en el icono de Extensiones en la Barra de Actividad en el lado de la ventana.
    • Busca e instala las siguientes extensiones:
      • Angular Language Service
      • TSLint
      • Prettier

  2. Configurar Prettier:

    • Crea un archivo .prettierrc en la raíz de tu proyecto con el siguiente contenido: 
      {
  "singleQuote": true,
  "trailingComma": "all"
}
    • Esta configuración asegura que se usen comillas simples y se agreguen comas finales donde sean válidas en ES5 (objetos, arreglos, etc.).

Conclusión

Configurar el entorno de desarrollo es un paso crucial para comenzar tu viaje con Angular. Al asegurarte de tener Node.js, npm y Angular CLI instalados, y configurar tu editor de texto o IDE con extensiones útiles, creas un espacio de trabajo productivo y eficiente. Con tu entorno listo, ahora puedes pasar a construir y desarrollar aplicaciones Angular con facilidad.
