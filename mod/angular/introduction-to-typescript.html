¿Qué es TypeScript?

TypeScript es un superconjunto de JavaScript con tipado estático que se compila en JavaScript plano. Añade tipos estáticos opcionales, clases e interfaces a JavaScript, lo que facilita la escritura y el mantenimiento de aplicaciones a gran escala.

  • Tipado Estático: TypeScript te permite definir tipos para variables, parámetros de funciones y valores de retorno.
  • Superconjunto de JavaScript: Todo el código JavaScript válido también es código TypeScript válido.
  • Se compila a JavaScript: El código TypeScript se transpila a JavaScript, que puede ejecutarse en cualquier entorno que soporte JavaScript.

¿Por qué usar TypeScript en Angular?

Angular está construido con TypeScript, y usar TypeScript en tus proyectos Angular ofrece varios beneficios:

  • Seguridad de Tipos: Ayuda a detectar errores en tiempo de compilación en lugar de en tiempo de ejecución.
  • Soporte Mejorado en IDE: Mejor autocompletado, navegación y herramientas de refactorización.
  • Mejor Legibilidad del Código: Código más claro y mantenible con tipos e interfaces.
  • Características Modernas de JavaScript: Acceso a características de ES6/ES7 y más allá.

Configuración de TypeScript

Para comenzar a usar TypeScript, necesitas instalarlo globalmente en tu máquina y configurar un archivo de configuración.

Instalando TypeScript

npm install -g typescript

Creando un tsconfig.json

El archivo tsconfig.json especifica los archivos raíz y las opciones del compilador necesarias para compilar el proyecto.

{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src"]
}

Sintaxis Básica de TypeScript

Variables y Tipos

TypeScript te permite definir tipos para variables.

let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];

Funciones

Puedes especificar tipos para los parámetros de las funciones y los valores de retorno.

function add(x: number, y: number): number {
  return x + y;
}

Interfaces

Las interfaces definen la estructura de un objeto.

interface Person {
  firstName: string;
  lastName: string;
}

function greet(person: Person) {
  return "Hello, " + person.firstName + " " + person.lastName;
}

let user = { firstName: "John", lastName: "Doe" };
console.log(greet(user));

Clases

TypeScript soporta clases de ES6 con características adicionales como modificadores de acceso.

class Animal {
  private name: string;

  constructor(name: string) {
    this.name = name;
  }

  public move(distanceInMeters: number): void {
    console.log(`${this.name} moved ${distanceInMeters}m.`);
  }
}

let dog = new Animal("Dog");
dog.move(10);

TypeScript en Angular

Angular CLI y TypeScript

Cuando creas un nuevo proyecto Angular usando Angular CLI, TypeScript se configura por defecto.

ng new my-angular-app

Usando Tipos en Componentes de Angular

En Angular, puedes usar TypeScript para definir los tipos de las propiedades y métodos del componente.

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title: string = 'my-angular-app';

  changeTitle(newTitle: string): void {
    this.title = newTitle;
  }
}

Conclusión

TypeScript es una herramienta poderosa que mejora JavaScript añadiendo tipos estáticos, interfaces y características modernas de JavaScript. Es una parte integral del desarrollo en Angular, proporcionando seguridad de tipos, mejores herramientas y mejor mantenibilidad del código. Al entender los conceptos básicos de TypeScript, puedes escribir aplicaciones Angular más robustas y escalables.
