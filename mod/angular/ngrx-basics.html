Conceptos Básicos de NgRx
NgRx Avanzado ►

NgRx es una poderosa biblioteca para gestionar el estado en aplicaciones Angular utilizando el patrón Redux. Esta sección te introducirá a los conceptos básicos de NgRx, sus beneficios y cómo implementarlo en tus proyectos Angular.

Introducción a NgRx

NgRx es una biblioteca de gestión de estado para aplicaciones Angular, inspirada en Redux. Ayuda a gestionar el estado de una aplicación de manera predecible, facilitando la depuración y las pruebas.

Conceptos Clave

  • Estado: Representa el estado de la aplicación en un momento dado.
  • Acciones: Objetos simples que describen lo que sucedió en la aplicación.
  • Reductores: Funciones que toman el estado actual y una acción, y devuelven un nuevo estado.
  • Selectores: Funciones que seleccionan una parte del estado de la tienda.
  • Efectos: Efectos secundarios que interactúan con servicios externos y despachan acciones.

Configuración de NgRx

Para comenzar a usar NgRx en tu proyecto Angular, necesitas instalar los paquetes necesarios y configurar la tienda.

Instalación

Ejecuta el siguiente comando para instalar los paquetes de NgRx:

npm install @ngrx/store @ngrx/effects @ngrx/entity @ngrx/store-devtools

Configuración de la Tienda

Crea un nuevo archivo app.reducer.ts para definir el estado raíz y los reductores.

import { ActionReducerMap } from '@ngrx/store';
import { counterReducer } from './counter.reducer';

export interface AppState {
  counter: number;
}

export const reducers: ActionReducerMap<AppState> = {
  counter: counterReducer
};

En tu app.module.ts, importa el StoreModule y proporciona los reductores.

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { StoreModule } from '@ngrx/store';
import { reducers } from './app.reducer';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    StoreModule.forRoot(reducers)
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Acciones

Las acciones se despachan para señalar que algo sucedió en la aplicación.

Definición de Acciones

Crea un nuevo archivo counter.actions.ts para definir acciones para el contador.

import { createAction } from '@ngrx/store';

export const increment = createAction('[Counter] Increment');
export const decrement = createAction('[Counter] Decrement');
export const reset = createAction('[Counter] Reset');

Reductores

Los reductores especifican cómo cambia el estado de la aplicación en respuesta a las acciones.

Creación de un Reductor

Crea un nuevo archivo counter.reducer.ts para definir el reductor del contador.

import { createReducer, on } from '@ngrx/store';
import { increment, decrement, reset } from './counter.actions';

export const initialState = 0;

const _counterReducer = createReducer(
  initialState,
  on(increment, state => state + 1),
  on(decrement, state => state - 1),
  on(reset, state => 0)
);

export function counterReducer(state, action) {
  return _counterReducer(state, action);
}

Selectores

Los selectores se utilizan para consultar la tienda en busca de piezas específicas del estado.

Creación de Selectores

Crea un nuevo archivo counter.selectors.ts para definir selectores para el contador.

import { createSelector, createFeatureSelector } from '@ngrx/store';
import { AppState } from './app.reducer';

export const selectCounter = createFeatureSelector<AppState, number>('counter');

Efectos

Los efectos manejan efectos secundarios como llamadas a API y despachan nuevas acciones basadas en los resultados.

Creación de Efectos

Crea un nuevo archivo counter.effects.ts para definir efectos para el contador.

import { Injectable } from '@angular/core';
import { Actions, createEffect, ofType } from '@ngrx/effects';
import { of } from 'rxjs';
import { map, mergeMap, catchError } from 'rxjs/operators';
import { increment, decrement, reset } from './counter.actions';

@Injectable()
export class CounterEffects {
  constructor(private actions$: Actions) {}

  increment$ = createEffect(() =>
    this.actions$.pipe(
      ofType(increment),
      mergeMap(() => {
        // Simular una llamada a API
        return of({}).pipe(
          map(() => increment()),
          catchError(() => of(reset()))
        );
      })
    )
  );
}

Conclusión

NgRx proporciona un marco robusto para gestionar el estado en aplicaciones Angular. Al comprender e implementar acciones, reductores, selectores y efectos, puedes crear aplicaciones escalables y mantenibles. Practica creando una aplicación de contador simple y agrega gradualmente más complejidad a medida que te sientas cómodo con los conceptos.
