Introducción a las Tuberías

Las tuberías en Angular son una característica poderosa que te permite transformar datos directamente en tus plantillas. Son funciones simples que aceptan un valor de entrada y devuelven un valor transformado. Angular proporciona varias tuberías integradas que cubren necesidades comunes de transformación de datos.

Tuberías Integradas Clave

Angular viene con una variedad de tuberías integradas que puedes usar directamente. Aquí tienes algunas de las más comúnmente utilizadas:

  • DatePipe: Formatea un valor de fecha según las reglas de la localidad.
  • UpperCasePipe: Transforma texto a mayúsculas.
  • LowerCasePipe: Transforma texto a minúsculas.
  • CurrencyPipe: Transforma un número en una cadena de moneda.
  • DecimalPipe: Transforma un número en una cadena con un formato decimal especificado.
  • PercentPipe: Transforma un número en una cadena de porcentaje.
  • JsonPipe: Convierte un valor en una cadena JSON.
  • SlicePipe: Crea un nuevo arreglo o cadena que contiene un subconjunto (segmento) de los elementos.

Uso de Tuberías Integradas

DatePipe

La DatePipe se usa para formatear valores de fecha. Puedes especificar el formato en el que quieres que se muestre la fecha.

<p>{{ today | date:'fullDate' }}</p>

En este ejemplo, today es un objeto de fecha, y fullDate es un formato predefinido que muestra la fecha completa.

UpperCasePipe y LowerCasePipe

Estas tuberías se usan para transformar texto a mayúsculas o minúsculas.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

La CurrencyPipe se usa para formatear números como moneda.

<p>{{ price | currency:'USD' }}</p>

Aquí, price es un número, y 'USD' especifica el formato de la moneda.

DecimalPipe

La DecimalPipe formatea números con un formato decimal especificado.

<p>{{ 3.14159 | number:'1.2-2' }}</p>

En este ejemplo, 1.2-2 especifica que el número debe tener al menos un dígito antes del punto decimal, y entre dos y dos dígitos después del punto decimal.

PercentPipe

La PercentPipe formatea números como porcentajes.

<p>{{ 0.25 | percent }}</p>

JsonPipe

La JsonPipe convierte un valor en una cadena JSON, lo cual es útil para depuración.

<p>{{ myObject | json }}</p>

SlicePipe

La SlicePipe crea un nuevo arreglo o cadena que contiene un subconjunto de los elementos.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

Comparación de Tuberías Comunes

| Tubería | Propósito | Ejemplo de Uso | |---------------|----------------------------------------------|------------------------------------------| | DatePipe | Formatea valores de fecha | {{ today | date:'shortDate' }} | | UpperCasePipe | Transforma texto a mayúsculas | {{ 'hello' | uppercase }} | | LowerCasePipe | Transforma texto a minúsculas | {{ 'HELLO' | lowercase }} | | CurrencyPipe | Formatea números como moneda | {{ price | currency:'USD' }} | | DecimalPipe | Formatea números con lugares decimales especificados | {{ 3.14159 | number:'1.2-2' }} | | PercentPipe | Formatea números como porcentajes | {{ 0.25 | percent }} | | JsonPipe | Convierte un valor en una cadena JSON | {{ myObject | json }} | | SlicePipe | Crea un subconjunto de un arreglo o cadena | {{ 'Hello' | slice:0:2 }} |

Conclusión

Las tuberías integradas en Angular proporcionan una manera conveniente de transformar datos directamente en tus plantillas. Ayudan a mantener tu código limpio y legible al manejar tareas comunes de transformación de datos. Entender y utilizar estas tuberías puede mejorar significativamente tus aplicaciones Angular al hacer tus plantillas más expresivas y fáciles de mantener.
