Variables y Mutabilidad en Rust
Tipos de Datos ►

Introducción

En Rust, las variables son bloques de construcción fundamentales que almacenan datos. Entender cómo declarar, usar y modificar variables es crucial para escribir programas efectivos en Rust. Esta sección cubrirá los conceptos básicos de las variables, el concepto de mutabilidad y cómo trabajar con variables mutables e inmutables.

Declaración de Variables

En Rust, las variables se declaran usando la palabra clave let. Por defecto, las variables son inmutables, lo que significa que sus valores no pueden cambiar una vez establecidos.

fn main() {
    let x = 5;
    println!("El valor de x es: {}", x);
}
  • let: Palabra clave usada para declarar una variable.
  • x: Nombre de la variable.
  • 5: Valor inicial asignado a la variable x.

Mutabilidad

Para permitir que el valor de una variable cambie, necesitas declararla como mutable usando la palabra clave mut.

fn main() {
    let mut y = 10;
    println!("El valor inicial de y es: {}", y);
    y = 15;
    println!("El nuevo valor de y es: {}", y);
}
  • mut: Palabra clave que indica que la variable es mutable.
  • y = 15;: Cambiando el valor de y después de su declaración inicial.

Sombras

Rust permite declarar una nueva variable con el mismo nombre que una variable anterior. Esto se conoce como sombreado y permite reutilizar nombres de variables mientras se transforman sus valores.

fn main() {
    let z = 6;
    let z = z + 1;
    println!("El valor de z después del sombreado es: {}", z);
}
  • El sombreado permite transformaciones sin necesidad de declarar nuevos nombres de variables.
  • El nuevo z sombrea al viejo z, pero son variables diferentes.

Constantes

Las constantes son similares a las variables inmutables pero se declaran usando la palabra clave const y deben tener su tipo especificado. Las constantes son siempre inmutables y pueden ser declaradas en cualquier ámbito, incluyendo el ámbito global.

const MAX_POINTS: u32 = 100_000;
  • const: Palabra clave usada para declarar una constante.
  • MAX_POINTS: Nombre de la constante, convencionalmente escrito en mayúsculas.
  • u32: Tipo de la constante.
  • 100_000: Valor asignado a la constante.

Tipos de Variables

Rust es un lenguaje de tipado estático, lo que significa que el tipo de una variable debe ser conocido en tiempo de compilación. Rust a menudo puede inferir el tipo basado en el valor, pero también puedes especificar el tipo explícitamente.

fn main() {
    let a: i32 = 42; // Anotación de tipo explícita
    let b = 3.14;    // Tipo inferido como f64
}
  • i32: Entero con signo de 32 bits.
  • f64: Número de punto flotante de 64 bits.

Ejemplos y Ejercicios

Ejemplo 1: Variable Inmutable

fn main() {
    let name = "Alice";
    println!("¡Hola, {}!", name);
}
  • Este ejemplo demuestra una variable inmutable name que contiene una cadena.

Ejemplo 2: Variable Mutable

fn main() {
    let mut counter = 1;
    println!("Contador: {}", counter);
    counter += 1;
    println!("Contador: {}", counter);
}
  • Este ejemplo muestra una variable mutable counter cuyo valor se incrementa.

Ejercicio 1: Declarar y Modificar una Variable Mutable

  1. Declara una variable mutable score y establece su valor inicial en 50.
  2. Imprime el valor de score.
  3. Cambia el valor de score a 75.
  4. Imprime el nuevo valor de score.

Ejercicio 2: Usar Sombreado

  1. Declara una variable inmutable temperature y establece su valor en 20.
  2. Sombrea temperature añadiendo 5 a su valor.
  3. Imprime el nuevo valor de temperature.

Resumen

  • Las variables en Rust son inmutables por defecto.
  • Usa la palabra clave mut para hacer una variable mutable.
  • El sombreado permite reutilizar nombres de variables para transformaciones.
  • Las constantes son siempre inmutables y deben tener su tipo especificado.
  • Rust es de tipado estático, y los tipos pueden ser inferidos o especificados explícitamente.

Entender las variables y la mutabilidad es esencial para gestionar datos y estados en tus programas Rust. Practica declarando, modificando y sombreando variables para volverte competente en el manejo de datos en Rust.
