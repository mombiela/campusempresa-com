Introducción

El programa "¡Hola, Mundo!" es un programa simple pero fundamental que a menudo se utiliza para introducir un nuevo lenguaje de programación. En esta sección, cubriremos cómo escribir y ejecutar un programa "¡Hola, Mundo!" en Rust, explicando cada parte del código en detalle.

Escribiendo el programa "¡Hola, Mundo!"

Configurando tu entorno

Antes de escribir nuestro primer programa en Rust, asegúrate de tener Rust instalado en tu máquina. Puedes instalar Rust siguiendo las instrucciones en el sitio web oficial de Rust.

Creando un nuevo proyecto

Rust utiliza un sistema de construcción y gestor de paquetes llamado Cargo. Para crear un nuevo proyecto, abre tu terminal y ejecuta:

cargo new hello_world
cd hello_world

Esto creará un nuevo directorio llamado hello_world con la siguiente estructura:

hello_world
├── Cargo.toml
└── src
    └── main.rs

Escribiendo el código

Abre el archivo src/main.rs en tu editor de texto favorito y reemplaza su contenido con el siguiente código:

fn main() {
    println!("¡Hola, mundo!");
}

Explicación

  • fn main() { ... }: Esto define la función principal, que es el punto de entrada de un programa en Rust.
  • println!("¡Hola, mundo!");: Esta línea imprime "¡Hola, mundo!" en la consola. println! es una macro en Rust utilizada para imprimir texto.

Ejecutando el programa

Para ejecutar tu programa, utiliza el siguiente comando en tu terminal:

cargo run

Deberías ver la salida:

¡Hola, mundo!

Conceptos clave

  • Funciones: En Rust, las funciones se definen usando la palabra clave fn. La función main es especial porque es el punto de entrada del programa.
  • Macros: La macro println! se utiliza para imprimir texto en la consola. Las macros en Rust se invocan con un ! y pueden realizar generación de código en tiempo de compilación.
  • Cargo: Cargo es el sistema de construcción y gestor de paquetes de Rust, que simplifica la gestión de dependencias y la construcción de proyectos.

Conclusión

En esta sección, cubrimos cómo escribir y ejecutar un simple programa "¡Hola, Mundo!" en Rust. Introdujimos la estructura básica de un programa en Rust, incluyendo la función principal y el uso de la macro println!. Entender estos conceptos básicos es crucial ya que forman la base para programas más complejos en Rust.
