Flujo de Control en Rust
Entendiendo la Propiedad ►

El flujo de control es un concepto fundamental en la programación que te permite dictar el orden en el que se ejecutan las declaraciones y expresiones. En Rust, el flujo de control se gestiona utilizando declaraciones condicionales, bucles y coincidencia de patrones. Esta sección te guiará desde construcciones básicas hasta avanzadas de flujo de control en Rust.

Declaraciones Condicionales

Las declaraciones condicionales te permiten ejecutar código basado en ciertas condiciones.

Declaraciones `if`

La declaración if se usa para ejecutar código solo si una condición es verdadera.

fn main() {
    let number = 7;

    if number < 10 {
        println!("El número es menor que 10");
    }
}

`else` y `else if`

Puedes extender la declaración if con else y else if para manejar múltiples condiciones.

fn main() {
    let number = 7;

    if number < 5 {
        println!("El número es menor que 5");
    } else if number == 5 {
        println!("El número es exactamente 5");
    } else {
        println!("El número es mayor que 5");
    }
}

Usando `if` en una Declaración `let`

En Rust, puedes usar if en una declaración let para asignar un valor basado en una condición.

fn main() {
    let condition = true;
    let number = if condition { 5 } else { 10 };

    println!("El valor del número es: {}", number);
}

Bucles

Los bucles te permiten ejecutar un bloque de código repetidamente.

`loop`

La palabra clave loop crea un bucle infinito.

fn main() {
    let mut counter = 0;

    loop {
        counter += 1;
        println!("Contador: {}", counter);

        if counter == 5 {
            break;
        }
    }
}

`while`

El bucle while se ejecuta mientras una condición sea verdadera.

fn main() {
    let mut number = 3;

    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }

    println!("¡DESPEGUE!!!");
}

`for`

El bucle for se usa para iterar sobre una colección.

fn main() {
    let a = [10, 20, 30, 40, 50];

    for element in a.iter() {
        println!("El valor es: {}", element);
    }
}

Coincidencia de Patrones

La coincidencia de patrones en Rust es una característica poderosa que te permite comparar un valor contra una serie de patrones.

`match`

La declaración match se usa para la coincidencia de patrones.

fn main() {
    let number = 6;

    match number {
        1 => println!("¡Uno!"),
        2 => println!("¡Dos!"),
        3 => println!("¡Tres!"),
        4 | 5 | 6 => println!("¡Cuatro, cinco o seis!"),
        _ => println!("¡Algo más!"),
    }
}

Desestructuración con `match`

Puedes desestructurar tipos de datos complejos con match.

fn main() {
    let pair = (2, -2);

    match pair {
        (x, y) if x == y => println!("Los números son iguales"),
        (x, y) if x + y == 0 => println!("Los números son opuestos"),
        (x, _) => println!("El primer número es {}", x),
    }
}

Conclusión

El flujo de control en Rust se gestiona a través de declaraciones condicionales, bucles y coincidencia de patrones. Entender estas construcciones es esencial para escribir código eficiente y legible en Rust. Al dominar if, else, loop, while, for y match, puedes manejar una amplia gama de escenarios de programación de manera efectiva.
