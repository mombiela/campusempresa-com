Introducción a las Cadenas

Las cadenas son una parte fundamental de la programación, permitiéndote trabajar con texto. En Rust, las cadenas son más complejas que en algunos otros lenguajes debido al enfoque de Rust en la seguridad y el rendimiento.

Conceptos Clave

  • Tipos de Cadenas: Rust tiene dos tipos principales de cadenas: String y &str.
  • Propiedad y Préstamo: Entender cómo Rust maneja la memoria con las cadenas.
  • Codificación UTF-8: Las cadenas en Rust están codificadas en UTF-8, lo que afecta cómo trabajas con ellas.

Tipos de Cadenas

`String`

El tipo String es una estructura de datos asignada en el montón y que puede crecer. Se usa cuando necesitas una cadena propia que puedas modificar.

Ejemplo

fn main() {
    let mut s = String::from("Hello");
    s.push_str(", world!");
    println!("{}", s); // Imprime: Hello, world!
}
  • String::from("Hello"): Crea una nueva String a partir de un literal de cadena.
  • s.push_str(", world!"): Añade una porción de cadena a la String.

`&str`

El tipo &str, también conocido como una porción de cadena, es una referencia a una cadena. Se usa cuando necesitas una vista inmutable de una cadena.

Ejemplo

fn main() {
    let s = "Hello, world!";
    println!("{}", s); // Imprime: Hello, world!
}
  • "Hello, world!": Un literal de cadena, que es de tipo &str.

Propiedad y Préstamo con Cadenas

Propiedad

En Rust, String posee sus datos, lo que significa que es responsable de gestionar la memoria donde se almacena la cadena.

Ejemplo

fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 se mueve a s2
    // println!("{}", s1); // Esto causaría un error en tiempo de compilación
}
  • s1 se mueve a s2, y s1 ya no es válido.

Préstamo

Puedes prestar una cadena usando una referencia, permitiéndote usar la cadena sin tomar posesión.

Ejemplo

fn main() {
    let s1 = String::from("hello");
    let len = calculate_length(&s1);
    println!("La longitud de '{}' es {}.", s1, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}
  • &s1: Se pasa una referencia a s1 a la función.
  • s.len(): Devuelve la longitud de la cadena.

Codificación UTF-8

Las cadenas en Rust están codificadas en UTF-8, lo que significa que pueden contener cualquier secuencia UTF-8 válida. Esto afecta cómo puedes indexar y dividir cadenas.

Ejemplo

fn main() {
    let hello = "Здравствуйте";
    let s = &hello[0..4];
    println!("{}", s); // Imprime: Зд
}
  • &hello[0..4]: Divide la cadena, pero debes tener cuidado de dividir en límites UTF-8 válidos.

Operaciones Comunes con Cadenas

Concatenación

Puedes concatenar cadenas usando el operador + o la macro format!.

Ejemplo

fn main() {
    let s1 = String::from("Hello, ");
    let s2 = String::from("world!");
    let s3 = s1 + &s2; // s1 se mueve aquí y ya no puede ser usado
    println!("{}", s3); // Imprime: Hello, world!
}

Iteración

Puedes iterar sobre cadenas por caracteres o bytes.

Ejemplo

fn main() {
    let s = String::from("hello");
    for c in s.chars() {
        println!("{}", c);
    }
}
  • s.chars(): Devuelve un iterador sobre los caracteres de la cadena.

Conclusión

Las cadenas en Rust son una forma poderosa y flexible de manejar texto, pero vienen con complejidades debido a las reglas de propiedad y préstamo de Rust y la codificación UTF-8. Entender las diferencias entre String y &str, cómo gestionar la propiedad y cómo realizar operaciones comunes te ayudará a trabajar eficazmente con cadenas en Rust.
