Introducción a Rust

Rust es un lenguaje de programación de sistemas que se enfoca en la velocidad, la seguridad de la memoria y el paralelismo. Es sintácticamente similar a C++, pero garantiza la seguridad de la memoria utilizando un verificador de préstamos para validar las referencias.

Conceptos Clave

  • Variables y Mutabilidad
  • Tipos de Datos
  • Funciones
  • Flujo de Control
  • Propiedad y Préstamo

Variables y Mutabilidad

En Rust, las variables son inmutables por defecto. Para hacer una variable mutable, necesitas usar la palabra clave mut.

fn main() {
    let x = 5; // variable inmutable
    println!("El valor de x es: {}", x);

    let mut y = 10; // variable mutable
    println!("El valor de y es: {}", y);
    y = 15;
    println!("El nuevo valor de y es: {}", y);
}

Explicación

  • let x = 5; declara una variable inmutable x con el valor 5.
  • let mut y = 10; declara una variable mutable y con el valor 10.
  • y = 15; cambia el valor de y a 15.

Tipos de Datos

Rust es un lenguaje de tipo estático, lo que significa que debe conocer los tipos de todas las variables en tiempo de compilación. Los principales tipos de datos incluyen:

  • Tipos Escalares: enteros, números de punto flotante, booleanos y caracteres.
  • Tipos Compuestos: tuplas y arreglos.

Ejemplo de Tipos Escalares

fn main() {
    let integer: i32 = 42;
    let float: f64 = 3.14;
    let boolean: bool = true;
    let character: char = 'R';

    println!("Entero: {}", integer);
    println!("Flotante: {}", float);
    println!("Booleano: {}", boolean);
    println!("Carácter: {}", character);
}

Ejemplo de Tipos Compuestos

fn main() {
    let tuple: (i32, f64, u8) = (500, 6.4, 1);
    let array: [i32; 3] = [1, 2, 3];

    println!("Tupla: ({}, {}, {})", tuple.0, tuple.1, tuple.2);
    println!("Arreglo: [{}, {}, {}]", array[0], array[1], array[2]);
}

Funciones

Las funciones en Rust se definen usando la palabra clave fn. Pueden tomar parámetros y devolver valores.

fn main() {
    let result = add(5, 3);
    println!("La suma es: {}", result);
}

fn add(x: i32, y: i32) -> i32 {
    x + y
}

Explicación

  • fn main() es el punto de entrada del programa.
  • fn add(x: i32, y: i32) -> i32 define una función llamada add que toma dos parámetros i32 y devuelve un i32.

Flujo de Control

Rust proporciona constructos comunes de flujo de control como if, else, loop, while y for.

Ejemplo de If-Else

fn main() {
    let number = 7;

    if number < 5 {
        println!("El número es menor que 5");
    } else {
        println!("El número es mayor o igual a 5");
    }
}

Ejemplo de Loop

fn main() {
    let mut count = 0;

    loop {
        count += 1;
        if count == 5 {
            break;
        }
    }

    println!("El bucle se detuvo en el conteo: {}", count);
}

Propiedad y Préstamo

La propiedad es la característica más única de Rust, que permite la seguridad de la memoria sin un recolector de basura. Cada valor en Rust tiene una variable que se llama su propietario. Solo puede haber un propietario a la vez, y cuando el propietario sale del alcance, el valor será eliminado.

Ejemplo de Propiedad

fn main() {
    let s1 = String::from("hola");
    let s2 = s1; // s1 se mueve a s2

    // println!("{}", s1); // Esta línea causaría un error en tiempo de compilación
    println!("{}", s2);
}

Ejemplo de Préstamo

fn main() {
    let s1 = String::from("hola");
    let len = calculate_length(&s1); // s1 es prestado

    println!("La longitud de '{}' es {}.", s1, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

Explicación

  • let s2 = s1; mueve s1 a s2, haciendo que s1 ya no sea válido.
  • &s1 crea una referencia a s1 que se puede usar sin tomar posesión.

Conclusión

Entender la sintaxis y estructura básica de Rust es crucial para escribir código seguro y eficiente. Al dominar variables, tipos de datos, funciones, flujo de control y propiedad, estableces una base sólida para conceptos de programación en Rust más avanzados. Practica estos conceptos básicos a fondo para volverte competente en Rust.
