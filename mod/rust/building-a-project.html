Construyendo un Proyecto
Creando una Construcción de Liberación ►

Construir un proyecto en Rust implica varios pasos, desde configurar tu entorno hasta escribir, probar y desplegar tu código. Este tema te guiará a través del proceso, proporcionando explicaciones claras, ejemplos y ejercicios para ayudarte a entender cada paso.

Configurando Tu Entorno

Antes de comenzar a construir un proyecto, necesitas configurar tu entorno de desarrollo Rust.

  • Instalar Rust: Usa rustup, el instalador de la cadena de herramientas de Rust. 
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Actualizar Rust: Asegúrate de tener la última versión. 
    rustup update
  • Instalar un IDE o Editor de Texto: Las opciones populares incluyen Visual Studio Code con la extensión de Rust, IntelliJ IDEA con el plugin de Rust, o Sublime Text.

Creando un Nuevo Proyecto

Los proyectos en Rust se gestionan usando Cargo, el gestor de paquetes y sistema de construcción de Rust.

  • Crear un nuevo proyecto:

    cargo new my_project
cd my_project

    Este comando crea un nuevo directorio llamado my_project con la siguiente estructura:

    my_project
├── Cargo.toml
└── src
    └── main.rs

  • Cargo.toml: Este archivo contiene metadatos sobre tu proyecto, como su nombre, versión y dependencias.

  • src/main.rs: Este es el archivo fuente principal donde residirá tu código Rust.

Escribiendo Tu Primer Programa

Vamos a escribir un simple programa de "¡Hola, mundo!" para empezar.

  • Editar src/main.rs:

    fn main() {
    println!("¡Hola, mundo!");
}

  • Construir y ejecutar el proyecto:

    cargo run

    Este comando compila tu código y ejecuta el ejecutable resultante.

Añadiendo Dependencias

A medida que tu proyecto crezca, probablemente necesitarás añadir bibliotecas externas (crates).

  • Añadir una dependencia: Abre Cargo.toml y añade la dependencia bajo [dependencies].

    [dependencies]
rand = "0.8"

  • Usar la dependencia en tu código:

    use rand::Rng;

fn main() {
    let mut rng = rand::thread_rng();
    let n: u32 = rng.gen_range(0..10);
    println!("Número aleatorio: {}", n);
}

Estructura del Proyecto

A medida que tu proyecto se vuelva más complejo, necesitarás organizar tu código en módulos y archivos separados.

  • Crear un módulo: Crea un nuevo archivo en el directorio src, por ejemplo, src/lib.rs.

    // src/lib.rs
pub fn greet() {
    println!("¡Hola desde la biblioteca!");
}

  • Usar el módulo en main.rs:

    mod lib;

fn main() {
    lib::greet();
}

Probando Tu Código

Rust tiene soporte integrado para escribir y ejecutar pruebas.

  • Escribir una prueba: Añade una función de prueba en src/lib.rs.

    #[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_greet() {
        greet();
    }
}

  • Ejecutar las pruebas:

    cargo test

Construyendo para la Liberación

Cuando tu proyecto esté listo para el despliegue, querrás construirlo en modo de liberación para un rendimiento optimizado.

  • Construir en modo de liberación: 
    cargo build --release
    El binario optimizado se ubicará en el directorio target/release.

Conclusión

Construir un proyecto en Rust implica configurar tu entorno, crear un nuevo proyecto, escribir y organizar tu código, añadir dependencias, probar y finalmente construir para la liberación. Siguiendo estos pasos, puedes gestionar y desarrollar eficientemente tus proyectos en Rust de principio a fin. Practica estos pasos con pequeños proyectos para familiarizarte más con el ecosistema de Rust y sus herramientas.
