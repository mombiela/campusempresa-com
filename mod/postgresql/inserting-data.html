Insertar Datos
Consulta de Datos ►

En esta sección, cubriremos cómo insertar datos en tablas de PostgreSQL. Esta es una habilidad fundamental para cualquier usuario de bases de datos, ya que te permite poblar tus tablas con los datos necesarios para tus aplicaciones. Comenzaremos con inserciones básicas y avanzaremos hacia técnicas más avanzadas.

Inserciones Básicas

La forma más sencilla de insertar datos en una tabla de PostgreSQL es usando la declaración INSERT INTO.

Sintaxis

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Ejemplo

Supongamos que tenemos una tabla llamada students con la siguiente estructura:

CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    age INT,
    grade VARCHAR(10)
);

Para insertar un nuevo estudiante en esta tabla, usarías la siguiente declaración SQL:

INSERT INTO students (name, age, grade)
VALUES ('John Doe', 20, 'A');

Explicación

  • INSERT INTO students: Especifica la tabla donde se insertarán los datos.
  • (name, age, grade): Enumera las columnas que recibirán los datos.
  • VALUES ('John Doe', 20, 'A'): Proporciona los datos reales que se insertarán en las columnas especificadas.

Insertar Múltiples Filas

Puedes insertar múltiples filas en una sola declaración INSERT INTO separando los valores de cada fila con una coma.

Ejemplo

INSERT INTO students (name, age, grade)
VALUES 
    ('Jane Smith', 22, 'B'),
    ('Emily Johnson', 21, 'A'),
    ('Michael Brown', 23, 'C');

Explicación

  • Cada conjunto de valores dentro de la cláusula VALUES corresponde a una nueva fila en la tabla students.

Insertar Datos desde Otra Tabla

También puedes insertar datos en una tabla seleccionando datos de otra tabla. Esto es útil para copiar datos o transformar datos de una tabla a otra.

Sintaxis

INSERT INTO target_table (column1, column2, ...)
SELECT column1, column2, ...
FROM source_table
WHERE condition;

Ejemplo

Supongamos que tenemos otra tabla new_students con la misma estructura que students. Para copiar todos los datos de new_students a students, usarías:

INSERT INTO students (name, age, grade)
SELECT name, age, grade
FROM new_students;

Explicación

  • SELECT name, age, grade FROM new_students: Selecciona los datos de la tabla new_students.
  • INSERT INTO students (name, age, grade): Inserta los datos seleccionados en la tabla students.

Usar Valores Predeterminados

Si una columna tiene un valor predeterminado, puedes omitirla en la declaración INSERT INTO. PostgreSQL usará automáticamente el valor predeterminado para esa columna.

Ejemplo

Supongamos que la columna grade en la tabla students tiene un valor predeterminado de 'B'. Puedes insertar un nuevo estudiante sin especificar la calificación:

INSERT INTO students (name, age)
VALUES ('Alice Green', 19);

Explicación

  • La columna grade se establecerá automáticamente en 'B' para la nueva fila.

Insertar Datos JSON

PostgreSQL admite tipos de datos JSON, lo que te permite insertar datos JSON directamente en una tabla.

Ejemplo

Supongamos que tenemos una tabla student_info con una columna JSON:

CREATE TABLE student_info (
    id SERIAL PRIMARY KEY,
    info JSON
);

Para insertar datos JSON en esta tabla:

INSERT INTO student_info (info)
VALUES ('{"name": "Tom White", "age": 24, "grade": "A"}');

Explicación

  • La columna info almacena el objeto JSON como un valor único.

Conclusión

Insertar datos en tablas de PostgreSQL es una operación fundamental que se puede realizar de varias maneras dependiendo de tus necesidades. Ya sea que estés insertando filas individuales, múltiples filas, copiando datos de otra tabla o trabajando con datos JSON, PostgreSQL proporciona herramientas flexibles y poderosas para gestionar tus datos de manera eficiente. Entender estas técnicas te ayudará a poblar y gestionar tus bases de datos de manera efectiva.
