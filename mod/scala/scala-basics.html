Introducción

Scala es un poderoso lenguaje de programación que combina paradigmas de programación orientada a objetos y funcional. Está diseñado para ser conciso, elegante y seguro en cuanto a tipos. Esta sección te introducirá a la sintaxis y estructura básica de Scala, proporcionando una base para temas más avanzados.

Conceptos Clave

  • Variables Inmutables: Variables que no pueden ser cambiadas una vez asignadas.
  • Variables Mutables: Variables que pueden ser cambiadas después de la asignación.
  • Expresiones y Declaraciones: Bloques básicos de código en Scala.
  • Funciones: Bloques reutilizables de código que realizan tareas específicas.
  • Clases y Objetos: Componentes centrales de la programación orientada a objetos en Scala.

Variables Inmutables y Mutables

En Scala, las variables pueden ser definidas usando val y var.

  • val: Define una variable inmutable.
  • var: Define una variable mutable.

Ejemplo

val immutableVariable = 10  // Este valor no puede ser cambiado
var mutableVariable = 20    // Este valor puede ser cambiado

mutableVariable = 30        // Operación válida
// immutableVariable = 15   // Esto causará un error de compilación

Expresiones y Declaraciones

Scala es un lenguaje orientado a expresiones, lo que significa que la mayoría de los constructos son expresiones que devuelven un valor.

Ejemplo

val result = if (5 > 3) "Greater" else "Lesser"  // "Greater" es asignado a result

Funciones

Las funciones en Scala son ciudadanos de primera clase, lo que significa que pueden ser asignadas a variables, pasadas como argumentos y devueltas desde otras funciones.

Definiendo Funciones

Las funciones pueden ser definidas usando la palabra clave def.

Ejemplo

def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)  // sum es 8

Clases y Objetos

Scala soporta programación orientada a objetos con clases y objetos.

Definiendo una Clase

Una clase se define usando la palabra clave class.

Ejemplo

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hello, my name is $name and I am $age years old."
}

val person = new Person("Alice", 30)
println(person.greet())  // Salida: Hello, my name is Alice and I am 30 years old.

Objetos Singleton

Scala tiene un concepto de objetos singleton, que son definidos usando la palabra clave object.

Ejemplo

object Singleton {
  def greet(): String = "Hello from the singleton object!"
}

println(Singleton.greet())  // Salida: Hello from the singleton object!

Conclusión

En esta sección, cubrimos los conceptos básicos de la sintaxis y estructura de Scala, incluyendo variables inmutables y mutables, expresiones y declaraciones, funciones, y clases y objetos. Entender estos fundamentos es crucial para progresar a temas más avanzados de Scala. Practica escribiendo programas simples en Scala para reforzar estos conceptos y construir una base sólida.
