Introducción

En esta sección, cubriremos las operaciones y expresiones fundamentales en Scala. Entender estos conceptos básicos es crucial para escribir código Scala efectivo. Exploraremos operaciones aritméticas, operaciones relacionales y lógicas, y cómo usar expresiones para crear lógica más compleja.

Operaciones Aritméticas

Las operaciones aritméticas son los bloques de construcción básicos de cualquier lenguaje de programación. En Scala, puedes realizar operaciones aritméticas estándar como suma, resta, multiplicación, división y módulo.

Conceptos Clave

  • Suma (+): Suma dos números.
  • Resta (-): Resta el segundo número del primero.
  • Multiplicación (*): Multiplica dos números.
  • División (/): Divide el primer número por el segundo.
  • Módulo (%): Devuelve el resto de la división del primer número por el segundo.

Ejemplos de Código

val a = 10
val b = 3

val sum = a + b        // 13
val difference = a - b // 7
val product = a * b    // 30
val quotient = a / b   // 3
val remainder = a % b  // 1

Operaciones Relacionales

Las operaciones relacionales se utilizan para comparar dos valores. El resultado de una operación relacional es un valor Booleano (true o false).

Conceptos Clave

  • Igual a (==): Verifica si dos valores son iguales.
  • No igual a (!=): Verifica si dos valores no son iguales.
  • Mayor que (>): Verifica si el primer valor es mayor que el segundo.
  • Menor que (<): Verifica si el primer valor es menor que el segundo.
  • Mayor o igual a (>=): Verifica si el primer valor es mayor o igual al segundo.
  • Menor o igual a (<=): Verifica si el primer valor es menor o igual al segundo.

Ejemplos de Código

val x = 5
val y = 10

val isEqual = x == y       // false
val isNotEqual = x != y    // true
val isGreater = x > y      // false
val isLess = x < y         // true
val isGreaterOrEqual = x >= y // false
val isLessOrEqual = x <= y // true

Operaciones Lógicas

Las operaciones lógicas se utilizan para combinar múltiples expresiones Booleanas. El resultado de una operación lógica también es un valor Booleano.

Conceptos Clave

  • AND Lógico (&&): Devuelve true si ambas expresiones son true.
  • OR Lógico (||): Devuelve true si al menos una de las expresiones es true.
  • NOT Lógico (!): Invierte el valor Booleano.

Ejemplos de Código

val a = true
val b = false

val andResult = a && b // false
val orResult = a || b  // true
val notResult = !a     // false

Expresiones

Las expresiones son combinaciones de variables, operadores y valores que producen un resultado. En Scala, casi todo es una expresión, incluyendo las sentencias if y los bloques de código.

Conceptos Clave

  • Expresiones Simples: Cálculos o evaluaciones directas.
  • Expresiones Complejas: Combinaciones de múltiples operaciones y funciones.

Ejemplos de Código

// Expresión simple
val simpleExpression = 5 * 2 + 3 // 13

// Expresión compleja
val complexExpression = {
  val x = 10
  val y = 20
  x + y * 2 // 50
}

// Usando if como una expresión
val condition = true
val result = if (condition) "Sí" else "No" // "Sí"

Resumen

En esta sección, cubrimos las operaciones y expresiones básicas en Scala. Exploramos operaciones aritméticas, relacionales y lógicas, y cómo usar expresiones para crear lógica más compleja. Entender estos fundamentos es esencial para progresar a temas más avanzados en la programación con Scala. Practica estos conceptos con varios ejemplos para solidificar tu comprensión.
