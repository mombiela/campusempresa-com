Introducción

Las funciones son bloques de construcción fundamentales en Scala, permitiéndote encapsular lógica y reutilizar código de manera eficiente. Esta sección te guiará a través de los conceptos básicos de la definición e invocación de funciones, progresando de conceptos simples a más avanzados.

Definiendo Funciones

Definición Básica de Función

En Scala, una función se define usando la palabra clave def seguida del nombre de la función, la lista de parámetros y el tipo de retorno.

def add(a: Int, b: Int): Int = {
  a + b
}
  • def: Palabra clave para definir una función.
  • add: Nombre de la función.
  • (a: Int, b: Int): Lista de parámetros con tipos.
  • : Int: Tipo de retorno.
  • { a + b }: Cuerpo de la función.

Funciones de Expresión Única

Para funciones con una sola expresión, puedes omitir las llaves.

def subtract(a: Int, b: Int): Int = a - b

Funciones Sin Parámetros

Las funciones también pueden definirse sin parámetros.

def greet(): String = "¡Hola, Mundo!"

Funciones Sin Tipo de Retorno

Si una función no retorna un valor, su tipo de retorno es Unit.

def printMessage(message: String): Unit = {
  println(message)
}

Invocando Funciones

Invocación Básica

Para invocar una función, usa el nombre de la función seguido de argumentos entre paréntesis.

val sum = add(5, 3) // sum será 8

Omitiendo Paréntesis

Para funciones sin parámetros, puedes omitir los paréntesis al invocarlas.

val greeting = greet // greeting será "¡Hola, Mundo!"

Argumentos Nombrados

Puedes usar argumentos nombrados para hacer la invocación más clara.

val result = add(a = 10, b = 5) // result será 15

Parámetros por Defecto

Las funciones pueden tener valores por defecto para los parámetros.

def multiply(a: Int, b: Int = 2): Int = a * b

val product1 = multiply(4) // product1 será 8
val product2 = multiply(4, 3) // product2 será 12

Funciones de Orden Superior

Funciones como Parámetros

Scala soporta funciones de orden superior, que pueden tomar otras funciones como parámetros.

def applyOperation(a: Int, b: Int, operation: (Int, Int) => Int): Int = {
  operation(a, b)
}

val sum = applyOperation(5, 3, add) // sum será 8

Funciones Anónimas

Puedes pasar funciones anónimas (lambdas) como argumentos.

val result = applyOperation(5, 3, (x, y) => x * y) // result será 15

Recursión

Funciones Recursivas

Las funciones pueden llamarse a sí mismas, lo cual se conoce como recursión.

def factorial(n: Int): Int = {
  if (n == 0) 1
  else n * factorial(n - 1)
}

val fact5 = factorial(5) // fact5 será 120

Recursión de Cola

Las funciones recursivas de cola son optimizadas por el compilador para prevenir desbordamientos de pila.

def factorialTailRec(n: Int, acc: Int = 1): Int = {
  if (n == 0) acc
  else factorialTailRec(n - 1, n * acc)
}

val fact5TailRec = factorialTailRec(5) // fact5TailRec será 120

Conclusión

En esta sección, aprendiste cómo definir e invocar funciones en Scala, incluyendo definiciones básicas, funciones de orden superior y recursión. Entender estos conceptos es crucial para escribir código eficiente y reutilizable. Practica definiendo tus propias funciones y experimentando con diferentes tipos de invocaciones de funciones para solidificar tu comprensión.
