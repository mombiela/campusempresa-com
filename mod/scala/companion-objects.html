Introducción

En Scala, los objetos compañeros son una característica poderosa que te permite definir tanto miembros estáticos como a nivel de instancia para una clase. Este tema te guiará a través del concepto de objetos compañeros, su uso y mejores prácticas, desde un nivel principiante hasta uno avanzado.

¿Qué es un Objeto Compañero?

Un objeto compañero en Scala es un objeto que:

  • Comparte el mismo nombre que una clase.
  • Se define en el mismo archivo que la clase.
  • Puede acceder a los miembros privados de la clase y viceversa.

Conceptos Clave

  • Miembros Estáticos: En Scala, no hay miembros estáticos como en Java. En su lugar, se utilizan objetos compañeros para definir miembros que se comparten entre todas las instancias de una clase.
  • Miembros de Instancia: Estos son miembros que pertenecen a una instancia específica de una clase.

Definiendo un Objeto Compañero

Un objeto compañero se define usando la palabra clave object, y debe estar en el mismo archivo que la clase con la que está asociado.

Ejemplo

class MyClass(val name: String) {
  def greet(): Unit = {
    println(s"¡Hola, $name!")
  }
}

object MyClass {
  def apply(name: String): MyClass = new MyClass(name)
}

En este ejemplo:

  • MyClass es una clase con un miembro de instancia name y un método greet.
  • El objeto compañero MyClass contiene un método de fábrica apply que crea una instancia de MyClass.

Usando Objetos Compañeros

Los objetos compañeros se utilizan a menudo para:

  • Métodos de fábrica.
  • Funciones utilitarias.
  • Constantes.

Ejemplo de Método de Fábrica

class Circle(val radius: Double) {
  def area: Double = Math.PI * radius * radius
}

object Circle {
  def apply(radius: Double): Circle = new Circle(radius)
}

Uso:

val circle = Circle(5.0)  // Llama al método apply en el objeto compañero
println(circle.area)      // Imprime el área del círculo

Ejemplo de Funciones Utilitarias

class MathUtils {
  def square(x: Int): Int = x * x
}

object MathUtils {
  def cube(x: Int): Int = x * x * x
}

Uso:

val utils = new MathUtils()
println(utils.square(4))  // Imprime 16

println(MathUtils.cube(3))  // Imprime 27

Uso Avanzado

Patrón Singleton

Los objetos compañeros se pueden usar para implementar el patrón Singleton, asegurando que solo exista una instancia de una clase.

class Singleton private() {
  def show(): Unit = println("Instancia Singleton")
}

object Singleton {
  private val instance = new Singleton()
  
  def getInstance: Singleton = instance
}

Uso:

val singleton = Singleton.getInstance
singleton.show()  // Imprime "Instancia Singleton"

Accediendo a Miembros Privados

Los objetos compañeros pueden acceder a los miembros privados de su clase compañera y viceversa.

class Secret(val secretCode: String) {
  private def reveal(): String = secretCode
}

object Secret {
  def revealSecret(secret: Secret): String = secret.reveal()
}

Uso:

val secret = new Secret("1234")
println(Secret.revealSecret(secret))  // Imprime "1234"

Conclusión

Los objetos compañeros en Scala proporcionan una forma flexible de definir miembros similares a los estáticos y funciones utilitarias. Mejoran las capacidades orientadas a objetos del lenguaje al permitir una interacción cercana entre las clases y sus objetos compañeros. Comprender y utilizar eficazmente los objetos compañeros puede llevar a un código más limpio y mantenible.
