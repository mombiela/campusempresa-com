Introducción

En Scala, las clases y los objetos son bloques de construcción fundamentales para crear código estructurado y reutilizable. Esta sección te guiará a través de los conceptos básicos de la definición de clases y objetos, y cómo usarlos efectivamente en tus programas Scala.

Clases en Scala

Una clase en Scala es un plano para crear objetos. Puede contener campos (variables) y métodos (funciones) para definir el comportamiento y el estado de los objetos.

Definiendo una Clase

Para definir una clase en Scala, se usa la palabra clave class seguida del nombre de la clase y una lista de parámetros opcional.

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hola, mi nombre es $name y tengo $age años."
}
  • class Person(val name: String, val age: Int): Esto define una clase llamada Person con dos parámetros, name y age.
  • def greet(): String = ...: Esto define un método llamado greet que devuelve una cadena de saludo.

Creando Objetos a partir de una Clase

Para crear un objeto a partir de una clase, se usa la palabra clave new.

val person = new Person("Alice", 30)
println(person.greet()) // Salida: Hola, mi nombre es Alice y tengo 30 años.

Objetos en Scala

En Scala, un objeto es una instancia singleton de una clase. Los objetos se definen usando la palabra clave object y pueden contener métodos y valores.

Definiendo un Objeto

Un objeto es similar a una clase pero no puede tomar parámetros.

object MathUtils {
  def add(a: Int, b: Int): Int = a + b
  def subtract(a: Int, b: Int): Int = a - b
}
  • object MathUtils: Esto define un objeto singleton llamado MathUtils.
  • def add(a: Int, b: Int): Int = ...: Esto define un método llamado add que devuelve la suma de dos enteros.

Usando un Objeto

Puedes llamar a los métodos de un objeto directamente sin crear una instancia.

val sum = MathUtils.add(5, 3)
println(sum) // Salida: 8

Objetos Compañeros

En Scala, un objeto compañero es un objeto que comparte el mismo nombre que una clase y se define en el mismo archivo. Los objetos compañeros se usan para definir métodos y valores que están relacionados con la clase pero no están vinculados a una instancia específica.

Ejemplo de Objeto Compañero

class Circle(val radius: Double) {
  def area: Double = Circle.pi * radius * radius
}

object Circle {
  val pi: Double = 3.14159
}
  • class Circle(val radius: Double): Esto define una clase llamada Circle.
  • object Circle: Esto define un objeto compañero para la clase Circle.
  • val pi: Double = 3.14159: Esto define un valor constante pi en el objeto compañero.

Usando Objetos Compañeros

Puedes acceder a los miembros de un objeto compañero usando el nombre de la clase.

val circle = new Circle(5.0)
println(circle.area) // Salida: 78.53975

Clases Case

Las clases case son un tipo especial de clase en Scala que son inmutables y vienen con métodos incorporados para comparación y coincidencia de patrones.

Definiendo una Clase Case

case class Point(x: Int, y: Int)
  • case class Point(x: Int, y: Int): Esto define una clase case llamada Point con dos parámetros, x y y.

Creando y Usando Clases Case

Puedes crear instancias de clases case sin usar la palabra clave new.

val point1 = Point(1, 2)
val point2 = Point(1, 2)
println(point1 == point2) // Salida: true

Resumen

En esta sección, cubrimos los conceptos básicos de clases y objetos en Scala, incluyendo:

  • Cómo definir e instanciar clases.
  • Cómo definir y usar objetos singleton.
  • El concepto de objetos compañeros y su uso.
  • Las características especiales de las clases case.

Entender estos conceptos es crucial para escribir código organizado y reutilizable en Scala. A medida que avances, verás cómo estos bloques de construcción encajan en aplicaciones Scala más complejas.
