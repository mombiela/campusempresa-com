Introducción

Scala proporciona un conjunto rico de colecciones y operaciones funcionales que permiten a los desarrolladores escribir código conciso y expresivo. Esta sección cubrirá los conceptos básicos de las colecciones de Scala, incluyendo listas, conjuntos, mapas y secuencias, así como operaciones funcionales como map, filter y reduce.

Colecciones en Scala

Las colecciones de Scala se dividen en tres categorías principales:

  • Secuencias: Colecciones ordenadas de elementos.
  • Conjuntos: Colecciones desordenadas de elementos únicos.
  • Mapas: Colecciones de pares clave-valor.

Secuencias

Las secuencias son colecciones ordenadas de elementos. Los tipos de secuencia más comunes en Scala son List, Array y Vector.

List

Una List es una secuencia inmutable. Así es como se crea y se usa una lista:

val fruits = List("apple", "banana", "cherry")
println(fruits.head) // Output: apple
println(fruits.tail) // Output: List(banana, cherry)

Array

Un Array es una secuencia mutable. Así es como se crea y se usa un array:

val numbers = Array(1, 2, 3, 4, 5)
numbers(0) = 10
println(numbers.mkString(", ")) // Output: 10, 2, 3, 4, 5

Vector

Un Vector es una secuencia inmutable que proporciona acceso aleatorio rápido. Así es como se crea y se usa un vector:

val vector = Vector(1, 2, 3, 4, 5)
println(vector(2)) // Output: 3

Conjuntos

Los conjuntos son colecciones de elementos únicos. Scala proporciona tanto conjuntos mutables como inmutables.

Conjunto Inmutable

Así es como se crea y se usa un conjunto inmutable:

val set = Set(1, 2, 3, 4, 5)
println(set.contains(3)) // Output: true
println(set.contains(6)) // Output: false

Conjunto Mutable

Así es como se crea y se usa un conjunto mutable:

import scala.collection.mutable

val mutableSet = mutable.Set(1, 2, 3)
mutableSet += 4
println(mutableSet) // Output: Set(1, 2, 3, 4)

Mapas

Los mapas son colecciones de pares clave-valor. Scala proporciona tanto mapas mutables como inmutables.

Mapa Inmutable

Así es como se crea y se usa un mapa inmutable:

val map = Map("a" -> 1, "b" -> 2, "c" -> 3)
println(map("a")) // Output: 1

Mapa Mutable

Así es como se crea y se usa un mapa mutable:

import scala.collection.mutable

val mutableMap = mutable.Map("a" -> 1, "b" -> 2)
mutableMap("c") = 3
println(mutableMap) // Output: Map(a -> 1, b -> 2, c -> 3)

Operaciones Funcionales

Las operaciones funcionales te permiten manipular colecciones en un estilo de programación funcional. Las operaciones funcionales más comunes son map, filter y reduce.

Map

La operación map aplica una función a cada elemento de una colección y devuelve una nueva colección con los resultados.

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)
println(doubled) // Output: List(2, 4, 6, 8, 10)

Filter

La operación filter selecciona elementos de una colección que satisfacen un predicado.

val numbers = List(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter(_ % 2 == 0)
println(evenNumbers) // Output: List(2, 4)

Reduce

La operación reduce combina los elementos de una colección usando una función binaria.

val numbers = List(1, 2, 3, 4, 5)
val sum = numbers.reduce(_ + _)
println(sum) // Output: 15

Conclusión

En esta sección, cubrimos los conceptos básicos de las colecciones y operaciones funcionales de Scala. Exploramos diferentes tipos de colecciones como listas, arrays, vectores, conjuntos y mapas, y aprendimos cómo realizar operaciones funcionales como map, filter y reduce. Entender estos conceptos es crucial para escribir código eficiente y expresivo en Scala. Practica usando estas colecciones y operaciones para volverte más competente en la programación en Scala.
