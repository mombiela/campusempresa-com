Agregaciones en Elasticsearch
Scripting ►

Las agregaciones en Elasticsearch te permiten realizar operaciones complejas de análisis y resumen de datos en tus datos indexados. Este tema te guiará desde los conceptos básicos de las agregaciones hasta conceptos más avanzados, proporcionando explicaciones claras, ejemplos y ejercicios.

Introducción a las Agregaciones

Las agregaciones son una característica poderosa en Elasticsearch que te permiten extraer información de tus datos. Te permiten agrupar, filtrar y realizar cálculos en tus datos.

Conceptos Clave

  • Buckets: Contenedores que contienen documentos que coinciden con un criterio específico.
  • Métricas: Cálculos realizados en los documentos de un bucket.
  • Agregaciones de Pipeline: Agregaciones que toman la salida de otras agregaciones como su entrada.

Tipos de Agregaciones

Elasticsearch ofrece varios tipos de agregaciones, cada uno con diferentes propósitos. Cubriremos los más comúnmente utilizados.

Agregaciones de Bucket

Las agregaciones de bucket crean buckets de documentos basados en ciertos criterios.

Agregación de Términos

Agrupa documentos por un campo específico.

{
  "aggs": {
    "genres": {
      "terms": {
        "field": "genre.keyword"
      }
    }
  }
}

Agregación de Rango

Agrupa documentos en rangos.

{
  "aggs": {
    "price_ranges": {
      "range": {
        "field": "price",
        "ranges": [
          { "to": 50 },
          { "from": 50, "to": 100 },
          { "from": 100 }
        ]
      }
    }
  }
}

Agregaciones de Métricas

Las agregaciones de métricas calculan métricas, como promedios y sumas, de los documentos en un bucket.

Agregación de Promedio

Calcula el valor promedio de un campo numérico.

{
  "aggs": {
    "average_price": {
      "avg": {
        "field": "price"
      }
    }
  }
}

Agregación de Suma

Calcula la suma de un campo numérico.

{
  "aggs": {
    "total_sales": {
      "sum": {
        "field": "sales"
      }
    }
  }
}

Agregaciones de Pipeline

Las agregaciones de pipeline toman la salida de otras agregaciones como su entrada.

Agregación de Derivada

Calcula la derivada de una métrica especificada.

{
  "aggs": {
    "sales_over_time": {
      "date_histogram": {
        "field": "date",
        "interval": "month"
      },
      "aggs": {
        "sales": {
          "sum": {
            "field": "sales"
          }
        },
        "sales_derivative": {
          "derivative": {
            "buckets_path": "sales"
          }
        }
      }
    }
  }
}

Agregaciones Avanzadas

Agregaciones Anidadas

Las agregaciones anidadas te permiten realizar agregaciones dentro de documentos anidados.

{
  "aggs": {
    "nested_comments": {
      "nested": {
        "path": "comments"
      },
      "aggs": {
        "avg_rating": {
          "avg": {
            "field": "comments.rating"
          }
        }
      }
    }
  }
}

Combinación de Agregaciones

Combinar múltiples agregaciones puede proporcionar información más profunda.

{
  "aggs": {
    "genres": {
      "terms": {
        "field": "genre.keyword"
      },
      "aggs": {
        "average_price": {
          "avg": {
            "field": "price"
          }
        }
      }
    }
  }
}

Ejercicios

  1. Agregación de Términos Básica: Crea una agregación de términos para agrupar documentos por el campo category.
  2. Agregación de Rango: Crea una agregación de rango para agrupar documentos por rangos de precio: 0-50, 50-100 y 100+.
  3. Combinación de Agregaciones: Combina una agregación de términos en category con una agregación de promedio en price.

Conclusión

Las agregaciones en Elasticsearch son una herramienta versátil para el análisis de datos, permitiéndote agrupar, filtrar y calcular métricas en tus datos. Al entender y utilizar diferentes tipos de agregaciones, desde agregaciones básicas de bucket y métricas hasta agregaciones anidadas y de pipeline más avanzadas, puedes extraer información valiosa de tus datos. Practica con los ejercicios proporcionados para solidificar tu comprensión y volverte competente en el uso de agregaciones en Elasticsearch.
