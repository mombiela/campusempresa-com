Introducción

Elasticsearch es un motor de búsqueda y análisis potente que se utiliza ampliamente para diversas aplicaciones, incluyendo el análisis de datos de registros y eventos, búsqueda de texto completo y más. Esta sección te guiará a través del proceso de instalación de Elasticsearch en diferentes sistemas operativos.

Requisitos previos

Antes de instalar Elasticsearch, asegúrate de tener los siguientes requisitos previos:

  • Java Development Kit (JDK) 8 o superior instalado.
  • Recursos del sistema suficientes (RAM, CPU, espacio en disco).

Instalación de Elasticsearch en diferentes sistemas operativos

Instalación en Windows

  1. Descargar Elasticsearch:

    • Visita la página de descargas de Elasticsearch.
    • Descarga el archivo ZIP para Windows.

  2. Extraer el archivo ZIP:

    • Extrae el archivo ZIP descargado en un directorio de tu elección, por ejemplo, C:\elasticsearch.

  3. Ejecutar Elasticsearch:

    • Abre una ventana de Símbolo del sistema.
    • Navega al directorio bin dentro de la carpeta extraída.
    • Ejecuta el siguiente comando: 
      elasticsearch.bat
    • Elasticsearch se iniciará, y puedes acceder a él navegando a http://localhost:9200 en tu navegador web.

Instalación en macOS

  1. Descargar Elasticsearch:

    • Visita la página de descargas de Elasticsearch.
    • Descarga el archivo TAR para macOS.

  2. Extraer el archivo TAR:

    • Abre una ventana de Terminal.
    • Navega al directorio donde se descargó el archivo TAR.
    • Ejecuta el siguiente comando para extraer el archivo: 
      tar -xzf elasticsearch-<version>.tar.gz

  3. Ejecutar Elasticsearch:

    • Navega al directorio bin dentro de la carpeta extraída.
    • Ejecuta el siguiente comando: 
      ./elasticsearch
    • Elasticsearch se iniciará, y puedes acceder a él navegando a http://localhost:9200 en tu navegador web.

  1. Descargar Elasticsearch:

    • Visita la página de descargas de Elasticsearch.

Instalación mediante gestores de paquetes

Usando Homebrew en macOS

  1. Instalar Homebrew:

    • Si no tienes Homebrew instalado, instálalo ejecutando el siguiente comando en Terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instalar Elasticsearch:

    • Ejecuta el siguiente comando para instalar Elasticsearch: 
      brew tap elastic/tap
brew install elastic/tap/elasticsearch-full

  3. Ejecutar Elasticsearch:

    • Inicia Elasticsearch ejecutando: 
      brew services start elastic/tap/elasticsearch-full

Usando APT en Linux basado en Debian

  1. Instalar APT Transport HTTPS:

    • Ejecuta el siguiente comando: 
      sudo apt-get install apt-transport-https

  2. Agregar la clave GPG de Elasticsearch:

    • Ejecuta el siguiente comando: 
      wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -

  3. Agregar el repositorio de Elasticsearch:

    • Ejecuta el siguiente comando: 
      echo "deb https://artifacts.elastic.co/packages/7.x/apt stable main" | sudo tee -a /etc/apt/sources.list.d/elastic-7.x.list

  4. Instalar Elasticsearch:

    • Actualiza la lista de paquetes e instala Elasticsearch: 
      sudo apt-get update
sudo apt-get install elasticsearch

  5. Ejecutar Elasticsearch:

    • Inicia Elasticsearch ejecutando: 
      sudo systemctl start elasticsearch

Verificación de la instalación

Después de instalar Elasticsearch, verifica que esté funcionando correctamente navegando a http://localhost:9200 en tu navegador web. Deberías ver una respuesta JSON similar a la siguiente:

{
  "name" : "your-node-name",
  "cluster_name" : "elasticsearch",
  "cluster_uuid" : "your-cluster-uuid",
  "version" : {
    "number" : "7.x.x",
    "build_flavor" : "default",
    "build_type" : "tar",
    "build_hash" : "your-build-hash",
    "build_date" : "your-build-date",
    "build_snapshot" : false,
    "lucene_version" : "8.x.x",
    "minimum_wire_compatibility_version" : "6.x.x",
    "minimum_index_compatibility_version" : "6.x.x"
  },
  "tagline" : "You Know, for Search"
}

Conclusión

En esta sección, cubrimos los pasos para instalar Elasticsearch en varios sistemas operativos, incluyendo Windows, macOS y Linux. También discutimos cómo verificar la instalación. Con Elasticsearch en funcionamiento, ahora estás listo para comenzar a explorar sus potentes capacidades de búsqueda y análisis.
