Introducción a Google Cloud Platform (GCP)

Google Cloud Platform (GCP) es un conjunto de servicios de computación en la nube ofrecidos por Google. Este curso te llevará de un nivel principiante a un nivel avanzado, cubriendo todos los aspectos esenciales de GCP.

¿Qué es GCP?

  • Una colección de servicios en la nube que incluyen computación, almacenamiento de datos, análisis de datos y aprendizaje automático.
  • Proporciona infraestructura como servicio (IaaS), plataforma como servicio (PaaS) y entornos de computación sin servidor.

Conceptos Clave

  • Regiones y Zonas: Ubicaciones físicas donde se alojan los recursos.
  • Proyectos: Unidades organizativas para gestionar recursos.
  • Facturación: Entender cómo GCP cobra por los servicios.
  • IAM (Gestión de Identidad y Acceso): Gestión de permisos y acceso.

Servicios Principales de GCP

Compute Engine

  • Máquinas Virtuales (VMs) en GCP.
  • Creación y gestión de VMs.
# Crear una instancia de VM
gcloud compute instances create my-vm --zone=us-central1-a --machine-type=e2-medium

App Engine

  • Plataforma como Servicio (PaaS) para construir aplicaciones web escalables.
  • Desplegar una aplicación web simple.
# Configuración de app.yaml para Python App Engine
runtime: python38

# main.py para una aplicación web simple
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello():
    return '¡Hola, Mundo!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Kubernetes Engine

  • Servicio de Kubernetes gestionado.
  • Desplegar una aplicación en contenedor.
# Crear un clúster de Kubernetes
gcloud container clusters create my-cluster --zone us-central1-a

# Desplegar una aplicación
kubectl create deployment hello-server --image=gcr.io/google-samples/hello-app:1.0

# Exponer la aplicación
kubectl expose deployment hello-server --type=LoadBalancer --port 80 --target-port 8080

Almacenamiento de Datos y Bases de Datos

Cloud Storage

  • Almacenamiento de objetos para datos no estructurados.
  • Creación y gestión de buckets de almacenamiento.
# Crear un bucket de almacenamiento
gsutil mb gs://my-bucket

# Subir un archivo al bucket
gsutil cp my-file.txt gs://my-bucket/

Cloud SQL

  • Servicio de base de datos relacional gestionado.
  • Configuración de una instancia de MySQL.
# Crear una instancia de Cloud SQL
gcloud sql instances create my-instance --database-version=MYSQL_5_7 --tier=db-n1-standard-1 --region=us-central1

# Conectarse a la instancia
gcloud sql connect my-instance --user=root

Big Data y Aprendizaje Automático

BigQuery

  • Almacén de datos totalmente gestionado para análisis de datos a gran escala.
  • Ejecutar consultas SQL en BigQuery.
