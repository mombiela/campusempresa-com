# Ejemplo: Despliegue de una aplicación simple de Python en App Engine

# Archivo de configuración app.yaml
runtime: python39

# main.py - Aplicación simple de Flask
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return '¡Hola, GCP!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Explicación:

  1. app.yaml: Este archivo especifica el entorno de ejecución para la aplicación. En este caso, está configurado para Python 3.9.
  2. main.py: Una aplicación simple de Flask que define una única ruta (/) que devuelve "¡Hola, GCP!" cuando se accede.

Para desplegar esta aplicación:

  1. Asegúrate de tener instalado y autenticado el Google Cloud SDK.
  2. Navega al directorio que contiene app.yaml y main.py.
  3. Ejecuta el comando: 
    gcloud app deploy
  4. Una vez desplegada, puedes acceder a tu aplicación a través de la URL proporcionada por GCP.

Resumen

En este curso, cubrimos lo esencial para convertirte en un GCP Professional Cloud Developer, desde la configuración de tu entorno hasta el despliegue y gestión de aplicaciones. Exploramos servicios principales, soluciones de almacenamiento, redes, seguridad y temas avanzados como machine learning y big data. Siguiendo las mejores prácticas y aprovechando las poderosas herramientas de GCP, puedes construir soluciones en la nube escalables, resilientes y rentables. Continúa explorando la extensa documentación y recursos de GCP para profundizar tus conocimientos y habilidades.
