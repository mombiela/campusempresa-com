Introducción a Stackdriver Logging

Stackdriver Logging, ahora conocido como Google Cloud Logging, es un servicio completamente gestionado que te permite almacenar, buscar, analizar, monitorear y alertar sobre datos y eventos de registros de Google Cloud Platform (GCP) y Amazon Web Services (AWS). Esta sección te introducirá a los conceptos básicos de Stackdriver Logging y su importancia en entornos en la nube.

  • Conceptos Clave:
    • Entradas de Registro: Registros individuales de eventos.
    • Sumideros de Registro: Mecanismos para exportar registros a otros servicios.
    • Métricas Basadas en Registros: Métricas derivadas de datos de registros.
    • Vistas de Registro: Filtros para controlar qué registros son visibles.

Configuración de Stackdriver Logging

Antes de que puedas comenzar a usar Stackdriver Logging, necesitas configurarlo en tu proyecto de GCP.

Habilitar la API de Logging

Para usar Stackdriver Logging, necesitas habilitar la API de Logging en tu proyecto de GCP.

gcloud services enable logging.googleapis.com

Crear una Cuenta de Servicio

Se requiere una cuenta de servicio para interactuar con Stackdriver Logging programáticamente.

gcloud iam service-accounts create logging-sa --display-name "Logging Service Account"

Asignar Roles

Asigna los roles necesarios a la cuenta de servicio para permitirle escribir registros.

gcloud projects add-iam-policy-binding [PROJECT_ID] \
    --member "serviceAccount:logging-sa@[PROJECT_ID].iam.gserviceaccount.com" \
    --role "roles/logging.logWriter"

Escribir Registros

Puedes escribir registros en Stackdriver Logging usando varios métodos, incluyendo la Consola de GCP, la herramienta de línea de comandos gcloud y bibliotecas cliente.

Escribir Registros Usando la Consola de GCP

  1. Navega a la sección de Logging en la Consola de GCP.
  2. Selecciona el tipo de registro y recurso apropiado.
  3. Escribe una entrada de registro y envíala.

Escribir Registros Usando la Herramienta de Línea de Comandos `gcloud`

gcloud logging write my-log "¡Hola, Stackdriver Logging!"

Escribir Registros Usando Bibliotecas Cliente

Aquí tienes un ejemplo de cómo escribir registros usando la biblioteca cliente de Python:

from google.cloud import logging

client = logging.Client()
logger = client.logger("my-log")

logger.log_text("¡Hola, Stackdriver Logging!")

Ver y Analizar Registros

Una vez que los registros están escritos, puedes verlos y analizarlos usando la Consola de GCP o la herramienta de línea de comandos gcloud.

Ver Registros en la Consola de GCP

  1. Navega a la sección de Logging en la Consola de GCP.
  2. Usa el Visor de Registros para filtrar y buscar registros.

Ver Registros Usando la Herramienta de Línea de Comandos `gcloud`

gcloud logging read "logName=projects/[PROJECT_ID]/logs/my-log"

Exportar Registros

Puedes exportar registros a otros servicios de GCP como BigQuery, Cloud Storage o Pub/Sub para un análisis más profundo y almacenamiento a largo plazo.

Crear un Sumidero de Registro

  1. Navega a la sección de Logging en la Consola de GCP.
  2. Ve a la pestaña "Exportaciones" y crea un nuevo sumidero.
  3. Elige el destino (BigQuery, Cloud Storage o Pub/Sub).

Ejemplo: Exportar Registros a BigQuery

gcloud logging sinks create my-sink bigquery.googleapis.com/projects/[PROJECT_ID]/datasets/[DATASET_ID]

Crear Métricas Basadas en Registros

Las métricas basadas en registros te permiten crear métricas personalizadas basadas en datos de registros, que pueden ser usadas para monitoreo y alertas.

Crear una Métrica Basada en Registros

  1. Navega a la sección de Logging en la Consola de GCP.
  2. Ve a la pestaña "Métricas" y crea una nueva métrica.
  3. Define el filtro y el tipo de métrica.

Ejemplo: Crear una Métrica Basada en Registros Usando `gcloud`

gcloud logging metrics create my-metric \
    --description="Una métrica personalizada basada en datos de registros" \
    --log-filter="logName=projects/[PROJECT_ID]/logs/my-log"

Conclusión

Stackdriver Logging es una herramienta poderosa para gestionar y analizar datos de registros en GCP. Al entender cómo configurar, escribir, ver, exportar y crear métricas a partir de registros, puedes monitorear y solucionar problemas de tus aplicaciones e infraestructura de manera efectiva. Ya seas un principiante o un usuario avanzado, dominar Stackdriver Logging mejorará significativamente tu capacidad para gestionar recursos en la nube de manera eficiente.
