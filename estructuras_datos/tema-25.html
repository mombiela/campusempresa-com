Introducción

En esta sección del curso sobre Estructuras de Datos, aprenderemos sobre las operaciones básicas con árboles. Los árboles son estructuras de datos jerárquicas que consisten en nodos conectados por aristas. Cada nodo contiene un valor y puede tener hijos, formando una estructura en forma de árbol.

Conceptos Clave

  • Nodo: Elemento básico de un árbol que contiene un valor o dato.
  • Raíz: El nodo superior de un árbol, que no tiene padres.
  • Hijo: Nodo que desciende de otro nodo.
  • Padre: Nodo que tiene uno o más hijos.
  • Hoja: Nodo que no tiene hijos.
  • Subárbol: Árbol formado por un nodo y todos sus descendientes.
  • Altura: Longitud del camino más largo desde la raíz hasta una hoja.
  • Profundidad: Longitud del camino desde la raíz hasta un nodo específico.

Operaciones Básicas

1. Inserción

La inserción es el proceso de agregar un nuevo nodo al árbol. La ubicación del nuevo nodo depende del tipo de árbol (por ejemplo, árbol binario de búsqueda).

Ejemplo de Código: Inserción en un Árbol Binario de Búsqueda (BST)

class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def insert(root, key):
    if root is None:
        return Node(key)
    else:
        if root.val < key:
            root.right = insert(root.right, key)
        else:
            root.left = insert(root.left, key)
    return root

# Ejemplo de uso
root = Node(50)
root = insert(root, 30)
root = insert(root, 20)
root = insert(root, 40)
root = insert(root, 70)
root = insert(root, 60)
root = insert(root, 80)

2. Búsqueda

La búsqueda es el proceso de encontrar un nodo con un valor específico en el árbol.

Ejemplo de Código: Búsqueda en un Árbol Binario de Búsqueda (BST)

def search(root, key):
    if root is None or root.val == key:
        return root
    if root.val < key:
        return search(root.right, key)
    return search(root.left, key)

# Ejemplo de uso
result = search(root, 40)
if result:
    print("Nodo encontrado:", result.val)
else:
    print("Nodo no encontrado")

3. Eliminación

La eliminación es el proceso de eliminar un nodo del árbol. Este proceso puede ser más complejo, ya que se deben considerar varios casos (nodo sin hijos, con un hijo o con dos hijos).

Ejemplo de Código: Eliminación en un Árbol Binario de Búsqueda (BST)

def minValueNode(node):
    current = node
    while current.left is not None:
        current = current.left
    return current

def deleteNode(root, key):
    if root is None:
        return root
    if key < root.val:
        root.left = deleteNode(root.left, key)
    elif key > root.val:
        root.right = deleteNode(root.right, key)
    else:
        if root.left is None:
            return root.right
        elif root.right is None:
            return root.left
        temp = minValueNode(root.right)
        root.val = temp.val
        root.right = deleteNode(root.right, temp.val)
    return root

# Ejemplo de uso
root = deleteNode(root, 20)

4. Recorridos

Los recorridos son métodos para visitar todos los nodos de un árbol en un orden específico. Los tipos más comunes son:

  • Inorden (In-order): Visita el subárbol izquierdo, el nodo raíz y luego el subárbol derecho.
  • Preorden (Pre-order): Visita el nodo raíz, el subárbol izquierdo y luego el subárbol derecho.
  • Postorden (Post-order): Visita el subárbol izquierdo, el subárbol derecho y luego el nodo raíz.

Ejemplo de Código: Recorridos en un Árbol Binario

def inorder(root):
    if root:
        inorder(root.left)
        print(root.val, end=' ')
        inorder(root.right)

def preorder(root):
    if root:
        print(root.val, end=' ')
        preorder(root.left)
        preorder(root.right)

def postorder(root):
    if root:
        postorder(root.left)
        postorder(root.right)
        print(root.val, end=' ')

# Ejemplo de uso
print("Inorden:")
inorder(root)
print("\nPreorden:")
preorder(root)
print("\nPostorden:")
postorder(root)

Conclusión

En esta sección, hemos cubierto las operaciones básicas con árboles, incluyendo la inserción, búsqueda, eliminación y recorridos. Estas operaciones son fundamentales para trabajar con árboles y son aplicables en diversas estructuras de datos y algoritmos. Asegúrate de practicar estos conceptos y ejemplos de código para consolidar tu comprensión.
