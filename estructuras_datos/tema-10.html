Introducción a las Pilas

Las pilas son una estructura de datos fundamental en la informática, utilizada para almacenar y gestionar datos de manera ordenada. Se caracterizan por seguir el principio LIFO (Last In, First Out), lo que significa que el último elemento en entrar es el primero en salir.

Conceptos Clave

  • LIFO (Last In, First Out): El último elemento que se añade a la pila es el primero en ser removido.
  • Operaciones Básicas:
    • Push: Añadir un elemento a la pila.
    • Pop: Remover el elemento superior de la pila.
    • Peek/Top: Obtener el elemento superior sin removerlo.
    • isEmpty: Verificar si la pila está vacía.
  • Aplicaciones Comunes:
    • Manejo de expresiones aritméticas.
    • Implementación de algoritmos de retroceso (backtracking).
    • Gestión de llamadas a funciones (pila de llamadas).

Implementación de Pilas en Python

A continuación, se presenta una implementación básica de una pila utilizando una lista en Python.

class Pila:
    def __init__(self):
        self.items = []

    def esta_vacia(self):
        return len(self.items) == 0

    def apilar(self, item):
        self.items.append(item)

    def desapilar(self):
        if not self.esta_vacia():
            return self.items.pop()
        else:
            raise IndexError("La pila está vacía")

    def ver_tope(self):
        if not self.esta_vacia():
            return self.items[-1]
        else:
            raise IndexError("La pila está vacía")

Explicación del Código

  • __init__: Inicializa una lista vacía para almacenar los elementos de la pila.
  • esta_vacia: Devuelve True si la pila está vacía, de lo contrario False.
  • apilar: Añade un elemento al final de la lista (parte superior de la pila).
  • desapilar: Remueve y devuelve el último elemento de la lista. Lanza una excepción si la pila está vacía.
  • ver_tope: Devuelve el último elemento de la lista sin removerlo. Lanza una excepción si la pila está vacía.

Ejemplo de Uso

A continuación, se muestra un ejemplo de cómo utilizar la clase Pila.

mi_pila = Pila()
mi_pila.apilar(1)
mi_pila.apilar(2)
mi_pila.apilar(3)

print(mi_pila.ver_tope())  # Salida: 3
print(mi_pila.desapilar())  # Salida: 3
print(mi_pila.ver_tope())  # Salida: 2

Conclusión

Las pilas son una estructura de datos esencial en la programación, especialmente útil en situaciones donde se requiere un acceso LIFO. La implementación en Python es sencilla y puede ser extendida para aplicaciones más complejas. Comprender cómo funcionan las pilas y cómo implementarlas es fundamental para cualquier profesional de TI, ya que son la base de muchos algoritmos y estructuras más avanzadas.
