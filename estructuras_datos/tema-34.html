Introducción

En esta sección del curso sobre Estructuras de Datos, nos enfocaremos en los grafos, una estructura fundamental en la informática y las ciencias de la computación. Los grafos se utilizan para modelar relaciones y conexiones entre diferentes entidades. A través de ejercicios prácticos, aprenderemos a implementar y manipular grafos en diferentes contextos.

Conceptos Clave

  • Grafo: Conjunto de nodos (o vértices) y aristas (o enlaces) que conectan pares de nodos.
  • Tipos de Grafos:
    • Dirigido: Las aristas tienen una dirección.
    • No dirigido: Las aristas no tienen dirección.
    • Ponderado: Las aristas tienen un peso asociado.
    • No ponderado: Las aristas no tienen peso.
  • Representaciones de Grafos:
    • Lista de Adyacencia: Utiliza listas para almacenar los nodos adyacentes.
    • Matriz de Adyacencia: Utiliza una matriz para representar las conexiones entre nodos.

Ejercicio 1: Implementación de un Grafo No Dirigido

Objetivo

Implementar un grafo no dirigido utilizando una lista de adyacencia.

Instrucciones

  1. Crear una clase Grafo.
  2. Implementar métodos para agregar nodos y aristas.
  3. Implementar un método para imprimir el grafo.

Ejemplo de Código

class Grafo:
    def __init__(self):
        self.grafo = {}

    def agregar_nodo(self, nodo):
        if nodo not in self.grafo:
            self.grafo[nodo] = []

    def agregar_arista(self, nodo1, nodo2):
        if nodo1 in self.grafo and nodo2 in self.grafo:
            self.grafo[nodo1].append(nodo2)
            self.grafo[nodo2].append(nodo1)

    def imprimir_grafo(self):
        for nodo in self.grafo:
            print(f"{nodo}: {self.grafo[nodo]}")

# Ejemplo de uso
g = Grafo()
g.agregar_nodo("A")
g.agregar_nodo("B")
g.agregar_nodo("C")
g.agregar_arista("A", "B")
g.agregar_arista("A", "C")
g.imprimir_grafo()

Explicación

  • Clase Grafo: Define la estructura del grafo.
  • Método agregar_nodo: Añade un nodo al grafo.
  • Método agregar_arista: Añade una arista entre dos nodos.
  • Método imprimir_grafo: Imprime la lista de adyacencia del grafo.

Ejercicio 2: Implementación de un Grafo Dirigido Ponderado

Objetivo

Implementar un grafo dirigido y ponderado utilizando una lista de adyacencia.

Instrucciones

  1. Crear una clase GrafoDirigidoPonderado.
  2. Implementar métodos para agregar nodos y aristas con pesos.
  3. Implementar un método para imprimir el grafo.

Ejemplo de Código

class GrafoDirigidoPonderado:
    def __init__(self):
        self.grafo = {}

    def agregar_nodo(self, nodo):
        if nodo not in self.grafo:
            self.grafo[nodo] = []

    def agregar_arista(self, nodo1, nodo2, peso):
        if nodo1 in self.grafo and nodo2 in self.grafo:
            self.grafo[nodo1].append((nodo2, peso))

    def imprimir_grafo(self):
        for nodo in self.grafo:
            print(f"{nodo}: {self.grafo[nodo]}")

# Ejemplo de uso
g = GrafoDirigidoPonderado()
g.agregar_nodo("A")
g.agregar_nodo("B")
g.agregar_nodo("C")
g.agregar_arista("A", "B", 5)
g.agregar_arista("A", "C", 10)
g.imprimir_grafo()

Explicación

  • Clase GrafoDirigidoPonderado: Define la estructura del grafo dirigido y ponderado.
  • Método agregar_nodo: Añade un nodo al grafo.
  • Método agregar_arista: Añade una arista con peso entre dos nodos.
  • Método imprimir_grafo: Imprime la lista de adyacencia del grafo con pesos.

Ejercicio 3: Búsqueda en Grafos

Objetivo

Implementar algoritmos de búsqueda en grafos: BFS (Breadth-First Search) y DFS (Depth-First Search).

Instrucciones

  1. Crear métodos para BFS y DFS en la clase Grafo.
  2. Implementar la lógica de los algoritmos de búsqueda.

Ejemplo de Código

class Grafo:
    def __init__(self):
        self.grafo = {}

    def agregar_nodo(self, nodo):
        if nodo not in self.grafo:
            self.grafo[nodo] = []

    def agregar_arista(self, nodo1, nodo2):
        if nodo1 in self.grafo and nodo2 in self.grafo:
            self.grafo[nodo1].append(nodo2)
            self.grafo[nodo2].append(nodo1)

    def bfs(self, inicio):
        visitados = set()
        cola = [inicio]

        while cola:
            nodo = cola.pop(0)
            if nodo not in visitados:
                print(nodo, end=" ")
                visitados.add(nodo)
                cola.extend(self.grafo[nodo])

    def dfs(self, inicio, visitados=None):
        if visitados is None:
            visitados = set()
        visitados.add(inicio)
        print(inicio, end=" ")

        for vecino in self.grafo[inicio]:
            if vecino not in visitados:
                self.dfs(vecino, visitados)

# Ejemplo de uso
g = Grafo()
g.agregar_nodo("A")
g.agregar_nodo("B")
g.agregar_nodo("C")
g.agregar_nodo("D")
g.agregar_arista("A", "B")
g.agregar_arista("A", "C")
g.agregar_arista("B", "D")
print("BFS:")
g.bfs("A")
print("\nDFS:")
g.dfs("A")

Explicación

  • Método bfs: Implementa la búsqueda en anchura.
  • Método dfs: Implementa la búsqueda en profundidad.

Conclusión

En esta sección, hemos aprendido a implementar y manipular grafos utilizando listas de adyacencia. Hemos cubierto la implementación de grafos no dirigidos, dirigidos y ponderados, así como los algoritmos de búsqueda BFS y DFS. Estos ejercicios prácticos proporcionan una base sólida para trabajar con grafos en aplicaciones del mundo real.
