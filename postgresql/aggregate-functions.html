Funciones Agregadas en PostgreSQL
Funciones de Ventana ►

Las funciones agregadas son esenciales en SQL para realizar cálculos en múltiples filas de una columna de una tabla y devolver un solo valor. PostgreSQL ofrece una variedad de funciones agregadas para ayudarte a resumir y analizar tus datos de manera efectiva.

Funciones Agregadas Clave

COUNT

La función COUNT devuelve el número de filas de entrada que coinciden con una condición específica.

SELECT COUNT(*) FROM employees;

Esta consulta devuelve el número total de filas en la tabla employees.

SUM

La función SUM calcula la suma total de una columna numérica.

SELECT SUM(salary) FROM employees;

Esta consulta devuelve la suma total de todos los salarios en la tabla employees.

AVG

La función AVG calcula el valor promedio de una columna numérica.

SELECT AVG(salary) FROM employees;

Esta consulta devuelve el salario promedio de todos los empleados.

MAX

La función MAX devuelve el valor máximo de una columna.

SELECT MAX(salary) FROM employees;

Esta consulta devuelve el salario más alto en la tabla employees.

MIN

La función MIN devuelve el valor mínimo de una columna.

SELECT MIN(salary) FROM employees;

Esta consulta devuelve el salario más bajo en la tabla employees.

Uso de Funciones Agregadas con GROUP BY

La cláusula GROUP BY se usa con funciones agregadas para agrupar el conjunto de resultados por una o más columnas.

SELECT department, AVG(salary) 
FROM employees 
GROUP BY department;

Esta consulta devuelve el salario promedio para cada departamento.

Cláusula HAVING

La cláusula HAVING se usa para filtrar grupos basados en una condición, similar a la cláusula WHERE pero para grupos.

SELECT department, AVG(salary) 
FROM employees 
GROUP BY department 
HAVING AVG(salary) > 50000;

Esta consulta devuelve los departamentos donde el salario promedio es mayor a 50,000.

Combinando Funciones Agregadas

Puedes usar múltiples funciones agregadas en una sola consulta para obtener un resumen completo.

SELECT department, COUNT(*), AVG(salary), MAX(salary), MIN(salary) 
FROM employees 
GROUP BY department;

Esta consulta devuelve el conteo de empleados, salario promedio, salario máximo y salario mínimo para cada departamento.

Funciones Agregadas con DISTINCT

La palabra clave DISTINCT se puede usar dentro de funciones agregadas para considerar solo valores únicos.

SELECT COUNT(DISTINCT department) 
FROM employees;

Esta consulta devuelve el número de departamentos únicos en la tabla employees.

Funciones de Ventana

Las funciones de ventana te permiten realizar cálculos agregados en un conjunto de filas de la tabla que están de alguna manera relacionadas con la fila actual.

SELECT employee_id, department, salary, 
       AVG(salary) OVER (PARTITION BY department) AS avg_department_salary 
FROM employees;

Esta consulta devuelve el salario de cada empleado junto con el salario promedio de su departamento.

Conclusión

Las funciones agregadas en PostgreSQL son herramientas poderosas para resumir y analizar datos. Al dominar estas funciones, puedes realizar cálculos complejos y obtener valiosos conocimientos de tus conjuntos de datos. Recuerda usar las cláusulas GROUP BY y HAVING para refinar tus consultas y extraer información significativa. Practica usando estas funciones con diferentes conjuntos de datos para volverte competente en el análisis de datos con PostgreSQL.
