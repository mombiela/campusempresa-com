Funciones de Cadena en PostgreSQL
Funciones Definidas por el Usuario ►

Las funciones de cadena son herramientas esenciales en PostgreSQL para manipular y consultar datos de texto. Esta sección cubrirá varias funciones de cadena desde niveles básicos hasta avanzados, proporcionando explicaciones claras, ejemplos y ejercicios.

Funciones Básicas de Cadena

`LENGTH()`

  • Descripción: Devuelve el número de caracteres en una cadena.
  • Sintaxis: LENGTH(string)
SELECT LENGTH('PostgreSQL'); -- Devuelve 10

`UPPER()` y `LOWER()`

  • Descripción: Convierte una cadena a mayúsculas o minúsculas.
  • Sintaxis: UPPER(string), LOWER(string)
SELECT UPPER('postgresql'); -- Devuelve 'POSTGRESQL'
SELECT LOWER('PostgreSQL'); -- Devuelve 'postgresql'

`TRIM()`

  • Descripción: Elimina los espacios en blanco al principio y al final de una cadena.
  • Sintaxis: TRIM(string)
SELECT TRIM('  PostgreSQL  '); -- Devuelve 'PostgreSQL'

`SUBSTRING()`

  • Descripción: Extrae una subcadena de una cadena.
  • Sintaxis: SUBSTRING(string FROM start FOR length)
SELECT SUBSTRING('PostgreSQL' FROM 1 FOR 4); -- Devuelve 'Post'

Funciones Intermedias de Cadena

`POSITION()`

  • Descripción: Encuentra la posición de una subcadena dentro de una cadena.
  • Sintaxis: POSITION(substring IN string)
SELECT POSITION('SQL' IN 'PostgreSQL'); -- Devuelve 8

`CONCAT()`

  • Descripción: Concatenar dos o más cadenas.
  • Sintaxis: CONCAT(string1, string2, ...)
SELECT CONCAT('Post', 'gre', 'SQL'); -- Devuelve 'PostgreSQL'

`REPLACE()`

  • Descripción: Reemplaza todas las ocurrencias de una subcadena dentro de una cadena con otra subcadena.
  • Sintaxis: REPLACE(string, old_substring, new_substring)
SELECT REPLACE('PostgreSQL', 'Post', 'My'); -- Devuelve 'MygreSQL'

`LPAD()` y `RPAD()`

  • Descripción: Rellena una cadena en el lado izquierdo o derecho hasta una longitud especificada.
  • Sintaxis: LPAD(string, length, pad_string), RPAD(string, length, pad_string)
SELECT LPAD('SQL', 5, '0'); -- Devuelve '00SQL'
SELECT RPAD('SQL', 5, '0'); -- Devuelve 'SQL00'

Funciones Avanzadas de Cadena

`REGEXP_MATCHES()`

  • Descripción: Devuelve todas las coincidencias de un patrón de expresión regular en una cadena.
  • Sintaxis: REGEXP_MATCHES(string, pattern)
SELECT REGEXP_MATCHES('abc123xyz', '[0-9]+'); -- Devuelve {123}

`SPLIT_PART()`

  • Descripción: Divide una cadena en un delimitador especificado y devuelve la enésima subcadena.
  • Sintaxis: SPLIT_PART(string, delimiter, field)
SELECT SPLIT_PART('PostgreSQL es increíble', ' ', 2); -- Devuelve 'es'

`TO_CHAR()`

  • Descripción: Convierte varios tipos de datos a una cadena formateada según un formato especificado.
  • Sintaxis: TO_CHAR(value, format)
SELECT TO_CHAR(12345, '99999'); -- Devuelve '12345'
SELECT TO_CHAR(current_timestamp, 'YYYY-MM-DD'); -- Devuelve la fecha actual en formato 'YYYY-MM-DD'

Ejercicios

  1. Ejercicio Básico: Escribe una consulta para convertir la cadena 'hello world' a mayúsculas.

    SELECT UPPER('hello world');

  2. Ejercicio Intermedio: Escribe una consulta para reemplazar 'SQL' con 'Database' en la cadena 'PostgreSQL'.

    SELECT REPLACE('PostgreSQL', 'SQL', 'Database');

  3. Ejercicio Avanzado: Escribe una consulta para extraer el año de la fecha actual usando TO_CHAR().

    SELECT TO_CHAR(current_date, 'YYYY');

Conclusión

Las funciones de cadena en PostgreSQL son herramientas poderosas para la manipulación y consulta de texto. Desde funciones básicas como LENGTH() y UPPER() hasta funciones avanzadas como REGEXP_MATCHES() y TO_CHAR(), dominar estas funciones mejorará tu capacidad para manejar y analizar datos de texto de manera efectiva. Practica los ejercicios proporcionados para solidificar tu comprensión y aplicar estas funciones en escenarios del mundo real.
