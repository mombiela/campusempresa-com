Listas en Prolog
Recursión ►

Las listas son una estructura de datos fundamental en Prolog, utilizada para almacenar secuencias de elementos. Entender cómo trabajar con listas es crucial para una programación efectiva en Prolog.

Conceptos Clave

  • Estructura de la Lista: Las listas en Prolog están encerradas en corchetes [] y pueden contener cualquier número de elementos, incluidas otras listas.
  • Cabeza y Cola: El primer elemento de una lista se llama cabeza, y el resto de la lista se llama cola.
  • Lista Vacía: La lista vacía se representa por [].
  • Operaciones con Listas: Las operaciones comunes incluyen verificar la pertenencia, concatenación y cálculo de longitud.

Estructura de la Lista

Una lista en Prolog se escribe como [Elemento1, Elemento2, ..., ElementoN]. Por ejemplo:

[1, 2, 3, 4, 5]

Cabeza y Cola

La cabeza de una lista es su primer elemento, y la cola es la lista de elementos restantes. Por ejemplo, en la lista [1, 2, 3, 4, 5], la cabeza es 1 y la cola es [2, 3, 4, 5].

Puedes descomponer una lista en su cabeza y cola usando el operador |:

[Head | Tail] = [1, 2, 3, 4, 5].
% Head = 1
% Tail = [2, 3, 4, 5]

Operaciones Comunes con Listas

Verificar Pertenencia

Para verificar si un elemento es miembro de una lista, puedes usar el predicado incorporado member/2:

member(X, [1, 2, 3, 4, 5]).
% X = 1;
% X = 2;
% X = 3;
% X = 4;
% X = 5;
% false.

Concatenación

Para concatenar dos listas, puedes usar el predicado incorporado append/3:

append([1, 2, 3], [4, 5], Result).
% Result = [1, 2, 3, 4, 5].

Cálculo de Longitud

Para calcular la longitud de una lista, usa el predicado incorporado length/2:

length([1, 2, 3, 4, 5], Length).
% Length = 5.

Ejemplos y Ejercicios

Ejemplo 1: Descomponiendo una Lista

% Descomponer la lista [a, b, c, d] en cabeza y cola
[a, b, c, d] = [Head | Tail].
% Head = a
% Tail = [b, c, d]

Ejemplo 2: Verificar Pertenencia

% Verificar si 'c' es miembro de la lista [a, b, c, d]
member(c, [a, b, c, d]).
% true.

Ejercicio 1: Concatenar Listas

Escribe una consulta para concatenar las listas [1, 2] y [3, 4, 5].

% Tu consulta aquí
append([1, 2], [3, 4, 5], Result).
% Result = [1, 2, 3, 4, 5].

Ejercicio 2: Calcular Longitud

Escribe una consulta para encontrar la longitud de la lista [apple, banana, cherry].

% Tu consulta aquí
length([apple, banana, cherry], Length).
% Length = 3.

Conclusión

Las listas son una estructura de datos versátil y esencial en Prolog. Al entender cómo crear, descomponer y manipular listas, puedes realizar una amplia gama de operaciones de manera eficiente. Practica los ejemplos y ejercicios proporcionados para solidificar tu comprensión de las listas en Prolog.
