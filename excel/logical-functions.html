Introducción

Las funciones lógicas en Excel son herramientas esenciales para realizar operaciones condicionales. Te permiten crear fórmulas que pueden tomar decisiones basadas en criterios específicos. Las funciones lógicas más comúnmente usadas son SI, Y, y O. Esta sección te guiará a través de estas funciones desde un nivel principiante hasta un nivel avanzado.

Función SI

La función SI se usa para realizar una prueba lógica y devolver un valor si la prueba es verdadera y otro valor si la prueba es falsa.

Sintaxis

SI(prueba_lógica, valor_si_verdadero, valor_si_falso)

Ejemplo

Supongamos que tienes una lista de calificaciones de estudiantes y quieres determinar si cada estudiante ha aprobado o reprobado. La calificación mínima para aprobar es 50.

| Estudiante | Calificación | Resultado | |------------|--------------|-----------| | John | 45 | | | Alice | 78 | | | Bob | 52 | |

Puedes usar la siguiente fórmula en la columna "Resultado":

=SI(B2 >= 50, "Aprobado", "Reprobado")

Explicación

  • prueba_lógica: B2 >= 50 (verifica si la calificación es mayor o igual a 50)
  • valor_si_verdadero: "Aprobado" (se devuelve si la prueba es verdadera)
  • valor_si_falso: "Reprobado" (se devuelve si la prueba es falsa)

Función Y

La función Y se usa para probar múltiples condiciones al mismo tiempo. Devuelve VERDADERO si todas las condiciones son verdaderas y FALSO si alguna condición es falsa.

Sintaxis

Y(lógico1, [lógico2], ...)

Ejemplo

Supongamos que quieres determinar si un estudiante ha aprobado tanto los exámenes de Matemáticas como de Inglés. La calificación mínima para ambos sujetos es 50.

| Estudiante | Calificación de Matemáticas | Calificación de Inglés | Resultado | |------------|-----------------------------|------------------------|-----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Puedes usar la siguiente fórmula en la columna "Resultado":

=SI(Y(B2 >= 50, C2 >= 50), "Aprobado", "Reprobado")

Explicación

  • lógico1: B2 >= 50 (verifica si la calificación de Matemáticas es mayor o igual a 50)
  • lógico2: C2 >= 50 (verifica si la calificación de Inglés es mayor o igual a 50)

Función O

La función O se usa para probar múltiples condiciones al mismo tiempo. Devuelve VERDADERO si alguna de las condiciones es verdadera y FALSO si todas las condiciones son falsas.

Sintaxis

O(lógico1, [lógico2], ...)

Ejemplo

Supongamos que quieres determinar si un estudiante ha aprobado ya sea el examen de Matemáticas o el de Inglés. La calificación mínima para ambos sujetos es 50.

| Estudiante | Calificación de Matemáticas | Calificación de Inglés | Resultado | |------------|-----------------------------|------------------------|-----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Puedes usar la siguiente fórmula en la columna "Resultado":

=SI(O(B2 >= 50, C2 >= 50), "Aprobado", "Reprobado")

Explicación

  • lógico1: B2 >= 50 (verifica si la calificación de Matemáticas es mayor o igual a 50)
  • lógico2: C2 >= 50 (verifica si la calificación de Inglés es mayor o igual a 50)

Combinando SI con Y/O

Puedes combinar la función SI con Y y O para crear pruebas lógicas más complejas.

Ejemplo

Supongamos que quieres determinar si un estudiante ha aprobado tanto los exámenes de Matemáticas como de Inglés, pero si ha aprobado al menos uno, se le debe dar una opción de "Repetir".

| Estudiante | Calificación de Matemáticas | Calificación de Inglés | Resultado | |------------|-----------------------------|------------------------|-----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Puedes usar la siguiente fórmula en la columna "Resultado":

=SI(Y(B2 >= 50, C2 >= 50), "Aprobado", SI(O(B2 >= 50, C2 >= 50), "Repetir", "Reprobado"))

Explicación

  • El primer SI verifica si ambas calificaciones son >= 50.
  • Si es verdadero, devuelve "Aprobado".
  • Si es falso, verifica si alguna calificación es >= 50.
  • Si es verdadero, devuelve "Repetir".
  • Si es falso, devuelve "Reprobado".

Conclusión

Las funciones lógicas como SI, Y, y O son herramientas poderosas en Excel que te permiten realizar operaciones condicionales complejas. Al entender y combinar estas funciones, puedes crear fórmulas dinámicas y flexibles para satisfacer una amplia gama de necesidades. Practica usando estas funciones con diferentes conjuntos de datos para volverte competente en su aplicación.
