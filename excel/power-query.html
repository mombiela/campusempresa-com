Introducción a Power Query

Power Query es una potente tecnología de conexión de datos que te permite descubrir, conectar, combinar y refinar datos de una amplia variedad de fuentes. Está disponible en Excel y se puede utilizar para automatizar tareas de preparación de datos.

Conceptos Clave

  • Fuentes de Datos: Power Query puede conectarse a varias fuentes de datos como archivos de Excel, bases de datos, páginas web y más.
  • Editor de Consultas: La interfaz donde puedes dar forma y transformar tus datos.
  • Pasos: Cada transformación que aplicas a tus datos se registra como un paso en el Editor de Consultas.
  • Lenguaje M: El lenguaje de fórmulas utilizado por Power Query para realizar transformaciones de datos.

Empezando con Power Query

Cargando Datos

  1. Desde un Libro de Excel:

    • Ve a la pestaña Datos.
    • Haz clic en Obtener Datos > Desde Archivo > Desde Libro.
    • Selecciona el archivo de Excel y haz clic en Importar.

  2. Desde la Web:

    • Ve a la pestaña Datos.
    • Haz clic en Obtener Datos > Desde Otras Fuentes > Desde Web.
    • Ingresa la URL de la página web y haz clic en Aceptar.

Transformaciones Básicas

  • Eliminar Columnas: Selecciona las columnas que deseas eliminar, haz clic derecho y elige Eliminar Columnas.
  • Filtrar Filas: Haz clic en la flecha desplegable en el encabezado de la columna y selecciona los valores que deseas filtrar.
  • Cambiar Tipos de Datos: Haz clic en el encabezado de la columna, ve a la pestaña Transformar y selecciona el tipo de datos apropiado.

Ejemplo

1. Cargar datos desde un libro de Excel.
2. Eliminar columnas innecesarias.
3. Filtrar filas para incluir solo datos del año 2022.
4. Cambiar el tipo de datos de la columna 'Fecha' a Fecha.

Técnicas Intermedias de Power Query

Fusionar Consultas

  • Unión Interna: Solo incluye filas donde hay una coincidencia en ambas tablas.
  • Unión Externa Izquierda: Incluye todas las filas de la tabla izquierda y las filas coincidentes de la tabla derecha.

Ejemplo

1. Cargar datos desde dos libros de Excel diferentes.
2. Fusionar las consultas usando una Unión Interna en la columna 'ID'.
3. Eliminar columnas duplicadas de la tabla fusionada.

Agrupar Datos

  • Agrupar Por: Resume tus datos agrupándolos en base a una o más columnas.
  • Agregaciones: Realiza cálculos como suma, promedio, conteo, etc., en los datos agrupados.

Ejemplo

1. Cargar datos de ventas.
2. Agrupar por 'Región' y calcular las ventas totales para cada región.
3. Ordenar los datos agrupados por ventas totales en orden descendente.

Técnicas Avanzadas de Power Query

Columnas Personalizadas

  • Agregar Columnas Personalizadas: Usa la pestaña Agregar Columna para crear nuevas columnas basadas en datos existentes.
  • Lenguaje M: Escribe código M personalizado para realizar transformaciones complejas.

Ejemplo

1. Cargar datos de clientes.
2. Agregar una columna personalizada para calcular la edad de cada cliente en base a su fecha de nacimiento.
3. Usar el lenguaje M para crear una columna condicional que categorice a los clientes como 'Menor' o 'Adulto'.

Transformaciones Avanzadas de Datos

  • Pivotar y Despivotar: Reorganiza tus datos pivotando columnas en filas o viceversa.
  • Parametrización: Crea parámetros para hacer tus consultas más dinámicas y reutilizables.

Ejemplo

1. Cargar datos de ventas de productos.
2. Despivotar los datos de ventas para transformar columnas en filas.
3. Crear un parámetro para el umbral de ventas y filtrar los datos en base a este parámetro.

Conclusión

Power Query es una herramienta esencial para cualquiera que trabaje con datos en Excel. Simplifica el proceso de preparación de datos, permitiéndote conectar, transformar y combinar datos de varias fuentes con facilidad. Al dominar Power Query, puedes automatizar tareas repetitivas, mejorar la precisión de los datos y hacer tu análisis de datos más eficiente.

Resumen

  • Introducción: Entender los conceptos básicos de Power Query y sus conceptos clave.
  • Empezando: Cargar datos y realizar transformaciones básicas.
  • Técnicas Intermedias: Fusionar consultas y agrupar datos.
  • Técnicas Avanzadas: Agregar columnas personalizadas, transformaciones avanzadas y parametrización.

Siguiendo este enfoque estructurado, podrás aprovechar todo el potencial de Power Query en Excel, desde niveles principiantes hasta avanzados.
