Introducción

El texturizado y sombreado son aspectos cruciales del modelado 3D que dan vida a tus modelos al agregar color, textura y propiedades de material. Esta sección te guiará desde los conceptos básicos hasta técnicas avanzadas en Blender.

Conceptos Básicos del Texturizado

El texturizado implica aplicar imágenes o patrones a la superficie de un modelo 3D para darle detalle y realismo.

Mapeo UV

  • Definición: El mapeo UV es el proceso de proyectar una textura de imagen 2D sobre un modelo 3D.
  • Pasos:
    1. Selecciona tu modelo en Blender.
    2. Entra en el Modo Edición (Tecla Tab).
    3. Desenvuelve el modelo (Tecla U) y elige un método de desenvuelto (por ejemplo, Proyecto UV Inteligente).
    4. Abre el Editor UV/Imagen para ver el mapa UV desenvuelto.
    5. Carga o crea una textura y aplícala al mapa UV.

Aplicación de Texturas

  • Texturas de Imagen:
    1. Ve al espacio de trabajo Sombreado.
    2. Añade un nuevo material en la pestaña Propiedades del Material.
    3. Usa el Editor de Sombras para añadir un nodo de Textura de Imagen.
    4. Conecta el nodo de Textura de Imagen a la entrada de Color Base del sombreado Principled BSDF.
    5. Carga tu textura de imagen en el nodo de Textura de Imagen.
# Ejemplo de una configuración básica de nodos para aplicar una textura de imagen
import bpy

# Crear un nuevo material
material = bpy.data.materials.new(name="MiMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes

# Limpiar nodos predeterminados
for node in nodes:
    nodes.remove(node)

# Añadir nodo Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = 0,0

# Añadir nodo de Textura de Imagen
texture = nodes.new(type='ShaderNodeTexImage')
texture.location = -300,0
texture.image = bpy.data.images.load("ruta/a/tu/imagen.png")

# Enlazar nodos
material.node_tree.links.new(bsdf.inputs['Base Color'], texture.outputs['Color'])

# Asignar material al objeto
obj = bpy.context.active_object
if obj.data.materials:
    obj.data.materials[0] = material
else:
    obj.data.materials.append(material)

Técnicas Intermedias de Texturizado

Texturas Procedurales

Las texturas procedurales se generan usando algoritmos matemáticos en lugar de archivos de imagen.

  • Textura de Ruido:

    • Añade un nodo de Textura de Ruido en el Editor de Sombras.
    • Conéctalo a la entrada de Color Base del sombreado Principled BSDF.
    • Ajusta los parámetros de Escala, Detalle y Aspereza para lograr el efecto deseado.

  • Textura Voronoi:

    • Añade un nodo de Textura Voronoi.
    • Conéctalo a la entrada de Color Base.
    • Experimenta con los parámetros de Escala y Aleatoriedad.

Pintura de Texturas

  • Pasos:
    1. Selecciona tu modelo y entra en el modo Pintura de Texturas.
    2. Crea una nueva textura en el panel Ranuras de Textura.
    3. Usa las herramientas de pintura para pintar directamente sobre el modelo.

Conceptos Básicos del Sombreado

El sombreado define cómo la luz interactúa con las superficies de tu modelo.

Sombreado Principled BSDF

  • Visión General: El sombreado Principled BSDF es un sombreado versátil que combina múltiples capas de sombreado en uno.
  • Entradas Clave:
    • Color Base: El color base del material.
    • Metálico: Determina si el material es metálico.
    • Aspereza: Controla la suavidad de la superficie.
    • Especular: Ajusta la intensidad de las reflexiones especulares.

Añadiendo Sombras

  • Sombra Difusa:

    • Usar para superficies no reflectantes y mates.
    • Conecta el nodo Diffuse BSDF a la entrada Surface del nodo Material Output.

  • Sombra Brillante:

    • Usar para superficies reflectantes.
    • Conecta el nodo Glossy BSDF a la entrada Surface.
# Ejemplo de una configuración básica de nodos para una sombra brillante
import bpy

# Crear un nuevo material
material = bpy.data.materials.new(name="MaterialBrillante")
material.use_nodes = True
nodes = material.node_tree.nodes

# Limpiar nodos predeterminados
for node in nodes:
    nodes.remove(node)

# Añadir nodo Glossy BSDF
glossy = nodes.new(type='ShaderNodeBsdfGlossy')
glossy.location = 0,0

# Enlazar nodos
material.node_tree.links.new(glossy.outputs['BSDF'], nodes.new(type='ShaderNodeOutputMaterial').inputs['Surface'])

# Asignar material al objeto
obj = bpy.context.active_object
if obj.data.materials:
    obj.data.materials[0] = material
else:
    obj.data.materials.append(material)

Técnicas Avanzadas de Sombreado

Sombreado Basado en Nodos

  • Mezcla de Sombras:

    • Usa el nodo Mix Shader para mezclar dos sombras diferentes.
    • Conecta la salida del nodo Mix Shader a la entrada Surface del nodo Material Output.

  • Uso de Coordenadas de Textura:

    • Añade un nodo de Coordenada de Textura para controlar cómo se mapean las texturas en el modelo.
    • Conecta la salida UV a la entrada Vector de los nodos de textura.

Dispersión Subsuperficial (SSS)

  • Definición: SSS simula la forma en que la luz penetra la superficie de un material translúcido, se dispersa dentro y sale en un punto diferente.
  • Uso:
    • Habilita Subsurface en el sombreado Principled BSDF.
    • Ajusta el Radio de Subsurface y el Color de Subsurface para lograr el efecto deseado.

Conclusión

El texturizado y sombreado son habilidades esenciales para crear modelos 3D realistas y visualmente atractivos en Blender. Al dominar el mapeo UV, las texturas procedurales, la pintura de texturas y las técnicas avanzadas de sombreado, puedes mejorar significativamente la calidad de tus proyectos 3D. Practica estas técnicas regularmente para volverte competente en la creación de modelos detallados y realistas.
