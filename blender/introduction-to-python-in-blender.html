Introducción a Python en Blender
Scripting Básico ►

Resumen

Blender es una potente suite de creación 3D de código abierto que soporta toda la línea de producción 3D. Python es un lenguaje de programación versátil que Blender utiliza para scripting y automatización. Esta sección te introducirá al uso de Python dentro de Blender, cubriendo lo básico y proporcionando ejemplos para que comiences.

¿Por qué usar Python en Blender?

  • Automatización: Automatiza tareas repetitivas para ahorrar tiempo.
  • Personalización: Crea herramientas y complementos personalizados para extender la funcionalidad de Blender.
  • Eficiencia: Mejora la eficiencia del flujo de trabajo mediante scripting de operaciones complejas.

Configuración de Python en Blender

Blender viene con un intérprete de Python integrado, por lo que no es necesario realizar configuraciones adicionales. Puedes comenzar a hacer scripting directamente dentro de Blender.

Accediendo a la Consola de Python

  1. Abre Blender.
  2. Cambia al espacio de trabajo "Scripting" desde el menú superior.
  3. Verás la Consola de Python donde puedes escribir y ejecutar comandos de Python.

Comandos básicos de Python en Blender

Comencemos con algunos comandos básicos para familiarizarnos con la API de Python en Blender.

Imprimiendo un Mensaje

print("¡Hola, Blender!")
  • Este comando imprimirá "¡Hola, Blender!" en la consola.

Accediendo a Datos de Blender

import bpy

# Imprime el nombre de la escena actual
print(bpy.context.scene.name)
  • bpy es el módulo de Python de Blender.
  • bpy.context.scene te da acceso a la escena actual.
  • .name recupera el nombre de la escena.

Creando y Manipulando Objetos

Añadiendo un Cubo

import bpy

# Añade un cubo a la escena
bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
  • bpy.ops.mesh.primitive_cube_add es el operador para añadir un cubo.
  • size establece el tamaño del cubo.
  • location establece la posición del cubo en el espacio 3D.

Moviendo un Objeto

import bpy

# Selecciona el cubo
cube = bpy.context.selected_objects[0]

# Mueve el cubo
cube.location.x += 2
  • bpy.context.selected_objects devuelve una lista de objetos seleccionados.
  • cube.location.x accede a la coordenada x de la ubicación del cubo y lo mueve 2 unidades.

Creando un Script Simple

Vamos a crear un script simple que añada un cubo y lo mueva.

import bpy

def add_and_move_cube():
    # Añade un cubo
    bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
    
    # Selecciona el cubo
    cube = bpy.context.selected_objects[0]
    
    # Mueve el cubo
    cube.location.x += 2
    cube.location.y += 3
    cube.location.z += 1

# Ejecuta la función
add_and_move_cube()
  • Este script define una función add_and_move_cube que añade un cubo y lo mueve a una nueva ubicación.
  • Luego se llama a la función para ejecutar las operaciones.

Conclusión

En esta sección, hemos introducido los conceptos básicos del uso de Python en Blender. Aprendiste cómo acceder a la consola de Python, ejecutar comandos simples y crear un script básico para añadir y manipular objetos. El scripting en Python en Blender puede mejorar significativamente tu productividad y permitirte crear herramientas personalizadas adaptadas a tu flujo de trabajo. A medida que avances en el curso, profundizarás en técnicas de scripting más avanzadas y aplicaciones.
