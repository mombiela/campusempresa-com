Introducción al Mapeo UV

El mapeo UV es una técnica utilizada en el modelado 3D para proyectar una textura de imagen 2D sobre un modelo 3D. Este proceso es esencial para agregar texturas detalladas a tus modelos en Blender.

Conceptos Clave

  • Coordenadas UV: Son las coordenadas que asignan la textura 2D al modelo 3D.
  • Desenvolvimiento: El proceso de aplanar la superficie de un modelo 3D en un plano 2D.
  • Costuras: Bordes marcados para definir dónde se 'cortará' el modelo para desenvolverse.
  • Textura: La imagen 2D aplicada al modelo 3D.

Mapeo UV Básico

Configuración de Tu Modelo

  1. Crear o Importar un Modelo: Comienza con un modelo simple como un cubo o una esfera.
  2. Cambiar al Espacio de Trabajo de Edición UV: Ve al espacio de trabajo de Edición UV en Blender.

Desenvolviendo Tu Modelo

  1. Marcar Costuras:
    • Selecciona los bordes donde deseas crear costuras.
    • Presiona Ctrl + E y elige "Marcar Costura".
  2. Desenvolver:
    • Selecciona todo el modelo en Modo Edición.
    • Presiona U y elige "Desenvolver".

Aplicando una Textura

  1. Cargar una Textura:
    • En el Editor UV, abre un archivo de imagen para usar como textura.
  2. Asignar la Textura:
    • En el Editor de Sombras, agrega un nodo de Textura de Imagen y conéctalo al Color Base del shader Principled BSDF.
    • Carga el mismo archivo de imagen en el nodo de Textura de Imagen.

Ejemplo

# Este es un script de Python para automatizar el desenvolvimiento UV en Blender
import bpy

# Seleccionar el objeto
obj = bpy.context.active_object

# Entrar en Modo Edición
bpy.ops.object.mode_set(mode='EDIT')

# Marcar costuras
bpy.ops.mesh.mark_seam(clear=False)

# Desenvolver
bpy.ops.uv.unwrap(method='ANGLE_BASED', margin=0.001)

# Volver a Modo Objeto
bpy.ops.object.mode_set(mode='OBJECT')

Mapeo UV Intermedio

Técnicas Avanzadas de Desenvolvimiento

  • Proyecto UV Inteligente: Desenvolver automáticamente el modelo basado en un umbral de ángulo.
  • Paquete de Mapa de Luz: Crea un mapa UV adecuado para mapas de luz.

Ajustando UVs

  • Escalar y Rotar UVs: Usa S para escalar y R para rotar islas UV.
  • Fijar UVs: Fija vértices en el Editor UV para mantenerlos fijos durante futuros desenvolvimientos.

Ejemplo

# Ajustando UVs usando Python
import bpy

# Seleccionar el objeto
obj = bpy.context.active_object

# Entrar en Modo Edición
bpy.ops.object.mode_set(mode='EDIT')

# Seleccionar todos los vértices UV
bpy.ops.uv.select_all(action='SELECT')

# Escalar UVs
bpy.ops.transform.resize(value=(1.5, 1.5, 1.5))

# Rotar UVs
bpy.ops.transform.rotate(value=0.785398, orient_axis='Z')

# Volver a Modo Objeto
bpy.ops.object.mode_set(mode='OBJECT')

Mapeo UV Avanzado

Mapeo UV de Modelos Complejos

  • Usando Múltiples Mapas UV: Crea y gestiona múltiples mapas UV para diferentes texturas.
  • Diseños UV Personalizados: Ajusta manualmente las islas UV para un uso óptimo del espacio de textura.

Horneado de Texturas

  • Oclusión Ambiental: Hornea mapas de oclusión ambiental para agregar sombreado realista.
  • Mapas Normales: Hornea mapas normales para agregar detalles de superficie sin aumentar la geometría.

Ejemplo

# Horneando un mapa de Oclusión Ambiental
import bpy

# Seleccionar el objeto
obj = bpy.context.active_object

# Asegurarse de que el objeto tenga un mapa UV
if not obj.data.uv_layers:
    bpy.ops.uv.smart_project()

# Crear una nueva imagen para hornear
bpy.ops.image.new(name="AO_Bake", width=1024, height=1024)

# Establecer la imagen como activa
bpy.data.images['AO_Bake'].use_fake_user = True

# Establecer el tipo de horneado a Oclusión Ambiental
bpy.context.scene.render.bake.use_pass_ambient_occlusion = True

# Hornear la textura
bpy.ops.object.bake(type='AO')

# Guardar la imagen horneada
bpy.data.images['AO_Bake'].save_render(filepath="/path/to/save/AO_Bake.png")

Conclusión

El mapeo UV es una habilidad crítica para agregar texturas detalladas a tus modelos 3D en Blender. Comenzando con el desenvolvimiento básico y la aplicación de texturas, puedes progresar a técnicas más avanzadas como diseños UV personalizados y horneado de texturas. El dominio del mapeo UV mejorará significativamente el realismo y el detalle de tus proyectos 3D.
