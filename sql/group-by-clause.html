Cláusula GROUP BY
Cláusula HAVING ►

La cláusula GROUP BY en SQL se utiliza para organizar datos idénticos en grupos. Esto es particularmente útil cuando se combina con funciones agregadas como COUNT(), SUM(), AVG(), MAX(), y MIN(). En esta sección, exploraremos la cláusula GROUP BY desde un nivel principiante hasta uno avanzado.

Introducción a GROUP BY

La cláusula GROUP BY se utiliza para agrupar filas que tienen los mismos valores en columnas especificadas en filas de resumen. A menudo se usa con funciones agregadas para realizar cálculos en cada grupo de datos.

Sintaxis Básica

SELECT column1, aggregate_function(column2)
FROM table_name
GROUP BY column1;
  • column1: La columna por la cual deseas agrupar los datos.
  • aggregate_function(column2): La función agregada que se aplicará a cada grupo.

Ejemplo

Considera una tabla Sales con los siguientes datos:

| SaleID | Product | Quantity | Price | |--------|---------|----------|-------| | 1 | Apple | 10 | 1.00 | | 2 | Orange | 5 | 0.80 | | 3 | Apple | 7 | 1.00 | | 4 | Banana | 3 | 0.50 | | 5 | Orange | 10 | 0.80 |

Para encontrar la cantidad total vendida de cada producto, puedes usar la cláusula GROUP BY:

SELECT Product, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY Product;

Esto producirá el siguiente resultado:

| Product | TotalQuantity | |---------|---------------| | Apple | 17 | | Orange | 15 | | Banana | 3 |

Usando GROUP BY con Múltiples Columnas

Puedes agrupar por múltiples columnas para obtener resultados más detallados.

Ejemplo

Considera la misma tabla Sales. Para encontrar la cantidad total vendida de cada producto a cada precio:

SELECT Product, Price, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY Product, Price;

Esto producirá el siguiente resultado:

| Product | Price | TotalQuantity | |---------|-------|---------------| | Apple | 1.00 | 17 | | Orange | 0.80 | 15 | | Banana | 0.50 | 3 |

Filtrando Grupos con HAVING

La cláusula HAVING se utiliza para filtrar grupos basados en una condición. Es similar a la cláusula WHERE pero se usa para grupos.

Ejemplo

Para encontrar productos con una cantidad total vendida mayor a 10:

SELECT Product, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY Product
HAVING SUM(Quantity) > 10;

Esto producirá el siguiente resultado:

| Product | TotalQuantity | |---------|---------------| | Apple | 17 | | Orange | 15 |

Técnicas Avanzadas de GROUP BY

Conjuntos de Agrupación

GROUPING SETS te permite definir múltiples agrupaciones en la misma consulta.

Ejemplo

Para obtener la cantidad total vendida de cada producto, cada precio y el total general:

SELECT Product, Price, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY GROUPING SETS ((Product), (Price), ());

Esto producirá el siguiente resultado:

| Product | Price | TotalQuantity | |---------|-------|---------------| | Apple | NULL | 17 | | Orange | NULL | 15 | | Banana | NULL | 3 | | NULL | 1.00 | 17 | | NULL | 0.80 | 15 | | NULL | 0.50 | 3 | | NULL | NULL | 35 |

Rollup

ROLLUP se usa para crear subtotales y un total general.

Ejemplo

Para obtener subtotales de cada producto y un total general:

SELECT Product, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY ROLLUP(Product);

Esto producirá el siguiente resultado:

| Product | TotalQuantity | |---------|---------------| | Apple | 17 | | Orange | 15 | | Banana | 3 | | NULL | 35 |

Cube

CUBE genera un conjunto de resultados que muestra agregados para todas las combinaciones de valores en las columnas seleccionadas.

Ejemplo

Para obtener todos los posibles subtotales y un total general:

SELECT Product, Price, SUM(Quantity) AS TotalQuantity
FROM Sales
GROUP BY CUBE(Product, Price);

Esto producirá el siguiente resultado:

| Product | Price | TotalQuantity | |---------|-------|---------------| | Apple | 1.00 | 17 | | Orange | 0.80 | 15 | | Banana | 0.50 | 3 | | Apple | NULL | 17 | | Orange | NULL | 15 | | Banana | NULL | 3 | | NULL | 1.00 | 17 | | NULL | 0.80 | 15 | | NULL | 0.50 | 3 | | NULL | NULL | 35 |

Conclusión

La cláusula GROUP BY es una herramienta poderosa en SQL que te permite resumir y agregar datos de maneras significativas. Al dominar GROUP BY, HAVING, y técnicas avanzadas como GROUPING SETS, ROLLUP, y CUBE, puedes realizar análisis de datos complejos e informes. Practica estos conceptos con varios conjuntos de datos para volverte competente en el uso efectivo de la cláusula GROUP BY.
