Introducción

En esta sección, exploraremos cómo usar las utilidades del sistema en JCL (Job Control Language). Las utilidades del sistema son herramientas esenciales que ayudan a gestionar y manipular conjuntos de datos, realizar mantenimiento del sistema y automatizar diversas tareas. Entender estas utilidades es crucial tanto para usuarios principiantes como avanzados de JCL.

Principales Utilidades del Sistema

IEBGENER

IEBGENER es una utilidad utilizada para copiar conjuntos de datos, crear copias de seguridad e imprimir conjuntos de datos.

  • Propósito: Copiar datos de un conjunto de datos a otro.
  • Casos de Uso Comunes:
    • Copiar conjuntos de datos secuenciales.
    • Crear copias de seguridad de conjuntos de datos.
    • Imprimir conjuntos de datos.

Ejemplo

//COPYJOB  JOB (ACCT),'COPY DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEBGENER
//SYSPRINT DD  SYSOUT=A
//SYSUT1   DD  DSN=INPUT.DATASET,DISP=SHR
//SYSUT2   DD  DSN=OUTPUT.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=*.SYSUT1
//SYSIN    DD  DUMMY
  • Explicación:
    • SYSUT1 especifica el conjunto de datos de entrada.
    • SYSUT2 especifica el conjunto de datos de salida.
    • SYSIN se establece en DUMMY porque no se necesitan declaraciones de control.

IEFBR14

IEFBR14 es un programa de utilidad simple que no hace nada más que devolver el control al sistema operativo. A menudo se utiliza para asignar o eliminar conjuntos de datos.

  • Propósito: Asignar o eliminar conjuntos de datos.
  • Casos de Uso Comunes:
    • Asignar nuevos conjuntos de datos.
    • Eliminar conjuntos de datos existentes.

Ejemplo

//ALLOCJOB JOB (ACCT),'ALLOCATE DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//NEWDS    DD  DSN=NEW.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)
  • Explicación:
    • NEWDS especifica el nuevo conjunto de datos a asignar.
    • DISP=(NEW,CATLG,DELETE) indica que el conjunto de datos debe ser catalogado si el trabajo se completa con éxito y eliminado si falla.

IDCAMS

IDCAMS (Integrated Data Cluster Access Method Services) es una utilidad versátil utilizada para gestionar conjuntos de datos VSAM y realizar tareas relacionadas con el catálogo.

  • Propósito: Gestionar conjuntos de datos VSAM y catálogos.
  • Casos de Uso Comunes:
    • Definir conjuntos de datos VSAM.
    • Eliminar conjuntos de datos VSAM.
    • Listar entradas de catálogo.

Ejemplo

//IDCAMSJOB JOB (ACCT),'IDCAMS EXAMPLE',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD  SYSOUT=A
//SYSIN    DD  *
  DELETE MY.VSAM.DATASET CLUSTER
  DEFINE CLUSTER (NAME(MY.VSAM.DATASET) -
                  INDEXED -
                  RECORDSIZE(80 80) -
                  KEYS(8 0) -
                  TRACKS(1 1) -
                  FREESPACE(3 3))
/*
  • Explicación:
    • El comando DELETE elimina el conjunto de datos VSAM especificado.
    • El comando DEFINE CLUSTER crea un nuevo conjunto de datos VSAM con atributos especificados.

SORT

SORT es una utilidad utilizada para ordenar, fusionar y copiar conjuntos de datos. Es altamente flexible y puede manejar tareas complejas de manipulación de datos.

  • Propósito: Ordenar, fusionar y copiar conjuntos de datos.
  • Casos de Uso Comunes:
    • Ordenar conjuntos de datos basados en campos especificados.
    • Fusionar múltiples conjuntos de datos.
    • Filtrar y copiar conjuntos de datos.

Ejemplo

//SORTJOB  JOB (ACCT),'SORT EXAMPLE',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=SORT
//SORTIN   DD  DSN=INPUT.DATASET,DISP=SHR
//SORTOUT  DD  DSN=OUTPUT.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=*.SORTIN
//SYSIN    DD  *
  SORT FIELDS=(1,8,CH,A)
/*
  • Explicación:
    • SORTIN especifica el conjunto de datos de entrada.
    • SORTOUT especifica el conjunto de datos de salida.
    • SORT FIELDS=(1,8,CH,A) ordena el conjunto de datos basado en los primeros 8 caracteres en orden ascendente.

Conclusión

Entender y utilizar las utilidades del sistema en JCL es esencial para una gestión eficiente de conjuntos de datos y automatización de trabajos. Al dominar utilidades como IEBGENER, IEFBR14, IDCAMS y SORT, puedes realizar una amplia gama de tareas, desde la simple copia de conjuntos de datos hasta la manipulación compleja de datos. Practica usando estas utilidades con diferentes conjuntos de datos y escenarios para volverte competente en JCL.
