Introducción

Leer archivos es una habilidad fundamental en la programación, que te permite procesar y manipular datos almacenados en archivos. En Ruby, leer archivos es sencillo y se puede hacer utilizando varios métodos. Esta sección te guiará a través de los conceptos básicos de la lectura de archivos, desde la lectura simple de archivos hasta técnicas más avanzadas.

Lectura Básica de Archivos

Ruby proporciona una clase File incorporada para manejar operaciones de archivos. Los métodos más comunes para leer archivos son File.open y File.read.

Usando `File.read`

El método File.read lee todo el contenido de un archivo en una cadena.

# Ejemplo: Leyendo todo el contenido de un archivo
file_content = File.read('example.txt')
puts file_content
  • Explicación:
    • File.read('example.txt'): Lee todo el contenido de example.txt en la variable file_content.
    • puts file_content: Muestra el contenido del archivo en la consola.

Usando `File.open` con un Bloque

El método File.open se puede usar con un bloque para asegurar que el archivo se cierre correctamente después de leerlo.

# Ejemplo: Leyendo un archivo usando File.open con un bloque
File.open('example.txt', 'r') do |file|
  file.each_line do |line|
    puts line
  end
end
  • Explicación:
    • File.open('example.txt', 'r'): Abre example.txt en modo de lectura ('r').
    • file.each_line: Itera sobre cada línea en el archivo.
    • puts line: Muestra cada línea en la consola.

Leyendo Archivos Línea por Línea

Leer archivos línea por línea es útil para procesar archivos grandes sin cargar todo el contenido en la memoria.

Usando `File.foreach`

El método File.foreach lee un archivo línea por línea.

# Ejemplo: Leyendo un archivo línea por línea usando File.foreach
File.foreach('example.txt') do |line|
  puts line
end
  • Explicación:
    • File.foreach('example.txt'): Itera sobre cada línea en example.txt.
    • puts line: Muestra cada línea en la consola.

Usando `IO.readlines`

El método IO.readlines lee todas las líneas de un archivo en un array.

# Ejemplo: Leyendo todas las líneas en un array usando IO.readlines
lines = IO.readlines('example.txt')
lines.each do |line|
  puts line
end
  • Explicación:
    • IO.readlines('example.txt'): Lee todas las líneas de example.txt en el array lines.
    • lines.each: Itera sobre cada línea en el array.
    • puts line: Muestra cada línea en la consola.

Lectura Avanzada de Archivos

Para una lectura de archivos más avanzada, puedes usar métodos para manejar diferentes codificaciones, leer archivos binarios o usar flujos de archivos.

Manejo de Diferentes Codificaciones

Ruby te permite especificar la codificación al abrir un archivo.

# Ejemplo: Leyendo un archivo con una codificación específica
File.open('example.txt', 'r:UTF-8') do |file|
  file.each_line do |line|
    puts line
  end
end
  • Explicación:
    • File.open('example.txt', 'r:UTF-8'): Abre example.txt en modo de lectura con codificación UTF-8.

Leyendo Archivos Binarios

Para leer archivos binarios, abre el archivo en modo binario.

# Ejemplo: Leyendo un archivo binario
File.open('example.bin', 'rb') do |file|
  binary_data = file.read
  puts binary_data
end
  • Explicación:
    • File.open('example.bin', 'rb'): Abre example.bin en modo de lectura binaria ('rb').
    • file.read: Lee los datos binarios del archivo.
    • puts binary_data: Muestra los datos binarios en la consola (puede no ser legible para humanos).

Conclusión

Leer archivos en Ruby es una habilidad versátil y esencial. Ya sea que necesites leer todo el contenido de un archivo, procesarlo línea por línea, o manejar diferentes codificaciones y datos binarios, Ruby proporciona métodos sencillos para lograr estas tareas. Practica estas técnicas para volverte competente en el manejo de archivos, lo cual es crucial para muchas aplicaciones del mundo real.
