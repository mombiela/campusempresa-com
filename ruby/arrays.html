Introducción a los Arrays

Los arrays son colecciones ordenadas e indexadas por enteros de cualquier objeto. Son una estructura de datos fundamental en Ruby, que te permite almacenar y manipular listas de datos.

Conceptos Clave

  • Colección Ordenada: Los elementos en un array se almacenan en un orden específico.
  • Indexación: Los arrays usan indexación basada en cero, lo que significa que el primer elemento está en el índice 0.
  • Tamaño Dinámico: Los arrays pueden crecer o reducirse según sea necesario.

Creación de Arrays

Puedes crear arrays usando corchetes [] o la clase Array.

# Usando corchetes
numbers = [1, 2, 3, 4, 5]

# Usando la clase Array
words = Array.new
words = Array.new(3) # Crea un array con 3 elementos nil
words = Array.new(3, "hello") # Crea un array con 3 elementos "hello"

Acceso a Elementos

Accede a los elementos usando su índice.

numbers = [1, 2, 3, 4, 5]
puts numbers[0] # Salida: 1
puts numbers[4] # Salida: 5

Modificación de Arrays

Puedes modificar arrays asignando valores a índices específicos o usando métodos como push, pop, shift y unshift.

numbers = [1, 2, 3, 4, 5]
numbers[2] = 10 # Modifica el tercer elemento
numbers.push(6) # Añade 6 al final del array
numbers.pop # Elimina el último elemento
numbers.shift # Elimina el primer elemento
numbers.unshift(0) # Añade 0 al principio del array

Métodos de Array

Ruby proporciona un conjunto rico de métodos para manipular arrays.

Métodos Comunes

  • length o size: Devuelve el número de elementos.
  • empty?: Verifica si el array está vacío.
  • include?: Verifica si un elemento está presente.
  • sort: Ordena el array.
  • reverse: Invierte el array.
  • join: Convierte el array en una cadena.
numbers = [3, 1, 4, 1, 5, 9]
puts numbers.length # Salida: 6
puts numbers.empty? # Salida: false
puts numbers.include?(4) # Salida: true
puts numbers.sort # Salida: [1, 1, 3, 4, 5, 9]
puts numbers.reverse # Salida: [9, 5, 1, 4, 1, 3]
puts numbers.join(", ") # Salida: "3, 1, 4, 1, 5, 9"

Iteración Sobre Arrays

Usa bucles o iteradores para procesar cada elemento en un array.

numbers = [1, 2, 3, 4, 5]

# Usando un bucle for
for number in numbers
  puts number
end

# Usando el método each
numbers.each do |number|
  puts number
end

Técnicas Avanzadas de Array

Arrays Anidados

Los arrays pueden contener otros arrays, creando arrays multidimensionales.

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
puts matrix[1][2] # Salida: 6

Transformaciones de Array

Usa métodos como map, select y reject para transformar arrays.

numbers = [1, 2, 3, 4, 5]

# Usando map para crear un nuevo array con cada elemento al cuadrado
squares = numbers.map { |number| number ** 2 }
puts squares # Salida: [1, 4, 9, 16, 25]

# Usando select para filtrar elementos
evens = numbers.select { |number| number.even? }
puts evens # Salida: [2, 4]

# Usando reject para filtrar elementos
odds = numbers.reject { |number| number.even? }
puts odds # Salida: [1, 3, 5]

Operaciones de Conjunto de Array

Los arrays soportan operaciones de conjunto como unión, intersección y diferencia.

a = [1, 2, 3]
b = [3, 4, 5]

# Unión
puts (a | b) # Salida: [1, 2, 3, 4, 5]

# Intersección
puts (a & b) # Salida: [3]

# Diferencia
puts (a - b) # Salida: [1, 2]

Conclusión

Los arrays son una estructura de datos versátil y poderosa en Ruby, esencial para manejar colecciones de datos. Desde operaciones básicas como acceder y modificar elementos hasta técnicas avanzadas como transformaciones y operaciones de conjunto, dominar los arrays mejorará significativamente tus habilidades de programación en Ruby. Practica usando arrays en diferentes escenarios para volverte competente en su uso.
