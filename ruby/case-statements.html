Introducción a las declaraciones case

Las declaraciones case en Ruby son una herramienta poderosa para controlar el flujo de tu programa basado en diferentes condiciones. Son similares a las declaraciones switch en otros lenguajes de programación y proporcionan una alternativa más legible a múltiples declaraciones if-elsif-else.

Sintaxis básica

La sintaxis básica de una declaración case en Ruby es la siguiente:

case expresión
when valor1
  # código a ejecutar si expresión == valor1
when valor2
  # código a ejecutar si expresión == valor2
else
  # código a ejecutar si ninguna de las condiciones anteriores se cumple
end

Ejemplo

Aquí hay un ejemplo simple para ilustrar el uso de una declaración case:

día = "Sábado"

case día
when "Lunes"
  puts "Inicio de la semana laboral"
when "Viernes"
  puts "Fin de la semana laboral"
when "Sábado", "Domingo"
  puts "¡Fin de semana!"
else
  puts "Día entre semana"
end

En este ejemplo:

  • Si día es "Lunes", imprime "Inicio de la semana laboral".
  • Si día es "Viernes", imprime "Fin de la semana laboral".
  • Si día es "Sábado" o "Domingo", imprime "¡Fin de semana!".
  • Para cualquier otro valor de día, imprime "Día entre semana".

Uso avanzado

Usando rangos

También puedes usar rangos en las declaraciones case para manejar un rango de valores.

puntuación = 85

case puntuación
when 90..100
  puts "Excelente"
when 75..89
  puts "Bueno"
when 50..74
  puts "Aprobado"
else
  puts "Reprobado"
end

Usando expresiones regulares

Las declaraciones case también pueden coincidir con expresiones regulares.

nombre = "Alicia"

case nombre
when /Alic/
  puts "El nombre contiene 'Alic'"
when /^A/
  puts "El nombre empieza con 'A'"
else
  puts "El nombre no coincide con ningún patrón"
end

Usando múltiples condiciones

Puedes especificar múltiples condiciones en una sola cláusula when separándolas con comas.

fruta = "Manzana"

case fruta
when "Manzana", "Banana", "Cereza"
  puts "Esta es una fruta común"
else
  puts "Esta es una fruta poco común"
end

Mejores prácticas

  • Legibilidad: Usa declaraciones case para mejorar la legibilidad de tu código, especialmente cuando se trata de múltiples condiciones.
  • Simplicidad: Mantén las condiciones simples y evita lógica excesivamente compleja dentro de una sola declaración case.
  • Caso por defecto: Siempre incluye una cláusula else para manejar valores inesperados.

Resumen

Las declaraciones case en Ruby proporcionan una forma limpia y legible de manejar múltiples condiciones. Pueden usarse con valores simples, rangos, expresiones regulares y múltiples condiciones. Siguiendo las mejores prácticas, puedes hacer tu código más mantenible y fácil de entender.

Puntos clave

  • Las declaraciones case son una alternativa a múltiples declaraciones if-elsif-else.
  • Mejoran la legibilidad y mantenibilidad del código.
  • Pueden manejar rangos, expresiones regulares y múltiples condiciones.
  • Siempre incluye una cláusula else para manejar valores inesperados.

Dominando las declaraciones case, puedes escribir código Ruby más eficiente y legible.
