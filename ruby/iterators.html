Introducción a los Iteradores

Los iteradores son un concepto fundamental en Ruby, que te permite recorrer colecciones como arreglos y hashes. Proporcionan una forma de acceder a cada elemento en una colección sin necesidad de gestionar el índice manualmente.

Conceptos Clave

  • Iterador: Un método que recorre una colección, cediendo cada elemento a un bloque.
  • Bloque: Un fragmento de código encerrado en do...end o llaves {...} que puede ser pasado a métodos como los iteradores.

Iteradores Básicos

Ruby proporciona varios métodos de iterador incorporados, siendo el más común each.

Usando `each` con Arreglos

El método each se utiliza para iterar sobre cada elemento en un arreglo.

numbers = [1, 2, 3, 4, 5]
numbers.each do |number|
  puts number
end
  • Explicación: El método each recorre el arreglo numbers, cediendo cada elemento al bloque. La variable de bloque |number| representa el elemento actual.

Usando `each` con Hashes

El método each también puede ser usado con hashes, cediendo pares clave-valor.

person = {name: "Alice", age: 30, city: "New York"}
person.each do |key, value|
  puts "#{key}: #{value}"
end
  • Explicación: El método each recorre el hash person, cediendo cada par clave-valor al bloque. Las variables de bloque |key, value| representan la clave y el valor actuales.

Iteradores Avanzados

Ruby ofrece iteradores más avanzados para casos de uso específicos.

`map`

El método map crea un nuevo arreglo que contiene los valores devueltos por el bloque.

numbers = [1, 2, 3, 4, 5]
squares = numbers.map do |number|
  number * number
end
puts squares.inspect
  • Explicación: El método map itera sobre el arreglo numbers, aplicando el bloque a cada elemento y devolviendo un nuevo arreglo con los resultados.

`select`

El método select devuelve un nuevo arreglo que contiene elementos para los cuales el bloque devuelve true.

numbers = [1, 2, 3, 4, 5]
even_numbers = numbers.select do |number|
  number.even?
end
puts even_numbers.inspect
  • Explicación: El método select itera sobre el arreglo numbers, seleccionando elementos que son pares.

`reduce`

El método reduce combina todos los elementos de un arreglo aplicando una operación binaria, especificada por un bloque o símbolo.

numbers = [1, 2, 3, 4, 5]
sum = numbers.reduce(0) do |accumulator, number|
  accumulator + number
end
puts sum
  • Explicación: El método reduce itera sobre el arreglo numbers, acumulando la suma de todos los elementos.

Iteradores Personalizados

Puedes crear iteradores personalizados definiendo métodos que ceden a un bloque.

def custom_each(array)
  for element in array
    yield(element)
  end
end

custom_each([1, 2, 3, 4, 5]) do |number|
  puts number
end
  • Explicación: El método custom_each toma un arreglo y cede cada elemento al bloque, similar al método each incorporado.

Conclusión

Los iteradores son una característica poderosa en Ruby, que te permite recorrer y manipular colecciones con facilidad. Al entender y utilizar iteradores como each, map, select y reduce, puedes escribir código más conciso y legible. Los iteradores personalizados amplían aún más esta flexibilidad, permitiéndote crear comportamientos de iteración especializados adaptados a tus necesidades.
