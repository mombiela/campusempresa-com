Introducción a las Cadenas

Las cadenas son secuencias de caracteres utilizadas para representar texto en Ruby. Son uno de los tipos de datos más comúnmente usados y son esenciales para manejar datos textuales.

Creación de Cadenas

  • Comillas Simples: Las cadenas pueden ser creadas usando comillas simples (').
  • Comillas Dobles: Las cadenas también pueden ser creadas usando comillas dobles (").
cadena_comillas_simples = '¡Hola, Mundo!'
cadena_comillas_dobles = "¡Hola, Mundo!"

Diferencias Entre Comillas Simples y Dobles

  • Interpolación: Las cadenas con comillas dobles soportan la interpolación de cadenas, mientras que las cadenas con comillas simples no.
  • Secuencias de Escape: Las cadenas con comillas dobles soportan más secuencias de escape que las cadenas con comillas simples.
nombre = "Alicia"
comillas_simples = 'Hola, #{nombre}'  # Salida: Hola, #{nombre}
comillas_dobles = "Hola, #{nombre}"  # Salida: Hola, Alicia

escape_comillas_simples = 'Es un día soleado.'  # Salida: Es un día soleado.
escape_comillas_dobles = "Línea1\nLínea2"       # Salida: Línea1
                                                #          Línea2

Métodos de Cadenas

Ruby proporciona un conjunto rico de métodos para manipular cadenas.

Métodos Comunes de Cadenas

  • Longitud: Devuelve el número de caracteres en la cadena.
cadena = "Hola"
puts cadena.length  # Salida: 4
  • Mayúsculas/Minúsculas: Convierte todos los caracteres a mayúsculas o minúsculas.
puts cadena.upcase   # Salida: HOLA
puts cadena.downcase # Salida: hola
  • Subcadenas: Extrae una parte de la cadena.
puts cadena[0, 2]  # Salida: Ho
  • Concatenación: Combina cadenas juntas.
saludo = "Hola"
nombre = "Alicia"
puts saludo + " " + nombre  # Salida: Hola Alicia
  • Interpolación: Inserta variables o expresiones dentro de una cadena.
edad = 25
puts "Tengo #{edad} años."  # Salida: Tengo 25 años.

Métodos Avanzados de Cadenas

  • Dividir: Divide una cadena en un array basado en un delimitador.
frase = "¡Hola, Mundo!"
palabras = frase.split(", ")
puts palabras  # Salida: ["¡Hola", "Mundo!"]
  • Unir: Combina elementos de un array en una sola cadena.
palabras = ["¡Hola", "Mundo!"]
frase = palabras.join(", ")
puts frase  # Salida: ¡Hola, Mundo!
  • Gsub: Sustituye todas las ocurrencias de un patrón con un reemplazo.
texto = "El rápido zorro marrón"
nuevo_texto = texto.gsub("rápido", "lento")
puts nuevo_texto  # Salida: El lento zorro marrón

Comparación de Cadenas

Las cadenas pueden ser comparadas usando varios operadores.

Operadores de Comparación

  • Igualdad (==): Verifica si dos cadenas son idénticas.
puts "hola" == "hola"  # Salida: true
puts "hola" == "mundo"  # Salida: false
  • Desigualdad (!=): Verifica si dos cadenas no son idénticas.
puts "hola" != "mundo"  # Salida: true
  • Comparación Lexicográfica (<, >, <=, >=): Compara cadenas basadas en el orden alfabético.
puts "manzana" < "banana"  # Salida: true
puts "manzana" > "banana"  # Salida: false

Codificación de Cadenas

Ruby soporta múltiples codificaciones de cadenas, lo cual es esencial para manejar la internacionalización.

Verificación y Cambio de Codificación

  • Codificación: Devuelve la codificación de la cadena.
cadena = "Hola"
puts cadena.encoding  # Salida: UTF-8
  • Forzar Codificación: Cambia la codificación de la cadena.
cadena = "Hola"
cadena.force_encoding("ASCII-8BIT")
puts cadena.encoding  # Salida: ASCII-8BIT

Conclusión

Las cadenas son una parte fundamental de la programación en Ruby. Entender cómo crear, manipular y comparar cadenas es crucial para manejar datos textuales de manera efectiva. Al dominar los diversos métodos y técnicas discutidos, puedes trabajar eficientemente con cadenas en tus aplicaciones Ruby.
