Introducción

En Ruby, las variables y métodos de instancia son conceptos fundamentales que ayudan a definir el estado y el comportamiento de los objetos. Esta sección cubrirá qué son las variables y métodos de instancia, cómo usarlos, y proporcionará ejemplos y ejercicios para solidificar tu comprensión.

Variables de Instancia

¿Qué son las Variables de Instancia?

  • Las variables de instancia son variables que pertenecen a una instancia de una clase.
  • Se utilizan para almacenar el estado de un objeto.
  • Las variables de instancia se prefijan con un símbolo @.

Definiendo Variables de Instancia

Las variables de instancia se definen típicamente dentro de los métodos de instancia. Se inicializan cuando se crea un objeto y pueden ser accedidas por cualquier método de instancia dentro del mismo objeto.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_info
    puts "Name: #{@name}, Age: #{@age}"
  end
end

person = Person.new("Alice", 30)
person.display_info  # Output: Name: Alice, Age: 30

Accediendo a Variables de Instancia

Las variables de instancia pueden ser accedidas y modificadas dentro de los métodos de instancia. Sin embargo, no son accesibles directamente desde fuera de la clase.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def set_name(new_name)
    @name = new_name
  end

  def get_name
    @name
  end
end

person = Person.new("Alice", 30)
puts person.get_name  # Output: Alice
person.set_name("Bob")
puts person.get_name  # Output: Bob

Métodos de Instancia

¿Qué son los Métodos de Instancia?

  • Los métodos de instancia son métodos que pertenecen a una instancia de una clase.
  • Definen el comportamiento de un objeto.
  • Los métodos de instancia pueden acceder y modificar las variables de instancia.

Definiendo Métodos de Instancia

Los métodos de instancia se definen usando la palabra clave def dentro de una clase. Pueden realizar operaciones usando variables de instancia y otros métodos.

class Calculator
  def initialize
    @result = 0
  end

  def add(number)
    @result += number
  end

  def subtract(number)
    @result -= number
  end

  def result
    @result
  end
end

calc = Calculator.new
calc.add(10)
calc.subtract(3)
puts calc.result  # Output: 7

Ejemplo: Una Cuenta Bancaria Simple

Vamos a crear una clase de cuenta bancaria simple para demostrar variables y métodos de instancia.

class BankAccount
  def initialize(owner, balance = 0)
    @owner = owner
    @balance = balance
  end

  def deposit(amount)
    @balance += amount
    puts "Deposited #{amount}. New balance: #{@balance}"
  end

  def withdraw(amount)
    if amount > @balance
      puts "Insufficient funds!"
    else
      @balance -= amount
      puts "Withdrew #{amount}. New balance: #{@balance}"
    end
  end

  def display_balance
    puts "Account owner: #{@owner}, Balance: #{@balance}"
  end
end

account = BankAccount.new("John Doe", 100)
account.deposit(50)
account.withdraw(30)
account.display_balance

Ejercicios

  1. Crear una clase Car:

    • Define variables de instancia para make, model, y year.
    • Crea métodos de instancia para arrancar el coche, detener el coche, y mostrar los detalles del coche.

  2. Mejorar la clase Person:

    • Añade un método de instancia para actualizar la edad.
    • Añade un método de instancia para mostrar un mensaje de saludo que incluya el nombre y la edad de la persona.

  3. Construir una clase Library:

    • Define variables de instancia para name y books (un array de títulos de libros).
    • Crea métodos de instancia para añadir un libro, eliminar un libro, y listar todos los libros.

Conclusión

Las variables y métodos de instancia son cruciales para definir el estado y el comportamiento de los objetos en Ruby. Al entender cómo usarlos, puedes crear clases más complejas y funcionales. Practica creando tus propias clases y experimentando con variables y métodos de instancia para profundizar tu comprensión.
