Configuración del Entorno de Go
Tu Primer Programa en Go ►

Configurar el entorno de Go es el primer paso para comenzar a programar en Go. Esta sección te guiará a través del proceso de instalación de Go, configuración de tu espacio de trabajo y verificación de tu instalación.

Instalando Go

Para comenzar con Go, necesitas instalar el compilador y las herramientas de Go en tu máquina. Sigue estos pasos para instalar Go:

  1. Descargar Go:

    • Visita el sitio web oficial de Go: golang.org
    • Descarga el instalador para tu sistema operativo (Windows, macOS o Linux).

  2. Instalar Go:

    • Windows:
      • Ejecuta el archivo MSI descargado y sigue las indicaciones.
    • macOS:
      • Abre el archivo .pkg descargado y sigue las instrucciones de instalación.
    • Linux:
      • Extrae el tarball a /usr/local usando el siguiente comando: 
        tar -C /usr/local -xzf go1.x.x.linux-amd64.tar.gz

  3. Configurar Variables de Entorno:

    • Agrega el directorio binario de Go a tu variable de entorno PATH.
    • Windows:
      • Abre la Búsqueda de Inicio, escribe "env" y selecciona "Editar las variables de entorno del sistema".
      • Haz clic en "Variables de entorno" y encuentra la variable PATH en la sección "Variables del sistema".
      • Haz clic en "Editar" y agrega la ruta al binario de Go (por ejemplo, C:\Go\bin).
    • macOS y Linux:
      • Agrega la siguiente línea a tu archivo .bashrc, .zshrc o .profile: 
        export PATH=$PATH:/usr/local/go/bin

Configuración de Tu Espacio de Trabajo

Go utiliza una estructura de espacio de trabajo específica para gestionar el código. El espacio de trabajo es una jerarquía de directorios con tres directorios principales: src, pkg y bin.

  1. Crear Tu Espacio de Trabajo:

    • Elige una ubicación para tu espacio de trabajo, por ejemplo, ~/go.
    • Dentro de este directorio, crea los siguientes subdirectorios: 
      mkdir -p ~/go/{src,pkg,bin}

  2. Configurar la Variable de Entorno GOPATH:

    • La variable de entorno GOPATH especifica la ubicación de tu espacio de trabajo.
    • Windows:
      • Abre la Búsqueda de Inicio, escribe "env" y selecciona "Editar las variables de entorno del sistema".
      • Haz clic en "Variables de entorno" y crea una nueva variable de usuario llamada GOPATH con el valor de la ruta de tu espacio de trabajo (por ejemplo, C:\Users\TuNombre\go).
    • macOS y Linux:
      • Agrega la siguiente línea a tu archivo .bashrc, .zshrc o .profile: 
        export GOPATH=~/go

Verificación de Tu Instalación

Para asegurarte de que Go está instalado correctamente y tu entorno está configurado adecuadamente, sigue estos pasos:

  1. Verificar la Versión de Go:

    • Abre una terminal o símbolo del sistema.
    • Ejecuta el siguiente comando: 
      go version
    • Deberías ver una salida similar a go version go1.x.x <os>/<arch>.

  2. Crear un Programa Simple en Go:

    • Crea un directorio para tu proyecto dentro del directorio src de tu espacio de trabajo: 
      mkdir -p ~/go/src/hello
    • Crea un archivo llamado hello.go dentro de este directorio con el siguiente contenido: 
      package main

import "fmt"

func main() {
    fmt.Println("¡Hola, Mundo!")
}

  3. Compilar y Ejecutar el Programa:

    • Navega al directorio que contiene tu archivo Go: 
      cd ~/go/src/hello
    • Compila el programa usando el comando go build: 
      go build
    • Ejecuta el ejecutable: 
      ./hello
    • Deberías ver la salida ¡Hola, Mundo!.

Conclusión

Configurar el entorno de Go implica instalar Go, configurar tu espacio de trabajo y verificar la instalación. Siguiendo estos pasos, aseguras que tu entorno de desarrollo esté listo para programar en Go. Esta configuración fundamental te permitirá progresar sin problemas en el aprendizaje y desarrollo de aplicaciones en Go.
