Resumen

Go, también conocido como Golang, es un lenguaje de programación compilado y tipado estáticamente diseñado por Google. Es conocido por su simplicidad, eficiencia y fuerte soporte para la concurrencia. Esta sección te introducirá a los conceptos básicos de Go, incluyendo su historia, características clave y cómo configurar tu entorno de desarrollo.

Historia y Antecedentes

  • Creado por: Robert Griesemer, Rob Pike y Ken Thompson en Google.
  • Primera versión: 2009.
  • Objetivos de diseño:
    • Mejorar la productividad en la programación.
    • Proveer un lenguaje moderno con fuerte soporte para la concurrencia.
    • Asegurar una compilación y ejecución rápidas.

Características Clave de Go

  • Simplicidad: Diseño minimalista con una pequeña biblioteca estándar.
  • Concurrencia: Soporte incorporado usando goroutines y canales.
  • Rendimiento: Lenguaje compilado con ejecución eficiente.
  • Recolección de Basura: Gestión automática de memoria.
  • Tipado Fuerte: Tipado estático con inferencia de tipos.

Configuración del Entorno de Desarrollo de Go

  1. Descargar e Instalar Go:

    • Visita el sitio web oficial de Go y descarga el instalador para tu sistema operativo.
    • Sigue las instrucciones de instalación proporcionadas en el sitio web.

  2. Verificar la Instalación:

    • Abre una terminal o símbolo del sistema.
    • Ejecuta el comando go version para verificar si Go está instalado correctamente.

  3. Configurar el Espacio de Trabajo:

    • Crea un directorio para tus proyectos de Go, por ejemplo, ~/go.
    • Configura la variable de entorno GOPATH para que apunte a tu directorio de espacio de trabajo.

  4. Programa Hola Mundo:

    • Crea un nuevo directorio para tu proyecto, por ejemplo, ~/go/src/hello.

    • Crea un archivo llamado main.go con el siguiente contenido:

      package main

import "fmt"

func main() {
    fmt.Println("¡Hola, Mundo!")
}

    • Navega al directorio del proyecto y ejecuta el programa usando el comando go run main.go.

Sintaxis Básica y Estructura

  • Paquetes: Cada programa en Go está compuesto por paquetes. El paquete main es el punto de entrada del programa.
  • Importaciones: Usa la palabra clave import para incluir otros paquetes.
  • Funciones: Definidas usando la palabra clave func.

Ejemplo: Estructura Básica del Programa

package main

import "fmt"

func main() {
    fmt.Println("¡Bienvenido a Go!")
}
  • Explicación:
    • package main: Define el nombre del paquete.
    • import "fmt": Importa el paquete fmt para entrada/salida formateada.
    • func main() { ... }: La función principal donde comienza la ejecución del programa.

Variables y Tipos de Datos

  • Declaración de Variables: Usa la palabra clave var o la forma abreviada :=.
  • Tipos de Datos Básicos: int, float64, string, bool.

Ejemplo: Declaración de Variables

package main

import "fmt"

func main() {
    var a int = 10
    b := 20
    var c, d int = 30, 40
    var e, f = 50, "hola"
    fmt.Println(a, b, c, d, e, f)
}
  • Explicación:
    • var a int = 10: Declara una variable entera a con valor 10.
    • b := 20: Forma abreviada para declarar e inicializar b con valor 20.
    • var c, d int = 30, 40: Declara dos variables enteras c y d.
    • var e, f = 50, "hola": Declara e como un entero y f como una cadena.

Estructuras de Control

  • Sentencias If-Else: Ejecución condicional.
  • Bucles For: El único constructo de bucle en Go.
  • Sentencias Switch: Simplifica múltiples ramas condicionales.

Ejemplo: Estructuras de Control

package main

import "fmt"

func main() {
    // If-Else
    x := 10
    if x > 5 {
        fmt.Println("x es mayor que 5")
    } else {
        fmt.Println("x es 5 o menor")
    }

    // Bucle For
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }

    // Sentencia Switch
    day := "Lunes"
    switch day {
    case "Lunes":
        fmt.Println("Inicio de la semana laboral")
    case "Viernes":
        fmt.Println("Fin de la semana laboral")
    default:
        fmt.Println("Día entre semana")
    }
}

Conclusión

En esta sección introductoria, hemos cubierto los conceptos básicos de Go, incluyendo su historia, características clave y cómo configurar tu entorno de desarrollo. También hemos explorado la sintaxis y estructura básica de los programas en Go, la declaración de variables y las estructuras de control. Con esta base, ahora estás listo para profundizar en la programación en Go y explorar temas más avanzados en secciones posteriores.
