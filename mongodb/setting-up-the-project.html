Configurar un proyecto de MongoDB es el primer paso para construir aplicaciones que aprovechen las potentes capacidades de la base de datos NoSQL de MongoDB. Esta sección te guiará a través del proceso de instalación de MongoDB, configuración de tu entorno de desarrollo y creación de tu primer proyecto MongoDB.

Instalación de MongoDB

Antes de que puedas empezar a usar MongoDB, necesitas instalarlo en tu máquina. Sigue estos pasos para instalar MongoDB en diferentes sistemas operativos:

Windows

  1. Descargar MongoDB:

    • Visita el Centro de Descargas de MongoDB y descarga el instalador para Windows.

  2. Ejecutar el Instalador:

    • Haz doble clic en el archivo .msi descargado y sigue las instrucciones de instalación.
    • Elige el tipo de configuración "Completa".

  3. Configurar la Variable de Entorno:

    • Añade el directorio bin de MongoDB (C:\Program Files\MongoDB\Server\<version>\bin) a la variable de entorno PATH de tu sistema.

  4. Verificar la Instalación:

    • Abre el Símbolo del sistema y escribe mongo --version para verificar la instalación.

macOS

  1. Instalar Homebrew (si no está ya instalado):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instalar MongoDB:

    brew tap mongodb/brew
brew install mongodb-community@<version>

  3. Iniciar MongoDB:

    brew services start mongodb/brew/mongodb-community

  4. Verificar la Instalación:

    • Abre Terminal y escribe mongo --version para verificar la instalación.

Linux

  1. Importar la Clave Pública:

    wget -qO - https://www.mongodb.org/static/pgp/server-<version>.asc | sudo apt-key add -

  2. Crear un Archivo de Lista:

    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/<version> multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-<version>.list

  3. Instalar MongoDB:

    sudo apt-get update
sudo apt-get install -y mongodb-org

  4. Iniciar MongoDB:

    sudo systemctl start mongod

  5. Verificar la Instalación:

    • Abre Terminal y escribe mongo --version para verificar la instalación.

Configuración del Entorno de Desarrollo

Una vez que MongoDB esté instalado, necesitas configurar tu entorno de desarrollo. Esto implica elegir un lenguaje de programación, instalar los controladores necesarios y configurar un Entorno de Desarrollo Integrado (IDE).

Elegir un Lenguaje de Programación

MongoDB soporta varios lenguajes de programación. Algunas opciones populares incluyen:

  • JavaScript (Node.js)
  • Python
  • Java
  • C#

Para este curso, usaremos Node.js debido a su popularidad y facilidad de uso.

Instalación de Node.js y npm

  1. Descargar Node.js:

    • Visita el sitio web de Node.js y descarga la versión LTS.

  2. Instalar Node.js:

    • Sigue las instrucciones de instalación para tu sistema operativo.

  3. Verificar la Instalación:

    node --version
npm --version

Configuración de un IDE

Elige un IDE que soporte desarrollo en JavaScript. Algunas opciones populares incluyen:

  • Visual Studio Code
  • WebStorm
  • Atom

Para este curso, usaremos Visual Studio Code.

  1. Descargar Visual Studio Code:

    • Visita el sitio web de Visual Studio Code y descarga el instalador.

  2. Instalar Visual Studio Code:

    • Sigue las instrucciones de instalación para tu sistema operativo.

Creación de tu Primer Proyecto MongoDB

Ahora que tu entorno de desarrollo está configurado, vamos a crear tu primer proyecto MongoDB.

Inicializar un Proyecto Node.js

  1. Crear un Directorio de Proyecto:

    mkdir my-mongodb-project
cd my-mongodb-project

  2. Inicializar el Proyecto:

    npm init -y

Instalar el Controlador de MongoDB

  1. Instalar el Controlador de MongoDB: 
    npm install mongodb

Conectar a MongoDB

Crea un archivo llamado app.js y añade el siguiente código para conectarte a MongoDB:

const { MongoClient } = require('mongodb');

// URL de Conexión
const url = 'mongodb://localhost:27017';
const client = new MongoClient(url);

// Nombre de la Base de Datos
const dbName = 'myDatabase';

async function main() {
  // Usa el método connect para conectarte al servidor
  await client.connect();
  console.log('Conectado exitosamente al servidor');
  const db = client.db(dbName);

  // Cierra la conexión
  await client.close();
}

main().catch(console.error);

Ejecutar el Proyecto

  1. Ejecutar la Aplicación:

    node app.js

  2. Verificar la Conexión:

    • Deberías ver el mensaje "Conectado exitosamente al servidor" en la consola.

Conclusión

En esta sección, cubrimos los pasos esenciales para configurar un proyecto MongoDB. Aprendiste cómo instalar MongoDB en diferentes sistemas operativos, configurar tu entorno de desarrollo con Node.js y Visual Studio Code, y crear una simple aplicación Node.js que se conecta a una base de datos MongoDB. Con tu entorno listo, ahora estás preparado para profundizar en MongoDB y comenzar a construir aplicaciones poderosas.
