Proyecto Final

Introducción

El Proyecto Final es una parte integral del curso de Gestión de Proyectos. Este proyecto tiene como objetivo consolidar los conocimientos adquiridos a lo largo del curso mediante la aplicación práctica de las metodologías y herramientas de gestión de proyectos. Los estudiantes trabajarán en equipos para planificar, ejecutar y cerrar un proyecto simulado, siguiendo las mejores prácticas de la industria.

Objetivos del Proyecto Final

  • Aplicar los conceptos y técnicas de gestión de proyectos aprendidos durante el curso.
  • Desarrollar habilidades de trabajo en equipo y comunicación efectiva.
  • Gestionar un proyecto de principio a fin, incluyendo la planificación, ejecución, monitoreo y cierre.
  • Utilizar herramientas de gestión de proyectos para documentar y controlar el progreso del proyecto.

Descripción del Proyecto

El proyecto consiste en la creación de una aplicación web para la gestión de tareas. Los estudiantes deberán seguir todas las fases de un ciclo de vida de proyecto, desde la iniciación hasta el cierre. A continuación se detallan las fases y los entregables esperados:

Fase de Iniciación

  • Definición del Alcance: Describir el objetivo del proyecto, los entregables principales y los criterios de éxito.
  • Identificación de Stakeholders: Listar a todas las partes interesadas y sus roles en el proyecto.
  • Acta de Constitución del Proyecto: Crear un documento formal que autorice el proyecto y describa sus objetivos y restricciones.

Fase de Planificación

  • Plan de Gestión del Proyecto: Desarrollar un plan detallado que incluya el cronograma, el presupuesto, los recursos y los riesgos.
  • Estructura de Desglose del Trabajo (EDT): Crear una EDT para descomponer el proyecto en tareas más manejables.
  • Cronograma del Proyecto: Utilizar herramientas como Gantt o PERT para planificar el tiempo de cada tarea.

Fase de Ejecución

  • Desarrollo de la Aplicación: Implementar la aplicación web siguiendo las especificaciones del proyecto.
  • Gestión del Equipo: Asignar tareas y responsabilidades a los miembros del equipo.
  • Comunicación: Mantener una comunicación efectiva entre los miembros del equipo y los stakeholders.

Fase de Monitoreo y Control

  • Seguimiento del Progreso: Utilizar herramientas de gestión de proyectos para monitorear el avance del proyecto.
  • Gestión de Riesgos: Identificar y mitigar riesgos que puedan afectar el proyecto.
  • Control de Calidad: Asegurar que los entregables cumplen con los estándares de calidad establecidos.

Fase de Cierre

  • Entrega de la Aplicación: Presentar la aplicación web finalizada a los stakeholders.
  • Documentación Final: Compilar toda la documentación del proyecto, incluyendo el informe final y las lecciones aprendidas.
  • Evaluación del Proyecto: Realizar una retrospectiva para evaluar el desempeño del equipo y del proyecto en general.

Ejemplo de Código

A continuación se presenta un ejemplo de código para una parte de la aplicación web de gestión de tareas. Este ejemplo muestra cómo crear una tarea utilizando un formulario en HTML y procesarla con JavaScript.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Gestión de Tareas</title>
</head>
<body>
    <h1>Crear Nueva Tarea</h1>
    <form id="taskForm">
        <label for="taskName">Nombre de la Tarea:</label>
        <input type="text" id="taskName" name="taskName" required>
        <br>
        <label for="taskDescription">Descripción:</label>
        <textarea id="taskDescription" name="taskDescription" required></textarea>
        <br>
        <button type="submit">Agregar Tarea</button>
    </form>

    <script>
        document.getElementById('taskForm').addEventListener('submit', function(event) {
            event.preventDefault();
            const taskName = document.getElementById('taskName').value;
            const taskDescription = document.getElementById('taskDescription').value;
            console.log(`Tarea creada: ${taskName} - ${taskDescription}`);
            // Aquí se podría agregar la lógica para guardar la tarea en una base de datos
        });
    </script>
</body>
</html>

Conclusión

El Proyecto Final es una excelente oportunidad para aplicar los conocimientos adquiridos en un entorno práctico y colaborativo. A través de este proyecto, los estudiantes no solo reforzarán sus habilidades técnicas y de gestión, sino que también desarrollarán competencias clave como el trabajo en equipo y la comunicación efectiva. Al finalizar el proyecto, los estudiantes estarán mejor preparados para enfrentar desafíos reales en la gestión de proyectos.
