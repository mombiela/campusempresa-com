Autenticación de Usuarios en Flask
Implementación de Inicio y Cierre de Sesión ►

Introducción

La autenticación de usuarios es un aspecto crucial del desarrollo web, asegurando que los usuarios puedan iniciar sesión de manera segura y acceder a sus datos. En esta sección, cubriremos los conceptos básicos de la autenticación de usuarios en Flask, progresando desde conceptos para principiantes hasta avanzados.

Configuración de Flask

Antes de profundizar en la autenticación de usuarios, asegúrate de tener Flask instalado y configurado.

pip install Flask

Crea una aplicación básica de Flask:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Autenticación Básica de Usuarios

Creación de un Modelo de Usuario

Para manejar los datos de los usuarios, crearemos un modelo de Usuario. Para simplificar, usaremos un diccionario para almacenar la información del usuario.

users = {
    "user1": {"username": "user1", "password": "password1"},
    "user2": {"username": "user2", "password": "password2"}
}

Ruta de Inicio de Sesión

Crea una ruta para manejar el inicio de sesión del usuario.

from flask import request, jsonify

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        return jsonify({"message": "¡Inicio de sesión exitoso!"}), 200
    return jsonify({"message": "Credenciales inválidas"}), 401

Protección de Rutas

Para proteger ciertas rutas, podemos usar un enfoque simple basado en sesiones.

from flask import session

app.secret_key = 'supersecretkey'

@app.route('/protected')
def protected():
    if 'username' in session:
        return f"¡Hola, {session['username']}!"
    return "¡No has iniciado sesión!", 401

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        session['username'] = username
        return jsonify({"message": "¡Inicio de sesión exitoso!"}), 200
    return jsonify({"message": "Credenciales inválidas"}), 401

@app.route('/logout')
def logout():
    session.pop('username', None)
    return "¡Sesión cerrada!"

Autenticación Intermedia de Usuarios

Uso de Flask-Login

Flask-Login proporciona gestión de sesiones de usuario para Flask. Instala Flask-Login:

pip install Flask-Login

Configuración de Flask-Login

Integra Flask-Login en tu aplicación.

from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin):
    def __init__(self, id):
        self.id = id

@login_manager.user_loader
def load_user(user_id):
    return User(user_id)

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "¡Inicio de sesión exitoso!"}), 200
    return jsonify({"message": "Credenciales inválidas"}), 401

@app.route('/protected')
@login_required
def protected():
    return f"¡Hola, {current_user.id}!"

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return "¡Sesión cerrada!"

Autenticación Avanzada de Usuarios

Hashing de Contraseñas

Almacenar contraseñas en texto plano es inseguro. Usa una biblioteca de hashing como werkzeug.security para hashear contraseñas.

from werkzeug.security import generate_password_hash, check_password_hash

# Hashing de contraseñas
users = {
    "user1": {"username": "user1", "password": generate_password_hash("password1")},
    "user2": {"username": "user2", "password": generate_password_hash("password2")}
}

# Verificación de contraseñas
@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and check_password_hash(user['password'], password):
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "¡Inicio de sesión exitoso!"}), 200
    return jsonify({"message": "Credenciales inválidas"}), 401

Uso de Flask-SQLAlchemy

Para una solución más robusta, integra Flask-SQLAlchemy para gestionar los datos de los usuarios en una base de datos.

pip install Flask-SQLAlchemy
from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///users.db'
db = SQLAlchemy(app)

class User(db.Model, UserMixin):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

# Crear la base de datos
db.create_all()

# Añadir un usuario
new_user = User(username='user1', password=generate_password_hash('password1'))
db.session.add(new_user)
db.session.commit()

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = User.query.filter_by(username=username).first()
    if user and check_password_hash(user.password, password):
        login_user(user)
        return jsonify({"message": "¡Inicio de sesión exitoso!"}), 200
    return jsonify({"message": "Credenciales inválidas"}), 401

Conclusión

La autenticación de usuarios es un aspecto fundamental del desarrollo web.
