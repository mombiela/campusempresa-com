Introducción

En esta sección, exploraremos cómo realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) usando Flask. Las operaciones CRUD son fundamentales para las aplicaciones web, permitiendo a los usuarios interactuar con la base de datos. Cubriremos cada operación paso a paso, proporcionando ejemplos de código y explicaciones para asegurarnos de que entiendas cómo implementarlas en tu aplicación Flask.

Configuración del Entorno

Antes de sumergirnos en las operaciones CRUD, configuremos nuestro entorno Flask.

  1. Instalar Flask y SQLAlchemy:

    pip install Flask SQLAlchemy

  2. Crear una aplicación básica de Flask:

    from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///test.db'
db = SQLAlchemy(app)

if __name__ == '__main__':
    app.run(debug=True)

Creación del Modelo de Base de Datos

Define un modelo para la base de datos. Para este ejemplo, crearemos un modelo simple User.

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.name}>'

Ejecuta los siguientes comandos para crear la base de datos:

from your_application import db
db.create_all()

Operación de Creación

Para agregar nuevos usuarios a la base de datos, necesitamos crear una ruta que maneje la creación de nuevas instancias de User.

@app.route('/user', methods=['POST'])
def create_user():
    name = request.json['name']
    email = request.json['email']
    new_user = User(name=name, email=email)

    try:
        db.session.add(new_user)
        db.session.commit()
        return {"message": "Usuario creado exitosamente"}, 201
    except:
        return {"message": "Hubo un problema al crear el usuario"}, 500

Operación de Lectura

Para leer o recuperar usuarios de la base de datos, podemos crear rutas para obtener todos los usuarios o un usuario específico por ID.

  • Obtener todos los usuarios:

    @app.route('/users', methods=['GET'])
def get_users():
    users = User.query.all()
    return {"users": [{"id": user.id, "name": user.name, "email": user.email} for user in users]}

  • Obtener usuario por ID:

    @app.route('/user/<int:id>', methods=['GET'])
def get_user(id):
    user = User.query.get_or_404(id)
    return {"id": user.id, "name": user.name, "email": user.email}

Operación de Actualización

Para actualizar un usuario existente, necesitamos crear una ruta que maneje la lógica de actualización.

@app.route('/user/<int:id>', methods=['PUT'])
def update_user(id):
    user = User.query.get_or_404(id)
    name = request.json['name']
    email = request.json['email']

    try:
        user.name = name
        user.email = email
        db.session.commit()
        return {"message": "Usuario actualizado exitosamente"}
    except:
        return {"message": "Hubo un problema al actualizar el usuario"}, 500

Operación de Eliminación

Para eliminar un usuario de la base de datos, necesitamos crear una ruta que maneje la eliminación.

@app.route('/user/<int:id>', methods=['DELETE'])
def delete_user(id):
    user = User.query.get_or_404(id)

    try:
        db.session.delete(user)
        db.session.commit()
        return {"message": "Usuario eliminado exitosamente"}
    except:
        return {"message": "Hubo un problema al eliminar el usuario"}, 500

Conclusión

En esta sección, cubrimos cómo realizar operaciones CRUD en una aplicación Flask usando SQLAlchemy. Aprendimos cómo:

  • Configurar el entorno Flask y la base de datos.
  • Crear un modelo de base de datos.
  • Implementar rutas para crear, leer, actualizar y eliminar registros.

Entender las operaciones CRUD es esencial para construir aplicaciones web dinámicas. Practica estos conceptos para solidificar tu comprensión y aplicarlos a tus proyectos.
