Implementación de Inicio y Cierre de Sesión

En esta sección, cubriremos cómo implementar la autenticación de usuarios en una aplicación Flask. Esto incluye crear un sistema de inicio y cierre de sesión. Comenzaremos con lo básico y gradualmente avanzaremos a temas más avanzados.

Configuración de Flask y Extensiones

Antes de sumergirnos en la implementación, asegúrate de tener Flask y las extensiones necesarias instaladas. Puedes instalarlas usando pip:

pip install Flask Flask-Login Flask-WTF
  • Flask: El marco principal.
  • Flask-Login: Gestiona las sesiones de usuario.
  • Flask-WTF: Simplifica el manejo de formularios.

Estructura Básica del Proyecto

Aquí hay una estructura básica para nuestra aplicación Flask:

/flask_auth
    /static
    /templates
    app.py
    config.py
    models.py
  • app.py: Archivo principal de la aplicación.
  • config.py: Configuración de ajustes.
  • models.py: Modelos de base de datos.

Configuración de la Aplicación

Primero, configuremos la configuración en config.py:

# config.py
import os

class Config:
    SECRET_KEY = os.environ.get('SECRET_KEY') or 'you-will-never-guess'
    SQLALCHEMY_DATABASE_URI = os.environ.get('DATABASE_URL') or 'sqlite:///site.db'
    SQLALCHEMY_TRACK_MODIFICATIONS = False

Creación del Modelo de Usuario

En models.py, define el modelo de Usuario:

# models.py
from flask_sqlalchemy import SQLAlchemy
from flask_login import UserMixin

db = SQLAlchemy()

class User(db.Model, UserMixin):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    email = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

Inicialización de Flask y Extensiones

En app.py, inicializa Flask y las extensiones:

# app.py
from flask import Flask
from config import Config
from models import db, User
from flask_login import LoginManager

app = Flask(__name__)
app.config.from_object(Config)

db.init_app(app)
login = LoginManager(app)
login.login_view = 'login'

@login.user_loader
def load_user(user_id):
    return User.query.get(int(user_id))

if __name__ == '__main__':
    app.run(debug=True)

Creación de Formularios

Crea formularios para inicio de sesión y registro usando Flask-WTF en forms.py:

# forms.py
from flask_wtf import FlaskForm
from wtforms import StringField, PasswordField, SubmitField
from wtforms.validators import DataRequired, Email, EqualTo

class LoginForm(FlaskForm):
    email = StringField('Email', validators=[DataRequired(), Email()])
    password = PasswordField('Password', validators=[DataRequired()])
    submit = SubmitField('Login')

class RegistrationForm(FlaskForm):
    username = StringField('Username', validators=[DataRequired()])
    email = StringField('Email', validators=[DataRequired(), Email()])
    password = PasswordField('Password', validators=[DataRequired()])
    confirm_password = PasswordField('Confirm Password', validators=[DataRequired(), EqualTo('password')])
    submit = SubmitField('Register')

Implementación de Rutas

Agrega rutas para inicio de sesión, cierre de sesión y registro en app.py:

# app.py (continuación)
from flask import render_template, redirect, url_for, flash
from forms import LoginForm, RegistrationForm
from flask_login import login_user, logout_user, login_required

@app.route('/login', methods=['GET', 'POST'])
def login():
    form = LoginForm()
    if form.validate_on_submit():
        user = User.query.filter_by(email=form.email.data).first()
        if user and user.password == form.password.data:
            login_user(user)
            return redirect(url_for('index'))
        else:
            flash('Inicio de sesión fallido. Por favor, verifica el correo electrónico y la contraseña', 'danger')
    return render_template('login.html', title='Login', form=form)

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return redirect(url_for('index'))

@app.route('/register', methods=['GET', 'POST'])
def register():
    form = RegistrationForm()
    if form.validate_on_submit():
        user = User(username=form.username.data, email=form.email.data, password=form.password.data)
        db.session.add(user)
        db.session.commit()
        flash('¡Cuenta creada para {form.username.data}!', 'success')
        return redirect(url_for('login'))
    return render_template('register.html', title='Register', form=form)

Conclusión

En esta sección, hemos cubierto los conceptos básicos de la implementación de la funcionalidad de inicio y cierre de sesión en una aplicación Flask. Comenzamos configurando la estructura del proyecto, configurando la aplicación, creando el modelo de usuario e inicializando Flask y sus extensiones. Luego, creamos formularios para inicio de sesión y registro, implementamos las rutas necesarias y creamos plantillas HTML para los formularios.

Siguiendo estos pasos, ahora deberías tener una comprensión básica de cómo implementar la autenticación de usuarios en Flask. A medida que progreses, puedes mejorar este sistema agregando características como el hash de contraseñas, la verificación de correo electrónico y funcionalidades más avanzadas de gestión de usuarios.
