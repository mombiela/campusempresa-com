Introducción a los Formularios en Flask

Los formularios son una parte fundamental de las aplicaciones web, permitiendo a los usuarios interactuar con la aplicación enviando datos. Flask proporciona varias formas de manejar formularios, desde formularios HTML básicos hasta un manejo más avanzado de formularios utilizando Flask-WTF.

Formularios HTML Básicos

Para comenzar con los formularios en Flask, puedes usar formularios HTML simples. Aquí tienes un ejemplo sencillo:

<!-- templates/form.html -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario Simple</title>
</head>
<body>
    <form action="/submit" method="post">
        <label for="name">Nombre:</label>
        <input type="text" id="name" name="name">
        <input type="submit" value="Enviar">
    </form>
</body>
</html>

Manejo del Envío del Formulario

Para manejar el envío del formulario, necesitas crear una ruta en tu aplicación Flask:

# app.py
from flask import Flask, request, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('form.html')

@app.route('/submit', methods=['POST'])
def submit():
    name = request.form['name']
    return f'¡Hola, {name}!'

if __name__ == '__main__':
    app.run(debug=True)
  • request.form: Un objeto diccionario que contiene los datos del formulario.
  • methods=['POST']: Especifica que la ruta debe manejar solicitudes POST.

Usando Flask-WTF para Manejo de Formularios

Flask-WTF es una extensión que integra Flask con WTForms, proporcionando una forma más robusta de manejar formularios.

Instalación

Primero, instala Flask-WTF:

pip install Flask-WTF

Creando una Clase de Formulario

Define una clase de formulario usando Flask-WTF:

# forms.py
from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField
from wtforms.validators import DataRequired

class NameForm(FlaskForm):
    name = StringField('Nombre', validators=[DataRequired()])
    submit = SubmitField('Enviar')

Renderizando el Formulario

Actualiza tu ruta para usar la clase de formulario:

# app.py
from flask import Flask, render_template, redirect, url_for
from forms import NameForm

app = Flask(__name__)
app.config['SECRET_KEY'] = 'your_secret_key'

@app.route('/', methods=['GET', 'POST'])
def index():
    form = NameForm()
    if form.validate_on_submit():
        name = form.name.data
        return redirect(url_for('greet', name=name))
    return render_template('form_wtf.html', form=form)

@app.route('/greet/<name>')
def greet(name):
    return f'¡Hola, {name}!'

if __name__ == '__main__':
    app.run(debug=True)

Plantilla para el Formulario Flask-WTF

Crea una plantilla para renderizar el formulario:

<!-- templates/form_wtf.html -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Formulario Flask-WTF</title>
</head>
<body>
    <form method="post">
        {{ form.hidden_tag() }}
        <p>
            {{ form.name.label }}<br>
            {{ form.name(size=32) }}<br>
            {% for error in form.name.errors %}
                <span style="color: red;">[{{ error }}]</span>
            {% endfor %}
        </p>
        <p>{{ form.submit() }}</p>
    </form>
</body>
</html>
  • form.hidden_tag(): Genera campos ocultos para la protección CSRF.
  • form.name(size=32): Renderiza el campo de entrada con un tamaño especificado.
  • form.submit(): Renderiza el botón de envío.

Manejo Avanzado de Formularios

Carga de Archivos

Manejar la carga de archivos requiere una configuración adicional. Aquí tienes un ejemplo:

# forms.py
from flask_wtf.file import FileField, FileRequired

class UploadForm(FlaskForm):
    file = FileField('Archivo', validators=[FileRequired()])
    submit = SubmitField('Subir')
# app.py
from flask import Flask, render_template, request
from forms import UploadForm
import os

app = Flask(__name__)
app.config['SECRET_KEY'] = 'your_secret_key'
app.config['UPLOAD_FOLDER'] = 'uploads/'

@app.route('/upload', methods=['GET', 'POST'])
def upload():
    form = UploadForm()
    if form.validate_on_submit():
        file = form.file.data
        file.save(os.path.join(app.config['UPLOAD_FOLDER'], file.filename))
        return '¡Archivo subido exitosamente!'
    return render_template('upload.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)

Plantilla para el Formulario de Carga de Archivos

<!-- templates/upload.html -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Carga de Archivos</title>
</head>
<body>
    <form method="post" enctype="multipart/form-data">
        {{ form.hidden_tag() }}
        <p>
            {{ form.file.label }}<br>
            {{ form.file() }}<br>
            {% for error in form.file.errors %}
                <span style="color: red;">[{{ error }}]</span>
            {% endfor %}
        </p>
        <p>{{ form.submit() }}</p>
    </form>
</body>
</html>

Conclusión

Manejar formularios en Flask puede ir desde formularios HTML simples hasta formularios más complejos usando Flask-WTF. Entender ambos métodos te permite elegir el mejor enfoque para las necesidades de tu aplicación. Flask-WTF proporciona una forma más estructurada y segura de manejar formularios, incluyendo características como protección CSRF y validación de formularios.

Al dominar el manejo de formularios en Flask, puedes crear aplicaciones web interactivas y amigables para el usuario que capturen y procesen efectivamente la entrada del usuario.
