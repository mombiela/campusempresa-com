Introducción

Las cadenas son un tipo de dato fundamental en PowerShell, utilizado para representar texto. Esta sección cubrirá los conceptos básicos de trabajar con cadenas, incluyendo la creación, manipulación y formateo de cadenas.

Creación de Cadenas

En PowerShell, las cadenas pueden ser creadas usando comillas simples (') o comillas dobles (").

Cadenas con Comillas Simples

Las cadenas con comillas simples son cadenas literales. Las variables y las secuencias de escape no son evaluadas.

$singleQuotedString = 'Esta es una cadena con comillas simples: $variable'
Write-Output $singleQuotedString

Cadenas con Comillas Dobles

Las cadenas con comillas dobles permiten la interpolación de variables y secuencias de escape.

$variable = "PowerShell"
$doubleQuotedString = "Esta es una cadena con comillas dobles: $variable"
Write-Output $doubleQuotedString

Manipulación de Cadenas

PowerShell proporciona varios métodos y operadores para manipular cadenas.

Concatenación

Las cadenas pueden ser concatenadas usando el operador + o simplemente colocándolas una al lado de la otra.

$string1 = "Hola"
$string2 = "Mundo"
$concatenatedString = $string1 + " " + $string2
Write-Output $concatenatedString

Subcadena

Extrae una parte de una cadena usando el método Substring.

$fullString = "PowerShell"
$subString = $fullString.Substring(0, 5)
Write-Output $subString

Longitud de la Cadena

Obtén la longitud de una cadena usando la propiedad Length.

$string = "PowerShell"
$length = $string.Length
Write-Output $length

Reemplazar

Reemplaza ocurrencias de una subcadena con otra subcadena usando el método Replace.

$string = "Hola, Mundo"
$replacedString = $string.Replace("Mundo", "PowerShell")
Write-Output $replacedString

Dividir

Divide una cadena en un array de subcadenas basado en un delimitador.

$string = "manzana,banana,cereza"
$array = $string.Split(',')
Write-Output $array

Formateo de Cadenas

PowerShell soporta varias formas de formatear cadenas.

Formateo Compuesto

Usa marcadores de posición y el operador -f para el formateo compuesto.

$name = "Juan"
$age = 30
$formattedString = "Nombre: {0}, Edad: {1}" -f $name, $age
Write-Output $formattedString

Interpolación de Cadenas

Usa cadenas con comillas dobles para la interpolación de cadenas.

$name = "Juan"
$age = 30
$interpolatedString = "Nombre: $name, Edad: $age"
Write-Output $interpolatedString

Operaciones Avanzadas con Cadenas

Para operaciones más avanzadas con cadenas, PowerShell proporciona cmdlets y métodos adicionales.

Expresiones Regulares

Usa expresiones regulares para la coincidencia de patrones y manipulación de texto.

$string = "La lluvia en España"
$pattern = "\b\w*ain\b"
if ($string -match $pattern) {
    Write-Output "Coincidencia encontrada: $($matches[0])"
}

Cadenas Multilínea

Crea cadenas multilínea usando símbolos @.

$multilineString = @"
Esta es una
cadena multilínea
en PowerShell.
"@
Write-Output $multilineString

Here-Strings

Los here-strings pueden ser usados para cadenas multilínea y preservan el formato.

$hereString = @"
Este es un here-string.
Preserva el formato
    y la indentación.
"@
Write-Output $hereString

Conclusión

Trabajar con cadenas en PowerShell es esencial para tareas de scripting y automatización. Esta sección cubrió los conceptos básicos de creación, manipulación y formateo de cadenas, así como algunas operaciones avanzadas. Al dominar estos conceptos, podrás manejar datos de texto de manera más efectiva en tus scripts de PowerShell.
