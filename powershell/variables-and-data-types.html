Variables y Tipos de Datos en PowerShell
Operadores en PowerShell ►

Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en PowerShell. Entender estos conceptos fundamentales es crucial para escribir scripts efectivos y automatizar tareas.

Variables en PowerShell

Las variables en PowerShell se utilizan para almacenar datos que pueden ser referenciados y manipulados a lo largo de tus scripts.

Declaración de Variables

  • Las variables en PowerShell se declaran usando el símbolo $ seguido del nombre de la variable.
  • Los nombres de las variables no distinguen entre mayúsculas y minúsculas.
  • No necesitas especificar un tipo de dato al declarar una variable.
# Declarando una variable
$miVariable = "¡Hola, PowerShell!"

Uso de Variables

  • Las variables pueden almacenar diferentes tipos de datos como cadenas, enteros, arreglos y objetos.
  • Puedes referenciar una variable usando su nombre.
# Usando una variable
$saludo = "¡Hola, Mundo!"
Write-Output $saludo

Alcance de las Variables

  • Las variables en PowerShell tienen diferentes alcances: Global, Local, Script y Privado.
  • Por defecto, las variables son locales al alcance en el que se crean.
# Ejemplo de alcance de variables
$global:varGlobal = "Soy global"
$varLocal = "Soy local"

Tipos de Datos en PowerShell

PowerShell soporta una variedad de tipos de datos, los cuales pueden ser categorizados en tipos simples y complejos.

Tipos de Datos Simples

  • Cadena: Una secuencia de caracteres.
  • Entero: Números enteros.
  • Booleano: Representa True o False.
  • Flotante/Doble: Números con puntos decimales.
# Ejemplos de tipos de datos simples
$cadenaVar = "Esto es una cadena"
$enteroVar = 42
$booleanoVar = $true
$flotanteVar = 3.14

Tipos de Datos Complejos

  • Arreglo: Una colección de valores.
  • Tabla Hash: Una colección de pares clave-valor.
  • Objeto: Instancias de clases .NET.
# Ejemplos de tipos de datos complejos
$arregloVar = @(1, 2, 3, 4, 5)
$tablaHashVar = @{Nombre="Juan"; Edad=30}
$objetoVar = New-Object -TypeName PSObject -Property @{Nombre="Ana"; Edad=25}

Conversión de Tipos

  • PowerShell puede convertir automáticamente entre tipos de datos cuando sea necesario.
  • La conversión explícita de tipos se puede hacer usando casting.
# Conversión automática de tipos
$numero = "123"
$suma = $numero + 10  # Convierte cadena a entero

# Conversión explícita de tipos
$enteroExplicito = [int]"456"

Ejemplos y Ejercicios

Ejemplo 1: Uso Básico de Variables

# Declarar y usar una variable
$mensaje = "¡Bienvenido a PowerShell!"
Write-Output $mensaje

Ejemplo 2: Trabajando con Arreglos

# Declarar un arreglo y acceder a sus elementos
$numeros = @(10, 20, 30, 40, 50)
Write-Output $numeros[2]  # Muestra 30

Ejercicio 1: Crear y Manipular Variables

  1. Declara una variable para almacenar tu nombre.
  2. Declara una variable para almacenar tu edad.
  3. Muestra un mensaje que incluya ambas variables.
# Solución
$nombre = "Alicia"
$edad = 28
Write-Output "Mi nombre es $nombre y tengo $edad años."

Ejercicio 2: Trabajar con Tablas Hash

  1. Crea una tabla hash para almacenar información sobre un libro (título, autor, año).
  2. Muestra el título y el autor del libro.
# Solución
$libro = @{Título="1984"; Autor="George Orwell"; Año=1949}
Write-Output "El libro '$($libro.Título)' fue escrito por $($libro.Autor)."

Conclusión

En esta sección, hemos cubierto los conceptos básicos de variables y tipos de datos en PowerShell. Aprendiste cómo declarar y usar variables, entender diferentes tipos de datos y realizar conversiones de tipos. Estas habilidades fundamentales son esenciales para escribir scripts efectivos en PowerShell. Practica los ejercicios proporcionados para reforzar tu comprensión, y estarás en buen camino para dominar la escritura de scripts en PowerShell.
