Introducción

PowerShell proporciona un conjunto robusto de cmdlets para gestionar archivos y directorios. Estos cmdlets te permiten realizar una variedad de tareas como crear, copiar, mover y eliminar archivos y directorios. Esta sección te guiará a través de los cmdlets esenciales para la gestión de archivos y directorios, desde el uso básico hasta el avanzado.

Cmdlets Básicos para la Gestión de Archivos y Directorios

Get-ChildItem

El cmdlet Get-ChildItem recupera los elementos y elementos secundarios en una o más ubicaciones especificadas. Es similar al comando dir en el Símbolo del sistema.

# Listar todos los archivos y directorios en el directorio actual
Get-ChildItem

# Listar todos los archivos y directorios en una ruta especificada
Get-ChildItem -Path "C:\Example\Path"

New-Item

El cmdlet New-Item crea un nuevo elemento, como un archivo o directorio.

# Crear un nuevo directorio
New-Item -Path "C:\Example\NewDirectory" -ItemType Directory

# Crear un nuevo archivo
New-Item -Path "C:\Example\NewFile.txt" -ItemType File

Remove-Item

El cmdlet Remove-Item elimina uno o más elementos.

# Eliminar un archivo
Remove-Item -Path "C:\Example\NewFile.txt"

# Eliminar un directorio y su contenido
Remove-Item -Path "C:\Example\NewDirectory" -Recurse

Copy-Item

El cmdlet Copy-Item copia un elemento de una ubicación a otra.

# Copiar un archivo a una nueva ubicación
Copy-Item -Path "C:\Example\SourceFile.txt" -Destination "C:\Example\DestinationFile.txt"

# Copiar un directorio y su contenido a una nueva ubicación
Copy-Item -Path "C:\Example\SourceDirectory" -Destination "C:\Example\DestinationDirectory" -Recurse

Move-Item

El cmdlet Move-Item mueve un elemento de una ubicación a otra.

# Mover un archivo a una nueva ubicación
Move-Item -Path "C:\Example\SourceFile.txt" -Destination "C:\Example\DestinationFile.txt"

# Mover un directorio a una nueva ubicación
Move-Item -Path "C:\Example\SourceDirectory" -Destination "C:\Example\DestinationDirectory"

Cmdlets y Técnicas Avanzadas

Get-Content y Set-Content

El cmdlet Get-Content recupera el contenido de un archivo, mientras que el cmdlet Set-Content escribe o reemplaza el contenido en un archivo.

# Recuperar el contenido de un archivo
Get-Content -Path "C:\Example\File.txt"

# Escribir contenido en un archivo
Set-Content -Path "C:\Example\File.txt" -Value "Este es un nuevo contenido."

Add-Content

El cmdlet Add-Content agrega contenido a un archivo sin sobrescribir el contenido existente.

# Agregar contenido a un archivo
Add-Content -Path "C:\Example\File.txt" -Value "Este es contenido adicional."

Get-Item y Set-Item

El cmdlet Get-Item recupera las propiedades de un elemento especificado, mientras que el cmdlet Set-Item cambia el valor de un elemento.

# Recuperar propiedades de un archivo
Get-Item -Path "C:\Example\File.txt"

# Cambiar el valor de un elemento (por ejemplo, renombrar un archivo)
Set-Item -Path "C:\Example\File.txt" -NewName "NewFileName.txt"

Uso de Filtros y Comodines

Los cmdlets de PowerShell admiten filtros y comodines para refinar la selección de elementos.

# Listar todos los archivos .txt en un directorio
Get-ChildItem -Path "C:\Example" -Filter "*.txt"

# Listar todos los archivos que comienzan con 'Report'
Get-ChildItem -Path "C:\Example" -Filter "Report*"

Resumen

En esta sección, cubrimos los cmdlets esenciales para gestionar archivos y directorios en PowerShell. Comenzamos con operaciones básicas como listar, crear, copiar, mover y eliminar archivos y directorios. Luego exploramos cmdlets y técnicas más avanzadas, incluyendo recuperar y establecer contenido, agregar contenido y usar filtros y comodines.

Al dominar estos cmdlets, puedes gestionar eficientemente archivos y directorios en tus scripts de PowerShell, automatizando tareas rutinarias y mejorando tu productividad.
