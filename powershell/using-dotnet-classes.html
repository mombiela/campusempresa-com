Introducción

PowerShell está construido sobre el framework .NET, lo que significa que tiene acceso a una vasta biblioteca de clases .NET. Comprender cómo usar estas clases puede mejorar significativamente tus capacidades de scripting. Esta sección te guiará a través de los conceptos básicos del uso de clases .NET en PowerShell, progresando hacia un uso más avanzado.

Accediendo a Clases .NET

Uso Básico

  • Cargando Clases .NET: Puedes acceder a las clases .NET directamente en PowerShell utilizando la sintaxis [Namespace.ClassName].
  • Creando Instancias: Usa el cmdlet New-Object o el método ::new() para crear instancias de clases .NET.

Ejemplo: Creando un Objeto .NET

# Usando New-Object
$object = New-Object System.DateTime

# Usando el método ::new() (PowerShell 5.0+)
$object = [System.DateTime]::new()

Métodos y Propiedades Estáticas

  • Métodos Estáticos: Llamados en la clase misma en lugar de una instancia.
  • Propiedades Estáticas: Accedidas directamente desde la clase.

Ejemplo: Usando Métodos y Propiedades Estáticas

# Método Estático
$currentDate = [System.DateTime]::Now

# Propiedad Estática
$maxValue = [System.Int32]::MaxValue

Clases .NET Comúnmente Usadas

System.String

  • Descripción: Representa texto como una serie de caracteres Unicode.
  • Métodos Comunes: Substring(), Replace(), ToUpper(), ToLower()

Ejemplo: Manipulando Cadenas

$string = "Hello, World!"
$substring = $string.Substring(0, 5)  # Salida: Hello
$replacedString = $string.Replace("World", "PowerShell")  # Salida: Hello, PowerShell!

System.IO.File

  • Descripción: Proporciona métodos estáticos para la creación, copia, eliminación, movimiento y apertura de archivos.
  • Métodos Comunes: ReadAllText(), WriteAllText(), Copy(), Delete()

Ejemplo: Operaciones con Archivos

$filePath = "C:\example.txt"

# Escribir texto en un archivo
[System.IO.File]::WriteAllText($filePath, "Hello, PowerShell!")

# Leer texto de un archivo
$content = [System.IO.File]::ReadAllText($filePath)
Write-Output $content  # Salida: Hello, PowerShell!

System.Net.WebClient

  • Descripción: Proporciona métodos comunes para enviar datos a y recibir datos de un recurso identificado por un URI.
  • Métodos Comunes: DownloadString(), UploadString()

Ejemplo: Descargando Contenido de la Web

$url = "http://example.com"
$webClient = [System.Net.WebClient]::new()
$content = $webClient.DownloadString($url)
Write-Output $content

Uso Avanzado

Usando Colecciones .NET

  • System.Collections.Generic.List: Una lista fuertemente tipada de objetos.
  • System.Collections.Hashtable: Representa una colección de pares clave/valor.

Ejemplo: Usando una Lista Genérica

# Crear una nueva Lista de enteros
$list = [System.Collections.Generic.List[int]]::new()
$list.Add(1)
$list.Add(2)
$list.Add(3)

# Iterar a través de la lista
foreach ($item in $list) {
    Write-Output $item
}

Manejo de Excepciones con Excepciones .NET

  • Try/Catch/Finally: PowerShell soporta el manejo estructurado de excepciones usando excepciones .NET.

Ejemplo: Manejo de Excepciones

try {
    # Intentar abrir un archivo inexistente
    [System.IO.File]::Open("C:\nonexistentfile.txt", [System.IO.FileMode]::Open)
} catch [System.IO.FileNotFoundException] {
    Write-Output "Archivo no encontrado."
} finally {
    Write-Output "Manejo de excepciones completo."
}

Conclusión

Usar clases .NET en PowerShell abre una amplia gama de funcionalidades que pueden mejorar tus scripts. Desde la creación básica de objetos hasta el manejo avanzado de excepciones, comprender cómo aprovechar estas clases es crucial tanto para usuarios principiantes como avanzados de PowerShell. Practica usando diferentes clases .NET y explora la documentación de .NET para expandir aún más tu conocimiento y capacidades.
