Introducción

Las declaraciones condicionales son fundamentales en la programación, ya que te permiten ejecutar código basado en ciertas condiciones. En PowerShell, las declaraciones condicionales ayudan a automatizar tareas tomando decisiones dentro de los scripts.

Conceptos clave

  • Declaración If: Ejecuta un bloque de código si una condición especificada es verdadera.
  • Declaración Else: Ejecuta un bloque de código si la condición en la declaración if es falsa.
  • Declaración ElseIf: Añade condiciones adicionales a una declaración if.
  • Declaración Switch: Evalúa una sola expresión contra múltiples valores posibles.

Declaración If

La declaración if se usa para probar una condición. Si la condición es verdadera, se ejecuta el bloque de código dentro de la declaración if.

Sintaxis

if (<condition>) {
    <code to execute if condition is true>
}

Ejemplo

$number = 10

if ($number -gt 5) {
    Write-Output "El número es mayor que 5"
}

En este ejemplo, la condición $number -gt 5 verifica si $number es mayor que 5. Dado que $number es 10, la condición es verdadera y se imprime el mensaje "El número es mayor que 5".

Declaración Else

La declaración else se usa para ejecutar un bloque de código si la condición en la declaración if es falsa.

Sintaxis

if (<condition>) {
    <code to execute if condition is true>
} else {
    <code to execute if condition is false>
}

Ejemplo

$number = 3

if ($number -gt 5) {
    Write-Output "El número es mayor que 5"
} else {
    Write-Output "El número no es mayor que 5"
}

Aquí, la condición $number -gt 5 es falsa porque $number es 3. Por lo tanto, se imprime el mensaje "El número no es mayor que 5".

Declaración ElseIf

La declaración elseif te permite probar múltiples condiciones.

Sintaxis

if (<condition1>) {
    <code to execute if condition1 is true>
} elseif (<condition2>) {
    <code to execute if condition2 is true>
} else {
    <code to execute if all conditions are false>
}

Ejemplo

$number = 7

if ($number -gt 10) {
    Write-Output "El número es mayor que 10"
} elseif ($number -gt 5) {
    Write-Output "El número es mayor que 5 pero menor o igual a 10"
} else {
    Write-Output "El número es 5 o menos"
}

En este ejemplo, la primera condición $number -gt 10 es falsa, pero la segunda condición $number -gt 5 es verdadera. Por lo tanto, se imprime el mensaje "El número es mayor que 5 pero menor o igual a 10".

Declaración Switch

La declaración switch evalúa una sola expresión contra múltiples valores posibles y ejecuta el bloque de código correspondiente.

Sintaxis

switch (<expression>) {
    <value1> { <code to execute if expression equals value1> }
    <value2> { <code to execute if expression equals value2> }
    default { <code to execute if expression does not match any value> }
}

Ejemplo

$day = "Tuesday"

switch ($day) {
    "Monday" { Write-Output "Hoy es lunes" }
    "Tuesday" { Write-Output "Hoy es martes" }
    "Wednesday" { Write-Output "Hoy es miércoles" }
    default { Write-Output "Hoy no es lunes, martes ni miércoles" }
}

En este ejemplo, se evalúa la expresión $day. Dado que $day es "Tuesday", se imprime el mensaje "Hoy es martes".

Resumen

Las declaraciones condicionales en PowerShell son esenciales para controlar el flujo de los scripts basados en condiciones específicas. Las declaraciones if, else, elseif y switch proporcionan herramientas poderosas para tomar decisiones y ejecutar código en consecuencia. Comprender y usar estas declaraciones de manera efectiva mejorará tu capacidad para escribir scripts de PowerShell dinámicos y receptivos.
