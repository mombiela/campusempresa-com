Introducción

Leer y escribir archivos son tareas fundamentales en la creación de scripts y la automatización. PowerShell proporciona cmdlets robustos para manejar operaciones de archivos de manera eficiente. Esta sección te guiará desde las técnicas básicas hasta las avanzadas para gestionar archivos en PowerShell.

Operaciones Básicas de Archivos

Lectura de Archivos

PowerShell facilita la lectura del contenido de un archivo utilizando el cmdlet Get-Content.

Ejemplo:

# Leer el contenido de un archivo y mostrarlo
Get-Content -Path "C:\example\file.txt"
  • Get-Content: Lee el contenido del archivo especificado.
  • -Path: Especifica la ruta al archivo.

Escritura de Archivos

Para escribir contenido en un archivo, puedes usar los cmdlets Set-Content o Add-Content.

Ejemplo:

# Escribir contenido en un nuevo archivo o sobrescribir un archivo existente
Set-Content -Path "C:\example\file.txt" -Value "¡Hola, Mundo!"

# Añadir contenido a un archivo existente
Add-Content -Path "C:\example\file.txt" -Value "Esta es una línea adicional."
  • Set-Content: Escribe o reemplaza el contenido de un archivo.
  • Add-Content: Añade contenido a un archivo existente.
  • -Value: Especifica el contenido a escribir o añadir.

Operaciones Intermedias de Archivos

Lectura de Archivos Grandes

Para archivos grandes, Get-Content puede ser ineficiente. En su lugar, usa el parámetro -ReadCount para leer el archivo en fragmentos.

Ejemplo:

# Leer un archivo grande en fragmentos de 100 líneas
Get-Content -Path "C:\example\largefile.txt" -ReadCount 100 | ForEach-Object {
    # Procesar cada fragmento aquí
    $_
}
  • -ReadCount: Especifica el número de líneas a leer a la vez.

Escritura de Datos Estructurados

PowerShell puede manejar formatos de datos estructurados como CSV, JSON y XML.

Ejemplo: Escritura de CSV

# Crear un array de objetos
$data = @(
    [PSCustomObject]@{Name="John"; Age=30},
    [PSCustomObject]@{Name="Jane"; Age=25}
)

# Exportar el array a un archivo CSV
$data | Export-Csv -Path "C:\example\data.csv" -NoTypeInformation
  • Export-Csv: Convierte objetos en un archivo CSV.
  • -NoTypeInformation: Omite la información de tipo del archivo CSV.

Lectura de Datos Estructurados

De manera similar, puedes importar datos estructurados desde archivos.

Ejemplo: Lectura de CSV

# Importar datos desde un archivo CSV
$data = Import-Csv -Path "C:\example\data.csv"

# Mostrar los datos importados
$data
  • Import-Csv: Lee un archivo CSV y lo convierte en objetos.

Operaciones Avanzadas de Archivos

Manejo de Archivos Binarios

Para archivos binarios, usa Get-Content con el parámetro -AsByteStream.

Ejemplo:

# Leer un archivo binario
$binaryData = Get-Content -Path "C:\example\file.bin" -AsByteStream

# Escribir datos binarios en un nuevo archivo
[System.IO.File]::WriteAllBytes("C:\example\newfile.bin", $binaryData)
  • -AsByteStream: Lee el archivo como una matriz de bytes.

Flujos de Archivos

PowerShell te permite trabajar con flujos de archivos para un control más granular.

Ejemplo:

# Abrir un flujo de archivo para lectura
$fileStream = [System.IO.File]::OpenRead("C:\example\file.txt")
$reader = New-Object System.IO.StreamReader($fileStream)

# Leer el contenido del archivo
$content = $reader.ReadToEnd()

# Cerrar el flujo
$reader.Close()
$fileStream.Close()

# Mostrar el contenido
$content
  • System.IO.File::OpenRead: Abre un flujo de archivo para lectura.
  • System.IO.StreamReader: Lee caracteres de un flujo de bytes.

Conclusión

PowerShell proporciona un conjunto completo de cmdlets y métodos para leer y escribir archivos, desde archivos de texto simples hasta datos estructurados complejos y archivos binarios. Al dominar estas técnicas, puedes automatizar una amplia gama de operaciones de archivos de manera eficiente.

Resumen

  • Operaciones Básicas: Get-Content, Set-Content, Add-Content.
  • Operaciones Intermedias: Manejo de archivos grandes, datos estructurados (CSV, JSON, XML).
  • Operaciones Avanzadas: Archivos binarios, flujos de archivos.

Con estas herramientas, puedes manejar prácticamente cualquier tarea relacionada con archivos en PowerShell, mejorando tus capacidades de scripting y automatización.
