Configurar tu proyecto Dart es el primer paso para construir aplicaciones usando el lenguaje de programación Dart. Esta sección te guiará a través del proceso de instalación de Dart, configuración de tu entorno de desarrollo y creación de tu primer proyecto Dart.

Instalando Dart

Antes de que puedas empezar a escribir código Dart, necesitas instalar el SDK de Dart (Software Development Kit) en tu máquina. El SDK de Dart incluye la VM de Dart, las bibliotecas principales de Dart y varias herramientas de línea de comandos.

Pasos para Instalar Dart

  1. Visita la Página de Descarga del SDK de Dart:

    • Ve a la página oficial de descarga del SDK de Dart: Dart SDK.

  2. Elige Tu Sistema Operativo:

    • Selecciona el método de instalación apropiado para tu sistema operativo (Windows, macOS o Linux).

  3. Sigue las Instrucciones de Instalación:

    • Windows: Descarga el archivo zip del SDK de Dart, extráelo y añade el directorio bin a la variable PATH de tu sistema.
    • macOS: Usa Homebrew para instalar Dart ejecutando el comando: 
      brew tap dart-lang/dart
brew install dart
    • Linux: Usa el gestor de paquetes apt para instalar Dart ejecutando los comandos: 
      sudo apt update
sudo apt install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt update
sudo apt install dart

  4. Verifica la Instalación:

    • Abre una terminal o símbolo del sistema y ejecuta: 
      dart --version
    • Deberías ver la versión de Dart instalada.

Configuración de Tu Entorno de Desarrollo

Para escribir y ejecutar código Dart de manera eficiente, necesitas un buen entorno de desarrollo. Visual Studio Code (VS Code) es una opción popular debido a sus potentes características y amplio soporte de plugins.

Instalando Visual Studio Code

  1. Descarga e Instala VS Code:

    • Visita la página de descarga de Visual Studio Code y descarga el instalador para tu sistema operativo.
    • Sigue las instrucciones de instalación para instalar VS Code.

  2. Instala el Plugin de Dart:

    • Abre VS Code.
    • Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana o presionando Ctrl+Shift+X.
    • Busca "Dart" y haz clic en el botón Instalar para el plugin de Dart.

Creando Tu Primer Proyecto Dart

Ahora que tienes Dart y VS Code instalados, puedes crear tu primer proyecto Dart.

Pasos para Crear un Proyecto Dart

  1. Abre VS Code:

    • Inicia Visual Studio Code.

  2. Abre la Paleta de Comandos:

    • Presiona Ctrl+Shift+P para abrir la Paleta de Comandos.

  3. Crea un Nuevo Proyecto Dart:

    • Escribe Dart: New Project y selecciónalo de la lista.
    • Elige Console Application cuando se te pida.
    • Selecciona una ubicación para tu nuevo proyecto y proporciona un nombre para él.

  4. Explora la Estructura del Proyecto:

    • VS Code creará un nuevo proyecto Dart con la siguiente estructura: 
      my_project/
├── bin/
│   └── main.dart
├── lib/
├── test/
└── pubspec.yaml
    • bin/main.dart: El punto de entrada principal de tu aplicación.
    • lib/: Directorio para bibliotecas reutilizables.
    • test/: Directorio para pruebas unitarias.
    • pubspec.yaml: Archivo de configuración para las dependencias de tu proyecto.

Ejecutando Tu Aplicación Dart

  1. Abre main.dart:

    • Navega a bin/main.dart y ábrelo.

  2. Escribe Tu Primer Código Dart:

    • Reemplaza el código existente con el siguiente: 
      void main() {
  print('¡Hola, Dart!');
}

  3. Ejecuta la Aplicación:

    • Abre la terminal en VS Code presionando `Ctrl+``.
    • Asegúrate de estar en el directorio del proyecto y ejecuta: 
      dart run bin/main.dart
    • Deberías ver la salida: 
      ¡Hola, Dart!

Conclusión

Configurar un proyecto Dart implica instalar el SDK de Dart, configurar un entorno de desarrollo como Visual Studio Code y crear un nuevo proyecto Dart. Siguiendo estos pasos, has configurado exitosamente tu primer proyecto Dart y ejecutado una simple aplicación Dart. Esta configuración fundamental te permitirá comenzar a explorar y construir aplicaciones Dart más complejas a medida que avances en el curso.
