Introducción

En Dart, las variables se utilizan para almacenar datos que pueden ser manipulados y referenciados a lo largo de tu programa. Entender las variables y los tipos de datos es fundamental para programar en Dart, ya que te permite manejar diferentes tipos de datos de manera efectiva.

Variables

Las variables en Dart se utilizan para almacenar valores. Puedes declarar una variable usando las palabras clave var, final o const.

Declaración de Variables

  • var: El tipo de la variable es inferido por el compilador.
  • final: Una variable que solo puede ser establecida una vez.
  • const: Una constante en tiempo de compilación.
void main() {
  var name = 'Alice'; // Tipo String inferido
  final age = 30; // Tipo int inferido, no puede ser cambiado
  const pi = 3.14; // Tipo double, constante en tiempo de compilación
}

Reglas de Nomenclatura de Variables

  • Deben comenzar con una letra o guion bajo (_).
  • Pueden contener letras, dígitos y guiones bajos.
  • Sensibles a mayúsculas y minúsculas.
void main() {
  var firstName = 'Alice';
  var _lastName = 'Smith';
  var age1 = 25;
}

Tipos de Datos

Dart es un lenguaje de tipado estático, lo que significa que cada variable tiene un tipo. Los tipos de datos comunes en Dart incluyen:

  • Números: int y double
  • Cadenas: String
  • Booleanos: bool
  • Listas: List
  • Mapas: Map

Números

Dart tiene dos tipos de números: int para enteros y double para números de punto flotante.

void main() {
  int age = 25;
  double height = 5.9;
}

Cadenas

Las cadenas representan una secuencia de caracteres. Están encerradas en comillas simples o dobles.

void main() {
  String greeting = '¡Hola, Mundo!';
  String name = "Alice";
}

Booleanos

Los booleanos representan valores verdaderos o falsos.

void main() {
  bool isStudent = true;
  bool isEmployed = false;
}

Listas

Las listas son colecciones ordenadas de objetos. Pueden ser de longitud fija o variable.

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  List<String> names = ['Alice', 'Bob', 'Charlie'];
}

Mapas

Los mapas son colecciones de pares clave-valor. Las claves y los valores pueden ser de cualquier tipo.

void main() {
  Map<String, int> phoneBook = {
    'Alice': 12345,
    'Bob': 67890,
  };
}

Inferencia de Tipos

Dart puede inferir el tipo de una variable basado en el valor asignado. Esto se hace usando la palabra clave var.

void main() {
  var name = 'Alice'; // Inferido como String
  var age = 30; // Inferido como int
  var height = 5.9; // Inferido como double
}

Anotación de Tipos

Puedes especificar explícitamente el tipo de una variable.

void main() {
  String name = 'Alice';
  int age = 30;
  double height = 5.9;
}

Conversión de Tipos

A veces necesitas convertir un tipo de dato a otro. Dart proporciona métodos para la conversión de tipos.

String a int

void main() {
  var one = int.parse('1'); // 1
}

String a double

void main() {
  var onePointOne = double.parse('1.1'); // 1.1
}

int a String

void main() {
  String oneAsString = 1.toString(); // '1'
}

double a String

void main() {
  String piAsString = 3.14159.toStringAsFixed(2); // '3.14'
}

Resumen

Entender las variables y los tipos de datos es crucial para programar de manera efectiva en Dart. Las variables almacenan datos, y los tipos de datos definen el tipo de datos que una variable puede contener. Dart soporta varios tipos de datos, incluyendo números, cadenas, booleanos, listas y mapas. Puedes declarar variables usando var, final y const, y Dart puede inferir tipos o puedes anotarlos explícitamente. Además, Dart proporciona métodos para la conversión de tipos para manejar diferentes tipos de datos sin problemas.

Al dominar estos conceptos, estarás bien equipado para manejar datos en tus programas Dart de manera eficiente.
