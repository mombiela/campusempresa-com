Introducción

Dart es un lenguaje optimizado para clientes para aplicaciones rápidas en cualquier plataforma. Es desarrollado por Google y se utiliza para construir aplicaciones móviles, de escritorio, servidor y web. Entender la sintaxis y estructura básica de Dart es esencial para escribir código eficiente y legible.

Conceptos Clave

  • Variables y Tipos de Datos
  • Funciones
  • Sentencias de Control de Flujo
  • Comentarios
  • Importaciones

Variables y Tipos de Datos

En Dart, las variables se utilizan para almacenar datos. Dart es un lenguaje de tipo estático, lo que significa que necesitas especificar el tipo de una variable cuando la declaras.

Declaración de Variables

void main() {
  int age = 25; // Entero
  double height = 5.9; // Doble
  String name = 'John Doe'; // Cadena
  bool isStudent = true; // Booleano
}

Inferencia de Tipos

Dart puede inferir el tipo de una variable basado en el valor asignado.

void main() {
  var age = 25; // Dart infiere que age es un int
  var height = 5.9; // Dart infiere que height es un double
  var name = 'John Doe'; // Dart infiere que name es un String
  var isStudent = true; // Dart infiere que isStudent es un bool
}

Funciones

Las funciones son los bloques de construcción de un programa Dart. Permiten encapsular código en bloques reutilizables.

Definición de Funciones

void main() {
  printGreeting();
  print(add(5, 3));
}

void printGreeting() {
  print('¡Hola, Mundo!');
}

int add(int a, int b) {
  return a + b;
}

Funciones Flecha

Para funciones simples, Dart proporciona una sintaxis abreviada conocida como funciones flecha.

int add(int a, int b) => a + b;

Sentencias de Control de Flujo

Las sentencias de control de flujo se utilizan para controlar el flujo de ejecución del programa.

Sentencias If-Else

void main() {
  int age = 20;

  if (age >= 18) {
    print('Eres un adulto.');
  } else {
    print('Eres un menor.');
  }
}

Sentencias Switch

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excelente');
      break;
    case 'B':
      print('Bueno');
      break;
    case 'C':
      print('Regular');
      break;
    case 'D':
      print('Pobre');
      break;
    default:
      print('Reprobado');
  }
}

Bucles

Bucle For

void main() {
  for (int i = 0; i < 5; i++) {
    print(i);
  }
}

Bucle While

void main() {
  int i = 0;

  while (i < 5) {
    print(i);
    i++;
  }
}

Comentarios de Una Línea

Importaciones

Las importaciones se utilizan para incluir bibliotecas y otros archivos Dart en tu programa.

Importación de Bibliotecas Básicas

import 'dart:math';

Importación de Archivos Personalizados

import 'my_custom_file.dart';

Conclusión

Entender la sintaxis y estructura básica de Dart es crucial para escribir código eficiente y legible. Esto incluye saber cómo declarar variables, definir funciones, usar sentencias de control de flujo, escribir comentarios e importar bibliotecas. Dominar estos conceptos básicos proporcionará una base sólida para conceptos de programación Dart más avanzados.
