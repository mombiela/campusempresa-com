Configuración del Entorno de Desarrollo
Tu Primer Programa en Dart ►

Configurar el entorno de desarrollo es el primer paso para comenzar a programar en Dart. Esta sección te guiará a través del proceso de instalación de Dart, configuración de un IDE y ejecución de tu primer programa en Dart.

Instalación de Dart

Para comenzar a programar en Dart, necesitas instalar el SDK de Dart (Kit de Desarrollo de Software).

Pasos para Instalar el SDK de Dart

  1. Descargar el SDK de Dart:

    • Visita la página de descarga del SDK de Dart.
    • Elige la versión adecuada para tu sistema operativo (Windows, macOS o Linux).

  2. Instalar el SDK de Dart:

    • Windows:
      • Descarga el archivo zip del SDK de Dart.
      • Extrae el archivo zip en una ubicación deseada.
      • Agrega el directorio bin a la variable de entorno PATH de tu sistema.
    • macOS:
      • Usa Homebrew para instalar Dart ejecutando el comando: 
        brew tap dart-lang/dart
brew install dart
    • Linux:
      • Usa APT para instalar Dart ejecutando el comando: 
        sudo apt update
sudo apt install dart

  3. Verificar la Instalación:

    • Abre una terminal o símbolo del sistema.
    • Ejecuta el comando: 
      dart --version
    • Deberías ver la versión de Dart instalada.

Configuración de un Entorno de Desarrollo Integrado (IDE)

Un IDE te ayuda a escribir, depurar y gestionar tu código Dart de manera eficiente. Los IDEs populares para Dart incluyen Visual Studio Code e IntelliJ IDEA.

Configuración de Visual Studio Code (VS Code)

  1. Instalar VS Code:

    • Descarga e instala VS Code desde el sitio web oficial.

  2. Instalar la Extensión de Dart:

    • Abre VS Code.
    • Ve a la vista de Extensiones haciendo clic en el icono de Extensiones en la Barra de Actividad en el lado de la ventana.
    • Busca "Dart" e instala la extensión de Dart.

  3. Configurar el SDK de Dart:

    • Abre la Paleta de Comandos (Ctrl+Shift+P o Cmd+Shift+P).
    • Escribe Dart: Use Recommended Settings y selecciónalo.
    • Esto configurará VS Code para usar el SDK de Dart.

Configuración de IntelliJ IDEA

  1. Instalar IntelliJ IDEA:

    • Descarga e instala IntelliJ IDEA desde el sitio web oficial.

  2. Instalar el Plugin de Dart:

    • Abre IntelliJ IDEA.
    • Ve a File > Settings (o IntelliJ IDEA > Preferences en macOS).
    • Navega a Plugins y busca "Dart".
    • Instala el plugin de Dart y reinicia IntelliJ IDEA si se te solicita.

  3. Configurar el SDK de Dart:

    • Ve a File > Settings (o IntelliJ IDEA > Preferences en macOS).
    • Navega a Languages & Frameworks > Dart.
    • Establece la ruta del SDK de Dart a la ubicación donde instalaste el SDK de Dart.

Ejecutar tu Primer Programa en Dart

Ahora que tu entorno de desarrollo está configurado, vamos a ejecutar un programa simple en Dart.

Escribir un Programa Simple en Dart

  1. Crear un Nuevo Archivo Dart:

    • En tu IDE, crea un nuevo archivo llamado hello.dart.

  2. Escribir el Código:

    • Abre hello.dart y escribe el siguiente código: 
      void main() {
  print('¡Hola, Dart!');
}

  3. Ejecutar el Programa:

    • En VS Code:
      • Abre la terminal dentro de VS Code (Ctrl+ o Cmd+).
      • Navega al directorio que contiene hello.dart.
      • Ejecuta el comando: 
        dart hello.dart
    • En IntelliJ IDEA:
      • Haz clic derecho en hello.dart y selecciona Run 'hello.dart'.

Explicación del Código

  • void main() { ... }: Esta es la función principal, el punto de entrada de cada aplicación Dart.
  • print('¡Hola, Dart!');: Esta línea imprime la cadena '¡Hola, Dart!' en la consola.

Conclusión

Configurar el entorno de desarrollo para Dart implica instalar el SDK de Dart, configurar un IDE y ejecutar un programa simple en Dart. Siguiendo estos pasos, ahora estás listo para comenzar a explorar la programación en Dart. En las siguientes secciones, profundizaremos en la sintaxis de Dart, sus características y temas avanzados.
