Introducción al Manejo de Excepciones

El manejo de excepciones es un aspecto crucial del desarrollo de software robusto. Permite a los desarrolladores gestionar y responder a eventos inesperados o errores que ocurren durante la ejecución de un programa. En Dart, el manejo de excepciones se realiza utilizando bloques try, catch y finally.

Conceptos Clave

  • Excepción: Un evento que interrumpe el flujo normal de un programa.
  • Bloque try: Contiene código que podría lanzar una excepción.
  • Bloque catch: Maneja la excepción.
  • Bloque finally: Ejecuta código independientemente de si se lanzó una excepción o no.
  • throw: Se utiliza para lanzar una excepción.

Manejo Básico de Excepciones

En Dart, puedes manejar excepciones utilizando bloques try y catch.

Ejemplo

void main() {
  try {
    int result = 12 ~/ 0; // Esto lanzará una excepción
    print(result);
  } catch (e) {
    print('Ocurrió un error: $e');
  }
}
  • Explicación:
    • El bloque try contiene código que podría lanzar una excepción.
    • El bloque catch captura la excepción e imprime un mensaje de error.

Captura de Excepciones Específicas

Puedes capturar tipos específicos de excepciones para manejarlas de manera diferente.

Ejemplo

void main() {
  try {
    int result = 12 ~/ 0; // Esto lanzará una IntegerDivisionByZeroException
    print(result);
  } on IntegerDivisionByZeroException {
    print('¡No se puede dividir por cero!');
  } catch (e) {
    print('Ocurrió un error: $e');
  }
}
  • Explicación:
    • La palabra clave on se utiliza para capturar excepciones específicas.
    • Si la excepción es del tipo IntegerDivisionByZeroException, se imprime un mensaje específico.
    • El bloque catch maneja cualquier otro tipo de excepción.

Uso del Bloque finally

El bloque finally contiene código que siempre se ejecutará, independientemente de si se lanzó una excepción o no.

Ejemplo

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('Ocurrió un error: $e');
  } finally {
    print('Este código siempre se ejecuta.');
  }
}
  • Explicación:
    • El bloque finally se ejecuta después de los bloques try y catch, asegurando que el código dentro de él siempre se ejecute.

Lanzamiento de Excepciones

Puedes lanzar excepciones utilizando la palabra clave throw.

Ejemplo

void checkAge(int age) {
  if (age < 18) {
    throw Exception('La edad debe ser 18 o mayor.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('Ocurrió un error: $e');
  }
}
  • Explicación:
    • La función checkAge lanza una excepción si la edad es menor de 18.
    • El bloque try llama a checkAge, y el bloque catch maneja la excepción.

Excepciones Personalizadas

Puedes crear excepciones personalizadas extendiendo la clase Exception.

Ejemplo

class AgeException implements Exception {
  String cause;
  AgeException(this.cause);
}

void checkAge(int age) {
  if (age < 18) {
    throw AgeException('La edad debe ser 18 o mayor.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('Ocurrió un error: $e');
  }
}
  • Explicación:
    • La clase AgeException implementa la interfaz Exception.
    • La función checkAge lanza una AgeException si la edad es menor de 18.
    • El bloque catch maneja la excepción personalizada.

Resumen

El manejo de excepciones en Dart es una característica poderosa que ayuda a gestionar errores y eventos inesperados en tu código. Utilizando try, catch, finally y throw, puedes asegurarte de que tus programas sean robustos y puedan manejar diversas condiciones de error de manera elegante. Además, crear excepciones personalizadas permite un manejo de errores más específico y adaptado a las necesidades de tu aplicación.
