Introducción

Flutter es un kit de herramientas de desarrollo de software de interfaz de usuario de código abierto creado por Google. Se utiliza para desarrollar aplicaciones multiplataforma para Android, iOS, Linux, Mac, Windows, Google Fuchsia y la web desde una única base de código. En esta sección, te guiaremos a través de la configuración de Flutter en tu máquina.

Requisitos previos

Antes de configurar Flutter, asegúrate de tener lo siguiente:

  • Una computadora con macOS, Windows o Linux.
  • Conocimiento básico del lenguaje de programación Dart.
  • Un IDE (Entorno de Desarrollo Integrado) como Android Studio, IntelliJ IDEA o Visual Studio Code.

Configuración paso a paso

1. Descargar Flutter SDK

  • Visita el sitio web oficial de Flutter.
  • Elige tu sistema operativo (Windows, macOS o Linux).
  • Descarga el archivo zip del SDK de Flutter.

2. Extraer el SDK de Flutter

  • Extrae el archivo zip descargado en una ubicación deseada en tu sistema.
  • Windows: Extrae a C:\src\flutter (o cualquier otra ubicación).
  • macOS/Linux: Extrae en tu directorio de inicio o en cualquier otra ubicación.

3. Actualizar tu PATH

Para ejecutar comandos de Flutter en la terminal, necesitas agregar Flutter a tu PATH del sistema.

Windows

  1. Abre la búsqueda de inicio, escribe "env" y selecciona "Editar las variables de entorno del sistema".
  2. En la ventana de Propiedades del sistema, haz clic en el botón "Variables de entorno".
  3. Bajo "Variables del sistema", encuentra la variable Path, selecciónala y haz clic en "Editar".
  4. Haz clic en "Nuevo" y agrega la ruta al directorio flutter\bin (por ejemplo, C:\src\flutter\bin).
  5. Haz clic en OK para guardar los cambios.

macOS/Linux

  1. Abre una terminal.
  2. Abre tu perfil de bash en un editor de texto: 
    nano ~/.bash_profile
  3. Agrega la siguiente línea al archivo: 
    export PATH="$PATH:`pwd`/flutter/bin"
  4. Guarda el archivo y ejecuta: 
    source ~/.bash_profile

4. Verificar la instalación

Abre una terminal y ejecuta el siguiente comando para verificar que Flutter esté instalado correctamente:

flutter doctor

Este comando verifica tu entorno y muestra un informe del estado de tu instalación de Flutter. La salida debería verse algo así:

Doctor summary (to see all details, run flutter doctor -v):
[✓] Flutter (Channel stable, 2.5.3, on macOS 11.6 20G165 darwin-x64, locale en-US)
[✓] Android toolchain - develop for Android devices (Android SDK version 31.0.0)
[✓] Xcode - develop for iOS and macOS (Xcode 13.1)
[✓] Chrome - develop for the web
[✓] Android Studio (version 2020.3)
[✓] VS Code (version 1.61.2)
[✓] Connected device (2 available)

Asegúrate de que todas las verificaciones estén marcadas con una marca de verificación verde. Si hay algún problema, sigue las instrucciones proporcionadas para resolverlos.

5. Configurar un editor

Flutter admite varios editores. Aquí, cubriremos la configuración de Visual Studio Code y Android Studio.

Visual Studio Code

  1. Instala Visual Studio Code desde aquí.
  2. Abre Visual Studio Code.
  3. Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana o presionando Ctrl+Shift+X.
  4. Busca "Flutter" e instala la extensión de Flutter por Dart Code.

Android Studio

  1. Descarga e instala Android Studio desde aquí.
  2. Abre Android Studio.
  3. Ve a File > Settings > Plugins.
  4. Busca "Flutter" y haz clic en "Instalar".
  5. Sigue las indicaciones para instalar también el plugin de Dart.

6. Crear un nuevo proyecto Flutter

Una vez que tu editor esté configurado, puedes crear un nuevo proyecto Flutter.

Visual Studio Code

  1. Abre Visual Studio Code.
  2. Presiona Ctrl+Shift+P para abrir la paleta de comandos.
  3. Escribe Flutter: New Project y selecciónalo.
  4. Sigue las indicaciones para nombrar tu proyecto y elegir una ubicación para guardarlo.

Android Studio

  1. Abre Android Studio.
  2. Haz clic en Start a new Flutter project.
  3. Selecciona Flutter Application y haz clic en Next.
  4. Sigue las indicaciones para configurar tu nuevo proyecto Flutter.

7. Ejecutar tu primera aplicación Flutter

Para ejecutar tu aplicación Flutter, sigue estos pasos:

Visual Studio Code

  1. Abre el archivo Dart principal (lib/main.dart).
  2. Presiona F5 para iniciar la depuración.
  3. Selecciona tu dispositivo objetivo (por ejemplo, un emulador de Android o un dispositivo conectado).

Android Studio

  1. Abre el archivo Dart principal (lib/main.dart).
  2. Haz clic en el botón verde de reproducción en la barra de herramientas.
  3. Selecciona tu dispositivo objetivo (por ejemplo, un emulador de Android o un dispositivo conectado).

Conclusión

Configurar Flutter implica descargar el SDK, actualizar tu PATH del sistema, verificar la instalación, configurar un editor, crear un nuevo proyecto y ejecutar tu primera aplicación. Siguiendo estos pasos, ahora deberías tener un entorno de desarrollo Flutter completamente funcional. Esta configuración te permitirá construir y ejecutar aplicaciones Flutter sin problemas.
