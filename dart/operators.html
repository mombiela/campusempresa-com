Introducción

Los operadores son símbolos especiales que realizan operaciones en variables y valores. Dart admite una amplia gama de operadores, desde aritmética básica hasta operadores lógicos y de bits más avanzados. Comprender estos operadores es crucial para escribir código Dart eficiente y efectivo.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas comunes.

  • + Suma
  • - Resta
  • * Multiplicación
  • / División
  • % Módulo (resto)
  • ~/ División Entera

Ejemplos

void main() {
  int a = 10;
  int b = 3;

  print(a + b);  // Salida: 13
  print(a - b);  // Salida: 7
  print(a * b);  // Salida: 30
  print(a / b);  // Salida: 3.3333333333333335
  print(a % b);  // Salida: 1
  print(a ~/ b); // Salida: 3
}

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • = Asignar
  • += Sumar y asignar
  • -= Restar y asignar
  • *= Multiplicar y asignar
  • /= Dividir y asignar
  • %= Módulo y asignar

Ejemplos

void main() {
  int a = 5;
  a += 3; // Equivalente a a = a + 3
  print(a); // Salida: 8

  a -= 2; // Equivalente a a = a - 2
  print(a); // Salida: 6

  a *= 2; // Equivalente a a = a * 2
  print(a); // Salida: 12

  a /= 3; // Equivalente a a = a / 3
  print(a); // Salida: 4.0

  a %= 2; // Equivalente a a = a % 2
  print(a); // Salida: 0.0
}

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores y devolver un resultado booleano.

  • == Igual a
  • != No igual a
  • > Mayor que
  • < Menor que
  • >= Mayor o igual que
  • <= Menor o igual que

Ejemplos

void main() {
  int a = 5;
  int b = 3;

  print(a == b); // Salida: false
  print(a != b); // Salida: true
  print(a > b);  // Salida: true
  print(a < b);  // Salida: false
  print(a >= b); // Salida: true
  print(a <= b); // Salida: false
}

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas.

  • && AND lógico
  • || OR lógico
  • ! NOT lógico

Ejemplos

void main() {
  bool a = true;
  bool b = false;

  print(a && b); // Salida: false
  print(a || b); // Salida: true
  print(!a);     // Salida: false
}

Operadores de Bits

Los operadores de bits se utilizan para realizar operaciones a nivel de bits.

  • & AND de bits
  • | OR de bits
  • ^ XOR de bits
  • ~ NOT de bits
  • << Desplazamiento a la izquierda
  • >> Desplazamiento a la derecha

Ejemplos

void main() {
  int a = 5;  // 0101 en binario
  int b = 3;  // 0011 en binario

  print(a & b);  // Salida: 1 (0001 en binario)
  print(a | b);  // Salida: 7 (0111 en binario)
  print(a ^ b);  // Salida: 6 (0110 en binario)
  print(~a);     // Salida: -6 (invierte todos los bits)
  print(a << 1); // Salida: 10 (1010 en binario)
  print(a >> 1); // Salida: 2 (0010 en binario)
}

Operador Condicional (Ternario)

El operador condicional es una abreviatura para una declaración if-else.

  • condición ? expr1 : expr2

Ejemplo

void main() {
  int a = 5;
  int b = 3;

  String result = a > b ? 'a es mayor' : 'b es mayor';
  print(result); // Salida: a es mayor
}

Operadores de Conocimiento Nulo

Los operadores de conocimiento nulo se utilizan para manejar valores nulos de manera más elegante.

  • ?? Si es nulo
  • ??= Asignar si es nulo
  • ?. Acceso condicional a miembros

Ejemplos

void main() {
  int? a;
  int b = 3;

  print(a ?? b); // Salida: 3

  a ??= 10;
  print(a); // Salida: 10

  String? name;
  print(name?.length); // Salida: null
}

Resumen

Los operadores en Dart son herramientas esenciales para realizar una amplia gama de operaciones, desde aritmética básica hasta manipulaciones lógicas y de bits complejas. Comprender cómo usar estos operadores de manera efectiva te ayudará a escribir código Dart más eficiente y legible. Practica usando estos operadores en diferentes escenarios para familiarizarte más con su sintaxis y funcionalidad.
