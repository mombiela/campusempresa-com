Introducción a los Mapas

Los mapas son una colección de pares clave-valor donde cada clave es única. Son similares a los diccionarios en otros lenguajes de programación. Los mapas son útiles cuando necesitas asociar valores con claves y recuperarlos de manera eficiente.

  • Conceptos Clave:
    • Clave: Un identificador único para un valor.
    • Valor: Los datos asociados con una clave.
    • Mapa: Una colección de pares clave-valor.

Creación de Mapas

Puedes crear mapas en Dart usando la clase Map o la sintaxis literal de mapas.

Usando la Clase Map

void main() {
  Map<String, int> ages = Map();
  ages['Alice'] = 25;
  ages['Bob'] = 30;
  print(ages); // Salida: {Alice: 25, Bob: 30}
}

Usando Literales de Mapas

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages); // Salida: {Alice: 25, Bob: 30}
}

Acceso a Elementos del Mapa

Puedes acceder a los elementos del mapa usando sus claves.

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages['Alice']); // Salida: 25
}

Modificación de Mapas

Puedes agregar, actualizar o eliminar elementos de un mapa.

Agregar Elementos

void main() {
  var ages = {
    'Alice': 25,
  };
  ages['Bob'] = 30;
  print(ages); // Salida: {Alice: 25, Bob: 30}
}

Actualizar Elementos

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  ages['Alice'] = 26;
  print(ages); // Salida: {Alice: 26, Bob: 30}
}

Eliminar Elementos

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  ages.remove('Bob');
  print(ages); // Salida: {Alice: 25}
}

Iteración Sobre Mapas

Puedes iterar sobre las claves, valores o pares clave-valor en un mapa.

Iteración Sobre Claves

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var key in ages.keys) {
    print(key); // Salida: Alice Bob
  }
}

Iteración Sobre Valores

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var value in ages.values) {
    print(value); // Salida: 25 30
  }
}

Iteración Sobre Pares Clave-Valor

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var entry in ages.entries) {
    print('${entry.key}: ${entry.value}'); // Salida: Alice: 25 Bob: 30
  }
}

Operaciones Avanzadas con Mapas

Comprobación de Clave o Valor

Puedes comprobar si un mapa contiene una clave o un valor específico.

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages.containsKey('Alice')); // Salida: true
  print(ages.containsValue(30)); // Salida: true
}

Fusión de Mapas

Puedes fusionar dos mapas usando el método addAll.

void main() {
  var ages1 = {
    'Alice': 25,
  };
  var ages2 = {
    'Bob': 30,
  };
  ages1.addAll(ages2);
  print(ages1); // Salida: {Alice: 25, Bob: 30}
}

Conclusión

Los mapas son una forma poderosa y flexible de gestionar pares clave-valor en Dart. Permiten una recuperación y manipulación de datos eficiente. Entender cómo crear, acceder, modificar e iterar sobre mapas es esencial para una programación efectiva en Dart. Al dominar estos conceptos, puedes manejar estructuras de datos complejas y mejorar el rendimiento de tus aplicaciones.
