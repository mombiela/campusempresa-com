Introducción a Clases y Objetos

En Dart, las clases son el plano para crear objetos. Un objeto es una instancia de una clase, y puede tener propiedades (variables) y métodos (funciones) que definen su comportamiento.

Conceptos Clave

  • Clase: Una plantilla para crear objetos.
  • Objeto: Una instancia de una clase.
  • Constructor: Un método especial utilizado para inicializar objetos.
  • Propiedades: Variables que pertenecen a un objeto.
  • Métodos: Funciones que pertenecen a un objeto.

Definiendo una Clase

Para definir una clase en Dart, usa la palabra clave class seguida del nombre de la clase y un par de llaves {}.

class Animal {
  // Propiedades
  String name;
  int age;

  // Constructor
  Animal(this.name, this.age);

  // Método
  void displayInfo() {
    print('Name: $name, Age: $age');
  }
}

Explicación

  • Propiedades: name y age son propiedades de la clase Animal.
  • Constructor: Animal(this.name, this.age); inicializa las propiedades.
  • Método: displayInfo() es un método que imprime la información del objeto.

Creando Objetos

Para crear un objeto, usa la palabra clave new seguida del nombre de la clase y los parámetros requeridos.

void main() {
  // Creando un objeto
  Animal dog = new Animal('Buddy', 3);

  // Accediendo a propiedades
  print(dog.name); // Salida: Buddy

  // Llamando a métodos
  dog.displayInfo(); // Salida: Name: Buddy, Age: 3
}

Explicación

  • Creación de Objeto: Animal dog = new Animal('Buddy', 3); crea un objeto llamado dog.
  • Accediendo a Propiedades: dog.name accede a la propiedad name.
  • Llamando a Métodos: dog.displayInfo() llama al método displayInfo.

Conceptos Avanzados

Herencia

La herencia permite que una clase herede propiedades y métodos de otra clase.

class Dog extends Animal {
  String breed;

  Dog(String name, int age, this.breed) : super(name, age);

  void displayBreed() {
    print('Breed: $breed');
  }
}

Explicación

  • Herencia: class Dog extends Animal significa que Dog hereda de Animal.
  • Constructor: Dog(String name, int age, this.breed) : super(name, age); llama al constructor de la clase padre.

Polimorfismo

El polimorfismo permite que los métodos hagan cosas diferentes según el objeto sobre el que actúan.

void main() {
  Animal myDog = new Dog('Buddy', 3, 'Golden Retriever');
  myDog.displayInfo(); // Salida: Name: Buddy, Age: 3
  // myDog.displayBreed(); // Esto causará un error
}

Explicación

  • Polimorfismo: Animal myDog = new Dog('Buddy', 3, 'Golden Retriever'); crea un objeto Dog pero lo trata como un Animal.

Encapsulación

La encapsulación restringe el acceso directo a algunos de los componentes de un objeto.

class Cat {
  String _name; // Propiedad privada

  Cat(this._name);

  String get name => _name; // Getter

  set name(String name) {
    _name = name;
  }
}

Explicación

  • Propiedad Privada: _name es privada para la clase Cat.
  • Getter y Setter: get name y set name proporcionan acceso controlado a _name.

Resumen

  • Clases son planos para crear objetos.
  • Objetos son instancias de clases con propiedades y métodos.
  • Constructores inicializan objetos.
  • Herencia permite que una clase herede de otra clase.
  • Polimorfismo permite que los métodos actúen de manera diferente según el objeto.
  • Encapsulación restringe el acceso directo a las propiedades del objeto.

Entender estos conceptos es crucial para dominar la programación orientada a objetos en Dart. Practica creando clases, objetos y usando la herencia para construir una base sólida.
