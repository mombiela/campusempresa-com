Introducción a las Listas

Las listas son una estructura de datos fundamental en Dart, utilizada para almacenar colecciones ordenadas de elementos. Son similares a los arrays en otros lenguajes de programación, pero vienen con más flexibilidad y funcionalidad.

Conceptos Clave

  • Inicialización de Listas: Crear listas usando diferentes métodos.
  • Acceso a Elementos: Recuperar elementos de una lista.
  • Modificación de Listas: Agregar, eliminar y actualizar elementos.
  • Iteración sobre Listas: Usar bucles para recorrer listas.
  • Métodos Comunes de Listas: Métodos incorporados útiles para la manipulación de listas.

Inicialización de Listas

En Dart, las listas se pueden crear de varias maneras. Aquí hay algunos métodos comunes:

Usando Literales de Lista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  print(numbers); // Output: [1, 2, 3, 4, 5]
}
  • Explicación: Esto crea una lista de enteros usando un literal de lista.

Usando el Constructor de Listas

void main() {
  List<String> fruits = List<String>.filled(3, 'Apple');
  print(fruits); // Output: [Apple, Apple, Apple]
}
  • Explicación: Esto crea una lista de cadenas con una longitud fija de 3, todas inicializadas a 'Apple'.

Acceso a Elementos

Puedes acceder a los elementos en una lista usando su índice, que comienza en 0.

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits[1]); // Output: Banana
}
  • Explicación: Esto recupera el elemento en el índice 1 de la lista.

Modificación de Listas

Agregar Elementos

  • Usando el Método add:
void main() {
  List<int> numbers = [1, 2, 3];
  numbers.add(4);
  print(numbers); // Output: [1, 2, 3, 4]
}

  • Explicación: Agrega un solo elemento al final de la lista.

  • Usando el Método addAll:

void main() {
  List<int> numbers = [1, 2, 3];
  numbers.addAll([4, 5, 6]);
  print(numbers); // Output: [1, 2, 3, 4, 5, 6]
}
  • Explicación: Agrega múltiples elementos al final de la lista.

Eliminar Elementos

  • Usando el Método remove:
void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits.remove('Banana');
  print(fruits); // Output: [Apple, Cherry]
}

  • Explicación: Elimina la primera ocurrencia del elemento especificado.

  • Usando el Método removeAt:

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits.removeAt(1);
  print(fruits); // Output: [Apple, Cherry]
}
  • Explicación: Elimina el elemento en el índice especificado.

Actualizar Elementos

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits[1] = 'Blueberry';
  print(fruits); // Output: [Apple, Blueberry, Cherry]
}
  • Explicación: Actualiza el elemento en el índice 1 a 'Blueberry'.

Iteración sobre Listas

Usando un Bucle `for`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  for (int i = 0; i < fruits.length; i++) {
    print(fruits[i]);
  }
}
  • Explicación: Recorre la lista usando un bucle for tradicional.

Usando un Bucle `for-in`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  for (var fruit in fruits) {
    print(fruit);
  }
}
  • Explicación: Recorre la lista usando un bucle for-in, que es más conciso.

Métodos Comunes de Listas

`contains`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits.contains('Banana')); // Output: true
}
  • Explicación: Verifica si la lista contiene el elemento especificado.

`indexOf`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits.indexOf('Cherry')); // Output: 2
}
  • Explicación: Devuelve el índice de la primera ocurrencia del elemento especificado.

`sort`

void main() {
  List<int> numbers = [3, 1, 4, 1, 5, 9];
  numbers.sort();
  print(numbers); // Output: [1, 1, 3, 4, 5, 9]
}
  • Explicación: Ordena la lista en orden ascendente.

`reversed`

void main() {
  List<int> numbers = [1, 2, 3];
  var reversedNumbers = numbers.reversed;
  print(reversedNumbers); // Output: (3, 2, 1)
}
  • Explicación: Devuelve un iterable con los elementos en orden inverso.

Conclusión

Las listas son una parte versátil y esencial de la programación en Dart. Te permiten almacenar y manipular colecciones ordenadas de elementos de manera eficiente. Al comprender cómo inicializar, acceder, modificar e iterar sobre listas, así como utilizar métodos comunes de listas, puedes manejar una amplia gama de tareas de programación de manera más efectiva. Practica estos conceptos con varios ejemplos para volverte competente en el uso de listas en Dart.
