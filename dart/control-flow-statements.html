Declaraciones de Control de Flujo en Dart
Funciones ►

Las declaraciones de control de flujo son fundamentales en la programación ya que te permiten dictar el orden en el que se ejecutan las declaraciones. En Dart, las declaraciones de control de flujo incluyen condicionales, bucles y declaraciones de ramificación. Esta sección cubrirá estos conceptos desde un nivel principiante hasta uno avanzado.

Condicionales

Los condicionales se utilizan para ejecutar código basado en ciertas condiciones.

Declaraciones if y else

La declaración if te permite ejecutar un bloque de código si una condición es verdadera. La declaración else se puede usar para ejecutar un bloque de código si la condición es falsa.

void main() {
  int number = 10;

  if (number > 0) {
    print('El número es positivo.');
  } else {
    print('El número no es positivo.');
  }
}

Declaración else if

La declaración else if te permite verificar múltiples condiciones.

void main() {
  int number = 0;

  if (number > 0) {
    print('El número es positivo.');
  } else if (number == 0) {
    print('El número es cero.');
  } else {
    print('El número es negativo.');
  }
}

Operador Ternario

El operador ternario es una forma abreviada de las declaraciones if-else.

void main() {
  int number = 10;
  String result = number > 0 ? 'Positivo' : 'No positivo';
  print(result);
}

Switch Case

La declaración switch te permite ejecutar un bloque de código entre muchas opciones.

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excelente');
      break;
    case 'B':
      print('Bueno');
      break;
    case 'C':
      print('Regular');
      break;
    case 'D':
      print('Pobre');
      break;
    default:
      print('Reprobado');
  }
}

Bucles

Los bucles se utilizan para ejecutar un bloque de código múltiples veces.

Bucle for

El bucle for se utiliza cuando sabes de antemano cuántas veces quieres ejecutar una declaración o un bloque de declaraciones.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteración $i');
  }
}

Bucle while

El bucle while se utiliza cuando quieres ejecutar un bloque de código mientras una condición sea verdadera.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteración $i');
    i++;
  }
}

Bucle do-while

El bucle do-while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez.

void main() {
  int i = 0;

  do {
    print('Iteración $i');
    i++;
  } while (i < 5);
}

Declaraciones de Ramificación

Las declaraciones de ramificación te permiten cambiar el flujo de ejecución.

Declaración break

La declaración break se utiliza para salir de un bucle o una declaración switch.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      break;
    }
    print('Iteración $i');
  }
}

Declaración continue

La declaración continue salta la iteración actual de un bucle y procede a la siguiente iteración.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      continue;
    }
    print('Iteración $i');
  }
}

Conclusión

Las declaraciones de control de flujo son esenciales para dirigir la ejecución de tu código. Entender cómo usar condicionales, bucles y declaraciones de ramificación te permitirá escribir programas en Dart más complejos y eficientes. Practica estos conceptos con varios ejemplos para volverte competente en el control del flujo de tus aplicaciones Dart.
