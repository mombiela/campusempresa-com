Descripción general

Tableau Server es una herramienta poderosa que permite a las organizaciones compartir, distribuir y colaborar en visualizaciones de datos creadas en Tableau Desktop. Esta sección te introducirá a los conceptos, características y funcionalidades principales de Tableau Server.

Conceptos clave

  • Tableau Server: Una plataforma para compartir y gestionar libros de trabajo, tableros y fuentes de datos de Tableau.
  • Libros de trabajo: Colecciones de tableros, hojas e historias creadas en Tableau Desktop.
  • Tableros: Visualizaciones de datos interactivas que combinan múltiples hojas.
  • Fuentes de datos: Conexiones a datos que pueden ser usadas en libros de trabajo de Tableau.
  • Usuarios y grupos: Individuos y colecciones de individuos que tienen acceso a Tableau Server.
  • Permisos: Reglas que definen lo que los usuarios y grupos pueden hacer con el contenido en Tableau Server.
  • Proyectos: Unidades organizativas dentro de Tableau Server usadas para gestionar contenido.

Configuración de Tableau Server

Instalación

  1. Requisitos del sistema: Asegúrate de que tu sistema cumpla con los requisitos de hardware y software.
  2. Descargar e instalar: Obtén el instalador de Tableau Server desde el sitio web de Tableau y sigue las instrucciones de instalación.
  3. Configuración: Configura los ajustes de Tableau Server como puertos, métodos de autenticación y notificaciones por correo electrónico.

Configuración inicial

  • Cuenta de administrador: Crea una cuenta de administrador durante el proceso de instalación.
  • Licenciamiento: Activa tu licencia de Tableau Server.
  • Conexiones de datos: Configura conexiones a tus fuentes de datos.

Gestión de usuarios y permisos

Añadir usuarios

  • Adición manual: Añade usuarios individualmente a través de la interfaz de Tableau Server.
  • Importación masiva: Usa archivos CSV para importar múltiples usuarios a la vez.
  • Integración con Active Directory: Integra con Active Directory para la gestión de usuarios.

Configuración de permisos

  • Permisos de contenido: Define qué acciones pueden realizar los usuarios en libros de trabajo, tableros y fuentes de datos.
  • Permisos de proyectos: Establece permisos a nivel de proyecto para gestionar el acceso a todo el contenido dentro de un proyecto.

Publicación y compartición de contenido

Publicación de libros de trabajo

  1. Desde Tableau Desktop: Publica libros de trabajo directamente desde Tableau Desktop a Tableau Server.
  2. Autoría web: Crea y edita libros de trabajo directamente en Tableau Server.

Compartición y colaboración

  • Vistas: Comparte vistas específicas de tableros con usuarios.
  • Suscripciones: Permite a los usuarios suscribirse a tableros y recibir actualizaciones por correo electrónico.
  • Comentarios: Habilita a los usuarios para comentar en los tableros para discusiones colaborativas.

Monitoreo y mantenimiento

Monitoreo del estado del servidor

  • Estado de Tableau Server: Usa la página de estado de Tableau Server para monitorear la salud y el rendimiento del servidor.
  • Registros: Revisa los registros del servidor para la solución de problemas y análisis de rendimiento.

Mantenimiento regular

  • Copias de seguridad: Realiza copias de seguridad regularmente de los datos de Tableau Server.
  • Actualizaciones: Mantén Tableau Server actualizado con los últimos parches y versiones.

Conclusión

Tableau Server es una herramienta esencial para las organizaciones que buscan aprovechar al máximo el poder de sus visualizaciones de datos. Al comprender los conceptos clave, configurar el servidor, gestionar usuarios y permisos, y publicar y compartir contenido de manera efectiva, puedes asegurarte de que tu organización maximice el valor de sus conocimientos de datos. El monitoreo y mantenimiento regular ayudarán a mantener tu Tableau Server funcionando de manera fluida y eficiente.
