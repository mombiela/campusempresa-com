Introducción a los Cálculos de Tabla

Los cálculos de tabla en Tableau son transformaciones que puedes aplicar a los valores en una visualización. Se calculan en función de los datos en la vista y se pueden usar para crear cálculos complejos sin necesidad de alterar la fuente de datos subyacente.

Conceptos Clave

  • Cálculo de Tabla: Una transformación aplicada a los valores en la vista.
  • Alcance: El subconjunto de datos utilizado para calcular el cálculo.
  • Dirección: El orden en el que se aplica el cálculo.

Tipos de Cálculos de Tabla

Tableau proporciona varios cálculos de tabla integrados que se pueden usar para realizar tareas comunes.

Cálculos de Tabla Comunes

  • Total Acumulado: Calcula un total acumulado.
  • Porcentaje del Total: Calcula el porcentaje que cada valor contribuye al total.
  • Diferencia: Calcula la diferencia entre valores.
  • Diferencia Porcentual: Calcula la diferencia porcentual entre valores.
  • Rango: Asigna un rango a cada valor.

Ejemplo: Total Acumulado

Para crear un total acumulado, sigue estos pasos:

  1. Arrastra una medida al estante de Filas.
  2. Haz clic derecho en la medida y selecciona "Cálculo de Tabla Rápido" > "Total Acumulado".

Cálculos de Tabla Personalizados

Además de los cálculos integrados, puedes crear cálculos de tabla personalizados usando campos calculados.

Crear un Cálculo de Tabla Personalizado

  1. Haz clic derecho en el panel de Datos y selecciona "Crear Campo Calculado".
  2. Ingresa un nombre para el cálculo.
  3. Ingresa una fórmula usando funciones de cálculo de tabla.

Ejemplo: Promedio Móvil Personalizado

Para crear un promedio móvil de 3 meses:

  1. Crea un campo calculado con la siguiente fórmula:
WINDOW_AVG(SUM([Sales]), -2, 0)

Esta fórmula calcula el promedio del mes actual y los dos meses anteriores.

Cálculos de Tabla Avanzados

Los cálculos de tabla avanzados implican transformaciones más complejas y a menudo requieren una comprensión más profunda de los datos y las funciones de cálculo.

Cálculos de Tabla Anidados

Los cálculos de tabla anidados implican usar un cálculo de tabla como parte de otro.

Ejemplo: Cálculo de Tabla Anidado

  1. Crea un campo calculado para el total acumulado:
RUNNING_SUM(SUM([Sales]))
  1. Crea otro campo calculado para el porcentaje del total basado en el total acumulado:
SUM([Sales]) / RUNNING_SUM(SUM([Sales]))

Ejercicios Prácticos

Ejercicio 1: Crear un Total Acumulado

  1. Carga el conjunto de datos de Superstore.
  2. Crea un gráfico de líneas que muestre las ventas mensuales.
  3. Agrega un cálculo de tabla de total acumulado a la medida de ventas.

Ejercicio 2: Calcular Porcentaje del Total

  1. Carga el conjunto de datos de Superstore.
  2. Crea un gráfico de barras que muestre las ventas por categoría.
  3. Agrega un cálculo de tabla de porcentaje del total a la medida de ventas.

Ejercicio 3: Promedio Móvil Personalizado

  1. Carga el conjunto de datos de Superstore.
  2. Crea un gráfico de líneas que muestre las ventas mensuales.
  3. Crea un campo calculado para un promedio móvil de 3 meses y agrégalo al gráfico.

Conclusión

Los cálculos de tabla en Tableau son herramientas poderosas para transformar y analizar datos directamente dentro de tus visualizaciones. Al comprender y utilizar tanto los cálculos de tabla integrados como los personalizados, puedes realizar análisis complejos sin alterar tu fuente de datos subyacente. Practica con diferentes tipos de cálculos de tabla para volverte competente en usarlos y obtener conocimientos más profundos de tus datos.
