Introducción

En esta sección, cubriremos los conceptos básicos para configurar un servidor GraphQL. Usaremos Node.js y Express junto con las bibliotecas graphql y express-graphql para crear un servidor GraphQL simple. Al final de esta sección, tendrás un servidor GraphQL funcional que podrás consultar.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente instalado:

  • Node.js y npm
  • Conocimientos básicos de JavaScript y Node.js

Paso 1: Inicializar un Proyecto Node.js

Primero, crea un nuevo directorio para tu proyecto y navega dentro de él. Luego, inicializa un nuevo proyecto Node.js.

mkdir graphql-server
cd graphql-server
npm init -y

Paso 2: Instalar Paquetes Requeridos

A continuación, instala los paquetes necesarios: express, graphql y express-graphql.

npm install express graphql express-graphql

Paso 3: Crear el Servidor

Crea un nuevo archivo llamado server.js y configura un servidor básico de Express.

const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

const app = express();
const port = 4000;

// Definir un esquema
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// Definir un resolver
const root = {
  hello: () => {
    return '¡Hola, mundo!';
  },
};

// Configurar el endpoint de GraphQL
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(port, () => {
  console.log(`El servidor está corriendo en http://localhost:${port}/graphql`);
});

Explicación

  • Configuración de Express: Creamos una aplicación de Express y la configuramos para escuchar en el puerto 4000.
  • Esquema de GraphQL: Definimos un esquema simple con una única consulta hello que devuelve un String.
  • Resolver: Creamos un resolver para la consulta hello que devuelve la cadena '¡Hola, mundo!'.
  • Endpoint de GraphQL: Configuramos el endpoint /graphql usando express-graphql y habilitamos la interfaz GraphiQL para pruebas.

Paso 4: Ejecutar el Servidor

Ejecuta el servidor usando Node.js.

node server.js

Deberías ver el mensaje El servidor está corriendo en http://localhost:4000/graphql en tu terminal. Abre tu navegador y navega a http://localhost:4000/graphql. Deberías ver la interfaz de GraphiQL donde puedes ejecutar la siguiente consulta:

{
  hello
}

Deberías obtener la siguiente respuesta:

{
  "data": {
    "hello": "¡Hola, mundo!"
  }
}

Conclusión

En esta sección, hemos configurado con éxito un servidor GraphQL básico usando Node.js, Express y la biblioteca express-graphql. Definimos un esquema y un resolver simples, y probamos nuestra configuración usando la interfaz GraphiQL. Esta configuración fundamental servirá como base para temas más avanzados en GraphQL.
