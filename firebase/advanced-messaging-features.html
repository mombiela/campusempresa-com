Introducción

Firebase Cloud Messaging (FCM) es una herramienta poderosa para enviar notificaciones y mensajes a usuarios en diferentes plataformas. En esta sección, exploraremos funciones avanzadas de FCM que permiten estrategias de mensajería más sofisticadas.

Conceptos Clave

  • Mensajería por Temas: Te permite enviar mensajes a múltiples dispositivos que se han suscrito a un tema en particular.
  • Mensajería de Grupo de Dispositivos: Permite enviar mensajes a un grupo de dispositivos que pertenecen a un solo usuario.
  • Prioridades de Mensajes: Controla la velocidad de entrega y las condiciones de los mensajes.
  • Programación de Mensajes: Programa mensajes para ser enviados en un momento específico.
  • Filtrado de Mensajes: Filtra mensajes basados en propiedades o condiciones del usuario.

Mensajería por Temas

La mensajería por temas te permite enviar un mensaje a múltiples dispositivos que se han suscrito a un tema en particular. Esto es útil para transmitir mensajes a una gran audiencia.

Ejemplo

// Suscribirse a un tema
firebase.messaging().subscribeToTopic(token, "news");

// Enviar un mensaje al tema
const message = {
  notification: {
    title: "Noticias de Última Hora",
    body: "¡Nuevo artículo disponible!"
  },
  topic: "news"
};

admin.messaging().send(message)
  .then((response) => {
    console.log("Mensaje enviado exitosamente:", response);
  })
  .catch((error) => {
    console.log("Error al enviar el mensaje:", error);
  });

En este ejemplo, los dispositivos se suscriben al tema "news" y reciben notificaciones cuando se envía un nuevo mensaje a ese tema.

Mensajería de Grupo de Dispositivos

La mensajería de grupo de dispositivos te permite enviar un mensaje a múltiples dispositivos que pertenecen a un solo usuario. Esto es útil para dirigirse a todos los dispositivos de un usuario en particular.

Ejemplo

// Crear una clave de notificación para un grupo de dispositivos
const notificationKey = "your-notification-key";
const registrationTokens = ["token1", "token2", "token3"];

const message = {
  notification: {
    title: "Hola Usuario",
    body: "Este es un mensaje para todos tus dispositivos."
  },
  notification_key: notificationKey
};

admin.messaging().sendToDeviceGroup(notificationKey, message)
  .then((response) => {
    console.log("Mensaje enviado exitosamente:", response);
  })
  .catch((error) => {
    console.log("Error al enviar el mensaje:", error);
  });

En este ejemplo, se envía un mensaje a todos los dispositivos asociados con una clave de notificación en particular.

Prioridades de Mensajes

FCM te permite establecer la prioridad de los mensajes para controlar su velocidad de entrega y condiciones.

Niveles de Prioridad

  • Alta Prioridad: Los mensajes se entregan inmediatamente y despiertan el dispositivo si es necesario.
  • Prioridad Normal: Los mensajes se entregan cuando el dispositivo está activo.

Ejemplo

const message = {
  notification: {
    title: "Actualización Urgente",
    body: "Por favor, revisa la aplicación para obtener información importante."
  },
  android: {
    priority: "high"
  },
  token: "user-device-token"
};

admin.messaging().send(message)
  .then((response) => {
    console.log("Mensaje de alta prioridad enviado exitosamente:", response);
  })
  .catch((error) => {
    console.log("Error al enviar el mensaje:", error);
  });

En este ejemplo, se envía un mensaje de alta prioridad a un dispositivo específico.

Programación de Mensajes

Puedes programar mensajes para ser enviados en un momento específico usando el parámetro sendAt.

Ejemplo

const message = {
  notification: {
    title: "Mensaje Programado",
    body: "Este mensaje fue programado."
  },
  android: {
    ttl: 3600 * 1000  // 1 hora en milisegundos
  },
  token: "user-device-token"
};

const sendTime = new Date().getTime() + 3600 * 1000; // 1 hora a partir de ahora

admin.messaging().send(message, { sendAt: sendTime })
  .then((response) => {
    console.log("Mensaje programado exitosamente:", response);
  })
  .catch((error) => {
    console.log("Error al programar el mensaje:", error);
  });

En este ejemplo, se programa un mensaje para ser enviado 1 hora después del tiempo actual.

Filtrado de Mensajes

El filtrado de mensajes te permite enviar mensajes basados en propiedades o condiciones del usuario.

Ejemplo

const message = {
  notification: {
    title: "Mensaje Filtrado",
    body: "Este mensaje es solo para usuarios premium."
  },
  condition: "'premiumUser' in topics"
};

admin.messaging().send(message)
  .then((response) => {
    console.log("Mensaje filtrado enviado exitosamente:", response);
  })
  .catch((error) => {
    console.log("Error al enviar el mensaje:", error);
  });

En este ejemplo, se envía un mensaje solo a los usuarios que están suscritos al tema "premiumUser".

Conclusión

Las funciones avanzadas de mensajería en Firebase Cloud Messaging proporcionan herramientas poderosas para dirigir y entregar mensajes a los usuarios. Al aprovechar los temas, grupos de dispositivos, prioridades de mensajes, programación y filtrado, puedes crear estrategias de mensajería sofisticadas que mejoran el compromiso y la experiencia del usuario.
