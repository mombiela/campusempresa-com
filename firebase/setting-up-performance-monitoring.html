Introducción al Monitoreo de Rendimiento

El Monitoreo de Rendimiento en Firebase te ayuda a obtener información sobre las características de rendimiento de tu aplicación. Te permite rastrear y analizar el rendimiento de tu aplicación en tiempo real, ayudándote a identificar y solucionar problemas de rendimiento.

Conceptos Clave

  • Rastreo: Un rastreo es un informe que contiene datos capturados entre dos puntos en el tiempo en tu aplicación.
  • Métrica: Una métrica es una medición de datos de rendimiento capturados durante un rastreo.
  • Rastreo Personalizado: Los rastreos personalizados son definidos por ti para medir el rendimiento de partes específicas de tu aplicación.
  • Rastreo Automático: Estos son rastreos predefinidos que Firebase recopila automáticamente para ti.

Configuración del Monitoreo de Rendimiento en Firebase

Paso 1: Agregar Firebase a Tu Proyecto

Antes de poder usar el Monitoreo de Rendimiento, necesitas agregar Firebase a tu proyecto.

Para Android

  1. Agrega el SDK de Firebase a tus archivos build.gradle.

    // build.gradle a nivel de proyecto
buildscript {
    dependencies {
        // Agrega el classpath de los servicios de Google
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// build.gradle a nivel de aplicación
dependencies {
    // Agrega el SDK de Monitoreo de Rendimiento
    implementation 'com.google.firebase:firebase-perf:20.0.3'
}

  2. Inicializa Firebase en tu clase Application.

    public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();
        // Inicializa Firebase
        FirebaseApp.initializeApp(this);
    }
}

Para iOS

  1. Agrega el SDK de Firebase a tu Podfile.

    pod 'Firebase/Performance'

  2. Inicializa Firebase en tu AppDelegate.

    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicializa Firebase
        FirebaseApp.configure()
        return true
    }
}

Paso 2: Habilitar el Monitoreo de Rendimiento en la Consola de Firebase

  1. Ve a la Consola de Firebase.
  2. Selecciona tu proyecto.
  3. Navega a la sección de Monitoreo de Rendimiento.
  4. Haz clic en el botón "Habilitar Monitoreo de Rendimiento".

Paso 3: Verificar la Instalación

Para verificar que el Monitoreo de Rendimiento esté configurado correctamente:

  • Ejecuta tu aplicación.
  • Realiza algunas acciones en tu aplicación para generar datos de rendimiento.
  • Revisa la Consola de Firebase para ver si aparecen los datos.

Usando el Monitoreo de Rendimiento

Rastreos Automáticos

Firebase recopila automáticamente varios rastreos para ti, tales como:

  • Rastreo de inicio de la aplicación: Mide el tiempo que tarda en iniciar tu aplicación.
  • Rastreo de renderizado de pantalla: Mide el tiempo que tarda en renderizar una pantalla.

Rastreos Personalizados

Puedes definir rastreos personalizados para medir partes específicas de tu aplicación.

Ejemplo: Rastreo Personalizado en Android

// Inicia un rastreo personalizado
Trace myTrace = FirebasePerformance.getInstance().newTrace("test_trace");
myTrace.start();

// Código que quieres medir
doSomeWork();

// Detén el rastreo personalizado
myTrace.stop();

Ejemplo: Rastreo Personalizado en iOS

// Inicia un rastreo personalizado
let trace = Performance.startTrace(name: "test_trace")

// Código que quieres medir
doSomeWork()

// Detén el rastreo personalizado
trace.stop()

Agregando Métricas Personalizadas

Puedes agregar métricas personalizadas a tus rastreos para medir eventos específicos.

Ejemplo: Métrica Personalizada en Android

// Inicia un rastreo personalizado
Trace myTrace = FirebasePerformance.getInstance().newTrace("test_trace");
myTrace.start();

// Incrementa una métrica personalizada
myTrace.incrementMetric("custom_metric", 1);

// Detén el rastreo personalizado
myTrace.stop();

Ejemplo: Métrica Personalizada en iOS

// Inicia un rastreo personalizado
let trace = Performance.startTrace(name: "test_trace")

// Incrementa una métrica personalizada
trace.incrementMetric("custom_metric", by: 1)

// Detén el rastreo personalizado
trace.stop()

Conclusión

Configurar el Monitoreo de Rendimiento en Firebase es un proceso sencillo que puede proporcionar información valiosa sobre el rendimiento de tu aplicación. Al aprovechar tanto los rastreos automáticos como los personalizados, puedes monitorear y mejorar el rendimiento de tu aplicación, asegurando una mejor experiencia de usuario. Recuerda revisar regularmente la Consola de Firebase para obtener datos de rendimiento y utilizarlos para identificar y abordar cualquier problema de rendimiento.
