Conjuntos en Delphi/Object Pascal
Tipos Enumerados ►

Introducción a los Conjuntos

Los conjuntos en Delphi/Object Pascal son colecciones de elementos del mismo tipo, típicamente utilizados para representar un grupo de valores relacionados. Son particularmente útiles para manejar un rango fijo de valores de manera eficiente.

Conceptos Clave

  • Definición: Un conjunto es una colección de valores del mismo tipo.
  • Tipo de Elemento: El tipo de elementos en un conjunto debe ser un tipo ordinal (por ejemplo, enteros, caracteres, enumeraciones).
  • Rango: Los conjuntos solo pueden contener valores dentro de un rango específico.

Declaración de Conjuntos

Para declarar un conjunto en Delphi/Object Pascal, se utiliza la palabra clave set of seguida del tipo de elementos que contendrá el conjunto.

Ejemplo

type
  TDays = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);
  TWorkDays = set of TDays;

En este ejemplo:

  • TDays es una enumeración que representa los días de la semana.
  • TWorkDays es un conjunto que puede contener cualquier combinación de valores de TDays.

Trabajando con Conjuntos

Creación e Inicialización de Conjuntos

Puedes crear e inicializar conjuntos usando constructores de conjuntos, que están encerrados entre corchetes.

Ejemplo

var
  WorkDays: TWorkDays;
begin
  WorkDays := [Mon, Tue, Wed, Thu, Fri];
end;

En este ejemplo, WorkDays se inicializa para contener de lunes a viernes.

Agregar y Eliminar Elementos

Puedes agregar o eliminar elementos de un conjunto usando los operadores + y -.

Ejemplo

begin
  WorkDays := WorkDays + [Sat];  // Agregar sábado
  WorkDays := WorkDays - [Mon];  // Eliminar lunes
end;

Verificación de Pertenencia

Para verificar si un elemento es parte de un conjunto, usa el operador in.

Ejemplo

begin
  if Mon in WorkDays then
    WriteLn('Monday is a workday.')
  else
    WriteLn('Monday is not a workday.');
end;

Operaciones con Conjuntos

Delphi/Object Pascal soporta varias operaciones con conjuntos, incluyendo unión, intersección y diferencia.

Unión

La unión de dos conjuntos contiene todos los elementos que están en cualquiera de los conjuntos.

Ejemplo

var
  Weekend: TWorkDays;
begin
  Weekend := [Sat, Sun];
  WorkDays := WorkDays + Weekend;  // Unión de WorkDays y Weekend
end;

Intersección

La intersección de dos conjuntos contiene solo los elementos que están en ambos conjuntos.

Ejemplo

var
  CommonDays: TWorkDays;
begin
  CommonDays := WorkDays * [Mon, Tue, Wed];  // Intersección de WorkDays y [Mon, Tue, Wed]
end;

Diferencia

La diferencia de dos conjuntos contiene los elementos que están en el primer conjunto pero no en el segundo.

Ejemplo

begin
  WorkDays := WorkDays - [Fri];  // Diferencia de WorkDays y [Fri]
end;

Características Avanzadas de los Conjuntos

Tipos de Subrango

Los conjuntos pueden definirse sobre subrangos de tipos ordinales para limitar su rango.

Ejemplo

type
  TSmallNumbers = 1..10;
  TSmallNumberSet = set of TSmallNumbers;

Conjunto de Caracteres

Los conjuntos también pueden usarse con caracteres, lo cual es útil para tareas como verificar entradas válidas.

Ejemplo

var
  Vowels: set of Char;
begin
  Vowels := ['A', 'E', 'I', 'O', 'U', 'a', 'e', 'i', 'o', 'u'];
  if 'A' in Vowels then
    WriteLn('A is a vowel.');
end;

Conclusión

Los conjuntos en Delphi/Object Pascal proporcionan una forma poderosa de manejar colecciones de valores ordinales de manera eficiente. Soportan varias operaciones como unión, intersección y diferencia, lo que los hace versátiles para muchas tareas de programación. Entender cómo declarar, inicializar y manipular conjuntos es esencial para una programación efectiva en Delphi/Object Pascal.
