Declaraciones condicionales en Delphi/Object Pascal
Bucles: For, While y Repeat ►

Las declaraciones condicionales son fundamentales en la programación, ya que permiten la ejecución de código basado en ciertas condiciones. En Delphi/Object Pascal, las declaraciones condicionales incluyen if, case y else. Este tema cubrirá estas declaraciones desde un nivel principiante hasta un nivel avanzado.

Declaraciones If

La declaración if se usa para ejecutar un bloque de código si una condición especificada es verdadera.

Declaración If básica

La forma más simple de una declaración if verifica una sola condición.

var
  number: Integer;
begin
  number := 10;
  if number > 5 then
    WriteLn('Number is greater than 5');
end.

Declaración If-Else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

var
  number: Integer;
begin
  number := 3;
  if number > 5 then
    WriteLn('Number is greater than 5')
  else
    WriteLn('Number is 5 or less');
end.

Declaraciones If anidadas

Puedes anidar declaraciones if para verificar múltiples condiciones.

var
  number: Integer;
begin
  number := 7;
  if number > 5 then
  begin
    if number < 10 then
      WriteLn('Number is between 6 and 9');
  end
  else
    WriteLn('Number is 5 or less');
end.

Declaraciones Case

La declaración case se usa para ejecutar uno de varios bloques de código basado en el valor de una expresión.

Declaración Case básica

Una declaración case simple verifica el valor de una sola variable.

var
  grade: Char;
begin
  grade := 'B';
  case grade of
    'A': WriteLn('Excellent');
    'B': WriteLn('Good');
    'C': WriteLn('Average');
    'D': WriteLn('Below Average');
    'F': WriteLn('Fail');
  else
    WriteLn('Invalid grade');
  end;
end.

Declaración Case con rangos

La declaración case también puede manejar rangos de valores.

var
  score: Integer;
begin
  score := 85;
  case score of
    90..100: WriteLn('Grade A');
    80..89: WriteLn('Grade B');
    70..79: WriteLn('Grade C');
    60..69: WriteLn('Grade D');
    0..59: WriteLn('Grade F');
  else
    WriteLn('Invalid score');
  end;
end.

Declaraciones condicionales avanzadas

Combinando condiciones con operadores lógicos

Puedes combinar múltiples condiciones usando operadores lógicos como and, or y not.

var
  age: Integer;
  hasLicense: Boolean;
begin
  age := 18;
  hasLicense := True;
  if (age >= 18) and hasLicense then
    WriteLn('You can drive')
  else
    WriteLn('You cannot drive');
end.

Usando funciones en condiciones

Puedes llamar a funciones dentro de declaraciones condicionales para tomar decisiones basadas en lógica más compleja.

function IsEven(number: Integer): Boolean;
begin
  Result := (number mod 2) = 0;
end;

var
  number: Integer;
begin
  number := 4;
  if IsEven(number) then
    WriteLn('Number is even')
  else
    WriteLn('Number is odd');
end.

Conclusión

Las declaraciones condicionales son esenciales para controlar el flujo de un programa. Delphi/Object Pascal proporciona constructos poderosos como if, else y case para manejar varias condiciones. Al dominar estas declaraciones, puedes escribir programas más dinámicos y receptivos. Practica usando estos constructos con diferentes escenarios para volverte competente en su uso.
