Introducción a las Cadenas en Delphi/Object Pascal

Las cadenas son secuencias de caracteres utilizadas para representar texto en programación. En Delphi/Object Pascal, las cadenas son un tipo de dato fundamental, y entender cómo manipularlas es crucial para una programación efectiva.

Conceptos Clave

  • Tipos de Cadenas: Diferentes tipos de cadenas en Delphi, como ShortString, AnsiString, WideString y UnicodeString.
  • Declaración de Cadenas: Cómo declarar e inicializar cadenas.
  • Operaciones con Cadenas: Operaciones básicas como concatenación, comparación y determinación de longitud.

Tipos de Cadenas en Delphi

Delphi soporta varios tipos de cadenas, cada uno con sus características y casos de uso.

ShortString

  • Una cadena de longitud fija con un máximo de 255 caracteres.
  • Útil para la compatibilidad con código Pascal más antiguo.
var
  s: ShortString;
begin
  s := '¡Hola, Delphi!';
end;

AnsiString

  • Una cadena asignada dinámicamente que puede contener cualquier número de caracteres.
  • Adecuada para manejar texto ASCII.
var
  s: AnsiString;
begin
  s := '¡Hola, Delphi!';
end;

WideString

  • Una cadena asignada dinámicamente diseñada para contener caracteres Unicode.
  • Útil para la internacionalización.
var
  s: WideString;
begin
  s := '¡Hola, Delphi!';
end;

UnicodeString

  • El tipo de cadena predeterminado en las versiones modernas de Delphi.
  • Soporta completamente Unicode, lo que lo hace ideal para la mayoría de las aplicaciones.
var
  s: UnicodeString;
begin
  s := '¡Hola, Delphi!';
end;

Declaración e Inicialización de Cadenas

Declarar e inicializar cadenas en Delphi es sencillo.

var
  greeting: string;
begin
  greeting := '¡Hola, Mundo!';
end;

Operaciones Básicas con Cadenas

Concatenación

Combinar dos o más cadenas en una sola.

var
  s1, s2, result: string;
begin
  s1 := 'Hola, ';
  s2 := 'Mundo!';
  result := s1 + s2; // result es 'Hola, Mundo!'
end;

Comparación

Comparar dos cadenas para igualdad u orden.

var
  s1, s2: string;
  isEqual: Boolean;
begin
  s1 := 'Hola';
  s2 := 'Mundo';
  isEqual := s1 = s2; // isEqual es False
end;

Determinación de Longitud

Encontrar el número de caracteres en una cadena.

var
  s: string;
  len: Integer;
begin
  s := '¡Hola, Mundo!';
  len := Length(s); // len es 13
end;

Manejo Avanzado de Cadenas

Funciones de Cadenas

Delphi proporciona un conjunto rico de funciones integradas para la manipulación avanzada.

  • Copy: Extrae una subcadena de una cadena.
  • Pos: Encuentra la posición de una subcadena dentro de una cadena.
  • Delete: Elimina una parte de una cadena.
  • Insert: Inserta una subcadena en una cadena.
var
  s, sub: string;
  pos: Integer;
begin
  s := '¡Hola, Delphi!';
  sub := Copy(s, 8, 6); // sub es 'Delphi'
  pos := Pos('Delphi', s); // pos es 8
  Delete(s, 8, 6); // s es '¡Hola, !'
  Insert('Mundo', s, 8); // s es '¡Hola, Mundo!'
end;

Formateo de Cadenas

Formateo de cadenas usando la función Format.

var
  s: string;
begin
  s := Format('¡Hola, %s! Tienes %d mensajes nuevos.', ['Alice', 5]);
  // s es '¡Hola, Alice! Tienes 5 mensajes nuevos.'
end;

Conversión de Cadenas

Convertir entre diferentes tipos de datos y cadenas.

var
  i: Integer;
  s: string;
begin
  i := 123;
  s := IntToStr(i); // s es '123'
  i := StrToInt(s); // i es 123
end;

Conclusión

Las cadenas son una parte versátil y esencial de la programación en Delphi. Entender los diferentes tipos de cadenas, las operaciones básicas y las técnicas de manejo avanzado te permitirá manipular texto de manera efectiva en tus aplicaciones. Practica estos conceptos con varios ejercicios para solidificar tu comprensión y mejorar tu competencia en el manejo de cadenas.
