Introducción a los Arrays

Los arrays son una estructura de datos fundamental en Delphi/Object Pascal, que te permite almacenar múltiples valores del mismo tipo en una sola variable. Esta sección cubrirá los conceptos básicos de los arrays, incluyendo su declaración, inicialización y uso.

Conceptos Clave

  • Declaración de Arrays: Cómo declarar arrays en Delphi/Object Pascal.
  • Inicialización de Arrays: Cómo inicializar arrays con valores.
  • Acceso a Elementos del Array: Cómo leer y escribir elementos del array.
  • Arrays Estáticos vs. Dinámicos: Entender la diferencia entre arrays estáticos y dinámicos.

Declaración de Arrays

En Delphi/Object Pascal, los arrays pueden ser declarados de varias maneras dependiendo de su tipo y tamaño.

Arrays Estáticos

Los arrays estáticos tienen un tamaño fijo que se determina en tiempo de compilación.

var
  Numbers: array[1..5] of Integer;

En este ejemplo:

  • Numbers es un array que puede contener 5 enteros.
  • El índice varía de 1 a 5.

Arrays Dinámicos

Los arrays dinámicos pueden cambiar de tamaño en tiempo de ejecución.

var
  Numbers: array of Integer;

En este ejemplo:

  • Numbers es un array dinámico de enteros.
  • El tamaño del array no se especifica en la declaración.

Inicialización de Arrays

Después de declarar un array, puedes inicializarlo con valores.

Arrays Estáticos

Los arrays estáticos pueden ser inicializados elemento por elemento o usando un bucle.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  Numbers[2] := 20;
  Numbers[3] := 30;
  Numbers[4] := 40;
  Numbers[5] := 50;
end;

Arrays Dinámicos

Los arrays dinámicos necesitan ser redimensionados antes de la inicialización.

var
  Numbers: array of Integer;
begin
  SetLength(Numbers, 5); // Redimensionar el array para contener 5 elementos
  Numbers[0] := 10;
  Numbers[1] := 20;
  Numbers[2] := 30;
  Numbers[3] := 40;
  Numbers[4] := 50;
end;

Acceso a Elementos del Array

Puedes acceder y modificar elementos del array usando su índice.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  WriteLn(Numbers[1]); // Salida: 10
end;

Arrays Estáticos vs. Dinámicos

Entender las diferencias entre arrays estáticos y dinámicos es crucial para una gestión eficiente de la memoria y el rendimiento.

| Característica | Arrays Estáticos | Arrays Dinámicos | |-----------------|---------------------------------|---------------------------------| | Tamaño | Fijo en tiempo de compilación | Puede redimensionarse en tiempo de ejecución | | Asignación de Memoria | Asignado en la pila | Asignado en el montón | | Flexibilidad | Menos flexible | Más flexible | | Rendimiento | Acceso más rápido | Acceso más lento debido a la asignación en el montón |

Operaciones Avanzadas con Arrays

A medida que te vuelvas más competente, puedes realizar operaciones más complejas con arrays.

Arrays Multidimensionales

Delphi/Object Pascal soporta arrays multidimensionales.

var
  Matrix: array[1..3, 1..3] of Integer;
begin
  Matrix[1, 1] := 1;
  Matrix[1, 2] := 2;
  Matrix[1, 3] := 3;
  // Continuar inicializando otros elementos
end;

Arrays Multidimensionales Dinámicos

Los arrays multidimensionales dinámicos también pueden ser creados y redimensionados.

var
  Matrix: array of array of Integer;
begin
  SetLength(Matrix, 3, 3); // Crear una matriz de 3x3
  Matrix[0][0] := 1;
  Matrix[0][1] := 2;
  Matrix[0][2] := 3;
  // Continuar inicializando otros elementos
end;

Array de Registros

Puedes crear arrays de tipos complejos como registros.

type
  TPerson = record
    Name: string;
    Age: Integer;
  end;

var
  People: array[1..3] of TPerson;
begin
  People[1].Name := 'Alice';
  People[1].Age := 30;
  // Continuar inicializando otros elementos
end;

Conclusión

Los arrays son una estructura de datos poderosa y versátil en Delphi/Object Pascal. Entender cómo declarar, inicializar y manipular arrays es esencial para una programación eficiente. Ya sea que estés trabajando con arrays estáticos o dinámicos, dominar estos conceptos te permitirá manejar colecciones de datos de manera efectiva. A medida que avances, puedes explorar operaciones más complejas con arrays, como arrays multidimensionales y arrays de registros, para mejorar aún más tus habilidades de programación.
