Alcance y Duración de las Variables en Delphi/Object Pascal
Recursión ►

Entender el alcance y la duración de las variables es crucial para escribir código eficiente y libre de errores en Delphi/Object Pascal. Este tema cubrirá los conceptos básicos, proporcionará ejemplos e incluirá ejercicios para reforzar el aprendizaje.

Alcance de las Variables

El alcance de una variable se refiere a la región del código donde la variable es accesible. En Delphi/Object Pascal, hay varios tipos de alcance:

  • Alcance Local
  • Alcance Global
  • Alcance de Unidad
  • Alcance de Clase

Alcance Local

Las variables declaradas dentro de un procedimiento, función o método tienen alcance local. Solo son accesibles dentro de ese bloque de código.

procedure ExampleProcedure;
var
  LocalVar: Integer; // Variable local
begin
  LocalVar := 10;
  WriteLn(LocalVar); // Salida: 10
end;

Alcance Global

Las variables declaradas fuera de cualquier procedimiento, función o método tienen alcance global. Son accesibles desde cualquier parte del programa.

var
  GlobalVar: Integer; // Variable global

procedure ExampleProcedure;
begin
  GlobalVar := 20;
  WriteLn(GlobalVar); // Salida: 20
end;

begin
  GlobalVar := 10;
  ExampleProcedure;
  WriteLn(GlobalVar); // Salida: 20
end.

Alcance de Unidad

Las variables declaradas en la sección de interfaz de una unidad son accesibles desde otras unidades que usan esta unidad. Las variables en la sección de implementación solo son accesibles dentro de la misma unidad.

unit ExampleUnit;

interface

var
  UnitVar: Integer; // Variable de alcance de unidad

implementation

procedure ExampleProcedure;
begin
  UnitVar := 30;
  WriteLn(UnitVar); // Salida: 30
end;

end.

Alcance de Clase

Las variables declaradas dentro de una clase son accesibles para todos los métodos de esa clase. Estas también se conocen como campos o variables miembro.

type
  TExampleClass = class
  private
    FClassVar: Integer; // Variable de alcance de clase
  public
    procedure SetVar(Value: Integer);
    procedure ShowVar;
  end;

procedure TExampleClass.SetVar(Value: Integer);
begin
  FClassVar := Value;
end;

procedure TExampleClass.ShowVar;
begin
  WriteLn(FClassVar);
end;

var
  Example: TExampleClass;

begin
  Example := TExampleClass.Create;
  Example.SetVar(40);
  Example.ShowVar; // Salida: 40
  Example.Free;
end.

Duración de las Variables

La duración de una variable se refiere al tiempo durante el cual la variable existe en la memoria. En Delphi/Object Pascal, la duración de las variables depende de su alcance.

Variables Locales

Las variables locales se crean cuando se llama al procedimiento, función o método y se destruyen cuando este finaliza.

procedure ExampleProcedure;
var
  LocalVar: Integer;
begin
  LocalVar := 50;
  WriteLn(LocalVar); // Salida: 50
end;

begin
  ExampleProcedure;
  // LocalVar ya no es accesible aquí
end.

Variables Globales

Las variables globales existen durante toda la duración del programa.

var
  GlobalVar: Integer;

begin
  GlobalVar := 60;
  WriteLn(GlobalVar); // Salida: 60
  // GlobalVar existe hasta que el programa termina
end.

Variables de Unidad

Las variables de unidad tienen una duración que abarca la duración del programa, similar a las variables globales.

unit ExampleUnit;

interface

var
  UnitVar: Integer;

implementation

procedure ExampleProcedure;
begin
  UnitVar := 70;
  WriteLn(UnitVar); // Salida: 70
end;

end.

Variables de Clase

Las variables de clase (campos) existen mientras exista la instancia de la clase. Se crean cuando se crea la instancia y se destruyen cuando la instancia se libera.

type
  TExampleClass = class
  private
    FClassVar: Integer;
  public
    procedure SetVar(Value: Integer);
    procedure ShowVar;
  end;

procedure TExampleClass.SetVar(Value: Integer);
begin
  FClassVar := Value;
end;

procedure TExampleClass.ShowVar;
begin
  WriteLn(FClassVar);
end;

var
  Example: TExampleClass;

begin
  Example := TExampleClass.Create;
  Example.SetVar(80);
  Example.ShowVar; // Salida: 80
  Example.Free;
  // FClassVar ya no es accesible aquí
end.

Conclusión

Entender el alcance y la duración de las variables es esencial para gestionar la memoria y asegurar que tu código se comporte como se espera. Aquí tienes un resumen rápido:

  • Alcance Local: Las variables son accesibles solo dentro del bloque donde se declaran y existen solo durante la ejecución de ese bloque.
  • Alcance Global: Las variables son accesibles desde cualquier parte del programa y existen durante toda la duración del programa.
  • Alcance de Unidad: Las variables son accesibles dentro de la unidad y, si se declaran en la sección de interfaz, también desde otras unidades.
  • Alcance de Clase: Las variables son accesibles para todos los métodos de la clase y existen mientras exista la instancia de la clase.

Al dominar estos conceptos, podrás escribir código en Delphi/Object Pascal más eficiente y mantenible.
