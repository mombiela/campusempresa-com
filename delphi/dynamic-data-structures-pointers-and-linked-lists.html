Introducción

Las estructuras de datos dinámicas son esenciales para una gestión eficiente de la memoria y una manipulación flexible de los datos. En este tema, exploraremos punteros y listas enlazadas en Delphi/Object Pascal, comenzando desde lo básico y progresando hacia conceptos más avanzados.

Punteros en Delphi/Object Pascal

¿Qué es un Puntero?

  • Un puntero es una variable que contiene la dirección de memoria de otra variable.
  • Los punteros permiten la asignación dinámica de memoria y la manipulación de estructuras de datos como listas enlazadas.

Declaración y Uso de Punteros

var
  p: ^Integer; // Declara un puntero a un entero
  num: Integer;
begin
  num := 10;
  p := @num; // p ahora contiene la dirección de num
  WriteLn('Valor de num: ', num); // Imprime 10
  WriteLn('Valor apuntado por p: ', p^); // Imprime 10
end;
  • ^ se usa para declarar un tipo de puntero.
  • El operador @ se usa para obtener la dirección de una variable.
  • p^ desreferencia el puntero para acceder al valor almacenado en la dirección.

Asignación Dinámica de Memoria

  • Use New y Dispose para la asignación y desasignación dinámica de memoria.
var
  p: ^Integer;
begin
  New(p); // Asigna memoria para un entero
  p^ := 20;
  WriteLn('Valor apuntado por p: ', p^); // Imprime 20
  Dispose(p); // Desasigna la memoria
end;

Listas Enlazadas

¿Qué es una Lista Enlazada?

  • Una lista enlazada es una estructura de datos dinámica que consiste en nodos.
  • Cada nodo contiene datos y un puntero al siguiente nodo.

Tipos de Listas Enlazadas

  • Lista Enlazada Simple: Cada nodo apunta al siguiente nodo.
  • Lista Enlazada Doble: Cada nodo apunta tanto al siguiente como al nodo anterior.

Implementación de una Lista Enlazada Simple

Estructura del Nodo

type
  PNode = ^TNode;
  TNode = record
    Data: Integer;
    Next: PNode;
  end;
  • PNode es un puntero a TNode.
  • TNode contiene un campo de datos entero y un puntero al siguiente nodo.

Creación y Manipulación de una Lista Enlazada

var
  Head, Temp: PNode;

procedure AddNode(var Head: PNode; Value: Integer);
var
  NewNode: PNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := Head;
  Head := NewNode;
end;

procedure PrintList(Head: PNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

begin
  Head := nil;
  AddNode(Head, 10);
  AddNode(Head, 20);
  AddNode(Head, 30);
  PrintList(Head);
end;
  • El procedimiento AddNode añade un nuevo nodo al principio de la lista.
  • El procedimiento PrintList imprime todos los nodos de la lista.

Implementación de una Lista Enlazada Doble

Estructura del Nodo

type
  PDNode = ^TDNode;
  TDNode = record
    Data: Integer;
    Prev, Next: PDNode;
  end;
  • PDNode es un puntero a TDNode.
  • TDNode contiene un campo de datos entero, un puntero al nodo anterior y un puntero al siguiente nodo.

Creación y Manipulación de una Lista Enlazada Doble

var
  Head, Tail, Temp: PDNode;

procedure AddNode(var Head, Tail: PDNode; Value: Integer);
var
  NewNode: PDNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := nil;
  NewNode^.Prev := Tail;
  if Tail <> nil then
    Tail^.Next := NewNode;
  Tail := NewNode;
  if Head = nil then
    Head := NewNode;
end;

procedure PrintListForward(Head: PDNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

procedure PrintListBackward(Tail: PDNode);
begin
  Temp := Tail;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Prev;
  end;
end;

begin
  Head := nil;
  Tail := nil;
  AddNode(Head, Tail, 10);
  AddNode(Head, Tail, 20);
  AddNode(Head, Tail, 30);
  PrintListForward(Head);
  PrintListBackward(Tail);
end;
  • El procedimiento AddNode añade un nuevo nodo al final de la lista.
  • El procedimiento PrintListForward imprime todos los nodos de cabeza a cola.
  • El procedimiento PrintListBackward imprime todos los nodos de cola a cabeza.

Conclusión

Los punteros y las listas enlazadas son herramientas poderosas en Delphi/Object Pascal para gestionar estructuras de datos dinámicas. Entender cómo usar punteros para la asignación dinámica de memoria y cómo implementar y manipular listas enlazadas es crucial para la programación avanzada. Practique estos conceptos con varios ejercicios para solidificar su comprensión y mejorar sus habilidades de programación.
