Variables y Tipos de Datos en Delphi/Object Pascal
Operadores y Expresiones ►

Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y tipos de datos en Delphi/Object Pascal. Comprender estos conceptos es crucial para escribir código eficiente y sin errores. Cubriremos los conceptos básicos de la declaración de variables, los diferentes tipos de datos disponibles en Delphi y cómo usarlos de manera efectiva.

Variables

Las variables se utilizan para almacenar datos que pueden ser manipulados por el programa. En Delphi, las variables deben ser declaradas antes de que puedan ser utilizadas.

Declaración de Variables

Para declarar una variable en Delphi, se utiliza la palabra clave var seguida del nombre de la variable y su tipo.

var
  age: Integer;
  name: String;
  isStudent: Boolean;
  • age es una variable entera.
  • name es una variable de cadena.
  • isStudent es una variable booleana.

Reglas de Nomenclatura de Variables

  • Deben comenzar con una letra o guion bajo.
  • Pueden contener letras, dígitos y guiones bajos.
  • No pueden ser una palabra clave reservada.

Asignación de Valores

Los valores pueden ser asignados a las variables usando el operador de asignación (:=).

age := 25;
name := 'John Doe';
isStudent := True;

Tipos de Datos

Los tipos de datos definen el tipo de datos que una variable puede contener. Delphi proporciona varios tipos de datos integrados.

Tipos Enteros

Se utilizan para almacenar números enteros.

| Tipo | Rango | Tamaño (bytes) | |----------|------------------------------|----------------| | Byte | 0 a 255 | 1 | | ShortInt | -128 a 127 | 1 | | Integer | -2,147,483,648 a 2,147,483,647 | 4 | | LongInt | -2,147,483,648 a 2,147,483,647 | 4 | | Int64 | -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807 | 8 |

Tipos de Punto Flotante

Se utilizan para almacenar números con partes fraccionarias.

| Tipo | Rango | Tamaño (bytes) | |----------|------------------------------|----------------| | Single | 1.5 x 10^-45 a 3.4 x 10^38 | 4 | | Double | 5.0 x 10^-324 a 1.7 x 10^308 | 8 | | Extended | 3.6 x 10^-4951 a 1.1 x 10^4932 | 10 |

Tipo Booleano

Se utiliza para almacenar True o False.

var
  isActive: Boolean;
begin
  isActive := True;
end;

Tipos de Caracteres

Se utilizan para almacenar caracteres individuales o cadenas.

| Tipo | Descripción | |--------|-------------------------------| | Char | Carácter individual (1 byte) | | String | Secuencia de caracteres |

var
  initial: Char;
  fullName: String;
begin
  initial := 'A';
  fullName := 'Alice Johnson';
end;

Tipos Enumerados

Tipos de datos definidos por el usuario que consisten en un conjunto de valores nombrados.

type
  TDays = (Sun, Mon, Tue, Wed, Thu, Fri, Sat);
var
  today: TDays;
begin
  today := Mon;
end;

Tipos Subrango

Definen un subconjunto de un tipo ordinal existente.

type
  TAge = 0..120;
var
  myAge: TAge;
begin
  myAge := 30;
end;

Temas Avanzados

Registros

Los registros se utilizan para agrupar diferentes tipos de datos juntos.

type
  TPerson = record
    Name: String;
    Age: Integer;
    IsStudent: Boolean;
  end;
var
  person: TPerson;
begin
  person.Name := 'John Doe';
  person.Age := 25;
  person.IsStudent := True;
end;

Arreglos

Los arreglos se utilizan para almacenar múltiples valores del mismo tipo.

var
  numbers: array[1..5] of Integer;
begin
  numbers[1] := 10;
  numbers[2] := 20;
  numbers[3] := 30;
  numbers[4] := 40;
  numbers[5] := 50;
end;

Arreglos Dinámicos

Los arreglos dinámicos pueden cambiar de tamaño en tiempo de ejecución.

var
  dynArray: array of Integer;
begin
  SetLength(dynArray, 3);
  dynArray[0] := 1;
  dynArray[1] := 2;
  dynArray[2] := 3;
end;

Conclusión

Comprender las variables y los tipos de datos es fundamental para programar en Delphi/Object Pascal. Este conocimiento te permite almacenar y manipular datos de manera efectiva. A medida que avances, encontrarás estructuras de datos más complejas y aprenderás a usarlas para resolver problemas del mundo real. Practica declarando variables, asignando valores y usando diferentes tipos de datos para volverte competente en la programación en Delphi.
