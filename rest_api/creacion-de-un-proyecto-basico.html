Creación de un proyecto básico
Definición de rutas y controladores ►

En esta sección, aprenderemos cómo crear un proyecto básico para una REST API. Este proyecto servirá como base para construir y expandir funcionalidades en futuras lecciones. Utilizaremos Node.js y Express para este propósito.

Requisitos previos

Antes de comenzar, asegúrate de tener instalados los siguientes componentes:

  • Node.js (versión 14 o superior)
  • npm (Node Package Manager)

Paso 1: Configuración del entorno de desarrollo

Instalación de Node.js y npm

Si aún no tienes Node.js y npm instalados, puedes descargarlos e instalarlos desde nodejs.org.

Creación del directorio del proyecto

  1. Abre tu terminal o línea de comandos.

  2. Crea un nuevo directorio para tu proyecto y navega dentro de él:

    mkdir mi-proyecto-rest-api
cd mi-proyecto-rest-api

Inicialización del proyecto con npm

  1. Inicializa un nuevo proyecto de Node.js:

    npm init -y

    Esto creará un archivo package.json con la configuración predeterminada.

Paso 2: Instalación de dependencias

Para este proyecto, utilizaremos Express, un framework minimalista para Node.js que facilita la creación de APIs REST.

  1. Instala Express:

    npm install express

Paso 3: Creación del servidor básico

Estructura del proyecto

Organiza tu proyecto de la siguiente manera:

mi-proyecto-rest-api/
├── node_modules/
├── package.json
└── index.js

Configuración del servidor

  1. Crea un archivo index.js en el directorio raíz de tu proyecto.

  2. Abre index.js y agrega el siguiente código para configurar un servidor básico con Express:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('¡Hola, mundo!');
});

app.listen(port, () => {
    console.log(`Servidor escuchando en http://localhost:${port}`);
});

Ejecución del servidor

  1. En tu terminal, ejecuta el siguiente comando para iniciar el servidor:

    node index.js

  2. Abre tu navegador web y navega a http://localhost:3000. Deberías ver el mensaje "¡Hola, mundo!".

Paso 4: Creación de rutas básicas

Rutas GET

  1. Agrega una nueva ruta GET en index.js para manejar solicitudes a /api/saludo:

    app.get('/api/saludo', (req, res) => {
    res.json({ mensaje: '¡Hola desde la API!' });
});

Rutas POST

  1. Para manejar solicitudes POST, primero necesitamos configurar Express para que pueda procesar datos JSON. Agrega la siguiente línea antes de definir las rutas:

    app.use(express.json());

  2. Agrega una nueva ruta POST en index.js para manejar solicitudes a /api/saludo:

    app.post('/api/saludo', (req, res) => {
    const nombre = req.body.nombre;
    res.json({ mensaje: `¡Hola, ${nombre}!` });
});

Ejemplo de código completo

El archivo index.js completo debería verse así:

const express = require('express');
const app = express();
const port = 3000;

app.use(express.json());

app.get('/', (req, res) => {
    res.send('¡Hola, mundo!');
});

app.get('/api/saludo', (req, res) => {
    res.json({ mensaje: '¡Hola desde la API!' });
});

app.post('/api/saludo', (req, res) => {
    const nombre = req.body.nombre;
    res.json({ mensaje: `¡Hola, ${nombre}!` });
});

app.listen(port, () => {
    console.log(`Servidor escuchando en http://localhost:${port}`);
});

Conclusión

En esta lección, hemos configurado un proyecto básico de REST API utilizando Node.js y Express. Hemos aprendido a:

  • Configurar un entorno de desarrollo.
  • Crear un servidor básico.
  • Definir rutas GET y POST.

Este proyecto básico servirá como base para agregar más funcionalidades y explorar conceptos avanzados en futuras lecciones. ¡Sigue practicando y experimentando con nuevas rutas y funcionalidades!
