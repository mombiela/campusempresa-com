Introducción

La estadística es una herramienta fundamental en las ciencias sociales, ya que permite analizar datos y extraer conclusiones significativas sobre fenómenos sociales. Este tema abordará los conceptos clave de la estadística aplicada a las ciencias sociales, proporcionando ejemplos y ejercicios prácticos para facilitar la comprensión.

Conceptos Clave

  • Población y Muestra:

    • Población: Conjunto completo de individuos o elementos que se estudian.
    • Muestra: Subconjunto de la población que se selecciona para el estudio.

  • Variables:

    • Cualitativas: Describen cualidades o categorías (e.g., género, estado civil).
    • Cuantitativas: Describen cantidades numéricas (e.g., ingresos, edad).

  • Medidas de Tendencia Central:

    • Media: Promedio aritmético de un conjunto de datos.
    • Mediana: Valor central de un conjunto de datos ordenados.
    • Moda: Valor que aparece con mayor frecuencia en un conjunto de datos.

  • Medidas de Dispersión:

    • Rango: Diferencia entre el valor máximo y el mínimo.
    • Varianza: Medida de la dispersión de los datos respecto a la media.
    • Desviación estándar: Raíz cuadrada de la varianza.

  • Distribuciones de Probabilidad:

    • Distribución Normal: Distribución simétrica en forma de campana.
    • Distribución Binomial: Distribución de resultados de experimentos con dos posibles resultados (éxito o fracaso).

Ejemplos Prácticos

Ejemplo 1: Cálculo de la Media, Mediana y Moda

Supongamos que tenemos los siguientes datos sobre los ingresos mensuales (en dólares) de un grupo de individuos:

$500, $600, $700, $800, $900, $1000, $1100, $1200, $1300, $1400

  • Media:

    ingresos = [500, 600, 700, 800, 900, 1000, 1100, 1200, 1300, 1400]
media = sum(ingresos) / len(ingresos)
print("Media:", media)

  • Mediana:

    ingresos.sort()
n = len(ingresos)
mediana = (ingresos[n//2 - 1] + ingresos[n//2]) / 2 if n % 2 == 0 else ingresos[n//2]
print("Mediana:", mediana)

  • Moda:

    from collections import Counter
moda = Counter(ingresos).most_common(1)[0][0]
print("Moda:", moda)

Ejemplo 2: Distribución Normal

Supongamos que queremos generar una distribución normal de datos sobre la altura de individuos en una población:

import numpy as np
import matplotlib.pyplot as plt

# Generar datos
mu, sigma = 170, 10  # media y desviación estándar
alturas = np.random.normal(mu, sigma, 1000)

# Graficar
plt.hist(alturas, bins=30, density=True, alpha=0.6, color='g')
plt.title('Distribución Normal de Alturas')
plt.xlabel('Altura (cm)')
plt.ylabel('Frecuencia')
plt.show()

  2. Análisis de Distribución:

    • Genera una distribución binomial para un experimento con 10 ensayos y una probabilidad de éxito de 0.5. Grafica la distribución y analiza los resultados.

Conclusión

La estadística en ciencias sociales es esencial para interpretar datos y tomar decisiones informadas. Comprender conceptos como la media, mediana, moda, y las distribuciones de probabilidad permite a los profesionales analizar fenómenos sociales de manera efectiva. Los ejemplos y ejercicios proporcionados en este tema son una base sólida para aplicar la estadística en investigaciones y estudios sociales.
