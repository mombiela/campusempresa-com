Introducción a las Sentencias Case

Las sentencias case en Ada son una estructura de control utilizada para ejecutar un bloque de código de entre múltiples bloques posibles, basándose en el valor de una expresión. Son similares a las sentencias switch en otros lenguajes de programación como C o Java. Las sentencias case mejoran la legibilidad y el mantenimiento del código al delinear claramente diferentes caminos de ejecución.

Sintaxis Básica

La sintaxis básica de una sentencia case en Ada es la siguiente:

case expression is
   when choice1 =>
      -- Sentencias para choice1
   when choice2 =>
      -- Sentencias para choice2
   when others =>
      -- Sentencias para todas las demás opciones
end case;
  • expression: El valor que se está evaluando.
  • choice1, choice2, etc.: Valores posibles de la expresión.
  • others: Un caso general para cualquier valor no listado explícitamente.

Ejemplo

Aquí hay un ejemplo simple de una sentencia case en Ada:

with Ada.Text_IO; use Ada.Text_IO;

procedure Case_Example is
   Grade : Character := 'B';
begin
   case Grade is
      when 'A' =>
         Put_Line("¡Excelente!");
      when 'B' =>
         Put_Line("¡Buen trabajo!");
      when 'C' =>
         Put_Line("¡Bien hecho!");
      when 'D' =>
         Put_Line("Pasaste.");
      when 'F' =>
         Put_Line("Mejor inténtalo de nuevo.");
      when others =>
         Put_Line("Calificación inválida.");
   end case;
end Case_Example;

En este ejemplo, el programa imprime un mensaje basado en el valor de la variable Grade.

Conceptos Clave

  • Expresión: El valor que la sentencia case evalúa.
  • Opciones: Los valores posibles que la expresión puede tomar.
  • Others: Un caso por defecto que maneja cualquier valor no listado explícitamente.

Uso Avanzado

Rangos y Múltiples Opciones

Ada permite especificar rangos y múltiples opciones en una sentencia case. Esto puede ser útil para agrupar casos similares juntos.

Ejemplo con Rangos

with Ada.Text_IO; use Ada.Text_IO;

procedure Case_Range_Example is
   Score : Integer := 85;
begin
   case Score is
      when 90 .. 100 =>
         Put_Line("¡Excelente!");
      when 80 .. 89 =>
         Put_Line("¡Muy bien!");
      when 70 .. 79 =>
         Put_Line("¡Bien!");
      when 60 .. 69 =>
         Put_Line("Aprobado.");
      when others =>
         Put_Line("Reprobado.");
   end case;
end Case_Range_Example;

En este ejemplo, el programa imprime un mensaje basado en el rango en el que cae la variable Score.

Sentencias Case Anidadas

Puedes anidar sentencias case dentro de otras para manejar procesos de toma de decisiones más complejos.

Ejemplo de Sentencias Case Anidadas

with Ada.Text_IO; use Ada.Text_IO;

procedure Nested_Case_Example is
   Day : Integer := 3;
   Weather : Character := 'S';
begin
   case Day is
      when 1 | 2 | 3 | 4 | 5 =>
         Put_Line("Día laborable");
         case Weather is
            when 'S' =>
               Put_Line("Soleado");
            when 'R' =>
               Put_Line("Lluvioso");
            when others =>
               Put_Line("Clima desconocido");
         end case;
      when 6 | 7 =>
         Put_Line("Fin de semana");
         case Weather is
            when 'S' =>
               Put_Line("Soleado");
            when 'R' =>
               Put_Line("Lluvioso");
            when others =>
               Put_Line("Clima desconocido");
         end case;
      when others =>
         Put_Line("Día inválido");
   end case;
end Nested_Case_Example;

En este ejemplo, el programa primero determina si el día es laborable o fin de semana y luego verifica la condición del clima.

Conclusión

Las sentencias case en Ada son una herramienta poderosa para controlar el flujo de un programa basado en el valor de una expresión. Mejoran la legibilidad y el mantenimiento del código al separar claramente diferentes caminos de ejecución. Al comprender la sintaxis básica, usar rangos y múltiples opciones, y anidar sentencias case, puedes manejar procesos de toma de decisiones complejos de manera efectiva. Practica con los ejemplos y ejercicios proporcionados para volverte competente en el uso de sentencias case en Ada.
