Definición de Clases y Objetos en Ada

Introducción

En esta sección, exploraremos cómo definir clases y objetos en Ada. Ada es un lenguaje de programación de alto nivel, tipado estáticamente, que soporta la programación orientada a objetos (OOP). Entender cómo definir y usar clases y objetos es fundamental para aprovechar todo el poder de Ada.

Conceptos Clave

  • Clase: En Ada, una clase se define usando tipos etiquetados. Los tipos etiquetados permiten la herencia y el polimorfismo.
  • Objeto: Una instancia de una clase (tipo etiquetado) se llama objeto.
  • Encapsulación: La encapsulación se logra definiendo componentes privados dentro de un paquete.
  • Herencia: La herencia se soporta a través de la extensión de tipos.
  • Polimorfismo: El polimorfismo se logra usando tipos de clase amplia y operaciones de despacho.

Definición de una Clase (Tipo Etiquetado)

En Ada, una clase se define usando un tipo etiquetado. Aquí hay un ejemplo simple:

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move (S : in out Shape; New_X, New_Y : Integer);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move (S : in out Shape; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;
end Shape_Pkg;

Explicación

  • Paquete: El paquete Shape_Pkg encapsula la definición del tipo Shape y sus operaciones.
  • Tipo Etiquetado: Shape es un tipo etiquetado, lo que significa que puede ser extendido.
  • Procedimiento: El procedimiento Move modifica las coordenadas X y Y de un objeto Shape.

Creación y Uso de Objetos

Para crear y usar objetos de la clase Shape, puedes hacer lo siguiente:

with Shape_Pkg;
use Shape_Pkg;

procedure Main is
    My_Shape : Shape := (X => 0, Y => 0);
begin
    Move(My_Shape, 10, 20);
    -- My_Shape ahora tiene coordenadas (10, 20)
end Main;

Explicación

  • Creación de Objetos: My_Shape es un objeto de tipo Shape inicializado con coordenadas (0, 0).
  • Uso de Procedimientos: El procedimiento Move se llama para cambiar las coordenadas de My_Shape.

Herencia y Extensión de Tipos

Ada soporta la herencia a través de la extensión de tipos. Aquí hay un ejemplo de extensión del tipo Shape:

package Circle_Pkg is
    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Resize (C : in out Circle; New_Radius : Integer);
end Circle_Pkg;

package body Circle_Pkg is
    procedure Resize (C : in out Circle; New_Radius : Integer) is
    begin
        C.Radius := New_Radius;
    end Resize;
end Circle_Pkg;

Explicación

  • Extensión de Tipo: Circle es un nuevo tipo que extiende Shape añadiendo un campo Radius.
  • Procedimiento: El procedimiento Resize modifica el Radius de un objeto Circle.

Polimorfismo

El polimorfismo en Ada se logra usando tipos de clase amplia y operaciones de despacho. Aquí hay un ejemplo:

procedure Draw (S : in out Shape'Class) is
begin
    -- Operación de despacho basada en el tipo real de S
end Draw;

procedure Main is
    My_Circle : Circle := (X => 0, Y => 0, Radius => 5);
    My_Shape : Shape'Class := My_Circle;
begin
    Draw(My_Shape);
    -- El procedimiento Draw despachará a la operación apropiada para Circle
end Main;

Explicación

  • Tipo de Clase Amplia: Shape'Class es un tipo de clase amplia que puede contener cualquier tipo derivado de Shape.
  • Operación de Despacho: El procedimiento Draw llamará a la operación apropiada basada en el tipo real de S.

Conclusión

En esta sección, hemos cubierto los conceptos básicos de la definición de clases y objetos en Ada. Exploramos:

  • Cómo definir una clase usando tipos etiquetados.
  • Cómo crear y usar objetos.
  • Cómo extender tipos para lograr herencia.
  • Cómo usar polimorfismo con tipos de clase amplia y operaciones de despacho.

Entender estos conceptos es crucial para una programación orientada a objetos efectiva en Ada. Practica definiendo tus propias clases y objetos para solidificar tu comprensión.
