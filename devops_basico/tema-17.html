¿Qué es Docker?

Docker es una plataforma de contenedorización que permite a los desarrolladores empaquetar aplicaciones y sus dependencias en un contenedor, asegurando que la aplicación se ejecute de manera uniforme en cualquier entorno.

Conceptos Clave

  • Contenedor: Un contenedor es una unidad estándar de software que empaqueta el código y todas sus dependencias para que la aplicación se ejecute de manera rápida y confiable de un entorno informático a otro.
  • Imagen: Una imagen de Docker es una plantilla de solo lectura que contiene un conjunto de instrucciones para crear un contenedor que puede ejecutarse en la plataforma Docker.
  • Dockerfile: Un archivo de texto que contiene todas las instrucciones (comandos) que un usuario podría llamar en la línea de comandos para ensamblar una imagen de Docker.
  • Docker Hub: Un servicio de registro de imágenes de Docker que permite almacenar y compartir imágenes de contenedores.

Instalación de Docker

Para comenzar a usar Docker, primero necesitas instalarlo en tu sistema. Aquí te mostramos cómo hacerlo en diferentes sistemas operativos:

Instalación en Windows

  1. Descarga Docker Desktop desde Docker Hub.
  2. Ejecuta el instalador y sigue las instrucciones en pantalla.
  3. Una vez instalado, abre Docker Desktop y asegúrate de que Docker esté corriendo.

Instalación en macOS

  1. Descarga Docker Desktop desde Docker Hub.
  2. Abre el archivo .dmg y arrastra Docker a la carpeta de Aplicaciones.
  3. Inicia Docker desde la carpeta de Aplicaciones.

Instalación en Linux

Para Ubuntu:

sudo apt-get update
sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    software-properties-common

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

sudo add-apt-repository \
   "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
   $(lsb_release -cs) \
   stable"

sudo apt-get update
sudo apt-get install docker-ce

Primeros Pasos con Docker

Ejecutar tu Primer Contenedor

Una vez que Docker esté instalado, puedes ejecutar tu primer contenedor. Vamos a usar una imagen de prueba llamada hello-world.

docker run hello-world

Este comando descarga una imagen de prueba de Docker y ejecuta un contenedor que imprime "Hello from Docker!".

Crear una Imagen de Docker

Para crear tu propia imagen de Docker, necesitas un Dockerfile. Aquí tienes un ejemplo simple:

# Usa una imagen base de Python
FROM python:3.8-slim

# Establece el directorio de trabajo
WORKDIR /app

# Copia el archivo de requerimientos y la aplicación
COPY requirements.txt requirements.txt
COPY app.py app.py

# Instala las dependencias
RUN pip install -r requirements.txt

# Define el comando por defecto
CMD ["python", "app.py"]

Para construir esta imagen, usa el siguiente comando:

docker build -t my-python-app .

Ejecutar un Contenedor desde tu Imagen

Una vez que hayas creado tu imagen, puedes ejecutar un contenedor basado en ella:

docker run my-python-app

Comandos Básicos de Docker

Aquí hay una tabla con algunos comandos básicos de Docker y su descripción:

| Comando | Descripción | |--------------------------|--------------------------------------------------| | docker run | Ejecuta un contenedor | | docker ps | Lista los contenedores en ejecución | | docker images | Lista las imágenes disponibles | | docker build | Construye una imagen desde un Dockerfile | | docker pull | Descarga una imagen desde un repositorio | | docker push | Sube una imagen a un repositorio | | docker stop | Detiene un contenedor en ejecución | | docker rm | Elimina un contenedor | | docker rmi | Elimina una imagen |

Conclusión

Docker es una herramienta poderosa que facilita la creación, despliegue y ejecución de aplicaciones en contenedores. Al aprender los conceptos básicos y comandos esenciales, puedes comenzar a aprovechar las ventajas de la contenedorización en tus proyectos de desarrollo y operaciones. En las siguientes lecciones, profundizaremos en temas más avanzados como la orquestación de contenedores y la integración continua con Docker.
