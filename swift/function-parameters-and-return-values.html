Parámetros y Valores de Retorno de Funciones
Closures ►

Introducción

En Swift, las funciones son un componente fundamental de tu código. Te permiten encapsular piezas reutilizables de lógica y pueden recibir entradas (parámetros) y producir salidas (valores de retorno). Entender cómo definir y usar parámetros y valores de retorno de funciones es crucial para escribir código Swift eficiente y legible.

Parámetros de Función

Definiendo Parámetros

Los parámetros son las entradas de una función. Los defines dentro de los paréntesis después del nombre de la función.

func greet(name: String) {
    print("¡Hola, \(name)!")
}
  • name es un parámetro de tipo String.
  • Al llamar a la función, necesitas proporcionar un argumento que coincida con el tipo del parámetro.

Llamando Funciones con Parámetros

Para llamar a una función con parámetros, pasas los argumentos en el mismo orden en que los parámetros están definidos.

greet(name: "Alice")
// Salida: ¡Hola, Alice!

Múltiples Parámetros

Una función puede tener múltiples parámetros, separados por comas.

func add(a: Int, b: Int) -> Int {
    return a + b
}

let sum = add(a: 3, b: 5)
print(sum) // Salida: 8
  • a y b son parámetros de tipo Int.
  • La función retorna la suma de a y b.

Etiquetas de Parámetros y Etiquetas de Argumentos

Swift te permite definir etiquetas de parámetros y etiquetas de argumentos para hacer tus llamadas a funciones más legibles.

func multiply(_ number1: Int, by number2: Int) -> Int {
    return number1 * number2
}

let product = multiply(4, by: 5)
print(product) // Salida: 20
  • _ se usa para omitir la etiqueta de argumento para number1.
  • by es la etiqueta de argumento para number2.

Valores Predeterminados de Parámetros

Puedes proporcionar valores predeterminados para los parámetros, haciéndolos opcionales al llamar a la función.

func greet(name: String = "Guest") {
    print("¡Hola, \(name)!")
}

greet() // Salida: ¡Hola, Guest!
greet(name: "Bob") // Salida: ¡Hola, Bob!

Valores de Retorno

Valor de Retorno Único

Las funciones pueden retornar un valor usando la sintaxis -> seguida del tipo de retorno.

func square(number: Int) -> Int {
    return number * number
}

let result = square(number: 4)
print(result) // Salida: 16
  • La función square retorna un Int.

Valores de Retorno Múltiples con Tuplas

Swift permite que las funciones retornen múltiples valores usando tuplas.

func minMax(numbers: [Int]) -> (min: Int, max: Int)? {
    guard let first = numbers.first else { return nil }
    var min = first
    var max = first
    for number in numbers.dropFirst() {
        if number < min {
            min = number
        } else if number > max {
            max = number
        }
    }
    return (min, max)
}

if let result = minMax(numbers: [3, 5, 1, 9, 2]) {
    print("Min: \(result.min), Max: \(result.max)")
    // Salida: Min: 1, Max: 9
}
  • La función minMax retorna una tupla opcional (min: Int, max: Int)?.

Tipo de Retorno Void

Si una función no retorna un valor, puedes usar Void o simplemente omitir el tipo de retorno.

func sayHello() {
    print("¡Hola, Mundo!")
}

sayHello() // Salida: ¡Hola, Mundo!

Temas Avanzados

Parámetros In-Out

Los parámetros in-out te permiten modificar el valor de un parámetro dentro de la función.

func swapValues(_ a: inout Int, _ b: inout Int) {
    let temp = a
    a = b
    b = temp
}

var x = 3
var y = 5
swapValues(&x, &y)
print("x: \(x), y: \(y)") // Salida: x: 5, y: 3
  • La palabra clave inout se usa para indicar que un parámetro puede ser modificado.
  • El símbolo & se usa al pasar argumentos a parámetros in-out.

Tipos de Funciones como Parámetros y Valores de Retorno

Las funciones pueden tomar otras funciones como parámetros y también pueden retornar funciones.

func performOperation(_ operation: (Int, Int) -> Int, on a: Int, and b: Int) -> Int {
    return operation(a, b)
}

let result = performOperation(add, on: 3, and: 5)
print(result) // Salida: 8
  • operation es un parámetro de tipo (Int, Int) -> Int.
  • La función performOperation toma una función y dos enteros, y retorna el resultado de la operación.

Conclusión

Entender los parámetros y valores de retorno de funciones es esencial para escribir código Swift efectivo. Al dominar estos conceptos, puedes crear funciones más flexibles, reutilizables y legibles. Practica definiendo funciones con varios tipos de parámetros y valores de retorno para volverte competente en la programación en Swift.
