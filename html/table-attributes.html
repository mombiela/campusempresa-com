Introducción

En HTML, las tablas se utilizan para mostrar datos en un formato estructurado usando filas y columnas. Entender los atributos de las tablas es esencial para crear tablas bien organizadas y visualmente atractivas. Esta sección cubrirá los diversos atributos que se pueden aplicar a las tablas HTML para mejorar su funcionalidad y apariencia.

Estructura Básica de una Tabla

Antes de profundizar en los atributos de las tablas, revisemos la estructura básica de una tabla HTML:

<table>
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>
  • <table>: Define la tabla.
  • <tr>: Define una fila de la tabla.
  • <th>: Define un encabezado de la tabla.
  • <td>: Define una celda de la tabla.

Atributos Comunes de la Tabla

`border`

El atributo border especifica el ancho del borde alrededor de la tabla y sus celdas.

<table border="1">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`cellpadding`

El atributo cellpadding especifica el espacio entre el contenido de la celda y sus bordes.

<table border="1" cellpadding="10">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`cellspacing`

El atributo cellspacing especifica el espacio entre las celdas.

<table border="1" cellspacing="10">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`width` y `height`

Los atributos width y height especifican las dimensiones de la tabla.

<table border="1" width="50%" height="100">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`align`

El atributo align especifica la alineación de la tabla en la página. Este atributo está obsoleto en HTML5, y se debe usar CSS en su lugar.

<table border="1" align="center">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`bgcolor`

El atributo bgcolor especifica el color de fondo de la tabla. Este atributo también está obsoleto en HTML5, y se debe usar CSS en su lugar.

<table border="1" bgcolor="lightblue">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

Atributos Avanzados de la Tabla

`colspan` y `rowspan`

Los atributos colspan y rowspan permiten que las celdas abarquen múltiples columnas o filas.

<table border="1">
  <tr>
    <th colspan="2">Encabezado abarcando 2 columnas</th>
  </tr>
  <tr>
    <td rowspan="2">Fila abarcando 2 filas</td>
    <td>Dato 1</td>
  </tr>
  <tr>
    <td>Dato 2</td>
  </tr>
</table>

`scope`

El atributo scope se usa para especificar si una celda de encabezado es un encabezado para una columna, fila o grupo de columnas o filas.

<table border="1">
  <tr>
    <th scope="col">Encabezado 1</th>
    <th scope="col">Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

`headers`

El atributo headers se usa para especificar los encabezados asociados con una celda, mejorando la accesibilidad.

<table border="1">
  <tr>
    <th id="header1">Encabezado 1</th>
    <th id="header2">Encabezado 2</th>
  </tr>
  <tr>
    <td headers="header1">Dato 1</td>
    <td headers="header2">Dato 2</td>
  </tr>
</table>

Conclusión

Entender y usar los atributos de las tablas en HTML te permite crear tablas más estructuradas, accesibles y visualmente atractivas. Aunque algunos atributos están obsoletos en HTML5, conocerlos puede ser útil para mantener código antiguo. Para el desarrollo web moderno, se recomienda usar CSS para propósitos de estilo y diseño. Practica usando estos atributos para volverte competente en la creación y gestión de tablas HTML.
