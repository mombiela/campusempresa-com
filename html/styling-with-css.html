Introducción a CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML. CSS controla el diseño, colores, fuentes y la apariencia visual general de las páginas web.

  • Sintaxis de CSS: Las reglas de CSS están compuestas por selectores y declaraciones. 
    selector {
  property: value;
}
  • Selectores: Seleccionan los elementos HTML que deseas estilizar.
  • Declaraciones: Contienen propiedades y valores que definen los estilos.

Añadiendo CSS a HTML

Hay tres formas principales de añadir CSS a documentos HTML:

  1. CSS en línea: Usando el atributo style dentro de las etiquetas HTML.

    <p style="color: blue;">Este es un párrafo azul.</p>

  2. CSS interno: Usando la etiqueta <style> dentro de la sección <head> del documento HTML.

    <head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>

  3. CSS externo: Enlazando un archivo CSS externo usando la etiqueta <link>.

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

Selectores básicos de CSS

  • Selector de elementos: Selecciona todos los elementos de un tipo dado.

    p {
  color: blue;
}

  • Selector de clases: Selecciona elementos con un atributo de clase específico, precedido por un punto (.).

    .myClass {
  color: green;
}

  • Selector de ID: Selecciona un solo elemento con un atributo de ID específico, precedido por un hash (#).

    #myId {
  color: red;
}

  • Selector de atributos: Selecciona elementos con un atributo específico.

    [type="text"] {
  border: 1px solid black;
}

Selectores avanzados de CSS

  • Selector de descendientes: Selecciona elementos que son descendientes de un elemento especificado.

    div p {
  color: purple;
}

  • Selector de hijos: Selecciona elementos que son hijos directos de un elemento especificado.

    div > p {
  color: orange;
}

  • Pseudo-clases: Selecciona elementos basados en su estado.

    a:hover {
  color: pink;
}

Modelo de caja de CSS

El modelo de caja de CSS describe los cuadros rectangulares generados para los elementos en el árbol del documento y consiste en:

  • Contenido: El contenido real de la caja, donde aparecen texto e imágenes.
  • Relleno: Limpia un área alrededor del contenido. El relleno es transparente.
  • Borde: Un borde que rodea el relleno y el contenido.
  • Margen: Limpia un área fuera del borde. El margen también es transparente.
div {
  width: 300px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Técnicas de diseño de CSS

  • Flexbox: Un modelo de diseño que permite a los elementos alinearse y distribuir espacio dentro de un contenedor.

    .container {
  display: flex;
}

  • Grid: Un sistema de diseño para crear diseños web complejos fácilmente.

    .grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
}

Diseño responsivo

El diseño responsivo asegura que las páginas web se vean bien en todos los dispositivos. Las técnicas clave incluyen:

  • Consultas de medios: Aplicar diferentes estilos para diferentes dispositivos o tamaños de pantalla.

    @media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

  • Diseños fluidos: Usar unidades relativas como porcentajes en lugar de unidades fijas como píxeles.

    .container {
  width: 100%;
}

Animaciones y transiciones en CSS

  • Transiciones: Cambiar suavemente las propiedades durante una duración especificada.

    .box {
  transition: width 2s;
}
.box:hover {
  width: 300px;
}

  • Animaciones: Crear animaciones complejas usando fotogramas clave.

    @keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}
.box {
  animation: example 5s infinite;
}

Conclusión

CSS es una herramienta poderosa para controlar la presentación visual de las páginas web. Al entender y utilizar varias técnicas de CSS, desde selectores básicos hasta sistemas de diseño avanzados y diseño responsivo, puedes crear páginas web visualmente atractivas y funcionales. La práctica y la experimentación son clave para dominar CSS y mejorar tus habilidades en desarrollo web.
