Introducción

En este tema, te guiaremos a través del proceso de construir un juego simple usando Lua. Esto te ayudará a entender los conceptos básicos del desarrollo de juegos, incluyendo bucles de juego, manejo de eventos y renderización de gráficos. Al final de esta sección, tendrás un juego funcional y una base sólida para construir juegos más complejos.

Configurando el Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Lua: Descarga e instala Lua desde el sitio web oficial.
  • Editor de Texto: Usa cualquier editor de texto como Visual Studio Code, Sublime Text o Notepad++.

Estructura Básica del Juego

Un juego simple típicamente consiste en los siguientes componentes:

  • Inicialización: Configuración del entorno del juego.
  • Bucle de Juego: Un bucle que se ejecuta continuamente para actualizar el estado del juego y renderizar gráficos.
  • Manejo de Eventos: Gestión de entradas del usuario y otros eventos.
  • Renderización: Dibujo de gráficos en la pantalla.

Inicialización

Primero, vamos a inicializar nuestro entorno de juego. Crearemos una ventana básica usando el framework Love2D, que es una elección popular para el desarrollo de juegos en Lua.

function love.load()
    love.window.setTitle("Juego Simple")
    love.window.setMode(800, 600)
end

Bucle de Juego

El bucle de juego consiste en dos funciones principales: love.update(dt) y love.draw().

  • love.update(dt): Actualiza el estado del juego. El parámetro dt representa el tiempo transcurrido desde la última actualización.
  • love.draw(): Renderiza los gráficos del juego.
function love.update(dt)
    -- Actualizar el estado del juego aquí
end

function love.draw()
    -- Renderizar los gráficos del juego aquí
end

Manejo de Eventos

Necesitamos manejar las entradas del usuario, como eventos de teclado y ratón. Love2D proporciona funciones como love.keypressed y love.mousepressed para este propósito.

function love.keypressed(key)
    if key == "escape" then
        love.event.quit()
    end
end

Creando Objetos del Juego

Vamos a crear un objeto de juego simple, como un personaje jugador. Definiremos sus propiedades y métodos.

player = {
    x = 400,
    y = 300,
    speed = 200
}

function player.update(dt)
    if love.keyboard.isDown("left") then
        player.x = player.x - player.speed * dt
    elseif love.keyboard.isDown("right") then
        player.x = player.x + player.speed * dt
    end
    if love.keyboard.isDown("up") then
        player.y = player.y - player.speed * dt
    elseif love.keyboard.isDown("down") then
        player.y = player.y + player.speed * dt
    end
end

function player.draw()
    love.graphics.rectangle("fill", player.x, player.y, 50, 50)
end

Integrando Objetos del Juego

Ahora, integra el objeto jugador en el bucle de juego.

function love.update(dt)
    player.update(dt)
end

function love.draw()
    player.draw()
end

Añadiendo Más Características

Para hacer el juego más interesante, añadamos algunas características adicionales.

Detección de Colisiones

Añadiremos una detección de colisiones simple para evitar que el jugador se mueva fuera de los límites de la ventana.

function player.update(dt)
    if love.keyboard.isDown("left") and player.x > 0 then
        player.x = player.x - player.speed * dt
    elseif love.keyboard.isDown("right") and player.x < love.graphics.getWidth() - 50 then
        player.x = player.x + player.speed * dt
    end
    if love.keyboard.isDown("up") and player.y > 0 then
        player.y = player.y - player.speed * dt
    elseif love.keyboard.isDown("down") and player.y < love.graphics.getHeight() - 50 then
        player.y = player.y + player.speed * dt
    end
end

Sistema de Puntuación

Añadamos un sistema de puntuación simple que incremente la puntuación cuando el jugador se mueva.

score = 0

function player.update(dt)
    local moved = false
    if love.keyboard.isDown("left") and player.x > 0 then
        player.x = player.x - player.speed * dt
        moved = true
    elseif love.keyboard.isDown("right") and player.x < love.graphics.getWidth() - 50 then
        player.x = player.x + player.speed * dt
        moved = true
    end
    if love.keyboard.isDown("up") and player.y > 0 then
        player.y = player.y - player.speed * dt
        moved = true
    elseif love.keyboard.isDown("down") and player.y < love.graphics.getHeight() - 50 then
        player.y = player.y + player.speed * dt
        moved = true
    end

    if moved then
        score = score + 1
    end
end

function love.draw()
    player.draw()
    love.graphics.print("Puntuación: " .. score, 10, 10)
end

Conclusión

En este tema, cubrimos los conceptos básicos de construir un juego simple usando Lua y el framework Love2D. Aprendimos cómo configurar el entorno del juego, crear un bucle de juego, manejar eventos y renderizar gráficos. También añadimos características como detección de colisiones y un sistema de puntuación. Con estos fundamentos, puedes comenzar a crear juegos más complejos y atractivos. ¡Feliz codificación!
