Variables y Tipos de Datos en Lua

Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y tipos de datos en Lua. Entender estos conceptos básicos es crucial para escribir programas en Lua efectivos y eficientes.

Variables

Las variables se utilizan para almacenar datos que pueden ser referenciados y manipulados en un programa. En Lua, las variables son de tipo dinámico, lo que significa que no necesitas declarar su tipo explícitamente.

Declaración de Variables

Para declarar una variable en Lua, simplemente le asignas un valor usando el operador =.

-- Declarando una variable
local miVariable = 10
  • Palabra clave local: Limita el alcance de la variable al bloque donde se declara.
  • miVariable: El nombre de la variable.
  • 10: El valor asignado a la variable.

Variables Globales vs Locales

  • Variables Globales: Accesibles desde cualquier parte del programa.
  • Variables Locales: Accesibles solo dentro del bloque donde se declaran.
-- Variable global
globalVar = "Soy global"

-- Variable local
local localVar = "Soy local"

Tipos de Datos

Lua soporta varios tipos de datos básicos, cada uno con diferentes propósitos. Los principales tipos de datos en Lua son:

  • nil
  • boolean
  • number
  • string
  • table
  • function
  • userdata
  • thread

nil

El tipo nil representa la ausencia de un valor.

local sinValor = nil

boolean

El tipo boolean tiene dos valores: true y false.

local esLuaDivertido = true
local estaCansado = false

number

El tipo number representa números reales (de punto flotante).

local edad = 25
local pi = 3.14

string

El tipo string representa secuencias de caracteres.

local saludo = "¡Hola, Mundo!"

table

El tipo table es una estructura de datos poderosa y flexible que puede usarse como arreglos, diccionarios y más.

-- Tabla tipo arreglo
local frutas = {"manzana", "banana", "cereza"}

-- Tabla tipo diccionario
local persona = {
    nombre = "Juan",
    edad = 30
}

function

El tipo function representa bloques de código ejecutables.

local function saludar(nombre)
    return "Hola, " .. nombre
end

userdata

El tipo userdata permite almacenar datos arbitrarios de C en variables Lua.

thread

El tipo thread representa hilos de ejecución independientes para corrutinas.

Ejemplos y Ejercicios

Ejemplo 1: Trabajando con Variables

local nombre = "Alicia"
local edad = 28
local esEstudiante = true

print("Nombre: " .. nombre)
print("Edad: " .. edad)
print("Es Estudiante: " .. tostring(esEstudiante))

Ejercicio 1: Crear e Imprimir Variables

  1. Crea una variable para almacenar tu color favorito.
  2. Crea una variable para almacenar tu altura en centímetros.
  3. Imprime ambas variables.

Ejemplo 2: Usando Tablas

local coche = {
    marca = "Toyota",
    modelo = "Corolla",
    año = 2020
}

print("Marca del Coche: " .. coche.marca)
print("Modelo del Coche: " .. coche.modelo)
print("Año del Coche: " .. coche.año)

Ejercicio 2: Crear y Manipular una Tabla

  1. Crea una tabla para almacenar información sobre un libro (título, autor, año).
  2. Imprime cada pieza de información de la tabla.

Conclusión

Entender las variables y los tipos de datos es esencial para cualquier programador en Lua. Las variables te permiten almacenar y manipular datos, mientras que los tipos de datos definen el tipo de datos con los que estás trabajando. Al dominar estos conceptos básicos, estarás bien preparado para manejar tareas de programación más complejas en Lua.
