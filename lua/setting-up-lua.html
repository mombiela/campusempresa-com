Configuración del Entorno Lua
Tu Primer Programa en Lua ►

Configurar el entorno Lua es el primer paso para comenzar a programar en Lua. Esta sección te guiará a través del proceso de instalación de Lua en tu sistema, la configuración de un Entorno de Desarrollo Integrado (IDE) y la ejecución de tu primer script en Lua.

Instalación de Lua

Windows

  1. Descargar Binarios de Lua:

    • Visita el sitio web oficial de Lua: lua.org
    • Descarga la última versión de Lua para Windows.

  2. Extraer e Instalar:

    • Extrae el archivo ZIP descargado.
    • Mueve la carpeta extraída a una ubicación deseada (por ejemplo, C:\Lua).

  3. Configurar Variables de Entorno:

    • Abre el Menú de Inicio, busca "Variables de Entorno" y selecciona "Editar las variables de entorno del sistema."
    • Haz clic en "Variables de Entorno."
    • Bajo "Variables del sistema," encuentra la variable Path y haz clic en "Editar."
    • Agrega la ruta al ejecutable de Lua (por ejemplo, C:\Lua\bin).

  4. Verificar Instalación:

    • Abre el Símbolo del sistema y escribe lua -v.
    • Deberías ver la información de la versión de Lua.

macOS

  1. Usando Homebrew:

    • Abre Terminal.
    • Instala Homebrew si no lo has hecho ya: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instala Lua: brew install lua

  2. Verificar Instalación:

    • En Terminal, escribe lua -v.
    • Deberías ver la información de la versión de Lua.

Linux

  1. Usando el Gestor de Paquetes:

    • Abre Terminal.
    • Para sistemas basados en Debian (por ejemplo, Ubuntu): sudo apt-get install lua5.3
    • Para sistemas basados en Red Hat (por ejemplo, Fedora): sudo dnf install lua

  2. Verificar Instalación:

    • En Terminal, escribe lua -v.
    • Deberías ver la información de la versión de Lua.

Configuración de un IDE

Aunque puedes escribir código Lua en cualquier editor de texto, usar un IDE puede mejorar significativamente tu productividad. Aquí hay algunas opciones populares:

  • ZeroBrane Studio:

    • Un IDE ligero para Lua con soporte para depuración.
    • Descárgalo desde ZeroBrane Studio.
    • Sigue las instrucciones de instalación en el sitio web.

  • Visual Studio Code:

    • Un editor de código versátil con soporte para Lua a través de extensiones.
    • Descárgalo desde Visual Studio Code.
    • Instala la extensión "Lua" desde el Marketplace de Extensiones.

Ejecutar tu Primer Script en Lua

Escribir un Script Simple

  1. Crear un Nuevo Archivo:

    • Abre tu IDE o editor de texto.
    • Crea un nuevo archivo y guárdalo como hello.lua.

  2. Escribir Código Lua:

    • Agrega el siguiente código a hello.lua: 
      print("¡Hola, Mundo!")

  3. Ejecutar el Script:

    • Abre una terminal o símbolo del sistema.
    • Navega al directorio donde se guardó hello.lua.
    • Ejecuta el script escribiendo: lua hello.lua
    • Deberías ver la salida: ¡Hola, Mundo!

Explicación del Código

  • print("¡Hola, Mundo!"): Esta línea de código imprime la cadena "¡Hola, Mundo!" en la consola. La función print es una función incorporada de Lua utilizada para la salida.

Conclusión

Configurar el entorno Lua implica instalar Lua en tu sistema operativo, elegir y configurar un IDE, y ejecutar tu primer script en Lua. Siguiendo estos pasos, aseguras que tu entorno de desarrollo esté listo para programación Lua más avanzada. En las siguientes secciones, profundizaremos en la sintaxis de Lua, estructuras de datos y características más avanzadas.
