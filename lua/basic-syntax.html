Introducción a Lua

Lua es un lenguaje de programación liviano, de alto nivel y multiparadigma, diseñado principalmente para uso embebido en aplicaciones. Es conocido por su simplicidad y flexibilidad, lo que lo convierte en una opción popular para el desarrollo de juegos, scripting y prototipado rápido.

Conceptos Clave

  • Sensibilidad a Mayúsculas y Minúsculas: Lua distingue entre mayúsculas y minúsculas, lo que significa que Variable y variable se consideran identificadores diferentes.
  • Comentarios: Lua soporta comentarios de una sola línea y de múltiples líneas.
  • Espacios en Blanco: Los espacios en blanco generalmente se ignoran en Lua, pero se usan para separar tokens.

Comentarios

  • Comentarios de una sola línea: Usa -- para iniciar un comentario de una sola línea.
  • Comentarios de múltiples líneas: Usa --[[ para iniciar y ]] para terminar un comentario de múltiples líneas.
-- Este es un comentario de una sola línea

--[[
Este es un
comentario de múltiples líneas
]]

Variables y Tipos de Datos

  • Declaración de Variables: Las variables en Lua son de tipo dinámico y pueden ser declaradas sin especificar un tipo.
  • Tipos de Datos: Lua soporta varios tipos de datos básicos, incluyendo nil, boolean, number, string, function, userdata, thread y table.
local myNumber = 10       -- número
local myString = "Hola"   -- cadena
local myBoolean = true    -- booleano
local myNil = nil         -- nil

Operadores

Lua soporta una variedad de operadores para operaciones aritméticas, relacionales, lógicas y otras.

Operadores Aritméticos

  • + (suma)
  • - (resta)
  • * (multiplicación)
  • / (división)
  • % (módulo)
  • ^ (exponenciación)
local a = 5
local b = 2
print(a + b)  -- Salida: 7
print(a - b)  -- Salida: 3
print(a * b)  -- Salida: 10
print(a / b)  -- Salida: 2.5
print(a % b)  -- Salida: 1
print(a ^ b)  -- Salida: 25

Operadores Relacionales

  • == (igual a)
  • ~= (diferente de)
  • < (menor que)
  • > (mayor que)
  • <= (menor o igual que)
  • >= (mayor o igual que)
local x = 10
local y = 20
print(x == y)  -- Salida: false
print(x ~= y)  -- Salida: true
print(x < y)   -- Salida: true
print(x > y)   -- Salida: false
print(x <= y)  -- Salida: true
print(x >= y)  -- Salida: false

Operadores Lógicos

  • and
  • or
  • not
local a = true
local b = false
print(a and b)  -- Salida: false
print(a or b)   -- Salida: true
print(not a)    -- Salida: false

Estructuras de Control

Lua proporciona varias estructuras de control para gestionar el flujo del programa.

Sentencias If

local age = 18
if age >= 18 then
    print("Eres un adulto.")
else
    print("Eres un menor.")
end

Bucles While

local count = 1
while count <= 5 do
    print(count)
    count = count + 1
end

Bucles For

for i = 1, 5 do
    print(i)
end

Bucles Repeat-Until

local count = 1
repeat
    print(count)
    count = count + 1
until count > 5

Funciones

Las funciones en Lua son valores de primera clase y pueden ser almacenadas en variables, pasadas como argumentos y devueltas desde otras funciones.

Definiendo una Función

function greet(name)
    return "Hola, " .. name
end

print(greet("Mundo"))  -- Salida: Hola, Mundo

Funciones Anónimas

local add = function(a, b)
    return a + b
end

print(add(2, 3))  -- Salida: 5

Tablas

Las tablas son la única estructura de datos en Lua y pueden ser usadas como arreglos, diccionarios o incluso objetos.

Creando una Tabla

local person = {
    name = "Juan",
    age = 30
}

print(person.name)  -- Salida: Juan
print(person.age)   -- Salida: 30

Iterando Sobre una Tabla

local fruits = {"manzana", "banana", "cereza"}

for index, value in ipairs(fruits) do
    print(index, value)
end

Conclusión

Entender la sintaxis y estructura básica de Lua es crucial para escribir código efectivo y eficiente. Esto incluye saber cómo declarar variables, usar operadores, controlar el flujo del programa, definir funciones y trabajar con tablas. Con estos fundamentos, estás bien equipado para explorar características y aplicaciones más avanzadas de Lua.
