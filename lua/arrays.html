Introducción a los Arrays

Los arrays son una estructura de datos fundamental en la programación, utilizada para almacenar múltiples valores en una sola variable. En Lua, los arrays se implementan utilizando tablas, que son estructuras de datos versátiles que pueden contener varios tipos de valores.

Conceptos Clave

  • Tablas: La estructura de datos principal de Lua, utilizada para crear arrays.
  • Indexación: Los arrays en Lua están indexados desde 1, lo que significa que el primer elemento se accede con el índice 1.
  • Inicialización: Los arrays pueden inicializarse con valores o crearse vacíos y llenarse más tarde.

Creación de Arrays

Los arrays en Lua se crean utilizando tablas. Aquí hay algunas formas de crear e inicializar arrays:

Ejemplo 1: Inicializando un Array con Valores

-- Creando un array con valores iniciales
local fruits = {"apple", "banana", "cherry"}
print(fruits[1])  -- Salida: apple
print(fruits[2])  -- Salida: banana
print(fruits[3])  -- Salida: cherry

Ejemplo 2: Creando un Array Vacío y Agregando Valores

-- Creando un array vacío
local numbers = {}

-- Agregando valores al array
numbers[1] = 10
numbers[2] = 20
numbers[3] = 30

print(numbers[1])  -- Salida: 10
print(numbers[2])  -- Salida: 20
print(numbers[3])  -- Salida: 30

Accediendo y Modificando Elementos del Array

Puedes acceder y modificar elementos en un array utilizando sus índices.

Ejemplo 3: Accediendo y Modificando Elementos

local colors = {"red", "green", "blue"}

-- Accediendo a elementos
print(colors[1])  -- Salida: red

-- Modificando elementos
colors[2] = "yellow"
print(colors[2])  -- Salida: yellow

Iterando Sobre Arrays

Lua proporciona varias formas de iterar sobre arrays, incluyendo bucles for y la función ipairs.

Ejemplo 4: Usando un Bucle `for`

local animals = {"cat", "dog", "bird"}

for i = 1, #animals do
    print(animals[i])
end
-- Salida:
-- cat
-- dog
-- bird

Ejemplo 5: Usando `ipairs`

local cars = {"Toyota", "Ford", "BMW"}

for index, value in ipairs(cars) do
    print(index, value)
end
-- Salida:
-- 1   Toyota
-- 2   Ford
-- 3   BMW

Técnicas Avanzadas de Arrays

A medida que te sientas más cómodo con los arrays, puedes explorar técnicas más avanzadas como arrays multidimensionales y funciones de manipulación de arrays.

Arrays Multidimensionales

Los arrays multidimensionales son arrays de arrays. Pueden usarse para representar matrices o cuadrículas.

Ejemplo 6: Creando un Array 2D

local matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
}

print(matrix[1][1])  -- Salida: 1
print(matrix[2][3])  -- Salida: 6

Funciones de Manipulación de Arrays

Lua proporciona varias funciones integradas para manipular arrays, como table.insert, table.remove y table.sort.

Ejemplo 7: Usando `table.insert` y `table.remove`

local list = {"a", "b", "c"}

-- Insertando un elemento
table.insert(list, 2, "x")
print(table.concat(list, ", "))  -- Salida: a, x, b, c

-- Eliminando un elemento
table.remove(list, 3)
print(table.concat(list, ", "))  -- Salida: a, x, c

Ejemplo 8: Ordenando un Array

local numbers = {5, 3, 8, 1, 2}

table.sort(numbers)
for i, v in ipairs(numbers) do
    print(v)
end
-- Salida:
-- 1
-- 2
-- 3
-- 5
-- 8

Conclusión

Los arrays en Lua son poderosos y flexibles, gracias a la versatilidad de las tablas. Al entender cómo crear, acceder, modificar y manipular arrays, puedes gestionar eficazmente colecciones de datos en tus programas Lua. Practica con los ejemplos proporcionados y explora técnicas más avanzadas para volverte competente en el uso de arrays.
