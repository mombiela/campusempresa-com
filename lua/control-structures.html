Estructuras de Control en Lua
Bucles ►

Las estructuras de control son fundamentales en la programación ya que te permiten controlar el flujo de tu código. En Lua, las estructuras de control incluyen condicionales, bucles y otros constructos que ayudan a gestionar la ejecución de tu código. Esta sección cubrirá los aspectos básicos y avanzados de las estructuras de control en Lua.

Condicionales

Los condicionales se usan para ejecutar código basado en ciertas condiciones. Lua soporta las declaraciones if, else y elseif.

Declaración If

La declaración if te permite ejecutar un bloque de código si una condición especificada es verdadera.

local age = 20

if age >= 18 then
    print("Eres un adulto.")
end

Declaración If-Else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

local age = 16

if age >= 18 then
    print("Eres un adulto.")
else
    print("Eres un menor.")
end

Declaración If-Elseif-Else

La declaración if-elseif-else te permite verificar múltiples condiciones.

local score = 85

if score >= 90 then
    print("Calificación: A")
elseif score >= 80 then
    print("Calificación: B")
elseif score >= 70 then
    print("Calificación: C")
else
    print("Calificación: F")
end

Bucles

Los bucles se usan para ejecutar un bloque de código múltiples veces. Lua soporta los bucles while, for y repeat-until.

Bucle While

El bucle while ejecuta un bloque de código mientras una condición especificada sea verdadera.

local count = 1

while count <= 5 do
    print("El conteo es: " .. count)
    count = count + 1
end

Bucle For

El bucle for se usa para iterar sobre un rango de valores.

for i = 1, 5 do
    print("Iteración: " .. i)
end

Bucle Repeat-Until

El bucle repeat-until ejecuta un bloque de código hasta que una condición especificada se vuelva verdadera.

local count = 1

repeat
    print("El conteo es: " .. count)
    count = count + 1
until count > 5

Estructuras de Control Avanzadas

Bucles Anidados

Los bucles pueden anidarse unos dentro de otros para manejar escenarios más complejos.

for i = 1, 3 do
    for j = 1, 2 do
        print("i: " .. i .. ", j: " .. j)
    end
end

Declaración Break

La declaración break se usa para salir de un bucle prematuramente.

for i = 1, 10 do
    if i == 5 then
        break
    end
    print("i: " .. i)
end

Comportamiento Similar a Continue

Lua no tiene una declaración continue incorporada, pero puedes lograr un comportamiento similar usando goto.

for i = 1, 10 do
    if i % 2 == 0 then
        goto continue
    end
    print("i: " .. i)
    ::continue::
end

Resumen

Las estructuras de control son esenciales para gestionar el flujo de tus programas en Lua. Al dominar los condicionales, bucles y constructos de control avanzados, puedes escribir código más eficiente y legible. Practica usando estas estructuras en varios escenarios para volverte competente en la programación en Lua.
