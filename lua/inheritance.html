Herencia en Lua
Polimorfismo ►

La herencia es un concepto fundamental en la programación orientada a objetos (OOP) que permite a una clase (u objeto) heredar propiedades y métodos de otra clase (u objeto). En Lua, la herencia se logra utilizando metatables y el metamétodo __index.

Conceptos Clave

  • Clase: Un plano para crear objetos (instancias).
  • Objeto: Una instancia de una clase.
  • Metatable: Una tabla que puede cambiar el comportamiento de otra tabla.
  • Metamétodo __index: Se usa para buscar valores en una tabla cuando no se encuentran en la tabla original.

Creando una Clase Básica

En Lua, las clases se representan típicamente usando tablas. Aquí se muestra cómo puedes crear una clase simple:

-- Definir una clase básica
Person = {}
Person.__index = Person

-- Constructor
function Person:new(name, age)
    local instance = setmetatable({}, Person)
    instance.name = name
    instance.age = age
    return instance
end

-- Método para mostrar información
function Person:display()
    print("Nombre: " .. self.name .. ", Edad: " .. self.age)
end

-- Crear una instancia de Person
local person1 = Person:new("Alice", 30)
person1:display()  -- Salida: Nombre: Alice, Edad: 30

Explicación

  • Person = {}: Define una nueva tabla para representar la clase.
  • Person.__index = Person: Establece el metamétodo __index a la clase misma, habilitando la herencia.
  • Person:new(name, age): Función constructora para crear una nueva instancia.
  • setmetatable({}, Person): Establece la metatable de la nueva instancia a Person.
  • self.name y self.age: Variables de instancia.

Implementando Herencia

Para implementar la herencia, creamos una nueva clase que hereda de una clase existente usando el metamétodo __index.

-- Definir una subclase
Student = setmetatable({}, {__index = Person})
Student.__index = Student

-- Constructor
function Student:new(name, age, studentID)
    local instance = Person.new(self, name, age)
    setmetatable(instance, Student)
    instance.studentID = studentID
    return instance
end

-- Método para mostrar información del estudiante
function Student:display()
    Person.display(self)
    print("ID del Estudiante: " .. self.studentID)
end

-- Crear una instancia de Student
local student1 = Student:new("Bob", 22, "S12345")
student1:display()
-- Salida:
-- Nombre: Bob, Edad: 22
-- ID del Estudiante: S12345

Explicación

  • Student = setmetatable({}, {__index = Person}): Crea una nueva tabla para la subclase y establece su __index a Person.
  • Student:new(name, age, studentID): Función constructora para la subclase.
  • Person.new(self, name, age): Llama al constructor de la clase padre.
  • setmetatable(instance, Student): Establece la metatable de la nueva instancia a Student.
  • Person.display(self): Llama al método de la clase padre.

Resumen

La herencia en Lua se logra utilizando metatables y el metamétodo __index. Al establecer el __index de una subclase a su clase padre, podemos heredar propiedades y métodos. Esto permite la reutilización de código y una estructura jerárquica clara.

Puntos Clave

  • Usa tablas para representar clases.
  • Usa el metamétodo __index para habilitar la herencia.
  • Llama a los constructores y métodos de la clase padre usando el parámetro self.

Al entender y utilizar la herencia, puedes crear programas Lua más complejos y organizados, haciendo tu código más modular y fácil de mantener.
