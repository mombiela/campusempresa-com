Creando Tu Primera Aplicación Spring Boot
Entendiendo la Estructura del Proyecto Spring Boot ►

En esta sección, te guiaremos a través del proceso de crear tu primera aplicación Spring Boot. Esto incluirá configurar tu entorno de desarrollo, crear un nuevo proyecto Spring Boot y entender la estructura básica de una aplicación Spring Boot.

Configurando Tu Entorno de Desarrollo

Antes de comenzar a construir tu aplicación Spring Boot, necesitas configurar tu entorno de desarrollo.

  • Java Development Kit (JDK): Asegúrate de tener instalado JDK 8 o superior. Puedes descargarlo desde el sitio oficial de Oracle.
  • Entorno de Desarrollo Integrado (IDE): Recomendamos usar IntelliJ IDEA o Eclipse. Ambos tienen un excelente soporte para Spring Boot.
  • Maven o Gradle: Estas son herramientas de construcción que gestionan dependencias y construyen el proyecto. Maven es más comúnmente usado con Spring Boot.

Instalando JDK

  1. Descarga el JDK desde el sitio web de Oracle.
  2. Sigue las instrucciones de instalación para tu sistema operativo.
  3. Verifica la instalación ejecutando el siguiente comando en tu terminal: 
    java -version

Configurando Tu IDE

  1. IntelliJ IDEA:

    • Descarga e instala IntelliJ IDEA desde el sitio oficial de JetBrains.
    • Abre IntelliJ IDEA y configúralo para el desarrollo en Java.

  2. Eclipse:

    • Descarga e instala Eclipse desde el sitio oficial de Eclipse.
    • Abre Eclipse y configúralo para el desarrollo en Java.

Creando un Nuevo Proyecto Spring Boot

Puedes crear un nuevo proyecto Spring Boot usando Spring Initializr, una herramienta basada en la web proporcionada por el equipo de Spring.

Usando Spring Initializr

  1. Ve a Spring Initializr.
  2. Rellena los detalles del proyecto:
    • Project: Maven Project
    • Language: Java
    • Spring Boot: 2.5.4 (o la última versión estable)
    • Project Metadata:
      • Group: com.example
      • Artifact: demo
      • Name: demo
      • Description: Proyecto de demostración para Spring Boot
      • Package name: com.example.demo
      • Packaging: Jar
      • Java: 8 (o superior)
  3. Añade dependencias:
    • Spring Web: Para construir aplicaciones web, incluyendo servicios RESTful.
  4. Haz clic en el botón "Generate" para descargar el proyecto como un archivo ZIP.
  5. Extrae el archivo ZIP y abre el proyecto en tu IDE.

Una vez que hayas creado tu proyecto, verás la siguiente estructura:

Componentes Clave

  • DemoApplication.java: Esta es la clase principal que inicia la aplicación Spring Boot.

    package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
    • @SpringBootApplication: Esta anotación marca la clase principal de una aplicación Spring Boot.
    • SpringApplication.run(): Este método lanza la aplicación.

  • application.properties: Este archivo se utiliza para configurar tu aplicación Spring Boot. Puedes establecer varias propiedades como el puerto del servidor, configuraciones de la base de datos, etc.

Ejecutando Tu Aplicación Spring Boot

Para ejecutar tu aplicación Spring Boot, sigue estos pasos:

  1. Abre tu IDE y navega al archivo DemoApplication.java.
  2. Haz clic derecho en el archivo y selecciona "Run DemoApplication.main()".
