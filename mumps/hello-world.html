Introducción

MUMPS (Massachusetts General Hospital Utility Multi-Programming System), también conocido como M, es un lenguaje de programación con una rica historia en la industria de la salud. Es conocido por sus capacidades de base de datos integradas y todavía se utiliza en varios sistemas médicos y financieros. En esta sección, cubriremos los conceptos básicos para escribir un simple programa de "Hola Mundo" en MUMPS.

Conceptos Clave

  • Sintaxis de MUMPS: Entender la sintaxis básica de MUMPS.
  • Escribir un Programa: Cómo escribir y ejecutar un simple programa en MUMPS.
  • Salida: Mostrar salida en la consola.

Escribiendo un Simple Programa de "Hola Mundo"

Sintaxis de MUMPS

MUMPS tiene una sintaxis única que puede parecer inusual para aquellos familiarizados con lenguajes de programación más modernos. Aquí hay algunos puntos clave:

  • Comandos: Los comandos de MUMPS a menudo están abreviados (por ejemplo, WRITE puede abreviarse como W).
  • Continuación de Línea: Las líneas pueden continuarse usando un espacio o un punto (.).
  • Variables: Las variables no tienen tipo y no requieren declaración.

Código de Ejemplo

Escribamos un simple programa de "Hola Mundo" en MUMPS.

HELLO ; Esto es una etiqueta
    WRITE "Hola, Mundo!",!
    QUIT

Explicación

  • Etiqueta: HELLO es una etiqueta que marca el comienzo del programa. Las etiquetas son opcionales pero útiles para la organización.
  • Comando WRITE: El comando WRITE (o W) muestra texto en la consola. En este caso, escribe "Hola, Mundo!".
  • Nueva Línea: El ! al final de la cadena mueve el cursor a la siguiente línea.
  • Comando QUIT: El comando QUIT termina la ejecución del programa.

Ejecutando el Programa

Para ejecutar el programa, necesitas:

  1. Guardar el código en un archivo con una extensión .m, por ejemplo, hola.m.
  2. Abrir tu entorno MUMPS (por ejemplo, GT.M o Cache).
  3. Cargar y ejecutar el programa usando los siguientes comandos:
DO ^hola

Conclusión

Escribir un programa de "Hola Mundo" en MUMPS es una forma sencilla de familiarizarse con la sintaxis y los comandos básicos del lenguaje. Entender cómo escribir y ejecutar programas simples es el primer paso para dominar MUMPS. En las siguientes secciones, profundizaremos en características y funcionalidades más complejas del lenguaje.
