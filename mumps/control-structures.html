Estructuras de Control: IF, ELSE, FOR, WHILE

Introducción

Las estructuras de control son fundamentales en la programación ya que te permiten controlar el flujo de ejecución basado en condiciones y bucles. En esta sección, exploraremos las estructuras de control en MUMPS (M), incluyendo IF, ELSE, FOR y WHILE.

Sentencias IF y ELSE

La sentencia IF en MUMPS se usa para ejecutar un bloque de código solo si una condición especificada es verdadera. La sentencia ELSE se puede usar para ejecutar un bloque de código alternativo si la condición es falsa.

Sintaxis

IF condition DO
. ; código a ejecutar si la condición es verdadera
ELSE  DO
. ; código a ejecutar si la condición es falsa

Ejemplo

SET age=20
IF age<18 DO
. WRITE "Eres menor de edad.",!
ELSE  DO
. WRITE "Eres adulto.",!
  • Explicación: En este ejemplo, el programa verifica si la edad es menor de 18. Si es verdadero, imprime "Eres menor de edad." De lo contrario, imprime "Eres adulto."

Bucle FOR

El bucle FOR en MUMPS se usa para repetir un bloque de código un número específico de veces.

Sintaxis

FOR variable=start:step:end DO
. ; código a ejecutar en cada iteración

Ejemplo

FOR i=1:1:5 DO
. WRITE "Iteración: ",i,!
  • Explicación: Este bucle iterará de 1 a 5, incrementando i en 1 cada vez, e imprimirá el número de iteración actual.

Bucle WHILE

El bucle WHILE en MUMPS se usa para repetir un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

WHILE condition DO
. ; código a ejecutar mientras la condición sea verdadera

Ejemplo

SET count=1
WHILE count<=5 DO
. WRITE "Cuenta: ",count,!
. SET count=count+1
  • Explicación: Este bucle continuará ejecutándose mientras count sea menor o igual a 5. Imprime la cuenta actual y luego incrementa count en 1.

Combinando Estructuras de Control

Puedes combinar IF, ELSE, FOR y WHILE para crear flujos de control más complejos.

Ejemplo

SET num=1
FOR i=1:1:10 DO
. IF num#2=0 DO
. . WRITE "Número par: ",num,!
. ELSE  DO
. . WRITE "Número impar: ",num,!
. SET num=num+1
  • Explicación: Este ejemplo usa un bucle FOR para iterar 10 veces. Dentro del bucle, usa una sentencia IF para verificar si num es par o impar e imprime el mensaje apropiado. Luego, num se incrementa en 1.

Conclusión

Entender y usar efectivamente estructuras de control como IF, ELSE, FOR y WHILE es crucial para controlar el flujo de tus programas en MUMPS. Estos constructos te permiten tomar decisiones, repetir tareas y crear lógica compleja en tu código. Practica usando estas estructuras en diferentes escenarios para volverte competente en la programación en MUMPS.
