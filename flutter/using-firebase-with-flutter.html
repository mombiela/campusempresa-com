Usando Firebase con Flutter

Introducción a Firebase y Flutter

  • Firebase: Una plataforma desarrollada por Google para crear aplicaciones móviles y web. Proporciona una variedad de herramientas y servicios para ayudar a los desarrolladores a crear aplicaciones de alta calidad, incluyendo autenticación, bases de datos, análisis y más.
  • Flutter: Un kit de desarrollo de software de interfaz de usuario de código abierto creado por Google. Se utiliza para desarrollar aplicaciones para Android, iOS, Linux, Mac, Windows, Google Fuchsia y la web desde una única base de código.

Configuración de Firebase con Flutter

Requisitos previos

  • Conocimientos básicos de Flutter y Dart.
  • Una cuenta de Google para acceder a los servicios de Firebase.
  • SDK de Flutter instalado en tu máquina.

Creación de un proyecto de Firebase

  1. Ve a la Consola de Firebase.
  2. Haz clic en "Agregar proyecto" y sigue las instrucciones para crear un nuevo proyecto.
  3. Una vez creado el proyecto, agrega una aplicación de Android y/o iOS al proyecto.

Configuración de Firebase en Flutter

Configuración de Android

  1. Descarga el archivo google-services.json desde la Consola de Firebase y colócalo en el directorio android/app.
  2. Modifica el archivo android/build.gradle: 
    buildscript {
    dependencies {
        // Agrega esta línea
        classpath 'com.google.gms:google-services:4.3.3'
    }
}
  3. Modifica el archivo android/app/build.gradle: 
    apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services' // Agrega esta línea

Configuración de iOS

  1. Descarga el archivo GoogleService-Info.plist desde la Consola de Firebase y colócalo en el directorio ios/Runner.
  2. Abre ios/Runner.xcworkspace en Xcode.
  3. Agrega el archivo GoogleService-Info.plist al proyecto en Xcode.

Agregar dependencias de Firebase

Agrega las dependencias necesarias de Firebase a tu archivo pubspec.yaml:

dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^1.10.0
  firebase_auth: ^3.3.3
  cloud_firestore: ^3.1.5

Ejecuta flutter pub get para instalar las dependencias.

Autenticación de Firebase

Configuración de la autenticación

  1. En la Consola de Firebase, navega a la sección "Autenticación".
  2. Habilita los métodos de autenticación deseados (por ejemplo, Email/Contraseña, Google Sign-In).

Implementación de la autenticación en Flutter

Ejemplo de autenticación por Email/Contraseña

import 'package:firebase_auth/firebase_auth.dart';

class AuthService {
  final FirebaseAuth _auth = FirebaseAuth.instance;

  // Iniciar sesión con email y contraseña
  Future<User?> signInWithEmailAndPassword(String email, String password) async {
    try {
      UserCredential result = await _auth.signInWithEmailAndPassword(email: email, password: password);
      User? user = result.user;
      return user;
    } catch (e) {
      print(e.toString());
      return null;
    }
  }

  // Registrar con email y contraseña
  Future<User?> registerWithEmailAndPassword(String email, String password) async {
    try {
      UserCredential result = await _auth.createUserWithEmailAndPassword(email: email, password: password);
      User? user = result.user;
      return user;
    } catch (e) {
      print(e.toString());
      return null;
    }
  }

  // Cerrar sesión
  Future<void> signOut() async {
    try {
      return await _auth.signOut();
    } catch (e) {
      print(e.toString());
      return null;
    }
  }
}

Cloud Firestore

Configuración de Firestore

  1. En la Consola de Firebase, navega a la sección "Base de datos de Firestore".
  2. Haz clic en "Crear base de datos" y sigue las instrucciones para configurar Firestore.

Implementación de Firestore en Flutter

Agregar datos a Firestore

import 'package:cloud_firestore/cloud_firestore.dart';

class DatabaseService {
  final CollectionReference userCollection = FirebaseFirestore.instance.collection('users');

  Future<void> addUser(String uid, String name, String email) async {
    return await userCollection.doc(uid).set({
      'name': name,
      'email': email,
    });
  }
}

Recuperar datos de Firestore

import 'package:cloud_firestore/cloud_firestore.dart';

class DatabaseService {
  final CollectionReference userCollection = FirebaseFirestore.instance.collection('users');

  // Obtener datos del usuario
  Future<DocumentSnapshot> getUserData(String uid) async {
    return await userCollection.doc(uid).get();
  }

  // Flujo de datos del usuario
  Stream<DocumentSnapshot> getUserStream(String uid) {
    return userCollection.doc(uid).snapshots();
  }
}

Almacenamiento de Firebase

Configuración del almacenamiento de Firebase

  1. En la Consola de Firebase, navega a la sección "Almacenamiento".
  2. Haz clic en "Comenzar" y sigue las instrucciones para configurar el almacenamiento de Firebase.

Implementación del almacenamiento de Firebase en Flutter

Subir archivos al almacenamiento de Firebase

import 'package:firebase_storage/firebase_storage.dart';
import 'dart:io';

class StorageService {
  final FirebaseStorage _storage = FirebaseStorage.instance;

  Future<String> uploadFile(File file, String path) async {
    try {
      TaskSnapshot snapshot = await _storage.ref().child(path).putFile(file);
      String downloadUrl = await snapshot.ref.getDownloadURL();
      return downloadUrl;
    } catch (e) {
      print(e.toString());
      return '';
    }
  }
}

Descargar archivos del almacenamiento de Firebase

import 'package:firebase_storage/firebase_storage.dart';

class StorageService {
  final FirebaseStorage _storage = FirebaseStorage.instance;

  Future<String> getFileDownloadUrl(String path) async {
    try {
      String downloadUrl = await _storage.ref().child(path).getDownloadURL();
      return downloadUrl;
    } catch (e) {
      print(e.toString());
      return '';
    }
  }
}
