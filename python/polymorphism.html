Polimorfismo en Python
Encapsulación ►

El polimorfismo es un concepto central en la programación orientada a objetos (OOP) que permite que objetos de diferentes clases sean tratados como objetos de una superclase común. Es una característica poderosa que permite flexibilidad e integración en el código.

¿Qué es el Polimorfismo?

  • Definición: El polimorfismo significa "muchas formas" y permite que los métodos hagan cosas diferentes según el objeto sobre el que actúan.
  • Propósito: Ayuda a diseñar sistemas que son más modulares y fáciles de extender.

Tipos de Polimorfismo

  1. Polimorfismo en tiempo de compilación (Sobrecarga de métodos):

    • No es directamente soportado en Python.
    • Se logra utilizando argumentos predeterminados o argumentos de longitud variable.

  2. Polimorfismo en tiempo de ejecución (Sobrescritura de métodos):

    • Soportado en Python.
    • Permite que una subclase proporcione una implementación específica de un método que ya está definido en su superclase.

Sobrecarga de Métodos en Python

Python no soporta la sobrecarga de métodos tradicional como se ve en otros lenguajes como Java o C++. Sin embargo, puedes lograr una funcionalidad similar utilizando argumentos predeterminados o argumentos de longitud variable.

Ejemplo de Sobrecarga de Métodos Usando Argumentos Predeterminados

class MathOperations:
    def add(self, a, b=0, c=0):
        return a + b + c

math_op = MathOperations()
print(math_op.add(1))       # Salida: 1
print(math_op.add(1, 2))    # Salida: 3
print(math_op.add(1, 2, 3)) # Salida: 6

Ejemplo de Sobrecarga de Métodos Usando Argumentos de Longitud Variable

class MathOperations:
    def add(self, *args):
        return sum(args)

math_op = MathOperations()
print(math_op.add(1))       # Salida: 1
print(math_op.add(1, 2))    # Salida: 3
print(math_op.add(1, 2, 3)) # Salida: 6

Sobrescritura de Métodos en Python

La sobrescritura de métodos permite que una subclase proporcione una implementación específica de un método que ya está definido en su superclase.

Ejemplo de Sobrescritura de Métodos

class Animal:
    def sound(self):
        return "Some generic sound"

class Dog(Animal):
    def sound(self):
        return "Bark"

class Cat(Animal):
    def sound(self):
        return "Meow"

animals = [Dog(), Cat(), Animal()]

for animal in animals:
    print(animal.sound())

Salida:

Bark
Meow
Some generic sound

Polimorfismo con Herencia

El polimorfismo se usa a menudo con la herencia para permitir que una sola interfaz se use para una clase general de acciones.

Ejemplo de Polimorfismo con Herencia

class Shape:
    def area(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius * self.radius

shapes = [Rectangle(2, 3), Circle(5)]

for shape in shapes:
    print(shape.area())

Salida:

6
78.5

Polimorfismo con Funciones y Objetos

El polimorfismo también se puede lograr utilizando funciones que pueden tomar objetos de diferentes clases y realizar acciones basadas en su tipo de clase.

Ejemplo de Polimorfismo con Funciones

class Bird:
    def fly(self):
        return "Flying high"

class Airplane:
    def fly(self):
        return "Flying at 30,000 feet"

def let_it_fly(flying_object):
    print(flying_object.fly())

bird = Bird()
airplane = Airplane()

let_it_fly(bird)       # Salida: Flying high
let_it_fly(airplane)   # Salida: Flying at 30,000 feet

Resumen

El polimorfismo es un concepto fundamental en OOP que permite flexibilidad e integración en el código. Se puede lograr en Python a través de la sobrecarga de métodos (usando argumentos predeterminados o de longitud variable) y la sobrescritura de métodos. Al usar el polimorfismo, puedes diseñar sistemas que son más modulares y fáciles de extender, haciendo que tu código sea más mantenible y escalable.
