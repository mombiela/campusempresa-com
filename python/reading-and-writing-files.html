Introducción

Leer y escribir archivos son tareas fundamentales en la programación. En Python, estas operaciones son sencillas y se pueden realizar utilizando funciones y métodos integrados. Esta sección te guiará a través de los conceptos básicos del manejo de archivos, desde abrir y cerrar archivos hasta técnicas más avanzadas como manejar diferentes formatos de archivos.

Conceptos Básicos del Manejo de Archivos

Abrir y Cerrar Archivos

  • Abrir un Archivo: Usa la función open(). 
    file = open('example.txt', 'r')  # 'r' para modo lectura
  • Cerrar un Archivo: Siempre cierra el archivo después de las operaciones usando el método close(). 
    file.close()

Modos de Archivo

  • Leer (r): Abre un archivo para lectura.
  • Escribir (w): Abre un archivo para escritura (crea un nuevo archivo o trunca un archivo existente).
  • Añadir (a): Abre un archivo para añadir (crea un nuevo archivo si no existe).
  • Leer y Escribir (r+): Abre un archivo tanto para lectura como para escritura.

Ejemplo: Leer un Archivo

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)
  • Explicación: La declaración with asegura que el archivo se cierre correctamente después de que su bloque termine, incluso si se produce una excepción.

Ejemplo: Escribir en un Archivo

with open('example.txt', 'w') as file:
    file.write('¡Hola, Mundo!')
  • Explicación: Esto escribirá la cadena '¡Hola, Mundo!' en example.txt. Si el archivo ya existe, será sobrescrito.

Leer Archivos

Leer Todo el Archivo

  • Usando read(): Lee todo el contenido del archivo. 
    with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Leer Línea por Línea

  • Usando readline(): Lee una línea a la vez.

    with open('example.txt', 'r') as file:
    line = file.readline()
    while line:
        print(line, end='')
        line = file.readline()

  • Usando readlines(): Lee todas las líneas en una lista.

    with open('example.txt', 'r') as file:
    lines = file.readlines()
    for line in lines:
        print(line, end='')

Escribir Archivos

Escribir Múltiples Líneas

  • Usando writelines(): Escribe una lista de cadenas en un archivo. 
    lines = ['Primera línea\n', 'Segunda línea\n', 'Tercera línea\n']
with open('example.txt', 'w') as file:
    file.writelines(lines)

Añadir a un Archivo

  • Usando el modo a: Añade contenido al final del archivo. 
    with open('example.txt', 'a') as file:
    file.write('Esta es una línea añadida.\n')

Manejo Avanzado de Archivos

Trabajar con Archivos Binarios

  • Leer Archivos Binarios: Usa el modo rb.

    with open('example.bin', 'rb') as file:
    content = file.read()
    print(content)

  • Escribir Archivos Binarios: Usa el modo wb.

    with open('example.bin', 'wb') as file:
    file.write(b'\x00\x01\x02\x03')

Manejar Diferentes Formatos de Archivos

  • Archivos CSV: Usa el módulo csv.

    import csv

# Leer un archivo CSV
with open('example.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

# Escribir en un archivo CSV
with open('example.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Nombre', 'Edad'])
    writer.writerow(['Alicia', 30])

  • Archivos JSON: Usa el módulo json.

    import json

# Leer un archivo JSON
with open('example.json', 'r') as file:
    data = json.load(file)
    print(data)

# Escribir en un archivo JSON
data = {'nombre': 'Alicia', 'edad': 30}
with open('example.json', 'w') as file:
    json.dump(data, file)

Resumen

En esta sección, aprendiste los conceptos básicos y algunas técnicas avanzadas para leer y escribir archivos en Python. Ahora sabes cómo:

  • Abrir y cerrar archivos.
  • Leer y escribir en archivos de texto.
  • Manejar diferentes modos de archivo.
  • Trabajar con archivos binarios.
  • Leer y escribir archivos CSV y JSON.

Entender el manejo de archivos es crucial para muchas aplicaciones, como el procesamiento de datos, la gestión de configuraciones y el registro de eventos. Practica estos conceptos para volverte competente en la gestión de archivos en Python.
