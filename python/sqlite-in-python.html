Introducción a SQLite

SQLite es una biblioteca en lenguaje C que implementa un motor de base de datos SQL pequeño, rápido, autónomo, de alta fiabilidad y con todas las funciones. Es el motor de base de datos más ampliamente desplegado en el mundo. En esta sección, aprenderemos cómo usar SQLite con Python.

Configuración de SQLite en Python

Para usar SQLite en Python, necesitas importar el módulo sqlite3, que está incluido en la Biblioteca Estándar de Python.

import sqlite3

Creación de una Base de Datos

Puedes crear una nueva base de datos SQLite conectándote a un archivo de base de datos. Si el archivo no existe, SQLite lo creará.

conn = sqlite3.connect('example.db')
  • example.db es el nombre del archivo de la base de datos.
  • conn es el objeto de conexión.

Creación de una Tabla

Para crear una tabla, necesitas ejecutar un comando SQL usando el objeto de conexión.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
''')

conn.commit()
conn.close()
  • cursor.execute ejecuta el comando SQL.
  • conn.commit guarda los cambios.
  • conn.close cierra la conexión.

Inserción de Datos

Puedes insertar datos en la tabla usando el comando SQL INSERT INTO.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
INSERT INTO users (name, age) VALUES (?, ?)
''', ('Alice', 30))

conn.commit()
conn.close()
  • Los marcadores de posición ? se usan para prevenir la inyección SQL.
  • Los valores se proporcionan como una tupla.

Consulta de Datos

Para recuperar datos, puedes usar el comando SQL SELECT.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('SELECT * FROM users')
rows = cursor.fetchall()

for row in rows:
    print(row)

conn.close()
  • cursor.fetchall recupera todas las filas del conjunto de resultados.

Actualización de Datos

Para actualizar datos existentes, usa el comando SQL UPDATE.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
UPDATE users SET age = ? WHERE name = ?
''', (31, 'Alice'))

conn.commit()
conn.close()

Eliminación de Datos

Para eliminar datos, usa el comando SQL DELETE.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
DELETE FROM users WHERE name = ?
''', ('Alice',))

conn.commit()
conn.close()

Uso de Administradores de Contexto

Usar administradores de contexto (instrucción with) asegura que la conexión se cierre correctamente incluso si ocurre un error.

with sqlite3.connect('example.db') as conn:
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)

Manejo de Excepciones

Es importante manejar excepciones para evitar fallos y asegurar que el programa se ejecute sin problemas.

try:
    conn = sqlite3.connect('example.db')
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)
except sqlite3.Error as e:
    print(f"An error occurred: {e}")
finally:
    if conn:
        conn.close()

Resumen

En esta sección, cubrimos los conceptos básicos de usar SQLite con Python, incluyendo:

  • Configuración de SQLite en Python.
  • Creación de una base de datos y tablas.
  • Inserción, consulta, actualización y eliminación de datos.
  • Uso de administradores de contexto y manejo de excepciones.

SQLite es una herramienta poderosa para gestionar bases de datos pequeñas a medianas, y su integración con Python lo convierte en una excelente opción para muchas aplicaciones.
