Introducción a JSON

  • JSON (JavaScript Object Notation) es un formato de intercambio de datos ligero.
  • Es fácil de leer y escribir para los humanos, y fácil de analizar y generar para las máquinas.
  • JSON se utiliza a menudo para transmitir datos entre un servidor y una aplicación web.

Conceptos Básicos de JSON

  • Los datos JSON se escriben como pares clave/valor.
  • Las claves son cadenas, y los valores pueden ser cadenas, números, objetos, matrices, true, false o null.

Ejemplo de Datos JSON

{
  "name": "John",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science"],
  "address": {
    "street": "123 Main St",
    "city": "Anytown"
  }
}

Trabajando con JSON en Python

  • Python proporciona un módulo incorporado llamado json para manejar datos JSON.

Importando el Módulo JSON

import json

Leyendo Datos JSON

  • Los datos JSON se pueden leer desde una cadena o un archivo.

Leyendo JSON desde una Cadena

import json

json_string = '{"name": "John", "age": 30, "isStudent": false}'
data = json.loads(json_string)

print(data)
# Salida: {'name': 'John', 'age': 30, 'isStudent': False}

Leyendo JSON desde un Archivo

import json

with open('data.json', 'r') as file:
    data = json.load(file)

print(data)
# Salida: {'name': 'John', 'age': 30, 'isStudent': False}

Escribiendo Datos JSON

  • Los datos JSON se pueden escribir en una cadena o un archivo.

Escribiendo JSON en una Cadena

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

json_string = json.dumps(data)
print(json_string)
# Salida: {"name": "John", "age": 30, "isStudent": false}

Escribiendo JSON en un Archivo

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

with open('data.json', 'w') as file:
    json.dump(data, file)

Manejo Avanzado de JSON

  • Personalización de la codificación y decodificación JSON.
  • Manejo de tipos de datos complejos.

Personalización de la Codificación JSON

  • Usa el parámetro default en json.dumps() para manejar objetos no serializables.
import json
from datetime import datetime

def datetime_handler(x):
    if isinstance(x, datetime):
        return x.isoformat()
    raise TypeError("Unknown type")

data = {
    "name": "John",
    "joined": datetime(2021, 5, 17)
}

json_string = json.dumps(data, default=datetime_handler)
print(json_string)
# Salida: {"name": "John", "joined": "2021-05-17T00:00:00"}

Personalización de la Decodificación JSON

  • Usa el parámetro object_hook en json.loads() para manejar la decodificación personalizada.
import json
from datetime import datetime

def datetime_parser(dct):
    for k, v in dct.items():
        if isinstance(v, str) and len(v) == 10:
            try:
                dct[k] = datetime.strptime(v, '%Y-%m-%d')
            except ValueError:
                pass
    return dct

json_string = '{"name": "John", "joined": "2021-05-17"}'
data = json.loads(json_string, object_hook=datetime_parser)
print(data)
# Salida: {'name': 'John', 'joined': datetime.datetime(2021, 5, 17, 0, 0)}

Conclusión

  • JSON es un formato versátil para el intercambio de datos.
  • El módulo json de Python proporciona métodos simples para leer y escribir datos JSON.
  • La personalización avanzada permite manejar tipos de datos complejos y codificación/decodificación personalizada.
  • Dominar el manejo de JSON en Python es esencial para trabajar con APIs y servicios web.
