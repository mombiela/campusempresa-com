Introducción a las Bases de Datos NoSQL

Las bases de datos NoSQL están diseñadas para manejar grandes volúmenes de datos, altas cargas de usuarios y cambios frecuentes en las estructuras de datos. A diferencia de las bases de datos SQL tradicionales, las bases de datos NoSQL no utilizan un esquema fijo, lo que las hace más flexibles y escalables.

Conceptos Clave

  • Sin esquema: Las bases de datos NoSQL no requieren un esquema predefinido.
  • Escalabilidad: Diseñadas para escalar distribuyendo datos a través de múltiples servidores.
  • Tipos de Bases de Datos NoSQL:
    • Almacenes de Documentos: Almacenan datos en documentos similares a JSON (por ejemplo, MongoDB).
    • Almacenes Clave-Valor: Almacenan datos como pares clave-valor (por ejemplo, Redis).
    • Almacenes de Columnas: Almacenan datos en columnas en lugar de filas (por ejemplo, Cassandra).
    • Bases de Datos de Grafos: Almacenan datos en estructuras de grafos (por ejemplo, Neo4j).

Configuración del Entorno

Antes de interactuar con bases de datos NoSQL en Python, necesitas instalar las bibliotecas necesarias. Para este curso, usaremos MongoDB como ejemplo.

Instalación de MongoDB y pymongo

  1. Instalar MongoDB: Sigue las instrucciones en el sitio web oficial de MongoDB para instalar MongoDB en tu sistema.
  2. Instalar pymongo: Usa pip para instalar la biblioteca pymongo, que permite a Python interactuar con MongoDB. 
    pip install pymongo

Conectarse a MongoDB

Para interactuar con MongoDB, necesitas establecer una conexión usando la biblioteca pymongo.

Ejemplo: Conectarse a MongoDB

from pymongo import MongoClient

# Crear un MongoClient para la instancia de MongoDB en ejecución
client = MongoClient('localhost', 27017)

# Acceder a la base de datos 'test'
db = client.test

print("Conectado a MongoDB")
  • MongoClient: Esta clase se usa para crear una conexión con el servidor MongoDB.
  • localhost: El nombre del host donde se está ejecutando MongoDB.
  • 27017: El número de puerto predeterminado para MongoDB.

Operaciones CRUD

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las operaciones básicas que puedes realizar en una base de datos.

Crear

Para insertar un documento en una colección, usa los métodos insert_one o insert_many.

Ejemplo: Insertar un Documento

# Definir un documento de ejemplo
document = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}

# Insertar el documento en la colección 'users'
result = db.users.insert_one(document)

print(f"Documento insertado con ID: {result.inserted_id}")

Leer

Para recuperar documentos de una colección, usa los métodos find o find_one.

Ejemplo: Recuperar Documentos

# Recuperar un solo documento
user = db.users.find_one({"name": "Alice"})
print(user)

# Recuperar múltiples documentos
users = db.users.find({"age": {"$gt": 25}})
for user in users:
    print(user)
  • find_one: Recupera un solo documento que coincida con la consulta.
  • find: Recupera múltiples documentos que coincidan con la consulta.
  • $gt: Un operador de consulta de MongoDB que significa "mayor que".

Actualizar

Para actualizar documentos, usa los métodos update_one o update_many.

Ejemplo: Actualizar un Documento

# Actualizar la edad del usuario llamado 'Alice'
result = db.users.update_one(
    {"name": "Alice"},  # Consulta para coincidir con el documento
    {"$set": {"age": 31}}  # Operación de actualización
)

print(f"Documentos coincidentes: {result.matched_count}, Documentos modificados: {result.modified_count}")
  • $set: Un operador de actualización de MongoDB usado para establecer el valor de un campo.

Eliminar

Para eliminar documentos, usa los métodos delete_one o delete_many.

Ejemplo: Eliminar un Documento

# Eliminar el usuario llamado 'Alice'
result = db.users.delete_one({"name": "Alice"})

print(f"Documentos eliminados: {result.deleted_count}")

Resumen

En este tema, cubrimos los conceptos básicos de la interacción con bases de datos NoSQL usando Python, enfocándonos específicamente en MongoDB. Discutimos:

  • Los conceptos clave de las bases de datos NoSQL.
  • Configuración del entorno e instalación de bibliotecas necesarias.
  • Establecimiento de una conexión con MongoDB.
  • Realización de operaciones CRUD: Crear, Leer, Actualizar y Eliminar.

Al entender estas operaciones fundamentales, puedes gestionar y manipular datos de manera efectiva en una base de datos NoSQL usando Python.
