Componentes de UI Comunes
ConstraintLayout ►

En esta sección, exploraremos los componentes de UI comunes utilizados en el desarrollo de Android. Comprender estos componentes es esencial para crear aplicaciones intuitivas y visualmente atractivas. Cubriremos componentes desde básicos hasta avanzados, proporcionando ejemplos y ejercicios para solidificar tu comprensión.

Componentes Básicos de UI

TextView

  • Descripción: Muestra texto al usuario.
  • Atributos: text, textColor, textSize, gravity, etc.
<TextView
    android:id="@+id/textView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="¡Hola, Mundo!"
    android:textSize="18sp"
    android:textColor="#000000"
    android:gravity="center"/>

EditText

  • Descripción: Permite al usuario ingresar y editar texto.
  • Atributos: hint, inputType, text, etc.
<EditText
    android:id="@+id/editText"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:hint="Ingresa tu nombre"
    android:inputType="textPersonName"/>

Button

  • Descripción: Un botón clicable que realiza una acción.
  • Atributos: text, onClick, background, etc.
<Button
    android:id="@+id/button"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Enviar"
    android:onClick="onSubmit"/>

ImageView

  • Descripción: Muestra una imagen.
  • Atributos: src, scaleType, contentDescription, etc.
<ImageView
    android:id="@+id/imageView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:src="@drawable/ic_launcher_foreground"
    android:contentDescription="Icono de la App"/>

Componentes Intermedios de UI

RecyclerView

  • Descripción: Una vista flexible para proporcionar una ventana limitada a un gran conjunto de datos.
  • Atributos: layoutManager, adapter, etc.
<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/recyclerView"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:layoutManager="LinearLayoutManager"/>

CardView

  • Descripción: Un diseño de marco con esquinas redondeadas y una sombra.
  • Atributos: cardCornerRadius, cardElevation, etc.
<androidx.cardview.widget.CardView
    android:id="@+id/cardView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:layout_margin="8dp"
    app:cardCornerRadius="4dp"
    app:cardElevation="2dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Ejemplo de CardView"/>
</androidx.cardview.widget.CardView>

Toolbar

  • Descripción: Una generalización de ActionBar para usar dentro de los diseños de la aplicación.
  • Atributos: title, subtitle, navigationIcon, etc.
<com.google.android.material.appbar.Toolbar
    android:id="@+id/toolbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="?attr/colorPrimary"
    android:title="Mi Toolbar"/>

Componentes Avanzados de UI

NavigationView

  • Descripción: Un menú de navegación estándar para la aplicación.
  • Atributos: menu, headerLayout, etc.
<com.google.android.material.navigation.NavigationView
    android:id="@+id/navigationView"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:layout_gravity="start"
    app:menu="@menu/navigation_menu"
    app:headerLayout="@layout/nav_header"/>

ViewPager2

  • Descripción: Permite al usuario deslizarse hacia la izquierda y derecha a través de páginas de datos.
  • Atributos: orientation, adapter, etc.
<androidx.viewpager2.widget.ViewPager2
    android:id="@+id/viewPager"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="horizontal"/>

ConstraintLayout

  • Descripción: Un diseño que te permite crear diseños grandes y complejos con una jerarquía de vista plana.
  • Atributos: constraintSet, layout_constraint, etc.
<androidx.constraintlayout.widget.ConstraintLayout
    android:id="@+id/constraintLayout"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Haz clic aquí"
        app:layout_constraintTop_toTopOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>
</androidx.constraintlayout.widget.ConstraintLayout>

Resumen

En este tema, cubrimos una variedad de componentes de UI comunes en el desarrollo de Android, desde elementos básicos como TextView y Button hasta componentes más avanzados como RecyclerView y ConstraintLayout. Cada componente tiene un propósito específico y puede ser personalizado a través de varios atributos para adaptarse a las necesidades de tu aplicación. Al comprender y utilizar estos componentes, puedes crear interfaces más dinámicas y amigables para el usuario en tus aplicaciones de Android.
