Introducción a Jetpack Compose

Jetpack Compose es la herramienta moderna de Android para construir interfaces de usuario nativas. Simplifica y acelera el desarrollo de UI en Android con menos código, herramientas poderosas y APIs intuitivas de Kotlin.

Conceptos Clave

  • UI Declarativa: Compose te permite describir tu UI de manera declarativa.
  • Funciones Composables: Funciones anotadas con @Composable que definen tus componentes de UI.
  • Gestión de Estado: Manejo del estado de la UI de manera reactiva.
  • Layouts: Organizar componentes de UI de manera flexible y eficiente.

Configuración de Jetpack Compose

Antes de sumergirte en Jetpack Compose, asegúrate de que tu entorno de desarrollo esté configurado correctamente.

Pasos para Configurar

  1. Instalar Android Studio: Asegúrate de tener la última versión de Android Studio.
  2. Crear un Nuevo Proyecto: Inicia un nuevo proyecto con Jetpack Compose habilitado.
  3. Agregar Dependencias: Asegúrate de que tus archivos build.gradle incluyan las dependencias necesarias para Jetpack Compose.
dependencies {
    implementation "androidx.compose.ui:ui:1.0.0"
    implementation "androidx.compose.material:material:1.0.0"
    implementation "androidx.compose.ui:ui-tooling-preview:1.0.0"
    implementation "androidx.lifecycle:lifecycle-runtime-ktx:2.3.1"
    implementation "androidx.activity:activity-compose:1.3.0-alpha06"
}

Construyendo tu Primer Composable

Comencemos creando una función composable simple que muestre un texto.

Ejemplo: Hola Mundo

import androidx.compose.material.Text
import androidx.compose.runtime.Composable

@Composable
fun Greeting(name: String) {
    Text(text = "Hello, $name!")
}

Explicación

  • @Composable: Esta anotación marca la función como composable.
  • Text: Un composable básico que muestra texto en la pantalla.
  • Parámetros: El parámetro name permite contenido dinámico.

Layouts en Jetpack Compose

Compose proporciona varios composables de layout para organizar tus componentes de UI.

Ejemplo: Column y Row

import androidx.compose.foundation.layout.Column
import androidx.compose.foundation.layout.Row
import androidx.compose.material.Text
import androidx.compose.runtime.Composable

@Composable
fun ColumnExample() {
    Column {
        Text("First item")
        Text("Second item")
        Text("Third item")
    }
}

@Composable
fun RowExample() {
    Row {
        Text("First item")
        Text("Second item")
        Text("Third item")
    }
}

Explicación

  • Column: Organiza los hijos verticalmente.
  • Row: Organiza los hijos horizontalmente.

Gestión de Estado

Gestionar el estado en Jetpack Compose es crucial para construir UIs interactivas.

Ejemplo: Estado en Compose

import androidx.compose.runtime.*
import androidx.compose.material.*

@Composable
fun Counter() {
    var count by remember { mutableStateOf(0) }

    Column {
        Text("Count: $count")
        Button(onClick = { count++ }) {
            Text("Increment")
        }
    }
}

Explicación

  • remember: Se usa para recordar el estado a través de recomposiciones.
  • mutableStateOf: Crea un estado mutable.
  • Button: Un composable que desencadena una acción cuando se hace clic.

Temas Avanzados

A medida que te sientas más cómodo con Jetpack Compose, puedes explorar temas más avanzados.

Composables Personalizados

Crear composables personalizados permite componentes de UI reutilizables y modulares.

@Composable
fun CustomCard(content: @Composable () -> Unit) {
    Card(elevation = 4.dp) {
        content()
    }
}

Tematización y Estilización

Compose te permite definir temas y estilos para mantener una apariencia y sensación consistentes.

import androidx.compose.material.MaterialTheme
import androidx.compose.material.Typography

@Composable
fun MyApp() {
    MaterialTheme(
        typography = Typography(
            h1 = TextStyle(
                fontWeight = FontWeight.Bold,
                fontSize = 30.sp
            )
        )
    ) {
        // Tu contenido de la app
    }
}

Conclusión

Jetpack Compose es una herramienta poderosa que simplifica el desarrollo de UI en Android. Al entender los conceptos básicos de funciones composables, layouts y gestión de estado, puedes construir UIs dinámicas y responsivas. A medida que avances, explorar composables personalizados y tematización te ayudará a crear aplicaciones más complejas y pulidas. ¡Sigue experimentando y construyendo para dominar Jetpack Compose!
