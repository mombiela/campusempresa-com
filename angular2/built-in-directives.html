Introducción

Angular proporciona un conjunto de directivas integradas que te ayudan a gestionar formularios, listas, estilos y más. Estas directivas son esenciales para construir aplicaciones dinámicas y responsivas. En esta sección, cubriremos las directivas integradas más comúnmente usadas, sus propósitos y cómo utilizarlas de manera efectiva.

Directivas Estructurales

Las directivas estructurales son responsables de alterar el diseño del DOM añadiendo o eliminando elementos. Las directivas estructurales más comúnmente usadas en Angular son *ngIf, *ngFor y *ngSwitch.

`*ngIf`

La directiva *ngIf incluye condicionalmente una plantilla basada en el valor de una expresión.

Ejemplo:

<div *ngIf="isLoggedIn">
  ¡Bienvenido, usuario!
</div>

En este ejemplo, el elemento div solo se renderizará si isLoggedIn es true.

`*ngFor`

La directiva *ngFor se utiliza para repetir una plantilla para cada elemento en una lista.

Ejemplo:

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

Aquí, se crea una lista desordenada, y cada elemento li representa un elemento en el array items.

`*ngSwitch`

La directiva *ngSwitch se utiliza para incluir condicionalmente una de varias plantillas posibles basadas en el valor de una expresión.

Ejemplo:

<div [ngSwitch]="status">
  <p *ngSwitchCase="'active'">Activo</p>
  <p *ngSwitchCase="'inactive'">Inactivo</p>
  <p *ngSwitchDefault>Estado desconocido</p>
</div>

En este ejemplo, se renderizará el párrafo que coincida con el valor de status.

Directivas de Atributo

Las directivas de atributo cambian la apariencia o el comportamiento de un elemento, componente u otra directiva. Las directivas de atributo comunes incluyen ngClass, ngStyle y ngModel.

`ngClass`

La directiva ngClass añade y elimina clases CSS en un elemento HTML.

Ejemplo:

<div [ngClass]="{'active': isActive, 'disabled': isDisabled}">
  Este es un div con clases dinámicas.
</div>

En este ejemplo, el elemento div tendrá la clase active si isActive es true y la clase disabled si isDisabled es true.

`ngStyle`

La directiva ngStyle te permite establecer estilos en línea de manera dinámica.

Ejemplo:

<div [ngStyle]="{'color': textColor, 'font-size': fontSize + 'px'}">
  Este es un div con estilos dinámicos.
</div>

Aquí, el color del texto y el tamaño de la fuente del elemento div se establecen basándose en los valores de textColor y fontSize.

`ngModel`

La directiva ngModel vincula un elemento input, select o textarea a una propiedad en el scope.

Ejemplo:

<input [(ngModel)]="username" placeholder="Introduce tu nombre de usuario">
<p>Tu nombre de usuario es: {{ username }}</p>

En este ejemplo, el campo de entrada está vinculado a la propiedad username, y cualquier cambio en la entrada actualizará la propiedad username y viceversa.

Resumen

Las directivas integradas en Angular son herramientas poderosas que te permiten manipular el DOM y gestionar la apariencia y el comportamiento de tu aplicación de manera dinámica. Entender y usar efectivamente estas directivas es crucial para construir aplicaciones robustas y responsivas en Angular. Cubrimos las directivas estructurales y de atributo más comúnmente usadas, incluyendo *ngIf, *ngFor, *ngSwitch, ngClass, ngStyle y ngModel. Practica usando estas directivas en tus proyectos para volverte más competente en el desarrollo con Angular.
