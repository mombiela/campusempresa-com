Introducción a las Tuberías

Las tuberías en Angular son una característica poderosa que te permite transformar datos directamente en tus plantillas. Pueden usarse para formatear datos, como fechas, números y cadenas, haciendo que tus plantillas sean más limpias y legibles.

Conceptos Clave

  • Tubería: Una función que toma datos como entrada y los transforma en una salida deseada.
  • Tuberías Integradas: Angular proporciona varias tuberías integradas para transformaciones de datos comunes.
  • Tuberías Personalizadas: Puedes crear tus propias tuberías para transformaciones de datos personalizadas (cubierto en otra sección).

Tuberías Integradas Comunes

Angular proporciona varias tuberías integradas que puedes usar de inmediato:

  1. DatePipe: Formatea un valor de fecha según las reglas de la configuración regional.
  2. UpperCasePipe: Transforma texto a mayúsculas.
  3. LowerCasePipe: Transforma texto a minúsculas.
  4. CurrencyPipe: Transforma un número en una cadena de moneda.
  5. DecimalPipe: Transforma un número en una cadena con un formato decimal especificado.
  6. PercentPipe: Transforma un número en una cadena de porcentaje.
  7. JsonPipe: Convierte un valor en una cadena JSON.
  8. SlicePipe: Crea un nuevo arreglo o cadena que contiene un subconjunto (rebanada) de los elementos.

Uso de Tuberías Integradas

DatePipe

El DatePipe se usa para formatear valores de fecha.

<p>{{ today | date }}</p>
<p>{{ today | date:'short' }}</p>
<p>{{ today | date:'fullDate' }}</p>
<p>{{ today | date:'yyyy-MM-dd' }}</p>

UpperCasePipe y LowerCasePipe

Estas tuberías transforman texto a mayúsculas o minúsculas, respectivamente.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

El CurrencyPipe formatea un número como moneda.

<p>{{ 1234.56 | currency }}</p>
<p>{{ 1234.56 | currency:'USD' }}</p>
<p>{{ 1234.56 | currency:'EUR':'symbol':'4.2-2' }}</p>

DecimalPipe

El DecimalPipe formatea un número con un formato decimal especificado.

<p>{{ 1234.56 | number }}</p>
<p>{{ 1234.56 | number:'3.1-2' }}</p>

PercentPipe

El PercentPipe formatea un número como un porcentaje.

<p>{{ 0.1234 | percent }}</p>
<p>{{ 0.1234 | percent:'2.2-2' }}</p>

JsonPipe

El JsonPipe convierte un objeto en una cadena JSON.

<p>{{ { name: 'John', age: 30 } | json }}</p>

SlicePipe

El SlicePipe crea un nuevo arreglo o cadena que contiene un subconjunto (rebanada) de los elementos.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

Ejemplos y Ejercicios

Ejemplo 1: Formateo de Fechas

Crea un componente que muestre la fecha actual en varios formatos usando el DatePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-date-example',
  template: `
    <p>Default: {{ today | date }}</p>
    <p>Short: {{ today | date:'short' }}</p>
    <p>Full Date: {{ today | date:'fullDate' }}</p>
    <p>Custom: {{ today | date:'yyyy-MM-dd' }}</p>
  `
})
export class DateExampleComponent {
  today: number = Date.now();
}

Ejercicio 1: Uso de CurrencyPipe

Crea un componente que muestre el precio de un producto en diferentes monedas usando el CurrencyPipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-currency-example',
  template: `
    <p>USD: {{ price | currency:'USD' }}</p>
    <p>EUR: {{ price | currency:'EUR' }}</p>
    <p>GBP: {{ price | currency:'GBP' }}</p>
  `
})
export class CurrencyExampleComponent {
  price: number = 1234.56;
}

Ejercicio 2: Transformación de Texto

Crea un componente que muestre una cadena en mayúsculas y minúsculas usando UpperCasePipe y LowerCasePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-text-transform-example',
  template: `
    <p>Original: {{ text }}</p>
    <p>Uppercase: {{ text | uppercase }}</p>
    <p>Lowercase: {{ text | lowercase }}</p>
  `
})
export class TextTransformExampleComponent {
  text: string = 'Hello Angular';
}

Conclusión

Las tuberías integradas en Angular proporcionan una manera conveniente de formatear y transformar datos directamente en tus plantillas. Al entender y utilizar estas tuberías, puedes hacer que tus plantillas sean más limpias y legibles. Practica usando estas tuberías en diferentes escenarios para familiarizarte más con su sintaxis y capacidades.
