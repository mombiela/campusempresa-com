Se encarga de diseñar y administrar sistemas de inteligencia de negocios que permiten analizar grandes volúmenes de datos para la toma de decisiones.

Competencias técnicas

Python
Lenguaje de programación multipropósito popular por su sencillez y gran ecosistema para ciencia de datos, backend y más.
Power BI
Herramienta de análisis de datos de Microsoft que permite crear informes y dashboards interactivos para la visualización de datos empresariales.
SQL
Lenguaje de consulta estructurado utilizado para gestionar y manipular bases de datos relacionales.
Tableau
Plataforma de visualización de datos que permite crear gráficos interactivos y dashboards para la toma de decisiones basada en datos.
R
Lenguaje de programación especializado en análisis estadístico y visualización de datos.
BigQuery
Plataforma de análisis de datos de Google Cloud que permite el procesamiento de grandes volúmenes de datos en tiempo real.
MongoDB
Base de datos NoSQL orientada a documentos, utilizada para almacenar datos de manera flexible y escalable.

Conocimientos

Arquitecturas de Datos
Diseño de infraestructuras de almacenamiento y gestión de datos que soporten los objetivos de análisis y procesamiento en una organización.
Fundamentos de Bases de Datos
Introducción a los principios de bases de datos relacionales y no relacionales, diseño de esquemas y normalización.
Estadística
Conocimiento de los conceptos y métodos estadísticos utilizados para analizar y resumir datos, incluyendo distribuciones, inferencia y probabilidad.
Analítica de Negocios
Conocimiento de técnicas y herramientas que permiten analizar el rendimiento de las operaciones comerciales y optimizar la toma de decisiones basada en datos.
Análisis de Datos
Técnicas y métodos utilizados para examinar, limpiar y modelar datos con el objetivo de descubrir información útil y apoyar la toma de decisiones.
Visualización de Datos
Técnicas y herramientas utilizadas para representar gráficamente los datos, facilitando su interpretación y análisis.

Competencias sociales

Pensamiento analítico
Habilidad para analizar información de manera lógica y detallada para tomar decisiones fundamentadas.
Gestión del tiempo
Habilidad para organizar y planificar el tiempo de manera eficiente, cumpliendo con plazos establecidos.
Presentación de resultados
Habilidad para comunicar de manera efectiva los hallazgos, análisis o resultados a audiencias técnicas o no técnicas.
