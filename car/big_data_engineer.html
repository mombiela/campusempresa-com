Desarrolla y gestiona la infraestructura necesaria para almacenar y procesar grandes volúmenes de datos, asegurando su eficiencia y escalabilidad.

Competencias técnicas

Python
Lenguaje de programación multipropósito popular por su sencillez y gran ecosistema para ciencia de datos, backend y más.
Scala
Lenguaje de programación que combina programación funcional y orientada a objetos, usado para aplicaciones escalables.
GCP
Google Cloud Platform, conjunto de servicios en la nube de Google que incluye infraestructura, bases de datos y análisis de datos.
Kafka
Plataforma de procesamiento de flujos de datos en tiempo real, utilizada para gestionar grandes volúmenes de eventos y datos en sistemas distribuidos.
Apache Spark
Motor de procesamiento distribuido de código abierto utilizado para el análisis de grandes volúmenes de datos.
Terraform
Herramienta de infraestructura como código utilizada para automatizar la creación y gestión de entornos cloud.
Bash
Shell de Unix utilizado para la automatización de tareas y la ejecución de comandos en sistemas operativos basados en Unix.
Hadoop
Framework de código abierto para el almacenamiento y procesamiento de grandes volúmenes de datos distribuidos.
AWS
Plataforma de servicios en la nube de Amazon que ofrece infraestructura, almacenamiento, bases de datos y más.

Conocimientos

Arquitecturas Distribuidas
Conocimiento de sistemas distribuidos y cómo se gestionan y procesan datos a gran escala de manera eficiente y segura.
Big Data
Conjunto de tecnologías y prácticas utilizadas para almacenar, procesar y analizar grandes volúmenes de datos.
Procesamiento de Datos Masivos
Técnicas y tecnologías utilizadas para manejar grandes volúmenes de datos, optimizando el almacenamiento, procesamiento y análisis en tiempo real.
IaaS, PaaS y SaaS
Modelos de servicios en la nube: Infraestructura como Servicio, Plataforma como Servicio y Software como Servicio.
CI/CD
Integración y despliegue continuo (CI/CD), prácticas para automatizar la construcción, pruebas y entrega de software.

Competencias sociales

Gestión de proyectos (Soft Skill)
Capacidad para planificar, organizar y dirigir proyectos, asegurando su éxito dentro de plazos y presupuestos establecidos.
Resolución de problemas
Capacidad para identificar problemas y desarrollar soluciones eficientes y efectivas.
Adaptabilidad
Capacidad para ajustarse rápidamente a nuevas circunstancias, tecnologías o métodos de trabajo.
