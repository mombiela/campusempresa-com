Competencias técnicas Conocimientos Competencias sociales

Protege sistemas, redes y datos contra ciberataques mediante la implementación de medidas de seguridad y la gestión de riesgos.

Competencias técnicas

Kubernetes
Sistema de orquestación para la automatización del despliegue, escalado y gestión de aplicaciones en contenedores.
Docker
Plataforma de contenedores que permite empaquetar aplicaciones y sus dependencias para un entorno aislado.
Ansible
Herramienta de automatización de TI utilizada para la configuración y gestión de sistemas.
Git
Sistema de control de versiones distribuido utilizado para el seguimiento de cambios en el código.
GCP
Google Cloud Platform, conjunto de servicios en la nube de Google que incluye infraestructura, bases de datos y análisis de datos.
Linux
Sistema operativo de código abierto ampliamente utilizado en servidores y entornos de desarrollo.
Jenkins
Herramienta de integración continua utilizada para automatizar la construcción, pruebas y despliegue de aplicaciones.
Bash
Shell de Unix utilizado para la automatización de tareas y la ejecución de comandos en sistemas operativos basados en Unix.
AWS
Plataforma de servicios en la nube de Amazon que ofrece infraestructura, almacenamiento, bases de datos y más.
Azure
Plataforma de servicios en la nube de Microsoft, que ofrece soluciones de almacenamiento, cómputo y bases de datos.

Conocimientos

OWASP
Conjunto de directrices y estándares para la seguridad en aplicaciones web, creado por la Open Web Application Security Project.
Redes
Curso sobre conceptos básicos de redes, protocolos de comunicación, y modelos como el TCP/IP y OSI.
Fundamentos de Seguridad Informática
Curso sobre principios básicos de seguridad, ciberseguridad, y criptografía, cubriendo riesgos y medidas de protección.
Pentesting
Técnicas de pruebas de penetración para evaluar la seguridad de sistemas y aplicaciones mediante simulación de ciberataques.
Fundamentos de Sistemas Operativos
Introducción a los sistemas operativos, gestión de recursos, concurrencia y estructuras de archivos.

Competencias sociales

Pensamiento crítico
Capacidad para evaluar y cuestionar información de manera lógica y objetiva para tomar decisiones bien fundamentadas.
Atención al detalle
Habilidad para trabajar con precisión y cuidado, revisando minuciosamente el trabajo para evitar errores.
Trabajo bajo presión
Habilidad para mantener la calma y el enfoque cuando se trabaja en situaciones de alta demanda o con plazos ajustados.
