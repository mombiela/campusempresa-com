Introducción al Núcleo de Linux

El núcleo de Linux es el componente central del sistema operativo Linux. Gestiona el hardware y los recursos del sistema, permitiendo que el software interactúe con el hardware de manera controlada.

  • Conceptos Clave:
    • Espacio del Núcleo vs. Espacio de Usuario: El núcleo opera en un modo privilegiado llamado espacio del núcleo, mientras que las aplicaciones se ejecutan en el espacio de usuario.
    • Núcleo Monolítico: El núcleo de Linux es monolítico, lo que significa que incluye todos los controladores y servicios del sistema necesarios dentro de un único binario grande.
    • Llamadas al Sistema: Interfaces proporcionadas por el núcleo para que las aplicaciones en espacio de usuario soliciten servicios.

Entendiendo los Módulos del Núcleo

Los módulos del núcleo son piezas de código que pueden cargarse y descargarse en el núcleo según demanda. Extienden la funcionalidad del núcleo sin necesidad de reiniciar el sistema.

  • Conceptos Clave:
    • Carga Dinámica: Los módulos pueden cargarse y descargarse dinámicamente usando comandos como insmod y rmmod.
    • Tipos de Módulos: Controladores de dispositivos, controladores de sistemas de archivos, controladores de red, etc.
    • Dependencias de Módulos: Los módulos pueden depender de otros módulos, que deben cargarse primero.

Trabajando con Módulos del Núcleo

Cargando y Descargando Módulos

Los módulos pueden gestionarse usando varios comandos.

  • Cargando un Módulo:

    sudo insmod mymodule.ko

    Este comando inserta el módulo mymodule.ko en el núcleo.

  • Descargando un Módulo:

    sudo rmmod mymodule

    Este comando elimina el módulo mymodule del núcleo.

  • Listando Módulos Cargados:

    lsmod

    Este comando lista todos los módulos actualmente cargados.

Escribiendo un Módulo de Núcleo Simple

Vamos a escribir un módulo de núcleo simple "Hola Mundo".

  • hello.c:

    #include <linux/module.h>
#include <linux/kernel.h>

int init_module(void) {
    printk(KERN_INFO "Hola, mundo!\n");
    return 0;
}

void cleanup_module(void) {
    printk(KERN_INFO "Adiós, mundo!\n");
}

MODULE_LICENSE("GPL");
MODULE_DESCRIPTION("Un módulo simple de Hola Mundo");
MODULE_AUTHOR("Tu Nombre");

  • Compilando el Módulo: Crea un Makefile para compilar el módulo.

    obj-m += hello.o

all:
    make -C /lib/modules/$(shell uname -r)/build M=$(PWD) modules

clean:
    make -C /lib/modules/$(shell uname -r)/build M=$(PWD) clean

    Ejecuta make para compilar el módulo.

  • Cargando y Probando el Módulo:

    sudo insmod hello.ko
dmesg | tail
sudo rmmod hello
dmesg | tail

    El comando dmesg muestra mensajes del núcleo, donde deberías ver los mensajes "Hola, mundo!" y "Adiós, mundo!".

Temas Avanzados

Parámetros del Módulo del Núcleo

Los módulos pueden aceptar parámetros en el momento de la carga.

  • Ejemplo con Parámetros:

    #include <linux/module.h>
#include <linux/kernel.h>
#include <linux/init.h>

static int myint = 0;
module_param(myint, int, 0);
MODULE_PARM_DESC(myint, "Un entero");

int init_module(void) {
    printk(KERN_INFO "Hola, mundo! myint = %d\n", myint);
    return 0;
}

void cleanup_module(void) {
    printk(KERN_INFO "Adiós, mundo!\n");
}

MODULE_LICENSE("GPL");
MODULE_DESCRIPTION("Un módulo simple con parámetros");
MODULE_AUTHOR("Tu Nombre");

    Carga el módulo con un parámetro:

    sudo insmod hello.ko myint=42
dmesg | tail

Depuración de Módulos del Núcleo

Depurar módulos del núcleo puede ser un desafío debido a la falta de herramientas de depuración estándar.

  • Usando printk: El método principal para depurar es usando declaraciones printk.
  • Depurador del Núcleo (KGDB): Una herramienta más avanzada para depurar código del núcleo.

Conclusión

Entender el núcleo de Linux y sus módulos es crucial para la programación avanzada en Linux y la administración del sistema. Los módulos del núcleo proporcionan una manera flexible de extender la funcionalidad del núcleo sin reiniciar el sistema. Al aprender a escribir, cargar y depurar módulos del núcleo, obtienes una comprensión más profunda del funcionamiento interno del sistema operativo Linux.
