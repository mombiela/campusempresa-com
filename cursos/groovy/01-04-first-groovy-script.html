Primer Script en Groovy

Siguiente ►
►

En esta sección, aprenderás a escribir y ejecutar tu primer script en Groovy. Este es un paso fundamental para familiarizarte con la sintaxis y el entorno de ejecución de Groovy.

Objetivos

  • Escribir un script básico en Groovy.
  • Ejecutar el script desde la línea de comandos.
  • Comprender la salida del script.

Requisitos Previos

  • Haber completado la configuración del entorno de Groovy.
  • Conocimientos básicos de programación (variables, operadores, estructuras de control).

Escribiendo tu Primer Script

Paso 1: Crear un Archivo de Script

  1. Abre tu editor de texto favorito (puede ser un IDE como IntelliJ IDEA, Eclipse, o un editor de texto simple como VSCode o Sublime Text).
  2. Crea un nuevo archivo y nómbralo HelloWorld.groovy.

Paso 2: Escribir el Código

En el archivo HelloWorld.groovy, escribe el siguiente código:

// Este es un comentario en Groovy
println 'Hola, Mundo!'

Explicación del Código

  • // Este es un comentario en Groovy: Los comentarios en Groovy se inician con // y son ignorados por el intérprete. Son útiles para agregar notas o explicaciones en tu código.
  • println 'Hola, Mundo!': Esta línea imprime el texto 'Hola, Mundo!' en la consola. println es una función incorporada en Groovy que imprime una línea de texto seguida de un salto de línea.

Paso 3: Ejecutar el Script

Para ejecutar tu script, sigue estos pasos:

  1. Abre una terminal o línea de comandos.
  2. Navega hasta el directorio donde guardaste tu archivo HelloWorld.groovy.
  3. Ejecuta el siguiente comando:
groovy HelloWorld.groovy

Salida Esperada

Al ejecutar el comando anterior, deberías ver la siguiente salida en la consola:

Hola, Mundo!

Ejercicio Práctico

Ejercicio 1: Modificar el Script

Modifica el script HelloWorld.groovy para que imprima tu nombre en lugar de "Hola, Mundo!".

Solución

// Este es un comentario en Groovy
println 'Hola, [Tu Nombre]!'

Reemplaza [Tu Nombre] con tu nombre real. Guarda el archivo y ejecútalo nuevamente para ver la salida.

Ejercicio 2: Añadir Más Líneas

Añade más líneas al script para imprimir tres mensajes diferentes.

Solución

// Este es un comentario en Groovy
println 'Hola, Mundo!'
println 'Bienvenido al curso de Groovy.'
println '¡Vamos a aprender mucho!'

Guarda el archivo y ejecútalo para ver la salida.

Errores Comunes y Consejos

  • Error de Sintaxis: Asegúrate de que cada línea de código esté correctamente escrita. Groovy es sensible a la sintaxis.
  • Archivo No Encontrado: Verifica que estás en el directorio correcto y que el nombre del archivo es exacto.
  • Permisos de Ejecución: En algunos sistemas, puede que necesites permisos de ejecución para correr scripts. Usa chmod +x HelloWorld.groovy en sistemas Unix/Linux si es necesario.

Conclusión

En esta sección, has aprendido a escribir y ejecutar tu primer script en Groovy. Este es un paso crucial para comenzar a explorar las capacidades del lenguaje. A medida que avances en el curso, construirás sobre esta base para crear scripts y aplicaciones más complejas.

En la próxima sección, profundizaremos en las variables y tipos de datos en Groovy, lo que te permitirá manejar y manipular datos de manera efectiva. ¡Sigue adelante!
