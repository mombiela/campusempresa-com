Configuración de tu cuenta de GCP

Siguiente ►
►

En esta sección, aprenderás cómo configurar tu cuenta de Google Cloud Platform (GCP). Este es un paso fundamental para comenzar a utilizar los servicios de GCP y desarrollar tus proyectos en la nube.

  1. Crear una cuenta de Google

Si ya tienes una cuenta de Google, puedes saltar este paso. Si no, sigue estos pasos para crear una:

  1. Ve a accounts.google.com.
  2. Haz clic en "Crear cuenta".
  3. Completa el formulario con tu información personal.
  4. Sigue las instrucciones para verificar tu cuenta.

  1. Acceder a Google Cloud Platform

  1. Ve a cloud.google.com.
  2. Haz clic en "Ir a la consola" en la esquina superior derecha.
  3. Inicia sesión con tu cuenta de Google.

  1. Crear un proyecto en GCP

Los proyectos en GCP son contenedores que organizan todos tus recursos de Google Cloud. Cada proyecto tiene su propia configuración, permisos y facturación.

Pasos para crear un proyecto:

  1. En la consola de GCP, haz clic en el menú desplegable de selección de proyecto en la parte superior de la página.
  2. Haz clic en "Nuevo proyecto".
  3. Completa el formulario:
    • Nombre del proyecto: Elige un nombre descriptivo para tu proyecto.
    • ID del proyecto: Este es un identificador único que GCP generará automáticamente, pero puedes personalizarlo si lo deseas.
    • Ubicación: Selecciona la organización o carpeta en la que deseas crear el proyecto (si aplica).
  4. Haz clic en "Crear".

  1. Configurar la facturación

Para utilizar la mayoría de los servicios de GCP, necesitas configurar la facturación. GCP ofrece un crédito gratuito inicial para nuevos usuarios, lo que te permite explorar y probar los servicios sin costo.

Pasos para configurar la facturación:

  1. En la consola de GCP, ve al menú de navegación y selecciona "Facturación".
  2. Haz clic en "Vincular una cuenta de facturación".
  3. Si no tienes una cuenta de facturación, sigue las instrucciones para crear una:
    • Nombre de la cuenta de facturación: Elige un nombre para tu cuenta de facturación.
    • País: Selecciona tu país.
    • Dirección de facturación: Proporciona tu dirección de facturación.
  4. Ingresa la información de tu tarjeta de crédito o débito.
  5. Revisa y acepta los términos y condiciones.
  6. Haz clic en "Iniciar mi prueba gratuita" (si es aplicable) o "Guardar y continuar".

  1. Habilitar las API necesarias

Para utilizar ciertos servicios de GCP, es posible que necesites habilitar las API correspondientes.

Pasos para habilitar una API:

  1. En la consola de GCP, ve al menú de navegación y selecciona "API y servicios" > "Biblioteca".
  2. Busca la API que necesitas habilitar (por ejemplo, "Compute Engine API").
  3. Haz clic en la API y luego en "Habilitar".

  1. Configurar permisos y roles

GCP utiliza el sistema de Gestión de Identidad y Acceso (IAM) para controlar quién puede hacer qué en tu proyecto.

Pasos para asignar roles:

  1. En la consola de GCP, ve al menú de navegación y selecciona "IAM y administración" > "IAM".
  2. Haz clic en "Agregar" en la parte superior de la página.
  3. Ingresa la dirección de correo electrónico del usuario al que deseas asignar un rol.
  4. Selecciona el rol adecuado para el usuario (por ejemplo, "Editor" o "Visualizador").
  5. Haz clic en "Guardar".

Ejercicio Práctico

Ejercicio 1: Crear un proyecto y habilitar la facturación

  1. Crea un nuevo proyecto en GCP llamado "Proyecto de Prueba".
  2. Configura la facturación para tu proyecto utilizando la prueba gratuita de GCP.
  3. Habilita la API de Compute Engine para tu proyecto.

Solución:

  1. Sigue los pasos descritos en la sección "Crear un proyecto en GCP" para crear un proyecto llamado "Proyecto de Prueba".
  2. Configura la facturación siguiendo los pasos en la sección "Configurar la facturación".
  3. Habilita la API de Compute Engine siguiendo los pasos en la sección "Habilitar las API necesarias".

Conclusión

En esta sección, has aprendido cómo configurar tu cuenta de GCP, crear un proyecto, configurar la facturación, habilitar las API necesarias y asignar roles y permisos. Estos son los primeros pasos esenciales para comenzar a utilizar Google Cloud Platform y aprovechar sus servicios. En el próximo módulo, exploraremos la consola de GCP y sus características principales.
