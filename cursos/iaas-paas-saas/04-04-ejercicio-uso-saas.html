Ejercicio Práctico: Uso de una Aplicación SaaS

Siguiente ►
►

En este ejercicio práctico, aprenderás a utilizar una aplicación SaaS (Software como Servicio). Para este propósito, utilizaremos Google Docs, una de las aplicaciones SaaS más populares y ampliamente utilizadas. Google Docs permite crear, editar y compartir documentos en línea sin necesidad de instalar software adicional en tu computadora.

Objetivos del Ejercicio

  1. Familiarizarse con la interfaz de Google Docs.
  2. Crear y editar un documento en Google Docs.
  3. Compartir el documento con otros usuarios.
  4. Colaborar en tiempo real con otros usuarios.

Pasos a Seguir

Paso 1: Acceder a Google Docs

  1. Abre tu navegador web y ve a Google Docs.
  2. Si no tienes una cuenta de Google, regístrate para crear una.
  3. Inicia sesión con tu cuenta de Google.

Paso 2: Crear un Nuevo Documento

  1. Una vez que hayas iniciado sesión, haz clic en el botón "Blank" (en blanco) para crear un nuevo documento.
  2. Se abrirá una nueva pestaña con un documento en blanco.

Paso 3: Familiarizarse con la Interfaz

  1. Barra de Menú: Contiene opciones como Archivo, Editar, Ver, Insertar, Formato, Herramientas, etc.
  2. Barra de Herramientas: Incluye herramientas de formato como negrita, cursiva, subrayado, alineación de texto, etc.
  3. Área de Trabajo: Es el espacio donde escribirás y editarás tu documento.
  4. Nombre del Documento: Haz clic en "Untitled document" (Documento sin título) en la esquina superior izquierda para renombrar tu documento.

Paso 4: Editar el Documento

  1. Escribe un párrafo de texto en el área de trabajo.
  2. Utiliza las herramientas de formato para cambiar el estilo del texto (negrita, cursiva, color de texto, etc.).
  3. Inserta una imagen en el documento:
    • Haz clic en Insertar > Imagen > Subir desde la computadora.
    • Selecciona una imagen desde tu computadora y haz clic en Abrir.

Paso 5: Compartir el Documento

  1. Haz clic en el botón "Share" (Compartir) en la esquina superior derecha.
  2. En la ventana emergente, introduce la dirección de correo electrónico de la persona con la que deseas compartir el documento.
  3. Selecciona el nivel de acceso (Editor, Comentarista, Lector).
  4. Haz clic en "Send" (Enviar).

Paso 6: Colaborar en Tiempo Real

  1. Pide a la persona con la que compartiste el documento que lo abra.
  2. Observa cómo pueden editar el documento en tiempo real.
  3. Utiliza el chat integrado para comunicarte con ellos mientras colaboran.

Ejercicio Adicional: Uso de Funciones Avanzadas

  1. Comentarios: Añade un comentario en alguna parte del texto:
    • Selecciona el texto.
    • Haz clic en el icono de comentario en la barra de herramientas.
    • Escribe tu comentario y haz clic en "Comment" (Comentar).
  2. Historial de Versiones: Revisa el historial de versiones del documento:
    • Haz clic en Archivo > Historial de versiones > Ver historial de versiones.
    • Observa las diferentes versiones y quién hizo cada cambio.

Solución y Retroalimentación

Solución

  1. Acceso a Google Docs: Debes haber podido acceder a Google Docs y crear un nuevo documento.
  2. Edición del Documento: Debes haber escrito un párrafo, formateado el texto e insertado una imagen.
  3. Compartir el Documento: Debes haber compartido el documento con otro usuario y seleccionado el nivel de acceso adecuado.
  4. Colaboración en Tiempo Real: Debes haber observado la edición en tiempo real y utilizado el chat integrado.

Retroalimentación sobre Errores Comunes

  1. Problemas de Acceso: Asegúrate de estar utilizando una cuenta de Google válida y de haber iniciado sesión correctamente.
  2. Formato de Texto: Si el formato no se aplica como esperas, revisa que hayas seleccionado el texto correcto antes de aplicar el formato.
  3. Compartir Documento: Verifica que la dirección de correo electrónico sea correcta y que hayas seleccionado el nivel de acceso adecuado.
  4. Colaboración: Si no ves los cambios en tiempo real, asegúrate de que ambos usuarios estén conectados a Internet y tengan el documento abierto.

Conclusión

En este ejercicio, has aprendido a utilizar Google Docs, una aplicación SaaS, para crear, editar y compartir documentos. También has experimentado la colaboración en tiempo real, una de las características más poderosas de las aplicaciones SaaS. Este conocimiento te será útil para trabajar de manera más eficiente y colaborativa en entornos profesionales y personales.
