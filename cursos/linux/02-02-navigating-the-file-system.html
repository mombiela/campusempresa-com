Navegando el Sistema de Archivos

Siguiente ►
►

En este tema, aprenderás a moverte por el sistema de archivos de Linux utilizando la línea de comandos. Entender cómo navegar por el sistema de archivos es fundamental para cualquier usuario de Linux, ya que te permitirá acceder y gestionar archivos y directorios de manera eficiente.

Conceptos Clave

  1. Directorio Raíz (/): El punto de partida de todo el sistema de archivos en Linux.
  2. Directorios y Subdirectorios: Estructura jerárquica que organiza los archivos.
  3. Rutas Absolutas y Relativas: Formas de especificar la ubicación de archivos y directorios.
  4. Comandos de Navegación: Comandos básicos para moverse por el sistema de archivos.

Comandos Básicos de Navegación

pwd (Print Working Directory)

Muestra la ruta completa del directorio actual.

$ pwd
/home/usuario

ls (List)

Lista los archivos y directorios en el directorio actual.

$ ls
documentos  descargas  imágenes  música  vídeos

Opciones Comunes de ls

  • -l: Muestra detalles en formato largo.
  • -a: Incluye archivos ocultos.
  • -h: Formato legible para humanos (tamaños de archivo).
$ ls -lah
total 28K
drwxr-xr-x  6 usuario usuario 4.0K Oct  1 12:34 .
drwxr-xr-x  3 root    root    4.0K Sep 30 10:00 ..
-rw-r--r--  1 usuario usuario  220 Oct  1 12:34 .bash_logout
-rw-r--r--  1 usuario usuario 3.7K Oct  1 12:34 .bashrc
drwxr-xr-x  2 usuario usuario 4.0K Oct  1 12:34 documentos

cd (Change Directory)

Cambia el directorio actual.

$ cd documentos
$ pwd
/home/usuario/documentos

Rutas Absolutas y Relativas

  • Ruta Absoluta: Especifica la ubicación completa desde el directorio raíz. 
    $ cd /home/usuario/documentos
  • Ruta Relativa: Especifica la ubicación en relación al directorio actual. 
    $ cd documentos

.. y .

  • ..: Representa el directorio padre. 
    $ cd ..
  • .: Representa el directorio actual. 
    $ cd .

~ (Tilde)

Representa el directorio home del usuario actual.

$ cd ~
$ pwd
/home/usuario

Ejemplos Prácticos

Ejemplo 1: Navegar a un Directorio Específico

  1. Verifica tu directorio actual.

    $ pwd
/home/usuario

  2. Lista los contenidos del directorio actual.

    $ ls
documentos  descargas  imágenes  música  vídeos

  3. Cambia al directorio documentos.

    $ cd documentos

  4. Verifica que has cambiado de directorio.

    $ pwd
/home/usuario/documentos

Ejemplo 2: Usar Rutas Relativas y Absolutas

  1. Cambia al directorio raíz.

    $ cd /

  2. Cambia al directorio home usando una ruta relativa.

    $ cd home

  3. Cambia al directorio usuario usando una ruta absoluta.

    $ cd /home/usuario

  4. Verifica tu ubicación actual.

    $ pwd
/home/usuario

Ejercicios Prácticos

Ejercicio 1: Navegación Básica

  1. Cambia al directorio raíz.
  2. Lista los contenidos del directorio raíz.
  3. Cambia al directorio home usando una ruta relativa.
  4. Cambia al directorio usuario usando una ruta absoluta.
  5. Verifica tu ubicación actual.

Ejercicio 2: Uso de .. y .

  1. Cambia al directorio documentos.
  2. Cambia al directorio padre.
  3. Verifica tu ubicación actual.
  4. Cambia al directorio documentos nuevamente.
  5. Usa . para verificar tu ubicación actual.

Soluciones

Solución al Ejercicio 1

  1. Cambia al directorio raíz.

    $ cd /

  2. Lista los contenidos del directorio raíz.

    $ ls
bin  boot  dev  etc  home  lib  lib64  media  mnt  opt  proc  root  run  sbin  srv  sys  tmp  usr  var

  3. Cambia al directorio home usando una ruta relativa.

    $ cd home

  4. Cambia al directorio usuario usando una ruta absoluta.

    $ cd /home/usuario

  5. Verifica tu ubicación actual.

    $ pwd
/home/usuario

Solución al Ejercicio 2

  1. Cambia al directorio documentos.

    $ cd documentos

  2. Cambia al directorio padre.

    $ cd ..

  3. Verifica tu ubicación actual.

    $ pwd
/home/usuario

  4. Cambia al directorio documentos nuevamente.

    $ cd documentos

  5. Usa . para verificar tu ubicación actual.

    $ cd .
$ pwd
/home/usuario/documentos

Conclusión

En esta sección, has aprendido los comandos básicos para navegar por el sistema de archivos de Linux. Estos comandos son fundamentales para cualquier tarea que realices en la línea de comandos. Asegúrate de practicar estos comandos hasta que te sientas cómodo moviéndote por el sistema de archivos, ya que esta habilidad será esencial para los temas más avanzados que abordaremos en los próximos módulos.
