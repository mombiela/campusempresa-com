Introducción

Las redes son el pilar fundamental de cualquier infraestructura TI, permitiendo la comunicación y transferencia de datos entre dispositivos. En esta sección, exploraremos los conceptos básicos de redes, los tipos de redes y los componentes esenciales que las conforman.

Conceptos Básicos de Redes

  1. ¿Qué es una Red?

Una red es un conjunto de dispositivos interconectados que pueden comunicarse entre sí para compartir recursos e información. Los dispositivos en una red pueden incluir computadoras, servidores, impresoras, y otros dispositivos de red.

  1. Tipos de Redes

Las redes se pueden clasificar según su tamaño, alcance y propósito. Los tipos más comunes son:

  • LAN (Local Area Network): Red de área local que conecta dispositivos en un área geográfica limitada, como una oficina o un edificio.
  • WAN (Wide Area Network): Red de área amplia que conecta dispositivos en áreas geográficas extensas, como ciudades o países.
  • MAN (Metropolitan Area Network): Red de área metropolitana que conecta dispositivos en una ciudad o área metropolitana.
  • PAN (Personal Area Network): Red de área personal que conecta dispositivos en un rango cercano, como dispositivos Bluetooth.

  1. Topologías de Red

La topología de red se refiere a la disposición física o lógica de los dispositivos en una red. Las topologías comunes incluyen:

  • Topología en Estrella: Todos los dispositivos están conectados a un nodo central (hub o switch).
  • Topología en Bus: Todos los dispositivos están conectados a un único cable central.
  • Topología en Anillo: Los dispositivos están conectados en un anillo cerrado.
  • Topología en Malla: Cada dispositivo está conectado a varios otros dispositivos, proporcionando múltiples rutas para los datos.

  1. Protocolos de Red

Los protocolos de red son reglas y estándares que permiten la comunicación entre dispositivos en una red. Algunos de los protocolos más comunes son:

  • TCP/IP (Transmission Control Protocol/Internet Protocol): Protocolo fundamental para la comunicación en Internet.
  • HTTP/HTTPS (HyperText Transfer Protocol/Secure): Protocolo para la transferencia de páginas web.
  • FTP (File Transfer Protocol): Protocolo para la transferencia de archivos.
  • SMTP (Simple Mail Transfer Protocol): Protocolo para el envío de correos electrónicos.

Componentes de una Red

  1. Dispositivos de Red

  • Router: Dispositivo que dirige el tráfico de datos entre diferentes redes.
  • Switch: Dispositivo que conecta múltiples dispositivos en una red y utiliza direcciones MAC para enviar datos al destino correcto.
  • Hub: Dispositivo que conecta múltiples dispositivos en una red, pero envía datos a todos los dispositivos conectados.
  • Access Point: Dispositivo que permite la conexión inalámbrica de dispositivos a una red cableada.

  1. Medios de Transmisión

  • Cables de Par Trenzado (UTP/STP): Cables comunes para redes LAN.
  • Fibra Óptica: Cables que utilizan luz para transmitir datos a altas velocidades y largas distancias.
  • Medios Inalámbricos: Tecnologías como Wi-Fi y Bluetooth que permiten la transmisión de datos sin cables.

  1. Direccionamiento IP

Cada dispositivo en una red necesita una dirección IP única para comunicarse. Las direcciones IP pueden ser:

  • IPv4: Formato de 32 bits, escrito como cuatro números decimales separados por puntos (e.g., 192.168.1.1).
  • IPv6: Formato de 128 bits, escrito como ocho grupos de cuatro caracteres hexadecimales separados por dos puntos (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

Ejemplo Práctico: Configuración Básica de una Red LAN

Paso 1: Conectar los Dispositivos

  1. Conecta el router a la fuente de alimentación y a la línea de Internet.
  2. Conecta el switch al router utilizando un cable Ethernet.
  3. Conecta las computadoras y otros dispositivos al switch utilizando cables Ethernet.

Paso 2: Configurar Direcciones IP

  1. Accede a la configuración del router a través de un navegador web (generalmente ingresando la dirección IP del router, como 192.168.1.1).
  2. Configura el rango de direcciones IP que el router asignará a los dispositivos (DHCP).
  3. En cada computadora, asegúrate de que la configuración de red esté configurada para obtener una dirección IP automáticamente (DHCP).

Paso 3: Verificar la Conectividad

  1. En cada computadora, abre una terminal o símbolo del sistema.
  2. Usa el comando ping para verificar la conectividad con otros dispositivos en la red. Por ejemplo: 
    ping 192.168.1.2
  3. Si recibes respuestas, la red está configurada correctamente.

Ejercicio Práctico

Ejercicio 1: Identificación de Componentes de Red

Instrucciones:

  1. Enumera los dispositivos de red que encuentras en tu entorno (hogar, oficina, etc.).
  2. Clasifica cada dispositivo según su función (router, switch, access point, etc.).

Solución:

  1. Router: Dispositivo que conecta la red local a Internet.
  2. Switch: Dispositivo que conecta múltiples computadoras en una red.
  3. Access Point: Dispositivo que permite la conexión inalámbrica de dispositivos.

Ejercicio 2: Configuración de una Red Simple

Instrucciones:

  1. Dibuja un diagrama de una red LAN simple que incluya al menos un router, un switch y tres computadoras.
  2. Asigna direcciones IP a cada dispositivo en el rango 192.168.1.x.

Solución:

  1. Diagrama de red: 
    [Internet] -- [Router] -- [Switch] -- [PC1: 192.168.1.2]
                                   -- [PC2: 192.168.1.3]
                                   -- [PC3: 192.168.1.4]
  2. Asignación de direcciones IP:
    • Router: 192.168.1.1
    • PC1: 192.168.1.2
    • PC2: 192.168.1.3
    • PC3: 192.168.1.4

Conclusión

En esta sección, hemos cubierto los fundamentos de las redes, incluyendo los conceptos básicos, tipos de redes, topologías, protocolos y componentes esenciales. Estos conocimientos son cruciales para gestionar y mantener una infraestructura TI eficiente y segura. En el próximo módulo, profundizaremos en el diseño y configuración de redes, proporcionando una base sólida para construir y optimizar redes en entornos empresariales.
