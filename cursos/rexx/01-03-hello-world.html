Hola Mundo en REXX

Siguiente ►
►

En esta lección, aprenderemos a escribir nuestro primer programa en REXX: el clásico "Hola Mundo". Este ejercicio nos permitirá familiarizarnos con la sintaxis básica y el flujo de trabajo de un programa REXX.

Objetivos

  • Escribir y ejecutar un programa simple en REXX.
  • Comprender la estructura básica de un script REXX.
  • Familiarizarse con la salida de texto en REXX.

Estructura Básica de un Programa REXX

Un programa REXX es un archivo de texto que contiene una serie de instrucciones que el intérprete de REXX ejecuta secuencialmente. La extensión común para los archivos REXX es .rexx o .rex.

Comentarios

Los comentarios en REXX se inician con /* y terminan con */. Todo lo que esté entre estos delimitadores será ignorado por el intérprete.

/* Este es un comentario en REXX */

Instrucción SAY

La instrucción SAY se utiliza para imprimir texto en la salida estándar (generalmente la consola).

SAY 'Hola Mundo'

Escribiendo el Programa "Hola Mundo"

Vamos a escribir un programa simple que imprima "Hola Mundo" en la consola.

Paso 1: Crear el Archivo del Programa

  1. Abre tu editor de texto favorito.
  2. Crea un nuevo archivo y guárdalo con el nombre hola_mundo.rexx.

Paso 2: Escribir el Código

Escribe el siguiente código en tu archivo hola_mundo.rexx:

/* Programa Hola Mundo en REXX */
SAY 'Hola Mundo'

Paso 3: Ejecutar el Programa

Para ejecutar el programa, abre una terminal o línea de comandos y navega hasta el directorio donde guardaste hola_mundo.rexx. Luego, ejecuta el siguiente comando:

rexx hola_mundo.rexx

Deberías ver la salida:

Hola Mundo

Explicación del Código

  • /* Programa Hola Mundo en REXX */: Este es un comentario que describe el propósito del programa.
  • SAY 'Hola Mundo': Esta instrucción imprime el texto 'Hola Mundo' en la consola.

Ejercicio Práctico

Ejercicio 1: Modificar el Mensaje

Modifica el programa para que imprima "¡Bienvenidos al curso de REXX!".

Solución

/* Programa de Bienvenida en REXX */
SAY '¡Bienvenidos al curso de REXX!'

Ejercicio 2: Añadir Más Mensajes

Añade dos líneas más de texto que se impriman después de "Hola Mundo". Por ejemplo, "Este es un curso de programación" y "¡Vamos a aprender REXX!".

Solución

/* Programa Hola Mundo en REXX con mensajes adicionales */
SAY 'Hola Mundo'
SAY 'Este es un curso de programación'
SAY '¡Vamos a aprender REXX!'

Resumen

En esta lección, hemos aprendido a:

  • Escribir un programa básico en REXX.
  • Utilizar la instrucción SAY para imprimir texto en la consola.
  • Ejecutar un programa REXX desde la línea de comandos.

Con estos conocimientos básicos, estamos listos para explorar más sobre la sintaxis y las estructuras de control en REXX en las próximas lecciones. ¡Sigue practicando y nos vemos en la siguiente lección!
