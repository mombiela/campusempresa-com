Instalando Redis

Siguiente ►
►

En esta sección, aprenderás cómo instalar Redis en diferentes sistemas operativos. Redis es compatible con la mayoría de los sistemas operativos, incluidos Linux, macOS y Windows. A continuación, se detallan los pasos para instalar Redis en cada uno de estos sistemas.

Instalación en Linux

Usando el Administrador de Paquetes (Debian/Ubuntu)

  1. Actualizar el índice de paquetes:

    sudo apt update

  2. Instalar Redis:

    sudo apt install redis-server

  3. Verificar la instalación:

    redis-server --version

Usando el Administrador de Paquetes (CentOS/RHEL)

  1. Agregar el repositorio EPEL:

    sudo yum install epel-release

  2. Instalar Redis:

    sudo yum install redis

  3. Iniciar el servicio Redis:

    sudo systemctl start redis

  4. Habilitar Redis para que se inicie al arrancar:

    sudo systemctl enable redis

  5. Verificar la instalación:

    redis-server --version

Instalación en macOS

Usando Homebrew

  1. Instalar Homebrew (si no está instalado):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Actualizar Homebrew:

    brew update

  3. Instalar Redis:

    brew install redis

  4. Iniciar Redis:

    brew services start redis

  5. Verificar la instalación:

    redis-server --version

Instalación en Windows

Usando el Instalador de Redis

  1. Descargar el instalador de Redis para Windows desde el repositorio de Microsoft: Redis para Windows

  2. Ejecutar el instalador y seguir las instrucciones en pantalla.

  3. Verificar la instalación: Abre una terminal de comandos y ejecuta:

    redis-server --version

Verificación de la Instalación

Independientemente del sistema operativo, puedes verificar que Redis se ha instalado correctamente ejecutando el siguiente comando en tu terminal:

redis-server --version

Deberías ver una salida similar a:

Redis server v=6.2.5 sha=00000000:0 malloc=libc bits=64 build=00000000

Ejecución de Redis

Para iniciar el servidor Redis, simplemente ejecuta:

redis-server

Esto iniciará el servidor Redis en el puerto 6379 por defecto.

Para conectarte al servidor Redis usando la CLI de Redis, abre otra terminal y ejecuta:

redis-cli

Deberías ver un prompt de Redis:

127.0.0.1:6379>

Conclusión

En esta sección, has aprendido cómo instalar Redis en diferentes sistemas operativos. Ahora que tienes Redis instalado, estás listo para explorar sus características y comandos básicos en la siguiente sección. Asegúrate de que Redis esté funcionando correctamente antes de continuar.
