Pivotando datos

Siguiente ►
►

En este tema, aprenderás cómo pivotar datos en Tableau. Pivotar datos es una técnica esencial para transformar tus datos de un formato amplio a un formato largo, lo que puede facilitar el análisis y la visualización. A continuación, se desglosarán los conceptos clave, se proporcionarán ejemplos prácticos y se incluirán ejercicios para reforzar el aprendizaje.

Conceptos clave

  1. Formato amplio vs. Formato largo:

    • Formato amplio: Los datos están organizados en columnas, donde cada columna representa una variable diferente.
    • Formato largo: Los datos están organizados en filas, donde cada fila representa una observación única y las variables se almacenan en una columna separada.

  2. Pivotar datos:

    • Pivotar a largo: Convertir datos de formato amplio a formato largo.
    • Pivotar a ancho: Convertir datos de formato largo a formato amplio (menos común en Tableau).

Ejemplo práctico

Paso 1: Conectar a la fuente de datos

  1. Abre Tableau y conéctate a tu fuente de datos. Para este ejemplo, usaremos un archivo de Excel llamado ventas.xlsx.

Paso 2: Cargar los datos

  1. En la ventana de conexión de datos, selecciona la hoja de cálculo que contiene los datos que deseas pivotar. Supongamos que la hoja se llama Ventas.

Paso 3: Identificar los datos a pivotar

  1. Observa la estructura de tus datos. Supongamos que tienes las siguientes columnas: Producto, Enero, Febrero, Marzo.
Producto Enero Febrero Marzo
A 100 150 200
B 200 250 300
C 300 350 400

Paso 4: Pivotar los datos

  1. Selecciona las columnas que deseas pivotar (Enero, Febrero, Marzo).
  2. Haz clic derecho en las columnas seleccionadas y elige la opción Pivotar.

Paso 5: Renombrar las columnas pivotadas

  1. Después de pivotar, verás dos nuevas columnas: Pivote Nombre y Pivote Valor.
  2. Renombra Pivote Nombre a Mes y Pivote Valor a Ventas.

El resultado debería ser algo así:

Producto Mes Ventas
A Enero 100
A Febrero 150
A Marzo 200
B Enero 200
B Febrero 250
B Marzo 300
C Enero 300
C Febrero 350
C Marzo 400

Paso 6: Guardar y usar los datos pivotados

  1. Ahora puedes usar estos datos pivotados para crear visualizaciones más efectivas en Tableau.

Ejercicio práctico

Ejercicio 1: Pivotar datos de ventas trimestrales

  1. Conéctate a un archivo de Excel llamado ventas_trimestrales.xlsx.
  2. La hoja de cálculo contiene las siguientes columnas: Producto, Q1, Q2, Q3, Q4.
  3. Pivotar las columnas Q1, Q2, Q3, Q4 para convertir los datos a formato largo.
  4. Renombra las columnas resultantes a Trimestre y Ventas.
  5. Crea una visualización de barras que muestre las ventas por trimestre para cada producto.

Solución del ejercicio 1

  1. Conéctate a ventas_trimestrales.xlsx y selecciona la hoja de cálculo.
  2. Selecciona las columnas Q1, Q2, Q3, Q4.
  3. Haz clic derecho y selecciona Pivotar.
  4. Renombra Pivote Nombre a Trimestre y Pivote Valor a Ventas.
  5. Crea una visualización de barras:
    • Arrastra Producto a Filas.
    • Arrastra Trimestre a Columnas.
    • Arrastra Ventas a Texto en la tarjeta de marcas.

Resumen

En esta sección, aprendiste cómo pivotar datos en Tableau para transformar tus datos de un formato amplio a un formato largo. Esta técnica es crucial para preparar tus datos para análisis y visualización. Practica pivotando diferentes conjuntos de datos para familiarizarte con el proceso y mejorar tus habilidades en Tableau.

En el siguiente tema, exploraremos técnicas de limpieza de datos para asegurarnos de que nuestros datos estén listos para el análisis.
