Tipos de gráficos avanzados

Siguiente ►
►

En este módulo, exploraremos varios tipos de gráficos avanzados que puedes crear en Tableau. Estos gráficos te permitirán visualizar datos de maneras más complejas y detalladas, proporcionando una comprensión más profunda de tus datos.

Gráficos de Gantt

¿Qué es un gráfico de Gantt?

Un gráfico de Gantt es una herramienta de gestión de proyectos que muestra las tareas a lo largo del tiempo. Es útil para visualizar la duración de las tareas y su secuencia.

Cómo crear un gráfico de Gantt en Tableau

  1. Conectar a la fuente de datos: Asegúrate de tener una fuente de datos que incluya las tareas, las fechas de inicio y fin.
  2. Arrastrar los campos:
    • Arrastra el campo de tareas a la columna.
    • Arrastra el campo de fechas de inicio a la fila.
  3. Cambiar el tipo de marca:
    • En la tarjeta de marcas, selecciona "Gantt Bar".
  4. Agregar duración:
    • Arrastra el campo de duración a la tarjeta de tamaño.

Ejemplo de código

1. Conectar a la fuente de datos.
2. Arrastrar [Tarea] a la columna.
3. Arrastrar [Fecha de Inicio] a la fila.
4. Cambiar el tipo de marca a "Gantt Bar".
5. Arrastrar [Duración] a la tarjeta de tamaño.

Ejercicio práctico

Ejercicio: Crea un gráfico de Gantt para visualizar el cronograma de un proyecto con las siguientes tareas y fechas:

Tarea Fecha de Inicio Fecha de Fin
Investigación 01/01/2023 15/01/2023
Desarrollo 16/01/2023 28/02/2023
Pruebas 01/03/2023 15/03/2023
Implementación 16/03/2023 31/03/2023

Solución:

  1. Conectar a la fuente de datos.
  2. Arrastrar [Tarea] a la columna.
  3. Arrastrar [Fecha de Inicio] a la fila.
  4. Cambiar el tipo de marca a "Gantt Bar".
  5. Crear un campo calculado para la duración: DATEDIFF('day', [Fecha de Inicio], [Fecha de Fin]).
  6. Arrastrar el campo calculado de duración a la tarjeta de tamaño.

Gráficos de Violin

¿Qué es un gráfico de Violin?

Un gráfico de Violin combina aspectos de un gráfico de caja y un gráfico de densidad de kernel. Es útil para visualizar la distribución de datos y su densidad.

Cómo crear un gráfico de Violin en Tableau

  1. Conectar a la fuente de datos: Asegúrate de tener una fuente de datos con valores numéricos.
  2. Crear un gráfico de caja:
    • Arrastra el campo numérico a la fila.
    • Arrastra la dimensión a la columna.
    • Cambia el tipo de marca a "Box Plot".
  3. Agregar densidad de kernel:
    • Duplica el eje de la fila.
    • Cambia el tipo de marca del eje duplicado a "Density".

Ejemplo de código

1. Conectar a la fuente de datos.
2. Arrastrar [Valor] a la fila.
3. Arrastrar [Categoría] a la columna.
4. Cambiar el tipo de marca a "Box Plot".
5. Duplicar el eje de la fila.
6. Cambiar el tipo de marca del eje duplicado a "Density".

Ejercicio práctico

Ejercicio: Crea un gráfico de Violin para visualizar la distribución de las puntuaciones de los estudiantes en diferentes materias.

Estudiante Materia Puntuación
Juan Matemáticas 85
Ana Matemáticas 90
Luis Ciencias 78
María Ciencias 88
Pedro Historia 92
Laura Historia 85

Solución:

  1. Conectar a la fuente de datos.
  2. Arrastrar [Puntuación] a la fila.
  3. Arrastrar [Materia] a la columna.
  4. Cambiar el tipo de marca a "Box Plot".
  5. Duplicar el eje de la fila.
  6. Cambiar el tipo de marca del eje duplicado a "Density".

Gráficos de Control

¿Qué es un gráfico de Control?

Un gráfico de control es una herramienta de control de calidad que muestra cómo un proceso cambia con el tiempo. Es útil para identificar variaciones y tendencias.

Cómo crear un gráfico de Control en Tableau

  1. Conectar a la fuente de datos: Asegúrate de tener una fuente de datos con valores numéricos y fechas.
  2. Crear un gráfico de líneas:
    • Arrastra el campo de fecha a la columna.
    • Arrastra el campo numérico a la fila.
    • Cambia el tipo de marca a "Line".
  3. Agregar líneas de referencia:
    • Haz clic derecho en el eje de la fila y selecciona "Add Reference Line".
    • Agrega líneas de referencia para el promedio y los límites de control superior e inferior.

Ejemplo de código

1. Conectar a la fuente de datos.
2. Arrastrar [Fecha] a la columna.
3. Arrastrar [Valor] a la fila.
4. Cambiar el tipo de marca a "Line".
5. Agregar líneas de referencia para el promedio y los límites de control.

Ejercicio práctico

Ejercicio: Crea un gráfico de control para visualizar la variación de la producción diaria en una fábrica durante un mes.

Fecha Producción
01/01/2023 100
02/01/2023 105
03/01/2023 98
... ...
31/01/2023 110

Solución:

  1. Conectar a la fuente de datos.
  2. Arrastrar [Fecha] a la columna.
  3. Arrastrar [Producción] a la fila.
  4. Cambiar el tipo de marca a "Line".
  5. Agregar líneas de referencia para el promedio y los límites de control.

Gráficos de Marimekko

¿Qué es un gráfico de Marimekko?

Un gráfico de Marimekko es una herramienta de visualización que muestra la distribución de datos categóricos en dos dimensiones. Es útil para comparar proporciones.

Cómo crear un gráfico de Marimekko en Tableau

  1. Conectar a la fuente de datos: Asegúrate de tener una fuente de datos con dos dimensiones categóricas y una medida.
  2. Crear un gráfico de barras apiladas:
    • Arrastra una dimensión a la columna.
    • Arrastra la otra dimensión a la fila.
    • Arrastra la medida a la tarjeta de tamaño.
    • Cambia el tipo de marca a "Bar".
  3. Normalizar las barras:
    • Crea un campo calculado para la proporción de cada categoría.
    • Usa el campo calculado para el tamaño de las barras.

Ejemplo de código

1. Conectar a la fuente de datos.
2. Arrastrar [Categoría 1] a la columna.
3. Arrastrar [Categoría 2] a la fila.
4. Arrastrar [Medida] a la tarjeta de tamaño.
5. Cambiar el tipo de marca a "Bar".
6. Crear un campo calculado para la proporción.
7. Usar el campo calculado para el tamaño de las barras.

Ejercicio práctico

Ejercicio: Crea un gráfico de Marimekko para visualizar la distribución de ventas por región y producto.

Región Producto Ventas
Norte A 100
Norte B 150
Sur A 200
Sur B 250

Solución:

  1. Conectar a la fuente de datos.
  2. Arrastrar [Región] a la columna.
  3. Arrastrar [Producto] a la fila.
  4. Arrastrar [Ventas] a la tarjeta de tamaño.
  5. Cambiar el tipo de marca a "Bar".
  6. Crear un campo calculado para la proporción: [Ventas] / TOTAL([Ventas]).
  7. Usar el campo calculado para el tamaño de las barras.

Gráficos de Sankey

¿Qué es un gráfico de Sankey?

Un gráfico de Sankey es una herramienta de visualización que muestra el flujo de datos entre diferentes estados o categorías. Es útil para visualizar procesos y flujos.

Cómo crear un gráfico de Sankey en Tableau

  1. Conectar a la fuente de datos: Asegúrate de tener una fuente de datos con categorías y medidas.
  2. Crear un gráfico de líneas:
    • Arrastra las categorías a la columna y la fila.
    • Arrastra la medida a la tarjeta de tamaño.
    • Cambia el tipo de marca a "Line".
  3. Agregar curvas:
    • Usa un campo calculado para crear las curvas de flujo.

Ejemplo de código

1. Conectar a la fuente de datos.
2. Arrastrar [Categoría de Origen] a la columna.
3. Arrastrar [Categoría de Destino] a la fila.
4. Arrastrar [Medida] a la tarjeta de tamaño.
5. Cambiar el tipo de marca a "Line".
6. Crear un campo calculado para las curvas de flujo.

Ejercicio práctico

Ejercicio: Crea un gráfico de Sankey para visualizar el flujo de clientes entre diferentes etapas de un embudo de ventas.

Etapa de Origen Etapa de Destino Clientes
Prospecto Contactado 100
Contactado Calificado 80
Calificado Oportunidad 60
Oportunidad Cliente 40

Solución:

  1. Conectar a la fuente de datos.
  2. Arrastrar [Etapa de Origen] a la columna.
  3. Arrastrar [Etapa de Destino] a la fila.
  4. Arrastrar [Clientes] a la tarjeta de tamaño.
  5. Cambiar el tipo de marca a "Line".
  6. Crear un campo calculado para las curvas de flujo.

Conclusión

En esta sección, hemos explorado varios tipos de gráficos avanzados en Tableau, incluyendo gráficos de Gantt, gráficos de Violin, gráficos de Control, gráficos de Marimekko y gráficos de Sankey. Cada uno de estos gráficos ofrece una manera única de visualizar datos complejos y puede ser utilizado para obtener insights más profundos. Asegúrate de practicar estos gráficos con tus propios datos para dominar su uso y aplicación.
