Publicando libros de trabajo

Siguiente ►
►

En este tema, aprenderás cómo publicar tus libros de trabajo de Tableau en Tableau Server o Tableau Online. Publicar un libro de trabajo permite compartir tus visualizaciones y análisis con otros usuarios de tu organización o con el público en general. A continuación, se detallan los pasos y consideraciones para realizar esta tarea.

Objetivos de aprendizaje

  • Entender el proceso de publicación de libros de trabajo en Tableau Server y Tableau Online.
  • Configurar permisos y opciones de publicación.
  • Conocer las mejores prácticas para la publicación de libros de trabajo.

Pasos para publicar un libro de trabajo

  1. Preparar el libro de trabajo

Antes de publicar, asegúrate de que tu libro de trabajo esté completo y listo para ser compartido. Revisa los siguientes puntos:

  • Verifica los datos: Asegúrate de que los datos sean precisos y estén actualizados.
  • Revisa las visualizaciones: Comprueba que todas las visualizaciones sean claras y efectivas.
  • Optimiza el rendimiento: Asegúrate de que el libro de trabajo esté optimizado para un rendimiento rápido.

  1. Conectar a Tableau Server o Tableau Online

Para publicar un libro de trabajo, primero debes conectarte a Tableau Server o Tableau Online.

1. Abre Tableau Desktop.
2. En la barra de menú, selecciona `Server` > `Sign In`.
3. Ingresa la URL de tu Tableau Server o Tableau Online y tus credenciales de inicio de sesión.

  1. Publicar el libro de trabajo

Una vez conectado, sigue estos pasos para publicar tu libro de trabajo:

1. En Tableau Desktop, abre el libro de trabajo que deseas publicar.
2. En la barra de menú, selecciona `Server` > `Publish Workbook`.
3. En el cuadro de diálogo `Publish Workbook`, selecciona el proyecto en el que deseas publicar el libro de trabajo.
4. Configura las opciones de publicación:
   - **Name**: Ingresa un nombre para el libro de trabajo.
   - **Description**: Proporciona una descripción opcional.
   - **Tags**: Añade etiquetas para facilitar la búsqueda.
   - **Permissions**: Configura los permisos para controlar quién puede ver y editar el libro de trabajo.
   - **Data Sources**: Elige si deseas publicar las fuentes de datos junto con el libro de trabajo.
5. Haz clic en `Publish`.

  1. Configurar permisos

Configurar los permisos es crucial para controlar el acceso a tu libro de trabajo. Puedes establecer permisos a nivel de proyecto o a nivel de libro de trabajo.

Permisos a nivel de proyecto

Los permisos a nivel de proyecto se aplican a todos los libros de trabajo dentro de un proyecto específico.

1. En Tableau Server o Tableau Online, navega al proyecto deseado.
2. Haz clic en `...` (más opciones) y selecciona `Permissions`.
3. Configura los permisos para los usuarios y grupos según sea necesario.

Permisos a nivel de libro de trabajo

Los permisos a nivel de libro de trabajo permiten un control más granular sobre quién puede ver y editar un libro de trabajo específico.

1. En Tableau Server o Tableau Online, navega al libro de trabajo publicado.
2. Haz clic en `...` (más opciones) y selecciona `Permissions`.
3. Configura los permisos para los usuarios y grupos según sea necesario.

Mejores prácticas para la publicación

  1. Optimiza el rendimiento

  • Usa extractos: Considera usar extractos en lugar de conexiones en vivo para mejorar el rendimiento.
  • Reduce el tamaño del libro de trabajo: Elimina hojas y datos innecesarios.

  1. Documenta tu trabajo

  • Añade descripciones: Proporciona descripciones claras para cada hoja y dashboard.
  • Usa comentarios: Añade comentarios en los cálculos y campos calculados para facilitar la comprensión.

  1. Configura alertas y suscripciones

  • Alertas: Configura alertas para notificar a los usuarios sobre cambios importantes en los datos.
  • Suscripciones: Permite que los usuarios se suscriban a actualizaciones periódicas del libro de trabajo.

Ejercicio práctico

Ejercicio 1: Publicar un libro de trabajo en Tableau Online

  1. Crea un libro de trabajo simple en Tableau Desktop con una visualización básica.
  2. Conéctate a Tableau Online.
  3. Publica el libro de trabajo en un proyecto específico.
  4. Configura los permisos para que solo ciertos usuarios puedan ver el libro de trabajo.

Solución del ejercicio

  1. Crear el libro de trabajo:
    • Abre Tableau Desktop y conecta a una fuente de datos.
    • Crea una visualización básica (por ejemplo, un gráfico de barras).
  2. Conectarse a Tableau Online:
    • En Tableau Desktop, selecciona Server > Sign In.
    • Ingresa la URL de Tableau Online y tus credenciales.
  3. Publicar el libro de trabajo:
    • Selecciona Server > Publish Workbook.
    • Elige el proyecto y configura las opciones de publicación.
    • Haz clic en Publish.
  4. Configurar permisos:
    • En Tableau Online, navega al libro de trabajo publicado.
    • Haz clic en ... y selecciona Permissions.
    • Configura los permisos para los usuarios deseados.

Conclusión

Publicar libros de trabajo en Tableau Server o Tableau Online es una habilidad esencial para compartir tus análisis y visualizaciones con otros. Asegúrate de seguir las mejores prácticas para optimizar el rendimiento y documentar tu trabajo adecuadamente. Con la práctica, te volverás más eficiente en la publicación y gestión de tus libros de trabajo en Tableau.
