Descripción general de la interfaz de Tableau

Siguiente ►
►

En esta sección, exploraremos la interfaz de Tableau para familiarizarnos con sus componentes y funcionalidades básicas. Comprender la interfaz es crucial para navegar y utilizar Tableau de manera eficiente.

Componentes principales de la interfaz de Tableau

  1. Barra de Menú

La barra de menú se encuentra en la parte superior de la ventana de Tableau y contiene varias opciones y comandos que se utilizan para realizar diferentes tareas. Los menús principales incluyen:

  • Archivo: Para abrir, guardar y exportar archivos.
  • Datos: Para conectar, actualizar y gestionar fuentes de datos.
  • Hoja: Para añadir nuevas hojas de trabajo, dashboards y historias.
  • Análisis: Para realizar operaciones analíticas como agregar líneas de tendencia y cálculos.
  • Mapa: Para configurar y personalizar mapas.
  • Formato: Para ajustar el formato de las visualizaciones.
  • Ayuda: Para acceder a la documentación y soporte de Tableau.

  1. Barra de Herramientas

La barra de herramientas proporciona acceso rápido a las funciones más utilizadas, como:

  • Nuevo: Crear una nueva hoja de trabajo.
  • Guardar: Guardar el libro de trabajo.
  • Deshacer/Rehacer: Deshacer o rehacer acciones.
  • Actualizar: Actualizar la conexión de datos.
  • Mostrar/Ocultar Panel de Datos: Alternar la visibilidad del panel de datos.

  1. Panel de Datos

El panel de datos se encuentra en el lado izquierdo de la ventana y muestra todas las fuentes de datos conectadas, así como los campos disponibles. Se divide en:

  • Datos: Muestra las fuentes de datos y sus campos.
  • Análisis: Proporciona herramientas analíticas como líneas de referencia, bandas y distribuciones.

  1. Área de Trabajo

El área de trabajo es donde se crean y editan las visualizaciones. Se compone de:

  • Hojas de Trabajo: Espacios individuales para crear gráficos y visualizaciones.
  • Dashboards: Combinaciones de múltiples hojas de trabajo en una sola vista.
  • Historias: Secuencias de hojas de trabajo y dashboards que cuentan una narrativa con los datos.

  1. Estante de Filtros

El estante de filtros se utiliza para aplicar filtros a los datos en la visualización. Los filtros pueden arrastrarse desde el panel de datos y soltarse en este estante.

  1. Estante de Marcas

El estante de marcas permite personalizar la apariencia de las visualizaciones. Incluye opciones para:

  • Color: Cambiar el color de los elementos.
  • Tamaño: Ajustar el tamaño de los elementos.
  • Etiqueta: Añadir etiquetas a los elementos.
  • Detalle: Añadir detalles adicionales a los elementos.
  • Forma: Cambiar la forma de los elementos.

  1. Estante de Filas y Columnas

Los estantes de filas y columnas se utilizan para definir la estructura de la visualización. Los campos arrastrados a estos estantes determinan cómo se organizan los datos en la visualización.

  1. Tarjetas de Filtros y Leyendas

Las tarjetas de filtros y leyendas se encuentran en el lado derecho de la ventana y proporcionan controles adicionales para interactuar con los filtros y ver las leyendas de las visualizaciones.

Ejemplo Práctico: Explorando la Interfaz

Vamos a crear una simple visualización para explorar la interfaz de Tableau.

  1. Conectar a una Fuente de Datos:

    • Abre Tableau y selecciona "Conectar" en la barra de herramientas.
    • Elige una fuente de datos, por ejemplo, un archivo de Excel o una base de datos.

  2. Crear una Nueva Hoja de Trabajo:

    • Haz clic en el icono de "Nueva Hoja" en la barra de herramientas.

  3. Arrastrar Campos a los Estantes:

    • Desde el panel de datos, arrastra un campo de dimensión (por ejemplo, "Categoría") al estante de columnas.
    • Arrastra un campo de medida (por ejemplo, "Ventas") al estante de filas.

  4. Personalizar la Visualización:

    • Utiliza el estante de marcas para cambiar el color de los elementos.
    • Añade etiquetas arrastrando el campo "Ventas" al estante de etiquetas.

  5. Aplicar Filtros:

    • Arrastra un campo (por ejemplo, "Región") al estante de filtros y selecciona las regiones que deseas incluir.

Ejercicio Práctico

Objetivo: Familiarizarse con la interfaz de Tableau creando una visualización básica.

  1. Conectar a una Fuente de Datos:

    • Conéctate a una fuente de datos de tu elección (por ejemplo, un archivo CSV).

  2. Crear una Nueva Hoja de Trabajo:

    • Crea una nueva hoja de trabajo.

  3. Arrastrar Campos a los Estantes:

    • Arrastra un campo de dimensión al estante de columnas.
    • Arrastra un campo de medida al estante de filas.

  4. Personalizar la Visualización:

    • Cambia el color de los elementos utilizando el estante de marcas.
    • Añade etiquetas a los elementos.

  5. Aplicar Filtros:

    • Aplica un filtro a la visualización.

Solución:

  1. Conectar a una Fuente de Datos:

    • Abre Tableau y selecciona "Conectar" en la barra de herramientas.
    • Elige un archivo CSV y conéctate a él.

  2. Crear una Nueva Hoja de Trabajo:

    • Haz clic en el icono de "Nueva Hoja" en la barra de herramientas.

  3. Arrastrar Campos a los Estantes:

    • Desde el panel de datos, arrastra el campo "Categoría" al estante de columnas.
    • Arrastra el campo "Ventas" al estante de filas.

  4. Personalizar la Visualización:

    • En el estante de marcas, selecciona "Color" y elige un color para los elementos.
    • Arrastra el campo "Ventas" al estante de etiquetas para añadir etiquetas.

  5. Aplicar Filtros:

    • Arrastra el campo "Región" al estante de filtros y selecciona las regiones que deseas incluir.

Conclusión

En esta sección, hemos explorado la interfaz de Tableau y sus componentes principales. Ahora deberías sentirte cómodo navegando por la interfaz y utilizando las herramientas básicas para crear visualizaciones. En la próxima sección, aprenderemos a conectar Tableau a diferentes fuentes de datos.
