Requisitos previos y configuración inicial

Siguiente ►
►

En esta sección, abordaremos los requisitos previos necesarios para comenzar a utilizar Google Data Studio y los pasos para realizar la configuración inicial. Este módulo es fundamental para asegurarnos de que todos los estudiantes tengan el entorno adecuado para seguir el curso sin problemas.

Requisitos previos

Antes de comenzar a trabajar con Google Data Studio, es importante asegurarse de cumplir con los siguientes requisitos:

  1. Cuenta de Google:

    • Necesitarás una cuenta de Google para acceder a Google Data Studio. Si no tienes una, puedes crear una de forma gratuita en accounts.google.com.

  2. Conexión a Internet:

    • Google Data Studio es una herramienta basada en la web, por lo que necesitarás una conexión a Internet estable para acceder y trabajar en la plataforma.

  3. Navegador web compatible:

    • Google Data Studio es compatible con los navegadores web más populares. Asegúrate de tener la última versión de uno de los siguientes navegadores:
      • Google Chrome
      • Mozilla Firefox
      • Microsoft Edge
      • Safari

  4. Conocimiento básico de hojas de cálculo:

    • Aunque no es estrictamente necesario, tener un conocimiento básico de hojas de cálculo (como Google Sheets o Microsoft Excel) puede ser útil para comprender mejor cómo se estructuran y manipulan los datos.

Configuración inicial

Una vez que hayas verificado que cumples con los requisitos previos, puedes proceder con la configuración inicial de Google Data Studio. Sigue estos pasos para comenzar:

  1. Acceder a Google Data Studio

  1. Abre tu navegador web y dirígete a Google Data Studio.
  2. Inicia sesión con tu cuenta de Google si aún no lo has hecho.

  1. Familiarizarse con la interfaz

Al acceder a Google Data Studio por primera vez, te encontrarás con la pantalla de inicio. Aquí hay algunos elementos clave de la interfaz que debes conocer:

  • Panel de navegación: Ubicado en el lado izquierdo, este panel te permite acceder a tus informes, fuentes de datos y otros recursos.
  • Botón "Crear": En la esquina superior izquierda, encontrarás un botón para crear nuevos informes y fuentes de datos.
  • Galería de plantillas: Google Data Studio ofrece una variedad de plantillas predefinidas que puedes utilizar como punto de partida para tus informes.

  1. Crear un nuevo informe

Para comenzar a trabajar con Google Data Studio, es útil crear un informe de prueba. Sigue estos pasos:

  1. Haz clic en el botón "Crear" y selecciona "Informe".
  2. Se abrirá una nueva ventana donde podrás elegir una plantilla o comenzar desde cero. Para este ejercicio, selecciona "Empezar desde cero".
  3. Asigna un nombre a tu informe y haz clic en "Crear".

  1. Conectar una fuente de datos

Un informe en Google Data Studio necesita datos para visualizar. A continuación, te mostramos cómo conectar una fuente de datos básica:

  1. En el nuevo informe, haz clic en "Añadir datos".
  2. Selecciona una de las fuentes de datos disponibles. Para este ejemplo, elige "Google Sheets".
  3. Autoriza a Google Data Studio a acceder a tus hojas de cálculo de Google.
  4. Selecciona una hoja de cálculo y una pestaña específica que contenga los datos que deseas utilizar.
  5. Haz clic en "Añadir" para conectar la fuente de datos a tu informe.

  1. Guardar y compartir tu informe

Una vez que hayas creado y configurado tu informe, es importante saber cómo guardarlo y compartirlo:

  1. Haz clic en el botón "Guardar" en la esquina superior derecha para guardar tu informe.
  2. Para compartir tu informe, haz clic en el botón "Compartir" y selecciona las opciones de compartir que prefieras (por ejemplo, compartir con personas específicas o generar un enlace de visualización).

Ejercicio práctico

Para reforzar los conceptos aprendidos en esta sección, realiza el siguiente ejercicio práctico:

  1. Crea un nuevo informe en Google Data Studio.
  2. Conecta una hoja de cálculo de Google Sheets que contenga datos de ejemplo (puedes crear una hoja de cálculo simple con datos ficticios si no tienes una disponible).
  3. Añade un gráfico básico (por ejemplo, un gráfico de barras) utilizando los datos conectados.
  4. Guarda y comparte tu informe con un compañero de clase o un amigo.

Solución del ejercicio

  1. Accede a Google Data Studio y crea un nuevo informe desde cero.
  2. Conecta una hoja de cálculo de Google Sheets siguiendo los pasos descritos anteriormente.
  3. Añade un gráfico de barras:
    • Haz clic en "Añadir un gráfico" y selecciona "Gráfico de barras".
    • Arrastra y suelta los campos de datos correspondientes en los ejes del gráfico.
  4. Guarda el informe haciendo clic en "Guardar".
  5. Comparte el informe haciendo clic en "Compartir" y elige las opciones de compartir adecuadas.

Conclusión

En esta sección, hemos cubierto los requisitos previos y los pasos para la configuración inicial de Google Data Studio. Ahora deberías estar listo para comenzar a explorar y crear informes básicos. En el próximo módulo, profundizaremos en cómo conectar y gestionar diversas fuentes de datos para enriquecer tus informes.
