Instalación de Power BI Desktop

Siguiente ►
►

Introducción

Power BI Desktop es una herramienta gratuita que permite a los usuarios conectarse a múltiples fuentes de datos, transformarlos y visualizarlos en informes interactivos. En esta lección, aprenderás cómo descargar e instalar Power BI Desktop en tu computadora.

Requisitos del Sistema

Antes de comenzar con la instalación, asegúrate de que tu sistema cumpla con los siguientes requisitos mínimos:

  • Sistema Operativo: Windows 7 / Windows Server 2008 R2, o versiones posteriores.
  • Memoria: Al menos 2 GB de RAM; se recomienda 4 GB o más.
  • Espacio en Disco: Al menos 1 GB de espacio disponible.
  • Resolución de Pantalla: Mínimo 1440 x 900 o 1600 x 900 (se recomienda 1920 x 1080 o superior).

Pasos para la Instalación

  1. Descargar Power BI Desktop

  1. Visita el sitio web oficial de Power BI:

    • Abre tu navegador web y dirígete a la página de descarga de Power BI Desktop: Descargar Power BI Desktop.

  2. Inicia la descarga:

    • Haz clic en el botón "Descargar gratis" o "Download Free".

  1. Ejecutar el Instalador

  1. Abrir el archivo descargado:

    • Una vez que la descarga haya finalizado, abre el archivo ejecutable (PBIDesktopSetup.exe).

  2. Iniciar el proceso de instalación:

    • Aparecerá una ventana de configuración. Haz clic en "Siguiente" para continuar.

  1. Aceptar los Términos de Licencia

  1. Leer y aceptar los términos:
    • Lee los términos del acuerdo de licencia. Si estás de acuerdo, selecciona la casilla "Acepto los términos del acuerdo de licencia" y haz clic en "Siguiente".

  1. Seleccionar la Ubicación de Instalación

  1. Elegir la carpeta de destino:
    • Puedes aceptar la ubicación predeterminada o hacer clic en "Cambiar" para seleccionar una carpeta diferente. Luego, haz clic en "Siguiente".

  1. Completar la Instalación

  1. Instalar Power BI Desktop:

    • Haz clic en "Instalar" para comenzar la instalación. Este proceso puede tardar unos minutos.

  2. Finalizar la instalación:

    • Una vez completada la instalación, haz clic en "Finalizar". Power BI Desktop se abrirá automáticamente.

Primer Inicio de Power BI Desktop

  1. Abrir Power BI Desktop

  • Si Power BI Desktop no se abre automáticamente, puedes iniciarlo desde el menú de inicio de Windows o buscando "Power BI Desktop" en la barra de búsqueda.

  1. Configuración Inicial

  • Pantalla de bienvenida:

    • Al abrir Power BI Desktop por primera vez, verás una pantalla de bienvenida. Puedes optar por iniciar sesión con tu cuenta de Microsoft o cerrar la ventana para continuar sin iniciar sesión.

  • Actualizaciones:

    • Power BI Desktop se actualiza regularmente. Asegúrate de mantener la aplicación actualizada para acceder a las últimas características y mejoras.

Ejercicio Práctico

Ejercicio 1: Instalación de Power BI Desktop

  1. Descarga e instala Power BI Desktop siguiendo los pasos descritos anteriormente.
  2. Abre Power BI Desktop y explora la interfaz inicial.

Solución

  • Descarga: Visita Descargar Power BI Desktop y descarga el instalador.
  • Instalación: Ejecuta el archivo PBIDesktopSetup.exe y sigue las instrucciones en pantalla.
  • Primer Inicio: Abre Power BI Desktop desde el menú de inicio y explora la pantalla de bienvenida.

Conclusión

En esta lección, has aprendido cómo descargar e instalar Power BI Desktop en tu computadora. Ahora estás listo para comenzar a explorar las capacidades de Power BI y conectar tus primeras fuentes de datos. En la próxima lección, revisaremos la interfaz de Power BI Desktop para que te familiarices con sus componentes y funcionalidades básicas.
