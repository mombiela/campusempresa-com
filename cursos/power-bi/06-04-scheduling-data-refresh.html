Programación de actualización de datos

Siguiente ►
►

En este tema, aprenderás cómo programar la actualización de datos en Power BI para asegurarte de que tus informes y paneles siempre reflejen la información más reciente. La programación de actualizaciones es crucial para mantener la integridad y la relevancia de los datos en tus visualizaciones.

Conceptos Clave

  1. Actualización de Datos: Proceso de refrescar los datos en tus informes y paneles para reflejar los cambios más recientes en las fuentes de datos.
  2. Gateway de Datos: Herramienta que permite la conexión segura entre Power BI y las fuentes de datos locales.
  3. Programación de Actualización: Configuración de horarios específicos para que Power BI actualice automáticamente los datos.

Pasos para Programar la Actualización de Datos

  1. Configuración del Gateway de Datos

Si tus datos están en una fuente local (por ejemplo, una base de datos SQL Server en tu red local), necesitarás configurar un Gateway de Datos.

  1. Instalar el Gateway de Datos:

    • Descarga e instala el Gateway de Datos desde el sitio web de Power BI.
    • Sigue las instrucciones de instalación y asegúrate de que el Gateway esté en funcionamiento.

  2. Configurar el Gateway en Power BI:

    • Ve a la sección de "Gateways" en el servicio de Power BI.
    • Añade un nuevo Gateway y proporciona la información necesaria para conectarlo a tus fuentes de datos.

  1. Publicar el Informe en el Servicio de Power BI

  1. Publicar desde Power BI Desktop:
    • Abre tu informe en Power BI Desktop.
    • Haz clic en "Publicar" y selecciona el espacio de trabajo en el servicio de Power BI donde deseas publicar el informe.

  1. Configurar la Programación de Actualización

  1. Acceder a la Configuración del Conjunto de Datos:

    • En el servicio de Power BI, navega hasta el conjunto de datos que deseas actualizar.
    • Haz clic en los tres puntos (...) junto al conjunto de datos y selecciona "Configuración".

  2. Configurar la Programación de Actualización:

    • En la sección de "Programación de actualización", activa la opción de "Mantener los datos actualizados".
    • Configura la frecuencia de actualización (diaria, semanal, etc.) y las horas específicas en las que deseas que se realice la actualización.

  3. Configurar las Credenciales de la Fuente de Datos:

    • Asegúrate de que las credenciales para acceder a la fuente de datos estén configuradas correctamente.
    • Si estás utilizando un Gateway de Datos, selecciona el Gateway adecuado y proporciona las credenciales necesarias.

  1. Verificar y Monitorear la Actualización

  1. Verificar el Estado de la Actualización:

    • Después de configurar la programación, puedes verificar el estado de las actualizaciones en la sección de "Historial de actualización" del conjunto de datos.
    • Aquí podrás ver si las actualizaciones se han realizado correctamente o si ha habido algún error.

  2. Monitorear las Actualizaciones:

    • Power BI enviará notificaciones por correo electrónico si alguna actualización falla.
    • Revisa regularmente el historial de actualizaciones para asegurarte de que todo esté funcionando correctamente.

Ejemplo Práctico

A continuación, se muestra un ejemplo de cómo configurar la programación de actualización para un conjunto de datos en Power BI:

1. Publica tu informe desde Power BI Desktop al servicio de Power BI.
2. En el servicio de Power BI, navega hasta el conjunto de datos publicado.
3. Haz clic en los tres puntos (...) junto al conjunto de datos y selecciona "Configuración".
4. En la sección de "Programación de actualización", activa la opción de "Mantener los datos actualizados".
5. Configura la frecuencia de actualización y las horas específicas.
6. Asegúrate de que las credenciales de la fuente de datos estén configuradas correctamente.
7. Verifica el estado de la actualización en la sección de "Historial de actualización".

Ejercicio Práctico

Ejercicio: Configura la programación de actualización para un conjunto de datos que se conecta a una base de datos SQL Server local.

  1. Instala y configura el Gateway de Datos.
  2. Publica un informe desde Power BI Desktop al servicio de Power BI.
  3. Configura la programación de actualización para que se realice diariamente a las 8:00 AM.
  4. Verifica el estado de la actualización en el historial de actualizaciones.

Solución:

  1. Instalar y configurar el Gateway de Datos:

    • Descarga e instala el Gateway de Datos.
    • Configura el Gateway en el servicio de Power BI.

  2. Publicar un informe desde Power BI Desktop:

    • Abre el informe en Power BI Desktop.
    • Haz clic en "Publicar" y selecciona el espacio de trabajo en el servicio de Power BI.

  3. Configurar la programación de actualización:

    • En el servicio de Power BI, navega hasta el conjunto de datos.
    • Haz clic en los tres puntos (...) y selecciona "Configuración".
    • Activa la opción de "Mantener los datos actualizados".
    • Configura la frecuencia de actualización diaria a las 8:00 AM.
    • Asegúrate de que las credenciales de la fuente de datos estén configuradas correctamente.

  4. Verificar el estado de la actualización:

    • Navega a la sección de "Historial de actualización" del conjunto de datos.
    • Verifica que la actualización se haya realizado correctamente.

Conclusión

La programación de actualización de datos en Power BI es una funcionalidad esencial para mantener tus informes y paneles actualizados con la información más reciente. A través de la configuración adecuada del Gateway de Datos y la programación de actualizaciones, puedes automatizar este proceso y asegurarte de que tus visualizaciones siempre reflejen datos precisos y actuales.
