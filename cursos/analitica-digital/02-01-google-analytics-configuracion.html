Google Analytics: Configuración Inicial

Siguiente ►
►

Introducción

Google Analytics es una herramienta esencial para cualquier profesional de la analítica digital. Permite recopilar, analizar y reportar datos sobre el tráfico y el comportamiento de los usuarios en un sitio web. En esta sección, aprenderás cómo realizar la configuración inicial de Google Analytics para empezar a recopilar datos de tu sitio web.

Objetivos de Aprendizaje

Al finalizar esta sección, deberías ser capaz de:

  • Crear una cuenta de Google Analytics.
  • Configurar una propiedad y una vista.
  • Implementar el código de seguimiento en tu sitio web.

Pasos para la Configuración Inicial

  1. Crear una Cuenta de Google Analytics

Paso 1: Accede a Google Analytics

  • Visita Google Analytics.
  • Inicia sesión con tu cuenta de Google.

Paso 2: Crear una Nueva Cuenta

  • Haz clic en "Admin" en la parte inferior izquierda del panel.
  • En la columna "Cuenta", haz clic en "Crear cuenta".
  • Completa los detalles de la cuenta:
    • Nombre de la cuenta: Un nombre descriptivo para tu cuenta.
    • Configuración de intercambio de datos: Selecciona las opciones que prefieras.

  1. Configurar una Propiedad

Paso 1: Crear una Nueva Propiedad

  • En la columna "Propiedad", haz clic en "Crear propiedad".
  • Completa los detalles de la propiedad:
    • Nombre de la propiedad: Un nombre descriptivo para tu sitio web.
    • Zona horaria de informes: Selecciona la zona horaria adecuada.
    • Moneda: Selecciona la moneda que prefieras.

Paso 2: Configuración Avanzada de la Propiedad

  • Selecciona "Mostrar opciones avanzadas".
  • Activa la opción "Crear una propiedad de Universal Analytics".
  • Introduce la URL de tu sitio web.
  • Selecciona "Crear una propiedad de Universal Analytics solamente" si solo deseas usar Universal Analytics.

  1. Configurar una Vista

Paso 1: Crear una Nueva Vista

  • En la columna "Vista", haz clic en "Crear vista".
  • Completa los detalles de la vista:
    • Nombre de la vista: Un nombre descriptivo para la vista (por ejemplo, "Vista sin filtros").
    • Zona horaria de informes: Selecciona la zona horaria adecuada.

  1. Implementar el Código de Seguimiento

Paso 1: Obtener el Código de Seguimiento

  • En la columna "Propiedad", haz clic en "Información de seguimiento" y luego en "Código de seguimiento".
  • Copia el código de seguimiento que aparece en la sección "Etiqueta de sitio global (gtag.js)".

Paso 2: Añadir el Código de Seguimiento a tu Sitio Web

  • Abre el archivo HTML de tu sitio web.
  • Pega el código de seguimiento justo antes de la etiqueta </head>.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Sitio Web</title>
    <!-- Código de seguimiento de Google Analytics -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
    <script>
        window.dataLayer = window.dataLayer || [];
        function gtag(){dataLayer.push(arguments);}
        gtag('js', new Date());

        gtag('config', 'UA-XXXXXXXXX-X');
    </script>
    <!-- Fin del código de seguimiento -->
</head>
<body>
    <!-- Contenido del sitio web -->
</body>
</html>

  1. Verificar la Implementación

Paso 1: Verificar en Tiempo Real

  • Vuelve a Google Analytics.
  • Ve a "Informes" y selecciona "Tiempo real" > "Visión general".
  • Abre tu sitio web en una nueva pestaña y navega por él.
  • Deberías ver actividad en el informe de tiempo real.

Ejercicio Práctico

Ejercicio 1: Configuración de Google Analytics

  1. Crea una cuenta de Google Analytics para un sitio web ficticio.
  2. Configura una propiedad y una vista para este sitio web.
  3. Implementa el código de seguimiento en un archivo HTML básico.
  4. Verifica que el seguimiento esté funcionando correctamente utilizando el informe de tiempo real.

Solución del Ejercicio

  1. Crear una cuenta:

    • Nombre de la cuenta: "Mi Sitio Web Ficticio".
    • Configuración de intercambio de datos: Selecciona según tus preferencias.

  2. Configurar una propiedad:

    • Nombre de la propiedad: "Mi Sitio Web Ficticio".
    • Zona horaria: "GMT-5".
    • Moneda: "USD".

  3. Configurar una vista:

    • Nombre de la vista: "Vista sin filtros".
    • Zona horaria: "GMT-5".

  4. Implementar el código de seguimiento:

    • Copia el código de seguimiento y pégalo en el archivo HTML básico proporcionado anteriormente.

  5. Verificar la implementación:

    • Abre el informe de tiempo real en Google Analytics y navega por tu sitio web para verificar que los datos se están recopilando correctamente.

Conclusión

En esta sección, has aprendido a realizar la configuración inicial de Google Analytics, incluyendo la creación de una cuenta, la configuración de una propiedad y una vista, y la implementación del código de seguimiento en tu sitio web. Estos pasos son fundamentales para empezar a recopilar datos y analizar el comportamiento de los usuarios en tu sitio web. En la próxima sección, aprenderemos a utilizar Google Tag Manager para una implementación más avanzada y flexible.
