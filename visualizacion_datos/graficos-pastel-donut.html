Introducción

Los gráficos de pastel y donut son herramientas visuales populares en la visualización de datos, especialmente útiles para mostrar la proporción de diferentes categorías dentro de un conjunto de datos. Este tema cubrirá los conceptos básicos, las diferencias entre ambos tipos de gráficos, y cómo crearlos e interpretarlos.

Conceptos Clave

  • Gráfico de Pastel: Representa datos en forma de un círculo dividido en sectores, donde cada sector muestra una proporción del total.
  • Gráfico de Donut: Similar al gráfico de pastel, pero con un agujero en el centro, lo que permite una mejor visualización de múltiples series de datos.

Ventajas y Desventajas

| Ventajas | Desventajas | |---------------------------|---------------------------------| | Fácil de entender | Difícil de leer con muchas categorías | | Visualmente atractivo | No muestra cambios en el tiempo | | Útil para proporciones | Comparaciones precisas son difíciles |

Creación de Gráficos de Pastel

Ejemplo en Python con Matplotlib

import matplotlib.pyplot as plt

# Datos
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Resalta el primer sector

# Crear gráfico de pastel
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)
plt.axis('equal')  # Asegura que el gráfico sea un círculo
plt.show()
  • labels: Etiquetas para cada sector.
  • sizes: Tamaño de cada sector.
  • colors: Colores para cada sector.
  • explode: Desplaza un sector para resaltarlo.
  • autopct: Formato para mostrar porcentajes.
  • shadow: Añade sombra para efecto visual.
  • startangle: Ángulo de inicio del primer sector.

Creación de Gráficos de Donut

Ejemplo en Python con Matplotlib

import matplotlib.pyplot as plt

# Datos
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Resalta el primer sector

# Crear gráfico de pastel
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)

# Añadir círculo blanco en el centro para crear el efecto de donut
centre_circle = plt.Circle((0,0),0.70,fc='white')
fig = plt.gcf()
fig.gca().add_artist(centre_circle)

plt.axis('equal')  # Asegura que el gráfico sea un círculo
plt.show()
  • centre_circle: Círculo blanco en el centro para crear el efecto de donut.

Ejercicios Prácticos

  1. Crear un gráfico de pastel: Utiliza un conjunto de datos propio para crear un gráfico de pastel que muestre las proporciones de diferentes categorías.
  2. Modificar un gráfico de pastel: Añade etiquetas, colores personalizados y resalta un sector específico.
  3. Crear un gráfico de donut: Convierte tu gráfico de pastel en un gráfico de donut añadiendo un círculo en el centro.

Conclusión

Los gráficos de pastel y donut son herramientas visuales efectivas para mostrar proporciones dentro de un conjunto de datos. Aunque tienen limitaciones, su simplicidad y atractivo visual los hacen útiles en muchas situaciones. Con la práctica, podrás crear y personalizar estos gráficos para comunicar tus datos de manera clara y efectiva.
