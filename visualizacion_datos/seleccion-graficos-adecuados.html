Introducción

La selección del gráfico adecuado es crucial para una visualización de datos efectiva. Un gráfico bien elegido puede comunicar información compleja de manera clara y concisa, mientras que una mala elección puede llevar a malentendidos y confusión. En esta sección, aprenderemos cómo seleccionar el gráfico más adecuado según el tipo de datos y el mensaje que queremos transmitir.

Tipos de Datos

Antes de seleccionar un gráfico, es importante entender el tipo de datos que tenemos. Los datos pueden clasificarse en varias categorías:

  • Datos Categóricos: Datos que se dividen en categorías distintas y no tienen un orden natural (e.g., colores, tipos de productos).
  • Datos Ordinales: Datos categóricos con un orden natural (e.g., clasificaciones, niveles de satisfacción).
  • Datos Numéricos: Datos que son números y pueden ser continuos o discretos (e.g., ingresos, temperatura).

Tipos de Gráficos

Gráficos de Barras

  • Uso: Comparar cantidades entre diferentes categorías.
  • Datos: Categóricos o ordinales.
  • Ejemplo: Comparar ventas de diferentes productos.
import matplotlib.pyplot as plt

categories = ['Producto A', 'Producto B', 'Producto C']
values = [100, 150, 80]

plt.bar(categories, values)
plt.xlabel('Productos')
plt.ylabel('Ventas')
plt.title('Ventas por Producto')
plt.show()

Gráficos de Líneas

  • Uso: Mostrar tendencias a lo largo del tiempo.
  • Datos: Numéricos (generalmente en el eje Y) y ordinales (generalmente en el eje X).
  • Ejemplo: Evolución de las ventas mensuales.
import matplotlib.pyplot as plt

months = ['Enero', 'Febrero', 'Marzo', 'Abril']
sales = [100, 120, 150, 130]

plt.plot(months, sales)
plt.xlabel('Meses')
plt.ylabel('Ventas')
plt.title('Evolución de Ventas Mensuales')
plt.show()

Gráficos de Dispersión

  • Uso: Mostrar la relación entre dos variables numéricas.
  • Datos: Numéricos.
  • Ejemplo: Relación entre la inversión en publicidad y las ventas.
import matplotlib.pyplot as plt

investment = [1000, 2000, 3000, 4000]
sales = [150, 180, 220, 260]

plt.scatter(investment, sales)
plt.xlabel('Inversión en Publicidad')
plt.ylabel('Ventas')
plt.title('Relación entre Inversión y Ventas')
plt.show()

Gráficos de Pastel

  • Uso: Mostrar la proporción de partes de un todo.
  • Datos: Categóricos.
  • Ejemplo: Distribución del mercado por empresa.
import matplotlib.pyplot as plt

labels = ['Empresa A', 'Empresa B', 'Empresa C']
sizes = [40, 35, 25]

plt.pie(sizes, labels=labels, autopct='%1.1f%%')
plt.title('Distribución del Mercado por Empresa')
plt.show()

Comparación de Gráficos

| Tipo de Gráfico | Uso Principal | Tipo de Datos | |-----------------------|----------------------------------------|---------------------| | Gráfico de Barras | Comparar cantidades entre categorías | Categóricos, Ordinales | | Gráfico de Líneas | Mostrar tendencias a lo largo del tiempo | Numéricos, Ordinales | | Gráfico de Dispersión | Mostrar relación entre dos variables | Numéricos | | Gráfico de Pastel | Mostrar proporciones | Categóricos |

Ejercicios

  1. Ejercicio 1: Crea un gráfico de barras que compare las ventas de cuatro productos diferentes.
  2. Ejercicio 2: Utiliza un gráfico de líneas para mostrar la evolución de la temperatura diaria durante una semana.
  3. Ejercicio 3: Realiza un gráfico de dispersión para mostrar la relación entre el número de horas estudiadas y las calificaciones obtenidas por los estudiantes.
  4. Ejercicio 4: Crea un gráfico de pastel que muestre la distribución de los gastos mensuales en diferentes categorías (e.g., alimentación, transporte, entretenimiento).

Conclusión

Seleccionar el gráfico adecuado es esencial para una visualización de datos efectiva. Al entender el tipo de datos y el mensaje que queremos comunicar, podemos elegir el gráfico que mejor represente la información. Practicar con diferentes tipos de gráficos y datos ayudará a mejorar nuestras habilidades en visualización de datos.
