Introducción a IoT y GCP

  • ¿Qué es IoT?
    • Internet de las Cosas (IoT) se refiere a la red de objetos físicos incrustados con sensores, software y otras tecnologías para conectar e intercambiar datos con otros dispositivos y sistemas a través de internet.
  • ¿Por qué usar GCP para IoT?
    • Escalabilidad
    • Seguridad
    • Integración con otros servicios de Google
    • Capacidades avanzadas de análisis y aprendizaje automático

Google Cloud IoT Core

  • Descripción General de IoT Core
    • Servicio gestionado para conectar, gestionar e ingerir datos de dispositivos dispersos globalmente de manera segura.
  • Conceptos Clave
    • Registro de Dispositivos: Un contenedor para dispositivos.
    • Dispositivo: Representa un dispositivo físico.
    • Configuración del Dispositivo: Configuraciones y ajustes para dispositivos.
    • Estado del Dispositivo: El estado actual del dispositivo.
    • Telemetría: Datos enviados desde dispositivos a la nube.
  • Configuración de IoT Core
    • Creación de un registro de dispositivos.
    • Registro de un dispositivo.
    • Configuración de credenciales del dispositivo.

Ejemplo: Creación de un Registro de Dispositivos

from google.cloud import iot_v1

client = iot_v1.DeviceManagerClient()
parent = client.location_path('your-project-id', 'us-central1')
registry = {
    'id': 'my-registry',
    'event_notification_configs': [{
        'pubsub_topic_name': 'projects/your-project-id/topics/your-topic'
    }]
}
response = client.create_device_registry(parent, registry)
print('Created registry:', response)
  • Explicación: Este fragmento de código demuestra cómo crear un registro de dispositivos en GCP usando Python. Reemplaza 'your-project-id' y 'your-topic' con tu ID de proyecto y tema de Pub/Sub reales.

Ejemplo: Ingesta de Datos con Pub/Sub

from google.cloud import pubsub_v1

publisher = pubsub_v1.PublisherClient()
topic_path = publisher.topic_path('your-project-id', 'your-topic')

data = 'Hello, IoT!'
data = data.encode('utf-8')
future = publisher.publish(topic_path, data)
print(future.result())
  • Explicación: Este fragmento de código muestra cómo publicar un mensaje en un tema de Pub/Sub. Reemplaza 'your-project-id' y 'your-topic' con tu ID de proyecto y nombre de tema reales.
