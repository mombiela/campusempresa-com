Introducción a la Iluminación
Animación básica ►

La iluminación es un aspecto crucial para crear entornos realistas y visualmente atractivos en Unreal Engine. Este tema te guiará a través de los conceptos básicos de la iluminación, los diferentes tipos de luces y las técnicas avanzadas de iluminación para mejorar tus proyectos.

Conceptos Clave de la Iluminación

  • Tipos de Luz: Comprender los diferentes tipos de luces disponibles en Unreal Engine.
  • Propiedades de la Luz: Aprender sobre las propiedades y configuraciones que se pueden ajustar para cada tipo de luz.
  • Técnicas de Iluminación: Explorar varias técnicas para lograr efectos de iluminación realistas.
  • Consideraciones de Rendimiento: Equilibrar la calidad visual y el rendimiento.

Tipos de Luz

Unreal Engine proporciona varios tipos de luces, cada uno adecuado para diferentes propósitos:

  1. Luz Direccional:

    • Simula la luz del sol o de la luna.
    • Emite rayos de luz paralelos.
    • Ideal para escenas al aire libre.

  2. Luz Puntual:

    • Emite luz en todas las direcciones desde un solo punto.
    • Similar a una bombilla.
    • Útil para fuentes de luz localizadas.

  3. Luz Focal:

    • Emite luz en forma de cono.
    • Similar a una linterna o un foco de escenario.
    • Ideal para efectos de iluminación enfocados.

  4. Luz Rectangular:

    • Emite luz desde un área rectangular.
    • Útil para simular luces fluorescentes o pantallas de televisión.

  5. Luz de Cielo:

    • Captura las partes distantes de la escena y aplica esa iluminación a todo el nivel.
    • Ideal para iluminación ambiental y reflejos.

Propiedades de la Luz

Cada tipo de luz tiene varias propiedades que se pueden ajustar para lograr el efecto deseado:

  • Intensidad: Controla el brillo de la luz.
  • Color: Establece el color de la luz.
  • Radio de Atenuación: Define el área afectada por la luz.
  • Proyectar Sombras: Determina si la luz proyecta sombras.
  • Radio/Longitud de la Fuente: Ajusta el tamaño de la fuente de luz para sombras más suaves o más duras.

Ejemplo: Configuración de una Luz Puntual

// Añadiendo una Luz Puntual en C++
// Asumiendo que tienes una clase básica de Actor

#include "Components/PointLightComponent.h"

APointLightActor::APointLightActor()
{
    // Crear un Componente de Luz Puntual
    PointLight = CreateDefaultSubobject<UPointLightComponent>(TEXT("PointLight"));
    PointLight->Intensity = 3000.0f; // Establecer la intensidad
    PointLight->bVisible = true; // Hacerla visible
    PointLight->SetupAttachment(RootComponent); // Adjuntar al componente raíz
}

En Blueprints, puedes añadir un componente de Luz Puntual a un actor y ajustar sus propiedades en el panel de Detalles.

Técnicas de Iluminación

  1. Iluminación Estática:

    • Iluminación precomputada.
    • Mejor para objetos estáticos.
    • Sombras e iluminación de alta calidad pero sin cambios dinámicos.

  2. Iluminación Estacionaria:

    • Parcialmente dinámica.
    • Permite algunos cambios en las propiedades de la luz en tiempo de ejecución.
    • Adecuada para objetos que se mueven ocasionalmente.

  3. Iluminación Dinámica:

    • Totalmente dinámica.
    • Iluminación y sombras en tiempo real.
    • Ideal para objetos que se mueven frecuentemente pero más intensiva en rendimiento.

Ejemplo: Uso de Iluminación Dinámica

// Habilitando Iluminación Dinámica para una Luz Direccional en C++

#include "Components/DirectionalLightComponent.h"

ADirectionalLightActor::ADirectionalLightActor()
{
    // Crear un Componente de Luz Direccional
    DirectionalLight = CreateDefaultSubobject<UDirectionalLightComponent>(TEXT("DirectionalLight"));
    DirectionalLight->Mobility = EComponentMobility::Movable; // Establecer como Movible para iluminación dinámica
    DirectionalLight->Intensity = 10.0f; // Establecer la intensidad
    DirectionalLight->SetupAttachment(RootComponent); // Adjuntar al componente raíz
}

Consideraciones de Rendimiento

Equilibrar la calidad visual y el rendimiento es esencial:

  • Usa Iluminación Estática para objetos no móviles para ahorrar en rendimiento.
  • Limita el número de Luces Dinámicas para evitar impactos en el rendimiento.
  • Optimiza la Configuración de Sombras: Usa sombras de menor resolución para objetos distantes.
  • Usa Horneado de Luz: Precomputa la iluminación para objetos estáticos para mejorar el rendimiento en tiempo de ejecución.

Conclusión

La iluminación en Unreal Engine es una herramienta poderosa que puede mejorar significativamente la calidad visual de tus proyectos. Al comprender los diferentes tipos de luces, sus propiedades y varias técnicas de iluminación, puedes crear entornos realistas e inmersivos. Siempre considera las implicaciones de rendimiento y optimiza tu configuración de iluminación para lograr el mejor equilibrio entre fidelidad visual y rendimiento.
