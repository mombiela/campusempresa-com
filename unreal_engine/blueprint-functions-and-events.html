Introducción

Blueprints son un sistema de scripting visual en Unreal Engine que te permite crear elementos de juego sin escribir código. Este tema cubrirá las Funciones y Eventos de Blueprint, componentes esenciales para crear mecánicas de juego interactivas y reutilizables.

Funciones de Blueprint

¿Qué son las Funciones de Blueprint?

  • Definición: Las Funciones de Blueprint son nodos reutilizables que encapsulan un conjunto de instrucciones o lógica. Ayudan a organizar y reutilizar código dentro de los Blueprints.
  • Propósito: Las funciones se utilizan para realizar tareas específicas y pueden ser llamadas múltiples veces desde diferentes lugares dentro de tus Blueprints.

Crear una Función de Blueprint

  1. Abre tu Blueprint: Navega al Editor de Blueprint.
  2. Añadir una Función: Haz clic en la pestaña "Functions" en el panel My Blueprint y luego haz clic en el botón "+" para crear una nueva función.
  3. Nombra la Función: Dale a tu función un nombre significativo que describa su propósito.
  4. Añadir Nodos: Arrastra y suelta nodos en el gráfico de la función para definir su comportamiento.

Ejemplo: Función Simple de Cálculo de Salud

// Nombre de la Función: CalculateHealth
// Entradas: CurrentHealth (float), Damage (float)
// Salidas: NewHealth (float)

1. Crea una nueva función llamada "CalculateHealth".
2. Añade dos parámetros de entrada: CurrentHealth (float) y Damage (float).
3. Añade un parámetro de salida: NewHealth (float).
4. Dentro del gráfico de la función, resta Damage de CurrentHealth.
5. Asigna el resultado a NewHealth.

Gráfico:
[CurrentHealth] -- [Subtract] -- [Damage] --> [NewHealth]

Conceptos Clave

  • Parámetros de Entrada: Variables pasadas a la función.
  • Parámetros de Salida: Variables devueltas por la función.
  • Variables Locales: Variables utilizadas dentro de la función que no son accesibles fuera de ella.

Eventos de Blueprint

¿Qué son los Eventos de Blueprint?

  • Definición: Los eventos son nodos que responden a acciones o condiciones específicas en el juego, como la entrada del jugador, colisiones o temporizadores.
  • Propósito: Los eventos se utilizan para desencadenar piezas específicas de lógica cuando se cumplen ciertas condiciones.

Tipos Comunes de Eventos

  • Begin Play: Se desencadena cuando el juego comienza.
  • Tick: Se desencadena cada frame.
  • Eventos Personalizados: Eventos definidos por el usuario que pueden ser llamados desde otros Blueprints o dentro del mismo Blueprint.

Crear un Evento Personalizado

  1. Abre tu Blueprint: Navega al Editor de Blueprint.
  2. Añadir un Evento: Haz clic derecho en el Gráfico de Eventos y selecciona "Add Custom Event".
  3. Nombra el Evento: Dale a tu evento un nombre significativo.
  4. Añadir Lógica: Arrastra y suelta nodos para definir lo que sucede cuando se desencadena el evento.

Ejemplo: Evento Personalizado para Imprimir un Mensaje

// Nombre del Evento: PrintMessage
// Acción: Imprimir "Hello, Unreal!" en la pantalla

1. Crea un nuevo evento personalizado llamado "PrintMessage".
2. Añade un nodo "Print String".
3. Conecta el pin de ejecución de "PrintMessage" al nodo "Print String".
4. Establece la entrada de cadena del nodo "Print String" a "Hello, Unreal!".

Gráfico:
[PrintMessage] --> [Print String ("Hello, Unreal!")]

Conceptos Clave

  • Gráfico de Eventos: El área principal donde se definen los eventos y su lógica correspondiente.
  • Flujo de Ejecución: El orden en el que se ejecutan los nodos, típicamente controlado por pines de ejecución blancos.

Combinando Funciones y Eventos

Ejemplo: Usar una Función en un Evento

// Evento: OnTakeDamage
// Función: CalculateHealth

1. Crea un evento personalizado llamado "OnTakeDamage".
2. Añade parámetros de entrada: CurrentHealth (float) y Damage (float).
3. Llama a la función "CalculateHealth" dentro del evento.
4. Usa la salida de "CalculateHealth" para actualizar la salud del jugador.

Gráfico:
[OnTakeDamage] --> [CalculateHealth (CurrentHealth, Damage)] --> [Set Player Health]

Resumen

  • Funciones de Blueprint: Encapsulan lógica reutilizable, haciendo tus Blueprints más organizados y modulares.
  • Eventos de Blueprint: Responden a acciones o condiciones específicas, permitiéndote crear elementos de juego interactivos.
  • Combinando Funciones y Eventos: Mejora la flexibilidad y reutilización de tus Blueprints, permitiendo mecánicas de juego más complejas e interactivas.

Al dominar las Funciones y Eventos de Blueprint, puedes crear elementos de juego eficientes, reutilizables e interactivos en Unreal Engine, allanando el camino para un desarrollo de juegos más avanzado.
