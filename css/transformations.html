Introducción a las Transformaciones

Las transformaciones CSS te permiten modificar el espacio de coordenadas del modelo de formato visual de CSS. Esto significa que puedes rotar, escalar, sesgar o trasladar elementos en el espacio 2D o 3D.

  • Funciones de Transformación: Las funciones principales utilizadas en las transformaciones CSS son translate, rotate, scale y skew.
  • Propiedad Transform: La propiedad transform aplica una transformación 2D o 3D a un elemento.

Transformaciones 2D

Translate

La función translate mueve un elemento desde su posición actual.

.element {
  transform: translate(50px, 100px);
}
  • Explicación: Este código mueve el elemento 50 píxeles a la derecha y 100 píxeles hacia abajo desde su posición original.

Rotate

La función rotate rota un elemento alrededor de un punto fijo.

.element {
  transform: rotate(45deg);
}
  • Explicación: Este código rota el elemento 45 grados en el sentido de las agujas del reloj.

Scale

La función scale redimensiona un elemento.

.element {
  transform: scale(1.5, 2);
}
  • Explicación: Este código escala el elemento a 1.5 veces su ancho y 2 veces su altura.

Skew

La función skew sesga un elemento a lo largo de los ejes X e Y.

.element {
  transform: skew(30deg, 20deg);
}
  • Explicación: Este código sesga el elemento 30 grados a lo largo del eje X y 20 grados a lo largo del eje Y.

Transformaciones 3D

Perspective

La función perspective define la perspectiva desde la cual se ven los elementos transformados en 3D.

.container {
  perspective: 500px;
}
  • Explicación: Este código establece la distancia de perspectiva a 500 píxeles.

RotateX, RotateY, RotateZ

Estas funciones rotan un elemento alrededor de los ejes X, Y y Z, respectivamente.

.element {
  transform: rotateX(45deg);
}
  • Explicación: Este código rota el elemento 45 grados alrededor del eje X.

Scale3d

La función scale3d escala un elemento en el espacio 3D.

.element {
  transform: scale3d(1.5, 2, 1);
}
  • Explicación: Este código escala el elemento a 1.5 veces su ancho, 2 veces su altura y 1 vez su profundidad.

Translate3d

La función translate3d mueve un elemento en el espacio 3D.

.element {
  transform: translate3d(50px, 100px, 200px);
}
  • Explicación: Este código mueve el elemento 50 píxeles a lo largo del eje X, 100 píxeles a lo largo del eje Y y 200 píxeles a lo largo del eje Z.

Combinando Transformaciones

Puedes combinar múltiples transformaciones usando una lista separada por espacios.

.element {
  transform: translate(50px, 100px) rotate(45deg) scale(1.5, 2);
}
  • Explicación: Este código mueve el elemento 50 píxeles a la derecha y 100 píxeles hacia abajo, lo rota 45 grados en el sentido de las agujas del reloj y lo escala a 1.5 veces su ancho y 2 veces su altura.

Origen de la Transformación

La propiedad transform-origin te permite cambiar el punto alrededor del cual se aplica una transformación.

.element {
  transform-origin: top left;
  transform: rotate(45deg);
}
  • Explicación: Este código establece el origen de la transformación en la esquina superior izquierda del elemento y luego lo rota 45 grados.

Ejemplos Prácticos

Ejemplo 1: Efecto Hover en Botón

Crea un botón que se escala ligeramente cuando se pasa el cursor sobre él.

.button {
  transition: transform 0.3s;
}

.button:hover {
  transform: scale(1.1);
}
  • Explicación: Este código escala el botón a 1.1 veces su tamaño original cuando se pasa el cursor sobre él, con un efecto de transición suave.

Ejemplo 2: Volteo de Tarjeta en 3D

Crea un efecto de volteo de tarjeta en 3D al pasar el cursor sobre ella.

.card {
  perspective: 1000px;
}

.card-inner {
  transition: transform 0.6s;
  transform-style: preserve-3d;
}

.card:hover .card-inner {
  transform: rotateY(180deg);
}

.card-front, .card-back {
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}
  • Explicación: Este código crea un efecto de volteo de tarjeta en 3D. Cuando se pasa el cursor sobre la tarjeta, esta rota 180 grados alrededor del eje Y, revelando el lado posterior.

Conclusión

Las transformaciones CSS son herramientas poderosas que te permiten manipular elementos tanto en el espacio 2D como en el 3D. Al dominar las funciones translate, rotate, scale y skew, junto con la comprensión de cómo combinar transformaciones y establecer orígenes de transformación, puedes crear diseños web dinámicos y visualmente atractivos. Practica con los ejemplos proporcionados y experimenta con diferentes combinaciones de transformaciones para mejorar tus habilidades en CSS.
