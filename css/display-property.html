Propiedad Display
Posicionamiento ►

La propiedad display en CSS es una de las propiedades más fundamentales y poderosas. Determina cómo se muestra un elemento en la página web. Comprender la propiedad display es crucial para crear diseños web bien estructurados y visualmente atractivos.

Conceptos Clave

  • Elementos de Nivel de Bloque: Estos elementos ocupan todo el ancho disponible y comienzan en una nueva línea.
  • Elementos en Línea: Estos elementos ocupan solo el ancho necesario y no comienzan en una nueva línea.
  • Elementos en Línea-Bloque: Estos elementos son como los elementos en línea pero pueden tener ancho y alto establecidos.
  • Ninguno: Este valor oculta el elemento; no se mostrará en la página.

Valores Comunes de Display

Bloque

Los elementos de nivel de bloque ocupan todo el ancho de su contenedor y comienzan en una nueva línea. Ejemplos incluyen <div>, <h1>, <p>, etc.

<div style="display: block;">
  Este es un elemento de nivel de bloque.
</div>

En Línea

Los elementos en línea ocupan solo el ancho necesario y no comienzan en una nueva línea. Ejemplos incluyen <span>, <a>, <strong>, etc.

<span style="display: inline;">
  Este es un elemento en línea.
</span>

En Línea-Bloque

Los elementos en línea-bloque son similares a los elementos en línea pero pueden tener ancho y alto establecidos.

<div style="display: inline-block; width: 100px; height: 50px; background-color: lightblue;">
  Este es un elemento en línea-bloque.
</div>

Ninguno

El valor none oculta el elemento; no se mostrará en la página.

<div style="display: none;">
  Este elemento está oculto.
</div>

Valores Avanzados de Display

Flex

El valor flex habilita un contenedor flexible, permitiendo diseños más complejos con diseño flexible y responsivo.

<div style="display: flex;">
  <div style="flex: 1;">Elemento Flex 1</div>
  <div style="flex: 2;">Elemento Flex 2</div>
</div>

Grid

El valor grid habilita un contenedor de cuadrícula, proporcionando un sistema poderoso para crear diseños complejos con filas y columnas.

<div style="display: grid; grid-template-columns: 1fr 1fr;">
  <div>Elemento de Cuadrícula 1</div>
  <div>Elemento de Cuadrícula 2</div>
</div>

Ejercicios

  1. Ejercicio Básico: Crea una página HTML simple con una mezcla de elementos de bloque, en línea y en línea-bloque. Experimenta con diferentes valores de display y observa los cambios.
  2. Ejercicio Intermedio: Crea un contenedor flexible con tres elementos. Ajusta las propiedades flex para ver cómo afectan el diseño.
  3. Ejercicio Avanzado: Crea un diseño de cuadrícula con múltiples filas y columnas. Experimenta con diferentes propiedades de cuadrícula para entender cómo funcionan juntas.

Conclusión

La propiedad display es una piedra angular de las técnicas de diseño en CSS. Al dominar sus diversos valores, desde los básicos bloque y en línea hasta los avanzados flex y grid, puedes crear diseños web sofisticados y responsivos. Practica usando estas propiedades en diferentes escenarios para volverte competente en la creación de páginas web bien estructuradas.
