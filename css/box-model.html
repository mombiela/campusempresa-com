Modelo de Caja
Márgenes, relleno y bordes ►

El Modelo de Caja de CSS es un concepto fundamental que define cómo se estructuran y muestran los elementos en una página web. Comprender el Modelo de Caja es crucial para un diseño y disposición efectivos en el desarrollo web.

Conceptos Clave

  • Contenido: El contenido real de la caja, donde aparecen texto e imágenes.
  • Relleno (Padding): El espacio entre el contenido y el borde. El relleno es transparente.
  • Borde: El área que rodea el relleno (si lo hay) y el contenido. Los bordes pueden tener diferentes estilos, anchos y colores.
  • Margen: El espacio fuera del borde. Los márgenes también son transparentes y se pueden usar para crear espacio entre elementos.

Propiedades CSS

Contenido

El área de contenido se define por las propiedades de ancho y alto.

div {
  width: 200px;
  height: 100px;
}

Relleno

El relleno agrega espacio dentro del elemento, alrededor del contenido.

div {
  padding: 20px; /* Agrega 20px de relleno en todos los lados */
}

También puedes especificar el relleno para cada lado individualmente:

div {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;
}

Borde

Los bordes rodean el relleno y el contenido. Puedes establecer el ancho, estilo y color del borde.

div {
  border: 2px solid black; /* Borde sólido negro de 2px de ancho */
}

También puedes especificar los bordes para cada lado individualmente:

div {
  border-top: 2px solid black;
  border-right: 2px solid red;
  border-bottom: 2px solid green;
  border-left: 2px solid blue;
}

Margen

Los márgenes crean espacio fuera del borde. También se pueden establecer para cada lado individualmente.

div {
  margin: 20px; /* Agrega 20px de margen en todos los lados */
}

Márgenes individuales:

div {
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;
}

Tamaño de Caja

Por defecto, el ancho y alto de un elemento solo incluyen el área de contenido. Para incluir el relleno y el borde en el ancho y alto total del elemento, usa la propiedad box-sizing.

div {
  box-sizing: border-box;
}

Ejemplo

Aquí tienes un ejemplo completo que demuestra el Modelo de Caja:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .box {
      width: 200px;
      height: 100px;
      padding: 20px;
      border: 5px solid black;
      margin: 30px;
      box-sizing: border-box;
    }
  </style>
</head>
<body>
  <div class="box">Este es un ejemplo del modelo de caja.</div>
</body>
</html>

Resumen

El Modelo de Caja de CSS es esencial para entender cómo se muestran y espacian los elementos en una página web. Consiste en el contenido, relleno, borde y margen. Al dominar el Modelo de Caja, puedes crear diseños más precisos y visualmente atractivos. Recuerda usar la propiedad box-sizing para controlar cómo se calculan el ancho y alto de los elementos, lo que puede simplificar tu diseño de disposición.
