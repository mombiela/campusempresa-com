Introducción a CSS

CSS (Hojas de Estilo en Cascada) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. CSS controla el diseño, los colores, las fuentes y la apariencia visual general de las páginas web.

Conceptos clave de CSS

Selectores

Los selectores se utilizan para apuntar a elementos HTML que deseas estilizar.

  • Selector de Elemento: Apunta a todas las instancias de un elemento HTML específico. 
    p {
  color: blue;
}
  • Selector de Clase: Apunta a elementos con un atributo de clase específico. 
    .example {
  color: red;
}
  • Selector de ID: Apunta a un solo elemento con un atributo de id específico. 
    #unique {
  color: green;
}
  • Selector de Atributo: Apunta a elementos con un atributo específico. 
    [type="text"] {
  border: 1px solid black;
}

Propiedades y Valores

Las propiedades de CSS se utilizan para definir los estilos aplicados a los elementos seleccionados.

  • Color: Establece el color del texto. 
    color: blue;
  • Tamaño de Fuente: Establece el tamaño del texto. 
    font-size: 16px;
  • Margen: Establece el espacio fuera del elemento. 
    margin: 10px;
  • Relleno: Establece el espacio dentro del elemento. 
    padding: 10px;

Cascada y Especificidad

CSS sigue un orden en cascada para determinar qué estilos se aplican cuando múltiples reglas coinciden con el mismo elemento.

  • Especificidad: Determina qué regla se aplica dando peso a diferentes tipos de selectores.
    • Los estilos en línea tienen la mayor especificidad.
    • Los selectores de ID tienen mayor especificidad que los selectores de clase.
    • Los selectores de clase tienen mayor especificidad que los selectores de elemento.
  • Herencia: Algunas propiedades se heredan de los elementos padre a los elementos hijo, como color y font-family.

Modelo de Caja

El modelo de caja de CSS describe los cuadros rectangulares generados para los elementos en el árbol del documento.

  • Contenido: El contenido real del elemento.
  • Relleno: Espacio entre el contenido y el borde.
  • Borde: El borde que rodea el relleno (si lo hay).
  • Margen: Espacio fuera del borde.
div {
  width: 100px;
  height: 100px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Posicionamiento

CSS proporciona varias formas de posicionar elementos en la página web.

  • Estático: Posicionamiento predeterminado, los elementos se posicionan según el flujo normal del documento.
  • Relativo: Posicionado en relación con su posición normal. 
    .relative {
  position: relative;
  top: 10px;
  left: 10px;
}
  • Absoluto: Posicionado en relación con el ancestro posicionado más cercano. 
    .absolute {
  position: absolute;
  top: 50px;
  left: 50px;
}
  • Fijo: Posicionado en relación con la ventana gráfica, permanece en el mismo lugar incluso cuando se desplaza la página. 
    .fixed {
  position: fixed;
  top: 0;
  left: 0;
}
  • Adherente: Cambia entre posicionamiento relativo y fijo dependiendo de la posición de desplazamiento. 
    .sticky {
  position: sticky;
  top: 0;
}

Conceptos avanzados de CSS

Flexbox

Flexbox es un modelo de diseño que permite la disposición de elementos de una manera predecible.

  • Propiedades del Contenedor:
    • display: flex;
    • flex-direction: row | column;
    • justify-content: flex-start | center | space-between;
    • align-items: flex-start | center | stretch;
  • Propiedades del Ítem:
    • flex-grow: 1;
    • flex-shrink: 1;
    • flex-basis: auto;
.container {
  display: flex;
  justify-content: space-between;
}

.item {
  flex-grow: 1;
}

Diseño de Cuadrícula

El diseño de cuadrícula de CSS es un sistema de diseño bidimensional para la web.

  • Propiedades del Contenedor:
    • display: grid;
    • grid-template-columns: repeat(3, 1fr);
    • grid-template-rows: auto;
    • gap: 10px;
  • Propiedades del Ítem:
    • grid-column: 1 / span 2;
    • grid-row: 1 / 2;
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  grid-column: 1 / span 2;
}

Consultas de Medios

Las consultas de medios te permiten aplicar estilos basados en las características del dispositivo, como su ancho.

@media (max-width: 600px) {
  .responsive {
    font-size: 12px;
  }
}

Conclusión

Entender cómo funciona CSS es fundamental para crear páginas web visualmente atractivas y responsivas. Al dominar los selectores, las propiedades, el modelo de caja, el posicionamiento, Flexbox, el diseño de cuadrícula y las consultas de medios, puedes controlar el diseño y la apariencia de tus proyectos web de manera efectiva. Practica estos conceptos con ejemplos del mundo real para solidificar tu conocimiento y convertirte en un experto en CSS.
