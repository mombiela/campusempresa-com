Introducción a las Consultas de Medios

Las Consultas de Medios son una parte fundamental del diseño web responsivo, permitiéndote aplicar reglas CSS basadas en las características del dispositivo que renderiza el contenido, como su ancho, altura u orientación.

Conceptos Clave

  • Diseño Responsivo: Crear páginas web que se vean bien en todos los dispositivos.
  • Viewport: El área visible de una página web en un dispositivo.
  • Puntos de Ruptura: Puntos específicos donde el diseño de la página web cambia para adaptarse a diferentes tamaños de pantalla.

Sintaxis Básica

Las Consultas de Medios usan la regla @media para aplicar estilos condicionalmente.

@media (condición) {
  /* Reglas CSS */
}

Ejemplo

/* Estilos por defecto */
body {
  background-color: white;
  color: black;
}

/* Estilos para pantallas más anchas de 600px */
@media (min-width: 600px) {
  body {
    background-color: lightblue;
  }
}

En este ejemplo:

  • El color de fondo por defecto es blanco.
  • Cuando el ancho de la pantalla es de 600px o más, el color de fondo cambia a azul claro.

Características Comunes de los Medios

  • Ancho y Altura: min-width, max-width, min-height, max-height
  • Orientación: orientation: portrait o orientation: landscape
  • Resolución: min-resolution, max-resolution

Ejemplo con Múltiples Condiciones

@media (min-width: 600px) and (orientation: landscape) {
  body {
    background-color: lightgreen;
  }
}

Esta regla se aplica cuando la pantalla tiene al menos 600px de ancho y está en orientación horizontal.

Usando Consultas de Medios para Diseño Responsivo

Puntos de Ruptura

Los puntos de ruptura son anchos de pantalla específicos donde el diseño cambia. Los puntos de ruptura comunes incluyen:

  • 480px (dispositivos pequeños)
  • 768px (tabletas)
  • 1024px (escritorios)

Ejemplo con Múltiples Puntos de Ruptura

/* Estilos por defecto */
body {
  font-size: 16px;
}

/* Dispositivos pequeños */
@media (max-width: 480px) {
  body {
    font-size: 14px;
  }
}

/* Tabletas */
@media (min-width: 481px) and (max-width: 768px) {
  body {
    font-size: 15px;
  }
}

/* Escritorios */
@media (min-width: 769px) {
  body {
    font-size: 18px;
  }
}

Consultas de Medios Avanzadas

Combinando Características de Medios

Puedes combinar múltiples características de medios usando operadores lógicos como and, not, y only.

Ejemplo con Operadores Lógicos

/* Aplica estilos solo si el dispositivo está en modo retrato y el ancho es al menos 600px */
@media only screen and (min-width: 600px) and (orientation: portrait) {
  body {
    background-color: lightcoral;
  }
}

Consultas de Medios para Pantallas de Alta Resolución

Las pantallas de alta resolución, como las pantallas Retina, requieren estilos diferentes para una mejor claridad.

@media (min-resolution: 2dppx) {
  body {
    font-size: 20px;
  }
}

Ejercicios Prácticos

  1. Consulta de Medios Básica: Crea una página web simple con un color de fondo que cambie cuando el ancho de la pantalla sea de 600px o más.
  2. Texto Responsivo: Ajusta el tamaño de fuente de un párrafo basado en diferentes anchos de pantalla (480px, 768px, 1024px).
  3. Cambio de Orientación: Cambia el diseño de una página web cuando la orientación del dispositivo cambie entre retrato y paisaje.

Conclusión

Las Consultas de Medios son una herramienta poderosa en CSS para crear diseños responsivos que se adapten a varios tamaños de pantalla y orientaciones. Al entender y usar las consultas de medios efectivamente, puedes asegurar que tus páginas web proporcionen una gran experiencia de usuario en todos los dispositivos. Practica con diferentes puntos de ruptura y condiciones para volverte competente en diseño web responsivo.
