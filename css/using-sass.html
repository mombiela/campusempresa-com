Introducción a Sass

Sass (Syntactically Awesome Style Sheets) es un preprocesador de CSS que añade poder y elegancia al lenguaje básico. Te permite usar características que aún no existen en CSS, como variables, reglas anidadas, mixins y funciones.

  • Preprocesador de CSS: Una herramienta que extiende CSS con variables, reglas anidadas y más.
  • Sintaxis: Sass tiene dos sintaxis: SCSS (Sassy CSS) y la sintaxis indentada.

Instalando Sass

Para empezar a usar Sass, necesitas instalarlo en tu máquina. La forma más fácil de instalar Sass es a través de npm (Node Package Manager).

npm install -g sass

Después de la instalación, puedes compilar Sass a CSS usando el siguiente comando:

sass input.scss output.css

Variables

Las variables en Sass te permiten almacenar valores (como colores, fuentes o cualquier valor de CSS) en una variable, que puedes reutilizar a lo largo de tu hoja de estilos.

$primary-color: #333;

body {
  color: $primary-color;
}

Anidación

Sass te permite anidar tus selectores de CSS de una manera que sigue la misma jerarquía visual de tu HTML.

nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }

  li { display: inline-block; }

  a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
  }
}

Parciales e Importación

Los parciales en Sass se usan para crear pequeños archivos CSS que puedes incluir en otros archivos Sass. Esto ayuda a organizar tu CSS.

  • Archivo Parcial: Un archivo Sass nombrado con un guion bajo al principio (por ejemplo, _partial.scss).
  • Importando Parciales: Usa la directiva @import para incluir parciales.
// _reset.scss
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

// styles.scss
@import 'reset';

body {
  font-family: Arial, sans-serif;
}

Mixins

Los mixins te permiten crear fragmentos reutilizables de CSS. Puedes pasar argumentos a los mixins para hacerlos más flexibles.

@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }

Funciones

Las funciones en Sass te permiten crear piezas reutilizables de código que devuelven un valor. Son similares a los mixins pero se usan para cálculos y devolver valores.

@function calculate-rem($size) {
  @return $size / 16px * 1rem;
}

body {
  font-size: calculate-rem(16px);
}

Extender/Heredar

La directiva @extend te permite compartir un conjunto de propiedades CSS de un selector a otro.

.message {
  border: 1px solid #ccc;
  padding: 10px;
  color: #333;
}

.success {
  @extend .message;
  border-color: green;
}

.error {
  @extend .message;
  border-color: red;
}

Directivas de Control y Expresiones

SassScript soporta directivas de control para escribir hojas de estilo complejas.

  • @if: Declaraciones condicionales.
  • @for: Bucle a través de un rango de números.
  • @each: Bucle a través de una lista o mapa.
  • @while: Bucle mientras una condición sea verdadera.
@for $i from 1 through 3 {
  .item-#{$i} { width: 2em * $i; }
}

$colors: (primary: #333, secondary: #666);

@each $name, $color in $colors {
  .#{$name}-color { color: $color; }
}

Conclusión

Sass es una herramienta poderosa que mejora las capacidades de CSS. Usando variables, anidación, mixins, funciones y directivas de control, puedes escribir hojas de estilo más mantenibles y escalables. A medida que te sientas más cómodo con Sass, descubrirás que mejora significativamente tu flujo de trabajo y la calidad de tu CSS.

Recuerda practicar convirtiendo algunos de tus proyectos CSS existentes a Sass y explora sus características más profundamente. ¡Feliz codificación!
