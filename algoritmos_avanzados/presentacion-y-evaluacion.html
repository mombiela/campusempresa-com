Presentación y Evaluación

Introducción

En esta sección, presentaremos los objetivos del curso y los criterios de evaluación. Este curso está diseñado para proporcionar una comprensión profunda de los algoritmos avanzados, incluyendo su diseño, análisis y aplicación en problemas complejos.

Objetivos del Curso

Al finalizar este curso, los estudiantes serán capaces de:

  • Comprender y aplicar algoritmos avanzados en la resolución de problemas complejos.
  • Analizar la eficiencia y complejidad de diferentes algoritmos.
  • Implementar algoritmos avanzados en un lenguaje de programación.
  • Evaluar y comparar diferentes enfoques algorítmicos para un mismo problema.

Criterios de Evaluación

La evaluación del curso se basará en los siguientes componentes:

  • Participación en clase y foros: 10%
  • Tareas y ejercicios prácticos: 40%
  • Exámenes parciales: 20%
  • Proyecto final: 30%

Ejemplos de Evaluación

Ejemplo de Tarea Práctica

Problema: Implementar el algoritmo de Dijkstra para encontrar el camino más corto en un grafo dirigido con pesos no negativos.

import heapq

def dijkstra(graph, start):
    # Inicialización
    queue = []
    heapq.heappush(queue, (0, start))
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    visited = set()

    while queue:
        current_distance, current_vertex = heapq.heappop(queue)

        if current_vertex in visited:
            continue

        visited.add(current_vertex)

        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))

    return distances

# Ejemplo de uso
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

print(dijkstra(graph, 'A'))

Explicación:

  • Se utiliza una cola de prioridad (heapq) para seleccionar el vértice con la distancia más corta.
  • Se inicializan las distancias de todos los vértices a infinito, excepto el vértice de inicio.
  • Se actualizan las distancias a medida que se exploran los vecinos de cada vértice.

Ejemplo de Proyecto Final

Descripción: Desarrollar un sistema de recomendación basado en el algoritmo de filtrado colaborativo. El proyecto debe incluir:

  • Análisis del problema y diseño del algoritmo.
  • Implementación del algoritmo en Python.
  • Evaluación del rendimiento del algoritmo utilizando un conjunto de datos real.
  • Comparación con otros algoritmos de recomendación.

Criterios de Evaluación:

  • Correctitud del algoritmo: 30%
  • Eficiencia del código: 20%
  • Calidad del análisis y diseño: 20%
  • Documentación y presentación: 30%

Conclusión

La sección de presentación y evaluación establece las expectativas y los objetivos del curso, proporcionando una guía clara sobre cómo se evaluará el progreso de los estudiantes. A través de tareas prácticas y un proyecto final, los estudiantes tendrán la oportunidad de aplicar los conceptos aprendidos y demostrar su comprensión de los algoritmos avanzados.
