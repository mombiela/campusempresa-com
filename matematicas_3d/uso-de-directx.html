Introducción a DirectX

DirectX es una colección de APIs (Interfaces de Programación de Aplicaciones) desarrolladas por Microsoft para manejar tareas relacionadas con multimedia, especialmente programación de juegos y video, en plataformas Microsoft. En el contexto de las matemáticas 3D, DirectX proporciona herramientas esenciales para la creación y manipulación de gráficos tridimensionales.

Conceptos Clave de DirectX

  • Direct3D: Es la API de DirectX utilizada para renderizar gráficos en 3D. Es fundamental para la creación de aplicaciones gráficas y juegos.
  • Shaders: Programas que se ejecutan en la GPU para realizar operaciones de gráficos avanzadas.
  • Buffers: Áreas de memoria que almacenan datos que la GPU necesita para renderizar gráficos.
  • Matrices de Transformación: Utilizadas para realizar operaciones de transformación en objetos 3D, como traslación, rotación y escalado.

Instalación y Configuración de DirectX

Para comenzar a usar DirectX, es necesario instalar el SDK de DirectX y configurar el entorno de desarrollo. Aquí hay un ejemplo de cómo hacerlo en Visual Studio:

  1. Instalar el SDK de DirectX: Descargue e instale el SDK de DirectX desde el sitio web de Microsoft.
  2. Configurar Visual Studio:
    • Abra Visual Studio y cree un nuevo proyecto.
    • Vaya a Project > Properties.
    • En la sección VC++ Directories, agregue las rutas de los directorios de inclusión y bibliotecas del SDK de DirectX.
    • En la sección Linker > Input, agregue las bibliotecas de DirectX necesarias (por ejemplo, d3d11.lib).

Creación de una Ventana Básica con DirectX

A continuación, se presenta un ejemplo de código para crear una ventana básica utilizando DirectX:

#include <windows.h>
#include <d3d11.h>

// Variables globales
HINSTANCE hInstance;
HWND hwnd;
ID3D11Device* device = nullptr;
ID3D11DeviceContext* deviceContext = nullptr;
IDXGISwapChain* swapChain = nullptr;
ID3D11RenderTargetView* renderTargetView = nullptr;

// Prototipos de funciones
LRESULT CALLBACK WndProc(HWND, UINT, WPARAM, LPARAM);
void InitD3D(HWND hwnd);
void RenderFrame();
void CleanD3D();

int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow) {
    // Registro de la clase de ventana
    WNDCLASSEX wc;
    ZeroMemory(&wc, sizeof(WNDCLASSEX));
    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WndProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = "WindowClass";
    RegisterClassEx(&wc);

    // Creación de la ventana
    hwnd = CreateWindowEx(NULL, "WindowClass", "DirectX Window", WS_OVERLAPPEDWINDOW, 100, 100, 800, 600, NULL, NULL, hInstance, NULL);
    ShowWindow(hwnd, nCmdShow);

    // Inicialización de Direct3D
    InitD3D(hwnd);

    // Bucle de mensajes
    MSG msg;
    while (TRUE) {
        if (PeekMessage(&msg, NULL, 0, 0, PM_REMOVE)) {
            if (msg.message == WM_QUIT) break;
            TranslateMessage(&msg);
            DispatchMessage(&msg);
        } else {
            RenderFrame();
        }
    }

    // Limpieza de Direct3D
    CleanD3D();

    return msg.wParam;
}

LRESULT CALLBACK WndProc(HWND hwnd, UINT msg, WPARAM wParam, LPARAM lParam) {
    switch (msg) {
        case WM_DESTROY:
            PostQuitMessage(0);
            return 0;
    }
    return DefWindowProc(hwnd, msg, wParam, lParam);
}

void InitD3D(HWND hwnd) {
    // Creación del dispositivo y el contexto de dispositivo
    D3D_FEATURE_LEVEL featureLevel;
    D3D11CreateDevice(NULL, D3D_DRIVER_TYPE_HARDWARE, NULL, 0, NULL, 0, D3D11_SDK_VERSION, &device, &featureLevel, &deviceContext);

    // Creación de la cadena de intercambio
    DXGI_SWAP_CHAIN_DESC scd;
    ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));
    scd.BufferCount = 1;
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;
    scd.OutputWindow = hwnd;
    scd.SampleDesc.Count = 4;
    scd.Windowed = TRUE;
    IDXGIFactory* factory;
    CreateDXGIFactory(__uuidof(IDXGIFactory), (void**)&factory);
    factory->CreateSwapChain(device, &scd, &swapChain);

    // Creación de la vista de destino de renderizado
    ID3D11Texture2D* backBuffer;
    swapChain->GetBuffer(0, __uuidof(ID3D11Texture2D), (void**)&backBuffer);
    device->CreateRenderTargetView(backBuffer, NULL, &renderTargetView);
    backBuffer->Release();
    deviceContext->OMSetRenderTargets(1, &renderTargetView, NULL);
}

void RenderFrame() {
    // Limpieza del buffer
    float color[4] = {0.0f, 0.2f, 0.4f, 1.0f};
    deviceContext->ClearRenderTargetView(renderTargetView, color);

    // Presentación del buffer
    swapChain->Present(0, 0);
}

void CleanD3D() {
    swapChain->Release();
    renderTargetView->Release();
    deviceContext->Release();
    device->Release();
}

Transformaciones en 3D con DirectX

Las transformaciones en 3D son esenciales para manipular objetos en el espacio tridimensional. DirectX utiliza matrices para realizar estas transformaciones:

  • Traslación: Mueve un objeto de un lugar a otro.
  • Rotación: Gira un objeto alrededor de un eje.
  • Escalado: Cambia el tamaño de un objeto.

Ejemplo de código para aplicar una transformación de traslación:

#include <DirectXMath.h>

using namespace DirectX;

XMMATRIX translationMatrix = XMMatrixTranslation(3.0f, 4.0f, 5.0f);

Conclusión

DirectX es una herramienta poderosa para la creación de gráficos en 3D, proporcionando una amplia gama de funcionalidades para manejar gráficos y multimedia. En este tema, hemos cubierto los conceptos básicos de DirectX, cómo configurar el entorno de desarrollo, y cómo crear una ventana básica y aplicar transformaciones en 3D. Con esta base, los estudiantes pueden explorar más a fondo las capacidades de DirectX y su aplicación en proyectos más complejos.
