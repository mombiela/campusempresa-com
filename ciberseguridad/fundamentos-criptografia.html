Introducción a la Criptografía

La criptografía es una disciplina fundamental en el campo de la ciberseguridad que se encarga de proteger la información mediante técnicas matemáticas. Su objetivo principal es garantizar la confidencialidad, integridad y autenticidad de los datos.

Conceptos Clave

  • Confidencialidad: Garantiza que la información solo sea accesible para las personas autorizadas.
  • Integridad: Asegura que la información no ha sido alterada de manera no autorizada.
  • Autenticidad: Verifica la identidad de las partes involucradas en la comunicación.
  • No repudio: Previene que una parte niegue haber realizado una acción.

Tipos de Criptografía

Criptografía Simétrica

En la criptografía simétrica, la misma clave se utiliza tanto para cifrar como para descifrar la información.

  • Ventajas:
    • Rápida y eficiente.
    • Menor consumo de recursos.
  • Desventajas:
    • Distribución de claves segura es un desafío.
    • No es escalable para grandes sistemas.

Ejemplo de Criptografía Simétrica

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clave de 16 bytes
key = get_random_bytes(16)

# Crear un objeto de cifrado AES
cipher = AES.new(key, AES.MODE_EAX)

# Mensaje a cifrar
data = b"Mensaje secreto"

# Cifrar el mensaje
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(data)

print("Mensaje cifrado:", ciphertext)

Criptografía Asimétrica

La criptografía asimétrica utiliza un par de claves: una clave pública para cifrar y una clave privada para descifrar.

  • Ventajas:
    • Facilita la distribución de claves.
    • Escalable para grandes sistemas.
  • Desventajas:
    • Más lenta y consume más recursos.
    • Complejidad en la gestión de claves.

Ejemplo de Criptografía Asimétrica

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generar un par de claves RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objeto de cifrado RSA
cipher_rsa = PKCS1_OAEP.new(public_key)

# Mensaje a cifrar
data = b"Mensaje secreto"

# Cifrar el mensaje
ciphertext = cipher_rsa.encrypt(data)

print("Mensaje cifrado:", ciphertext)

Algoritmos Criptográficos Comunes

| Algoritmo | Tipo | Uso Principal | |----------------|-------------|---------------------| | AES | Simétrico | Cifrado de datos | | DES | Simétrico | Cifrado de datos | | RSA | Asimétrico | Cifrado y firmas | | ECC | Asimétrico | Cifrado y firmas | | SHA-256 | Hash | Integridad de datos |

Aplicaciones de la Criptografía

  • Cifrado de datos: Protege la información almacenada y transmitida.
  • Firmas digitales: Verifica la autenticidad y la integridad de los mensajes.
  • Certificados digitales: Autentica la identidad de las entidades en una red.
  • Protocolos seguros: SSL/TLS para comunicaciones seguras en la web.

Conclusión

La criptografía es una herramienta esencial en la ciberseguridad que permite proteger la información contra accesos no autorizados y alteraciones. Comprender los fundamentos de la criptografía, incluyendo los tipos de criptografía y los algoritmos comunes, es crucial para cualquier profesional de TI que desee asegurar la integridad y confidencialidad de los datos en un entorno digital.

Este tema proporciona una base sólida para entender cómo la criptografía se aplica en la ciberseguridad. En los siguientes módulos, profundizaremos en técnicas avanzadas y casos de uso específicos para fortalecer aún más tus habilidades en este campo.
