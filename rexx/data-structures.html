Introducción a las Estructuras de Datos

Las estructuras de datos son esenciales para organizar y gestionar datos de manera eficiente. En REXX, entender cómo usar y manipular varias estructuras de datos te ayudará a escribir programas más efectivos y eficientes.

Tipos de Datos Básicos

Antes de sumergirse en estructuras de datos complejas, es importante entender los tipos de datos básicos en REXX:

  • Cadenas: El tipo de dato más común en REXX, usado para texto y números.
  • Números: REXX trata los números como cadenas pero permite operaciones aritméticas.

Ejemplo: Tipos de Datos Básicos

/* Ejemplo de Cadena */
name = "John Doe"
say "Nombre: " name

/* Ejemplo de Número */
age = 30
say "Edad: " age

Arreglos

Los arreglos en REXX son arreglos asociativos, también conocidos como variables de tallo. Te permiten almacenar y recuperar datos usando claves.

Creación y Uso de Arreglos

  • Definir un Arreglo: Usa una variable de tallo seguida de un punto.
  • Asignar Valores: Usa la variable de tallo con un índice.
  • Acceder a Valores: Usa el índice para recuperar el valor.

Ejemplo: Arreglos

/* Definir un arreglo */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Acceder a elementos del arreglo */
say "Estudiante 1: " students.1
say "Estudiante 2: " students.2
say "Estudiante 3: " students.3

Listas

REXX no tiene estructuras de datos de listas incorporadas, pero puedes simular listas usando arreglos o cadenas.

Ejemplo: Listas Usando Arreglos

/* Definir una lista usando un arreglo */
list.1 = "apple"
list.2 = "banana"
list.3 = "cherry"

/* Iterar sobre la lista */
do i = 1 to 3
    say "Ítem " i ": " list.i
end

Ejemplo: Listas Usando Cadenas

/* Definir una lista usando una cadena */
list = "apple banana cherry"

/* Dividir la cadena en palabras */
parse var list item1 item2 item3

say "Ítem 1: " item1
say "Ítem 2: " item2
say "Ítem 3: " item3

Pilas y Colas

Las pilas y colas pueden ser implementadas usando arreglos o listas. Una pila sigue el orden Último en Entrar, Primero en Salir (LIFO), mientras que una cola sigue el orden Primero en Entrar, Primero en Salir (FIFO).

Ejemplo: Implementación de Pila

/* Definir una pila */
stack.1 = "first"
stack.2 = "second"
stack.3 = "third"

/* Extraer de la pila */
do i = 3 to 1 by -1
    say "Extraído: " stack.i
end

Ejemplo: Implementación de Cola

/* Definir una cola */
queue.1 = "first"
queue.2 = "second"
queue.3 = "third"

/* Desencolar */
do i = 1 to 3
    say "Desencolado: " queue.i
end

Tablas Hash

Los arreglos asociativos de REXX pueden ser usados para crear tablas hash, donde cada clave se asigna a un valor.

Ejemplo: Tabla Hash

/* Definir una tabla hash */
hashTable.["name"] = "John Doe"
hashTable.["age"] = 30
hashTable.["city"] = "New York"

/* Acceder a elementos de la tabla hash */
say "Nombre: " hashTable.["name"]
say "Edad: " hashTable.["age"]
say "Ciudad: " hashTable.["city"]

Estructuras de Datos Avanzadas

Para estructuras de datos más complejas, puede que necesites combinar estructuras básicas o usar bibliotecas externas.

Ejemplo: Arreglos Anidados

/* Definir un arreglo anidado */
students.1.name = "Alice"
students.1.age = 20
students.2.name = "Bob"
students.2.age = 22

/* Acceder a elementos del arreglo anidado */
say "Estudiante 1: " students.1.name ", Edad: " students.1.age
say "Estudiante 2: " students.2.name ", Edad: " students.2.age

Conclusión

Entender y utilizar estructuras de datos en REXX es crucial para la programación eficiente. Desde arreglos básicos hasta estructuras anidadas más complejas, dominar estos conceptos mejorará tu capacidad para gestionar y manipular datos de manera efectiva. Practica con los ejemplos proporcionados y explora más para profundizar tu conocimiento.
