Introducción

Leer archivos es una habilidad fundamental en la programación, permitiéndote procesar y analizar datos almacenados en archivos externos. En esta sección, cubriremos cómo leer archivos usando REXX, comenzando desde lo básico y avanzando hacia técnicas más avanzadas.

Lectura Básica de Archivos

Abrir un Archivo

Para leer un archivo en REXX, primero necesitas abrirlo. Esto se hace usando la función OPEN.

/* Abrir un archivo para lectura */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error al abrir el archivo'
ELSE
  SAY 'Archivo abierto con éxito'
  • file: El nombre del archivo que deseas abrir.
  • file_handle: Un manejador que representa el archivo abierto. Si el archivo no se puede abrir, file_handle será 0.

Leer Líneas de un Archivo

Una vez que el archivo está abierto, puedes leer su contenido línea por línea usando la función LINEIN.

/* Leer líneas del archivo */
DO WHILE LINES(file_handle) > 0
  line = LINEIN(file_handle)
  SAY line
END
  • LINES(file_handle): Devuelve el número de líneas restantes en el archivo.
  • LINEIN(file_handle): Lee la siguiente línea del archivo.

Cerrar un Archivo

Después de terminar de leer el archivo, es importante cerrarlo usando la función CLOSE.

/* Cerrar el archivo */
CALL CLOSE file_handle

Lectura Avanzada de Archivos

Leer Líneas Específicas

Puedes leer líneas específicas de un archivo especificando el número de línea en la función LINEIN.

/* Leer la quinta línea del archivo */
line = LINEIN(file_handle, 5)
SAY line

Leer con Manejo de Errores

Es una buena práctica incluir manejo de errores al trabajar con archivos para gestionar problemas inesperados.

/* Abrir un archivo con manejo de errores */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error al abrir el archivo'
ELSE DO
  /* Leer líneas con manejo de errores */
  DO WHILE LINES(file_handle) > 0
    line = LINEIN(file_handle)
    IF RC \= 0 THEN
      SAY 'Error al leer la línea'
    ELSE
      SAY line
  END
  /* Cerrar el archivo */
  CALL CLOSE file_handle
END

Leer Archivos Grandes

Para archivos grandes, podrías querer leer fragmentos de datos en lugar de línea por línea para mejorar el rendimiento.

/* Leer fragmentos de datos del archivo */
DO WHILE LINES(file_handle) > 0
  chunk = CHARIN(file_handle, 1, 1024) /* Leer 1024 caracteres */
  SAY chunk
END
  • CHARIN(file_handle, start, length): Lee un número especificado de caracteres comenzando desde una posición dada.

Resumen

En esta sección, cubrimos lo básico de la lectura de archivos en REXX, incluyendo abrir, leer y cerrar archivos. También exploramos técnicas avanzadas como leer líneas específicas, manejo de errores y leer archivos grandes en fragmentos. Dominar estas habilidades te permitirá procesar y analizar eficientemente datos almacenados en archivos externos usando REXX.
