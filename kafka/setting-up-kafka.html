Configuración de Kafka
Temas y Particiones de Kafka ►

Configurar Kafka implica varios pasos, desde descargar e instalar Kafka hasta configurarlo según tus necesidades específicas. Esta sección te guiará a través del proceso, asegurando que tengas un entorno de Kafka funcional al final.

Requisitos previos

Antes de configurar Kafka, asegúrate de tener lo siguiente:

  • Java Development Kit (JDK): Kafka requiere Java para ejecutarse. Asegúrate de tener instalado JDK 8 o una versión posterior.
  • Zookeeper: Kafka utiliza Zookeeper para gestionar su clúster. Zookeeper debe estar instalado y en funcionamiento antes de iniciar Kafka.

Descargando Kafka

  1. Visita la página de descargas de Kafka: Ve a la página de descargas de Apache Kafka.
  2. Selecciona una versión: Elige la versión de Kafka que deseas descargar. Se recomienda usar la última versión estable.
  3. Descarga el binario: Descarga los archivos binarios para tu sistema operativo.

Instalando Kafka

  1. Extrae el archivo: Una vez descargado, extrae el archivo de Kafka en el directorio deseado.

    tar -xzf kafka_2.13-2.8.0.tgz
cd kafka_2.13-2.8.0

  2. Estructura del directorio: Después de la extracción, deberías ver la siguiente estructura de directorios:

    kafka_2.13-2.8.0/
├── bin
├── config
├── libs
├── logs
└── site-docs

Configurando Zookeeper

Kafka requiere Zookeeper para gestionar su clúster. Zookeeper está incluido en la distribución de Kafka.

  1. Inicia Zookeeper: Usa el script proporcionado para iniciar Zookeeper.

    bin/zookeeper-server-start.sh config/zookeeper.properties

  2. Verifica Zookeeper: Asegúrate de que Zookeeper esté en funcionamiento revisando los registros o usando un cliente de Zookeeper.

Configurando Kafka

  1. Edita la configuración de Kafka: Abre el archivo config/server.properties para configurar Kafka.

    • Broker ID: Identificador único para cada broker de Kafka. 
      broker.id=0
    • Directorio de registros: Directorio donde Kafka almacenará sus registros. 
      log.dirs=/tmp/kafka-logs
    • Conexión a Zookeeper: Dirección de la instancia de Zookeeper. 
      zookeeper.connect=localhost:2181

  2. Inicia Kafka: Usa el script proporcionado para iniciar Kafka.

    bin/kafka-server-start.sh config/server.properties

Verificando la instalación

  1. Crea un tema: Usa el script de Kafka para crear un nuevo tema.

    bin/kafka-topics.sh --create --topic test --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1

  2. Lista los temas: Verifica que el tema fue creado.

    bin/kafka-topics.sh --list --bootstrap-server localhost:9092

  3. Envía mensajes: Usa el productor de consola de Kafka para enviar mensajes al tema.

    bin/kafka-console-producer.sh --topic test --bootstrap-server localhost:9092
>Hola Kafka
>Este es un mensaje de prueba

  4. Consume mensajes: Usa el consumidor de consola de Kafka para leer mensajes del tema.

    bin/kafka-console-consumer.sh --topic test --from-beginning --bootstrap-server localhost:9092

Resumen

Configurar Kafka implica descargar los binarios de Kafka, configurar Zookeeper e iniciar tanto los servidores de Zookeeper como de Kafka. Siguiendo los pasos descritos anteriormente, deberías tener un entorno de Kafka funcional. Esta configuración te permite crear temas, producir mensajes y consumir mensajes, formando la base para operaciones más avanzadas con Kafka.
