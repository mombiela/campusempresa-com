Introducción a los Operadores

Los operadores son componentes fundamentales en Perl que te permiten realizar varias operaciones en variables y valores. Entender los operadores es crucial para escribir código Perl efectivo y eficiente.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

  • Adición (+): Suma dos números.
  • Sustracción (-): Resta el segundo número del primero.
  • Multiplicación (*): Multiplica dos números.
  • División (/): Divide el primer número por el segundo.
  • Módulo (%): Devuelve el resto de la división del primer número por el segundo.
  • Exponenciación (**): Eleva el primer número a la potencia del segundo.

Ejemplo

my $a = 10;
my $b = 3;

print $a + $b;  # Salida: 13
print $a - $b;  # Salida: 7
print $a * $b;  # Salida: 30
print $a / $b;  # Salida: 3.33333333333333
print $a % $b;  # Salida: 1
print $a ** $b; # Salida: 1000

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores.

  • Comparación Numérica:

    • Igual (==)
    • No igual (!=)
    • Mayor que (>)
    • Menor que (<)
    • Mayor o igual que (>=)
    • Menor o igual que (<=)

  • Comparación de Cadenas:

    • Igual (eq)
    • No igual (ne)
    • Mayor que (gt)
    • Menor que (lt)
    • Mayor o igual que (ge)
    • Menor o igual que (le)

Ejemplo

my $num1 = 5;
my $num2 = 10;
my $str1 = "apple";
my $str2 = "banana";

print $num1 == $num2;  # Salida: (vacío, falso)
print $num1 != $num2;  # Salida: 1 (verdadero)
print $str1 eq $str2;  # Salida: (vacío, falso)
print $str1 lt $str2;  # Salida: 1 (verdadero)

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

  • Y (&&): Devuelve verdadero si ambas condiciones son verdaderas.
  • O (||): Devuelve verdadero si al menos una condición es verdadera.
  • No (!): Invierte el estado lógico de su operando.

Ejemplo

my $x = 5;
my $y = 10;

print ($x < 10 && $y > 5);  # Salida: 1 (verdadero)
print ($x > 10 || $y > 5);  # Salida: 1 (verdadero)
print !($x == $y);          # Salida: 1 (verdadero)

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. También pueden realizar operaciones y asignar el resultado a la variable.

  • Asignación Simple (=)
  • Asignación de Adición (+=)
  • Asignación de Sustracción (-=)
  • Asignación de Multiplicación (*=)
  • Asignación de División (/=)
  • Asignación de Módulo (%=)
  • Asignación de Exponenciación (**=)

Ejemplo

my $a = 5;
$a += 3;  # Equivalente a $a = $a + 3; Salida: 8
$a -= 2;  # Equivalente a $a = $a - 2; Salida: 6
$a *= 4;  # Equivalente a $a = $a * 4; Salida: 24
$a /= 2;  # Equivalente a $a = $a / 2; Salida: 12
$a %= 5;  # Equivalente a $a = $a % 5; Salida: 2
$a **= 3; # Equivalente a $a = $a ** 3; Salida: 8

Operadores de Cadenas

Los operadores de cadenas se utilizan para manipular cadenas.

  • Concatenación (.): Une dos cadenas.
  • Repetición (x): Repite una cadena un número especificado de veces.

Ejemplo

my $str1 = "Hello";
my $str2 = "World";

print $str1 . " " . $str2;  # Salida: Hello World
print $str1 x 3;            # Salida: HelloHelloHello

Operador Condicional (Operador Ternario)

El operador condicional es una forma abreviada de una declaración if-else.

  • Sintaxis: condition ? true_value : false_value

Ejemplo

my $age = 18;
my $status = ($age >= 18) ? "adult" : "minor";

print $status;  # Salida: adult

Operador de Rango

El operador de rango se utiliza para crear una lista de valores.

  • Sintaxis: ..

Ejemplo

my @numbers = (1..5);  # Salida: (1, 2, 3, 4, 5)
my @letters = ('a'..'e');  # Salida: ('a', 'b', 'c', 'd', 'e')

Resumen

Los operadores en Perl son herramientas esenciales para realizar una amplia variedad de tareas, desde cálculos aritméticos hasta manipulaciones de cadenas y comparaciones lógicas. Al dominar estos operadores, puedes escribir código Perl más conciso y eficiente. Practica usando estos operadores en diferentes contextos para familiarizarte más con su sintaxis y funcionalidad.
