Introducción a los Arrays

Los arrays son listas ordenadas de escalares en Perl. Son una de las estructuras de datos fundamentales y se utilizan para almacenar múltiples valores en una sola variable.

  • Los arrays se denotan con el símbolo @.
  • Cada elemento en un array puede ser accedido usando un índice, comenzando desde 0.

Declaración e Inicialización de Arrays

Para declarar e inicializar un array, puedes usar la siguiente sintaxis:

# Declarando un array
my @fruits;

# Inicializando un array
my @fruits = ('apple', 'banana', 'cherry');

Acceso a Elementos del Array

Puedes acceder a elementos individuales de un array usando su índice:

# Accediendo al primer elemento
print $fruits[0]; # Salida: apple

# Accediendo al segundo elemento
print $fruits[1]; # Salida: banana

Modificación de Elementos del Array

Puedes modificar elementos en un array asignando nuevos valores a índices específicos:

# Cambiando el segundo elemento
$fruits[1] = 'blueberry';
print $fruits[1]; # Salida: blueberry

Añadir y Eliminar Elementos

Perl proporciona varias funciones para añadir o eliminar elementos de los arrays:

  • push: Añade elementos al final de un array.
  • pop: Elimina el último elemento de un array.
  • unshift: Añade elementos al principio de un array.
  • shift: Elimina el primer elemento de un array.
# Añadiendo elementos al final
push(@fruits, 'date');
print "@fruits"; # Salida: apple blueberry cherry date

# Eliminando el último elemento
pop(@fruits);
print "@fruits"; # Salida: apple blueberry cherry

# Añadiendo elementos al principio
unshift(@fruits, 'apricot');
print "@fruits"; # Salida: apricot apple blueberry cherry

# Eliminando el primer elemento
shift(@fruits);
print "@fruits"; # Salida: apple blueberry cherry

Funciones y Operaciones de Arrays

Perl proporciona una variedad de funciones para manipular arrays:

  • scalar @array: Devuelve el número de elementos en el array.
  • @sorted = sort @array: Ordena el array.
  • @reversed = reverse @array: Invierte el array.
# Número de elementos en el array
my $count = scalar @fruits;
print $count; # Salida: 3

# Ordenando el array
my @sorted_fruits = sort @fruits;
print "@sorted_fruits"; # Salida: apple blueberry cherry

# Invirtiendo el array
my @reversed_fruits = reverse @fruits;
print "@reversed_fruits"; # Salida: cherry blueberry apple

Slicing de Arrays

El slicing de arrays te permite acceder a múltiples elementos de un array a la vez:

# Slicing del array
my @slice = @fruits[0, 2];
print "@slice"; # Salida: apple cherry

Arrays Multidimensionales

Perl no soporta arrays multidimensionales verdaderos, pero puedes crear arrays de arrays:

# Creando un array multidimensional
my @matrix = (
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
);

# Accediendo a elementos en un array multidimensional
print $matrix[1][2]; # Salida: 6

Técnicas Avanzadas de Arrays

Referencias de Arrays

Las referencias de arrays te permiten crear estructuras de datos complejas como arrays de arrays o arrays de hashes:

# Creando una referencia de array
my $array_ref = \@fruits;

# Desreferenciando una referencia de array
print @$array_ref; # Salida: apple blueberry cherry

Uso de `map` y `grep`

  • map: Aplica un bloque de código a cada elemento de un array y devuelve un nuevo array con los resultados.
  • grep: Filtra elementos de un array basado en una condición.
# Usando map para transformar elementos
my @lengths = map { length($_) } @fruits;
print "@lengths"; # Salida: 5 9 6

# Usando grep para filtrar elementos
my @filtered = grep { length($_) > 5 } @fruits;
print "@filtered"; # Salida: blueberry cherry

Conclusión

Los arrays en Perl son herramientas versátiles y poderosas para gestionar listas de datos. Desde operaciones básicas como añadir y eliminar elementos hasta técnicas avanzadas como el uso de referencias y funciones de orden superior, entender los arrays es crucial para una programación efectiva en Perl. Practica usando arrays con diferentes funciones y operaciones para volverte competente en el manejo de estructuras de datos complejas.
