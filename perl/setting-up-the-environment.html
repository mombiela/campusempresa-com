Configurar tu entorno de desarrollo es el primer paso para comenzar a programar en Perl. Esta sección te guiará a través del proceso de instalación de Perl, configuración de un editor de texto o Entorno de Desarrollo Integrado (IDE), y verificación de tu instalación.

Instalación de Perl

Perl es un lenguaje altamente portátil y se puede instalar en varios sistemas operativos, incluyendo Windows, macOS y Linux.

Windows

  1. Descargar Strawberry Perl:

    • Visita el sitio web de Strawberry Perl.
    • Descarga la última versión de Strawberry Perl.

  2. Instalar Strawberry Perl:

    • Ejecuta el instalador descargado.
    • Sigue las indicaciones de instalación y acepta la configuración predeterminada.

  3. Verificar la Instalación:

    • Abre el Símbolo del sistema.
    • Escribe perl -v y presiona Enter.
    • Deberías ver la información de la versión de Perl.

macOS

  1. Usando Homebrew:

    • Abre Terminal.
    • Instala Homebrew si aún no lo has hecho: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instala Perl usando Homebrew: 
      brew install perl

  2. Verificar la Instalación:

    • En Terminal, escribe perl -v y presiona Enter.
    • Deberías ver la información de la versión de Perl.

Linux

  1. Usando el Gestor de Paquetes:

    • Abre Terminal.
    • Para sistemas basados en Debian (por ejemplo, Ubuntu): 
      sudo apt-get update
sudo apt-get install perl
    • Para sistemas basados en Red Hat (por ejemplo, Fedora): 
      sudo dnf install perl

  2. Verificar la Instalación:

    • En Terminal, escribe perl -v y presiona Enter.
    • Deberías ver la información de la versión de Perl.

Configuración de un Editor de Texto o IDE

Elegir el editor de texto o IDE adecuado puede mejorar significativamente tu productividad. Aquí hay algunas opciones populares:

Editores de Texto

  • Notepad++ (Windows):

    • Descarga desde el sitio web de Notepad++.
    • Soporta resaltado de sintaxis para Perl.

  • Sublime Text (Multiplataforma):

    • Descarga desde el sitio web de Sublime Text.
    • Instala el paquete de resaltado de sintaxis para Perl.

  • Vim (Multiplataforma):

    • Disponible por defecto en la mayoría de los sistemas basados en Unix.
    • Instala a través del gestor de paquetes si no está ya instalado.

Entornos de Desarrollo Integrados (IDEs)

  • Komodo IDE:

    • Descarga desde el sitio web de Komodo IDE.
    • Específicamente diseñado para lenguajes dinámicos como Perl.

  • Eclipse con el Plugin EPIC:

    • Descarga Eclipse desde el sitio web de Eclipse.
    • Instala el plugin EPIC para soporte de Perl.

Verificación de tu Configuración

Para asegurarte de que todo está configurado correctamente, puedes escribir y ejecutar un simple script en Perl.

  1. Crear un Script en Perl:

    • Abre tu editor de texto o IDE.
    • Escribe el siguiente código: 
      #!/usr/bin/perl
use strict;
use warnings;
print "¡Hola, Mundo!\n";

  2. Guardar el Script:

    • Guarda el archivo con una extensión .pl, por ejemplo, hola_mundo.pl.

  3. Ejecutar el Script:

    • Abre el Símbolo del sistema o Terminal.
    • Navega al directorio donde guardaste el script.
    • Ejecuta el script escribiendo: 
      perl hola_mundo.pl
    • Deberías ver la salida: ¡Hola, Mundo!

Conclusión

Configurar el entorno de Perl implica instalar Perl, elegir un editor de texto o IDE adecuado, y verificar la instalación ejecutando un script simple. Esta configuración básica asegura que estés listo para comenzar a programar en Perl de manera eficiente. En las próximas secciones, profundizaremos en los conceptos y prácticas de programación en Perl.
