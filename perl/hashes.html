Introducción a los Hashes

Los hashes, también conocidos como arreglos asociativos, son una estructura de datos fundamental en Perl. Te permiten almacenar datos en pares clave-valor, facilitando la búsqueda de valores basados en una clave única.

  • Pares Clave-Valor: Cada elemento en un hash es un par clave-valor.
  • Claves Únicas: Las claves deben ser únicas dentro de un hash.
  • Desordenados: Los hashes no mantienen el orden de los elementos.

Creación y Acceso a Hashes

Creación de un Hash

Puedes crear un hash usando la palabra clave my y una lista de pares clave-valor.

my %colores_frutas = (
    manzana  => 'rojo',
    banana => 'amarillo',
    uva  => 'morado',
);

Acceso a Elementos del Hash

Accede a los elementos de un hash usando la clave dentro de llaves.

print $colores_frutas{'manzana'};  # Imprime: rojo

Añadir y Modificar Elementos

Puedes añadir o modificar elementos asignando un valor a una clave.

$colores_frutas{'naranja'} = 'naranja';  # Añade un nuevo par clave-valor
$colores_frutas{'banana'} = 'verde';   # Modifica el valor existente

Eliminar Elementos

Usa la función delete para eliminar un par clave-valor de un hash.

delete $colores_frutas{'uva'};  # Elimina la clave 'uva' y su valor

Iteración sobre Hashes

Usando `each`

La función each te permite iterar sobre pares clave-valor en un hash.

while (my ($clave, $valor) = each %colores_frutas) {
    print "$clave: $valor\n";
}

Usando `keys` y `values`

También puedes iterar solo sobre las claves o solo sobre los valores.

foreach my $clave (keys %colores_frutas) {
    print "$clave\n";
}

foreach my $valor (values %colores_frutas) {
    print "$valor\n";
}

Funciones de Hash

`exists`

Verifica si una clave existe en un hash.

if (exists $colores_frutas{'manzana'}) {
    print "La manzana está en el hash.\n";
}

`keys` y `values`

Recupera todas las claves o valores de un hash.

my @claves = keys %colores_frutas;
my @valores = values %colores_frutas;

Contexto `scalar`

En contexto escalar, keys devuelve el número de pares clave-valor.

my $num_elementos = scalar keys %colores_frutas;
print "Número de elementos: $num_elementos\n";

Técnicas Avanzadas de Hash

Hash de Hashes

Puedes crear estructuras de datos complejas como un hash de hashes.

my %estudiantes = (
    'Juan' => { edad => 20, carrera => 'Matemáticas' },
    'Ana' => { edad => 22, carrera => 'Biología' },
);

print $estudiantes{'Juan'}{'edad'};  # Imprime: 20

Autovivificación

Perl crea automáticamente estructuras de datos anidadas según sea necesario.

$estudiantes{'Alicia'}{'edad'} = 23;  # Crea la clave 'Alicia' y su hash anidado

Errores Comunes

Uso de Claves No Definidas

Acceder a un hash con una clave no definida devuelve undef.

my $color = $colores_frutas{'pera'};  # Devuelve undef

Sobrescribir Claves

Asignar un valor a una clave existente sobrescribe el valor anterior.

$colores_frutas{'manzana'} = 'verde';  # 'rojo' es sobrescrito con 'verde'

Conclusión

Los hashes son una estructura de datos versátil y poderosa en Perl, permitiendo un almacenamiento y recuperación eficientes de datos usando pares clave-valor. Al entender cómo crear, acceder y manipular hashes, puedes manejar estructuras de datos más complejas y mejorar la eficiencia de tus programas en Perl. Practica usando hashes con varios ejemplos para volverte competente en su uso.
