Introducción

En este curso, construiremos una red social utilizando Django, un framework web de alto nivel para Python. Este proyecto te llevará de nivel principiante a avanzado, cubriendo conceptos y características esenciales de Django.

Configuración del Entorno

  • Instalar Python y Django
  • Configurar un entorno virtual
  • Crear un nuevo proyecto Django

Ejemplo de Código: Configuración de un Proyecto Django

# Instalar Django
pip install django

# Crear un nuevo proyecto Django
django-admin startproject social_network

# Navegar al directorio del proyecto
cd social_network

# Iniciar el servidor de desarrollo
python manage.py runserver

Creando el Modelo de Usuario

  • Usar el modelo de Usuario incorporado de Django
  • Extender el modelo de Usuario con un modelo de Perfil

Ejemplo de Código: Extender el Modelo de Usuario

# models.py
from django.contrib.auth.models import User
from django.db import models

class Profile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    bio = models.TextField(max_length=500, blank=True)
    location = models.CharField(max_length=30, blank=True)
    birth_date = models.DateField(null=True, blank=True)

    def __str__(self):
        return self.user.username

Registro y Autenticación de Usuarios

  • Crear formularios de registro
  • Manejar el inicio y cierre de sesión de usuarios
  • Funcionalidad de restablecimiento de contraseña

Ejemplo de Código: Formulario de Registro de Usuario

# forms.py
from django import forms
from django.contrib.auth.models import User
from django.contrib.auth.forms import UserCreationForm

class SignUpForm(UserCreationForm):
    email = forms.EmailField(max_length=200, help_text='Required')

    class Meta:
        model = User
        fields = ('username', 'email', 'password1', 'password2')

Construyendo las Funcionalidades de la Red Social

  • Crear y gestionar publicaciones
  • Implementar perfiles de usuario
  • Añadir un sistema de seguimiento

Ejemplo de Código: Creación de un Modelo de Publicación

# models.py
from django.db import models
from django.contrib.auth.models import User

class Post(models.Model):
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.content[:20]

Implementando el Sistema de Seguimiento

  • Crear un modelo de Seguimiento
  • Gestionar relaciones de seguimiento
  • Mostrar listas de seguidores y seguidos

Ejemplo de Código: Modelo de Seguimiento

# models.py
class Follow(models.Model):
    follower = models.ForeignKey(User, related_name='following', on_delete=models.CASCADE)
    following = models.ForeignKey(User, related_name='followers', on_delete=models.CASCADE)
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.follower} sigue a {self.following}"

Funcionalidades Avanzadas

  • Añadir comentarios y "me gusta" a las publicaciones

Ejemplo de Código: Añadir Comentarios a las Publicaciones

# models.py
class Comment(models.Model):
    post = models.ForeignKey(Post, related_name='comments', on_delete=models.CASCADE)
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.content[:20]

Pruebas y Despliegue

  • Escribir pruebas unitarias para tu aplicación
  • Configurar un entorno de producción
  • Desplegar tu aplicación Django

Ejemplo de Código: Escribir una Prueba Simple

# tests.py
from django.test import TestCase
from django.contrib.auth.models import User
from .models import Post

class PostModelTest(TestCase):
    def setUp(self):
        self.user = User.objects.create_user(username='testuser', password='12345')
        self.post = Post.objects.create(author=self.user, content='Just a test post')

    def test_post_content(self):
        self.assertEqual(self.post.content, 'Just a test post')
