Introducción

Django es un framework web de alto nivel para Python que fomenta el desarrollo rápido y el diseño limpio y pragmático. Fue diseñado para ayudar a los desarrolladores a llevar las aplicaciones desde el concepto hasta la finalización lo más rápido posible. Django enfatiza la reutilización y "enchufabilidad" de componentes, menos código, bajo acoplamiento y el principio de "no te repitas" (DRY).

Conceptos Clave

  • Framework Web: Una colección de módulos y herramientas que ayudan a los desarrolladores a construir aplicaciones web.
  • Framework de Alto Nivel: Proporciona abstracciones y herramientas para manejar tareas comunes de desarrollo web, permitiendo a los desarrolladores centrarse en escribir la lógica de su aplicación.
  • Basado en Python: Escrito en Python, aprovechando su simplicidad y legibilidad.

Características de Django

  • ORM (Mapeo Objeto-Relacional): Permite a los desarrolladores interactuar con bases de datos usando código Python en lugar de SQL.
  • Interfaz de Administración: Panel de administración generado automáticamente para gestionar los datos de la aplicación.
  • Enrutamiento de URL: Mapea URLs a vistas usando una sintaxis limpia y legible.
  • Motor de Plantillas: Facilita la separación del diseño (HTML) del código Python.
  • Manejo de Formularios: Simplifica la creación, validación y procesamiento de formularios.
  • Seguridad: Incluye protección incorporada contra ataques web comunes como inyección SQL, cross-site scripting (XSS) y cross-site request forgery (CSRF).
  • Escalabilidad: Diseñado para manejar sitios web y aplicaciones de alto tráfico.

Ejemplo de Código: Una Aplicación Simple de Django

Paso 1: Configuración de Django

Primero, necesitas instalar Django. Puedes hacerlo usando pip:

pip install django

Paso 2: Creación de un Proyecto Django

Crea un nuevo proyecto Django usando el comando django-admin:

django-admin startproject myproject

Paso 3: Creación de una Aplicación Django

Navega a tu directorio de proyecto y crea una nueva aplicación:

cd myproject
python manage.py startapp myapp

Paso 4: Definición de un Modelo

En myapp/models.py, define un modelo simple:

from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.title

Paso 5: Registrar el Modelo en Admin

En myapp/admin.py, registra el modelo para hacerlo disponible en la interfaz de administración:

from django.contrib import admin
from .models import Post

admin.site.register(Post)

Paso 6: Creación de una Vista

En myapp/views.py, crea una vista simple para mostrar publicaciones:

from django.shortcuts import render
from .models import Post

def post_list(request):
    posts = Post.objects.all()
    return render(request, 'post_list.html', {'posts': posts})

Paso 7: Mapeo de la URL

En myapp/urls.py, mapea la URL a la vista:

from django.urls import path
from .views import post_list

urlpatterns = [
    path('', post_list, name='post_list'),
]

Paso 8: Creación de una Plantilla

En myapp/templates/post_list.html, crea una plantilla simple para mostrar las publicaciones:

<!DOCTYPE html>
<html>
<head>
    <title>Lista de Publicaciones</title>
</head>
<body>
    <h1>Publicaciones</h1>
    <ul>
        {% for post in posts %}
            <li>{{ post.title }} - {{ post.created_at }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Paso 9: Ejecución del Servidor

Ejecuta el servidor de desarrollo para ver tu aplicación en acción:

python manage.py runserver

Visita http://127.0.0.1:8000/ en tu navegador para ver la lista de publicaciones.

Conclusión

Django es un framework poderoso y flexible que simplifica el proceso de construcción de aplicaciones web. Sus características incorporadas como ORM, interfaz de administración y mecanismos de seguridad lo convierten en una excelente opción tanto para principiantes como para desarrolladores avanzados. Siguiendo los pasos descritos anteriormente, puedes configurar rápidamente una aplicación básica de Django y comenzar a explorar sus capacidades. A medida que avances en este curso, aprenderás características más avanzadas y mejores prácticas para construir aplicaciones web robustas y escalables con Django.
