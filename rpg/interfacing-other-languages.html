Interfaz con Otros Lenguajes
Seguridad y Mejores Prácticas ►

La interfaz con otros lenguajes de programación es una habilidad crucial para los desarrolladores de RPG (Report Program Generator). Este tema te guiará a través de las técnicas básicas y avanzadas de integración de RPG con otros lenguajes, como C, Java y SQL.

Introducción a la Interfaz

La interfaz permite que los programas RPG aprovechen las funcionalidades de otros lenguajes, mejorando sus capacidades y rendimiento. Esta sección cubre los conceptos fundamentales y los beneficios de la interfaz.

  • Conceptos Clave:
    • Interoperabilidad: La capacidad de diferentes sistemas y organizaciones para trabajar juntos.
    • APIs (Interfaces de Programación de Aplicaciones): Conjuntos de rutinas, protocolos y herramientas para construir software y aplicaciones.
    • Intercambio de Datos: Métodos para transferir datos entre diferentes entornos de programación.

Llamar Funciones C desde RPG

RPG puede llamar a funciones C para realizar tareas que no son posibles o son ineficientes en RPG. Esta sección explica cómo llamar a funciones C desde RPG.

  • Pasos para Llamar Funciones C:

    1. Definir el Prototipo de la Función C en RPG: 
      D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE
    2. Llamar a la Función C en RPG: 
      C                   eval      result = myCFunction(10 : 20)

  • Ejemplo:

    // Definición de la función C
int myCFunction(int a, int b) {
    return a + b;
}

    // Programa RPG llamando a la función C
D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = myCFunction(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Integración de Java con RPG

Java ofrece una amplia gama de bibliotecas y funcionalidades que pueden ser utilizadas en programas RPG. Esta sección cubre la integración de Java con RPG.

  • Pasos para Llamar Métodos Java:

    1. Definir la Clase y el Método Java: 
      public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}
    2. Usar RPG para Llamar al Método Java: 
      D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

  • Ejemplo:

    // Definición de la clase Java
public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}

    // Programa RPG llamando al método Java
D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Uso de SQL en RPG

SQL puede ser incrustado en programas RPG para realizar operaciones de base de datos de manera eficiente. Esta sección explica cómo usar SQL dentro de RPG.

  • Pasos para Incrustar SQL en RPG:

    1. Incluir la Directiva del Precompilador SQL: 
      /EXEC SQL
    2. Escribir Sentencias SQL dentro del Programa RPG: 
      /EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

  • Ejemplo:

    // Programa RPG con SQL incrustado
D recordCount     S               10I 0

/EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

C                   dsply     recordCount
C                   eval      *inlr = *on

Conclusión

La interfaz de RPG con otros lenguajes como C, Java y SQL mejora significativamente la funcionalidad y el rendimiento de las aplicaciones RPG. Al comprender y aplicar estas técnicas de interfaz, los desarrolladores de RPG pueden crear programas más robustos y versátiles. Practica estos ejemplos y explora más para dominar el arte de la interfaz en RPG.
