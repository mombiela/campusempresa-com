Creación de Funciones Personalizadas
Bucles: DO, FOR y WHILE ►

Introducción

Crear funciones personalizadas es una habilidad fundamental en la programación RPG (Report Program Generator). Las funciones te permiten encapsular código reutilizable, haciendo que tus programas sean más modulares, legibles y mantenibles. Esta sección te guiará a través del proceso de creación de funciones personalizadas en RPG, desde niveles básicos hasta avanzados.

Conceptos Clave

  • Definición de Función: Cómo definir una función en RPG.
  • Parámetros: Pasar datos a las funciones.
  • Valores de Retorno: Devolver datos desde las funciones.
  • Ámbito: Entender las variables locales y globales dentro de las funciones.
  • Modularidad: Descomponer tareas complejas en funciones más simples.

Definición Básica de Función

En RPG, las funciones se definen utilizando las palabras clave DCL-PROC y END-PROC. Aquí hay un ejemplo simple de una función que suma dos números:

DCL-PROC AddNumbers;
  DCL-PI AddNumbers INT(10);
    num1 INT(10);
    num2 INT(10);
  END-PI;

  DCL-S result INT(10);

  result = num1 + num2;

  RETURN result;
END-PROC;

Explicación

  • DCL-PROC AddNumbers;: Declara el inicio de un procedimiento llamado AddNumbers.
  • DCL-PI AddNumbers INT(10);: Declara la interfaz del procedimiento, especificando que AddNumbers devuelve un entero de longitud 10.
  • num1 INT(10); y num2 INT(10);: Parámetros de la función, ambos enteros de longitud 10.
  • DCL-S result INT(10);: Declara una variable local result para almacenar la suma.
  • result = num1 + num2;: Suma los dos parámetros y almacena el resultado.
  • RETURN result;: Devuelve el resultado al llamador.
  • END-PROC;: Marca el final del procedimiento.

Parámetros y Valores de Retorno

Las funciones pueden tomar múltiples parámetros y devolver un solo valor. Aquí hay un ejemplo de una función que calcula el área de un rectángulo:

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

Explicación

  • La función CalculateArea toma dos parámetros: length y width.
  • Calcula el área multiplicando length y width.
  • El resultado se almacena en la variable local area y se devuelve al llamador.

Ámbito de Variables

Entender el ámbito de las variables es crucial cuando se trabaja con funciones. Las variables declaradas dentro de una función son locales a esa función y no pueden ser accedidas fuera de ella.

DCL-PROC ExampleScope;
  DCL-PI ExampleScope;
  END-PI;

  DCL-S localVar INT(10);

  localVar = 100;

  RETURN;
END-PROC;

Explicación

  • localVar es una variable local y solo puede ser accedida dentro de la función ExampleScope.
  • Intentar acceder a localVar fuera de ExampleScope resultará en un error.

Modularidad

Descomponer tareas complejas en funciones más simples hace que tu código sea más modular y fácil de mantener. Aquí hay un ejemplo de un programa que utiliza múltiples funciones para realizar una serie de cálculos:

DCL-PROC Main;
  DCL-PI Main;
  END-PI;

  DCL-S length INT(10) INZ(5);
  DCL-S width INT(10) INZ(10);
  DCL-S area INT(10);
  DCL-S perimeter INT(10);

  area = CalculateArea(length: width);
  perimeter = CalculatePerimeter(length: width);

  RETURN;
END-PROC;

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

DCL-PROC CalculatePerimeter;
  DCL-PI CalculatePerimeter INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S perimeter INT(10);

  perimeter = 2 * (length + width);

  RETURN perimeter;
END-PROC;

Explicación

  • El procedimiento Main inicializa length y width.
  • Llama a CalculateArea y CalculatePerimeter para calcular el área y el perímetro de un rectángulo.
  • Cada función realiza una tarea específica, haciendo que el código sea más modular y fácil de entender.

Conclusión

Crear funciones personalizadas en RPG te permite escribir código más modular, mantenible y reutilizable. Al entender cómo definir funciones, pasar parámetros, devolver valores y gestionar el ámbito de las variables, puedes descomponer tareas complejas en piezas más simples y manejables. Practica creando tus propias funciones para volverte competente en esta habilidad esencial de programación.
