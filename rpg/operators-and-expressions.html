Introducción

En RPG (Report Program Generator), los operadores y expresiones son componentes fundamentales que te permiten realizar cálculos, tomar decisiones y manipular datos. Esta sección cubrirá los diversos tipos de operadores disponibles en RPG y cómo usarlos eficazmente en expresiones.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos.

  • Adición (+): Suma dos operandos.
  • Sustracción (-): Resta el segundo operando del primero.
  • Multiplicación (*): Multiplica dos operandos.
  • División (/): Divide el primer operando por el segundo.
  • Exponenciación (**): Eleva el primer operando a la potencia del segundo.

Ejemplo

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S result Int(10);

result = num1 + num2; // result es 15
result = num1 - num2; // result es 5
result = num1 * num2; // result es 50
result = num1 / num2; // result es 2
result = num1 ** num2; // result es 100000

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores y devolver un resultado Booleano (True o False).

  • Igual a (= o EQ)
  • No igual a (<> o NE)
  • Mayor que (> o GT)
  • Menor que (< o LT)
  • Mayor o igual a (>= o GE)
  • Menor o igual a (<= o LE)

Ejemplo

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S isEqual Ind;

isEqual = (num1 = num2); // isEqual es *False
isEqual = (num1 <> num2); // isEqual es *True
isEqual = (num1 > num2); // isEqual es *True
isEqual = (num1 < num2); // isEqual es *False
isEqual = (num1 >= num2); // isEqual es *True
isEqual = (num1 <= num2); // isEqual es *False

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones.

  • AND (& o AND): Devuelve True si ambas condiciones son True.
  • OR (| o OR): Devuelve True si al menos una condición es True.
  • NOT (NOT): Invierte el valor Booleano de la condición.

Ejemplo

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S num3 Int(10) Inz(15);
Dcl-S condition Ind;

condition = (num1 > num2) AND (num1 < num3); // condition es *True
condition = (num1 > num2) OR (num1 > num3); // condition es *True
condition = NOT(num1 = num2); // condition es *True

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación simple (=): Asigna el valor del operando derecho al operando izquierdo.
  • Asignación de adición (+=): Suma el operando derecho al operando izquierdo y asigna el resultado al operando izquierdo.
  • Asignación de sustracción (-=): Resta el operando derecho del operando izquierdo y asigna el resultado al operando izquierdo.
  • Asignación de multiplicación (*=): Multiplica el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.
  • Asignación de división (/=): Divide el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.

Ejemplo

Dcl-S num Int(10) Inz(10);

num = 5; // num es 5
num += 3; // num es 8
num -= 2; // num es 6
num *= 4; // num es 24
num /= 3; // num es 8

Operadores de Cadenas

Los operadores de cadenas se utilizan para manipular cadenas de texto.

  • Concatenación (+): Combina dos cadenas en una sola.

Ejemplo

Dcl-S firstName Char(20) Inz('John');
Dcl-S lastName Char(20) Inz('Doe');
Dcl-S fullName Char(40);

fullName = firstName + ' ' + lastName; // fullName es 'John Doe'

Resumen

En esta sección, cubrimos los diversos tipos de operadores disponibles en RPG, incluyendo operadores aritméticos, de comparación, lógicos, de asignación y de cadenas. Entender cómo usar estos operadores eficazmente es crucial para realizar cálculos, tomar decisiones y manipular datos en tus programas RPG. Practica usando estos operadores en diferentes escenarios para volverte competente en escribir expresiones en RPG.
