Introducción

JavaScript es un lenguaje de programación versátil y poderoso, utilizado principalmente para el desarrollo web. Esta sección cubrirá la sintaxis fundamental y los conceptos básicos que son esenciales para cualquiera que comience con JavaScript.

Conceptos Clave

Variables

Las variables se utilizan para almacenar datos que pueden ser referenciados y manipulados en un programa. JavaScript usa tres palabras clave para declarar variables:

  • var
  • let
  • const

Ejemplo:

var name = "John";
let age = 30;
const isStudent = true;

Tipos de Datos

JavaScript soporta varios tipos de datos, incluyendo:

  • String: Datos textuales
  • Number: Datos numéricos
  • Boolean: Valores verdaderos o falsos
  • Array: Lista ordenada de valores
  • Object: Colección de pares clave-valor
  • Null: Representa la ausencia intencional de cualquier valor de objeto
  • Undefined: Variable que ha sido declarada pero no asignada un valor

Ejemplo:

let message = "Hello, World!"; // String
let count = 42; // Number
let isActive = false; // Boolean
let fruits = ["Apple", "Banana", "Cherry"]; // Array
let person = { firstName: "John", lastName: "Doe" }; // Object
let emptyValue = null; // Null
let notAssigned; // Undefined

Operadores

Los operadores se utilizan para realizar operaciones en variables y valores. Los operadores comunes incluyen:

  • Operadores Aritméticos: +, -, *, /, %
  • Operadores de Asignación: =, +=, -=, *=, /=
  • Operadores de Comparación: ==, ===, !=, !==, >, <, >=, <=
  • Operadores Lógicos: &&, ||, !

Ejemplo:

let a = 10;
let b = 20;
let sum = a + b; // Aritmético
a += 5; // Asignación
let isEqual = (a == b); // Comparación
let isBothTrue = (a > 5 && b < 30); // Lógico

Estructuras de Control

Las estructuras de control se utilizan para dictar el flujo del programa. Las estructuras de control comunes incluyen:

  • Sentencias Condicionales: if, else if, else
  • Sentencias Switch: switch
  • Bucles: for, while, do...while

Ejemplo:

// Sentencia Condicional
if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// Sentencia Switch
let day = 3;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    case 3:
        console.log("Wednesday");
        break;
    default:
        console.log("Invalid day");
}

// Bucle For
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Bucle While
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Bucle Do...While
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);

Funciones

Las funciones son bloques de código diseñados para realizar una tarea particular. Se ejecutan cuando son llamadas.

Ejemplo:

// Declaración de Función
function greet(name) {
    return "Hello, " + name + "!";
}

// Expresión de Función
const greet = function(name) {
    return "Hello, " + name + "!";
};

// Función Flecha (ES6)
const greet = (name) => {
    return "Hello, " + name + "!";
};

// Llamada a una Función
console.log(greet("Alice"));

Conclusión

Entender los conceptos básicos de la sintaxis de JavaScript es crucial para construir una base sólida en programación. Esta sección cubrió variables, tipos de datos, operadores, estructuras de control y funciones. Dominar estos conceptos te permitirá escribir código JavaScript simple pero efectivo y te preparará para temas más avanzados. Sigue practicando y experimentando con estos conceptos básicos para sentirte más cómodo con JavaScript.
