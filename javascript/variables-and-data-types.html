Introducción

En JavaScript, las variables y los tipos de datos son conceptos fundamentales que forman los bloques de construcción de cualquier programa. Entender cómo declarar, asignar y manipular variables, así como conocer los diferentes tipos de datos disponibles, es crucial para escribir código efectivo y eficiente.

Variables

Declaración de Variables

En JavaScript, puedes declarar variables usando tres palabras clave: var, let y const.

  • var: La forma más antigua de declarar una variable. Tiene ámbito de función.
  • let: Introducido en ES6, tiene ámbito de bloque.
  • const: También introducido en ES6, declara una variable constante que no puede ser reasignada.

Ejemplos:

var name = "John"; // Ámbito de función
let age = 30;      // Ámbito de bloque
const PI = 3.14;   // Ámbito de bloque y no puede ser reasignado

Ámbito de Variables

  • Ámbito de Función: Las variables declaradas con var tienen ámbito en la función en la que se declaran.
  • Ámbito de Bloque: Las variables declaradas con let y const tienen ámbito en el bloque (por ejemplo, dentro de un bucle o una declaración if) en el que se declaran.

Ejemplo:

function exampleFunction() {
    var functionScoped = "I am function scoped";
    if (true) {
        let blockScoped = "I am block scoped";
        const alsoBlockScoped = "I am also block scoped";
        console.log(blockScoped); // Funciona
        console.log(alsoBlockScoped); // Funciona
    }
    console.log(functionScoped); // Funciona
    // console.log(blockScoped); // Error: blockScoped no está definido
    // console.log(alsoBlockScoped); // Error: alsoBlockScoped no está definido
}

Tipos de Datos

Tipos de Datos Primitivos

JavaScript tiene varios tipos de datos primitivos:

  • String: Representa datos textuales.
  • Number: Representa tanto números enteros como de punto flotante.
  • Boolean: Representa verdadero o falso.
  • Undefined: Representa una variable que ha sido declarada pero no asignada un valor.
  • Null: Representa la ausencia intencional de cualquier valor de objeto.
  • Symbol: Representa un valor único e inmutable (introducido en ES6).
  • BigInt: Representa enteros con precisión arbitraria (introducido en ES2020).

Ejemplos:

let str = "Hello, World!"; // String
let num = 42;              // Number
let isTrue = true;         // Boolean
let notAssigned;           // Undefined
let emptyValue = null;     // Null
let uniqueId = Symbol("id"); // Symbol
let bigNumber = 1234567890123456789012345678901234567890n; // BigInt

Tipos de Datos No Primitivos

JavaScript también tiene tipos de datos no primitivos, principalmente objetos.

  • Object: Una colección de propiedades, donde cada propiedad es un par clave-valor.
  • Array: Un tipo especial de objeto utilizado para almacenar colecciones ordenadas de valores.

Ejemplos:

let person = {
    name: "Alice",
    age: 25
}; // Objeto

let numbers = [1, 2, 3, 4, 5]; // Array

Conversión de Tipos

JavaScript es un lenguaje de tipado débil, lo que significa que puedes convertir entre diferentes tipos de datos.

  • Conversión Implícita: JavaScript convierte automáticamente los tipos cuando es necesario.
  • Conversión Explícita: Puedes convertir manualmente los tipos usando funciones como String(), Number() y Boolean().

Ejemplos:

let x = "5" + 2; // Conversión implícita: "52" (string)
let y = "5" - 2; // Conversión implícita: 3 (number)

let strToNum = Number("123"); // Conversión explícita: 123 (number)
let numToStr = String(123);   // Conversión explícita: "123" (string)
let boolToNum = Number(true); // Conversión explícita: 1 (number)

Resumen

Entender las variables y los tipos de datos es esencial para cualquier programador de JavaScript. Las variables pueden ser declaradas usando var, let o const, cada una con diferentes reglas de ámbito. JavaScript soporta varios tipos de datos, incluyendo tipos primitivos como strings, números y booleanos, así como tipos no primitivos como objetos y arrays. Saber cómo trabajar con estos tipos y convertir entre ellos te ayudará a escribir código más robusto y flexible.
