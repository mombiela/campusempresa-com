¿Qué es el DOM?

El Modelo de Objetos del Documento (DOM) es una interfaz de programación para documentos web. Representa la página de manera que los programas pueden cambiar la estructura, el estilo y el contenido del documento. El DOM representa el documento como nodos y objetos; de esa manera, los lenguajes de programación pueden interactuar con la página.

  • Documento: El documento HTML o XML.
  • Objeto: Cada elemento en el documento es un objeto.
  • Modelo: La estructura del documento.

¿Por qué es importante el DOM?

Entender el DOM es crucial para el desarrollo web porque te permite:

  • Cambiar dinámicamente el contenido de una página web.
  • Modificar la estructura del documento.
  • Cambiar los estilos de los elementos.
  • Reaccionar a eventos del usuario (por ejemplo, clics, pulsaciones de teclas).

Conceptos básicos

Nodos y Elementos

En el DOM, todo es un nodo. Los nodos pueden ser:

  • Nodos de elementos: Representan elementos HTML.
  • Nodos de texto: Representan texto dentro de los elementos.
  • Nodos de atributos: Representan atributos de los elementos.
  • Nodos de comentarios: Representan comentarios en el HTML.

Estructura de árbol

El DOM está estructurado como un árbol de nodos. El objeto documento es la raíz, y cada elemento es una rama.

<!DOCTYPE html>
<html>
  <head>
    <title>Documento</title>
  </head>
  <body>
    <h1>¡Hola, Mundo!</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

En el HTML anterior, la estructura del árbol se vería así:

Documento
 └── html
      ├── head
      │    └── title
      └── body
           ├── h1
           └── p

Accediendo al DOM

Seleccionando Elementos

Para manipular el DOM, primero necesitas seleccionar los elementos con los que quieres trabajar. JavaScript proporciona varios métodos para esto:

  • document.getElementById(id): Selecciona un elemento por su ID.
  • document.getElementsByClassName(className): Selecciona todos los elementos con la clase especificada.
  • document.getElementsByTagName(tagName): Selecciona todos los elementos con el nombre de etiqueta especificado.
  • document.querySelector(selector): Selecciona el primer elemento que coincide con el selector CSS.
  • document.querySelectorAll(selector): Selecciona todos los elementos que coinciden con el selector CSS.

Ejemplo

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">¡Hola, Mundo!</h1>
    <p class="text">Este es un párrafo.</p>
    <p class="text">Este es otro párrafo.</p>

    <script>
      // Seleccionar elemento por ID
      const header = document.getElementById('header');
      console.log(header); // <h1 id="header">¡Hola, Mundo!</h1>

      // Seleccionar elementos por nombre de clase
      const paragraphs = document.getElementsByClassName('text');
      console.log(paragraphs); // Colección HTML de elementos <p>

      // Seleccionar elementos por nombre de etiqueta
      const allParagraphs = document.getElementsByTagName('p');
      console.log(allParagraphs); // Colección HTML de elementos <p>

      // Seleccionar el primer elemento que coincide con el selector CSS
      const firstParagraph = document.querySelector('.text');
      console.log(firstParagraph); // <p class="text">Este es un párrafo.</p>

      // Seleccionar todos los elementos que coinciden con el selector CSS
      const allTextElements = document.querySelectorAll('.text');
      console.log(allTextElements); // Lista de nodos de elementos <p>
    </script>
  </body>
</html>

Manipulando el DOM

Cambiando el Contenido

Puedes cambiar el contenido de un elemento usando las propiedades innerHTML o textContent.

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">¡Hola, Mundo!</h1>

    <script>
      const header = document.getElementById('header');
      header.innerHTML = '¡Hola, JavaScript!'; // Cambia el contenido HTML
      header.textContent = '¡Hola, DOM!'; // Cambia el contenido de texto
    </script>
  </body>
</html>

Cambiando Estilos

Puedes cambiar los estilos de un elemento usando la propiedad style.

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">¡Hola, Mundo!</h1>

    <script>
      const header = document.getElementById('header');
      header.style.color = 'blue'; // Cambia el color del texto a azul
      header.style.fontSize = '24px'; // Cambia el tamaño de la fuente a 24px
    </script>
  </body>
</html>

Añadiendo y Eliminando Elementos

Puedes añadir y eliminar elementos usando métodos como appendChild, removeChild, createElement e insertBefore.

<!DOCTYPE html>
<html>
  <body>
    <div id="container">
      <p>Este es un párrafo.</p>
    </div>

    <script>
      const container = document.getElementById('container');

      // Crear un nuevo elemento
      const newParagraph = document.createElement('p');
      newParagraph.textContent = 'Este es un nuevo párrafo.';

      // Añadir el nuevo elemento
      container.appendChild(newParagraph);

      // Eliminar un elemento
      const oldParagraph = container.getElementsByTagName('p')[0];
      container.removeChild(oldParagraph);
    </script>
  </body>
</html>

Manejo de Eventos

Los eventos son acciones que ocurren en el navegador, como clics, pulsaciones de teclas o movimientos del ratón. Puedes usar JavaScript para escuchar estos eventos y ejecutar código en respuesta.

Añadiendo Listeners de Eventos

Puedes añadir listeners de eventos usando el método addEventListener.

<!DOCTYPE html>
<html>
  <body>
    <button id="myButton">¡Haz clic en mí!</button>

    <script>
      const button = document.getElementById('myButton');

      button.addEventListener('click', function() {
        alert('¡El botón fue clicado!');
      });
    </script>
  </body>
</html>

Conclusión

El DOM es una interfaz poderosa que te permite interactuar y manipular documentos web. Al entender el DOM, puedes crear páginas web dinámicas e interactivas. Esta introducción cubrió los conceptos básicos de qué es el DOM, cómo acceder y manipularlo, y cómo manejar eventos. A medida que progreses, aprenderás técnicas más avanzadas y mejores prácticas para trabajar con el DOM.
