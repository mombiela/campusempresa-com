Introducción

El paso a paso a través del código es una técnica esencial de depuración que te permite ejecutar tu código VBA línea por línea. Esto te ayuda a entender el flujo de tu programa, identificar errores lógicos e inspeccionar los valores de las variables en diferentes etapas de la ejecución.

Conceptos Clave

  • Puntos de interrupción: Marcadores que puedes establecer en tu código para pausar la ejecución en una línea específica.
  • Paso dentro: Ejecuta la siguiente línea de código y, si es una llamada a un procedimiento, entra en el procedimiento.
  • Paso sobre: Ejecuta la siguiente línea de código pero no entra en los procedimientos llamados.
  • Paso fuera: Ejecuta las líneas restantes del procedimiento actual y se pausa después de regresar al procedimiento que lo llamó.
  • Ventana Inmediata: Una herramienta donde puedes ejecutar declaraciones VBA e inspeccionar variables durante la depuración.

Establecer Puntos de Interrupción

Los puntos de interrupción se utilizan para pausar la ejecución de tu código en puntos específicos. Esto te permite inspeccionar el estado de tu programa y entender su comportamiento.

Cómo Establecer un Punto de Interrupción

  1. Abre el editor de VBA (Presiona Alt + F11).
  2. Haz clic en el margen izquierdo junto a la línea de código donde deseas establecer el punto de interrupción, o presiona F9 mientras el cursor está en esa línea.
Sub EjemploMacro()
    Dim x As Integer
    x = 10 ' Establece un punto de interrupción aquí
    x = x + 5
    MsgBox x
End Sub

Paso a Paso a Través del Código

El paso a paso a través del código implica ejecutar tu programa una línea a la vez para observar su comportamiento.

Paso Dentro

  • Usa F8 para entrar en la siguiente línea de código. Si la línea contiene una llamada a otro procedimiento, entrará en ese procedimiento.
Sub Principal()
    Llamar SubProcedimiento
End Sub

Sub SubProcedimiento()
    Dim y As Integer
    y = 20
    MsgBox y
End Sub

Paso Sobre

  • Usa Shift + F8 para pasar sobre la siguiente línea de código. Si la línea contiene una llamada a otro procedimiento, ejecutará todo el procedimiento sin entrar en él.
Sub Principal()
    Llamar SubProcedimiento ' Pasa sobre esta línea
End Sub

Sub SubProcedimiento()
    Dim y As Integer
    y = 20
    MsgBox y
End Sub

Paso Fuera

  • Usa Ctrl + Shift + F8 para salir del procedimiento actual. Esto ejecutará las líneas restantes del procedimiento actual y se pausará después de regresar al procedimiento que lo llamó.
Sub Principal()
    Llamar SubProcedimiento
    MsgBox "De vuelta en Principal"
End Sub

Sub SubProcedimiento()
    Dim y As Integer
    y = 20
    MsgBox y ' Salir desde aquí
End Sub

Uso de la Ventana Inmediata

La Ventana Inmediata es una herramienta poderosa para la depuración. Puedes usarla para ejecutar declaraciones VBA, inspeccionar y modificar valores de variables, e imprimir resultados.

Ejemplo de Uso

  1. Abre la Ventana Inmediata (Presiona Ctrl + G).
  2. Escribe ? x para imprimir el valor de x.
  3. Escribe x = 15 para cambiar el valor de x.
Sub EjemploVentanaInmediata()
    Dim x As Integer
    x = 10
    Debug.Print x ' Imprime 10 en la Ventana Inmediata
    x = x + 5
    Debug.Print x ' Imprime 15 en la Ventana Inmediata
End Sub

Conclusión

El paso a paso a través del código es una habilidad fundamental para depurar programas VBA. Al usar puntos de interrupción, entrar, pasar sobre y salir del código, y aprovechar la Ventana Inmediata, puedes obtener una comprensión más profunda del flujo de tu programa y rápidamente identificar y corregir errores. Practica estas técnicas para volverte competente en la depuración y mejorar la fiabilidad de tus aplicaciones VBA.
