Creación de Funciones Personalizadas
Interactuando con Otras Aplicaciones de Office ►

Crear funciones personalizadas en VBA te permite extender las capacidades de Excel y otras aplicaciones de Office escribiendo tu propio código reutilizable. Este tema te guiará desde los conceptos básicos hasta técnicas avanzadas en la creación de funciones personalizadas.

Introducción a las Funciones Personalizadas

Las funciones personalizadas, también conocidas como Funciones Definidas por el Usuario (UDFs), son funciones que creas para realizar cálculos o tareas específicas que no están disponibles en las funciones integradas de Excel.

Conceptos Clave

  • Definición de Función: La estructura y sintaxis de una función.
  • Argumentos de Función: Parámetros que pasas a la función.
  • Valor de Retorno: El resultado que la función devuelve.

Escribiendo Tu Primera Función Personalizada

Comencemos escribiendo una función personalizada simple en VBA.

Ejemplo: Una Función de Suma Simple

Function AddNumbers(a As Double, b As Double) As Double
    AddNumbers = a + b
End Function

Explicación

  • Función AddNumbers: Esto define una nueva función llamada AddNumbers.
  • Parámetros (a As Double, b As Double): La función toma dos argumentos, a y b, ambos de tipo Double.
  • Valor de Retorno (As Double): La función devuelve un valor de tipo Double.
  • Cuerpo de la Función (AddNumbers = a + b): La función suma los dos argumentos y devuelve el resultado.

Usando la Función Personalizada en Excel

  1. Abre Excel y presiona Alt + F11 para abrir el editor de VBA.
  2. Inserta un nuevo módulo haciendo clic en Insertar > Módulo.
  3. Copia y pega la función AddNumbers en el módulo.
  4. Regresa a Excel y usa la función como cualquier otra función de Excel: 
    =AddNumbers(5, 10)

Agregando Manejo de Errores

Es importante manejar errores en tus funciones para hacerlas robustas.

Ejemplo: División con Manejo de Errores

Function SafeDivide(a As Double, b As Double) As Variant
    If b = 0 Then
        SafeDivide = "Error: División por cero"
    Else
        SafeDivide = a / b
    End If
End Function

Explicación

  • Tipo de Retorno (As Variant): Usar Variant permite que la función devuelva un número o un mensaje de error.
  • Chequeo de Error (If b = 0): Verifica si el denominador es cero.
  • Mensaje de Error (SafeDivide = "Error: División por cero"): Devuelve un mensaje de error si el denominador es cero.
  • División (SafeDivide = a / b): Realiza la división si el denominador no es cero.

Funciones Personalizadas Avanzadas

Las funciones personalizadas avanzadas pueden incluir lógica más compleja, múltiples pasos e interacción con objetos de Excel.

Ejemplo: Calcular el Promedio de un Rango

Function AverageRange(rng As Range) As Double
    Dim cell As Range
    Dim total As Double
    Dim count As Integer
    
    total = 0
    count = 0
    
    For Each cell In rng
        If IsNumeric(cell.Value) Then
            total = total + cell.Value
            count = count + 1
        End If
    Next cell
    
    If count > 0 Then
        AverageRange = total / count
    Else
        AverageRange = 0
    End If
End Function

Explicación

  • Parámetro (rng As Range): La función toma un rango de celdas como argumento.
  • Variables (Dim total As Double, count As Integer): Inicializa variables para almacenar la suma total y el conteo de celdas numéricas.
  • Bucle (For Each cell In rng): Itera a través de cada celda en el rango.
  • Chequeo (If IsNumeric(cell.Value)): Verifica si la celda contiene un valor numérico.
  • Suma y Conteo (total = total + cell.Value, count = count + 1): Suma el valor de la celda al total e incrementa el conteo.
  • Cálculo del Promedio (AverageRange = total / count): Calcula el promedio si hay celdas numéricas, de lo contrario devuelve 0.

Conclusión

Crear funciones personalizadas en VBA te permite adaptar Excel a tus necesidades específicas, haciendo tu trabajo más eficiente y efectivo. Al entender los conceptos básicos de la definición de funciones, manejo de errores e implementación de lógica avanzada, puedes crear funciones poderosas y reutilizables.

Resumen

  • Las funciones personalizadas extienden las capacidades de Excel.
  • Comienza con funciones simples y gradualmente añade complejidad.
  • Maneja errores para hacer tus funciones robustas.
  • Usa bucles y condiciones para crear funciones avanzadas.

Al dominar estas técnicas, podrás crear funciones personalizadas que ahorren tiempo y mejoren tu productividad en Excel.
