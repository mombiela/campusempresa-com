Introducción

Las estructuras de control son fundamentales en la programación ya que te permiten dictar el flujo de tu programa basado en ciertas condiciones. En VBA (Visual Basic for Applications), la declaración If...Then...Else es una herramienta poderosa para la toma de decisiones. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de If...Then...Else en VBA.

Declaración Básica If...Then

La forma más simple de estructura de control es la declaración If...Then. Te permite ejecutar un bloque de código solo si una condición especificada es verdadera.

Sintaxis

If condition Then
    ' Código a ejecutar si la condición es verdadera
End If

Ejemplo

Sub CheckValue()
    Dim value As Integer
    value = 10
    
    If value > 5 Then
        MsgBox "El valor es mayor que 5"
    End If
End Sub

En este ejemplo, el cuadro de mensaje mostrará "El valor es mayor que 5" porque la condición value > 5 es verdadera.

Declaración If...Then...Else

Cuando necesitas ejecutar un bloque de código si una condición es verdadera y otro bloque si es falsa, utilizas la declaración If...Then...Else.

Sintaxis

If condition Then
    ' Código a ejecutar si la condición es verdadera
Else
    ' Código a ejecutar si la condición es falsa
End If

Ejemplo

Sub CheckValue()
    Dim value As Integer
    value = 3
    
    If value > 5 Then
        MsgBox "El valor es mayor que 5"
    Else
        MsgBox "El valor es 5 o menos"
    End If
End Sub

En este ejemplo, el cuadro de mensaje mostrará "El valor es 5 o menos" porque la condición value > 5 es falsa.

Declaración If...Then...ElseIf...Else

Para múltiples condiciones, puedes usar la palabra clave ElseIf para verificar condiciones adicionales.

Sintaxis

If condition1 Then
    ' Código a ejecutar si la condición1 es verdadera
ElseIf condition2 Then
    ' Código a ejecutar si la condición2 es verdadera
Else
    ' Código a ejecutar si ninguna de las condiciones anteriores es verdadera
End If

Ejemplo

Sub CheckValue()
    Dim value As Integer
    value = 7
    
    If value > 10 Then
        MsgBox "El valor es mayor que 10"
    ElseIf value > 5 Then
        MsgBox "El valor es mayor que 5 pero 10 o menos"
    Else
        MsgBox "El valor es 5 o menos"
    End If
End Sub

En este ejemplo, el cuadro de mensaje mostrará "El valor es mayor que 5 pero 10 o menos" porque la condición value > 5 es verdadera.

Declaraciones If Anidadas

Puedes anidar declaraciones If...Then...Else dentro de otras para manejar decisiones más complejas.

Ejemplo

Sub CheckValue()
    Dim value As Integer
    value = 12
    
    If value > 10 Then
        If value > 15 Then
            MsgBox "El valor es mayor que 15"
        Else
            MsgBox "El valor es mayor que 10 pero 15 o menos"
        End If
    Else
        MsgBox "El valor es 10 o menos"
    End If
End Sub

En este ejemplo, el cuadro de mensaje mostrará "El valor es mayor que 10 pero 15 o menos" porque la condición externa value > 10 es verdadera, pero la condición interna value > 15 es falsa.

Uso Avanzado

Uso de Operadores Lógicos

Puedes combinar múltiples condiciones usando operadores lógicos como And, Or y Not.

Ejemplo

Sub CheckValue()
    Dim value As Integer
    value = 8
    
    If value > 5 And value < 10 Then
        MsgBox "El valor está entre 5 y 10"
    Else
        MsgBox "El valor no está entre 5 y 10"
    End If
End Sub

En este ejemplo, el cuadro de mensaje mostrará "El valor está entre 5 y 10" porque ambas condiciones value > 5 y value < 10 son verdaderas.

Conclusión

La declaración If...Then...Else es una estructura de control versátil en VBA que te permite tomar decisiones basadas en condiciones. Al entender y utilizar sus diversas formas, desde simples declaraciones If...Then hasta complejas verificaciones de condiciones anidadas y lógicas, puedes controlar el flujo de tus programas VBA de manera efectiva. Practica con diferentes escenarios para volverte competente en el uso de estas estructuras de control.
