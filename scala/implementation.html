Introducción a la Implementación en Scala

En esta sección, exploraremos cómo implementar varios constructos y conceptos de programación en Scala. Comenzaremos con implementaciones básicas y gradualmente nos moveremos a temas más avanzados. Al final de esta sección, deberías ser capaz de escribir y entender código Scala que implemente una variedad de funcionalidades.

Implementaciones Básicas

Variables y Tipos de Datos

Scala soporta varios tipos de datos y te permite definir variables usando val (inmutable) y var (mutable).

val immutableVariable: Int = 10
var mutableVariable: Int = 20

// Intentar reasignar immutableVariable causará un error de compilación
// immutableVariable = 15 // Esto no compilará

mutableVariable = 25 // Esto está permitido
  • Conceptos Clave:
    • val para variables inmutables.
    • var para variables mutables.
    • Tipos de datos comunes: Int, Double, String, Boolean.

Estructuras de Control

Scala proporciona estructuras de control estándar como if-else, for y while.

val x = 10
if (x > 5) {
  println("x es mayor que 5")
} else {
  println("x es 5 o menos")
}

for (i <- 1 to 5) {
  println(i)
}

var count = 0
while (count < 5) {
  println(count)
  count += 1
}
  • Conceptos Clave:
    • Declaraciones condicionales: if, else.
    • Constructos de bucle: for, while.

Implementaciones Intermedias

Funciones

Las funciones en Scala pueden definirse usando la palabra clave def.

def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)
println(sum) // Salida: 8
  • Conceptos Clave:
    • Definición de funciones usando def.
    • Parámetros de función y tipos de retorno.

Colecciones

Scala proporciona un conjunto rico de tipos de colecciones como List, Set y Map.

val numbers: List[Int] = List(1, 2, 3, 4, 5)
val uniqueNumbers: Set[Int] = Set(1, 2, 2, 3)
val nameToAge: Map[String, Int] = Map("Alice" -> 25, "Bob" -> 30)

println(numbers) // Salida: List(1, 2, 3, 4, 5)
println(uniqueNumbers) // Salida: Set(1, 2, 3)
println(nameToAge) // Salida: Map(Alice -> 25, Bob -> 30)
  • Conceptos Clave:
    • Colecciones inmutables: List, Set, Map.
    • Operaciones básicas en colecciones.

Implementaciones Avanzadas

Clases y Objetos

Scala es un lenguaje orientado a objetos y te permite definir clases y objetos.

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hola, mi nombre es $name y tengo $age años."
}

val alice = new Person("Alice", 25)
println(alice.greet()) // Salida: Hola, mi nombre es Alice y tengo 25 años.
  • Conceptos Clave:
    • Definición de clases usando class.
    • Parámetros del constructor.
    • Métodos dentro de las clases.

Traits y Mixins

Los traits se utilizan para compartir interfaces y campos entre clases.

trait Greeter {
  def greet(name: String): String = s"Hola, $name!"
}

class FriendlyPerson(val name: String) extends Greeter

val bob = new FriendlyPerson("Bob")
println(bob.greet(bob.name)) // Salida: Hola, Bob!
  • Conceptos Clave:
    • Definición de traits usando trait.
    • Mezcla de traits en clases usando extends y with.

Coincidencia de Patrones

La coincidencia de patrones es una característica poderosa en Scala para verificar un valor contra un patrón.

val number = 2
val numberType = number match {
  case 1 => "Uno"
  case 2 => "Dos"
  case _ => "Otro"
}

println(numberType) // Salida: Dos
  • Conceptos Clave:
    • Coincidencia de patrones usando match.
    • Declaraciones de caso.

Resumen

En esta sección, cubrimos la implementación de varios constructos en Scala, comenzando desde variables y estructuras de control básicas hasta temas más avanzados como clases, traits y coincidencia de patrones. Entender estas implementaciones es crucial para escribir código Scala efectivo. Practica estos conceptos con los ejemplos y ejercicios proporcionados para solidificar tu comprensión.
