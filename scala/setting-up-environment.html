Configuración del Entorno de Desarrollo
Conceptos Básicos de Scala: Sintaxis y Estructura ►

Configurar el entorno de desarrollo es el primer paso para comenzar a programar en Scala. Esta sección te guiará a través del proceso de instalación de las herramientas necesarias y la configuración de tu sistema para escribir, compilar y ejecutar programas en Scala de manera eficiente.

Requisitos Previos

Antes de configurar el entorno de desarrollo de Scala, asegúrate de tener lo siguiente:

  • Una computadora con un sistema operativo (Windows, macOS o Linux).
  • Conexión a Internet para descargar el software necesario.

Instalación del Kit de Desarrollo de Java (JDK)

Scala se ejecuta en la Máquina Virtual de Java (JVM), por lo que necesitas instalar el JDK.

  1. Descargar JDK:

    • Visita la página de descarga de Oracle JDK o utiliza una alternativa de código abierto como OpenJDK.
    • Elige la versión compatible con tu sistema operativo.

  2. Instalar JDK:

    • Sigue las instrucciones de instalación proporcionadas en la página de descarga.
    • Configura la variable de entorno JAVA_HOME para que apunte al directorio de instalación del JDK.

Verificación de la Instalación del JDK

Abre una terminal o símbolo del sistema y escribe:

java -version

Deberías ver una salida similar a:

java version "1.8.0_231"
Java(TM) SE Runtime Environment (build 1.8.0_231-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.231-b11, mixed mode)

Instalación de Scala

Scala se puede instalar utilizando varios métodos. Aquí, cubriremos los más comunes: usando el instalador oficial de Scala y usando SDKMAN.

Método 1: Usando el Instalador Oficial de Scala

  1. Descargar Scala:

    • Visita la página de descarga de Scala.
    • Elige el instalador apropiado para tu sistema operativo.

  2. Instalar Scala:

    • Ejecuta el instalador y sigue las instrucciones en pantalla.

Método 2: Usando SDKMAN

SDKMAN es una herramienta para gestionar versiones paralelas de múltiples Kits de Desarrollo de Software en la mayoría de los sistemas basados en Unix.

  1. Instalar SDKMAN:

    • Abre una terminal y ejecuta el siguiente comando: 
      curl -s "https://get.sdkman.io" | bash
    • Sigue las instrucciones para completar la instalación.

  2. Instalar Scala usando SDKMAN:

    • Abre una nueva terminal y ejecuta: 
      sdk install scala

Verificación de la Instalación de Scala

Abre una terminal o símbolo del sistema y escribe:

scala -version

Deberías ver una salida similar a:

Scala code runner version 2.13.3 -- Copyright 2002-2020, LAMP/EPFL

Instalación de un Entorno de Desarrollo Integrado (IDE)

Un IDE puede mejorar enormemente tu productividad al proporcionar características como autocompletado de código, depuración y gestión de proyectos. Aquí hay algunos IDEs populares para Scala:

  • IntelliJ IDEA:

    • Descárgalo desde el sitio web de JetBrains.
    • Instala el plugin de Scala a través del gestor de plugins.

  • Visual Studio Code:

    • Descárgalo desde el sitio web de Visual Studio Code.
    • Instala la extensión Metals para Scala.

Configuración de IntelliJ IDEA para Scala

  1. Instalar IntelliJ IDEA:

    • Sigue las instrucciones de instalación en el sitio web de JetBrains.

  2. Instalar el Plugin de Scala:

    • Abre IntelliJ IDEA.
    • Ve a File > Settings > Plugins.
    • Busca "Scala" e instala el plugin.

  3. Crear un Proyecto Scala:

    • Ve a File > New > Project.
    • Selecciona Scala de la lista y sigue las indicaciones para configurar tu proyecto.

Configuración de Visual Studio Code para Scala

  1. Instalar Visual Studio Code:

    • Sigue las instrucciones de instalación en el sitio web de Visual Studio Code.

  2. Instalar la Extensión Metals:

    • Abre Visual Studio Code.
    • Ve a la vista de Extensiones haciendo clic en el icono de Extensiones en la Barra de Actividades en el lado de la ventana.
    • Busca "Metals" e instala la extensión.

  3. Crear un Proyecto Scala:

    • Abre la terminal en Visual Studio Code.
    • Usa sbt (Scala Build Tool) para crear un nuevo proyecto: 
      sbt new scala/hello-world.g8

Conclusión

Configurar el entorno de desarrollo es un paso crucial en tu viaje de programación en Scala. Siguiendo los pasos descritos en esta sección, deberías tener un entorno de desarrollo de Scala completamente funcional. Ya sea que elijas IntelliJ IDEA o Visual Studio Code, asegúrate de que tus herramientas estén correctamente configuradas para maximizar tu productividad. Ahora que tu entorno está listo, puedes comenzar a escribir y ejecutar programas en Scala.
