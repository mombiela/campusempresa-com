Introducción a Scala

Scala es un poderoso lenguaje de programación que combina paradigmas de programación orientada a objetos y funcional. Está diseñado para ser conciso, elegante y seguro en tipos, lo que lo convierte en una excelente opción tanto para principiantes como para desarrolladores experimentados.

Empezando con Scala

  • Instalación: Cómo instalar Scala en diferentes sistemas operativos.
  • Scala REPL: Introducción al Read-Eval-Print Loop (REPL) de Scala para programación interactiva.
  • Hola Mundo: Escribir y ejecutar tu primer programa en Scala.
object HelloWorld {
  def main(args: Array[String]): Unit = {
    println("¡Hola, Mundo!")
  }
}

Sintaxis Básica y Tipos de Datos

  • Variables y Valores: Entendiendo var y val.
  • Tipos de Datos Básicos: Int, Double, String, Boolean, etc.
  • Inferencia de Tipos: Dejando que Scala infiera los tipos por ti.
val immutableValue: Int = 10
var mutableVariable: String = "Hola"
mutableVariable = "Mundo"

Estructuras de Control

  • Declaraciones Condicionales: if, else if, else.
  • Bucles: for, while, do-while.
  • Coincidencia de Patrones: Una característica poderosa para el flujo de control.
val number = 10
if (number > 0) {
  println("Positivo")
} else if (number < 0) {
  println("Negativo")
} else {
  println("Cero")
}

for (i <- 1 to 5) {
  println(i)
}

number match {
  case 1 => println("Uno")
  case 2 => println("Dos")
  case _ => println("Otro")
}

Funciones y Métodos

  • Definiendo Funciones: Sintaxis y uso.
  • Funciones de Orden Superior: Funciones que toman otras funciones como parámetros.
  • Funciones Anónimas: Expresiones lambda.
def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)

val multiply = (x: Int, y: Int) => x * y
val product = multiply(4, 2)

Colecciones

  • Listas: Secuencias inmutables.
  • Arrays: Secuencias mutables.
  • Mapas: Pares clave-valor.
  • Tuplas: Colecciones de tamaño fijo de elementos.
val list = List(1, 2, 3, 4, 5)
val array = Array(1, 2, 3, 4, 5)
val map = Map("a" -> 1, "b" -> 2)
val tuple = (1, "Scala", true)

Programación Orientada a Objetos

  • Clases y Objetos: Definiendo y usando clases.
  • Herencia: Extendiendo clases.
  • Traits: Similar a las interfaces en otros lenguajes.
class Animal {
  def speak(): Unit = {
    println("El animal habla")
  }
}

class Dog extends Animal {
  override def speak(): Unit = {
    println("El perro ladra")
  }
}

val dog = new Dog()
dog.speak()

trait Friendly {
  def greet(): Unit = {
    println("¡Hola!")
  }
}

class Person extends Friendly

val person = new Person()
person.greet()

Programación Funcional

  • Inmutabilidad: Beneficios y uso.
  • Funciones Puras: Funciones sin efectos secundarios.
  • Monadas: Entendiendo Option, Either y Future.
val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)

val maybeNumber: Option[Int] = Some(5)
val result = maybeNumber match {
  case Some(value) => value * 2
  case None => 0
}

Temas Avanzados

  • Conversiones Implícitas: Convertir tipos automáticamente.
  • Clases de Tipo: Polimorfismo ad-hoc.
  • Concurrencia: Usando Futures y Promises para programación asíncrona.
implicit def intToString(x: Int): String = x.toString

def printString(s: String): Unit = {
  println(s)
}

printString(123) // Convierte implícitamente Int a String

import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

val futureResult = Future {
  Thread.sleep(1000)
  42
}

futureResult.map(result => println(s"Resultado: $result"))

Conclusión

Scala es un lenguaje versátil que ofrece un rico conjunto de características tanto para la programación orientada a objetos como funcional. Al dominar los conceptos básicos y explorar temas avanzados, puedes escribir código eficiente, conciso y elegante. Practica regularmente y explora las extensas bibliotecas y frameworks de Scala para volverte competente en este poderoso lenguaje.
