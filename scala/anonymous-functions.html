Introducción

Las funciones anónimas, también conocidas como lambdas, son una característica poderosa en Scala que te permite definir funciones sin darles un nombre. Son particularmente útiles para funciones cortas y desechables que se usan solo una vez o unas pocas veces. En esta sección, exploraremos cómo crear y usar funciones anónimas en Scala, desde el uso básico hasta el avanzado.

Sintaxis Básica

  • Las funciones anónimas en Scala se definen usando el símbolo =>.
  • La sintaxis general es (parámetros) => expresión.

Ejemplo

val add = (x: Int, y: Int) => x + y
println(add(3, 4))  // Output: 7

En este ejemplo, add es una función anónima que toma dos enteros y devuelve su suma.

Uso de Funciones Anónimas con Colecciones

Las funciones anónimas se usan a menudo con métodos de colección como map, filter y reduce.

Ejemplo con `map`

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(x => x * 2)
println(doubled)  // Output: List(2, 4, 6, 8, 10)

Aquí, la función anónima x => x * 2 se aplica a cada elemento de la lista numbers.

Ejemplo con `filter`

val numbers = List(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter(x => x % 2 == 0)
println(evenNumbers)  // Output: List(2, 4)

En este ejemplo, la función anónima x => x % 2 == 0 filtra los números pares de la lista.

Simplificando Funciones Anónimas

Scala proporciona varias formas de simplificar las funciones anónimas, haciéndolas más concisas.

Sintaxis de Marcador de Posición

  • Puedes usar guiones bajos (_) como marcadores de posición para los parámetros.

Ejemplo

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)
println(doubled)  // Output: List(2, 4, 6, 8, 10)

Aquí, _ * 2 es una abreviatura de x => x * 2.

Inferencia de Tipos

  • Scala puede a menudo inferir los tipos de los parámetros, permitiéndote omitirlos.

Ejemplo

val add = (_: Int) + (_: Int)
println(add(3, 4))  // Output: 7

En este caso, los tipos de los parámetros se infieren del contexto.

Uso Avanzado

Las funciones anónimas pueden usarse en escenarios más complejos, como funciones de orden superior y currificación.

Funciones de Orden Superior

  • Funciones que toman otras funciones como parámetros o devuelven funciones como resultados.

Ejemplo

def applyOperation(x: Int, y: Int, operation: (Int, Int) => Int): Int = {
  operation(x, y)
}

val sum = applyOperation(3, 4, (a, b) => a + b)
println(sum)  // Output: 7

Aquí, applyOperation es una función de orden superior que toma una función anónima como parámetro.

Currificación

  • Transformar una función que toma múltiples argumentos en una serie de funciones que cada una toma un solo argumento.

Ejemplo

def curriedAdd(x: Int)(y: Int): Int = x + y
val addThree = curriedAdd(3) _
println(addThree(4))  // Output: 7

En este ejemplo, curriedAdd es una función currificada, y addThree es una función parcialmente aplicada.

Conclusión

Las funciones anónimas (lambdas) en Scala son una herramienta versátil que puede simplificar tu código y hacerlo más expresivo. Son particularmente útiles para funciones cortas y desechables y se usan comúnmente con métodos de colección. Al entender la sintaxis básica, las técnicas de simplificación y el uso avanzado, puedes aprovechar todo el poder de las funciones anónimas en tus programas Scala.
