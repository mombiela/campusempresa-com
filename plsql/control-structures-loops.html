Introducción

En PL/SQL, los bucles son estructuras de control esenciales que te permiten ejecutar una secuencia de sentencias múltiples veces. Este tema cubrirá los diferentes tipos de bucles disponibles en PL/SQL, cómo usarlos, y proporcionará ejemplos y ejercicios para ayudarte a entender su aplicación.

Tipos de Bucles en PL/SQL

PL/SQL soporta varios tipos de bucles:

  • Bucle Básico
  • Bucle WHILE
  • Bucle FOR
  • Bucle Cursor FOR

Bucle Básico

Un bucle básico ejecuta repetidamente un bloque de código hasta que se encuentra una sentencia EXIT.

Sintaxis

LOOP
    -- Sentencias a ejecutar
    EXIT WHEN condition;
END LOOP;

Ejemplo

DECLARE
    counter NUMBER := 1;
BEGIN
    LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
        counter := counter + 1;
        EXIT WHEN counter > 5;
    END LOOP;
END;
/

En este ejemplo, el bucle imprime el valor de counter y lo incrementa en 1 hasta que counter excede 5.

Bucle WHILE

Un bucle WHILE ejecuta un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

WHILE condition LOOP
    -- Sentencias a ejecutar
END LOOP;

Ejemplo

DECLARE
    counter NUMBER := 1;
BEGIN
    WHILE counter <= 5 LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
        counter := counter + 1;
    END LOOP;
END;
/

Este ejemplo es similar al bucle básico pero utiliza un bucle WHILE para verificar la condición antes de cada iteración.

Bucle FOR

Un bucle FOR itera sobre un rango especificado de valores.

Sintaxis

FOR counter IN lower_bound..upper_bound LOOP
    -- Sentencias a ejecutar
END LOOP;

Ejemplo

BEGIN
    FOR counter IN 1..5 LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
    END LOOP;
END;
/

En este ejemplo, el bucle incrementa automáticamente el counter de 1 a 5.

Bucle Cursor FOR

Un bucle cursor FOR itera sobre las filas devueltas por una consulta.

Sintaxis

FOR record IN cursor_name LOOP
    -- Sentencias a ejecutar
END LOOP;

Ejemplo

DECLARE
    CURSOR emp_cursor IS
        SELECT employee_id, first_name, last_name FROM employees;
BEGIN
    FOR emp_record IN emp_cursor LOOP
        DBMS_OUTPUT.PUT_LINE('Employee ID: ' || emp_record.employee_id || 
                             ', Name: ' || emp_record.first_name || ' ' || emp_record.last_name);
    END LOOP;
END;
/

Este ejemplo itera sobre cada fila en la tabla employees e imprime el ID y nombre del empleado.

Comparación de Tipos de Bucles

| Tipo de Bucle | Verificación de Condición | Caso de Uso | |-----------------|---------------------------|------------------------------------------------| | Bucle Básico | Dentro del bucle | Cuando el número de iteraciones no es conocido | | Bucle WHILE | Antes del bucle | Cuando el número de iteraciones no es conocido | | Bucle FOR | Antes del bucle | Cuando el número de iteraciones es conocido | | Bucle Cursor FOR| Antes del bucle | Al iterar sobre filas devueltas por una consulta|

Ejercicios

  1. Ejercicio de Bucle Básico: Escribe un bloque PL/SQL que use un bucle básico para imprimir los números del 10 al 1.
  2. Ejercicio de Bucle WHILE: Escribe un bloque PL/SQL que use un bucle WHILE para imprimir los primeros 10 números pares.
  3. Ejercicio de Bucle FOR: Escribe un bloque PL/SQL que use un bucle FOR para imprimir los cuadrados de los números del 1 al 5.
  4. Ejercicio de Bucle Cursor FOR: Escribe un bloque PL/SQL que use un bucle cursor FOR para imprimir los nombres de los departamentos de la tabla departments.

Conclusión

Los bucles son estructuras de control poderosas en PL/SQL que te permiten ejecutar un bloque de código múltiples veces. Entender los diferentes tipos de bucles y sus casos de uso te permitirá escribir código PL/SQL más eficiente y efectivo. Practica usando cada tipo de bucle con los ejercicios proporcionados para solidificar tu comprensión.
