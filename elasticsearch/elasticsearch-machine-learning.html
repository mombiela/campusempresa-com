Introducción a Elasticsearch con Machine Learning

Elasticsearch es un potente motor de búsqueda y análisis, mientras que el machine learning (ML) puede proporcionar conocimientos predictivos y detección de anomalías. La combinación de estos dos puede mejorar significativamente las capacidades de análisis de datos.

  • Elasticsearch: Un motor de búsqueda y análisis distribuido y RESTful.
  • Machine Learning: Algoritmos y modelos estadísticos que permiten a las computadoras realizar tareas sin instrucciones explícitas.

Configuración de Elasticsearch para Machine Learning

Antes de sumergirse en el machine learning con Elasticsearch, asegúrese de tener un clúster de Elasticsearch en funcionamiento con los complementos necesarios.

Requisitos previos

  • Elasticsearch instalado y en funcionamiento.
  • X-Pack instalado (para funciones de machine learning).

Instalación

# Descargar e instalar Elasticsearch
wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-7.10.0-linux-x86_64.tar.gz
tar -xzf elasticsearch-7.10.0-linux-x86_64.tar.gz
cd elasticsearch-7.10.0

# Iniciar Elasticsearch
./bin/elasticsearch

Preparación de datos

Los modelos de machine learning requieren datos bien preparados. Elasticsearch proporciona herramientas para ingerir, transformar y almacenar datos de manera eficiente.

Ingesta de datos

Use la API Bulk para ingerir grandes conjuntos de datos.

POST _bulk
{ "index" : { "_index" : "my_index", "_id" : "1" } }
{ "field1" : "value1", "field2" : "value2" }
{ "index" : { "_index" : "my_index", "_id" : "2" } }
{ "field1" : "value3", "field2" : "value4" }

Transformación de datos

Use Ingest Pipelines para preprocesar datos.

PUT _ingest/pipeline/my_pipeline
{
  "description" : "describe pipeline",
  "processors" : [
    {
      "set" : {
        "field": "field3",
        "value": "new_value"
      }
    }
  ]
}

Trabajos de Machine Learning en Elasticsearch

Elasticsearch le permite crear varios trabajos de machine learning, como detección de anomalías y pronósticos.

Detección de anomalías

Detecte patrones inusuales en sus datos.

Creación de un trabajo de detección de anomalías

POST _ml/anomaly_detectors/my_job
{
  "description": "Trabajo de detección de anomalías para mis datos",
  "analysis_config": {
    "bucket_span": "15m",
    "detectors": [
      {
        "function": "mean",
        "field_name": "field1"
      }
    ]
  },
  "data_description": {
    "time_field": "timestamp"
  }
}

Pronósticos

Prediga valores futuros basados en datos históricos.

Creación de un trabajo de pronóstico

POST _ml/anomaly_detectors/my_job/_forecast
{
  "duration": "7d"
}

Visualización de resultados de Machine Learning

Elasticsearch se integra con Kibana para visualizar los resultados de machine learning.

Configuración de Kibana

Instale e inicie Kibana.

# Descargar e instalar Kibana
wget https://artifacts.elastic.co/downloads/kibana/kibana-7.10.0-linux-x86_64.tar.gz
tar -xzf kibana-7.10.0-linux-x86_64.tar.gz
cd kibana-7.10.0

# Iniciar Kibana
./bin/kibana

Creación de visualizaciones

Use Kibana para crear visualizaciones para sus trabajos de machine learning.

  • Anomaly Explorer: Visualice las anomalías detectadas por sus trabajos.
  • Single Metric Viewer: Vea datos de una sola métrica y pronósticos.

Temas avanzados

Modelos de Machine Learning personalizados

Integre modelos de machine learning personalizados utilizando las capacidades de inferencia de Elasticsearch.

Subida de un modelo

PUT _ml/trained_models/my_model
{
  "description": "Modelo personalizado para clasificación",
  "input": {
    "field_names": ["field1", "field2"]
  },
  "definition": {
    "preprocessors": [],
    "trained_model": {
      "tree": {
        "feature_names": ["field1", "field2"],
        "tree_structure": [
          {
            "node_index": 0,
            "split_feature": 0,
            "split_gain": 0.1,
            "threshold": 0.5,
            "decision_type": "lte",
            "left_child": 1,
            "right_child": 2
          },
          {
            "node_index": 1,
            "leaf_value": 0.1
          },
          {
            "node_index": 2,
            "leaf_value": -0.1
          }
        ]
      }
    }
  }
}

Conclusión

Combinar Elasticsearch con machine learning puede mejorar significativamente sus capacidades de análisis de datos.
