Operaciones Básicas con Tensores
Autograd: Diferenciación Automática ►

Introducción

En esta sección, cubriremos las operaciones fundamentales que puedes realizar en tensores usando PyTorch. Los tensores son la estructura de datos central en PyTorch, similar a los arrays en NumPy pero con capacidades adicionales para la aceleración por GPU. Entender estas operaciones básicas es crucial para construir modelos de aprendizaje automático más complejos.

Creación de Tensores

Antes de realizar operaciones, necesitas saber cómo crear tensores. PyTorch proporciona varios métodos para crear tensores:

import torch

# Creando un tensor a partir de una lista
tensor_from_list = torch.tensor([1, 2, 3, 4])

# Creando un tensor con valores aleatorios
random_tensor = torch.rand(3, 3)

# Creando un tensor lleno de ceros
zeros_tensor = torch.zeros(2, 2)

# Creando un tensor lleno de unos
ones_tensor = torch.ones(2, 2)
  • torch.tensor(data): Crea un tensor a partir de datos (lista, tupla, etc.).
  • torch.rand(size): Crea un tensor con valores aleatorios distribuidos uniformemente entre 0 y 1.
  • torch.zeros(size): Crea un tensor lleno de ceros.
  • torch.ones(size): Crea un tensor lleno de unos.

Operaciones Básicas con Tensores

Operaciones Aritméticas

Puedes realizar operaciones aritméticas básicas en tensores tal como lo harías con arrays de NumPy.

a = torch.tensor([1, 2, 3])
b = torch.tensor([4, 5, 6])

# Adición
c = a + b  # o torch.add(a, b)

# Sustracción
d = a - b  # o torch.sub(a, b)

# Multiplicación
e = a * b  # o torch.mul(a, b)

# División
f = a / b  # o torch.div(a, b)

Operaciones de Matrices

Las operaciones de matrices son esenciales para muchos algoritmos de aprendizaje automático.

# Multiplicación de matrices
matrix1 = torch.tensor([[1, 2], [3, 4]])
matrix2 = torch.tensor([[5, 6], [7, 8]])

result = torch.mm(matrix1, matrix2)  # o matrix1 @ matrix2

Operaciones de Agregación

Las operaciones de agregación ayudan a resumir datos.

tensor = torch.tensor([1, 2, 3, 4, 5])

# Suma
sum_result = torch.sum(tensor)

# Media
mean_result = torch.mean(tensor.float())  # Convertir a float para el cálculo de la media

# Máximo
max_result = torch.max(tensor)

# Mínimo
min_result = torch.min(tensor)

Cambio de Forma de Tensores

Cambiar la forma de los tensores es a menudo necesario para preparar datos para los modelos.

tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])

# Cambiar forma a un tensor 3x2
reshaped_tensor = tensor.view(3, 2)

# Aplanar el tensor
flattened_tensor = tensor.view(-1)

Corte e Indexación

Al igual que los arrays de NumPy, puedes cortar e indexar tensores.

tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])

# Indexación
element = tensor[0, 1]  # Acceder al elemento en la fila 0, columna 1

# Corte
slice_tensor = tensor[:, 1]  # Acceder a todas las filas, columna 1

Comparación de Operaciones entre PyTorch y NumPy

| Operación | Sintaxis en PyTorch | Sintaxis en NumPy | |-----------------|----------------------------|----------------------------| | Adición | a + b o torch.add(a, b)| a + b o np.add(a, b) | | Sustracción | a - b o torch.sub(a, b)| a - b o np.subtract(a, b)| | Multiplicación | a * b o torch.mul(a, b)| a * b o np.multiply(a, b)| | División | a / b o torch.div(a, b)| a / b o np.divide(a, b)| | Multiplicación de Matrices | torch.mm(a, b) o a @ b | np.dot(a, b) o a @ b | | Cambio de Forma | a.view(new_shape) | a.reshape(new_shape) | | Suma | torch.sum(a) | np.sum(a) | | Media | torch.mean(a) | np.mean(a) |

Conclusión

En esta sección, cubrimos los conceptos básicos de las operaciones con tensores en PyTorch, incluyendo la creación de tensores, la realización de operaciones aritméticas y de matrices, la agregación, el cambio de forma y el corte. Estas operaciones forman la base para temas más avanzados en PyTorch y son esenciales para cualquier tarea de aprendizaje automático o aprendizaje profundo. Asegúrate de practicar estas operaciones para familiarizarte con la manipulación de tensores, ya que son una parte crítica del trabajo con PyTorch.
