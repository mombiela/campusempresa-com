Introducción al Aprendizaje por Refuerzo

El Aprendizaje por Refuerzo (RL) es un tipo de aprendizaje automático donde un agente aprende a tomar decisiones realizando acciones en un entorno para maximizar la recompensa acumulada.

Conceptos Clave

  • Agente: El aprendiz o tomador de decisiones.
  • Entorno: El sistema externo con el que interactúa el agente.
  • Estado: Una representación de la situación actual del agente.
  • Acción: Un conjunto de todos los movimientos posibles que el agente puede realizar.
  • Recompensa: Retroalimentación del entorno para evaluar la acción.
  • Política: Una estrategia utilizada por el agente para decidir acciones basadas en el estado.
  • Función de Valor: Una predicción de recompensas futuras.

Configuración del Entorno

Antes de sumergirse en RL, asegúrese de tener PyTorch instalado. Puede instalar PyTorch usando el siguiente comando:

pip install torch torchvision

Construyendo un Modelo RL Simple con PyTorch

En esta sección, construiremos un modelo RL simple usando PyTorch. Usaremos el entorno OpenAI Gym para este propósito.

Paso 1: Instalación de OpenAI Gym

Primero, instale la biblioteca OpenAI Gym:

pip install gym

Paso 2: Creación del Entorno

Vamos a crear un entorno simple usando OpenAI Gym:

import gym

env = gym.make('CartPole-v1')
state = env.reset()
print(state)

Paso 3: Definición de la Red Neuronal

Definiremos una red neuronal simple para aproximar la función de valor Q.

import torch
import torch.nn as nn
import torch.optim as optim

class QNetwork(nn.Module):
    def __init__(self, state_size, action_size):
        super(QNetwork, self).__init__()
        self.fc1 = nn.Linear(state_size, 24)
        self.fc2 = nn.Linear(24, 24)
        self.fc3 = nn.Linear(24, action_size)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = self.fc3(x)
        return x

Paso 4: Entrenamiento del Agente

Ahora entrenaremos al agente usando el algoritmo de aprendizaje Q.

import numpy as np

def train(env, q_network, episodes=1000, gamma=0.99, epsilon=0.1, lr=0.001):
    optimizer = optim.Adam(q_network.parameters(), lr=lr)
    criterion = nn.MSELoss()

    for episode in range(episodes):
        state = env.reset()
        done = False
        total_reward = 0

        while not done:
            state_tensor = torch.FloatTensor(state).unsqueeze(0)
            q_values = q_network(state_tensor)
            if np.random.rand() < epsilon:
                action = env.action_space.sample()
            else:
                action = torch.argmax(q_values).item()

            next_state, reward, done, _ = env.step(action)
            total_reward += reward

            next_state_tensor = torch.FloatTensor(next_state).unsqueeze(0)
            next_q_values = q_network(next_state_tensor)
            max_next_q_value = torch.max(next_q_values).item()

            target_q_value = reward + (gamma * max_next_q_value * (1 - done))
            target_q_values = q_values.clone()
            target_q_values[0][action] = target_q_value

            loss = criterion(q_values, target_q_values)
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            state = next_state

        print(f"Episodio {episode + 1}: Recompensa Total: {total_reward}")

# Inicializar y entrenar la red Q
state_size = env.observation_space.shape[0]
action_size = env.action_space.n
q_network = QNetwork(state_size, action_size)
train(env, q_network)

Conclusión

En este tema, cubrimos los conceptos básicos del aprendizaje por refuerzo e implementamos un agente de aprendizaje Q simple usando PyTorch. Comenzamos configurando el entorno, definiendo la red neuronal y luego entrenando al agente. Este conocimiento fundamental le ayudará a comprender algoritmos y proyectos de RL más complejos en el futuro.
