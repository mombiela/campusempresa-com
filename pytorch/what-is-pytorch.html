¿Qué es PyTorch?
Configurando tu Entorno ►

Introducción

PyTorch es una biblioteca de aprendizaje automático de código abierto desarrollada por el laboratorio de Investigación en IA de Facebook (FAIR). Es ampliamente utilizada para aplicaciones como el procesamiento del lenguaje natural y la visión por computadora. PyTorch proporciona una plataforma flexible e intuitiva para construir modelos de aprendizaje profundo, lo que la convierte en una opción popular entre investigadores y desarrolladores.

Características Clave de PyTorch

  • Grafos de Computación Dinámicos: PyTorch utiliza grafos de computación dinámicos, lo que significa que el grafo se construye sobre la marcha a medida que se ejecutan las operaciones. Esto proporciona mayor flexibilidad y facilidad de depuración.
  • Cálculo de Tensores: PyTorch ofrece una robusta biblioteca de arreglos N-dimensionales (tensores), similar a NumPy, pero con aceleración por GPU.
  • Diferenciación Automática: El módulo autograd de PyTorch calcula automáticamente los gradientes, lo cual es esencial para entrenar redes neuronales.
  • Ecosistema Rico: PyTorch tiene un ecosistema rico de herramientas y bibliotecas para diversas tareas de aprendizaje automático, incluyendo TorchVision para visión por computadora, TorchText para procesamiento del lenguaje natural, y más.
  • Comunidad y Soporte: PyTorch tiene una comunidad grande y activa, con documentación extensa y numerosos tutoriales disponibles.

Instalación de PyTorch

Para comenzar con PyTorch, necesitas instalarlo. Puedes instalar PyTorch usando pip o conda. Aquí hay un ejemplo usando pip:

pip install torch torchvision torchaudio

Para usuarios de conda:

conda install pytorch torchvision torchaudio -c pytorch

Conceptos Básicos

Tensores

Los tensores son los bloques fundamentales en PyTorch. Son similares a los arreglos de NumPy pero también pueden ser utilizados en una GPU para acelerar el cálculo.

Creación de Tensores

Aquí hay algunos ejemplos de creación de tensores en PyTorch:

import torch

# Crear un tensor a partir de una lista
tensor_from_list = torch.tensor([1, 2, 3, 4])
print(tensor_from_list)

# Crear un tensor con valores aleatorios
random_tensor = torch.rand(3, 3)
print(random_tensor)

# Crear un tensor lleno de ceros
zeros_tensor = torch.zeros(2, 2)
print(zeros_tensor)

Diferenciación Automática

El módulo autograd de PyTorch proporciona diferenciación automática para todas las operaciones en tensores. Esto es particularmente útil para la retropropagación en redes neuronales.

Ejemplo de Diferenciación Automática

import torch

# Crear un tensor y habilitar el cálculo de gradientes
x = torch.tensor(2.0, requires_grad=True)

# Realizar algunas operaciones
y = x ** 2
z = y + 3

# Calcular los gradientes
z.backward()

# Imprimir el gradiente de x
print(x.grad)  # Salida: tensor(4.0)

Construyendo una Red Neuronal Simple

Vamos a construir una red neuronal simple usando PyTorch para entender cómo se integra todo.

Ejemplo: Modelo de Regresión Lineal

import torch
import torch.nn as nn
import torch.optim as optim

# Definir el modelo
class LinearRegressionModel(nn.Module):
    def __init__(self):
        super(LinearRegressionModel, self).__init__()
        self.linear = nn.Linear(1, 1)

    def forward(self, x):
        return self.linear(x)

# Crear el modelo, definir la función de pérdida y el optimizador
model = LinearRegressionModel()
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# Datos de entrenamiento
x_train = torch.tensor([[1.0], [2.0], [3.0], [4.0]])
y_train = torch.tensor([[2.0], [4.0], [6.0], [8.0]])

# Bucle de entrenamiento
for epoch in range(100):
    # Paso hacia adelante
    outputs = model(x_train)
    loss = criterion(outputs, y_train)

    # Paso hacia atrás y optimización
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

    if (epoch+1) % 10 == 0:
        print(f'Epoch [{epoch+1}/100], Loss: {loss.item():.4f}')

# Probar el modelo
with torch.no_grad():
    predicted = model(torch.tensor([[5.0]]))
    print(f'Predicción para la entrada 5.0: {predicted.item():.4f}')

Conclusión

PyTorch es un marco de aprendizaje profundo poderoso y flexible que es fácil de usar y entender. Su grafo de computación dinámico, capacidades de cálculo de tensores y diferenciación automática lo convierten en una excelente opción tanto para principiantes como para usuarios avanzados. Al entender los conceptos básicos de PyTorch, como los tensores y la diferenciación automática, puedes comenzar a construir y entrenar tus propias redes neuronales. A medida que avances en este curso, aprenderás técnicas y aplicaciones más avanzadas de PyTorch.
