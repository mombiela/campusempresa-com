Carga y Preprocesamiento de Datos
Conceptos Básicos del Bucle de Entrenamiento ►

La carga y el preprocesamiento de datos son pasos cruciales en cualquier pipeline de aprendizaje automático. En esta sección, cubriremos cómo cargar y preprocesar datos de manera eficiente usando PyTorch. Esto incluye entender los conjuntos de datos, los cargadores de datos y varias técnicas de preprocesamiento.

Introducción a la Carga de Datos

En PyTorch, la carga de datos se maneja mediante el módulo torch.utils.data. Este módulo proporciona dos clases principales:

  • Dataset: Una clase que representa un conjunto de datos.
  • DataLoader: Una clase que carga datos de un Dataset y proporciona un iterable sobre el conjunto de datos.

Clase Dataset

La clase Dataset es una clase abstracta que representa un conjunto de datos. Para crear un conjunto de datos personalizado, necesitas subclasificar Dataset y sobrescribir los siguientes métodos:

  • __len__: Devuelve el tamaño del conjunto de datos.
  • __getitem__: Recupera una muestra del conjunto de datos en el índice dado.

Ejemplo: Conjunto de Datos Personalizado

import torch
from torch.utils.data import Dataset

class CustomDataset(Dataset):
    def __init__(self, data, labels):
        self.data = data
        self.labels = labels

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        sample = {'data': self.data[idx], 'label': self.labels[idx]}
        return sample

# Ejemplo de uso
data = torch.randn(100, 3, 32, 32)  # 100 muestras de imágenes de 3x32x32
labels = torch.randint(0, 10, (100,))  # 100 etiquetas para 10 clases

dataset = CustomDataset(data, labels)
print(len(dataset))  # Salida: 100
print(dataset[0])  # Salida: {'data': tensor(...), 'label': tensor(...)}

Clase DataLoader

La clase DataLoader proporciona un iterable sobre un conjunto de datos con soporte para batching, shuffling y carga de datos en paralelo.

Ejemplo: Uso de DataLoader

from torch.utils.data import DataLoader

dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

for batch in dataloader:
    print(batch['data'].shape, batch['label'])

Técnicas de Preprocesamiento

El preprocesamiento es esencial para asegurar que los datos estén en el formato correcto y normalizados para el entrenamiento. PyTorch proporciona varias utilidades para el preprocesamiento, especialmente para datos de imágenes.

Pasos Comunes de Preprocesamiento

  • Normalización: Ajustar el rango de valores de los píxeles.
  • Redimensionamiento: Cambiar las dimensiones de las imágenes.
  • Aumento de Datos: Técnicas como volteo, rotación y recorte para aumentar artificialmente el tamaño del conjunto de datos.

Uso de `torchvision.transforms`

El módulo torchvision.transforms proporciona varias operaciones comunes de preprocesamiento.

Ejemplo: Preprocesamiento de Imágenes

from torchvision import transforms
from PIL import Image

# Definir una serie de transformaciones
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.RandomHorizontalFlip(),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

# Cargar una imagen
image = Image.open('path/to/image.jpg')

# Aplicar transformaciones
transformed_image = transform(image)
print(transformed_image.shape)  # Salida: torch.Size([3, 128, 128])

Integración de Transformaciones con Dataset

Puedes integrar estas transformaciones directamente en tu conjunto de datos personalizado.

Ejemplo: Conjunto de Datos Personalizado con Transformaciones

class CustomDataset(Dataset):
    def __init__(self, data, labels, transform=None):
        self.data = data
        self.labels = labels
        self.transform = transform

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        sample = {'data': self.data[idx], 'label': self.labels[idx]}
        if self.transform:
            sample['data'] = self.transform(sample['data'])
        return sample

# Ejemplo de uso
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

dataset = CustomDataset(data, labels, transform=transform)
dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

for batch in dataloader:
    print(batch['data'].shape, batch['label'])

Conclusión

En esta sección, cubrimos los conceptos básicos de la carga y el preprocesamiento de datos en PyTorch. Aprendimos cómo crear conjuntos de datos personalizados usando la clase Dataset, cómo usar la clase DataLoader para cargar datos de manera eficiente y cómo aplicar técnicas comunes de preprocesamiento usando torchvision.transforms. Estas habilidades son fundamentales para preparar datos para el entrenamiento de modelos de aprendizaje automático en PyTorch.
