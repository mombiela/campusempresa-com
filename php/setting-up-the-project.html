Introducción

Configurar un proyecto PHP es el primer paso hacia el desarrollo de una aplicación web. Esta sección te guiará a través del proceso de configuración de un entorno de desarrollo PHP, la creación de una estructura de proyecto y la ejecución de tu primer script PHP.

Conceptos Clave

  • Instalación de PHP: Instalación de PHP en tu máquina local.
  • Servidor Web: Configuración de un servidor web como Apache o Nginx.
  • Base de Datos: Configuración de un servidor de base de datos como MySQL o MariaDB.
  • IDE/Editor de Código: Elección de un Entorno de Desarrollo Integrado (IDE) o editor de código.
  • Estructura del Proyecto: Organización de tus archivos y directorios del proyecto.

Instalación de PHP

Para ejecutar scripts PHP, necesitas tener PHP instalado en tu máquina. Sigue estos pasos para instalar PHP:

Windows

  1. Descarga la última versión de PHP desde el sitio web oficial de PHP.
  2. Extrae el archivo descargado en un directorio, por ejemplo, C:\php.
  3. Añade el directorio de PHP a la variable de entorno PATH de tu sistema.

macOS

  1. Abre Terminal.
  2. Instala PHP usando Homebrew: 
    brew install php

Linux (Ubuntu)

  1. Abre Terminal.
  2. Actualiza tu lista de paquetes: 
    sudo apt update
  3. Instala PHP: 
    sudo apt install php

Configuración de un Servidor Web

Se requiere un servidor web para servir tus archivos PHP. Puedes usar Apache o Nginx.

Apache

  • Windows: Instala XAMPP o WAMP, que incluyen Apache, PHP y MySQL.
  • macOS: Instala MAMP, que incluye Apache, PHP y MySQL.
  • Linux: 
    sudo apt install apache2
sudo apt install libapache2-mod-php

Nginx

  • Windows: Instala Nginx desde el sitio web oficial.
  • macOS: Instala Nginx usando Homebrew: 
    brew install nginx
  • Linux: 
    sudo apt install nginx

Configuración de una Base de Datos

La mayoría de las aplicaciones PHP requieren una base de datos. MySQL es una opción popular.

Instalación de MySQL

  • Windows: Instala MySQL desde el sitio web oficial.
  • macOS: Instala MySQL usando Homebrew: 
    brew install mysql
  • Linux: 
    sudo apt install mysql-server

Elección de un IDE/Editor de Código

Elegir el IDE o editor de código adecuado puede mejorar tu productividad. Aquí hay algunas opciones populares:

  • Visual Studio Code: Un editor de código gratuito y de código abierto con extensiones para PHP.
  • PHPStorm: Un IDE comercial diseñado específicamente para el desarrollo en PHP.
  • Sublime Text: Un editor de código ligero y rápido con soporte para PHP.

Creación de una Estructura de Proyecto

Una estructura de proyecto bien organizada hace que tu código sea más mantenible. Aquí tienes una estructura básica para un proyecto PHP:

mi_proyecto/
├── public/
│   └── index.php
├── src/
│   └── main.php
├── tests/
│   └── main_test.php
├── vendor/
├── composer.json
└── README.md
  • public/: Contiene archivos accesibles públicamente como index.php.
  • src/: Contiene tu código fuente PHP.
  • tests/: Contiene tus scripts de prueba.
  • vendor/: Contiene bibliotecas de terceros (gestionadas por Composer).
  • composer.json: Archivo de configuración de Composer.
  • README.md: Documentación del proyecto.

Ejecución de tu Primer Script PHP

Vamos a crear un script PHP simple para asegurarnos de que todo esté configurado correctamente.

  1. Crea un archivo llamado index.php en el directorio public/.
  2. Añade el siguiente código: 
    <?php
echo "¡Hola, Mundo!";
?>
  3. Inicia tu servidor web y navega a http://localhost/index.php en tu navegador web. Deberías ver "¡Hola, Mundo!" en pantalla.

Conclusión

Configurar un proyecto PHP implica instalar PHP, configurar un servidor web, configurar una base de datos, elegir un IDE y organizar la estructura de tu proyecto. Siguiendo estos pasos, tendrás una base sólida para comenzar a desarrollar aplicaciones PHP.
