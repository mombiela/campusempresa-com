¿Qué es una Base de Datos?

Una base de datos es una colección organizada de datos, generalmente almacenada y accesada electrónicamente desde un sistema informático. Las bases de datos son esenciales para almacenar, gestionar y recuperar datos de manera eficiente.

Tipos de Bases de Datos

  • Bases de Datos Relacionales: Almacenan datos en tablas con filas y columnas. Ejemplos incluyen MySQL, PostgreSQL y SQLite.
  • Bases de Datos NoSQL: Almacenan datos en varios formatos como pares clave-valor, documentos, grafos o almacenes de columnas amplias. Ejemplos incluyen MongoDB, Redis y Cassandra.

¿Por qué Usar Bases de Datos con PHP?

  • Persistencia de Datos: Almacenar datos permanentemente.
  • Gestión de Datos: Gestionar eficientemente grandes cantidades de datos.
  • Recuperación de Datos: Recuperar datos rápidamente usando consultas.
  • Seguridad: Almacenar información sensible de manera segura.

Configuración de una Base de Datos

Instalando MySQL

  1. Windows: Descarga e instala MySQL desde el sitio web oficial.
  2. macOS: Usa Homebrew: brew install mysql
  3. Linux: Usa tu gestor de paquetes, por ejemplo, sudo apt-get install mysql-server

Creando una Base de Datos

Una vez que MySQL está instalado, puedes crear una base de datos usando los siguientes comandos:

CREATE DATABASE mi_base_de_datos;

Conectando PHP a una Base de Datos

Usando MySQLi

MySQLi (MySQL Mejorado) es una extensión de PHP diseñada para trabajar con bases de datos MySQL.

Ejemplo: Conectando a una Base de Datos MySQL

<?php
$servername = "localhost";
$username = "usuario";
$password = "contraseña";
$dbname = "mi_base_de_datos";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}
echo "Conectado exitosamente";
?>

Usando PDO (PHP Data Objects)

PDO es una forma más flexible y segura de conectar a bases de datos en PHP.

Ejemplo: Conectando a una Base de Datos MySQL con PDO

<?php
$dsn = 'mysql:host=localhost;dbname=mi_base_de_datos';
$username = 'usuario';
$password = 'contraseña';

try {
    $pdo = new PDO($dsn, $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Conectado exitosamente";
} catch (PDOException $e) {
    echo "Conexión fallida: " . $e->getMessage();
}
?>

Realizando Operaciones CRUD

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las operaciones básicas que puedes realizar en una base de datos.

Crear

Ejemplo: Insertando Datos en una Tabla

<?php
$sql = "INSERT INTO usuarios (nombre_usuario, correo) VALUES ('juan_perez', 'juan@ejemplo.com')";
if ($conn->query($sql) === TRUE) {
    echo "Nuevo registro creado exitosamente";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}
?>

Leer

Ejemplo: Recuperando Datos de una Tabla

<?php
$sql = "SELECT id, nombre_usuario, correo FROM usuarios";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Nombre: " . $row["nombre_usuario"]. " - Correo: " . $row["correo"]. "<br>";
    }
} else {
    echo "0 resultados";
}
?>

Actualizar

Ejemplo: Actualizando Datos en una Tabla

<?php
$sql = "UPDATE usuarios SET correo='nuevo_correo@ejemplo.com' WHERE nombre_usuario='juan_perez'";
if ($conn->query($sql) === TRUE) {
    echo "Registro actualizado exitosamente";
} else {
    echo "Error actualizando registro: " . $conn->error;
}
?>

Eliminar

Ejemplo: Eliminando Datos de una Tabla

<?php
$sql = "DELETE FROM usuarios WHERE nombre_usuario='juan_perez'";
if ($conn->query($sql) === TRUE) {
    echo "Registro eliminado exitosamente";
} else {
    echo "Error eliminando registro: " . $conn->error;
}
?>

Conclusión

En esta sección, cubrimos los conceptos básicos de las bases de datos, incluyendo qué son, por qué son importantes y cómo conectarse a ellas usando PHP. También exploramos cómo realizar operaciones básicas CRUD. Entender estos fundamentos es crucial para cualquier desarrollador PHP, ya que las bases de datos son integrales para la mayoría de las aplicaciones web.
