Introducción a PHP

PHP (Hypertext Preprocessor) es un popular lenguaje de scripting del lado del servidor diseñado para el desarrollo web, pero también utilizado como un lenguaje de programación de propósito general. Esta sección cubrirá la sintaxis fundamental y los constructos básicos de PHP para que comiences a escribir scripts en PHP.

Sintaxis Básica de PHP

  • Etiquetas PHP: El código PHP se incrusta dentro del HTML usando etiquetas PHP. 
    <?php
// El código PHP va aquí
?>
  • Declaración Echo: Se utiliza para imprimir texto o variables. 
    <?php
echo "¡Hola, Mundo!";
?>

Variables y Tipos de Datos

  • Variables: Se declaran usando el símbolo $ seguido del nombre de la variable. 
    <?php
$saludo = "¡Hola, Mundo!";
echo $saludo;
?>
  • Tipos de Datos:
    • Cadena (String)
    • Entero (Integer)
    • Flotante (Float) (Doble)
    • Booleano (Boolean)
    • Matriz (Array)
    • Objeto (Object)
    • NULL

Ejemplo de Diferentes Tipos de Datos

<?php
$cadena = "¡Hola, Mundo!";
$entero = 42;
$flotante = 3.14;
$booleano = true;
$matriz = array("manzana", "plátano", "cereza");
$objeto = (object) ['propiedad' => 'valor'];
$nulo = NULL;

echo $cadena;
echo $entero;
echo $flotante;
echo $booleano;
print_r($matriz);
echo $objeto->propiedad;
echo $nulo;
?>

Operadores

  • Operadores Aritméticos: +, -, *, /, % 
    <?php
$suma = 5 + 3; // 8
$diferencia = 5 - 3; // 2
$producto = 5 * 3; // 15
$cociente = 5 / 3; // 1.6667
$resto = 5 % 3; // 2
?>
  • Operadores de Asignación: =, +=, -=, *=, /=, %= 
    <?php
$x = 10;
$x += 5; // 15
$x -= 3; // 12
$x *= 2; // 24
$x /= 4; // 6
$x %= 5; // 1
?>
  • Operadores de Comparación: ==, ===, !=, !==, >, <, >=, <= 
    <?php
$a = 5;
$b = 10;
var_dump($a == $b); // falso
var_dump($a === $b); // falso
var_dump($a != $b); // verdadero
var_dump($a !== $b); // verdadero
var_dump($a > $b); // falso
var_dump($a < $b); // verdadero
var_dump($a >= $b); // falso
var_dump($a <= $b); // verdadero
?>

Estructuras de Control

  • Declaración If-Else: Ejecución condicional de bloques de código. 
    <?php
$edad = 20;
if ($edad >= 18) {
    echo "Eres un adulto.";
} else {
    echo "Eres un menor.";
}
?>
  • Declaración Switch: Selecciona uno de muchos bloques de código para ser ejecutado. 
    <?php
$dia = "Lunes";
switch ($dia) {
    case "Lunes":
        echo "Inicio de la semana laboral.";
        break;
    case "Viernes":
        echo "Fin de la semana laboral.";
        break;
    default:
        echo "Días entre semana.";
}
?>

Bucles

  • Bucle While: Repite un bloque de código mientras la condición especificada sea verdadera. 
    <?php
$i = 0;
while ($i < 5) {
    echo $i;
    $i++;
}
?>
  • Bucle For: Recorre un bloque de código un número especificado de veces. 
    <?php
for ($i = 0; $i < 5; $i++) {
    echo $i;
}
?>
  • Bucle Foreach: Recorre cada par clave/valor en una matriz. 
    <?php
$frutas = array("manzana", "plátano", "cereza");
foreach ($frutas as $fruta) {
    echo $fruta;
}
?>

Funciones

  • Definiendo Funciones: Las funciones son bloques de código que pueden ser llamados repetidamente. 
    <?php
function saludar($nombre) {
    return "Hola, " . $nombre;
}
echo saludar("Alicia");
?>

Conclusión

En esta sección, cubrimos la sintaxis básica y los constructos de PHP, incluyendo variables, tipos de datos, operadores, estructuras de control, bucles y funciones. Estos fundamentos forman los bloques de construcción para una programación PHP más avanzada. Practica estos conceptos con los ejemplos proporcionados para solidificar tu comprensión antes de pasar a temas más complejos.
