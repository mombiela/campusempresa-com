Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en PHP. Entender estos conceptos es crucial para escribir código PHP eficiente y efectivo. Comenzaremos con los fundamentos de las variables, luego pasaremos a los diferentes tipos de datos disponibles en PHP.

Variables

¿Qué es una Variable?

  • Una variable es una ubicación de almacenamiento identificada por un nombre que contiene datos que pueden ser modificados durante la ejecución del programa.
  • En PHP, las variables se denotan con un signo de dólar ($) seguido del nombre de la variable.

Declarando Variables

  • Las variables en PHP se declaran asignándoles un valor usando el operador de asignación (=).
  • PHP es un lenguaje de tipado débil, lo que significa que no necesitas declarar el tipo de dato de una variable.
<?php
$nombreVariable = "¡Hola, Mundo!";
$numero = 42;
?>

Reglas para Nombrar Variables

  • Los nombres de las variables deben comenzar con una letra o un guion bajo (_).
  • Los nombres de las variables solo pueden contener caracteres alfanuméricos y guiones bajos (A-z, 0-9, y _).
  • Los nombres de las variables son sensibles a mayúsculas y minúsculas ($Variable y $variable son diferentes).

Ejemplos

<?php
$nombre = "Juan";
$edad = 30;
$esEstudiante = true;
?>

Tipos de Datos

Descripción General

PHP soporta varios tipos de datos, que pueden ser categorizados en tipos escalares, tipos compuestos y tipos especiales.

Tipos Escalares

  • Entero: Números enteros, positivos o negativos.
  • Flotante: Números con punto decimal o en forma exponencial.
  • Cadena: Una secuencia de caracteres.
  • Booleano: Representa dos estados posibles: TRUE o FALSE.

Tipos Compuestos

  • Array: Una colección de valores.
  • Objeto: Una instancia de una clase.

Tipos Especiales

  • NULL: Una variable sin valor.
  • Recurso: Una variable especial que contiene una referencia a un recurso externo (por ejemplo, conexión a una base de datos).

Entero

<?php
$intVar = 100;
echo $intVar; // Salida: 100
?>

Flotante

<?php
$floatVar = 10.5;
echo $floatVar; // Salida: 10.5
?>

Cadena

<?php
$stringVar = "¡Hola, PHP!";
echo $stringVar; // Salida: ¡Hola, PHP!
?>

Booleano

<?php
$boolVar = true;
echo $boolVar; // Salida: 1 (true se representa como 1)
?>

Array

<?php
$arrayVar = array("manzana", "banana", "cereza");
echo $arrayVar[0]; // Salida: manzana
?>

Objeto

<?php
class Coche {
    public $color;
    public function __construct($color) {
        $this->color = $color;
    }
}

$miCoche = new Coche("rojo");
echo $miCoche->color; // Salida: rojo
?>

NULL

<?php
$nullVar = NULL;
echo $nullVar; // Salida: (nada)
?>

Recurso

  • Los recursos son variables especiales que contienen referencias a recursos externos.
  • Comúnmente usados con conexiones a bases de datos, manejadores de archivos, etc.
<?php
$file = fopen("test.txt", "r");
echo $file; // Salida: Resource id #3
fclose($file);
?>

Manipulación de Tipos y Conversión de Tipos

Manipulación de Tipos

  • PHP convierte automáticamente las variables al tipo de dato correcto basado en su valor.
  • Esto se conoce como manipulación de tipos.
<?php
$var = "10";
$var = $var + 2;
echo $var; // Salida: 12 (la cadena "10" se convierte en entero 10)
?>

Conversión de Tipos

  • Puedes convertir explícitamente una variable a un tipo de dato diferente usando la conversión de tipos.
<?php
$var = "10";
$intVar = (int)$var;
echo $intVar; // Salida: 10
?>

Conclusión

En esta sección, hemos cubierto los conceptos básicos de variables y tipos de datos en PHP. Aprendimos cómo declarar variables, las reglas para nombrarlas y los diversos tipos de datos disponibles en PHP. Entender estos conceptos es fundamental para escribir código PHP efectivo. A medida que avances, verás cómo estos bloques de construcción básicos se utilizan para crear aplicaciones web más complejas y dinámicas.
