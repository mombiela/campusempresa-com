Operadores en PHP
Estructuras de Control: Condicionales ►

Introducción

Los operadores son caracteres o símbolos especiales en PHP que se utilizan para realizar operaciones en variables y valores. Son esenciales para manipular datos y controlar el flujo de un programa. Esta sección cubrirá los diferentes tipos de operadores disponibles en PHP, proporcionando explicaciones claras, ejemplos y ejercicios para ayudarte a entender su uso.

Tipos de Operadores

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas comunes.

  • Adición (+): Suma dos operandos.
  • Sustracción (-): Resta el segundo operando del primero.
  • Multiplicación (*): Multiplica dos operandos.
  • División (/): Divide el primer operando por el segundo.
  • Módulo (%): Devuelve el resto de la división del primer operando por el segundo.
  • Exponenciación (**): Eleva el primer operando a la potencia del segundo operando.
<?php
$a = 10;
$b = 3;

echo $a + $b; // Salida: 13
echo $a - $b; // Salida: 7
echo $a * $b; // Salida: 30
echo $a / $b; // Salida: 3.3333333333333
echo $a % $b; // Salida: 1
echo $a ** $b; // Salida: 1000
?>

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación Básica (=): Asigna el valor del operando derecho al operando izquierdo.
  • Asignación de Adición (+=): Suma el operando derecho al operando izquierdo y asigna el resultado al operando izquierdo.
  • Asignación de Sustracción (-=): Resta el operando derecho del operando izquierdo y asigna el resultado al operando izquierdo.
  • Asignación de Multiplicación (*=): Multiplica el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.
  • Asignación de División (/=): Divide el operando izquierdo por el operando derecho y asigna el resultado al operando izquierdo.
  • Asignación de Módulo (%=): Toma el módulo usando dos operandos y asigna el resultado al operando izquierdo.
<?php
$a = 10;
$b = 5;

$a += $b; // $a ahora es 15
$a -= $b; // $a ahora es 10
$a *= $b; // $a ahora es 50
$a /= $b; // $a ahora es 10
$a %= $b; // $a ahora es 0
?>

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores.

  • Igual (==): Devuelve verdadero si ambos operandos son iguales.
  • Idéntico (===): Devuelve verdadero si ambos operandos son iguales y del mismo tipo.
  • No Igual (!= o <>): Devuelve verdadero si los operandos no son iguales.
  • No Idéntico (!==): Devuelve verdadero si los operandos no son iguales o no son del mismo tipo.
  • Mayor Que (>): Devuelve verdadero si el operando izquierdo es mayor que el operando derecho.
  • Menor Que (<): Devuelve verdadero si el operando izquierdo es menor que el operando derecho.
  • Mayor o Igual Que (>=): Devuelve verdadero si el operando izquierdo es mayor o igual que el operando derecho.
  • Menor o Igual Que (<=): Devuelve verdadero si el operando izquierdo es menor o igual que el operando derecho.
<?php
$a = 10;
$b = 20;

var_dump($a == $b); // Salida: bool(false)
var_dump($a === $b); // Salida: bool(false)
var_dump($a != $b); // Salida: bool(true)
var_dump($a !== $b); // Salida: bool(true)
var_dump($a > $b); // Salida: bool(false)
var_dump($a < $b); // Salida: bool(true)
var_dump($a >= $b); // Salida: bool(false)
var_dump($a <= $b); // Salida: bool(true)
?>

Operadores Lógicos

Los operadores lógicos se utilizan para combinar declaraciones condicionales.

  • Y (&& o and): Devuelve verdadero si ambos operandos son verdaderos.
  • O (|| o or): Devuelve verdadero si al menos uno de los operandos es verdadero.
  • No (!): Devuelve verdadero si el operando es falso.
<?php
$a = true;
$b = false;

var_dump($a && $b); // Salida: bool(false)
var_dump($a || $b); // Salida: bool(true)
var_dump(!$a); // Salida: bool(false)
?>

Operadores de Incremento/Decremento

Los operadores de incremento y decremento se utilizan para aumentar o disminuir el valor de una variable en uno.

  • Incremento (++): Aumenta el valor de la variable en uno.
  • Decremento (--): Disminuye el valor de la variable en uno.
<?php
$a = 10;

echo ++$a; // Salida: 11 (Pre-incremento)
echo $a++; // Salida: 11 (Post-incremento)
echo $a;   // Salida: 12

echo --$a; // Salida: 11 (Pre-decremento)
echo $a--; // Salida: 11 (Post-decremento)
echo $a;   // Salida: 10
?>

Operadores de Cadenas

Los operadores de cadenas se utilizan para manipular valores de cadenas.

  • Concatenación (.): Concatenar dos cadenas.
  • Asignación de Concatenación (.=): Añade el operando derecho al operando izquierdo y asigna el resultado al operando izquierdo.
<?php
$a = "Hola";
$b = " Mundo";

echo $a . $b; // Salida: Hola Mundo

$a .= $b;
echo $a; // Salida: Hola Mundo
?>

Operadores de Arrays

Los operadores de arrays se utilizan para comparar arrays.

  • Unión (+): Fusiona dos arrays.
  • Igualdad (==): Devuelve verdadero si dos arrays tienen los mismos pares clave/valor.
  • Identidad (===): Devuelve verdadero si dos arrays tienen los mismos pares clave/valor en el mismo orden y del mismo tipo.
  • Desigualdad (!= o <>): Devuelve verdadero si dos arrays no son iguales.
  • No Identidad (!==): Devuelve verdadero si dos arrays no son idénticos.
<?php
$a = array("a" => "manzana", "b" => "banana");
$b = array("c" => "cereza", "d" => "dátil");

$c = $a + $b; // Unión de $a y $b
var_dump($c); // Salida: array(4) { ["a"]=> string(7) "manzana" ["b"]=> string(6) "banana" ["c"]=> string(6) "cereza" ["d"]=> string(5) "dátil" }

var_dump($a == $b); // Salida: bool(false)
var_dump($a === $b); // Salida: bool(false)
var_dump($a != $b); // Salida: bool(true)
var_dump($a !== $b); // Salida: bool(true)
?>

Conclusión

Los operadores son fundamentales en PHP para realizar diversas operaciones en datos. Entender cómo usar operadores aritméticos, de asignación, de comparación, lógicos, de incremento/decremento, de cadenas y de arrays mejorará significativamente tu capacidad para escribir código PHP eficiente y efectivo. Practica usando estos operadores en diferentes escenarios para familiarizarte más con su sintaxis y funcionalidad.
