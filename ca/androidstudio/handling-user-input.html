Gestió de l'Entrada d'Usuari
Comprenent les Activitats ►

La gestió de l'entrada d'usuari és un aspecte fonamental del desenvolupament d'aplicacions Android. Aquest tema et guiarà a través dels conceptes bàsics de capturar l'entrada de l'usuari, processar-la i respondre-hi a les teves aplicacions Android. Al final d'aquesta secció, seràs capaç de crear aplicacions interactives que responguin a les accions dels usuaris.

Introducció a l'Entrada d'Usuari

L'entrada d'usuari en Android pot provenir de diverses fonts com gestos tàctils, entrada de teclat i sensors. En aquesta secció, ens centrarem en els mètodes més comuns de gestió de l'entrada d'usuari: EditText per a l'entrada de text i Button per a capturar esdeveniments de clic.

Conceptes Clau

  • EditText: Un element d'interfície d'usuari que permet als usuaris introduir i editar text.
  • Button: Un element d'interfície d'usuari que els usuaris poden clicar per realitzar una acció.
  • OnClickListener: Una interfície utilitzada per definir l'acció que ocorre quan un usuari clica un botó.
  • TextWatcher: Una interfície utilitzada per monitoritzar els canvis en el text d'un EditText.

Captura de l'Entrada de Text amb EditText

EditText és un widget comunament utilitzat en Android per capturar l'entrada de text de l'usuari. Aquí tens com utilitzar-lo:

Exemple: Ús Bàsic d'EditText

<!-- activity_main.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <EditText
        android:id="@+id/editText"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:hint="Enter your text here" />

    <Button
        android:id="@+id/button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Submit" />
</LinearLayout>

Gestió de l'Entrada d'EditText a l'Activitat

// MainActivity.java
package com.example.userinput;

import android.os.Bundle;
import android.view.View;
import android.widget.Button;
import android.widget.EditText;
import android.widget.Toast;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {

    private EditText editText;
    private Button button;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        editText = findViewById(R.id.editText);
        button = findViewById(R.id.button);

        button.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                String userInput = editText.getText().toString();
                Toast.makeText(MainActivity.this, "You entered: " + userInput, Toast.LENGTH_SHORT).show();
            }
        });
    }
}

Explicació

  • EditText: El widget EditText està definit al disseny XML amb un ID de editText.
  • Button: El widget Button està definit al disseny XML amb un ID de button.
  • OnClickListener: A la MainActivity, establim un OnClickListener al botó per capturar l'esdeveniment de clic. Quan es clica el botó, el text de l'EditText es recupera i es mostra utilitzant un Toast.

Gestió Avançada de l'Entrada d'Usuari

Per a una gestió més complexa de l'entrada, pots utilitzar interfícies com TextWatcher per monitoritzar els canvis en l'EditText en temps real.

Exemple: Ús de TextWatcher

// MainActivity.java
package com.example.userinput;

import android.os.Bundle;
import android.text.Editable;
import android.text.TextWatcher;
import android.widget.EditText;
import android.widget.TextView;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {

    private EditText editText;
    private TextView textView;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        editText = findViewById(R.id.editText);
        textView = findViewById(R.id.textView);

        editText.addTextChangedListener(new TextWatcher() {
            @Override
            public void beforeTextChanged(CharSequence s, int start, int count, int after) {
                // Fer alguna cosa abans que el text canviï
            }

            @Override
            public void onTextChanged(CharSequence s, int start, int before, int count) {
                // Fer alguna cosa mentre el text està canviant
                textView.setText(s.toString());
            }

            @Override
            public void afterTextChanged(Editable s) {
                // Fer alguna cosa després que el text hagi canviat
            }
        });
    }
}

Explicació

  • TextWatcher: La interfície TextWatcher proporciona tres mètodes: beforeTextChanged, onTextChanged i afterTextChanged. Aquests mètodes et permeten respondre als canvis en l'EditText en temps real.
  • onTextChanged: En aquest exemple, actualitzem un TextView amb el text actual de l'EditText mentre s'està escrivint.

Conclusió

La gestió de l'entrada d'usuari és crucial per crear aplicacions Android interactives. Utilitzant widgets com EditText i Button, i interfícies com OnClickListener i TextWatcher, pots capturar i respondre a l'entrada d'usuari de manera efectiva. Practica aquests conceptes creant aplicacions senzilles que requereixin interacció de l'usuari, i gradualment passa a escenaris més complexos.
