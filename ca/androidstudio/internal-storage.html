L'emmagatzematge intern a Android es refereix a l'espai d'emmagatzematge privat disponible al dispositiu on una aplicació pot desar dades. Aquestes dades són privades per a l'aplicació i no poden ser accedides per altres aplicacions. Entendre com utilitzar efectivament l'emmagatzematge intern és crucial per desenvolupar aplicacions Android robustes.

Conceptes Clau

  • Emmagatzematge Intern: Espai d'emmagatzematge privat al dispositiu.
  • Gestió d'Arxius: Llegir i escriure arxius.
  • Privacitat de Dades: Assegurar que les dades només siguin accessibles per l'aplicació.

Escriure a l'Emmagatzematge Intern

Per escriure dades a l'emmagatzematge intern, pots utilitzar el mètode openFileOutput. Aquest mètode retorna un FileOutputStream que pots utilitzar per escriure dades.

Exemple: Escriure una Cadena a un Arxiu

String filename = "myfile.txt";
String fileContents = "Hola, Món!";
FileOutputStream fos = null;

try {
    fos = openFileOutput(filename, Context.MODE_PRIVATE);
    fos.write(fileContents.getBytes());
} catch (Exception e) {
    e.printStackTrace();
} finally {
    if (fos != null) {
        try {
            fos.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
  • Context.MODE_PRIVATE: Assegura que l'arxiu sigui privat per a l'aplicació.
  • openFileOutput: Obre un arxiu per escriure.

Llegir de l'Emmagatzematge Intern

Per llegir dades de l'emmagatzematge intern, pots utilitzar el mètode openFileInput. Aquest mètode retorna un FileInputStream que pots utilitzar per llegir dades.

Exemple: Llegir una Cadena d'un Arxiu

String filename = "myfile.txt";
FileInputStream fis = null;
StringBuilder stringBuilder = new StringBuilder();

try {
    fis = openFileInput(filename);
    int character;
    while ((character = fis.read()) != -1) {
        stringBuilder.append((char) character);
    }
} catch (Exception e) {
    e.printStackTrace();
} finally {
    if (fis != null) {
        try {
            fis.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

String fileContents = stringBuilder.toString();
  • openFileInput: Obre un arxiu per llegir.
  • StringBuilder: Construeix eficientment la cadena a partir del contingut de l'arxiu.

Esborrar un Arxiu

Per esborrar un arxiu de l'emmagatzematge intern, pots utilitzar el mètode deleteFile.

Exemple: Esborrar un Arxiu

String filename = "myfile.txt";
boolean deleted = deleteFile(filename);

if (deleted) {
    // Arxiu esborrat amb èxit
} else {
    // Error en esborrar l'arxiu
}
  • deleteFile: Esborra l'arxiu especificat.

Comprovar l'Existència d'un Arxiu

Per comprovar si un arxiu existeix a l'emmagatzematge intern, pots utilitzar el mètode fileList i cercar el nom de l'arxiu a l'array retornat.

Exemple: Comprovar si un Arxiu Existeix

String filename = "myfile.txt";
String[] files = fileList();
boolean fileExists = Arrays.asList(files).contains(filename);

if (fileExists) {
    // L'arxiu existeix
} else {
    // L'arxiu no existeix
}
  • fileList: Retorna un array de tots els arxius a l'emmagatzematge intern.

Resum

En aquesta secció, hem cobert els conceptes bàsics de l'emmagatzematge intern a Android:

  • Escriure a l'emmagatzematge intern utilitzant openFileOutput.
  • Llegir de l'emmagatzematge intern utilitzant openFileInput.
  • Esborrar arxius amb deleteFile.
  • Comprovar l'existència d'arxius amb fileList.

Entendre aquestes operacions és essencial per gestionar dades dins de les teves aplicacions Android. Dominant aquestes tècniques, pots assegurar-te que les dades de la teva aplicació es desin de manera segura i eficient.
