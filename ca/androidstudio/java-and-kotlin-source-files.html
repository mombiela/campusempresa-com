Introducció

En el desenvolupament d'Android, Java i Kotlin són els dos llenguatges de programació principals utilitzats. Entendre com crear i gestionar fitxers font en ambdós llenguatges és essencial per desenvolupar aplicacions Android. Aquesta secció et guiarà a través dels conceptes bàsics dels fitxers font de Java i Kotlin, les seves estructures i com treballar amb ells a Android Studio.

Fitxers Font de Java

Java ha estat el llenguatge tradicional per al desenvolupament d'Android. Aquí, cobrirem els conceptes bàsics dels fitxers font de Java, incloent la seva estructura i com crear-los a Android Studio.

Estructura d'un Fitxer Font de Java

Un fitxer font típic de Java inclou:

  • Declaració del paquet
  • Declaracions d'importació
  • Declaració de la classe
  • Mètodes i variables

Exemple:

package com.example.myapp;

import android.os.Bundle;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }
}

Creació d'un Fitxer Font de Java a Android Studio

  1. Obre Android Studio i navega al teu projecte.
  2. Clica amb el botó dret sobre el directori java a la vista Project.
  3. Selecciona New > Java Class.
  4. Introdueix el nom de la teva classe i clica OK.

Fitxers Font de Kotlin

Kotlin és un llenguatge modern, tipat estàticament, que és totalment interoperable amb Java. S'ha convertit en el llenguatge preferit per al desenvolupament d'Android a causa de la seva sintaxi concisa i característiques millorades.

Estructura d'un Fitxer Font de Kotlin

Un fitxer font típic de Kotlin inclou:

  • Declaració del paquet
  • Declaracions d'importació
  • Declaració de la classe
  • Mètodes i variables

Exemple:

package com.example.myapp

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}

Creació d'un Fitxer Font de Kotlin a Android Studio

  1. Obre Android Studio i navega al teu projecte.
  2. Clica amb el botó dret sobre el directori java a la vista Project (els fitxers de Kotlin també es col·loquen aquí).
  3. Selecciona New > Kotlin Class/File.
  4. Introdueix el nom de la teva classe i clica OK.

Comparació de Fitxers Font de Java i Kotlin

Aquí tens una comparació dels fitxers font de Java i Kotlin per destacar les diferències i similituds:

Conclusió

Entendre l'estructura i el procés de creació dels fitxers font de Java i Kotlin és fonamental per al desenvolupament d'Android. Mentre que Java ha estat el llenguatge tradicional, Kotlin ofereix una sintaxi més moderna i concisa, fent-lo l'opció preferida per a molts desenvolupadors. Dominant ambdós, pots aprofitar els punts forts de cada llenguatge i millorar les teves habilitats de desenvolupament d'Android.
