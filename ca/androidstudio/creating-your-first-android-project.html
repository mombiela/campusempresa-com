Introducció

En aquesta secció, et guiarem a través del procés de crear el teu primer projecte Android utilitzant Android Studio. Aquest és un pas crucial per a qualsevol desenvolupador Android, ja que estableix la base per a tot el treball de desenvolupament futur.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Android Studio instal·lat al teu ordinador.
  • Coneixement bàsic de Java o Kotlin (els llenguatges principals utilitzats per al desenvolupament Android).
  • Un dispositiu Android funcional o un emulador configurat a Android Studio.

Guia pas a pas per crear el teu primer projecte Android

Pas 1: Obre Android Studio

  • Llança Android Studio des del menú d'aplicacions o la drecera d'escriptori.

Pas 2: Comença un nou projecte d'Android Studio

  • Fes clic a "Start a new Android Studio project" des de la pantalla de benvinguda.

Pas 3: Configura el teu nou projecte

  • Nom: Introdueix un nom per al teu projecte (per exemple, "MyFirstApp").
  • Nom del paquet: Normalment és en el format com.example.myfirstapp. Ha de ser únic per evitar conflictes.
  • Ubicació de desament: Tria un directori on vulguis desar el teu projecte.
  • Llenguatge: Selecciona Java o Kotlin. Per a aquest exemple, utilitzarem Kotlin.
  • Nivell mínim d'API: Tria la versió mínima d'Android que la teva aplicació suportarà. Una elecció comuna és API 21: Android 5.0 (Lollipop).

Pas 4: Selecciona el factor de forma i la plantilla

  • Telèfon i tauleta: Assegura't que estigui marcat.
  • Plantilla: Selecciona "Empty Activity" per començar amb una plantilla bàsica.

Pas 5: Configura l'activitat

  • Nom de l'activitat: Per defecte és "MainActivity". Pots deixar-ho tal com està.
  • Nom del disseny: Per defecte és "activity_main". Aquest és el fitxer XML on dissenyaràs la teva interfície d'usuari.
  • Generar fitxer de disseny: Assegura't que estigui marcat.
  • Compatibilitat amb versions anteriors: Assegura't que "Use a legacy support library" estigui marcat si necessites suportar versions antigues d'Android.

Pas 6: Finalitza i construeix el projecte

  • Fes clic a "Finish" per crear el teu projecte. Android Studio configurarà el projecte i construirà els fitxers necessaris.

Entenent l'estructura del projecte

Un cop el teu projecte estigui creat, veuràs diversos directoris i fitxers. Aquí tens els components clau:

  • app/src/main/java/com.example.myfirstapp/MainActivity.kt: Aquest és el fitxer principal de l'activitat on escriuràs el teu codi Kotlin.
  • app/src/main/res/layout/activity_main.xml: Aquest és el fitxer de disseny on dissenyaràs la teva interfície d'usuari.
  • app/src/main/AndroidManifest.xml: Aquest fitxer conté informació essencial sobre la teva aplicació, com ara els seus components i permisos.

Escriure el teu primer codi

Afegim un simple missatge de "Hola, món!" a la teva aplicació.

MainActivity.kt

package com.example.myfirstapp

import androidx.appcompat.app.AppCompatActivity
import android.os.Bundle
import android.widget.TextView

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        // Troba el TextView en el disseny
        val textView: TextView = findViewById(R.id.textView)
        // Estableix el text a "Hola, món!"
        textView.text = "Hola, món!"
    }
}

activity_main.xml

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hola, món!"
        android:layout_centerInParent="true"/>
</RelativeLayout>

Executant la teva aplicació

  • Connecta el teu dispositiu Android o inicia l'emulador.
  • Fes clic al botó "Run" (icona de reproducció verda) a Android Studio.
  • Selecciona el teu dispositiu o emulador de la llista.
  • Android Studio construirà i instal·larà l'aplicació al teu dispositiu/emulador. Hauries de veure "Hola, món!" mostrat a la pantalla.

Conclusió

Felicitats! Has creat i executat amb èxit el teu primer projecte Android. Aquest pas fonamental és crucial mentre continues desenvolupant aplicacions més complexes. En les següents seccions, aprofundirem en els components d'Android, el disseny de la interfície d'usuari i funcions avançades per millorar les teves habilitats de desenvolupament d'aplicacions.
