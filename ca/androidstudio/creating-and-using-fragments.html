Introducció als Fragments

Els fragments són una característica poderosa en el desenvolupament d'Android que et permeten crear components modulars i reutilitzables dins de la teva aplicació. Representen una porció de la interfície d'usuari en una Activitat i es poden combinar per crear una interfície d'usuari de múltiples panells.

  • Cicle de vida del Fragment: Similar al cicle de vida de l'Activitat però amb estats addicionals.
  • FragmentManager: Gestiona els fragments en una activitat.
  • FragmentTransaction: S'utilitza per afegir, eliminar, substituir i realitzar altres accions amb fragments.

Creació d'un Fragment

Per crear un fragment, necessites estendre la classe Fragment i sobreescriure els seus mètodes de cicle de vida.

Guia Pas a Pas

  1. Crear una nova classe Fragment:

    public class ExampleFragment extends Fragment {
    @Nullable
    @Override
    public View onCreateView(LayoutInflater inflater, @Nullable ViewGroup container, @Nullable Bundle savedInstanceState) {
        // Inflar el disseny per a aquest fragment
        return inflater.inflate(R.layout.fragment_example, container, false);
    }
}

  2. Definir el disseny del Fragment:

    <!-- res/layout/fragment_example.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hola des del Fragment!" />
</LinearLayout>

Afegir un Fragment a una Activitat

Els fragments es poden afegir a una activitat estàticament via XML o dinàmicament via codi.

Afegit Estàtic

  • Definir el Fragment en el disseny de l'Activitat: 
    <!-- res/layout/activity_main.xml -->
<FrameLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/fragment_container"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />

<fragment
    android:id="@+id/example_fragment"
    android:name="com.example.ExampleFragment"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />

Afegit Dinàmic

  • Afegir el Fragment programàticament: 
    FragmentManager fragmentManager = getSupportFragmentManager();
FragmentTransaction fragmentTransaction = fragmentManager.beginTransaction();
ExampleFragment exampleFragment = new ExampleFragment();
fragmentTransaction.add(R.id.fragment_container, exampleFragment);
fragmentTransaction.commit();

Comunicació Entre Fragments i Activitats

Els fragments sovint necessiten comunicar-se amb la seva activitat host o altres fragments.

Ús d'Interfícies

  1. Definir una interfície en el Fragment:

    public interface OnFragmentInteractionListener {
    void onFragmentInteraction(String data);
}

  2. Implementar la interfície en l'Activitat:

    public class MainActivity extends AppCompatActivity implements ExampleFragment.OnFragmentInteractionListener {
    @Override
    public void onFragmentInteraction(String data) {
        // Gestionar la interacció
    }
}

  3. Invocar el mètode de la interfície des del Fragment:

    public class ExampleFragment extends Fragment {
    OnFragmentInteractionListener mListener;

    @Override
    public void onAttach(Context context) {
        super.onAttach(context);
        if (context instanceof OnFragmentInteractionListener) {
            mListener = (OnFragmentInteractionListener) context;
        } else {
            throw new RuntimeException(context.toString() + " must implement OnFragmentInteractionListener");
        }
    }

    // Cridar mListener.onFragmentInteraction(data) quan sigui necessari
}

Tècniques Avançades de Fragments

Transaccions de Fragments

  • Substituir un Fragment: 
    FragmentTransaction transaction = getSupportFragmentManager().beginTransaction();
transaction.replace(R.id.fragment_container, new AnotherFragment());
transaction.addToBackStack(null); // Opcional: afegir a la pila enrere
transaction.commit();

Pila Enrere de Fragments

  • Gestionar la pila enrere:
    • Afegir transaccions a la pila enrere per permetre als usuaris navegar enrere.
    • Utilitzar el mètode addToBackStack() en FragmentTransaction.

Arguments de Fragments

  • Passar dades a un Fragment:

    Bundle args = new Bundle();
args.putString("key", "value");
ExampleFragment fragment = new ExampleFragment();
fragment.setArguments(args);

  • Recuperar dades en el Fragment:

    @Override
public void onCreate(@Nullable Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    if (getArguments() != null) {
        String value = getArguments().getString("key");
    }
}

Conclusió

Els fragments són essencials per crear components d'interfície d'usuari flexibles i reutilitzables en aplicacions d'Android. Entenent com crear, afegir i gestionar fragments, així com com gestionar la comunicació entre fragments i activitats, pots construir aplicacions més dinàmiques i responsives. Practica l'ús de fragments en diferents escenaris per convertir-te en expert en el seu ús.
