Firebase és una plataforma potent proporcionada per Google que ofereix una varietat d'eines i serveis per ajudar els desenvolupadors a crear aplicacions d'alta qualitat. Aquest tema et guiarà a través de la integració de Firebase en el teu projecte d'Android Studio, des de nivells principiants fins a avançats.

Introducció a Firebase

Firebase proporciona serveis de backend com una base de dades NoSQL en temps real, autenticació, emmagatzematge al núvol i més. Integrar Firebase a la teva aplicació d'Android pot millorar significativament la seva funcionalitat i escalabilitat.

Conceptes Clau

  • Consola de Firebase: La interfície web on pots gestionar els teus projectes de Firebase.
  • SDK de Firebase: Kit de Desenvolupament de Programari que integres al teu projecte d'Android.
  • Serveis de Firebase: Inclou Autenticació, Firestore, Base de Dades en Temps Real, Emmagatzematge al Núvol, Missatgeria al Núvol i més.

Configurant Firebase a Android Studio

Pas 1: Crear un Projecte de Firebase

  1. Ves a la Consola de Firebase.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

Pas 2: Afegeix Firebase al Teu Projecte d'Android

  1. A la Consola de Firebase, fes clic a "Afegeix aplicació" i selecciona "Android".
  2. Registra la teva aplicació amb el nom del paquet del teu projecte d'Android.
  3. Descarrega el fitxer google-services.json i col·loca'l al directori app del teu projecte d'Android.

Pas 3: Configura el Teu Projecte

  1. Obre el teu fitxer build.gradle (a nivell de Projecte) i afegeix el següent: 
    buildscript {
    dependencies {
        // Afegeix aquesta línia
        classpath 'com.google.gms:google-services:4.3.10'
    }
}
  2. Obre el teu fitxer build.gradle (a nivell d'App) i afegeix: 
    dependencies {
    // Afegeix les dependències per als SDKs de Firebase que necessitis
    implementation 'com.google.firebase:firebase-analytics:19.0.0'
}

// Afegeix aquesta línia al final del fitxer
apply plugin: 'com.google.gms.google-services'

Utilitzant l'Autenticació de Firebase

Autenticació amb Correu Electrònic i Contrasenya

  1. Activa l'autenticació amb Correu Electrònic/Contrasenya a la Consola de Firebase sota la secció d'Autenticació.
  2. Afegeix la dependència de l'Autenticació de Firebase al teu fitxer build.gradle: 
    implementation 'com.google.firebase:firebase-auth:21.0.1'
  3. Utilitza el següent codi per registrar un nou usuari: 
    FirebaseAuth mAuth = FirebaseAuth.getInstance();

mAuth.createUserWithEmailAndPassword(email, password)
    .addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
        @Override
        public void onComplete(@NonNull Task<AuthResult> task) {
            if (task.isSuccessful()) {
                // Sign in success
                FirebaseUser user = mAuth.getCurrentUser();
                // Actualitza la UI amb la informació de l'usuari registrat
            } else {
                // Si el registre falla, mostra un missatge a l'usuari.
            }
        }
    });

Iniciar Sessió d'Usuaris Existents

Utilitzant Firebase Firestore

Configurant Firestore

  1. Activa Firestore a la Consola de Firebase.
  2. Afegeix la dependència de Firestore al teu fitxer build.gradle: 
    implementation 'com.google.firebase:firebase-firestore:24.0.0'

Afegint Dades a Firestore

FirebaseFirestore db = FirebaseFirestore.getInstance();

// Crea un nou usuari amb un nom i cognom
Map<String, Object> user = new HashMap<>();
user.put("first", "Ada");
user.put("last", "Lovelace");
user.put("born", 1815);

// Afegeix un nou document amb un ID generat
db.collection("users")
    .add(user)
    .addOnSuccessListener(new OnSuccessListener<DocumentReference>() {
        @Override
        public void onSuccess(DocumentReference documentReference) {
            Log.d(TAG, "DocumentSnapshot added with ID: " + documentReference.getId());
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            Log.w(TAG, "Error adding document", e);
        }
    });

Llegint Dades de Firestore

db.collection("users")
    .get()
    .addOnCompleteListener(new OnCompleteListener<QuerySnapshot>() {
        @Override
        public void onComplete(@NonNull Task<QuerySnapshot> task) {
            if (task.isSuccessful()) {
                for (QueryDocumentSnapshot document : task.getResult()) {
                    Log.d(TAG, document.getId() + " => " + document.getData());
                }
            } else {
                Log.w(TAG, "Error getting documents.", task.getException());
            }
        }
    });

Funcionalitats Avançades de Firebase

Firebase Cloud Messaging (FCM)

  1. Activa FCM a la Consola de Firebase.
  2. Afegeix la dependència de FCM al teu fitxer build.gradle: 
    implementation 'com.google.firebase:firebase-messaging:23.0.0'

Enviant Notificacions

  • Utilitza la Consola de Firebase per enviar notificacions.
  • Gestiona les notificacions a la teva aplicació estenent FirebaseMessagingService: 
    public class MyFirebaseMessagingService extends FirebaseMessagingService {
    @Override
    public void onMessageReceived(RemoteMessage remoteMessage) {
        // Gestiona els missatges de FCM aquí.
        Log.d(TAG, "From: " + remoteMessage.getFrom());
        Log.d(TAG, "Notification Message Body: " + remoteMessage.getNotification().getBody());
    }
}

Conclusió

Integrar Firebase a la teva aplicació d'Android pot millorar significativament les seves capacitats, des de l'autenticació d'usuaris fins a la gestió de dades en temps real i la missatgeria al núvol. Seguint els passos descrits en aquest tema, pots configurar i utilitzar diversos serveis de Firebase de manera efectiva. A mesura que et familiaritzis més amb Firebase, pots explorar les seves funcionalitats avançades per millorar encara més la funcionalitat i l'experiència d'usuari de la teva aplicació.
