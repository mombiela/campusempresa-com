L'emmagatzematge extern a Android es refereix a l'emmagatzematge que és accessible per a totes les aplicacions i l'usuari. Es pot utilitzar per emmagatzemar fitxers grans i dades que necessiten ser compartides entre diferents aplicacions o amb l'usuari. Aquesta secció et guiarà a través dels conceptes bàsics de l'emmagatzematge extern, com llegir i escriure fitxers, i les millors pràctiques per gestionar l'emmagatzematge extern en les teves aplicacions Android.

Conceptes clau

  • Tipus d'emmagatzematge extern:

    • Emmagatzematge extern primari: L'emmagatzematge extern principal, normalment l'emmagatzematge compartit intern del dispositiu.
    • Emmagatzematge extern secundari: Opcions d'emmagatzematge addicionals com targetes SD o unitats USB.

  • Permisos:

    • READ_EXTERNAL_STORAGE: Requerit per llegir fitxers de l'emmagatzematge extern.
    • WRITE_EXTERNAL_STORAGE: Requerit per escriure fitxers a l'emmagatzematge extern.

  • Directoris d'emmagatzematge:

    • Environment.getExternalStorageDirectory(): Retorna el directori d'emmagatzematge extern primari.
    • Environment.getExternalStoragePublicDirectory(String type): Retorna un directori públic per emmagatzemar fitxers d'un tipus específic (per exemple, imatges, música).

Llegir i escriure fitxers

Sol·licitud de permisos

Abans d'accedir a l'emmagatzematge extern, has de sol·licitar els permisos necessaris en el teu fitxer AndroidManifest.xml:

<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.myapp">

    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>

    <application
        ... >
        ...
    </application>
</manifest>

Comprovar permisos en temps d'execució

Des d'Android 6.0 (nivell d'API 23) en endavant, has de sol·licitar permisos en temps d'execució:

if (ContextCompat.checkSelfPermission(this, Manifest.permission.WRITE_EXTERNAL_STORAGE)
        != PackageManager.PERMISSION_GRANTED) {
    ActivityCompat.requestPermissions(this, new String[]{Manifest.permission.WRITE_EXTERNAL_STORAGE}, REQUEST_CODE);
}

Escriure a l'emmagatzematge extern

Per escriure un fitxer a l'emmagatzematge extern, segueix aquests passos:

  1. Comprova si l'emmagatzematge està disponible:
String state = Environment.getExternalStorageState();
if (!Environment.MEDIA_MOUNTED.equals(state)) {
    // Gestiona l'error: l'emmagatzematge extern no està disponible
}
  1. Obté el directori d'emmagatzematge extern:
File externalStorageDir = Environment.getExternalStorageDirectory();
File myFile = new File(externalStorageDir, "myfile.txt");
  1. Escriu al fitxer:
try (FileOutputStream fos = new FileOutputStream(myFile)) {
    fos.write("Hello, World!".getBytes());
} catch (IOException e) {
    e.printStackTrace();
}

Llegir de l'emmagatzematge extern

Per llegir un fitxer de l'emmagatzematge extern, segueix aquests passos:

  1. Comprova si l'emmagatzematge està disponible:
String state = Environment.getExternalStorageState();
if (!Environment.MEDIA_MOUNTED.equals(state) && !Environment.MEDIA_MOUNTED_READ_ONLY.equals(state)) {
    // Gestiona l'error: l'emmagatzematge extern no està disponible o no és llegible
}
  1. Obté el directori d'emmagatzematge extern:
File externalStorageDir = Environment.getExternalStorageDirectory();
File myFile = new File(externalStorageDir, "myfile.txt");
  1. Llegeix del fitxer:
try (FileInputStream fis = new FileInputStream(myFile)) {
    int content;
    while ((content = fis.read()) != -1) {
        System.out.print((char) content);
    }
} catch (IOException e) {
    e.printStackTrace();
}

Millors pràctiques

  • Comprova la disponibilitat de l'emmagatzematge: Sempre comprova si l'emmagatzematge extern està disponible abans de llegir o escriure.
  • Gestiona els permisos: Gestiona correctament les sol·licituds de permisos i les respostes.
  • Utilitza directoris públics: Emmagatzema fitxers en directoris públics quan necessitin ser accessibles per altres aplicacions o l'usuari.
  • Gestiona l'emmagatzematge de manera eficient: Neteja els fitxers innecessaris per evitar omplir l'emmagatzematge.

Resum

L'emmagatzematge extern és una característica poderosa a Android que permet a les aplicacions emmagatzemar i compartir fitxers grans. Entenent els tipus d'emmagatzematge extern, gestionant correctament els permisos i seguint les millors pràctiques, pots gestionar efectivament l'emmagatzematge extern en les teves aplicacions Android. Assegura't sempre de comprovar la disponibilitat de l'emmagatzematge i gestionar correctament els permisos per proporcionar una experiència d'usuari sense problemes.
