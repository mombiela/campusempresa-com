Introducció als Serveis

Els serveis a Android són components que permeten a una aplicació realitzar operacions de llarga durada en segon pla. No proporcionen una interfície d'usuari. Els serveis es poden utilitzar per a diversos propòsits, com ara reproduir música, gestionar transaccions de xarxa o interactuar amb proveïdors de contingut.

Conceptes Clau

  • Cicle de Vida del Servei: Entendre el cicle de vida d'un servei és crucial per gestionar el seu comportament i recursos.
  • Tipus de Serveis: Hi ha diferents tipus de serveis, cadascun adequat per a tasques específiques.
  • Enllaçar-se a un Servei: Permet als components interactuar amb el servei i realitzar comunicació entre processos.

Cicle de Vida del Servei

El cicle de vida d'un servei és diferent del d'una activitat. És important gestionar el cicle de vida correctament per assegurar un ús eficient dels recursos.

Mètodes del Cicle de Vida

  • onCreate(): Es crida quan el servei es crea per primera vegada.
  • onStartCommand(): Es crida cada vegada que un client inicia el servei utilitzant startService().
  • onBind(): Es crida quan un client es vincula al servei utilitzant bindService().
  • onUnbind(): Es crida quan tots els clients s'han desconnectat d'una interfície específica publicada pel servei.
  • onDestroy(): Es crida quan el servei ja no s'utilitza i s'està destruint.
public class MyService extends Service {
    @Override
    public void onCreate() {
        super.onCreate();
        // Inicialitza el teu servei aquí
    }

    @Override
    public int onStartCommand(Intent intent, int flags, int startId) {
        // Gestiona la sol·licitud d'inici del servei aquí
        return START_STICKY;
    }

    @Override
    public IBinder onBind(Intent intent) {
        // Retorna el canal de comunicació al servei
        return null;
    }

    @Override
    public void onDestroy() {
        super.onDestroy();
        // Neteja els recursos aquí
    }
}

Tipus de Serveis

Hi ha tres tipus principals de serveis a Android:

Serveis en Primer Pla

  • S'utilitzen per a tasques que són notables per a l'usuari, com ara reproduir música.
  • Han de mostrar una notificació a l'usuari.

Serveis en Segon Pla

  • S'utilitzen per a tasques que no són directament notades per l'usuari, com ara la sincronització de dades.
  • Subjectes a límits d'execució en segon pla en versions més noves d'Android.

Serveis Vinculats

  • Permet als components vincular-se al servei i interactuar amb ell.
  • Útil per a la comunicació entre processos.

Crear un Servei Simple

Creem un servei simple que registra un missatge cada vegada que s'inicia.

Exemple Pas a Pas

  1. Crear la Classe del Servei:
public class SimpleService extends Service {
    private static final String TAG = "SimpleService";

    @Override
    public void onCreate() {
        super.onCreate();
        Log.d(TAG, "Servei Creat");
    }

    @Override
    public int onStartCommand(Intent intent, int flags, int startId) {
        Log.d(TAG, "Servei Iniciat");
        return START_STICKY;
    }

    @Override
    public IBinder onBind(Intent intent) {
        return null;
    }

    @Override
    public void onDestroy() {
        super.onDestroy();
        Log.d(TAG, "Servei Destruït");
    }
}
  1. Declarar el Servei al Manifest:
<service android:name=".SimpleService" />
  1. Iniciar el Servei des d'una Activitat:
Intent intent = new Intent(this, SimpleService.class);
startService(intent);

Enllaçar-se a un Servei

Enllaçar-se a un servei et permet interactuar amb ell i realitzar operacions. Això és útil per a serveis que necessiten proporcionar una interfície client-servidor.

Exemple d'un Servei Vinculat

  1. Crear la Classe del Servei:
public class BoundService extends Service {
    private final IBinder binder = new LocalBinder();

    public class LocalBinder extends Binder {
        BoundService getService() {
            return BoundService.this;
        }
    }

    @Override
    public IBinder onBind(Intent intent) {
        return binder;
    }

    public String getGreeting() {
        return "Hola des del Servei Vinculat!";
    }
}
  1. Enllaçar-se al Servei des d'una Activitat:
public class MainActivity extends AppCompatActivity {
    BoundService boundService;
    boolean isBound = false;

    private ServiceConnection connection = new ServiceConnection() {
        @Override
        public void onServiceConnected(ComponentName name, IBinder service) {
            BoundService.LocalBinder binder = (BoundService.LocalBinder) service;
            boundService = binder.getService();
            isBound = true;
        }

        @Override
        public void onServiceDisconnected(ComponentName name) {
            isBound = false;
        }
    };

    @Override
    protected void onStart() {
        super.onStart();
        Intent intent = new Intent(this, BoundService.class);
        bindService(intent, connection, Context.BIND_AUTO_CREATE);
    }

    @Override
    protected void onStop() {
        super.onStop();
        if (isBound) {
            unbindService(connection);
            isBound = false;
        }
    }
}

Conclusió

Els serveis són un component poderós a Android que permeten el processament en segon pla i la comunicació entre processos. Entendre el cicle de vida i els tipus de serveis és crucial per a un desenvolupament eficient d'aplicacions. Dominant els serveis, pots crear aplicacions Android més responsives i robustes.
