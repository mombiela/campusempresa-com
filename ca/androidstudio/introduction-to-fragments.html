Introducció als Fragments
Cicle de Vida del Fragment ►

Els fragments són un component crucial en el desenvolupament d'Android, permetent dissenys d'interfície d'usuari més flexibles i modulars. Aquesta secció et guiarà a través dels conceptes bàsics dels fragments, el seu cicle de vida i com implementar-los en les teves aplicacions Android.

Què és un Fragment?

Un fragment representa una porció reutilitzable de la interfície d'usuari de la teva aplicació. Es pot pensar com una secció modular d'una activitat, que té el seu propi cicle de vida i es pot afegir o eliminar mentre l'activitat està en execució.

Conceptes Clau

  • Modularitat: Els fragments permeten descompondre la teva interfície d'usuari en components més petits i reutilitzables.
  • Cicle de vida: Els fragments tenen el seu propi cicle de vida, que està estretament lligat al cicle de vida de l'activitat que els allotja.
  • Interfície d'usuari dinàmica: Els fragments permeten dissenys d'interfície d'usuari dinàmics i flexibles, com adaptar-se a diferents mides i orientacions de pantalla.

Cicle de Vida del Fragment

Entendre el cicle de vida del fragment és essencial per gestionar el seu comportament i interaccions. Els mètodes del cicle de vida són similars als d'una activitat però tenen callbacks addicionals.

Mètodes del Cicle de Vida del Fragment

  1. onAttach(): Es crida quan el fragment s'adjunta per primera vegada al seu context.
  2. onCreate(): Es crida per inicialitzar el fragment.
  3. onCreateView(): Es crida per crear la jerarquia de vistes associada amb el fragment.
  4. onActivityCreated(): Es crida quan el mètode onCreate() de l'activitat ha retornat.
  5. onStart(): Es crida quan el fragment és visible per a l'usuari.
  6. onResume(): Es crida quan el fragment està actiu i interactuant amb l'usuari.
  7. onPause(): Es crida quan el fragment ja no està interactuant amb l'usuari.
  8. onStop(): Es crida quan el fragment ja no és visible.
  9. onDestroyView(): Es crida per netejar els recursos associats amb la vista.
  10. onDestroy(): Es crida per fer la neteja final.
  11. onDetach(): Es crida quan el fragment es desenganxa del seu context.

Implementació d'un Fragment

Vegem com crear un fragment senzill i afegir-lo a una activitat.

Pas 1: Crear una Classe Fragment

Crea una nova classe que extengui Fragment.

public class ExampleFragment extends Fragment {

    @Nullable
    @Override
    public View onCreateView(@NonNull LayoutInflater inflater, @Nullable ViewGroup container, @Nullable Bundle savedInstanceState) {
        // Inflar el layout per a aquest fragment
        return inflater.inflate(R.layout.fragment_example, container, false);
    }
}

Pas 2: Definir el Layout del Fragment

Crea un fitxer de layout per al fragment (fragment_example.xml).

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hola, Fragment!" />
</LinearLayout>

Pas 3: Afegir el Fragment a una Activitat

Modifica el layout de l'activitat per incloure un contenidor per al fragment (activity_main.xml).

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <FrameLayout
        android:id="@+id/fragment_container"
        android:layout_width="match_parent"
        android:layout_height="match_parent" />
</RelativeLayout>

En el codi de l'activitat, afegeix el fragment al contenidor.

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Comprova si el contenidor del fragment està disponible
        if (findViewById(R.id.fragment_container) != null) {
            // Prevenir fragments superposats
            if (savedInstanceState != null) {
                return;
            }

            // Crea una instància d'ExampleFragment
            ExampleFragment exampleFragment = new ExampleFragment();

            // Afegeix el fragment al contenidor del fragment
            getSupportFragmentManager().beginTransaction()
                .add(R.id.fragment_container, exampleFragment).commit();
        }
    }
}

Conclusió

Els fragments són eines poderoses per crear interfícies d'usuari dinàmiques i flexibles en aplicacions Android. Permeten modularitzar la interfície d'usuari de la teva aplicació, facilitant-ne la gestió i l'adaptació a diferents mides i orientacions de pantalla. Entenent el cicle de vida dels fragments i com implementar-los, pots crear aplicacions Android més robustes i mantenibles.
