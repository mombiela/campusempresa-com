L'enllaç de dades és una característica potent a Android que et permet enllaçar components de la interfície d'usuari en els teus dissenys a fonts de dades en la teva aplicació utilitzant un format declaratiu en lloc de programàticament. Aquest tema et guiarà des dels conceptes bàsics fins als avançats de l'enllaç de dades a Android Studio.

Introducció a l'Enllaç de Dades

La Biblioteca d'Enllaç de Dades és una biblioteca de suport que et permet enllaçar components de la interfície d'usuari en els teus dissenys a fonts de dades en la teva aplicació utilitzant un format declaratiu en lloc de programàticament.

Conceptes Clau

  • Dissenys d'Enllaç: Com crear i utilitzar dissenys d'enllaç.
  • Expressions d'Enllaç: Com utilitzar expressions dins dels teus dissenys XML.
  • Dades Observables: Com fer que les teves dades siguin observables per actualitzar la interfície d'usuari automàticament.
  • Enllaç de Dades Bidireccional: Com enllaçar dades en ambdues direccions entre la interfície d'usuari i la font de dades.

Configuració de l'Enllaç de Dades

Per utilitzar l'enllaç de dades en el teu projecte Android, has d'habilitar-lo en el teu fitxer build.gradle.

android {
    ...
    viewBinding {
        enabled = true
    }
}

Dissenys d'Enllaç

Creació d'un Disseny d'Enllaç

Per crear un disseny d'enllaç, has d'envoltar el teu fitxer de disseny XML amb una etiqueta <layout>.

<layout xmlns:android="http://schemas.android.com/apk/res/android">
    <data>
        <variable
            name="user"
            type="com.example.User"/>
    </data>
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:orientation="vertical">

        <TextView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="@{user.name}"/>

        <TextView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="@{user.age}"/>
    </LinearLayout>
</layout>

Expressions d'Enllaç

Les expressions d'enllaç et permeten escriure expressions directament en els teus dissenys XML.

  • Expressions Simples: @{user.name}
  • Crides a Mètodes: @{user.getName()}
  • Operacions Aritmètiques: @{user.age + 1}
  • Operacions Lògiques: @{user.age > 18 ? "Adult" : "Minor"}

Dades Observables

Per actualitzar automàticament la interfície d'usuari quan les dades canvien, pots utilitzar tipus de dades observables.

Ús de Camps Observables

public class User extends BaseObservable {
    private String name;
    private int age;

    @Bindable
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
        notifyPropertyChanged(BR.name);
    }

    @Bindable
    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
        notifyPropertyChanged(BR.age);
    }
}

Ús de LiveData

public class UserViewModel extends ViewModel {
    private MutableLiveData<User> user;

    public LiveData<User> getUser() {
        if (user == null) {
            user = new MutableLiveData<User>();
            loadUser();
        }
        return user;

Enllaç de Dades Bidireccional

L'enllaç de dades bidireccional permet que els canvis en la interfície d'usuari actualitzin la font de dades i viceversa.

Exemple d'Enllaç de Dades Bidireccional

<layout xmlns:android="http://schemas.android.com/apk/res/android">
    <data>
        <variable
            name="viewModel"
            type="com.example.UserViewModel"/>
    </data>
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:orientation="vertical">

        <EditText
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:text="@={viewModel.user.name}"/>
    </LinearLayout>
</layout>

Conceptes Avançats d'Enllaç de Dades

Adaptadors d'Enllaç Personalitzats

Els adaptadors d'enllaç personalitzats et permeten crear comportaments personalitzats per als teus components de la interfície d'usuari.

public class BindingAdapters {
    @BindingAdapter("app:imageUrl")
    public static void loadImage(ImageView view, String imageUrl) {
        Glide.with(view.getContext())
             .load(imageUrl)
             .into(view);
    }
}

Enllaç de Col·leccions

Enllaçar col·leccions com llistes o arrays a RecyclerView.

<layout xmlns:android="http://schemas.android.com/apk/res/android">
    <data>
        <variable
            name="viewModel"
            type="com.example.UserViewModel"/>
    </data>
    <androidx.recyclerview.widget.RecyclerView
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        app:items="@{viewModel.users}"/>
</layout>

Conclusió

L'enllaç de dades a Android Studio simplifica el procés d'enllaçar components de la interfície d'usuari a fonts de dades, fent el teu codi més net i mantenible. Entenent i utilitzant l'enllaç de dades, pots crear aplicacions Android més eficients i responsives. Aquest tema ha cobert els conceptes bàsics de la configuració de l'enllaç de dades, la creació de dissenys d'enllaç, l'ús de dades observables, la implementació de l'enllaç de dades bidireccional i conceptes avançats com els adaptadors d'enllaç personalitzats i l'enllaç de col·leccions.
