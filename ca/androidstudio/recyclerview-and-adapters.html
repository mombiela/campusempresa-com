Introducció

RecyclerView és un component potent i flexible per mostrar un gran conjunt de dades en aplicacions Android. És una versió més avançada i eficient de ListView. Els adapters s'utilitzen per connectar la font de dades amb el RecyclerView, permetent la visualització dinàmica de dades.

Configuració de RecyclerView

Per utilitzar RecyclerView, cal afegir les dependències necessàries al teu projecte.

Afegir Dependències

Primer, afegeix la dependència de RecyclerView al teu fitxer build.gradle:

dependencies {
    implementation 'androidx.recyclerview:recyclerview:1.2.1'
}

Layout XML

A continuació, afegeix un RecyclerView al teu fitxer de disseny XML:

<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/recyclerView"
    android:layout_width="match_parent"
    android:layout_height="match_parent"/>

Creació de l'Adapter

L'Adapter és responsable de crear el ViewHolder i vincular-hi les dades.

Classe ViewHolder

Crea una classe ViewHolder que extengui RecyclerView.ViewHolder:

public class MyViewHolder extends RecyclerView.ViewHolder {
    TextView textView;

    public MyViewHolder(@NonNull View itemView) {
        super(itemView);
        textView = itemView.findViewById(R.id.textView);
    }
}

Classe Adapter

Crea una classe Adapter que extengui RecyclerView.Adapter<MyViewHolder>:

public class MyAdapter extends RecyclerView.Adapter<MyViewHolder> {
    private List<String> dataList;

    public MyAdapter(List<String> dataList) {
        this.dataList = dataList;
    }

    @NonNull
    @Override
    public MyViewHolder onCreateViewHolder(@NonNull ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(parent.getContext()).inflate(R.layout.item_layout, parent, false);
        return new MyViewHolder(view);
    }

    @Override
    public void onBindViewHolder(@NonNull MyViewHolder holder, int position) {
        String data = dataList.get(position);
        holder.textView.setText(data);
    }

    @Override
    public int getItemCount() {
        return dataList.size();
    }
}

Layout XML de l'Item

Crea un fitxer de disseny XML per als elements individuals del RecyclerView:

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Sample Text"
        android:textSize="18sp"/>
</LinearLayout>

Connectar RecyclerView i Adapter

A la teva Activity o Fragment, configura el RecyclerView i l'Adapter.

Inicialitzar RecyclerView

RecyclerView recyclerView = findViewById(R.id.recyclerView);
recyclerView.setLayoutManager(new LinearLayoutManager(this));
MyAdapter adapter = new MyAdapter(dataList);
recyclerView.setAdapter(adapter);

Funcionalitats Avançades

Gestionar Esdeveniments de Clic

Per gestionar esdeveniments de clic, pots definir una interfície al teu Adapter i implementar-la a la teva Activity o Fragment.

Optimitzar el Rendiment

  • Utilitza RecyclerView.setHasFixedSize(true) si la mida del RecyclerView no canvia.
  • Utilitza DiffUtil per actualitzar eficientment el RecyclerView quan el conjunt de dades canvia.

Utilitzar Diferents Tipus de Vista

Pots sobreescriure getItemViewType al teu Adapter per mostrar diferents dissenys per a diferents elements.

Conclusió

RecyclerView i Adapters són components essencials per mostrar dades dinàmiques en aplicacions Android. Entenent com configurar i utilitzar aquests components, pots crear llistes i graelles eficients i flexibles. Recorda optimitzar el rendiment i gestionar les interaccions dels usuaris per proporcionar una experiència d'usuari fluida.
