SharedPreferences és una eina poderosa en el desenvolupament d'Android, que et permet emmagatzemar i recuperar petites quantitats de dades primitives com a parells clau-valor. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de SharedPreferences a Android Studio.

Introducció a SharedPreferences

SharedPreferences s'utilitza per a:

  • Emmagatzemar tipus de dades simples (boolean, float, int, long, string).
  • Persistir preferències i configuracions de l'usuari.
  • Emmagatzemar en memòria cau petites quantitats de dades.

Conceptes Clau

  • SharedPreferences: Interfície per accedir i modificar dades de preferències.
  • Editor: Interfície utilitzada per modificar valors en un objecte SharedPreferences.

Ús Bàsic de SharedPreferences

Crear i Accedir a SharedPreferences

Per crear o accedir a SharedPreferences, utilitza el mètode getSharedPreferences().

// Crear o accedir a SharedPreferences
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);

Emmagatzemar Dades

Per emmagatzemar dades, utilitza la interfície SharedPreferences.Editor.

// Emmagatzemar dades a SharedPreferences
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("username", "JohnDoe");
editor.putInt("userAge", 25);
editor.apply(); // o editor.commit();

Recuperar Dades

Per recuperar dades, utilitza el mètode get apropiat.

// Recuperar dades de SharedPreferences
String username = sharedPreferences.getString("username", "defaultName");
int userAge = sharedPreferences.getInt("userAge", 0);

Ús Avançat de SharedPreferences

Eliminar Dades

Per eliminar un parell clau-valor específic, utilitza el mètode remove().

// Eliminar dades de SharedPreferences
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.remove("username");
editor.apply();

Esborrar Totes les Dades

Per esborrar totes les dades a SharedPreferences, utilitza el mètode clear().

// Esborrar totes les dades de SharedPreferences
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.clear();
editor.apply();

Utilitzar SharedPreferences amb Objectes Personalitzats

SharedPreferences no suporta l'emmagatzematge d'objectes personalitzats directament. Necessites serialitzar l'objecte a una cadena JSON.

Exemple: Emmagatzemar un Objecte Personalitzat

// Objecte personalitzat
class User {
    String name;
    int age;

    // Constructor, getters, i setters
}

// Convertir objecte a cadena JSON
Gson gson = new Gson();
User user = new User("JohnDoe", 25);
String json = gson.toJson(user);

// Emmagatzemar cadena JSON a SharedPreferences
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("user", json);
editor.apply();

Exemple: Recuperar un Objecte Personalitzat

// Recuperar cadena JSON de SharedPreferences
String json = sharedPreferences.getString("user", "");

// Convertir cadena JSON de nou a objecte
User user = gson.fromJson(json, User.class);

Millors Pràctiques

  • Utilitza apply() en lloc de commit(): apply() és asíncron i no bloqueja el fil principal, mentre que commit() és síncron.
  • Mantingues les claus organitzades: Utilitza una convenció de noms consistent per a les claus per evitar conflictes.
  • Limita l'ús de SharedPreferences: Utilitza'l només per a petites quantitats de dades. Per a conjunts de dades més grans, considera utilitzar una base de dades.

Conclusió

SharedPreferences és una eina simple però poderosa per emmagatzemar petites quantitats de dades en aplicacions Android. Entenent el seu ús bàsic i avançat, pots gestionar efectivament les preferències de l'usuari i altres requisits de petites dades a la teva aplicació. Recorda seguir les millors pràctiques per assegurar un emmagatzematge de dades eficient i organitzat.
