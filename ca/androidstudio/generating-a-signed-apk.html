Introducció

Generar un APK (Paquet d'Android) signat és un pas crucial en el procés de desenvolupament d'aplicacions Android. Es requereix un APK signat per publicar la teva aplicació a Google Play Store o distribuir-la als usuaris. Aquesta secció et guiarà a través del procés de generar un APK signat utilitzant Android Studio, des de nivells bàsics fins a avançats.

Prerequisits

Abans de començar, assegura't de tenir:

  • Android Studio instal·lat.
  • Un projecte d'Android complet que vulguis generar un APK signat.
  • Comprensió bàsica de l'estructura del projecte d'Android.

Passos per Generar un APK Signat

1. Obre el Teu Projecte a Android Studio

  • Llança Android Studio.
  • Obre el projecte pel qual vols generar un APK signat.

2. Menú de Compilació

  • Navega a la barra de menú superior.
  • Fes clic a Build > Generate Signed Bundle / APK....

3. Selecciona APK

  • En el diàleg que apareix, selecciona APK i fes clic a Next.

4. Crea o Tria un Key Store

  • Necessitaràs un Key Store per signar el teu APK. Si no en tens un, pots crear-lo dins d'Android Studio.
  • Fes clic a Create new... si no tens un Key Store existent.

Creant un Nou Key Store

  • Key store path: Tria una ubicació per guardar el teu fitxer Key Store.
  • Password: Crea una contrasenya per al teu Key Store.
  • Key alias: Crea un àlies per la teva clau.
  • Key password: Crea una contrasenya per la teva clau.
  • Validity (years): Estableix el període de validesa per la teva clau (per exemple, 25 anys).
  • Certificate: Omple la informació requerida (Nom i Cognoms, Unitat Organitzativa, Organització, Ciutat o Localitat, Estat o Província, Codi de País).
Key store path: /path/to/your/keystore.jks
Password: **
Key alias: my-key-alias
Key password: **
Validity (years): 25
Certificate:
  First and Last Name: John Doe
  Organizational Unit: Development
  Organization: MyCompany
  City or Locality: MyCity
  State or Province: MyState
  Country Code: US

5. Selecciona Destinació i Tipus de Compilació

  • Tria la carpeta de destinació on vols guardar l'APK signat.
  • Selecciona el tipus de compilació (per exemple, release).
  • Fes clic a Finish.

6. Procés de Compilació

  • Android Studio ara compilarà el teu projecte i generarà l'APK signat.
  • Un cop el procés estigui complet, veuràs una notificació amb la ubicació de l'APK generat.

Opcions Avançades

ProGuard i Signatura d'Aplicacions

  • ProGuard és una eina que pot reduir i ofuscar el teu codi, fent-lo més difícil de revertir.
  • Per habilitar ProGuard, afegeix les següents línies al teu fitxer build.gradle:
buildTypes {
    release {
        minifyEnabled true
        proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
    }
}

Múltiples APKs per Diferents Densitats de Pantalla

  • Pots generar múltiples APKs per diferents densitats de pantalla per optimitzar la mida de l'aplicació per a diversos dispositius.
  • Afegeix la següent configuració al teu fitxer build.gradle:
android {
    splits {
        density {
            enable true
            reset()
            include 'ldpi', 'mdpi', 'hdpi', 'xhdpi', 'xxhdpi', 'xxxhdpi'
            compatibleScreens 'small', 'normal', 'large', 'xlarge'
        }
    }
}

Conclusió

Generar un APK signat és un pas essencial per preparar la teva aplicació d'Android per a la distribució. Seguint els passos descrits anteriorment, pots crear un APK signat utilitzant Android Studio, assegurant que la teva aplicació està llesta per al llançament. A més, opcions avançades com ProGuard i múltiples APKs poden ajudar a optimitzar la teva aplicació per a un millor rendiment i seguretat.
