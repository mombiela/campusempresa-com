Passant Dades entre Activitats ►

Navegar entre activitats és un concepte fonamental en el desenvolupament d'Android. Aquest tema et guiarà a través dels conceptes bàsics de la navegació d'activitats, des de la creació d'intents fins al pas de dades entre activitats. Al final d'aquesta secció, seràs capaç de crear transicions fluides entre diferents pantalles a la teva aplicació Android.

Introducció a les activitats

  • Activity: Una cosa única i enfocada que l'usuari pot fer. Gairebé totes les activitats interactuen amb l'usuari, així que la classe Activity s'encarrega de crear una finestra per a tu en la qual pots col·locar la teva interfície d'usuari amb setContentView(View).
  • Intent: Una descripció abstracta d'una operació a realitzar. Es pot utilitzar amb startActivity per llançar una activitat.

Creació i inici d'una activitat

Per navegar d'una activitat a una altra, necessites crear un intent i utilitzar-lo per iniciar la nova activitat.

Exemple: Iniciant una nova activitat

  1. Crear una nova activitat: Suposem que tens una activitat principal (MainActivity) i vols navegar a una segona activitat (SecondActivity).
// MainActivity.java
package com.example.myfirstapp;

import android.content.Intent;
import android.os.Bundle;
import android.view.View;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }

    public void navigateToSecondActivity(View view) {
        Intent intent = new Intent(this, SecondActivity.class);
        startActivity(intent);
    }
}
  1. Definir la nova activitat al Manifest: Assegura't que SecondActivity està declarada al teu AndroidManifest.xml.
<!-- AndroidManifest.xml -->
<application
    ... >
    <activity android:name=".MainActivity">
        <intent-filter>
            <action android:name="android.intent.action.MAIN" />
            <category android:name="android.intent.category.LAUNCHER" />
        </intent-filter>
    </activity>
    <activity android:name=".SecondActivity"></activity>
</application>
  1. Crear la nova classe d'activitat: Implementa la SecondActivity.
// SecondActivity.java
package com.example.myfirstapp;

import android.os.Bundle;
import androidx.appcompat.app.AppCompatActivity;

public class SecondActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_second);
    }
}
  1. Crear el disseny per a la nova activitat: Defineix la interfície d'usuari per a SecondActivity a activity_second.xml.
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Benvingut a la Segona Activitat!" />
</LinearLayout>

Passar dades entre activitats

Sovint, necessites passar dades d'una activitat a una altra. Això es pot fer utilitzant els extres de Intent.

Exemple: Passar dades amb extres d'Intent

  1. Enviar dades des de MainActivity:
// MainActivity.java
public void navigateToSecondActivity(View view) {
    Intent intent = new Intent(this, SecondActivity.class);
    intent.putExtra("EXTRA_MESSAGE", "Hola, Segona Activitat!");
    startActivity(intent);
}
  1. Rebre dades a SecondActivity:
// SecondActivity.java
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_second);

    Intent intent = getIntent();
    String message = intent.getStringExtra("EXTRA_MESSAGE");

    TextView textView = findViewById(R.id.textView);
    textView.setText(message);
}
  1. Actualitzar el disseny per incloure un TextView:
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Benvingut a la Segona Activitat!" />
</LinearLayout>

Navegar de tornada a l'activitat anterior

Android gestiona la navegació enrere per tu, però també pots acabar explícitament una activitat.

Exemple: Acabant una activitat

// SecondActivity.java
public void finishActivity(View view) {
    finish();
}

Conclusió

Navegar entre activitats és una part crucial del desenvolupament d'Android. Entenent com crear intents, iniciar activitats i passar dades entre elles, pots crear aplicacions més dinàmiques i interactives. Practica aquests conceptes creant múltiples activitats i navegant entre elles, passant diferents tipus de dades per familiaritzar-te amb el procés.
