Introducció

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un desenvolupador d'Android competent. Aquesta secció et guiarà a través del procés d'instal·lació i configuració d'Android Studio, l'Entorn de Desenvolupament Integrat (IDE) oficial per al desenvolupament d'Android.

Prerequisits

Abans de començar, assegura't que el teu sistema compleixi els següents requisits:

  • Sistema Operatiu: Windows, macOS o Linux
  • RAM: Mínim 4 GB, 8 GB recomanats
  • Espai en Disc: Mínim 2 GB per a Android Studio, més almenys 4 GB per a l'SDK d'Android i les imatges del sistema de l'emulador
  • Java Development Kit (JDK): JDK 8 o superior

Descarregar i Instal·lar Android Studio

  1. Descarregar Android Studio:

    • Visita la pàgina oficial de descàrrega d'Android Studio.
    • Tria la versió adequada per al teu sistema operatiu i descarrega l'instal·lador.

  2. Instal·lar Android Studio:

    • Windows:
      • Executa el fitxer .exe descarregat.
      • Segueix l'assistent de configuració per completar la instal·lació.
    • macOS:
      • Obre el fitxer .dmg descarregat.
      • Arrossega i deixa anar Android Studio a la carpeta d'Aplicacions.
    • Linux:
      • Extreu el fitxer .zip descarregat.
      • Obre una terminal i navega fins a la carpeta extreta.
      • Executa ./studio.sh per començar la instal·lació.

  3. Configuració inicial:

    • Llança Android Studio.
    • Segueix l'assistent de configuració per instal·lar els components necessaris de l'SDK.
    • Configura els paràmetres per defecte segons les teves preferències.

Configurar Android Studio

  1. SDK Manager:

    • Obre Android Studio.
    • Navega a File > Settings > Appearance & Behavior > System Settings > Android SDK (en macOS, Android Studio > Preferences).
    • Assegura't que les plataformes i eines necessàries de l'SDK estiguin instal·lades.

  2. AVD Manager:

    • Navega a Tools > AVD Manager.
    • Crea un nou dispositiu virtual seleccionant una definició de dispositiu i una imatge del sistema.
    • Configura els paràmetres del dispositiu i fes clic a Finish.

  3. Plugins:

    • Navega a File > Settings > Plugins (en macOS, Android Studio > Preferences).
    • Navega i instal·la plugins que millorin la teva experiència de desenvolupament, com Kotlin, Firebase i més.

Crear el teu Primer Projecte

  1. Iniciar un Nou Projecte:

    • Obre Android Studio.
    • Fes clic a Start a new Android Studio project.
    • Tria una plantilla de projecte (per exemple, Empty Activity) i fes clic a Next.

  2. Configurar el teu Projecte:

    • Nomena el teu projecte i especifica la ubicació de desament.
    • Tria el llenguatge (Java/Kotlin) i el nivell mínim de l'API.
    • Fes clic a Finish per crear el projecte.

  3. Estructura del Projecte:

    • Familiaritza't amb l'estructura del projecte:
      • app/src/main/java: Conté els teus fitxers de codi font Java/Kotlin.
      • app/src/main/res: Conté els teus fitxers de recursos (dissenys, cadenes de text, imatges).
      • build.gradle: Fitxers de configuració per al teu projecte i mòduls.

Executar la teva Primera Aplicació

  1. Connectar un Dispositiu:

    • Dispositiu Físic:
      • Activa les Opcions de Desenvolupador i el Depuració USB al teu dispositiu Android.
      • Connecta el teu dispositiu a l'ordinador via USB.
    • Emulador:
      • Obre l'AVD Manager i inicia un dispositiu virtual.

  2. Executar l'Aplicació:

    • Fes clic al botó Run a Android Studio.
    • Selecciona el dispositiu objectiu (físic o emulador).
    • Espera que l'aplicació es construeixi i s'instal·li al dispositiu.

  3. Depuració:

    • Utilitza la finestra de Logcat per veure els missatges de registre.
    • Estableix punts de ruptura i utilitza l'eina de Depuració per inspeccionar variables i controlar el flux d'execució.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu camí com a desenvolupador d'Android. Seguint els passos descrits en aquesta secció, ara hauries de tenir una configuració completa d'Android Studio, llesta per començar a construir i executar les teves primeres aplicacions d'Android. Recorda mantenir el teu SDK i eines actualitzats per aprofitar les últimes característiques i millores. Feliç programació!
