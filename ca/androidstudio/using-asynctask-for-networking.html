Introducció

En aquesta secció, explorarem com utilitzar AsyncTask per a networking en aplicacions Android. AsyncTask és una classe que et permet realitzar operacions en segon pla i publicar resultats en el fil de la interfície d'usuari sense haver de manipular fils i manejadors.

Conceptes Clau

  • AsyncTask: Una classe que permet un ús correcte i fàcil del fil de la interfície d'usuari. Permet realitzar operacions en segon pla i actualitzar el fil de la interfície d'usuari.
  • Networking: El procés de connectar-se i comunicar-se amb servidors remots per obtenir o enviar dades.

Entenent AsyncTask

AsyncTask està dissenyat per ser una classe auxiliar al voltant de Thread i Handler i no constitueix un marc de fils genèric. AsyncTask és ideal per a operacions curtes (uns pocs segons com a màxim).

Estructura Bàsica d'AsyncTask

Un AsyncTask es defineix per tres tipus genèrics:

  1. Params: El tipus dels paràmetres enviats a la tasca en executar-se.
  2. Progress: El tipus de les unitats de progrés publicades durant el càlcul en segon pla.
  3. Result: El tipus del resultat del càlcul en segon pla.
private class MyAsyncTask extends AsyncTask<Params, Progress, Result> {
    @Override
    protected void onPreExecute() {
        // S'executa en el fil de la interfície d'usuari abans que comenci el càlcul en segon pla.
        // Inicialitza qualsevol recurs necessari abans que comenci la tasca.
    }

    @Override
    protected Result doInBackground(Params... params) {
        // S'executa en un fil en segon pla.
        // Realitza el càlcul en segon pla aquí.
        return result;
    }

    @Override
    protected void onProgressUpdate(Progress... values) {
        // S'executa en el fil de la interfície d'usuari després que es cridi a publishProgress.
        // Actualitza la interfície d'usuari amb el progrés.
    }

    @Override
    protected void onPostExecute(Result result) {
        // S'executa en el fil de la interfície d'usuari després que acabi el càlcul en segon pla.
        // Actualitza la interfície d'usuari amb el resultat.
    }
}

Utilitzant AsyncTask per a Networking

Per realitzar operacions de networking utilitzant AsyncTask, segueix aquests passos:

Pas 1: Afegeix Permís d'Internet

Primer, assegura't que la teva aplicació tingui permís per accedir a internet. Afegeix la següent línia al teu fitxer AndroidManifest.xml:

<uses-permission android:name="android.permission.INTERNET" />

Pas 2: Crea la Classe AsyncTask

Crea una nova classe que estengui AsyncTask. Per exemple, creem una classe que obtingui dades d'una URL.

public class FetchDataTask extends AsyncTask<String, Void, String> {
    @Override
    protected void onPreExecute() {
        super.onPreExecute();
        // Inicialitza qualsevol recurs necessari abans que comenci la tasca.
    }

    @Override
    protected String doInBackground(String... urls) {
        String result = "";
        try {
            URL url = new URL(urls[0]);
            HttpURLConnection urlConnection = (HttpURLConnection) url.openConnection();
            InputStream in = new BufferedInputStream(urlConnection.getInputStream());
            result = readStream(in);
        } catch (Exception e) {
            e.printStackTrace();
        }
        return result;
    }

    @Override
    protected void onPostExecute(String result) {
        super.onPostExecute(result);
        // Actualitza la interfície d'usuari amb el resultat.
    }

    private String readStream(InputStream in) {
        BufferedReader reader = new BufferedReader(new InputStreamReader(in));
        StringBuilder sb = new StringBuilder();
        String line;
        try {
            while ((line = reader.readLine()) != null) {
                sb.append(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                in.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }
        return sb.toString();
    }
}

Pas 3: Executa l'AsyncTask

Per executar l'AsyncTask, crea una instància de la classe i crida al mètode execute, passant la URL com a paràmetre.

new FetchDataTask().execute("https://api.example.com/data");

Exemple: Obtenint Dades JSON

Creem un exemple complet on obtenim dades JSON d'una URL i les mostrem en un TextView.

Pas 1: Fitxer de Disseny

Crea un fitxer de disseny activity_main.xml amb un TextView.

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:id="@+id/textView"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:text="Fetching data..."
        android:textSize="18sp" />
</LinearLayout>

Pas 2: MainActivity

A MainActivity, executa l'AsyncTask i actualitza el TextView amb les dades obtingudes.

public class MainActivity extends AppCompatActivity {
    private TextView textView;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        textView = findViewById(R.id.textView);
        new FetchDataTask().execute("https://api.example.com/data");
    }

    private class FetchDataTask extends AsyncTask<String, Void, String> {
        @Override
        protected void onPreExecute() {
            super.onPreExecute();
            // Inicialitza qualsevol recurs necessari abans que comenci la tasca.
        }

        @Override
        protected String doInBackground(String... urls) {
            String result = "";
            try {
                URL url = new URL(urls[0]);
                HttpURLConnection urlConnection = (HttpURLConnection) url.openConnection();
                InputStream in = new BufferedInputStream(urlConnection.getInputStream());
                result = readStream(in);
            } catch (Exception e) {
                e.printStackTrace();
            }
            return result;
        }

        @Override
        protected void onPostExecute(String result) {
            super.onPostExecute(result);
            textView.setText(result);
        }

        private String readStream(InputStream in) {
            BufferedReader reader = new BufferedReader(new InputStreamReader(in));
            StringBuilder sb = new StringBuilder();
            String line;
            try {
                while ((line = reader.readLine()) != null) {
                    sb.append(line);
                }
            } catch (IOException e) {
                e.printStackTrace();
            } finally {
                try {
                    in.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
            return sb.toString();
        }
    }
}

Conclusió

En aquesta secció, hem après com utilitzar AsyncTask per a networking en aplicacions Android. Hem cobert l'estructura bàsica de AsyncTask, com realitzar operacions de networking, i hem proporcionat un exemple complet d'obtenció de dades JSON d'una URL i mostrar-les en un TextView. Tot i que AsyncTask és útil per a tasques senzilles, per a operacions més complexes o de llarga durada, considera utilitzar altres marcs de concurrència com RxJava o WorkManager.
