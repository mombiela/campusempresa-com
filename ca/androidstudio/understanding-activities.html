Introducció a les Activitats

Les activitats són un component fonamental de les aplicacions Android. Representen una sola pantalla amb una interfície d'usuari. Una aplicació pot tenir múltiples activitats, i cada activitat és independent de les altres.

Conceptes Clau

  • Cicle de Vida de l'Activitat: La sèrie d'estats pels quals passa una activitat des de la seva creació fins a la seva destrucció.
  • Intencions: Objectes de missatgeria utilitzats per sol·licitar una acció d'un altre component de l'aplicació.
  • Components de la IU: Ginys i dissenys que formen la interfície d'usuari d'una activitat.

Cicle de Vida de l'Activitat

Comprendre el cicle de vida de l'activitat és crucial per gestionar l'estat de la teva aplicació i assegurar una experiència d'usuari suau.

Mètodes del Cicle de Vida

  • onCreate(): Es crida quan l'activitat es crea per primera vegada. Aquí és on inicialitzes la teva activitat.
  • onStart(): Es crida quan l'activitat es fa visible per a l'usuari.
  • onResume(): Es crida quan l'activitat comença a interactuar amb l'usuari.
  • onPause(): Es crida quan el sistema està a punt de començar a reprendre una altra activitat.
  • onStop(): Es crida quan l'activitat ja no és visible per a l'usuari.
  • onDestroy(): Es crida abans que l'activitat sigui destruïda.

Diagrama del Cicle de Vida

Aquí tens un diagrama simplificat del cicle de vida de l'activitat:

onCreate() -> onStart() -> onResume()
   ^            |            |
   |            v            v
onDestroy() <- onStop() <- onPause()

Exemple de Codi

Aquí tens una implementació bàsica d'una activitat amb mètodes del cicle de vida:

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        Log.d("Lifecycle", "onCreate called");
    }

    @Override
    protected void onStart() {
        super.onStart();
        Log.d("Lifecycle", "onStart called");
    }

    @Override
    protected void onResume() {
        super.onResume();
        Log.d("Lifecycle", "onResume called");
    }

    @Override
    protected void onPause() {
        super.onPause();
        Log.d("Lifecycle", "onPause called");
    }

    @Override
    protected void onStop() {
        super.onStop();
        Log.d("Lifecycle", "onStop called");
    }

    @Override
    protected void onDestroy() {
        super.onDestroy();
        Log.d("Lifecycle", "onDestroy called");
    }
}

Intencions

Les intencions s'utilitzen per iniciar activitats i comunicar-se entre diferents components d'una aplicació.

Tipus d'Intencions

  • Intencions Explícites: Especifica el component a iniciar pel seu nom.
  • Intencions Implícites: No especifiquen directament el component; en canvi, declaren una acció general a realitzar.

Exemple de Codi

Aquí tens un exemple d'ús d'una intenció explícita per iniciar una nova activitat:

Intent intent = new Intent(this, SecondActivity.class);
startActivity(intent);

I aquí tens un exemple d'ús d'una intenció implícita per obrir una pàgina web:

Intent intent = new Intent(Intent.ACTION_VIEW);
intent.setData(Uri.parse("http://www.example.com"));
startActivity(intent);

Components de la IU

Les activitats es componen de diversos components de la IU, com ara botons, vistes de text i dissenys.

Components Comuns de la IU

  • TextView: Mostra text a l'usuari.
  • Button: Un botó clicable.
  • EditText: Un camp de text que permet l'entrada de l'usuari.
  • LinearLayout: Un disseny que organitza els seus fills en una sola columna o fila.

Exemple de Codi

Aquí tens un exemple d'un disseny d'activitat simple utilitzant XML:

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hola, Món!" />

    <Button
        android:id="@+id/button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Clica'm" />
</LinearLayout>

Conclusió

Comprendre les activitats és essencial per desenvolupar aplicacions Android. En dominar el cicle de vida de l'activitat, les intencions i els components de la IU, pots crear aplicacions robustes i fàcils d'utilitzar. Recorda gestionar adequadament els estats del cicle de vida per assegurar una experiència d'usuari suau i utilitza les intencions per navegar efectivament entre diferents parts de la teva aplicació.
