Passar dades entre activitats és un concepte fonamental en el desenvolupament d'Android. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades per compartir dades entre activitats en una aplicació Android.

Introducció als Intents

Els intents són el mètode principal per iniciar noves activitats i passar dades entre elles.

  • Intent: Una descripció abstracta d'una operació a realitzar.
  • Intent Explícit: Especifica l'activitat exacta a iniciar.
  • Intent Implícit: Declara una acció general a realitzar, permetent al sistema trobar una activitat adequada.

Creant un Intent

Per iniciar una nova activitat, crees un objecte Intent i crides startActivity().

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
startActivity(intent);

Passar Dades amb Intents

Pots passar dades entre activitats utilitzant el mètode putExtra de la classe Intent.

Tipus de Dades Bàsiques

Pots passar tipus de dades bàsiques com int, String, boolean, etc.

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MESSAGE", "Hola, Món!");
intent.putExtra("EXTRA_NUMBER", 123);
startActivity(intent);

Recuperant Dades a l'Activitat Destí

A l'activitat destí, recuperes les dades utilitzant el mètode getIntent() i els mètodes get adequats.

Intent intent = getIntent();
String message = intent.getStringExtra("EXTRA_MESSAGE");
int number = intent.getIntExtra("EXTRA_NUMBER", 0);

Passar Tipus de Dades Complexes

Per a tipus de dades més complexes, com objectes personalitzats, necessites implementar la interfície Parcelable o Serializable.

Utilitzant Parcelable

Parcelable és el mètode preferit per passar tipus de dades complexes perquè és més eficient que Serializable.

Creant una Classe Parcelable

public class MyObject implements Parcelable {
    private int id;
    private String name;

    // Constructor
    public MyObject(int id, String name) {
        this.id = id;
        this.name = name;
    }

    // Implementació de Parcelable
    protected MyObject(Parcel in) {
        id = in.readInt();
        name = in.readString();
    }

    public static final Creator<MyObject> CREATOR = new Creator<MyObject>() {
        @Override
        public MyObject createFromParcel(Parcel in) {
            return new MyObject(in);
        }

        @Override
        public MyObject[] newArray(int size) {
            return new MyObject[size];
        }
    };

    @Override
    public int describeContents() {
        return 0;
    }

    @Override
    public void writeToParcel(Parcel dest, int flags) {
        dest.writeInt(id);
        dest.writeString(name);
    }
}

Passant un Objecte Parcelable

MyObject myObject = new MyObject(1, "Exemple");
Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MY_OBJECT", myObject);
startActivity(intent);

Recuperant un Objecte Parcelable

Intent intent = getIntent();
MyObject myObject = intent.getParcelableExtra("EXTRA_MY_OBJECT");

Tècniques Avançades

Utilitzant Bundles

Els bundles són útils per passar múltiples peces de dades juntes.

Bundle bundle = new Bundle();
bundle.putString("EXTRA_MESSAGE", "Hola, Món!");
bundle.putInt("EXTRA_NUMBER", 123);

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtras(bundle);
startActivity(intent);

Utilitzant Shared Preferences

Per a dades persistents que necessiten ser accessibles a través de múltiples activitats, considera utilitzar Shared Preferences.

// Guardant dades
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("EXTRA_MESSAGE", "Hola, Món!");
editor.apply();

// Recuperant dades
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
String message = sharedPreferences.getString("EXTRA_MESSAGE", "Valor per Defecte");

Conclusió

Passar dades entre activitats és una habilitat crucial en el desenvolupament d'Android. Dominant intents, bundles i shared preferences, pots compartir dades de manera eficient i crear una experiència d'usuari fluida. Practica aquestes tècniques per convertir-te en un expert en la gestió del flux de dades en les teves aplicacions Android.
