Introducció a Jetpack Compose

Jetpack Compose és l'eina moderna d'Android per construir interfícies d'usuari natives. Simplifica i accelera el desenvolupament d'interfícies d'usuari a Android amb menys codi, eines potents i APIs intuïtives de Kotlin.

Conceptes Clau

  • Interfície Declarativa: Compose et permet descriure la teva interfície d'usuari de manera declarativa.
  • Funcions Composables: Funcions anotades amb @Composable que defineixen els teus components d'interfície d'usuari.
  • Gestió d'Estat: Gestionar l'estat de la interfície d'usuari de manera reactiva.
  • Dissenys: Organitzar els components de la interfície d'usuari de manera flexible i eficient.

Configurant Jetpack Compose

Abans de submergir-te en Jetpack Compose, assegura't que el teu entorn de desenvolupament està configurat correctament.

Passos per Configurar

  1. Instal·la Android Studio: Assegura't que tens la versió més recent d'Android Studio.
  2. Crea un Nou Projecte: Comença un nou projecte amb Jetpack Compose habilitat.
  3. Afegeix Dependències: Assegura't que els teus fitxers build.gradle inclouen les dependències necessàries per Jetpack Compose.
dependencies {
    implementation "androidx.compose.ui:ui:1.0.0"
    implementation "androidx.compose.material:material:1.0.0"
    implementation "androidx.compose.ui:ui-tooling-preview:1.0.0"
    implementation "androidx.lifecycle:lifecycle-runtime-ktx:2.3.1"
    implementation "androidx.activity:activity-compose:1.3.0-alpha06"
}

Construint el teu Primer Composable

Comencem creant una funció composable simple que mostri un text.

Exemple: Hola Món

import androidx.compose.material.Text
import androidx.compose.runtime.Composable

@Composable
fun Greeting(name: String) {
    Text(text = "Hola, $name!")
}

Explicació

  • @Composable: Aquesta anotació marca la funció com a composable.
  • Text: Un composable bàsic que mostra text a la pantalla.
  • Paràmetres: El paràmetre name permet contingut dinàmic.

Dissenys en Jetpack Compose

Compose proporciona diversos composables de disseny per organitzar els teus components d'interfície d'usuari.

Exemple: Column i Row

import androidx.compose.foundation.layout.Column
import androidx.compose.foundation.layout.Row
import androidx.compose.material.Text
import androidx.compose.runtime.Composable

@Composable
fun ColumnExample() {
    Column {
        Text("Primer element")
        Text("Segon element")
        Text("Tercer element")
    }
}

@Composable
fun RowExample() {
    Row {
        Text("Primer element")
        Text("Segon element")
        Text("Tercer element")
    }
}

Explicació

  • Column: Organitza els fills verticalment.
  • Row: Organitza els fills horitzontalment.

Gestió d'Estat

Gestionar l'estat en Jetpack Compose és crucial per construir interfícies d'usuari interactives.

Exemple: Estat en Compose

import androidx.compose.runtime.*
import androidx.compose.material.*

@Composable
fun Counter() {
    var count by remember { mutableStateOf(0) }

    Column {
        Text("Compte: $count")
        Button(onClick = { count++ }) {
            Text("Incrementar")
        }
    }
}

Explicació

  • remember: Utilitzat per recordar l'estat a través de recomposicions.
  • mutableStateOf: Crea un estat mutable.
  • Button: Un composable que desencadena una acció quan es fa clic.

Temes Avançats

A mesura que et familiaritzis amb Jetpack Compose, pots explorar temes més avançats.

Composables Personalitzats

Crear composables personalitzats permet components d'interfície d'usuari reutilitzables i modulars.

@Composable
fun CustomCard(content: @Composable () -> Unit) {
    Card(elevation = 4.dp) {
        content()
    }
}

Tematització i Estilització

Compose et permet definir temes i estils per mantenir una aparença consistent.

import androidx.compose.material.MaterialTheme
import androidx.compose.material.Typography

@Composable
fun MyApp() {
    MaterialTheme(
        typography = Typography(
            h1 = TextStyle(
                fontWeight = FontWeight.Bold,
                fontSize = 30.sp
            )
        )
    ) {
        // El teu contingut de l'aplicació
    }
}

Conclusió

Jetpack Compose és una eina poderosa que simplifica el desenvolupament d'interfícies d'usuari a Android. Entenent els conceptes bàsics de les funcions composables, els dissenys i la gestió d'estat, pots construir interfícies d'usuari dinàmiques i responsives. A mesura que avancis, explorar composables personalitzats i tematització t'ajudarà a crear aplicacions més complexes i polides. Continua experimentant i construint per dominar Jetpack Compose!
