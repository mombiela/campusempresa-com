Introducció

En el desenvolupament d'Android, entendre com crear vistes personalitzades i grups de vistes és essencial per construir interfícies d'usuari úniques i altament interactives. Aquesta secció et guiarà a través del procés de creació de vistes personalitzades i grups de vistes, des dels conceptes bàsics fins a tècniques avançades.

Vistes personalitzades

Què és una vista personalitzada?

  • Una vista personalitzada és un component definit per l'usuari que amplia la funcionalitat de les vistes d'Android existents.
  • Les vistes personalitzades permeten als desenvolupadors crear components d'interfície d'usuari reutilitzables que es poden adaptar a necessitats específiques.

Quan utilitzar vistes personalitzades

  • Quan les vistes existents no compleixen els teus requisits de disseny.
  • Quan necessites crear un component reutilitzable que es pugui utilitzar en diverses activitats o fragments.
  • Quan vols optimitzar el rendiment reduint el nombre de jerarquies de vistes.

Creació d'una vista personalitzada simple

  1. Extendre la classe View

    public class MyCustomView extends View {
    public MyCustomView(Context context) {
        super(context);
    }

    public MyCustomView(Context context, AttributeSet attrs) {
        super(context, attrs);
    }

    @Override
    protected void onDraw(Canvas canvas) {
        super.onDraw(canvas);
        // Codi de dibuix personalitzat aquí
        Paint paint = new Paint();
        paint.setColor(Color.RED);
        canvas.drawCircle(50, 50, 30, paint);
    }
}

  2. Utilitzar la vista personalitzada en XML

    <com.example.myapp.MyCustomView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"/>

Gestió d'atributs personalitzats

  • Defineix atributs personalitzats a res/values/attrs.xml.

    <declare-styleable name="MyCustomView">
    <attr name="customColor" format="color"/>
</declare-styleable>

  • Recupera els atributs personalitzats en el constructor.

    public MyCustomView(Context context, AttributeSet attrs) {
    super(context, attrs);
    TypedArray a = context.getTheme().obtainStyledAttributes(
        attrs,
        R.styleable.MyCustomView,
        0, 0);

    try {
        int color = a.getColor(R.styleable.MyCustomView_customColor, Color.BLACK);
        // Utilitza el color
    } finally {
        a.recycle();
    }
}

Grups de vistes personalitzats

Què és un grup de vistes personalitzat?

  • Un grup de vistes personalitzat és un contenidor que pot contenir múltiples vistes filles i gestionar el seu disseny i dibuix.
  • Els grups de vistes personalitzats amplien la classe ViewGroup i sobreescriuen mètodes per controlar la posició i la mida de les vistes filles.

Creació d'un grup de vistes personalitzat simple

  1. Extendre la classe ViewGroup

    public class MyCustomViewGroup extends ViewGroup {
    public MyCustomViewGroup(Context context) {
        super(context);
    }

    public MyCustomViewGroup(Context context, AttributeSet attrs) {
        super(context, attrs);
    }

    @Override
    protected void onLayout(boolean changed, int l, int t, int r, int b) {
        // Lògica de disseny per a les vistes filles
        int childCount = getChildCount();
        for (int i = 0; i < childCount; i++) {
            View child = getChildAt(i);
            child.layout(l, t, r, b);
        }
    }
}

  2. Utilitzar el grup de vistes personalitzat en XML

    <com.example.myapp.MyCustomViewGroup
    android:layout_width="match_parent"
    android:layout_height="match_parent">
    <!-- Afegeix vistes filles aquí -->
    <Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Button"/>
</com.example.myapp.MyCustomViewGroup>

Lògica de disseny personalitzada

  • Sobreescriu onMeasure per mesurar la mida de les vistes filles. 
    @Override
protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec) {
    int width = 0;
    int height = 0;

    int childCount = getChildCount();
    for (int i = 0; i < childCount; i++) {
        View child = getChildAt(i);
        measureChild(child, widthMeasureSpec, heightMeasureSpec);
        width += child.getMeasuredWidth();
        height = Math.max(height, child.getMeasuredHeight());
    }

    setMeasuredDimension(width, height);
}

Tècniques avançades

Dibuix i animacions personalitzades

  • Utilitza la classe Canvas per al dibuix personalitzat.
  • Implementa animacions utilitzant ValueAnimator o ObjectAnimator.

Optimització del rendiment

  • Minimitza el nombre de vistes i grups de vistes personalitzats.
  • Utilitza ViewStub per a vistes que no sempre són necessàries.
  • Optimitza els mètodes onDraw i onMeasure.

Conclusió

Crear vistes i grups de vistes personalitzats a Android permet components d'interfície d'usuari altament personalitzats i reutilitzables. Entenent els conceptes bàsics d'extendre View i ViewGroup, gestionar atributs personalitzats i optimitzar el rendiment, pots crear interfícies d'usuari eficients i visualment atractives. Practica creant els teus propis components personalitzats i integrant-los en els teus projectes per consolidar el teu coneixement.
