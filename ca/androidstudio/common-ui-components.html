En aquesta secció, explorarem els components d'UI comuns utilitzats en el desenvolupament d'Android. Entendre aquests components és essencial per crear aplicacions intuïtives i visualment atractives. Cobrirem components bàsics a avançats, proporcionant exemples i exercicis per consolidar la teva comprensió.

Components d'UI Bàsics

TextView

  • Descripció: Mostra text a l'usuari.
  • Atributs: text, textColor, textSize, gravity, etc.
<TextView
    android:id="@+id/textView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hola, Món!"
    android:textSize="18sp"
    android:textColor="#000000"
    android:gravity="center"/>

EditText

  • Descripció: Permet a l'usuari introduir i editar text.
  • Atributs: hint, inputType, text, etc.
<EditText
    android:id="@+id/editText"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:hint="Introdueix el teu nom"
    android:inputType="textPersonName"/>

Button

  • Descripció: Un botó clicable que realitza una acció.
  • Atributs: text, onClick, background, etc.
<Button
    android:id="@+id/button"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Enviar"
    android:onClick="onSubmit"/>

ImageView

  • Descripció: Mostra una imatge.
  • Atributs: src, scaleType, contentDescription, etc.
<ImageView
    android:id="@+id/imageView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:src="@drawable/ic_launcher_foreground"
    android:contentDescription="Icona de l'App"/>

Components d'UI Intermedis

RecyclerView

  • Descripció: Una vista flexible per proporcionar una finestra limitada a un conjunt de dades gran.
  • Atributs: layoutManager, adapter, etc.
<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/recyclerView"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:layoutManager="LinearLayoutManager"/>

CardView

  • Descripció: Un disseny de marc amb cantonades arrodonides i una ombra.
  • Atributs: cardCornerRadius, cardElevation, etc.
<androidx.cardview.widget.CardView
    android:id="@+id/cardView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:layout_margin="8dp"
    app:cardCornerRadius="4dp"
    app:cardElevation="2dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Exemple de CardView"/>
</androidx.cardview.widget.CardView>

Toolbar

  • Descripció: Una generalització de ActionBar per a ús dins dels dissenys de l'aplicació.
  • Atributs: title, subtitle, navigationIcon, etc.
<com.google.android.material.appbar.Toolbar
    android:id="@+id/toolbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="?attr/colorPrimary"
    android:title="La Meva Barra d'Eines"/>

Components d'UI Avançats

NavigationView

  • Descripció: Un menú de navegació estàndard per a l'aplicació.
  • Atributs: menu, headerLayout, etc.
<com.google.android.material.navigation.NavigationView
    android:id="@+id/navigationView"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:layout_gravity="start"
    app:menu="@menu/navigation_menu"
    app:headerLayout="@layout/nav_header"/>

ViewPager2

  • Descripció: Permet a l'usuari passar pàgines de dades cap a l'esquerra i la dreta.
  • Atributs: orientation, adapter, etc.
<androidx.viewpager2.widget.ViewPager2
    android:id="@+id/viewPager"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="horizontal"/>

ConstraintLayout

  • Descripció: Un disseny que et permet crear dissenys grans i complexos amb una jerarquia de vista plana.
  • Atributs: constraintSet, layout_constraint, etc.
<androidx.constraintlayout.widget.ConstraintLayout
    android:id="@+id/constraintLayout"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Clica'm"
        app:layout_constraintTop_toTopOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>
</androidx.constraintlayout.widget.ConstraintLayout>

Resum

En aquest tema, hem cobert una gamma de components d'UI comuns en el desenvolupament d'Android, des d'elements bàsics com TextView i Button fins a components més avançats com RecyclerView i ConstraintLayout. Cada component serveix a un propòsit específic i es pot personalitzar a través de diversos atributs per adaptar-se a les necessitats de la teva aplicació. Entenent i utilitzant aquests components, pots crear interfícies més dinàmiques i fàcils d'usar en les teves aplicacions d'Android.
