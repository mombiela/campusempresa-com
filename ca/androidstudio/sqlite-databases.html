Introducció a SQLite en Android

SQLite és una base de dades lleugera basada en disc que no requereix un procés de servidor separat i permet l'accés a la base de dades utilitzant una variant no estàndard del llenguatge de consulta SQL. És una opció popular per a l'emmagatzematge local en aplicacions Android a causa de la seva simplicitat i eficiència.

Configuració de SQLite a Android Studio

Per començar a utilitzar SQLite en el teu projecte Android, segueix aquests passos:

  1. Crea un nou projecte Android a Android Studio.
  2. Afegeix una nova classe per gestionar les operacions de la base de dades.

Exemple: Creació d'una classe Helper per a la base de dades

public class MyDatabaseHelper extends SQLiteOpenHelper {

    private static final String DATABASE_NAME = "mydatabase.db";
    private static final int DATABASE_VERSION = 1;

    public MyDatabaseHelper(Context context) {
        super(context, DATABASE_NAME, null, DATABASE_VERSION);
    }

    @Override
    public void onCreate(SQLiteDatabase db) {
        // Crear taules
        String CREATE_TABLE = "CREATE TABLE my_table (id INTEGER PRIMARY KEY, name TEXT)";
        db.execSQL(CREATE_TABLE);
    }

    @Override
    public void onUpgrade(SQLiteDatabase db, int oldVersion, int newVersion) {
        // Lògica d'actualització
        db.execSQL("DROP TABLE IF EXISTS my_table");
        onCreate(db);
    }
}
  • DATABASE_NAME: El nom del fitxer de la base de dades.
  • DATABASE_VERSION: La versió de l'esquema de la base de dades.
  • onCreate: Es crida quan la base de dades es crea per primera vegada.
  • onUpgrade: Es crida quan la base de dades necessita ser actualitzada.

Operacions bàsiques CRUD

CRUD significa Crear, Llegir, Actualitzar i Eliminar. Aquestes són les quatre operacions bàsiques que pots realitzar en una base de dades.

Crear (Insertar)

public void addRecord(String name) {
    SQLiteDatabase db = this.getWritableDatabase();
    ContentValues values = new ContentValues();
    values.put("name", name);
    db.insert("my_table", null, values);
    db.close();
}
  • ContentValues: Una estructura de parell clau/valor per inserir dades a la base de dades.

Llegir (Consultar)

public List<String> getAllRecords() {
    List<String> records = new ArrayList<>();
    SQLiteDatabase db = this.getReadableDatabase();
    Cursor cursor = db.rawQuery("SELECT * FROM my_table", null);
    if (cursor.moveToFirst()) {
        do {
            records.add(cursor.getString(1)); // 1 és l'índex de la columna 'name'
        } while (cursor.moveToNext());
    }
    cursor.close();
    db.close();
    return records;
}
  • Cursor: S'utilitza per iterar sobre el conjunt de resultats d'una consulta a la base de dades.

Actualitzar

public void updateRecord(int id, String newName) {
    SQLiteDatabase db = this.getWritableDatabase();
    ContentValues values = new ContentValues();
    values.put("name", newName);
    db.update("my_table", values, "id = ?", new String[]{String.valueOf(id)});
    db.close();
}
  • update: Actualitza les files especificades a la base de dades.

Eliminar

public void deleteRecord(int id) {
    SQLiteDatabase db = this.getWritableDatabase();
    db.delete("my_table", "id = ?", new String[]{String.valueOf(id)});
    db.close();
}
  • delete: Elimina les files especificades de la base de dades.

Funcions avançades de SQLite

Transaccions

Les transaccions et permeten executar una sèrie d'operacions de base de dades com una única unitat de treball.

public void performTransaction() {
    SQLiteDatabase db = this.getWritableDatabase();
    db.beginTransaction();
    try {
        // Realitzar múltiples operacions
        db.execSQL("INSERT INTO my_table (name) VALUES ('John')");
        db.execSQL("INSERT INTO my_table (name) VALUES ('Doe')");
        db.setTransactionSuccessful();
    } finally {
        db.endTransaction();
    }
    db.close();
}
  • beginTransaction: Comença una nova transacció.
  • setTransactionSuccessful: Marca la transacció actual com a exitosa.
  • endTransaction: Finalitza la transacció actual.

Índexs

Els índexs poden millorar el rendiment de les teves consultes a la base de dades.

@Override
public void onCreate(SQLiteDatabase db) {
    String CREATE_TABLE = "CREATE TABLE my_table (id INTEGER PRIMARY KEY, name TEXT)";
    db.execSQL(CREATE_TABLE);
    String CREATE_INDEX = "CREATE INDEX index_name ON my_table(name)";
    db.execSQL(CREATE_INDEX);
}
  • CREATE INDEX: Crea un índex a la columna o columnes especificades.

Millors pràctiques

  • Utilitza consultes parametritzades per prevenir la injecció SQL.
  • Tanca les connexions a la base de dades per evitar fuites de memòria.
  • Utilitza transaccions per a operacions en bloc per assegurar la integritat de les dades.
  • Optimitza les consultes amb índexs i un disseny d'esquema adequat.

Conclusió

SQLite és una solució de base de dades potent i eficient per a aplicacions Android. Comprenent i implementant funcions bàsiques i avançades, pots gestionar eficaçment l'emmagatzematge de dades locals a les teves aplicacions. Practica les operacions CRUD, utilitza transaccions per a operacions complexes i segueix les millors pràctiques per assegurar que la base de dades de la teva aplicació sigui robusta i eficient.
