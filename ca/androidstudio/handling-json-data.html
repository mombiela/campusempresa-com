Introducció

Gestionar dades JSON (JavaScript Object Notation) és una habilitat crucial per als desenvolupadors d'Android. JSON és un format lleuger d'intercanvi de dades que és fàcil de llegir i escriure per als humans i fàcil de parsejar i generar per a les màquines. En aquest tema, cobrirem com gestionar dades JSON a Android Studio des de nivells bàsics fins a avançats.

Què és JSON?

  • JSON és un format de text que és completament independent del llenguatge.
  • S'utilitza principalment per transmetre dades entre un servidor i una aplicació web.
  • Les dades JSON es representen com a parells clau-valor.

Exemple de Dades JSON

{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}

Parsejar Dades JSON a Android

Utilitzant JSONObject i JSONArray

Android proporciona el paquet org.json per parsejar dades JSON.

Parsejar un Objecte JSON Simple

String jsonString = "{\"name\":\"John Doe\",\"age\":30,\"isStudent\":false}";
try {
    JSONObject jsonObject = new JSONObject(jsonString);
    String name = jsonObject.getString("name");
    int age = jsonObject.getInt("age");
    boolean isStudent = jsonObject.getBoolean("isStudent");

    // Utilitzar les dades parsejades
    Log.d("JSON Parsing", "Name: " + name + ", Age: " + age + ", Is Student: " + isStudent);
} catch (JSONException e) {
    e.printStackTrace();
}

Parsejar un Array JSON

String jsonString = "[{\"name\":\"John Doe\",\"age\":30},{\"name\":\"Jane Doe\",\"age\":25}]";
try {
    JSONArray jsonArray = new JSONArray(jsonString);
    for (int i = 0; i < jsonArray.length(); i++) {
        JSONObject jsonObject = jsonArray.getJSONObject(i);
        String name = jsonObject.getString("name");
        int age = jsonObject.getInt("age");

        // Utilitzar les dades parsejades
        Log.d("JSON Parsing", "Name: " + name + ", Age: " + age);
    }
} catch (JSONException e) {
    e.printStackTrace();
}

Parseig Avançat de JSON amb Gson

Gson és una biblioteca popular proporcionada per Google per convertir objectes Java a JSON i viceversa.

Afegir la Dependència de Gson

Afegeix la següent dependència al teu fitxer build.gradle:

implementation 'com.google.code.gson:gson:2.8.8'

Parsejar JSON a Objectes Java

Definir Classes Java

public class Person {
    private String name;
    private int age;
    private boolean isStudent;

    // Getters i Setters
}

Parsejar una Cadena JSON a un Objecte Java

String jsonString = "{\"name\":\"John Doe\",\"age\":30,\"isStudent\":false}";
Gson gson = new Gson();
Person person = gson.fromJson(jsonString, Person.class);

// Utilitzar les dades parsejades
Log.d("Gson Parsing", "Name: " + person.getName() + ", Age: " + person.getAge() + ", Is Student: " + person.isStudent());

Serialitzar Objectes Java a JSON

Person person = new Person();
person.setName("John Doe");
person.setAge(30);
person.setStudent(false);

Gson gson = new Gson();
String jsonString = gson.toJson(person);

// Utilitzar la cadena JSON
Log.d("Gson Serialization", "JSON: " + jsonString);

Gestió d'Objectes JSON Imbricats

Exemple de JSON Imbricat

{
  "name": "John Doe",
  "age": 30,
  "address": {
    "street": "123 Main St",
    "city": "New York",
    "zip": "10001"
  }
}

Parsejar JSON Imbricat amb Gson

Definir Classes Java

public class Address {
    private String street;
    private String city;
    private String zip;

    // Getters i Setters
}

public class Person {
    private String name;
    private int age;
    private Address address;

    // Getters i Setters
}

Parsejar una Cadena JSON Imbricada a un Objecte Java

String jsonString = "{\"name\":\"John Doe\",\"age\":30,\"address\":{\"street\":\"123 Main St\",\"city\":\"New York\",\"zip\":\"10001\"}}";
Gson gson = new Gson();
Person person = gson.fromJson(jsonString, Person.class);

// Utilitzar les dades parsejades
Log.d("Gson Parsing", "Name: " + person.getName() + ", Age: " + person.getAge() + ", Street: " + person.getAddress().getStreet());

Conclusió

Gestionar dades JSON és una habilitat essencial per als desenvolupadors d'Android. Tant si estàs parsejant objectes JSON simples com si estàs tractant amb estructures imbricades complexes, entendre com utilitzar tant el paquet org.json com la biblioteca Gson millorarà enormement la teva capacitat per gestionar dades a les teves aplicacions. Practica aquestes tècniques per convertir-te en un expert en la gestió de dades JSON a Android Studio.
