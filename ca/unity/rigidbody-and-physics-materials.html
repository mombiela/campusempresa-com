Introducció

A Unity, els RigidBody i els Materials Físics són components essencials per crear comportaments físics realistes en els teus jocs. Aquest tema et guiarà des dels conceptes bàsics fins als avançats, assegurant que entenguis com utilitzar efectivament aquests components en els teus projectes.

RigidBody

Què és un RigidBody?

Un RigidBody és un component que permet que un GameObject sigui afectat per forces físiques, com la gravetat i les col·lisions. És crucial per a qualsevol objecte que necessiti moure's o interactuar físicament dins del món del joc.

Afegir un RigidBody

Per afegir un RigidBody a un GameObject:

  1. Selecciona el GameObject a la Jerarquia.
  2. A l'Inspector, fes clic a "Add Component".
  3. Escriu "Rigidbody" i selecciona'l de la llista.

Propietats del RigidBody

  • Mass: El pes de l'objecte. Afecta com les forces impacten l'objecte.
  • Drag: Resistència al moviment. Valors més alts ralenteixen l'objecte.
  • Angular Drag: Resistència a la rotació.
  • Use Gravity: Si està marcat, l'objecte serà afectat per la gravetat.
  • Is Kinematic: Si està marcat, l'objecte no serà afectat per forces físiques però encara pot interactuar amb altres objectes físics.

Exemple bàsic de RigidBody

using UnityEngine;

public class SimpleRigidbody : MonoBehaviour
{
    private Rigidbody rb;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
        rb.mass = 2.0f;
        rb.drag = 0.5f;
        rb.angularDrag = 0.05f;
        rb.useGravity = true;
    }

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector3.up * 5.0f, ForceMode.Impulse);
        }
    }
}

En aquest exemple:

  • Obtenim el component RigidBody adjunt al GameObject.
  • Establim diverses propietats del RigidBody.
  • Apliquem una força cap amunt quan es prem la tecla d'espai.

Materials Físics

Què són els Materials Físics?

Els Materials Físics defineixen com les superfícies interactuen entre elles en termes de fricció i rebot. S'utilitzen per crear interaccions físiques més realistes.

Crear un Material Físic

  1. A la finestra del Projecte, fes clic dret i selecciona "Create > Physics Material".
  2. Dona un nom al teu nou Material Físic.
  3. Selecciona el Material Físic i ajusta les seves propietats a l'Inspector.

Propietats del Material Físic

  • Dynamic Friction: Fricció quan l'objecte es mou.
  • Static Friction: Fricció quan l'objecte està estacionari.
  • Bounciness: Quant rebota l'objecte després d'una col·lisió.
  • Friction Combine: Com es combina la fricció amb altres materials (Average, Minimum, Maximum, Multiply).
  • Bounce Combine: Com es combina el rebot amb altres materials (Average, Minimum, Maximum, Multiply).

Aplicar un Material Físic

Per aplicar un Material Físic a un Collider:

  1. Selecciona el GameObject amb el Collider.
  2. Al component Collider, troba el camp "Material".
  3. Arrossega i deixa anar el teu Material Físic en aquest camp.

Exemple bàsic de Material Físic

using UnityEngine;

public class SimplePhysicsMaterial : MonoBehaviour
{
    void Start()
    {
        Collider col = GetComponent<Collider>();
        PhysicsMaterial2D mat = new PhysicsMaterial2D();
        mat.friction = 0.4f;
        mat.bounciness = 0.8f;
        col.sharedMaterial = mat;
    }
}

En aquest exemple:

  • Creem un nou Material Físic en codi.
  • Establim les seves propietats de fricció i rebot.
  • L'apliquem al Collider del GameObject.

Conceptes Avançats

Restriccions del RigidBody

Les restriccions del RigidBody et permeten restringir el moviment i la rotació d'un RigidBody. Això és útil per crear objectes que només s'han de moure de maneres específiques.

  • Freeze Position: Prevé el moviment al llarg dels eixos especificats.
  • Freeze Rotation: Prevé la rotació al voltant dels eixos especificats.

Interpolació del RigidBody

La interpolació suavitza el moviment d'un RigidBody, fent que sembli més fluid. Això és particularment útil per a objectes que es mouen ràpidament o tenen interaccions a alta velocitat.

  • None: Sense interpolació.
  • Interpolate: Suavitza el moviment basat en el fotograma anterior.
  • Extrapolate: Preveu el moviment basat en el següent fotograma.

Combinació de RigidBody i Materials Físics

La combinació de RigidBody i Materials Físics pot crear comportaments complexos i realistes. Per exemple, una pilota amb un Material Físic de rebot alt i un RigidBody amb poca massa rebotarà de manera més realista.

Conclusió

Entendre els RigidBody i els Materials Físics és crucial per crear interaccions físiques realistes a Unity. En dominar aquests components, pots millorar el realisme i la interactivitat dels teus objectes de joc. Experimenta amb diferents propietats i combinacions per veure com afecten el teu món del joc.
