Reduir les crides de dibuix a Unity
Optimitzant Scripts ►

Reduir les crides de dibuix és una tècnica d'optimització crucial a Unity per millorar el rendiment del teu joc. Aquest tema et guiarà des dels conceptes bàsics fins a estratègies avançades per minimitzar les crides de dibuix.

Què són les crides de dibuix?

  • Definició: Una crida de dibuix és una ordre enviada a la GPU per dibuixar un conjunt de vèrtexs. Cada crida de dibuix implica una sobrecàrrega de la CPU.
  • Impacte en el rendiment: Un nombre elevat de crides de dibuix pot reduir significativament la taxa de fotogrames del teu joc, especialment en dispositius mòbils.

Estratègies bàsiques per reduir les crides de dibuix

Combinació de malles

  • Batching estàtic: Combina objectes estàtics (no mòbils) en una sola malla en el moment de la compilació. 
    // Exemple: Marcar un GameObject com a estàtic per al batching
gameObject.isStatic = true;
  • Batching dinàmic: Combina petits objectes dinàmics (mòbils) en una sola malla en temps d'execució. 
    // Assegura't que els objectes siguin prou petits per ser agrupats dinàmicament
// Unity s'encarrega d'això automàticament si es compleixen les condicions

Ús de materials eficients

  • Instanciació de materials: Utilitza el mateix material per a múltiples objectes per reduir les crides de dibuix. 
    // Exemple: Assignar el mateix material a múltiples objectes
Renderer renderer1 = object1.GetComponent<Renderer>();
Renderer renderer2 = object2.GetComponent<Renderer>();
renderer2.material = renderer1.material;
  • Atlases de textures: Combina múltiples textures en una sola textura per reduir els canvis de material. 
    // Exemple: Ús d'un atlas de textures
// Crea un atlas de textures en un programa d'edició d'imatges i aplica'l als teus materials

Estratègies intermèdies per reduir les crides de dibuix

Nivell de detall (LOD)

  • Definició: Utilitza diferents models amb diversos nivells de detall segons la distància de la càmera. 
    // Exemple: Configuració del LOD a Unity
LODGroup lodGroup = gameObject.AddComponent<LODGroup>();
LOD[] lods = new LOD[3];
lods[0] = new LOD(0.6f, new Renderer[] { highDetailRenderer });
lods[1] = new LOD(0.3f, new Renderer[] { mediumDetailRenderer });
lods[2] = new LOD(0.1f, new Renderer[] { lowDetailRenderer });
lodGroup.SetLODs(lods);
lodGroup.RecalculateBounds();

Occlusion Culling

  • Definició: Evita renderitzar objectes que no són visibles per la càmera. 
    // Exemple: Configuració de l'occlusion culling a Unity
// Activa l'occlusion culling en la configuració del component Camera
Camera.main.useOcclusionCulling = true;

Estratègies avançades per reduir les crides de dibuix

Instanciació de la GPU

  • Definició: Dibuixa moltes instàncies de la mateixa malla amb una sola crida de dibuix. 
    // Exemple: Activar la instanciació de la GPU en un material
material.enableInstancing = true;
  • Shaders personalitzats: Escriu shaders que suportin la instanciació. 
    // Exemple: Shader bàsic d'instanciació
Shader "Custom/InstancingShader" {
    Properties {
        _Color ("Color", Color) = (1,1,1,1)
    }
    SubShader {
        Pass {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile_instancing
            #include "UnityCG.cginc"

            struct appdata {
                float4 vertex : POSITION;
                UNITY_INSTANCE_ID
            };

            struct v2f {
                float4 pos : SV_POSITION;
            };

            UNITY_INSTANCING_BUFFER_START(Props)
                UNITY_INSTANCING_BUFFER_END(Props)

            v2f vert (appdata v) {
                v2f o;
                UNITY_SETUP_INSTANCE_ID(v);
                o.pos = UnityObjectToClipPos(v.vertex);
                return o;
            }

            fixed4 frag (v2f i) : SV_Target {
                return fixed4(1,1,1,1);
            }
            ENDCG
        }
    }
}

Buffers de comandes

  • Definició: Et permet controlar el pipeline de renderització de manera més directa. 
    // Exemple: Ús de buffers de comandes
CommandBuffer commandBuffer = new CommandBuffer();
commandBuffer.DrawRenderer(renderer, material);
Camera.main.AddCommandBuffer(CameraEvent.AfterForwardOpaque, commandBuffer);

Resum

Reduir les crides de dibuix és essencial per optimitzar el rendiment del teu joc a Unity. Combinant malles, utilitzant materials eficients, implementant LOD i aprofitant tècniques avançades com la instanciació de la GPU i els buffers de comandes, pots reduir significativament el nombre de crides de dibuix i millorar la taxa de fotogrames del teu joc. Sempre perfil·la el teu joc per identificar colls d'ampolla i aplica aquestes estratègies on tindran més impacte.
