Creant i Personalitzant Elements de la UI ►

Visió general

La UI de Unity és un sistema potent i flexible per crear interfícies d'usuari en aplicacions de Unity. Aquesta secció cobrirà els conceptes bàsics de la UI de Unity, incloent com crear i manipular elements de la UI, gestionar dissenys i manejar interaccions d'usuari.

Conceptes clau

  • Canvas: El component arrel per a tots els elements de la UI a Unity. Actua com un contenidor per als elements de la UI i determina com es renderitzen.
  • Elements de la UI: Blocs bàsics per crear interfícies d'usuari, com Text, Image, Button, Slider, i més.
  • RectTransform: Un component especial per als elements de la UI que permet una posició i escalat flexibles.
  • Ancoratges i Pivots: Eines per posicionar i alinear elements de la UI en relació amb els seus contenidors pares.
  • Sistema d'Esdeveniments: Gestiona esdeveniments d'entrada i interaccions amb els elements de la UI.

Creant una UI bàsica

Configurant el Canvas

  1. Crear un Canvas: Clic dret a la finestra Hierarchy, anar a UI > Canvas.
  2. Escalador de Canvas: Ajustar el component Canvas Scaler per escalar la UI basant-se en la mida de la pantalla o altres criteris.

Afegint elements de la UI

  1. Text: Clic dret al Canvas, anar a UI > Text. Personalitzar el contingut i l'aparença del text utilitzant el component Text.
  2. Button: Clic dret al Canvas, anar a UI > Button. Personalitzar l'aparença del botó i afegir un esdeveniment OnClick per manejar interaccions d'usuari.

Exemple: Creant un botó simple

using UnityEngine;
using UnityEngine.UI;

public class ButtonExample : MonoBehaviour
{
    public Button myButton;

    void Start()
    {
        myButton.onClick.AddListener(OnButtonClick);
    }

    void OnButtonClick()
    {
        Debug.Log("Button clicked!");
    }
}
  • Explicació:
    • Adjuntar aquest script a un GameObject a la teva escena.
    • Assignar el component Button al camp myButton a l'Inspector.
    • Quan es faci clic al botó, es cridarà el mètode OnButtonClick, registrant un missatge a la consola.

Gestionant dissenys

Grups de disseny

  • Grup de Disseny Horitzontal: Disposa els elements fills en una línia horitzontal.
  • Grup de Disseny Vertical: Disposa els elements fills en una línia vertical.
  • Grup de Disseny de Graella: Disposa els elements fills en un format de graella.

Exemple: Utilitzant un Grup de Disseny Vertical

  1. Crear un Panell: Clic dret al Canvas, anar a UI > Panel.
  2. Afegir Grup de Disseny Vertical: Seleccionar el Panell, anar a Add Component > Vertical Layout Group.
  3. Afegir elements fills: Clic dret al Panell, anar a UI > Text o UI > Button per afegir elements fills. Es disposaran automàticament en vertical.

Manejant interaccions d'usuari

Sistema d'Esdeveniments

  • Mòdul d'Entrada Autònom: Gestiona l'entrada del ratolí, teclat i tàctil.
  • Activadors d'Esdeveniments: Permet definir esdeveniments personalitzats per als elements de la UI.

Exemple: Utilitzant Activadors d'Esdeveniments

  1. Afegir Activador d'Esdeveniments: Seleccionar un element de la UI (per exemple, un Botó), anar a Add Component > Event Trigger.
  2. Afegir Tipus d'Esdeveniment: Clicar Add New Event Type, seleccionar PointerEnter.
  3. Assignar Mètode: Afegir un mètode que es cridarà quan es desencadeni l'esdeveniment.
using UnityEngine;
using UnityEngine.EventSystems;

public class EventTriggerExample : MonoBehaviour, IPointerEnterHandler
{
    public void OnPointerEnter(PointerEventData eventData)
    {
        Debug.Log("Pointer entered!");
    }
}
  • Explicació:
    • Adjuntar aquest script a l'element de la UI.
    • Quan el punter entri a l'element, es cridarà el mètode OnPointerEnter, registrant un missatge a la consola.

Conclusió

La UI de Unity proporciona un conjunt complet d'eines per crear i gestionar interfícies d'usuari en els teus projectes de Unity. Entenent els conceptes bàsics de Canvas, elements de la UI, RectTransform i el Sistema d'Esdeveniments, pots crear interfícies interactives i visualment atractives. Practica creant diferents dissenys de la UI i manejant diverses interaccions d'usuari per convertir-te en un expert en la UI de Unity.
