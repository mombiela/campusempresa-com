Unity Physics és una característica poderosa que permet als desenvolupadors crear interaccions i comportaments realistes en els seus jocs. Aquest tema cobrirà els conceptes bàsics de Unity Physics, incloent conceptes clau, components i exemples pràctics per ajudar-te a començar.

Conceptes Clau

  • Motor de Física: Unity utilitza el motor NVIDIA PhysX per simular la física. Aquest motor maneja els càlculs necessaris per al moviment realista, col·lisions i altres interaccions físiques.
  • Rigidbody: Un component que permet que un GameObject sigui afectat per forces físiques com la gravetat i les col·lisions.
  • Colliders: Components que defineixen la forma d'un GameObject amb finalitats de col·lisions físiques.
  • Materials Físics: Actius que defineixen com interactuen les superfícies entre si, incloent propietats com la fricció i la rebotabilitat.
  • Forces: Mètodes per aplicar forces físiques als GameObjects, com la gravetat, impulsos i forces contínues.

Rigidbody

El component Rigidbody és essencial per a qualsevol GameObject que vulguis moure o que sigui afectat per la física.

Afegir un Rigidbody

Per afegir un Rigidbody a un GameObject:

  1. Selecciona el GameObject a la Jerarquia.
  2. A l'Inspector, fes clic a "Afegir Component".
  3. Cerca "Rigidbody" i afegeix-lo.

Propietats del Rigidbody

  • Mass: El pes del GameObject.
  • Drag: Resistència al moviment.
  • Angular Drag: Resistència a la rotació.
  • Use Gravity: Si el GameObject és afectat per la gravetat.
  • Is Kinematic: Si és cert, el GameObject no és afectat per forces físiques però encara pot interactuar amb altres objectes físics.

Exemple: Afegir un Rigidbody a un Cub

using UnityEngine;

public class AddRigidbody : MonoBehaviour
{
    void Start()
    {
        // Afegeix un component Rigidbody al GameObject al qual està adjunt aquest script
        Rigidbody rb = gameObject.AddComponent<Rigidbody>();
        rb.mass = 2.0f; // Estableix la massa del Rigidbody
    }
}

Colliders

Els Colliders defineixen la forma d'un GameObject per a la detecció de col·lisions. Hi ha diversos tipus de colliders:

  • Box Collider: Una forma de prisma rectangular.
  • Sphere Collider: Una forma esfèrica.
  • Capsule Collider: Una forma de càpsula.
  • Mesh Collider: Utilitza la malla del GameObject per a la detecció de col·lisions.

Afegir un Collider

Per afegir un Collider a un GameObject:

  1. Selecciona el GameObject a la Jerarquia.
  2. A l'Inspector, fes clic a "Afegir Component".
  3. Cerca el tipus de Collider desitjat (per exemple, "Box Collider") i afegeix-lo.

Exemple: Afegir un Box Collider a un Cub

using UnityEngine;

public class AddBoxCollider : MonoBehaviour
{
    void Start()
    {
        // Afegeix un component Box Collider al GameObject al qual està adjunt aquest script
        BoxCollider boxCollider = gameObject.AddComponent<BoxCollider>();
        boxCollider.size = new Vector3(1, 1, 1); // Estableix la mida del Box Collider
    }
}

Materials Físics

Els Materials Físics defineixen com interactuen les superfícies entre si. Tenen propietats com la fricció i la rebotabilitat.

Crear un Material Físic

  1. A la finestra del Projecte, fes clic dret i selecciona "Crear" -> "Material Físic".
  2. Nomena el nou Material Físic.
  3. A l'Inspector, estableix les propietats desitjades (per exemple, Fricció Dinàmica, Fricció Estàtica, Rebotabilitat).

Aplicar un Material Físic

  1. Selecciona el GameObject amb un Collider.
  2. Al component Collider, troba la propietat "Material".
  3. Arrossega el Material Físic des de la finestra del Projecte fins a la propietat "Material".

Exemple: Crear i Aplicar un Material Rebotador

using UnityEngine;

public class ApplyPhysicsMaterial : MonoBehaviour
{
    public PhysicsMaterial2D bouncyMaterial;

    void Start()
    {
        // Afegeix un component Box Collider al GameObject al qual està adjunt aquest script
        BoxCollider2D boxCollider = gameObject.AddComponent<BoxCollider2D>();
        // Aplica el material rebotador al Box Collider
        boxCollider.sharedMaterial = bouncyMaterial;
    }
}

Aplicar Forces

Les forces es poden aplicar als Rigidbodies per crear moviment i interaccions.

Tipus de Forces

  • AddForce: Aplica una força al Rigidbody.
  • AddTorque: Aplica una força rotacional.
  • AddExplosionForce: Aplica una força com si fos d'una explosió.

Exemple: Aplicar una Força a un Rigidbody

using UnityEngine;

public class ApplyForce : MonoBehaviour
{
    public float forceAmount = 10.0f;

    void Start()
    {
        Rigidbody rb = gameObject.GetComponent<Rigidbody>();
        if (rb != null)
        {
            // Aplica una força al Rigidbody
            rb.AddForce(Vector3.up * forceAmount, ForceMode.Impulse);
        }
    }
}
