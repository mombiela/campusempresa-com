Introducció

A Unity, els blocs de construcció fonamentals de qualsevol joc són els Objectes de Joc i els Components. Comprendre com aquests elements treballen junts és crucial per crear entorns de joc interactius i dinàmics. Aquesta secció cobrirà els conceptes bàsics dels Objectes de Joc i els Components, els seus rols i com utilitzar-los eficaçment a Unity.

Objectes de Joc

Els Objectes de Joc són les entitats a Unity que representen personatges, accessoris, escenaris, càmeres, punts de referència i més. Essencialment, qualsevol cosa que aparegui a l'escena del teu joc és un Objecte de Joc.

Conceptes Clau

  • Objecte de Joc: La classe base per a totes les entitats en les escenes de Unity.
  • Transform: Cada Objecte de Joc té un component Transform que determina la seva posició, rotació i escala a l'escena.
  • Jerarquia: Els Objectes de Joc poden ser pares d'altres Objectes de Joc, creant una jerarquia que afecta les seves transformacions.

Exemple

// Creant un nou Objecte de Joc a Unity
GameObject myGameObject = new GameObject("MyGameObject");

// Accedint al component Transform
Transform myTransform = myGameObject.transform;
myTransform.position = new Vector3(0, 0, 0);

Components

Els Components són les peces funcionals adjuntes als Objectes de Joc que defineixen el seu comportament i aparença. Cada Objecte de Joc pot tenir múltiples components.

Conceptes Clau

  • Component: La classe base per a tots els components que es poden adjuntar als Objectes de Joc.
  • Components Integrats: Unity proporciona una varietat de components integrats com Rigidbody, Collider i Renderer.
  • Components Personalitzats: Pots crear els teus propis components heretant de la classe MonoBehaviour.

Exemple

// Afegint un component Rigidbody a un Objecte de Joc
Rigidbody rb = myGameObject.AddComponent<Rigidbody>();

// Creant un component personalitzat
public class MyCustomComponent : MonoBehaviour
{
    void Start()
    {
        Debug.Log("Component Personalitzat Adjuntat!");
    }
}

// Afegint el component personalitzat a un Objecte de Joc
myGameObject.AddComponent<MyCustomComponent>();

Treballant amb Objectes de Joc i Components

Comprendre com manipular Objectes de Joc i Components és essencial per al desenvolupament de jocs a Unity. Això implica crear, modificar i gestionar aquests elements a través de scripts i l'Editor de Unity.

Creant Objectes de Joc

Pots crear Objectes de Joc ja sigui a través de l'Editor de Unity o mitjançant scripts.

  • Editor de Unity: Fes clic dret a la finestra de Jerarquia i selecciona Create > Empty o qualsevol altre Objecte de Joc predefinit.
  • Script: Utilitza el constructor new GameObject().

Afegint Components

Els components es poden afegir als Objectes de Joc a través de l'Editor de Unity o mitjançant scripts.

  • Editor de Unity: Selecciona un Objecte de Joc, després fes clic a Add Component a la finestra de l'Inspector.
  • Script: Utilitza el mètode AddComponent<T>().

Exemple

// Creant un Objecte de Joc i afegint un component d'una sola vegada
GameObject myGameObjectWithComponent = new GameObject("MyGameObjectWithComponent", typeof(Rigidbody));

Exercici Pràctic

Per solidificar la teva comprensió, creem un script senzill que crea un Objecte de Joc, afegeix un component Rigidbody i estableix la seva posició.

Passos de l'Exercici

  1. Obre Unity i crea un nou script anomenat GameObjectCreator.
  2. Substitueix el contingut amb el següent codi:
using UnityEngine;

public class GameObjectCreator : MonoBehaviour
{
    void Start()
    {
        // Crear un nou Objecte de Joc
        GameObject myGameObject = new GameObject("MyDynamicGameObject");

        // Afegir un component Rigidbody
        Rigidbody rb = myGameObject.AddComponent<Rigidbody>();

        // Establir la posició de l'Objecte de Joc
        myGameObject.transform.position = new Vector3(0, 5, 0);

        Debug.Log("Objecte de Joc Creat amb Rigidbody a la posició (0, 5, 0)");
    }
}
  1. Adjunta l'script GameObjectCreator a un Objecte de Joc buit a la teva escena.
  2. Prem Play i observa com es crea el nou Objecte de Joc a l'escena.

Conclusió

Comprendre els Objectes de Joc i els Components és fonamental per treballar amb Unity. Els Objectes de Joc serveixen com a contenidors per a totes les entitats del teu joc, mentre que els Components defineixen el seu comportament i aparença. Dominant aquests conceptes, pots crear entorns de joc complexos i interactius. Continua practicant experimentant amb diferents components i creant comportaments personalitzats per millorar les teves habilitats de desenvolupament a Unity.
