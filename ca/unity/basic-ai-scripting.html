Introducció

En aquesta secció, cobrirem els conceptes bàsics del scripting d'IA a Unity. Aprendràs a crear comportaments simples d'IA utilitzant les eines integrades de Unity i el scripting en C#. Al final d'aquesta secció, seràs capaç de crear IA bàsica per a personatges en els teus projectes de Unity.

Configuració del teu projecte

Abans d'endinsar-nos en el scripting d'IA, configurem un nou projecte de Unity.

  1. Obre Unity Hub i crea un nou projecte 3D.
  2. Anomena el teu projecte "BasicAIScripting" i tria una ubicació per desar-lo.
  3. Fes clic a "Create" per configurar el teu nou projecte.

Conceptes bàsics d'IA

Entendre alguns conceptes fonamentals d'IA és crucial abans de començar a escriure scripts.

  • Agent d'IA: Una entitat en el joc que exhibeix un comportament intel·ligent.
  • Màquina d'estats: Un sistema que gestiona els estats i les transicions d'un agent d'IA.
  • Mesh de navegació (NavMesh): Un mesh que defineix les àrees caminables per als agents d'IA.

Creació d'un agent d'IA simple

Creem un agent d'IA simple que pugui moure's cap a un objectiu.

Pas 1: Configuració de l'escena

  1. Crea un pla que serveixi com a terra.
  2. Afegeix un objecte 3D (per exemple, un cub) per actuar com a agent d'IA.
  3. Afegeix un altre objecte 3D (per exemple, una esfera) per actuar com a objectiu.

Pas 2: Afegir components de NavMesh

  1. Selecciona el pla i ves a Window > AI > Navigation.
  2. A la finestra de Navigation, fes clic a la pestanya Bake i després fes clic a Bake per generar el NavMesh.

Pas 3: Escriure el script d'IA

Crea un nou script en C# anomenat SimpleAI i adjunta'l a l'agent d'IA (cub).

using UnityEngine;
using UnityEngine.AI;

public class SimpleAI : MonoBehaviour
{
    public Transform target; // L'objectiu cap al qual es mourà l'IA
    private NavMeshAgent agent;

    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
    }

    void Update()
    {
        if (target != null)
        {
            agent.SetDestination(target.position);
        }
    }
}

Pas 4: Configuració de l'agent d'IA

  1. Selecciona l'agent d'IA (cub) i afegeix un component NavMeshAgent.
  2. Assigna l'objectiu (esfera) al camp target en el script SimpleAI.

Afegir una màquina d'estats bàsica

Per fer la nostra IA més sofisticada, podem implementar una màquina d'estats bàsica.

Pas 1: Definició dels estats

Crea un enum per definir diferents estats.

public enum AIState
{
    Idle,
    Chasing,
    Attacking
}

Pas 2: Implementació de la màquina d'estats

Actualitza el script SimpleAI per incloure la gestió d'estats.

public class SimpleAI : MonoBehaviour
{
    public Transform target;
    private NavMeshAgent agent;
    private AIState currentState;

    void Start()
    {
        agent = GetComponent<NavMeshAgent>();
        currentState = AIState.Idle;
    }

    void Update()
    {
        switch (currentState)
        {
            case AIState.Idle:
                // Comportament en estat Idle
                if (Vector3.Distance(transform.position, target.position) < 10f)
                {
                    currentState = AIState.Chasing;
                }
                break;

            case AIState.Chasing:
                // Comportament en estat Chasing
                agent.SetDestination(target.position);
                if (Vector3.Distance(transform.position, target.position) < 2f)
                {
                    currentState = AIState.Attacking;
                }
                break;

            case AIState.Attacking:
                // Comportament en estat Attacking
                Debug.Log("Atacant l'objectiu!");
                break;
        }
    }
}

Comportaments avançats d'IA

Un cop et sentis còmode amb el scripting bàsic d'IA, pots explorar comportaments més avançats com:

  • Pathfinding: Utilitzar el NavMesh de Unity per crear camins de navegació complexos.
  • Arbres de decisió: Implementar processos de presa de decisions per als agents d'IA.
  • Arbres de comportament: Crear comportaments d'IA més modulars i reutilitzables.
