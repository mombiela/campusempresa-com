Introducció

En aquesta secció, introduirem el concepte de jocs multijugador i com Unity es pot utilitzar per crear-los. Cobrirem els conceptes bàsics de xarxes, les eines que proporciona Unity i l'estructura general d'un joc multijugador.

  • Jocs Multijugador: Jocs que permeten a múltiples jugadors interactuar i jugar junts a través d'una xarxa.
  • Xarxes de Unity: Unity proporciona diverses eines i marcs per facilitar el desenvolupament de jocs multijugador, com ara la xarxa integrada de Unity (UNet) i solucions de tercers com Photon.

Configurant el Projecte

Abans d'entrar en el codi, necessitem configurar el nostre projecte de Unity per al desenvolupament multijugador.

  1. Crear un Nou Projecte de Unity: Obre Unity Hub, fes clic a "New Project" i selecciona la plantilla 3D.
  2. Importar el Paquet de Xarxes: Ves a l'Asset Store i importa un paquet de xarxes com Photon PUN 2.
  3. Estructura del Projecte: Organitza el teu projecte amb carpetes per Scripts, Prefabs, Scenes, etc.

Conceptes Bàsics de Xarxes

Entendre els conceptes bàsics de xarxes és crucial per al desenvolupament de jocs multijugador.

  • Model Client-Servidor: Un jugador (el servidor) allotja el joc, i altres jugadors (clients) s'hi connecten.
  • Model Peer-to-Peer: Tots els jugadors estan connectats entre ells sense un servidor central.
  • Sincronització: Assegurar-se que tots els jugadors veuen el mateix estat del joc.

Implementant un Joc Multijugador Simple

Crearem un joc multijugador bàsic on els jugadors poden moure's en un entorn compartit.

Configurant Photon

  1. Crear un Compte de Photon: Registra't al lloc web de Photon i crea una nova aplicació.
  2. Importar Photon PUN 2: Importa el paquet Photon PUN 2 al teu projecte de Unity.
  3. Configurar Photon: Configura el teu App ID de Photon a Unity navegant a Window > Photon Unity Networking > PUN Wizard.

Creant el Prefab del Jugador

  1. Crear un GameObject del Jugador: Crea un simple GameObject de jugador (per exemple, un cub) i afegeix-hi un component PhotonView.
  2. Script del Jugador: Crea un script per gestionar el moviment del jugador.
using Photon.Pun;
using UnityEngine;

public class PlayerController : MonoBehaviourPunCallbacks
{
    public float speed = 5f;

    void Update()
    {
        if (photonView.IsMine)
        {
            float moveHorizontal = Input.GetAxis("Horizontal");
            float moveVertical = Input.GetAxis("Vertical");

            Vector3 movement = new Vector3(moveHorizontal, 0.0f, moveVertical);
            transform.Translate(movement * speed * Time.deltaTime);
        }
    }
}

Generant Jugadors

  1. Script del NetworkManager: Crea un script per gestionar la generació de jugadors.
using Photon.Pun;
using UnityEngine;

public class NetworkManager : MonoBehaviourPunCallbacks
{
    public GameObject playerPrefab;

    void Start()
    {
        PhotonNetwork.ConnectUsingSettings();
    }

    public override void OnConnectedToMaster()
    {
        PhotonNetwork.JoinLobby();
    }

    public override void OnJoinedLobby()
    {
        PhotonNetwork.JoinRandomRoom();
    }

    public override void OnJoinRandomFailed(short returnCode, string message)
    {
        PhotonNetwork.CreateRoom(null, new Photon.Realtime.RoomOptions { MaxPlayers = 4 });
    }

    public override void OnJoinedRoom()
    {
        PhotonNetwork.Instantiate(playerPrefab.name, Vector3.zero, Quaternion.identity);
    }
}

Provant el Joc

  1. Compilar i Executar: Compila i executa el joc per provar la funcionalitat multijugador.
  2. Múltiples Instàncies: Obre múltiples instàncies del joc per veure els jugadors movent-se en l'entorn compartit.

Funcions Avançades

Un cop la funcionalitat multijugador bàsica estigui funcionant, podem afegir funcions més avançades.

  • Sistema de Xat: Implementa un simple sistema de xat perquè els jugadors puguin comunicar-se.
  • Sincronització d'Objectes Complexos: Sincronitza objectes més complexos com animacions i física.
  • Compensació de Retard: Implementa tècniques per gestionar la latència de la xarxa.

Conclusió

En aquest tema, hem cobert els conceptes bàsics de crear un joc multijugador simple utilitzant Unity i Photon. Hem configurat el projecte, implementat el moviment del jugador i gestionat la generació de jugadors. Amb aquests fonaments, pots començar a construir jocs multijugador més complexos i explorar funcions avançades de xarxes.

Seguint aquest enfocament estructurat, ara hauríeu de tenir una base sòlida per desenvolupar jocs multijugador a Unity. Feliç codificació!
