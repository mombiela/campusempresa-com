Visió General

En aquesta secció, cobrirem els conceptes bàsics de treballar amb àudio a Unity. Aprendràs com importar fitxers d'àudio, utilitzar fonts i oients d'àudio, i controlar la reproducció d'àudio mitjançant scripts. Al final d'aquesta secció, tindràs una base sòlida per integrar àudio als teus projectes de Unity.

Conceptes Clau

  • Fonts d'Àudio: Components que reprodueixen clips d'àudio a l'escena.
  • Oients d'Àudio: Components que capturen àudio de les fonts d'àudio i el reprodueixen a l'usuari.
  • Clips d'Àudio: Els fitxers d'àudio reals que són reproduïts per les fonts d'àudio.
  • Configuracions de So 3D: Configuracions que controlen com es espacialitza l'àudio en un entorn 3D.
  • Script d'Àudio: Utilitzar scripts en C# per controlar la reproducció i les propietats de l'àudio.

Importar Fitxers d'Àudio

Per utilitzar àudio a Unity, primer has d'importar els teus fitxers d'àudio al projecte.

  1. Importar Àudio:
    • Arrossega i deixa anar els teus fitxers d'àudio (per exemple, .mp3, .wav) a la carpeta Assets a l'Editor de Unity.
    • Unity importarà automàticament aquests fitxers i els farà disponibles per al seu ús en el teu projecte.

Utilitzar Fonts i Oients d'Àudio

Fonts d'Àudio

Un component Font d'Àudio s'utilitza per reproduir clips d'àudio a l'escena.

  1. Afegir una Font d'Àudio:

    • Selecciona el GameObject al qual vols afegir l'àudio.
    • Fes clic a Add Component a la finestra de l'Inspector.
    • Cerca Audio Source i afegeix-lo.

  2. Assignar un Clip d'Àudio:

    • Al component Font d'Àudio, veuràs un camp etiquetat Audio Clip.
    • Arrossega el teu fitxer d'àudio importat a aquest camp.

  3. Configuracions Bàsiques:

    • Play On Awake: Si està marcat, l'àudio es reproduirà automàticament quan comenci l'escena.
    • Loop: Si està marcat, l'àudio es reproduirà en bucle contínuament.

Oients d'Àudio

Un component Oient d'Àudio normalment està adjunt a la Càmera Principal i és responsable de capturar àudio de totes les Fonts d'Àudio a l'escena.

  1. Configuració Per Defecte:
    • Per defecte, la Càmera Principal a la teva escena tindrà un component Oient d'Àudio adjunt.
    • Assegura't que només un Oient d'Àudio estigui actiu a l'escena per evitar conflictes.

Controlar la Reproducció d'Àudio amb Scripts

Pots controlar la reproducció i les propietats de l'àudio utilitzant scripts en C#.

Exemple: Script Bàsic de Control d'Àudio

using UnityEngine;

public class AudioController : MonoBehaviour
{
    public AudioSource audioSource;

    void Start()
    {
        // Reprodueix l'àudio quan comença el joc
        audioSource.Play();
    }

    void Update()
    {
        // Alterna la reproducció d'àudio amb la tecla espaiadora
        if (Input.GetKeyDown(KeyCode.Space))
        {
            if (audioSource.isPlaying)
            {
                audioSource.Pause();
            }
            else
            {
                audioSource.Play();
            }
        }
    }
}
  • Explicació:
    • audioSource.Play(): Comença a reproduir el clip d'àudio assignat.
    • audioSource.Pause(): Pausa la reproducció de l'àudio.
    • audioSource.isPlaying: Comprova si l'àudio s'està reproduint actualment.

Configuracions de So 3D

Unity et permet crear experiències d'àudio immersives ajustant les configuracions espacials de les Fonts d'Àudio.

  1. Barreja Espacial:

    • Aquesta configuració controla la barreja entre so 2D i 3D.
    • Un valor de 0 significa que el so és 2D (no afectat per la posició de l'oient).
    • Un valor d'1 significa que el so és completament 3D (afectat per la posició de l'oient).

  2. Desacceleració del Volum:

    • Aquesta configuració controla com disminueix el volum a mesura que l'oient s'allunya de la font d'àudio.
    • Pots triar entre modes de desacceleració Lineal, Logarítmica i Personalitzada.

Resum

En aquesta secció, has après els conceptes bàsics de treballar amb àudio a Unity. Ara saps com importar fitxers d'àudio, utilitzar components Font d'Àudio i Oient d'Àudio, controlar la reproducció d'àudio mitjançant scripts, i ajustar les configuracions de so 3D. Aquestes habilitats et permetran afegir experiències d'àudio riques als teus projectes de Unity.
