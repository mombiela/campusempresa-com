El Projecte Sobre nosaltres Contribuir Donacions Llicència

Benvingut al teu viatge a Unity! En aquesta secció, et guiarem a través del procés de crear el teu primer projecte a Unity. Aquest és un pas essencial que establirà la base per al teu futur treball de desenvolupament.

Configurant Unity

Abans de submergir-nos en la creació d'un projecte, assegura't que tens Unity instal·lat al teu ordinador. Pots descarregar-lo des del lloc web de Unity.

Unity Hub

Unity Hub és una eina de gestió que t'ajuda a gestionar múltiples instal·lacions i projectes de Unity. Assegura't de tenir Unity Hub instal·lat també.

  1. Descarrega i instal·la Unity Hub: Segueix les instruccions al lloc web de Unity per descarregar i instal·lar Unity Hub.
  2. Instal·la Unity Editor: Obre Unity Hub, ves a la pestanya "Installs" i fes clic a "Add" per instal·lar l'última versió de Unity Editor.

Creant un nou projecte

Un cop Unity Hub i Unity Editor estiguin instal·lats, pots crear el teu primer projecte.

  1. Obre Unity Hub: Llença Unity Hub des de les teves aplicacions.
  2. Crea un nou projecte:
    • Fes clic al botó "New".
    • Tria una plantilla. Per als principiants, recomanem començar amb la plantilla "3D".
    • Posa un nom al teu projecte (per exemple, "MyFirstUnityProject").
    • Tria una ubicació per desar el teu projecte.
    • Fes clic a "Create".

Entenent la interfície de Unity

Quan el teu projecte s'obri, veuràs la interfície de Unity Editor. Aquí tens els components clau:

  • Vista de l'escena: Aquí és on construeixes i organitzes els objectes del teu joc.
  • Vista del joc: Això mostra com es veurà el teu joc quan s'executi.
  • Jerarquia: Això llista tots els objectes del joc a la teva escena.
  • Inspector: Això mostra propietats i configuracions per a l'objecte del joc seleccionat.
  • Projecte: Aquí és on es guarden tots els teus recursos (models, scripts, textures, etc.).
  • Consola: Això mostra missatges, advertències i errors.

Afegint un objecte de joc

Anem a afegir un objecte de joc simple a la teva escena.

  1. Crea un cub:

    • Fes clic amb el botó dret a la finestra de la Jerarquia.
    • Selecciona 3D Object > Cube.
    • Un cub apareixerà a la teva vista de l'escena i serà llistat a la Jerarquia.

  2. Posiciona el cub:

    • Selecciona el cub a la Jerarquia.
    • A l'Inspector, estableix la Posició a (0, 0, 0).

Escriure el teu primer script

Ara, afegim una mica d'interactivitat amb un script simple.

  1. Crea un script:

    • A la finestra del Projecte, fes clic amb el botó dret i selecciona Create > C# Script.
    • Posa el nom MoveCube a l'script.

  2. Edita l'script:

    • Fes doble clic a l'script MoveCube per obrir-lo al teu editor de codi (per exemple, Visual Studio).
    • Substitueix el codi per defecte pel següent:
using UnityEngine;

public class MoveCube : MonoBehaviour
{
    public float speed = 5f;

    void Update()
    {
        float move = Input.GetAxis("Horizontal") * speed * Time.deltaTime;
        transform.Translate(move, 0, 0);
    }
}
  1. Adjunta l'script al cub:
    • Arrossega l'script MoveCube des de la finestra del Projecte fins al cub a la Jerarquia.

Executant el teu projecte

  1. Desa la teva escena:

    • Ves a File > Save Scene.
    • Posa un nom a la teva escena (per exemple, "MainScene").

  2. Juga al joc:

    • Fes clic al botó "Play" a la part superior de Unity Editor.
    • Utilitza les tecles de fletxa esquerra i dreta per moure el cub.

Resum

Felicitats! Has creat el teu primer projecte a Unity, has afegit un objecte de joc i has escrit un script per fer-lo interactiu. Aquí tens un breu resum del que hem cobert:

  • Configurant Unity i Unity Hub.
  • Creant un nou projecte.
  • Entenent la interfície de Unity.
  • Afegint i posicionant un objecte de joc.
  • Escriure i adjuntar un script simple.
  • Executant el teu projecte.

Aquest coneixement fonamental serà crucial mentre continues explorant i desenvolupant projectes més complexos a Unity. Feliç codificació!
