El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, cobrim els passos essencials per construir i exportar el teu joc utilitzant Unity. Aquest procés implica preparar el teu joc per a diferents plataformes, configurar els paràmetres de construcció, i finalment exportar el joc per a la seva distribució.

Preparar el Teu Joc per a la Construcció

Abans de començar el procés de construcció, assegura't que el teu joc està llest per a l'exportació. Això implica comprovar si hi ha errors, optimitzar els recursos, i configurar les configuracions necessàries.

  • Comprovar Errors: Utilitza la Consola de Unity per identificar i corregir qualsevol error o advertència.
  • Optimitzar Recursos: Assegura't que els teus recursos (textures, models, àudio) estan optimitzats per al rendiment.
  • Configurar Paràmetres del Jugador: Configura la resolució del teu joc, els paràmetres de qualitat, i altres preferències del jugador.

Exemple: Configurar Paràmetres del Jugador

  1. Ves a Edit > Project Settings > Player.
  2. Configura els paràmetres de Resolution and Presentation:
    • Defineix l'amplada i l'alçada de pantalla per defecte.
    • Tria si el joc ha de funcionar en mode de pantalla completa.
  3. Ajusta els paràmetres de Quality:
    • Defineix diferents nivells de qualitat (per exemple, Baixa, Mitjana, Alta).
    • Assigna el nivell de qualitat per defecte per al teu joc.
// Exemple de script per establir la resolució de pantalla per defecte
void Start() {
    Screen.SetResolution(1920, 1080, true);
}

Configurar Paràmetres de Construcció

Unity et permet construir el teu joc per a múltiples plataformes. Configurar correctament els paràmetres de construcció és crucial per assegurar que el teu joc funcioni sense problemes a la plataforma objectiu.

  • Seleccionar la Plataforma: Tria la plataforma per a la qual vols construir (per exemple, Windows, Mac, iOS, Android).
  • Configurar les Opcions de Construcció: Configura opcions com la construcció de desenvolupament, el mètode de compressió, i el backend de scripting.

Exemple: Configurar Paràmetres de Construcció per a Windows

  1. Ves a File > Build Settings.
  2. Selecciona PC, Mac & Linux Standalone.
  3. Tria Windows com a plataforma objectiu.
  4. Fes clic a Switch Platform.
  5. Configura les opcions de construcció:
    • Marca Development Build si vols incloure informació de depuració.
    • Tria el Compression Method apropiat (per exemple, LZ4, LZ4HC).
// Exemple de script per comprovar la plataforma i ajustar els paràmetres
void Start() {
    #if UNITY_STANDALONE_WIN
    Debug.Log("Executant a Windows");
    #endif
}

Exportar el Teu Joc

Un cop el teu joc està configurat i llest, pots procedir a exportar-lo. Això implica crear els fitxers de construcció finals que es poden distribuir als jugadors.

  • Construir el Joc: Utilitza la finestra de Build Settings per començar el procés de construcció.
  • Provar la Construcció: Executa el joc construït per assegurar-te que tot funciona com s'espera.
  • Distribuir el Joc: Empaqueta els fitxers de construcció i distribueix-los a través de la plataforma que hagis triat (per exemple, Steam, App Store).

Exemple: Construir i Provar el Teu Joc

  1. Ves a File > Build Settings.
  2. Fes clic a Build per començar el procés de construcció.
  3. Tria una ubicació per desar els fitxers de construcció.
  4. Un cop la construcció estigui completa, navega a la ubicació de la construcció i executa el fitxer executable per provar el joc.

Conclusió

Construir i exportar el teu joc és un pas crucial en el procés de desenvolupament de jocs. Seguint els passos descrits en aquesta secció, pots assegurar-te que el teu joc està optimitzat, configurat correctament, i llest per a la distribució. Recorda provar les teves construccions a fons per detectar qualsevol problema abans de llançar el teu joc al públic. Feliç desenvolupament!
