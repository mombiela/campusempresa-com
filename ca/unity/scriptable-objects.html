Introducció als Objectes Scriptables

Els Objectes Scriptables són una característica poderosa a Unity que et permet crear contenidors de dades que es poden utilitzar per emmagatzemar grans quantitats de dades compartides independentment de les instàncies de classe. Són particularment útils per gestionar dades del joc, configuracions i ajustaments.

Conceptes Clau

  • Contenidors de Dades: Els Objectes Scriptables s'utilitzen per emmagatzemar dades que es poden compartir entre múltiples instàncies.
  • Creació d'Actius: Els Objectes Scriptables es creen com a actius a l'Editor de Unity.
  • Serialització: Es serialitzen i es guarden com a actius, fent-los persistents entre sessions de joc.

Creació d'Objectes Scriptables

Guia Pas a Pas

  1. Crear una Classe d'Objecte Scriptable:
    • Defineix una classe que hereti de ScriptableObject.
    • Utilitza l'atribut CreateAssetMenu per facilitar la creació d'instàncies des de l'Editor de Unity.
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameData", menuName = "Game Data", order = 51)]
public class GameData : ScriptableObject
{
    public string gameName;
    public int maxPlayers;
    public float gameDuration;
}
  1. Crear una Instància a l'Editor:
    • Fes clic dret a la finestra del Projecte.
    • Selecciona Create > Game Data (o el nom que hagis donat al teu Objecte Scriptable).
    • Omple els camps de dades a l'Inspector.

Exemple

// GameData.cs
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameData", menuName = "Game Data", order = 51)]
public class GameData : ScriptableObject
{
    public string gameName = "My Game";
    public int maxPlayers = 4;
    public float gameDuration = 60.0f;
}

Utilitzar Objectes Scriptables en el Teu Joc

Accedir als Objectes Scriptables

  • Referència en Scripts: Pots referenciar Objectes Scriptables en els teus scripts de MonoBehaviour per accedir a les seves dades.
using UnityEngine;

public class GameManager : MonoBehaviour
{
    public GameData gameData;

    void Start()
    {
        Debug.Log("Game Name: " + gameData.gameName);
        Debug.Log("Max Players: " + gameData.maxPlayers);
        Debug.Log("Game Duration: " + gameData.gameDuration);
    }
}

Beneficis

  • Gestió Centralitzada de Dades: Gestiona i actualitza fàcilment les dades del joc des d'una única ubicació.
  • Reducció de l'Ús de Memòria: Els Objectes Scriptables es comparteixen entre instàncies, reduint la càrrega de memòria.
  • Integració amb l'Inspector: Modifica les dades directament a l'Editor de Unity sense recompilar scripts.

Ús Avançat

Crear Editors Personalitzats

  • Inspector Personalitzat: Millora l'experiència de l'usuari creant inspectors personalitzats per als teus Objectes Scriptables.
using UnityEditor;
using UnityEngine;

[CustomEditor(typeof(GameData))]
public class GameDataEditor : Editor
{
    public override void OnInspectorGUI()
    {
        GameData gameData = (GameData)target;

        gameData.gameName = EditorGUILayout.TextField("Game Name", gameData.gameName);
        gameData.maxPlayers = EditorGUILayout.IntField("Max Players", gameData.maxPlayers);
        gameData.gameDuration = EditorGUILayout.FloatField("Game Duration", gameData.gameDuration);

        if (GUILayout.Button("Reset Data"))
        {
            gameData.gameName = "My Game";
            gameData.maxPlayers = 4;
            gameData.gameDuration = 60.0f;
        }

        EditorUtility.SetDirty(gameData);
    }
}

Utilitzar Objectes Scriptables per a Sistemes d'Esdeveniments

  • Gestió d'Esdeveniments: Utilitza Objectes Scriptables per gestionar esdeveniments del joc i desacoblar la gestió d'esdeveniments de la lògica del joc.
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameEvent", menuName = "Game Event", order = 52)]
public class GameEvent : ScriptableObject
{
    private readonly List<GameEventListener> eventListeners = new List<GameEventListener>();

    public void Raise()
    {
        for (int i = eventListeners.Count - 1; i >= 0; i--)
        {
            eventListeners[i].OnEventRaised();
        }
    }

    public void RegisterListener(GameEventListener listener)
    {
        if (!eventListeners.Contains(listener))
            eventListeners.Add(listener);
    }

    public void UnregisterListener(GameEventListener listener)
    {
        if (eventListeners.Contains(listener))
            eventListeners.Remove(listener);
    }
}

Conclusió

Els Objectes Scriptables són una manera versàtil i eficient de gestionar dades a Unity. Ajuden a organitzar i centralitzar dades, reduir l'ús de memòria i millorar l'arquitectura general del teu joc. Dominant els Objectes Scriptables, pots crear projectes de Unity més mantenibles i escalables.
