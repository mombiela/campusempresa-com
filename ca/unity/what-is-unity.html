Introducció

Unity és una plataforma de desenvolupament de jocs potent i àmpliament utilitzada que permet als desenvolupadors crear experiències en 2D, 3D, VR i AR. Proporciona un conjunt complet d'eines i funcions que agiliten el procés de desenvolupament, fent-lo accessible tant per a principiants com per a usuaris avançats.

Conceptes Clau

  • Motor de Joc: Unity és un motor de joc, el que significa que proporciona la infraestructura necessària per crear i executar jocs. Això inclou renderitzar gràfics, gestionar la física i administrar la lògica del joc.
  • Multiplataforma: Unity admet el desenvolupament per a múltiples plataformes, incloent Windows, macOS, Linux, iOS, Android i diverses consoles de jocs.
  • Arquitectura Basada en Components: Unity utilitza una arquitectura basada en components on els objectes de joc es componen de diversos components que defineixen el seu comportament i aparença.
  • Scripting: Unity utilitza principalment C# per a scripting, permetent als desenvolupadors escriure lògica de joc personalitzada i controlar el comportament dels objectes de joc.

Característiques de Unity

  • Editor: Una interfície fàcil d'usar on els desenvolupadors poden dissenyar i organitzar les seves escenes de joc.
  • Asset Store: Un mercat on els desenvolupadors poden comprar i vendre recursos, com models 3D, textures i scripts.
  • Motor de Física: Motor de física integrat per simular la física del món real.
  • Sistema d'Animació: Eines per crear i gestionar animacions.
  • Sistema d'UI: Components per construir interfícies d'usuari.
  • Networking: Suport per a jocs multijugador i experiències en xarxa.

Començar amb Unity

Per començar amb Unity, segueix aquests passos:

  1. Descarrega i Instal·la Unity: Visita el lloc web de Unity i descarrega el Unity Hub. Utilitza el Hub per instal·lar la versió més recent de Unity.
  2. Crea un Nou Projecte: Obre Unity Hub, fes clic a "New Project", selecciona una plantilla (per exemple, 2D, 3D) i dona un nom al teu projecte.
  3. Explora l'Editor de Unity: Familiaritza't amb la interfície de l'Editor de Unity, incloent la vista d'Escena, la vista de Joc, la Jerarquia, l'Inspector i la finestra de Projecte.

Exemple Bàsic: Crear una Escena Senzilla

Creem una escena senzilla amb un cub en moviment per entendre els conceptes bàsics de Unity.

Pas 1: Crear un Cub

  1. Obre el teu projecte de Unity.
  2. A la finestra de Jerarquia, fes clic dret i selecciona 3D Object > Cube.
  3. Un cub apareixerà a la vista d'Escena i es mostrarà a la Jerarquia.

Pas 2: Afegir un Script

  1. A la finestra de Projecte, fes clic dret i selecciona Create > C# Script. Anomena'l MoveCube.
  2. Fes doble clic al script per obrir-lo al teu editor de codi (per exemple, Visual Studio).

Pas 3: Escriure el Script

Substitueix el codi per defecte pel següent:

using UnityEngine;

public class MoveCube : MonoBehaviour
{
    public float speed = 5f;

    void Update()
    {
        float move = speed * Time.deltaTime;
        transform.Translate(move, 0, 0);
    }
}

Pas 4: Adjuntar el Script al Cub

  1. Arrossega el script MoveCube des de la finestra de Projecte fins al Cub a la Jerarquia.
  2. Selecciona el Cub i mira la finestra de l'Inspector per veure el component MoveCube adjunt.

Pas 5: Executar l'Escena

  1. Fes clic al botó de Reproducció a la part superior de l'Editor de Unity.
  2. Observa el cub movent-se al llarg de l'eix x.

Conclusió

Unity és una plataforma de desenvolupament de jocs versàtil i potent adequada per crear una àmplia gamma d'experiències interactives. Entenent els seus conceptes clau i característiques, i practicant amb exemples senzills, pots començar el teu viatge en el desenvolupament de jocs. A mesura que avancis en aquest curs, aprofundiràs en les capacitats de Unity, aprenent tècniques avançades i millors pràctiques per crear jocs i aplicacions de qualitat professional.
