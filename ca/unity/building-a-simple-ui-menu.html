Construir un Menú d'Interfície d'Usuari Simple
Introducció a l'Àudio a Unity ►

Crear un menú d'interfície d'usuari (UI) és una habilitat fonamental a Unity, essencial per desenvolupar aplicacions interactives i jocs. Aquest tema et guiarà a través del procés de construir un menú d'UI simple, des de conceptes bàsics fins a característiques més avançades.

Introducció a la UI de Unity

Abans d'endinsar-nos en la construcció d'un menú d'UI, és important entendre els conceptes bàsics del sistema d'UI de Unity.

  • Canvas: L'àrea on es renderitzen tots els elements d'UI.
  • Elements d'UI: Components com botons, text, imatges i panells que formen la UI.
  • Sistema d'Esdeveniments: Gestiona els esdeveniments d'entrada per a la UI.

Configuració del Canvas

  1. Crear un Canvas:

    • Fes clic dret a la finestra Hierarchy.
    • Selecciona UI > Canvas.

  2. Ajustar la Configuració del Canvas:

    • Assegura't que el Canvas estigui configurat a Screen Space - Overlay per a una UI 2D simple.
// Exemple: Crear un Canvas en C#
GameObject canvas = new GameObject("Canvas");
canvas.AddComponent<Canvas>().renderMode = RenderMode.ScreenSpaceOverlay;
canvas.AddComponent<CanvasScaler>();
canvas.AddComponent<GraphicRaycaster>();

Afegir Elements d'UI

Crear un Menú Bàsic

  1. Afegir un Panell:

    • Fes clic dret sobre el Canvas.
    • Selecciona UI > Panel.

  2. Afegir Botons:

    • Fes clic dret sobre el Panell.
    • Selecciona UI > Button.

  3. Personalitzar Botons:

    • Canvia el text, la mida i la posició dels botons a l'Inspector.
// Exemple: Afegir un Botó en C#
GameObject button = new GameObject("Button");
button.AddComponent<RectTransform>();
button.AddComponent<CanvasRenderer>();
button.AddComponent<Image>();
button.AddComponent<Button>();

// Establir el pare del botó al Canvas
button.transform.SetParent(canvas.transform, false);

// Afegir Text al Botó
GameObject buttonText = new GameObject("Text");
buttonText.transform.SetParent(button.transform, false);
Text text = buttonText.AddComponent<Text>();
text.text = "Click Me!";
text.font = Resources.GetBuiltinResource<Font>("Arial.ttf");
text.alignment = TextAnchor.MiddleCenter;

Gestionar Clics de Botons

Afegir Funcionalitat als Botons

  1. Crear un Script:

    • Fes clic dret a la finestra Project.
    • Selecciona Create > C# Script.
    • Anomena'l MenuController.

  2. Adjuntar el Script a un GameObject:

    • Arrossega el script al Canvas o a un GameObject buit.

  3. Escriure Mètodes de Clic de Botons:

using UnityEngine;
using UnityEngine.UI;

public class MenuController : MonoBehaviour
{
    public void OnStartButtonClicked()
    {
        Debug.Log("Start Button Clicked");
        // Carregar l'escena del joc o iniciar el joc
    }

    public void OnExitButtonClicked()
    {
        Debug.Log("Exit Button Clicked");
        // Sortir de l'aplicació
        Application.Quit();
    }
}
  1. Enllaçar Botons a Mètodes:
    • Selecciona el Botó a la Hierarchy.
    • A l'Inspector, troba el component Button.
    • Afegeix un esdeveniment OnClick i enllaça'l al mètode apropiat a MenuController.

Característiques Avançades d'UI

Afegir Animacions

  1. Crear un Controlador d'Animator:

    • Fes clic dret a la finestra Project.
    • Selecciona Create > Animator Controller.

  2. Afegir Clips d'Animació:

    • Obre la finestra Animator.
    • Crea i configura clips d'animació per a les transicions d'UI.

  3. Enllaçar Animacions a Elements d'UI:

    • Afegeix un component Animator a l'element d'UI.
    • Assigna el Controlador d'Animator.

Utilitzar Grups de Disseny

  1. Afegir un Grup de Disseny:

    • Selecciona el Panell.
    • Afegeix un component Vertical Layout Group o Horizontal Layout Group.

  2. Configurar la Configuració del Disseny:

    • Ajusta el padding, l'espaiat i l'alineació a l'Inspector.
// Exemple: Afegir un Grup de Disseny Vertical en C#
VerticalLayoutGroup layoutGroup = panel.AddComponent<VerticalLayoutGroup>();
layoutGroup.padding = new RectOffset(10, 10, 10, 10);
layoutGroup.spacing = 5;
layoutGroup.childAlignment = TextAnchor.MiddleCenter;

Conclusió

Construir un menú d'UI simple a Unity implica entendre els components bàsics d'UI, configurar un Canvas, afegir i personalitzar elements d'UI, i gestionar les interaccions dels usuaris a través de scripts. A mesura que avancis, pots millorar la teva UI amb animacions i grups de disseny per crear interfícies més dinàmiques i responsives. Practica aquests passos per convertir-te en un expert en la creació de menús intuïtius i interactius per als teus projectes a Unity.
