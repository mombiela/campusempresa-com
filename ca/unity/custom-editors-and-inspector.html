Introducció

Els editors i inspectors personalitzats a Unity et permeten crear interfícies personalitzades per als teus scripts i components dins de l'Editor de Unity. Això pot millorar molt el teu flux de treball proporcionant eines més intuïtives i potents per al desenvolupament de jocs.

Per què utilitzar editors i inspectors personalitzats?

  • Millora del flux de treball: Adapta l'Editor de Unity per satisfer millor les teves necessitats específiques.
  • Usabilitat millorada: Crea interfícies més fàcils d'usar per a scripts complexos.
  • Depuració i proves: Més fàcil de visualitzar i manipular dades durant el desenvolupament.

Començant amb editors personalitzats

Creació d'un editor personalitzat bàsic

Per crear un editor personalitzat, necessites:

  1. Crear un nou script a la carpeta Editor.
  2. Estendre la classe Editor.
  3. Utilitzar l'atribut CustomEditor per enllaçar l'editor a un component específic.
// ExampleComponent.cs
using UnityEngine;

public class ExampleComponent : MonoBehaviour
{
    public float exampleFloat;
    public string exampleString;
}

// ExampleComponentEditor.cs
using UnityEditor;
using UnityEngine;

[CustomEditor(typeof(ExampleComponent))]
public class ExampleComponentEditor : Editor
{
    public override void OnInspectorGUI()
    {
        ExampleComponent exampleComponent = (ExampleComponent)target;

        exampleComponent.exampleFloat = EditorGUILayout.FloatField("Example Float", exampleComponent.exampleFloat);
        exampleComponent.exampleString = EditorGUILayout.TextField("Example String", exampleComponent.exampleString);

        if (GUILayout.Button("Press Me"))
        {
            Debug.Log("Button Pressed!");
        }

        // Assegura't que es guardin els canvis
        if (GUI.changed)
        {
            EditorUtility.SetDirty(exampleComponent);
        }
    }
}

Conceptes clau

  • EditorGUILayout: Proporciona un conjunt de controls que es poden utilitzar per crear camps d'inspector personalitzats.
  • GUILayout: Utilitzat per a controls basats en disseny com botons i etiquetes.
  • EditorUtility.SetDirty: Marca l'objectiu com a modificat, assegurant que es guardin els canvis.

Editors personalitzats avançats

Dibuixadors de propietats personalitzats

Els dibuixadors de propietats personalitzats et permeten crear interfícies d'usuari personalitzades per a camps individuals en els teus scripts.

// CustomProperty.cs
using UnityEngine;

public class CustomProperty : MonoBehaviour
{
    public Range range;
}

[System.Serializable]
public class Range
{
    public float min;
    public float max;
}

// RangeDrawer.cs
using UnityEditor;
using UnityEngine;

[CustomPropertyDrawer(typeof(Range))]
public class RangeDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position, SerializedProperty property, GUIContent label)
    {
        EditorGUI.BeginProperty(position, label, property);

        SerializedProperty min = property.FindPropertyRelative("min");
        SerializedProperty max = property.FindPropertyRelative("max");

        float minValue = min.floatValue;
        float maxValue = max.floatValue;

        EditorGUI.MinMaxSlider(position, label, ref minValue, ref maxValue, 0, 100);

        min.floatValue = minValue;
        max.floatValue = maxValue;

        EditorGUI.EndProperty();
    }
}

Finestres d'editor

Crear finestres d'editor personalitzades pot proporcionar eines més complexes i independents dins de Unity.

// CustomEditorWindow.cs
using UnityEditor;
using UnityEngine;

public class CustomEditorWindow : EditorWindow
{
    [MenuItem("Window/Custom Editor Window")]
    public static void ShowWindow()
    {
        GetWindow<CustomEditorWindow>("Custom Editor Window");
    }

    private void OnGUI()
    {
        GUILayout.Label("This is a custom editor window", EditorStyles.boldLabel);

        if (GUILayout.Button("Do Something"))
        {
            Debug.Log("Button in custom window pressed!");
        }
    }
}

Millors pràctiques

  • Mantén-ho simple: Evita complicar massa la interfície; mantén-la intuïtiva.
  • Rendiment: Assegura't que els teus editors personalitzats siguin eficients i no degradin el rendiment.
  • Consistència: Mantén un aspecte i sensació consistents amb la resta de l'Editor de Unity.

Conclusió

Els editors i inspectors personalitzats a Unity són eines poderoses que poden millorar significativament el teu flux de treball de desenvolupament. Creant interfícies personalitzades adaptades a les teves necessitats, pots fer que el teu procés de desenvolupament sigui més eficient i agradable. Comença amb editors personalitzats bàsics i explora gradualment funcions avançades com dibuixadors de propietats personalitzats i finestres d'editor per aprofitar al màxim les capacitats de l'scripting de l'editor de Unity.
