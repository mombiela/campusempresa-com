Introducció

El maneig d'esdeveniments de la interfície d'usuari és un aspecte crucial per crear aplicacions interactives a Unity. Aquest tema et guiarà des dels conceptes bàsics del maneig d'esdeveniments de la interfície d'usuari fins a tècniques més avançades, assegurant que puguis crear interfícies responsives i fàcils d'usar.

Conceptes bàsics dels esdeveniments de la interfície d'usuari

Unity proporciona un sistema robust per manejar esdeveniments de la interfície d'usuari a través del sistema d'esdeveniments i diversos components de la interfície d'usuari.

Conceptes clau

  • Sistema d'esdeveniments: Gestiona i dirigeix els esdeveniments d'entrada als elements de la interfície d'usuari apropiats.
  • Components de la interfície d'usuari: Elements com botons, barres de desplaçament i camps d'entrada que poden desencadenar esdeveniments.

Exemple: Maneig de clics de botons

Per manejar un clic de botó, necessites:

  1. Crear un botó de la interfície d'usuari.
  2. Adjuntar un script per manejar l'esdeveniment de clic.
using UnityEngine;
using UnityEngine.UI;

public class ButtonClickHandler : MonoBehaviour
{
    public Button myButton;

    void Start()
    {
        myButton.onClick.AddListener(OnButtonClick);
    }

    void OnButtonClick()
    {
        Debug.Log("S'ha fet clic al botó!");
    }
}

Explicació

  • myButton: Referència al component Button.
  • Start(): Mètode de Unity que es crida abans de l'actualització del primer fotograma.
  • onClick.AddListener(OnButtonClick): Registra el mètode OnButtonClick per ser cridat quan es faci clic al botó.
  • OnButtonClick(): Mètode que s'executa quan es fa clic al botó.

Maneig intermedi d'esdeveniments de la interfície d'usuari

A mesura que progressis, necessitaràs manejar esdeveniments i interaccions més complexes.

Maneig de múltiples esdeveniments

Pots manejar múltiples tipus d'esdeveniments en un sol element de la interfície d'usuari.

using UnityEngine;
using UnityEngine.EventSystems;

public class MultiEventHandler : MonoBehaviour, IPointerEnterHandler, IPointerExitHandler, IPointerClickHandler
{
    public void OnPointerEnter(PointerEventData eventData)
    {
        Debug.Log("El punter ha entrat a l'element de la interfície d'usuari.");
    }

    public void OnPointerExit(PointerEventData eventData)
    {
        Debug.Log("El punter ha sortit de l'element de la interfície d'usuari.");
    }

    public void OnPointerClick(PointerEventData eventData)
    {
        Debug.Log("S'ha fet clic a l'element de la interfície d'usuari.");
    }
}

Explicació

  • IPointerEnterHandler, IPointerExitHandler, IPointerClickHandler: Interfícies que defineixen mètodes per manejar esdeveniments de punter.
  • OnPointerEnter, OnPointerExit, OnPointerClick: Mètodes que manegen els esdeveniments respectius.

Maneig avançat d'esdeveniments de la interfície d'usuari

El maneig avançat d'esdeveniments implica esdeveniments personalitzats i interaccions més complexes.

Esdeveniments personalitzats

Pots crear i manejar esdeveniments personalitzats per gestionar interaccions específiques a la teva aplicació.

using UnityEngine;
using UnityEngine.Events;

public class CustomEventHandler : MonoBehaviour
{
    [System.Serializable]
    public class MyCustomEvent : UnityEvent<string> { }

    public MyCustomEvent onCustomEvent;

    void Start()
    {
        if (onCustomEvent == null)
            onCustomEvent = new MyCustomEvent();

        onCustomEvent.AddListener(OnCustomEventTriggered);
    }

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            onCustomEvent.Invoke("S'ha premut la tecla d'espai!");
        }
    }

    void OnCustomEventTriggered(string message)
    {
        Debug.Log(message);
    }
}

Explicació

  • MyCustomEvent: Esdeveniment personalitzat de Unity que pren un paràmetre de tipus string.
  • onCustomEvent: Instància de l'esdeveniment personalitzat.
  • onCustomEvent.AddListener(OnCustomEventTriggered): Registra el mètode OnCustomEventTriggered per manejar l'esdeveniment personalitzat.
  • onCustomEvent.Invoke("S'ha premut la tecla d'espai!"): Desencadena l'esdeveniment personalitzat amb un missatge.

Resum

El maneig d'esdeveniments de la interfície d'usuari a Unity va des de clics de botons simples fins a esdeveniments personalitzats complexos. Entenent i utilitzant el sistema d'esdeveniments de Unity i els components de la interfície d'usuari, pots crear aplicacions interactives i responsives. Comença amb el maneig bàsic d'esdeveniments, progressa cap al maneig de múltiples esdeveniments i eventualment crea esdeveniments personalitzats per adaptar-se a les necessitats de la teva aplicació.
