Introducció

En aquesta secció, ens endinsarem en tècniques avançades de física a Unity. Aquestes tècniques t'ajudaran a crear interaccions més realistes i complexes en els teus jocs. Cobrirem temes com la manipulació de Rigidbody, la detecció de col·lisions i materials de física avançada.

Manipulació de Rigidbody

Els components Rigidbody són essencials per afegir comportament físic als objectes a Unity. Permeten que els objectes reaccionin a forces i col·lisions de manera realista.

  • Afegir un Component Rigidbody: Per afegir un component Rigidbody a un GameObject, selecciona l'objecte a la jerarquia, ves a la finestra Inspector i fes clic a "Add Component" -> "Rigidbody".
  • Propietats Clau:
    • Mass: Determina el pes de l'objecte.
    • Drag: Controla el drag lineal, que redueix la velocitat de l'objecte amb el temps.
    • Angular Drag: Controla el drag rotacional.
    • Use Gravity: Si està marcat, l'objecte serà afectat per la gravetat.
    • Is Kinematic: Si està marcat, l'objecte no serà afectat per forces físiques però encara pot interactuar amb altres objectes físics.

Exemple: Aplicar Força a un Rigidbody

using UnityEngine;

public class ApplyForce : MonoBehaviour
{
    public Rigidbody rb;
    public float forceAmount = 500f;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector3.up * forceAmount);
        }
    }
}

Explicació:

  • Obtenim el component Rigidbody adjunt al GameObject.
  • Quan es prem la tecla espai, s'aplica una força cap amunt al Rigidbody.

Detecció de Col·lisions

La detecció de col·lisions és crucial per crear entorns interactius. Unity proporciona diverses maneres de detectar col·lisions i triggers.

  • Col·lisió vs. Trigger:
    • Col·lisió: Interacció física entre objectes (per exemple, rebotar, lliscar).
    • Trigger: Interacció no física utilitzada per detectar quan els objectes entren, es mantenen o surten d'una àrea específica.

Exemple: Detectar Col·lisions

using UnityEngine;

public class CollisionDetection : MonoBehaviour
{
    void OnCollisionEnter(Collision collision)
    {
        Debug.Log("Col·lisió detectada amb " + collision.gameObject.name);
    }

    void OnTriggerEnter(Collider other)
    {
        Debug.Log("Trigger activat per " + other.gameObject.name);
    }
}

Explicació:

  • OnCollisionEnter es crida quan el GameObject col·lideix amb un altre objecte.
  • OnTriggerEnter es crida quan un altre objecte entra al trigger collider adjunt al GameObject.

Materials de Física Avançada

Els materials de física defineixen com interactuen les superfícies entre elles. Es poden utilitzar per crear efectes com la rebotabilitat i la fricció.

  • Crear un Material de Física:
    • Ves a la finestra Project, fes clic dret i selecciona "Create" -> "Physics Material".
    • Ajusta propietats com Bounciness i Friction.

Exemple: Aplicar un Material de Física

using UnityEngine;

public class ApplyPhysicsMaterial : MonoBehaviour
{
    public PhysicMaterial physicMaterial;

    void Start()
    {
        Collider collider = GetComponent<Collider>();
        if (collider != null)
        {
            collider.material = physicMaterial;
        }
    }
}

Explicació:

  • Assignem un Material de Física al component Collider del GameObject, que afectarà les seves interaccions físiques.

Resum

En aquest tema, hem explorat tècniques avançades de física a Unity, incloent la manipulació de Rigidbody, la detecció de col·lisions i l'ús de materials de física. Aquestes tècniques són essencials per crear entorns de joc realistes i interactius. Entenent i aplicant aquests conceptes, pots millorar significativament el realisme físic dels teus projectes a Unity.
