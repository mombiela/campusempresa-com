El Projecte Sobre nosaltres Contribuir Donacions Llicència

Publicar el teu joc de Unity a diferents plataformes pot ser un procés complex, però és essencial per arribar a una audiència més àmplia. Aquesta secció et guiarà a través dels passos necessaris per publicar el teu joc en diverses plataformes, des d'escriptori fins a mòbils i consoles.

Plataformes d'Escriptori

Windows, macOS, i Linux

Publicar a plataformes d'escriptori és un dels processos més senzills a Unity. Aquí tens com fer-ho:

  1. Configuració de la Compilació:

    • Ves a File > Build Settings.
    • Selecciona la plataforma per a la qual vols compilar (Windows, macOS, o Linux).
    • Fes clic a Switch Platform.

  2. Configuració del Jugador:

    • Fes clic a Player Settings a la finestra de Configuració de la Compilació.
    • Configura opcions com el nom de l'empresa, el nom del producte i la versió.
    • Defineix la resolució i les opcions de presentació.

  3. Compilar i Executar:

    • Fes clic a Build and Run per compilar el teu joc.
    • Tria una ubicació per desar la compilació.
    • Unity crearà un fitxer executable per a la plataforma seleccionada.
// Exemple: Script senzill per sortir de l'aplicació
using UnityEngine;

public class QuitGame : MonoBehaviour
{
    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            Application.Quit();
        }
    }
}

Plataformes Mòbils

Android

  1. Instal·lar l'SDK d'Android:

    • Descarrega i instal·la Android Studio.
    • Configura Unity per utilitzar l'SDK i NDK d'Android.

  2. Configuració de la Compilació:

    • Ves a File > Build Settings.
    • Selecciona Android i fes clic a Switch Platform.

  3. Configuració del Jugador:

    • Configura opcions com el nom del paquet, la versió i el nivell mínim de l'API.
    • Configura icones i pantalles de benvinguda.

  4. Compilar i Executar:

    • Connecta el teu dispositiu Android via USB.
    • Fes clic a Build and Run.
// Exemple: Script senzill per gestionar el botó enrere d'Android
using UnityEngine;

public class AndroidBackButton : MonoBehaviour
{
    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            // Gestionar la pressió del botó enrere
            Application.Quit();
        }
    }
}

iOS

  1. Instal·lar Xcode:

    • Descarrega i instal·la Xcode des de la Mac App Store.

  2. Configuració de la Compilació:

    • Ves a File > Build Settings.
    • Selecciona iOS i fes clic a Switch Platform.

  3. Configuració del Jugador:

    • Configura opcions com l'identificador del paquet, la versió i el dispositiu objectiu.
    • Configura icones i pantalles de benvinguda.

  4. Compilar i Executar:

    • Fes clic a Build per generar un projecte d'Xcode.
    • Obre el projecte a Xcode i configura la signatura.
    • Compila i executa el projecte al teu dispositiu iOS.
// Exemple: Script senzill per gestionar el botó d'inici d'iOS
using UnityEngine;

public class iOSHomeButton : MonoBehaviour
{
    void OnApplicationPause(bool pauseStatus)
    {
        if (pauseStatus)
        {
            // Gestionar la pressió del botó d'inici
            Debug.Log("Aplicació en pausa");
        }
    }
}

Plataformes Web

WebGL

  1. Configuració de la Compilació:

    • Ves a File > Build Settings.
    • Selecciona WebGL i fes clic a Switch Platform.

  2. Configuració del Jugador:

    • Configura opcions com la resolució i les plantilles de WebGL.

  3. Compilar i Executar:

    • Fes clic a Build and Run.
    • Unity generarà una compilació WebGL que es pot allotjar en un servidor web.
// Exemple: Script senzill per gestionar l'entrada a WebGL
using UnityEngine;

public class WebGLInput : MonoBehaviour
{
    void Update()
    {
        if (Input.GetMouseButtonDown(0))
        {
            Debug.Log("Botó del ratolí clicat");
        }
    }
}

Plataformes de Consola

PlayStation, Xbox, i Nintendo Switch

Publicar a consoles requereix permisos especials i kits de desenvolupament de les respectives companyies. Aquí tens els passos generals:

  1. Obtenir Kits de Desenvolupament:

    • Sol·licita programes de desenvolupadors de Sony, Microsoft, o Nintendo.
    • Obteniu els kits de desenvolupament i SDKs necessaris.

  2. Configuració de la Compilació:

    • Ves a File > Build Settings.
    • Selecciona la plataforma de consola i fes clic a Switch Platform.

  3. Configuració del Jugador:

    • Configura opcions específiques per a la consola, com la resolució i l'entrada.

  4. Compilar i Executar:

    • Segueix les guies específiques de la plataforma per compilar i desplegar el teu joc.
// Exemple: Script senzill per gestionar l'entrada específica de consola
using UnityEngine;

public class ConsoleInput : MonoBehaviour
{
    void Update()
    {
        if (Input.GetButtonDown("Fire1"))
        {
            Debug.Log("Botó de foc pressionat");
        }
    }
}

Conclusió

Publicar el teu joc de Unity a diferents plataformes implica configurar opcions específiques de cada plataforma i seguir guies específiques. Entenent els requisits i processos per a cada plataforma, pots assegurar una experiència de publicació suau i arribar a una audiència més àmplia. Tant si estàs orientat a escriptori, mòbil, web o consoles, Unity proporciona les eines i la flexibilitat per fer que el teu joc estigui disponible per a jugadors de tot arreu.
