Visió General

En aquesta secció, explorarem els fonaments de la creació de jocs multijugador en Unity. Cobrirem els conceptes bàsics, eines i tècniques necessàries per començar amb el desenvolupament de jocs multijugador. Al final d'aquesta secció, tindràs una comprensió sòlida de com configurar un joc multijugador senzill en Unity.

Conceptes Clau

  • Xarxes: El procés de connectar múltiples dispositius per compartir dades.
  • Model Client-Servidor: Una arquitectura comuna on els clients (jugadors) es connecten a un servidor central.
  • Model Peer-to-Peer: Una arquitectura on cada jugador es connecta directament a altres jugadors.
  • Sincronització: Assegurar que tots els jugadors vegin el mateix estat del joc.
  • Latència: El retard entre una acció i el seu efecte en el joc, sovint degut als temps de transmissió de la xarxa.

Eines i Frameworks

  • Unity Networking (UNet): La solució de xarxes integrada de Unity (descontinuada però encara utilitzada en projectes antics).
  • Mirror: Una biblioteca de xarxes impulsada per la comunitat per a Unity, basada en UNet.
  • Photon Unity Networking (PUN): Una solució de xarxes de tercers popular per a Unity.

Configuració d'un Joc Multijugador Bàsic

Pas 1: Configuració del Projecte

  1. Crear un Nou Projecte de Unity:

    • Obre Unity Hub i crea un nou projecte 3D.
    • Anomena el teu projecte "MultiplayerGame".

  2. Instal·lar el Paquet de Xarxes:

    • Per a aquest exemple, utilitzarem Mirror.
    • Ves a la Unity Asset Store i cerca "Mirror".
    • Importa el paquet Mirror al teu projecte.

Pas 2: Creació del Network Manager

  1. Afegir Network Manager:
    • Crea un GameObject buit a la teva escena i anomena'l "NetworkManager".
    • Afegeix els components NetworkManager i NetworkManagerHUD a aquest GameObject.
using Mirror;

public class MyNetworkManager : NetworkManager
{
    public override void OnServerAddPlayer(NetworkConnection conn)
    {
        // Crida la funcionalitat base (afegeix el jugador)
        base.OnServerAddPlayer(conn);

        // La configuració addicional per al jugador es pot fer aquí
    }
}
  1. Configurar Network Manager:
    • Configura el Player Prefab al teu GameObject de jugador.
    • Assegura't que el component NetworkManagerHUD estigui habilitat per a una interfície d'usuari senzilla per iniciar/aturar el servidor i el client.

Pas 3: Creació del Prefab del Jugador

  1. Crear el GameObject del Jugador:

    • Crea un nou GameObject i anomena'l "Player".
    • Afegeix un component NetworkIdentity al GameObject del Jugador.
    • Afegeix un component NetworkTransform per sincronitzar la posició del jugador a través de la xarxa.

  2. Crear el Script del Jugador:

    • Crea un nou script anomenat PlayerController i adjunta'l al GameObject del Jugador.
using UnityEngine;
using Mirror;

public class PlayerController : NetworkBehaviour
{
    public float moveSpeed = 5f;

    void Update()
    {
        if (!isLocalPlayer) return;

        float moveX = Input.GetAxis("Horizontal") * moveSpeed * Time.deltaTime;
        float moveZ = Input.GetAxis("Vertical") * moveSpeed * Time.deltaTime;

        transform.Translate(new Vector3(moveX, 0, moveZ));
    }
}
  1. Convertir el Jugador en Prefab:
    • Arrossega el GameObject del Jugador a la finestra del Projecte per crear un prefab.
    • Elimina el GameObject del Jugador de l'escena.
    • Assigna el prefab del Jugador al camp Player Prefab en el NetworkManager.

Pas 4: Provar el Joc Multijugador

  1. Executar el Joc:
    • Fes clic al botó Play a Unity.
    • Utilitza el NetworkManagerHUD per iniciar un Host (que actua com a servidor i client).
    • Obre una segona instància del joc i connecta't com a client.
    • Mou tots dos jugadors i observa la sincronització.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per configurar un joc multijugador en Unity utilitzant la biblioteca de xarxes Mirror. Hem discutit conceptes clau de xarxes, hem configurat un NetworkManager, hem creat un prefab de jugador i hem provat la funcionalitat multijugador. Aquest coneixement fonamental servirà com a base per a funcions multijugador més avançades i optimitzacions en Unity.
