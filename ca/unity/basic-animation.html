Introducció a l'animació a Unity

Unity és un potent motor de desenvolupament de jocs que permet als desenvolupadors crear animacions impressionants. Entendre els conceptes bàsics de l'animació és crucial per donar vida als teus personatges i entorns de joc.

Conceptes clau

  • Clip d'animació: Un contenidor per a dades d'animació que es poden aplicar a GameObjects.
  • Controlador d'animació: Gestiona múltiples clips d'animació i les transicions entre ells.
  • Fotogrames clau: Punts específics en el temps on defineixes el valor d'una propietat.
  • Línia de temps: Una representació visual de la seqüència d'animació.

Creant la teva primera animació

Configurant l'escena

  1. Crea un nou projecte de Unity.
  2. Afegeix un GameObject (per exemple, un cub) a l'escena.
  3. Obre la finestra d'animació (Window > Animation > Animation).

Creant un clip d'animació

  1. Selecciona el GameObject.
  2. Fes clic a "Create" a la finestra d'animació per crear un nou clip d'animació.
  3. Desa el clip d'animació amb un nom descriptiu (per exemple, "CubeBounce").

Afegint fotogrames clau

  1. Mou el cursor de la línia de temps al començament (0:00).
  2. Modifica les propietats del GameObject (per exemple, posició, rotació, escala).
  3. Fes clic al botó "Add Keyframe" per enregistrar els valors de les propietats.
  4. Mou el cursor de la línia de temps a una nova posició (per exemple, 1:00).
  5. Canvia les propietats del GameObject de nou.
  6. Afegeix un altre fotograma clau.

Exemple de codi: Movent un cub amunt i avall

using UnityEngine;

public class CubeBounce : MonoBehaviour
{
    void Update()
    {
        float y = Mathf.PingPong(Time.time, 2);
        transform.position = new Vector3(transform.position.x, y, transform.position.z);
    }
}
  • Explicació:
    • Mathf.PingPong(Time.time, 2) crea un valor que oscil·la entre 0 i 2.
    • Aquest valor s'assigna a la coordenada y de la posició del cub, fent que es mogui amunt i avall.

Utilitzant el controlador d'animació

Creant un controlador d'animació

  1. Fes clic dret a la finestra del projecte i selecciona Create > Animator Controller.
  2. Nomena el controlador d'animació (per exemple, "CubeAnimator").
  3. Selecciona el GameObject i assigna el controlador d'animació al component Animator.

Afegint clips d'animació al controlador d'animació

  1. Obre la finestra d'Animator (Window > Animation > Animator).
  2. Arrossega i deixa anar els clips d'animació a la finestra d'Animator.
  3. Crea transicions entre els estats d'animació.

Exemple: Transició entre dues animacions

using UnityEngine;

public class AnimationSwitcher : MonoBehaviour
{
    private Animator animator;

    void Start()
    {
        animator = GetComponent<Animator>();
    }

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.Space))
        {
            animator.SetTrigger("Jump");
        }
    }
}
  • Explicació:
    • animator.SetTrigger("Jump") activa una transició a l'animació "Jump" quan es prem la barra espaiadora.

Tècniques avançades d'animació

Blend Trees

  • Blend Trees permeten transicions suaus entre múltiples animacions basades en paràmetres.
  • Exemple: Barrejar entre animacions de caminar i córrer basant-se en la velocitat del personatge.

Esdeveniments d'animació

  • Esdeveniments d'animació poden cridar funcions en punts específics d'una animació.
  • Exemple: Activar un efecte de so quan el peu d'un personatge toca el terra.

Cinemàtica inversa (IK)

  • IK permet animacions més realistes ajustant automàticament les posicions de les extremitats.
  • Exemple: Fer que la mà d'un personatge arribi a un objecte específic.

Resum

En aquest tema, hem cobert els conceptes bàsics de l'animació a Unity, incloent la creació i gestió de clips d'animació, l'ús del controlador d'animació i la implementació de tècniques avançades com Blend Trees, esdeveniments d'animació i cinemàtica inversa. Amb aquestes habilitats, pots crear animacions dinàmiques i atractives per als teus projectes de Unity.
