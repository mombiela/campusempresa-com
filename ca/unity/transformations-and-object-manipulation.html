Introducció

A Unity, les transformacions i la manipulació d'objectes són conceptes fonamentals que et permeten controlar la posició, la rotació i l'escala dels objectes del joc. Aquest tema cobrirà les bases de les transformacions, com manipular objectes a través de codi, i tècniques avançades per crear escenes dinàmiques i interactives.

Conceptes Clau

  • Component Transform: Cada objecte del joc a Unity té un component Transform que emmagatzema la seva posició, rotació i escala.
  • Posició: La ubicació de l'objecte a l'espai 3D.
  • Rotació: L'orientació de l'objecte a l'espai 3D.
  • Escala: La mida de l'objecte a l'espai 3D.
  • Relacions Pare-Fill: Relacions jeràrquiques entre objectes que afecten les seves transformacions.

Component Transform

El component Transform és crucial per a qualsevol objecte del joc a Unity. Et permet controlar la posició, la rotació i l'escala de l'objecte.

Posició

La posició d'un objecte es defineix per les seves coordenades a l'espai 3D (x, y, z).

// Establint la posició d'un objecte del joc
transform.position = new Vector3(0, 1, 0);

Rotació

La rotació d'un objecte es defineix pels seus angles Euler o Quaternion.

// Establint la rotació d'un objecte del joc utilitzant angles Euler
transform.rotation = Quaternion.Euler(0, 90, 0);

Escala

L'escala d'un objecte es defineix per la seva mida a l'espai 3D.

// Establint l'escala d'un objecte del joc
transform.localScale = new Vector3(2, 2, 2);

Manipulació d'Objectes a Través de Codi

Moure Objectes

Pots moure objectes canviant la seva posició al llarg del temps.

void Update() {
    // Moure l'objecte cap endavant
    transform.Translate(Vector3.forward * Time.deltaTime);
}

Rotar Objectes

Pots rotar objectes canviant la seva rotació al llarg del temps.

void Update() {
    // Rotar l'objecte al voltant de l'eix y
    transform.Rotate(Vector3.up * 45 * Time.deltaTime);
}

Escalar Objectes

Pots escalar objectes canviant la seva propietat localScale.

void Update() {
    // Augmentar l'escala de l'objecte
    transform.localScale += new Vector3(0.1f, 0.1f, 0.1f) * Time.deltaTime;
}

Relacions Pare-Fill

Els objectes a Unity poden tenir relacions jeràrquiques, on la transformació d'un objecte pare afecta els seus fills.

Crear Relacions Pare-Fill

Pots crear relacions pare-fill establint la propietat parent d'un Transform.

// Establint el pare d'un objecte del joc
childObject.transform.parent = parentObject.transform;

Afectar les Transformacions dels Fills

Quan mous, rotes o escales un objecte pare, els seus fills es veuen afectats en conseqüència.

void Update() {
    // Moure l'objecte pare
    parentObject.transform.Translate(Vector3.forward * Time.deltaTime);
    // L'objecte fill també es mourà cap endavant
}

Tècniques Avançades

Interpolacions

Les tècniques d'interpolació com Lerp (Interpolació Lineal) es poden utilitzar per a transicions suaus.

void Update() {
    // Moure suaument l'objecte a una nova posició
    transform.position = Vector3.Lerp(transform.position, new Vector3(10, 0, 0), Time.deltaTime);
}

Utilitzar Coroutines per a Transicions Suaus

Les coroutines es poden utilitzar per crear transicions suaus al llarg del temps.

IEnumerator MoveOverTime(Vector3 targetPosition, float duration) {
    Vector3 startPosition = transform.position;
    float elapsedTime = 0;

    while (elapsedTime < duration) {
        transform.position = Vector3.Lerp(startPosition, targetPosition, elapsedTime / duration);
        elapsedTime += Time.deltaTime;
        yield return null;
    }

    transform.position = targetPosition;
}

Conclusió

Entendre les transformacions i la manipulació d'objectes és essencial per crear escenes dinàmiques i interactives a Unity. Dominant aquests conceptes, podràs controlar la posició, la rotació i l'escala dels objectes del joc, crear relacions jeràrquiques i implementar transicions suaus. Practica aquestes tècniques per millorar les teves habilitats de desenvolupament a Unity i crear jocs més atractius.
