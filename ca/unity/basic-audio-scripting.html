Introducció

En aquesta secció, cobrirem els fonaments del scripting d'àudio a Unity. Aprendràs a afegir, controlar i manipular fonts d'àudio i oients mitjançant scripting. Al final d'aquest tema, hauràs de ser capaç de crear experiències d'àudio dinàmiques en els teus projectes de Unity.

Conceptes Clau

  • Font d'Àudio: El component que reprodueix un clip d'àudio a l'escena.
  • Oient d'Àudio: El component que captura l'àudio i l'envia als altaveus.
  • Clip d'Àudio: L'arxiu d'àudio que és reproduït per la Font d'Àudio.
  • So 3D: Àudio que canvia segons la posició de l'oient en l'espai 3D.

Configuració de Fonts i Oients d'Àudio

Per començar a treballar amb àudio a Unity, necessites entendre com configurar Fonts d'Àudio i Oients.

Afegir una Font d'Àudio

  1. Crea un GameObject buit o selecciona'n un d'existent.

  2. Afegeix el component Font d'Àudio:

    // Script en C# per afegir un component Font d'Àudio
void Start()
{
    gameObject.AddComponent<AudioSource>();
}

  3. Assigna un Clip d'Àudio a la Font d'Àudio:

    public AudioClip myClip;

void Start()
{
    AudioSource audioSource = gameObject.AddComponent<AudioSource>();
    audioSource.clip = myClip;
}

Afegir un Oient d'Àudio

  1. Assegura't que només hi hagi un Oient d'Àudio a l'escena, típicament adjunt a la Càmera Principal.
  2. Comprova si la Càmera Principal té un Oient d'Àudio: 
    void Start()
{
    if (Camera.main.GetComponent<AudioListener>() == null)
    {
        Camera.main.gameObject.AddComponent<AudioListener>();
    }
}

Reproducció i Control d'Àudio

Un cop hagis configurat la teva Font d'Àudio i Oient, pots controlar la reproducció dels clips d'àudio.

Reproduir un Clip d'Àudio

  • Reprodueix el clip d'àudio: 
    void Start()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.Play();
}

Aturar un Clip d'Àudio

  • Atura el clip d'àudio: 
    void StopAudio()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.Stop();
}

Pausar i Despausar

  • Pausa i Despausa: 
    void PauseAudio()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.Pause();
}

void UnpauseAudio()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.UnPause();
}

Ajustar Propietats d'Àudio

Pots ajustar diverses propietats de la Font d'Àudio per crear experiències d'àudio més dinàmiques.

Control de Volum

  • Ajusta el volum: 
    void SetVolume(float volume)
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.volume = volume;
}

Control de To

  • Ajusta el to: 
    void SetPitch(float pitch)
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.pitch = pitch;
}

Looping i Mescla Espacial

  • Habilita el looping:

    void EnableLooping()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.loop = true;
}

  • Estableix la mescla espacial (0 per 2D, 1 per 3D):

    void SetSpatialBlend(float blend)
{
    AudioSource audioSource = GetComponent<AudioSource>();
    audioSource.spatialBlend = blend;
}

Scripting d'Àudio Avançat

Per a scripting d'àudio més avançat, pots manipular l'àudio mitjançant codi per crear comportaments complexos.

Reproduir Múltiples Clips d'Àudio

  • Reprodueix múltiples clips d'àudio: 
    public AudioClip[] audioClips;

void PlayRandomClip()
{
    AudioSource audioSource = GetComponent<AudioSource>();
    int index = Random.Range(0, audioClips.Length);
    audioSource.clip = audioClips[index];
    audioSource.Play();
}

Esvaïment d'Àudio

  • Esvaeix dins i fora: 
    IEnumerator FadeOut(AudioSource audioSource, float FadeTime)
{
    float startVolume = audioSource.volume;

    while (audioSource.volume > 0)
    {
        audioSource.volume -= startVolume * Time.deltaTime / FadeTime;
        yield return null;
    }

    audioSource.Stop();
    audioSource.volume = startVolume;
}

IEnumerator FadeIn(AudioSource audioSource, float FadeTime)
{
    audioSource.Play();
    audioSource.volume = 0f;

    while (audioSource.volume < 1.0f)
    {
        audioSource.volume += Time.deltaTime / FadeTime;
        yield return null;
    }
}

Conclusió

En aquest tema, hem cobert els conceptes bàsics del scripting d'àudio a Unity, incloent la configuració de fonts i oients d'àudio, el control de la reproducció, l'ajust de propietats d'àudio i la implementació de comportaments d'àudio més avançats. Amb aquestes habilitats, pots crear experiències d'àudio immersives en els teus projectes de Unity. Continua practicant i experimentant amb diferents configuracions d'àudio per dominar el scripting d'àudio a Unity.
