Cerca de camins i navegació

Introducció

La cerca de camins i la navegació són aspectes crucials del desenvolupament de jocs, especialment en la creació de personatges AI intel·ligents i responsius. A Unity, aquests conceptes s'implementen utilitzant el sistema NavMesh, que permet als personatges navegar pel món del joc de manera eficient.

Conceptes clau

  • NavMesh: Un mesh de navegació que representa les superfícies caminables en el món del joc.
  • NavMesh Agent: Un component que permet als personatges navegar pel NavMesh.
  • NavMesh Obstacles: Objectes que poden afectar dinàmicament el NavMesh, fent que els agents naveguin al seu voltant.
  • Algoritmes de cerca de camins: Tècniques utilitzades per trobar el camí més curt d'un punt a un altre.

Configuració del NavMesh

Creació d'un NavMesh

  1. Coure el NavMesh:
    • Ves a Window > AI > Navigation.
    • Selecciona la pestanya Bake.
    • Ajusta els paràmetres com Agent Radius, Agent Height, i Max Slope.
    • Fes clic a Bake per generar el NavMesh.
// Exemple: Coure el NavMesh en codi (ús avançat)
using UnityEngine;
using UnityEngine.AI;

public class NavMeshBaker : MonoBehaviour
{
    void Start()
    {
        NavMeshSurface surface = GetComponent<NavMeshSurface>();
        surface.BuildNavMesh();
    }
}
  1. Component NavMesh Surface:
    • Adjunta el component NavMesh Surface a un GameObject.
    • Configura el component i fes clic a Bake.

Afegir NavMesh Agents

  1. Adjuntar NavMesh Agent:

    • Selecciona el GameObject del personatge.
    • Afegeix el component NavMesh Agent.

  2. Configurar NavMesh Agent:

    • Configura propietats com Speed, Angular Speed, i Stopping Distance.
// Exemple: Moure un NavMesh Agent a una posició objectiu
using UnityEngine;
using UnityEngine.AI;

public class AgentMover : MonoBehaviour
{
    public Transform target;

    void Start()
    {
        NavMeshAgent agent = GetComponent<NavMeshAgent>();
        agent.SetDestination(target.position);
    }
}

Navegació Dinàmica

NavMesh Obstacles

  • Obstacles Estàtics: Objectes que no es mouen però afecten el NavMesh.
  • Obstacles Dinàmics: Objectes que es mouen i actualitzen dinàmicament el NavMesh.
// Exemple: Afegir un NavMesh Obstacle
using UnityEngine;

public class ObstacleCreator : MonoBehaviour
{
    void Start()
    {
        GameObject obstacle = GameObject.CreatePrimitive(PrimitiveType.Cube);
        obstacle.AddComponent<NavMeshObstacle>();
        obstacle.GetComponent<NavMeshObstacle>().carving = true;
    }
}

Enllaços Off-Mesh

  • Enllaços Manuals: Enllaços col·locats manualment per salts, escalades, etc.
  • Enllaços Automàtics: Enllaços generats automàticament per una navegació sense interrupcions.
// Exemple: Crear un Off-Mesh Link
using UnityEngine;
using UnityEngine.AI;

public class OffMeshLinkCreator : MonoBehaviour
{
