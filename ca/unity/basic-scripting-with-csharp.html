Introducció

Benvingut al curs "Scripting bàsic amb C#" per a Unity. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat en scripting amb C# dins de l'entorn Unity. Al final d'aquest curs, seràs capaç de crear jocs interactius i dinàmics utilitzant scripts en C#.

Començant amb C# a Unity

Configurant el teu entorn

  • Instal·la Unity Hub i Unity Editor: Assegura't de tenir instal·lada la versió més recent de Unity.
  • Crea un nou projecte: Obre Unity Hub, fes clic a "New Project" i selecciona la plantilla "3D".
  • Familiaritza't amb la interfície de Unity: Entén les finestres Scene, Game, Hierarchy, Project i Inspector.

El teu primer script

  • Creant un script:

    1. Fes clic dret a la finestra Project.
    2. Selecciona Create > C# Script.
    3. Anomena el teu script HelloWorld.

  • Editant el script:

    using UnityEngine;

public class HelloWorld : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        Debug.Log("Hello, World!");
    }

    // Update is called once per frame
    void Update()
    {

    }
}

  • Adjuntant el script a un GameObject:

    1. Crea un nou GameObject a la finestra Hierarchy.
    2. Arrossega i deixa anar el script HelloWorld sobre el GameObject.

Conceptes clau

  • MonoBehaviour: La classe base de la qual deriva cada script de Unity.
  • Mètodes Start() i Update():
    • Start(): Es crida abans de l'actualització del primer frame.
    • Update(): Es crida una vegada per frame.

Variables i tipus de dades

Declarant variables

  • Sintaxi: 
    int score = 0;
float speed = 5.5f;
string playerName = "Player1";
bool isGameOver = false;

Tipus de dades comuns

  • int: Nombres enters.
  • float: Nombres de punt flotant.
  • string: Text.
  • bool: Valors booleans (true/false).

Exemple: Utilitzant variables a Unity

using UnityEngine;

public class Player : MonoBehaviour
{
    public int score = 0;
    public float speed = 5.5f;
    public string playerName = "Player1";
    public bool isGameOver = false;

    void Start()
    {
        Debug.Log("Player Name: " + playerName);
        Debug.Log("Initial Score: " + score);
    }

    void Update()
    {
        if (isGameOver)
        {
            Debug.Log("Game Over!");
        }
    }
}

Estructures de control

Declaracions condicionals

  • if-else: 
    if (score >= 100)
{
    Debug.Log("You win!");
}
else
{
    Debug.Log("Keep trying!");
}

Bucles

  • for Loop:

    for (int i = 0; i < 10; i++)
{
    Debug.Log("Count: " + i);
}

  • while Loop:

    int count = 0;
while (count < 10)
{
    Debug.Log("Count: " + count);
    count++;
}

Funcions i mètodes

Creant funcions

  • Sintaxi: 
    void PrintMessage(string message)
{
    Debug.Log(message);
}

Cridant funcions

  • Exemple: 
    void Start()
{
    PrintMessage("Hello from a function!");
}

Conceptes específics de Unity

GameObject i Components

  • GameObject: Els objectes fonamentals en les escenes de Unity.
  • Components: Scripts, física, renderització i altres funcionalitats adjuntes als GameObjects.

Component Transform

  • Posició, Rotació, Escala: 
    void Update()
{
    transform.position = new Vector3(0, 1, 0);
    transform.rotation = Quaternion.Euler(0, 45, 0);
    transform.localScale = new Vector3(1, 2, 1);
}

Gestionant l'entrada

  • Entrada de teclat: 
    void Update()
{
    if (Input.GetKeyDown(KeyCode.Space))
    {
        Debug.Log("Space key was pressed.");
    }
}
