Què són els Prefabs?

Els Prefabs (abreviatura de "objectes prefabricats") són una característica poderosa a Unity que et permet crear, configurar i emmagatzemar un GameObject complet amb tots els seus components, propietats i objectes fills. Un cop creats, els Prefabs es poden reutilitzar al llarg del teu projecte, facilitant la gestió i la instanciació d'objectes complexos.

Conceptes Clau

  • Plantilles Reutilitzables: Els Prefabs actuen com a plantilles que es poden instanciar múltiples vegades en una escena.
  • Consistència: Els canvis fets a un asset Prefab es reflecteixen a totes les instàncies d'aquest Prefab.
  • Eficiència: Els Prefabs ajuden a reduir la redundància i a millorar l'eficiència del flux de treball.

Creació d'un Prefab

Guia Pas a Pas

  1. Crear un GameObject: Comença creant un GameObject a la teva escena. Això podria ser qualsevol cosa, des d'un cub simple fins a un model de personatge complex.
  2. Configurar el GameObject: Afegeix components, estableix propietats i organitza objectes fills segons sigui necessari.
  3. Crear l'Asset Prefab: Arrossega el GameObject des de la finestra Hierarchy a la finestra Project. Aquesta acció crea un nou asset Prefab.
  4. Instanciar el Prefab: Arrossega l'asset Prefab des de la finestra Project de nou a la finestra Hierarchy per crear instàncies del Prefab.

Exemple

// Exemple de script per instanciar un Prefab
using UnityEngine;

public class PrefabSpawner : MonoBehaviour
{
    public GameObject prefab; // Referència al Prefab

    void Start()
    {
        // Instanciar el Prefab a l'origen sense rotació
        Instantiate(prefab, Vector3.zero, Quaternion.identity);
    }
}

Edició de Prefabs

Edició Directa

  • Obrir Mode Prefab: Fes doble clic a l'asset Prefab a la finestra Project per obrir-lo en Mode Prefab. Aquí, pots fer canvis directament al Prefab.
  • Aplicar Canvis: Després de fer canvis, fes clic al botó "Apply" a l'Inspector per desar els canvis a l'asset Prefab.

Sobreescriptures d'Instància

  • Canvis Específics d'Instància: Pots fer canvis a instàncies individuals d'un Prefab a l'escena. Aquests canvis es diuen "overrides".
  • Revertir o Aplicar Sobreescriptures: Utilitza el botó "Revert" per descartar les sobreescriptures o el botó "Apply" per aplicar-les a l'asset Prefab.

Conceptes Avançats de Prefabs

Prefabs Niats

  • Definició: Els Prefabs poden contenir altres Prefabs, permetent estructures jeràrquiques complexes.
  • Cas d'Ús: Útil per crear components modulars que es poden reutilitzar en diverses combinacions.

Variants de Prefab

  • Definició: Les variants de Prefab són una manera de crear variacions d'un Prefab base mantenint una connexió amb l'original.
  • Cas d'Ús: Ideal per crear diferents versions d'un objecte (per exemple, diferents tipus d'enemics) que comparteixen propietats comunes.

Exemple

// Exemple de script per demostrar Variants de Prefab
using UnityEngine;

public class EnemySpawner : MonoBehaviour
{
    public GameObject baseEnemyPrefab; // Referència al Prefab base de l'enemic
    public GameObject fastEnemyVariant; // Referència a una variant ràpida de l'enemic

    void Start()
    {
        // Instanciar l'enemic base
        Instantiate(baseEnemyPrefab, new Vector3(0, 0, 0), Quaternion.identity);

        // Instanciar la variant ràpida de l'enemic
        Instantiate(fastEnemyVariant, new Vector3(2, 0, 0), Quaternion.identity);
    }
}

Conclusió

Els Prefabs són una característica essencial a Unity que milloren significativament el teu flux de treball promovent la reutilització, la consistència i l'eficiència. En dominar els Prefabs, pots crear projectes complexos, modulars i fàcilment gestionables. Tant si estàs treballant en un joc simple com en una aplicació a gran escala, entendre i utilitzar els Prefabs agilitzarà el teu procés de desenvolupament i millorarà la teva productivitat.
