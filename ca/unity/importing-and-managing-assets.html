Introducció

A Unity, els actius són els blocs de construcció del teu joc. Inclouen models 3D, textures, arxius d'àudio, scripts i més. Importar i gestionar adequadament aquests actius és crucial per a un desenvolupament de jocs eficient.

Importació d'Actius

Importació des de la Unity Asset Store

  • Unity Asset Store: Un mercat on pots trobar actius gratuïts i de pagament.
    1. Obre la finestra de l'Asset Store: Window > Asset Store.
    2. Cerca l'actiu desitjat.
    3. Fes clic a l'actiu i selecciona Download o Purchase.
    4. Un cop descarregat, fes clic a Import per portar l'actiu al teu projecte.

Importació d'Actius Personalitzats

  • Arrossegar i Deixar Caure: Simplement arrossega els arxius des del teu explorador de fitxers a la carpeta Assets a Unity.
  • Utilitzant el Diàleg d'Importació:
    1. Fes clic dret a la carpeta Assets a la finestra del Projecte.
    2. Selecciona Import New Asset....
    3. Tria l'arxiu que vols importar i fes clic a Import.

Tipus d'Arxius Suportats

  • Models 3D: .fbx, .obj, .dae, etc.
  • Textures: .png, .jpg, .tga, etc.
  • Àudio: .mp3, .wav, .ogg, etc.
  • Scripts: .cs (scripts en C#).

Gestió d'Actius

Organització de la Carpeta d'Actius

  • Estructura de Carpetes: Crea una estructura de carpetes lògica per mantenir els teus actius organitzats.
    • Exemple: 
      Assets/
├── Models/
├── Textures/
├── Audio/
├── Scripts/
├── Prefabs/
└── Scenes/
  • Convencions de Nomenclatura: Utilitza convencions de nomenclatura clares i consistents per als teus actius.

Etiquetes d'Actius

  • Utilitzant Etiquetes: Les etiquetes t'ajuden a categoritzar i trobar actius ràpidament.
    1. Selecciona l'actiu a la finestra del Projecte.
    2. A l'Inspector, fes clic al camp Asset Labels.
    3. Afegeix o crea noves etiquetes.

Paquets d'Actius

  • Creació de Paquets d'Actius: Els paquets d'actius et permeten empaquetar actius junts per a una càrrega eficient.
    1. Selecciona els actius que vols empaquetar.
    2. A l'Inspector, estableix el camp AssetBundle a un nom de paquet nou o existent.
    3. Utilitza el mètode BuildPipeline.BuildAssetBundles per crear els paquets.

Configuració d'Importació

  • Configuració d'Importació de Models:
    • Factor d'Escala: Ajusta la mida del model importat.
    • Compressió de Malla: Optimitza la malla per al rendiment.
    • Lectura/Escriptura Habilitada: Permet que la malla sigui modificada en temps d'execució.
  • Configuració d'Importació de Textures:
    • Tipus de Texture: Tria el tipus adequat (per exemple, Per Defecte, Sprite, Mapa de Normals).
    • Mida Màxima: Estableix la resolució màxima.
    • Compressió: Tria el mètode de compressió per equilibrar qualitat i rendiment.
  • Configuració d'Importació d'Àudio:
    • Tipus de Càrrega: Tria entre Decompress on Load, Compressed in Memory, o Streaming.
    • Format de Compressió: Selecciona el format (per exemple, PCM, Vorbis).
// Exemple de configuració de paràmetres d'importació per a una textura
using UnityEditor;
using UnityEngine;

public class TextureImportSettings : AssetPostprocessor
{
    void OnPreprocessTexture()
    {
        TextureImporter importer = (TextureImporter)assetImporter;
        importer.textureType = TextureImporterType.Default;
        importer.maxTextureSize = 1024;
        importer.textureCompression = TextureImporterCompression.Compressed;
    }
}

Conclusió

Importar i gestionar actius eficientment és essencial per a un desenvolupament de jocs fluid a Unity. Entenent com importar actius de diverses fonts, organitzar-los efectivament i configurar els seus paràmetres d'importació, pots assegurar-te que el teu projecte es mantingui manejable i funcioni bé. Practica aquestes tècniques per convertir-te en un expert en la gestió d'actius a Unity.
