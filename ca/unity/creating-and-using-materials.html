Introducció

A Unity, els materials defineixen com apareixen els objectes a la pantalla. Controlen el color, la textura i la reflectivitat dels objectes, entre altres propietats. Entendre com crear i utilitzar materials és essencial per fer que el teu joc sigui visualment atractiu.

Creació de Materials

Creació Bàsica de Materials

  1. Obre la Finestra del Projecte: Navega a la finestra del Projecte a Unity.

  2. Crea un Nou Material:

    • Fes clic dret a la finestra del Projecte.
    • Selecciona Create > Material.
    • Dona un nom al teu nou material (per exemple, "MyMaterial").

  3. Assignació d'un Shader:

    • Per defecte, el nou material utilitza el Shader Estàndard.
    • Pots canviar el shader seleccionant el material i triant un shader diferent del menú desplegable Shader a l'Inspector.

Configuració de les Propietats del Material

  • Color: Canvia el color base del material.
  • Textura: Afegeix una textura al material arrossegant un fitxer de textura a la ranura Albedo.
  • Metàl·lic i Suavitat: Ajusta aquests controls lliscants per canviar la reflectivitat i la suavitat del material.
// Exemple: Creació i assignació d'un material via script
using UnityEngine;

public class MaterialExample : MonoBehaviour
{
    void Start()
    {
        // Crea un nou material amb un shader estàndard
        Material myMaterial = new Material(Shader.Find("Standard"));
        
        // Configura el color a vermell
        myMaterial.color = Color.red;
        
        // Assigna el material al renderer de l'objecte
        GetComponent<Renderer>().material = myMaterial;
    }
}

Ús de Materials

Aplicació de Materials a Objectes

  1. Arrossegar i Deixar Caure:

    • Arrossega el material des de la finestra del Projecte fins a l'objecte a la finestra de l'Escena o Hierarquia.

  2. Via Inspector:

    • Selecciona l'objecte a la finestra de l'Escena o Hierarquia.
    • A l'Inspector, troba el component Mesh Renderer.
    • Arrossega el material a la ranura Element 0 sota Materials.

Instàncies de Material

  • Material Compartit: Els canvis al material afecten tots els objectes que l'utilitzen.
  • Material d'Instància: Els canvis afecten només la instància específica del material a l'objecte.
// Exemple: Modificació d'una instància de material
using UnityEngine;

public class InstanceMaterialExample : MonoBehaviour
{
    void Start()
    {
        // Obté el component renderer
        Renderer renderer = GetComponent<Renderer>();
        
        // Crea una instància del material
        Material instanceMaterial = renderer.material;
        
        // Canvia el color del material d'instància
        instanceMaterial.color = Color.blue;
    }
}

Tècniques Avançades de Material

Ús de Shaders Personalitzats

  • Shader Graph: Eina visual de Unity per crear shaders sense escriure codi.
  • Codi de Shader Personalitzat: Escriu els teus propis shaders utilitzant HLSL.

Blocs de Propietats de Material

  • Canviar Propietats de Material de Manera Eficient: Utilitza MaterialPropertyBlock per canviar propietats sense crear noves instàncies de material.
// Exemple: Ús de MaterialPropertyBlock
using UnityEngine;

public class MaterialPropertyBlockExample : MonoBehaviour
{
    void Start()
    {
        Renderer renderer = GetComponent<Renderer>();
        MaterialPropertyBlock block = new MaterialPropertyBlock();
        
        // Configura una propietat de color
        block.SetColor("_Color", Color.green);
        
        // Aplica el bloc de propietats al renderer
        renderer.SetPropertyBlock(block);
    }
}

Conclusió

Crear i utilitzar materials a Unity és una habilitat fonamental per al desenvolupament de jocs. Entenent com crear materials, configurar les seves propietats i aplicar-los als objectes, pots millorar significativament la qualitat visual dels teus jocs. Tècniques avançades com shaders personalitzats i MaterialPropertyBlocks permeten encara més control i optimització. Practica aquests conceptes per convertir-te en un expert en la gestió de materials a Unity.
