El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, explorarem com importar i utilitzar clips d'àudio a Unity. Això cobrirà tot, des dels conceptes bàsics d'importació d'arxius d'àudio fins a tècniques més avançades com controlar la reproducció d'àudio mitjançant scripts.

Importació de Clips d'Àudio

Per utilitzar àudio en el teu projecte de Unity, primer has d'importar els arxius d'àudio al teu projecte.

Passos per Importar Clips d'Àudio

  1. Prepara els Teus Arxius d'Àudio: Assegura't que els teus arxius d'àudio estan en un format compatible (per exemple, .mp3, .wav, .ogg).
  2. Importació:
    • Obre el teu projecte de Unity.
    • Arrossega i deixa anar els teus arxius d'àudio a la carpeta Assets a la finestra del Projecte.
    • Alternativament, pots utilitzar Assets > Import New Asset des del menú superior i seleccionar els teus arxius d'àudio.

Exemple

Assets/
  Audio/
    background_music.mp3
    jump_sound.wav

Ús de Clips d'Àudio a Unity

Un cop els teus arxius d'àudio estan importats, pots utilitzar-los a les teves escenes.

Afegir Fonts d'Àudio

Es requereix un component de Font d'Àudio per reproduir clips d'àudio a Unity.

  1. Crea un GameObject Buit:

    • Fes clic dret a la finestra de Jerarquia i selecciona Create Empty.
    • Anomena'l AudioManager.

  2. Afegeix el Component de Font d'Àudio:

    • Selecciona el GameObject AudioManager.
    • A la finestra de l'Inspector, fes clic a Add Component i busca Audio Source.
    • Afegeix el component Audio Source.

  3. Assigna el Clip d'Àudio:

    • Arrossega el teu clip d'àudio des de la finestra del Projecte al camp Audio Clip en el component de Font d'Àudio.

Exemple

using UnityEngine;

public class AudioManager : MonoBehaviour
{
    public AudioSource audioSource;
    public AudioClip backgroundMusic;

    void Start()
    {
        audioSource.clip = backgroundMusic;
        audioSource.Play();
    }
}

En aquest exemple:

  • Creem un script AudioManager i l'enganxem al GameObject AudioManager.
  • L'script fa referència a un AudioSource i un AudioClip.
  • En el mètode Start, el clip d'àudio s'assigna a la font d'àudio i es reprodueix.

Control de la Reproducció d'Àudio Mitjançant Scripts

Pots controlar la reproducció d'àudio utilitzant scripts per crear experiències d'àudio més dinàmiques.

Exemple: Reproduir, Pausar i Aturar

using UnityEngine;

public class AudioController : MonoBehaviour
{
    public AudioSource audioSource;

    void Update()
    {
        if (Input.GetKeyDown(KeyCode.P))
        {
            audioSource.Play();
        }
        if (Input.GetKeyDown(KeyCode.S))
        {
            audioSource.Stop();
        }
        if (Input.GetKeyDown(KeyCode.U))
        {
            audioSource.Pause();
        }
    }
}

En aquest exemple:

  • L'script AudioController et permet reproduir, aturar i pausar l'àudio utilitzant les tecles P, S i U, respectivament.

Ús de Mescladors d'Àudio

Els Mescladors d'Àudio et permeten gestionar configuracions d'àudio complexes, incloent-hi el control de volum, efectes i encaminament.

  1. Crea un Mesclador d'Àudio:

    • Fes clic dret a la finestra del Projecte i selecciona Create > Audio Mixer.
    • Anomena'l MainMixer.

  2. Assigna el Mesclador d'Àudio a la Font d'Àudio:

    • Selecciona el GameObject AudioManager.
    • En el component de Font d'Àudio, estableix la Output a MainMixer.

  3. Controla el Volum Mitjançant Script:

using UnityEngine;
using UnityEngine.Audio;

public class VolumeController : MonoBehaviour
{
    public AudioMixer mainMixer;

    public void SetVolume(float volume)
    {
        mainMixer.SetFloat("MasterVolume", volume);
    }
}

En aquest exemple:

  • L'script VolumeController et permet establir el volum del MainMixer mitjançant un mètode públic SetVolume.

Exemple: Esvaïment d'Àudio

using UnityEngine;

public class AudioFader : MonoBehaviour
{
    public AudioSource audioSource;
    public float fadeDuration = 1.0f;

    public void FadeOut()
    {
        StartCoroutine(FadeOutCoroutine());
    }

    private IEnumerator FadeOutCoroutine()
    {
        float startVolume = audioSource.volume;

        for (float t = 0; t < fadeDuration; t += Time.deltaTime)
        {
            audioSource.volume = Mathf.Lerp(startVolume, 0, t / fadeDuration);
            yield return null;
        }

        audioSource.Stop();
        audioSource.volume = startVolume;
    }
}

En aquest exemple:

  • L'script AudioFader redueix gradualment el volum de la font d'àudio durant una durada especificada i després atura l'àudio.
