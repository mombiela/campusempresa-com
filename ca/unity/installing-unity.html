Introducció

Unity és un motor de desenvolupament de jocs potent i àmpliament utilitzat que permet als desenvolupadors crear experiències en 2D, 3D, VR i AR. Aquesta secció et guiarà a través del procés d'instal·lació de Unity i la configuració del teu entorn de desenvolupament.

Instal·lar Unity

Pas 1: Descarregar Unity Hub

Unity Hub és una eina de gestió que t'ajuda a gestionar múltiples instal·lacions i projectes de Unity. Segueix aquests passos per descarregar i instal·lar Unity Hub:

  1. Ves a la Pàgina de Descàrrega de Unity.
  2. Fes clic al botó "Download Unity Hub".
  3. Un cop completada la descàrrega, obre l'instal·lador i segueix les instruccions en pantalla per instal·lar Unity Hub.

Pas 2: Instal·lar Unity Editor

Després d'instal·lar Unity Hub, necessites instal·lar Unity Editor, que és l'eina principal que utilitzaràs per crear els teus projectes.

  1. Obre Unity Hub.
  2. Fes clic a la pestanya "Installs".
  3. Fes clic al botó "Add" per afegir una nova versió de Unity.
  4. Tria la versió de Unity que vols instal·lar. Es recomana utilitzar la versió Long-Term Support (LTS) més recent per a més estabilitat.
  5. Selecciona els mòduls que necessites (per exemple, Android Build Support, iOS Build Support) i fes clic a "Next".
  6. Revisa els termes i condicions, després fes clic a "Done" per començar la instal·lació.

Pas 3: Crear un Unity ID

Es necessita un Unity ID per utilitzar els serveis de Unity i accedir a l'Asset Store.

  1. Obre Unity Hub.
  2. Fes clic a la icona del perfil a la cantonada superior dreta.
  3. Selecciona "Create account" i segueix les instruccions per crear el teu Unity ID.

Configurar el teu Primer Projecte

Pas 1: Crear un Nou Projecte

  1. Obre Unity Hub.
  2. Fes clic a la pestanya "Projects".
  3. Fes clic al botó "New" per crear un nou projecte.
  4. Tria una plantilla (per exemple, 2D, 3D, URP, HDRP) segons les necessitats del teu projecte.
  5. Introdueix un nom de projecte i selecciona una ubicació per desar el teu projecte.
  6. Fes clic a "Create" per crear el teu nou projecte.

Pas 2: Entendre la Interfície de Unity

Un cop creat el teu projecte, seràs rebut amb la interfície de Unity Editor. Aquí tens els components clau:

  • Scene View: Aquí és on pots dissenyar i organitzar visualment els objectes del teu joc.
  • Game View: Això mostra com es veurà el teu joc quan s'executi.
  • Hierarchy: Això llista tots els objectes del joc a la teva escena.
  • Inspector: Això mostra les propietats de l'objecte del joc seleccionat.
  • Project: Això mostra tots els actius del teu projecte.
  • Console: Això mostra missatges de registre, advertències i errors.

Pas 3: Configuració Bàsica

Abans de començar a desenvolupar, és important configurar alguns ajustaments bàsics:

  1. Configuració del Projecte:

    • Ves a Edit > Project Settings.
    • Configura ajustaments com Player, Quality i Input segons els requisits del teu projecte.

  2. Configuració de la Construcció:

    • Ves a File > Build Settings.
    • Selecciona la teva plataforma objectiu (per exemple, PC, Android, iOS).
    • Fes clic a "Switch Platform" si és necessari.

Conclusió

Seguint aquests passos, has instal·lat amb èxit Unity i has configurat el teu primer projecte. Ara estàs preparat per començar a explorar les característiques de Unity i iniciar el teu viatge en el desenvolupament de jocs. En les següents seccions, aprofundirem en les capacitats de Unity, començant per entendre els objectes del joc i els components.
