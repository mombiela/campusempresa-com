L'il·luminació i l'ombrejat són conceptes crucials en gràfics per ordinador, ja que milloren significativament el realisme i l'atractiu visual de les escenes 3D. Aquest tema et guiarà a través dels conceptes bàsics d'il·luminació i ombrejat en DirectX, progressant cap a tècniques més avançades.

Introducció a la Il·luminació

L'il·luminació en DirectX implica simular com la llum interactua amb els objectes en una escena 3D. Hi ha diversos tipus de llums que pots utilitzar:

  • Llum Ambiental: Proporciona un nivell base de llum a tots els objectes de l'escena, assegurant que cap part estigui completament fosca.
  • Llum Direccional: Simula la llum que prové d'una direcció específica, com la llum solar.
  • Llum Puntual: Emet llum en totes direccions des d'un punt únic, similar a una bombeta.
  • Foco: Emet llum en forma de con, com una llanterna.

Llum Ambiental

La llum ambiental és la forma més simple d'il·luminació. Afecta tots els objectes uniformement, proporcionant un nivell base d'il·luminació.

float4 AmbientLighting(float4 color, float3 ambientColor, float ambientIntensity) {
    return color * ambientColor * ambientIntensity;
}

Llum Direccional

Les llums direccionals simulen la llum d'una font distant, com el sol. Tenen una direcció però no una posició.

struct DirectionalLight {
    float3 direction;
    float3 color;
};

float4 DirectionalLighting(float4 color, float3 normal, DirectionalLight light) {
    float NdotL = max(dot(normal, -light.direction), 0.0);
    return color * light.color * NdotL;
}

Llum Puntual

Les llums puntuals emeten llum en totes direccions des d'un punt únic. Tenen una posició i un abast.

struct PointLight {
    float3 position;
    float3 color;
    float range;
};

float4 PointLighting(float4 color, float3 position, float3 normal, PointLight light) {
    float3 lightDir = light.position - position;
    float distance = length(lightDir);
    lightDir = normalize(lightDir);
    float attenuation = saturate(1.0 - distance / light.range);
    float NdotL = max(dot(normal, lightDir), 0.0);
    return color * light.color * NdotL * attenuation;
}

Foco

Els focos són similars a les llums puntuals però emeten llum en forma de con.

struct Spotlight {
    float3 position;
    float3 direction;
    float3 color;
    float range;
    float spotAngle;
};

float4 SpotlightLighting(float4 color, float3 position, float3 normal, Spotlight light) {
    float3 lightDir = light.position - position;
    float distance = length(lightDir);
    lightDir = normalize(lightDir);
    float attenuation = saturate(1.0 - distance / light.range);
    float NdotL = max(dot(normal, lightDir), 0.0);
    
    float spotEffect = dot(lightDir, -light.direction);
    if (spotEffect > cos(light.spotAngle)) {
        spotEffect = pow(spotEffect, 4.0);
        return color * light.color * NdotL * attenuation * spotEffect;
    }
    return float4(0, 0, 0, 0);
}

Introducció a l'Ombrejat

L'ombrejat determina com la superfície d'un objecte interactua amb la llum. Els models d'ombrejat més comuns són:

  • Ombrejat Pla: Cada polígon s'ombreja amb un sol color.
  • Ombrejat Gouraud: Els colors s'interpolen a través dels vèrtexs dels polígons.
  • Ombrejat Phong: Els colors s'interpolen a través de la superfície dels polígons, proporcionant transicions més suaus.

Ombrejat Pla

L'ombrejat pla aplica un sol color a cada polígon, basat en la normal del polígon.

float4 FlatShading(float4 color, float3 normal, DirectionalLight light) {
    float NdotL = max(dot(normal, -light.direction), 0.0);
    return color * light.color * NdotL;
}

Ombrejat Gouraud

L'ombrejat Gouraud interpola els colors dels vèrtexs a través de la superfície del polígon.

struct Vertex {
    float3 position;
    float3 normal;
    float4 color;
};

float4 GouraudShading(Vertex v0, Vertex v1, Vertex v2, DirectionalLight light) {
    float4 color0 = DirectionalLighting(v0.color, v0.normal, light);
    float4 color1 = DirectionalLighting(v1.color, v1.normal, light);
    float4 color2 = DirectionalLighting(v2.color, v2.normal, light);
    // Interpolar colors a través del triangle
    return (color0 + color1 + color2) / 3.0;
}

Ombrejat Phong

L'ombrejat Phong interpola les normals a través de la superfície i calcula el color per píxel.

float4 PhongShading(float3 position, float3 normal, DirectionalLight light) {
    float3 interpolatedNormal = normalize(normal);
    return DirectionalLighting(float4(1, 1, 1, 1), interpolatedNormal, light);
}

Tècniques Avançades d'Il·luminació

Les tècniques avançades d'il·luminació inclouen:

  • Mapatge de Normals: Afegeix detall a les superfícies sense augmentar el recompte de polígons.
  • Reflexos Especulars: Simula superfícies brillants afegint punts brillants on la llum es reflecteix directament.
  • Mapatge d'Ombres: Crea ombres realistes determinat quines parts de l'escena estan ocultes de la font de llum.

Mapatge de Normals

El mapatge de normals utilitza una textura per pertorbar les normals de la superfície, afegint detall.

float3 PerturbNormal(float3 normal, float3 tangent, float3 bitangent, float3 normalMap) {
    float3x3 TBN = float3x3(tangent, bitangent, normal);
    return normalize(mul(normalMap, TBN));
}

Reflexos Especulars

Els reflexos especulars simulen superfícies brillants.

float4 SpecularLighting(float4 color, float3 position, float3 normal, float3 viewDir, DirectionalLight light, float shininess) {
    float3 lightDir = normalize(-light.direction);
    float3 reflectDir = reflect(-lightDir, normal);
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), shininess);
    return color + light.color * spec;
}

Mapatge d'Ombres

El mapatge d'ombres implica renderitzar l'escena des de la perspectiva de la llum per determinar quines àrees estan en ombra.

float ShadowCalculation(float4 fragPosLightSpace, sampler2D shadowMap) {
    float shadow = 0.0;
    float currentDepth = fragPosLightSpace.z / fragPosLightSpace.w;
    float closestDepth = tex2D(shadowMap, fragPosLightSpace.xy).r;
    if (currentDepth > closestDepth) {
        shadow = 1.0;
    }
    return shadow;
}

Conclusió

L'il·luminació i l'ombrejat són essencials per crear escenes 3D realistes i visualment atractives en DirectX. Comprenent i implementant diversos tipus d'il·luminació i models d'ombrejat, pots millorar significativament la qualitat dels teus gràfics. A mesura que progressis cap a tècniques més avançades com el mapatge de normals, els reflexos especulars i el mapatge d'ombres, podràs crear escenes encara més detallades i dinàmiques. Continua experimentant i refinant les teves habilitats per dominar l'art de l'il·luminació i l'ombrejat en DirectX.
