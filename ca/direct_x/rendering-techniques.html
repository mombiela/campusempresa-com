Introducció a les Tècniques de Renderització

La renderització és el procés de generar una imatge a partir d'un model mitjançant programes informàtics. En DirectX, la renderització implica diverses tècniques per produir visuals d'alta qualitat. Aquesta secció et guiarà des de tècniques bàsiques fins a avançades de renderització en DirectX.

Tècniques Bàsiques de Renderització

Configuració del Pipeline de Renderització

  • Processament de Vèrtexs: Transformar coordenades 3D a coordenades de pantalla 2D.
  • Rasterització: Convertir gràfics vectorials en una imatge raster (píxels o punts).
  • Processament de Píxels: Determinar el color de cada píxel.
// Configuració bàsica per a un pipeline de renderització DirectX
void SetupPipeline() {
    // Inicialitzar Direct3D
    D3D11CreateDeviceAndSwapChain(...);

    // Crear vista de destí de renderització
    ID3D11RenderTargetView* renderTargetView;
    device->CreateRenderTargetView(..., &renderTargetView);

    // Establir el destí de renderització
    context->OMSetRenderTargets(1, &renderTargetView, nullptr);

    // Establir la vista
    D3D11_VIEWPORT viewport = {};
    viewport.Width = static_cast<float>(width);
    viewport.Height = static_cast<float>(height);
    viewport.MinDepth = 0.0f;
    viewport.MaxDepth = 1.0f;
    context->RSSetViewports(1, &viewport);
}

Dibuixar Formes Bàsiques

  • Vèrtexs i Índexs: Definir la geometria de les formes.
  • Buffers: Emmagatzemar dades de vèrtexs i índexs.
// Definir vèrtexs per a un triangle
struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT4 color;
};

Vertex vertices[] = {
    { DirectX::XMFLOAT3(0.0f, 0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(-0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 0.0f, 1.0f, 1.0f) }
};

// Crear buffer de vèrtexs
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

Tècniques Intermèdies de Renderització

Texturització

  • Mapatge de Textures: Aplicar imatges a models 3D.
  • Estats de Sampler: Controlar com es mostren les textures.
// Carregar textura
ID3D11ShaderResourceView* textureView;
CreateWICTextureFromFile(device, context, L"texture.jpg", nullptr, &textureView);

// Establir textura i estat de sampler
context->PSSetShaderResources(0, 1, &textureView);

D3D11_SAMPLER_DESC samplerDesc = {};
samplerDesc.Filter = D3D11_FILTER_MIN_MAG_MIP_LINEAR;
samplerDesc.AddressU = D3D11_TEXTURE_ADDRESS_WRAP;
samplerDesc.AddressV = D3D11_TEXTURE_ADDRESS_WRAP;
samplerDesc.AddressW = D3D11_TEXTURE_ADDRESS_WRAP;

ID3D11SamplerState* samplerState;
device->CreateSamplerState(&samplerDesc, &samplerState);
context->PSSetSamplers(0, 1, &samplerState);

Il·luminació

  • Models Bàsics d'Il·luminació: Implementar il·luminació ambiental, difusa i especular.
  • Shaders: Escriure codi HLSL per a càlculs d'il·luminació.
// Shader de píxels simple per a il·luminació
cbuffer LightBuffer : register(b0) {
    float4 lightDirection;
    float4 lightColor;
};

struct PSInput {
    float4 position : SV_POSITION;
    float4 color : COLOR;
    float3 normal : NORMAL;
};

float4 main(PSInput input) : SV_TARGET {
    float3 norm = normalize(input.normal);
    float3 lightDir = normalize(lightDirection.xyz);

    // Il·luminació difusa
    float diffuse = max(dot(norm, lightDir), 0.0);
    float4 diffuseColor = diffuse * lightColor;

    return input.color * diffuseColor;
}

Tècniques Avançades de Renderització

Tècniques Avançades d'Ombrejat

  • Mapatge de Normals: Afegir detall a les superfícies sense augmentar la complexitat de la geometria.
  • Mapatge de Parallax: Simular profunditat en textures.
// Mapatge de normals en shader de píxels
cbuffer NormalMapBuffer : register(b1) {
    float4x4 normalMatrix;
};

Texture2D normalMap : register(t1);

float4 main(PSInput input) : SV_TARGET {
    float3 normal = normalMap.Sample(samplerState, input.texCoord).xyz * 2.0 - 1.0;
    normal = mul(normal, (float3x3)normalMatrix);

    // Càlculs d'il·luminació amb mapatge de normals
    float3 lightDir = normalize(lightDirection.xyz);
    float diffuse = max(dot(normal, lightDir), 0.0);
    float4 diffuseColor = diffuse * lightColor;

    return input.color * diffuseColor;
}

Efectes de Postprocessament

  • Bloom: Crear un efecte de resplendor al voltant de les àrees brillants.
  • HDR: Renderització d'Alta Gamma Dinàmica per a una il·luminació més realista.
// Efecte de bloom simple en shader de píxels
Texture2D sceneTexture : register(t0);
Texture2D bloomTexture : register(t1);

float4 main(PSInput input) : SV_TARGET {
    float4 sceneColor = sceneTexture.Sample(samplerState, input.texCoord);
    float4 bloomColor = bloomTexture.Sample(samplerState, input.texCoord);

    // Combinar textures de l'escena i bloom
    return sceneColor + bloomColor * 0.5;
}

Conclusió

Les tècniques de renderització en DirectX van des de la configuració bàsica i el dibuix de formes fins a l'ombrejat avançat i els efectes de postprocessament. En dominar aquestes tècniques, pots crear aplicacions i jocs visualment impressionants. Practica cada tècnica, experimenta amb diferents shaders i explora les vastes possibilitats que DirectX ofereix per a la renderització.
