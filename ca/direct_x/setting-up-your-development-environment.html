El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el teu entorn de desenvolupament és el primer pas crucial en el desenvolupament d'aplicacions amb DirectX. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar, assegurant que tinguis una base sòlida tant per a la programació de DirectX per a principiants com per a avançats.

Eines i requisits de programari

Abans de submergir-te en la programació de DirectX, necessites instal·lar i configurar diverses eines i paquets de programari. Aquí tens una llista del que necessitaràs:

  • Sistema Operatiu Windows: DirectX és una col·lecció d'APIs per gestionar tasques relacionades amb multimèdia, especialment la programació de jocs i vídeo, en plataformes de Microsoft.
  • Visual Studio: Un entorn de desenvolupament integrat (IDE) de Microsoft. La Visual Studio Community Edition és gratuïta i suficient per als nostres propòsits.
  • DirectX SDK: Tot i que el DirectX SDK ara forma part del Windows SDK, algunes mostres i eines antigues encara poden requerir el DirectX SDK independent.

Instal·lació de Visual Studio

  1. Descarrega Visual Studio: Visita la pàgina de descàrrega de Visual Studio i descarrega la Community Edition.
  2. Instal·la Visual Studio:
    • Executa l'instal·lador i selecciona la càrrega de treball "Desenvolupament d'escriptori amb C++".
    • Assegura't que l'opció "Desenvolupament de jocs amb C++" també estigui seleccionada, ja que inclou eines i biblioteques per al desenvolupament de DirectX.
  3. Configura Visual Studio:
    • Obre Visual Studio i ves a Eines > Opcions.
    • Navega a Projectes i solucions > Directoris de VC++ i assegura't que les rutes per als fitxers d'inclusió, fitxers de biblioteca i fitxers executables estiguin configurades correctament.

Configuració del DirectX SDK

Tot i que la majoria dels components de DirectX estan inclosos en el Windows SDK, algunes eines i mostres antigues poden requerir el DirectX SDK independent.

  1. Descarrega el DirectX SDK: Pots trobar el DirectX SDK (juny de 2010) al lloc web de Microsoft.
  2. Instal·la el DirectX SDK:
    • Executa l'instal·lador i segueix les instruccions en pantalla.
    • Nota la ruta d'instal·lació, ja que hauràs de referenciar-la a Visual Studio.

Configuració de Visual Studio per al desenvolupament de DirectX

  1. Inclou les rutes del DirectX SDK:

    • Obre Visual Studio i ves a Eines > Opcions.
    • Navega a Projectes i solucions > Directoris de VC++.
    • Afegeix les rutes d'inclusió i biblioteca del DirectX SDK als respectius directoris.

  2. Crea un nou projecte:

    • Ves a Fitxer > Nou > Projecte.
    • Selecciona Visual C++ i després Projecte Win32.
    • Nomena el teu projecte i fes clic a D'acord.
    • A l'Assistent d'Aplicacions Win32, selecciona Configuració de l'aplicació i tria Projecte buit.

  3. Afegeix les biblioteques de DirectX:

    • Fes clic dret sobre el teu projecte a l'Explorador de solucions i selecciona Propietats.
    • Ves a Propietats de configuració > Enllaçador > Entrada.
    • Afegeix les biblioteques necessàries de DirectX (per exemple, d3d11.lib, d3dx11.lib, dxgi.lib) al camp Dependències addicionals.

Escriure el teu primer programa DirectX

Ara que el teu entorn està configurat, escrivim un programa simple de DirectX per assegurar-nos que tot funciona correctament.

#include <windows.h>
#include <d3d11.h>

// Enllaça les biblioteques necessàries
#pragma comment (lib, "d3d11.lib")

// Declaracions globals
IDXGISwapChain *swapchain;             // el punter a la interfície de la cadena d'intercanvi
ID3D11Device *dev;                     // el punter a la nostra interfície de dispositiu Direct3D
ID3D11DeviceContext *devcon;           // el punter al nostre context de dispositiu Direct3D

// Prototips de funcions
void InitD3D(HWND hWnd);    // configura i inicialitza Direct3D
void CleanD3D(void);        // tanca Direct3D i allibera memòria

// El punt d'entrada per a qualsevol programa de Windows
int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow)
{
    HWND hWnd;
    WNDCLASSEX wc;

    ZeroMemory(&wc, sizeof(WNDCLASSEX));

    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WindowProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = L"WindowClass";

    RegisterClassEx(&wc);

    hWnd = CreateWindowEx(NULL,
                          L"WindowClass",
                          L"Our First Direct3D Program",
                          WS_OVERLAPPEDWINDOW,
                          300, 300,
                          800, 600,
                          NULL,
                          NULL,
                          hInstance,
                          NULL);

    ShowWindow(hWnd, nCmdShow);

    // configura i inicialitza Direct3D
    InitD3D(hWnd);

    // entra al bucle principal:

    MSG msg;

    while(TRUE)
    {
        while(PeekMessage(&msg, NULL, 0, 0, PM_REMOVE))
        {
            TranslateMessage(&msg);
            DispatchMessage(&msg);
        }

        if(msg.message == WM_QUIT)
            break;
    }

    // neteja Direct3D i COM
    CleanD3D();

    return msg.wParam;
}

// aquest és el controlador principal de missatges per al programa
LRESULT CALLBACK WindowProc(HWND hWnd, UINT message, WPARAM wParam, LPARAM lParam)
{
    switch(message)
    {
        case WM_DESTROY:
            {
                PostQuitMessage(0);
                return 0;
            } break;
    }

    return DefWindowProc (hWnd, message, wParam, lParam);
}

// aquesta funció inicialitza i prepara Direct3D per al seu ús
void InitD3D(HWND hWnd)
{
    // crea una estructura per contenir informació sobre la cadena d'intercanvi
    DXGI_SWAP_CHAIN_DESC scd;

    // neteja l'estructura per al seu ús
    ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));

    // omple la descripció de la cadena d'intercanvi
    scd.BufferCount = 1;                                    // un buffer de fons
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;     // utilitza color de 32 bits
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;      // com s'utilitzarà la cadena d'intercanvi
    scd.OutputWindow = hWnd;                                // la finestra que s'utilitzarà
    scd.SampleDesc.Count = 4;                               // quants mostreigs múltiples
    scd.Windowed = TRUE;                                    // mode de finestra/pantalla completa

    // crea un dispositiu, context de dispositiu i cadena d'intercanvi utilitzant la informació a l'estructura scd
    D3D11CreateDeviceAndSwapChain(NULL,
                                  D3D_DRIVER_TYPE_HARDWARE,
                                  NULL,
                                  NULL,
                                  NULL,
                                  NULL,
                                  D3D11_SDK_VERSION,
                                  &scd,
                                  &swapchain,
                                  &dev,
                                  NULL,
                                  &devcon);
}

// aquesta és la funció que neteja Direct3D i COM
void CleanD3D(void)
{
    // tanca i allibera tots els objectes COM existents
    swapchain->Release();
    dev->Release();
    devcon->Release();
}

Explicació del codi

  • Capçaleres i biblioteques: Incloem les capçaleres necessàries i enllacem la biblioteca de Direct3D.
  • Declaracions globals: Declarem punters globals per a la cadena d'intercanvi, el dispositiu i el context del dispositiu.
  • Funció WinMain: Aquest és el punt d'entrada per a l'aplicació de Windows. Configura la finestra i entra al bucle principal.
  • Funció WindowProc: Aquesta gestiona els missatges enviats a la finestra.
  • Funció InitD3D: Aquesta inicialitza Direct3D, configurant la cadena d'intercanvi, el dispositiu i el context del dispositiu.
  • Funció CleanD3D: Aquesta allibera els recursos de Direct3D quan el programa acaba.

Conclusió

Configurar el teu entorn de desenvolupament per a la programació de DirectX implica instal·lar i configurar Visual Studio, configurar el DirectX SDK i escriure un programa bàsic de DirectX per assegurar-se que tot funciona correctament. Amb el teu entorn a punt, ara pots passar a temes més avançats i començar a desenvolupar aplicacions sofisticades amb DirectX.
