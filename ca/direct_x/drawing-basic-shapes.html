Introducció

En aquesta secció, explorarem com dibuixar formes bàsiques utilitzant DirectX. Això és fonamental per a qualsevol persona que vulgui crear aplicacions gràfiques o jocs. Començarem amb formes simples com línies i rectangles i gradualment passarem a formes més complexes.

Configuració de DirectX

Abans de començar a dibuixar formes, assegura't que tens DirectX configurat en el teu entorn de desenvolupament. Si encara no ho has fet, consulta la guia de configuració proporcionada a la introducció del curs.

Dibuixar una Línia

Dibuixar una línia és una de les tasques més senzilles en DirectX. Utilitzarem el mètode ID3D11DeviceContext::Draw per renderitzar una línia.

Conceptes Clau

  • Búfer de Vèrtexs: Emmagatzema els vèrtexs de la forma.
  • Disposició d'Entrada: Descriu el format del vèrtex a la GPU.
  • Shaders: Programes que s'executen a la GPU per processar vèrtexs i píxels.

Exemple de Codi

// Defineix els vèrtexs per a una línia
struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT4 color;
};

Vertex vertices[] = {
    { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(1.0f, 1.0f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) }
};

// Crea el búfer de vèrtexs
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Estableix el búfer de vèrtexs
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Dibuixa la línia
context->Draw(2, 0);

Explicació

  • Estructura de Vèrtex: Defineix la posició i el color de cada vèrtex.
  • Creació del Búfer de Vèrtexs: Assigna memòria a la GPU i copia les dades dels vèrtexs.
  • Establiment del Búfer de Vèrtexs: Enllaça el búfer de vèrtexs a l'etapa d'assemblador d'entrada.
  • Dibuix: Emet una crida de dibuix per renderitzar la línia.

Dibuixar un Rectangle

Dibuixar un rectangle implica definir quatre vèrtexs i dos triangles.

Conceptes Clau

  • Búfer d'Índexs: Especifica l'ordre en què els vèrtexs s'utilitzen per formar triangles.

Exemple de Codi

// Defineix els vèrtexs per a un rectangle
Vertex vertices[] = {
    { DirectX::XMFLOAT3(-0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(-0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 0.0f, 1.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 0.0f, 1.0f) }
};

// Defineix els índexs per al rectangle
unsigned short indices[] = {
    0, 1, 2,
    0, 2, 3
};

// Crea el búfer de vèrtexs
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Crea el búfer d'índexs
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(indices);
bufferDesc.BindFlags = D3D11_BIND_INDEX_BUFFER;

initData.pSysMem = indices;

ID3D11Buffer* indexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &indexBuffer);

// Estableix el búfer de vèrtexs
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Estableix el búfer d'índexs
context->IASetIndexBuffer(indexBuffer, DXGI_FORMAT_R16_UINT, 0);

// Dibuixa el rectangle
context->DrawIndexed(6, 0, 0);

Explicació

  • Vèrtexs i Índexs: Defineix la forma del rectangle i com connectar els vèrtexs.
  • Creació del Búfer d'Índexs: Assigna memòria a la GPU i copia les dades dels índexs.
  • Establiment dels Búfers: Enllaça els búfers de vèrtexs i índexs a l'etapa d'assemblador d'entrada.
  • Dibuix: Emèt una crida de dibuix per renderitzar el rectangle utilitzant els índexs.

Dibuixar un Cercle

Dibuixar un cercle és més complex i implica aproximar el cercle amb un polígon.

Conceptes Clau

  • Aproximació del Cercle: Utilitzar un polígon amb molts costats per aproximar un cercle.

Exemple de Codi

#include <vector>
#include <cmath>

// Defineix el nombre de segments
const int numSegments = 36;
const float radius = 1.0f;
std::vector<Vertex> vertices(numSegments + 1);

// Vèrtex central
vertices[0] = { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 1.0f, 1.0f) };

// Defineix els vèrtexs per al cercle
for (int i = 1; i <= numSegments; ++i) {
    float angle = i * (2.0f * DirectX::XM_PI / numSegments);
    vertices[i] = { DirectX::XMFLOAT3(radius * cosf(angle), radius * sinf(angle), 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) };
}

// Crea el búfer de vèrtexs
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(Vertex) * vertices.size();
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices.data();

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Estableix el búfer de vèrtexs
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Dibuixa el cercle
context->Draw(vertices.size(), 0);

Explicació

  • Vèrtexs del Cercle: Calcula els vèrtexs al voltant de la circumferència del cercle.
  • Creació del Búfer de Vèrtexs: Assigna memòria a la GPU i copia les dades dels vèrtexs.
  • Establiment del Búfer de Vèrtexs: Enllaça el búfer de vèrtexs a l'etapa d'assemblador d'entrada.
  • Dibuix: Emèt una crida de dibuix per renderitzar el cercle.

Conclusió

Dibuixar formes bàsiques en DirectX implica entendre els búfers de vèrtexs, els búfers d'índexs i com emetre crides de dibuix. En dominar aquests conceptes, pots crear formes més complexes i eventualment construir aplicacions gràfiques sofisticades. Practica experimentant amb diferents formes i configuracions per aprofundir en la teva comprensió.
