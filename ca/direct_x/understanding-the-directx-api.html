Introducció a DirectX

DirectX és una col·lecció d'APIs dissenyades per gestionar tasques relacionades amb multimèdia, especialment la programació de jocs i vídeo, en plataformes de Microsoft. Proporciona un conjunt d'eines per als desenvolupadors per crear aplicacions d'alt rendiment.

Conceptes Clau

  • API (Interfície de Programació d'Aplicacions): Un conjunt de funcions i procediments que permeten la creació d'aplicacions que accedeixen a característiques o dades d'un sistema operatiu, aplicació o altres serveis.
  • Direct3D: Una part de DirectX utilitzada per renderitzar gràfics tridimensionals en aplicacions.
  • DirectInput: Gestiona l'entrada de dispositius com teclats, ratolins i controladors de jocs.
  • DirectSound: Gestiona la reproducció i gravació de sons en forma d'ona.
  • DirectDraw: Utilitzat per renderitzar gràfics en aplicacions on el rendiment és crític.

Configuració de DirectX

Abans de submergir-te en la codificació, necessites configurar el teu entorn de desenvolupament.

Instal·lació

  1. Descarrega el DirectX SDK: Assegura't de tenir la versió més recent del Kit de Desenvolupament de Programari (SDK) de DirectX.
  2. Instal·la Visual Studio: El desenvolupament de DirectX es fa típicament utilitzant Microsoft Visual Studio.
  3. Configura el Projecte: Configura el teu projecte de Visual Studio per incloure les biblioteques i capçaleres de DirectX.

Exemple: Configuració d'un Projecte Bàsic

#include <windows.h>
#include <d3d11.h>

#pragma comment (lib, "d3d11.lib")

LRESULT CALLBACK WindowProc(HWND hwnd, UINT uMsg, WPARAM wParam, LPARAM lParam) {
    switch (uMsg) {
        case WM_DESTROY:
            PostQuitMessage(0);
            return 0;
    }
    return DefWindowProc(hwnd, uMsg, wParam, lParam);
}

int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow) {
    HWND hwnd;
    WNDCLASSEX wc;

    ZeroMemory(&wc, sizeof(WNDCLASSEX));
    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WindowProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = "WindowClass";

    RegisterClassEx(&wc);

    hwnd = CreateWindowEx(NULL, "WindowClass", "DirectX Window", WS_OVERLAPPEDWINDOW, 300, 300, 800, 600, NULL, NULL, hInstance, NULL);

    ShowWindow(hwnd, nCmdShow);

    MSG msg;
    while (GetMessage(&msg, NULL, 0, 0)) {
        TranslateMessage(&msg);
        DispatchMessage(&msg);
    }

    return msg.wParam;
}

Aquest codi configura una finestra bàsica utilitzant l'API Win32, que és un requisit previ per inicialitzar DirectX.

Conceptes Bàsics de Direct3D

Direct3D és la part més utilitzada de DirectX per renderitzar gràfics 3D.

Inicialització de Direct3D

  1. Crear un Dispositiu i una Cadena de Canvi: Aquests són essencials per a la renderització.
  2. Configurar Objectius de Renderització: Definir on es mostrarà la sortida de la renderització.
  3. Crear un Viewport: Definir l'àrea de la finestra on es produirà la renderització.

Exemple: Inicialització de Direct3D

IDXGISwapChain *swapchain;
ID3D11Device *dev;
ID3D11DeviceContext *devcon;

DXGI_SWAP_CHAIN_DESC scd;
ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));

scd.BufferCount = 1;
scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;
scd.OutputWindow = hwnd;
scd.SampleDesc.Count = 4;
scd.Windowed = TRUE;

D3D11CreateDeviceAndSwapChain(NULL, D3D_DRIVER_TYPE_HARDWARE, NULL, NULL, NULL, NULL, D3D11_SDK_VERSION, &scd, &swapchain, &dev, NULL, &devcon);

Aquest codi inicialitza Direct3D creant un dispositiu, un context de dispositiu i una cadena de canvi.

Tècniques Avançades de DirectX

Un cop et sentis còmode amb els conceptes bàsics, pots explorar temes més avançats.

Shaders

Els shaders són programes que s'executen a la GPU per controlar la pipeline de renderització.

  • Vertex Shaders: Processen cada vèrtex.
  • Pixel Shaders: Processen cada píxel.

Exemple: Codi Bàsic de Shader

// Vertex Shader
struct VS_OUTPUT {
    float4 Pos : SV_POSITION;
};

VS_OUTPUT VS(float4 inPos : POSITION) {
    VS_OUTPUT output;
    output.Pos = inPos;
    return output;
}

// Pixel Shader
float4 PS(VS_OUTPUT input) : SV_TARGET {
    return float4(1.0f, 0.0f, 0.0f, 1.0f); // Color vermell
}

Aquest codi de shader configura un shader bàsic de vèrtex i píxel que renderitza objectes en vermell.

Optimització del Rendiment

  • Renderització per Lots: Reduir el nombre de crides de dibuix.
  • Nivell de Detall (LOD): Utilitzar models més simples per a objectes distants.
  • Culling d'Oclusió: Evitar renderitzar objectes no visibles per la càmera.

Conclusió

Comprendre l'API DirectX implica entendre els seus diversos components, configurar l'entorn de desenvolupament i aprendre a renderitzar gràfics utilitzant Direct3D. A mesura que avancis, podràs explorar shaders i tècniques d'optimització del rendiment per crear aplicacions d'alt rendiment. Amb pràctica i experimentació, podràs aprofitar DirectX per desenvolupar aplicacions multimèdia i jocs sofisticats.
