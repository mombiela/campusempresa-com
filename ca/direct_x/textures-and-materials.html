Introducció

En aquesta secció, explorarem com utilitzar textures i materials en DirectX per millorar la qualitat visual de les teves aplicacions 3D. Les textures afegeixen detall a les superfícies dels teus models 3D, mentre que els materials defineixen com aquestes superfícies interactuen amb la llum.

Textures

Les textures són imatges aplicades a les superfícies dels models 3D per donar-los més detall i realisme.

Carregar Textures

Per carregar una textura en DirectX, normalment utilitzes el DirectX Tool Kit o la biblioteca DirectXTex. Aquí tens un exemple bàsic de com carregar una textura utilitzant el DirectX Tool Kit:

#include "DDSTextureLoader.h"

// Suposem que device és un punter vàlid a ID3D11Device
ID3D11ShaderResourceView* textureView = nullptr;
HRESULT hr = DirectX::CreateDDSTextureFromFile(device, L"texture.dds", nullptr, &textureView);
if (FAILED(hr)) {
    // Gestionar error
}

Aplicar Textures

Un cop carregada, has de vincular la textura a la pipeline. Això es fa establint la vista de recurs del shader a l'etapa del pixel shader.

// Suposem que context és un punter vàlid a ID3D11DeviceContext
context->PSSetShaderResources(0, 1, &textureView);

Coordenades de Textura

Les coordenades de textura (coordenades UV) mapegen la textura al model 3D. Van des de (0,0) fins a (1,1).

struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT2 texCoord; // Coordenades de textura
};

Materials

Els materials defineixen com les superfícies interactuen amb la llum, incloent propietats com el color, la brillantor i la reflectivitat.

Propietats Bàsiques del Material

  • Color Difús: El color base del material.
  • Color Especular: El color dels reflexos.
  • Brillantor: Controla la mida dels reflexos especulars.

Definir Materials

Els materials normalment es defineixen en una estructura i es passen als shaders.

struct Material {
    DirectX::XMFLOAT4 diffuseColor;
    DirectX::XMFLOAT4 specularColor;
    float shininess;
};

Utilitzar Materials en Shaders

En el teu shader HLSL, pots definir una estructura de material i utilitzar-la per calcular el color final d'un píxel.

cbuffer MaterialBuffer : register(b0) {
    float4 diffuseColor;
    float4 specularColor;
    float shininess;
};

float4 PSMain(VertexOut input) : SV_TARGET {
    // Calcular il·luminació aquí
    float4 color = diffuseColor; // Exemple simplificat
    return color;
}

Combinació de Textures i Materials

Per aconseguir un renderitzat realista, sovint combines textures i materials. Per exemple, podries utilitzar una textura per al color difús i un material per a les propietats especulars.

Exemple: Superfície Texturitzada i Brillant

Aquí tens un exemple de com combinar una textura amb propietats de material en un pixel shader:

Texture2D diffuseTexture : register(t0);
SamplerState samplerState : register(s0);

cbuffer MaterialBuffer : register(b0) {
    float4 specularColor;
    float shininess;
};

float4 PSMain(VertexOut input) : SV_TARGET {
    float4 diffuseColor = diffuseTexture.Sample(samplerState, input.texCoord);
    // Calcular il·luminació aquí
    float4 finalColor = diffuseColor; // Exemple simplificat
    return finalColor;
}

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les textures i materials en DirectX. Has après com carregar i aplicar textures, definir propietats de materials i combinar-los per crear superfícies realistes. Dominar aquests conceptes millorarà significativament la qualitat visual de les teves aplicacions 3D. Continua practicant experimentant amb diferents textures i propietats de materials per veure els seus efectes en els teus models.
