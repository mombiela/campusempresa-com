Introducció

El maneig de cadenes és un aspecte fonamental de la programació en C++. Aquest tema cobrirà els conceptes bàsics de les cadenes, diverses operacions que es poden realitzar amb cadenes i tècniques avançades de maneig de cadenes. Al final d'aquesta secció, tindràs una comprensió sòlida de com manipular i utilitzar cadenes en els teus programes en C++.

Conceptes Bàsics de les Cadenes

En C++, les cadenes es poden manejar utilitzant dos mètodes principals: cadenes d'estil C i la classe std::string de la Biblioteca Estàndard.

Cadenes d'Estil C

  • Les cadenes d'estil C són arrays de caràcters terminats per un caràcter nul ('\0').
  • Es defineixen utilitzant arrays de caràcters.
#include <iostream>

int main() {
    char greeting[] = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

Classe `std::string`

  • La classe std::string proporciona una manera més flexible i potent de manejar cadenes.
  • És part de la Biblioteca Estàndard de C++ i requereix la inclusió del header <string>.
#include <iostream>
#include <string>

int main() {
    std::string greeting = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

Operacions amb Cadenes

Concatenació

  • La concatenació és el procés d'unir dues cadenes.

Utilitzant Cadenes d'Estil C

#include <iostream>
#include <cstring>

int main() {
    char str1[20] = "Hello, ";
    char str2[] = "World!";
    strcat(str1, str2); // Concatena str2 a str1
    std::cout << str1 << std::endl;
    return 0;
}

Utilitzant std::string

#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello, ";
    std::string str2 = "World!";
    std::string result = str1 + str2; // Concatena str1 i str2
    std::cout << result << std::endl;
    return 0;
}

Longitud d'una Cadena

  • Determinar la longitud d'una cadena és una operació comuna.

Utilitzant Cadenes d'Estil C

#include <iostream>
#include <cstring>

int main() {
    char str[] = "Hello, World!";
    std::cout << "Longitud de la cadena: " << strlen(str) << std::endl;
    return 0;
}

Utilitzant std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::cout << "Longitud de la cadena: " << str.length() << std::endl;
    return 0;
}

Accés a Caràcters

  • Els caràcters individuals en una cadena es poden accedir utilitzant la indexació d'arrays.

Utilitzant Cadenes d'Estil C

#include <iostream>

int main() {
    char str[] = "Hello";
    std::cout << "Primer caràcter: " << str[0] << std::endl;
    return 0;
}

Utilitzant std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello";
    std::cout << "Primer caràcter: " << str[0] << std::endl;
    return 0;
}

Subcadenes

  • Extreure una subcadena d'una cadena és un requisit comú.

Utilitzant std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::string substr = str.substr(7, 5); // Extreu "World"
    std::cout << "Subcadena: " << substr << std::endl;
    return 0;
}

Maneig Avançat de Cadenes

Fluxos de Cadenes

  • Els fluxos de cadenes permeten realitzar operacions d'entrada i sortida en cadenes.
#include <iostream>
#include <sstream>
#include <string>

int main() {
    std::string str = "123 456 789";
    std::istringstream iss(str);
    int a, b, c;
    iss >> a >> b >> c;
    std::cout << "Nombres extrets: " << a << ", " << b << ", " << c << std::endl;
    return 0;
}

Expressions Regulars

  • Les expressions regulars proporcionen una manera potent de cercar i manipular cadenes.
#include <iostream>
#include <regex>
#include <string>

int main() {
    std::string str = "The quick brown fox jumps over the lazy dog.";
    std::regex pattern("(\\w+)"); // Coincideix amb paraules
    std::sregex_iterator begin(str.begin(), str.end(), pattern);
    std::sregex_iterator end;
    for (std::sregex_iterator i = begin; i != end; ++i) {
        std::smatch match = *i;
        std::cout << match.str() << std::endl;
    }
    return 0;
}

Conclusió

El maneig de cadenes en C++ abasta des d'operacions bàsiques amb cadenes d'estil C fins a manipulacions més avançades utilitzant la classe std::string i altres utilitats com fluxos de cadenes i expressions regulars. Entendre aquests conceptes és crucial per a un processament i manipulació de text efectius en els teus programes. Practica aquestes tècniques per convertir-te en un expert en el maneig de cadenes en C++.
