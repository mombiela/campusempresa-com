La depuració és una habilitat crucial per a qualsevol programador. En aquesta secció, explorarem diverses tècniques de depuració en C++ que van des de nivells bàsics fins a avançats. Al final d'aquest tema, estaràs equipat amb les eines i els coneixements per identificar i solucionar errors de manera efectiva en els teus programes en C++.

Introducció a la Depuració

  • Definició: La depuració és el procés d'identificar, analitzar i eliminar errors o bugs d'un programa.
  • Importància: Assegura que el programa s'executi correctament i eficientment.

Tècniques Bàsiques de Depuració

Utilitzar Sentències de Print

Una de les tècniques de depuració més simples i comunes és utilitzar sentències de print per seguir el flux d'execució i els valors de les variables.

#include <iostream>

int main() {
    int a = 5;
    int b = 10;
    int sum = a + b;

    std::cout << "a: " << a << ", b: " << b << ", sum: " << sum << std::endl;

    return 0;
}
  • Explicació: Imprimint els valors de a, b i sum, pots verificar que les variables continguin els valors esperats.

Utilitzar Assertions

Les assertions s'utilitzen per verificar les suposicions fetes pel programa i poden ajudar a detectar errors de manera precoç.

#include <cassert>

int main() {
    int a = 5;
    int b = 10;
    int sum = a + b;

    assert(sum == 15);

    return 0;
}
  • Explicació: La sentència assert comprova si sum és igual a 15. Si no ho és, el programa es tancarà i proporcionarà un missatge d'error.

Tècniques Intermèdies de Depuració

Utilitzar un Depurador

Un depurador és una eina que et permet executar el teu programa pas a pas, inspeccionar variables i controlar el flux d'execució.

  • Depuradors Comuns: GDB (GNU Debugger), LLDB, Visual Studio Debugger.
  • Funcions Clau:
    • Breakpoints: Pausar l'execució en una línia específica.
    • Step Over: Executar la següent línia de codi.
    • Step Into: Entrar en funcions per veure la seva execució.
    • Watch Variables: Monitoritzar els valors de les variables.

Exemple amb GDB

#include <iostream>

int main() {
    int a = 5;
    int b = 10;
    int sum = a + b;

    std::cout << "Sum: " << sum << std::endl;

    return 0;
}
  • Passos:
    1. Compilar amb informació de depuració: g++ -g -o program program.cpp
    2. Iniciar GDB: gdb ./program
    3. Posar un breakpoint: break main
    4. Executar el programa: run
    5. Passar pel codi: next o step
    6. Inspeccionar variables: print a, print b, print sum

Utilitzar Logging

El logging implica escriure missatges a un fitxer de registre o consola per seguir l'execució del programa.

#include <iostream>
#include <fstream>

int main() {
    std::ofstream log("log.txt");

    int a = 5;
    int b = 10;
    int sum = a + b;

    log << "a: " << a << ", b: " << b << ", sum: " << sum << std::endl;

    log.close();

    return 0;
}
  • Explicació: Els registres es poden revisar més tard per entendre el comportament del programa i identificar problemes.

Tècniques Avançades de Depuració

Depuració de Memòria

Els errors relacionats amb la memòria, com les fuites i la corrupció, poden ser difícils d'identificar. Eines com Valgrind poden ajudar.

Exemple amb Valgrind

#include <iostream>

int main() {
    int* arr = new int[10];
    arr[0] = 1;

    delete[] arr;

    return 0;
}
  • Passos:
    1. Compilar el programa: g++ -o program program.cpp
    2. Executar amb Valgrind: valgrind ./program
    3. Analitzar la sortida per trobar fuites de memòria i errors.

Utilitzar Eines d'Anàlisi Estàtica

Les eines d'anàlisi estàtica analitzen el teu codi sense executar-lo per trobar possibles errors.

  • Eines Comuns: Clang Static Analyzer, Cppcheck.
  • Exemple:
    • Executar Cppcheck: cppcheck program.cpp
    • Revisar la sortida per advertències i errors.

Conclusió

La depuració és una habilitat essencial per a qualsevol programador de C++. Dominant diverses tècniques de depuració, des de simples sentències de print fins a eines avançades com Valgrind i analitzadors estàtics, podràs identificar i resoldre problemes de manera eficient en el teu codi. Practica aquestes tècniques regularment per convertir-te en un expert en depuració i assegurar-te que els teus programes s'executin de manera fluida i correcta.
