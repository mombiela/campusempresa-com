Introducció

En C++, les variables i els tipus de dades són conceptes fonamentals que formen els blocs de construcció de qualsevol programa. Entendre com declarar, inicialitzar i utilitzar variables, així com conèixer els diferents tipus de dades disponibles, és crucial per escriure codi eficient i efectiu.

Variables

Les variables són ubicacions d'emmagatzematge amb nom a la memòria que contenen dades que es poden modificar durant l'execució del programa.

Declaració de Variables

Per declarar una variable en C++, cal especificar el tipus de dada seguit del nom de la variable.

int edat;
double salari;
char nota;

Inicialització de Variables

Les variables es poden inicialitzar en el moment de la declaració.

int edat = 25;
double salari = 50000.50;
char nota = 'A';

Regles per Nomenar Variables

  • Han de començar amb una lletra (a-z, A-Z) o un guió baix (_).
  • Poden contenir lletres, dígits (0-9) i guions baixos.
  • Sensibles a majúscules i minúscules (per exemple, edat i Edat són variables diferents).
  • No poden ser una paraula reservada (per exemple, int, double).

Tipus de Dades

Els tipus de dades especifiquen el tipus de dades que una variable pot contenir. C++ proporciona diversos tipus de dades integrats.

Tipus de Dades Bàsics

  • int: Tipus enter.
  • float: Tipus de punt flotant.
  • double: Tipus de punt flotant de doble precisió.
  • char: Tipus caràcter.
  • bool: Tipus booleà (true o false).

Exemple de Tipus de Dades Bàsics

int edat = 30;
float alçada = 5.9f;
double pes = 70.5;
char inicial = 'J';
bool ésEstudiant = true;

Tipus de Dades Derivats

  • Arrays: Col·lecció d'elements del mateix tipus.
  • Punteres: Variables que emmagatzemen adreces de memòria.
  • Referències: Àlies per a una altra variable.

Exemple de Tipus de Dades Derivats

int nombres[5] = {1, 2, 3, 4, 5}; // Array
int* ptr = &edat; // Puntera
int& ref = edat; // Referència

Tipus de Dades Definits per l'Usuari

  • struct: Grup de variables de diferents tipus.
  • union: Similar a struct però amb memòria compartida.
  • enum: Tipus d'enumeració.

Exemple de Tipus de Dades Definits per l'Usuari

struct Persona {
    int edat;
    double salari;
    char nota;
};

union Dades {
    int valorInt;
    float valorFloat;
};

enum Color { VERMELL, VERD, BLAU };

Modificadors de Tipus

Els modificadors de tipus alteren la mida i el rang dels tipus de dades.

  • signed: Per defecte per als tipus enters.
  • unsigned: Només valors positius.
  • short: Rang escurçat.
  • long: Rang estès.

Exemple de Modificadors de Tipus

unsigned int nombrePositiu = 100;
long int nombreGran = 1000000;
short int nombrePetit = 10;

Conversió de Tipus

La conversió de tipus és el procés de convertir un tipus de dada en un altre. Pot ser implícita o explícita.

Conversió Implícita

Realitzada automàticament pel compilador.

int num = 10;
double resultat = num; // Conversió implícita de int a double

Conversió Explícita (Type Casting)

Realitzada manualment pel programador.

double pi = 3.14;
int intPi = (int)pi; // Conversió explícita de double a int

Conclusió

Entendre les variables i els tipus de dades és essencial per a qualsevol programador de C++. Les variables emmagatzemen dades, i els tipus de dades defineixen el tipus de dades que es poden emmagatzemar. En dominar aquests conceptes, pots escriure programes en C++ més eficients i efectius. Practica declarant, inicialitzant i utilitzant diferents tipus de variables per consolidar la teva comprensió.
