Introducció a les Referències

Les referències en C++ són un concepte essencial que et permet crear un àlies per a una altra variable. Sovint s'utilitzen per passar variables a funcions sense copiar-les, cosa que pot ser més eficient i permet que la funció modifiqui la variable original.

  • Definició: Una referència és un àlies per a una altra variable.
  • Sintaxi: Per declarar una referència, utilitza el símbol &.

Exemple:

int main() {
    int a = 10;
    int &ref = a; // ref és una referència a a
    ref = 20; // Això canvia el valor de a a 20
    std::cout << "a = " << a << std::endl; // Sortida: a = 20
    return 0;
}

Conceptes Clau

1. Declaració i Inicialització

  • Declaració: Una referència ha de ser inicialitzada quan es declara.
  • Inicialització: La referència ha de ser inicialitzada a una variable del mateix tipus.
int x = 5;
int &ref = x; // Correcte
int &ref2; // Error: la referència ha de ser inicialitzada

2. Referències vs Punteres

  • Referències: No poden ser nul·les, han de ser inicialitzades i no poden ser canviades per referenciar una altra variable.
  • Punteres: Poden ser nul·les, poden ser reassignades i necessiten ser desreferenciades per accedir al valor.

| Característica | Referència | Puntera | |------------------|-----------------------|------------------------| | Nullabilitat | No pot ser nul·la | Pot ser nul·la | | Inicialització | Ha de ser inicialitzada| Pot ser no inicialitzada| | Reassignació | No pot ser reassignada| Pot ser reassignada | | Sintaxi | int &ref = var; | int *ptr = &var; |

3. Passar per Referència

  • Paràmetres de Funció: Passar per referència permet que la funció modifiqui la variable original.
void increment(int &num) {
    num++;
}

int main() {
    int a = 5;
    increment(a);
    std::cout << "a = " << a << std::endl; // Sortida: a = 6
    return 0;
}

4. Referències Const

  • Referències de Només Lectura: Utilitza const per crear una referència que no pot modificar la variable original.
void printValue(const int &num) {
    std::cout << "Valor: " << num << std::endl;
}

int main() {
    int a = 10;
    printValue(a); // Sortida: Valor: 10
    return 0;
}

5. Referències a Punteres

  • Combinació de Referències i Punteres: Pots crear una referència a una puntera.
int main() {
    int a = 10;
    int *ptr = &a;
    int *&refPtr = ptr; // refPtr és una referència a la puntera ptr
    *refPtr = 20; // Això canvia el valor de a a 20
    std::cout << "a = " << a << std::endl; // Sortida: a = 20
    return 0;
}

Temes Avançats

1. Retornar Referències des de Funcions

  • Retorn per Referència: Les funcions poden retornar referències, permetent que el cridant modifiqui el valor retornat.
int& getElement(int arr[], int index) {
    return arr[index];
}

int main() {
    int arr[3] = {1, 2, 3};
    getElement(arr, 1) = 10; // Canvia arr[1] a 10
    std::cout << "arr[1] = " << arr[1] << std::endl; // Sortida: arr[1] = 10
    return 0;
}

2. Membres Referència en Classes

  • Membres de Classe: Les referències poden ser utilitzades com a membres de classe, però han de ser inicialitzades en el constructor.
class MyClass {
    int &ref;
public:
    MyClass(int &r) : ref(r) {}
    void setValue(int val) {
        ref = val;
    }
};

int main() {
    int a = 5;
    MyClass obj(a);
    obj.setValue(10);
    std::cout << "a = " << a << std::endl; // Sortida: a = 10
    return 0;
}

Conclusió

Les referències en C++ proporcionen una manera poderosa de crear àlies per a variables, permetent una passada de paràmetres de funció eficient i la manipulació de variables. Entendre les diferències entre referències i punteres, així com l'ús avançat com retornar referències des de funcions i utilitzar referències en classes, és crucial per escriure codi C++ eficient i efectiu. Practica utilitzant referències en diverses situacions per familiaritzar-te amb el seu comportament i beneficis.
