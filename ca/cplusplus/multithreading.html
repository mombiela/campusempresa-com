El multithreading és una característica poderosa en C++ que permet a un programa realitzar múltiples tasques de manera concurrent. Aquest tema et guiarà des dels conceptes bàsics del multithreading fins a conceptes avançats, proporcionant explicacions clares, exemples i exercicis.

Introducció al Multithreading

El multithreading implica executar múltiples fils simultàniament dins d'un únic programa. Cada fil s'executa de manera independent, permetent l'execució paral·lela de tasques.

Conceptes Clau

  • Fil: La unitat més petita d'un procés que pot ser programada per a l'execució.
  • Concurrència: La capacitat d'executar múltiples fils en períodes superposats.
  • Paral·lelisme: L'execució simultània de múltiples fils.

Creació de Fils

En C++, la biblioteca <thread> proporciona les eines necessàries per crear i gestionar fils.

Creació Bàsica de Fils

Per crear un fil, pots utilitzar la classe std::thread.

#include <iostream>
#include <thread>

void printMessage() {
    std::cout << "Hola des del fil!" << std::endl;
}

int main() {
    std::thread t(printMessage); // Crear un nou fil que executa printMessage
    t.join(); // Esperar que el fil acabi
    return 0;
}

Explicació

  • std::thread t(printMessage): Crea un nou fil que executa la funció printMessage.
  • t.join(): Espera que el fil t acabi l'execució abans de continuar.

Passar Arguments als Fils

Pots passar arguments a la funció del fil utilitzant el constructor de std::thread.

#include <iostream>
#include <thread>

void printNumber(int n) {
    std::cout << "Número: " << n << std::endl;
}

int main() {
    std::thread t(printNumber, 42); // Passar 42 com a argument a printNumber
    t.join();
    return 0;
}

Explicació

  • std::thread t(printNumber, 42): Crea un nou fil que executa printNumber amb 42 com a argument.

Gestió de Fils

Desenganxar Fils

Un fil desenganxat s'executa de manera independent del fil principal.

#include <iostream>
#include <thread>

void printMessage() {
    std::cout << "Hola des del fil desenganxat!" << std::endl;
}

int main() {
    std::thread t(printMessage);
    t.detach(); // Desenganxar el fil
    std::this_thread::sleep_for(std::chrono::seconds(1)); // Assegurar que el fil principal espera
    return 0;
}

Explicació

  • t.detach(): El fil t s'executa de manera independent del fil principal.
  • std::this_thread::sleep_for: Assegura que el fil principal espera prou temps perquè el fil desenganxat s'executi.

Sincronització

Quan múltiples fils accedeixen a recursos compartits, la sincronització és necessària per evitar conflictes.

Mutex

Un std::mutex s'utilitza per protegir dades compartides.

#include <iostream>
#include <thread>
#include <mutex>

std::mutex mtx;

void printMessage(const std::string& msg) {
    std::lock_guard<std::mutex> lock(mtx);
    std::cout << msg << std::endl;
}

int main() {
    std::thread t1(printMessage, "Hola des del fil 1");
    std::thread t2(printMessage, "Hola des del fil 2");
    t1.join();
    t2.join();
    return 0;
}

Explicació

  • std::mutex mtx: Declara un mutex.
  • std::lock_guardstd::mutex lock(mtx): Bloqueja el mutex durant la durada de la funció printMessage.

Temes Avançats

Variables de Condició

Les variables de condició permeten que els fils esperin fins que es compleixin certes condicions.

#include <iostream>
#include <thread>
#include <mutex>
#include <condition_variable>

std::mutex mtx;
std::condition_variable cv;
bool ready = false;

void printMessage() {
    std::unique_lock<std::mutex> lock(mtx);
    cv.wait(lock, []{ return ready; });
    std::cout << "El fil està en execució!" << std::endl;
}

int main() {
    std::thread t(printMessage);
    {
        std::lock_guard<std::mutex> lock(mtx);
        ready = true;
    }
    cv.notify_one();
    t.join();
    return 0;
}

Explicació

  • std::condition_variable cv: Declara una variable de condició.
  • cv.wait(lock, []{ return ready; }): Espera fins que ready sigui true.
  • cv.notify_one(): Notifica un fil en espera.

Conclusió

El multithreading en C++ permet l'execució concurrent de tasques, millorant el rendiment i la capacitat de resposta. Entenent la creació, gestió i sincronització de fils, pots utilitzar efectivament el multithreading en les teves aplicacions. Practica amb els exemples i exercicis proporcionats per aprofundir en la teva comprensió.
