L'assignació dinàmica de memòria és un concepte crucial en C++ que permet als programes sol·licitar memòria en temps d'execució. Això és essencial per crear aplicacions flexibles i eficients que puguin gestionar quantitats variables de dades.

Introducció a l'Assignació Dinàmica de Memòria

L'assignació dinàmica de memòria permet als programes assignar memòria segons sigui necessari durant l'execució, en lloc de fer-ho en temps de compilació. Això és particularment útil per gestionar estructures de dades la mida de les quals no es pot determinar prèviament.

Conceptes Clau

  • Heap vs. Stack:
    • Stack: Memòria assignada en temps de compilació, gestionada automàticament i limitada en mida.
    • Heap: Memòria assignada en temps d'execució, gestionada manualment i generalment més gran que la stack.
  • Punteres: Variables que emmagatzemen adreces de memòria, essencials per a l'assignació dinàmica de memòria.
  • Operadors: new i delete per a objectes individuals, new[] i delete[] per a arrays.

Assignació Dinàmica de Memòria Bàsica

Assignació i Desassignació d'Objectes Individuals

Per assignar memòria per a un objecte individual, utilitza l'operador new. Per desassignar, utilitza l'operador delete.

int* ptr = new int; // Assignar memòria per a un enter
*ptr = 10;         // Assignar un valor a la memòria assignada
delete ptr;        // Desassignar la memòria

Assignació i Desassignació d'Arrays

Per a arrays, utilitza new[] i delete[].

int* arr = new int[5]; // Assignar memòria per a un array de 5 enters
for (int i = 0; i < 5; ++i) {
    arr[i] = i * 10;  // Assignar valors a l'array
}
delete[] arr;         // Desassignar la memòria

Assignació Dinàmica de Memòria Avançada

Eliminadors Personalitzats

Quan s'utilitzen punteres intel·ligents, es poden especificar eliminadors personalitzats per gestionar la neteja de recursos.

#include <memory>
#include <iostream>

void customDeleter(int* ptr) {
    std::cout << "Eliminador personalitzat cridat\n";
    delete ptr;
}

int main() {
    std::unique_ptr<int, void(*)(int*)> ptr(new int, customDeleter);
    *ptr = 20;
    return 0;
}

Fuites de Memòria i Gestió

Una gestió adequada de la memòria és crucial per evitar fuites de memòria, on la memòria assignada no es desassigna correctament.

  • Exemple de Fuita de Memòria:
void memoryLeak() {
    int* ptr = new int(10);
    // Oblidat desassignar ptr, causant una fuita de memòria
}
  • Evitant Fuites de Memòria:
void noMemoryLeak() {
    int* ptr = new int(10);
    delete ptr; // Desassignar correctament la memòria
}

Punteres Intel·ligents

Les punteres intel·ligents (std::unique_ptr, std::shared_ptr, std::weak_ptr) ajuden a gestionar la memòria dinàmica automàticament.

  • Puntera Única:
#include <memory>

std::unique_ptr<int> uniquePtr(new int(10));
  • Puntera Compartida:
#include <memory>

std::shared_ptr<int> sharedPtr1 = std::make_shared<int>(10);
std::shared_ptr<int> sharedPtr2 = sharedPtr1; // Propietat compartida

Resum

L'assignació dinàmica de memòria en C++ permet una gestió flexible i eficient de la memòria en temps d'execució. Els conceptes clau inclouen l'ús de l'heap, les punteres i els operadors new i delete. Les tècniques avançades impliquen eliminadors personalitzats i punteres intel·ligents per gestionar la memòria automàticament i evitar fuites. Una comprensió i aplicació adequades d'aquests conceptes són essencials per desenvolupar aplicacions robustes en C++.
