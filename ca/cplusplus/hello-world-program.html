Introducció

El programa "Hello World" és un programa senzill però fonamental que serveix com a introducció a qualsevol llenguatge de programació. En C++, ajuda els principiants a entendre l'estructura bàsica d'un programa en C++, incloent-hi la sintaxi, la compilació i l'execució.

Conceptes Clau

  • Sintaxi Bàsica: Entendre l'estructura d'un programa en C++.
  • Fitxers de Capçalera: Introducció a la directiva #include.
  • Funció Principal: El punt d'entrada d'un programa en C++.
  • Sortida: Utilitzar std::cout per imprimir text a la consola.
  • Compilació i Execució: Passos per compilar i executar un programa en C++.

Escriure el Teu Primer Programa en C++

Estructura Bàsica d'un Programa en C++

Un programa en C++ típicament consta dels següents components:

  1. Directives del Preprocessador: Instruccions per al compilador per incloure biblioteques.
  2. Funció Principal: El punt d'inici del programa.
  3. Instruccions: Instruccions que el programa executa.

Exemple: Programa Hello World

Aquí tens un senzill programa "Hello World" en C++:

#include <iostream>  // Directiva del preprocessador per incloure la biblioteca iostream

int main() {  // Funció principal on comença l'execució del programa
    std::cout << "Hello, World!" << std::endl;  // Instrucció de sortida
    return 0;  // Instrucció de retorn que indica una execució exitosa
}

Explicació del Codi

  • #include <iostream>: Aquesta línia diu al compilador que inclogui la biblioteca de flux d'entrada-sortida estàndard, que és necessària per utilitzar std::cout.
  • int main() { ... }: Això defineix la funció principal, que és el punt d'entrada del programa. L'execució del programa comença en aquesta funció.
  • std::cout << "Hello, World!" << std::endl;: Aquesta línia imprimeix el text "Hello, World!" a la consola. std::cout s'utilitza per a la sortida, i std::endl s'utilitza per inserir un caràcter de nova línia.
  • return 0;: Aquesta instrucció finalitza la funció principal i retorna 0 al sistema operatiu, indicant que el programa s'ha executat correctament.

Compilació i Execució

Per executar el programa "Hello World", segueix aquests passos:

  1. Escriu el Codi: Desa el codi en un fitxer amb una extensió .cpp, per exemple, hello_world.cpp.
  2. Compila el Codi: Utilitza un compilador de C++ per compilar el codi. Per exemple, utilitzant g++: 
    g++ hello_world.cpp -o hello_world
    Aquesta comanda compila hello_world.cpp i genera un executable anomenat hello_world.
  3. Executa l'Executable: Executa el programa compilat: 
    ./hello_world
    Hauries de veure la sortida: 
    Hello, World!

Resum

El programa "Hello World" és un punt de partida senzill però essencial per aprendre C++. Introduïx l'estructura bàsica d'un programa en C++, incloent-hi directives del preprocessador, la funció principal i instruccions de sortida. Entendre com escriure, compilar i executar aquest programa posa les bases per a conceptes més avançats de programació en C++.
