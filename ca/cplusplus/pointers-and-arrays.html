Introducció

Els punteros i els arrays són conceptes fonamentals en C++ que permeten una gestió eficient de la memòria i la manipulació de dades. Entendre com utilitzar els punteros i els arrays de manera efectiva és crucial per escriure programes C++ optimitzats i potents.

Punteros

Els punteros són variables que emmagatzemen l'adreça de memòria d'una altra variable. Són eines poderoses que permeten l'accés directe a la memòria i la seva manipulació.

Conceptes Clau

  • Declaració de Puntero: Declarar un puntero implica especificar el tipus de dades al qual apunta, seguit d'un asterisc (*) i el nom del puntero.
  • Inicialització de Puntero: Els punteros es poden inicialitzar amb l'adreça d'una variable utilitzant l'operador d'adreça (&).
  • Desreferenciació: Accedir al valor emmagatzemat a l'adreça de memòria a la qual apunta un puntero utilitzant l'operador de desreferenciació (*).

Exemple de Codi

#include <iostream>
using namespace std;

int main() {
    int var = 10;
    int *ptr = &var; // Inicialització de puntero

    cout << "Valor de var: " << var << endl;
    cout << "Adreça de var: " << &var << endl;
    cout << "Valor emmagatzemat a ptr: " << *ptr << endl; // Desreferenciació

    return 0;
}

Explicació:

  • int *ptr = &var; inicialitza el puntero ptr amb l'adreça de var.
  • *ptr desreferencia el puntero per accedir al valor emmagatzemat a l'adreça.

Arrays

Els arrays són col·leccions d'elements del mateix tipus emmagatzemats en ubicacions de memòria contigües. Proporcionen una manera d'emmagatzemar múltiples valors en una sola variable.

Conceptes Clau

  • Declaració d'Array: Declarar un array implica especificar el tipus d'elements, el nom de l'array i el nombre d'elements.
  • Inicialització d'Array: Els arrays es poden inicialitzar en el moment de la declaració.
  • Accés als Elements: Els elements d'un array es poden accedir utilitzant l'operador d'índex ([]).

Exemple de Codi

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5}; // Inicialització d'array

    for (int i = 0; i < 5; i++) {
        cout << "Element a l'índex " << i << ": " << arr[i] << endl;
    }

    return 0;
}

Explicació:

  • int arr[5] = {1, 2, 3, 4, 5}; declara i inicialitza un array de 5 enters.
  • arr[i] accedeix a l'element a l'índex i.

Punteros i Arrays

Els punteros i els arrays estan estretament relacionats en C++. El nom d'un array actua com un puntero constant al primer element de l'array.

Conceptes Clau

  • Nom d'Array com a Puntero: El nom d'un array és un puntero al seu primer element.
  • Aritmètica de Punteros: Els punteros es poden incrementar o decrementar per recórrer un array.

Exemple de Codi

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *ptr = arr; // Nom d'array com a puntero

    for (int i = 0; i < 5; i++) {
        cout << "Element a l'índex " << i << ": " << *(ptr + i) << endl; // Aritmètica de punteros
    }

    return 0;
}

Explicació:

  • int *ptr = arr; inicialitza el puntero ptr al primer element de l'array arr.
  • *(ptr + i) utilitza l'aritmètica de punteros per accedir als elements de l'array.

Temes Avançats

Arrays Dinàmics

Els arrays dinàmics permeten una assignació flexible de memòria durant el temps d'execució utilitzant punteros i la paraula clau new.

Exemple de Codi

#include <iostream>
using namespace std;

int main() {
    int size;
    cout << "Introdueix la mida de l'array: ";
    cin >> size;

    int *arr = new int[size]; // Assignació dinàmica d'array

    for (int i = 0; i < size; i++) {
        arr[i] = i * 10; // Inicialització dels elements de l'array
    }

    for (int i = 0; i < size; i++) {
        cout << "Element a l'índex " << i << ": " << arr[i] << endl;
    }

    delete[] arr; // Desassignació de memòria

    return 0;
}

Explicació:

  • int *arr = new int[size]; assigna dinàmicament un array d'enters.
  • delete[] arr; desassigna la memòria per evitar fuites de memòria.

Conclusió

Els punteros i els arrays són eines essencials en C++ per a una gestió eficient de la memòria i la manipulació de dades. Entendre la seva relació i com utilitzar-los de manera efectiva pot millorar significativament les teves habilitats de programació. Practica utilitzant punteros i arrays en diverses situacions per convertir-te en un expert en el seu ús.
