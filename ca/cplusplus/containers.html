Introducció als Contenidors

Els contenidors són components fonamentals en C++ que et permeten emmagatzemar i gestionar col·leccions de dades. Formen part de la Biblioteca de Plantilles Estàndard (STL) i proporcionen diverses maneres d'organitzar i manipular dades de manera eficient.

Conceptes Clau

  • Contenidors de Seqüència: Emmagatzemen dades de manera lineal.
  • Contenidors Associatius: Emmagatzemen dades de manera que permeten una recuperació ràpida mitjançant claus.
  • Adaptadors de Contenidors: Proporcionen una interfície diferent per als contenidors seqüencials.

Contenidors de Seqüència

Els contenidors de seqüència mantenen l'ordre dels elements.

Vector

Una matriu dinàmica que pot redimensionar-se automàticament quan s'afegeix o s'elimina un element.

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Afegeix un element al final

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Llista

Una llista doblement enllaçada que permet una inserció i eliminació eficient d'elements.

#include <iostream>
#include <list>

int main() {
    std::list<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Afegeix un element al final

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Deque

Una cua doblement acabada que permet la inserció i eliminació a ambdós extrems.

#include <iostream>
#include <deque>

int main() {
    std::deque<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_front(0); // Afegeix un element al davant

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Contenidors Associatius

Els contenidors associatius emmagatzemen elements formats per una combinació d'un valor clau i un valor mapat, i permeten una recuperació ràpida dels elements basats en les claus.

Mapa

Una col·lecció de parelles clau-valor amb claus úniques.

#include <iostream>
#include <map>

int main() {
    std::map<int, std::string> students;
    students[1] = "Alice";
    students[2] = "Bob";

    for (const auto& pair : students) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
    return 0;
}

Conjunt

Una col·lecció de claus úniques.

#include <iostream>
#include <set>

int main() {
    std::set<int> numbers = {1, 2, 3, 4, 5};
    numbers.insert(6); // Afegeix un element

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Adaptadors de Contenidors

Els adaptadors de contenidors proporcionen una interfície diferent per als contenidors seqüencials.

Pila

Una estructura de dades LIFO (Last In, First Out).

#include <iostream>
#include <stack>

int main() {
    std::stack<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Cua

Una estructura de dades FIFO (First In, First Out).

#include <iostream>
#include <queue>

int main() {
    std::queue<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.front() << " ";
        numbers.pop();
    }
    return 0;
}

Cua de Prioritat

Una estructura de dades on els elements es processen segons la prioritat.

#include <iostream>
#include <queue>

int main() {
    std::priority_queue<int> numbers;
    numbers.push(3);
    numbers.push(1);
    numbers.push(2);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Resum

En aquesta secció, hem cobert els diferents tipus de contenidors disponibles en C++:

  • Contenidors de Seqüència: Vector, Llista, Deque.
  • Contenidors Associatius: Mapa, Conjunt.
  • Adaptadors de Contenidors: Pila, Cua, Cua de Prioritat.

Entendre aquests contenidors i els seus casos d'ús apropiats és crucial per a una gestió i manipulació eficient de dades en C++. Cada tipus de contenidor ofereix avantatges únics i és adequat per a escenaris específics, convertint-los en eines poderoses en la caixa d'eines d'un programador de C++.
