Pointers to Pointers in C++
Referències ►

Introducció

Els punters són un concepte fonamental en C++ que et permeten manipular directament la memòria. Un punter a un punter és una forma de múltiples indirections o una cadena de punters. Aquest concepte és crucial per entendre estructures de dades complexes i l'assignació dinàmica de memòria.

Conceptes Clau

  • Conceptes Bàsics de Punter: Un punter és una variable que emmagatzema l'adreça de memòria d'una altra variable.
  • Punter a Punter: Un punter a un punter és una variable que emmagatzema l'adreça d'un altre punter.

Entenent els Punters a Punters

Recapitulació Bàsica de Punter

Abans d'endinsar-nos en els punters a punters, fem una ràpida recapitulació dels punters bàsics.

int var = 10;
int *ptr = &var; // ptr conté l'adreça de var

En aquest exemple:

  • var és una variable sencera.
  • ptr és un punter a un enter, que conté l'adreça de var.

Punter a Punter

Un punter a un punter emmagatzema l'adreça d'un altre punter.

int var = 10;
int *ptr = &var; // ptr conté l'adreça de var
int **pptr = &ptr; // pptr conté l'adreça de ptr

En aquest exemple:

  • var és una variable sencera.
  • ptr és un punter a un enter, que conté l'adreça de var.
  • pptr és un punter a un punter, que conté l'adreça de ptr.

Representació Visual

Per entendre-ho millor, visualitzem les adreces de memòria:

| Variable | Valor | Adreça | |----------|-------|---------| | var | 10 | 0x100 | | ptr | 0x100 | 0x200 | | pptr | 0x200 | 0x300 |

Accedint als Valors

Per accedir al valor de var utilitzant pptr:

#include <iostream>
using namespace std;

int main() {
    int var = 10;
    int *ptr = &var;
    int **pptr = &ptr;

    cout << "Valor de var: " << var << endl;
    cout << "Valor utilitzant ptr: " << *ptr << endl;
    cout << "Valor utilitzant pptr: " << **pptr << endl;

    return 0;
}

Sortida:

Valor de var: 10
Valor utilitzant ptr: 10
Valor utilitzant pptr: 10

Aplicacions Pràctiques

Assignació Dinàmica de Memòria

Els punters a punters sovint s'utilitzen en l'assignació dinàmica de memòria, especialment per a arrays multidimensionals.

#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 4;
    int **arr = new int*[rows];

    for(int i = 0; i < rows; ++i)
        arr[i] = new int[cols];

    // Assignar valors
    for(int i = 0; i < rows; ++i)
        for(int j = 0; j < cols; ++j)
            arr[i][j] = i + j;

    // Imprimir valors
    for(int i = 0; i < rows; ++i) {
        for(int j = 0; j < cols; ++j)
            cout << arr[i][j] << " ";
        cout << endl;
    }

    // Desallotjar memòria
    for(int i = 0; i < rows; ++i)
        delete[] arr[i];
    delete[] arr;

    return 0;
}

Sortida:

0 1 2 3 
1 2 3 4 
2 3 4 5

Punters a Funcions

Els punters a punters també es poden utilitzar amb punters a funcions per crear estructures de codi més complexes i flexibles.

#include <iostream>
using namespace std;

void func(int **pptr) {
    **pptr = 20;
}

int main() {
    int var = 10;
    int *ptr = &var;
    int **pptr = &ptr;

    cout << "Abans de la crida a func: " << var << endl;
    func(pptr);
    cout << "Després de la crida a func: " << var << endl;

    return 0;
}

Sortida:

Abans de la crida a func: 10
Després de la crida a func: 20

Conclusió

Els punters a punters són una característica poderosa en C++ que permeten múltiples nivells d'indirecció. Són essencials per a l'assignació dinàmica de memòria, la gestió d'arrays multidimensionals i la creació d'estructures de dades complexes. Entendre aquest concepte millorarà significativament la teva capacitat per escriure programes en C++ eficients i flexibles.
