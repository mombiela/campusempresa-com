Introducció als Algorismes

Els algorismes són procediments o fórmules pas a pas per resoldre problemes. En programació, els algorismes són essencials per realitzar tasques de manera eficient i efectiva.

Conceptes Clau

  • Definició: Un conjunt d'instruccions dissenyades per realitzar una tasca específica.
  • Característiques:
    • Entrada: L'algorisme pren zero o més entrades.
    • Sortida: L'algorisme produeix una o més sortides.
    • Definició: Cada pas està clarament definit.
    • Finitud: L'algorisme acaba després d'un nombre finit de passos.
    • Efectivitat: Cada pas és prou bàsic per ser realitzat.

Algorismes Bàsics

Algorismes de Classificació

Classificar és el procés d'ordenar dades en un ordre particular.

Bubble Sort

Bubble Sort és un algorisme de classificació simple basat en comparacions.

#include <iostream>
using namespace std;

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // Intercanviar arr[j] i arr[j+1]
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    cout << "Array ordenat: \n";
    for (int i=0; i < n; i++)
        cout << arr[i] << " ";
    return 0;
}

Selection Sort

Selection Sort és un altre algorisme de classificació simple basat en comparacions.

#include <iostream>
using namespace std;

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        int min_idx = i;
        for (int j = i+1; j < n; j++)
            if (arr[j] < arr[min_idx])
                min_idx = j;
        // Intercanviar l'element mínim trobat amb el primer element
        int temp = arr[min_idx];
        arr[min_idx] = arr[i];
        arr[i] = temp;
    }
}

int main() {
    int arr[] = {64, 25, 12, 22, 11};
    int n = sizeof(arr)/sizeof(arr[0]);
    selectionSort(arr, n);
    cout << "Array ordenat: \n";
    for (int i=0; i < n; i++)
        cout << arr[i] << " ";
    return 0;
}

Algorismes de Cerca

Cercar és el procés de trobar un element en una estructura de dades.

Cerca Lineal

La Cerca Lineal és un algorisme de cerca simple que comprova cada element fins que es troba l'element desitjat.

#include <iostream>
using namespace std;

int linearSearch(int arr[], int n, int x) {
    for (int i = 0; i < n; i++)
        if (arr[i] == x)
            return i;
    return -1;
}

int main() {
    int arr[] = {2, 3, 4, 10, 40};
    int x = 10;
    int n = sizeof(arr)/sizeof(arr[0]);
    int result = linearSearch(arr, n, x);
    (result == -1) ? cout << "L'element no és present a l'array"
                   : cout << "L'element és present a l'índex " << result;
    return 0;
}

Cerca Binària

La Cerca Binària és un algorisme de cerca més eficient que funciona en arrays ordenats.

#include <iostream>
using namespace std;

int binarySearch(int arr[], int l, int r, int x) {
    while (l <= r) {
        int m = l + (r - l) / 2;
        // Comprovar si x és present a la meitat
        if (arr[m] == x)
            return m;
        // Si x és més gran, ignorar la meitat esquerra
        if (arr[m] < x)
            l = m + 1;
        // Si x és més petit, ignorar la meitat dreta
        else
            r = m - 1;
    }
    // Si arribem aquí, l'element no estava present
    return -1;
}

int main() {
    int arr[] = {2, 3, 4, 10, 40};
    int x = 10;
    int n = sizeof(arr)/sizeof(arr[0]);
    int result = binarySearch(arr, 0, n-1, x);
    (result == -1) ? cout << "L'element no és present a l'array"
                   : cout << "L'element és present a l'índex " << result;
    return 0;
}

Algorismes Avançats

Programació Dinàmica

La Programació Dinàmica és un mètode per resoldre problemes complexos descomponent-los en subproblemes més simples.

Seqüència de Fibonacci

La seqüència de Fibonacci és un exemple clàssic de programació dinàmica.

#include <iostream>
using namespace std;

int fib(int n) {
    int f[n+1];
    f[0] = 0;
    f[1] = 1;
    for (int i = 2; i <= n; i++)
        f[i] = f[i-1] + f[i-2];
    return f[n];
}

int main() {
    int n = 9;
    cout << "El nombre de Fibonacci és " << fib(n);
    return 0;
}

Algorismes de Grafs

Els algorismes de grafs s'utilitzen per resoldre problemes relacionats amb la teoria de grafs.

Cerca en Profunditat (DFS)

DFS és un algorisme per travessar o cercar estructures de dades d'arbres o grafs.

#include <iostream>
#include <list>
using namespace std;

class Graph {
    int V;
    list<int> *adj;
    void DFSUtil(int v, bool visited[]);
public:
    Graph(int V);
    void addEdge(int v, int w);
    void DFS(int v);
};

Graph::Graph(int V) {
    this->V = V;
    adj = new list<int>[V];
}

void Graph::addEdge(int v, int w) {
    adj[v].push_back(w);
}

void Graph::DFSUtil(int v, bool visited[]) {
    visited[v] = true;
    cout << v << " ";
    list<int>::iterator i;
    for (i = adj[v].begin(); i != adj[v].end(); ++i)
        if (!visited[*i])
            DFSUtil(*i, visited);
}

void Graph::DFS(int v) {
    bool *visited = new bool[V];
    for (int i = 0; i < V; i++)
        visited[i] = false;
    DFSUtil(v, visited);
}

int main() {
    Graph g(4);
    g.addEdge(0, 1);
    g.addEdge(0, 2);
    g.addEdge(1, 2);
    g.addEdge(2, 0);
    g.addEdge(2, 3);
    g.addEdge(3, 3);
    cout << "Recorregut en Profunditat (començant des del vèrtex 2) \n";
    g.DFS(2);
    return 0;
}
