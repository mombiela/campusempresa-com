La semàntica de moviment és una característica poderosa introduïda en C++11 que permet que els recursos dels objectes temporals es moguin en lloc de copiar-se. Això pot millorar significativament el rendiment eliminant còpies profundes innecessàries. En aquesta secció, explorarem el concepte de la semàntica de moviment, els seus beneficis i com implementar-la en els teus programes C++.

Conceptes clau

  • Lvalue i Rvalue:

    • Lvalue: Un objecte que ocupa una ubicació identificable en la memòria (per exemple, variables).
    • Rvalue: Un objecte temporal que no ocupa una ubicació de memòria persistent (per exemple, el resultat d'una expressió).

  • Constructor de moviment: Un constructor que transfereix recursos d'un objecte temporal a un nou objecte.

  • Operador d'assignació de moviment: Un operador que transfereix recursos d'un objecte temporal a un objecte existent.

  • std::move: Una funció de la biblioteca estàndard que converteix un objecte en una referència rvalue, permetent la semàntica de moviment.

Constructor de moviment

Un constructor de moviment permet que un objecte prengui possessió dels recursos d'un objecte temporal. Això es fa utilitzant referències rvalue.

Exemple

#include <iostream>
#include <utility> // per std::move

class MyClass {
public:
    int* data;
    
    // Constructor
    MyClass(int value) : data(new int(value)) {
        std::cout << "Constructor called\n";
    }
    
    // Constructor de moviment
    MyClass(MyClass&& other) noexcept : data(other.data) {
        other.data = nullptr; // Anul·lar el punter de la font
        std::cout << "Move Constructor called\n";
    }
    
    // Destructor
    ~MyClass() {
        delete data;
        std::cout << "Destructor called\n";
    }
};

int main() {
    MyClass obj1(10);
    MyClass obj2(std::move(obj1)); // Es crida el constructor de moviment
    return 0;
}

Explicació

  • Constructor: Assigna memòria i inicialitza el membre data.
  • Constructor de moviment: Transfereix la possessió del punter data de other al nou objecte i anul·la el punter data en other.
  • Destructor: Elimina la memòria assignada.

Operador d'assignació de moviment

L'operador d'assignació de moviment permet que un objecte existent prengui possessió dels recursos d'un objecte temporal.

Exemple

#include <iostream>
#include <utility> // per std::move

class MyClass {
public:
    int* data;
    
    // Constructor
    MyClass(int value) : data(new int(value)) {
        std::cout << "Constructor called\n";
    }
    
    // Constructor de moviment
    MyClass(MyClass&& other) noexcept : data(other.data) {
        other.data = nullptr; // Anul·lar el punter de la font
        std::cout << "Move Constructor called\n";
    }
    
    // Operador d'assignació de moviment
    MyClass& operator=(MyClass&& other) noexcept {
        if (this != &other) {
            delete data; // Alliberar el recurs existent
            data = other.data; // Transferir la possessió
            other.data = nullptr; // Anul·lar el punter de la font
            std::cout << "Move Assignment Operator called\n";
        }
        return *this;
    }
    
    // Destructor
    ~MyClass() {
        delete data;
        std::cout << "Destructor called\n";
    }
};

int main() {
    MyClass obj1(10);
    MyClass obj2(20);
    obj2 = std::move(obj1); // Es crida l'operador d'assignació de moviment
    return 0;
}

Explicació

  • Operador d'assignació de moviment: Comprova l'autoassignació, elimina el recurs existent, transfereix la possessió del punter data de other a l'objecte actual i anul·la el punter data en other.

std::move

La funció std::move s'utilitza per convertir un objecte en una referència rvalue, permetent la semàntica de moviment.

Exemple

#include <iostream>
#include <utility> // per std::move

void process(MyClass obj) {
    // Funció que pren possessió de obj
}

int main() {
    MyClass obj1(10);
    process(std::move(obj1)); // std::move converteix obj1 en una referència rvalue
    return 0;
}

Explicació

  • std::move: Converteix obj1 en una referència rvalue, permetent que la funció process prengui possessió dels seus recursos.

Conclusió

La semàntica de moviment és una característica crucial en el C++ modern que millora el rendiment eliminant còpies profundes innecessàries. Entenent i implementant constructors de moviment i operadors d'assignació de moviment, pots escriure programes C++ més eficients i amigables amb els recursos. Recorda utilitzar std::move per permetre la semàntica de moviment quan tractis amb objectes temporals.
