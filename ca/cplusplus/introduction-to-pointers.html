Què són els Punteros?

Els punteros són un concepte fonamental en C++ que et permeten accedir i manipular directament la memòria. Són variables que emmagatzemen l'adreça de memòria d'una altra variable.

Conceptes Clau

  • Adreça de Memòria: La ubicació d'una variable a la memòria.
  • Variable Puntero: Una variable que emmagatzema l'adreça de memòria d'una altra variable.
  • Desreferenciació: Accedir al valor emmagatzemat a l'adreça de memòria que conté un puntero.

Declarar i Inicialitzar Punteros

Per declarar un puntero, utilitzes el símbol asterisc (*). La sintaxi és:

type *nomPuntero;

Per exemple:

int *ptr; // Declara un puntero a un enter

Exemple

int var = 10;
int *ptr = &var; // ptr ara conté l'adreça de var

En aquest exemple:

  • var és una variable enter.
  • &var és l'adreça de var.
  • ptr és un puntero que conté l'adreça de var.

Desreferenciació de Punteros

Desreferenciar un puntero significa accedir al valor emmagatzemat a l'adreça de memòria que conté el puntero. Ho fas utilitzant l'operador asterisc (*).

Exemple

int var = 10;
int *ptr = &var;
cout << *ptr; // Mostra 10

En aquest exemple:

  • *ptr accedeix al valor emmagatzemat a l'adreça que conté ptr.

Aritmètica de Punteros

Els punteros es poden incrementar o decrementar per apuntar a la següent o anterior ubicació de memòria del mateix tipus de dades.

Exemple

int arr[3] = {10, 20, 30};
int *ptr = arr; // Apunta al primer element de l'array

cout << *ptr << endl; // Mostra 10
ptr++;
cout << *ptr << endl; // Mostra 20

En aquest exemple:

  • ptr++ mou el puntero al següent enter de l'array.

Punteros i Arrays

Els arrays i els punteros estan estretament relacionats en C++. El nom d'un array actua com un puntero al seu primer element.

Exemple

int arr[3] = {10, 20, 30};
int *ptr = arr; // Equivalent a int *ptr = &arr[0];

for (int i = 0; i < 3; i++) {
    cout << *(ptr + i) << endl; // Mostra 10, 20, 30
}

En aquest exemple:

  • ptr + i apunta a l'element i-èssim de l'array.
  • *(ptr + i) accedeix al valor de l'element i-èssim.

Punteros a Punteros

Pots tenir punteros que apuntin a altres punteros. Això és útil per a l'assignació dinàmica de memòria i arrays multidimensionals.

Exemple

int var = 10;
int *ptr = &var;
int **pptr = &ptr;

cout << **pptr; // Mostra 10

En aquest exemple:

  • pptr és un puntero a ptr.
  • **pptr accedeix al valor emmagatzemat a l'adreça que conté ptr.

Errors Comuns

  • Punteros No Inicialitzats: Sempre inicialitza els punteros, ja que els punteros no inicialitzats poden portar a un comportament indefinit.
  • Punteros Penjants: Punteros que fan referència a memòria que ha estat alliberada.
  • Fuites de Memòria: No alliberar la memòria assignada dinàmicament.

Conclusió

Els punteros són una característica poderosa en C++ que proporcionen accés directe a la memòria i permeten la manipulació eficient d'estructures de dades. Entendre els punteros és crucial per a tasques de programació avançades com l'assignació dinàmica de memòria i la implementació d'estructures de dades complexes.

En dominar els punteros, obtindràs una comprensió més profunda de com funciona la memòria en C++ i estaràs millor preparat per escriure codi eficient i efectiu.
