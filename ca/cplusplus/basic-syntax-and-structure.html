Introducció

C++ és un llenguatge de programació potent i d'alt rendiment que suporta tant els paradigmes de programació procedimental com orientada a objectes. Entendre la sintaxi i l'estructura bàsiques de C++ és essencial per escriure codi eficient i efectiu. Aquesta secció cobrirà els elements fonamentals de la sintaxi i l'estructura de C++, proporcionant una base sòlida per a un aprenentatge posterior.

Conceptes Clau

  • Comentaris
  • Tipus de Dades
  • Variables
  • Operadors
  • Estructures de Control
  • Funcions

Comentaris

Els comentaris s'utilitzen per explicar el codi i fer-lo més llegible. Són ignorats pel compilador.

  • Comentaris d'una sola línia: Utilitza // per comentar una sola línia.
  • Comentaris de múltiples línies: Utilitza /* */ per comentar múltiples línies.
// Aquest és un comentari d'una sola línia

/*
Aquest és un 
comentari de múltiples línies
*/

Tipus de Dades

C++ suporta diversos tipus de dades, que es poden categoritzar àmpliament en:

  • Tipus de Dades Primitives: int, float, double, char, bool
  • Tipus de Dades Derivades: Arrays, Pointers, References
  • Tipus de Dades Definides per l'Usuari: Structures, Unions, Classes, Enums

Exemple:

int age = 25;         // Tipus enter
float height = 5.9;   // Tipus de punt flotant
char grade = 'A';     // Tipus caràcter
bool isStudent = true; // Tipus booleà

Variables

Les variables s'utilitzen per emmagatzemar dades que poden ser manipulades pel programa. Han de ser declarades abans d'usar-les.

Sintaxi:

data_type variable_name = value;

Exemple:

int number = 10;
float temperature = 36.6;
char initial = 'J';
bool isValid = false;

Operadors

Els operadors són símbols que realitzen operacions sobre variables i valors. Els operadors comuns inclouen:

  • Operadors Aritmètics: +, -, *, /, %
  • Operadors Relacionals: ==, !=, >, <, >=, <=
  • Operadors Lògics: &&, ||, !
  • Operadors d'Assignació: =, +=, -=, *=, /=, %=

Exemple:

int a = 10, b = 20;
int sum = a + b; // Operador aritmètic
bool result = (a == b); // Operador relacional
bool logicalResult = (a > 5 && b < 30); // Operador lògic
a += 5; // Operador d'assignació

Estructures de Control

Les estructures de control dictaminen el flux del programa. Les estructures de control comunes inclouen:

  • Declaracions Condicionals: if, else if, else, switch
  • Bucles: for, while, do-while

Exemple:

// Declaració Condicional
int num = 10;
if (num > 0) {
    cout << "Nombre positiu";
} else {
    cout << "Nombre no positiu";
}

// Bucle
for (int i = 0; i < 5; i++) {
    cout << i << " ";
}

Funcions

Les funcions són blocs de codi que realitzen una tasca específica. Ajuden a modularitzar el codi i reutilitzar-lo.

Sintaxi:

return_type function_name(parameters) {
    // Cos de la funció
}

Exemple:

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    cout << "Suma: " << result;
    return 0;
}

Conclusió

Entendre la sintaxi i l'estructura bàsiques de C++ és crucial per a qualsevol programador. Aquesta secció ha cobert els elements essencials, incloent comentaris, tipus de dades, variables, operadors, estructures de control i funcions. Dominar aquests conceptes bàsics proporcionarà una base sòlida per a temes més avançats en C++.
