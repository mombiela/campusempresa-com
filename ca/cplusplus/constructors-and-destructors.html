Introducció

En C++, els constructors i destructors són funcions membres especials d'una classe que s'utilitzen per inicialitzar i netejar objectes, respectivament. Entendre com utilitzar-los de manera efectiva és crucial per gestionar recursos i assegurar l'estabilitat dels teus programes.

Constructors

Què és un Constructor?

  • Un constructor és una funció membre especial que es crida automàticament quan es crea un objecte d'una classe.
  • S'utilitza per inicialitzar els membres de dades de l'objecte.

Punts Clau

  • Mateix Nom que la Classe: El constructor té el mateix nom que la classe.
  • Sense Tipus de Retorn: Els constructors no tenen tipus de retorn, ni tan sols void.
  • Es Pot Sobrecargar: Pots tenir múltiples constructors amb diferents paràmetres.

Tipus de Constructors

  1. Constructor per Defecte: Un constructor que no pren arguments.
  2. Constructor Parametritzat: Un constructor que pren un o més arguments.
  3. Constructor de Còpia: Un constructor que inicialitza un objecte utilitzant un altre objecte de la mateixa classe.

Exemples

Constructor per Defecte

class MyClass {
public:
    MyClass() {
        // Codi d'inicialització
        std::cout << "Constructor per Defecte Cridat" << std::endl;
    }
};

int main() {
    MyClass obj; // Es crida el constructor per defecte
    return 0;
}

Constructor Parametritzat

class MyClass {
public:
    int x;
    MyClass(int val) : x(val) {
        std::cout << "Constructor Parametritzat Cridat amb valor " << val << std::endl;
    }
};

int main() {
    MyClass obj(10); // Es crida el constructor parametritzat
    return 0;
}

Constructor de Còpia

class MyClass {
public:
    int x;
    MyClass(int val) : x(val) {}
    MyClass(const MyClass &obj) {
        x = obj.x;
        std::cout << "Constructor de Còpia Cridat" << std::endl;
    }
};

int main() {
    MyClass obj1(10);
    MyClass obj2 = obj1; // Es crida el constructor de còpia
    return 0;
}

Destructors

Què és un Destructor?

  • Un destructor és una funció membre especial que es crida automàticament quan un objecte surt de l'àmbit o es suprimeix explícitament.
  • S'utilitza per netejar recursos que l'objecte pot haver adquirit durant la seva vida útil.

Punts Clau

  • Mateix Nom que la Classe amb una Titlla (~): El destructor té el mateix nom que la classe, prefixat amb una titlla (~).
  • Sense Tipus de Retorn i Sense Paràmetres: Els destructors no tenen tipus de retorn i no prenen cap paràmetre.
  • Només Un Destructor: Una classe només pot tenir un destructor.

Exemple

class MyClass {
public:
    MyClass() {
        std::cout << "Constructor Cridat" << std::endl;
    }
    ~MyClass() {
        std::cout << "Destructor Cridat" << std::endl;
    }
};

int main() {
    MyClass obj; // Es crida el constructor
    // El destructor es cridarà automàticament quan obj surti de l'àmbit
    return 0;
}

Taula Comparativa

| Característica | Constructor | Destructor | |-----------------|--------------------------------------|-------------------------------------| | Nom | Mateix que el nom de la classe | Mateix que el nom de la classe prefixat amb ~ | | Tipus de Retorn | Cap | Cap | | Paràmetres | Pot tenir paràmetres | No pot tenir paràmetres | | Sobrecàrrega | Es pot sobrecarregar | No es pot sobrecarregar | | Propòsit | Inicialitzar objectes | Netejar abans que l'objecte sigui destruït |

Conclusió

Els constructors i destructors són conceptes fonamentals en C++ que ajuden a gestionar el cicle de vida dels objectes. Els constructors inicialitzen objectes, mentre que els destructors netegen recursos. Entendre com utilitzar-los de manera efectiva assegura que els teus programes siguin robustos i eficients en recursos. En dominar aquests conceptes, estaràs millor preparat per afrontar desafiaments de programació més complexos en C++.
