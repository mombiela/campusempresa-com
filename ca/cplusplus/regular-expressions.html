Les expressions regulars (regex) són eines poderoses per a la coincidència de patrons i el processament de text. En C++, la biblioteca <regex> proporciona suport per treballar amb expressions regulars. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de les expressions regulars en C++.

Introducció a les Expressions Regulars

  • Definició: Les expressions regulars són seqüències de caràcters que defineixen patrons de cerca, principalment per a l'ús en la coincidència de patrons amb cadenes de text.
  • Usos: Validació d'entrada, cerca dins de cadenes, substitució de subcadenes i anàlisi de text.

Conceptes Bàsics de les Expressions Regulars

Incloure la Biblioteca Regex

Per utilitzar expressions regulars en C++, cal incloure el capçal <regex>:

#include <regex>

Sintaxi Bàsica

  • Caràcters Literals: Coincideixen amb ells mateixos.
  • Metacaràcters: Caràcters amb significats especials (per exemple, . per a qualsevol caràcter, ^ per al començament d'una cadena, $ per al final d'una cadena).

Exemple Simple

#include <iostream>
#include <regex>

int main() {
    std::string text = "Hola, Món!";
    std::regex pattern("Món");

    if (std::regex_search(text, pattern)) {
        std::cout << "Patró trobat!" << std::endl;
    } else {
        std::cout << "Patró no trobat." << std::endl;
    }

    return 0;
}

Explicació

  • std::regex: Representa una expressió regular compilada.
  • std::regex_search: Cerca el patró dins del text.

Patrons Regex Comuns

  • Classes de Caràcters: [abc] coincideix amb qualsevol caràcter a, b o c.
  • Classes de Caràcters Predefinides: \d per a dígits, \w per a caràcters de paraula, \s per a espais en blanc.
  • Quantificadors: * (0 o més), + (1 o més), ? (0 o 1), {n} (exactament n), {n,} (n o més), {n,m} (entre n i m).

Exemple amb Quantificadors

#include <iostream>
#include <regex>

int main() {
    std::string text = "El número és 12345.";
    std::regex pattern("\\d+");

    if (std::regex_search(text, pattern)) {
        std::cout << "Número trobat!" << std::endl;
    } else {
        std::cout << "Número no trobat." << std::endl;
    }

    return 0;
}

Expressions Regulars Avançades

Grups de Captura

  • Definició: Els parèntesis () s'utilitzen per crear grups de captura.
  • Ús: Extreure parts específiques d'una coincidència.

Exemple amb Grups de Captura

#include <iostream>
#include <regex>

int main() {
    std::string text = "El meu correu és exemple@exemple.com.";
    std::regex pattern("(\\w+)@(\\w+\\.\\w+)");

    std::smatch matches;
    if (std::regex_search(text, matches, pattern)) {
        std::cout << "Nom d'usuari: " << matches[1] << std::endl;
        std::cout << "Domini: " << matches[2] << std::endl;
    } else {
        std::cout << "No s'ha trobat cap coincidència." << std::endl;
    }

    return 0;
}

Explicació

  • std::smatch: Conté els resultats de la cerca.
  • matches[n]: Accedeix al n-è grup de captura.

Substitució amb Regex

  • Funció: std::regex_replace substitueix les ocurrències del patró per una cadena especificada.

Exemple de Substitució amb Regex

#include <iostream>
#include <regex>

int main() {
    std::string text = "Tinc 2 pomes i 3 taronges.";
    std::regex pattern("\\d");
    std::string replacement = "#";

    std::string result = std::regex_replace(text, pattern, replacement);
    std::cout << "Resultat: " << result << std::endl;

    return 0;
}

Explicació

  • std::regex_replace: Substitueix totes les ocurrències del patró per la cadena de substitució.

Consideracions de Rendiment

  • Compilació: Compilar una regex pot ser costós. Reutilitza objectes regex compilats sempre que sigui possible.
  • Complexitat: Tingues cura amb patrons complexos, ja que poden causar problemes de rendiment.

Conclusió

Les expressions regulars són una eina versàtil en C++ per al processament de text. Des de la coincidència bàsica de patrons fins a la manipulació avançada de text, entendre les regex pot millorar significativament la teva capacitat per gestionar cadenes de manera efectiva. Practica amb diferents patrons i casos d'ús per convertir-te en un expert en l'ús d'expressions regulars en els teus programes en C++.
