Introducció

En C++, les funcions i classes amigues proporcionen una manera de concedir privilegis d'accés especials a funcions o classes específiques. Això els permet accedir a membres privats i protegits d'una altra classe, cosa que pot ser útil en certs escenaris on cal evitar l'encapsulació per raons específiques.

Conceptes Clau

  • Funció Amiga: Una funció que no és membre d'una classe però té accés als seus membres privats i protegits.
  • Classe Amiga: Una classe que té accés als membres privats i protegits d'una altra classe.

Funcions Amigues

Definició i Sintaxi

Una funció amiga es defineix utilitzant la paraula clau friend dins de la classe els membres privats o protegits de la qual necessita accedir.

class MyClass {
private:
    int secretValue;

public:
    MyClass(int value) : secretValue(value) {}

    // Declarar funció amiga
    friend void showSecretValue(const MyClass& obj);
};

// Definició de la funció amiga
void showSecretValue(const MyClass& obj) {
    std::cout << "Secret Value: " << obj.secretValue << std::endl;
}

Explicació

  • La paraula clau friend s'utilitza dins de la classe MyClass per declarar la funció showSecretValue com a amiga.
  • La funció amiga showSecretValue pot accedir al membre privat secretValue de MyClass.

Exemple

Aquí tens un exemple complet que demostra l'ús d'una funció amiga:

#include <iostream>

class MyClass {
private:
    int secretValue;

public:
    MyClass(int value) : secretValue(value) {}

    // Declarar funció amiga
    friend void showSecretValue(const MyClass& obj);
};

// Definició de la funció amiga
void showSecretValue(const MyClass& obj) {
    std::cout << "Secret Value: " << obj.secretValue << std::endl;
}

int main() {
    MyClass obj(42);
    showSecretValue(obj);  // Output: Secret Value: 42
    return 0;
}

Usos

  • Quan una funció no membre necessita accedir a membres privats o protegits d'una classe.
  • Quan es sobrecarreguen operadors que requereixen accedir a membres privats o protegits.

Classes Amigues

Definició i Sintaxi

Una classe amiga es declara utilitzant la paraula clau friend dins de la classe els membres privats o protegits de la qual necessita accedir.

class MyClass {
private:
    int secretValue;

public:
    MyClass(int value) : secretValue(value) {}

    // Declarar classe amiga
    friend class FriendClass;
};

class FriendClass {
public:
    void showSecretValue(const MyClass& obj) {
        std::cout << "Secret Value: " << obj.secretValue << std::endl;
    }
};

Explicació

  • La paraula clau friend s'utilitza dins de la classe MyClass per declarar FriendClass com a amiga.
  • La funció membre showSecretValue de FriendClass pot accedir al membre privat secretValue de MyClass.

Exemple

Aquí tens un exemple complet que demostra l'ús d'una classe amiga:

#include <iostream>

class MyClass {
private:
    int secretValue;

public:
    MyClass(int value) : secretValue(value) {}

    // Declarar classe amiga
    friend class FriendClass;
};

class FriendClass {
public:
    void showSecretValue(const MyClass& obj) {
        std::cout << "Secret Value: " << obj.secretValue << std::endl;
    }
};

int main() {
    MyClass obj(42);
    FriendClass friendObj;
    friendObj.showSecretValue(obj);  // Output: Secret Value: 42
    return 0;
}

Usos

  • Quan una altra classe necessita accedir a membres privats o protegits d'una classe.
  • Quan s'implementen classes estretament lligades que necessiten compartir dades privades.

Resum

Les funcions i classes amigues en C++ proporcionen una manera d'accedir a membres privats i protegits d'una classe, cosa que pot ser útil en escenaris específics. Tot i que trenquen el principi d'encapsulació, ofereixen flexibilitat i poden simplificar certs dissenys. No obstant això, s'han d'utilitzar amb prudència per mantenir la integritat i seguretat de les dades de la classe.

Entenent i utilitzant adequadament les funcions i classes amigues, pots millorar les teves habilitats de programació en C++ i crear codi més versàtil i eficient.
