Introducció als Iteradors

Els iteradors són un concepte fonamental en C++ que et permeten recórrer els elements d'un contenidor (com ara arrays, vectors, llistes, etc.) d'una manera consistent. Proporcionen una manera d'accedir als elements d'un contenidor sense exposar l'estructura subjacent del contenidor.

Conceptes Clau

  • Tipus d'Iteradors: Entrada, Sortida, Endavant, Bidireccional i Accés Aleatori.
  • Operacions d'Iteradors: Desreferenciació, Increment, Decrement.
  • Categories d'Iteradors: Basades en les capacitats i restriccions dels iteradors.

Ús Bàsic dels Iteradors

Què és un Iterador?

Un iterador és un objecte que apunta a un element dins d'un contenidor i es pot utilitzar per recórrer el contenidor.

Exemple: Ús Bàsic d'un Iterador amb `std::vector`

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it;

    for (it = numbers.begin(); it != numbers.end(); ++it) {
        std::cout << *it << " ";
    }
    return 0;
}
  • Explicació:
    • numbers.begin() retorna un iterador al primer element.
    • numbers.end() retorna un iterador a un passat l'últim element.
    • *it desreferencia l'iterador per accedir al valor al qual apunta.
    • ++it incrementa l'iterador per apuntar al següent element.

Tipus d'Iteradors

Iteradors d'Entrada i Sortida

  • Iteradors d'Entrada: Poden llegir elements d'un contenidor.
  • Iteradors de Sortida: Poden escriure elements a un contenidor.

Exemple: Iterador d'Entrada

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it = numbers.begin();

    while (it != numbers.end()) {
        std::cout << *it << " ";
        ++it;
    }
    return 0;
}

Exemple: Iterador de Sortida

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers(5);
    std::vector<int>::iterator it = numbers.begin();

    int value = 1;
    while (it != numbers.end()) {
        *it = value++;
        ++it;
    }

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Iteradors Endavant, Bidireccionals i d'Accés Aleatori

  • Iteradors Endavant: Poden llegir i escriure elements i moure's endavant.
  • Iteradors Bidireccionals: Poden moure's tant endavant com enrere.
  • Iteradors d'Accés Aleatori: Poden moure's a qualsevol element en temps constant.

Exemple: Iterador Bidireccional amb `std::list`

#include <iostream>
#include <list>

int main() {
    std::list<int> numbers = {1, 2, 3, 4, 5};
    std::list<int>::iterator it = numbers.begin();

    // Moure's endavant
    ++it;
    std::cout << *it << " "; // Sortida: 2

    // Moure's enrere
    --it;
    std::cout << *it << " "; // Sortida: 1

    return 0;
}

Exemple: Iterador d'Accés Aleatori amb `std::vector`

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it = numbers.begin();

    // Moure's endavant per 3 posicions
    it += 3;
    std::cout << *it << " "; // Sortida: 4

    // Moure's enrere per 2 posicions
    it -= 2;
    std::cout << *it << " "; // Sortida: 2

    return 0;
}

Tècniques Avançades d'Iteradors

Ús de `std::iterator` Traits

Els traits de std::iterator proporcionen una manera de definir iteradors personalitzats heretant de std::iterator.

Exemple: Iterador Personalitzat

#include <iostream>
#include <iterator>
#include <vector>

template <typename T>
class CustomIterator : public std::iterator<std::input_iterator_tag, T> {
    T* p;
public:
    CustomIterator(T* x) : p(x) {}
    CustomIterator(const CustomIterator& mit) : p(mit.p) {}
    CustomIterator& operator++() { ++p; return *this; }
    CustomIterator operator++(int) { CustomIterator tmp(*this); operator++(); return tmp; }
    bool operator==(const CustomIterator& rhs) const { return p == rhs.p; }
    bool operator!=(const CustomIterator& rhs) const { return p != rhs.p; }
    T& operator*() { return *p; }
};

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    CustomIterator<int> begin(numbers.data());
    CustomIterator<int> end(numbers.data() + numbers.size());

    while (begin != end) {
        std::cout << *begin << " ";
        ++begin;
    }
    return 0;
}

Conclusió

Els iteradors són una característica poderosa en C++ que proporcionen una manera uniforme d'accedir als elements d'un contenidor. Venen en diversos tipus i categories, cadascun adequat per a diferents tasques. Entendre i utilitzar els iteradors de manera efectiva pot millorar molt la teva capacitat per treballar amb col·leccions en C++.
