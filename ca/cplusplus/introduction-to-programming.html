Visió General

Benvingut al curs "Introducció a la Programació en C++"! Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat en programació en C++. Cobrirem conceptes fonamentals, sintaxi i característiques avançades de C++.

Què és C++?

C++ és un llenguatge de programació potent i d'alt rendiment que suporta paradigmes de programació procedimental, orientada a objectes i genèrica. És àmpliament utilitzat en el desenvolupament de programari, desenvolupament de jocs i programació de sistemes.

Per què Aprendre C++?

  • Rendiment: C++ és conegut pel seu alt rendiment i eficiència.
  • Versatilitat: Es pot utilitzar per a una àmplia gamma d'aplicacions.
  • Demanda a la Indústria: Moltes indústries requereixen competència en C++.
  • Fonament: Aprendre C++ proporciona una base sòlida per entendre altres llenguatges de programació.

Configurant el Teu Entorn

Abans de començar a codificar, configurem l'entorn de desenvolupament.

Instal·lant un Compilador

  • Windows: Instal·la MinGW o Visual Studio.
  • Mac: Instal·la Xcode o utilitza Homebrew per instal·lar GCC.
  • Linux: Utilitza el gestor de paquets per instal·lar GCC o Clang.

Configurant un IDE

  • Visual Studio Code: Un editor lleuger i versàtil.
  • CLion: Un IDE potent específicament per a C++.
  • Code::Blocks: Un IDE gratuït per a C++.

Sintaxi i Estructura Bàsica

Comencem amb l'estructura bàsica d'un programa en C++.

Programa Hola Món

#include <iostream>

int main() {
    std::cout << "Hola, Món!" << std::endl;
    return 0;
}
  • #include <iostream>: Inclou la biblioteca de flux d'entrada i sortida.
  • int main() { ... }: La funció principal on comença l'execució del programa.
  • std::cout: S'utilitza per imprimir sortida a la consola.
  • return 0;: Indica que el programa ha acabat amb èxit.

Variables i Tipus de Dades

Les variables s'utilitzen per emmagatzemar dades. C++ suporta diversos tipus de dades.

Tipus de Dades Comuns

  • int: Tipus enter.
  • float: Tipus de punt flotant.
  • double: Tipus de punt flotant de doble precisió.
  • char: Tipus caràcter.
  • bool: Tipus booleà.

Exemple

int edat = 25;
float alçada = 5.9;
char nota = 'A';
bool ésEstudiant = true;

Estructures de Control

Les estructures de control et permeten controlar el flux del teu programa.

Sentències Condicionals

  • if: Executa un bloc de codi si una condició és certa.
  • else: Executa un bloc de codi si la condició és falsa.

Exemple

int nombre = 10;
if (nombre > 0) {
    std::cout << "Nombre positiu" << std::endl;
} else {
    std::cout << "Nombre no positiu" << std::endl;
}

Bucles

  • for: Executa un bloc de codi un nombre especificat de vegades.
  • while: Executa un bloc de codi mentre una condició sigui certa.
  • do-while: Executa un bloc de codi almenys una vegada i després repeteix mentre una condició sigui certa.

Exemple

for (int i = 0; i < 5; i++) {
    std::cout << "i = " << i << std::endl;
}

int j = 0;
while (j < 5) {
    std::cout << "j = " << j << std::endl;
    j++;
}

int k = 0;
do {
    std::cout << "k = " << k << std::endl;
    k++;
} while (k < 5);

Funcions

Les funcions són blocs de codi que realitzen una tasca específica.

Definint una Funció

int suma(int a, int b) {
    return a + b;
}

Cridant una Funció

int resultat = suma(5, 3);
std::cout << "Resultat: " << resultat << std::endl;

Conclusió

En aquest tema introductori, hem cobert els conceptes bàsics de la programació en C++, incloent la configuració del teu entorn, la comprensió de la sintaxi bàsica, l'ús de variables i tipus de dades, estructures de control i funcions. Aquests fonaments són crucials a mesura que avancem cap a temes més avançats. Continua practicant, i aviat seràs competent en programació en C++!
