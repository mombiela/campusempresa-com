Introducció

En aquesta secció, explorarem les instruccions break i continue en C++. Aquestes instruccions de control de flux s'utilitzen per alterar el flux normal dels bucles, proporcionant més control sobre com i quan s'executen els bucles.

Instrucció Break

L'instrucció break s'utilitza per sortir d'un bucle prematurament. Quan es troba una instrucció break dins d'un bucle, el bucle es termina immediatament, i el control del programa es reprèn a la següent instrucció després del bucle.

Conceptes Clau

  • Terminació Immediata: L'instrucció break fa que el bucle es termini immediatament.
  • Bucles Aplicables: Es pot utilitzar en bucles for, while i do-while.
  • Bucles Imbricats: Quan s'utilitza en bucles imbricats, break només surt del bucle més intern.

Exemple

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break; // Sortir del bucle quan i és 5
        }
        cout << i << " ";
    }
    cout << "Bucle terminat." << endl;
    return 0;
}

Explicació: En aquest exemple, el bucle imprimirà números del 0 al 4. Quan i és igual a 5, s'executa la instrucció break, terminant el bucle.

Instrucció Continue

L'instrucció continue salta la iteració actual del bucle i procedeix amb la següent iteració. A diferència de break, no termina el bucle sinó que simplement salta el codi restant en la iteració actual.

Conceptes Clau

  • Saltar Iteració: L'instrucció continue salta la resta del codi en la iteració actual del bucle.
  • Bucles Aplicables: Es pot utilitzar en bucles for, while i do-while.
  • Bucles Imbricats: Quan s'utilitza en bucles imbricats, continue només afecta el bucle més intern.

Exemple

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue; // Saltar la resta del cos del bucle per a números parells
        }
        cout << i << " ";
    }
    cout << "Bucle completat." << endl;
    return 0;
}

Explicació: En aquest exemple, el bucle imprimirà només números senars del 1 al 9. Quan i és parell, s'executa la instrucció continue, saltant la instrucció cout per a aquesta iteració.

Comparació de Break i Continue

| Característica | Break | Continue | |-------------------|--------------------------------|--------------------------------| | Efecte | Termina el bucle | Salta la iteració actual | | Terminació del Bucle | Sí | No | | Efecte en Bucles Imbricats | Afecta només el bucle més intern | Afecta només el bucle més intern |

Exercicis

  1. Exercici 1: Escriu un programa que utilitzi un bucle for per imprimir números de l'1 al 20. Utilitza la instrucció break per sortir del bucle quan el número sigui 15.
  2. Exercici 2: Escriu un programa que utilitzi un bucle while per imprimir números de l'1 al 10. Utilitza la instrucció continue per saltar la impressió del número 5.

Solució de l'Exercici 1

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 20; i++) {
        if (i == 15) {
            break; // Sortir del bucle quan i és 15
        }
        cout << i << " ";
    }
    cout << "Bucle terminat a 15." << endl;
    return 0;
}

Solució de l'Exercici 2

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 10) {
        if (i == 5) {
            i++;
            continue; // Saltar la resta del cos del bucle quan i és 5
        }
        cout << i << " ";
        i++;
    }
    cout << "Bucle completat, saltat el 5." << endl;
    return 0;
}

Conclusió

Les instruccions break i continue són eines poderoses per controlar el flux dels bucles en C++. La instrucció break et permet sortir d'un bucle prematurament, mentre que la instrucció continue et permet saltar la iteració actual i procedir amb la següent. Entendre com i quan utilitzar aquestes instruccions pot ajudar-te a escriure codi més eficient i llegible.
