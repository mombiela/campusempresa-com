Polimorfisme en C++
Encapsulació i Abstracció ►

El polimorfisme és un dels conceptes fonamentals de la Programació Orientada a Objectes (POO) en C++. Permet que els objectes siguin tractats com a instàncies de la seva classe pare en lloc de la seva classe real. Els dos principals tipus de polimorfisme en C++ són el polimorfisme en temps de compilació (o estàtic) i el polimorfisme en temps d'execució (o dinàmic).

Polimorfisme en Temps de Compilació

El polimorfisme en temps de compilació s'aconsegueix mitjançant la sobrecàrrega de funcions i la sobrecàrrega d'operadors.

Sobrecàrrega de Funcions

La sobrecàrrega de funcions permet que múltiples funcions tinguin el mateix nom però diferents paràmetres.

#include <iostream>
using namespace std;

class Print {
public:
    void show(int i) {
        cout << "Enter: " << i << endl;
    }
    void show(double d) {
        cout << "Double: " << d << endl;
    }
    void show(string s) {
        cout << "String: " << s << endl;
    }
};

int main() {
    Print obj;
    obj.show(5);
    obj.show(5.5);
    obj.show("Hello");
    return 0;
}

En aquest exemple, la funció show està sobrecarregada per manejar diferents tipus de paràmetres.

Sobrecàrrega d'Operadors

La sobrecàrrega d'operadors permet redefinir la manera com els operadors funcionen per a tipus definits per l'usuari.

#include <iostream>
using namespace std;

class Complex {
private:
    float real;
    float imag;
public:
    Complex() : real(0), imag(0) {}
    Complex(float r, float i) : real(r), imag(i) {}

    Complex operator + (const Complex& obj) {
        Complex temp;
        temp.real = real + obj.real;
        temp.imag = imag + obj.imag;
        return temp;
    }

    void display() {
        cout << "Real: " << real << " Imaginary: " << imag << endl;
    }
};

int main() {
    Complex c1(3.3, 4.4), c2(1.1, 2.2);
    Complex c3 = c1 + c2;
    c3.display();
    return 0;
}

En aquest exemple, l'operador + està sobrecarregat per sumar dos objectes Complex.

Polimorfisme en Temps d'Execució

El polimorfisme en temps d'execució s'aconsegueix mitjançant la herència i les funcions virtuals.

Herència

La herència permet que una classe hereti propietats i comportaments d'una altra classe.

#include <iostream>
using namespace std;

class Base {
public:
    void show() {
        cout << "Base class" << endl;
    }
};

class Derived : public Base {
public:
    void show() {
        cout << "Derived class" << endl;
    }
};

int main() {
    Derived d;
    d.show();
    return 0;
}

En aquest exemple, la classe Derived hereta de la classe Base i sobreescriu la funció show.

Funcions Virtuals

Les funcions virtuals permeten que les classes derivades sobreescriguin mètodes en una classe base, permetent el polimorfisme en temps d'execució.

#include <iostream>
using namespace std;

class Base {
public:
    virtual void show() {
        cout << "Base class" << endl;
    }
};

class Derived : public Base {
public:
    void show() override {
        cout << "Derived class" << endl;
    }
};

int main() {
    Base* b;
    Derived d;
    b = &d;
    b->show();
    return 0;
}

En aquest exemple, la funció show en la classe Base es declara com a virtual, permetent que la classe Derived la sobreescrigui. Quan la funció show es crida a través d'un punter de la classe base, s'executa la versió de la classe derivada.

Resum

El polimorfisme en C++ és una característica poderosa que permet un codi més flexible i mantenible. Es pot aconseguir mitjançant:

  • Polimorfisme en Temps de Compilació: Usant la sobrecàrrega de funcions i la sobrecàrrega d'operadors.
  • Polimorfisme en Temps d'Execució: Usant la herència i les funcions virtuals.

Entendre i utilitzar efectivament el polimorfisme pot millorar enormement la teva capacitat per escriure programes en C++ robustos i escalables.
