Introducció a Classes i Objectes

En C++, les classes i els objectes són conceptes fonamentals de la Programació Orientada a Objectes (OOP). Ajuden a organitzar el codi i crear components reutilitzables.

Conceptes Clau

  • Classe: Un plànol per crear objectes. Defineix un tipus de dada agrupant dades i mètodes que treballen sobre les dades.
  • Objecte: Una instància d'una classe. Es crea utilitzant la definició de la classe.

Exemple

#include <iostream>
using namespace std;

class Car {
public:
    string brand;
    string model;
    int year;

    void displayInfo() {
        cout << "Brand: " << brand << ", Model: " << model << ", Year: " << year << endl;
    }
};

int main() {
    Car car1;
    car1.brand = "Toyota";
    car1.model = "Corolla";
    car1.year = 2020;

    car1.displayInfo();

    return 0;
}

En aquest exemple:

  • Car és una classe amb tres atributs públics: brand, model i year.
  • displayInfo és un mètode que imprimeix els detalls del cotxe.
  • car1 és un objecte de la classe Car.

Definició de Classes

Una classe es defineix utilitzant la paraula clau class seguida del nom de la classe i un parell de claus {}.

Sintaxi

class ClassName {
public:
    // Membres públics
private:
    // Membres privats
protected:
    // Membres protegits
};

Especificadors d'Accés

  • public: Els membres són accessibles des de fora de la classe.
  • private: Els membres són accessibles només dins de la classe.
  • protected: Els membres són accessibles dins de la classe i per la classe derivada.

Exemple

class Person {
public:
    string name;
    int age;

    void introduce() {
        cout << "Hi, I'm " << name << " and I'm " << age << " years old." << endl;
    }
};

Creació d'Objectes

Els objectes són instàncies d'una classe. Es creen declarant una variable del tipus de la classe.

Exemple

Person person1;
person1.name = "Alice";
person1.age = 30;
person1.introduce();

Constructors i Destructors

Els constructors i destructors són funcions membres especials d'una classe.

Constructors

  • Constructor: Una funció especial que es crida automàticament quan es crea un objecte. Inicialitza l'objecte.
  • Sintaxi: El constructor té el mateix nom que la classe i no té tipus de retorn.

Exemple

class Person {
public:
    string name;
    int age;

    // Constructor
    Person(string n, int a) {
        name = n;
        age = a;
    }

    void introduce() {
        cout << "Hi, I'm " << name << " and I'm " << age << " years old." << endl;
    }
};

int main() {
    Person person1("Alice", 30);
    person1.introduce();

    return 0;
}

Destructors

  • Destructor: Una funció especial que es crida automàticament quan es destrueix un objecte. Neteja els recursos.
  • Sintaxi: El destructor té el mateix nom que la classe precedit per una tilde ~ i no té tipus de retorn.

Exemple

class Person {
public:
    string name;
    int age;

    // Constructor
    Person(string n, int a) {
        name = n;
        age = a;
    }

    // Destructor
    ~Person() {
        cout << "Destructor called for " << name << endl;
    }

    void introduce() {
        cout << "Hi, I'm " << name << " and I'm " << age << " years old." << endl;
    }
};

int main() {
    Person person1("Alice", 30);
    person1.introduce();

    return 0;
}

Temes Avançats

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

Exemple

class Animal {
public:
    void eat() {
        cout << "This animal is eating." << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "The dog is barking." << endl;
    }
};

int main() {
    Dog dog1;
    dog1.eat();  // Mètode heretat
    dog1.bark(); // Mètode propi del gos

    return 0;
}

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen.

Exemple

class Animal {
public:
    virtual void sound() {
        cout << "This animal makes a sound." << endl;
    }
};

class Dog : public Animal {
public:
    void sound() override {
        cout << "The dog barks." << endl;
    }
};

class Cat : public Animal {
public:
    void sound() override {
        cout << "The cat meows." << endl;
    }
};

int main() {
    Animal* animal1 = new Dog();
    Animal* animal2 = new Cat();

    animal1->sound(); // Outputs: The dog barks.
    animal2->sound(); // Outputs: The cat meows.

    delete animal1;
    delete animal2;

    return 0;
}

Resum

  • Classes i Objectes: Conceptes fonamentals en C++ OOP.
  • Classe: Plànol per a objectes, que conté dades i mètodes.
  • Objecte: Instància d'una classe.
  • Constructors i Destructors: Funcions especials per inicialitzar i netejar objectes.
  • Herència: Mecanisme per crear una nova classe a partir d'una classe existent.
  • Polimorfisme: Capacitat de cridar el mateix mètode en diferents objectes i que cadascun respongui a la seva manera.

Entendre aquests conceptes és crucial per dominar C++ i escriure codi eficient i reutilitzable.
