El Projecte Sobre nosaltres Contribuir Donacions Llicència

L'herència és un concepte fonamental en la programació orientada a objectes (OOP) que permet a una classe heretar propietats i comportaments (mètodes) d'una altra classe. Això promou la reutilització de codi i estableix una jerarquia natural entre classes.

Què és l'Herència?

L'herència permet que una nova classe (classe derivada) hereti atributs i mètodes d'una classe existent (classe base). Això permet que la classe derivada reutilitzi codi de la classe base i ampliï o modifiqui el seu comportament.

Conceptes Clau

  • Classe Base: La classe les propietats i mètodes de la qual són heretats.
  • Classe Derivada: La classe que hereta propietats i mètodes de la classe base.
  • Especificadors d'Accés: Paraules clau que defineixen l'accessibilitat dels membres de la classe base en la classe derivada (public, protected, private).
  • Sobreescriptura: Redefinició d'un mètode de la classe base en la classe derivada.
  • Polimorfisme: La capacitat de cridar mètodes de la classe derivada a través d'una referència de la classe base.

Sintaxi Bàsica

Definint una Classe Base

class Base {
public:
    int baseValue;
    void display() {
        std::cout << "Funció display de la classe base" << std::endl;
    }
};

Definint una Classe Derivada

class Derived : public Base {
public:
    int derivedValue;
    void show() {
        std::cout << "Funció show de la classe derivada" << std::endl;
    }
};

Creant Objectes i Accedint a Membres

int main() {
    Derived obj;
    obj.baseValue = 10; // Accedint al membre de la classe base
    obj.derivedValue = 20; // Accedint al membre de la classe derivada
    obj.display(); // Cridant al mètode de la classe base
    obj.show(); // Cridant al mètode de la classe derivada
    return 0;
}

Especificadors d'Accés en l'Herència

Els especificadors d'accés controlen l'accessibilitat dels membres de la classe base en la classe derivada.

| Especificador d'Accés | Accés als Membres de la Classe Base en la Classe Derivada | Accés Extern | |-----------------------|-----------------------------------------------------------|--------------| | public | Públic | Públic | | protected | Protegit | Privat | | private | Privat | Privat |

Exemple amb Especificadors d'Accés

class Base {
protected:
    int protectedValue;
public:
    int publicValue;
};

class Derived : public Base {
public:
    void setValues() {
        protectedValue = 10; // Accessible
        publicValue = 20; // Accessible
    }
};

Sobreescriptura de Mètodes

Una classe derivada pot sobreescriure un mètode de la classe base per proporcionar una implementació específica.

Exemple de Sobreescriptura de Mètodes

class Base {
public:
    virtual void display() {
        std::cout << "Funció display de la classe base" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Funció display de la classe derivada" << std::endl;
    }
};

int main() {
    Base* basePtr;
    Derived derivedObj;
    basePtr = &derivedObj;
    basePtr->display(); // Crida a la funció display de la classe derivada
    return 0;
}

Polimorfisme

El polimorfisme permet utilitzar la mateixa interfície per a diferents tipus de dades. En C++, això sovint s'aconsegueix a través de la sobreescriptura de funcions i els punters o referències a tipus de classe base.

Exemple de Polimorfisme

class Base {
public:
    virtual void display() {
        std::cout << "Funció display de la classe base" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Funció display de la classe derivada" << std::endl;
    }
};

void showDisplay(Base* basePtr) {
    basePtr->display();
}

int main() {
    Base baseObj;
    Derived derivedObj;
    showDisplay(&baseObj); // Crida a la funció display de la classe base
    showDisplay(&derivedObj); // Crida a la funció display de la classe derivada
    return 0;
}

Conclusió

L'herència és una característica poderosa en C++ que promou la reutilització de codi i estableix una jerarquia natural entre classes. Comprenent i utilitzant l'herència, pots crear codi més modular, mantenible i escalable. Conceptes clau com els especificadors d'accés, la sobreescriptura de mètodes i el polimorfisme són essencials per dominar l'herència en C++. Practica aquests conceptes amb diversos exemples per consolidar la teva comprensió.
