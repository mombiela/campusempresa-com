El Projecte Sobre nosaltres Contribuir Donacions Llicència

Les declaracions condicionals són fonamentals en la programació ja que permeten executar diferents blocs de codi basats en certes condicions. En C++, les principals declaracions condicionals són if, else if, else i switch.

Declaració If

La declaració if s'utilitza per executar un bloc de codi si una condició especificada és certa.

Sintaxi

if (condició) {
    // codi a executar si la condició és certa
}

Exemple

#include <iostream>
using namespace std;

int main() {
    int number = 10;

    if (number > 5) {
        cout << "El número és més gran que 5" << endl;
    }

    return 0;
}

En aquest exemple, el missatge "El número és més gran que 5" es mostrarà perquè la condició number > 5 és certa.

Declaració Else

La declaració else s'utilitza per executar un bloc de codi si la condició en la declaració if és falsa.

Sintaxi

if (condició) {
    // codi a executar si la condició és certa
} else {
    // codi a executar si la condició és falsa
}

Exemple

#include <iostream>
using namespace std;

int main() {
    int number = 3;

    if (number > 5) {
        cout << "El número és més gran que 5" << endl;
    } else {
        cout << "El número no és més gran que 5" << endl;
    }

    return 0;
}

En aquest exemple, el missatge "El número no és més gran que 5" es mostrarà perquè la condició number > 5 és falsa.

Declaració Else If

La declaració else if s'utilitza per especificar una nova condició a provar si la primera condició és falsa.

Sintaxi

if (condició1) {
    // codi a executar si la condició1 és certa
} else if (condició2) {
    // codi a executar si la condició2 és certa
} else {
    // codi a executar si totes les condicions són falses
}

Exemple

#include <iostream>
using namespace std;

int main() {
    int number = 7;

    if (number > 10) {
        cout << "El número és més gran que 10" << endl;
    } else if (number > 5) {
        cout << "El número és més gran que 5 però menor o igual a 10" << endl;
    } else {
        cout << "El número és 5 o menor" << endl;
    }

    return 0;
}

En aquest exemple, el missatge "El número és més gran que 5 però menor o igual a 10" es mostrarà perquè la condició number > 5 és certa.

Declaració Switch

La declaració switch s'utilitza per executar un bloc de codi d'entre múltiples opcions basades en el valor d'una variable.

Sintaxi

switch (expressió) {
    case constant1:
        // codi a executar si l'expressió és igual a constant1
        break;
    case constant2:
        // codi a executar si l'expressió és igual a constant2
        break;
    // pots tenir qualsevol nombre de declaracions case
    default:
        // codi a executar si l'expressió no coincideix amb cap case
}

Exemple

#include <iostream>
using namespace std;

int main() {
    int day = 3;

    switch (day) {
        case 1:
            cout << "Dilluns" << endl;
            break;
        case 2:
            cout << "Dimarts" << endl;
            break;
        case 3:
            cout << "Dimecres" << endl;
            break;
        case 4:
            cout << "Dijous" << endl;
            break;
        case 5:
            cout << "Divendres" << endl;
            break;
        case 6:
            cout << "Dissabte" << endl;
            break;
        case 7:
            cout << "Diumenge" << endl;
            break;
        default:
            cout << "Dia invàlid" << endl;
    }

    return 0;
}

En aquest exemple, el missatge "Dimecres" es mostrarà perquè el valor de day és 3.

Resum

  • Declaració If: Executa un bloc de codi si una condició especificada és certa.
  • Declaració Else: Executa un bloc de codi si la condició en la declaració if és falsa.
  • Declaració Else If: Especifica una nova condició a provar si la primera condició és falsa.
  • Declaració Switch: Executa un bloc de codi d'entre múltiples opcions basades en el valor d'una variable.

Entendre i utilitzar les declaracions condicionals de manera efectiva et permet controlar el flux del teu programa i prendre decisions basades en diferents condicions. Practica escrivint i combinant aquestes declaracions per convertir-te en un expert en manejar diverses situacions en els teus programes en C++.
