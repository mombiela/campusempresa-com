Les expressions lambda, també conegudes com a funcions lambda o clausures, són una característica poderosa en C++ que et permeten definir funcions anònimes directament dins del teu codi. Són particularment útils per a funcions curtes i d'un sol ús que no es reutilitzen enlloc més.

Introducció a les Expressions Lambda

  • Definició: Una expressió lambda és una funció anònima que pot capturar variables del seu entorn.
  • Sintaxi: La sintaxi bàsica d'una expressió lambda és: 
    [captura](paràmetres) -> tipus_de_retorn {
    // cos de la funció
};

Sintaxi Bàsica i Ús

  • Clàusula de Captura: La clàusula de captura (els claudàtors []) s'utilitza per especificar quines variables de l'entorn són accessibles dins del lambda.
  • Paràmetres: La llista de paràmetres és similar a la d'una funció regular.
  • Tipus de Retorn: El tipus de retorn és opcional i sovint pot ser inferit pel compilador.
  • Cos de la Funció: El cos de la funció conté el codi que s'executarà quan es cridi el lambda.

Exemple: Expressió Lambda Bàsica

#include <iostream>

int main() {
    auto add = [](int a, int b) -> int {
        return a + b;
    };

    std::cout << "Sum: " << add(3, 4) << std::endl; // Output: Sum: 7
    return 0;
}

En aquest exemple:

  • auto add declara un lambda que pren dos enters i retorna la seva suma.
  • El lambda es crida amb els arguments 3 i 4, i el resultat es mostra.

Captura de Variables

Les expressions lambda poden capturar variables del seu entorn de diverses maneres:

  • Per Valor: Captura una còpia de la variable.
  • Per Referència: Captura una referència a la variable.
  • Captura Implícita: Captura automàticament les variables utilitzades en el lambda per valor o per referència.

Exemple: Captura per Valor i Referència

#include <iostream>

int main() {
    int x = 10;
    int y = 20;

    auto captureByValue = [x, y]() {
        return x + y;
    };

    auto captureByReference = [&x, &y]() {
        return x + y;
    };

    x = 15;
    y = 25;

    std::cout << "Capture by value: " << captureByValue() << std::endl; // Output: 30
    std::cout << "Capture by reference: " << captureByReference() << std::endl; // Output: 40

    return 0;
}

En aquest exemple:

  • captureByValue captura x i y per valor, així que els canvis a x i y després de definir el lambda no afecten el resultat.
  • captureByReference captura x i y per referència, així que els canvis a x i y es reflecteixen en el resultat del lambda.

Característiques Avançades

Lambdas Mutables

Per defecte, els lambdas que capturen variables per valor no poden modificar aquestes variables. Per permetre la modificació, pots utilitzar la paraula clau mutable.

#include <iostream>

int main() {
    int x = 10;

    auto mutableLambda = [x]() mutable {
        x += 5;
        return x;
    };

    std::cout << "Mutable lambda: " << mutableLambda() << std::endl; // Output: 15
    std::cout << "Original x: " << x << std::endl; // Output: 10

    return 0;
}

Lambdas Genèrics

C++14 va introduir els lambdas genèrics, que et permeten utilitzar auto en la llista de paràmetres per crear un lambda que pugui acceptar arguments de qualsevol tipus.

#include <iostream>

int main() {
    auto genericLambda = [](auto a, auto b) {
        return a + b;
    };

    std::cout << "Sum of integers: " << genericLambda(3, 4) << std::endl; // Output: 7
    std::cout << "Sum of doubles: " << genericLambda(3.5, 4.5) << std::endl; // Output: 8

    return 0;
}

Aplicacions Pràctiques

Ús de Lambdas amb Algorismes Estàndard

Els lambdas sovint s'utilitzen amb algorismes estàndard com std::sort, std::for_each i std::transform.

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    std::for_each(numbers.begin(), numbers.end(), [](int &n) {
        n *= 2;
    });

    for (int n : numbers) {
        std::cout << n << " "; // Output: 2 4 6 8 10
    }

    return 0;
}

En aquest exemple:

  • std::for_each aplica el lambda a cada element del vector, duplicant cada valor.

Conclusió

Les expressions lambda en C++ proporcionen una manera concisa i poderosa de definir funcions anònimes. Són particularment útils per a funcions curtes i d'un sol ús i poden capturar variables del seu entorn. Entendre com utilitzar els lambdas de manera efectiva pot millorar molt la teva capacitat per escriure codi C++ net i eficient.
