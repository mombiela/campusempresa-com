Els punteros intel·ligents són una característica poderosa en C++ que ajuden a gestionar la memòria dinàmica automàticament. Asseguren que la memòria es desallotgi correctament quan ja no és necessària, prevenint fuites de memòria i punteros penjants.

Introducció als Punteros Intel·ligents

  • Què són els Punteros Intel·ligents?

    • Els punteros intel·ligents són objectes que actuen com punteros però proporcionen una gestió automàtica de la memòria.
    • Formen part de la Biblioteca Estàndard de C++, introduïda en C++11.

  • Per què Utilitzar Punteros Intel·ligents?

    • Per evitar fuites de memòria.
    • Per assegurar la seguretat en cas d'excepcions.
    • Per simplificar la gestió de la memòria.

Tipus de Punteros Intel·ligents

C++ proporciona diversos tipus de punteros intel·ligents, cadascun amb diferents propòsits:

  1. std::unique_ptr
  2. std::shared_ptr
  3. std::weak_ptr

`std::unique_ptr`

  • Característiques:

    • Posseeix un recurs exclusivament.
    • No es pot copiar, només moure.
    • Elimina automàticament el recurs quan surt de l'àmbit.

  • Exemple d'Ús:

#include <iostream>
#include <memory>

void uniquePtrExample() {
    std::unique_ptr<int> ptr1 = std::make_unique<int>(10);
    std::cout << "Valor: " << *ptr1 << std::endl;

    // std::unique_ptr<int> ptr2 = ptr1; // Error: no es pot copiar unique_ptr
    std::unique_ptr<int> ptr2 = std::move(ptr1); // Transferència de propietat

    if (!ptr1) {
        std::cout << "ptr1 ara és nul." << std::endl;
    }
    std::cout << "Valor: " << *ptr2 << std::endl;
}

int main() {
    uniquePtrExample();
    return 0;
}

`std::shared_ptr`

  • Característiques:

    • Propietat compartida d'un recurs.
    • Mecanisme de recompte de referències per fer un seguiment del nombre de propietaris.
    • El recurs s'elimina quan l'últim shared_ptr que el posseeix es destrueix.

  • Exemple d'Ús:

#include <iostream>
#include <memory>

void sharedPtrExample() {
    std::shared_ptr<int> ptr1 = std::make_shared<int>(20);
    std::cout << "Valor: " << *ptr1 << std::endl;
    std::cout << "Recompte d'ús: " << ptr1.use_count() << std::endl;

    {
        std::shared_ptr<int> ptr2 = ptr1;
        std::cout << "Recompte d'ús després de copiar: " << ptr1.use_count() << std::endl;
    }

    std::cout << "Recompte d'ús després que ptr2 surti de l'àmbit: " << ptr1.use_count() << std::endl;
}

int main() {
    sharedPtrExample();
    return 0;
}

`std::weak_ptr`

  • Característiques:

    • No posseeix el recurs.
    • S'utilitza per trencar referències circulars.
    • Es pot convertir a std::shared_ptr si és necessari.

  • Exemple d'Ús:

#include <iostream>
#include <memory>

void weakPtrExample() {
    std::shared_ptr<int> sharedPtr = std::make_shared<int>(30);
    std::weak_ptr<int> weakPtr = sharedPtr;

    std::cout << "Recompte d'ús: " << sharedPtr.use_count() << std::endl;

    if (auto tempPtr = weakPtr.lock()) {
        std::cout << "Valor: " << *tempPtr << std::endl;
    } else {
        std::cout << "El recurs ja no existeix." << std::endl;
    }
}

int main() {
    weakPtrExample();
    return 0;
}

Comparació de Punteros Intel·ligents

| Característica | std::unique_ptr | std::shared_ptr | std::weak_ptr | |-----------------------|-------------------|-------------------|-----------------| | Propietat | Exclusiva | Compartida | No propietària | | Copiable | No | Sí | No | | Movible | Sí | Sí | No | | Recompte de Referències | No | Sí | Sí | | Cas d'Ús | Propietat única | Propietat compartida | Trencar cicles |

Conclusió

Els punteros intel·ligents en C++ proporcionen una manera robusta de gestionar la memòria dinàmica, reduint el risc de fuites de memòria i punteros penjants. Entenent i utilitzant std::unique_ptr, std::shared_ptr i std::weak_ptr, pots escriure codi C++ més segur i eficient. Tria sempre el puntero intel·ligent adequat basant-te en la semàntica de propietat i els requisits del cas d'ús.
