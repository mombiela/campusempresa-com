Visió General

El projecte final està dissenyat per consolidar els teus coneixements i habilitats adquirides al llarg d'aquest curs de C++. Aquest projecte et desafiarà a aplicar diversos conceptes, des de la sintaxi bàsica fins a tècniques de programació avançades, d'una manera cohesionada i pràctica. Al final d'aquest projecte, hauràs d'haver adquirit una comprensió completa de C++ i ser capaç de desenvolupar una aplicació totalment funcional.

Descripció del Projecte

Crearàs un joc senzill basat en text utilitzant C++. El joc implicarà un jugador navegant a través d'una sèrie d'habitacions, recollint objectes i resolent trencaclosques per aconseguir un objectiu específic. Aquest projecte requerirà que utilitzis:

  • Sintaxi bàsica i estructures de control
  • Funcions i programació modular
  • Programació Orientada a Objectes (POO)
  • Operacions d'entrada/sortida de fitxers
  • Gestió dinàmica de memòria
  • Gestió d'errors

Requisits

Estructura Bàsica del Joc

  1. Menú Principal:

    • Mostrar opcions per començar un nou joc, carregar un joc guardat o sortir.
    • Exemple: 
      void displayMainMenu() {
    std::cout << "1. Començar Nou Joc\n";
    std::cout << "2. Carregar Joc\n";
    std::cout << "3. Sortir\n";
    std::cout << "Introdueix la teva elecció: ";
}

  2. Bucle del Joc:

    • Correr contínuament fins que el jugador guanyi o decideixi sortir.
    • Exemple: 
      bool gameRunning = true;
while (gameRunning) {
    // Lògica del joc aquí
    if (playerWins || playerExits) {
        gameRunning = false;
    }
}

Classes de Jugador i Habitació

  1. Classe Jugador:

    • Atributs: nom, salut, inventari.
    • Mètodes: moure, recollir objecte, utilitzar objecte.
    • Exemple: 
      class Player {
public:
    std::string name;
    int health;
    std::vector<std::string> inventory;

    void move();
    void pickUpItem(const std::string& item);
    void useItem(const std::string& item);
};

  2. Classe Habitació:

    • Atributs: descripció, objectes, sortides.
    • Mètodes: afegir objecte, eliminar objecte, descriure habitació.
    • Exemple: 
      class Room {
public:
    std::string description;
    std::vector<std::string> items;
    std::map<std::string, Room*> exits;

    void addItem(const std::string& item);
    void removeItem(const std::string& item);
    void describeRoom();
};

Mecàniques del Joc

  1. Navegació:

    • Permetre al jugador moure's entre habitacions.
    • Exemple: 
      void Player::move(Room* currentRoom, const std::string& direction) {
    if (currentRoom->exits.find(direction) != currentRoom->exits.end()) {
        currentRoom = currentRoom->exits[direction];
        std::cout << "T'has mogut cap a " << direction << ".\n";
    } else {
        std::cout << "No pots anar en aquesta direcció.\n";
    }
}

  2. Gestió d'Objectes:

    • Recollir i utilitzar objectes.
    • Exemple: 
      void Player::pickUpItem(const std::string& item) {
    inventory.push_back(item);
    std::cout << "Has recollit un " << item << ".\n";
}

void Player::useItem(const std::string& item) {
    auto it = std::find(inventory.begin(), inventory.end(), item);
    if (it != inventory.end()) {
        inventory.erase(it);
        std::cout << "Has utilitzat el " << item << ".\n";
    } else {
        std::cout << "No tens un " << item << ".\n";
    }
}

Guardar i Carregar l'Estat del Joc

  1. Entrada/Sortida de Fitxers:
    • Guardar l'estat actual del joc en un fitxer.
    • Carregar un estat de joc guardat des d'un fitxer.
    • Exemple: 
      void saveGame(const Player& player, const Room& room) {
    std::ofstream outFile("savegame.txt");
    outFile << player.name << "\n" << player.health << "\n";
    for (const auto& item : player.inventory) {
        outFile << item << "\n";
    }
    outFile.close();
}

void loadGame(Player& player, Room& room) {
    std::ifstream inFile("savegame.txt");
    inFile >> player.name >> player.health;
    std::string item;
    while (inFile >> item) {
        player.inventory.push_back(item);
    }
    inFile.close();
}
