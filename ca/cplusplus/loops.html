Introducció als Bucles

Els bucles són constructes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En C++, hi ha tres tipus principals de bucles:

  • Bucle for
  • Bucle while
  • Bucle do-while

Cada tipus de bucle serveix per a diferents propòsits i es pot utilitzar segons els requisits específics del teu programa.

El Bucle `for`

El bucle for s'utilitza típicament quan el nombre d'iteracions es coneix per endavant. Consta de tres parts: inicialització, condició i increment/decrement.

Sintaxi

for (inicialització; condició; increment/decrement) {
    // Codi a executar
}

Exemple

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << "Iteració " << i << endl;
    }
    return 0;
}

En aquest exemple:

  • Inicialització: int i = 0
  • Condició: i < 5
  • Increment: i++

El bucle s'executarà 5 vegades, imprimint el número d'iteració cada vegada.

El Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix i depèn d'una condició que sigui certa.

Sintaxi

while (condició) {
    // Codi a executar
}

Exemple

#include <iostream>
using namespace std;

int main() {
    int i = 0;
    while (i < 5) {
        cout << "Iteració " << i << endl;
        i++;
    }
    return 0;
}

En aquest exemple, el bucle continuarà executant-se mentre i sigui menor que 5.

El Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

Sintaxi

do {
    // Codi a executar
} while (condició);

Exemple

#include <iostream>
using namespace std;

int main() {
    int i = 0;
    do {
        cout << "Iteració " << i << endl;
        i++;
    } while (i < 5);
    return 0;
}

En aquest exemple, el bucle s'executarà almenys una vegada i després continuarà executant-se mentre i sigui menor que 5.

Bucles Imbricats

Els bucles es poden imbricar entre si. Això és útil per treballar amb estructures de dades multidimensionals com matrius.

Exemple

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            cout << "i: " << i << ", j: " << j << endl;
        }
    }
    return 0;
}

En aquest exemple, el bucle extern s'executa 3 vegades, i per a cada iteració del bucle extern, el bucle intern s'executa 3 vegades.

Sentències de Control de Bucles

C++ proporciona diverses sentències de control per alterar el flux dels bucles:

  • break: Surt del bucle immediatament.
  • continue: Salta l'iteració actual i procedeix a la següent iteració.
  • goto: Transfereix el control a una sentència etiquetada (utilitzar amb precaució).

Exemple amb `break` i `continue`

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break; // Surt del bucle quan i és 5
        }
        if (i % 2 == 0) {
            continue; // Salta els números parells
        }
        cout << "Iteració " << i << endl;
    }
    return 0;
}

En aquest exemple:

  • El bucle surt quan i és 5 a causa de la sentència break.
  • La sentència continue salta els números parells.

Resum

  • Bucle for: Millor utilitzat quan el nombre d'iteracions es coneix.
  • Bucle while: Millor utilitzat quan el nombre d'iteracions és desconegut i depèn d'una condició.
  • Bucle do-while: Garanteix que el bloc de codi s'executi almenys una vegada.
  • Bucles imbricats: Útils per a estructures de dades multidimensionals.
  • Sentències de control de bucles: break, continue i goto poden alterar el flux dels bucles.

Entendre i dominar els bucles és crucial per a una programació eficient i efectiva en C++. Practica utilitzant diferents tipus de bucles i sentències de control per convertir-te en un expert en el seu ús.
