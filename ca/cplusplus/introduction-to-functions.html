Paràmetres de Funció i Tipus de Retorn ►

Què és una Funció?

  • Una funció és un bloc de codi reutilitzable que realitza una tasca específica.
  • Les funcions ajuden a descompondre problemes complexos en peces més petites i manejables.
  • Promouen la reutilització del codi i milloren la llegibilitat.

Sintaxi d'una Funció

  • Una funció en C++ consisteix en un tipus de retorn, un nom de funció i una llista de paràmetres tancada entre parèntesis.
  • El cos de la funció està tancat entre claus {}.

Exemple:

#include <iostream>

// Declaració de la funció
int add(int a, int b);

int main() {
    int result = add(5, 3); // Crida a la funció
    std::cout << "La suma és: " << result << std::endl;
    return 0;
}

// Definició de la funció
int add(int a, int b) {
    return a + b;
}

Declaració i Definició de Funcions

  • Declaració de la Funció (Prototip): Informa al compilador sobre el nom de la funció, el tipus de retorn i els paràmetres. Acaba amb un punt i coma.
  • Definició de la Funció: Conté el cos real de la funció.

Exemple:

// Declaració
int multiply(int, int);

// Definició
int multiply(int x, int y) {
    return x * y;
}

Tipus de Retorn

  • El tipus de retorn especifica el tipus de valor que la funció retornarà.
  • Si una funció no retorna cap valor, el seu tipus de retorn és void.

Exemple:

void printMessage() {
    std::cout << "Hola, Món!" << std::endl;
}

Paràmetres i Arguments

  • Paràmetres: Variables llistades en la declaració/definició de la funció.
  • Arguments: Valors passats a la funció quan és cridada.

Exemple:

void greet(std::string name) {
    std::cout << "Hola, " << name << "!" << std::endl;
}

int main() {
    greet("Alice"); // Argument: "Alice"
    return 0;
}

Paràmetres per Defecte

  • C++ permet que les funcions tinguin valors de paràmetres per defecte.
  • Si no es proporciona cap argument, s'utilitza el valor per defecte.

Exemple:

void display(int x = 10) {
    std::cout << "Valor: " << x << std::endl;
}

int main() {
    display();    // Utilitza el valor per defecte 10
    display(5);   // Utilitza el valor proporcionat 5
    return 0;
}

Sobrecàrrega de Funcions

  • La sobrecàrrega de funcions permet que múltiples funcions tinguin el mateix nom amb diferents paràmetres.
  • El compilador les diferencia per les seves llistes de paràmetres.

Exemple:

void print(int i) {
    std::cout << "Enter: " << i << std::endl;
}

void print(double f) {
    std::cout << "Double: " << f << std::endl;
}

void print(std::string s) {
    std::cout << "Cadena: " << s << std::endl;
}

int main() {
    print(10);
    print(3.14);
    print("Hola");
    return 0;
}

Funcions Inline

  • Les funcions inline es defineixen amb la paraula clau inline.
  • Suggereixen al compilador que insereixi el cos de la funció on es fa la crida, per reduir la sobrecàrrega d'una crida a funció.

Exemple:

inline int square(int x) {
    return x * x;
}

int main() {
    std::cout << "Quadrat de 5: " << square(5) << std::endl;
    return 0;
}

Recursivitat

  • Una funció que es crida a si mateixa es coneix com una funció recursiva.
  • La recursivitat és útil per a tasques que es poden definir en termes de subtasques similars.

Exemple:

int factorial(int n) {
    if (n <= 1) return 1;
    else return n * factorial(n - 1);
}

int main() {
    std::cout << "Factorial de 5: " << factorial(5) << std::endl;
    return 0;
}

Resum

  • Les funcions són essencials per descompondre problemes complexos i promoure la reutilització del codi.
  • Entendre la declaració, definició, paràmetres, tipus de retorn i sobrecàrrega de funcions és crucial.
  • Les funcions inline i la recursivitat són conceptes avançats que ofereixen beneficis de rendiment i solucions elegants a certs problemes.
  • Practica escrivint i utilitzant funcions per convertir-te en un expert en estructurar els teus programes en C++ de manera efectiva.
