Plantilles en C++
Gestió d'Excepcions ►

Les plantilles són una característica poderosa en C++ que et permeten escriure codi genèric i reutilitzable. Permeten que funcions i classes operin amb tipus genèrics, que es poden especificar quan la funció o classe és instanciada. Aquesta secció et guiarà des dels conceptes bàsics de les plantilles fins a usos més avançats.

Introducció a les Plantilles

Les plantilles et permeten definir funcions i classes amb un marcador de posició per al tipus de dades. Això fa que el teu codi sigui més flexible i reutilitzable.

Plantilles de Funció

Les plantilles de funció et permeten crear una funció que pot operar amb diferents tipus de dades sense haver de reescriure tota la funció per a cada tipus.

Exemple:

#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(3, 4) << endl;       // Mostra 7
    cout << add(3.5, 4.5) << endl;   // Mostra 8.0
    return 0;
}
  • template <typename T>: Aquesta línia declara una plantilla amb un marcador de posició de tipus T.
  • T add(T a, T b): Aquesta funció pren dos paràmetres de tipus T i retorna un valor de tipus T.

Plantilles de Classe

Les plantilles de classe et permeten crear una classe que pot manejar qualsevol tipus de dades.

Exemple:

#include <iostream>
using namespace std;

template <typename T>
class Box {
private:
    T value;
public:
    Box(T val) : value(val) {}
    T getValue() { return value; }
};

int main() {
    Box<int> intBox(123);
    Box<double> doubleBox(456.78);

    cout << intBox.getValue() << endl;       // Mostra 123
    cout << doubleBox.getValue() << endl;    // Mostra 456.78
    return 0;
}
  • template <typename T>: Aquesta línia declara una plantilla amb un marcador de posició de tipus T.
  • Box<T>: Aquesta classe ara pot manejar qualsevol tipus de dades especificat quan es crea un objecte.

Plantilles Avançades

Especialització de Plantilles

L'especialització de plantilles et permet definir una implementació diferent d'una plantilla per a un tipus de dades específic.

Exemple:

#include <iostream>
using namespace std;

template <typename T>
class Box {
public:
    T value;
    Box(T val) : value(val) {}
    void print() {
        cout << "Plantilla genèrica: " << value << endl;
    }
};

// Especialització per a int
template <>
class Box<int> {
public:
    int value;
    Box(int val) : value(val) {}
    void print() {
        cout << "Plantilla especialitzada per a int: " << value << endl;
    }
};

int main() {
    Box<double> doubleBox(3.14);
    Box<int> intBox(123);

    doubleBox.print();  // Mostra: Plantilla genèrica: 3.14
    intBox.print();     // Mostra: Plantilla especialitzada per a int: 123

    return 0;
}
  • template <> class Box<int>: Aquesta línia declara una versió especialitzada de la classe Box per al tipus int.
  • La classe especialitzada pot tenir una implementació diferent de la plantilla genèrica.

Paràmetres de Plantilla

Les plantilles poden tenir paràmetres no-tipus, que et permeten passar valors a les plantilles.

Exemple:

#include <iostream>
using namespace std;

template <typename T, int size>
class Array {
private:
    T arr[size];
public:
    void set(int index, T value) {
        if (index >= 0 && index < size) {
            arr[index] = value;
        }
    }
    T get(int index) {
        if (index >= 0 && index < size) {
            return arr[index];
        }
        return T(); // Retorna el valor per defecte de T
    }
};

int main() {
    Array<int, 5> intArray;
    intArray.set(0, 10);
    intArray.set(1, 20);

    cout << intArray.get(0) << endl;  // Mostra 10
    cout << intArray.get(1) << endl;  // Mostra 20

    return 0;
}
  • template <typename T, int size>: Aquesta línia declara una plantilla amb un paràmetre de tipus T i un paràmetre no-tipus size.
  • Array<int, 5>: Això crea un objecte Array que pot contenir 5 enters.

Conclusió

Les plantilles en C++ proporcionen una manera poderosa d'escriure codi genèric i reutilitzable. Et permeten crear funcions i classes que poden operar amb qualsevol tipus de dades, reduint la redundància i augmentant la flexibilitat. Entenent i utilitzant les plantilles, pots escriure programes en C++ més eficients i mantenibles.
