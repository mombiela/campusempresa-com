Introducció a l'STL
Contenidors ►

La Standard Template Library (STL) és un conjunt potent de classes de plantilles de C++ que proporciona classes i funcions d'ús general amb plantilles que implementen molts algoritmes i estructures de dades populars i comunament utilitzats com vectors, llistes, cues i piles. Aquesta secció t'introduirà a l'STL, els seus components i com utilitzar-los de manera efectiva.

Què és l'STL?

L'STL és una col·lecció de classes de plantilles de C++ que proporcionen estructures de dades i algoritmes. Inclou:

  • Containers: Objectes que emmagatzemen dades.
  • Algoritmes: Procediments per processar dades.
  • Iteradors: Objectes que apunten a elements dins dels contenidors.

Per què utilitzar l'STL?

  • Eficiència: L'STL està optimitzat per al rendiment.
  • Reutilització: Proporciona components de codi reutilitzables.
  • Consistència: Interfície estandarditzada a través de diferents contenidors i algoritmes.

Components clau de l'STL

Contenidors

Els contenidors s'utilitzen per gestionar col·leccions d'objectes. Els contenidors més comunament utilitzats són:

  • Contenidors de seqüència: Emmagatzemen dades de manera lineal.
    • vector
    • deque
    • list
  • Contenidors associatius: Emmagatzemen dades en un ordre ordenat.
    • set
    • map
    • multiset
    • multimap
  • Contenidors desordenats: Emmagatzemen dades de manera desordenada.
    • unordered_set
    • unordered_map
    • unordered_multiset
    • unordered_multimap

Iteradors

Els iteradors s'utilitzen per apuntar als elements d'un contenidor. Són similars als punters i es poden utilitzar per recórrer els elements d'un contenidor.

  • Iteradors d'entrada: Llegeixen del contenidor.
  • Iteradors de sortida: Escriuen al contenidor.
  • Iteradors de avançament: Llegeixen i escriuen, poden avançar.
  • Iteradors bidireccionals: Llegeixen i escriuen, poden avançar i retrocedir.
  • Iteradors d'accés aleatori: Llegeixen i escriuen, poden moure's a qualsevol element directament.

Algoritmes

Els algoritmes són un conjunt de procediments per processar els elements dels contenidors. Alguns algoritmes comuns inclouen:

  • Cerca: find, binary_search
  • Ordenació: sort, stable_sort
  • Modificació: copy, swap
  • Operacions de conjunt: set_union, set_intersection

Exemples

Utilitzant un Vector

Els vectors són arrays dinàmics que poden canviar de mida.

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers;

    // Afegint elements al vector
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    // Accedint als elements
    for (int i = 0; i < numbers.size(); ++i) {
        std::cout << numbers[i] << " ";
    }
    std::cout << std::endl;

    // Utilitzant un iterador
    for (std::vector<int>::iterator it = numbers.begin(); it != numbers.end(); ++it) {
        std::cout << *it << " ";
    }
    std::cout << std::endl;

    return 0;
}

Utilitzant un Map

Els mapes emmagatzemen parelles clau-valor i permeten una recuperació ràpida basada en claus.

#include <iostream>
#include <map>

int main() {
    std::map<std::string, int> ageMap;

    // Afegint elements al mapa
    ageMap["Alice"] = 30;
    ageMap["Bob"] = 25;
    ageMap["Charlie"] = 35;

    // Accedint als elements
    std::cout << "L'edat de l'Alice: " << ageMap["Alice"] << std::endl;

    // Utilitzant un iterador
    for (std::map<std::string, int>::iterator it = ageMap.begin(); it != ageMap.end(); ++it) {
        std::cout << it->first << ": " << it->second << std::endl;
    }

    return 0;
}

Conclusió

L'STL és una part essencial de la programació moderna en C++, proporcionant un conjunt ric d'eines per gestionar dades i realitzar operacions comunes de manera eficient. En dominar els contenidors, iteradors i algoritmes de l'STL, pots escriure codi més eficient, llegible i mantenible. Practica utilitzant diferents components de l'STL per convertir-te en un expert en aprofitar tot el seu potencial.
