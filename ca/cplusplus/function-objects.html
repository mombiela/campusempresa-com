Els objectes funció, també coneguts com a functors, són objectes que es poden tractar com si fossin una funció o un punter a funció. Són instàncies de classes que tenen definit l'operator(). Això permet que es puguin utilitzar en qualsevol context on es faria servir un punter a funció, com ara en algorismes o com a funcions de retorn de trucada.

Conceptes Clau

  • Objecte Funció (Functor): Un objecte que es pot cridar com si fos una funció.
  • operator(): L'operador de crida de funció que cal sobrecarregar a la classe per convertir-la en un objecte funció.
  • Functors amb Estat: Functors que mantenen l'estat entre crides.
  • Functors sense Estat: Functors que no mantenen l'estat entre crides.

Exemple Bàsic

Comencem amb un exemple senzill per il·lustrar com funcionen els objectes funció.

#include <iostream>

// Definir una classe amb operator() sobrecarregat
class Add {
public:
    int operator()(int a, int b) const {
        return a + b;
    }
};

int main() {
    Add add; // Crear una instància de la classe Add
    int result = add(2, 3); // Utilitzar la instància com una funció
    std::cout << "2 + 3 = " << result << std::endl; // Sortida: 2 + 3 = 5
    return 0;
}

En aquest exemple:

  • La classe Add té un operator() sobrecarregat que pren dos enters i retorna la seva suma.
  • Es crea una instància de Add i s'utilitza com una funció per sumar dos números.

Functors amb Estat

Els functors amb estat poden mantenir l'estat entre crides. Això pot ser útil per acumular valors o mantenir un compte.

#include <iostream>

// Definir una classe amb un estat
class Counter {
private:
    int count;
public:
    Counter() : count(0) {}
    int operator()() {
        return ++count;
    }
};

int main() {
    Counter counter;
    std::cout << counter() << std::endl; // Sortida: 1
    std::cout << counter() << std::endl; // Sortida: 2
    std::cout << counter() << std::endl; // Sortida: 3
    return 0;
}

En aquest exemple:

  • La classe Counter manté un estat (count) que s'incrementa cada vegada que es crida el functor.

Utilització de Functors amb Algorismes Estàndard

Els functors sovint s'utilitzen amb algorismes estàndard de la Biblioteca Estàndard de C++.

#include <iostream>
#include <vector>
#include <algorithm>

// Definir un functor per a la comparació
class IsEven {
public:
    bool operator()(int n) const {
        return n % 2 == 0;
    }
};

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5, 6};
    auto it = std::find_if(numbers.begin(), numbers.end(), IsEven());
    
    if (it != numbers.end()) {
        std::cout << "Primer número parell: " << *it << std::endl; // Sortida: Primer número parell: 2
    } else {
        std::cout << "No s'ha trobat cap número parell." << std::endl;
    }
    return 0;
}

En aquest exemple:

  • El functor IsEven s'utilitza amb l'algorisme std::find_if per trobar el primer número parell en un vector.

Ús Avançat: Functors Parametritzats

Els functors també es poden parametritzar per fer-los més flexibles.

#include <iostream>

// Definir un functor parametritzat
class Multiply {
private:
    int factor;
public:
    Multiply(int f) : factor(f) {}
    int operator()(int n) const {
        return n * factor;
    }
};

int main() {
    Multiply multiplyBy2(2);
    Multiply multiplyBy3(3);
    
    std::cout << "5 * 2 = " << multiplyBy2(5) << std::endl; // Sortida: 5 * 2 = 10
    std::cout << "5 * 3 = " << multiplyBy3(5) << std::endl; // Sortida: 5 * 3 = 15
    return 0;
}

En aquest exemple:

  • El functor Multiply està parametritzat amb un factor, permetent que diferents instàncies multipliquin per diferents valors.

Conclusió

Els objectes funció en C++ proporcionen una manera potent i flexible d'encapsular funcions i estat. Es poden utilitzar en lloc de punters a funció i són particularment útils amb algorismes estàndard. Sobrecàrregant l'operator(), pots crear objectes que es comporten com funcions, amb el benefici addicional de mantenir l'estat si cal. Entendre i utilitzar functors pot millorar molt la teva capacitat d'escriure codi C++ net i eficient.
