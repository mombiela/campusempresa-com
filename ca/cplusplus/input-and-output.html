Introducció

L'entrada i sortida (I/O) són conceptes fonamentals en qualsevol llenguatge de programació. En C++, les operacions d'I/O es gestionen principalment utilitzant fluxos. Aquesta secció cobrirà els conceptes bàsics de l'I/O en C++, incloent com llegir des del teclat i escriure a la consola, així com temes més avançats com l'I/O de fitxers i manipuladors de flux personalitzats.

Entrada i Sortida Bàsica a la Consola

Llegir des de la Consola

Per llegir l'entrada des de la consola, C++ proporciona l'objecte cin, que forma part de la biblioteca estàndard.

Exemple:

#include <iostream>

int main() {
    int number;
    std::cout << "Entra un número: ";
    std::cin >> number;
    std::cout << "Has entrat: " << number << std::endl;
    return 0;
}
  • std::cin s'utilitza per llegir l'entrada des de l'entrada estàndard (teclat).
  • std::cout s'utilitza per escriure la sortida a la sortida estàndard (consola).
  • std::endl s'utilitza per inserir un caràcter de nova línia i buidar el flux.

Escriure a la Consola

Per escriure la sortida a la consola, C++ utilitza l'objecte cout.

Exemple:

#include <iostream>

int main() {
    std::cout << "Hola, Món!" << std::endl;
    return 0;
}
  • << és l'operador d'inserció utilitzat per enviar dades al flux de sortida.

Entrada i Sortida Avançada a la Consola

Formatant la Sortida

C++ proporciona diversos manipuladors per formatar la sortida.

Exemple:

#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159;
    std::cout << "Per defecte: " << pi << std::endl;
    std::cout << "Fixat: " << std::fixed << pi << std::endl;
    std::cout << "Científic: " << std::scientific << pi << std::endl;
    std::cout << "Precisió: " << std::setprecision(2) << pi << std::endl;
    return 0;
}
  • std::fixed i std::scientific s'utilitzen per formatar nombres de punt flotant.
  • std::setprecision estableix el nombre de dígits a mostrar després del punt decimal.

I/O de Fitxers

Llegir des d'un Fitxer

Per llegir des d'un fitxer, C++ utilitza la classe ifstream.

Exemple:

#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream inputFile("example.txt");
    std::string line;

    if (inputFile.is_open()) {
        while (getline(inputFile, line)) {
            std::cout << line << std::endl;
        }
        inputFile.close();
    } else {
        std::cout << "No es pot obrir el fitxer" << std::endl;
    }

    return 0;
}
  • std::ifstream s'utilitza per llegir des de fitxers.
  • getline llegeix una línia del fitxer.

Escriure a un Fitxer

Per escriure a un fitxer, C++ utilitza la classe ofstream.

Exemple:

#include <iostream>
#include <fstream>

int main() {
    std::ofstream outputFile("example.txt");

    if (outputFile.is_open()) {
        outputFile << "Aquesta és una línia.\n";
        outputFile << "Aquesta és una altra línia.\n";
        outputFile.close();
    } else {
        std::cout << "No es pot obrir el fitxer" << std::endl;
    }

    return 0;
}
  • std::ofstream s'utilitza per escriure a fitxers.
  • L'operador << s'utilitza per enviar dades al fitxer.

Manipuladors de Flux Personalitzats

C++ permet crear manipuladors personalitzats per formatar la sortida de maneres específiques.

Exemple:

#include <iostream>
#include <iomanip>

std::ostream& customManipulator(std::ostream& os) {
    return os << std::setw(10) << std::setfill('*');
}

int main() {
    std::cout << customManipulator << 123 << std::endl;
    return 0;
}
  • std::setw estableix l'amplada del camp de sortida.
  • std::setfill estableix el caràcter de farciment.

Conclusió

Entendre l'entrada i sortida en C++ és crucial per a una programació efectiva. Aquesta secció ha cobert els conceptes bàsics de l'I/O a la consola, tècniques avançades de formatació, I/O de fitxers i manipuladors de flux personalitzats. Dominar aquests conceptes et permetrà gestionar dades de manera eficient i presentar-les d'una manera amigable per a l'usuari.
