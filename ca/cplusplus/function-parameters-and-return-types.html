Introducció

En C++, les funcions són blocs fonamentals que et permeten encapsular codi en unitats reutilitzables. Entendre els paràmetres de funció i els tipus de retorn és crucial per escriure codi eficient i mantenible. Aquesta secció cobrirà els conceptes bàsics dels paràmetres de funció i els tipus de retorn, progressant cap a conceptes més avançats.

Paràmetres de Funció

Conceptes Bàsics dels Paràmetres de Funció

Els paràmetres de funció són variables passades a una funció per proporcionar dades d'entrada. Aquests paràmetres es especifiquen en la definició de la funció i s'utilitzen dins del cos de la funció.

Exemple:

#include <iostream>

// Funció amb dos paràmetres
void printSum(int a, int b) {
    int sum = a + b;
    std::cout << "Sum: " << sum << std::endl;
}

int main() {
    printSum(5, 3); // Cridant la funció amb els arguments 5 i 3
    return 0;
}
  • int a i int b són paràmetres.
  • printSum(5, 3); crida la funció amb els arguments 5 i 3.

Tipus de Paràmetres

  • Paràmetres per Valor: Passa una còpia de l'argument.
  • Paràmetres per Referència: Passa una referència a l'argument, permetent que la funció modifiqui la variable original.
  • Paràmetres per Referència Const: Passa una referència a l'argument però impedeix la modificació.

Exemple:

#include <iostream>

// Funció amb paràmetre per valor
void incrementValue(int a) {
    a++;
    std::cout << "Dins de la funció (valor): " << a << std::endl;
}

// Funció amb paràmetre per referència
void incrementReference(int &a) {
    a++;
    std::cout << "Dins de la funció (referència): " << a << std::endl;
}

// Funció amb paràmetre per referència const
void printValue(const int &a) {
    std::cout << "Valor: " << a << std::endl;
}

int main() {
    int x = 10;
    incrementValue(x);
    std::cout << "Fora de la funció (valor): " << x << std::endl;

    incrementReference(x);
    std::cout << "Fora de la funció (referència): " << x << std::endl;

    printValue(x);
    return 0;
}
  • incrementValue(int a) utilitza un paràmetre per valor.
  • incrementReference(int &a) utilitza un paràmetre per referència.
  • printValue(const int &a) utilitza un paràmetre per referència const.

Paràmetres per Defecte

C++ et permet especificar valors per defecte per als paràmetres, que s'utilitzen si no es proporciona cap argument.

Exemple:

#include <iostream>

// Funció amb paràmetres per defecte
void greet(std::string name = "Guest") {
    std::cout << "Hello, " << name << "!" << std::endl;
}

int main() {
    greet(); // Utilitza el paràmetre per defecte
    greet("Alice"); // Utilitza l'argument proporcionat
    return 0;
}
  • std::string name = "Guest" estableix un valor per defecte per al paràmetre.

Tipus de Retorn

Conceptes Bàsics dels Tipus de Retorn

El tipus de retorn d'una funció especifica el tipus de valor que la funció retorna. Si una funció no retorna cap valor, el seu tipus de retorn és void.

Exemple:

#include <iostream>

// Funció amb tipus de retorn int
int add(int a, int b) {
    return a + b;
}

// Funció amb tipus de retorn void
void printMessage() {
    std::cout << "Hello, World!" << std::endl;
}

int main() {
    int result = add(5, 3);
    std::cout << "Result: " << result << std::endl;

    printMessage();
    return 0;
}
  • int add(int a, int b) retorna un enter.
  • void printMessage() no retorna cap valor.

Retornant Múltiples Valors

C++ no suporta retornar múltiples valors directament, però pots utilitzar estructures, classes, o std::tuple per aconseguir-ho.

Exemple amb std::tuple:

#include <iostream>
#include <tuple>

// Funció que retorna una tupla
std::tuple<int, int> getMinMax(int a, int b) {
    if (a < b) {
        return std::make_tuple(a, b);
    } else {
        return std::make_tuple(b, a);
    }
}

int main() {
    int min, max;
    std::tie(min, max) = getMinMax(5, 3);
    std::cout << "Min: " << min << ", Max: " << max << std::endl;
    return 0;
}
  • std::tuple<int, int> getMinMax(int a, int b) retorna una tupla que conté dos enters.
  • std::tie(min, max) = getMinMax(5, 3); desempaqueta la tupla en min i max.

Conceptes Avançats

Sobrecàrrega de Funcions

La sobrecàrrega de funcions permet múltiples funcions amb el mateix nom però amb diferents paràmetres.

Exemple:

#include <iostream>

// Funcions sobrecarregades
void display(int a) {
    std::cout << "Integer: " << a << std::endl;
}

void display(double a) {
    std::cout << "Double: " << a << std::endl;
}

void display(std::string a) {
    std::cout << "String: " << a << std::endl;
}

int main() {
    display(5);
    display(3.14);
    display("Hello");
    return 0;
}
  • display està sobrecarregada per manejar int, double, i std::string.

Funcions Inline

Les funcions inline suggereixen al compilador inserir el cos de la funció on es crida, potencialment millorant el rendiment.

Exemple:

#include <iostream>

// Funció inline
inline int square(int x) {
    return x * x;
}

int main() {
    std::cout << "Square of 5: " << square(5) << std::endl;
    return 0;
}
  • inline int square(int x) és una funció inline.

Resum

Entendre els paràmetres de funció i els tipus de retorn és essencial per escriure programes efectius en C++. Aquesta secció ha cobert:

  • Tipus de paràmetres bàsics i avançats.
  • Paràmetres per defecte.
  • Diversos tipus de retorn i mètodes per retornar múltiples valors.
  • Conceptes avançats com la sobrecàrrega de funcions i les funcions inline.

En dominar aquests conceptes, podràs escriure codi en C++ més flexible i eficient.
