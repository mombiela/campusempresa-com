Introducció

La instrucció switch en C++ és una instrucció de control que permet provar una variable per a la igualtat contra una llista de valors. Cada valor s'anomena un cas, i la variable que es canvia es comprova per a cada cas.

Sintaxi Bàsica

La sintaxi bàsica d'una instrucció switch és la següent:

switch (expression) {
    case constant1:
        // codi a executar si l'expressió és igual a constant1;
        break;
    case constant2:
        // codi a executar si l'expressió és igual a constant2;
        break;
    // pots tenir qualsevol nombre d'instruccions case
    default:
        // codi a executar si l'expressió no coincideix amb cap cas
}
  • expression: La variable o valor que es compara.
  • case: Cada cas va seguit d'un valor constant i un dos punts.
  • break: Aquesta instrucció finalitza el cas del switch. Si s'ometeix, el programa continuarà executant el següent cas.
  • default: Això és opcional i es pot utilitzar per executar un bloc de codi si cap dels casos coincideix.

Exemple: Cas Bàsic de Switch

#include <iostream>
using namespace std;

int main() {
    int day = 4;

    switch (day) {
        case 1:
            cout << "Monday";
            break;
        case 2:
            cout << "Tuesday";
            break;
        case 3:
            cout << "Wednesday";
            break;
        case 4:
            cout << "Thursday";
            break;
        case 5:
            cout << "Friday";
            break;
        case 6:
            cout << "Saturday";
            break;
        case 7:
            cout << "Sunday";
            break;
        default:
            cout << "Invalid day";
    }

    return 0;
}

Explicació

  • La variable day es compara amb cada cas.
  • Quan day és igual a 4, el programa imprimeix "Thursday".
  • La instrucció break evita que el programa executi el següent cas.

Ús Avançat

Múltiples Casos per a un Sol Bloc

Pots agrupar múltiples casos junts si han d'executar el mateix bloc de codi.

#include <iostream>
using namespace std;

int main() {
    char grade = 'B';

    switch (grade) {
        case 'A':
        case 'B':
        case 'C':
            cout << "You passed.";
            break;
        case 'D':
        case 'F':
            cout << "You failed.";
            break;
        default:
            cout << "Invalid grade.";
    }

    return 0;
}

Explicació

  • Els casos A, B i C estan agrupats per executar el mateix bloc de codi.
  • De manera similar, els casos D i F estan agrupats.

Instruccions Switch Imbricades

Les instruccions switch es poden imbricar entre elles.

#include <iostream>
using namespace std;

int main() {
    int num1 = 1, num2 = 2;

    switch (num1) {
        case 1:
            switch (num2) {
                case 2:
                    cout << "num1 is 1 and num2 is 2";
                    break;
                default:
                    cout << "num1 is 1 but num2 is not 2";
            }
            break;
        default:
            cout << "num1 is not 1";
    }

    return 0;
}

Explicació

  • El switch extern comprova el valor de num1.
  • Si num1 és 1, el switch intern comprova el valor de num2.

Millors Pràctiques

  • Inclou sempre un cas default per gestionar valors inesperats.
  • Utilitza instruccions break per evitar la caiguda a menys que es desitgi intencionadament.
  • Mantingues els casos del switch simples i evita la lògica complexa dins d'ells.

Conclusió

La instrucció switch és una eina poderosa per controlar el flux d'un programa basat en el valor d'una variable. És particularment útil quan tens múltiples condicions que depenen de la mateixa variable. En comprendre i utilitzar la instrucció switch, pots escriure codi més llegible i eficient.
