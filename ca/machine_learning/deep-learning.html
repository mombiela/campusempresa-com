Introducció a Deep Learning

Deep Learning és una subdisciplina del Machine Learning que es basa en xarxes neuronals artificials amb múltiples capes (també conegudes com a xarxes neuronals profundes). Aquestes xarxes són capaces d'aprendre representacions de dades amb múltiples nivells d'abstracció, cosa que les fa especialment útils per a tasques complexes com el reconeixement d'imatges, el processament del llenguatge natural i la conducció autònoma.

Conceptes Clau

  • Xarxa Neuronal Artificial (ANN): Model computacional inspirat en l'estructura del cervell humà, compost per neurones artificials.
  • Capes (Layers): Conjunt de neurones en una xarxa neuronal. Les capes poden ser d'entrada, ocultes o de sortida.
  • Neurones: Unitats bàsiques d'una xarxa neuronal que processen i transmeten informació.
  • Funció d'Activació: Funció matemàtica que determina si una neurona ha d'activar-se o no.
  • Backpropagation: Algorisme utilitzat per ajustar els pesos de les connexions en la xarxa neuronal durant l'entrenament.
  • Overfitting: Problema on el model s'ajusta massa a les dades d'entrenament i no generalitza bé a dades noves.

Arquitectures de Xarxes Neuronals

Xarxes Neuronals Feedforward

Les xarxes neuronals feedforward són el tipus més simple de xarxa neuronal artificial. La informació es mou en una sola direcció, des de la capa d'entrada, a través de les capes ocultes, fins a la capa de sortida.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# Crear un model seqüencial
model = Sequential()

# Afegir una capa d'entrada amb 10 neurones
model.add(Dense(10, input_shape=(784,), activation='relu'))

# Afegir una capa oculta amb 10 neurones
model.add(Dense(10, activation='relu'))

# Afegir una capa de sortida amb 10 neurones
model.add(Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

Xarxes Neuronals Convolucionals (CNN)

Les CNN són especialment útils per a tasques de processament d'imatges. Utilitzen capes convolucionals que apliquen filtres a les entrades per extreure característiques importants.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

# Crear un model seqüencial
model = Sequential()

# Afegir una capa convolucional amb 32 filtres i una mida de kernel de 3x3
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))

# Afegir una capa de pooling
model.add(MaxPooling2D(pool_size=(2, 2)))

# Aplanar la sortida
model.add(Flatten())

# Afegir una capa densa
model.add(Dense(128, activation='relu'))

# Afegir una capa de sortida
model.add(Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

Xarxes Neuronals Recurrentes (RNN)

Les RNN són adequades per a dades seqüencials com sèries temporals i text. Utilitzen connexions recurrents per mantenir informació d'estats anteriors.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import SimpleRNN, Dense

# Crear un model seqüencial
model = Sequential()

# Afegir una capa recurrent
model.add(SimpleRNN(50, input_shape=(100, 1), activation='relu'))

# Afegir una capa de sortida
model.add(Dense(1))

# Compilar el model
model.compile(optimizer='adam', loss='mean_squared_error')

# Resum del model
model.summary()

Comparació d'Arquitectures

| Arquitectura | Aplicacions Principals | Avantatges | Desavantatges | |--------------|------------------------|------------|---------------| | Feedforward | Classificació bàsica, regressió | Simplicitat, fàcil d'implementar | No captura relacions temporals o espacials | | CNN | Reconeixement d'imatges, vídeo | Captura característiques espacials, eficient en processament d'imatges | Requereix gran quantitat de dades i poder computacional | | RNN | Sèries temporals, processament del llenguatge natural | Captura relacions temporals, adequat per a dades seqüencials | Problemes de gradient, difícil d'entrenar |

Exercicis Pràctics

  1. Implementar una xarxa neuronal feedforward: Crear i entrenar una xarxa neuronal simple per classificar dígits escrits a mà utilitzant el dataset MNIST.
  2. Construir una CNN: Implementar una xarxa neuronal convolucional per classificar imatges de moda utilitzant el dataset Fashion MNIST.
  3. Desenvolupar una RNN: Crear una xarxa neuronal recurrent per predir la temperatura diària utilitzant un dataset de sèries temporals.

Conclusió

Deep Learning és una eina poderosa dins del Machine Learning que permet abordar problemes complexos mitjançant l'ús de xarxes neuronals profundes. Comprendre les diferents arquitectures i les seves aplicacions és crucial per seleccionar el model adequat per a cada tasca. Amb la pràctica i l'experimentació, els professionals poden aprofitar el potencial de Deep Learning per resoldre problemes avançats en diversos dominis.
