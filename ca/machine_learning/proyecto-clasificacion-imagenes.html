Introducció

En aquest projecte, ens enfocarem en la classificació d'imatges utilitzant tècniques de Machine Learning. La classificació d'imatges és una tasca fonamental en el camp de la visió per computadora i té aplicacions en diverses àrees com la medicina, la seguretat i l'entreteniment.

Objectius del Projecte

  • Comprendre els conceptes bàsics de la classificació d'imatges.
  • Aprendre a preprocessar i preparar dades d'imatges per a models de Machine Learning.
  • Implementar i entrenar un model de classificació d'imatges utilitzant una xarxa neuronal convolucional (CNN).
  • Avaluar el rendiment del model i ajustar els seus hiperparàmetres per millorar la precisió.

Conceptes Clau

  • Classificació d'Imatges: Procés d'assignar una etiqueta a una imatge d'un conjunt predefinit de categories.
  • Xarxa Neuronal Convolucional (CNN): Tipus de xarxa neuronal especialment eficaç per a tasques de visió per computadora.
  • Preprocessament d'Imatges: Tècniques per preparar imatges abans d'entrenar el model, com l'escalat i la normalització.
  • Dataset: Conjunt de dades que inclou imatges i les seves etiquetes corresponents.

Eines i Biblioteques

  • Python: Llenguatge de programació utilitzat per implementar el projecte.
  • TensorFlow/Keras: Biblioteca de Machine Learning que facilita la creació i entrenament de models de xarxes neuronals.
  • NumPy: Biblioteca per a operacions numèriques en Python.
  • Matplotlib: Biblioteca per a la visualització de dades.

Dataset

Per a aquest projecte, utilitzarem el dataset CIFAR-10, que conté 60,000 imatges de 32x32 píxels en 10 classes diferents.

| Classe | Descripció | |---------------|-----------------------| | Avió | Imatges d'avions | | Automòbil | Imatges d'automòbils | | Ocell | Imatges d'ocells | | Gat | Imatges de gats | | Cérvol | Imatges de cérvols | | Gos | Imatges de gossos | | Granota | Imatges de granotes | | Cavall | Imatges de cavalls | | Vaixell | Imatges de vaixells | | Camió | Imatges de camions |

Preprocessament de Dades

Abans d'entrenar el nostre model, és crucial preprocessar les dades d'imatges:

  1. Càrrega del Dataset:

    from tensorflow.keras.datasets import cifar10
(x_train, y_train), (x_test, y_test) = cifar10.load_data()

  2. Normalització: Escalar els valors de píxels a un rang de 0 a 1.

    x_train = x_train.astype('float32') / 255.0
x_test = x_test.astype('float32') / 255.0

  3. Codificació d'Etiquetes: Convertir les etiquetes a format one-hot.

    from tensorflow.keras.utils import to_categorical
y_train = to_categorical(y_train, 10)
y_test = to_categorical(y_test, 10)

Construcció del Model

Implementarem una xarxa neuronal convolucional (CNN) utilitzant Keras:

from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

model = Sequential([
    Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)),
    MaxPooling2D((2, 2)),
    Conv2D(64, (3, 3), activation='relu'),
    MaxPooling2D((2, 2)),
    Flatten(),
    Dense(64, activation='relu'),
    Dense(10, activation='softmax')
])

Compilació i Entrenament del Model

Compilem el model i l'entrenem amb les dades preprocessades:

model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

history = model.fit(x_train, y_train, epochs=10, validation_data=(x_test, y_test))

Avaluació del Model

Avaluem el rendiment del model en el conjunt de prova:

test_loss, test_acc = model.evaluate(x_test, y_test)
print(f'Test accuracy: {test_acc}')

Ajust d'Hiperparàmetres

Per millorar la precisió del model, podem ajustar diversos hiperparàmetres com el nombre de capes, la mida dels filtres, i la taxa d'aprenentatge.

Conclusió

En aquest projecte, hem après a classificar imatges utilitzant una xarxa neuronal convolucional. Hem cobert des del preprocessament de dades fins a l'avaluació i ajust del model. La classificació d'imatges és una tasca complexa, però amb les eines i tècniques adequades, podem construir models precisos i eficients.

Aquest projecte proporciona una base sòlida per abordar problemes més avançats en la visió per computadora i el Machine Learning.
