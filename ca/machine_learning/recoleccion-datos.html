Introducció

La recol·lecció de dades és una fase crucial en qualsevol projecte de Machine Learning. La qualitat i quantitat de les dades recollides poden determinar l'èxit o fracàs d'un model. En aquesta secció, aprendrem sobre les diferents fonts de dades, mètodes de recol·lecció, i com preparar les dades per al seu ús en models de Machine Learning.

Fonts de Dades

Existeixen diverses fonts de dades que poden ser utilitzades en projectes de Machine Learning. Algunes de les més comunes inclouen:

  • Bases de dades relacionals: MySQL, PostgreSQL.
  • APIs: Twitter API, Google Maps API.
  • Arxius: CSV, JSON, Excel.
  • Web Scraping: BeautifulSoup, Scrapy.
  • Sensors i dispositius IoT: Sensors de temperatura, acceleròmetres.

Exemple de Connexió a una Base de Dades Relacional

import mysql.connector

# Connexió a la base de dades
conn = mysql.connector.connect(
    host="localhost",
    user="el_teu_usuari",
    password="la_teva_contrasenya",
    database="nom_de_la_base_de_dades"
)

# Creació d'un cursor per executar consultes
cursor = conn.cursor()

# Execució d'una consulta
cursor.execute("SELECT * FROM la_teva_taula")

# Recuperació de les dades
resultats = cursor.fetchall()

# Tancament de la connexió
conn.close()

# Mostrar els resultats
for fila in resultats:
    print(fila)

Mètodes de Recol·lecció de Dades

Depenent del tipus de dades i la font, els mètodes de recol·lecció poden variar. Aquí es presenten alguns dels mètodes més comuns:

  • Extracció de dades de bases de dades: Utilitzant SQL per consultar i extreure dades.
  • Consum d'APIs: Utilitzant llibreries com requests en Python per fer peticions HTTP.
  • Web Scraping: Utilitzant eines com BeautifulSoup o Scrapy per extreure dades de pàgines web.
  • Lectura d'arxius: Utilitzant llibreries com pandas per llegir arxius CSV, Excel, etc.

Exemple de Consum d'una API

import requests

# URL de l'API
url = "https://api.example.com/data"

# Fer una petició GET a l'API
response = requests.get(url)

# Verificar l'estat de la resposta
if response.status_code == 200:
    # Parsejar la resposta JSON
    dades = response.json()
    print(dades)
else:
    print("Error en la petició:", response.status_code)

Preparació de Dades

Un cop recollides les dades, és necessari preparar-les abans d'utilitzar-les en models de Machine Learning. La preparació de dades inclou:

  • Neteja de dades: Eliminació de valors nuls, duplicats, i correcció d'errors.
  • Transformació de dades: Normalització, estandardització, i codificació de variables categòriques.
  • Divisió de dades: Separació en conjunts d'entrenament i prova.

Exemple de Neteja i Transformació de Dades

import pandas as pd
from sklearn.preprocessing import StandardScaler, OneHotEncoder

# Carregar dades des d'un arxiu CSV
df = pd.read_csv("dades.csv")

# Eliminar files amb valors nuls
df = df.dropna()

# Eliminar duplicats
df = df.drop_duplicates()

# Normalitzar dades numèriques
scaler = StandardScaler()
df[['columna_numerica']] = scaler.fit_transform(df[['columna_numerica']])

# Codificar variables categòriques
encoder = OneHotEncoder()
categorical_data = encoder.fit_transform(df[['columna_categorica']]).toarray()

# Afegir les columnes codificades al DataFrame original
df = df.join(pd.DataFrame(categorical_data, columns=encoder.get_feature_names_out()))

# Mostrar els primers registres del DataFrame preparat
print(df.head())

Conclusió

La recol·lecció de dades és una etapa fonamental en el desenvolupament de projectes de Machine Learning. L'elecció de les fonts de dades i els mètodes de recol·lecció adequats, així com una correcta preparació de les dades, són essencials per construir models precisos i eficients. En les properes seccions del curs, aprofundirem en tècniques avançades de preparació de dades i en la implementació de models de Machine Learning.
