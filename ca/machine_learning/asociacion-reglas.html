Introducció

L'associació de regles és una tècnica de mineria de dades que s'utilitza per descobrir relacions interessants entre variables en grans bases de dades. És especialment útil en l'anàlisi de cistelles de mercat, on es busca identificar productes que es compren junts amb freqüència.

Conceptes Clau

  • Itemset: Un conjunt d'un o més ítems.
  • Freqüència d'un Itemset: El nombre de transaccions en què apareix un itemset.
  • Suport (Support): La proporció de transaccions en la base de dades que contenen l'itemset.
  • Confiança (Confidence): La proporció de transaccions que contenen l'itemset A que també contenen l'itemset B.
  • Lift: La raó entre la confiança observada i la confiança esperada si A i B fossin independents.

Exemple d'Associació de Regles

Per il·lustrar aquests conceptes, considerem una base de dades de transaccions en un supermercat:

| Transacció | Ítems Comprats | |------------|------------------------------| | 1 | Llet, Pa, Mantega | | 2 | Llet, Pa | | 3 | Llet, Mantega | | 4 | Pa, Mantega | | 5 | Llet, Pa, Mantega, Formatge |

Càlcul del Suport

El suport d'un itemset es calcula com:

\[ \text{Suport}(A) = \frac{\text{Nombre de transaccions que contenen A}}{\text{Nombre total de transaccions}} \]

Per exemple, el suport de l'itemset {Llet, Pa} és:

\[ \text{Suport}({Llet, Pa}) = \frac{3}{5} = 0.6 \]

Càlcul de la Confiança

La confiança d'una regla A → B es calcula com:

\[ \text{Confiança}(A \rightarrow B) = \frac{\text{Nombre de transaccions que contenen A i B}}{\text{Nombre de transaccions que contenen A}} \]

Per exemple, la confiança de la regla {Llet} → {Pa} és:

\[ \text{Confiança}({Llet} \rightarrow {Pa}) = \frac{3}{4} = 0.75 \]

Càlcul del Lift

El lift d'una regla A → B es calcula com:

\[ \text{Lift}(A \rightarrow B) = \frac{\text{Confiança}(A \rightarrow B)}{\text{Suport}(B)} \]

Per exemple, el lift de la regla {Llet} → {Pa} és:

\[ \text{Lift}({Llet} \rightarrow {Pa}) = \frac{0.75}{0.8} = 0.9375 \]

Implementació en Python

A continuació, es mostra un exemple de com implementar l'associació de regles utilitzant la biblioteca mlxtend en Python.

# Importar les biblioteques necessàries
import pandas as pd
from mlxtend.frequent_patterns import apriori, association_rules

# Crear un DataFrame amb les transaccions
data = {'Llet': [1, 1, 1, 0, 1],
        'Pa': [1, 1, 0, 1, 1],
        'Mantega': [1, 0, 1, 1, 1],
        'Formatge': [0, 0, 0, 0, 1]}
df = pd.DataFrame(data)

# Aplicar l'algoritme Apriori per trobar itemsets freqüents
frequent_itemsets = apriori(df, min_support=0.6, use_colnames=True)

# Generar les regles d'associació
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.7)

# Mostrar les regles
print(rules)

Explicació del Codi

  1. Importació de Biblioteques: S'importen pandas per a la manipulació de dades i mlxtend per a la implementació de l'algoritme Apriori.
  2. Creació del DataFrame: Es crea un DataFrame amb les transaccions, on cada columna representa un ítem i cada fila una transacció.
  3. Algoritme Apriori: S'aplica l'algoritme Apriori per trobar itemsets freqüents amb un suport mínim de 0.6.
  4. Generació de Regles: Es generen les regles d'associació amb una confiança mínima de 0.7.
  5. Mostrar Regles: S'imprimeixen les regles generades.

Conclusió

L'associació de regles és una tècnica poderosa per descobrir relacions ocultes en grans conjunts de dades. A través de conceptes com suport, confiança i lift, podem identificar patrons significatius que poden ser utilitzats per prendre decisions informades en diverses aplicacions, com la venda al detall i el màrqueting. La implementació en Python amb biblioteques com mlxtend facilita el procés de descobriment d'aquestes regles, permetent als professionals del sector aplicar aquestes tècniques de manera eficient.
