Introducció a Ensemble Learning

Ensemble Learning és una tècnica en Machine Learning que combina múltiples models per millorar la precisió i la robustesa de les prediccions. La idea principal és que un conjunt de models (un "ensemble") pot superar el rendiment d'un sol model en reduir el risc d'errors individuals.

Tipus de Ensemble Learning

Hi ha diversos mètodes de Ensemble Learning, cadascun amb les seves pròpies característiques i aplicacions. Els més comuns són:

  • Bagging (Bootstrap Aggregating)
  • Boosting
  • Stacking

Bagging (Bootstrap Aggregating)

Bagging és una tècnica que millora la precisió del model en reduir la variabilitat. Es basa en la idea d'entrenar múltiples models en diferents subconjunts de dades i després promigiar les seves prediccions.

  • Procés:

    1. Crear múltiples subconjunts de dades mitjançant mostreig amb reemplaçament.
    2. Entrenar un model en cada subconjunt.
    3. Promigiar les prediccions de tots els models.

  • Exemple de codi en Python:

    from sklearn.ensemble import BaggingClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# Carregar dades
iris = load_iris()
X, y = iris.data, iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear el model Bagging
bagging = BaggingClassifier(base_estimator=DecisionTreeClassifier(), n_estimators=50, random_state=42)
bagging.fit(X_train, y_train)

# Prediccions i avaluació
y_pred = bagging.predict(X_test)
print(f'Accuracy: {accuracy_score(y_test, y_pred)}')

Boosting

Boosting és una tècnica que ajusta seqüencialment models febles, corregint els errors dels models anteriors. L'objectiu és convertir models febles en un model fort.

  • Procés:

    1. Entrenar un model en les dades originals.
    2. Ajustar el següent model en els errors del model anterior.
    3. Repetir el procés i combinar els models.

  • Exemple de codi en Python:

    from sklearn.ensemble import AdaBoostClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# Carregar dades
iris = load_iris()
X, y = iris.data, iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear el model Boosting
boosting = AdaBoostClassifier(base_estimator=DecisionTreeClassifier(), n_estimators=50, random_state=42)
boosting.fit(X_train, y_train)

# Prediccions i avaluació
y_pred = boosting.predict(X_test)
print(f'Accuracy: {accuracy_score(y_test, y_pred)}')

Stacking

Stacking combina múltiples models mitjançant un model de nivell superior (meta-model) que aprèn a combinar les prediccions dels models base.

  • Procés:

    1. Entrenar múltiples models base en les dades originals.
    2. Usar les prediccions dels models base com a característiques per al meta-model.
    3. Entrenar el meta-model en aquestes noves característiques.

  • Exemple de codi en Python:

    from sklearn.ensemble import StackingClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.tree import DecisionTreeClassifier
from sklearn.svm import SVC
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# Carregar dades
iris = load_iris()
X, y = iris.data, iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear els models base
estimators = [
    ('dt', DecisionTreeClassifier()),
    ('svc', SVC(probability=True))
]

# Crear el model Stacking
stacking = StackingClassifier(estimators=estimators, final_estimator=LogisticRegression())
stacking.fit(X_train, y_train)

# Prediccions i avaluació
y_pred = stacking.predict(X_test)
print(f'Accuracy: {accuracy_score(y_test, y_pred)}')

Comparació de Mètodes

| Mètode | Avantatges | Desavantatges | |----------|-------------------------------------------------|-----------------------------------------------| | Bagging | Redueix la variància, millora l'estabilitat | No redueix el biaix | | Boosting | Redueix tant el biaix com la variància | Pot sobreajustar si no es controla | | Stacking | Combina múltiples models de manera flexible | Més complex, requereix més temps d'entrenament |

Conclusió

Ensemble Learning és una poderosa tècnica en Machine Learning que pot millorar significativament el rendiment dels models. En combinar múltiples models, es poden reduir els errors i augmentar la precisió de les prediccions. Cada mètode de Ensemble Learning té les seves pròpies avantatges i desavantatges, i l'elecció del mètode adequat depèn del problema específic i de les dades disponibles.
