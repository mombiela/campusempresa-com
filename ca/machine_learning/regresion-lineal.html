Introducció a la Regressió Lineal

La regressió lineal és un dels algoritmes més simples i àmpliament utilitzats en Machine Learning. S'utilitza per modelar la relació entre una variable dependent i una o més variables independents. En aquest tema, aprendrem els conceptes bàsics de la regressió lineal, com implementar-la i com avaluar el seu rendiment.

Conceptes Clau

  • Variable Dependent (Y): La variable que estem intentant predir o explicar.
  • Variable Independent (X): Les variables que utilitzem per fer prediccions sobre la variable dependent.
  • Coeficients (β0, β1, ..., βn): Els paràmetres que el model aprèn durant l'entrenament.
  • Error (ε): La diferència entre el valor real i el valor predit pel model.

Fórmula de la Regressió Lineal

La fórmula general de la regressió lineal és:

\[ Y = β0 + β1X1 + β2X2 + ... + βnXn + ε \]

On:

  • \( Y \) és la variable dependent.
  • \( X1, X2, ..., Xn \) són les variables independents.
  • \( β0 \) és l'intercepto.
  • \( β1, β2, ..., βn \) són els coeficients de les variables independents.
  • \( ε \) és l'error.

Implementació en Python

A continuació, es mostra un exemple de com implementar una regressió lineal simple utilitzant Python i la biblioteca scikit-learn.

Exemple de Codi

import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
Y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió lineal
model = LinearRegression()

# Entrenar el model
model.fit(X, Y)

# Fer prediccions
Y_pred = model.predict(X)

# Visualitzar els resultats
plt.scatter(X, Y, color='blue')  # Dades reals
plt.plot(X, Y_pred, color='red')  # Línia de regressió
plt.xlabel('Variable Independent (X)')
plt.ylabel('Variable Dependent (Y)')
plt.title('Regressió Lineal Simple')
plt.show()

Explicació del Codi

  1. Importació de Biblioteques: Importem numpy per manejar les dades i matplotlib per a la visualització. scikit-learn s'utilitza per crear i entrenar el model de regressió lineal.
  2. Dades d'Exemple: Creem un conjunt de dades d'exemple amb una variable independent X i una variable dependent Y.
  3. Creació del Model: Utilitzem LinearRegression de scikit-learn per crear el model de regressió lineal.
  4. Entrenament del Model: Ajustem el model a les dades utilitzant el mètode fit.
  5. Prediccions: Utilitzem el model entrenat per fer prediccions sobre les dades d'entrada.
  6. Visualització: Grafiquem les dades reals i la línia de regressió per visualitzar l'ajust del model.

Avaluació del Model

Per avaluar el rendiment d'un model de regressió lineal, podem utilitzar diverses mètriques, com:

  • Coeficient de Determinació (R²): Indica la proporció de la variància en la variable dependent que és previsible a partir de les variables independents.
  • Error Quadràtic Mitjà (MSE): La mitjana dels quadrats dels errors, que mesura la magnitud mitjana dels errors.

Exemple d'Avaluació

from sklearn.metrics import mean_squared_error, r2_score

# Calcular el MSE
mse = mean_squared_error(Y, Y_pred)

# Calcular el R²
r2 = r2_score(Y, Y_pred)

print(f"Mean Squared Error: {mse}")
print(f"R² Score: {r2}")

Conclusió

La regressió lineal és una tècnica fonamental en Machine Learning que permet modelar relacions lineals entre variables. A través d'aquest tema, hem après els conceptes bàsics, com implementar una regressió lineal en Python i com avaluar el seu rendiment. Amb aquesta base, estaràs preparat per explorar models més complexos i tècniques avançades en Machine Learning.
