Introducció

La neteja de dades és una etapa crucial en el procés de Machine Learning. Consisteix a identificar i corregir errors, inconsistències i valors faltants en les dades per millorar la qualitat i la precisió dels models predictius. Aquest procés assegura que les dades siguin adequades per a l'anàlisi i la construcció de models.

Importància de la Neteja de Dades

  • Millora la qualitat del model: Dades netes i precises condueixen a models més fiables i precisos.
  • Reducció d'errors: Minimitza la presència de valors atípics i errors que poden distorsionar els resultats.
  • Eficiència: Facilita el procés d'anàlisi i modelatge, estalviant temps i recursos.

Passos en la Neteja de Dades

1. Identificació de Valors Faltants

  • Detecció: Identificar les columnes i files que contenen valors faltants.
  • Tractament:
    • Eliminació: Treure files o columnes amb valors faltants.
    • Imputació: Omplir valors faltants amb la mitjana, mediana, moda o mitjançant tècniques més avançades com KNN.
import pandas as pd
from sklearn.impute import SimpleImputer

# Carregar dades
data = pd.read_csv('data.csv')

# Identificar valors faltants
print(data.isnull().sum())

# Imputació amb la mitjana
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data)

2. Detecció i Tractament de Valors Atípics

  • Detecció: Utilitzar mètodes estadístics com el rang interquartílic (IQR) o visualitzacions com boxplots.
  • Tractament:
    • Eliminació: Treure els valors atípics.
    • Transformació: Aplicar transformacions per reduir l'impacte dels valors atípics.
import numpy as np

# Detecció de valors atípics usant IQR
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1

# Filtrar valors atípics
data_filtered = data[~((data < (Q1 - 1.5 * IQR)) | (data > (Q3 + 1.5 * IQR))).any(axis=1)]

3. Normalització i Escalat de Dades

  • Normalització: Escalar les dades perquè tinguin una mitjana de 0 i una desviació estàndard d'1.
  • Escalat: Ajustar les dades perquè estiguin en un rang específic, com [0, 1].
from sklearn.preprocessing import StandardScaler, MinMaxScaler

# Normalització
scaler = StandardScaler()
data_normalized = scaler.fit_transform(data)

# Escalat
scaler = MinMaxScaler()
data_scaled = scaler.fit_transform(data)

4. Maneig de Dades Duplicades

  • Detecció: Identificar files duplicades en el conjunt de dades.
  • Eliminació: Treure les files duplicades per evitar redundàncies.
# Detecció de duplicats
duplicates = data.duplicated()

# Eliminació de duplicats
data_cleaned = data.drop_duplicates()

Eines i Llibreries Comunes

  • Pandas: Per a manipulació i anàlisi de dades.
  • NumPy: Per a operacions numèriques i maneig de matrius.
  • Scikit-learn: Per a preprocessament i modelatge de dades.

Conclusió

La neteja de dades és un pas essencial en el procés de Machine Learning que no s'ha de subestimar. A través de la identificació i correcció de valors faltants, atípics i duplicats, així com la normalització i escalat de dades, podem assegurar que els nostres models siguin més precisos i eficients. Utilitzar eines i llibreries adequades facilita aquest procés i millora la qualitat de les nostres anàlisis i prediccions.
