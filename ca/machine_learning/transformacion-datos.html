Introducció

La transformació de dades és un pas crucial en el procés de Machine Learning. Consisteix a convertir dades brutes en un format adequat per a l'anàlisi i la modelització. Aquest procés pot incloure la neteja de dades, la normalització, la codificació de variables categòriques, entre altres.

Importància de la Transformació de Dades

  • Millora la qualitat de les dades: Elimina inconsistències i errors.
  • Facilita la modelització: Transforma les dades en un format que els algoritmes de Machine Learning poden processar eficientment.
  • Augmenta la precisió del model: Dades ben transformades poden millorar significativament el rendiment del model.

Tipus de Transformacions de Dades

Normalització i Estandardització

  • Normalització: Escala les dades perquè estiguin en un rang específic, generalment [0, 1].
  • Estandardització: Transforma les dades perquè tinguin una mitjana de 0 i una desviació estàndard d'1.
from sklearn.preprocessing import MinMaxScaler, StandardScaler

# Normalització
scaler = MinMaxScaler()
data_normalized = scaler.fit_transform(data)

# Estandardització
scaler = StandardScaler()
data_standardized = scaler.fit_transform(data)

Codificació de Variables Categòriques

  • One-Hot Encoding: Converteix variables categòriques en una sèrie de variables binàries.
  • Label Encoding: Assigna un número enter a cada categoria.
from sklearn.preprocessing import OneHotEncoder, LabelEncoder

# One-Hot Encoding
encoder = OneHotEncoder()
data_encoded = encoder.fit_transform(data_categorical)

# Label Encoding
encoder = LabelEncoder()
data_encoded = encoder.fit_transform(data_categorical)

Maneig de Valors Faltants

  • Imputació: Omple els valors faltants amb la mitjana, mediana o moda.
  • Eliminació: Elimina files o columnes amb valors faltants.
from sklearn.impute import SimpleImputer

# Imputació amb la mitjana
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data_with_missing_values)

Transformacions Logarítmiques i de Potència

  • Transformació Logarítmica: Utilitzada per reduir l'asimetria de la distribució de les dades.
  • Transformació de Potència: Utilitzada per estabilitzar la variància i fer que les dades s'assemblin més a una distribució normal.
import numpy as np

# Transformació Logarítmica
data_log_transformed = np.log(data)

# Transformació de Potència
from sklearn.preprocessing import PowerTransformer
pt = PowerTransformer()
data_power_transformed = pt.fit_transform(data)

Exemple Complet de Transformació de Dades

import pandas as pd
from sklearn.preprocessing import MinMaxScaler, OneHotEncoder, SimpleImputer

# Carregar dades
data = pd.read_csv('data.csv')

# Maneig de valors faltants
imputer = SimpleImputer(strategy='mean')
data_imputed = imputer.fit_transform(data)

# Normalització
scaler = MinMaxScaler()
data_normalized = scaler.fit_transform(data_imputed)

# Codificació de variables categòriques
encoder = OneHotEncoder()
data_encoded = encoder.fit_transform(data_normalized)

# Convertir a DataFrame per facilitar la visualització
data_transformed = pd.DataFrame(data_encoded.toarray())
print(data_transformed.head())

Conclusió

La transformació de dades és un pas essencial en el pipeline de Machine Learning. Assegura que les dades estiguin en un format adequat per a l'anàlisi i la modelització, millorant així la qualitat i la precisió dels models. En dominar les tècniques de transformació de dades, els professionals poden construir models més robustos i eficients.
