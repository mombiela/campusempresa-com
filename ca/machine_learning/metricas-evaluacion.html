Introducció

En Machine Learning, les mètriques d'avaluació són crucials per mesurar el rendiment dels models. Aquestes mètriques ens permeten entendre com de bé està funcionant un model i comparar-lo amb altres models. En aquest tema, explorarem les mètriques més comunes utilitzades per avaluar models de classificació i regressió.

Mètriques per a Models de Classificació

Precisió (Accuracy)

  • Definició: La proporció de prediccions correctes sobre el total de prediccions.
  • Fórmula: \[ \text{Precisió} = \frac{\text{Nombre de prediccions correctes}}{\text{Nombre total de prediccions}} \]

Precisió (Precision)

  • Definició: La proporció de veritables positius sobre el total de prediccions positives.
  • Fórmula: \[ \text{Precisió} = \frac{\text{Veritables Positius}}{\text{Veritables Positius} + \text{Falsos Positius}} \]

Recall (Sensibilitat)

  • Definició: La proporció de veritables positius sobre el total de positius reals.
  • Fórmula: \[ \text{Recall} = \frac{\text{Veritables Positius}}{\text{Veritables Positius} + \text{Falsos Negatius}} \]

F1-Score

  • Definició: La mitjana harmònica de la precisió i el recall.
  • Fórmula: \[ \text{F1-Score} = 2 \times \frac{\text{Precisió} \times \text{Recall}}{\text{Precisió} + \text{Recall}} \]

Exemple de Codi

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

# Suposem que tenim les següents etiquetes veritables i prediccions
y_true = [0, 1, 1, 0, 1, 0, 1, 1]
y_pred = [0, 1, 0, 0, 1, 0, 1, 1]

# Calcular les mètriques
accuracy = accuracy_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)

print(f"Precisió: {accuracy}")
print(f"Precisió: {precision}")
print(f"Recall: {recall}")
print(f"F1-Score: {f1}")

Mètriques per a Models de Regressió

Error Absolut Mitjà (MAE)

  • Definició: La mitjana de les diferències absolutes entre les prediccions i els valors reals.
  • Fórmula: \[ \text{MAE} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i| \]

Error Quadràtic Mitjà (MSE)

  • Definició: La mitjana dels quadrats de les diferències entre les prediccions i els valors reals.
  • Fórmula: \[ \text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

Arrel de l'Error Quadràtic Mitjà (RMSE)

  • Definició: L'arrel quadrada del MSE.
  • Fórmula: \[ \text{RMSE} = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2} \]

Coeficient de Determinació (R²)

  • Definició: La proporció de la variància en la variable dependent que és previsible a partir de les variables independents.
  • Fórmula: \[ R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}i)^2}{\sum{i=1}^{n} (y_i - \bar{y})^2} \]

Exemple de Codi

from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
import numpy as np

# Suposem que tenim les següents etiquetes veritables i prediccions
y_true = np.array([3.0, -0.5, 2.0, 7.0])
y_pred = np.array([2.5, 0.0, 2.0, 8.0])

# Calcular les mètriques
mae = mean_absolute_error(y_true, y_pred)
mse = mean_squared_error(y_true, y_pred)
rmse = np.sqrt(mse)
r2 = r2_score(y_true, y_pred)

print(f"MAE: {mae}")
print(f"MSE: {mse}")
print(f"RMSE: {rmse}")
print(f"R²: {r2}")

Comparació de Mètriques

| Mètrica | Tipus de Model | Interpretació | |---------------|-----------------|-------------------------------------------------------------------------------| | Precisió | Classificació | Proporció de prediccions correctes. | | Precisió | Classificació | Proporció de veritables positius sobre el total de prediccions positives. | | Recall | Classificació | Proporció de veritables positius sobre el total de positius reals. | | F1-Score | Classificació | Mitjana harmònica de la precisió i el recall. | | MAE | Regressió | Mitjana de les diferències absolutes entre les prediccions i els valors reals. | | MSE | Regressió | Mitjana dels quadrats de les diferències entre les prediccions i els valors reals. | | RMSE | Regressió | Arrel quadrada del MSE. | | R² | Regressió | Proporció de la variància explicada pel model. |

Conclusió

Les mètriques d'avaluació són essencials per entendre el rendiment dels models de Machine Learning. L'elecció de la mètrica adequada depèn del tipus de problema i dels objectius específics del projecte. És fonamental conèixer i aplicar correctament aquestes mètriques per desenvolupar models efectius i prendre decisions informades.
