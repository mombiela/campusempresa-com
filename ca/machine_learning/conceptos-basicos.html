Introducció a Machine Learning

Machine Learning (ML) és una branca de la intel·ligència artificial (IA) que se centra en el desenvolupament d'algoritmes i models que permeten als ordinadors aprendre i fer prediccions o decisions basades en dades. A continuació, es presenten els conceptes bàsics que tot professional ha de conèixer per iniciar-se en el món del Machine Learning.

Conceptes Clau

  • Dades: La base de qualsevol model de Machine Learning. Les dades poden ser estructurades (taules, bases de dades) o no estructurades (imatges, text).
  • Característiques (Features): Atributs o propietats individuals de les dades que s'utilitzen per a la predicció.
  • Etiquetes (Labels): La sortida o resultat esperat en un problema supervisat.
  • Model: Una representació matemàtica que s'entrena usant dades per fer prediccions o decisions.
  • Entrenament: El procés d'ajustar un model a les dades.
  • Validació: Avaluar el rendiment del model en un conjunt de dades diferent al d'entrenament.
  • Prova (Testing): Avaluar el rendiment final del model en un conjunt de dades completament nou.
  • Sobreajustament (Overfitting): Quan un model s'ajusta massa a les dades d'entrenament i no generalitza bé a noves dades.
  • Subajustament (Underfitting): Quan un model és massa simple i no captura la complexitat de les dades.

Tipus d'Aprenentatge

Aprenentatge Supervisat

En l'aprenentatge supervisat, el model s'entrena amb un conjunt de dades etiquetades. L'objectiu és aprendre una funció que mapegi entrades a sortides.

  • Regressió: Predicció de valors continus.
    • Exemple: Predir el preu d'una casa.
  • Classificació: Predicció de categories o classes.
    • Exemple: Classificar correus electrònics com a spam o no spam.

Aprenentatge No Supervisat

En l'aprenentatge no supervisat, el model s'entrena amb dades sense etiquetar. L'objectiu és trobar patrons o estructures ocultes en les dades.

  • Clustering: Agrupació de dades en clusters o grups.
    • Exemple: Agrupar clients en segments de mercat.
  • Reducció de Dimensionalitat: Reduir el nombre de característiques mantenint la major quantitat d'informació possible.
    • Exemple: PCA (Anàlisi de Components Principals).

Aprenentatge per Reforç

En l'aprenentatge per reforç, un agent aprèn a prendre decisions mitjançant la interacció amb un entorn i la recepció de recompenses o càstigs.

  • Agent: L'entitat que pren decisions.
  • Entorn: El món amb el qual interactua l'agent.
  • Recompensa: La retroalimentació que rep l'agent per avaluar les seves accions.

Exemple de Codi: Regressió Lineal en Python

A continuació, es presenta un exemple bàsic de com implementar una regressió lineal utilitzant la biblioteca scikit-learn en Python.

import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió lineal
model = LinearRegression()

# Entrenar el model
model.fit(X, y)

# Fer prediccions
y_pred = model.predict(X)

# Visualitzar els resultats
plt.scatter(X, y, color='blue', label='Dades Reals')
plt.plot(X, y_pred, color='red', label='Línia de Regressió')
plt.xlabel('X')
plt.ylabel('y')
plt.legend()
plt.show()

Explicació del Codi

  1. Importació de Biblioteques: S'importen numpy per manejar les dades, matplotlib per a la visualització i LinearRegression de scikit-learn per crear el model.
  2. Dades d'Exemple: Es creen les dades d'entrada X i les etiquetes y.
  3. Creació del Model: S'instancia el model de regressió lineal.
  4. Entrenament del Model: S'ajusta el model a les dades d'entrada i etiquetes.
  5. Prediccions: S'utilitzen les dades d'entrada per fer prediccions.
  6. Visualització: Es grafiquen les dades reals i la línia de regressió.

Conclusió

Aquests fonaments són essencials per comprendre i desenvolupar models de Machine Learning més avançats.
