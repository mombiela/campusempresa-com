Introducció

Els arbres de decisió són una tècnica d'aprenentatge supervisat utilitzada tant per a problemes de classificació com de regressió. La seva estructura jeràrquica permet prendre decisions basades en una sèrie de preguntes binàries, cosa que els fa intuïtius i fàcils d'interpretar.

Conceptes Clau

  • Node Arrel: El node superior de l'arbre que representa la característica inicial per a la divisió.
  • Nodes Interns: Representen les característiques en què es divideix el conjunt de dades.
  • Fulles: Els nodes terminals que representen la decisió final o la classe.
  • Branca: Connecta dos nodes, representant la relació entre una característica i el seu resultat.

Construcció d'un Arbre de Decisió

  1. Selecció de la Característica: Triar la característica que millor divideix el conjunt de dades.
  2. Criteri de Divisió: Utilitzar mètriques com el guany d'informació, índex Gini o reducció de variància per determinar la millor divisió.
  3. Divisió Recursiva: Repetir el procés de selecció i divisió per a cada subconjunt resultant.
  4. Condició de Parada: Determinar quan aturar la divisió, ja sigui per assolir un nombre mínim de mostres en un node o una profunditat màxima de l'arbre.

Exemple de Codi

A continuació, es mostra un exemple de com construir un arbre de decisió utilitzant la biblioteca scikit-learn en Python.

# Importar les biblioteques necessàries
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn import metrics

# Carregar el conjunt de dades Iris
iris = load_iris()
X = iris.data
y = iris.target

# Dividir el conjunt de dades en entrenament i prova
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=1)

# Crear el classificador d'arbre de decisió
clf = DecisionTreeClassifier()

# Entrenar el classificador
clf = clf.fit(X_train, y_train)

# Realitzar prediccions
y_pred = clf.predict(X_test)

# Avaluar el model
print("Precisió:", metrics.accuracy_score(y_test, y_pred))

Avantatges i Desavantatges

Avantatges

  • Interpretabilitat: Fàcil d'entendre i visualitzar.
  • No requereix normalització: Els arbres de decisió no necessiten que les dades siguin escalades o normalitzades.
  • Maneig de dades categòriques: Poden manejar tant dades numèriques com categòriques.

Desavantatges

  • Sobreajust: Els arbres de decisió poden sobreajustar-se fàcilment si no es poden adequadament.
  • Inestabilitat: Petites variacions en les dades poden resultar en un arbre completament diferent.
  • Biaix: Poden ser esbiaixats si una classe domina.

Comparació de Criteris de Divisió

| Criteri | Descripció | Ús Comú | |-------------------|-----------------------------------------------------------------------------|--------------------------| | Guany d'Informació | Mesura la reducció en l'entropia després d'una divisió. | Arbres de decisió ID3 | | Índex Gini | Mesura la puresa dels nodes. | Arbres de decisió CART | | Reducció de Variància | Utilitzat per a problemes de regressió, mesura la reducció en la variància. | Arbres de regressió |

Conclusió

Els arbres de decisió són una eina poderosa i versàtil en el camp del machine learning. La seva facilitat d'interpretació i capacitat per manejar diferents tipus de dades els fa una opció popular per a molts problemes. No obstant això, és crucial tenir en compte les seves limitacions i aplicar tècniques com la poda per evitar el sobreajust.
