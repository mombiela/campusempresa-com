Introducció al Clustering

El clustering és una tècnica d'aprenentatge no supervisat que agrupa un conjunt d'objectes de tal manera que els objectes en el mateix grup (anomenat clúster) són més similars entre si que als d'altres grups. K-Means és un dels algoritmes de clustering més populars i àmpliament utilitzats.

Conceptes Clau de K-Means

  • Centroides: Punts que representen el centre de cada clúster.
  • Assignació: Cada punt de dades s'assigna al clúster el centreide del qual està més proper.
  • Actualització: Els centroides es recalculen com la mitjana de tots els punts de dades assignats a aquest clúster.
  • Iteració: El procés d'assignació i actualització es repeteix fins que els centroides ja no canvien significativament.

Algoritme K-Means

  1. Inicialització: Seleccionar K punts inicials (centroides) a l'atzar.
  2. Assignació de Clústers: Assignar cada punt de dades al centreide més proper.
  3. Recalcular Centroides: Calcular la mitjana dels punts de dades en cada clúster per obtenir nous centroides.
  4. Repetir: Repetir els passos 2 i 3 fins que els centroides no canviïn significativament o s'assoleixi un nombre màxim d'iteracions.

Exemple de Codi en Python

A continuació, es presenta un exemple de com implementar K-Means utilitzant la biblioteca scikit-learn en Python.

import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans

# Generar dades d'exemple
X = np.array([[1, 2], [1, 4], [1, 0],
              [4, 2], [4, 4], [4, 0]])

# Crear el model K-Means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Predir els clústers
labels = kmeans.predict(X)

# Obtenir els centroides
centroids = kmeans.cluster_centers_

# Visualitzar els resultats
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.scatter(centroids[:, 0], centroids[:, 1], s=300, c='red')
plt.show()

Explicació del Codi

  • Importació de Biblioteques: S'importen numpy per a la manipulació de dades, matplotlib per a la visualització i KMeans de scikit-learn per a l'algoritme de clustering.
  • Generació de Dades: Es crea un conjunt de dades d'exemple.
  • Creació del Model: S'inicialitza el model K-Means amb 2 clústers i s'ajusta a les dades.
  • Predicció de Clústers: Es prediuen els clústers per a cada punt de dades.
  • Obtenció de Centroides: S'obtenen els centroides dels clústers.
  • Visualització: Es visualitzen els punts de dades i els centroides.

Avantatges i Desavantatges de K-Means

| Avantatges | Desavantatges | |-------------------------------------|----------------------------------------| | Simple i fàcil d'implementar | Sensible a la inicialització | | Escalable per a grans conjunts de dades | No maneja bé clústers de formes no esfèriques | | Convergència ràpida | Requereix especificar K per endavant |

Conclusió

K-Means és una tècnica poderosa i eficient per al clustering de dades, especialment quan es té una idea aproximada del nombre de clústers. No obstant això, és important tenir en compte les seves limitacions, com la sensibilitat a la inicialització i la necessitat d'especificar el nombre de clústers. Amb una comprensió clara d'aquests conceptes i la pràctica amb exemples de codi, els professionals poden aplicar K-Means de manera efectiva en els seus projectes de Machine Learning.
