Introducció a Scikit-Learn

Scikit-Learn és una biblioteca de Python àmpliament utilitzada per a l'aprenentatge automàtic. Proporciona eines simples i eficients per a la mineria de dades i l'anàlisi de dades. És accessible per a tothom i reutilitzable en diversos contextos.

  • Fàcil d'usar: API simple i coherent.
  • Eficient: Basat en NumPy, SciPy i Matplotlib.
  • Versàtil: Compatible amb una varietat d'algoritmes d'aprenentatge supervisat i no supervisat.

Instal·lació de Scikit-Learn

Per instal·lar Scikit-Learn, pots usar pip, el gestor de paquets de Python:

pip install scikit-learn

Conceptes Clau en Scikit-Learn

  • Models: Representen algoritmes d'aprenentatge automàtic.
  • Datasets: Conjunts de dades que s'utilitzen per entrenar i avaluar models.
  • Preprocessament: Transformacions aplicades a les dades abans d'entrenar un model.
  • Avaluació: Mètodes per mesurar el rendiment d'un model.

Exemple Pràctic: Classificació amb Scikit-Learn

Anem a implementar un exemple de classificació utilitzant el conjunt de dades Iris, que és un clàssic en l'aprenentatge automàtic.

Pas 1: Importar Biblioteques Necessàries

import numpy as np
import pandas as pd
from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

Pas 2: Carregar el Conjunt de Dades

iris = datasets.load_iris()
X = iris.data
y = iris.target

Pas 3: Dividir el Conjunt de Dades

Dividim el conjunt de dades en conjunts d'entrenament i prova.

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

Pas 4: Preprocessament de Dades

Estandarditzem les característiques perquè tinguin una mitjana de 0 i una desviació estàndard d'1.

scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test = scaler.transform(X_test)

Pas 5: Entrenar el Model

Usem l'algoritme K-Nearest Neighbors (KNN) per a la classificació.

knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(X_train, y_train)

Pas 6: Fer Prediccions

Realitzem prediccions sobre el conjunt de prova.

y_pred = knn.predict(X_test)

Pas 7: Avaluar el Model

Calculem la precisió del model.

accuracy = accuracy_score(y_test, y_pred)
print(f'Precisió del model: {accuracy * 100:.2f}%')

Conclusió

En aquest tema, hem cobert els conceptes bàsics de Scikit-Learn i com implementar-lo en Python per resoldre un problema de classificació. Hem après a:

  • Instal·lar i configurar Scikit-Learn.
  • Carregar i dividir conjunts de dades.
  • Preprocessar dades.
  • Entrenar i avaluar un model de classificació.

Scikit-Learn és una eina poderosa i flexible que facilita la implementació d'algoritmes d'aprenentatge automàtic en Python. Amb pràctica i experimentació, pots explorar més algoritmes i tècniques avançades per millorar els teus models.
