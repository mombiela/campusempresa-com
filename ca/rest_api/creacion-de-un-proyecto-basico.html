Creació d'un projecte bàsic
Definició de rutes i controladors ►

En aquesta secció, aprendrem com crear un projecte bàsic per a una REST API. Aquest projecte servirà com a base per construir i expandir funcionalitats en futures lliçons. Utilitzarem Node.js i Express per a aquest propòsit.

Requisits previs

Abans de començar, assegura't de tenir instal·lats els següents components:

  • Node.js (versió 14 o superior)
  • npm (Node Package Manager)

Pas 1: Configuració de l'entorn de desenvolupament

Instal·lació de Node.js i npm

Si encara no tens Node.js i npm instal·lats, pots descarregar-los i instal·lar-los des de nodejs.org.

Creació del directori del projecte

  1. Obre el teu terminal o línia de comandes.

  2. Crea un nou directori per al teu projecte i navega dins d'ell:

    mkdir mi-proyecto-rest-api
cd mi-proyecto-rest-api

Inicialització del projecte amb npm

  1. Inicialitza un nou projecte de Node.js:

    npm init -y

    Això crearà un fitxer package.json amb la configuració predeterminada.

Pas 2: Instal·lació de dependències

Per a aquest projecte, utilitzarem Express, un framework minimalista per a Node.js que facilita la creació d'APIs REST.

  1. Instal·la Express:

    npm install express

Pas 3: Creació del servidor bàsic

Estructura del projecte

Organitza el teu projecte de la següent manera:

mi-proyecto-rest-api/
├── node_modules/
├── package.json
└── index.js

Configuració del servidor

  1. Crea un fitxer index.js en el directori arrel del teu projecte.

  2. Obre index.js i afegeix el següent codi per configurar un servidor bàsic amb Express:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('¡Hola, mundo!');
});

app.listen(port, () => {
    console.log(`Servidor escoltant a http://localhost:${port}`);
});

Execució del servidor

  1. En el teu terminal, executa el següent comandament per iniciar el servidor:

    node index.js

  2. Obre el teu navegador web i navega a http://localhost:3000. Hauries de veure el missatge "¡Hola, mundo!".

Pas 4: Creació de rutes bàsiques

Rutes GET

  1. Afegeix una nova ruta GET en index.js per manejar sol·licituds a /api/saludo:

    app.get('/api/saludo', (req, res) => {
    res.json({ mensaje: '¡Hola desde la API!' });
});

Rutes POST

  1. Per manejar sol·licituds POST, primer necessitem configurar Express perquè pugui processar dades JSON. Afegeix la següent línia abans de definir les rutes:

    app.use(express.json());

  2. Afegeix una nova ruta POST en index.js per manejar sol·licituds a /api/saludo:

    app.post('/api/saludo', (req, res) => {
    const nombre = req.body.nombre;
    res.json({ mensaje: `¡Hola, ${nombre}!` });
});

Exemple de codi complet

El fitxer index.js complet hauria de veure's així:

const express = require('express');
const app = express();
const port = 3000;

app.use(express.json());

app.get('/', (req, res) => {
    res.send('¡Hola, mundo!');
});

app.get('/api/saludo', (req, res) => {
    res.json({ mensaje: '¡Hola desde la API!' });
});

app.post('/api/saludo', (req, res) => {
    const nombre = req.body.nombre;
    res.json({ mensaje: `¡Hola, ${nombre}!` });
});

app.listen(port, () => {
    console.log(`Servidor escoltant a http://localhost:${port}`);
});

Conclusió

En aquesta lliçó, hem configurat un projecte bàsic de REST API utilitzant Node.js i Express. Hem après a:

  • Configurar un entorn de desenvolupament.
  • Crear un servidor bàsic.
  • Definir rutes GET i POST.

Aquest projecte bàsic servirà com a base per afegir més funcionalitats i explorar conceptes avançats en futures lliçons. Continua practicant i experimentant amb noves rutes i funcionalitats!
