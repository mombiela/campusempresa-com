En aquesta secció, aprendràs a configurar el teu entorn de desenvolupament per treballar amb REST APIs. Aquest pas és crucial per assegurar que tinguis totes les eines necessàries i que el teu entorn estigui optimitzat per al desenvolupament eficient i efectiu d'APIs.

Requisits previs

Abans de començar, assegura't de tenir els següents elements instal·lats al teu sistema:

  • Sistema operatiu: Windows, macOS o Linux.
  • Editor de codi: Visual Studio Code, Sublime Text, IntelliJ IDEA, etc.
  • Llenguatge de programació: Node.js, Python, Java, etc.
  • Gestor de paquets: npm (Node.js), pip (Python), Maven (Java), etc.
  • Client HTTP: Postman, Insomnia, cURL, etc.

Instal·lació d'eines essencials

1. Instal·lació de l'editor de codi

Per a aquest curs, farem servir Visual Studio Code (VS Code) a causa de la seva popularitat i extensibilitat.

Passos per instal·lar VS Code:

  1. Visita la pàgina de descàrregues de VS Code.
  2. Descarrega l'instal·lador corresponent al teu sistema operatiu.
  3. Segueix les instruccions de l'instal·lador per completar la instal·lació.

2. Instal·lació del llenguatge de programació

Per a aquest curs, utilitzarem Node.js. A continuació, es detallen els passos per a la seva instal·lació:

Passos per instal·lar Node.js:

  1. Visita la pàgina de descàrregues de Node.js.
  2. Descarrega la versió LTS (Long Term Support) recomanada.
  3. Segueix les instruccions de l'instal·lador per completar la instal·lació.
  4. Verifica la instal·lació executant els següents comandaments al teu terminal: 
    node -v
npm -v

3. Instal·lació del client HTTP

Postman és una eina popular per provar i desenvolupar APIs. A continuació, es detallen els passos per a la seva instal·lació:

Passos per instal·lar Postman:

  1. Visita la pàgina de descàrregues de Postman.
  2. Descarrega l'instal·lador corresponent al teu sistema operatiu.
  3. Segueix les instruccions de l'instal·lador per completar la instal·lació.

Configuració del projecte

1. Creació d'un nou projecte

Passos per crear un nou projecte en Node.js:

  1. Obre el teu terminal i navega al directori on desitges crear el teu projecte.
  2. Executa el següent comandament per inicialitzar un nou projecte: 
    npm init -y
  3. Això crearà un fitxer package.json amb la configuració bàsica del projecte.

2. Instal·lació de dependències

Per a aquest curs, utilitzarem Express.js, un framework minimalista per a Node.js que facilita la creació d'APIs REST.

Passos per instal·lar Express.js:

  1. Executa el següent comandament al teu terminal: 
    npm install express

3. Configuració del servidor bàsic

A continuació, configurarem un servidor bàsic utilitzant Express.js.

Exemple de codi:

// Importar el mòdul Express
const express = require('express');

// Crear una instància de l'aplicació Express
const app = express();

// Definir un port per al servidor
const PORT = process.env.PORT || 3000;

// Definir una ruta bàsica
app.get('/', (req, res) => {
  res.send('Hola, món!');
});

// Iniciar el servidor
app.listen(PORT, () => {
  console.log(`Servidor escoltant al port ${PORT}`);
});

4. Execució del servidor

Passos per executar el servidor:

  1. Desa el fitxer anterior com index.js al directori arrel del teu projecte.
  2. Executa el següent comandament al teu terminal: 
    node index.js
  3. Obre el teu navegador i navega a http://localhost:3000. Hauries de veure el missatge "Hola, món!".

Conclusió

En aquesta secció, hem cobert els passos essencials per configurar el teu entorn de desenvolupament per treballar amb REST APIs. Des de la instal·lació d'eines clau com VS Code i Postman, fins a la configuració d'un servidor bàsic utilitzant Node.js i Express.js. Amb aquest entorn configurat, estaràs llest per aprofundir en el desenvolupament i la implementació d'APIs REST en les següents seccions del curs.
