Introducció a Postman

Postman és una eina popular per al desenvolupament i prova d'APIs. Permet als desenvolupadors enviar sol·licituds HTTP, desar respostes i automatitzar proves. És especialment útil per treballar amb APIs RESTful.

Característiques principals de Postman

  • Interfície amigable: Permet crear i gestionar sol·licituds HTTP de manera intuïtiva.
  • Col·leccions: Agrupa múltiples sol·licituds en un sol lloc per facilitar l'organització.
  • Variables: Permet l'ús de variables per reutilitzar valors en diferents sol·licituds.
  • Scripts: Suporta scripts pre-sol·licitud i de prova per automatitzar tasques.
  • Documentació: Genera documentació automàticament a partir de les col·leccions.

Instal·lació i Configuració

Per començar a usar Postman, primer has d'instal·lar-lo i configurar-lo.

Passos per a la instal·lació

  1. Descarregar Postman: Visita Postman i descarrega la versió adequada per al teu sistema operatiu.
  2. Instal·lar: Segueix les instruccions d'instal·lació específiques per al teu sistema operatiu.
  3. Crear un compte: Registra't o inicia sessió per sincronitzar les teves col·leccions i configuracions.

Creació de Sol·licituds HTTP

Postman permet crear diferents tipus de sol·licituds HTTP, com GET, POST, PUT, DELETE, entre altres.

Exemple de sol·licitud GET

Per realitzar una sol·licitud GET a una API, segueix aquests passos:

  1. Obrir Postman.
  2. Crear una nova sol·licitud: Fes clic a "New" i selecciona "Request".
  3. Configurar la sol·licitud:
    • Selecciona el mètode HTTP (GET).
    • Introdueix la URL de l'API (per exemple, https://jsonplaceholder.typicode.com/posts).
  4. Enviar la sol·licitud: Fes clic a "Send".
GET /posts HTTP/1.1
Host: jsonplaceholder.typicode.com

Exemple de sol·licitud POST

Per realitzar una sol·licitud POST, segueix aquests passos:

  1. Crear una nova sol·licitud: Fes clic a "New" i selecciona "Request".
  2. Configurar la sol·licitud:
    • Selecciona el mètode HTTP (POST).
    • Introdueix la URL de l'API (per exemple, https://jsonplaceholder.typicode.com/posts).
    • A la pestanya "Body", selecciona "raw" i tria "JSON" com a tipus de contingut.
    • Introdueix el cos de la sol·licitud en format JSON.
{
  "title": "foo",
  "body": "bar",
  "userId": 1
}
  1. Enviar la sol·licitud: Fes clic a "Send".
POST /posts HTTP/1.1
Host: jsonplaceholder.typicode.com
Content-Type: application/json

{
  "title": "foo",
  "body": "bar",
  "userId": 1
}

Ús de Col·leccions i Variables

Les col·leccions i variables en Postman són eines poderoses per organitzar i reutilitzar sol·licituds.

Creació de Col·leccions

  1. Crear una nova col·lecció: Fes clic a "New" i selecciona "Collection".
  2. Afegir sol·licituds a la col·lecció: Arrossega i deixa anar les sol·licituds existents o crea noves sol·licituds dins de la col·lecció.

Ús de Variables

  1. Definir variables: A la pestanya "Variables" d'una col·lecció, defineix les variables necessàries.
  2. Usar variables en sol·licituds: Utilitza la sintaxi {{variableName}} per inserir variables en les URL, encapçalaments, cossos de sol·licitud, etc.
{
  "title": "{{postTitle}}",
  "body": "{{postBody}}",
  "userId": {{userId}}
}

Automatització de Proves amb Scripts

Postman permet escriure scripts per automatitzar proves i validar respostes.

Scripts Pre-sol·licitud

Aquests scripts s'executen abans d'enviar la sol·licitud. Pots usar-los per configurar variables o modificar la sol·licitud.

pm.environment.set("currentTime", new Date().toISOString());

Scripts de Prova

Aquests scripts s'executen després de rebre la resposta. Pots usar-los per validar el contingut de la resposta.

pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

pm.test("Response time is less than 200ms", function () {
    pm.expect(pm.response.responseTime).to.be.below(200);
});

Conclusió

Postman és una eina essencial per a qualsevol desenvolupador que treballi amb APIs. La seva interfície intuïtiva, juntament amb característiques avançades com col·leccions, variables i scripts, facilita enormement el procés de desenvolupament i prova d'APIs. En dominar Postman, podràs millorar l'eficiència i qualitat de les teves proves d'API, assegurant que les teves aplicacions funcionin correctament i compleixin amb els requisits esperats.
