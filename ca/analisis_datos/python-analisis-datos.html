Introducció

L'anàlisi de dades és una disciplina fonamental en l'era de la informació. Python s'ha convertit en una eina essencial per als analistes de dades a causa de la seva simplicitat i la potència de les seves biblioteques. Aquest tema cobrirà els conceptes bàsics i avançats de l'ús de Python per a l'anàlisi de dades.

Instal·lació i Configuració

Abans de començar amb l'anàlisi de dades en Python, és necessari configurar l'entorn de treball.

Instal·lació de Python

  • Descarrega i instal·la Python des de python.org.
  • Verifica la instal·lació executant python --version a la terminal.

Instal·lació de Biblioteques Essencials

Per a l'anàlisi de dades, utilitzarem diverses biblioteques populars de Python:

pip install numpy pandas matplotlib seaborn
  • NumPy: Biblioteca fonamental per al càlcul numèric.
  • Pandas: Biblioteca per a la manipulació i anàlisi de dades.
  • Matplotlib: Biblioteca per a la visualització de dades.
  • Seaborn: Biblioteca per a la visualització de dades basada en Matplotlib.

Manipulació de Dades amb Pandas

Pandas és una biblioteca poderosa per a la manipulació i anàlisi de dades. Proporciona estructures de dades com Series i DataFrames.

Series

Una Sèrie és una estructura de dades unidimensional similar a una llista.

import pandas as pd

# Crear una Sèrie
data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

DataFrames

Un DataFrame és una estructura de dades bidimensional similar a una taula.

# Crear un DataFrame
data = {
    'Nom': ['Ana', 'Lluís', 'Carles'],
    'Edat': [28, 34, 29],
    'Ciutat': ['Madrid', 'Barcelona', 'València']
}
df = pd.DataFrame(data)
print(df)

Operacions Bàsiques amb DataFrames

  • Selecció de columnes:
print(df['Nom'])
  • Filtrat de files:
print(df[df['Edat'] > 30])
  • Afegir noves columnes:
df['Salari'] = [3000, 4000, 3500]
print(df)

Anàlisi Estadística amb NumPy

NumPy proporciona suport per a grans matrius multidimensionals i funcions matemàtiques d'alt nivell.

Creació d'Arrays

import numpy as np

# Crear un array
array = np.array([1, 2, 3, 4, 5])
print(array)

Operacions Bàsiques

  • Suma:
print(np.sum(array))
  • Mitjana:
print(np.mean(array))
  • Desviació estàndard:
print(np.std(array))

Visualització de Dades

La visualització de dades és crucial per interpretar i comunicar els resultats de l'anàlisi.

Gràfics amb Matplotlib

import matplotlib.pyplot as plt

# Crear un gràfic de línies
plt.plot([1, 2, 3, 4, 5], [1, 4, 9, 16, 25])
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Gràfic de Línies')
plt.show()

Gràfics amb Seaborn

import seaborn as sns

# Crear un gràfic de dispersió
sns.scatterplot(x='Edat', y='Salari', data=df)
plt.show()

Conclusió

Python ofereix un conjunt robust d'eines per a l'anàlisi de dades. Des de la manipulació de dades amb Pandas, passant per l'anàlisi estadística amb NumPy, fins a la visualització amb Matplotlib i Seaborn, Python facilita el procés de convertir dades en informació útil. Practicar amb aquests exemples i exercicis ajudarà a consolidar els coneixements adquirits.
