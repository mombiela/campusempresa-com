Introducció a l'Anàlisi Exploratori de Dades (EDA)

L'Anàlisi Exploratori de Dades (EDA) és una fase crucial en el procés d'anàlisi de dades que se centra a resumir les principals característiques d'un conjunt de dades, sovint amb mètodes visuals. L'objectiu principal de l'EDA és comprendre millor les dades abans d'aplicar models predictius o d'inferència.

Objectius de l'EDA

  • Comprendre l'estructura de les dades: Identificar patrons, detectar anomalies i verificar supòsits.
  • Resumir les característiques principals: Utilitzar estadístiques descriptives i visualitzacions.
  • Guiar la selecció de models i tècniques: Informar sobre les decisions de modelatge i transformació de dades.

Eines i Tècniques Comunes en EDA

Estadístiques Descriptives

  • Mesures de tendència central: Mitjana, mediana, moda.
  • Mesures de dispersió: Rang, variància, desviació estàndard.
  • Mesures de forma: Asimetria, curtosi.

Visualitzacions

  • Gràfics de barres i histogrames: Per distribució de variables categòriques i contínues.
  • Diagrames de caixa i bigotis (boxplots): Per identificar outliers i distribució de dades.
  • Diagrames de dispersió (scatter plots): Per analitzar relacions entre variables.
  • Mapes de calor (heatmaps): Per visualitzar correlacions entre variables.

Exemple d'Estadístiques Descriptives en Python

import pandas as pd

# Carregar un conjunt de dades
df = pd.read_csv('data.csv')

# Calcular estadístiques descriptives
descriptive_stats = df.describe()
print(descriptive_stats)

Exemple de Visualització en Python

import matplotlib.pyplot as plt
import seaborn as sns

# Histograma d'una variable contínua
plt.figure(figsize=(10, 6))
sns.histplot(df['variable_continua'], kde=True)
plt.title('Distribució de Variable Contínua')
plt.xlabel('Valor')
plt.ylabel('Freqüència')
plt.show()

# Diagrama de dispersió entre dues variables
plt.figure(figsize=(10, 6))
sns.scatterplot(x='variable1', y='variable2', data=df)
plt.title('Diagrama de Dispersió entre Variable1 i Variable2')
plt.xlabel('Variable1')
plt.ylabel('Variable2')
plt.show()

Anàlisi de Correlació

  • Coeficient de correlació de Pearson: Mesura la relació lineal entre dues variables.
  • Coeficient de correlació de Spearman: Mesura la relació monòtona entre dues variables.

Exemple d'Anàlisi de Correlació en Python

# Calcular la matriu de correlació
correlation_matrix = df.corr()

# Visualitzar la matriu de correlació amb un mapa de calor
plt.figure(figsize=(12, 8))
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm')
plt.title('Matriu de Correlació')
plt.show()

Identificació d'Outliers

  • Mètodes estadístics: Utilitzar el rang interquartílic (IQR) i els z-scores.
  • Visualitzacions: Diagrames de caixa i bigotis, gràfics de dispersió.

Exemple d'Identificació d'Outliers en Python

# Identificació d'outliers usant el rang interquartílic (IQR)
Q1 = df['variable_continua'].quantile(0.25)
Q3 = df['variable_continua'].quantile(0.75)
IQR = Q3 - Q1

outliers = df[(df['variable_continua'] < (Q1 - 1.5 * IQR)) | (df['variable_continua'] > (Q3 + 1.5 * IQR))]
print(outliers)

Conclusió

L'Anàlisi Exploratori de Dades (EDA) és una etapa fonamental en l'anàlisi de dades que permet als analistes comprendre millor les seves dades, identificar patrons i anomalies, i prendre decisions informades sobre el modelatge i la transformació de dades. Utilitzant estadístiques descriptives i visualitzacions, els analistes poden obtenir una visió clara i detallada de les seves dades, cosa que facilita el procés d'anàlisi i millora la qualitat dels resultats obtinguts.
