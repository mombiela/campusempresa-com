Introducció

L'avaluació de models és una etapa crucial en l'anàlisi de dades que permet determinar l'eficàcia i precisió dels models predictius. Aquesta secció del curs se centrarà en les tècniques i mètriques utilitzades per avaluar models de machine learning, assegurant que els estudiants comprenguin com interpretar els resultats i millorar els seus models.

Mètriques d'Avaluació

Per avaluar un model, és fonamental utilitzar mètriques adequades que reflecteixin el seu rendiment. A continuació, es descriuen algunes de les mètriques més comunes:

  • Exactitud (Accuracy): Proporció de prediccions correctes sobre el total de prediccions.
  • Precisió (Precision): Proporció de veritables positius sobre el total de positius predites.
  • Sensibilitat (Recall) o Taxa de Veritables Positius: Proporció de veritables positius sobre el total de positius reals.
  • F1-Score: Mitjana harmònica de la precisió i la sensibilitat.
  • AUC-ROC: Àrea sota la corba ROC, que representa la capacitat del model per distingir entre classes.

Exemple de Codi: Càlcul de Mètriques

A continuació, es mostra un exemple de com calcular aquestes mètriques utilitzant Python i la biblioteca scikit-learn.

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score, roc_auc_score
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# Dades d'exemple
X = [[1, 2], [3, 4], [5, 6], [7, 8]]
y = [0, 0, 1, 1]

# Divisió de les dades en entrenament i prova
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.5, random_state=42)

# Entrenament del model
model = RandomForestClassifier()
model.fit(X_train, y_train)

# Prediccions
y_pred = model.predict(X_test)
y_prob = model.predict_proba(X_test)[:, 1]

# Càlcul de mètriques
accuracy = accuracy_score(y_test, y_pred)
precision = precision_score(y_test, y_pred)
recall = recall_score(y_test, y_pred)
f1 = f1_score(y_test, y_pred)
roc_auc = roc_auc_score(y_test, y_prob)

print(f'Accuracy: {accuracy}')
print(f'Precision: {precision}')
print(f'Recall: {recall}')
print(f'F1 Score: {f1}')
print(f'ROC AUC: {roc_auc}')

Validació Creuada

La validació creuada és una tècnica utilitzada per avaluar la capacitat de generalització d'un model. Consisteix a dividir el conjunt de dades en múltiples subconjunts i entrenar el model diverses vegades, cada vegada amb un subconjunt diferent com a conjunt de prova.

  • K-Fold Cross Validation: Divideix les dades en k parts iguals. El model s'entrena k vegades, cada vegada utilitzant una part diferent com a conjunt de prova i les restants com a conjunt d'entrenament.
  • Leave-One-Out Cross Validation (LOOCV): Cada observació s'utilitza una vegada com a conjunt de prova i la resta com a conjunt d'entrenament.

Exemple de Codi: Validació Creuada

from sklearn.model_selection import cross_val_score

# Dades d'exemple
X = [[1, 2], [3, 4], [5, 6], [7, 8]]
y = [0, 0, 1, 1]

# Model
model = RandomForestClassifier()

# K-Fold Cross Validation
scores = cross_val_score(model, X, y, cv=4)

print(f'Cross-Validation Scores: {scores}')
print(f'Mean Score: {scores.mean()}')

Conclusió

L'avaluació de models és una etapa essencial en l'anàlisi de dades que permet determinar l'eficàcia dels models predictius. Utilitzant mètriques d'avaluació i tècniques com la validació creuada, els analistes poden comparar diferents models i seleccionar el més adequat per al seu problema específic. La pràctica contínua i la comprensió profunda d'aquestes tècniques són clau per millorar la precisió i la capacitat de generalització dels models.
