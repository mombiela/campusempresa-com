Introducció als Models Estadístics

Els models estadístics són eines fonamentals en l'anàlisi de dades. Permeten als analistes comprendre i predir comportaments basats en dades històriques. En aquesta secció, explorarem els conceptes clau, tipus de models i com implementar-los en Python.

Conceptes Clau

  • Variable Dependent: La variable que s'intenta predir o explicar.
  • Variable Independent: Les variables que s'utilitzen per predir la variable dependent.
  • Coeficients: Valors que multipliquen les variables independents en un model lineal.
  • Intercepció: El valor de la variable dependent quan totes les variables independents són zero.
  • Error: La diferència entre els valors observats i els valors preditos pel model.

Tipus de Models Estadístics

Models Lineals

Els models lineals assumeixen una relació lineal entre la variable dependent i una o més variables independents.

Exemple de Regressió Lineal Simple

import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Dades d'exemple
X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
y = np.array([1, 3, 2, 5, 4])

# Crear el model
model = LinearRegression()
model.fit(X, y)

# Prediccions
y_pred = model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_pred, color='red')
plt.xlabel('Variable Independent')
plt.ylabel('Variable Dependent')
plt.title('Regressió Lineal Simple')
plt.show()

Models No Lineals

Els models no lineals capturen relacions més complexes entre les variables.

Exemple de Regressió Polinòmica

from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline

# Dades d'exemple
X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
y = np.array([1, 3, 2, 5, 4])

# Crear el model polinòmic
poly_model = make_pipeline(PolynomialFeatures(degree=2), LinearRegression())
poly_model.fit(X, y)

# Prediccions
y_poly_pred = poly_model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_poly_pred, color='red')
plt.xlabel('Variable Independent')
plt.ylabel('Variable Dependent')
plt.title('Regressió Polinòmica')
plt.show()

Avaluació de Models

Per avaluar l'eficàcia d'un model, s'utilitzen diverses mètriques:

  • R² (Coeficient de Determinació): Indica la proporció de la variància en la variable dependent que és previsible a partir de les variables independents.
  • Error Quadràtic Mitjà (MSE): La mitjana dels quadrats dels errors.
  • Error Absolut Mitjà (MAE): La mitjana dels valors absoluts dels errors.

Exemple d'Avaluació

from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score

# Calcular mètriques
mse = mean_squared_error(y, y_pred)
mae = mean_absolute_error(y, y_pred)
r2 = r2_score(y, y_pred)

print(f'MSE: {mse}')
print(f'MAE: {mae}')
print(f'R²: {r2}')

Conclusió

Els models estadístics són eines poderoses per a l'anàlisi de dades. Des de models lineals simples fins a models no lineals més complexos, cada tipus de model té el seu lloc depenent de la naturalesa de les dades i els objectius de l'anàlisi. L'avaluació de models és crucial per assegurar que les prediccions siguin precises i útils. Amb la pràctica i l'ús de biblioteques com scikit-learn, els analistes poden implementar i avaluar aquests models de manera efectiva.
