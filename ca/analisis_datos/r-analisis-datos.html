Introducció a R

R és un llenguatge de programació i un entorn de programari lliure per a l'anàlisi estadística i la visualització de dades. És àmpliament utilitzat per estadístics, analistes de dades i científics de dades a causa de la seva capacitat per manejar, analitzar i visualitzar grans conjunts de dades.

Avantatges d'usar R

  • Gratuït i de codi obert: R és completament gratuït i el seu codi font està disponible per a tothom.
  • Àmplia comunitat: Existeix una gran comunitat d'usuaris i desenvolupadors que contribueixen amb paquets i suport.
  • Extensibilitat: R té una gran quantitat de paquets que estenen la seva funcionalitat.
  • Visualització de dades: R és conegut per les seves capacitats avançades de visualització de dades.

Instal·lació i Configuració de R

Per començar a usar R, és necessari instal·lar tant R com RStudio, un entorn de desenvolupament integrat (IDE) popular per a R.

Instal·lació de R

  1. Visita el lloc web del CRAN (Comprehensive R Archive Network): https://cran.r-project.org/
  2. Descarrega i instal·la la versió adequada per al teu sistema operatiu (Windows, macOS, Linux).

Instal·lació de RStudio

  1. Visita el lloc web de RStudio: https://rstudio.com/products/rstudio/download/
  2. Descarrega i instal·la la versió gratuïta de RStudio Desktop.

Conceptes Bàsics de R

Abans de profunditzar en l'anàlisi de dades, és important familiaritzar-se amb alguns conceptes bàsics de R.

Tipus de Dades

  • Numèrics: Nombres enters i decimals.
  • Caràcters: Cadenes de text.
  • Factors: Variables categòriques.
  • Lògics: Valors TRUE o FALSE.

Estructures de Dades

  • Vectors: Seqüències de dades del mateix tipus.
  • Matrius: Conjunts bidimensionals de dades del mateix tipus.
  • Data Frames: Taules de dades on cada columna pot contenir diferents tipus de dades.
  • Llistes: Col·leccions d'elements que poden ser de diferents tipus.

Exemple de Codi

# Crear un vector numèric
numeros <- c(1, 2, 3, 4, 5)

# Crear un data frame
dades <- data.frame(
  nom = c("Ana", "Lluís", "Joan"),
  edat = c(23, 25, 30),
  salari = c(50000, 55000, 60000)
)

# Mostrar el data frame
print(dades)

Importació i Exportació de Dades

R permet importar i exportar dades des de i cap a diferents formats de fitxer.

Importació de Dades

  • CSV: Comma-Separated Values
  • Excel: Fitxers de Microsoft Excel
  • Bases de Dades: Connexions a bases de dades SQL

Exemple d'Importació d'un Fitxer CSV

# Llegir un fitxer CSV
dades <- read.csv("ruta/al/fitxer.csv")

# Mostrar els primers 6 registres
head(dades)

Exportació de Dades

  • write.csv(): Per exportar dades a un fitxer CSV.
  • write.xlsx(): Per exportar dades a un fitxer Excel (requereix el paquet openxlsx).

Exemple d'Exportació d'un Data Frame a CSV

# Exportar el data frame a un fitxer CSV
write.csv(dades, "ruta/al/fitxer_exportat.csv")

Anàlisi de Dades amb R

R proporciona una àmplia gamma de funcions i paquets per realitzar anàlisi de dades.

Estadístiques Descriptives

  • summary(): Proporciona un resum estadístic bàsic.
  • mean(), median(), sd(): Càlcul de la mitjana, mediana i desviació estàndard.

Exemple d'Estadístiques Descriptives

# Calcular estadístiques descriptives
summary(dades$salari)

# Calcular la mitjana del salari
mean(dades$salari)

# Calcular la desviació estàndard del salari
sd(dades$salari)

Visualització de Dades

  • ggplot2: Un paquet poderós per a la creació de gràfics.
  • plot(): Funció bàsica per a gràfics.

Exemple de Visualització amb ggplot2

# Instal·lar i carregar el paquet ggplot2
install.packages("ggplot2")
library(ggplot2)

# Crear un gràfic de dispersió
ggplot(dades, aes(x = edat, y = salari)) +
  geom_point() +
  labs(title = "Relació entre Edat i Salari",
       x = "Edat",
       y = "Salari")

Conclusió

R és una eina poderosa i versàtil per a l'anàlisi de dades. Des de la importació i manipulació de dades fins a l'anàlisi estadístic i la visualització, R ofereix una àmplia gamma de funcionalitats que el fan indispensable per als professionals de l'anàlisi de dades. Amb la pràctica i l'ús continu, els usuaris poden aprofitar al màxim les capacitats de R per obtenir insights valuosos de les seves dades.
