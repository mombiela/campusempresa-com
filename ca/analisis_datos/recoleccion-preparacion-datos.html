Introducció

La recol·lecció i preparació de dades és una fase crucial en l'anàlisi de dades. Aquesta etapa implica l'obtenció de dades de diverses fonts i la seva posterior neteja i transformació perquè siguin adequades per a l'anàlisi. En aquest mòdul, aprendràs sobre les tècniques i eines necessàries per recollir i preparar dades de manera efectiva.

Recol·lecció de Dades

La recol·lecció de dades és el primer pas en el procés d'anàlisi de dades. Implica l'obtenció de dades de diverses fonts, que poden ser internes o externes a l'organització.

Fonts de Dades

  • Bases de Dades Relacionals: MySQL, PostgreSQL, SQL Server.
  • APIs: RESTful APIs, SOAP APIs.
  • Arxius: CSV, Excel, JSON, XML.
  • Web Scraping: BeautifulSoup, Scrapy.
  • Sensors i Dispositius IoT: Dades en temps real de sensors.

Exemple de Recol·lecció de Dades des d'una API

import requests

# URL de l'API
url = "https://api.example.com/data"

# Realitzar la sol·licitud GET
response = requests.get(url)

# Verificar l'estat de la resposta
if response.status_code == 200:
    data = response.json()
    print("Dades recollides amb èxit")
else:
    print("Error en recollir dades")

Preparació de Dades

Un cop les dades han estat recollides, és necessari preparar-les per a l'anàlisi. Això inclou la neteja, transformació i estructuració de les dades.

Neteja de Dades

La neteja de dades implica l'eliminació de dades duplicades, el maneig de valors faltants i la correcció d'errors en les dades.

  • Eliminació de Duplicats: Identificar i eliminar registres duplicats.
  • Maneig de Valors Faltants: Imputació de valors faltants, eliminació de registres incomplets.
  • Correcció d'Errors: Corregir errors tipogràfics, inconsistències en les dades.

Exemple de Neteja de Dades amb Pandas

import pandas as pd

# Carregar dades des d'un arxiu CSV
df = pd.read_csv("data.csv")

# Eliminar duplicats
df = df.drop_duplicates()

# Manejar valors faltants
df = df.fillna(method='ffill')

# Corregir errors tipogràfics
df['column_name'] = df['column_name'].str.replace('erroor', 'error')

print("Dades netejades amb èxit")

Transformació de Dades

La transformació de dades implica canviar el format o l'estructura de les dades perquè siguin adequades per a l'anàlisi.

  • Normalització: Escalar les dades perquè estiguin en un rang comú.
  • Codificació: Convertir dades categòriques en dades numèriques.
  • Agrupació: Resumir les dades en grups.

Exemple de Transformació de Dades

from sklearn.preprocessing import StandardScaler, LabelEncoder

# Normalització de dades
scaler = StandardScaler()
df['normalized_column'] = scaler.fit_transform(df[['column_name']])

# Codificació de dades categòriques
encoder = LabelEncoder()
df['encoded_column'] = encoder.fit_transform(df['categorical_column'])

print("Dades transformades amb èxit")

Conclusió

La recol·lecció i preparació de dades són passos fonamentals en l'anàlisi de dades. La recol·lecció de dades implica obtenir dades de diverses fonts, mentre que la preparació de dades inclou la neteja i transformació de les dades perquè siguin adequades per a l'anàlisi. Dominar aquestes tècniques et permetrà treballar amb dades de manera més eficient i obtenir resultats més precisos en les teves anàlisis.
