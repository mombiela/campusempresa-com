Introducció

L'anàlisi de sèries temporals és una tècnica crucial en el camp de l'anàlisi de dades, especialment quan es tracta de dades que es recullen al llarg del temps. Aquest tipus d'anàlisi s'utilitza per entendre les tendències, patrons i comportaments de les dades temporals, i és fonamental en àrees com l'economia, la meteorologia, l'enginyeria i les ciències socials.

Conceptes Clau

  • Sèrie Temporal: Conjunt de dades recollides en intervals de temps regulars.
  • Tendència: Direcció general en la qual es mou la sèrie temporal al llarg del temps.
  • Estacionalitat: Patrons que es repeteixen en intervals regulars de temps.
  • Sorra: Variabilitat aleatòria en les dades que no segueix un patró específic.
  • Descomposició: Procés de separar una sèrie temporal en components de tendència, estacionalitat i sorra.

Tipus de Sèries Temporals

  • Sèries Temporals Univariades: Dades d'una sola variable recollides al llarg del temps.
  • Sèries Temporals Multivariades: Dades de múltiples variables recollides al llarg del temps.

Mètodes d'Anàlisi

Descomposició de Sèries Temporals

La descomposició de sèries temporals és una tècnica que permet separar una sèrie temporal en els seus components bàsics: tendència, estacionalitat i sorra.

import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose

# Carregar dades d'exemple
data = pd.read_csv('path_to_your_data.csv', index_col='date', parse_dates=True)
series = data['your_time_series_column']

# Descomposició
result = seasonal_decompose(series, model='additive')
result.plot()
plt.show()

Models ARIMA

El model ARIMA (AutoRegressive Integrated Moving Average) és un dels mètodes més utilitzats per a l'anàlisi i la predicció de sèries temporals.

  • AR (AutoRegressiu): Model que utilitza les dependències entre una observació i un nombre de retards (valors anteriors).
  • I (Integrat): Diferenciació de les dades per fer-les estacionàries.
  • MA (Mitjana Mòbil): Model que utilitza la dependència entre una observació i un error residual d'un model de mitjana mòbil aplicat a retards.
from statsmodels.tsa.arima.model import ARIMA

# Ajustar el model ARIMA
model = ARIMA(series, order=(5,1,0))
model_fit = model.fit()

# Resum del model
print(model_fit.summary())

# Prediccions
predictions = model_fit.forecast(steps=10)
print(predictions)

Models de Suavitzat Exponencial

Els models de suavitzat exponencial són una altra tècnica popular per a l'anàlisi de sèries temporals, especialment útil per a dades amb tendència i estacionalitat.

  • SES (Simple Exponential Smoothing): Adequat per a dades sense tendència ni estacionalitat.
  • Holt’s Linear Trend Model: Adequat per a dades amb tendència.
  • Holt-Winters Seasonal Model: Adequat per a dades amb tendència i estacionalitat.
from statsmodels.tsa.holtwinters import ExponentialSmoothing

# Ajustar el model Holt-Winters
model = ExponentialSmoothing(series, seasonal='add', seasonal_periods=12)
model_fit = model.fit()

# Prediccions
predictions = model_fit.forecast(steps=12)
print(predictions)

Exercicis Pràctics

  1. Descomposició de Sèries Temporals: Utilitza un conjunt de dades de la teva elecció i aplica la descomposició per identificar els components de tendència, estacionalitat i sorra.
  2. Model ARIMA: Ajusta un model ARIMA a un conjunt de dades i realitza prediccions per als pròxims 10 períodes.
  3. Model de Suavitzat Exponencial: Aplica el model Holt-Winters a un conjunt de dades amb estacionalitat i realitza prediccions per al pròxim any.

Conclusió

L'anàlisi de sèries temporals és una eina poderosa en l'anàlisi de dades, permetent als professionals identificar patrons i fer prediccions basades en dades històriques. La comprensió dels conceptes clau i l'aplicació de mètodes com la descomposició, ARIMA i el suavitzat exponencial són essencials per a qualsevol analista de dades que treballi amb dades temporals.
