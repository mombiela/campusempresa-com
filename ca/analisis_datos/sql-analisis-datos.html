Introducció a SQL

SQL (Structured Query Language) és un llenguatge de programació utilitzat per gestionar i manipular bases de dades relacionals. És una eina essencial per als analistes de dades, ja que permet extreure, filtrar i transformar dades de manera eficient.

Conceptes Bàsics de SQL

  • Bases de Dades Relacionals: Conjunt de taules relacionades entre si.
  • Taules: Estructures que emmagatzemen dades en files i columnes.
  • Files: Cada fila representa un registre únic.
  • Columnes: Cada columna representa un atribut del registre.

Operacions Bàsiques en SQL

SELECT

La instrucció SELECT s'utilitza per consultar dades d'una o més taules.

SELECT columna1, columna2
FROM nom_taula;

WHERE

La clàusula WHERE s'utilitza per filtrar registres que compleixen una condició específica.

SELECT columna1, columna2
FROM nom_taula
WHERE condició;

Exemple

SELECT nom, edat
FROM empleats
WHERE edat > 30;

Aquest exemple selecciona els noms i edats dels empleats que tenen més de 30 anys.

Funcions d'Agregació

Les funcions d'agregació permeten realitzar càlculs sobre un conjunt de valors i retornar un sol valor.

  • COUNT: Compta el nombre de files.
  • SUM: Suma els valors d'una columna.
  • AVG: Calcula la mitjana dels valors d'una columna.
  • MAX: Troba el valor màxim.
  • MIN: Troba el valor mínim.

Exemple

SELECT COUNT(*), AVG(sou), MAX(sou), MIN(sou)
FROM empleats;

Aquest exemple compta el nombre d'empleats i calcula el sou mitjà, màxim i mínim.

Joins

Els JOINs s'utilitzen per combinar files de dues o més taules basades en una columna relacionada entre elles.

  • INNER JOIN: Retorna files quan hi ha una coincidència en ambdues taules.
  • LEFT JOIN: Retorna totes les files de la taula esquerra i les files coincidents de la taula dreta.
  • RIGHT JOIN: Retorna totes les files de la taula dreta i les files coincidents de la taula esquerra.
  • FULL JOIN: Retorna files quan hi ha una coincidència en una de les taules.

Exemple

SELECT empleats.nom, departaments.nom
FROM empleats
INNER JOIN departaments ON empleats.departament_id = departaments.id;

Aquest exemple selecciona els noms dels empleats i els noms dels departaments on treballen, mostrant només les coincidències.

Subconsultes

Les subconsultes són consultes niades dins d'una altra consulta.

Exemple

SELECT nom, sou
FROM empleats
WHERE sou > (SELECT AVG(sou) FROM empleats);

Aquest exemple selecciona els noms i sous dels empleats que guanyen més que el sou mitjà.

Agrupació de Dades

La clàusula GROUP BY s'utilitza per agrupar files que tenen els mateixos valors en columnes especificades.

Exemple

SELECT departament_id, COUNT(*)
FROM empleats
GROUP BY departament_id;

Aquest exemple compta el nombre d'empleats en cada departament.

Ordenació de Dades

La clàusula ORDER BY s'utilitza per ordenar els resultats d'una consulta.

Exemple

SELECT nom, sou
FROM empleats
ORDER BY sou DESC;

Aquest exemple selecciona els noms i sous dels empleats, ordenant-los de major a menor sou.

Conclusió

SQL és una eina poderosa per a l'anàlisi de dades, permetent als analistes extreure, filtrar i transformar dades de manera eficient. Comprendre les operacions bàsiques, funcions d'agregació, joins, subconsultes, agrupació i ordenació de dades és fonamental per a qualsevol professional de l'anàlisi de dades. Practicar amb exemples i exercicis és crucial per dominar aquestes habilitats.
