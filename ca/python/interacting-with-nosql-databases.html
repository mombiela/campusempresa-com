Introducció a les Bases de Dades NoSQL

Les bases de dades NoSQL estan dissenyades per gestionar grans volums de dades, càrregues elevades d'usuaris i canvis freqüents en les estructures de dades. A diferència de les bases de dades SQL tradicionals, les bases de dades NoSQL no utilitzen un esquema fix, cosa que les fa més flexibles i escalables.

Conceptes Clau

  • Sense Esquema: Les bases de dades NoSQL no requereixen un esquema predefinit.
  • Escalabilitat: Dissenyades per escalar distribuint les dades a través de múltiples servidors.
  • Tipus de Bases de Dades NoSQL:
    • Magatzems de Documents: Emmagatzemen dades en documents similars a JSON (per exemple, MongoDB).
    • Magatzems Clau-Valor: Emmagatzemen dades com a parelles clau-valor (per exemple, Redis).
    • Magatzems de Columnes: Emmagatzemen dades en columnes en lloc de files (per exemple, Cassandra).
    • Bases de Dades de Grafs: Emmagatzemen dades en estructures de graf (per exemple, Neo4j).

Configuració de l'Entorn

Abans d'interactuar amb bases de dades NoSQL en Python, cal instal·lar les biblioteques necessàries. Per a aquest curs, utilitzarem MongoDB com a exemple.

Instal·lació de MongoDB i pymongo

  1. Instal·lar MongoDB: Segueix les instruccions al lloc web oficial de MongoDB per instal·lar MongoDB al teu sistema.
  2. Instal·lar pymongo: Utilitza pip per instal·lar la biblioteca pymongo, que permet a Python interactuar amb MongoDB. 
    pip install pymongo

Connexió a MongoDB

Per interactuar amb MongoDB, cal establir una connexió utilitzant la biblioteca pymongo.

Exemple: Connexió a MongoDB

from pymongo import MongoClient

# Crear un MongoClient per a la instància de MongoDB en execució
client = MongoClient('localhost', 27017)

# Accedir a la base de dades 'test'
db = client.test

print("Connectat a MongoDB")
  • MongoClient: Aquesta classe s'utilitza per crear una connexió amb el servidor MongoDB.
  • localhost: El nom de l'amfitrió on s'està executant MongoDB.
  • 27017: El número de port per defecte per a MongoDB.

Operacions CRUD

CRUD significa Crear, Llegir, Actualitzar i Eliminar. Aquestes són les operacions bàsiques que es poden realitzar en una base de dades.

Crear

Per inserir un document en una col·lecció, utilitza els mètodes insert_one o insert_many.

Exemple: Inserir un Document

# Definir un document de mostra
document = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}

# Inserir el document a la col·lecció 'users'
result = db.users.insert_one(document)

print(f"Document inserit amb ID: {result.inserted_id}")

Llegir

Per recuperar documents d'una col·lecció, utilitza els mètodes find o find_one.

Exemple: Recuperar Documents

# Recuperar un únic document
user = db.users.find_one({"name": "Alice"})
print(user)

# Recuperar múltiples documents
users = db.users.find({"age": {"$gt": 25}})
for user in users:
    print(user)
  • find_one: Recupera un únic document que coincideixi amb la consulta.
  • find: Recupera múltiples documents que coincideixin amb la consulta.
  • $gt: Un operador de consulta de MongoDB que significa "més gran que".

Actualitzar

Per actualitzar documents, utilitza els mètodes update_one o update_many.

Exemple: Actualitzar un Document

# Actualitzar l'edat de l'usuari anomenat 'Alice'
result = db.users.update_one(
    {"name": "Alice"},  # Consulta per coincidir amb el document
    {"$set": {"age": 31}}  # Operació d'actualització
)

print(f"Coincideixen {result.matched_count} document(s) i s'han modificat {result.modified_count} document(s)")
  • $set: Un operador d'actualització de MongoDB utilitzat per establir el valor d'un camp.

Eliminar

Per eliminar documents, utilitza els mètodes delete_one o delete_many.

Exemple: Eliminar un Document

# Eliminar l'usuari anomenat 'Alice'
result = db.users.delete_one({"name": "Alice"})

print(f"Eliminat(s) {result.deleted_count} document(s)")
