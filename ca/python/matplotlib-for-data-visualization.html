Introducció a Matplotlib

Matplotlib és una biblioteca completa per crear visualitzacions estàtiques, animades i interactives en Python. És àmpliament utilitzada per a la visualització de dades gràcies a la seva flexibilitat i facilitat d'ús.

Conceptes Clau

  • Figure: La finestra o pàgina sencera on es dibuixa tot.
  • Axes: L'àrea on es ploten les dades, que inclou els eixos x i y (i possiblement z).
  • Plot: La representació real de les dades (per exemple, línia, barra, dispersió).

Instal·lació

Per instal·lar Matplotlib, utilitza la següent comanda:

pip install matplotlib

Ploteig Bàsic

Creació d'un Gràfic de Línies Simple

import matplotlib.pyplot as plt

# Dades d'exemple
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Creació del gràfic
plt.plot(x, y)

# Afegint títol i etiquetes
plt.title('Gràfic de Línies Simple')
plt.xlabel('Eix X')
plt.ylabel('Eix Y')

# Mostrant el gràfic
plt.show()
  • plt.plot(): Plota les dades.
  • plt.title(): Afegeix un títol al gràfic.
  • plt.xlabel() i plt.ylabel(): Etiqueta els eixos x i y.
  • plt.show(): Mostra el gràfic.

Personalització de Gràfics

Afegint Estils i Colors

plt.plot(x, y, color='green', linestyle='--', marker='o')
plt.title('Gràfic de Línies Estilitzat')
plt.xlabel('Eix X')
plt.ylabel('Eix Y')
plt.show()
  • color: Canvia el color de la línia.
  • linestyle: Canvia l'estil de la línia (per exemple, discontínua).
  • marker: Afegeix marcadors als punts de dades.

Afegint Llegendes

plt.plot(x, y, label='Nombres Primers')
plt.title('Gràfic de Línies amb Llegenda')
plt.xlabel('Eix X')
plt.ylabel('Eix Y')
plt.legend()
plt.show()
  • label: Afegeix una etiqueta al gràfic.
  • plt.legend(): Mostra la llegenda al gràfic.

Ploteig Avançat

Subgràfics

fig, axs = plt.subplots(2, 2)

# Primer subgràfic
axs[0, 0].plot(x, y)
axs[0, 0].set_title('Subgràfic 1')

# Segon subgràfic
axs[0, 1].plot(x, [i**2 for i in x])
axs[0, 1].set_title('Subgràfic 2')

# Tercer subgràfic
axs[1, 0].plot(x, [i**3 for i in x])
axs[1, 0].set_title('Subgràfic 3')

# Quart subgràfic
axs[1, 1].plot(x, [i**0.5 for i in x])
axs[1, 1].set_title('Subgràfic 4')

plt.tight_layout()
plt.show()
  • plt.subplots(): Crea una graella de subgràfics.
  • axs[row, col]: Accedeix a un subgràfic específic.

Gràfics de Barres

categories = ['A', 'B', 'C', 'D']
values = [4, 7, 1, 8]

plt.bar(categories, values)
plt.title('Gràfic de Barres')
plt.xlabel('Categories')
plt.ylabel('Valors')
plt.show()
  • plt.bar(): Crea un gràfic de barres.

Gràfics de Dispersió

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]
sizes = [20, 50, 80, 200, 500]

plt.scatter(x, y, s=sizes, c='red', alpha=0.5)
plt.title('Gràfic de Dispersió')
plt.xlabel('Eix X')
plt.ylabel('Eix Y')
plt.show()
  • plt.scatter(): Crea un gràfic de dispersió.
  • s: Estableix la mida dels marcadors.
  • c: Estableix el color dels marcadors.
  • alpha: Estableix la transparència dels marcadors.

Conclusió

Matplotlib és una eina poderosa per a la visualització de dades en Python. Permet la creació d'una àmplia gamma de gràfics i diagrames, des de gràfics de línies simples fins a subgràfics complexos i gràfics de dispersió. Dominant Matplotlib, pots comunicar efectivament les teves idees a través de visualitzacions convincents.
