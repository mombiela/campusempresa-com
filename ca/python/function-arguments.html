Introducció

En Python, les funcions són un bloc fonamental que et permet encapsular codi en blocs reutilitzables. Entendre com passar arguments a les funcions és crucial per escriure codi flexible i mantenible. Aquesta secció cobrirà els diferents tipus d'arguments de funció en Python, des de conceptes bàsics fins a avançats.

Tipus d'Arguments de Funció

1. Arguments Posicionals

Els arguments posicionals són el tipus més senzill d'arguments. Es passen a la funció en el mateix ordre en què es defineixen.

def greet(name, age):
    print(f"Hola, {name}! Tens {age} anys.")

greet("Alice", 30)
  • Explicació: En l'exemple anterior, name i age són arguments posicionals. Quan es crida greet("Alice", 30), "Alice" s'assigna a name i 30 a age.

2. Arguments per Paraula Clau

Els arguments per paraula clau es passen a la funció especificant explícitament el nom del paràmetre juntament amb el seu valor.

def greet(name, age):
    print(f"Hola, {name}! Tens {age} anys.")

greet(name="Bob", age=25)
  • Explicació: Aquí, name="Bob" i age=25 són arguments per paraula clau. L'ordre dels arguments no importa quan s'utilitzen arguments per paraula clau.

3. Arguments per Defecte

Els arguments per defecte et permeten definir valors per defecte per als paràmetres. Si no es proporciona cap valor per a un paràmetre amb un valor per defecte, s'utilitza el valor per defecte.

def greet(name, age=20):
    print(f"Hola, {name}! Tens {age} anys.")

greet("Charlie")
greet("Diana", 22)
  • Explicació: En aquest exemple, age té un valor per defecte de 20. Quan es crida greet("Charlie"), s'utilitza el valor per defecte. Quan es crida greet("Diana", 22), el valor proporcionat 22 substitueix el valor per defecte.

4. Arguments de Longitud Variable

Python permet que les funcions acceptin un nombre arbitrari d'arguments utilitzant *args per a arguments no per paraula clau i **kwargs per a arguments per paraula clau.

*args

*args et permet passar un nombre variable d'arguments no per paraula clau a una funció.

def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # Sortida: 10
  • Explicació: El paràmetre *args recull tots els arguments posicionals addicionals en una tupla. En aquest cas, sum_all(1, 2, 3, 4) suma tots els números proporcionats.

**kwargs

**kwargs et permet passar un nombre variable d'arguments per paraula clau a una funció.

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Eve", age=28, city="Nova York")
  • Explicació: El paràmetre **kwargs recull tots els arguments per paraula clau addicionals en un diccionari. La funció després itera sobre aquest diccionari per imprimir cada parell clau-valor.

Conceptes Avançats

1. Combinació de Diferents Tipus d'Arguments

Pots combinar arguments posicionals, per paraula clau, per defecte i de longitud variable en una sola funció. L'ordre dels paràmetres hauria de ser: posicional, *args, per defecte, **kwargs.

def complex_function(a, b, *args, c=10, **kwargs):
    print(f"a: {a}, b: {b}, c: {c}")
    print(f"args: {args}")
    print(f"kwargs: {kwargs}")

complex_function(1, 2, 3, 4, 5, d=6, e=7)
  • Explicació: En aquest exemple, a i b són arguments posicionals, *args recull arguments posicionals addicionals, c és un argument per defecte, i **kwargs recull arguments per paraula clau addicionals.

2. Desempaquetament d'Arguments

Pots desempaquetar llistes o diccionaris en arguments de funció utilitzant * i ** respectivament.

def greet(name, age):
    print(f"Hola, {name}! Tens {age} anys.")

args = ["Frank", 35]
kwargs = {"name": "Grace", "age": 40}

greet(*args)
greet(**kwargs)
  • Explicació: La sintaxi *args desempaqueta la llista en arguments posicionals, mentre que **kwargs desempaqueta el diccionari en arguments per paraula clau.

Conclusió

Entendre els arguments de funció en Python és essencial per escriure codi flexible i reutilitzable. Aquesta secció ha cobert arguments posicionals, arguments per paraula clau, arguments per defecte i arguments de longitud variable, juntament amb conceptes avançats com la combinació de diferents tipus d'arguments i el desempaquetament d'arguments. Dominar aquests conceptes millorarà significativament la teva capacitat per escriure codi Python eficient i net.
