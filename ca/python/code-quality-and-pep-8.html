Introducció a la Qualitat del Codi

  • Definició: La qualitat del codi es refereix al grau en què el codi s'adhereix als estàndards i millors pràctiques acordats, fent-lo llegible, mantenible i eficient.
  • Importància:
    • Llegibilitat: Més fàcil per als altres (i per a tu mateix) de llegir i entendre.
    • Mantenibilitat: Simplifica el procés d'actualització i correcció del codi.
    • Eficiència: Assegura que el codi s'executi de manera òptima.

Què és el PEP 8?

  • Definició: PEP 8 és la Proposta de Millora de Python que descriu les convencions per escriure codi Python llegible i consistent.
  • Propòsit: Millorar la llegibilitat i consistència del codi Python.

Principals Directrius del PEP 8

Indentació

  • Utilitza 4 espais per nivell d'indentació.
def example_function():
    if True:
        print("Follow PEP 8!")

Longitud de la Línia

  • Limita totes les línies a un màxim de 79 caràcters.
# Aquest és un comentari que té exactament 79 caràcters de llarg per demostrar la directriu.

Línies en Blanc

  • Utilitza línies en blanc per separar definicions de funcions i classes de nivell superior, i dins de les funcions per separar seccions lògiques.
class MyClass:
    def __init__(self):
        self.value = 0

    def increment(self):
        self.value += 1

Imports

  • Els imports haurien d'estar normalment en línies separades.
  • Ordena els imports: biblioteca estàndard, biblioteques de tercers, imports específics de l'aplicació/biblioteca local.
import os
import sys

import requests

from mymodule import myfunction

Convencions de Nomenclatura

  • Variables i Funcions: Utilitza lower_case_with_underscores.
  • Classes: Utilitza CapWords.
  • Constants: Utilitza ALL_CAPS_WITH_UNDERSCORES.
# Noms de variables i funcions
my_variable = 10

def my_function():
    pass

# Nom de la classe
class MyClass:
    pass

# Nom de la constant
MAX_LIMIT = 100

Comentaris

  • Utilitza comentaris per explicar per què es fa alguna cosa, no què es fa.
  • Utilitza # per a comentaris en línia i comentaris de bloc.
# Aquest és un comentari de bloc
x = 42  # Aquest és un comentari en línia

Docstrings

  • Utilitza cometes triples per a docstrings en funcions, classes i mòduls.
def my_function():
    """
    Aquest és un docstring.
    Explica què fa la funció.
    """
    pass

Eines per Enforçar el PEP 8

  • pycodestyle: Una eina per comprovar el teu codi Python contra algunes de les convencions d'estil del PEP 8.
  • autopep8: Una eina que formata automàticament el codi Python per conformar-se a la guia d'estil del PEP 8.
  • flake8: Una eina que combina pycodestyle, pyflakes i altres plugins per comprovar l'estil i la qualitat del codi Python.

Exemple de Codi Complint el PEP 8

import os
import sys

class ExampleClass:
    """Aquesta és una classe d'exemple que segueix les directrius del PEP 8."""

    def __init__(self, value):
        self.value = value

    def increment(self):
        """Incrementa el valor en un."""
        self.value += 1

def main():
    """Funció principal per executar l'exemple."""
    example = ExampleClass(0)
    example.increment()
    print(example.value)

if __name__ == "__main__":
    main()

Conclusió

  • Resum: Adherir-se a les directrius del PEP 8 assegura que el teu codi Python sigui llegible, mantenible i consistent. Utilitzar eines com pycodestyle, autopep8 i flake8 pot ajudar a enforçar aquests estàndards.
  • Pròxims Passos: Practica escriure codi complint el PEP 8 i utilitza les eines esmentades per comprovar i millorar la qualitat del teu codi.
