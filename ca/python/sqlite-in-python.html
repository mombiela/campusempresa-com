Introducció a SQLite

SQLite és una biblioteca en llenguatge C que implementa un motor de base de dades SQL petit, ràpid, autònom, d'alta fiabilitat i amb totes les característiques. És el motor de base de dades més desplegat al món. En aquesta secció, aprendrem com utilitzar SQLite amb Python.

Configuració de SQLite en Python

Per utilitzar SQLite en Python, cal importar el mòdul sqlite3, que està inclòs a la Biblioteca Estàndard de Python.

import sqlite3

Creació d'una Base de Dades

Pots crear una nova base de dades SQLite connectant-te a un fitxer de base de dades. Si el fitxer no existeix, SQLite el crearà.

conn = sqlite3.connect('example.db')
  • example.db és el nom del fitxer de la base de dades.
  • conn és l'objecte de connexió.

Creació d'una Taula

Per crear una taula, cal executar una comanda SQL utilitzant l'objecte de connexió.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
''')

conn.commit()
conn.close()
  • cursor.execute executa la comanda SQL.
  • conn.commit guarda els canvis.
  • conn.close tanca la connexió.

Inserció de Dades

Pots inserir dades a la taula utilitzant la comanda SQL INSERT INTO.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
INSERT INTO users (name, age) VALUES (?, ?)
''', ('Alice', 30))

conn.commit()
conn.close()
  • Els marcadors de posició ? s'utilitzen per prevenir la injecció SQL.
  • Els valors es proporcionen com una tupla.

Consulta de Dades

Per recuperar dades, pots utilitzar la comanda SQL SELECT.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('SELECT * FROM users')
rows = cursor.fetchall()

for row in rows:
    print(row)

conn.close()
  • cursor.fetchall recupera totes les files del conjunt de resultats.

Actualització de Dades

Per actualitzar dades existents, utilitza la comanda SQL UPDATE.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
UPDATE users SET age = ? WHERE name = ?
''', (31, 'Alice'))

conn.commit()
conn.close()

Eliminació de Dades

Per eliminar dades, utilitza la comanda SQL DELETE.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
DELETE FROM users WHERE name = ?
''', ('Alice',))

conn.commit()
conn.close()

Ús de Gestors de Context

Utilitzar gestors de context (with statement) assegura que la connexió es tanqui correctament fins i tot si es produeix un error.

with sqlite3.connect('example.db') as conn:
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)

Gestió d'Exceptions

És important gestionar les excepcions per evitar fallades i assegurar que el programa funcioni sense problemes.

try:
    conn = sqlite3.connect('example.db')
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)
except sqlite3.Error as e:
    print(f"An error occurred: {e}")
finally:
    if conn:
        conn.close()

Resum

En aquesta secció, hem cobert els conceptes bàsics d'utilitzar SQLite amb Python, incloent:

  • Configuració de SQLite en Python.
  • Creació d'una base de dades i taules.
  • Inserció, consulta, actualització i eliminació de dades.
  • Ús de gestors de context i gestió d'exceptions.

SQLite és una eina poderosa per gestionar bases de dades petites a mitjanes, i la seva integració amb Python la fa una gran elecció per a moltes aplicacions.
