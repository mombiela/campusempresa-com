Introducció

En aquesta secció, ens endinsarem en les estructures de dades avançades en Python. Entendre aquestes estructures t'ajudarà a escriure codi més eficient i optimitzat. Cobrirem els següents conceptes clau:

  • Heaps
  • Trie
  • Grafs
  • Arbres de Segment
  • Conjunt Disjunt (Union-Find)

Heaps

Els heaps són estructures de dades basades en arbres especialitzades que satisfan la propietat del heap. S'utilitzen comunament per implementar cues de prioritat.

Conceptes Clau

  • Min-Heap: La clau a l'arrel ha de ser la mínima entre totes les claus presents en el Heap Binari.
  • Max-Heap: La clau a l'arrel ha de ser la màxima entre totes les claus presents en el Heap Binari.

Exemple de Codi

import heapq

# Min-Heap
min_heap = []
heapq.heappush(min_heap, 10)
heapq.heappush(min_heap, 20)
heapq.heappush(min_heap, 5)

print("Min-Heap:", min_heap)
print("Element més petit:", heapq.heappop(min_heap))

# Max-Heap
max_heap = []
heapq.heappush(max_heap, -10)
heapq.heappush(max_heap, -20)
heapq.heappush(max_heap, -5)

print("Max-Heap:", [-x for x in max_heap])
print("Element més gran:", -heapq.heappop(max_heap))

Explicació

  • heapq.heappush afegeix un element al heap.
  • heapq.heappop elimina i retorna l'element més petit del heap.
  • Per a un max-heap, inserim valors negatius per simular el comportament.

Trie

Un Trie (pronunciat com "try") és un tipus especial d'arbre utilitzat per emmagatzemar estructures de dades associatives. Una aplicació comuna d'un Trie és emmagatzemar un diccionari de text predictiu o autocompletar.

Conceptes Clau

  • Nodes: Cada node representa un caràcter de les cadenes emmagatzemades.
  • Arestes: Les arestes connecten nodes i representen la seqüència de caràcters.

Exemple de Codi

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

# Ús
trie = Trie()
trie.insert("hello")
print(trie.search("hello"))  # Sortida: True
print(trie.search("hell"))   # Sortida: False

Explicació

  • El mètode insert afegeix una paraula al Trie.
  • El mètode search comprova si una paraula és present al Trie.

Grafs

Els grafs s'utilitzen per representar xarxes de nodes connectats. S'utilitzen àmpliament en diverses aplicacions com xarxes socials, algorismes de rutes, etc.

Conceptes Clau

  • Vèrtexs (Nodes): Unitats fonamentals del graf.
  • Arestes: Connexions entre els vèrtexs.

Exemple de Codi

class Graph:
    def __init__(self):
        self.graph = {}

    def add_edge(self, u, v):
        if u not in self.graph:
            self.graph[u] = []
        self.graph[u].append(v)

    def bfs(self, start):
        visited = set()
        queue = [start]
        while queue:
            vertex = queue.pop(0)
            if vertex not in visited:
                print(vertex, end=" ")
                visited.add(vertex)
                queue.extend(set(self.graph[vertex]) - visited)

# Ús
g = Graph()
g.add_edge(0, 1)
g.add_edge(0, 2)
g.add_edge(1, 2)
g.add_edge(2, 0)
g.add_edge(2, 3)
g.add_edge(3, 3)

print("Cerca en Amplada començant des del vèrtex 2:")
g.bfs(2)  # Sortida: 2 0 3 1

Explicació

  • El mètode add_edge afegeix una aresta entre dos vèrtexs.
  • El mètode bfs realitza una Cerca en Amplada començant des d'un vèrtex donat.

Arbres de Segment

Els arbres de segment s'utilitzen per emmagatzemar intervals o segments. Permeten consultar quins dels segments emmagatzemats contenen un punt donat de manera eficient.

Conceptes Clau

  • Construcció: Construir l'arbre a partir d'una matriu.
  • Consulta: Trobar la suma, mínim o màxim en un interval donat.

Exemple de Codi

class SegmentTree:
    def __init__(self, data):
        self.n = len(data)
        self.tree = [0] * (2 * self.n)
        self.build(data)

    def build(self, data):
        for i in range(self.n):
            self.tree[self.n + i] = data[i]
        for i in range(self.n - 1, 0, -1):
            self.tree[i] = self.tree[i * 2] + self.tree[i * 2 + 1]

    def update(self, pos, value):
        pos += self.n
        self.tree[pos] = value
        while pos > 1:
            pos //= 2
            self.tree[pos] = self.tree[2 * pos] + self.tree[2 * pos + 1]

    def query(self, left, right):
        left += self.n
        right += self.n
        result = 0
        while left < right:
            if left % 2:
                result += self.tree[left]
                left += 1
            if right % 2:
                right -= 1
                result += self.tree[right]
            left //= 2
            right //= 2
        return result

# Ús
data = [1, 3, 5, 7, 9, 11]
seg_tree = SegmentTree(data)
print("Suma de valors en l'interval [1, 5):", seg_tree.query(1, 5))  # Sortida: 24
seg_tree.update(1, 10)
print("Suma de valors en l'interval [1, 5) després de l'actualització:", seg_tree.query(1, 5))  # Sortida: 31

Explicació

  • El mètode build construeix l'arbre de segment.
  • El mètode update actualitza el valor en una posició específica.
  • El mètode query retorna la suma de valors en un interval donat.

Conjunt Disjunt (Union-Find)

El Conjunt Disjunt, també conegut com Union-Find, és una estructura de dades que fa un seguiment d'una partició d'un conjunt en subconjunts disjunts (no superposats).

Conceptes Clau

  • Unió: Fusionar dos subconjunts en un únic subconjunt.
  • Trobar: Determinar a quin subconjunt pertany un element en particular.

Exemple de Codi

class DisjointSet:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, u):
        if self.parent[u] != u:
            self.parent[u] = self.find(self.parent[u])
        return self.parent[u]

    def union(self, u, v):
        root_u = self.find(u)
        root_v = self.find(v)
        if root_u != root_v:
            if self.rank[root_u] > self.rank[root_v]:
                self.parent[root_v] = root_u
            elif self.rank[root_u] < self.rank[root_v]:
                self.parent[root_u] = root_v
            else:
                self.parent[root_v] = root_u
                self.rank[root_u] += 1

# Ús
ds = DisjointSet(5)
ds.union(0, 2)
ds.union(4, 2)
ds.union(3, 1)
print(ds.find(4))  # Sortida: 0
print(ds.find(3))  # Sortida: 1

Explicació

  • El mètode find retorna l'arrel del conjunt que conté l'element.
  • El mètode union fusiona dos conjunts.
