Introducció als Generadors

Els generadors són una classe especial de funcions en Python que et permeten crear iteradors d'una manera simple i concisa. Et permeten iterar sobre dades sense emmagatzemar tot el conjunt de dades a la memòria, cosa que és especialment útil per a conjunts de dades grans o fluxos de dades.

Conceptes Clau

  • Iterador: Un objecte que es pot iterar (recórrer) amb un bucle.
  • Iterable: Un objecte que retorna un iterador.
  • Funció Generadora: Una funció que utilitza la paraula clau yield per retornar un valor i pausar el seu estat.
  • Objecte Generador: Un iterador creat per una funció generadora.

Creació de Generadors

Els generadors es creen utilitzant funcions i l'instrucció yield.

Exemple: Generador Simple

def simple_generator():
    yield 1
    yield 2
    yield 3

# Utilitzant el generador
gen = simple_generator()
print(next(gen))  # Sortida: 1
print(next(gen))  # Sortida: 2
print(next(gen))  # Sortida: 3

En aquest exemple:

  • La funció simple_generator retorna tres valors: 1, 2 i 3.
  • Cada crida a next(gen) reprèn la funció des d'on es va quedar.

Exemple: Generador amb Bucle

def countdown(n):
    while n > 0:
        yield n
        n -= 1

# Utilitzant el generador
for number in countdown(5):
    print(number)

En aquest exemple:

  • La funció generadora countdown retorna valors des de n fins a 1.
  • El bucle for itera sobre cada valor retornat pel generador.

Expressions Generadores

Les expressions generadores proporcionen una manera concisa de crear generadors. Són similars a les comprensions de llistes però utilitzen parèntesis en lloc de claudàtors.

Exemple: Expressió Generadora

gen_exp = (x * x per x in range(5))

# Utilitzant l'expressió generadora
for value in gen_exp:
    print(value)

En aquest exemple:

  • L'expressió generadora (x * x per x in range(5)) genera quadrats de nombres del 0 al 4.
  • El bucle for itera sobre cada valor retornat per l'expressió generadora.

Avantatges dels Generadors

  • Eficiència de Memòria: Els generadors no emmagatzemen tot el conjunt de dades a la memòria.
  • Avaluació Mandrosa: Els valors es generen sobre la marxa i només quan es necessiten.
  • Codi Llegible: Els generadors poden simplificar el codi que implica iteració.

Ús Avançat

Els generadors es poden utilitzar en escenaris més complexos, incloent-hi pipelines i coroutines.

Exemple: Pipeline de Generadors

def generate_numbers(n):
    for i in range(n):
        yield i

def square_numbers(numbers):
    for number in numbers:
        yield number * number

# Creant un pipeline
numbers = generate_numbers(5)
squared_numbers = square_numbers(numbers)

# Utilitzant el pipeline
for value in squared_numbers:
    print(value)

En aquest exemple:

  • generate_numbers retorna nombres des de 0 fins a n-1.
  • square_numbers pren un iterable i retorna el quadrat de cada nombre.
  • El pipeline permet encadenar generadors per processar dades en etapes.

Conclusió

Els generadors són una característica poderosa en Python que proporcionen una manera eficient de gestionar grans conjunts de dades i fluxos de dades. Utilitzant la paraula clau yield, pots crear iteradors que generen valors sobre la marxa, cosa que condueix a un codi més eficient en termes de memòria i més llegible. Tant si treballes amb bucles simples com amb pipelines de dades complexos, els generadors poden ajudar-te a escriure programes en Python més eficients i mantenibles.
