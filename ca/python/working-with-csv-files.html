Introducció

Els fitxers CSV (Valors Separats per Coma) són un format comú per emmagatzemar dades tabulars. Són fitxers de text senzills on cada línia representa una fila de dades, i cada valor de la fila està separat per una coma. Python proporciona diverses maneres de treballar amb fitxers CSV, facilitant la lectura i escriptura d'aquests fitxers.

Llegir fitxers CSV

Utilitzant el mòdul `csv`

El mòdul csv de la biblioteca estàndard de Python proporciona funcionalitats tant per llegir com per escriure fitxers CSV.

Llegir un fitxer CSV

Per llegir un fitxer CSV, pots utilitzar l'objecte csv.reader.

import csv

with open('example.csv', mode='r') as file:
    csv_reader = csv.reader(file)
    for row in csv_reader:
        print(row)
  • Explicació:
    • open('example.csv', mode='r'): Obre el fitxer en mode lectura.
    • csv.reader(file): Crea un objecte lector de CSV.
    • for row in csv_reader: Itera sobre cada fila del fitxer CSV.

Llegir un fitxer CSV amb capçaleres

Si el teu fitxer CSV té capçaleres, pots utilitzar l'objecte csv.DictReader per llegir el fitxer en un diccionari.

import csv

with open('example_with_headers.csv', mode='r') as file:
    csv_reader = csv.DictReader(file)
    for row in csv_reader:
        print(row)
  • Explicació:
    • csv.DictReader(file): Llegeix el fitxer CSV en un diccionari on les claus són les capçaleres de les columnes.

Escriure fitxers CSV

Utilitzant el mòdul `csv`

Per escriure en un fitxer CSV, pots utilitzar l'objecte csv.writer.

Escriure en un fitxer CSV

import csv

data = [
    ['Nom', 'Edat', 'Ciutat'],
    ['Alice', 30, 'Nova York'],
    ['Bob', 25, 'Los Angeles'],
    ['Charlie', 35, 'Chicago']
]

with open('output.csv', mode='w', newline='') as file:
    csv_writer = csv.writer(file)
    csv_writer.writerows(data)
  • Explicació:
    • open('output.csv', mode='w', newline=''): Obre el fitxer en mode escriptura.
    • csv.writer(file): Crea un objecte escriptor de CSV.
    • csv_writer.writerows(data): Escriu múltiples files al fitxer CSV.

Escriure un fitxer CSV amb capçaleres

Si vols escriure un fitxer CSV amb capçaleres, pots utilitzar l'objecte csv.DictWriter.

import csv

data = [
    {'Nom': 'Alice', 'Edat': 30, 'Ciutat': 'Nova York'},
    {'Nom': 'Bob', 'Edat': 25, 'Ciutat': 'Los Angeles'},
    {'Nom': 'Charlie', 'Edat': 35, 'Ciutat': 'Chicago'}
]

with open('output_with_headers.csv', mode='w', newline='') as file:
    fieldnames = ['Nom', 'Edat', 'Ciutat']
    csv_writer = csv.DictWriter(file, fieldnames=fieldnames)
    csv_writer.writeheader()
    csv_writer.writerows(data)
  • Explicació:
    • csv.DictWriter(file, fieldnames=fieldnames): Crea un objecte escriptor de CSV amb els noms dels camps especificats.
    • csv_writer.writeheader(): Escriu la fila de capçalera al fitxer CSV.

Gestió avançada de CSV

Utilitzant `pandas` per a fitxers CSV

Per a una gestió més avançada de fitxers CSV, es recomana la biblioteca pandas. Proporciona capacitats poderoses de manipulació de dades.

Llegir un fitxer CSV amb pandas

import pandas as pd

df = pd.read_csv('example.csv')
print(df)
  • Explicació:
    • pd.read_csv('example.csv'): Llegeix el fitxer CSV en un DataFrame.

Escriure en un fitxer CSV amb pandas

import pandas as pd

data = {
    'Nom': ['Alice', 'Bob', 'Charlie'],
    'Edat': [30, 25, 35],
    'Ciutat': ['Nova York', 'Los Angeles', 'Chicago']
}

df = pd.DataFrame(data)
df.to_csv('output.csv', index=False)
  • Explicació:
    • pd.DataFrame(data): Crea un DataFrame a partir de les dades.
    • df.to_csv('output.csv', index=False): Escriu el DataFrame en un fitxer CSV sense l'índex.

Conclusió

Treballar amb fitxers CSV en Python és senzill amb el mòdul csv integrat per a tasques bàsiques i la biblioteca pandas per a operacions més avançades. Entendre com llegir i escriure fitxers CSV és essencial per a tasques de manipulació i anàlisi de dades. En dominar aquestes tècniques, podràs gestionar fitxers CSV de manera eficient en els teus projectes de Python.
