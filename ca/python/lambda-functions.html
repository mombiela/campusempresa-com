Introducció a les Funcions Lambda

Les funcions lambda, també conegudes com a funcions anònimes, són una manera concisa de crear funcions petites i d'un sol ús en Python. Sovint s'utilitzen per a operacions curtes que no són prou complexes per requerir una definició completa de funció.

Conceptes Clau

  • Funcions Anònimes: Les funcions lambda no tenen nom.
  • Expressió Única: Estan limitades a una sola expressió.
  • Sintaxi: La sintaxi és lambda arguments: expressió.

Sintaxi Bàsica

La sintaxi bàsica d'una funció lambda és:

lambda arguments: expression
  • lambda és la paraula clau.
  • arguments són les entrades de la funció.
  • expression és l'operació única que es realitza i es retorna.

Exemple

Aquí teniu un exemple senzill d'una funció lambda que suma dos números:

add = lambda x, y: x + y
print(add(2, 3))  # Output: 5

En aquest exemple:

  • lambda x, y: x + y crea una funció lambda que pren dos arguments, x i y, i retorna la seva suma.
  • add és una variable que conté la funció lambda.
  • add(2, 3) crida la funció lambda amb 2 i 3 com a arguments, retornant 5.

Usos de les Funcions Lambda

Les funcions lambda sovint s'utilitzen en situacions on es necessita una funció petita per un període curt de temps. Els casos d'ús comuns inclouen:

  • Ordenació i Filtratge: Utilitzades amb funcions com sorted(), filter(), i map().
  • Funcions Inline: Utilitzades en llocs on es requereix una funció com a argument.

Ordenació amb Funcions Lambda

Les funcions lambda es poden utilitzar per ordenar llistes de tuples o diccionaris per elements específics.

# Llista de tuples
points = [(1, 2), (3, 1), (5, -1)]
# Ordenar pel segon element de cada tuple
sorted_points = sorted(points, key=lambda point: point[1])
print(sorted_points)  # Output: [(5, -1), (3, 1), (1, 2)]

Filtratge amb Funcions Lambda

Les funcions lambda es poden utilitzar per filtrar llistes basant-se en una condició.

# Llista de números
numbers = [1, 2, 3, 4, 5, 6]
# Filtrar els números parells
odd_numbers = list(filter(lambda x: x % 2 != 0, numbers))
print(odd_numbers)  # Output: [1, 3, 5]

Mapeig amb Funcions Lambda

Les funcions lambda es poden utilitzar per aplicar una funció a cada element d'una llista.

# Llista de números
numbers = [1, 2, 3, 4, 5]
# Elevar al quadrat cada número
squared_numbers = list(map(lambda x: x ** 2, numbers))
print(squared_numbers)  # Output: [1, 4, 9, 16, 25]

Ús Avançat de les Funcions Lambda

Les funcions lambda també es poden utilitzar en escenaris més complexos, com dins de funcions d'ordre superior o com a valors de retorn d'altres funcions.

Funcions d'Ordre Superior

Una funció d'ordre superior és una funció que pren una altra funció com a argument o retorna una funció com a resultat.

def make_incrementor(n):
    return lambda x: x + n

increment_by_5 = make_incrementor(5)
print(increment_by_5(10))  # Output: 15

En aquest exemple:

  • make_incrementor és una funció d'ordre superior que retorna una funció lambda.
  • increment_by_5 és una funció lambda que afegeix 5 a la seva entrada.

Conclusió

Les funcions lambda són una característica poderosa en Python que permeten la creació de funcions petites i anònimes sobre la marxa. Són particularment útils per a operacions curtes i es poden utilitzar efectivament amb funcions com sorted(), filter(), i map(). Tot i que estan limitades a una sola expressió, la seva sintaxi concisa les converteix en una eina valuosa per als programadors de Python. Entendre quan i com utilitzar les funcions lambda pot millorar significativament la teva capacitat per escriure codi net i eficient.
