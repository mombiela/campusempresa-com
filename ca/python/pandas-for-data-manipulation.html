Pandas és una biblioteca de Python potent i flexible utilitzada per a la manipulació i anàlisi de dades. Proporciona estructures de dades i funcions necessàries per manipular dades estructurades sense problemes. Aquest tema et guiarà des dels conceptes bàsics fins a les característiques avançades de Pandas, assegurant que puguis gestionar diverses tasques de manipulació de dades de manera eficient.

Introducció a Pandas

  • Què és Pandas?

    • Pandas és una biblioteca de codi obert per a l'anàlisi i manipulació de dades en Python.
    • Proporciona estructures de dades com Series i DataFrame.

  • Instal·lant Pandas

    !pip install pandas

  • Important Pandas

    import pandas as pd

Estructures de Dades de Pandas

Series

  • Creant una Series

    import pandas as pd

data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

  • Series amb Índex Personalitzat

    data = [1, 2, 3, 4, 5]
index = ['a', 'b', 'c', 'd', 'e']
series = pd.Series(data, index=index)
print(series)

DataFrame

  • Creant un DataFrame

    data = {
    'Nom': ['Alice', 'Bob', 'Charlie'],
    'Edat': [25, 30, 35],
    'Ciutat': ['New York', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)

  • Llegint Dades d'un Fitxer CSV

    df = pd.read_csv('data.csv')
print(df.head())

Manipulació de Dades amb Pandas

Seleccionant Dades

  • Seleccionant Columnes

    # Columna única
print(df['Nom'])

# Múltiples columnes
print(df[['Nom', 'Edat']])

  • Seleccionant Files

    # Per índex
print(df.iloc[0])

# Per etiqueta
print(df.loc[0])

Filtrant Dades

  • Utilitzant Condicions 
    # Filtrar files on Edat > 30
filtered_df = df[df['Edat'] > 30]
print(filtered_df)

Afegint i Modificant Columnes

  • Afegint una Nova Columna

    df['Salari'] = [50000, 60000, 70000]
print(df)

  • Modificant una Columna Existent

    df['Edat'] = df['Edat'] + 1
print(df)

Gestionant Dades Faltants

  • Comprovant Dades Faltants

    print(df.isnull())
print(df.isnull().sum())

  • Omplint Dades Faltants

    df['Edat'].fillna(df['Edat'].mean(), inplace=True)
print(df)

  • Eliminant Dades Faltants

    df.dropna(inplace=True)
print(df)

Agrupant i Agregant Dades

  • Agrupant Dades

    grouped = df.groupby('Ciutat')
print(grouped.mean())

  • Agregant Dades

    agg_df = df.groupby('Ciutat').agg({'Edat': 'mean', 'Salari': 'sum'})
print(agg_df)

Fusionant i Unint DataFrames

  • Fusionant DataFrames

    df1 = pd.DataFrame({
    'Nom': ['Alice', 'Bob'],
    'Edat': [25, 30]
})

df2 = pd.DataFrame({
    'Nom': ['Alice', 'Bob'],
    'Salari': [50000, 60000]
})

merged_df = pd.merge(df1, df2, on='Nom')
print(merged_df)

  • Unint DataFrames

    df1 = df1.set_index('Nom')
df2 = df2.set_index('Nom')

joined_df = df1.join(df2)
print(joined_df)

Tècniques Avançades de Pandas

Taules Dinàmiques

  • Creant una Taula Dinàmica 
    pivot_table = df.pivot_table(values='Salari', index='Ciutat', columns='Edat', aggfunc='mean')
print(pivot_table)

Anàlisi de Sèries Temporals

  • Gestionant Dades de Data i Hora

    df['Data'] = pd.to_datetime(df['Data'])
print(df)

  • Re-mostrant Dades de Sèries Temporals

    ts = df.set_index('Data')
resampled_ts = ts.resample('M').mean()
print(resampled_ts)

Conclusió

Pandas és una eina essencial per a la manipulació de dades en Python. Proporciona estructures de dades i funcions potents que fan que les tasques d'anàlisi de dades siguin senzilles i eficients. En dominar Pandas, pots gestionar una àmplia gamma de tasques de manipulació de dades, des de la neteja simple de dades fins a transformacions i agregacions de dades complexes. Continua practicant amb conjunts de dades reals per millorar les teves habilitats i convertir-te en un expert en l'ús de Pandas per a la manipulació de dades.
