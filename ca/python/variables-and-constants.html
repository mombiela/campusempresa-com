Introducció

En Python, les variables i constants són conceptes fonamentals que et permeten emmagatzemar i manipular dades. Entendre com utilitzar-les de manera efectiva és crucial per escriure codi eficient i llegible.

Variables

Què és una Variable?

  • Una variable és una ubicació amb nom utilitzada per emmagatzemar dades en memòria.
  • El valor emmagatzemat en una variable pot canviar durant l'execució del programa.

Declaració de Variables

En Python, no cal declarar explícitament el tipus d'una variable. Simplement assignes un valor a un nom de variable.

# Exemple de declaració de variable
x = 10
nom = "Alice"
és_estudiant = True

Regles de Nomenclatura de Variables

  • Els noms de les variables han de començar amb una lletra o un guió baix (_).
  • La resta del nom pot contenir lletres, números o guions baixos.
  • Els noms de les variables són sensibles a majúscules i minúscules (edat i Edat són variables diferents).

Exemples de Noms de Variables Vàlids i Invàlids

| Noms Vàlids | Noms Invàlids | |-------------|---------------| | edat | 1edat | | _nom | nom! | | suma_total | suma-total |

Actualització de Variables

Pots actualitzar el valor d'una variable assignant-li un nou valor.

# Actualització d'una variable
x = 10
x = 20  # x ara conté el valor 20

Tipus de Variables

Python és dinàmicament tipat, el que significa que el tipus d'una variable es determina en temps d'execució. Els tipus comuns inclouen:

  • int per a enters
  • float per a nombres de punt flotant
  • str per a cadenes de text
  • bool per a valors booleans
# Exemples de diferents tipus
edat = 25          # int
alçada = 5.9       # float
nom = "Alice"      # str
és_estudiant = True # bool

Constants

Què és una Constant?

  • Una constant és un tipus de variable el valor de la qual no pot canviar un cop assignat.
  • Python no té tipus de constants integrats, però per convenció, les constants s'escriuen en majúscules.

Declaració de Constants

Tot i que Python no imposa valors constants, pots indicar que una variable ha de ser tractada com una constant utilitzant lletres majúscules.

# Exemple de declaració de constants
PI = 3.14159
USUARIS_MÀXIMS = 100

Ús de Constants

Les constants s'utilitzen per definir valors que no haurien de canviar al llarg del programa, com ara valors de configuració o paràmetres fixos.

# Ús de constants en un programa
PI = 3.14159
radi = 5
àrea = PI * (radi ** 2)
print("Àrea del cercle:", àrea)

Millors Pràctiques

Convencions de Nomenclatura

  • Utilitza noms descriptius per a les variables per fer el teu codi més llegible.
  • Utilitza lletres majúscules per a les constants per distingir-les de les variables regulars.

Evita Nombres Màgics

  • Evita utilitzar valors codificats directament en el teu codi (nombres màgics). En lloc d'això, utilitza constants per donar-los noms significatius.
# Evita nombres màgics
descompte = 0.1

# Millor enfocament utilitzant constants
TAXA_DESCOMPTE = 0.1

Conclusió

Entendre les variables i constants és essencial per a qualsevol programador de Python. Les variables et permeten emmagatzemar i manipular dades, mentre que les constants t'ajuden a definir valors fixos que no haurien de canviar. Seguint les millors pràctiques i utilitzant noms significatius, pots escriure codi més llegible i mantenible.
