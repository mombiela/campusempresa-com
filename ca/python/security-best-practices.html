Introducció

La seguretat és un aspecte crític del desenvolupament de programari. En aquesta secció, cobrirem les millors pràctiques de seguretat essencials per a la programació en Python, des de nivells bàsics fins a avançats. Al final d'aquest tema, entendràs com escriure codi Python segur i protegir les teves aplicacions de vulnerabilitats comunes.

Pràctiques Bàsiques de Seguretat

Validació d'Entrades

  • Sanititzar les Entrades de l'Usuari: Sempre valida i sanititza les entrades de l'usuari per prevenir atacs d'injecció. 
    import re

def sanitize_input(user_input):
    return re.sub(r'[^\w\s]', '', user_input)

user_input = "<script>alert('hack');</script>"
safe_input = sanitize_input(user_input)
print(safe_input)  # Sortida: scriptalerthackscript

Ús d'Emmagatzematge Segur de Contrasenyes

  • Hash de Contrasenyes: Mai emmagatzemis contrasenyes en text pla. Utilitza biblioteques com bcrypt per fer hash de les contrasenyes de manera segura. 
    import bcrypt

password = b"supersecretpassword"
hashed = bcrypt.hashpw(password, bcrypt.gensalt())

# Per comprovar una contrasenya
if bcrypt.checkpw(password, hashed):
    print("La contrasenya coincideix")
else:
    print("La contrasenya no coincideix")

Evitar Codificar Informació Sensible

  • Variables d'Entorn: Utilitza variables d'entorn per emmagatzemar informació sensible com claus API i credencials de bases de dades. 
    import os

api_key = os.getenv('API_KEY')

Pràctiques Intermèdies de Seguretat

Comunicació Segura

  • Ús d'HTTPS: Assegura't que la teva aplicació utilitza HTTPS per xifrar les dades transmeses per la xarxa.
  • SSL/TLS: Utilitza biblioteques com ssl per afegir suport SSL/TLS a les teves aplicacions Python. 
    import ssl
import socket

context = ssl.create_default_context()
with socket.create_connection(('www.example.com', 443)) as sock:
    with context.wrap_socket(sock, server_hostname='www.example.com') as ssock:
        print(ssock.version())

Dependències Segures

  • Gestió de Dependències: Actualitza regularment les teves dependències per corregir vulnerabilitats conegudes.
  • Ús de pip amb requirements.txt: Especifica versions exactes de les dependències per evitar introduir vulnerabilitats. 
    # requirements.txt
requests==2.25.1
flask==1.1.2

Gestió d'Errors

  • Evitar Missatges d'Error Detallats: No exposis missatges d'error detallats als usuaris finals ja que poden revelar informació sensible. 
    try:
    # Algunes línies de codi que poden generar una excepció
    pass
except Exception as e:
    print("S'ha produït un error. Si us plau, torna-ho a intentar més tard.")
    # Registra l'error detallat per a ús intern
    with open('error.log', 'a') as f:
        f.write(str(e))

Pràctiques Avançades de Seguretat

Pràctiques de Codificació Segura

  • Principi del Menor Privilegi: Concedeix els permisos mínims necessaris perquè el teu codi funcioni.
  • Revisions de Codi: Revisa regularment el codi per detectar vulnerabilitats de seguretat.

Ús de Biblioteques de Seguretat

  • Configuració Segura: Utilitza biblioteques com secure per fer complir les capçaleres de seguretat en aplicacions web. 
    from flask import Flask
from secure import SecureHeaders

app = Flask(__name__)
secure_headers = SecureHeaders()

@app.after_request
def set_secure_headers(response):
    secure_headers.flask(response)
    return response

Monitorització i Registre

  • Monitoritzar Incidents de Seguretat: Implementa registres i monitorització per detectar i respondre a incidents de seguretat. 
    import logging

logging.basicConfig(filename='app.log', level=logging.INFO)
logging.info('Aquest és un missatge informatiu')

Conclusió

La seguretat és un procés continu que requereix vigilància i millora contínua. Seguint aquestes millors pràctiques, pots reduir significativament el risc de vulnerabilitats de seguretat en les teves aplicacions Python. Mantén-te sempre actualitzat amb les últimes tendències i actualitzacions de seguretat per mantenir les teves aplicacions segures.
