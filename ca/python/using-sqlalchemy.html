Introducció a SQLAlchemy

SQLAlchemy és una potent eina SQL i una biblioteca de Mapatge Objecte-Relacional (ORM) per a Python. Proporciona un conjunt complet de patrons de persistència de nivell empresarial ben coneguts, dissenyats per a un accés eficient i d'alt rendiment a la base de dades.

  • SQLAlchemy Core: Proporciona una capa d'abstracció SQL centrada en l'esquema.
  • SQLAlchemy ORM: Proporciona una interfície de nivell superior centrada en l'ORM.

Instal·lant SQLAlchemy

Per començar amb SQLAlchemy, cal instal·lar-lo utilitzant pip.

pip install sqlalchemy

SQLAlchemy Core

SQLAlchemy Core proporciona un Llenguatge d'Expressió SQL i un Mapatge Objecte-Relacional (ORM). Començarem amb el Core per entendre els conceptes bàsics de SQLAlchemy.

Connectant-se a una Base de Dades

Primer, cal crear un motor, que és el punt de partida per a qualsevol aplicació SQLAlchemy.

from sqlalchemy import create_engine

# Substitueix 'sqlite:///example.db' amb la URL de la teva base de dades
engine = create_engine('sqlite:///example.db')

Creant Taules

Pots definir taules utilitzant la classe Table i metadades.

from sqlalchemy import Table, Column, Integer, String, MetaData

metadata = MetaData()

users = Table('users', metadata,
    Column('id', Integer, primary_key=True),
    Column('name', String),
    Column('age', Integer)
)

metadata.create_all(engine)  # Crea la taula

Inserint Dades

Pots inserir dades a la taula utilitzant la funció insert.

from sqlalchemy import insert

ins = users.insert().values(name='John Doe', age=30)
conn = engine.connect()
result = conn.execute(ins)

Consultar Dades

Pots consultar dades utilitzant la funció select.

from sqlalchemy import select

s = select([users])
result = conn.execute(s)

for row in result:
    print(row)

SQLAlchemy ORM

L'ORM et permet mapar classes de Python a taules de bases de dades, proporcionant una abstracció de nivell superior.

Definint un Model

Defineixes un model creant una classe que hereta de Base.

from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy import Column, Integer, String

Base = declarative_base()

class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    name = Column(String)
    age = Column(Integer)

Creant una Sessió

Es requereix una sessió per interactuar amb la base de dades utilitzant l'ORM.

from sqlalchemy.orm import sessionmaker

Session = sessionmaker(bind=engine)
session = Session()

Afegint i Cometent Dades

Pots afegir i cometre dades utilitzant la sessió.

new_user = User(name='Jane Doe', age=25)
session.add(new_user)
session.commit()

Consultar Dades

Pots consultar dades utilitzant la sessió.

users = session.query(User).all()
for user in users:
    print(user.name, user.age)

Temes Avançats

Relacions

SQLAlchemy suporta la definició de relacions entre taules.

from sqlalchemy import ForeignKey
from sqlalchemy.orm import relationship

class Address(Base):
    __tablename__ = 'addresses'
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey('users.id'))
    email_address = Column(String)
    user = relationship("User", back_populates="addresses")

User.addresses = relationship("Address", order_by=Address.id, back_populates="user")

Migracions amb Alembic

Alembic és una eina lleugera de migració de bases de dades per utilitzar amb SQLAlchemy.

  • Instal·lar Alembic: pip install alembic
  • Inicialitzar Alembic: alembic init alembic
  • Crear una Migració: alembic revision --autogenerate -m "Initial migration"
  • Aplicar Migracions: alembic upgrade head

Conclusió

SQLAlchemy és una biblioteca versàtil i potent per a les interaccions amb bases de dades en Python. Tant si utilitzes el Core per a expressions SQL directes com l'ORM per a abstraccions de nivell superior, SQLAlchemy proporciona les eines necessàries per a una gestió de bases de dades eficient i efectiva. Dominant SQLAlchemy, pots gestionar operacions complexes de bases de dades amb facilitat, fent d'aquesta una habilitat essencial per a qualsevol desenvolupador de Python.
