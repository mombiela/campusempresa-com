Introducció

En aquesta secció, tractarem els conceptes fonamentals de les operacions d'entrada i sortida en Python. Entendre com gestionar l'entrada i la sortida és crucial per interactuar amb els usuaris i altres sistemes.

Entrada en Python

Les operacions d'entrada permeten que el teu programa prengui dades de l'usuari o altres fonts.

Utilitzant la funció `input()`

  • La funció input() llegeix una línia de l'entrada (normalment de l'usuari), la converteix en una cadena i la retorna.

Exemple:

name = input("Introdueix el teu nom: ")
print("Hola, " + name + "!")
  • En aquest exemple, el programa demana a l'usuari que introdueixi el seu nom i després el saluda.

Convertint l'entrada

  • Com que input() retorna una cadena, potser necessitaràs convertir-la a altres tipus de dades com enters o floats.

Exemple:

age = input("Introdueix la teva edat: ")
age = int(age)  # Converteix l'entrada a un enter
print("Tens " + str(age) + " anys.")
  • Aquí, l'entrada de l'usuari es converteix d'una cadena a un enter abans de ser utilitzada.

Sortida en Python

Les operacions de sortida permeten que el teu programa mostri dades a l'usuari o altres sistemes.

Utilitzant la funció `print()`

  • La funció print() mostra dades a la sortida estàndard (normalment la consola).

Exemple:

print("Hola, Món!")
  • Aquest simple exemple imprimeix "Hola, Món!" a la consola.

Formatant la sortida

  • Pots formatar cadenes utilitzant diversos mètodes per fer la sortida més llegible.

Utilitzant concatenació:

name = "Alice"
print("Hola, " + name + "!")

Utilitzant el mètode str.format():

name = "Alice"
print("Hola, {}!".format(name))

Utilitzant f-Strings (Python 3.6+):

name = "Alice"
print(f"Hola, {name}!")

Imprimint múltiples elements

  • Pots imprimir múltiples elements separant-los amb comes.

Exemple:

name = "Alice"
age = 30
print("Nom:", name, "Edat:", age)
  • Això imprimeix "Nom: Alice Edat: 30" amb espais entre els elements.

Entrada i sortida avançades

Llegint de fitxers

  • Pots llegir l'entrada de fitxers utilitzant la funció open() i els mètodes de fitxers.

Exemple:

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)
  • Això llegeix tot el contingut de example.txt i l'imprimeix.

Escriure a fitxers

  • Pots escriure la sortida a fitxers utilitzant la funció open() amb mode d'escriptura.

Exemple:

with open('output.txt', 'w') as file:
    file.write("Hola, Món!")
  • Això escriu "Hola, Món!" a output.txt.

Gestionant excepcions

  • És important gestionar les excepcions que poden ocórrer durant les operacions d'entrada i sortida.

Exemple:

try:
    with open('example.txt', 'r') as file:
        content = file.read()
        print(content)
except FileNotFoundError:
    print("El fitxer no s'ha trobat.")
  • Això gestiona el cas en què example.txt no existeix.

Resum

  • Entrada: Utilitza input() per obtenir l'entrada de l'usuari i converteix-la al tipus de dada requerit.
  • Sortida: Utilitza print() per mostrar dades. Format la sortida utilitzant concatenació, str.format(), o f-strings.
  • Operacions de fitxers: Utilitza open() per llegir i escriure fitxers. Gestiona les excepcions per manejar els errors de manera elegant.

Entendre les operacions bàsiques d'entrada i sortida és essencial per crear programes interactius i fàcils d'usar en Python. Practica aquests conceptes per convertir-te en un expert en la gestió de dades en les teves aplicacions.
