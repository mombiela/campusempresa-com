Introducció

En aquest curs, iniciaràs un viatge des dels conceptes bàsics de Python fins a conceptes avançats de programació. Aquest enfocament basat en projectes assegura que no només aprenguis la sintaxi i les característiques de Python, sinó que també entenguis com aplicar-los en escenaris del món real. Al final d'aquest curs, tindràs una base sòlida en Python i la capacitat d'afrontar desafiaments de programació complexos.

Estructura del Curs

El curs està dividit en tres seccions principals:

  1. Nivell Principiant: Introducció a Python, sintaxi bàsica i conceptes fonamentals de programació.
  2. Nivell Intermedi: Temes més avançats com estructures de dades, programació orientada a objectes i mòduls.
  3. Nivell Avançat: Temes complexos incloent decoradors, generadors, gestors de context i treball amb biblioteques externes.

Conceptes Clau

Nivell Principiant

  • Introducció a Python:

    • Instal·lar Python i configurar l'entorn.
    • Escriure el teu primer programa en Python.
    print("Hola, Món!")

  • Sintaxi Bàsica:

    • Variables i Tipus de Dades
    • Operadors Bàsics
    • Flux de Control (if, for, while)
    x = 10
if x > 5:
    print("x és més gran que 5")

  • Funcions:

    • Definir i cridar funcions
    • Arguments de funció i valors de retorn
    def saluda(nom):
    return f"Hola, {nom}!"

print(saluda("Alice"))

Nivell Intermedi

  • Estructures de Dades:

    • Llistes, Tuples, Conjunts i Diccionaris
    • Comprensions de Llistes
    nombres = [1, 2, 3, 4, 5]
quadrats = [x**2 per x en nombres]
print(quadrats)

  • Programació Orientada a Objectes (OOP):

    • Classes i Objectes
    • Herència i Polimorfisme
    class Animal:
    def __init__(self, nom):
        self.nom = nom

    def parla(self):
        pass

class Gos(Animal):
    def parla(self):
        return "Bup!"

el_meu_gos = Gos("Buddy")
print(el_meu_gos.parla())

  • Mòduls i Paquets:

    • Importar i utilitzar mòduls
    • Crear els teus propis mòduls
    import math
print(math.sqrt(16))

Nivell Avançat

  • Decoradors:

    • Entendre i crear decoradors
    def el_meu_decorador(func):
    def embolcall():
        print("Alguna cosa està passant abans que es cridi la funció.")
        func()
        print("Alguna cosa està passant després que es cridi la funció.")
    return embolcall

@el_meu_decorador
def diu_hola():
    print("Hola!")

diu_hola()

  • Generadors:

    • Crear i utilitzar generadors
    def compte_enrere(n):
    while n > 0:
        yield n
        n -= 1

per nombre en compte_enrere(5):
    print(nombre)

  • Gestors de Context:

    • Utilitzar l'instrucció with i crear gestors de context personalitzats
    with open('exemple.txt', 'w') com fitxer:
    fitxer.write('Hola, Món!')

  • Treballar amb Biblioteques Externes:

    • Introducció a biblioteques populars com NumPy, Pandas i Requests
    import requests
resposta = requests.get('https://api.github.com')
print(resposta.status_code)

Conclusió

Aquest curs està dissenyat per portar-te des d'un novell en Python fins a un programador avançat capaç de construir aplicacions complexes. Cada secció es construeix sobre l'anterior, assegurant una experiència d'aprenentatge suau i completa. Al final d'aquest curs, tindràs una comprensió profunda de Python i estaràs ben preparat per afrontar qualsevol desafiament de programació.
