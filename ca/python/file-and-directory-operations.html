Operacions amb Fitxers i Directoris
Comprensió de les Excepcions ►

Introducció

Les operacions amb fitxers i directoris són habilitats essencials per a qualsevol programador. En Python, els mòduls os i shutil proporcionen una àmplia gamma de funcions per interactuar amb el sistema de fitxers. Aquesta secció cobrirà els conceptes bàsics de les operacions amb fitxers i directoris, des de llegir i escriure fitxers fins a tasques més avançades com recórrer directoris i gestionar permisos de fitxers.

Treballant amb Fitxers

Obrir i Tancar Fitxers

  • Obrir un fitxer: Utilitza la funció open() per obrir un fitxer.
  • Tancar un fitxer: Sempre tanca el fitxer utilitzant el mètode close() per alliberar recursos del sistema.
# Obrir un fitxer per llegir
file = open('example.txt', 'r')

# Realitzar operacions amb el fitxer

# Tancar el fitxer
file.close()

Llegir Fitxers

  • Llegir tot el fitxer: Utilitza el mètode read().
  • Llegir línia per línia: Utilitza els mètodes readline() o readlines().
# Llegir tot el fitxer
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Llegir el fitxer línia per línia
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

Escriure a Fitxers

  • Escriure text a un fitxer: Utilitza el mètode write().
  • Afegir text a un fitxer: Obre el fitxer en mode d'afegir ('a').
# Escriure a un fitxer
with open('example.txt', 'w') as file:
    file.write('Hello, World!\n')

# Afegir a un fitxer
with open('example.txt', 'a') as file:
    file.write('Appending a new line.\n')

Treballant amb Directoris

Crear i Eliminar Directoris

  • Crear un directori: Utilitza os.mkdir() o os.makedirs() per a directoris niats.
  • Eliminar un directori: Utilitza os.rmdir() o shutil.rmtree() per a directoris no buits.
import os
import shutil

# Crear un directori
os.mkdir('new_directory')

# Crear directoris niats
os.makedirs('parent_directory/child_directory')

# Eliminar un directori buit
os.rmdir('new_directory')

# Eliminar un directori no buit
shutil.rmtree('parent_directory')

Llistar el Contingut del Directori

  • Llistar fitxers i directoris: Utilitza os.listdir() o os.scandir() per obtenir informació més detallada.
# Llistar fitxers i directoris
contents = os.listdir('.')
print(contents)

# Llistar amb informació detallada
with os.scandir('.') as entries:
    for entry in entries:
        print(entry.name, entry.is_file(), entry.is_dir())

Operacions Avançades amb Fitxers i Directoris

Recórrer Directoris

  • Recórrer directoris: Utilitza os.walk() per generar noms de fitxers en un arbre de directoris.
# Recórrer directoris
for dirpath, dirnames, filenames in os.walk('.'):
    print(f'Found directory: {dirpath}')
    for filename in filenames:
        print(f'\t{filename}')

Permisos de Fitxers

  • Canviar permisos de fitxers: Utilitza os.chmod() per canviar el mode d'un fitxer.
  • Comprovar permisos de fitxers: Utilitza os.access() per comprovar els permisos d'accés del fitxer.
# Canviar permisos de fitxers
os.chmod('example.txt', 0o644)  # Lectura i escriptura per al propietari, lectura per als altres

# Comprovar permisos de fitxers
if os.access('example.txt', os.R_OK):
    print('File is readable')
else:
    print('File is not readable')

Resum

Has après com obrir, llegir, escriure i tancar fitxers, així com com crear, eliminar i llistar directoris.
