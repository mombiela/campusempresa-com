Introducció a NumPy

NumPy (Numerical Python) és una biblioteca potent per al càlcul numèric en Python. Proporciona suport per a arrays, matrius i moltes funcions matemàtiques per operar amb aquestes estructures de dades.

  • Conceptes Clau:
    • Arrays i la seva importància en el càlcul numèric.
    • Diferències entre les llistes de Python i els arrays de NumPy.
    • Operacions bàsiques amb arrays de NumPy.
import numpy as np

# Creant un array de NumPy
array = np.array([1, 2, 3, 4, 5])
print(array)

Creació i Manipulació d'Arrays

Aprèn a crear i manipular arrays utilitzant NumPy.

  • Creació d'Arrays:
    • Utilitzant np.array()
    • Utilitzant funcions com np.zeros(), np.ones(), np.arange(), i np.linspace()
# Creant arrays utilitzant diferents mètodes
zeros_array = np.zeros((3, 3))
ones_array = np.ones((2, 2))
range_array = np.arange(10)
linspace_array = np.linspace(0, 1, 5)

print("Zeros Array:\n", zeros_array)
print("Ones Array:\n", ones_array)
print("Range Array:\n", range_array)
print("Linspace Array:\n", linspace_array)
  • Manipulació d'Arrays:
    • Remodelant arrays amb reshape()
    • Aplanant arrays amb flatten()
    • Concatenant arrays amb concatenate()
# Remodelant i aplanant arrays
reshaped_array = range_array.reshape((2, 5))
flattened_array = reshaped_array.flatten()

print("Reshaped Array:\n", reshaped_array)
print("Flattened Array:\n", flattened_array)

# Concatenant arrays
array1 = np.array([1, 2, 3])
array2 = np.array([4, 5, 6])
concatenated_array = np.concatenate((array1, array2))

print("Concatenated Array:\n", concatenated_array)

Operacions amb Arrays

Realitza diverses operacions amb arrays, incloent operacions aritmètiques, operacions estadístiques i més.

  • Operacions Aritmètiques:
    • Operacions element a element
    • Broadcasting
# Operacions element a element
array1 = np.array([1, 2, 3])
array2 = np.array([4, 5, 6])
sum_array = array1 + array2
product_array = array1 * array2

print("Sum Array:\n", sum_array)
print("Product Array:\n", product_array)

# Broadcasting
scalar = 2
broadcasted_array = array1 * scalar

print("Broadcasted Array:\n", broadcasted_array)
  • Operacions Estadístiques:
    • Mitjana, mediana, desviació estàndard, etc.
# Operacions estadístiques
data = np.array([1, 2, 3, 4, 5])
mean = np.mean(data)
median = np.median(data)
std_dev = np.std(data)

print("Mean:", mean)
print("Median:", median)
print("Standard Deviation:", std_dev)

Manipulacions Avançades d'Arrays

Explora tècniques més avançades per treballar amb arrays.

  • Slicing i Indexació:
    • Slicing bàsic
    • Indexació booleana
    • Indexació fancy
# Slicing bàsic
array = np.array([1, 2, 3, 4, 5])
slice_array = array[1:4]

print("Sliced Array:", slice_array)

# Indexació booleana
bool_array = array[array > 2]

print("Boolean Indexed Array:", bool_array)

# Indexació fancy
indices = [0, 2, 4]
fancy_indexed_array = array[indices]

print("Fancy Indexed Array:", fancy_indexed_array)
  • Matemàtiques amb Arrays:
    • Operacions d'àlgebra lineal
    • Generació de nombres aleatoris
# Operacions d'àlgebra lineal
matrix1 = np.array([[1, 2], [3, 4]])
matrix2 = np.array([[5, 6], [7, 8]])
dot_product = np.dot(matrix1, matrix2)

print("Dot Product:\n", dot_product)

# Generació de nombres aleatoris
random_array = np.random.rand(3, 3)

print("Random Array:\n", random_array)

Conclusió

NumPy és una biblioteca essencial per al càlcul numèric en Python. Proporciona eines potents per crear i manipular arrays, realitzar operacions matemàtiques i gestionar grans conjunts de dades de manera eficient. Dominar NumPy millorarà significativament la teva capacitat per realitzar càlculs científics i numèrics en Python.

  • Resum:
    • Els arrays de NumPy són més eficients i convenients que les llistes de Python per a operacions numèriques.
    • Pots crear arrays utilitzant diverses funcions de NumPy i manipular-los amb remodelació, slicing i indexació.
    • NumPy suporta una àmplia gamma d'operacions matemàtiques, incloent operacions aritmètiques, estadístiques i d'àlgebra lineal.
    • Tècniques avançades com el broadcasting i la generació de nombres aleatoris també estan disponibles en NumPy.

Comprenent i utilitzant NumPy, podràs realitzar càlculs numèrics complexos amb facilitat i eficiència.
