El desenvolupament web és un terme ampli que engloba totes les activitats implicades en la creació d'un lloc web o aplicació web. Aquest tema t'introduirà als conceptes bàsics del desenvolupament web utilitzant Python, cobrint conceptes essencials, eines i frameworks.

Què és el Desenvolupament Web?

El desenvolupament web implica construir i mantenir llocs web. Es pot dividir en tres categories principals:

  • Desenvolupament Front-end: Es tracta dels aspectes visuals d'un lloc web amb els quals els usuaris interactuen.
  • Desenvolupament Back-end: Es centra en la lògica del servidor, bases de dades i funcionalitat de l'aplicació.
  • Desenvolupament Full-stack: Combina tant el desenvolupament front-end com el back-end.

Per què Utilitzar Python per al Desenvolupament Web?

Python és un llenguatge de programació versàtil i potent que s'utilitza àmpliament en el desenvolupament web a causa de:

  • Llegibilitat i Simplicitat: La sintaxi de Python és neta i fàcil d'entendre.
  • Biblioteques i Frameworks Extensos: Python ofereix biblioteques i frameworks robustos com Django i Flask.
  • Suport Comunitari: Una gran comunitat de desenvolupadors proporciona recursos i suport extensos.

Configuració del Teu Entorn

Abans de submergir-te en el desenvolupament web amb Python, necessites configurar el teu entorn de desenvolupament.

Instal·lant Python

Assegura't de tenir Python instal·lat al teu sistema. Pots descarregar-lo des de python.org.

Instal·lant un Editor de Codi

Tria un editor de codi o IDE (Entorn de Desenvolupament Integrat) com:

  • VS Code
  • PyCharm
  • Sublime Text

Instal·lant un Entorn Virtual

Un entorn virtual ajuda a gestionar les dependències dels teus projectes. Instal·la'l utilitzant pip:

pip install virtualenv

Crea un entorn virtual:

virtualenv myenv

Activa l'entorn virtual:

  • A Windows: 
    myenv\Scripts\activate
  • A macOS/Linux: 
    source myenv/bin/activate

Introducció a Flask

Flask és un framework web lleuger per a Python. És fàcil de configurar i perfecte per a principiants.

Instal·lant Flask

Instal·la Flask utilitzant pip:

pip install Flask

Creant una Aplicació Simple amb Flask

Crea un fitxer anomenat app.py i afegeix el següent codi:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, World!"

if __name__ == '__main__':
    app.run(debug=True)

Explicació:

  • Importar Flask: Importa la classe Flask del mòdul flask.
  • Crear una Instància: Crea una instància de la classe Flask.
  • Definir una Ruta: Utilitza el decorador @app.route per definir la ruta URL per a la pàgina principal.
  • Executar l'Aplicació: Utilitza app.run() per executar l'aplicació.

Executa l'aplicació:

python app.py

Visita http://127.0.0.1:5000/ al teu navegador web per veure el resultat.

Introducció a Django

Django és un framework web de Python d'alt nivell que fomenta el desenvolupament ràpid i un disseny net i pragmàtic.

Instal·lant Django

Instal·la Django utilitzant pip:

pip install django

Creant un Projecte Django

Crea un nou projecte Django:

django-admin startproject myproject

Navega al directori del projecte i executa el servidor de desenvolupament:

cd myproject
python manage.py runserver

Visita http://127.0.0.1:8000/ per veure la pàgina de benvinguda per defecte de Django.

Creant una Aplicació Django

Crea una nova aplicació dins del teu projecte:

python manage.py startapp myapp

Afegeix l'aplicació a la configuració del teu projecte a myproject/settings.py:

INSTALLED_APPS = [
    ...
    'myapp',
]

Defineix una vista a myapp/views.py:

from django.http import HttpResponse

def home(request):
    return HttpResponse("Hello, World!")

Mapeja la vista a una URL a myapp/urls.py:

from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]

Inclou les URLs de l'aplicació al urls.py del projecte:

from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

Executa el servidor de nou i visita http://127.0.0.1:8000/ per veure el resultat.
