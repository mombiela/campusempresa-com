La biblioteca estàndard de Python és una col·lecció de mòduls i paquets que vénen amb Python, proporcionant una àmplia gamma de funcionalitats de manera immediata. Aquesta visió general t'introduirà a alguns dels mòduls més utilitzats, ajudant-te a entendre el seu propòsit i com utilitzar-los de manera efectiva.

Funcions integrades

Python ve amb un conjunt de funcions integrades que estan sempre disponibles. Aquestes funcions realitzen operacions bàsiques i són essencials per a la programació diària.

  • print(): Mostra dades a la consola.
  • len(): Retorna la longitud d'un objecte.
  • type(): Retorna el tipus d'un objecte.
  • int(), float(), str(): Funcions de conversió de tipus.

Exemple

print("Hola, món!")  # Mostra: Hola, món!
print(len("Hola"))   # Mostra: 4
print(type(42))      # Mostra: <class 'int'>
print(int("123"))    # Mostra: 123

Mòdul `math`

El mòdul math proporciona funcions i constants matemàtiques.

  • Funcions comunes:
    • math.sqrt(x): Retorna l'arrel quadrada de x.
    • math.pow(x, y): Retorna x elevat a la potència de y.
    • math.sin(x), math.cos(x), math.tan(x): Funcions trigonomètriques.
  • Constants:
    • math.pi: El valor de π.
    • math.e: El valor d'e (nombre d'Euler).

Exemple

import math

print(math.sqrt(16))    # Mostra: 4.0
print(math.pow(2, 3))   # Mostra: 8.0
print(math.pi)          # Mostra: 3.141592653589793

Mòdul `datetime`

El mòdul datetime proporciona classes per manipular dates i hores.

  • Classes:
    • datetime.date: Representa una data (any, mes, dia).
    • datetime.time: Representa una hora (hora, minut, segon, microsegon).
    • datetime.datetime: Combina data i hora.
  • Mètodes:
    • datetime.datetime.now(): Retorna la data i hora actuals.
    • datetime.datetime.strptime(date_string, format): Analitza una cadena en un objecte datetime.
    • datetime.datetime.strftime(format): Dona format a un objecte datetime en una cadena.

Exemple

from datetime import datetime

now = datetime.now()
print(now)  # Mostra: data i hora actuals

date_string = "2023-10-01"
date_object = datetime.strptime(date_string, "%Y-%m-%d")
print(date_object)  # Mostra: 2023-10-01 00:00:00

formatted_date = now.strftime("%Y-%m-%d %H:%M:%S")
print(formatted_date)  # Mostra: data i hora actuals en el format especificat

Mòdul `os`

El mòdul os proporciona una manera d'utilitzar funcionalitats dependents del sistema operatiu.

  • Funcions comunes:
    • os.getcwd(): Retorna el directori de treball actual.
    • os.listdir(path): Retorna una llista d'entrades en el directori donat per path.
    • os.mkdir(path): Crea un directori anomenat path.
    • os.remove(path): Elimina el fitxer path.

Exemple

import os

print(os.getcwd())  # Mostra: directori de treball actual

os.mkdir("new_directory")
print(os.listdir("."))  # Mostra: llista de fitxers i directoris en el directori actual

os.remove("new_directory")

Mòdul `sys`

El mòdul sys proporciona accés a algunes variables utilitzades o mantingudes per l'intèrpret i a funcions que interactuen fortament amb l'intèrpret.

  • Funcions comunes:
    • sys.argv: Una llista d'arguments de línia de comandes passats al script.
    • sys.exit([arg]): Surt de Python.
    • sys.path: Una llista de cadenes que especifica el camí de cerca per als mòduls.

Exemple

import sys

print(sys.argv)  # Mostra: llista d'arguments de línia de comandes

sys.exit()  # Surt del script

Mòdul `json`

El mòdul json proporciona una manera fàcil de codificar i descodificar dades en format JSON.

  • Funcions comunes:
    • json.dumps(obj): Serialitza obj en una cadena amb format JSON.
    • json.loads(s): Deserialitza s (una cadena amb format JSON) en un objecte Python.

Exemple

import json

data = {"name": "Alice", "age": 25}
json_string = json.dumps(data)
print(json_string)  # Mostra: {"name": "Alice", "age": 25}

parsed_data = json.loads(json_string)
print(parsed_data)  # Mostra: {'name': 'Alice', 'age': 25}

Conclusió

La biblioteca estàndard de Python és una eina poderosa que proporciona una àmplia gamma de funcionalitats, des d'operacions bàsiques fins a manipulacions complexes de dades. Familiaritzar-te amb aquests mòduls millorarà enormement la teva capacitat per escriure codi Python eficient i efectiu. A mesura que progressis de nivell principiant a avançat, et trobaràs confiant més en aquests mòduls integrats per resoldre problemes i construir aplicacions.
