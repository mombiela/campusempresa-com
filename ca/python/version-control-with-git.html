Control de Versions amb Git

Introducció al Control de Versions

  • Definició: El control de versions és un sistema que registra els canvis en un fitxer o conjunt de fitxers al llarg del temps perquè puguis recordar versions específiques més tard.
  • Importància: Ajuda a fer un seguiment dels canvis, col·laborar amb altres i mantenir un historial del teu projecte.

Què és Git?

  • Definició: Git és un sistema de control de versions distribuït dissenyat per gestionar tot, des de projectes petits fins a molt grans, amb velocitat i eficiència.
  • Característiques Clau:
    • Distribuït: Cada desenvolupador té una còpia completa del repositori.
    • Rendiment: Gestió ràpida i eficient de projectes grans.
    • Ramificació i Fusió: Fàcil de crear, gestionar i fusionar branques.

Configuració de Git

Instal·lació

  • Windows: Descarrega des de git-scm.com i segueix les instruccions d'instal·lació.
  • Mac: Usa Homebrew: brew install git
  • Linux: Usa el gestor de paquets: sudo apt-get install git (basat en Debian) o sudo yum install git (basat en Red Hat)

Configuració

  • Configura el teu nom d'usuari i correu electrònic: 
    git config --global user.name "El Teu Nom"
git config --global user.email "el.teu.correu@example.com"

Comandes Bàsiques de Git

Inicialitzar un Repositori

  • Comanda: git init
  • Explicació: Inicialitza un nou repositori Git al directori actual.

Clonar un Repositori

  • Comanda: git clone <repository-url>
  • Explicació: Crea una còpia d'un repositori existent.

Comprovar l'Estat

  • Comanda: git status
  • Explicació: Mostra l'estat dels canvis com a no seguits, modificats o preparats.

Afegir Canvis

  • Comanda: git add <file>
  • Explicació: Afegeix els canvis en el fitxer especificat a l'àrea de preparació.
  • Exemple: 
    git add script.py

Cometre Canvis

  • Comanda: git commit -m "missatge del commit"
  • Explicació: Registra els canvis al repositori amb un missatge.
  • Exemple: 
    git commit -m "Commit inicial"

Veure l'Historial de Commits

  • Comanda: git log
  • Explicació: Mostra l'historial de commits del repositori.

Ramificació i Fusió

Crear una Branca

  • Comanda: git branch <branch-name>
  • Explicació: Crea una nova branca.
  • Exemple: 
    git branch feature-xyz

Canviar de Branca

  • Comanda: git checkout <branch-name>
  • Explicació: Canvia a la branca especificada.
  • Exemple: 
    git checkout feature-xyz

Fusionar Branques

  • Comanda: git merge <branch-name>
  • Explicació: Fusiona la branca especificada a la branca actual.
  • Exemple: 
    git merge feature-xyz

Treballar amb Repositoris Remots

Afegir un Repositori Remot

  • Comanda: git remote add <name> <url>
  • Explicació: Afegeix un repositori remot.
  • Exemple: 
    git remote add origin https://github.com/user/repo.git

Empènyer Canvis

  • Comanda: git push <remote> <branch>
  • Explicació: Empeny els canvis al repositori remot i branca especificats.
  • Exemple: 
    git push origin main

Estirar Canvis

  • Comanda: git pull <remote> <branch>
  • Explicació: Recupera i fusiona els canvis del repositori remot i branca especificats.
  • Exemple: 
    git pull origin main

Gestionar Conflictes

  • Definició: Els conflictes ocorren quan els canvis en diferents branques es contradiuen.
  • Resolució:
    1. Identifica el conflicte usant git status.
    2. Obre el fitxer conflictiu i resol manualment el conflicte.
    3. Afegeix el fitxer resolt usant git add <file>.
    4. Comet els canvis usant git commit.

Funcions Avançades de Git

Emmagatzemar Canvis

  • Comanda: git stash
  • Explicació: Desa temporalment els canvis que no estan llestos per ser comesos.
  • Exemple: 
    git stash

Rebase

  • Comanda: git rebase <branch>
  • Explicació: Reaplica els commits sobre una altra punta base.
  • Exemple: 
    git rebase main

Conclusió

El control de versions amb Git és una habilitat essencial per a qualsevol programador. Et permet fer un seguiment dels canvis, col·laborar amb altres i gestionar l'historial del teu projecte de manera eficient. Dominant les comandes de Git i entenent les seves característiques, pots millorar significativament el teu flux de treball i productivitat.
