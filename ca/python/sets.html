Introducció als Conjunts

Els conjunts són un tipus de dades de col·lecció en Python que són desordenats, mutables i no permeten elements duplicats. Són útils per emmagatzemar elements únics i realitzar operacions com la unió, la intersecció i la diferència.

Conceptes Clau

  • Desordenats: Els elements d'un conjunt no tenen un ordre definit.
  • Mutables: Pots afegir o eliminar elements d'un conjunt.
  • Sense Duplicats: Els conjunts gestionen automàticament les entrades duplicades emmagatzemant només una instància de cada element.

Creació de Conjunts

Pots crear un conjunt utilitzant claus {} o la funció set().

Exemples

# Creació d'un conjunt amb claus
my_set = {1, 2, 3, 4, 5}
print(my_set)  # Sortida: {1, 2, 3, 4, 5}

# Creació d'un conjunt utilitzant la funció set()
another_set = set([1, 2, 3, 4, 5])
print(another_set)  # Sortida: {1, 2, 3, 4, 5}

Operacions Bàsiques amb Conjunts

Afegir i Eliminar Elements

  • Afegir: Utilitza el mètode add().
  • Eliminar: Utilitza els mètodes remove() o discard().

Exemples

# Afegir un element
my_set.add(6)
print(my_set)  # Sortida: {1, 2, 3, 4, 5, 6}

# Eliminar un element
my_set.remove(3)
print(my_set)  # Sortida: {1, 2, 4, 5, 6}

# Descartar un element (sense error si l'element no es troba)
my_set.discard(10)  # Sense error
print(my_set)  # Sortida: {1, 2, 4, 5, 6}

Operacions amb Conjunts

Unió, Intersecció, Diferència i Diferència Simètrica

  • Unió: Combina tots els elements únics de tots dos conjunts.
  • Intersecció: Retorna només els elements que estan presents en tots dos conjunts.
  • Diferència: Retorna els elements que estan en el primer conjunt però no en el segon.
  • Diferència Simètrica: Retorna els elements que estan en qualsevol dels conjunts però no en tots dos.

Exemples

set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}

# Unió
union_set = set_a.union(set_b)
print(union_set)  # Sortida: {1, 2, 3, 4, 5, 6}

# Intersecció
intersection_set = set_a.intersection(set_b)
print(intersection_set)  # Sortida: {3, 4}

# Diferència
difference_set = set_a.difference(set_b)
print(difference_set)  # Sortida: {1, 2}

# Diferència Simètrica
sym_diff_set = set_a.symmetric_difference(set_b)
print(sym_diff_set)  # Sortida: {1, 2, 5, 6}

Operacions Avançades amb Conjunts

Subconjunt i Superconjunt

  • Subconjunt: Un conjunt és un subconjunt d'un altre si tots els elements del primer conjunt estan en el segon conjunt.
  • Superconjunt: Un conjunt és un superconjunt d'un altre si conté tots els elements del segon conjunt.

Exemples

set_c = {1, 2}
set_d = {1, 2, 3, 4}

# Subconjunt
is_subset = set_c.issubset(set_d)
print(is_subset)  # Sortida: True

# Superconjunt
is_superset = set_d.issuperset(set_c)
print(is_superset)  # Sortida: True

Conjunts Congelats

Els conjunts congelats són versions immutables dels conjunts. Un cop creats, no es poden modificar.

Exemples

# Creació d'un conjunt congelat
frozen_set = frozenset([1, 2, 3, 4])
print(frozen_set)  # Sortida: frozenset({1, 2, 3, 4})

# Intentar afegir un element generarà un error
# frozen_set.add(5)  # AttributeError: 'frozenset' object has no attribute 'add'

Conclusió

Els conjunts en Python són una eina poderosa per gestionar col·leccions d'elements únics. Admeten diverses operacions que poden ser molt útils en diferents escenaris de programació, des de proves simples de pertinença fins a operacions matemàtiques complexes. Entendre com utilitzar eficaçment els conjunts pot millorar enormement la teva capacitat per escriure codi Python eficient i net.
