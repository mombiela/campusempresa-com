Introducció a PyTest

PyTest és un potent marc de proves per a Python que facilita l'escriptura de casos de prova simples i escalables. És àmpliament utilitzat per la seva simplicitat i flexibilitat.

  • Facilitat d'ús: PyTest és fàcil de començar i requereix una configuració mínima.
  • Flexibilitat: Suporta fixtures, proves parametritzades i plugins.
  • Compatibilitat: Funciona amb suites de proves existents de unittest o nose.

Instal·lant PyTest

Per començar amb PyTest, cal instal·lar-lo. Pots instal·lar PyTest utilitzant pip:

pip install pytest

Escriure la teva primera prova

Comencem amb un exemple simple per entendre com funciona PyTest.

Exemple: Provant una funció simple

Considera una funció simple que suma dos números:

# my_module.py
def add(a, b):
    return a + b

Per provar aquesta funció utilitzant PyTest, crea un nou fitxer anomenat test_my_module.py:

# test_my_module.py
from my_module import add

def test_add():
    assert add(1, 2) == 3
    assert add(-1, 1) == 0
    assert add(-1, -1) == -2

Per executar la prova, simplement executa la següent comanda al teu terminal:

pytest

Entenent les afirmacions

Les afirmacions són la columna vertebral de qualsevol prova. En PyTest, utilitzes l'instrucció assert per comprovar si una condició és certa. Si la condició és falsa, la prova falla.

  • Afirmació bàsica: assert expressió
  • Amb missatge: assert expressió, "missatge"

Exemple: Utilitzant afirmacions

def test_example():
    assert 1 == 1
    assert "hello".upper() == "HELLO", "La conversió de cadena a majúscules ha fallat"

Fixtures

Les fixtures són una manera de proporcionar una base fixa sobre la qual les proves poden executar-se de manera fiable i repetida. Ajuden a configurar i desmuntar l'entorn per a les proves.

Exemple: Utilitzant fixtures

import pytest

@pytest.fixture
def sample_data():
    return {"key1": "value1", "key2": "value2"}

def test_sample_data(sample_data):
    assert sample_data["key1"] == "value1"
    assert sample_data["key2"] == "value2"

Proves parametritzades

PyTest et permet executar una funció de prova diverses vegades amb diferents arguments utilitzant el decorador @pytest.mark.parametrize.

Exemple: Proves parametritzades

import pytest

@pytest.mark.parametrize("a, b, expected", [
    (1, 2, 3),
    (-1, 1, 0),
    (-1, -1, -2)
])
def test_add(a, b, expected):
    assert add(a, b) == expected

Executant proves

Pots executar proves de diverses maneres utilitzant PyTest:

  • Executar totes les proves: pytest
  • Executar un fitxer de prova específic: pytest test_my_module.py
  • Executar una funció de prova específica: pytest test_my_module.py::test_add

Funcions avançades

Marcadors

Els marcadors s'utilitzen per categoritzar proves i executar-les selectivament.

@pytest.mark.slow
def test_slow_function():
    pass

# Executar proves marcades com a lentes
pytest -m slow

Plugins

PyTest suporta una àmplia gamma de plugins per estendre la seva funcionalitat. Pots trobar plugins a PyPI.

Exemple: Utilitzant un plugin

pip install pytest-cov

Afegeix el següent a la teva comanda de prova per generar un informe de cobertura:

pytest --cov=my_module

Conclusió

PyTest és un marc de proves versàtil i fàcil d'utilitzar que pot manejar proves unitàries simples així com proves funcionals complexes. Aprofitant les fixtures, les proves parametritzades i els plugins, pots crear suites de proves robustes i mantenibles.

  • Comença simple: Comença amb afirmacions bàsiques i proves simples.
  • Utilitza fixtures: Simplifica els processos de configuració i desmuntatge.
  • Parametritza: Executa proves amb múltiples conjunts de dades.
  • Explora plugins: Extén la funcionalitat segons sigui necessari.

Dominant PyTest, pots assegurar-te que el teu codi Python sigui fiable i lliure d'errors.
