Introducció a les Funcions

Les funcions són un bloc fonamental en la programació en Python. Permeten encapsular codi en blocs reutilitzables, fent els teus programes més modulars, llegibles i mantenibles.

Conceptes Clau

  • Definició de Funció: El procés de crear una funció.
  • Crida de Funció: El procés d'executar una funció.
  • Paràmetres: Variables que es passen a una funció.
  • Valor de Retorn: El valor que una funció retorna després de l'execució.

Definició d'una Funció Simple

Per definir una funció en Python, utilitzes la paraula clau def seguida del nom de la funció i parèntesis ().

Exemple

def greet():
    print("Hola, Món!")
  • def: Paraula clau per definir una funció.
  • greet: Nom de la funció.
  • (): Parèntesis que poden contenir paràmetres.
  • print("Hola, Món!"): El cos de la funció, que està indentat.

Crida d'una Funció

Per executar la funció, simplement la crides pel seu nom seguit de parèntesis.

greet()

Funcions amb Paràmetres

Les funcions poden prendre paràmetres, que permeten passar dades a elles.

Exemple

def greet(name):
    print(f"Hola, {name}!")
  • name: Paràmetre que la funció accepta.

Crida d'una Funció amb Paràmetres

greet("Alice")

Això mostrarà:

Hola, Alice!

Funcions amb Múltiples Paràmetres

Pots definir funcions amb múltiples paràmetres separant-los amb comes.

Exemple

def add(a, b):
    return a + b
  • a, b: Paràmetres que la funció accepta.
  • return a + b: La funció retorna la suma de a i b.

Crida d'una Funció amb Múltiples Paràmetres

result = add(3, 5)
print(result)

Això mostrarà:

8

Valors de Paràmetre per Defecte

Pots proporcionar valors per defecte per als paràmetres. Si no es proporciona un paràmetre durant la crida de la funció, s'utilitza el valor per defecte.

Exemple

def greet(name="Món"):
    print(f"Hola, {name}!")

Crida d'una Funció amb Paràmetres per Defecte

greet()
greet("Alice")

Això mostrarà:

Hola, Món!
Hola, Alice!

Arguments per Paraula Clau

Quan crides funcions, pots utilitzar arguments per paraula clau per especificar quins valors de paràmetre estàs proporcionant.

Exemple

def describe_pet(pet_name, animal_type="gos"):
    print(f"Tinc un {animal_type} anomenat {pet_name}.")

Crida d'una Funció amb Arguments per Paraula Clau

describe_pet(pet_name="Whiskers", animal_type="gat")
describe_pet(pet_name="Buddy")

Això mostrarà:

Tinc un gat anomenat Whiskers.
Tinc un gos anomenat Buddy.

Arguments de Longitud Variable

Python et permet definir funcions que accepten un nombre arbitrari d'arguments utilitzant *args i **kwargs.

Exemple amb `*args`

def sum_all(*args):
    return sum(args)

Crida d'una Funció amb `*args`

result = sum_all(1, 2, 3, 4)
print(result)

Això mostrarà:

10

Exemple amb `**kwargs`

def print_details(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

Crida d'una Funció amb `**kwargs`

print_details(name="Alice", age=30, city="Nova York")

Això mostrarà:

name: Alice
age: 30
city: Nova York

Funcions Lambda

Les funcions lambda són petites funcions anònimes definides utilitzant la paraula clau lambda. Són útils per a funcions curtes i d'un sol ús.

Exemple

add = lambda x, y: x + y
print(add(2, 3))

Això mostrarà:

5

Conclusió

Les funcions són essencials per escriure codi net, eficient i reutilitzable en Python. Entenent com definir i utilitzar funcions, pots millorar significativament l'estructura i la llegibilitat dels teus programes. Practica definint diversos tipus de funcions i utilitzant diferents tipus de paràmetres per convertir-te en un expert en l'ús de funcions.
