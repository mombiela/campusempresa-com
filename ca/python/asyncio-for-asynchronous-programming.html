Introducció a Asyncio

Asyncio és una biblioteca en Python utilitzada per escriure codi concurrent utilitzant la sintaxi async/await. Està dissenyada per gestionar operacions d'E/S asíncrones, fent-la ideal per a tasques com sol·licituds de xarxa, E/S de fitxers i altres operacions que es poden realitzar concurrentment.

Conceptes Clau

  • Programació Asíncrona: Un paradigma de programació que permet operacions no bloquejants, permetent a un programa gestionar múltiples tasques alhora.
  • Bucle d'Esdeveniments: El nucli de cada aplicació asyncio. Executa tasques i callbacks asíncrons, realitza operacions d'E/S de xarxa i gestiona subprocessos.
  • Coroutines: Funcions especials definides amb async def que es poden pausar i reprendre, permetent que altres operacions s'executin concurrentment.
  • Tasques: Embolcalls per a coroutines que permeten que siguin programades i executades pel bucle d'esdeveniments.
  • Futures: Objectes que representen el resultat d'una operació asíncrona.

Configurant Asyncio

Per utilitzar asyncio, cal importar-lo al vostre script de Python:

import asyncio

Escriure la Teva Primera Coroutine

Una coroutine es defineix utilitzant la sintaxi async def. Aquí tens un exemple senzill:

import asyncio

async def say_hello():
    print("Hola")
    await asyncio.sleep(1)
    print("Món")

# Executant la coroutine
asyncio.run(say_hello())

Explicació

  • async def say_hello(): Defineix una funció asíncrona.
  • await asyncio.sleep(1): Pausa la coroutine durant 1 segon, permetent que altres tasques s'executin.
  • asyncio.run(say_hello()): Executa la coroutine fins que es completi.

Utilitzant el Bucle d'Esdeveniments

El bucle d'esdeveniments és responsable d'executar tasques asíncrones. Pots obtenir el bucle d'esdeveniments actual utilitzant asyncio.get_event_loop() i executar tasques amb loop.run_until_complete().

import asyncio

async def main():
    print("Inici")
    await asyncio.sleep(2)
    print("Fi")

loop = asyncio.get_event_loop()
loop.run_until_complete(main())

Crear i Executar Múltiples Tasques

Pots executar múltiples coroutines concurrentment utilitzant asyncio.create_task().

import asyncio

async def task1():
    await asyncio.sleep(1)
    print("Tasca 1 completada")

async def task2():
    await asyncio.sleep(2)
    print("Tasca 2 completada")

async def main():
    task1 = asyncio.create_task(task1())
    task2 = asyncio.create_task(task2())
    
    await task1
    await task2

asyncio.run(main())

Explicació

  • asyncio.create_task(): Programa l'execució d'una coroutine.
  • await task1, await task2: Espera que les tasques es completin.

Gestionar Excepcions en Asyncio

Pots gestionar excepcions en tasques asyncio utilitzant blocs try-except.

import asyncio

async def faulty_task():
    await asyncio.sleep(1)
    raise ValueError("S'ha produït un error")

async def main():
    try:
        await faulty_task()
    except ValueError as e:
        print(f"S'ha capturat una excepció: {e}")

asyncio.run(main())

Ús Avançat: Sincronitzar Tasques

Asyncio proporciona diversos primitius de sincronització com asyncio.Lock, asyncio.Event, i asyncio.Queue.

Exemple: Utilitzant asyncio.Lock

import asyncio

lock = asyncio.Lock()

async def critical_section():
    async with lock:
        print("Entrant a la secció crítica")
        await asyncio.sleep(1)
        print("Sortint de la secció crítica")

async def main():
    await asyncio.gather(critical_section(), critical_section())

asyncio.run(main())

Explicació

  • asyncio.Lock(): Crea un objecte de bloqueig.
  • async with lock: Adquireix el bloqueig abans d'entrar a la secció crítica i l'allibera després de sortir.

Conclusió

Asyncio és una biblioteca potent per escriure codi concurrent en Python. Entenent els seus conceptes bàsics com el bucle d'esdeveniments, coroutines, tasques i primitius de sincronització, pots gestionar eficientment operacions asíncrones en les teves aplicacions. Practica amb els exemples proporcionats i explora característiques més avançades per dominar la programació asíncrona amb asyncio.
