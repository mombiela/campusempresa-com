Les eines de control de flux són essencials en la programació, ja que et permeten dictar l'ordre en què s'executen les instruccions. En Python, aquestes eines inclouen instruccions condicionals, bucles i altres estructures de control. Aquesta secció et guiarà des dels conceptes bàsics fins als més avançats.

Instruccions condicionals

Les instruccions condicionals et permeten executar certs fragments de codi basant-se en si una condició és certa o falsa.

Instrucció `if`

La instrucció if és la forma més bàsica d'instrucció condicional.

x = 10
if x > 5:
    print("x és més gran que 5")
  • Explicació: Si x és més gran que 5, es mostrarà el missatge "x és més gran que 5".

Instrucció `if-else`

La instrucció if-else proporciona un camí alternatiu d'execució quan la condició és falsa.

x = 3
if x > 5:
    print("x és més gran que 5")
else:
    print("x no és més gran que 5")
  • Explicació: Si x no és més gran que 5, es mostrarà el missatge "x no és més gran que 5".

Instrucció `if-elif-else`

La instrucció if-elif-else et permet comprovar múltiples condicions.

x = 7
if x > 10:
    print("x és més gran que 10")
elif x > 5:
    print("x és més gran que 5 però menor o igual a 10")
else:
    print("x és 5 o menys")
  • Explicació: Aquesta estructura comprova múltiples condicions en seqüència i executa el bloc de codi corresponent a la primera condició certa.

Bucles

Els bucles et permeten executar un bloc de codi múltiples vegades.

Bucle `while`

El bucle while continua executant-se mentre la condició sigui certa.

count = 0
while count < 5:
    print("El compte és:", count)
    count += 1
  • Explicació: Aquest bucle mostrarà el valor de count de 0 a 4.

Bucle `for`

El bucle for itera sobre una seqüència (com una llista, tupla o cadena).

fruits = ["poma", "plàtan", "cirera"]
for fruit in fruits:
    print(fruit)
  • Explicació: Aquest bucle mostrarà cada fruit de la llista.

Funció `range()`

La funció range() genera una seqüència de números, que sovint s'utilitza amb bucles for.

for i in range(5):
    print(i)
  • Explicació: Aquest bucle mostrarà números de 0 a 4.

Eines de control de flux en bucles

Instrucció `break`

La instrucció break surt del bucle prematurament.

for i in range(10):
    if i == 5:
        break
    print(i)
  • Explicació: Aquest bucle mostrarà números de 0 a 4 i després sortirà quan i sigui igual a 5.

Instrucció `continue`

La instrucció continue salta la iteració actual i procedeix a la següent.

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
  • Explicació: Aquest bucle mostrarà només números senars entre 0 i 9.

Clàusula `else` en bucles

La clàusula else en bucles s'executa després que el bucle es completi normalment (sense trobar una instrucció break).

for i in range(5):
    print(i)
else:
    print("Bucle completat")
  • Explicació: Aquest bucle mostrarà números de 0 a 4 i després mostrarà "Bucle completat".

Eines de control de flux avançades

Bucles imbricats

Els bucles es poden imbricar dins d'altres bucles.

for i in range(3):
    for j in range(2):
        print(f"i: {i}, j: {j}")
  • Explicació: Això mostrarà totes les combinacions de i i j dins dels rangs especificats.

Comprensions de llistes

Les comprensions de llistes proporcionen una manera concisa de crear llistes.

squares = [x**2 for x in range(10)]
print(squares)
  • Explicació: Això crearà una llista de quadrats per a números de 0 a 9.

Expressions generadores

Les expressions generadores són similars a les comprensions de llistes però utilitzen parèntesis i generen elements un a un.

squares_gen = (x**2 for x in range(10))
for square in squares_gen:
    print(square)
  • Explicació: Això generarà i mostrarà quadrats per a números de 0 a 9, un a un.

Conclusió

Les eines de control de flux són fonamentals en la programació en Python. Et permeten controlar l'execució del teu codi basant-se en condicions i bucles. Entendre aquestes eines et permetrà escriure codi més eficient i llegible. Practica utilitzant aquestes eines amb diversos exemples per convertir-te en un expert en la gestió del flux dels teus programes en Python.
