Introducció als Diccionaris

Els diccionaris són una estructura de dades fonamental en Python, utilitzada per emmagatzemar dades en parelles clau-valor. Són col·leccions mutables i desordenades que permeten una recuperació ràpida dels valors basant-se en les seves claus.

  • Parelles Clau-Valor: Cada element en un diccionari és una parella que consisteix en una clau i un valor.
  • Mutable: Els diccionaris es poden canviar després de la seva creació.
  • Desordenat: L'ordre dels elements en un diccionari no està garantit.

Creació de Diccionaris

Pots crear un diccionari utilitzant claus {} amb parelles clau-valor separades per dos punts :.

# Creació d'un diccionari buit
empty_dict = {}

# Creació d'un diccionari amb parelles clau-valor inicials
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}

Accés als Valors

Els valors en un diccionari es poden accedir utilitzant les seves claus corresponents.

# Accés als valors utilitzant claus
name = person["name"]
age = person["age"]

print(name)  # Sortida: Alice
print(age)   # Sortida: 30

Afegir i Modificar Entrades

Pots afegir noves parelles clau-valor o modificar les existents assignant un valor a una clau.

# Afegir una nova parella clau-valor
person["email"] = "alice@example.com"

# Modificar una parella clau-valor existent
person["age"] = 31

print(person)
# Sortida: {'name': 'Alice', 'age': 31, 'city': 'New York', 'email': 'alice@example.com'}

Eliminar Entrades

Les entrades es poden eliminar utilitzant la instrucció del o el mètode pop.

# Utilitzant la instrucció del
del person["city"]

# Utilitzant el mètode pop
email = person.pop("email")

print(person)  # Sortida: {'name': 'Alice', 'age': 31}
print(email)   # Sortida: alice@example.com

Mètodes del Diccionari

Els diccionaris tenen diversos mètodes integrats que els fan versàtils i potents.

| Mètode | Descripció | |----------------|-------------------------------------------------| | clear() | Elimina tots els elements del diccionari | | copy() | Retorna una còpia superficial del diccionari | | get(key) | Retorna el valor per a la clau especificada | | items() | Retorna un objecte vista dels elements del diccionari | | keys() | Retorna un objecte vista de les claus del diccionari | | values() | Retorna un objecte vista dels valors del diccionari| | update(dict) | Actualitza el diccionari amb un altre diccionari |

Exemple d'Ús dels Mètodes

# Utilitzant el mètode get
age = person.get("age")
print(age)  # Sortida: 31

# Utilitzant el mètode keys
keys = person.keys()
print(keys)  # Sortida: dict_keys(['name', 'age'])

# Utilitzant el mètode values
values = person.values()
print(values)  # Sortida: dict_values(['Alice', 31])

# Utilitzant el mètode items
items = person.items()
print(items)  # Sortida: dict_items([('name', 'Alice'), ('age', 31)])

# Utilitzant el mètode update
person.update({"city": "Boston", "email": "alice@newdomain.com"})
print(person)
# Sortida: {'name': 'Alice', 'age': 31, 'city': 'Boston', 'email': 'alice@newdomain.com'}

Conceptes Avançats de Diccionaris

Comprensions de Diccionaris

Les comprensions de diccionaris proporcionen una manera concisa de crear diccionaris.

# Creació d'un diccionari utilitzant comprensió
squares = {x: x*x per x en range(6)}
print(squares)
# Sortida: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Diccionaris Imbricats

Els diccionaris poden contenir altres diccionaris, permetent estructures de dades complexes.

# Creació d'un diccionari imbricat
students = {
    "Alice": {"age": 30, "city": "New York"},
    "Bob": {"age": 25, "city": "Los Angeles"}
}

# Accés als valors del diccionari imbricat
alice_age = students["Alice"]["age"]
bob_city = students["Bob"]["city"]

print(alice_age)  # Sortida: 30
print(bob_city)   # Sortida: Los Angeles

Maneig de Claus Inexistents

Utilitzar el mètode get o defaultdict del mòdul collections pot ajudar a manejar claus inexistents de manera elegant.

# Utilitzant el mètode get amb un valor per defecte
age = person.get("age", "No Disponible")
print(age)  # Sortida: 31

# Utilitzant defaultdict
from collections import defaultdict

# Creació d'un defaultdict amb tipus per defecte int
default_dict = defaultdict(int)
default_dict["count"] += 1

print(default_dict)  # Sortida: defaultdict(<class 'int'>, {'count': 1})

Conclusió

Els diccionaris són una estructura de dades versàtil i potent en Python, adequada per a una àmplia gamma d'aplicacions. Permeten un emmagatzematge i recuperació eficient de dades utilitzant parelles clau-valor. Comprenent els conceptes bàsics i avançats dels diccionaris, pots aprofitar tot el seu potencial en els teus projectes de programació en Python.
