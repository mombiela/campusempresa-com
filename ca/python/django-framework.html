Introducció a Django

Django és un framework web de Python d'alt nivell que fomenta el desenvolupament ràpid i el disseny net i pragmàtic. Està construït per desenvolupadors experimentats per fer-se càrrec de gran part de les molèsties del desenvolupament web, de manera que et puguis centrar en escriure la teva aplicació sense necessitat de reinventar la roda.

  • Característiques Clau:
    • Desenvolupament ràpid
    • Segur
    • Escalable
    • Versàtil

Configuració de Django

Abans de submergir-te en Django, necessites configurar el teu entorn.

  • Instal·lació:

    pip install django

  • Creació d'un Projecte:

    django-admin startproject myproject

  • Executar el Servidor de Desenvolupament:

    cd myproject
python manage.py runserver

Estructura del Projecte Django

Entendre l'estructura d'un projecte Django és crucial per a un desenvolupament efectiu.

  • Components Clau:
    • manage.py: Una utilitat de línia de comandes que et permet interactuar amb el teu projecte.
    • settings.py: Configuració per al teu projecte Django.
    • urls.py: Declaracions d'URL per al teu projecte.
    • wsgi.py: Un punt d'entrada per a servidors web compatibles amb WSGI per servir el teu projecte.

Creació d'una Aplicació Django

Els projectes Django es componen de múltiples aplicacions, cadascuna amb un propòsit específic.

  • Creació d'una Aplicació:

    python manage.py startapp myapp

  • Registrar l'Aplicació: Afegeix la teva aplicació a la llista INSTALLED_APPS a settings.py:

    INSTALLED_APPS = [
    ...
    'myapp',
]

Models i Bases de Dades

Els models de Django defineixen l'estructura de les teves taules de base de dades.

  • Creació d'un Model:

    from django.db import models

class Article(models.Model):
    title = models.CharField(maxlength=100)
    content = models.TextField()
    published_date = models.DateTimeField(auto_now_add=True)

  • Aplicar Migracions:

    python manage.py makemigrations
python manage.py migrate

Vistes i Plantilles

Les vistes gestionen la lògica de la teva aplicació, mentre que les plantilles gestionen la presentació.

  • Creació d'una Vista:

    from django.shortcuts import render
from .models import Article

def index(request):
    articles = Article.objects.all()
    return render(request, 'index.html', {'articles': articles})

  • Creació d'una Plantilla:

    <!-- templates/index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Llista d'Articles</title>
</head>
<body>
    <h1>Articles</h1>
    <ul>
        {% for article in articles %}
            <li>{{ article.title }} - {{ article.published_date }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Enrutament d'URL

L'enrutament d'URL mapatge URLs a vistes.

  • Definició d'URLs: 
    from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

Formularis i Entrada d'Usuari

Els formularis de Django gestionen l'entrada d'usuari i la validació.

  • Creació d'un Formulari:

    from django import forms

class ArticleForm(forms.ModelForm):
    class Meta:
        model = Article
        fields = ['title', 'content']

  • Gestió de l'Enviament del Formulari:

    from django.shortcuts import render, redirect
from .forms import ArticleForm

def create_article(request):
    if request.method == 'POST':
        form = ArticleForm(request.POST)
        if form.is_valid():
            form.save()
            return redirect('index')
    else:
        form = ArticleForm()
    return render(request, 'create_article.html', {'form': form})

  • Plantilla per al Formulari:

    <!-- templates/create_article.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Crear Article</title>
</head>
<body>
    <h1>Crear Article</h1>
    <form method="post">
        {% csrf_token %}
        {{ form.as_p }}
        <button type="submit">Guardar</button>
    </form>
</body>
</html>

Interfície d'Administració

Django proporciona una interfície d'administració integrada per gestionar els teus models.

  • Activació de la Interfície d'Administració:

    from django.contrib import admin
from .models import Article

admin.site.register(Article)

  • Accés a la Interfície d'Administració:

    • Executa el servidor de desenvolupament i navega a /admin.
    • Inicia sessió utilitzant les credencials de superusuari que has creat.

Conclusió

Django és un framework potent que simplifica el desenvolupament web proporcionant un conjunt robust d'eines i característiques. Entenent els seus components bàsics—models, vistes, plantilles i enrutament d'URL—pots construir aplicacions web escalables i mantenibles de manera eficient. Continua explorant la documentació extensa de Django i els recursos de la comunitat per aprofundir en els teus coneixements i habilitats.
