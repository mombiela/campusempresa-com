Introducció

En Python, les excepcions són una manera de gestionar els errors de manera elegant. Tot i que Python proporciona moltes excepcions integrades, de vegades necessites crear les teves pròpies excepcions personalitzades per gestionar situacions específiques en el teu codi. Aquesta secció et guiarà a través del procés de crear i utilitzar excepcions personalitzades, des de nivells bàsics fins a avançats.

Per què utilitzar excepcions personalitzades?

  • Especificitat: Les excepcions personalitzades et permeten gestionar condicions d'error específiques en el teu codi.
  • Llegibilitat: Fan que el teu codi sigui més llegible i mantenible indicant clarament què ha anat malament.
  • Depuració: Les excepcions personalitzades poden proporcionar missatges d'error més detallats, facilitant la depuració.

Creació d'una excepció personalitzada bàsica

Crear una excepció personalitzada en Python és senzill. Simplement defineixes una nova classe que hereta de la classe integrada Exception.

Exemple

class MyCustomError(Exception):
    """Excepció llançada per condicions d'error personalitzades."""
    pass

# Ús
try:
    raise MyCustomError("Alguna cosa ha anat malament!")
except MyCustomError as e:
    print(e)

Explicació

  • Definició de la classe: class MyCustomError(Exception): defineix una nova classe d'excepció que hereta de Exception.
  • Docstring: El docstring proporciona una descripció de l'excepció.
  • Llançament de l'excepció: raise MyCustomError("Alguna cosa ha anat malament!") llança l'excepció personalitzada.
  • Gestió de l'excepció: El bloc except MyCustomError as e captura i gestiona l'excepció personalitzada.

Afegint més funcionalitat

Pots afegir més funcionalitat a les teves excepcions personalitzades sobreescrivint el mètode __init__ i afegint atributs personalitzats.

Exemple

class DetailedError(Exception):
    """Excepció llançada per errors amb més detalls."""
    
    def __init__(self, message, error_code):
        self.message = message
        self.error_code = error_code
        super().__init__(self.message)

# Ús
try:
    raise DetailedError("Operació no vàlida", 404)
except DetailedError as e:
    print(f"Error: {e.message}, Codi: {e.error_code}")

Explicació

  • Atributs personalitzats: self.message i self.error_code emmagatzemen informació addicional sobre l'error.
  • Inicialització: super().__init__(self.message) crida el constructor de la classe base per inicialitzar l'excepció.

Millors pràctiques

  • Noms significatius: Dóna a les teves excepcions personalitzades noms significatius que indiquin clarament la condició d'error.
  • Documentació: Documenta sempre les teves excepcions personalitzades per explicar quan s'han d'utilitzar.
  • Herència: Considera crear una classe base d'excepció personalitzada si tens múltiples excepcions personalitzades relacionades.

Exemple

class BaseCustomError(Exception):
    """Classe base per a altres excepcions personalitzades"""
    pass

class InputError(BaseCustomError):
    """Excepció llançada per entrada no vàlida"""
    pass

class CalculationError(BaseCustomError):
    """Excepció llançada per errors en càlculs"""
    pass

Ús avançat

En escenaris més avançats, potser voldràs crear una jerarquia d'excepcions personalitzades o integrar-les amb mecanismes de registre.

Exemple

import logging

class BaseCustomError(Exception):
    """Classe base per a altres excepcions personalitzades"""
    pass

class InputError(BaseCustomError):
    """Excepció llançada per entrada no vàlida"""
    
    def __init__(self, message):
        self.message = message
        logging.error(f"InputError: {self.message}")
        super().__init__(self.message)

# Ús
try:
    raise InputError("Entrada d'usuari no vàlida")
except InputError as e:
    print(e)

Explicació

  • Registre: logging.error(f"InputError: {self.message}") registra el missatge d'error.
  • Herència: InputError hereta de BaseCustomError, permetent una jerarquia d'excepcions estructurada.

Conclusió

Les excepcions personalitzades en Python proporcionen una manera poderosa de gestionar condicions d'error específiques en el teu codi. Creant excepcions personalitzades significatives i ben documentades, pots fer que el teu codi sigui més llegible, mantenible i fàcil de depurar. Tant si ets un programador principiant com avançat, entendre com crear i utilitzar excepcions personalitzades és una habilitat essencial en la programació amb Python.
