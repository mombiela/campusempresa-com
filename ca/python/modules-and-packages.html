Introducció als Mòduls

Els mòduls en Python són fitxers que contenen codi Python. Aquest codi pot definir funcions, classes i variables. Un mòdul també pot incloure codi executable.

Crear un Mòdul

Per crear un mòdul, simplement guarda el codi que vols en un fitxer amb l'extensió .py.

# my_module.py

def greet(name):
    return f"Hola, {name}!"

def add(a, b):
    return a + b

Importar un Mòdul

Pots utilitzar la instrucció import per portar un mòdul al teu script.

# main.py

import my_module

print(my_module.greet("Alice"))  # Sortida: Hola, Alice!
print(my_module.add(5, 3))       # Sortida: 8

Utilitzar `from` per Importar Funcions Específiques

Pots importar funcions o variables específiques d'un mòdul utilitzant la paraula clau from.

# main.py

from my_module import greet

print(greet("Bob"))  # Sortida: Hola, Bob!

Aliasing de Mòduls

Els mòduls poden tenir un àlies utilitzant la paraula clau as.

# main.py

import my_module as mm

print(mm.greet("Charlie"))  # Sortida: Hola, Charlie!

Mòduls Incorporats

Python ve amb una biblioteca estàndard de mòduls que pots utilitzar directament.

Mòduls Incorporats Comunament Utilitzats

  • math: Proporciona funcions matemàtiques.
  • datetime: Proporciona classes per manipular dates i hores.
  • os: Proporciona una manera d'utilitzar funcionalitats dependents del sistema operatiu.
import math
import datetime
import os

print(math.sqrt(16))  # Sortida: 4.0
print(datetime.datetime.now())  # Sortida: Data i hora actuals
print(os.getcwd())  # Sortida: Directori de treball actual

Introducció als Paquets

Un paquet és una manera d'organitzar mòduls relacionats en una jerarquia de directoris. Un paquet és un directori que conté un fitxer especial anomenat __init__.py i pot contenir múltiples mòduls i sub-paquets.

Crear un Paquet

  1. Crea un directori per al paquet.
  2. Afegeix un fitxer __init__.py al directori.
  3. Afegeix mòduls al directori.
my_package/
    __init__.py
    module1.py
    module2.py

Importar des d'un Paquet

Pots importar mòduls d'un paquet utilitzant la notació de punts.

# my_package/module1.py

def func1():
    return "Funció 1"

# my_package/module2.py

def func2():
    return "Funció 2"

# main.py

from my_package import module1, module2

print(module1.func1())  # Sortida: Funció 1
print(module2.func2())  # Sortida: Funció 2

Importar Sub-paquets

Els paquets poden contenir sub-paquets, que són paquets dins de paquets.

my_package/
    __init__.py
    sub_package/
        __init__.py
        sub_module.py
# my_package/sub_package/sub_module.py

def sub_func():
    return "Sub Funció"

# main.py

from my_package.sub_package import sub_module

print(sub_module.sub_func())  # Sortida: Sub Funció

Conceptes Avançats de Mòduls i Paquets

Importacions Relatives

Les importacions relatives utilitzen un punt (.) per indicar els paquets actuals i pares implicats.

# my_package/module1.py

from .module2 import func2

def combined_func():
    return func2() + " & Funció 1"

La Variable `__all__`

La variable __all__ es pot utilitzar en un mòdul per definir una llista d'objectes públics d'aquest mòdul.

# my_package/module1.py

__all__ = ['func1']

def func1():
    return "Funció 1"

def _hidden_func():
    return "Funció Oculta"

El Fitxer `__init__.py`

El fitxer __init__.py es pot utilitzar per inicialitzar un paquet i també pot executar codi d'inicialització per al paquet.

# my_package/__init__.py

print("Inicialitzant my_package")

from .module1 import func1
from .module2 import func2

Conclusió

Els mòduls i paquets són conceptes fonamentals en Python que ajuden a organitzar i gestionar el codi de manera eficient. Entenent com crear, importar i utilitzar mòduls i paquets, pots escriure codi més modular, mantenible i reutilitzable. Tant si estàs treballant en un petit script com en una gran aplicació, aprofitar aquests conceptes millorarà significativament les teves habilitats de programació.
