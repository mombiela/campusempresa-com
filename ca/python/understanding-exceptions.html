Introducció a les Excepcions

Les excepcions són esdeveniments que poden alterar el flux normal d'un programa. Normalment s'utilitzen per gestionar errors o altres condicions excepcionals que es produeixen durant l'execució d'un programa.

Conceptes Clau

  • Excepció: Un error que es produeix durant l'execució d'un programa.
  • Gestió d'Excepcions: El procés de respondre a les excepcions quan es produeixen.
  • Bloc Try: Un bloc de codi que es prova per detectar errors mentre s'executa.
  • Bloc Except: Un bloc de codi que s'executa si es produeix un error en el bloc try.
  • Bloc Finally: Un bloc de codi que s'executa independentment de si es produeix una excepció o no.

Gestió Bàsica d'Excepcions

En Python, les excepcions es gestionen utilitzant els blocs try, except i opcionalment finally.

Exemple

try:
    # Codi que pot generar una excepció
    result = 10 / 0
except ZeroDivisionError:
    # Codi per gestionar l'excepció
    print("No es pot dividir per zero!")
finally:
    # Codi que s'executarà sigui com sigui
    print("Execució completada.")

Explicació

  • El bloc try conté codi que podria generar una excepció.
  • El bloc except captura i gestiona l'excepció.
  • El bloc finally conté codi que sempre s'executarà, independentment de si s'ha generat una excepció o no.

Gestió de Múltiples Excepcions

Pots gestionar múltiples excepcions especificant múltiples blocs except.

Exemple

try:
    value = int(input("Introdueix un número: "))
    result = 10 / value
except ValueError:
    print("Entrada no vàlida! Si us plau, introdueix un número.")
except ZeroDivisionError:
    print("No es pot dividir per zero!")
finally:
    print("Execució completada.")

Explicació

  • Múltiples blocs except et permeten gestionar diferents tipus d'excepcions per separat.
  • Això fa que el teu codi sigui més robust i fàcil de depurar.

Utilitzar Else amb Try-Except

El bloc else es pot utilitzar per executar codi si no es generen excepcions en el bloc try.

Exemple

try:
    value = int(input("Introdueix un número: "))
    result = 10 / value
except ZeroDivisionError:
    print("No es pot dividir per zero!")
else:
    print(f"El resultat és {result}")
finally:
    print("Execució completada.")

Explicació

  • El bloc else només s'executa si el bloc try no genera cap excepció.
  • Això pot ser útil per a codi que només s'hauria d'executar si no s'han produït errors.

Generar Excepcions

Pots generar excepcions utilitzant la paraula clau raise.

Exemple

def check_age(age):
    if age < 18:
        raise ValueError("L'edat ha de ser almenys 18.")
    return "Accés concedit."

try:
    print(check_age(15))
except ValueError as e:
    print(e)

Explicació

  • La paraula clau raise s'utilitza per desencadenar manualment una excepció.
  • Això pot ser útil per fer complir certes condicions en el teu codi.

Excepcions Personalitzades

Pots definir les teves pròpies excepcions creant una nova classe que hereti de la classe Exception incorporada.

Exemple

class CustomError(Exception):
    pass

def check_value(value):
    if value < 0:
        raise CustomError("El valor no pot ser negatiu.")
    return value

try:
    print(check_value(-5))
except CustomError as e:
    print(e)

Explicació

  • Les excepcions personalitzades et permeten crear missatges d'error més específics i rutines de gestió.
  • Això pot fer que el teu codi sigui més llegible i mantenible.

Resum

  • Les excepcions s'utilitzen per gestionar errors i altres condicions excepcionals en un programa.
  • Els blocs try, except i finally s'utilitzen per gestionar excepcions.
  • Es poden gestionar múltiples excepcions utilitzant múltiples blocs except.
  • El bloc else es pot utilitzar per executar codi si no es generen excepcions.
  • Les excepcions es poden generar manualment utilitzant la paraula clau raise.
  • Es poden crear excepcions personalitzades definint noves classes que heretin de la classe Exception.

Comprendre i utilitzar efectivament les excepcions és crucial per escriure programes Python robustos i resistents als errors.
