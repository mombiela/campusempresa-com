Introducció

Python és un llenguatge de programació versàtil i àmpliament utilitzat, conegut per la seva llegibilitat i simplicitat. Aquesta secció cobrirà la sintaxi fonamental i els tipus de dades bàsics en Python, proporcionant una base sòlida per als principiants.

Sintaxi de Python

La sintaxi de Python es refereix al conjunt de regles que defineixen com s'escriu i s'interpreta un programa en Python.

Conceptes Clau

  • Indentació: Python utilitza la indentació per definir blocs de codi. La indentació consistent és crucial.
  • Comentaris: Utilitza # per a comentaris d'una sola línia i cometes triples (''' o """) per a comentaris de múltiples línies.
  • Variables: Les variables no requereixen una declaració explícita i poden canviar de tipus dinàmicament.

Exemples de Codi

Indentació

if True:
    print("Això està indentat")
    if True:
        print("Això està més indentat")

Comentaris

# Això és un comentari d'una sola línia

"""
Això és un
comentari de múltiples línies
"""

Variables

x = 5       # x és un enter
x = "Hola"  # Ara x és una cadena

Tipus de Dades Bàsics

Python admet diversos tipus de dades bàsics que són essencials per a qualsevol tasca de programació.

Tipus de Dades Clau

  • Enters: Nombres sencers, per exemple, 1, 42, -7
  • Flotants: Nombres decimals, per exemple, 3.14, 0.001, -2.5
  • Cadenes: Text, per exemple, "Hola, Món!", 'Python'
  • Booleans: Valors True o False, per exemple, True, False

Exemples de Codi

Enters i Flotants

a = 10      # Enter
b = 3.14    # Flotant
c = a + b   # El resultat és un flotant: 13.14

Cadenes

salutació = "Hola"
nom = 'Alice'
missatge = salutació + ", " + nom + "!"  # Concatenació
print(missatge)  # Sortida: Hola, Alice!

Booleans

python_divertit = True
cel_blau = False

print(python_divertit)  # Sortida: True
print(cel_blau)         # Sortida: False

Resum

Entendre la sintaxi de Python i els tipus de dades bàsics és essencial per escriure programes efectius en Python. Punts clau a recordar:

  • Python utilitza la indentació per definir blocs de codi.
  • Els comentaris es creen utilitzant # per a una sola línia i cometes triples per a múltiples línies.
  • Les variables poden canviar de tipus dinàmicament.
  • Els tipus de dades bàsics inclouen enters, flotants, cadenes i booleans.

En dominar aquests fonaments, estaràs ben preparat per abordar conceptes i tasques de programació més complexes en Python.
