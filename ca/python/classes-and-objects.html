Introducció a Classes i Objectes

En Python, les classes i els objectes són conceptes fonamentals de la Programació Orientada a Objectes (OOP). Ajuden a organitzar el codi i crear components reutilitzables.

Conceptes Clau

  • Classe: Un plànol per crear objectes. Defineix un conjunt d'atributs i mètodes que els objectes creats tindran.
  • Objecte: Una instància d'una classe. Es crea utilitzant la classe i pot utilitzar els atributs i mètodes definits en la classe.
  • Atributs: Variables que pertanyen a una classe o un objecte.
  • Mètodes: Funcions que pertanyen a una classe i normalment operen sobre els seus atributs.

Definint una Classe

Per definir una classe en Python, utilitza la paraula clau class seguida del nom de la classe i un dos punts. Dins de la classe, defineix atributs i mètodes.

class Dog:
    # Atribut de classe
    species = "Canis familiaris"

    # Inicialitzador / Atributs de la instància
    def __init__(self, name, age):
        self.name = name
        self.age = age

    # Mètode
    def description(self):
        return f"{self.name} is {self.age} years old"

Explicació

  • class Dog:: Defineix una nova classe anomenada Dog.
  • species: Un atribut de classe compartit per totes les instàncies de la classe.
  • __init__: El mètode inicialitzador que es crida quan es crea un objecte. Inicialitza els atributs de la instància name i age.
  • description: Un mètode que retorna una cadena que descriu el gos.

Creant Objectes

Per crear un objecte (instància) d'una classe, crida la classe utilitzant el seu nom i passa els arguments necessaris.

# Creant una instància de la classe Dog
my_dog = Dog("Buddy", 3)

# Accedint a atributs i mètodes
print(my_dog.name)  # Sortida: Buddy
print(my_dog.age)   # Sortida: 3
print(my_dog.description())  # Sortida: Buddy is 3 years old

Atributs de la Instància vs. Atributs de Classe

Els atributs de la instància són específics per a cada objecte, mentre que els atributs de classe són compartits per totes les instàncies de la classe.

class Dog:
    species = "Canis familiaris"  # Atribut de classe

    def __init__(self, name, age):
        self.name = name  # Atribut de la instància
        self.age = age    # Atribut de la instància

# Creant instàncies
dog1 = Dog("Buddy", 3)
dog2 = Dog("Lucy", 5)

# Accedint a l'atribut de classe
print(dog1.species)  # Sortida: Canis familiaris
print(dog2.species)  # Sortida: Canis familiaris

# Accedint als atributs de la instància
print(dog1.name)  # Sortida: Buddy
print(dog2.name)  # Sortida: Lucy

Mètodes en Classes

Els mètodes són funcions definides dins d'una classe i s'utilitzen per realitzar operacions utilitzant els atributs de la classe.

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self, sound):
        return f"{self.name} says {sound}"

# Creant una instància
my_dog = Dog("Buddy", 3)

# Cridant un mètode
print(my_dog.speak("Woof Woof"))  # Sortida: Buddy says Woof Woof

Herència

L'herència permet que una classe hereti atributs i mètodes d'una altra classe. Això promou la reutilització del codi i la jerarquia lògica.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("Subclass must implement this method")

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof Woof"

class Cat(Animal):
    def speak(self):
        return f"{self.name} says Meow"

# Creant instàncies
dog = Dog("Buddy")
cat = Cat("Whiskers")

# Cridant mètodes
print(dog.speak())  # Sortida: Buddy says Woof Woof
print(cat.speak())  # Sortida: Whiskers says Meow

Polimorfisme

El polimorfisme permet que els mètodes es puguin utilitzar de manera intercanviable entre diferents classes, sempre que comparteixin el mateix nom de mètode.

class Bird:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} says Chirp"

# Creant instàncies
animals = [Dog("Buddy"), Cat("Whiskers"), Bird("Tweety")]

# Cridant el mateix mètode en diferents objectes
for animal in animals:
    print(animal.speak())
# Sortida:
# Buddy says Woof Woof
# Whiskers says Meow
# Tweety says Chirp

Encapsulació

L'encapsulació restringeix l'accés directe a alguns components d'un objecte, cosa que pot evitar la modificació accidental de dades.

class Car:
    def __init__(self, make, model, year):
        self.__make = make  # Atribut privat
        self.__model = model  # Atribut privat
        self.__year = year  # Atribut privat

    def get_info(self):
        return f"{self.__year} {self.__make} {self.__model}"

# Creant una instància
my_car = Car("Toyota", "Corolla", 2020)

# Accedint a les dades a través d'un mètode
print(my_car.get_info())  # Sortida: 2020 Toyota Corolla

# Intentar accedir als atributs privats directament resultarà en un error
# print(my_car.__make)  # AttributeError: 'Car' object has no attribute '__make'

Resum

  • Les classes i els objectes són el nucli de la OOP en Python.
  • Les classes defineixen el plànol, mentre que els objectes són instàncies de les classes.
  • Els atributs poden ser específics de la instància o compartits entre totes les instàncies (atributs de classe).
  • Els mètodes defineixen el comportament dels objectes.
  • L'herència permet que les classes heretin atributs i mètodes d'altres classes.
  • El polimorfisme permet l'ús d'una interfície unificada per a diferents formes subjacents (tipus de dades).
  • L'encapsulació protegeix l'estat intern d'un objecte restringint l'accés a certs atributs.

Entendre aquests conceptes és crucial per escriure codi organitzat, reutilitzable i mantenible en Python.
