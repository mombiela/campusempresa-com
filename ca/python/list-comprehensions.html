Introducció a les Comprensions de Llistes

Les comprensions de llistes proporcionen una manera concisa de crear llistes en Python. Sovint són més llegibles i ràpides que la creació de llistes basada en bucles for tradicionals.

Conceptes Clau

  • Sintaxi: [expressió per a element en iterable si condició]
  • Expressió: El valor de sortida per a cada element de la llista.
  • Iterable: Qualsevol iterable de Python (per exemple, llista, range, cadena).
  • Condició (opcional): Un filtre que selecciona quins elements incloure.

Exemples Bàsics

Comencem amb un exemple senzill per entendre la sintaxi bàsica.

Exemple 1: Creant una Llista de Quadrats

squares = [x**2 per a x en range(10)]
print(squares)
  • Explicació: Aquesta comprensió de llista itera sobre cada número x en el rang de 0 a 9, calcula x**2, i recull aquests valors en una nova llista.

Exemple 2: Filtrant amb Condicions

even_squares = [x**2 per a x en range(10) si x % 2 == 0]
print(even_squares)
  • Explicació: Aquest exemple afegeix una condició per filtrar només els números parells abans de calcular els seus quadrats.

Comprensions de Llistes Avançades

A mesura que et sentis més còmode amb les comprensions de llistes, pots començar a utilitzar expressions més complexes i comprensions imbricades.

Comprensions de Llistes Imbricades

Les comprensions de llistes imbricades es poden utilitzar per aplanar una llista de llistes o per crear una matriu.

Exemple 3: Aplanant una Llista de Llistes

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flattened = [num per a row en matrix per a num en row]
print(flattened)
  • Explicació: Aquesta comprensió itera sobre cada fila de la matriu, i després sobre cada número en la fila, recollint tots els números en una sola llista.

Utilitzant Funcions en Comprensions de Llistes

També pots cridar funcions dins d'una comprensió de llista.

Exemple 4: Aplicant una Funció

def square(x):
    return x**2

squares = [square(x) per a x en range(10)]
print(squares)
  • Explicació: Aquest exemple defineix una funció square i després la utilitza dins de la comprensió de llista per calcular els quadrats dels números de 0 a 9.

Combinant Múltiples Condicions

Pots combinar múltiples condicions utilitzant operadors lògics.

Exemple 5: Múltiples Condicions

filtered = [x per a x en range(20) si x % 2 == 0 i x % 3 == 0]
print(filtered)
  • Explicació: Aquesta comprensió filtra números que són divisibles tant per 2 com per 3.

Consideracions de Rendiment

Les comprensions de llistes són generalment més ràpides que els bucles for tradicionals perquè estan optimitzades per al rendiment. No obstant això, la llegibilitat no s'ha de sacrificar per guanys de rendiment menors.

Comparant el Rendiment

Aquí hi ha una comparació simple entre un bucle for i una comprensió de llista.

Exemple 6: Bucle For vs Comprensió de Llista

import time

# Utilitzant un bucle for
start = time.time()
squares_loop = []
for x in range(1000000):
    squares_loop.append(x**2)
end = time.time()
print("Temps del bucle for:", end - start)

# Utilitzant una comprensió de llista
start = time.time()
squares_comp = [x**2 per a x en range(1000000)]
end = time.time()
print("Temps de la comprensió de llista:", end - start)
  • Explicació: Aquest codi mesura el temps que es triga a crear una llista de quadrats utilitzant tant un bucle for com una comprensió de llista.

Conclusió

Les comprensions de llistes són una característica poderosa en Python que pot fer que el teu codi sigui més concís i llegible. Són adequades per a una àmplia gamma de tasques, des de la creació simple de llistes fins a operacions més complexes que impliquen bucles imbricats i condicions. A mesura que practiquis, trobaràs que són una eina inestimable en la teva caixa d'eines de programació en Python.
