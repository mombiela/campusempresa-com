Introducció a l'Herència

L'herència és un concepte fonamental en la programació orientada a objectes (OOP) que permet que una classe hereti atributs i mètodes d'una altra classe. Això promou la reutilització del codi i pot conduir a una base de codi més organitzada i modular.

  • Classe Base (Classe Pare): La classe les propietats i mètodes de la qual són heretats.
  • Classe Derivada (Classe Filla): La classe que hereta propietats i mètodes de la classe base.

Herència Bàsica

Definint una Classe Base

Comencem definint una classe base simple anomenada Animal.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} fa un so."

Creant una Classe Derivada

Ara, creem una classe derivada anomenada Dog que hereta d'Animal.

class Dog(Animal):
    def speak(self):
        return f"{self.name} borda."

Utilitzant la Classe Derivada

Ara podem crear una instància de Dog i utilitzar els seus mètodes.

dog = Dog("Buddy")
print(dog.speak())  # Sortida: Buddy borda.

Sobreescriptura de Mètodes

En l'exemple anterior, el mètode speak a la classe Dog sobreescriu el mètode speak a la classe Animal. Això permet que la classe Dog proporcioni una implementació específica del mètode.

Exemple de Sobreescriptura de Mètodes

class Cat(Animal):
    def speak(self):
        return f"{self.name} miola."

cat = Cat("Whiskers")
print(cat.speak())  # Sortida: Whiskers miola.

Utilitzant `super()`

La funció super() permet cridar mètodes de la classe base a la classe derivada. Això és útil quan es vol estendre la funcionalitat del mètode de la classe base.

Exemple amb `super()`

class Bird(Animal):
    def speak(self):
        base_speak = super().speak()
        return f"{base_speak} Piu piu!"

bird = Bird("Tweety")
print(bird.speak())  # Sortida: Tweety fa un so. Piu piu!

Herència Múltiple

Python suporta l'herència múltiple, on una classe pot heretar de més d'una classe base. Això pot ser útil però s'ha d'utilitzar amb precaució per evitar la complexitat.

Exemple d'Herència Múltiple

class CanFly:
    def fly(self):
        return "Puc volar!"

class FlyingBird(Animal, CanFly):
    def speak(self):
        return f"{self.name} piula."

flying_bird = FlyingBird("Sky")
print(flying_bird.speak())  # Sortida: Sky piula.
print(flying_bird.fly())    # Sortida: Puc volar!

Herència i la Funció `isinstance()`

La funció isinstance() es pot utilitzar per comprovar si un objecte és una instància d'una classe particular o d'una tupla de classes.

Exemple amb `isinstance()`

print(isinstance(dog, Dog))       # Sortida: True
print(isinstance(dog, Animal))    # Sortida: True
print(isinstance(dog, Cat))       # Sortida: False

Resum

  • Herència permet que una classe hereti atributs i mètodes d'una altra classe.
  • Classe Base és la classe de la qual s'hereta, i Classe Derivada és la classe que hereta.
  • Sobreescriptura de Mètodes permet que una classe derivada proporcioni una implementació específica d'un mètode ja definit a la seva classe base.
  • La funció super() s'utilitza per cridar mètodes de la classe base a la classe derivada.
  • Herència Múltiple permet que una classe hereti de més d'una classe base però s'ha d'utilitzar amb cura.
  • La funció isinstance() comprova si un objecte és una instància d'una classe específica o d'una tupla de classes.

Entendre l'herència és crucial per dominar l'OOP en Python, ja que ajuda a crear codi més modular, reutilitzable i mantenible.
