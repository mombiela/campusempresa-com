Introducció a Flask

Flask és un framework lleuger d'aplicacions web WSGI en Python. Està dissenyat amb simplicitat i flexibilitat en ment, fent-lo una elecció popular tant per a principiants com per a desenvolupadors avançats.

  • WSGI: Web Server Gateway Interface, una especificació per a una interfície simple i universal entre servidors web i aplicacions web.
  • Lleuger: Flask no requereix eines o biblioteques particulars.
  • Flexibilitat: Flask permet als desenvolupadors triar les seves eines i biblioteques.

Configuració de Flask

Abans de poder començar a utilitzar Flask, cal instal·lar-lo.

pip install Flask

Creant la teva primera aplicació Flask

Aquí tens un exemple senzill d'una aplicació Flask:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)
  • Explicació:
    • from flask import Flask: Importa la classe Flask.
    • app = Flask(__name__): Crea una instància de la classe Flask.
    • @app.route('/'): Defineix la ruta per a la pàgina principal.
    • def hello_world(): Defineix la funció de vista per a la pàgina principal.
    • app.run(debug=True): Executa l'aplicació en mode debug.

Routing a Flask

El routing s'utilitza per vincular una URL a una funció.

  • Ruta bàsica:

    @app.route('/about')
def about():
    return 'About Page'

  • Ruta dinàmica:

    @app.route('/user/<username>')
def show_user_profile(username):
    return f'User {username}'

  • Explicació:

    • <username>: Un component dinàmic que pot capturar part de la URL i passar-lo a la funció com a argument.

Plantilles a Flask

Flask utilitza el motor de plantilles Jinja2 per renderitzar HTML.

Creant una plantilla

  1. Crea una carpeta anomenada templates.
  2. Dins de la carpeta templates, crea un fitxer HTML, per exemple, index.html.
<!doctype html>
<html>
  <head>
    <title>{{ title }}</title>
  </head>
  <body>
    <h1>{{ heading }}</h1>
    <p>{{ content }}</p>
  </body>
</html>

Renderitzant una plantilla

from flask import render_template

@app.route('/')
def home():
    return render_template('index.html', title='Home Page', heading='Welcome', content='This is the home page.')
  • Explicació:
    • render_template('index.html', ...): Renderitza la plantilla index.html i passa variables a aquesta.

Formularis i entrada d'usuari

Gestionar formularis i l'entrada d'usuari és una tasca comuna en aplicacions web.

Creant un formulari

<form method="POST" action="/submit">
  <input type="text" name="username">
  <input type="submit" value="Submit">
</form>

Gestionant l'enviament del formulari

from flask import request

@app.route('/submit', methods=['POST'])
def submit():
    username = request.form['username']
    return f'Hello, {username}!'
  • Explicació:
    • request.form['username']: Accedeix a les dades del formulari enviades via sol·licitud POST.

Integració amb bases de dades

Flask es pot integrar amb diverses bases de dades utilitzant extensions com Flask-SQLAlchemy.

Configurant Flask-SQLAlchemy

  1. Instal·la Flask-SQLAlchemy:

    pip install Flask-SQLAlchemy

  2. Configura i inicialitza la base de dades:

    from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///site.db'
db = SQLAlchemy(app)

Creant un model

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'
  • Explicació:
    • db.Model: Classe base per a tots els models.
    • db.Column: Defineix una columna a la base de dades.
    • db.Integer, db.String: Tipus de dades per a les columnes.
    • primary_key=True: Estableix la columna com a clau primària.
    • unique=True: Assegura que els valors de la columna siguin únics.
    • nullable=False: Assegura que la columna no pugui ser nul·la.

Temes avançats

Blueprints

Els Blueprints ajuden a organitzar la teva aplicació en mòduls.

from flask import Blueprint

admin = Blueprint('admin', __name__)

@admin.route('/admin')
def admin_home():
    return 'Admin Home'

Gestió d'errors

Es poden crear pàgines d'error personalitzades per gestionar errors.

@app.errorhandler(404)
def page_not_found(e):
    return render_template('404.html'), 404

Proves

Flask proporciona eines per provar la teva aplicació.

import unittest

class FlaskTestCase(unittest.TestCase):
    def setUp(self):
        app.config['TESTING'] = True
        self.app = app.test_client()

    def test_home(self):
        response = self.app.get('/')
        self.assertEqual(response.status_code, 200)

if __name__ == '__main__':
    unittest.main()

Conclusió

Flask és un framework potent i flexible per construir aplicacions web en Python. És adequat tant per a principiants com per a desenvolupadors avançats gràcies a la seva simplicitat i extensibilitat. En dominar Flask, pots crear aplicacions web robustes i integrar-les amb diverses eines i biblioteques.
