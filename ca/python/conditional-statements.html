Introducció a les declaracions condicionals

Les declaracions condicionals et permeten executar certs fragments de codi basant-se en si una condició és certa o falsa. Aquest és un concepte fonamental en la programació que permet prendre decisions en el teu codi.

La declaració `if`

La declaració if és la forma més bàsica de declaració condicional en Python.

Sintaxi

if condition:
    # codi a executar si la condició és certa

Exemple

age = 18
if age >= 18:
    print("Ets un adult.")

En aquest exemple, es mostrarà el missatge "Ets un adult." perquè la condició age >= 18 és certa.

La declaració `else`

La declaració else es pot utilitzar per executar un bloc de codi si la condició en la declaració if és falsa.

Sintaxi

if condition:
    # codi a executar si la condició és certa
else:
    # codi a executar si la condició és falsa

Exemple

age = 16
if age >= 18:
    print("Ets un adult.")
else:
    print("No ets un adult.")

En aquest exemple, es mostrarà el missatge "No ets un adult." perquè la condició age >= 18 és falsa.

La declaració `elif`

La declaració elif (abreviatura de else if) et permet comprovar múltiples condicions.

Sintaxi

if condition1:
    # codi a executar si la condició1 és certa
elif condition2:
    # codi a executar si la condició2 és certa
else:
    # codi a executar si tant la condició1 com la condició2 són falses

Exemple

age = 20
if age < 13:
    print("Ets un nen.")
elif age < 18:
    print("Ets un adolescent.")
else:
    print("Ets un adult.")

En aquest exemple, es mostrarà el missatge "Ets un adult." perquè la condició age < 18 és falsa, però la condició age >= 18 és certa.

Declaracions condicionals niades

Pots niar declaracions condicionals dins d'altres per comprovar múltiples condicions.

Exemple

age = 20
if age >= 18:
    if age < 21:
        print("Ets un adult però no prou gran per beure alcohol als EUA.")
    else:
        print("Ets un adult i prou gran per beure alcohol als EUA.")
else:
    print("No ets un adult.")

En aquest exemple, es mostrarà el missatge "Ets un adult però no prou gran per beure alcohol als EUA." perquè la condició age >= 18 és certa i age < 21 també és certa.

Operadors lògics

Els operadors lògics es poden utilitzar per combinar múltiples condicions en una sola declaració if.

Operadors

  • and: Cert si ambdues condicions són certes.
  • or: Cert si almenys una condició és certa.
  • not: Cert si la condició és falsa.

Exemple

age = 20
has_id = True

if age >= 18 and has_id:
    print("Tens permís per entrar al club.")
else:
    print("No tens permís per entrar al club.")

En aquest exemple, es mostrarà el missatge "Tens permís per entrar al club." perquè ambdues condicions age >= 18 i has_id són certes.

Expressions condicionals (Operador ternari)

Python admet una sintaxi més curta per a les declaracions condicionals coneguda com l'operador ternari.

Sintaxi

value_if_true if condition else value_if_false

Exemple

age = 20
status = "adult" if age >= 18 else "minor"
print(status)

En aquest exemple, la variable status es configurarà com "adult" perquè la condició age >= 18 és certa.

Resum

  • Les declaracions condicionals (if, else, elif) et permeten executar codi basat en condicions.
  • Les declaracions condicionals niades permeten una presa de decisions més complexa.
  • Els operadors lògics (and, or, not) poden combinar múltiples condicions.
  • L'operador ternari proporciona una manera concisa d'escriure declaracions condicionals simples.

Entendre i utilitzar les declaracions condicionals de manera efectiva et permetrà escriure programes en Python més dinàmics i responsius. Practica utilitzant aquests conceptes en diversos escenaris per convertir-te en un expert.
