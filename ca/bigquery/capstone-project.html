Introducció

El projecte capstone és un exercici integral dissenyat per consolidar el teu aprenentatge i demostrar la teva competència en l'ús de BigQuery. Aquest projecte implicarà dades del món real i requerirà que apliquis diversos conceptes i tècniques coberts al llarg del curs, des de consultes bàsiques fins a anàlisis de dades avançades i optimització.

Visió General del Projecte

En aquest projecte capstone, hauràs de:

  • Seleccionar un Conjunt de Dades: Tria un conjunt de dades públicament disponible o utilitza un conjunt de dades proporcionat pel curs.
  • Definir Objectius: Especifica clarament els objectius de la teva anàlisi.
  • Realitzar Exploració de Dades: Utilitza consultes SQL per explorar i entendre el conjunt de dades.
  • Neteja i Transformació de Dades: Neteja i transforma les dades segons sigui necessari.
  • Anàlisi Avançada: Aplica funcions avançades de BigQuery com funcions de finestra, funcions definides per l'usuari i models d'aprenentatge automàtic.
  • Visualització i Informes: Crea visualitzacions i genera informes per presentar les teves troballes.

Guia Pas a Pas

1. Seleccionar un Conjunt de Dades

Tria un conjunt de dades que t'interessi. Algunes fonts populars de conjunts de dades inclouen:

  • Google Cloud Public Datasets
  • Kaggle Datasets
  • UCI Machine Learning Repository

2. Definir Objectius

Defineix clarament què vols aconseguir amb la teva anàlisi. Per exemple:

  • Identificar tendències en les dades.
  • Predir resultats futurs basats en dades històriques.
  • Trobar correlacions entre diferents variables.

3. Realitzar Exploració de Dades

Utilitza consultes SQL per explorar el conjunt de dades. Aquí tens algunes consultes d'exemple:

-- Comptar el nombre de files en el conjunt de dades
SELECT COUNT(*) AS total_rows
FROM `project.dataset.table`;

-- Obtenir un resum d'una columna específica
SELECT column_name, COUNT(*) AS count
FROM `project.dataset.table`
GROUP BY column_name
ORDER BY count DESC;

4. Neteja i Transformació de Dades

Neteja i transforma les dades per fer-les adequades per a l'anàlisi. Això pot implicar gestionar valors perduts, eliminar duplicats o crear nous camps calculats.

-- Eliminar duplicats
CREATE OR REPLACE TABLE `project.dataset.cleaned_table` AS
SELECT DISTINCT *
FROM `project.dataset.table`;

-- Gestionar valors perduts reemplaçant-los amb la mitjana
CREATE OR REPLACE TABLE `project.dataset.cleaned_table` AS
SELECT *,
  IFNULL(numeric_column, (SELECT AVG(numeric_column) FROM `project.dataset.table`)) AS numeric_column
FROM `project.dataset.table`;

5. Anàlisi Avançada

Aplica funcions avançades de BigQuery per obtenir una comprensió més profunda.

Funcions de Finestra

-- Calcular el total acumulat d'una columna numèrica
SELECT column_name,
  SUM(numeric_column) OVER (ORDER BY date_column) AS running_total
FROM `project.dataset.table`;

Funcions Definides per l'Usuari

-- Crear una funció definida per l'usuari per categoritzar dades
CREATE FUNCTION `project.dataset.categorize`(value FLOAT64) RETURNS STRING AS ((
  CASE
    WHEN value < 10 THEN 'Low'
    WHEN value BETWEEN 10 AND 20 THEN 'Medium'
    ELSE 'High'
  END
));

-- Utilitzar la funció en una consulta
SELECT column_name,
  `project.dataset.categorize`(numeric_column) AS category
FROM `project.dataset.table`;

Models d'Aprenentatge Automàtic

-- Crear un model de regressió lineal
CREATE OR REPLACE MODEL `project.dataset.model`
OPTIONS(model_type='linear_reg') AS
SELECT
  feature1,
  feature2,
  target
FROM `project.dataset.table`;

-- Avaluar el model
SELECT
  *
FROM
  ML.EVALUATE(MODEL `project.dataset.model`, (
    SELECT
      feature1,
      feature2,
      target
    FROM `project.dataset.table`
  ));

6. Visualització i Informes

Crea visualitzacions i genera informes per presentar les teves troballes. Pots utilitzar eines com Google Data Studio, Tableau, o fins i tot integrar amb biblioteques de Python com Matplotlib o Seaborn.

-- Consulta d'exemple per a visualització
SELECT
  date_column,
  SUM(numeric_column) AS total_value
FROM `project.dataset.table`
GROUP BY date_column
ORDER BY date_column;

Conclusió

El projecte capstone és una oportunitat per aplicar tot el que has après en un escenari pràctic i real. En completar aquest projecte, aconseguiràs:

  • Experiència pràctica amb BigQuery.
  • Desenvolupar habilitats de resolució de problemes enfrontant-te a desafiaments reals de dades.
  • Crear una peça de portafoli que demostri la teva expertesa en BigQuery.

Recorda documentar el teu procés, explicar les teves decisions i reflexionar sobre les teves troballes. Bona sort!
