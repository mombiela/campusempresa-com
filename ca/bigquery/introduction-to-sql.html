Visió general

En aquesta secció, introduirem SQL (Structured Query Language) i la seva aplicació a Google BigQuery. Això servirà com a base per entendre com interactuar i manipular dades dins de BigQuery.

Què és SQL?

SQL és un llenguatge estàndard per accedir i manipular bases de dades. S'utilitza per realitzar tasques com consultar dades, actualitzar registres i gestionar estructures de bases de dades.

Conceptes clau

  • Base de dades: Una col·lecció estructurada de dades.
  • Taula: Una col·lecció d'entrades de dades relacionades que consisteixen en files i columnes.
  • Consulta: Una sol·licitud de dades o informació d'una base de dades.

Sintaxi bàsica de SQL

Comencem amb alguns comandaments bàsics de SQL que són essencials per treballar amb BigQuery.

Instrucció SELECT

La instrucció SELECT s'utilitza per consultar dades d'una base de dades.

SELECT column1, column2, ...
FROM table_name;

Exemple:

SELECT name, age
FROM employees;

Clàusula WHERE

La clàusula WHERE s'utilitza per filtrar registres.

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Exemple:

SELECT name, age
FROM employees
WHERE age > 30;

Clàusula ORDER BY

La clàusula ORDER BY s'utilitza per ordenar el conjunt de resultats.

SELECT column1, column2, ...
FROM table_name
ORDER BY column1 [ASC|DESC];

Exemple:

SELECT name, age
FROM employees
ORDER BY age DESC;

Clàusula LIMIT

La clàusula LIMIT s'utilitza per especificar el nombre de registres a retornar.

SELECT column1, column2, ...
FROM table_name
LIMIT number;

Exemple:

SELECT name, age
FROM employees
LIMIT 5;

SQL a BigQuery

BigQuery utilitza SQL estàndard (també conegut com BigQuery SQL) per consultar dades. Aquí hi ha algunes característiques específiques i diferències quan s'utilitza SQL a BigQuery.

Característiques clau

  • Escalabilitat: BigQuery pot gestionar grans conjunts de dades de manera eficient.
  • Rendiment: Optimitzat per a un rendiment ràpid de les consultes.
  • Integració: Es pot integrar fàcilment amb altres serveis de Google Cloud.

Exemple de consulta a BigQuery

SELECT name, age
FROM `project.dataset.employees`
WHERE age > 30
ORDER BY age DESC
LIMIT 5;

Explicació:

  • Projecte: L'ID del projecte de Google Cloud.
  • Dataset: El conjunt de dades dins del projecte.
  • Taula: La taula dins del conjunt de dades.

Funcions SQL comunes a BigQuery

BigQuery proporciona una varietat de funcions integrades per realitzar operacions sobre dades.

Funcions agregades

  • COUNT(): Retorna el nombre de files.
  • SUM(): Retorna la suma d'una columna numèrica.
  • AVG(): Retorna el valor mitjà d'una columna numèrica.
  • MAX(): Retorna el valor màxim.
  • MIN(): Retorna el valor mínim.

Exemple:

SELECT COUNT(*), AVG(age), MAX(age), MIN(age)
FROM `project.dataset.employees`;

Funcions de cadena

  • UPPER(): Converteix una cadena a majúscules.
  • LOWER(): Converteix una cadena a minúscules.
  • CONCAT(): Concatena dues o més cadenes.

Exemple:

SELECT UPPER(name), LOWER(name), CONCAT(first_name, ' ', last_name) AS full_name
FROM `project.dataset.employees`;

Funcions de data

  • CURRENT_DATE(): Retorna la data actual.
  • DATE_ADD(): Afegeix un interval de temps especificat a una data.
  • FORMAT_DATE(): Dona format a una data segons un format especificat.

Exemple:

SELECT CURRENT_DATE(), DATE_ADD(CURRENT_DATE(), INTERVAL 7 DAY) AS next_week
FROM `project.dataset.employees`;

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de SQL i la seva aplicació a BigQuery. Hem discutit comandaments fonamentals de SQL com SELECT, WHERE, ORDER BY i LIMIT, i hem explorat com utilitzar aquests comandaments a BigQuery. A més, hem introduït algunes funcions SQL comunes disponibles a BigQuery. Aquest coneixement fonamental serà crucial a mesura que avancem cap a temes més avançats a BigQuery.
