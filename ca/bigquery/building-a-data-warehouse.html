gcloud projects create my-bigquery-project

# Establir el projecte
gcloud config set project my-bigquery-project

# Activar l'API de BigQuery

Consultar dades a BigQuery

  • Escriure consultes SQL a BigQuery
  • Utilitzar SQL estàndard vs. SQL llegat
  • Millors pràctiques per a l'optimització de consultes

Exemple: Consulta SQL bàsica

-- Seleccionar tots els registres de vendes
SELECT * FROM my_dataset.sales;

-- Agregar vendes per producte
SELECT 
  p.product_name,
  SUM(s.sale_amount) AS total_sales
FROM 
  my_dataset.sales s
JOIN 
  my_dataset.products p
ON 
  s.product_id = p.product_id
GROUP BY 
  p.product_name;

Temes avançats

  • Utilitzar BigQuery ML per a l'aprenentatge automàtic
  • Integrar BigQuery amb altres serveis de Google Cloud
  • Implementar millors pràctiques de seguretat

Exemple: Entrenament d'un model d'aprenentatge automàtic a BigQuery ML

-- Crear un model per predir l'import de les vendes
CREATE OR REPLACE MODEL my_dataset.sales_model
OPTIONS(model_type='linear_reg') AS
SELECT
  sale_amount,
  product_id,
  customer_id,
  EXTRACT(DAYOFWEEK FROM sale_date) AS day_of_week
FROM
  my_dataset.sales;
