Introducció

Les funcions agregades són eines essencials en SQL i BigQuery per resumir i analitzar dades. Permeten realitzar càlculs sobre múltiples files de dades i retornar un únic valor. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de funcions agregades en BigQuery.

Funcions Agregades Clau

  • COUNT(): Compta el nombre de files.
  • SUM(): Calcula la suma total d'una columna numèrica.
  • AVG(): Calcula el valor mitjà d'una columna numèrica.
  • MIN(): Troba el valor mínim en una columna.
  • MAX(): Troba el valor màxim en una columna.

Ús Bàsic

COUNT()

La funció COUNT() retorna el nombre de files que coincideixen amb una condició especificada.

SELECT COUNT(*) AS total_rows
FROM `project.dataset.table`;

SUM()

La funció SUM() suma tots els valors d'una columna numèrica.

SELECT SUM(sales) AS total_sales
FROM `project.dataset.table`;

AVG()

La funció AVG() calcula el valor mitjà d'una columna numèrica.

SELECT AVG(sales) AS average_sales
FROM `project.dataset.table`;

MIN() i MAX()

Les funcions MIN() i MAX() retornen els valors més petits i més grans d'una columna, respectivament.

SELECT MIN(sales) AS minimum_sales, MAX(sales) AS maximum_sales
FROM `project.dataset.table`;

Agrupant Dades amb Funcions Agregades

Clàusula GROUP BY

Per aplicar funcions agregades a subconjunts de dades, utilitza la clàusula GROUP BY.

SELECT region, SUM(sales) AS total_sales
FROM `project.dataset.table`
GROUP BY region;

Clàusula HAVING

La clàusula HAVING filtra grups basant-se en els resultats de funcions agregades.

SELECT region, SUM(sales) AS total_sales
FROM `project.dataset.table`
GROUP BY region
HAVING total_sales > 10000;

Ús Avançat

Combinant Múltiples Funcions Agregades

Pots combinar múltiples funcions agregades en una sola consulta.

SELECT region, 
       COUNT(*) AS total_orders, 
       SUM(sales) AS total_sales, 
       AVG(sales) AS average_sales
FROM `project.dataset.table`
GROUP BY region;

Funcions de Finestra

Les funcions de finestra permeten realitzar càlculs agregats a través d'un conjunt de files de taula que estan d'alguna manera relacionades amb la fila actual.

SELECT region, 
       sales, 
       SUM(sales) OVER (PARTITION BY region) AS total_sales_per_region
FROM `project.dataset.table`;

Tractament de Valors NULL

Les funcions agregades generalment ignoren els valors NULL. No obstant això, pots utilitzar COALESCE() per tractar els valors NULL.

SELECT region, 
       SUM(COALESCE(sales, 0)) AS total_sales
FROM `project.dataset.table`
GROUP BY region;

Resum

Les funcions agregades en BigQuery són eines poderoses per resumir i analitzar dades. Dominant funcions com COUNT(), SUM(), AVG(), MIN() i MAX(), i entenent com utilitzar-les amb les clàusules GROUP BY i HAVING, pots realitzar anàlisis de dades complexes de manera eficient. Tècniques avançades com les funcions de finestra i el tractament de valors NULL milloren encara més la teva capacitat per obtenir informació de les teves dades.
