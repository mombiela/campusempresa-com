Introducció

En aquesta secció, explorarem com utilitzar Google BigQuery amb Google Data Studio. Aquesta integració et permet visualitzar i analitzar les teves dades de BigQuery utilitzant les potents eines de creació d'informes de Data Studio.

Conceptes Clau

  • BigQuery: Un magatzem de dades totalment gestionat i sense servidor que permet l'anàlisi escalable de petabytes de dades.
  • Data Studio: Una eina gratuïta que converteix les teves dades en panells i informes informatius, fàcils de llegir, fàcils de compartir i totalment personalitzables.

Configuració de BigQuery i Data Studio

Abans de poder utilitzar les dades de BigQuery a Data Studio, has de configurar ambdós serveis i assegurar-te que poden comunicar-se entre ells.

Prerequisits

  • Un compte de Google Cloud Platform (GCP).
  • Un projecte amb facturació habilitada a GCP.
  • API de BigQuery habilitada per al teu projecte.
  • Accés a Data Studio.

Passos per Connectar BigQuery a Data Studio

  1. Obre Data Studio: Navega a Google Data Studio.
  2. Crea un Nou Informe: Fes clic al botó "Crear" i selecciona "Informe".
  3. Afegeix Font de Dades: Fes clic a "Afegeix dades" i selecciona "BigQuery" de la llista de connectors.
  4. Autoritza l'Accés: Si se't demana, autoritza Data Studio per accedir a les teves dades de BigQuery.
  5. Selecciona Projecte i Conjunt de Dades: Tria el teu projecte de GCP, conjunt de dades i taula que vols visualitzar.
  6. Afegeix a l'Informe: Fes clic a "Afegeix" per incloure les dades seleccionades al teu informe.

Visualització de Dades de BigQuery a Data Studio

Un cop la teva font de dades està connectada, pots començar a crear visualitzacions.

Creació de Gràfics i Taules

  • Gràfics de Barres: Útils per comparar diferents categories.
  • Gràfics de Línies: Ideals per mostrar tendències al llarg del temps.
  • Gràfics de Pastís: Bons per mostrar proporcions d'un total.
  • Taules: Útils per a la representació detallada de dades.

Exemple: Creació d'un Gràfic de Barres

  1. Afegeix un Gràfic: Fes clic al botó "Afegeix un gràfic" i selecciona "Gràfic de barres".
  2. Configura les Dades: Arrossega i deixa anar camps de la teva font de dades de BigQuery a les seccions "Dimensió" i "Mètrica".
  3. Personalitza l'Aparença: Utilitza la pestanya d'estil per canviar colors, etiquetes i altres aspectes visuals.
-- Exemple de consulta SQL de BigQuery per obtenir dades per a la visualització
SELECT
  category,
  COUNT(*) as count
FROM
  `your_project.your_dataset.your_table`
GROUP BY
  category
ORDER BY
  count DESC;

Filtratge de Dades

Data Studio et permet afegir filtres als teus informes per centrar-te en subconjunts específics de les teves dades.

  • Filtre de Rang de Dates: Útil per a dades basades en el temps.
  • Filtre de Categoria: Permet als usuaris seleccionar categories específiques per veure.

Tècniques Avançades

Per a usuaris més avançats, Data Studio ofereix funcions addicionals per millorar els teus informes.

Consultes SQL Personalitzades

Pots utilitzar consultes SQL personalitzades per obtenir dades específiques de BigQuery.

  1. Edita la Font de Dades: A Data Studio, ves a la teva font de dades i fes clic a "Edita".
  2. Afegeix Consulta Personalitzada: Fes clic a "Crea nova font de dades" i selecciona "Consulta Personalitzada".
  3. Introdueix la Consulta SQL: Escriu la teva consulta SQL per obtenir les dades desitjades.
-- Exemple d'una consulta SQL personalitzada
SELECT
  user_id,
  SUM(purchase_amount) as total_spent
FROM
  `your_project.your_dataset.purchases`
WHERE
  purchase_date BETWEEN '2023-01-01' AND '2023-12-31'
GROUP BY
  user_id
ORDER BY
  total_spent DESC;

Combinació de Dades

La combinació de dades et permet combinar dades de múltiples fonts.

  1. Afegeix Fonts de Dades: Assegura't que totes les fonts de dades estan afegides al teu informe.
  2. Combina Dades: Fes clic a "Recurs" > "Gestiona dades combinades" > "Afegeix una combinació".
  3. Configura la Unió: Selecciona els camps per unir i configura la combinació.

Conclusió

Utilitzar BigQuery amb Data Studio proporciona una manera potent de visualitzar i analitzar grans conjunts de dades. Seguint els passos descrits en aquesta secció, pots connectar BigQuery a Data Studio, crear visualitzacions informatives i aprofitar funcions avançades per obtenir una comprensió més profunda de les teves dades. Tant si ets un principiant com un usuari avançat, aquesta integració pot millorar significativament les teves capacitats d'anàlisi de dades.
