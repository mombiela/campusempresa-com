Introducció a BigQuery

BigQuery és un magatzem de dades totalment gestionat i sense servidor proporcionat per Google Cloud Platform (GCP). Permet realitzar consultes SQL molt ràpides utilitzant la potència de processament de la infraestructura de Google. BigQuery està dissenyat per gestionar anàlisis de dades a gran escala i pot processar terabytes de dades en segons.

Característiques clau de BigQuery

  • Arquitectura sense servidor: No cal gestionar la infraestructura; Google s'encarrega de l'escalabilitat i el rendiment.
  • Anàlisi en temps real: Capacitat per consultar dades en streaming en temps real.
  • Escalabilitat: S'escala automàticament per gestionar grans conjunts de dades.
  • Eficiència de costos: Pagues només per les dades que consultes.
  • Seguretat: Integrat amb les característiques de seguretat de Google Cloud, incloent xifratge i IAM (Gestió d'Identitat i Accés).

Com funciona BigQuery

BigQuery utilitza un format d'emmagatzematge columnar i una arquitectura d'arbre per executar consultes, la qual cosa li permet escanejar grans quantitats de dades ràpidament.

Components de BigQuery

  • Conjunts de dades: Contenidors que contenen taules.
  • Taules: Dades estructurades que es poden consultar.
  • Treballs: Tasques que BigQuery executa, com ara executar consultes o carregar dades.
  • Projectes: Contenidors de nivell superior a GCP que organitzen recursos.

Exemple: Consulta bàsica a BigQuery

Aquí teniu un exemple senzill de consulta a un conjunt de dades públic a BigQuery:

SELECT name, birth_year
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
ORDER BY birth_year DESC
LIMIT 10;

Explicació:

  • SELECT name, birth_year: Selecciona les columnes name i birth_year.
  • FROM bigquery-public-data.usa_names.usa_1910_2013: Especifica el conjunt de dades i la taula.
  • WHERE state = 'TX': Filtra els resultats per incloure només els registres on l'estat és Texas.
  • ORDER BY birth_year DESC: Ordena els resultats per birth_year en ordre descendent.
  • LIMIT 10: Limita els resultats a 10 files.

Característiques avançades de BigQuery

A mesura que et familiaritzis més amb BigQuery, podràs aprofitar les seves característiques avançades per realitzar anàlisis de dades més complexes.

Taules particionades

Les taules particionades permeten dividir la teva taula en segments, fent les consultes més eficients.

CREATE TABLE mydataset.newtable
PARTITION BY DATE(timestamp_column)
AS
SELECT * FROM mydataset.oldtable;

Funcions definides per l'usuari (UDFs)

Pots crear funcions personalitzades a BigQuery utilitzant JavaScript.

CREATE FUNCTION mydataset.custom_function(x INT64) 
RETURNS INT64 
LANGUAGE js AS """
  return x * x;
""";

Aprenentatge automàtic amb BigQuery ML

BigQuery ML et permet crear i executar models d'aprenentatge automàtic directament dins de BigQuery utilitzant SQL.

CREATE MODEL mydataset.my_model
OPTIONS(model_type='linear_reg') AS
SELECT
  input_feature,
  target
FROM
  mydataset.training_data;

Conclusió

BigQuery és una eina poderosa per a l'anàlisi de dades, oferint una gamma de característiques des de consultes SQL bàsiques fins a capacitats avançades d'aprenentatge automàtic. La seva arquitectura sense servidor i escalabilitat la fan una excel·lent opció per gestionar grans conjunts de dades de manera eficient. Comprenent els conceptes bàsics i explorant les seves característiques avançades, podràs aprofitar BigQuery per obtenir informació valuosa de les teves dades.
