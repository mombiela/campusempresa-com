Introducció

Les subconsultes i les consultes imbricades són eines poderoses en SQL que et permeten realitzar operacions complexes de recuperació de dades. A BigQuery, aquests conceptes són essencials per escriure consultes eficients i llegibles. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat de subconsultes i consultes imbricades a BigQuery.

Què són les subconsultes?

Una subconsulta és una consulta imbricada dins d'una altra consulta. Es pot utilitzar en diverses parts d'una instrucció SQL, com ara les claus SELECT, FROM, WHERE i HAVING.

Conceptes clau

  • Consulta interna: La consulta dins de la consulta principal.
  • Consulta externa: La consulta principal que conté la subconsulta.
  • Subconsulta correlacionada: Una subconsulta que fa referència a columnes de la consulta externa.

Exemple bàsic

SELECT name
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
AND year = (SELECT MAX(year) FROM `bigquery-public-data.usa_names.usa_1910_2013` WHERE state = 'TX');
  • Explicació: Aquesta consulta recupera els noms de l'estat de Texas per a l'any més recent disponible al conjunt de dades.

Tipus de subconsultes

Subconsultes escalar

Una subconsulta escalar retorna un valor únic. Es pot utilitzar en llocs on s'espera un valor únic.

SELECT name, year
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE year = (SELECT MAX(year) FROM `bigquery-public-data.usa_names.usa_1910_2013`);
  • Explicació: Aquesta consulta recupera noms i anys on l'any és l'any màxim al conjunt de dades.

Subconsultes correlacionades

Una subconsulta correlacionada fa referència a columnes de la consulta externa. S'avalua una vegada per cada fila processada per la consulta externa.

SELECT name, year, state
FROM `bigquery-public-data.usa_names.usa_1910_2013` AS outer
WHERE year = (SELECT MAX(year) FROM `bigquery-public-data.usa_names.usa_1910_2013` AS inner WHERE inner.state = outer.state);
  • Explicació: Aquesta consulta recupera noms, anys i estats on l'any és l'any màxim per a cada estat.

Consultes imbricades

Les consultes imbricades són subconsultes que s'utilitzen dins d'una altra consulta. Es poden utilitzar per desglossar consultes complexes en parts més senzilles.

Ús de consultes imbricades a la clàusula FROM

SELECT state, AVG(year) AS avg_year
FROM (
  SELECT state, year
  FROM `bigquery-public-data.usa_names.usa_1910_2013`
  WHERE name = 'John'
)
GROUP BY state;
  • Explicació: Aquesta consulta primer selecciona l'estat i l'any per al nom 'John' i després calcula l'any mitjà per a cada estat.

Ús de consultes imbricades a la clàusula SELECT

SELECT name,
  (SELECT MAX(year) FROM `bigquery-public-data.usa_names.usa_1910_2013` WHERE state = 'TX') AS max_year
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX';
  • Explicació: Aquesta consulta selecciona noms i l'any màxim per a l'estat de Texas.

Ús avançat

Combinació de múltiples subconsultes

SELECT name, year, state
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE year = (SELECT MAX(year) FROM `bigquery-public-data.usa_names.usa_1910_2013`)
AND state IN (SELECT DISTINCT state FROM `bigquery-public-data.usa_names.usa_1910_2013` WHERE name = 'John');
  • Explicació: Aquesta consulta recupera noms, anys i estats on l'any és l'any màxim i l'estat és un on apareix el nom 'John'.

Consideracions de rendiment

  • Índexs: Assegura't que les columnes utilitzades en subconsultes estiguin indexades per obtenir un millor rendiment.
  • Optimització de consultes: Utilitza EXPLAIN per entendre el pla d'execució de la consulta i optimitzar en conseqüència.
  • Evitar subconsultes correlacionades: Quan sigui possible, evita les subconsultes correlacionades ja que poden ser menys eficients.

Conclusió

Les subconsultes i les consultes imbricades són essencials per escriure consultes SQL complexes i eficients a BigQuery. Entendre com utilitzar-les de manera efectiva pot millorar enormement les teves capacitats de recuperació de dades. Practica escrivint diferents tipus de subconsultes i consultes imbricades per convertir-te en un expert en el seu ús.
