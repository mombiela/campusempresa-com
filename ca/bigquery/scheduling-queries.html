Introducció

Programar consultes a BigQuery et permet automatitzar l'execució de consultes SQL a intervals especificats. Això és especialment útil per a tasques com generar informes regulars, actualitzar conjunts de dades o realitzar anàlisis de dades rutinàries.

Conceptes clau

  • Consultes programades: Consultes que es configuren per executar-se automàticament a intervals especificats.
  • Cloud Scheduler: Un servei de treballs cron totalment gestionat que et permet programar pràcticament qualsevol treball, incloent-hi consultes de BigQuery.
  • BigQuery Data Transfer Service: Un servei que automatitza el moviment de dades cap a BigQuery de manera programada i gestionada.

Configuració de consultes programades

Utilitzant la interfície de BigQuery

  1. Navega a la consola de BigQuery:
    • Ves a la interfície web de BigQuery a la Google Cloud Console.
  2. Crea una nova consulta programada:
    • Fes clic a la pestanya "Consultes programades".
    • Fes clic a "Crear consulta programada".
  3. Configura la consulta:
    • Introdueix la consulta SQL que vols programar.
    • Defineix la taula de destinació on es desaran els resultats.
  4. Defineix la programació:
    • Defineix la freqüència (per exemple, diària, setmanal) i l'hora específica per executar la consulta.
  5. Desa i activa:
    • Desa la consulta programada i assegura't que estigui activada.

Utilitzant Cloud Scheduler

  1. Crea una feina de Cloud Scheduler:
    • Navega a la pàgina de Cloud Scheduler a la Google Cloud Console.
    • Fes clic a "Crear feina".
  2. Configura la feina:
    • Defineix la freqüència utilitzant la sintaxi cron.
    • Tria el tipus de destinació com "HTTP".
    • Defineix l'URL per activar una funció de Cloud Function o un servei de Cloud Run que executi la consulta de BigQuery.
  3. Escriu la funció de Cloud Function:
    • Crea una funció de Cloud Function que contingui la lògica de la consulta de BigQuery.
    • Exemple: 
      from google.cloud import bigquery

def run_query(request):
    client = bigquery.Client()
    query = """
    SELECT name, COUNT(*) as name_count
    FROM `project.dataset.table`
    GROUP BY name
    """
    query_job = client.query(query)
    results = query_job.result()
    return "Query executed successfully"
  4. Desplega la funció de Cloud Function:
    • Desplega la funció de Cloud Function i pren nota de l'URL.
  5. Enllaça Cloud Scheduler amb Cloud Function:
    • Utilitza l'URL de la funció de Cloud Function desplegada a la feina de Cloud Scheduler.

Utilitzant BigQuery Data Transfer Service

  1. Navega a BigQuery Data Transfer Service:
    • Ves a la pàgina de BigQuery Data Transfer Service a la Google Cloud Console.
  2. Crea una transferència:
    • Fes clic a "Crear transferència".
  3. Selecciona la font de dades:
    • Tria "Consulta programada" com a font de dades.
  4. Configura la consulta:
    • Introdueix la consulta SQL i defineix la taula de destinació.
  5. Defineix la programació:
    • Defineix la freqüència i l'hora d'inici per a la consulta.
  6. Desa i activa:
    • Desa la configuració de la transferència i assegura't que estigui activada.

Exemple: Programar un informe diari de vendes

Consulta SQL

SELECT
  DATE(order_date) as date,
  SUM(total_amount) as total_sales
FROM
  `project.dataset.orders`
GROUP BY
  date
ORDER BY
  date;

Passos per programar

  1. Utilitzant la interfície de BigQuery:

    • Introdueix la consulta SQL anterior.
    • Defineix la taula de destinació a project.dataset.daily_sales_report.
    • Programa-la per executar-se diàriament a les 2 AM.

  2. Utilitzant Cloud Scheduler i Cloud Function:

    • Escriu una funció de Cloud Function amb la consulta anterior.
    • Desplega la funció de Cloud Function.
    • Crea una feina de Cloud Scheduler per activar la funció de Cloud Function diàriament a les 2 AM.

  3. Utilitzant BigQuery Data Transfer Service:

    • Crea una transferència amb la consulta SQL anterior.
    • Defineix la taula de destinació a project.dataset.daily_sales_report.
    • Programa-la per executar-se diàriament a les 2 AM.

Conclusió

Programar consultes a BigQuery és una característica potent que pot estalviar temps i garantir la consistència en l'anàlisi i els informes de dades. Tant si utilitzes la interfície de BigQuery, Cloud Scheduler o BigQuery Data Transfer Service, pots automatitzar els teus fluxos de treball de dades de manera eficient. Entendre aquests mètodes et permetrà triar l'enfocament més adequat per a les teves necessitats específiques i integrar el processament automatitzat de dades en els teus projectes de manera fluida.
