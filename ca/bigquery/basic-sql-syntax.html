Introducció

BigQuery és un magatzem de dades totalment gestionat i sense servidor que permet una anàlisi escalable de petabytes de dades. Utilitza SQL (Structured Query Language) per a consultar i gestionar dades. Aquesta secció cobrirà la sintaxi bàsica de SQL necessària per començar amb BigQuery.

Conceptes clau

  • SQL (Structured Query Language): Un llenguatge estàndard per accedir i manipular bases de dades.
  • BigQuery: Un servei de Google Cloud Platform per a magatzematge de dades i anàlisi.
  • Dataset: Una col·lecció de taules.
  • Table: Una col·lecció de files i columnes.
  • Query: Una sol·licitud de dades o informació d'una base de dades.

Declaracions bàsiques de SQL

Declaració SELECT

La declaració SELECT s'utilitza per consultar dades d'una base de dades.

SELECT column1, column2, ...
FROM table_name;

Exemple:

SELECT name, age
FROM employees;

Explicació: Aquesta consulta selecciona les columnes name i age de la taula employees.

Clàusula WHERE

La clàusula WHERE s'utilitza per filtrar registres.

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Exemple:

SELECT name, age
FROM employees
WHERE age > 30;

Explicació: Aquesta consulta selecciona les columnes name i age de la taula employees on l'edat és superior a 30.

Clàusula ORDER BY

La clàusula ORDER BY s'utilitza per ordenar el conjunt de resultats.

SELECT column1, column2, ...
FROM table_name
ORDER BY column1 [ASC|DESC];

Exemple:

SELECT name, age
FROM employees
ORDER BY age DESC;

Explicació: Aquesta consulta selecciona les columnes name i age de la taula employees i ordena els resultats per edat en ordre descendent.

Clàusula LIMIT

La clàusula LIMIT s'utilitza per especificar el nombre de registres a retornar.

SELECT column1, column2, ...
FROM table_name
LIMIT number;

Exemple:

SELECT name, age
FROM employees
LIMIT 5;

Explicació: Aquesta consulta selecciona les columnes name i age de la taula employees i retorna només els primers 5 registres.

Funcions d'agregació

Les funcions d'agregació realitzen un càlcul sobre un conjunt de valors i retornen un únic valor.

  • COUNT(): Retorna el nombre de files.
  • SUM(): Retorna la suma total d'una columna numèrica.
  • AVG(): Retorna el valor mitjà d'una columna numèrica.
  • MAX(): Retorna el valor màxim.
  • MIN(): Retorna el valor mínim.

Exemple:

SELECT COUNT(*), AVG(age), MAX(age), MIN(age)
FROM employees;

Explicació: Aquesta consulta retorna el nombre total d'empleats, l'edat mitjana, l'edat màxima i l'edat mínima de la taula employees.

Clàusula GROUP BY

La clàusula GROUP BY agrupa files que tenen els mateixos valors en files de resum.

SELECT column1, COUNT(*)
FROM table_name
GROUP BY column1;

Exemple:

SELECT department, COUNT(*)
FROM employees
GROUP BY department;

Explicació: Aquesta consulta agrupa els empleats per departament i retorna el nombre d'empleats en cada departament.

Clàusula JOIN

La clàusula JOIN s'utilitza per combinar files de dues o més taules.

  • INNER JOIN: Retorna registres que tenen valors coincidents en ambdues taules.
  • LEFT JOIN: Retorna tots els registres de la taula esquerra, i els registres coincidents de la taula dreta.
  • RIGHT JOIN: Retorna tots els registres de la taula dreta, i els registres coincidents de la taula esquerra.
  • FULL OUTER JOIN: Retorna tots els registres quan hi ha una coincidència en qualsevol de les taules.

Exemple:

SELECT employees.name, departments.department_name
FROM employees
INNER JOIN departments ON employees.department_id = departments.id;

Explicació: Aquesta consulta selecciona el name de la taula employees i el department_name de la taula departments on el department_id en employees coincideix amb l'id en departments.

Conclusió

Entendre la sintaxi bàsica de SQL és essencial per a consultar i gestionar dades en BigQuery. Aquesta secció ha cobert declaracions fonamentals de SQL com SELECT, WHERE, ORDER BY, LIMIT, funcions d'agregació, GROUP BY i JOIN. Dominar aquests conceptes bàsics proporcionarà una base sòlida per a consultes SQL més avançades i anàlisi de dades en BigQuery.
