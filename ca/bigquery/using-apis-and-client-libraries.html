Introducció

BigQuery és un magatzem de dades totalment gestionat i sense servidor que permet una anàlisi escalable de petabytes de dades. Per interactuar amb BigQuery programàticament, pots utilitzar APIs i llibreries de clients. Aquest tema et guiarà a través de l'ús de les APIs i llibreries de clients de BigQuery des de nivells principiants fins a avançats.

Conceptes Clau

  • API (Interfície de Programació d'Aplicacions): Un conjunt de regles que permeten que diferents entitats de programari es comuniquin entre elles.
  • Llibreria de Clients: Una col·lecció de codi preescrit que els desenvolupadors poden utilitzar per interactuar amb APIs més fàcilment.
  • API de BigQuery: Proporciona accés programàtic a les funcions de BigQuery.
  • Autenticació: El procés de verificar la identitat d'un usuari o aplicació.

Configuració

Abans de poder utilitzar l'API de BigQuery o les llibreries de clients, has de configurar el teu entorn.

Prerequisits

  • Un compte de Google Cloud Platform (GCP).
  • Un projecte de GCP amb BigQuery habilitat.
  • Credencials de compte de servei per a l'autenticació.

Autenticació

  1. Crear un Compte de Servei:

    • Ves a la Consola de GCP.
    • Navega a IAM & Admin > Comptes de Servei.
    • Fes clic a "Crear Compte de Servei" i segueix les instruccions.
    • Descarrega el fitxer de clau JSON.

  2. Configura la Variable d'Entorn:

    export GOOGLE_APPLICATION_CREDENTIALS="/path/to/your/service-account-file.json"

Ús de l'API de BigQuery

L'API de BigQuery et permet interactuar amb BigQuery programàticament. A continuació es mostren els passos per realitzar operacions bàsiques utilitzant l'API.

Fer Sol·licituds API

Pots utilitzar eines com curl o Postman per fer sol·licituds HTTP a l'API de BigQuery.

Exemple: Llistar Conjunts de Dades

curl -X GET \
  -H "Authorization: Bearer $(gcloud auth application-default print-access-token)" \
  "https://bigquery.googleapis.com/bigquery/v2/projects/your-project-id/datasets"

Analitzar Respostes API

Les respostes API solen estar en format JSON. Pots utilitzar eines com jq per analitzar respostes JSON.

Exemple: Analitzar Resposta JSON

curl -X GET \
  -H "Authorization: Bearer $(gcloud auth application-default print-access-token)" \
  "https://bigquery.googleapis.com/bigquery/v2/projects/your-project-id/datasets" | jq .

Ús de Llibreries de Clients

Les llibreries de clients proporcionen una manera més convenient d'interactuar amb BigQuery. Google proporciona llibreries de clients per a diversos llenguatges de programació.

Llibreria de Clients de Python

Python és un dels llenguatges més populars per a l'anàlisi de dades. A continuació es mostren els passos per utilitzar la llibreria de clients de BigQuery per a Python.

Instal·lació

pip install google-cloud-bigquery

Operacions Bàsiques

  1. Inicialitzar el Client:

    from google.cloud import bigquery

client = bigquery.Client()

  2. Llistar Conjunts de Dades:

    datasets = list(client.list_datasets())
for dataset in datasets:
    print(dataset.dataset_id)

  3. Executar una Consulta:

    query = """
SELECT name, SUM(number) as total
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
GROUP BY name
ORDER BY total DESC
LIMIT 10
"""
query_job = client.query(query)

for row in query_job:
    print(f"name: {row['name']}, total: {row['total']}")

Llibreria de Clients de JavaScript

JavaScript és àmpliament utilitzat per a aplicacions web. A continuació es mostren els passos per utilitzar la llibreria de clients de BigQuery per a JavaScript.

Instal·lació

npm install --save @google-cloud/bigquery

Operacions Bàsiques

  1. Inicialitzar el Client:

    const {BigQuery} = require('@google-cloud/bigquery');
const bigquery = new BigQuery();

  2. Llistar Conjunts de Dades:

    async function listDatasets() {
  const [datasets] = await bigquery.getDatasets();
  datasets.forEach(dataset => console.log(dataset.id));
}

listDatasets();

  3. Executar una Consulta:

    async function runQuery() {
  const query = `
  SELECT name, SUM(number) as total
  FROM \`bigquery-public-data.usa_names.usa_1910_2013\`
  WHERE state = 'TX'
  GROUP BY name
  ORDER BY total DESC
  LIMIT 10
  `;
  const [rows] = await bigquery.query(query);
  rows.forEach(row => console.log(`name: ${row.name}, total: ${row.total}`));
}

runQuery();

Conclusió

Utilitzar APIs i llibreries de clients per interactuar amb BigQuery permet realitzar operacions de dades potents i flexibles. Tant si utilitzes Python, JavaScript, o un altre llenguatge, entendre com autenticar-se, fer sol·licituds i gestionar respostes és crucial. En dominar aquestes habilitats, pots aprofitar tot el potencial de BigQuery per a les teves necessitats d'anàlisi de dades.
