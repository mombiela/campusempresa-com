Introducció

Analitzar datasets públics és una habilitat crucial per a científics de dades i analistes. Google BigQuery proporciona una plataforma potent per consultar i analitzar grans datasets de manera eficient. Aquest tema et guiarà a través del procés d'analitzar datasets públics utilitzant BigQuery, des de nivells principiants fins a avançats.

Començant amb BigQuery

Configurant el Teu Entorn

  • Crea un compte a Google Cloud Platform (GCP): Registra't per obtenir una capa gratuïta si no tens un compte.
  • Habilita l'API de BigQuery: Navega a la biblioteca d'API al consola de GCP i habilita BigQuery.
  • Crea un nou projecte: Això t'ajudarà a organitzar els teus datasets i consultes.

Accedint a Datasets Públics

  • Datasets Públics de BigQuery: Google proporciona una varietat de datasets públics als quals pots accedir directament des de BigQuery.
  • Navegador de Datasets: Utilitza la interfície web de BigQuery per navegar pels datasets públics disponibles.

Exemple: Carregant un Dataset Públic

-- Exemple de SQL per explorar un dataset públic
SELECT 
  name, 
  birth_year 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` 
LIMIT 10;
  • Explicació: Aquesta consulta selecciona les columnes name i birth_year del dataset usa_names, limitant els resultats a 10 files.

Tècniques Bàsiques de Consulta

Seleccionant Dades

  • Instrucció SELECT: Utilitzada per especificar les columnes que vols recuperar.
  • Clàusula FROM: Especifica el dataset i la taula.

Filtrant Dades

  • Clàusula WHERE: Utilitzada per filtrar registres basats en condicions.
SELECT 
  name, 
  birth_year 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` 
WHERE 
  gender = 'F' 
LIMIT 10;
  • Explicació: Aquesta consulta filtra el dataset per incloure només noms femenins.

Agregant Dades

  • Clàusula GROUP BY: Utilitzada per agrupar files que tenen els mateixos valors en columnes especificades.
  • Funcions d'Agregació: Com COUNT(), SUM(), AVG(), MIN(), i MAX().
SELECT 
  state, 
  COUNT(name) AS name_count 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` 
GROUP BY 
  state 
ORDER BY 
  name_count DESC 
LIMIT 10;
  • Explicació: Aquesta consulta compta el nombre de noms per estat i ordena els resultats en ordre descendent.

Tècniques Intermèdies de Consulta

Unint Taules

  • INNER JOIN: Combina files de dues taules basant-se en una columna relacionada.
SELECT 
  a.name, 
  b.population 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` AS a 
INNER JOIN 
  `bigquery-public-data.census_bureau_international.population_by_country` AS b 
ON 
  a.state = b.country_code 
LIMIT 10;
  • Explicació: Aquesta consulta uneix el dataset usa_names amb el dataset population_by_country en les columnes state i country_code.

Utilitzant Subconsultes

  • Subconsulta: Una consulta niada dins d'una altra consulta.
SELECT 
  name, 
  birth_year 
FROM 
  (SELECT 
    name, 
    birth_year 
  FROM 
    `bigquery-public-data.usa_names.usa_1910_2013` 
  WHERE 
    gender = 'M') 
WHERE 
  birth_year > 2000 
LIMIT 10;
  • Explicació: Aquesta consulta primer filtra per noms masculins i després filtra més per noms amb anys de naixement posteriors al 2000.

Tècniques Avançades de Consulta

Funcions de Finestra

  • Clàusula OVER: Utilitzada amb funcions de finestra per realitzar càlculs a través d'un conjunt de files de la taula.
SELECT 
  name, 
  birth_year, 
  COUNT(*) OVER (PARTITION BY birth_year) AS birth_year_count 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` 
LIMIT 10;
  • Explicació: Aquesta consulta compta el nombre de noms per cada any de naixement utilitzant una funció de finestra.

Funcions Definides per l'Usuari (UDFs)

  • UDFs en JavaScript: Et permeten escriure funcions personalitzades en JavaScript.
CREATE TEMP FUNCTION MultiplyByTwo(x INT64) 
RETURNS INT64 
LANGUAGE js AS """
  return x * 2;
""";

SELECT 
  name, 
  MultiplyByTwo(birth_year) AS birth_year_multiplied 
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013` 
LIMIT 10;
  • Explicació: Aquesta consulta crea una funció temporal que multiplica un número per dos i l'aplica a la columna birth_year.

Conclusió

Analitzar datasets públics amb BigQuery implica entendre com accedir a datasets, escriure consultes SQL bàsiques a avançades, i utilitzar les potents característiques de BigQuery com unions, subconsultes, funcions de finestra i UDFs. En dominar aquestes tècniques, pots analitzar grans datasets de manera eficient i obtenir informació significativa.

Resum

  • Configuració: Crea un compte a GCP, habilita l'API de BigQuery i crea un projecte.
  • Consultes Bàsiques: SELECT, WHERE, GROUP BY.
  • Consultes Intermèdies: JOINs, subconsultes.
  • Consultes Avançades: Funcions de finestra, UDFs.
  • Pràctica: Practica regularment consultant diferents datasets públics per millorar les teves habilitats.
