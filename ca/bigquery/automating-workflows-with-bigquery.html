Automatització de Fluxos de Treball amb BigQuery
Programació de consultes ►

Introducció

L'automatització de fluxos de treball a BigQuery pot millorar significativament la productivitat reduint la intervenció manual i assegurant un processament de dades consistent. Aquesta secció cobrirà els aspectes essencials de l'automatització de tasques a BigQuery, des de la programació de consultes fins a la integració amb altres serveis de Google Cloud.

Conceptes Clau

  • Consultes Programades: Automatitza l'execució de consultes SQL a intervals especificats.
  • Servei de Transferència de Dades de BigQuery: Automatitza el moviment de dades de diverses fonts cap a BigQuery.
  • Funcions Cloud: Activa operacions de BigQuery basades en esdeveniments.
  • Cloud Composer: Orquestra fluxos de treball complexos que involucren BigQuery i altres serveis.
  • APIs i Llibreries de Clients: Interactua programàticament amb BigQuery per a l'automatització.

Consultes Programades

Les consultes programades permeten executar consultes SQL de manera recurrent. Això és útil per a tasques com informes diaris o agregació periòdica de dades.

Exemple: Creació d'una Consulta Programada

-- Exemple de consulta SQL per agregar dades de vendes diàries
SELECT 
  DATE(TIMESTAMP_TRUNC(order_timestamp, DAY)) AS order_date,
  SUM(order_amount) AS total_sales
FROM 
  `project.dataset.orders`
GROUP BY 
  order_date;

Per programar aquesta consulta:

  1. Ves a la consola de BigQuery.
  2. Navega a la secció "Consultes Programades".
  3. Fes clic a "Crear Consulta Programada".
  4. Introdueix la consulta SQL i estableix el programa (per exemple, diàriament a mitjanit).

Servei de Transferència de Dades de BigQuery

El Servei de Transferència de Dades de BigQuery automatitza la importació de dades de diverses fonts, com Google Ads, Google Analytics i bases de dades externes.

Exemple: Configuració d'una Transferència de Dades

  1. A la consola de BigQuery, ves a la secció "Transferències".
  2. Fes clic a "Crear Transferència".
  3. Selecciona la font de dades (per exemple, Google Analytics).
  4. Configura els paràmetres de la transferència, incloent el conjunt de dades de destinació i el programa.

Funcions Cloud

Les Funcions Cloud es poden utilitzar per activar operacions de BigQuery basades en esdeveniments, com ara càrregues de fitxers a Cloud Storage o canvis a Firestore.

Exemple: Activació d'una Consulta de BigQuery amb Funcions Cloud

const {BigQuery} = require('@google-cloud/bigquery');
const bigquery = new BigQuery();

exports.triggerBigQuery = async (event, context) => {
  const query = `SELECT * FROM \`project.dataset.table\` LIMIT 10`;
  const options = {
    query: query,
    location: 'US',
  };

  const [job] = await bigquery.createQueryJob(options);
  console.log(`Job ${job.id} started.`);
};
  1. Desplega aquesta funció a Funcions Cloud.
  2. Configura un activador, com un esdeveniment de Cloud Storage.

Cloud Composer

Cloud Composer és un servei d'orquestració de fluxos de treball gestionat basat en Apache Airflow. Permet crear, programar i monitoritzar fluxos de treball complexos.

Exemple: Creació d'un Flux de Treball amb Cloud Composer

  1. Crea un nou entorn a Cloud Composer.
  2. Defineix un DAG (Graf Dirigit Acíclic) en Python:
from airflow import DAG
from airflow.providers.google.cloud.operators.bigquery import BigQueryExecuteQueryOperator
from datetime import datetime

default_args = {
    'start_date': datetime(2023, 1, 1),
}

with DAG('bigquery_workflow', default_args=default_args, schedule_interval='@daily') as dag:
    execute_query = BigQueryExecuteQueryOperator(
        task_id='execute_query',
        sql='SELECT * FROM `project.dataset.table` LIMIT 10',
        use_legacy_sql=False,
    )
  1. Desplega el DAG al teu entorn de Cloud Composer.

APIs i Llibreries de Clients

BigQuery proporciona APIs i llibreries de clients per a diversos llenguatges de programació, permetent automatitzar tasques programàticament.

Exemple: Ús de la Llibreria de Clients de Python

from google.cloud import bigquery

client = bigquery.Client()

query = """
SELECT 
  name, 
  SUM(number) as total
FROM 
  `bigquery-public-data.usa_names.usa_1910_2013`
GROUP BY 
  name
ORDER BY 
  total DESC
LIMIT 
  10
"""

query_job = client.query(query)

for row in query_job:
    print(f"{row.name}: {row.total}")
  1. Instal·la la llibreria de clients de BigQuery: pip install google-cloud-bigquery.
  2. Executa el script per executar la consulta i processar els resultats.

Conclusió

L'automatització de fluxos de treball amb BigQuery pot agilitzar el processament i l'anàlisi de dades, facilitant la gestió de grans conjunts de dades i operacions complexes. Mitjançant l'ús de consultes programades, el Servei de Transferència de Dades de BigQuery, Funcions Cloud, Cloud Composer i APIs, pots crear fluxos de treball de dades eficients i fiables. Comença identificant tasques repetitives en la teva pipeline de dades i explora com aquestes eines poden ajudar-te a automatitzar-les.
