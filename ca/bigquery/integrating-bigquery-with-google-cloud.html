Introducció

Integrar BigQuery amb altres serveis de Google Cloud pot millorar significativament les teves capacitats de processament, anàlisi i visualització de dades. Aquesta secció cobrirà com connectar BigQuery amb diversos serveis de Google Cloud, incloent Google Cloud Storage, Google Cloud Pub/Sub, Google Data Studio i Google Cloud Functions.

Google Cloud Storage

Google Cloud Storage (GCS) és un servei d'emmagatzematge d'objectes escalable que et permet emmagatzemar i recuperar qualsevol quantitat de dades en qualsevol moment.

Carregar dades des de Google Cloud Storage a BigQuery

Per carregar dades des de GCS a BigQuery, pots utilitzar la interfície web de BigQuery, l'eina de línia de comandes bq o l'API de BigQuery.

Utilitzant la interfície web de BigQuery

  1. Ves a la pàgina de BigQuery al Google Cloud Console.
  2. Al panell d'Explorador, selecciona el conjunt de dades on vols carregar les dades.
  3. Fes clic al botó "Crear Taula".
  4. A la secció "Origen", selecciona "Google Cloud Storage" i proporciona l'URI del fitxer de GCS.
  5. Configura la resta de paràmetres i fes clic a "Crear Taula".

Utilitzant l'eina de línia de comandes bq

bq load --source_format=CSV mydataset.mytable gs://mybucket/myfile.csv
  • mydataset.mytable: El conjunt de dades i la taula de destinació a BigQuery.
  • gs://mybucket/myfile.csv: L'URI del fitxer a GCS.

Exportar dades de BigQuery a Google Cloud Storage

També pots exportar dades de BigQuery a GCS per fer còpies de seguretat o per a un processament posterior.

Utilitzant la interfície web de BigQuery

  1. Ves a la pàgina de BigQuery al Google Cloud Console.
  2. Al panell d'Explorador, selecciona la taula que vols exportar.
  3. Fes clic al botó "Exportar".
  4. Proporciona l'URI de destinació a GCS i configura els paràmetres d'exportació.
  5. Fes clic a "Exportar".

Utilitzant l'eina de línia de comandes bq

bq extract --destination_format=CSV mydataset.mytable gs://mybucket/myfile.csv
  • mydataset.mytable: El conjunt de dades i la taula d'origen a BigQuery.
  • gs://mybucket/myfile.csv: L'URI de destinació a GCS.

Google Cloud Pub/Sub

Google Cloud Pub/Sub és un servei de missatgeria per intercanviar dades d'esdeveniments entre aplicacions i serveis.

Transmissió de dades de Pub/Sub a BigQuery

Pots crear una pipeline de dades per transmetre dades de Pub/Sub a BigQuery utilitzant Dataflow.

Configurant la pipeline de Dataflow

  1. Crea un tema i una subscripció de Pub/Sub.
  2. Crea una feina de Dataflow per llegir missatges de la subscripció de Pub/Sub i escriure'ls a BigQuery.

Exemple de codi de pipeline de Dataflow en Python:

import apache_beam as beam
from apache_beam.options.pipeline_options import PipelineOptions
from apache_beam.io.gcp.bigquery import WriteToBigQuery

def run():
    options = PipelineOptions()
    p = beam.Pipeline(options=options)

    (p
     | 'Read from Pub/Sub' >> beam.io.ReadFromPubSub(subscription='projects/myproject/subscriptions/mysubscription')
     | 'Transform Data' >> beam.Map(lambda x: {'field1': x['field1'], 'field2': x['field2']})
     | 'Write to BigQuery' >> WriteToBigQuery('myproject:mydataset.mytable'))

    p.run().wait_until_finish()

if __name__ == '__main__':
    run()
  • myproject:mydataset.mytable: El conjunt de dades i la taula de destinació a BigQuery.
  • projects/myproject/subscriptions/mysubscription: La subscripció de Pub/Sub.

Google Data Studio

Google Data Studio és una eina de visualització de dades que et permet crear quadres de comandament i informes interactius.

Connectar BigQuery a Google Data Studio

  1. Obre Google Data Studio.
  2. Fes clic a "Crear" i selecciona "Font de Dades".
  3. Tria "BigQuery" com a connector.
  4. Selecciona el teu projecte, conjunt de dades i taula.
  5. Fes clic a "Connectar" i configura la font de dades.

Google Cloud Functions

Google Cloud Functions és un entorn d'execució sense servidor per construir i connectar serveis al núvol.

Activar feines de BigQuery amb Cloud Functions

Pots utilitzar Cloud Functions per activar feines de BigQuery basades en esdeveniments, com ara la càrrega de fitxers a GCS.

Exemple de funció al núvol en Python:

from google.cloud import bigquery

def trigger_bigquery_job(event, context):
    client = bigquery.Client()
    query = """
    SELECT *
    FROM `myproject.mydataset.mytable`
    WHERE timestamp >= TIMESTAMP_SUB(CURRENT_TIMESTAMP(), INTERVAL 1 DAY)
    """
    job = client.query(query)
    results = job.result()
    for row in results:
        print(row)
  • myproject.mydataset.mytable: El conjunt de dades i la taula de destinació a BigQuery.

Conclusió

Integrar BigQuery amb altres serveis de Google Cloud pot agilitzar els teus fluxos de treball de dades i millorar les teves capacitats analítiques. Tant si estàs carregant dades des de Google Cloud Storage, transmetent dades amb Pub/Sub, visualitzant dades a Google Data Studio o automatitzant tasques amb Cloud Functions, aquestes integracions poden ajudar-te a aprofitar tot el potencial de Google Cloud.
