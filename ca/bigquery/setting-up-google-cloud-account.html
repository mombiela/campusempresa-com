◄ Què és BigQuery?
Visió general de la consola de BigQuery ►

Introducció

Abans d'endinsar-te en BigQuery, és essencial configurar el teu compte de Google Cloud. Aquesta secció et guiarà a través del procés, assegurant que tinguis tot el que necessites per començar a treballar amb BigQuery.

Creació d'un compte de Google Cloud

Per utilitzar BigQuery, primer necessites un compte de Google Cloud. Segueix aquests passos per crear-ne un:

  1. Visita el lloc web de Google Cloud: Ves a Google Cloud.
  2. Registra't: Fes clic al botó "Comença gratis".
  3. Compte de Google: Si ja tens un compte de Google, inicia sessió. Si no, crea un nou compte de Google.
  4. Informació de facturació: Introdueix la teva informació de facturació. Google Cloud ofereix un nivell gratuït amb 300 dòlars en crèdits per a nous usuaris, que pots utilitzar per explorar BigQuery i altres serveis.

Configuració d'un projecte

A Google Cloud, els recursos s'organitzen en projectes. Necessites crear un projecte per utilitzar BigQuery.

  1. Navega a la Consola: Ves a la Consola de Google Cloud.
  2. Crea un nou projecte:
    • Fes clic al desplegable de projectes a la part superior de la pàgina.
    • Selecciona "Nou projecte".
    • Introdueix un nom de projecte i selecciona el teu compte de facturació.
    • Fes clic a "Crear".

Activació de l'API de BigQuery

Per utilitzar BigQuery, necessites activar l'API de BigQuery per al teu projecte.

  1. API i Serveis: A la Consola de Google Cloud, navega a "API i Serveis" > "Biblioteca".
  2. Cerca BigQuery: Escriu "BigQuery" a la barra de cerca.
  3. Activa l'API: Fes clic a "API de BigQuery" i després fes clic a "Activa".

Configuració de la facturació

BigQuery cobra per l'emmagatzematge de dades, les consultes i el processament de dades. Assegura't que la teva facturació estigui configurada correctament.

  1. Secció de facturació: A la Consola de Google Cloud, ves a "Facturació".
  2. Enllaça el compte de facturació: Enllaça el teu projecte a un compte de facturació. Si no en tens cap, hauràs de crear-ne un.

Creació d'un compte de servei

Els comptes de servei s'utilitzen per autenticar i autoritzar aplicacions per accedir als recursos de Google Cloud.

  1. IAM i Administració: A la Consola de Google Cloud, navega a "IAM i Administració" > "Comptes de servei".
  2. Crea un compte de servei:
    • Fes clic a "Crea un compte de servei".
    • Introdueix un nom i una descripció.
    • Fes clic a "Crear".
  3. Assigna rols: Assigna els rols necessaris al compte de servei. Per a BigQuery, podries necessitar rols com "Usuari de BigQuery" o "Administrador de BigQuery".
  4. Crea una clau: Crea una clau per al compte de servei. Tria el format JSON i descarrega el fitxer de clau. Aquest fitxer s'utilitzarà per autenticar les teves aplicacions.

Accés a BigQuery

Pots accedir a BigQuery a través de diverses metodologies:

  • Consola de BigQuery: La interfície web disponible a la Consola de Google Cloud.
  • Eina de línia de comandes bq: Una eina de línia de comandes per interactuar amb BigQuery.
  • Biblioteques de clients: Biblioteques disponibles per a diversos llenguatges de programació com Python, Java i Node.js.

Exemple: Accés a BigQuery mitjançant Python

Per accedir a BigQuery utilitzant Python, necessites instal·lar la biblioteca client de Google Cloud BigQuery i autenticar-te utilitzant la clau del compte de servei.

  1. Instal·la la biblioteca client:

    pip install google-cloud-bigquery

  2. Autentica't i consulta BigQuery:

    from google.cloud import bigquery

# Configura l'autenticació
client = bigquery.Client.from_service_account_json('path/to/service_account_key.json')

# Defineix una consulta
query = """
SELECT name, SUM(number) as total
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
GROUP BY name
ORDER BY total DESC
LIMIT 10
"""

# Executa la consulta
query_job = client.query(query)

# Imprimeix els resultats
for row in query_job:
    print(f"{row.name}: {row.total}")

Conclusió

Configurar el teu compte de Google Cloud és el primer pas per utilitzar BigQuery. Seguint els passos descrits anteriorment, tindràs un projecte preparat amb BigQuery activat, la facturació configurada i un compte de servei creat per a l'autenticació. Amb aquesta base, pots començar a explorar i consultar dades a BigQuery.
