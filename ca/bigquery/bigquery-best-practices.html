Introducció

BigQuery és un magatzem de dades totalment gestionat i sense servidor que permet l'anàlisi escalable de petabytes de dades. Per treure el màxim profit de BigQuery, és essencial seguir les millors pràctiques que assegurin consultes i gestió de dades eficients, rendibles i mantenibles.

Optimització de Consultes

Optimitzar les teves consultes pot reduir significativament els costos i millorar el rendiment.

  • *Evita SELECT : En lloc de seleccionar totes les columnes, especifica només les columnes que necessites.

    -- Ineficient
SELECT * FROM `project.dataset.table`;

-- Eficient
SELECT column1, column2 FROM `project.dataset.table`;

  • Utilitza Taules Particionades: La partició ajuda a reduir la quantitat de dades escanejades.

    CREATE TABLE `project.dataset.partitioned_table`
PARTITION BY DATE(timestamp_column)
AS SELECT * FROM `project.dataset.original_table`;

  • Aprofita el Clustering: El clustering pot millorar el rendiment de les consultes organitzant les dades segons els valors d'una o més columnes.

    CREATE TABLE `project.dataset.clustered_table`
CLUSTER BY column1, column2
AS SELECT * FROM `project.dataset.original_table`;

  • Filtra Primer: Aplica filtres tan aviat com sigui possible en la teva consulta per minimitzar la quantitat de dades processades.

    -- Menys eficient
SELECT column1, column2 FROM `project.dataset.table`
WHERE column3 = 'value';

-- Més eficient
WITH filtered_data AS (
  SELECT column1, column2 FROM `project.dataset.table`
  WHERE column3 = 'value'
)
SELECT * FROM filtered_data;

Gestió de Costos

Gestionar els costos és crucial quan es treballa amb grans conjunts de dades.

  • Estima els Costos de les Consultes: Utilitza la declaració EXPLAIN per estimar el cost de les teves consultes.

    EXPLAIN SELECT column1, column2 FROM `project.dataset.table` WHERE column3 = 'value';

  • Utilitza la Memòria Cau: BigQuery emmagatzema en memòria cau els resultats de les consultes durant 24 hores. Reutilitzar els resultats emmagatzemats pot estalviar costos.

    -- Primera execució
SELECT column1, column2 FROM `project.dataset.table` WHERE column3 = 'value';

-- Execució posterior dins de les 24 hores (utilitza la memòria cau)
SELECT column1, column2 FROM `project.dataset.table` WHERE column3 = 'value';

  • Estableix Límits de Consulta: Limita la quantitat de dades processades establint límits de consulta.

    -- Limita el nombre de files retornades
SELECT column1, column2 FROM `project.dataset.table` WHERE column3 = 'value' LIMIT 1000;

Gestió de Dades

Una gestió adequada de les dades assegura la integritat i l'accessibilitat de les dades.

  • Utilitza Esquemes: Defineix esquemes per a les teves taules per assegurar la consistència de les dades.

    CREATE TABLE `project.dataset.table` (
  column1 STRING,
  column2 INT64,
  column3 TIMESTAMP
);

  • Tipus de Dades: Utilitza tipus de dades adequats per optimitzar l'emmagatzematge i el rendiment.

    -- Exemple de definició de tipus de dades
CREATE TABLE `project.dataset.table` (
  name STRING,
  age INT64,
  birthdate DATE
);

  • Ingestió de Dades: Utilitza mètodes eficients per a la ingestió de dades, com ara la càrrega per lots o el streaming.

    -- Exemple de càrrega per lots
LOAD DATA INTO `project.dataset.table`
FROM 'gs://bucket/path/to/data.csv'
WITH
  FORMAT = CSV,
  FIELD_DELIMITER = ',',
  SKIP_LEADING_ROWS = 1;

Millors Pràctiques de Seguretat

Assegurar la seguretat de les teves dades és fonamental.

  • Controls d'Accés: Utilitza rols IAM per controlar l'accés a conjunts de dades i taules.

    # Concedir un rol a un usuari
gcloud projects add-iam-policy-binding project-id 
        --member=user:email@example.com 
        --role=roles/bigquery.dataViewer

  • Xifrat de Dades: Assegura't que les dades estiguin xifrades tant en trànsit com en repòs.

    • BigQuery xifra automàticament les dades en repòs.
    • Utilitza TLS/SSL per a les dades en trànsit.

  • Registres d'Auditoria: Activa els registres d'auditoria per fer un seguiment de l'accés i els canvis a les teves dades.

    # Activació dels registres d'auditoria
gcloud logging sinks create my-sink bigquery.googleapis.com/projects/project-id/datasets/audit_logs

Conclusió

Seguir aquestes millors pràctiques t'ajudarà a optimitzar l'ús de BigQuery, gestionar els costos de manera efectiva, mantenir la integritat de les dades i assegurar la seguretat. Implementant aquestes estratègies, podràs treure el màxim profit de les potents capacitats de BigQuery mentre mantens les teves operacions de dades eficients i segures.
