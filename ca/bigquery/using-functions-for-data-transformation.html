Introducció

En aquesta secció, explorarem com utilitzar funcions a BigQuery per transformar dades. La transformació de dades és un pas crucial en l'anàlisi i preparació de dades, permetent-te netejar, manipular i preparar les teves dades per a una anàlisi o informes posteriors.

Conceptes Clau

  • Funcions: Operacions predefinides que es poden aplicar a les dades per realitzar tasques específiques.
  • Transformació de Dades: El procés de convertir dades d'un format o estructura a un altre.
  • BigQuery SQL: El dialecte SQL utilitzat a Google BigQuery per consultar i manipular dades.

Funcions Comunes a BigQuery

BigQuery proporciona una àmplia gamma de funcions que es poden utilitzar per a la transformació de dades. Aquí hi ha algunes de les més utilitzades:

Funcions de Cadena

  • UPPER(): Converteix una cadena a majúscules.
  • LOWER(): Converteix una cadena a minúscules.
  • CONCAT(): Concatena dues o més cadenes.
  • SUBSTR(): Extreu una subcadena d'una cadena.
SELECT 
  UPPER('hello world') AS upper_case,
  LOWER('HELLO WORLD') AS lower_case,
  CONCAT('Hello', ' ', 'World') AS concatenated_string,
  SUBSTR('Hello World', 1, 5) AS substring;

Funcions Numèriques

  • ROUND(): Arrodoneix un número a un nombre especificat de decimals.
  • CEIL(): Arrodoneix un número cap amunt al nombre enter més proper.
  • FLOOR(): Arrodoneix un número cap avall al nombre enter més proper.
  • ABS(): Retorna el valor absolut d'un número.
SELECT 
  ROUND(3.14159, 2) AS rounded_value,
  CEIL(3.14159) AS ceiling_value,
  FLOOR(3.14159) AS floor_value,
  ABS(-42) AS absolute_value;

Funcions de Data

  • CURRENT_DATE(): Retorna la data actual.
  • DATE_ADD(): Afegeix un interval de temps especificat a una data.
  • DATE_SUB(): Resta un interval de temps especificat d'una data.
  • FORMAT_DATE(): Dona format a una data segons un format especificat.
SELECT 
  CURRENT_DATE() AS current_date,
  DATE_ADD(CURRENT_DATE(), INTERVAL 7 DAY) AS next_week,
  DATE_SUB(CURRENT_DATE(), INTERVAL 1 MONTH) AS last_month,
  FORMAT_DATE('%Y-%m-%d', CURRENT_DATE()) AS formatted_date;

Funcions Condicionals

  • IF(): Retorna un valor basat en una condició.
  • CASE: Avalua una llista de condicions i retorna un resultat corresponent.
SELECT 
  IF(1 > 0, 'True', 'False') AS if_example,
  CASE 
    WHEN 1 > 0 THEN 'Positive'
    WHEN 1 < 0 THEN 'Negative'
    ELSE 'Zero'
  END AS case_example;

Transformació Avançada de Dades

A mesura que et familiaritzis amb les funcions bàsiques, pots començar a combinar-les per realitzar transformacions més complexes.

Combinació de Funcions

Pots niar funcions dins d'altres per realitzar múltiples transformacions en una sola consulta.

SELECT 
  UPPER(CONCAT('Hello', ' ', 'World')) AS upper_concatenated_string,
  ROUND(ABS(-3.14159), 2) AS rounded_absolute_value;

Ús de Funcions de Finestra

Les funcions de finestra et permeten realitzar càlculs a través d'un conjunt de files de taula que estan relacionades amb la fila actual.

SELECT 
  name,
  score,
  RANK() OVER (ORDER BY score DESC) AS rank
FROM 
  students;

Pivotar Dades

Pivotar dades és una transformació comuna que implica convertir files en columnes.

SELECT 
  name,
  MAX(CASE WHEN subject = 'Math' THEN score END) AS math_score,
  MAX(CASE WHEN subject = 'Science' THEN score END) AS science_score
FROM 
  student_scores
GROUP BY 
  name;

Conclusió

Utilitzar funcions per a la transformació de dades a BigQuery és una manera poderosa de manipular i preparar les teves dades per a l'anàlisi. Dominant aquestes funcions, pots netejar, formatar i transformar les teves dades de manera eficient. Comença amb funcions bàsiques i gradualment passa a transformacions més complexes a mesura que et familiaritzis amb BigQuery SQL.
