Introducció a les Funcions Definides per l'Usuari (UDFs)

Les Funcions Definides per l'Usuari (UDFs) a BigQuery et permeten ampliar les capacitats de SQL escrivint funcions personalitzades utilitzant JavaScript o SQL. Les UDFs es poden utilitzar per encapsular lògica complexa, realitzar càlculs personalitzats i reutilitzar codi en diverses consultes.

Conceptes Clau

  • Tipus de UDFs: BigQuery admet dos tipus de UDFs:
    • SQL UDFs: Escrites en SQL.
    • JavaScript UDFs: Escrites en JavaScript.
  • Sintaxi: La sintaxi per crear UDFs varia lleugerament entre SQL i JavaScript.
  • Reutilització: Les UDFs es poden reutilitzar en diverses consultes, fent que el teu codi sigui més modular i mantenible.

Creació de SQL UDFs

Les SQL UDFs es redacten utilitzant sintaxi SQL estàndard i es poden utilitzar per encapsular lògica SQL.

Exemple: Simple SQL UDF

Creem una simple SQL UDF que calcula el quadrat d'un número.

CREATE OR REPLACE FUNCTION my_dataset.square(x FLOAT64) 
RETURNS FLOAT64 AS (
  x * x
);

Explicació

  • CREATE OR REPLACE FUNCTION: Aquesta instrucció crea una nova funció o reemplaça una existent.
  • my_dataset.square: El nom de la funció, on my_dataset és el conjunt de dades que conté la funció.
  • x FLOAT64: El paràmetre d'entrada i el seu tipus.
  • RETURNS FLOAT64: El tipus de retorn de la funció.
  • AS ( x * x ): El cos de la funció, que conté l'expressió SQL a executar.

Ús de la SQL UDF

Pots utilitzar la UDF en una consulta com qualsevol altra funció.

SELECT my_dataset.square(3) AS result;

Resultat

| result | |--------| | 9 |

Creació de JavaScript UDFs

Les JavaScript UDFs et permeten escriure funcions personalitzades utilitzant JavaScript, cosa que pot ser útil per a lògica més complexa que és difícil d'expressar en SQL.

Exemple: Simple JavaScript UDF

Creem una JavaScript UDF que inverteix una cadena.

CREATE OR REPLACE FUNCTION my_dataset.reverse_string(input STRING) 
RETURNS STRING 
LANGUAGE js AS """
  return input.split('').reverse().join('');
""";

Explicació

  • CREATE OR REPLACE FUNCTION: Aquesta instrucció crea una nova funció o reemplaça una existent.
  • my_dataset.reverse_string: El nom de la funció.
  • input STRING: El paràmetre d'entrada i el seu tipus.
  • RETURNS STRING: El tipus de retorn de la funció.
  • LANGUAGE js: Especifica que la funció està escrita en JavaScript.
  • AS """ ... """: El cos de la funció, escrit en JavaScript.

Ús de la JavaScript UDF

Pots utilitzar la UDF en una consulta com qualsevol altra funció.

SELECT my_dataset.reverse_string('BigQuery') AS reversed;

Resultat

| reversed | |----------| | yreuQgiB |

Ús Avançat de les UDFs

Gestió de Múltiples Paràmetres

Les UDFs poden acceptar múltiples paràmetres. Aquí tens un exemple d'una SQL UDF que concatena dues cadenes.

CREATE OR REPLACE FUNCTION my_dataset.concat_strings(str1 STRING, str2 STRING) 
RETURNS STRING AS (
  CONCAT(str1, ' ', str2)
);

Ús de UDFs amb Arrays

També pots crear UDFs que treballen amb arrays. Aquí tens un exemple d'una JavaScript UDF que suma els elements d'un array.

CREATE OR REPLACE FUNCTION my_dataset.sum_array(arr ARRAY<INT64>) 
RETURNS INT64 
LANGUAGE js AS """
  return arr.reduce((a, b) => a + b, 0);
""";

Consideracions de Rendiment

  • Temps d'Execució: Les UDFs poden introduir temps d'execució addicional, especialment si contenen lògica complexa.
  • Ús de Recursos: Tingues en compte els recursos que consumeixen les teves UDFs, ja que poden afectar el rendiment global de les teves consultes.

Conclusió

Les Funcions Definides per l'Usuari (UDFs) a BigQuery proporcionen una manera poderosa d'ampliar les capacitats de SQL amb lògica personalitzada. Tant si utilitzes SQL com JavaScript, les UDFs poden ajudar-te a encapsular lògica complexa, realitzar càlculs personalitzats i fer que les teves consultes siguin més modulars i mantenibles. Comprenent com crear i utilitzar UDFs de manera efectiva, pots aprofitar tot el potencial de BigQuery per a les teves tasques d'anàlisi de dades.
