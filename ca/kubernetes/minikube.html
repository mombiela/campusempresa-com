Estudi de Cas 1: Migració a Kubernetes ►

Minikube és una eina que facilita l'execució de Kubernetes localment. Executa un clúster de Kubernetes d'un sol node dins d'una màquina virtual (VM) al teu ordinador portàtil. Això és especialment útil per a desenvolupament, proves i aprenentatge.

Conceptes Clau

  • Clúster de Kubernetes d'un Sol Node: Minikube executa un clúster de Kubernetes d'un sol node, el que significa que inclou tant les funcionalitats del node mestre com del node treballador en una sola VM.
  • Màquina Virtual: Minikube utilitza una VM per executar el clúster de Kubernetes. Suporta diversos controladors de VM com VirtualBox, VMware, Hyper-V i Docker.
  • Desenvolupament Local: Minikube està dissenyat per al desenvolupament i proves locals, facilitant l'experimentació amb Kubernetes sense necessitat d'un clúster a gran escala.

Instal·lació

Prerequisits

Abans d'instal·lar Minikube, assegura't de tenir el següent:

  • Un hipervisor instal·lat (per exemple, VirtualBox, VMware, Hyper-V o Docker).
  • kubectl instal·lat. kubectl és l'eina de línia de comandes per interactuar amb clústers de Kubernetes.

Instal·lant Minikube

  1. Descarregar Minikube:

    • Per a macOS: 
      brew install minikube
    • Per a Linux: 
      curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
sudo install minikube-linux-amd64 /usr/local/bin/minikube
    • Per a Windows: Descarrega l'instal·lador des de la pàgina de llançaments de Minikube.

  2. Iniciar Minikube:

    minikube start

    Aquesta comanda inicia un clúster local de Kubernetes. Pots especificar el controlador de VM utilitzant el flag --driver, per exemple:

    minikube start --driver=virtualbox

Ús Bàsic

Interactuant amb Minikube

  • Comprovar l'Estat del Clúster:

    minikube status

  • Accedir al Tauler de Kubernetes: Minikube proporciona un tauler de Kubernetes integrat per gestionar el clúster.

    minikube dashboard

Desplegant Aplicacions

  1. Crear un Desplegament:

    kubectl create deployment hello-minikube --image=k8s.gcr.io/echoserver:1.4

  2. Exposar el Desplegament:

    kubectl expose deployment hello-minikube --type=NodePort --port=8080

  3. Accedir a l'Aplicació: Obtenir l'URL per accedir a l'aplicació:

    minikube service hello-minikube --url

Gestionant Minikube

  • Aturar Minikube:

    minikube stop

  • Eliminar el Clúster de Minikube:

    minikube delete

Funcionalitats Avançades

Configurant Recursos

Pots configurar la quantitat de CPU i memòria assignada a la VM de Minikube:

minikube start --cpus=4 --memory=8192

Complements

Minikube suporta diversos complements per ampliar la seva funcionalitat. Alguns complements populars inclouen:

  • Ingress:

    minikube addons enable ingress

  • Metrics Server:

    minikube addons enable metrics-server

Emmagatzematge Persistent

Minikube suporta emmagatzematge persistent, permetent-te crear recursos PersistentVolume i PersistentVolumeClaim.

  1. Crear un PersistentVolume:

    apiVersion: v1
kind: PersistentVolume
metadata:
  name: pv0001
spec:
  capacity:
    storage: 1Gi
  accessModes:
    - ReadWriteOnce
  hostPath:
    path: "/mnt/data"

  2. Crear un PersistentVolumeClaim:

    apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: pvc0001
spec:
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 1Gi

  3. Aplicar les configuracions:

    kubectl apply -f pv.yaml
kubectl apply -f pvc.yaml

Conclusió

Minikube és una excel·lent eina per a desenvolupadors i aprenents per obtenir experiència pràctica amb Kubernetes. Simplifica el procés de configuració d'un entorn local de Kubernetes, facilitant el desenvolupament, les proves i l'experimentació amb les funcionalitats de Kubernetes. Entenent com instal·lar, configurar i utilitzar Minikube, pots simular efectivament un clúster de Kubernetes a la teva màquina local i obtenir valuoses perspectives sobre les operacions de Kubernetes.
