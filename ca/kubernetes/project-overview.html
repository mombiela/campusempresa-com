Introducció a Kubernetes

Kubernetes, sovint abreujat com K8s, és una plataforma de codi obert dissenyada per automatitzar el desplegament, l'escalat i l'operació de contenidors d'aplicacions. Agrupa els contenidors que formen una aplicació en unitats lògiques per a una gestió i descobriment fàcils. Aquest curs et guiarà des dels conceptes bàsics de Kubernetes fins als conceptes avançats, assegurant que tinguis una comprensió completa de la plataforma.

Conceptes Clau

  • Contenidors: Paquets lleugers, autònoms i executables que inclouen tot el necessari per executar un tros de programari.
  • Pods: L'objecte més petit i senzill de Kubernetes. Un pod representa un conjunt de contenidors en execució al teu clúster.
  • Nodes: Màquines treballadores a Kubernetes, que poden ser virtuals o físiques.
  • Clústers: Un conjunt de nodes que executen aplicacions contenidoritzades gestionades per Kubernetes.
  • Serveis: Una manera abstracta d'exposar una aplicació que s'executa en un conjunt de Pods com un servei de xarxa.
  • Desplegaments: Un controlador que proporciona actualitzacions declaratives a les aplicacions.

Estructura del Curs

Nivell Principiant

Comprensió dels Contenidors i Conceptes Bàsics de Kubernetes

  • Introducció als Contenidors:

    • Explicació de Docker i la contenidorització.
    • Comandes bàsiques de Docker per crear i gestionar contenidors.
    • Exemple: 
      # Descarregar una imatge de Docker
docker pull nginx

# Executar un contenidor de Docker
docker run -d -p 80:80 nginx

  • Arquitectura de Kubernetes:

    • Visió general dels components de Kubernetes: Node Mestre, Nodes Treballadors, etc.
    • Explicació de Pods, Nodes i Clústers.
    • Exemple: 
      # Definició simple d'un Pod
apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
  containers:
  - name: nginx
    image: nginx:latest

Nivell Intermedi

Gestió de Recursos de Kubernetes

  • Desplegaments i Serveis:

    • Creació i gestió de Desplegaments.
    • Exposició de Desplegaments utilitzant Serveis.
    • Exemple: 
      # Definició de Desplegament
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:latest

# Definició de Servei
apiVersion: v1
kind: Service
metadata:
  name: nginx-service
spec:
  selector:
    app: nginx
  ports:
  - protocol: TCP
    port: 80
    targetPort: 80

  • ConfigMaps i Secrets:

    • Gestió de dades de configuració amb ConfigMaps.
    • Emmagatzematge d'informació sensible amb Secrets.
    • Exemple: 
      # Definició de ConfigMap
apiVersion: v1
kind: ConfigMap
metadata:
  name: example-config
data:
  key1: value1
  key2: value2

# Definició de Secret
apiVersion: v1
kind: Secret
metadata:
  name: example-secret
type: Opaque
data:
  username: YWRtaW4=  # codificat en base64
  password: MWYyZDFlMmU2N2Rm  # codificat en base64

Nivell Avançat

Funcionalitats Avançades de Kubernetes

  • StatefulSets i DaemonSets:

    • Gestió d'aplicacions amb estat amb StatefulSets.
    • Execució d'una còpia d'un Pod en tots els nodes amb DaemonSets.
    • Exemple: 
      # Definició de StatefulSet
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: web
spec:
  serviceName: "nginx"
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:latest

# Definició de DaemonSet
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: fluentd
spec:
  selector:
    matchLabels:
      name: fluentd
  template:
    metadata:
      labels:
        name: fluentd
    spec:
      containers:
      - name: fluentd
        image: fluent/fluentd:latest

  • Helm Charts:

    • Introducció a Helm i els seus beneficis.
    • Creació i desplegament d'aplicacions utilitzant Helm charts.
    • Exemple: 
      # Instal·lació d'un Helm chart
helm install my-nginx stable/nginx

Conclusió

Al final d'aquest curs, tindràs una sòlida comprensió de Kubernetes, des dels conceptes bàsics fins a les funcionalitats avançades. Seràs capaç de desplegar, gestionar i escalar aplicacions contenidoritzades de manera eficient utilitzant Kubernetes. Aquest coneixement et permetrà afrontar escenaris i desafiaments reals en la gestió d'aplicacions contenidoritzades.
