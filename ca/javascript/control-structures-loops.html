Introducció als Bucles

Els bucles són estructures de control fonamentals en JavaScript que et permeten executar un bloc de codi múltiples vegades. Són essencials per a tasques que requereixen repetició, com iterar sobre arrays o realitzar accions fins que es compleixi una certa condició.

Tipus de Bucles en JavaScript

JavaScript proporciona diversos tipus de bucles per manejar diferents escenaris:

  • Bucle for
  • Bucle while
  • Bucle do...while
  • Bucle for...in
  • Bucle for...of

Bucle `for`

El bucle for és el bucle més utilitzat en JavaScript. És ideal per a escenaris on saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi:

for (inicialització; condició; increment) {
  // bloc de codi a executar
}

Exemple:

for (let i = 0; i < 5; i++) {
  console.log("Número d'iteració: " + i);
}

Explicació:

  • Inicialització: let i = 0 estableix el punt de partida.
  • Condició: i < 5 comprova si el bucle ha de continuar.
  • Increment: i++ actualitza el comptador després de cada iteració.

Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix per endavant. Continua executant-se mentre la condició especificada sigui true.

Sintaxi:

while (condició) {
  // bloc de codi a executar
}

Exemple:

let i = 0;
while (i < 5) {
  console.log("Número d'iteració: " + i);
  i++;
}

Explicació:

  • El bucle continuarà executant-se mentre i < 5 sigui true.
  • i++ incrementa el comptador per acabar finalment el bucle.

Bucle `do...while`

El bucle do...while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada abans de comprovar la condició.

Sintaxi:

do {
  // bloc de codi a executar
} while (condició);

Exemple:

let i = 0;
do {
  console.log("Número d'iteració: " + i);
  i++;
} while (i < 5);

Explicació:

  • El bloc de codi dins de la instrucció do s'executa primer.
  • La condició i < 5 es comprova després de la primera execució.

Bucle `for...in`

El bucle for...in s'utilitza per iterar sobre les propietats d'un objecte.

Sintaxi:

for (clau in objecte) {
  // bloc de codi a executar
}

Exemple:

const person = {fname: "John", lname: "Doe", age: 25};
for (let clau in person) {
  console.log(clau + ": " + person[clau]);
}

Explicació:

  • El bucle itera sobre cada propietat de l'objecte person.
  • clau representa el nom de la propietat, i person[clau] accedeix al valor de la propietat.

Bucle `for...of`

El bucle for...of s'utilitza per iterar sobre objectes iterables com arrays, cadenes de text, mapes i conjunts.

Sintaxi:

for (variable of iterable) {
  // bloc de codi a executar
}

Exemple:

const numbers = [1, 2, 3, 4, 5];
for (let number of numbers) {
  console.log(number);
}

Explicació:

  • El bucle itera sobre cada element de l'array numbers.
  • number representa l'element actual en cada iteració.

Comparació d'Estructures de Bucle

| Tipus de Bucle | Cas d'Ús | Garanteix l'Execució Almenys una Vegada | |----------------|-----------------------------------------------|----------------------------------------| | for | Nombre conegut d'iteracions | No | | while | Nombre desconegut d'iteracions | No | | do...while | Nombre desconegut d'iteracions | Sí | | for...in | Iterar sobre propietats d'objectes | No | | for...of | Iterar sobre objectes iterables (arrays, etc.)| No |

Exercicis

  1. Bucle for Bàsic: Escriu un bucle for que imprimeixi els números de l'1 al 10.

    for (let i = 1; i <= 10; i++) {
  console.log(i);
}

  2. Bucle while: Escriu un bucle while que imprimeixi els números del 10 a l'1.

    let i = 10;
while (i > 0) {
  console.log(i);
  i--;
}

  3. Bucle do...while: Escriu un bucle do...while que imprimeixi els números de l'1 al 5.

    let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);

  4. Bucle for...in: Escriu un bucle for...in per imprimir totes les propietats i valors del següent objecte:

    const car = {make: "Toyota", model: "Corolla", year: 2020};
for (let clau in car) {
  console.log(clau + ": " + car[clau]);
}

  5. Bucle for...of: Escriu un bucle for...of per imprimir cada caràcter de la cadena de text "JavaScript".

    const str = "JavaScript";
for (let char of str) {
  console.log(char);
}

Conclusió

Els bucles són eines poderoses en JavaScript que et permeten realitzar tasques repetitives de manera eficient. Entendre els diferents tipus de bucles i els seus casos d'ús et permetrà escriure codi més efectiu i concís. Practica utilitzant cada tipus de bucle per familiaritzar-te amb la seva sintaxi i funcionalitat.
