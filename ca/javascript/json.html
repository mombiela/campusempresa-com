Introducció al JSON

JSON (JavaScript Object Notation) és un format lleuger d'intercanvi de dades que és fàcil de llegir i escriure per als humans, i fàcil de parsejar i generar per a les màquines. S'utilitza comunament per transmetre dades en aplicacions web.

  • Conceptes Clau:
    • JSON és text, escrit amb notació d'objectes de JavaScript.
    • JSON és independent del llenguatge, però utilitza convencions que són familiars per als programadors de la família de llenguatges C, incloent C, C++, C#, Java, JavaScript, Perl, Python, i molts altres.

Sintaxi JSON

La sintaxi JSON es deriva de la sintaxi de notació d'objectes de JavaScript:

  • Les dades estan en parells nom/valor:

    {
  "name": "John",
  "age": 30
}

  • Les dades estan separades per comes:

    {
  "name": "John",
  "age": 30,
  "city": "New York"
}

  • Les claus contenen objectes:

    {
  "name": "John",
  "age": 30,
  "city": "New York"
}

  • Els claudàtors contenen arrays:

    {
  "employees": [
    { "firstName": "John", "lastName": "Doe" },
    { "firstName": "Anna", "lastName": "Smith" },
    { "firstName": "Peter", "lastName": "Jones" }
  ]
}

JSON vs Objecte JavaScript

Tot i que JSON i els objectes JavaScript semblen similars, hi ha diferències clau:

| Característica | JSON | Objecte JavaScript | |------------------|-------------------------------------------|------------------------------------------| | Format de Dades | Format de text | Format d'objecte | | Noms de Clau | Han de ser cadenes entre cometes dobles | Poden ser cadenes, símbols o números | | Valors | Cadenes, números, arrays, objectes, booleans, null | Qualsevol expressió JavaScript vàlida | | Ús | Intercanvi de dades | Lògica de programació i manipulació de dades |

Parsejar JSON

Per parsejar dades JSON en JavaScript, utilitza JSON.parse():

let jsonString = '{"name": "John", "age": 30, "city": "New York"}';
let jsonObject = JSON.parse(jsonString);

console.log(jsonObject.name); // Sortida: John
console.log(jsonObject.age);  // Sortida: 30
console.log(jsonObject.city); // Sortida: New York

Convertir a JSON

Per convertir un objecte JavaScript en una cadena JSON, utilitza JSON.stringify():

let jsonObject = { name: "John", age: 30, city: "New York" };
let jsonString = JSON.stringify(jsonObject);

console.log(jsonString); // Sortida: {"name":"John","age":30,"city":"New York"}

Treballar amb JSON en Aplicacions Web

JSON s'utilitza sovint per intercanviar dades entre un servidor web i un client. Aquí tens un exemple utilitzant l'API Fetch per obtenir dades JSON d'un servidor:

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error('Error:', error);
  });

Tècniques Avançades de JSON

Objectes JSON Niats

JSON pot contenir objectes i arrays niats:

{
  "person": {
    "name": "John",
    "age": 30,
    "address": {
      "street": "123 Main St",
      "city": "New York"
    }
  }
}

JSON amb Funcions

Tot i que JSON en si no pot contenir funcions, pots convertir objectes amb funcions a JSON i de tornada, però les funcions es perdran:

let obj = {
  name: "John",
  greet: function() { return "Hello " + this.name; }
};

let jsonString = JSON.stringify(obj);
console.log(jsonString); // Sortida: {"name":"John"}

let parsedObj = JSON.parse(jsonString);
console.log(parsedObj.greet); // Sortida: undefined

Gestionar Dates en JSON

Les dates no són un tipus de dada estàndard en JSON, però pots convertir dates a cadenes i de tornada:

let obj = { name: "John", birthdate: new Date() };
let jsonString = JSON.stringify(obj);

console.log(jsonString); // Sortida: {"name":"John","birthdate":"2023-10-01T12:00:00.000Z"}

let parsedObj = JSON.parse(jsonString);
parsedObj.birthdate = new Date(parsedObj.birthdate);

console.log(parsedObj.birthdate); // Sortida: 2023-10-01T12:00:00.000Z

Conclusió

JSON és un format potent i flexible per a l'intercanvi de dades en aplicacions web. Entendre com parsejar i convertir a JSON, així com gestionar escenaris més avançats, és crucial per al desenvolupament modern en JavaScript. Dominant JSON, pots gestionar efectivament l'intercanvi de dades entre les teves aplicacions del costat del client i del servidor.
