JavaScript és un llenguatge de programació versàtil i potent utilitzat principalment per al desenvolupament web. Permet als desenvolupadors crear pàgines web dinàmiques i interactives manipulant el Document Object Model (DOM), gestionant esdeveniments i comunicant-se amb servidors.

Conceptes Clau de JavaScript

  • Scripting del Costat del Client: JavaScript s'executa en el navegador de l'usuari, fent-lo ideal per crear pàgines web interactives.
  • Llenguatge Interpretat: A diferència dels llenguatges compilats, el codi JavaScript s'executa línia per línia pel navegador.
  • Basat en Esdeveniments: JavaScript pot respondre a accions de l'usuari com clics, enviaments de formularis i esdeveniments del teclat.
  • Orientat a Objectes: JavaScript suporta principis de programació orientada a objectes, permetent la creació de codi reutilitzable.
  • Multiplataforma: JavaScript s'executa en qualsevol dispositiu amb un navegador web, fent-lo altament versàtil.

Sintaxi i Estructura Bàsica

La sintaxi de JavaScript és el conjunt de regles que defineixen un programa JavaScript estructurat correctament. Aquí hi ha alguns elements bàsics:

Variables

Les variables s'utilitzen per emmagatzemar valors de dades. En JavaScript, pots declarar variables utilitzant var, let o const.

var name = "John"; // Declara una variable anomenada 'name' amb el valor "John"
let age = 30;      // Declara una variable de bloc anomenada 'age' amb el valor 30
const pi = 3.14;   // Declara una constant anomenada 'pi' amb el valor 3.14

Tipus de Dades

JavaScript suporta diversos tipus de dades, incloent:

  • String: Dades textuals, per exemple, "Hello, World!"
  • Number: Dades numèriques, per exemple, 42 o 3.14
  • Boolean: Dades lògiques, per exemple, true o false
  • Array: Col·lecció ordenada de valors, per exemple, [1, 2, 3]
  • Object: Col·lecció de parells clau-valor, per exemple, {name: "John", age: 30}

Funcions

Les funcions són blocs de codi dissenyats per realitzar una tasca particular. S'executen quan són "cridades" o "invocades".

function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("Alice")); // Mostra: Hello, Alice!

Estructures de Control

JavaScript inclou estructures de control com condicionals i bucles per controlar el flux del programa.

Condicionals

let score = 85;

if (score >= 90) {
    console.log("A");
} else if (score >= 80) {
    console.log("B");
} else {
    console.log("C");
}

Bucles

for (let i = 0; i < 5; i++) {
    console.log(i); // Mostra: 0, 1, 2, 3, 4
}

Comparació de `var`, `let` i `const`

| Característica | var | let | const | |------------------|-------------------------|-------------------------|-------------------------| | Àmbit | Àmbit de funció | Àmbit de bloc | Àmbit de bloc | | Reassignació | Permesa | Permesa | No permesa | | Hoisting | Sí, inicialitzat com undefined | Sí, però no inicialitzat | Sí, però no inicialitzat |

Conclusió

JavaScript és un llenguatge essencial per al desenvolupament web, permetent la creació d'experiències d'usuari dinàmiques i interactives. Entendre la seva sintaxi bàsica, tipus de dades i estructures de control és el primer pas per dominar JavaScript. A mesura que avancis en aquest curs, aprofundiràs en temes més avançats, construint una base sòlida en la programació amb JavaScript.
