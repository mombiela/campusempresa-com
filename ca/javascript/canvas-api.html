L'API Canvas és una característica potent en HTML5 que et permet dibuixar gràfics utilitzant JavaScript. Aquest tema cobrirà els conceptes bàsics de l'API Canvas, progressant cap a tècniques més avançades.

Introducció a l'API Canvas

  • Què és l'API Canvas?

    • Una API de JavaScript per dibuixar gràfics 2D.
    • Permet el renderitzat dinàmic i scriptable de formes i imatges.

  • Configuració del Canvas

    • Element HTML <canvas>.
    • Accés al context del canvas en JavaScript.

Exemple: Configuració bàsica del Canvas

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Canvas</title>
</head>
<body>
    <canvas id="myCanvas" width="500" height="500" style="border:1px solid #000000;"></canvas>
    <script>
        const canvas = document.getElementById('myCanvas');
        const ctx = canvas.getContext('2d');
    </script>
</body>
</html>

Dibuix de formes

  • Rectangles
    • fillRect(x, y, width, height)
    • strokeRect(x, y, width, height)
    • clearRect(x, y, width, height)

Exemple: Dibuix de rectangles

ctx.fillStyle = 'green';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'blue';
ctx.strokeRect(130, 10, 100, 100);

ctx.clearRect(50, 50, 60, 60);
  • Camins
    • beginPath()
    • moveTo(x, y)
    • lineTo(x, y)
    • closePath()
    • stroke()
    • fill()

Exemple: Dibuix de camins

ctx.beginPath();
ctx.moveTo(50, 50);
ctx.lineTo(200, 50);
ctx.lineTo(200, 200);
ctx.closePath();
ctx.stroke();

Treballant amb colors i estils

  • Estils de farciment i traç
    • fillStyle
    • strokeStyle
    • globalAlpha

Exemple: Ús de colors i estils

ctx.fillStyle = 'rgba(255, 0, 0, 0.5)';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'rgba(0, 0, 255, 0.5)';
ctx.strokeRect(130, 10, 100, 100);

Dibuix de text

  • Mètodes de text
    • fillText(text, x, y)
    • strokeText(text, x, y)
    • font
    • textAlign
    • textBaseline

Exemple: Dibuix de text

ctx.font = '30px Arial';
ctx.fillText('Hola Canvas', 10, 50);

ctx.strokeText('Hola Canvas', 10, 100);

Transformacions

  • Escalat

    • scale(x, y)

  • Rotació

    • rotate(angle)

  • Translació

    • translate(x, y)

Exemple: Aplicació de transformacions

ctx.translate(100, 100);
ctx.rotate(Math.PI / 4);
ctx.fillStyle = 'purple';
ctx.fillRect(0, 0, 100, 100);

Tècniques avançades

  • Gradients
    • createLinearGradient(x0, y0, x1, y1)
    • createRadialGradient(x0, y0, r0, x1, y1, r1)

Exemple: Ús de gradients

const gradient = ctx.createLinearGradient(0, 0, 200, 0);
gradient.addColorStop(0, 'red');
gradient.addColorStop(1, 'blue');

ctx.fillStyle = gradient;
ctx.fillRect(10, 10, 200, 100);
  • Patrons
    • createPattern(image, repetition)

Exemple: Ús de patrons

const img = new Image();
img.src = 'path/to/image.jpg';
img.onload = function() {
    const pattern = ctx.createPattern(img, 'repeat');
    ctx.fillStyle = pattern;
    ctx.fillRect(0, 0, 500, 500);
};

Animacions

  • Bucle d'animació bàsic
    • requestAnimationFrame(callback)

Exemple: Animació bàsica

let x = 0;
function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    ctx.fillRect(x, 50, 50, 50);
    x += 1;
    requestAnimationFrame(draw);
}
draw();

Conclusió

L'API Canvas és una eina versàtil per crear gràfics dinàmics en aplicacions web. Entenent els conceptes bàsics de dibuixar formes, treballar amb colors, aplicar transformacions i crear animacions, pots construir visualitzacions complexes i interactives. Practica aquestes tècniques per convertir-te en un expert en l'ús de l'API Canvas per als teus projectes.
