Introducció als Objectes

Els objectes són una part fonamental de JavaScript i s'utilitzen per emmagatzemar col·leccions de dades i entitats més complexes. Un objecte és una col·lecció de propietats, i una propietat és una associació entre un nom (o clau) i un valor.

Conceptes Clau

  • Literals d'Objecte: La manera més senzilla de crear un objecte.
  • Propietats: Parells clau-valor dins d'un objecte.
  • Mètodes: Funcions emmagatzemades com a propietats d'objecte.
  • Objectes Imbricats: Objectes dins d'objectes.
  • Accés a Propietats: Notació de punt i notació de claudàtors.
  • Modificació d'Objectes: Afegir, actualitzar i eliminar propietats.

Creació d'Objectes

Literals d'Objecte

Un literal d'objecte és una llista separada per comes de parells clau-valor envoltada per claus.

let person = {
    name: "John",
    age: 30,
    isStudent: false
};

Utilitzant la Sintaxi `new Object()`

Una altra manera de crear un objecte és utilitzant la sintaxi new Object().

let person = new Object();
person.name = "John";
person.age = 30;
person.isStudent = false;

Propietats i Mètodes

Accés a Propietats

Pots accedir a les propietats d'un objecte utilitzant la notació de punt o la notació de claudàtors.

// Notació de punt
console.log(person.name); // Sortida: John

// Notació de claudàtors
console.log(person['age']); // Sortida: 30

Afegir i Modificar Propietats

Pots afegir o modificar propietats utilitzant la notació de punt o la notació de claudàtors.

person.email = "john@example.com"; // Afegint una nova propietat
person.age = 31; // Modificant una propietat existent

Eliminar Propietats

Pots eliminar propietats utilitzant l'operador delete.

delete person.isStudent;

Mètodes

Els mètodes són funcions emmagatzemades com a propietats d'objecte.

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

person.greet(); // Sortida: Hello, my name is John

Objectes Imbricats

Els objectes poden contenir altres objectes com a propietats.

let person = {
    name: "John",
    age: 30,
    address: {
        street: "123 Main St",
        city: "New York"
    }
};

console.log(person.address.city); // Sortida: New York

Conceptes Avançats d'Objectes

Desestructuració d'Objectes

La desestructuració permet desempaquetar valors d'objectes en variables distintes.

let person = {
    name: "John",
    age: 30,
    email: "john@example.com"
};

let { name, age } = person;
console.log(name); // Sortida: John
console.log(age); // Sortida: 30

Mètodes d'Objecte: `Object.keys()`, `Object.values()`, `Object.entries()`

Aquests mètodes s'utilitzen per treballar amb les propietats dels objectes.

let person = {
    name: "John",
    age: 30,
    email: "john@example.com"
};

console.log(Object.keys(person)); // Sortida: ["name", "age", "email"]
console.log(Object.values(person)); // Sortida: ["John", 30, "john@example.com"]
console.log(Object.entries(person)); // Sortida: [["name", "John"], ["age", 30], ["email", "john@example.com"]]

Paraula Clau `this`

La paraula clau this es refereix a l'objecte al qual pertany.

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

person.greet(); // Sortida: Hello, my name is John

Conclusió

Els objectes són una part versàtil i essencial de JavaScript, permetent-te emmagatzemar i manipular dades de manera eficient. Des de la creació de literals d'objecte simples fins a l'ús de mètodes avançats i la desestructuració, entendre els objectes millorarà significativament la teva capacitat per escriure codi JavaScript robust i mantenible. Practica creant, modificant i accedint a objectes per convertir-te en un expert en l'ús d'aquests en els teus projectes.
