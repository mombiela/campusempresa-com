Introducció

JavaScript és un llenguatge de programació versàtil i potent utilitzat principalment per al desenvolupament web. Aquesta secció cobrirà la sintaxi fonamental i els conceptes bàsics que són essencials per a qualsevol persona que comenci amb JavaScript.

Conceptes Clau

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden referenciar i manipular en un programa. JavaScript utilitza tres paraules clau per declarar variables:

  • var
  • let
  • const

Exemple:

var name = "John";
let age = 30;
const isStudent = true;

Tipus de Dades

JavaScript admet diversos tipus de dades, incloent:

  • String: Dades textuals
  • Number: Dades numèriques
  • Boolean: Valors veritable o fals
  • Array: Llista ordenada de valors
  • Object: Col·lecció de parelles clau-valor
  • Null: Representa l'absència intencional de qualsevol valor d'objecte
  • Undefined: Variable que ha estat declarada però no assignada un valor

Exemple:

let message = "Hello, World!"; // String
let count = 42; // Number
let isActive = false; // Boolean
let fruits = ["Apple", "Banana", "Cherry"]; // Array
let person = { firstName: "John", lastName: "Doe" }; // Object
let emptyValue = null; // Null
let notAssigned; // Undefined

Operadors

Els operadors s'utilitzen per realitzar operacions sobre variables i valors. Els operadors comuns inclouen:

  • Operadors Aritmètics: +, -, *, /, %
  • Operadors d'Assignació: =, +=, -=, *=, /=
  • Operadors de Comparació: ==, ===, !=, !==, >, <, >=, <=
  • Operadors Lògics: &&, ||, !

Exemple:

let a = 10;
let b = 20;
let sum = a + b; // Aritmètic
a += 5; // Assignació
let isEqual = (a == b); // Comparació
let isBothTrue = (a > 5 && b < 30); // Lògic

Estructures de Control

Les estructures de control s'utilitzen per dictar el flux del programa. Les estructures de control comunes inclouen:

  • Declaracions Condicionals: if, else if, else
  • Declaracions Switch: switch
  • Bucles: for, while, do...while

Exemple:

// Declaració Condicional
if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// Declaració Switch
let day = 3;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    case 3:
        console.log("Wednesday");
        break;
    default:
        console.log("Invalid day");
}

// Bucle For
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Bucle While
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Bucle Do...While
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);

Funcions

Les funcions són blocs de codi dissenyats per realitzar una tasca particular. S'executen quan són cridades.

Exemple:

// Declaració de Funció
function greet(name) {
    return "Hello, " + name + "!";
}

// Expressió de Funció
const greet = function(name) {
    return "Hello, " + name + "!";
};

// Funció Fletxa (ES6)
const greet = (name) => {
    return "Hello, " + name + "!";
};

// Cridar una Funció
console.log(greet("Alice"));

Conclusió

Entendre els conceptes bàsics de la sintaxi de JavaScript és crucial per construir una base sòlida en programació. Aquesta secció ha cobert variables, tipus de dades, operadors, estructures de control i funcions. Dominar aquests conceptes et permetrà escriure codi JavaScript senzill però efectiu i et prepararà per a temes més avançats. Continua practicant i experimentant amb aquests conceptes bàsics per sentir-te més còmode amb JavaScript.
