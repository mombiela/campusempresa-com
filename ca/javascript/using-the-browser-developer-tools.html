Introducció

Les eines de desenvolupador del navegador són essencials per als desenvolupadors de JavaScript. T'ajuden a inspeccionar, depurar i optimitzar el teu codi directament al navegador. Aquesta secció et guiarà des de les funcions bàsiques fins a les avançades d'aquestes eines.

Accés a les eines de desenvolupador

  • Google Chrome: Fes clic amb el botó dret en qualsevol pàgina web i selecciona "Inspecciona" o prem Ctrl+Shift+I (Windows/Linux) o Cmd+Opt+I (Mac).
  • Mozilla Firefox: Fes clic amb el botó dret en qualsevol pàgina web i selecciona "Inspecciona l'element" o prem Ctrl+Shift+I (Windows/Linux) o Cmd+Opt+I (Mac).
  • Microsoft Edge: Fes clic amb el botó dret en qualsevol pàgina web i selecciona "Inspecciona" o prem Ctrl+Shift+I (Windows/Linux) o Cmd+Opt+I (Mac).

Panells clau en les eines de desenvolupador

Panell d'elements

  • Propòsit: Inspeccionar i modificar l'HTML i el CSS d'una pàgina web.
  • Ús:
    • Inspeccionar elements: Passa el cursor sobre els elements per veure el seu HTML i CSS.
    • Editar HTML/CSS: Fes doble clic en els elements o estils per editar-los.
    • Afegir nous estils: Utilitza el panell "Estils" per afegir noves regles CSS.

Panell de consola

  • Propòsit: Executar codi JavaScript i veure registres, errors i advertències.
  • Ús:
    • Registre de missatges: Utilitza console.log(), console.error() i console.warn() per registrar missatges.
    • Executar codi: Escriu codi JavaScript directament a la consola i prem Enter per executar-lo.
    • Veure errors: Comprova errors i advertències de JavaScript.
console.log("Hola, món!"); // Registra un missatge a la consola
console.error("Això és un error!"); // Registra un missatge d'error
console.warn("Això és una advertència!"); // Registra un missatge d'advertència

Panell de fonts

  • Propòsit: Veure i depurar fitxers JavaScript.
  • Ús:
    • Veure fitxers de fonts: Navega i visualitza tots els fitxers JavaScript carregats per la pàgina web.
    • Establir punts de ruptura: Fes clic al número de línia per establir punts de ruptura per a la depuració.
    • Passar pel codi: Utilitza els botons de reproducció, pas per sobre, pas a dins i pas a fora per controlar l'execució del codi.

Panell de xarxa

  • Propòsit: Monitoritzar les sol·licituds i respostes de xarxa.
  • Ús:
    • Veure sol·licituds: Veure totes les sol·licituds de xarxa fetes per la pàgina web.
    • Inspeccionar respostes: Fes clic en una sol·licitud per veure els seus encapçalaments, resposta i temps.
    • Analitzar el rendiment: Comprova el temps de càrrega i la mida dels recursos.

Panell de rendiment

  • Propòsit: Analitzar el rendiment de la teva pàgina web.
  • Ús:
    • Registrar el rendiment: Fes clic al botó de registre per començar a registrar les mètriques de rendiment.
    • Analitzar la línia de temps: Visualitza la línia de temps per veure quant triguen diferents tasques.
    • Identificar colls d'ampolla: Busca tasques llargues o desplaçaments de disseny que alentegen la teva pàgina.

Panell d'aplicació

  • Propòsit: Inspeccionar i gestionar l'emmagatzematge, els treballadors de servei i altres recursos de l'aplicació web.
  • Ús:
    • Veure emmagatzematge: Inspecciona l'emmagatzematge local, l'emmagatzematge de sessió, les galetes i IndexedDB.
    • Gestionar treballadors de servei: Visualitza i controla els treballadors de servei.
    • Inspeccionar el manifest: Comprova el manifest de l'aplicació web per a aplicacions web progressives (PWAs).

Exemples pràctics

Exemple 1: Depurar JavaScript amb punts de ruptura

  1. Obre les eines de desenvolupador i ves al panell de fonts.
  2. Obre un fitxer JavaScript.
  3. Fes clic al número de línia per establir un punt de ruptura.
  4. Recarrega la pàgina i l'execució es pararà al punt de ruptura.
  5. Utilitza els botons de pas per navegar pel codi.
function add(a, b) {
    return a + b;
}

let result = add(2, 3);
console.log(result); // Estableix un punt de ruptura aquí

Exemple 2: Inspeccionar sol·licituds de xarxa

  1. Obre les eines de desenvolupador i ves al panell de xarxa.
  2. Recarrega la pàgina per veure totes les sol·licituds de xarxa.
  3. Fes clic en una sol·licitud per veure els seus detalls, com ara encapçalaments i resposta.
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Conclusió

Les eines de desenvolupador del navegador són poderoses per inspeccionar, depurar i optimitzar el teu codi JavaScript. En dominar aquestes eines, pots millorar significativament el teu flux de treball de desenvolupament i el rendiment de les teves aplicacions web. Practica utilitzant aquestes eines regularment per convertir-te en un expert en identificar i resoldre problemes de manera eficient.
