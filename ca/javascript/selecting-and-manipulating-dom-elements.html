Introducció

En aquesta secció, explorarem com seleccionar i manipular elements del DOM (Document Object Model) utilitzant JavaScript. Aquesta és una habilitat fonamental per al desenvolupament web, que et permet interactuar dinàmicament amb les pàgines web i modificar-les.

Seleccionar Elements del DOM

Utilitzant `getElementById`

  • Descripció: Selecciona un únic element pel seu atribut id.
  • Sintaxi: document.getElementById('id')
  • Exemple: 
    const element = document.getElementById('myElement');
console.log(element); // Mostra l'element amb id 'myElement'

Utilitzant `getElementsByClassName`

  • Descripció: Selecciona tots els elements amb un nom de classe donat.
  • Sintaxi: document.getElementsByClassName('className')
  • Exemple: 
    const elements = document.getElementsByClassName('myClass');
console.log(elements); // Mostra una HTMLCollection en viu d'elements amb la classe 'myClass'

Utilitzant `getElementsByTagName`

  • Descripció: Selecciona tots els elements amb un nom de etiqueta donat.
  • Sintaxi: document.getElementsByTagName('tagName')
  • Exemple: 
    const elements = document.getElementsByTagName('div');
console.log(elements); // Mostra una HTMLCollection en viu de tots els elements <div>

Utilitzant `querySelector`

  • Descripció: Selecciona el primer element que coincideix amb un selector CSS.
  • Sintaxi: document.querySelector('selector')
  • Exemple: 
    const element = document.querySelector('.myClass');
console.log(element); // Mostra el primer element amb la classe 'myClass'

Utilitzant `querySelectorAll`

  • Descripció: Selecciona tots els elements que coincideixen amb un selector CSS.
  • Sintaxi: document.querySelectorAll('selector')
  • Exemple: 
    const elements = document.querySelectorAll('.myClass');
console.log(elements); // Mostra una NodeList de tots els elements amb la classe 'myClass'

Manipular Elements del DOM

Canviar Contingut

  • Inner HTML: Modificar el contingut HTML dins d'un element.

    const element = document.getElementById('myElement');
element.innerHTML = '<p>Nou contingut</p>';

  • Text Content: Modificar el contingut de text dins d'un element.

    const element = document.querySelector('.myClass');
element.textContent = 'Nou contingut de text';

Canviar Atributs

  • Set Attribute: Canviar o afegir un atribut a un element.

    const element = document.querySelector('img');
element.setAttribute('src', 'newImage.jpg');

  • Remove Attribute: Eliminar un atribut d'un element.

    const element = document.querySelector('img');
element.removeAttribute('alt');

Canviar Estils

  • Inline Styles: Modificar directament l'estil d'un element. 
    const element = document.getElementById('myElement');
element.style.color = 'red';
element.style.backgroundColor = 'yellow';

Afegir i Eliminar Classes

  • Add Class: Afegir una classe a un element.

    const element = document.querySelector('.myClass');
element.classList.add('newClass');

  • Remove Class: Eliminar una classe d'un element.

    const element = document.querySelector('.myClass');
element.classList.remove('oldClass');

  • Toggle Class: Alternar una classe en un element.

    const element = document.querySelector('.myClass');
element.classList.toggle('active');

Gestió d'Esdeveniments

  • Add Event Listener: Adjuntar un gestor d'esdeveniments a un element.

    const button = document.querySelector('button');
button.addEventListener('click', function() {
    alert('Botó clicat!');
});

  • Remove Event Listener: Eliminar un gestor d'esdeveniments d'un element.

    const button = document.querySelector('button');
function handleClick() {
    alert('Botó clicat!');
}
button.addEventListener('click', handleClick);
button.removeEventListener('click', handleClick);

Conclusió

Seleccionar i manipular elements del DOM és una part crucial del desenvolupament web. Dominant aquestes tècniques, pots crear pàgines web dinàmiques i interactives. Practica utilitzant els diferents mètodes per seleccionar elements i manipular el seu contingut, atributs i estils per convertir-te en un expert en la manipulació del DOM amb JavaScript.
