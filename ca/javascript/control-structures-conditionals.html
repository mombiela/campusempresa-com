Introducció als condicionals

Els condicionals són blocs fonamentals en la programació que et permeten executar diferents codi basat en certes condicions. En JavaScript, els condicionals t'ajuden a prendre decisions en el teu codi.

Instrucció `if`

La instrucció if és la forma més bàsica de condicional. Executa un bloc de codi si una condició especificada és certa.

Sintaxi

if (condition) {
  // codi a executar si la condició és certa
}

Exemple

let age = 18;

if (age >= 18) {
  console.log("You are an adult.");
}

En aquest exemple, el missatge "You are an adult." es mostrarà a la consola perquè la condició age >= 18 és certa.

Instrucció `if...else`

La instrucció if...else et permet executar un bloc de codi si la condició és certa, i un altre bloc si la condició és falsa.

Sintaxi

if (condition) {
  // codi a executar si la condició és certa
} else {
  // codi a executar si la condició és falsa
}

Exemple

let age = 16;

if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

En aquest exemple, el missatge "You are a minor." es mostrarà perquè la condició age >= 18 és falsa.

Instrucció `if...else if...else`

La instrucció if...else if...else s'utilitza per provar múltiples condicions.

Sintaxi

if (condition1) {
  // codi a executar si la condició1 és certa
} else if (condition2) {
  // codi a executar si la condició2 és certa
} else {
  // codi a executar si tant la condició1 com la condició2 són falses
}

Exemple

let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else if (score >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}

En aquest exemple, el missatge "Grade: B" es mostrarà perquè la condició score >= 80 és certa.

Operador ternari

L'operador ternari és una forma abreujada de la instrucció if...else. És útil per a condicions simples.

Sintaxi

condition ? expressionIfTrue : expressionIfFalse;

Exemple

let age = 20;
let message = age >= 18 ? "You are an adult." : "You are a minor.";
console.log(message);

En aquest exemple, el missatge "You are an adult." es mostrarà perquè la condició age >= 18 és certa.

Instrucció `switch`

La instrucció switch s'utilitza per realitzar diferents accions basades en diferents condicions. És una alternativa a utilitzar múltiples instruccions else if.

Sintaxi

switch (expression) {
  case value1:
    // codi a executar si expression === value1
    break;
  case value2:
    // codi a executar si expression === value2
    break;
  // més casos...
  default:
    // codi a executar si expression no coincideix amb cap cas
}

Exemple

let day = 3;
let dayName;

switch (day) {
  case 1:
    dayName = "Monday";
    break;
  case 2:
    dayName = "Tuesday";
    break;
  case 3:
    dayName = "Wednesday";
    break;
  case 4:
    dayName = "Thursday";
    break;
  case 5:
    dayName = "Friday";
    break;
  case 6:
    dayName = "Saturday";
    break;
  case 7:
    dayName = "Sunday";
    break;
  default:
    dayName = "Invalid day";
}

console.log(dayName);

En aquest exemple, el missatge "Wednesday" es mostrarà perquè el valor de day és 3.

Resum

  • Instrucció if: Executa un bloc de codi si una condició és certa.
  • Instrucció if...else: Executa un bloc de codi si una condició és certa, i un altre bloc si és falsa.
  • Instrucció if...else if...else: Prova múltiples condicions.
  • Operador ternari: Una forma abreujada de les instruccions if...else.
  • Instrucció switch: Una alternativa a múltiples instruccions else if per realitzar diferents accions basades en diferents condicions.

Entendre i utilitzar aquestes estructures de control de manera efectiva et permetrà escriure codi JavaScript més dinàmic i flexible.
