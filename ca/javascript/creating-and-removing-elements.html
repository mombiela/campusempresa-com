Crear i Eliminar Elements
API d'Emmagatzematge Web ►

En aquesta secció, explorarem com crear i eliminar elements en el DOM (Model d'Objectes del Document) utilitzant JavaScript. Aquesta és una habilitat fonamental per manipular pàgines web dinàmicament.

Crear Elements

Crear elements en JavaScript implica utilitzar el mètode document.createElement. Aquest mètode et permet crear nous elements HTML que es poden afegir al DOM.

Conceptes Clau

  • document.createElement(tagName): Crea un nou element amb el nom de l'etiqueta especificada.
  • element.appendChild(childElement): Afegeix un element fill a l'element pare especificat.
  • element.innerHTML: Estableix o obté el contingut HTML dins d'un element.

Exemple: Crear un Nou Paràgraf

// Crear un nou element de paràgraf
const newParagraph = document.createElement('p');

// Establir el contingut de text del paràgraf
newParagraph.textContent = 'Aquest és un nou paràgraf creat amb JavaScript.';

// Trobar l'element pare on s'afegirà el nou paràgraf
const parentElement = document.getElementById('content');

// Afegir el nou paràgraf a l'element pare
parentElement.appendChild(newParagraph);

En aquest exemple:

  1. Creem un nou element <p>.
  2. Establim el seu contingut de text.
  3. Trobem l'element pare amb l'ID content.
  4. Afegim el nou paràgraf a l'element pare.

Exemple: Crear un Nou Element de Llista

// Crear un nou element de llista
const newItem = document.createElement('li');

// Establir el contingut de text de l'element de llista
newItem.textContent = 'Nou element de llista';

// Trobar l'element pare (una llista desordenada)
const list = document.getElementById('myList');

// Afegir el nou element de llista a la llista desordenada
list.appendChild(newItem);

Eliminar Elements

Eliminar elements del DOM es pot fer utilitzant el mètode removeChild o el mètode remove.

Conceptes Clau

  • parentElement.removeChild(childElement): Elimina un element fill especificat de l'element pare.
  • element.remove(): Elimina l'element del DOM.

Exemple: Eliminar un Element Fill

// Trobar l'element pare
const parentElement = document.getElementById('content');

// Trobar l'element fill que s'ha d'eliminar
const childElement = document.getElementById('paragraphToRemove');

// Eliminar l'element fill de l'element pare
parentElement.removeChild(childElement);

Exemple: Eliminar un Element Directament

// Trobar l'element que s'ha d'eliminar
const elementToRemove = document.getElementById('itemToRemove');

// Eliminar l'element del DOM
elementToRemove.remove();

Resum

En aquesta secció, hem cobert els conceptes bàsics de crear i eliminar elements en el DOM utilitzant JavaScript. Hem après com:

  • Crear nous elements amb document.createElement.
  • Afegir nous elements al DOM amb appendChild.
  • Eliminar elements utilitzant removeChild i remove.

Aquestes habilitats són essencials per manipular pàgines web dinàmicament, permetent-te crear experiències d'usuari interactives i responsives. Practica aquestes tècniques amb diferents elements HTML per sentir-te més còmode amb la manipulació del DOM.
