Introducció

Benvingut al teu primer pas en l'aprenentatge de JavaScript! En aquesta secció, et guiarem a través de l'escriptura del teu primer programa en JavaScript. Al final d'aquesta lliçó, tindràs una comprensió bàsica de com escriure i executar codi JavaScript.

Configurant el teu entorn

Abans de començar a codificar, configurem l'entorn on escriuràs i executaràs el teu codi JavaScript.

Utilitzant un editor de text

Pots escriure codi JavaScript en qualsevol editor de text. Algunes opcions populars inclouen:

  • Visual Studio Code
  • Sublime Text
  • Atom

Executant JavaScript en un navegador

JavaScript s'utilitza principalment per al desenvolupament web, així que pots executar el teu codi directament en un navegador web. Tots els navegadors moderns (Chrome, Firefox, Safari, Edge) tenen motors de JavaScript integrats.

Utilitzant la consola del navegador

Per provar ràpidament el codi JavaScript, pots utilitzar la consola del navegador:

  1. Obre el teu navegador.
  2. Fes clic dret a la pàgina i selecciona "Inspect" o prem Ctrl+Shift+I (Windows/Linux) o Cmd+Option+I (Mac).
  3. Ves a la pestanya "Console".

Escriure el teu primer programa

Escrivim un programa senzill que mostri un missatge a la consola.

El codi

console.log("Hello, World!");

Explicació

  • console: Aquest és un objecte proporcionat pel navegador que et permet interactuar amb la consola de depuració del navegador.
  • log(): Aquest és un mètode de l'objecte console que mostra un missatge a la consola.
  • "Hello, World!": Aquesta és una cadena, una seqüència de caràcters tancada entre cometes, que es mostrarà a la consola.

Executant el codi

  1. Obre la consola del teu navegador.
  2. Escriu o enganxa el codi console.log("Hello, World!"); a la consola.
  3. Prem Enter.

Hauries de veure el missatge "Hello, World!" mostrat a la consola.

Comprenent la sintaxi bàsica

Desglossem algunes regles bàsiques de sintaxi en JavaScript:

  • Declaracions: Cada línia de codi s'anomena declaració i hauria d'acabar amb un punt i coma (;).
  • Comentaris: Pots afegir comentaris al teu codi utilitzant // per a comentaris d'una sola línia o /* */ per a comentaris de múltiples línies.

Exemple

// Aquest és un comentari d'una sola línia
console.log("Hello, World!"); // Això imprimirà "Hello, World!" a la consola

/*
Aquest és un comentari de múltiples línies
Pot abastar diverses línies
*/
console.log("JavaScript és divertit!");

Exercicis

Prova els següents exercicis per practicar el que has après:

  1. Escriu un programa que imprimeixi el teu nom a la consola.
  2. Afegeix un comentari d'una sola línia a sobre del teu codi explicant què fa.
  3. Escriu un programa que imprimeixi el següent missatge a la consola: "Aprenent JavaScript pas a pas."

Conclusió

Felicitats per escriure el teu primer programa en JavaScript! Has après com configurar el teu entorn, escriure codi JavaScript bàsic i comprendre algunes regles fonamentals de sintaxi. Continua practicant escrivint més codi i experimentant amb diferents declaracions. En la propera lliçó, aprofundirem en les variables i els tipus de dades en JavaScript. Feliç codificació!
