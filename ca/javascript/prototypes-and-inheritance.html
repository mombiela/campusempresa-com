Introducció

En JavaScript, entendre els prototips i l'herència és crucial per dominar la programació orientada a objectes. Aquesta secció cobrirà els conceptes bàsics dels prototips, com funciona l'herència en JavaScript i com utilitzar aquests conceptes per crear codi més eficient i organitzat.

Prototips

Què és un Prototip?

  • Cada objecte de JavaScript té un prototip.
  • Un prototip també és un objecte.
  • Els prototips són el mecanisme pel qual els objectes de JavaScript hereten característiques els uns dels altres.

Cadena de Prototips

  • Els objectes poden tenir un prototip, que al seu torn pot tenir el seu propi prototip, formant una cadena.
  • La cadena acaba quan un objecte té null com a prototip.

Exemple de Cadena de Prototips

let animal = {
    menja: true
};

let conill = {
    salta: true
};

conill.__proto__ = animal; // establint animal com el prototip de conill

console.log(conill.menja); // true
console.log(conill.salta); // true

Accedint als Prototips

  • Utilitza Object.getPrototypeOf(obj) per obtenir el prototip d'un objecte.
  • Utilitza Object.setPrototypeOf(obj, prototype) per establir el prototip d'un objecte.

Exemple

let animal = {
    menja: true
};

let conill = Object.create(animal); // conill hereta d'animal

console.log(Object.getPrototypeOf(conill) === animal); // true

Herència

Què és l'Herència?

  • L'herència és una característica que permet a un objecte heretar propietats i mètodes d'un altre objecte.
  • Això ajuda a reutilitzar codi i crear una estructura jeràrquica.

Herència Clàssica vs Prototipal

  • L'herència clàssica implica classes i és comuna en llenguatges com Java i C++.
  • L'herència prototipal implica objectes i és més flexible.

Creant una Cadena d'Herència

  • Utilitza Object.create() per crear un nou objecte amb un prototip especificat.

Exemple

let animal = {
    menja: true,
    camina() {
        console.log("L'animal camina");
    }
};

let conill = Object.create(animal);
conill.salta = true;

conill.camina(); // L'animal camina
console.log(conill.menja); // true
console.log(conill.salta); // true

Funcions Constructores i Prototips

Funcions Constructores

  • Les funcions constructores s'utilitzen per crear múltiples objectes similars.
  • Utilitza la paraula clau new per crear una instància d'un objecte.

Exemple

function Animal(nom) {
    this.nom = nom;
    this.menja = true;
}

Animal.prototype.camina = function() {
    console.log(this.nom + " camina");
};

let conill = new Animal("Conill");
conill.camina(); // Conill camina

Propietat Prototype

  • Cada funció en JavaScript té una propietat prototype.
  • Aquesta propietat s'utilitza per adjuntar mètodes i propietats que haurien de ser heretats per les instàncies de la funció constructora.

Exemple

function Animal(nom) {
    this.nom = nom;
}

Animal.prototype.menja = true;

let conill = new Animal("Conill");
console.log(conill.menja); // true

Classes ES6

Introducció a les Classes

  • ES6 va introduir les classes per facilitar l'escriptura i la comprensió de l'herència en JavaScript.
  • Les classes són sucre sintàctic sobre l'herència basada en prototips existent en JavaScript.

Exemple

class Animal {
    constructor(nom) {
        this.nom = nom;
        this.menja = true;
    }

    camina() {
        console.log(this.nom + " camina");
    }
}

let conill = new Animal("Conill");
conill.camina(); // Conill camina

Herència amb Classes

  • Utilitza la paraula clau extends per crear una classe que hereti d'una altra classe.

Exemple

class Animal {
    constructor(nom) {
        this.nom = nom;
    }

    camina() {
        console.log(this.nom + " camina");
    }
}

class Conill extends Animal {
    constructor(nom, color) {
        super(nom); // crida al constructor del pare
        this.color = color;
    }

    salta() {
        console.log(this.nom + " salta");
    }
}

let conill = new Conill("Conill", "blanc");
conill.camina(); // Conill camina
conill.salta(); // Conill salta

Resum

  • Els prototips són fonamentals per entendre el model d'herència de JavaScript.
  • Els objectes hereten propietats i mètodes dels seus prototips.
  • Les funcions constructores i la paraula clau new s'utilitzen per crear instàncies amb propietats i mètodes compartits.
  • Les classes ES6 proporcionen una sintaxi més intuïtiva per crear i gestionar l'herència en JavaScript.
  • Entendre aquests conceptes permet un codi més eficient i organitzat, aprofitant el poder de la programació orientada a objectes en JavaScript.
