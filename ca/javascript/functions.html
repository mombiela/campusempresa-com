Introducció a les Funcions

Les funcions són un dels blocs fonamentals en JavaScript. Una funció és un bloc de codi reutilitzable dissenyat per realitzar una tasca particular. Les funcions permeten encapsular codi per a la seva reutilització, fent que els teus programes siguin més modulars i fàcils de mantenir.

Conceptes Clau

  • Declaració de Funció: Defineix una funció amb un nom especificat.
  • Expressió de Funció: Defineix una funció com a part d'una expressió.
  • Funcions Fletxa: Una sintaxi més curta per escriure expressions de funció.
  • Paràmetres i Arguments: Entrades a les funcions.
  • Valors de Retorn: Sortides de les funcions.
  • Àmbit: El context en el qual les variables són accessibles.

Declaració de Funció

Una declaració de funció defineix una funció amb nom. La sintaxi és senzilla:

function functionName(parameters) {
  // codi a executar
}

Exemple

function greet(name) {
  console.log('Hello, ' + name + '!');
}

greet('Alice'); // Sortida: Hello, Alice!

Explicació

  • function greet(name): Declara una funció anomenada greet amb un paràmetre name.
  • console.log('Hello, ' + name + '!');: Mostra un missatge de salutació a la consola.
  • greet('Alice');: Crida la funció greet amb l'argument 'Alice'.

Expressió de Funció

Una expressió de funció defineix una funció com a part d'una expressió. Pot ser anònima (sense nom) o amb nom.

Exemple

const greet = function(name) {
  console.log('Hello, ' + name + '!');
};

greet('Bob'); // Sortida: Hello, Bob!

Explicació

  • const greet = function(name) { ... };: Defineix una funció anònima i l'assigna a la variable greet.
  • greet('Bob');: Crida la funció greet amb l'argument 'Bob'.

Funcions Fletxa

Les funcions fletxa proporcionen una sintaxi concisa per escriure expressions de funció. Són especialment útils per a funcions curtes.

Exemple

const greet = (name) => {
  console.log('Hello, ' + name + '!');
};

greet('Charlie'); // Sortida: Hello, Charlie!

Explicació

  • const greet = (name) => { ... };: Defineix una funció fletxa i l'assigna a la variable greet.
  • greet('Charlie');: Crida la funció greet amb l'argument 'Charlie'.

Paràmetres i Arguments

Els paràmetres són variables llistades com a part de la definició de la funció. Els arguments són els valors reals passats a la funció quan és cridada.

Exemple

function add(a, b) {
  return a + b;
}

const sum = add(5, 3); // sum és 8

Explicació

  • function add(a, b) { ... }: Defineix una funció add amb dos paràmetres a i b.
  • return a + b;: Retorna la suma de a i b.
  • const sum = add(5, 3);: Crida la funció add amb els arguments 5 i 3, i assigna el resultat a sum.

Valors de Retorn

Les funcions poden retornar valors utilitzant la instrucció return. Si no s'utilitza cap instrucció return, la funció retorna undefined.

Exemple

function multiply(a, b) {
  return a * b;
}

const product = multiply(4, 7); // product és 28

Explicació

  • function multiply(a, b) { ... }: Defineix una funció multiply amb dos paràmetres a i b.
  • return a * b;: Retorna el producte de a i b.
  • const product = multiply(4, 7);: Crida la funció multiply amb els arguments 4 i 7, i assigna el resultat a product.

Àmbit

L'àmbit determina l'accessibilitat de les variables. JavaScript té àmbit de funció i àmbit de bloc.

Exemple

function outerFunction() {
  const outerVar = 'I am outside!';
  
  function innerFunction() {
    const innerVar = 'I am inside!';
    console.log(outerVar); // Accessible
    console.log(innerVar); // Accessible
  }
  
  innerFunction();
  console.log(outerVar); // Accessible
  // console.log(innerVar); // Inaccessible, causaria un error
}

outerFunction();

Explicació

  • outerFunction defineix una variable outerVar i una innerFunction nidificada.
  • innerFunction pot accedir a outerVar perquè està dins del mateix àmbit de funció.
  • outerFunction no pot accedir a innerVar perquè està definida dins de innerFunction.

Conclusió

Les funcions són una característica poderosa en JavaScript que permeten escriure codi reutilitzable i modular. Entendre les declaracions de funció, expressions, funcions fletxa, paràmetres, arguments, valors de retorn i àmbit és essencial per escriure codi JavaScript eficient i fàcil de mantenir. Practica escrivint diferents tipus de funcions i experimenta amb el seu àmbit per aprofundir en la teva comprensió.
