Les funcions d'ordre superior són un concepte fonamental en JavaScript i en la programació funcional. Són funcions que poden prendre altres funcions com a arguments o retornar funcions com a resultat. Aquest tema et guiarà des dels conceptes bàsics fins a usos més avançats de les funcions d'ordre superior.

Què són les Funcions d'Ordre Superior?

  • Definició: Una funció d'ordre superior és una funció que pren una o més funcions com a arguments o retorna una funció com a resultat.
  • Importància: Permeten un codi més abstracte i flexible, habilitant patrons poderosos com les callbacks, la composició de funcions i més.

Exemples de Funcions d'Ordre Superior

Exemple Bàsic

function higherOrderFunction(callback) {
    return callback();
}

function sayHello() {
    return "Hello, World!";
}

console.log(higherOrderFunction(sayHello)); // Output: Hello, World!

En aquest exemple:

  • higherOrderFunction és una funció d'ordre superior perquè pren una altra funció (callback) com a argument.
  • sayHello és una funció simple que retorna una cadena.
  • Quan higherOrderFunction es crida amb sayHello com a argument, executa sayHello i retorna el seu resultat.

Funcions d'Ordre Superior Comunes en JavaScript

Mètodes d'Array

JavaScript proporciona diverses funcions d'ordre superior integrades per a arrays, com map, filter i reduce.

map

  • Propòsit: Crea un nou array poblat amb els resultats de cridar una funció proporcionada en cada element de l'array que crida.
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6, 8]

filter

  • Propòsit: Crea un nou array amb tots els elements que passen la prova implementada per la funció proporcionada.
const numbers = [1, 2, 3, 4];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // Output: [2, 4]

reduce

  • Propòsit: Executa una funció reductora (que tu proporciones) en cada element de l'array, resultant en un únic valor de sortida.
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum); // Output: 10

Ús Avançat de les Funcions d'Ordre Superior

Composició de Funcions

  • Definició: Combinació de dues o més funcions per produir una nova funció.
const add = x => y => x + y;
const multiply = x => y => x * y;

const addAndMultiply = (a, b, c) => multiply(add(a)(b))(c);

console.log(addAndMultiply(1, 2, 3)); // Output: 9

Currificació

  • Definició: El procés de transformar una funció que pren múltiples arguments en una seqüència de funcions que cadascuna pren un únic argument.
function curry(f) {
    return function(a) {
        return function(b) {
            return f(a, b);
        };
    };
}

function sum(a, b) {
    return a + b;
}

const curriedSum = curry(sum);
console.log(curriedSum(1)(2)); // Output: 3

Exercicis

  1. Exercici Bàsic: Escriu una funció d'ordre superior que prengui una funció i un número, i apliqui la funció al número tres vegades.
function applyThrice(func, num) {
    return func(func(func(num)));
}

function increment(x) {
    return x + 1;
}

console.log(applyThrice(increment, 5)); // Output: 8
  1. Exercici Intermedi: Utilitza map per crear un nou array que contingui les longituds de cada cadena en un array de cadenes.
const strings = ["hello", "world", "JavaScript"];
const lengths = strings.map(str => str.length);
console.log(lengths); // Output: [5, 5, 10]
  1. Exercici Avançat: Escriu una funció compose que prengui dues funcions i retorni una nova funció que sigui la seva composició.
function compose(f, g) {
    return function(x) {
        return f(g(x));
    };
}

const add1 = x => x + 1;
const double = x => x * 2;

const add1ThenDouble = compose(double, add1);
console.log(add1ThenDouble(5)); // Output: 12

Conclusió

Les funcions d'ordre superior són una característica poderosa en JavaScript que permeten un codi més abstracte i flexible. Entenent i utilitzant les funcions d'ordre superior, pots escriure codi més concís i expressiu. Practica utilitzant funcions d'ordre superior integrades com map, filter i reduce, i intenta crear les teves pròpies funcions d'ordre superior per aprofundir en la teva comprensió.
