Introducció als Arrays

Els arrays són una estructura de dades fonamental en JavaScript utilitzada per emmagatzemar múltiples valors en una sola variable. Són essencials per gestionar col·leccions de dades i s'utilitzen àmpliament en diversos escenaris de programació.

Conceptes Clau

  • Definició: Un array és una variable especial que pot contenir més d'un valor alhora.
  • Indexació: Els arrays utilitzen una indexació basada en zero, és a dir, el primer element està a l'índex 0.
  • Elements de l'Array: Cada element en un array s'anomena element.
  • Longitud de l'Array: El nombre d'elements en un array es pot determinar utilitzant la propietat length.

Creació d'Arrays

Pots crear arrays en JavaScript utilitzant diferents mètodes.

Utilitzant Literals d'Array

let fruits = ["Poma", "Plàtan", "Cirera"];

Utilitzant el Constructor d'Array

let fruits = new Array("Poma", "Plàtan", "Cirera");

Accés als Elements de l'Array

Pots accedir als elements d'un array utilitzant el seu índex.

let fruits = ["Poma", "Plàtan", "Cirera"];
console.log(fruits[0]); // Sortida: Poma
console.log(fruits[1]); // Sortida: Plàtan
console.log(fruits[2]); // Sortida: Cirera

Modificació dels Elements de l'Array

Pots canviar el valor d'un element d'un array accedint al seu índex.

let fruits = ["Poma", "Plàtan", "Cirera"];
fruits[1] = "Nabiu";
console.log(fruits); // Sortida: ["Poma", "Nabiu", "Cirera"]

Mètodes d'Array

JavaScript proporciona diversos mètodes integrats per manipular arrays.

Afegir i Eliminar Elements

  • push(): Afegeix un o més elements al final d'un array.
  • pop(): Elimina l'últim element d'un array.
  • unshift(): Afegeix un o més elements al començament d'un array.
  • shift(): Elimina el primer element d'un array.
let fruits = ["Poma", "Plàtan"];
fruits.push("Cirera"); // ["Poma", "Plàtan", "Cirera"]
fruits.pop(); // ["Poma", "Plàtan"]
fruits.unshift("Mango"); // ["Mango", "Poma", "Plàtan"]
fruits.shift(); // ["Poma", "Plàtan"]

Iteració sobre Arrays

Pots recórrer els elements d'un array utilitzant diversos mètodes.

Utilitzant un bucle for

let fruits = ["Poma", "Plàtan", "Cirera"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Utilitzant forEach()

let fruits = ["Poma", "Plàtan", "Cirera"];
fruits.forEach(function(fruit) {
    console.log(fruit);
});

Altres Mètodes Útils

  • map(): Crea un nou array amb els resultats de cridar una funció a cada element.
  • filter(): Crea un nou array amb tots els elements que passen una prova.
  • reduce(): Executa una funció reductora a cada element, resultant en un únic valor de sortida.
let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(function(num) {
    return num * 2;
}); // [2, 4, 6, 8, 10]

let evens = numbers.filter(function(num) {
    return num % 2 === 0;
}); // [2, 4]

let sum = numbers.reduce(function(total, num) {
    return total + num;
}, 0); // 15

Conceptes Avançats d'Array

A mesura que et tornis més proficient amb els arrays, pots explorar temes més avançats.

Arrays Multidimensionals

Els arrays poden contenir altres arrays, creant arrays multidimensionals.

let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];
console.log(matrix[0][0]); // Sortida: 1
console.log(matrix[1][2]); // Sortida: 6

Desestructuració d'Array

Una manera convenient d'extreure valors d'arrays.

let fruits = ["Poma", "Plàtan", "Cirera"];
let [first, second, third] = fruits;
console.log(first); // Sortida: Poma
console.log(second); // Sortida: Plàtan
console.log(third); // Sortida: Cirera

Operador Spread

L'operador spread (...) et permet expandir elements d'un array.

let fruits = ["Poma", "Plàtan"];
let moreFruits = ["Cirera", ...fruits];
console.log(moreFruits); // Sortida: ["Cirera", "Poma", "Plàtan"]

Conclusió

Els arrays són una característica versàtil i poderosa en JavaScript, que et permeten emmagatzemar i manipular col·leccions de dades de manera eficient. Des d'operacions bàsiques com accedir i modificar elements fins a tècniques avançades com la desestructuració i l'ús de l'operador spread, dominar els arrays és crucial per a qualsevol desenvolupador de JavaScript. Practica utilitzant arrays amb diferents mètodes i escenaris per aprofundir en la teva comprensió i millorar les teves habilitats de programació.
