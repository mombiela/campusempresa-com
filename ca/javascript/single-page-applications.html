Introducció a les aplicacions d'una sola pàgina (SPA)

Les aplicacions d'una sola pàgina (SPA) són aplicacions web que carreguen una sola pàgina HTML i actualitzen dinàmicament el contingut a mesura que l'usuari interactua amb l'aplicació. Aquest enfocament proporciona una experiència d'usuari més fluida, similar a la d'una aplicació d'escriptori.

Conceptes clau

  • Càrrega d'una sola pàgina: Només es carrega una pàgina HTML inicialment.
  • Contingut dinàmic: El contingut s'actualitza dinàmicament sense refrescar tota la pàgina.
  • Enrutament del costat del client: La navegació dins de l'aplicació es gestiona al costat del client.
  • AJAX: JavaScript asíncron i XML (AJAX) s'utilitza per obtenir dades del servidor sense recarregar la pàgina.

Beneficis de les SPA

  • Millora de l'experiència d'usuari: Interaccions més ràpides ja que només es fan actualitzacions parcials de la pàgina.
  • Reducció de la càrrega del servidor: Menys recàrregues completes de pàgina signifiquen menys processament del servidor.
  • Facilitat de depuració: Eines com Chrome DevTools faciliten la depuració d'aplicacions amb molt de JavaScript.

Exemple bàsic de SPA

Creem una SPA simple utilitzant JavaScript pur.

Estructura HTML

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SPA Simple</title>
</head>
<body>
    <nav>
        <a href="#" id="home">Inici</a>
        <a href="#" id="about">Sobre nosaltres</a>
    </nav>
    <div id="content">
        <h1>Benvingut a la nostra SPA</h1>
    </div>

    <script src="app.js"></script>
</body>
</html>

JavaScript per al contingut dinàmic

document.addEventListener('DOMContentLoaded', () => {
    const contentDiv = document.getElementById('content');

    document.getElementById('home').addEventListener('click', () => {
        contentDiv.innerHTML = '<h1>Benvingut a la nostra SPA</h1>';
    });

    document.getElementById('about').addEventListener('click', () => {
        contentDiv.innerHTML = '<h1>Sobre nosaltres</h1><p>Aquest és un exemple simple de SPA.</p>';
    });
});

Explicació

  • HTML: Conté una barra de navegació i un div de contingut on es carregarà el contingut dinàmic.
  • JavaScript: Escolta els esdeveniments de clic en els enllaços de navegació i actualitza el div de contingut sense recarregar la pàgina.

Conceptes avançats de SPA

A mesura que progressis, potser voldràs utilitzar frameworks i biblioteques per gestionar SPA més complexes.

Frameworks i biblioteques populars per a SPA

  • React: Una biblioteca de JavaScript per construir interfícies d'usuari.
  • Vue.js: Un framework progressiu per construir interfícies d'usuari.
  • Angular: Una plataforma per construir aplicacions web mòbils i d'escriptori.

Taula comparativa de frameworks per a SPA

Exemple utilitzant React

Creem una SPA simple utilitzant React.

Configuració de React

Primer, configura un nou projecte de React utilitzant Create React App:

npx create-react-app my-spa
cd my-spa
npm start

Exemple de component React

import React, { useState } from 'react';

function App() {
    const [page, setPage] = useState('home');

    const renderContent = () => {
        switch (page) {
            case 'home':
                return <h1>Benvingut a la nostra SPA</h1>;
            case 'about':
                return <div><h1>Sobre nosaltres</h1><p>Aquest és un exemple simple de SPA utilitzant React.</p></div>;
            default:
                return <h1>Benvingut a la nostra SPA</h1>;
        }
    };

    return (
        <div>
            <nav>
                <button onClick={() => setPage('home')}>Inici</button>
                <button onClick={() => setPage('about')}>Sobre nosaltres</button>
            </nav>
            <div id="content">
                {renderContent()}
            </div>
        </div>
    );
}

export default App;

Explicació

  • React: Utilitza components i estat per gestionar el contingut dinàmic.
  • Hook useState: Gestiona l'estat de la pàgina actual.
  • Funció renderContent: Renderitza diferents continguts segons l'estat de la pàgina actual.

Conclusió

Les aplicacions d'una sola pàgina (SPA) proporcionen una experiència d'usuari contínua i responsiva actualitzant dinàmicament el contingut sense recàrregues completes de pàgina. Tot i que es pot utilitzar JavaScript pur per crear SPA simples, frameworks com React, Vue.js i Angular ofereixen eines potents per gestionar aplicacions més complexes. Entendre els conceptes bàsics de les SPA i explorar conceptes avançats amb frameworks populars t'ajudarà a construir aplicacions web eficients i fàcils d'utilitzar.
