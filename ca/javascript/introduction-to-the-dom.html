Què és el DOM?

El Document Object Model (DOM) és una interfície de programació per a documents web. Representa la pàgina de manera que els programes poden canviar l'estructura, l'estil i el contingut del document. El DOM representa el document com a nodes i objectes; d'aquesta manera, els llenguatges de programació poden interactuar amb la pàgina.

  • Document: El document HTML o XML.
  • Object: Cada element del document és un objecte.
  • Model: L'estructura del document.

Per què és important el DOM?

Entendre el DOM és crucial per al desenvolupament web perquè et permet:

  • Canviar dinàmicament el contingut d'una pàgina web.
  • Modificar l'estructura del document.
  • Canviar els estils dels elements.
  • Reaccionar als esdeveniments de l'usuari (per exemple, clics, premses de tecles).

Conceptes bàsics

Nodes i elements

En el DOM, tot és un node. Els nodes poden ser:

  • Nodes d'element: Representen elements HTML.
  • Nodes de text: Representen text dins dels elements.
  • Nodes d'atribut: Representen atributs dels elements.
  • Nodes de comentari: Representen comentaris en l'HTML.

Estructura d'arbre

El DOM està estructurat com un arbre de nodes. L'objecte document és l'arrel, i cada element és una branca.

<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

En l'HTML anterior, l'estructura de l'arbre seria així:

Document
 └── html
      ├── head
      │    └── title
      └── body
           ├── h1
           └── p

Accedint al DOM

Seleccionant elements

Per manipular el DOM, primer has de seleccionar els elements amb els quals vols treballar. JavaScript proporciona diversos mètodes per a això:

  • document.getElementById(id): Selecciona un element pel seu ID.
  • document.getElementsByClassName(className): Selecciona tots els elements amb la classe especificada.
  • document.getElementsByTagName(tagName): Selecciona tots els elements amb el nom de l'etiqueta especificada.
  • document.querySelector(selector): Selecciona el primer element que coincideix amb el selector CSS.
  • document.querySelectorAll(selector): Selecciona tots els elements que coincideixen amb el selector CSS.

Exemple

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">Hello, World!</h1>
    <p class="text">This is a paragraph.</p>
    <p class="text">This is another paragraph.</p>

    <script>
      // Selecciona l'element per ID
      const header = document.getElementById('header');
      console.log(header); // <h1 id="header">Hello, World!</h1>

      // Selecciona els elements per nom de classe
      const paragraphs = document.getElementsByClassName('text');
      console.log(paragraphs); // HTMLCollection of <p> elements

      // Selecciona els elements per nom d'etiqueta
      const allParagraphs = document.getElementsByTagName('p');
      console.log(allParagraphs); // HTMLCollection of <p> elements

      // Selecciona el primer element que coincideix amb el selector CSS
      const firstParagraph = document.querySelector('.text');
      console.log(firstParagraph); // <p class="text">This is a paragraph.</p>

      // Selecciona tots els elements que coincideixen amb el selector CSS
      const allTextElements = document.querySelectorAll('.text');
      console.log(allTextElements); // NodeList of <p> elements
    </script>
  </body>
</html>

Manipulant el DOM

Canviant el contingut

Pots canviar el contingut d'un element utilitzant les propietats innerHTML o textContent.

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">Hello, World!</h1>

    <script>
      const header = document.getElementById('header');
      header.innerHTML = 'Hello, JavaScript!'; // Canvia el contingut HTML
      header.textContent = 'Hello, DOM!'; // Canvia el contingut de text
    </script>
  </body>
</html>

Canviant els estils

Pots canviar els estils d'un element utilitzant la propietat style.

<!DOCTYPE html>
<html>
  <body>
    <h1 id="header">Hello, World!</h1>

    <script>
      const header = document.getElementById('header');
      header.style.color = 'blue'; // Canvia el color del text a blau
      header.style.fontSize = '24px'; // Canvia la mida de la font a 24px
    </script>
  </body>
</html>

Afegint i eliminant elements

Pots afegir i eliminar elements utilitzant mètodes com appendChild, removeChild, createElement i insertBefore.

<!DOCTYPE html>
<html>
  <body>
    <div id="container">
      <p>This is a paragraph.</p>
    </div>

    <script>
      const container = document.getElementById('container');

      // Crea un nou element
      const newParagraph = document.createElement('p');
      newParagraph.textContent = 'This is a new paragraph.';

      // Afegeix el nou element
      container.appendChild(newParagraph);

      // Elimina un element
      const oldParagraph = container.getElementsByTagName('p')[0];
      container.removeChild(oldParagraph);
    </script>
  </body>
</html>

Gestió d'esdeveniments

Els esdeveniments són accions que ocorren al navegador, com ara clics, premses de tecles o moviments del ratolí. Pots utilitzar JavaScript per escoltar aquests esdeveniments i executar codi en resposta.

Afegint escoltadors d'esdeveniments

Pots afegir escoltadors d'esdeveniments utilitzant el mètode addEventListener.

<!DOCTYPE html>
<html>
  <body>
    <button id="myButton">Click Me!</button>

    <script>
      const button = document.getElementById('myButton');

      button.addEventListener('click', function() {
        alert('Button was clicked!');
      });
    </script>
  </body>
</html>

Conclusió

El DOM és una interfície poderosa que et permet interactuar i manipular documents web. Entenent el DOM, pots crear pàgines web dinàmiques i interactives. Aquesta introducció ha cobert els conceptes bàsics de què és el DOM, com accedir-hi i manipular-lo, i com gestionar esdeveniments. A mesura que progressis, aprendràs tècniques més avançades i millors pràctiques per treballar amb el DOM.
