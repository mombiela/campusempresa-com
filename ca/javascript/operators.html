Introducció

Els operadors són blocs de construcció fonamentals en JavaScript que et permeten realitzar diverses operacions sobre variables i valors. Entendre com utilitzar els operadors de manera efectiva és crucial per escriure codi eficient i efectiu.

Tipus d'Operadors

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques.

  • Suma (+)

    let a = 5;
let b = 3;
let sum = a + b; // sum és 8

  • Resta (-)

    let difference = a - b; // difference és 2

  • Multiplicació (*)

    let product = a * b; // product és 15

  • Divisió (/)

    let quotient = a / b; // quotient és 1.6667

  • Mòdul (%)

    let remainder = a % b; // remainder és 2

  • Exponenciació (**)

    let power = a ** b; // power és 125

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació (=)

    let x = 10;

  • Assignació de Suma (+=)

    x += 5; // x és ara 15

  • Assignació de Resta (-=)

    x -= 3; // x és ara 12

  • Assignació de Multiplicació (*=)

    x *= 2; // x és ara 24

  • Assignació de Divisió (/=)

    x /= 4; // x és ara 6

  • Assignació de Mòdul (%=)

    x %= 5; // x és ara 1

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors i retornar un resultat booleà.

  • Igual (==)

    let isEqual = (a == b); // false

  • Igual Estricte (===)

    let isStrictEqual = (a === b); // false

  • No Igual (!=)

    let isNotEqual = (a != b); // true

  • No Igual Estricte (!==)

    let isStrictNotEqual = (a !== b); // true

  • Major Que (>)

    let isGreaterThan = (a > b); // true

  • Menor Que (<)

    let isLessThan = (a < b); // false

  • Major o Igual Que (>=)

    let isGreaterThanOrEqual = (a >= b); // true

  • Menor o Igual Que (<=)

    let isLessThanOrEqual = (a <= b); // false

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleanes.

  • AND Lògic (&&)

    let andResult = (a > 0 && b > 0); // true

  • OR Lògic (||)

    let orResult = (a > 0 || b < 0); // true

  • NOT Lògic (!)

    let notResult = !(a > 0); // false

Operadors a Nivell de Bits

Els operadors a nivell de bits s'utilitzen per realitzar operacions a nivell de bits.

  • AND (&)

    let bitwiseAnd = a & b; // 1 (binari: 0101 & 0011 = 0001)

  • OR (|)

    let bitwiseOr = a | b; // 7 (binari: 0101 | 0011 = 0111)

  • XOR (^)

    let bitwiseXor = a ^ b; // 6 (binari: 0101 ^ 0011 = 0110)

  • NOT (~)

    let bitwiseNot = ~a; // -6 (binari: ~0101 = 1010)

  • Desplaçament a l'Esquerra (<<)

    let leftShift = a << 1; // 10 (binari: 0101 << 1 = 1010)

  • Desplaçament a la Dreta (>>)

    let rightShift = a >> 1; // 2 (binari: 0101 >> 1 = 0010)

Operador Ternari

L'operador ternari és una forma abreujada de l'instrucció if-else.

  • Sintaxi: condition ? expr1 : expr2 
    let age = 18;
let canVote = (age >= 18) ? 'Yes' : 'No'; // canVote és 'Yes'

Operadors de Tipus

Els operadors de tipus s'utilitzen per determinar el tipus d'una variable.

  • typeof

    let typeOfA = typeof a; // 'number'

  • instanceof

    let isInstanceOf = a instanceof Number; // false

Conclusió

Els operadors són eines essencials en JavaScript que et permeten manipular dades i controlar el flux dels teus programes. Dominant els operadors aritmètics, d'assignació, de comparació, lògics, a nivell de bits, ternaris i de tipus, podràs escriure codi més eficient i efectiu. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en el seu ús.
