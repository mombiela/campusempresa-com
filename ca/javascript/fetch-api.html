La Fetch API proporciona una manera moderna, flexible i potent de fer sol·licituds de xarxa en JavaScript. És un reemplaçament per a l'antiga API XMLHttpRequest (XHR) i està integrada en els navegadors moderns.

Introducció a la Fetch API

  • Què és la Fetch API?

    • Una interfície moderna per fer sol·licituds HTTP.
    • Retorna Promeses, fent més fàcil treballar amb operacions asíncrones.
    • Suporta diversos mètodes HTTP com GET, POST, PUT, DELETE, etc.

  • Per què utilitzar la Fetch API?

    • Sintaxi més simple i neta en comparació amb XMLHttpRequest.
    • Millor suport per manejar dades JSON.
    • Suport integrat per a Promeses, fent més fàcil manejar codi asíncron.

Ús bàsic

Fer una sol·licitud GET

Per obtenir dades d'un servidor, pots utilitzar la funció fetch. Aquí tens un exemple bàsic de com fer una sol·licitud GET:

fetch('https://api.example.com/data')
  .then(response => response.json()) // Analitza el JSON de la resposta
  .then(data => console.log(data)) // Maneja les dades
  .catch(error => console.error('Error:', error)); // Maneja qualsevol error
  • Explicació:
    • fetch('https://api.example.com/data'): Inicia una sol·licitud GET a l'URL especificat.
    • .then(response => response.json()): Converteix la resposta a JSON.
    • .then(data => console.log(data)): Mostra les dades a la consola.
    • .catch(error => console.error('Error:', error)): Captura i mostra qualsevol error.

Fer una sol·licitud POST

Per enviar dades a un servidor, pots utilitzar el mètode POST:

fetch('https://api.example.com/data', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ key: 'value' }) // Converteix les dades a una cadena JSON
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
  • Explicació:
    • method: 'POST': Especifica el mètode HTTP.
    • headers: { 'Content-Type': 'application/json' }: Estableix el tipus de contingut a JSON.
    • body: JSON.stringify({ key: 'value' }): Converteix l'objecte JavaScript a una cadena JSON.

Ús avançat

Manejar diferents tipus de resposta

Fetch pot manejar diversos tipus de resposta com text, blob i arrayBuffer:

fetch('https://api.example.com/data')
  .then(response => response.text()) // Per a resposta en text pla
  .then(text => console.log(text))
  .catch(error => console.error('Error:', error));

Manejar errors HTTP

Fetch no rebutja automàticament errors HTTP (per exemple, 404 o 500). Has de manejar-los manualment:

fetch('https://api.example.com/data')
  .then(response => {
    if (!response.ok) {
      throw new Error('La resposta de la xarxa no va ser correcta ' + response.statusText);
    }
    return response.json();
  })
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Utilitzar Async/Await amb Fetch

La sintaxi async/await pot fer que el teu codi sigui més llegible:

async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    if (!response.ok) {
      throw new Error('La resposta de la xarxa no va ser correcta ' + response.statusText);
    }
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
}

fetchData();

Comparació: Fetch API vs. XMLHttpRequest

| Característica | Fetch API | XMLHttpRequest | |--------------------------|------------------------------------|----------------------------------| | Sintaxi | Més neta i concisa | Verbosa i complexa | | Promeses | Suport integrat | Requereix maneig manual | | Maneig de Respostes | Més fàcil amb mètodes integrats | Requereix anàlisi manual | | Maneig d'Errors | Manual però senzill | Manual i complex | | Funcions Modernes | Suporta streaming i altres funcions web modernes | Limitat |

Conclusió

La Fetch API és una eina potent i flexible per fer sol·licituds HTTP en JavaScript. La seva sintaxi moderna i el suport integrat per a Promeses la fan una opció preferida sobre l'antiga XMLHttpRequest. Entenent tant l'ús bàsic com l'avançat, pots manejar efectivament diverses operacions de xarxa en les teves aplicacions.
