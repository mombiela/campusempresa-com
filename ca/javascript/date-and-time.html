Introducció

Gestionar la data i l'hora és un requisit comú en moltes aplicacions JavaScript. JavaScript proporciona un objecte Date integrat per treballar amb dates i hores. Aquesta secció cobrirà els conceptes bàsics de l'objecte Date, com manipular dates i hores, i algunes tècniques avançades per treballar amb dates i hores en JavaScript.

Creació de Dates

L'objecte Date de JavaScript es pot crear de diverses maneres:

  • Data i Hora Actuals:

    const now = new Date();
console.log(now); // Mostra la data i hora actuals

  • Data i Hora Específiques:

    const specificDate = new Date('2023-10-01T10:20:30Z');
console.log(specificDate); // Mostra: 2023-10-01T10:20:30.000Z

  • Components de la Data:

    const dateComponents = new Date(2023, 9, 1, 10, 20, 30);
console.log(dateComponents); // Mostra: Dg Oct 01 2023 10:20:30 GMT+0000 (Hora Universal Coordinada)

Obtenir Components de la Data i Hora

Pots extreure diversos components d'un objecte Date utilitzant diferents mètodes:

  • Any, Mes, Dia:

    const date = new Date();
console.log(date.getFullYear()); // Mostra l'any
console.log(date.getMonth()); // Mostra el mes (0-11)
console.log(date.getDate()); // Mostra el dia del mes (1-31)

  • Hores, Minuts, Segons:

    const time = new Date();
console.log(time.getHours()); // Mostra les hores (0-23)
console.log(time.getMinutes()); // Mostra els minuts (0-59)
console.log(time.getSeconds()); // Mostra els segons (0-59)

Establir Components de la Data i Hora

També pots establir diversos components d'un objecte Date:

  • Establir Any, Mes, Dia:

    const date = new Date();
date.setFullYear(2025);
date.setMonth(11); // Desembre
date.setDate(25);
console.log(date); // Mostra: Dj Des 25 2025

  • Establir Hores, Minuts, Segons:

    const time = new Date();
time.setHours(15);
time.setMinutes(45);
time.setSeconds(30);
console.log(time); // Mostra: data actual amb l'hora establerta a 15:45:30

Formatació de Dates

JavaScript proporciona diversos mètodes per formatar dates:

  • toLocaleDateString():

    const date = new Date();
console.log(date.toLocaleDateString()); // Mostra la data en format local

  • toLocaleTimeString():

    const time = new Date();
console.log(time.toLocaleTimeString()); // Mostra l'hora en format local

  • toISOString():

    const isoDate = new Date();
console.log(isoDate.toISOString()); // Mostra la data en format ISO 8601

Aritmètica de Dates

Pots realitzar operacions aritmètiques amb dates:

  • Afegir Dies:

    const date = new Date();
date.setDate(date.getDate() + 5);
console.log(date); // Mostra la data d'aquí a 5 dies

  • Restar Dies:

    const date = new Date();
date.setDate(date.getDate() - 5);
console.log(date); // Mostra la data de fa 5 dies

Comparació de Dates

Per comparar dates, pots utilitzar operadors de comparació estàndard:

  • Igualtat:

    const date1 = new Date('2023-10-01');
const date2 = new Date('2023-10-01');
console.log(date1.getTime() === date2.getTime()); // Mostra: true

  • Major/Menor Que:

    const date1 = new Date('2023-10-01');
const date2 = new Date('2023-10-02');
console.log(date1 < date2); // Mostra: true

Treballar amb Fusos Horaris

Els objectes Date de JavaScript es basen en el fus horari local del navegador, però pots treballar amb diferents fusos horaris utilitzant biblioteques com moment-timezone:

  • Utilitzant moment-timezone: 
    // Inclou la biblioteca moment-timezone
const moment = require('moment-timezone');
const date = moment.tz('2023-10-01 10:20:30', 'America/New_York');
console.log(date.format()); // Mostra: 2023-10-01T10:20:30-04:00

Conclusió

Treballar amb dates i hores en JavaScript pot ser complex a causa de diversos factors com els fusos horaris i els diferents formats de data. No obstant això, l'objecte Date integrat proporciona un conjunt robust de mètodes per gestionar les tasques més comunes. Per a una manipulació més avançada de dates i hores, biblioteques com moment.js o date-fns poden ser molt útils. Entendre aquests conceptes bàsics et permetrà gestionar efectivament la data i l'hora en les teves aplicacions JavaScript.
