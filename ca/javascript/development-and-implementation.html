Introducció

En aquesta secció, tractarem el desenvolupament i la implementació d'aplicacions JavaScript. Això inclou la configuració del teu entorn de desenvolupament, la comprensió del flux de treball de desenvolupament i el desplegament de les teves aplicacions. Al final d'aquesta secció, tindràs una comprensió sòlida de com desenvolupar i implementar aplicacions JavaScript des de zero.

Configuració del Teu Entorn de Desenvolupament

Abans de començar a codificar, és essencial configurar un entorn de desenvolupament adequat.

  • Editors de Text i IDEs: Tria un editor de text o un Entorn de Desenvolupament Integrat (IDE) que s'adapti a les teves necessitats.
    • Les opcions populars inclouen Visual Studio Code, Sublime Text i Atom.
  • Node.js i npm: Instal·la Node.js i npm (Node Package Manager) per gestionar els teus paquets i dependències de JavaScript.
    • Descarrega des del lloc web oficial de Node.js.

Exemple: Instal·lant Node.js i npm

  1. Visita la pàgina de descàrrega de Node.js.
  2. Descarrega l'instal·lador per al teu sistema operatiu.
  3. Executa l'instal·lador i segueix les instruccions.
  4. Verifica la instal·lació executant les següents comandes al teu terminal: 
    node -v
npm -v

Flux de Treball de Desenvolupament

Comprendre el flux de treball de desenvolupament és crucial per a una codificació eficient.

  • Control de Versions: Utilitza Git per al control de versions per mantenir un registre dels canvis i col·laborar amb altres.
    • Plataformes com GitHub, GitLab i Bitbucket són populars per allotjar repositoris.
  • Linting i Formatació de Codi: Utilitza eines com ESLint i Prettier per mantenir la qualitat i la consistència del codi.
  • Proves: Implementa marcs de proves com Jest o Mocha per assegurar-te que el teu codi funcioni com s'espera.

Exemple: Comandes Bàsiques de Git

# Inicialitza un nou repositori Git
git init

# Clona un repositori existent
git clone <repository-url>

# Afegeix canvis a l'àrea d'staging
git add .

# Comet els canvis
git commit -m "El teu missatge de commit"

# Puja els canvis al repositori remot
git push origin main

Construcció i Agrupació

Les aplicacions modernes de JavaScript sovint requereixen construcció i agrupació per optimitzar el rendiment.

  • Agrupadors de Mòduls: Eines com Webpack, Rollup i Parcel agrupen els teus fitxers JavaScript en un sol fitxer.
  • Transpiladors: Utilitza Babel per convertir el codi ES6+ en ES5 per a una millor compatibilitat amb els navegadors.

Exemple: Configuració Bàsica de Webpack

  1. Instal·la Webpack i Webpack CLI: 
    npm install --save-dev webpack webpack-cli
  2. Crea un fitxer webpack.config.js: 
    const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
        },
      },
    ],
  },
};
  3. Afegeix un script de construcció a package.json: 
    "scripts": {
  "build": "webpack"
}
  4. Executa l'script de construcció: 
    npm run build

Desplegament

Desplegar la teva aplicació JavaScript implica fer-la accessible als usuaris.

  • Allotjament de Llocs Estàtics: Plataformes com GitHub Pages, Netlify i Vercel són excel·lents per desplegar llocs estàtics.
  • Desplegament del Costat del Servidor: Utilitza serveis com Heroku, AWS o DigitalOcean per a aplicacions del costat del servidor.

Exemple: Desplegant a GitHub Pages

  1. Inicialitza un repositori Git i puja el teu codi a GitHub.
  2. Instal·la el paquet gh-pages: 
    npm install --save-dev gh-pages
  3. Afegeix scripts de desplegament a package.json: 
    "scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d dist"
}
  4. Desplega la teva aplicació: 
    npm run deploy

Conclusió

En aquesta secció, hem cobert els aspectes essencials del desenvolupament i la implementació d'aplicacions JavaScript. Has après a configurar el teu entorn de desenvolupament, comprendre el flux de treball de desenvolupament, construir i agrupar el teu codi, i desplegar les teves aplicacions. Amb aquestes habilitats, estàs ben preparat per crear i desplegar aplicacions JavaScript robustes.
