Introducció a les clausures

Les clausures són un concepte fonamental en JavaScript que permeten a les funcions accedir a variables d'un àmbit envoltant, fins i tot després que aquest àmbit hagi tancat. Entendre les clausures és crucial per dominar JavaScript, ja que s'utilitzen àmpliament en diversos patrons i tècniques de programació.

Conceptes clau

  • Àmbit lèxic: Les funcions en JavaScript formen clausures al voltant de l'àmbit en què van ser definides.
  • Àmbit de funció: Les variables definides dins d'una funció no són accessibles fora d'ella, però les clausures permeten que les funcions internes accedeixin a aquestes variables.
  • Estat persistent: Les clausures poden mantenir un estat persistent a través de múltiples invocacions.

Exemple bàsic d'una clausura

Comencem amb un exemple senzill per il·lustrar què és una clausura.

function outerFunction() {
    let outerVariable = 'I am outside!';

    function innerFunction() {
        console.log(outerVariable); // Accedint a outerVariable des de l'àmbit extern
    }

    return innerFunction;
}

const myClosure = outerFunction();
myClosure(); // Mostra: 'I am outside!'

Explicació

  • outerFunction defineix una variable outerVariable.
  • innerFunction es defineix dins de outerFunction i accedeix a outerVariable.
  • outerFunction retorna innerFunction, creant una clausura.
  • Quan myClosure es crida, encara té accés a outerVariable tot i que outerFunction ha acabat d'executar-se.

Usos pràctics de les clausures

Les clausures no són només teòriques; tenen aplicacions pràctiques en la programació del món real.

Privacitat de dades

Les clausures es poden utilitzar per crear variables privades.

function createCounter() {
    let count = 0;

    return {
        increment: function() {
            count++;
            return count;
        },
        decrement: function() {
            count--;
            return count;
        },
        getCount: function() {
            return count;
        }
    };
}

const counter = createCounter();
console.log(counter.increment()); // 1
console.log(counter.increment()); // 2
console.log(counter.decrement()); // 1
console.log(counter.getCount()); // 1

Explicació

  • La funció createCounter crea una variable privada count.
  • L'objecte retornat conté mètodes que formen clausures al voltant de count, permetent un accés controlat a aquesta.

Controladors d'esdeveniments

Les clausures s'utilitzen sovint en controladors d'esdeveniments per mantenir l'estat.

function setupButton() {
    let clickCount = 0;

    document.getElementById('myButton').addEventListener('click', function() {
        clickCount++;
        console.log(`Button clicked ${clickCount} times`);
    });
}

setupButton();

Explicació

  • setupButton inicialitza clickCount.
  • La funció del controlador d'esdeveniments forma una clausura al voltant de clickCount, mantenint el seu estat a través de múltiples clics.

Conceptes avançats

Clausures en bucles

Les clausures poden de vegades conduir a comportaments inesperats en bucles si no es gestionen correctament.

for (var i = 1; i <= 3; i++) {
    setTimeout(function() {
        console.log(i);
    }, 1000);
}
// Mostra: 4, 4, 4

Explicació

  • Les funcions de setTimeout formen clausures al voltant de la mateixa variable i, que acaba sent 4 després del bucle.

Solució amb IIFE

Una Expressió de Funció Immediatament Invocada (IIFE) es pot utilitzar per crear un nou àmbit per a cada iteració.

for (var i = 1; i <= 3; i++) {
    (function(i) {
        setTimeout(function() {
            console.log(i);
        }, 1000);
    })(i);
}
// Mostra: 1, 2, 3

Explicació

  • L'IIFE crea un nou àmbit per a cada iteració, preservant el valor actual de i.

Conclusió

Les clausures són una característica poderosa en JavaScript que permeten a les funcions accedir a variables d'un àmbit envoltant. Permeten la privacitat de dades, mantenen l'estat en operacions asíncrones i són essencials per a molts patrons de programació avançats. Entendre i dominar les clausures millorarà significativament les teves habilitats de programació en JavaScript.
