Introducció

En JavaScript, les variables i els tipus de dades són conceptes fonamentals que formen els blocs de construcció de qualsevol programa. Entendre com declarar, assignar i manipular variables, així com conèixer els diferents tipus de dades disponibles, és crucial per escriure codi efectiu i eficient.

Variables

Declarar Variables

En JavaScript, pots declarar variables utilitzant tres paraules clau: var, let i const.

  • var: La manera més antiga de declarar una variable. Té àmbit de funció.
  • let: Introduït a ES6, té àmbit de bloc.
  • const: També introduït a ES6, declara una variable constant que no es pot reasignar.

Exemples:

var name = "John"; // Àmbit de funció
let age = 30;      // Àmbit de bloc
const PI = 3.14;   // Àmbit de bloc i no es pot reasignar

Àmbit de les Variables

  • Àmbit de Funció: Les variables declarades amb var tenen àmbit a la funció en la qual es declaren.
  • Àmbit de Bloc: Les variables declarades amb let i const tenen àmbit al bloc (per exemple, dins d'un bucle o una instrucció if) en el qual es declaren.

Exemple:

function exampleFunction() {
    var functionScoped = "I am function scoped";
    if (true) {
        let blockScoped = "I am block scoped";
        const alsoBlockScoped = "I am also block scoped";
        console.log(blockScoped); // Funciona
        console.log(alsoBlockScoped); // Funciona
    }
    console.log(functionScoped); // Funciona
    // console.log(blockScoped); // Error: blockScoped no està definit
    // console.log(alsoBlockScoped); // Error: alsoBlockScoped no està definit
}

Tipus de Dades

Tipus de Dades Primitives

JavaScript té diversos tipus de dades primitives:

  • String: Representa dades textuals.
  • Number: Representa tant nombres enters com de punt flotant.
  • Boolean: Representa true o false.
  • Undefined: Representa una variable que ha estat declarada però no assignada un valor.
  • Null: Representa l'absència intencional de qualsevol valor d'objecte.
  • Symbol: Representa un valor únic i immutable (introduït a ES6).
  • BigInt: Representa enters amb precisió arbitrària (introduït a ES2020).

Exemples:

let str = "Hello, World!"; // String
let num = 42;              // Number
let isTrue = true;         // Boolean
let notAssigned;           // Undefined
let emptyValue = null;     // Null
let uniqueId = Symbol("id"); // Symbol
let bigNumber = 1234567890123456789012345678901234567890n; // BigInt

Tipus de Dades No Primitives

JavaScript també té tipus de dades no primitives, principalment objectes.

  • Object: Una col·lecció de propietats, on cada propietat és un parell clau-valor.
  • Array: Un tipus especial d'objecte utilitzat per emmagatzemar col·leccions ordenades de valors.

Exemples:

let person = {
    name: "Alice",
    age: 25
}; // Objecte

let numbers = [1, 2, 3, 4, 5]; // Array

Conversió de Tipus

JavaScript és un llenguatge de tipus lliure, el que significa que pots convertir entre diferents tipus de dades.

  • Conversió Implícita: JavaScript converteix automàticament els tipus quan és necessari.
  • Conversió Explícita: Pots convertir manualment els tipus utilitzant funcions com String(), Number() i Boolean().

Exemples:

let x = "5" + 2; // Conversió implícita: "52" (string)
let y = "5" - 2; // Conversió implícita: 3 (number)

let strToNum = Number("123"); // Conversió explícita: 123 (number)
let numToStr = String(123);   // Conversió explícita: "123" (string)
let boolToNum = Number(true); // Conversió explícita: 1 (number)

Resum

Entendre les variables i els tipus de dades és essencial per a qualsevol programador de JavaScript. Les variables es poden declarar utilitzant var, let o const, cadascuna amb diferents regles d'àmbit. JavaScript admet diversos tipus de dades, incloent-hi tipus primitius com cadenes, nombres i booleans, així com tipus no primitius com objectes i arrays. Saber com treballar amb aquests tipus i convertir entre ells t'ajudarà a escriure codi més robust i flexible.
