Introducció als Mòduls

Els mòduls de JavaScript et permeten descompondre el teu codi en peces més petites i reutilitzables. Això fa que el teu codi sigui més organitzat, fàcil de mantenir i ajuda a evitar conflictes de noms.

Conceptes Clau

  • Mòdul: Un fitxer que conté codi JavaScript que es pot importar i utilitzar en altres fitxers JavaScript.
  • Exportar: La paraula clau utilitzada per fer funcions, objectes o valors primitius disponibles per importar en altres mòduls.
  • Importar: La paraula clau utilitzada per portar funcions, objectes o valors primitius d'altres mòduls.

Crear i Exportar Mòduls

Per crear un mòdul, simplement escrius codi JavaScript en un fitxer separat i utilitzes la paraula clau export per fer parts del codi disponibles per importar.

Exportacions Nomenades

Les exportacions nomenades et permeten exportar múltiples valors d'un mòdul.

// math.js
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;

Exportacions Per Defecte

Les exportacions per defecte s'utilitzen per exportar un sol valor d'un mòdul.

// greet.js
const greet = (name) => `Hola, ${name}!`;
export default greet;

Importar Mòduls

Per utilitzar els valors exportats en un altre fitxer, utilitzes la paraula clau import.

Importar Exportacions Nomenades

Quan importes exportacions nomenades, has d'utilitzar els mateixos noms que els valors exportats.

// main.js
import { add, subtract } from './math.js';

console.log(add(2, 3)); // 5
console.log(subtract(5, 2)); // 3

Importar Exportacions Per Defecte

Quan importes exportacions per defecte, pots utilitzar qualsevol nom que vulguis.

// main.js
import greet from './greet.js';

console.log(greet('Alice')); // Hola, Alice!

Importar Totes les Exportacions Nomenades

Pots importar totes les exportacions nomenades d'un mòdul utilitzant la sintaxi *.

// main.js
import * as math from './math.js';

console.log(math.add(2, 3)); // 5
console.log(math.subtract(5, 2)); // 3

Combinació d'Exportacions Per Defecte i Nomenades

Un mòdul pot tenir tant exportacions per defecte com nomenades.

// utils.js
export const multiply = (a, b) => a * b;
const divide = (a, b) => a / b;
export default divide;
// main.js
import divide, { multiply } from './utils.js';

console.log(multiply(2, 3)); // 6
console.log(divide(6, 2)); // 3

Conceptes Avançats de Mòduls

Importacions Dinàmiques

Les importacions dinàmiques et permeten carregar mòduls de manera condicional o sota demanda.

// main.js
if (condition) {
  import('./moduleA.js').then(moduleA => {
    moduleA.doSomething();
  });
} else {
  import('./moduleB.js').then(moduleB => {
    moduleB.doSomethingElse();
  });
}

Agrupadors de Mòduls

Eines com Webpack, Rollup i Parcel poden agrupar múltiples mòduls en un sol fitxer per al desplegament.

Mòduls ES vs CommonJS

JavaScript té dos sistemes principals de mòduls: Mòduls ES (ESM) i CommonJS (CJS). ESM és l'estàndard per al JavaScript modern, mentre que CJS s'utilitza principalment en Node.js.

| Característica | Mòduls ES (ESM) | CommonJS (CJS) | |-------------------|------------------|----------------| | Sintaxi | import/export| require/module.exports | | Exportació Per Defecte | export default | module.exports = | | Estàtic/Dinàmic | Estàtic | Dinàmic | | Suport al Navegador | Sí | No |

Conclusió

Els mòduls són una característica poderosa en JavaScript que t'ajuden a organitzar i gestionar el teu codi de manera més efectiva. Entenent com crear, exportar i importar mòduls, pots escriure codi més net i mantenible. A mesura que avancis, pots explorar les importacions dinàmiques i els agrupadors de mòduls per optimitzar encara més els teus projectes.
