El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert en JavaScript. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar, tant si ets un principiant com si vols avançar en les teves habilitats.

1. Triar un editor de codi

Un editor de codi és on escriuràs el teu codi JavaScript. Aquí tens algunes opcions populars:

  • Visual Studio Code (VS Code)
  • Sublime Text
  • Atom
  • WebStorm

Visual Studio Code (VS Code)

VS Code és altament recomanat per les seves àmplies característiques i suport comunitari. Configurem-lo:

  1. Descarregar i instal·lar VS Code:

    • Ves al lloc web de VS Code.
    • Descarrega l'instal·lador per al teu sistema operatiu (Windows, macOS o Linux).
    • Segueix les instruccions d'instal·lació.

  2. Instal·lar extensions:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca i instal·la les següents extensions:
      • ESLint: Eina de linting per identificar i corregir problemes en el teu codi JavaScript.
      • Prettier: Formatador de codi per assegurar un estil de codi consistent.
      • Live Server: Llança un servidor de desenvolupament local amb funció de recàrrega en viu per a pàgines estàtiques i dinàmiques.

2. Configurar Node.js i npm

Node.js és un entorn d'execució de JavaScript que et permet executar JavaScript al costat del servidor. npm (Node Package Manager) és un gestor de paquets per a JavaScript, que et permet instal·lar biblioteques i eines.

  1. Descarregar i instal·lar Node.js:

    • Ves al lloc web de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Verificar la instal·lació:

    • Obre el teu terminal (Command Prompt, PowerShell o Terminal).
    • Executa les següents comandes per comprovar la instal·lació: 
      node -v
npm -v
    • Hauries de veure els números de versió de Node.js i npm.

3. Configurar Git i GitHub

El control de versions és essencial per gestionar la teva base de codi. Git és un sistema de control de versions, i GitHub és una plataforma per allotjar i col·laborar en repositoris de Git.

  1. Descarregar i instal·lar Git:

    • Ves al lloc web de Git.
    • Descarrega l'instal·lador per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Configurar Git:

    • Obre el teu terminal.
    • Configura el teu nom d'usuari i correu electrònic: 
      git config --global user.name "El teu nom"
git config --global user.email "el.teu.email@example.com"

  3. Crear un compte de GitHub:

    • Ves al lloc web de GitHub.
    • Registra't per obtenir un compte gratuït.

  4. Connectar Git a GitHub:

    • Genera una clau SSH: 
      ssh-keygen -t rsa -b 4096 -C "el.teu.email@example.com"
    • Afegeix la clau SSH al teu compte de GitHub copiant la clau de ~/.ssh/id_rsa.pub i enganxant-la a la secció de claus SSH de la configuració del teu compte de GitHub.

4. Crear el teu primer projecte JavaScript

Ara que el teu entorn està configurat, creem un projecte JavaScript senzill.

  1. Crear un directori de projecte:

    • Obre el teu terminal.
    • Crea un nou directori i navega-hi: 
      mkdir my-js-project
cd my-js-project

  2. Inicialitzar un repositori Git:

    • Inicialitza un nou repositori Git: 
      git init

  3. Crear un fitxer JavaScript:

    • Obre VS Code.
    • Crea un nou fitxer anomenat app.js al teu directori de projecte.
    • Afegeix el següent codi a app.js: 
      console.log('Hola, Món!');

  4. Executar el teu fitxer JavaScript:

    • Obre el teu terminal.
    • Executa el fitxer JavaScript utilitzant Node.js: 
      node app.js
    • Hauries de veure Hola, Món! imprès al terminal.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu camí per convertir-te en un desenvolupador JavaScript expert. Triant les eines adequades i configurant-les correctament, pots optimitzar el teu flux de treball i centrar-te en escriure bon codi. En aquesta secció, hem cobert:

  • Triar i configurar un editor de codi (VS Code).
  • Instal·lar Node.js i npm.
  • Configurar Git i GitHub per al control de versions.
  • Crear i executar el teu primer projecte JavaScript.

Amb el teu entorn preparat, ara estàs llest per aprofundir en la programació en JavaScript. Feliç codificació!
