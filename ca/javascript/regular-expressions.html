Introducció a les Expressions Regulars

Les expressions regulars (regex) són patrons utilitzats per coincidir combinacions de caràcters en cadenes. En JavaScript, les expressions regulars són objectes que es poden utilitzar per realitzar funcions de coincidència de patrons i "cerca i reemplaçament" en text.

  • Sintaxi Bàsica: Les expressions regulars es creen de dues maneres:
    • Notació literal: Encloses entre barres (/patró/).
    • Funció constructora: Creat amb el constructor RegExp (new RegExp('patró')).

Exemple:

// Notació literal
const regex1 = /hello/;

// Funció constructora
const regex2 = new RegExp('hello');

Patrons Bàsics

  • Caràcters Literals: Coincideix amb els caràcters exactes en el patró.
  • Metacaràcters: Caràcters especials que tenen un significat únic en regex.

Metacaràcters Comuns:

  • .: Coincideix amb qualsevol caràcter excepte el salt de línia.
  • \d: Coincideix amb qualsevol dígit (equivalent a [0-9]).
  • \w: Coincideix amb qualsevol caràcter de paraula (alfanumèric + guió baix).
  • \s: Coincideix amb qualsevol caràcter d'espai en blanc (espais, tabuladors, salts de línia).
  • \b: Coincideix amb un límit de paraula.
  • ^: Coincideix amb el començament d'una cadena.
  • $: Coincideix amb el final d'una cadena.

Exemple:

const regex = /\d\w\s/;
const str = "3a ";
console.log(regex.test(str)); // true

Classes de Caràcters i Rangs

  • Classes de Caràcters: Definides utilitzant claudàtors [] per coincidir amb qualsevol dels caràcters inclosos.
  • Rangs: Definits utilitzant un guió - dins dels claudàtors per especificar un rang de caràcters.

Exemple:

const regex = /[a-z]/; // Coincideix amb qualsevol lletra minúscula
const str = "hello";
console.log(regex.test(str)); // true

Quantificadors

Els quantificadors especifiquen quantes vegades un caràcter o grup ha de coincidir.

Quantificadors Comuns:

  • *: Coincideix 0 o més vegades.
  • +: Coincideix 1 o més vegades.
  • ?: Coincideix 0 o 1 vegada.
  • {n}: Coincideix exactament n vegades.
  • {n,}: Coincideix n o més vegades.
  • {n,m}: Coincideix entre n i m vegades.

Exemple:

const regex = /a{2,4}/; // Coincideix amb 'a' entre 2 i 4 vegades
const str = "aaa";
console.log(regex.test(str)); // true

Grups i Alternança

  • Grups: Els parèntesis () s'utilitzen per crear grups.
  • Alternança: La barra vertical | s'utilitza per coincidir amb una cosa o una altra.

Exemple:

const regex = /(cat|dog)/; // Coincideix amb 'cat' o 'dog'
const str = "I have a cat";
console.log(regex.test(str)); // true

Bandes

Les bandes són paràmetres opcionals que modifiquen el comportament del regex.

Bandes Comunes:

  • g: Cerca global.
  • i: Cerca insensible a majúscules i minúscules.
  • m: Cerca multi-línia.

Exemple:

const regex = /hello/i; // Cerca insensible a majúscules i minúscules
const str = "Hello";
console.log(regex.test(str)); // true

Mètodes per a Expressions Regulars

  • test(): Prova una coincidència en una cadena. Retorna true o false.
  • exec(): Executa una cerca d'una coincidència en una cadena. Retorna una matriu d'informació o null.

Exemple:

const regex = /hello/;
const str = "hello world";
console.log(regex.test(str)); // true
console.log(regex.exec(str)); // ["hello"]

Mètodes de Cadena amb Expressions Regulars

  • match(): Recupera les coincidències quan es compara una cadena amb un regex.
  • replace(): Reemplaça subcadenes coincidents amb una nova subcadena.
  • search(): Prova una coincidència en una cadena i retorna l'índex de la coincidència.
  • split(): Divideix una cadena en una matriu de subcadenes utilitzant un regex.

Exemple:

const str = "hello world";
const regex = /o/;

console.log(str.match(regex)); // ["o"]
console.log(str.replace(regex, 'a')); // "hella world"
console.log(str.search(regex)); // 4
console.log(str.split(regex)); // ["hell", " w", "rld"]

Patrons Avançats

  • Lookaheads i Lookbehinds: Afirmacions que coincideixen amb un grup abans o després d'un patró principal sense incloure'l en el resultat.
    • Lookahead Positiu: (?=...)
    • Lookahead Negatiu: (?!...)
    • Lookbehind Positiu: (?<=...)
    • Lookbehind Negatiu: (?<!...)

Exemple:

const regex = /foo(?=bar)/; // Coincideix amb 'foo' només si va seguit de 'bar'
const str = "foobar";
console.log(regex.test(str)); // true

Conclusió

Les expressions regulars són una eina poderosa per a la coincidència de patrons i la manipulació de text en JavaScript. Entenent la sintaxi bàsica, les classes de caràcters, els quantificadors, els grups, l'alternança, les bandes i els mètodes, pots realitzar tasques complexes de processament de text de manera eficient. Practica utilitzant expressions regulars amb diferents exemples per convertir-te en un expert en el seu ús.
