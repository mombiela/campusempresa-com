Introducció a la Validació de Formularis

La validació de formularis és un aspecte crucial del desenvolupament web que assegura que les dades introduïdes pels usuaris són exactes i compleixen els criteris requerits abans de ser enviades al servidor. Això ajuda a mantenir la integritat de les dades i a proporcionar una millor experiència d'usuari.

Conceptes Clau

  • Validació del Costat del Client: Validació realitzada al navegador de l'usuari abans que les dades siguin enviades al servidor.
  • Validació del Costat del Servidor: Validació realitzada al servidor després que les dades siguin enviades.
  • Validació HTML5: Funcions de validació integrades proporcionades per HTML5.
  • Validació Personalitzada: Regles i lògica personalitzades implementades amb JavaScript.

Validació de Formularis HTML5

HTML5 proporciona diversos atributs que es poden utilitzar per validar les entrades del formulari sense escriure cap JavaScript.

Exemple

<form id="myForm">
  <label for="username">Nom d'usuari:</label>
  <input type="text" id="username" name="username" required minlength="3" maxlength="15">
  
  <label for="email">Correu electrònic:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="age">Edat:</label>
  <input type="number" id="age" name="age" min="18" max="100">
  
  <input type="submit" value="Enviar">
</form>

Explicació

  • required: Assegura que el camp no estigui buit.
  • minlength i maxlength: Especifica la longitud mínima i màxima de l'entrada.
  • type="email": Assegura que l'entrada sigui una adreça de correu electrònic vàlida.
  • min i max: Especifica els valors mínims i màxims per a l'entrada numèrica.

Validació Bàsica de Formularis amb JavaScript

JavaScript es pot utilitzar per afegir lògica de validació personalitzada que va més enllà del que ofereix HTML5.

Exemple

<form id="myForm">
  <label for="username">Nom d'usuari:</label>
  <input type="text" id="username" name="username">
  
  <label for="email">Correu electrònic:</label>
  <input type="email" id="email" name="email">
  
  <label for="age">Edat:</label>
  <input type="number" id="age" name="age">
  
  <input type="submit" value="Enviar">
</form>

<script>
document.getElementById('myForm').addEventListener('submit', function(event) {
  var username = document.getElementById('username').value;
  var email = document.getElementById('email').value;
  var age = document.getElementById('age').value;
  
  if (username.length < 3 || username.length > 15) {
    alert('El nom d\'usuari ha de tenir entre 3 i 15 caràcters.');
    event.preventDefault();
  }
  
  var emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  if (!emailPattern.test(email)) {
    alert('Si us plau, introdueix una adreça de correu electrònic vàlida.');
    event.preventDefault();
  }
  
  if (age < 18 || age > 100) {
    alert('L\'edat ha de ser entre 18 i 100.');
    event.preventDefault();
  }
});
</script>

Explicació

  • Event Listener: Escolta l'esdeveniment submit del formulari.
  • Lògica de Validació: Comprova si la longitud del nom d'usuari està dins del rang especificat, si el correu electrònic coincideix amb un patró d'expressió regular, i si l'edat està dins del rang especificat.
  • Prevent Default: Evita que el formulari sigui enviat si alguna validació falla.

Validació Avançada de Formularis amb JavaScript

La validació avançada implica una lògica més complexa, com la validació asíncrona, missatges d'error personalitzats i la integració amb biblioteques de tercers.

Exemple

<form id="myForm">
  <label for="username">Nom d'usuari:</label>
  <input type="text" id="username" name="username">
  
  <label for="email">Correu electrònic:</label>
  <input type="email" id="email" name="email">
  
  <label for="age">Edat:</label>
  <input type="number" id="age" name="age">
  
  <input type="submit" value="Enviar">
</form>

<script>
async function checkUsernameAvailability(username) {
  // Simula una crida a l'API per comprovar la disponibilitat del nom d'usuari
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(username !== 'takenUsername');
    }, 1000);
  });
}

document.getElementById('myForm').addEventListener('submit', async function(event) {
  var username = document.getElementById('username').value;
  var email = document.getElementById('email').value;
  var age = document.getElementById('age').value;
  
  if (username.length < 3 || username.length > 15) {
    alert('El nom d\'usuari ha de tenir entre 3 i 15 caràcters.');
    event.preventDefault();
    return;
  }
  
  var isAvailable = await checkUsernameAvailability(username);
  if (!isAvailable) {
    alert('El nom d\'usuari ja està agafat.');
    event.preventDefault();
    return;
  }
  
  var emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  if (!emailPattern.test(email)) {
    alert('Si us plau, introdueix una adreça de correu electrònic vàlida.');
    event.preventDefault();
    return;
  }
  
  if (age < 18 || age > 100) {
    alert('L\'edat ha de ser entre 18 i 100.');
    event.preventDefault();
  }
});
</script>

Explicació

  • Validació Asíncrona: Utilitza async/await per gestionar operacions asíncrones, com comprovar la disponibilitat del nom d'usuari.
  • Missatges d'Error Personalitzats: Proporciona retroalimentació específica a l'usuari basada en els resultats de la validació.

Resum

La validació de formularis és una part essencial del desenvolupament web que assegura la integritat de les dades i millora l'experiència de l'usuari. HTML5 proporciona funcions bàsiques de validació, però JavaScript permet una lògica de validació més complexa i personalitzada. Entenent i implementant tècniques de validació bàsiques i avançades, els desenvolupadors poden crear formularis robustos i fàcils d'utilitzar.
