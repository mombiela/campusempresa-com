Introducció a les Classes

Les classes en JavaScript són un plànol per crear objectes. Encapsulen dades amb codi per treballar amb aquestes dades. Tot i que JavaScript és un llenguatge basat en prototips, les classes proporcionen una sintaxi més familiar per crear objectes i tractar amb la herència.

Conceptes Clau

  • Declaració de Classe: La sintaxi per definir una classe.
  • Mètode Constructor: Un mètode especial per crear i inicialitzar un objecte creat amb una classe.
  • Mètodes: Funcions definides dins d'una classe.
  • Herència: Mecanisme per crear una nova classe basada en una classe existent.

Declaració de Classe

Una classe es defineix utilitzant la paraula clau class seguida del nom de la classe.

class Person {
  // Mètode Constructor
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  // Mètode
  greet() {
    console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
  }
}

// Creant una instància de la classe
const person1 = new Person('Alice', 30);
person1.greet(); // Output: Hello, my name is Alice and I am 30 years old.

Explicació

  • Declaració de Classe: class Person { ... }
  • Mètode Constructor: constructor(name, age) { ... }
  • Mètode: greet() { ... }
  • Creació d'Instància: const person1 = new Person('Alice', 30);

Mètode Constructor

El mètode constructor és un mètode especial per crear i inicialitzar objectes. Es crida automàticament quan es crea una nova instància de la classe.

class Car {
  constructor(brand, model) {
    this.brand = brand;
    this.model = model;
  }

  displayInfo() {
    console.log(`This car is a ${this.brand} ${this.model}.`);
  }
}

const car1 = new Car('Toyota', 'Corolla');
car1.displayInfo(); // Output: This car is a Toyota Corolla.

Explicació

  • Paràmetres del Constructor: brand, model
  • Propietats de la Instància: this.brand, this.model

Mètodes

Els mètodes són funcions definides dins d'una classe. Es poden utilitzar per realitzar accions sobre les dades emmagatzemades a la classe.

class Rectangle {
  constructor(width, height) {
    this.width = width;
    this.height = height;
  }

  // Mètode per calcular l'àrea
  calculateArea() {
    return this.width * this.height;
  }
}

const rectangle1 = new Rectangle(5, 10);
console.log(rectangle1.calculateArea()); // Output: 50

Explicació

  • Definició del Mètode: calculateArea() { ... }
  • Invocació del Mètode: rectangle1.calculateArea()

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe. Això es fa utilitzant la paraula clau extends.

class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks.`);
  }
}

const dog1 = new Dog('Rex');
dog1.speak(); // Output: Rex barks.

Explicació

  • Classe Base: class Animal { ... }
  • Classe Derivada: class Dog extends Animal { ... }
  • Sobreescriptura del Mètode: speak() { ... }

Resum

  • Classes en JavaScript proporcionen un plànol per crear objectes.
  • Els mètodes constructors inicialitzen nous objectes.
  • Els mètodes són funcions definides dins d'una classe per realitzar accions sobre les dades de la classe.
  • L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

Entenent aquests conceptes, pots crear codi més estructurat i reutilitzable en JavaScript. Practica creant classes, utilitzant constructors, definint mètodes i implementant herència per consolidar la teva comprensió.
