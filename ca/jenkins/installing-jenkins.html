Introducció

Jenkins és un servidor d'automatització de codi obert que ajuda a automatitzar les parts del desenvolupament de programari relacionades amb la construcció, prova i desplegament, facilitant la integració contínua i el lliurament continu (CI/CD). Aquesta secció et guiarà a través del procés d'instal·lació de Jenkins en diferents sistemes operatius.

Prerequisits

Abans d'instal·lar Jenkins, assegura't de tenir els següents prerequisits:

  • Un sistema que executi un sistema operatiu compatible (Windows, macOS o Linux).
  • Kit de desenvolupament de Java (JDK) instal·lat (Jenkins requereix Java per funcionar).

Instal·lant Jenkins a Windows

Pas 1: Descarregar Jenkins

  1. Visita la pàgina de descàrrega de Jenkins.
  2. Fes clic a la pestanya "Windows".
  3. Descarrega el paquet d'instal·lació per a Windows (fitxer .msi).

Pas 2: Instal·lar Jenkins

  1. Executa l'instal·lador .msi descarregat.
  2. Segueix l'assistent d'instal·lació:
    • Accepta l'acord de llicència.
    • Tria el directori d'instal·lació.
    • Selecciona els components a instal·lar (es recomanen les opcions per defecte).
    • Fes clic a "Instal·lar" per començar la instal·lació.

Pas 3: Iniciar Jenkins

  1. Després de la instal·lació, Jenkins s'iniciarà automàticament.
  2. Obre un navegador web i navega a http://localhost:8080.
  3. Segueix les instruccions en pantalla per completar la configuració.

Instal·lant Jenkins a macOS

Pas 1: Instal·lar Homebrew

Si no tens Homebrew instal·lat, obre Terminal i executa:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Pas 2: Instal·lar Jenkins

  1. Obre Terminal.
  2. Executa la següent comanda per instal·lar Jenkins: 
    brew install jenkins-lts

Pas 3: Iniciar Jenkins

  1. Inicia Jenkins executant: 
    brew services start jenkins-lts
  2. Obre un navegador web i navega a http://localhost:8080.
  3. Segueix les instruccions en pantalla per completar la configuració.

Instal·lant Jenkins a Linux

Pas 1: Afegir el Repositori de Jenkins

  1. Obre Terminal.
  2. Afegeix el repositori i la clau de Jenkins: 
    wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -
sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'

Pas 2: Instal·lar Jenkins

  1. Actualitza la llista de paquets: 
    sudo apt-get update
  2. Instal·la Jenkins: 
    sudo apt-get install jenkins

Pas 3: Iniciar Jenkins

  1. Inicia el servei de Jenkins: 
    sudo systemctl start jenkins
  2. Habilita Jenkins per iniciar-se en arrencar: 
    sudo systemctl enable jenkins
  3. Obre un navegador web i navega a http://localhost:8080.
  4. Segueix les instruccions en pantalla per completar la configuració.

Configuració Post-Instal·lació

Després d'instal·lar Jenkins, has de realitzar alguns passos inicials de configuració:

  1. Desbloquejar Jenkins:
    • Localitza la contrasenya inicial d'administrador al fitxer especificat a la pantalla de configuració (per exemple, /var/lib/jenkins/secrets/initialAdminPassword).
    • Introdueix la contrasenya a l'assistent de configuració.
  2. Personalitzar Jenkins:
    • Instal·la els complements suggerits o selecciona complements específics segons les teves necessitats.
    • Crea el primer usuari administrador.
    • Configura la URL de la instància.

Conclusió

Instal·lar Jenkins és un procés senzill que varia lleugerament segons el sistema operatiu. Seguint els passos descrits anteriorment, pots configurar Jenkins a Windows, macOS o Linux. Un cop instal·lat, Jenkins proporciona una plataforma potent per automatitzar els teus processos de desenvolupament de programari, permetent la integració contínua i el lliurament continu.
