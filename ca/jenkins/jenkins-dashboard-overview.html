Introducció

El quadre de comandament de Jenkins és el centre neuràlgic per gestionar la teva instància de Jenkins. Proporciona una visió completa de tots els teus projectes, compilacions i configuracions. Entendre el quadre de comandament és crucial per utilitzar Jenkins de manera efectiva, tant si ets un principiant com un usuari avançat.

Components clau del quadre de comandament de Jenkins

Menú principal

  • Quadre de comandament: La vista predeterminada que mostra tots els teus treballs i els seus estats.
  • Nou element: Crear un nou treball o projecte.
  • Persones: Veure i gestionar comptes d'usuari.
  • Historial de compilacions: Una línia de temps de totes les compilacions de tots els treballs.
  • Gestionar Jenkins: Accedir a la configuració i configuracions administratives.
  • Les meves vistes: Personalitzar vistes per a treballs o projectes específics.
  • Credencials: Gestionar credencials per accedir a sistemes externs.
  • Tancar sessió: Sortir de Jenkins.

Llista de treballs

  • Mostra tots els treballs configurats a Jenkins.
  • Mostra l'estat de cada treball (per exemple, èxit, fallada, en progrés).
  • Proporciona accés ràpid a accions específiques del treball com compilar, configurar i eliminar.

Cua de compilació

  • Llista els treballs que estan esperant ser executats.
  • Mostra l'estat actual i la posició a la cua.

Estat de l'executor de compilacions

  • Mostra l'estat dels executors de compilacions (inactiu, compilant, etc.).
  • Indica quins treballs s'estan executant actualment.

Navegant pel quadre de comandament

Veure detalls del treball

Fes clic en qualsevol nom de treball per veure informació detallada sobre aquest treball, incloent:

  • Historial de compilacions: Una llista de totes les compilacions del treball.
  • Espai de treball: Fitxers i directoris utilitzats pel treball.
  • Configurar: Configuracions i configuracions per al treball.
  • Compilar ara: Activar una nova compilació manualment.

Configurant Jenkins

Accedeix a la secció "Gestionar Jenkins" per:

  • Configurar la configuració del sistema.
  • Gestionar plugins.
  • Configurar la seguretat i els permisos d'usuari.
  • Monitoritzar els registres del sistema i el rendiment.

Exemple: Crear un nou treball

  1. Fes clic a "Nou element" al menú principal.
  2. Introdueix un nom per al treball.
  3. Selecciona el tipus de treball (per exemple, projecte Freestyle, Pipeline).
  4. Fes clic a "D'acord" i configura els paràmetres del treball segons sigui necessari.
  5. Desa la configuració i torna al quadre de comandament.
pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                echo 'Building...'
            }
        }
        stage('Test') {
            steps {
                echo 'Testing...'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying...'
            }
        }
    }
}
  • Aquest exemple mostra un script de Pipeline de Jenkins senzill amb tres etapes: Build, Test i Deploy.
  • Cada etapa conté passos que realitzen accions específiques, com ara imprimir missatges a la consola.

Resum

El quadre de comandament de Jenkins és una eina essencial per gestionar el teu entorn de Jenkins. Proporciona una visió clara de tots els treballs, els seus estats i els historials de compilacions. Familiaritzant-te amb els components i la navegació del quadre de comandament, pots gestionar i monitoritzar eficientment els teus projectes de Jenkins. Tant si estàs creant nous treballs, configurant paràmetres o monitoritzant compilacions, el quadre de comandament és la teva interfície principal per a totes les activitats de Jenkins.
