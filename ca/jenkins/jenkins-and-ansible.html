Introducció a Jenkins i Ansible

En aquesta secció, introduirem Jenkins i Ansible, explicant què són i com es poden utilitzar junts per automatitzar i agilitzar els teus processos DevOps.

  • Jenkins: Un servidor d'automatització de codi obert que ajuda a automatitzar les parts del desenvolupament de programari relacionades amb la construcció, prova i desplegament, facilitant la integració contínua i el lliurament continu (CI/CD).
  • Ansible: Una eina d'automatització de codi obert utilitzada per a la gestió de configuracions, desplegament d'aplicacions i automatització de tasques. Utilitza un llenguatge senzill i llegible per humans (YAML) per descriure treballs d'automatització.

Configuració de Jenkins

Abans d'integrar Jenkins amb Ansible, cal tenir Jenkins configurat i en funcionament.

Instal·lació de Jenkins

  1. Descarregar Jenkins: Visita el lloc web de Jenkins i descarrega l'instal·lador adequat per al teu sistema operatiu.
  2. Instal·lar Jenkins: Segueix les instruccions d'instal·lació per al teu sistema operatiu.
  3. Iniciar Jenkins: Un cop instal·lat, inicia el servei de Jenkins.

Configuració de Jenkins

  1. Configuració inicial: Obre un navegador web i navega a http://localhost:8080. Segueix l'assistent de configuració per completar la configuració inicial.
  2. Instal·lar complements: Durant la configuració, instal·la els complements suggerits o selecciona manualment els complements que necessitis.

Configuració d'Ansible

Ansible ha d'estar instal·lat a la màquina on s'està executant Jenkins.

Instal·lació d'Ansible

  1. Utilitzant pip: 
    pip install ansible
  2. Utilitzant el gestor de paquets:
    • Ubuntu: 
      sudo apt update
sudo apt install ansible
    • CentOS: 
      sudo yum install ansible

Integració de Jenkins amb Ansible

Per utilitzar Ansible dins de Jenkins, cal configurar Jenkins per executar playbooks d'Ansible.

Instal·lació del complement d'Ansible a Jenkins

  1. Navegar al Tauler de Jenkins: Ves a Manage Jenkins > Manage Plugins.
  2. Instal·lar el complement d'Ansible: Cerca el complement "Ansible" i instal·la'l.

Configuració d'Ansible a Jenkins

  1. Afegir una instal·lació d'Ansible: Ves a Manage Jenkins > Global Tool Configuration.
  2. Configurar Ansible: A la secció d'Ansible, afegeix una nova instal·lació d'Ansible proporcionant un nom i el camí a l'executable d'Ansible.

Creació d'un treball de Jenkins per executar playbooks d'Ansible

  1. Crear un nou treball: Des del tauler de Jenkins, fes clic a New Item, introdueix un nom i selecciona Freestyle project.
  2. Configurar el treball:
    • Gestió del codi font: Configura el teu repositori de codi font (per exemple, Git).
    • Entorn de construcció: Marca Provide Node & Label si és necessari.
    • Passos de construcció: Afegeix un pas de construcció per Invoke Ansible Playbook.

Exemple: Executar un playbook senzill d'Ansible

  1. Crear un playbook: Crea un playbook senzill d'Ansible anomenat example-playbook.yml. 
    ---
- name: Example Playbook
  hosts: localhost
  tasks:
    - name: Ensure Apache is installed
      apt:
        name: apache2
        state: present
  2. Configurar el treball de Jenkins:
    • Pas de construcció: Afegeix Invoke Ansible Playbook i configura'l per utilitzar example-playbook.yml.

Executar el treball

  1. Construir el treball: Fes clic a Build Now per executar el treball.
  2. Comprovar la sortida: Monitora la sortida de la consola per assegurar-te que el playbook s'executa correctament.

Integració avançada de Jenkins i Ansible

Per a casos d'ús més avançats, pots integrar Jenkins i Ansible per gestionar escenaris de desplegament complexos.

Utilitzar pipelines de Jenkins amb Ansible

  1. Crear un treball de pipeline: Des del tauler de Jenkins, fes clic a New Item, introdueix un nom i selecciona Pipeline.
  2. Definir el script del pipeline: 
    pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo.git'
            }
        }
        stage('Run Ansible Playbook') {
            steps {
                ansiblePlaybook playbook: 'example-playbook.yml'
            }
        }
    }
}

Exemple: Desplegar una aplicació web

  1. Playbook per al desplegament: 
    ---
- name: Deploy Web Application
  hosts: webservers
  tasks:
    - name: Ensure Nginx is installed
      apt:
        name: nginx
        state: present
    - name: Deploy application code
      copy:
        src: /path/to/app
        dest: /var/www/html
  2. Script del pipeline: 
    pipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo.git'
            }
        }
        stage('Run Ansible Playbook') {
            steps {
                ansiblePlaybook playbook: 'deploy-webapp.yml'
            }
        }
    }
}

Conclusió

Integrar Jenkins i Ansible et permet automatitzar processos de desplegament complexos, assegurant consistència i fiabilitat. Seguint aquesta guia, hauràs de poder configurar Jenkins, configurar-lo per utilitzar Ansible, i crear treballs i pipelines que executin playbooks d'Ansible. Aquesta integració és una eina poderosa en la caixa d'eines DevOps, permetent fluxos de treball CI/CD sense problemes.
