Personalització de la interfície de Jenkins
Escriure plugins de Jenkins ►

Personalitzar la interfície de Jenkins pot millorar significativament l'experiència de l'usuari i l'eficiència del teu equip. Aquest tema et guiarà a través de diverses maneres de personalitzar i millorar la interfície de Jenkins, des de modificacions bàsiques fins a personalitzacions avançades.

Introducció a la personalització de la interfície de Jenkins

  • Per què personalitzar la interfície de Jenkins?
    • Millorar l'experiència de l'usuari.
    • Augmentar la productivitat adaptant la interfície a les teves necessitats.
    • Reflectir la marca de la teva organització.

Personalitzacions bàsiques

Canviant el tema

Jenkins et permet canviar l'aspecte de la interfície utilitzant temes.

  • Utilitzant el Simple Theme Plugin:
    1. Instal·la el Simple Theme Plugin:
      • Navega a Manage Jenkins > Manage Plugins.
      • Cerca "Simple Theme Plugin" i instal·la'l.
    2. Configura el plugin:
      • Ves a Manage Jenkins > Configure System.
      • Desplaça't cap avall fins a la secció "Theme".
      • Introdueix l'URL del teu fitxer CSS personalitzat.
/* Exemple de CSS per canviar el color de fons */
body {
    background-color: #f0f0f0;
}

Personalitzant el tauler de control

  • Utilitzant el Dashboard View Plugin:
    1. Instal·la el Dashboard View Plugin:
      • Navega a Manage Jenkins > Manage Plugins.
      • Cerca "Dashboard View Plugin" i instal·la'l.
    2. Crea un nou tauler de control:
      • Ves a la pàgina principal de Jenkins.
      • Fes clic a New View.
      • Selecciona Dashboard i configura els widgets segons sigui necessari.

Personalitzacions intermèdies

Personalitzant les vistes de treballs

  • Utilitzant el View Job Filters Plugin:
    1. Instal·la el View Job Filters Plugin:
      • Navega a Manage Jenkins > Manage Plugins.
      • Cerca "View Job Filters Plugin" i instal·la'l.
    2. Configura els filtres de treballs:
      • Ves a Manage Jenkins > Configure System.
      • Afegeix filtres per organitzar els treballs segons criteris com l'estat, el nom, etc.

Afegint logotips i icones personalitzats

  • Utilitzant el Custom Logo Plugin:
    1. Instal·la el Custom Logo Plugin:
      • Navega a Manage Jenkins > Manage Plugins.
      • Cerca "Custom Logo Plugin" i instal·la'l.
    2. Configura el logotip:
      • Ves a Manage Jenkins > Configure System.
      • Desplaça't cap avall fins a la secció "Custom Logo".
      • Carrega el teu fitxer de logotip.

Personalitzacions avançades

Personalitzant Jenkins amb scripts Groovy

  • Utilitzant la consola d'scripts:
    • Jenkins proporciona una consola d'scripts per executar scripts Groovy per personalitzar la interfície i realitzar tasques administratives.
    • Accedeix a la consola d'scripts:
      • Navega a Manage Jenkins > Script Console.
// Exemple d'script Groovy per canviar el missatge del sistema
Jenkins.instance.setSystemMessage("Benvingut a la interfície personalitzada de Jenkins!")

Creant plugins personalitzats

  • Desenvolupant un plugin personalitzat:
    • Jenkins et permet desenvolupar plugins personalitzats per ampliar la seva funcionalitat.
    • Passos bàsics per crear un plugin personalitzat:
      1. Configura un entorn de desenvolupament.
      2. Utilitza l'Arquetip de Plugin de Jenkins per crear un nou projecte de plugin.
      3. Implementa la funcionalitat desitjada.
      4. Empaqueta i instal·la el plugin.
// Exemple de codi Java per a un plugin senzill de Jenkins
package org.jenkinsci.plugins.sample;

import hudson.Extension;
import hudson.model.RootAction;

@Extension
public class SamplePlugin implements RootAction {
    public String getIconFileName() {
        return "/plugin/sample/icon.png";
    }

    public String getDisplayName() {
        return "Sample Plugin";
    }

    public String getUrlName() {
        return "sample";
    }
}

Conclusió

Personalitzar la interfície de Jenkins pot millorar enormement la usabilitat i l'eficiència de la teva pipeline de CI/CD. Des de canvis simples de tema fins a scripts Groovy avançats i plugins personalitzats, hi ha nombroses maneres d'adaptar Jenkins a les necessitats del teu equip. Comença amb personalitzacions bàsiques i gradualment explora opcions més avançades a mesura que et familiaritzis amb Jenkins.

Al final d'aquest tema, hauries de ser capaç de:

  • Canviar el tema i el tauler de control de Jenkins.
  • Personalitzar les vistes de treballs i afegir logotips personalitzats.
  • Utilitzar scripts Groovy per a personalitzacions avançades.
  • Desenvolupar i instal·lar plugins personalitzats de Jenkins.

Feliç personalització!
