Introducció a Jenkins i Docker

En aquesta secció, explorarem els conceptes bàsics de Jenkins i Docker, i com es poden utilitzar junts per optimitzar la teva pipeline de CI/CD.

Què és Jenkins?

  • Jenkins és un servidor d'automatització de codi obert.
  • Ajuda a automatitzar parts del procés de desenvolupament de programari, incloent la construcció, prova i desplegament de codi.
  • Jenkins suporta la integració contínua (CI) i el lliurament continu (CD).

Què és Docker?

  • Docker és una plataforma per desenvolupar, enviar i executar aplicacions dins de contenidors.
  • Els contenidors són lleugers, portables i asseguren la consistència a través de diferents entorns.

Configuració de Jenkins amb Docker

En aquesta secció, veurem com configurar Jenkins utilitzant Docker.

Prerequisits

  • Comprensió bàsica de Docker i Jenkins.
  • Docker instal·lat a la teva màquina.

Executar Jenkins en un Contenidor Docker

  1. Descarregar la Imatge Docker de Jenkins: 
    docker pull jenkins/jenkins:lts
  2. Executar el Contenidor de Jenkins: 
    docker run -d -p 8080:8080 -p 50000:50000 --name jenkins jenkins/jenkins:lts
    • -d: Executar el contenidor en mode desenganxat.
    • -p 8080:8080: Mapar el port 8080 de l'amfitrió al port 8080 del contenidor.
    • -p 50000:50000: Mapar el port 50000 de l'amfitrió al port 50000 del contenidor.
    • --name jenkins: Anomenar el contenidor "jenkins".

Accedir a Jenkins

  • Obre un navegador web i navega a http://localhost:8080.
  • Segueix les instruccions en pantalla per completar la configuració.

Integració de Docker amb Jenkins

En aquesta secció, aprendrem com integrar Docker amb Jenkins per construir i desplegar aplicacions.

Instal·lació del Plugin Docker a Jenkins

  1. Navegar al Tauler de Jenkins:
    • Ves a Manage Jenkins > Manage Plugins.
  2. Instal·lar el Plugin Docker:
    • Cerca "Docker" i instal·la el plugin Docker.

Configuració de Docker a Jenkins

  1. Navegar al Tauler de Jenkins:
    • Ves a Manage Jenkins > Configure System.
  2. Afegir Amfitrió Docker:
    • Sota Cloud, fes clic a Add a new cloud i selecciona Docker.
    • Configura els paràmetres de l'amfitrió Docker (per exemple, URL de Docker, credencials).

Crear una Pipeline de Jenkins amb Docker

  1. Crear un Nou Treball de Pipeline:
    • Ves a New Item, introdueix un nom i selecciona Pipeline.
  2. Definir l'Script de Pipeline:
    • Utilitza el següent script d'exemple per construir i executar un contenidor Docker: 
      pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                script {
                    docker.build('my-app-image')
                }
            }
        }
        stage('Run') {
            steps {
                script {
                    docker.image('my-app-image').inside {
                        sh 'echo "Running inside Docker container"'
                    }
                }
            }
        }
    }
}

Ús Avançat de Jenkins i Docker

En aquesta secció, veurem temes més avançats, com utilitzar Docker Compose amb Jenkins i configurar aplicacions multi-contenidor.

Utilitzar Docker Compose en Pipelines de Jenkins

  1. Instal·lar Docker Compose:
    • Assegura't que Docker Compose està instal·lat al teu servidor Jenkins.
  2. Definir el Fitxer Docker Compose:
    • Crea un fitxer docker-compose.yml per a la teva aplicació. 
      version: '3'
services:
  web:
    image: nginx
    ports:
      - "80:80"
  db:
    image: mysql
    environment:
      MYSQL_ROOT_PASSWORD: example
  3. Actualitzar l'Script de Pipeline de Jenkins:
    • Utilitza el següent script d'exemple per executar Docker Compose: 
      pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                script {
                    sh 'docker-compose build'
                }
            }
        }
        stage('Run') {
            steps {
                script {
                    sh 'docker-compose up -d'
                }
            }
        }
    }
}

Conclusió

En aquest tema, hem cobert la integració de Jenkins i Docker, des de la configuració bàsica fins a l'ús avançat. Utilitzant Docker dins de Jenkins, pots crear una pipeline de CI/CD robusta i flexible que assegura la consistència a través de diferents entorns. Aquesta integració permet una construcció, prova i desplegament eficients d'aplicacions, accelerant el procés de desenvolupament i millorant la qualitat del programari.
