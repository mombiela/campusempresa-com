Introducció als plugins de Jenkins

Els plugins de Jenkins amplien la funcionalitat de Jenkins, permetent integrar diverses eines, afegir noves característiques i personalitzar la teva pipeline CI/CD. Entendre com utilitzar i gestionar els plugins és essencial per aprofitar tot el potencial de Jenkins.

Conceptes clau

  • Gestor de plugins: La interfície dins de Jenkins per instal·lar, actualitzar i gestionar plugins.
  • Dependències de plugins: Alguns plugins requereixen altres plugins per funcionar correctament.
  • Actualitzacions de plugins: Mantenir els plugins actualitzats per garantir la seguretat i la funcionalitat.
  • Plugins personalitzats: Crear els teus propis plugins per satisfer necessitats específiques.

Instal·lació de plugins de Jenkins

Utilitzant el gestor de plugins

  1. Navega a Manage Jenkins > Manage Plugins.
  2. Ves a la pestanya Available.
  3. Utilitza la barra de cerca per trobar el plugin desitjat.
  4. Selecciona el plugin i fes clic a Install without restart o Download now and install after restart.

Exemple: Instal·lació del plugin Git

1. Ves a `Manage Jenkins` > `Manage Plugins`.
2. Fes clic a la pestanya `Available`.
3. Cerca `Git Plugin`.
4. Marca la casella al costat de `Git Plugin`.
5. Fes clic a `Install without restart`.
6. Espera que la instal·lació es completi.

Configuració de plugins de Jenkins

Exemple: Configuració del plugin Git

  1. Després de la instal·lació, navega a Manage Jenkins > Configure System.
  2. Desplaça't cap avall fins a la secció Git.
  3. Configura els paràmetres globals com el camí de l'executable de Git si és necessari.
  4. Desa la configuració.

Exemple de bloc de codi de configuració

// Aquest és un exemple de script Groovy per configurar els paràmetres del plugin Git
import jenkins.model.*
import hudson.plugins.git.*

def instance = Jenkins.getInstance()

def gitDescriptor = instance.getDescriptorByType(GitSCM.DescriptorImpl.class)
gitDescriptor.setGlobalConfigName("Your Name")
gitDescriptor.setGlobalConfigEmail("your.email@example.com")

instance.save()

Gestió de dependències de plugins

  • Gestió automàtica: Jenkins gestiona la majoria de les dependències automàticament durant la instal·lació del plugin.
  • Gestió manual: De vegades, pot ser necessari instal·lar manualment les dependències si la instal·lació automàtica falla.

Exemple: Comprovar les dependències dels plugins

  1. Ves a Manage Jenkins > Manage Plugins.
  2. Fes clic a la pestanya Installed.
  3. Troba el plugin i fes clic a l'enllaç Dependencies per veure els plugins requerits.

Actualització de plugins de Jenkins

  1. Navega a Manage Jenkins > Manage Plugins.
  2. Ves a la pestanya Updates.
  3. Selecciona els plugins que vols actualitzar.
  4. Fes clic a Download now and install after restart o Install without restart.

Exemple: Actualització de tots els plugins

1. Ves a `Manage Jenkins` > `Manage Plugins`.
2. Fes clic a la pestanya `Updates`.
3. Marca la casella `Select All`.
4. Fes clic a `Download now and install after restart`.
5. Reinicia Jenkins per completar les actualitzacions.

Creació de plugins personalitzats de Jenkins

  • Entorn de desenvolupament de plugins: Configura un entorn de desenvolupament utilitzant eines com Maven.
  • Estructura del plugin: Entén l'estructura bàsica d'un plugin de Jenkins (per exemple, pom.xml, src/main/java, src/main/resources).
  • Desenvolupament del plugin: Escriu el codi del plugin, empaqueta'l i prova'l en una instància de Jenkins.

Exemple: Estructura bàsica del plugin

my-plugin/
├── pom.xml
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/
│   │   │       └── example/
│   │   │           └── MyPlugin.java
│   │   └── resources/
│   │       └── index.jelly
│   └── test/
│       └── java/
│           └── com/
│               └── example/
│                   └── MyPluginTest.java

Resum

Els plugins de Jenkins són eines poderoses que milloren les capacitats de la teva instància de Jenkins. Entenent com instal·lar, configurar, gestionar dependències, actualitzar i fins i tot crear plugins personalitzats, pots adaptar Jenkins per satisfer les teves necessitats específiques de CI/CD. Actualitzar regularment els plugins i gestionar les dependències garanteix un entorn de Jenkins fluid i segur.
