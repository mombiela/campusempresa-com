Visió General

En aquesta secció, revisarem els conceptes clau coberts al llarg del curs i proporcionarem exercicis pràctics per reforçar la vostra comprensió. Això us ajudarà a consolidar els vostres coneixements i assegurar-vos que esteu ben preparats per utilitzar Jenkins en escenaris del món real.

Exercicis Pràctics

Exercici 1: Creació Bàsica de Treball

Creeu un treball simple de Jenkins que imprimeixi "Hola, Jenkins!" a la consola.

pipeline {
    agent any
    stages {
        stage('Hola') {
            steps {
                echo 'Hola, Jenkins!'
            }
        }
    }
}

Exercici 2: Construint una Aplicació Java

Configureu un pipeline de Jenkins per construir una aplicació Java utilitzant Maven.

pipeline {
    agent any
    tools {
        maven 'Maven 3.6.3' // Assegureu-vos que Maven estigui instal·lat i configurat a Jenkins
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-java-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
    }
}

Exercici 3: Utilitzant Docker amb Jenkins

Creeu un pipeline de Jenkins que construeixi una imatge Docker i la pugi a Docker Hub.

pipeline {
    agent any
    environment {
        DOCKERHUB_CREDENTIALS = credentials('dockerhub-credentials-id')
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-docker-app.git'
            }
        }
        stage('Build Docker Image') {
            steps {
                script {
                    dockerImage = docker.build("your-dockerhub-username/your-docker-app:${env.BUILD_ID}")
                }
            }
        }
        stage('Push Docker Image') {
            steps {
                script {
                    docker.withRegistry('https://index.docker.io/v1/', 'DOCKERHUB_CREDENTIALS') {
                        dockerImage.push()
                    }
                }
            }
        }
    }
}

Exercici 4: Configurant un Pipeline de CI/CD

Creeu un pipeline complet de CI/CD que inclogui etapes de checkout de codi, construcció, prova i desplegament.

pipeline {
    agent any
    tools {
        maven 'Maven 3.6.3'
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-java-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Desplegant aplicació...'
                // Afegiu aquí els vostres passos de desplegament
            }
        }
    }
}

Resum

En aquesta secció, hem revisat els conceptes essencials de Jenkins, incloent la creació de treballs, pipelines, plugins i pràctiques de CI/CD. Els exercicis pràctics han proporcionat experiència pràctica amb la creació de treballs i pipelines de Jenkins, la integració amb sistemes de control de versions, l'ús de Docker i la configuració d'un pipeline complet de CI/CD. En completar aquests exercicis, ara hauríeu de tenir una comprensió sòlida de com utilitzar Jenkins per automatitzar els vostres processos de desenvolupament de programari de manera efectiva.
