Introducció a App Engine

Google App Engine és una plataforma sense servidor totalment gestionada per desenvolupar i allotjar aplicacions web a escala. Abstrau la gestió de la infraestructura, permetent als desenvolupadors centrar-se en escriure codi.

Conceptes Clau

  • Totalment Gestionat: No cal gestionar servidors, xarxes o emmagatzematge.
  • Escalabilitat: Escala automàticament la teva aplicació amunt i avall segons el trànsit.
  • Suporta Múltiples Llenguatges: Python, Java, Node.js, Go, PHP, Ruby i .NET.
  • Serveis Integrats: Integra fàcilment amb altres serveis de GCP com Cloud Datastore, Cloud SQL i Cloud Storage.

Configuració del Teu Primer Projecte App Engine

Prerequisits

  • Un compte de Google Cloud Platform.
  • Google Cloud SDK instal·lat a la teva màquina local.

Passos per Crear un Nou Projecte App Engine

  1. Crear un Nou Projecte: 
    gcloud projects create my-app-engine-project
gcloud config set project my-app-engine-project
  2. Activar App Engine: 
    gcloud app create --region=us-central
  3. Desplegar la Teva Aplicació:
    • Crear un simple fitxer de configuració app.yaml: 
      runtime: python39
entrypoint: gunicorn -b :$PORT main:app
    • Crear un simple fitxer main.py: 
      from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
    • Desplegar l'aplicació: 
      gcloud app deploy

App Engine Entorn Estàndard vs. Flexible

Entorn Estàndard

  • Runtimes Predefinits: Limitat a versions específiques dels llenguatges suportats.
  • Escalat Ràpid: Escala ràpidament amunt i avall.
  • Nivell Gratuït: Ofereix un nivell gratuït per aplicacions de baix trànsit.

Entorn Flexible

  • Runtimes Personalitzats: Permet contenidors Docker personalitzats.
  • Major Flexibilitat: Suporta aplicacions més complexes.
  • Temps d'Inici Més Llargs: Temps d'inici lleugerament més llargs en comparació amb l'entorn estàndard.

| Característica | Entorn Estàndard | Entorn Flexible | |------------------------|----------------------|----------------------| | Suport de Llenguatges | Limitat | Extens | | Escalat | Ràpid | Flexible | | Personalització | Limitada | Alta | | Temps d'Inici | Ràpid | Moderat | | Cost | Inferior | Superior |

Funcions Avançades

Divisió de Trànsit

  • Permet dividir el trànsit entre diferents versions de la teva aplicació.

  • Útil per a proves A/B i desplegaments graduals.

    gcloud app services set-traffic default --splits version1=0.5,version2=0.5

Cues de Tasques

  • Permet el processament en segon pla de tasques.
  • Defineix un fitxer queue.yaml: 
    queue:
- name: my-queue
  rate: 5/s

Tasques Programades

  • Programa tasques per executar-se a intervals específics.
  • Defineix un fitxer cron.yaml: 
    cron:
- description: daily summary job
  url: /tasks/summary
  schedule: every 24 hours

Monitorització i Registre

  • Stackdriver Logging: Registre integrat per a les teves aplicacions.
  • Stackdriver Monitoring: Proporciona mètriques i alertes.

Exemple: Veure Registres

gcloud app logs read

Exemple: Configurar Alertes

  • Crea una política d'alerta al Consola de GCP per monitoritzar mètriques específiques com l'ús de CPU o la latència de les sol·licituds.

Conclusió

Google App Engine proporciona una plataforma potent, escalable i totalment gestionada per desplegar aplicacions web. Tant si estàs construint un lloc web senzill com una aplicació complexa, les funcions d'App Engine i les integracions amb altres serveis de GCP el fan una opció versàtil. Comprenent les diferències entre els entorns Estàndard i Flexible, aprofitant funcions avançades com la divisió de trànsit i les cues de tasques, i utilitzant eines de monitorització i registre, pots gestionar i escalar efectivament les teves aplicacions a Google Cloud Platform.
