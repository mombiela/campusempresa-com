Introducció a BigQuery

BigQuery és el magatzem de dades totalment gestionat i sense servidor de Google que permet consultes SQL super-ràpides utilitzant la potència de processament de la infraestructura de Google. Està dissenyat per manejar anàlisis de dades a gran escala.

Conceptes Clau

  • Arquitectura sense servidor: No cal gestionar la infraestructura.
  • Escalabilitat: S'escala automàticament per manejar grans conjunts de dades.
  • Consultes SQL: Utilitza SQL estàndard per a les consultes.
  • Integració: S'integra fàcilment amb altres serveis de GCP.

Configuració de BigQuery

Abans d'entrar en l'emmagatzematge de dades, cal configurar BigQuery en el teu projecte de GCP.

Passos per Configurar BigQuery

  1. Crear un Projecte de GCP: Si no en tens un, crea un nou projecte al Consola de GCP.
  2. Habilitar l'API de BigQuery: Navega al tauler d'API i Serveis i habilita l'API de BigQuery.
  3. Crear un Conjunt de Dades: Els conjunts de dades són contenidors de nivell superior que contenen taules.
  4. Crear una Taula: Les taules contenen les dades reals.
-- Exemple de SQL per crear un conjunt de dades i una taula
CREATE SCHEMA my_dataset;

CREATE TABLE my_dataset.my_table (
  id INT64,
  name STRING,
  age INT64
);

Carregar Dades a BigQuery

BigQuery admet la càrrega de dades des de diverses fonts com ara fitxers CSV, fitxers JSON, Google Cloud Storage i Google Drive.

Carregar Dades des de Google Cloud Storage

  1. Carrega el teu fitxer de dades a Google Cloud Storage.
  2. Utilitza la Consola de BigQuery o CLI per carregar dades.
bq load --source_format=CSV my_dataset.my_table gs://my_bucket/my_data.csv

Carregar Dades des de Fitxers Locals

També pots carregar dades directament des del teu ordinador local.

bq load --source_format=CSV my_dataset.my_table ./my_data.csv

Consultar Dades a BigQuery

BigQuery utilitza SQL estàndard per consultar dades. Aquí tens algunes consultes bàsiques i avançades.

Consultes Bàsiques

-- Selecciona tots els registres
SELECT * FROM my_dataset.my_table;

-- Selecciona columnes específiques
SELECT name, age FROM my_dataset.my_table;

-- Filtra registres
SELECT * FROM my_dataset.my_table WHERE age > 30;

Consultes Avançades

-- Funcions d'agregació
SELECT COUNT(*) AS total_records FROM my_dataset.my_table;

-- Agrupar per
SELECT age, COUNT(*) AS count FROM my_dataset.my_table GROUP BY age;

-- Unir Taules
SELECT a.id, a.name, b.salary
FROM my_dataset.my_table a
JOIN my_dataset.salary_table b
ON a.id = b.id;

Optimitzar Consultes

BigQuery cobra segons la quantitat de dades processades per les teves consultes. Per tant, optimitzar les teves consultes pot estalviar costos i millorar el rendiment.

Consells per a l'Optimització

  • Utilitza SELECT * amb moderació: Selecciona només les columnes que necessites.
  • Filtra aviat: Utilitza claus WHERE per filtrar dades tan aviat com sigui possible.
  • Particiona Taules: La partició pot millorar el rendiment de les consultes i reduir costos.
  • Utilitza la Memòria Cau: BigQuery emmagatzema en memòria cau els resultats de les consultes, que es poden reutilitzar per estalviar costos.
-- Exemple de partició d'una taula
CREATE TABLE my_dataset.partitioned_table
PARTITION BY DATE(timestamp_column)
AS
SELECT * FROM my_dataset.my_table;

Integrar BigQuery amb Altres Serveis de GCP

BigQuery es pot integrar amb diversos altres serveis de GCP per a una funcionalitat millorada.

Exemples d'Integració

  • Dataflow: Per al processament de dades en temps real.
  • Dataproc: Per executar treballs d'Apache Hadoop i Spark.
  • Looker Studio: Per a la visualització i informes de dades.
# Exemple d'ús de BigQuery amb Dataflow (Python)
from apache_beam.io.gcp.bigquery import ReadFromBigQuery
import apache_beam as beam

def run():
    with beam.Pipeline() as p:
        rows = (p
                | 'Read from BigQuery' >> ReadFromBigQuery(query='SELECT * FROM my_dataset.my_table')
                | 'Process Data' >> beam.Map(lambda row: process(row))
                | 'Write to BigQuery' >> beam.io.WriteToBigQuery('my_dataset.processed_table'))

Conclusió

BigQuery és una eina potent per a l'emmagatzematge de dades a GCP, oferint escalabilitat, facilitat d'ús i integració amb altres serveis de GCP. Comprenent com configurar, carregar dades, consultar i optimitzar el teu entorn de BigQuery, pots aprofitar tot el seu potencial per a les teves necessitats d'anàlisi de dades.
