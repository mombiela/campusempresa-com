Introducció a Kubernetes Engine

Google Kubernetes Engine (GKE) és un entorn gestionat i preparat per a la producció per desplegar aplicacions contenidoritzades. Proporciona una manera potent i flexible de gestionar, desplegar i escalar aplicacions utilitzant Kubernetes.

Conceptes Clau

  • Kubernetes: Un sistema de codi obert per automatitzar el desplegament, l'escalat i la gestió d'aplicacions contenidoritzades.
  • GKE: Un servei gestionat de Kubernetes proporcionat per Google Cloud Platform.
  • Contenidors: Paquets de programari lleugers, autònoms i executables que inclouen tot el necessari per executar una peça de programari, incloent el codi, el temps d'execució, les biblioteques i les dependències.

Configuració de GKE

Prerequisits

  • Un compte de Google Cloud Platform.
  • Comprensió bàsica de Docker i Kubernetes.
  • Google Cloud SDK instal·lat a la teva màquina local.

Passos per Configurar GKE

  1. Crear un Projecte GCP:

    gcloud projects create my-gke-project
gcloud config set project my-gke-project

  2. Habilitar l'API de Kubernetes Engine:

    gcloud services enable container.googleapis.com

  3. Crear un Clúster GKE:

    gcloud container clusters create my-cluster --zone us-central1-a

  4. Obtenir Credencials d'Autenticació per al Clúster:

    gcloud container clusters get-credentials my-cluster --zone us-central1-a

  5. Verificar el Clúster:

    kubectl get nodes

Desplegament d'Aplicacions a GKE

Creació d'un Desplegament

  • Fitxer YAML de Desplegament:

    apiVersion: apps/v1
kind: Deployment
metadata:
  name: my-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: my-app
  template:
    metadata:
      labels:
        app: my-app
    spec:
      containers:
      - name: my-app
        image: gcr.io/my-gke-project/my-app:latest
        ports:
        - containerPort: 80

  • Aplicar el Desplegament:

    kubectl apply -f deployment.yaml

Exposar el Desplegament

  • Fitxer YAML de Servei:

    apiVersion: v1
kind: Service
metadata:
  name: my-app-service
spec:
  type: LoadBalancer
  selector:
    app: my-app
  ports:
  - protocol: TCP
    port: 80
    targetPort: 80

  • Aplicar el Servei:

    kubectl apply -f service.yaml

Escalat d'Aplicacions

Escalat Manual

  • Escalar el Desplegament: 
    kubectl scale deployment my-app --replicas=5

Autoscalat

  • Habilitar l'Autoscalat: 
    kubectl autoscale deployment my-app --cpu-percent=80 --min=1 --max=10

Monitorització i Registre

Integració amb Stackdriver

  • Habilitar el Registre i la Monitorització de Stackdriver:

    gcloud container clusters create my-cluster --zone us-central1-a --enable-stackdriver-kubernetes

  • Veure Registres:

    gcloud logging read "resource.type=k8s_container AND resource.labels.cluster_name=my-cluster"

Temes Avançats

Definicions de Recursos Personalitzats (CRDs)

  • Definir un CRD:

    apiVersion: apiextensions.k8s.io/v1
kind: CustomResourceDefinition
metadata:
  name: crontabs.stable.example.com
spec:
  group: stable.example.com
  versions:
  - name: v1
    served: true
    storage: true
  scope: Namespaced
  names:
    plural: crontabs
    singular: crontab
    kind: CronTab
    shortNames:
    - ct

  • Aplicar el CRD:

    kubectl apply -f crd.yaml

Polítiques de Xarxa

  • Definir una Política de Xarxa:

    apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: allow-same-namespace
spec:
  podSelector: {}
  policyTypes:
  - Ingress
  ingress:
  - from:
    - podSelector: {}

  • Aplicar la Política de Xarxa:

    kubectl apply -f network-policy.yaml

Conclusió

Google Kubernetes Engine (GKE) simplifica el desplegament, la gestió i l'escalat d'aplicacions contenidoritzades utilitzant Kubernetes. Aprofitant GKE, els desenvolupadors poden centrar-se en escriure codi i desplegar aplicacions sense preocupar-se per la infraestructura subjacent. Aquest curs ha cobert els conceptes bàsics de la configuració de GKE, el desplegament d'aplicacions, l'escalat, la monitorització i temes avançats com CRDs i polítiques de xarxa. Amb aquest coneixement, estàs ben preparat per gestionar i desplegar aplicacions a GKE de manera efectiva.
