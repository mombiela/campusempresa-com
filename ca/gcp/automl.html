Introducció a AutoML

AutoML (Automated Machine Learning) és un conjunt d'eines i serveis de machine learning que automatitzen el procés d'aplicar machine learning a problemes del món real. Google Cloud Platform (GCP) ofereix una varietat de serveis AutoML que permeten als usuaris construir i desplegar models de machine learning amb un mínim de codificació i experiència en machine learning.

Conceptes Clau

  • Enginyeria de Característiques Automatitzada: Generar i seleccionar automàticament les característiques més rellevants per a un model.
  • Selecció de Model: Seleccionar automàticament la millor arquitectura de model per a un conjunt de dades determinat.
  • Ajust de Hiperparàmetres: Ajustar automàticament els hiperparàmetres d'un model per optimitzar el rendiment.
  • Entrenament de Model: Entrenar el model utilitzant les característiques, l'arquitectura i els hiperparàmetres seleccionats.
  • Avaluació de Model: Avaluar el rendiment del model entrenat utilitzant diverses mètriques.
  • Desplegament de Model: Desplegar el model entrenat per fer prediccions sobre noves dades.

Començar amb AutoML a GCP

Per utilitzar AutoML a GCP, cal configurar un projecte GCP i habilitar les API necessàries.

Passos per Configurar

  1. Crear un Projecte GCP:

    • Aneu a la Consola GCP.
    • Feu clic al desplegable del projecte i seleccioneu "Nou Projecte".
    • Introduïu un nom de projecte i feu clic a "Crear".

  2. Habilitar les API d'AutoML:

    • Navegueu a la secció "APIs & Services".
    • Cerqueu "AutoML" i habiliteu les API rellevants (per exemple, AutoML Vision, AutoML Natural Language).

  3. Configurar la Facturació:

    • Assegureu-vos que la facturació està habilitada per al vostre projecte.

  4. Instal·lar el Google Cloud SDK:

    • Seguiu les instruccions aquí per instal·lar el SDK.

  5. Autenticar el SDK:

    gcloud auth login
gcloud config set project YOUR_PROJECT_ID

AutoML Vision

AutoML Vision us permet construir models personalitzats de classificació d'imatges.

Passos per Crear un Model AutoML Vision

  1. Pujar el Vostre Conjunt de Dades:

    • Prepareu un conjunt de dades amb imatges etiquetades.
    • Pugeu el conjunt de dades a un bucket de Google Cloud Storage.

  2. Crear un Conjunt de Dades a AutoML Vision:

    • Aneu a la Interfície d'Usuari d'AutoML Vision.
    • Feu clic a "Nou Conjunt de Dades" i seguiu les indicacions per importar les vostres imatges des del bucket de Cloud Storage.

  3. Entrenar el Model:

    • Després de crear el conjunt de dades, feu clic a "Entrenar Nou Model".
    • Trieu el tipus de model i el pressupost d'entrenament.
    • Feu clic a "Iniciar Entrenament".

  4. Avaluar el Model:

    • Un cop completat l'entrenament, reviseu les mètriques d'avaluació proporcionades per AutoML Vision.

  5. Desplegar el Model:

    • Feu clic a "Desplegar Model" per fer el model disponible per a prediccions.

Exemple de Codi per Fer Prediccions

from google.cloud import automl_v1beta1 as automl

project_id = "YOUR_PROJECT_ID"
model_id = "YOUR_MODEL_ID"
file_path = "path/to/your/image.jpg"

client = automl.PredictionServiceClient()
model_full_id = client.model_path(project_id, "us-central1", model_id)

with open(file_path, "rb") as image_file:
    content = image_file.read()

payload = {"image": {"image_bytes": content}}
params = {}

response = client.predict(name=model_full_id, payload=payload, params=params)

for result in response.payload:
    print("Nom de la classe predita: {}".format(result.display_name))
    print("Puntuació de la classe predita: {}".format(result.classification.score))

AutoML Natural Language

AutoML Natural Language us permet construir models personalitzats per a la classificació de text, l'extracció d'entitats i l'anàlisi de sentiment.

Passos per Crear un Model AutoML Natural Language

  1. Preparar el Vostre Conjunt de Dades:

    • Creeu un fitxer CSV amb les vostres dades de text i etiquetes.
    • Pugeu el fitxer CSV a un bucket de Google Cloud Storage.

  2. Crear un Conjunt de Dades a AutoML Natural Language:

    • Aneu a la Interfície d'Usuari d'AutoML Natural Language.
    • Feu clic a "Nou Conjunt de Dades" i seguiu les indicacions per importar el vostre fitxer CSV.

  3. Entrenar el Model:

    • Després de crear el conjunt de dades, feu clic a "Entrenar Nou Model".
    • Trieu el tipus de model i el pressupost d'entrenament.
    • Feu clic a "Iniciar Entrenament".

  4. Avaluar el Model:

    • Un cop completat l'entrenament, reviseu les mètriques d'avaluació proporcionades per AutoML Natural Language.

  5. Desplegar el Model:

    • Feu clic a "Desplegar Model" per fer el model disponible per a prediccions.

Exemple de Codi per Fer Prediccions

from google.cloud import automl_v1beta1 as automl

project_id = "YOUR_PROJECT_ID"
model_id = "YOUR_MODEL_ID"
text_content = "El teu text per analitzar"

client = automl.PredictionServiceClient()
model_full_id = client.model_path(project_id, "us-central1", model_id)

payload = {"text_snippet": {"content": text_content, "mime_type": "text/plain"}}
params = {}

response = client.predict(name=model_full_id, payload=payload, params=params)

for result in response.payload:
    print("Nom de la classe predita: {}".format(result.display_name))
    print("Puntuació de la classe predita: {}".format(result.classification.score))

Ajust de Hiperparàmetres

  • Cerca Exhaustiva: Cercar exhaustivament a través d'un subconjunt especificat manualment de l'espai d'hiperparàmetres.

  • Importància de les Característiques: Entendre quines característiques tenen més impacte en les prediccions del model.
  • Valors SHAP: Un mètode per explicar prediccions individuals.

Conclusió

AutoML a GCP proporciona un conjunt d'eines potent per construir i desplegar models de machine learning amb un mínim esforç. Automatitzant molts dels passos complexos implicats en el machine learning, AutoML permet als usuaris centrar-se en resoldre els seus problemes empresarials específics. Tant si treballeu amb imatges, text o dades estructurades, els serveis AutoML de GCP us poden ajudar a crear models d'alta qualitat de manera ràpida i eficient.
