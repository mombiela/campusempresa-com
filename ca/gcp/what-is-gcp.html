Google Cloud Platform (GCP) és un conjunt de serveis de computació en núvol proporcionats per Google. Ofereix una gamma de serveis que inclouen computació, emmagatzematge, anàlisi de dades i aprenentatge automàtic. GCP permet als desenvolupadors construir, desplegar i escalar aplicacions, llocs web i serveis en la mateixa infraestructura que Google utilitza internament.

Conceptes Clau de Google Cloud Platform

  • Computació en Núvol: El lliurament de serveis de computació a través d'internet ("el núvol") per oferir una innovació més ràpida, recursos flexibles i economies d'escala.
  • Infraestructura com a Servei (IaaS): Proporciona recursos de computació virtualitzats a través d'internet. Exemples inclouen Google Compute Engine.
  • Plataforma com a Servei (PaaS): Proporciona una plataforma que permet als clients desenvolupar, executar i gestionar aplicacions sense tractar amb la infraestructura. Exemples inclouen Google App Engine.
  • Programari com a Servei (SaaS): Lliura aplicacions de programari a través d'internet, amb una subscripció. Exemples inclouen Google Workspace.

Serveis Bàsics de GCP

Serveis de Computació

  • Google Compute Engine (GCE): Proporciona màquines virtuals (VMs) escalables que s'executen en la infraestructura de Google.
  • Google Kubernetes Engine (GKE): Servei de Kubernetes gestionat per desplegar, gestionar i escalar aplicacions contenidoritzades.
  • Google App Engine (GAE): Oferta PaaS que permet als desenvolupadors construir i desplegar aplicacions sense gestionar la infraestructura subjacent.

Serveis d'Emmagatzematge

  • Google Cloud Storage: Servei d'emmagatzematge d'objectes per emmagatzemar i accedir a dades en la infraestructura de Google.
  • Google Cloud SQL: Servei de base de dades relacional gestionat per MySQL, PostgreSQL i SQL Server.
  • Google Cloud Spanner: Servei de base de dades distribuïda globalment i escalable horitzontalment.

Serveis de Xarxa

  • Google Virtual Private Cloud (VPC): Proporciona funcionalitat de xarxa als recursos de GCP.
  • Cloud Load Balancing: Distribueix el trànsit entrant entre múltiples instàncies de màquines virtuals.
  • Cloud CDN: Xarxa de distribució de contingut per lliurar contingut als usuaris amb alta disponibilitat i rendiment.

Serveis d'Anàlisi de Dades

  • BigQuery: Magatzem de dades completament gestionat i sense servidor per a l'anàlisi de dades a gran escala.
  • Dataflow: Servei de processament de dades en flux i per lots.
  • Dataproc: Servei gestionat de Spark i Hadoop per al processament de dades massives.

Serveis d'Aprenentatge Automàtic

  • AI Platform: Conjunt d'eines i serveis per construir, desplegar i gestionar models d'aprenentatge automàtic.
  • AutoML: Permet als desenvolupadors amb poca experiència en aprenentatge automàtic entrenar models d'alta qualitat específics per a les seves necessitats.
  • TensorFlow on GCP: Marc d'aprenentatge automàtic de codi obert que es pot utilitzar amb els serveis de GCP.

Exemple: Desplegar una Aplicació Web Simple a Google App Engine

Vegem un exemple simple de desplegar una aplicació web utilitzant Google App Engine.

Pas 1: Configurar el Teu Entorn

Assegura't de tenir el Google Cloud SDK instal·lat i inicialitzat.

# Instal·lar Google Cloud SDK
curl https://sdk.cloud.google.com | bash

# Inicialitzar el SDK
gcloud init

Pas 2: Crear la Teva Aplicació

Crea una aplicació web simple en Python.

# main.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hola, Món!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Pas 3: Crear el Fitxer de Configuració d'App Engine

Crea un fitxer app.yaml per configurar el desplegament del teu App Engine.

# app.yaml
runtime: python39

handlers:
- url: /.*
  script: auto

Pas 4: Desplegar la Teva Aplicació

Desplega l'aplicació a Google App Engine.

# Desplegar l'aplicació
gcloud app deploy

Pas 5: Accedir a la Teva Aplicació

Després del desplegament, pots accedir a la teva aplicació a https://<your-project-id>.appspot.com.

Conclusió

Google Cloud Platform és un conjunt potent de serveis en núvol que permet als desenvolupadors construir, desplegar i escalar aplicacions de manera eficient. Amb una varietat de serveis que van des de la computació i l'emmagatzematge fins a l'anàlisi de dades i l'aprenentatge automàtic, GCP proporciona les eines necessàries tant per a principiants com per a usuaris avançats per aprofitar la computació en núvol de manera efectiva. Entendre els serveis bàsics i com desplegar aplicacions a GCP és el primer pas per dominar aquesta robusta plataforma.
