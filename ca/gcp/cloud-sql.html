Introducció a Cloud SQL

Cloud SQL és un servei de base de dades relacional totalment gestionat per MySQL, PostgreSQL i SQL Server. T'ajuda a configurar, mantenir, gestionar i administrar les teves bases de dades relacionals a Google Cloud Platform.

Conceptes Clau

  • Servei Totalment Gestionat: Cloud SQL automatitza les còpies de seguretat, la replicació i el failover per garantir una alta disponibilitat i fiabilitat.
  • Escalabilitat: Escala fàcilment els recursos de càlcul i emmagatzematge de la teva base de dades amb un temps d'inactivitat mínim.
  • Seguretat: Integrat amb Google Cloud IAM i VPC per a un accés segur i xifrat de dades.
  • Compatibilitat: Suporta MySQL, PostgreSQL i SQL Server.

Configuració de Cloud SQL

Creació d'una Instància de Cloud SQL

  1. Navega a Cloud SQL al Consola de GCP:

    • Ves a la Consola de GCP.
    • Selecciona "SQL" des del menú de l'esquerra.

  2. Crea una Nova Instància:

    • Fes clic a "Create Instance".
    • Tria el motor de base de dades desitjat (MySQL, PostgreSQL o SQL Server).

  3. Configura els Paràmetres de la Instància:

    • Defineix l'ID de la instància, la contrasenya de root i la regió.
    • Configura el tipus de màquina i les opcions d'emmagatzematge.

  4. Finalitza i Crea:

    • Revisa els teus paràmetres i fes clic a "Create".

Exemple: Creació d'una Instància MySQL

gcloud sql instances create my-instance \
    --database-version=MYSQL_5_7 \
    --tier=db-n1-standard-1 \
    --region=us-central1

Connexió a Cloud SQL

  • Utilitzant Cloud Shell: 
    gcloud sql connect my-instance --user=root
  • Utilitzant un Client Local:
    • Instal·la el proxy d'autenticació de Cloud SQL.
    • Inicia el proxy: 
      ./cloud_sql_proxy -instances=<INSTANCE_CONNECTION_NAME>=tcp:3306
    • Connecta't utilitzant el teu client MySQL local: 
      mysql -u root -p --host 127.0.0.1 --port 3306

Gestió d'Instàncies de Cloud SQL

Còpies de Seguretat i Restauracions

  • Còpies de Seguretat Automatitzades: Activa les còpies de seguretat automatitzades per garantir la recuperació de dades.
  • Còpies de Seguretat Manuals: Crea còpies de seguretat manuals per a punts específics en el temps.

Exemple: Activació de Còpies de Seguretat Automatitzades

gcloud sql instances patch my-instance --backup-start-time 23:00

Monitorització i Registre

  • Integració amb Stackdriver: Utilitza Stackdriver per a la monitorització i el registre.
  • Query Insights: Analitza el rendiment de les consultes i optimitza la teva base de dades.

Escalabilitat i Optimització del Rendiment

  • Escalabilitat Vertical: Augmenta el tipus de màquina o la mida de l'emmagatzematge.
  • Escalabilitat Horitzontal: Utilitza rèpliques de lectura per a l'equilibri de càrrega.

Exemple: Escalabilitat de l'Emmagatzematge

gcloud sql instances patch my-instance --storage-size=100

Funcions Avançades de Cloud SQL

Alta Disponibilitat i Replicació

  • Alta Disponibilitat (HA): Configura HA per a un failover automàtic.
  • Rèpliques de Lectura: Crea rèpliques de lectura per distribuir el trànsit de lectura.

Exemple: Creació d'una Rèplica de Lectura

gcloud sql instances create my-replica --master-instance-name=my-instance

Millors Pràctiques de Seguretat

  • Rols IAM: Assigna els rols IAM adequats per al control d'accés.
  • VPC Peering: Utilitza VPC peering per a connexions segures i privades.
  • SSL/TLS: Activa SSL/TLS per a connexions xifrades.

Exemple: Activació de SSL/TLS

gcloud sql instances patch my-instance --require-ssl

Conclusió

Cloud SQL a GCP proporciona una solució robusta, escalable i segura per a la gestió de bases de dades relacionals. Aprofitant els seus serveis totalment gestionats, còpies de seguretat automatitzades, alta disponibilitat i funcions avançades de seguretat, pots centrar-te en desenvolupar les teves aplicacions sense preocupar-te per l'administració de bases de dades. Tant si ets un principiant configurant la teva primera instància com si ets un usuari avançat optimitzant el rendiment i la seguretat, Cloud SQL ofereix les eines i la flexibilitat per satisfer les teves necessitats.
