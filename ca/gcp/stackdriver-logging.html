Introducció a Stackdriver Logging

Stackdriver Logging, ara conegut com Google Cloud Logging, és un servei totalment gestionat que et permet emmagatzemar, cercar, analitzar, monitoritzar i alertar sobre dades de registres i esdeveniments de Google Cloud Platform (GCP) i Amazon Web Services (AWS). Aquesta secció t'introduirà als conceptes bàsics de Stackdriver Logging i la seva importància en entorns de núvol.

  • Conceptes Clau:
    • Entrades de Registre: Registres individuals d'esdeveniments.
    • Destinacions de Registre: Mecanismes per exportar registres a altres serveis.
    • Mètriques Basades en Registres: Mètriques derivades de dades de registres.
    • Vistes de Registres: Filtres per controlar quins registres són visibles.

Configuració de Stackdriver Logging

Abans de poder començar a utilitzar Stackdriver Logging, cal configurar-lo al teu projecte de GCP.

Activació de l'API de Logging

Per utilitzar Stackdriver Logging, cal activar l'API de Logging al teu projecte de GCP.

gcloud services enable logging.googleapis.com

Creació d'un Compte de Servei

Es requereix un compte de servei per interactuar amb Stackdriver Logging programàticament.

gcloud iam service-accounts create logging-sa --display-name "Logging Service Account"

Assignació de Rols

Assigna els rols necessaris al compte de servei per permetre-li escriure registres.

gcloud projects add-iam-policy-binding [PROJECT_ID] \
    --member "serviceAccount:logging-sa@[PROJECT_ID].iam.gserviceaccount.com" \
    --role "roles/logging.logWriter"

Escriure Registres

Pots escriure registres a Stackdriver Logging utilitzant diversos mètodes, incloent la Consola de GCP, l'eina de línia de comandes gcloud i biblioteques de clients.

Escriure Registres Utilitzant la Consola de GCP

  1. Navega a la secció de Logging a la Consola de GCP.
  2. Selecciona el tipus de registre i el recurs adequat.
  3. Escriu una entrada de registre i envia-la.

Escriure Registres Utilitzant l'Eina de Línia de Comandes `gcloud`

gcloud logging write my-log "Hello, Stackdriver Logging!"

Escriure Registres Utilitzant Biblioteques de Clients

Aquí tens un exemple d'escriure registres utilitzant la biblioteca de clients de Python:

from google.cloud import logging

client = logging.Client()
logger = client.logger("my-log")

logger.log_text("Hello, Stackdriver Logging!")

Veure i Analitzar Registres

Un cop els registres estan escrits, pots veure'ls i analitzar-los utilitzant la Consola de GCP o l'eina de línia de comandes gcloud.

Veure Registres a la Consola de GCP

  1. Navega a la secció de Logging a la Consola de GCP.
  2. Utilitza el Visualitzador de Registres per filtrar i cercar registres.

Veure Registres Utilitzant l'Eina de Línia de Comandes `gcloud`

gcloud logging read "logName=projects/[PROJECT_ID]/logs/my-log"

Exportar Registres

Pots exportar registres a altres serveis de GCP com BigQuery, Cloud Storage o Pub/Sub per a una anàlisi més detallada i emmagatzematge a llarg termini.

Creació d'una Destinació de Registre

  1. Navega a la secció de Logging a la Consola de GCP.
  2. Ves a la pestanya "Exports" i crea una nova destinació.
  3. Tria la destinació (BigQuery, Cloud Storage o Pub/Sub).

Exemple: Exportar Registres a BigQuery

gcloud logging sinks create my-sink bigquery.googleapis.com/projects/[PROJECT_ID]/datasets/[DATASET_ID]

Creació de Mètriques Basades en Registres

Les mètriques basades en registres et permeten crear mètriques personalitzades basades en dades de registres, que es poden utilitzar per a monitorització i alertes.

Creació d'una Mètrica Basada en Registres

  1. Navega a la secció de Logging a la Consola de GCP.
  2. Ves a la pestanya "Metrics" i crea una nova mètrica.
  3. Defineix el filtre i el tipus de mètrica.

Exemple: Creació d'una Mètrica Basada en Registres Utilitzant `gcloud`

gcloud logging metrics create my-metric \
    --description="A custom metric based on log data" \
    --log-filter="logName=projects/[PROJECT_ID]/logs/my-log"

Conclusió

Stackdriver Logging és una eina poderosa per gestionar i analitzar dades de registres a GCP. Comprenent com configurar, escriure, veure, exportar i crear mètriques a partir de registres, pots monitoritzar i resoldre problemes de les teves aplicacions i infraestructura de manera efectiva. Tant si ets un principiant com un usuari avançat, dominar Stackdriver Logging millorarà significativament la teva capacitat per gestionar recursos de núvol de manera eficient.
