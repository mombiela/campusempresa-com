Introducció a Dataproc

Google Cloud Dataproc és un servei al núvol ràpid, fàcil d'utilitzar i totalment gestionat per executar clústers d'Apache Spark i Apache Hadoop. Et permet processar grans conjunts de dades de manera ràpida i eficient.

Conceptes Clau

  • Clúster: Una col·lecció de màquines virtuals (VMs) que executen treballs de Hadoop i Spark.
  • Treball: Una unitat de treball que envies a un clúster de Dataproc.
  • Flux de treball: Una seqüència de treballs que es poden executar en un ordre específic.
  • Autoscaling: Ajustar automàticament el nombre de nodes de treball en un clúster segons la càrrega de treball.

Configuració de Dataproc

Creació d'un Clúster de Dataproc

Per crear un clúster de Dataproc, segueix aquests passos:

  1. Navega a la secció de Dataproc al Consola de GCP.

  2. Fes clic a "Crear Clúster".

  3. Configura el clúster:

    • Nom del Clúster: Dona un nom únic al teu clúster.
    • Regió i Zona: Selecciona la regió i zona apropiades.
    • Mode del Clúster: Tria entre Estàndard, Node Únic o Alta Disponibilitat.
    • Configuració de Nodes: Especifica el nombre i tipus de nodes mestre i de treball.
    • Components Opcional: Afegeix components opcionals com Jupyter, Zeppelin, etc.

  4. Fes clic a "Crear" per llançar el clúster.

Exemple: Creació d'un Clúster Usant la Comanda gcloud

gcloud dataproc clusters create my-cluster \
    --region=us-central1 \
    --zone=us-central1-a \
    --master-machine-type=n1-standard-2 \
    --worker-machine-type=n1-standard-2 \
    --num-workers=2

Aquesta comanda crea un clúster de Dataproc anomenat my-cluster amb 1 node mestre i 2 nodes de treball a la zona us-central1-a.

Execució de Treballs a Dataproc

Enviament d'un Treball

Pots enviar treballs a un clúster de Dataproc utilitzant la Consola de GCP, l'eina de línia de comandes gcloud o l'API de Dataproc.

Exemple: Enviament d'un Treball Spark Usant la Comanda gcloud

gcloud dataproc jobs submit spark \
    --cluster=my-cluster \
    --region=us-central1 \
    --class=org.apache.spark.examples.SparkPi \
    --jars=file:///usr/lib/spark/examples/jars/spark-examples.jar \
    -- 1000

Aquesta comanda envia un treball Spark per calcular Pi utilitzant l'exemple SparkPi.

Monitorització de Treballs

Pots monitoritzar l'estat dels teus treballs a la Consola de GCP sota la secció de Dataproc. La consola proporciona registres i mètriques detallades per a cada treball.

Funcions Avançades de Dataproc

Clústers amb Autoscaling

L'autoscaling permet que el teu clúster ajusti automàticament el nombre de nodes de treball segons la càrrega de treball.

Exemple: Activació de l'Autoscaling

gcloud dataproc clusters update my-cluster \
    --region=us-central1 \
    --autoscaling-policy=basic-autoscaling-policy

Aquesta comanda actualitza el clúster my-cluster per utilitzar la basic-autoscaling-policy.

Ús de Fluxos de Treball de Dataproc

Els Fluxos de Treball de Dataproc et permeten definir una seqüència de treballs que es poden executar en un ordre específic.

Exemple: Creació d'una Plantilla de Flux de Treball

gcloud dataproc workflow-templates create my-workflow-template \
    --region=us-central1

gcloud dataproc workflow-templates set-managed-cluster my-workflow-template \
    --region=us-central1 \
    --cluster-name=my-cluster \
    --zone=us-central1-a \
    --master-machine-type=n1-standard-2 \
    --worker-machine-type=n1-standard-2 \
    --num-workers=2

gcloud dataproc workflow-templates add-job spark \
    --step-id=spark-job \
    --class=org.apache.spark.examples.SparkPi \
    --jars=file:///usr/lib/spark/examples/jars/spark-examples.jar \
    --workflow-template=my-workflow-template \
    --region=us-central1 \
    -- 1000

gcloud dataproc workflow-templates instantiate my-workflow-template \
    --region=us-central1

Aquesta seqüència de comandes crea una plantilla de flux de treball, afegeix un treball Spark a aquesta, i després insta el flux de treball.

Conclusió

Google Cloud Dataproc és una eina poderosa per gestionar clústers de Hadoop i Spark al núvol. Simplifica el procés de configuració, gestió i escalat de clústers, permetent-te centrar-te en processar les teves dades. Entenent els conceptes bàsics de creació de clústers, enviament de treballs i utilització de funcions avançades com l'autoscaling i els fluxos de treball, pots aprofitar efectivament Dataproc per a les teves necessitats de big data.
