# Exemple: Desplegant una aplicació simple de Python a App Engine

# fitxer de configuració app.yaml
runtime: python39

# main.py - Aplicació simple de Flask
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, GCP!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Explicació:

  1. app.yaml: Aquest fitxer especifica l'entorn d'execució per a l'aplicació. En aquest cas, està configurat a Python 3.9.
  2. main.py: Una aplicació simple de Flask que defineix una única ruta (/) que retorna "Hello, GCP!" quan s'accedeix.

Per desplegar aquesta aplicació:

  1. Assegura't de tenir instal·lat i autenticat el Google Cloud SDK.
  2. Navega al directori que conté app.yaml i main.py.
  3. Executa la comanda: 
    gcloud app deploy
  4. Un cop desplegada, pots accedir a la teva aplicació a través de l'URL proporcionada per GCP.
