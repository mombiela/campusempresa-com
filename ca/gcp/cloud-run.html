Introducció a Cloud Run

Cloud Run és una plataforma de computació totalment gestionada que escala automàticament els teus contenidors sense estat. Abstrau tota la gestió de la infraestructura, permetent-te centrar-te en escriure codi.

  • Totalment Gestionat: No cal gestionar servidors o clústers.
  • Escalable: Escala automàticament cap amunt i cap avall segons el trànsit.
  • Contenidors Sense Estat: Executa qualsevol contenidor sense estat.

Configuració de Cloud Run

Abans de desplegar a Cloud Run, necessites configurar el teu entorn de Google Cloud Platform (GCP).

  1. Crear un Projecte GCP:

    • Ves al GCP Console.
    • Fes clic al desplegable de projectes i selecciona "Nou Projecte".
    • Dona un nom al teu projecte i fes clic a "Crear".

  2. Activar la Facturació:

    • Ves a la secció de Facturació al GCP Console.
    • Enllaça un compte de facturació al teu projecte.

  3. Activar l'API de Cloud Run:

    • Navega a la secció d'API i Serveis.
    • Cerca "Cloud Run" i activa l'API.

  4. Instal·lar Google Cloud SDK:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu des de la documentació de Google Cloud SDK.

Desplegament d'un Contenidor a Cloud Run

Guia Pas a Pas

  1. Escriu la Teva Aplicació:

    • Crea un servidor web senzill utilitzant el teu llenguatge preferit. Aquí tens un exemple en Python utilitzant Flask:
    # app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return "Hola, Món!"

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

  2. Crea un Dockerfile:

    • Dockerfile s'utilitza per conteniritzar la teva aplicació.
    # Dockerfile
FROM python:3.8-slim

WORKDIR /app

COPY requirements.txt requirements.txt
RUN pip install -r requirements.txt

COPY . .

CMD ["python", "app.py"]

  3. Construeix i Puja la Imatge del Contenidor:

    • Utilitza Google Container Registry (GCR) per emmagatzemar les teves imatges de contenidors.
    # Construeix la imatge Docker
docker build -t gcr.io/[PROJECT-ID]/[IMAGE-NAME]:[TAG] .

# Puja la imatge Docker a GCR
docker push gcr.io/[PROJECT-ID]/[IMAGE-NAME]:[TAG]

  4. Desplega a Cloud Run:

    • Utilitza l'eina de línia de comandes gcloud per desplegar el teu contenidor.
    gcloud run deploy --image gcr.io/[PROJECT-ID]/[IMAGE-NAME]:[TAG] --platform managed
    • Segueix les indicacions per especificar el nom del servei i la regió.

Funcions Avançades de Cloud Run

Dominis Personalitzats

  • Mapeig de Dominis Personalitzats: Pots mapar dominis personalitzats als teus serveis de Cloud Run.
    • Ves a la pàgina de detalls del servei de Cloud Run.
    • Fes clic a "Gestiona dominis personalitzats" i segueix les instruccions.

Gestió del Trànsit

  • Divisió del Trànsit: Pots dividir el trànsit entre diferents revisions del teu servei.
    • Navega a la pàgina de detalls del servei de Cloud Run.
    • Fes clic a "Gestiona el trànsit" i assigna percentatges a diferents revisions.

Autenticació i Autorització

  • Rols IAM: Controla qui pot desplegar i invocar els teus serveis utilitzant rols IAM.
    • Assigna rols com roles/run.admin i roles/run.invoker a usuaris o comptes de servei.

Monitoratge i Registre

  • Cloud Monitoring: Utilitza Cloud Monitoring per fer el seguiment del rendiment dels teus serveis.
    • Configura quadres de comandament personalitzats i alertes.
  • Cloud Logging: Recull automàticament registres dels teus serveis.
    • Visualitza els registres a la secció de Cloud Logging del GCP Console.

Millors Pràctiques

  • Serveis Sense Estat: Assegura't que els teus serveis siguin sense estat per a una millor escalabilitat.
  • Variables d'Entorn: Utilitza variables d'entorn per gestionar la configuració.
  • Controls de Salut: Implementa controls de salut per assegurar-te que el teu servei funciona correctament.

Conclusió

Cloud Run simplifica el desplegament i la gestió d'aplicacions conteniritzades abstraient la infraestructura subjacent. Ofereix funcions potents com l'escalat automàtic, dominis personalitzats, gestió del trànsit i un monitoratge i registre robustos. Seguint les millors pràctiques i aprofitant aquestes funcions, pots desplegar i gestionar eficientment les teves aplicacions a GCP.

Al final d'aquest tema, hauràs de tenir una comprensió sòlida de com configurar, desplegar i gestionar aplicacions utilitzant Cloud Run a Google Cloud Platform.
