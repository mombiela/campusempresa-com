Introducció als Pipelines de Machine Learning

  • Definició: Un pipeline de machine learning és una seqüència de passos de processament de dades, entrenament de models i etapes d'avaluació que automatitzen el flux de treball de construcció i desplegament de models de machine learning.
  • Importància: Els pipelines asseguren la reproductibilitat, escalabilitat i eficiència en projectes de machine learning.

Configuració del Teu Entorn GCP

  • Crear un Compte GCP: Registra't per a un compte de Google Cloud i configura la facturació.
  • Habilitar APIs: Habilita les APIs necessàries com Compute Engine, Cloud Storage i AI Platform.
  • Instal·lar Google Cloud SDK: Instal·la i configura el Google Cloud SDK a la teva màquina local.
# Instal·lar el Google Cloud SDK
curl https://sdk.cloud.google.com | bash

# Inicialitzar el SDK
gcloud init

Ingestió i Preprocessament de Dades

  • Opcions d'Emmagatzematge de Dades:

    • Cloud Storage: Per emmagatzemar grans conjunts de dades.
    • BigQuery: Per gestionar dades estructurades i executar consultes SQL.

  • Carregar Dades:

    • Exemple de càrrega de dades des de Cloud Storage: 
      from google.cloud import storage

def download_blob(bucket_name, source_blob_name, destination_file_name):
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    blob = bucket.blob(source_blob_name)
    blob.download_to_filename(destination_file_name)
    print(f"Downloaded {source_blob_name} to {destination_file_name}.")

download_blob('my-bucket', 'data.csv', 'local-data.csv')

  • Preprocessament de Dades:

    • Utilitzant Pandas per a la neteja i transformació de dades: 
      import pandas as pd

# Carregar dades
df = pd.read_csv('local-data.csv')

# Neteja de dades
df.dropna(inplace=True)

# Enginyeria de característiques
df['new_feature'] = df['existing_feature'] * 2

Entrenament del Model

  • Triar un Marc de Treball de Machine Learning:

    • TensorFlow
    • Scikit-learn

  • Entrenament a AI Platform:

    • Exemple de submissió d'una feina d'entrenament utilitzant TensorFlow: 
      gcloud ai-platform jobs submit training my_job 
              --region=us-central1 
              --module-name=trainer.task 
              --package-path=trainer/ 
              --job-dir=gs://my-bucket/job-dir 
              --runtime-version=2.3 
              --python-version=3.7

  • Entrenament Distribuït: Aprofitant múltiples GPUs o TPUs per a un entrenament més ràpid.

Avaluació del Model

  • Mètriques d'Avaluació: Precisió, Precisió, Record, F1 Score.
  • Utilitzant AI Platform per a l'Avaluació:
    • Exemple d'avaluació d'un model: 
      from google.cloud import aiplatform

model = aiplatform.Model('projects/my-project/models/my-model')
evaluation = model.evaluate(
    test_data='gs://my-bucket/test-data.csv',
    metrics=['accuracy', 'precision', 'recall']
)
print(evaluation)

Desplegament del Model

  • Desplegar a AI Platform:

    • Exemple de desplegament d'un model entrenat: 
      gcloud ai-platform models create my_model 
              --regions=us-central1

gcloud ai-platform versions create v1 
              --model=my_model 
              --origin=gs://my-bucket/model-dir 
              --runtime-version=2.3 
              --python-version=3.7

  • Servei de Prediccions:

    • Exemple de fer prediccions utilitzant el model desplegat: 
      from google.cloud import aiplatform

model = aiplatform.Model('projects/my-project/models/my_model/versions/v1')
predictions = model.predict(instances=[[5.1, 3.5, 1.4, 0.2]])
print(predictions)

Monitorització i Manteniment

  • Monitorització del Rendiment del Model: Utilitzant les eines de monitorització d'AI Platform per fer un seguiment del rendiment del model al llarg del temps.
  • Actualització de Models: Reentrenament i redeployament de models a mesura que es disposa de noves dades.

Conclusió

En aquest tema, hem cobert els passos essencials per crear un pipeline de machine learning a Google Cloud Platform, des de la configuració de l'entorn fins al desplegament i monitorització de models. Seguint aquests passos, pots assegurar-te que els teus fluxos de treball de machine learning siguin eficients, escalables i reproductibles.
