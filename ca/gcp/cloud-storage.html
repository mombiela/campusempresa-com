Introducció a l'emmagatzematge al núvol

Google Cloud Storage és un servei per emmagatzemar i accedir a les teves dades a la infraestructura de Google. Proporciona una solució d'emmagatzematge d'objectes escalable, segura i duradora per a desenvolupadors i empreses.

  • Conceptes clau:
    • Buckets: Contenidors per emmagatzemar objectes.
    • Objectes: Peces individuals de dades emmagatzemades en buckets.
    • Classes: Diferents classes d'emmagatzematge per a diversos casos d'ús (per exemple, Standard, Nearline, Coldline, Archive).
    • Regions: Ubicacions geogràfiques on s'emmagatzemen les dades.

Configuració de Google Cloud Storage

Creació d'un projecte de Google Cloud

  1. Inicia sessió a Google Cloud Console:

    • Navega a Google Cloud Console.
    • Inicia sessió amb el teu compte de Google.

  2. Crea un nou projecte:

    • Fes clic al desplegable del projecte i selecciona "New Project."
    • Introdueix un nom de projecte i fes clic a "Create."

Activació de l'API de Cloud Storage

  1. Navega a la Biblioteca d'API:

    • A Google Cloud Console, ves a "Navigation Menu" > "APIs & Services" > "Library."

  2. Activa l'API de Cloud Storage:

    • Cerca "Cloud Storage" i fes clic sobre ella.
    • Fes clic a "Enable."

Treballant amb Buckets

Creació d'un Bucket

from google.cloud import storage

def create_bucket(bucket_name):
    """Creates a new bucket."""
    storage_client = storage.Client()
    bucket = storage_client.create_bucket(bucket_name)
    print(f"Bucket {bucket.name} created.")

create_bucket('my-new-bucket')
  • Explicació:
    • Importa el mòdul storage de la biblioteca google.cloud.
    • Crea una instància de storage.Client.
    • Utilitza el mètode create_bucket per crear un nou bucket amb el nom especificat.

Llistat de Buckets

def list_buckets():
    """Lists all buckets."""
    storage_client = storage.Client()
    buckets = storage_client.list_buckets()
    for bucket in buckets:
        print(bucket.name)

list_buckets()
  • Explicació:
    • Utilitza el mètode list_buckets per recuperar tots els buckets del projecte.
    • Itera a través dels buckets i imprimeix els seus noms.

Treballant amb Objectes

Pujada d'un Objecte

def upload_blob(bucket_name, source_file_name, destination_blob_name):
    """Uploads a file to the bucket."""
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    blob = bucket.blob(destination_blob_name)
    blob.upload_from_filename(source_file_name)
    print(f"File {source_file_name} uploaded to {destination_blob_name}.")

upload_blob('my-new-bucket', 'local-file.txt', 'remote-file.txt')
  • Explicació:
    • Obté el bucket utilitzant el mètode bucket.
    • Crea un objecte blob utilitzant el mètode blob.
    • Pujar el fitxer utilitzant el mètode upload_from_filename.

Descàrrega d'un Objecte

def download_blob(bucket_name, source_blob_name, destination_file_name):
    """Downloads a blob from the bucket."""
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    blob = bucket.blob(source_blob_name)
    blob.download_to_filename(destination_file_name)
    print(f"Blob {source_blob_name} downloaded to {destination_file_name}.")

download_blob('my-new-bucket', 'remote-file.txt', 'local-file.txt')
  • Explicació:
    • Obté l'objecte blob utilitzant el mètode blob.
    • Descarrega el fitxer utilitzant el mètode download_to_filename.

Classes d'Emmagatzematge i Gestió del Cicle de Vida

Classes d'Emmagatzematge

Google Cloud Storage ofereix diferents classes d'emmagatzematge per optimitzar el cost i el rendiment per a diversos casos d'ús:

| Classe d'Emmagatzematge | Cas d'Ús | Durabilitat | Disponibilitat | | ----------------------- | -------- | ----------- | -------------- | | Standard | Dades accedides freqüentment | 99.999999999% | 99.99% | | Nearline | Dades accedides infreqüentment (un cop al mes) | 99.999999999% | 99.95% | | Coldline | Dades accedides rarament (un cop a l'any) | 99.999999999% | 99.90% | | Archive | Emmagatzematge a llarg termini | 99.999999999% | 99.90% |

Gestió del Cicle de Vida

La gestió del cicle de vida et permet definir regles per a transicionar automàticament objectes a diferents classes d'emmagatzematge o eliminar-los després d'un cert període.

def set_bucket_lifecycle(bucket_name):
    """Sets a lifecycle management policy on a bucket."""
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    rule = {
        "action": {"type": "SetStorageClass", "storageClass": "COLDLINE"},
        "condition": {"age": 365}
    }
    lifecycle = bucket.lifecycle_rules
    lifecycle.append(rule)
    bucket.patch()
    print(f"Lifecycle management policy set for bucket {bucket_name}.")

set_bucket_lifecycle('my-new-bucket')
  • Explicació:
    • Defineix una regla per transicionar objectes a la classe d'emmagatzematge Coldline després de 365 dies.
    • Afegeix la regla a les regles del cicle de vida del bucket i aplica els canvis utilitzant el mètode patch.

Seguretat i Control d'Accés

Polítiques IAM

Les polítiques IAM (Identity and Access Management) et permeten controlar qui té accés als teus buckets i objectes.

  • Rols:
    • roles/storage.admin: Control complet sobre buckets i objectes.
    • roles/storage.objectAdmin: Control complet sobre objectes.
    • roles/storage.objectViewer: Accés de només lectura als objectes.

Configuració de Permisos del Bucket

def set_bucket_iam_policy(bucket_name):
    """Sets an IAM policy on a bucket."""
    storage_client = storage.Client()
    bucket = storage_client.bucket(bucket_name)
    policy = bucket.get_iam_policy()
    policy["bindings"].append(
        {
            "role": "roles/storage.objectViewer",
            "members": {"user:example-user@gmail.com"},
        }
    )
    bucket.set_iam_policy(policy)
    print(f"IAM policy set for bucket {bucket_name}.")

set_bucket_iam_policy('my-new-bucket')
  • Explicació:
    • Recupera la política IAM actual utilitzant el mètode get_iam_policy.
    • Afegeix una nova vinculació per concedir accés de només lectura a un usuari específic.
    • Aplica la política actualitzada utilitzant el mètode set_iam_policy.

Conclusió

Comprenent els conceptes clau, configurant buckets, gestionant objectes, utilitzant diferents classes d'emmagatzematge i implementant mesures de seguretat, pots aprofitar efectivament Google Cloud Storage per a una àmplia gamma d'aplicacions. Tant si ets un principiant com un usuari avançat, dominar aquestes habilitats et permetrà construir solucions al núvol escalables i segures.
