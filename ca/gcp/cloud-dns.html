Introducció a Cloud DNS

Google Cloud DNS és un servei escalable, fiable i gestionat del Sistema de Noms de Domini (DNS) autoritatiu que funciona sobre la mateixa infraestructura que Google. Traduïx noms de domini com www.example.com en adreces IP com 192.0.2.1, que són utilitzades pels ordinadors per comunicar-se entre ells.

Conceptes Clau

  • DNS: Sistema de Noms de Domini, un sistema jeràrquic per traduir noms de domini fàcils de recordar en adreces IP.
  • DNS gestionat: Un servei DNS proporcionat per un tercer, en aquest cas, Google Cloud, que maneja la infraestructura i l'escalabilitat.
  • Zones: Una zona DNS és una porció de l'espai de noms DNS gestionada per una organització o administrador específic.
  • Registres: Els registres DNS són entrades en una zona DNS que mapegen noms de domini a adreces IP o altres recursos.

Configuració de Cloud DNS

Creació d'una Zona DNS

Per crear una zona DNS a Google Cloud:

  1. Aneu a la pàgina de Cloud DNS a la Consola de Google Cloud.
  2. Feu clic a "Crear zona".
  3. Ompliu els següents camps:
    • Nom de la zona: Un nom únic per a la zona.
    • Nom DNS: El nom de domini per a la zona (per exemple, example.com).
    • Descripció: Una descripció opcional per a la zona.
    • DNSSEC: Activeu o desactiveu DNSSEC (Extensions de Seguretat del Sistema de Noms de Domini).
gcloud dns managed-zones create my-zone \
    --description="My first DNS zone" \
    --dns-name="example.com." \
    --visibility="public"

Afegir Registres DNS

Un cop creada la zona, podeu afegir registres DNS:

  1. Aneu a la pàgina de Cloud DNS a la Consola de Google Cloud.
  2. Feu clic a la zona que heu creat.
  3. Feu clic a "Afegir conjunt de registres".
  4. Ompliu els següents camps:
    • Nom DNS: El subdomini per al registre (per exemple, www).
    • Tipus de registre de recurs: El tipus de registre DNS (per exemple, A, CNAME, MX).
    • TTL: Temps de vida, la durada en segons que el registre es manté en la memòria cau dels resolutors DNS.
    • TTL: Temps de vida, la durada en segons que el registre es manté en la memòria cau dels resolutors DNS.
    • Dades: L'adreça IP o altres dades per al registre.
gcloud dns record-sets transaction start --zone=my-zone
gcloud dns record-sets transaction add --zone=my-zone \
    --name="www.example.com." --ttl=300 --type=A "192.0.2.1"
gcloud dns record-sets transaction execute --zone=my-zone

Configuració Avançada

DNSSEC

DNSSEC afegeix una capa addicional de seguretat al vostre DNS permetent signatures criptogràfiques als vostres registres DNS.

  • Activació de DNSSEC: Això es pot fer durant la creació de la zona DNS o més tard editant la configuració de la zona.
  • Gestió de claus: Google Cloud DNS maneja la gestió de claus automàticament, però també podeu gestionar les claus manualment si cal.

Zones DNS Privades

Les zones DNS privades us permeten gestionar registres DNS per a la vostra xarxa interna, que només són accessibles des del vostre VPC (Núvol Privat Virtual).

  1. Creeu una zona DNS privada:

    gcloud dns managed-zones create my-private-zone 
           --description="My private DNS zone" 
           --dns-name="internal.example.com." 
           --visibility="private" 
           --networks="my-vpc-network"

  2. Afegiu registres a la zona privada com ho faríeu amb una zona pública.

Monitorització i Registre

Google Cloud proporciona eines per monitoritzar i registrar consultes i respostes DNS:

  • Cloud Logging: Activeu el registre per a les vostres zones DNS per capturar registres de consultes DNS.
  • Cloud Monitoring: Configureu alertes i quadres de comandament per monitoritzar la salut i el rendiment de les vostres zones DNS.
gcloud dns managed-zones update my-zone --logging

Conclusió

Google Cloud DNS és un servei DNS potent i flexible que pot manejar tant necessitats DNS públiques com privades. Entenent els conceptes bàsics de DNS, configurant zones i registres, i aprofitant funcions avançades com DNSSEC i zones privades, podeu gestionar efectivament la infraestructura DNS del vostre domini. Les capacitats de monitorització i registre milloren encara més la vostra capacitat per mantenir un entorn DNS robust i segur.
