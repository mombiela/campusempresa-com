Introducció a Google Cloud Platform (GCP)

Google Cloud Platform (GCP) és un conjunt de serveis de computació en núvol oferts per Google. Aquest curs et portarà des d'un nivell principiant fins a un nivell avançat, cobrint tots els aspectes essencials de GCP.

Què és GCP?

  • Una col·lecció de serveis en núvol que inclouen computació, emmagatzematge de dades, anàlisi de dades i aprenentatge automàtic.
  • Proporciona infraestructura com a servei (IaaS), plataforma com a servei (PaaS) i entorns de computació sense servidor.

Conceptes Clau

  • Regions i Zones: Ubicacions físiques on es troben allotjats els recursos.
  • Projectes: Unitats organitzatives per gestionar recursos.
  • Facturació: Comprendre com GCP cobra pels serveis.
  • IAM (Gestió d'Identitat i Accés): Gestionar permisos i accessos.

Serveis Bàsics de GCP

Compute Engine

  • Màquines Virtuals (VMs) a GCP.
  • Crear i gestionar VMs.
# Crear una instància de VM
gcloud compute instances create my-vm --zone=us-central1-a --machine-type=e2-medium

App Engine

  • Plataforma com a Servei (PaaS) per construir aplicacions web escalables.
  • Desplegar una aplicació web simple.
# Configuració app.yaml per a Python App Engine
runtime: python38

# main.py per a una aplicació web simple
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hola, món!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Kubernetes Engine

  • Servei de Kubernetes gestionat.
  • Desplegar una aplicació contenidoritzada.
# Crear un clúster de Kubernetes
gcloud container clusters create my-cluster --zone us-central1-a

# Desplegar una aplicació
kubectl create deployment hello-server --image=gcr.io/google-samples/hello-app:1.0

# Exposar l'aplicació
kubectl expose deployment hello-server --type=LoadBalancer --port 80 --target-port 8080

Emmagatzematge de Dades i Bases de Dades

Cloud Storage

  • Emmagatzematge d'objectes per a dades no estructurades.
  • Crear i gestionar cubs d'emmagatzematge.
# Crear un cub d'emmagatzematge
gsutil mb gs://my-bucket

# Pujar un fitxer al cub
gsutil cp my-file.txt gs://my-bucket/

Cloud SQL

  • Servei de base de dades relacional gestionat.
  • Configurar una instància de MySQL.
# Crear una instància de Cloud SQL
gcloud sql instances create my-instance --database-version=MYSQL_5_7 --tier=db-n1-standard-1 --region=us-central1

# Connectar-se a la instància
gcloud sql connect my-instance --user=root

Big Data i Aprenentatge Automàtic

BigQuery

Seguretat i Compliment

Gestió d'Identitat i Accés (IAM)

  • Gestionar usuaris i permisos.

# Afegir un usuari a un projecte amb un rol específic

Temes Avançats de GCP

Infraestructura com a Codi amb Terraform

  • Utilitzar Terraform per gestionar recursos de GCP.
  • Escriure i desplegar configuracions de Terraform.
# main.tf per crear una instància de VM
provider "google" {
  project = "my-project"
  region  = "us-central1"
}

resource "google_compute_instance" "default" {
  name         = "my-vm"
  machine_type = "e2-medium"
  zone         = "us-central1-a"

  boot_disk {
    initialize_params {
      image = "debian-cloud/debian-9"
    }
  }

  network_interface {
    network = "default"
    access_config {
    }
  }
}
