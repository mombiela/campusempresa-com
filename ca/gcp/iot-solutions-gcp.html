Introducció a IoT i GCP

  • Què és IoT?
    • Internet of Things (IoT) es refereix a la xarxa d'objectes físics incrustats amb sensors, programari i altres tecnologies per connectar i intercanviar dades amb altres dispositius i sistemes a través d'internet.

    • Escalabilitat
    • Seguretat
    • Integració amb altres serveis de Google
    • Capacitats avançades d'analítica i aprenentatge automàtic

Google Cloud IoT Core

  • Visió general d'IoT Core
    • Servei gestionat per connectar, gestionar i ingerir dades de dispositius dispersos globalment de manera segura.
  • Conceptes clau
    • Registre de dispositius: Un contenidor per a dispositius.
    • Dispositiu: Representa un dispositiu físic.
    • Configuració del dispositiu: Configuracions i ajustaments per a dispositius.
    • Estat del dispositiu: L'estat actual del dispositiu.
    • Telemetria: Dades enviades dels dispositius al núvol.
  • Configuració d'IoT Core
    • Creació d'un registre de dispositius.
    • Registre d'un dispositiu.
    • Configuració de les credencials del dispositiu.

Exemple: Creació d'un registre de dispositius

from google.cloud import iot_v1

client = iot_v1.DeviceManagerClient()
parent = client.location_path('your-project-id', 'us-central1')
registry = {
    'id': 'my-registry',
    'event_notification_configs': [{
        'pubsub_topic_name': 'projects/your-project-id/topics/your-topic'
    }]
}
response = client.create_device_registry(parent, registry)
print('Created registry:', response)
  • Explicació: Aquest fragment de codi demostra com crear un registre de dispositius a GCP utilitzant Python. Substitueix 'your-project-id' i 'your-topic' pel teu ID de projecte i tema de Pub/Sub reals.

Exemple: Ingestió de dades amb Pub/Sub

from google.cloud import pubsub_v1

publisher = pubsub_v1.PublisherClient()
topic_path = publisher.topic_path('your-project-id', 'your-topic')

data = 'Hello, IoT!'
data = data.encode('utf-8')
future = publisher.publish(topic_path, data)
print(future.result())
  • Explicació: Aquest fragment de codi mostra com publicar un missatge a un tema de Pub/Sub. Substitueix 'your-project-id' i 'your-topic' pel teu ID de projecte i nom del tema reals.
