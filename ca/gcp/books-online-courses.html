En aquesta secció, explorarem diversos recursos disponibles per aprendre Google Cloud Platform (GCP) des de nivells principiants fins a avançats. Aquests recursos inclouen llibres i cursos en línia que proporcionen camins d'aprenentatge estructurats, exemples pràctics i exercicis pràctics.

Llibres

Els llibres són una gran manera d'obtenir coneixements en profunditat i poden servir com a guia de referència a mesura que avances en el teu viatge amb GCP. Aquí tens alguns llibres recomanats per a diferents nivells:

Nivell Principiant

  • "Google Cloud Platform for Beginners" de Ted Hunter i Steven Porter

    • Conceptes Clau:
      • Introducció a GCP
      • Configuració d'un compte GCP
      • Serveis bàsics com Compute Engine, Cloud Storage i BigQuery
    • Exemple: 
      Capítol 3: Configuració de la teva primera màquina virtual

      gcloud compute instances create my-first-vm --zone=us-central1-a --machine-type=n1-standard-1 --image-family=debian-9 --image-project=debian-cloud
      • Aquesta comanda crea una nova instància de màquina virtual a GCP.

  • "Google Cloud Platform in Action" de JJ Geewax

    • Conceptes Clau:
      • Visió general dels serveis de GCP
      • Projectes pràctics
      • Millors pràctiques per utilitzar GCP
    • Exemple: 
      Capítol 5: Desplegant la teva primera aplicació

      # app.py
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
      • Aquest fragment de codi Python demostra com desplegar una aplicació simple de Flask a GCP.

Nivell Intermedi

  • "Architecting Google Cloud Solutions" de Victor Dantas i Antonio Gulli
    • Conceptes Clau:
      • Disseny d'arquitectures escalables i resilients
      • Xarxes avançades i seguretat
      • Processament de dades i aprenentatge automàtic
    • Exemple: 
      Capítol 7: Construint una pipeline de dades amb Dataflow

      import apache_beam as beam

def run():
    p = beam.Pipeline()
    (p
     | 'ReadFromPubSub' >> beam.io.ReadFromPubSub(subscription='projects/my-project/subscriptions/my-subscription')
     | 'TransformData' >> beam.Map(lambda x: x.upper())
     | 'WriteToBigQuery' >> beam.io.WriteToBigQuery('my-project:my_dataset.my_table'))
    p.run().wait_until_finish()

if __name__ == '__main__':
    run()
      • Aquest fragment de codi mostra com crear una pipeline de dades utilitzant Apache Beam i Dataflow.

Nivell Avançat

  • "Site Reliability Engineering: How Google Runs Production Systems" de Niall Richard Murphy et al.
    • Conceptes Clau:
      • Principis de l'Enginyeria de Fiabilitat del Lloc (SRE)
      • Monitorització i alertes
      • Resposta i gestió d'incidents
    • Exemple: 
      Capítol 10: Monitorització de sistemes distribuïts

      - alert: HighErrorRate
  expr: job:request_latency_seconds:mean5m{job="my-service"} > 0.5
  for: 10m
  labels:
    severity: page
  annotations:
    summary: "High error rate detected"
    description: "Error rate is above 0.5 for more than 10 minutes."
      • Aquesta configuració YAML configura una alerta per a taxes d'error altes utilitzant Prometheus.

Cursos en Línia

Els cursos en línia ofereixen experiències d'aprenentatge interactives i sovint pràctiques. Aquí tens alguns cursos recomanats per a diferents nivells:

Nivell Principiant

  • Coursera: "Google Cloud Platform Fundamentals: Core Infrastructure"
    • Conceptes Clau:
      • Introducció a GCP
      • Serveis i eines bàsiques
      • Laboratoris pràctics
    • Exemple: 
      Laboratori: Creant una màquina virtual

      gcloud compute instances create my-vm --zone=us-central1-a --machine-type=f1-micro --image-family=debian-9 --image-project=debian-cloud
      • Aquest laboratori et guia a través de la creació d'una instància de màquina virtual.

Nivell Intermedi

  • Udacity: "Developing Scalable Apps in Python"
    • Conceptes Clau:
      • Construcció d'aplicacions escalables
      • Utilització d'App Engine i Datastore
      • Optimització del rendiment
    • Exemple: 
      Projecte: Construint una aplicació web escalable

      # main.py
from google.appengine.ext import ndb
from flask import Flask, request

app = Flask(__name__)

class Message(ndb.Model):
    content = ndb.StringProperty()

@app.route('/add', methods=['POST'])
def add_message():
    content = request.form['content']
    message = Message(content=content)
    message.put()
    return 'Message added!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
      • Aquest projecte demostra com construir una aplicació web escalable utilitzant App Engine i Datastore.

Nivell Avançat

  • Pluralsight: "Architecting with Google Kubernetes Engine"
    • Conceptes Clau:
      • Fonaments de Kubernetes
      • Desplegament i gestió d'aplicacions contenidoritzades
      • Funcions avançades de Kubernetes
    • Exemple: 
      Mòdul: Desplegant una aplicació Kubernetes

      apiVersion: apps/v1
kind: Deployment
metadata:
  name: my-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: my-app
  template:
    metadata:
      labels:
        app: my-app
    spec:
      containers:
      - name: my-app
        image: gcr.io/my-project/my-app:v1
        ports:
        - containerPort: 8080
      • Aquest fitxer YAML defineix un desplegament de Kubernetes per a una aplicació contenidoritzada.
