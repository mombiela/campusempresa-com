Descripció del Projecte

Introducció

En aquest curs sobre Processament de Dades Massives, explorarem com manejar grans volums de dades utilitzant diverses eines i tècniques. Aquest projecte té com a objectiu proporcionar una comprensió profunda dels conceptes i pràctiques essencials en el processament de dades massives, des de la recollida i emmagatzematge fins a l'anàlisi i visualització.

Objectius del Projecte

  • Comprendre els conceptes bàsics del processament de dades massives: Introducció a Big Data, característiques de les dades massives (Volum, Velocitat, Varietat, Veracitat i Valor).
  • Familiaritzar-se amb les eines i tecnologies clau: Hadoop, Spark, NoSQL, i altres tecnologies rellevants.
  • Desenvolupar habilitats pràctiques: Implementació de solucions de processament de dades massives mitjançant exemples i exercicis pràctics.
  • Analitzar i visualitzar dades massives: Ús d'eines d'anàlisi i visualització per extreure informació valuosa de grans conjunts de dades.

Estructura del Projecte

Recollida de Dades

  • Fonts de dades massives: Sensors IoT, xarxes socials, transaccions financeres, etc.
  • Tècniques de recollida: Web scraping, APIs, bases de dades distribuïdes.

Emmagatzematge de Dades

  • Sistemes de fitxers distribuïts: HDFS (Hadoop Distributed File System).
  • Bases de dades NoSQL: MongoDB, Cassandra, HBase.
  • Comparació entre bases de dades relacionals i NoSQL:

| Característica | Bases de Dades Relacionals | Bases de Dades NoSQL | |----------------------|----------------------------|----------------------| | Model de dades | Taules | Documents, Claus-Valor, Columnes, Grafs | | Escalabilitat | Vertical | Horitzontal | | Flexibilitat | Baixa | Alta | | Consistència | Alta | Eventual |

Processament de Dades

  • MapReduce: Conceptes bàsics i exemples.
  • Apache Spark: Introducció i comparació amb MapReduce.
  • Exemple de codi en Spark:
from pyspark.sql import SparkSession

# Crear una sessió de Spark
spark = SparkSession.builder.appName("ExempleSpark").getOrCreate()

# Llegir un fitxer CSV en un DataFrame
df = spark.read.csv("dades.csv", header=True, inferSchema=True)

# Mostrar les primeres files del DataFrame
df.show()

# Realitzar una operació d'agregació
df.groupBy("categoria").count().show()

# Aturar la sessió de Spark
spark.stop()

Anàlisi i Visualització de Dades

  • Eines d'anàlisi: Pandas, NumPy.
  • Eines de visualització: Matplotlib, Seaborn, Tableau.
  • Exemple de visualització amb Matplotlib:
import matplotlib.pyplot as plt
import pandas as pd

# Llegir dades des d'un fitxer CSV
data = pd.read_csv("dades.csv")

# Crear un gràfic de barres
plt.figure(figsize=(10, 6))
data['categoria'].value_counts().plot(kind='bar')
plt.title('Distribució de Categories')
plt.xlabel('Categoria')
plt.ylabel('Freqüència')
plt.show()

Conclusió

Aquest projecte proporciona una guia integral per al processament de dades massives, abastant des de la recollida i emmagatzematge fins a l'anàlisi i visualització. A través d'exemples pràctics i exercicis, els estudiants desenvoluparan habilitats essencials per manejar grans volums de dades i extreure informació valuosa. En finalitzar el curs, els participants estaran equipats amb el coneixement i les eines necessàries per abordar desafiaments en l'àmbit del Big Data.
