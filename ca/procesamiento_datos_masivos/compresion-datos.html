Compressió de Dades

Introducció a la Compressió de Dades

La compressió de dades és una tècnica essencial en el processament de dades massives, ja que permet reduir la mida de les dades per facilitar-ne l'emmagatzematge i transmissió. En aquest tema, explorarem els conceptes fonamentals de la compressió de dades, els algorismes més comuns i com aplicar-los en el context de grans volums de dades.

Conceptes Clau

  • Compressió sense pèrdua: Tècnica de compressió que permet recuperar les dades originals exactament després de la descompressió. Exemples inclouen Huffman Coding i Lempel-Ziv-Welch (LZW).
  • Compressió amb pèrdua: Tècnica de compressió que redueix la mida de les dades eliminant part de la informació, la qual cosa resulta en una pèrdua de qualitat. Exemples inclouen JPEG i MP3.
  • Ràtio de compressió: Mesura de l'eficàcia d'un algorisme de compressió, calculada com la mida original dividida per la mida comprimida.
  • Entropia: Mesura de la quantitat d'informació continguda en un conjunt de dades, que influeix en la capacitat de compressió.

Algorismes de Compressió

Compressió sense pèrdua

  1. Huffman Coding

    • Algorisme de compressió basat en la freqüència d'aparició dels caràcters.
    • Construeix un arbre binari on els caràcters més freqüents tenen codis més curts.
    import heapq
from collections import defaultdict, Counter

def huffman_encoding(data):
    frequency = Counter(data)
    heap = [[weight, [symbol, ""]] for symbol, weight in frequency.items()]
    heapq.heapify(heap)
    mentre len(heap) > 1:
        lo = heapq.heappop(heap)
        hi = heapq.heappop(heap)
        per pair in lo[1:]:
            pair[1] = '0' + pair[1]
        per pair in hi[1:]:
            pair[1] = '1' + pair[1]
        heapq.heappush(heap, [lo[0] + hi[0]] + lo[1:] + hi[1:])
    return sorted(heapq.heappop(heap)[1:], key=lambda p: (len(p[-1]), p))

data = "this is an example for huffman encoding"
huff = huffman_encoding(data)
print("Symbol\tWeight\tHuffman Code")
per p in huff:
    print(f"{p[0]}\t{frequency[p[0]]}\t{p[1]}")

  2. Lempel-Ziv-Welch (LZW)

    • Algorisme de compressió basat en diccionaris que reemplaça subcadenes repetides amb referències a un diccionari.
    def lzw_compress(uncompressed):
    dict_size = 256
    dictionary = {chr(i): i for i in range(dict_size)}
    w = ""
    result = []
    per c in uncompressed:
        wc = w + c
        if wc in dictionary:
            w = wc
        else:
            result.append(dictionary[w])
            dictionary[wc] = dict_size
            dict_size += 1
            w = c
    if w:
        result.append(dictionary[w])
    return result

data = "TOBEORNOTTOBEORTOBEORNOT"
compressed = lzw_compress(data)
print("Compressed data:", compressed)

Compressió amb pèrdua

  1. JPEG (Joint Photographic Experts Group)

    • Algorisme de compressió d'imatges que redueix la mida eliminant detalls que són menys perceptibles per a l'ull humà.
    • Utilitza transformades de cosinus discret (DCT) i quantització.

  2. MP3 (MPEG-1 Audio Layer III)

    • Algorisme de compressió d'àudio que elimina sons inaudibles per a l'oïda humana.
    • Utilitza transformades de Fourier i quantització perceptual.

Comparació d'Algorismes

| Algorisme | Tipus | Ús Comú | Avantatges | Desavantatges | |-----------------|---------------|------------------|------------------------------------|----------------------------------| | Huffman Coding | Sense pèrdua | Textos, arxius | Simple i eficient | No sempre aconsegueix alta compressió | | LZW | Sense pèrdua | Textos, imatges | Bona compressió per a dades repetitives | Pot ser ineficient per a dades no repetitives | | JPEG | Amb pèrdua | Imatges | Alta compressió, bona qualitat visual | Pèrdua de qualitat irreversible | | MP3 | Amb pèrdua | Àudio | Alta compressió, bona qualitat auditiva | Pèrdua de qualitat irreversible |

Conclusió

La compressió de dades és una eina poderosa en el processament de dades massives, permetent reduir significativament la mida de les dades per al seu emmagatzematge i transmissió. Comprendre els diferents algorismes de compressió i les seves aplicacions és crucial per seleccionar la tècnica adequada segons el tipus de dades i els requisits específics del projecte.
