Introducció a l'EDA

L'Anàlisi Exploratori de Dades (EDA, per les seves sigles en anglès) és una fase crucial en el processament de dades massives. El seu objectiu és comprendre l'estructura, les característiques i les relacions en les dades abans d'aplicar models predictius o algoritmes d'aprenentatge automàtic. L'EDA ajuda a identificar patrons, detectar anomalies, provar hipòtesis i verificar supòsits.

Objectius de l'EDA

  • Comprendre la distribució de les dades: Identificar com es distribueixen les variables i si hi ha biaixos.
  • Detectar valors atípics: Trobar dades que es desvien significativament de la resta.
  • Identificar relacions entre variables: Descobrir correlacions i dependències.
  • Avaluar la qualitat de les dades: Verificar la presència de dades mancants, duplicades o inconsistents.

Eines i Tècniques Comunes

Estadístiques Descriptives

Les estadístiques descriptives proporcionen un resum ràpid de les dades.

  • Mesures de tendència central: Mitjana, mediana i moda.
  • Mesures de dispersió: Rang, variància i desviació estàndard.
  • Mesures de forma: Asimetria i curtosi.
import pandas as pd

# Carregar dades
data = pd.read_csv('data.csv')

# Estadístiques descriptives
print(data.describe())

Visualització de Dades

La visualització és una eina poderosa per a l'EDA. Permet identificar patrons i anomalies de manera intuïtiva.

  • Histogramas: Per veure la distribució d'una variable.
  • Diagrames de caixa (box plots): Per detectar valors atípics.
  • Diagrames de dispersió (scatter plots): Per analitzar relacions entre dues variables.
  • Mapes de calor (heatmaps): Per visualitzar correlacions entre múltiples variables.
import matplotlib.pyplot as plt
import seaborn as sns

# Histograma
data['variable'].hist()
plt.show()

# Diagrama de caixa
sns.boxplot(x=data['variable'])
plt.show()

# Diagrama de dispersió
sns.scatterplot(x='variable1', y='variable2', data=data)
plt.show()

# Mapa de calor
sns.heatmap(data.corr(), annot=True, cmap='coolwarm')
plt.show()

Gestió de Dades Mancants

Les dades mancants poden afectar significativament els anàlisis i models predictius.

  • Eliminació de dades mancants: Útil quan la quantitat de dades mancants és petita.
  • Imputació de dades mancants: Omplir els valors mancants amb la mitjana, mediana o mitjançant tècniques més avançades com la imputació múltiple.
# Eliminar files amb dades mancants
data_cleaned = data.dropna()

# Imputar dades mancants amb la mitjana
data_filled = data.fillna(data.mean())

Detecció de Valors Atípics

Els valors atípics poden distorsionar els anàlisis i han de ser identificats i tractats adequadament.

  • Mètode del rang interquartílic (IQR): Identificar valors atípics basats en el rang interquartílic.
  • Mètode Z-score: Detectar valors atípics utilitzant la desviació estàndard.
# Mètode IQR
Q1 = data['variable'].quantile(0.25)
Q3 = data['variable'].quantile(0.75)
IQR = Q3 - Q1
outliers = data[(data['variable'] < (Q1 - 1.5 * IQR)) | (data['variable'] > (Q3 + 1.5 * IQR))]

# Mètode Z-score
from scipy import stats
z_scores = stats.zscore(data['variable'])
outliers = data[(z_scores > 3) | (z_scores < -3)]

Conclusió

L'Anàlisi Exploratori de Dades és una etapa fonamental en el processament de dades massives. Permet als analistes i científics de dades comprendre millor les seves dades, identificar problemes i preparar les dades per a anàlisis més avançades. Utilitzant tècniques d'estadístiques descriptives, visualització, gestió de dades mancants i detecció de valors atípics, es pot obtenir una visió clara i detallada de les dades, la qual cosa facilita la presa de decisions informades i la construcció de models predictius més precisos.
