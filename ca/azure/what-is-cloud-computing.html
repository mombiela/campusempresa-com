Introducció al Cloud Computing

El cloud computing és la distribució de serveis de computació a través d'internet ("el núvol"), incloent emmagatzematge, potència de processament, bases de dades, xarxes, programari, anàlisi i intel·ligència. Ofereix una innovació més ràpida, recursos flexibles i economies d'escala.

Conceptes Clau

  • Autoservei a Demanda: Els usuaris poden aprovisionar capacitats de computació segons les necessitats automàticament sense requerir interacció humana amb cada proveïdor de serveis.
  • Accés Ampli a la Xarxa: Les capacitats estan disponibles a través de la xarxa i s'accedeix a elles mitjançant mecanismes estàndard que promouen l'ús per plataformes de clients heterogènies, fines o gruixudes.
  • Agrupació de Recursos: Els recursos de computació del proveïdor estan agrupats per servir múltiples consumidors utilitzant un model multi-llogater.
  • Elasticitat Ràpida: Les capacitats es poden aprovisionar i alliberar elàsticament per escalar ràpidament cap a fora i cap a dins segons la demanda.
  • Servei Mesurat: Els sistemes de núvol controlen i optimitzen automàticament l'ús dels recursos aprofitant una capacitat de mesurament a algun nivell d'abstracció adequat al tipus de servei.

Tipus de Serveis de Núvol

Els serveis de núvol es categoritzen típicament en tres tipus:

  1. Infraestructura com a Servei (IaaS):

    • Proporciona recursos de computació virtualitzats a través d'internet.
    • Exemples: Màquines virtuals, emmagatzematge i xarxes.
    • Exemple d'Azure: Azure Virtual Machines.

  2. Plataforma com a Servei (PaaS):

    • Proporciona eines de maquinari i programari a través d'internet, típicament les necessàries per al desenvolupament d'aplicacions.
    • Exemples: Eines de desenvolupament, sistemes de gestió de bases de dades, middleware.
    • Exemple d'Azure: Azure App Services.

  3. Programari com a Servei (SaaS):

    • Distribueix aplicacions de programari a través d'internet, amb una subscripció.
    • Exemples: Correu electrònic, CRM, eines de col·laboració.
    • Exemple d'Azure: Microsoft Office 365.

Models de Desplegament de Núvol

Hi ha diversos models de desplegament per als serveis de núvol:

  • Núvol Públic: Els serveis es distribueixen a través d'internet públic i es comparteixen entre organitzacions.
  • Núvol Privat: Els serveis es mantenen en una xarxa privada i s'utilitzen exclusivament per una organització.
  • Núvol Híbrid: Combina núvols públics i privats, permetent que les dades i les aplicacions es comparteixin entre ells.

Beneficis del Cloud Computing

  • Eficiència de Costos: Redueix la despesa de capital en comprar maquinari i programari.
  • Velocitat: Es poden aprovisionar grans quantitats de recursos de computació en minuts.
  • Escala Global: Proporciona la quantitat adequada de recursos TI quan es necessiten i des de la ubicació geogràfica adequada.
  • Productivitat: Elimina la necessitat de moltes tasques, com la configuració de maquinari i l'aplicació de pegats de programari.
  • Rendiment: Ofereix serveis de núvol principals que funcionen en una xarxa mundial de centres de dades segurs.
  • Fiabilitat: La còpia de seguretat de dades, la recuperació de desastres i la continuïtat del negoci són més fàcils i menys costoses.

Començant amb Azure

Azure és la plataforma de cloud computing de Microsoft, que ofereix una àmplia gamma de serveis per construir, desplegar i gestionar aplicacions a través de centres de dades gestionats per Microsoft.

Exemple: Crear una Màquina Virtual a Azure

Aquí hi ha un exemple senzill de com crear una màquina virtual utilitzant Azure CLI:

# Iniciar sessió a Azure
az login

# Crear un grup de recursos
az group create --name myResourceGroup --location eastus

# Crear una màquina virtual
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Explicació:

  • az login: Autentica el teu compte d'Azure.
  • az group create: Crea un grup de recursos, que és un contenidor que conté recursos relacionats.
  • az vm create: Crea una màquina virtual amb els paràmetres especificats.

Conclusió

El cloud computing és una tecnologia transformadora que ofereix solucions escalables, flexibles i rendibles per a les necessitats modernes de TI. Azure, com a plataforma de núvol líder, proporciona una suite completa de serveis a través dels models IaaS, PaaS i SaaS, atenent una àmplia gamma d'aplicacions i indústries. Entendre els conceptes bàsics del cloud computing i les ofertes d'Azure és el primer pas per aprofitar el poder del núvol per als teus projectes i necessitats empresarials.
