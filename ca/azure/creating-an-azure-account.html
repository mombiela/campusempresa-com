Crear un compte d'Azure és el primer pas per aprofitar els serveis al núvol de Microsoft per a les teves aplicacions i projectes. Aquesta secció et guiarà a través del procés de configuració d'un compte d'Azure, assegurant-te que tens la base necessària per explorar i utilitzar les àmplies ofertes d'Azure.

Per què crear un compte d'Azure?

  • Accés als serveis al núvol: Aconsegueix accés a una àmplia gamma de serveis al núvol, incloent computació, analítica, emmagatzematge i xarxes.
  • Escalabilitat: Escala fàcilment les teves aplicacions i serveis segons les teves necessitats.
  • Gestió de costos: Paga només pel que utilitzes amb els models de preus flexibles d'Azure.
  • Seguretat: Beneficia't de les robustes mesures de seguretat i certificacions de compliment d'Azure.

Prerequisits

Abans de crear un compte d'Azure, assegura't de tenir el següent:

  • Una adreça de correu electrònic vàlida.
  • Un número de telèfon per a la verificació.
  • Una targeta de crèdit per a la verificació d'identitat (no se't cobrarà tret que actualitzis a un pla de pagament).

Passos per crear un compte d'Azure

  1. Visita el lloc web d'Azure:

    • Ves al lloc web d'Azure.

  2. Comença gratuïtament:

    • Fes clic al botó "Start free" a la pàgina principal.

  3. Inicia sessió o crea un compte de Microsoft:

    • Si ja tens un compte de Microsoft, inicia sessió.
    • Si no, fes clic a "Create one!" per configurar un nou compte de Microsoft.

  4. Omple les teves dades:

    • Proporciona la teva informació personal, incloent nom, correu electrònic i número de telèfon.
    • Verifica el teu número de telèfon mitjançant un codi enviat al teu telèfon.

  5. Verificació d'identitat:

    • Introdueix les dades de la teva targeta de crèdit per a la verificació d'identitat. Tingues en compte que això és un procediment estàndard i no se't cobrarà tret que actualitzis explícitament a un pla de pagament.

  6. Accepta els termes:

    • Revisa i accepta l'acord de Microsoft Azure i la declaració de privacitat.

  7. Completa la configuració:

    • Un cop totes les dades estiguin verificades, el teu compte d'Azure es crearà i podràs començar a utilitzar els serveis d'Azure.

Beneficis del compte gratuït d'Azure

Quan crees un nou compte d'Azure, obtens accés a diversos beneficis:

  • Crèdit de $200: Utilitza aquest crèdit dins dels primers 30 dies per explorar i provar qualsevol servei d'Azure.
  • Serveis gratuïts durant 12 mesos: Accedeix a serveis populars de manera gratuïta durant 12 mesos, incloent:
    • 750 hores de màquina virtual B1S Windows.
    • 5 GB d'emmagatzematge de fitxers LRS.
    • 250 GB d'emmagatzematge de base de dades SQL.
  • Serveis sempre gratuïts: Alguns serveis són sempre gratuïts, com ara:
    • Servei d'Aplicacions d'Azure amb 1 milió de sol·licituds per mes.
    • Funcions d'Azure amb 1 milió de sol·licituds per mes.

Exemple: Crear una màquina virtual

Un cop el teu compte d'Azure estigui configurat, pots començar a crear recursos. Aquí tens un exemple senzill de com crear una màquina virtual (VM):

# Inicia sessió al teu compte d'Azure
az login

# Crea un grup de recursos
az group create --name myResourceGroup --location eastus

# Crea una màquina virtual
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys
  • az login: Comanda per iniciar sessió al teu compte d'Azure.
  • az group create: Crea un grup de recursos anomenat myResourceGroup a la regió eastus.
  • az vm create: Crea una VM anomenada myVM amb la imatge Ubuntu LTS, establint azureuser com a nom d'usuari administrador i generant claus SSH per a un accés segur.

Conclusió

Crear un compte d'Azure és un procés senzill que obre un món de possibilitats per a la computació al núvol. Amb el teu nou compte, pots explorar la gran varietat de serveis d'Azure, aprofitar els crèdits gratuïts i començar a construir i desplegar les teves aplicacions al núvol. A les següents seccions, aprofundirem en serveis específics d'Azure i com utilitzar-los efectivament per satisfer les necessitats del teu projecte.
