Introducció a l'emmagatzematge de fitxers d'Azure

L'emmagatzematge de fitxers d'Azure és un servei d'emmagatzematge de fitxers al núvol que et permet crear comparticions de fitxers al núvol. Està dissenyat per ser altament disponible, durador i escalable, fent-lo una excel·lent elecció per a una varietat d'aplicacions, des de la compartició simple de fitxers fins a solucions d'emmagatzematge a nivell empresarial complexes.

Conceptes clau

  • Compartició de fitxers: Una compartició de fitxers és un agrupament lògic de fitxers que es poden accedir a través de la xarxa.
  • Compte d'emmagatzematge: Un compte d'emmagatzematge és un contenidor que conté totes les teves dades d'emmagatzematge d'Azure, incloent comparticions de fitxers.
  • Nivells d'accés: L'emmagatzematge de fitxers d'Azure ofereix diferents nivells d'accés (Hot, Cool i Archive) per optimitzar els costos basats en la freqüència d'accés a les dades.
  • Protocol SMB: Server Message Block (SMB) és un protocol de compartició de fitxers de xarxa que permet a les aplicacions llegir i escriure fitxers i sol·licitar serveis de programes de servidor en una xarxa d'ordinadors.

Configuració de l'emmagatzematge de fitxers d'Azure

Creació d'un compte d'emmagatzematge

Per utilitzar l'emmagatzematge de fitxers d'Azure, primer has de crear un compte d'emmagatzematge.

  1. Navega al portal d'Azure: Ves a Azure Portal.
  2. Crea un compte d'emmagatzematge:
    • Fes clic a "Crear un recurs" i selecciona "Compte d'emmagatzematge".
    • Omple els detalls requerits com Subscripció, Grup de recursos, Nom del compte d'emmagatzematge i Ubicació.
    • Tria el rendiment (Standard o Premium) i les opcions de replicació (LRS, GRS, etc.) basades en les teves necessitats.
    • Fes clic a "Revisar + crear" i després a "Crear".

Creació d'una compartició de fitxers

Un cop tinguis un compte d'emmagatzematge, pots crear una compartició de fitxers.

  1. Navega al teu compte d'emmagatzematge: Ves al compte d'emmagatzematge que has creat.
  2. Crea una compartició de fitxers:
    • Fes clic a "Comparticions de fitxers" sota la secció "Emmagatzematge de dades".
    • Fes clic a "+ Compartició de fitxers".
    • Introdueix un nom per a la compartició de fitxers i especifica la quota (en GB).
    • Fes clic a "Crear".

Accés a l'emmagatzematge de fitxers d'Azure

Muntar la compartició de fitxers a Windows

Pots muntar la compartició de fitxers d'Azure en una màquina Windows utilitzant el protocol SMB.

  1. Obtenir la cadena de connexió:

    • Navega a la teva compartició de fitxers al portal d'Azure.
    • Fes clic a "Connectar" i selecciona "Windows".
    • Copia el script proporcionat.

  2. Executar l'script:

    • Obre PowerShell a la teva màquina Windows.
    • Enganxa i executa l'script. Serà semblant a això:
    $connectTestResult = Test-Path -Path "Z:"
if ($connectTestResult -eq $false) {
    New-PSDrive -Name Z -PSProvider FileSystem -Root "\\<storage_account_name>.file.core.windows.net\<file_share_name>" -Persist -Credential (Get-Credential)
}

Accés a la compartició de fitxers des de Linux

També pots muntar la compartició de fitxers en una màquina Linux utilitzant el protocol SMB.

  1. Instal·lar CIFS Utils:

    sudo apt-get update
sudo apt-get install cifs-utils

  2. Muntar la compartició de fitxers:

    sudo mount -t cifs //<storage_account_name>.file.core.windows.net/<file_share_name> /mnt/<mount_point> -o vers=3.0,username=<storage_account_name>,password=<storage_account_key>,dir_mode=0777,file_mode=0777

Funcions avançades

Sincronització de fitxers d'Azure

La sincronització de fitxers d'Azure et permet centralitzar les teves comparticions de fitxers a Azure Files, mentre mantens la flexibilitat, el rendiment i la compatibilitat d'un servidor de fitxers local.

  1. Instal·la l'agent de sincronització de fitxers d'Azure: Descarrega i instal·la l'agent de sincronització de fitxers d'Azure al teu servidor Windows.
  2. Registra el servidor: Utilitza l'agent de sincronització de fitxers d'Azure per registrar el teu servidor amb la teva subscripció d'Azure.
  3. Crea un grup de sincronització: Al portal d'Azure, crea un grup de sincronització i afegeix la teva compartició de fitxers i el servidor registrat.

Control d'accés i seguretat

  • Integració amb Azure Active Directory (AAD): Pots integrar Azure Files amb Azure AD per gestionar els permisos d'accés.
  • Seguretat de xarxa: Utilitza la xarxa virtual (VNet) i les regles de tallafocs per restringir l'accés a les teves comparticions de fitxers.

Conclusió

L'emmagatzematge de fitxers d'Azure proporciona una solució robusta, escalable i segura per emmagatzemar i accedir a fitxers al núvol. Entenent els conceptes bàsics de la creació i gestió de comparticions de fitxers, així com aprofitant les funcions avançades com la sincronització de fitxers d'Azure i el control d'accés, pots utilitzar efectivament l'emmagatzematge de fitxers d'Azure per satisfer les teves necessitats d'emmagatzematge.
