Introducció

Gestionar màquines virtuals (VMs) a Azure és una habilitat fonamental per a la computació en núvol. Aquest tema et guiarà a través del procés des d'un nivell principiant fins a un nivell avançat, cobrint conceptes clau, exemples pràctics i exercicis.

Començant amb les Màquines Virtuals d'Azure

Què és una Màquina Virtual?

  • Una màquina virtual (VM) és una emulació d'un sistema informàtic.
  • Les VMs s'executen en maquinari físic però estan aïllades d'aquest.
  • Permeten executar múltiples sistemes operatius en una sola màquina física.

Creant la Teva Primera VM

  1. Inicia sessió al Portal d'Azure: Navega a Azure Portal.
  2. Crea un Recurs: Fes clic a "Create a resource" i selecciona "Virtual Machine".
  3. Configura els Paràmetres Bàsics:
    • Subscripció: Tria la teva subscripció d'Azure.
    • Grup de Recursos: Crea un nou grup de recursos o selecciona'n un d'existent.
    • Nom de la VM: Dona un nom únic a la teva VM.
    • Regió: Selecciona la regió on vols desplegar la teva VM.
    • Imatge: Tria una imatge del sistema operatiu (per exemple, Windows Server, Ubuntu).
    • Mida: Selecciona la mida de la VM segons les teves necessitats.
# Exemple: Comanda Azure CLI per crear una VM
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Connectant-te a la Teva VM

  • SSH per a VMs Linux: Utilitza un client SSH per connectar-te.
  • RDP per a VMs Windows: Utilitza el Protocol d'Escriptori Remot (RDP).
# Exemple: SSH en una VM Linux
ssh azureuser@<public-ip-address>

Tasques de Gestió Intermèdies

Gestió de Mides de VM i Escalat

  • Redimensionant una VM: Canvia la mida de la teva VM per escalar cap amunt o cap avall segons les teves necessitats.
  • Auto-escalat: Utilitza Azure Scale Sets per escalar automàticament les VMs segons la demanda.
# Exemple: Redimensionar una VM utilitzant Azure CLI
az vm resize \
  --resource-group myResourceGroup \
  --name myVM \
  --size Standard_DS3_v2

Gestió de Discos

  • Afegint Discos de Dades: Adjunta discos addicionals a la teva VM per obtenir més emmagatzematge.
  • Tipus de Discos: Entén la diferència entre HDD Estàndard, SSD Estàndard i SSD Premium.
# Exemple: Adjuntar un nou disc de dades a una VM
az vm disk attach \
  --resource-group myResourceGroup \
  --vm-name myVM \
  --name myDataDisk \
  --size-gb 128

Xarxes

  • Xarxes Virtuals (VNet): Crea i gestiona VNets per controlar la configuració de xarxa de la VM.
  • Grups de Seguretat de Xarxa (NSG): Configura NSGs per gestionar el trànsit entrant i sortint.
# Exemple: Crear una VNet i Subxarxa
az network vnet create \
  --resource-group myResourceGroup \
  --name myVNet \
  --address-prefix 10.0.0.0/16 \
  --subnet-name mySubnet \
  --subnet-prefix 10.0.1.0/24

Tasques de Gestió Avançades

Automatització amb Plantilles d'Azure Resource Manager (ARM)

  • Plantilles ARM: Defineix la teva infraestructura com a codi utilitzant plantilles JSON.
  • Desplegant Plantilles ARM: Utilitza Azure CLI o el Portal d'Azure per desplegar plantilles.
{
  "$schema": "https://schema.management.azure.com/schemas/2019-04-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "resources": [
    {
      "type": "Microsoft.Compute/virtualMachines",
      "apiVersion": "2019-12-01",
      "name": "myVM",
      "location": "[resourceGroup().location]",
      "properties": {
        "hardwareProfile": {
          "vmSize": "Standard_DS1_v2"
        },
        "storageProfile": {
          "imageReference": {
            "publisher": "Canonical",
            "offer": "UbuntuServer",
            "sku": "18.04-LTS",
            "version": "latest"
          },
          "osDisk": {
            "createOption": "FromImage"
          }
        },
        "osProfile": {
          "computerName": "myVM",
          "adminUsername": "azureuser",
          "linuxConfiguration": {
            "disablePasswordAuthentication": true,
            "ssh": {
              "publicKeys": [
                {
                  "path": "/home/azureuser/.ssh/authorized_keys",
                  "keyData": "ssh-rsa AAAAB3NzaC1yc2EAAAADAQABAAABAQC..."
                }
              ]
            }
          }
        },
        "networkProfile": {
          "networkInterfaces": [
            {
              "id": "[resourceId('Microsoft.Network/networkInterfaces', 'myNic')]"
            }
          ]
        }
      }
    }
  ]
}

Monitorització i Diagnòstics

  • Azure Monitor: Recull i analitza dades de telemetria de les teves VMs.
  • Log Analytics: Consulta i analitza registres per obtenir informació més profunda.
# Exemple: Activar la monitorització en una VM
az vm diagnostics set \
  --resource-group myResourceGroup \
  --vm-name myVM \
  --settings '{"storageAccount": "mystorageaccount"}'

Còpia de Seguretat i Recuperació

  • Azure Backup: Configura polítiques de còpia de seguretat per protegir les teves VMs.
  • Recuperació de Desastres: Utilitza Azure Site Recovery per a la replicació i recuperació de VMs.
# Exemple: Activar la còpia de seguretat per a una VM
az backup protection enable-for-vm \
  --resource-group myResourceGroup \
  --vault-name myRecoveryServicesVault \
  --vm myVM \
  --policy-name DefaultPolicy

Conclusió

Gestionar màquines virtuals a Azure implica una sèrie de tasques des de la creació i connexió bàsiques fins a l'automatització i monitorització avançades. Dominant aquestes habilitats, pots aprofitar efectivament les capacitats d'Azure per desplegar, escalar i mantenir la teva infraestructura virtual. Practica regularment i explora la documentació extensa d'Azure per aprofundir en la teva comprensió i competència.
