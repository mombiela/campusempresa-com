Introducció a Azure Machine Learning

Azure Machine Learning és un servei basat en el núvol proporcionat per Microsoft que permet als desenvolupadors i científics de dades construir, entrenar i desplegar models d'aprenentatge automàtic. Proporciona una suite completa d'eines i serveis per facilitar tot el cicle de vida de l'aprenentatge automàtic.

Conceptes Clau

  • Workspace: Un lloc centralitzat per gestionar tots els recursos necessaris per a l'aprenentatge automàtic.
  • Datasets: Col·leccions de dades utilitzades per entrenar i provar models d'aprenentatge automàtic.
  • Experiments: Execucions d'un script o pipeline d'aprenentatge automàtic.
  • Compute Targets: Recursos on es poden executar els experiments d'aprenentatge automàtic, com ara màquines locals, màquines virtuals o clústers.
  • Models: Models d'aprenentatge automàtic entrenats que es poden desplegar per a inferència.
  • Endpoints: Serveis RESTful que allotgen els teus models i fan prediccions.

Configuració del Teu Entorn

Abans de començar a utilitzar Azure Machine Learning, cal configurar el teu entorn.

Passos per Configurar

  1. Crear un Compte d'Azure: Si no en tens un, registra't per obtenir un compte gratuït.
  2. Crear un Workspace: Utilitza el portal d'Azure per crear un nou workspace d'Azure Machine Learning.
  3. Instal·lar Azure CLI: Instal·la la Interfície de Línia de Comandes d'Azure (CLI) per interactuar amb els serveis d'Azure des del teu terminal.
  4. Instal·lar Azure Machine Learning SDK: Utilitza pip per instal·lar l'SDK d'Azure ML per a Python.
pip install azureml-sdk

Creació i Gestió de Datasets

Els datasets són crucials per entrenar models d'aprenentatge automàtic. Azure Machine Learning suporta diversos tipus de datasets, incloent datasets tabulars i de fitxers.

Creació d'un Dataset

Pots crear un dataset a partir d'un fitxer local o de dades emmagatzemades a Azure Blob Storage.

from azureml.core import Workspace, Dataset

# Connectar-se al workspace
ws = Workspace.from_config()

# Crear un dataset a partir d'un fitxer local
dataset = Dataset.Tabular.from_delimited_files(path='data.csv')

# Registrar el dataset
dataset = dataset.register(workspace=ws, name='my_dataset')

Gestió de Datasets

Pots llistar, actualitzar i eliminar datasets utilitzant l'SDK d'Azure ML.

# Llistar tots els datasets al workspace
datasets = ws.datasets
for name, dataset in datasets.items():
    print(name, dataset.id)

# Eliminar un dataset
dataset_to_delete = datasets['my_dataset']
dataset_to_delete.unregister_all_versions()

Construcció i Entrenament de Models

Construir i entrenar models implica crear experiments, definir compute targets i executar scripts d'entrenament.

Creació d'un Experiment

Un experiment és una col·lecció de proves (execucions individuals) que representen intents de resoldre un problema d'aprenentatge automàtic.

from azureml.core import Experiment

# Crear un experiment
experiment = Experiment(workspace=ws, name='my_experiment')

Definició de Compute Targets

Els compute targets són els recursos on executes els teus scripts d'entrenament.

from azureml.core.compute import ComputeTarget, AmlCompute
from azureml.core.compute_target import ComputeTargetException

# Definir la configuració de compute
compute_config = AmlCompute.provisioning_configuration(vm_size='STANDARD_D2_V2', max_nodes=4)

# Crear el compute target
compute_target = ComputeTarget.create(ws, 'my-compute-cluster', compute_config)
compute_target.wait_for_completion(show_output=True)

Execució d'Scripts d'Entrenament

Pots executar els teus scripts d'entrenament als compute targets definits.

from azureml.core import ScriptRunConfig

# Definir la configuració de l'execució de l'script
src = ScriptRunConfig(source_directory='.', script='train.py', compute_target=compute_target)

# Enviar l'experiment
run = experiment.submit(src)
run.wait_for_completion(show_output=True)

Desplegament de Models

Un cop has entrenat un model, el següent pas és desplegar-lo perquè es pugui utilitzar per a inferència.

Registre del Model

Registra el model entrenat al teu workspace.

from azureml.core import Model

# Registrar el model
model = Model.register(workspace=ws, model_name='my_model', model_path='outputs/model.pkl')

Creació d'una Configuració d'Inferència

Defineix l'entorn i l'script de scoring per al teu model.

from azureml.core.environment import Environment
from azureml.core.model import InferenceConfig

# Definir l'entorn
env = Environment.from_conda_specification(name='myenv', file_path='environment.yml')

# Definir la configuració d'inferència
inference_config = InferenceConfig(entry_script='score.py', environment=env)

Desplegament del Model

Desplega el model a una Azure Container Instance (ACI) o Azure Kubernetes Service (AKS).

from azureml.core.webservice import AciWebservice, Webservice

# Definir la configuració del desplegament
deployment_config = AciWebservice.deploy_configuration(cpu_cores=1, memory_gb=1)

# Desplegar el model
service = Model.deploy(workspace=ws, name='my-service', models=[model], inference_config=inference_config, deployment_config=deployment_config)
service.wait_for_deployment(show_output=True)

Conclusió

Azure Machine Learning proporciona una plataforma robusta per construir, entrenar i desplegar models d'aprenentatge automàtic. Entenent els conceptes clau i seguint els passos descrits en aquest curs, pots aprofitar efectivament Azure Machine Learning per resoldre problemes complexos i desplegar solucions d'aprenentatge automàtic escalables.
