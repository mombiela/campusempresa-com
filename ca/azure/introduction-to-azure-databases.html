Visió General

Les Bases de Dades d'Azure ofereixen una àmplia gamma de serveis de bases de dades que s'adapten a diverses necessitats, des de bases de dades relacionals fins a bases de dades NoSQL. Aquesta secció us introduirà als diferents tipus de bases de dades disponibles a Azure, les seves característiques clau i com començar a utilitzar-les.

Tipus de Bases de Dades d'Azure

Bases de Dades Relacionals

Les bases de dades relacionals emmagatzemen dades en taules amb files i columnes. Azure ofereix diversos serveis de bases de dades relacionals:

  • Azure SQL Database: Un servei de bases de dades relacionals totalment gestionat basat en Microsoft SQL Server.
  • Azure Database for MySQL: Un servei de bases de dades MySQL totalment gestionat.
  • Azure Database for PostgreSQL: Un servei de bases de dades PostgreSQL totalment gestionat.

Bases de Dades NoSQL

Les bases de dades NoSQL estan dissenyades per a dades no estructurades i ofereixen esquemes flexibles:

  • Azure Cosmos DB: Un servei de bases de dades multi-model distribuït globalment que admet models de dades de document, clau-valor, gràfic i column-family.
  • Azure Table Storage: Un magatzem de clau-valor NoSQL per a desenvolupament ràpid utilitzant conjunts de dades semi-estructurats massius.

Bases de Dades en Memòria

Les bases de dades en memòria emmagatzemen dades en RAM per a un accés més ràpid:

  • Azure Cache for Redis: Un servei de memòria cau Redis totalment gestionat que ajuda a millorar el rendiment i l'escalabilitat de les vostres aplicacions.

Conceptes Clau

Serveis Gestionats

Les bases de dades d'Azure són totalment gestionades, és a dir, Microsoft s'encarrega del manteniment, les còpies de seguretat i les actualitzacions, permetent-vos centrar-vos en el desenvolupament de la vostra aplicació.

Escalabilitat

Les bases de dades d'Azure ofereixen diverses opcions d'escalabilitat:

  • Escalabilitat Vertical: Augmentar la mida de la instància de la base de dades.
  • Escalabilitat Horitzontal: Distribuir la base de dades a través de múltiples instàncies.

Seguretat

Les bases de dades d'Azure vénen amb funcions de seguretat integrades:

  • Xifratge: Les dades estan xifrades en repòs i en trànsit.
  • Control d'Accés: Control d'accés basat en rols (RBAC) i identitats gestionades.

Alta Disponibilitat

Les bases de dades d'Azure proporcionen alta disponibilitat mitjançant:

  • Replicació: Les dades es repliquen a través de múltiples regions.
  • Failover: Els mecanismes automàtics de failover asseguren un temps d'inactivitat mínim.

Començar amb Azure SQL Database

Crear una Azure SQL Database

  1. Navegar al Portal d'Azure: Aneu a Portal d'Azure.
  2. Crear un Recurs: Feu clic a "Crear un recurs" i cerqueu "SQL Database".
  3. Configurar la Base de Dades: Ompliu els detalls requerits com el nom de la base de dades, el servidor i el nivell de preus.
  4. Revisar i Crear: Reviseu la vostra configuració i feu clic a "Crear".

Connectar-se a Azure SQL Database

Podeu connectar-vos a la vostra Azure SQL Database utilitzant diverses eines com SQL Server Management Studio (SSMS) o Azure Data Studio.

Exemple: Connectar-se amb Python

import pyodbc

# Definir la cadena de connexió
conn_str = (
    "DRIVER={ODBC Driver 17 for SQL Server};"
    "SERVER=tcp:<your_server>.database.windows.net,1433;"
    "DATABASE=<your_database>;"
    "UID=<your_username>;"
    "PWD=<your_password>"
)

# Establir la connexió
conn = pyodbc.connect(conn_str)

# Crear un objecte cursor
cursor = conn.cursor()

# Executar una consulta
cursor.execute("SELECT TOP 10 * FROM your_table")

# Recuperar i imprimir els resultats
for row in cursor.fetchall():
    print(row)

# Tancar la connexió
conn.close()

Resum

Azure ofereix una suite completa de serveis de bases de dades que s'adapten a diverses necessitats, des de bases de dades relacionals fins a NoSQL i bases de dades en memòria. Aquests serveis són totalment gestionats, escalables, segurs i d'alta disponibilitat, fent-los una excel·lent opció per al desenvolupament d'aplicacions modernes. Comprenent els conceptes clau i adquirint experiència pràctica, podeu aprofitar les bases de dades d'Azure per construir aplicacions robustes i escalables.
