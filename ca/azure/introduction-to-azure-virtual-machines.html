Crear i Configurar Màquines Virtuals ►

Visió General

Les Màquines Virtuals (VMs) d'Azure són un dels serveis principals proporcionats per Microsoft Azure. Et permeten crear i gestionar entorns de computació virtualitzats al núvol. Aquesta secció t'introduirà als conceptes bàsics de les VMs d'Azure, incloent els seus beneficis, casos d'ús i com començar.

Conceptes Clau

  • Màquina Virtual (VM): Una emulació de programari d'un ordinador físic que executa un sistema operatiu i aplicacions.
  • Portal d'Azure: Una aplicació basada en web que proporciona una consola unificada per gestionar recursos d'Azure.
  • Grup de Recursos: Un contenidor que manté recursos relacionats per a una solució d'Azure.
  • Xarxa Virtual (VNet): Una representació de la teva pròpia xarxa al núvol.

Beneficis d'Utilitzar VMs d'Azure

  • Escalabilitat: Escala fàcilment cap amunt o cap avall segons la demanda.
  • Cost-Efectiu: Paga només pel que utilitzes amb opcions de preus flexibles.
  • Flexibilitat: Tria entre una àmplia gamma de mides i configuracions de VM.
  • Abast Global: Desplega VMs en centres de dades arreu del món.

Casos d'Ús

  • Desenvolupament i Proves: Configura ràpidament entorns per al desenvolupament i les proves.
  • Execució d'Aplicacions: Allotja aplicacions que requereixen alta disponibilitat i escalabilitat.
  • Recuperació de Desastres: Utilitza VMs per a solucions de còpia de seguretat i recuperació de desastres.

Començar amb les VMs d'Azure

Pas 1: Crear un Compte d'Azure

Per començar a utilitzar les VMs d'Azure, necessites un compte d'Azure. Pots registrar-te per a un compte gratuït a Azure Free Account.

Pas 2: Accedir al Portal d'Azure

Un cop tinguis un compte, inicia sessió al Portal d'Azure.

Pas 3: Crear una Màquina Virtual

Segueix aquests passos per crear la teva primera VM:

  1. Navega al servei de Màquines Virtuals:

    • Al Portal d'Azure, fes clic a "Crear un recurs".
    • Selecciona "Computació" i després "Màquina Virtual".

  2. Configura la Configuració Bàsica:

    • Subscripció: Tria la teva subscripció.
    • Grup de Recursos: Crea un nou grup de recursos o selecciona'n un d'existent.
    • Nom de la Màquina Virtual: Introdueix un nom per a la teva VM.
    • Regió: Tria la regió on vols desplegar la VM.
    • Imatge: Selecciona la imatge del sistema operatiu (per exemple, Windows Server, Ubuntu).
    • Mida: Tria la mida de la VM segons els teus requisits.

  3. Configura el Compte Administratiu:

    • Nom d'Usuari: Introdueix un nom d'usuari per a la VM.
    • Contrasenya: Introdueix una contrasenya forta.

  4. Configura la Xarxa:

    • Xarxa Virtual: Crea una nova VNet o selecciona'n una d'existent.
    • Subxarxa: Tria una subxarxa dins de la VNet.
    • IP Pública: Assigna una adreça IP pública si cal.

  5. Revisa i Crea:

    • Revisa la teva configuració i fes clic a "Crear" per desplegar la VM.

Exemple: Crear una VM utilitzant Azure CLI

També pots crear una VM utilitzant la Interfície de Línia d'Ordres (CLI) d'Azure. Aquí tens un exemple:

# Inicia sessió a Azure
az login

# Crea un grup de recursos
az group create --name myResourceGroup --location eastus

# Crea una màquina virtual
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Aquest script inicia sessió a Azure, crea un grup de recursos i després crea una VM amb Ubuntu LTS.

Gestionar les VMs d'Azure

Un cop la teva VM està creada, pots gestionar-la a través del Portal d'Azure o utilitzant Azure CLI. Les tasques clau de gestió inclouen:

  • Iniciar i Aturar VMs: Controla l'estat de la teva VM.
  • Monitorització i Diagnòstic: Utilitza Azure Monitor per fer un seguiment del rendiment i la salut.
  • Escalabilitat: Ajusta la mida i el nombre de VMs segons la demanda.

Conclusió

Les Màquines Virtuals d'Azure proporcionen una solució flexible i escalable per executar aplicacions i càrregues de treball al núvol. Comprenent els conceptes bàsics de creació i gestió de VMs, pots aprofitar el poder d'Azure per satisfer les teves necessitats de computació. Tant si estàs desenvolupant aplicacions, executant càrregues de treball empresarials, com si estàs configurant una solució de recuperació de desastres, les VMs d'Azure ofereixen les eines i capacitats per donar suport als teus objectius.
