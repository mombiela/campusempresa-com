Azure Automation és un servei d'automatització i configuració basat en el núvol que proporciona una gestió consistent a través dels teus entorns Azure i no Azure. T'ajuda a automatitzar tasques freqüents, que consumeixen temps i són propenses a errors.

Introducció a Azure Automation

  • Què és Azure Automation?

    • Un servei per automatitzar tasques i orquestrar fluxos de treball.
    • Ajuda a gestionar entorns en el núvol i locals.
    • Suporta l'automatització de tasques repetitives, la gestió de configuracions i la gestió d'actualitzacions.

  • Beneficis Clau:

    • Eficiència: Automatitza tasques repetitives per estalviar temps.
    • Consistència: Assegura que les tasques es realitzin de la mateixa manera cada vegada.
    • Estalvi de costos: Redueix els costos operatius automatitzant processos manuals.
    • Escalabilitat: Escala fàcilment les tasques d'automatització a mesura que el teu entorn creix.

Començar amb Azure Automation

  • Crear un Compte d'Automatització:

    1. Inicia sessió al portal d'Azure.
    2. Navega a "Crear un recurs" > "Automatització".
    3. Omple els detalls requerits (Nom, Grup de Recursos, Ubicació).
    4. Fes clic a "Crear".

  • Components del Compte d'Automatització:

    • Runbooks: Scripts que realitzen tasques.
    • Actius: Recursos com variables, credencials i connexions.
    • Horaris: Defineixen quan han de córrer els runbooks.
    • Mòduls: Biblioteques de cmdlets de PowerShell.

Exemple: Crear un Runbook Simple

# Exemple de Runbook de PowerShell
Write-Output "Hola, Azure Automation!"
  • Passos per Crear i Executar un Runbook:
    1. Al teu compte d'Automatització, ves a "Runbooks".
    2. Fes clic a "Afegir un runbook" > "Crear un nou runbook".
    3. Introdueix un nom i selecciona "PowerShell" com a tipus de runbook.
    4. Enganxa el codi d'exemple anterior.
    5. Desa i publica el runbook.
    6. Fes clic a "Iniciar" per executar el runbook.

Conceptes Intermedis d'Azure Automation

  • Ús de Variables i Credencials:
    • Variables: Emmagatzema valors que es poden reutilitzar en runbooks.
    • Credencials: Emmagatzema i utilitza de manera segura credencials en runbooks.

Exemple: Ús de Variables en un Runbook

# Recupera un valor de variable
$myVariable = Get-AutomationVariable -Name "MyVariable"
Write-Output "El valor de MyVariable és $myVariable"
  • Horaris i Webhooks:
    • Horaris: Automatitza l'execució de runbooks en moments específics.
    • Webhooks: Activa runbooks mitjançant sol·licituds HTTP.

Exemple: Crear un Horari

  1. Navega a "Horaris" al teu compte d'Automatització.
  2. Fes clic a "Afegir un horari".
  3. Defineix l'horari (per exemple, diàriament a les 8 AM).
  4. Enllaça l'horari a un runbook.

Tècniques Avançades d'Azure Automation

  • Treballadors de Runbook Híbrids:

    • Executa runbooks en entorns locals o en altres entorns de núvol.
    • Útil per a tasques que requereixen accés a recursos locals.

  • Integració amb Control de Versions:

    • Integra amb GitHub o Azure Repos per al control de versions de runbooks.

  • Tipus Avançats de Runbook:

    • Runbooks Gràfics: Crea runbooks utilitzant una interfície de drag-and-drop.
    • Runbooks de Python: Utilitza scripts de Python per a tasques d'automatització.

Exemple: Crear un Runbook Gràfic

  1. Al teu compte d'Automatització, ves a "Runbooks".
  2. Fes clic a "Afegir un runbook" > "Crear un nou runbook".
  3. Introdueix un nom i selecciona "Gràfic" com a tipus de runbook.
  4. Utilitza l'editor gràfic per dissenyar el teu flux de treball.
  5. Desa i publica el runbook.

Conclusió

Azure Automation és una eina poderosa per automatitzar i gestionar els teus entorns de núvol i locals. Aprofitant els runbooks, horaris i altres components del compte d'automatització, pots optimitzar operacions, reduir errors i estalviar temps. Tant si acabes de començar com si busques implementar tècniques avançades d'automatització, Azure Automation proporciona la flexibilitat i escalabilitat per satisfer les teves necessitats.
