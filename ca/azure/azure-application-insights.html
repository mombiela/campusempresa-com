Azure Application Insights és una eina potent per monitoritzar i analitzar el rendiment i l'ús de les teves aplicacions. T'ajuda a detectar problemes, diagnosticar fallades i entendre el comportament dels usuaris. Aquest tema et guiarà des dels conceptes bàsics fins a les característiques avançades d'Azure Application Insights.

Introducció a Azure Application Insights

  • Què és Azure Application Insights?

    • Un servei que proporciona monitorització i anàlisi per a aplicacions web.
    • Part de la suite Azure Monitor.
    • Ajuda a detectar anomalies de rendiment, diagnosticar problemes i entendre el comportament dels usuaris.

  • Característiques clau:

    • Recollida automàtica de registres i mètriques de l'aplicació.
    • Monitorització i alertes en temps real.
    • Integració amb diverses eines i marcs de desenvolupament.
    • Quadres de comandament i informes personalitzables.

Configuració d'Azure Application Insights

  • Creació d'un recurs d'Application Insights:

    1. Navega al Portal d'Azure.
    2. Fes clic a "Crear un recurs" i cerca "Application Insights".
    3. Omple els detalls necessaris com el Grup de Recursos, la Regió i el Tipus d'Aplicació.
    4. Fes clic a "Revisar + crear" i després a "Crear".

  • Integració amb la teva aplicació:

    • Aplicacions .NET:

Monitorització i Diagnòstic

  • Flux de Mètriques en Temps Real:

    • Proporciona dades en temps real sobre el rendiment de la teva aplicació.
    • Útil per detectar problemes a mesura que succeeixen.

  • Mapa de l'Aplicació:

    • Representació visual dels components de la teva aplicació i les seves interaccions.
    • Ajuda a identificar colls d'ampolla de rendiment i punts de fallada.

  • Errors i Excepcions:

    • Recull i informa automàticament sobre excepcions.
    • Proporciona traçats de pila detallats i missatges d'error.

  • Mètriques de Rendiment:

    • Fa un seguiment dels temps de resposta, les taxes de sol·licitud i les taxes de fallada.
    • Ajuda a identificar dependències lentes i optimitzar el rendiment.

Telemetria Personalitzada

  • Esdeveniments Personalitzats:

    • Fa un seguiment d'accions o esdeveniments específics dins de la teva aplicació.
    telemetryClient.TrackEvent("UserLogin", new Dictionary<string, string> { { "UserId", "12345" } });

  • Mètriques Personalitzades:

    • Fa un seguiment de mètriques de rendiment personalitzades.
    telemetryClient.GetMetric("CustomMetric").TrackValue(42);

  • Traces Personalitzades:

    • Registra missatges personalitzats per a depuració i monitorització.
    telemetryClient.TrackTrace("Custom trace message", SeverityLevel.Information);

Característiques Avançades

  • Mostreig:

    • Redueix el volum de dades de telemetria enviant només un subconjunt de dades.
    • Configurable per equilibrar entre volum de dades i informació.

  • Exportació Contínua:

    • Exporta dades de telemetria a emmagatzematge extern per a anàlisi a llarg termini.
    • Suporta l'exportació a Azure Blob Storage o Azure Event Hubs.

  • Integració amb DevOps:

    • Integra amb Azure DevOps per a monitorització i desplegament continus.
    • Configura alertes i quadres de comandament per monitoritzar la salut de les teves pipelines CI/CD.

Conclusió

Azure Application Insights és una eina completa per monitoritzar i analitzar el rendiment i l'ús de les teves aplicacions. Integrant-la amb la teva aplicació, pots obtenir informació valuosa sobre el comportament dels usuaris, detectar i diagnosticar problemes i optimitzar el rendiment. Tant si ets un usuari principiant com avançat, Azure Application Insights proporciona les eines que necessites per assegurar que les teves aplicacions funcionin de manera suau i eficient.
