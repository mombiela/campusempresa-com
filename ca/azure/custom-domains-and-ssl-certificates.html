Introducció

En aquest mòdul, explorarem com configurar dominis personalitzats i certificats SSL a Azure. Això és essencial per assegurar les teves aplicacions web i proporcionar una aparença professional amb noms de domini personalitzats.

Dominis personalitzats

Què és un domini personalitzat?

Un domini personalitzat és una adreça única i llegible per humans que pots utilitzar per accedir a la teva aplicació web. En lloc d'utilitzar el domini per defecte proporcionat per Azure (per exemple, yourapp.azurewebsites.net), pots utilitzar un domini com www.yourdomain.com.

Per què utilitzar dominis personalitzats?

  • Branding: Millora la identitat de la teva marca.
  • Confiança: Augmenta la confiança i credibilitat dels usuaris.
  • SEO: Millora l'optimització en motors de cerca.

Configuració d'un domini personalitzat

  1. Compra un domini: Pots comprar un domini a registradors de dominis com GoDaddy, Namecheap, etc.
  2. Afegeix un domini personalitzat a Azure:
    • Navega a la teva aplicació web d'Azure.
    • Ves a la secció "Dominis personalitzats".
    • Fes clic a "Afegeix domini personalitzat".
    • Introdueix el teu nom de domini personalitzat.
  3. Verifica la propietat del domini:
    • Azure proporcionarà un registre TXT que has d'afegir a la configuració DNS del teu domini.
    • Un cop verificat el registre TXT, el teu domini estarà enllaçat a la teva aplicació web d'Azure.

Exemple: Afegir un domini personalitzat

# Navega a la teva aplicació web d'Azure
az webapp config hostname add --resource-group <ResourceGroupName> --webapp-name <AppName> --hostname www.yourdomain.com

Certificats SSL

Què és un certificat SSL?

Un certificat SSL (Secure Sockets Layer) encripta les dades transferides entre el navegador de l'usuari i el teu servidor web, assegurant la privacitat i seguretat de les dades.

Per què utilitzar certificats SSL?

  • Seguretat: Encripta dades sensibles.
  • Confiança: Els usuaris confien en llocs web amb HTTPS.
  • SEO: Els motors de cerca prefereixen llocs web amb HTTPS.

Tipus de certificats SSL

  • Certificats SSL gratuïts: Proporcionats per serveis com Let's Encrypt.
  • Certificats SSL de pagament: Oferts per diverses Autoritats de Certificació (CAs) amb característiques addicionals i garanties.

Afegir un certificat SSL a Azure

  1. Obtenir un certificat SSL:
    • Pots obtenir un certificat SSL gratuït de Let's Encrypt o comprar-ne un a una CA.
  2. Pujar el certificat SSL a Azure:
    • Navega a la teva aplicació web d'Azure.
    • Ves a la secció "Configuració TLS/SSL".
    • Fes clic a "Certificats de clau privada (.pfx)" i puja el teu certificat.
  3. Enllaçar el certificat SSL al domini personalitzat:
    • Després de pujar-lo, ves a la secció "Dominis personalitzats".
    • Selecciona el teu domini personalitzat i enllaça el certificat SSL.

Exemple: Afegir un certificat SSL

# Pujar el certificat SSL a l'aplicació web d'Azure
az webapp config ssl upload --resource-group <ResourceGroupName> --name <AppName> --certificate-file <PathToPfxFile> --certificate-password <Password>

# Enllaçar el certificat SSL al domini personalitzat
az webapp config ssl bind --resource-group <ResourceGroupName> --name <AppName> --certificate-thumbprint <Thumbprint> --ssl-type SNI

Conclusió

Configurar dominis personalitzats i certificats SSL a Azure és un pas crucial per assegurar i donar marca a les teves aplicacions web. Seguint els passos descrits en aquest mòdul, pots millorar la seguretat de la teva aplicació i la confiança dels usuaris. Assegura't sempre que els teus certificats SSL estiguin actualitzats i renova'ls abans que caduquin per mantenir una seguretat contínua.
