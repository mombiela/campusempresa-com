Azure Logic Apps
Aprenentatge Automàtic d'Azure ►

Azure Logic Apps és un servei al núvol que t'ajuda a automatitzar i orquestrar tasques, processos empresarials i fluxos de treball. Aquest tema et guiarà des dels conceptes bàsics fins als avançats d'Azure Logic Apps.

Introducció a Azure Logic Apps

  • Què són Azure Logic Apps?

    • Azure Logic Apps és un servei basat en el núvol que et permet automatitzar fluxos de treball i integrar aplicacions, dades, serveis i sistemes.
    • Proporciona un dissenyador visual per crear fluxos de treball sense escriure codi.

  • Característiques clau:

    • Dissenyador visual: Interfície de drag-and-drop per dissenyar fluxos de treball.
    • Connectors: Connectors preconstruïts per integrar-se amb diversos serveis com Office 365, SQL Server i més.
    • Desencadenants i accions: Defineix què inicia el flux de treball (desencadenant) i què passa després (accions).

Configurant la teva primera Logic App

  • Prerequisits:

    • Un compte d'Azure.
    • Comprensió bàsica dels conceptes del núvol.

  • Creant una Logic App:

    1. Navega al portal d'Azure:
      • Ves al portal d'Azure (https://portal.azure.com).
    2. Crea una Logic App:
      • Fes clic a "Crear un recurs" i cerca "Logic App".
      • Fes clic a "Crear" i omple els detalls necessaris com el grup de recursos, el nom i la regió.
    3. Dissenya el flux de treball:
      • Utilitza el dissenyador visual per afegir un desencadenant i accions.

Exemple: Enviar un correu electrònic quan es puja un fitxer a OneDrive

{
  "definition": {
    "$schema": "https://schema.management.azure.com/providers/Microsoft.Logic/schemas/2016-06-01/workflowdefinition.json#",
    "actions": {
      "Send_an_email_(V2)": {
        "inputs": {
          "body": {
            "Content": "S'ha pujat un nou fitxer a OneDrive.",
            "Subject": "Notificació de pujada de fitxer",
            "To": "example@example.com"
          },
          "host": {
            "connection": {
              "name": "@parameters('$connections')['office365']['connectionId']"
            }
          }
        },
        "runAfter": {},
        "type": "ApiConnection"
      }
    },
    "triggers": {
      "When_a_file_is_created": {
        "inputs": {
          "host": {
            "connection": {
              "name": "@parameters('$connections')['onedrive']['connectionId']"
            }
          },
          "method": "get",
          "path": "/datasets/default/files"
        },
        "recurrence": {
          "frequency": "Minute",
          "interval": 5
        },
        "type": "ApiConnection"
      }
    }
  }
}

Conceptes avançats de Logic Apps

  • Connectors personalitzats:

    • Crea els teus propis connectors per integrar-te amb serveis no disponibles en els connectors preconstruïts.
    • Útil per integrar-se amb sistemes interns o propietaris.

  • Gestió d'errors:

    • Implementa la gestió d'errors per gestionar fallades en els fluxos de treball.
    • Utilitza escopes i configura els paràmetres de run-after per gestionar els errors de manera elegant.

  • Fluxos de treball amb estat vs sense estat:

    • Amb estat: Manté informació d'estat entre execucions.
    • Sense estat: No manté informació d'estat, adequat per a escenaris d'alt rendiment.

Exemple: Gestió d'errors amb escopes

{
  "definition": {
    "actions": {
      "Scope": {
        "actions": {
          "Action1": {
            "type": "Http",
            "inputs": {
              "method": "GET",
              "uri": "https://example.com/api"
            }
          }
        },
        "runAfter": {},
        "type": "Scope"
      },
      "ErrorHandling": {
        "actions": {
          "SendEmail": {
            "type": "ApiConnection",
            "inputs": {
              "body": {
                "Content": "S'ha produït un error en el flux de treball.",
                "Subject": "Notificació d'error",
                "To": "admin@example.com"
              },
              "host": {
                "connection": {
                  "name": "@parameters('$connections')['office365']['connectionId']"
                }
              }
            }
          }
        },
        "runAfter": {
          "Scope": [
            "Failed"
          ]
        },
        "type": "Scope"
      }
    },
    "triggers": {
      "manual": {
        "type": "Request",
        "kind": "Http",
        "inputs": {
          "schema": {}
        }
      }
    }
  }
}

Millors pràctiques

  • Fluxos de treball modulars:

    • Divideix fluxos de treball complexos en components més petits i reutilitzables.
    • Utilitza Logic Apps niades per a una millor gestió.

  • Monitorització i registre:

    • Activa diagnòstics i registre per monitoritzar el rendiment i la salut de les teves Logic Apps.
    • Utilitza Azure Monitor i Application Insights per a anàlisis detallades.

  • Seguretat:

    • Assegura les teves Logic Apps utilitzant Azure Active Directory (AAD) i identitats gestionades.
    • Assegura't que les dades sensibles estiguin encriptades i que l'accés estigui controlat.

Conclusió

Azure Logic Apps proporciona una manera potent i flexible d'automatitzar fluxos de treball i integrar diversos serveis. Comprenent els conceptes bàsics i explorant les funcions avançades, pots crear fluxos de treball eficients i fiables per satisfer les necessitats del teu negoci. Recorda seguir les millors pràctiques per a la modularitat, la monitorització i la seguretat per assegurar que les teves Logic Apps siguin robustes i fàcils de mantenir.
