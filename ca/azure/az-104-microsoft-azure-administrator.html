Introducció

Benvingut al curs AZ-104: Administrador de Microsoft Azure. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat en la gestió i administració de recursos d'Azure. Al final d'aquest curs, estaràs ben preparat per fer l'examen de certificació AZ-104 i excel·lir en el teu rol com a Administrador d'Azure.

Esquema del curs

  1. Introducció a Azure
  2. Gestió de Subscripcions i Recursos d'Azure
  3. Implementació i Gestió d'Emmagatzematge
  4. Desplegament i Gestió de Màquines Virtuals (VMs)
  5. Configuració i Gestió de Xarxes Virtuals
  6. Gestió d'Identitats
  7. Monitorització i Còpia de Seguretat de Recursos d'Azure

1. Introducció a Azure

Què és Azure?

  • Azure és la plataforma de computació al núvol de Microsoft.
  • Proporciona una gamma de serveis que inclouen computació, analítica, emmagatzematge i xarxes.

Conceptes Clau

  • Grups de Recursos: Contenidors lògics per als recursos desplegats a Azure.
  • Regions: Àrees geogràfiques on es troben els centres de dades d'Azure.
  • Zones de Disponibilitat: Ubicacions físicament separades dins d'una regió d'Azure.

Exemple: Creació d'un Grup de Recursos

# Crear un grup de recursos a la regió Est dels EUA
az group create --name MyResourceGroup --location eastus

Explicació: Aquesta comanda crea un nou grup de recursos anomenat MyResourceGroup a la regió eastus.

2. Gestió de Subscripcions i Recursos d'Azure

Subscripcions d'Azure

  • Una subscripció és un contenidor lògic utilitzat per aprovisionar recursos a Azure.
  • Cada subscripció té un ID únic i està vinculada a un compte de facturació.

Gestió de Recursos

  • Azure Resource Manager (ARM): El servei de desplegament i gestió per a Azure.
  • Etiquetes: Parells clau-valor que ajuden a organitzar recursos.

Exemple: Aplicació d'Etiquetes a un Recurs

# Aplicar una etiqueta a un recurs
az resource tag --tags Environment=Production --resource-id /subscriptions/{subscription-id}/resourceGroups/{resource-group}/providers/Microsoft.Compute/virtualMachines/{vm-name}

Explicació: Aquesta comanda aplica l'etiqueta Environment=Production a una màquina virtual especificada.

3. Implementació i Gestió d'Emmagatzematge

Tipus de Comptes d'Emmagatzematge

  • Blob Storage: Optimitzat per emmagatzemar grans quantitats de dades no estructurades.
  • File Storage: Comparticions de fitxers gestionades que es poden accedir mitjançant el protocol SMB.
  • Queue Storage: Proporciona cua de missatges per a grans càrregues de treball.
  • Table Storage: Emmagatzematge NoSQL de clau-valor per a desenvolupament ràpid.

Exemple: Creació d'un Compte d'Emmagatzematge

# Crear un compte d'emmagatzematge
az storage account create --name mystorageaccount --resource-group MyResourceGroup --location eastus --sku Standard_LRS

Explicació: Aquesta comanda crea un nou compte d'emmagatzematge anomenat mystorageaccount al grup de recursos MyResourceGroup amb l'SKU Standard_LRS (Emmagatzematge Redundant Localment).

4. Desplegament i Gestió de Màquines Virtuals (VMs)

Creació de Màquines Virtuals

  • Azure Marketplace: Ofereix una àmplia gamma d'imatges de VM preconfigurades.
  • Imatges Personalitzades: Crear VMs a partir d'imatges personalitzades.

Exemple: Creació d'una Màquina Virtual

# Crear una màquina virtual
az vm create --resource-group MyResourceGroup --name MyVM --image UbuntuLTS --admin-username azureuser --generate-ssh-keys

Explicació: Aquesta comanda crea una nova màquina virtual anomenada MyVM al grup de recursos MyResourceGroup utilitzant la imatge UbuntuLTS.

5. Configuració i Gestió de Xarxes Virtuals

Xarxes Virtuals (VNet)

  • Subxarxes: Segments dins d'una VNet.
  • Grups de Seguretat de Xarxa (NSGs): Controlen el trànsit entrant i sortint.

Exemple: Creació d'una Xarxa Virtual i Subxarxa

# Crear una xarxa virtual
az network vnet create --resource-group MyResourceGroup --name MyVNet --address-prefix 10.0.0.0/16

# Crear una subxarxa
az network vnet subnet create --resource-group MyResourceGroup --vnet-name MyVNet --name MySubnet --address-prefix 10.0.1.0/24

Explicació: Aquestes comandes creen una xarxa virtual anomenada MyVNet amb un espai d'adreces de 10.0.0.0/16 i una subxarxa anomenada MySubnet amb un prefix d'adreça de 10.0.1.0/24.

6. Gestió d'Identitats

Azure Active Directory (AAD)

  • Usuaris i Grups: Gestionar l'accés als recursos.
  • Rols i Assignacions de Rols: Definir permisos.

Exemple: Creació d'un Usuari a Azure AD

# Crear un usuari a Azure AD
az ad user create --display-name JohnDoe --user-principal-name johndoe@mydomain.com --password P@ssw0rd!

Explicació: Aquesta comanda crea un nou usuari anomenat JohnDoe amb el nom principal d'usuari johndoe@mydomain.com.

7. Monitorització i Còpia de Seguretat de Recursos d'Azure

Monitorització

  • Azure Monitor: Proporciona monitorització de pila completa.
  • Log Analytics: Recull i analitza dades de registres.

Exemple: Configuració d'Azure Monitor

# Crear un espai de treball de Log Analytics
az monitor log-analytics workspace create --resource-group MyResourceGroup --workspace-name MyWorkspace

# Activar la monitorització per a una VM
az monitor diagnostic-settings create --resource /subscriptions/{subscription-id}/resourceGroups/{resource-group}/providers/Microsoft.Compute/virtualMachines/{vm-name} --workspace MyWorkspace --name MyDiagnosticSetting --metrics '[{"category": "AllMetrics", "enabled": true}]' --logs '[{"category": "AuditLogs", "enabled": true}]'

Explicació: Aquestes comandes creen un espai de treball de Log Analytics anomenat MyWorkspace i activen la monitorització per a una màquina virtual especificada.

Conclusió

En aquest curs, hem cobert els temes essencials necessaris per convertir-te en un Administrador d'Azure competent. Des de comprendre els conceptes bàsics d'Azure fins a gestionar recursos complexos i solucions de monitorització, ara tens els coneixements i habilitats per gestionar entorns d'Azure de manera efectiva. Continua practicant i explorant els serveis d'Azure per aprofundir en la teva comprensió i preparar-te per a l'examen de certificació AZ-104. Bona sort!
