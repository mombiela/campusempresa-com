Introducció a Microsoft Azure

Microsoft Azure és una plataforma i servei de computació en el núvol creat per Microsoft. Proporciona una gamma de serveis en el núvol, incloent-hi serveis de computació, anàlisi, emmagatzematge i xarxes. Els usuaris poden triar entre aquests serveis per desenvolupar i escalar noves aplicacions, o executar aplicacions existents, en el núvol públic.

Conceptes clau de Microsoft Azure

  • Computació en el núvol: El lliurament de serveis de computació a través d'Internet ("el núvol") per oferir una innovació més ràpida, recursos flexibles i economies d'escala.
  • Portal d'Azure: Una aplicació web que et permet gestionar els teus recursos d'Azure.
  • Grups de recursos: Contenidors que contenen recursos relacionats per a una solució d'Azure.
  • Màquines virtuals (VMs): Sistemes informàtics emulats que proporcionen la funcionalitat d'un ordinador físic.
  • Emmagatzematge d'Azure: Emmagatzematge en el núvol escalable per a dades, fitxers i còpies de seguretat.
  • Xarxes d'Azure: Serveis que proporcionen capacitats de xarxa per connectar i gestionar els teus recursos d'Azure.

Visió general dels serveis d'Azure

Serveis de computació

  • Màquines virtuals: Recursos de computació escalables i sota demanda.
  • Serveis d'aplicacions: Entorn d'allotjament gestionat per a aplicacions web.
  • Funcions d'Azure: Servei de computació sense servidor i impulsat per esdeveniments.

Serveis d'emmagatzematge

  • Emmagatzematge de Blobs: Emmagatzematge d'objectes per a dades no estructurades.
  • Emmagatzematge de fitxers: Comparticions de fitxers totalment gestionades en el núvol.
  • Emmagatzematge de cues: Servei de missatgeria per a la comunicació entre components d'aplicacions.

Serveis de xarxes

  • Xarxa virtual (VNet): Permet que molts tipus de recursos d'Azure es comuniquin de manera segura entre ells.
  • Equilibrador de càrrega: Distribueix el trànsit de xarxa entrant entre múltiples destinacions.
  • Passarel·la VPN: Estableix una connectivitat segura entre premisses.

Serveis de bases de dades

  • Base de dades SQL d'Azure: Servei de base de dades relacional gestionat.
  • Cosmos DB: Servei de base de dades multimodel distribuït globalment.
  • Base de dades d'Azure per a MySQL/PostgreSQL: Serveis de bases de dades gestionats per a MySQL i PostgreSQL.

Com començar amb Azure

Crear un compte d'Azure

Per començar a utilitzar Azure, necessites crear un compte. Microsoft ofereix un nivell gratuït que inclou una quantitat limitada de serveis gratuïts durant 12 mesos.

Navegar pel portal d'Azure

El portal d'Azure és la teva interfície principal per gestionar els recursos d'Azure. Aquí tens una guia ràpida per començar:

  1. Tauler: La teva pàgina d'inici personalitzable.
  2. Tots els serveis: Una llista de tots els serveis disponibles d'Azure.
  3. Grups de recursos: Gestiona els teus grups de recursos.
  4. Subscripcions: Visualitza i gestiona les teves subscripcions d'Azure.

Crear una màquina virtual

Crear una VM és un dels primers passos per obtenir experiència pràctica amb Azure. Aquí tens un exemple senzill utilitzant el portal d'Azure:

  1. Navega al portal d'Azure.
  2. Fes clic a "Crear un recurs".
  3. Selecciona "Màquina virtual".
  4. Omple els detalls necessaris:
    • Subscripció: Tria la teva subscripció.
    • Grup de recursos: Crea un nou grup de recursos o selecciona'n un d'existent.
    • Nom de la VM: Posa un nom a la teva VM.
    • Regió: Tria la regió on vols desplegar la teva VM.
    • Imatge: Selecciona la imatge del sistema operatiu (per exemple, Windows Server, Ubuntu).
    • Mida: Tria la mida de la VM segons les teves necessitats.
  5. Configura la resta de paràmetres i fes clic a "Revisar + crear".
  6. Revisa els teus paràmetres i fes clic a "Crear".
# Exemple de creació d'una VM utilitzant Azure CLI
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Conclusió

Microsoft Azure és una plataforma de núvol completa que ofereix una àmplia gamma de serveis per satisfer diverses necessitats de computació. Des de màquines virtuals fins a bases de dades i xarxes, Azure proporciona les eines necessàries per construir, desplegar i gestionar aplicacions en el núvol. Entendre els conceptes bàsics d'Azure, com crear un compte, navegar pel portal i desplegar recursos, estableix la base per a temes i serveis més avançats. A mesura que avancis en aquest curs, obtindràs coneixements més profunds i experiència pràctica amb les potents capacitats d'Azure.
