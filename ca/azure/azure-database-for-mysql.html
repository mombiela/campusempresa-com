Introducció a Azure Database for MySQL

Azure Database for MySQL és un servei de base de dades totalment gestionat proporcionat per Microsoft Azure. Ofereix alta disponibilitat integrada, còpies de seguretat automatitzades i capacitats d'escalat, fent-lo una excel·lent elecció per als desenvolupadors que volen centrar-se en construir aplicacions sense preocupar-se per la gestió de la base de dades.

Característiques Clau

  • Servei Totalment Gestionat: Còpies de seguretat automatitzades, aplicació de pegats i manteniment.
  • Alta Disponibilitat: Alta disponibilitat integrada sense configuració addicional.
  • Escalabilitat: Escala el càlcul i l'emmagatzematge de manera independent.
  • Seguretat: Funcions avançades de seguretat, incloent xifratge de dades en repòs i en trànsit.
  • Compliment: Compleix amb diversos estàndards de compliment com GDPR, HIPAA i ISO.

Configuració d'Azure Database for MySQL

Creació d'una Base de Dades MySQL

  1. Navega al Portal d'Azure: Ves al Portal d'Azure.

  2. Crea un Recurs: Fes clic a "Crear un recurs" i cerca "Azure Database for MySQL".

  3. Configura els Paràmetres Bàsics:

    • Subscripció: Tria la teva subscripció d'Azure.
    • Grup de Recursos: Crea un nou grup de recursos o selecciona'n un d'existent.
    • Nom del Servidor: Proporciona un nom únic per al teu servidor MySQL.
    • Regió: Selecciona la regió més propera als teus usuaris.
    • Nom d'Usuari i Contrasenya de l'Administrador: Estableix el nom d'usuari i la contrasenya de l'administrador.

  4. Configura el Nivell de Preus: Tria el nivell de preus adequat segons les teves necessitats.

  5. Revisa i Crea: Revisa la teva configuració i fes clic a "Crear".

Connexió a la Teva Base de Dades MySQL

Per connectar-te a la teva base de dades MySQL, pots utilitzar diverses eines com MySQL Workbench, Azure Data Studio o eines de línia de comandes.

Utilitzant MySQL Workbench

  1. Descarrega i Instal·la MySQL Workbench: Si no el tens instal·lat, descarrega'l des del lloc web oficial.

  2. Crea una Nova Connexió:

    • Nom del Host: El nom del teu servidor seguit de .mysql.database.azure.com.
    • Port: 3306 (port per defecte de MySQL).
    • Nom d'Usuari: El teu nom d'usuari d'administrador seguit de @elnomdelteuserver.
    • Contrasenya: La teva contrasenya d'administrador.

  3. Prova la Connexió: Fes clic a "Prova la Connexió" per assegurar-te que pots connectar-te a la base de dades.

  4. Desa i Connecta: Desa la configuració de la connexió i connecta't a la teva base de dades.

Gestió de la Teva Base de Dades MySQL

Escalat de la Teva Base de Dades

Azure Database for MySQL et permet escalar el càlcul i l'emmagatzematge de manera independent.

Escalat del Càlcul

  1. Navega al Teu Servidor MySQL: Ves al Portal d'Azure i selecciona el teu servidor MySQL.
  2. Càlcul + Emmagatzematge: Fes clic a "Càlcul + Emmagatzematge" a la secció "Configuració".
  3. Ajusta el Càlcul: Utilitza el control lliscant per ajustar els vCores i la memòria.
  4. Desa els Canvis: Fes clic a "Desa" per aplicar els canvis.

Escalat de l'Emmagatzematge

  1. Navega al Teu Servidor MySQL: Ves al Portal d'Azure i selecciona el teu servidor MySQL.
  2. Càlcul + Emmagatzematge: Fes clic a "Càlcul + Emmagatzematge" a la secció "Configuració".
  3. Ajusta l'Emmagatzematge: Utilitza el control lliscant per ajustar la mida de l'emmagatzematge.
  4. Desa els Canvis: Fes clic a "Desa" per aplicar els canvis.

Còpia de Seguretat i Restauració

Azure Database for MySQL proporciona còpies de seguretat automatitzades i capacitats de restauració en un punt en el temps.

Còpies de Seguretat Automatitzades

  • Període de Retenció: Per defecte, les còpies de seguretat es retenen durant 7 dies, però es pot configurar fins a 35 dies.
  • Freqüència de les Còpies de Seguretat: Es fan còpies de seguretat completes setmanalment, còpies de seguretat diferencials diàriament i còpies de seguretat del registre de transaccions cada 5 minuts.

Restauració en un Punt en el Temps

  1. Navega al Teu Servidor MySQL: Ves al Portal d'Azure i selecciona el teu servidor MySQL.
  2. Restaurar: Fes clic a "Restaurar" a la secció "Configuració".
  3. Punt de Restauració: Selecciona el punt en el temps al qual vols restaurar.
  4. Nou Nom de Servidor: Proporciona un nou nom de servidor per a la base de dades restaurada.
  5. Restaurar: Fes clic a "D'acord" per iniciar el procés de restauració.

Configuració Avançada

Optimització del Rendiment

  • Query Performance Insight: Utilitza Query Performance Insight per identificar consultes lentes.
  • Indexació: Assegura't que les teves taules estan correctament indexades per millorar el rendiment de les consultes.
  • Pooling de Connexions: Utilitza el pooling de connexions per gestionar les connexions a la base de dades de manera eficient.

Millors Pràctiques de Seguretat

  • Regles de Tallafocs: Configura regles de tallafocs per restringir l'accés a la teva base de dades.
  • Xifratge SSL/TLS: Assegura't que el xifratge SSL/TLS està habilitat per a les dades en trànsit.
  • Autenticació d'Azure AD: Utilitza l'Autenticació d'Azure Active Directory per millorar la seguretat.

Conclusió

Azure Database for MySQL proporciona un servei de base de dades robust i totalment gestionat que simplifica les tasques de gestió de bases de dades. Aprofitant les seves característiques com les còpies de seguretat automatitzades, l'escalat i la seguretat avançada, els desenvolupadors poden centrar-se en construir aplicacions d'alt rendiment. Tant si ets un usuari principiant com avançat, Azure Database for MySQL ofereix les eines i capacitats per satisfer les teves necessitats de base de dades de manera eficient.
