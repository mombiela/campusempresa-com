Introducció a Azure Storage
Emmagatzematge de Blobs ►

Azure Storage és una solució d'emmagatzematge al núvol proporcionada per Microsoft Azure que ofereix opcions d'emmagatzematge altament disponibles, segures, duradores i escalables. Aquesta secció et guiarà a través dels conceptes bàsics d'Azure Storage, els seus components clau i com començar a utilitzar-lo.

Conceptes Clau d'Azure Storage

  • Blob Storage: Utilitzat per emmagatzemar dades no estructurades com text o dades binàries. És ideal per servir imatges o documents directament a un navegador, emmagatzemar fitxers per a accés distribuït i transmetre vídeo i àudio.
  • File Storage: Proporciona comparticions de fitxers totalment gestionades al núvol que són accessibles mitjançant el protocol Server Message Block (SMB).
  • Queue Storage: Et permet emmagatzemar grans quantitats de missatges que poden ser accessibles des de qualsevol lloc mitjançant trucades autenticades utilitzant HTTP o HTTPS.
  • Table Storage: Un magatzem de valors clau NoSQL per al desenvolupament ràpid utilitzant conjunts de dades semi-estructurades massives.

Blob Storage

Blob Storage està dissenyat per emmagatzemar grans quantitats de dades no estructurades. Admet tres tipus de blobs:

  • Block Blobs: Optimitzats per a la transmissió i l'emmagatzematge d'objectes al núvol.
  • Append Blobs: Optimitzats per a operacions d'addició, cosa que els fa ideals per a registres.
  • Page Blobs: Optimitzats per a operacions de lectura/escriptura aleatòries i utilitzats per a fitxers de disc dur virtual (VHD).

Exemple: Pujant un Blob

from azure.storage.blob import BlobServiceClient, BlobClient, ContainerClient

# Inicialitza una connexió al servei de Blob
blob_service_client = BlobServiceClient.from_connection_string("your_connection_string")

# Crea un contenidor
container_name = "mycontainer"
container_client = blob_service_client.create_container(container_name)

# Puja un blob
blob_client = blob_service_client.get_blob_client(container=container_name, blob="myblob.txt")
with open("sample.txt", "rb") as data:
    blob_client.upload_blob(data)

File Storage

Azure File Storage et permet crear comparticions de fitxers al núvol que poden ser accessibles mitjançant el protocol SMB. Això és útil per a aplicacions que necessiten emmagatzematge compartit accessible per múltiples màquines virtuals.

Exemple: Muntant una Compartició de Fitxers d'Azure

# Instal·la cifs-utils si no està ja instal·lat
sudo apt-get install cifs-utils

# Munta la compartició de fitxers
sudo mount -t cifs //your_storage_account.file.core.windows.net/your_file_share /mnt/your_mount_point -o vers=3.0,username=your_storage_account,password=your_storage_account_key,dir_mode=0777,file_mode=0777,sec=ntlmssp

Queue Storage

Queue Storage s'utilitza per emmagatzemar i recuperar missatges. És útil per desacoblar components d'una aplicació al núvol per millorar l'escalabilitat i la fiabilitat.

Exemple: Afegint un Missatge a una Cua

from azure.storage.queue import QueueServiceClient

# Inicialitza una connexió al servei de Queue
queue_service_client = QueueServiceClient.from_connection_string("your_connection_string")

# Crea una cua
queue_name = "myqueue"
queue_client = queue_service_client.create_queue(queue_name)

# Afegeix un missatge a la cua
queue_client.send_message("Hello, Azure Queue!")

Table Storage

Table Storage és un magatzem de dades NoSQL que permet l'emmagatzematge de grans quantitats de dades estructurades. És sense esquema, cosa que el fa flexible i escalable.

Exemple: Inserint una Entitat en una Taula

from azure.data.tables import TableServiceClient, TableEntity

# Inicialitza una connexió al servei de Table
table_service_client = TableServiceClient.from_connection_string("your_connection_string")

# Crea una taula
table_name = "mytable"
table_client = table_service_client.create_table_if_not_exists(table_name)

# Insereix una entitat
entity = TableEntity()
entity.PartitionKey = "partition1"
entity.RowKey = "row1"
entity["Name"] = "John Doe"
entity["Age"] = 30

table_client.create_entity(entity=entity)

Resum

Azure Storage proporciona una varietat de solucions d'emmagatzematge per satisfer diferents necessitats, des de l'emmagatzematge de dades no estructurades amb Blob Storage fins a l'emmagatzematge de dades estructurades amb Table Storage. Entendre aquests serveis i com utilitzar-los és crucial per construir aplicacions al núvol escalables i fiables. Aquesta introducció ha cobert els conceptes bàsics de cada tipus d'emmagatzematge, juntament amb exemples per ajudar-te a començar. A mesura que progressis, aprendràs funcions més avançades i millors pràctiques per utilitzar Azure Storage de manera efectiva.
