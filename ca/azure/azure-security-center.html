Azure Security Center és un sistema unificat de gestió de la seguretat de la infraestructura que enforteix la postura de seguretat dels teus centres de dades i proporciona protecció avançada contra amenaces a través de les teves càrregues de treball híbrides al núvol i a les instal·lacions locals.

Introducció a Azure Security Center

Azure Security Center t'ajuda a prevenir, detectar i respondre a les amenaces amb una major visibilitat i control sobre la seguretat dels teus recursos d'Azure.

Conceptes Clau

  • Gestió de la Postura de Seguretat: Avalua contínuament els teus recursos per detectar vulnerabilitats de seguretat.
  • Protecció contra Amenaces: Proporciona capacitats avançades de detecció i resposta a amenaces.
  • Recomanacions de Seguretat: Ofereix recomanacions accionables per millorar la teva postura de seguretat.
  • Compliment: T'ajuda a complir amb els requisits de compliment normatiu.

Configuració d'Azure Security Center

Per començar a utilitzar Azure Security Center, has d'habilitar-lo a la teva subscripció d'Azure.

Passos per Habilitar Azure Security Center

  1. Navega al Portal d'Azure: Ves al portal d'Azure a https://portal.azure.com.
  2. Cerca Security Center: A la barra de cerca, escriu "Security Center" i selecciona'l dels resultats.
  3. Habilita Security Center: Segueix les instruccions per habilitar Security Center per a la teva subscripció.
# Exemple de comanda Azure CLI per habilitar Azure Security Center
az security auto-provisioning-setting update --name default --auto-provision "On"

Gestió de la Postura de Seguretat

Azure Security Center avalua contínuament l'estat de seguretat dels teus recursos. Proporciona una puntuació de seguretat que reflecteix la teva postura de seguretat.

Puntuació de Seguretat

  • Definició: Un valor numèric que representa la salut de seguretat dels teus recursos.
  • Millora: Segueix les recomanacions de seguretat per millorar la teva puntuació de seguretat.

Recomanacions de Seguretat

Security Center proporciona recomanacions per ajudar-te a assegurar els teus recursos. Aquestes recomanacions es basen en les millors pràctiques de seguretat.

{
  "recommendations": [
    {
      "name": "Habilitar MFA",
      "description": "Habilita l'Autenticació Multifactor per a tots els usuaris."
    },
    {
      "name": "Aplicar Actualitzacions de Seguretat",
      "description": "Assegura't que totes les màquines virtuals tinguin instal·lades les últimes actualitzacions de seguretat."
    }
  ]
}

Protecció contra Amenaces

Azure Security Center proporciona protecció avançada contra amenaces per a les teves càrregues de treball.

Detecció d'Amenaces

  • Alertes: Security Center genera alertes quan detecta amenaces.
  • Investigació: Utilitza les alertes per investigar i respondre a possibles incidents de seguretat.

Exemple: Detecció d'un Atac de Força Bruta

Security Center pot detectar un atac de força bruta a les teves màquines virtuals.

{
  "alert": {
    "name": "Atac de Força Bruta",
    "description": "S'han detectat múltiples intents fallits d'inici de sessió a la VM 'myVM'.",
    "severity": "Alta"
  }
}

Resposta a les Amenaces

  • Resposta Automàtica: Configura respostes automàtiques a certs tipus d'amenaces.
  • Resposta Manual: Investiga i respon manualment a amenaces més complexes.

Compliment

Azure Security Center t'ajuda a complir amb els requisits de compliment normatiu proporcionant informes de compliment i avaluació contínua.

Tauler de Compliment

  • Visió General: Proporciona una visió general del teu estat de compliment.
  • Informes: Genera informes de compliment per a diversos estàndards (per exemple, ISO 27001, PCI DSS).

Resum

Azure Security Center és una eina poderosa per gestionar la seguretat dels teus recursos d'Azure. Proporciona una avaluació contínua de la seguretat, protecció avançada contra amenaces i t'ajuda a complir amb els requisits de compliment. Seguint les recomanacions i responent a les alertes, pots millorar significativament la teva postura de seguretat i protegir les teves càrregues de treball contra amenaces.
