Introducció a les bases de dades d'Azure

Azure ofereix una varietat de serveis de bases de dades que s'adapten a diferents necessitats, des de bases de dades relacionals fins a bases de dades NoSQL. Aquesta secció proporcionarà una visió general dels diferents tipus de bases de dades disponibles a Azure i els seus casos d'ús.

  • Azure SQL Database: Una base de dades relacional totalment gestionada amb intel·ligència integrada.
  • Azure Cosmos DB: Un servei de base de dades multi-model distribuït globalment.
  • Azure Database for MySQL: Un servei de base de dades MySQL gestionat.
  • Azure Database for PostgreSQL: Un servei de base de dades PostgreSQL gestionat.
  • Azure Database for MariaDB: Un servei de base de dades MariaDB gestionat.

Azure SQL Database

Azure SQL Database és un servei de base de dades relacional totalment gestionat que proporciona alta disponibilitat, escalabilitat i seguretat.

Característiques clau

  • Servei gestionat: Còpies de seguretat automàtiques, aplicació de pegats i actualitzacions.
  • Escalabilitat: Escala cap amunt o cap avall segons les teves necessitats.
  • Seguretat: Protecció avançada contra amenaces i xifratge.
  • Rendiment: Intel·ligència integrada per a l'optimització del rendiment.

Començar amb Azure SQL Database

  1. Crear una Azure SQL Database:

    az sql db create --resource-group myResourceGroup --server myServer --name myDatabase --service-objective S0

  2. Connectar-se a Azure SQL Database:

    import pyodbc

server = 'myServer.database.windows.net'
database = 'myDatabase'
username = 'myUsername'
password = 'myPassword'
driver= '{ODBC Driver 17 for SQL Server}'

cnxn = pyodbc.connect('DRIVER=' + driver + ';SERVER=' + server + ';PORT=1433;DATABASE=' + database + ';UID=' + username + ';PWD=' + password)
cursor = cnxn.cursor()

  3. Realitzar operacions CRUD:

    # Crear
cursor.execute("INSERT INTO myTable (name, age) VALUES (?, ?)", ('John Doe', 30))
cnxn.commit()

# Llegir
cursor.execute("SELECT * FROM myTable")
for row in cursor:
    print(row)

# Actualitzar
cursor.execute("UPDATE myTable SET age = ? WHERE name = ?", (31, 'John Doe'))
cnxn.commit()

# Esborrar
cursor.execute("DELETE FROM myTable WHERE name = ?", ('John Doe',))
cnxn.commit()

Azure Cosmos DB

Azure Cosmos DB és un servei de base de dades multi-model distribuït globalment dissenyat per a alta disponibilitat i baixa latència.

Característiques clau

  • Distribució global: Distribueix dades a través de múltiples regions.
  • Multi-model: Suporta models de dades de documents, clau-valor, gràfics i columnes.
  • Escalabilitat: Escalament automàtic del rendiment i l'emmagatzematge.
  • Nivells de consistència: Tria entre cinc nivells de consistència per equilibrar rendiment i consistència.

Començar amb Azure Cosmos DB

  1. Crear un compte d'Azure Cosmos DB:

    az cosmosdb create --name myCosmosDBAccount --resource-group myResourceGroup --kind MongoDB

  2. Connectar-se a Azure Cosmos DB:

    from pymongo import MongoClient

client = MongoClient("mongodb://myCosmosDBAccount.documents.azure.com:10255/?ssl=true&replicaSet=globaldb", username="myUsername", password="myPassword")
db = client.myDatabase

  3. Realitzar operacions CRUD:

    # Crear
db.myCollection.insert_one({"name": "John Doe", "age": 30})

# Llegir
for doc in db.myCollection.find():
    print(doc)

# Actualitzar
db.myCollection.update_one({"name": "John Doe"}, {"$set": {"age": 31}})

# Esborrar
db.myCollection.delete_one({"name": "John Doe"})

Azure Database for MySQL

Azure Database for MySQL és un servei de base de dades MySQL gestionat que proporciona alta disponibilitat i seguretat.

Característiques clau

  • Servei gestionat: Còpies de seguretat automàtiques, aplicació de pegats i actualitzacions.
  • Escalabilitat: Escala el càlcul i l'emmagatzematge de manera independent.
  • Seguretat: Protecció avançada contra amenaces i xifratge.
  • Rendiment: Monitorització i ajustament del rendiment integrats.

Començar amb Azure Database for MySQL

  1. Crear una Azure Database for MySQL:

    az mysql server create --resource-group myResourceGroup --name myMySQLServer --admin-user myAdmin --admin-password myPassword --sku-name GP_Gen5_2

  2. Connectar-se a Azure Database for MySQL:

    import mysql.connector

cnx = mysql.connector.connect(user='myAdmin', password='myPassword',
                              host='myMySQLServer.mysql.database.azure.com',
                              database='myDatabase')
cursor = cnx.cursor()

  3. Realitzar operacions CRUD:

    # Crear
cursor.execute("INSERT INTO myTable (name, age) VALUES (%s, %s)", ('John Doe', 30))
cnx.commit()

# Llegir
cursor.execute("SELECT * FROM myTable")
for row in cursor:
    print(row)

# Actualitzar
cursor.execute("UPDATE myTable SET age = %s WHERE name = %s", (31, 'John Doe'))
cnx.commit()

# Esborrar
cursor.execute("DELETE FROM myTable WHERE name = %s", ('John Doe',))
cnx.commit()

Conclusió

Azure ofereix una àmplia gamma de serveis de bases de dades que s'adapten a diverses necessitats, des de bases de dades relacionals com Azure SQL Database, MySQL, PostgreSQL i MariaDB fins a bases de dades NoSQL com Azure Cosmos DB. Cada servei proporciona característiques i beneficis únics, facilitant l'elecció de la base de dades adequada per a la teva aplicació. Comprenent les característiques clau i començant amb operacions CRUD bàsiques, pots aprofitar el poder de les bases de dades d'Azure per construir aplicacions escalables, segures i d'alt rendiment.
