Visió General

La Xarxa d'Azure és un component crític de Microsoft Azure, proporcionant la infraestructura i els serveis necessaris per connectar i gestionar els teus recursos al núvol. Aquesta secció t'introduirà als conceptes fonamentals i serveis dins de la Xarxa d'Azure, ajudant-te a entendre com dissenyar i implementar una arquitectura de xarxa robusta a Azure.

Conceptes Clau

  • Xarxes Virtuals (VNets): La base de la xarxa d'Azure, permetent-te crear xarxes aïllades dins del núvol d'Azure.
  • Subxarxes: Segments dins d'una VNet que ajuden a organitzar i assegurar els recursos.
  • Grups de Seguretat de Xarxa (NSGs): Regles que controlen el trànsit entrant i sortint cap a les interfícies de xarxa (NICs), VMs i subxarxes.
  • Azure Load Balancer: Distribueix el trànsit de xarxa entrant entre múltiples VMs per assegurar alta disponibilitat i fiabilitat.
  • Azure VPN Gateway: Estableix una connectivitat segura entre les teves VNets d'Azure i les xarxes locals.
  • Azure ExpressRoute: Proporciona una connexió dedicada i privada entre la teva infraestructura local i els centres de dades d'Azure.
  • Azure DNS: Un servei d'allotjament per a dominis DNS, proporcionant resolució de noms utilitzant la infraestructura de Microsoft Azure.
  • Azure Traffic Manager: Un equilibrador de càrrega de trànsit basat en DNS que et permet distribuir el trànsit de manera òptima als serveis a través de les regions globals d'Azure.

Xarxes Virtuals (VNets)

Creació d'una Xarxa Virtual

Una Xarxa Virtual (VNet) és una representació de la teva pròpia xarxa al núvol. És un aïllament lògic del núvol d'Azure dedicat a la teva subscripció.

# Crear un grup de recursos
az group create --name MyResourceGroup --location eastus

# Crear una xarxa virtual
az network vnet create \
  --resource-group MyResourceGroup \
  --name MyVNet \
  --address-prefix 10.0.0.0/16 \
  --subnet-name MySubnet \
  --subnet-prefix 10.0.1.0/24

Punts Clau

  • Les VNets es poden segmentar en subxarxes.
  • Les VNets es poden connectar a altres VNets o xarxes locals.
  • Les VNets suporten diversos serveis d'Azure com VMs, Azure Kubernetes Service (AKS), i més.

Subxarxes

Les subxarxes et permeten dividir la teva VNet en seccions més petites i gestionables. Cada subxarxa pot allotjar diferents serveis i aplicar diferents regles de seguretat.

# Afegir una subxarxa a una VNet existent
az network vnet subnet create \
  --resource-group MyResourceGroup \
  --vnet-name MyVNet \
  --name MyAdditionalSubnet \
  --address-prefix 10.0.2.0/24

Punts Clau

  • Les subxarxes ajuden a organitzar i assegurar els recursos.
  • Cada subxarxa pot tenir els seus propis Grups de Seguretat de Xarxa (NSGs).

Grups de Seguretat de Xarxa (NSGs)

Els NSGs contenen regles de seguretat que permeten o deneguen el trànsit entrant i sortint cap als recursos.

# Crear un Grup de Seguretat de Xarxa
az network nsg create \
  --resource-group MyResourceGroup \
  --name MyNSG

# Crear una regla de seguretat
az network nsg rule create \
  --resource-group MyResourceGroup \
  --nsg-name MyNSG \
  --name AllowSSH \
  --protocol Tcp \
  --direction Inbound \
  --source-address-prefix '*' \
  --source-port-range '*' \
  --destination-address-prefix '*' \
  --destination-port-range 22 \
  --access Allow \
  --priority 1000

Punts Clau

  • Els NSGs es poden associar amb subxarxes o interfícies de xarxa individuals.
  • Les regles es valoren en ordre de prioritat, amb números més baixos tenint major prioritat.

Azure Load Balancer

Azure Load Balancer distribueix el trànsit entrant entre instàncies saludables dels serveis a la teva VNet.

# Crear una adreça IP pública
az network public-ip create \
  --resource-group MyResourceGroup \
  --name MyPublicIP

# Crear un equilibrador de càrrega
az network lb create \
  --resource-group MyResourceGroup \
  --name MyLoadBalancer \
  --public-ip-address MyPublicIP \
  --frontend-ip-name MyFrontEndPool \
  --backend-pool-name MyBackEndPool

Punts Clau

  • Suporta tant l'equilibrament de càrrega públic com intern.
  • Assegura alta disponibilitat i fiabilitat distribuint el trànsit.

Azure VPN Gateway

Azure VPN Gateway proporciona connectivitat segura entre premisses.

# Crear una passarel·la de xarxa virtual
az network vnet-gateway create \
  --resource-group MyResourceGroup \
  --name MyVPNGateway \
  --public-ip-address MyPublicIP \
  --vnet MyVNet \
  --gateway-type Vpn \
  --vpn-type RouteBased \
  --sku VpnGw1 \
  --no-wait

Punts Clau

  • Suporta connexions site-to-site, point-to-site i VNet-to-VNet.
  • Proporciona connexions segures i encriptades.

Azure ExpressRoute

Azure ExpressRoute ofereix una connexió privada a Azure, evitant l'internet públic.

Punts Clau

  • Proporciona major seguretat, fiabilitat i velocitat comparat amb VPN.
  • Ideal per a escenaris que requereixen connectivitat consistent i d'alt rendiment.

Azure DNS

Azure DNS et permet allotjar els teus dominis DNS a Azure.

# Crear una zona DNS
az network dns zone create \
  --resource-group MyResourceGroup \
  --name mydomain.com

# Crear un registre DNS
az network dns record-set a add-record \
  --resource-group MyResourceGroup \
  --zone-name mydomain.com \
  --record-set-name www \
  --ipv4-address 10.0.0.4

Punts Clau

  • Proporciona serveis DNS ràpids i fiables.
  • Integrat amb altres serveis d'Azure.

Azure Traffic Manager

Azure Traffic Manager és un equilibrador de càrrega de trànsit basat en DNS.

# Crear un perfil de Traffic Manager
az network traffic-manager profile create \
  --resource-group MyResourceGroup \
  --name MyTrafficManagerProfile \
  --routing-method Performance \
  --unique-dns-name myapptrafficmanager

# Afegir un punt final al perfil de Traffic Manager
az network traffic-manager endpoint create \
  --resource-group MyResourceGroup \
  --profile-name MyTrafficManagerProfile \
  --name MyEndpoint \
  --type azureEndpoints \
  --target-resource-id /subscriptions/{subscription-id}/resourceGroups/MyResourceGroup/providers/Microsoft.Network/publicIPAddresses/MyPublicIP

Punts Clau

  • Distribueix el trànsit a través de múltiples regions.
  • Millora la disponibilitat i la capacitat de resposta de les aplicacions.

Conclusió

La Xarxa d'Azure proporciona una suite completa de serveis per construir, gestionar i assegurar la teva infraestructura de xarxa al núvol. Entenent i utilitzant aquests serveis, pots crear arquitectures de xarxa escalables, fiables i segures que compleixin amb les necessitats del teu negoci. Ja sigui connectant VNets, assegurant recursos amb NSGs, o optimitzant el trànsit amb Load Balancers i Traffic Manager, la Xarxa d'Azure té les eines que necessites per tenir èxit.
