Introducció

En aquesta secció, explorarem com crear i configurar Màquines Virtuals (VMs) a Microsoft Azure. Aquest tema és essencial per a qualsevol persona que vulgui aprofitar la computació al núvol per a una infraestructura escalable i flexible.

Prerequisits

Abans de començar, assegura't de tenir:

  • Una subscripció activa a Azure.
  • Coneixements bàsics dels conceptes de computació al núvol.
  • Familiaritat amb el portal d'Azure.

Creació d'una Màquina Virtual

Guia Pas a Pas

  1. Navega al Portal d'Azure:

    • Ves a Azure Portal.
    • Inicia sessió amb el teu compte d'Azure.

  2. Crea una Nova Màquina Virtual:

    • Fes clic a "Crear un recurs" al menú de l'esquerra.
    • Selecciona "Computació" i després "Màquina Virtual."

  3. Configura la Configuració Bàsica:

    • Subscripció: Tria la teva subscripció.
    • Grup de Recursos: Crea un nou grup de recursos o selecciona'n un d'existent.
    • Nom de la Màquina Virtual: Introdueix un nom únic per a la teva VM.
    • Regió: Selecciona la regió on vols desplegar la teva VM.
    • Opcions de Disponibilitat: Tria conjunt de disponibilitat o zona de disponibilitat si cal.
    • Imatge: Selecciona la imatge del sistema operatiu (per exemple, Windows Server, Ubuntu).
    • Mida: Tria la mida de la VM segons les teves necessitats.

  4. Compte d'Administrador:

    • Nom d'Usuari: Introdueix un nom d'usuari per al compte d'administrador.
    • Contrasenya: Introdueix una contrasenya forta o utilitza una clau pública SSH per a VMs Linux.

  5. Regles de Ports Entrants:

    • Selecciona els ports que vols obrir (per exemple, RDP per a Windows, SSH per a Linux).

  6. Revisar + Crear:

    • Revisa la teva configuració i fes clic a "Crear."

Exemple

# Exemple de CLI d'Azure per crear una VM
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Configuració d'una Màquina Virtual

Accedir a la VM

  • VM de Windows: Utilitza el Protocol d'Escriptori Remot (RDP) per connectar-te.
  • VM de Linux: Utilitza Secure Shell (SSH) per connectar-te.

Instal·lació de Programari

  • VM de Windows:

    • Utilitza la connexió RDP per instal·lar programari manualment.
    • Utilitza scripts de PowerShell per a l'automatització.

  • VM de Linux:

    • Utilitza SSH per connectar-te i instal·lar programari utilitzant gestors de paquets com apt o yum.

Exemple

# Exemple d'instal·lació d'Apache en una VM Ubuntu
sudo apt update
sudo apt install apache2 -y

Configuració Avançada

Xarxes

  • Xarxa Virtual (VNet): Configura VNets per gestionar el trànsit de xarxa.
  • Grups de Seguretat de Xarxa (NSG): Defineix regles de seguretat per controlar el trànsit entrant i sortint.

Emmagatzematge

  • Discos Gestionats: Utilitza discos gestionats per a millor rendiment i fiabilitat.
  • Discos de Dades: Adjunta discos de dades addicionals per a necessitats d'emmagatzematge.

Escalat

  • Conjunts d'Escalat: Utilitza conjunts d'escalat de VM per escalar automàticament el nombre de VMs segons la demanda.

Exemple

# Exemple de CLI d'Azure per afegir un disc de dades a una VM
az vm disk attach \
  --resource-group myResourceGroup \
  --vm-name myVM \
  --name myDataDisk \
  --size-gb 128

Monitorització i Manteniment

Monitorització

  • Azure Monitor: Utilitza Azure Monitor per fer un seguiment del rendiment i la salut de les teves VMs.
  • Log Analytics: Recull i analitza registres per a la resolució de problemes.

Manteniment

  • Actualitzacions: Actualitza regularment les teves VMs per garantir la seguretat i el rendiment.
  • Còpies de Seguretat: Utilitza Azure Backup per crear còpies de seguretat regulars de les teves VMs.

Conclusió

Crear i configurar VMs a Azure és una habilitat fonamental per aprofitar la infraestructura al núvol. Seguint els passos descrits en aquesta guia, pots configurar VMs adaptades a les teves necessitats específiques, configurar-les per a un rendiment òptim i assegurar-te que siguin segures i ben mantingudes. A mesura que avancis, explora configuracions i integracions més complexes per utilitzar plenament les capacitats d'Azure.
