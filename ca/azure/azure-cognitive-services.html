Els Serveis Cognitius d'Azure són un conjunt d'APIs, SDKs i serveis disponibles per als desenvolupadors per fer que les seves aplicacions siguin més intel·ligents, atractives i descobribles. Aquests serveis permeten als desenvolupadors afegir fàcilment funcions cognitives a les seves aplicacions sense tenir habilitats directes en IA o ciència de dades.

Introducció als Serveis Cognitius d'Azure

Els Serveis Cognitius d'Azure proporcionen una gamma d'APIs que permeten als desenvolupadors integrar capacitats d'IA a les seves aplicacions. Aquests serveis es classifiquen en cinc àrees principals:

  • Visió
  • Veu
  • Llenguatge
  • Decisió
  • Cerca

Serveis de Visió

Els serveis de visió permeten a les aplicacions processar imatges i vídeos per extreure informació, categoritzar contingut i més.

Serveis de Visió Clau

  • Visió per Computador: Extreu informació d'imatges, com ara objectes, cares i text.
  • API de Cares: Detecta i reconeix cares humanes en imatges.
  • Reconeguidor de Formularis: Extreu text, parells clau-valor i taules de documents.
  • Visió Personalitzada: Et permet construir classificadors d'imatges personalitzats.

Exemple: Utilitzant l'API de Visió per Computador

import requests

subscription_key = "your_subscription_key"
endpoint = "https://your_endpoint.cognitiveservices.azure.com/"

analyze_url = endpoint + "vision/v3.1/analyze"

# Estableix la URL de la imatge
image_url = "https://example.com/image.jpg"

headers = {'Ocp-Apim-Subscription-Key': subscription_key}
params = {'visualFeatures': 'Categories,Description,Color'}
data = {'url': image_url}

response = requests.post(analyze_url, headers=headers, params=params, json=data)
response.raise_for_status()

analysis = response.json()
print(analysis)

En aquest exemple, utilitzem l'API de Visió per Computador per analitzar una imatge i extreure informació com categories, descripcions i colors.

Serveis de Veu

Els serveis de veu permeten a les aplicacions convertir llenguatge parlat en text, text en llenguatge parlat i més.

Serveis de Veu Clau

  • Veu a Text: Converteix llenguatge parlat en text.
  • Text a Veu: Converteix text en llenguatge parlat.
  • Traducció de Veu: Traduïx llenguatge parlat en temps real.
  • Reconeixement de Parladors: Identifica i verifica parladors.

Exemple: Utilitzant l'API de Veu a Text

import azure.cognitiveservices.speech as speechsdk

speech_key = "your_speech_key"
service_region = "your_service_region"

speech_config = speechsdk.SpeechConfig(subscription=speech_key, region=service_region)
audio_config = speechsdk.AudioConfig(filename="path_to_audio_file.wav")

speech_recognizer = speechsdk.SpeechRecognizer(speech_config=speech_config, audio_config=audio_config)

result = speech_recognizer.recognize_once()
print(result.text)

En aquest exemple, utilitzem l'API de Veu a Text per convertir un fitxer d'àudio en text.

Serveis de Llenguatge

Els serveis de llenguatge proporcionen capacitats per entendre i processar el llenguatge natural.

Serveis de Llenguatge Clau

  • Anàlisi de Text: Extreu informació com sentiment, frases clau, entitats nomenades i llenguatge del text.
  • Traductor: Traduïx text d'un idioma a un altre.
  • Comprensió del Llenguatge (LUIS): Construeix comprensió del llenguatge natural en aplicacions, bots i dispositius IoT.
  • Creador de QnA: Crea una capa de preguntes i respostes conversacionals sobre les teves dades.

Exemple: Utilitzant l'API d'Anàlisi de Text

from azure.ai.textanalytics import TextAnalyticsClient
from azure.core.credentials import AzureKeyCredential

endpoint = "https://your_endpoint.cognitiveservices.azure.com/"
key = "your_key"

text_analytics_client = TextAnalyticsClient(endpoint=endpoint, credential=AzureKeyCredential(key))

documents = ["I had a wonderful experience! The rooms were wonderful and the staff was helpful."]

response = text_analytics_client.analyze_sentiment(documents=documents)[0]
print("Sentiment:", response.sentiment)

En aquest exemple, utilitzem l'API d'Anàlisi de Text per analitzar el sentiment d'un text donat.

Serveis de Decisió

Els serveis de decisió proporcionen APIs per prendre decisions informades basades en dades.

Serveis de Decisió Clau

  • Personalitzador: Proporciona recomanacions personalitzades.
  • Moderador de Contingut: Detecta contingut potencialment ofensiu o no desitjat.
  • Detector d'Anomalies: Detecta anomalies en dades de sèries temporals.

Exemple: Utilitzant l'API de Detector d'Anomalies

import requests

subscription_key = "your_subscription_key"
endpoint = "https://your_endpoint.cognitiveservices.azure.com/"

anomaly_url = endpoint + "anomalydetector/v1.0/timeseries/entire/detect"

headers = {'Ocp-Apim-Subscription-Key': subscription_key}
data = {
    "series": [
        {"timestamp": "2023-01-01T00:00:00Z", "value": 10},
        {"timestamp": "2023-01-02T00:00:00Z", "value": 20},
        # Afegeix més punts de dades
    ],
    "granularity": "daily"
}

response = requests.post(anomaly_url, headers=headers, json=data)
response.raise_for_status()

anomalies = response.json()
print(anomalies)

En aquest exemple, utilitzem l'API de Detector d'Anomalies per detectar anomalies en un conjunt de dades de sèries temporals.

Serveis de Cerca

Els serveis de cerca proporcionen APIs per afegir capacitats de cerca a les teves aplicacions.

Serveis de Cerca Clau

  • Cerca Cognitiva d'Azure: Proporciona una solució de cerca com a servei.
  • APIs de Cerca de Bing: Et permeten integrar capacitats de cerca de Bing a les teves aplicacions.

Exemple: Utilitzant la Cerca Cognitiva d'Azure

from azure.search.documents import SearchClient
from azure.core.credentials import AzureKeyCredential

endpoint = "https://your_search_service.search.windows.net"
index_name = "your_index_name"
key = "your_key"

search_client = SearchClient(endpoint=endpoint, index_name=index_name, credential=AzureKeyCredential(key))

results = search_client.search("search_term")
for result in results:
    print(result)

En aquest exemple, utilitzem la Cerca Cognitiva d'Azure per cercar un terme dins d'un índex especificat.

Conclusió

Els Serveis Cognitius d'Azure proporcionen un conjunt complet d'APIs que permeten als desenvolupadors afegir funcions intel·ligents a les seves aplicacions. Aprofitant aquests serveis, els desenvolupadors poden millorar les seves aplicacions amb capacitats com l'anàlisi d'imatges i vídeos, el reconeixement de veu, la comprensió del llenguatge natural, la presa de decisions i la cerca. Tant si ets un desenvolupador principiant com avançat, els Serveis Cognitius d'Azure ofereixen eines i recursos per ajudar-te a construir aplicacions més intel·ligents.
