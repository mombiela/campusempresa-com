Introducció a Azure Monitor

Azure Monitor és una solució integral per recollir, analitzar i actuar sobre la telemetria dels teus entorns en el núvol i locals. T'ajuda a maximitzar la disponibilitat i el rendiment de les teves aplicacions i serveis.

Conceptes Clau

  • Dades de Telemetria: Dades recollides de diverses fonts, incloent aplicacions, sistemes operatius i recursos.
  • Mètriques: Dades numèriques que descriuen el rendiment d'un sistema en un moment determinat.
  • Logs: Registres d'esdeveniments que ocorren dins del teu sistema.
  • Alertes: Notificacions activades per condicions específiques en les teves mètriques o logs.
  • Quadres de comandament: Vistes personalitzables que mostren mètriques i logs.

Configurant Azure Monitor

Per començar a utilitzar Azure Monitor, necessites configurar el teu entorn i configurar els components necessaris.

Passos per Configurar Azure Monitor

  1. Crear un Compte d'Azure: Si no tens un compte d'Azure, crea'n un a Azure Portal.
  2. Navegar a Azure Monitor: Al Portal d'Azure, cerca "Azure Monitor" a la barra de cerca.
  3. Habilitar el Monitoratge per a Recursos: Selecciona els recursos que vols monitorar i habilita el monitoratge.

Exemple: Habilitant el Monitoratge per a una Màquina Virtual

# Script de PowerShell per habilitar el monitoratge per a una VM
$vmName = "myVM"
$resourceGroup = "myResourceGroup"
Set-AzVMMonitoringExtension -ResourceGroupName $resourceGroup -VMName $vmName

Recollint i Analitzant Dades

Azure Monitor recull dades de diverses fonts i proporciona eines per analitzar-les.

Fonts de Dades

  • Application Insights: Monitoritza aplicacions en viu i proporciona informació sobre el seu rendiment i ús.
  • Azure Log Analytics: Recull i analitza dades de logs de diverses fonts.
  • Azure Metrics: Recull i analitza dades de mètriques de recursos d'Azure.

Analitzant Dades amb Log Analytics

Log Analytics utilitza un llenguatge de consultes anomenat Kusto Query Language (KQL) per analitzar dades de logs.

Exemple: Consulta Bàsica de KQL

// Obtenir els últims 10 registres de la taula 'Event'
Event
| take 10

Creant Alertes i Notificacions

Les alertes et notifiquen quan es compleixen certes condicions en les teves mètriques o logs.

Tipus d'Alertes

  • Alertes de Mètriques: Activades per valors de mètriques.
  • Alertes de Logs: Activades per dades de logs.
  • Alertes de Registre d'Activitat: Activades per registres d'activitat d'Azure.

Exemple: Creant una Alerta de Mètrica

# Script de PowerShell per crear una alerta de mètrica
$actionGroup = New-AzActionGroup -Name "myActionGroup" -ResourceGroupName "myResourceGroup" -ShortName "AG"
Add-AzMetricAlertRuleV2 -Name "myMetricAlert" -ResourceGroupName "myResourceGroup" -WindowSize (New-TimeSpan -Minutes 5) -Frequency (New-TimeSpan -Minutes 1) -TargetResourceId "/subscriptions/{subscription-id}/resourceGroups/myResourceGroup/providers/Microsoft.Compute/virtualMachines/myVM" -Condition "avg Percentage CPU > 80" -ActionGroupId $actionGroup.Id

Visualitzant Dades amb Quadres de Comandament

Els quadres de comandament proporcionen una vista personalitzable de les teves mètriques i logs.

Creant un Quadre de Comandament

  1. Navegar a Quadres de Comandament: Al Portal d'Azure, ves a la secció "Dashboards".
  2. Crear un Nou Quadre de Comandament: Fes clic a "New Dashboard" i personalitza'l afegint rajoles.
  3. Afegir Mètriques i Logs: Afegeix rajoles per a mètriques i logs per visualitzar les teves dades.

Exemple: Afegint una Rajola de Mètrica

# Script de PowerShell per afegir una rajola de mètrica a un quadre de comandament
$dashboard = New-AzPortalDashboard -Name "myDashboard" -ResourceGroupName "myResourceGroup"
Add-AzPortalDashboardTile -Dashboard $dashboard -TileType "metric" -MetricName "Percentage CPU" -ResourceId "/subscriptions/{subscription-id}/resourceGroups/myResourceGroup/providers/Microsoft.Compute/virtualMachines/myVM"

Conclusió

Azure Monitor és una eina poderosa per monitorar i gestionar el rendiment i la disponibilitat de les teves aplicacions i serveis. Entenent els seus conceptes clau, configurant-lo correctament, recollint i analitzant dades, creant alertes i visualitzant dades amb quadres de comandament, pots assegurar-te que els teus sistemes funcionin de manera suau i eficient.
