Introducció

En aquest tema, explorarem com crear i desplegar aplicacions web utilitzant Microsoft Azure. Aquesta guia està estructurada per portar-te des dels conceptes bàsics fins als avançats, assegurant que tinguis una comprensió completa del procés.

Començant amb Azure

Abans d'endinsar-nos en la creació d'aplicacions web, és essencial configurar el teu entorn Azure.

Configurant un Compte d'Azure

  • Registrar-se: Ves al lloc web d'Azure i registra't per obtenir un compte gratuït.
  • Portal d'Azure: Familiaritza't amb el Portal d'Azure, on gestionaràs els teus recursos.

Instal·lant Azure CLI

Azure CLI és una eina de línia de comandes que et permet gestionar recursos d'Azure.

  • Instal·lació: Segueix la documentació oficial per instal·lar Azure CLI al teu sistema.
  • Inici de sessió: Utilitza la comanda az login per autenticar el teu compte.

Creant una Aplicació Web

Azure proporciona diverses maneres de crear aplicacions web. Ens centrarem en utilitzar el Portal d'Azure i Azure CLI.

Utilitzant el Portal d'Azure

  1. Navega al Portal d'Azure.

  2. Crear un Recurs: Fes clic a "Crear un recurs" i selecciona "Aplicació Web".

  3. Configura els Paràmetres Bàsics:

    • Subscripció: Tria la teva subscripció.
    • Grup de Recursos: Crea un nou grup de recursos o utilitza un d'existent.
    • Nom: Proporciona un nom únic per a la teva aplicació web.
    • Publicar: Tria "Codi" o "Contenidor Docker".
    • Stack de Runtime: Selecciona el stack de runtime (per exemple, .NET, Node.js, Python).
    • Regió: Tria una regió propera als teus usuaris.

  4. Revisar i Crear: Revisa els teus paràmetres i fes clic a "Crear".

Utilitzant Azure CLI

# Crear un grup de recursos
az group create --name myResourceGroup --location eastus

# Crear un pla de servei d'aplicacions
az appservice plan create --name myAppServicePlan --resource-group myResourceGroup --sku FREE

# Crear una aplicació web
az webapp create --resource-group myResourceGroup --plan myAppServicePlan --name myUniqueAppName --runtime "NODE|14-lts"
  • Explicació:
    • az group create: Crea un grup de recursos.
    • az appservice plan create: Crea un pla de servei d'aplicacions.
    • az webapp create: Crea l'aplicació web amb el runtime especificat.

Desplegant la teva Aplicació Web

Un cop la teva aplicació web està creada, necessites desplegar el teu codi.

Utilitzant Git

  1. Inicialitzar el Repositori Git:

    git init
git add .
git commit -m "Commit inicial"

  2. Configurar el Desplegament:

    az webapp deployment source config-local-git --name myUniqueAppName --resource-group myResourceGroup

  3. Empènyer el Codi:

    git remote add azure <deployment-url>
git push azure master

Utilitzant Azure DevOps

  1. Crear un Projecte: A Azure DevOps, crea un nou projecte.
  2. Configurar una Pipeline: Utilitza el servei Azure Pipelines per configurar una pipeline CI/CD.
  3. Configurar el Desplegament: Enllaça el teu projecte Azure DevOps amb la teva aplicació web Azure per a un desplegament continu.

Monitorització i Escalat

Azure proporciona eines per monitoritzar i escalar la teva aplicació web.

Monitorització

  • Application Insights: Integra Application Insights per monitoritzar el rendiment i diagnosticar problemes.
  • Registres: Utilitza Azure Monitor per veure registres i configurar alertes.

Escalat

  • Escalar Cap Amunt: Augmenta els recursos del teu pla de servei d'aplicacions.
  • Escalar Cap a Fora: Augmenta el nombre d'instàncies que executen la teva aplicació web.

Temes Avançats

Dominis Personalitzats i SSL

  • Domini Personalitzat: Mapa un domini personalitzat a la teva aplicació web.
  • Certificats SSL: Assegura la teva aplicació web amb certificats SSL.

Utilitzant Docker

  • Suport Docker: Desplega la teva aplicació web utilitzant contenidors Docker.
  • Azure Container Registry: Emmagatzema i gestiona imatges Docker.

Serverless amb Azure Functions

  • Azure Functions: Crea funcions serverless per gestionar tasques específiques.
  • Integració: Integra Azure Functions amb la teva aplicació web per a funcionalitats addicionals.

Conclusió

Crear i desplegar aplicacions web a Azure és una manera poderosa d'aprofitar la infraestructura al núvol. Seguint aquesta guia, ara hauries de ser capaç de:

  • Configurar el teu entorn Azure.
  • Crear i desplegar aplicacions web utilitzant el Portal d'Azure i Azure CLI.
  • Monitoritzar i escalar les teves aplicacions web.
  • Explorar temes avançats com dominis personalitzats, SSL, Docker i funcions serverless.

Amb aquestes habilitats, pots gestionar i desplegar aplicacions web a Azure de manera eficient, assegurant alta disponibilitat i rendiment per als teus usuaris.
