Azure Pipelines és un servei al núvol que pots utilitzar per construir i provar automàticament el teu projecte de codi i fer-lo disponible per a altres usuaris. Combina la integració contínua (CI) i el lliurament continu (CD) per provar i construir constantment el teu codi i enviar-lo a qualsevol destinació.

Introducció a Azure Pipelines

  • Integració Contínua (CI): Construir i provar automàticament el codi cada vegada que un membre de l'equip fa canvis.
  • Lliurament Continu (CD): Desplegar automàticament l'aplicació a entorns de producció o de proves després de construccions exitoses.

Configuració d'Azure Pipelines

Prerequisits

  • Un compte d'Azure DevOps.
  • Un projecte a Azure DevOps.
  • Repositori de codi font (per exemple, GitHub, Azure Repos).

Creació del teu Primer Pipeline

  1. Navegar a Pipelines: Ves al teu projecte d'Azure DevOps i selecciona "Pipelines" des del menú de l'esquerra.
  2. Crear Pipeline: Fes clic a "New Pipeline".
  3. Seleccionar Repositori: Tria la font on està allotjat el teu codi (per exemple, GitHub, Azure Repos).
  4. Configurar Pipeline: Utilitza l'editor YAML per definir el teu pipeline o tria una plantilla inicial.

Exemple de Pipeline YAML

# .azure-pipelines.yml
trigger:
- main

pool:
  vmImage: 'ubuntu-latest'

steps:
- task: UseNode@2
  inputs:
    version: '14.x'
- script: |
    npm install
    npm run build
    npm test
  displayName: 'Install dependencies, build and test'

Explicació:

  • trigger: Especifica la branca que desencadena el pipeline.
  • pool: Defineix la imatge de VM a utilitzar per l'agent de construcció.
  • steps: Llista les tasques a executar en el pipeline.

Conceptes Clau a Azure Pipelines

  • Agents: Màquines que executen les tasques en el teu pipeline.
  • Jobs: Una col·lecció de passos que s'executen seqüencialment en el mateix agent.
  • Tasks: Els blocs de construcció més petits d'un pipeline, representant una acció única (per exemple, executar un script, instal·lar dependències).

Funcionalitats Avançades del Pipeline

Pipelines Multi-Etapa

Els pipelines multi-etapa et permeten definir múltiples etapes en el teu pipeline, com ara construir, provar i desplegar.

stages:
- stage: Build
  jobs:
  - job: BuildJob
    pool:
      vmImage: 'ubuntu-latest'
    steps:
    - script: echo Building...
      displayName: 'Build Stage'

- stage: Test
  dependsOn: Build
  jobs:
  - job: TestJob
    pool:
      vmImage: 'ubuntu-latest'
    steps:
    - script: echo Testing...
      displayName: 'Test Stage'

- stage: Deploy
  dependsOn: Test
  jobs:
  - job: DeployJob
    pool:
      vmImage: 'ubuntu-latest'
    steps:
    - script: echo Deploying...
      displayName: 'Deploy Stage'

Estratègies de Desplegament

  • Desplegament Progressiu: Reemplaçar gradualment les instàncies de la versió anterior amb la nova versió.
  • Desplegament Blau-Verd: Desplegar la nova versió al costat de la versió antiga, després canviar el trànsit a la nova versió.
  • Llançament Canari: Desplegar gradualment la nova versió a un petit subconjunt d'usuaris abans del desplegament complet.

Integració amb Altres Serveis

  • GitHub Actions: Utilitza GitHub Actions per desencadenar Azure Pipelines.
  • Azure Key Vault: Gestiona de manera segura secrets i claus utilitzades en el teu pipeline.

Millors Pràctiques

  • Utilitza Pipelines YAML: Els pipelines YAML estan controlats per versió i són més fàcils de gestionar.
  • Modularitza els teus Pipelines: Divideix pipelines complexos en plantilles reutilitzables.
  • Assegura els teus Pipelines: Utilitza connexions de servei i Azure Key Vault per gestionar secrets de manera segura.
  • Monitora i Optimitza: Monitora contínuament el rendiment del pipeline i optimitza per a l'eficiència.

Conclusió

Azure Pipelines és una eina poderosa per automatitzar els processos de construcció, prova i desplegament de les teves aplicacions. Comprenent els seus conceptes bàsics i aprofitant les funcionalitats avançades, pots crear pipelines CI/CD robustos que millorin el teu flux de treball de desenvolupament i assegurin un lliurament de programari d'alta qualitat. Tant si acabes de començar com si busques optimitzar els teus pipelines existents, Azure Pipelines ofereix la flexibilitat i escalabilitat per satisfer les teves necessitats.
