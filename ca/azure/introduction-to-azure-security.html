Introducció a la Seguretat d'Azure
Centre de Seguretat d'Azure ►

Visió General

La seguretat d'Azure és un aspecte crític de la gestió i operació d'aplicacions i serveis a la plataforma de núvol de Microsoft Azure. Aquesta secció t'introduirà als conceptes fonamentals, eines i millors pràctiques per assegurar el teu entorn d'Azure.

Conceptes Clau

  • Model de Responsabilitat Compartida: Comprendre la divisió de responsabilitats de seguretat entre Microsoft i el client.
  • Gestió d'Identitat i Accés (IAM): Gestionar qui té accés als teus recursos d'Azure.
  • Seguretat de Xarxa: Protegir la teva infraestructura de xarxa.
  • Protecció de Dades: Assegurar que les dades estan segures tant en repòs com en trànsit.
  • Monitoratge i Gestió de Seguretat: Monitoritzar i gestionar contínuament les amenaces de seguretat.

Model de Responsabilitat Compartida

Al núvol, la seguretat és una responsabilitat compartida entre el proveïdor de núvol (Microsoft) i el client. La divisió de responsabilitats varia segons el tipus de servei de núvol:

| Model de Servei | Responsabilitat de Microsoft | Responsabilitat del Client | |-----------------|------------------------------|----------------------------| | IaaS | Seguretat física, infraestructura de l'host, controls de xarxa | Dades, aplicacions, gestió d'identitat i accés, configuració del SO | | PaaS | Seguretat física, infraestructura de l'host, controls de xarxa, gestió del SO | Dades, aplicacions, gestió d'identitat i accés | | SaaS | Seguretat física, infraestructura de l'host, controls de xarxa, gestió del SO, gestió d'aplicacions | Dades, gestió d'identitat i accés |

Gestió d'Identitat i Accés (IAM)

IAM és crucial per assegurar que només els usuaris autoritzats poden accedir als teus recursos d'Azure.

  • Azure Active Directory (AAD): Una solució completa de gestió d'identitat i accés al núvol.
  • Control d'Accés Basat en Rols (RBAC): Assignar rols als usuaris per controlar el seu accés als recursos.

Exemple: Crear un Usuari i Assignar un Rol

# Crear un nou usuari a Azure Active Directory
New-AzureADUser -DisplayName "John Doe" -UserPrincipalName "johndoe@yourdomain.com" -PasswordProfile (New-Object -TypeName Microsoft.Open.AzureAD.Model.PasswordProfile -Property @{Password = "P@ssw0rd"})

# Assignar el rol de 'Lector' a l'usuari per a un grup de recursos específic
New-AzRoleAssignment -ObjectId (Get-AzureADUser -UserPrincipalName "johndoe@yourdomain.com").ObjectId -RoleDefinitionName "Reader" -ResourceGroupName "YourResourceGroup"

Seguretat de Xarxa

Assegurar la teva infraestructura de xarxa és essencial per protegir-se contra accessos no autoritzats i atacs.

  • Grups de Seguretat de Xarxa (NSGs): Controlar el trànsit entrant i sortint als recursos d'Azure.
  • Azure Firewall: Un servei de seguretat de xarxa gestionat i basat en el núvol.
  • Connexió de Xarxes Virtuals (VNet Peering): Connectar VNets per permetre la comunicació entre elles.

Exemple: Crear un Grup de Seguretat de Xarxa i Afegir una Regla

# Crear un nou Grup de Seguretat de Xarxa
$nsg = New-AzNetworkSecurityGroup -ResourceGroupName "YourResourceGroup" -Location "EastUS" -Name "MyNSG"

# Afegir una regla de seguretat per permetre el trànsit HTTP
$rule = Add-AzNetworkSecurityRuleConfig -Name "AllowHTTP" -NetworkSecurityGroup $nsg -Protocol "Tcp" -Direction "Inbound" -Priority 100 -SourceAddressPrefix "*" -SourcePortRange "*" -DestinationAddressPrefix "*" -DestinationPortRange 80 -Access "Allow"

# Aplicar la regla
Set-AzNetworkSecurityGroup -NetworkSecurityGroup $nsg

Protecció de Dades

Protegir les teves dades és fonamental, ja sigui en repòs o en trànsit.

  • Xifrat: Utilitza el Xifrat del Servei d'Emmagatzematge d'Azure (SSE) per a dades en repòs i el Xifrat de Discos d'Azure per a màquines virtuals.
  • Azure Key Vault: Emmagatzema i gestiona de manera segura informació sensible com claus, secrets i certificats.

Exemple: Xifrar un Disc de VM

# Activar el xifrat de disc en una VM
Set-AzVMDiskEncryptionExtension -ResourceGroupName "YourResourceGroup" -VMName "YourVM" -DiskEncryptionKeyVaultUrl "https://yourkeyvault.vault.azure.net/" -KeyEncryptionKeyUrl "https://yourkeyvault.vault.azure.net/keys/yourkey"

Monitoratge i Gestió de Seguretat

El monitoratge i la gestió contínua de les amenaces de seguretat són essencials per mantenir un entorn segur.

  • Azure Security Center: Proporciona gestió de seguretat unificada i protecció avançada contra amenaces.
  • Azure Sentinel: Una solució escalable i nativa del núvol per a la gestió d'informació d'esdeveniments de seguretat (SIEM) i resposta automatitzada d'orquestració de seguretat (SOAR).

Exemple: Activar Azure Security Center

# Activar el nivell Estàndard d'Azure Security Center
Set-AzSecurityPricing -Name "default" -PricingTier "Standard"

Conclusió

La seguretat d'Azure abasta una àmplia gamma de pràctiques i eines dissenyades per protegir el teu entorn de núvol. Comprenent i implementant el model de responsabilitat compartida, IAM, seguretat de xarxa, protecció de dades i monitoratge continu, pots millorar significativament la postura de seguretat dels teus recursos d'Azure. Recorda, la seguretat és un procés continu que requereix atenció i millora constants.
