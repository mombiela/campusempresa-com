Azure DevOps és un conjunt d'eines de desenvolupament proporcionades per Microsoft per donar suport a tot el cicle de vida del desenvolupament de programari. Inclou serveis per planificar, desenvolupar, lliurar i operar aplicacions. Aquest tema us introduirà als conceptes i serveis bàsics d'Azure DevOps, proporcionant una base per a una exploració i domini més profunds.

Conceptes Clau d'Azure DevOps

  • Azure Boards: Eines de planificació àgil (taulers Kanban, backlogs, panells de l'equip) per fer el seguiment del treball.
  • Azure Repos: Repositoris Git privats al núvol il·limitats per al vostre projecte.
  • Azure Pipelines: Compila, prova i desplega amb CI/CD que funciona amb qualsevol llenguatge, plataforma i núvol.
  • Azure Test Plans: Solució completa per a proves planificades i exploratòries.
  • Azure Artifacts: Crea, allotja i comparteix paquets amb el teu equip, i afegeix artefactes a les teves pipelines de CI/CD.

Configuració d'Azure DevOps

Creació d'un Compte d'Azure DevOps

  1. Aneu al lloc web d'Azure DevOps.
  2. Feu clic a "Start free" o "Sign in" si ja teniu un compte de Microsoft.
  3. Seguiu les instruccions per crear la vostra organització d'Azure DevOps.

Creació d'un Nou Projecte

  1. Un cop iniciada la sessió, feu clic a "New Project".
  2. Introduïu un nom i una descripció del projecte.
  3. Trieu la visibilitat (públic o privat).
  4. Feu clic a "Create".

Azure Boards

Azure Boards proporciona un conjunt ric de capacitats, incloent-hi suport natiu per a Scrum i Kanban, panells personalitzables i informes integrats.

Creació i Gestió d'Elements de Treball

  • Elements de Treball: Unitats bàsiques de treball, com ara històries d'usuari, tasques, errors.
  • Taulers: Visualitzeu i gestioneu el treball amb taulers Kanban.
  • Backlogs: Prioritzeu el treball i planifiqueu sprints.

Exemple de creació d'un element de treball:

1. Navegueu a Boards > Work Items.
2. Feu clic a "New Work Item" i seleccioneu el tipus (per exemple, User Story).
3. Ompliu els detalls i deseu.

Azure Repos

Azure Repos proporciona repositoris Git o Team Foundation Version Control (TFVC) per al control de versions del vostre codi.

Creació d'un Repositori

  1. Navegueu a Repos > Files.
  2. Feu clic a "New Repository".
  3. Trieu Git o TFVC.
  4. Inicialitzeu amb un README si ho desitgeu i feu clic a "Create".

Comandes Bàsiques de Git

# Clonar un repositori
git clone https://dev.azure.com/yourorganization/yourproject/_git/yourrepo

# Afegir un fitxer a l'staging
git add filename

# Fer un commit dels canvis
git commit -m "Your commit message"

# Empènyer els canvis a Azure Repos
git push origin main

Azure Pipelines

Azure Pipelines us permet compilar, provar i desplegar codi automàticament.

Creació d'una Pipeline

  1. Navegueu a Pipelines > Pipelines.
  2. Feu clic a "New Pipeline".
  3. Seleccioneu la font del vostre repositori (per exemple, GitHub, Azure Repos).
  4. Configureu la vostra pipeline utilitzant YAML o l'editor clàssic.

Exemple de configuració de pipeline YAML:

trigger:
- main

pool:
  vmImage: 'ubuntu-latest'

steps:
- script: echo Hello, world!
  displayName: 'Run a one-line script'

Azure Test Plans

Azure Test Plans proporciona eines per a proves manuals i exploratòries.

Creació d'un Pla de Proves

  1. Navegueu a Test Plans > Test Plans.
  2. Feu clic a "New Test Plan".
  3. Introduïu un nom i una descripció.
  4. Afegiu casos de prova i configureu suites de prova.

Azure Artifacts

Azure Artifacts us permet crear i compartir feeds de paquets Maven, npm, NuGet i Python de fonts públiques i privades.

Creació d'un Feed

  1. Navegueu a Artifacts > Feeds.
  2. Feu clic a "New Feed".
  3. Introduïu un nom i configureu els permisos.
  4. Feu clic a "Create".

Ús d'Azure Artifacts en una Pipeline

Exemple d'afegir un paquet NuGet a una pipeline:

steps:
- task: UseDotNet@2
  inputs:
    packageType: 'sdk'
    version: '5.x'
    installationPath: $(Agent.ToolsDirectory)/dotnet

- task: NuGetCommand@2
  inputs:
    command: 'restore'
    feedsToUse: 'select'
    vstsFeed: 'your-feed-id'

Conclusió

Azure DevOps és un conjunt complet d'eines dissenyat per donar suport a tot el cicle de vida del desenvolupament de programari. Comprenent i utilitzant Azure Boards, Repos, Pipelines, Test Plans i Artifacts, podeu optimitzar el vostre procés de desenvolupament, millorar la col·laboració i lliurar programari d'alta qualitat de manera més eficient. Aquesta introducció proporciona una base per a una exploració i domini més profunds d'Azure DevOps.
