Introducció a Azure Key Vault

Azure Key Vault és un servei al núvol que proporciona un magatzem segur per a secrets, claus i certificats. Ajuda a protegir claus criptogràfiques i secrets utilitzats per aplicacions i serveis al núvol.

Conceptes Clau

  • Secrets: Emmagatzema de manera segura i controla estrictament l'accés a tokens, contrasenyes, certificats, claus API i altres secrets.
  • Claus: Gestiona i utilitza claus criptogràfiques per a xifratge, desxifratge i signatura.
  • Certificats: Proporciona, gestiona i desplega certificats SSL/TLS públics i privats.

Configuració d'Azure Key Vault

Per començar a utilitzar Azure Key Vault, necessites crear un Key Vault a la teva subscripció d'Azure.

Passos per Crear un Key Vault

  1. Navega al Portal d'Azure: Ves al Portal d'Azure.
  2. Crea un Recurs: Fes clic a "Create a resource" i cerca "Key Vault".
  3. Configura el Key Vault:
    • Nom: Proporciona un nom únic per al teu Key Vault.
    • Regió: Selecciona la regió on vols desplegar el teu Key Vault.
    • Nivell de Preus: Tria el nivell de preus (Standard o Premium).
    • Grup de Recursos: Selecciona un grup de recursos existent o crea'n un de nou.
  4. Revisa i Crea: Revisa la teva configuració i fes clic a "Create".

Exemple: Creant un Key Vault utilitzant Azure CLI

# Variables
RESOURCE_GROUP="myResourceGroup"
KEY_VAULT_NAME="myKeyVault"
LOCATION="eastus"

# Crear Grup de Recursos
az group create --name $RESOURCE_GROUP --location $LOCATION

# Crear Key Vault
az keyvault create --name $KEY_VAULT_NAME --resource-group $RESOURCE_GROUP --location $LOCATION

Gestió de Secrets a Azure Key Vault

Els secrets són valors que vols mantenir confidencials, com claus API, contrasenyes o cadenes de connexió.

Afegint un Secret

Pots afegir secrets al teu Key Vault utilitzant el Portal d'Azure, Azure CLI o Azure PowerShell.

Exemple: Afegint un Secret utilitzant Azure CLI

# Variables
SECRET_NAME="MySecret"
SECRET_VALUE="MySecretValue"

# Afegir Secret
az keyvault secret set --vault-name $KEY_VAULT_NAME --name $SECRET_NAME --value $SECRET_VALUE

Recuperant un Secret

Per recuperar un secret, pots utilitzar Azure CLI o Azure SDKs.

Exemple: Recuperant un Secret utilitzant Azure CLI

# Recuperar Secret
az keyvault secret show --vault-name $KEY_VAULT_NAME --name $SECRET_NAME

Gestió de Claus a Azure Key Vault

Les claus s'utilitzen per a operacions de xifratge, desxifratge i signatura.

Creant una Clau

Pots crear claus al teu Key Vault utilitzant el Portal d'Azure, Azure CLI o Azure PowerShell.

Exemple: Creant una Clau utilitzant Azure CLI

# Variables
KEY_NAME="MyKey"

# Crear Clau
az keyvault key create --vault-name $KEY_VAULT_NAME --name $KEY_NAME --protection software

Utilitzant una Clau per a Xifratge

Pots utilitzar les claus emmagatzemades a Azure Key Vault per a xifrar dades.

Exemple: Xifrant Dades utilitzant Azure CLI

# Variables
PLAINTEXT="Hello, World!"

# Xifrar Dades
az keyvault key encrypt --vault-name $KEY_VAULT_NAME --name $KEY_NAME --algorithm RSA-OAEP --value $PLAINTEXT

Gestió de Certificats a Azure Key Vault

Els certificats s'utilitzen per assegurar comunicacions i establir identitat.

Important un Certificat

Pots importar certificats al teu Key Vault utilitzant el Portal d'Azure, Azure CLI o Azure PowerShell.

Exemple: Important un Certificat utilitzant Azure CLI

# Variables
CERT_NAME="MyCert"
CERT_FILE="/path/to/certificate.pfx"
CERT_PASSWORD="password"

# Importar Certificat
az keyvault certificate import --vault-name $KEY_VAULT_NAME --name $CERT_NAME --file $CERT_FILE --password $CERT_PASSWORD

Polítiques d'Accés i Seguretat

Les polítiques d'accés a Azure Key Vault controlen l'accés als secrets, claus i certificats emmagatzemats al vault.

Configurant Polítiques d'Accés

Pots configurar polítiques d'accés per concedir o denegar accés a usuaris o aplicacions específiques.

Exemple: Configurant Polítiques d'Accés utilitzant Azure CLI

# Variables
USER_OBJECT_ID="user-object-id"

# Establir Política d'Accés
az keyvault set-policy --name $KEY_VAULT_NAME --object-id $USER_OBJECT_ID --secret-permissions get list --key-permissions get list --certificate-permissions get list

Integrant Azure Key Vault amb Aplicacions

Pots integrar Azure Key Vault amb les teves aplicacions per gestionar de manera segura secrets, claus i certificats.

Exemple: Utilitzant Azure Key Vault en una Aplicació .NET

using Azure.Identity;
using Azure.Security.KeyVault.Secrets;

var client = new SecretClient(new Uri("https://<YourKeyVaultName>.vault.azure.net/"), new DefaultAzureCredential());
KeyVaultSecret secret = client.GetSecret("MySecret");

Console.WriteLine($"Secret Value: {secret.Value}");

Conclusió

Azure Key Vault és una eina poderosa per gestionar secrets, claus i certificats de manera segura i escalable. Comprenent com crear i gestionar Key Vaults, afegir i recuperar secrets, crear i utilitzar claus, i importar certificats, pots millorar la seguretat de les teves aplicacions i serveis. Integrar Azure Key Vault amb les teves aplicacions assegura que la informació sensible es guarda de manera segura i només és accessible per entitats autoritzades.
