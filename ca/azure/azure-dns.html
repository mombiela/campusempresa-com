Azure DNS és un servei d'allotjament per a dominis DNS, que proporciona resolució de noms utilitzant la infraestructura de Microsoft Azure. Allotjant els teus dominis a Azure, pots gestionar els teus registres DNS utilitzant les mateixes credencials, API, eines i facturació que els teus altres serveis d'Azure.

Introducció al DNS

  • DNS (Domain Name System): Un sistema jeràrquic per traduir noms de domini fàcils de recordar (com www.example.com) en adreces IP (com 192.0.2.1).
  • Registres DNS: Entrades a la base de dades DNS que mapegen noms de domini a adreces IP i altres recursos.

Per què utilitzar Azure DNS?

  • Abast global: Aprofita la xarxa global de servidors DNS de Microsoft per a una resolució de noms ràpida i fiable.
  • Gestió integrada: Gestiona els registres DNS utilitzant el mateix portal d'Azure, API i eines que altres serveis d'Azure.
  • Seguretat: Beneficia't de les característiques de seguretat d'Azure, incloent-hi el control d'accés basat en rols (RBAC) i els registres d'activitat.

Configuració d'Azure DNS

Creació d'una zona DNS

Una zona DNS s'utilitza per allotjar els registres DNS per a un domini particular.

  1. Navega al portal d'Azure: Ves al portal d'Azure a https://portal.azure.com.
  2. Crea un recurs: Fes clic a "Crear un recurs" i cerca "zona DNS".
  3. Configura la zona DNS: Omple els detalls necessaris com la subscripció, el grup de recursos i el nom de la zona DNS (per exemple, example.com).
  4. Revisa i crea: Revisa la teva configuració i fes clic a "Crear".

Afegir registres DNS

Un cop la zona DNS està creada, pots afegir registres DNS.

  1. Navega a la zona DNS: Ves a la zona DNS que has creat.
  2. Afegeix un conjunt de registres: Fes clic a "+ Conjunt de registres" per afegir un nou registre DNS.
  3. Configura el registre: Omple els detalls com el nom, tipus (A, CNAME, MX, etc.), TTL (Time to Live) i valor.
  4. Desa: Fes clic a "OK" per desar el registre.

Exemple: Afegir un registre A

Nom: www
Tipus: A
TTL: 3600
Valor: 192.0.2.1

Aquest registre mapega www.example.com a l'adreça IP 192.0.2.1.

Gestió del DNS amb Azure CLI

Azure CLI proporciona una interfície de línia de comandes per gestionar recursos d'Azure, incloent-hi DNS.

Instal·lació d'Azure CLI

Segueix les instruccions a Instal·lar Azure CLI per instal·lar l'Azure CLI al teu sistema.

Creació d'una zona DNS amb Azure CLI

az network dns zone create --resource-group MyResourceGroup --name example.com

Afegir un registre DNS amb Azure CLI

az network dns record-set a add-record --resource-group MyResourceGroup --zone-name example.com --record-set-name www --ipv4-address 192.0.2.1

Característiques avançades del DNS

Registres d'àlies

Els registres d'àlies permeten mapar un nom DNS a un recurs d'Azure, com una adreça IP pública o un perfil de Traffic Manager.

  • Cas d'ús: Simplifica la gestió del DNS quan l'adreça IP del recurs canvia.

Zones DNS privades

Les zones DNS privades proporcionen resolució DNS per a xarxes virtuals dins d'Azure.

  • Cas d'ús: Útil per a escenaris on necessites resolució DNS per a recursos interns que no estan exposats a internet.

Traffic Manager DNS

Azure Traffic Manager és un equilibrador de càrrega basat en DNS que et permet distribuir el tràfic de manera òptima als serveis a través de regions globals d'Azure.

  • Cas d'ús: Millora la disponibilitat i la capacitat de resposta de les teves aplicacions dirigint el tràfic dels usuaris al punt final més proper o més disponible.

Resum

Azure DNS és un servei potent i flexible per gestionar registres DNS al núvol. S'integra perfectament amb altres serveis d'Azure, proporcionant una experiència de gestió unificada. Entenent els conceptes bàsics del DNS, configurant zones DNS i aprofitant característiques avançades com els registres d'àlies i les zones DNS privades, pots gestionar eficaçment els registres DNS del teu domini utilitzant Azure DNS.

Punts clau

  • Azure DNS proporciona un servei DNS distribuït globalment i d'alta disponibilitat.
  • Pots gestionar zones i registres DNS a través del portal d'Azure, Azure CLI i API.
  • Característiques avançades com els registres d'àlies i les zones DNS privades ofereixen flexibilitat i control addicional.

Dominant Azure DNS, pots assegurar una resolució de noms fiable i eficient per a les teves aplicacions i serveis.
