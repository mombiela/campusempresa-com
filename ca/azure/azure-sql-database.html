◄ Introducció a les Bases de Dades d'Azure
Azure Cosmos DB ►

Azure SQL Database és un servei de base de dades relacional totalment gestionat proporcionat per Microsoft Azure. Ofereix alta disponibilitat, escalabilitat i seguretat per a les teves necessitats de dades. Aquest tema et guiarà des dels conceptes bàsics fins als avançats d'Azure SQL Database.

Introducció a Azure SQL Database

  • Què és Azure SQL Database?

    • Un servei de base de dades relacional basat en el núvol.
    • Totalment gestionat per Microsoft.
    • Suporta característiques de SQL Server.

  • Beneficis Clau:

    • Alta disponibilitat i recuperació davant desastres.
    • Escalabilitat i ajust de rendiment.
    • Seguretat i compliment.
    • Còpies de seguretat i actualitzacions automàtiques.

Començant amb Azure SQL Database

Crear una Azure SQL Database

  1. Inicia sessió al Portal d'Azure:

    • Navega a Azure Portal.

  2. Crea una Nova SQL Database:

    • Ves a "Crear un recurs" > "Bases de dades" > "SQL Database".
    • Omple els detalls requerits com el nom de la base de dades, servidor i nivell de preus.

  3. Configura el Servidor:

    • Si no tens un servidor existent, crea'n un de nou especificant el nom del servidor, inici de sessió d'administrador i contrasenya.

  4. Selecciona el Nivell de Preus:

    • Tria un nivell de preus basat en les teves necessitats de rendiment i emmagatzematge.

  5. Revisa i Crea:

    • Revisa la teva configuració i fes clic a "Crear".

Connectar-se a Azure SQL Database

  • Utilitzant SQL Server Management Studio (SSMS):

    Server: <your_server_name>.database.windows.net
Database: <your_database_name>
Authentication: SQL Server Authentication
Login: <your_admin_login>
Password: <your_password>

  • Exemple de Cadena de Connexió:

    Server=tcp:<your_server_name>.database.windows.net,1433;Initial Catalog=<your_database_name>;Persist Security Info=False;User ID=<your_admin_login>;Password=<your_password>;MultipleActiveResultSets=False;Encrypt=True;TrustServerCertificate=False;Connection Timeout=30;

Gestionant Azure SQL Database

Escalar la Base de Dades

  • Escalat Vertical:

    • Augmenta o disminueix la mida de computació (DTUs o vCores).
    • Canvia el nivell de preus.

  • Escalat Horitzontal:

    • Utilitza particionament o grups elàstics per distribuir la càrrega.

Còpia de Seguretat i Restauració

  • Còpies de Seguretat Automàtiques:

    • Azure SQL Database fa còpies de seguretat automàticament.
    • El període de retenció depèn del nivell de servei.

  • Restauració Puntual:

    • Restaura la teva base de dades a qualsevol punt dins del període de retenció.

  • Retenció a Llarg Termini:

    • Configura la retenció de còpies de seguretat a llarg termini per al compliment.

Característiques de Seguretat

  • Regles de Tallafocs:

    • Configura regles de tallafocs a nivell de servidor i base de dades per controlar l'accés.

  • Protecció Avançada contra Amenaces:

    • Detecta i respon a possibles amenaces.

  • Xifrat Transparent de Dades (TDE):

    • Xifra les dades en repòs.

  • Sempre Xifrat:

    • Protegeix les dades sensibles xifrant-les al costat del client.

Temes Avançats

Ajust de Rendiment

  • Informes de Rendiment de Consultes:

    • Analitza el rendiment de les consultes i identifica colls d'ampolla.

  • Gestió d'Índexs:

    • Crea i gestiona índexs per optimitzar el rendiment de les consultes.

  • Ajust Automàtic:

    • Activa l'ajust automàtic perquè Azure optimitzi la teva base de dades.

Alta Disponibilitat i Recuperació davant Desastres

  • Geo-Replicació:

    • Configura la geo-replicació activa per a alta disponibilitat.

  • Grups de Failover:

    • Configura grups de failover per a failover automàtic.

Monitoratge i Alertes

  • Azure Monitor:

    • Utilitza Azure Monitor per seguir les mètriques de rendiment i configurar alertes.

  • SQL Analytics:

    • Utilitza SQL Analytics per obtenir una visió més profunda del rendiment de la teva base de dades.

Conclusió

Azure SQL Database proporciona una solució robusta, escalable i segura per a les teves necessitats de bases de dades relacionals. Aprofitant les seves característiques, pots assegurar alta disponibilitat, optimitzar el rendiment i mantenir el compliment de seguretat. Tant si estàs començant com si busques implementar configuracions avançades, Azure SQL Database ofereix les eines i capacitats per satisfer els teus requisits.
