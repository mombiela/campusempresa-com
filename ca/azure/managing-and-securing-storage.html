Introducció

En aquest mòdul, explorarem com gestionar i assegurar l'emmagatzematge a Microsoft Azure. Això inclou entendre els diferents tipus d'emmagatzematge disponibles, com gestionar comptes d'emmagatzematge i les millors pràctiques per assegurar les teves dades.

Tipus d'Emmagatzematge d'Azure

Azure proporciona diversos tipus de solucions d'emmagatzematge per satisfer diferents necessitats:

  • Emmagatzematge de Blobs: Utilitzat per emmagatzemar grans quantitats de dades no estructurades com text o dades binàries.
  • Emmagatzematge de Fitxers: Proporciona comparticions de fitxers totalment gestionades al núvol que són accessibles mitjançant el protocol SMB.
  • Emmagatzematge de Cues: Utilitzat per emmagatzemar grans quantitats de missatges que poden ser accessibles des de qualsevol lloc mitjançant trucades autenticades.
  • Emmagatzematge de Taules: Un magatzem NoSQL per a l'emmagatzematge sense esquema de dades estructurades.

Emmagatzematge de Blobs

L'emmagatzematge de Blobs és ideal per a:

  • Servir imatges o documents directament a un navegador.
  • Emmagatzemar fitxers per a accés distribuït.
  • Transmetre vídeo i àudio.
  • Escriure en fitxers de registre.

Exemple: Pujar un Blob

from azure.storage.blob import BlobServiceClient, BlobClient, ContainerClient

# Inicialitzar una connexió al servei de Blobs
blob_service_client = BlobServiceClient.from_connection_string("your_connection_string")

# Crear un contenidor
container_name = "mycontainer"
container_client = blob_service_client.create_container(container_name)

# Pujar un fitxer al contenidor
blob_client = blob_service_client.get_blob_client(container=container_name, blob="myfile.txt")
with open("localfile.txt", "rb") as data:
    blob_client.upload_blob(data)

Emmagatzematge de Fitxers

L'emmagatzematge de Fitxers és ideal per a:

  • Migrar comparticions de fitxers locals al núvol.
  • Emmagatzemar configuracions compartides d'aplicacions.
  • Emmagatzemar perfils d'usuari.

Exemple: Muntar una Compartició de Fitxers d'Azure a Windows

# Muntar la compartició de fitxers
net use Z: \\<storage_account_name>.file.core.windows.net\<file_share_name> /u:<storage_account_name> <storage_account_key>

Emmagatzematge de Cues

L'emmagatzematge de Cues és ideal per a:

  • Desacoblar components d'aplicacions.
  • Construir aplicacions flexibles i escalables.

Exemple: Afegir un Missatge a una Cua

from azure.storage.queue import QueueServiceClient

# Inicialitzar una connexió al servei de Cues
queue_service_client = QueueServiceClient.from_connection_string("your_connection_string")

# Crear una cua
queue_name = "myqueue"
queue_client = queue_service_client.create_queue(queue_name)

# Afegir un missatge a la cua
queue_client.send_message("Hello, World!")

Emmagatzematge de Taules

L'emmagatzematge de Taules és ideal per a:

  • Emmagatzemar dades estructurades que no requereixen unions complexes, claus foranes o procediments emmagatzemats.
  • Emmagatzemar conjunts de dades que requereixen alta disponibilitat i escalabilitat.

Exemple: Inserir una Entitat en una Taula

from azure.data.tables import TableServiceClient, TableEntity

# Inicialitzar una connexió al servei de Taules
table_service_client = TableServiceClient.from_connection_string("your_connection_string")

# Crear una taula
table_name = "mytable"
table_client = table_service_client.create_table(table_name)

# Inserir una entitat
entity = TableEntity()
entity.PartitionKey = "partition1"
entity.RowKey = "row1"
entity["Name"] = "John Doe"
entity["Age"] = 30
table_client.create_entity(entity=entity)

Gestió de Comptes d'Emmagatzematge

Els comptes d'emmagatzematge proporcionen un espai de noms únic per a les teves dades d'Azure Storage. Pots gestionar comptes d'emmagatzematge utilitzant el portal d'Azure, Azure CLI o Azure PowerShell.

Crear un Compte d'Emmagatzematge utilitzant Azure CLI

az storage account create \
  --name mystorageaccount \
  --resource-group myResourceGroup \
  --location eastus \
  --sku Standard_LRS

Gestió de Claus d'Accés

Les claus d'accés s'utilitzen per autenticar l'accés al teu compte d'emmagatzematge. És important gestionar aquestes claus de manera segura.

Rotació de Claus d'Accés

# Llistar les claus d'accés
az storage account keys list --resource-group myResourceGroup --account-name mystorageaccount

# Regenerar una clau d'accés
az storage account keys renew --resource-group myResourceGroup --account-name mystorageaccount --key primary

Assegurar l'Emmagatzematge

Assegurar el teu emmagatzematge és crucial per protegir les teves dades contra accessos no autoritzats i violacions.

Millors Pràctiques per Assegurar l'Emmagatzematge

  • Utilitzar Signatures d'Accés Compartit (SAS): SAS proporciona accés limitat als teus recursos d'emmagatzematge sense exposar les teves claus de compte.
  • Habilitar el Xifratge d'Emmagatzematge d'Azure: Les dades es xifren i desxifren de manera transparent utilitzant xifratge AES de 256 bits.
  • Utilitzar Punts de Servei de Xarxa Virtual: Restringir l'accés al teu compte d'emmagatzematge a xarxes virtuals específiques.
  • Habilitar la Protecció Avançada contra Amenaces: Detecta intents inusuals i potencialment nocius d'accedir o explotar el teu compte d'emmagatzematge.

Exemple: Generar un Testimoni SAS

from azure.storage.blob import generate_blob_sas, BlobSasPermissions
from datetime import datetime, timedelta

sas_token = generate_blob_sas(
    account_name="your_account_name",
    container_name="your_container_name",
    blob_name="your_blob_name",
    account_key="your_account_key",
    permission=BlobSasPermissions(read=True),
    expiry=datetime.utcnow() + timedelta(hours=1)
)
print(sas_token)
