Configuració del clúster HDInsight

  • Prerequisits
  • Guia pas a pas per crear un clúster HDInsight
  • Opcions de configuració
# Exemple: Creació d'un clúster HDInsight utilitzant Azure CLI
az hdinsight create \
    --name myHDInsightCluster \
    --resource-group myResourceGroup \
    --cluster-type spark \
    --location eastus \
    --version 4.0 \
    --component-version spark=2.4 \
    --headnode-size Standard_D3_V2 \
    --workernode-size Standard_D3_V2 \
    --workernode-count 3

Treballant amb dades a HDInsight

  • Carregar dades a HDInsight
  • Processament de dades amb Apache Spark
  • Exemple: Programa de recompte de paraules en PySpark
# Exemple: Recompte de paraules en PySpark
from pyspark import SparkContext

sc = SparkContext("local", "WordCount")
text_file = sc.textFile("hdfs:///path/to/input.txt")
counts = text_file.flatMap(lambda line: line.split(" ")) \
                  .map(lambda word: (word, 1)) \
                  .reduceByKey(lambda a, b: a + b)
counts.saveAsTextFile("hdfs:///path/to/output")

Configuració d'Azure Data Lake Storage

  • Prerequisits
  • Guia pas a pas per crear un compte de Data Lake Storage
  • Opcions de configuració
# Exemple: Creació d'un compte d'Azure Data Lake Storage utilitzant Azure CLI
az storage account create \
    --name mydatalakestorage \
    --resource-group myResourceGroup \
    --location eastus \
    --sku Standard_LRS \
    --kind StorageV2 \
    --hierarchical-namespace true

Treballant amb dades a Data Lake Storage

  • Carregar i gestionar dades

    --file-system myfilesystem \
    --path /path/to/uploadedfile.txt \
    --source /local/path/to/file.txt
