Azure Resource Manager (ARM)
Desplegar una Aplicació Multi-Nivell ►

Azure Resource Manager (ARM) és el servei de desplegament i gestió per a Azure. Proporciona una capa de gestió consistent que et permet crear, actualitzar i eliminar recursos en el teu compte d'Azure.

Introducció a Azure Resource Manager

  • Definició: ARM és un marc de gestió que et permet desplegar, gestionar i monitoritzar recursos d'Azure com un grup.
  • Beneficis:
    • Consistència: Assegura que els recursos es despleguin en un estat consistent.
    • Seguretat: Proporciona control d'accés basat en rols (RBAC) per assegurar els recursos.
    • Etiquetatge: Et permet etiquetar recursos per a una gestió i facturació més fàcil.
    • Plantilles: Suporta Infraestructura com a Codi (IaC) a través de plantilles ARM.

Conceptes Clau

  • Grup de Recursos: Un contenidor que manté recursos relacionats per a una solució d'Azure.
  • Plantilla ARM: Un fitxer JSON que defineix la infraestructura i configuració per al teu projecte.
  • Desplegament: El procés de desplegar recursos utilitzant plantilles ARM.
  • Proveïdor de Recursos: Un servei que subministra recursos d'Azure.

Treballant amb Grups de Recursos

  • Creant un Grup de Recursos:

    az group create --name myResourceGroup --location eastus

    Aquesta comanda crea un nou grup de recursos anomenat myResourceGroup a la regió eastus.

  • Llistant Grups de Recursos:

    az group list --output table

    Aquesta comanda llista tots els grups de recursos en la teva subscripció en format de taula.

  • Eliminant un Grup de Recursos:

    az group delete --name myResourceGroup --yes --no-wait

    Aquesta comanda elimina el grup de recursos especificat i tots els recursos continguts dins d'ell.

Plantilles ARM

  • Estructura d'una Plantilla ARM:

    {
  "$schema": "https://schema.management.azure.com/schemas/2019-04-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "resources": [
    {
      "type": "Microsoft.Storage/storageAccounts",
      "apiVersion": "2019-04-01",
      "name": "mystorageaccount",
      "location": "eastus",
      "sku": {
        "name": "Standard_LRS"
      },
      "kind": "StorageV2",
      "properties": {}
    }
  ]
}
    • $schema: Defineix la ubicació del fitxer d'esquema JSON que descriu la versió del llenguatge de la plantilla.
    • contentVersion: Especifica la versió de la plantilla.
    • resources: Una matriu que conté els recursos a desplegar o actualitzar.

  • Desplegant una Plantilla ARM:

    az deployment group create --resource-group myResourceGroup --template-file template.json

    Aquesta comanda desplega els recursos definits a template.json al grup de recursos especificat.

Característiques Avançades de les Plantilles ARM

  • Paràmetres: Et permeten passar valors durant el desplegament per personalitzar les propietats dels recursos.

    {
  "parameters": {
    "storageAccountType": {
      "type": "string",
      "defaultValue": "Standard_LRS",
      "allowedValues": [
        "Standard_LRS",
        "Standard_GRS",
        "Standard_ZRS",
        "Premium_LRS"
      ],
      "metadata": {
        "description": "Tipus de Compte d'Emmagatzematge"
      }
    }
  }
}

  • Variables: Emmagatzemen valors que reutilitzes en la teva plantilla.

    {
  "variables": {
    "storageAccountName": "[concat('storage', uniqueString(resourceGroup().id))]"
  }
}

  • Sortides: Retornen valors dels recursos desplegats.

    {
  "outputs": {
    "storageAccountName": {
      "type": "string",
      "value": "[variables('storageAccountName')]"
    }
  }
}

Millors Pràctiques

  • Plantilles Modulars: Divideix plantilles grans en plantilles més petites i reutilitzables.
  • Control de Versions: Emmagatzema les teves plantilles en un sistema de control de versions com Git.
  • Proves: Prova les teves plantilles en un entorn de proves abans de desplegar-les a producció.
  • Documentació: Documenta les teves plantilles per fer-les més fàcils d'entendre i mantenir.

Conclusió

Azure Resource Manager (ARM) és una eina poderosa per gestionar recursos d'Azure. Entenent i utilitzant plantilles ARM, grups de recursos i les diverses característiques d'ARM, pots aconseguir desplegaments consistents, repetibles i automatitzats. Això no només millora l'eficiència de la gestió dels teus recursos d'Azure, sinó que també assegura que la teva infraestructura sigui escalable i mantenible.
