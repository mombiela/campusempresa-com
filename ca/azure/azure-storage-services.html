Serveis d'emmagatzematge d'Azure
Serveis de Xarxa d'Azure ►

Els serveis d'emmagatzematge d'Azure proporcionen solucions d'emmagatzematge al núvol escalables, duradores i altament disponibles. Aquest tema cobrirà els conceptes bàsics fins als avançats dels serveis d'emmagatzematge d'Azure, incloent-hi l'emmagatzematge de Blobs, l'emmagatzematge de Taules, l'emmagatzematge de Cues i l'emmagatzematge d'Arxius.

Introducció als serveis d'emmagatzematge d'Azure

Azure Storage és un servei gestionat per Microsoft que proporciona emmagatzematge al núvol altament disponible, segur, durador, escalable i redundant. Ofereix diversos tipus de serveis d'emmagatzematge per satisfer diferents necessitats.

  • Emmagatzematge de Blobs: Optimitzat per emmagatzemar grans quantitats de dades no estructurades.
  • Emmagatzematge de Taules: Un magatzem NoSQL de clau-valor per al desenvolupament ràpid utilitzant conjunts de dades semiestructurades.
  • Emmagatzematge de Cues: Proporciona missatgeria fiable per al processament de fluxos de treball i la comunicació entre components.
  • Emmagatzematge d'Arxius: Comparticions d'arxius gestionades per a desplegaments al núvol o locals.

Emmagatzematge de Blobs

L'emmagatzematge de Blobs està dissenyat per emmagatzemar grans quantitats de dades no estructurades, com ara dades de text o binàries.

Conceptes clau

  • Contenidors: Els contenidors organitzen un conjunt de blobs, similar als directoris en un sistema d'arxius.
  • Blobs: Arxius individuals emmagatzemats a Azure Storage. Els tipus inclouen Blobs de Blocs, Blobs d'Afegit i Blobs de Pàgines.

Exemple

from azure.storage.blob import BlobServiceClient, BlobClient, ContainerClient

# Crear un BlobServiceClient
blob_service_client = BlobServiceClient.from_connection_string("your_connection_string")

# Crear un contenidor
container_name = "mycontainer"
container_client = blob_service_client.create_container(container_name)

# Pujar un blob
blob_client = blob_service_client.get_blob_client(container=container_name, blob="myblob.txt")
with open("sample.txt", "rb") as data:
    blob_client.upload_blob(data)

Conceptes avançats

  • Nivells de Blob: Nivells Calent, Fred i Arxiu per a emmagatzematge rendible.
  • Gestió del cicle de vida: Automatitzar la classificació i eliminació de blobs.

Emmagatzematge de Taules

L'emmagatzematge de Taules és un magatzem NoSQL que proporciona accés ràpid a grans quantitats de dades.

Conceptes clau

  • Taules: Col·leccions d'entitats.
  • Entitats: Similar a les files en una taula.
  • Propietats: Similar a les columnes en una taula.

Exemple

from azure.data.tables import TableServiceClient, TableClient

# Crear un TableServiceClient
table_service_client = TableServiceClient.from_connection_string("your_connection_string")

# Crear una taula
table_name = "mytable"
table_client = table_service_client.create_table_if_not_exists(table_name)

# Inserir una entitat
entity = {
    'PartitionKey': 'partition1',
    'RowKey': 'row1',
    'Name': 'John Doe',
    'Age': 30
}
table_client.create_entity(entity=entity)

Conceptes avançats

  • Particionament: Distribuir dades a través de múltiples servidors per a l'escalabilitat.
  • Consultes: Utilitzar consultes OData per filtrar dades.

Emmagatzematge de Cues

L'emmagatzematge de Cues proporciona una solució de missatgeria fiable per a la comunicació asíncrona entre components d'aplicacions.

Conceptes clau

  • Cues: Emmagatzemen missatges.
  • Missatges: Contenen dades per ser processades.

Exemple

from azure.storage.queue import QueueServiceClient, QueueClient

# Crear un QueueServiceClient
queue_service_client = QueueServiceClient.from_connection_string("your_connection_string")

# Crear una cua
queue_name = "myqueue"
queue_client = queue_service_client.create_queue(queue_name)

# Afegir un missatge a la cua
queue_client.send_message("Hello, World!")

Conceptes avançats

  • TTL del missatge: Temps de vida per als missatges.
  • Operacions per lots: Processar múltiples missatges en una sola operació.

Emmagatzematge d'Arxius

L'emmagatzematge d'Arxius proporciona comparticions d'arxius completament gestionades al núvol que són accessibles mitjançant el protocol SMB.

Conceptes clau

  • Comparticions d'arxius: Emmagatzematge compartit accessible per múltiples màquines virtuals.
  • Directoris: Organitzen arxius dins d'una compartició.
  • Arxius: Arxius individuals emmagatzemats en una compartició.

Exemple

from azure.storage.fileshare import ShareServiceClient, ShareClient, ShareDirectoryClient

# Crear un ShareServiceClient
share_service_client = ShareServiceClient.from_connection_string("your_connection_string")

# Crear una compartició d'arxius
share_name = "myshare"
share_client = share_service_client.create_share(share_name)

# Pujar un arxiu a la compartició
file_path = "sample.txt"
with open(file_path, "rb") as source_file:
    share_client.upload_file(file_path, source_file)

Conceptes avançats

  • Instantània: Crear versions de només lectura de comparticions d'arxius.
  • Sincronització d'Arxius d'Azure: Sincronitzar servidors d'arxius locals amb comparticions d'arxius d'Azure.

Conclusió

Els serveis d'emmagatzematge d'Azure ofereixen una varietat de solucions per a diferents necessitats d'emmagatzematge, des de dades no estructurades amb l'emmagatzematge de Blobs fins a dades NoSQL estructurades amb l'emmagatzematge de Taules, missatgeria fiable amb l'emmagatzematge de Cues i comparticions d'arxius gestionades amb l'emmagatzematge d'Arxius. Comprendre aquests serveis i les seves característiques avançades pot ajudar-te a dissenyar solucions d'emmagatzematge al núvol escalables i eficients.
