Introducció a les Subrutines

Les subrutines, també conegudes com a funcions o mètodes en altres llenguatges de programació, són blocs de codi reutilitzables que realitzen una tasca específica. En Perl, les subrutines ajuden a organitzar el codi, reduir la redundància i millorar la llegibilitat.

Conceptes Clau

  • Definició: Una subrutina es defineix utilitzant la paraula clau sub seguida del nom de la subrutina i un bloc de codi.
  • Crida a una Subrutina: Les subrutines es criden pel seu nom seguit de parèntesis.
  • Arguments: Les subrutines poden acceptar arguments, que es passen com una llista.
  • Valors de Retorn: Les subrutines poden retornar valors utilitzant la paraula clau return.

Definició i Crida de Subrutines

Per definir una subrutina en Perl, utilitza la paraula clau sub seguida del nom de la subrutina i un bloc de codi.

sub say_hello {
    print "Hola, Món!\n";
}

Per cridar la subrutina, simplement utilitza el seu nom seguit de parèntesis:

say_hello();

Exemple

sub greet {
    my $name = shift;  # Recupera el primer argument
    print "Hola, $name!\n";
}

greet("Alice");  # Sortida: Hola, Alice!

Passar Arguments a Subrutines

Els arguments es passen a les subrutines com una llista. Dins de la subrutina, pots accedir a aquests arguments utilitzant l'array especial @_.

Exemple

sub add {
    my ($a, $b) = @_;  # Desempaqueta els arguments
    return $a + $b;
}

my $sum = add(3, 4);  # $sum ara conté el valor 7
print "Sum: $sum\n";  # Sortida: Sum: 7

Retornar Valors des de Subrutines

Les subrutines poden retornar valors utilitzant la paraula clau return. Si return s'omet, l'última expressió avaluada es retorna per defecte.

Exemple

sub multiply {
    my ($x, $y) = @_;
    return $x * $y;
}

my $product = multiply(5, 6);  # $product ara conté el valor 30
print "Product: $product\n";  # Sortida: Product: 30

Característiques Avançades de les Subrutines

Sensibilitat al Context

Les subrutines de Perl poden comportar-se de manera diferent segons el context en què es criden (context escalar o de llista).

Exemple

sub context_example {
    return wantarray ? (1, 2, 3) : "scalar";
}

my @array = context_example();  # Context de llista
print "@array\n";  # Sortida: 1 2 3

my $scalar = context_example();  # Context escalar
print "$scalar\n";  # Sortida: scalar

Subrutines Anònimes

Les subrutines anònimes són subrutines sense nom, sovint utilitzades com arguments per a altres funcions o per crear closures.

Exemple

my $anon_sub = sub {
    my ($x, $y) = @_;
    return $x - $y;
};

my $result = $anon_sub->(10, 3);  # $result ara conté el valor 7
print "Result: $result\n";  # Sortida: Result: 7

Conclusió

Les subrutines en Perl són eines poderoses per organitzar i reutilitzar codi. Entenent com definir, cridar i passar arguments a subrutines, així com com manejar valors de retorn i sensibilitat al context, pots escriure programes en Perl més modulars i mantenibles. Practica creant i utilitzant subrutines per convertir-te en un expert en el seu ús.
