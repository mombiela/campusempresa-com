Estructures de control en Perl
Bucles ►

Les estructures de control són fonamentals en qualsevol llenguatge de programació, i Perl no és una excepció. Et permeten controlar el flux del teu programa, prenent decisions, iterant a través de dades i gestionant excepcions. Aquest tema cobrirà les estructures de control essencials en Perl, des de nivells bàsics fins a avançats.

Declaracions condicionals

Les declaracions condicionals et permeten executar codi basat en certes condicions.

Declaració if

La declaració if s'utilitza per executar un bloc de codi si una condició especificada és certa.

my $age = 18;

if ($age >= 18) {
    print "You are an adult.\n";
}

Declaració if-else

La declaració if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

my $age = 16;

if ($age >= 18) {
    print "You are an adult.\n";
} else {
    print "You are a minor.\n";
}

Declaració if-elsif-else

La declaració if-elsif-else permet que es comprovin múltiples condicions en seqüència.

my $age = 20;

if ($age < 13) {
    print "You are a child.\n";
} elsif ($age < 18) {
    print "You are a teenager.\n";
} else {
    print "You are an adult.\n";
}

Estructures de bucle

Les estructures de bucle et permeten executar un bloc de codi múltiples vegades.

Bucle while

El bucle while executa un bloc de codi mentre la condició sigui certa.

my $count = 0;

while ($count < 5) {
    print "Count is $count\n";
    $count++;
}

Bucle for

El bucle for s'utilitza per iterar sobre un rang de valors.

for (my $i = 0; $i < 5; $i++) {
    print "Iteration $i\n";
}

Bucle foreach

El bucle foreach s'utilitza per iterar sobre una llista de valors.

my @colors = ('red', 'blue', 'green');

foreach my $color (@colors) {
    print "Color: $color\n";
}

Bucle until

El bucle until és l'oposat del bucle while; executa un bloc de codi mentre la condició sigui falsa.

my $count = 0;

until ($count == 5) {
    print "Count is $count\n";
    $count++;
}

Modificadors de flux de control

Els modificadors de flux de control es poden utilitzar per alterar el comportament dels bucles i les declaracions condicionals.

last

La declaració last s'utilitza per sortir immediatament d'un bucle.

for (my $i = 0; $i < 10; $i++) {
    if ($i == 5) {
        last;
    }
    print "Iteration $i\n";
}

next

La declaració next s'utilitza per saltar la resta de la iteració actual i passar a la següent iteració.

for (my $i = 0; $i < 10; $i++) {
    if ($i % 2 == 0) {
        next;
    }
    print "Odd number: $i\n";
}

redo

La declaració redo reinicia la iteració actual sense avaluar de nou la condició del bucle.

my $count = 0;

while ($count < 5) {
    $count++;
    if ($count == 3) {
        redo;
    }
    print "Count is $count\n";
}

Estructures de control avançades

given-when

L'estructura given-when és similar a la declaració switch-case que es troba en altres llenguatges.

use feature 'switch';

my $fruit = 'apple';

given ($fruit) {
    when ('apple') {
        print "It's an apple.\n";
    }
    when ('banana') {
        print "It's a banana.\n";
    }
    default {
        print "Unknown fruit.\n";
    }
}

Conclusió

Les estructures de control són essencials per dirigir el flux dels teus programes en Perl. Des de declaracions condicionals bàsiques com if i if-else fins a estructures de bucle més complexes com while, for i foreach, entendre aquests constructes et permetrà escriure codi més eficient i llegible. A més, els modificadors de flux de control com last, next i redo proporcionen un control addicional sobre els teus bucles, mentre que estructures avançades com given-when ofereixen maneres més expressives de gestionar múltiples condicions. Practica aquests constructes per convertir-te en un expert en la gestió del flux dels teus programes en Perl.
