Estructures de Dades Complexes en Perl
Perl Orientat a Objectes ►

Introducció

En aquesta secció, explorarem estructures de dades complexes en Perl, com ara arrays d'arrays, hashes d'arrays, arrays de hashes i hashes de hashes. Entendre aquestes estructures és crucial per gestionar dades més sofisticades en els teus programes.

Arrays d'Arrays

Els arrays d'arrays, també coneguts com a arrays multidimensionals, et permeten crear una estructura semblant a una matriu.

Creant un Array d'Arrays

my @matrix = (
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
);

Accedint a Elements

Per accedir a elements en un array d'arrays:

print $matrix[0][1];  # Mostra 2
print $matrix[2][0];  # Mostra 7

Iterant sobre un Array d'Arrays

foreach my $row (@matrix) {
    foreach my $element (@$row) {
        print "$element ";
    }
    print "\n";
}

Hashes d'Arrays

Els hashes d'arrays et permeten associar múltiples valors amb una sola clau.

Creant un Hash d'Arrays

my %hash_of_arrays = (
    fruits => ['apple', 'banana', 'cherry'],
    vegetables => ['carrot', 'pea', 'potato']
);

Accedint a Elements

Per accedir a elements en un hash d'arrays:

print $hash_of_arrays{fruits}[1];  # Mostra 'banana'
print $hash_of_arrays{vegetables}[2];  # Mostra 'potato'

Iterant sobre un Hash d'Arrays

foreach my $key (keys %hash_of_arrays) {
    print "$key: ";
    foreach my $item (@{$hash_of_arrays{$key}}) {
        print "$item ";
    }
    print "\n";
}

Arrays de Hashes

Els arrays de hashes et permeten emmagatzemar una llista de referències de hash, útils per representar registres.

Creant un Array de Hashes

my @array_of_hashes = (
    { name => 'Alice', age => 30 },
    { name => 'Bob', age => 25 },
    { name => 'Charlie', age => 35 }
);

Accedint a Elements

Per accedir a elements en un array de hashes:

print $array_of_hashes[0]{name};  # Mostra 'Alice'
print $array_of_hashes[1]{age};  # Mostra 25

Iterant sobre un Array de Hashes

foreach my $record (@array_of_hashes) {
    print "Name: $record->{name}, Age: $record->{age}\n";
}

Hashes de Hashes

Els hashes de hashes et permeten crear una estructura de hash niada, útil per representar relacions complexes.

Creant un Hash de Hashes

my %hash_of_hashes = (
    person1 => { name => 'Alice', age => 30 },
    person2 => { name => 'Bob', age => 25 },
    person3 => { name => 'Charlie', age => 35 }
);

Accedint a Elements

Per accedir a elements en un hash de hashes:

print $hash_of_hashes{person1}{name};  # Mostra 'Alice'
print $hash_of_hashes{person2}{age};  # Mostra 25

Iterant sobre un Hash de Hashes

foreach my $person (keys %hash_of_hashes) {
    print "Name: $hash_of_hashes{$person}{name}, Age: $hash_of_hashes{$person}{age}\n";
}

Resum

En aquesta secció, hem cobert:

  • Arrays d'arrays per a dades multidimensionals.
  • Hashes d'arrays per associar múltiples valors amb una sola clau.
  • Arrays de hashes per emmagatzemar llistes de registres.
  • Hashes de hashes per a estructures de hash niades.

Aquestes estructures de dades complexes són essencials per gestionar i organitzar dades en Perl, permetent-te escriure codi més eficient i llegible. Practica creant i manipulant aquestes estructures per convertir-te en un expert en la gestió de dades complexes en els teus programes Perl.
