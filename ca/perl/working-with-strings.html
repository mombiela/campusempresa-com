Introducció a les Cadenes

Les cadenes són una part fonamental de qualsevol llenguatge de programació, i Perl no és una excepció. En Perl, les cadenes són seqüències de caràcters que poden incloure lletres, números, símbols i espais en blanc. Entendre com manipular cadenes és crucial per a una programació efectiva en Perl.

Conceptes Clau

  • Literals de Cadena: Text tancat entre cometes.
  • Operadors de Cadena: Operadors utilitzats per manipular cadenes.
  • Funcions de Cadena: Funcions incorporades per a la manipulació de cadenes.

Literals de Cadena

En Perl, les cadenes es poden definir utilitzant cometes simples (') o cometes dobles (").

  • Cadenes amb cometes simples: Conserven el valor literal de cada caràcter dins de les cometes.
  • Cadenes amb cometes dobles: Permeten la interpolació de variables i seqüències de caràcters especials.

Exemples

# Cadena amb cometes simples
my $single_quoted = 'Hola, Món!';

# Cadena amb cometes dobles
my $name = "Alice";
my $double_quoted = "Hola, $name!";

Operadors de Cadena

Perl proporciona diversos operadors per a la manipulació de cadenes.

  • Operador de concatenació (.): Combina dues cadenes.
  • Operador de repetició (x): Repeteix una cadena un nombre especificat de vegades.

Exemples

# Concatenació
my $greeting = "Hola, " . "Món!";  # Resultat: "Hola, Món!"

# Repetició
my $repeat = "abc" x 3;  # Resultat: "abcabcabc"

Funcions de Cadena

Perl ofereix un conjunt ric de funcions incorporades per a la manipulació de cadenes.

Funcions de Cadena Comunes

  • length: Retorna la longitud d'una cadena.
  • index: Troba la posició d'una subcadena.
  • substr: Extreu una subcadena.
  • uc: Converteix una cadena a majúscules.
  • lc: Converteix una cadena a minúscules.
  • chomp: Elimina el caràcter de nova línia del final d'una cadena.

Exemples

# Longitud d'una cadena
my $str = "Hola, Món!";
my $len = length($str);  # Resultat: 13

# Índex d'una subcadena
my $position = index($str, "Món");  # Resultat: 7

# Extracció de subcadena
my $substr = substr($str, 7, 3);  # Resultat: "Món"

# Conversió a majúscules
my $upper = uc($str);  # Resultat: "HOLA, MÓN!"

# Conversió a minúscules
my $lower = lc($str);  # Resultat: "hola, món!"

# Exemple de chomp
my $line = "Hola, Món!\n";
chomp($line);  # Resultat: "Hola, Món!"

Manipulació Avançada de Cadenes

Per a operacions de cadena més complexes, Perl proporciona eines poderoses com les expressions regulars i la funció sprintf.

Expressions Regulars

Les expressions regulars (regex) són patrons utilitzats per coincidir combinacions de caràcters en cadenes. Les capacitats regex de Perl són extenses i molt flexibles.

Exemples

# Coincidència simple amb regex
my $text = "El ràpid guineu marró";
if ($text =~ /ràpid/) {
    print "Coincidència trobada!\n";  # Sortida: Coincidència trobada!
}

# Substitució amb regex
$text =~ s/ràpid/lent/;
print $text;  # Sortida: El lent guineu marró

Funció `sprintf`

La funció sprintf formata cadenes utilitzant espais reservats.

Exemples

# Formatant una cadena
my $number = 42;
my $formatted = sprintf("La resposta és %d", $number);  # Resultat: "La resposta és 42"

Conclusió

Treballar amb cadenes en Perl implica entendre com definir-les, manipular-les i formatar-les utilitzant diversos operadors i funcions. Des de la concatenació i repetició bàsiques fins a expressions regulars avançades i formatació, dominar aquestes tècniques és essencial per a una programació efectiva en Perl. Practica aquests conceptes amb els exemples i exercicis proporcionats per obtenir una base sòlida en la manipulació de cadenes.
