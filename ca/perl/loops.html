Introducció als Bucles

Els bucles són construccions fonamentals en la programació que et permeten executar un bloc de codi repetidament. En Perl, hi ha diversos tipus de bucles, cadascun adequat per a diferents situacions. Entendre com utilitzar els bucles de manera efectiva pot fer que el teu codi sigui més eficient i fàcil de llegir.

Tipus de Bucles en Perl

Bucle `while`

El bucle while continua executant un bloc de codi mentre la condició especificada sigui certa.

Sintaxi

while (condició) {
    # Codi a executar
}

Exemple

my $count = 0;
while ($count < 5) {
    print "Count is $count\n";
    $count++;
}
  • Explicació: Aquest bucle imprimirà el valor de $count de 0 a 4. El bucle incrementa $count en 1 en cada iteració fins que $count ja no sigui menor que 5.

Bucle `until`

El bucle until és l'oposat del bucle while. Continua executant un bloc de codi mentre la condició especificada sigui falsa.

Sintaxi

until (condició) {
    # Codi a executar
}

Exemple

my $count = 0;
until ($count == 5) {
    print "Count is $count\n";
    $count++;
}
  • Explicació: Aquest bucle també imprimirà el valor de $count de 0 a 4. El bucle incrementa $count en 1 en cada iteració fins que $count sigui igual a 5.

Bucle `for`

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi

for (inicialització; condició; increment) {
    # Codi a executar
}

Exemple

for (my $i = 0; $i < 5; $i++) {
    print "Value of i is $i\n";
}
  • Explicació: Aquest bucle imprimirà el valor de $i de 0 a 4. El bucle inicialitza $i a 0, comprova la condició $i < 5, i incrementa $i en 1 en cada iteració.

Bucle `foreach`

El bucle foreach s'utilitza per iterar sobre una llista de valors.

Sintaxi

foreach my $item (@array) {
    # Codi a executar
}

Exemple

my @colors = ('red', 'green', 'blue');
foreach my $color (@colors) {
    print "Color is $color\n";
}
  • Explicació: Aquest bucle imprimirà cada color de l'array @colors. El bucle itera sobre cada element de l'array i l'assigna a $color en cada iteració.

Instruccions de Control de Bucles

`next`

La instrucció next s'utilitza per saltar la resta de la iteració actual i passar a la següent iteració del bucle.

Exemple

for (my $i = 0; $i < 5; $i++) {
    next if $i == 2;
    print "Value of i is $i\n";
}
  • Explicació: Aquest bucle imprimirà el valor de $i de 0 a 4, excepte per al 2. Quan $i és igual a 2, la instrucció next salta la instrucció de print i passa a la següent iteració.

`last`

La instrucció last s'utilitza per sortir del bucle immediatament.

Exemple

for (my $i = 0; $i < 5; $i++) {
    last if $i == 3;
    print "Value of i is $i\n";
}
  • Explicació: Aquest bucle imprimirà el valor de $i de 0 a 2. Quan $i és igual a 3, la instrucció last surt del bucle.

`redo`

La instrucció redo reinicia la iteració actual del bucle sense avaluar novament la condició del bucle.

Exemple

my $i = 0;
while ($i < 5) {
    $i++;
    redo if $i == 3;
    print "Value of i is $i\n";
}
  • Explicació: Aquest bucle imprimirà el valor de $i de 1 a 5, excepte per al 3. Quan $i és igual a 3, la instrucció redo reinicia la iteració actual, efectivament saltant la instrucció de print per a aquesta iteració.

Conclusió

Els bucles són construccions essencials en Perl que et permeten realitzar tasques repetitives de manera eficient. Entendre els diferents tipus de bucles (while, until, for, foreach) i les instruccions de control de bucles (next, last, redo) et permetrà escriure codi més efectiu i concís. Practica utilitzant aquests bucles i instruccions de control per esdevenir proficient en la gestió de tasques repetitives en els teus programes Perl.
