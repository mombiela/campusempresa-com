Introducció a la Programació de Xarxes

La programació de xarxes implica escriure programes que es comuniquen amb altres programes a través d'una xarxa d'ordinadors. En Perl, això es fa típicament utilitzant sockets, que proporcionen una manera d'enviar i rebre dades a través d'una xarxa.

Conceptes Clau

  • Sockets: Punts finals per enviar i rebre dades.
  • Protocols: Regles per a l'intercanvi de dades (per exemple, TCP, UDP).
  • Adreces IP i Ports: Identificadors per a interfícies de xarxa i serveis específics.

Configuració del Teu Entorn

Abans d'endinsar-te en la programació de xarxes, assegura't de tenir Perl instal·lat al teu sistema. Pots descarregar-lo des de Perl.org.

Instal·lació de Mòduls Necessaris

Perl té diversos mòduls per a la programació de xarxes. Els més utilitzats són:

  • IO::Socket
  • Net::Ping
  • LWP::UserAgent

Pots instal·lar aquests mòduls utilitzant CPAN:

cpan IO::Socket Net::Ping LWP::UserAgent

Programació Bàsica de Sockets

Creació d'un Client TCP Simple

Un client TCP es connecta a un servidor i envia/rep dades.

use strict;
use warnings;
use IO::Socket::INET;

# Crear un nou socket
my $socket = IO::Socket::INET->new(
    PeerAddr => '127.0.0.1',
    PeerPort => '5000',
    Proto    => 'tcp'
) or die "No s'ha pogut crear el socket: $!\n";

# Enviar dades al servidor
print $socket "Hola des del client!\n";

# Rebre resposta del servidor
my $response = <$socket>;
print "Rebut del servidor: $response\n";

# Tancar el socket
close($socket);

Creació d'un Servidor TCP Simple

Un servidor TCP escolta connexions entrants i les processa.

use strict;
use warnings;
use IO::Socket::INET;

# Crear un nou socket
my $socket = IO::Socket::INET->new(
    LocalAddr => '127.0.0.1',
    LocalPort => '5000',
    Proto     => 'tcp',
    Listen    => 5,
    Reuse     => 1
) or die "No s'ha pogut crear el socket: $!\n";

print "Servidor esperant connexió del client al port 5000\n";

# Acceptar connexió entrant
while (my $client_socket = $socket->accept()) {
    # Llegir dades del client
    my $data = <$client_socket>;
    print "Rebut del client: $data\n";

    # Enviar resposta al client
    print $client_socket "Hola des del servidor!\n";

    # Tancar el socket del client
    close($client_socket);
}

# Tancar el socket del servidor
close($socket);

Programació Avançada de Sockets

Sockets No Bloquejants

Els sockets no bloquejants permeten que un programa continuï executant-se mentre espera que les operacions de xarxa es completin.

use strict;
use warnings;
use IO::Socket::INET;
use IO::Select;

my $socket = IO::Socket::INET->new(
    LocalAddr => '127.0.0.1',
    LocalPort => '5000',
    Proto     => 'tcp',
    Listen    => 5,
    Reuse     => 1
) or die "No s'ha pogut crear el socket: $!\n";

my $select = IO::Select->new($socket);

while (1) {
    my @ready = $select->can_read(0);
    foreach my $fh (@ready) {
        if ($fh == $socket) {
            my $new_socket = $socket->accept();
            $select->add($new_socket);
        } else {
            my $data = <$fh>;
            if (defined $data) {
                print "Rebut: $data";
                print $fh "Eco: $data";
            } else {
                $select->remove($fh);
                close($fh);
            }
        }
    }
}

Treballant amb UDP

UDP és un protocol sense connexió, que és més ràpid però menys fiable que TCP.

Creació d'un Client UDP Simple

use strict;
use warnings;
use IO::Socket::INET;

my $socket = IO::Socket::INET->new(
    PeerAddr => '127.0.0.1',
    PeerPort => '5000',
    Proto    => 'udp'
) or die "No s'ha pogut crear el socket: $!\n";

$socket->send("Hola des del client UDP!\n");

my $response;
$socket->recv($response, 1024);
print "Rebut del servidor: $response\n";

close($socket);

Creació d'un Servidor UDP Simple

use strict;
use warnings;
use IO::Socket::INET;

my $socket = IO::Socket::INET->new(
    LocalAddr => '127.0.0.1',
    LocalPort => '5000',
    Proto     => 'udp'
) or die "No s'ha pogut crear el socket: $!\n";

while (1) {
    my $data;
    $socket->recv($data, 1024);
    print "Rebut: $data\n";
    $socket->send("Hola des del servidor UDP!\n");
}

close($socket);

Programació HTTP

HTTP és un protocol per obtenir recursos com documents HTML. El mòdul LWP::UserAgent de Perl facilita la interacció amb servidors web.

Realització de Peticions HTTP

use strict;
use warnings;
use LWP::UserAgent;

my $ua = LWP::UserAgent->new;
my $response = $ua->get('http://www.example.com');

if ($response->is_success) {
    print $response->decoded_content;
} else {
    die $response->status_line;
}

Anàlisi de Respostes HTTP

use strict;
use warnings;
use LWP::UserAgent;
use HTTP::Response;

my $ua = LWP::UserAgent->new;
my $response = $ua->get('http://www.example.com');

if ($response->is_success) {
    print "Content-Type: " . $response->header('Content-Type') . "\n";
    print "Content: " . $response->decoded_content;
} else {
    die $response->status_line;
}
