Introducció a Perl

Perl, que significa "Practical Extraction and Report Language" (Llenguatge Pràctic d'Extracció i Informe), és un llenguatge de programació d'alt nivell i de propòsit general conegut per les seves capacitats de processament de text. Va ser desenvolupat originalment per Larry Wall el 1987 com un llenguatge de script per a Unix per facilitar el processament d'informes.

Característiques Clau de Perl

  • Processament de Text: Perl excel·leix en la manipulació de text i sovint s'utilitza per a tasques que impliquen expressions regulars.
  • Multiplataforma: Perl funciona en més de 100 plataformes, des de portàtils fins a mainframes.
  • CPAN: La Xarxa Arxivística Completa de Perl (CPAN) és un gran repositori de programari i mòduls de Perl.
  • Flexibilitat: Perl admet tant la programació procedimental com la orientada a objectes.
  • Comunitat: Perl té una comunitat forta i activa que contribueix al seu desenvolupament i suport.

Conceptes Bàsics de la Sintaxi de Perl

La sintaxi de Perl està dissenyada per ser fàcil d'utilitzar i entendre, especialment per a aquells familiaritzats amb altres llenguatges de programació com C, Python o script de shell.

Exemple de "Hello, World!"

Comencem amb un programa senzill de "Hello, World!" en Perl:

#!/usr/bin/perl
use strict;
use warnings;

print "Hello, World!\n";
  • #!/usr/bin/perl: Aquesta és la línia shebang que indica al sistema que utilitzi l'intèrpret de Perl.
  • use strict; i use warnings;: Aquests pragmes ajuden a detectar errors comuns i a aplicar bones pràctiques de codificació.
  • print "Hello, World!\n";: Aquesta línia imprimeix la cadena "Hello, World!" seguida d'un caràcter de nova línia.

Variables en Perl

Perl té tres tipus principals de variables:

  • Escalars: Representen valors únics (nombres o cadenes). Prefixats amb $.
  • Arrays: Llistes ordenades d'escalars. Prefixats amb @.
  • Hashes: Conjunts desordenats de parelles clau-valor. Prefixats amb %.

Exemple d'Escalar

my $name = "Alice";
my $age = 30;
print "Nom: $name, Edat: $age\n";

Exemple d'Array

my @colors = ("red", "green", "blue");
print "Primer color: $colors[0]\n";  # Els arrays comencen en zero

Exemple de Hash

my %fruit_colors = (
    apple  => "red",
    banana => "yellow",
    grape  => "purple"
);
print "Color de la poma: $fruit_colors{apple}\n";

Estructures de Control

Perl admet estructures de control comunes com if, unless, while, for i foreach.

Exemple d'If-Else

my $number = 10;

if ($number > 5) {
    print "El número és més gran que 5\n";
} else {
    print "El número és 5 o menys\n";
}

Exemple de Bucle

my @numbers = (1, 2, 3, 4, 5);

foreach my $num (@numbers) {
    print "Número: $num\n";
}

Expressions Regulars

Una de les característiques més poderoses de Perl és el seu suport per a expressions regulars, que s'utilitzen per a la coincidència de patrons i la manipulació de text.

Exemple d'Expressió Regular

my $text = "The quick brown fox jumps over the lazy dog";

if ($text =~ /quick/) {
    print "La paraula 'quick' és en el text\n";
}

$text =~ s/dog/cat/;
print "Text modificat: $text\n";
  • =~ /pattern/: Comprova si el patró coincideix amb el text.
  • s/old/new/: Substitueix el patró antic pel nou.

Conclusió

Perl és un llenguatge versàtil i potent, especialment adequat per a tasques de processament de text i scripts. La seva sintaxi és flexible i ofereix un conjunt ric de característiques que el converteixen en una eina valuosa tant per a principiants com per a programadors avançats. Amb una comunitat forta i un ampli suport bibliotecari a través de CPAN, Perl continua sent un llenguatge rellevant i útil en molts dominis.
