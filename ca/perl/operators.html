Introducció als Operadors

Els operadors són components fonamentals en Perl que et permeten realitzar diverses operacions sobre variables i valors. Entendre els operadors és crucial per escriure codi Perl efectiu i eficient.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • Suma (+): Suma dos números.
  • Resta (-): Resta el segon número del primer.
  • Multiplicació (*): Multiplica dos números.
  • Divisió (/): Divideix el primer número pel segon.
  • Mòdul (%): Retorna el residu de la divisió del primer número pel segon.
  • Exponenciació (**): Eleva el primer número a la potència del segon.

Exemple

my $a = 10;
my $b = 3;

print $a + $b;  # Output: 13
print $a - $b;  # Output: 7
print $a * $b;  # Output: 30
print $a / $b;  # Output: 3.33333333333333
print $a % $b;  # Output: 1
print $a ** $b; # Output: 1000

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors.

  • Comparació Numèrica:

    • Igual (==)
    • No igual (!=)
    • Major que (>)
    • Menor que (<)
    • Major o igual que (>=)
    • Menor o igual que (<=)

  • Comparació de Cadenes:

    • Igual (eq)
    • No igual (ne)
    • Major que (gt)
    • Menor que (lt)
    • Major o igual que (ge)
    • Menor o igual que (le)

Exemple

my $num1 = 5;
my $num2 = 10;
my $str1 = "poma";
my $str2 = "plàtan";

print $num1 == $num2;  # Output: (buit, fals)
print $num1 != $num2;  # Output: 1 (cert)
print $str1 eq $str2;  # Output: (buit, fals)
print $str1 lt $str2;  # Output: 1 (cert)

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions.

  • I (&&): Retorna cert si ambdues condicions són certes.
  • O (||): Retorna cert si almenys una condició és certa.
  • No (!): Reverteix l'estat lògic del seu operand.

Exemple

my $x = 5;
my $y = 10;

print ($x < 10 && $y > 5);  # Output: 1 (cert)
print ($x > 10 || $y > 5);  # Output: 1 (cert)
print !($x == $y);          # Output: 1 (cert)

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. També poden realitzar operacions i assignar el resultat a la variable.

  • Assignació Simple (=)
  • Assignació de Suma (+=)
  • Assignació de Resta (-=)
  • Assignació de Multiplicació (*=)
  • Assignació de Divisió (/=)
  • Assignació de Mòdul (%=)
  • Assignació d'Exponenciació (**=)

Exemple

my $a = 5;
$a += 3;  # Equivalent a $a = $a + 3; Output: 8
$a -= 2;  # Equivalent a $a = $a - 2; Output: 6
$a *= 4;  # Equivalent a $a = $a * 4; Output: 24
$a /= 2;  # Equivalent a $a = $a / 2; Output: 12
$a %= 5;  # Equivalent a $a = $a % 5; Output: 2
$a **= 3; # Equivalent a $a = $a ** 3; Output: 8

Operadors de Cadenes

Els operadors de cadenes s'utilitzen per manipular cadenes.

  • Concatenació (.): Uneix dues cadenes.
  • Repetició (x): Repeteix una cadena un nombre especificat de vegades.

Exemple

my $str1 = "Hola";
my $str2 = "Món";

print $str1 . " " . $str2;  # Output: Hola Món
print $str1 x 3;            # Output: HolaHolaHola

Operador Condicional (Operador Ternari)

L'operador condicional és una forma abreujada d'una instrucció if-else.

  • Sintaxi: condició ? valor_cert : valor_fals

Exemple

my $age = 18;
my $status = ($age >= 18) ? "adult" : "menor";

print $status;  # Output: adult

Operador de Rang

L'operador de rang s'utilitza per crear una llista de valors.

  • Sintaxi: ..

Exemple

my @numbers = (1..5);  # Output: (1, 2, 3, 4, 5)
my @letters = ('a'..'e');  # Output: ('a', 'b', 'c', 'd', 'e')

Resum

Els operadors en Perl són eines essencials per realitzar una gran varietat de tasques, des de càlculs aritmètics fins a manipulacions de cadenes i comparacions lògiques. En dominar aquests operadors, pots escriure codi Perl més concís i eficient. Practica utilitzant aquests operadors en diferents contextos per sentir-te més còmode amb la seva sintaxi i funcionalitat.
