El Teu Primer Script en Perl ►

Configurar el teu entorn de desenvolupament és el primer pas per començar a programar en Perl. Aquesta secció et guiarà a través del procés d'instal·lació de Perl, configuració d'un editor de text o Entorn de Desenvolupament Integrat (IDE), i verificació de la teva instal·lació.

Instal·lant Perl

Perl és un llenguatge altament portable i es pot instal·lar en diversos sistemes operatius, incloent Windows, macOS i Linux.

Windows

  1. Descarregar Strawberry Perl:

    • Visita el lloc web de Strawberry Perl.
    • Descarrega l'última versió de Strawberry Perl.

  2. Instal·lar Strawberry Perl:

    • Executa l'instal·lador descarregat.
    • Segueix les indicacions d'instal·lació i accepta la configuració per defecte.

  3. Verificar la Instal·lació:

    • Obre el Símbol del Sistema.
    • Escriu perl -v i prem Enter.
    • Hauries de veure la informació de la versió de Perl.

macOS

  1. Utilitzant Homebrew:

    • Obre Terminal.
    • Instal·la Homebrew si encara no ho has fet: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la Perl utilitzant Homebrew: 
      brew install perl

  2. Verificar la Instal·lació:

    • A Terminal, escriu perl -v i prem Enter.
    • Hauries de veure la informació de la versió de Perl.

Linux

  1. Utilitzant el Gestor de Paquets:

    • Obre Terminal.
    • Per a sistemes basats en Debian (p. ex., Ubuntu): 
      sudo apt-get update
sudo apt-get install perl
    • Per a sistemes basats en Red Hat (p. ex., Fedora): 
      sudo dnf install perl

  2. Verificar la Instal·lació:

    • A Terminal, escriu perl -v i prem Enter.
    • Hauries de veure la informació de la versió de Perl.

Configurant un Editor de Text o IDE

Escollir l'editor de text o IDE adequat pot millorar significativament la teva productivitat. Aquí tens algunes opcions populars:

Editors de Text

  • Notepad++ (Windows):

    • Descarrega'l des del lloc web de Notepad++.
    • Suporta ressaltat de sintaxi per a Perl.

  • Sublime Text (Multiplataforma):

    • Descarrega'l des del lloc web de Sublime Text.
    • Instal·la el paquet de ressaltat de sintaxi per a Perl.

  • Vim (Multiplataforma):

    • Disponible per defecte en la majoria de sistemes basats en Unix.
    • Instal·la'l via gestor de paquets si no està ja instal·lat.

Entorns de Desenvolupament Integrats (IDEs)

  • Komodo IDE:

    • Descarrega'l des del lloc web de Komodo IDE.
    • Dissenyat específicament per a llenguatges dinàmics com Perl.

  • Eclipse amb EPIC Plugin:

    • Descarrega Eclipse des del lloc web de Eclipse.
    • Instal·la el plugin EPIC per a suport de Perl.

Verificant la Teva Configuració

Per assegurar-te que tot està configurat correctament, pots escriure i executar un script simple en Perl.

  1. Crear un Script en Perl:

    • Obre el teu editor de text o IDE.
    • Escriu el següent codi: 
      #!/usr/bin/perl
use strict;
use warnings;
print "Hello, World!\n";

  2. Desar el Script:

    • Desa el fitxer amb una extensió .pl, per exemple, hello_world.pl.

  3. Executar el Script:

    • Obre el Símbol del Sistema o Terminal.
    • Navega al directori on has desat el script.
    • Executa el script escrivint: 
      perl hello_world.pl
    • Hauries de veure la sortida: Hello, World!

Conclusió

Configurar l'entorn de Perl implica instal·lar Perl, escollir un editor de text o IDE adequat, i verificar la instal·lació executant un script simple. Aquesta configuració bàsica assegura que estàs preparat per començar a programar en Perl de manera eficient. En les següents seccions, aprofundirem en els conceptes i pràctiques de programació en Perl.
