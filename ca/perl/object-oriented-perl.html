Introducció a la Programació Orientada a Objectes (POO) en Perl

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Els objectes són instàncies de classes, que poden contenir tant dades (atributs) com funcions (mètodes).

En Perl, la POO s'implementa utilitzant paquets i referències. Aquesta secció cobrirà els conceptes bàsics de la POO en Perl, incloent la creació de classes, objectes i mètodes.

Creació d'una Classe en Perl

En Perl, una classe és simplement un paquet. Un paquet és un espai de noms separat que pot contenir variables i subrutines.

Exemple: Creació d'una Classe Simple

package Animal;

# Constructor
sub new {
    my $class = shift;
    my $self = {
        name => shift,
        sound => shift,
    };
    bless $self, $class;
    return $self;
}

# Mètode per obtenir el nom
sub get_name {
    my $self = shift;
    return $self->{name};
}

# Mètode per obtenir el so
sub get_sound {
    my $self = shift;
    return $self->{sound};
}

1; # Fi del paquet

Explicació

  • Declaració del Paquet: package Animal; declara un nou paquet anomenat Animal.
  • Constructor: sub new és un mètode constructor que inicialitza un nou objecte. bless associa l'objecte amb la classe.
  • Mètodes: get_name i get_sound són mètodes per accedir als atributs de l'objecte.

Creació i Ús d'Objectes

Un cop definida una classe, pots crear objectes (instàncies de la classe) i cridar mètodes sobre ells.

Exemple: Ús de la Classe Animal

use Animal;

# Crear un nou objecte Animal
my $cat = Animal->new("Cat", "Meow");

# Accedir als mètodes de l'objecte
print "Animal: " . $cat->get_name() . "\n"; # Sortida: Animal: Cat
print "Sound: " . $cat->get_sound() . "\n"; # Sortida: Sound: Meow

Explicació

  • Creació d'Objectes: Animal->new("Cat", "Meow") crea un nou objecte Animal.
  • Crides a Mètodes: $cat->get_name() i $cat->get_sound() criden els mètodes respectius sobre l'objecte.

Herència en Perl

L'herència permet que una classe hereti els atributs i mètodes d'una altra classe.

Exemple: Creació d'una Subclasse

package Dog;
use parent 'Animal';

# Constructor
sub new {
    my $class = shift;
    my $self = $class->SUPER::new(shift, "Bark");
    bless $self, $class;
    return $self;
}

1; # Fi del paquet

Explicació

  • Herència: use parent 'Animal'; fa que Dog sigui una subclasse de Animal.
  • Paraula Clau SUPER: $class->SUPER::new crida el constructor de la classe pare.

Ús de la Subclasse

use Dog;

# Crear un nou objecte Dog
my $dog = Dog->new("Dog");

# Accedir als mètodes heretats
print "Animal: " . $dog->get_name() . "\n"; # Sortida: Animal: Dog
print "Sound: " . $dog->get_sound() . "\n"; # Sortida: Sound: Bark

Polimorfisme en Perl

El polimorfisme permet que els mètodes es puguin utilitzar de manera intercanviable entre classes.

Exemple: Polimorfisme

sub make_sound {
    my $animal = shift;
    print $animal->get_name() . " says " . $animal->get_sound() . "\n";
}

# Ús del polimorfisme
make_sound($cat); # Sortida: Cat says Meow
make_sound($dog); # Sortida: Dog says Bark

Explicació

  • Mètode Polimòrfic: make_sound pot acceptar qualsevol objecte que tingui els mètodes get_name i get_sound.

Resum

En aquest tema, hem cobert els conceptes bàsics de la Programació Orientada a Objectes en Perl:

  • Creació de classes utilitzant paquets.
  • Definició i ús de constructors i mètodes.
  • Creació i ús d'objectes.
  • Implementació de l'herència per crear subclasses.
  • Ús del polimorfisme per escriure codi flexible i reutilitzable.

Entenent aquests conceptes, pots dissenyar aplicacions Perl més modulars i mantenibles utilitzant els principis de la POO.
