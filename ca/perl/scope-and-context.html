Àmbit i Context
Funcions Integrades ►

Entendre l'àmbit i el context és crucial per escriure programes Perl efectius i sense errors. Aquest tema cobrirà els conceptes bàsics de l'àmbit de les variables, la diferència entre variables locals i globals, i com el context afecta les operacions en Perl.

Àmbit de les Variables

En Perl, l'àmbit d'una variable determina on es pot accedir o modificar en el teu codi. Hi ha dos tipus principals d'àmbit:

  • Àmbit Global
  • Àmbit Lexical

Àmbit Global

Les variables declarades amb la paraula clau our o sense cap paraula clau tenen àmbit global. Es pot accedir a elles des de qualsevol lloc del programa.

# Variable global
$global_var = "Sóc global";

sub print_global {
    print "$global_var\n";  # Accessible aquí
}

print_global();
print "$global_var\n";  # Accessible aquí també

Àmbit Lexical

Les variables declarades amb la paraula clau my tenen àmbit lexical. Només són accessibles dins del bloc, fitxer o eval on es declaren.

sub print_lexical {
    my $lexical_var = "Sóc lexical";
    print "$lexical_var\n";  # Accessible aquí
}

print_lexical();
# print "$lexical_var\n";  # Error: $lexical_var no és accessible aquí

Variables Locals vs. Lexicals

Perl també té una paraula clau local, que temporalment fa una còpia de seguretat del valor d'una variable global i el restaura després que el bloc hagi acabat.

$global_var = "Sóc global";

sub modify_global {
    local $global_var = "Sóc temporalment local";
    print "$global_var\n";  # Imprimeix "Sóc temporalment local"
}

modify_global();
print "$global_var\n";  # Imprimeix "Sóc global"

Taula Comparativa

| Paraula Clau | Tipus d'Àmbit | Descripció | |--------------|--------------------|-------------------------------------------------| | our | Global | Accessible des de qualsevol lloc del paquet | | my | Lexical | Accessible només dins del bloc/fitxer/eval | | local | Temporalment Local | Temporalment fa una còpia de seguretat i restaura la variable global |

Context en Perl

El context en Perl es refereix al tipus de valor esperat (escalar o llista) que una expressió ha de retornar. Hi ha dos contextos principals:

  • Context Escalar
  • Context de Llista

Context Escalar

En context escalar, s'espera que una expressió retorni un sol valor.

my $scalar = 5 + 10;  # Context escalar: $scalar és 15
my $length = length("Hola");  # Context escalar: $length és 5

Context de Llista

En context de llista, s'espera que una expressió retorni una llista de valors.

my @array = (1, 2, 3, 4);  # Context de llista: @array conté (1, 2, 3, 4)
my @split = split(/,/, "a,b,c");  # Context de llista: @split conté ("a", "b", "c")

Sensibilitat al Context

Algunes funcions es comporten de manera diferent segons el context en què es criden.

my @array = (1, 2, 3, 4);

my $count = @array;  # Context escalar: $count és 4 (nombre d'elements)
my @copy = @array;   # Context de llista: @copy conté (1, 2, 3, 4)

Conclusió

Entendre l'àmbit i el context és essencial per escriure codi Perl clar i mantenible. Coneixent la diferència entre l'àmbit global i lexical, i com el context afecta les operacions, pots evitar errors comuns i escriure programes més eficients. Practica aquests conceptes amb els exemples i exercicis proporcionats per consolidar la teva comprensió.
