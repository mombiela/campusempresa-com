Introducció als Arrays

Els arrays són llistes ordenades d'escalars en Perl. Són una de les estructures de dades fonamentals i s'utilitzen per emmagatzemar múltiples valors en una sola variable.

  • Els arrays es denoten amb el símbol @.
  • Cada element en un array es pot accedir utilitzant un índex, començant des de 0.

Declarar i Inicialitzar Arrays

Per declarar i inicialitzar un array, pots utilitzar la següent sintaxi:

# Declarar un array
my @fruits;

# Inicialitzar un array
my @fruits = ('apple', 'banana', 'cherry');

Accedir als Elements d'un Array

Pots accedir als elements individuals d'un array utilitzant el seu índex:

# Accedir al primer element
print $fruits[0]; # Sortida: apple

# Accedir al segon element
print $fruits[1]; # Sortida: banana

Modificar els Elements d'un Array

Pots modificar els elements d'un array assignant nous valors a índexs específics:

# Canviar el segon element
$fruits[1] = 'blueberry';
print $fruits[1]; # Sortida: blueberry

Afegir i Eliminar Elements

Perl proporciona diverses funcions per afegir o eliminar elements dels arrays:

  • push: Afegeix elements al final d'un array.
  • pop: Elimina l'últim element d'un array.
  • unshift: Afegeix elements al començament d'un array.
  • shift: Elimina el primer element d'un array.
# Afegir elements al final
push(@fruits, 'date');
print "@fruits"; # Sortida: apple blueberry cherry date

# Eliminar l'últim element
pop(@fruits);
print "@fruits"; # Sortida: apple blueberry cherry

# Afegir elements al començament
unshift(@fruits, 'apricot');
print "@fruits"; # Sortida: apricot apple blueberry cherry

# Eliminar el primer element
shift(@fruits);
print "@fruits"; # Sortida: apple blueberry cherry

Funcions i Operacions d'Array

Perl proporciona una varietat de funcions per manipular arrays:

  • scalar @array: Retorna el nombre d'elements en l'array.
  • @sorted = sort @array: Ordena l'array.
  • @reversed = reverse @array: Inverteix l'array.
# Nombre d'elements en l'array
my $count = scalar @fruits;
print $count; # Sortida: 3

# Ordenar l'array
my @sorted_fruits = sort @fruits;
print "@sorted_fruits"; # Sortida: apple blueberry cherry

# Invertir l'array
my @reversed_fruits = reverse @fruits;
print "@reversed_fruits"; # Sortida: cherry blueberry apple

Slicing Arrays

El slicing d'array et permet accedir a múltiples elements d'un array alhora:

# Slicing de l'array
my @slice = @fruits[0, 2];
print "@slice"; # Sortida: apple cherry

Arrays Multidimensionals

Perl no suporta arrays multidimensionals veritables, però pots crear arrays d'arrays:

# Crear un array multidimensional
my @matrix = (
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
);

# Accedir als elements d'un array multidimensional
print $matrix[1][2]; # Sortida: 6

Tècniques Avançades d'Array

Referències d'Array

Les referències d'array et permeten crear estructures de dades complexes com arrays d'arrays o arrays de hashes:

# Crear una referència d'array
my $array_ref = \@fruits;

# Desreferenciar una referència d'array
print @$array_ref; # Sortida: apple blueberry cherry

Utilitzar `map` i `grep`

  • map: Aplica un bloc de codi a cada element d'un array i retorna un nou array amb els resultats.
  • grep: Filtra elements d'un array basant-se en una condició.
# Utilitzar map per transformar elements
my @lengths = map { length($_) } @fruits;
print "@lengths"; # Sortida: 5 9 6

# Utilitzar grep per filtrar elements
my @filtered = grep { length($_) > 5 } @fruits;
print "@filtered"; # Sortida: blueberry cherry

Conclusió

Els arrays en Perl són eines versàtils i potents per gestionar llistes de dades. Des d'operacions bàsiques com afegir i eliminar elements fins a tècniques avançades com utilitzar referències i funcions d'ordre superior, entendre els arrays és crucial per a una programació efectiva en Perl. Practica utilitzant arrays amb diferents funcions i operacions per esdevenir proficient en la gestió d'estructures de dades complexes.
