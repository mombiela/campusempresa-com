Introducció

Perl, conegut per les seves capacitats de processament de text, és un llenguatge de programació versàtil i potent. Entendre la seva sintaxi i estructura bàsiques és essencial per escriure codi eficient i llegible. Aquesta secció cobrirà els elements fonamentals de la sintaxi de Perl, incloent variables, operadors, estructures de control i operacions bàsiques d'entrada/sortida.

Variables

Les variables en Perl s'utilitzen per emmagatzemar dades. Hi ha tres tipus principals de variables:

  • Escalars: Emmagatzemen valors únics (nombres, cadenes).
  • Arrays: Emmagatzemen llistes ordenades d'escalars.
  • Hashes: Emmagatzemen conjunts de parelles clau-valor.

Escalars

Els escalars es prefixen amb un signe de dòlar ($).

# Variables escalars
my $name = "John";
my $age = 30;
my $pi = 3.14;

Arrays

Els arrays es prefixen amb un signe d'arrova (@).

# Variables d'array
my @colors = ("red", "green", "blue");
print $colors[0]; # Mostra: red

Hashes

Els hashes es prefixen amb un signe de percentatge (%).

# Variables de hash
my %fruit_colors = (
    apple  => "red",
    banana => "yellow",
    grape  => "purple"
);
print $fruit_colors{"apple"}; # Mostra: red

Operadors

Perl admet diversos operadors per realitzar operacions sobre variables.

Operadors Aritmètics

  • Suma: +
  • Resta: -
  • Multiplicació: *
  • Divisió: /
  • Mòdul: %
my $sum = 5 + 10; # 15
my $product = 4 * 3; # 12

Operadors de Cadena

  • Concatenació: .
my $greeting = "Hello, " . "world!"; # Hello, world!

Operadors de Comparació

  • Numèrics: ==, !=, <, >, <=, >=
  • De cadena: eq, ne, lt, gt, le, ge
if (5 == 5) {
    print "Els nombres són iguals\n";
}

if ("apple" eq "apple") {
    print "Les cadenes són iguals\n";
}

Estructures de Control

Les estructures de control gestionen el flux del programa.

Declaracions Condicionals

  • if, elsif, else
my $number = 10;

if ($number > 10) {
    print "El nombre és més gran que 10\n";
} elsif ($number == 10) {
    print "El nombre és igual a 10\n";
} else {
    print "El nombre és menor que 10\n";
}

Declaracions de Bucle

  • while
  • for
  • foreach
# bucle while
my $i = 0;
while ($i < 5) {
    print "i = $i\n";
    $i++;
}

# bucle for
for (my $j = 0; $j < 5; $j++) {
    print "j = $j\n";
}

# bucle foreach
my @numbers = (1, 2, 3, 4, 5);
foreach my $num (@numbers) {
    print "num = $num\n";
}

Entrada i Sortida

Les operacions bàsiques d'entrada i sortida són essencials per interactuar amb els usuaris i els fitxers.

Entrada Estàndard

Utilitzant el manejador de fitxers <STDIN> per llegir l'entrada de l'usuari.

print "Introdueix el teu nom: ";
my $name = <STDIN>;
chomp($name); # Elimina el caràcter de nova línia
print "Hola, $name!\n";

Sortida Estàndard

Utilitzant la funció print per mostrar la sortida.

print "Aquest és un missatge simple\n";

Conclusió

Entendre la sintaxi i estructura bàsiques de Perl és el primer pas per dominar el llenguatge. Aquesta secció ha cobert els aspectes essencials, incloent variables, operadors, estructures de control i operacions bàsiques d'entrada/sortida. Amb aquests fonaments, pots començar a escriure scripts Perl senzills i gradualment passar a tasques més complexes. Continua practicant i explorant per aprofundir la teva comprensió i habilitat en Perl.
