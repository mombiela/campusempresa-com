Introducció

Perl, un llenguatge de programació altament capaç i ric en funcionalitats, ve amb una àmplia gamma de funcions integrades. Aquestes funcions estan dissenyades per realitzar una àmplia varietat de tasques, des de la manipulació bàsica de cadenes fins al processament complex de dades. Entendre aquestes funcions integrades és crucial per escriure codi Perl eficient i efectiu.

Funcions Integrades Bàsiques

Print i Say

  • print: Mostra una cadena o una llista de cadenes a la sortida estàndard.
  • say: Similar a print, però afegeix automàticament una nova línia al final de la sortida (disponible en Perl 5.10 i posteriors).
print "Hola, Món!";
say "Hola, Món!";  # Mostra "Hola, Món!" seguit d'una nova línia

Funcions Escalars

  • chomp: Elimina el caràcter de nova línia del final d'una cadena.
  • length: Retorna la longitud d'una cadena.
my $string = "Hola, Món!\n";
chomp($string);  # $string ara és "Hola, Món!"
my $length = length($string);  # $length és 12

Funcions Numèriques

  • int: Retorna la part entera d'un número.
  • rand: Genera un número aleatori entre 0 i el valor especificat.
my $number = 3.14;
my $integer = int($number);  # $integer és 3
my $random = rand(10);  # $random és un número entre 0 i 10

Funcions Integrades Intermèdies

Funcions de Llista

  • push: Afegeix elements al final d'una matriu.
  • pop: Elimina l'últim element d'una matriu.
  • shift: Elimina el primer element d'una matriu.
  • unshift: Afegeix elements al començament d'una matriu.
my @array = (1, 2, 3);
push(@array, 4);  # @array ara és (1, 2, 3, 4)
pop(@array);  # @array ara és (1, 2, 3)
shift(@array);  # @array ara és (2, 3)
unshift(@array, 1);  # @array ara és (1, 2, 3)

Funcions de Cadena

  • substr: Extreu una subcadena d'una cadena.
  • index: Troba la posició d'una subcadena dins d'una cadena.
  • split: Divideix una cadena en una llista de cadenes basant-se en un delimitador.
my $string = "Hola, Món!";
my $substring = substr($string, 0, 5);  # $substring és "Hola"
my $position = index($string, "Món");  # $position és 7
my @words = split(/, /, $string);  # @words és ("Hola", "Món!")

Funcions de Hash

  • keys: Retorna una llista de claus d'un hash.
  • values: Retorna una llista de valors d'un hash.
  • each: Itera sobre parelles clau-valor en un hash.
my %hash = (foo => 1, bar => 2);
my @keys = keys %hash;  # @keys és ("foo", "bar")
my @values = values %hash;  # @values és (1, 2)
while (my ($key, $value) = each %hash) {
    print "$key: $value\n";
}

Funcions Integrades Avançades

Funcions de Gestió de Fitxers

  • open: Obre un fitxer per llegir, escriure o afegir.
  • close: Tanca un manejador de fitxer obert.
  • read: Llegeix dades d'un manejador de fitxer.
  • write: Escriu dades a un manejador de fitxer.
open(my $fh, '<', 'file.txt') or die "No es pot obrir el fitxer: $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

Funcions d'Interacció amb el Sistema

  • system: Executa una comanda externa i espera que acabi.
  • exec: Reemplaça el procés actual amb una comanda externa.
system("ls -l");  # Executa la comanda 'ls -l'
exec("perl script.pl");  # Reemplaça el procés actual amb 'perl script.pl'

Funcions d'Expressió Regular

  • m//: Compara una cadena amb una expressió regular.
  • s///: Substitueix una part d'una cadena que coincideix amb una expressió regular.
  • tr///: Translitera caràcters en una cadena.
my $string = "Hola, Món!";
if ($string =~ m/Món/) {
    print "Coincidència trobada!\n";
}
$string =~ s/Món/Perl/;  # $string ara és "Hola, Perl!"
$string =~ tr/A-Z/a-z/;  # $string ara és "hola, perl!"

Conclusió

Les funcions integrades de Perl són eines poderoses que poden simplificar moltes tasques de programació. Des de les operacions bàsiques d'entrada/sortida fins a la manipulació complexa de dades, aquestes funcions proporcionen una base sòlida per escriure codi Perl eficient. En dominar aquestes funcions integrades, pots millorar significativament les teves habilitats de programació en Perl i abordar una àmplia gamma de problemes amb facilitat.
