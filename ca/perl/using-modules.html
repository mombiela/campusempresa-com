Introducció als Mòduls

Els mòduls en Perl són paquets que encapsulen codi i dades, fent-los reutilitzables i més fàcils de gestionar. Ajuden a organitzar el codi, evitar col·lisions d'espais de noms i promoure la reutilització del codi.

Conceptes Clau

  • Paquet: Un espai de noms separat per a variables i subrutines.
  • Mòdul: Un fitxer que conté un paquet i és reutilitzable en diferents programes.
  • CPAN: Comprehensive Perl Archive Network, un repositori de mòduls Perl.

Creació i Ús de Mòduls

Creació d'un Mòdul Simple

Per crear un mòdul, necessites:

  1. Crear un paquet.
  2. Desar-lo en un fitxer amb una extensió .pm.
  3. Retornar un valor veritable al final del fitxer.

Exemple: Creació d'un Mòdul Simple

# MyModule.pm
package MyModule;

use strict;
use warnings;

sub hello {
    return "Hola, Món!";
}

1; # Assegura't que el mòdul retorna un valor veritable

Ús d'un Mòdul

Per utilitzar un mòdul en el teu script Perl, utilitzes la paraula clau use.

Exemple: Ús d'un Mòdul Simple

# main.pl
use strict;
use warnings;
use MyModule;

print MyModule::hello(), "\n";

Exportació de Funcions

Els mòduls poden exportar funcions a l'espai de noms del cridant utilitzant el mòdul Exporter.

Exemple: Exportació de Funcions

# MyModule.pm
package MyModule;

use strict;
use warnings;
use Exporter 'import';

our @EXPORT_OK = ('hello');

sub hello {
    return "Hola, Món!";
}

1;

Exemple: Ús de Funcions Exportades

# main.pl
use strict;
use warnings;
use MyModule qw(hello);

print hello(), "\n";

Característiques Avançades dels Mòduls

Mòduls Orientats a Objectes

Perl suporta la programació orientada a objectes. Pots crear classes i objectes utilitzant mòduls.

Exemple: Mòdul Orientat a Objectes

# MyClass.pm
package MyClass;

use strict;
use warnings;

sub new {
    my $class = shift;
    my $self = {
        name => shift,
    };
    bless $self, $class;
    return $self;
}

sub get_name {
    my $self = shift;
    return $self->{name};
}

1;

Exemple: Ús d'un Mòdul Orientat a Objectes

# main.pl
use strict;
use warnings;
use MyClass;

my $object = MyClass->new("Perl");
print $object->get_name(), "\n";

Mòduls CPAN

CPAN és un vast repositori de mòduls Perl. Pots instal·lar mòduls CPAN utilitzant la comanda cpan.

Exemple: Instal·lació d'un Mòdul CPAN

cpan Some::Module

Exemple: Ús d'un Mòdul CPAN

# main.pl
use strict;
use warnings;
use Some::Module;

# Utilitza la funcionalitat del mòdul

Conclusió

Els mòduls són una característica poderosa en Perl que ajuden a organitzar i reutilitzar el codi. Des de la creació de paquets simples reutilitzables fins a aprofitar el vast repositori de CPAN, entendre com utilitzar els mòduls de manera efectiva és crucial per a qualsevol programador Perl. En dominar els mòduls, pots escriure programes Perl més nets, més mantenibles i escalables.
