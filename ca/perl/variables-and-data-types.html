Variables and Data Types in Perl
Operadors ►

Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en Perl. Entendre aquests conceptes bàsics és crucial per escriure scripts Perl efectius i eficients.

Variables en Perl

Les variables en Perl s'utilitzen per emmagatzemar dades que es poden manipular al llarg del teu programa. Perl admet tres tipus principals de variables:

  • Escalars: Emmagatzemen valors únics (nombres, cadenes, etc.).
  • Arrays: Emmagatzemen llistes ordenades d'escalars.
  • Hashes: Emmagatzemen conjunts de parelles clau-valor.

Escalars

Els escalars són el tipus més simple de variable en Perl. Poden emmagatzemar un valor únic, que pot ser un nombre, una cadena o una referència.

Sintaxi

my $scalar_variable = "Hola, món!";
my $number = 42;
my $decimal = 3.14;

Explicació

  • my s'utilitza per declarar una nova variable.
  • $ denota una variable escalar.
  • Els escalars poden contenir diferents tipus de dades, com cadenes i nombres.

Arrays

Els arrays són llistes ordenades d'escalars. Cada element en un array es pot accedir pel seu índex, començant des de 0.

Sintaxi

my @array = (1, 2, 3, 4, 5);
my @words = ("poma", "plàtan", "cirera");

Accés als Elements

print $array[0]; # Mostra: 1
print $words[1]; # Mostra: plàtan

Explicació

  • @ denota una variable d'array.
  • Els elements es poden accedir utilitzant claudàtors [] amb l'índex de l'element.

Hashes

Els hashes són conjunts de parelles clau-valor, on cada clau és única.

Sintaxi

my %hash = (
    "nom" => "Joan",
    "edat" => 30,
    "ciutat" => "Nova York"
);

Accés als Elements

print $hash{"nom"}; # Mostra: Joan
print $hash{"edat"};  # Mostra: 30

Explicació

  • % denota una variable hash.
  • Les claus i els valors es separen amb =>.
  • Els elements es poden accedir utilitzant claus {} amb la clau.

Tipus de Dades en Perl

Perl és un llenguatge de tipus dinàmic, el que significa que el tipus d'una variable es determina en temps d'execució. Els principals tipus de dades en Perl són:

  • Nombres: Enters i nombres de punt flotant.
  • Cadenes: Seqüències de caràcters.
  • Referències: Escalars que contenen la ubicació d'un altre valor.

Nombres

Els nombres en Perl poden ser enters o nombres de punt flotant.

Exemples

my $integer = 10;
my $float = 3.14;

Cadenes

Les cadenes són seqüències de caràcters tancades entre cometes.

Exemples

my $single_quoted = 'Hola, món!';
my $double_quoted = "Hola, $single_quoted";

Explicació

  • Les cadenes amb cometes simples no interpolen variables.
  • Les cadenes amb cometes dobles interpolen variables i caràcters especials.

Referències

Les referències són escalars que contenen l'adreça de memòria d'una altra variable.

Sintaxi

my $array_ref = \@array;
my $hash_ref = \%hash;

Desreferenciació

print $array_ref->[0]; # Mostra: 1
print $hash_ref->{"nom"}; # Mostra: Joan

Explicació

  • \ s'utilitza per crear una referència.
  • -> s'utilitza per desreferenciar i accedir a les dades originals.

Conclusió

Entendre les variables i els tipus de dades és essencial per programar en Perl. Els escalars, arrays i hashes són els blocs de construcció per emmagatzemar i manipular dades. El tipus dinàmic de Perl permet flexibilitat, però és important mantenir un seguiment dels tipus de dades que contenen les teves variables per evitar comportaments inesperats. Practica utilitzant aquests conceptes amb diversos exemples per convertir-te en un programador proficient en Perl.
