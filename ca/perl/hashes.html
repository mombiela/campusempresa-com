Introducció als Hashes

Els hashes, també coneguts com a arrays associatius, són una estructura de dades fonamental en Perl. Et permeten emmagatzemar dades en parelles clau-valor, facilitant la cerca de valors basats en una clau única.

  • Parelles Clau-Valor: Cada element en un hash és una parella clau-valor.
  • Claus Úniques: Les claus han de ser úniques dins d'un hash.
  • Desordenats: Els hashes no mantenen l'ordre dels elements.

Creació i Accés als Hashes

Creació d'un Hash

Pots crear un hash utilitzant la paraula clau my i una llista de parelles clau-valor.

my %fruit_colors = (
    apple  => 'red',
    banana => 'yellow',
    grape  => 'purple',
);

Accés als Elements d'un Hash

Accedeix als elements d'un hash utilitzant la clau dins de claus.

print $fruit_colors{'apple'};  # Mostra: red

Afegir i Modificar Elements

Pots afegir o modificar elements assignant un valor a una clau.

$fruit_colors{'orange'} = 'orange';  # Afegeix una nova parella clau-valor
$fruit_colors{'banana'} = 'green';   # Modifica el valor existent

Eliminar Elements

Utilitza la funció delete per eliminar una parella clau-valor d'un hash.

delete $fruit_colors{'grape'};  # Elimina la clau 'grape' i el seu valor

Iteració sobre Hashes

Utilitzant `each`

La funció each et permet iterar sobre les parelles clau-valor d'un hash.

while (my ($key, $value) = each %fruit_colors) {
    print "$key: $value\n";
}

Utilitzant `keys` i `values`

També pots iterar només sobre les claus o només sobre els valors.

foreach my $key (keys %fruit_colors) {
    print "$key\n";
}

foreach my $value (values %fruit_colors) {
    print "$value\n";
}

Funcions de Hash

`exists`

Comprova si una clau existeix en un hash.

if (exists $fruit_colors{'apple'}) {
    print "Apple és al hash.\n";
}

`keys` i `values`

Recupera totes les claus o valors d'un hash.

my @keys = keys %fruit_colors;
my @values = values %fruit_colors;

Context `scalar`

En context escalar, keys retorna el nombre de parelles clau-valor.

my $num_elements = scalar keys %fruit_colors;
print "Nombre d'elements: $num_elements\n";

Tècniques Avançades de Hash

Hash de Hashes

Pots crear estructures de dades complexes com un hash de hashes.

my %students = (
    'John' => { age => 20, major => 'Math' },
    'Jane' => { age => 22, major => 'Biology' },
);

print $students{'John'}{'age'};  # Mostra: 20

Autovivificació

Perl crea automàticament estructures de dades niades segons sigui necessari.

$students{'Alice'}{'age'} = 23;  # Crea la clau 'Alice' i el seu hash niat

Errors Comuns

Utilitzar Claus No Definides

Accedir a un hash amb una clau no definida retorna undef.

my $color = $fruit_colors{'pear'};  # Retorna undef

Sobreescriure Claus

Assignar un valor a una clau existent sobreescriu el valor anterior.

$fruit_colors{'apple'} = 'green';  # 'red' és sobreescrit amb 'green'

Conclusió

Els hashes són una estructura de dades versàtil i poderosa en Perl, permetent l'emmagatzematge i recuperació eficient de dades utilitzant parelles clau-valor. Entenent com crear, accedir i manipular hashes, pots gestionar estructures de dades més complexes i millorar l'eficiència dels teus programes en Perl. Practica utilitzant hashes amb diversos exemples per convertir-te en un expert en el seu ús.
