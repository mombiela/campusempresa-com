  • Instal·lació de mòduls necessaris utilitzant CPAN.

Conceptes bàsics del web

  • Comprensió de HTTP i com funcionen els servidors web.
  • Introducció a CGI (Common Gateway Interface).

Escriure el teu primer script Perl CGI

  • Creació d'un simple script Perl CGI.
  • Explicació de l'estructura de l'script.
  • Executar l'script en un servidor web.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Hello World');
print $cgi->h1('Hello World from Perl CGI!');
print $cgi->end_html;

Gestió de dades de formularis

  • Creació de formularis HTML.
  • Recuperació de dades de formularis utilitzant Perl CGI.
  • Validació i processament de l'entrada del formulari.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Form Example');
print $cgi->start_form;
print $cgi->textfield(-name=>'name');
print $cgi->submit;
print $cgi->end_form;

if ($cgi->param) {
    my $name = $cgi->param('name');
    print $cgi->h1("Hello, $name!");
}
print $cgi->end_html;

Integració de bases de dades

  • Connexió a una base de dades utilitzant DBI.
  • Realització d'operacions CRUD.
  • Utilització d'ORM (Object-Relational Mapping) amb DBIx::Class.
use DBI;

my $dbh = DBI->connect("DBI:mysql:database=testdb;host=localhost", "user", "password", {'RaiseError' => 1});

my $sth = $dbh->prepare("SELECT * FROM users");
$sth->execute();

while (my @row = $sth->fetchrow_array) {
    print "User: $row[0], Email: $row[1]\n";
}

$sth->finish;
$dbh->disconnect;
