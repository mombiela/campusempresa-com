Interactuar amb Bases de Dades
Programació Web amb Perl ►

Introducció

Interactuar amb bases de dades és una habilitat crucial per a qualsevol programador. En aquesta secció, explorarem com connectar-se, fer consultes i manipular bases de dades utilitzant Perl. Cobrirem tant tècniques bàsiques com avançades, assegurant-nos que tinguis una comprensió sòlida de les interaccions amb bases de dades.

Configuració de l'Entorn

Abans de submergir-nos en les interaccions amb bases de dades, necessitem configurar el nostre entorn.

  • Instal·lar el Mòdul DBI: DBI (Database Interface) és el mòdul d'interfície de base de dades estàndard per a Perl. 
    cpan DBI
  • Instal·lar el Mòdul DBD: Depenent de la base de dades que estiguis utilitzant (per exemple, MySQL, PostgreSQL), necessitaràs el mòdul DBD (Database Driver) apropiat. 
    cpan DBD::mysql

Connectar-se a una Base de Dades

Per interactuar amb una base de dades, primer necessites establir una connexió.

  • DBI->connect: El mètode connect s'utilitza per establir una connexió amb la base de dades. 
    use DBI;

my $dsn = "DBI:mysql:database=testdb;host=localhost";
my $username = "root";
my $password = "password";

my $dbh = DBI->connect($dsn, $username, $password, {
    RaiseError => 1,
    AutoCommit => 1,
}) or die $DBI::errstr;

Executar Consultes SQL

Un cop connectat, pots executar consultes SQL per interactuar amb la base de dades.

  • Consulta SELECT: Recuperar dades de la base de dades.

    my $sth = $dbh->prepare("SELECT * FROM users");
$sth->execute();

while (my @row = $sth->fetchrow_array) {
    print "User: $row[0], Email: $row[1]\n";
}

  • Consulta INSERT: Inserir dades a la base de dades.

    my $sth = $dbh->prepare("INSERT INTO users (name, email) VALUES (?, ?)");
$sth->execute('John Doe', 'john.doe@example.com');

  • Consulta UPDATE: Actualitzar dades existents a la base de dades.

    my $sth = $dbh->prepare("UPDATE users SET email = ? WHERE name = ?");
$sth->execute('john.new@example.com', 'John Doe');

  • Consulta DELETE: Esborrar dades de la base de dades.

    my $sth = $dbh->prepare("DELETE FROM users WHERE name = ?");
$sth->execute('John Doe');

Gestió d'Errors

La gestió d'errors és crucial quan s'interactua amb bases de dades per assegurar que la teva aplicació pugui gestionar els problemes de manera elegant.

  • RaiseError: Eleva automàticament un error si un mètode DBI falla.
  • PrintError: Imprimeix errors a STDERR si un mètode DBI falla.

Exemple:

my $dbh = DBI->connect($dsn, $username, $password, {
    RaiseError => 1,
    PrintError => 0,
    AutoCommit => 1,
}) or die $DBI::errstr;

Temes Avançats

Transaccions

Les transaccions et permeten executar una sèrie d'operacions com una única unitat de treball.

  • Iniciar Transacció: Comença una transacció.

    $dbh->begin_work;

  • Confirmar Transacció: Confirma la transacció.

    $dbh->commit;

  • Desfer Transacció: Desfés la transacció en cas d'error.

    $dbh->rollback;

Exemple:

eval {
    $dbh->begin_work;
    $dbh->do("INSERT INTO users (name, email) VALUES ('Jane Doe', 'jane.doe@example.com')");
    $dbh->do("UPDATE users SET email = 'jane.new@example.com' WHERE name = 'Jane Doe'");
    $dbh->commit;
};
if ($@) {
    warn "Transacció avortada: $@";
    $dbh->rollback;
}

Ús de Marcadors de Posició

Els marcadors de posició ajuden a prevenir la injecció SQL i fan que les teves consultes siguin més eficients.

  • Ús de Marcadors de Posició: Utilitza ? com a marcador de posició en les teves instruccions SQL. 
    my $sth = $dbh->prepare("INSERT INTO users (name, email) VALUES (?, ?)");
$sth->execute('John Doe', 'john.doe@example.com');

Resum

En aquesta secció, hem cobert les bases d'interactuar amb bases de dades utilitzant Perl. Hem après com configurar l'entorn, connectar-se a una base de dades, executar diverses consultes SQL, gestionar errors, i utilitzar transaccions i marcadors de posició. Amb aquestes habilitats, pots gestionar eficientment les operacions de bases de dades en les teves aplicacions Perl.
