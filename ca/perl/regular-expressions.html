Introducció a les expressions regulars

Les expressions regulars (regex) són eines poderoses per a la coincidència de patrons i la manipulació de text. En Perl, les regex estan integrades directament en el llenguatge, convertint-les en una eina versàtil per al processament de cadenes.

Conceptes clau

  • Coincidència de patrons: Trobar patrons específics dins de cadenes.
  • Substitució: Substituir parts d'una cadena que coincideixen amb un patró.
  • Modificadors: Canviar el comportament de les operacions regex.

Sintaxi bàsica

Perl utilitza l'operador =~ per aplicar una regex a una cadena.

Exemple

my $string = "Hello, world!";
if ($string =~ /world/) {
    print "Match found!\n";
}
  • /world/ és el patró regex.
  • =~ aplica el patró a $string.

Metacaràcters comuns

  • .: Coincideix amb qualsevol caràcter únic excepte el salt de línia.
  • *: Coincideix amb 0 o més de l'element precedent.
  • +: Coincideix amb 1 o més de l'element precedent.
  • ?: Coincideix amb 0 o 1 de l'element precedent.
  • \d: Coincideix amb qualsevol dígit.
  • \w: Coincideix amb qualsevol caràcter de paraula (alfanumèric més guió baix).
  • \s: Coincideix amb qualsevol caràcter d'espai en blanc.

Classes de caràcters i quantificadors

Les classes de caràcters permeten coincidir amb qualsevol dels diversos caràcters.

Exemple

my $string = "abc123";
if ($string =~ /[a-z]/) {
    print "Contains a lowercase letter.\n";
}
  • [a-z] coincideix amb qualsevol lletra minúscula.

Quantificadors

  • {n}: Coincideix exactament amb n ocurrències.
  • {n,}: Coincideix amb n o més ocurrències.
  • {n,m}: Coincideix entre n i m ocurrències.

Exemple

my $string = "aaaab";
if ($string =~ /a{3,5}/) {
    print "Matches between 3 and 5 'a's.\n";
}

Agrupació i captura

Els parèntesis () s'utilitzen per agrupar parts d'una regex i capturar subcadenes coincidents.

Exemple

my $string = "Hello, world!";
if ($string =~ /(Hello), (world)/) {
    print "First part: $1\n";
    print "Second part: $2\n";
}
  • $1 i $2 emmagatzemen les subcadenes coincidents.

Substitució

L'operador s/// s'utilitza per a la substitució.

Exemple

my $string = "Hello, world!";
$string =~ s/world/Perl/;
print "$string\n";  # Output: Hello, Perl!
  • s/world/Perl/ substitueix "world" per "Perl".

Temes avançats

Lookahead i Lookbehind

Les afirmacions lookahead i lookbehind permeten coincidir amb un patró només si està (o no està) seguit o precedit per un altre patró.

Exemple

my $string = "abc123";
if ($string =~ /abc(?=\d)/) {
    print "abc is followed by a digit.\n";
}
  • (?=\d) és un lookahead positiu.

Grups no capturadors

Els grups no capturadors (?:...) s'utilitzen per agrupar parts d'una regex sense capturar les subcadenes coincidents.

Exemple

my $string = "Hello, world!";
if ($string =~ /(?:Hello), (world)/) {
    print "Matched: $1\n";  # Output: Matched: world
}

Modificadors

Els modificadors canvien el comportament de les operacions regex.

Modificadors comuns

  • i: Coincidència insensible a majúscules i minúscules.
  • m: Tracta la cadena com a múltiples línies.
  • s: Tracta la cadena com a una sola línia.
  • g: Coincidència global.

Exemple

my $string = "Hello, world!";
if ($string =~ /HELLO/i) {
    print "Case-insensitive match found!\n";
}
  • /HELLO/i coincideix amb "HELLO" sense tenir en compte majúscules i minúscules.

Exercicis pràctics

Exercici 1: Coincidència bàsica

Escriu un script Perl per comprovar si una cadena conté la paraula "Perl".

Exercici 2: Substitució

Escriu un script Perl per substituir totes les ocurrències de "foo" per "bar" en una cadena donada.

Exercici 3: Grups de captura

Escriu un script Perl per extreure el nom de domini d'una adreça de correu electrònic.

Conclusió

Les expressions regulars en Perl són una eina poderosa per al processament de text. Comprenent la sintaxi bàsica, les classes de caràcters, els quantificadors, l'agrupació i les característiques avançades com el lookahead i el lookbehind, pots realitzar tasques complexes de coincidència de patrons i manipulació de text de manera eficient. Practica amb els exercicis proporcionats per consolidar la teva comprensió i convertir-te en un expert en l'ús de regex en Perl.
