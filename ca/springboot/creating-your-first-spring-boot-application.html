En aquesta secció, et guiarem a través del procés de creació de la teva primera aplicació Spring Boot. Això inclourà la configuració del teu entorn de desenvolupament, la creació d'un nou projecte Spring Boot i la comprensió de l'estructura bàsica d'una aplicació Spring Boot.

Configuració del teu entorn de desenvolupament

Abans de començar a construir la teva aplicació Spring Boot, has de configurar el teu entorn de desenvolupament.

  • Java Development Kit (JDK): Assegura't de tenir instal·lat el JDK 8 o superior. Pots descarregar-lo des del lloc web oficial d'Oracle.
  • Entorn de desenvolupament integrat (IDE): Recomanem utilitzar IntelliJ IDEA o Eclipse. Tots dos tenen un excel·lent suport per a Spring Boot.
  • Maven o Gradle: Aquests són eines de construcció que gestionen les dependències i construeixen el projecte. Maven és més comunament utilitzat amb Spring Boot.

Instal·lació del JDK

  1. Descarrega el JDK des del lloc web d'Oracle.
  2. Segueix les instruccions d'instal·lació per al teu sistema operatiu.
  3. Verifica la instal·lació executant la següent comanda al teu terminal: 
    java -version

Configuració del teu IDE

  1. IntelliJ IDEA:

    • Descarrega i instal·la IntelliJ IDEA des del lloc web oficial de JetBrains.
    • Obre IntelliJ IDEA i configura'l per al desenvolupament en Java.

  2. Eclipse:

    • Descarrega i instal·la Eclipse des del lloc web oficial d'Eclipse.
    • Obre Eclipse i configura'l per al desenvolupament en Java.

Creació d'un nou projecte Spring Boot

Pots crear un nou projecte Spring Boot utilitzant Spring Initializr, una eina basada en web proporcionada per l'equip de Spring.

Utilitzant Spring Initializr

  1. Ves a Spring Initializr.
  2. Omple els detalls del projecte:
    • Projecte: Maven Project
    • Llenguatge: Java
    • Spring Boot: 2.5.4 (o la versió estable més recent)
    • Metadades del projecte:
      • Grup: com.example
      • Artifact: demo
      • Nom: demo
      • Descripció: Projecte de demostració per a Spring Boot
      • Nom del paquet: com.example.demo
      • Packaging: Jar
      • Java: 8 (o superior)
  3. Afegeix dependències:
    • Spring Web: Per construir aplicacions web, incloent serveis RESTful.
  4. Fes clic al botó "Generate" per descarregar el projecte com un fitxer ZIP.
  5. Extreu el fitxer ZIP i obre el projecte al teu IDE.

Components clau

  • DemoApplication.java: Aquesta és la classe principal que inicia l'aplicació Spring Boot.

    package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
    • @SpringBootApplication: Aquesta anotació marca la classe principal d'una aplicació Spring Boot.
    • SpringApplication.run(): Aquest mètode llança l'aplicació.

  • application.properties: Aquest fitxer s'utilitza per configurar la teva aplicació Spring Boot. Pots establir diverses propietats com el port del servidor, configuracions de la base de dades, etc.

Executant la teva aplicació Spring Boot

Per executar la teva aplicació Spring Boot, segueix aquests passos:

  1. Obre el teu IDE i navega fins al fitxer DemoApplication.java.
  2. Fes clic dret sobre el fitxer i selecciona "Run DemoApplication.main()".
