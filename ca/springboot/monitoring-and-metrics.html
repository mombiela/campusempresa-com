Introducció al Monitoratge i Mètriques

El monitoratge i les mètriques són crucials per entendre la salut i el rendiment de les teves aplicacions Spring Boot. Aquesta secció cobrirà els conceptes bàsics del monitoratge, per què és important i com implementar-lo en Spring Boot.

  • Monitoratge: El procés de recollir, analitzar i utilitzar informació per fer un seguiment del rendiment i la salut d'una aplicació.
  • Mètriques: Mesures quantitatives utilitzades per avaluar el rendiment i la salut d'una aplicació.

Configuració de l'Actuator

Spring Boot Actuator proporciona característiques preparades per a la producció per ajudar-te a monitoritzar i gestionar la teva aplicació.

Afegir la Dependència de l'Actuator

Per començar amb l'Actuator, has d'afegir la següent dependència al teu fitxer pom.xml:

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

Activar els Endpoints de l'Actuator

Per defecte, només uns pocs endpoints estan activats. Pots activar més endpoints configurant el teu fitxer application.properties:

management.endpoints.web.exposure.include=*

Endpoints Comuns de l'Actuator

  • /actuator/health: Mostra informació sobre la salut de l'aplicació.
  • /actuator/info: Mostra informació arbitrària de l'aplicació.
  • /actuator/metrics: Exposa les mètriques recollides per l'aplicació.

Personalització dels Endpoints de l'Actuator

Pots personalitzar el comportament dels endpoints de l'Actuator per adaptar-los a les teves necessitats.

Exemple: Indicador de Salut Personalitzat

Crea un indicador de salut personalitzat implementant la interfície HealthIndicator:

import org.springframework.boot.actuate.health.Health;
import org.springframework.boot.actuate.health.HealthIndicator;
import org.springframework.stereotype.Component;

@Component
public class CustomHealthIndicator implements HealthIndicator {

    @Override
    public Health health() {
        boolean isHealthy = checkHealth();
        if (isHealthy) {
            return Health.up().withDetail("Custom Health", "All systems go").build();
        } else {
            return Health.down().withDetail("Custom Health", "Something is wrong").build();
        }
    }

    private boolean checkHealth() {
        // Lògica personalitzada de verificació de salut
        return true;
    }
}

Recollida de Mètriques

Spring Boot Actuator s'integra amb Micrometer per proporcionar un sistema de recollida de mètriques flexible i potent.

Afegir la Dependència de Micrometer

Afegeix la dependència de Micrometer al teu fitxer pom.xml:

<dependency>
    <groupId>io.micrometer</groupId>
    <artifactId>micrometer-core</artifactId>
</dependency>

Mètriques Comunes

  • Mètriques de JVM: Ús de memòria, recollida d'escombraries, fils.
  • Mètriques HTTP: Comptador de sol·licituds, temps de resposta.
  • Mètriques de Base de Dades: Ús del pool de connexions, rendiment de les consultes.

Exemple: Mètrica Personalitzada

Pots crear mètriques personalitzades utilitzant el MeterRegistry de Micrometer:

import io.micrometer.core.instrument.MeterRegistry;
import org.springframework.stereotype.Component;

import javax.annotation.PostConstruct;

@Component
public class CustomMetrics {

    private final MeterRegistry meterRegistry;

    public CustomMetrics(MeterRegistry meterRegistry) {
        this.meterRegistry = meterRegistry;
    }

    @PostConstruct
    public void init() {
        meterRegistry.counter("custom.metric.counter").increment();
    }
}

Integració amb Eines de Monitoratge

Spring Boot Actuator i Micrometer es poden integrar amb diverses eines de monitoratge com Prometheus, Grafana i New Relic.

Exemple: Integració amb Prometheus

Afegeix la dependència de Prometheus al teu fitxer pom.xml:

<dependency>
    <groupId>io.micrometer</groupId>
    <artifactId>micrometer-registry-prometheus</artifactId>
</dependency>

Configura Prometheus al teu fitxer application.properties:

management.metrics.export.prometheus.enabled=true

Prometheus recollirà les mètriques des de l'endpoint /actuator/prometheus.

Conclusió

El monitoratge i les mètriques són essencials per mantenir la salut i el rendiment de les teves aplicacions Spring Boot. Utilitzant Spring Boot Actuator i Micrometer, pots recollir i analitzar fàcilment les mètriques, personalitzar les verificacions de salut i integrar-te amb eines de monitoratge populars. Això assegura que puguis identificar i resoldre problemes de manera proactiva, conduint a aplicacions més fiables i eficients.
