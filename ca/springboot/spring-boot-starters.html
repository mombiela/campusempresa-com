Spring Boot Starters són un conjunt de descriptors de dependències convenients que pots incloure a la teva aplicació. Simplifiquen el procés de configuració i configuració de la teva aplicació Spring Boot proporcionant un conjunt seleccionat de dependències i configuracions.

Què són els Spring Boot Starters?

  • Definició: Els Spring Boot Starters són plantilles predefinides que inclouen un conjunt de dependències i configuracions per arrencar ràpidament la teva aplicació Spring Boot.
  • Propòsit: El seu objectiu és reduir la quantitat de codi i configuració repetitiva que necessites escriure, permetent-te centrar-te en desenvolupar la lògica de negoci de la teva aplicació.

Spring Boot Starters més utilitzats

Aquí tens alguns dels Spring Boot Starters més utilitzats:

  • spring-boot-starter-web: Per construir aplicacions web, incloent-hi RESTful, utilitzant Spring MVC. Utilitza Tomcat com a contenidor incrustat per defecte.
  • spring-boot-starter-data-jpa: Per utilitzar Spring Data JPA amb Hibernate.
  • spring-boot-starter-security: Per afegir Spring Security a la teva aplicació.
  • spring-boot-starter-test: Per provar aplicacions Spring Boot amb biblioteques com JUnit, Hamcrest i Mockito.

Afegir Spring Boot Starters al teu projecte

Per afegir un Spring Boot Starter al teu projecte, has d'incloure'l al teu fitxer pom.xml si utilitzes Maven, o al teu fitxer build.gradle si utilitzes Gradle.

Utilitzant Maven

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-jpa</artifactId>
    </dependency>
</dependencies>

Utilitzant Gradle

dependencies {
    implementation 'org.springframework.boot:spring-boot-starter-web'
    implementation 'org.springframework.boot:spring-boot-starter-data-jpa'
}

Exemple: Crear una aplicació web senzilla

Creem una aplicació web senzilla utilitzant el starter spring-boot-starter-web.

Pas 1: Configura el teu projecte

  1. Crea un nou projecte Spring Boot utilitzant Spring Initializr (https://start.spring.io/).
  2. Selecciona "Maven Project" i "Java".
  3. Afegeix la dependència "Spring Web".
  4. Genera el projecte i descomprimeix-lo.

Pas 2: Crea un controlador

Crea una nova classe Java anomenada HelloController al directori src/main/java/com/example/demo.

package com.example.demo;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String sayHello() {
        return "Hello, World!";
    }
}

Pas 3: Executa la teva aplicació

Executa la teva aplicació executant el mètode main a la classe DemoApplication.

package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Pas 4: Prova la teva aplicació

Obre un navegador web i navega a http://localhost:8080/hello. Hauries de veure el missatge "Hello, World!".

Conclusió

Els Spring Boot Starters són una característica poderosa que simplifica el procés de configuració i configuració de les teves aplicacions Spring Boot. Incloent un starter al teu projecte, pots començar ràpidament amb un conjunt de dependències i configuracions preconfigurades. Això et permet centrar-te en escriure la lògica de negoci de la teva aplicació en lloc de tractar amb codi i configuració repetitiva.

En aquest tema, hem cobert:

  • Què són els Spring Boot Starters i el seu propòsit.
  • Spring Boot Starters més utilitzats.
  • Com afegir Spring Boot Starters al teu projecte utilitzant Maven i Gradle.
  • Un exemple pas a pas de com crear una aplicació web senzilla utilitzant el starter spring-boot-starter-web.

Entenent i utilitzant els Spring Boot Starters, pots accelerar significativament el teu procés de desenvolupament i crear aplicacions Spring Boot robustes amb facilitat.
