Spring Boot és un framework de codi obert basat en Java utilitzat per crear aplicacions autònomes i de grau de producció basades en Spring amb un esforç mínim. Està dissenyat per simplificar el procés de desenvolupament proporcionant un conjunt de convencions i valors per defecte per reduir el codi boilerplate i la configuració.

Conceptes Clau de Spring Boot

  • Convenció sobre Configuració: Spring Boot segueix el principi de convenció sobre configuració, el que significa que proporciona valors per defecte raonables per reduir la necessitat d'una configuració extensa.
  • Aplicacions Autònomes: Les aplicacions Spring Boot poden funcionar de manera independent sense requerir un servidor d'aplicacions extern.
  • Servidors Integrats: Spring Boot inclou servidors integrats com Tomcat, Jetty o Undertow, el que facilita el desplegament d'aplicacions.
  • Funcions Preparades per a Producció: Inclou funcions com mètriques, comprovacions de salut i configuració externalitzada per fer que les aplicacions estiguin preparades per a la producció.
  • Spring Boot Starters: Aquests són un conjunt de descriptors de dependències convenients per simplificar la inclusió de dependències en el teu projecte.
  • Spring Boot CLI: Una eina de línia de comandes que es pot utilitzar per prototipar ràpidament amb Spring.

Per què Utilitzar Spring Boot?

  • Gestió de Dependències Simplificada: Els starters de Spring Boot simplifiquen la gestió de dependències proporcionant un conjunt de dependències preconfigurades.
  • Auto-Configuració: Configura automàticament la teva aplicació Spring basant-se en les dependències que has afegit.
  • Suport per a Microserveis: Ideal per construir microserveis degut a la seva naturalesa lleugera i modular.
  • Desenvolupament Ràpid: Accelera el procés de desenvolupament reduint el codi boilerplate i la configuració.
  • Comunitat i Ecosistema: Fort suport comunitari i un ric ecosistema de plugins i extensions.

Començant amb Spring Boot

Configurant un Projecte Spring Boot

Pots configurar un projecte Spring Boot utilitzant diversos mètodes, incloent Spring Initializr, plugins d'IDE o configurant manualment el teu projecte.

Utilitzant Spring Initializr

Spring Initializr és una eina basada en web per arrencar el teu projecte Spring Boot.

  1. Ves a Spring Initializr.
  2. Omple els detalls del projecte:
    • Projecte: Projecte Maven
    • Llenguatge: Java
    • Spring Boot: (selecciona la versió estable més recent)
    • Metadades del Projecte: Grup, Artifact, Nom, Descripció, Nom del Paquet, Embalatge, Versió de Java
  3. Afegeix dependències (per exemple, Spring Web, Spring Data JPA, H2 Database).
  4. Fes clic a "Generate" per descarregar el projecte com un fitxer ZIP.
  5. Extreu el fitxer ZIP i importa el projecte al teu IDE.

Exemple: Aplicació Bàsica Spring Boot

package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}

Explicació:

  • @SpringBootApplication: Aquesta anotació és una combinació de tres anotacions: @EnableAutoConfiguration, @ComponentScan i @Configuration. Marca la classe principal d'una aplicació Spring Boot.
  • SpringApplication.run(DemoApplication.class, args): Aquest mètode estàtic llança l'aplicació Spring Boot.

Executant l'Aplicació

Per executar l'aplicació Spring Boot, pots utilitzar els següents mètodes:

  • Des de l'IDE: Fes clic dret a la classe principal (DemoApplication) i selecciona "Run".
  • Des de la Línia de Comandes: Navega al directori del projecte i executa mvn spring-boot:run.

Conclusió

Spring Boot és un potent framework que simplifica el desenvolupament d'aplicacions Spring proporcionant un conjunt de convencions i valors per defecte. Permet als desenvolupadors crear aplicacions autònomes i preparades per a la producció amb una configuració mínima. Aprofitant funcions com l'auto-configuració, els servidors integrats i els starters de Spring Boot, els desenvolupadors poden centrar-se més en escriure la lògica de negoci en lloc de tractar amb codi boilerplate i configuració.

En les següents seccions d'aquest curs, aprofundirem en diversos aspectes de Spring Boot, incloent la gestió de dependències, la construcció d'APIs RESTful, l'accés a dades, la seguretat i el desplegament d'aplicacions Spring Boot.
