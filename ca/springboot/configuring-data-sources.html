En aquesta secció, explorarem com configurar fonts de dades en Spring Boot. Aquest és un aspecte crucial de qualsevol aplicació que interactua amb una base de dades. Cobrirem els conceptes bàsics de la configuració d'una font de dades, la configuració de múltiples fonts de dades i configuracions avançades.

Introducció a les Fonts de Dades

Una font de dades és una fàbrica per a connexions a una font de dades física, com ara una base de dades. En Spring Boot, configurar una font de dades és senzill gràcies a les seves capacitats d'auto-configuració.

Conceptes Clau

  • DataSource: Una interfície que proporciona una manera estàndard de connectar-se a una base de dades.
  • JdbcTemplate: Una utilitat de Spring que simplifica les operacions JDBC.
  • EntityManager: Una interfície JPA utilitzada per interactuar amb el context de persistència.

Configuració Bàsica de la Font de Dades

Spring Boot simplifica les configuracions de bases de dades proporcionant configuracions i propietats per defecte. Comencem amb una configuració bàsica.

Propietats de l'Aplicació

Spring Boot utilitza application.properties o application.yml per configurar la font de dades. Aquí teniu un exemple utilitzant application.properties:

spring.datasource.url=jdbc:mysql://localhost:3306/mydb
spring.datasource.username=root
spring.datasource.password=secret
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

Exemple de Codi

També podeu configurar la font de dades programàticament en una classe @Configuration:

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.boot.jdbc.DataSourceBuilder;
import javax.sql.DataSource;

@Configuration
public class DataSourceConfig {

    @Bean
    public DataSource dataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/mydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }
}

Configurant Múltiples Fonts de Dades

En algunes aplicacions, potser necessiteu connectar-vos a múltiples bases de dades. Spring Boot us permet configurar múltiples fonts de dades fàcilment.

Propietats de l'Aplicació

Definiu múltiples fonts de dades en application.properties:

# Font de Dades Primària
spring.datasource.primary.url=jdbc:mysql://localhost:3306/primarydb
spring.datasource.primary.username=root
spring.datasource.primary.password=secret
spring.datasource.primary.driver-class-name=com.mysql.cj.jdbc.Driver

# Font de Dades Secundària
spring.datasource.secondary.url=jdbc:mysql://localhost:3306/secondarydb
spring.datasource.secondary.username=root
spring.datasource.secondary.password=secret
spring.datasource.secondary.driver-class-name=com.mysql.cj.jdbc.Driver

Exemple de Codi

Creeu una classe de configuració per definir múltiples fonts de dades:

import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.boot.jdbc.DataSourceBuilder;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.Primary;

import javax.sql.DataSource;

@Configuration
public class MultiDataSourceConfig {

    @Primary
    @Bean(name = "primaryDataSource")
    public DataSource primaryDataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/primarydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }

    @Bean(name = "secondaryDataSource")
    public DataSource secondaryDataSource() {
        return DataSourceBuilder.create()
                .url("jdbc:mysql://localhost:3306/secondarydb")
                .username("root")
                .password("secret")
                .driverClassName("com.mysql.cj.jdbc.Driver")
                .build();
    }
}

Configuració Avançada de la Font de Dades

Per a configuracions avançades, potser necessiteu personalitzar el pool de connexions, la gestió de transaccions o utilitzar JPA.

Pool de Connexions

Spring Boot admet diverses biblioteques de pool de connexions com HikariCP, Tomcat i DBCP2. HikariCP és el predeterminat.

Exemple amb HikariCP

spring.datasource.hikari.maximum-pool-size=10
spring.datasource.hikari.minimum-idle=5
spring.datasource.hikari.idle-timeout=30000
spring.datasource.hikari.pool-name=MyHikariCP

Configuració de JPA

Si esteu utilitzant JPA, necessiteu configurar el EntityManagerFactory i el TransactionManager.

Exemple de Configuració

import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.boot.orm.jpa.EntityManagerFactoryBuilder;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.data.jpa.repository.config.EnableJpaRepositories;
import org.springframework.orm.jpa.JpaTransactionManager;
import org.springframework.orm.jpa.LocalContainerEntityManagerFactoryBean;
import org.springframework.transaction.PlatformTransactionManager;

import javax.persistence.EntityManagerFactory;
import javax.sql.DataSource;

@Configuration
@EnableJpaRepositories(
        basePackages = "com.example.repository",
        entityManagerFactoryRef = "entityManagerFactory",
        transactionManagerRef = "transactionManager"
)
public class JpaConfig {

    @Bean(name = "entityManagerFactory")
    public LocalContainerEntityManagerFactoryBean entityManagerFactory(
            EntityManagerFactoryBuilder builder, @Qualifier("primaryDataSource") DataSource dataSource) {
        return builder
                .dataSource(dataSource)
                .packages("com.example.model")
                .persistenceUnit("primary")
                .build();
    }

    @Bean(name = "transactionManager")
    public PlatformTransactionManager transactionManager(
            @Qualifier("entityManagerFactory") EntityManagerFactory entityManagerFactory) {
        return new JpaTransactionManager(entityManagerFactory);
    }
}

Conclusió

Configurar fonts de dades en Spring Boot és una habilitat fonamental per a qualsevol desenvolupador que treballi amb bases de dades. Hem cobert:

  • Configuració bàsica de la font de dades utilitzant application.properties i configuració programàtica.
  • Configuració de múltiples fonts de dades.
  • Configuracions avançades com el pool de connexions i la configuració de JPA.

Entendre aquestes configuracions us permetrà construir aplicacions Spring Boot robustes i escalables que interactuen eficientment amb bases de dades.
