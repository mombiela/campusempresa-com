Configurar el teu entorn de desenvolupament és el primer pas per construir aplicacions amb Spring Boot. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar, des d'instal·lar Java fins a configurar el teu IDE i crear el teu primer projecte Spring Boot.

Prerequisits

Abans de començar, assegura't de tenir els següents prerequisits instal·lats a la teva màquina:

  • Java Development Kit (JDK) 8 o superior
  • Entorn de Desenvolupament Integrat (IDE)
  • Maven o Gradle (eines d'automatització de construcció)
  • Git (sistema de control de versions)

Instal·lació del Java Development Kit (JDK)

Spring Boot requereix Java per funcionar. Segueix aquests passos per instal·lar el JDK:

  1. Descarrega el JDK:

    • Visita la pàgina de descàrrega de l'Oracle JDK o OpenJDK.
    • Tria la versió adequada per al teu sistema operatiu.

  2. Instal·la el JDK:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.
    • Configura la variable d'entorn JAVA_HOME per apuntar al directori d'instal·lació del JDK.

  3. Verifica la instal·lació:

    java -version

Configuració del teu IDE

Un Entorn de Desenvolupament Integrat (IDE) simplifica el procés de desenvolupament. Les opcions populars per a Spring Boot inclouen IntelliJ IDEA, Eclipse i Visual Studio Code.

IntelliJ IDEA

  1. Descarrega i Instal·la:

    • Visita la pàgina de descàrrega d'IntelliJ IDEA.
    • Tria l'edició Community o Ultimate i segueix les instruccions d'instal·lació.

  2. Configura IntelliJ IDEA:

    • Obre IntelliJ IDEA i configura el JDK:
      • Ves a File > Project Structure > Project.
      • Configura el Project SDK al JDK que has instal·lat anteriorment.

Eclipse

  1. Descarrega i Instal·la:

    • Visita la pàgina de descàrrega d'Eclipse.
    • Tria l'Eclipse IDE per a desenvolupadors Java i segueix les instruccions d'instal·lació.

  2. Configura Eclipse:

    • Obre Eclipse i configura el JDK:
      • Ves a Window > Preferences > Java > Installed JREs.
      • Afegeix el JDK que has instal·lat anteriorment.

Visual Studio Code

  1. Descarrega i Instal·la:

    • Visita la pàgina de descàrrega de Visual Studio Code.
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  2. Instal·la Extensions:

    • Instal·la les següents extensions:
      • Java Extension Pack
      • Spring Boot Extension Pack

Configuració de Maven o Gradle

Els projectes Spring Boot es poden construir utilitzant Maven o Gradle. Tria un basant-te en la teva preferència.

Maven

  1. Descarrega i Instal·la:

    • Visita la pàgina de descàrrega de Maven.
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  2. Verifica la instal·lació:

    mvn -v

Gradle

  1. Descarrega i Instal·la:

    • Visita la pàgina de descàrrega de Gradle.
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  2. Verifica la instal·lació:

    gradle -v

Creació del teu primer projecte Spring Boot

Pots crear un projecte Spring Boot utilitzant Spring Initializr, una eina basada en web que genera una estructura de projecte Spring Boot.

  1. Visita Spring Initializr:

    • Ves a start.spring.io.

  2. Configura el teu projecte:

    • Tria les següents configuracions:
      • Projecte: Maven Project o Gradle Project
      • Llenguatge: Java
      • Spring Boot: Última versió estable
      • Metadades del projecte: Group, Artifact, Name, Description, Package Name
      • Dependències: Afegeix dependències com Spring Web, Spring Data JPA, etc.

  3. Genera i Descarrega:

    • Fes clic al botó Generate per descarregar el projecte com un fitxer ZIP.
    • Extreu el fitxer ZIP a la teva ubicació desitjada.

  4. Importa el projecte al teu IDE:

    • Obre el teu IDE i importa el projecte:
      • Per a IntelliJ IDEA: File > Open i selecciona el directori del projecte.
      • Per a Eclipse: File > Import > Existing Maven Projects i selecciona el directori del projecte.
      • Per a Visual Studio Code: File > Open Folder i selecciona el directori del projecte.

Executar la teva aplicació Spring Boot

Un cop hagis configurat el teu projecte, pots executar la teva aplicació Spring Boot.

  1. Navega al directori del projecte:

    cd path/to/your/project

  2. Executa l'aplicació:

    • Utilitzant Maven: 
      mvn spring-boot:run
    • Utilitzant Gradle: 
      gradle bootRun

  3. Verifica que l'aplicació està en funcionament:

    • Obre un navegador web i ves a http://localhost:8080.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial per començar amb Spring Boot. Seguint els passos descrits en aquesta secció, hauràs de tenir un entorn completament funcional preparat per desenvolupar aplicacions Spring Boot. Assegura't de tenir les eines necessàries instal·lades, configura el teu IDE i crea el teu primer projecte Spring Boot per començar el teu viatge.
