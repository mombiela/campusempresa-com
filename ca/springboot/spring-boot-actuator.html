Spring Boot Actuator és un conjunt d'eines potent que proporciona característiques preparades per a la producció per ajudar-te a monitoritzar i gestionar la teva aplicació Spring Boot. Inclou una varietat de punts finals integrats que es poden utilitzar per recollir mètriques, entendre el trànsit o fins i tot canviar configuracions sense reiniciar l'aplicació.

Conceptes Clau

  • Punts Finals: Actuator proporciona diversos punts finals que exposen diferents aspectes de l'aplicació.
  • Mètriques: Recollir i exposar mètriques sobre l'aplicació.
  • Controls de Salut: Monitoritzar la salut de l'aplicació i les seves dependències.
  • Punts Finals Personalitzats: Crear els teus propis punts finals per exposar informació personalitzada.
  • Seguretat: Assegurar els punts finals de l'actuator per prevenir l'accés no autoritzat.

Començant amb Spring Boot Actuator

Afegint Actuator al Teu Projecte

Per començar a utilitzar Spring Boot Actuator, necessites afegir la dependència spring-boot-starter-actuator al teu fitxer pom.xml:

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

Activant Punts Finals

Per defecte, només uns quants punts finals estan activats. Pots activar més punts finals configurant el teu fitxer application.properties:

management.endpoints.web.exposure.include=*

Aquesta configuració exposarà tots els punts finals disponibles.

Punts Finals Comuns d'Actuator

Aquí tens alguns dels punts finals d'Actuator més utilitzats:

  • /actuator/health: Proporciona informació sobre la salut de l'aplicació.
  • /actuator/info: Mostra informació arbitrària de l'aplicació.
  • /actuator/metrics: Mostra informació de mètriques.
  • /actuator/env: Exposa propietats de l'Environment de Spring.

Exemple: Accedint al Punt Final de Salut

Un cop hagis afegit la dependència d'Actuator i activat els punts finals, pots accedir al punt final de salut navegant a http://localhost:8080/actuator/health al teu navegador o utilitzant una eina com curl:

curl http://localhost:8080/actuator/health

La resposta serà semblant a aquesta:

{
    "status": "UP"
}

Personalitzant Punts Finals d'Actuator

Indicadors de Salut Personalitzats

Pots crear indicadors de salut personalitzats per incloure controls de salut addicionals. Aquí tens un exemple d'un indicador de salut personalitzat:

import org.springframework.boot.actuate.health.Health;
import org.springframework.boot.actuate.health.HealthIndicator;
import org.springframework.stereotype.Component;

@Component
public class CustomHealthIndicator implements HealthIndicator {

    @Override
    public Health health() {
        // Lògica de control de salut personalitzada
        boolean isHealthy = checkCustomHealth();
        if (isHealthy) {
            return Health.up().withDetail("CustomService", "Available").build();
        } else {
            return Health.down().withDetail("CustomService", "Not Available").build();
        }
    }

    private boolean checkCustomHealth() {
        // Implementa la teva lògica de control de salut aquí
        return true;
    }
}

Mètriques Personalitzades

També pots crear mètriques personalitzades per monitoritzar aspectes específics de la teva aplicació. Aquí tens un exemple:

import io.micrometer.core.instrument.MeterRegistry;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Component;

import javax.annotation.PostConstruct;

@Component
public class CustomMetrics {

    @Autowired
    private MeterRegistry meterRegistry;

    @PostConstruct
    public void init() {
        meterRegistry.counter("custom.metric.counter").increment();
    }
}

Assegurant els Punts Finals d'Actuator

Per assegurar els teus punts finals d'Actuator, pots configurar Spring Security. Aquí tens un exemple de configuració:

import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.WebSecurityConfigurerAdapter;

@Configuration
public class SecurityConfig extends WebSecurityConfigurerAdapter {

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http
            .authorizeRequests()
            .antMatchers("/actuator/**").hasRole("ADMIN")
            .anyRequest().authenticated()
            .and()
            .httpBasic();
    }
}

En aquesta configuració, només els usuaris amb el rol ADMIN poden accedir als punts finals d'Actuator.

Conclusió

Spring Boot Actuator és una eina essencial per monitoritzar i gestionar les teves aplicacions Spring Boot. Proporciona una àmplia gamma de punts finals integrats per ajudar-te a obtenir informació sobre la salut, les mètriques i l'entorn de la teva aplicació. Personalitzant i assegurant aquests punts finals, pots adaptar l'Actuator a les teves necessitats específiques i assegurar-te que la teva aplicació es mantingui robusta i segura en entorns de producció.
