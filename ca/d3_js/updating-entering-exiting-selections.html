En aquesta secció, explorarem com gestionar documents impulsats per dades utilitzant D3.js mitjançant l'actualització, l'entrada i la sortida de seleccions. Aquests conceptes són fonamentals per crear visualitzacions dinàmiques que responen a canvis en les dades.

Conceptes Clau

  • Seleccions: D3.js utilitza seleccions per vincular dades a elements del DOM.
  • Selecció d'Entrada: Gestiona els punts de dades que són nous i necessiten ser afegits al DOM.
  • Selecció d'Actualització: Gestiona els punts de dades que ja existeixen en el DOM i necessiten ser actualitzats.
  • Selecció de Sortida: Gestiona els punts de dades que ja no existeixen en les dades i necessiten ser eliminats del DOM.

Seleccions en D3.js

Les seleccions en D3.js s'utilitzen per vincular dades a elements del DOM. Aquesta és la base per crear visualitzacions dinàmiques.

// Selecciona tots els cercles i vincula les dades a ells
const circles = d3.selectAll('circle').data(data);

Selecció d'Entrada

La selecció d'entrada s'utilitza per crear nous elements per als punts de dades entrants que encara no tenen elements corresponents en el DOM.

// Matriu de dades
const data = [30, 70, 110];

// Selecciona tots els cercles i vincula les dades
const circles = d3.select('svg').selectAll('circle').data(data);

// Selecció d'entrada: crea nous cercles per als nous punts de dades
circles.enter()
  .append('circle')
  .attr('cx', (d, i) => (i + 1) * 50)
  .attr('cy', 50)
  .attr('r', d => d / 2)
  .attr('fill', 'blue');

Explicació:

  • La matriu data conté els punts de dades.
  • selectAll('circle') selecciona tots els elements cercle.
  • data(data) vincula les dades a la selecció.
  • enter() crea una selecció per als punts de dades que no tenen elements corresponents en el DOM.
  • append('circle') crea nous elements cercle per a cada nou punt de dades.

Selecció d'Actualització

La selecció d'actualització s'utilitza per modificar els elements existents del DOM per reflectir els canvis en les dades.

// Matriu de dades
const data = [30, 70, 110];

// Selecciona tots els cercles i vincula les dades
const circles = d3.select('svg').selectAll('circle').data(data);

// Selecció d'actualització: actualitza els cercles existents
circles
  .attr('r', d => d / 2)
  .attr('fill', 'green');

Explicació:

  • La selecció circles s'actualitza amb nous atributs.
  • attr('r', d => d / 2) actualitza el radi de cada cercle basat en les dades.
  • attr('fill', 'green') canvia el color de farciment dels cercles.

Selecció de Sortida

La selecció de sortida s'utilitza per eliminar els elements del DOM que ja no tenen punts de dades corresponents.

// Matriu de dades
const data = [30, 70];

// Selecciona tots els cercles i vincula les dades
const circles = d3.select('svg').selectAll('circle').data(data);

// Selecció de sortida: elimina els cercles que ja no tenen dades
circles.exit().remove();

Explicació:

  • La selecció circles està vinculada a la nova matriu de dades.
  • exit() crea una selecció per als elements del DOM que ja no tenen punts de dades corresponents.
  • remove() elimina aquests elements del DOM.

Exemple Combinat

Combinem les seleccions d'entrada, actualització i sortida en un sol exemple.

// Matriu de dades inicial
let data = [30, 70, 110];

// Funció per actualitzar la visualització
function update(data) {
  // Selecciona tots els cercles i vincula les dades
  const circles = d3.select('svg').selectAll('circle').data(data);

  // Selecció d'entrada: crea nous cercles per als nous punts de dades
  circles.enter()
    .append('circle')
    .attr('cx', (d, i) => (i + 1) * 50)
    .attr('cy', 50)
    .attr('r', d => d / 2)
    .attr('fill', 'blue');

  // Selecció d'actualització: actualitza els cercles existents
  circles
    .attr('r', d => d / 2)
    .attr('fill', 'green');

  // Selecció de sortida: elimina els cercles que ja no tenen dades
  circles.exit().remove();
}

// Renderització inicial
update(data);

// Actualitza les dades i torna a renderitzar després de 2 segons
setTimeout(() => {
  data = [50, 90];
  update(data);
}, 2000);

Explicació:

  • La funció update gestiona les seleccions d'entrada, actualització i sortida.
  • Inicialment, la matriu data conté tres punts de dades.
  • Després de 2 segons, la matriu data s'actualitza per contenir dos punts de dades, i la funció update es crida de nou per reflectir aquests canvis en la visualització.

Conclusió

Entendre com gestionar les seleccions en D3.js és crucial per crear visualitzacions dinàmiques i interactives. Dominant les seleccions d'entrada, actualització i sortida, pots assegurar-te que les teves visualitzacions reflecteixin amb precisió els canvis en les dades subjacents. Practica aquests conceptes amb diferents conjunts de dades per convertir-te en un expert en la manipulació de documents impulsats per dades utilitzant D3.js.
