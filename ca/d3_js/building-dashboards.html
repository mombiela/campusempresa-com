Construcció de Quadres de Comandament amb D3.js

Introducció

Construir quadres de comandament interactius és una manera poderosa de visualitzar dades i obtenir informació. D3.js (Documents Basats en Dades) és una biblioteca de JavaScript que et permet crear visualitzacions de dades dinàmiques i interactives en navegadors web. Aquest tema et guiarà a través del procés de construcció de quadres de comandament utilitzant D3.js, començant pels conceptes bàsics i progressant cap a tècniques avançades.

Començant amb D3.js

Configuració del Teu Entorn

  • Instal·la D3.js via CDN o npm.
  • Crea una plantilla HTML bàsica per allotjar el teu quadre de comandament.
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Quadre de Comandament D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
    <style>
        /* Estil bàsic per al quadre de comandament */
        .chart {
            display: inline-block;
            margin: 10px;
        }
    </style>
</head>
<body>
    <div id="dashboard"></div>
    <script src="app.js"></script>
</body>
</html>

Conceptes Bàsics de D3.js

  • Seleccions: Selecciona elements del DOM i vincula dades a ells.
  • Vinculació de Dades: Utilitza .data() per vincular dades als elements.
  • Entrar, Actualitzar, Sortir: Gestiona els canvis basats en dades al DOM.
// app.js
const data = [10, 20, 30, 40, 50];

d3.select("#dashboard")
  .selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "steelblue")
  .style("margin", "5px");

Creació de Gràfics Bàsics

Gràfic de Barres

  • Preparació de Dades: Formata les teves dades.
  • Escales: Mapa els valors de les dades a dimensions visuals.
  • Eixos: Crea i estilitza els eixos.
const svg = d3.select("#dashboard")
              .append("svg")
              .attr("width", 500)
              .attr("height", 300);

const xScale = d3.scaleBand()
                 .domain(data.map((d, i) => i))
                 .range([0, 500])
                 .padding(0.1);

const yScale = d3.scaleLinear()
                 .domain([0, d3.max(data)])
                 .range([300, 0]);

svg.selectAll(".bar")
   .data(data)
   .enter()
   .append("rect")
   .attr("class", "bar")
   .attr("x", (d, i) => xScale(i))
   .attr("y", d => yScale(d))
   .attr("width", xScale.bandwidth())
   .attr("height", d => 300 - yScale(d))
   .attr("fill", "steelblue");

Gràfic de Línies

  • Generador de Línies: Utilitza d3.line() per crear un camí.
  • Vinculació de Dades: Vincula dades a l'element camí.
const lineData = [
  { x: 0, y: 30 },
  { x: 1, y: 50 },
  { x: 2, y: 80 },
  { x: 3, y: 40 },
  { x: 4, y: 60 }
];

const line = d3.line()
               .x(d => xScale(d.x))
               .y(d => yScale(d.y));

svg.append("path")
   .datum(lineData)
   .attr("fill", "none")
   .attr("stroke", "steelblue")
   .attr("stroke-width", 2)
   .attr("d", line);

Tècniques Intermèdies

Afegir Interactivitat

  • Esdeveniments: Afegeix esdeveniments de ratolí als elements.
  • Transicions: Anima els canvis en les dades.
svg.selectAll(".bar")
   .on("mouseover", function(event, d) {
       d3.select(this).attr("fill", "orange");
   })
   .on("mouseout", function(event, d) {
       d3.select(this).attr("fill", "steelblue");
   });

svg.selectAll(".bar")
   .transition()
   .duration(1000)
   .attr("height", d => 300 - yScale(d))
   .attr("y", d => yScale(d));

Disseny Responsiu

  • Vista SVG: Fes que els teus gràfics siguin responsius.
  • Escalat Dinàmic: Ajusta les escales segons la mida de la finestra.
const svg = d3.select("#dashboard")
              .append("svg")
              .attr("viewBox", "0 0 500 300")
              .attr("preserveAspectRatio", "xMidYMid meet");

Tècniques Avançades

Integració amb Altres Biblioteques

  • Crossfilter: Gestiona grans conjunts de dades.
  • DC.js: Simplifica la creació de gràfics.
// Exemple d'integració amb Crossfilter
const cf = crossfilter(data);
const dimension = cf.dimension(d => d.value);
const group = dimension.group().reduceCount();

dc.barChart("#chart")
  .dimension(dimension)
  .group(group)
  .x(d3.scaleLinear().domain([0, 100]));

Visualitzacions Personalitzades

  • Formes Personalitzades: Crea elements visuals únics.
  • Interaccions Complexes: Implementa arrossegar i deixar anar, zoom i panoràmica.
const drag = d3.drag()
               .on("start", function(event, d) {
                   d3.select(this).raise().attr("stroke", "black");
               })
               .on("drag", function(event, d) {
                   d3.select(this).attr("cx", d.x = event.x).attr("cy", d.y = event.y);
               })
               .on("end", function(event, d) {
                   d3.select(this).attr("stroke", null);
               });

svg.selectAll("circle")
   .data(data)
   .enter()
   .append("circle")
   .attr("cx", d => xScale(d.x))
   .attr("cy", d => yScale(d.y))
   .attr("r", 5)
   .call(drag);

Conclusió

Construir quadres de comandament amb D3.js implica entendre els conceptes bàsics de la vinculació de dades, crear diversos tipus de gràfics i afegir interactivitat i capacitat de resposta. A mesura que progressis cap a tècniques més avançades, podràs integrar altres biblioteques i crear visualitzacions personalitzades per satisfer necessitats específiques. La pràctica i l'experimentació són clau per dominar D3.js i crear quadres de comandament efectius i interactius.
