Introducció a l'Explicació de Dades Basada en Dades

L'explicació de dades basada en dades és la pràctica d'utilitzar dades per explicar històries convincents. Implica l'ús de tècniques de visualització de dades per presentar les dades d'una manera que sigui tant informativa com atractiva.

  • Importància de l'Explicació de Dades Basada en Dades:
    • Fa que les dades complexes siguin comprensibles.
    • Enganxa l'audiència.
    • Suporta la presa de decisions basada en dades.

Començant amb D3.js

D3.js (Documents Basats en Dades) és una biblioteca de JavaScript per crear visualitzacions de dades dinàmiques i interactives en navegadors web.

  • Conceptes Clau:
    • Seleccions: Seleccionar elements del DOM.
    • Enllaç de Dades: Enllaçar dades als elements del DOM.
    • Entrar, Actualitzar, Sortir: Gestionar els canvis basats en dades al DOM.

Exemple Bàsic: Creant un Gràfic de Barres Simple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Gràfic de Barres Simple</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
    </style>
</head>
<body>
    <script>
        const data = [4, 8, 15, 16, 23, 42];

        const width = 420;
        const barHeight = 20;

        const x = d3.scaleLinear()
            .domain([0, d3.max(data)])
            .range([0, width]);

        const chart = d3.select("body")
            .append("svg")
            .attr("width", width)
            .attr("height", barHeight * data.length);

        const bar = chart.selectAll("g")
            .data(data)
            .enter().append("g")
            .attr("transform", (d, i) => `translate(0,${i * barHeight})`);

        bar.append("rect")
            .attr("width", x)
            .attr("height", barHeight - 1)
            .attr("class", "bar");

        bar.append("text")
            .attr("x", d => x(d) - 3)
            .attr("y", barHeight / 2)
            .attr("dy", ".35em")
            .text(d => d);
    </script>
</body>
</html>

Tècniques Intermèdies

A mesura que et familiaritzis més amb D3.js, pots començar a explorar visualitzacions i interaccions més complexes.

  • Escales i Eixos:

    • Les escales mapegen valors de dades a valors visuals.
    • Els eixos proporcionen marques de referència per a les escales.

  • Transicions:

    • Animar els canvis al DOM.

Exemple: Afegint Transicions al Gràfic de Barres

bar.select("rect")
    .transition()
    .duration(750)
    .attr("width", x);

Tècniques Avançades

Les tècniques avançades de D3.js impliquen crear visualitzacions altament interactives i complexes.

  • Interactivitat:

    • Afegint tooltips.
    • Gestionant esdeveniments d'usuari (clic, passar el ratolí).

  • Dissenys Complexos:

    • Gràfics dirigits per forces.
    • Dissenys jeràrquics (per exemple, mapes d'arbres, gràfics de sol).

Exemple: Afegint Tooltips

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Gràfic de Barres amb Tooltips</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
        .tooltip {
            position: absolute;
            text-align: center;
            width: 60px;
            height: 28px;
            padding: 2px;
            font: 12px sans-serif;
            background: lightsteelblue;
            border: 0px;
            border-radius: 8px;
            pointer-events: none;
        }
    </style>
</head>
<body>
    <script>
        const data = [4, 8, 15, 16, 23, 42];

        const width = 420;
        const barHeight = 20;

        const x = d3.scaleLinear()
            .domain([0, d3.max(data)])
            .range([0, width]);

        const chart = d3.select("body")
            .append("svg")
            .attr("width", width)
            .attr("height", barHeight * data.length);

        const bar = chart.selectAll("g")
            .data(data)
            .enter().append("g")
            .attr("transform", (d, i) => `translate(0,${i * barHeight})`);

        bar.append("rect")
            .attr("width", x)
            .attr("height", barHeight - 1)
            .attr("class", "bar");

        bar.append("text")
            .attr("x", d => x(d) - 3)
            .attr("y", barHeight / 2)
            .attr("dy", ".35em")
            .text(d => d);

        const tooltip = d3.select("body").append("div")
            .attr("class", "tooltip")
            .style("opacity", 0);

        bar.on("mouseover", function(event, d) {
            tooltip.transition()
                .duration(200)
                .style("opacity", .9);
            tooltip.html(d)
                .style("left", (event.pageX) + "px")
                .style("top", (event.pageY - 28) + "px");
        })
        .on("mouseout", function(d) {
            tooltip.transition()
                .duration(500)
                .style("opacity", 0);
        });
    </script>
</body>
</html>

Conclusió

L'explicació de dades basada en dades amb D3.js et permet crear visualitzacions potents que poden comunicar dades complexes d'una manera atractiva i comprensible. Dominant els conceptes bàsics, les tècniques intermèdies i les característiques avançades de D3.js, pots construir visualitzacions interactives i dinàmiques que expliquin històries convincents amb les teves dades.

  • Resum:
    • Comença amb visualitzacions bàsiques.
    • Aprèn a utilitzar escales, eixos i transicions.
    • Afegeix interactivitat i explora dissenys complexos.
    • Practica i experimenta per millorar les teves habilitats.
