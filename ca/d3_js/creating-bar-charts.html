Introducció

En aquesta secció, explorarem com crear gràfics de barres utilitzant D3.js, una potent biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Els gràfics de barres són un tipus fonamental de gràfic que representa dades amb barres rectangulars, on la longitud de cada barra és proporcional al valor que representa.

Configuració del Teu Entorn

Abans d'endinsar-nos en la creació de gràfics de barres, assegura't de tenir el següent:

  • Una comprensió bàsica d'HTML, CSS i JavaScript.
  • La biblioteca D3.js inclosa en el teu projecte.

Per incloure D3.js en el teu projecte, pots utilitzar un CDN:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Gràfic de Barres amb D3.js</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
</head>
<body>
    <!-- El teu contingut anirà aquí -->
</body>
</html>

Creació d'un Gràfic de Barres Simple

Pas 1: Preparar les Dades

Primer, necessitem algunes dades per visualitzar. Per simplicitat, utilitzem una matriu de números:

const data = [30, 86, 168, 281, 303, 365];

Pas 2: Configurar el Contenidor SVG

Crearem un contenidor SVG on es dibuixarà el nostre gràfic de barres:

<body>
    <svg width="500" height="300"></svg>
</body>

Pas 3: Crear les Barres

Ara, utilitzem D3.js per crear les barres:

const svg = d3.select("svg");
const width = 500;
const height = 300;
const barPadding = 5;
const barWidth = (width / data.length);

svg.selectAll("rect")
    .data(data)
    .enter()
    .append("rect")
    .attr("x", (d, i) => i * barWidth)
    .attr("y", d => height - d)
    .attr("width", barWidth - barPadding)
    .attr("height", d => d)
    .attr("fill", "teal");

Explicació

  • d3.select("svg"): Selecciona l'element SVG.
  • svg.selectAll("rect"): Selecciona tots els elements rect (encara no n'hi ha cap).
  • .data(data): Vincula les dades a la selecció.
  • .enter(): Crea un marcador de posició per a cada punt de dades.
  • .append("rect"): Afegeix un element rect per a cada punt de dades.
  • .attr("x", (d, i) => i * barWidth): Estableix la posició x de cada barra.
  • .attr("y", d => height - d): Estableix la posició y de cada barra.
  • .attr("width", barWidth - barPadding): Estableix l'amplada de cada barra.
  • .attr("height", d => d): Estableix l'alçada de cada barra.
  • .attr("fill", "teal"): Estableix el color de cada barra.

Afegir Etiquetes

Per fer el gràfic més informatiu, afegim etiquetes a cada barra:

svg.selectAll("text")
    .data(data)
    .enter()
    .append("text")
    .text(d => d)
    .attr("x", (d, i) => i * barWidth + (barWidth - barPadding) / 2)
    .attr("y", d => height - d - 5)
    .attr("text-anchor", "middle")
    .attr("fill", "white");

Explicació

  • .append("text"): Afegeix un element text per a cada punt de dades.
  • .text(d => d): Estableix el contingut del text al valor de les dades.
  • .attr("x", (d, i) => i * barWidth + (barWidth - barPadding) / 2): Centra el text horitzontalment dins de cada barra.
  • .attr("y", d => height - d - 5): Col·loca el text lleugerament per sobre de cada barra.
  • .attr("text-anchor", "middle"): Centra el text.
  • .attr("fill", "white"): Estableix el color del text a blanc.

Personalitzacions Avançades

Afegir Eixos

Per afegir eixos, necessitem utilitzar les funcions d'escala i eix de D3:

const xScale = d3.scaleBand()
    .domain(data.map((d, i) => i))
    .range([0, width])
    .padding(0.1);

const yScale = d3.scaleLinear()
    .domain([0, d3.max(data)])
    .range([height, 0]);

const xAxis = d3.axisBottom(xScale);
const yAxis = d3.axisLeft(yScale);

svg.append("g")
    .attr("transform", `translate(0, ${height})`)
    .call(xAxis);

svg.append("g")
    .call(yAxis);

Explicació

  • d3.scaleBand(): Crea una escala de banda per a l'eix x.
  • d3.scaleLinear(): Crea una escala lineal per a l'eix y.
  • d3.axisBottom(xScale): Crea un eix inferior utilitzant l'escala xScale.
  • d3.axisLeft(yScale): Crea un eix esquerre utilitzant l'escala yScale.
  • .attr("transform", translate(0, ${height})): Col·loca l'eix x a la part inferior del SVG.
  • .call(xAxis): Renderitza l'eix x.
  • .call(yAxis): Renderitza l'eix y.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la creació de gràfics de barres utilitzant D3.js. Hem començat configurant l'entorn, preparant les dades i creant un gràfic de barres simple. Després hem afegit etiquetes i hem explorat personalitzacions avançades com afegir eixos. Amb aquestes habilitats, pots crear gràfics de barres més complexos i interactius per visualitzar les teves dades de manera efectiva.
