◄ Què és D3.js?
Conceptes bàsics i terminologia ►

Introducció

Abans d'endinsar-te en D3.js, és essencial configurar correctament el teu entorn de desenvolupament. Això assegura que tens totes les eines i biblioteques necessàries per crear i visualitzar dades de manera efectiva. Aquesta secció et guiarà a través de la instal·lació del programari necessari i la configuració d'una estructura bàsica de projecte.

Prerequisits

Abans de configurar D3.js, assegura't de tenir el següent:

  • Un navegador web modern (per exemple, Chrome, Firefox, Safari)
  • Un editor de codi (per exemple, Visual Studio Code, Sublime Text, Atom)
  • Coneixements bàsics d'HTML, CSS i JavaScript

Instal·lació de Node.js i npm

Node.js i npm (Node Package Manager) són eines essencials per gestionar biblioteques i dependències de JavaScript.

  1. Descarregar i instal·lar Node.js:

    • Visita el lloc web de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Verificar la instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa les següents comandes per comprovar les versions instal·lades: 
      node -v
npm -v
    • Hauries de veure els números de versió tant de Node.js com de npm.

Configuració d'un nou projecte

  1. Crear un directori de projecte:

    • Obre el teu terminal o símbol del sistema.
    • Navega fins al directori on vols crear el teu projecte.
    • Executa la següent comanda per crear un nou directori: 
      mkdir d3-project
cd d3-project

  2. Inicialitzar un nou projecte npm:

    • Executa la següent comanda per inicialitzar un nou projecte npm: 
      npm init -y
    • Això crearà un fitxer package.json amb la configuració per defecte.

Instal·lació de D3.js

  1. Instal·lar D3.js via npm:

    • Executa la següent comanda per instal·lar D3.js: 
      npm install d3

  2. Verificar la instal·lació:

    • Comprova el directori node_modules per assegurar-te que D3.js s'ha instal·lat.
    • El fitxer package.json també hauria de llistar D3.js com a dependència.

Configuració d'una plantilla HTML bàsica

Crea un fitxer HTML bàsic per començar a utilitzar D3.js.

  1. Crear un fitxer index.html:

    • Al teu directori de projecte, crea un fitxer anomenat index.html.
    • Afegeix la següent estructura HTML bàsica: 
      <!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Projecte D3.js</title>
</head>
<body>
    <h1>Benvingut a D3.js</h1>
    <script src="node_modules/d3/dist/d3.min.js"></script>
    <script src="app.js"></script>
</body>
</html>

  2. Crear un fitxer app.js:

    • Al teu directori de projecte, crea un fitxer anomenat app.js.
    • Afegeix el següent codi per verificar que D3.js funciona: 
      // Selecciona l'element body i afegeix un paràgraf
d3.select("body").append("p").text("D3.js està funcionant!");

  3. Obrir index.html en un navegador:

    • Obre el fitxer index.html al teu navegador web.
    • Hauries de veure el missatge "D3.js està funcionant!" mostrat a la pàgina.

Ús d'un servidor de desenvolupament local

Per a una experiència de desenvolupament més fluida, utilitza un servidor de desenvolupament local per servir els teus fitxers.

  1. Instal·lar live-server:

    • Executa la següent comanda per instal·lar live-server globalment: 
      npm install -g live-server

  2. Iniciar el servidor:

    • Al teu directori de projecte, executa la següent comanda: 
      live-server
    • Això iniciarà un servidor local i obrirà el teu projecte al navegador web per defecte. Qualsevol canvi que facis als teus fitxers refrescarà automàticament el navegador.

Conclusió

Configurar el teu entorn és el primer pas per dominar D3.js. Assegurant-te que tens Node.js, npm i un servidor de desenvolupament local, pots gestionar eficientment les dependències del teu projecte i agilitzar el teu procés de desenvolupament. Amb el teu entorn a punt, ara pots centrar-te en aprendre i crear impressionants visualitzacions de dades amb D3.js.
