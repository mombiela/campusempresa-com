En aquesta secció, explorarem com crear formes bàsiques utilitzant D3.js. Aquesta és una habilitat fonamental per a qualsevol persona que vulgui crear visualitzacions de dades, ja que les formes constitueixen els blocs de construcció de gràfics més complexos.

Introducció a D3.js

D3.js (Documents Basats en Dades) és una biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Utilitza estàndards HTML, SVG i CSS.

Conceptes Clau

  • SVG (Gràfics Vectorials Escalables): Un llenguatge per descriure gràfics 2D en XML.
  • Seleccions D3: Mètodes per seleccionar i manipular elements del DOM.
  • Associació de Dades: Associar dades amb elements del DOM.

Configurant el Teu Entorn

Abans de començar a crear formes, assegura't de tenir un fitxer HTML bàsic configurat i inclou la biblioteca D3.js.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>D3.js Formes Bàsiques</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <svg width="500" height="500"></svg>
    <script>
        // El teu codi D3.js anirà aquí
    </script>
</body>
</html>

Creant un Cercle

Per crear un cercle en D3.js, utilitzaràs l'element SVG circle.

Exemple

const svg = d3.select("svg");

svg.append("circle")
    .attr("cx", 250) // coordenada x del centre
    .attr("cy", 250) // coordenada y del centre
    .attr("r", 50)   // radi
    .attr("fill", "blue"); // color de farciment

Explicació

  • d3.select("svg"): Selecciona l'element SVG.
  • append("circle"): Afegeix un element cercle a l'SVG.
  • attr("cx", 250): Estableix la coordenada x del centre del cercle a 250.
  • attr("cy", 250): Estableix la coordenada y del centre del cercle a 250.
  • attr("r", 50): Estableix el radi del cercle a 50.
  • attr("fill", "blue"): Estableix el color de farciment del cercle a blau.

Creant un Rectangle

Per crear un rectangle, utilitza l'element SVG rect.

Exemple

svg.append("rect")
    .attr("x", 100) // coordenada x de la cantonada superior esquerra
    .attr("y", 100) // coordenada y de la cantonada superior esquerra
    .attr("width", 200) // amplada del rectangle
    .attr("height", 100) // alçada del rectangle
    .attr("fill", "green"); // color de farciment

Explicació

  • attr("x", 100): Estableix la coordenada x de la cantonada superior esquerra del rectangle a 100.
  • attr("y", 100): Estableix la coordenada y de la cantonada superior esquerra del rectangle a 100.
  • attr("width", 200): Estableix l'amplada del rectangle a 200.
  • attr("height", 100): Estableix l'alçada del rectangle a 100.
  • attr("fill", "green"): Estableix el color de farciment del rectangle a verd.

Creant una Línia

Per crear una línia, utilitza l'element SVG line.

Exemple

svg.append("line")
    .attr("x1", 50) // coordenada x del punt d'inici
    .attr("y1", 50) // coordenada y del punt d'inici
    .attr("x2", 450) // coordenada x del punt final
    .attr("y2", 450) // coordenada y del punt final
    .attr("stroke", "red") // color del traç
    .attr("stroke-width", 2); // amplada del traç

Explicació

  • attr("x1", 50): Estableix la coordenada x del punt d'inici de la línia a 50.
  • attr("y1", 50): Estableix la coordenada y del punt d'inici de la línia a 50.
  • attr("x2", 450): Estableix la coordenada x del punt final de la línia a 450.
  • attr("y2", 450): Estableix la coordenada y del punt final de la línia a 450.
  • attr("stroke", "red"): Estableix el color del traç de la línia a vermell.
  • attr("stroke-width", 2): Estableix l'amplada del traç de la línia a 2.

Creant una El·lipse

Per crear una el·lipse, utilitza l'element SVG ellipse.

Exemple

svg.append("ellipse")
    .attr("cx", 250) // coordenada x del centre
    .attr("cy", 250) // coordenada y del centre
    .attr("rx", 100) // radi x
    .attr("ry", 50) // radi y
    .attr("fill", "purple"); // color de farciment

Explicació

  • attr("cx", 250): Estableix la coordenada x del centre de l'el·lipse a 250.
  • attr("cy", 250): Estableix la coordenada y del centre de l'el·lipse a 250.
  • attr("rx", 100): Estableix el radi x de l'el·lipse a 100.
  • attr("ry", 50): Estableix el radi y de l'el·lipse a 50.
  • attr("fill", "purple"): Estableix el color de farciment de l'el·lipse a porpra.

Conclusió

En aquesta secció, has après a crear formes bàsiques com cercles, rectangles, línies i el·lipses utilitzant D3.js. Aquestes formes són la base per construir visualitzacions més complexes. Practica creant aquestes formes i experimenta amb diferents atributs per obtenir una millor comprensió de com D3.js treballa amb elements SVG.
