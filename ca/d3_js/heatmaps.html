Els mapes de calor són una eina de visualització potent utilitzada per representar dades on els valors individuals es representen com a colors. En aquesta secció, explorarem com crear mapes de calor utilitzant D3.js, des de conceptes bàsics fins a tècniques avançades.

Introducció als Mapes de Calor

  • Definició: Un mapa de calor és una representació gràfica de dades on els valors individuals es representen amb colors.
  • Usos: Comunament utilitzats en l'anàlisi de dades per visualitzar la magnitud d'un fenomen, com ara el trànsit web, dades del mercat de valors o patrons meteorològics.

Configuració del Teu Entorn

Abans d'endinsar-nos en la creació de mapes de calor, assegura't de tenir el següent:

  • Biblioteca D3.js: Inclou D3.js en el teu projecte afegint la següent etiqueta de script al teu fitxer HTML: 
    <script src="https://d3js.org/d3.v6.min.js"></script>
  • Estructura HTML Bàsica: Crea una estructura HTML bàsica per contenir el teu mapa de calor. 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mapa de Calor amb D3.js</title>
</head>
<body>
    <div id="heatmap"></div>
    <script src="script.js"></script>
</body>
</html>

Creació d'un Mapa de Calor Bàsic

  1. Preparar les Dades: Els mapes de calor requereixen dades en format de matriu. Aquí tens un exemple de conjunt de dades:

    const data = [
    [30, 20, 10],
    [60, 50, 40],
    [90, 80, 70]
];

  2. Configurar el Contenidor SVG: Crea un contenidor SVG per contenir el mapa de calor.

    const width = 300;
const height = 300;
const svg = d3.select("#heatmap")
              .append("svg")
              .attr("width", width)
              .attr("height", height);

  3. Definir l'Escala de Colors: Utilitza una escala de colors per mapar els valors de les dades a colors.

    const colorScale = d3.scaleSequential(d3.interpolateBlues)
                     .domain([0, 100]);

  4. Renderitzar el Mapa de Calor: Utilitza D3.js per renderitzar el mapa de calor.

    const cellSize = 100; // Mida de cada cel·la

data.forEach((row, i) => {
    row.forEach((value, j) => {
        svg.append("rect")
           .attr("x", j * cellSize)
           .attr("y", i * cellSize)
           .attr("width", cellSize)
           .attr("height", cellSize)
           .attr("fill", colorScale(value));
    });
});

Millorant el Mapa de Calor

  • Afegint Etiquetes: Afegeix etiquetes a les cel·les del mapa de calor per a una millor llegibilitat.

    data.forEach((row, i) => {
    row.forEach((value, j) => {
        svg.append("text")
           .attr("x", j * cellSize + cellSize / 2)
           .attr("y", i * cellSize + cellSize / 2)
           .attr("dy", ".35em")
           .attr("text-anchor", "middle")
           .text(value)
           .attr("fill", "black");
    });
});

  • Característiques Interactives: Afegeix interactivitat com ara tooltips per mostrar els valors de les dades en passar el ratolí per sobre.

    const tooltip = d3.select("body").append("div")
                  .style("position", "absolute")
                  .style("background", "#f9f9f9")
                  .style("padding", "5px")
                  .style("border", "1px solid #d3d3d3")
                  .style("border-radius", "5px")
                  .style("visibility", "hidden");

svg.selectAll("rect")
   .on("mouseover", function(event, d) {
       tooltip.style("visibility", "visible")
              .text(d3.select(this).data());
   })
   .on("mousemove", function(event) {
       tooltip.style("top", (event.pageY - 10) + "px")
              .style("left", (event.pageX + 10) + "px");
   })
   .on("mouseout", function() {
       tooltip.style("visibility", "hidden");
   });

Tècniques Avançades

  • Càrrega Dinàmica de Dades: Carrega dades dinàmicament des de fonts externes com fitxers CSV o JSON.

    d3.csv("data.csv").then(data => {
    // Processa i renderitza el mapa de calor utilitzant les dades carregades
});

  • Escales de Colors Personalitzades: Crea escales de colors personalitzades per tenir més control sobre l'aparença del mapa de calor.

    const customColorScale = d3.scaleLinear()
                           .domain([0, 50, 100])
                           .range(["green", "yellow", "red"]);

  • Mapes de Calor Responsius: Fes que el mapa de calor sigui responsiu a diferents mides de pantalla.

    const svg = d3.select("#heatmap")
              .append("svg")
              .attr("viewBox", `0 0 ${width} ${height}`)
              .attr("preserveAspectRatio", "xMidYMid meet");

Conclusió

Els mapes de calor són una manera versàtil i visualment atractiva de representar dades. Utilitzant D3.js, pots crear mapes de calor dinàmics i interactius que proporcionen informació valuosa sobre les teves dades. Des de configurar un mapa de calor bàsic fins a implementar funcions avançades, aquesta guia ha cobert els passos essencials per ajudar-te a començar amb els mapes de calor en D3.js. Experimenta amb diferents conjunts de dades i opcions de personalització per crear mapes de calor que s'adaptin millor a les teves necessitats.
