Introducció a les Disposicions de Força

Les disposicions de força són una característica poderosa en D3.js que et permeten crear visualitzacions dinàmiques i interactives de dades de xarxa. Utilitzen simulació física per posicionar nodes i enllaços d'una manera que revela l'estructura de les dades.

Conceptes Clau

  • Nodes: Representen entitats en la xarxa.
  • Enllaços: Representen relacions entre nodes.
  • Forces: Simulen forces físiques per posicionar nodes i enllaços.

Configurant una Disposició de Força Bàsica

Per començar amb les disposicions de força en D3.js, necessites configurar el teu entorn HTML i JavaScript.

Configuració HTML

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Disposició de Força</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .node {
            stroke: #fff;
            stroke-width: 1.5px;
        }
        .link {
            stroke: #999;
            stroke-opacity: 0.6;
        }
    </style>
</head>
<body>
    <svg width="960" height="600"></svg>
    <script src="force-layout.js"></script>
</body>
</html>

Configuració JavaScript

const svg = d3.select("svg"),
    width = +svg.attr("width"),
    height = +svg.attr("height");

const nodes = [
    { id: 1 }, { id: 2 }, { id: 3 }, { id: 4 }
];

const links = [
    { source: 1, target: 2 },
    { source: 2, target: 3 },
    { source: 3, target: 4 },
    { source: 4, target: 1 }
];

const simulation = d3.forceSimulation(nodes)
    .force("link", d3.forceLink(links).id(d => d.id))
    .force("charge", d3.forceManyBody())
    .force("center", d3.forceCenter(width / 2, height / 2));

const link = svg.append("g")
    .attr("class", "links")
    .selectAll("line")
    .data(links)
    .enter().append("line")
    .attr("class", "link");

const node = svg.append("g")
    .attr("class", "nodes")
    .selectAll("circle")
    .data(nodes)
    .enter().append("circle")
    .attr("class", "node")
    .attr("r", 5)
    .call(d3.drag()
        .on("start", dragstarted)
        .on("drag", dragged)
        .on("end", dragended));

simulation
    .nodes(nodes)
    .on("tick", ticked);

simulation.force("link")
    .links(links);

function ticked() {
    link
        .attr("x1", d => d.source.x)
        .attr("y1", d => d.source.y)
        .attr("x2", d => d.target.x)
        .attr("y2", d => d.target.y);

    node
        .attr("cx", d => d.x)
        .attr("cy", d => d.y);
}

function dragstarted(event, d) {
    if (!event.active) simulation.alphaTarget(0.3).restart();
    d.fx = d.x;
    d.fy = d.y;
}

function dragged(event, d) {
    d.fx = event.x;
    d.fy = event.y;
}

function dragended(event, d) {
    if (!event.active) simulation.alphaTarget(0);
    d.fx = null;
    d.fy = null;
}

Personalitzant la Disposició de Força

Pots personalitzar el comportament de la disposició de força ajustant les forces aplicades als nodes i enllaços.

Ajustant Forces

  • Força d'Enllaç: Controla la distància entre nodes enllaçats.
  • Força de Càrrega: Controla la repulsió o atracció entre nodes.
  • Força de Centrament: Manté el gràfic centrat en l'SVG.
simulation
    .force("link", d3.forceLink(links).id(d => d.id).distance(100))
    .force("charge", d3.forceManyBody().strength(-50))
    .force("center", d3.forceCenter(width / 2, height / 2));

Afegint Etiquetes

Pots afegir etiquetes als nodes per fer la visualització més informativa.

const labels = svg.append("g")
    .attr("class", "labels")
    .selectAll("text")
    .data(nodes)
    .enter().append("text")
    .attr("class", "label")
    .text(d => d.id);

function ticked() {
    link
        .attr("x1", d => d.source.x)
        .attr("y1", d => d.source.y)
        .attr("x2", d => d.target.x)
        .attr("y2", d => d.target.y);

    node
        .attr("cx", d => d.x)
        .attr("cy", d => d.y);

    labels
        .attr("x", d => d.x)
        .attr("y", d => d.y);
}

Personalitzacions Avançades

Per a personalitzacions més avançades, pots crear forces personalitzades o utilitzar mòduls addicionals de D3.js.

Forces Personalitzades

Pots crear forces personalitzades per aconseguir comportaments específics en la teva disposició de força.

function customForce(alpha) {
    nodes.forEach(node => {
        node.vx += Math.random() * 2 - 1;
        node.vy += Math.random() * 2 - 1;
    });
}

simulation.force("custom", customForce);

Utilitzant Mòduls Addicionals

D3.js proporciona mòduls addicionals que es poden utilitzar per millorar la teva disposició de força.

import { forceCollide } from "d3-force";

simulation.force("collide", forceCollide(10));

Conclusió

Les disposicions de força en D3.js ofereixen una manera flexible i poderosa de visualitzar dades de xarxa. Entenent la configuració bàsica i les opcions de personalització, pots crear visualitzacions dinàmiques i informatives. Experimenta amb diferents forces i personalitzacions per veure com afecten la disposició i descobreix la millor manera de representar les teves dades.
