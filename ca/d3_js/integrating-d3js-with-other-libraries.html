En aquesta secció, explorarem com integrar D3.js amb altres llibreries i frameworks populars de JavaScript. Aquesta integració et pot ajudar a aprofitar els punts forts de múltiples eines per crear visualitzacions de dades més potents i eficients.

Per què Integrar D3.js amb Altres Llibreries?

  • Funcionalitat Millorada: Combinant D3.js amb altres llibreries es poden afegir característiques que D3.js per si sol no proporciona.
  • Rendiment Millorat: Algunes llibreries estan optimitzades per a tasques específiques, cosa que pot millorar el rendiment de les teves visualitzacions.
  • Facilitat d'Ús: Algunes llibreries ofereixen APIs més senzilles per a tasques que podrien ser complexes en D3.js.
  • Suport Comunitari: Les llibreries populars sovint tenen grans comunitats i documentació extensa, cosa que facilita trobar ajuda i recursos.

Integrar D3.js amb React

React és una llibreria popular de JavaScript per construir interfícies d'usuari. Integrar D3.js amb React et permet crear visualitzacions de dades dinàmiques i interactives dins d'una aplicació React.

Conceptes Clau

  • Arquitectura Basada en Components: Els components de React poden encapsular visualitzacions de D3.js.
  • Gestió de l'Estat: Utilitza l'estat de React per gestionar les dades i desencadenar actualitzacions a les visualitzacions de D3.js.
  • Mètodes del Cicle de Vida: Utilitza els mètodes del cicle de vida de React per inicialitzar i actualitzar les visualitzacions de D3.js.

Exemple: Gràfic de Barres Simple amb React i D3.js

import React, { useRef, useEffect } from 'react';
import * as d3 from 'd3';

const BarChart = ({ data }) => {
  const ref = useRef();

  useEffect(() => {
    const svg = d3.select(ref.current)
      .attr('width', 500)
      .attr('height', 300);

    svg.selectAll('rect')
      .data(data)
      .enter()
      .append('rect')
      .attr('x', (d, i) => i * 30)
      .attr('y', d => 300 - d * 10)
      .attr('width', 25)
      .attr('height', d => d * 10)
      .attr('fill', 'blue');
  }, [data]);

  return <svg ref={ref}></svg>;
};

export default BarChart;

Explicació

  • useRef: Un hook per crear una referència a l'element SVG.
  • useEffect: Un hook per executar el codi de D3.js després que el component es munta i sempre que les dades canvien.
  • d3.select: Selecciona l'element SVG i vincula les dades per crear el gràfic de barres.

Integrar D3.js amb Angular

Angular és una plataforma per construir aplicacions web mòbils i d'escriptori. Integrar D3.js amb Angular et pot ajudar a crear visualitzacions complexes dins d'una aplicació Angular.

Conceptes Clau

  • Directives: Utilitza directives d'Angular per encapsular visualitzacions de D3.js.
  • Injecció de Dependències: Aprofita la injecció de dependències d'Angular per gestionar serveis i dades.
  • Detecció de Canvis: Utilitza la detecció de canvis d'Angular per actualitzar les visualitzacions de D3.js quan les dades canvien.

Exemple: Gràfic de Línies Simple amb Angular i D3.js

import { Component, ElementRef, Input, OnChanges, ViewChild } from '@angular/core';
import * as d3 from 'd3';

@Component({
  selector: 'app-line-chart',
  template: '<svg #chart></svg>',
  styleUrls: ['./line-chart.component.css']
})
export class LineChartComponent implements OnChanges {
  @ViewChild('chart') private chartContainer: ElementRef;
  @Input() data: number[];

  ngOnChanges(): void {
    this.createChart();
  }

  private createChart(): void {
    const element = this.chartContainer.nativeElement;
    const svg = d3.select(element)
      .attr('width', 500)
      .attr('height', 300);

    const x = d3.scaleLinear().domain([0, this.data.length - 1]).range([0, 500]);
    const y = d3.scaleLinear().domain([0, d3.max(this.data)]).range([300, 0]);

    const line = d3.line<number>()
      .x((d, i) => x(i))
      .y(d => y(d));

    svg.append('path')
      .datum(this.data)
      .attr('fill', 'none')
      .attr('stroke', 'blue')
      .attr('d', line);
  }
}

Explicació

  • @ViewChild: Un decorador per obtenir una referència a l'element SVG.
  • @Input: Un decorador per acceptar dades com a entrada des d'un component pare.
  • ngOnChanges: Un hook del cicle de vida per detectar canvis en les dades d'entrada i actualitzar el gràfic.

Integrar D3.js amb Vue.js

Vue.js és un framework progressiu per construir interfícies d'usuari. Integrar D3.js amb Vue.js et permet crear visualitzacions reactives i interactives dins d'una aplicació Vue.

Conceptes Clau

  • Components de Vue: Encapsula visualitzacions de D3.js dins de components de Vue.
  • Dades Reactives: Utilitza el sistema de reactivitat de Vue per gestionar les dades i actualitzar les visualitzacions.
  • Directives: Crea directives personalitzades per gestionar visualitzacions de D3.js.

Exemple: Gràfic de Pastís Simple amb Vue.js i D3.js

<template>
  <svg ref="chart"></svg>
</template>

<script>
import * as d3 from 'd3';

export default {
  props: ['data'],
  mounted() {
    this.createChart();
  },
  watch: {
    data() {
      this.createChart();
    }
  },
  methods: {
    createChart() {
      const svg = d3.select(this.$refs.chart)
        .attr('width', 500)
        .attr('height', 300);

      const radius = Math.min(500, 300) / 2;
      const g = svg.append('g').attr('transform', `translate(${radius}, ${radius})`);

      const pie = d3.pie().value(d => d);
      const arc = d3.arc().innerRadius(0).outerRadius(radius);

      const arcs = g.selectAll('.arc')
        .data(pie(this.data))
        .enter().append('g')
        .attr('class', 'arc');

      arcs.append('path')
        .attr('d', arc)
        .attr('fill', (d, i) => d3.schemeCategory10[i]);
    }
  }
};
</script>

<style scoped>
svg {
  display: block;
  margin: auto;
}
</style>

Explicació

  • props: Accepta dades com a prop des d'un component pare.
  • mounted: Un hook del cicle de vida per crear el gràfic després que el component es munta.
  • watch: Observa els canvis en el prop de dades i actualitza el gràfic en conseqüència.
  • d3.pie: Crea un layout de pastís.
  • d3.arc: Crea formes d'arc per al gràfic de pastís.

Aprofitant els punts forts d'aquestes llibreries, pots crear visualitzacions més dinàmiques, interactives i eficients. Entendre com combinar aquestes eines de manera efectiva et farà un desenvolupador més versàtil i habilidós.
