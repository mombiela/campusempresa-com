Seleccionar elements és un concepte fonamental en D3.js, ja que et permet vincular dades al DOM i manipular elements basats en aquestes dades. Aquest tema et guiarà des de tècniques bàsiques fins a avançades per seleccionar elements en D3.js.

Selecció Bàsica

Seleccionar un Sol Element

  • Utilitza d3.select() per seleccionar un sol element.
  • Aquest mètode selecciona el primer element coincident en el DOM.
// Selecciona el primer element <p> i canvia el seu color de text a vermell
d3.select("p").style("color", "red");

Seleccionar Múltiples Elements

  • Utilitza d3.selectAll() per seleccionar múltiples elements.
  • Aquest mètode selecciona tots els elements coincidents en el DOM.
// Selecciona tots els elements <p> i canvia el seu color de text a blau
d3.selectAll("p").style("color", "blue");

Selecció Intermèdia

Enllaçar Mètodes

  • D3.js permet l'enllaç de mètodes, el que significa que pots aplicar múltiples mètodes en una sola declaració.
// Selecciona tots els elements <p>, canvia el seu color de text a verd, i estableix la seva mida de font
d3.selectAll("p")
  .style("color", "green")
  .style("font-size", "20px");

Utilitzar Selectors

  • Pots utilitzar selectors CSS per fer seleccions més complexes.
// Selecciona tots els elements <p> amb la classe 'highlight' i canvia el seu color de fons
d3.selectAll("p.highlight").style("background-color", "yellow");

Selecció Avançada

Seleccionar Elements Niats

  • Utilitza seleccions niades per seleccionar elements dins d'un element pare específic.
// Selecciona tots els elements <li> dins d'un <ul> amb l'id 'list' i canvia el seu color de text
d3.select("#list").selectAll("li").style("color", "purple");

Utilitzar Funcions per a Selecció Dinàmica

  • Pots utilitzar funcions per establir dinàmicament estils o atributs basats en dades.
// Selecciona tots els elements <p> i estableix el seu color de text basat en el seu índex
d3.selectAll("p").style("color", function(d, i) {
  return i % 2 === 0 ? "red" : "blue";
});

Combinant Dades amb Seleccions

  • Utilitza data() per vincular dades als elements i enter() per gestionar nous punts de dades.
// Array de dades
var data = [10, 20, 30, 40, 50];

// Selecciona tots els elements <div> i vincula dades
var divs = d3.select("body").selectAll("div")
  .data(data);

// Selecció d'entrada per crear nous elements <div> per a nous punts de dades
divs.enter().append("div")
  .style("width", function(d) { return d + "px"; })
  .style("height", "20px")
  .style("background-color", "teal")
  .text(function(d) { return d; });

Conclusió

Seleccionar elements és una habilitat crucial en D3.js, permetent-te vincular dades al DOM i manipular elements dinàmicament. Començant per seleccions bàsiques utilitzant d3.select() i d3.selectAll(), pots progressar a tècniques més avançades com seleccions niades, estil dinàmic, i combinació de dades amb seleccions. Dominar aquestes tècniques et proporcionarà una base sòlida per crear visualitzacions complexes i interactives amb D3.js.
