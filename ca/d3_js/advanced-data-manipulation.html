Introducció

En aquesta secció, aprofundirem en tècniques avançades de manipulació de dades utilitzant D3.js. Això inclourà treballar amb conjunts de dades complexos, realitzar transformacions de dades i crear visualitzacions dinàmiques. Al final d'aquesta secció, hauríeu de ser capaços de gestionar tasques sofisticades de manipulació de dades i crear visualitzacions més interactives i informatives.

Conceptes Clau

  • Enllaç de Dades: Associar dades amb elements del DOM.
  • Unions de Dades: Gestionar les seleccions d'entrada, actualització i sortida.
  • Transformació de Dades: Modificar les dades per adaptar-les a les necessitats de la vostra visualització.
  • Escales i Eixos: Mapar valors de dades a dimensions visuals.
  • Dades Niuades: Gestionar dades jeràrquiques o agrupades.

Enllaç de Dades

L'enllaç de dades en D3.js és el procés d'associar dades amb elements del DOM. Aquest és un concepte fonamental que us permet crear visualitzacions dinàmiques.

// Exemple: Enllaçar dades a una selecció d'elements <div>
const data = [10, 20, 30, 40, 50];

d3.selectAll('div')
  .data(data)
  .style('height', d => `${d}px`);

En aquest exemple, l'alçada de cada element <div> es defineix en funció del valor de dades corresponent.

Unions de Dades

Les unions de dades són crucials per gestionar com es vinculen les dades als elements, especialment quan el conjunt de dades canvia.

// Exemple: Seleccions d'entrada, actualització i sortida
const data = [10, 20, 30, 40, 50];

const divs = d3.selectAll('div')
  .data(data);

// Selecció d'entrada: Crear nous elements per a noves dades
divs.enter().append('div')
  .style('height', d => `${d}px`);

// Selecció d'actualització: Actualitzar elements existents
divs.style('height', d => `${d}px`);

// Selecció de sortida: Eliminar elements que ja no tenen dades
divs.exit().remove();

Aquest exemple demostra com gestionar punts de dades nous, existents i eliminats.

Transformació de Dades

Transformar dades sovint és necessari per preparar-les per a la visualització. Això pot incloure filtrar, ordenar i agregar dades.

// Exemple: Filtrar i ordenar dades
const data = [10, 20, 30, 40, 50];

// Filtrar dades per incloure només valors superiors a 20
const filteredData = data.filter(d => d > 20);

// Ordenar dades en ordre descendent
const sortedData = filteredData.sort((a, b) => b - a);

console.log(sortedData); // Sortida: [50, 40, 30]

En aquest exemple, filtrem i ordenem les dades per preparar-les per a la visualització.

Escales i Eixos

Les escales i els eixos s'utilitzen per mapar valors de dades a dimensions visuals, com ara posicions de píxels o colors.

// Exemple: Crear una escala lineal i un eix
const data = [10, 20, 30, 40, 50];

const scale = d3.scaleLinear()
  .domain([0, d3.max(data)])
  .range([0, 500]);

const axis = d3.axisBottom(scale);

d3.select('svg')
  .append('g')
  .attr('transform', 'translate(0, 50)')
  .call(axis);

Aquest exemple mostra com crear una escala lineal i un eix basat en les dades.

Dades Niuades

Gestionar dades niuades o jeràrquiques és essencial per a visualitzacions com gràfics de barres agrupades o diagrames d'arbre.

// Exemple: Niuar dades
const data = [
  {category: 'A', value: 30},
  {category: 'B', value: 80},
  {category: 'A', value: 45},
  {category: 'B', value: 60}
];

const nestedData = d3.nest()
  .key(d => d.category)
  .entries(data);

console.log(nestedData);

Aquest exemple demostra com agrupar dades per una clau específica.

Conclusió

En aquesta secció, hem cobert tècniques avançades de manipulació de dades en D3.js, incloent l'enllaç de dades, unions de dades, transformació de dades, escales i eixos, i la gestió de dades niuades. Dominar aquestes tècniques us permetrà crear visualitzacions més complexes i dinàmiques, proporcionant una comprensió més profunda de les vostres dades. Continueu practicant aquests conceptes per convertir-vos en experts en manipulació avançada de dades amb D3.js.
