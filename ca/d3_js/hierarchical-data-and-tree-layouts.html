Introducció a les Dades Jeràrquiques

Les dades jeràrquiques són un tipus d'estructura de dades on els elements estan organitzats en una estructura semblant a un arbre. Això és comú en molts escenaris del món real, com ara organigrames, directoris de fitxers i taxonomies biològiques.

  • Estructura d'Arbre: Un arbre és una col·lecció de nodes connectats per arestes. Cada node conté un valor o dades, i els nodes estan connectats en una relació pare-fill.
  • Node Arrel: El node superior en un arbre, que no té pare.
  • Nodes Fulla: Nodes que no tenen cap fill.
  • Nodes Interns: Nodes que tenen almenys un fill.

Entenent les Disposicions Jeràrquiques de D3.js

D3.js proporciona eines potents per visualitzar dades jeràrquiques. Les disposicions més comunes per a dades jeràrquiques en D3.js són les disposicions d'arbre i de clúster.

  • Disposició d'Arbre: Una estructura d'arbre tradicional on cada node està connectat als seus fills.
  • Disposició de Clúster: Similar a una disposició d'arbre però dissenyada per produir una representació més compacta.

Creant una Disposició d'Arbre Bàsica

Per crear una disposició d'arbre en D3.js, segueix aquests passos:

  1. Prepara les Dades: Assegura't que les teves dades estiguin en un format jeràrquic, típicament un objecte JSON niat.
  2. Crea la Disposició d'Arbre: Utilitza la funció d3.tree() de D3 per crear la disposició.
  3. Dibuixa l'Arbre: Utilitza elements SVG per dibuixar els nodes i els enllaços.

Exemple: Disposició d'Arbre Bàsica

// Dades jeràrquiques de mostra
const data = {
  name: "Arrel",
  children: [
    { name: "Fill 1" },
    { 
      name: "Fill 2",
      children: [
        { name: "Nét 1" },
        { name: "Nét 2" }
      ]
    }
  ]
};

// Configura les dimensions del llenç SVG
const width = 600;
const height = 400;

const svg = d3.select("body").append("svg")
  .attr("width", width)
  .attr("height", height);

// Crea una disposició d'arbre
const treeLayout = d3.tree().size([width, height - 100]);

// Converteix les dades en una jerarquia
const root = d3.hierarchy(data);

// Genera la disposició d'arbre
treeLayout(root);

// Dibuixa els enllaços (arestes)
svg.selectAll('line')
  .data(root.links())
  .enter()
  .append('line')
  .attr('x1', d => d.source.x)
  .attr('y1', d => d.source.y)
  .attr('x2', d => d.target.x)
  .attr('y2', d => d.target.y)
  .attr('stroke', 'black');

// Dibuixa els nodes
svg.selectAll('circle')
  .data(root.descendants())
  .enter()
  .append('circle')
  .attr('cx', d => d.x)
  .attr('cy', d => d.y)
  .attr('r', 5)
  .attr('fill', 'blue');

Personalitzant les Disposicions d'Arbre

Pots personalitzar l'aparença i el comportament de la teva disposició d'arbre ajustant diverses propietats i afegint interactivitat.

  • Mida i Forma del Node: Canvia la mida i la forma dels nodes per representar millor les dades.
  • Estils d'Enllaç: Personalitza els enllaços entre nodes amb diferents colors, amplades i estils.
  • Interactivitat: Afegeix eines emergents, esdeveniments de clic i altres característiques interactives per millorar l'experiència de l'usuari.

Exemple: Personalitzant Nodes i Enllaços

// Personalitzant nodes amb diferents colors i mides
svg.selectAll('circle')
  .data(root.descendants())
  .enter()
  .append('circle')
  .attr('cx', d => d.x)
  .attr('cy', d => d.y)
  .attr('r', d => d.children ? 10 : 5) // Cercles més grans per a nodes interns
  .attr('fill', d => d.children ? 'red' : 'green'); // Diferents colors per a nodes interns i fulla

// Personalitzant enllaços amb diferents estils
svg.selectAll('line')
  .data(root.links())
  .enter()
  .append('line')
  .attr('x1', d => d.source.x)
  .attr('y1', d => d.source.y)
  .attr('x2', d => d.target.x)
  .attr('y2', d => d.target.y)
  .attr('stroke', 'black')
  .attr('stroke-width', 2)
  .attr('stroke-dasharray', '5,5'); // Línies discontínues

Disposicions d'Arbre Avançades

Per a visualitzacions més avançades, pots explorar funcions i tècniques addicionals de D3.js.

  • Arbres Col·lapsables: Implementa nodes col·lapsables per permetre als usuaris expandir i col·lapsar branques de l'arbre.
  • Arbres Radials: Crea disposicions d'arbre radials per a una representació circular de les dades jeràrquiques.
  • Dades Dinàmiques: Actualitza la disposició de l'arbre dinàmicament basant-se en interaccions de l'usuari o dades en temps real.

Exemple: Arbre Col·lapsable

// Funció per alternar els fills en fer clic
function toggle(d) {
  if (d.children) {
    d._children = d.children;
    d.children = null;
  } else {
    d.children = d._children;
    d._children = null;
  }
  update(d);
}

// Funció d'actualització per redibuixar l'arbre
function update(source) {
  const nodes = root.descendants();
  const links = root.links();

  // Actualitza els nodes
  const node = svg.selectAll('circle')
    .data(nodes, d => d.id || (d.id = ++i));

  node.enter().append('circle')
    .attr('r', 5)
    .attr('fill', 'blue')
    .on('click', toggle);

  node.transition()
    .attr('cx', d => d.x)
    .attr('cy', d => d.y);

  // Actualitza els enllaços
  const link = svg.selectAll('line')
    .data(links, d => d.target.id);

  link.enter().append('line')
    .attr('stroke', 'black');

  link.transition()
    .attr('x1', d => d.source.x)
    .attr('y1', d => d.source.y)
    .attr('x2', d => d.target.x)
    .attr('y2', d => d.target.y);
}

// Actualització inicial
update(root);

Conclusió

Les dades jeràrquiques i les disposicions d'arbre són essencials per visualitzar relacions complexes d'una manera estructurada. D3.js proporciona eines robustes per crear i personalitzar aquestes visualitzacions, des de disposicions d'arbre bàsiques fins a representacions interactives i dinàmiques avançades. En dominar aquestes tècniques, pots comunicar dades jeràrquiques de manera efectiva i visualment atractiva.
