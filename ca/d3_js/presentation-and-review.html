Presentació i Revisió

Introducció

En aquesta secció, presentarem i revisarem els conceptes i tècniques clau coberts en el curs de D3.js. Això ajudarà a consolidar la teva comprensió i assegurar que estàs ben preparat per aplicar D3.js en projectes del món real.

Conceptes Clau

Revisem els conceptes i tècniques fonamentals que hem après al llarg del curs:

  • Seleccions: Com seleccionar elements del DOM utilitzant D3.js.
  • Associació de Dades: Associar dades amb elements del DOM.
  • Patró Enter, Update, Exit: Gestionar els canvis impulsats per dades en el DOM.
  • Escales i Eixos: Mapar valors de dades a propietats visuals.
  • Formes i Camins: Crear i manipular formes SVG.
  • Transicions i Animacions: Afegir comportament dinàmic a les visualitzacions.
  • Interactivitat: Fer les visualitzacions interactives amb esdeveniments.
  • Disposicions: Utilitzar disposicions de D3.js per a visualitzacions complexes com gràfics de força dirigida i mapes de calor.

Seleccions

Les seleccions són la base de D3.js, permetent-te seleccionar i manipular elements del DOM.

// Seleccionar tots els elements de paràgraf
d3.selectAll("p")
  .style("color", "blue");

Explicació:

  • d3.selectAll("p"): Selecciona tots els elements <p>.
  • .style("color", "blue"): Canvia el color del text a blau.

Associació de Dades

L'associació de dades és el procés d'associar dades amb elements del DOM.

// Associar dades amb elements de paràgraf
const data = [10, 20, 30, 40, 50];
d3.selectAll("p")
  .data(data)
  .text(d => `Valor: ${d}`);

Explicació:

  • data: Una matriu de valors.
  • .data(data): Associa la matriu de dades amb els elements seleccionats.
  • .text(d => Valor: ${d}): Estableix el contingut del text de cada paràgraf al valor de dades corresponent.

Patró Enter, Update, Exit

Aquest patró és crucial per gestionar dades dinàmiques.

const data = [10, 20, 30];
const p = d3.select("body").selectAll("p")
  .data(data);

// Selecció Enter
p.enter().append("p")
  .text(d => `Nou Valor: ${d}`);

// Selecció Update
p.text(d => `Valor Actualitzat: ${d}`);

// Selecció Exit
p.exit().remove();

Explicació:

  • p.enter(): Gestiona nous elements de dades.
  • p.text(): Actualitza els elements existents.
  • p.exit(): Elimina els elements que ja no tenen dades corresponents.

Escales i Eixos

Les escales mapejen valors de dades a propietats visuals, i els eixos proporcionen marques de referència.

const scale = d3.scaleLinear()
  .domain([0, 100])
  .range([0, 500]);

const axis = d3.axisBottom(scale);

d3.select("svg")
  .append("g")
  .attr("transform", "translate(0, 50)")
  .call(axis);

Explicació:

  • d3.scaleLinear(): Crea una escala lineal.
  • .domain([0, 100]): Estableix el domini d'entrada.
  • .range([0, 500]): Estableix el rang de sortida.
  • d3.axisBottom(scale): Crea un eix inferior utilitzant l'escala.
  • .call(axis): Renderitza l'eix.

Formes i Camins

Crear i manipular formes SVG és una part essencial de D3.js.

const line = d3.line()
  .x(d => d.x)
  .y(d => d.y);

const data = [{x: 0, y: 0}, {x: 50, y: 50}, {x: 100, y: 0}];

d3.select("svg")
  .append("path")
  .datum(data)
  .attr("d", line)
  .attr("stroke", "black")
  .attr("fill", "none");

Explicació:

  • d3.line(): Crea un generador de línies.
  • .x(d => d.x): Estableix l'accessor de la coordenada x.
  • .y(d => d.y): Estableix l'accessor de la coordenada y.
  • .datum(data): Associa les dades amb l'element path.
  • .attr("d", line): Genera les dades del camí.
  • .attr("stroke", "black"): Estableix el color del traç.
  • .attr("fill", "none"): Elimina el farciment.

Transicions i Animacions

Les transicions afegeixen comportament dinàmic a les visualitzacions.

d3.select("circle")
  .transition()
  .duration(1000)
  .attr("r", 50);

Explicació:

  • .transition(): Inicia una transició.
  • .duration(1000): Estableix la durada a 1000 mil·lisegons.
  • .attr("r", 50): Canvia el radi a 50.

Interactivitat

L'interactivitat fa que les visualitzacions siguin més atractives.

d3.select("circle")
  .on("mouseover", function() {
    d3.select(this).attr("fill", "red");
  })
  .on("mouseout", function() {
    d3.select(this).attr("fill", "blue");
  });

Explicació:

  • .on("mouseover", function() {...}): Afegeix un oient d'esdeveniments mouseover.
  • .on("mouseout", function() {...}): Afegeix un oient d'esdeveniments mouseout.
  • d3.select(this).attr("fill", "red"): Canvia el color de farciment a vermell en mouseover.
  • d3.select(this).attr("fill", "blue"): Canvia el color de farciment a blau en mouseout.

Disposicions

Les disposicions ajuden a crear visualitzacions complexes.

const data = [10, 20, 30, 40, 50];
const pie = d3.pie()(data);

const arc = d3.arc()
  .innerRadius(0)
  .outerRadius(100);

d3.select("svg")
  .selectAll("path")
  .data(pie)
  .enter()
  .append("path")
  .attr("d", arc)
  .attr("fill", (d, i) => d3.schemeCategory10[i]);

Explicació:

  • d3.pie(): Crea una disposició de pastís.
  • d3.arc(): Crea un generador d'arcs.
  • .innerRadius(0): Estableix el radi interior.
  • .outerRadius(100): Estableix el radi exterior.
  • .data(pie): Associa les dades del pastís amb els elements path.
  • .attr("d", arc): Genera les dades del camí de l'arc.
  • .attr("fill", (d, i) => d3.schemeCategory10[i]): Estableix el color de farciment utilitzant un esquema de colors.

Conclusió

En aquesta secció, hem revisat els conceptes i tècniques clau de D3.js, incloent seleccions, associació de dades, el patró enter-update-exit, escales i eixos, formes i camins, transicions i animacions, interactivitat i disposicions. Dominant aquests conceptes, estàs ben equipat per crear visualitzacions de dades potents i dinàmiques amb D3.js. Continua practicant i experimentant amb diferents conjunts de dades i tipus de visualitzacions per millorar encara més les teves habilitats.
