Introducció

Els gràfics de línies són un tipus fonamental de visualització de dades utilitzat per mostrar informació com una sèrie de punts de dades connectats per segments de línia recta. D3.js (Documents Basats en Dades) és una potent biblioteca de JavaScript per crear visualitzacions de dades dinàmiques i interactives en navegadors web. Aquesta secció et guiarà a través del procés de creació de gràfics de línies utilitzant D3.js, des de nivells bàsics fins a avançats.

Configuració del Teu Entorn

Abans de crear un gràfic de línies, assegura't de tenir la configuració necessària:

  • Inclou la biblioteca D3.js en el teu fitxer HTML.
  • Prepara un contenidor HTML (com un element SVG) on es renderitzarà el gràfic.

Incloent D3.js

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Gràfic de Línies amb D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <svg width="800" height="400"></svg>
</body>
</html>

Gràfic de Línies Bàsic

Preparació de Dades

Comença preparant les teves dades. Per simplicitat, utilitzarem una matriu d'objectes on cada objecte representa un punt de dades.

const data = [
    { date: new Date(2023, 0, 1), value: 30 },
    { date: new Date(2023, 1, 1), value: 40 },
    { date: new Date(2023, 2, 1), value: 35 },
    { date: new Date(2023, 3, 1), value: 50 },
    { date: new Date(2023, 4, 1), value: 55 }
];

Creació d'Escales

Les escales són funcions que mapegen valors de dades a valors de píxels. Crearem escales per als eixos x i y.

const svg = d3.select("svg");
const margin = { top: 20, right: 30, bottom: 30, left: 40 };
const width = +svg.attr("width") - margin.left - margin.right;
const height = +svg.attr("height") - margin.top - margin.bottom;

const x = d3.scaleTime()
    .domain(d3.extent(data, d => d.date))
    .range([0, width]);

const y = d3.scaleLinear()
    .domain([0, d3.max(data, d => d.value)])
    .range([height, 0]);

Dibuixant la Línia

Utilitza la funció d3.line per crear un generador de línies.

const line = d3.line()
    .x(d => x(d.date))
    .y(d => y(d.value));

const g = svg.append("g")
    .attr("transform", `translate(${margin.left},${margin.top})`);

g.append("path")
    .datum(data)
    .attr("fill", "none")
    .attr("stroke", "steelblue")
    .attr("stroke-width", 1.5)
    .attr("d", line);

Afegint Eixos

Afegeix els eixos x i y al gràfic.

g.append("g")
    .attr("transform", `translate(0,${height})`)
    .call(d3.axisBottom(x));

g.append("g")
    .call(d3.axisLeft(y));

Gràfic de Línies Intermedi

Afegint Interactivitat

Millora el gràfic afegint tooltips que mostrin els valors de les dades quan es passa el cursor sobre els punts.

const tooltip = d3.select("body").append("div")
    .attr("class", "tooltip")
    .style("opacity", 0);

g.selectAll("dot")
    .data(data)
    .enter().append("circle")
    .attr("cx", d => x(d.date))
    .attr("cy", d => y(d.value))
    .attr("r", 5)
    .attr("fill", "red")
    .on("mouseover", (event, d) => {
        tooltip.transition()
            .duration(200)
            .style("opacity", .9);
        tooltip.html(`Data: ${d.date}<br/>Valor: ${d.value}`)
            .style("left", (event.pageX + 5) + "px")
            .style("top", (event.pageY - 28) + "px");
    })
    .on("mouseout", d => {
        tooltip.transition()
            .duration(500)
            .style("opacity", 0);
    });

Estilitzant el Gràfic

Millora l'atractiu visual afegint estils CSS.

.tooltip {
    position: absolute;
    text-align: center;
    width: 60px;
    height: 28px;
    padding: 2px;
    font: 12px sans-serif;
    background: lightsteelblue;
    border: 0px;
    border-radius: 8px;
    pointer-events: none;
}

Gràfic de Línies Avançat

Gestionant Dades en Temps Real

Actualitza el gràfic dinàmicament amb dades en temps real.

function updateChart(newData) {
    // Actualitza les escales
    x.domain(d3.extent(newData, d => d.date));
    y.domain([0, d3.max(newData, d => d.value)]);

    // Actualitza la línia
    g.select("path")
        .datum(newData)
        .attr("d", line);

    // Actualitza els punts
    const circles = g.selectAll("circle")
        .data(newData);

    circles.enter().append("circle")
        .attr("r", 5)
        .attr("fill", "red")
        .merge(circles)
        .attr("cx", d => x(d.date))
        .attr("cy", d => y(d.value));

    circles.exit().remove();

    // Actualitza els eixos
    g.select(".x.axis")
        .call(d3.axisBottom(x));

    g.select(".y.axis")
        .call(d3.axisLeft(y));
}

Afegint Zoom i Pan

Habilita el zoom i el pan per a una millor exploració de les dades.

const zoom = d3.zoom()
    .scaleExtent([1, 10])
    .translateExtent([[-width, -height], [2 * width, 2 * height]])
    .on("zoom", zoomed);

svg.call(zoom);

function zoomed(event) {
    const transform = event.transform;
    const newX = transform.rescaleX(x);
    const newY = transform.rescaleY(y);

    g.select(".x.axis").call(d3.axisBottom(newX));
    g.select(".y.axis").call(d3.axisLeft(newY));

    g.select("path").attr("d", line.x(d => newX(d.date)).y(d => newY(d.value)));
    g.selectAll("circle")
        .attr("cx", d => newX(d.date))
        .attr("cy", d => newY(d.value));
}

Conclusió

Crear gràfics de línies amb D3.js implica diversos passos, des de la configuració de l'entorn i la preparació de dades fins al dibuix del gràfic i l'afegit d'interactivitat. En dominar aquestes tècniques, pots crear visualitzacions dinàmiques i interactives que comuniquin eficaçment les dades. A mesura que avancis, podràs incorporar actualitzacions de dades en temps real i habilitar el zoom i el pan per millorar l'experiència de l'usuari.
