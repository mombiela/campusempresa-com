El Projecte Sobre nosaltres Contribuir Donacions Llicència

La vinculació de dades és un concepte fonamental en D3.js, que permet la connexió entre les dades i els elements en el DOM. Aquest tema et guiarà des dels conceptes bàsics fins a les tècniques avançades de vinculació de dades en D3.js.

Què és la vinculació de dades?

La vinculació de dades en D3.js es refereix al procés de connectar dades amb elements del DOM. Això et permet crear, actualitzar i manipular dinàmicament els elements basant-te en les dades.

Conceptes clau

  • Selecció: El procés de seleccionar elements del DOM.
  • Unió de dades: El procés de vincular dades amb els elements seleccionats.
  • Entrar, Actualitzar, Sortir: Les tres fases de la vinculació de dades en D3.js.

Vinculació de dades bàsica

Seleccionar elements

Per vincular dades amb elements, primer has de seleccionar els elements. D3.js utilitza selectors CSS per seleccionar elements.

d3.select("body") // Selecciona l'element body
d3.selectAll("p") // Selecciona tots els elements de paràgraf

Vincular dades

Un cop hagis seleccionat els elements, pots vincular dades a ells utilitzant el mètode .data().

const data = [10, 20, 30, 40, 50];

d3.selectAll("p")
  .data(data)
  .text(d => d); // Estableix el text de cada paràgraf al valor de dades corresponent

Entrar, Actualitzar, Sortir

Quan vincules dades, D3.js categoritza els elements en tres grups:

  • Entrar: Elements que necessiten ser creats.
  • Actualitzar: Elements que necessiten ser actualitzats.
  • Sortir: Elements que necessiten ser eliminats.
const data = [10, 20, 30, 40, 50];

const p = d3.select("body").selectAll("p")
  .data(data);

// Fase d'entrar
p.enter().append("p")
  .text(d => d);

// Fase d'actualitzar
p.text(d => d);

// Fase de sortir
p.exit().remove();

Vinculació de dades avançada

Vincular dades a elements complexos

Pots vincular dades a elements més complexos com formes SVG.

const data = [10, 20, 30, 40, 50];

const svg = d3.select("body").append("svg")
  .attr("width", 500)
  .attr("height", 100);

const circles = svg.selectAll("circle")
  .data(data);

// Fase d'entrar
circles.enter().append("circle")
  .attr("cx", (d, i) => i * 50 + 25)
  .attr("cy", 50)
  .attr("r", d => d / 2);

// Fase d'actualitzar
circles.attr("r", d => d / 2);

// Fase de sortir
circles.exit().remove();

Funcions clau

Quan treballes amb dades més complexes, potser necessitaràs utilitzar funcions clau per assegurar-te que les dades es vinculen correctament als elements.

const data = [
  { id: 1, value: 10 },
  { id: 2, value: 20 },
  { id: 3, value: 30 }
];

const circles = svg.selectAll("circle")
  .data(data, d => d.id); // Utilitzant l'id com a funció clau

// Fase d'entrar
circles.enter().append("circle")
  .attr("cx", (d, i) => i * 50 + 25)
  .attr("cy", 50)
  .attr("r", d => d.value / 2);

// Fase d'actualitzar
circles.attr("r", d => d.value / 2);

// Fase de sortir
circles.exit().remove();

Conclusió

La vinculació de dades és una característica poderosa de D3.js que et permet crear, actualitzar i eliminar dinàmicament elements basant-te en les teves dades. Comprendre les fases d'entrar, actualitzar i sortir és crucial per utilitzar efectivament D3.js. En dominar aquests conceptes, pots crear visualitzacions complexes i interactives que responen als canvis en les teves dades.
