Personalització d'eixos ►

En aquesta secció, explorarem com crear i personalitzar eixos en D3.js. Els eixos són essencials per visualitzar dades de manera significativa, proporcionant context i escala als punts de dades mostrats en gràfics i diagrames.

Introducció als Eixos

  • Eixos: Els eixos són línies que defineixen el sistema de coordenades per a un gràfic. Ajuden a mapar els valors de les dades a posicions en el gràfic.
  • Eixos de D3.js: D3.js proporciona funcions integrades per crear i personalitzar eixos fàcilment.

Configuració de l'Entorn

Abans d'endinsar-nos en la creació d'eixos, assegura't de tenir la següent configuració:

  • Fitxer HTML: Crea un fitxer HTML per incloure el teu script de D3.js.
  • Biblioteca D3.js: Inclou la biblioteca D3.js en el teu fitxer HTML.
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Eixos amb D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <script src="script.js"></script>
</body>
</html>

Creació d'un Eix Bàsic

  1. Defineix el Contenidor SVG: Crea un element SVG on es dibuixarà l'eix.
const svg = d3.select("body")
    .append("svg")
    .attr("width", 500)
    .attr("height", 300);
  1. Crea Escales: Les escales mapen els valors de les dades a valors de píxels. Per als eixos, normalment utilitzem escales lineals.
const xScale = d3.scaleLinear()
    .domain([0, 100])  // Domini d'entrada
    .range([0, 480]);  // Rang de sortida

const yScale = d3.scaleLinear()
    .domain([0, 100])
    .range([280, 0]);
  1. Genera Eixos: Utilitza els generadors d'eixos de D3 per crear els eixos.
const xAxis = d3.axisBottom(xScale);
const yAxis = d3.axisLeft(yScale);
  1. Renderitza Eixos: Afegeix els eixos al contenidor SVG.
svg.append("g")
    .attr("transform", "translate(10, 280)")  // Posiciona l'eix x
    .call(xAxis);

svg.append("g")
    .attr("transform", "translate(10, 0)")  // Posiciona l'eix y
    .call(yAxis);

Personalització d'Eixos

Marques i Format de Marques

  • Marques: Controla el nombre de marques en l'eix.
  • Format de Marques: Personalitza el format de les etiquetes de les marques.
const xAxis = d3.axisBottom(xScale)
    .ticks(5)  // Estableix el nombre de marques
    .tickFormat(d3.format(".0f"));  // Format de marques com a enters

const yAxis = d3.axisLeft(yScale)
    .ticks(5)
    .tickFormat(d3.format(".0%"));  // Format de marques com a percentatges

Estilització d'Eixos

  • CSS: Utilitza CSS per estilitzar els eixos.
<style>
    .axis path,
    .axis line {
        fill: none;
        stroke: black;
        shape-rendering: crispEdges;
    }

    .axis text {
        font-family: sans-serif;
        font-size: 12px;
    }
</style>
  • Aplicació de Classes CSS: Aplica classes CSS als eixos.
svg.append("g")
    .attr("class", "x axis")
    .attr("transform", "translate(10, 280)")
    .call(xAxis);

svg.append("g")
    .attr("class", "y axis")
    .attr("transform", "translate(10, 0)")
    .call(yAxis);

Personalitzacions Avançades

Línies de Graella

  • Línies de Graella: Afegeix línies de graella al gràfic per a una millor llegibilitat.
const xGridlines = d3.axisBottom(xScale)
    .tickSize(-280)
    .tickFormat("");

const yGridlines = d3.axisLeft(yScale)
    .tickSize(-480)
    .tickFormat("");

svg.append("g")
    .attr("class", "grid")
    .attr("transform", "translate(10, 280)")
    .call(xGridlines);

svg.append("g")
    .attr("class", "grid")
    .attr("transform", "translate(10, 0)")
    .call(yGridlines);

Rotació d'Etiquetes de Marques

  • Rotació d'Etiquetes: Rota les etiquetes de les marques per a una millor llegibilitat quan són massa llargues.
svg.append("g")
    .attr("class", "x axis")
    .attr("transform", "translate(10, 280)")
    .call(xAxis)
    .selectAll("text")
    .attr("transform", "rotate(-45)")
    .style("text-anchor", "end");

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de creació i personalització d'eixos en D3.js. Hem après a:

  • Configurar l'entorn i crear un eix bàsic.
  • Personalitzar les marques i els formats de marques.
  • Estilitzar els eixos utilitzant CSS.
  • Afegir personalitzacions avançades com línies de graella i rotació d'etiquetes de marques.

Entendre com crear i personalitzar eixos és crucial per construir visualitzacions de dades efectives. Practica aquests conceptes per convertir-te en un expert en l'ús de D3.js per als teus projectes de visualització de dades.
