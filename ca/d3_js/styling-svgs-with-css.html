Introducció

Estilitzar SVGs (Scalable Vector Graphics) amb CSS és una tècnica poderosa que et permet millorar l'aparença visual de les teves visualitzacions amb D3.js. Aquesta secció cobrirà els conceptes bàsics de l'estilització de SVG, com aplicar estils CSS als elements SVG, i tècniques avançades per crear visualitzacions dinàmiques i interactives.

Conceptes Clau

  • Conceptes Bàsics de SVG: Entendre l'estructura dels elements SVG.
  • Selectors CSS: Aplicar estils als elements SVG utilitzant selectors CSS.
  • Estils Inline vs. Fulls d'Estil Externs: Diferents mètodes per estilitzar SVGs.
  • Propietats CSS per a SVGs: Propietats CSS clau que es poden utilitzar amb elements SVG.
  • Transicions i Animacions: Afegir efectes dinàmics als SVGs.

Conceptes Bàsics de SVG

Abans d'endinsar-nos en l'estilització, és essencial entendre l'estructura bàsica dels elements SVG. Un SVG és un format basat en XML per a gràfics vectorials que es pot incrustar directament en HTML.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

En aquest exemple, tenim un element SVG que conté un cercle amb atributs específics.

Selectors CSS

Els selectors CSS s'utilitzen per aplicar estils a elements SVG específics. Pots utilitzar selectors d'elements, selectors de classe i selectors d'ID.

Selector d'Element

circle {
  fill: blue;
}

Selector de Classe

<circle class="my-circle" cx="50" cy="50" r="40" />
.my-circle {
  fill: green;
}

Selector d'ID

<circle id="unique-circle" cx="50" cy="50" r="40" />
#unique-circle {
  fill: yellow;
}

Estils Inline vs. Fulls d'Estil Externs

Hi ha dos mètodes principals per estilitzar SVGs: estils inline i fulls d'estil externs.

Estils Inline

Els estils inline es defineixen directament dins dels elements SVG.

<circle cx="50" cy="50" r="40" style="fill: purple; stroke: black; stroke-width: 3;" />

Fulls d'Estil Externs

Els fulls d'estil externs es defineixen en un fitxer CSS separat i s'enllacen al document HTML.

<link rel="stylesheet" type="text/css" href="styles.css">
circle {
  fill: orange;
  stroke: black;
  stroke-width: 3;
}

Propietats CSS per a SVGs

Diverses propietats CSS es poden utilitzar per estilitzar elements SVG. Aquí tens algunes propietats clau:

  • fill: Defineix el color dins de la forma SVG.
  • stroke: Defineix el color del contorn de la forma SVG.
  • stroke-width: Defineix l'amplada del contorn.
  • opacity: Defineix el nivell de transparència de l'element SVG.
  • transform: Aplica transformacions com rotar, escalar i traslladar.

Exemple

circle {
  fill: lightblue;
  stroke: darkblue;
  stroke-width: 2;
  opacity: 0.8;
  transform: scale(1.2);
}

Transicions i Animacions

Les transicions i animacions CSS poden afegir efectes dinàmics als teus elements SVG.

Transicions

Les transicions permeten canviar els valors de les propietats de manera suau durant una durada determinada.

circle {
  transition: fill 0.5s ease;
}

circle:hover {
  fill: pink;
}

Animacions

Les animacions poden crear seqüències de canvis més complexes.

@keyframes pulse {
  0% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.5);
  }
  100% {
    transform: scale(1);
  }
}

circle {
  animation: pulse 2s infinite;
}

Conclusió

Estilitzar SVGs amb CSS en D3.js et permet crear visualitzacions visualment atractives i interactives. Entenent els conceptes bàsics de l'estructura SVG, els selectors CSS i les propietats CSS clau, pots estilitzar efectivament els teus elements SVG. A més, aprofitant les transicions i animacions, pots millorar l'experiència de l'usuari afegint efectes dinàmics a les teves visualitzacions. Practica aquestes tècniques per dominar l'art d'estilitzar SVGs amb CSS en els teus projectes D3.js.
