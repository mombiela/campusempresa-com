Introducció a D3.js

D3.js (Documents impulsats per dades) és una potent biblioteca de JavaScript per crear visualitzacions de dades dinàmiques i interactives en navegadors web. Utilitza estàndards web com HTML, SVG i CSS per donar vida a les dades.

Conceptes clau

Seleccions

  • Seleccions: D3 utilitza seleccions per vincular dades a elements del DOM. Una selecció és un grup d'elements sobre els quals pots operar com una unitat única. 
    d3.select("body") // Selecciona el primer element <body>
d3.selectAll("p") // Selecciona tots els elements <p>

Vinculació de dades

  • Vinculació de dades: Aquest és el procés de connectar dades a elements del DOM. D3 proporciona mètodes com .data() i .enter() per facilitar això. 
    const data = [10, 20, 30];
d3.selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .text(d => d);

Escales

  • Escales: Les escales són funcions que mapegen d'un domini d'entrada a un rang de sortida. Són essencials per traduir valors de dades en representacions visuals. 
    const scale = d3.scaleLinear()
  .domain([0, 100])
  .range([0, 500]);
console.log(scale(50)); // Sortida: 250

Eixos

  • Eixos: D3 proporciona funcions integrades per crear i personalitzar eixos, que són essencials per a molts tipus de visualitzacions. 
    const xAxis = d3.axisBottom(scale);
d3.select("svg")
  .append("g")
  .attr("transform", "translate(0, 500)")
  .call(xAxis);

Elements SVG

  • Elements SVG: D3 utilitza gràfics vectorials escalables (SVG) per crear formes com cercles, rectangles i línies. 
    d3.select("svg")
  .append("circle")
  .attr("cx", 50)
  .attr("cy", 50)
  .attr("r", 40)
  .style("fill", "blue");

Transicions

  • Transicions: D3 et permet animar canvis a les teves visualitzacions amb transicions. 
    d3.select("circle")
  .transition()
  .duration(1000)
  .attr("r", 20);

Exemple: Creació d'un gràfic de barres simple

Posem aquests conceptes junts per crear un gràfic de barres simple.

  1. Configuració HTML:

    <!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Gràfic de barres simple</title>
  <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
  <svg width="500" height="300"></svg>
</body>
</html>

  2. Codi JavaScript:

    const data = [30, 86, 168, 281, 303, 365];
const svg = d3.select("svg");
const width = +svg.attr("width");
const height = +svg.attr("height");

const xScale = d3.scaleBand()
  .domain(d3.range(data.length))
  .range([0, width])
  .padding(0.1);

const yScale = d3.scaleLinear()
  .domain([0, d3.max(data)])
  .range([height, 0]);

svg.selectAll("rect")
  .data(data)
  .enter()
  .append("rect")
  .attr("x", (d, i) => xScale(i))
  .attr("y", d => yScale(d))
  .attr("width", xScale.bandwidth())
  .attr("height", d => height - yScale(d))
  .attr("fill", "steelblue");

Conclusió

Entendre els conceptes bàsics i la terminologia de D3.js és crucial per crear visualitzacions de dades efectives. Dominant les seleccions, la vinculació de dades, les escales, els eixos, els elements SVG i les transicions, pots construir una àmplia gamma de visualitzacions. En les següents seccions, aprofundirem en cadascun d'aquests conceptes i explorarem funcions més avançades de D3.js.
