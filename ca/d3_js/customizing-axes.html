El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, explorarem com personalitzar els eixos en D3.js, des de tècniques bàsiques fins a avançades. Els eixos són crucials per a la visualització de dades, ja que proporcionen punts de referència per interpretar les dades. Cobrirem com crear i estilitzar eixos, afegir etiquetes i personalitzar les marques de graduació.

Creació d'eixos bàsics

Abans de personalitzar els eixos, cal saber com crear-los. D3.js proporciona funcions integrades per generar eixos.

Exemple: Creació d'eixos bàsics

// Configurar les dimensions del llenç SVG
const width = 500;
const height = 300;
const margin = { top: 20, right: 30, bottom: 30, left: 40 };

// Crear un element SVG
const svg = d3.select("body").append("svg")
    .attr("width", width)
    .attr("height", height);

// Definir escales
const xScale = d3.scaleLinear()
    .domain([0, 100])
    .range([margin.left, width - margin.right]);

const yScale = d3.scaleLinear()
    .domain([0, 100])
    .range([height - margin.bottom, margin.top]);

// Definir eixos
const xAxis = d3.axisBottom(xScale);
const yAxis = d3.axisLeft(yScale);

// Afegir eixos al SVG
svg.append("g")
    .attr("transform", `translate(0,${height - margin.bottom})`)
    .call(xAxis);

svg.append("g")
    .attr("transform", `translate(${margin.left},0)`)
    .call(yAxis);

En aquest exemple, creem escales lineals per als eixos x i y i després utilitzem d3.axisBottom i d3.axisLeft per crear els eixos. Finalment, afegim aquests eixos a l'element SVG.

Personalització de les marques de graduació

Les marques de graduació són les petites línies als eixos que indiquen valors específics. Personalitzar-les pot millorar la llegibilitat del teu gràfic.

Exemple: Personalització de les marques de graduació

// Personalitzar el nombre de marques de graduació
const xAxis = d3.axisBottom(xScale)
    .ticks(10); // Especificar el nombre de marques de graduació

const yAxis = d3.axisLeft(yScale)
    .ticks(5); // Especificar el nombre de marques de graduació

// Afegir eixos personalitzats al SVG
svg.append("g")
    .attr("transform", `translate(0,${height - margin.bottom})`)
    .call(xAxis);

svg.append("g")
    .attr("transform", `translate(${margin.left},0)`)
    .call(yAxis);

En aquest exemple, especifiquem el nombre de marques de graduació per als eixos x i y utilitzant el mètode .ticks().

Afegir etiquetes als eixos

Les etiquetes ajuden els usuaris a entendre què representen els eixos. Pots afegir etiquetes tant als eixos x com y.

Exemple: Afegir etiquetes als eixos

// Afegir etiqueta a l'eix x
svg.append("text")
    .attr("class", "x label")
    .attr("text-anchor", "end")
    .attr("x", width - margin.right)
    .attr("y", height - 6)
    .text("Etiqueta Eix X");

// Afegir etiqueta a l'eix y
svg.append("text")
    .attr("class", "y label")
    .attr("text-anchor", "end")
    .attr("x", -margin.top)
    .attr("y", 6)
    .attr("dy", ".75em")
    .attr("transform", "rotate(-90)")
    .text("Etiqueta Eix Y");

En aquest exemple, utilitzem el mètode .append("text") per afegir etiquetes als eixos x i y. L'atribut transform s'utilitza per rotar l'etiqueta de l'eix y.

Personalitzacions avançades

Per a personalitzacions més avançades, pots modificar l'aparença de les marques de graduació, les línies de la graella i els camins dels eixos.

Exemple: Personalitzacions avançades

// Personalitzar marques de graduació i línies de la graella
const xAxis = d3.axisBottom(xScale)
    .ticks(10)
    .tickSize(-height + margin.top + margin.bottom)
    .tickPadding(10);

const yAxis = d3.axisLeft(yScale)
    .ticks(5)
    .tickSize(-width + margin.left + margin.right)
    .tickPadding(10);

// Afegir eixos personalitzats al SVG
svg.append("g")
    .attr("transform", `translate(0,${height - margin.bottom})`)
    .call(xAxis)
    .selectAll(".tick line")
    .attr("stroke", "#ccc");

svg.append("g")
    .attr("transform", `translate(${margin.left},0)`)
    .call(yAxis)
    .selectAll(".tick line")
    .attr("stroke", "#ccc");

// Personalitzar camins dels eixos
svg.selectAll(".domain")
    .attr("stroke", "#333");

En aquest exemple, utilitzem .tickSize() per estendre les línies de les marques de graduació a través del gràfic i .tickPadding() per ajustar l'espai entre les marques de graduació i les etiquetes. També personalitzem l'aparença de les línies de les marques de graduació i els camins dels eixos utilitzant .attr().

Conclusió

Personalitzar els eixos en D3.js et permet crear gràfics més llegibles i visualment atractius. Ajustant les marques de graduació, afegint etiquetes i modificant l'aparença dels eixos, pots millorar l'efectivitat general de les teves visualitzacions de dades. Practica aquestes tècniques per convertir-te en un expert en la creació d'eixos personalitzats per als teus projectes D3.js.
