Introducció a les Escales

En D3.js, les escales són funcions que mapegen d'un domini d'entrada a un rang de sortida. Són essencials per crear visualitzacions perquè et permeten transformar valors de dades en propietats visuals com la posició, la longitud i el color.

Conceptes Clau

  • Domini: El conjunt de valors d'entrada.
  • Rang: El conjunt de valors de sortida.
  • Tipus d'Escales: Diferents tipus d'escales per a diferents tipus de dades (per exemple, lineal, ordinal, temporal).

Tipus d'Escales

Escales Lineals

Les escales lineals s'utilitzen per a dades quantitatives contínues. Mapeen un domini d'entrada continu a un rang de sortida continu.

// Crear una escala lineal
const linearScale = d3.scaleLinear()
    .domain([0, 100])  // Domini d'entrada
    .range([0, 500]);  // Rang de sortida

// Utilitzar l'escala
console.log(linearScale(50));  // Sortida: 250

Escales Ordinals

Les escales ordinals s'utilitzen per a dades categòriques. Mapeen un domini discret a un rang discret.

// Crear una escala ordinal
const ordinalScale = d3.scaleOrdinal()
    .domain(['A', 'B', 'C'])  // Domini d'entrada
    .range([0, 100, 200]);    // Rang de sortida

// Utilitzar l'escala
console.log(ordinalScale('B'));  // Sortida: 100

Escales Temporals

Les escales temporals s'utilitzen per a dades basades en el temps. Mapeen un domini temporal continu a un rang de sortida continu.

// Crear una escala temporal
const timeScale = d3.scaleTime()
    .domain([new Date(2020, 0, 1), new Date(2020, 11, 31)])  // Domini d'entrada
    .range([0, 500]);  // Rang de sortida

// Utilitzar l'escala
console.log(timeScale(new Date(2020, 5, 15)));  // Sortida: 250

Personalitzar les Escales

Clamping

El clamping restringeix el rang de sortida al rang especificat, fins i tot si el valor d'entrada està fora del domini.

// Crear una escala lineal amb clamping
const clampedScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 500])
    .clamp(true);

// Utilitzar l'escala
console.log(clampedScale(150));  // Sortida: 500 (clamped)

Nice

El mètode nice amplia el domini a valors rodons, cosa que pot fer que les etiquetes dels eixos siguin més llegibles.

// Crear una escala lineal i aplicar nice
const niceScale = d3.scaleLinear()
    .domain([0.123, 99.987])
    .range([0, 500])
    .nice();

// Utilitzar l'escala
console.log(niceScale.domain());  // Sortida: [0, 100]

Exemples Pràctics

Crear un Gràfic de Barres

Utilitzar escales per crear un gràfic de barres simple.

// Dades de mostra
const data = [10, 20, 30, 40, 50];

// Crear escales
const xScale = d3.scaleBand()
    .domain(data.map((d, i) => i))
    .range([0, 500])
    .padding(0.1);

const yScale = d3.scaleLinear()
    .domain([0, d3.max(data)])
    .range([400, 0]);

// Crear contenidor SVG
const svg = d3.select('body').append('svg')
    .attr('width', 600)
    .attr('height', 500);

// Crear barres
svg.selectAll('rect')
    .data(data)
    .enter().append('rect')
    .attr('x', (d, i) => xScale(i))
    .attr('y', d => yScale(d))
    .attr('width', xScale.bandwidth())
    .attr('height', d => 400 - yScale(d))
    .attr('fill', 'steelblue');

Conclusió

Comprendre les escales en D3.js és crucial per a una visualització de dades efectiva. Les escales et permeten mapar valors de dades a propietats visuals, fent que els teus gràfics i diagrames siguin precisos i significatius. En dominar diferents tipus d'escales i les seves opcions de personalització, pots crear una àmplia gamma de visualitzacions que comuniquin efectivament les teves dades.
