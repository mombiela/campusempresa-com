Introducció als mapes coroplètics

Els mapes coroplètics són un tipus de mapa temàtic on les àrees es tenyeixen o es patronegen en proporció al valor d'una variable representada. Aquests mapes són útils per visualitzar dades a través de regions geogràfiques.

Conceptes clau

  • Dades geogràfiques: Dades que inclouen ubicacions geogràfiques, com ara països, estats o comtats.
  • Escales de colors: Una gamma de colors utilitzada per representar diferents valors al mapa.
  • GeoJSON: Un format per codificar una varietat d'estructures de dades geogràfiques utilitzant JavaScript Object Notation (JSON).

Configuració del teu entorn

Abans de crear un mapa coroplètic, assegura't de tenir les eines i biblioteques necessàries instal·lades.

Prerequisits

  • Comprensió bàsica d'HTML, CSS i JavaScript.
  • Familiaritat amb els conceptes bàsics de D3.js.

Instal·lació

Per començar, inclou D3.js en el teu projecte. Pots fer-ho afegint la següent etiqueta de script al teu fitxer HTML:

<script src="https://d3js.org/d3.v6.min.js"></script>

Creació d'un mapa coroplètic bàsic

Creem un mapa coroplètic simple utilitzant D3.js.

Pas 1: Preparar l'estructura HTML

Crea un fitxer HTML amb una estructura bàsica:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Mapa Coroplètic</title>
    <style>
        .map {
            width: 800px;
            height: 600px;
        }
    </style>
</head>
<body>
    <div class="map" id="map"></div>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <script src="script.js"></script>
</body>
</html>

Pas 2: Carregar dades GeoJSON

Al teu fitxer script.js, carrega les dades GeoJSON per a les regions geogràfiques que vols mostrar. Per a aquest exemple, utilitzarem un fitxer GeoJSON de mostra per als estats dels EUA.

d3.json('path/to/us-states.json').then(function(data) {
    console.log(data);
});

Pas 3: Crear un element SVG

Crea un element SVG per contenir el mapa:

const width = 800;
const height = 600;

const svg = d3.select("#map")
    .append("svg")
    .attr("width", width)
    .attr("height", height);

Pas 4: Definir una projecció i un generador de camins

Defineix una projecció per convertir coordenades geogràfiques a coordenades de pantalla i un generador de camins per dibuixar el mapa:

const projection = d3.geoAlbersUsa()
    .translate([width / 2, height / 2])
    .scale(1000);

const path = d3.geoPath()
    .projection(projection);

Pas 5: Dibuixar el mapa

Utilitza les dades GeoJSON per dibuixar el mapa:

d3.json('path/to/us-states.json').then(function(data) {
    svg.selectAll("path")
        .data(data.features)
        .enter()
        .append("path")
        .attr("d", path)
        .attr("fill", "#ccc")
        .attr("stroke", "#333");
});

Afegir dades al mapa

Per fer el mapa més informatiu, afegirem dades i utilitzarem escales de colors per representar-les.

Pas 1: Preparar les dades

Suposem que tenim un conjunt de dades que inclou valors per a cada estat:

const data = [
    { state: 'California', value: 100 },
    { state: 'Texas', value: 80 },
    // Afegeix més dades aquí
];

Pas 2: Crear una escala de colors

Crea una escala de colors per representar els valors de les dades:

const colorScale = d3.scaleQuantize()
    .domain([0, 100])
    .range(d3.schemeBlues[9]);

Pas 3: Fusionar dades amb GeoJSON

Fusiona les dades amb les característiques GeoJSON:

d3.json('path/to/us-states.json').then(function(geoData) {
    data.forEach(d => {
        const state = geoData.features.find(f => f.properties.name === d.state);
        if (state) {
            state.properties.value = d.value;
        }
    });

    svg.selectAll("path")
        .data(geoData.features)
        .enter()
        .append("path")
        .attr("d", path)
        .attr("fill", d => colorScale(d.properties.value))
        .attr("stroke", "#333");
});

Funcionalitats avançades

Millora el teu mapa coroplètic amb funcionalitats interactives i tècniques addicionals de visualització de dades.

Tooltips

Afegeix tooltips per mostrar els valors de les dades quan es passa el cursor per sobre de les regions:

const tooltip = d3.select("body").append("div")
    .attr("class", "tooltip")
    .style("opacity", 0);

svg.selectAll("path")
    .on("mouseover", function(event, d) {
        tooltip.transition()
            .duration(200)
            .style("opacity", .9);
        tooltip.html(d.properties.name + "<br/>" + d.properties.value)
            .style("left", (event.pageX + 5) + "px")
            .style("top", (event.pageY - 28) + "px");
    })
    .on("mouseout", function(d) {
        tooltip.transition()
            .duration(500)
            .style("opacity", 0);
    });

Llegendes

Afegeix una llegenda per explicar l'escala de colors:

const legend = svg.append("g")
    .attr("class", "legend")
    .attr("transform", "translate(20,20)");

const legendScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 300]);

const legendAxis = d3.axisBottom(legendScale)
    .ticks(5);

legend.selectAll("rect")
    .data(colorScale.range().map(color => {
        const d = colorScale.invertExtent(color);
        if (!d[0]) d[0] = legendScale.domain()[0];
        if (!d[1]) d[1] = legendScale.domain()[1];
        return d;
    }))
    .enter().append("rect")
    .attr("height", 20)
    .attr("x", d => legendScale(d[0]))
    .attr("width", d => legendScale(d[1]) - legendScale(d[0]))
    .attr("fill", d => colorScale(d[0]));

legend.append("g")
    .attr("transform", "translate(0,20)")
    .call(legendAxis);

Conclusió

Els mapes coroplètics són una eina poderosa per visualitzar dades geogràfiques. Seguint els passos descrits en aquesta guia, pots crear mapes informatius i interactius utilitzant D3.js. A mesura que et familiaritzis amb aquestes tècniques, pots explorar funcionalitats addicionals i personalitzacions per millorar encara més els teus mapes.
