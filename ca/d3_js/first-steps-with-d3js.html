Introducció a D3.js

D3.js (Documents Basats en Dades) és una potent biblioteca de JavaScript per crear visualitzacions de dades dinàmiques i interactives en navegadors web. Permet als desenvolupadors vincular dades a un Model d'Objectes del Document (DOM) i aplicar transformacions basades en dades al document.

Conceptes Clau

  • Seleccions: Seleccionar elements del DOM.
  • Vinculació de Dades: Associar dades amb elements del DOM.
  • Enter, Update, Exit: Gestionar canvis basats en dades en el DOM.
  • Escales: Mapar valors de dades a propietats visuals.
  • Eixos: Crear i personalitzar eixos per a visualitzacions.

Configuració del Teu Entorn

Abans de submergir-te en D3.js, necessites configurar el teu entorn de desenvolupament.

Passos per Configurar

  1. Instal·la un Editor de Codi: Utilitza editors com Visual Studio Code, Sublime Text o Atom.
  2. Inclou D3.js en el Teu Projecte:
    • Via CDN: 
      <script src="https://d3js.org/d3.v7.min.js"></script>
    • Via npm: 
      npm install d3

Seleccions Bàsiques i Vinculació de Dades

Les seleccions i la vinculació de dades són fonamentals per treballar amb D3.js.

Seleccionar Elements

Utilitza D3.js per seleccionar elements del DOM:

// Selecciona un sol element
d3.select("body").style("background-color", "lightblue");

// Selecciona múltiples elements
d3.selectAll("p").style("color", "blue");

Vincular Dades a Elements

Vincula dades a elements del DOM per crear visualitzacions dinàmiques:

// Dades d'exemple
const data = [10, 20, 30, 40, 50];

// Selecciona elements i vincula dades
d3.select("body").selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "teal")
  .style("margin", "5px");

Patró Enter, Update, i Exit

El patró enter, update, i exit és crucial per gestionar dades dinàmiques.

Selecció Enter

Gestiona nous punts de dades:

const data = [10, 20, 30];

const divs = d3.select("body").selectAll("div")
  .data(data);

divs.enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "teal")
  .style("margin", "5px");

Selecció Update

Actualitza elements existents:

const updatedData = [15, 25, 35];

const divs = d3.select("body").selectAll("div")
  .data(updatedData);

divs.style("width", d => `${d * 10}px`);

Selecció Exit

Elimina elements que ja no tenen dades corresponents:

const reducedData = [15, 25];

const divs = d3.select("body").selectAll("div")
  .data(reducedData);

divs.exit().remove();

Creació de Visualitzacions Bàsiques

Aprèn a crear visualitzacions bàsiques com gràfics de barres i diagrames de dispersió.

Exemple de Gràfic de Barres

const data = [10, 20, 30, 40, 50];

const svg = d3.select("body").append("svg")
  .attr("width", 500)
  .attr("height", 100);

svg.selectAll("rect")
  .data(data)
  .enter()
  .append("rect")
  .attr("x", (d, i) => i * 30)
  .attr("y", d => 100 - d)
  .attr("width", 25)
  .attr("height", d => d)
  .attr("fill", "teal");

Exemple de Diagrama de Dispersió

const data = [
  { x: 30, y: 20 },
  { x: 50, y: 80 },
  { x: 90, y: 50 },
  { x: 120, y: 70 },
  { x: 150, y: 30 }
];

const svg = d3.select("body").append("svg")
  .attr("width", 200)
  .attr("height", 100);

svg.selectAll("circle")
  .data(data)
  .enter()
  .append("circle")
  .attr("cx", d => d.x)
  .attr("cy", d => d.y)
  .attr("r", 5)
  .attr("fill", "teal");

Conclusió

En aquesta secció, has après els conceptes bàsics de D3.js, incloent la configuració del teu entorn, la realització de seleccions, la vinculació de dades i l'ús del patró enter, update, i exit. També has creat visualitzacions bàsiques com gràfics de barres i diagrames de dispersió. Aquestes habilitats fonamentals t'ajudaran a construir visualitzacions de dades més complexes i interactives a mesura que avancis en el curs.
