Introducció a D3.js

D3.js, o Data-Driven Documents, és una biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Aprofita la potència dels estàndards web moderns com HTML, SVG i CSS per donar vida a les dades.

Conceptes Clau

  • Enllaç de Dades: D3.js enllaça dades a un Document Object Model (DOM), permetent la creació de visualitzacions complexes.
  • Seleccions: D3.js utilitza seleccions per aplicar transformacions basades en dades als elements del DOM.
  • Enter, Update, Exit: D3.js proporciona un patró potent per gestionar el cicle de vida dels elements de dades.

Per què utilitzar D3.js?

D3.js és àmpliament utilitzat perquè proporciona:

  • Flexibilitat: A diferència d'altres biblioteques de gràfics, D3.js no ve amb gràfics preconstruïts. En canvi, proporciona les eines per construir qualsevol tipus de visualització.
  • Rendiment: D3.js és altament eficient, capaç de gestionar grans conjunts de dades de manera eficient.
  • Interactivitat: D3.js facilita l'addició d'elements interactius a les visualitzacions, millorant l'experiència de l'usuari.

Exemple Bàsic

Comencem amb un exemple senzill per il·lustrar com funciona D3.js. Crearem un gràfic de barres bàsic.

Configuració HTML

Primer, configureu un fitxer HTML bàsic:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple Bàsic de D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <div id="chart"></div>
</body>
</html>

Codi JavaScript

A continuació, afegiu el següent codi JavaScript per crear un gràfic de barres:

// Dades de mostra
const data = [30, 86, 168, 281, 303, 365];

// Selecciona el contenidor del gràfic
const chart = d3.select("#chart")
    .append("svg")
    .attr("width", 500)
    .attr("height", 300);

// Crea barres
chart.selectAll("rect")
    .data(data)
    .enter()
    .append("rect")
    .attr("width", 40)
    .attr("height", d => d)
    .attr("x", (d, i) => i * 45)
    .attr("y", d => 300 - d)
    .attr("fill", "steelblue");

Explicació

  • Enllaç de Dades: La matriu data està enllaçada als elements rect.
  • Seleccions: d3.select("#chart") selecciona el div amb l'id chart.
  • Enter, Update, Exit: El mètode enter() crea nous elements rect per a cada punt de dades.

Conclusió

D3.js és una biblioteca potent per crear visualitzacions de dades personalitzades. Comprenent els seus conceptes bàsics com l'enllaç de dades, les seleccions i el patró enter-update-exit, podeu crear una gran varietat de visualitzacions interactives i dinàmiques. Aquest exemple bàsic és només el començament; a mesura que avanceu en aquest curs, aprendreu a aprofitar tot el potencial de D3.js per crear visualitzacions sofisticades.
