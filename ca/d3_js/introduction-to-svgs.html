Què és SVG?

SVG (Scalable Vector Graphics) és un format basat en XML per descriure gràfics vectorials bidimensionals. A diferència de les imatges raster (per exemple, JPEG, PNG), els SVGs es poden escalar a qualsevol mida sense perdre qualitat, cosa que els fa ideals per al disseny web responsiu i la visualització de dades.

Característiques clau de SVG:

  • Escalabilitat: Les imatges SVG es poden escalar a qualsevol mida sense perdre qualitat.
  • Basat en text: Els fitxers SVG són basats en text, cosa que els fa fàcils d'editar i manipular.
  • Interactivitat: Els SVGs suporten interactivitat i animació.
  • Accessibilitat: Els SVGs es poden fer accessibles als lectors de pantalla i altres tecnologies assistives.

Estructura bàsica d'un SVG

Un fitxer SVG és essencialment un document XML amb una estructura específica. Aquí teniu un exemple senzill d'un fitxer SVG:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Explicació:

  • <svg>: L'element arrel d'un document SVG.
  • width i height: Defineixen les dimensions del llenç SVG.
  • xmlns: Especifica l'espai de noms XML per a SVG.
  • <circle>: Un element de forma que dibuixa un cercle.
    • cx i cy: Defineixen les coordenades del centre del cercle.
    • r: Defineix el radi del cercle.
    • stroke: Defineix el color de la vora del cercle.
    • stroke-width: Defineix l'amplada de la vora del cercle.
    • fill: Defineix el color de farciment del cercle.

Elements comuns de SVG

Aquí teniu alguns elements SVG comunament utilitzats:

  • : Dibuixa rectangles.
  • : Dibuixa cercles.
  • : Dibuixa el·lipses.
  • : Dibuixa línies rectes.
  • : Dibuixa una sèrie de línies rectes connectades.
  • : Dibuixa una forma tancada amb múltiples costats.
  • : Dibuixa formes complexes utilitzant una sèrie de comandes.

Exemple:

<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <rect x="10" y="10" width="50" height="50" stroke="blue" fill="yellow" />
  <ellipse cx="100" cy="100" rx="30" ry="20" stroke="green" fill="purple" />
  <line x1="150" y1="10" x2="150" y2="50" stroke="red" />
  <polyline points="10,150 50,180 90,150" stroke="black" fill="none" />
  <polygon points="120,150 160,180 200,150" stroke="black" fill="orange" />
  <path d="M10 200 C 20 100, 40 100, 50 200" stroke="brown" fill="none" />
</svg>

Estilització dels SVGs

Els elements SVG es poden estilitzar utilitzant CSS. Podeu aplicar estils directament dins del fitxer SVG o a través de CSS extern.

Estilització en línia:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" style="stroke: black; stroke-width: 3; fill: red;" />
</svg>

CSS extern:

<style>
  .my-circle {
    stroke: black;
    stroke-width: 3;
    fill: red;
  }
</style>

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" class="my-circle" />
</svg>

Utilitzant SVGs amb D3.js

D3.js (Data-Driven Documents) és una potent biblioteca de JavaScript per crear visualitzacions de dades dinàmiques i interactives utilitzant estàndards web com SVG.

Exemple: Creant un SVG amb D3.js

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Exemple de SVG amb D3.js</title>
  <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
  <script>
    // Crear un contenidor SVG
    const svg = d3.select("body")
                  .append("svg")
                  .attr("width", 200)
                  .attr("height", 200);

    // Afegir un cercle al SVG
    svg.append("circle")
       .attr("cx", 100)
       .attr("cy", 100)
       .attr("r", 50)
       .attr("stroke", "black")
       .attr("stroke-width", 3)
       .attr("fill", "red");
  </script>
</body>
</html>

Explicació:

  • d3.select("body").append("svg"): Selecciona l'element body i hi afegeix un element SVG.
  • .attr("width", 200).attr("height", 200): Defineix l'amplada i l'alçada del SVG.
  • svg.append("circle"): Afegeix un element cercle al SVG.
  • .attr("cx", 100).attr("cy", 100).attr("r", 50): Defineix les coordenades del centre del cercle i el radi.
  • .attr("stroke", "black").attr("stroke-width", 3).attr("fill", "red"): Defineix el color de la vora del cercle, l'amplada de la vora i el color de farciment.

Conclusió

Els SVGs són una eina versàtil i potent per crear gràfics escalables i interactius a la web. Entendre els conceptes bàsics de SVG és crucial per aprofitar tot el potencial de D3.js en la visualització de dades. Combinant SVG amb D3.js, podeu crear visualitzacions dinàmiques i basades en dades que són tant visualment atractives com altament interactives.
