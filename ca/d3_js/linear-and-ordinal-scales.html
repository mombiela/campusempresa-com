En aquesta secció, explorarem els conceptes d'escales lineals i ordinals en D3.js. Les escales són una part fonamental de D3.js, ja que permeten mapar valors de dades a dimensions visuals. Entendre com utilitzar les escales de manera efectiva és crucial per crear visualitzacions significatives.

Escales Lineals

Les escales lineals s'utilitzen per mapar dominis d'entrada continus a rangs de sortida continus. Són particularment útils quan es tracta de dades numèriques que necessiten ser representades proporcionalment.

Conceptes Clau

  • Domini: El conjunt de valors d'entrada.
  • Rang: El conjunt de valors de sortida.
  • Mapatge: El procés de convertir un valor del domini al rang.

Exemple

Creem una escala lineal simple que mapegi un domini de [0, 100] a un rang de [0, 500].

// Crear una escala lineal
const linearScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 500]);

// Provar l'escala
console.log(linearScale(0));   // Sortida: 0
console.log(linearScale(50));  // Sortida: 250
console.log(linearScale(100)); // Sortida: 500

Explicació

  • d3.scaleLinear(): Crea una nova escala lineal.
  • domain([0, 100]): Estableix el domini d'entrada per anar de 0 a 100.
  • range([0, 500]): Estableix el rang de sortida per anar de 0 a 500.

Escales Ordinals

Les escales ordinals s'utilitzen per mapar dominis d'entrada discrets a rangs de sortida discrets. Són útils quan es tracta de dades categòriques.

Conceptes Clau

  • Domini: El conjunt de valors d'entrada discrets.
  • Rang: El conjunt de valors de sortida discrets.
  • Mapatge: El procés de convertir un valor del domini al rang.

Exemple

Creem una escala ordinal que mapegi un domini de ["A", "B", "C"] a un rang de [0, 100, 200].

// Crear una escala ordinal
const ordinalScale = d3.scaleOrdinal()
    .domain(["A", "B", "C"])
    .range([0, 100, 200]);

// Provar l'escala
console.log(ordinalScale("A")); // Sortida: 0
console.log(ordinalScale("B")); // Sortida: 100
console.log(ordinalScale("C")); // Sortida: 200

Explicació

  • d3.scaleOrdinal(): Crea una nova escala ordinal.
  • domain(["A", "B", "C"]): Estableix el domini d'entrada als valors discrets "A", "B" i "C".
  • range([0, 100, 200]): Estableix el rang de sortida als valors discrets 0, 100 i 200.

Comparació d'Escales Lineals i Ordinals

| Característica | Escala Lineal | Escala Ordinal | |-----------------|------------------------------------|-------------------------------------| | Domini d'Entrada| Continu (p. ex., [0, 100]) | Discret (p. ex., ["A", "B", "C"]) | | Rang de Sortida | Continu (p. ex., [0, 500]) | Discret (p. ex., [0, 100, 200]) | | Ús | Dades numèriques | Dades categòriques | | Mapatge | Proporcional | No proporcional |

Conclusió

Entendre les escales lineals i ordinals és essencial per crear visualitzacions efectives en D3.js. Les escales lineals són ideals per a dades numèriques, proporcionant un mapatge proporcional de l'entrada a la sortida. Les escales ordinals, en canvi, són perfectes per a dades categòriques, mapant valors d'entrada discrets a valors de sortida discrets.

En dominar aquestes escales, pots assegurar-te que les teves dades es representen amb precisió en les teves visualitzacions, fent-les més informatives i atractives.
