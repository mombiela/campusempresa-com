Creació de Gràfics de Pastís
Creació de diagrames de dispersió ►

En aquesta secció, explorarem com crear gràfics de pastís utilitzant D3.js, una potent biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Començarem amb els conceptes bàsics i gradualment avançarem cap a temes més avançats.

Introducció als Gràfics de Pastís

Els gràfics de pastís són gràfics circulars dividits en sectors, cadascun representant una proporció del total. Són útils per mostrar dades que formen part d'un tot, com ara percentatges.

Conceptes Clau:

  • Generador d'Arcs: Una funció en D3.js que crea les dades del camí SVG per als arcs.
  • Generador de Pastís: Una funció en D3.js que calcula els angles d'inici i final per a cada porció del gràfic de pastís.
  • Elements SVG: Elements de Gràfics Vectorials Escalables utilitzats per dibuixar el gràfic de pastís.

Configuració del Teu Entorn

Abans de començar a crear gràfics de pastís, assegura't de tenir el següent:

  • Un coneixement bàsic d'HTML, CSS i JavaScript.
  • La biblioteca D3.js inclosa en el teu projecte.

Incloent D3.js

Pots incloure D3.js en el teu fitxer HTML afegint la següent etiqueta de script:

<script src="https://d3js.org/d3.v7.min.js"></script>

Creació d'un Gràfic de Pastís Bàsic

Pas 1: Preparar les Dades

Primer, necessitem preparar les dades que visualitzarem. Creem un conjunt de dades senzill:

const data = [10, 20, 30, 40, 50];

Pas 2: Configurar el Contenidor SVG

A continuació, configurem un contenidor SVG on es dibuixarà el gràfic de pastís:

const width = 400;
const height = 400;
const radius = Math.min(width, height) / 2;

const svg = d3.select("body")
  .append("svg")
  .attr("width", width)
  .attr("height", height)
  .append("g")
  .attr("transform", `translate(${width / 2}, ${height / 2})`);

Pas 3: Crear els Generadors de Pastís i Arc

Necessitem crear els generadors de pastís i arc:

const pie = d3.pie();

const arc = d3.arc()
  .innerRadius(0)
  .outerRadius(radius);

Pas 4: Dibuixar el Gràfic de Pastís

Ara, utilitzem els generadors de pastís i arc per dibuixar el gràfic de pastís:

const arcs = svg.selectAll("arc")
  .data(pie(data))
  .enter()
  .append("g")
  .attr("class", "arc");

arcs.append("path")
  .attr("d", arc)
  .attr("fill", (d, i) => d3.schemeCategory10[i]);

Afegint Etiquetes al Gràfic de Pastís

Per fer el gràfic de pastís més informatiu, podem afegir etiquetes a cada porció:

arcs.append("text")
  .attr("transform", d => `translate(${arc.centroid(d)})`)
  .attr("text-anchor", "middle")
  .text(d => d.data);

Personalitzacions Avançades

Afegint Interactivitat

Podem afegir interactivitat al gràfic de pastís, com ara ressaltar una porció en passar-hi el ratolí per sobre:

arcs.on("mouseover", function(event, d) {
  d3.select(this).select("path").transition()
    .duration(200)
    .attr("d", d3.arc().innerRadius(0).outerRadius(radius + 10));
})
.on("mouseout", function(event, d) {
  d3.select(this).select("path").transition()
    .duration(200)
    .attr("d", arc);
});

Utilitzant un Gràfic de Donut

Un gràfic de donut és una variació d'un gràfic de pastís amb un forat al centre. Podem crear un gràfic de donut establint el innerRadius del generador d'arc:

const arc = d3.arc()
  .innerRadius(radius / 2)
  .outerRadius(radius);

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la creació de gràfics de pastís utilitzant D3.js, des de la configuració de l'entorn fins al dibuix del gràfic i l'afegiment d'etiquetes. També hem explorat personalitzacions avançades com la interactivitat i la creació de gràfics de donut. Els gràfics de pastís són una manera poderosa de visualitzar relacions part-tot en les teves dades, i amb D3.js, pots crear gràfics altament personalitzables i interactius.
