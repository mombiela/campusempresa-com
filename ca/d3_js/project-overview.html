Introducció a D3.js

D3.js (Documents impulsats per dades) és una potent biblioteca de JavaScript utilitzada per crear visualitzacions de dades dinàmiques i interactives en navegadors web. Aquest curs et portarà des de nivells principiants fins a avançats, cobrint els conceptes fonamentals i les tècniques avançades necessàries per dominar D3.js.

Conceptes clau

  • Enllaç de dades: Aprèn a enllaçar dades a elements del DOM.
  • Seleccions: Entén com seleccionar i manipular elements del DOM.
  • Escales: Utilitza escales per mapar valors de dades a propietats visuals.
  • Eixos: Crea i personalitza eixos per a les teves visualitzacions.
  • Formes: Dibuixa formes bàsiques com línies, rectangles i cercles.
  • Transicions: Afegeix animacions i transicions a les teves visualitzacions.
  • Interactivitat: Fes que les teves visualitzacions siguin interactives amb oients d'esdeveniments.
  • Disposicions: Utilitza disposicions de D3 per a visualitzacions complexes com gràfics de sectors, gràfics dirigits per força i mapes d'arbres.

Exemple de codi

Aquí tens un exemple bàsic de creació d'un gràfic de barres simple amb D3.js:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Gràfic de barres simple</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
    </style>
</head>
<body>
    <script>
        // Dades de mostra
        const data = [30, 86, 168, 281, 303, 365];

        // Crea el contenidor SVG
        const svg = d3.select("body").append("svg")
            .attr("width", 500)
            .attr("height", 300);

        // Crea les barres
        svg.selectAll("rect")
            .data(data)
            .enter().append("rect")
            .attr("class", "bar")
            .attr("x", (d, i) => i * 50)
            .attr("y", d => 300 - d)
            .attr("width", 40)
            .attr("height", d => d);
    </script>
</body>
</html>

Explicació

  • Dades: Una matriu de números que representa les dades a visualitzar.
  • Contenidor SVG: Es crea un element SVG i s'afegeix al cos del document HTML.
  • Barres: Es creen rectangles per a cada punt de dades. L'atribut x posiciona les barres horitzontalment, l'atribut y les posiciona verticalment, i l'atribut height estableix la seva alçada basada en els valors de les dades.
