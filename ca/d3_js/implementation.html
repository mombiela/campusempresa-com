Introducció a D3.js

D3.js (Documents Basats en Dades) és una biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Utilitza HTML, SVG i CSS per donar vida a les dades. Aquesta secció et guiarà a través dels conceptes bàsics per implementar D3.js en els teus projectes.

Conceptes Clau

  • Seleccions: D3.js utilitza seleccions per vincular dades a elements DOM.
  • Vinculació de Dades: Associar dades amb elements.
  • Enter, Update, i Exit: Gestionar els canvis de dades.
  • Escales: Mapar valors de dades a valors visuals.
  • Eixos: Crear i estilitzar eixos per a les visualitzacions.
  • Transicions: Animar els canvis en la visualització.

Configuració del Teu Entorn

Abans d'endinsar-te en D3.js, assegura't de tenir un entorn de desenvolupament adequat configurat.

Passos per Configurar

  1. Incloure D3.js en el Teu Projecte:

    • Pots incloure D3.js a través d'un CDN: 
      <script src="https://d3js.org/d3.v7.min.js"></script>
    • O instal·lar-lo utilitzant npm: 
      npm install d3

  2. Crear una Plantilla HTML Bàsica:

    <!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple D3.js</title>
</head>
<body>
    <script src="https://d3js.org/d3.v7.min.js"></script>
    <script src="app.js"></script>
</body>
</html>

Implementació Bàsica

Crear un Gràfic de Barres Simple

  1. Preparar les Teves Dades:

    const data = [30, 86, 168, 281, 303, 365];

  2. Seleccionar el Contenidor i Vincular les Dades:

    const width = 500;
const height = 300;
const barWidth = width / data.length;

const svg = d3.select("body")
    .append("svg")
    .attr("width", width)
    .attr("height", height);

svg.selectAll("rect")
    .data(data)
    .enter()
    .append("rect")
    .attr("width", barWidth - 1)
    .attr("height", d => d)
    .attr("x", (d, i) => i * barWidth)
    .attr("y", d => height - d);

Explicació

  • Seleccions: d3.select i d3.selectAll s'utilitzen per seleccionar elements DOM.
  • Vinculació de Dades: data(data) vincula les dades als elements seleccionats.
  • Enter: enter() crea espais reservats per a nous elements de dades.
  • Atributs: attr() estableix atributs com l'amplada, l'alçada, les posicions x i y.

Implementació Intermèdia

Afegir Escales i Eixos

  1. Definir Escales:

    const xScale = d3.scaleBand()
    .domain(d3.range(data.length))
    .range([0, width])
    .padding(0.1);

const yScale = d3.scaleLinear()
    .domain([0, d3.max(data)])
    .range([height, 0]);

  2. Actualitzar el Gràfic de Barres amb Escales:

    svg.selectAll("rect")
    .data(data)
    .enter()
    .append("rect")
    .attr("x", (d, i) => xScale(i))
    .attr("y", d => yScale(d))
    .attr("width", xScale.bandwidth())
    .attr("height", d => height - yScale(d));

  3. Afegir Eixos:

    const xAxis = d3.axisBottom(xScale);
const yAxis = d3.axisLeft(yScale);

svg.append("g")
    .attr("transform", `translate(0, ${height})`)
    .call(xAxis);

svg.append("g")
    .call(yAxis);

Explicació

  • Escales: d3.scaleBand i d3.scaleLinear s'utilitzen per mapar valors de dades a valors visuals.
  • Eixos: d3.axisBottom i d3.axisLeft creen els eixos.

Implementació Avançada

Afegir Transicions i Interactivitat

  1. Afegir Transicions:

    svg.selectAll("rect")
    .data(data)
    .enter()
    .append("rect")
    .attr("x", (d, i) => xScale(i))
    .attr("y", height)
    .attr("width", xScale.bandwidth())
    .attr("height", 0)
    .transition()
    .duration(1000)
    .attr("y", d => yScale(d))
    .attr("height", d => height - yScale(d));

  2. Afegir Interactivitat:

    svg.selectAll("rect")
    .on("mouseover", function(event, d) {
        d3.select(this)
            .attr("fill", "taronja");
    })
    .on("mouseout", function(event, d) {
        d3.select(this)
            .attr("fill", "blau acer");
    });

Explicació

  • Transicions: transition() i duration() s'utilitzen per animar els canvis.
  • Interactivitat: on("mouseover") i on("mouseout") afegeixen oients d'esdeveniments per a la interactivitat.

Conclusió

Implementar D3.js implica entendre com vincular dades a elements DOM, utilitzar escales i eixos, i afegir transicions i interactivitat. Dominant aquests conceptes, pots crear visualitzacions de dades dinàmiques i interactives que donen vida a les teves dades. Practica amb diferents conjunts de dades i tipus de visualitzacions per aprofundir en la teva comprensió i competència amb D3.js.
