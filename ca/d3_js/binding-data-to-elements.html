Introducció

Vincular dades a elements és un concepte fonamental en D3.js, que et permet crear visualitzacions dinàmiques i interactives. Aquesta secció et guiarà a través dels conceptes bàsics de la vinculació de dades, des d'exemples simples fins a tècniques més avançades.

Conceptes Clau

  • Vinculació de Dades: El procés de connectar dades a elements del DOM.
  • Enter, Update, Exit: Les tres fases principals de la vinculació de dades en D3.js.
  • Unions de Dades: El mecanisme pel qual D3.js vincula dades a elements.

Vinculació Bàsica de Dades

Seleccionar Elements

Per vincular dades a elements, primer has de seleccionar els elements utilitzant els selectors de D3.js.

// Selecciona tots els elements de paràgraf
d3.selectAll("p");

Vincular Dades

Pots vincular dades als elements seleccionats utilitzant el mètode .data().

// Matriu de dades d'exemple
const data = [10, 20, 30, 40, 50];

// Vincula dades als elements de paràgraf
d3.selectAll("p")
  .data(data)
  .text(d => d);

En aquest exemple:

  • d3.selectAll("p") selecciona tots els elements de paràgraf.
  • .data(data) vincula la matriu de dades als elements seleccionats.
  • .text(d => d) estableix el contingut de text de cada paràgraf al valor de dades corresponent.

Enter, Update, Exit

Selecció Enter

La selecció enter representa els elements que necessiten ser creats per coincidir amb les dades.

const data = [10, 20, 30, 40, 50];

// Selecciona i vincula dades
const p = d3.select("body").selectAll("p")
  .data(data);

// Selecció enter
p.enter()
  .append("p")
  .text(d => d);

Selecció Update

La selecció update representa els elements que ja existeixen i necessiten ser actualitzats.

const data = [10, 20, 30, 40, 50];

// Selecciona i vincula dades
const p = d3.select("body").selectAll("p")
  .data(data);

// Selecció update
p.text(d => d);

Selecció Exit

La selecció exit representa els elements que necessiten ser eliminats perquè ja no tenen dades corresponents.

const data = [10, 20, 30];

// Selecciona i vincula dades
const p = d3.select("body").selectAll("p")
  .data(data);

// Selecció exit
p.exit().remove();

Vinculació Avançada de Dades

Funcions Clau

Les funcions clau ajuden D3.js a identificar quines dades corresponen a quins elements, especialment útils per a actualitzacions dinàmiques de dades.

const data = [
  { id: 1, value: 10 },
  { id: 2, value: 20 },
  { id: 3, value: 30 }
];

// Selecciona i vincula dades amb funció clau
const p = d3.select("body").selectAll("p")
  .data(data, d => d.id);

// Selecció enter
p.enter()
  .append("p")
  .text(d => d.value);

// Selecció update
p.text(d => d.value);

// Selecció exit
p.exit().remove();

Fusionar Seleccions

Pots fusionar les seleccions enter i update per aplicar operacions comunes a ambdues.

const data = [10, 20, 30, 40, 50];

// Selecciona i vincula dades
const p = d3.select("body").selectAll("p")
  .data(data);

// Selecció enter i update
p.enter()
  .append("p")
  .merge(p)
  .text(d => d);

Conclusió

Vincular dades a elements és un concepte central en D3.js que et permet crear visualitzacions dinàmiques i interactives. Entenent les seleccions enter, update i exit, així com utilitzant funcions clau i fusionant seleccions, pots gestionar i actualitzar efectivament les teves visualitzacions a mesura que les teves dades canvien. Practica aquests conceptes amb diferents conjunts de dades per convertir-te en un expert en la vinculació de dades amb D3.js.
