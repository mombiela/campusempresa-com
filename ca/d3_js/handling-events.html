La gestió d'esdeveniments és un aspecte crucial per crear visualitzacions interactives amb D3.js. Aquest tema et guiarà a través dels conceptes bàsics de la gestió d'esdeveniments, progressant cap a tècniques més avançades.

Introducció a la Gestió d'Esdeveniments

La gestió d'esdeveniments a D3.js et permet fer que les teves visualitzacions siguin interactives. Els esdeveniments poden ser accions de l'usuari com clics, moviments del ratolí o premses de tecles.

Conceptes Clau

  • Listeners d'Esdeveniments: Funcions que es criden quan ocorre un esdeveniment específic.
  • Tipus d'Esdeveniments: Diferents tipus d'esdeveniments com click, mouseover, mouseout, etc.
  • Objecte d'Esdeveniment: Conté informació sobre l'esdeveniment, com l'element objectiu i les coordenades del ratolí.

Afegint Listeners d'Esdeveniments

Per afegir un listener d'esdeveniments a una selecció de D3, utilitza el mètode on.

Exemple: Afegint un Esdeveniment de Clic

d3.select("circle")
  .on("click", function(event) {
    d3.select(this)
      .attr("fill", "red");
  });
  • Explicació: Aquest codi selecciona un element circle i canvia el seu color de farciment a vermell quan es fa clic sobre ell.

Exercici

  1. Selecciona un element rect i canvia la seva amplada quan es fa clic sobre ell.
  2. Selecciona un element text i canvia la seva mida de font en passar el ratolí per sobre.

Gestió de Diferents Tipus d'Esdeveniments

D3.js suporta diversos tipus d'esdeveniments. Aquí en tens alguns de comuns:

  • click
  • dblclick
  • mouseover
  • mouseout
  • mousemove

Exemple: Gestió de Mouseover i Mouseout

d3.select("rect")
  .on("mouseover", function(event) {
    d3.select(this)
      .attr("fill", "blue");
  })
  .on("mouseout", function(event) {
    d3.select(this)
      .attr("fill", "black");
  });
  • Explicació: Aquest codi canvia el color de farciment d'un element rect a blau en passar el ratolí per sobre i torna a negre en treure el ratolí.

Exercici

  1. Selecciona un element circle i canvia el seu radi en fer doble clic.
  2. Selecciona un element line i canvia el seu color de traç en moure el ratolí.

Utilitzant l'Objecte d'Esdeveniment

L'objecte d'esdeveniment proporciona informació útil sobre l'esdeveniment.

Exemple: Accedint a les Coordenades del Ratolí

d3.select("svg")
  .on("mousemove", function(event) {
    console.log("Coordenades del ratolí: ", d3.pointer(event));
  });
  • Explicació: Aquest codi registra les coordenades del ratolí relatives a l'element SVG cada vegada que es mou el ratolí.

Exercici

  1. Registra les coordenades d'un esdeveniment de clic sobre un element rect.
  2. Mostra les coordenades d'un esdeveniment de moviment del ratolí dins d'un element text.

Gestió Avançada d'Esdeveniments

Per a interaccions més complexes, potser necessitaràs gestionar múltiples esdeveniments o utilitzar listeners d'esdeveniments personalitzats.

Exemple: Arrossegar i Deixar Caure

const drag = d3.drag()
  .on("start", function(event) {
    d3.select(this).raise().attr("stroke", "black");
  })
  .on("drag", function(event) {
    d3.select(this)
      .attr("cx", event.x)
      .attr("cy", event.y);
  })
  .on("end", function(event) {
    d3.select(this).attr("stroke", null);
  });

d3.select("circle").call(drag);
  • Explicació: Aquest codi permet la funcionalitat d'arrossegar i deixar caure per a un element circle. La posició del cercle s'actualitza mentre es desplaça.

Exercici

  1. Implementa arrossegar i deixar caure per a un element rect.
  2. Crea un listener d'esdeveniments personalitzat que canviï el color d'un element quan es desplaça.

Resum

En aquest tema, has après com gestionar esdeveniments a D3.js, des de listeners d'esdeveniments bàsics fins a interaccions avançades com arrossegar i deixar caure. La gestió d'esdeveniments és essencial per crear visualitzacions interactives i dinàmiques. Practica els exercicis per reforçar la teva comprensió i experimenta amb diferents tipus d'esdeveniments i gestors per crear visualitzacions més atractives.
