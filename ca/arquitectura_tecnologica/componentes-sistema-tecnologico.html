Introducció

En aquesta secció del curs sobre Arquitectura Tecnològica, explorarem els components fonamentals d'un sistema tecnològic. Comprendre aquests components és crucial per dissenyar, implementar i mantenir sistemes eficients i escalables.

Components Principals

Maquinari

El maquinari és la base física de qualsevol sistema tecnològic. Inclou tots els dispositius i equips físics que són necessaris per al funcionament del sistema.

  • Servidors: Màquines que proporcionen serveis a altres programes o dispositius.
  • Dispositius d'emmagatzematge: Unitats de disc dur (HDD), unitats d'estat sòlid (SSD), i sistemes d'emmagatzematge en xarxa (NAS).
  • Xarxes: Routers, switches, i altres dispositius que faciliten la comunicació entre diferents parts del sistema.
  • Dispositius d'entrada/sortida: Teclats, ratolins, monitors, impressores, etc.

Programari

El programari és el conjunt d'instruccions que diu al maquinari què fer. Es pot dividir en diverses categories:

  • Sistemes Operatius: Programari que gestiona el maquinari i proporciona serveis per a altres programes. Exemples inclouen Windows, Linux, i macOS.
  • Aplicacions: Programes dissenyats per realitzar tasques específiques. Exemples inclouen navegadors web, processadors de text, i sistemes de gestió de bases de dades.
  • Middleware: Programari que connecta diferents aplicacions o serveis. Exemples inclouen servidors d'aplicacions i sistemes de missatgeria.

Xarxes

Les xarxes permeten la comunicació entre diferents components del sistema tecnològic. Poden ser locals (LAN) o globals (WAN).

  • Protocol d'Internet (IP): Defineix com s'envien i reben les dades a través de la xarxa.
  • Protocol de Transferència d'Hipertext (HTTP/HTTPS): Utilitzat per a la comunicació a la web.
  • Protocol de Control de Transmissió (TCP): Proporciona una comunicació fiable i ordenada entre aplicacions.

Bases de Dades

Les bases de dades són sistemes que emmagatzemen i gestionen dades de manera estructurada.

  • Bases de Dades Relacionals (RDBMS): Utilitzen taules per emmagatzemar dades i SQL per gestionar-los. Exemples inclouen MySQL, PostgreSQL, i Oracle.
  • Bases de Dades NoSQL: Utilitzen diferents models de dades com documents, grafos, i parells clau-valor. Exemples inclouen MongoDB, Cassandra, i Redis.

Seguretat

La seguretat és un component crític que protegeix el sistema i les dades contra accessos no autoritzats i atacs.

  • Autenticació i Autorització: Verificació d'identitat i control d'accés.
  • Xifrat: Protecció de dades mitjançant tècniques de xifrat.
  • Firewalls: Sistemes que controlen el trànsit de xarxa basat en regles de seguretat.

Exemple de Codi

A continuació, es mostra un exemple de com es pot configurar una connexió a una base de dades en Python utilitzant una base de dades relacional com PostgreSQL.

import psycopg2

try:
    # Connexió a la base de dades
    connection = psycopg2.connect(
        user="el_teu_usuari",
        password="la_teva_contrasenya",
        host="127.0.0.1",
        port="5432",
        database="la_teva_base_de_dades"
    )

    cursor = connection.cursor()
    # Executar una consulta
    cursor.execute("SELECT version();")
    record = cursor.fetchone()
    print("Connectat a - ", record, "\n")

except (Exception, psycopg2.Error) as error:
    print("Error en connectar a la base de dades", error)
finally:
    # Tancar la connexió
    if connection:
        cursor.close()
        connection.close()
        print("Connexió a la base de dades tancada")

Conclusió

Comprendre els components d'un sistema tecnològic és essencial per a qualsevol professional de TI. Des del maquinari i el programari fins a les xarxes i la seguretat, cada component juga un paper crucial en el funcionament del sistema. A mesura que avancem en el curs, aprofundirem en cadascun d'aquests components per entendre millor com s'integren i funcionen junts.
