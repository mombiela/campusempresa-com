Exemple Pràctic: Desplegament d'una Aplicació Web en Google App Engine

Introducció

En aquesta secció, aprendrem a desplegar una aplicació web en Google App Engine (GAE), una plataforma com a servei (PaaS) que permet als desenvolupadors construir i desplegar aplicacions sense preocupar-se per la infraestructura subjacent. Aquest exemple pràctic és ideal per entendre com funciona una PaaS i com pot facilitar el procés de desplegament i escalabilitat d'aplicacions web.

Requisits Previs

Abans de començar, assegura't de tenir el següent:

  • Un compte de Google Cloud Platform (GCP).
  • Google Cloud SDK instal·lat a la teva màquina.
  • Coneixements bàsics de Python i Flask (per a aquest exemple utilitzarem una aplicació web senzilla en Flask).

Pas 1: Configuració de l'Entorn

  1. Instal·lar Google Cloud SDK:

    • Descarrega i instal·la el SDK des de Google Cloud SDK.
    • Inicialitza el SDK amb el comandament: 
      gcloud init

  2. Crear un Projecte en GCP:

    • Ves a la Consola de Google Cloud.
    • Crea un nou projecte i pren nota de l'ID del projecte.

  3. Configurar el Projecte en el SDK:

    • Configura el projecte en el SDK amb el comandament: 
      gcloud config set project [YOUR_PROJECT_ID]

Pas 2: Crear una Aplicació Web Senzilla

Anem a crear una aplicació web senzilla utilitzant Flask.

  1. Instal·lar Flask:

    • Assegura't de tenir pip instal·lat i executa: 
      pip install Flask

  2. Crear l'Aplicació:

    • Crea un fitxer anomenat main.py amb el següent contingut: 
      from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Pas 3: Configurar Google App Engine

  1. Crear el Fitxer app.yaml:

    • Aquest fitxer li diu a GAE com configurar i desplegar l'aplicació.
    • Crea un fitxer app.yaml amb el següent contingut: 
      runtime: python39

handlers:
- url: /.*
  script: auto

  2. Estructura del Projecte:

    • Assegura't que el teu projecte tingui la següent estructura: 
      my-flask-app/
├── app.yaml
└── main.py

Pas 4: Desplegar l'Aplicació

  1. Desplegar en Google App Engine:

    • Executa el següent comandament per desplegar la teva aplicació: 
      gcloud app deploy

  2. Verificar el Desplegament:

    • Una vegada completat el desplegament, pots accedir a la teva aplicació a la URL proporcionada per GAE. Generalment, serà alguna cosa com https://[YOUR_PROJECT_ID].appspot.com.

Conclusió

En aquest exemple pràctic, hem cobert els passos necessaris per desplegar una aplicació web senzilla en Google App Engine. Aquest procés il·lustra com una PaaS com GAE pot simplificar el desplegament i la gestió d'aplicacions web, permetent als desenvolupadors centrar-se en el codi i la funcionalitat en lloc de la infraestructura.

Recorda que GAE ofereix moltes més funcionalitats avançades, com l'escalabilitat automàtica, la integració amb altres serveis de Google Cloud i opcions de configuració més detallades. Explorar aquestes característiques pot ajudar-te a aprofitar al màxim aquesta poderosa plataforma.
