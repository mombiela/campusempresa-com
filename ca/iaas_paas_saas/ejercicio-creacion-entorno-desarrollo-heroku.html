Exercici: Creació d'un Entorn de Desenvolupament a Heroku

Introducció

En aquest exercici, aprendràs a crear un entorn de desenvolupament a Heroku, una plataforma com a servei (PaaS) que permet als desenvolupadors construir, executar i operar aplicacions completament al núvol. Heroku simplifica el procés de desplegament i gestió d'aplicacions, permetent als desenvolupadors centrar-se en l'escriptura de codi.

Objectius de l'Exercici

  • Configurar un compte a Heroku.
  • Instal·lar i configurar la CLI de Heroku.
  • Crear i desplegar una aplicació bàsica a Heroku.
  • Gestionar l'entorn de desenvolupament a Heroku.

Requisits Previs

Abans de començar, assegura't de tenir:

  • Un compte a Heroku (pots registrar-te a Heroku).
  • Git instal·lat a la teva màquina.
  • Node.js i npm instal·lats (per a l'exemple d'aplicació).

Passos per Crear un Entorn de Desenvolupament a Heroku

1. Configuració del Compte a Heroku

  1. Registre a Heroku:

    • Visita Heroku.
    • Fes clic a "Sign Up" i segueix les instruccions per crear un compte.

  2. Verificació del Compte:

    • Verifica el teu compte a través del correu electrònic enviat per Heroku.

2. Instal·lació de la CLI de Heroku

  1. Descarrega i Instal·lació:

    • Descarrega la CLI de Heroku des de Heroku CLI.
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  2. Verificació de la Instal·lació:

    • Obre una terminal i executa el següent comandament per verificar la instal·lació: 
      heroku --version
    • Hauries de veure la versió de la CLI de Heroku instal·lada.

3. Creació i Desplegament d'una Aplicació Bàsica

  1. Inici de Sessió a Heroku:

    • A la terminal, executa: 
      heroku login
    • Segueix les instruccions per iniciar sessió al teu compte de Heroku.

  2. Creació d'una Aplicació Node.js:

    • Crea un nou directori per a la teva aplicació i navega-hi: 
      mkdir my-heroku-app
cd my-heroku-app
    • Inicialitza un nou projecte Node.js: 
      npm init -y
    • Instal·la Express.js: 
      npm install express
    • Crea un fitxer index.js amb el següent contingut: 
      const express = require('express');
const app = express();
const PORT = process.env.PORT || 3000;

app.get('/', (req, res) => {
  res.send('Hello, Heroku!');
});

app.listen(PORT, () => {
  console.log(`Server is running on port ${PORT}`);
});

  3. Preparació per al Desplegament:

    • Crea un fitxer Procfile al directori arrel amb el següent contingut: 
      web: node index.js
    • Inicialitza un repositori Git i fes un commit: 
      git init
git add .
git commit -m "Initial commit"

  4. Desplegament a Heroku:

    • Crea una nova aplicació a Heroku: 
      heroku create
    • Desplega l'aplicació: 
      git push heroku master

  5. Verificació del Desplegament:

    • Un cop completat el desplegament, obre l'aplicació al teu navegador: 
      heroku open

4. Gestió de l'Entorn de Desenvolupament

  1. Veure Logs de l'Aplicació:

    • Per veure els logs de la teva aplicació, executa: 
      heroku logs --tail

  2. Escalat de Dynos:

    • Per escalar el nombre de dynos (instàncies de la teva aplicació), usa: 
      heroku ps:scale web=1

  3. Configuració de Variables d'Entorn:

    • Per configurar variables d'entorn, usa: 
      heroku config:set MY_VAR=value

Conclusió

En aquest exercici, has après a configurar un entorn de desenvolupament a Heroku, des de la creació d'un compte fins al desplegament i gestió d'una aplicació bàsica. Heroku facilita el procés de desplegament i gestió d'aplicacions, permetent als desenvolupadors centrar-se en l'escriptura de codi i en la millora de les seves aplicacions.
