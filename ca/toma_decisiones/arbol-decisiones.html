Introducció

L'Arbre de Decisions és una eina visual i analítica que ajuda a prendre decisions complexes de manera estructurada. S'utilitza per mapar possibles resultats, costos, utilitats i probabilitats de diferents opcions. Aquesta tècnica és especialment útil en entorns laborals on les decisions han de ser justificades i basades en dades.

Conceptes Clau

  • Node de Decisió: Representa un punt on s'ha de prendre una decisió. Es dibuixa com un quadrat.
  • Node d'Oportunitat: Representa un punt on ocorre un esdeveniment aleatori. Es dibuixa com un cercle.
  • Branca: Representa una possible acció o resultat que segueix a un node. Es dibuixa com una línia.
  • Fulla: Representa el resultat final d'una sèrie de decisions i esdeveniments. Es dibuixa com un triangle.

Passos per Crear un Arbre de Decisions

  1. Identificar la decisió a prendre: Defineix clarament el problema o la decisió que necessites analitzar.
  2. Dibuixar el node de decisió: Col·loca un quadrat al punt inicial del teu diagrama.
  3. Afegir branques per a cada opció: Dibuixa línies que surtin del node de decisió, cadascuna representant una opció diferent.
  4. Afegir nodes d'oportunitat: Als extrems de les branques, col·loca cercles per representar esdeveniments aleatoris que poden ocórrer.
  5. Continuar el procés: Repeteix els passos anteriors fins que totes les possibles decisions i esdeveniments estiguin representats.
  6. Assignar valors i probabilitats: Afegeix costos, utilitats i probabilitats a cada branca.
  7. Calcular el valor esperat: Per a cada node de decisió, calcula el valor esperat per determinar la millor opció.

Exemple Pràctic

Suposem que una empresa ha de decidir si llançar un nou producte. Les opcions són: llançar el producte o no llançar-lo. Si es llança, hi ha dos possibles resultats: èxit o fracàs.

| Decisió | Esdeveniment Aleatori | Probabilitat | Utilitat | |-------------------|-----------------------|--------------|----------| | Llançar Producte | Èxit | 0.7 | $100,000 | | | Fracàs | 0.3 | -$50,000 | | No Llançar Producte| N/A | 1.0 | $0 |

Arbre de Decisions Simplificat

  1. Node de Decisió: Llançar o no llançar el producte.

  2. Branques:

    • Llançar Producte
      • Node d'Oportunitat: Èxit (70%) o Fracàs (30%)
    • No Llançar Producte

  3. Resultats:

    • Èxit: $100,000
    • Fracàs: -$50,000
    • No Llançar: $0

Càlcul del Valor Esperat

  • Llançar Producte: (0.7 * $100,000) + (0.3 * -$50,000) = $70,000 - $15,000 = $55,000
  • No Llançar Producte: $0

Atès que el valor esperat de llançar el producte ($55,000) és més gran que no llançar-lo ($0), la millor decisió seria llançar el producte.

Conclusió

L'Arbre de Decisions és una eina poderosa per visualitzar i analitzar decisions complexes. En descompondre una decisió en els seus components bàsics i avaluar les probabilitats i utilitats de cada resultat, es pot prendre una decisió més informada i justificada. Aquesta tècnica no només facilita la presa de decisions, sinó que també proporciona una base sòlida per justificar les decisions preses davant altres stakeholders.
