Introducció als mapes de fonts

Els mapes de fonts són una eina crucial per a la depuració en el desenvolupament web modern. Mapegen el codi transformat i minimitzat al codi font original, facilitant la depuració.

  • Propòsit: Ajudar els desenvolupadors a depurar el seu codi font original fins i tot després que hagi estat transformat o minimitzat.
  • Ús: Comunament utilitzats conjuntament amb eines de construcció com Webpack.

Com funcionen els mapes de fonts

Els mapes de fonts creen un mapatge entre el codi font original i el codi transformat.

  • Codi font original: El codi que escrius.
  • Codi transformat: El codi després que hagi estat processat per eines com Babel, TypeScript, o minimitzat per Webpack.
  • Fitxer de mapa de fonts: Un fitxer que mapega el codi transformat al codi font original.

Exemple

Considera el següent codi JavaScript original:

// src/index.js
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet('World');

Després de la minimització, podria semblar així:

function greet(n){console.log("Hello, "+n+"!")}greet("World");

Un fitxer de mapa de fonts maparà el codi minimitzat al fitxer original src/index.js.

Configuració de mapes de fonts a Webpack

Webpack pot generar mapes de fonts per ajudar amb la depuració.

Configuració bàsica

Per habilitar els mapes de fonts a Webpack, necessites establir la propietat devtool al teu fitxer de configuració de Webpack.

// webpack.config.js
module.exports = {
    mode: 'development',
    devtool: 'source-map',
    entry: './src/index.js',
    output: {
        filename: 'bundle.js',
        path: __dirname + '/dist'
    }
};
  • mode: Establert a 'development' per habilitar una millor depuració.
  • devtool: Establert a 'source-map' per generar un fitxer de mapa de fonts.

Diferents opcions de mapes de fonts

Webpack ofereix diverses opcions per als mapes de fonts, cadascuna amb diferents compromisos entre la velocitat de construcció i la qualitat dels mapes de fonts.

| Opció | Descripció | Velocitat de construcció | Qualitat dels mapes de fonts | |----------------------|---------------------------------------------------------------------------|--------------------------|------------------------------| | eval | Cada mòdul s'executa amb eval(), i //@ sourceURL. | Més ràpid | Pobra | | source-map | Es genera un fitxer de mapa de fonts complet separat. | Lenta | Excel·lent | | cheap-source-map | Un mapa de fonts sense mapatges de columna, ignorant els mapes de fonts del carregador. | Ràpid | Bona | | inline-source-map | El mapa de fonts està inclòs com a DataUrl. | Lenta | Excel·lent | | cheap-module-source-map | Similar a cheap-source-map, però inclou els mapes de fonts del carregador. | Ràpid | Bona |

Exemple amb diferents opcions

// webpack.config.js
module.exports = {
    mode: 'development',
    devtool: 'cheap-module-source-map', // Canvia això per experimentar amb diferents opcions
    entry: './src/index.js',
    output: {
        filename: 'bundle.js',
        path: __dirname + '/dist'
    }
};

Depuració amb mapes de fonts

Un cop configurats els mapes de fonts, pots utilitzar-los a les eines de desenvolupador del teu navegador.

  • Chrome DevTools: Obre les DevTools (F12 o clic dret > Inspecciona), ves a la pestanya "Sources", i veuràs els teus fitxers de codi font originals.
  • Firefox Developer Tools: Obre les Developer Tools (F12 o clic dret > Inspecciona l'element), ves a la pestanya "Debugger", i veuràs els teus fitxers de codi font originals.

Exemple de sessió de depuració

  1. Obre la teva aplicació web al navegador.
  2. Obre les eines de desenvolupador (F12).
  3. Navega a la pestanya "Sources" o "Debugger".
  4. Estableix punts de ruptura al teu codi font original.
  5. Recarrega la pàgina i veu com els punts de ruptura es disparen al codi font original.

Conclusió

Els mapes de fonts són una eina essencial per a la depuració d'aplicacions web modernes. Configurant Webpack per generar mapes de fonts, pots depurar el teu codi font original fins i tot després que hagi estat transformat o minimitzat. Experimenta amb diferents opcions de mapes de fonts per trobar l'equilibri adequat entre la velocitat de construcció i la qualitat de la depuració per al teu projecte.
