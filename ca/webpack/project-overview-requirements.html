Introducció

En aquesta secció, proporcionarem una visió general del projecte en què treballaràs al llarg d'aquest curs de Webpack. També esbossarem els requisits necessaris per completar amb èxit el projecte. Això t'ajudarà a entendre l'abast del projecte i assegurar-te que tens tot el que necessites abans d'entrar en els detalls tècnics.

Visió general del projecte

El projecte per a aquest curs és una aplicació web senzilla que evolucionarà en complexitat a mesura que avancem des de nivells bàsics fins a avançats. Al final del curs, tindràs una aplicació web totalment optimitzada i llesta per a producció.

Característiques clau del projecte

  • Gestió d'actius estàtics: Gestiona eficientment fitxers CSS, JavaScript i imatges.
  • Agrupació de mòduls: Agrupa mòduls JavaScript per a un millor rendiment.
  • Divisió de codi: Divideix el codi en trossos més petits per millorar els temps de càrrega.
  • Servidor de desenvolupament: Configura un servidor de desenvolupament local amb reemplaçament de mòduls en calent.
  • Optimització per a producció: Optimitza l'aplicació per a producció amb minificació i sacsejada d'arbre.

Requisits

Abans de començar el projecte, assegura't de tenir els següents requisits previs i eines instal·lades:

Requisits previs

  • Coneixements bàsics de JavaScript: Comprensió de la sintaxi ES6+.
  • HTML i CSS bàsics: Familiaritat amb la creació i estilització de pàgines web.
  • Node.js i npm: Assegura't de tenir Node.js i npm instal·lats a la teva màquina. Pots descarregar-los des del lloc web oficial de Node.js.

Eines i biblioteques

  • Editor de codi: Un editor de codi com Visual Studio Code, Sublime Text o Atom.
  • Navegador: Un navegador web modern com Google Chrome o Firefox per a proves.
  • Terminal: Accés a un terminal o símbol del sistema per executar comandes.

Configuració inicial

  1. Crea un directori de projecte: Crea un nou directori per al teu projecte.

    mkdir webpack-project
cd webpack-project

  2. Inicialitza npm: Inicialitza un nou projecte npm.

    npm init -y

    Això crearà un fitxer package.json al teu directori de projecte.

  3. Instal·la Webpack i Webpack CLI: Instal·la Webpack i la seva interfície de línia de comandes.

    npm install --save-dev webpack webpack-cli

Estructura bàsica del projecte

Aquí tens una estructura bàsica del directori del projecte:

webpack-project/
├── src/
│   ├── index.js
│   └── style.css
├── dist/
├── package.json
└── webpack.config.js
  • src/: Aquest directori contindrà els teus fitxers font.
    • index.js: El fitxer principal de JavaScript.
    • style.css: El fitxer principal de CSS.
  • dist/: Aquest directori contindrà els fitxers de sortida agrupats.
  • package.json: Conté metadades del projecte i dependències.
  • webpack.config.js: El fitxer de configuració per a Webpack.

Exemple de fitxer de configuració

Aquí tens un exemple bàsic d'un fitxer webpack.config.js:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist')
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
  • entry: Especifica el punt d'entrada de l'aplicació.
  • output: Defineix el fitxer i directori de sortida.
  • module: Conté regles per a com s'han de tractar diferents tipus de mòduls.

Conclusió

En aquesta secció, hem cobert la visió general del projecte i els requisits necessaris per començar. Assegura't de tenir tots els requisits previs i eines instal·lades abans de passar a les següents seccions. Amb la configuració inicial completa, ara estàs llest per endinsar-te en el món de Webpack i començar a construir la teva aplicació web.
