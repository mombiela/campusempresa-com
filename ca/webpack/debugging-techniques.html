Introducció

Depurar és una habilitat essencial per a qualsevol desenvolupador, i Webpack, sent un potent empaquetador de mòduls, ve amb el seu propi conjunt d'eines i tècniques de depuració. Aquesta secció et guiarà a través de diverses metodologies de depuració, des de nivells bàsics fins a avançats, per ajudar-te a solucionar i resoldre problemes de manera eficient en els teus projectes Webpack.

Tècniques Bàsiques de Depuració

Comprendre Errors i Advertències de Webpack

  • Errors: Aquests són problemes crítics que impedeixen que Webpack completi el procés de construcció.
  • Advertències: Aquests són problemes no crítics que no aturen la construcció però indiquen problemes potencials.

Exemple d'un error de Webpack:

ERROR in ./src/index.js
Module not found: Error: Can't resolve './nonexistent-file' in '/path/to/project/src'

Ús de Mapes de Fonts

Els mapes de fonts t'ajuden a mapar el codi empaquetat al codi font original, facilitant la depuració.

  • Habilitar Mapes de Fonts: Afegeix el següent al teu webpack.config.js:

    module.exports = {
  devtool: 'source-map',
  // altres configuracions
};

  • Inspeccionar Mapes de Fonts: Utilitza les eines de desenvolupador del navegador per inspeccionar el codi font original.

Analitzar la Sortida de la Construcció

  • Sortida Verbosa: Executa Webpack amb el flag --verbose per obtenir informació detallada sobre el procés de construcció. 
    webpack --verbose

Tècniques Intermèdies de Depuració

Ús del Servidor de Desenvolupament de Webpack

El Servidor de Desenvolupament de Webpack proporciona un servidor de desenvolupament amb recàrrega en viu, facilitant la depuració de canvis en temps real.

  • Configurar el Servidor de Desenvolupament: Afegeix el següent al teu webpack.config.js:

    const path = require('path');

module.exports = {
  // altres configuracions
  devServer: {
    contentBase: path.join(__dirname, 'dist'),
    compress: true,
    port: 9000,
    hot: true,
  },
};

  • Iniciar el Servidor de Desenvolupament: Executa la següent comanda:

    webpack serve

Depuració de Carregadors i Plugins

  • Depuració de Carregadors: Utilitza l'opció debug en els carregadors per obtenir més informació.

    module: {
  rules: [
    {
      test: /\.js$/,
      use: [
        {
          loader: 'babel-loader',
          options: {
            debug: true,
          },
        },
      ],
    },
  ],
};

  • Depuració de Plugins: Utilitza l'opció debug en els plugins si està disponible, o afegeix registres personalitzats.

    const MyPlugin = require('my-plugin');

module.exports = {
  plugins: [
    new MyPlugin({
      debug: true,
    }),
  ],
};

Tècniques Avançades de Depuració

Perfilat del Rendiment de la Construcció

El perfilat t'ajuda a entendre on el procés de construcció està gastant temps i recursos.

  • Analitzador de Paquets de Webpack: Utilitza aquest plugin per visualitzar la mida dels fitxers de sortida de Webpack.

    const { BundleAnalyzerPlugin } = require('webpack-bundle-analyzer');

module.exports = {
  plugins: [
    new BundleAnalyzerPlugin(),
  ],
};

  • Plugin de Mesura de Velocitat: Mesura la velocitat de construcció de la teva configuració de Webpack.

    const SpeedMeasurePlugin = require('speed-measure-webpack-plugin');
const smp = new SpeedMeasurePlugin();

module.exports = smp.wrap({
  // la teva configuració de webpack
});

Registres Personalitzats

Afegeix registres personalitzats a la teva configuració de Webpack per obtenir més informació durant el procés de construcció.

  • Exemple: Afegeix un plugin personalitzat per registrar missatges. 
    class CustomLoggerPlugin {
  apply(compiler) {
    compiler.hooks.done.tap('CustomLoggerPlugin', (stats) => {
      console.log('La construcció ha finalitzat!');
    });
  }
}

module.exports = {
  plugins: [
    new CustomLoggerPlugin(),
  ],
};

Conclusió

La depuració en Webpack pot anar des de simples comprovacions d'errors i advertències fins a perfilat avançat i registres personalitzats. Aprofitant aquestes tècniques, pots identificar i resoldre problemes de manera eficient en els teus projectes Webpack, assegurant un procés de desenvolupament més suau. Recorda començar amb tècniques bàsiques i avançar gradualment cap a mètodes més avançats segons sigui necessari. Bona depuració!
