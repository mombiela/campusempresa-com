Introducció

Webpack és un potent empaquetador de mòduls per a aplicacions JavaScript. Pren mòduls amb dependències i genera actius estàtics que representen aquests mòduls. Aquesta secció et guiarà a través de la configuració del teu primer projecte Webpack des de zero.

Prerequisits

Abans de començar, assegura't de tenir instal·lat el següent:

  • Node.js (es recomana la versió LTS)
  • npm (ve amb Node.js)

Guia pas a pas

1. Inicialitza el teu projecte

Primer, crea un nou directori per al teu projecte i navega-hi:

mkdir my-webpack-project
cd my-webpack-project

Inicialitza un nou projecte Node.js:

npm init -y

Aquesta comanda crea un fitxer package.json amb configuracions per defecte.

2. Instal·la Webpack i Webpack CLI

A continuació, instal·la Webpack i Webpack CLI com a dependències de desenvolupament:

npm install --save-dev webpack webpack-cli

3. Crea l'estructura del projecte

Crea la següent estructura de directoris:

my-webpack-project/
├── src/
│   └── index.js
├── dist/
├── package.json
└── webpack.config.js

4. Configura Webpack

Crea un fitxer webpack.config.js a l'arrel del teu projecte:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  mode: 'development',
};
  • entry: El punt d'entrada de la teva aplicació.
  • output: On Webpack ha de generar els fitxers empaquetats.
  • mode: Configurat a 'development' per facilitar el depurament.

5. Afegeix un script a package.json

Afegeix un script de build al teu package.json:

"scripts": {
  "build": "webpack"
}

6. Crea un fitxer JavaScript senzill

Crea un fitxer index.js senzill al directori src:

console.log('Hello, Webpack!');

7. Construeix el projecte

Executa l'script de build:

npm run build

Aquesta comanda generarà un fitxer bundle.js al directori dist.

8. Crea un fitxer HTML

Crea un fitxer index.html al directori dist per incloure el JavaScript empaquetat:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Webpack Project</title>
</head>
<body>
  <script src="bundle.js"></script>
</body>
</html>

9. Obre el fitxer HTML

Obre index.html al teu navegador per veure el resultat. Hauries de veure "Hello, Webpack!" a la consola.

Resum

En aquesta secció, has après a configurar un projecte bàsic de Webpack. Has:

  • Inicialitzat un projecte Node.js.
  • Instal·lat Webpack i Webpack CLI.
  • Creat una estructura de projecte.
  • Configurat Webpack.
  • Construït i servit una aplicació JavaScript senzilla.

Aquesta configuració fonamental servirà com a base per a configuracions i optimitzacions més avançades de Webpack en futures lliçons.
