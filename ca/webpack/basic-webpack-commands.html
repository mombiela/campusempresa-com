Introducció

Webpack és un potent empaquetador de mòduls per a aplicacions JavaScript. Pren mòduls amb dependències i genera actius estàtics que representen aquests mòduls. Entendre les comandes bàsiques de Webpack és essencial per a qualsevol desenvolupador que vulgui optimitzar el seu flux de treball i construir aplicacions eficients.

Conceptes clau

  • Entrada: El punt d'entrada per a l'aplicació.
  • Sortida: On es generaran els fitxers empaquetats.
  • Carregadors: Transformacions aplicades al codi font d'un mòdul.
  • Plugins: Amplien o personalitzen el procés de construcció de Webpack.

Comandes bàsiques

Instal·lant Webpack

Abans d'utilitzar Webpack, cal instal·lar-lo a través de npm (Node Package Manager).

npm install --save-dev webpack webpack-cli

Executant Webpack

Per empaquetar la teva aplicació, pots utilitzar la següent comanda:

npx webpack

Aquesta comanda utilitzarà el fitxer de configuració per defecte (webpack.config.js) per empaquetar la teva aplicació.

Especificant un fitxer de configuració

Si tens un fitxer de configuració personalitzat, pots especificar-lo utilitzant l'opció --config:

npx webpack --config custom-webpack.config.js

Establint el Mode

Webpack pot executar-se en diferents modes: development, production i none. Pots especificar el mode utilitzant l'opció --mode:

npx webpack --mode development
npx webpack --mode production

Veient canvis

Per recompilar automàticament el teu codi quan els fitxers canviïn, utilitza l'opció --watch:

npx webpack --watch

Especificant Entrada i Sortida

Pots especificar l'entrada i la sortida directament des de la línia de comandes:

npx webpack --entry ./src/index.js --output ./dist/bundle.js

Utilitzant Carregadors

Els carregadors s'utilitzen per transformar fitxers. Per exemple, per utilitzar el babel-loader per a fitxers JavaScript, primer cal instal·lar-lo:

npm install --save-dev babel-loader @babel/core @babel/preset-env

Després, pots especificar-lo en el teu webpack.config.js:

module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env']
          }
        }
      }
    ]
  }
};

Utilitzant Plugins

Els plugins amplien la funcionalitat de Webpack. Per exemple, per utilitzar el HtmlWebpackPlugin, primer cal instal·lar-lo:

npm install --save-dev html-webpack-plugin

Després, pots afegir-lo al teu webpack.config.js:

const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html'
    })
  ]
};

Resum

Entendre i utilitzar les comandes bàsiques de Webpack és crucial per configurar i gestionar els teus projectes JavaScript de manera eficient. Dominant aquestes comandes, pots personalitzar el teu procés de construcció, optimitzar el teu codi i agilitzar el teu flux de treball de desenvolupament. A mesura que progressis, descobriràs que la flexibilitat i el poder de Webpack poden millorar significativament la teva productivitat i el rendiment de les teves aplicacions.
