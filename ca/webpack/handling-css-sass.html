Introducció

En aquesta secció, explorarem com gestionar CSS i SASS en un projecte Webpack. Començarem amb els conceptes bàsics de carregar fitxers CSS i després passarem a temes més avançats com l'ús de SASS, l'optimització de CSS per a producció i l'aprofitament dels mòduls CSS.

Carregar Fitxers CSS

Per carregar fitxers CSS en un projecte Webpack, cal utilitzar css-loader i style-loader.

Passos per Carregar Fitxers CSS

  1. Instal·la els carregadors necessaris: 
    npm install css-loader style-loader --save-dev
  2. Actualitza la configuració de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader'],
      },
    ],
  },
};
  3. Importa CSS en els teus fitxers JavaScript: 
    // src/index.js
import './styles.css';

Explicació

  • css-loader: Aquest carregador interpreta @import i url() com import/require() i els resoldrà.
  • style-loader: Aquest carregador injecta CSS al DOM.

Ús de SASS

SASS és un potent preprocesador de CSS que et permet utilitzar variables, regles niades, mixins i més.

Passos per Carregar Fitxers SASS

  1. Instal·la els carregadors necessaris: 
    npm install sass-loader sass css-loader style-loader --save-dev
  2. Actualitza la configuració de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: ['style-loader', 'css-loader', 'sass-loader'],
      },
    ],
  },
};
  3. Importa SASS en els teus fitxers JavaScript: 
    // src/index.js
import './styles.scss';

Explicació

  • sass-loader: Aquest carregador compila SASS a CSS.
  • L'ordre dels carregadors és important: style-loader -> css-loader -> sass-loader.

Optimització de CSS per a Producció

Quan es construeix per a producció, vols minimitzar i optimitzar els teus fitxers CSS.

Passos per Optimitzar CSS

  1. Instal·la els plugins necessaris: 
    npm install mini-css-extract-plugin css-minimizer-webpack-plugin --save-dev
  2. Actualitza la configuració de Webpack: 
    // webpack.config.js
const MiniCssExtractPlugin = require('mini-css-extract-plugin');
const CssMinimizerPlugin = require('css-minimizer-webpack-plugin');

module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [MiniCssExtractPlugin.loader, 'css-loader'],
      },
      {
        test: /\.scss$/,
        use: [MiniCssExtractPlugin.loader, 'css-loader', 'sass-loader'],
      },
    ],
  },
  plugins: [
    new MiniCssExtractPlugin({
      filename: '[name].[contenthash].css',
    }),
  ],
  optimization: {
    minimizer: [
      `...`,
      new CssMinimizerPlugin(),
    ],
  },
};

Explicació

  • MiniCssExtractPlugin: Aquest plugin extreu CSS en fitxers separats. Crea un fitxer CSS per cada fitxer JS que conté CSS.
  • CssMinimizerPlugin: Aquest plugin optimitza i minimitza el CSS.

Mòduls CSS

Els mòduls CSS et permeten delimitar el CSS creant automàticament un nom de classe únic.

Passos per Utilitzar Mòduls CSS

  1. Actualitza la configuració de Webpack: 
    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.module\.css$/,
        use: [
          'style-loader',
          {
            loader: 'css-loader',
            options: {
              modules: true,
            },
          },
        ],
      },
    ],
  },
};
  2. Importa Mòduls CSS en els teus fitxers JavaScript: 
    // src/index.js
import styles from './styles.module.css';

const element = document.createElement('div');
element.className = styles.myClass;
document.body.appendChild(element);

Explicació

  • L'opció modules: true en css-loader habilita els mòduls CSS.
  • Els mòduls CSS delimiten automàticament el CSS generant noms de classe únics.

Conclusió

En aquesta secció, hem cobert com gestionar CSS i SASS en un projecte Webpack. Hem començat amb els conceptes bàsics de carregar fitxers CSS, hem passat a l'ús de SASS, hem discutit l'optimització de CSS per a producció i finalment hem explorat els mòduls CSS. Seguint aquests passos, pots gestionar i optimitzar eficientment els teus estils en un projecte Webpack.
