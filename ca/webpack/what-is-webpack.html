Introducció a Webpack

Webpack és un empaquetador de mòduls potent i popular per a aplicacions JavaScript. Pren mòduls amb dependències i genera actius estàtics que representen aquests mòduls. Webpack és altament configurable i es pot estendre amb plugins per realitzar una àmplia gamma de tasques.

Conceptes Clau

  • Mòdul: Un tros de codi (per exemple, un fitxer JavaScript) que encapsula funcionalitat.
  • Punt d'Entrada: El fitxer on Webpack comença el procés d'empaquetat.
  • Sortida: El fitxer o fitxers finals generats per Webpack.
  • Loaders: Transformacions aplicades al codi font d'un mòdul (per exemple, convertir ES6 a ES5).
  • Plugins: Amplien les capacitats de Webpack (per exemple, optimitzar els paquets, gestionar actius).

Per què utilitzar Webpack?

  • Gestió de Dependències: Resol i empaqueta automàticament les dependències.
  • Divisió de Codi: Divideix el codi en trossos més petits per millorar el rendiment.
  • Gestió d'Actius: Gestiona actius com imatges, fonts i fulls d'estil.
  • Eines de Desenvolupament: Proporciona funcions com la substitució de mòduls en calent i la recàrrega en viu.

Configuració Bàsica de Webpack

Per començar amb Webpack, cal instal·lar-lo i crear un fitxer de configuració bàsic.

Instal·lació

Primer, instal·la Webpack i Webpack CLI via npm:

npm install --save-dev webpack webpack-cli

Configuració Bàsica

Crea un fitxer webpack.config.js a l'arrel del teu projecte:

const path = require('path');

module.exports = {
  entry: './src/index.js', // Punt d'entrada
  output: {
    filename: 'bundle.js', // Fitxer de sortida
    path: path.resolve(__dirname, 'dist'), // Directori de sortida
  },
  module: {
    rules: [
      {
        test: /\.js$/, // Aplica aquesta regla als fitxers .js
        exclude: /node_modules/, // Exclou node_modules
        use: {
          loader: 'babel-loader', // Utilitza Babel loader
        },
      },
    ],
  },
};

Executar Webpack

Afegeix un script al teu package.json per executar Webpack:

"scripts": {
  "build": "webpack"
}

Executa l'script de build:

npm run build

Loaders i Plugins

Els loaders i plugins són essencials per ampliar la funcionalitat de Webpack.

Loaders

Els loaders transformen fitxers abans que s'afegeixin al paquet. Per exemple, per utilitzar CSS en el teu JavaScript:

npm install --save-dev css-loader style-loader

Actualitza webpack.config.js:

module.exports = {
  // ... altres configuracions
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader'], // Aplica loaders per als fitxers CSS
      },
    ],
  },
};

Plugins

Els plugins realitzen una gamma més àmplia de tasques. Per exemple, per generar un fitxer HTML que inclogui el teu paquet:

npm install --save-dev html-webpack-plugin

Actualitza webpack.config.js:

const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  // ... altres configuracions
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html', // Fitxer plantilla
    }),
  ],
};

Funcions Avançades

Webpack ofereix funcions avançades com la divisió de codi, el tree shaking i la memòria cau.

Divisió de Codi

La divisió de codi ajuda a carregar només el codi necessari per a una pàgina, millorant el rendiment.

module.exports = {
  // ... altres configuracions
  optimization: {
    splitChunks: {
      chunks: 'all',
    },
  },
};

Tree Shaking

El tree shaking elimina el codi no utilitzat del paquet final.

module.exports = {
  // ... altres configuracions
  mode: 'production', // Activa el tree shaking establint el mode a producció
};

Memòria Cau

La memòria cau ajuda a millorar el rendiment de la build emmagatzemant en memòria cau els mòduls que no han canviat.

module.exports = {
  // ... altres configuracions
  output: {
    filename: '[name].[contenthash].js', // Utilitza hash de contingut per a la memòria cau
    path: path.resolve(__dirname, 'dist'),
  },
};

Conclusió

Webpack és una eina versàtil i potent per empaquetar aplicacions JavaScript. Simplifica la gestió de dependències, optimitza el rendiment i proporciona un ric ecosistema de loaders i plugins. Entenent els conceptes bàsics i explorant les funcions avançades, pots aprofitar Webpack per construir aplicacions web eficients i escalables.
