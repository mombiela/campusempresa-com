Introducció a Webpack

  • Què és Webpack?

    • Webpack és un empaquetador de mòduls per a aplicacions JavaScript.
    • Pren mòduls amb dependències i genera actius estàtics que representen aquests mòduls.

  • Per què utilitzar Webpack?

    • Gestiona eficientment les dependències.
    • Optimitza el codi per a producció.
    • Suporta característiques i eines modernes de JavaScript.

Configurant Webpack

  • Instal·lant Webpack

    • Instal·la Webpack i Webpack CLI via npm: 
      npm install --save-dev webpack webpack-cli

  • Creant l'Estructura del Projecte

    • Estructura bàsica del projecte: 
      my-app/
├── src/
│   └── index.js
├── dist/
├── package.json
└── webpack.config.js

Configurant Webpack

  • Configuració Bàsica

    • Crea webpack.config.js al directori arrel: 
      const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist')
  },
  mode: 'development'
};

  • Executant Webpack

    • Afegeix un script a package.json: 
      "scripts": {
  "build": "webpack"
}
    • Executa el procés de construcció: 
      npm run build

Carregant i Empaquetant Actius

  • Gestionant CSS

    • Instal·la carregadors: 
      npm install --save-dev style-loader css-loader
    • Actualitza webpack.config.js: 
      module.exports = {
  // ...altres configuracions
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};

  • Gestionant Imatges

    • Instal·la file-loader: 
      npm install --save-dev file-loader
    • Actualitza webpack.config.js: 
      module.exports = {
  // ...altres configuracions
  module: {
    rules: [
      {
        test: /\.(png|svg|jpg|gif)$/,
        use: ['file-loader']
      }
    ]
  }
};

Configuració Avançada

  • Utilitzant Plugins

    • Instal·la HTML Webpack Plugin: 
      npm install --save-dev html-webpack-plugin
    • Actualitza webpack.config.js: 
      const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  // ...altres configuracions
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html'
    })
  ]
};

  • Divisió de Codi

    • Modifica webpack.config.js per habilitar la divisió de codi: 
      module.exports = {
  // ...altres configuracions
  optimization: {
    splitChunks: {
      chunks: 'all'
    }
  }
};

Eines de Desenvolupament

  • Utilitzant Webpack Dev Server
    • Instal·la Webpack Dev Server: 
      npm install --save-dev webpack-dev-server
    • Actualitza webpack.config.js: 
      module.exports = {
  // ...altres configuracions
  devServer: {
    contentBase: './dist'
  }
};
    • Afegeix un script a package.json: 
      "scripts": {
  "start": "webpack serve --open"
}
    • Inicia el servidor de desenvolupament: 
      npm start

Conclusió

  • Resum

    • Webpack és una eina poderosa per empaquetar aplicacions JavaScript.
    • Simplifica la gestió de dependències i optimitza el codi per a producció.
    • Utilitzant carregadors i plugins, Webpack pot gestionar diversos tipus d'actius i millorar l'experiència de desenvolupament.

  • Pròxims Passos

    • Explora característiques més avançades de Webpack com el tree shaking i plugins personalitzats.
    • Integra Webpack amb altres eines i frameworks com React o Vue.js.
    • Continua practicant construint aplicacions i configuracions més complexes.
