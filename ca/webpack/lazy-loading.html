La càrrega peresosa és un patró de disseny comunament utilitzat en el desenvolupament web per ajornar la càrrega de recursos fins que realment es necessiten. Això pot millorar significativament el rendiment de les teves aplicacions web reduint el temps de càrrega inicial i el consum de banda ampla.

Què és la Càrrega Peresosa?

  • Definició: La càrrega peresosa és una tècnica que retarda la càrrega de recursos no essencials en la càrrega inicial de la pàgina i, en canvi, els carrega quan són necessaris.
  • Beneficis:
    • Millora el temps de càrrega inicial.
    • Redueix l'ús de banda ampla.
    • Millora l'experiència de l'usuari carregant el contingut progressivament.

Com Funciona la Càrrega Peresosa a Webpack

Webpack és un potent empaquetador de mòduls que et permet implementar la càrrega peresosa amb facilitat. Pot dividir el teu codi en trossos més petits i carregar-los sota demanda.

Conceptes Clau

  • Divisió de Codi: Webpack pot dividir el teu codi en diversos paquets que es poden carregar sota demanda.
  • Importacions Dinàmiques: Utilitzar la sintaxi import() per carregar mòduls dinàmicament.

Implementant la Càrrega Peresosa a Webpack

Pas 1: Configurar Webpack

Primer, assegura't que tens Webpack instal·lat i configurat en el teu projecte. Si no, pots instal·lar-lo utilitzant npm:

npm install --save-dev webpack webpack-cli

Pas 2: Configurar Webpack

En el teu webpack.config.js, assegura't de tenir la següent configuració bàsica:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  mode: 'development',
};

Pas 3: Crear un Mòdul per Carregar Peresosament

Crea un mòdul que vulguis carregar peresosament. Per exemple, crea un fitxer lazyModule.js:

// src/lazyModule.js
export function lazyFunction() {
  console.log('Aquesta és una funció carregada peresosament!');
}

Pas 4: Utilitzar Importacions Dinàmiques

En el teu fitxer d'entrada principal, utilitza importacions dinàmiques per carregar el mòdul peresosament:

// src/index.js
document.getElementById('loadButton').addEventListener('click', () => {
  import('./lazyModule').then(module => {
    module.lazyFunction();
  }).catch(err => {
    console.error('No s’ha pogut carregar el mòdul', err);
  });
});

Pas 5: Actualitzar HTML

Assegura't de tenir un fitxer HTML amb un botó per activar la càrrega peresosa:

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Webpack Càrrega Peresosa</title>
</head>
<body>
  <button id="loadButton">Carregar Mòdul</button>
  <script src="dist/bundle.js"></script>
</body>
</html>

Pas 6: Compilar i Provar

Executa Webpack per compilar el teu projecte:

npx webpack

Obre el fitxer index.html en el teu navegador i fes clic al botó "Carregar Mòdul". Hauries de veure el missatge de lazyFunction a la consola.

Tècniques Avançades de Càrrega Peresosa

Prefetching i Preloading

Webpack suporta prefetching i preloading per optimitzar encara més la càrrega peresosa.

  • Prefetching: Indica al navegador que descarregui recursos quan estigui inactiu.
  • Preloading: Obliga al navegador a descarregar recursos tan aviat com sigui possible.

Exemple de Prefetching

// src/index.js
document.getElementById('loadButton').addEventListener('click', () => {
  import(/* webpackPrefetch: true */ './lazyModule').then(module => {
    module.lazyFunction();
  }).catch(err => {
    console.error('No s’ha pogut carregar el mòdul', err);
  });
});

Exemple de Preloading

// src/index.js
document.getElementById('loadButton').addEventListener('click', () => {
  import(/* webpackPreload: true */ './lazyModule').then(module => {
    module.lazyFunction();
  }).catch(err => {
    console.error('No s’ha pogut carregar el mòdul', err);
  });
});

Conclusió

La càrrega peresosa és una tècnica poderosa per optimitzar el rendiment de les teves aplicacions web. Utilitzant la divisió de codi i les importacions dinàmiques de Webpack, pots implementar fàcilment la càrrega peresosa en els teus projectes. Tècniques avançades com el prefetching i el preloading poden millorar encara més l'experiència de l'usuari optimitzant la càrrega de recursos.
