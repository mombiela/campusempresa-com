Introducció a Webpack i Vue

Webpack és un potent empaquetador de mòduls per a aplicacions JavaScript. Vue.js és un framework JavaScript progressiu per construir interfícies d'usuari. Combinar Webpack amb Vue permet un desenvolupament eficient i compilacions de producció optimitzades.

Configuració de Webpack per a un Projecte Vue

Instal·lant Dependències

Per començar, necessites instal·lar Webpack i Vue juntament amb alguns plugins i carregadors essencials.

npm install --save-dev webpack webpack-cli vue-loader vue-template-compiler
npm install vue

Configuració Bàsica de Webpack

Crea un fitxer webpack.config.js a l'arrel del teu projecte.

const path = require('path');
const VueLoaderPlugin = require('vue-loader/lib/plugin');

module.exports = {
  entry: './src/main.js',
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.vue$/,
        loader: 'vue-loader'
      },
      {
        test: /\.js$/,
        exclude: /node_modules/,
        loader: 'babel-loader'
      },
      {
        test: /\.css$/,
        use: [
          'style-loader',
          'css-loader'
        ]
      }
    ]
  },
  plugins: [
    new VueLoaderPlugin()
  ],
  resolve: {
    alias: {
      'vue$': 'vue/dist/vue.esm.js'
    },
    extensions: ['*', '.js', '.vue', '.json']
  }
};

Creant un Component Vue

Crea un directori src amb un fitxer main.js i un directori components amb un fitxer HelloWorld.vue.

src/main.js

import Vue from 'vue';
import HelloWorld from './components/HelloWorld.vue';

new Vue({
  render: h => h(HelloWorld),
}).$mount('#app');

src/components/HelloWorld.vue

<template>
  <div>
    <h1>Hello, Webpack with Vue!</h1>
  </div>
</template>

<script>
export default {
  name: 'HelloWorld'
};
</script>

<style scoped>
h1 {
  color: #42b983;
}
</style>

Executant la Compilació

Afegeix un script de compilació al teu fitxer package.json.

"scripts": {
  "build": "webpack --mode development"
}

Executa l'script de compilació:

npm run build

Configuració Avançada

Servidor de Desenvolupament

Per a una millor experiència de desenvolupament, pots utilitzar webpack-dev-server.

npm install --save-dev webpack-dev-server

Actualitza el teu webpack.config.js:

devServer: {
  contentBase: path.join(__dirname, 'dist'),
  compress: true,
  port: 9000
}

Actualitza els scripts del teu package.json:

"scripts": {
  "start": "webpack serve --mode development"
}

Executa el servidor de desenvolupament:

npm start

Substitució de Mòduls en Calent (HMR)

Activa HMR per a una experiència de desenvolupament més dinàmica.

Actualitza el teu webpack.config.js:

plugins: [
  new VueLoaderPlugin(),
  new webpack.HotModuleReplacementPlugin()
],
devServer: {
  contentBase: path.join(__dirname, 'dist'),
  compress: true,
  port: 9000,
  hot: true
}

Divisió de Codi

Optimitza la teva aplicació dividint el teu codi en fragments més petits.

Actualitza el teu webpack.config.js:

output: {
  path: path.resolve(__dirname, 'dist'),
  filename: '[name].bundle.js'
},
optimization: {
  splitChunks: {
    chunks: 'all'
  }
}

Variables d'Entorn

Utilitza variables d'entorn per gestionar diferents configuracions per a desenvolupament i producció.

Instal·la dotenv:

npm install dotenv --save

Crea un fitxer .env:

VUE_APP_API_URL=https://api.example.com

Actualitza el teu webpack.config.js:

const webpack = require('webpack');
const dotenv = require('dotenv').config({ path: __dirname + '/.env' });

plugins: [
  new VueLoaderPlugin(),
  new webpack.DefinePlugin({
    'process.env': JSON.stringify(dotenv.parsed)
  })
]

Conclusió

Webpack i Vue junts proporcionen una combinació poderosa per construir aplicacions web modernes. Entenent els conceptes bàsics de la configuració de Webpack amb Vue, i explorant configuracions avançades com HMR, divisió de codi i variables d'entorn, pots crear aplicacions eficients i escalables. Continua experimentant amb diferents configuracions per optimitzar el teu flux de treball de desenvolupament i les compilacions de producció.
