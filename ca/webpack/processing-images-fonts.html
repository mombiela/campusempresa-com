En aquesta secció, explorarem com gestionar imatges i fonts en un projecte Webpack. Això inclou carregar, optimitzar i gestionar aquests recursos de manera eficient per assegurar que les teves aplicacions web siguin rendibles i visualment atractives.

Conceptes Clau

  • Loaders: Webpack utilitza loaders per processar fitxers que no són JavaScript.
  • Mòduls d'Actius: Una nova característica en Webpack 5 que simplifica la gestió d'actius.
  • Optimització: Tècniques per reduir la mida de les imatges i fonts per a temps de càrrega més ràpids.

Configuració de Loaders per a Imatges i Fonts

Per processar imatges i fonts, necessites configurar Webpack per utilitzar els loaders adequats. Aquests loaders t'ajudaran a importar aquests recursos als teus fitxers JavaScript.

Instal·lació dels Loaders Necessaris

Primer, instal·la els loaders requerits:

npm install --save-dev file-loader url-loader

Configuració de Webpack

A continuació, actualitza el teu webpack.config.js per incloure regles per gestionar imatges i fonts.

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  module: {
    rules: [
      {
        test: /\.(png|jpg|gif)$/i,
        type: 'asset/resource',
      },
      {
        test: /\.(woff|woff2|eot|ttf|otf)$/i,
        type: 'asset/resource',
      },
    ],
  },
};

Explicació

  • test: Una expressió regular que coincideix amb els tipus de fitxers que vols processar.
  • type: Especifica el tipus de mòdul d'actiu. asset/resource emet un fitxer separat i exporta l'URL.

Ús de Mòduls d'Actius

Webpack 5 va introduir els Mòduls d'Actius, que simplifiquen el procés de gestionar actius com imatges i fonts.

Exemple de Configuració

A continuació, es mostra com pots configurar Webpack per utilitzar Mòduls d'Actius:

module.exports = {
  module: {
    rules: [
      {
        test: /\.(png|jpg|jpeg|gif|svg)$/i,
        type: 'asset/resource',
      },
      {
        test: /\.(woff|woff2|eot|ttf|otf)$/i,
        type: 'asset/resource',
      },
    ],
  },
};

Explicació

  • asset/resource: Emet un fitxer separat i exporta l'URL. Això és útil per a fitxers més grans.
  • asset/inline: Exporta l'actiu com una URI de dades. Això és útil per a fitxers més petits.
  • asset/source: Exporta el codi font de l'actiu.
  • asset: Tria automàticament entre exportar una URI de dades i emetre un fitxer separat basat en la mida del fitxer.

Optimització d'Imatges i Fonts

L'optimització és crucial per millorar el rendiment de la teva aplicació web. Webpack proporciona diversos plugins per ajudar amb això.

Optimització d'Imatges

Instal·la image-webpack-loader:

npm install --save-dev image-webpack-loader

Actualitza el teu webpack.config.js:

module.exports = {
  module: {
    rules: [
      {
        test: /\.(png|jpg|jpeg|gif|svg)$/i,
        use: [
          {
            loader: 'file-loader',
          },
          {
            loader: 'image-webpack-loader',
            options: {
              mozjpeg: {
                progressive: true,
                quality: 65,
              },
              // Optimitza imatges PNG
              optipng: {
                enabled: true,
              },
              // Optimitza imatges GIF
              gifsicle: {
                interlaced: false,
              },
              // Optimitza imatges SVG
              svgo: {
                plugins: [
                  {
                    removeViewBox: false,
                  },
                ],
              },
            },
          },
        ],
      },
    ],
  },
};

Optimització de Fonts

Les fonts també es poden optimitzar utilitzant el url-loader per incloure fitxers de fonts petits com a URI de dades.

Instal·la url-loader:

npm install --save-dev url-loader

Actualitza el teu webpack.config.js:

module.exports = {
  module: {
    rules: [
      {
        test: /\.(woff|woff2|eot|ttf|otf)$/i,
        use: [
          {
            loader: 'url-loader',
            options: {
              limit: 8192, // Inclou fitxers més petits de 8KB
              name: '[name].[hash].[ext]',
              outputPath: 'fonts/',
            },
          },
        ],
      },
    ],
  },
};

Conclusió

Gestionar imatges i fonts en Webpack implica configurar loaders i utilitzar mòduls d'actius per gestionar aquests recursos de manera eficient. Optimitzant aquests recursos, pots millorar significativament el rendiment de les teves aplicacions web. Amb els coneixements adquirits en aquesta secció, hauràs de ser capaç de:

  • Configurar Webpack per processar imatges i fonts.
  • Utilitzar Mòduls d'Actius per simplificar la gestió d'actius.
  • Optimitzar imatges i fonts per a un millor rendiment.

Dominant aquestes tècniques, asseguraràs que les teves aplicacions web siguin no només visualment atractives, sinó també rendibles i eficients.
