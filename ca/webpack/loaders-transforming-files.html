Introducció als Carregadors

Els carregadors són una part essencial de Webpack, permetent-te preprocessar fitxers mentre els importes o carregues. Transformen fitxers d'un altre llenguatge o format en mòduls JavaScript que Webpack pot entendre i empaquetar.

Conceptes Clau

  • Propòsit: Els carregadors et permeten preprocessar fitxers, com convertir TypeScript a JavaScript, o compilar Sass a CSS.
  • Ús: Els carregadors es configuren en el fitxer de configuració de Webpack (webpack.config.js).
  • Cadenables: Es poden encadenar múltiples carregadors per gestionar transformacions complexes.

Configurant Carregadors

Per utilitzar carregadors, has d'especificar-los a l'array module.rules dins del teu fitxer de configuració de Webpack.

Exemple: Carregant CSS

Per carregar i empaquetar fitxers CSS, necessites utilitzar css-loader i style-loader.

  1. Instal·la els carregadors:

    npm install --save-dev css-loader style-loader

  2. Actualitza webpack.config.js:

    module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [
          'style-loader', // Injecta estils al DOM
          'css-loader'    // Converteix CSS a CommonJS
        ]
      }
    ]
  }
};

Exemple: Carregant Imatges

Per carregar fitxers d'imatge, pots utilitzar file-loader o url-loader.

  1. Instal·la el carregador:

    npm install --save-dev file-loader

  2. Actualitza webpack.config.js:

    module.exports = {
  module: {
    rules: [
      {
        test: /\.(png|svg|jpg|jpeg|gif)$/i,
        type: 'asset/resource', // Utilitza file-loader o mòduls d'actius
      }
    ]
  }
};

Configuracions Avançades de Carregadors

Els carregadors es poden configurar amb opcions per personalitzar el seu comportament.

Exemple: Carregant Sass/SCSS

Per compilar fitxers Sass/SCSS a CSS, necessites sass-loader, css-loader, i style-loader.

  1. Instal·la els carregadors:

    npm install --save-dev sass-loader sass css-loader style-loader

  2. Actualitza webpack.config.js:

    module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: [
          'style-loader', // Injecta estils al DOM
          'css-loader',   // Converteix CSS a CommonJS
          'sass-loader'   // Compila Sass a CSS
        ]
      }
    ]
  }
};

Exemple: Transpilant ES6+ a ES5

Per transpilar JavaScript modern (ES6+) a ES5, necessites babel-loader.

  1. Instal·la el carregador i Babel:

    npm install --save-dev babel-loader @babel/core @babel/preset-env

  2. Actualitza webpack.config.js:

    module.exports = {
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@babel/preset-env']
          }
        }
      }
    ]
  }
};

Resum

Els carregadors són una característica poderosa de Webpack que et permeten preprocessar i transformar fitxers abans que siguin empaquetats. Configurant carregadors en el teu webpack.config.js, pots gestionar una varietat de tipus de fitxers i transformacions, des de compilar Sass fins a carregar imatges i transpilar JavaScript modern. Entendre i utilitzar els carregadors de manera efectiva et permetrà crear processos de construcció més eficients i mantenibles per a les teves aplicacions web.
