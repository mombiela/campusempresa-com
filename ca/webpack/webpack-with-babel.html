Introducció

Webpack i Babel són eines poderoses que poden millorar significativament el teu flux de treball de desenvolupament en JavaScript. Webpack és un empaquetador de mòduls que pren mòduls amb dependències i genera actius estàtics que representen aquests mòduls. Babel és un compilador de JavaScript que et permet utilitzar JavaScript de nova generació, avui. Aquesta secció et guiarà a través de la integració de Babel amb Webpack per crear un entorn de desenvolupament de JavaScript modern.

Conceptes Clau

  • Webpack: Un empaquetador de mòduls per a aplicacions JavaScript.
  • Babel: Un compilador de JavaScript que transforma el codi ES6+ en JavaScript compatible amb versions anteriors.
  • Loaders: Webpack utilitza loaders per preprocessar fitxers. Babel-loader s'utilitza per transpilar fitxers JavaScript utilitzant Babel i Webpack.

Configurant Webpack amb Babel

Pas 1: Inicialitza el teu Projecte

Primer, crea un nou directori per al teu projecte i inicialitza'l amb npm.

mkdir webpack-babel-project
cd webpack-babel-project
npm init -y

Pas 2: Instal·la Webpack i Babel

Instal·la Webpack, Webpack CLI, Babel, i els presets necessaris de Babel.

npm install --save-dev webpack webpack-cli babel-loader @babel/core @babel/preset-env

Pas 3: Configura Babel

Crea un fitxer .babelrc a l'arrel del teu projecte per configurar Babel.

{
  "presets": ["@babel/preset-env"]
}
  • @babel/preset-env: Un preset intel·ligent que et permet utilitzar l'últim JavaScript sense necessitat de microgestionar quines transformacions de sintaxi (i opcionalment, polyfills de navegador) són necessàries per al teu entorn objectiu.

Pas 4: Configura Webpack

Crea un fitxer webpack.config.js a l'arrel del teu projecte per configurar Webpack.

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist')
  },
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader'
        }
      }
    ]
  }
};
  • entry: El punt d'entrada de la teva aplicació.
  • output: La configuració de sortida per al fitxer empaquetat.
  • module.rules: Una matriu de regles que diu a Webpack com manejar diferents tipus de fitxers. Aquí, utilitzem babel-loader per transpilar fitxers JavaScript.

Pas 5: Crea Fitxers de Font

Crea un directori src i afegeix un fitxer index.js.

// src/index.js
const greet = () => {
  console.log('Hola, Webpack amb Babel!');
};

greet();

Pas 6: Construeix el Projecte

Afegeix un script de construcció al teu package.json.

"scripts": {
  "build": "webpack"
}

Executa l'script de construcció.

npm run build

Això crearà un directori dist amb un fitxer bundle.js que conté el teu codi transpilat i empaquetat.

Conclusió

Integrar Babel amb Webpack et permet escriure JavaScript modern mentre assegures la compatibilitat amb navegadors antics. Seguint els passos descrits en aquesta secció, pots configurar un entorn de desenvolupament que aprofiti el poder de les dues eines. Aquesta configuració et permetrà utilitzar les últimes característiques de JavaScript i agilitzar el teu procés de desenvolupament.
