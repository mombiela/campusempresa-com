La gestió d'estat és un concepte crucial en el desenvolupament amb React. Et permet gestionar i sincronitzar l'estat de la teva aplicació de manera eficient. Aquest tema et guiarà a través dels conceptes bàsics de la gestió d'estat, des de la comprensió del que és l'estat fins a la implementació de tècniques avançades de gestió d'estat.

Què és l'Estat?

L'estat es refereix a les dades que un component manté. Pot canviar amb el temps i afectar com es renderitza i es comporta el component.

  • Estat Local: Gestionat dins d'un sol component.
  • Estat Global: Compartit entre múltiples components.

Gestionant l'Estat en React

Utilitzant el Hook useState

El hook useState és la manera més senzilla de gestionar l'estat local en un component funcional.

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Has clicat {count} vegades</p>
      <button onClick={() => setCount(count + 1)}>
        Clica'm
      </button>
    </div>
  );
}
  • useState: Inicialitza l'estat.
  • setCount: Actualitza l'estat.

Utilitzant el Hook useReducer

Per a una lògica d'estat més complexa, useReducer pot ser una millor opció.

import React, { useReducer } from 'react';

const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}

function Counter() {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Compte: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        +
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        -
      </button>
    </div>
  );
}
  • useReducer: Gestiona la lògica d'estat complexa.
  • dispatch: Envia accions a la funció reductora.

Gestió d'Estat Global

API de Context

L'API de Context et permet compartir l'estat entre múltiples components sense passar props manualment a cada nivell.

import React, { createContext, useContext, useState } from 'react';

const CountContext = createContext();

function CounterProvider({ children }) {
  const [count, setCount] = useState(0);

  return (
    <CountContext.Provider value={{ count, setCount }}>
      {children}
    </CountContext.Provider>
  );
}

function Counter() {
  const { count, setCount } = useContext(CountContext);

  return (
    <div>
      <p>Compte: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Incrementar
      </button>
    </div>
  );
}

function App() {
  return (
    <CounterProvider>
      <Counter />
    </CounterProvider>
  );
}
  • createContext: Crea un objecte de context.
  • useContext: Consumeix el valor del context.

Redux

Redux és una biblioteca popular per gestionar l'estat global en aplicacions més grans.

// store.js
import { createStore } from 'redux';

const initialState = { count: 0 };

function counterReducer(state = initialState, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

const store = createStore(counterReducer);

export default store;

// App.js
import React from 'react';
import { Provider, useDispatch, useSelector } from 'react-redux';
import store from './store';

function Counter() {
  const count = useSelector(state => state.count);
  const dispatch = useDispatch();

  return (
    <div>
      <p>Compte: {count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        Incrementar
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        Decrementar
      </button>
    </div>
  );
}

function App() {
  return (
    <Provider store={store}>
      <Counter />
    </Provider>
  );
}

export default App;
  • createStore: Crea una botiga de Redux.
  • Provider: Fa que la botiga de Redux estigui disponible per a l'aplicació.
  • useSelector: Accedeix a l'estat de la botiga de Redux.
  • useDispatch: Envia accions a la botiga de Redux.

Conclusió

La gestió d'estat és un aspecte fonamental del desenvolupament amb React. Comprendre com gestionar tant l'estat local com el global de manera eficient pot millorar enormement el rendiment i la mantenibilitat de les teves aplicacions. Comença amb els conceptes bàsics utilitzant useState i useReducer, i després explora tècniques més avançades com l'API de Context i Redux per a aplicacions més grans.
