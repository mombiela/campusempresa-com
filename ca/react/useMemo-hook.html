Introducció a useMemo

El hook useMemo és una eina poderosa en React que ajuda a optimitzar el rendiment memorizant càlculs costosos. Assegura que una funció només es torni a calcular quan les seves dependències canvien, prevenint renders innecessaris.

  • Propòsit: Optimitzar el rendiment memorizant càlculs costosos.
  • Quan utilitzar-lo: Quan tens una funció que realitza un càlcul pesat i el seu resultat és necessari en el procés de renderització.

Sintaxi Bàsica

El hook useMemo pren dos arguments: una funció i una matriu de dependències.

const memoizedValue = useMemo(() => computeExpensiveValue(a, b), [a, b]);
  • Funció: La funció que calcula el valor.
  • Matriu de Dependències: La llista de dependències que desencadenen la recomputació quan canvien.

Exemple: Ús Bàsic

Comencem amb un exemple senzill per entendre com funciona useMemo.

Sense useMemo

import React, { useState } from 'react';

function App() {
  const [count, setCount] = useState(0);
  const [input, setInput] = useState('');

  const computeExpensiveValue = (num) => {
    console.log('Computant...');
    return num * 2;
  };

  const value = computeExpensiveValue(count);

  return (
    <div>
      <input value={input} onChange={(e) => setInput(e.target.value)} />
      <button onClick={() => setCount(count + 1)}>Incrementar</button>
      <p>Valor Computat: {value}</p>
    </div>
  );
}

export default App;

En aquest exemple, computeExpensiveValue es crida en cada render, fins i tot quan input canvia.

Amb useMemo

import React, { useState, useMemo } from 'react';

function App() {
  const [count, setCount] = useState(0);
  const [input, setInput] = useState('');

  const computeExpensiveValue = (num) => {
    console.log('Computant...');
    return num * 2;
  };

  const memoizedValue = useMemo(() => computeExpensiveValue(count), [count]);

  return (
    <div>
      <input value={input} onChange={(e) => setInput(e.target.value)} />
      <button onClick={() => setCount(count + 1)}>Incrementar</button>
      <p>Valor Computat: {memoizedValue}</p>
    </div>
  );
}

export default App;

Ara, computeExpensiveValue només es crida quan count canvia, optimitzant el rendiment.

Ús Avançat

Memorizant Objectes Complexos

useMemo també es pot utilitzar per memoritzar objectes complexos o arrays.

import React, { useState, useMemo } from 'react';

function App() {
  const [count, setCount] = useState(0);

  const complexObject = useMemo(() => {
    return { value: count };
  }, [count]);

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>Incrementar</button>
      <p>Valor de l'Objecte Complex: {complexObject.value}</p>
    </div>
  );
}

export default App;

Evitant Renders Innecessaris

Utilitzar useMemo pot ajudar a evitar renders innecessaris assegurant que es retorna la mateixa referència tret que les dependències canviïn.

import React, { useState, useMemo } from 'react';

function App() {
  const [count, setCount] = useState(0);
  const [input, setInput] = useState('');

  const memoizedValue = useMemo(() => {
    return { value: count };
  }, [count]);

  return (
    <div>
      <input value={input} onChange={(e) => setInput(e.target.value)} />
      <button onClick={() => setCount(count + 1)}>Incrementar</button>
      <p>Valor Memorizat: {memoizedValue.value}</p>
    </div>
  );
}

export default App;

Errors Comuns

  • Sobreutilitzar useMemo: No totes les funcions necessiten ser memorizades. Utilitza-ho només quan notis problemes de rendiment.
  • Dependències Incorrectes: Assegura't que la matriu de dependències està correctament especificada per evitar valors obsolets o recomputacions innecessàries.

Conclusió

El hook useMemo és una eina valuosa per optimitzar el rendiment en aplicacions React. Memorizant càlculs costosos, pots prevenir renders innecessaris i millorar l'eficiència dels teus components. Recorda utilitzar-lo amb prudència i assegurar-te que les teves matrius de dependències són precises per evitar errors comuns.
