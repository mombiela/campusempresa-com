Desplegament d'Aplicacions React
Millors Pràctiques i Estàndards de Codificació ►

Desplegar una aplicació React implica preparar la teva aplicació per a producció, assegurar-te que funcioni eficientment i fer-la accessible als usuaris. Aquest tema et guiarà a través dels passos necessaris per desplegar una aplicació React, des de la configuració bàsica fins a estratègies avançades de desplegament.

Preparant la Teva Aplicació React per al Desplegament

Abans de desplegar, has d'assegurar-te que la teva aplicació React està llesta per a producció.

Compilar la Teva Aplicació

  • Crear una compilació de producció: React proporciona una comanda integrada per crear una compilació de producció optimitzada de la teva aplicació.
npm run build
  • Entendre la carpeta de compilació: La carpeta build conté els fitxers estàtics que poden ser servits per un servidor web. Això inclou:
    • index.html
    • Fitxers JavaScript minificats
    • Imatges optimitzades

Variables d'Entorn

  • Configurar variables d'entorn: Utilitza fitxers .env per gestionar variables específiques de l'entorn.
REACT_APP_API_URL=https://api.example.com
  • Accedir a les variables d'entorn en el teu codi:
const apiUrl = process.env.REACT_APP_API_URL;

Desplegant a Hostings de Llocs Estàtics

GitHub Pages

  • Instal·lar el paquet gh-pages:
npm install --save gh-pages
  • Afegir scripts de desplegament a package.json:
"homepage": "https://yourusername.github.io/your-repo-name",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d build"
}
  • Desplegar la teva aplicació:
npm run deploy

Netlify

  • Connectar el teu repositori: Enllaça el teu repositori de GitHub a Netlify.
  • Configurar els paràmetres de compilació:
    • Comanda de compilació: npm run build
    • Directori de publicació: build
  • Desplegar: Netlify construirà i desplegarà automàticament el teu lloc.

Vercel

  • Instal·lar Vercel CLI:
npm install -g vercel
  • Desplegar la teva aplicació:
vercel
  • Configurar els paràmetres: Segueix les indicacions per configurar el teu projecte.

Desplegant a Proveïdors de Núvol

AWS S3 i CloudFront

  • Crear un bucket S3: Assegura't que el bucket sigui públic.
  • Pujar la carpeta de compilació: Utilitza l'AWS CLI o la consola S3.
  • Configurar CloudFront: Configura una distribució CloudFront per servir el contingut del teu S3.

Google Cloud Platform (GCP)

  • Instal·lar el Google Cloud SDK: Segueix les instruccions al lloc web de GCP.

  • Desplegar a Firebase Hosting:

    • Instal·lar Firebase CLI:
    npm install -g firebase-tools
    • Inicialitzar Firebase:
    firebase init
    • Desplegar la teva aplicació:
    firebase deploy

Estratègies Avançades de Desplegament

Integració Contínua/Desplegament Continu (CI/CD)

  • Configurar pipelines de CI/CD: Utilitza eines com GitHub Actions, GitLab CI o CircleCI per automatitzar el procés de desplegament.
  • Exemple de workflow de GitHub Actions:
name: Deploy React App

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v2

      - name: Install dependencies
        run: npm install

      - name: Build project
        run: npm run build

      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./build

Renderitzat del Costat del Servidor (SSR) amb Next.js

  • Instal·lar Next.js:
npx create-next-app
  • Desplegar a Vercel: Vercel proporciona una integració perfecta amb Next.js per a SSR.

Conclusió

Desplegar una aplicació React implica diversos passos, des de crear una compilació de producció fins a triar el servei d'allotjament adequat. Tant si estàs desplegant a un hosting de llocs estàtics com GitHub Pages, a un proveïdor de núvol com AWS, o utilitzant pipelines avançats de CI/CD, entendre aquests processos assegura que la teva aplicació sigui accessible i funcioni bé en un entorn de producció. Amb aquestes eines i tècniques, pots desplegar les teves aplicacions React amb confiança i mantenir-les funcionant sense problemes.
