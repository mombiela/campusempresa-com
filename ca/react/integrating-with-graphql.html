Introducció a GraphQL

GraphQL és un llenguatge de consulta per a la teva API i un entorn d'execució del costat del servidor per executar consultes utilitzant un sistema de tipus que defineixes per a les teves dades. Proporciona una alternativa més eficient, potent i flexible a REST.

  • Conceptes Clau:
    • Esquema: Defineix l'estructura de les dades.
    • Consulta: Sol·licita dades específiques.
    • Mutació: Modifica dades.
    • Resolver: Funcions que gestionen la recuperació de les dades per a una consulta o mutació.

Configuració d'un servidor GraphQL

Abans d'integrar GraphQL amb React, necessites un servidor GraphQL. Per a aquest curs, utilitzarem Apollo Server.

Exemple: Configuració d'Apollo Server

const { ApolloServer, gql } = require('apollo-server');

// Les definicions de tipus defineixen la "forma" de les teves dades i especifiquen quines consultes i mutacions estan disponibles.
const typeDefs = gql`
  type Query {
    hello: String
  }
`;

// Els resolvers defineixen la tècnica per recuperar els tipus definits a l'esquema.
const resolvers = {
  Query: {
    hello: () => 'Hello world!',
  },
};

// El constructor ApolloServer requereix dos paràmetres: la teva definició d'esquema i el teu conjunt de resolvers.
const server = new ApolloServer({ typeDefs, resolvers });

// El mètode `listen` llança un servidor web.
server.listen().then(({ url }) => {
  console.log(`🚀  Server ready at ${url}`);
});

Integració de GraphQL amb React

Per integrar GraphQL amb React, utilitzarem Apollo Client, una biblioteca de gestió d'estat completa per a JavaScript que et permet gestionar tant dades locals com remotes amb GraphQL.

Instal·lació d'Apollo Client

Primer, instal·la els paquets necessaris:

npm install @apollo/client graphql

Configuració d'Apollo Client a React

Crea una instància d'Apollo Client i embolcalla la teva aplicació React amb ApolloProvider.

import React from 'react';
import ReactDOM from 'react-dom';
import { ApolloClient, InMemoryCache, ApolloProvider } from '@apollo/client';
import App from './App';

// Inicialitza Apollo Client
const client = new ApolloClient({
  uri: 'http://localhost:4000',
  cache: new InMemoryCache()
});

ReactDOM.render(
  <ApolloProvider client={client}>
    <App />
  </ApolloProvider>,
  document.getElementById('root')
);

Recuperació de dades amb consultes

Per recuperar dades, utilitza el hook useQuery proporcionat per Apollo Client.

Exemple: Recuperació de dades amb `useQuery`

import React from 'react';
import { useQuery, gql } from '@apollo/client';

// Defineix la consulta GraphQL
const GET_HELLO = gql`
  query GetHello {
    hello
  }
`;

function Hello() {
  const { loading, error, data } = useQuery(GET_HELLO);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error :(</p>;

  return <h1>{data.hello}</h1>;
}

export default Hello;

Modificació de dades amb mutacions

Per modificar dades, utilitza el hook useMutation.

Exemple: Ús de `useMutation`

import React from 'react';
import { useMutation, gql } from '@apollo/client';

// Defineix la mutació GraphQL
const ADD_TODO = gql`
  mutation AddTodo($text: String!) {
    addTodo(text: $text) {
      id
      text
    }
  }
`;

function AddTodo() {
  let input;
  const [addTodo, { data }] = useMutation(ADD_TODO);

  return (
    <div>
      <form
        onSubmit={e => {
          e.preventDefault();
          addTodo({ variables: { text: input.value } });
          input.value = '';
        }}
      >
        <input
          ref={node => {
            input = node;
          }}
        />
        <button type="submit">Add Todo</button>
      </form>
    </div>
  );
}

export default AddTodo;

Conclusió

Integrar GraphQL amb React utilitzant Apollo Client permet una recuperació de dades eficient i una gestió d'estat. Entenent els conceptes bàsics de la configuració d'un servidor GraphQL, l'ús d'Apollo Client i la realització de consultes i mutacions, pots construir aplicacions potents i flexibles.

  • Punts Clau:
    • GraphQL proporciona una alternativa més eficient i flexible a REST.
    • Apollo Client simplifica la integració de GraphQL amb React.
    • Utilitza useQuery per recuperar dades i useMutation per modificar dades.

Dominant aquests conceptes, pots aprofitar tot el potencial de GraphQL a les teves aplicacions React.
