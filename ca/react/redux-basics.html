Introducció a Redux

Redux és un contenidor d'estat previsible per a aplicacions JavaScript, sovint utilitzat amb React. Ajuda a gestionar l'estat de la teva aplicació d'una manera més estructurada i previsible.

Conceptes clau

  • Estat: La font única de veritat per a la teva aplicació.
  • Accions: Objectes JavaScript simples que descriuen què ha passat.
  • Reductors: Funcions que determinen com canvia l'estat en resposta a les accions.
  • Magatzem: L'objecte que uneix accions i reductors, mantenint l'estat de l'aplicació.

Configuració de Redux

Per començar a utilitzar Redux en una aplicació React, cal instal·lar els paquets necessaris.

npm install redux react-redux

Creació d'un magatzem Redux

El magatzem és la peça central d'una aplicació Redux. Manté l'estat i permet que es pugui actualitzar mitjançant accions.

import { createStore } from 'redux';

// Un reductor simple
const initialState = { count: 0 };
const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case 'INCREMENT':
      return { count: state.count + 1 };
    case 'DECREMENT':
      return { count: state.count - 1 };
    default:
      return state;
  }
};

// Crear el magatzem
const store = createStore(counterReducer);

Connectar Redux amb React

Per connectar Redux amb React, utilitzes el component Provider de react-redux.

import React from 'react';
import ReactDOM from 'react-dom';
import { Provider } from 'react-redux';
import App from './App'; // El teu component principal App

ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
);

Enviar accions

Les accions s'envien al magatzem per actualitzar l'estat.

// Creadors d'accions
const increment = () => ({ type: 'INCREMENT' });
const decrement = () => ({ type: 'DECREMENT' });

// Enviar accions
store.dispatch(increment());
store.dispatch(decrement());

Utilitzar `connect` per accedir a l'estat i enviar accions en components

La funció connect de react-redux s'utilitza per connectar components React al magatzem Redux.

import React from 'react';
import { connect } from 'react-redux';

// Un component simple
const Counter = ({ count, increment, decrement }) => (
  <div>
    <h1>{count}</h1>
    <button onClick={increment}>Incrementar</button>
    <button onClick={decrement}>Disminuir</button>
  </div>
);

// Mapar estat i enviar a props
const mapStateToProps = state => ({ count: state.count });
const mapDispatchToProps = dispatch => ({
  increment: () => dispatch(increment()),
  decrement: () => dispatch(decrement())
});

// Connectar el component al magatzem Redux
export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Resum

Redux ajuda a gestionar l'estat de la teva aplicació d'una manera previsible. Entenent els conceptes bàsics—estat, accions, reductors i magatzem—pots utilitzar efectivament Redux en les teves aplicacions React. Configurar Redux implica crear un magatzem, connectar-lo a la teva aplicació React utilitzant el Provider, enviar accions per actualitzar l'estat i utilitzar connect per accedir a l'estat i enviar accions en els teus components. Amb aquests conceptes bàsics, pots començar a construir aplicacions més complexes i escalables.
