Introducció als Props

Els props (abreviatura de "properties") són un concepte fonamental en React que et permeten passar dades d'un component a un altre. S'utilitzen per personalitzar i configurar components, fent-los més reutilitzables i dinàmics.

  • Els props són de només lectura: No haurien de ser modificats pel component receptor.
  • Els props es passen de pare a fill: Les dades flueixen en una direcció, des dels components pare als fills.

Ús dels Props en Components Funcionals

Els components funcionals són més senzills i concisos. Aquí tens com pots utilitzar els props en un component funcional:

import React from 'react';

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

export default Greeting;

En l'exemple anterior:

  • Greeting és un component funcional que pren props com a argument.
  • props.name s'utilitza per mostrar el nom passat al component.

Exemple d'Ús

import React from 'react';
import ReactDOM from 'react-dom';
import Greeting from './Greeting';

ReactDOM.render(<Greeting name="Alice" />, document.getElementById('root'));

Aquí, el component Greeting es renderitza amb el prop name establert a "Alice".

Ús dels Props en Components de Classe

Els components de classe també poden utilitzar props. Aquí tens un exemple:

import React, { Component } from 'react';

class Greeting extends Component {
  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}

export default Greeting;

En aquest exemple:

  • Greeting és un component de classe que accedeix als props utilitzant this.props.

Exemple d'Ús

import React from 'react';
import ReactDOM from 'react-dom';
import Greeting from './Greeting';

ReactDOM.render(<Greeting name="Bob" />, document.getElementById('root'));

Aquí, el component Greeting es renderitza amb el prop name establert a "Bob".

Props per Defecte

Pots definir valors per defecte per als props en cas que no siguin proporcionats pel component pare.

Props per Defecte en Components Funcionals

import React from 'react';

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Greeting.defaultProps = {
  name: 'Stranger'
};

export default Greeting;

Props per Defecte en Components de Classe

import React, { Component } from 'react';

class Greeting extends Component {
  static defaultProps = {
    name: 'Stranger'
  };

  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}

export default Greeting;

Tipus de Props

Els tipus de props s'utilitzen per fer complir el tipus i l'estructura dels props passats a un component. Això ajuda a detectar errors i assegura que els components s'utilitzen correctament.

Ús de PropTypes

import React from 'react';
import PropTypes from 'prop-types';

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Greeting.propTypes = {
  name: PropTypes.string
};

export default Greeting;

En aquest exemple:

  • PropTypes.string assegura que el prop name és una cadena.

Passar Múltiples Props

Pots passar múltiples props a un component incloent-los com a atributs:

import React from 'react';

function UserCard(props) {
  return (
    <div>
      <h2>{props.name}</h2>
      <p>Age: {props.age}</p>
      <p>Email: {props.email}</p>
    </div>
  );
}

export default UserCard;

Exemple d'Ús

import React from 'react';
import ReactDOM from 'react-dom';
import UserCard from './UserCard';

ReactDOM.render(
  <UserCard name="Charlie" age={30} email="charlie@example.com" />,
  document.getElementById('root')
);

Resum

Els props són una característica poderosa en React que et permeten passar dades i configuració des dels components pare als fills. Són de només lectura i no haurien de ser modificats pel component receptor. Utilitzant props per defecte i tipus de props, pots fer els teus components més robustos i fàcils d'utilitzar. Entendre com utilitzar efectivament els props és essencial per construir components React dinàmics i reutilitzables.
