Introducció a React Router
Configuració de React Router ►

React Router és una biblioteca potent que permet el routing dinàmic en una aplicació React. Et permet crear aplicacions d'una sola pàgina amb navegació sense que la pàgina es refresqui mentre l'usuari navega. Aquest tema cobrirà els conceptes bàsics de React Router, incloent la seva configuració, definició de rutes i l'ús de diversos components proporcionats per la biblioteca.

Configurant React Router

Per començar amb React Router, necessites instal·lar la biblioteca en el teu projecte React. Pots fer-ho utilitzant npm o yarn.

# Utilitzant npm
npm install react-router-dom

# Utilitzant yarn
yarn add react-router-dom

Un cop instal·lat, necessites embolicar la teva aplicació amb el component BrowserRouter per habilitar el routing.

import React from 'react';
import ReactDOM from 'react-dom';
import { BrowserRouter } from 'react-router-dom';
import App from './App';

ReactDOM.render(
  <BrowserRouter>
    <App />
  </BrowserRouter>,
  document.getElementById('root')
);

Definint Rutes

Les rutes es defineixen utilitzant el component Route. Cada component Route especifica un camí i el component que s'ha de renderitzar quan el camí coincideix amb l'URL.

import React from 'react';
import { Route, Switch } from 'react-router-dom';
import Home from './Home';
import About from './About';
import Contact from './Contact';

function App() {
  return (
    <Switch>
      <Route exact path="/" component={Home} />
      <Route path="/about" component={About} />
      <Route path="/contact" component={Contact} />
    </Switch>
  );
}

export default App;

Conceptes Clau

  • BrowserRouter: Una implementació de router que utilitza l'API d'historial HTML5 per mantenir la teva UI sincronitzada amb l'URL.
  • Route: Renderitza un component UI quan el seu camí coincideix amb l'URL actual.
  • Switch: Renderitza el primer fill Route que coincideix amb l'URL actual. Això és útil per assegurar que només una ruta es renderitzi alhora.

Navegació

Per navegar entre rutes, pots utilitzar el component Link. Aquest component és similar a una etiqueta d'ancoratge HTML però evita que la pàgina es recarregui.

import React from 'react';
import { Link } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <ul>
        <li><Link to="/">Home</Link></li>
        <li><Link to="/about">About</Link></li>
        <li><Link to="/contact">Contact</Link></li>
      </ul>
    </nav>
  );
}

export default Navigation;

Rutes Imbricades

React Router et permet definir rutes imbricades, que poden ser útils per crear dissenys complexos.

import React from 'react';
import { Route, Switch, Link, useRouteMatch } from 'react-router-dom';

function Topics() {
  let { path, url } = useRouteMatch();

  return (
    <div>
      <h2>Topics</h2>
      <ul>
        <li><Link to={`${url}/components`}>Components</Link></li>
        <li><Link to={`${url}/props-v-state`}>Props v. State</Link></li>
      </ul>

      <Switch>
        <Route exact path={path}>
          <h3>Please select a topic.</h3>
        </Route>
        <Route path={`${path}/:topicId`}>
          <Topic />
        </Route>
      </Switch>
    </div>
  );
}

function Topic() {
  let { topicId } = useParams();
  return <h3>Requested topic ID: {topicId}</h3>;
}

Redireccions i Navegació Programàtica

De vegades necessites redirigir usuaris o navegar programàticament. React Router proporciona el component Redirect i el hook useHistory per a aquests propòsits.

Component Redirect

import React from 'react';
import { Redirect } from 'react-router-dom';

function OldPage() {
  return <Redirect to="/new-page" />;
}

Hook useHistory

import React from 'react';
import { useHistory } from 'react-router-dom';

function HomePage() {
  let history = useHistory();

  function handleClick() {
    history.push('/about');
  }

  return (
    <div>
      <button onClick={handleClick}>Go to About Page</button>
    </div>
  );
}

Resum

React Router és una eina essencial per construir aplicacions d'una sola pàgina amb React. Et permet:

  • Definir rutes utilitzant el component Route.
  • Navegar entre rutes utilitzant el component Link.
  • Gestionar rutes imbricades per a dissenys complexos.
  • Redirigir i navegar programàticament utilitzant el component Redirect i el hook useHistory.

Dominant React Router, pots crear aplicacions web dinàmiques i responsives que proporcionen una experiència d'usuari sense interrupcions.
