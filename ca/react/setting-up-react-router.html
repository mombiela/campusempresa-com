Introducció

React Router és una biblioteca potent que permet la navegació entre vistes o components en una aplicació React. Et permet construir una aplicació d'una sola pàgina amb navegació sense que la pàgina es recarregui mentre l'usuari navega. Aquesta secció et guiarà a través de la configuració de React Router en el teu projecte React.

Instal·lació de React Router

Per començar amb React Router, necessites instal·lar el paquet react-router-dom. Pots fer-ho utilitzant npm o yarn.

Utilitzant npm

npm install react-router-dom

Utilitzant yarn

yarn add react-router-dom

Configuració Bàsica

Un cop hagis instal·lat react-router-dom, pots començar a configurar les teves rutes.

Importació dels Components Necessaris

Primer, importa els components necessaris de react-router-dom en el teu fitxer principal de l'aplicació (normalment App.js o index.js).

import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';

Creació de Rutes Bàsiques

A continuació, embolcalla la teva aplicació amb el component Router i defineix les teves rutes utilitzant el component Route. El component Switch s'utilitza per renderitzar només la primera ruta que coincideix amb la ubicació actual.

import React from 'react';
import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Contact from './components/Contact';

function App() {
  return (
    <Router>
      <Switch>
        <Route exact path="/" component={Home} />
        <Route path="/about" component={About} />
        <Route path="/contact" component={Contact} />
      </Switch>
    </Router>
  );
}

export default App;

Explicació

  • Router: Embolcalla tota l'aplicació i permet la navegació.
  • Switch: Assegura que només una ruta es renderitzi a la vegada.
  • Route: Defineix un camí i el component que s'ha de renderitzar quan el camí coincideix.

Rutes Imbricades

React Router també suporta rutes imbricades, que et permeten definir rutes dins d'altres rutes.

Exemple de Rutes Imbricades

import React from 'react';
import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';

function App() {
  return (
    <Router>
      <Switch>
        <Route path="/dashboard" component={Dashboard} />
      </Switch>
    </Router>
  );
}

function Dashboard() {
  return (
    <div>
      <h2>Dashboard</h2>
      <Switch>
        <Route path="/dashboard/profile" component={Profile} />
        <Route path="/dashboard/settings" component={Settings} />
      </Switch>
    </div>
  );
}

export default App;

Explicació

  • Switch Imbricat: El component Switch dins del component Dashboard permet la navegació imbricada dins del tauler de control.

Enllaços de Navegació

Per navegar entre rutes, pots utilitzar el component Link de react-router-dom.

Exemple d'Enllaços de Navegació

import React from 'react';
import { BrowserRouter as Router, Route, Switch, Link } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Contact from './components/Contact';

function App() {
  return (
    <Router>
      <nav>
        <ul>
          <li><Link to="/">Home</Link></li>
          <li><Link to="/about">About</Link></li>
          <li><Link to="/contact">Contact</Link></li>
        </ul>
      </nav>
      <Switch>
        <Route exact path="/" component={Home} />
        <Route path="/about" component={About} />
        <Route path="/contact" component={Contact} />
      </Switch>
    </Router>
  );
}

export default App;

Explicació

  • Link: El component Link s'utilitza per crear enllaços de navegació. Evita que la pàgina es recarregui i permet transicions suaus entre rutes.

Conclusió

Configurar React Router és essencial per construir aplicacions d'una sola pàgina amb React. Seguint els passos descrits en aquesta secció, pots crear rutes bàsiques i imbricades, i navegar entre elles utilitzant el component Link. Aquest coneixement fonamental et permetrà construir estructures de navegació més complexes a mesura que la teva aplicació creixi.
