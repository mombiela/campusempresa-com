El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert en React. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar amb el desenvolupament en React.

Prerequisits

Abans de configurar l'entorn, assegura't de tenir els següents prerequisits:

  • Node.js: React depèn de Node.js per a la gestió de paquets i l'execució de scripts.
  • npm o Yarn: Aquests són gestors de paquets utilitzats per instal·lar biblioteques i dependències.

Instal·lació de Node.js i npm

  1. Descarrega Node.js: Visita el lloc web oficial de Node.js i descarrega la versió LTS.
  2. Instal·la Node.js: Segueix les instruccions d'instal·lació per al teu sistema operatiu.
  3. Verifica la Instal·lació: 
    node -v
npm -v
    Això hauria de mostrar les versions instal·lades de Node.js i npm.

Instal·lació de Yarn (Opcional)

Yarn és una alternativa a npm. Per instal·lar Yarn, executa:

npm install --global yarn

Verifica la instal·lació:

yarn -v

Configuració d'un Projecte React

Els projectes React es configuren típicament utilitzant Create React App (CRA), una eina que proporciona un esborrany per a aplicacions React.

Utilitzant Create React App

  1. Instal·la Create React App:

    npx create-react-app my-app

    Substitueix my-app pel nom desitjat del teu projecte.

  2. Navega al Directori del Projecte:

    cd my-app

  3. Inicia el Servidor de Desenvolupament:

    npm start

    Això iniciarà el servidor de desenvolupament i obrirà la teva nova aplicació React en el navegador web predeterminat.

Estructura del Projecte

Després de configurar el projecte, veuràs una estructura similar a aquesta:

my-app/
├── node_modules/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── App.css
│   ├── App.js
│   ├── App.test.js
│   ├── index.css
│   ├── index.js
│   └── ...
├── .gitignore
├── package.json
└── README.md
  • node_modules/: Conté totes les dependències del projecte.
  • public/: Conté fitxers estàtics com index.html.
  • src/: Conté el codi font de la teva aplicació React.
  • package.json: Llista les dependències i scripts del projecte.
  • README.md: Conté informació sobre el projecte.

Editor de Codi

Un bon editor de codi pot millorar significativament la teva productivitat. Les opcions populars inclouen:

  • Visual Studio Code (VS Code): Altament personalitzable amb una gran quantitat d'extensions.
  • Sublime Text: Lleuger i ràpid.
  • Atom: Desenvolupat per GitHub, altament personalitzable.

Extensions Recomanades per a VS Code

  • ESLint: Integra ESLint en VS Code.
  • Prettier: Formatador de codi.
  • Reactjs code snippets: Proporciona fragments de codi específics per a React.
  • Bracket Pair Colorizer: Coloreja els claudàtors coincidents per fer el codi més llegible.

Control de Versions amb Git

Utilitzar el control de versions és crucial per gestionar els canvis de codi i la col·laboració.

Configuració de Git

  1. Instal·la Git: Descarrega i instal·la Git des del lloc web oficial.
  2. Configura Git: 
    git config --global user.name "El teu Nom"
git config --global user.email "el.teu.email@example.com"

Inicialització d'un Repositori Git

  1. Inicialitza Git:

    git init

  2. Afegeix Fitxers:

    git add .

  3. Fes un Commit dels Canvis:

    git commit -m "Commit inicial"

Conclusió

Configurar l'entorn de desenvolupament és un pas crucial en el teu viatge amb React. Assegurant-te que tens Node.js, npm o Yarn, un bon editor de codi i control de versions amb Git, estaràs ben preparat per submergir-te en el desenvolupament amb React. Aquesta configuració fonamental agilitzarà el teu flux de treball i millorarà la teva productivitat a mesura que progressis des de nivell principiant fins a avançat en React.
