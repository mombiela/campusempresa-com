Introducció a React

React és una popular biblioteca de JavaScript per construir interfícies d'usuari, especialment aplicacions d'una sola pàgina on necessites una experiència d'usuari ràpida i interactiva. Va ser desenvolupada per Facebook i és mantinguda per Facebook i una comunitat de desenvolupadors individuals i empreses.

Conceptes Clau

  • Arquitectura Basada en Components: Les aplicacions React es construeixen utilitzant components, que són peces reutilitzables de la interfície d'usuari.
  • DOM Virtual: React utilitza un DOM virtual per optimitzar les actualitzacions i el renderitzat, fent l'aplicació més ràpida.
  • JSX: JavaScript XML (JSX) és una extensió de sintaxi per a JavaScript que et permet escriure HTML directament dins de JavaScript.
  • Flux de Dades Unidireccional: Les dades en React flueixen en una direcció, fent l'aplicació més predictible i fàcil de depurar.

Per què utilitzar React?

  • Rendiment: El DOM virtual assegura que les actualitzacions siguin ràpides i eficients.
  • Reutilitzabilitat: Els components es poden reutilitzar en diferents parts de l'aplicació, reduint la redundància.
  • Comunitat i Ecosistema: Una gran comunitat i un ric ecosistema de biblioteques i eines faciliten el desenvolupament.
  • Flexibilitat: React es pot utilitzar amb altres biblioteques o marcs, com Redux per a la gestió d'estat o Next.js per al renderitzat del costat del servidor.

Exemple Bàsic

Comencem amb un exemple senzill per il·lustrar com funciona React. Crearem un component bàsic que mostra "Hola, Món!".

Exemple de Codi

import React from 'react';
import ReactDOM from 'react-dom';

// Definir un component funcional
function HelloWorld() {
  return <h1>Hola, Món!</h1>;
}

// Renderitzar el component al DOM
ReactDOM.render(<HelloWorld />, document.getElementById('root'));

Explicació

  • Declaracions d'Importació: Importem React i ReactDOM dels seus respectius paquets.
  • Component Funcional: HelloWorld és un component funcional que retorna un element JSX.
  • Renderització: ReactDOM.render s'utilitza per renderitzar el component HelloWorld dins de l'element DOM amb l'id root.

Components en React

Els components són els blocs de construcció d'una aplicació React. Hi ha dos tipus principals de components:

  • Components Funcionals: Aquests són funcions simples que retornen JSX.
  • Components de Classe: Aquests són classes ES6 que estenen React.Component i tenen un mètode render.

Exemple de Component Funcional

function Greeting(props) {
  return <h1>Hola, {props.name}!</h1>;
}

Exemple de Component de Classe

class Greeting extends React.Component {
  render() {
    return <h1>Hola, {this.props.name}!</h1>;
  }
}

JSX

JSX és una extensió de sintaxi que et permet escriure codi semblant a HTML dins de JavaScript. Fa que el codi sigui més llegible i fàcil d'escriure.

Exemple de JSX

const element = <h1>Hola, Món!</h1>;

Explicació

  • Sintaxi semblant a HTML: El codi sembla HTML però és realment JavaScript.
  • Transpilació: JSX es transpila a JavaScript utilitzant eines com Babel.

DOM Virtual

El DOM virtual és una còpia lleugera del DOM real. React l'utilitza per optimitzar les actualitzacions i el renderitzat.

Com Funciona

  1. Renderització Inicial: React crea un arbre de DOM virtual.
  2. Actualització: Quan l'estat o les propietats canvien, React crea un nou arbre de DOM virtual.
  3. Comparació: React compara el nou arbre de DOM virtual amb l'anterior.
  4. Reconciliació: React actualitza només les parts del DOM real que han canviat.

Conclusió

React és una biblioteca potent per construir interfícies d'usuari ràpides i interactives. La seva arquitectura basada en components, el DOM virtual i la sintaxi JSX la fan una elecció popular entre els desenvolupadors. Entendre aquests conceptes bàsics és essencial per construir aplicacions React eficients i escalables. A mesura que avancis en aquest curs, aprofundiràs en aquests temes i aprendràs a construir aplicacions complexes utilitzant React.
