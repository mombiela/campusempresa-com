Introducció

La navegació programàtica en React es refereix a la capacitat de navegar entre diferents vistes o pàgines en una aplicació React utilitzant codi, en lloc de dependre únicament de les interaccions de l'usuari com fer clic en enllaços. Això és particularment útil per a escenaris com redirigir usuaris després de l'enviament de formularis, gestionar fluxos d'autenticació o navegar basant-se en l'estat de l'aplicació.

Conceptes Clau

  • React Router: Una biblioteca popular per gestionar el routing en aplicacions React.
  • Objecte History: Un objecte proporcionat per React Router que permet la navegació programàtica.
  • Hook useNavigate: Un hook introduït en React Router v6 per a una navegació més fàcil.
  • Component Redirect: Un component utilitzat en versions anteriors de React Router per a la navegació.

Configuració de React Router

Abans d'endinsar-nos en la navegació programàtica, cal configurar React Router en el teu projecte.

  1. Instal·lar React Router:

    npm install react-router-dom

  2. Configuració Bàsica:

    import React from 'react';
import { BrowserRouter as Router, Route, Routes } from 'react-router-dom';
import Home from './Home';
import About from './About';

function App() {
  return (
    <Router>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </Router>
  );
}

export default App;

Navegació Programàtica Utilitzant el Hook useNavigate

El hook useNavigate és la manera recomanada de navegar programàticament en React Router v6.

Exemple

import React from 'react';
import { useNavigate } from 'react-router-dom';

function Home() {
  const navigate = useNavigate();

  const goToAbout = () => {
    navigate('/about');
  };

  return (
    <div>
      <h1>Pàgina d'Inici</h1>
      <button onClick={goToAbout}>Anar a Sobre Nosaltres</button>
    </div>
  );
}

export default Home;

Explicació

  • useNavigate: Aquest hook retorna una funció que es pot utilitzar per navegar a diferents rutes.
  • navigate('/about'): Aquesta crida de funció canvia la ruta a /about.

Gestionar la Navegació amb Estat

També pots passar estat juntament amb la navegació per compartir dades entre components.

Exemple

import React from 'react';
import { useNavigate } from 'react-router-dom';

function Home() {
  const navigate = useNavigate();

  const goToAbout = () => {
    navigate('/about', { state: { fromHome: true } });
  };

  return (
    <div>
      <h1>Pàgina d'Inici</h1>
      <button onClick={goToAbout}>Anar a Sobre Nosaltres</button>
    </div>
  );
}

export default Home;

Accedir a l'Estat en el Component Destí

import React from 'react';
import { useLocation } from 'react-router-dom';

function About() {
  const location = useLocation();
  const { fromHome } = location.state || {};

  return (
    <div>
      <h1>Pàgina Sobre Nosaltres</h1>
      {fromHome && <p>Benvingut des de la Pàgina d'Inici!</p>}
    </div>
  );
}

export default About;

Explicació

  • navigate('/about', { state: { fromHome: true } }): Això navega a la ruta /about i passa estat.
  • useLocation: Aquest hook retorna l'objecte de la ubicació actual, que inclou l'estat passat durant la navegació.

Redirigir Després de l'Enviament d'un Formulari

Redirigir usuaris després de l'enviament de formularis és un cas d'ús comú per a la navegació programàtica.

Exemple

import React, { useState } from 'react';
import { useNavigate } from 'react-router-dom';

function LoginForm() {
  const [username, setUsername] = useState('');
  const navigate = useNavigate();

  const handleSubmit = (e) => {
    e.preventDefault();
    // Realitzar la lògica d'inici de sessió aquí
    navigate('/dashboard');
  };

  return (
    <form onSubmit={handleSubmit}>
      <label>
        Nom d'usuari:
        <input type="text" value={username} onChange={(e) => setUsername(e.target.value)} />
      </label>
      <button type="submit">Iniciar Sessió</button>
    </form>
  );
}

export default LoginForm;

Explicació

  • handleSubmit: Aquesta funció gestiona l'enviament del formulari i navega a la ruta /dashboard després d'un inici de sessió exitós.

Conclusió

La navegació programàtica en React és una característica poderosa que millora l'experiència de l'usuari permetent als desenvolupadors controlar el flux de navegació a través del codi. Utilitzant eines com useNavigate i useLocation de React Router, pots implementar fàcilment escenaris de navegació complexos en les teves aplicacions. Entendre aquests conceptes és crucial per construir aplicacions React dinàmiques i responsives.
