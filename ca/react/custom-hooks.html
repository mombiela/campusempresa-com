Introducció als Hooks personalitzats

Els Hooks personalitzats són una característica poderosa de React que et permet extreure i reutilitzar la lògica a través de múltiples components. Et permeten encapsular la lògica amb estat i els efectes secundaris, fent que el teu codi sigui més modular i fàcil de mantenir.

Conceptes clau

  • Reutilització: Els Hooks personalitzats et permeten reutilitzar la lògica amb estat sense duplicar codi.
  • Encapsulació: Ajuden a encapsular la lògica complexa, fent que els components siguin més nets i enfocats en el renderitzat.
  • Composició: Els Hooks personalitzats es poden compondre junts per construir funcionalitats més complexes.

Creació d'un Hook personalitzat

Per crear un hook personalitzat, simplement escrius una funció que utilitzi hooks de React com useState, useEffect, etc., i retornes els valors o funcions necessaris.

Exemple: Hook useFetch

Creem un hook personalitzat anomenat useFetch que obté dades d'una API.

import { useState, useEffect } from 'react';

function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const response = await fetch(url);
        if (!response.ok) {
          throw new Error('Network response was not ok');
        }
        const result = await response.json();
        setData(result);
      } catch (error) {
        setError(error);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, [url]);

  return { data, loading, error };
}

export default useFetch;

Ús del Hook useFetch

Ara que tenim el nostre hook useFetch, utilitzem-lo en un component.

import React from 'react';
import useFetch from './useFetch';

function DataFetchingComponent() {
  const { data, loading, error } = useFetch('https://api.example.com/data');

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error: {error.message}</p>;

  return (
    <div>
      <h1>Fetched Data</h1>
      <pre>{JSON.stringify(data, null, 2)}</pre>
    </div>
  );
}

export default DataFetchingComponent;

Hooks personalitzats avançats

Els Hooks personalitzats també poden gestionar estats i efectes secundaris més complexos, i es poden compondre junts per construir funcionalitats més sofisticades.

Exemple: Hook useLocalStorage

Creem un hook personalitzat anomenat useLocalStorage que interactua amb l'emmagatzematge local del navegador.

import { useState } from 'react';

function useLocalStorage(key, initialValue) {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      console.error(error);
      return initialValue;
    }
  });

  const setValue = (value) => {
    try {
      const valueToStore = value instanceof Function ? value(storedValue) : value;
      setStoredValue(valueToStore);
      window.localStorage.setItem(key, JSON.stringify(valueToStore));
    } catch (error) {
      console.error(error);
    }
  };

  return [storedValue, setValue];
}

export default useLocalStorage;

Ús del Hook useLocalStorage

Aquí tens com pots utilitzar el hook useLocalStorage en un component.

import React from 'react';
import useLocalStorage from './useLocalStorage';

function LocalStorageComponent() {
  const [name, setName] = useLocalStorage('name', 'John Doe');

  return (
    <div>
      <h1>Hello, {name}!</h1>
      <input
        type="text"
        value={name}
        onChange={(e) => setName(e.target.value)}
      />
    </div>
  );
}

export default LocalStorageComponent;

Conclusió

Els Hooks personalitzats són una eina versàtil a React que t'ajuden a escriure codi més net i fàcil de mantenir encapsulant la lògica amb estat i els efectes secundaris. Promouen la reutilització i la composició, fent que els teus components estiguin més enfocats en el renderitzat i menys en la gestió de la lògica. Entenent i utilitzant els hooks personalitzats, pots millorar significativament l'estructura i la llegibilitat de les teves aplicacions React.
