Introducció a la Context API

La Context API és una característica poderosa de React que et permet gestionar l'estat global i compartir dades a través de l'arbre de components sense haver de passar propietats manualment a cada nivell. És particularment útil per gestionar temes, autenticació d'usuaris i altres configuracions globals.

Conceptes Clau

  • Context: Una manera de passar dades a través de l'arbre de components sense haver de passar propietats manualment a cada nivell.
  • Provider: Un component que proporciona el valor del context als seus fills.
  • Consumer: Un component que consumeix el valor del context.

Creant un Context

Per crear un context, utilitzes la funció React.createContext. Aquesta funció retorna un objecte de context amb un component Provider i un component Consumer.

import React from 'react';

// Crear un Context
const MyContext = React.createContext();

Utilitzant el Provider

El component Provider s'utilitza per envoltar la part de la teva aplicació on vols que el context estigui disponible. Accepta una propietat value que serà passada als components consumidors.

import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';

// Crear un Context
const MyContext = React.createContext();

const MyProvider = ({ children }) => {
  const value = { name: 'John Doe' };

  return (
    <MyContext.Provider value={value}>
      {children}
    </MyContext.Provider>
  );
};

ReactDOM.render(
  <MyProvider>
    <App />
  </MyProvider>,
  document.getElementById('root')
);

Consumint Context

Hi ha múltiples maneres de consumir context en un component de React:

Utilitzant el Component Consumer

El component Consumer utilitza el patró de render prop per accedir al valor del context.

import React from 'react';
import { MyContext } from './MyProvider';

const MyComponent = () => {
  return (
    <MyContext.Consumer>
      {value => <div>{value.name}</div>}
    </MyContext.Consumer>
  );
};

export default MyComponent;

Utilitzant el Hook useContext

El hook useContext és una manera més senzilla de consumir context en components funcionals.

import React, { useContext } from 'react';
import { MyContext } from './MyProvider';

const MyComponent = () => {
  const contextValue = useContext(MyContext);

  return <div>{contextValue.name}</div>;
};

export default MyComponent;

Actualitzant Context

Per actualitzar el valor del context, pots utilitzar la gestió d'estat dins del component Provider.

import React, { useState } from 'react';
import ReactDOM from 'react-dom';
import App from './App';

const MyContext = React.createContext();

const MyProvider = ({ children }) => {
  const [name, setName] = useState('John Doe');

  const value = {
    name,
    setName,
  };

  return (
    <MyContext.Provider value={value}>
      {children}
    </MyContext.Provider>
  );
};

ReactDOM.render(
  <MyProvider>
    <App />
  </MyProvider>,
  document.getElementById('root')
);

Exemple d'Actualització de Context

import React, { useContext } from 'react';
import { MyContext } from './MyProvider';

const MyComponent = () => {
  const { name, setName } = useContext(MyContext);

  const handleChange = () => {
    setName('Jane Doe');
  };

  return (
    <div>
      <div>{name}</div>
      <button onClick={handleChange}>Canviar Nom</button>
    </div>
  );
};

export default MyComponent;

Context API vs. Redux

| Característica | Context API | Redux | |------------------|---------------------------------|--------------------------------| | Configuració | Configuració mínima | Requereix biblioteques addicionals | | Corba d'aprenentatge | Més fàcil per a principiants | Corba d'aprenentatge més pronunciada | | Cas d'ús | Gestió d'estat simple | Gestió d'estat complex | | Rendiment | Pot causar renders innecessaris | Optimitzat per a aplicacions grans |

Conclusió

La Context API és una eina poderosa i flexible per gestionar l'estat global en aplicacions React. És particularment útil per a escenaris de gestió d'estat més simples i pot reduir significativament la necessitat de passar propietats. Per a necessitats de gestió d'estat més complexes, podries considerar utilitzar Redux o altres biblioteques de gestió d'estat. Entendre quan i com utilitzar la Context API t'ajudarà a construir aplicacions React més mantenibles i escalables.
