Introducció a React Native

React Native és un framework popular per construir aplicacions mòbils utilitzant JavaScript i React. Permet als desenvolupadors crear aplicacions mòbils renderitzades nativament per a iOS i Android utilitzant una sola base de codi.

Conceptes Clau

  • Desenvolupament Multiplataforma: Escriu una vegada, executa en ambdós iOS i Android.
  • Components Natius: Utilitza components natius com View, Text i Image per construir la interfície d'usuari.
  • Hot Reloading: Veure els canvis en temps real sense reconstruir tota l'aplicació.
  • JavaScript i React: Aprofita el teu coneixement existent de JavaScript i React.

Configuració de l'Entorn de Desenvolupament

Abans de començar a construir amb React Native, necessites configurar el teu entorn de desenvolupament.

Passos per Configurar

  1. Instal·lar Node.js: React Native requereix Node.js. Descarrega i instal·la'l des del lloc web oficial de Node.js.
  2. Instal·lar React Native CLI: 
    npm install -g react-native-cli
  3. Configurar Android Studio: Per al desenvolupament d'Android, instal·la Android Studio i configura l'SDK d'Android.
  4. Configurar Xcode: Per al desenvolupament d'iOS, instal·la Xcode des de la Mac App Store.

Crear un Nou Projecte

Per crear un nou projecte de React Native, utilitza la següent comanda:

npx react-native init MyNewProject

Construint la Teva Primera Aplicació React Native

Construïm una aplicació simple de "Hello, World!" per familiaritzar-nos amb React Native.

Estructura Bàsica

Una aplicació de React Native típicament consisteix en els següents fitxers:

  • App.js: El fitxer principal del component.
  • index.js: Punt d'entrada de l'aplicació.

Codi d'Exemple

// App.js
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Hello, World!</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  text: {
    fontSize: 20,
  },
});

export default App;

Explicació

  • View: Un contenidor que suporta disseny amb flexbox, estil, maneig de tocs i controls d'accessibilitat.
  • Text: Un component per mostrar text.
  • StyleSheet: Una utilitat per crear estils.

Treballant amb Components i Estat

Els components de React Native poden ser funcionals o basats en classes. La gestió de l'estat és crucial per a aplicacions dinàmiques.

Components Funcionals amb Hooks

import React, { useState } from 'react';
import { View, Text, Button } from 'react-native';

const Counter = () => {
  const [count, setCount] = useState(0);

  return (
    <View>
      <Text>{count}</Text>
      <Button title="Increment" onPress={() => setCount(count + 1)} />
    </View>
  );
};

export default Counter;

Components de Classe

import React, { Component } from 'react';
import { View, Text, Button } from 'react-native';

class Counter extends Component {
  state = {
    count: 0,
  };

  increment = () => {
    this.setState({ count: this.state.count + 1 });
  };

  render() {
    return (
      <View>
        <Text>{this.state.count}</Text>
        <Button title="Increment" onPress={this.increment} />
      </View>
    );
  }
}

export default Counter;

Navegació en React Native

La navegació és essencial per a aplicacions amb múltiples pantalles. React Navigation és una biblioteca popular per gestionar la navegació.

Instal·lant React Navigation

npm install @react-navigation/native
npm install @react-navigation/stack

Exemple Bàsic de Navegació

import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
import { Button, View, Text } from 'react-native';

const HomeScreen = ({ navigation }) => (
  <View>
    <Text>Home Screen</Text>
    <Button title="Go to Details" onPress={() => navigation.navigate('Details')} />
  </View>
);

const DetailsScreen = () => (
  <View>
    <Text>Details Screen</Text>
  </View>
);

const Stack = createStackNavigator();

const App = () => (
  <NavigationContainer>
    <Stack.Navigator initialRouteName="Home">
      <Stack.Screen name="Home" component={HomeScreen} />
      <Stack.Screen name="Details" component={DetailsScreen} />
    </Stack.Navigator>
  </NavigationContainer>
);

export default App;

Estilització en React Native

L'estilització en React Native es fa utilitzant objectes JavaScript, similar al CSS però amb algunes diferències.

Exemple d'Estils

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  text: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
});

Temes Avançats

Optimització del Rendiment

  • Utilitza PureComponent: Evita renders innecessaris.
  • Memorització: Utilitza React.memo i useMemo per emmagatzemar en memòria cau càlculs costosos.
  • Càrrega Peresosa: Carrega components només quan sigui necessari utilitzant React.lazy.

Integració de Mòduls Natius

De vegades necessites utilitzar codi natiu per raons de rendiment o funcionalitat.

Exemple: Mòdul Natiu

react-native init MyNativeModuleApp
cd MyNativeModuleApp
npm install react-native-create-library
react-native-create-library MyNativeModule

Conclusió

React Native és un framework potent per construir aplicacions mòbils multiplataforma. Aprofitant el teu coneixement existent de JavaScript i React, pots crear aplicacions mòbils natives d'alt rendiment. Aquest curs ha cobert els conceptes bàsics de la configuració del teu entorn, la construcció de components simples, la gestió de l'estat, la navegació entre pantalles i l'estilització de la teva aplicació. A mesura que avancis, pots explorar tècniques d'optimització del rendiment i integrar mòduls natius per millorar la funcionalitat de la teva aplicació. Feliç codificació!
