Introducció al Middleware de Redux

El middleware de Redux proporciona un punt d'extensió de tercers entre l'enviament d'una acció i el moment en què arriba al reductor. El middleware es pot utilitzar per a la registració, informes d'errors, realització de tasques asíncrones, etc.

  • Middleware: Funcions que intercepten accions abans que arribin al reductor.
  • Propòsit: Millorar Redux amb funcionalitats addicionals.
  • Usos Comuns: Registració, maneig d'accions asíncrones, informes d'errors.

Com Funciona el Middleware

El middleware es troba entre l'enviament d'una acció i el reductor. Pot inspeccionar, modificar o retardar accions.

Exemple Bàsic de Middleware

const loggerMiddleware = store => next => action => {
  console.log('Enviant:', action);
  let result = next(action);
  console.log('Següent Estat:', store.getState());
  return result;
};
  • store: La botiga de Redux.
  • next: El següent middleware en la cadena o el reductor si no hi ha cap altre middleware present.
  • action: L'acció que s'està enviant.

Aplicació del Middleware

Per aplicar el middleware, utilitza la funció applyMiddleware de Redux.

Exemple d'Aplicació del Middleware

import { createStore, applyMiddleware } from 'redux';
import rootReducer from './reducers';
import loggerMiddleware from './middlewares/logger';

const store = createStore(
  rootReducer,
  applyMiddleware(loggerMiddleware)
);
  • createStore: Crea la botiga de Redux.
  • applyMiddleware: Millora la botiga amb middleware.
  • rootReducer: El reductor principal per a l'aplicació.

Biblioteques de Middleware Comuns

Diverses biblioteques de middleware són comunament utilitzades en aplicacions de Redux:

  • redux-thunk: Permet escriure creadors d'accions que retornen una funció en lloc d'una acció.
  • redux-saga: Gestiona efectes secundaris en una aplicació de Redux utilitzant funcions generadores.
  • redux-logger: Registra accions i canvis d'estat a la consola.

Exemple de Middleware Thunk

import { createStore, applyMiddleware } from 'redux';
import thunk from 'redux-thunk';
import rootReducer from './reducers';

const store = createStore(
  rootReducer,
  applyMiddleware(thunk)
);

// Creador d'accions utilitzant thunk
const fetchData = () => {
  return dispatch => {
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(data => dispatch({ type: 'FETCH_SUCCESS', payload: data }))
      .catch(error => dispatch({ type: 'FETCH_ERROR', error }));
  };
};
  • redux-thunk: Middleware que permet als creadors d'accions retornar funcions.
  • fetchData: Un creador d'accions que realitza una operació de recuperació asíncrona.

Conceptes Avançats de Middleware

El middleware es pot composar i utilitzar conjuntament per crear funcionalitats complexes.

Composició de Middleware

import { createStore, applyMiddleware } from 'redux';
import thunk from 'redux-thunk';
import logger from 'redux-logger';
import rootReducer from './reducers';

const store = createStore(
  rootReducer,
  applyMiddleware(thunk, logger)
);
  • applyMiddleware(thunk, logger): Composa múltiples funcions de middleware.

Middleware Personalitzat

Crear middleware personalitzat et permet adaptar la funcionalitat a les necessitats de la teva aplicació.

Exemple de Middleware Personalitzat

const customMiddleware = store => next => action => {
  if (action.type === 'SPECIAL_ACTION') {
    console.log('Acció especial detectada!');
  }
  return next(action);
};

const store = createStore(
  rootReducer,
  applyMiddleware(customMiddleware)
);
  • customMiddleware: Un middleware que registra un missatge per a un tipus d'acció específic.

Conclusió

El middleware de Redux és una eina poderosa per a l'extensió de la funcionalitat de Redux. Entenent com funciona el middleware i com aplicar-lo, pots millorar les teves aplicacions de Redux amb registració, accions asíncrones i més. Ja sigui utilitzant biblioteques comunes com redux-thunk o creant middleware personalitzat, dominar el middleware millorarà significativament la teva capacitat per gestionar efectes secundaris i lògica complexa en les teves aplicacions de Redux.
