Redux Toolkit és la manera oficial i recomanada d'escriure la lògica de Redux. Proporciona un conjunt d'eines i millors pràctiques que simplifiquen el procés d'escriure aplicacions Redux. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de Redux Toolkit en el context d'una aplicació React.

Introducció a Redux Toolkit

Redux Toolkit està dissenyat per ajudar-te a escriure la lògica de Redux de manera més fàcil i eficient. Inclou utilitats per crear i gestionar fragments d'estat, escriure reductors i manejar la lògica asincrònica.

Conceptes Clau

  • Fragments: Un fragment és una col·lecció de lògica de reductor de Redux i accions per a una sola característica de la teva aplicació.
  • Reductors: Funcions que determinen els canvis en l'estat d'una aplicació.
  • Accions: Objectes simples que representen una intenció de canviar l'estat.
  • Store: L'objecte que uneix accions i reductors.

Configuració de Redux Toolkit

Instal·lació

Per començar amb Redux Toolkit, necessites instal·lar els paquets necessaris:

npm install @reduxjs/toolkit react-redux

Creació d'un Store de Redux

La funció configureStore de Redux Toolkit simplifica el procés de configurar el store.

import { configureStore } from '@reduxjs/toolkit';

const store = configureStore({
  reducer: {
    // Afegeix els teus reductors aquí
  },
});

export default store;

Creació de Fragments

Un fragment conté la lògica del reductor i les accions per a una part específica de l'estat.

Exemple: Fragment de Comptador

import { createSlice } from '@reduxjs/toolkit';

const counterSlice = createSlice({
  name: 'counter',
  initialState: 0,
  reducers: {
    increment: state => state + 1,
    decrement: state => state - 1,
    incrementByAmount: (state, action) => state + action.payload,
  },
});

export const { increment, decrement, incrementByAmount } = counterSlice.actions;
export default counterSlice.reducer;

Utilització del Fragment en el Store

import { configureStore } from '@reduxjs/toolkit';
import counterReducer from './counterSlice';

const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
});

export default store;

Connectant Redux a React

Configuració del Proveïdor

Per utilitzar Redux en una aplicació React, embolica la teva aplicació amb el component Provider de react-redux.

import React from 'react';
import ReactDOM from 'react-dom';
import { Provider } from 'react-redux';
import store from './store';
import App from './App';

ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
);

Utilització de l'Estat i les Accions de Redux en Components

Utilitza els hooks useSelector i useDispatch de react-redux per interactuar amb el store de Redux en els teus components.

import React from 'react';
import { useSelector, useDispatch } from 'react-redux';
import { increment, decrement, incrementByAmount } from './counterSlice';

const Counter = () => {
  const count = useSelector(state => state.counter);
  const dispatch = useDispatch();

  return (
    <div>
      <div>{count}</div>
      <button onClick={() => dispatch(increment())}>Incrementar</button>
      <button onClick={() => dispatch(decrement())}>Decrementar</button>
      <button onClick={() => dispatch(incrementByAmount(5))}>Incrementar per 5</button>
    </div>
  );
};

export default Counter;

Funcionalitats Avançades de Redux Toolkit

Lògica Asincrònica amb createAsyncThunk

createAsyncThunk simplifica el procés de manejar accions asincròniques.

import { createAsyncThunk, createSlice } from '@reduxjs/toolkit';
import axios from 'axios';

export const fetchUser = createAsyncThunk('user/fetchUser', async (userId) => {
  const response = await axios.get(`/api/user/${userId}`);
  return response.data;
});

const userSlice = createSlice({
  name: 'user',
  initialState: { user: null, status: 'idle' },
  reducers: {},
  extraReducers: (builder) => {
    builder
      .addCase(fetchUser.pending, (state) => {
        state.status = 'loading';
      })
      .addCase(fetchUser.fulfilled, (state, action) => {
        state.status = 'succeeded';
        state.user = action.payload;
      })
      .addCase(fetchUser.rejected, (state) => {
        state.status = 'failed';
      });
  },
});

export default userSlice.reducer;

Middleware

El middleware es pot afegir al store de Redux per manejar efectes secundaris i altra lògica personalitzada.

import { configureStore, getDefaultMiddleware } from '@reduxjs/toolkit';
import logger from 'redux-logger';
import counterReducer from './counterSlice';

const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
  middleware: (getDefaultMiddleware) => getDefaultMiddleware().concat(logger),
});

export default store;

Conclusió

Redux Toolkit simplifica el procés d'escriure la lògica de Redux proporcionant un conjunt d'eines i millors pràctiques. Utilitzant fragments, configureStore i createAsyncThunk, pots gestionar l'estat i manejar accions asincròniques de manera més eficient. Integrar Redux amb React utilitzant els hooks useSelector i useDispatch permet una experiència de gestió d'estat sense problemes. A mesura que avances, pots aprofitar el middleware i altres funcionalitats avançades per construir aplicacions robustes.
