Introducció a la generació de llocs estàtics (SSG)

La generació de llocs estàtics (SSG) és un mètode de pre-renderització de pàgines web en temps de compilació, que resulta en fitxers HTML estàtics. Això pot millorar significativament el rendiment i el SEO.

  • Pre-renderització: Generació d'HTML en temps de compilació.
  • Rendiment: Carregues de pàgina més ràpides ja que l'HTML està pre-generat.
  • SEO: Millor indexació dels motors de cerca gràcies al contingut estàtic.

Per què utilitzar Next.js per a SSG?

Next.js és un framework popular de React que proporciona suport integrat per a SSG, facilitant la creació de llocs web estàtics.

  • Facilitat d'ús: Simplifica la configuració i la configuració.
  • Enrutament integrat: Enrutament automàtic basat en el sistema de fitxers.
  • Obtenció de dades: Suporta diversos mètodes d'obtenció de dades per a SSG.

Configuració d'un projecte Next.js

Per començar amb Next.js, necessites configurar un nou projecte.

  1. Instal·la Next.js: 
    npx create-next-app@latest my-nextjs-site
cd my-nextjs-site
npm run dev
  2. Estructura del projecte:
    • pages/: Conté tots els components de pàgina.
    • public/: Actius estàtics com imatges.
    • styles/: Fitxers CSS.

Creació d'una pàgina estàtica

Next.js et permet crear pàgines estàtiques simplement exportant un component React des d'un fitxer al directori pages/.

  1. Crea una nova pàgina:

    // pages/index.js
import React from 'react';

const HomePage = () => {
  return (
    <div>
      <h1>Benvingut al meu lloc estàtic</h1>
      <p>Aquesta pàgina està generada estàticament.</p>
    </div>
  );
};

export default HomePage;

  2. Executa el servidor de desenvolupament:

    npm run dev

    Obre http://localhost:3000 per veure la teva pàgina estàtica.

Obtenció de dades amb `getStaticProps`

Per obtenir dades en temps de compilació, Next.js proporciona la funció getStaticProps.

  1. Exemple amb getStaticProps: 
    // pages/index.js
import React from 'react';

export async function getStaticProps() {
  // Obteniu dades d'una API o sistema de fitxers
  const data = await fetch('https://api.example.com/data').then(res => res.json());

  return {
    props: {
      data,
    },
  };
}

const HomePage = ({ data }) => {
  return (
    <div>
      <h1>Benvingut al meu lloc estàtic</h1>
      <p>Dades obtingudes en temps de compilació:</p>
      <pre>{JSON.stringify(data, null, 2)}</pre>
    </div>
  );
};

export default HomePage;

Rutes dinàmiques amb `getStaticPaths`

Per a pàgines que requereixen enrutament dinàmic, pots utilitzar getStaticPaths juntament amb getStaticProps.

  1. Exemple amb rutes dinàmiques: 
    // pages/posts/[id].js
import React from 'react';

export async function getStaticPaths() {
  // Obteniu la llista d'IDs d'una API o sistema de fitxers
  const posts = await fetch('https://api.example.com/posts').then(res => res.json());
  const paths = posts.map(post => ({
    params: { id: post.id.toString() },
  }));

  return { paths, fallback: false };
}

export async function getStaticProps({ params }) {
  // Obteniu dades per a una sola publicació
  const post = await fetch(`https://api.example.com/posts/${params.id}`).then(res => res.json());

  return {
    props: {
      post,
    },
  };
}

const PostPage = ({ post }) => {
  return (
    <div>
      <h1>{post.title}</h1>
      <p>{post.content}</p>
    </div>
  );
};

export default PostPage;

Comparació de SSG amb altres mètodes d'obtenció de dades

| Mètode | Descripció | Cas d'ús | |-----------------|----------------------------------------------|-----------------------------------------------| | Generació de llocs estàtics (SSG) | Pre-renderitza pàgines en temps de compilació. | Contingut estàtic que no canvia sovint. | | Renderització del costat del servidor (SSR) | Renderitza pàgines en cada sol·licitud. | Contingut dinàmic que canvia freqüentment. | | Renderització del costat del client (CSR) | Obté dades al costat del client després de la càrrega. | Contingut altament interactiu o específic per a l'usuari. |

Conclusió

La generació de llocs estàtics amb Next.js és una manera poderosa de crear llocs web ràpids i amigables amb el SEO. Aprofitant getStaticProps i getStaticPaths, pots obtenir dades en temps de compilació i generar pàgines estàtiques de manera eficient. Aquest mètode és ideal per a contingut que no canvia freqüentment, proporcionant un gran equilibri entre rendiment i facilitat d'ús.

Dominant SSG en Next.js, pots construir llocs web estàtics robustos i d'alt rendiment que són fàcils de mantenir i escalar.
