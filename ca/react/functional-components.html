Els components funcionals són un dels blocs fonamentals en React. Són més simples i concisos en comparació amb els components de classe i s'utilitzen àmpliament en el desenvolupament modern de React.

Què són els Components Funcionals?

  • Els components funcionals són funcions JavaScript que retornen elements React.
  • No tenen el seu propi estat ni mètodes de cicle de vida (abans dels React Hooks).
  • Són més fàcils de llegir i provar a causa de la seva simplicitat.

Exemple d'un Component Funcional

import React from 'react';

function Greeting() {
  return <h1>Hello, World!</h1>;
}

export default Greeting;

En aquest exemple:

  • Greeting és un component funcional.
  • Retorna un únic element React, una etiqueta <h1> amb el text "Hello, World!".

Props en Components Funcionals

  • Les props (abreviatura de propietats) s'utilitzen per passar dades dels components pare als components fill.
  • Els components funcionals reben props com a argument.

Exemple amb Props

import React from 'react';

function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}

export default Greeting;

Ús:

import React from 'react';
import Greeting from './Greeting';

function App() {
  return <Greeting name="Alice" />;
}

export default App;

En aquest exemple:

  • El component Greeting rep name com a prop.
  • El component App passa el valor "Alice" al component Greeting.

React Hooks en Components Funcionals

Els React Hooks permeten que els components funcionals utilitzin estat i altres característiques de React.

Hook useState

  • useState és un Hook que et permet afegir estat de React als components funcionals.

Exemple amb useState

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}

export default Counter;

En aquest exemple:

  • useState inicialitza la variable d'estat count amb un valor de 0.
  • setCount és una funció que actualitza l'estat.
  • Fer clic al botó incrementa l'estat count.

Hook useEffect

  • useEffect és un Hook que et permet realitzar efectes secundaris en components funcionals.

Exemple amb useEffect

import React, { useEffect, useState } from 'react';

function Timer() {
  const [seconds, setSeconds] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setSeconds(prevSeconds => prevSeconds + 1);
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return <div>Seconds: {seconds}</div>;
}

export default Timer;

En aquest exemple:

  • useEffect configura un temporitzador que incrementa l'estat seconds cada segon.
  • La funció de neteja clearInterval es crida quan el component es desmunta.

Comparació: Components Funcionals vs Components de Classe

| Característica | Components Funcionals | Components de Classe | |------------------------|-----------------------|-------------------------| | Sintaxi | Funció | Classe | | Gestió de l'Estat | Hook useState | this.state | | Mètodes de Cicle de Vida | Hook useEffect | componentDidMount, etc. | | Complexitat del Codi | Més simple | Més complex | | Rendiment | Lleugerament millor | Lleugerament pitjor |

Conclusió

Els components funcionals són una manera poderosa i eficient de construir aplicacions React. Són més simples d'escriure i entendre, especialment amb la introducció dels Hooks, que permeten als components funcionals gestionar l'estat i els efectes secundaris. Dominant els components funcionals, pots escriure codi React més net i mantenible.
