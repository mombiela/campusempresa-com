La gestió d'esdeveniments en React és una habilitat fonamental que permet als desenvolupadors crear aplicacions web interactives i dinàmiques. Aquest tema cobrirà els conceptes bàsics de la gestió d'esdeveniments, tècniques avançades i millors pràctiques per assegurar que puguis gestionar eficaçment les interaccions dels usuaris en les teves aplicacions React.

Conceptes Bàsics de la Gestió d'Esdeveniments

React proporciona una manera de gestionar esdeveniments que és similar a la gestió d'esdeveniments en JavaScript pur, però amb algunes diferències sintàctiques.

Afegir Gestors d'Esdeveniments

En React, afegeixes gestors d'esdeveniments directament als elements utilitzant la sintaxi camelCase. Per exemple, l'esdeveniment onclick en HTML es converteix en onClick en React.

import React from 'react';

function App() {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <button onClick={handleClick}>Click Me</button>
  );
}

export default App;
  • Nomenclatura d'Esdeveniments: Utilitza camelCase per als noms dels esdeveniments.
  • Funció de Gestió d'Esdeveniments: Defineix la funció que gestiona l'esdeveniment.

Passar Arguments als Gestors d'Esdeveniments

De vegades necessites passar arguments al teu gestor d'esdeveniments. Pots aconseguir-ho utilitzant una funció fletxa.

import React from 'react';

function App() {
  const handleClick = (message) => {
    alert(message);
  };

  return (
    <button onClick={() => handleClick('Button clicked!')}>Click Me</button>
  );
}

export default App;
  • Funcions Fletxa: Utilitza funcions fletxa per passar arguments als gestors d'esdeveniments.

Esdeveniments Sintètics

React normalitza els esdeveniments perquè tinguin propietats consistents en diferents navegadors. Aquests esdeveniments normalitzats es diuen Esdeveniments Sintètics.

Propietats dels Esdeveniments

Els esdeveniments sintètics tenen la mateixa interfície que els esdeveniments natius però funcionen de manera idèntica en tots els navegadors.

import React from 'react';

function App() {
  const handleClick = (event) => {
    console.log(event.type); // 'click'
  };

  return (
    <button onClick={handleClick}>Click Me</button>
  );
}

export default App;
  • Tipus d'Esdeveniment: Accedeix al tipus d'esdeveniment utilitzant event.type.
  • Propietats dels Esdeveniments: Els esdeveniments sintètics tenen propietats com els esdeveniments natius (per exemple, event.target, event.preventDefault()).

Gestió d'Esdeveniments de Formularis

Gestionar esdeveniments de formularis com els canvis d'entrada i les submissions de formularis és crucial per crear formularis interactius.

Gestionar Canvis d'Entrada

Per gestionar canvis d'entrada, pots utilitzar l'esdeveniment onChange.

import React, { useState } from 'react';

function App() {
  const [value, setValue] = useState('');

  const handleChange = (event) => {
    setValue(event.target.value);
  };

  return (
    <input type="text" value={value} onChange={handleChange} />
  );
}

export default App;
  • Gestió d'Estat: Utilitza l'estat per gestionar el valor de l'entrada.
  • Esdeveniment onChange: Actualitza l'estat quan el valor de l'entrada canvia.

Gestionar la Submissió de Formularis

Per gestionar la submissió de formularis, utilitza l'esdeveniment onSubmit i evita el comportament per defecte de la submissió de formularis.

import React, { useState } from 'react';

function App() {
  const [value, setValue] = useState('');

  const handleSubmit = (event) => {
    event.preventDefault();
    alert(`Form submitted with value: ${value}`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" value={value} onChange={(e) => setValue(e.target.value)} />
      <button type="submit">Submit</button>
    </form>
  );
}

export default App;
  • Evitar el Comportament per Defecte: Utilitza event.preventDefault() per evitar la submissió per defecte del formulari.
  • Submissió de Formularis: Gestiona les dades del formulari en l'esdeveniment onSubmit.

Gestió Avançada d'Esdeveniments

Delegació d'Esdeveniments

La delegació d'esdeveniments implica adjuntar un únic gestor d'esdeveniments a un element pare per gestionar esdeveniments per a múltiples elements fills.

import React from 'react';

function App() {
  const handleClick = (event) => {
    if (event.target.tagName === 'BUTTON') {
      alert(`Button ${event.target.textContent} clicked!`);
    }
  };

  return (
    <div onClick={handleClick}>
      <button>Button 1</button>
      <button>Button 2</button>
      <button>Button 3</button>
    </div>
  );
}

export default App;
  • Delegació d'Esdeveniments: Adjunta un gestor d'esdeveniments a un element pare.
  • Objectiu de l'Esdeveniment: Utilitza event.target per determinar quin element fill va desencadenar l'esdeveniment.

Esdeveniments Personalitzats

Pots crear esdeveniments personalitzats per gestionar interaccions més complexes.

import React, { useState } from 'react';

function App() {
  const [count, setCount] = useState(0);

  const handleCustomEvent = () => {
    const event = new CustomEvent('increment', { detail: { count: count + 1 } });
    window.dispatchEvent(event);
  };

  window.addEventListener('increment', (event) => {
    setCount(event.detail.count);
  });

  return (
    <div>
      <button onClick={handleCustomEvent}>Increment</button>
      <p>Count: {count}</p>
    </div>
  );
}

export default App;
  • CustomEvent: Crea i envia esdeveniments personalitzats.
  • Gestors d'Esdeveniments: Afegeix gestors d'esdeveniments per a esdeveniments personalitzats.

Resum

Gestionar esdeveniments en React és essencial per crear aplicacions interactives. Des de la gestió bàsica d'esdeveniments fins a tècniques avançades com la delegació d'esdeveniments i els esdeveniments personalitzats, entendre com gestionar les interaccions dels usuaris millorarà significativament les teves habilitats de desenvolupament en React. Recorda utilitzar la gestió d'estat de manera efectiva, aprofitar els Esdeveniments Sintètics per a la compatibilitat entre navegadors i explorar patrons avançats segons sigui necessari.
