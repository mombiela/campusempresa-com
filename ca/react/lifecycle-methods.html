Introducció als mètodes de cicle de vida

Els mètodes de cicle de vida a React són mètodes especials que es criden en diferents etapes de la vida d'un component. Et permeten connectar-te a moments específics del cicle de vida del component, com quan es munta, s'actualitza o es desmunta. Entendre aquests mètodes és crucial per gestionar efectes secundaris, optimitzar el rendiment i gestionar l'estat del component.

Mètodes de cicle de vida dels components de classe

Els components de classe de React tenen diversos mètodes de cicle de vida que pots sobreescriure per executar codi en moments específics del cicle de vida del component.

Muntatge

El muntatge és la fase en què un component s'està inserint al DOM.

  • constructor(): Es crida abans que el component es munti. S'utilitza per inicialitzar l'estat i vincular els manejadors d'esdeveniments.
  • componentDidMount(): Es crida immediatament després que el component s'hagi muntat. És un bon lloc per fer crides a API o configurar subscripcions.
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
    this.state = { data: null };
  }

  componentDidMount() {
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(data => this.setState({ data }));
  }

  render() {
    return <div>{this.state.data ? this.state.data : 'Loading...'}</div>;
  }
}

Actualització

L'actualització és la fase en què un component s'està tornant a renderitzar a causa de canvis en les propietats o l'estat.

  • componentDidUpdate(prevProps, prevState): Es crida immediatament després que es produeixi l'actualització. És útil per realitzar operacions DOM o fer sol·licituds de xarxa basades en l'estat o les propietats anteriors.
class MyComponent extends React.Component {
  componentDidUpdate(prevProps, prevState) {
    if (this.props.id !== prevProps.id) {
      this.fetchData(this.props.id);
    }
  }

  fetchData(id) {
    // Obtenir noves dades basades en l'id
  }

  render() {
    return <div>{this.props.id}</div>;
  }
}

Desmuntatge

El desmuntatge és la fase en què un component s'està eliminant del DOM.

  • componentWillUnmount(): Es crida immediatament abans que un component es desmunti i es destrueixi. S'utilitza per netejar subscripcions, temporitzadors o qualsevol altre recurs.
class MyComponent extends React.Component {
  componentDidMount() {
    this.timerID = setInterval(() => this.tick(), 1000);
  }

  componentWillUnmount() {
    clearInterval(this.timerID);
  }

  tick() {
    // Actualitzar l'estat o realitzar alguna acció
  }

  render() {
    return <div>Timer</div>;
  }
}

Components funcionals i Hooks

Amb la introducció dels Hooks a React 16.8, pots utilitzar l'estat i altres característiques de React sense escriure una classe. El Hook useEffect serveix per a una finalitat similar als mètodes de cicle de vida en components de classe.

Hook useEffect

El Hook useEffect et permet realitzar efectes secundaris en components funcionals. Combina les funcionalitats de componentDidMount, componentDidUpdate i componentWillUnmount.

import React, { useState, useEffect } from 'react';

function MyComponent({ id }) {
  const [data, setData] = useState(null);

  useEffect(() => {
    fetch(`https://api.example.com/data/${id}`)
      .then(response => response.json())
      .then(data => setData(data));

    return () => {
      // Codi de neteja, similar a componentWillUnmount
    };
  }, [id]); // Matriu de dependències

  return <div>{data ? data : 'Loading...'}</div>;
}

Matriu de dependències

El segon argument de useEffect és la matriu de dependències. Determina quan hauria de córrer l'efecte.

  • Si la matriu està buida ([]), l'efecte només s'executa una vegada, similar a componentDidMount.
  • Si la matriu conté variables, l'efecte s'executa sempre que alguna d'aquestes variables canviï, similar a componentDidUpdate.
  • La funció de neteja dins de useEffect s'executa abans que el component es desmunti, similar a componentWillUnmount.

Resum

Els mètodes de cicle de vida a React són essencials per gestionar efectes secundaris, optimitzar el rendiment i gestionar l'estat del component. Els components de classe tenen mètodes de cicle de vida distintius com componentDidMount, componentDidUpdate i componentWillUnmount. Amb l'aparició dels Hooks, els components funcionals poden aconseguir la mateixa funcionalitat utilitzant el Hook useEffect. Entendre i utilitzar efectivament aquests mètodes o hooks és crucial per construir aplicacions React robustes i eficients.
