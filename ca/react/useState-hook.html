Introducció al hook useState

El hook useState és un dels hooks fonamentals en React que et permet afegir estat als components funcionals. Abans dels hooks, la gestió de l'estat només era possible en components de classe. Amb useState, ara pots gestionar l'estat en components funcionals, fent-los més potents i versàtils.

Ús bàsic de useState

Per utilitzar el hook useState, necessites importar-lo de la biblioteca de React. El hook useState retorna una matriu amb dos elements: l'estat actual i una funció per actualitzar aquest estat.

Exemple

import React, { useState } from 'react';

function Counter() {
  // Declara una variable d'estat anomenada 'count' i una funció per actualitzar-la
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Has fet clic {count} vegades</p>
      <button onClick={() => setCount(count + 1)}>
        Fes clic aquí
      </button>
    </div>
  );
}

export default Counter;

Explicació

  • useState(0): Inicialitza la variable d'estat count amb un valor de 0.
  • const [count, setCount]: Desestructura la matriu retornada per useState en count (la variable d'estat) i setCount (la funció per actualitzar count).
  • setCount(count + 1): Actualitza la variable d'estat count incrementant el seu valor en 1.

Gestió de múltiples variables d'estat

Pots utilitzar múltiples hooks useState per gestionar diferents variables d'estat dins del mateix component.

Exemple

import React, { useState } from 'react';

function UserProfile() {
  const [name, setName] = useState('');
  const [age, setAge] = useState(0);

  return (
    <div>
      <input
        type="text"
        placeholder="Introdueix el teu nom"
        value={name}
        onChange={(e) => setName(e.target.value)}
      />
      <input
        type="number"
        placeholder="Introdueix la teva edat"
        value={age}
        onChange={(e) => setAge(Number(e.target.value))}
      />
      <p>Nom: {name}</p>
      <p>Edat: {age}</p>
    </div>
  );
}

export default UserProfile;

Explicació

  • const [name, setName] = useState(''): Inicialitza la variable d'estat name amb una cadena buida.
  • const [age, setAge] = useState(0): Inicialitza la variable d'estat age amb un valor de 0.
  • setName(e.target.value): Actualitza la variable d'estat name amb el valor del camp d'entrada.
  • setAge(Number(e.target.value)): Actualitza la variable d'estat age amb el valor numèric del camp d'entrada.

Actualitzacions funcionals

Quan actualitzes l'estat basant-te en l'estat anterior, es recomana utilitzar una actualització funcional per assegurar-te que tens el valor d'estat més recent.

Exemple

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Has fet clic {count} vegades</p>
      <button onClick={() => setCount(prevCount => prevCount + 1)}>
        Fes clic aquí
      </button>
    </div>
  );
}

export default Counter;

Explicació

  • setCount(prevCount => prevCount + 1): Utilitza una actualització funcional per incrementar la variable d'estat count en 1, assegurant-se que l'actualització es basa en el valor d'estat més recent.

Inicialització mandrosa

Pots inicialitzar l'estat de manera mandrosa passant una funció a useState. Aquesta funció només s'executarà durant el renderitzat inicial.

Exemple

import React, { useState } from 'react';

function ExpensiveComponent() {
  const [value, setValue] = useState(() => {
    // Simula un càlcul costós
    const initialValue = computeExpensiveValue();
    return initialValue;
  });

  return (
    <div>
      <p>Valor: {value}</p>
      <button onClick={() => setValue(value + 1)}>
        Incrementa
      </button>
    </div>
  );
}

function computeExpensiveValue() {
  // Simula un càlcul costós
  return 42;
}

export default ExpensiveComponent;

Explicació

  • useState(() => computeExpensiveValue()): Inicialitza la variable d'estat value cridant computeExpensiveValue només durant el renderitzat inicial, evitant càlculs innecessaris en renderitzats posteriors.

Conclusió

El hook useState és una eina potent per gestionar l'estat en components funcionals. Proporciona una API senzilla i intuïtiva per inicialitzar, actualitzar i gestionar l'estat. Comprenent el seu ús bàsic, la gestió de múltiples variables d'estat, les actualitzacions funcionals i la inicialització mandrosa, pots gestionar eficaçment l'estat en les teves aplicacions React.
