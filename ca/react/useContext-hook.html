Introducció al Hook useContext

El Hook useContext és una característica potent de React que et permet compartir estat entre múltiples components sense haver de passar les props manualment a cada nivell. Això pot simplificar molt el teu codi i fer-lo més mantenible.

Conceptes Clau

  • Context: Una manera de passar dades a través de l'arbre de components sense haver de passar les props manualment a cada nivell.
  • Provider: Un component que proporciona el valor del context als seus fills.
  • Consumer: Un component que consumeix el valor del context.

Crear i Utilitzar Context

Pas 1: Crear un Context

Primer, necessites crear un context utilitzant la funció createContext.

import React, { createContext } from 'react';

// Crear un Context
const MyContext = createContext();

Pas 2: Proporcionar el Context

Embolica el teu arbre de components amb el component Provider i passa el valor del context.

import React, { createContext } from 'react';

// Crear un Context
const MyContext = createContext();

const App = () => {
  return (
    <MyContext.Provider value="Hola, Món!">
      <ChildComponent />
    </MyContext.Provider>
  );
};

Pas 3: Consumir el Context

Utilitza el Hook useContext per consumir el valor del context en un component fill.

import React, { useContext } from 'react';
import { MyContext } from './App'; // Ajusta el camí d'importació segons sigui necessari

const ChildComponent = () => {
  const value = useContext(MyContext);
  return <div>{value}</div>;
};

Exemple: Context de Tema

Crear un Context de Tema

Creem un context de tema per gestionar els temes clar i fosc.

import React, { createContext, useState } from 'react';

// Crear un Context de Tema
const ThemeContext = createContext();

const ThemeProvider = ({ children }) => {
  const [theme, setTheme] = useState('light');

  const toggleTheme = () => {
    setTheme((prevTheme) => (prevTheme === 'light' ? 'dark' : 'light'));
  };

  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
};

export { ThemeContext, ThemeProvider };

Utilitzar el Context de Tema

Ara, utilitzem el ThemeContext en un component per alternar entre els temes clar i fosc.

import React, { useContext } from 'react';
import { ThemeContext } from './ThemeProvider'; // Ajusta el camí d'importació segons sigui necessari

const ThemedComponent = () => {
  const { theme, toggleTheme } = useContext(ThemeContext);

  return (
    <div style={{ background: theme === 'light' ? '#fff' : '#333', color: theme === 'light' ? '#000' : '#fff' }}>
      <p>El tema actual és {theme}</p>
      <button onClick={toggleTheme}>Alternar Tema</button>
    </div>
  );
};

export default ThemedComponent;

Integrar ThemeProvider en App

Finalment, embolica la teva aplicació amb el ThemeProvider.

import React from 'react';
import { ThemeProvider } from './ThemeProvider'; // Ajusta el camí d'importació segons sigui necessari
import ThemedComponent from './ThemedComponent'; // Ajusta el camí d'importació segons sigui necessari

const App = () => {
  return (
    <ThemeProvider>
      <ThemedComponent />
    </ThemeProvider>
  );
};

export default App;

Conclusió

El Hook useContext és una eina potent per gestionar l'estat entre múltiples components sense la necessitat de passar props. Creant un context, proporcionant-lo a un nivell superior i consumint-lo en components fills, pots simplificar la teva gestió d'estat i fer el teu codi més mantenible. Practica utilitzant useContext amb diferents exemples per obtenir una millor comprensió de les seves capacitats i com es pot aplicar en aplicacions del món real.
