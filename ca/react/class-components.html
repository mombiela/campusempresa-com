Introducció als Components de Classe

Els components de classe són un dels blocs fonamentals en React, especialment en bases de codi més antigues. Permeten crear components amb una lògica i una gestió d'estat més complexes en comparació amb els components funcionals. Tot i que els components funcionals amb hooks són més comuns en el desenvolupament modern de React, entendre els components de classe és crucial per mantenir i comprendre el codi llegat.

Creació d'un Component de Classe

Per crear un component de classe en React, cal estendre la classe React.Component i definir un mètode render que retorni el JSX a renderitzar.

Exemple

import React, { Component } from 'react';

class Welcome extends Component {
  render() {
    return <h1>Hello, {this.props.name}</h1>;
  }
}

export default Welcome;

Explicació

  • import React, { Component } from 'react';: Importa React i la classe Component de la llibreria React.
  • class Welcome extends Component { ... }: Defineix una nova classe Welcome que estèn React.Component.
  • render() { ... }: Cada component de classe ha de tenir un mètode render que retorni JSX.
  • this.props.name: Accedeix a les props passades al component utilitzant this.props.

Estat en Components de Classe

Els components de classe poden tenir i gestionar el seu propi estat utilitzant l'objecte this.state.

Exemple

import React, { Component } from 'react';

class Counter extends Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  increment = () => {
    this.setState({ count: this.state.count + 1 });
  };

  render() {
    return (
      <div>
        <p>Count: {this.state.count}</p>
        <button onClick={this.increment}>Increment</button>
      </div>
    );
  }
}

export default Counter;

Explicació

  • constructor(props) { ... }: Inicialitza l'estat del component en el constructor.
  • this.state = { count: 0 };: Estableix l'estat inicial.
  • this.setState({ count: this.state.count + 1 });: Actualitza l'estat utilitzant setState.
  • onClick={this.increment}: Associa el mètode increment a l'esdeveniment onClick del botó.

Mètodes de Cicle de Vida

Els components de classe tenen diversos mètodes de cicle de vida que permeten executar codi en punts específics del cicle de vida d'un component.

Mètodes de Cicle de Vida Comuns

  • componentDidMount(): Es crida després que el component es monta.
  • componentDidUpdate(prevProps, prevState): Es crida després que el component s'actualitza.
  • componentWillUnmount(): Es crida abans que el component es desmunti i es destrueixi.

Exemple

import React, { Component } from 'react';

class Timer extends Component {
  constructor(props) {
    super(props);
    this.state = { seconds: 0 };
  }

  componentDidMount() {
    this.interval = setInterval(() => this.tick(), 1000);
  }

  componentDidUpdate(prevProps, prevState) {
    if (this.state.seconds !== prevState.seconds) {
      console.log(`Seconds updated: ${this.state.seconds}`);
    }
  }

  componentWillUnmount() {
    clearInterval(this.interval);
  }

  tick() {
    this.setState(state => ({
      seconds: state.seconds + 1
    }));
  }

  render() {
    return (
      <div>
        <p>Seconds: {this.state.seconds}</p>
      </div>
    );
  }
}

export default Timer;

Explicació

  • componentDidMount(): Comença un temporitzador quan el component es monta.
  • componentDidUpdate(prevProps, prevState): Registra un missatge sempre que l'estat seconds s'actualitza.
  • componentWillUnmount(): Esborra el temporitzador quan el component es desmunta.
  • tick(): Incrementa l'estat seconds cada segon.

Props vs Estat

Entendre la diferència entre props i estat és crucial en React.

| Característica | Props | Estat | |---------|-------|-------| | Definició | Dades passades al component | Dades gestionades dins del component | | Mutabilitat | Immutable | Mutable | | Ús | Només lectura | Es pot actualitzar amb setState | | Origen | Component pare | Dins del component mateix |

Exemple

import React, { Component } from 'react';

class Display extends Component {
  render() {
    return <h1>{this.props.message}</h1>;
  }
}

class App extends Component {
  constructor(props) {
    super(props);
    this.state = { message: 'Hello, World!' };
  }

  changeMessage = () => {
    this.setState({ message: 'Hello, React!' });
  };

  render() {
    return (
      <div>
        <Display message={this.state.message} />
        <button onClick={this.changeMessage}>Change Message</button>
      </div>
    );
  }
}

export default App;

Explicació

  • El component Display rep message com una prop i la mostra.
  • El component App gestiona l'estat message i el passa a Display com una prop.
  • El mètode changeMessage actualitza l'estat message quan es fa clic al botó.

Conclusió

Els components de classe són una característica potent en React, que permeten una gestió complexa de l'estat i mètodes de cicle de vida. Tot i que els components funcionals amb hooks són més prevalents en el desenvolupament modern de React, entendre els components de classe és essencial per treballar amb codi llegat i obtenir una comprensió més profunda de les capacitats de React. En dominar els components de classe, estaràs ben preparat per gestionar una àmplia gamma d'escenaris en el desenvolupament de React.
