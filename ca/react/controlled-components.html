Components No Controlats ►

Introducció

Els components controlats són un concepte fonamental en React que es refereix a components on React controla l'estat dels elements del formulari. Això significa que les dades del formulari són gestionades pel component de React, cosa que facilita la seva gestió i manipulació.

Conceptes Clau

  • Gestió de l'Estat: En els components controlats, l'estat dels elements del formulari és gestionat pel component de React.
  • Gestió d'Esdeveniments: Els components controlats depenen dels gestors d'esdeveniments per actualitzar l'estat.
  • Font Única de Veritat: L'estat dins del component de React serveix com a font única de veritat per a les dades del formulari.

Exemple d'un Component Controlat

Comencem amb un exemple senzill d'un camp d'entrada controlat:

import React, { useState } from 'react';

function ControlledInput() {
  const [inputValue, setInputValue] = useState('');

  const handleChange = (event) => {
    setInputValue(event.target.value);
  };

  return (
    <div>
      <input type="text" value={inputValue} onChange={handleChange} />
      <p>Current Input: {inputValue}</p>
    </div>
  );
}

export default ControlledInput;

Explicació

  • Hook useState: Utilitzem el hook useState per crear una variable d'estat inputValue i una funció setInputValue per actualitzar-la.
  • Funció handleChange: Aquesta funció actualitza l'estat cada vegada que el valor de l'entrada canvia.
  • Element d'Entrada: L'atribut value de l'element d'entrada està establert a inputValue, fent-lo un component controlat. El gestor d'esdeveniments onChange crida a handleChange per actualitzar l'estat.

Components Controlats vs No Controlats

| Característica | Components Controlats | Components No Controlats | |--------------------------|------------------------------------------------|------------------------------------------------| | Gestió de l'Estat | Gestionat per l'estat del component de React | Gestionat pel DOM | | Gestió d'Esdeveniments | Requereix gestors d'esdeveniments per actualitzar l'estat | Utilitza refs per accedir directament als elements del DOM | | Font Única de Veritat | Estat dins del component | L'estat és gestionat pel DOM | | Facilitat de Proves | Més fàcil de provar degut a l'estat previsible | Més difícil de provar degut a la dependència del DOM |

Exemple Avançat: Formulari Controlat

Creem un exemple més complex amb un formulari controlat que conté múltiples camps d'entrada:

import React, { useState } from 'react';

function ControlledForm() {
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    message: ''
  });

  const handleChange = (event) => {
    const { name, value } = event.target;
    setFormData({
      ...formData,
      [name]: value
    });
  };

  const handleSubmit = (event) => {
    event.preventDefault();
    console.log('Form Data Submitted:', formData);
  };

  return (
    <form onSubmit={handleSubmit}>
      <div>
        <label>Name:</label>
        <input type="text" name="name" value={formData.name} onChange={handleChange} />
      </div>
      <div>
        <label>Email:</label>
        <input type="email" name="email" value={formData.email} onChange={handleChange} />
      </div>
      <div>
        <label>Message:</label>
        <textarea name="message" value={formData.message} onChange={handleChange}></textarea>
      </div>
      <button type="submit">Submit</button>
    </form>
  );
}

export default ControlledForm;

Explicació

  • Inicialització de l'Estat: L'estat formData s'inicialitza amb un objecte que conté els camps name, email i message.
  • Funció handleChange: Aquesta funció actualitza el camp corresponent en l'estat formData basant-se en l'atribut name de l'entrada.
  • Funció handleSubmit: Aquesta funció prevé el comportament per defecte de l'enviament del formulari i registra les dades del formulari a la consola.
  • Elements del Formulari: L'atribut value de cada element del formulari està vinculat al camp corresponent en l'estat formData, i el gestor d'esdeveniments onChange actualitza l'estat.

Conclusió

Els components controlats són essencials per gestionar les dades dels formularis en aplicacions React. Controlant l'estat dins del component de React, obtens més control sobre el comportament del formulari i pots manipular i validar fàcilment les dades. Entendre els components controlats és crucial per construir aplicacions React robustes i mantenibles.
