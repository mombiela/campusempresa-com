Introducció

React és una biblioteca popular de JavaScript per construir interfícies d'usuari, particularment aplicacions d'una sola pàgina. Permet als desenvolupadors crear components d'UI reutilitzables. En aquesta secció, cobrirem els conceptes bàsics per crear una aplicació de "Hola Món" en React.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Coneixements bàsics de JavaScript i HTML.
  • Node.js i npm instal·lats a la teva màquina.

Configurant l'Entorn

Per crear una aplicació React, utilitzarem Create React App, una eina que configura un nou projecte React amb una configuració predeterminada raonable.

  1. Instal·lar Create React App: Obre el teu terminal i executa:

    npx create-react-app hello-world

    Aquesta comanda crearà un nou directori anomenat hello-world amb tots els fitxers i dependències necessaris.

  2. Navegar al Directori del Projecte:

    cd hello-world

  3. Iniciar el Servidor de Desenvolupament:

    npm start

    Això iniciarà el servidor de desenvolupament i obrirà la teva nova aplicació React al navegador.

Entenent l'Estructura del Projecte

Quan crees una nova aplicació React, l'estructura del projecte es veu així:

hello-world/
├── node_modules/
├── public/
│   ├── index.html
│   └── ...
├── src/
│   ├── App.css
│   ├── App.js
│   ├── App.test.js
│   ├── index.css
│   ├── index.js
│   └── ...
├── .gitignore
├── package.json
├── README.md
└── ...
  • public/index.html: L'únic fitxer HTML que serveix la teva aplicació React.
  • src/index.js: El punt d'entrada de JavaScript de la teva aplicació.
  • src/App.js: Un component React de mostra.

Creant el Component "Hola Món"

Modifiquem el component React predeterminat per mostrar "Hola Món".

  1. Obrir src/App.js:

    import React from 'react';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>Hola Món</h1>
      </header>
    </div>
  );
}

export default App;

  2. Explicació:

    • Importar React: La instrucció import React from 'react'; és necessària per utilitzar JSX, una extensió de sintaxi per JavaScript que sembla HTML.
    • Component Funcional: App és un component funcional que retorna JSX. JSX ens permet escriure codi semblant a HTML dins de JavaScript.
    • JSX: La instrucció return conté JSX que renderitza un div amb un header i un element h1 que mostra "Hola Món".

Executant l'Aplicació

Després de desar els canvis, el teu servidor de desenvolupament hauria de recarregar automàticament, i hauries de veure "Hola Món" mostrat al teu navegador.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per configurar un entorn React i crear una simple aplicació de "Hola Món". Aquest coneixement fonamental és crucial a mesura que avancem cap a temes més complexos en React. Entendre com configurar i modificar components serà essencial per construir aplicacions més avançades.
