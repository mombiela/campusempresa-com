Configurar l'entorn de Go és el primer pas per començar a programar en Go. Aquesta secció et guiarà a través del procés d'instal·lació de Go, configuració del teu espai de treball i verificació de la teva instal·lació.

Instal·lant Go

Per començar amb Go, necessites instal·lar el compilador i les eines de Go a la teva màquina. Segueix aquests passos per instal·lar Go:

  1. Descarregar Go:

    • Visita el lloc web oficial de Go: golang.org
    • Descarrega l'instal·lador per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·lar Go:

    • Windows:
      • Executa l'arxiu MSI descarregat i segueix les instruccions.
    • macOS:
      • Obre l'arxiu .pkg descarregat i segueix les instruccions d'instal·lació.
    • Linux:
      • Extreu el tarball a /usr/local utilitzant la següent comanda: 
        tar -C /usr/local -xzf go1.x.x.linux-amd64.tar.gz

  3. Configurar Variables d'Entorn:

    • Afegeix el directori binari de Go a la teva variable d'entorn PATH.
    • Windows:
      • Obre la Cerca d'Inici, escriu "env" i selecciona "Edita les variables d'entorn del sistema".
      • Fes clic a "Variables d'Entorn" i troba la variable PATH a la secció "Variables del sistema".
      • Fes clic a "Edita" i afegeix el camí al binari de Go (per exemple, C:\Go\bin).
    • macOS i Linux:
      • Afegeix la següent línia al teu arxiu .bashrc, .zshrc o .profile: 
        export PATH=$PATH:/usr/local/go/bin

Configuració del Teu Espai de Treball

Go utilitza una estructura específica d'espai de treball per gestionar el codi. L'espai de treball és una jerarquia de directoris amb tres directoris principals: src, pkg i bin.

  1. Crear el Teu Espai de Treball:

    • Tria una ubicació per al teu espai de treball, per exemple, ~/go.
    • Dins d'aquest directori, crea els següents subdirectoris: 
      mkdir -p ~/go/{src,pkg,bin}

  2. Configurar la Variable d'Entorn GOPATH:

    • La variable d'entorn GOPATH especifica la ubicació del teu espai de treball.
    • Windows:
      • Obre la Cerca d'Inici, escriu "env" i selecciona "Edita les variables d'entorn del sistema".
      • Fes clic a "Variables d'Entorn" i crea una nova variable d'usuari anomenada GOPATH amb el valor del camí del teu espai de treball (per exemple, C:\Users\YourName\go).
    • macOS i Linux:
      • Afegeix la següent línia al teu arxiu .bashrc, .zshrc o .profile: 
        export GOPATH=~/go

Verificant la Teva Instal·lació

Per assegurar-te que Go està instal·lat correctament i que el teu entorn està configurat adequadament, segueix aquests passos:

  1. Comprovar la Versió de Go:

    • Obre una terminal o símbol del sistema.
    • Executa la següent comanda: 
      go version
    • Hauries de veure una sortida similar a go version go1.x.x <os>/<arch>.

  2. Crear un Programa Simple en Go:

    • Crea un directori per al teu projecte dins del directori src del teu espai de treball: 
      mkdir -p ~/go/src/hello
    • Crea un arxiu anomenat hello.go dins d'aquest directori amb el següent contingut: 
      package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

  3. Compilar i Executar el Programa:

    • Navega al directori que conté el teu arxiu Go: 
      cd ~/go/src/hello
    • Compila el programa utilitzant la comanda go build: 
      go build
    • Executa l'executable: 
      ./hello
    • Hauries de veure la sortida Hello, World!.

Conclusió

Configurar l'entorn de Go implica instal·lar Go, configurar el teu espai de treball i verificar la instal·lació. Seguint aquests passos, assegures que el teu entorn de desenvolupament està llest per a programar en Go. Aquesta configuració fonamental et permetrà progressar sense problemes en l'aprenentatge i desenvolupament d'aplicacions en Go.
