El teu primer programa en Go
Sintaxi i Estructura de Go ►

Benvingut al teu viatge d'aprenentatge de Go! En aquesta secció, et guiarem a través de l'escriptura del teu primer programa en Go. Al final d'aquesta secció, tindràs una comprensió bàsica de la sintaxi i l'estructura de Go.

Configurant el teu entorn

Abans de començar a codificar, assegurem-nos que tens tot configurat:

  1. Instal·la Go: Descarrega i instal·la Go des del lloc web oficial.
  2. Configura el teu espai de treball: Crea un directori per als teus projectes en Go. Per exemple, ~/go-projects.
  3. Verifica la instal·lació: Obre el teu terminal i executa go version per comprovar si Go està instal·lat correctament.

Escriure el teu primer programa

Escrivim un simple programa de "Hola, Món!" en Go.

Creant el fitxer del programa

  1. Crea un nou fitxer: Obre el teu editor de text o IDE i crea un nou fitxer anomenat main.go.
  2. Escriu el codi: Introdueix el següent codi a main.go:
package main

import "fmt"

func main() {
    fmt.Println("Hola, Món!")
}

Explicació del codi

  • package main: Cada programa en Go comença amb una declaració de package. El paquet main és un paquet especial que indica al compilador de Go que aquest és un programa executable.
  • import "fmt": La declaració import s'utilitza per incloure el paquet fmt, que conté funcions per formatar text, incloent imprimir a la consola.
  • func main(): La funció main és el punt d'entrada del programa. Quan executes el programa, el codi dins de main s'executa.
  • fmt.Println("Hola, Món!"): Aquesta línia crida la funció Println del paquet fmt per imprimir "Hola, Món!" a la consola.

Executant el teu programa

Per executar el teu programa, segueix aquests passos:

  1. Obre el teu terminal.
  2. Navega al teu directori de projectes: Utilitza cd ~/go-projects (o el directori que has creat).
  3. Executa el programa: Executa go run main.go.

Hauries de veure la sortida:

Hola, Món!

Conceptes clau

  • Paquets: Els programes en Go estan organitzats en paquets. El paquet main s'utilitza per a programes executables.
  • Imports: Utilitza la declaració import per incloure biblioteques estàndard o altres paquets.
  • Funcions: Les funcions es defineixen utilitzant la paraula clau func. La funció main és el punt de partida del programa.
  • Impressió: El paquet fmt proporciona funcions com Println per imprimir sortida a la consola.

Resum

Felicitats! Has escrit i executat el teu primer programa en Go. Aquí tens un breu resum del que hem cobert:

  • Configurar el teu entorn de Go.
  • Escriure un simple programa de "Hola, Món!".
  • Comprendre l'estructura bàsica d'un programa en Go.
  • Executar el teu programa en Go des del terminal.

En la següent secció, aprofundirem en la sintaxi de Go i explorarem programes més complexos. Continua practicant i experimentant amb el codi per consolidar la teva comprensió. Feliç codificació!
