L'instrucció select en Go és una estructura de control potent utilitzada per gestionar múltiples operacions de canal. Permet que una goroutine esperi en múltiples operacions de comunicació, fent-la essencial per a la programació concurrent.

Conceptes Clau

  • Canals: Els canals són la manera de Go de comunicar-se entre goroutines. Es poden considerar com a tubs a través dels quals es poden enviar i rebre dades.
  • Bloqueig: Les operacions en canals són de bloqueig per defecte. Una operació d'enviament bloquejarà fins que una altra goroutine estigui preparada per rebre, i viceversa.
  • Instrucció Select: L'instrucció select permet que una goroutine esperi en múltiples operacions de canal. Bloqueja fins que un dels seus casos pugui procedir, i després executa aquest cas.

Sintaxi Bàsica

L'instrucció select s'assembla a una instrucció switch però opera en canals:

select {
case <-ch1:
    // codi a executar quan ch1 rep un valor
case ch2 <- value:
    // codi a executar quan el valor és enviat a ch2
default:
    // codi a executar si cap dels canals està preparat
}

Exemples

Exemple 1: Instrucció Select Bàsica

Aquest exemple demostra un ús bàsic de l'instrucció select amb dos canals.

package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string)
    ch2 := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        ch1 <- "Missatge de ch1"
    }()

    go func() {
        time.Sleep(2 * time.Second)
        ch2 <- "Missatge de ch2"
    }()

    select {
    case msg1 := <-ch1:
        fmt.Println(msg1)
    case msg2 := <-ch2:
        fmt.Println(msg2)
    }
}

En aquest exemple:

  • Es creen dos canals ch1 i ch2.
  • Es llancen dues goroutines, cadascuna enviant un missatge a un dels canals després d'un retard.
  • L'instrucció select espera que un dels canals rebi un missatge i després el mostra.

Exemple 2: Ús del Cas Default

El cas default en una instrucció select s'executa si cap dels canals està preparat.

package main

import (
    "fmt"
    "time"
)

func main() {
    ch := make(chan string)

    go func() {
        time.Sleep(2 * time.Second)
        ch <- "Missatge de ch"
    }()

    select {
    case msg := <-ch:
        fmt.Println(msg)
    default:
        fmt.Println("No s'ha rebut cap missatge")
    }
}

En aquest exemple:

  • Es crea un únic canal ch.
  • Una goroutine envia un missatge al canal després d'un retard de 2 segons.
  • L'instrucció select inclou un cas default que mostra un missatge si cap canal està preparat. Com que la goroutine triga 2 segons a enviar el missatge, el cas default s'executarà immediatament.

Conclusió

L'instrucció select és una eina fonamental en Go per gestionar múltiples operacions de canal. Et permet escriure programes concurrents més eficients i responsius permetent que una goroutine esperi en múltiples canals. Entendre com utilitzar select de manera efectiva t'ajudarà a gestionar patrons de comunicació complexos en les teves aplicacions Go.
