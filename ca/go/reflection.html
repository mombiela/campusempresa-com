Reflexió en Go
Proves en Go ►

La reflexió és una característica poderosa en Go que permet a un programa inspeccionar i manipular la seva pròpia estructura i comportament en temps d'execució. Aquest tema cobrirà els conceptes bàsics de la reflexió, com utilitzar-la de manera efectiva i algunes tècniques avançades.

Què és la Reflexió?

La reflexió en Go és la capacitat d'inspeccionar el tipus i el valor de les variables en temps d'execució. Forma part del paquet reflect de la biblioteca estàndard.

  • Inspecció de Tipus: Determinar el tipus d'una variable.
  • Inspecció de Valor: Accedir al valor d'una variable.
  • Manipulació: Modificar el valor d'una variable.

Per què Utilitzar la Reflexió?

La reflexió pot ser útil en diverses situacions, com ara:

  • Gestió Dinàmica de Tipus: Treballar amb interfícies i tipus desconeguts.
  • Serialització/Deserialització: Convertir estructures de dades a i des de formats com JSON o XML.
  • Proves: Crear funcions de prova genèriques que puguin gestionar diferents tipus.

Conceptes Bàsics

El Paquet `reflect`

El paquet reflect proporciona les eines necessàries per a la reflexió. Els tipus i funcions més importants són:

  • reflect.Type: Representa el tipus d'una variable.
  • reflect.Value: Representa el valor d'una variable.
  • reflect.TypeOf(): Retorna el reflect.Type d'una variable.
  • reflect.ValueOf(): Retorna el reflect.Value d'una variable.

Exemple: Inspeccionant Tipus i Valors

Comencem amb un exemple senzill per inspeccionar el tipus i el valor d'una variable.

package main

import (
    "fmt"
    "reflect"
)

func main() {
    var x float64 = 3.4
    fmt.Println("Type:", reflect.TypeOf(x))
    fmt.Println("Value:", reflect.ValueOf(x))
}

Sortida:

Type: float64
Value: 3.4

En aquest exemple, reflect.TypeOf(x) retorna el tipus de x, i reflect.ValueOf(x) retorna el valor de x.

Treballant amb Structs

La reflexió és particularment útil quan es treballa amb structs. Pots inspeccionar i manipular els camps d'un struct en temps d'execució.

Exemple: Inspeccionant Camps d'un Struct

package main

import (
    "fmt"
    "reflect"
)

type Person struct {
    Name string
    Age  int
}

func main() {
    p := Person{"Alice", 30}
    t := reflect.TypeOf(p)
    v := reflect.ValueOf(p)

    for i := 0; i < t.NumField(); i++ {
        field := t.Field(i)
        value := v.Field(i)
        fmt.Printf("%s: %v = %v\n", field.Name, field.Type, value)
    }
}

Sortida:

Name: string = Alice
Age: int = 30

En aquest exemple, utilitzem reflect.TypeOf(p) per obtenir el tipus del struct i reflect.ValueOf(p) per obtenir el seu valor. Després iterem sobre els camps utilitzant t.NumField() i accedim al nom, tipus i valor de cada camp.

Modificant Valors

La reflexió també et permet modificar els valors de les variables. No obstant això, necessites utilitzar punters per modificar la variable original.

Exemple: Modificant Camps d'un Struct

package main

import (
    "fmt"
    "reflect"
)

type Person struct {
    Name string
    Age  int
}

func main() {
    p := Person{"Alice", 30}
    v := reflect.ValueOf(&p).Elem()

    nameField := v.FieldByName("Name")
    if nameField.CanSet() {
        nameField.SetString("Bob")
    }

    fmt.Println(p)
}

Sortida:

{Bob 30}

En aquest exemple, utilitzem reflect.ValueOf(&p).Elem() per obtenir un reflect.Value que podem modificar. Després utilitzem FieldByName("Name") per obtenir el camp Name i SetString("Bob") per canviar el seu valor.

Tècniques Avançades

Afirmacions de Tipus amb Reflexió

La reflexió es pot combinar amb afirmacions de tipus per gestionar tipus desconeguts de manera dinàmica.

Exemple: Gestionant Tipus Desconeguts

package main

import (
    "fmt"
    "reflect"
)

func PrintTypeAndValue(i interface{}) {
    v := reflect.ValueOf(i)
    switch v.Kind() {
    case reflect.Int:
        fmt.Printf("Type: int, Value: %d\n", v.Int())
    case reflect.String:
        fmt.Printf("Type: string, Value: %s\n", v.String())
    default:
        fmt.Printf("Type: %s, Value: %v\n", v.Kind(), v)
    }
}

func main() {
    PrintTypeAndValue(42)
    PrintTypeAndValue("hello")
    PrintTypeAndValue(3.14)
}

Sortida:

Type: int, Value: 42
Type: string, Value: hello
Type: float64, Value: 3.14

En aquest exemple, PrintTypeAndValue utilitza reflect.ValueOf(i) per obtenir el valor de la interfície i una instrucció switch sobre v.Kind() per gestionar diferents tipus.

Conclusió

La reflexió en Go és una eina poderosa que et permet inspeccionar i manipular tipus i valors en temps d'execució. És particularment útil per a la gestió dinàmica de tipus, la serialització/deserialització i les proves. No obstant això, s'ha d'utilitzar amb prudència, ja que pot fer que el codi sigui més complex i difícil d'entendre.

Comprenent els conceptes bàsics del paquet reflect i practicant amb exemples, pots aprofitar la reflexió per resoldre problemes complexos de manera flexible i dinàmica.
