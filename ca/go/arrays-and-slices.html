Introducció

En Go, els arrays i slices són estructures de dades fonamentals que et permeten emmagatzemar i gestionar col·leccions d'elements. Entendre com utilitzar-los de manera efectiva és crucial per escriure codi eficient i idiomàtic en Go.

Arrays

Conceptes Clau

  • Mida Fixa: Els arrays tenen una mida fixa que es defineix en el moment de la declaració.
  • Elements Homogenis: Tots els elements d'un array han de ser del mateix tipus.
  • Inicialització amb Valor Zero: Els arrays s'inicialitzen amb valors zero per al seu tipus.

Declaració d'Arrays

Per declarar un array en Go, especifiques la mida i el tipus dels seus elements.

var arr [5]int // Un array de 5 enters

Inicialització d'Arrays

Pots inicialitzar un array en el moment de la declaració.

var arr = [5]int{1, 2, 3, 4, 5}

També pots deixar que Go infereixi la mida de l'array.

var arr = [...]int{1, 2, 3, 4, 5}

Accés als Elements de l'Array

Els elements de l'array s'accedeixen utilitzant un índex basat en zero.

fmt.Println(arr[0]) // Mostra: 1

Modificació dels Elements de l'Array

Pots modificar els elements assignant nous valors a índexs específics.

arr[0] = 10
fmt.Println(arr[0]) // Mostra: 10

Iteració sobre Arrays

Pots utilitzar un bucle for per iterar sobre els elements de l'array.

for i := 0; i < len(arr); i++ {
    fmt.Println(arr[i])
}

Alternativament, pots utilitzar un bucle range.

for index, value := range arr {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

Slices

Conceptes Clau

  • Mida Dinàmica: Els slices tenen una mida dinàmica, cosa que els fa més flexibles que els arrays.
  • Tipus de Referència: Els slices són referències a arrays.
  • Inicialització amb Valor Zero: El valor zero d'un slice és nil.

Declaració de Slices

Pots declarar un slice sense especificar la seva mida.

var slice []int

Creació de Slices

Pots crear un slice utilitzant la funció make.

slice := make([]int, 5) // Crea un slice de longitud 5

També pots crear un slice a partir d'un array existent.

arr := [5]int{1, 2, 3, 4, 5}
slice := arr[1:4] // Crea un slice [2, 3, 4]

Afegir Elements a Slices

Pots afegir elements a un slice utilitzant la funció append.

slice := []int{1, 2, 3}
slice = append(slice, 4, 5)
fmt.Println(slice) // Mostra: [1, 2, 3, 4, 5]

Copiar Slices

Pots copiar elements d'un slice a un altre utilitzant la funció copy.

src := []int{1, 2, 3}
dst := make([]int, len(src))
copy(dst, src)
fmt.Println(dst) // Mostra: [1, 2, 3]

Slicing de Slices

Pots crear un nou slice a partir d'un slice existent.

slice := []int{1, 2, 3, 4, 5}
newSlice := slice[1:4] // Crea un slice [2, 3, 4]

Iteració sobre Slices

Pots iterar sobre slices igual que amb els arrays.

for index, value := range slice {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

Arrays vs. Slices

| Característica | Arrays | Slices | |------------------|---------------------------------|---------------------------------| | Mida | Fixa | Dinàmica | | Inicialització | Valor zero | Valor zero (nil) | | Flexibilitat | Menys flexible | Més flexible | | Sobrecàrrega de Memòria | Menys | Més (degut al redimensionament dinàmic) | | Cas d'Ús | Quan la mida és coneguda i fixa | Quan la mida pot variar |

Conclusió

Els arrays i slices són eines essencials en Go per gestionar col·leccions de dades. Els arrays proporcionen una manera eficient de mida fixa per emmagatzemar elements, mentre que els slices ofereixen flexibilitat i redimensionament dinàmic. Entendre les diferències i els casos d'ús apropiats per a cadascun t'ajudarà a escriure codi més eficient i idiomàtic en Go. Practica utilitzant arrays i slices amb diversos exemples per esdevenir proficient en el seu ús.
