Introducció

A Go, les variables i constants són conceptes fonamentals que et permeten emmagatzemar i gestionar dades. Entendre com declarar, inicialitzar i utilitzar-les és crucial per escriure programes Go efectius. Aquesta secció cobrirà les bases de les variables i constants, les seves diferències i les millors pràctiques per utilitzar-les.

Variables

Declaració de Variables

A Go, les variables es poden declarar utilitzant la paraula clau var o la notació abreujada. Aquí tens les diferents maneres de declarar variables:

  • Utilitzant la paraula clau var:
var name string
var age int
  • Notació abreujada:
name := "John"
age := 30

Inicialització de Variables

Les variables es poden inicialitzar en el moment de la declaració:

  • Amb la paraula clau var:
var name string = "John"
var age int = 30
  • Utilitzant la notació abreujada:
name := "John"
age := 30

Valors Zero

A Go, les variables que es declaren sense un valor inicial s'assignen un "valor zero" per defecte basat en el seu tipus:

  • Valors zero per a tipus bàsics:

| Tipus | Valor Zero | |---------|------------| | int | 0 | | float64 | 0.0 | | string | "" | | bool | false |

Exemple:

var number int
var price float64
var message string
var isAvailable bool

fmt.Println(number)      // Sortida: 0
fmt.Println(price)       // Sortida: 0.0
fmt.Println(message)     // Sortida: ""
fmt.Println(isAvailable) // Sortida: false

Declaracions Múltiples de Variables

Go et permet declarar múltiples variables en una sola línia:

var x, y, z int
x, y, z := 1, 2, 3

Constants

Declaració de Constants

Les constants es declaren utilitzant la paraula clau const i s'han d'assignar un valor en el moment de la declaració. Les constants no es poden canviar un cop establertes.

const Pi = 3.14
const Greeting = "Hello, World!"

Constants Tipades i No Tipades

Les constants poden ser tipades o no tipades:

  • Constants tipades:
const Pi float64 = 3.14
  • Constants no tipades:
const Pi = 3.14

Constants Enumerades

Go suporta constants enumerades utilitzant l'identificador iota, que simplifica la creació d'una seqüència de constants relacionades.

const (
    Red = iota
    Green
    Blue
)

fmt.Println(Red, Green, Blue) // Sortida: 0 1 2

Millors Pràctiques

  • Utilitza noms significatius: Tria noms de variables i constants que descriguin clarament el seu propòsit.
  • Prefereix constants sobre variables: Utilitza constants per a valors que no canvien per fer el teu codi més llegible i mantenible.
  • Agrupa constants relacionades: Utilitza iota per crear constants relacionades per millorar l'organització del codi.

Conclusió

Entendre les variables i constants és essencial per escriure programes Go efectius. Les variables et permeten emmagatzemar i manipular dades, mentre que les constants proporcionen una manera de definir valors fixos que no canvien. Seguint les millors pràctiques i utilitzant els mètodes de declaració adequats, pots escriure codi Go clar i mantenible.
