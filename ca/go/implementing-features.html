Introducció

En aquesta secció, explorarem com implementar diverses funcions en Go. Començarem amb la implementació bàsica de funcions i gradualment avançarem cap a conceptes més avançats. Al final d'aquesta secció, hauríeu de ser capaços d'implementar funcions amb confiança en les vostres aplicacions Go.

Implementació Bàsica de Funcions

Funcions

Les funcions són els blocs de construcció de qualsevol programa Go. Permeten encapsular codi en blocs reutilitzables.

  • Definició d'una Funció: 
    package main

import "fmt"

// Funció que suma dos números
func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 4)
    fmt.Println("Sum:", result)
}
    • func add(a int, b int) int: Defineix una funció anomenada add que pren dos enters i retorna un enter.
    • result := add(3, 4): Crida la funció add amb els arguments 3 i 4.

Structs

Els structs s'utilitzen per crear tipus de dades complexos que agrupen variables.

  • Definició i Ús de Structs: 
    package main

import "fmt"

// Definir un struct
type Person struct {
    Name string
    Age  int
}

func main() {
    // Crear una instància del struct
    person := Person{Name: "Alice", Age: 30}
    fmt.Println("Name:", person.Name)
    fmt.Println("Age:", person.Age)
}
    • type Person struct { ... }: Defineix un struct anomenat Person.
    • person := Person{Name: "Alice", Age: 30}: Crea una instància de Person.

Mètodes

Els mètodes són funcions amb un argument receptor especial.

  • Definició de Mètodes: 
    package main

import "fmt"

type Rectangle struct {
    Width  float64
    Height float64
}

// Mètode per calcular l'àrea
func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

func main() {
    rect := Rectangle{Width: 10, Height: 5}
    fmt.Println("Area:", rect.Area())
}
    • func (r Rectangle) Area() float64: Defineix un mètode Area per al struct Rectangle.

Implementació Intermèdia de Funcions

Interfícies

Les interfícies permeten definir un conjunt de signatures de mètodes que un tipus ha d'implementar.

  • Definició i Implementació d'Interfícies: 
    package main

import "fmt"

// Definir una interfície
type Shape interface {
    Area() float64
}

type Circle struct {
    Radius float64
}

// Implementar la interfície
func (c Circle) Area() float64 {
    return 3.14 * c.Radius * c.Radius
}

func main() {
    var s Shape
    s = Circle{Radius: 5}
    fmt.Println("Area:", s.Area())
}
    • type Shape interface { ... }: Defineix una interfície anomenada Shape.
    • func (c Circle) Area() float64: Implementa el mètode Area per al struct Circle.

Gestió d'Errors

La gestió d'errors en Go es fa utilitzant el tipus error.

  • Gestió Bàsica d'Errors: 
    package main

import (
    "errors"
    "fmt"
)

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

func main() {
    result, err := divide(4, 0)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Result:", result)
    }
}
    • func divide(a, b float64) (float64, error): Defineix una funció que retorna un resultat i un error.
    • errors.New("division by zero"): Crea un nou error.

Implementació Avançada de Funcions

Concurrència

Go proporciona suport integrat per a la programació concurrent amb goroutines i canals.

  • Ús de Goroutines:

    package main

import (
    "fmt"
    "time"
)

func printNumbers() {
    for i := 1; i <= 5; i++ {
        fmt.Println(i)
        time.Sleep(1 * time.Second)
    }
}

func main() {
    go printNumbers() // Inicia una nova goroutine
    fmt.Println("Goroutine started")
    time.Sleep(6 * time.Second) // Espera que la goroutine acabi
}
    • go printNumbers(): Inicia una nova goroutine.

  • Ús de Canals:

    package main

import "fmt"

func sum(a, b int, result chan int) {
    result <- a + b
}

func main() {
    result := make(chan int)
    go sum(3, 4, result)
    fmt.Println("Sum:", <-result)
}
    • result := make(chan int): Crea un nou canal.
    • result <- a + b: Envia un valor al canal.
    • fmt.Println("Sum:", <-result): Rep un valor del canal.
