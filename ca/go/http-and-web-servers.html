Introducció a HTTP i Servidors Web

  • HTTP (HyperText Transfer Protocol): La base de qualsevol intercanvi de dades a la Web, i és un protocol utilitzat per transmetre sol·licituds i informació d'hipertext entre servidors i clients.
  • Servidor Web: Un servidor que utilitza HTTP per servir fitxers que formen pàgines web als usuaris en resposta a les seves sol·licituds, que són reenviades pels clients HTTP dels seus ordinadors.

Configuració d'un Servidor Web Bàsic en Go

  • Go té una potent biblioteca estàndard que inclou el paquet net/http, que facilita la creació de servidors web.

Exemple: Servidor HTTP Bàsic

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    http.HandleFunc("/", handler)
    fmt.Println("Starting server at port 8080")
    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • http.HandleFunc("/", handler): Registra la funció handler per al camí URL arrel.
    • http.ListenAndServe(":8080", nil): Inicia el servidor al port 8080.

Gestió de Diferents Rutes

  • Pots gestionar diferents rutes registrant múltiples funcions handler.

Exemple: Gestió de Múltiples Rutes

package main

import (
    "fmt"
    "net/http"
)

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func aboutHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "About Page")
}

func main() {
    http.HandleFunc("/", helloHandler)
    http.HandleFunc("/about", aboutHandler)
    fmt.Println("Starting server at port 8080")
    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • http.HandleFunc("/about", aboutHandler): Registra la funció aboutHandler per al camí URL /about.

Servir Fitxers Estàtics

  • Servir fitxers estàtics com HTML, CSS i JavaScript és un requisit comú per als servidors web.

Exemple: Servir Fitxers Estàtics

package main

import (
    "net/http"
)

func main() {
    fs := http.FileServer(http.Dir("static"))
    http.Handle("/", fs)

    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • http.FileServer(http.Dir("static")): Crea un servidor de fitxers que serveix fitxers des del directori static.
    • http.Handle("/", fs): Registra el servidor de fitxers per gestionar sol·licituds al camí URL arrel.

Gestió de Formularis i Paràmetres de Consulta

  • Gestionar l'entrada de l'usuari a través de formularis i paràmetres de consulta és essencial per a aplicacions web dinàmiques.

Exemple: Gestió de Formularis

package main

import (
    "fmt"
    "net/http"
)

func formHandler(w http.ResponseWriter, r *http.Request) {
    si r.Method == "POST" {
        name := r.FormValue("name")
        fmt.Fprintf(w, "Hello, %s!", name)
    } else {
        http.ServeFile(w, r, "form.html")
    }
}

func main() {
    http.HandleFunc("/form", formHandler)
    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • r.Method == "POST": Comprova si el mètode de la sol·licitud és POST.
    • r.FormValue("name"): Recupera el valor del camp del formulari anomenat "name".
    • http.ServeFile(w, r, "form.html"): Serveix el fitxer HTML del formulari per a sol·licituds GET.

Exemple: Gestió de Paràmetres de Consulta

package main

import (
    "fmt"
    "net/http"
)

func queryHandler(w http.ResponseWriter, r *http.Request) {
    name := r.URL.Query().Get("name")
    fmt.Fprintf(w, "Hello, %s!", name)
}

func main() {
    http.HandleFunc("/query", queryHandler)
    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • r.URL.Query().Get("name"): Recupera el valor del paràmetre de consulta anomenat "name".

Middleware en Go

  • Middleware és una manera de processar sol·licituds abans que arribin al handler final.

Exemple: Middleware Simple

package main

import (
    "fmt"
    "net/http"
)

func loggingMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        fmt.Println("Request received")
        next.ServeHTTP(w, r)
    })
}

func helloHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    hello := http.HandlerFunc(helloHandler)
    http.Handle("/", loggingMiddleware(hello))
    http.ListenAndServe(":8080", nil)
}
  • Explicació:
    • loggingMiddleware(next http.Handler) http.Handler: Defineix una funció middleware que registra les sol·licituds.
    • next.ServeHTTP(w, r): Crida el següent handler en la cadena.
