Visió general

Go, també conegut com Golang, és un llenguatge de programació compilat i tipat estàticament dissenyat per Google. És conegut per la seva simplicitat, eficiència i fort suport per a la concurrència. Aquesta secció us introduirà als conceptes bàsics de Go, incloent-hi la seva història, característiques clau i com configurar el vostre entorn de desenvolupament.

Història i antecedents

  • Creat per: Robert Griesemer, Rob Pike i Ken Thompson a Google.
  • Primera versió: 2009.
  • Objectius de disseny:
    • Millorar la productivitat de la programació.
    • Proporcionar un llenguatge modern amb fort suport per a la concurrència.
    • Assegurar una compilació i execució ràpides.

Característiques clau de Go

  • Simplicitat: Disseny minimalista amb una biblioteca estàndard petita.
  • Concurrència: Suport integrat utilitzant goroutines i canals.
  • Rendiment: Llenguatge compilat amb execució eficient.
  • Recollida d'escombraries: Gestió automàtica de la memòria.
  • Tipatge fort: Tipat estàtic amb inferència de tipus.

Configuració de l'entorn de desenvolupament de Go

  1. Descarregar i instal·lar Go:

    • Visiteu el lloc web oficial de Go i descarregueu l'instal·lador per al vostre sistema operatiu.
    • Seguiu les instruccions d'instal·lació proporcionades al lloc web.

  2. Verificar la instal·lació:

    • Obriu una terminal o símbol del sistema.
    • Executeu la comanda go version per comprovar si Go està instal·lat correctament.

  3. Configurar l'espai de treball:

    • Creeu un directori per als vostres projectes Go, per exemple, ~/go.
    • Configureu la variable d'entorn GOPATH per apuntar al vostre directori d'espai de treball.

  4. Programa Hello World:

    • Creeu un nou directori per al vostre projecte, per exemple, ~/go/src/hello.

    • Creeu un fitxer anomenat main.go amb el següent contingut:

      package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

    • Navegueu al directori del projecte i executeu el programa utilitzant la comanda go run main.go.

Sintaxi i estructura bàsiques

  • Paquets: Cada programa Go està format per paquets. El paquet main és el punt d'entrada del programa.
  • Imports: Utilitzeu la paraula clau import per incloure altres paquets.
  • Funcions: Definides utilitzant la paraula clau func.

Exemple: Estructura bàsica del programa

package main

import "fmt"

func main() {
    fmt.Println("Welcome to Go!")
}
  • Explicació:
    • package main: Defineix el nom del paquet.
    • import "fmt": Importa el paquet fmt per a I/O formatat.
    • func main() { ... }: La funció principal on comença l'execució del programa.

Variables i tipus de dades

  • Declaració de variables: Utilitzeu la paraula clau var o la forma abreujada :=.
  • Tipus de dades bàsics: int, float64, string, bool.

Exemple: Declaració de variables

package main

import "fmt"

func main() {
    var a int = 10
    b := 20
    var c, d int = 30, 40
    var e, f = 50, "hello"
    fmt.Println(a, b, c, d, e, f)
}
  • Explicació:
    • var a int = 10: Declara una variable enter a amb valor 10.
    • b := 20: Forma abreujada per declarar i inicialitzar b amb valor 20.
    • var c, d int = 30, 40: Declara dues variables enteres c i d.
    • var e, f = 50, "hello": Declara e com a enter i f com a cadena.

Estructures de control

  • Sentències If-Else: Execució condicional.
  • Bucles For: L'única estructura de bucle a Go.
  • Sentències Switch: Simplifica múltiples branques condicionals.

Exemple: Estructures de control

package main

import "fmt"

func main() {
    // If-Else
    x := 10
    if x > 5 {
        fmt.Println("x és més gran que 5")
    } else {
        fmt.Println("x és 5 o menys")
    }

    // Bucle For
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }

    // Sentència Switch
    dia := "Dilluns"
    switch dia {
    case "Dilluns":
        fmt.Println("Inici de la setmana laboral")
    case "Divendres":
        fmt.Println("Final de la setmana laboral")
    default:
        fmt.Println("Dia de la setmana")
    }
}
