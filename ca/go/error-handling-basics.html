Errors Personalitzats ►

La gestió d'errors és un aspecte crucial de la programació, i Go proporciona una manera única i efectiva de gestionar errors. Aquesta secció cobrirà els conceptes bàsics de la gestió d'errors en Go, incloent com crear, gestionar i propagar errors.

Conceptes Clau

  • Tipus d'Error: En Go, els errors estan representats pel tipus incorporat error.
  • Creació d'Errors: Utilitza la funció errors.New o la funció fmt.Errorf per crear nous valors d'error.
  • Gestió d'Errors: Comprova els errors utilitzant simples sentències if.
  • Propagació d'Errors: Retorna errors des de les funcions per propagar-los a la pila de crides.
  • Errors Personalitzats: Implementa la interfície error per crear tipus d'errors personalitzats.

Creació d'Errors

En Go, pots crear un error utilitzant la funció errors.New del paquet errors o la funció fmt.Errorf del paquet fmt.

Utilitzant `errors.New`

package main

import (
    "errors"
    "fmt"
)

func main() {
    err := errors.New("alguna cosa ha anat malament")
    if err != nil {
        fmt.Println(err)
    }
}

Utilitzant `fmt.Errorf`

package main

import (
    "fmt"
)

func main() {
    err := fmt.Errorf("s'ha produït un error: %v", "alguna cosa ha anat malament")
    if err != nil {
        fmt.Println(err)
    }
}

Gestió d'Errors

La gestió d'errors en Go es fa típicament utilitzant sentències if. Si una funció retorna un error, hauries de comprovar si l'error és nil per determinar si s'ha produït un error.

package main

import (
    "fmt"
    "strconv"
)

func main() {
    _, err := strconv.Atoi("123a")
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Conversió reeixida")
    }
}

Propagació d'Errors

Quan escrius funcions, hauries de retornar errors per permetre que el cridant els gestioni adequadament. Això es coneix com a propagació d'errors.

package main

import (
    "errors"
    "fmt"
)

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("no es pot dividir per zero")
    }
    return a / b, nil
}

func main() {
    result, err := divide(4, 0)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Resultat:", result)
    }
}

Errors Personalitzats

Pots crear tipus d'errors personalitzats implementant la interfície error. Això et permet afegir context addicional als teus errors.

package main

import (
    "fmt"
)

type MyError struct {
    Codi    int
    Missatge string
}

func (e *MyError) Error() string {
    return fmt.Sprintf("Error %d: %s", e.Codi, e.Missatge)
}

func doSomething(flag bool) error {
    if !flag {
        return &MyError{Codi: 123, Missatge: "alguna cosa ha anat malament"}
    }
    return nil
}

func main() {
    err := doSomething(false)
    if err != nil {
        fmt.Println(err)
    }
}

Resum

  • Go utilitza el tipus incorporat error per a la gestió d'errors.
  • Crea errors utilitzant errors.New o fmt.Errorf.
  • Gestiona errors utilitzant sentències if per comprovar si l'error és nil.
  • Propaga errors retornant-los des de les funcions.
  • Implementa la interfície error per crear tipus d'errors personalitzats.

Entendre aquests conceptes bàsics t'ajudarà a escriure programes en Go més robustos i mantenibles. A mesura que progressis, aprendràs tècniques més avançades per a la gestió d'errors, com utilitzar el paquet errors per embolicar i desempaquetar errors.
