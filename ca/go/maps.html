Els mapes són un tipus de dades integrat en Go que associa claus amb valors. Són similars als diccionaris en Python o les taules de hash en altres llenguatges. Aquesta secció cobrirà els conceptes bàsics dels mapes, com utilitzar-los i algunes tècniques avançades.

Introducció als Mapes

  • Els mapes són col·leccions de parelles clau-valor.
  • Les claus han de ser d'un tipus que sigui comparable (per exemple, cadenes, enters).
  • Els valors poden ser de qualsevol tipus.
  • Els mapes proporcionen cerques, actualitzacions i eliminacions ràpides.

Crear i Inicialitzar Mapes

Pots crear un mapa utilitzant la funció make o un literal de mapa.

// Utilitzant la funció make
var m1 map[string]int
m1 = make(map[string]int)

// Utilitzant un literal de mapa
m2 := map[string]int{
    "Alice": 25,
    "Bob":   30,
}

Afegir i Recuperar Elements

Per afegir o actualitzar elements en un mapa, utilitza l'operador d'assignació. Per recuperar un element, utilitza la clau.

m := make(map[string]int)
m["Alice"] = 25
m["Bob"] = 30

// Recuperar valor
age := m["Alice"]
fmt.Println("L'edat de l'Alice:", age) // Sortida: L'edat de l'Alice: 25

Comprovar l'Existència

Per comprovar si una clau existeix en un mapa, utilitza l'assignació de dos valors.

age, exists := m["Charlie"]
if exists {
    fmt.Println("L'edat d'en Charlie:", age)
} else {
    fmt.Println("En Charlie no és al mapa")
}

Eliminar Elements

Per eliminar un element d'un mapa, utilitza la funció delete.

delete(m, "Bob")

Iterar Sobre Mapes

Pots iterar sobre un mapa utilitzant un bucle for.

for key, value := range m {
    fmt.Printf("%s: %d\n", key, value)
}

Tècniques Avançades amb Mapes

Mapes Imbricats

Els mapes poden contenir altres mapes com a valors, permetent estructures de dades més complexes.

nestedMap := map[string]map[string]int{
    "Grup1": {
        "Alice": 25,
        "Bob":   30,
    },
    "Grup2": {
        "Charlie": 35,
        "Dave":    40,
    },
}

Mapes amb Structs

Els mapes també poden utilitzar structs com a valors per a dades més estructurades.

type Person struct {
    Age  int
    City string
}

people := map[string]Person{
    "Alice": {25, "Nova York"},
    "Bob":   {30, "San Francisco"},
}

Consideracions de Rendiment

  • Els mapes proporcionen una complexitat temporal mitjana de O(1) per a cerques, insercions i eliminacions.
  • No obstant això, els mapes no són segurs per a fils. Utilitza mecanismes de sincronització com sync.Mutex o sync.RWMutex quan accedeixis a mapes des de múltiples goroutines.

Errors Comuns

  • Valor Zero: El valor zero d'un mapa és nil. Un mapa nil es comporta com un mapa buit quan es llegeix, però causa un pànic en temps d'execució quan s'escriu.
  • Tipus de Claus: Només els tipus que són comparables poden ser utilitzats com a claus. Per exemple, les slices no poden ser utilitzades com a claus.

Conclusió

Els mapes són una estructura de dades poderosa i flexible en Go, proporcionant un emmagatzematge i recuperació de claus-valors eficient. Entendre com crear, manipular i iterar sobre mapes és essencial per a una programació efectiva en Go. Dominant els mapes, podràs gestionar una àmplia gamma de tasques d'emmagatzematge i recuperació de dades de manera eficient.
