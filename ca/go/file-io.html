Introducció a l'Entrada/Sortida de Fitxers

L'entrada/sortida de fitxers (Input/Output) és un aspecte fonamental de la programació que permet a un programa llegir i escriure fitxers. En Go, el paquet os proporciona una interfície independent de la plataforma per a la funcionalitat del sistema operatiu, incloent-hi l'entrada/sortida de fitxers.

Llegir Fitxers

Obrir un Fitxer

Per llegir un fitxer, primer cal obrir-lo utilitzant la funció os.Open. Aquesta funció retorna un descriptor de fitxer i un error.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("example.txt")
    if err != nil {
        fmt.Println("Error opening file:", err)
        return
    }
    defer file.Close()
}

Llegir el Contingut del Fitxer

Un cop el fitxer està obert, pots llegir el seu contingut utilitzant diversos mètodes dels paquets bufio i io/ioutil.

Utilitzant bufio.Scanner

El bufio.Scanner és útil per llegir un fitxer línia per línia.

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("example.txt")
    if (err != nil) {
        fmt.Println("Error opening file:", err)
        return
    }
    defer file.Close()

    scanner := bufio.NewScanner(file)
    for scanner.Scan() {
        fmt.Println(scanner.Text())
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("Error reading file:", err)
    }
}

Utilitzant ioutil.ReadFile

La funció ioutil.ReadFile llegeix tot el contingut del fitxer a la memòria.

package main

import (
    "fmt"
    "io/ioutil"
)

func main() {
    content, err := ioutil.ReadFile("example.txt")
    if err != nil {
        fmt.Println("Error reading file:", err)
        return
    }
    fmt.Println(string(content))
}

Escriure Fitxers

Crear o Obrir un Fitxer per Escriure

Per escriure a un fitxer, pots utilitzar os.Create o os.OpenFile. La funció os.Create crea un fitxer si no existeix o el trunca si ho fa.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creating file:", err)
        return
    }
    defer file.Close()
}

Escriure Contingut a un Fitxer

Pots escriure a un fitxer utilitzant els mètodes Write o WriteString.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creating file:", err)
        return
    }
    defer file.Close()

    _, err = file.WriteString("Hello, World!\n")
    if err != nil {
        fmt.Println("Error writing to file:", err)
    }
}

Utilitzant `bufio.Writer`

Per escriure amb buffer, pots utilitzar bufio.Writer.

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creating file:", err)
        return
    }
    defer file.Close()

    writer := bufio.NewWriter(file)
    _, err = writer.WriteString("Hello, Buffered World!\n")
    if err != nil {
        fmt.Println("Error writing to file:", err)
    }
    writer.Flush()
}

Operacions Avançades amb Fitxers

Permisos de Fitxers

Quan crees o obres fitxers, pots especificar els permisos del fitxer utilitzant os.OpenFile.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.OpenFile("example.txt", os.O_CREATE|os.O_WRONLY, 0644)
    if err != nil {
        fmt.Println("Error opening file:", err)
        return
    }
    defer file.Close()

    _, err = file.WriteString("Hello with permissions!\n")
    if err != nil {
        fmt.Println("Error writing to file:", err)
    }
}

Informació del Fitxer

Pots obtenir informació del fitxer utilitzant la funció os.Stat.

package main

import (
    "fmt"
    "os"
)

func main() {
    fileInfo, err := os.Stat("example.txt")
    if err != nil {
        fmt.Println("Error getting file info:", err)
        return
    }
    fmt.Printf("File Name: %s\n", fileInfo.Name())
    fmt.Printf("Size: %d bytes\n", fileInfo.Size())
    fmt.Printf("Permissions: %s\n", fileInfo.Mode())
    fmt.Printf("Last Modified: %s\n", fileInfo.ModTime())
}

Conclusió

L'entrada/sortida de fitxers és una habilitat crucial per a qualsevol programador. En Go, els paquets os, bufio i io/ioutil proporcionen eines potents per treballar amb fitxers. Comprenent com llegir i escriure fitxers, gestionar errors i administrar permisos de fitxers, pots crear aplicacions robustes que interactuen amb el sistema de fitxers de manera efectiva.
