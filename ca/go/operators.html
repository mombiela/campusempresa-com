Introducció

Els operadors són blocs fonamentals en qualsevol llenguatge de programació. Permeten realitzar diverses operacions sobre variables i valors. En Go, els operadors es classifiquen en diversos tipus, incloent-hi operadors aritmètics, relacionals, lògics, de bits i d'assignació. Aquesta secció cobrirà cada tipus d'operador, proporcionant explicacions clares i exemples per ajudar-te a entendre com utilitzar-los de manera efectiva.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques com l'addició, la sostracció, la multiplicació, la divisió i el mòdul.

  • + : Addició
  • - : Sostracció
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul

Exemple

package main

import "fmt"

func main() {
    a := 10
    b := 3

    fmt.Println("Addició:", a + b)        // Output: 13
    fmt.Println("Sostracció:", a - b)     // Output: 7
    fmt.Println("Multiplicació:", a * b)  // Output: 30
    fmt.Println("Divisió:", a / b)        // Output: 3
    fmt.Println("Mòdul:", a % b)          // Output: 1
}

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors. Retornen un resultat booleà (true o false).

  • == : Igual a
  • != : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual que
  • <= : Menor o igual que

Exemple

package main

import "fmt"

func main() {
    a := 10
    b := 20

    fmt.Println("a == b:", a == b)  // Output: false
    fmt.Println("a != b:", a != b)  // Output: true
    fmt.Println("a > b:", a > b)    // Output: false
    fmt.Println("a < b:", a < b)    // Output: true
    fmt.Println("a >= b:", a >= b)  // Output: false
    fmt.Println("a <= b:", a <= b)  // Output: true
}

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions o valors booleans.

  • && : AND lògic
  • || : OR lògic
  • ! : NOT lògic

Exemple

package main

import "fmt"

func main() {
    a := true
    b := false

    fmt.Println("a && b:", a && b)  // Output: false
    fmt.Println("a || b:", a || b)  // Output: true
    fmt.Println("!a:", !a)          // Output: false
}

Operadors de Bits

Els operadors de bits realitzen operacions a nivell de bit sobre tipus enters.

  • & : AND de bits
  • | : OR de bits
  • ^ : XOR de bits
  • &^ : Clear de bits (AND NOT)
  • << : Desplaçament a l'esquerra
  • >> : Desplaçament a la dreta

Exemple

package main

import "fmt"

func main() {
    a := 10  // 1010 en binari
    b := 3   // 0011 en binari

    fmt.Println("a & b:", a & b)    // Output: 2  (0010 en binari)
    fmt.Println("a | b:", a & b)    // Output: 11 (1011 en binari)
    fmt.Println("a ^ b:", a ^ b)    // Output: 9  (1001 en binari)
    fmt.Println("a &^ b:", a &^ b)  // Output: 8  (1000 en binari)
    fmt.Println("a << 1:", a << 1)  // Output: 20 (10100 en binari)
    fmt.Println("a >> 1:", a >> 1)  // Output: 5  (0101 en binari)
}

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. Inclouen l'operador d'assignació estàndard = i els operadors d'assignació composta.

  • = : Assignar
  • += : Sumar i assignar
  • -= : Restar i assignar
  • *= : Multiplicar i assignar
  • /= : Dividir i assignar
  • %= : Mòdul i assignar
  • &= : AND de bits i assignar
  • |= : OR de bits i assignar
  • ^= : XOR de bits i assignar
  • <<=: Desplaçament a l'esquerra i assignar
  • >>=: Desplaçament a la dreta i assignar

Exemple

package main

import "fmt"

func main() {
    a := 10
    b := 3

    a += b  // Equivalent a a = a + b
    fmt.Println("a += b:", a)  // Output: 13

    a -= b  // Equivalent a a = a - b
    fmt.Println("a -= b:", a)  // Output: 10

    a *= b  // Equivalent a a = a * b
    fmt.Println("a *= b:", a)  // Output: 30

    a /= b  // Equivalent a a = a / b
    fmt.Println("a /= b:", a)  // Output: 10

    a %= b  // Equivalent a a = a % b
    fmt.Println("a %= b:", a)  // Output: 1
}
