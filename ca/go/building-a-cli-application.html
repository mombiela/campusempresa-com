Introducció

Les aplicacions d'Interfície de Línia de Comandes (CLI) són eines poderoses que permeten als usuaris interactuar amb el programari mitjançant comandes de text. En aquesta secció, aprendrem a construir una aplicació CLI utilitzant Go. Cobrirem els conceptes bàsics, explorarem biblioteques populars i construirem una eina CLI simple des de zero.

Conceptes Clau

  • Aplicació CLI: Un programa que processa comandes introduïdes pels usuaris en un terminal o símbol del sistema.
  • Flags: Arguments de línia de comandes que modifiquen el comportament de l'aplicació CLI.
  • Subcomandes: Comandes dins d'una aplicació CLI que realitzen tasques específiques.
  • Biblioteques: Paquets que proporcionen utilitats per simplificar la creació d'aplicacions CLI.

Configurant el Teu Entorn

Abans de començar a construir la nostra aplicació CLI, assegura't de tenir Go instal·lat a la teva màquina. Pots descarregar-lo des del lloc web oficial de Go.

Aplicació CLI Bàsica

Comencem amb una aplicació CLI simple que accepta un nom com a argument i imprimeix una salutació.

Exemple: Hello World CLI

package main

import (
    "flag"
    "fmt"
)

func main() {
    name := flag.String("name", "World", "a name to say hello to")
    flag.Parse()
    fmt.Printf("Hello, %s!\n", *name)
}

Explicació

  • flag.String: Defineix una flag de cadena amb un valor per defecte ("World") i una descripció.
  • flag.Parse: Analitza les flags de la línia de comandes.
  • fmt.Printf: Imprimeix la cadena formatada a la consola.

Utilitzant una Biblioteca CLI

Tot i que el paquet flag de la biblioteca estàndard és útil, hi ha biblioteques més poderoses com cobra i urfave/cli que proporcionen funcions addicionals i una millor estructura per a aplicacions CLI complexes.

Exemple: Utilitzant Cobra

Primer, instal·la el paquet cobra:

go get -u github.com/spf13/cobra/cobra

Ara, creem una aplicació CLI simple utilitzant cobra.

package main

import (
    "fmt"
    "github.com/spf13/cobra"
)

func main() {
    var rootCmd = &cobra.Command{
        Use:   "greet",
        Short: "Greet és una eina CLI per saludar la gent",
        Run: func(cmd *cobra.Command, args []string) {
            fmt.Println("Hello, World!")
        },
    }

    var name string
    var greetCmd = &cobra.Command{
        Use:   "hello",
        Short: "Imprimeix un missatge de salutació",
        Run: func(cmd *cobra.Command, args []string) {
            fmt.Printf("Hello, %s!\n", name)
        },
    }

    greetCmd.Flags().StringVarP(&name, "name", "n", "World", "nom per saludar")
    rootCmd.AddCommand(greetCmd)
    rootCmd.Execute()
}

Explicació

  • cobra.Command: Defineix una comanda amb ús, descripció i una funció per executar.
  • Flags().StringVarP: Afegeix una flag de cadena a la comanda.
  • AddCommand: Afegeix una subcomanda a la comanda arrel.
  • Execute: Executa la comanda arrel.

Construint una Aplicació CLI Més Complexa

Construïm una aplicació CLI més complexa amb múltiples subcomandes i flags.

Exemple: Gestor de Tasques CLI

package main

import (
    "fmt"
    "github.com/spf13/cobra"
)

var tasks = []string{}

func main() {
    var rootCmd = &cobra.Command{Use: "task"}

    var addCmd = &cobra.Command{
        Use:   "add [task]",
        Short: "Afegeix una nova tasca",
        Args:  cobra.MinimumNArgs(1),
        Run: func(cmd *cobra.Command, args []string) {
            task := args[0]
            tasks = append(tasks, task)
            fmt.Printf("Afegida tasca: %s\n", task)
        },
    }

    var listCmd = &cobra.Command{
        Use:   "list",
        Short: "Llista totes les tasques",
        Run: func(cmd *cobra.Command, args []string) {
            fmt.Println("Tasques:")
            for i, task := range tasks {
                fmt.Printf("%d: %s\n", i+1, task)
            }
        },
    }

    rootCmd.AddCommand(addCmd, listCmd)
    rootCmd.Execute()
}

Explicació

  • Args: cobra.MinimumNArgs(1): Assegura que la comanda rebi almenys un argument.
  • tasks: Un slice per emmagatzemar tasques.
  • addCmd: Afegeix una nova tasca a la llista.
  • listCmd: Llista totes les tasques.

Conclusió

Construir aplicacions CLI en Go pot ser senzill amb les eines i biblioteques adequades. Hem començat amb un exemple bàsic utilitzant la biblioteca estàndard i després hem explorat la biblioteca cobra per construir aplicacions més complexes. Entenent les flags, subcomandes i com estructurar la teva aplicació, pots crear eines CLI poderoses i fàcils d'utilitzar.

En les properes seccions, aprofundirem en temes avançats com provar aplicacions CLI, gestionar errors i empaquetar la teva eina CLI per a la distribució.
