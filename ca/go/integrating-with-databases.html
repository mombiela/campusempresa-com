Introducció

Integrar-se amb bases de dades és una habilitat crucial per a qualsevol desenvolupador de Go. Aquesta secció cobrirà com connectar-se a bases de dades, realitzar operacions CRUD (Crear, Llegir, Actualitzar, Esborrar) i utilitzar eines ORM (Mapatge Objecte-Relacional) en Go. Al final d'aquesta secció, seràs capaç d'interactuar eficientment amb bases de dades en les teves aplicacions Go.

Connectar-se a una Base de Dades

Per interactuar amb una base de dades en Go, normalment utilitzes el paquet database/sql juntament amb un controlador de base de dades.

Instal·lar un Controlador de Base de Dades

Primer, necessites instal·lar un controlador de base de dades. Per exemple, per connectar-te a una base de dades MySQL, pots utilitzar el paquet go-sql-driver/mysql.

go get -u github.com/go-sql-driver/mysql

Establir una Connexió

Aquí tens com pots establir una connexió a una base de dades MySQL:

package main

import (
    "database/sql"
    "fmt"
    _ "github.com/go-sql-driver/mysql"
)

func main() {
    dsn := "username:password@tcp(127.0.0.1:3306)/dbname"
    db, err := sql.Open("mysql", dsn)
    if err != nil {
        panic(err)
    }
    defer db.Close()

    // Comprovar si la connexió és exitosa
    err = db.Ping()
    if err != nil {
        panic(err)
    }

    fmt.Println("Connectat a la base de dades amb èxit!")
}
  • dsn: Nom de la Font de Dades, que inclou el nom d'usuari, contrasenya, host, port i nom de la base de dades.
  • sql.Open: Obre una connexió a la base de dades.
  • db.Ping: Verifica la connexió a la base de dades.

Realitzar Operacions CRUD

Les operacions CRUD són fonamentals quan es treballa amb bases de dades. Exploreu com realitzar aquestes operacions en Go.

Crear

Per inserir dades en una base de dades, utilitzes el mètode Exec.

func createUser(db *sql.DB, name string, age int) error {
    query := "INSERT INTO users (name, age) VALUES (?, ?)"
    _, err := db.Exec(query, name, age)
    return err
}

Llegir

Per llegir dades d'una base de dades, utilitzes el mètode Query o QueryRow.

func getUser(db *sql.DB, id int) (string, int, error) {
    var name string
    var age int
    query := "SELECT name, age FROM users WHERE id = ?"
    err := db.QueryRow(query, id).Scan(&name, &age)
    return name, age, err
}

Actualitzar

Per actualitzar dades existents, utilitzes el mètode Exec.

func updateUser(db *sql.DB, id int, name string, age int) error {
    query := "UPDATE users SET name = ?, age = ? WHERE id = ?"
    _, err := db.Exec(query, name, age, id)
    return err
}

Esborrar

Per esborrar dades, també utilitzes el mètode Exec.

func deleteUser(db *sql.DB, id int) error {
    query := "DELETE FROM users WHERE id = ?"
    _, err := db.Exec(query, id)
    return err
}

Utilitzar ORM amb Go

Les eines de Mapeig Objecte-Relacional (ORM) poden simplificar les interaccions amb la base de dades permetent-te treballar amb objectes en lloc de consultes SQL crues. Un ORM popular per a Go és GORM.

Instal·lar GORM

Primer, instal·la GORM i el controlador de base de dades.

go get -u gorm.io/gorm
go get -u gorm.io/driver/mysql

Connectar-se amb GORM

Aquí tens com establir una connexió utilitzant GORM:

package main

import (
    "gorm.io/driver/mysql"
    "gorm.io/gorm"
    "fmt"
)

func main() {
    dsn := "username:password@tcp(127.0.0.1:3306)/dbname"
    db, err := gorm.Open(mysql.Open(dsn), &gorm.Config{})
    if (err != nil) {
        panic(err)
    }

    fmt.Println("Connectat a la base de dades amb èxit amb GORM!")
}

Definir Models

Defineix un model per representar una taula a la base de dades.

type User struct {
    ID   uint   `gorm:"primaryKey"`
    Name string
    Age  int
}

Realitzar Operacions CRUD amb GORM

GORM fa que les operacions CRUD siguin senzilles.

Crear

func createUser(db *gorm.DB, user User) error {
    return db.Create(&user).Error
}

Llegir

func getUser(db *gorm.DB, id uint) (User, error) {
    var user User
    err := db.First(&user, id).Error
    return user, err
}

Actualitzar

func updateUser(db *gorm.DB, user User) error {
    return db.Save(&user).Error
}

Esborrar

func deleteUser(db *gorm.DB, id uint) error {
    return db.Delete(&User{}, id).Error
}

Conclusió

Integrar-se amb bases de dades en Go implica entendre com connectar-se a una base de dades, realitzar operacions CRUD i, opcionalment, utilitzar un ORM per a aplicacions més complexes. Amb el paquet database/sql i eines com GORM, pots gestionar eficientment les interaccions amb la base de dades en les teves aplicacions Go. Practica aquests conceptes amb diferents bases de dades i escenaris per solidificar la teva comprensió i convertir-te en un expert en integració de bases de dades amb Go.
