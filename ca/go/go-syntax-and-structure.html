Introducció

Go, també conegut com a Golang, és un llenguatge de programació compilat i tipat estàticament dissenyat per a la simplicitat i l'eficiència. Entendre la sintaxi i l'estructura de Go és crucial per escriure programes efectius i eficients. Aquesta secció cobrirà els conceptes bàsics de la sintaxi i l'estructura de Go, proporcionant una base per a temes més avançats.

Sintaxi Bàsica

Hola Món

El punt de partida tradicional per aprendre qualsevol llenguatge de programació és el programa "Hola, Món!". Així és com es veu en Go:

package main

import "fmt"

func main() {
    fmt.Println("Hola, Món!")
}
  • package main: Defineix el nom del paquet. El paquet main és especial perquè defineix un programa executable independent.
  • import "fmt": Importa el paquet fmt, que conté funcions per a I/O formatat.
  • func main() { ... }: La funció main és el punt d'entrada del programa.

Variables i Constants

Les variables en Go es declaren utilitzant la paraula clau var, mentre que les constants es declaren utilitzant la paraula clau const.

var name string = "John"
var age int = 30
const pi float64 = 3.14159
  • var name string = "John": Declara una variable name de tipus string i la inicialitza amb "John".
  • const pi float64 = 3.14159: Declara una constant pi de tipus float64.

Declaracions Curtes de Variables

Go també suporta declaracions curtes de variables utilitzant la sintaxi :=.

name := "John"
age := 30
  • name := "John": Declara i inicialitza name amb "John". El tipus es dedueix.

Tipus de Dades Bàsics

Go té diversos tipus de dades bàsics:

  • int, float64, string, bool
  • Exemple:
var a int = 10
var b float64 = 3.14
var c string = "Hola"
var d bool = true

Estructures de Control

Sentències If-Else

Les sentències condicionals en Go són similars a altres llenguatges.

if age > 18 {
    fmt.Println("Adult")
} else {
    fmt.Println("Menor")
}

Bucles For

Go utilitza la paraula clau for per als bucles.

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

Sentències Switch

Les sentències switch proporcionen una manera d'executar codi diferent basat en el valor d'una variable.

switch day := "Monday"; day {
case "Monday":
    fmt.Println("Inici de la setmana")
case "Friday":
    fmt.Println("Final de la setmana")
default:
    fmt.Println("Mitja setmana")
}

Funcions

Declaració Bàsica de Funcions

Les funcions en Go es declaren utilitzant la paraula clau func.

func add(a int, b int) int {
    return a + b
}
  • func add(a int, b int) int: Declara una funció add que pren dos paràmetres int i retorna un int.

Valors de Retorn Múltiples

Les funcions en Go poden retornar múltiples valors.

func swap(x, y string) (string, string) {
    return y, x
}
  • func swap(x, y string) (string, string): Declara una funció swap que pren dos paràmetres string i retorna dos valors string.

Structs i Mètodes

Structs

Els structs s'utilitzen per agrupar dades relacionades.

type Person struct {
    Name string
    Age  int
}
  • type Person struct { ... }: Defineix un nou tipus de struct Person.

Mètodes

Els mètodes són funcions amb un argument receptor.

func (p Person) greet() string {
    return "Hola, em dic " + p.Name
}
  • func (p Person) greet() string: Declara un mètode greet en el tipus Person.

Paquets i Mòduls

Creació d'un Paquet

Els paquets s'utilitzen per organitzar el codi en components reutilitzables.

// file: mathutil/mathutil.go
package mathutil

func Add(a, b int) int {
    return a + b
}

Ús d'un Paquet

Per utilitzar un paquet, cal importar-lo.

package main

import (
    "fmt"
    "path/to/mathutil"
)

func main() {
    result := mathutil.Add(1, 2)
    fmt.Println(result)
}

Conclusió

Entendre la sintaxi i l'estructura de Go és essencial per escriure codi eficient i llegible. Aquesta secció ha cobert els conceptes bàsics, incloent la declaració de variables, estructures de control, funcions, structs i paquets. Amb aquesta base, estàs ben preparat per explorar temes més avançats en la programació amb Go.
