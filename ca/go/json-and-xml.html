Introducció a JSON i XML

JSON (JavaScript Object Notation) i XML (eXtensible Markup Language) són dos formats populars per a l'intercanvi de dades. Ambdós són àmpliament utilitzats en serveis web, APIs i fitxers de configuració.

  • JSON: Lleuger, fàcil de llegir i escriure, i àmpliament utilitzat en aplicacions web.
  • XML: Més verbós, suporta estructures de dades complexes, i s'utilitza en diverses aplicacions empresarials.

Treballant amb JSON en Go

Codificació i Decodificació de JSON

Go proporciona el paquet encoding/json per treballar amb dades JSON. Aquest paquet permet codificar estructures de dades de Go en JSON i decodificar dades JSON en estructures de dades de Go.

Codificant Estructures de Dades de Go a JSON

Per codificar una estructura de dades de Go a JSON, pots utilitzar la funció json.Marshal.

package main

import (
    "encoding/json"
    "fmt"
)

type Person struct {
    Name string `json:"name"`
    Age  int    `json:"age"`
}

func main() {
    person := Person{Name: "Alice", Age: 30}
    jsonData, err := json.Marshal(person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Println(string(jsonData))
}

Decodificant JSON a Estructures de Dades de Go

Per decodificar dades JSON en una estructura de dades de Go, pots utilitzar la funció json.Unmarshal.

package main

import (
    "encoding/json"
    "fmt"
)

type Person struct {
    Name string `json:"name"`
    Age  int    `json:"age"`
}

func main() {
    jsonData := `{"name":"Alice","age":30}`
    var person Person
    err := json.Unmarshal([]byte(jsonData), &person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Printf("%+v\n", person)
}

Gestionant JSON amb Estructures Complexes

JSON pot representar estructures de dades complexes com objectes niats i arrays. El paquet encoding/json de Go també pot gestionar aquestes.

package main

import (
    "encoding/json"
    "fmt"
)

type Address struct {
    Street string `json:"street"`
    City   string `json:"city"`
}

type Person struct {
    Name    string   `json:"name"`
    Age     int      `json:"age"`
    Address Address  `json:"address"`
    Hobbies []string `json:"hobbies"`
}

func main() {
    jsonData := `{
        "name": "Alice",
        "age": 30,
        "address": {
            "street": "123 Main St",
            "city": "Wonderland"
        },
        "hobbies": ["reading", "biking"]
    }`
    var person Person
    err := json.Unmarshal([]byte(jsonData), &person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Printf("%+v\n", person)
}

Treballant amb XML en Go

Codificació i Decodificació de XML

Go proporciona el paquet encoding/xml per treballar amb dades XML. Aquest paquet permet codificar estructures de dades de Go en XML i decodificar dades XML en estructures de dades de Go.

Codificant Estructures de Dades de Go a XML

Per codificar una estructura de dades de Go a XML, pots utilitzar la funció xml.Marshal.

package main

import (
    "encoding/xml"
    "fmt"
)

type Person struct {
    XMLName xml.Name `xml:"person"`
    Name    string   `xml:"name"`
    Age     int      `xml:"age"`
}

func main() {
    person := Person{Name: "Alice", Age: 30}
    xmlData, err := xml.Marshal(person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Println(string(xmlData))
}

Decodificant XML a Estructures de Dades de Go

Per decodificar dades XML en una estructura de dades de Go, pots utilitzar la funció xml.Unmarshal.

package main

import (
    "encoding/xml"
    "fmt"
)

type Person struct {
    XMLName xml.Name `xml:"person"`
    Name    string   `xml:"name"`
    Age     int      `xml:"age"`
}

func main() {
    xmlData := `<person><name>Alice</name><age>30</age></person>`
    var person Person
    err := xml.Unmarshal([]byte(xmlData), &person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Printf("%+v\n", person)

Gestionant XML amb Estructures Complexes

XML pot representar estructures de dades complexes com elements niats i atributs. El paquet encoding/xml de Go també pot gestionar aquestes.

package main

import (
    "encoding/xml"
    "fmt"
)

type Address struct {
    Street string `xml:"street"`
    City   string `xml:"city"`
}

type Person struct {
    XMLName xml.Name `xml:"person"`
    Name    string   `xml:"name"`
    Age     int      `xml:"age"`
    Address Address  `xml:"address"`
    Hobbies []string `xml:"hobbies>hobby"`
}

func main() {
    xmlData := `<person>
        <name>Alice</name>
        <age>30</age>
        <address>
            <street>123 Main St</street>
            <city>Wonderland</city>
        </address>
        <hobbies>
            <hobby>reading</hobby>
            <hobby>biking</hobby>
        </hobbies>
    </person>`
    var person Person
    err := xml.Unmarshal([]byte(xmlData), &person)
    if err != nil {
        fmt.Println(err)
        return
    }
    fmt.Printf("%+v\n", person)
}

JSON vs. XML: Una Comparació

| Característica | JSON | XML | |-------------------|------------------------------|------------------------------| | Sintaxi | Lleugera, fàcil de llegir | Verbosa, basada en etiquetes | | Tipus de Dades | Suporta tipus de dades bàsics| Suporta tipus de dades complexos | | Llegibilitat | Llegible per humans | Llegible per humans però verbós | | Suport d'Esquemes | No té suport d'esquemes integrat | Suporta esquemes (XSD) | | Usos | APIs web, fitxers de configuració | Aplicacions empresarials, fitxers de configuració |

Conclusió

En aquesta secció, hem explorat com treballar amb JSON i XML en Go. Hem cobert la codificació i decodificació d'estructures de dades, la gestió d'estructures complexes, i hem comparat els dos formats. JSON és generalment preferit per a aplicacions web a causa de la seva simplicitat i facilitat d'ús, mentre que XML s'utilitza en escenaris que requereixen una representació de dades més complexa i validació d'esquemes. Entendre ambdós formats i com treballar amb ells en Go és essencial per a qualsevol programador que tracti amb l'intercanvi de dades.
