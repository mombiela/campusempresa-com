Introducció a les Interfícies

Les interfícies en Go proporcionen una manera d'especificar el comportament d'un objecte. Si un tipus implementa tots els mètodes declarats per una interfície, es diu que implementa aquesta interfície. Aquesta és una característica poderosa que permet un codi flexible i modular.

Conceptes Clau

  • Definició d'Interfície: Una interfície es defineix utilitzant la paraula clau type seguida del nom de la interfície i la paraula clau interface.
  • Conjunt de Mètodes: El conjunt de mètodes que un tipus ha d'implementar per satisfer una interfície.
  • Implementació Implícita: En Go, un tipus implementa una interfície de manera implícita. No cal declarar explícitament que un tipus implementa una interfície.

Definició i Implementació d'Interfícies

Definició d'una Interfície

Una interfície es defineix especificant un conjunt de signatures de mètodes. Aquí teniu un exemple de definició d'una interfície:

type Shape interface {
    Area() float64
    Perimeter() float64
}

Implementació d'una Interfície

Per implementar una interfície, un tipus ha de definir tots els mètodes declarats per la interfície. Aquí teniu un exemple amb un tipus Rectangle:

type Rectangle struct {
    Width, Height float64
}

func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Width + r.Height)
}

En aquest exemple, Rectangle implementa la interfície Shape perquè té els mètodes Area i Perimeter.

Ús de les Interfícies

Un cop un tipus implementa una interfície, es pot utilitzar allà on s'espera aquesta interfície. Aquí teniu un exemple:

func PrintShapeInfo(s Shape) {
    fmt.Printf("Area: %f\n", s.Area())
    fmt.Printf("Perimeter: %f\n", s.Perimeter())
}

func main() {
    rect := Rectangle{Width: 10, Height: 5}
    PrintShapeInfo(rect)
}

Exemples Pràctics

Implementacions Múltiples

Diferents tipus poden implementar la mateixa interfície. Aquí teniu un exemple amb un altre tipus, Circle:

type Circle struct {
    Radius float64
}

func (c Circle) Area() float64 {
    return math.Pi * c.Radius * c.Radius
}

func (c Circle) Perimeter() float64 {
    return 2 * math.Pi * c.Radius
}

Ara, tant Rectangle com Circle implementen la interfície Shape:

func main() {
    rect := Rectangle{Width: 10, Height: 5}
    circ := Circle{Radius: 7}
    
    PrintShapeInfo(rect)
    PrintShapeInfo(circ)
}

Interfície Buida

La interfície buida interface{} és un cas especial en Go. Pot contenir valors de qualsevol tipus perquè cada tipus implementa almenys zero mètodes.

func PrintAnything(a interface{}) {
    fmt.Println(a)
}

func main() {
    PrintAnything(42)
    PrintAnything("Hello, World!")
    PrintAnything(Rectangle{Width: 10, Height: 5})
}

Temes Avançats

Afirmacions de Tipus

Les afirmacions de tipus proporcionen accés al valor concret subjacent d'una interfície.

func main() {
    var i interface{} = "hello"
    
    s, ok := i.(string)
    if ok {
        fmt.Println(s)
    } else {
        fmt.Println("Not a string")
    }
}

Commutadors de Tipus

Els commutadors de tipus són una manera de manejar múltiples tipus que satisfan una interfície.

func doSomething(i interface{}) {
    switch v := i.(type) {
    case int:
        fmt.Println("Integer:", v)
    case string:
        fmt.Println("String:", v)
    default:
        fmt.Println("Unknown type")
    }
}

func main() {
    doSomething(10)
    doSomething("hello")
    doSomething(3.14)
}

Conclusió

Les interfícies en Go són una característica poderosa que permet un codi flexible i modular. Entenent com definir i implementar interfícies, pots crear components més abstractes i reutilitzables. Recorda que Go utilitza la implementació implícita, la qual cosa pot fer que el teu codi sigui més net i intuïtiu. Practica l'ús d'interfícies amb diferents tipus i explora característiques avançades com les afirmacions de tipus i els commutadors de tipus per convertir-te en un expert en l'ús d'interfícies en Go.
