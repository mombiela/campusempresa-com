Introducció als Structs

Els structs a Go són tipus de dades compostes que agrupen variables sota un sol nom. Aquestes variables, conegudes com a camps, poden ser de diferents tipus. Els structs són útils per crear estructures de dades més complexes.

Conceptes Clau

  • Definició: Un struct es defineix utilitzant les paraules clau type i struct.
  • Camps: Variables dins d'un struct, cadascuna amb un nom i un tipus.
  • Instanciació: Crear una instància d'un struct.
  • Accés als Camps: Utilitzar la notació de punt per accedir o modificar els camps.
  • Camps Anònims: Camps sense nom, permetent definicions més concises.
  • Structs Embeguts: Structs dins d'altres structs, permetent la composició.

Definició i Ús dels Structs

Definició Bàsica d'un Struct

Per definir un struct, utilitzes la paraula clau type seguida del nom del struct i la paraula clau struct.

type Person struct {
    Name string
    Age  int
}

Instanciació d'un Struct

Pots crear una instància d'un struct utilitzant el nom del struct seguit de claus.

func main() {
    var p1 Person
    p1.Name = "Alice"
    p1.Age = 30

    p2 := Person{Name: "Bob", Age: 25}
    p3 := Person{"Charlie", 35} // Els noms dels camps es poden ometre si els valors estan en ordre

    fmt.Println(p1)
    fmt.Println(p2)
    fmt.Println(p3)
}

Accés i Modificació dels Camps

Els camps d'un struct es poden accedir i modificar utilitzant la notació de punt.

func main() {
    p := Person{Name: "Alice", Age: 30}
    fmt.Println(p.Name) // Sortida: Alice
    p.Age = 31
    fmt.Println(p.Age)  // Sortida: 31
}

Característiques Avançades dels Structs

Camps Anònims

Els camps anònims permeten definir un struct sense nomenar explícitament els camps.

type Point struct {
    int
    int
}

func main() {
    p := Point{1, 2}
    fmt.Println(p.int) // Error: selector ambigu p.int
    fmt.Println(p.int1) // Sortida: 1
    fmt.Println(p.int2) // Sortida: 2
}

Structs Embeguts

Els structs es poden embegir dins d'altres structs, permetent la composició i la reutilització de camps.

type Address struct {
    City, State string
}

type Employee struct {
    Name    string
    Age     int
    Address // Struct embegut
}

func main() {
    e := Employee{
        Name: "John",
        Age:  28,
        Address: Address{
            City:  "New York",
            State: "NY",
        },
    }

    fmt.Println(e.Name)    // Sortida: John
    fmt.Println(e.City)    // Sortida: New York
    fmt.Println(e.Address) // Sortida: {New York NY}
}

Mètodes dels Structs

Els mètodes es poden associar amb structs, permetent definir comportaments per als teus tipus de dades.

type Rectangle struct {
    Width, Height int
}

// Mètode amb receptor de valor
func (r Rectangle) Area() int {
    return r.Width * r.Height
}

// Mètode amb receptor de punter
func (r *Rectangle) Scale(factor int) {
    r.Width *= factor
    r.Height *= factor
}

func main() {
    rect := Rectangle{Width: 10, Height: 5}
    fmt.Println(rect.Area()) // Sortida: 50

    rect.Scale(2)
    fmt.Println(rect.Area()) // Sortida: 200
}

Conclusió

Els structs a Go proporcionen una manera poderosa d'agrupar dades relacionades i definir estructures de dades complexes. Entenent com definir, instanciar i manipular structs, així com utilitzar característiques avançades com els camps anònims i els structs embeguts, pots escriure codi Go més organitzat i mantenible. Els mètodes dels structs milloren encara més les capacitats dels structs permetent definir comportaments associats amb els teus tipus de dades.
