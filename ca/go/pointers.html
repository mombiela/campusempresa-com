Introducció als Pointers

Els pointers són un concepte fonamental en molts llenguatges de programació, incloent Go. Et permeten accedir i manipular directament les adreces de memòria de les variables. Entendre els pointers és crucial per a una gestió eficient de la memòria i per a certes tècniques avançades de programació.

Què és un Pointer?

  • Un pointer és una variable que emmagatzema l'adreça de memòria d'una altra variable.
  • En Go, els pointers es representen utilitzant el símbol * (asterisc).

Per què Utilitzar Pointers?

  • Eficiència: Passar estructures grans o arrays per pointer pot ser més eficient que passar-los per valor.
  • Mutabilitat: Els pointers permeten que les funcions modifiquin la variable original.
  • Estructures de Dades Dinàmiques: Els pointers són essencials per crear llistes enllaçades, arbres i altres estructures de dades dinàmiques.

Declarar i Utilitzar Pointers

Declarar un Pointer

Per declarar un pointer, utilitzes el símbol * seguit del tipus de la variable a la qual apunta.

var ptr *int

Inicialitzar un Pointer

Pots inicialitzar un pointer assignant-li l'adreça d'una variable utilitzant l'operador & (adreça de).

var x int = 10
var ptr *int = &x

Desreferenciar un Pointer

Desreferenciar un pointer significa accedir al valor emmagatzemat a l'adreça de memòria a la qual apunta el pointer. Això es fa utilitzant l'operador *.

fmt.Println(*ptr) // Mostra: 10

Operacions amb Pointers

Aritmètica de Pointers

Go no suporta l'aritmètica de pointers (per exemple, incrementar l'adreça emmagatzemada en un pointer) directament, a diferència de llenguatges com C o C++.

Comparar Pointers

Pots comparar pointers utilitzant els operadors == i !=.

var a, b int
var ptr1, ptr2 *int = &a, &b

if ptr1 == ptr2 {
    fmt.Println("Els pointers són iguals")
} else {
    fmt.Println("Els pointers no són iguals")
}

Pointers i Funcions

Passar Pointers a Funcions

Passar un pointer a una funció permet que la funció modifiqui la variable original.

func increment(x *int) {
    *x++
}

func main() {
    var num int = 5
    increment(&num)
    fmt.Println(num) // Mostra: 6
}

Retornar Pointers des de Funcions

Una funció també pot retornar un pointer.

func newInt() *int {
    var x int = 100
    return &x
}

func main() {
    ptr := newInt()
    fmt.Println(*ptr) // Mostra: 100
}

Conceptes Avançats de Pointers

Pointers a Pointers

Go permet pointers a pointers, tot i que són menys comuns.

var x int = 10
var ptr *int = &x
var pptr **int = &ptr

fmt.Println(**pptr) // Mostra: 10

Structs i Pointers

Utilitzar pointers amb structs pot ser molt potent, especialment per a structs grans.

type Person struct {
    name string
    age  int
}

func changeName(p *Person, newName string) {
    p.name = newName
}

func main() {
    person := Person{name: "Alice", age: 30}
    changeName(&person, "Bob")
    fmt.Println(person.name) // Mostra: Bob
}

Pointers Nil

Un pointer que no està inicialitzat explícitament és nil.

var ptr *int
fmt.Println(ptr) // Mostra: <nil>

Resum

  • Els pointers emmagatzemen l'adreça de memòria de les variables.
  • Utilitza l'operador * per declarar i desreferenciar pointers.
  • L'operador & s'utilitza per obtenir l'adreça d'una variable.
  • Els pointers es poden passar a funcions i retornar-ne per permetre la modificació directa de les variables.
  • Els conceptes avançats inclouen pointers a pointers i l'ús de pointers amb structs.
  • Entendre els pointers és essencial per a una gestió eficient de la memòria i tècniques avançades de programació en Go.
