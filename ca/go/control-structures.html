El Projecte Sobre nosaltres Contribuir Donacions Llicència

Les estructures de control són fonamentals per a la programació, ja que et permeten dictar el flux del teu programa basant-te en condicions i bucles. En Go, les estructures de control són senzilles i fàcils d'entendre. Aquesta secció cobrirà les estructures de control essencials en Go, incloent-hi les instruccions if, els bucles for, les instruccions switch i les instruccions select.

Instruccions If

La instrucció if et permet executar un bloc de codi només si una condició especificada és certa.

Instrucció If bàsica

package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x és més gran que 5")
    }
}
  • Explicació: El codi comprova si x és més gran que 5. Si és cert, imprimeix "x és més gran que 5".

Instrucció If-Else

package main

import "fmt"

func main() {
    x := 3
    if x > 5 {
        fmt.Println("x és més gran que 5")
    } else {
        fmt.Println("x no és més gran que 5")
    }
}
  • Explicació: Si x no és més gran que 5, s'executa el bloc else, imprimint "x no és més gran que 5".

Instrucció If-Else If-Else

package main

import "fmt"

func main() {
    x := 7
    if x > 10 {
        fmt.Println("x és més gran que 10")
    } else if x > 5 {
        fmt.Println("x és més gran que 5 però menor o igual a 10")
    } else {
        fmt.Println("x és 5 o menys")
    }
}
  • Explicació: Aquesta estructura permet comprovar múltiples condicions en seqüència.

Bucles For

El bucle for és l'única construcció de bucle en Go, però és prou versàtil per manejar totes les necessitats de bucles.

Bucle For bàsic

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}
  • Explicació: Aquest bucle imprimeix números del 0 al 4.

Bucle For amb condició

package main

import "fmt"

func main() {
    x := 0
    for x < 5 {
        fmt.Println(x)
        x++
    }
}
  • Explicació: Aquest bucle continua mentre x sigui menor que 5.

Bucle infinit

package main

import "fmt"

func main() {
    for {
        fmt.Println("Bucle infinit")
        break // Per evitar un bucle infinit real
    }
}
  • Explicació: Es pot crear un bucle infinit utilitzant for sense cap condició. La instrucció break s'utilitza per sortir del bucle.

Instruccions Switch

La instrucció switch proporciona una manera d'executar un bloc de codi entre molts basant-se en el valor d'una variable.

Instrucció Switch bàsica

package main

import "fmt"

func main() {
    day := "Dimarts"
    switch day {
    case "Dilluns":
        fmt.Println("Inici de la setmana laboral")
    case "Dimarts":
        fmt.Println("Segon dia de la setmana laboral")
    default:
        fmt.Println("Un altre dia")
    }
}
  • Explicació: El codi imprimeix un missatge basat en el valor de day.

Switch amb múltiples casos

package main

import "fmt"

func main() {
    day := "Dissabte"
    switch day {
    case "Dissabte", "Diumenge":
        fmt.Println("Cap de setmana")
    default:
        fmt.Println("Dia laborable")
    }
}
  • Explicació: Es poden combinar múltiples casos per manejar el mateix bloc de codi.

Instruccions Select

La instrucció select s'utilitza amb canals per manejar múltiples operacions de canals.

Instrucció Select bàsica

package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string)
    ch2 := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        ch1 <- "un"
    }()

    go func() {
        time.Sleep(2 * time.Second)
        ch2 <- "dos"
    }()

    select {
    case msg1 := <-ch1:
        fmt.Println("Rebut", msg1)
    case msg2 := <-ch2:
        fmt.Println("Rebut", msg2)
    }
}
  • Explicació: La instrucció select espera que un dels canals enviï un missatge i després executa el cas corresponent.

Conclusió

Les estructures de control en Go són essencials per gestionar el flux del teu programa. Dominant les instruccions if, els bucles for, les instruccions switch i les instruccions select, podràs escriure codi més eficient i llegible. Practica utilitzant aquestes construccions en diversos escenaris per convertir-te en un expert en controlar el flux dels teus programes en Go.
