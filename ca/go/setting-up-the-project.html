Configuració del Projecte
Implementació de Funcionalitats ►

Configurar un projecte Go correctament és crucial per a un desenvolupament i manteniment eficients. Aquesta secció et guiarà a través del procés de configurar un projecte Go des de zero, assegurant que tinguis una base sòlida per al teu viatge de codificació.

Instal·lant Go

Abans de començar un projecte Go, necessites tenir Go instal·lat a la teva màquina.

  1. Descarrega Go: Visita el lloc web oficial de Go i descarrega l'instal·lador per al teu sistema operatiu.

  2. Instal·la Go: Segueix les instruccions d'instal·lació proporcionades per al teu sistema operatiu.

  3. Verifica la Instal·lació: Obre un terminal i executa la següent comanda per verificar que Go està instal·lat correctament:

    go version

    Hauries de veure una sortida similar a go version go1.17.1 darwin/amd64, indicant la versió de Go instal·lada.

Configurant el Teu Espai de Treball

Go utilitza una estructura específica d'espai de treball per gestionar el codi. L'espai de treball és una jerarquia de directoris amb tres directoris a la seva arrel:

  • src: Conté els fitxers de codi font de Go.
  • pkg: Conté els objectes de paquet.
  • bin: Conté les comandes executables.

Creant l'Espai de Treball

  1. Crea un Directori: Crea un directori per al teu espai de treball de Go. Per exemple:

    mkdir -p $HOME/go/{bin,pkg,src}

  2. Configura les Variables d'Entorn: Configura la variable d'entorn GOPATH per apuntar al teu directori d'espai de treball. Afegeix les següents línies al teu fitxer de configuració de la shell (per exemple, .bashrc, .zshrc):

    export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

  3. Recarrega la Configuració: Recarrega la configuració de la teva shell per aplicar els canvis:

    source ~/.bashrc

Creant un Nou Projecte

Ara que el teu espai de treball està configurat, pots crear un nou projecte Go.

  1. Navega a l'Espai de Treball: Canvia al directori src del teu espai de treball:

    cd $GOPATH/src

  2. Crea el Directori del Projecte: Crea un directori per al teu nou projecte. Per exemple, si el teu projecte es diu hello, executaries:

    mkdir -p hello
cd hello

  3. Inicialitza el Mòdul: Inicialitza un nou mòdul Go al directori del teu projecte. Aquest pas crea un fitxer go.mod, que fa el seguiment de les dependències del teu projecte:

    go mod init hello

    El fitxer go.mod tindrà un aspecte semblant a aquest:

    module hello

go 1.17

Escriu el Teu Primer Programa Go

Amb el teu projecte configurat, ara pots escriure el teu primer programa Go.

  1. Crea un Fitxer Principal: Crea un fitxer anomenat main.go al directori del teu projecte:

    touch main.go

  2. Escriu el Codi: Obre main.go al teu editor de text favorit i afegeix el següent codi:

    package main

import "fmt"

func main() {
    fmt.Println("Hola, Món!")
}

  3. Executa el Programa: Desa el fitxer i executa el teu programa utilitzant la comanda go run:

    go run main.go

    Hauries de veure la sortida:

    Hola, Món!

Resum

En aquesta secció, has après com configurar un projecte Go des de zero. Has instal·lat Go, configurat el teu espai de treball, creat un nou projecte i escrit el teu primer programa Go. Aquesta configuració fonamental et permetrà gestionar i desenvolupar eficientment els teus projectes Go a mesura que progressis en el curs.
