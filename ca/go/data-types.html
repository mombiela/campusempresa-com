Introducció

Els tipus de dades són fonamentals en qualsevol llenguatge de programació, i Go no és una excepció. Entendre els tipus de dades és crucial per escriure codi eficient i sense errors. En aquesta secció, cobrirem els tipus de dades bàsics i avançats en Go, el seu ús i com treballar amb ells.

Tipus de Dades Bàsics

Enters

Go suporta enters signats i no signats en diverses mides:

  • int i uint: Mida dependent de la plataforma (32 o 64 bits)
  • int8, int16, int32, int64: Enters signats de 8, 16, 32 i 64 bits
  • uint8, uint16, uint32, uint64: Enters no signats de 8, 16, 32 i 64 bits
package main

import "fmt"

func main() {
    var a int = 10
    var b int8 = -128
    var c uint16 = 65535

    fmt.Println(a, b, c)
}

Nombres de Punt Flotant

Go suporta nombres de punt flotant:

  • float32: Nombres de punt flotant de 32 bits
  • float64: Nombres de punt flotant de 64 bits
package main

import "fmt"

func main() {
    var a float32 = 3.14
    var b float64 = 2.718281828459045

    fmt.Println(a, b)
}

Booleà

El tipus bool representa un valor booleà, que pot ser true o false.

package main

import "fmt"

func main() {
    var isGoFun bool = true
    fmt.Println(isGoFun)
}

Cadenes

Les cadenes en Go són seqüències de bytes. Són immutables, és a dir, un cop creades, no es poden canviar.

package main

import "fmt"

func main() {
    var greeting string = "Hola, Go!"
    fmt.Println(greeting)
}

Tipus de Dades Compostos

Arrays

Els arrays són seqüències de mida fixa d'elements del mateix tipus.

package main

import "fmt"

func main() {
    var arr [5]int = [5]int{1, 2, 3, 4, 5}
    fmt.Println(arr)
}

Slices

Els slices són més flexibles que els arrays. Són de mida dinàmica i proporcionen operacions més potents i convenients.

package main

import "fmt"

func main() {
    var slice []int = []int{1, 2, 3, 4, 5}
    fmt.Println(slice)
}

Maps

Els maps són col·leccions de parelles clau-valor. Les claus han de ser úniques i d'un tipus comparable.

package main

import "fmt"

func main() {
    var m map[string]int = map[string]int{"one": 1, "two": 2}
    fmt.Println(m)
}

Structs

Els structs són col·leccions de camps. S'utilitzen per agrupar dades.

package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    var p Person = Person{Name: "Alice", Age: 30}
    fmt.Println(p)
}

Tipus de Dades Avançats

Pointers

Els pointers contenen l'adreça de memòria d'una variable. Són útils per a la manipulació eficient de dades i per evitar copiar estructures grans.

package main

import "fmt"

func main() {
    var a int = 10
    var p *int = &a

    fmt.Println("Valor de a:", a)
    fmt.Println("Adreça de a:", p)
    fmt.Println("Valor a l'adreça p:", *p)
}

Interfaces

Les interfaces defineixen un conjunt de signatures de mètodes. Qualsevol tipus que implementi aquests mètodes satisfà la interfície.

package main

import "fmt"

type Speaker interface {
    Speak() string
}

type Dog struct{}

func (d Dog) Speak() string {
    return "Woof!"
}

func main() {
    var s Speaker = Dog{}
    fmt.Println(s.Speak())
}

Conclusió

Entendre els tipus de dades en Go és essencial per escriure programes efectius. Hem cobert tipus de dades bàsics com enters, nombres de punt flotant, booleans i cadenes, així com tipus de dades compostos com arrays, slices, maps i structs. També hem tocat tipus de dades avançats com pointers i interfaces. Dominar aquests conceptes t'ajudarà a escriure codi Go més eficient i robust.
