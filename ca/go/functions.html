Introducció a les Funcions

Les funcions són blocs fonamentals en Go, que et permeten encapsular codi en unitats reutilitzables. Ajuden a organitzar el codi, millorar la llegibilitat i reduir la redundància.

Conceptes Clau

  • Definició: Una funció es defineix utilitzant la paraula clau func seguida del nom de la funció, paràmetres, tipus de retorn i el cos de la funció.
  • Paràmetres: Les funcions poden tenir zero o més paràmetres.
  • Valors de Retorn: Les funcions poden retornar zero o més valors.
  • Àmbit: Les variables declarades dins d'una funció són locals a aquesta funció.

Sintaxi Bàsica de les Funcions

Aquí tens la sintaxi bàsica per definir una funció en Go:

func functionName(parameter1 type1, parameter2 type2) returnType {
    // cos de la funció
    return value
}

Exemple: Funció Simple

Comencem amb una funció simple que suma dos enters:

package main

import "fmt"

// la funció add pren dos enters i retorna la seva suma
func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 4)
    fmt.Println("Sum:", result) // Sortida: Sum: 7
}

Valors de Retorn Múltiples

Les funcions en Go poden retornar múltiples valors, cosa que és especialment útil per al maneig d'errors.

package main

import "fmt"

// la funció divide retorna el quocient i un booleà que indica l'èxit
func divide(a int, b int) (int, bool) {
    if b == 0 {
        return 0, false
    }
    return a / b, true
}

func main() {
    quotient, success := divide(10, 2)
    if success {
        fmt.Println("Quotient:", quotient) // Sortida: Quotient: 5
    } else {
        fmt.Println("Division by zero")
    }
}

Conceptes Avançats de Funcions

Valors de Retorn Nomenats

Les funcions poden tenir valors de retorn nomenats, cosa que pot fer el codi més llegible.

package main

import "fmt"

// la funció swap intercanvia dos enters i els retorna
func swap(a, b int) (x int, y int) {
    x = b
    y = a
    return
}

func main() {
    a, b := swap(1, 2)
    fmt.Println("Swapped:", a, b) // Sortida: Swapped: 2 1
}

Funcions Variàdiques

Les funcions variàdiques poden prendre un nombre variable d'arguments.

package main

import "fmt"

// la funció sum pren un nombre variable d'enters i retorna la seva suma
func sum(numbers ...int) int {
    total := 0
    per _, number := range numbers {
        total += number
    }
    return total
}

func main() {
    fmt.Println("Sum:", sum(1, 2, 3, 4)) // Sortida: Sum: 10
}

Funcions Anònimes i Clausures

Go suporta funcions anònimes, que poden ser útils per a operacions de curta durada. Les clausures són funcions anònimes que capturen variables del seu àmbit circumdant.

package main

import "fmt"

func main() {
    // Funció anònima
    func(message string) {
        fmt.Println(message)
    }("Hello, Go!") // Sortida: Hello, Go!

    // Exemple de clausura
    counter := func() func() int {
        count := 0
        return func() int {
            count++
            return count
        }
    }()

    fmt.Println(counter()) // Sortida: 1
    fmt.Println(counter()) // Sortida: 2
}

Funcions d'Ordre Superior

Les funcions poden prendre altres funcions com a arguments o retornar-les.

package main

import "fmt"

// la funció apply pren una funció i dos enters, i aplica la funció als enters
func apply(op func(int, int) int, a int, b int) int {
    return op(a, b)
}

func main() {
    add := func(a int, b int) int {
        return a + b
    }

    result := apply(add, 3, 4)
    fmt.Println("Result:", result) // Sortida: Result: 7
}

Resum

Les funcions en Go són versàtils i poderoses, permetent un codi net, reutilitzable i mantenible. Des de definicions bàsiques de funcions fins a conceptes avançats com clausures i funcions d'ordre superior, dominar les funcions és essencial per a qualsevol programador de Go. Practica escrivint diferents tipus de funcions i utilitzant-les en diversos escenaris per aprofundir en la teva comprensió.
