Desplegar una aplicació Go implica diversos passos, des de construir l'executable fins a configurar l'entorn on l'aplicació s'executarà. Aquesta secció et guiarà a través del procés de desplegar una aplicació Go, cobrint tant estratègies de desplegament bàsiques com avançades.

Construint l'Aplicació Go

Abans de desplegar, necessites construir la teva aplicació Go en un binari executable.

  • Construint per al sistema actual:

    go build -o myapp

    Aquesta comanda compila el codi font de Go i produeix un executable anomenat myapp per al sistema operatiu i arquitectura actuals.

  • Compilant per a diferents sistemes:

    GOOS=linux GOARCH=amd64 go build -o myapp-linux

    Aquesta comanda estableix el sistema operatiu objectiu a Linux i l'arquitectura a AMD64, produint un executable anomenat myapp-linux.

Preparant l'Entorn

Assegura't que l'entorn objectiu està llest per executar la teva aplicació Go.

  • Configurant el servidor:

    • Tria un servidor (per exemple, AWS, DigitalOcean, Google Cloud).
    • Instal·la les dependències necessàries (per exemple, Docker, sistemes de bases de dades).

  • Variables d'entorn:

    • Emmagatzema informació sensible com claus d'API i credencials de bases de dades en variables d'entorn.
    • Exemple en un fitxer .env: 
      DB_HOST=localhost
DB_USER=root
DB_PASS=password

Desplegant amb Docker

Docker simplifica el desplegament empaquetant la teva aplicació i les seves dependències en un contenidor.

  • Creant un Dockerfile:

    # Utilitza la imatge oficial de Golang com a imatge base
FROM golang:1.19-alpine

# Estableix el Directori de Treball Actual dins del contenidor
WORKDIR /app

# Copia els fitxers go mod i sum
COPY go.mod go.sum ./

# Descarrega totes les dependències. Les dependències es cachejaran si els fitxers go.mod i go.sum no canvien
RUN go mod download

# Copia el codi font des del directori actual al Directori de Treball dins del contenidor
COPY . .

# Construeix l'aplicació Go
RUN go build -o myapp

# Comanda per executar l'executable
CMD ["./myapp"]

  • Construint i executant el contenidor Docker:

    docker build -t myapp .
docker run -p 8080:8080 myapp

Desplegant a un Servei al Núvol

Desplegant a AWS Elastic Beanstalk

  • Crea una aplicació Elastic Beanstalk:

    • Utilitza la Consola de Gestió d'AWS per crear una nova aplicació Elastic Beanstalk.
    • Tria una plataforma (per exemple, Docker si utilitzes un contenidor Docker).

  • Desplega l'aplicació:

    • Empaqueta el codi de la teva aplicació i els fitxers de configuració.
    • Utilitza el CLI d'Elastic Beanstalk per desplegar: 
      eb init -p docker myapp
eb create myapp-env
eb deploy

Desplegant a Google Cloud Run

  • Containeritza l'aplicació:

    • Assegura't que tens un Dockerfile com es mostra anteriorment.

  • Desplega utilitzant el CLI de gcloud:

    gcloud builds submit --tag gcr.io/PROJECT-ID/myapp
gcloud run deploy --image gcr.io/PROJECT-ID/myapp --platform managed

Conclusió

Desplegar una aplicació Go implica construir l'executable, preparar l'entorn i triar una estratègia de desplegament. Tant si utilitzes Docker, AWS Elastic Beanstalk, com Google Cloud Run, la clau és assegurar-te que la teva aplicació està empaquetada correctament i que l'entorn objectiu està configurat adequadament. Seguint aquests passos, pots desplegar les teves aplicacions Go de manera eficient i fiable.
