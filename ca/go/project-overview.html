Introducció

En aquest curs, et guiarem a través del procés d'aprenentatge de Go (Golang) des d'un nivell principiant fins a un nivell avançat. Go és un llenguatge de programació compilat i tipat estàticament dissenyat a Google. És conegut per la seva simplicitat, eficiència i fort suport a la concurrència. Aquest curs està estructurat per proporcionar una comprensió completa de Go, començant pels conceptes bàsics i avançant gradualment cap a temes més complexos.

Conceptes Clau

  • Conceptes Bàsics de Go: Comprendre la sintaxi, els tipus de dades i els conceptes bàsics.
  • Estructures de Control: Aprendre sobre bucles, condicionals i maneig d'errors.
  • Funcions i Mètodes: Escriure codi reutilitzable amb funcions i mètodes.
  • Estructures de Dades: Treballar amb arrays, slices, mapes i structs.
  • Concurrència: Utilitzar el potent model de concurrència de Go amb goroutines i canals.
  • Llibreria Estàndard: Explorar l'extensa llibreria estàndard de Go.
  • Proves i Depuració: Escriure proves i depurar programes Go.
  • Temes Avançats: Endinsar-se en conceptes avançats com la reflexió, les interfícies i la gestió de paquets.

Estructura del Curs

Conceptes Bàsics de Go

Començarem amb els blocs fonamentals de Go, incloent:

  • Variables i Constants: Com declarar-les i utilitzar-les.
  • Tipus de Dades Bàsics: Comprendre enters, floats, cadenes de text i booleans.
  • Sintaxi Bàsica: Escriure el teu primer programa en Go.
package main

import "fmt"

func main() {
    fmt.Println("Hola, Món!")
}

Estructures de Control

A continuació, cobrirem les estructures de control que et permeten gestionar el flux dels teus programes:

  • If Statements: Execució condicional.
  • For Loops: Construccions d'iteració.
  • Switch Statements: Ramificació múltiple.
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        if i%2 == 0 {
            fmt.Println(i, "és parell")
        } else {
            fmt.Println(i, "és senar")
        }
    }
}

Funcions i Mètodes

Explorarem com escriure codi reutilitzable utilitzant funcions i mètodes:

  • Declaració de Funcions: Sintaxi i ús.
  • Mètodes: Associar funcions amb tipus.
  • Funcions Anònimes: Utilitzar funcions com a valors.
package main

import "fmt"

func sumar(a int, b int) int {
    return a + b
}

func main() {
    resultat := sumar(3, 4)
    fmt.Println("3 + 4 =", resultat)
}

Estructures de Dades

Comprendre i utilitzar les estructures de dades de Go:

  • Arrays i Slices: Arrays de mida fixa i dinàmica.
  • Mapes: Parells clau-valor.
  • Structs: Tipus de dades personalitzats.
package main

import "fmt"

type Persona struct {
    Nom string
    Edat int
}

func main() {
    persona := Persona{Nom: "Alice", Edat: 30}
    fmt.Println(persona)
}

Concurrència

Aprofitar el model de concurrència de Go:

  • Goroutines: Fils lleugers.
  • Canals: Comunicació entre goroutines.
package main

import (
    "fmt"
    "time"
)

func dirHola() {
    fmt.Println("Hola")
}

func main() {
    go dirHola()
    time.Sleep(1 * segon)
    fmt.Println("Món")
}

Llibreria Estàndard

Explorar la llibreria estàndard de Go:

  • fmt: Format i impressió.
  • io: Entrada i sortida.
  • net/http: Construir servidors web.

Proves i Depuració

Assegurar que el teu codi funcioni correctament:

  • Escriure Proves: Utilitzant el paquet testing.
  • Depuració: Eines i tècniques per depurar programes Go.

Temes Avançats

Endinsar-se en àrees més complexes:

  • Reflexió: Inspeccionar tipus en temps d'execució.
  • Interfícies: Definir i utilitzar interfícies.
  • Gestió de Paquets: Gestionar dependències amb mòduls Go.

Conclusió

Al final d'aquest curs, tindràs una comprensió sòlida de Go i seràs capaç de construir aplicacions eficients i concurrents. També estaràs familiaritzat amb la llibreria estàndard de Go i les seves característiques avançades, cosa que et permetrà abordar projectes més complexos i contribuir a la comunitat de Go.
