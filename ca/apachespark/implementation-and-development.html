Introducció a Apache Spark

Apache Spark és un sistema de computació distribuïda de codi obert que proporciona una interfície per programar clústers sencers amb paral·lelisme de dades implícit i tolerància a fallades. Aquesta secció cobrirà els conceptes bàsics d'Apache Spark, la seva arquitectura i com s'integra en l'ecosistema de big data.

Abans d'endinsar-nos en la implementació i el desenvolupament, és crucial configurar correctament el vostre entorn.

  • Instal·lació en Clúster: Passos per configurar Apache Spark en un clúster.

Conceptes Bàsics de Spark

Entendre els conceptes bàsics de Spark és essencial per a una implementació i desenvolupament efectius.

Resilient Distributed Datasets (RDDs)

  • Definició: Col·leccions distribuïdes immutables d'objectes.
  • Operacions: Transformacions (per exemple, map, filter) i accions (per exemple, collect, count).
from pyspark import SparkContext

sc = SparkContext("local", "Simple App")
data = [1, 2, 3, 4, 5]
rdd = sc.parallelize(data)
squared = rdd.map(lambda x: x * x).collect()
print(squared)

DataFrames i Datasets

  • DataFrames: Col·leccions distribuïdes de dades organitzades en columnes amb nom.
  • Datasets: Col·leccions fortament tipades i immutables d'objectes.
from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("SimpleApp").getOrCreate()
df = spark.read.json("path/to/json/file")
df.show()

Desenvolupament amb Apache Spark

Aquesta secció cobreix el desenvolupament d'aplicacions Spark, incloent-hi l'escriptura, proves i desplegament de treballs Spark.

Escriptura d'Aplicacions Spark

  • SparkContext i SparkSession: Creació i ús d'aquests objectes.
  • Operacions Bàsiques: Exemples de transformacions i accions bàsiques.
from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("SimpleApp").getOrCreate()
data = [("Alice", 1), ("Bob", 2), ("Cathy", 3)]
df = spark.createDataFrame(data, ["Name", "Value"])
df.show()

Operacions Avançades

  • Unions: Realització d'unions en DataFrames.
  • Agregacions: Ús de groupBy i funcions d'agregació.
# Exemple d'una operació d'unió
df1 = spark.createDataFrame([("Alice", 1), ("Bob", 2)], ["Name", "Value"])
df2 = spark.createDataFrame([("Alice", "A"), ("Bob", "B")], ["Name", "Grade"])
joined_df = df1.join(df2, "Name")
joined_df.show()

Optimitzacions

  • Caché i Persistència: Tècniques per optimitzar els treballs Spark.
  • Variables de Difusió i Acumuladors: Ús d'aquests per a millores de rendiment.
# Exemple de caché
df.cache()
df.count()  # Això emmagatzemarà en memòria el DataFrame
