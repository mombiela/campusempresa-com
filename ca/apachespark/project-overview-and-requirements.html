Introducció

En aquesta secció, proporcionarem una visió general del projecte en què treballaràs al llarg d'aquest curs d'Apache Spark. Aquest projecte t'ajudarà a aplicar els conceptes i tècniques que aprens d'una manera pràctica i pràctica. També esbossarem els requisits i prerequisits necessaris per completar amb èxit el projecte.

Visió General del Projecte

El projecte d'aquest curs implica la construcció d'una pipeline de processament de dades utilitzant Apache Spark. La pipeline ingerirà, processarà i analitzarà un gran conjunt de dades per extreure informació significativa. Al final del projecte, tindràs una comprensió sòlida de com utilitzar Spark per al processament i l'anàlisi de grans volums de dades.

Objectius Clau

  • Ingestió de Dades: Carregar dades de diverses fonts a Spark.
  • Processament de Dades: Netejar, transformar i agregar dades utilitzant transformacions i accions de Spark.
  • Anàlisi de Dades: Realitzar una anàlisi exploratòria de dades (EDA) i generar informació.
  • Optimització: Optimitzar els treballs de Spark per obtenir un millor rendiment.
  • Visualització: Visualitzar els resultats utilitzant eines adequades.

Requisits

Prerequisits

Abans de començar el projecte, assegura't de tenir els següents prerequisits:

  • Coneixements Bàsics de Programació: Familiaritat amb Python o Scala.
  • Comprensió dels Conceptes de Big Data: Coneixements bàsics de tecnologies i conceptes de big data.
  • Entorn de Desenvolupament: Una instal·lació funcional d'Apache Spark i un IDE compatible (per exemple, IntelliJ IDEA, PyCharm).

Programari i Eines

  • Apache Spark: El motor principal per al processament de dades a gran escala.
  • Hadoop (Opcional): Per a HDFS si planejes utilitzar-lo com a capa d'emmagatzematge.
  • Jupyter Notebook: Per a l'anàlisi interactiva de dades i visualització.
  • Python/Scala: Llenguatges de programació per escriure aplicacions Spark.
  • Eines de Visualització: Biblioteques com Matplotlib, Seaborn o Plotly per a la visualització de dades.

Conjunt de Dades

Per a aquest projecte, utilitzarem un conjunt de dades disponible públicament. Pots triar entre diversos conjunts de dades disponibles en plataformes com Kaggle, UCI Machine Learning Repository o data.gov. Assegura't que el conjunt de dades sigui prou gran per demostrar les capacitats de Spark però manejable dins l'abast d'aquest curs.

Configuració de l'Entorn

Per configurar el teu entorn de desenvolupament, segueix aquests passos:

  1. Instal·lar Apache Spark:

    # Per a MacOS utilitzant Homebrew
brew install apache-spark

# Per a Ubuntu
sudo apt-get update
sudo apt-get install -y spark

  2. Instal·lar Jupyter Notebook:

    pip install notebook

  3. Instal·lar Biblioteques Requerides:

    pip install pyspark matplotlib seaborn

  4. Verificar la Instal·lació:

    import pyspark
print(pyspark.__version__)

Conclusió

En aquesta secció, hem esbossat el projecte en què treballaràs al llarg d'aquest curs d'Apache Spark. Hem cobert els objectius clau, prerequisits, programari i eines necessàries, selecció del conjunt de dades i configuració de l'entorn. Seguint aquestes directrius, estaràs ben preparat per començar a construir la teva pipeline de processament de dades amb Apache Spark. A la següent secció, ens endinsarem en els conceptes bàsics d'Apache Spark i els seus components principals.
