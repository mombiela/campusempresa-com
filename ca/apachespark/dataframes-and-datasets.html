Introducció als DataFrames i Datasets

Apache Spark és una eina poderosa per al processament de grans volums de dades, i entendre els DataFrames i Datasets és crucial per aprofitar tot el seu potencial. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats dels DataFrames i Datasets en Spark.

Què són els DataFrames?

  • DataFrame: Una col·lecció distribuïda de dades organitzada en columnes amb nom, similar a una taula en una base de dades relacional o un data frame en R/Python.
  • Esquema: L'estructura d'un DataFrame, que defineix els noms de les columnes i els tipus de dades.

Creació de DataFrames

Pots crear DataFrames a partir de diverses fonts de dades com ara fitxers CSV, JSON, Parquet, o RDDs existents.

from pyspark.sql import SparkSession

# Inicialitzar la sessió de Spark
spark = SparkSession.builder.appName("DataFrameExample").getOrCreate()

# Crear un DataFrame a partir d'un fitxer JSON
df = spark.read.json("path/to/json/file.json")

# Mostrar el DataFrame
df.show()

Operacions bàsiques amb DataFrames

  • Show: Mostrar el contingut del DataFrame.
  • Select: Seleccionar columnes específiques.
  • Filter: Filtrar files basades en una condició.
  • GroupBy: Agrupar dades i realitzar agregacions.
# Seleccionar columnes específiques
df.select("column1", "column2").show()

# Filtrar files
df.filter(df["column1"] > 10).show()

# Agrupar per i agregar
df.groupBy("column1").count().show()

Què són els Datasets?

  • Dataset: Una col·lecció distribuïda de dades fortament tipada. Els Datasets proporcionen els beneficis dels RDDs (tipatge fort i programació funcional) juntament amb el motor d'execució optimitzat dels DataFrames.

Creació de Datasets

Els Datasets estan disponibles en Scala i Java però no en Python. A continuació es mostra un exemple en Scala:

import org.apache.spark.sql.SparkSession

// Inicialitzar la sessió de Spark
val spark = SparkSession.builder.appName("DatasetExample").getOrCreate()

// Importar implicits per convertir RDDs a Datasets
import spark.implicits._

// Crear Dataset a partir d'una classe de cas
case class Person(name: String, age: Int)
val peopleDS = Seq(Person("Alice", 29), Person("Bob", 35)).toDS()

// Mostrar el Dataset
peopleDS.show()

Operacions bàsiques amb Datasets

  • Map: Aplicar una funció a cada element.
  • Filter: Filtrar elements basats en una condició.
  • GroupByKey: Agrupar dades i realitzar agregacions.
// Operació Map
val namesDS = peopleDS.map(person => person.name)

// Operació Filter
val adultsDS = peopleDS.filter(person => person.age > 18)

// GroupByKey i agregació
val ageGroupDS = peopleDS.groupByKey(_.age).count()

Comparació entre DataFrames i Datasets

| Característica | DataFrame | Dataset | |---------------------|------------------------------------|-----------------------------------| | Suport de llenguatge| Python, Scala, Java, R | Scala, Java | | Tipatge fort | No | Sí | | Seguretat en temps de compilació | No | Sí | | Rendiment | Motor d'execució optimitzat | Motor d'execució optimitzat | | Facilitat d'ús | Més fàcil per a la manipulació de dades | Més fàcil per a la programació funcional |

Operacions avançades

Joins

Unir DataFrames o Datasets és una operació comuna en el processament de dades.

# Unir dos DataFrames
df1 = spark.read.json("path/to/json1.json")
df2 = spark.read.json("path/to/json2.json")

joined_df = df1.join(df2, df1["id"] == df2["id"])
joined_df.show()

UDFs (Funcions Definides per l'Usuari)

Pots definir funcions personalitzades per aplicar a DataFrames o Datasets.

from pyspark.sql.functions import udf
from pyspark.sql.types import StringType

# Definir una UDF
def upper_case(s):
    return s.upper()

# Registrar la UDF
upper_case_udf = udf(upper_case, StringType())

# Aplicar la UDF
df.withColumn("upper_name", upper_case_udf(df["name"])).show()

Conclusió

Els DataFrames i Datasets són components essencials d'Apache Spark, proporcionant abstraccions poderoses per a la manipulació i processament de dades. Els DataFrames ofereixen facilitat d'ús i rendiment optimitzat, mentre que els Datasets proporcionen seguretat de tipus i capacitats de programació funcional. Entendre aquests conceptes et permetrà escriure aplicacions Spark eficients i robustes.
