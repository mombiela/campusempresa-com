Introducció

La recollida i preparació de dades són passos crítics en qualsevol pipeline de processament de dades. En Apache Spark, aquests passos impliquen recollir dades de diverses fonts, netejar-les i transformar-les en un format adequat per a l'anàlisi. Aquesta secció et guiarà a través dels conceptes bàsics de la recollida i preparació de dades utilitzant Spark, progressant des de nivells principiants fins a avançats.

Recollida de Dades

Llegir Dades de Diverses Fonts

Apache Spark suporta la lectura de dades de múltiples fonts, incloent fitxers locals, HDFS, S3 i diverses bases de dades.

Llegir de Fitxers Locals

Per llegir dades d'un fitxer local, pots utilitzar el mètode spark.read. Aquí tens un exemple de lectura d'un fitxer CSV:

from pyspark.sql import SparkSession

# Inicialitzar la sessió de Spark
spark = SparkSession.builder.appName("DataCollection").getOrCreate()

# Llegir fitxer CSV
df = spark.read.csv("path/to/localfile.csv", header=True, inferSchema=True)

# Mostrar les primeres files
df.show()

Llegir de HDFS

Llegir de HDFS és similar a llegir de fitxers locals, però has d'especificar el camí de HDFS:

df = spark.read.csv("hdfs://namenode:9000/path/to/hdfsfile.csv", header=True, inferSchema=True)
df.show()

Llegir de S3

Per llegir de S3, has de configurar la teva sessió de Spark amb les credencials d'AWS:

spark._jsc.hadoopConfiguration().set("fs.s3a.access.key", "YOUR_ACCESS_KEY")
spark._jsc.hadoopConfiguration().set("fs.s3a.secret.key", "YOUR_SECRET_KEY")

df = spark.read.csv("s3a://bucketname/path/to/s3file.csv", header=True, inferSchema=True)
df.show()

Llegir de Bases de Dades

Spark també pot llegir dades de bases de dades relacionals utilitzant JDBC:

jdbc_url = "jdbc:mysql://hostname:port/dbname"
properties = {
    "user": "username",
    "password": "password"
}

df = spark.read.jdbc(url=jdbc_url, table="tablename", properties=properties)
df.show()

Neteja de Dades

Gestionar Valors Perduts

Els valors perduts són comuns en dades del món real. Spark proporciona diversos mètodes per gestionar-los:

  • Eliminar valors perduts:
df_cleaned = df.na.drop()
df_cleaned.show()
  • Omplir valors perduts:
df_filled = df.na.fill({"column1": 0, "column2": "unknown"})
df_filled.show()

Eliminar Duplicats

Els duplicats poden distorsionar la teva anàlisi. Pots eliminar-los utilitzant el mètode dropDuplicates:

df_unique = df.dropDuplicates()
df_unique.show()

Transformació de Dades

Operacions amb Columnes

Pots realitzar diverses operacions en columnes de DataFrame, com ara canviar el nom, afegir i eliminar columnes.

  • Canviar el nom de columnes:
df_renamed = df.withColumnRenamed("oldName", "newName")
df_renamed.show()
  • Afegir noves columnes:
from pyspark.sql.functions import col

df_newcol = df.withColumn("newColumn", col("existingColumn") * 2)
df_newcol.show()
  • Eliminar columnes:
df_dropped = df.drop("columnToDrop")
df_dropped.show()

Filtrar Dades

Filtrar et permet seleccionar files basades en condicions específiques:

df_filtered = df.filter(df["column"] > 10)
df_filtered.show()

Agregacions

Les agregacions s'utilitzen per calcular estadístiques resum:

df_grouped = df.groupBy("column").count()
df_grouped.show()

Preparació Avançada de Dades

Funcions Definides per l'Usuari (UDFs)

Les UDFs et permeten aplicar transformacions personalitzades a columnes de DataFrame:

from pyspark.sql.functions import udf
from pyspark.sql.types import IntegerType

def multiply_by_two(x):
    return x * 2

multiply_by_two_udf = udf(multiply_by_two, IntegerType())

df_udf = df.withColumn("newColumn", multiply_by_two_udf(df["existingColumn"]))
df_udf.show()

Treballar amb Tipus de Dades Complexes

Spark suporta tipus de dades complexos com arrays i structs. Pots manipular-los utilitzant funcions integrades:

from pyspark.sql.functions import explode

# Explosió d'una columna array
df_exploded = df.withColumn("explodedColumn", explode(df["arrayColumn"]))
df_exploded.show()

Conclusió

La recollida i preparació de dades són passos fonamentals en qualsevol flux de treball de processament de dades. Apache Spark proporciona eines robustes per llegir dades de diverses fonts, netejar-les i transformar-les en un format adequat per a l'anàlisi. En dominar aquestes tècniques, pots assegurar-te que les teves dades estan preparades per a les següents etapes del teu pipeline de dades, ja sigui que impliquin aprenentatge automàtic, informes o anàlisis addicionals.
