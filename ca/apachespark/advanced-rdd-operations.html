Introducció

En aquesta secció, ens endinsarem en operacions avançades sobre conjunts de dades distribuïts resilients (RDD) a Apache Spark. Aquestes operacions són essencials per realitzar transformacions i accions complexes de dades de manera eficient. Cobriren conceptes clau, proporcionarem exemples de codi i inclourem exercicis per consolidar la teva comprensió.

Conceptes clau

  • Transformacions vs Accions: Comprendre la diferència entre aquests dos tipus d'operacions.
  • RDD de parells: RDD especials que consisteixen en parells clau-valor, permetent operacions més complexes.
  • Transformacions avançades: Operacions com aggregateByKey, combineByKey i cogroup.
  • Accions avançades: Operacions com countByKey, lookup i foreach.

Transformacions vs Accions

Les transformacions creen un nou RDD a partir d'un existent, mentre que les accions desencadenen l'execució de les transformacions per retornar un resultat o escriure dades a un sistema d'emmagatzematge extern.

Exemple:

val rdd = sc.parallelize(Seq(1, 2, 3, 4, 5))

// Transformació
val squaredRdd = rdd.map(x => x * x)

// Acció
val result = squaredRdd.collect()
println(result.mkString(", ")) // Sortida: 1, 4, 9, 16, 25

RDD de parells

Els RDD de parells són RDD de parells clau-valor, que són útils per agregar dades, realitzar unions i altres operacions.

Exemple:

val pairRdd = sc.parallelize(Seq(("a", 1), ("b", 2), ("a", 3)))

// Transformació
val groupedRdd = pairRdd.groupByKey()

// Acció
groupedRdd.collect().foreach(println)
// Sortida: (a,CompactBuffer(1, 3))
//         (b,CompactBuffer(2))

Transformacions avançades

`aggregateByKey`

Agrupa els valors de cada clau utilitzant funcions de combinació donades i un "valor zero" neutral.

Exemple:

val pairRdd = sc.parallelize(Seq(("a", 1), ("b", 2), ("a", 3), ("b", 4)))

// El valor zero és 0, seqOp suma valors dins d'una partició, combOp suma valors entre particions
val aggregatedRdd = pairRdd.aggregateByKey(0)(_ + _, _ + _)

// Acció
aggregatedRdd.collect().foreach(println)
// Sortida: (a, 4)
//         (b, 6)

`combineByKey`

Funció d'agregació generalitzada que permet diferents lògiques d'agregació per combinar valors dins d'una partició i entre particions.

Exemple:

val pairRdd = sc.parallelize(Seq(("a", 1), ("b", 2), ("a", 3), ("b", 4)))

// Crear combinador, fusionar valor dins de la partició, fusionar valor entre particions
val combinedRdd = pairRdd.combineByKey(
  (v: Int) => v,
  (c: Int, v: Int) => c + v,
  (c1: Int, c2: Int) => c1 + c2
)

// Acció
combinedRdd.collect().foreach(println)
// Sortida: (a, 4)
//         (b, 6)

`cogroup`

Agrupa dades de dos RDD per clau.

Exemple:

val rdd1 = sc.parallelize(Seq(("a", 1), ("b", 2)))
val rdd2 = sc.parallelize(Seq(("a", 3), ("b", 4), ("a", 5)))

// Transformació
val cogroupedRdd = rdd1.cogroup(rdd2)

// Acció
cogroupedRdd.collect().foreach(println)
// Sortida: (a,(CompactBuffer(1),CompactBuffer(3, 5)))
//         (b,(CompactBuffer(2),CompactBuffer(4)))

Accions avançades

`countByKey`

Compta el nombre d'elements per a cada clau.

Exemple:

val pairRdd = sc.parallelize(Seq(("a", 1), ("b", 2), ("a", 3), ("b", 4)))

// Acció
val countByKeyResult = pairRdd.countByKey()
println(countByKeyResult)
// Sortida: Map(a -> 2, b -> 2)

`lookup`

Retorna tots els valors associats amb una clau donada.

Exemple:

val pairRdd = sc.parallelize(Seq(("a", 1), ("b", 2), ("a", 3), ("b", 4)))

// Acció
val lookupResult = pairRdd.lookup("a")
println(lookupResult.mkString(", ")) // Sortida: 1, 3

`foreach`

Aplica una funció a cada element de l'RDD.

Exemple:

val rdd = sc.parallelize(Seq(1, 2, 3, 4, 5))

// Acció
rdd.foreach(x => println(x * x))
// Sortida: 1
//         4
//         9
//         16
//         25
