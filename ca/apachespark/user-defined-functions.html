Les Funcions Definides per l'Usuari (UDFs) et permeten crear funcions personalitzades per ampliar la funcionalitat de Spark SQL i les operacions de DataFrame. Aquest tema et guiarà des dels conceptes bàsics de les UDFs fins a l'ús avançat, assegurant que entenguis com crear, registrar i utilitzar UDFs de manera efectiva en les teves aplicacions Spark.

Introducció a les UDFs

  • Definició: Les UDFs són funcions personalitzades definides per l'usuari per realitzar operacions específiques que no estan disponibles en les funcions integrades de Spark.
  • Propòsit: S'utilitzen per aplicar transformacions personalitzades a les columnes de DataFrame.

Creació i Registre de UDFs

Creació d'una UDF Simple

Per crear una UDF, defineixes una funció en Python, Scala o Java i després la registres amb Spark.

Exemple en Python

from pyspark.sql import SparkSession
from pyspark.sql.functions import udf
from pyspark.sql.types import StringType

# Inicialitza la sessió de Spark
spark = SparkSession.builder.appName("UDF Example").getOrCreate()

# Defineix una funció simple en Python
def to_uppercase(s):
    return s.upper()

# Registra la funció com a UDF
uppercase_udf = udf(to_uppercase, StringType())

# Crea un DataFrame de mostra
data = [("alice",), ("bob",), ("charlie",)]
df = spark.createDataFrame(data, ["name"])

# Utilitza la UDF en una operació de DataFrame
df.withColumn("name_upper", uppercase_udf(df["name"])).show()

Exemple en Scala

import org.apache.spark.sql.SparkSession
import org.apache.spark.sql.functions.udf

// Inicialitza la sessió de Spark
val spark = SparkSession.builder.appName("UDF Example").getOrCreate()

// Defineix una funció simple en Scala
def toUppercase(s: String): String = s.toUpperCase

// Registra la funció com a UDF
val uppercaseUDF = udf(toUppercase _)

// Crea un DataFrame de mostra
val data = Seq(("alice"), ("bob"), ("charlie"))
val df = spark.createDataFrame(data).toDF("name")

// Utilitza la UDF en una operació de DataFrame
df.withColumn("name_upper", uppercaseUDF(df("name"))).show()

Registre de UDFs amb Spark SQL

També pots registrar UDFs per utilitzar-les en consultes SQL de Spark.

Exemple en Python

# Registra la UDF amb Spark SQL
spark.udf.register("to_uppercase_sql", to_uppercase, StringType())

# Utilitza la UDF en una consulta SQL
df.createOrReplaceTempView("people")
spark.sql("SELECT name, to_uppercase_sql(name) as name_upper FROM people").show()

Exemple en Scala

// Registra la UDF amb Spark SQL
spark.udf.register("toUppercaseSQL", toUppercase _)

// Utilitza la UDF en una consulta SQL
df.createOrReplaceTempView("people")
spark.sql("SELECT name, toUppercaseSQL(name) as name_upper FROM people").show()

Ús Avançat de UDFs

Gestió de Valors Null

Les UDFs necessiten gestionar explícitament els valors nulls per evitar errors en temps d'execució.

Exemple en Python

def safe_to_uppercase(s):
    return s.upper() if s is not None else None

safe_uppercase_udf = udf(safe_to_uppercase, StringType())
df.withColumn("name_upper", safe_uppercase_udf(df["name"])).show()

Consideracions de Rendiment

  • Sobrecàrrega de Serialització: Les UDFs poden introduir sobrecàrrega de serialització. Utilitza funcions integrades sempre que sigui possible.
  • UDFs Vectoritzades: En PySpark, utilitza pandas UDFs per obtenir un millor rendiment amb operacions vectoritzades.

Exemple en Python amb Pandas UDF

from pyspark.sql.functions import pandas_udf, PandasUDFType

@pandas_udf(StringType(), PandasUDFType.SCALAR)
def pandas_to_uppercase(s):
    return s.str.upper()

df.withColumn("name_upper", pandas_to_uppercase(df["name"])).show()

Resum

  • Les UDFs són funcions personalitzades que amplien les capacitats de Spark SQL i les operacions de DataFrame.
  • Es poden crear i registrar en Python, Scala o Java.
  • Gestiona explícitament els valors nulls en les UDFs per evitar errors.
  • Considera les implicacions de rendiment i utilitza UDFs vectoritzades (pandas UDFs) en PySpark per obtenir un millor rendiment.

Entenent i utilitzant les UDFs, pots realitzar transformacions complexes i ampliar la funcionalitat de les teves aplicacions Spark més enllà de les funcions integrades.
