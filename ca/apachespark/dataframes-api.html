Introducció als DataFrames

Els DataFrames són una estructura de dades fonamental a Apache Spark, dissenyada per gestionar el processament de dades a gran escala amb facilitat. Són similars a les taules en una base de dades relacional o als data frames en R/Python, però optimitzats per a la computació distribuïda.

  • Definició: Un DataFrame és una col·lecció distribuïda de dades organitzades en columnes amb nom.
  • Característiques Clau:
    • Aplicació d'esquemes
    • Plans d'execució optimitzats
    • Suport per a una àmplia gamma de fonts de dades

Creació de DataFrames

Crear DataFrames és el primer pas per treballar amb ells. Pots crear DataFrames a partir de diverses fonts de dades com CSV, JSON, Parquet, i més.

A partir de RDDs Existents

from pyspark.sql import SparkSession
from pyspark.sql import Row

spark = SparkSession.builder.appName("DataFrameExample").getOrCreate()

# Crear un RDD
rdd = spark.sparkContext.parallelize([
    Row(name="Alice", age=29),
    Row(name="Bob", age=35)
])

# Convertir RDD a DataFrame
df = spark.createDataFrame(rdd)
df.show()

A partir d'un Fitxer CSV

df = spark.read.csv("path/to/csvfile.csv", header=True, inferSchema=True)
df.show()

A partir d'un Fitxer JSON

df = spark.read.json("path/to/jsonfile.json")
df.show()

A partir d'un Fitxer Parquet

df = spark.read.parquet("path/to/parquetfile.parquet")
df.show()

Operacions Bàsiques en DataFrames

Un cop tens un DataFrame, pots realitzar diverses operacions sobre ell.

Visualització de Dades

df.show()  # Mostra les primeres 20 files
df.printSchema()  # Imprimeix l'esquema del DataFrame
df.head(5)  # Retorna les primeres 5 files com una llista

Selecció de Columnes

df.select("name", "age").show()

Filtratge de Files

df.filter(df.age > 30).show()

Afegir Noves Columnes

from pyspark.sql.functions import col

df = df.withColumn("age_after_5_years", col("age") + 5)
df.show()

Agrupació i Agregació

df.groupBy("age").count().show()

Operacions Avançades en DataFrames

A mesura que et familiaritzis amb les operacions bàsiques, pots explorar funcionalitats més avançades.

Joins

df1 = spark.createDataFrame([(1, "Alice"), (2, "Bob")], ["id", "name"])
df2 = spark.createDataFrame([(1, "HR"), (2, "Engineering")], ["id", "department"])

df_joined = df1.join(df2, df1.id == df2.id)
df_joined.show()

Funcions de Finestra

from pyspark.sql.window import Window
from pyspark.sql.functions import rank, col

windowSpec = Window.partitionBy("department").orderBy(col("salary").desc())
df = df.withColumn("rank", rank().over(windowSpec))
df.show()

Funcions Definides per l'Usuari (UDFs)

from pyspark.sql.functions import udf
from pyspark.sql.types import StringType

def upper_case(name):
    return name.upper()

upper_case_udf = udf(upper_case, StringType())

df = df.withColumn("upper_name", upper_case_udf(df.name))
df.show()

Optimització del Rendiment

Optimitzar el rendiment de les operacions de DataFrame és crucial per gestionar grans conjunts de dades de manera eficient.

  • Caché: Utilitza df.cache() per emmagatzemar DataFrames en memòria.
  • Joins de Difusió: Utilitza broadcast(df) per a DataFrames petits per optimitzar les operacions de join.
  • Particionament: Reparticiona DataFrames utilitzant df.repartition(num_partitions) per equilibrar la càrrega.

Conclusió

L'API de DataFrames a Apache Spark proporciona una manera potent i flexible de treballar amb dades estructurades. Des d'operacions bàsiques com la selecció i el filtratge fins a tècniques avançades com els joins i les funcions de finestra, dominar els DataFrames és essencial per a un processament de dades eficient a Spark. Recorda aprofitar les tècniques d'optimització del rendiment per optimitzar els teus fluxos de treball i gestionar grans conjunts de dades de manera efectiva.
