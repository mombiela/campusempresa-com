Introducció

En aquesta secció, explorarem com executar consultes SQL programàticament utilitzant Apache Spark. Aquesta és una habilitat crucial per a enginyers de dades i científics de dades que necessiten realitzar manipulacions i anàlisis de dades complexes. Cobrirà els conceptes bàsics de Spark SQL, com crear DataFrames i com executar consultes SQL sobre aquests DataFrames.

Conceptes Clau

  • SparkSession: El punt d'entrada per programar Spark amb l'API de Dataset i DataFrame.
  • DataFrame: Una col·lecció distribuïda de dades organitzada en columnes amb nom.
  • SQLContext: Un punt d'entrada per treballar amb dades estructurades (DataFrames) a Spark.

Configuració de SparkSession

Abans d'executar consultes SQL, cal configurar una SparkSession. Aquest és el punt d'entrada per a totes les funcionalitats de Spark.

from pyspark.sql import SparkSession

# Inicialitzar SparkSession
spark = SparkSession.builder \
    .appName("SQL Queries Programmatically") \
    .getOrCreate()

Creació de DataFrames

Els DataFrames són l'abstracció principal a Spark SQL. Pots crear DataFrames a partir de diverses fonts de dades, com ara fitxers JSON, CSV i Parquet.

Exemple: Creació d'un DataFrame a partir d'un fitxer JSON

# Carregar dades JSON en un DataFrame
df = spark.read.json("path/to/your/jsonfile.json")

# Mostrar l'esquema del DataFrame
df.printSchema()

# Mostrar les primeres files del DataFrame
df.show()

Executar Consultes SQL

Un cop tens un DataFrame, pots registrar-lo com una vista temporal i executar consultes SQL contra ell.

Exemple: Registrar un DataFrame com una Vista Temporal

# Registrar el DataFrame com una vista temporal
df.createOrReplaceTempView("my_table")

# Executar una consulta SQL
result = spark.sql("SELECT * FROM my_table WHERE age > 30")

# Mostrar el resultat
result.show()

Consultes SQL Avançades

A mesura que et familiaritzis amb les consultes SQL bàsiques, pots començar a explorar consultes més avançades, com ara unions, agregacions i funcions de finestra.

Exemple: Realitzar una Unió

# Carregar un altre DataFrame
df2 = spark.read.json("path/to/another/jsonfile.json")

# Registrar el segon DataFrame com una vista temporal
df2.createOrReplaceTempView("another_table")

# Realitzar una unió entre les dues taules
join_result = spark.sql("""
    SELECT a.name, b.salary
    FROM my_table a
    JOIN another_table b
    ON a.id = b.id
""")

# Mostrar el resultat de la unió
join_result.show()

Exemple: Agregacions

# Realitzar una consulta d'agregació
agg_result = spark.sql("""
    SELECT department, AVG(salary) as avg_salary
    FROM another_table
    GROUP BY department
""")

# Mostrar el resultat de l'agregació
agg_result.show()

Conclusió

Executar consultes SQL programàticament a Apache Spark et permet aprofitar el poder de la computació distribuïda per a l'anàlisi i manipulació de dades. Entenent com crear DataFrames, registrar-los com a vistes temporals i executar consultes SQL, pots realitzar operacions de dades complexes de manera eficient. A mesura que avancis, podràs explorar funcionalitats SQL més sofisticades per satisfer les teves necessitats de processament de dades.
