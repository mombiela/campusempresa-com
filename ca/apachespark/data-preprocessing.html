El preprocessament de dades és un pas crucial en el pipeline d'anàlisi de dades. Implica netejar, transformar i preparar dades brutes per a l'anàlisi. En aquesta secció, cobrirem els conceptes bàsics del preprocessament de dades utilitzant Apache Spark, progressant des de nivells principiants fins a avançats.

Introducció al Preprocessament de Dades

El preprocessament de dades implica diversos passos per assegurar que les dades estiguin en un format adequat per a l'anàlisi. Aquests passos inclouen:

  • Neteja de Dades
  • Transformació de Dades
  • Integració de Dades
  • Reducció de Dades

Configuració d'Apache Spark

Abans de submergir-nos en el preprocessament de dades, assegura't de tenir Apache Spark instal·lat i configurat. Pots seguir la documentació oficial d'Apache Spark per a les instruccions d'instal·lació.

Exemple: Inicialització de la Sessió Spark

from pyspark.sql import SparkSession

# Inicialitza una sessió Spark
spark = SparkSession.builder \
    .appName("DataPreprocessing") \
    .getOrCreate()

# Comprova la sessió Spark
print(spark)

Neteja de Dades

La neteja de dades implica gestionar valors perduts, eliminar duplicats i corregir errors en el conjunt de dades.

Gestió de Valors Perduts

  • Eliminar valors perduts: Eliminar files amb valors perduts.
  • Omplir valors perduts: Substituir valors perduts amb un valor específic o una mesura estadística (mitjana, mediana, etc.).

Exemple: Eliminació de Valors Perduts

# Carrega dades de mostra
data = [("Alice", 34, None), ("Bob", None, "HR"), ("Cathy", 29, "IT")]
columns = ["Name", "Age", "Department"]
df = spark.createDataFrame(data, columns)

# Elimina files amb qualsevol valor perdut
df_cleaned = df.na.drop()

df_cleaned.show()

Exemple: Omplir Valors Perduts

# Omple valors perduts amb un valor específic
df_filled = df.na.fill({"Age": 0, "Department": "Unknown"})

df_filled.show()

Transformació de Dades

La transformació de dades implica convertir dades en un format adequat per a l'anàlisi. Les transformacions comunes inclouen normalització, escalat i codificació de variables categòriques.

Normalització i Escalat

  • Normalització: Ajusta el rang de les dades.
  • Escalat: Ajusta l'escala de les dades.

Exemple: Escalat de Dades

from pyspark.ml.feature import MinMaxScaler
from pyspark.ml.linalg import Vectors

# Dades de mostra
data = [(0, Vectors.dense([1.0, 0.1, -1.0]),),
        (1, Vectors.dense([2.0, 1.1, 1.0]),),
        (2, Vectors.dense([3.0, 10.1, 3.0]),)]
df = spark.createDataFrame(data, ["id", "features"])

# Inicialitza MinMaxScaler
scaler = MinMaxScaler(inputCol="features", outputCol="scaledFeatures")

# Ajusta i transforma les dades
scalerModel = scaler.fit(df)
scaledData = scalerModel.transform(df)

scaledData.show()

Codificació de Variables Categòriques

  • One-Hot Encoding: Converteix variables categòriques en vectors binaris.
  • Indexació de Cadenes: Assigna índexs numèrics a variables categòriques.

Exemple: One-Hot Encoding

from pyspark.ml.feature import OneHotEncoder, StringIndexer

# Dades de mostra
data = [("HR",), ("IT",), ("HR",), ("Finance",)]
df = spark.createDataFrame(data, ["Department"])

# StringIndexer
indexer = StringIndexer(inputCol="Department", outputCol="DeptIndex")
df_indexed = indexer.fit(df).transform(df)

# OneHotEncoder
encoder = OneHotEncoder(inputCol="DeptIndex", outputCol="DeptVec")
df_encoded = encoder.fit(df_indexed).transform(df_indexed)

df_encoded.show()

Integració de Dades

La integració de dades implica combinar dades de diferents fonts en un únic conjunt de dades.

Exemple: Unió de DataFrames

# Dades de mostra
data1 = [("Alice", 34), ("Bob", 45)]
data2 = [("Alice", "HR"), ("Bob", "IT")]
columns1 = ["Name", "Age"]
columns2 = ["Name", "Department"]

df1 = spark.createDataFrame(data1, columns1)
df2 = spark.createDataFrame(data2, columns2)

# Uneix DataFrames
df_joined = df1.join(df2, on="Name", how="inner")

df_joined.show()

Reducció de Dades

La reducció de dades implica reduir el volum de dades mantenint la seva integritat. Les tècniques inclouen mostreig, selecció de característiques i reducció de dimensionalitat.

Exemple: Selecció de Característiques

from pyspark.ml.feature import VectorAssembler
from pyspark.ml.stat import Correlation

# Dades de mostra
data = [(1, 0.1, -1.0, 3.0), (2, 1.1, 1.0, 2.0), (3, 10.1, 3.0, 1.0)]
columns = ["id", "feature1", "feature2", "feature3"]
df = spark.createDataFrame(data, columns)

# Aglutina característiques en un únic vector
assembler = VectorAssembler(inputCols=["feature1", "feature2", "feature3"], outputCol="features")
df_vector = assembler.transform(df)

# Calcula la matriu de correlació
correlation_matrix = Correlation.corr(df_vector, "features").head()[0]
print("Matriu de correlació:\n", correlation_matrix)

Conclusió

El preprocessament de dades és un pas fonamental en el pipeline d'anàlisi de dades. Utilitzant Apache Spark, podem netejar, transformar, integrar i reduir dades de manera eficient a gran escala. En dominar aquestes tècniques, pots assegurar-te que les teves dades estiguin llestes per a l'anàlisi i la modelització, conduint a resultats més precisos i fiables.
