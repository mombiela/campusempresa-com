Apache Spark és un potent sistema de computació distribuïda de codi obert que proporciona una interfície per programar clústers sencers amb paral·lelisme de dades implícit i tolerància a fallades. Aquesta secció et guiarà a través del procés d'instal·lació i configuració d'Apache Spark a la teva màquina local.

Prerequisits

Abans de començar amb la instal·lació, assegura't de tenir els següents prerequisits:

  • Java Development Kit (JDK) 8 o posterior: Spark s'executa en Java, així que necessites tenir JDK instal·lat.
  • Scala (opcional): Si tens intenció d'escriure aplicacions Spark en Scala, necessites tenir Scala instal·lat.
  • Python (opcional): Si tens intenció d'escriure aplicacions Spark en Python, necessites tenir Python instal·lat.
  • Apache Hadoop (opcional): Si tens intenció d'utilitzar Spark amb Hadoop, necessites tenir Hadoop instal·lat.

Guia d'Instal·lació Pas a Pas

Pas 1: Instal·lar Java Development Kit (JDK)

  1. Descarregar JDK: Ves a la pàgina de descàrrega d'Oracle JDK i descarrega la versió adequada per al teu sistema operatiu.
  2. Instal·lar JDK: Segueix les instruccions d'instal·lació proporcionades a la pàgina de descàrrega.

Pas 2: Descarregar Apache Spark

  1. Visita la Pàgina de Descàrrega de Spark: Ves a la pàgina de descàrrega d'Apache Spark.
  2. Selecciona Versió: Tria la versió de Spark que vols descarregar. Es recomana descarregar l'última versió estable.
  3. Selecciona Tipus de Paquet: Tria un paquet precompilat per a Hadoop. Si no utilitzes Hadoop, selecciona "Pre-built for Apache Hadoop 2.7 and later".
  4. Descarregar: Fes clic a l'enllaç de descàrrega per descarregar el paquet de Spark.

Pas 3: Instal·lar Apache Spark

  1. Extreure el Paquet: Extreu el paquet descarregat de Spark a un directori de la teva elecció. Per exemple, en sistemes basats en Unix, pots utilitzar la següent comanda: 
    tar -xvf spark-<version>-bin-hadoop2.7.tgz
  2. Configurar Variables d'Entorn: Afegeix Spark a la variable d'entorn PATH del teu sistema. Per exemple, en sistemes basats en Unix, afegeix les següents línies al teu fitxer .bashrc o .zshrc: 
    export SPARK_HOME=/path/to/spark-<version>-bin-hadoop2.7
export PATH=$SPARK_HOME/bin:$PATH

Pas 4: Verificar la Instal·lació

  1. Obrir un Terminal: Obre un terminal o símbol del sistema.
  2. Executar Spark Shell: Escriu la següent comanda per iniciar el shell de Spark: 
    spark-shell
  3. Comprovar Sortida: Si Spark està instal·lat correctament, hauries de veure el prompt del shell de Spark: 
    Spark context Web UI available at http://<your-ip>:4040
Spark context available as 'sc' (master = local[*], app id = local-<timestamp>).
Spark session available as 'spark'.

Pas 5: Instal·lar Components Opcional

  • Scala: Si tens intenció d'escriure aplicacions Spark en Scala, descarrega i instal·la Scala des de la pàgina de descàrrega de Scala.
  • Python: Si tens intenció d'escriure aplicacions Spark en Python, descarrega i instal·la Python des de la pàgina de descàrrega de Python.

Executar una Aplicació Simple de Spark

Per assegurar-te que tot està configurat correctament, executem una aplicació simple de Spark.

  1. Crear un Script en Python: Crea un fitxer anomenat simple_app.py amb el següent contingut:

    from pyspark.sql import SparkSession

# Inicialitzar sessió de Spark
spark = SparkSession.builder.appName("SimpleApp").getOrCreate()

# Crear un DataFrame simple
data = [("Alice", 34), ("Bob", 45), ("Cathy", 29)]
df = spark.createDataFrame(data, ["Name", "Age"])

# Mostrar el DataFrame
df.show()

# Aturar la sessió de Spark
spark.stop()

  2. Executar l'Script: Executa l'script utilitzant la següent comanda:

    spark-submit simple_app.py

  3. Comprovar Sortida: Hauries de veure la següent sortida:

    +-----+---+
| Name|Age|
+-----+---+
|Alice| 34|
|  Bob| 45|
|Cathy| 29|
+-----+---+

Conclusió

En aquesta secció, hem cobert els passos per instal·lar i configurar Apache Spark a la teva màquina local. També hem executat una aplicació simple de Spark per verificar la instal·lació. Amb Spark instal·lat, ara estàs preparat per aprofundir en les seves potents capacitats de processament de dades.
