Introducció

L'avaluació i l'ajust del model són passos crítics en el pipeline d'aprenentatge automàtic. Asseguren que els models que construeixes no només siguin precisos, sinó que també es generalitzin bé a dades no vistes. En aquesta secció, explorarem com avaluar i ajustar models d'aprenentatge automàtic utilitzant MLlib d'Apache Spark.

Conceptes Clau

  • Avaluació del Model: El procés d'avaluar com de bé funciona un model.
  • Ajust del Model: El procés d'optimitzar els paràmetres del model per millorar el rendiment.
  • Validació Creuada: Una tècnica per avaluar el rendiment del model dividint les dades en conjunts d'entrenament i prova múltiples vegades.
  • Hiperparàmetres: Paràmetres que es configuren abans que comenci el procés d'aprenentatge i controlen el procés d'aprenentatge.

Avaluació del Model

Mètriques

Spark MLlib proporciona diverses mètriques per avaluar el rendiment dels models d'aprenentatge automàtic:

  • Mètriques de Classificació: Precisió, Precisió, Record, F1 Score, etc.
  • Mètriques de Regressió: Error Quadràtic Mitjà (MSE), Arrel de l'Error Quadràtic Mitjà (RMSE), Error Absolut Mitjà (MAE), R-quadrat, etc.

Exemple: Avaluació d'un Model de Classificació

Avaluem un model de regressió logística utilitzant la precisió com a mètrica.

from pyspark.ml.classification import LogisticRegression
from pyspark.ml.evaluation import MulticlassClassificationEvaluator
from pyspark.sql import SparkSession

# Inicialitzar la sessió de Spark
spark = SparkSession.builder.appName("ModelEvaluation").getOrCreate()

# Carregar dades
data = spark.read.format("libsvm").load("data/mllib/sample_libsvm_data.txt")

# Dividir les dades en conjunts d'entrenament i prova
train_data, test_data = data.randomSplit([0.7, 0.3])

# Entrenar un model de regressió logística
lr = LogisticRegression()
model = lr.fit(train_data)

# Fer prediccions
predictions = model.transform(test_data)

# Avaluar el model
evaluator = MulticlassClassificationEvaluator(metricName="accuracy")
accuracy = evaluator.evaluate(predictions)
print(f"Precisió de Prova: {accuracy}")

Exemple: Avaluació d'un Model de Regressió

Avaluem un model de regressió lineal utilitzant RMSE com a mètrica.

from pyspark.ml.regression import LinearRegression
from pyspark.ml.evaluation import RegressionEvaluator

# Carregar dades
data = spark.read.format("libsvm").load("data/mllib/sample_linear_regression_data.txt")

# Dividir les dades en conjunts d'entrenament i prova
train_data, test_data = data.randomSplit([0.7, 0.3])

# Entrenar un model de regressió lineal
lr = LinearRegression()
model = lr.fit(train_data)

# Fer prediccions
predictions = model.transform(test_data)

# Avaluar el model
evaluator = RegressionEvaluator(metricName="rmse")
rmse = evaluator.evaluate(predictions)
print(f"RMSE de Prova: {rmse}")

Ajust del Model

Ajust d'Hiperparàmetres

L'ajust d'hiperparàmetres implica trobar el millor conjunt d'hiperparàmetres per a un model. Spark MLlib proporciona eines com ParamGridBuilder i CrossValidator per facilitar aquest procés.

Exemple: Ajust d'Hiperparàmetres amb Validació Creuada

Ajustem els hiperparàmetres d'un model de regressió logística utilitzant validació creuada.

from pyspark.ml.tuning import ParamGridBuilder, CrossValidator

# Definir la graella de paràmetres
paramGrid = ParamGridBuilder() \
    .addGrid(lr.regParam, [0.1, 0.01]) \
    .addGrid(lr.elasticNetParam, [0.0, 0.5, 1.0]) \
    .build()

# Definir el validador creuat
crossval = CrossValidator(estimator=lr,
                          estimatorParamMaps=paramGrid,
                          evaluator=evaluator,
                          numFolds=3)

# Executar la validació creuada i triar el millor conjunt de paràmetres
cvModel = crossval.fit(train_data)

# Fer prediccions sobre les dades de prova
predictions = cvModel.transform(test_data)

# Avaluar el model ajustat
accuracy = evaluator.evaluate(predictions)
print(f"Precisió de Prova Ajustada: {accuracy}")

Conclusió

L'avaluació i l'ajust del model són essencials per construir models d'aprenentatge automàtic robustos. Apache Spark MLlib proporciona un conjunt complet d'eines per avaluar i ajustar models de manera eficient. Comprenent i aplicant aquestes tècniques, pots millorar significativament el rendiment i la fiabilitat de les teves aplicacions d'aprenentatge automàtic.
