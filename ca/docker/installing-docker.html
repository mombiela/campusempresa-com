Introducció

Docker és una plataforma que permet als desenvolupadors automatitzar el desplegament d'aplicacions dins de contenidors lleugers i portàtils. Aquesta secció et guiarà a través del procés d'instal·lació de Docker en diversos sistemes operatius.

Prerequisits

Abans d'instal·lar Docker, assegura't que el teu sistema compleixi els següents requisits:

  • Un sistema operatiu de 64 bits.
  • Almenys 4GB de RAM.
  • Privilegis administratius a la teva màquina.

Instal·lant Docker a Windows

Guia Pas a Pas

  1. Descarrega Docker Desktop per a Windows:

    • Visita el Docker Hub i descarrega l'instal·lador de Docker Desktop.

  2. Executa l'Instal·lador:

    • Fes doble clic al fitxer .exe descarregat per executar l'instal·lador.
    • Segueix les instruccions en pantalla per completar la instal·lació.

  3. Inicia Docker Desktop:

    • Un cop instal·lat, Docker Desktop s'iniciarà automàticament.
    • Potser necessitis tancar sessió i tornar a iniciar sessió per completar la instal·lació.

  4. Verifica la Instal·lació:

    • Obre una finestra de Command Prompt o PowerShell.
    • Executa la següent comanda per verificar que Docker està instal·lat correctament: 
      docker --version
    • Hauries de veure una sortida similar a Docker version 20.10.7, build f0df350.

Instal·lant Docker a macOS

Guia Pas a Pas

  1. Descarrega Docker Desktop per a Mac:

    • Visita el Docker Hub i descarrega l'instal·lador de Docker Desktop.

  2. Executa l'Instal·lador:

    • Fes doble clic al fitxer .dmg descarregat per obrir l'instal·lador.
    • Arrossega la icona de Docker a la carpeta d'Aplicacions.

  3. Inicia Docker Desktop:

    • Obre Docker des de la carpeta d'Aplicacions.
    • Docker Desktop s'iniciarà i potser necessitis autoritzar Docker amb la teva contrasenya de sistema.

  4. Verifica la Instal·lació:

    • Obre una finestra de Terminal.
    • Executa la següent comanda per verificar que Docker està instal·lat correctament: 
      docker --version
    • Hauries de veure una sortida similar a Docker version 20.10.7, build f0df350.

Instal·lant Docker a Linux

Guia Pas a Pas

  1. Actualitza l'Índex de Paquets:

    • Obre una terminal i executa: 
      sudo apt-get update

  2. Instal·la els Paquets Necessaris:

    • Executa la següent comanda per instal·lar els paquets necessaris: 
      sudo apt-get install apt-transport-https ca-certificates curl software-properties-common

  3. Afegeix la Clau GPG Oficial de Docker:

    • Executa la següent comanda: 
      curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

  4. Configura el Repositori Estable:

    • Afegeix el repositori de Docker a les fonts d'APT: 
      sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

  5. Instal·la Docker CE:

    • Actualitza l'índex de paquets de nou: 
      sudo apt-get update
    • Instal·la Docker: 
      sudo apt-get install docker-ce

  6. Verifica la Instal·lació:

    • Executa la següent comanda per verificar que Docker està instal·lat correctament: 
      docker --version
    • Hauries de veure una sortida similar a Docker version 20.10.7, build f0df350.

Passos Posteriors a la Instal·lació

Gestionant Docker com a Usuari No Root

Per defecte, les comandes de Docker només poden ser executades per l'usuari root o per un usuari del grup docker. Per evitar utilitzar sudo amb cada comanda de Docker, afegeix el teu usuari al grup docker:

  1. Crea el Grup Docker:

    sudo groupadd docker

  2. Afegeix el Teu Usuari al Grup Docker:

    sudo usermod -aG docker $USER

  3. Aplica la Nova Membresia de Grup:

    • Tanca sessió i torna a iniciar sessió perquè la teva membresia de grup sigui reavaluada.
    • Alternativament, pots executar: 
      newgrp docker

  4. Verifica la Membresia de Grup:

    • Executa una comanda de Docker sense sudo per verificar: 
      docker run hello-world

Conclusió

Instal·lar Docker és un procés senzill que varia lleugerament segons el teu sistema operatiu. Seguint els passos descrits en aquesta secció, ara hauries de tenir Docker instal·lat i funcionant a la teva màquina. Aquesta configuració et permetrà començar a construir i desplegar aplicacions contenidoritzades de manera eficient.
