Introducció a Docker Compose

Docker Compose és una eina per definir i executar aplicacions Docker amb múltiples contenidors. Amb Compose, utilitzes un fitxer YAML per configurar els serveis de la teva aplicació. Després, amb una sola comanda, crees i inicies tots els serveis des de la teva configuració.

Conceptes Clau

  • Servei: Un servei és un contenidor a la teva aplicació. Cada servei executa una imatge i es pot escalar especificant el nombre de contenidors a executar.
  • Volum: Els volums s'utilitzen per persistir dades generades i utilitzades pels contenidors Docker.
  • Xarxa: Les xarxes permeten que els contenidors es comuniquin entre ells, ja sigui en el mateix host o entre múltiples hosts.

Configurar Docker Compose

Per començar amb Docker Compose, necessites instal·lar Docker i Docker Compose a la teva màquina.

Instal·lació

  • Docker: Segueix la guia d'instal·lació oficial de Docker per al teu sistema operatiu.
  • Docker Compose: Docker Compose està inclòs a Docker Desktop per a Windows i Mac. Per a Linux, pots instal·lar-lo utilitzant la següent comanda: 
    sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

Crear un Fitxer Docker Compose

Un fitxer Docker Compose és un fitxer YAML que defineix serveis, xarxes i volums per a una aplicació Docker.

Estructura Bàsica

Aquí tens un exemple bàsic d'un fitxer docker-compose.yml:

version: '3'
services:
  web:
    image: nginx:latest
    ports:
      - "80:80"
  db:
    image: postgres:latest
    environment:
      POSTGRES_PASSWORD: example

Explicació

  • version: Especifica la versió del format del fitxer Compose.
  • services: Defineix els serveis a executar.
    • web: Un servei anomenat web que utilitza la imatge nginx:latest.
      • ports: Mapatge del port 80 a l'host al port 80 al contenidor.
    • db: Un servei anomenat db que utilitza la imatge postgres:latest.
      • environment: Defineix variables d'entorn per al contenidor.

Executar Docker Compose

Per iniciar els serveis definits al teu fitxer docker-compose.yml, navega al directori que conté el fitxer i executa:

docker-compose up

Aquesta comanda iniciarà tots els serveis en primer pla. Per executar-los en segon pla, utilitza:

docker-compose up -d

Gestionar Serveis

Docker Compose proporciona diverses comandes per gestionar els teus serveis.

Comandes Comunes

  • Iniciar Serveis: docker-compose start
  • Aturar Serveis: docker-compose stop
  • Reiniciar Serveis: docker-compose restart
  • Veure Registres: docker-compose logs
  • Escalar Serveis: docker-compose up --scale web=3 -d

Exemple

Per escalar el servei web a 3 instàncies, utilitzaries:

docker-compose up --scale web=3 -d

Utilitzar Volums i Xarxes

Els volums i les xarxes són essencials per a la persistència de dades i la comunicació entre serveis.

Volums

Defineix volums al teu fitxer docker-compose.yml:

services:
  db:
    image: postgres:latest
    volumes:
      - db-data:/var/lib/postgresql/data
volumes:
  db-data:

Xarxes

Defineix xarxes personalitzades per als teus serveis:

services:
  web:
    image: nginx:latest
    networks:
      - webnet
  db:
    image: postgres:latest
    networks:
      - webnet
networks:
  webnet:

Configuració Avançada

Docker Compose permet configuracions més avançades com utilitzar fitxers d'entorn, estendre serveis i utilitzar múltiples fitxers Compose.

Fitxers d'Entorn

Pots utilitzar un fitxer d'entorn per definir variables d'entorn:

services:
  web:
    image: nginx:latest
    env_file:
      - .env

Estendre Serveis

Pots estendre serveis existents per reutilitzar configuracions:

services:
  base:
    image: nginx:latest
  web:
    extends:
      service: base
      file: common-services.yml

Múltiples Fitxers Compose

Pots utilitzar múltiples fitxers Compose per sobreescriure configuracions:

docker-compose -f docker-compose.yml -f docker-compose.override.yml up -d

Conclusió

Gestionar projectes amb Docker Compose implica entendre l'estructura del fitxer docker-compose.yml, executar i gestionar serveis, i utilitzar volums i xarxes per a la persistència de dades i la comunicació. Amb aquestes eines, pots gestionar eficientment aplicacions Docker amb múltiples contenidors, fent que els teus processos de desenvolupament i desplegament siguin més àgils i efectius.
