Introducció a Docker Hub

Docker Hub és un repositori basat en el núvol on els usuaris de Docker poden crear, provar, emmagatzemar i distribuir imatges de contenidors. Serveix com un lloc central per trobar i compartir imatges de contenidors.

Conceptes Clau

  • Repositori: Una col·lecció d'imatges Docker relacionades, sovint utilitzada per distribuir una aplicació o servei específic.
  • Imatge: Un paquet executable lleuger, autònom, que inclou tot el necessari per executar un programari.
  • Etiqueta: Una etiqueta aplicada a una imatge Docker en un repositori, sovint utilitzada per denotar diferents versions d'una imatge.
  • Compte de Docker Hub: Un compte necessari per pujar imatges a Docker Hub i gestionar repositoris.

Configuració de Docker Hub

Per utilitzar Docker Hub, necessites crear un compte i configurar el teu client Docker per interactuar amb Docker Hub.

Passos per Configurar Docker Hub

  1. Crear un Compte de Docker Hub:

    • Visita Docker Hub.
    • Fes clic a "Sign Up" i segueix les instruccions per crear un compte.

  2. Iniciar Sessió a Docker Hub des de la Línia de Comandes:

    docker login
    • Introdueix el teu nom d'usuari i contrasenya de Docker Hub quan se't demani.

Ús de Docker Hub

Docker Hub et permet cercar imatges, descarregar-les a la teva màquina local i pujar les teves pròpies imatges al repositori.

Cerca d'Imatges

Pots cercar imatges a Docker Hub utilitzant la interfície web o la línia de comandes.

Exemple: Cercar una Imatge

docker search nginx

Aquesta comanda retornarà una llista d'imatges relacionades amb "nginx" disponibles a Docker Hub.

Descarregar Imatges

Per utilitzar una imatge de Docker Hub, necessites descarregar-la a la teva màquina local.

Exemple: Descarregar una Imatge

docker pull nginx

Aquesta comanda descarregarà la imatge "nginx" de Docker Hub a la teva màquina local.

Pujar Imatges

Per compartir les teves pròpies imatges, necessites pujar-les a Docker Hub.

Exemple: Pujar una Imatge

  1. Etiquetar la Imatge: 
    docker tag my-image:latest myusername/my-repo:my-tag
  2. Pujar la Imatge: 
    docker push myusername/my-repo:my-tag

Imatges Oficials

Les imatges oficials són imatges Docker que són curades i mantingudes per Docker, assegurant que segueixen les millors pràctiques i són actualitzades regularment.

Beneficis d'Utilitzar Imatges Oficials

  • Seguretat: Actualitzades regularment per abordar vulnerabilitats de seguretat.
  • Fiabilitat: Mantingudes per Docker i col·laboradors de confiança.
  • Millors Pràctiques: Segueixen les directrius de Docker per a la creació d'imatges.

Exemples d'Imatges Oficials

  • nginx: Un servidor web d'alt rendiment.
  • mysql: Un sistema de gestió de bases de dades relacionals popular.
  • python: Una imatge que conté el llenguatge de programació Python.

Exemple: Utilitzar una Imatge Oficial

docker pull python:3.9
docker run -it python:3.9 python

Aquesta comanda descarrega la imatge oficial de Python i inicia una sessió interactiva de Python.

Conclusió

Docker Hub és una eina essencial per gestionar i distribuir imatges Docker. Entenent com cercar, descarregar i pujar imatges, i aprofitant les imatges oficials, pots assegurar que les teves aplicacions contenidoritzades siguin segures, fiables i segueixin les millors pràctiques.
