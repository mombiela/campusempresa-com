El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquest projecte, crearem una aplicació web senzilla i la containeritzarem utilitzant Docker. Aquest projecte t'ajudarà a entendre els conceptes bàsics de Docker, incloent la creació de Dockerfiles, la construcció d'imatges i l'execució de contenidors.

Objectius

Al final d'aquest projecte, tu:

  • Entendràs com crear un Dockerfile.
  • Aprendràs com construir una imatge Docker.
  • Executaràs un contenidor Docker.
  • Exposaràs una aplicació web containeritzada al món exterior.

Prerequisits

Abans de començar aquest projecte, assegura't de tenir:

  • Coneixements bàsics de desenvolupament web (HTML, CSS, JavaScript).
  • Comprensió bàsica dels conceptes de Docker.
  • Docker instal·lat a la teva màquina.

Pas 1: Crear una Aplicació Web Senzilla

Primer, creem una aplicació web senzilla utilitzant Node.js.

  1. Inicialitza un projecte Node.js:

    mkdir simple-web-app
cd simple-web-app
npm init -y

  2. Instal·la Express.js:

    npm install express

  3. Crea app.js:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('Hola, Món!');
});

app.listen(port, () => {
    console.log(`Aplicació en funcionament a http://localhost:${port}`);
});

  4. Actualitza package.json per incloure un script d'inici:

    "scripts": {
    "start": "node app.js"
}

Pas 2: Crear un Dockerfile

Un Dockerfile és un document de text que conté totes les comandes per assemblar una imatge.

  1. Crea un Dockerfile a l'arrel del teu projecte:

    # Utilitza la imatge oficial de Node.js com a imatge base
FROM node:14

# Estableix el directori de treball
WORKDIR /usr/src/app

# Copia package.json i package-lock.json
COPY package*.json ./

# Instal·la les dependències
RUN npm install

# Copia la resta del codi de l'aplicació
COPY . .

# Exposa el port en el qual s'executa l'aplicació
EXPOSE 3000

# Comanda per executar l'aplicació
CMD ["npm", "start"]

Pas 3: Construir la Imatge Docker

Per construir la imatge Docker, utilitza la comanda docker build.

  1. Construeix la imatge:

    docker build -t simple-web-app .
    • -t simple-web-app: Etiqueta la imatge amb el nom simple-web-app.
    • .: Especifica el directori actual com a context de construcció.

Pas 4: Executar el Contenidor Docker

Un cop la imatge està construïda, pots executar-la utilitzant la comanda docker run.

  1. Executa el contenidor:

    docker run -p 3000:3000 simple-web-app
    • -p 3000:3000: Mapeja el port 3000 del teu host al port 3000 del contenidor.

  2. Accedeix a l'aplicació:

    Obre el teu navegador i navega a http://localhost:3000. Hauries de veure "Hola, Món!".

Conclusió

En aquest projecte, has après a:

  • Crear una aplicació web senzilla utilitzant Node.js i Express.
  • Escriure un Dockerfile per containeritzar l'aplicació.
  • Construir una imatge Docker a partir del Dockerfile.
  • Executar un contenidor Docker a partir de la imatge i exposar-lo al món exterior.

Aquest coneixement fonamental servirà com a base per a temes més avançats de Docker, com ara construccions multi-etapa, Docker Compose i l'orquestració de contenidors amb Kubernetes.
