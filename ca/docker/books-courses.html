Introducció

En aquesta secció, explorarem diversos llibres i cursos que poden ajudar-te a dominar Docker des d'un nivell principiant fins a un nivell avançat. Tant si prefereixes llegir llibres com si prefereixes fer cursos en línia, hi ha molts recursos disponibles per adaptar-se al teu estil d'aprenentatge.

Llibres

Nivell Principiant

  1. "Docker: Up & Running" de Karl Matthias i Sean P. Kane

    • Visió general: Aquest llibre proporciona una sòlida introducció a Docker, cobrint els conceptes bàsics de la containerització i com utilitzar Docker de manera efectiva.
    • Conceptes clau:
      • Introducció a Docker
      • Configuració de Docker en diferents plataformes
      • Comandes bàsiques de Docker
      • Construcció i execució de contenidors Docker

  2. "The Docker Book: Containerization is the new virtualization" de James Turnbull

    • Visió general: Una guia completa per a principiants, aquest llibre cobreix els fonaments de Docker i la containerització.
    • Conceptes clau:
      • Arquitectura de Docker
      • Treballar amb imatges i contenidors Docker
      • Xarxes Docker
      • Conceptes bàsics de Docker Compose

Nivell Intermedi

  1. "Docker in Action" de Jeff Nickoloff i Stephen Kuenzli

    • Visió general: Aquest llibre aprofundeix més en Docker, proporcionant exemples pràctics i tècniques avançades.
    • Conceptes clau:
      • Comandes avançades de Docker
      • Millors pràctiques de Dockerfile
      • Gestió de volums i xarxes Docker
      • Docker Compose per a aplicacions multi-contenidor

  2. "Using Docker: Developing and Deploying Software with Containers" de Adrian Mouat

    • Visió general: Es centra en l'ús de Docker en escenaris del món real, incloent desenvolupament, proves i desplegament.
    • Conceptes clau:
      • Integració i desplegament continus amb Docker
      • Docker Swarm i orquestració
      • Millors pràctiques de seguretat
      • Monitorització i registre

Nivell Avançat

  1. "Kubernetes Up & Running: Dive into the Future of Infrastructure" de Kelsey Hightower, Brendan Burns, i Joe Beda

    • Visió general: Tot i que es centra principalment en Kubernetes, aquest llibre és essencial per entendre l'orquestració avançada de contenidors amb Docker.
    • Conceptes clau:
      • Arquitectura de Kubernetes
      • Desplegament i gestió de contenidors a gran escala
      • Xarxes i emmagatzematge avançats
      • Monitorització i resolució de problemes

  2. "Docker Deep Dive" de Nigel Poulton

    • Visió general: Aquest llibre proporciona una visió profunda de Docker, cobrint temes avançats i millors pràctiques.
    • Conceptes clau:
      • Internals de Docker
      • Xarxes avançades de Docker
      • Seguretat i compliment
      • Optimització i ajust de rendiment

Cursos en Línia

Nivell Principiant

  1. "Docker for Beginners" a Udemy

    • Visió general: Un curs amigable per a principiants que cobreix els conceptes bàsics de Docker.
    • Conceptes clau:
      • Introducció a Docker
      • Instal·lació de Docker
      • Comandes bàsiques de Docker
      • Construcció i execució de contenidors

  2. "Getting Started with Docker" a Coursera

    • Visió general: Aquest curs proporciona una introducció completa a Docker i la containerització.
    • Conceptes clau:
      • Arquitectura de Docker
      • Treballar amb imatges i contenidors Docker
      • Conceptes bàsics de Docker Compose
      • Xarxes Docker

Nivell Intermedi

  1. "Docker Mastery: with Kubernetes +Swarm from a Docker Captain" a Udemy

    • Visió general: Aquest curs cobreix conceptes intermedis a avançats de Docker, incloent orquestració amb Kubernetes i Swarm.
    • Conceptes clau:
      • Docker Compose per a aplicacions multi-contenidor
      • Conceptes bàsics de Docker Swarm
      • Introducció a Kubernetes
      • Integració i desplegament continus

  2. "Docker and Kubernetes: The Complete Guide" a Udemy

    • Visió general: Un curs complet que cobreix Docker i Kubernetes en detall.
    • Conceptes clau:
      • Comandes avançades de Docker
      • Millors pràctiques de Dockerfile
      • Arquitectura de Kubernetes
      • Desplegament i gestió de contenidors amb Kubernetes

Nivell Avançat

  1. "Kubernetes for Developers: Core Concepts" a Coursera

    • Visió general: Aquest curs se centra en conceptes avançats de Kubernetes, essencials per dominar l'orquestració de contenidors.
    • Conceptes clau:
      • Arquitectura i components de Kubernetes
      • Desplegament i escalat d'aplicacions
      • Xarxes i emmagatzematge avançats
      • Monitorització i resolució de problemes

  2. "Advanced Docker for DevOps Engineers" a Udemy

    • Visió general: Aquest curs cobreix temes avançats de Docker i millors pràctiques per a enginyers DevOps.
    • Conceptes clau:
      • Internals de Docker
      • Xarxes i seguretat avançades
      • Optimització de rendiment
      • Docker en entorns de producció

Conclusió

Dominar Docker requereix una combinació de coneixement teòric i experiència pràctica. Els llibres i cursos esmentats anteriorment proporcionen un camí estructurat des de nivells principiants fins a nivells avançats, cobrint conceptes essencials i tècniques avançades. Aprofitant aquests recursos, pots construir una sòlida base en Docker i progressar cap a tasques més complexes de containerització i orquestració. Bona aprenentatge!
