En aquesta secció, cobrirem com gestionar contenidors Docker de manera efectiva. Això inclou iniciar, aturar, inspeccionar i eliminar contenidors, així com gestionar registres de contenidors i xarxes. Al final d'aquesta secció, tindràs una comprensió sòlida de com controlar i interactuar amb contenidors Docker.

Iniciar i Aturar Contenidors

Gestionar el cicle de vida dels contenidors és fonamental per utilitzar Docker. Aquí, aprendrem com iniciar, aturar i reiniciar contenidors.

Iniciar un Contenidor

Per iniciar un contenidor, utilitza la comanda docker run. Aquesta comanda crea i inicia un nou contenidor a partir d'una imatge especificada.

docker run -d --name my_container nginx
  • -d: Executa el contenidor en mode desconnectat (en segon pla).
  • --name my_container: Assigna un nom al contenidor.
  • nginx: La imatge a utilitzar per al contenidor.

Aturar un Contenidor

Per aturar un contenidor en execució, utilitza la comanda docker stop seguida del nom o ID del contenidor.

docker stop my_container

Reiniciar un Contenidor

Per reiniciar un contenidor aturat, utilitza la comanda docker start.

docker start my_container

També pots reiniciar un contenidor en execució utilitzant la comanda docker restart.

docker restart my_container

Inspeccionar Contenidors

Inspeccionar contenidors et permet veure informació detallada sobre ells, com la seva configuració, estat i ús de recursos.

Veure Detalls del Contenidor

Utilitza la comanda docker inspect per obtenir informació detallada sobre un contenidor.

docker inspect my_container

Aquesta comanda retorna un objecte JSON que conté tots els detalls sobre el contenidor.

Comprovar els Registres del Contenidor

Per veure els registres d'un contenidor, utilitza la comanda docker logs.

docker logs my_container

També pots seguir els registres en temps real utilitzant el flag -f.

docker logs -f my_container

Eliminar Contenidors

Eliminar contenidors que ja no són necessaris ajuda a alliberar recursos del sistema.

Eliminar un Contenidor Aturat

Per eliminar un contenidor aturat, utilitza la comanda docker rm.

docker rm my_container

Eliminar un Contenidor en Execució

Per eliminar un contenidor en execució, primer has d'aturar-lo o utilitzar el flag -f per eliminar-lo forçosament.

docker rm -f my_container

Gestionar Xarxes de Contenidors

Docker proporciona diverses opcions de xarxa per connectar contenidors entre ells i amb xarxes externes.

Llistar Xarxes

Per llistar totes les xarxes Docker disponibles, utilitza la comanda docker network ls.

docker network ls

Crear una Xarxa

Per crear una nova xarxa, utilitza la comanda docker network create.

docker network create my_network

Connectar un Contenidor a una Xarxa

Per connectar un contenidor en execució a una xarxa, utilitza la comanda docker network connect.

docker network connect my_network my_container

Desconnectar un Contenidor d'una Xarxa

Per desconnectar un contenidor d'una xarxa, utilitza la comanda docker network disconnect.

docker network disconnect my_network my_container

Resum

En aquesta secció, hem cobert les comandes i conceptes essencials per gestionar contenidors Docker. Has après com iniciar, aturar i reiniciar contenidors, inspeccionar els seus detalls, veure registres i eliminar-los quan ja no són necessaris. A més, hem explorat com gestionar la xarxa de contenidors, incloent la creació de xarxes i la connexió o desconnexió de contenidors d'aquestes. Dominar aquestes habilitats et permetrà controlar i interactuar efectivament amb contenidors Docker en els teus entorns de desenvolupament i producció.
