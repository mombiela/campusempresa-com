Docker Machine és una eina que et permet instal·lar Docker Engine en hosts virtuals i gestionar els hosts amb comandes docker-machine. Pots utilitzar Machine per crear hosts Docker a la teva màquina local, en proveïdors de núvol i en el teu centre de dades.

Conceptes Clau

  • Docker Machine: Una eina per aprovisionar i gestionar hosts Docker.
  • Driver: Un connector que permet a Docker Machine crear hosts en diverses plataformes (per exemple, VirtualBox, AWS, DigitalOcean).
  • Machine: Un host Docker creat i gestionat per Docker Machine.
  • Variables d'Entorn: Variables que configuren el client Docker per comunicar-se amb Docker Engine en una màquina específica.

Instal·lant Docker Machine

Per instal·lar Docker Machine, segueix aquests passos:

  1. Descarrega el binari de Docker Machine: 
    curl -L https://github.com/docker/machine/releases/download/v0.16.2/docker-machine-$(uname -s)-$(uname -m) >/tmp/docker-machine
  2. Fes el binari executable: 
    chmod +x /tmp/docker-machine
  3. Mou el binari al teu PATH: 
    sudo mv /tmp/docker-machine /usr/local/bin/docker-machine

Creant un Host Docker

Per crear un host Docker, utilitza la comanda docker-machine create juntament amb un driver. Per exemple, per crear un host Docker utilitzant VirtualBox:

docker-machine create --driver virtualbox my-docker-host

Explicació:

  • --driver virtualbox: Especifica el driver a utilitzar (VirtualBox en aquest cas).
  • my-docker-host: El nom de l'host Docker que s'està creant.

Gestionant Hosts Docker

Un cop hagis creat hosts Docker, pots gestionar-los utilitzant diverses comandes docker-machine.

Llistant Màquines Docker

Per llistar totes les màquines Docker:

docker-machine ls

Iniciant i Aturant Màquines Docker

Per iniciar una màquina Docker:

docker-machine start my-docker-host

Per aturar una màquina Docker:

docker-machine stop my-docker-host

Eliminant una Màquina Docker

Per eliminar una màquina Docker:

docker-machine rm my-docker-host

Configurant el Client Docker

Per configurar el client Docker per utilitzar una màquina Docker específica, necessites establir variables d'entorn. Docker Machine proporciona una manera convenient de fer-ho:

eval $(docker-machine env my-docker-host)

Explicació:

  • docker-machine env my-docker-host: Genera les variables d'entorn necessàries.
  • eval $(...): Avalua la sortida en la sessió actual del shell.

Exemple de Flux de Treball

Aquí tens un exemple de flux de treball per crear, gestionar i utilitzar una màquina Docker:

  1. Crea una màquina Docker: 
    docker-machine create --driver virtualbox my-docker-host
  2. Llista les màquines Docker: 
    docker-machine ls
  3. Configura el client Docker: 
    eval $(docker-machine env my-docker-host)
  4. Executa un contenidor Docker: 
    docker run hello-world
  5. Atura la màquina Docker: 
    docker-machine stop my-docker-host
  6. Elimina la màquina Docker: 
    docker-machine rm my-docker-host

Conclusió

Docker Machine simplifica el procés de crear i gestionar hosts Docker a diverses plataformes. Entenent com instal·lar Docker Machine, crear i gestionar hosts Docker, i configurar el client Docker, pots configurar i utilitzar entorns Docker de manera eficient adaptats a les teves necessitats. Aquest coneixement és essencial tant per a entorns de desenvolupament com de producció, fent de Docker Machine una eina valuosa en el teu conjunt d'eines Docker.
