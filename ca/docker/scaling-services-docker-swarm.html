Introducció a Docker Swarm

Docker Swarm és una eina nativa de clustering i orquestració per a contenidors Docker. Et permet gestionar un clúster de motors Docker com un sistema virtual únic. Aquesta secció cobrirà els conceptes bàsics de Docker Swarm i la seva importància en l'escalat de serveis.

  • Què és Docker Swarm?

    • Docker Swarm és una eina d'orquestració de contenidors que et permet gestionar múltiples hosts Docker com un únic clúster.
    • Proporciona alta disponibilitat, equilibrat de càrrega i capacitats d'escalat.

  • Per què utilitzar Docker Swarm?

    • Simplifica el desplegament i la gestió d'aplicacions contenidoritzades.
    • Assegura alta disponibilitat i tolerància a fallades.
    • Facilita l'escalat fàcil de serveis.

Configuració d'un Clúster Docker Swarm

Abans d'escalar serveis, necessites configurar un clúster Docker Swarm. Aquesta secció et guiarà pel procés d'inicialitzar un Swarm i afegir nodes a aquest.

  • Inicialitzar un Swarm

    docker swarm init --advertise-addr <MANAGER-IP>
    • --advertise-addr: L'adreça IP que el node gestor anuncia als altres nodes.

  • Afegir Nodes Treballadors al Swarm

    docker swarm join --token <SWARM-TOKEN> <MANAGER-IP>:2377
    • --token: El token proporcionat pel node gestor per unir-se al Swarm.
    • <MANAGER-IP>: L'adreça IP del node gestor.

Desplegament de Serveis en Docker Swarm

Un cop el clúster Swarm està configurat, pots desplegar serveis en aquest. Aquesta secció explicarà com crear i gestionar serveis en Docker Swarm.

  • Crear un Servei

    docker service create --name my-service --replicas 3 nginx
    • --name: El nom del servei.
    • --replicas: El nombre d'instàncies (rèpliques) del servei.
    • nginx: La imatge a utilitzar per al servei.

  • Llistar Serveis

    docker service ls

  • Inspeccionar un Servei

    docker service inspect my-service

  • Escalar un Servei

    docker service scale my-service=5
    • Aquesta comanda escala el servei a 5 rèpliques.

Gestió i Monitorització de Serveis

La gestió i monitorització efectiva són crucials per mantenir la salut i el rendiment dels teus serveis. Aquesta secció cobreix comandes i eines essencials per gestionar i monitoritzar serveis en Docker Swarm.

  • Actualitzar un Servei

    docker service update --image nginx:latest my-service
    • Actualitza el servei per utilitzar la imatge més recent de nginx.

  • Eliminar un Servei

    docker service rm my-service

  • Monitoritzar Serveis

    • Utilitza docker service ps per llistar les tasques d'un servei. 
      docker service ps my-service
    • Utilitza docker node ls per llistar tots els nodes en el Swarm. 
      docker node ls

Alta Disponibilitat i Equilibrat de Càrrega

Docker Swarm assegura alta disponibilitat i equilibrat de càrrega de manera nativa. Aquesta secció explica com funcionen aquestes característiques i com configurar-les.

  • Alta Disponibilitat

    • Docker Swarm redistribueix automàticament les tasques dels nodes fallits als nodes saludables.
    • Assegura't de tenir múltiples nodes gestors per a la tolerància a fallades.

  • Equilibrat de Càrrega

    • Docker Swarm utilitza un equilibrador de càrrega intern per distribuir les sol·licituds entre les rèpliques del servei.
    • També pots utilitzar equilibradors de càrrega externs per a configuracions més complexes.

Conclusió

Escalar serveis amb Docker Swarm és una manera poderosa de gestionar aplicacions contenidoritzades de manera eficient. Configurant un clúster Swarm, desplegant i escalant serveis, i aprofitant les característiques d'alta disponibilitat i equilibrat de càrrega, pots assegurar que les teves aplicacions siguin robustes i escalables.

  • Punts Clau:
    • Docker Swarm simplifica l'orquestració d'aplicacions contenidoritzades.
    • Configurar un clúster Swarm implica inicialitzar el Swarm i afegir nodes.
    • Els serveis es poden desplegar, escalar i gestionar fàcilment utilitzant comandes Docker Swarm.
    • Alta disponibilitat i equilibrat de càrrega són característiques integrades de Docker Swarm, assegurant un lliurament de serveis fiable i eficient.

Dominant aquests conceptes, estaràs ben equipat per escalar els teus serveis de manera efectiva utilitzant Docker Swarm.
