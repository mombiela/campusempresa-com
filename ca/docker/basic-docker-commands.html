Introducció

Docker és una plataforma potent per desenvolupar, enviar i executar aplicacions dins de contenidors. Entendre les comandes bàsiques de Docker és essencial per a qualsevol persona que comenci amb Docker. Aquesta secció cobrirà les comandes fonamentals que necessites per començar amb Docker.

Conceptes clau

  • Imatge Docker: Un paquet de programari lleuger, autònom i executable que inclou tot el necessari per executar una peça de programari.
  • Contenidor Docker: Una instància d'execució d'una imatge Docker.
  • Docker Hub: Un repositori basat en el núvol on els usuaris de Docker poden crear, provar, emmagatzemar i distribuir imatges de contenidors.

Instal·lació de Docker

Abans de submergir-te en les comandes de Docker, assegura't que Docker està instal·lat al teu sistema. Pots descarregar i instal·lar Docker des del lloc web oficial de Docker.

Comandes bàsiques de Docker

1. `docker --version`

  • Propòsit: Comprovar la versió de Docker instal·lada.
  • Ús: 
    docker --version
  • Exemple: 
    $ docker --version
Docker version 20.10.7, build f0df350

2. `docker pull`

  • Propòsit: Descarregar una imatge Docker des de Docker Hub.
  • Ús: 
    docker pull <image_name>
  • Exemple: 
    $ docker pull nginx
Using default tag: latest
latest: Pulling from library/nginx
...

3. `docker images`

  • Propòsit: Llistar totes les imatges Docker a la teva màquina local.
  • Ús: 
    docker images
  • Exemple: 
    $ docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
nginx               latest              4bb46517cac3        2 weeks ago         133MB

4. `docker run`

  • Propòsit: Crear i iniciar un nou contenidor a partir d'una imatge.
  • Ús: 
    docker run <image_name>
  • Exemple: 
    $ docker run nginx

5. `docker ps`

  • Propòsit: Llistar tots els contenidors en execució.
  • Ús: 
    docker ps
  • Exemple: 
    $ docker ps
CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS               NAMES
1a2b3c4d5e6f        nginx               "nginx -g 'daemon of…"   5 seconds ago       Up 4 seconds        80/tcp              loving_turing

6. `docker stop`

  • Propòsit: Aturar un contenidor en execució.
  • Ús: 
    docker stop <container_id>
  • Exemple: 
    $ docker stop 1a2b3c4d5e6f

7. `docker rm`

  • Propòsit: Eliminar un contenidor aturat.
  • Ús: 
    docker rm <container_id>
  • Exemple: 
    $ docker rm 1a2b3c4d5e6f

8. `docker rmi`

  • Propòsit: Eliminar una imatge Docker.
  • Ús: 
    docker rmi <image_id>
  • Exemple: 
    $ docker rmi 4bb46517cac3

Resum

En aquesta secció, hem cobert les comandes bàsiques de Docker essencials per a principiants. Aquestes comandes inclouen comprovar la versió de Docker, descarregar imatges, executar contenidors, llistar imatges i contenidors, aturar i eliminar contenidors, i eliminar imatges. Dominar aquestes comandes proporcionarà una base sòlida per treballar amb Docker. A mesura que progressis, aprendràs comandes i conceptes més avançats per gestionar entorns Docker de manera efectiva.
