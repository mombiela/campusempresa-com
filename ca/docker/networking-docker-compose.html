Introducció a les Xarxes de Docker

  • Conceptes Bàsics de Xarxes de Docker: Entendre com Docker gestiona les xarxes per defecte.
  • Xarxa Bridge: Aprendre sobre el tipus de xarxa per defecte en Docker.
  • Xarxa Host: Explorar com els contenidors poden compartir l'espai de noms de xarxa de l'amfitrió.
  • Xarxa Overlay: Descobrir com connectar múltiples daemons de Docker junts.

Configuració de Docker Compose

  • Instal·lació: Passos per instal·lar Docker Compose en diversos sistemes operatius.
  • Comandes Bàsiques: Familiaritzar-se amb les comandes essencials de Docker Compose com up, down, ps i logs.

Definició de Xarxes en Docker Compose

  • Estructura del Fitxer Compose: Entendre l'estructura d'un fitxer docker-compose.yml.
  • Configuració de Xarxes: Aprendre com definir xarxes personalitzades dins d'un fitxer docker-compose.yml.
version: '3'
services:
  web:
    image: nginx
    networks:
      - webnet
  db:
    image: mysql
    networks:
      - webnet

networks:
  webnet:

Explicació:

  • version: Especifica la versió del format del fitxer Docker Compose.
  • services: Defineix els serveis que es volen executar.
    • web: Un servei que utilitza la imatge nginx.
    • db: Un servei que utilitza la imatge mysql.
  • networks: Defineix xarxes personalitzades.
    • webnet: Una xarxa personalitzada que utilitzaran ambdós serveis.

Connexió de Serveis

  • Descobriment de Serveis: Entendre com els serveis poden descobrir-se entre ells utilitzant DNS.
  • Àlies de Xarxa: Aprendre com establir àlies de xarxa per als serveis.
version: '3'
services:
  web:
    image: nginx
    networks:
      webnet:
        aliases:
          - myweb
  db:
    image: mysql
    networks:
      webnet:
        aliases:
          - mydb

networks:
  webnet:

Explicació:

  • aliases: Proporciona noms alternatius per als serveis dins de la xarxa.

Conceptes Avançats de Xarxes

  • Múltiples Xarxes: Aprendre com connectar serveis a múltiples xarxes.
version: '3'
services:
  web:
    image: nginx
    networks:
      - frontend
      - backend
  db:
    image: mysql
    networks:
      - backend

networks:
  frontend:
  backend:

Explicació:

  • frontend: Una xarxa per a serveis de frontend.

  • backend: Una xarxa per a serveis de backend.

  • Controladors de Xarxa: Explorar diferents controladors de xarxa com bridge, host, overlay i macvlan.

| Controlador de Xarxa | Descripció | |----------------------|-------------| | bridge | Controlador de xarxa per defecte, adequat per a contenidors independents. | | host | Comparteix l'espai de noms de xarxa de l'amfitrió. | | overlay | Connecta múltiples daemons de Docker junts. | | macvlan | Assigna una adreça MAC a cada contenidor per a accés directe a la xarxa. |

Exercicis Pràctics

  1. Exercici 1: Crear un fitxer Docker Compose senzill amb dos serveis connectats a través d'una xarxa personalitzada.
  2. Exercici 2: Modificar el fitxer Docker Compose per afegir àlies de xarxa.
  3. Exercici 3: Connectar serveis a múltiples xarxes i observar el comportament.

Conclusió

En aquest tema, has après els fonaments de les xarxes de Docker i com aprofitar Docker Compose per gestionar configuracions de xarxa complexes. Definint xarxes personalitzades, establint àlies de xarxa i connectant serveis a múltiples xarxes, pots crear aplicacions contenidoritzades robustes i escalables. Continua practicant amb els exercicis proporcionats per consolidar la teva comprensió i preparar-te per a escenaris de xarxes de Docker més avançats.
