Kubernetes és una plataforma de codi obert dissenyada per automatitzar el desplegament, l'escalat i l'operació de contenidors d'aplicacions. Funciona perfectament amb Docker, convertint-se en una eina essencial per gestionar aplicacions contenidoritzades en un entorn clúster.

Què és Kubernetes?

Kubernetes, sovint abreujat com K8s, és un sistema d'orquestració de contenidors que ajuda a gestionar aplicacions contenidoritzades a través de múltiples hosts. Proporciona un marc per executar sistemes distribuïts de manera resilient, ocupant-se de l'escalat i la recuperació d'errors de la teva aplicació, proporcionant patrons de desplegament, i més.

Conceptes Clau

  • Clúster: Un conjunt de nodes (màquines) que executen aplicacions contenidoritzades gestionades per Kubernetes.
  • Node: Una sola màquina en el clúster de Kubernetes, que pot ser una màquina física o virtual.
  • Pod: L'objecte més petit i senzill de Kubernetes. Un pod representa una sola instància d'un procés en execució en el teu clúster.
  • Servei: Una abstracció que defineix un conjunt lògic de pods i una política per accedir-hi.
  • Desplegament: Un concepte de nivell superior que gestiona un conjunt de pods idèntics, assegurant que el nombre correcte de pods estiguin en execució i actualitzant-los segons sigui necessari.
  • Espai de noms: Una manera de dividir els recursos del clúster entre múltiples usuaris.

Per què Utilitzar Kubernetes amb Docker?

  • Escalabilitat: Escala automàticament les teves aplicacions amunt i avall segons la demanda.
  • Alta Disponibilitat: Assegura que les teves aplicacions estiguin sempre en execució amb capacitats d'auto-recuperació.
  • Optimització de Recursos: Utilitza eficientment els teus recursos de maquinari per maximitzar el rendiment.
  • Portabilitat: Mou fàcilment aplicacions a través de diferents entorns (desenvolupament, proves, producció).

Configuració d'un Clúster de Kubernetes

Per començar amb Kubernetes, necessites configurar un clúster. Això es pot fer localment utilitzant eines com Minikube o al núvol utilitzant serveis com Google Kubernetes Engine (GKE), Amazon Elastic Kubernetes Service (EKS) o Azure Kubernetes Service (AKS).

Exemple: Configuració de Minikube

Minikube és una eina que et permet executar Kubernetes localment. Crea un clúster de Kubernetes d'un sol node a la teva màquina local.

  1. Instal·lar Minikube:

    curl -Lo minikube https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
chmod +x minikube
sudo mv minikube /usr/local/bin/

  2. Iniciar Minikube:

    minikube start

  3. Verificar la Instal·lació:

    kubectl get nodes

Desplegament d'un Contenidor Docker amb Kubernetes

Despleguem un contenidor Docker senzill utilitzant Kubernetes. Utilitzarem un contenidor bàsic de Nginx per a aquest exemple.

  1. Crear un Desplegament:

    kubectl create deployment nginx --image=nginx

  2. Exposar el Desplegament com un Servei:

    kubectl expose deployment nginx --port=80 --type=NodePort

  3. Obtenir l'URL per Accedir al Servei:

    minikube service nginx --url

Gestió de Recursos de Kubernetes

Escalat d'Aplicacions

Kubernetes et permet escalar les teves aplicacions fàcilment. Per exemple, per escalar el desplegament de Nginx a 3 rèpliques:

kubectl scale deployment nginx --replicas=3

Actualització d'Aplicacions

Per actualitzar el desplegament de Nginx per utilitzar una versió d'imatge diferent:

kubectl set image deployment/nginx nginx=nginx:1.19.0

Monitorització i Registre

Kubernetes proporciona eines integrades per a la monitorització i el registre. Pots veure els registres d'un pod específic utilitzant:

kubectl logs <pod-name>

Per obtenir informació detallada sobre un recurs específic:

kubectl describe <resource-type> <resource-name>

Conclusió

Kubernetes és una eina poderosa que complementa Docker proporcionant capacitats robustes d'orquestració. Automatitza moltes de les tasques complexes involucrades en la gestió d'aplicacions contenidoritzades, com l'escalat, les actualitzacions i la recuperació d'errors. Entenent els conceptes bàsics de Kubernetes, pots millorar significativament la teva capacitat per desplegar i gestionar aplicacions en un entorn de producció.
