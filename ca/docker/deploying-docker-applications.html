El desplegament d'aplicacions Docker implica empaquetar la teva aplicació i les seves dependències en un contenidor Docker i després executar aquest contenidor en una màquina host. Aquesta secció et guiarà a través del procés des de nivells principiants fins a avançats.

Introducció al Desplegament Docker

  • Què és el Desplegament Docker?

    • El procés d'executar la teva aplicació en un contenidor Docker.
    • Assegura la consistència entre diferents entorns (desenvolupament, proves, producció).

  • Per què Desplegar amb Docker?

    • Simplifica la gestió de dependències.
    • Millora l'escalabilitat i la portabilitat.
    • Facilita la integració contínua i el desplegament continu (CI/CD).

Configurant el Teu Entorn

  • Prerequisits:

    • Docker instal·lat a la teva màquina.
    • Comprensió bàsica dels conceptes de Docker (imatges, contenidors, Dockerfile).

  • Instal·lant Docker:

    • Segueix la guia oficial d'instal·lació de Docker per al teu sistema operatiu: Docker Installation

Creant un Dockerfile

Un Dockerfile és un script que conté una sèrie d'instruccions sobre com construir una imatge Docker per a la teva aplicació.

  • Estructura Bàsica del Dockerfile:

    # Utilitza una imatge oficial de Python com a imatge pare
FROM python:3.8-slim

# Estableix el directori de treball dins del contenidor
WORKDIR /app

# Copia el contingut del directori actual al contenidor a /app
COPY . /app

# Instal·la qualsevol paquet necessari especificat a requirements.txt
RUN pip install --no-cache-dir -r requirements.txt

# Fa que el port 80 estigui disponible per al món fora d'aquest contenidor
EXPOSE 80

# Defineix una variable d'entorn
ENV NAME World

# Executa app.py quan el contenidor es llanci
CMD ["python", "app.py"]

  • Explicació:

    • FROM: Especifica la imatge base.
    • WORKDIR: Estableix el directori de treball dins del contenidor.
    • COPY: Copia fitxers de l'host al contenidor.
    • RUN: Executa comandes dins del contenidor.
    • EXPOSE: Exposa un port.
    • ENV: Estableix variables d'entorn.
    • CMD: Especifica la comanda a executar dins del contenidor.

Construint i Executant la Teva Imatge Docker

  • Construeix la Imatge Docker:

    docker build -t my-python-app .
    • -t my-python-app: Etiqueta la imatge amb el nom my-python-app.
    • .: Especifica el directori actual com a context de construcció.

  • Executa el Contenidor Docker:

    docker run -p 4000:80 my-python-app
    • -p 4000:80: Assigna el port 4000 de l'host al port 80 del contenidor.
    • my-python-app: El nom de la imatge a executar.

Estratègies Avançades de Desplegament

  • Docker Compose:

    • Eina per definir i executar aplicacions Docker de múltiples contenidors.
    • Exemple docker-compose.yml: 
      version: '3'
services:
  web:
    build: .
    ports:
      - "4000:80"
    volumes:
      - .:/code
    environment:
      FLASK_ENV: development
  redis:
    image: "redis:alpine"
    • Comandes: 
      docker-compose up
docker-compose down

  • Escalant Serveis:

    • Utilitza Docker Compose per escalar serveis.
    • Exemple: 
      docker-compose up --scale web=3

  • Desplegant a Producció:

    • Utilitza eines d'orquestració com Kubernetes o Docker Swarm.
    • Assegura una configuració adequada per a la seguretat, xarxes i emmagatzematge.

Monitorització i Registre

  • Eines de Monitorització:

    • Prometheus
    • Grafana
    • Docker Stats

  • Registre:

    • Utilitza controladors de registre Docker (json-file, syslog, journald).
    • Exemple: 
      docker run --log-driver syslog my-python-app

Conclusió

El desplegament d'aplicacions Docker implica crear un Dockerfile, construir una imatge i executar contenidors. Les estratègies avançades inclouen l'ús de Docker Compose per a aplicacions de múltiples contenidors i l'escalat de serveis. La monitorització i el registre són crucials per mantenir la salut i el rendiment de les teves aplicacions. Dominant aquests conceptes, pots assegurar que les teves aplicacions es despleguin consistentment en diferents entorns, millorant la fiabilitat i l'escalabilitat.
