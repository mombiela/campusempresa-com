Introducció

En aquesta secció, et guiarem a través de la creació de la teva primera aplicació React Native. Aquesta serà una aplicació simple de "Hola Món", que és un punt de partida tradicional per aprendre qualsevol nou llenguatge de programació o marc. Al final d'aquesta secció, tindràs una comprensió bàsica de com configurar un entorn React Native i crear una aplicació simple.

Configuració de l'Entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

Prerequisits

  • Node.js i npm (Node Package Manager)
  • React Native CLI o Expo CLI
  • Un editor de codi (per exemple, Visual Studio Code)
  • Android Studio o Xcode (per executar l'aplicació en un emulador)

Passos d'Instal·lació

  1. Instal·lar Node.js i npm: Descarrega i instal·la Node.js des de nodejs.org. npm està inclòs amb Node.js.

  2. Instal·lar React Native CLI: Obre el teu terminal i executa:

    npm install -g react-native-cli

  3. Instal·lar Expo CLI (Opcional): Si prefereixes utilitzar Expo, executa:

    npm install -g expo-cli

  4. Configurar Android Studio o Xcode:

    • Per Android: Descarrega i instal·la Android Studio des de developer.android.com.
    • Per iOS: Assegura't de tenir Xcode instal·lat des de la Mac App Store.

Creant la Teva Primera Aplicació React Native

Ara que el nostre entorn està configurat, creem la nostra primera aplicació.

Utilitzant React Native CLI

  1. Crear un Nou Projecte:

    npx react-native init HelloWorldApp

    Aquesta comanda crea un nou projecte React Native anomenat HelloWorldApp.

  2. Navegar al Directori del Projecte:

    cd HelloWorldApp

  3. Executar l'Aplicació:

    • Per Android: 
      npx react-native run-android
    • Per iOS: 
      npx react-native run-ios

Utilitzant Expo CLI

  1. Crear un Nou Projecte:

    expo init HelloWorldApp

    Tria una plantilla (per exemple, en blanc) i segueix les instruccions.

  2. Navegar al Directori del Projecte:

    cd HelloWorldApp

  3. Executar l'Aplicació:

    expo start

    Això obrirà una nova pestanya al teu navegador amb un codi QR. Escaneja el codi QR amb l'aplicació Expo Go al teu dispositiu mòbil per executar l'aplicació.

Escriure el Codi "Hola Món"

Ara, modifiquem el codi per defecte per mostrar "Hola Món".

Editant el Component App

Obre App.js al teu editor de codi i reemplaça el seu contingut amb el següent codi:

import React from 'react';
import { Text, View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Hello World</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  text: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
});

export default App;

Explicació

  • Imports: Importem React, Text, View, i StyleSheet de react-native.
  • Component App: Aquest és un component funcional que retorna un View que conté un element Text.
  • Estils: Definim alguns estils bàsics utilitzant StyleSheet.create.

Executant l'Aplicació

Després de desar els teus canvis, la teva aplicació s'hauria de recarregar automàticament i mostrar "Hello World" a la pantalla.

Conclusió

Felicitats! Has creat amb èxit la teva primera aplicació React Native. En aquesta secció, has après a:

  • Configurar el teu entorn de desenvolupament.
  • Crear un nou projecte React Native utilitzant tant React Native CLI com Expo CLI.
  • Escriure i executar una aplicació simple de "Hola Món".

Aquest coneixement fonamental serà crucial a mesura que avancem cap a temes més avançats en el desenvolupament de React Native.
