Introducció al Navegador de Pestanyes

La navegació per pestanyes és un patró comú en les aplicacions mòbils on els usuaris poden canviar entre diferents pantalles o vistes tocant les pestanyes. En React Native, la biblioteca react-navigation proporciona una manera potent i flexible d'implementar la navegació per pestanyes.

Configuració de l'Entorn

Abans de submergir-nos en la creació d'un Navegador de Pestanyes, assegura't de tenir les dependències necessàries instal·lades.

  1. Instal·la React Navigation:

    npm install @react-navigation/native

  2. Instal·la les dependències requerides:

    npm install @react-navigation/bottom-tabs
npm install react-native-screens react-native-safe-area-context

  3. Embolica la teva aplicació en NavigationContainer:

    import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';

export default function App() {
  return (
    <NavigationContainer>
      {/* Resta del codi de la teva aplicació */}
    </NavigationContainer>
  );
}

Creació d'un Navegador de Pestanyes Bàsic

Per crear un navegador de pestanyes bàsic, segueix aquests passos:

  1. Importa els mòduls necessaris:

    import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import HomeScreen from './screens/HomeScreen';
import SettingsScreen from './screens/SettingsScreen';

  2. Crea el Navegador de Pestanyes:

    const Tab = createBottomTabNavigator();

function MyTabs() {
  return (
    <Tab.Navigator>
      <Tab.Screen name="Home" component={HomeScreen} />
      <Tab.Screen name="Settings" component={SettingsScreen} />
    </Tab.Navigator>
  );
}

  3. Utilitza el Navegador de Pestanyes en la teva Aplicació:

    export default function App() {
  return (
    <NavigationContainer>
      <MyTabs />
    </NavigationContainer>
  );
}

Personalització del Navegador de Pestanyes

Pots personalitzar l'aparença i el comportament del Navegador de Pestanyes utilitzant diverses opcions.

Opcions de la Barra de Pestanyes

  • tabBarOptions: Personalitza l'aparença de la barra de pestanyes. 
    <Tab.Navigator
  tabBarOptions={{
    activeTintColor: 'tomato',
    inactiveTintColor: 'gray',
  }}
>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Opcions de Pantalla

  • tabBarIcon: Afegeix icones a les pestanyes. 
    import { Ionicons } from '@expo/vector-icons';

<Tab.Navigator
  screenOptions={({ route }) => ({
    tabBarIcon: ({ focused, color, size }) => {
      let iconName;

      if (route.name === 'Home') {
        iconName = focused ? 'home' : 'home-outline';
      } else if (route.name === 'Settings') {
        iconName = focused ? 'settings' : 'settings-outline';
      }

      return <Ionicons name={iconName} size={size} color={color} />;
    },
  })}
  tabBarOptions={{
    activeTintColor: 'tomato',
    inactiveTintColor: 'gray',
  }}
>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Personalització Avançada

Per a una personalització més avançada, pots utilitzar components personalitzats per a la barra de pestanyes.

Component Personalitzat de la Barra de Pestanyes

  • Creació d'una barra de pestanyes personalitzada: 
    import { View, Text, TouchableOpacity } from 'react-native';

function MyTabBar({ state, descriptors, navigation }) {
  return (
    <View style={{ flexDirection: 'row' }}>
      {state.routes.map((route, index) => {
        const { options } = descriptors[route.key];
        const label =
          options.tabBarLabel !== undefined
            ? options.tabBarLabel
            : options.title !== undefined
            ? options.title
            : route.name;

        const isFocused = state.index === index;

        const onPress = () => {
          const event = navigation.emit({
            type: 'tabPress',
            target: route.key,
          });

          if (!isFocused && !event.defaultPrevented) {
            navigation.navigate(route.name);
          }
        };

        const onLongPress = () => {
          navigation.emit({
            type: 'tabLongPress',
            target: route.key,
          });
        };

        return (
          <TouchableOpacity
            key={index}
            accessibilityRole="button"
            accessibilityStates={isFocused ? ['selected'] : []}
            accessibilityLabel={options.tabBarAccessibilityLabel}
            testID={options.tabBarTestID}
            onPress={onPress}
            onLongPress={onLongPress}
            style={{ flex: 1 }}
          >
            <Text style={{ color: isFocused ? '#673ab7' : '#222' }}>
              {label}
            </Text>
          </TouchableOpacity>
        );
      })}
    </View>
  );
}

<Tab.Navigator tabBar={props => <MyTabBar {...props} />}>
  <Tab.Screen name="Home" component={HomeScreen} />
  <Tab.Screen name="Settings" component={SettingsScreen} />
</Tab.Navigator>

Conclusió

La navegació per pestanyes és una part essencial de moltes aplicacions mòbils, proporcionant una manera fàcil d'usar per canviar entre diferents pantalles. El Navegador de Pestanyes de React Navigation ofereix una solució flexible i personalitzable per implementar la navegació per pestanyes en les teves aplicacions React Native. Comprenent els conceptes bàsics i explorant les opcions de personalització avançades, pots crear una experiència de navegació fluida i intuïtiva per als teus usuaris.
