Configuració de l'Entorn de Desenvolupament
Hola Món: La teva primera aplicació React Native ►

Configurar l'entorn de desenvolupament és el primer pas crucial en el teu viatge per convertir-te en un expert en React Native. Aquesta secció et guiarà a través del procés de configuració del teu entorn en diferents sistemes operatius, assegurant que tinguis totes les eines necessàries per començar a construir aplicacions React Native.

Prerequisits

Abans de submergir-te en la configuració, assegura't de tenir els següents prerequisits:

  • Coneixements bàsics de JavaScript.
  • Un editor de codi (es recomana Visual Studio Code).
  • Node.js instal·lat a la teva màquina.

Instal·lació de Node.js i npm

Node.js és un entorn d'execució de JavaScript que et permet executar JavaScript al costat del servidor. npm (Node Package Manager) ve amb Node.js i t'ajuda a gestionar les dependències.

  1. Descarrega i instal·la Node.js:

    • Ves al lloc web de Node.js.
    • Descarrega la versió LTS per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Verifica la instal·lació:

    node -v
npm -v

Configuració de React Native CLI

React Native CLI és una interfície de línia de comandes que t'ajuda a crear, construir i executar projectes React Native.

  1. Instal·la React Native CLI:

    npm install -g react-native-cli

  2. Verifica la instal·lació:

    react-native --version

Configuració de l'Entorn de Desenvolupament a macOS

Instal·lació de Xcode

Xcode és necessari per al desenvolupament d'iOS.

  1. Descarrega Xcode:

    • Obre l'App Store al teu Mac.
    • Cerca Xcode i instal·la'l.

  2. Instal·la les eines de línia de comandes:

    xcode-select --install

Instal·lació d'Android Studio

Android Studio és necessari per al desenvolupament d'Android.

  1. Descarrega Android Studio:

    • Ves al lloc web d'Android Studio.
    • Descarrega i instal·la Android Studio.

  2. Configura l'SDK d'Android:

    • Obre Android Studio.
    • Ves a Preferences > Appearance & Behavior > System Settings > Android SDK.
    • Selecciona la pestanya SDK Platforms i assegura't que Android 10.0 (Q) estigui marcat.
    • Selecciona la pestanya SDK Tools i assegura't que Android SDK Build-Tools, Android Emulator i Android SDK Platform-Tools estiguin marcats.

Configuració de les Variables d'Entorn

  1. Obre Terminal i edita el perfil:

    nano ~/.bash_profile

  2. Afegeix les següents línies:

    export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools

  3. Guarda i tanca:

    • Prem CTRL + X, després Y, i Enter.

  4. Aplica els canvis:

    source ~/.bash_profile

Configuració de l'Entorn de Desenvolupament a Windows

Instal·lació d'Android Studio

  1. Descarrega Android Studio:

    • Ves al lloc web d'Android Studio.
    • Descarrega i instal·la Android Studio.

  2. Configura l'SDK d'Android:

    • Obre Android Studio.
    • Ves a File > Settings > Appearance & Behavior > System Settings > Android SDK.
    • Selecciona la pestanya SDK Platforms i assegura't que Android 10.0 (Q) estigui marcat.
    • Selecciona la pestanya SDK Tools i assegura't que Android SDK Build-Tools, Android Emulator i Android SDK Platform-Tools estiguin marcats.

Configuració de les Variables d'Entorn

  1. Obre les Variables d'Entorn:

    • Fes clic amb el botó dret a Aquest PC > Propietats > Configuració avançada del sistema.
    • Fes clic a Variables d'entorn.

  2. Afegeix noves variables de sistema:

    • Fes clic a Nou sota Variables del sistema.
    • Afegeix ANDROID_HOME amb la ruta al teu SDK d'Android (per exemple, C:\Users\<El-Teu-Nom-Usuari>\AppData\Local\Android\Sdk).
    • Afegeix entrades PATH per a platform-tools i tools.

Creació del Teu Primer Projecte React Native

  1. Crea un nou projecte:

    npx react-native init MyFirstProject

  2. Navega al directori del projecte:

    cd MyFirstProject

  3. Executa el projecte:

    • Per a iOS: 
      npx react-native run-ios
    • Per a Android: 
      npx react-native run-android

Conclusió

Configurar l'entorn de desenvolupament per a React Native pot semblar intimidant al principi, però seguint aquests passos asseguraràs que tens tot el que necessites per començar a construir les teves aplicacions. Tant si estàs a macOS com a Windows, tenir Node.js, React Native CLI i les eines necessàries per al desenvolupament d'iOS i Android et posarà en el camí de l'èxit. Feliç codificació!
