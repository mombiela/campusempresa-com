Redux és un contenidor d'estat previsible per a aplicacions JavaScript, sovint utilitzat amb biblioteques com React i React Native per gestionar l'estat de l'aplicació. Aquest tema t'introduirà a Redux, els seus conceptes bàsics i com integrar-lo en una aplicació React Native.

Què és Redux?

Redux és una biblioteca de gestió d'estat que t'ajuda a gestionar l'estat de la teva aplicació d'una manera previsible. És particularment útil per a aplicacions amb interaccions d'estat complexes i fluxos de dades.

Conceptes Clau de Redux

  • Store: La font única de veritat que conté l'estat de tota la teva aplicació.
  • Actions: Objectes JavaScript simples que descriuen què ha passat a l'aplicació.
  • Reducers: Funcions que prenen l'estat actual i una acció, i retornen un nou estat.
  • Dispatch: Un mètode utilitzat per enviar accions a la store.
  • Selectors: Funcions que extreuen peces específiques de l'estat de la store.

Configurant Redux en un Projecte React Native

Per començar amb Redux en un projecte React Native, necessites instal·lar els paquets necessaris:

npm install redux react-redux

Creant la Store

La store és la peça central de Redux. Conté l'estat de la teva aplicació.

import { createStore } from 'redux';
import rootReducer from './reducers';

const store = createStore(rootReducer);

Definint Accions

Les accions són objectes JavaScript simples que descriuen què ha passat a la teva aplicació.

// actions.js
export const INCREMENT = 'INCREMENT';
export const DECREMENT = 'DECREMENT';

export const increment = () => ({
  type: INCREMENT,
});

export const decrement = () => ({
  type: DECREMENT,
});

Creant Reducers

Els reducers especifiquen com canvia l'estat de l'aplicació en resposta a les accions.

// counterReducer.js
import { INCREMENT, DECREMENT } from './actions';

const initialState = {
  count: 0,
};

const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case INCREMENT:
      return { ...state, count: state.count + 1 };
    case DECREMENT:
      return { ...state, count: state.count - 1 };
    default:
      return state;
  }
};

export default counterReducer;

Combinant Reducers

Si tens múltiples reducers, pots combinar-los utilitzant combineReducers.

// reducers/index.js
import { combineReducers } from 'redux';
import counterReducer from './counterReducer';

const rootReducer = combineReducers({
  counter: counterReducer,
});

export default rootReducer;

Proveint la Store a la Teva Aplicació

Utilitza el component Provider de react-redux per fer que la store estigui disponible a tota la teva aplicació.

// App.js
import React from 'react';
import { Provider } from 'react-redux';
import { createStore } from 'redux';
import rootReducer from './reducers';
import Counter from './Counter';

const store = createStore(rootReducer);

const App = () => (
  <Provider store={store}>
    <Counter />
  </Provider>
);

export default App;

Connectant Components a la Store

Utilitza la funció connect de react-redux per connectar els teus components a la store de Redux.

// Counter.js
import React from 'react';
import { connect } from 'react-redux';
import { increment, decrement } from './actions';

const Counter = ({ count, increment, decrement }) => (
  <View>
    <Text>{count}</Text>
    <Button onPress={increment} title="Incrementar" />
    <Button onPress={decrement} title="Decrementar" />
  </View>
);

const mapStateToProps = (state) => ({
  count: state.counter.count,
});

const mapDispatchToProps = {
  increment,
  decrement,
};

export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Resum

En aquest tema, hem cobert els conceptes bàsics de Redux i com integrar-lo en una aplicació React Native. Hem discutit els conceptes clau de Redux, incloent la store, les accions, els reducers, i com connectar components a la store. Seguint aquests passos, pots gestionar l'estat de les teves aplicacions React Native d'una manera previsible i escalable.
