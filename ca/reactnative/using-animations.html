Introducció a les Animacions en React Native

Les animacions poden millorar significativament l'experiència de l'usuari en aplicacions mòbils proporcionant retroalimentació visual, millorant la navegació i fent l'aplicació més atractiva. React Native ofereix diverses maneres de crear animacions, des de transicions simples fins a seqüències complexes.

Conceptes Clau

  • Animated API: La biblioteca principal per crear animacions en React Native.
  • LayoutAnimation: Utilitzat per animar canvis de disseny.
  • Reanimated: Una biblioteca més avançada per gestionar animacions, oferint millor rendiment i més funcions.

Animated API

L'API Animated en React Native és una eina poderosa per crear animacions suaus i interactives.

Exemple Bàsic

Comencem amb un exemple senzill d'animar l'opacitat d'una vista.

import React, { useRef, useEffect } from 'react';
import { Animated, View, Button, StyleSheet } from 'react-native';

const FadeInView = () => {
  const fadeAnim = useRef(new Animated.Value(0)).current;

  useEffect(() => {
    Animated.timing(fadeAnim, {
      toValue: 1,
      duration: 2000,
      useNativeDriver: true,
    }).start();
  }, [fadeAnim]);

  return (
    <Animated.View style={{ ...styles.fadingContainer, opacity: fadeAnim }}>
      <View style={styles.fadingView} />
    </Animated.View>
  );
};

const styles = StyleSheet.create({
  fadingContainer: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  fadingView: {
    width: 100,
    height: 100,
    backgroundColor: 'blue',
  },
});

export default FadeInView;

Explicació

  • useRef: Utilitzat per crear una referència al valor animat.
  • Animated.Value: Inicialitza el valor animat.
  • Animated.timing: Defineix el tipus d'animació i les propietats.
  • useNativeDriver: Millora el rendiment descarregant les animacions al fil natiu.

LayoutAnimation

LayoutAnimation és útil per animar canvis de disseny, com afegir o eliminar elements.

Exemple

import React, { useState } from 'react';
import { View, Button, LayoutAnimation, StyleSheet } from 'react-native';

const LayoutAnimationExample = () => {
  const [boxes, setBoxes] = useState([1]);

  const addBox = () => {
    LayoutAnimation.configureNext(LayoutAnimation.Presets.spring);
    setBoxes([...boxes, boxes.length + 1]);
  };

  return (
    <View style={styles.container}>
      <Button title="Add Box" onPress={addBox} />
      {boxes.map((box, index) => (
        <View key={index} style={styles.box} />
      ))}
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  box: {
    width: 50,
    height: 50,
    backgroundColor: 'red',
    margin: 10,
  },
});

export default LayoutAnimationExample;

Explicació

  • LayoutAnimation.configureNext: Configura el següent canvi de disseny per ser animat.
  • LayoutAnimation.Presets.spring: Utilitza un preset d'animació de molla per a un efecte natural.

Animacions Avançades amb Reanimated

Reanimated és una biblioteca més avançada que ofereix millor rendiment i més funcions per a animacions complexes.

Exemple

import React from 'react';
import { View, Button, StyleSheet } from 'react-native';
import Animated, { Easing } from 'react-native-reanimated';

const { Value, timing } = Animated;

const ReanimatedExample = () => {
  const opacity = new Value(0);

  const startAnimation = () => {
    timing(opacity, {
      toValue: 1,
      duration: 2000,
      easing: Easing.inOut(Easing.ease),
    }).start();
  };

  return (
    <View style={styles.container}>
      <Button title="Start Animation" onPress={startAnimation} />
      <Animated.View style={{ ...styles.animatedView, opacity }} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  animatedView: {
    width: 100,
    height: 100,
    backgroundColor: 'green',
  },
});

export default ReanimatedExample;

Explicació

  • Value: Inicialitza el valor animat.
  • timing: Defineix l'animació de temps.
  • Easing: Proporciona funcions d'easing per a animacions suaus.

Conclusió

Les animacions en React Native poden millorar enormement l'experiència de l'usuari fent l'aplicació més interactiva i visualment atractiva. Començant amb l'API bàsica Animated, pots crear animacions senzilles, mentre que LayoutAnimation ajuda amb els canvis de disseny. Per a animacions més complexes i amb millor rendiment, considera utilitzar la biblioteca Reanimated. Practica aquests exemples i explora la documentació per dominar les animacions en React Native.
