Introducció a TypeScript

TypeScript és un superconjunt tipat de JavaScript que es compila a JavaScript pla. Ofereix tipatge estàtic, que pot ajudar a detectar errors aviat en el procés de desenvolupament i millorar la qualitat del codi.

  • Tipatge Estàtic: TypeScript et permet definir tipus per a variables, paràmetres de funcions i valors de retorn.
  • Inferència de Tipus: TypeScript pot inferir tipus basant-se en els valors assignats a les variables.
  • Interfícies i Tipus: Defineix tipus personalitzats i interfícies per descriure la forma dels objectes.
  • Eines: Suport millorat per a eines amb editors com VS Code.

Configuració de TypeScript en un Projecte React Native

Per començar a utilitzar TypeScript en un projecte React Native, cal configurar l'entorn correctament.

Configuració Pas a Pas

  1. Inicialitza un Projecte React Native:

    npx react-native init MyTypeScriptApp
cd MyTypeScriptApp

  2. Instal·la TypeScript i les Tipificacions Necessàries:

    npm install --save-dev typescript @types/react @types/react-native

  3. Afegeix un Arxiu tsconfig.json: Crea un arxiu tsconfig.json a l'arrel del teu projecte:

    {
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "jsx": "react",
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "exclude": ["node_modules", "babel.config.js", "metro.config.js", "jest.config.js"]
}

  4. Reanomena Arxius a TypeScript: Reanomena els teus arxius JavaScript (.js) a arxius TypeScript (.tsx per a arxius que contenen JSX, .ts per a arxius TypeScript plans).

Exemple: Convertir un Component a TypeScript

Convertim un component simple de React Native de JavaScript a TypeScript.

Versió JavaScript

import React from 'react';
import { View, Text } from 'react-native';

const HelloWorld = () => {
  return (
    <View>
      <Text>Hello, World!</Text>
    </View>
  );
};

export default HelloWorld;

Versió TypeScript

import React from 'react';
import { View, Text } from 'react-native';

const HelloWorld: React.FC = () => {
  return (
    <View>
      <Text>Hello, World!</Text>
    </View>
  );
};

export default HelloWorld;
  • React.FC: Aquest és un tipus proporcionat per TypeScript per a components funcionals.

Funcionalitats de TypeScript en React Native

Props i Estat

TypeScript et permet definir tipus per a props i estat, fent els teus components més robustos.

Exemple: Tipatge de Props

import React from 'react';
import { View, Text } from 'react-native';

interface HelloWorldProps {
  name: string;
}

const HelloWorld: React.FC<HelloWorldProps> = ({ name }) => {
  return (
    <View>
      <Text>Hello, {name}!</Text>
    </View>
  );
};

export default HelloWorld;

Utilitzant Interfícies i Tipus

Les interfícies i tipus es poden utilitzar per definir la forma dels objectes i assegurar la seguretat de tipus.

Exemple: Definint una Interfície

interface User {
  id: number;
  name: string;
  email: string;
}

const user: User = {
  id: 1,
  name: 'John Doe',
  email: 'john.doe@example.com'
};

Inferència de Tipus

TypeScript pot inferir tipus, reduint la necessitat d'anotacions de tipus explícites.

Exemple: Inferència de Tipus

let count = 0; // TypeScript infereix el tipus com a número
count = 'a'; // Error: El tipus 'string' no es pot assignar al tipus 'number'

Funcionalitats Avançades de TypeScript

Genèrics

Els genèrics et permeten crear components i funcions reutilitzables que funcionen amb qualsevol tipus de dades.

Exemple: Funció Genèrica

function identity<T>(arg: T): T {
  return arg;
}

const num = identity<number>(42); // num és del tipus número
const str = identity<string>('Hello'); // str és del tipus string

Tipus d'Utilitat

TypeScript proporciona diversos tipus d'utilitat per facilitar transformacions de tipus comunes.

Exemple: Partial i Pick

interface User {
  id: number;
  name: string;
  email: string;
}

type PartialUser = Partial<User>; // Totes les propietats són opcionals
type UserName = Pick<User, 'name'>; // Només la propietat 'name'

Conclusió

Seguint els passos de configuració i aprofitant les funcionalitats de TypeScript, pots construir aplicacions React Native robustes i mantenibles.
