Introducció

En aquest curs, construirem una App de Tasques Pendents utilitzant React Native. Aquest projecte t'ajudarà a entendre els fonaments de React Native i com aplicar-los per crear una aplicació mòbil funcional. Començarem des dels conceptes bàsics i gradualment avançarem cap a temes més avançats.

Configuració de l'Entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar Node.js: Assegura't de tenir Node.js instal·lat. Pots descarregar-lo des de nodejs.org.
  • Instal·lar Expo CLI: Expo és un framework i plataforma per a aplicacions React universals. Instal·la'l utilitzant npm: 
    npm install -g expo-cli
  • Inicialitzar un Nou Projecte: Crea un nou projecte React Native utilitzant Expo CLI: 
    expo init ToDoApp
cd ToDoApp
expo start

Estructura Bàsica d'una App React Native

Entendre l'estructura bàsica d'una app React Native és crucial.

  • App.js: El punt d'entrada principal de l'aplicació.
  • Components: Peces reutilitzables de la interfície d'usuari.
  • Gestió de l'Estat: Gestionar l'estat de l'aplicació.

Exemple: Estructura Bàsica de l'App

import React from 'react';
import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text>Benvingut a l'App de Tasques Pendents!</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center',
  },
});

Crear el Component de la Llista de Tasques

Crearem un component de Llista de Tasques que mostrarà les nostres tasques.

  • Gestió de l'Estat: Utilitza el hook useState de React per gestionar la llista de tasques.
  • Renderitzar la Llista: Utilitza FlatList per renderitzar la llista de tasques.

Exemple: Component de la Llista de Tasques

import React, { useState } from 'react';
import { View, Text, FlatList, StyleSheet } from 'react-native';

const ToDoList = () => {
  const [tasks, setTasks] = useState([
    { key: '1', task: 'Comprar queviures' },
    { key: '2', task: 'Passejar el gos' },
    { key: '3', task: 'Llegir un llibre' },
  ]);

  return (
    <View style={styles.container}>
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoList;

Afegir Noves Tasques

Necessitem una manera d'afegir noves tasques a la nostra llista.

  • TextInput: Utilitza TextInput per capturar l'entrada de l'usuari.
  • Button: Utilitza Button per afegir la tasca a la llista.

Exemple: Afegir Noves Tasques

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Afegeix una nova tasca"
        value={text}
        onChangeText={setText}
      />
      <Button title="Afegeix Tasca" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;

Eliminar Tasques

Per fer la nostra app més funcional, necessitem afegir la capacitat d'eliminar tasques.

  • TouchableOpacity: Utilitza TouchableOpacity per fer cada element de la tasca clicable.
  • Funció de Filtre: Utilitza la funció de filtre per eliminar la tasca de la llista.

Exemple: Eliminar Tasques

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, TouchableOpacity, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  const deleteTask = (key) => {
    setTasks(tasks.filter(task => task.key !== key));
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Afegeix una nova tasca"
        value={text}
        onChangeText={setText}
      />
      <Button title="Afegeix Tasca" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => (
          <TouchableOpacity onPress={() => deleteTask(item.key)}>
            <Text style={styles.item}>{item.task}</Text>
          </TouchableOpacity>
        )}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;

Estilitzar l'App
