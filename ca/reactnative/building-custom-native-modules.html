Introducció

En aquesta secció, explorarem com crear mòduls natius personalitzats en React Native. Els mòduls natius et permeten ampliar la funcionalitat de la teva aplicació React Native escrivint codi natiu en Java (per a Android) o Objective-C/Swift (per a iOS). Això és particularment útil quan necessites accedir a APIs específiques de la plataforma o optimitzar el rendiment per a certes tasques.

Prerequisits

Abans d'endinsar-te en la creació de mòduls natius personalitzats, assegura't de tenir:

  • Coneixement bàsic de React Native.
  • Familiaritat amb JavaScript i React.
  • Coneixement bàsic de Java (per a Android) o Objective-C/Swift (per a iOS).

Configuració de l'Entorn

Per crear mòduls natius personalitzats, necessites configurar el teu entorn de desenvolupament tant per a Android com per a iOS.

Configuració d'Android

  1. Instal·la Android Studio: Assegura't de tenir Android Studio instal·lat i configurat.
  2. Configura Gradle: Assegura't que els teus fitxers android/build.gradle i android/app/build.gradle estiguin configurats correctament.

Configuració d'iOS

  1. Instal·la Xcode: Assegura't de tenir Xcode instal·lat.
  2. CocoaPods: Assegura't que CocoaPods estigui instal·lat i configurat.

Creació d'un Mòdul Natiu Personalitzat per a Android

Pas 1: Crea una Nova Classe Java

Crea una nova classe Java en el teu projecte Android. Aquesta classe definirà el mòdul natiu.

package com.yourapp;

import com.facebook.react.bridge.ReactApplicationContext;
import com.facebook.react.bridge.ReactContextBaseJavaModule;
import com.facebook.react.bridge.ReactMethod;
import com.facebook.react.bridge.Callback;

public class MyCustomModule extends ReactContextBaseJavaModule {

    MyCustomModule(ReactApplicationContext context) {
        super(context);
    }

    @Override
    public String getName() {
        return "MyCustomModule";
    }

    @ReactMethod
    public void customMethod(String message, Callback callback) {
        // Codi natiu aquí
        callback.invoke("Received message: " + message);
    }
}

Pas 2: Registra el Mòdul

Registra el mòdul en el teu MainApplication.java.

import com.yourapp.MyCustomModule; // Importa el teu mòdul

@Override
protected List<ReactPackage> getPackages() {
    return Arrays.<ReactPackage>asList(
        new MainReactPackage(),
        new MyCustomPackage() // Registra el teu paquet
    );
}

Pas 3: Crea un Paquet React

Crea un paquet per exposar el mòdul a JavaScript.

package com.yourapp;

import com.facebook.react.ReactPackage;
import com.facebook.react.bridge.NativeModule;
import com.facebook.react.bridge.ReactApplicationContext;
import com.facebook.react.uimanager.ViewManager;

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class MyCustomPackage implements ReactPackage {

    @Override
    public List<NativeModule> createNativeModules(ReactApplicationContext reactContext) {
        List<NativeModule> modules = new ArrayList<>();
        modules.add(new MyCustomModule(reactContext));
        return modules;
    }

    @Override
    public List<ViewManager> createViewManagers(ReactApplicationContext reactContext) {
        return Collections.emptyList();
    }
}

Creació d'un Mòdul Natiu Personalitzat per a iOS

Pas 1: Crea una Nova Classe Objective-C/Swift

Crea una nova classe Objective-C o Swift en el teu projecte iOS. Aquesta classe definirà el mòdul natiu.

Exemple en Objective-C

#import <React/RCTBridgeModule.h>

@interface RCT_EXTERN_MODULE(MyCustomModule, NSObject)

RCT_EXTERN_METHOD(customMethod:(NSString *)message callback:(RCTResponseSenderBlock)callback)

@end

Exemple en Swift

import Foundation

@objc(MyCustomModule)
class MyCustomModule: NSObject {
  
  @objc(customMethod:callback:)
  func customMethod(message: String, callback: RCTResponseSenderBlock) {
    // Codi natiu aquí
    callback(["Received message: \(message)"])
  }
}

Pas 2: Registra el Mòdul

Registra el mòdul en el teu AppDelegate.m o AppDelegate.swift.

Exemple en Objective-C

#import <React/RCTBridge.h>
#import <React/RCTRootView.h>
#import "MyCustomModule.h"

@implementation AppDelegate

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
  RCTBridge *bridge = [[RCTBridge alloc] initWithDelegate:self launchOptions:launchOptions];
  RCTRootView *rootView = [[RCTRootView alloc] initWithBridge:bridge moduleName:@"YourAppName" initialProperties:nil];
  
  // Altres codis de configuració
  return YES;
}

@end

Exemple en Swift

import UIKit
import React

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

  var window: UIWindow?

  func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    let bridge = RCTBridge(delegate: self, launchOptions: launchOptions)
    let rootView = RCTRootView(bridge: bridge, moduleName: "YourAppName", initialProperties: nil)
    
    // Altres codis de configuració
    return true
  }
}

Ús del Mòdul Natiu Personalitzat en JavaScript

Un cop el mòdul natiu està creat i registrat, pots utilitzar-lo en el teu codi JavaScript.

import { NativeModules } from 'react-native';
const { MyCustomModule } = NativeModules;

MyCustomModule.customMethod('Hello from JS', (response) => {
  console.log(response); // Sortida: Received message: Hello from JS
});

Conclusió

La creació de mòduls natius personalitzats en React Native et permet aprofitar les característiques específiques de la plataforma i optimitzar el rendiment per a certes tasques. Seguint els passos descrits en aquesta guia, pots crear i registrar mòduls natius personalitzats tant per a Android com per a iOS, i utilitzar-los sense problemes en el teu codi JavaScript. Això millora les capacitats de les teves aplicacions React Native i proporciona una experiència d'usuari més integrada.
