Introducció a AsyncStorage

AsyncStorage és un sistema d'emmagatzematge simple, no xifrat, asíncron, persistent i de clau-valor que és global per a l'aplicació. S'utilitza en React Native per emmagatzemar dades que necessiten persistir entre llançaments de l'aplicació.

  • Conceptes Clau:
    • Emmagatzematge de Clau-Valor: Les dades s'emmagatzemen com a parells clau-valor.
    • Asíncron: Les operacions no bloquegen, permetent un rendiment suau de la interfície d'usuari.
    • Persistent: Les dades romanen disponibles fins i tot després de tancar i reobrir l'aplicació.

Configuració de AsyncStorage

Per utilitzar AsyncStorage en el teu projecte de React Native, necessites instal·lar el paquet.

npm install @react-native-async-storage/async-storage

Després de la instal·lació, necessites importar-lo al teu component.

import AsyncStorage from '@react-native-async-storage/async-storage';

Operacions Bàsiques amb AsyncStorage

Emmagatzemar Dades

Per emmagatzemar dades, utilitza el mètode setItem. Aquest mètode pren dos arguments: la clau i el valor.

const storeData = async (key, value) => {
  try {
    await AsyncStorage.setItem(key, value);
    console.log('Dades desades correctament');
  } catch (e) {
    console.error('No s\'han pogut desar les dades a l\'emmagatzematge', e);
  }
};

Recuperar Dades

Per recuperar dades, utilitza el mètode getItem. Aquest mètode pren un argument: la clau.

const getData = async (key) => {
  try {
    const value = await AsyncStorage.getItem(key);
    if (value !== null) {
      console.log('Dades recuperades correctament', value);
      return value;
    }
  } catch (e) {
    console.error('No s\'han pogut recuperar les dades de l\'emmagatzematge', e);
  }
};

Eliminar Dades

Per eliminar dades, utilitza el mètode removeItem. Aquest mètode pren un argument: la clau.

const removeData = async (key) => {
  try {
    await AsyncStorage.removeItem(key);
    console.log('Dades eliminades correctament');
  } catch (e) {
    console.error('No s\'han pogut eliminar les dades de l\'emmagatzematge', e);
  }
};

Ús Avançat

Emmagatzemar Dades Complexes

AsyncStorage només pot emmagatzemar cadenes, així que per emmagatzemar tipus de dades complexes com objectes o arrays, necessites serialitzar-los utilitzant JSON.stringify.

const storeComplexData = async (key, value) => {
  try {
    const jsonValue = JSON.stringify(value);
    await AsyncStorage.setItem(key, jsonValue);
    console.log('Dades complexes desades correctament');
  } catch (e) {
    console.error('No s\'han pogut desar les dades complexes a l\'emmagatzematge', e);
  }
};

Recuperar Dades Complexes

Per recuperar dades complexes, necessites deserialitzar-les utilitzant JSON.parse.

const getComplexData = async (key) => {
  try {
    const jsonValue = await AsyncStorage.getItem(key);
    return jsonValue != null ? JSON.parse(jsonValue) : null;
  } catch (e) {
    console.error('No s\'han pogut recuperar les dades complexes de l\'emmagatzematge', e);
  }
};

Esborrar Totes les Dades

Per esborrar totes les dades emmagatzemades a AsyncStorage, utilitza el mètode clear.

const clearAllData = async () => {
  try {
    await AsyncStorage.clear();
    console.log('Totes les dades esborrades correctament');
  } catch (e) {
    console.error('No s\'ha pogut esborrar l\'emmagatzematge', e);
  }
};

Gestió d'Errors

Una correcta gestió d'errors és crucial quan es treballa amb AsyncStorage per assegurar una experiència d'usuari suau.

  • Blocs Try-Catch: Utilitza blocs try-catch per gestionar els errors de manera elegant.
  • Valors de Reserva: Proporciona valors de reserva quan la recuperació de dades falla.
const getDataWithFallback = async (key, fallbackValue) => {
  try {
    const value = await AsyncStorage.getItem(key);
    return value != null ? value : fallbackValue;
  } catch (e) {
    console.error('No s\'han pogut recuperar les dades de l\'emmagatzematge', e);
    return fallbackValue;
  }
};

Conclusió

AsyncStorage és una eina poderosa per a la persistència de dades en aplicacions React Native. Entenent les seves operacions bàsiques i avançades, pots gestionar efectivament l'emmagatzematge i la recuperació de dades, assegurant una experiència d'usuari sense interrupcions. Recorda gestionar els errors de manera elegant i considera les implicacions de rendiment de la teva estratègia d'emmagatzematge de dades.
