Introducció

L'obtenció de dades és un aspecte fonamental de moltes aplicacions, incloses les construïdes amb React Native. La Fetch API proporciona una interfície senzilla per obtenir recursos de manera asíncrona a través de la xarxa. Aquesta secció et guiarà a través dels conceptes bàsics d'ús de la Fetch API en React Native, des de nivells principiants fins a avançats.

Conceptes bàsics de Fetch API

La Fetch API és una interfície moderna que et permet fer sol·licituds de xarxa similars a XMLHttpRequest (XHR). No obstant això, és més potent i flexible.

Conceptes clau

  • Promises: La Fetch API utilitza promeses, cosa que facilita el treball amb operacions asíncrones.
  • Request i Response: La Fetch API proporciona una funció global fetch que pren una URL i retorna una promesa que es resol amb la resposta.
  • Gestió de JSON: La Fetch API pot gestionar dades JSON fàcilment, que és un format comú per a les API.

Exemple bàsic de Fetch

Aquí tens un exemple senzill de com utilitzar la Fetch API per obtenir dades d'una URL:

fetch('https://jsonplaceholder.typicode.com/posts')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error obtenint dades:', error));

Explicació:

  • fetch('https://jsonplaceholder.typicode.com/posts'): Inicia una sol·licitud GET a la URL especificada.
  • .then(response => response.json()): Converteix la resposta a JSON.
  • .then(data => console.log(data)): Mostra les dades a la consola.
  • .catch(error => console.error('Error obtenint dades:', error)): Captura i mostra qualsevol error que es produeixi durant la sol·licitud.

Obtenció de dades en React Native

En una aplicació React Native, l'obtenció de dades sovint es fa dins dels mètodes del cicle de vida d'un component o utilitzant hooks.

Ús dels mètodes del cicle de vida del component

Pots obtenir dades en el mètode componentDidMount d'un component de classe.

import React, { Component } from 'react';
import { View, Text } from 'react-native';

class DataFetchingComponent extends Component {
  state = {
    data: [],
    isLoading: true,
    error: null,
  };

  componentDidMount() {
    fetch('https://jsonplaceholder.typicode.com/posts')
      .then(response => response.json())
      .then(data => this.setState({ data, isLoading: false }))
      .catch(error => this.setState({ error, isLoading: false }));
  }

  render() {
    const { data, isLoading, error } = this.state;

    if (isLoading) {
      return <Text>Carregant...</Text>;
    }

    if (error) {
      return <Text>Error: {error.message}</Text>;
    }

    return (
      <View>
        {data.map(post => (
          <Text key={post.id}>{post.title}</Text>
        ))}
      </View>
    );
  }
}

export default DataFetchingComponent;

Ús de Hooks (Components funcionals)

Amb components funcionals, pots utilitzar el hook useEffect per obtenir dades.

import React, { useState, useEffect } from 'react';
import { View, Text } from 'react-native';

const DataFetchingComponent = () => {
  const [data, setData] = useState([]);
  const [isLoading, setIsLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetch('https://jsonplaceholder.typicode.com/posts')
      .then(response => response.json())
      .then(data => {
        setData(data);
        setIsLoading(false);
      })
      .catch(error => {
        setError(error);
        setIsLoading(false);
      });
  }, []);

  if (isLoading) {
    return <Text>Carregant...</Text>;
  }

  if (error) {
    return <Text>Error: {error.message}</Text>;
  }

  return (
    <View>
      {data.map(post => (
        <Text key={post.id}>{post.title}</Text>
      ))}
    </View>
  );
};

export default DataFetchingComponent;

Tècniques avançades d'obtenció de dades

A mesura que et familiaritzis amb els conceptes bàsics, pots explorar tècniques més avançades com ara gestionar diferents mètodes HTTP, establir capçaleres i gestionar l'autenticació.

Gestió de diferents mètodes HTTP

Pots utilitzar la Fetch API per fer sol·licituds POST, PUT, DELETE especificant el mètode en l'objecte d'opcions.

// Exemple de sol·licitud POST
fetch('https://jsonplaceholder.typicode.com/posts', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    title: 'foo',
    body: 'bar',
    userId: 1,
  }),
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Establir capçaleres

Pots establir capçaleres personalitzades en les opcions de la sol·licitud.

fetch('https://jsonplaceholder.typicode.com/posts', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN_HERE',
    'Content-Type': 'application/json',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Gestió de l'autenticació

Per a sol·licituds autenticades, sovint necessites incloure tokens en les capçaleres.

fetch('https://api.example.com/protected', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
  },
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Conclusió

Obtenir dades amb la Fetch API en React Native és una manera potent i flexible d'interactuar amb recursos remots. Comprenent els conceptes bàsics i explorant tècniques avançades, pots construir aplicacions robustes que es comuniquin eficaçment amb les API. Recorda gestionar els errors de manera adequada i gestionar els estats de càrrega per proporcionar una experiència d'usuari suau.
