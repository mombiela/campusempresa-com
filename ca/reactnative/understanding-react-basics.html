Introducció a React

React és una biblioteca de JavaScript per construir interfícies d'usuari, principalment per a aplicacions d'una sola pàgina. Permet als desenvolupadors crear components de la IU reutilitzables, gestionar l'estat de l'aplicació de manera eficient i manejar les interaccions dels usuaris sense problemes.

Conceptes Clau

  • Components: Els blocs de construcció d'una aplicació React. Els components poden ser funcionals o basats en classes.
  • JSX: Una extensió de sintaxi que permet escriure codi semblant a HTML dins de JavaScript.
  • State: Un objecte que determina com es renderitza i es comporta un component.
  • Props: Abreviatura de propietats, les props són atributs de només lectura utilitzats per passar dades dels components pare als fills.
  • Mètodes del Cicle de Vida: Mètodes especials en components de classe que permeten enganxar-se a diferents etapes del cicle de vida d'un component.

Configurant un Entorn de React

Per començar a construir aplicacions React, necessites configurar el teu entorn de desenvolupament.

Prerequisits

  • Node.js i npm (Node Package Manager) instal·lats a la teva màquina.
  • Un editor de codi, com Visual Studio Code.

Creant un Nou Projecte React

Utilitza l'eina Create React App per configurar ràpidament un nou projecte React.

npx create-react-app my-react-app
cd my-react-app
npm start

Això crearà un nou projecte React i iniciarà el servidor de desenvolupament.

Comprendre els Components

Els components són el nucli de les aplicacions React. Poden ser funcionals o basats en classes.

Components Funcionals

Els components funcionals són funcions simples de JavaScript que retornen JSX.

import React from 'react';

function Greeting() {
  return <h1>Hello, World!</h1>;
}

export default Greeting;

Components de Classe

Els components de classe són classes ES6 que estenen React.Component i han d'incloure un mètode render.

import React, { Component } from 'react';

class Greeting extends Component {
  render() {
    return <h1>Hello, World!</h1>;
  }
}

export default Greeting;

JSX: JavaScript XML

JSX et permet escriure codi semblant a HTML dins de JavaScript, facilitant la creació i visualització de components de la IU.

Exemple de JSX

const element = <h1>Hello, World!</h1>;

Incrustar Expressions en JSX

Pots incrustar expressions de JavaScript dins de JSX utilitzant claus {}.

const name = 'John';
const element = <h1>Hello, {name}!</h1>;

State i Props

State i props són essencials per gestionar dades i renderitzar contingut dinàmic en components React.

State

State és un objecte integrat que emmagatzema valors de propietat que pertanyen a un component. Quan l'objecte state canvia, el component es torna a renderitzar.

import React, { Component } from 'react';

class Counter extends Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  render() {
    return (
      <div>
        <p>Count: {this.state.count}</p>
        <button onClick={() => this.setState({ count: this.state.count + 1 })}>
          Increment
        </button>
      </div>
    );
  }
}

export default Counter;

Props

Les props s'utilitzen per passar dades dels components pare als fills.

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

function App() {
  return <Welcome name="Sara" />;
}

export default App;

Cicle de Vida del Component

Els components React tenen un cicle de vida al qual pots enganxar-te utilitzant mètodes del cicle de vida. Aquests mètodes només estan disponibles en components de classe.

Mètodes Comuns del Cicle de Vida

  • componentDidMount(): Cridat després que el component es renderitzi.
  • componentDidUpdate(prevProps, prevState): Cridat després que el component s'actualitzi.
  • componentWillUnmount(): Cridat abans que el component es retiri del DOM.
import React, { Component } from 'react';

class Timer extends Component {
  componentDidMount() {
    this.timerID = setInterval(() => this.tick(), 1000);
  }

  componentWillUnmount() {
    clearInterval(this.timerID);
  }

  tick() {
    this.setState({
      date: new Date()
    });
  }

  render() {
    return (
      <div>
        <h1>{this.state.date.toLocaleTimeString()}</h1>
      </div>
    );
  }
}

export default Timer;

Conclusió

Comprendre els conceptes bàsics de React és crucial per construir interfícies d'usuari eficients i escalables. En dominar components, JSX, state, props i mètodes del cicle de vida, pots crear aplicacions dinàmiques i interactives. A mesura que progressis, aquests conceptes fonamentals serviran com a blocs de construcció per a temes més avançats en el desenvolupament de React i React Native.
