Introducció

En aquest curs, et guiarem a través del procés de construir una aplicació de comerç electrònic totalment funcional utilitzant React Native. Aquest curs està dissenyat tant per a principiants com per a desenvolupadors avançats, cobrint tot des de la configuració del teu entorn de desenvolupament fins al desplegament de la teva aplicació.

Configurant l'Entorn de Desenvolupament

  • Instal·lar Node.js i npm
  • Instal·lar React Native CLI
  • Configurar Android Studio i Xcode per al desenvolupament d'iOS
  • Crear un nou projecte de React Native
# Instal·lar React Native CLI globalment
npm install -g react-native-cli

# Crear un nou projecte de React Native
react-native init EcommerceApp

Navegació

  • Instal·lar React Navigation
  • Configurar la navegació per stack
# Instal·lar React Navigation
npm install @react-navigation/native @react-navigation/stack

# Instal·lar dependències
npm install react-native-screens react-native-safe-area-context
import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
import HomeScreen from './HomeScreen';
import ProductScreen from './ProductScreen';

const Stack = createStackNavigator();

const App = () => {
  return (
    <NavigationContainer>
      <Stack.Navigator initialRouteName="Home">
        <Stack.Screen name="Home" component={HomeScreen} />
        <Stack.Screen name="Product" component={ProductScreen} />
      </Stack.Navigator>
    </NavigationContainer>
  );
};

export default App;

Construint la Llista de Productes

  • Obtenir dades d'una API
  • Mostrar una llista de productes utilitzant FlatList
import React, { useState, useEffect } from 'react';
import { View, Text, FlatList, Image } from 'react-native';

const ProductList = () => {
  const [products, setProducts] = useState([]);

  useEffect(() => {
    fetch('https://api.example.com/products')
      .then(response => response.json())
      .then(data => setProducts(data));
  }, []);

  return (
    <FlatList
      data={products}
      keyExtractor={item => item.id.toString()}
      renderItem={({ item }) => (
        <View>
          <Image source={{ uri: item.image }} style={{ width: 100, height: 100 }} />
          <Text>{item.name}</Text>
          <Text>${item.price}</Text>
        </View>
      )}
    />
  );
};

export default ProductList;

import React from 'react';

Autenticació d'Usuaris

  • Implementar registre i inici de sessió d'usuaris
  • Utilitzar Firebase per a l'autenticació
# Instal·lar Firebase
npm install @react-native-firebase/app @react-native-firebase/auth
import auth from '@react-native-firebase/auth';

const signUp = (email, password) => {
  auth()
    .createUserWithEmailAndPassword(email, password)
    .then(() => {
      console.log('Compte d'usuari creat i iniciat sessió!');
    })
    .catch(error => {
      if (error.code === 'auth/email-already-in-use') {
        console.log('Aquesta adreça de correu ja està en ús!');
      }

      if (error.code === 'auth/invalid-email') {
        console.log('Aquesta adreça de correu és invàlida!');
      }

      console.error(error);
    });
};
