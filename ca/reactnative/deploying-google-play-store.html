Desplegar la teva aplicació React Native a Google Play Store és un pas crucial per fer que la teva aplicació estigui disponible per als usuaris d'Android a tot el món. Aquesta secció et guiarà a través del procés de principi a fi, assegurant que entenguis cada pas i puguis publicar la teva aplicació amb confiança.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Una aplicació React Native llesta per al desplegament.
  • Un compte de desenvolupador de Google Play (es requereix una tarifa de registre única de 25 $).
  • Android Studio instal·lat al teu ordinador.
  • Coneixements bàsics de React Native i desenvolupament d'Android.

Preparant la teva aplicació React Native

1. Actualitza la informació de l'aplicació

Assegura't que el teu fitxer app.json o AndroidManifest.xml contingui la informació correcta sobre la teva aplicació.

{
  "expo": {
    "name": "YourAppName",
    "slug": "your-app-slug",
    "version": "1.0.0",
    "android": {
      "package": "com.yourcompany.yourappname"
    }
  }
}

2. Genera un APK signat

Per publicar la teva aplicació a Google Play Store, necessites generar un APK signat. Segueix aquests passos:

  1. Obre Android Studio i obre el teu projecte React Native.
  2. Build > Generate Signed Bundle / APK...
  3. Selecciona APK i fes clic a Next.
  4. Crea un nou key store path o utilitza'n un d'existent.
  5. Omple els camps requerits (Key alias, Key password, etc.).
  6. Selecciona release com a tipus de build.
  7. Fes clic a Finish.

3. Configura Proguard (Opcional)

Proguard ajuda a reduir i ofuscar el teu codi, fent-lo més difícil de revertir. Per habilitar Proguard, afegeix el següent al teu fitxer android/app/build.gradle:

buildTypes {
    release {
        minifyEnabled true
        proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
    }
}

Pujant a Google Play Console

1. Crea una nova aplicació

  1. Ves a Google Play Console.
  2. Fes clic a Create Application.
  3. Selecciona un idioma per defecte i introdueix un títol per a la teva aplicació.
  4. Fes clic a Create.

2. Prepara la llista de la botiga

Omple els camps requerits a la secció de la llista de la botiga:

  • Title
  • Short Description
  • Full Description
  • Screenshots (almenys 2)
  • App Icon (512x512, PNG de 32 bits)
  • Feature Graphic (1024x500, JPG o PNG de 24 bits)
  • Category i Tags
  • Contact Details

3. Pujar l'APK

  1. Navega a la secció App releases.
  2. Fes clic a Manage production > Create release.
  3. Puja l'APK signat que has generat anteriorment.
  4. Omple les notes de la versió.
  5. Fes clic a Save.

4. Configura la classificació de contingut

  1. Ves a la secció Content rating.
  2. Fes clic a Start questionnaire.
  3. Omple el qüestionari per rebre una classificació de contingut per a la teva aplicació.
  4. Fes clic a Save.

5. Configura els preus i la distribució

  1. Ves a la secció Pricing & distribution.
  2. Selecciona si la teva aplicació és Free o Paid.
  3. Tria els països on vols distribuir la teva aplicació.
  4. Opta per la secció de Google Play's Data Safety.

6. Revisa i publica

  1. Revisa tota la informació que has introduït.
  2. Assegura't que no hi hagi errors o camps incomplets.
  3. Fes clic a Review a la part inferior de la pàgina.
  4. Si tot és correcte, fes clic a Start rollout to production.
  5. Fes clic a Confirm.

Conclusió

Desplegar la teva aplicació React Native a Google Play Store implica diversos passos, des de preparar la teva aplicació i generar un APK signat fins a omplir la informació necessària a Google Play Console. Seguint aquesta guia, hauràs de poder publicar la teva aplicació amb èxit i fer-la disponible per a milions d'usuaris d'Android. Recorda mantenir la teva aplicació actualitzada i monitoritzar el seu rendiment a través de Google Play Console per assegurar una experiència d'usuari fluida.
