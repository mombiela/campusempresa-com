Les proves d'extrem a extrem (E2E) són crucials per assegurar que la teva aplicació React Native funcioni com s'espera des de la perspectiva de l'usuari. Detox és un marc popular per a proves E2E en React Native, proporcionant una manera robusta i fiable d'automatitzar les teves proves.

Introducció a Detox

  • Què és Detox?

    • Detox és una biblioteca de proves d'extrem a extrem per a aplicacions mòbils.
    • Et permet escriure proves que simulen interaccions d'usuari.
    • Detox funciona amb plataformes iOS i Android.

  • Per què utilitzar Detox?

    • Assegura que la teva aplicació funcioni correctament en diferents dispositius i versions de SO.
    • Automatitza tasques de prova repetitives.
    • Ajuda a detectar errors aviat en el cicle de desenvolupament.

Configuració de Detox

  • Prerequisits:

    • Node.js i npm/yarn instal·lats.
    • React Native CLI instal·lat.
    • Xcode (per a iOS) i Android Studio (per a Android).

  • Instal·lació:

    npm install -g detox-cli
npm install detox --save-dev

  • Configuració de Detox:

    • Afegeix la configuració de Detox al teu package.json: 
      {
  "detox": {
    "configurations": {
      "ios.sim.debug": {
        "binaryPath": "ios/build/Build/Products/Debug-iphonesimulator/YourApp.app",
        "build": "xcodebuild -workspace ios/YourApp.xcworkspace -scheme YourApp -configuration Debug -sdk iphonesimulator -derivedDataPath ios/build",
        "type": "ios.simulator",
        "name": "iPhone 11"
      }
    }
  }
}
    • Per a Android, la configuració serà diferent: 
      {
  "detox": {
    "configurations": {
      "android.emu.debug": {
        "binaryPath": "android/app/build/outputs/apk/debug/app-debug.apk",
        "build": "cd android && ./gradlew assembleDebug assembleAndroidTest -DtestBuildType=debug && cd ..",
        "type": "android.emulator",
        "name": "Pixel_3a_API_30_x86"
      }
    }
  }
}

Escriure la teva primera prova Detox

  • Creació d'un fitxer de prova:

    • Crea una carpeta e2e a l'arrel del teu projecte.
    • Afegeix un fitxer de prova, per exemple, firstTest.e2e.js.

  • Exemple de prova:

    describe('Example', () => {
  beforeAll(async () => {
    await device.launchApp();
  });

  beforeEach(async () => {
    await device.reloadReactNative();
  });

  it('should have welcome screen', async () => {
    await expect(element(by.id('welcome'))).toBeVisible();
  });

  it('should show hello screen after tap', async () => {
    await element(by.id('hello_button')).tap();
    await expect(element(by.id('hello_screen'))).toBeVisible();
  });
});

  • Explicació:

    • describe: Agrupa proves relacionades.
    • beforeAll: S'executa una vegada abans de totes les proves.
    • beforeEach: S'executa abans de cada prova.
    • it: Defineix una prova individual.
    • element: Selecciona un element de la interfície d'usuari.
    • by.id: Selector per elements pel seu testID.
    • tap: Simula una acció de tocar.
    • expect: Afirmació per comprovar si un element és visible.

Executar les proves Detox

  • Construir l'aplicació:

    detox build --configuration ios.sim.debug

  • Executar les proves:

    detox test --configuration ios.sim.debug

  • Comandes comunes:

    • detox build: Construeix l'aplicació per a proves.
    • detox test: Executa les proves.
    • detox clean-framework-cache: Neteja la memòria cau de Detox.

Funcions avançades de Detox

  • Mocking i Stubbing:

    • Utilitza Detox per simular sol·licituds de xarxa o altres dependències.
    • Exemple utilitzant jest per simular una sol·licitud de xarxa: 
      jest.mock('axios', () => ({
  get: jest.fn(() => Promise.resolve({ data: { message: 'Hello World' } }))
}));

  • Execució paral·lela de proves:

    • Detox suporta l'execució de proves en paral·lel per accelerar el procés de prova.
    • Configura l'execució paral·lela al teu package.json: 
      {
  "detox": {
    "test-runner": "jest",
    "runner-config": "e2e/config.json",
    "configurations": {
      "ios.sim.debug": {
        "binaryPath": "ios/build/Build/Products/Debug-iphonesimulator/YourApp.app",
        "build": "xcodebuild -workspace ios/YourApp.xcworkspace -scheme YourApp -configuration Debug -sdk iphonesimulator -derivedDataPath ios/build",
        "type": "ios.simulator",
        "name": "iPhone 11"
      }
    }
  }
}

Conclusió

Detox és una eina poderosa per a proves d'extrem a extrem en React Native, assegurant que la teva aplicació es comporti com s'espera en diferents dispositius i plataformes. Configurant Detox, escrivint proves i executant-les, pots automatitzar el procés de prova i detectar errors aviat. Funcions avançades com la simulació i l'execució paral·lela de proves milloren encara més les teves capacitats de prova. Comença a integrar Detox en el teu flux de treball per millorar la fiabilitat i la qualitat de les teves aplicacions React Native.
