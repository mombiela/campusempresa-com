Introducció a Redux

Redux és un contenidor d'estat previsible per a aplicacions JavaScript, sovint utilitzat amb biblioteques com React per gestionar l'estat de l'aplicació. T'ajuda a escriure aplicacions que es comporten de manera consistent, funcionen en diferents entorns (client, servidor i natiu) i són fàcils de provar.

Conceptes Clau de Redux

  • Store: La font única de veritat que conté l'estat de la teva aplicació.
  • Actions: Objectes JavaScript simples que descriuen què ha passat.
  • Reducers: Funcions que prenen l'estat actual i una acció, i retornen un nou estat.
  • Dispatch: Una funció per enviar accions a la botiga.
  • Selectors: Funcions per extreure i derivar dades de l'estat.

Configurant Redux en un Projecte React Native

Instal·lant Dependències

Per començar amb Redux en un projecte React Native, necessites instal·lar els paquets necessaris:

npm install redux react-redux

Creant la Botiga Redux

Crea un fitxer anomenat store.js per configurar la botiga Redux.

import { createStore } from 'redux';
import rootReducer from './reducers';

const store = createStore(rootReducer);

export default store;

Definint Accions

Crea un fitxer anomenat actions.js per definir les teves accions.

// actions.js
export const INCREMENT = 'INCREMENT';
export const DECREMENT = 'DECREMENT';

export const increment = () => ({
  type: INCREMENT,
});

export const decrement = () => ({
  type: DECREMENT,
});

Creant Reductors

Crea un fitxer anomenat reducers.js per definir els teus reductors.

// reducers.js
import { INCREMENT, DECREMENT } from './actions';

const initialState = {
  count: 0,
};

const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case INCREMENT:
      return { ...state, count: state.count + 1 };
    case DECREMENT:
      return { ...state, count: state.count - 1 };
    default:
      return state;
  }
};

export default counterReducer;

Combinant Reductors

Si tens múltiples reductors, els pots combinar utilitzant combineReducers.

// rootReducer.js
import { combineReducers } from 'redux';
import counterReducer from './counterReducer';

const rootReducer = combineReducers({
  counter: counterReducer,
});

export default rootReducer;

Connectant Redux a React Native

Embolica el teu component principal de l'aplicació amb el Provider de react-redux i passa-li la botiga.

// App.js
import React from 'react';
import { Provider } from 'react-redux';
import store from './store';
import Counter from './Counter';

const App = () => (
  <Provider store={store}>
    <Counter />
  </Provider>
);

export default App;

Utilitzant Redux en Components

Connecta els teus components a la botiga Redux utilitzant la funció connect de react-redux.

// Counter.js
import React from 'react';
import { View, Text, Button } from 'react-native';
import { connect } from 'react-redux';
import { increment, decrement } from './actions';

const Counter = ({ count, increment, decrement }) => (
  <View>
    <Text>{count}</Text>
    <Button title="Increment" onPress={increment} />
    <Button title="Decrement" onPress={decrement} />
  </View>
);

const mapStateToProps = (state) => ({
  count: state.counter.count,
});

const mapDispatchToProps = {
  increment,
  decrement,
};

export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Conclusió

Integrar Redux amb React Native implica configurar una botiga Redux, definir accions i reductors, i connectar els teus components React Native a la botiga Redux utilitzant la funció connect. Aquesta configuració ajuda a gestionar l'estat de la teva aplicació de manera previsible i mantenible. Seguint aquests passos, pots assegurar-te que la teva aplicació React Native tingui una solució robusta de gestió d'estat.
