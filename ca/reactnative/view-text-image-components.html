Introducció

En aquesta secció, explorarem els components fonamentals en React Native: View, Text i Image. Aquests components són els blocs de construcció per crear interfícies d'usuari en aplicacions React Native. Entendre com utilitzar aquests components de manera efectiva és crucial per a qualsevol desenvolupador de React Native, des de principiants fins a usuaris avançats.

Component View

El component View és un contenidor que suporta el disseny amb Flexbox, estil, maneig de tocs i controls d'accessibilitat. És anàleg a un div en HTML.

Conceptes Clau

  • Contenidor: Utilitzat per embolicar altres components.
  • Estilització: Es pot estilitzar utilitzant la prop style.
  • Flexbox: Suporta el disseny Flexbox per organitzar components fills.

Exemple

import React from 'react';
import { View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <View style={styles.box1} />
      <View style={styles.box2} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'row',
    justifyContent: 'space-around',
    alignItems: 'center',
  },
  box1: {
    width: 50,
    height: 50,
    backgroundColor: 'red',
  },
  box2: {
    width: 50,
    height: 50,
    backgroundColor: 'blue',
  },
});

export default App;

Explicació

  • Contenidor: El View extern actua com un contenidor per als components View interns.
  • Flexbox: Les propietats flexDirection, justifyContent i alignItems s'utilitzen per organitzar els components fills.

Component Text

El component Text s'utilitza per mostrar text a la teva aplicació. Suporta la nidificació, l'estilització i el maneig de tocs.

Conceptes Clau

  • Nidificació: Els components de text es poden niar dins d'altres.
  • Estilització: Es pot estilitzar utilitzant la prop style.
  • Maneig de Tocs: Pot gestionar esdeveniments de toc.

Exemple

import React from 'react';
import { Text, StyleSheet } from 'react-native';

const App = () => {
  return (
    <Text style={styles.text}>
      Hola, <Text style={styles.bold}>React Native</Text>!
    </Text>
  );
};

const styles = StyleSheet.create({
  text: {
    fontSize: 20,
    color: 'black',
  },
  bold: {
    fontWeight: 'bold',
  },
});

export default App;

Explicació

  • Nidificació: El component Text amb l'estil bold està niat dins del component Text pare.
  • Estilització: L'estil text estableix la mida de la font i el color, mentre que l'estil bold estableix el pes de la font.

Component Image

El component Image s'utilitza per mostrar imatges a la teva aplicació. Suporta diversos formats d'imatge i es pot estilitzar i redimensionar.

Conceptes Clau

  • Font: La prop source especifica la imatge a mostrar.
  • Estilització: Es pot estilitzar utilitzant la prop style.
  • Redimensionament: Suporta modes de redimensionament com cover, contain, etc.

Exemple

import React from 'react';
import { Image, StyleSheet, View } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Image
        style={styles.image}
        source={{ uri: 'https://reactnative.dev/img/tiny_logo.png' }}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
  image: {
    width: 100,
    height: 100,
  },
});

export default App;

Explicació

  • Font: La prop source s'utilitza per especificar la URL de la imatge.
  • Estilització: L'estil image estableix l'amplada i l'alçada de la imatge.
  • Redimensionament: La imatge es redimensionarà per ajustar-se a les dimensions especificades.

Conclusió

En aquesta secció, hem cobert els components bàsics en React Native: View, Text i Image. Aquests components són essencials per construir interfícies d'usuari. Hem après com utilitzar-los, estilitzar-los i gestionar les seves propietats. Dominar aquests components proporcionarà una base sòlida per crear aplicacions més complexes i interactives en React Native.
