Gestió de l'Entrada d'Usuari amb TextInput

Introducció

Gestionar l'entrada d'usuari és un aspecte fonamental de la construcció d'aplicacions interactives. A React Native, el component TextInput s'utilitza per capturar l'entrada de text de l'usuari. Aquesta secció et guiarà a través dels conceptes bàsics d'ús de TextInput, la gestió del seu estat i la implementació de funcions avançades.

Ús Bàsic de TextInput

Per començar amb TextInput, cal importar-lo de la biblioteca react-native i incloure'l en el teu component.

Exemple

import React, { useState } from 'react';
import { View, TextInput, Text, StyleSheet } from 'react-native';

const BasicTextInput = () => {
  const [text, setText] = useState('');

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Escriu aquí..."
        onChangeText={setText}
        value={text}
      />
      <Text style={styles.text}>Has escrit: {text}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
  text: {
    fontSize: 18,
  },
});

export default BasicTextInput;

Explicació

  • Importació de Components: TextInput, View, Text i StyleSheet són importats de react-native.
  • Gestió d'Estat: El hook useState s'utilitza per gestionar l'estat de l'entrada de text.
  • Component TextInput:
    • style: Aplica estils al TextInput.
    • placeholder: Mostra un text de marcador de posició quan l'entrada està buida.
    • onChangeText: Actualitza l'estat cada vegada que el text canvia.
    • value: Vincula l'estat al valor del TextInput.
  • Component Text: Mostra el text escrit per l'usuari.

Gestió de Diferents Tipus d'Entrada

TextInput pot gestionar diversos tipus d'entrada establint la propietat keyboardType.

Exemple

<TextInput
  style={styles.input}
  placeholder="Introdueix el teu correu electrònic"
  keyboardType="email-address"
  onChangeText={setEmail}
  value={email}
/>

Valors comuns de `keyboardType`

| Valor | Descripció | |----------------|--------------------------------------| | default | Teclat per defecte | | email-address| Teclat optimitzat per a l'entrada de correu electrònic | | numeric | Teclat numèric | | phone-pad | Teclat optimitzat per a números de telèfon |

Gestió d'Esdeveniments de Focus i Desenfocament

Pots gestionar esdeveniments de focus i desenfocament per millorar l'experiència de l'usuari.

Exemple

<TextInput
  style={styles.input}
  placeholder="Focalitza per veure l'efecte"
  onFocus={() => console.log('Entrada focalitzada')}
  onBlur={() => console.log('Entrada desenfocada')}
/>

Explicació

  • onFocus: Es desencadena quan el TextInput guanya el focus.
  • onBlur: Es desencadena quan el TextInput perd el focus.

Funcions Avançades

Controlar TextInput Programàticament

Pots utilitzar ref per controlar el TextInput programàticament.

Exemple

import React, { useRef } from 'react';
import { View, TextInput, Button, StyleSheet } from 'react-native';

const ControlledTextInput = () => {
  const inputRef = useRef(null);

  const focusInput = () => {
    inputRef.current.focus();
  };

  return (
    <View style={styles.container}>
      <TextInput
        ref={inputRef}
        style={styles.input}
        placeholder="Entrada controlada"
      />
      <Button title="Focalitza l'entrada" onPress={focusInput} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
});

export default ControlledTextInput;

Explicació

  • Hook useRef: S'utilitza per crear una referència al TextInput.
  • Mètode focus: Es crida per focalitzar programàticament el TextInput.

Conclusió

En aquesta secció, has après com gestionar l'entrada d'usuari utilitzant el component TextInput a React Native. Hem cobert l'ús bàsic, la gestió de diferents tipus d'entrada, la gestió d'esdeveniments de focus i desenfocament, i el control programàtic de TextInput. Dominar aquests conceptes t'ajudarà a construir aplicacions més interactives i fàcils d'usar.
