Introducció als Mapes en React Native

Els mapes són una part crucial de moltes aplicacions mòbils, proporcionant serveis basats en la ubicació i funcions de navegació als usuaris. En aquesta secció, explorarem com integrar i treballar amb mapes en aplicacions React Native.

Configuració de l'Entorn

Abans de començar a treballar amb mapes, necessitem configurar el nostre entorn de desenvolupament.

  1. Instal·lar React Native Maps:

    npm install react-native-maps

  2. Enllaçar la Biblioteca: Per React Native 0.60 i versions superiors, la biblioteca s'hauria d'enllaçar automàticament. Per a versions més antigues, potser caldrà enllaçar-la manualment:

    react-native link react-native-maps

  3. Configurar iOS:

    • Obre el teu projecte a Xcode.
    • Afegeix AirGoogleMaps i AirMaps a les biblioteques del teu projecte.
    • Assegura't de tenir els permisos necessaris al teu Info.plist: 
      <key>NSLocationWhenInUseUsageDescription</key>
<string>Necessitem la teva ubicació per mostrar mapes</string>

  4. Configurar Android:

    • Afegeix els següents permisos al teu AndroidManifest.xml: 
      <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />

Component Bàsic de Mapa

Comencem per renderitzar un component bàsic de mapa.

Codi d'Exemple

import React from 'react';
import { StyleSheet, View } from 'react-native';
import MapView from 'react-native-maps';

const BasicMap = () => {
  return (
    <View style={styles.container}>
      <MapView
        style={styles.map}
        initialRegion={{
          latitude: 37.78825,
          longitude: -122.4324,
          latitudeDelta: 0.0922,
          longitudeDelta: 0.0421,
        }}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    ...StyleSheet.absoluteFillObject,
    justifyContent: 'flex-end',
    alignItems: 'center',
  },
  map: {
    ...StyleSheet.absoluteFillObject,
  },
});

export default BasicMap;

Explicació

  • MapView: El component principal per renderitzar el mapa.
  • initialRegion: Defineix la regió inicial mostrada pel mapa.
    • latitude i longitude: Coordenades per al centre del mapa.
    • latitudeDelta i longitudeDelta: Controlen el nivell de zoom del mapa.

Afegint Marcadors

Els marcadors s'utilitzen per indicar ubicacions específiques al mapa.

Codi d'Exemple

import React from 'react';
import { StyleSheet, View } from 'react-native';
import MapView, { Marker } from 'react-native-maps';

const MapWithMarker = () => {
  return (
    <View style={styles.container}>
      <MapView
        style={styles.map}
        initialRegion={{
          latitude: 37.78825,
          longitude: -122.4324,
          latitudeDelta: 0.0922,
          longitudeDelta: 0.0421,
        }}
      >
        <Marker
          coordinate={{ latitude: 37.78825, longitude: -122.4324 }}
          title={"El Meu Marcador"}
          description={"Aquesta és una descripció del marcador"}
        />
      </MapView>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    ...StyleSheet.absoluteFillObject,
    justifyContent: 'flex-end',
    alignItems: 'center',
  },
  map: {
    ...StyleSheet.absoluteFillObject,
  },
});

export default MapWithMarker;

Explicació

  • Marker: S'utilitza per col·locar un marcador al mapa.
    • coordinate: Especifica la ubicació del marcador.
    • title i description: Proporcionen informació addicional sobre el marcador.

Personalitzant Marcadors

Pots personalitzar els marcadors amb imatges o vistes personalitzades.

Codi d'Exemple

import React from 'react';
import { StyleSheet, View, Image } from 'react-native';
import MapView, { Marker } from 'react-native-maps';

const CustomMarker = () => {
  return (
    <View style={styles.container}>
      <MapView
        style={styles.map}
        initialRegion={{
          latitude: 37.78825,
          longitude: -122.4324,
          latitudeDelta: 0.0922,
          longitudeDelta: 0.0421,
        }}
      >
        <Marker
          coordinate={{ latitude: 37.78825, longitude: -122.4324 }}
        >
          <Image
            source={require('./path/to/custom-marker.png')}
            style={{ height: 35, width: 35 }}
          />
        </Marker>
      </MapView>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    ...StyleSheet.absoluteFillObject,
    justifyContent: 'flex-end',
    alignItems: 'center',
  },
  map: {
    ...StyleSheet.absoluteFillObject,
  },
});

export default CustomMarker;

Explicació

  • Marcador Personalitzat: Pots utilitzar qualsevol component de React Native dins del component Marker per personalitzar-ne l'aparença.

Gestionant la Ubicació de l'Usuari

Per mostrar la ubicació actual de l'usuari al mapa, necessites sol·licitar permisos de localització i habilitar el seguiment de la ubicació.

Codi d'Exemple

import React, { useState, useEffect } from 'react';
import { StyleSheet, View, PermissionsAndroid, Platform } from 'react-native';
import MapView, { Marker } from 'react-native-maps';
import Geolocation from '@react-native-community/geolocation';

const UserLocationMap = () => {
  const [location, setLocation] = useState(null);

  useEffect(() => {
    const requestLocationPermission = async () => {
      if (Platform.OS === 'android') {
        const granted = await PermissionsAndroid.request(
          PermissionsAndroid.PERMISSIONS.ACCESS_FINE_LOCATION
        );
        if (granted !== PermissionsAndroid.RESULTS.GRANTED) {
          console.warn('Permís de localització denegat');
          return;
        }
      }
      Geolocation.getCurrentPosition(
        (position) => {
          setLocation({
            latitude: position.coords.latitude,
            longitude: position.coords.longitude,
            latitudeDelta: 0.0922,
            longitudeDelta: 0.0421,
          });
        },
        (error) => console.error(error),
        { enableHighAccuracy: true, timeout: 20000, maximumAge: 1000 }
      );
    };

    requestLocationPermission();
  }, []);

  return (
    <View style={styles.container}>
      <MapView
        style={styles.map}
        region={location}
        showsUserLocation={true}
      >
        {location && (
          <Marker
            coordinate={location}
            title={"Estàs aquí"}
          />
        )}
      </MapView>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    ...StyleSheet.absoluteFillObject,
    justifyContent: 'flex-end',
    alignItems: 'center',
  },
  map: {
    ...StyleSheet.absoluteFillObject,
  },
});

export default UserLocationMap;

Explicació

  • PermissionsAndroid: S'utilitza per sol·licitar permisos de localització a Android.
  • Geolocation: S'utilitza per obtenir la ubicació actual de l'usuari.
  • showsUserLocation: Una propietat de MapView que mostra la ubicació actual de l'usuari al mapa.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'integrar mapes en una aplicació React Native. Hem après a configurar l'entorn, renderitzar un mapa bàsic, afegir marcadors, personalitzar marcadors i gestionar la ubicació de l'usuari. Amb aquestes habilitats fonamentals, pots començar a construir funcions de mapes més complexes i interactives en les teves aplicacions React Native.
