React Navigation és una biblioteca popular per a l'encaminament i la navegació en aplicacions React Native. Proporciona una manera de navegar entre diferents pantalles i gestionar l'historial de navegació. Aquest tema cobrirà els conceptes bàsics de React Navigation, incloent la seva configuració, la creació de navegadors i la gestió d'accions de navegació.

Configuració de React Navigation

Per començar amb React Navigation, cal instal·lar els paquets necessaris. Seguiu aquests passos:

  1. Instal·leu React Navigation i les seves dependències:

    npm install @react-navigation/native
npm install @react-navigation/native-stack
npm install react-native-screens react-native-safe-area-context

  2. Emboliqueu la vostra aplicació amb NavigationContainer:

    import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';

export default function App() {
  return (
    <NavigationContainer>
      {/* Resta del codi de la vostra aplicació */}
    </NavigationContainer>
  );
}

Creació de Navegadors

React Navigation proporciona diversos tipus de navegadors per gestionar diferents patrons de navegació. Els navegadors més utilitzats són:

  • Stack Navigator
  • Tab Navigator
  • Drawer Navigator

Stack Navigator

Un Stack Navigator us permet navegar entre pantalles de manera apilada, on cada nova pantalla es col·loca a sobre de l'anterior.

  1. Instal·leu el stack navigator:

    npm install @react-navigation/stack

  2. Creeu un Stack Navigator:

    import { createNativeStackNavigator } from '@react-navigation/native-stack';

const Stack = createNativeStackNavigator();

function MyStack() {
  return (
    <Stack.Navigator>
      <Stack.Screen name="Home" component={HomeScreen} />
      <Stack.Screen name="Details" component={DetailsScreen} />
    </Stack.Navigator>
  );
}

  3. Utilitzeu el Stack Navigator a la vostra aplicació:

    export default function App() {
  return (
    <NavigationContainer>
      <MyStack />
    </NavigationContainer>
  );
}

Tab Navigator

Un Tab Navigator us permet crear un sistema de navegació basat en pestanyes.

  1. Instal·leu el tab navigator:

    npm install @react-navigation/bottom-tabs

  2. Creeu un Tab Navigator:

    import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';

const Tab = createBottomTabNavigator();

function MyTabs() {
  return (
    <Tab.Navigator>
      <Tab.Screen name="Home" component={HomeScreen} />
      <Tab.Screen name="Settings" component={SettingsScreen} />
    </Tab.Navigator>
  );
}

  3. Utilitzeu el Tab Navigator a la vostra aplicació:

    export default function App() {
  return (
    <NavigationContainer>
      <MyTabs />
    </NavigationContainer>
  );
}

Drawer Navigator

Un Drawer Navigator us permet crear un menú lateral que es desplaça des del costat de la pantalla.

  1. Instal·leu el drawer navigator:

    npm install @react-navigation/drawer

  2. Creeu un Drawer Navigator:

    import { createDrawerNavigator } from '@react-navigation/drawer';

const Drawer = createDrawerNavigator();

function MyDrawer() {
  return (
    <Drawer.Navigator>
      <Drawer.Screen name="Home" component={HomeScreen} />
      <Drawer.Screen name="Profile" component={ProfileScreen} />
    </Drawer.Navigator>
  );
}

  3. Utilitzeu el Drawer Navigator a la vostra aplicació:

    export default function App() {
  return (
    <NavigationContainer>
      <MyDrawer />
    </NavigationContainer>
  );
}

Gestió d'Accions de Navegació

React Navigation proporciona diversos mètodes per gestionar accions de navegació, com ara navegar a una nova pantalla, tornar a la pantalla anterior i restablir l'estat de navegació.

  • Navegar a una nova pantalla:

    function HomeScreen({ navigation }) {
  return (
    <Button
      title="Go to Details"
      onPress={() => navigation.navigate('Details')}
    />
  );
}

  • Tornar a la pantalla anterior:

    function DetailsScreen({ navigation }) {
  return (
    <Button
      title="Go back"
      onPress={() => navigation.goBack()}
    />
  );
}

  • Restablir l'estat de navegació:

    function HomeScreen({ navigation }) {
  return (
    <Button
      title="Reset to Home"
      onPress={() => navigation.reset({
        index: 0,
        routes: [{ name: 'Home' }],
      })}
    />
  );
}

Conclusió

React Navigation és una biblioteca potent i flexible per gestionar la navegació en aplicacions React Native. Comprenent com configurar i utilitzar diferents tipus de navegadors, podeu crear una experiència de navegació fluida i intuïtiva per als vostres usuaris. Practiqueu utilitzant els navegadors Stack, Tab i Drawer, i experimenteu amb accions de navegació per esdevenir proficient en la gestió de la navegació a les vostres aplicacions React Native.
