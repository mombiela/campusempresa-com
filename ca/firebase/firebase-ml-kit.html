Firebase ML Kit és un SDK mòbil que porta l'expertesa en aprenentatge automàtic de Google a les aplicacions d'Android i iOS en un paquet potent però fàcil d'utilitzar. Ofereix tant APIs en el dispositiu com basades en el núvol per ajudar-te a implementar funcions d'aprenentatge automàtic a la teva aplicació.

Introducció a Firebase ML Kit

  • Què és Firebase ML Kit?
    • Un conjunt d'APIs proporcionades per Firebase per integrar l'aprenentatge automàtic en aplicacions mòbils.
    • Suporta tant models d'aprenentatge automàtic en el dispositiu com basats en el núvol.
  • Per què utilitzar Firebase ML Kit?
    • Simplifica la integració de funcions d'aprenentatge automàtic.
    • Proporciona models preentrenats per a casos d'ús comuns.
    • Ofereix la possibilitat de desplegar models personalitzats.

Començant amb Firebase ML Kit

  • Configuració de Firebase en el teu projecte
    • Afegeix Firebase al teu projecte d'Android o iOS.
    • Activa ML Kit a la Consola de Firebase.
  • Afegint dependències
    • Per a Android: 
      dependencies {
    implementation 'com.google.firebase:firebase-ml-vision:24.0.3'
}
    • Per a iOS: 
      pod 'Firebase/MLVision'

APIs en el Dispositiu

  • Reconeixement de Text
    • Reconeix text en imatges.
    • Exemple: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionTextRecognizer detector = FirebaseVision.getInstance().getOnDeviceTextRecognizer();

detector.processImage(image)
    .addOnSuccessListener(new OnSuccessListener<FirebaseVisionText>() {
        @Override
        public void onSuccess(FirebaseVisionText texts) {
            // Tasca completada amb èxit
            for (FirebaseVisionText.TextBlock block : texts.getTextBlocks()) {
                String text = block.getText();
                Log.d("TextRecognition", text);
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // Tasca fallida amb una excepció
            Log.e("TextRecognition", e.getMessage());
        }
    });
  • Detecció de Cares
    • Detecta cares en imatges i proporciona informació com la posició, punts de referència i classificació.
    • Exemple: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionFaceDetectorOptions options =
    new FirebaseVisionFaceDetectorOptions.Builder()
        .setModeType(FirebaseVisionFaceDetectorOptions.FAST)
        .setLandmarkType(FirebaseVisionFaceDetectorOptions.ALL_LANDMARKS)
        .setClassificationType(FirebaseVisionFaceDetectorOptions.ALL_CLASSIFICATIONS)
        .build();
FirebaseVisionFaceDetector detector = FirebaseVision.getInstance().getVisionFaceDetector(options);

detector.detectInImage(image)
    .addOnSuccessListener(new OnSuccessListener<List<FirebaseVisionFace>>() {
        @Override
        public void onSuccess(List<FirebaseVisionFace> faces) {
            // Tasca completada amb èxit
            for (FirebaseVisionFace face : faces) {
                Rect bounds = face.getBoundingBox();
                Log.d("FaceDetection", "Límits de la cara: " + bounds.toString());
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // Tasca fallida amb una excepció
            Log.e("FaceDetection", e.getMessage());
        }
    });

APIs Basades en el Núvol

  • Reconeixement de Text en el Núvol
    • Reconeix text en imatges utilitzant models basats en el núvol.
    • Exemple: 
      FirebaseVisionImage image = FirebaseVisionImage.fromBitmap(bitmap);
FirebaseVisionCloudTextRecognizerOptions options =
    new FirebaseVisionCloudTextRecognizerOptions.Builder()
        .setLanguageHints(Arrays.asList("en", "fr"))
        .build();
FirebaseVisionTextRecognizer detector = FirebaseVision.getInstance().getCloudTextRecognizer(options);

detector.processImage(image)
    .addOnSuccessListener(new OnSuccessListener<FirebaseVisionText>() {
        @Override
        public void onSuccess(FirebaseVisionText texts) {
            // Tasca completada amb èxit
            for (FirebaseVisionText.TextBlock block : texts.getTextBlocks()) {
                String text = block.getText();
                Log.d("CloudTextRecognition", text);
            }
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // Tasca fallida amb una excepció
            Log.e("CloudTextRecognition", e.getMessage());
        }
    });

Models Personalitzats

  • Desplegant Models Personalitzats
    • Carrega els teus models personalitzats de TensorFlow Lite a Firebase.
    • Exemple: 
      FirebaseModelDownloadConditions conditions = new FirebaseModelDownloadConditions.Builder()
    .requireWifi()
    .build();
FirebaseCustomRemoteModel remoteModel = new FirebaseCustomRemoteModel.Builder("my_model")
    .build();
FirebaseModelManager.getInstance().download(remoteModel, conditions)
    .addOnSuccessListener(new OnSuccessListener<Void>() {
        @Override
        public void onSuccess(Void aVoid) {
            // Model descarregat amb èxit
            Log.d("CustomModel", "Model descarregat");
        }
    })
    .addOnFailureListener(new OnFailureListener() {
        @Override
        public void onFailure(@NonNull Exception e) {
            // Tasca fallida amb una excepció
            Log.e("CustomModel", e.getMessage());
        }
    });

Conclusió

Firebase ML Kit proporciona un conjunt complet d'eines per integrar l'aprenentatge automàtic a les teves aplicacions mòbils. Tant si utilitzes models preentrenats per a tasques comunes com el reconeixement de text i la detecció de cares, com si desplegues els teus models personalitzats, Firebase ML Kit simplifica el procés i et permet centrar-te en crear grans experiències d'usuari. Aprofitant tant les capacitats en el dispositiu com les basades en el núvol, pots crear aplicacions potents i eficients que utilitzen els últims avenços en aprenentatge automàtic.
