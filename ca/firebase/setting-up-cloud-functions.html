Introducció a Cloud Functions

Cloud Functions per a Firebase és un framework sense servidor que et permet executar automàticament codi backend en resposta a esdeveniments desencadenats per funcions de Firebase i sol·licituds HTTPS. Això et permet construir aplicacions complexes amb un esforç mínim.

Conceptes Clau

  • Sense servidor: No cal gestionar servidors; Firebase s'encarrega de la infraestructura.
  • Basat en esdeveniments: Les funcions es desencadenen per esdeveniments com canvis a la base de dades, autenticació d'usuaris o sol·licituds HTTP.
  • Escalable: S'escala automàticament amunt o avall segons la càrrega.

Prerequisits

Abans de configurar Cloud Functions, assegura't de tenir:

  • Un projecte de Firebase configurat.
  • Node.js i npm instal·lats a la teva màquina.
  • Firebase CLI instal·lat.

Configuració del Teu Entorn

Pas 1: Instal·lar Firebase CLI

Primer, has d'instal·lar Firebase CLI si encara no ho has fet.

npm install -g firebase-tools

Pas 2: Inicialitzar Firebase al Teu Projecte

Navega al directori del teu projecte i inicialitza Firebase.

firebase init

Durant la inicialització:

  • Selecciona Functions utilitzant les tecles de fletxa i la barra espaiadora.
  • Tria un projecte de Firebase existent o crea'n un de nou.
  • Selecciona JavaScript o TypeScript per a les teves Cloud Functions.
  • Tria si vols utilitzar ESLint per a la revisió de codi.

Pas 3: Instal·lar Dependències

Navega al directori functions i instal·la les dependències necessàries.

cd functions
npm install

Escriure la Teva Primera Cloud Function

Exemple: Funció HTTP

Escrivim una simple funció HTTP que retorna "Hello, World!".

  1. Obre index.js al directori functions.
  2. Afegeix el següent codi:
const functions = require('firebase-functions');

exports.helloWorld = functions.https.onRequest((request, response) => {
  response.send("Hello, World!");
});

Desplegar la Funció

Desplega la teva funció a Firebase.

firebase deploy --only functions

Provar la Funció

Després del desplegament, Firebase proporcionarà una URL per a la teva funció. Obre la URL al teu navegador per veure el missatge "Hello, World!".

Cloud Functions Avançades

Exemple: Desencadenador de Firestore

Creem una funció que es desencadena quan es crea un document a Firestore.

  1. Obre index.js al directori functions.
  2. Afegeix el següent codi:
const functions = require('firebase-functions');
const admin = require('firebase-admin');
admin.initializeApp();

exports.createUserProfile = functions.firestore
  .document('users/{userId}')
  .onCreate((snap, context) => {
    const newValue = snap.data();
    const name = newValue.name;
    console.log(`New user created: ${name}`);
    return null;
  });

Desplegar el Desencadenador de Firestore

Desplega la teva funció a Firebase.

firebase deploy --only functions

Provar el Desencadenador de Firestore

Afegeix un nou document a la col·lecció users a Firestore. La funció hauria de registrar el nom del nou usuari.

Millors Pràctiques

  • Modularitzar el Codi: Divideix les teves funcions en fitxers separats per a una millor mantenibilitat.
  • Gestió d'Errors: Inclou sempre la gestió d'errors per gestionar problemes inesperats.
  • Configuració d'Entorn: Utilitza la configuració d'entorn de Firebase per gestionar dades sensibles.

Conclusió

Configurar Cloud Functions a Firebase et permet construir serveis backend potents i escalables amb un esforç mínim. Seguint els passos descrits en aquesta guia, pots crear, desplegar i provar les teves funcions de manera eficient. A mesura que et familiaritzis amb Cloud Functions, explora funcions més avançades i millors pràctiques per optimitzar les teves aplicacions.
