Introducció a Firebase Storage

Firebase Storage és un servei d'emmagatzematge d'objectes potent, senzill i rendible construït per a l'escala de Google. Et permet emmagatzemar i servir contingut generat per l'usuari, com ara fotos o vídeos.

Conceptes Clau

  • Buckets: Contenidors d'emmagatzematge per als teus fitxers.
  • Objects: Fitxers emmagatzemats en buckets.
  • References: Punts de referència als fitxers en el teu bucket d'emmagatzematge.
  • Metadata: Informació sobre els teus fitxers, com ara mida, tipus de contingut, etc.

Configuració de Firebase Storage

Abans de poder gestionar fitxers, has de configurar Firebase Storage en el teu projecte.

Passos per Configurar Firebase Storage

  1. Crea un Projecte Firebase: Si encara no ho has fet, crea un projecte Firebase a la consola de Firebase.
  2. Afegeix Firebase a la Teva Aplicació: Segueix les instruccions per afegir Firebase a la teva aplicació web, Android o iOS.
  3. Habilita Firebase Storage: A la consola de Firebase, navega a la secció d'Emmagatzematge i fes clic a "Comença".

Exemple: Configuració de Firebase Storage en una Aplicació Web

<!DOCTYPE html>
<html>
<head>
  <title>Exemple de Firebase Storage</title>
  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-storage.js"></script>
</head>
<body>
  <script>
    // La configuració de Firebase de la teva aplicació web
    const firebaseConfig = {
      apiKey: "YOUR_API_KEY",
      authDomain: "YOUR_AUTH_DOMAIN",
      projectId: "YOUR_PROJECT_ID",
      storageBucket: "YOUR_STORAGE_BUCKET",
      messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
      appId: "YOUR_APP_ID"
    };

    // Inicialitza Firebase
    const app = firebase.initializeApp(firebaseConfig);
    const storage = firebase.storage();
  </script>
</body>
</html>

Pujada de Fitxers

Pujar fitxers a Firebase Storage és senzill. Pots pujar fitxers des d'una aplicació web, aplicació Android o aplicació iOS.

Exemple: Pujada d'un Fitxer des d'una Aplicació Web

<input type="file" id="fileInput" />
<button onclick="uploadFile()">Puja</button>

<script>
  function uploadFile() {
    const file = document.getElementById('fileInput').files[0];
    const storageRef = firebase.storage().ref();
    const fileRef = storageRef.child(file.name);

    fileRef.put(file).then((snapshot) => {
      console.log('Fitxer pujat!');
    });
  }
</script>

Exemple: Pujada d'un Fitxer des d'una Aplicació Android

Uri file = Uri.fromFile(new File("path/to/images/rivers.jpg"));
StorageReference storageRef = FirebaseStorage.getInstance().getReference();
StorageReference riversRef = storageRef.child("images/rivers.jpg");

riversRef.putFile(file)
    .addOnSuccessListener(taskSnapshot -> {
        // Fitxer pujat amb èxit
    })
    .addOnFailureListener(exception -> {
        // Gestió de pujades no reeixides
    });

Descàrrega de Fitxers

Descarregar fitxers de Firebase Storage també és senzill. Pots obtenir l'URL de descàrrega i utilitzar-lo per obtenir el fitxer.

Exemple: Descàrrega d'un Fitxer des d'una Aplicació Web

<button onclick="downloadFile()">Descarrega</button>

<script>
  function downloadFile() {
    const storageRef = firebase.storage().ref();
    const fileRef = storageRef.child('path/to/file.jpg');

    fileRef.getDownloadURL().then((url) => {
      window.open(url);
    }).catch((error) => {
      console.error('Error en descarregar el fitxer:', error);
    });
  }
</script>

Exemple: Descàrrega d'un Fitxer des d'una Aplicació Android

StorageReference storageRef = FirebaseStorage.getInstance().getReference();
StorageReference fileRef = storageRef.child("path/to/file.jpg");

fileRef.getDownloadUrl().addOnSuccessListener(uri -> {
    // Utilitza l'URL de descàrrega
}).addOnFailureListener(exception -> {
    // Gestió d'errors
});

Gestió de Metadades de Fitxers

Pots establir i recuperar metadades per als teus fitxers a Firebase Storage.

Exemple: Establir Metadades

const fileRef = firebase.storage().ref().child('path/to/file.jpg');
const newMetadata = {
  contentType: 'image/jpeg',
  customMetadata: {
    'uploadedBy': 'User123',
    'description': 'Foto de perfil'
  }
};

fileRef.updateMetadata(newMetadata).then((metadata) => {
  console.log('Metadades actualitzades:', metadata);
}).catch((error) => {
  console.error('Error en actualitzar les metadades:', error);
});

Exemple: Recuperar Metadades

const fileRef = firebase.storage().ref().child('path/to/file.jpg');

fileRef.getMetadata().then((metadata) => {
  console.log('Metadades del fitxer:', metadata);
}).catch((error) => {
  console.error('Error en recuperar les metadades:', error);
});

Eliminació de Fitxers

Eliminar fitxers de Firebase Storage es fa utilitzant el mètode delete.

Exemple: Eliminació d'un Fitxer

const fileRef = firebase.storage().ref().child('path/to/file.jpg');

fileRef.delete().then(() => {
  console.log('Fitxer eliminat amb èxit');
}).catch((error) => {
  console.error('Error en eliminar el fitxer:', error);
});

Conclusió

Gestionar fitxers a Firebase Storage implica configurar el teu emmagatzematge, pujar i descarregar fitxers, gestionar metadades i eliminar fitxers. Firebase Storage proporciona una solució robusta i escalable per gestionar contingut generat per l'usuari, convertint-lo en una eina essencial per a aplicacions web i mòbils modernes. En dominar aquests conceptes, podràs gestionar eficientment fitxers en els teus projectes Firebase.
