Integració de Firebase amb Altres Serveis ►

Firebase App Distribution és una eina potent que permet als desenvolupadors distribuir les seves aplicacions als provadors de manera ràpida i eficient. Aquesta secció cobrirà tot, des dels conceptes bàsics de la configuració de Firebase App Distribution fins a les funcions avançades i les millors pràctiques.

Introducció a Firebase App Distribution

Firebase App Distribution proporciona una manera de distribuir versions prèvies al llançament de la teva aplicació als provadors. Això ajuda a obtenir comentaris anticipats i identificar errors abans que l'aplicació es publiqui.

Conceptes Clau

  • Gestió de Provadors: Afegeix i gestiona els provadors que rebran les teves versions d'aplicació.
  • Distribució de Versions: Carrega i distribueix versions als provadors.
  • Recollida de Comentaris: Recull comentaris dels provadors per millorar la teva aplicació.
  • Integració amb CI/CD: Automatitza el procés de distribució utilitzant eines d'Integració Contínua/Desplegament Continu.

Configuració de Firebase App Distribution

Prerequisits

  • Un projecte de Firebase.
  • Una aplicació registrada en el teu projecte de Firebase.
  • Firebase CLI instal·lat a la teva màquina.

Passos per Configurar

  1. Afegeix Firebase a la Teva Aplicació:

    • Segueix les instruccions a la consola de Firebase per afegir Firebase a la teva aplicació iOS o Android.

  2. Instal·la Firebase CLI:

    npm install -g firebase-tools

  3. Autentica Firebase CLI:

    firebase login

  4. Inicialitza Firebase en el Teu Projecte:

    firebase init

  5. Afegeix App Distribution al Teu Projecte:

    • A la consola de Firebase, navega a App Distribution i segueix les instruccions de configuració.

Distribució de la Teva Primera Versió

Carregar una Versió

  1. Construeix la Teva Aplicació:

    • Per Android: 
      ./gradlew assembleDebug
    • Per iOS: 
      xcodebuild -scheme YourScheme -sdk iphoneos -configuration Debug archive -archivePath YourApp.xcarchive

  2. Distribueix la Versió:

    firebase appdistribution:distribute path/to/your/app.apk --app your-app-id --groups testers

Afegir Provadors

  • Navega a la secció de App Distribution a la consola de Firebase.
  • Fes clic a "Testers & Groups".
  • Afegeix provadors individuals o crea grups i afegeix provadors a aquests grups.

Recollida de Comentaris

Activar la Recollida de Comentaris

  • Assegura't que els provadors tinguin instal·lada l'aplicació Firebase App Tester.
  • Els provadors poden proporcionar comentaris directament a través de l'aplicació, que seran visibles a la consola de Firebase.

Veure Comentaris

  • Navega a la secció de App Distribution a la consola de Firebase.
  • Fes clic a la versió específica per veure els comentaris dels provadors.

Integració amb CI/CD

Utilitzant Fastlane

  • Instal·la Fastlane:

    gem install fastlane

  • Afegeix Fastlane al Teu Projecte:

    fastlane init

  • Configura Fastlane per Firebase App Distribution:

    lane :distribute do
  firebase_app_distribution(
    app: "your-app-id",
    testers: "tester1@example.com, tester2@example.com",
    release_notes: "This is a test build"
  )
end

Utilitzant GitHub Actions

  • Crea un Workflow de GitHub Actions: 
    name: Distribute App

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout code
      uses: actions/checkout@v2

    - name: Set up JDK 1.8
      uses: actions/setup-java@v1
      with:
        java-version: 1.8

    - name: Build with Gradle
      run: ./gradlew build

    - name: Distribute to Firebase App Distribution
      uses: wzieba/Firebase-Distribution-Github-Action@v1
      with:
        appId: ${{ secrets.FIREBASE_APP_ID }}
        token: ${{ secrets.FIREBASE_TOKEN }}
        groups: testers
        file: app/build/outputs/apk/debug/app-debug.apk

Millors Pràctiques

  • Actualitza Regularment els Provadors: Mantingues els teus provadors compromesos actualitzant-los regularment amb noves versions.
  • Automatitza la Distribució: Utilitza eines de CI/CD per automatitzar el procés de distribució, assegurant que els provadors sempre tinguin la darrera versió.
  • Recull i Actua sobre els Comentaris: Recull activament comentaris dels provadors i utilitza'ls per millorar la teva aplicació abans del llançament oficial.

Conclusió

Firebase App Distribution és una eina essencial per a qualsevol equip de desenvolupament d'aplicacions. Simplifica el procés de distribuir versions prèvies als provadors, recollir comentaris i integrar-se amb pipelines de CI/CD. Seguint els passos descrits en aquesta secció, pots gestionar eficientment el procés de distribució de la teva aplicació i assegurar-te que la teva aplicació estigui ben provada abans d'arribar als teus usuaris.
