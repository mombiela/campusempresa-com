Visió General

El Monitoratge de Rendiment de Firebase és una eina potent que t'ajuda a obtenir informació sobre les característiques de rendiment de la teva aplicació. Et permet fer un seguiment i analitzar diverses mètriques per assegurar-te que la teva aplicació funcioni de manera suau i eficient. Aquesta secció et guiarà a través dels conceptes bàsics de la configuració i l'ús del Monitoratge de Rendiment de Firebase, des de nivells per a principiants fins a avançats.

Conceptes Clau

  • SDK de Monitoratge de Rendiment: Una biblioteca que integres a la teva aplicació per recollir dades de rendiment.
  • Traça: Un informe que conté dades capturades entre dos punts en el temps a la teva aplicació.
  • Mètrica: Una mesura que es recull durant una traça, com ara el temps de resposta o l'ús de la CPU.
  • Traça Personalitzada: Una traça definida per l'usuari que mesura el rendiment de parts específiques de la teva aplicació.
  • Monitoratge de Sol·licituds de Xarxa: Fa un seguiment automàtic del rendiment de les sol·licituds HTTP/S.

Configuració del Monitoratge de Rendiment de Firebase

Pas 1: Afegeix Firebase al Teu Projecte

  1. Crea un projecte de Firebase:

    • Ves al Firebase Console.
    • Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

  2. Afegeix el SDK de Firebase a la teva aplicació:

    • Per a Android, afegeix el SDK de Firebase als teus fitxers build.gradle.
    • Per a iOS, utilitza CocoaPods per integrar el SDK de Firebase.

Pas 2: Afegeix el SDK de Monitoratge de Rendiment

  • Android:

    dependencies {
    // Afegeix el SDK de Monitoratge de Rendiment de Firebase
    implementation 'com.google.firebase:firebase-perf:20.0.3'
}

  • iOS:

    pod 'Firebase/Performance'

Pas 3: Inicialitza el Monitoratge de Rendiment

  • Android:

    import com.google.firebase.perf.FirebasePerformance;
import com.google.firebase.perf.metrics.Trace;

public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();
        FirebasePerformance.getInstance().setPerformanceCollectionEnabled(true);
    }
}

  • iOS:

    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()
        Performance.sharedInstance().isDataCollectionEnabled = true
        return true
    }
}

Ús del Monitoratge de Rendiment

Traces Automàtiques

Firebase recull automàticament dades per a diversos escenaris de rendiment comuns, com ara l'inici de l'aplicació, el renderitzat de pantalles i les sol·licituds de xarxa.

Traces Personalitzades

Les traces personalitzades et permeten mesurar el rendiment de parts específiques de la teva aplicació.

  • Android:

    Trace myTrace = FirebasePerformance.getInstance().newTrace("test_trace");
myTrace.start();
// Codi que vols traçar
myTrace.stop();

  • iOS:

    let trace = Performance.startTrace(name: "test_trace")
// Codi que vols traçar
trace.stop()

Monitoratge de Sol·licituds de Xarxa

El Monitoratge de Rendiment de Firebase fa un seguiment automàtic del rendiment de les sol·licituds de xarxa HTTP/S.

  • Android:

    // No es necessita codi addicional; les sol·licituds de xarxa es tracen automàticament.

  • iOS:

    // No es necessita codi addicional; les sol·licituds de xarxa es tracen automàticament.

Anàlisi de les Dades de Rendiment

Un cop la teva aplicació estigui funcionant amb el Monitoratge de Rendiment, pots veure les dades recollides al Firebase Console.

  • Panell: Proporciona una visió general de les mètriques de rendiment clau.
  • Traces: Informes detallats per a cada traça, incloent-hi les traces personalitzades.
  • Sol·licituds de Xarxa: Dades de rendiment per a les sol·licituds HTTP/S.

Exemple d'Anàlisi

  • Traça d'Inici de l'Aplicació: Mesura el temps que triga a iniciar-se la teva aplicació.
  • Traça de Renderitzat de Pantalla: Mesura el temps que triga a renderitzar-se una pantalla.
  • Traça Personalitzada: Mesura el rendiment d'una funció o característica específica.

Conclusió

El Monitoratge de Rendiment de Firebase és una eina essencial per assegurar que la teva aplicació funcioni de manera eficient. Integrant el SDK, configurant traces automàtiques i personalitzades, i analitzant les dades recollides, pots identificar i abordar problemes de rendiment. Això no només millora l'experiència de l'usuari, sinó que també ajuda a mantenir una aplicació d'alta qualitat.

A les següents seccions, aprofundirem en funcions avançades i millors pràctiques per utilitzar el Monitoratge de Rendiment de Firebase de manera efectiva.
