Introducció a Firebase Hosting

Firebase Hosting és una plataforma d'allotjament de contingut web de qualitat per a desenvolupadors. Et permet desplegar aplicacions web i contingut estàtic de manera ràpida i segura.

Conceptes Clau

  • Contingut Estàtic i Dinàmic: Firebase Hosting suporta tant fitxers estàtics (HTML, CSS, JavaScript) com contingut dinàmic a través de Funcions al Núvol.
  • SSL per Defecte: Tot el contingut es serveix a través de HTTPS, garantint la seguretat.
  • CDN Global: Firebase Hosting utilitza una xarxa de distribució de contingut global (CDN) per assegurar una entrega ràpida del contingut.

Començant amb Firebase Hosting

  1. Instal·la Firebase CLI: 
    npm install -g firebase-tools
  2. Inicialitza el Teu Projecte: 
    firebase init hosting
    • Segueix les instruccions per configurar el teu projecte.
  3. Desplega el Teu Lloc: 
    firebase deploy

Exemple: Desplegant una Pàgina HTML Senzilla

  1. Crea un fitxer index.html al directori del teu projecte: 
    <!DOCTYPE html>
<html>
<head>
    <title>El Meu Lloc Allotjat a Firebase</title>
</head>
<body>
    <h1>Hola, Firebase Hosting!</h1>
</body>
</html>
  2. Inicialitza Firebase Hosting: 
    firebase init hosting
  3. Desplega el teu lloc: 
    firebase deploy

Introducció a les Funcions al Núvol de Firebase

Les Funcions al Núvol de Firebase et permeten executar codi backend en resposta a esdeveniments desencadenats per funcionalitats de Firebase i sol·licituds HTTPS.

Conceptes Clau

  • Funcions Basades en Esdeveniments: Funcions que es desencadenen per esdeveniments de Firebase (per exemple, canvis a la base de dades, esdeveniments d'autenticació).
  • Funcions HTTPS: Funcions que es poden desencadenar a través de sol·licituds HTTP.
  • Escalabilitat: S'escala automàticament amunt i avall segons el nombre de sol·licituds entrants.

Configurant Funcions al Núvol

  1. Inicialitza Funcions al Núvol: 
    firebase init functions
    • Segueix les instruccions per configurar el teu projecte.
  2. Escriu la Teva Primera Funció: 
    const functions = require('firebase-functions');

// Crea i Desplega la Teva Primera Funció al Núvol
exports.helloWorld = functions.https.onRequest((request, response) => {
    response.send("Hola des de Firebase!");
});
  3. Desplega les Teves Funcions: 
    firebase deploy --only functions

Exemple: Creant una Funció HTTPS

  1. Inicialitza Funcions al Núvol: 
    firebase init functions
  2. Escriu una funció HTTPS senzilla a functions/index.js: 
    const functions = require('firebase-functions');

exports.helloWorld = functions.https.onRequest((request, response) => {
    response.send("Hola des de Firebase!");
});
  3. Desplega la teva funció: 
    firebase deploy --only functions

Temes Avançats en Hosting i Funcions al Núvol

Dominis Personalitzats

  • Configuració de Dominis Personalitzats: Firebase Hosting et permet connectar el teu domini personalitzat al teu projecte de Firebase.
    • Ves a la Consola de Firebase > Hosting > Connecta el Domini.
    • Segueix les instruccions per verificar el teu domini i actualitzar els registres DNS.

Utilitzant Funcions al Núvol amb Firestore

  • Desencadenant Funcions en Esdeveniments de Firestore: 
    const functions = require('firebase-functions');
const admin = require('firebase-admin');
admin.initializeApp();

exports.createUserProfile = functions.firestore
    .document('users/{userId}')
    .onCreate((snap, context) => {
        const newValue = snap.data();
        // Realitza operacions amb newValue
        return admin.firestore().collection('profiles').doc(context.params.userId).set(newValue);
    });

Combinant Hosting i Funcions al Núvol

  • Servint Contingut Dinàmic: Utilitza Funcions al Núvol per servir contingut dinàmic al costat de fitxers estàtics. 
    const functions = require('firebase-functions');
const express = require('express');
const app = express();

app.get('/timestamp', (req, res) => {
    res.send(`Timestamp: ${Date.now()}`);
});

exports.app = functions.https.onRequest(app);

Conclusió

Firebase Hosting i Funcions al Núvol proporcionen una combinació poderosa per desplegar tant aplicacions web estàtiques com dinàmiques. Amb Firebase Hosting, pots desplegar ràpidament contingut estàtic segur i de càrrega ràpida, mentre que les Funcions al Núvol et permeten afegir lògica backend que s'escala automàticament. En dominar aquestes eines, pots construir aplicacions web robustes i escalables amb facilitat.
