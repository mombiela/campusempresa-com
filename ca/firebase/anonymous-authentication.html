Autenticació Personalitzada ►

L'Autenticació Anònima a Firebase permet als usuaris iniciar sessió a la teva aplicació sense proporcionar cap credencial. Això pot ser útil per permetre als usuaris provar la teva aplicació abans de decidir-se a registrar-se o iniciar sessió amb un mètode d'autenticació més permanent.

Conceptes Clau

  • Usuari Anònim: Un usuari que està autenticat sense proporcionar cap credencial.
  • Compte Temporal: Un compte que es crea per a l'usuari anònim, que posteriorment es pot enllaçar a un compte permanent.
  • Autenticació de Firebase: Un servei que proporciona serveis de backend per ajudar a autenticar usuaris a la teva aplicació.

Configuració de l'Autenticació de Firebase

Abans de poder utilitzar l'Autenticació Anònima, has de configurar l'Autenticació de Firebase al teu projecte.

  1. Crea un Projecte de Firebase:

    • Ves al Firebase Console.
    • Fes clic a "Afegeix projecte" i segueix els passos de configuració.

  2. Afegeix Firebase a la teva Aplicació:

    • Segueix les instruccions per afegir Firebase a la teva aplicació web, iOS o Android.

  3. Habilita l'Autenticació Anònima:

    • Al Firebase Console, navega a la secció d'Autenticació.
    • Fes clic a la pestanya "Mètode d'inici de sessió".
    • Habilita l'autenticació "Anònima".

Implementació de l'Autenticació Anònima

Web

Per implementar l'Autenticació Anònima en una aplicació web, has d'utilitzar el Firebase JavaScript SDK.

<!DOCTYPE html>
<html>
<head>
  <title>Autenticació Anònima</title>
  <script src="https://www.gstatic.com/firebasejs/9.6.1/firebase-app.js"></script>
  <script src="https://www.gstatic.com/firebasejs/9.6.1/firebase-auth.js"></script>
</head>
<body>
  <script>
    // La configuració de Firebase de la teva aplicació web
    const firebaseConfig = {
      apiKey: "YOUR_API_KEY",
      authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
      projectId: "YOUR_PROJECT_ID",
      storageBucket: "YOUR_PROJECT_ID.appspot.com",
      messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
      appId: "YOUR_APP_ID"
    };

    // Inicialitza Firebase
    const app = firebase.initializeApp(firebaseConfig);
    const auth = firebase.auth();

    // Inicia sessió anònimament
    auth.signInAnonymously()
      .then(() => {
        console.log("Usuari ha iniciat sessió anònimament");
      })
      .catch((error) => {
        console.error("Error iniciant sessió anònimament:", error);
      });
  </script>
</body>
</html>

Android

Per implementar l'Autenticació Anònima en una aplicació Android, has d'utilitzar el Firebase Android SDK.

  1. Afegeix l'Autenticació de Firebase a la teva aplicació:

    • Afegeix la dependència de l'Autenticació de Firebase al teu fitxer build.gradle: 
      implementation 'com.google.firebase:firebase-auth:21.0.1'

  2. Inicia sessió Anònimament:

    FirebaseAuth auth = FirebaseAuth.getInstance();
auth.signInAnonymously()
    .addOnCompleteListener(this, task -> {
        if (task.isSuccessful()) {
            // Inici de sessió exitós
            FirebaseUser user = auth.getCurrentUser();
            Log.d("Auth", "signInAnonymously:success");
        } else {
            // Inici de sessió fallit
            Log.w("Auth", "signInAnonymously:failure", task.getException());
        }
    });

iOS

Per implementar l'Autenticació Anònima en una aplicació iOS, has d'utilitzar el Firebase iOS SDK.

  1. Afegeix l'Autenticació de Firebase a la teva aplicació:

    • Afegeix el pod de l'Autenticació de Firebase al teu Podfile: 
      pod 'Firebase/Auth'

  2. Inicia sessió Anònimament:

    import Firebase

// Inicialitza Firebase
FirebaseApp.configure()

let auth = Auth.auth()
auth.signInAnonymously { (authResult, error) in
    if let error = error {
        print("Error iniciant sessió anònimament: \\(error.localizedDescription)")
        return
    }
    // Usuari ha iniciat sessió
    let user = authResult?.user
    print("Usuari ha iniciat sessió anònimament amb UID: \\(user?.uid ?? "")")
}

Enllaçant Comptes Anònims

Els comptes anònims es poden enllaçar a comptes permanents per conservar les dades de l'usuari. Això és útil si un usuari decideix registrar-se després d'utilitzar l'aplicació anònimament.

Exemple: Enllaçant a un Compte de Correu Electrònic/Contrasenya

const email = "user@example.com";
const password = "password123";

// Inicia sessió anònimament
auth.signInAnonymously()
  .then(() => {
    // Enllaça el compte anònim a un compte de correu electrònic/contrasenya
    const credential = firebase.auth.EmailAuthProvider.credential(email, password);
    return auth.currentUser.linkWithCredential(credential);
  })
  .then((usercred) => {
    const user = usercred.user;
    console.log("Compte anònim actualitzat amb èxit", user);
  })
  .catch((error) => {
    console.error("Error actualitzant el compte anònim", error);
  });

Resum

L'Autenticació Anònima a Firebase és una característica potent que permet als usuaris interactuar amb la teva aplicació sense necessitat de registrar-se o iniciar sessió. Això pot millorar l'experiència de l'usuari reduint la fricció i permetent als usuaris provar la teva aplicació abans de comprometre's amb un compte permanent. Seguint els passos descrits anteriorment, pots implementar fàcilment l'Autenticació Anònima a les teves aplicacions web, Android o iOS. A més, pots enllaçar comptes anònims a comptes permanents, assegurant que les dades de l'usuari es conserven quan decideixen registrar-se.
