Introducció

Les propietats de l'usuari a Firebase són atributs que pots definir per descriure segments de la teva base d'usuaris. Aquestes propietats t'ajuden a entendre millor els teus usuaris i es poden utilitzar per dirigir audiències específiques per a anàlisis, notificacions i més.

Conceptes Clau

  • Propietats de l'Usuari: Atributs personalitzats que defineixes per descriure segments de la teva base d'usuaris.
  • Propietats de l'Usuari Predefinides: Firebase proporciona algunes propietats de l'usuari predefinides com first_open_time, last_sign_in_time, etc.
  • Propietats de l'Usuari Personalitzades: Pots definir fins a 25 propietats de l'usuari personalitzades per projecte.

Configuració de les Propietats de l'Usuari

Utilitzant la Consola de Firebase

  1. Navega a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Ves a la secció "Analytics".
  4. Fes clic a "User Properties".
  5. Afegeix una nova propietat de l'usuari definint el seu nom i descripció.

Utilitzant el SDK de Firebase

Android (Java)

// Importa Firebase Analytics
import com.google.firebase.analytics.FirebaseAnalytics;

// Obté la instància de Firebase Analytics
FirebaseAnalytics mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);

// Configura la propietat de l'usuari
mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

iOS (Swift)

// Importa Firebase
import Firebase

// Obté la instància de Firebase Analytics
let analytics = Analytics.self

// Configura la propietat de l'usuari
analytics.setUserProperty("favorite_food", forName: "pizza")

Web (JavaScript)

// Importa Firebase
import firebase from "firebase/app";
import "firebase/analytics";

// Inicialitza Firebase
firebase.initializeApp({
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID",
  measurementId: "YOUR_MEASUREMENT_ID"
});

// Obté la instància de Firebase Analytics
const analytics = firebase.analytics();

// Configura la propietat de l'usuari
analytics.setUserProperties({ favorite_food: "pizza" });

Utilitzant les Propietats de l'Usuari en Analytics

Creació d'Audiències

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció "Analytics".
  4. Fes clic a "Audiences".
  5. Crea una nova audiència definint condicions basades en propietats de l'usuari.

Filtratge d'Informes

  • Pots filtrar els teus informes d'analítica basant-te en propietats de l'usuari per obtenir informació sobre segments específics de la teva base d'usuaris.

Millors Pràctiques

  • Mantén-ho Simple: Utilitza noms clars i concisos per a les teves propietats de l'usuari.
  • Consistència: Assegura't de mantenir la consistència en les convencions de nomenclatura a través de diferents plataformes.
  • Privacitat: Evita utilitzar informació sensible com a propietats de l'usuari.

Exemples d'Ús

Dirigir Notificacions

Pots utilitzar propietats de l'usuari per dirigir notificacions a segments específics d'usuaris. Per exemple, si tens una propietat de l'usuari favorite_food, pots enviar una notificació sobre un descompte en pizza als usuaris que han configurat favorite_food com "pizza".

Personalització de l'Experiència de l'Usuari

Utilitza propietats de l'usuari per personalitzar l'experiència de l'usuari. Per exemple, si tens una propietat de l'usuari preferred_language, pots mostrar contingut en l'idioma preferit de l'usuari.

Conclusió

Les propietats de l'usuari a Firebase són eines poderoses per entendre i segmentar la teva base d'usuaris. Configurant i utilitzant les propietats de l'usuari de manera efectiva, pots obtenir informació valuosa i crear experiències més personalitzades per als teus usuaris. Recorda seguir les millors pràctiques per assegurar la consistència i la privacitat.
