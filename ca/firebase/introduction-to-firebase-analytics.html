Firebase Analytics és una eina potent que t'ajuda a entendre com els usuaris interactuen amb la teva aplicació. Proporciona informació sobre el comportament dels usuaris, que es pot utilitzar per prendre decisions informades per millorar el rendiment i l'experiència de l'usuari de la teva aplicació.

Què és Firebase Analytics?

Firebase Analytics és una solució gratuïta de mesura d'aplicacions que proporciona informació sobre l'ús de l'aplicació i el compromís dels usuaris. S'integra perfectament amb altres serveis de Firebase i productes de Google, cosa que la converteix en una eina versàtil per als desenvolupadors d'aplicacions.

Característiques Clau de Firebase Analytics

  • Seguiment d'Esdeveniments: Fa un seguiment automàtic de les interaccions dels usuaris i esdeveniments personalitzats.
  • Propietats dels Usuaris: Defineix segments de la teva base d'usuaris.
  • Segmentació d'Audiència: Crea audiències basades en el comportament dels usuaris.
  • Integració amb Altres Serveis de Firebase: Funciona bé amb Firebase Crashlytics, Firebase Remote Config, i més.
  • Suport Multiplataforma: Disponible per a aplicacions iOS, Android i web.

Configuració de Firebase Analytics

Pas 1: Crear un Projecte de Firebase

  1. Ves a la Consola de Firebase.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions en pantalla per crear un nou projecte.

Pas 2: Afegeix Firebase a la Teva Aplicació

Per a Android

  1. Afegeix el SDK de Firebase al teu projecte incloent el següent als teus arxius build.gradle:
// Project-level build.gradle
buildscript {
    dependencies {
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// App-level build.gradle
apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services'

dependencies {
    implementation 'com.google.firebase:firebase-analytics:19.0.0'
}
  1. Sincronitza el teu projecte amb els arxius de Gradle.

Per a iOS

  1. Afegeix el SDK de Firebase al teu projecte utilitzant CocoaPods. Afegeix el següent al teu Podfile:
pod 'Firebase/Analytics'
  1. Executa pod install i obre l'arxiu .xcworkspace.

Pas 3: Inicialitza Firebase a la Teva Aplicació

Per a Android

import com.google.firebase.analytics.FirebaseAnalytics;

public class MainActivity extends AppCompatActivity {
    private FirebaseAnalytics mFirebaseAnalytics;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Inicialitza Firebase Analytics
        mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);
    }
}

Per a iOS

import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicialitza Firebase
        FirebaseApp.configure()
        return true
    }
}

Seguiment d'Esdeveniments

Esdeveniments Recollits Automàticament

Firebase Analytics registra automàticament alguns esdeveniments, com ara first_open, app_update, i in_app_purchase.

Esdeveniments Personalitzats

Per registrar esdeveniments personalitzats, utilitza el mètode logEvent.

Per a Android

Bundle bundle = new Bundle();
bundle.putString(FirebaseAnalytics.Param.ITEM_ID, "id123");
bundle.putString(FirebaseAnalytics.Param.ITEM_NAME, "name123");
bundle.putString(FirebaseAnalytics.Param.CONTENT_TYPE, "image");
mFirebaseAnalytics.logEvent(FirebaseAnalytics.Event.SELECT_CONTENT, bundle);

Per a iOS

Analytics.logEvent(AnalyticsEventSelectContent, parameters: [
    AnalyticsParameterItemID: "id123" as NSObject,
    AnalyticsParameterItemName: "name123" as NSObject,
    AnalyticsParameterContentType: "image" as NSObject
])

Anàlisi de Dades

Consola de Firebase

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció "Analytics" per veure diversos informes i informació.

Integració amb Google Analytics

Firebase Analytics es pot integrar amb Google Analytics per a una anàlisi i informes més avançats.

Conclusió

Firebase Analytics és una eina essencial per entendre el comportament dels usuaris i millorar la teva aplicació. Configurant-lo i fent un seguiment tant d'esdeveniments automàtics com personalitzats, pots obtenir informació valuosa sobre com els usuaris interactuen amb la teva aplicació. Aquestes dades es poden utilitzar per prendre decisions informades per millorar l'experiència de l'usuari i el rendiment de l'aplicació.
