Introducció

Firestore és una base de dades flexible i escalable per al desenvolupament mòbil, web i de servidor de Firebase i Google Cloud Platform. En aquesta secció, cobrirem com configurar Firestore per al teu projecte, des de la configuració inicial fins a la integració a la teva aplicació.

Prerequisits

Abans de configurar Firestore, assegura't de tenir:

  • Un compte de Google.
  • Un projecte de Firebase creat.
  • Coneixements bàsics de la Consola de Firebase.

Guia Pas a Pas per Configurar Firestore

1. Crear un Projecte de Firebase

  1. Ves a la Consola de Firebase.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions en pantalla per crear un nou projecte.

2. Afegeix Firestore al Teu Projecte de Firebase

  1. A la Consola de Firebase, selecciona el teu projecte.
  2. Al menú de l'esquerra, fes clic a "Firestore Database".
  3. Fes clic a "Crea base de dades".
  4. Tria un mode inicial per a les teves Regles de Seguretat de Firestore:
    • Mode de prova: Permet a qualsevol amb la referència de la teva base de dades llegir i escriure a la teva base de dades. Adequat per al desenvolupament.
    • Mode bloquejat: Denega totes les lectures i escriptures des de clients mòbils i web. Adequat per a la producció.
  5. Selecciona una ubicació de Cloud Firestore i fes clic a "Habilita".

3. Integra Firestore a la Teva Aplicació

Per utilitzar Firestore a la teva aplicació, has d'afegir el SDK de Firebase al teu projecte.

Per a Aplicacions Web

  1. Afegeix el SDK de Firebase al teu projecte: 
    <!-- Afegeix els SDKs de Firebase -->
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js"></script>
  2. Inicialitza Firebase i Firestore: 
    // Importa les funcions que necessites dels SDKs que necessites
import { initializeApp } from "https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js";
import { getFirestore } from "https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js";

// La configuració de Firebase de la teva aplicació web
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};

// Inicialitza Firebase
const app = initializeApp(firebaseConfig);

// Inicialitza Firestore
const db = getFirestore(app);

Per a Aplicacions Android

  1. Afegeix el SDK de Firebase al teu projecte: 
    // Afegeix la dependència per al SDK de Firebase
implementation 'com.google.firebase:firebase-firestore:23.0.3'
  2. Inicialitza Firebase i Firestore al teu MainActivity: 
    // Importa les biblioteques necessàries de Firebase
import com.google.firebase.firestore.FirebaseFirestore;

// Inicialitza Firestore
FirebaseFirestore db = FirebaseFirestore.getInstance();

Per a Aplicacions iOS

  1. Afegeix el SDK de Firebase al teu projecte utilitzant CocoaPods: 
    pod 'Firebase/Firestore'
  2. Inicialitza Firebase i Firestore al teu AppDelegate: 
    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

    var window: UIWindow?

    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()

        // Inicialitza Firestore
        let db = Firestore.firestore()

        return true
    }
}

Conclusió

Configurar Firestore implica crear un projecte de Firebase, habilitar Firestore a la Consola de Firebase i integrar Firestore a la teva aplicació utilitzant els SDKs adequats. Seguint aquests passos, pots començar a utilitzar Firestore per emmagatzemar i gestionar les dades de la teva aplicació de manera eficient. A les següents seccions, explorarem com realitzar operacions CRUD i gestionar dades a Firestore.
