Introducció a Firebase Analytics

Firebase Analytics és una eina potent que et permet fer un seguiment del comportament dels usuaris i obtenir informació sobre com interactuen amb la teva aplicació. Proporciona informes detallats i panells de control per ajudar-te a entendre el compromís dels usuaris, la retenció i altres mètriques crítiques.

Configuració de Firebase Analytics

Abans d'endinsar-te en l'analítica avançada, assegura't que Firebase Analytics està correctament configurat en el teu projecte.

Passos per Configurar Firebase Analytics:

  1. Crea un Projecte Firebase:

    • Ves al Firebase Console.
    • Fes clic a "Afegeix projecte" i segueix les instruccions de configuració.

  2. Afegeix Firebase a la Teva Aplicació:

    • Per Android: Afegeix el fitxer google-services.json al teu projecte.
    • Per iOS: Afegeix el fitxer GoogleService-Info.plist al teu projecte.

  3. Afegeix el SDK de Firebase:

    • Per Android: 
      dependencies {
    implementation platform('com.google.firebase:firebase-bom:28.4.1')
    implementation 'com.google.firebase:firebase-analytics'
}
    • Per iOS: 
      pod 'Firebase/Analytics'

  4. Inicialitza Firebase:

    • Per Android: 
      import com.google.firebase.analytics.FirebaseAnalytics;

public class MainActivity extends AppCompatActivity {
    private FirebaseAnalytics mFirebaseAnalytics;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);
    }
}
    • Per iOS: 
      import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()
        return true
    }
}

Esdeveniments i Paràmetres Personalitzats

Els esdeveniments i paràmetres personalitzats et permeten fer un seguiment d'accions i atributs específics que són rellevants per a la teva aplicació.

Registre d'Esdeveniments Personalitzats

  • Android:

    Bundle bundle = new Bundle();
bundle.putString(FirebaseAnalytics.Param.ITEM_ID, "id123");
bundle.putString(FirebaseAnalytics.Param.ITEM_NAME, "name123");
bundle.putString(FirebaseAnalytics.Param.CONTENT_TYPE, "image");
mFirebaseAnalytics.logEvent(FirebaseAnalytics.Event.SELECT_CONTENT, bundle);

  • iOS:

    Analytics.logEvent(AnalyticsEventSelectContent, parameters: [
  AnalyticsParameterItemID: "id123" as NSObject,
  AnalyticsParameterItemName: "name123" as NSObject,
  AnalyticsParameterContentType: "image" as NSObject
])

Ús de Paràmetres Personalitzats

Els paràmetres personalitzats proporcionen context addicional als teus esdeveniments.

  • Android:

    Bundle bundle = new Bundle();
bundle.putString("custom_param", "value123");
mFirebaseAnalytics.logEvent("custom_event", bundle);

  • iOS:

    Analytics.logEvent("custom_event", parameters: [
  "custom_param": "value123" as NSObject
])

Propietats d'Usuari

Les propietats d'usuari són atributs que defineixes per descriure segments de la teva base d'usuaris, com ara la preferència d'idioma o la ubicació geogràfica.

Configuració de Propietats d'Usuari

  • Android:

    mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

  • iOS:

    Analytics.setUserProperty("favorite_food", forName: "pizza")

Tècniques d'Anàlisi Avançades

Les tècniques avançades impliquen l'ús de Firebase Analytics conjuntament amb altres serveis de Firebase i eines externes.

Anàlisi de Funnells

L'anàlisi de funnells t'ajuda a entendre els passos que prenen els usuaris per completar un objectiu a la teva aplicació.

Anàlisi de Cohorts

L'anàlisi de cohorts agrupa els usuaris basant-se en característiques compartides per analitzar el seu comportament al llarg del temps.

Integració amb BigQuery

BigQuery et permet realitzar consultes complexes sobre les dades de Firebase Analytics.

Exportació de Dades a BigQuery

  1. Habilita la Integració amb BigQuery:

    • Ves al Firebase Console.
    • Navega a la secció "Integracions".
    • Enllaça el teu projecte Firebase amb BigQuery.

  2. Consulta de Dades:

    SELECT
  event_name,
  COUNT(*) as event_count
FROM
  `your_project_id.analytics_1234567890.events_*`
WHERE
  _TABLE_SUFFIX BETWEEN '20210101' AND '20210131'
GROUP BY
  event_name
ORDER BY
  event_count DESC

Conclusió

Firebase Analytics proporciona una plataforma robusta per fer un seguiment i analitzar el comportament dels usuaris a la teva aplicació. Configurant esdeveniments personalitzats, paràmetres i propietats d'usuari, pots obtenir informació profunda sobre les interaccions dels usuaris. Les tècniques avançades com l'anàlisi de funnells i cohorts, juntament amb la integració amb BigQuery, et permeten realitzar anàlisis detallades i complexes, ajudant-te a prendre decisions basades en dades per millorar la teva aplicació.
