Introducció

Firebase Realtime Database és una base de dades NoSQL allotjada al núvol que et permet emmagatzemar i sincronitzar dades entre els teus usuaris en temps real. Aquesta secció et guiarà a través de la configuració de la Base de Dades en Temps Real des de zero, cobrint tant configuracions bàsiques com avançades.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un compte de Google
  • Coneixements bàsics de JavaScript (per a web) o Kotlin/Java (per a Android)
  • Node.js instal·lat (per a web)

Creació d'un Projecte Firebase

  1. Ves al Consola de Firebase: Navega a Firebase Console.
  2. Afegeix un Nou Projecte:
    • Fes clic a "Afegeix projecte".
    • Introdueix el nom del teu projecte i segueix les instruccions en pantalla.
    • Un cop creat el projecte, seràs redirigit al tauler del projecte.

Afegir la Base de Dades en Temps Real al Teu Projecte

  1. Navega a la Secció de Base de Dades:
    • A la Consola de Firebase, selecciona el teu projecte.
    • Fes clic a "Realtime Database" al menú de l'esquerra.
  2. Crea una Base de Dades:
    • Fes clic a "Crea base de dades".
    • Tria la ubicació de la base de dades (normalment la més propera als teus usuaris).
    • Selecciona les regles de seguretat (comença en mode de prova per a finalitats de desenvolupament).

Integrar la Base de Dades en Temps Real amb la Teva Aplicació

Per a Web

  1. Instal·la el SDK de Firebase: 
    npm install firebase
  2. Inicialitza Firebase a la Teva Aplicació: 
    // Importa Firebase
import firebase from 'firebase/app';
import 'firebase/database';

// La configuració de Firebase de la teva aplicació web
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  databaseURL: "https://YOUR_PROJECT_ID.firebaseio.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

// Inicialitza Firebase
firebase.initializeApp(firebaseConfig);

// Obté una referència al servei de base de dades
const database = firebase.database();

Per a Android

  1. Afegeix el SDK de Firebase:
    • Obre build.gradle (nivell de Projecte) i afegeix el següent: 
      classpath 'com.google.gms:google-services:4.3.3'
    • Obre build.gradle (nivell d'Aplicació) i afegeix el següent: 
      implementation 'com.google.firebase:firebase-database:19.5.1'
apply plugin: 'com.google.gms.google-services'
  2. Inicialitza Firebase a la Teva Aplicació: 
    // Importa Firebase
import com.google.firebase.database.DatabaseReference;
import com.google.firebase.database.FirebaseDatabase;

// Inicialitza Firebase
FirebaseDatabase database = FirebaseDatabase.getInstance();
DatabaseReference myRef = database.getReference("message");

// Escriu un missatge a la base de dades
myRef.setValue("Hello, World!");

Operacions Bàsiques CRUD

Crear i Llegir Dades

  • Web:

    // Crear dades
database.ref('users/1').set({
  username: "john_doe",
  email: "john@example.com"
});

// Llegir dades
database.ref('users/1').once('value').then((snapshot) => {
  const data = snapshot.val();
  console.log(data);
});

  • Android:

    // Crear dades
myRef.child("users").child("1").setValue(new User("john_doe", "john@example.com"));

// Llegir dades
myRef.child("users").child("1").addListenerForSingleValueEvent(new ValueEventListener() {
    @Override
    public void onDataChange(DataSnapshot dataSnapshot) {
        User user = dataSnapshot.getValue(User.class);
        System.out.println(user.username);
    }

    @Override
    public void onCancelled(DatabaseError databaseError) {
        // Gestiona possibles errors.
    }
});

Actualitzar i Eliminar Dades

  • Web:

    // Actualitzar dades
database.ref('users/1').update({
  username: "john_doe_updated"
});

// Eliminar dades
database.ref('users/1').remove();

  • Android:

    // Actualitzar dades
myRef.child("users").child("1").child("username").setValue("john_doe_updated");

// Eliminar dades
myRef.child("users").child("1").removeValue();

Configuració Avançada

Regles de Seguretat

  • Configuració de Regles:
    • Navega a la pestanya "Rules" a la secció de Realtime Database.
    • Defineix regles per assegurar les teves dades.
    {
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

Capacitats Fora de Línia

  • Web:

    // Habilita capacitats fora de línia
firebase.database().goOffline();
firebase.database().goOnline();

  • Android:

    // Habilita capacitats fora de línia
FirebaseDatabase.getInstance().setPersistenceEnabled(true);

Conclusió

Configurar Firebase Realtime Database implica crear un projecte Firebase, integrar la base de dades amb la teva aplicació i realitzar operacions bàsiques CRUD. A mesura que progressis, podràs configurar funcions avançades com les regles de seguretat i les capacitats fora de línia per millorar la funcionalitat i la seguretat de la teva aplicació. Amb aquestes bases, estàs ben equipat per construir aplicacions col·laboratives en temps real.
