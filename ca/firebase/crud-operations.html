Introducció a Firebase

Firebase és una plataforma desenvolupada per Google per crear aplicacions mòbils i web. Proporciona una varietat d'eines i serveis per ajudar els desenvolupadors a construir aplicacions d'alta qualitat, incloent-hi una base de dades NoSQL en temps real, autenticació, analítiques i més.

Què són les operacions CRUD?

CRUD significa Crear, Llegir, Actualitzar i Esborrar. Aquestes són les quatre operacions bàsiques que es poden realitzar sobre les dades en una base de dades. Entendre com realitzar aquestes operacions és fonamental per treballar amb qualsevol base de dades, incloent Firebase.

Configuració de Firebase

Abans d'endinsar-nos en les operacions CRUD, configurem Firebase per al teu projecte.

  1. Crear un projecte de Firebase:

    • Ves a la Consola de Firebase.
    • Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

  2. Afegeix Firebase a la teva aplicació:

    • Per a aplicacions web, afegeix el SDK de Firebase al teu projecte.
    • Per a aplicacions Android/iOS, segueix les instruccions específiques proporcionades a la Consola de Firebase.

  3. Inicialitza Firebase a la teva aplicació:

    // Importa els mòduls de Firebase que necessites a la teva aplicació.
import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";

// TODO: Reemplaça el següent amb la configuració del projecte de Firebase de la teva aplicació
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

// Inicialitza Firebase
const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

Operació de Creació

Crear dades a Firebase implica afegir nous documents a una col·lecció.

Exemple: Afegir un nou document

import { collection, addDoc } from "firebase/firestore"; 

async function addUser() {
  try {
    const docRef = await addDoc(collection(db, "users"), {
      first: "Ada",
      last: "Lovelace",
      born: 1815
    });
    console.log("Document escrit amb ID: ", docRef.id);
  } catch (e) {
    console.error("Error afegint document: ", e);
  }
}

addUser();
  • Explicació:
    • collection(db, "users"): Especifica la col·lecció on s'afegirà el document.
    • addDoc: Afegeix un nou document amb les dades especificades.

Operació de Lectura

Llegir dades de Firebase implica recuperar documents d'una col·lecció.

Exemple: Obtenir tots els documents d'una col·lecció

import { collection, getDocs } from "firebase/firestore"; 

async function getUsers() {
  const querySnapshot = await getDocs(collection(db, "users"));
  querySnapshot.forEach((doc) => {
    console.log(`${doc.id} => ${doc.data()}`);
  });
}

getUsers();
  • Explicació:
    • getDocs(collection(db, "users")): Recupera tots els documents de la col·lecció "users".
    • querySnapshot.forEach: Itera sobre cada document de la col·lecció.

Operació d'Actualització

Actualitzar dades a Firebase implica modificar un document existent.

Exemple: Actualitzar un document

import { doc, updateDoc } from "firebase/firestore"; 

async function updateUser(userId) {
  const userRef = doc(db, "users", userId);

  // Estableix el camp "last" de l'usuari
  await updateDoc(userRef, {
    last: "Byron",
    born: 1812
  });
}

updateUser("USER_DOCUMENT_ID");
  • Explicació:
    • doc(db, "users", userId): Especifica el document que s'actualitzarà.
    • updateDoc: Actualitza els camps especificats del document.

Operació d'Esborrat

Esborrar dades a Firebase implica eliminar un document d'una col·lecció.

Exemple: Esborrar un document

import { doc, deleteDoc } from "firebase/firestore"; 

async function deleteUser(userId) {
  await deleteDoc(doc(db, "users", userId));
}

deleteUser("USER_DOCUMENT_ID");
  • Explicació:
    • doc(db, "users", userId): Especifica el document que s'eliminarà.
    • deleteDoc: Elimina el document especificat.

Conclusió

Les operacions CRUD són fonamentals per gestionar dades en qualsevol base de dades, incloent Firebase. Dominant aquestes operacions, pots crear, llegir, actualitzar i esborrar dades de manera efectiva a les teves aplicacions de Firebase. Practica aquestes operacions per convertir-te en un expert en la gestió de Firebase Firestore, i explora funcions més avançades a mesura que et sentis més còmode amb els conceptes bàsics.
