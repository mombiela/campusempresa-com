Introducció a Firebase Hosting

Firebase Hosting és un servei d'allotjament de contingut web de qualitat per a desenvolupadors. Està dissenyat per allotjar la teva aplicació web, contingut estàtic i dinàmic, i microserveis.

Característiques Clau

  • Allotjament ràpid i segur: CDN global, SSL per defecte i memòria cau potent.
  • Desplegament amb un sol clic: Desplega la teva aplicació amb una sola comanda.
  • Dominis personalitzats: Connecta fàcilment el teu domini personalitzat.

Configuració de Firebase Hosting

Prerequisits

  • Node.js i npm instal·lats a la teva màquina.
  • Firebase CLI instal·lat globalment via npm.
  • Un projecte de Firebase creat a la Consola de Firebase.

Instal·lació de Firebase CLI

Per instal·lar Firebase CLI, executa la següent comanda al teu terminal:

npm install -g firebase-tools

Inicialització de Firebase al Teu Projecte

Navega al directori del teu projecte i executa:

firebase init
  • Selecciona "Hosting" de la llista de característiques de Firebase.
  • Tria un projecte de Firebase existent o crea'n un de nou.
  • Especifica el directori públic (per exemple, build o public).
  • Configura com a aplicació de pàgina única si és aplicable (per exemple, per a aplicacions React o Angular).

Exemple: Inicialització de Firebase Hosting

cd your-project-directory
firebase init

Segueix les indicacions:

  • Selecciona "Hosting".
  • Tria el teu projecte de Firebase.
  • Defineix el directori públic (per exemple, build).
  • Configura com a aplicació de pàgina única si utilitzes un framework com React.

Desplegament de la Teva Aplicació Web

Construcció de la Teva Aplicació Web

Abans de desplegar, assegura't que la teva aplicació web està construïda i llesta per a producció. Per exemple, si utilitzes React:

npm run build

Desplegament a Firebase Hosting

Executa la següent comanda per desplegar la teva aplicació:

firebase deploy

Aquesta comanda:

  • Carregarà els teus fitxers a Firebase Hosting.
  • Proporcionarà una URL on la teva aplicació està allotjada.

Exemple: Desplegament d'una Aplicació React

  1. Construeix la teva aplicació: 
    npm run build
  2. Desplega a Firebase: 
    firebase deploy

Configuració Avançada

Dominis Personalitzats

Pots afegir un domini personalitzat al teu projecte de Firebase:

  1. Ves a la Consola de Firebase.
  2. Navega a Hosting.
  3. Fes clic a "Add Custom Domain" i segueix les instruccions.

Redireccions i Reescriptures

Pots configurar redireccions i reescriptures al teu fitxer firebase.json:

{
  "hosting": {
    "public": "build",
    "rewrites": [
      {
        "source": "**",
        "destination": "/index.html"
      }
    ],
    "redirects": [
      {
        "source": "/old-page",
        "destination": "/new-page",
        "type": 301
      }
    ]
  }
}

Resum

En aquest tema, has après com desplegar una aplicació web utilitzant Firebase Hosting. Hem cobert:

  • Configuració de Firebase Hosting.
  • Construcció i desplegament de la teva aplicació web.
  • Configuracions avançades com dominis personalitzats, redireccions i reescriptures.

Firebase Hosting proporciona una manera ràpida, segura i fàcil de desplegar les teves aplicacions web, fent-lo una excel·lent opció per a desenvolupadors de tots els nivells.
