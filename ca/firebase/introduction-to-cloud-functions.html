Visió general

Les Funcions Cloud per Firebase són un marc sense servidor que et permet executar automàticament codi de backend en resposta a esdeveniments desencadenats per funcions de Firebase i sol·licituds HTTPS. Aquesta secció et guiarà a través dels conceptes bàsics de les Funcions Cloud, des de la configuració fins a casos d'ús avançats.

Què són les Funcions Cloud?

Les Funcions Cloud són funcions JavaScript de propòsit únic que s'executen en un entorn gestionat. Es poden desencadenar per diversos esdeveniments, com ara canvis a la teva base de dades de Firebase, nous registres d'usuaris o sol·licituds HTTP.

Conceptes clau

  • Sense servidor: No cal gestionar servidors; Firebase s'encarrega de la infraestructura.
  • Basat en esdeveniments: Les funcions es desencadenen per esdeveniments específics.
  • Escalable: S'escala automàticament cap amunt o cap avall segons la demanda.
  • Segur: S'executa en un entorn segur amb autenticació i autorització integrades.

Configuració de les Funcions Cloud

Abans de poder començar a escriure Funcions Cloud, has de configurar el teu entorn de desenvolupament.

Prerequisits

  • Node.js i npm instal·lats
  • Firebase CLI instal·lat
  • Un projecte de Firebase

Passos per configurar

  1. Instal·la Firebase CLI: 
    npm install -g firebase-tools
  2. Inicialitza Firebase al teu projecte: 
    firebase login
firebase init functions
  3. Tria el teu projecte de Firebase i configura el teu directori de funcions.

Escriure la teva primera Funció Cloud

Escrivim una funció HTTP simple que respon amb "Hello, World!".

Codi d'exemple

const functions = require('firebase-functions');

// Crea i desplega una funció HTTP
exports.helloWorld = functions.https.onRequest((request, response) => {
  response.send("Hello, World!");
});

Explicació

  • functions.https.onRequest: Això especifica que la funció es desencadenarà per una sol·licitud HTTP.
  • request i response: Aquests són els objectes estàndard d'Express.js utilitzats per gestionar sol·licituds i respostes HTTP.

Desplegament de Funcions Cloud

Un cop hagis escrit la teva funció, has de desplegar-la a Firebase.

Passos per desplegar

  1. Desplega la funció: 
    firebase deploy --only functions
  2. Accedeix a la teva funció: Firebase proporcionarà una URL on la teva funció està allotjada.

Casos d'ús avançats

Les Funcions Cloud poden fer molt més que només respondre a sol·licituds HTTP. Aquí hi ha alguns casos d'ús avançats:

Desencadenants de base de dades

Desencadena funcions en resposta a canvis a la teva base de dades en temps real de Firebase o Firestore.

Codi d'exemple

exports.onUserCreate = functions.auth.user().onCreate((user) => {
  console.log('Un nou usuari s\'ha registrat:', user.email);
});

Desencadenants de Firestore

Desencadena funcions quan es creen, actualitzen o eliminen documents a Firestore.

Codi d'exemple

exports.onDocumentCreate = functions.firestore
  .document('users/{userId}')
  .onCreate((snap, context) => {
    const newValue = snap.data();
    console.log('Nou document d\'usuari creat:', newValue);
  });

Funcions programades

Executa funcions segons un horari utilitzant la integració de Firebase amb Google Cloud Scheduler.

Codi d'exemple

exports.scheduledFunction = functions.pubsub.schedule('every 24 hours').onRun((context) => {
  console.log('Aquesta funció s\'executa cada 24 hores');
});

Conclusió

Les Funcions Cloud per Firebase proporcionen una manera poderosa d'extendre la teva aplicació de Firebase amb lògica de backend personalitzada. Són fàcils de configurar, escalables i segures. Tant si estàs responent a sol·licituds HTTP, canvis a la base de dades o esdeveniments programats, les Funcions Cloud poden ajudar-te a construir aplicacions robustes i dinàmiques.

En dominar les Funcions Cloud, pots millorar significativament les capacitats dels teus projectes de Firebase, fent-los més responsius i rics en funcionalitats.
