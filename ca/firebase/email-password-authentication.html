Introducció a l'Autenticació de Firebase

L'Autenticació de Firebase proporciona serveis de backend per ajudar a autenticar usuaris a la teva aplicació. Suporta diverses metodologies d'autenticació, incloent correu electrònic i contrasenya, inici de sessió amb xarxes socials, i més. En aquesta secció, ens centrarem en l'autenticació amb correu electrònic i contrasenya.

Configuració de Firebase al Teu Projecte

Abans d'endinsar-nos en l'autenticació, has de configurar Firebase al teu projecte.

  1. Crea un Projecte de Firebase:

    • Ves a la Consola de Firebase.
    • Fes clic a "Afegeix projecte" i segueix les instruccions de configuració.

  2. Afegeix Firebase a la Teva Aplicació:

    • Per a aplicacions web, afegeix el SDK de Firebase al teu projecte.
    • Per a Android i iOS, segueix les instruccions específiques proporcionades a la Consola de Firebase.

  3. Habilita l'Autenticació amb Correu Electrònic/Contrasenya:

    • A la Consola de Firebase, ves a la secció d'Autenticació.
    • Fes clic a la pestanya "Mètode d'inici de sessió".
    • Habilita l'autenticació amb "Correu Electrònic/Contrasenya".

Autenticació Bàsica amb Correu Electrònic i Contrasenya

Registrar un Nou Usuari

Per registrar un nou usuari amb un correu electrònic i contrasenya, utilitza el mètode createUserWithEmailAndPassword.

firebase.auth().createUserWithEmailAndPassword(email, password)
  .then((userCredential) => {
    // Sessió iniciada 
    var user = userCredential.user;
    console.log("Usuari registrat:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en registrar l'usuari:", errorCode, errorMessage);
  });
  • Explicació:
    • createUserWithEmailAndPassword(email, password): Aquest mètode crea un nou usuari amb el correu electrònic i contrasenya proporcionats.
    • userCredential: Conté informació sobre l'usuari acabat de crear.
    • La gestió d'errors és crucial per proporcionar retroalimentació a l'usuari.

Iniciar Sessió amb un Usuari Existent

Per iniciar sessió amb un usuari existent, utilitza el mètode signInWithEmailAndPassword.

firebase.auth().signInWithEmailAndPassword(email, password)
  .then((userCredential) => {
    // Sessió iniciada
    var user = userCredential.user;
    console.log("Usuari ha iniciat sessió:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en iniciar sessió:", errorCode, errorMessage);
  });
  • Explicació:
    • signInWithEmailAndPassword(email, password): Aquest mètode inicia sessió amb un usuari existent amb el correu electrònic i contrasenya proporcionats.
    • La gestió d'errors és similar al procés de registre.

Tancar Sessió d'un Usuari

Per tancar sessió de l'usuari actual, utilitza el mètode signOut.

firebase.auth().signOut().then(() => {
  console.log("Usuari ha tancat sessió");
}).catch((error) => {
  console.error("Error en tancar sessió:", error);
});
  • Explicació:
    • signOut(): Aquest mètode tanca sessió de l'usuari actualment connectat.

Autenticació Avançada amb Correu Electrònic i Contrasenya

Verificació de Correu Electrònic

Després que un usuari es registri, pots enviar un correu electrònic de verificació per confirmar la seva adreça de correu electrònic.

firebase.auth().currentUser.sendEmailVerification()
  .then(() => {
    console.log("Correu electrònic de verificació enviat");
  })
  .catch((error) => {
    console.error("Error en enviar el correu electrònic de verificació:", error);
  });
  • Explicació:
    • sendEmailVerification(): Envia un correu electrònic de verificació a l'usuari actualment connectat.

Restabliment de Contrasenya

Si un usuari oblida la seva contrasenya, pots enviar-li un correu electrònic per restablir-la.

firebase.auth().sendPasswordResetEmail(email)
  .then(() => {
    console.log("Correu electrònic de restabliment de contrasenya enviat");
  })
  .catch((error) => {
    console.error("Error en enviar el correu electrònic de restabliment de contrasenya:", error);
  });
  • Explicació:
    • sendPasswordResetEmail(email): Envia un correu electrònic de restabliment de contrasenya a l'adreça de correu electrònic especificada.

Actualització del Perfil de l'Usuari

Els usuaris poden actualitzar la seva informació de perfil, com ara el seu nom de visualització i la URL de la foto de perfil.

firebase.auth().currentUser.updateProfile({
  displayName: "Nou Nom",
  photoURL: "https://example.com/jane-q-user/profile.jpg"
}).then(() => {
  console.log("Perfil de l'usuari actualitzat");
}).catch((error) => {
  console.error("Error en actualitzar el perfil de l'usuari:", error);
});
  • Explicació:
    • updateProfile(): Actualitza la informació del perfil de l'usuari actualment connectat.

Millors Pràctiques de Seguretat

  • Utilitza HTTPS: Sempre utilitza HTTPS per protegir les dades dels usuaris durant la transmissió.
  • Valida les Adreces de Correu Electrònic: Assegura't que les adreces de correu electrònic siguin vàlides i estiguin correctament formatejades.
  • Contrasenyes Fortes: Encoratja els usuaris a crear contrasenyes fortes.
  • Gestió d'Errors: Proporciona missatges d'error clars als usuaris sense exposar informació sensible.

Conclusió

L'autenticació amb correu electrònic i contrasenya a Firebase és una manera potent i senzilla de gestionar l'autenticació d'usuaris a la teva aplicació. Seguint els passos descrits en aquest tema, pots implementar funcions d'autenticació bàsiques i avançades, assegurant una experiència segura i fàcil d'usar. Recorda seguir les millors pràctiques de seguretat per protegir les dades dels teus usuaris.
