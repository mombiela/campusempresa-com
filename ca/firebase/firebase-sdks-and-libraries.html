Firebase ofereix un conjunt de SDKs i biblioteques que permeten als desenvolupadors integrar els serveis de Firebase a les seves aplicacions de manera fluida. Aquest tema cobrirà els conceptes bàsics dels SDKs i biblioteques de Firebase, la seva instal·lació i ús en diferents plataformes.

Introducció als SDKs i biblioteques de Firebase

Els SDKs i biblioteques de Firebase proporcionen les eines necessàries per interactuar amb serveis de Firebase com Autenticació, Firestore, Realtime Database, Cloud Functions i més. Aquests SDKs estan disponibles per a diverses plataformes, incloent Android, iOS, Web i Unity.

Conceptes clau

  • SDK (Kit de Desenvolupament de Programari): Una col·lecció d'eines i biblioteques de programari que ajuden els desenvolupadors a crear aplicacions per a plataformes específiques.
  • Biblioteca: Una col·lecció de codi preescrit que els desenvolupadors poden utilitzar per optimitzar tasques.
  • Suport de Plataforma: Els SDKs de Firebase estan disponibles per a múltiples plataformes, assegurant la compatibilitat multiplataforma.

Configuració dels SDKs de Firebase

Android

  1. Afegeix Firebase al teu projecte Android:

    • Ves a la Consola de Firebase i crea un nou projecte.
    • Afegeix la teva aplicació Android al projecte registrant l'aplicació amb el seu nom de paquet.
    • Descarrega el fitxer google-services.json i col·loca'l al directori app del teu projecte Android.

  2. Afegeix les dependències del SDK de Firebase:

    // Afegeix el classpath dels serveis de Google al fitxer build.gradle a nivell de projecte
buildscript {
    dependencies {
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// Afegeix les dependències del SDK de Firebase al fitxer build.gradle a nivell d'aplicació
dependencies {
    implementation platform('com.google.firebase:firebase-bom:28.4.1')
    implementation 'com.google.firebase:firebase-analytics'
    // Afegeix altres SDKs de Firebase segons sigui necessari
}

// Aplica el plugin dels serveis de Google
apply plugin: 'com.google.gms.google-services'

iOS

  1. Afegeix Firebase al teu projecte iOS:

    • Ves a la Consola de Firebase i crea un nou projecte.
    • Afegeix la teva aplicació iOS al projecte registrant l'aplicació amb el seu ID de paquet.
    • Descarrega el fitxer GoogleService-Info.plist i afegeix-lo al teu projecte Xcode.

  2. Afegeix les dependències del SDK de Firebase utilitzant CocoaPods:

    # Afegeix el pod de Firebase al teu Podfile
pod 'Firebase/Analytics'
# Afegeix altres pods de Firebase segons sigui necessari

# Instal·la els pods
pod install

Web

  1. Afegeix Firebase al teu projecte web:

    • Ves a la Consola de Firebase i crea un nou projecte.
    • Afegeix la teva aplicació web al projecte registrant l'aplicació i obtenint l'objecte de configuració de Firebase.

  2. Inclou el SDK de Firebase al teu HTML:

    <!-- Afegeix els SDKs de Firebase -->
<script src="https://www.gstatic.com/firebasejs/9.1.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.1.0/firebase-analytics.js"></script>
<!-- Afegeix altres SDKs de Firebase segons sigui necessari -->

<script>
  // La configuració de Firebase de la teva aplicació web
  const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID",
    measurementId: "YOUR_MEASUREMENT_ID"
  };

  // Inicialitza Firebase
  const app = firebase.initializeApp(firebaseConfig);
  const analytics = firebase.analytics();
</script>

Ús dels SDKs de Firebase

Autenticació

L'Autenticació de Firebase proporciona serveis de backend per autenticar usuaris a la teva aplicació.

Exemple: Autenticació per correu electrònic i contrasenya (Android)

// Inicialitza Firebase Auth
FirebaseAuth mAuth = FirebaseAuth.getInstance();

// Inicia sessió amb correu electrònic i contrasenya
mAuth.signInWithEmailAndPassword(email, password)
    .addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
        @Override
        public void onComplete(@NonNull Task<AuthResult> task) {
            if (task.isSuccessful()) {
                // Inici de sessió exitós
                FirebaseUser user = mAuth.getCurrentUser();
                // Actualitza la UI amb la informació de l'usuari que ha iniciat sessió
            } else {
                // Si l'inici de sessió falla, mostra un missatge a l'usuari.
                Toast.makeText(MainActivity.this, "Autenticació fallida.", Toast.LENGTH_SHORT).show();
            }
        }
    });

Firestore

Firestore és una base de dades flexible i escalable per al desenvolupament mòbil, web i de servidor.

Exemple: Afegir dades a Firestore (Web)

// Inicialitza Firestore
const db = firebase.firestore();

// Afegeix un nou document amb un ID generat
db.collection("users").add({
    first: "Ada",
    last: "Lovelace",
    born: 1815
})
.then((docRef) => {
    console.log("Document escrit amb ID: ", docRef.id);
})
.catch((error) => {
    console.error("Error afegint el document: ", error);
});

Conclusió

Els SDKs i biblioteques de Firebase proporcionen un conjunt d'eines poderoses i fàcils d'utilitzar per integrar els serveis de Firebase a les teves aplicacions. Entenent com configurar i utilitzar aquests SDKs en diferents plataformes, pots aprofitar les capacitats de Firebase per millorar la funcionalitat i l'experiència d'usuari de la teva aplicació. Tant si estàs treballant en l'autenticació, la gestió de bases de dades o l'analítica, els SDKs de Firebase ofereixen la flexibilitat i escalabilitat necessàries per al desenvolupament d'aplicacions modernes.
