En aquesta secció, et guiarem a través de l'escriptura de la teva primera funció de Firebase. Les funcions de Firebase et permeten executar codi de backend en resposta a esdeveniments desencadenats per les característiques de Firebase i sol·licituds HTTPS. Aquesta és una habilitat crucial per construir aplicacions escalables i eficients.

Conceptes clau

  • Funcions de Firebase: Funcions sense servidor que s'executen en un entorn gestionat.
  • Basat en esdeveniments: Les funcions poden ser desencadenades per diversos esdeveniments de Firebase, com canvis a la base de dades, esdeveniments d'autenticació o sol·licituds HTTP.
  • Node.js: Les funcions de Firebase estan escrites en JavaScript o TypeScript i s'executen en Node.js.

Configuració del teu entorn

Abans d'escriure la teva primera funció, assegura't de tenir el següent configurat:

  1. Node.js i npm: Instal·la Node.js i npm des de nodejs.org.
  2. Firebase CLI: Instal·la el Firebase CLI executant: 
    npm install -g firebase-tools
  3. Projecte de Firebase: Crea un projecte de Firebase a la Consola de Firebase.

Inicialització de les funcions de Firebase

  1. Inicia sessió a Firebase: 
    firebase login
  2. Inicialitza Firebase al directori del teu projecte: 
    firebase init functions
    • Selecciona el teu projecte de Firebase.
    • Tria JavaScript o TypeScript.
    • Instal·la les dependències quan se't demani.

Escriure la teva primera funció

Escrivim una funció HTTP simple que retorna un missatge de "Hello, World!".

  1. Navega al directori functions:

    cd functions

  2. Obre index.js (o index.ts si vas triar TypeScript) i afegeix el següent codi:

    const functions = require('firebase-functions');

// Crea i desplega la teva primera funció
exports.helloWorld = functions.https.onRequest((request, response) => {
  response.send("Hello, World!");
});

Explicació

  • Importar les funcions de Firebase:

    const functions = require('firebase-functions');

    Aquesta línia importa el SDK de les funcions de Firebase.

  • Definir la funció:

    exports.helloWorld = functions.https.onRequest((request, response) => {
  response.send("Hello, World!");
});
    • exports.helloWorld: Això exporta la funció perquè Firebase la pugui utilitzar.
    • functions.https.onRequest: Això especifica que la funció serà desencadenada per una sol·licitud HTTP.
    • (request, response) => { ... }: Aquest és el cos de la funció, on request i response són els objectes de sol·licitud i resposta HTTP.

Desplegar la teva funció

  1. Desplega la funció: 
    firebase deploy --only functions
  2. Comprova l'URL: Després del desplegament, el Firebase CLI proporcionarà un URL per a la teva funció. Obre aquest URL al teu navegador per veure "Hello, World!".

Resum

En aquesta secció, has après a:

  • Configurar el teu entorn per a les funcions de Firebase.
  • Inicialitzar les funcions de Firebase al teu projecte.
  • Escriure una funció HTTP simple que retorna un missatge de "Hello, World!".
  • Desplegar la teva funció a Firebase.

Dominant aquests conceptes bàsics, ara estàs preparat per explorar característiques i casos d'ús més avançats de les funcions de Firebase. Continua experimentant i construint funcions més complexes per millorar les capacitats de la teva aplicació.
