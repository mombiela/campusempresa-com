Firebase Cloud Messaging (FCM) és una solució de missatgeria multiplataforma que et permet enviar missatges de manera fiable i sense cost. Aquest tema et guiarà a través dels conceptes bàsics de FCM, des de la configuració del teu projecte fins a l'enviament del teu primer missatge, i també cobrirà funcions avançades per a desenvolupadors més experimentats.

Què és Firebase Cloud Messaging?

Firebase Cloud Messaging (FCM) és un servei que et permet enviar notificacions i missatges als usuaris a través de plataformes com Android, iOS i aplicacions web. És una eina poderosa per involucrar els usuaris i mantenir-los informats sobre actualitzacions, promocions i altra informació important.

Característiques Clau de FCM

  • Suport Multiplataforma: Envia missatges a aplicacions Android, iOS i web.
  • Servei Gratuït: Sense cost per enviar missatges.
  • Missatgeria Dirigida: Envia missatges a dispositius específics, segments d'usuaris o temes.
  • Integració d'Analítiques: Fes un seguiment de l'efectivitat dels teus missatges amb Firebase Analytics.

Configuració de Firebase Cloud Messaging

Pas 1: Crear un Projecte de Firebase

  1. Ves al Firebase Console.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

Pas 2: Afegeix Firebase a la Teva Aplicació

  • Per Android:

    1. Afegeix el SDK de Firebase al teu projecte incloent les dependències necessàries en els teus arxius build.gradle.
    2. Registra la teva aplicació amb Firebase i descarrega l'arxiu google-services.json.
    3. Col·loca l'arxiu google-services.json al directori app del teu projecte.

  • Per iOS:

    1. Afegeix el SDK de Firebase al teu projecte utilitzant CocoaPods.
    2. Registra la teva aplicació amb Firebase i descarrega l'arxiu GoogleService-Info.plist.
    3. Afegeix l'arxiu GoogleService-Info.plist al teu projecte Xcode.

Pas 3: Configura FCM a la Teva Aplicació

  • Per Android:

    // En el teu MainActivity.java
import com.google.firebase.messaging.FirebaseMessaging;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Subscriu-te a un tema
        FirebaseMessaging.getInstance().subscribeToTopic("news")
            .addOnCompleteListener(new OnCompleteListener<Void>() {
                @Override
                public void onComplete(@NonNull Task<Void> task) {
                    String msg = "Subscriure's al tema de notícies";
                    if (!task.isSuccessful()) {
                        msg = "La subscripció ha fallat";
                    }
                    Log.d(TAG, msg);
                    Toast.makeText(MainActivity.this, msg, Toast.LENGTH_SHORT).show();
                }
            });
    }
}

  • Per iOS:

    // En el teu AppDelegate.swift
import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()

        // Subscriu-te a un tema
        Messaging.messaging().subscribe(toTopic: "news") { error in
            print("Subscriure's al tema de notícies")
        }

        return true
    }
}

Enviant el Teu Primer Missatge

Utilitzant el Firebase Console

  1. Ves al Firebase Console.
  2. Selecciona el teu projecte.
  3. Navega a la secció "Cloud Messaging".
  4. Fes clic a "Envia el teu primer missatge".
  5. Omple els detalls del missatge i selecciona el públic objectiu.
  6. Fes clic a "Envia".

Utilitzant l'API de FCM

  • Protocol HTTP:

    curl -X POST -H "Authorization: key=YOUR_SERVER_KEY" -H "Content-Type: application/json" -d '{
  "to": "/topics/news",
  "notification": {
    "title": "Notícies d'última hora",
    "body": "Nou article disponible!"
  }
}' https://fcm.googleapis.com/fcm/send

  • Exemple en Node.js:

    const admin = require('firebase-admin');
const serviceAccount = require('path/to/serviceAccountKey.json');

admin.initializeApp({
  credential: admin.credential.cert(serviceAccount)
});

const message = {
  notification: {
    title: 'Notícies d'última hora',
    body: 'Nou article disponible!'
  },
  topic: 'news'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Missatge enviat amb èxit:', response);
  })
  .catch((error) => {
    console.log('Error en enviar el missatge:', error);
  });

Funcions Avançades

Tipus de Missatges

  • Missatges de Notificació: Mostrats automàticament pel SDK de FCM.
  • Missatges de Dades: Gestionats per l'aplicació client.

Opcions de Direccionament

  • Dispositiu Únic: Utilitza el token de registre del dispositiu.
  • Grup de Dispositius: Utilitza una clau de notificació.
  • Temes: Subscriu els usuaris a temes.
  • Condició: Utilitza expressions lògiques per dirigir múltiples temes.

Gestionant Missatges a l'Aplicació

  • Missatges en Primer Pla: Gestiona els missatges quan l'aplicació està en primer pla.
  • Missatges en Segon Pla: Gestiona els missatges quan l'aplicació està en segon pla o terminada.

Conclusió

Firebase Cloud Messaging és una eina versàtil i poderosa per enviar notificacions i missatges als usuaris a través de diferents plataformes. Seguint els passos descrits en aquest tema, pots configurar FCM a la teva aplicació, enviar el teu primer missatge i explorar funcions avançades per millorar la participació dels usuaris. Tant si ets un principiant com un desenvolupador avançat, FCM proporciona les eines que necessites per comunicar-te eficaçment amb els teus usuaris.
