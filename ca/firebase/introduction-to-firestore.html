Introducció a Firestore
Configurar Firestore ►

Firestore és una base de dades flexible i escalable per al desenvolupament mòbil, web i de servidor de Firebase i Google Cloud Platform. És una base de dades de documents NoSQL que et permet emmagatzemar, sincronitzar i consultar dades fàcilment per a les teves aplicacions.

Conceptes Clau

  • Base de Dades NoSQL: Firestore és una base de dades NoSQL, la qual cosa significa que emmagatzema dades en un format no tabular.
  • Documents i Col·leccions: Les dades s'emmagatzemen en documents, que estan organitzats en col·leccions.
  • Actualitzacions en Temps Real: Firestore pot sincronitzar dades en temps real entre tots els clients.
  • Suport Fora de Línia: Firestore suporta la persistència de dades fora de línia, permetent que les aplicacions funcionin fins i tot quan no hi ha connexió de xarxa.
  • Regles de Seguretat: Firestore proporciona un llenguatge de regles potent per controlar l'accés a les teves dades.

Configuració de Firestore

  1. Crea un Projecte de Firebase: Ves a la Consola de Firebase i crea un nou projecte.
  2. Afegeix Firestore al Teu Projecte: Navega a la secció de Firestore a la Consola de Firebase i habilita Firestore.
  3. Instal·la el SDK de Firebase: Afegeix el SDK de Firebase al teu projecte.

Exemple: Afegir el SDK de Firebase a un Projecte Web

<!-- Afegeix els SDKs de Firebase -->
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js"></script>

<script>
  // La configuració de Firebase de la teva aplicació web
  var firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT_ID.appspot.com",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
  };
  // Inicialitza Firebase
  firebase.initializeApp(firebaseConfig);
  var db = firebase.firestore();
</script>

Treballant amb Firestore

Afegint Dades

Per afegir dades a Firestore, crees un document dins d'una col·lecció.

// Afegeix un nou document amb un ID generat
db.collection("users").add({
    first: "Ada",
    last: "Lovelace",
    born: 1815
})
.then((docRef) => {
    console.log("Document escrit amb ID: ", docRef.id);
})
.catch((error) => {
    console.error("Error afegint el document: ", error);
});

Llegint Dades

Pots llegir dades de Firestore consultant documents.

// Obteniu una llista d'usuaris
db.collection("users").get().then((querySnapshot) => {
    querySnapshot.forEach((doc) => {
        console.log(`${doc.id} => ${doc.data()}`);
    });
});

Actualitzant Dades

Per actualitzar un document, pots utilitzar el mètode update.

var userRef = db.collection("users").doc("USER_ID");

// Estableix el camp "capital" de la ciutat 'DC'
return userRef.update({
    capital: true
})
.then(() => {
    console.log("Document actualitzat amb èxit!");
})
.catch((error) => {
    console.error("Error actualitzant el document: ", error);
});

Eliminant Dades

Per eliminar un document, utilitza el mètode delete.

db.collection("users").doc("USER_ID").delete().then(() => {
    console.log("Document eliminat amb èxit!");
}).catch((error) => {
    console.error("Error eliminant el document: ", error);
});

Actualitzacions en Temps Real

Firestore et permet escoltar actualitzacions en temps real.

db.collection("users").onSnapshot((querySnapshot) => {
    var users = [];
    querySnapshot.forEach((doc) => {
        users.push(doc.data());
    });
    console.log("Usuaris actuals a la col·lecció: ", users.join(", "));
});

Regles de Seguretat

Les regles de seguretat de Firestore et permeten controlar l'accés a les teves dades.

service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Conclusió

Firestore és una potent base de dades NoSQL que ofereix sincronització de dades en temps real, suport fora de línia i funcions de seguretat robustes. Comprenent els conceptes bàsics de la configuració de Firestore, afegint, llegint, actualitzant i eliminant dades, així com implementant actualitzacions en temps real i regles de seguretat, pots gestionar eficaçment les necessitats de dades de la teva aplicació.
