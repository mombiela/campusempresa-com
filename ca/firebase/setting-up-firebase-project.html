Introducció

Firebase és una plataforma desenvolupada per Google per crear aplicacions mòbils i web. Proporciona una varietat d'eines i serveis per ajudar-te a desenvolupar aplicacions d'alta qualitat, fer créixer la teva base d'usuaris i guanyar més diners. Aquesta secció et guiarà a través del procés de configuració d'un projecte Firebase des de zero.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un compte de Google
  • Coneixements bàsics de desenvolupament web (HTML, CSS, JavaScript)
  • Node.js i npm instal·lats a la teva màquina (per a configuracions avançades)

Creació d'un Projecte Firebase

Pas 1: Inicia sessió a la Consola de Firebase

  1. Ves a la Consola de Firebase.
  2. Inicia sessió amb el teu compte de Google.

Pas 2: Afegeix un Nou Projecte

  1. Fes clic al botó "Afegeix projecte".
  2. Introdueix un nom de projecte. Aquest nom s'utilitzarà per identificar el teu projecte a la consola de Firebase.
  3. (Opcional) Pots habilitar Google Analytics per al teu projecte. Això és útil per fer un seguiment de les interaccions dels usuaris i el rendiment de l'aplicació.
  4. Fes clic a "Crea projecte" i espera que Firebase configuri el teu projecte.

Pas 3: Configura el Teu Projecte

  1. Un cop creat el teu projecte, seràs redirigit a la pàgina de visió general del projecte.
  2. Aquí, pots afegir serveis de Firebase al teu projecte, com ara Autenticació, Firestore, Base de Dades en Temps Real, etc.

Afegir Firebase a la Teva Aplicació Web

Pas 1: Registra la Teva Aplicació

  1. A la consola de Firebase, fes clic a la icona web (</>) per afegir Firebase a la teva aplicació web.
  2. Introdueix un sobrenom per a l'aplicació (opcional) i fes clic a "Registra aplicació".

Pas 2: Afegeix el SDK de Firebase

  1. Firebase et proporcionarà un objecte de configuració. Sembla una cosa així:

    // La configuració de Firebase de la teva aplicació web
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

  2. Copia aquest objecte de configuració.

Pas 3: Inicialitza Firebase a la Teva Aplicació

  1. Crea un fitxer index.html i inclou els SDKs de Firebase:

    <!DOCTYPE html>
<html>
<head>
  <title>Configuració de Firebase</title>
  <!-- Afegeix els SDKs de Firebase -->
  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-analytics.js"></script>
</head>
<body>
  <h1>Configuració de Firebase Completa</h1>
  <script>
    // La configuració de Firebase de la teva aplicació web
    const firebaseConfig = {
      apiKey: "YOUR_API_KEY",
      authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
      projectId: "YOUR_PROJECT_ID",
      storageBucket: "YOUR_PROJECT_ID.appspot.com",
      messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
      appId: "YOUR_APP_ID"
    };

    // Inicialitza Firebase
    firebase.initializeApp(firebaseConfig);
    firebase.analytics();
  </script>
</body>
</html>

  2. Substitueix els valors de marcador de posició a l'objecte firebaseConfig pels valors reals del teu projecte Firebase.

Afegir Firebase a la Teva Aplicació Mòbil

Per Android

  1. A la consola de Firebase, fes clic a la icona d'Android per afegir Firebase a la teva aplicació Android.
  2. Segueix les instruccions en pantalla per descarregar el fitxer google-services.json i col·loca'l al directori app del teu projecte Android.
  3. Afegeix el SDK de Firebase als teus fitxers build.gradle segons les instruccions.

Per iOS

  1. A la consola de Firebase, fes clic a la icona d'iOS per afegir Firebase a la teva aplicació iOS.
  2. Segueix les instruccions en pantalla per descarregar el fitxer GoogleService-Info.plist i afegeix-lo al teu projecte Xcode.
  3. Afegeix el SDK de Firebase al teu Podfile i executa pod install.

Conclusió

Configurar un projecte Firebase és un procés senzill que implica crear un projecte a la consola de Firebase, registrar la teva aplicació i afegir el SDK de Firebase al teu projecte. Tant si estàs treballant en una aplicació web com mòbil, Firebase proporciona una plataforma unificada per gestionar els serveis de backend de la teva aplicació, facilitant la concentració en la creació de grans experiències d'usuari.

Seguint els passos descrits en aquesta secció, ara hauries de tenir un projecte Firebase configurat i llest per utilitzar. A les següents seccions, explorarem com integrar diversos serveis de Firebase a la teva aplicació.
