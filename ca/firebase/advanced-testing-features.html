Introducció

En aquesta secció, aprofundirem en les funcions avançades de proves disponibles a Firebase. Les proves són una part crucial del procés de desenvolupament, assegurant que la teva aplicació es comporti com s'espera. Firebase proporciona diverses eines i funcions per facilitar proves completes.

Conceptes Clau

  • Firebase Test Lab: Una infraestructura de proves d'aplicacions basada en el núvol.
  • Firebase Emulator Suite: Entorn de proves local per als serveis de Firebase.
  • Proves Automatitzades: Utilitzar pipelines de CI/CD per automatitzar les proves.
  • Monitorització del Rendiment: Seguiment del rendiment de la teva aplicació durant les proves.

Firebase Test Lab

Firebase Test Lab et permet provar la teva aplicació en una àmplia gamma de dispositius i configuracions. Això ajuda a assegurar que la teva aplicació funcioni correctament en diferents entorns.

Funcions

  • Selecció de Dispositius: Tria entre una varietat de dispositius físics i virtuals.
  • Tipus de Proves: Suporta proves d'instrumentació, proves Robo i proves de bucle de joc.
  • Integració: S'integra fàcilment amb pipelines de CI/CD.

Exemple

gcloud firebase test android run \
  --type instrumentation \
  --app app-debug.apk \
  --test app-debug-androidTest.apk \
  --device model=Nexus6,version=23,locale=en,orientation=portrait

Aquesta comanda executa una prova d'instrumentació en un dispositiu Nexus 6 amb la versió 23 d'Android.

Firebase Emulator Suite

El Firebase Emulator Suite et permet executar i provar les teves aplicacions Firebase localment. Això és particularment útil per al desenvolupament i la depuració.

Funcions

  • Entorn Local: Emula serveis de Firebase com Firestore, Realtime Database i Autenticació.
  • Integració: Funciona perfectament amb el teu entorn de desenvolupament local.
  • Depuració: Proporciona registres detallats i informació de depuració.

Exemple

firebase emulators:start

Aquesta comanda inicia els emuladors de Firebase per a tots els serveis configurats.

Proves Automatitzades

Les proves automatitzades són essencials per mantenir la qualitat del codi i assegurar que els nous canvis no introdueixin errors.

Funcions

  • Integració CI/CD: S'integra amb eines populars de CI/CD com Jenkins, CircleCI i GitHub Actions.
  • Proves Scriptades: Escriu scripts per automatitzar el teu procés de proves.
  • Desplegament Continu: Desplega automàticament el codi provat a producció.

Exemple

name: Firebase CI

on: [push]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v2
    - name: Set up JDK 1.8
      uses: actions/setup-java@v1
      with:
        java-version: 1.8
    - name: Build with Gradle
      run: ./gradlew build
    - name: Run Tests
      run: ./gradlew test
    - name: Deploy to Firebase
      run: firebase deploy --token $FIREBASE_TOKEN

Aquest workflow de GitHub Actions construeix el projecte, executa proves i desplega a Firebase.

Monitorització del Rendiment

La Monitorització del Rendiment de Firebase t'ajuda a entendre les característiques de rendiment de la teva aplicació durant les proves.

Funcions

  • Rastreig del Rendiment: Mesura el temps que triguen a executar-se parts específiques de la teva aplicació.
  • Monitorització de Xarxa: Seguiment de les sol·licituds de xarxa i el seu rendiment.
  • Mètriques Personalitzades: Defineix mètriques personalitzades per monitoritzar aspectes específics de la teva aplicació.

Exemple

import { getPerformance } from "firebase/performance";

const perf = getPerformance();
const trace = perf.trace('test_trace');
trace.start();

// Codi que vols rastrejar
trace.stop();

Aquest fragment de codi inicia i atura un rastreig de rendiment al voltant d'un bloc de codi.

Conclusió

Les funcions avançades de proves a Firebase, com Firebase Test Lab, Emulator Suite, Proves Automatitzades i Monitorització del Rendiment, proporcionen eines potents per assegurar que la teva aplicació sigui robusta i rendible. Aprofitant aquestes eines, pots detectar problemes aviat, millorar la qualitat del codi i oferir una millor experiència d'usuari.
