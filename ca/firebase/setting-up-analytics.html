Firebase Analytics és una eina potent que t'ajuda a entendre com els usuaris interactuen amb la teva aplicació. Aquesta secció et guiarà a través de la configuració de Firebase Analytics, des dels conceptes bàsics fins a configuracions més avançades.

Introducció a Firebase Analytics

Firebase Analytics proporciona informació sobre el comportament i el compromís dels usuaris. Captura automàticament una àmplia gamma d'esdeveniments i propietats d'usuari, i pots registrar esdeveniments personalitzats per mesurar accions específiques.

Conceptes Clau

  • Esdeveniments: Accions que els usuaris realitzen a la teva aplicació, com ara visualitzacions de pantalla, clics de botons o compres.
  • Propietats d'Usuari: Atributs que defineixes per descriure segments de la teva base d'usuaris, com ara la preferència d'idioma o la ubicació geogràfica.
  • Quadres de Comandament: Representacions visuals de les teves dades, disponibles a la consola de Firebase.

Configuració de Firebase Analytics

Pas 1: Crear un Projecte de Firebase

  1. Ves a la Consola de Firebase.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

Pas 2: Afegeix Firebase a la Teva Aplicació

Per a Android

  1. A la consola de Firebase, selecciona el teu projecte.
  2. Fes clic a la icona d'Android per afegir una aplicació d'Android.
  3. Registra la teva aplicació amb el nom del paquet i segueix les instruccions per descarregar el fitxer google-services.json.
  4. Col·loca el fitxer google-services.json al directori app del teu projecte d'Android.
  5. Afegeix el SDK de Firebase als teus fitxers build.gradle: 
    // build.gradle a nivell de projecte
buildscript {
    dependencies {
        // Afegeix aquesta línia
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// build.gradle a nivell d'aplicació
apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services'

dependencies {
    // Afegeix aquesta línia
    implementation 'com.google.firebase:firebase-analytics:19.0.0'
}

Per a iOS

  1. A la consola de Firebase, selecciona el teu projecte.
  2. Fes clic a la icona d'iOS per afegir una aplicació d'iOS.
  3. Registra la teva aplicació amb l'ID del paquet i segueix les instruccions per descarregar el fitxer GoogleService-Info.plist.
  4. Afegeix el fitxer GoogleService-Info.plist al teu projecte d'Xcode.
  5. Afegeix el SDK de Firebase al teu Podfile: 
    # Descomenta la següent línia per definir una plataforma global per al teu projecte
# platform :ios, '9.0'

target 'YourApp' do
  # Afegeix aquesta línia
  pod 'Firebase/Analytics'
end
  6. Executa pod install per instal·lar el SDK.

Pas 3: Inicialitza Firebase a la Teva Aplicació

Per a Android

Al teu fitxer MainActivity.java o MainActivity.kt, inicialitza Firebase:

import com.google.firebase.analytics.FirebaseAnalytics;

public class MainActivity extends AppCompatActivity {
    private FirebaseAnalytics mFirebaseAnalytics;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Inicialitza Firebase Analytics
        mFirebaseAnalytics = FirebaseAnalytics.getInstance(this);
    }
}

Per a iOS

Al teu fitxer AppDelegate.swift, inicialitza Firebase:

import UIKit
import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicialitza Firebase
        FirebaseApp.configure()
        return true
    }
}

Registre d'Esdeveniments

Firebase Analytics registra automàticament alguns esdeveniments, però pots registrar esdeveniments personalitzats per fer un seguiment d'accions específiques.

Exemple: Registre d'un Esdeveniment Personalitzat

Per a Android

Bundle bundle = new Bundle();
bundle.putString(FirebaseAnalytics.Param.ITEM_ID, "id123");
bundle.putString(FirebaseAnalytics.Param.ITEM_NAME, "example_item");
bundle.putString(FirebaseAnalytics.Param.CONTENT_TYPE, "text");
mFirebaseAnalytics.logEvent(FirebaseAnalytics.Event.SELECT_CONTENT, bundle);

Per a iOS

Analytics.logEvent(AnalyticsEventSelectContent, parameters: [
    AnalyticsParameterItemID: "id123" as NSObject,
    AnalyticsParameterItemName: "example_item" as NSObject,
    AnalyticsParameterContentType: "text" as NSObject
])

Visualització de Dades d'Analytics

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció "Analytics" per veure quadres de comandament i informes.

Configuracions Avançades

Propietats d'Usuari

Defineix propietats d'usuari per segmentar la teva audiència:

Per a Android

mFirebaseAnalytics.setUserProperty("favorite_food", "pizza");

Per a iOS

Analytics.setUserProperty("pizza", forName: "favorite_food")

Depuració d'Analytics

Utilitza el DebugView a la Consola de Firebase per veure esdeveniments en temps real:

  1. Per a Android, activa el mode de depuració executant: 
    adb shell setprop debug.firebase.analytics.app <your_package_name>
  2. Per a iOS, activa el mode de depuració establint el següent argument a Xcode: 
    -FIRDebugEnabled

Conclusió

Configurar Firebase Analytics implica crear un projecte de Firebase, afegir Firebase a la teva aplicació, inicialitzar Firebase i registrar esdeveniments. Aprofitant Firebase Analytics, pots obtenir informació valuosa sobre el comportament dels usuaris i prendre decisions basades en dades per millorar la teva aplicació.
