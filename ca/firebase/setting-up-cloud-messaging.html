Firebase Cloud Messaging (FCM) és una solució de missatgeria multiplataforma que et permet enviar missatges de manera fiable i sense cost. Aquest tema et guiarà a través de la configuració de FCM des d'un nivell principiant fins a un nivell avançat.

Introducció a Firebase Cloud Messaging

  • Què és FCM?
    • Un servei per enviar notificacions i missatges als usuaris a través de diferents plataformes.
    • Suporta tant missatges directes com missatges basats en temes.
  • Per què utilitzar FCM?
    • Gratuït i escalable.
    • Fàcil integració amb els serveis de Firebase.
    • Suporta diversos tipus de missatges (notificació, dades, etc.).

Prerequisits

  • Coneixement bàsic de Firebase.
  • Projecte de Firebase configurat a la Consola de Firebase.
  • Coneixement bàsic del desenvolupament d'Android/iOS.

Configuració de FCM per a Android

Pas 1: Afegeix Firebase al Teu Projecte d'Android

  1. Crea un projecte de Firebase a la Consola de Firebase.
  2. Registra la teva aplicació amb Firebase:
    • Afegeix el nom del paquet de la teva aplicació d'Android.
    • Descarrega el fitxer google-services.json.
  3. Afegeix el fitxer google-services.json al teu projecte:
    • Col·loca'l al directori app.

Pas 2: Afegeix el SDK de Firebase al Teu Projecte

  1. Obre el teu fitxer build.gradle (a nivell de projecte) i afegeix el següent: 
    buildscript {
    dependencies {
        // Afegeix aquesta línia
        classpath 'com.google.gms:google-services:4.3.10'
    }
}
  2. Obre el teu fitxer build.gradle (a nivell d'aplicació) i afegeix el següent: 
    dependencies {
    // Afegeix el SDK de Firebase per a FCM
    implementation 'com.google.firebase:firebase-messaging:23.0.0'
}

// Afegeix aquesta línia al final del fitxer
apply plugin: 'com.google.gms.google-services'

Pas 3: Configura FCM a la Teva Aplicació

  1. Inicialitza Firebase al teu MainActivity.java: 
    import com.google.firebase.messaging.FirebaseMessaging;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        // Inicialitza Firebase
        FirebaseMessaging.getInstance().subscribeToTopic("news")
            .addOnCompleteListener(task -> {
                String msg = "Subscribed to news topic";
                if (!task.isSuccessful()) {
                    msg = "Subscription failed";
                }
                Log.d("FCM", msg);
            });
    }
}

Configuració de FCM per a iOS

Pas 1: Afegeix Firebase al Teu Projecte d'iOS

  1. Crea un projecte de Firebase a la Consola de Firebase.
  2. Registra la teva aplicació amb Firebase:
    • Afegeix l'ID del paquet de la teva aplicació d'iOS.
    • Descarrega el fitxer GoogleService-Info.plist.
  3. Afegeix el fitxer GoogleService-Info.plist al teu projecte:
    • Arrossega i deixa'l anar al teu projecte d'Xcode.

Pas 2: Afegeix el SDK de Firebase al Teu Projecte

  1. Obre el teu Podfile i afegeix el següent: 
    pod 'Firebase/Messaging'
  2. Executa pod install per instal·lar el SDK de Firebase.

Pas 3: Configura FCM a la Teva Aplicació

  1. Inicialitza Firebase al teu AppDelegate.swift: 
    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

    var window: UIWindow?

    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicialitza Firebase
        FirebaseApp.configure()

        // Subscriu-te a un tema
        Messaging.messaging().subscribe(toTopic: "news") { error in
            print("Subscribed to news topic")
        }

        return true
    }
}

Enviament de Missatges amb FCM

Enviament d'un Missatge de Notificació

  • Via Consola de Firebase:
    • Navega a la Consola de Firebase.
    • Selecciona el teu projecte i ves a la secció de Cloud Messaging.
    • Fes clic a "Send your first message".
    • Omple els detalls del missatge i l'objectiu (per exemple, tema "news").
    • Fes clic a "Send".

Enviament d'un Missatge de Dades

  • Via Firebase Admin SDK: 
    const admin = require('firebase-admin');
admin.initializeApp();

const message = {
  data: {
    score: '850',
    time: '2:45'
  },
  topic: 'news'
};

admin.messaging().send(message)
  .then((response) => {
    console.log('Successfully sent message:', response);
  })
  .catch((error) => {
    console.log('Error sending message:', error);
  });

Temes Avançats

Gestió de les Càrregues de Missatges

  • Missatges de Notificació: Mostrats automàticament pel sistema.
  • Missatges de Dades: Gestionats per l'aplicació en segon pla o en primer pla.

Personalització de les Notificacions

  • So Personalitzat: Especifica un fitxer de so personalitzat en la càrrega de la notificació.
  • Accions Personalitzades: Afegeix botons d'acció a les notificacions.

Gestió d'Errors

  • Errors Comuns:
    • Token de registre invàlid.
    • Taxa de missatges excedida.
  • Consells de Depuració:
    • Revisa els registres de la Consola de Firebase.
    • Utilitza el Firebase DebugView.

Conclusió

Configurar Firebase Cloud Messaging implica integrar el SDK de Firebase al teu projecte d'Android o iOS, configurar la teva aplicació per rebre missatges i enviar missatges via la Consola de Firebase o l'Admin SDK. Amb FCM, pots enviar tant missatges de notificació com de dades, personalitzar notificacions i gestionar errors de manera efectiva. Aquesta configuració proporciona una base sòlida per implementar missatgeria en temps real a les teves aplicacions.
