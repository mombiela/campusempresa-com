Introducció

Firebase és una plataforma potent per construir aplicacions mòbils i web, però el seu veritable potencial es desbloqueja quan s'integra amb altres serveis. Aquesta secció et guiarà a través del procés d'integració de Firebase amb diversos serveis de tercers, millorant la funcionalitat i l'experiència d'usuari de la teva aplicació.

Integració de Firebase amb Google Cloud Platform (GCP)

Google Cloud Platform (GCP) ofereix una suite de serveis de computació en núvol que es poden integrar perfectament amb Firebase.

Conceptes Clau

  • Google Cloud Functions: Amplia Firebase amb funcions sense servidor.
  • BigQuery: Analitza grans conjunts de dades emmagatzemats a Firebase.
  • Cloud Storage: Emmagatzema i serveix contingut generat per l'usuari.

Exemple: Utilitzant Cloud Functions amb Firebase

Cloud Functions et permeten executar codi backend en resposta a esdeveniments desencadenats per funcions de Firebase.

const functions = require('firebase-functions');
const admin = require('firebase-admin');
admin.initializeApp();

exports.addWelcomeMessage = functions.auth.user().onCreate((user) => {
  const email = user.email;
  const displayName = user.displayName;

  return admin.firestore().collection('messages').add({
    message: `Welcome ${displayName || email}!`,
    timestamp: admin.firestore.FieldValue.serverTimestamp()
  });
});
  • Explicació:
    • functions.auth.user().onCreate: Desencadenat quan es crea un nou usuari.
    • admin.firestore().collection('messages').add: Afegeix un missatge de benvinguda a la base de dades Firestore.

Integració de Firebase amb Proveïdors d'Autenticació de Tercers

Firebase Authentication admet la integració amb diversos proveïdors de tercers com Google, Facebook i GitHub.

Conceptes Clau

  • OAuth: Estàndard obert per a la delegació d'accés.
  • Firebase Authentication SDK: Simplifica el procés d'integració de l'autenticació de tercers.

Exemple: Google Sign-In

Integra Google Sign-In amb Firebase Authentication.

import firebase from 'firebase/app';
import 'firebase/auth';

const provider = new firebase.auth.GoogleAuthProvider();

firebase.auth().signInWithPopup(provider).then((result) => {
  const user = result.user;
  console.log(`User signed in: ${user.displayName}`);
}).catch((error) => {
  console.error(`Error during sign-in: ${error.message}`);
});
  • Explicació:
    • GoogleAuthProvider(): Crea una nova instància del proveïdor d'autenticació de Google.
    • signInWithPopup(provider): Obre una finestra emergent per a l'inici de sessió amb Google.

Integració de Firebase amb Eines d'Anàlisi

Firebase Analytics es pot integrar amb altres eines d'anàlisi com Google Analytics i Mixpanel per obtenir informació millorada de les dades.

Conceptes Clau

  • Seguiment d'Esdeveniments: Monitoritza les interaccions dels usuaris dins de la teva aplicació.
  • Propietats d'Usuari: Atributs que defineixes per descriure segments de la teva base d'usuaris.

Exemple: Registrar Esdeveniments Personalitzats

Registra esdeveniments personalitzats a Firebase Analytics.

firebase.analytics().logEvent('purchase', {
  item_id: 'P12345',
  item_name: 'Widget',
  value: 9.99
});
  • Explicació:
    • logEvent('purchase', {...}): Registra un esdeveniment personalitzat anomenat 'purchase' amb paràmetres addicionals.

Integració de Firebase amb Passarel·les de Pagament

Integra Firebase amb passarel·les de pagament com Stripe o PayPal per gestionar transaccions dins de la teva aplicació.

Conceptes Clau

  • Passarel·la de Pagament: Servei que processa pagaments amb targeta de crèdit.
  • Integració del Costat del Servidor: Gestiona de manera segura el processament de pagaments al servidor.

Exemple: Utilitzant Stripe amb Firebase Cloud Functions

Processa pagaments utilitzant Stripe i Firebase Cloud Functions.

const functions = require('firebase-functions');
const stripe = require('stripe')('your-stripe-secret-key');

exports.createStripeCharge = functions.https.onRequest(async (req, res) => {
  const { amount, currency, source } = req.body;

  try {
    const charge = await stripe.charges.create({
      amount,
      currency,
      source,
    });
    res.status(200).send(charge);
  } catch (error) {
    res.status(500).send(error);
  }
});
  • Explicació:
    • stripe.charges.create({...}): Crea un nou càrrec utilitzant Stripe.
    • functions.https.onRequest: Desencadenador HTTP per a la Cloud Function.

Conclusió

Integrar Firebase amb altres serveis pot millorar significativament les capacitats de la teva aplicació. Ja sigui aprofitant Google Cloud Platform per al processament backend, afegint autenticació de tercers, millorant l'anàlisi o processant pagaments, Firebase proporciona les eines i la flexibilitat necessàries per construir aplicacions robustes i escalables. Comprenent i utilitzant aquestes integracions, pots crear aplicacions més dinàmiques i amb més funcionalitats.
