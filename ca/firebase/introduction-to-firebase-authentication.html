L'Autenticació de Firebase és una eina poderosa que permet als desenvolupadors integrar fàcilment l'autenticació d'usuaris a les seves aplicacions. Aquesta secció cobrirà els conceptes bàsics de l'Autenticació de Firebase, incloent-hi com configurar-la, els diferents mètodes d'autenticació disponibles i com gestionar els usuaris.

Què és l'Autenticació de Firebase?

L'Autenticació de Firebase proporciona serveis de backend per ajudar a autenticar usuaris a la teva aplicació. Suporta diversos mètodes d'autenticació i s'integra perfectament amb altres serveis de Firebase.

Característiques Clau

  • Múltiples Mètodes d'Autenticació: Suporta correu electrònic/contrasenya, número de telèfon, Google, Facebook, Twitter i més.
  • Fàcil Integració: SDKs i APIs simples per a web, iOS i Android.
  • Segur: Funcions de seguretat integrades per protegir les dades dels usuaris.
  • Personalitzable: Permet sistemes d'autenticació personalitzats.

Configurant l'Autenticació de Firebase

Per començar a utilitzar l'Autenticació de Firebase, necessites configurar un projecte de Firebase i habilitar els mètodes d'autenticació que vols utilitzar.

Passos per Configurar l'Autenticació de Firebase

  1. Crear un Projecte de Firebase:

    • Ves a la Consola de Firebase.
    • Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

  2. Afegeix Firebase a la Teva Aplicació:

    • Segueix les instruccions per afegir Firebase a la teva aplicació web, iOS o Android.
    • Descarrega el google-services.json (per a Android) o el GoogleService-Info.plist (per a iOS) i afegeix-lo al teu projecte.

  3. Habilita els Mètodes d'Autenticació:

    • A la Consola de Firebase, navega a la secció "Autenticació".
    • Fes clic a la pestanya "Mètode d'inici de sessió".
    • Habilita els mètodes d'autenticació que vols utilitzar (per exemple, Correu electrònic/Contrasenya, Google, Facebook).

Mètodes d'Autenticació

L'Autenticació de Firebase suporta diversos mètodes per autenticar usuaris. Aquí tens alguns dels mètodes més utilitzats:

Autenticació amb Correu Electrònic/Contrasenya

Aquest és un dels mètodes més simples d'implementar. Els usuaris poden registrar-se i iniciar sessió utilitzant la seva adreça de correu electrònic i una contrasenya.

Exemple de Codi

// Inicialitza Firebase
firebase.initializeApp(firebaseConfig);

// Registra nous usuaris
firebase.auth().createUserWithEmailAndPassword(email, password)
  .then((userCredential) => {
    // Sessió iniciada 
    var user = userCredential.user;
    console.log("Usuari registrat:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en registrar-se:", errorCode, errorMessage);
  });

// Inicia sessió usuaris existents
firebase.auth().signInWithEmailAndPassword(email, password)
  .then((userCredential) => {
    // Sessió iniciada
    var user = userCredential.user;
    console.log("Usuari ha iniciat sessió:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en iniciar sessió:", errorCode, errorMessage);
  });

Autenticació amb Xarxes Socials

Firebase suporta l'autenticació a través de plataformes populars de xarxes socials com Google, Facebook i Twitter.

Exemple de Codi per a Inici de Sessió amb Google

// Inicialitza Firebase
firebase.initializeApp(firebaseConfig);

// Proveïdor de Google
var provider = new firebase.auth.GoogleAuthProvider();

// Inicia sessió amb popup
firebase.auth().signInWithPopup(provider)
  .then((result) => {
    // Això et dóna un Token d'Accés de Google.
    var token = result.credential.accessToken;
    // La informació de l'usuari que ha iniciat sessió.
    var user = result.user;
    console.log("Usuari ha iniciat sessió amb Google:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en iniciar sessió amb Google:", errorCode, errorMessage);
  });

Autenticació amb Número de Telèfon

L'autenticació amb número de telèfon permet als usuaris iniciar sessió utilitzant el seu número de telèfon. Firebase envia un codi de verificació al telèfon de l'usuari, que ha d'introduir per completar el procés d'inici de sessió.

Exemple de Codi

// Inicialitza Firebase
firebase.initializeApp(firebaseConfig);

// Inicia sessió amb número de telèfon
var phoneNumber = "+1234567890";
var appVerifier = new firebase.auth.RecaptchaVerifier('recaptcha-container');

firebase.auth().signInWithPhoneNumber(phoneNumber, appVerifier)
  .then((confirmationResult) => {
    // SMS enviat. Demana a l'usuari que introdueixi el codi del missatge.
    var code = window.prompt("Introdueix el codi de verificació que has rebut");
    return confirmationResult.confirm(code);
  })
  .then((result) => {
    // Usuari ha iniciat sessió amb èxit.
    var user = result.user;
    console.log("Usuari ha iniciat sessió amb número de telèfon:", user);
  })
  .catch((error) => {
    var errorCode = error.code;
    var errorMessage = error.message;
    console.error("Error en iniciar sessió amb número de telèfon:", errorCode, errorMessage);
  });

Gestió d'Usuaris

Firebase proporciona eines per gestionar usuaris, incloent-hi l'actualització de perfils d'usuari, la restaura de contrasenyes i l'eliminació de comptes.

Actualització de Perfils d'Usuari

Pots actualitzar els perfils d'usuari amb informació addicional com el nom de visualització i la URL de la foto.

Exemple de Codi

var user = firebase.auth().currentUser;

user.updateProfile({
  displayName: "Jane Doe",
  photoURL: "https://example.com/jane-doe/photo.jpg"
}).then(() => {
  // Perfil actualitzat amb èxit.
  console.log("Perfil d'usuari actualitzat");
}).catch((error) => {
  // Ha ocorregut un error.
  console.error("Error en actualitzar el perfil:", error);
});

Restabliment de Contrasenya

Firebase permet als usuaris restablir les seves contrasenyes via correu electrònic.

Exemple de Codi

var auth = firebase.auth();
var emailAddress = "user@example.com";

auth.sendPasswordResetEmail(emailAddress).then(() => {
  // Correu electrònic enviat.
  console.log("Correu electrònic de restabliment de contrasenya enviat");
}).catch((error) => {
  // Ha ocorregut un error.
  console.error("Error en enviar el correu electrònic de restabliment de contrasenya:", error);
});

Conclusió

L'Autenticació de Firebase simplifica el procés d'afegir autenticació d'usuaris a la teva aplicació. En suportar múltiples mètodes d'autenticació i proporcionar eines robustes de gestió d'usuaris, l'Autenticació de Firebase t'ajuda a crear aplicacions segures i fàcils d'utilitzar. En les següents seccions, aprofundirem en les funcions avançades i les personalitzacions disponibles a l'Autenticació de Firebase.
