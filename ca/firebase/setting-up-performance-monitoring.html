Introducció al Monitoratge del Rendiment

El Monitoratge del Rendiment a Firebase t'ajuda a obtenir informació sobre les característiques de rendiment de la teva aplicació. Et permet fer un seguiment i analitzar el rendiment de la teva aplicació en temps real, ajudant-te a identificar i solucionar problemes de rendiment.

Conceptes Clau

  • Traça: Una traça és un informe que conté dades capturades entre dos punts en el temps a la teva aplicació.
  • Mètrica: Una mètrica és una mesura de dades de rendiment capturades durant una traça.
  • Traça Personalitzada: Les traces personalitzades són definides per tu per mesurar el rendiment de parts específiques de la teva aplicació.
  • Traça Automàtica: Aquestes són traces predefinides que Firebase recull automàticament per tu.

Configuració del Monitoratge del Rendiment a Firebase

Pas 1: Afegeix Firebase al Teu Projecte

Abans de poder utilitzar el Monitoratge del Rendiment, has d'afegir Firebase al teu projecte.

Per Android

  1. Afegeix el SDK de Firebase als teus fitxers build.gradle.

    // build.gradle a nivell de projecte
buildscript {
    dependencies {
        // Afegeix el classpath de Google services
        classpath 'com.google.gms:google-services:4.3.10'
    }
}

// build.gradle a nivell d'aplicació
dependencies {
    // Afegeix el SDK de Monitoratge del Rendiment
    implementation 'com.google.firebase:firebase-perf:20.0.3'
}

  2. Inicialitza Firebase a la teva classe Application.

    public class MyApplication extends Application {
    @Override
    public void onCreate() {
        super.onCreate();
        // Inicialitza Firebase
        FirebaseApp.initializeApp(this);
    }
}

Per iOS

  1. Afegeix el SDK de Firebase al teu Podfile.

    pod 'Firebase/Performance'

  2. Inicialitza Firebase al teu AppDelegate.

    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {
    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        // Inicialitza Firebase
        FirebaseApp.configure()
        return true
    }
}

Pas 2: Activa el Monitoratge del Rendiment a la Consola de Firebase

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció de Monitoratge del Rendiment.
  4. Fes clic al botó "Activa el Monitoratge del Rendiment".

Pas 3: Verifica la Instal·lació

Per verificar que el Monitoratge del Rendiment està configurat correctament:

  • Executa la teva aplicació.
  • Realitza algunes accions a la teva aplicació per generar dades de rendiment.
  • Comprova la Consola de Firebase per veure si apareixen les dades.

Ús del Monitoratge del Rendiment

Traces Automàtiques

Firebase recull automàticament diverses traces per tu, com ara:

  • Traça d'inici de l'aplicació: Mesura el temps que triga la teva aplicació a iniciar-se.
  • Traça de renderització de pantalla: Mesura el temps que triga a renderitzar-se una pantalla.

Traces Personalitzades

Pots definir traces personalitzades per mesurar parts específiques de la teva aplicació.

Exemple: Traça Personalitzada a Android

// Inicia una traça personalitzada
Trace myTrace = FirebasePerformance.getInstance().newTrace("test_trace");
myTrace.start();

// Codi que vols mesurar
doSomeWork();

// Atura la traça personalitzada
myTrace.stop();

Exemple: Traça Personalitzada a iOS

// Inicia una traça personalitzada
let trace = Performance.startTrace(name: "test_trace")

// Codi que vols mesurar
doSomeWork()

// Atura la traça personalitzada
trace.stop()

Afegir Mètriques Personalitzades

Pots afegir mètriques personalitzades a les teves traces per mesurar esdeveniments específics.

Exemple: Mètrica Personalitzada a Android

// Inicia una traça personalitzada
Trace myTrace = FirebasePerformance.getInstance().newTrace("test_trace");
myTrace.start();

// Incrementa una mètrica personalitzada
myTrace.incrementMetric("custom_metric", 1);

// Atura la traça personalitzada
myTrace.stop();

Exemple: Mètrica Personalitzada a iOS

// Inicia una traça personalitzada
let trace = Performance.startTrace(name: "test_trace")

// Incrementa una mètrica personalitzada
trace.incrementMetric("custom_metric", by: 1)

// Atura la traça personalitzada
trace.stop()

Conclusió

Configurar el Monitoratge del Rendiment a Firebase és un procés senzill que pot proporcionar informació valuosa sobre el rendiment de la teva aplicació. Aprofitant tant les traces automàtiques com les personalitzades, pots monitoritzar i millorar el rendiment de la teva aplicació, assegurant una millor experiència d'usuari. Recorda revisar regularment la Consola de Firebase per obtenir dades de rendiment i utilitzar-les per identificar i abordar qualsevol problema de rendiment.
