Firebase Hosting et permet desplegar aplicacions web de manera ràpida i segura. Una de les característiques potents que ofereix és la capacitat d'utilitzar dominis personalitzats per a les teves aplicacions web. Aquesta secció et guiarà a través del procés de configuració i gestió de dominis personalitzats a Firebase, des de nivells bàsics fins a avançats.

Introducció als dominis personalitzats

Els dominis personalitzats et permeten utilitzar el teu propi nom de domini (per exemple, www.elteudomini.com) en lloc del subdomini predeterminat de Firebase (per exemple, elteuapp.firebaseapp.com). Això pot millorar la identitat de la teva marca i fer que la teva aplicació sigui més accessible.

Conceptes clau

  • Domini personalitzat: El teu propi nom de domini que vols utilitzar amb el teu lloc allotjat a Firebase.
  • Registres DNS: Configuracions que indiquen a Internet on trobar el teu domini.
  • Certificats SSL: Certificats de seguretat que xifren les dades entre els teus usuaris i el teu lloc.

Configuració d'un domini personalitzat

Pas 1: Compra un domini

Abans de poder configurar un domini personalitzat, has de comprar-ne un a un registrador de dominis com Google Domains, GoDaddy o Namecheap.

Pas 2: Afegeix el teu domini a Firebase

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Al menú de l'esquerra, fes clic a Hosting.
  4. Fes clic a Add Custom Domain.
  5. Introdueix el teu nom de domini i fes clic a Continue.

Pas 3: Verifica la propietat del domini

Firebase et proporcionarà un registre TXT que has d'afegir a la configuració del teu DNS per verificar que ets el propietari del domini.

Exemple:

Tipus: TXT
Nom: @
Valor: firebase=el-teu-codi-de-verificació

Pas 4: Actualitza els registres DNS

Després de verificar el teu domini, has d'actualitzar els registres DNS per apuntar a Firebase Hosting.

Exemple:

Tipus: A
Nom: @
Valor: 151.101.1.195

Pas 5: Desplega el teu lloc

Desplega el teu lloc utilitzant el Firebase CLI:

firebase deploy

Gestió de certificats SSL

Firebase provisiona automàticament certificats SSL per al teu domini personalitzat, assegurant que el teu lloc es serveixi a través de HTTPS.

Punts clau

  • SSL automàtic: Firebase gestiona els certificats SSL per a tu.
  • Renovació: Els certificats SSL es renoven automàticament per Firebase.
  • SSL personalitzat: Els usuaris avançats poden carregar els seus propis certificats SSL si cal.

Configuració avançada

Redireccions i reescriptures

Pots configurar redireccions i reescriptures al teu fitxer firebase.json per gestionar com es manegen les URL.

Exemple:

{
  "hosting": {
    "redirects": [
      {
        "source": "/old-page",
        "destination": "/new-page",
        "type": 301
      }
    ],
    "rewrites": [
      {
        "source": "/api/**",
        "function": "api"
      }
    ]
  }
}

Múltiples llocs

Firebase Hosting admet múltiples llocs dins d'un sol projecte de Firebase. Això és útil per gestionar diferents entorns (per exemple, staging i producció) o diferents parts d'una aplicació gran.

Exemple:

{
  "hosting": [
    {
      "target": "project1",
      "public": "public1",
      "ignore": ["firebase.json", "**/.*", "**/node_modules/**"]
    },
    {
      "target": "project2",
      "public": "public2",
      "ignore": ["firebase.json", "**/.*", "**/node_modules/**"]
    }
  ]
}

Resolució de problemes

Problemes comuns

  • Propagació DNS: Els canvis als registres DNS poden trigar fins a 48 hores a propagar-se.
  • Errors SSL: Assegura't que el teu domini està correctament verificat i que els registres DNS estan configurats correctament.
  • Problemes de desplegament: Revisa la sortida del Firebase CLI per detectar errors i assegura't que el teu firebase.json està configurat correctament.

Conclusió

Configurar un domini personalitzat a Firebase Hosting millora el professionalisme i l'accessibilitat de la teva aplicació web. Seguint els passos descrits en aquesta secció, pots configurar i gestionar fàcilment dominis personalitzats, assegurant una experiència segura i de marca per als teus usuaris. Tant si ets un usuari principiant com avançat, Firebase proporciona les eines i la flexibilitat necessàries per gestionar l'allotjament de la teva aplicació web de manera efectiva.
