Firebase Test Lab proporciona una infraestructura basada en el núvol per provar aplicacions Android i iOS. Et permet executar la teva aplicació en una àmplia gamma de dispositius i configuracions, assegurant que la teva aplicació funcioni bé per a tots els usuaris. Aquesta secció et guiarà a través de la configuració de Firebase Test Lab, des dels conceptes bàsics fins a configuracions més avançades.

Introducció a Firebase Test Lab

  • Què és Firebase Test Lab?

    • Una infraestructura de proves d'aplicacions basada en el núvol.
    • Suporta tant plataformes Android com iOS.
    • Permet provar en una varietat de dispositius i configuracions.

  • Per què utilitzar Firebase Test Lab?

    • Assegura la compatibilitat de l'aplicació en diferents dispositius.
    • Identifica errors i problemes de rendiment aviat.
    • Estalvia temps i recursos en comparació amb les proves manuals.

Configuració de Firebase Test Lab

Prerequisits

  • Projecte Firebase

    • Assegura't de tenir un projecte Firebase configurat. Si no, crea'n un al Firebase Console.

  • Projecte de Google Cloud

    • Firebase Test Lab requereix un projecte de Google Cloud. Enllaça el teu projecte Firebase a un projecte de Google Cloud si no està ja enllaçat.

Activació de Firebase Test Lab

  1. Navega al Firebase Console

    • Ves al Firebase Console.
    • Selecciona el teu projecte.

  2. Accedeix a Test Lab

    • Al menú de l'esquerra, fes clic a "Test Lab".
    • Si se't demana, activa l'API de Test Lab.

Executar la teva primera prova

  • Carrega la teva aplicació

    • Per Android: Carrega el teu fitxer APK.
    • Per iOS: Carrega el teu fitxer IPA.

  • Selecciona el tipus de prova

    • Prova Robo: Explora automàticament la teva aplicació.
    • Prova d'instrumentació: Executa els teus scripts de prova personalitzats.

  • Tria els dispositius

    • Selecciona els dispositius i configuracions en què vols provar. Firebase Test Lab proporciona una àmplia gamma de dispositius amb diferents versions de SO i mides de pantalla.

  • Executa la prova

    • Fes clic a "Run" per començar la prova. Firebase Test Lab executarà les proves en els dispositius seleccionats.

Visualització dels resultats de les proves

  • Accés als resultats

    • Un cop les proves estiguin completes, pots veure els resultats al Firebase Console a la secció "Test Lab".

  • Interpretació dels resultats

    • Resum de la prova: Proporciona una visió general dels resultats de la prova.
    • Registres i captures de pantalla: Registres detallats i captures de pantalla de les proves.
    • Vídeos: Gravacions de vídeo de les proves.

Configuracions avançades

Proves d'instrumentació personalitzades

  • Escriure proves d'instrumentació

    • Utilitza marcs com Espresso per Android o XCTest per iOS per escriure scripts de prova personalitzats.

  • Carregar scripts de prova

    • Carrega els teus scripts de prova juntament amb la teva aplicació a Firebase Test Lab.

Proves en paral·lel

  • Executar proves en paral·lel
    • Firebase Test Lab et permet executar proves en paral·lel en múltiples dispositius, reduint el temps total de prova.

Integració amb CI/CD

  • Integració contínua/Desplegament continu

    • Integra Firebase Test Lab amb pipelines de CI/CD utilitzant eines com Jenkins, CircleCI o GitHub Actions.

  • Exemple: Workflow de GitHub Actions

name: Firebase Test Lab

on: [push]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
    - name: Checkout code
      uses: actions/checkout@v2
    - name: Set up JDK
      uses: actions/setup-java@v1
      with:
        java-version: '11'
    - name: Build APK
      run: ./gradlew assembleDebug
    - name: Run tests on Firebase Test Lab
      run: |
        gcloud firebase test android run \
          --type instrumentation \
          --app app/build/outputs/apk/debug/app-debug.apk \
          --test app/build/outputs/apk/androidTest/debug/app-debug-androidTest.apk
      env:
        GOOGLE_APPLICATION_CREDENTIALS: ${{ secrets.GOOGLE_APPLICATION_CREDENTIALS }}

Conclusió

Firebase Test Lab és una eina poderosa per assegurar que la teva aplicació funcioni perfectament en una varietat de dispositius i configuracions. Seguint aquesta guia, pots configurar i executar proves en la teva aplicació, veure resultats detallats i integrar les proves en el teu pipeline de CI/CD. Això t'ajudarà a detectar errors aviat, millorar el rendiment de l'aplicació i oferir una millor experiència als teus usuaris.
