Firebase Hosting és un servei d'allotjament totalment gestionat per a contingut estàtic i dinàmic, així com microserveis. Proporciona un allotjament ràpid i segur per a aplicacions web, llocs web estàtics i serveis backend sense servidor.

Què és Firebase Hosting?

Firebase Hosting és part del conjunt d'eines i serveis de Firebase proporcionats per Google. Permet als desenvolupadors desplegar aplicacions web i contingut estàtic a una xarxa de distribució de contingut global (CDN) amb una sola comanda.

Característiques clau de Firebase Hosting

  • Ràpid i segur: Firebase Hosting serveix el teu contingut a través d'una connexió segura (HTTPS) i utilitza una CDN global per garantir una entrega ràpida.
  • Dominis personalitzats: Pots connectar el teu domini personalitzat a Firebase Hosting.
  • SSL automàtic: Firebase Hosting proporciona automàticament certificats SSL per als teus dominis personalitzats.
  • Versionat: Cada desplegament està versionat, permetent-te tornar a versions anteriors si cal.
  • Integració amb serveis de Firebase: S'integra perfectament amb altres serveis de Firebase com Firestore, Autenticació i Cloud Functions.

Configuració de Firebase Hosting

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un compte de Google.
  • Node.js i npm instal·lats a la teva màquina.
  • Firebase CLI instal·lat.

Guia pas a pas

  1. Instal·la Firebase CLI:

    npm install -g firebase-tools

  2. Inicia sessió a Firebase:

    firebase login

  3. Inicialitza Firebase al teu projecte: Navega al directori del teu projecte i executa:

    firebase init
    • Selecciona "Hosting" de la llista de característiques de Firebase.
    • Tria un projecte de Firebase existent o crea'n un de nou.
    • Especifica el directori públic (per exemple, public).
    • Configura com una aplicació d'una sola pàgina si és aplicable.

  4. Desplega el teu lloc:

    firebase deploy

Exemple: Desplegant un lloc HTML senzill

  1. Crea un directori de projecte:

    mkdir my-firebase-site
cd my-firebase-site

  2. Inicialitza Firebase:

    firebase init

  3. Crea un fitxer index.html:

    <!DOCTYPE html>
<html>
<head>
    <title>My Firebase Site</title>
</head>
<body>
    <h1>Welcome to My Firebase Site!</h1>
</body>
</html>

  4. Desplega el lloc:

    firebase deploy

Gestió del teu Firebase Hosting

Dominis personalitzats

Per afegir un domini personalitzat:

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció d'Hosting.
  4. Fes clic a "Add Custom Domain" i segueix les instruccions.

Tornar a una versió anterior

Firebase Hosting fa un seguiment de l'historial de desplegaments. Per tornar a una versió anterior:

  1. Ves a la Consola de Firebase.
  2. Selecciona el teu projecte.
  3. Navega a la secció d'Hosting.
  4. Fes clic al botó "View" al costat de la versió a la qual vols tornar.
  5. Fes clic a "Rollback" per desplegar la versió seleccionada.

Conclusió

Firebase Hosting és una eina poderosa per desplegar i gestionar aplicacions web. Ofereix una gamma de característiques que faciliten el desplegament, la gestió i la seguretat del teu contingut web. Seguint els passos descrits en aquesta guia, pots començar ràpidament amb Firebase Hosting i aprofitar les seves capacitats robustes.

En les següents seccions, explorarem característiques i integracions més avançades de Firebase Hosting, com l'ús de Cloud Functions i la integració amb altres serveis de Firebase.
