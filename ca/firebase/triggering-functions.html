Introducció a les Funcions de Firebase

Les Funcions de Firebase et permeten executar codi backend en resposta a esdeveniments desencadenats per característiques de Firebase i sol·licituds HTTPS. Això et permet estendre la funcionalitat de la teva aplicació sense gestionar els teus propis servidors.

Configuració de les Funcions de Firebase

Abans de submergir-te en l'activació de funcions, necessites configurar les Funcions de Firebase al teu projecte.

  1. Instal·la Firebase CLI:

    npm install -g firebase-tools

  2. Inicialitza Firebase al teu projecte:

    firebase init functions

  3. Instal·la les dependències:

    cd functions
npm install

Tipus de Triggers

Les Funcions de Firebase poden ser activades per diversos esdeveniments. Aquí tens els principals tipus:

  • HTTP Triggers: Activades per sol·licituds HTTP.
  • Auth Triggers: Activades per esdeveniments d'Autenticació de Firebase.
  • Firestore Triggers: Activades per esdeveniments de la base de dades Firestore.
  • Realtime Database Triggers: Activades per esdeveniments de la base de dades en temps real.
  • Storage Triggers: Activades per esdeveniments de Cloud Storage.
  • Analytics Triggers: Activades per esdeveniments d'Analytics de Firebase.
  • Pub/Sub Triggers: Activades per esdeveniments de Google Cloud Pub/Sub.

HTTP Triggers

Els triggers HTTP et permeten crear funcions que poden ser cridades mitjançant sol·licituds HTTP.

Exemple

const functions = require('firebase-functions');

exports.helloWorld = functions.https.onRequest((request, response) => {
    response.send("Hola des de Firebase!");
});

Explicació

  • functions.https.onRequest: Això especifica que la funció serà activada per una sol·licitud HTTP.
  • request: Conté informació sobre la sol·licitud HTTP.
  • response: S'utilitza per enviar una resposta al client.

Firestore Triggers

Els triggers de Firestore responen als canvis en els documents de Firestore.

Exemple

exports.onUserCreate = functions.firestore
    .document('users/{userId}')
    .onCreate((snap, context) => {
        const newValue = snap.data();
        console.log('Nou usuari creat:', newValue);
    });

Explicació

  • functions.firestore.document('users/{userId}'): Especifica el camí del document de Firestore a observar.
  • onCreate: Activat quan es crea un nou document.
  • snap: Conté la instantània del document creat.
  • context: Conté metadades sobre l'esdeveniment.

Realtime Database Triggers

Els triggers de la base de dades en temps real responen als canvis en la base de dades en temps real.

Exemple

exports.onDataChange = functions.database
    .ref('/messages/{messageId}')
    .onWrite((change, context) => {
        const beforeData = change.before.val();
        const afterData = change.after.val();
        console.log('Dades canviades de', beforeData, 'a', afterData);
    });

Explicació

  • functions.database.ref('/messages/{messageId}'): Especifica el camí de la base de dades en temps real a observar.
  • onWrite: Activat quan es creen, actualitzen o eliminen dades.
  • change: Conté les dades abans i després del canvi.
  • context: Conté metadades sobre l'esdeveniment.

Auth Triggers

Els triggers d'autenticació responen als esdeveniments d'Autenticació de Firebase.

Exemple

exports.onUserSignUp = functions.auth.user().onCreate((user) => {
    console.log('Nou usuari registrat:', user.email);
});

Explicació

  • functions.auth.user().onCreate: Activat quan un nou usuari es registra.
  • user: Conté informació sobre l'usuari recentment creat.

Storage Triggers

Els triggers d'emmagatzematge responen als canvis en Cloud Storage.

Exemple

exports.onFileUpload = functions.storage.object().onFinalize((object) => {
    console.log('Fitxer pujat:', object.name);
});

Explicació

  • functions.storage.object().onFinalize: Activat quan es puja un fitxer.
  • object: Conté informació sobre el fitxer pujat.

Conclusió

Les Funcions de Firebase proporcionen una manera poderosa d'estendre les capacitats del backend de la teva aplicació sense gestionar els teus propis servidors. Entenent i utilitzant diferents tipus de triggers, pots respondre a diversos esdeveniments en el teu projecte de Firebase, des de sol·licituds HTTP fins a canvis en la base de dades i esdeveniments d'autenticació d'usuaris. Practica creant i desplegant diferents tipus de funcions per convertir-te en un expert en l'ús de les Funcions de Firebase.
