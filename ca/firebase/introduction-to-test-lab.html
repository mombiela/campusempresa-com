Introducció a Firebase Test Lab
Configurar Test Lab ►

Firebase Test Lab és una infraestructura de proves d'aplicacions basada en el núvol que et permet provar la teva aplicació en una àmplia gamma de dispositius i configuracions. Això assegura que la teva aplicació funcioni bé en diferents entorns, ajudant-te a detectar problemes abans que arribin als teus usuaris.

Conceptes Clau

  • Matriu de Dispositius: Una combinació de diferents dispositius, versions de SO i localitzacions en què pots provar la teva aplicació.
  • Proves d'Instrumentació: Proves que s'executen en dispositius reals i poden interactuar amb la interfície d'usuari de l'aplicació.
  • Proves Robo: Proves automàtiques que exploren la interfície d'usuari de l'aplicació sense necessitat de scripts de prova.
  • Proves de Bucle de Joc: Dissenyades específicament per provar jocs, permetent simular interaccions d'usuari.
  • Resultats de les Proves: Informes detallats que inclouen registres, captures de pantalla i vídeos de les execucions de prova.

Configuració de Firebase Test Lab

Prerequisits

Abans de començar a utilitzar Firebase Test Lab, assegura't de tenir el següent:

  • Un projecte de Firebase configurat.
  • Una aplicació Android o iOS connectada al teu projecte de Firebase.
  • Coneixements bàsics d'escriptura de proves per a la teva aplicació.

Afegir Firebase Test Lab al Teu Projecte

  1. Navega a la Consola de Firebase:

    • Ves a la Consola de Firebase i selecciona el teu projecte.
    • Al menú de l'esquerra, fes clic a "Test Lab".

  2. Carrega la Teva Aplicació:

    • Fes clic a "Executar una prova".
    • Carrega l'arxiu APK (per a Android) o IPA (per a iOS) de la teva aplicació.

  3. Selecciona el Tipus de Prova:

    • Tria entre Proves d'Instrumentació, Proves Robo o Proves de Bucle de Joc.

  4. Configura la Matriu de Dispositius:

    • Selecciona els dispositius, versions de SO i localitzacions en què vols provar.

  5. Executa la Prova:

    • Fes clic a "Iniciar Prova" per començar el procés de prova.

Escriure Proves per a Firebase Test Lab

Proves d'Instrumentació

Les proves d'instrumentació s'escriuen utilitzant marcs de proves com Espresso per a Android o XCTest per a iOS. A continuació es mostra un exemple d'una prova d'Espresso per a una aplicació Android:

@RunWith(AndroidJUnit4.class)
public class ExampleInstrumentedTest {
    @Rule
    public ActivityTestRule<MainActivity> activityRule =
        new ActivityTestRule<>(MainActivity.class);

    @Test
    public void useAppContext() {
        // Context de l'aplicació sota prova.
        Context appContext = InstrumentationRegistry.getInstrumentation().getTargetContext();
        assertEquals("com.example.myapp", appContext.getPackageName());
    }

    @Test
    public void checkButtonClick() {
        // Realitza un clic al botó amb ID 'button'
        onView(withId(R.id.button)).perform(click());
        // Comprova si el text view amb ID 'textView' mostra el text esperat
        onView(withId(R.id.textView)).check(matches(withText("Button Clicked")));
    }
}

Proves Robo

Les proves Robo no requereixen cap script de prova. Firebase Test Lab explorarà automàticament la interfície d'usuari de la teva aplicació i realitzarà accions com fer clic a botons i introduir text.

Proves de Bucle de Joc

Les proves de Bucle de Joc estan dissenyades per a jocs i requereixen que defineixis escenaris específics. A continuació es mostra un exemple d'una prova simple de bucle de joc:

public class GameLoopTest {
    public static void main(String[] args) {
        // Inicialitza el joc
        Game game = new Game();
        game.start();

        // Simula interaccions d'usuari
        game.tapScreen();
        game.swipeLeft();
        game.swipeRight();

        // Finalitza el joc
        game.stop();
    }
}

Veure els Resultats de les Proves

Després que les teves proves s'hagin executat, pots veure els resultats a la Consola de Firebase:

  • Registres: Registres detallats de l'execució de la prova.
  • Captures de Pantalla: Captures de pantalla preses durant la prova.
  • Vídeos: Vídeos de l'execució de la prova.
  • Mètriques de Rendiment: Mètriques com l'ús de la CPU, l'ús de la memòria i l'activitat de xarxa.

Conclusió

Firebase Test Lab és una eina poderosa per assegurar que la teva aplicació funcioni bé en una varietat de dispositius i configuracions. Utilitzant Proves d'Instrumentació, Proves Robo i Proves de Bucle de Joc, pots detectar problemes aviat i lliurar una aplicació més robusta als teus usuaris. Assegura't d'executar proves regularment i revisar els resultats per mantenir una alta qualitat de l'aplicació.
