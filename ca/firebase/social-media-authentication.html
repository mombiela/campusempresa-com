Firebase proporciona un sistema d'autenticació robust i fàcil d'utilitzar que admet diverses plataformes de xarxes socials. Aquest tema et guiarà a través del procés d'integració de l'autenticació de xarxes socials a la teva aplicació utilitzant Firebase, des de nivells principiants fins a avançats.

Introducció a l'Autenticació de Firebase

L'objectiu de l'Autenticació de Firebase és facilitar que els teus usuaris iniciïn sessió sense haver de gestionar el procés d'inici de sessió manualment. Admet l'autenticació mitjançant contrasenyes, números de telèfon i proveïdors d'identitat federada populars com Google, Facebook i Twitter.

Conceptes Clau

  • Autenticació de Firebase: Un servei que pot autenticar usuaris utilitzant només codi del costat del client.
  • OAuth: Un estàndard obert per a la delegació d'accés comunament utilitzat com una manera de concedir a llocs web o aplicacions accés limitat a la informació de l'usuari sense exposar contrasenyes.
  • Proveïdor: Un servei que proporciona autenticació d'usuaris, com Google, Facebook o Twitter.

Configuració de Firebase al Teu Projecte

Abans de poder utilitzar l'Autenticació de Firebase, has de configurar Firebase al teu projecte.

Passos per Configurar Firebase

  1. Crear un Projecte de Firebase:

    • Ves a la Consola de Firebase.
    • Fes clic a "Afegeix projecte" i segueix els passos de configuració.

  2. Afegeix Firebase a la Teva Aplicació:

    • Per a aplicacions web, afegeix el SDK de Firebase al teu projecte.
    • Per a Android i iOS, descarrega el fitxer google-services.json o GoogleService-Info.plist i afegeix-lo al teu projecte.

  3. Habilita Proveïdors d'Autenticació:

    • A la Consola de Firebase, ves a la secció d'Autenticació.
    • Habilita els proveïdors de xarxes socials que vols utilitzar (per exemple, Google, Facebook, Twitter).

Implementació de l'Inici de Sessió amb Google

L'Inici de Sessió amb Google és un dels mètodes d'autenticació més comuns. Aquí tens com implementar-lo.

Passos per Implementar l'Inici de Sessió amb Google

  1. Habilita l'Inici de Sessió amb Google a la Consola de Firebase:

    • Ves a la secció d'Autenticació.
    • Habilita Google com a proveïdor d'inici de sessió.

  2. Afegeix l'Inici de Sessió amb Google a la Teva Aplicació:

    • Per a aplicacions web, inclou la biblioteca d'Inici de Sessió amb Google: 
      <script src="https://apis.google.com/js/platform.js" async defer></script>

  3. Autentica amb Firebase:

    • Utilitza el SDK d'Autenticació de Firebase per gestionar el procés d'inici de sessió.
    // Inicialitza Firebase
var firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
firebase.initializeApp(firebaseConfig);

// Inici de Sessió amb Google
var provider = new firebase.auth.GoogleAuthProvider();

function signInWithGoogle() {
  firebase.auth().signInWithPopup(provider).then((result) => {
    var user = result.user;
    console.log("Usuari ha iniciat sessió: ", user);
  }).catch((error) => {
    console.error("Error durant l'inici de sessió: ", error);
  });
}

Implementació de l'Inici de Sessió amb Facebook

L'Inici de Sessió amb Facebook és un altre mètode popular. Aquí tens com implementar-lo.

Passos per Implementar l'Inici de Sessió amb Facebook

  1. Habilita l'Inici de Sessió amb Facebook a la Consola de Firebase:

    • Ves a la secció d'Autenticació.
    • Habilita Facebook com a proveïdor d'inici de sessió i proporciona les credencials necessàries (ID d'Aplicació i Secret d'Aplicació).

  2. Afegeix el SDK de Facebook a la Teva Aplicació:

Temes Avançats

Gestió de l'Estat d'Autenticació

Per proporcionar una experiència d'usuari fluida, has de gestionar l'estat d'autenticació de l'usuari.

firebase.auth().onAuthStateChanged((user) => {
  if (user) {
    console.log("Usuari ha iniciat sessió: ", user);
  } else {
    console.log("Cap usuari ha iniciat sessió.");
  }
});

Enllaçar Múltiples Proveïdors

Els usuaris poden voler enllaçar múltiples proveïdors d'autenticació a un sol compte.

var googleProvider = new firebase.auth.GoogleAuthProvider();
var facebookProvider = new firebase.auth.FacebookAuthProvider();

firebase.auth().currentUser.linkWithPopup(googleProvider).then((result) => {
  console.log("Compte enllaçat amb Google: ", result.user);
}).catch((error) => {
  console.error("Error en enllaçar el compte: ", error);
});
