Pujar fitxers a Firebase és un requisit comú per a moltes aplicacions, ja sigui per emmagatzemar contingut generat per l'usuari, fotos de perfil o altres tipus de mitjans. Firebase proporciona un servei robust i fàcil d'usar anomenat Firebase Storage per gestionar la pujada de fitxers. Aquest tema et guiarà a través del procés de pujar fitxers a Firebase, des de nivells principiants fins a avançats.

Introducció a Firebase Storage

Firebase Storage és un servei d'emmagatzematge d'objectes potent, senzill i rendible construït per a l'escala de Google. Et permet emmagatzemar i servir contingut generat per l'usuari, com ara fotos o vídeos.

Conceptes Clau

  • Buckets: Contenidors d'emmagatzematge per als teus fitxers.
  • Referències: Punts de referència a la ubicació dels teus fitxers dins d'un bucket.
  • Metadades: Informació sobre els teus fitxers, com ara mida, tipus de contingut, etc.

Configurant Firebase Storage

Abans de poder pujar fitxers, necessites configurar Firebase Storage en el teu projecte.

Passos per Configurar Firebase Storage

  1. Crear un Projecte Firebase:

    • Ves a la Consola de Firebase.
    • Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.

  2. Afegeix Firebase a la Teva Aplicació:

    • Segueix les instruccions per afegir Firebase a la teva aplicació web, Android o iOS.

  3. Habilita Firebase Storage:

    • A la Consola de Firebase, navega a la secció "Storage".
    • Fes clic a "Comença" i segueix les instruccions per habilitar Firebase Storage.

Pujant Fitxers (Nivell Principiant)

Al nivell principiant, cobrirem els conceptes bàsics de pujar un fitxer a Firebase Storage utilitzant un formulari HTML senzill i JavaScript.

Formulari HTML per Pujar Fitxers

<!DOCTYPE html>
<html>
<head>
  <title>Pujar Fitxer a Firebase</title>
</head>
<body>
  <input type="file" id="fileInput" />
  <button onclick="uploadFile()">Pujar</button>

  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
  <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-storage.js"></script>
  <script src="path/to/your/firebase-config.js"></script>
  <script src="path/to/your/upload-script.js"></script>
</body>
</html>

JavaScript per Pujar Fitxers

// Inicialitza Firebase
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

firebase.initializeApp(firebaseConfig);
const storage = firebase.storage();

function uploadFile() {
  const fileInput = document.getElementById('fileInput');
  const file = fileInput.files[0];
  const storageRef = storage.ref('uploads/' + file.name);

  const uploadTask = storageRef.put(file);

  uploadTask.on('state_changed', 
    (snapshot) => {
      // Observa esdeveniments de canvi d'estat com ara progrés, pausa i reprendre
      const progress = (snapshot.bytesTransferred / snapshot.totalBytes) * 100;
      console.log('La pujada està al ' + progress + '% completat');
    }, 
    (error) => {
      // Gestiona pujades no reeixides
      console.error('La pujada ha fallat:', error);
    }, 
    () => {
      // Gestiona pujades reeixides en completar
      console.log('Pujada reeixida!');
    }
  );
}

Explicació

  • Formulari HTML: Un formulari senzill amb un input de fitxer i un botó per activar la pujada.
  • JavaScript:
    • Inicialitza Firebase amb la teva configuració.
    • Obté el fitxer de l'element input.
    • Crea una referència a la ubicació d'emmagatzematge.
    • Pujar el fitxer i monitoritzar l'estat de la pujada.

Pujant Fitxers (Nivell Intermedi)

Al nivell intermedi, gestionarem escenaris més complexos, com ara mostrar el progrés de la pujada i gestionar errors de manera més elegant.

Mostrant el Progrés de la Pujada

function uploadFile() {
  const fileInput = document.getElementById('fileInput');
  const file = fileInput.files[0];
  const storageRef = storage.ref('uploads/' + file.name);

  const uploadTask = storageRef.put(file);

  uploadTask.on('state_changed', 
    (snapshot) => {
      const progress = (snapshot.bytesTransferred / snapshot.totalBytes) * 100;
      console.log('La pujada està al ' + progress + '% completat');
      document.getElementById('progress').innerText = 'La pujada està al ' + progress + '% completat';
    }, 
    (error) => {
      console.error('La pujada ha fallat:', error);
      document.getElementById('error').innerText = 'La pujada ha fallat: ' + error.message;
    }, 
    () => {
      console.log('Pujada reeixida!');
      document.getElementById('success').innerText = 'Pujada reeixida!';
    }
  );
}

Pujant Fitxers (Nivell Avançat)

Al nivell avançat, cobrirem funcions més sofisticades com ara pujades reanudables, gestió de fitxers grans i configuració de metadades.

Pujades Reanudables i Metadades

function uploadFile() {
  const fileInput = document.getElementById('fileInput');
  const file = fileInput.files[0];
  const storageRef = storage.ref('uploads/' + file.name);

  const metadata = {
    contentType: file.type,
    customMetadata: {
      'uploadedBy': 'User123',
      'description': 'Foto de perfil'
    }
  };

  const uploadTask = storageRef.put(file, metadata);

  uploadTask.on('state_changed', 
    (snapshot) => {
      const progress = (snapshot.bytesTransferred / snapshot.totalBytes) * 100;
      console.log('La pujada està al ' + progress + '% completat');
      document.getElementById('progress').innerText = 'La pujada està al ' + progress + '% completat';
    }, 
    (error) => {
      console.error('La pujada ha fallat:', error);
      document.getElementById('error').innerText = 'La pujada ha fallat: ' + error.message;
    }, 
    () => {
      console.log('Pujada reeixida!');
      document.getElementById('success').innerText = 'Pujada reeixida!';
    }
  );
}

Explicació

  • Metadades: Afegeix informació addicional al fitxer que s'està pujant.
  • Pujades Reanudables: Firebase Storage gestiona automàticament les pujades reanudables, facilitant la pujada de fitxers grans.

Conclusió

Pujar fitxers a Firebase Storage és una funció potent que es pot implementar a diversos nivells de complexitat. Des d'una pujada de fitxers senzilla fins a la gestió de fitxers grans amb metadades, Firebase Storage proporciona una solució escalable i fàcil d'usar per a les necessitats d'emmagatzematge de la teva aplicació. Seguint els passos i exemples proporcionats, pots integrar la pujada de fitxers a la teva aplicació de manera eficient i efectiva.
