Introducció

Firebase Realtime Database és una base de dades NoSQL allotjada al núvol que et permet emmagatzemar i sincronitzar dades entre els teus usuaris en temps real. En aquesta secció, cobrirem les operacions fonamentals CRUD (Crear, Llegir, Actualitzar, Eliminar) a Firebase Realtime Database, adequades tant per a principiants com per a usuaris avançats.

Configuració de Firebase Realtime Database

Abans d'endinsar-nos en les operacions CRUD, assegura't d'haver configurat el teu projecte Firebase i d'haver inicialitzat Firebase a la teva aplicació.

Inicialitzant Firebase

// Importa i configura Firebase
import firebase from 'firebase/app';
import 'firebase/database';

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  databaseURL: "https://YOUR_PROJECT_ID.firebaseio.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

firebase.initializeApp(firebaseConfig);
const database = firebase.database();

Operació de Creació

Crear dades a Firebase Realtime Database implica escriure dades a una referència especificada.

Exemple: Afegir un Nou Usuari

const addUser = (userId, name, email) => {
  firebase.database().ref('users/' + userId).set({
    username: name,
    email: email
  });
};

// Ús
addUser('1', 'John Doe', 'john.doe@example.com');
  • Explicació: El mètode set escriu dades a la referència especificada (users/userId). Si la referència no existeix, es crearà.

Operació de Lectura

Llegir dades de Firebase Realtime Database es pot fer utilitzant els mètodes once o on.

Exemple: Recuperar Dades d'Usuari

const getUser = (userId) => {
  firebase.database().ref('users/' + userId).once('value').then((snapshot) => {
    const userData = snapshot.val();
    console.log(userData);
  });
};

// Ús
getUser('1');
  • Explicació: El mètode once llegeix les dades a la referència especificada una vegada. El mètode snapshot.val() recupera les dades.

Operació d'Actualització

Actualitzar dades a Firebase Realtime Database es pot fer utilitzant el mètode update.

Exemple: Actualitzar el Correu Electrònic d'un Usuari

const updateUserEmail = (userId, newEmail) => {
  firebase.database().ref('users/' + userId).update({
    email: newEmail
  });
};

// Ús
updateUserEmail('1', 'new.email@example.com');
  • Explicació: El mètode update actualitza els camps especificats a la referència sense sobrescriure totes les dades.

Operació d'Eliminació

Eliminar dades a Firebase Realtime Database es pot fer utilitzant el mètode remove.

Exemple: Eliminar un Usuari

const deleteUser = (userId) => {
  firebase.database().ref('users/' + userId).remove();
};

// Ús
deleteUser('1');
  • Explicació: El mètode remove elimina les dades a la referència especificada.

Conceptes Avançats

Per a usuaris avançats, Firebase Realtime Database ofereix operacions més complexes com ara transaccions i capacitats fora de línia.

Transaccions

Les transaccions asseguren que les dades siguin consistents i gestionen actualitzacions concurrents.

Exemple: Incrementar un Comptador

const incrementCounter = (counterId) => {
  const counterRef = firebase.database().ref('counters/' + counterId);
  counterRef.transaction((currentValue) => {
    return (currentValue || 0) + 1;
  });
};

// Ús
incrementCounter('pageViews');
  • Explicació: El mètode transaction assegura que el comptador s'incrementi atòmicament, fins i tot si múltiples clients l'estan actualitzant simultàniament.

Capacitats Fora de Línia

Firebase Realtime Database suporta la persistència de dades fora de línia, permetent que la teva aplicació romangui responsiva fins i tot quan l'usuari està fora de línia.

Activant la Persistència Fora de Línia

firebase.database().ref().keepSynced(true);
  • Explicació: El mètode keepSynced assegura que la referència especificada es mantingui sincronitzada fins i tot quan l'aplicació està fora de línia.

Conclusió

En aquesta secció, hem cobert les operacions essencials CRUD a Firebase Realtime Database, des de la creació bàsica de dades fins a la gestió avançada de transaccions. En dominar aquestes operacions, pots construir aplicacions robustes i en temps real que proporcionen una experiència d'usuari fluida.
