Introducció

Ada és un llenguatge de programació d'alt nivell estructurat, tipat estàticament, imperatiu i orientat a objectes. Està dissenyat per a aplicacions grans i de llarga durada on la fiabilitat i l'eficiència són essencials. Aquesta secció cobrirà la sintaxi i l'estructura bàsiques dels programes Ada, proporcionant una base per a un aprenentatge posterior.

Conceptes Clau

  • Estructura del Programa: Comprendre la disposició bàsica d'un programa Ada.
  • Comentaris: Com escriure comentaris en Ada.
  • Identificadors: Regles per a nomenar variables, constants i altres entitats.
  • Tipus de Dades: Tipus de dades bàsics disponibles en Ada.
  • Estructures de Control: Mecanismes bàsics de flux de control com bucles i condicionals.

Estructura del Programa

Un programa Ada consisteix en un o més paquets, procediments i funcions. L'estructura bàsica d'un programa Ada és la següent:

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;
  • with Ada.Text_IO; use Ada.Text_IO;: Aquestes línies importen el paquet Ada.Text_IO, que proporciona funcionalitats bàsiques d'entrada/sortida.
  • procedure Hello is: Aquesta línia declara un procediment anomenat Hello.
  • begin ... end Hello;: Aquest bloc conté les instruccions executables del procediment.

Comentaris

Els comentaris en Ada s'escriuen utilitzant -- per a comentaris d'una sola línia. No hi ha delimitadors de comentaris de múltiples línies en Ada.

-- Aquest és un comentari d'una sola línia
Put_Line("Hello, World!"); -- Això també és un comentari

Identificadors

Els identificadors en Ada són noms donats a diversos elements del programa com variables, constants, procediments, etc. Les regles per a nomenar identificadors són:

  • Han de començar amb una lletra.
  • Poden contenir lletres, dígits i guions baixos.
  • No distingeixen entre majúscules i minúscules.
My_Variable : Integer := 10;
anotherVariable : Integer := 20;

Tipus de Dades

Ada admet diversos tipus de dades bàsics, incloent:

  • Integer: Nombres enters.
  • Float: Nombres de punt flotant.
  • Boolean: Valors True o False.
  • Character: Caràcters individuals.
  • String: Seqüències de caràcters.

Exemple:

My_Integer : Integer := 42;
My_Float : Float := 3.14;
My_Boolean : Boolean := True;
My_Character : Character := 'A';
My_String : String := "Hello, Ada!";

Estructures de Control

Ada proporciona estructures de control estàndard per a la presa de decisions i la iteració.

Sentències If

if My_Integer > 10 then
   Put_Line("Greater than 10");
elsif My_Integer = 10 then
   Put_Line("Equal to 10");
else
   Put_Line("Less than 10");
end if;

Sentències Case

case My_Integer is
   when 1 => Put_Line("One");
   when 2 => Put_Line("Two");
   when others => Put_Line("Other");
end case;

Bucles

Bucle While

while My_Integer < 50 loop
   My_Integer := My_Integer + 1;
end loop;

Bucle For

for I in 1 .. 10 loop
   Put_Line(Integer'Image(I));
end loop;

Resum

En aquesta secció, hem cobert la sintaxi i l'estructura bàsiques dels programes Ada. Hem discutit la disposició general del programa, com escriure comentaris, regles per a nomenar identificadors, tipus de dades bàsics i estructures de control com sentències if, sentències case i bucles. Comprendre aquests fonaments és crucial per escriure i comprendre programes Ada més complexos.
