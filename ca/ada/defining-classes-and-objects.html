Introducció

En aquesta secció, explorarem com definir classes i objectes en Ada. Ada és un llenguatge de programació d'alt nivell, amb tipatge estàtic, que suporta la programació orientada a objectes (OOP). Entendre com definir i utilitzar classes i objectes és fonamental per aprofitar tot el potencial d'Ada.

Conceptes Clau

  • Classe: En Ada, una classe es defineix utilitzant tipus etiquetats. Els tipus etiquetats permeten la herència i el polimorfisme.
  • Objecte: Una instància d'una classe (tipus etiquetat) s'anomena objecte.
  • Encapsulació: L'encapsulació s'aconsegueix definint components privats dins d'un paquet.
  • Herència: L'herència es suporta mitjançant l'extensió de tipus.
  • Polimorfisme: El polimorfisme s'aconsegueix utilitzant tipus de classe ampla i operacions de despatx.

Definició d'una Classe (Tipus Etiquetat)

En Ada, una classe es defineix utilitzant un tipus etiquetat. Aquí hi ha un exemple senzill:

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move (S : in out Shape; New_X, New_Y : Integer);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move (S : in out Shape; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;
end Shape_Pkg;

Explicació

  • Paquet: El paquet Shape_Pkg encapsula la definició del tipus Shape i les seves operacions.
  • Tipus Etiquetat: Shape és un tipus etiquetat, el que significa que es pot estendre.
  • Procediment: El procediment Move modifica les coordenades X i Y d'un objecte Shape.

Creació i Ús d'Objectes

Per crear i utilitzar objectes de la classe Shape, pots fer el següent:

with Shape_Pkg;
use Shape_Pkg;

procedure Main is
    My_Shape : Shape := (X => 0, Y => 0);
begin
    Move(My_Shape, 10, 20);
    -- My_Shape ara té coordenades (10, 20)
end Main;

Explicació

  • Creació d'Objectes: My_Shape és un objecte de tipus Shape inicialitzat amb coordenades (0, 0).
  • Ús de Procediments: Es crida el procediment Move per canviar les coordenades de My_Shape.

Herència i Extensió de Tipus

Ada suporta l'herència mitjançant l'extensió de tipus. Aquí hi ha un exemple d'extensió del tipus Shape:

package Circle_Pkg is
    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Resize (C : in out Circle; New_Radius : Integer);
end Circle_Pkg;

package body Circle_Pkg is
    procedure Resize (C : in out Circle; New_Radius : Integer) is
    begin
        C.Radius := New_Radius;
    end Resize;
end Circle_Pkg;

Explicació

  • Extensió de Tipus: Circle és un nou tipus que estèn Shape afegint un camp Radius.
  • Procediment: El procediment Resize modifica el Radius d'un objecte Circle.

Polimorfisme

El polimorfisme en Ada s'aconsegueix utilitzant tipus de classe ampla i operacions de despatx. Aquí hi ha un exemple:

procedure Draw (S : in out Shape'Class) is
begin
    -- Operació de despatx basada en el tipus real de S
end Draw;

procedure Main is
    My_Circle : Circle := (X => 0, Y => 0, Radius => 5);
    My_Shape : Shape'Class := My_Circle;
begin
    Draw(My_Shape);
    -- El procediment Draw despatxarà a l'operació apropiada per Circle
end Main;

Explicació

  • Tipus de Classe Ampla: Shape'Class és un tipus de classe ampla que pot contenir qualsevol tipus derivat de Shape.
  • Operació de Despatx: El procediment Draw cridarà l'operació apropiada basada en el tipus real de S.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la definició de classes i objectes en Ada. Hem explorat:

  • Com definir una classe utilitzant tipus etiquetats.
  • Com crear i utilitzar objectes.
  • Com estendre tipus per aconseguir herència.
  • Com utilitzar el polimorfisme amb tipus de classe ampla i operacions de despatx.

Entendre aquests conceptes és crucial per a una programació orientada a objectes efectiva en Ada. Practica definint les teves pròpies classes i objectes per consolidar la teva comprensió.
