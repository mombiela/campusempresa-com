Introducció als Bucles en Ada

Els bucles són estructures de control fonamentals que permeten executar un bloc de codi múltiples vegades. Ada proporciona diversos tipus de bucles per gestionar diferents escenaris. Entendre com utilitzar aquests bucles de manera efectiva és crucial per escriure codi eficient i llegible.

Tipus de Bucles en Ada

Bucle Bàsic

El bucle bàsic en Ada és un bucle infinit que continua fins que es termina explícitament utilitzant una instrucció exit.

procedure Basic_Loop is
begin
   loop
      -- El teu codi aquí
      exit when Some_Condition;
   end loop;
end Basic_Loop;
  • Concepte Clau: La instrucció exit when s'utilitza per sortir del bucle quan es compleix una condició especificada.

Bucle While

El bucle while continua executant-se mentre una condició especificada sigui certa.

procedure While_Loop is
   Counter : Integer := 0;
begin
   while Counter < 10 loop
      -- El teu codi aquí
      Counter := Counter + 1;
   end loop;
end While_Loop;
  • Concepte Clau: La condició es valora abans de cada iteració. Si la condició és falsa inicialment, el cos del bucle pot no executar-se en absolut.

Bucle For

El bucle for itera sobre un rang de valors. S'utilitza comunament per iterar un nombre conegut de vegades.

procedure For_Loop is
begin
   for I in 1 .. 10 loop
      -- El teu codi aquí
   end loop;
end For_Loop;
  • Concepte Clau: La variable del bucle I pren automàticament cada valor en el rang especificat.

Bucles Imbricats

Els bucles poden estar imbricats dins d'altres per gestionar escenaris més complexos.

procedure Nested_Loops is
begin
   for I in 1 .. 5 loop
      for J in 1 .. 5 loop
         -- El teu codi aquí
      end loop;
   end loop;
end Nested_Loops;
  • Concepte Clau: Cada bucle manté la seva pròpia variable de bucle i àmbit.

Conceptes Avançats de Bucles

Bucle amb Sortida

Utilitzar instruccions exit dins dels bucles permet fluxos de control més complexos.

procedure Loop_With_Exit is
begin
   for I in 1 .. 10 loop
      if I = 5 then
         exit;
      end if;
      -- El teu codi aquí
   end loop;
end Loop_With_Exit;
  • Concepte Clau: La instrucció exit es pot utilitzar per sortir del bucle basant-se en una condició.

Bucle amb Continuació

Ada no té una instrucció continue incorporada com alguns altres llenguatges, però es pot aconseguir un comportament similar utilitzant exit i goto.

procedure Loop_With_Continue is
begin
   for I in 1 .. 10 loop
      if I mod 2 = 0 then
         goto Continue_Loop;
      end if;
      -- El teu codi aquí
      <<Continue_Loop>>;
   end loop;
end Loop_With_Continue;
  • Concepte Clau: Utilitzar goto per saltar la resta del cos del bucle i continuar amb la següent iteració.

Exercicis

  1. Exercici de Bucle Bàsic: Escriu un bucle bàsic que imprimeixi números de l'1 al 10.
  2. Exercici de Bucle While: Crea un bucle while que imprimeixi números parells entre l'1 i el 20.
  3. Exercici de Bucle For: Escriu un bucle for que calculi el factorial d'un nombre donat.
  4. Exercici de Bucles Imbricats: Implementa bucles imbricats per imprimir una taula de multiplicar per als números de l'1 al 5.
  5. Exercici de Bucle Avançat: Escriu un bucle que ometi imprimir números divisibles per 3 i surti quan el número sigui més gran que 15.

Conclusió

Els bucles són constructes essencials en Ada que permeten realitzar tasques repetitives de manera eficient. Dominant els bucles bàsics, bucles while, bucles for i bucles imbricats, pots gestionar una àmplia gamma d'escenaris de programació. Conceptes avançats com utilitzar exit i simular continue proporcionen un control addicional sobre l'execució dels bucles. Practica aquests conceptes amb els exercicis proporcionats per consolidar la teva comprensió.
