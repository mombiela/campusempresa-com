Visió general

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Ada, un llenguatge de programació tipat estàticament, estructurat i imperatiu, suporta els principis de la POO com la encapsulació, la herència i el polimorfisme.

Conceptes clau de la POO

  • Encapsulació: Agrupació de dades i mètodes que operen sobre les dades dins d'una unitat, per exemple, una classe.
  • Herència: Mecanisme pel qual una classe pot heretar atributs i mètodes d'una altra classe.
  • Polimorfisme: Capacitat de processar objectes de manera diferent segons el seu tipus de dades o classe.

Encapsulació en Ada

L'encapsulació en Ada s'aconsegueix utilitzant paquets i tipus privats. Aquí teniu un exemple senzill:

-- Definir un paquet per a una Pila
package Stack_Pkg is
    type Stack is private;
    procedure Push(S : in out Stack; Element : Integer);
    procedure Pop(S : in out Stack; Element : out Integer);
    function Is_Empty(S : Stack) return Boolean;
private
    type Stack is record
        Elements : array(1..100) of Integer;
        Top : Integer := 0;
    end record;
end Stack_Pkg;

package body Stack_Pkg is
    procedure Push(S : in out Stack; Element : Integer) is
    begin
        S.Top := S.Top + 1;
        S.Elements(S.Top) := Element;
    end Push;

    procedure Pop(S : in out Stack; Element : out Integer) is
    begin
        Element := S.Elements(S.Top);
        S.Top := S.Top - 1;
    end Pop;

    function Is_Empty(S : Stack) return Boolean is
    begin
        return S.Top = 0;
    end Is_Empty;
end Stack_Pkg;

Explicació:

  • Especificació del paquet: Defineix la interfície de la pila.
  • Tipus privat: La implementació real de la pila està oculta per a l'usuari.
  • Cos del paquet: Implementa els mètodes definits en l'especificació del paquet.

Herència en Ada

L'herència permet definir un nou tipus basat en un tipus existent. Aquí teniu un exemple:

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Integer);

    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Resize(C : in out Circle; New_Radius : Integer);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move(S : in out Shape; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Resize(C : in out Circle; New_Radius : Integer) is
    begin
        C.Radius := New_Radius;
    end Resize;
end Shape_Pkg;

Explicació:

  • Registre etiquetat: Shape és un registre etiquetat, el que significa que es pot estendre.
  • Tipus derivat: Circle es deriva de Shape i afegeix un nou atribut Radius.
  • Sobreescriptura de mètodes: Mètodes com Move i Resize es poden definir tant per a Shape com per a Circle.

Polimorfisme en Ada

El polimorfisme en Ada s'aconsegueix mitjançant el despatx dinàmic. Aquí teniu un exemple:

with Ada.Text_IO; use Ada.Text_IO;

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Integer) is abstract;
    procedure Display(S : in Shape) is abstract;

    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Move(S : in out Circle; New_X, New_Y : Integer);
    procedure Display(S : in Circle);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move(S : in out Circle; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Display(S : in Circle) is
    begin
        Put_Line("Circle at (" & Integer'Image(S.X) & ", " & Integer'Image(S.Y) & ") with radius " & Integer'Image(S.Radius));
    end Display;
end Shape_Pkg;

with Shape_Pkg; use Shape_Pkg;

procedure Main is
    C : Circle := (X => 0, Y => 0, Radius => 10);
begin
    C.Move(5, 5);
    C.Display;
end Main;

Explicació:

  • Mètodes abstractes: Move i Display són mètodes abstractes en Shape.
  • Despatx dinàmic: Quan es criden Move i Display en un Circle, s'utilitzen les implementacions específiques per a Circle.

Conclusió

En aquest tema, hem introduït els conceptes fonamentals de la Programació Orientada a Objectes en Ada, incloent la encapsulació, la herència i el polimorfisme. Hem proporcionat exemples per il·lustrar com s'implementen aquests conceptes en Ada. Entendre aquests principis és crucial per dissenyar sistemes de programari robustos i mantenibles. A mesura que avancis, veuràs com s'apliquen aquests conceptes en escenaris més complexos.
