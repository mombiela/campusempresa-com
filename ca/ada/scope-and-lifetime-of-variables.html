Entendre l'àmbit i la durada de les variables és crucial per escriure programes eficients i sense errors en Ada. Aquest tema et guiarà a través dels conceptes des d'un nivell principiant fins a un nivell avançat.

Introducció a l'Àmbit i la Durada

  • Àmbit: La regió del programa on una variable és accessible.
  • Durada: La durada durant la qual una variable existeix en memòria durant l'execució del programa.

Àmbit de les Variables

Àmbit Local

Les variables declarades dins d'un bloc (per exemple, un procediment, funció o bucle) són locals a aquest bloc.

procedure LocalScopeExample is
begin
   declare
      X : Integer := 10;
   begin
      -- X és accessible aquí
      Put_Line("X = " & Integer'Image(X));
   end;
   -- X no és accessible aquí
end LocalScopeExample;
  • Concepte Clau: Les variables declarades dins d'un bloc no són accessibles fora d'aquest bloc.

Àmbit Global

Les variables declarades fora de qualsevol bloc són globals i accessibles des de qualsevol part del programa.

package GlobalScopeExample is
   X : Integer := 10;
end GlobalScopeExample;

with GlobalScopeExample;
procedure UseGlobalScope is
begin
   Put_Line("X = " & Integer'Image(GlobalScopeExample.X));
end UseGlobalScope;
  • Concepte Clau: Les variables globals poden ser accedides des de qualsevol part del programa que with el paquet.

Àmbits Imbricats

Ada suporta àmbits imbricats, on els blocs interns poden accedir a variables dels blocs externs.

procedure NestedScopeExample is
   X : Integer := 10;
begin
   declare
      Y : Integer := 20;
   begin
      -- Tant X com Y són accessibles aquí
      Put_Line("X = " & Integer'Image(X) & ", Y = " & Integer'Image(Y));
   end;
   -- Només X és accessible aquí
end NestedScopeExample;
  • Concepte Clau: Els blocs interns poden accedir a variables dels blocs externs, però no a la inversa.

Durada de les Variables

Durada Estàtica

Les variables globals tenen una durada estàtica, és a dir, existeixen durant tota la durada del programa.

package StaticLifetimeExample is
   X : Integer := 10;
end StaticLifetimeExample;

with StaticLifetimeExample;
procedure UseStaticLifetime is
begin
   Put_Line("X = " & Integer'Image(StaticLifetimeExample.X));
end UseStaticLifetime;
  • Concepte Clau: Les variables globals es creen quan el programa comença i es destrueixen quan el programa acaba.

Durada Dinàmica

Les variables locals tenen una durada dinàmica, és a dir, existeixen només durant l'execució del bloc en què es declaren.

procedure DynamicLifetimeExample is
begin
   declare
      X : Integer := 10;
   begin
      -- X existeix aquí
      Put_Line("X = " & Integer'Image(X));
   end;
   -- X ja no existeix aquí
end DynamicLifetimeExample;
  • Concepte Clau: Les variables locals es creen quan s'entra al bloc i es destrueixen quan se surt del bloc.

Conceptes Avançats

Tipus d'Accés i Assignació Dinàmica

En Ada, pots utilitzar tipus d'accés (punters) per a l'assignació dinàmica de memòria, la qual cosa permet que les variables tinguin una durada més enllà de l'àmbit en què es van crear.

procedure AccessTypeExample is
   type Int_Ptr is access all Integer;
   Ptr : Int_Ptr;
begin
   Ptr := new Integer'(10);
   Put_Line("Value = " & Integer'Image(Ptr.all));
   -- La memòria s'ha de desassignar manualment
   Free(Ptr);
end AccessTypeExample;
  • Concepte Clau: Les variables assignades al heap utilitzant tipus d'accés existeixen fins que es desassignen explícitament.

Tipus Controlats

Els tipus controlats en Ada et permeten definir comportaments personalitzats d'inicialització, finalització i assignació, que poden ser útils per gestionar recursos.

with Ada.Finalization; use Ada.Finalization;

procedure ControlledTypeExample is
   type My_Controlled is new Controlled with record
      X : Integer := 10;
   end record;

   procedure Initialize(Object : in out My_Controlled) is
   begin
      Put_Line("Initializing X to " & Integer'Image(Object.X));
   end Initialize;

   procedure Finalize(Object : in out My_Controlled) is
   begin
      Put_Line("Finalizing X with value " & Integer'Image(Object.X));
   end Finalize;

   Obj : My_Controlled;
begin
   -- Obj s'inicialitza i finalitza automàticament
end ControlledTypeExample;
  • Concepte Clau: Els tipus controlats proporcionen ganxos per a la inicialització i finalització personalitzades, ajudant a gestionar la durada de recursos complexos.

Resum

  • Àmbit determina on una variable pot ser accedida.
    • Àmbit local: Les variables són accessibles només dins del bloc en què es declaren.
    • Àmbit global: Les variables són accessibles a tot el programa.
    • Àmbit imbricat: Els blocs interns poden accedir a variables dels blocs externs.
  • Durada determina quant de temps una variable existeix en memòria.
    • Durada estàtica: Les variables globals existeixen durant tota la durada del programa.
    • Durada dinàmica: Les variables locals existeixen només durant l'execució del seu bloc.
    • Els tipus d'accés i els tipus controlats proporcionen mecanismes avançats per gestionar la durada de les variables.

Entendre aquests conceptes és essencial per escriure programes robustos i mantenibles en Ada.
