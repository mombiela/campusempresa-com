Interfícies amb altres llenguatges és una habilitat crucial per als programadors d'Ada, permetent-los aprofitar biblioteques i sistemes existents escrits en diferents llenguatges de programació. Aquest tema cobrirà els conceptes bàsics d'interfícies d'Ada amb C, ja que és l'escenari més comú, i proporcionarà informació sobre interfícies amb altres llenguatges.

Introducció a les Interfícies

Les interfícies permeten que els programes Ada cridin funcions i utilitzin biblioteques escrites en altres llenguatges, i viceversa. Això és particularment útil per a:

  • Reutilitzar codi existent.
  • Aprofitar biblioteques especialitzades.
  • Integrar-se amb APIs a nivell de sistema.

Interfícies amb C

C és el llenguatge més comú amb el qual Ada s'interfície degut al seu ús generalitzat i compatibilitat. Ada proporciona una manera senzilla d'interfície amb C a través de les directives pragma Import, pragma Export i pragma Convention.

Cridar Funcions C des d'Ada

Per cridar una funció C des d'Ada, cal:

  1. Declarar la funció C en Ada.
  2. Utilitzar pragma Import per enllaçar la funció C.

Exemple

Suposem que tens una funció C int add(int a, int b);:

Codi C:

// add.c
int add(int a, int b) {
    return a + b;
}

Codi Ada:

-- add.ads
with Interfaces.C; use Interfaces.C;

function Add (A, B : C.int) return C.int;
pragma Import (C, Add, "add");

-- main.adb
with Ada.Text_IO; use Ada.Text_IO;
with Add;

procedure Main is
begin
    Put_Line ("5 + 3 = " & Integer'Image (Add (5, 3)));
end Main;

Exportar Funcions Ada a C

Per cridar una funció Ada des de C, cal:

  1. Declarar la funció Ada.
  2. Utilitzar pragma Export per fer la funció disponible per a C.

Exemple

Suposem que vols cridar una funció Ada des de C:

Codi Ada:

-- my_functions.ads
with Interfaces.C; use Interfaces.C;

procedure Print_Message (Message : in C.char_array);
pragma Export (C, Print_Message, "print_message");

-- my_functions.adb
with Ada.Text_IO; use Ada.Text_IO;

procedure Print_Message (Message : in Interfaces.C.char_array) is
begin
    Ada.Text_IO.Put_Line (Message.all);
end Print_Message;

Codi C:

// main.c
extern void print_message(const char* message);

int main() {
    print_message("Hello from C!");
    return 0;
}

Mapeig de Tipus de Dades

Quan s'interfície amb C, és essencial mapar els tipus d'Ada als seus corresponents tipus C. Aquí hi ha una taula per a mapatges de tipus comuns:

| Tipus Ada | Tipus C | |--------------------|------------------| | Interfaces.C.int | int | | Interfaces.C.char| char | | Interfaces.C.float| float | | Interfaces.C.double| double | | Interfaces.C.char_array| char* |

Interfícies amb Altres Llenguatges

Tot i que C és el llenguatge més comú amb el qual Ada s'interfície, també és possible interfície amb altres llenguatges com Fortran, C++ i Java. El procés és similar però pot requerir consideracions addicionals per a característiques específiques del llenguatge.

Interfícies amb Fortran

Fortran s'utilitza sovint en computació científica. Ada pot interfície amb Fortran utilitzant pragmes similars:

procedure Fortran_Subroutine;
pragma Import (Fortran, Fortran_Subroutine, "fortran_subroutine");

Interfícies amb C++

Interfície amb C++ pot ser més complex degut a característiques com classes i mangling de noms. No obstant això, les crides bàsiques de funcions es poden manejar de manera similar a C:

procedure Cpp_Function;
pragma Import (C, Cpp_Function, "_Z12cpp_functionv"); -- Nom mangled

Interfícies amb Java

Interfície amb Java típicament implica utilitzar la Java Native Interface (JNI). Això és més avançat i requereix entendre les convencions de JNI.

Conclusió

Interfície Ada amb altres llenguatges, especialment C, és una característica poderosa que et permet estendre les capacitats dels teus programes Ada. Entenent com declarar i enllaçar funcions, mapar tipus de dades i utilitzar les pragmes apropiades, pots integrar efectivament Ada amb una varietat d'altres llenguatges de programació. Aquesta habilitat és inavaluable per aprofitar bases de codi i biblioteques existents, fent que les teves aplicacions Ada siguin més versàtils i poderoses.
