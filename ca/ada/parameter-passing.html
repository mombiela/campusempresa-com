Introducció

El passatge de paràmetres és un concepte fonamental en programació que et permet passar dades a subprogrames (procediments i funcions). En Ada, el passatge de paràmetres és versàtil i proporciona diversos modes per controlar com es passen i manipulen les dades.

Modes de Paràmetres

Ada suporta tres modes de paràmetres principals:

  • In: Passa dades al subprograma.
  • Out: Passa dades fora del subprograma.
  • In Out: Passa dades tant al subprograma com fora d'aquest.

Mode In

El mode in s'utilitza quan vols passar dades a un subprograma sense permetre que el subprograma modifiqui les dades originals.

procedure Print_Value(Value : in Integer) is
begin
   Put_Line("The value is: " & Integer'Image(Value));
end Print_Value;
  • Concepte Clau: El paràmetre in és de només lectura dins del subprograma.

Mode Out

El mode out s'utilitza quan vols que el subprograma modifiqui el paràmetre i retorni el resultat.

procedure Increment(Value : out Integer) is
begin
   Value := Value + 1;
end Increment;
  • Concepte Clau: El paràmetre out és de només escriptura dins del subprograma. S'utilitza per retornar un valor al cridant.

Mode In Out

El mode in out permet que el subprograma llegeixi i modifiqui el paràmetre.

procedure Double_Value(Value : in out Integer) is
begin
   Value := Value * 2;
end Double_Value;
  • Concepte Clau: El paràmetre in out es pot llegir i modificar dins del subprograma.

Mecanismes de Passatge de Paràmetres

Ada utilitza diferents mecanismes per implementar el passatge de paràmetres:

  • Per Còpia: El valor real es copia al subprograma.
  • Per Referència: Es passa una referència a les dades reals al subprograma.

Per Còpia

Normalment s'utilitza per a paràmetres in. El subprograma treballa amb una còpia del paràmetre real.

procedure Show_Copy(Value : in Integer) is
   Temp : Integer := Value;
begin
   Temp := Temp + 1;
   Put_Line("Temp value: " & Integer'Image(Temp));
   Put_Line("Original value: " & Integer'Image(Value));
end Show_Copy;

Per Referència

Normalment s'utilitza per a paràmetres out i in out. El subprograma treballa directament amb el paràmetre real.

procedure Modify_Reference(Value : in out Integer) is
begin
   Value := Value + 10;
end Modify_Reference;

Exemples i Exercicis

Exemple 1: Utilitzant el Mode `in`

procedure Display_Sum(A, B : in Integer) is
begin
   Put_Line("Sum: " & Integer'Image(A + B));
end Display_Sum;

-- Ús
begin
   Display_Sum(5, 10);  -- Sortida: Sum: 15
end;

Exemple 2: Utilitzant el Mode `out`

procedure Initialize(Value : out Integer) is
begin
   Value := 0;
end Initialize;

-- Ús
declare
   Num : Integer;
begin
   Initialize(Num);
   Put_Line("Initialized value: " & Integer'Image(Num));  -- Sortida: Initialized value: 0
end;

Exemple 3: Utilitzant el Mode `in out`

procedure Square(Value : in out Integer) is
begin
   Value := Value * Value;
end Square;

-- Ús
declare
   Num : Integer := 4;
begin
   Square(Num);
   Put_Line("Squared value: " & Integer'Image(Num));  -- Sortida: Squared value: 16
end;

Exercici 1

Escriu un procediment Swap que prengui dos paràmetres in out i intercanviï els seus valors.

procedure Swap(A, B : in out Integer) is
   Temp : Integer;
begin
   Temp := A;
   A := B;
   B := Temp;
end Swap;

-- Prova el teu procediment
declare
   X : Integer := 5;
   Y : Integer := 10;
begin
   Swap(X, Y);
   Put_Line("X: " & Integer'Image(X));  -- Sortida: X: 10
   Put_Line("Y: " & Integer'Image(Y));  -- Sortida: Y: 5
end;

Conclusió

Entendre el passatge de paràmetres en Ada és crucial per escriure subprogrames eficients i efectius. Dominant els modes in, out i in out, pots controlar com es passen i manipulen les dades dins dels teus programes. Practica amb els exemples i exercicis proporcionats per consolidar la teva comprensió d'aquests conceptes.
