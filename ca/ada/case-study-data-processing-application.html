En aquest estudi de cas, desenvoluparem una aplicació de processament de dades utilitzant Ada. Això t'ajudarà a entendre com aplicar les característiques d'Ada en un escenari del món real. Començarem amb una tasca simple de processament de dades i gradualment afegirem complexitat, cobrint tant temes per a principiants com avançats.

Introducció a l'Aplicació de Processament de Dades

Abans d'entrar en el codi, anem a esbossar els principals objectius de la nostra aplicació de processament de dades:

  • Llegir dades d'un fitxer.
  • Processar les dades (per exemple, filtrar, ordenar, agregar).
  • Sortir les dades processades a un fitxer.

Llegir Dades d'un Fitxer

Conceptes Clau

  • Gestió de Fitxers en Ada: Aprendre a obrir, llegir i tancar fitxers.
  • Estructures de Dades: Utilitzar arrays o registres per emmagatzemar les dades.

Exemple de Codi

Comencem amb un exemple simple de lectura de dades d'un fitxer. Suposem que tenim un fitxer data.txt amb el següent contingut:

10
20
30
40
50

Aquí tens com pots llegir aquestes dades en un array en Ada:

with Ada.Text_IO; use Ada.Text_IO;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Read_Data is
   type Data_Array is array (1 .. 5) of Integer;
   Data : Data_Array;
   File : File_Type;
   Line : Integer;
begin
   Open (File, In_File, "data.txt");
   for I in Data'Range loop
      Get (File, Line);
      Data(I) := Line;
   end loop;
   Close (File);

   -- Imprimir les dades per verificar
   for I in Data'Range loop
      Put_Line (Integer'Image(Data(I)));
   end loop;
end Read_Data;

Explicació

  • Gestió de Fitxers: Obrim el fitxer utilitzant Open, llegim cada línia utilitzant Get, i tanquem el fitxer utilitzant Close.
  • Emmagatzematge de Dades: Utilitzem un array Data per emmagatzemar els enters llegits del fitxer.

Processar les Dades

Conceptes Clau

  • Bucles i Condicionals: Utilitzar bucles i condicionals per processar dades.
  • Disseny Modular: Descompondre el processament en funcions o procediments.

Exemple de Codi

Afegim un pas simple de processament de dades: filtrar els números menors de 30.

with Ada.Text_IO; use Ada.Text_IO;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Process_Data is
   type Data_Array is array (1 .. 5) of Integer;
   Data : Data_Array := (10, 20, 30, 40, 50);
   Filtered_Data : Data_Array := (others => 0);
   Index : Integer := 1;
begin
   for I in Data'Range loop
      if Data(I) >= 30 then
         Filtered_Data(Index) := Data(I);
         Index := Index + 1;
      end if;
   end loop;

   -- Imprimir les dades filtrades per verificar
   for I in Filtered_Data'Range loop
      if Filtered_Data(I) /= 0 then
         Put_Line (Integer'Image(Filtered_Data(I)));
      end if;
   end loop;
end Process_Data;

Explicació

  • Filtratge: Utilitzem un bucle i una instrucció if per filtrar els números menors de 30.
  • Emmagatzematge: Les dades filtrades s'emmagatzemen en un nou array Filtered_Data.

Sortir les Dades Processades a un Fitxer

Conceptes Clau

  • Escriptura de Fitxers en Ada: Aprendre a obrir, escriure i tancar fitxers.
  • Gestió d'Errors: Assegurar una correcta gestió d'errors quan es treballa amb fitxers.

Exemple de Codi

Ara, escrivim les dades filtrades a un nou fitxer filtered_data.txt.

with Ada.Text_IO; use Ada.Text_IO;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Write_Data is
   type Data_Array is array (1 .. 5) of Integer;
   Filtered_Data : Data_Array := (30, 40, 50, 0, 0);
   File : File_Type;
begin
   Create (File, Out_File, "filtered_data.txt");
   for I in Filtered_Data'Range loop
      if Filtered_Data(I) /= 0 then
         Put (File, Filtered_Data(I));
         New_Line (File);
      end if;
   end loop;
   Close (File);
end Write_Data;

Explicació

  • Gestió de Fitxers: Utilitzem Create per obrir el fitxer per escriure, Put per escriure dades, i Close per tancar el fitxer.
  • Escriptura Condicional: Només escrivim valors diferents de zero al fitxer.

Conclusió

En aquest estudi de cas, hem desenvolupat una simple aplicació de processament de dades en Ada. Hem cobert:

  • Llegir dades d'un fitxer.
  • Processar les dades filtrant.
  • Escriure les dades processades a un fitxer.

Aquest exemple demostra com gestionar fitxers, utilitzar arrays i implementar un processament bàsic de dades en Ada. A mesura que avancis, pots afegir passos de processament més complexos, gestió d'errors i optimitzacions per fer la teva aplicació més robusta i eficient.
