Introducció a Ada

Aquesta secció introdueix el llenguatge de programació Ada, la seva història i els seus usos principals. Estableix les bases per entendre per què Ada és un llenguatge potent per a certes aplicacions.

  • Història i Antecedents
    • Origen d'Ada
    • Característiques clau i objectius de disseny
  • Instal·lació i Configuració
    • Instal·lant GNAT (GNU NYU Ada Translator)
    • Configurant l'entorn de desenvolupament

Exemple: Hola Món en Ada

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;

Explicació:

  • with Ada.Text_IO; use Ada.Text_IO;: Importar el paquet Ada Text_IO per a operacions d'entrada/sortida.
  • procedure Hello is: Declarant un procediment anomenat Hello.
  • begin ... end Hello;: El cos del procediment, on Put_Line imprimeix "Hello, World!" a la consola.

Sintaxi i Construccions Bàsiques

Aquesta secció cobreix la sintaxi i les construccions fonamentals d'Ada, essencials per escriure programes bàsics.

  • Tipus de Dades i Variables
    • Tipus escalars: Integer, Float, Boolean, Character
    • Tipus compostos: Arrays, Records
  • Estructures de Control
    • Declaracions condicionals: if, case
    • Bucles: for, while
  • Subprogrames
    • Procediments
    • Funcions

Exemple: Declaració Condicional

procedure Check_Number is
   Number : Integer := 10;
begin
   if Number > 0 then
      Put_Line("Number is positive");
   elsif Number < 0 then
      Put_Line("Number is negative");
   else
      Put_Line("Number is zero");
   end if;
end Check_Number;

Explicació:

  • if Number > 0 then ... end if;: Comprova si Number és positiu, negatiu o zero i imprimeix el missatge corresponent.

Conceptes Intermedis

Basat en els conceptes bàsics, aquesta secció aprofundeix en temes més complexos com la programació modular i la gestió d'excepcions.

  • Paquets i Programació Modular
    • Creant i utilitzant paquets
    • Encapsulació i ocultació d'informació
  • Gestió d'Excepcions
    • Declarant i llançant excepcions
    • Gestionant excepcions amb begin ... exception ... end

Exemple: Gestió d'Excepcions

procedure Divide is
   A, B, Result : Integer;
begin
   A := 10;
   B := 0;
   begin
      Result := A / B;
   exception
      when Constraint_Error =>
         Put_Line("Division by zero is not allowed");
   end;
end Divide;

Explicació:

  • begin ... exception ... end;: Intenta dividir A per B i captura un Constraint_Error si B és zero.

Temes Avançats

Aquesta secció cobreix característiques avançades d'Ada com la tasca, els genèrics i la interfície amb altres llenguatges.

  • Tasques i Concurrència
    • Tipus de tasques i objectes de tasques
    • Sincronització amb objectes protegits
  • Genèrics
    • Definint i instanciant paquets i subprogrames genèrics
  • Interfície amb Altres Llenguatges
    • Interfície amb C i altres llenguatges

Exemple: Tasca

task type Worker is
end Worker;

task body Worker is
begin
   Put_Line("Worker task is running");
end Worker;

procedure Main is
   W : Worker;
begin
   null;
end Main;

Explicació:

  • task type Worker is ... end Worker;: Declara un tipus de tasca anomenat Worker.
  • task body Worker is ... end Worker;: Defineix el comportament de la tasca Worker.
  • W : Worker;: Instancia una tasca Worker en el procediment Main.

Conclusió

Aquest curs ha proporcionat un viatge complet a través d'Ada, començant pels conceptes bàsics i avançant cap a temes complexos. En dominar aquests conceptes, els estudiants estan ben preparats per afrontar problemes del món real utilitzant Ada. Les habilitats adquirides aquí seran valuoses per desenvolupar sistemes de programari fiables, mantenibles i eficients.
