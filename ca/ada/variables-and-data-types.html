Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en Ada. Entendre aquests conceptes és crucial per escriure programes Ada efectius i eficients. Cobrirem els següents temes:

  • Què són les variables?
  • Declarar variables en Ada
  • Tipus de dades bàsics en Ada
  • Conversió de tipus
  • Tipus de dades avançats

Què són les Variables?

Les variables són ubicacions d'emmagatzematge amb nom a la memòria que contenen dades que es poden modificar durant l'execució del programa. Són essencials per emmagatzemar i manipular informació en un programa.

Declarar Variables en Ada

En Ada, les variables s'han de declarar abans de poder ser utilitzades. Una declaració de variable especifica el nom de la variable i el seu tipus de dada.

Sintaxi

variable_name : data_type;

Exemple

-- Declarar una variable enter
age : Integer;

-- Declarar una variable de punt flotant
temperature : Float;

-- Declarar una variable de cadena
name : String(1..50);

Tipus de Dades Bàsics en Ada

Ada proporciona diversos tipus de dades bàsics que s'utilitzen per definir variables. Aquests inclouen:

  • Integer
  • Float
  • Boolean
  • Character
  • String

Integer

Els enters són nombres sencers sense part fraccionària.

-- Declarar i inicialitzar una variable enter
count : Integer := 10;

Float

Els nombres de punt flotant s'utilitzen per representar nombres reals amb una part fraccionària.

-- Declarar i inicialitzar una variable de punt flotant
pi : Float := 3.14159;

Boolean

Els tipus de dades booleans s'utilitzen per representar valors de veritat: True o False.

-- Declarar i inicialitzar una variable booleana
is_valid : Boolean := True;

Character

Els tipus de dades de caràcter s'utilitzen per representar caràcters individuals.

-- Declarar i inicialitzar una variable de caràcter
initial : Character := 'A';

String

Les cadenes són seqüències de caràcters.

-- Declarar i inicialitzar una variable de cadena
greeting : String(1..5) := "Hello";

Conversió de Tipus

La conversió de tipus és el procés de convertir un valor d'un tipus de dada a un altre. Ada proporciona funcions integrades per a la conversió de tipus.

Exemple

-- Convertir un enter a un punt flotant
integer_value : Integer := 42;
float_value : Float := Float(integer_value);

-- Convertir un punt flotant a un enter
float_value : Float := 3.99;
integer_value : Integer := Integer(float_value);

Tipus de Dades Avançats

Ada també suporta tipus de dades més complexos com ara arrays, registres i enumeracions.

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors del mateix tipus.

-- Declarar i inicialitzar un array d'enters
numbers : array(1..5) of Integer := (1, 2, 3, 4, 5);

Registres

Els registres s'utilitzen per agrupar dades relacionades.

-- Declarar un tipus de registre
type Person is record
    Name : String(1..50);
    Age  : Integer;
end record;

-- Declarar i inicialitzar una variable de registre
john : Person := (Name => "John Doe", Age => 30);

Enumeracions

Les enumeracions s'utilitzen per definir un conjunt de valors amb nom.

-- Declarar un tipus d'enumeració
type Day is (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday);

-- Declarar i inicialitzar una variable d'enumeració
today : Day := Friday;

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en Ada. Hem après com declarar variables, els tipus de dades bàsics disponibles en Ada i com realitzar conversions de tipus. També hem tocat tipus de dades avançats com ara arrays, registres i enumeracions. Entendre aquests conceptes és fonamental per escriure programes Ada efectius.
