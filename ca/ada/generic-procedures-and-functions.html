Introducció

A Ada, els procediments i funcions genèriques et permeten escriure codi flexible i reutilitzable. Els genèrics et permeten definir algorismes i estructures de dades que poden operar amb qualsevol tipus de dada. Aquesta secció et guiarà a través dels conceptes bàsics dels genèrics a Ada, des de la definició de procediments i funcions genèriques fins a la seva utilització en els teus programes.

Conceptes Clau

  • Genèrics: Plantilles que et permeten escriure procediments i funcions que poden operar sobre diferents tipus de dades.
  • Paràmetres Genèrics: Marcadors de posició per a tipus, subprogrames o valors que es especifiquen quan la unitat genèrica és instanciada.
  • Instanciació: El procés de crear una instància específica d'una unitat genèrica proporcionant paràmetres reals.

Definició de Procediments i Funcions Genèriques

Per definir un procediment o funció genèrica, utilitzes la paraula clau generic seguida de la llista de paràmetres genèrics. Aquí tens la sintaxi bàsica:

generic
   -- Llista de paràmetres genèrics
procedure Generic_Procedure (Parameter : in Type);

Exemple: Procediment Genèric de Canvi

Comencem amb un exemple senzill: un procediment genèric per canviar dos valors de qualsevol tipus.

generic
   type Element_Type is private;
procedure Swap (X, Y : in out Element_Type);

procedure Swap (X, Y : in out Element_Type) is
   Temp : Element_Type;
begin
   Temp := X;
   X := Y;
   Y := Temp;
end Swap;

En aquest exemple:

  • Element_Type és un paràmetre genèric que representa qualsevol tipus.
  • Swap és un procediment que pren dos paràmetres de tipus Element_Type i canvia els seus valors.

Instanciació del Procediment Genèric

Per utilitzar el procediment genèric Swap, necessites instanciar-lo amb un tipus específic.

procedure Swap_Integer is new Swap (Integer);

procedure Test_Swap is
   A, B : Integer := 1;
begin
   Swap_Integer (A, B);
   -- Ara A = 2 i B = 1
end Test_Swap;

Funcions Genèriques

Les funcions genèriques es defineixen de manera similar als procediments genèrics. Aquí tens la sintaxi bàsica:

generic
   -- Llista de paràmetres genèrics
function Generic_Function (Parameter : in Type) return Return_Type;

Exemple: Funció Genèrica Max

Creem una funció genèrica per trobar el màxim de dos valors.

generic
   type Element_Type is private;
   with function "<" (Left, Right : Element_Type) return Boolean;
function Max (X, Y : Element_Type) return Element_Type;

function Max (X, Y : Element_Type) return Element_Type is
begin
   if X < Y then
      return Y;
   else
      return X;
   end if;
end Max;

En aquest exemple:

  • Element_Type és un paràmetre genèric que representa qualsevol tipus.
  • La funció "<" és un paràmetre genèric que representa una funció de comparació per a Element_Type.
  • Max és una funció que retorna el màxim de dos valors de tipus Element_Type.

Instanciació de la Funció Genèrica

Per utilitzar la funció genèrica Max, necessites instanciar-la amb un tipus específic i una funció de comparació.

function Integer_Less (Left, Right : Integer) return Boolean is
begin
   return Left < Right;
end Integer_Less;

function Max_Integer is new Max (Integer, Integer_Less);

procedure Test_Max is
   A, B : Integer := 1;
   C : Integer;
begin
   C := Max_Integer (A, B);
   -- Ara C = 2
end Test_Max;

Resum

Els procediments i funcions genèriques a Ada proporcionen una manera poderosa d'escriure codi reutilitzable i segur per tipus. Definint unitats genèriques, pots crear algorismes i estructures de dades que funcionin amb qualsevol tipus de dada, millorant la flexibilitat i mantenibilitat dels teus programes. Recorda:

  • Definir paràmetres genèrics utilitzant la paraula clau generic.
  • Instanciar unitats genèriques amb tipus i subprogrames específics.
  • Utilitzar genèrics per crear components de codi flexibles i reutilitzables.

Amb aquestes eines, pots escriure programes Ada més generals i adaptables, fent que el teu codi sigui més robust i fàcil de mantenir.
