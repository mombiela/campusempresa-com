Introducció als Tipus d'Accés

Els tipus d'accés en Ada, comunament coneguts com a punteres en altres llenguatges de programació, s'utilitzen per referenciar objectes dinàmicament allotjats. Proporcionen una manera de crear i manipular estructures de dades com llistes enllaçades, arbres i gràfics.

Conceptes Clau

  • Declaració de Tipus d'Accés: Defineix un tipus que pot contenir referències a objectes d'un tipus especificat.
  • Allotjament Dinàmic: Allotja memòria en temps d'execució utilitzant la paraula clau new.
  • Desreferenciació: Accedeix al valor emmagatzemat a la ubicació apuntada per un tipus d'accés.
  • Accés Null: Representa un tipus d'accés que no apunta a cap objecte.

Declaració de Tipus d'Accés

Per declarar un tipus d'accés, utilitzeu la paraula clau access seguida del tipus d'objecte al qual apuntarà.

type Int_Ptr is access Integer;

En aquest exemple, Int_Ptr és un tipus d'accés que pot apuntar a un Integer.

Allotjament Dinàmic

L'allotjament dinàmic en Ada es realitza utilitzant la paraula clau new. Això allibera memòria per a un objecte i retorna un valor d'accés que hi apunta.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   -- Ptr ara apunta a un Integer amb valor 10
end;

Desreferenciació de Tipus d'Accés

Per accedir al valor emmagatzemat a la ubicació apuntada per un tipus d'accés, utilitzeu la notació .all.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   Put_Line(Integer'Image(Ptr.all)); -- Mostra: 10
   Ptr.all := 20;
   Put_Line(Integer'Image(Ptr.all)); -- Mostra: 20
end;

Accés Null

Un tipus d'accés es pot establir explícitament a null, indicant que no apunta a cap objecte.

declare
   Ptr : Int_Ptr := null;
begin
   if Ptr = null then
      Put_Line("El punter és null");
   end if;
end;

Exemple: Llista Enllaçada

Creem una llista enllaçada simple utilitzant tipus d'accés.

with Ada.Text_IO; use Ada.Text_IO;

procedure Linked_List_Example is
   type Node;
   type Node_Ptr is access Node;
   
   type Node is
      record
         Value : Integer;
         Next  : Node_Ptr;
      end record;
   
   Head : Node_Ptr := null;
   
   procedure Add_Node(Value : Integer) is
      New_Node : Node_Ptr := new Node'(Value, null);
   begin
      if Head = null then
         Head := New_Node;
      else
         New_Node.Next := Head;
         Head := New_Node;
      end if;
   end Add_Node;
   
   procedure Print_List is
      Current : Node_Ptr := Head;
   begin
      while Current /= null loop
         Put_Line(Integer'Image(Current.Value));
         Current := Current.Next;
      end loop;
   end Print_List;
   
begin
   Add_Node(10);
   Add_Node(20);
   Add_Node(30);
   
   Print_List; -- Mostra: 30 20 10
end Linked_List_Example;

Resum

Els tipus d'accés en Ada són eines poderoses per gestionar memòria dinàmica i crear estructures de dades complexes. Punts clau a recordar:

  • Els tipus d'accés es declaren utilitzant la paraula clau access.
  • L'allotjament de memòria dinàmica es fa utilitzant la paraula clau new.
  • La desreferenciació es realitza utilitzant la notació .all.
  • Els tipus d'accés es poden establir a null per indicar que no apunten a cap objecte.

Entendre i utilitzar els tipus d'accés de manera efectiva us permetrà construir programes eficients i flexibles en Ada.
