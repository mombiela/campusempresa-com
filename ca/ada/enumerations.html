Les enumeracions en Ada són un tipus de dada fonamental que et permet definir un conjunt de valors amb nom. Aquests valors s'anomenen literals d'enumeració i són útils per representar una col·lecció de constants relacionades d'una manera llegible i fàcil de mantenir.

Conceptes Clau

  • Tipus d'Enumeració: Un tipus definit per l'usuari que consisteix en un conjunt de valors amb nom.
  • Literals d'Enumeració: Els valors amb nom que pertanyen a un tipus d'enumeració.
  • Seguretat de Tipus: Les enumeracions en Ada proporcionen seguretat de tipus assegurant que només es poden assignar valors vàlids a variables del tipus d'enumeració.

Definició d'Enumeracions

Per definir un tipus d'enumeració en Ada, utilitzes la paraula clau type seguida del nom del tipus i la llista de literals d'enumeració entre parèntesis.

Exemple

type Day is (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday);

En aquest exemple, Day és un tipus d'enumeració amb set literals: Monday, Tuesday, Wednesday, Thursday, Friday, Saturday i Sunday.

Ús de les Enumeracions

Un cop has definit un tipus d'enumeració, pots declarar variables d'aquest tipus i assignar-los literals d'enumeració.

Exemple

procedure Enumerations_Example is
   type Day is (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday);
   Today : Day;
begin
   Today := Monday;
   if Today = Monday then
      Put_Line("It's Monday!");
   end if;
end Enumerations_Example;

En aquest exemple, la variable Today és del tipus Day, i se li assigna el valor Monday. El programa després comprova si Today és Monday i imprimeix un missatge en conseqüència.

Atributs d'Enumeració

Ada proporciona diversos atributs per als tipus d'enumeració que poden ser útils en diverses situacions.

  • 'Pos: Retorna el número de posició d'un literal d'enumeració.
  • 'Val: Converteix un número de posició en el literal d'enumeració corresponent.
  • 'Succ: Retorna el successor d'un literal d'enumeració.
  • 'Pred: Retorna el predecessor d'un literal d'enumeració.

Exemple

procedure Enumeration_Attributes is
   type Day is (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday);
   Today : Day := Monday;
begin
   Put_Line("Position of Monday: " & Integer'Image(Day'Pos(Today)));
   Put_Line("Literal at position 2: " & Day'Image(Day'Val(2)));
   Put_Line("Successor of Monday: " & Day'Image(Day'Succ(Today)));
   Put_Line("Predecessor of Wednesday: " & Day'Image(Day'Pred(Wednesday)));
end Enumeration_Attributes;

En aquest exemple, es fan servir diversos atributs per obtenir informació sobre els literals d'enumeració.

Ús Avançat

Clàusules de Representació d'Enumeració

Pots especificar la representació interna dels literals d'enumeració utilitzant clàusules de representació. Això pot ser útil per interfícies amb maquinari o altres llenguatges de programació.

Exemple

type Color is (Red, Green, Blue);
for Color use (Red => 1, Green => 2, Blue => 3);

En aquest exemple, els literals d'enumeració Red, Green i Blue se'ls assignen els valors interns 1, 2 i 3, respectivament.

Enumeració i Sentències Case

Les enumeracions sovint es fan servir amb sentències case per gestionar diferents valors d'una manera estructurada.

Exemple

procedure Case_Statement_Example is
   type Traffic_Light is (Red, Yellow, Green);
   Signal : Traffic_Light := Red;
begin
   case Signal is
      when Red =>
         Put_Line("Stop!");
      when Yellow =>
         Put_Line("Caution!");
      when Green =>
         Put_Line("Go!");
   end case;
end Case_Statement_Example;

En aquest exemple, es fa servir una sentència case per imprimir diferents missatges segons el valor de la variable Signal.

Conclusió

Les enumeracions en Ada proporcionen una manera poderosa de definir i treballar amb un conjunt de constants relacionades. Milloren la llegibilitat i mantenibilitat del codi mentre asseguren la seguretat de tipus. Entenent com definir, utilitzar i manipular enumeracions, pots escriure programes en Ada més robustos i clars.
