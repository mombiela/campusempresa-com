Introducció als Paquets Genèrics

Els paquets genèrics en Ada permeten escriure codi flexible i reutilitzable mitjançant la parametrització de paquets amb tipus, objectes o subprogrames. Aquesta característica és particularment útil per crear estructures de dades i algorismes que poden operar sobre diferents tipus sense reescriure el codi.

Conceptes Clau

  • Genèrics: Un mecanisme per crear plantilles per a paquets, subprogrames o tipus.
  • Paràmetres Genèrics: Tipus, objectes o subprogrames que es especifiquen quan el paquet genèric és instanciat.
  • Instanciació: El procés de crear una instància específica d'un paquet genèric amb paràmetres concrets.

Definició d'un Paquet Genèric

Per definir un paquet genèric, s'utilitza la paraula clau generic seguida dels paràmetres formals. Aquí hi ha un exemple senzill:

generic
   type Element_Type is private;
package Generic_Stack is
   procedure Push(Item : in Element_Type);
   procedure Pop(Item : out Element_Type);
   function Is_Empty return Boolean;
end Generic_Stack;

Explicació

  • generic: Indica l'inici d'una definició de paquet genèric.
  • type Element_Type is private;: Declara un paràmetre formal de tipus anomenat Element_Type que és privat, el que significa que pot ser qualsevol tipus.
  • Generic_Stack: El nom del paquet genèric.
  • Push, Pop, Is_Empty: Subprogrames que operen sobre la pila.

Instanciació d'un Paquet Genèric

Per utilitzar un paquet genèric, cal instanciar-lo amb tipus o valors específics. Aquí es mostra com es pot instanciar el paquet Generic_Stack per a un tipus Integer:

package Integer_Stack is new Generic_Stack(Element_Type => Integer);

Explicació

  • Integer_Stack: El nom de la nova instància del paquet.
  • new Generic_Stack(Element_Type => Integer);: Instancia el paquet Generic_Stack amb Element_Type establert a Integer.

Ús del Paquet Instanciat

Un cop instanciat, es pot utilitzar el paquet com qualsevol altre paquet. Aquí hi ha un exemple de com utilitzar el paquet Integer_Stack:

procedure Use_Stack is
   package Integer_Stack is new Generic_Stack(Element_Type => Integer);
   My_Stack : Integer_Stack;
   Item : Integer;
begin
   Integer_Stack.Push(My_Stack, 10);
   Integer_Stack.Push(My_Stack, 20);
   Integer_Stack.Pop(My_Stack, Item);
   Ada.Text_IO.Put_Line("Popped item: " & Integer'Image(Item));
end Use_Stack;

Explicació

  • My_Stack: Una instància del Integer_Stack.
  • Push, Pop: Operacions sobre la pila.
  • Ada.Text_IO.Put_Line: Utilitzat per imprimir l'element extret.

Ús Avançat dels Paquets Genèrics

Múltiples Paràmetres

Es pot definir un paquet genèric amb múltiples paràmetres, incloent tipus, objectes i subprogrames. Aquí hi ha un exemple:

generic
   type Key_Type is private;
   type Value_Type is private;
   with function Hash(Key : Key_Type) return Integer;
package Generic_Hash_Table is
   procedure Insert(Key : in Key_Type; Value : in Value_Type);
   function Retrieve(Key : in Key_Type) return Value_Type;
end Generic_Hash_Table;

Explicació

  • type Key_Type is private;: Un paràmetre formal de tipus per a la clau.
  • type Value_Type is private;: Un paràmetre formal de tipus per al valor.
  • with function Hash(Key : Key_Type) return Integer;: Un paràmetre formal de subprograma per a la funció de hash.

Instanciació amb Múltiples Paràmetres

function Integer_Hash(Key : Integer) return Integer is
begin
   return Key mod 101;
end Integer_Hash;

package Integer_String_Hash_Table is new Generic_Hash_Table(
   Key_Type => Integer,
   Value_Type => String,
   Hash => Integer_Hash
);

Explicació

  • Integer_Hash: Una funció de hash per a enters.
  • Integer_String_Hash_Table: Una instància de Generic_Hash_Table amb Integer com a tipus de clau i String com a tipus de valor.

Conclusió

Els paquets genèrics en Ada proporcionen una manera poderosa de crear codi reutilitzable i flexible. Mitjançant la parametrització de paquets amb tipus, objectes o subprogrames, es poden escriure algorismes i estructures de dades genèrics que funcionen amb qualsevol tipus. Entendre com definir, instanciar i utilitzar paquets genèrics és essencial per escriure programes en Ada eficients i mantenibles.
