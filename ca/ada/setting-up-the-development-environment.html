Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre Ada. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per començar a escriure i executar programes en Ada.

Instal·lació del Compilador Ada

Per escriure i executar programes en Ada, necessites un compilador. El compilador Ada més utilitzat és GNAT, que forma part de la GNU Compiler Collection (GCC).

Passos per Instal·lar GNAT

  1. Descarregar GNAT:

    • Visita el lloc web d'AdaCore per descarregar la GNAT Community Edition.
    • Tria la versió adequada per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·lar GNAT:

    • Windows: Executa l'instal·lador i segueix les instruccions en pantalla.
    • macOS: Obre el fitxer .dmg descarregat i arrossega la carpeta GNAT al teu directori d'Aplicacions.
    • Linux: Extreu el tarball descarregat i segueix les instruccions al fitxer README.

  3. Verificar la Instal·lació:

    • Obre un terminal o símbol del sistema.
    • Escriu gnat --version i prem Enter. Hauries de veure la informació de la versió de GNAT.
$ gnat --version
GNAT 11.2.0

Configuració d'un Entorn de Desenvolupament Integrat (IDE)

Tot i que pots escriure codi Ada en qualsevol editor de text, utilitzar un Entorn de Desenvolupament Integrat (IDE) pot millorar significativament la teva productivitat. Dos IDEs populars per Ada són GNAT Studio i Visual Studio Code.

GNAT Studio

  1. Descarregar GNAT Studio:

    • GNAT Studio està inclòs amb la GNAT Community Edition. Si ja has instal·lat GNAT, també hauries de tenir GNAT Studio.

  2. Llançar GNAT Studio:

    • Obre GNAT Studio des del teu directori d'Aplicacions o escrivint gnatstudio al teu terminal/símbol del sistema.

  3. Configurar GNAT Studio:

    • Configura el teu projecte creant un nou fitxer de projecte (.gpr).
    • Configura els paràmetres del projecte com ara els directoris de fonts i les opcions del compilador.

Visual Studio Code

  1. Instal·lar Visual Studio Code:

    • Descarrega i instal·la Visual Studio Code des del lloc web oficial.

  2. Instal·lar Extensions Ada:

    • Obre Visual Studio Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra.
    • Cerca "Ada" i instal·la l'extensió AdaCore Ada Language Support.

  3. Configurar Visual Studio Code:

    • Crea un nou espai de treball i afegeix el directori del teu projecte Ada.
    • Configura les tasques i els paràmetres de llançament per compilar i executar programes en Ada.

Escriure el teu Primer Programa en Ada

Un cop el teu entorn de desenvolupament estigui configurat, pots escriure i executar el teu primer programa en Ada.

Exemple: Hola Món

  1. Crear un nou fitxer:

    • Obre el teu IDE i crea un nou fitxer anomenat hello.adb.

  2. Escriure el codi:

    • Introdueix el següent codi a hello.adb:
with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;
  1. Compilar el programa:
    • Obre un terminal o utilitza la comanda de compilació al teu IDE.
    • Navega al directori que conté hello.adb.
    • Executa la següent comanda per compilar el programa:
$ gnatmake hello.adb
  1. Executar el programa:
    • Executa el programa compilat escrivint:
$ ./hello
Hello, World!

Conclusió

Configurar l'entorn de desenvolupament per Ada implica instal·lar el compilador GNAT, triar i configurar un IDE, i escriure el teu primer programa. Seguint aquests passos, estaràs preparat per començar el teu viatge en la programació amb Ada. Recorda, un entorn ben configurat pot millorar molt la teva experiència de codificació i productivitat.
