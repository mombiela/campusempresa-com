Introducció als Arrays

Els arrays són una estructura de dades fonamental en Ada que et permeten emmagatzemar i manipular col·leccions d'elements. Cada element en un array és identificat per un índex, i tots els elements han de ser del mateix tipus.

Conceptes Clau

  • Indexació: Els arrays en Ada estan indexats, és a dir, cada element s'accedeix utilitzant un índex.
  • Límits: Els arrays tenen límits definits, especificant el rang d'índexs vàlids.
  • Tipus: Tots els elements en un array han de ser del mateix tipus.
  • Arrays Estàtics i Dinàmics: Els arrays poden ser estàtics (mida fixa) o dinàmics (la mida pot canviar).

Declaració d'Arrays

En Ada, els arrays es declaren especificant el tipus d'elements i el rang d'índexs.

Arrays Estàtics

Els arrays estàtics tenen una mida fixa definida en temps de compilació.

-- Declaració d'un array estàtic amb índexs de l'1 al 5
type Index_Type is range 1 .. 5;
type Array_Type is array (Index_Type) of Integer;

-- Inicialització de l'array
My_Array : Array_Type := (1, 2, 3, 4, 5);

Arrays Dinàmics

Els arrays dinàmics poden canviar de mida durant l'execució utilitzant el tipus Unbounded_Array del paquet Ada.Containers.

with Ada.Containers.Indefinite_Vectors;
procedure Dynamic_Array_Example is
   package Integer_Vectors is new Ada.Containers.Indefinite_Vectors (Index_Type => Natural, Element_Type => Integer);
   My_Vector : Integer_Vectors.Vector;
begin
   -- Afegint elements a l'array dinàmic
   Integer_Vectors.Append (My_Vector, 10);
   Integer_Vectors.Append (My_Vector, 20);
end Dynamic_Array_Example;

Accés als Elements de l'Array

Els elements en un array s'accedeixen utilitzant els seus índexs.

-- Accés als elements en un array estàtic
My_Array(1) := 10;
Put_Line (Integer'Image (My_Array(1))); -- Sortida: 10

Iteració sobre Arrays

Pots iterar sobre arrays utilitzant bucles.

Utilitzant un Bucle `for`

-- Iteració sobre un array estàtic
for Index in My_Array'Range loop
   Put_Line (Integer'Image (My_Array(Index)));
end loop;

Utilitzant un Bucle `while`

-- Iteració sobre un array dinàmic
Index : Natural := My_Vector.First_Index;
while Index /= My_Vector.No_Element loop
   Put_Line (Integer'Image (My_Vector.Element (Index)));
   Index := My_Vector.Next_Index (Index);
end loop;

Arrays Multidimensionals

Ada suporta arrays multidimensionals, que són arrays d'arrays.

-- Declaració d'un array 2D
type Matrix_Type is array (1 .. 3, 1 .. 3) of Integer;
Matrix : Matrix_Type := ((1, 2, 3), (4, 5, 6), (7, 8, 9));

-- Accés als elements en un array 2D
Put_Line (Integer'Image (Matrix(2, 3))); -- Sortida: 6

Atributs d'Array

Ada proporciona diversos atributs per treballar amb arrays.

  • 'First: Retorna el primer índex de l'array.
  • 'Last: Retorna l'últim índex de l'array.
  • 'Length: Retorna el nombre d'elements en l'array.
  • 'Range: Retorna el rang d'índexs.
Put_Line (Integer'Image (My_Array'First));  -- Sortida: 1
Put_Line (Integer'Image (My_Array'Last));   -- Sortida: 5
Put_Line (Integer'Image (My_Array'Length)); -- Sortida: 5

Resum

Els arrays en Ada són estructures versàtils i potents que et permeten emmagatzemar i manipular col·leccions d'elements de manera eficient. Entendre com declarar, accedir i iterar sobre arrays és crucial per a una programació efectiva en Ada. Tant si treballes amb arrays estàtics com dinàmics, Ada proporciona característiques i atributs robustos per gestionar-los de manera efectiva.
