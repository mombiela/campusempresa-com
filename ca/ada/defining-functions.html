Introducció

Les funcions són blocs fonamentals en Ada, permetent encapsular codi reutilitzable. Aquesta secció et guiarà a través de la definició de funcions en Ada, des de la sintaxi bàsica fins a l'ús avançat.

Conceptes Clau

  • Definició de Funció: L'estructura i la sintaxi de la definició d'una funció.
  • Declaració de Funció: Com declarar una funció abans de la seva implementació.
  • Passatge de Paràmetres: Diferents maneres de passar paràmetres a les funcions.
  • Tipus de Retorn: Especificar i utilitzar tipus de retorn en funcions.
  • Sobrecàrrega de Funcions: Definir múltiples funcions amb el mateix nom però amb diferents paràmetres.

Definició de Funció

En Ada, una funció es defineix utilitzant la paraula clau function, seguida del nom de la funció, els paràmetres, el tipus de retorn i el cos de la funció.

Sintaxi Bàsica

function Function_Name (Parameter_List) return Return_Type is
begin
   -- Cos de la funció
end Function_Name;

Exemple

function Add (A, B : Integer) return Integer is
begin
   return A + B;
end Add;

En aquest exemple:

  • Add és el nom de la funció.
  • A i B són paràmetres de tipus Integer.
  • La funció retorna un Integer que és la suma de A i B.

Declaració de Funció

Abans d'implementar una funció, pots declarar-la. Això és útil per organitzar el codi i gestionar referències anticipades.

Sintaxi

function Function_Name (Parameter_List) return Return_Type;

Exemple

function Multiply (X, Y : Integer) return Integer;

Passatge de Paràmetres

Ada suporta diferents maneres de passar paràmetres a les funcions: in, out i in out.

Modes de Paràmetres

  • in: El paràmetre és només de lectura.
  • out: El paràmetre és només d'escriptura.
  • in out: El paràmetre es pot llegir i modificar.

Exemple

function Swap (X, Y : in out Integer) is
   Temp : Integer;
begin
   Temp := X;
   X := Y;
   Y := Temp;
end Swap;

En aquest exemple:

  • X i Y són paràmetres in out, el que significa que els seus valors es poden llegir i modificar dins de la funció.

Tipus de Retorn

Les funcions en Ada han d'especificar un tipus de retorn. Aquest tipus defineix el tipus de valor que la funció retornarà.

Exemple

function Square (N : Integer) return Integer is
begin
   return N * N;
end Square;

En aquest exemple:

  • Square retorna un Integer que és el quadrat de l'entrada N.

Sobrecàrrega de Funcions

Ada permet definir múltiples funcions amb el mateix nom però amb diferents llistes de paràmetres. Això es coneix com a sobrecàrrega de funcions.

Exemple

function Max (A, B : Integer) return Integer is
begin
   if A > B then
      return A;
   else
      return B;
   end if;
end Max;

function Max (A, B : Float) return Float is
begin
   if A > B then
      return A;
   else
      return B;
   end if;
end Max;

En aquest exemple:

  • Hi ha dues funcions Max, una per a paràmetres Integer i una altra per a paràmetres Float.

Conclusió

Definir funcions en Ada implica entendre la sintaxi i l'estructura de les definicions de funcions, declaracions, passatge de paràmetres, tipus de retorn i sobrecàrrega. En dominar aquests conceptes, podràs escriure codi reutilitzable i mantenible en Ada. Practica escrivint les teves pròpies funcions i experimentant amb diferents modes de paràmetres i tipus de retorn per aprofundir en la teva comprensió.
