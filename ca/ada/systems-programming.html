Introducció a la Programació de Sistemes

La programació de sistemes implica escriure programari que proporciona serveis al maquinari de l'ordinador, com ara sistemes operatius, controladors de dispositius i sistemes encastats. Ada és un llenguatge de programació d'alt nivell dissenyat per a la programació de sistemes, conegut per la seva tipificació forta, modularitat i suport per a sistemes en temps real.

Començant amb Ada

Abans d'endinsar-se en la programació de sistemes, és essencial entendre els conceptes bàsics d'Ada.

Sintaxi Bàsica

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;
  • with Ada.Text_IO; use Ada.Text_IO;: Inclou la biblioteca Ada.Text_IO per a operacions d'entrada/sortida.
  • procedure Hello is: Defineix un procediment anomenat Hello.
  • begin ... end Hello;: El bloc de codi que s'executa quan es crida el procediment.

Tipus de Dades

Ada admet diversos tipus de dades, incloent-hi enters, nombres de punt flotant i enumeracions.

procedure Data_Types is
   type Day is (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday);
   Today : Day := Monday;
   Number : Integer := 42;
   Pi : Float := 3.14159;
begin
   -- Codi per utilitzar aquestes variables
end Data_Types;

Programació de Baix Nivell en Ada

La programació de sistemes sovint requereix operacions de baix nivell, com ara l'accés directe a la memòria i la interacció amb el maquinari.

Tipus d'Accés (Punteres)

Els tipus d'accés en Ada són similars als punters en altres llenguatges.

procedure Access_Types is
   type Int_Ptr is access all Integer;
   Ptr : Int_Ptr := new Integer'(10);
begin
   Put_Line(Integer'Image(Ptr.all)); -- Mostra: 10
   Ptr.all := 20;
   Put_Line(Integer'Image(Ptr.all)); -- Mostra: 20
   Free(Ptr);
end Access_Types;
  • type Int_Ptr is access all Integer;: Defineix un tipus d'accés per a enters.
  • Ptr : Int_Ptr := new Integer'(10);: Assigna memòria per a un enter i l'inicialitza a 10.
  • Ptr.all: Desreferencia el punter per accedir al valor.

Interfície amb C

Ada pot interfície amb C per a operacions de baix nivell.

with Interfaces.C; use Interfaces.C;

procedure Interfacing_With_C is
   function C_Sqrt(X : Double) return Double;
   pragma Import(C, C_Sqrt, "sqrt");
   Result : Double := C_Sqrt(9.0);
begin
   Put_Line(Double'Image(Result)); -- Mostra: 3.0
end Interfacing_With_C;
  • pragma Import(C, C_Sqrt, "sqrt");: Importa la funció sqrt de C a Ada.

Sistemes en Temps Real

Ada és molt adequada per a sistemes en temps real gràcies al seu suport per a tasques i sincronització.

Tasques

Les tasques en Ada són similars als fils en altres llenguatges.

task type Worker is
   entry Start;
end Worker;

task body Worker is
begin
   accept Start do
      Put_Line("Worker started");
   end Start;
end Worker;

procedure Real_Time_Systems is
   W : Worker;
begin
   W.Start;
end Real_Time_Systems;
  • task type Worker is: Defineix un tipus de tasca anomenat Worker.
  • entry Start;: Declara un punt d'entrada per a la tasca.
  • accept Start do ... end Start;: Defineix el codi que s'executa quan es crida el punt d'entrada.

Sincronització

Ada proporciona objectes protegits per a la sincronització.

protected type Counter is
   procedure Increment;
   function Value return Integer;
private
   Count : Integer := 0;
end Counter;

protected body Counter is
   procedure Increment is
   begin
      Count := Count + 1;
   end Increment;

   function Value return Integer is
   begin
      return Count;
   end Value;
end Counter;

procedure Synchronization is
   C : Counter;
begin
   C.Increment;
   Put_Line(Integer'Image(C.Value)); -- Mostra: 1
end Synchronization;
  • protected type Counter is: Defineix un tipus protegit anomenat Counter.
  • procedure Increment; function Value return Integer;: Declara operacions per al tipus protegit.
  • private Count : Integer := 0;: Declara una variable privada.

Temes Avançats

La programació de sistemes avançada en Ada inclou temes com la gestió d'interrupcions i la manipulació directa del maquinari.

Gestió d'Interrupcions

Ada pot gestionar interrupcions de maquinari utilitzant la pragma Interrupt_Handler.

with System; use System;

procedure Interrupt_Handling is
   pragma Interrupt_Handler(Handler);
   procedure Handler is
   begin
      -- Codi de gestió d'interrupcions
   end Handler;
begin
   -- Codi d'inicialització
end Interrupt_Handling;
  • pragma Interrupt_Handler(Handler);: Declara Handler com a gestor d'interrupcions.

Manipulació Directa del Maquinari

Ada permet la manipulació directa dels registres de maquinari.

with System; use System;

procedure Hardware_Manipulation is
   Register : Unsigned_32 := 16#FFFF_FFFF#;
begin
   -- Manipula directament el registre
   Register := Register and 16#0000_0000#;
end Hardware_Manipulation;
  • Register : Unsigned_32 := 16#FFFF_FFFF#;: Declara un enter sense signe de 32 bits inicialitzat a tots 1s.

Conclusió

La programació de sistemes en Ada ofereix un conjunt robust de característiques per a aplicacions de baix nivell i en temps real. Des de la sintaxi bàsica i els tipus de dades fins a temes avançats com la gestió d'interrupcions i la manipulació directa del maquinari, Ada proporciona les eines necessàries per desenvolupar programari de sistemes fiable i eficient. En dominar aquests conceptes, els programadors poden aprofitar les fortaleses d'Ada per construir sistemes d'alt rendiment i fàcil manteniment.
