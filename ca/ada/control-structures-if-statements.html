Introducció

A Ada, les estructures de control són essencials per dirigir el flux d'un programa. Una de les estructures de control més fonamentals és la sentència if, que permet a un programa prendre decisions basades en condicions. Aquesta secció cobrirà els conceptes bàsics de les sentències if, incloent-hi la sintaxi, l'ús i exemples.

Sintaxi bàsica

La sentència if a Ada s'utilitza per executar un bloc de codi només si una condició especificada és certa. La sintaxi bàsica és la següent:

if condition then
   -- statements to execute if condition is true
end if;

Exemple

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 10;
begin
   if Number > 5 then
      Put_Line("Number is greater than 5");
   end if;
end Check_Number;

En aquest exemple, el programa comprova si la variable Number és més gran que 5. Si la condició és certa, imprimeix "Number is greater than 5".

Afegint Else i Elsif

Per gestionar múltiples condicions, pots utilitzar les clausules else i elsif.

Sintaxi amb Else

if condition then
   -- statements to execute if condition is true
else
   -- statements to execute if condition is false
end if;

Sintaxi amb Elsif

if condition1 then
   -- statements to execute if condition1 is true
elsif condition2 then
   -- statements to execute if condition2 is true
else
   -- statements to execute if none of the above conditions are true
end if;

Exemple amb Else i Elsif

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 3;
begin
   if Number > 5 then
      Put_Line("Number is greater than 5");
   elsif Number = 5 then
      Put_Line("Number is equal to 5");
   else
      Put_Line("Number is less than 5");
   end if;
end Check_Number;

En aquest exemple, el programa comprova múltiples condicions:

  • Si Number és més gran que 5, imprimeix "Number is greater than 5".
  • Si Number és igual a 5, imprimeix "Number is equal to 5".
  • Si cap de les condicions anteriors és certa, imprimeix "Number is less than 5".

Sentències If niades

Pots niar sentències if dins d'altres per gestionar condicions més complexes.

Exemple de sentències If niades

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 7;
begin
   if Number > 0 then
      if Number < 10 then
         Put_Line("Number is between 1 and 9");
      else
         Put_Line("Number is 10 or greater");
      end if;
   else
      Put_Line("Number is 0 or negative");
   end if;
end Check_Number;

En aquest exemple, el programa primer comprova si Number és més gran que 0. Si és cert, després comprova si Number és menor que 10. Depenent del resultat, imprimeix el missatge adequat.

Millors pràctiques

  • Llegibilitat: Assegura't que les teves sentències if siguin fàcils de llegir i entendre.
  • Evita la niació profunda: Les sentències if profundament niades poden ser difícils de seguir. Considera utilitzar operadors lògics o descompondre condicions complexes en altres més senzilles.
  • Format consistent: Utilitza una indentació i un format consistent per fer el teu codi més llegible.

Resum

La sentència if és una eina poderosa a Ada per prendre decisions basades en condicions. Entenent la seva sintaxi bàsica, així com com utilitzar else, elsif i sentències if niades, pots escriure programes més complexos i flexibles. Sempre apunta a la llegibilitat i mantenibilitat en el teu codi per assegurar-te que sigui fàcil d'entendre i modificar.
