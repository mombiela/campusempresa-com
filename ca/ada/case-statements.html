Introducció a les Sentències Case

Les sentències case en Ada són una estructura de control utilitzada per executar un bloc de codi d'entre múltiples blocs possibles basant-se en el valor d'una expressió. Són similars a les sentències switch en altres llenguatges de programació com C o Java. Les sentències case milloren la llegibilitat i mantenibilitat del codi delineant clarament diferents camins d'execució.

Sintaxi Bàsica

La sintaxi bàsica d'una sentència case en Ada és la següent:

case expression is
   when choice1 =>
      -- Sentències per a choice1
   when choice2 =>
      -- Sentències per a choice2
   when others =>
      -- Sentències per a totes les altres opcions
end case;
  • expression: El valor que s'està avaluant.
  • choice1, choice2, etc.: Valors possibles de l'expressió.
  • others: Un catch-all per a qualsevol valor no explícitament llistat.

Exemple

Aquí hi ha un exemple senzill d'una sentència case en Ada:

with Ada.Text_IO; use Ada.Text_IO;

procedure Case_Example is
   Grade : Character := 'B';
begin
   case Grade is
      when 'A' =>
         Put_Line("Excel·lent!");
      when 'B' =>
         Put_Line("Bona feina!");
      when 'C' =>
         Put_Line("Ben fet!");
      when 'D' =>
         Put_Line("Has aprovat.");
      when 'F' =>
         Put_Line("Millor torna-ho a intentar.");
      when others =>
         Put_Line("Nota invàlida.");
   end case;
end Case_Example;

En aquest exemple, el programa imprimeix un missatge basat en el valor de la variable Grade.

Conceptes Clau

  • Expressió: El valor que la sentència case avalua.
  • Opcions: Els valors possibles que l'expressió pot prendre.
  • Others: Un cas per defecte que maneja qualsevol valor no explícitament llistat.

Ús Avançat

Rangs i Múltiples Opcions

Ada permet especificar rangs i múltiples opcions en una sentència case. Això pot ser útil per agrupar casos similars junts.

Exemple amb Rangs

with Ada.Text_IO; use Ada.Text_IO;

procedure Case_Range_Example is
   Score : Integer := 85;
begin
   case Score is
      when 90 .. 100 =>
         Put_Line("Excel·lent!");
      when 80 .. 89 =>
         Put_Line("Molt Bé!");
      when 70 .. 79 =>
         Put_Line("Bé!");
      when 60 .. 69 =>
         Put_Line("Aprovat.");
      when others =>
         Put_Line("Suspès.");
   end case;
end Case_Range_Example;

En aquest exemple, el programa imprimeix un missatge basat en el rang en què cau la variable Score.

Sentències Case Imbricades

Pots imbricar sentències case dins d'altres per manejar processos de presa de decisions més complexos.

Exemple de Sentències Case Imbricades

with Ada.Text_IO; use Ada.Text_IO;

procedure Nested_Case_Example is
   Day : Integer := 3;
   Weather : Character := 'S';
begin
   case Day is
      when 1 | 2 | 3 | 4 | 5 =>
         Put_Line("Dia laborable");
         case Weather is
            when 'S' =>
               Put_Line("Assolellat");
            when 'R' =>
               Put_Line("Plujós");
            when others =>
               Put_Line("Temps desconegut");
         end case;
      when 6 | 7 =>
         Put_Line("Cap de setmana");
         case Weather is
            when 'S' =>
               Put_Line("Assolellat");
            when 'R' =>
               Put_Line("Plujós");
            when others =>
               Put_Line("Temps desconegut");
         end case;
      when others =>
         Put_Line("Dia invàlid");
   end case;
end Nested_Case_Example;

En aquest exemple, el programa primer determina si el dia és un dia laborable o cap de setmana i després comprova la condició meteorològica.

Conclusió

Les sentències case en Ada són una eina poderosa per controlar el flux d'un programa basat en el valor d'una expressió. Milloren la llegibilitat i mantenibilitat del codi separant clarament diferents camins d'execució. Entenent la sintaxi bàsica, utilitzant rangs i múltiples opcions, i imbricant sentències case, pots manejar processos de presa de decisions complexos de manera efectiva. Practica amb els exemples i exercicis proporcionats per esdevenir proficient en l'ús de les sentències case en Ada.
