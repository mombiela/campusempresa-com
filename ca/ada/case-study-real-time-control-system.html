Introducció

En aquest estudi de cas, explorarem com dissenyar i implementar un sistema de control en temps real utilitzant Ada. Els sistemes en temps real són crucials en diversos dominis com l'aeroespacial, l'automoció i l'automatització industrial, on les respostes oportunes i predictibles són essencials. Ada, amb el seu fort tipatge, suport per a la concurrència i capacitats en temps real, és una excel·lent elecció per a aquests sistemes.

Conceptes Clau

  • Sistemes en Temps Real: Sistemes que han de respondre a entrades o esdeveniments dins d'un estricte límit de temps.
  • Concurrència: La capacitat d'executar múltiples seqüències d'operacions simultàniament.
  • Tasques en Ada: Suport integrat d'Ada per a la programació concurrent a través de tasques.
  • Planificació en Temps Real: Tècniques per assegurar que les tasques compleixin els seus terminis.

Requisits del Sistema

Abans d'entrar en la implementació, anem a esbossar els requisits per al nostre sistema de control en temps real:

  1. Adquisició de Dades de Sensors: Llegir periòdicament dades dels sensors.
  2. Processament de Dades: Processar les dades adquirides per prendre decisions de control.
  3. Control d'Actuadors: Enviar ordres als actuadors basades en les dades processades.
  4. Restriccions de Temps: Assegurar que totes les tasques compleixin els seus terminis.

Disseny del Sistema

Estructura de Tasques

Dissenyarem el nostre sistema utilitzant tasques d'Ada per gestionar diferents components:

  • Tasca de Sensor: Llegeix dades dels sensors a intervals regulars.
  • Tasca de Processament: Processa les dades del sensor.
  • Tasca d'Actuador: Controla els actuadors basant-se en les dades processades.

Comunicació de Tasques

Les tasques es comunicaran utilitzant objectes protegits per assegurar la consistència de les dades i evitar condicions de carrera.

Implementació

Tasca de Sensor

La tasca de sensor llegirà dades d'un sensor cada 100 mil·lisegons.

task Sensor_Task is
    period : constant Duration := 0.1; -- 100 mil·lisegons
end Sensor_Task;

task body Sensor_Task is
    sensor_data : Integer;
begin
    loop
        -- Simular la lectura de dades del sensor
        sensor_data := Read_Sensor();
        -- Emmagatzemar les dades en un recurs compartit
        Sensor_Data_Protected.Store(sensor_data);
        delay period;
    end loop;
end Sensor_Task;

Objectes Protegits

S'utilitzaran objectes protegits per emmagatzemar i recuperar dades de manera segura entre tasques.

protected Sensor_Data_Protected is
    procedure Store(data : in Integer);
    function Retrieve return Integer;
private
    sensor_data : Integer := 0;
end Sensor_Data_Protected;

protected body Sensor_Data_Protected is
    procedure Store(data : in Integer) is
    begin
        sensor_data := data;
    end Store;

    function Retrieve return Integer is
    begin
        return sensor_data;
    end Retrieve;
end Sensor_Data_Protected;

protected Processed_Data_Protected is
    procedure Store(data : in Integer);
    function Retrieve return Integer;
private
    processed_data : Integer := 0;
end Processed_Data_Protected;

protected body Processed_Data_Protected is
    procedure Store(data : in Integer) is
    begin
        processed_data := data;
    end Store;

    function Retrieve return Integer is
    begin
        return processed_data;
    end Retrieve;
end Processed_Data_Protected;

Resum

En aquest estudi de cas, hem dissenyat i implementat un sistema de control en temps real utilitzant Ada. Hem cobert:

  • L'estructura i els requisits d'un sistema en temps real.
  • Com utilitzar les tasques d'Ada per a la concurrència.
  • Comunicació entre tasques utilitzant objectes protegits.

Aquest exemple demostra les capacitats d'Ada per gestionar restriccions en temps real i concurrència, fent-la una opció adequada per desenvolupar sistemes en temps real fiables i predictibles.
