Introducció als Genèrics
Procediments i Funcions Genèrics ►

Els genèrics en Ada proporcionen una manera poderosa de crear components de codi flexibles i reutilitzables. Permeten escriure algoritmes i estructures de dades que poden operar sobre qualsevol tipus de dades, fent que el teu codi sigui més modular i fàcil de mantenir.

Conceptes Clau

  • Unitats Genèriques: Aquests són plantilles per crear subprogrames o paquets que poden operar sobre diferents tipus de dades.
  • Paràmetres Genèrics: Aquests són els espais reservats per als tipus o valors que es especificaran quan la unitat genèrica sigui instanciada.
  • Instanciació: Aquest és el procés de crear una instància específica d'una unitat genèrica proporcionant paràmetres reals.

Unitats Genèriques

En Ada, pots crear procediments, funcions i paquets genèrics. Aquí tens un exemple bàsic d'un procediment genèric:

generic
   type Item_Type is private;
procedure Swap (X, Y : in out Item_Type);
  • La paraula clau generic indica l'inici d'una unitat genèrica.
  • type Item_Type is private; declara un paràmetre de tipus genèric.
  • procedure Swap és el procediment genèric que operarà sobre el Item_Type.

Instanciació de Genèrics

Per utilitzar una unitat genèrica, cal instanciar-la amb paràmetres reals. Aquí tens com instanciar el procediment Swap per a enters:

procedure Swap_Integer is new Swap (Item_Type => Integer);

Ara pots utilitzar Swap_Integer per intercanviar valors enters:

declare
   A, B : Integer := 10;
begin
   Swap_Integer (A, B);
   -- Ara A = 10 i B = 10
end;

Paquets Genèrics

Els paquets genèrics permeten crear components reutilitzables que es poden personalitzar amb diferents tipus o valors. Aquí tens un exemple d'un paquet genèric per a una pila:

generic
   type Element_Type is private;
package Generic_Stack is
   procedure Push (Item : in Element_Type);
   procedure Pop (Item : out Element_Type);
end Generic_Stack;

Per instanciar aquest paquet per a enters:

package Integer_Stack is new Generic_Stack (Element_Type => Integer);

Ara pots utilitzar Integer_Stack per crear i manipular piles d'enters.

Exemple: Algoritme de Classificació Genèric

Creem un paquet genèric per a un algoritme de classificació:

generic
   type Element_Type is private;
   with function "<" (Left, Right : Element_Type) return Boolean;
package Generic_Sort is
   procedure Sort (Array : in out array (Positive range <>) of Element_Type);
end Generic_Sort;

Per instanciar i utilitzar aquest paquet per classificar una matriu d'enters:

package Integer_Sort is new Generic_Sort (Element_Type => Integer, "<" => "<");

declare
   Arr : array (1 .. 5) of Integer := (5, 3, 4, 1, 2);
begin
   Integer_Sort.Sort (Arr);
   -- Ara Arr està ordenat: (1, 2, 3, 4, 5)
end;

Conclusió

Els genèrics en Ada proporcionen un mecanisme robust per crear components de codi flexibles i reutilitzables. Entenent com definir i instanciar unitats genèriques, pots escriure codi més modular i fàcil de mantenir. Practica creant i utilitzant genèrics amb diferents tipus i veu com poden simplificar les teves tasques de programació.
