Introducció

L'agrupament o clustering és una tècnica d'aprenentatge no supervisat utilitzada en intel·ligència artificial i aprenentatge automàtic per agrupar un conjunt d'objectes de tal manera que els objectes en el mateix grup (o clúster) siguin més similars entre si que els objectes en altres grups. Aquesta tècnica és fonamental per a l'anàlisi exploratori de dades i té aplicacions en diverses àrees com la segmentació de clients, la detecció d'anomalies i la compressió de dades.

Conceptes Clau

  • Agrupament (Clustering): Procés de dividir un conjunt de dades en grups o clústers, on les dades dins de cada grup són més similars entre si que a les dades d'altres grups.
  • Distància o Similitud: Mesura utilitzada per determinar què tan similars o diferents són dos objectes. Les mètriques comunes inclouen la distància euclidiana, la distància de Manhattan i la similitud del cosinus.
  • Centroides: Punts que representen el centre d'un clúster en algorismes com K-means.
  • Algorismes d'Agrupament: Mètodes específics utilitzats per realitzar l'agrupament, com K-means, DBSCAN i l'agrupament jeràrquic.

Algorismes d'Agrupament Comuns

K-means

K-means és un dels algorismes d'agrupament més populars i fàcils d'entendre. L'objectiu és dividir un conjunt de dades en K clústers, on cada clúster està representat pel centroide dels punts en aquest clúster.

Passos de l'algorisme K-means:

  1. Seleccionar K punts inicials com a centroides.
  2. Assignar cada punt de dades al clúster el centreide del qual estigui més proper.
  3. Recalcular els centroides com la mitjana dels punts assignats a cada clúster.
  4. Repetir els passos 2 i 3 fins que els centroides no canviïn significativament.
from sklearn.cluster import KMeans
import numpy as np

# Dades d'exemple
X = np.array([[1, 2], [1, 4], [1, 0],
              [4, 2], [4, 4], [4, 0]])

# Crear el model K-means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Predir els clústers
labels = kmeans.labels_

# Obtenir els centroides
centroids = kmeans.cluster_centers_

print("Labels:", labels)
print("Centroids:", centroids)

DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

DBSCAN és un algorisme d'agrupament basat en densitat que pot trobar clústers de forma arbitrària i manejar soroll (punts que no pertanyen a cap clúster).

Passos de l'algorisme DBSCAN:

  1. Seleccionar un punt aleatori no visitat.
  2. Recuperar tots els punts densament connectats a aquest punt.
  3. Si el punt és un punt central, formar un clúster.
  4. Si el punt és un punt de vora, marcar-lo com a soroll.
  5. Repetir fins que tots els punts hagin estat visitats.
from sklearn.cluster import DBSCAN

# Dades d'exemple
X = np.array([[1, 2], [2, 2], [2, 3],
              [8, 7], [8, 8], [25, 80]])

# Crear el model DBSCAN
dbscan = DBSCAN(eps=3, min_samples=2).fit(X)

# Predir els clústers
labels = dbscan.labels_

print("Labels:", labels)

Agrupament Jeràrquic

L'agrupament jeràrquic construeix una jerarquia de clústers mitjançant un enfocament aglomeratiu (de baix a dalt) o divisori (de dalt a baix).

Passos de l'algorisme aglomeratiu:

  1. Començar amb cada punt com un clúster individual.
  2. Combinar els dos clústers més propers en un sol clúster.
  3. Repetir el pas 2 fins que tots els punts estiguin en un sol clúster.
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# Dades d'exemple
X = np.array([[1, 2], [2, 2], [2, 3],
              [8, 7], [8, 8], [25, 80]])

# Crear el model d'agrupament jeràrquic
Z = linkage(X, 'ward')

# Dibuixar el dendrograma
plt.figure()
dendrogram(Z)
plt.show()

Comparació d'Algorismes

| Algorisme | Avantatges | Desavantatges | |-----------------|----------------------------------------------------|----------------------------------------------------| | K-means | Simple, ràpid, fàcil d'entendre | Sensible a la selecció de K, no maneja bé el soroll | | DBSCAN | Maneja bé el soroll, troba clústers arbitraris | Sensible als paràmetres eps i min_samples | | Jeràrquic | No requereix predefinir el nombre de clústers | Computacionalment costós per a grans datasets |

Conclusió

Els algorismes d'agrupament són eines poderoses per a l'anàlisi exploratori de dades en intel·ligència artificial. Cada algorisme té els seus propis avantatges i desavantatges, i l'elecció de l'algorisme adequat depèn de la naturalesa de les dades i el problema específic a resoldre. És fonamental comprendre les característiques i limitacions de cada mètode per aplicar-los de manera efectiva en la pràctica.
