Introducció a les Xarxes Neuronals Convolucionals

Les Xarxes Neuronals Convolucionals (CNN, per les seves sigles en anglès) són un tipus de xarxa neuronal especialment eficaç per a tasques de processament d'imatges i reconeixement de patrons. A diferència de les xarxes neuronals tradicionals, les CNN utilitzen capes convolucionals que permeten extreure característiques espacials i jeràrquiques de les dades d'entrada.

Arquitectura d'una CNN

Una CNN típica està composta per diverses capes, cadascuna amb una funció específica:

  1. Capa Convolucional (Convolutional Layer)
  2. Capa d'Agrupament (Pooling Layer)
  3. Capa de Normalització (Normalization Layer)
  4. Capa de Connexió Completa (Fully Connected Layer)

Capa Convolucional

La capa convolucional és el nucli d'una CNN. Utilitza filtres (o kernels) per realitzar convolucions sobre la imatge d'entrada, produint mapes de característiques.

  • Filtre/Kernels: Matrius petites que es desplacen sobre la imatge d'entrada.
  • Stride: El pas amb el qual el filtre es mou sobre la imatge.
  • Padding: Afegir vores a la imatge per controlar la mida de la sortida.

Exemple de codi:

import tensorflow as tf
from tensorflow.keras import layers

model = tf.keras.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

Capa d'Agrupament

La capa d'agrupament redueix les dimensions espacials dels mapes de característiques, disminuint la quantitat de paràmetres i el cost computacional.

  • Max Pooling: Selecciona el valor màxim en una finestra.
  • Average Pooling: Calcula la mitjana dels valors en una finestra.

Exemple de codi:

model.add(layers.MaxPooling2D((2, 2)))

Capa de Normalització

La normalització ajuda a accelerar l'entrenament i a millorar l'estabilitat de la xarxa.

  • Batch Normalization: Normalitza les activacions d'una capa per a cada mini-lot.

Exemple de codi:

model.add(layers.BatchNormalization())

Capa de Connexió Completa

Les capes de connexió completa (Fully Connected) són capes densament connectades que s'utilitzen al final de la xarxa per realitzar la classificació.

Exemple de codi:

model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

Exemple Complet d'una CNN

A continuació, es presenta un exemple complet d'una CNN per a la classificació d'imatges del conjunt de dades MNIST:

import tensorflow as tf
from tensorflow.keras import datasets, layers, models

# Carregar i preprocessar les dades
(train_images, train_labels), (test_images, test_labels) = datasets.mnist.load_data()
train_images = train_images.reshape((60000, 28, 28, 1)).astype('float32') / 255
test_images = test_images.reshape((10000, 28, 28, 1)).astype('float32') / 255

# Definir el model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# Compilar el model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Entrenar el model
model.fit(train_images, train_labels, epochs=5, validation_data=(test_images, test_labels))

Conclusió

Les Xarxes Neuronals Convolucionals són una eina poderosa per al processament d'imatges i altres tasques de reconeixement de patrons. La seva arquitectura, que inclou capes convolucionals, d'agrupament i de connexió completa, permet extreure i aprendre característiques complexes de les dades d'entrada. Amb la pràctica i l'experimentació, els professionals poden ajustar i optimitzar les CNN per a una varietat d'aplicacions en el camp de la intel·ligència artificial.
