Introducció a la Lògica Difusa

La lògica difusa és una extensió de la lògica clàssica que permet manejar la incertesa i la imprecisió. A diferència de la lògica binària tradicional, on les variables han de ser veritables o falses, la lògica difusa permet valors intermedis entre 0 i 1, la qual cosa la fa ideal per a aplicacions en intel·ligència artificial i sistemes de control.

Conceptes Clau

  • Conjunt Difús: Un conjunt en el qual els elements tenen graus de pertinença que varien entre 0 i 1.
  • Funció de Pertinença: Una funció que defineix com cada punt en l'espai d'entrada s'assigna a un grau de pertinença en el conjunt difús.
  • Operacions Difuses: Extensions de les operacions lògiques clàssiques (AND, OR, NOT) per manejar graus de pertinença.
  • Regles Difuses: Declaracions "si-llavors" que utilitzen lògica difusa per modelar relacions entre variables.
  • Inferència Difusa: El procés d'aplicar regles difuses per obtenir conclusions a partir de dades difuses.

Funcions de Pertinença

Les funcions de pertinença són fonamentals en la lògica difusa. Alguns tipus comuns inclouen:

  • Triangular: Definida per tres punts, formant un triangle.
  • Trapezoïdal: Definida per quatre punts, formant un trapezoide.
  • Gaussiana: Definida per una corba gaussiana.

Exemple de Funció de Pertinença Triangular

import numpy as np
import matplotlib.pyplot as plt

def triangular_membership(x, a, b, c):
    return np.maximum(0, np.minimum((x-a)/(b-a), (c-x)/(c-b)))

x = np.linspace(0, 10, 100)
y = triangular_membership(x, 2, 5, 8)

plt.plot(x, y)
plt.title('Funció de Pertinença Triangular')
plt.xlabel('x')
plt.ylabel('Grau de Pertinença')
plt.grid(True)
plt.show()

Operacions Difuses

Les operacions difuses són extensions de les operacions lògiques clàssiques:

  • AND (Intersecció): \( \min(\mu_A(x), \mu_B(x)) \)
  • OR (Unió): \( \max(\mu_A(x), \mu_B(x)) \)
  • NOT (Complement): \( 1 - \mu_A(x) \)

Regles Difuses

Les regles difuses són declaracions "si-llavors" que utilitzen lògica difusa. Per exemple:

  • Si la temperatura és alta, llavors el ventilador ha d'estar a alta velocitat.

Aquestes regles es poden combinar per formar sistemes d'inferència difusa.

Exemple de Regles Difuses

import skfuzzy as fuzz
from skfuzzy import control as ctrl

# Definir variables d'entrada i sortida
temperature = ctrl.Antecedent(np.arange(0, 41, 1), 'temperature')
fan_speed = ctrl.Consequent(np.arange(0, 101, 1), 'fan_speed')

# Definir funcions de pertinença
temperature['low'] = fuzz.trimf(temperature.universe, [0, 0, 20])
temperature['medium'] = fuzz.trimf(temperature.universe, [10, 20, 30])
temperature['high'] = fuzz.trimf(temperature.universe, [20, 40, 40])

fan_speed['low'] = fuzz.trimf(fan_speed.universe, [0, 0, 50])
fan_speed['medium'] = fuzz.trimf(fan_speed.universe, [25, 50, 75])
fan_speed['high'] = fuzz.trimf(fan_speed.universe, [50, 100, 100])

# Definir regles
rule1 = ctrl.Rule(temperature['low'], fan_speed['low'])
rule2 = ctrl.Rule(temperature['medium'], fan_speed['medium'])
rule3 = ctrl.Rule(temperature['high'], fan_speed['high'])

# Crear sistema de control
fan_ctrl = ctrl.ControlSystem([rule1, rule2, rule3])
fan = ctrl.ControlSystemSimulation(fan_ctrl)

# Simular el sistema
fan.input['temperature'] = 25
fan.compute()
print(fan.output['fan_speed'])

Conclusió

La lògica difusa és una eina poderosa en intel·ligència artificial per manejar la incertesa i la imprecisió. En permetre valors intermedis entre 0 i 1, la lògica difusa proporciona una forma més flexible i realista de modelar problemes complexos. Amb l'ús de funcions de pertinença, operacions difuses i regles difuses, és possible construir sistemes d'inferència difusa que poden prendre decisions informades en situacions incertes.
