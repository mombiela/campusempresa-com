Introducció a la Regressió

La regressió és una tècnica fonamental en el camp de la intel·ligència artificial i l'aprenentatge automàtic. S'utilitza per predir valors continus basats en dades d'entrada. A diferència de la classificació, que prediu categories discretes, la regressió se centra en predir valors numèrics.

Conceptes Clau

  • Variable Dependent (Y): La variable que es desitja predir.
  • Variable Independent (X): Les variables que s'utilitzen per fer la predicció.
  • Funció de Pèrdua: Una funció que mesura l'error de les prediccions del model.
  • Coeficients: Paràmetres del model que s'ajusten durant l'entrenament.

Tipus d'Algoritmes de Regressió

Regressió Lineal

La regressió lineal és un dels algoritmes més simples i àmpliament utilitzats. Es basa en la relació lineal entre la variable dependent i una o més variables independents.

Fórmula

\[ Y = \beta_0 + \beta_1X_1 + \beta_2X_2 + \ldots + \beta_nX_n \]

Exemple de Codi

import numpy as np
from sklearn.linear_model import LinearRegression

# Dades d'exemple
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([2, 3, 4, 5])

# Crear el model de regressió lineal
model = LinearRegression()

# Entrenar el model
model.fit(X, y)

# Fer prediccions
predictions = model.predict(X)

print("Prediccions:", predictions)

Regressió Polinòmica

La regressió polinòmica és una extensió de la regressió lineal que permet modelar relacions no lineals entre les variables independents i la variable dependent.

Fórmula

\[ Y = \beta_0 + \beta_1X + \beta_2X^2 + \ldots + \beta_nX^n \]

Exemple de Codi

from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import make_pipeline

# Dades d'exemple
X = np.array([[1], [2], [3], [4]])
y = np.array([2, 3, 5, 7])

# Crear el model de regressió polinòmica
degree = 2
model = make_pipeline(PolynomialFeatures(degree), LinearRegression())

# Entrenar el model
model.fit(X, y)

# Fer prediccions
predictions = model.predict(X)

print("Prediccions:", predictions)

Regressió Ridge

La regressió Ridge és una variant de la regressió lineal que inclou una penalització per evitar el sobreajustament. Aquesta tècnica és útil quan hi ha multicolinealitat entre les variables independents.

Fórmula

\[ \text{Minimitzar} \quad \sum_{i=1}^{n} (y_i - \hat{y}i)^2 + \lambda \sum{j=1}^{p} \beta_j^2 \]

Exemple de Codi

from sklearn.linear_model import Ridge

# Dades d'exemple
X = np.array([[1, 2], [2, 3], [3, 4], [4, 5]])
y = np.array([2, 3, 4, 5])

# Crear el model de regressió Ridge
model = Ridge(alpha=1.0)

# Entrenar el model
model.fit(X, y)

# Fer prediccions
predictions = model.predict(X)

print("Prediccions:", predictions)

Comparació d'Algoritmes de Regressió

| Algoritme | Avantatges | Desavantatges | |---------------------|-------------------------------------------------|----------------------------------------------| | Regressió Lineal | Simple, fàcil d'interpretar | No maneja relacions no lineals | | Regressió Polinòmica| Captura relacions no lineals | Pot sobreajustar si el grau del polinomi és molt alt | | Regressió Ridge | Redueix el sobreajustament, maneja multicolinealitat | Requereix ajust del paràmetre de penalització |

Conclusió

Els algoritmes de regressió són eines poderoses per predir valors continus en el camp de la intel·ligència artificial. Des de la regressió lineal simple fins a tècniques més avançades com la regressió polinòmica i Ridge, cada algoritme té els seus propis avantatges i limitacions. Comprendre aquests algoritmes i saber quan aplicar-los és crucial per a qualsevol professional que treballi amb dades i models predictius.
