Què és l'Aprenentatge Automàtic?

L'Aprenentatge Automàtic (Machine Learning, ML) és una branca de la intel·ligència artificial que se centra en el desenvolupament d'algoritmes i tècniques que permeten als ordinadors aprendre i fer prediccions o decisions basades en dades. En lloc de ser explícitament programades per realitzar una tasca, les màquines utilitzen patrons i inferències per millorar el seu rendiment en tasques específiques.

Tipus d'Aprenentatge Automàtic

Hi ha diversos tipus d'aprenentatge automàtic, cadascun amb les seves pròpies característiques i aplicacions:

  • Aprenentatge Supervisat: El model s'entrena amb un conjunt de dades etiquetades. Exemples inclouen la regressió i la classificació.
  • Aprenentatge No Supervisat: El model s'entrena amb dades no etiquetades i busca patrons o estructures ocultes. Exemples inclouen el clustering i la reducció de dimensionalitat.
  • Aprenentatge per Reforç: El model aprèn a prendre decisions mitjançant la interacció amb un entorn i la recepció de recompenses o càstigs.

Components Clau de l'Aprenentatge Automàtic

Per entendre com funciona l'aprenentatge automàtic, és important conèixer els seus components bàsics:

  • Dades: La matèria primera de l'aprenentatge automàtic. Les dades poden ser estructurades (taules) o no estructurades (imatges, text).
  • Model: Una representació matemàtica que s'ajusta a les dades per fer prediccions o decisions.
  • Algoritme: El procediment utilitzat per ajustar el model a les dades.
  • Funció de Pèrdua: Una mètrica que avalua què tan bé el model s'ajusta a les dades.
  • Optimització: El procés d'ajustar els paràmetres del model per minimitzar la funció de pèrdua.

Exemple d'Aprenentatge Supervisat

Vegem un exemple senzill d'aprenentatge supervisat utilitzant Python i la biblioteca scikit-learn.

Exemple: Classificació amb K-Nearest Neighbors (KNN)

# Importar biblioteques necessàries
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# Carregar el conjunt de dades Iris
iris = load_iris()
X = iris.data
y = iris.target

# Dividir les dades en conjunts d'entrenament i prova
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Crear el model KNN
knn = KNeighborsClassifier(n_neighbors=3)

# Entrenar el model
knn.fit(X_train, y_train)

# Fer prediccions
y_pred = knn.predict(X_test)

# Avaluar el model
accuracy = accuracy_score(y_test, y_pred)
print(f"Precisió del model: {accuracy:.2f}")

Explicació del Codi

  1. Importació de Biblioteques: S'importen les biblioteques necessàries per al maneig de dades, la creació del model i l'avaluació del rendiment.
  2. Càrrega de Dades: S'utilitza el conjunt de dades Iris, que és un conjunt de dades clàssic per a problemes de classificació.
  3. Divisió de Dades: Les dades es divideixen en conjunts d'entrenament i prova per avaluar el rendiment del model.
  4. Creació del Model: Es crea un model KNN amb 3 veïns.
  5. Entrenament del Model: El model s'entrena utilitzant les dades d'entrenament.
  6. Prediccions: Es fan prediccions sobre el conjunt de prova.
  7. Avaluació: Es calcula la precisió del model comparant les prediccions amb les etiquetes reals.

Conclusió

L'aprenentatge automàtic és una eina poderosa que permet a les màquines aprendre de les dades i millorar el seu rendiment en tasques específiques. En aquesta introducció, hem cobert els conceptes bàsics i hem vist un exemple pràctic d'aprenentatge supervisat utilitzant K-Nearest Neighbors. A mesura que avancem en el curs, explorarem més algoritmes i tècniques, així com les seves aplicacions en el món real.
