Què són les Xarxes Neuronals?

Les xarxes neuronals són un conjunt d'algoritmes dissenyats per reconèixer patrons. Són el nucli de la intel·ligència artificial i l'aprenentatge profund. Inspirades en l'estructura del cervell humà, les xarxes neuronals estan compostes per capes de neurones artificials que processen dades d'entrada i generen sortides.

Conceptes Clau

  • Neurona Artificial: Unitat bàsica d'una xarxa neuronal que rep una entrada, la processa i produeix una sortida.
  • Capa d'Entrada: La primera capa de la xarxa que rep les dades d'entrada.
  • Capes Ocultes: Capes intermèdies entre la capa d'entrada i la capa de sortida que processen la informació.
  • Capa de Sortida: L'última capa que produeix el resultat final.
  • Pesos i Biasos: Paràmetres ajustables que la xarxa utilitza per aprendre de les dades.
  • Funció d'Activació: Funció que introdueix no linealitat a la xarxa, permetent que aprengui i modelitzi dades complexes.

Arquitectura d'una Xarxa Neuronal

Una xarxa neuronal típica consta de diverses capes de neurones. A continuació, es presenta una taula comparativa dels components clau d'una xarxa neuronal:

| Component | Descripció | |------------------|-----------------------------------------------------------------------------| | Neurona | Unitat bàsica que processa una entrada i produeix una sortida. | | Capa d'Entrada | Rep les dades inicials. | | Capes Ocultes | Processen la informació i extreuen característiques. | | Capa de Sortida | Genera el resultat final. | | Pesos | Coeficients que s'ajusten durant l'entrenament per minimitzar l'error. | | Biasos | Valors addicionals que se sumen a l'entrada d'una neurona. | | Funció d'Activació | Introdueix no linealitat a la xarxa. |

Funcionament d'una Neurona Artificial

Cada neurona realitza una operació matemàtica simple sobre les seves entrades i produeix una sortida. La fórmula bàsica és:

\[ y = f\left(\sum_{i=1}^{n} w_i x_i + b\right) \]

On:

  • \( y \) és la sortida de la neurona.
  • \( f \) és la funció d'activació.
  • \( w_i \) són els pesos.
  • \( x_i \) són les entrades.
  • \( b \) és el bias.

Exemple de Codi

A continuació, es presenta un exemple bàsic d'una xarxa neuronal utilitzant Python i la biblioteca Keras:

import numpy as np
from keras.models import Sequential
from keras.layers import Dense

# Generar dades d'exemple
X = np.array([[0,0], [0,1], [1,0], [1,1]])
y = np.array([[0], [1], [1], [0]])

# Crear el model
model = Sequential()
model.add(Dense(2, input_dim=2, activation='relu'))
model.add(Dense(1, activation='sigmoid'))

# Compilar el model
model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])

# Entrenar el model
model.fit(X, y, epochs=1000, verbose=0)

# Avaluar el model
loss, accuracy = model.evaluate(X, y)
print(f'Precisió: {accuracy * 100:.2f}%')

Explicació del Codi

  1. Importació de Biblioteques: S'importen les biblioteques necessàries, numpy per manejar les dades i keras per construir la xarxa neuronal.
  2. Generació de Dades: Es creen dades d'entrada X i les seves corresponents sortides y.
  3. Creació del Model: Es defineix un model seqüencial amb una capa oculta de 2 neurones i una capa de sortida d'1 neurona.
  4. Compilació del Model: S'especifica la funció de pèrdua, l'optimitzador i la mètrica d'avaluació.
  5. Entrenament del Model: S'entrena el model amb les dades durant 1000 èpoques.
  6. Avaluació del Model: S'avalua la precisió del model amb les dades d'entrada.

Conclusió

Les xarxes neuronals són una eina poderosa en el camp de la intel·ligència artificial, capaces d'aprendre i generalitzar a partir de dades. Comprendre els components bàsics i el funcionament d'una neurona artificial és fonamental per avançar en l'estudi de xarxes neuronals més complexes i aplicacions pràctiques en l'aprenentatge profund.
