Introducció als Algorismes Genètics

Els algorismes genètics (AG) són una tècnica d'optimització inspirada en el procés de selecció natural de Charles Darwin. Utilitzen conceptes de genètica i evolució per resoldre problemes complexos de cerca i optimització.

Conceptes Clau

  • Població: Conjunt de possibles solucions al problema.
  • Cromosoma: Representació d'una solució individual.
  • Gen: Element d'un cromosoma que representa una característica de la solució.
  • Fitness: Mesura de la qualitat d'una solució.
  • Selecció: Procés de triar els millors cromosomes per reproduir-se.
  • Encreuament (Crossover): Combinació de dos cromosomes per produir descendència.
  • Mutació: Alteració aleatòria d'un o més gens en un cromosoma.
  • Generació: Iteració de l'algorisme que produeix una nova població.

Funcionament dels Algorismes Genètics

  1. Inicialització: Crear una població inicial de cromosomes aleatoris.
  2. Avaluació: Calcular el fitness de cada cromosoma en la població.
  3. Selecció: Triar cromosomes per reproduir-se basant-se en el seu fitness.
  4. Encreuament: Combinar parells de cromosomes seleccionats per crear nous cromosomes.
  5. Mutació: Aplicar mutacions aleatòries a alguns cromosomes.
  6. Reemplaçament: Formar una nova població reemplaçant l'antiga amb els nous cromosomes.
  7. Iteració: Repetir els passos 2-6 fins que es compleixi un criteri d'aturada (per exemple, un nombre fix de generacions o una solució òptima).

Exemple de Codi en Python

A continuació, es presenta un exemple bàsic d'un algorisme genètic en Python per maximitzar una funció simple.

import random

# Funció de fitness
def fitness_function(x):
    return x**2

# Crear una població inicial
def create_population(size, lower_bound, upper_bound):
    return [random.uniform(lower_bound, upper_bound) for _ in range(size)]

# Selecció per torneig
def selection(population, fitnesses):
    selected = []
    for _ in range(len(population)):
        i, j = random.sample(range(len(population)), 2)
        selected.append(population[i] if fitnesses[i] > fitnesses[j] else population[j])
    return selected

# Encreuament d'un punt
def crossover(parent1, parent2):
    point = random.randint(1, len(parent1) - 1)
    return parent1[:point] + parent2[point:]

# Mutació
def mutate(chromosome, mutation_rate, lower_bound, upper_bound):
    if random.random() < mutation_rate:
        return random.uniform(lower_bound, upper_bound)
    return chromosome

# Algorisme genètic
def genetic_algorithm(pop_size, lower_bound, upper_bound, generations, mutation_rate):
    population = create_population(pop_size, lower_bound, upper_bound)
    for generation in range(generations):
        fitnesses = [fitness_function(ind) for ind in population]
        population = selection(population, fitnesses)
        next_population = []
        for i in range(0, len(population), 2):
            parent1, parent2 = population[i], population[i+1]
            child1, child2 = crossover(parent1, parent2), crossover(parent2, parent1)
            next_population.extend([mutate(child1, mutation_rate, lower_bound, upper_bound),
                                    mutate(child2, mutation_rate, lower_bound, upper_bound)])
        population = next_population
    best_solution = max(population, key=fitness_function)
    return best_solution

# Paràmetres
pop_size = 10
lower_bound = -10
upper_bound = 10
generations = 20
mutation_rate = 0.1

# Executar l'algorisme genètic
best_solution = genetic_algorithm(pop_size, lower_bound, upper_bound, generations, mutation_rate)
print(f"La millor solució trobada és: {best_solution} amb un fitness de {fitness_function(best_solution)}")

Conclusió

Els algorismes genètics són una poderosa eina d'optimització que pot aplicar-se a una àmplia varietat de problemes. La seva capacitat per explorar grans espais de cerca i trobar solucions òptimes o properes a l'òptim els fa especialment útils en el camp de la intel·ligència artificial. A través de la selecció, l'encreuament i la mutació, els algorismes genètics imiten el procés d'evolució natural per millorar iterativament les solucions a un problema donat.
