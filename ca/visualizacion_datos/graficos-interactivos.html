Introducció als Gràfics Interactius

Els gràfics interactius són una eina poderosa en la visualització de dades, permetent als usuaris explorar i analitzar dades de manera dinàmica. A diferència dels gràfics estàtics, els gràfics interactius permeten la manipulació directa del gràfic per obtenir informació més detallada i personalitzada.

  • Definició: Gràfics que permeten la interacció de l'usuari per explorar dades.
  • Beneficis:
    • Millora la comprensió de les dades.
    • Facilita la identificació de patrons i tendències.
    • Permet la personalització i l'anàlisi en temps real.

Exemples de Gràfics Interactius

Exemple amb D3.js

D3.js és una de les biblioteques més flexibles i poderoses per crear gràfics interactius a la web. A continuació, es mostra un exemple bàsic de com crear un gràfic de barres interactiu amb D3.js.

<!DOCTYPE html>
<html>
<head>
  <script src="https://d3js.org/d3.v6.min.js"></script>
</head>
<body>
  <div id="chart"></div>
  <script>
    const data = [30, 86, 168, 281, 303, 365];
    const width = 500;
    const height = 300;
    const barWidth = width / data.length;

    const svg = d3.select("#chart")
                  .append("svg")
                  .attr("width", width)
                  .attr("height", height);

    svg.selectAll("rect")
       .data(data)
       .enter()
       .append("rect")
       .attr("width", barWidth - 1)
       .attr("height", d => d)
       .attr("x", (d, i) => i * barWidth)
       .attr("y", d => height - d)
       .on("mouseover", function(event, d) {
         d3.select(this).attr("fill", "orange");
       })
       .on("mouseout", function(event, d) {
         d3.select(this).attr("fill", "steelblue");
       })
       .attr("fill", "steelblue");
  </script>
</body>
</html>

Exemple amb Plotly en Python

Plotly és una altra biblioteca popular que permet crear gràfics interactius fàcilment. A continuació, es mostra un exemple de com crear un gràfic de dispersió interactiu amb Plotly en Python.

import plotly.express as px

# Dades d'exemple
df = px.data.iris()

# Crear gràfic de dispersió
fig = px.scatter(df, x='sepal_width', y='sepal_length', color='species',
                 title='Gràfic de Dispersió Interactiu')

# Mostrar gràfic
fig.show()

Exercicis Pràctics

Per consolidar l'aprenentatge, es proposen els següents exercicis:

  1. Exercici amb D3.js: Modifica l'exemple de D3.js perquè els colors de les barres canviïn a un color diferent en passar el ratolí sobre elles.
  2. Exercici amb Plotly: Crea un gràfic de línies interactiu utilitzant Plotly en Python amb un conjunt de dades de la teva elecció. Afegeix etiquetes i títols per millorar la comprensió del gràfic.

Conclusió

Els gràfics interactius són una eina essencial en la visualització de dades moderna. Permeten als usuaris explorar i analitzar dades de manera més efectiva i dinàmica. Amb eines com D3.js i Plotly, és possible crear gràfics interactius que milloren la comprensió i la presa de decisions basada en dades. Practicar amb aquestes eines i biblioteques ajudarà a desenvolupar habilitats valuoses en la visualització de dades.
