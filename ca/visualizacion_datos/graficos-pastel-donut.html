Introducció

Els gràfics de pastís i donut són eines visuals populars en la visualització de dades, especialment útils per mostrar la proporció de diferents categories dins d'un conjunt de dades. Aquest tema cobrirà els conceptes bàsics, les diferències entre ambdós tipus de gràfics, i com crear-los i interpretar-los.

Conceptes Clau

  • Gràfic de Pastís: Representa dades en forma d'un cercle dividit en sectors, on cada sector mostra una proporció del total.
  • Gràfic de Donut: Similar al gràfic de pastís, però amb un forat al centre, el que permet una millor visualització de múltiples sèries de dades.

Avantatges i Desavantatges

| Avantatges | Desavantatges | |---------------------------|-----------------------------------| | Fàcil d'entendre | Difícil de llegir amb moltes categories | | Visualment atractiu | No mostra canvis en el temps | | Útil per proporcions | Comparacions precises són difícils |

Creació de Gràfics de Pastís

Exemple en Python amb Matplotlib

import matplotlib.pyplot as plt

# Dades
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Ressalta el primer sector

# Crear gràfic de pastís
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)
plt.axis('equal')  # Assegura que el gràfic sigui un cercle
plt.show()
  • labels: Etiquetes per a cada sector.
  • sizes: Mida de cada sector.
  • colors: Colors per a cada sector.
  • explode: Desplaça un sector per ressaltar-lo.
  • autopct: Format per mostrar percentatges.
  • shadow: Afegeix ombra per efecte visual.
  • startangle: Angle d'inici del primer sector.

Creació de Gràfics de Donut

Exemple en Python amb Matplotlib

import matplotlib.pyplot as plt

# Dades
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Ressalta el primer sector

# Crear gràfic de pastís
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)

# Afegir cercle blanc al centre per crear l'efecte de donut
centre_circle = plt.Circle((0,0),0.70,fc='white')
fig = plt.gcf()
fig.gca().add_artist(centre_circle)

plt.axis('equal')  # Assegura que el gràfic sigui un cercle
plt.show()
  • centre_circle: Cercle blanc al centre per crear l'efecte de donut.

Exercicis Pràctics

  1. Crear un gràfic de pastís: Utilitza un conjunt de dades propi per crear un gràfic de pastís que mostri les proporcions de diferents categories.
  2. Modificar un gràfic de pastís: Afegeix etiquetes, colors personalitzats i ressalta un sector específic.
  3. Crear un gràfic de donut: Converteix el teu gràfic de pastís en un gràfic de donut afegint un cercle al centre.

Conclusió

Els gràfics de pastís i donut són eines visuals efectives per mostrar proporcions dins d'un conjunt de dades. Tot i que tenen limitacions, la seva simplicitat i atractiu visual els fan útils en moltes situacions. Amb la pràctica, podràs crear i personalitzar aquests gràfics per comunicar les teves dades de manera clara i efectiva.
