Introducció

Els gràfics de línies i àrees són eines fonamentals en la visualització de dades. Permeten representar dades contínues al llarg del temps o qualsevol altra dimensió contínua, facilitant la identificació de tendències, patrons i anomalies.

Gràfics de Línies

Conceptes Clau

  • Eix X (Horitzontal): Generalment representa el temps o una seqüència de categories.
  • Eix Y (Vertical): Representa els valors de la variable que s'està mesurant.
  • Punts de Dades: Cada punt en el gràfic representa un valor en el temps.
  • Línia de Connexió: Connecta els punts de dades per mostrar la tendència.

Exemple de Codi

A continuació, es mostra un exemple de com crear un gràfic de línies utilitzant Python i la biblioteca matplotlib.

import matplotlib.pyplot as plt

# Dades d'exemple
mesos = ['Gener', 'Febrer', 'Març', 'Abril', 'Maig']
vendes = [150, 200, 250, 300, 350]

# Crear el gràfic de línies
plt.plot(mesos, vendes, marker='o', linestyle='-', color='b')

# Afegir títols i etiquetes
plt.title('Vendes Mensuals')
plt.xlabel('Mesos')
plt.ylabel('Vendes')

# Mostrar el gràfic
plt.show()

Explicació del Codi

  • Importació de la Biblioteca: import matplotlib.pyplot as plt
  • Dades d'Exemple: Llistes mesos i vendes contenen les dades a visualitzar.
  • Creació del Gràfic: plt.plot crea el gràfic de línies.
  • Personalització: marker='o' afegeix marcadors en els punts de dades, linestyle='-' defineix l'estil de la línia, i color='b' estableix el color blau.
  • Etiquetes i Títols: plt.title, plt.xlabel, i plt.ylabel afegeixen títols i etiquetes als eixos.
  • Mostrar el Gràfic: plt.show() mostra el gràfic en pantalla.

Gràfics d'Àrees

Conceptes Clau

  • Àrea Sota la Línia: L'àrea entre la línia i l'eix X està plena, cosa que facilita la comparació de volums.
  • Transparència: Sovint s'utilitza per superposar múltiples àrees i facilitar la comparació.

Exemple de Codi

A continuació, es mostra un exemple de com crear un gràfic d'àrees utilitzant Python i la biblioteca matplotlib.

import matplotlib.pyplot as plt

# Dades d'exemple
mesos = ['Gener', 'Febrer', 'Març', 'Abril', 'Maig']
vendes = [150, 200, 250, 300, 350]

# Crear el gràfic d'àrees
plt.fill_between(mesos, vendes, color='skyblue', alpha=0.4)

# Afegir títols i etiquetes
plt.title('Vendes Mensuals')
plt.xlabel('Mesos')
plt.ylabel('Vendes')

# Mostrar el gràfic
plt.show()

Explicació del Codi

  • Importació de la Biblioteca: import matplotlib.pyplot as plt
  • Dades d'Exemple: Llistes mesos i vendes contenen les dades a visualitzar.
  • Creació del Gràfic d'Àrees: plt.fill_between crea el gràfic d'àrees, omplint l'àrea sota la línia.
  • Personalització: color='skyblue' estableix el color de l'àrea, i alpha=0.4 ajusta la transparència.
  • Etiquetes i Títols: plt.title, plt.xlabel, i plt.ylabel afegeixen títols i etiquetes als eixos.
  • Mostrar el Gràfic: plt.show() mostra el gràfic en pantalla.

Comparació entre Gràfics de Línies i Àrees

| Característica | Gràfic de Línies | Gràfic d'Àrees | |----------------------|------------------|----------------| | Visualització | Línia contínua | Àrea plena | | Ús Comú | Tendències | Volums | | Superposició | Menys clar | Més clar amb transparència | | Interpretació | Directa | Comparativa |

Conclusió

Els gràfics de línies i àrees són eines poderoses per a la visualització de dades contínues. Els gràfics de línies són ideals per mostrar tendències al llarg del temps, mentre que els gràfics d'àrees són útils per comparar volums i visualitzar acumulacions. Comprendre quan i com utilitzar cada tipus de gràfic és essencial per a una visualització de dades efectiva i clara.
