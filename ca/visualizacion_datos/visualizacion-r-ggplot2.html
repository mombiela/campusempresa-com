Introducció a ggplot2

ggplot2 és un paquet de R dissenyat per crear gràfics d'alta qualitat de manera eficient i flexible. És part de l'ecosistema tidyverse i es basa en la gramàtica de gràfics, la qual cosa permet construir gràfics de manera sistemàtica.

  • Gramàtica de gràfics: Concepte que permet construir gràfics mitjançant la combinació de components com dades, capes, escales i coordenades.
  • Tidyverse: Conjunt de paquets de R dissenyats per a la ciència de dades, que inclou ggplot2, dplyr, tidyr, entre altres.

Instal·lació i configuració

Per utilitzar ggplot2, primer necessites instal·lar-lo i carregar-lo en el teu entorn de R.

# Instal·lació del paquet ggplot2
install.packages("ggplot2")

# Carregar el paquet ggplot2
library(ggplot2)

Estructura bàsica d'un gràfic amb ggplot2

L'estructura bàsica d'un gràfic en ggplot2 es construeix utilitzant la funció ggplot(), seguida de capes addicionals que especifiquen els tipus de gràfics i les estètiques.

  • ggplot(): Funció principal que inicia un gràfic.
  • aes(): Funció que defineix les estètiques del gràfic, com els eixos x i y.
  • geom_*: Funcions que afegeixen capes de geometria al gràfic, com punts, línies i barres.

Exemple bàsic: Gràfic de dispersió

A continuació, es mostra un exemple de com crear un gràfic de dispersió utilitzant ggplot2.

# Crear un conjunt de dades d'exemple
data <- data.frame(
  x = rnorm(100),
  y = rnorm(100)
)

# Crear un gràfic de dispersió
ggplot(data, aes(x = x, y = y)) +
  geom_point()

Tipus de gràfics comuns

ggplot2 permet crear una àmplia varietat de gràfics. A continuació, es presenten alguns dels tipus més comuns.

Gràfic de barres

Els gràfics de barres són útils per mostrar la distribució d'una variable categòrica.

# Crear un conjunt de dades d'exemple
data <- data.frame(
  category = factor(c("A", "B", "C")),
  count = c(10, 20, 30)
)

# Crear un gràfic de barres
ggplot(data, aes(x = category, y = count)) +
  geom_bar(stat = "identity")

Histograma

Els histogrames són útils per mostrar la distribució d'una variable numèrica.

# Crear un conjunt de dades d'exemple
data <- data.frame(
  value = rnorm(100)
)

# Crear un histograma
ggplot(data, aes(x = value)) +
  geom_histogram(binwidth = 0.5)

Gràfic de línies

Els gràfics de línies són útils per mostrar tendències al llarg del temps.

# Crear un conjunt de dades d'exemple
data <- data.frame(
  time = 1:100,
  value = cumsum(rnorm(100))
)

# Crear un gràfic de línies
ggplot(data, aes(x = time, y = value)) +
  geom_line()

Personalització de gràfics

ggplot2 ofereix una àmplia gamma d'opcions per personalitzar els gràfics, incloent colors, temes i etiquetes.

Canviar colors

Pots canviar els colors dels elements del gràfic utilitzant la funció scale_*_manual().

# Crear un gràfic de dispersió amb colors personalitzats
ggplot(data, aes(x = x, y = y, color = factor(x > 0))) +
  geom_point() +
  scale_color_manual(values = c("red", "blue"))

Afegir etiquetes i títols

Per afegir títols i etiquetes als eixos, pots utilitzar les funcions labs() i ggtitle().

# Crear un gràfic de dispersió amb etiquetes i títol
ggplot(data, aes(x = x, y = y)) +
  geom_point() +
  labs(x = "Eix X", y = "Eix Y") +
  ggtitle("Gràfic de Dispersió")

Aplicar temes

ggplot2 inclou diversos temes predefinits que pots aplicar per canviar l'aspecte general del gràfic.

# Crear un gràfic de dispersió amb un tema personalitzat
ggplot(data, aes(x = x, y = y)) +
  geom_point() +
  theme_minimal()

Conclusió

ggplot2 és una eina poderosa i flexible per a la visualització de dades en R. El seu enfocament basat en la gramàtica de gràfics permet construir gràfics complexos de manera sistemàtica i personalitzable. Amb la pràctica, podràs crear visualitzacions efectives que comuniquin els teus descobriments de manera clara i atractiva.
