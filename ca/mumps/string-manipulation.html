Manipulació de cadenes en MUMPS (M)
Gestió d'Arxius ►

La manipulació de cadenes és una habilitat fonamental en MUMPS (M), un llenguatge conegut per les seves potents capacitats de gestió de cadenes. Aquest tema et guiarà des de tècniques bàsiques fins a avançades de manipulació de cadenes en MUMPS.

Introducció a les cadenes en MUMPS

  • Definició: En MUMPS, una cadena és una seqüència de caràcters.
  • Variables de cadena: Les cadenes es guarden en variables i es poden manipular utilitzant diverses funcions integrades.

Operacions bàsiques amb cadenes

Assignació de cadenes

SET str="Hello, World!"
WRITE str
  • Explicació: La comanda SET assigna la cadena "Hello, World!" a la variable str. La comanda WRITE mostra el valor de str.

Concatenació

SET str1="Hello"
SET str2="World"
SET str3=str1_" "_str2
WRITE str3
  • Explicació: L'operador _ concatena str1, un espai, i str2 en str3.

Longitud de la cadena

Utilitzant la funció $LENGTH

SET str="Hello, World!"
SET len=$LENGTH(str)
WRITE len
  • Explicació: La funció $LENGTH retorna la longitud de la cadena str.

Extracció de subcadena

Utilitzant la funció $EXTRACT

SET str="Hello, World!"
SET sub=$EXTRACT(str,1,5)
WRITE sub
  • Explicació: La funció $EXTRACT extreu una subcadena de str començant a la posició 1 i acabant a la posició 5.

Cerca en cadenes

Utilitzant la funció $FIND

SET str="Hello, World!"
SET pos=$FIND(str,"World")
WRITE pos
  • Explicació: La funció $FIND retorna la posició immediatament després de la subcadena "World" en str.

Substitució de cadenes

Utilitzant la funció $TRANSLATE

SET str="Hello, World!"
SET newstr=$TRANSLATE(str,"World","MUMPS")
WRITE newstr
  • Explicació: La funció $TRANSLATE substitueix les ocurrències de "World" per "MUMPS" en str.

Operacions intermèdies amb cadenes

Comparació de cadenes

Utilitzant la funció $ORDER

SET str1="Apple"
SET str2="Banana"
IF str1]str2 WRITE "str1 is greater"
ELSE  WRITE "str2 is greater"
  • Explicació: L'operador ] compara dues cadenes lexicogràficament.

Divisió de cadenes

Utilitzant la funció $PIECE

SET str="apple,banana,cherry"
SET fruit1=$PIECE(str,",",1)
SET fruit2=$PIECE(str,",",2)
SET fruit3=$PIECE(str,",",3)
WRITE fruit1,!,fruit2,!,fruit3
  • Explicació: La funció $PIECE divideix str pel delimitador , i extreu la peça especificada.

Operacions avançades amb cadenes

Manipulacions complexes de cadenes

Combinació de funcions

SET str="apple,banana,cherry"
SET newstr=$TRANSLATE($PIECE(str,",",2),"banana","grape")
WRITE newstr
  • Explicació: Aquest exemple combina $PIECE i $TRANSLATE per extreure "banana" i substituir-lo per "grape".

Expressions regulars (Avançat)

  • Nota: MUMPS no suporta nativament expressions regulars, però es poden aconseguir manipulacions complexes de cadenes utilitzant combinacions de les funcions discutides.

Conclusió

La manipulació de cadenes en MUMPS és una eina poderosa que permet una gestió i transformació eficient de dades de text. Des d'operacions bàsiques com la concatenació i la mesura de longitud fins a tècniques més avançades com les manipulacions complexes, dominar aquestes funcions millorarà significativament les teves habilitats de programació en MUMPS.

  • Pràctica: Practica regularment aquestes funcions per convertir-te en un expert.
  • Explora: Experimenta combinant diferents funcions per resoldre tasques complexes de manipulació de cadenes.

Comprenent i utilitzant aquestes tècniques de manipulació de cadenes, estaràs ben preparat per gestionar una àmplia gamma de tasques de processament de text en MUMPS.
