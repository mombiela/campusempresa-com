Configurar l'entorn MUMPS (Massachusetts General Hospital Utility Multi-Programming System) és el primer pas per començar a programar en MUMPS. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de l'entorn MUMPS al teu sistema.

Instal·lant MUMPS

Prerequisits

Abans d'instal·lar MUMPS, assegura't que el teu sistema compleixi els següents requisits:

  • Un sistema operatiu compatible (Linux, Windows, macOS).
  • Coneixements bàsics de la interfície de línia de comandes (CLI).

Passos d'Instal·lació

En Linux

  1. Actualitza la llista de paquets: 
    sudo apt-get update
  2. Instal·la GT.M (una implementació popular de MUMPS): 
    sudo apt-get install -y fis-gtm
  3. Verifica la instal·lació: 
    gtm
    Hauries de veure el prompt de GT.M (GTM>).

En Windows

  1. Descarrega l'instal·lador de GT.M des del lloc web oficial.
  2. Executa l'instal·lador i segueix les instruccions en pantalla.
  3. Configura les variables d'entorn:
    • Afegeix el directori binari de GT.M al teu PATH.
    • Configura la variable d'entorn gtm_dist al directori d'instal·lació de GT.M.
  4. Verifica la instal·lació obrint una línia de comandes i escrivint: 
    gtm
    Hauries de veure el prompt de GT.M (GTM>).

En macOS

  1. Instal·la Homebrew si encara no ho has fet: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instal·la GT.M utilitzant Homebrew: 
    brew install fis-gtm
  3. Verifica la instal·lació: 
    gtm
    Hauries de veure el prompt de GT.M (GTM>).

Configurant l'Entorn MUMPS

Configurant Directoris

Crea una estructura de directoris per als teus projectes MUMPS. Per exemple:

mkdir -p ~/mumps/projects
mkdir -p ~/mumps/data

Variables d'Entorn

Configura les variables d'entorn necessàries per assegurar que GT.M pugui trobar els teus projectes i fitxers de dades.

En Linux/macOS

Afegeix les següents línies al teu fitxer .bashrc o .zshrc:

export gtm_dist=/usr/lib/fis-gtm/V6.3-003
export gtmroutines="~/mumps/projects"
export gtmgbldir="~/mumps/data/global.gld"

Després, carrega el fitxer per aplicar els canvis:

source ~/.bashrc

En Windows

Configura les variables d'entorn a través de les Propietats del Sistema:

  1. Obre les Propietats del Sistema.
  2. Ves a la pestanya Avançat i fes clic a Variables d'entorn.
  3. Afegeix noves variables:
    • gtm_dist amb el camí al directori d'instal·lació de GT.M.
    • gtmroutines amb el camí al teu directori de projectes.
    • gtmgbldir amb el camí al teu directori global.

Executant el Teu Primer Programa MUMPS

Creant un Programa Simple

  1. Crea un nou fitxer anomenat hello.m al teu directori de projectes: 
    nano ~/mumps/projects/hello.m
  2. Escriu un programa MUMPS simple: 
    hello
write "Hello, MUMPS!",!
quit
  3. Guarda i surt de l'editor.

Executant el Programa

  1. Inicia l'entorn GT.M: 
    gtm
  2. Executa el teu programa: 
    GTM>do ^hello
Hello, MUMPS!
GTM>

Conclusió

Configurar l'entorn MUMPS implica instal·lar la implementació GT.M, configurar els directoris i les variables d'entorn necessàries, i executar el teu primer programa MUMPS. Amb l'entorn configurat, ara estàs preparat per aprofundir en la programació MUMPS. Aquesta configuració fonamental assegura que tinguis un entorn estable i consistent per desenvolupar i provar les teves aplicacions MUMPS.
