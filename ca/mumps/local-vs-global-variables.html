Introducció

En MUMPS (M), entendre la diferència entre variables locals i globals és crucial per a una programació efectiva. Aquesta secció cobrirà les definicions, l'ús i les diferències entre variables locals i globals, proporcionant exemples i exercicis per solidificar la teva comprensió.

Variables locals

Les variables locals en MUMPS són temporals i existeixen només dins del context del procés o rutina actual. No són accessibles fora de la rutina o procés en què es defineixen.

Conceptes clau

  • Abast: Les variables locals estan limitades a la rutina o procés.
  • Durada: Existeixen només durant la durada de la rutina o procés.
  • Sintaxi: Les variables locals normalment es nomenen sense cap prefix especial.

Exemple

MyRoutine
    NEW X,Y
    SET X=10
    SET Y=20
    WRITE "X: ", X, " Y: ", Y, !
    QUIT

En aquest exemple:

  • X i Y són variables locals.
  • Es defineixen i s'utilitzen dins de la rutina MyRoutine.
  • Deixen d'existir un cop la rutina finalitza l'execució.

Variables globals

Les variables globals en MUMPS són persistents i poden ser accedides per qualsevol rutina o procés. S'emmagatzemen a la base de dades i romanen disponibles fins i tot després que el procés que les va crear hagi acabat.

Conceptes clau

  • Abast: Les variables globals són accessibles a través de diferents rutines i processos.
  • Durada: Persisten fins que s'eliminen explícitament.
  • Sintaxi: Les variables globals es prefixen amb un accent circumflex (^).

Exemple

MyGlobalRoutine
    SET ^GlobalX=100
    SET ^GlobalY=200
    WRITE "^GlobalX: ", ^GlobalX, " ^GlobalY: ", ^GlobalY, !
    QUIT

En aquest exemple:

  • ^GlobalX i ^GlobalY són variables globals.
  • S'emmagatzemen a la base de dades i poden ser accedides per qualsevol rutina.

Comparació de variables locals i globals

| Característica | Variables locals | Variables globals | |----------------|----------------------------|----------------------------| | Abast | Limitat a rutina/procés | Accessible a través de rutines| | Durada | Durada de rutina/procés | Persistent fins a ser eliminada| | Sintaxi | Sense prefix especial | Prefixades amb ^ | | Emmagatzematge | En memòria | A la base de dades |

Exemple de codi: Ús de variables locals i globals

MyMixedRoutine
    NEW LocalA, LocalB
    SET LocalA=5
    SET LocalB=10
    SET ^GlobalA=15
    SET ^GlobalB=20
    WRITE "LocalA: ", LocalA, " LocalB: ", LocalB, !
    WRITE "^GlobalA: ", ^GlobalA, " ^GlobalB: ", ^GlobalB, !
    QUIT

En aquest exemple:

  • LocalA i LocalB són variables locals.
  • ^GlobalA i ^GlobalB són variables globals.
  • La rutina demostra l'ús simultani de variables locals i globals.

Exercicis

  1. Exercici 1: Crea una rutina que defineixi i utilitzi variables locals per realitzar un càlcul simple (per exemple, suma o multiplicació).
  2. Exercici 2: Crea una rutina que defineixi i utilitzi variables globals per emmagatzemar i recuperar dades.
  3. Exercici 3: Modifica la rutina de l'Exercici 1 per utilitzar també variables globals, i compara els resultats.

Conclusió

Entendre les diferències entre variables locals i globals en MUMPS és essencial per a una programació efectiva. Les variables locals són temporals i limitades en abast, mentre que les variables globals són persistents i accessibles a través de rutines. En dominar aquests conceptes, podràs escriure codi MUMPS més eficient i organitzat.
