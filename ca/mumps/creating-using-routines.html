Introducció a les Rutines en MUMPS

Les rutines en MUMPS (Massachusetts General Hospital Utility Multi-Programming System) són blocs fonamentals per organitzar i executar codi. Aquesta secció et guiarà des dels conceptes bàsics de la creació de rutines simples fins a tècniques avançades per gestionar-les i optimitzar-les.

Conceptes Bàsics de les Rutines

  • Definició: Una rutina en MUMPS és una seqüència de comandes emmagatzemades en un fitxer que es poden executar com una unitat.
  • Convencions de Nomenclatura: Els noms de les rutines han de començar amb una lletra i poden incloure lletres, números i guions baixos, però no han de superar els 8 caràcters.

Exemple d'una Rutina Simple

HELLO ; Una rutina simple per imprimir "Hello, World!"
    WRITE "Hello, World!",!
    QUIT
  • Explicació:
    • HELLO és el nom de la rutina.
    • ; inicia un comentari.
    • WRITE "Hello, World!",! imprimeix la cadena "Hello, World!" seguida d'una nova línia.
    • QUIT finalitza la rutina.

Creació de Rutines

  • Edició: Les rutines es creen i editen típicament utilitzant un editor de text.
  • Desament: Desa la rutina amb una extensió .m.

Exemple: Creació d'una Rutina per Sumar Dos Nombres

ADDNUM ; Rutina per sumar dos nombres
    NEW A,B,RESULT
    READ "Enter first number: ",A
    READ "Enter second number: ",B
    SET RESULT=A+B
    WRITE "The result is: ",RESULT,!
    QUIT
  • Explicació:
    • NEW A,B,RESULT declara noves variables locals.
    • READ "Enter first number: ",A demana a l'usuari que introdueixi el primer nombre.
    • READ "Enter second number: ",B demana a l'usuari que introdueixi el segon nombre.
    • SET RESULT=A+B calcula la suma.
    • WRITE "The result is: ",RESULT,! imprimeix el resultat.

Execució de Rutines

  • Execució Directa: Des del prompt de MUMPS, pots executar una rutina escrivint el seu nom. 
    >DO ^HELLO
  • Utilitzant la Comanda DO: La comanda DO s'utilitza per cridar una rutina. 
    >DO ^ADDNUM

Gestió Avançada de Rutines

  • Paràmetres: Les rutines poden acceptar paràmetres per fer-les més flexibles.
  • Gestió d'Errors: Implementa la gestió d'errors per manejar situacions inesperades.

Exemple: Rutina amb Paràmetres i Gestió d'Errors

ADDNUM(A,B) ; Rutina per sumar dos nombres amb paràmetres
    NEW RESULT
    SET $ETRAP="WRITE ""An error occurred!"",!"
    SET RESULT=A+B
    WRITE "The result is: ",RESULT,!
    QUIT
  • Explicació:
    • ADDNUM(A,B) defineix una rutina que accepta dos paràmetres.
    • SET $ETRAP="WRITE ""An error occurred!"",!" estableix una trampa d'errors per manejar els errors de manera elegant.

Millors Pràctiques

  • Modularitat: Divideix tasques complexes en rutines més petites.
  • Documentació: Utilitza comentaris per documentar el propòsit i la funcionalitat de les rutines.
  • Proves: Prova exhaustivament les rutines per assegurar-te que funcionen com s'espera.

Conclusió

Les rutines són essencials per estructurar i reutilitzar codi en MUMPS. Entenent com crear, executar i gestionar rutines, pots escriure programes més eficients i mantenibles. Practica creant rutines simples i incorpora gradualment funcions avançades com paràmetres i gestió d'errors per millorar les teves habilitats.
