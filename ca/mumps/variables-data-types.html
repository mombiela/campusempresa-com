Introducció

En aquesta secció, cobrirem els conceptes bàsics de les variables i els tipus de dades en MUMPS (M). Entendre aquests conceptes fonamentals és crucial per escriure codi MUMPS efectiu i eficient. Començarem amb una introducció a les variables, seguida d'una exploració dels diferents tipus de dades disponibles en MUMPS.

Variables

Les variables en MUMPS s'utilitzen per emmagatzemar dades que es poden manipular i recuperar durant l'execució d'un programa. A diferència de molts altres llenguatges de programació, MUMPS no requereix la declaració explícita de variables abans del seu ús.

Conceptes Clau

  • Declaració Implícita: Les variables es creen quan se'ls assigna un valor per primera vegada.
  • Sensibilitat a Majúscules i Minúscules: Els noms de les variables en MUMPS són sensibles a majúscules i minúscules.
  • Convencions de Nomenclatura: Els noms de les variables han de començar amb una lletra i poden ser seguits per lletres, números o el caràcter de subratllat.

Exemples

SET name="John Doe"
SET age=30
SET _score=95

En els exemples anteriors:

  • name és una variable que emmagatzema un valor de cadena.
  • age és una variable que emmagatzema un valor enter.
  • _score és una variable que emmagatzema un valor enter.

Tipus de Dades

MUMPS admet diversos tipus de dades, però principalment tracta amb cadenes i números. El llenguatge està dissenyat per manejar dades dinàmicament, de manera que el tipus de dades que una variable conté pot canviar durant l'execució.

Tipus de Dades Clau

  • Cadena: Una seqüència de caràcters.
  • Número: Pot ser un enter o un número de punt flotant.

Tipus de Dades de Cadena

Les cadenes en MUMPS estan tancades entre cometes dobles. Poden contenir qualsevol caràcter, incloent números i caràcters especials.

Exemples

SET greeting="Hello, World!"
SET address="123 Main St."

Tipus de Dades de Número

Els números en MUMPS poden ser enters o números de punt flotant. No estan tancats entre cometes.

Exemples

SET count=100
SET price=19.99

Tipatge Dinàmic

Les variables de MUMPS poden canviar de tipus dinàmicament en funció del valor assignat.

Exemple

SET var="123"  ; var és una cadena
SET var=123    ; var ara és un número

Operacions amb Variables

MUMPS permet diverses operacions amb variables, incloent operacions aritmètiques, concatenació de cadenes i operacions lògiques.

Operacions Aritmètiques

Podeu realitzar operacions aritmètiques estàndard amb variables numèriques.

Exemples

SET a=10
SET b=5
SET sum=a+b       ; sum és 15
SET difference=a-b ; difference és 5
SET product=a*b   ; product és 50
SET quotient=a/b  ; quotient és 2

Concatenació de Cadenes

Les cadenes es poden concatenar utilitzant l'operador de subratllat (_).

Exemple

SET firstName="John"
SET lastName="Doe"
SET fullName=firstName_" "_lastName  ; fullName és "John Doe"

Operacions Lògiques

Es poden realitzar operacions lògiques utilitzant operadors lògics estàndard.

Exemples

SET x=1
SET y=0
SET andResult=x&y  ; andResult és 0 (fals)
SET orResult=x!y   ; orResult és 1 (cert)

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en MUMPS. Hem après que les variables en MUMPS es declaren implícitament i poden canviar de tipus dinàmicament. També hem explorat els principals tipus de dades en MUMPS: cadenes i números, i hem vist com realitzar diverses operacions amb aquestes variables.

Entendre aquests conceptes fonamentals és essencial per progressar cap a temes més avançats en la programació MUMPS. Practiqueu creant i manipulant variables per sentir-vos més còmodes amb aquests conceptes.
