Introducció als Arrays

Els arrays en MUMPS (també conegut com a M) són una estructura de dades fonamental utilitzada per emmagatzemar i manipular col·leccions de dades. A diferència dels arrays en molts altres llenguatges de programació, els arrays de MUMPS són associatius, és a dir, estan indexats per claus que poden ser cadenes o números.

Conceptes Clau

  • Arrays Associatius: Arrays indexats per claus, que poden ser cadenes o números.
  • Subscripts: Les claus utilitzades per indexar arrays.
  • Arrays Globals: Arrays que s'emmagatzemen en disc i persisteixen entre sessions.
  • Arrays Locals: Arrays que existeixen només durant l'execució d'un programa.

Treballant amb Arrays Locals

Els arrays locals són temporals i existeixen només durant l'execució d'un programa. Són útils per emmagatzemar dades que no necessiten persistir.

Creant i Accedint a Arrays Locals

Per crear i accedir a arrays locals, utilitza la següent sintaxi:

SET array("key1") = "value1"
SET array("key2") = "value2"
WRITE array("key1"), !
WRITE array("key2"), !

Exemple

SET fruits("apple") = 10
SET fruits("banana") = 20
WRITE "Apple count: ", fruits("apple"), !
WRITE "Banana count: ", fruits("banana"), !

Iterant sobre Arrays Locals

Pots iterar sobre arrays locals utilitzant el bucle FOR:

SET fruits("apple") = 10
SET fruits("banana") = 20
SET fruits("cherry") = 30

SET key = ""
FOR  SET key=$ORDER(fruits(key)) QUIT:key=""  DO
. WRITE key, ": ", fruits(key), !

Treballant amb Arrays Globals

Els arrays globals s'emmagatzemen en disc i persisteixen entre sessions. S'identifiquen per un prefix de caret (^).

Creant i Accedint a Arrays Globals

Per crear i accedir a arrays globals, utilitza la següent sintaxi:

SET ^globalArray("key1") = "value1"
SET ^globalArray("key2") = "value2"
WRITE ^globalArray("key1"), !
WRITE ^globalArray("key2"), !

Exemple

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
WRITE "Apple inventory: ", ^inventory("apple"), !
WRITE "Banana inventory: ", ^inventory("banana"), !

Iterant sobre Arrays Globals

Pots iterar sobre arrays globals de manera similar als arrays locals:

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
SET ^inventory("cherry") = 300

SET key = ""
FOR  SET key=$ORDER(^inventory(key)) QUIT:key=""  DO
. WRITE key, ": ", ^inventory(key), !

Temes Avançats

Arrays Multidimensionals

MUMPS suporta arrays multidimensionals, que són arrays d'arrays.

Exemple

SET matrix(1,1) = 10
SET matrix(1,2) = 20
SET matrix(2,1) = 30
SET matrix(2,2) = 40

WRITE "Matrix(1,1): ", matrix(1,1), !
WRITE "Matrix(1,2): ", matrix(1,2), !
WRITE "Matrix(2,1): ", matrix(2,1), !
WRITE "Matrix(2,2): ", matrix(2,2), !

Subscripts Dinàmics

Els subscripts en MUMPS poden ser generats dinàmicament i utilitzats per accedir a elements de l'array.

Exemple

SET i = 1
SET j = 2
SET matrix(i,j) = 50
WRITE "Matrix(", i, ",", j, "): ", matrix(i,j), !

Conclusió

Els arrays i els subscripts són característiques poderoses en MUMPS que permeten un emmagatzematge i manipulació de dades flexibles i eficients. Entendre com treballar amb arrays locals i globals, així com conceptes avançats com arrays multidimensionals i subscripts dinàmics, és essencial per a una programació efectiva en MUMPS. Practica creant, accedint i iterant sobre arrays per convertir-te en un expert en l'ús d'aquestes estructures.
