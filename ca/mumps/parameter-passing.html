Introducció

El passatge de paràmetres és un concepte fonamental en programació que et permet passar dades a subrutines o funcions. En MUMPS (M), entendre com passar paràmetres de manera efectiva pot ajudar-te a escriure codi més modular i reutilitzable.

Conceptes Clau

  • Subrutines: Blocs de codi que realitzen tasques específiques i poden ser cridats des d'altres parts del programa.
  • Paràmetres: Dades passades a subrutines per ser utilitzades durant la seva execució.
  • Per Valor: Passar una còpia de les dades.
  • Per Referència: Passar una referència a les dades reals.

Passatge de Paràmetres a Subrutines

En MUMPS, pots passar paràmetres a subrutines utilitzant la comanda DO. Els paràmetres poden ser passats per valor o per referència.

Passatge per Valor

Quan passes paràmetres per valor, es passa una còpia de les dades a la subrutina. Qualsevol canvi fet als paràmetres dins de la subrutina no afecta les dades originals.

Exemple

Main
    NEW X
    SET X=10
    DO Subroutine(X)
    WRITE "Valor de X després de la crida a la subrutina: ", X, !
    QUIT

Subroutine(Value)
    SET Value=20
    WRITE "Valor dins de la subrutina: ", Value, !
    QUIT

Explicació:

  • Main estableix X a 10 i crida Subroutine amb X com a paràmetre.
  • Dins de Subroutine, Value es fixa a 20, però això no afecta X en Main.
  • Sortida: 
    Valor dins de la subrutina: 20
Valor de X després de la crida a la subrutina: 10

Passatge per Referència

Quan passes paràmetres per referència, es passa una referència a les dades reals a la subrutina. Els canvis fets als paràmetres dins de la subrutina afecten les dades originals.

Exemple

Main
    NEW X
    SET X=10
    DO Subroutine(.X)
    WRITE "Valor de X després de la crida a la subrutina: ", X, !
    QUIT

Subroutine(.Value)
    SET Value=20
    WRITE "Valor dins de la subrutina: ", Value, !
    QUIT

Explicació:

  • Main estableix X a 10 i crida Subroutine amb X com a referència.
  • Dins de Subroutine, Value es fixa a 20, cosa que afecta directament X en Main.
  • Sortida: 
    Valor dins de la subrutina: 20
Valor de X després de la crida a la subrutina: 20

Passatge de Paràmetres Avançat

Les tècniques avançades impliquen passar múltiples paràmetres i utilitzar arrays.

Passatge de Múltiples Paràmetres

Pots passar múltiples paràmetres a una subrutina separant-los amb comes.

Exemple

Main
    NEW A,B
    SET A=5, B=10
    DO Subroutine(A,B)
    WRITE "Valors després de la crida a la subrutina: A=", A, ", B=", B, !
    QUIT

Subroutine(Value1, Value2)
    SET Value1=Value1*2
    SET Value2=Value2+5
    WRITE "Valors dins de la subrutina: Value1=", Value1, ", Value2=", Value2, !
    QUIT

Explicació:

  • Main estableix A a 5 i B a 10 i crida Subroutine amb tots dos com a paràmetres.
  • Dins de Subroutine, Value1 i Value2 són modificats.
  • Sortida: 
    Valors dins de la subrutina: Value1=10, Value2=15
Valors després de la crida a la subrutina: A=5, B=10

Passatge d'Arrays

Els arrays poden ser passats per referència a subrutines, permetent-te manipular estructures de dades complexes.

Exemple

Main
    NEW ARR
    SET ARR(1)=1, ARR(2)=2, ARR(3)=3
    DO Subroutine(.ARR)
    WRITE "Array després de la crida a la subrutina: ", !
    ZWRITE ARR
    QUIT

Subroutine(.Array)
    SET Array(1)=Array(1)+10
    SET Array(2)=Array(2)+20
    SET Array(3)=Array(3)+30
    WRITE "Array dins de la subrutina: ", !
    ZWRITE Array
    QUIT

Explicació:

  • Main inicialitza un array ARR i crida Subroutine amb ARR com a referència.
  • Dins de Subroutine, els elements de l'array són modificats.
  • Sortida: 
    Array dins de la subrutina: 
ARR(1)=11
ARR(2)=22
ARR(3)=33
Array després de la crida a la subrutina: 
ARR(1)=11
ARR(2)=22
ARR(3)=33

Conclusió

Entendre el passatge de paràmetres en MUMPS és crucial per escriure codi eficient i modular. Dominant tant el passatge per valor com per referència, així com la gestió de múltiples paràmetres i arrays, pots crear subrutines més flexibles i potents. Practica aquests conceptes amb diversos exemples per solidificar la teva comprensió i millorar les teves habilitats de programació en MUMPS.
