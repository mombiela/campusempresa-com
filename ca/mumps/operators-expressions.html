Introducció

En aquesta secció, explorarem els diversos operadors i expressions utilitzats en MUMPS (M). Entendre aquests conceptes fonamentals és crucial per escriure codi MUMPS efectiu i eficient. Cobrirem operadors aritmètics, lògics i relacionals, així com com construir i avaluar expressions.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics. Els operadors aritmètics comuns en MUMPS inclouen:

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • # : Divisió Entera
  • ** : Exponenciació

Exemples

SET A=10, B=3
WRITE "Suma: ", A+B, !
WRITE "Resta: ", A-B, !
WRITE "Multiplicació: ", A*B, !
WRITE "Divisió: ", A/B, !
WRITE "Divisió Entera: ", A#B, !
WRITE "Exponenciació: ", A**B, !

Sortida:

Suma: 13
Resta: 7
Multiplicació: 30
Divisió: 3.333333
Divisió Entera: 3
Exponenciació: 1000

Operadors Lògics

Els operadors lògics s'utilitzen per realitzar operacions lògiques. Els operadors lògics comuns en MUMPS inclouen:

  • & : AND Lògic
  • ! : OR Lògic
  • ' : NOT Lògic

Exemples

SET X=1, Y=0
WRITE "AND Lògic: ", X&Y, !
WRITE "OR Lògic: ", X!Y, !
WRITE "NOT Lògic: ", 'X, !

Sortida:

AND Lògic: 0
OR Lògic: 1
NOT Lògic: 0

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors. Els operadors relacionals comuns en MUMPS inclouen:

  • = : Igual a
  • `'= : Diferent de
  • < : Menor que
  • > : Major que
  • <= : Menor o igual a
  • >= : Major o igual a

Exemples

SET A=5, B=10
WRITE "Igual a: ", A=B, !
WRITE "Diferent de: ", A'=B, !
WRITE "Menor que: ", A<B, !
WRITE "Major que: ", A>B, !
WRITE "Menor o igual a: ", A<=B, !
WRITE "Major o igual a: ", A>=B, !

Sortida:

Igual a: 0
Diferent de: 1
Menor que: 1
Major que: 0
Menor o igual a: 1
Major o igual a: 0

Combinació d'Operadors en Expressions

Els operadors es poden combinar per formar expressions complexes. L'ordre d'operacions (precedència) en MUMPS segueix les regles matemàtiques estàndard:

  1. Parèntesis ()
  2. Exponenciació **
  3. Multiplicació * i Divisió /
  4. Suma + i Resta -
  5. Operadors relacionals
  6. Operadors lògics

Exemples

SET A=5, B=10, C=2
WRITE "Expressió 1: ", (A+B)*C, !
WRITE "Expressió 2: ", A+B*C, !
WRITE "Expressió 3: ", A>B&(B>C), !
WRITE "Expressió 4: ", (A<B)!('C), !

Sortida:

Expressió 1: 30
Expressió 2: 25
Expressió 3: 0
Expressió 4: 1

Conclusió

Entendre i utilitzar de manera efectiva els operadors i expressions és essencial per programar en MUMPS. Aquesta secció ha cobert els operadors aritmètics, lògics i relacionals bàsics, i com combinar-los per formar expressions complexes. Practiqueu utilitzant aquests operadors en diferents escenaris per esdevenir proficient en l'escriptura de codi MUMPS.
