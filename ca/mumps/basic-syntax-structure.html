Introducció

MUMPS (Massachusetts General Hospital Utility Multi-Programming System), també conegut com a M, és un llenguatge de programació dissenyat per a la indústria sanitària. És conegut per la seva sintaxi única i les seves potents capacitats de base de dades. Aquesta secció t'introduirà a la sintaxi i estructura bàsiques de MUMPS, proporcionant una base per a temes més avançats.

Conceptes Clau

  • Línies i Comandes: El codi MUMPS s'escriu en línies, cadascuna de les quals conté una o més comandes.
  • Variables: Les variables en MUMPS poden ser locals o globals.
  • Tipus de Dades: MUMPS utilitza principalment cadenes i números.
  • Funcions: Les funcions integrades realitzen diverses operacions.
  • Estructures de Control: Les declaracions condicionals i els bucles controlen el flux del programa.

Línies i Comandes

En MUMPS, cada línia de codi pot contenir una o més comandes. Les comandes es separen per espais, i cada comanda pot tenir arguments.

Exemple:

WRITE "Hola, Món!",!
SET X=10
  • WRITE "Hola, Món!",! imprimeix "Hola, Món!" seguit d'un salt de línia.
  • SET X=10 assigna el valor 10 a la variable X.

Variables

MUMPS suporta dos tipus de variables: locals i globals.

  • Variables Locals: Aquestes són temporals i només existeixen durant l'execució d'un programa.
  • Variables Globals: Aquestes s'emmagatzemen a la base de dades i persisteixen entre execucions de programes.

Exemple:

SET localVar="Sóc local"
SET ^globalVar="Sóc global"
  • localVar és una variable local.
  • ^globalVar és una variable global, indicada pel símbol de la caret (^).

Tipus de Dades

MUMPS utilitza principalment dos tipus de dades: cadenes i números. No cal declarar explícitament el tipus.

Exemple:

SET str="Hola"
SET num=123
  • str és una variable cadena.
  • num és una variable numèrica.

Funcions

MUMPS té diverses funcions integrades per a diverses operacions, com la manipulació de cadenes, càlculs matemàtics i gestió de dates/hores.

Exemple:

SET length=$LENGTH("Hola") ; Retorna la longitud de la cadena "Hola"
SET upper=$ZCONVERT("hola","U") ; Converteix "hola" a majúscules
  • $LENGTH retorna la longitud d'una cadena.
  • $ZCONVERT converteix una cadena a majúscules.

Estructures de Control

MUMPS proporciona estructures de control com declaracions condicionals i bucles per controlar el flux del programa.

Declaracions Condicionals

IF X=10 WRITE "X és 10",!
ELSE  WRITE "X no és 10",!
  • IF comprova si X és igual a 10.
  • ELSE s'executa si la condició és falsa.

Bucles

SET i=1
FOR  SET i=$INCREMENT(i) QUIT:i>5  WRITE i,!
  • El bucle FOR incrementa i i imprimeix el seu valor fins que i sigui més gran que 5.

Conclusió

Entendre la sintaxi i estructura bàsiques de MUMPS és crucial per escriure programes efectius. Aquesta secció ha cobert els aspectes essencials, incloent línies i comandes, variables, tipus de dades, funcions i estructures de control. Amb aquesta base, ara estàs preparat per aprofundir en temes més avançats en la programació MUMPS.
