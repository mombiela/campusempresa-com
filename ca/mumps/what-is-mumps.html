Introducció a MUMPS

MUMPS, també conegut com a M, és un llenguatge de programació dissenyat originalment per a la indústria sanitària. És conegut per les seves característiques úniques que el fan especialment adequat per a la gestió de bases de dades i aplicacions en temps real.

Conceptes Clau de MUMPS

  • Llenguatge Integrat de Base de Dades: MUMPS combina tant capacitats de programació com de gestió de bases de dades en un sol llenguatge.
  • Base de Dades Jeràrquica: Utilitza un model de base de dades jeràrquic, que és diferent del model relacional utilitzat per les bases de dades SQL.
  • Orientat a Cadenes: MUMPS és altament eficient en la manipulació d'operacions amb cadenes.
  • Tipatge Dinàmic: Les variables en MUMPS no requereixen declaracions de tipus explícites.
  • Entorn Multiusuari: MUMPS suporta l'accés concurrent per múltiples usuaris, fent-lo ideal per a aplicacions en temps real.

Antecedents Històrics

MUMPS va ser desenvolupat a finals dels anys 60 a l'Hospital General de Massachusetts. El seu objectiu principal era gestionar els registres mèdics de manera eficient, però les seves robustes capacitats de base de dades han portat a la seva adopció en diverses altres indústries.

Sintaxi i Estructura Bàsiques

Exemple de Hello World

Comencem amb un simple programa de "Hello, World!" en MUMPS:

HELLO
  WRITE "Hello, World!", !
  QUIT
  • HELLO: Aquesta és l'etiqueta del programa.
  • WRITE: Aquesta comanda mostra text a la pantalla.
  • QUIT: Aquesta comanda finalitza el programa.

Variables i Tipus de Dades

En MUMPS, les variables són de tipus dinàmic i poden contenir diferents tipus de dades sense declaracions explícites.

SET name="John Doe"
SET age=30
WRITE "Name: ", name, " Age: ", age, !
  • SET: Aquesta comanda assigna un valor a una variable.
  • WRITE: Mostra el valor de les variables.

Operacions Bàsiques

Operacions Aritmètiques

SET a=10, b=5
SET sum=a+b
SET difference=a-b
SET product=a*b
SET quotient=a/b
WRITE "Sum: ", sum, " Difference: ", difference, " Product: ", product, " Quotient: ", quotient, !

Operacions amb Cadenes

SET str1="Hello"
SET str2="World"
SET combined=str1_" "_str2
WRITE combined, !
  • _: El guió baix s'utilitza per a la concatenació de cadenes.

Operacions de Base de Dades

Emmagatzemar Dades

MUMPS utilitza variables globals per emmagatzemar dades de manera persistent.

SET ^Patient(1)="John Doe"
SET ^Patient(2)="Jane Smith"

Recuperar Dades

WRITE ^Patient(1), !
WRITE ^Patient(2), !

Recórrer Dades

SET i=0
FOR  SET i=$ORDER(^Patient(i)) QUIT:i=""  DO
. WRITE "Patient ", i, ": ", ^Patient(i), !
  • $ORDER: Aquesta funció s'utilitza per iterar a través dels índexs d'una matriu global.

Conclusió

MUMPS és un llenguatge potent amb característiques úniques que el fan ideal per a la gestió de bases de dades i aplicacions en temps real. El seu enfocament integrat a la programació i les operacions de bases de dades, combinat amb la seva eficiència en la manipulació de cadenes i el tipatge dinàmic, el converteixen en una eina valuosa en diverses indústries, especialment la sanitària. Entendre els conceptes bàsics de MUMPS, com la seva sintaxi, la gestió de variables i les operacions de bases de dades, proporciona una base sòlida per a una exploració més profunda i un ús avançat del llenguatge.
