Introducció

En aquesta secció, explorarem estructures de control avançades en MUMPS (M). Aquestes estructures permeten un control de flux més sofisticat en els teus programes, permetent-te escriure codi més eficient i llegible. Cobrirem:

  • Construccions de bucles
  • Construccions condicionals
  • Gestió d'errors
  • Tècniques avançades de ramificació

Construccions de Bucles

Bucle For

El bucle For en MUMPS s'utilitza per iterar sobre un rang de valors. Es pot utilitzar amb o sense un rang especificat.

Sintaxi

FOR variable=start:increment:end DO
. ; bloc de codi

Exemple

FOR i=1:1:5 DO
. WRITE "Iteració: ", i, !

Aquest bucle imprimirà els números de l'1 al 5.

Bucle While

El bucle While continua executant-se mentre una condició especificada sigui certa.

Sintaxi

WHILE condition DO
. ; bloc de codi

Exemple

SET i=1
WHILE i<=5 DO
. WRITE "Iteració: ", i, !
. SET i=i+1

Aquest bucle també imprimirà els números de l'1 al 5.

Bucle Do

El bucle Do s'utilitza per executar un bloc de codi repetidament fins que es compleixi una certa condició.

Sintaxi

DO
. ; bloc de codi
QUIT:condition

Exemple

SET i=1
DO
. WRITE "Iteració: ", i, !
. SET i=i+1
QUIT:i>5

Aquest bucle imprimirà els números de l'1 al 5.

Construccions Condicionals

If-Else

La construcció If-Else permet l'execució condicional de blocs de codi.

Sintaxi

IF condition DO
. ; bloc de codi
ELSE  DO
. ; bloc de codi alternatiu

Exemple

SET x=10
IF x>5 DO
. WRITE "x és més gran que 5", !
ELSE  DO
. WRITE "x no és més gran que 5", !

Això imprimirà "x és més gran que 5".

If Imbricat

Pots imbricar declaracions If per a condicions més complexes.

Exemple

SET x=10, y=20
IF x>5 DO
. IF y>15 DO
. . WRITE "x és més gran que 5 i y és més gran que 15", !

Això imprimirà "x és més gran que 5 i y és més gran que 15".

Gestió d'Errors

Comanda Trap

La comanda Trap s'utilitza per gestionar errors de manera elegant.

Sintaxi

SET $ETRAP="DO errorHandler"

Exemple

SET $ETRAP="DO errorHandler"
SET x=10
SET y=0
SET result=x/y ; Això causarà un error de divisió per zero

errorHandler
WRITE "S'ha produït un error: ", $ZE, !
QUIT

Això imprimirà un missatge d'error quan es produeixi un error de divisió per zero.

Tècniques Avançades de Ramificació

Comanda Goto

La comanda Goto permet una ramificació incondicional a una etiqueta especificada.

Sintaxi

GOTO label

Exemple

WRITE "Això serà saltat", !
GOTO skip
WRITE "Això també serà saltat", !
skip
WRITE "Això serà executat", !

Això imprimirà "Això serà executat".

Comanda Quit

La comanda Quit s'utilitza per sortir d'un bloc de codi o una subrutina.

Sintaxi

QUIT

Exemple

DO
. WRITE "Això serà imprès", !
. QUIT
. WRITE "Això no serà imprès", !

Això imprimirà "Això serà imprès".

Conclusió

En aquesta secció, hem cobert estructures de control avançades en MUMPS, incloent construccions de bucles, construccions condicionals, gestió d'errors i tècniques avançades de ramificació. Dominar aquestes construccions et permetrà escriure codi MUMPS més eficient i llegible. Practica aquests exemples i intenta implementar-los en els teus propis projectes per obtenir una comprensió més profunda.
