Introducció

En aquesta secció, cobrirem els conceptes bàsics de les operacions d'entrada i sortida en MUMPS (M). Entendre com gestionar l'entrada i la sortida és crucial per interactuar amb els usuaris i gestionar dades dins dels teus programes.

Conceptes clau

  • Entrada: Llegir dades de l'usuari o d'una altra font.
  • Sortida: Mostrar dades a l'usuari o a una altra destinació.

Operacions d'entrada

Utilitzant la comanda `READ`

La comanda READ s'utilitza per prendre entrada de l'usuari.

Sintaxi

READ variable

Exemple

READ "Introdueix el teu nom: ", name
WRITE "Hola, ", name, "!"

En aquest exemple:

  • La comanda READ demana a l'usuari que introdueixi el seu nom.
  • La comanda WRITE després mostra un missatge de benvinguda utilitzant l'entrada.

Llegint múltiples variables

Pots llegir múltiples variables en una sola comanda READ.

Exemple

READ "Introdueix el teu nom: ", firstName, " Introdueix el teu cognom: ", lastName
WRITE "Hola, ", firstName, " ", lastName, "!"

En aquest exemple:

  • Es demana a l'usuari que introdueixi el seu nom i cognom.
  • La comanda WRITE mostra un missatge de benvinguda utilitzant ambdues entrades.

Operacions de sortida

Utilitzant la comanda `WRITE`

La comanda WRITE s'utilitza per mostrar dades a l'usuari.

Sintaxi

WRITE expression

Exemple

SET greeting = "Hola, Món!"
WRITE greeting

En aquest exemple:

  • La comanda SET assigna la cadena "Hola, Món!" a la variable greeting.
  • La comanda WRITE mostra el valor de greeting.

Formatant la sortida

Pots formatar la sortida utilitzant caràcters especials i expressions.

Exemple

SET age = 25
WRITE "Tens ", age, " anys."

En aquest exemple:

  • La comanda WRITE mostra un missatge formatat que inclou el valor de age.

Combinant entrada i sortida

Programa d'exemple

Aquí tens un programa senzill que combina operacions d'entrada i sortida:

READ "Introdueix la teva edat: ", age
IF age<18 WRITE "Ets menor d'edat."
ELSE  WRITE "Ets major d'edat."

En aquest exemple:

  • La comanda READ demana a l'usuari que introdueixi la seva edat.
  • La instrucció IF comprova si l'edat és menor de 18.
  • Depenent de la condició, la comanda WRITE mostra un missatge apropiat.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les operacions d'entrada i sortida en MUMPS. Has après com utilitzar la comanda READ per prendre entrada de l'usuari i la comanda WRITE per mostrar sortida. També hem explorat com formatar la sortida i combinar operacions d'entrada i sortida en un programa senzill. Dominar aquests conceptes bàsics et permetrà crear programes interactius en MUMPS que puguin comunicar-se eficaçment amb els usuaris.
