Introducció

La depuració és una habilitat crucial per a qualsevol programador, i MUMPS (M) no és una excepció. Aquesta secció et guiarà a través de diverses tècniques de depuració, des de nivells bàsics fins a avançats, per ajudar-te a identificar i solucionar problemes en el teu codi MUMPS de manera eficient.

Tècniques Bàsiques de Depuració

1. Utilitzar Sentències WRITE

  • Concepte: La manera més senzilla de depurar és inserint sentències WRITE en el teu codi per mostrar els valors de les variables i el flux del programa.
  • Exemple: 
    SET X=10
WRITE "Valor de X: ", X, !
SET Y=X+5
WRITE "Valor de Y: ", Y, !

2. Comprovar Codis d'Error

  • Concepte: MUMPS proporciona variables de sistema com $ECODE i $ZERROR per comprovar errors.
  • Exemple: 
    SET $ECODE=""
SET X=10/0
IF $ECODE'="" WRITE "S'ha produït un error: ", $ECODE, !

3. Utilitzar la Comanda BREAK

  • Concepte: La comanda BREAK es pot utilitzar per pausar l'execució i entrar en mode directe per a la depuració interactiva.
  • Exemple: 
    SET X=10
BREAK
SET Y=X+5

Tècniques Intermèdies de Depuració

1. Utilitzar la Comanda ZWRITE

  • Concepte: La comanda ZWRITE proporciona una vista detallada de les variables, incloent els seus noms i valors.
  • Exemple: 
    SET X=10, Y=20
ZWRITE

2. Traçar l'Execució del Programa

  • Concepte: Utilitza la comanda TRACE per seguir l'execució del teu programa pas a pas.
  • Exemple: 
    TRACE
SET X=10
SET Y=X+5

3. Utilitzar $STACK i $STACKTRACE

  • Concepte: Aquestes variables de sistema t'ajuden a entendre la pila de crides i a rastrejar errors.
  • Exemple: 
    DO Subroutine
QUIT

Subroutine
SET $ECODE=",U1,"
IF $ECODE'="" WRITE $STACKTRACE, !
QUIT

Tècniques Avançades de Depuració

1. Utilitzar $ZTRAP per al Maneig d'Errors

  • Concepte: La variable $ZTRAP et permet especificar una etiqueta a la qual saltar quan es produeix un error.
  • Exemple: 
    SET $ZTRAP="ErrorHandler"
SET X=10/0
QUIT

ErrorHandler
WRITE "S'ha produït un error: ", $ZERROR, !
QUIT

2. Depurar amb $ETRAP

  • Concepte: Similar a $ZTRAP, però $ETRAP proporciona més control i és més flexible.
  • Exemple: 
    SET $ETRAP="DO ErrorHandler"
SET X=10/0
QUIT

ErrorHandler
WRITE "S'ha produït un error: ", $ZERROR, !
SET $ECODE=""
QUIT

3. Utilitzar Eines Externes de Depuració

  • Concepte: Alguns entorns proporcionen eines externes de depuració que es poden integrar amb MUMPS.
  • Exemple: Això dependrà de l'eina específica i de l'entorn que estiguis utilitzant.

Conclusió

La depuració és una part essencial de la programació en MUMPS. Dominant aquestes tècniques, des de les sentències bàsiques WRITE fins al maneig avançat d'errors amb $ZTRAP i $ETRAP, pots identificar i resoldre problemes en el teu codi de manera eficient. Practica aquestes metodologies regularment per convertir-te en un expert en la depuració de programes MUMPS.
