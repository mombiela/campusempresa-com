Introducció

En MUMPS (M), les funcions i els procediments són blocs fonamentals que et permeten encapsular i reutilitzar codi. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats, assegurant que entenguis com crear, cridar i gestionar funcions i procediments de manera efectiva.

Conceptes Clau

  • Funcions: Retornen un valor i es poden utilitzar en expressions.
  • Procediments: Realitzen accions però no retornen un valor.
  • Arguments: Valors passats a funcions o procediments.
  • Variables Locals: Variables definides dins d'una funció o procediment.
  • Variables Globals: Variables accessibles a tot el programa.

Creació de Funcions

Les funcions en MUMPS es defineixen utilitzant la sintaxi FunctionName^RoutineName. Retornen un valor utilitzant la comanda QUIT.

Exemple

Sum(a, b)
    QUIT a + b

En aquest exemple, Sum és una funció que pren dos arguments a i b, i retorna la seva suma.

Cridar Funcions

Les funcions es criden utilitzant el seu nom seguit dels arguments entre parèntesis.

Exemple

SET result = $$Sum(3, 5)
WRITE result  ; Output: 8

Aquí, $$Sum(3, 5) crida la funció Sum amb els arguments 3 i 5, i emmagatzema el resultat en result.

Creació de Procediments

Els procediments són similars a les funcions però no retornen un valor. Es defineixen utilitzant la sintaxi ProcedureName^RoutineName.

Exemple

PrintSum(a, b)
    NEW sum
    SET sum = a + b
    WRITE "La suma és: ", sum, !
    QUIT

En aquest exemple, PrintSum és un procediment que calcula la suma de a i b i imprimeix el resultat.

Cridar Procediments

Els procediments es criden utilitzant la comanda DO seguida del nom del procediment i els arguments.

Exemple

DO PrintSum(3, 5)
; Output: La suma és: 8

Aquí, DO PrintSum(3, 5) crida el procediment PrintSum amb els arguments 3 i 5.

Variables Locals i Globals

  • Variables Locals: Definides dins d'una funció o procediment utilitzant la comanda NEW.
  • Variables Globals: Accessibles a tot el programa i no requereixen la comanda NEW.

Exemple

SumWithGlobal(a, b)
    NEW sum
    SET sum = a + b
    SET ^GlobalSum = sum
    QUIT sum

En aquest exemple, sum és una variable local, mentre que ^GlobalSum és una variable global.

Conceptes Avançats

Recursivitat

Les funcions i procediments en MUMPS poden cridar-se a si mateixos, permetent algoritmes recursius.

Exemple

Factorial(n)
    IF n=0 QUIT 1
    QUIT n * $$Factorial(n-1)

Aquest exemple defineix una funció recursiva Factorial que calcula el factorial de n.

Gestió d'Errors

Utilitza les comandes TRY i CATCH per gestionar errors dins de funcions i procediments.

Exemple

SafeDivide(a, b)
    NEW result
    TRY {
        IF b=0 THROW "DivideByZero"
        SET result = a / b
    } CATCH {
        WRITE "Error: Divisió per zero", !
        SET result = "indefinit"
    }
    QUIT result

En aquest exemple, SafeDivide gestiona els errors de divisió per zero de manera elegant.

Resum

  • Les funcions i els procediments són essencials per a la reutilització i organització del codi en MUMPS.
  • Les funcions retornen valors, mentre que els procediments realitzen accions sense retornar valors.
  • Utilitza variables locals per a dades temporals dins de funcions i procediments, i variables globals per a dades que necessiten ser accessibles a tot el programa.
  • Conceptes avançats com la recursivitat i la gestió d'errors poden millorar la robustesa i funcionalitat del teu codi.

En dominar aquests conceptes, estaràs ben preparat per escriure codi MUMPS eficient i mantenible.
