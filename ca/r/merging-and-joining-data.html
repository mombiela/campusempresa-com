Introducció

Fusionar i unir dades són tasques essencials en l'anàlisi i manipulació de dades. En R, aquestes operacions et permeten combinar conjunts de dades basats en columnes comunes o índexs de files. Aquesta secció cobrirà diversos mètodes per fusionar i unir dades, des de tècniques bàsiques fins a avançades.

Conceptes Clau

  • Fusionar: Combinar conjunts de dades basats en columnes comunes.
  • Unir: Similar a fusionar però sovint es refereix a operacions tipus SQL.
  • Inner Join: Retorna només les files amb claus coincidents en ambdós conjunts de dades.
  • Outer Join: Retorna totes les files de tots dos conjunts de dades, amb NA per a coincidències que falten.
  • Left Join: Retorna totes les files del conjunt de dades de l'esquerra i les files coincidents del conjunt de dades de la dreta.
  • Right Join: Retorna totes les files del conjunt de dades de la dreta i les files coincidents del conjunt de dades de l'esquerra.

Fusió Bàsica amb `merge()`

La funció merge() en R és una eina versàtil per fusionar data frames.

Exemple

# Crear data frames de mostra
df1 <- data.frame(ID = 1:4, Name = c("Alice", "Bob", "Charlie", "David"))
df2 <- data.frame(ID = c(2, 4, 5), Age = c(25, 30, 35))

# Realitzar una inner join
merged_df <- merge(df1, df2, by = "ID")
print(merged_df)

Explicació:

  • df1 i df2 es fusionen basats en la columna comuna ID.
  • Només les files amb valors ID coincidents en ambdós data frames s'inclouen en el resultat.

Tipus d'Unions

Inner Join

Retorna files amb claus coincidents en ambdós conjunts de dades.

inner_join_df <- merge(df1, df2, by = "ID")
print(inner_join_df)

Left Join

Retorna totes les files del conjunt de dades de l'esquerra i les files coincidents del conjunt de dades de la dreta.

left_join_df <- merge(df1, df2, by = "ID", all.x = TRUE)
print(left_join_df)

Right Join

Retorna totes les files del conjunt de dades de la dreta i les files coincidents del conjunt de dades de l'esquerra.

right_join_df <- merge(df1, df2, by = "ID", all.y = TRUE)
print(right_join_df)

Outer Join

Retorna totes les files de tots dos conjunts de dades, amb NA per a coincidències que falten.

outer_join_df <- merge(df1, df2, by = "ID", all = TRUE)
print(outer_join_df)

Fusió Avançada amb `dplyr`

El paquet dplyr proporciona funcions més intuïtives per fusionar i unir dades.

Exemple

library(dplyr)

# Crear data frames de mostra
df1 <- data.frame(ID = 1:4, Name = c("Alice", "Bob", "Charlie", "David"))
df2 <- data.frame(ID = c(2, 4, 5), Age = c(25, 30, 35))

# Realitzar una inner join
inner_join_df <- inner_join(df1, df2, by = "ID")
print(inner_join_df)

Altres Unions en `dplyr`

  • Left Join: left_join(df1, df2, by = "ID")
  • Right Join: right_join(df1, df2, by = "ID")
  • Full Join: full_join(df1, df2, by = "ID")

Comparació de `merge()` i Unions de `dplyr`

| Característica | Funció merge() | Funcions de dplyr | |----------------|---------------------|---------------------| | Sintaxi | Més verbosa | Més intuïtiva | | Rendiment | Generalment més lent| Generalment més ràpid| | Flexibilitat | Alta | Alta | | Llegibilitat | Moderada | Alta |

Conclusió

Fusionar i unir dades són operacions fonamentals en l'anàlisi de dades. R proporciona múltiples maneres d'aconseguir-ho, des de la funció base merge() fins al paquet més amigable dplyr. Entendre aquests mètodes et permet combinar conjunts de dades de manera eficient i realitzar una anàlisi de dades completa.

En dominar aquestes tècniques, podràs gestionar tasques complexes de manipulació de dades i assegurar-te que els teus conjunts de dades es combinen de manera precisa per a una anàlisi posterior.
