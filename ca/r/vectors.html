Els vectors són una de les estructures de dades més fonamentals a R. Són essencials per emmagatzemar i manipular dades. Aquesta secció cobrirà tot, des dels conceptes bàsics dels vectors fins a operacions avançades.

Introducció als Vectors

  • Definició: Un vector és una seqüència d'elements de dades del mateix tipus bàsic. A R, els vectors poden ser numèrics, de caràcters, lògics o complexos.
  • Creació: Els vectors es poden crear utilitzant la funció c().

Creació de Vectors

# Vector numèric
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Vector de caràcters
char_vector <- c("apple", "banana", "cherry")
print(char_vector)

# Vector lògic
logical_vector <- c(TRUE, FALSE, TRUE)
print(logical_vector)

Accés als Elements

  • Indexació: Els elements d'un vector es poden accedir utilitzant claudàtors [].
# Accedir al primer element d'un vector numèric
first_element <- numeric_vector[1]
print(first_element)

# Accedir a múltiples elements
subset_vector <- numeric_vector[1:3]
print(subset_vector)

Operacions amb Vectors

  • Operacions aritmètiques: Es poden realitzar operacions aritmètiques amb vectors element a element.
# Sumar dos vectors
vector1 <- c(1, 2, 3)
vector2 <- c(4, 5, 6)
sum_vector <- vector1 + vector2
print(sum_vector)

# Multiplicar un vector per un escalar
scalar_mult <- vector1 * 2
print(scalar_mult)
  • Operacions lògiques: També es poden realitzar operacions lògiques element a element.
# Comparació lògica
logical_result <- vector1 > 2
print(logical_result)

Operacions Avançades amb Vectors

Funcions de Vector

  • Funcions Comunes: R proporciona diverses funcions integrades per treballar amb vectors.
# Suma dels elements
sum_vector <- sum(numeric_vector)
print(sum_vector)

# Mitjana dels elements
mean_vector <- mean(numeric_vector)
print(mean_vector)

# Ordenar un vector
sorted_vector <- sort(numeric_vector)
print(sorted_vector)

Reciclatge de Vectors

  • Regla de Reciclatge: Quan es realitzen operacions amb vectors de diferents longituds, R recicla el vector més curt per igualar la longitud del més llarg.
# Exemple de reciclatge
short_vector <- c(1, 2)
long_vector <- c(10, 20, 30, 40)
recycled_sum <- short_vector + long_vector
print(recycled_sum)

Vectors Nomenats

  • Assignació de Noms: Es poden assignar noms als elements d'un vector per a una millor llegibilitat.
# Creació d'un vector nomenat
named_vector <- c(a = 1, b = 2, c = 3)
print(named_vector)

# Accedir als elements pel nom
element_b <- named_vector["b"]
print(element_b)

Conclusió

Els vectors són una estructura de dades versàtil i essencial a R. Permeten un emmagatzematge i manipulació eficient de dades. Entendre com crear, accedir i realitzar operacions amb vectors és crucial per a qualsevol programador de R. A mesura que avancis, dominar les funcions de vector, el reciclatge i la nomenclatura et permetrà escriure codi més eficient i llegible.
