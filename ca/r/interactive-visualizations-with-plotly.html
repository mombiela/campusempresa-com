Introducció a plotly

Plotly és una biblioteca potent per crear visualitzacions interactives en R. Permet als usuaris crear una àmplia gamma de gràfics i diagrames que es poden incrustar en aplicacions web, informes i panells de control.

  • Conceptes Clau:
    • Visualitzacions Interactives vs. Estàtiques
    • Visió general de la biblioteca plotly
    • Instal·lació i configuració
# Instal·lar i carregar plotly
install.packages("plotly")
library(plotly)

Gràfics Bàsics amb Plotly

Aprèn a crear gràfics bàsics amb plotly, com ara diagrames de dispersió, gràfics de línies i gràfics de barres.

  • Diagrama de Dispersió:
# Diagrama de Dispersió Bàsic
data <- data.frame(x = rnorm(50), y = rnorm(50))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  • Gràfic de Línies:
# Gràfic de Línies Bàsic
data <- data.frame(x = 1:10, y = cumsum(rnorm(10)))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
  • Gràfic de Barres:
# Gràfic de Barres Bàsic
data <- data.frame(x = c('A', 'B', 'C'), y = c(10, 20, 30))
plot_ly(data, x = ~x, y = ~y, type = 'bar')

Personalització de Visualitzacions

Aprèn a personalitzar les teves visualitzacions modificant colors, afegint títols i ajustant els eixos.

  • Personalització de Colors:
# Personalització de Colors en Diagrama de Dispersió
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', marker = list(color = 'red'))
  • Afegir Títols i Etiquetes:
# Afegir Títols i Etiquetes
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(title = 'Exemple de Diagrama de Dispersió', xaxis = list(title = 'Eix X'), yaxis = list(title = 'Eix Y'))
  • Ajustar Eixos:
# Ajustar Eixos
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(xaxis = list(range = c(-3, 3)), yaxis = list(range = c(-3, 3)))

Funcions Avançades de Plotly

Explora funcions avançades com subgràfics, gràfics 3D i animacions.

  • Subgràfics:
# Creació de Subgràfics
subplot(
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers'),
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
)
  • Gràfics 3D:
# Diagrama de Dispersió 3D
data <- data.frame(x = rnorm(50), y = rnorm(50), z = rnorm(50))
plot_ly(data, x = ~x, y = ~y, z = ~z, type = 'scatter3d', mode = 'markers')
  • Animacions:
# Diagrama de Dispersió Animat
data <- data.frame(x = rnorm(50), y = rnorm(50), frame = rep(1:5, each = 10))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', frame = ~frame)

Integració de Plotly amb Shiny

Aprèn a integrar visualitzacions de plotly en aplicacions Shiny per a aplicacions web interactives.

  • Aplicació Shiny Bàsica amb Plotly:
# Aplicació Shiny amb Plotly
library(shiny)
library(plotly)

ui <- fluidPage(
  plotlyOutput("plot")
)

server <- function(input, output) {
  output$plot <- renderPlotly({
    plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  })
}

shinyApp(ui, server)

Conclusió

Plotly és una eina versàtil i potent per crear visualitzacions interactives en R. En dominar els conceptes bàsics i explorar funcions avançades, pots crear visualitzacions atractives i dinàmiques per a una varietat d'aplicacions.

  • Resum:
    • Plotly ofereix una àmplia gamma de tipus de gràfics i opcions de personalització.
    • Funcions avançades com subgràfics, gràfics 3D i animacions poden millorar les teves visualitzacions.
    • La integració amb Shiny permet la creació d'aplicacions web interactives.

Seguint aquest curs, adquiriràs les habilitats necessàries per crear i personalitzar visualitzacions interactives utilitzant plotly en R, des de gràfics bàsics fins a aplicacions interactives avançades.
