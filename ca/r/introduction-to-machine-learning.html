Visió General

L'Aprenentatge Automàtic (ML) és una subcategoria de la intel·ligència artificial que implica l'ús d'algoritmes i models estadístics per permetre que els ordinadors realitzin tasques sense instruccions explícites. Aquest curs et guiarà a través dels conceptes bàsics de l'aprenentatge automàtic utilitzant R, un llenguatge potent per al càlcul estadístic i gràfics.

Què és l'Aprenentatge Automàtic?

  • Definició: L'Aprenentatge Automàtic és l'estudi d'algoritmes que milloren automàticament a través de l'experiència.
  • Tipus d'Aprenentatge Automàtic:
    • Aprenentatge Supervisat: Aprenentatge a partir de dades etiquetades (per exemple, classificació, regressió).
    • Aprenentatge No Supervisat: Aprenentatge a partir de dades no etiquetades (per exemple, agrupament, associació).
    • Aprenentatge per Reforç: Aprenentatge interactuant amb un entorn per maximitzar una noció de recompensa acumulativa.

Configuració de R per a l'Aprenentatge Automàtic

Abans d'endinsar-nos en l'aprenentatge automàtic, assegura't de tenir R i RStudio instal·lats. També necessitaràs instal·lar alguns paquets essencials.

Instal·lació de R i RStudio

  1. Descarrega i instal·la R des de CRAN.
  2. Descarrega i instal·la RStudio des de RStudio.

Instal·lació de Paquets Essencials

install.packages(c("tidyverse", "caret", "randomForest", "e1071"))

Preprocessament de Dades

El preprocessament de dades és un pas crucial en l'aprenentatge automàtic. Implica netejar i transformar dades en brut en un format adequat per al modelatge.

Carregar Dades

# Carregar les biblioteques necessàries
library(tidyverse)

# Carregar un conjunt de dades de mostra
data <- read.csv("path/to/your/dataset.csv")

# Mostrar les primeres files del conjunt de dades
head(data)

Tractament de Valors Perduts

# Comprovar si hi ha valors perduts
sum(is.na(data))

# Eliminar files amb valors perduts
data <- na.omit(data)

Escalat de Característiques

# Normalitzar les dades
data_scaled <- scale(data)

Aprenentatge Supervisat

L'aprenentatge supervisat implica entrenar un model amb dades etiquetades. Cobrirem dos tipus principals: classificació i regressió.

Classificació

Exemple: Predir Espècies de Flors d'Iris

# Carregar el conjunt de dades
data(iris)

# Dividir les dades en conjunts d'entrenament i prova
set.seed(123)
training_indices <- sample(1:nrow(iris), 0.7 * nrow(iris))
train_data <- iris[training_indices, ]
test_data <- iris[-training_indices, ]

# Entrenar un model de bosc aleatori
library(randomForest)
model <- randomForest(Species ~ ., data = train_data)

# Fer prediccions
predictions <- predict(model, test_data)

# Avaluar el model
confusion_matrix <- table(predictions, test_data$Species)
print(confusion_matrix)

Regressió

Exemple: Predir Preus de Cases

# Carregar el conjunt de dades
data <- read.csv("path/to/house_prices.csv")

# Dividir les dades en conjunts d'entrenament i prova
set.seed(123)
training_indices <- sample(1:nrow(data), 0.7 * nrow(data))
train_data <- data[training_indices, ]
test_data <- data[-training_indices, ]

# Entrenar un model de regressió lineal
model <- lm(price ~ ., data = train_data)

# Fer prediccions
predictions <- predict(model, test_data)

# Avaluar el model
mse <- mean((predictions - test_data$price)^2)
print(mse)

Aprenentatge No Supervisat

L'aprenentatge no supervisat implica trobar patrons ocults en dades no etiquetades. Cobrirem l'agrupament i la reducció de dimensionalitat.

Agrupament

Exemple: Agrupament K-means

# Carregar el conjunt de dades
data <- read.csv("path/to/your/dataset.csv")

# Realitzar l'agrupament K-means
set.seed(123)
clusters <- kmeans(data, centers = 3)

# Afegir assignacions de clúster a les dades
data$cluster <- clusters$cluster

# Visualitzar els clústers
library(ggplot2)
ggplot(data, aes(x = feature1, y = feature2, color = as.factor(cluster))) +
  geom_point() +
  labs(color = "Clúster")

Reducció de Dimensionalitat

Exemple: Anàlisi de Components Principals (PCA)

# Carregar el conjunt de dades
data <- read.csv("path/to/your/dataset.csv")

# Realitzar PCA
pca <- prcomp(data, scale. = TRUE)

# Visualitzar els dos primers components principals
pca_data <- data.frame(pca$x)
ggplot(pca_data, aes(x = PC1, y = PC2)) +
  geom_point()
