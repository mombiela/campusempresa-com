Introducció a les Llistes

Les llistes a R són una estructura de dades versàtil i potent que pot contenir elements de diferents tipus i longituds. A diferència dels vectors, que només poden contenir elements del mateix tipus, les llistes poden contenir una barreja de números, cadenes, vectors, altres llistes i més.

Conceptes Clau

  • Definició: Una llista és una col·lecció ordenada d'elements, on cada element pot ser d'un tipus diferent.
  • Creació: Les llistes es creen utilitzant la funció list().
  • Accés als Elements: Els elements d'una llista es poden accedir utilitzant doble claudàtor [[ ]] o l'operador $.
  • Modificació de Llistes: Les llistes es poden modificar assignant nous valors a elements específics.
  • Llistes Imbricades: Les llistes poden contenir altres llistes, fent-les una eina potent per a estructures de dades complexes.

Creació de Llistes

Per crear una llista a R, utilitza la funció list(). Aquí tens un exemple senzill:

# Creant una llista amb diferents tipus d'elements
my_list <- list(name = "John", age = 30, scores = c(85, 90, 88))

# Imprimint la llista
print(my_list)

Explicació

  • name = "John": Una cadena de caràcters.
  • age = 30: Un valor numèric.
  • scores = c(85, 90, 88): Un vector numèric.

Accés als Elements de la Llista

Pots accedir als elements d'una llista utilitzant doble claudàtor [[ ]] o l'operador $.

Utilitzant Doble Claudàtor

# Accedint a l'element 'name'
name <- my_list[[1]]
print(name)

# Accedint a l'element 'scores'
scores <- my_list[[3]]
print(scores)

Utilitzant l'Operador `$`

# Accedint a l'element 'name'
name <- my_list$name
print(name)

# Accedint a l'element 'scores'
scores <- my_list$scores
print(scores)

Modificació de Llistes

Pots modificar els elements d'una llista assignant nous valors a elements específics.

# Modificant l'element 'age'
my_list$age <- 31

# Afegint un nou element
my_list$city <- "New York"

# Imprimint la llista modificada
print(my_list)

Llistes Imbricades

Les llistes poden contenir altres llistes, la qual cosa permet la creació d'estructures de dades complexes.

# Creant una llista imbricada
nested_list <- list(person = my_list, job = list(title = "Data Scientist", salary = 90000))

# Accedint a elements d'una llista imbricada
job_title <- nested_list$job$title
print(job_title)

Resum

Les llistes a R són una estructura de dades flexible i potent que pot contenir elements de diferents tipus i longituds. Es creen utilitzant la funció list() i es poden accedir i modificar utilitzant doble claudàtor [[ ]] o l'operador $. Les llistes també poden ser imbricades, permetent la creació d'estructures de dades complexes.

En dominar les llistes, seràs capaç de gestionar una gran varietat de tipus i estructures de dades a R, fent les teves tasques d'anàlisi de dades i programació més eficients i efectives.
