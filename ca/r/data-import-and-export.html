Introducció

La importació i exportació de dades són habilitats fonamentals en la programació en R, permetent-te llegir dades de diverses fonts i guardar els teus resultats. Aquesta secció et guiarà des de les tècniques bàsiques fins a les avançades d'importació i exportació de dades en R.

Importació de dades

Llegir dades de fitxers CSV

Els fitxers CSV (Valors Separats per Comes) són un dels formats més comuns per a l'emmagatzematge de dades.

  • Importació bàsica de CSV:

    data <- read.csv("path/to/your/file.csv")
    • read.csv() és una funció que llegeix un fitxer en format de taula i crea un data frame a partir d'ell.
    • "path/to/your/file.csv" és la ruta al teu fitxer CSV.

  • Especificar delimitadors:

    data <- read.csv("path/to/your/file.csv", sep = ";")
    • El paràmetre sep et permet especificar el delimitador utilitzat en el teu fitxer.

Llegir dades de fitxers Excel

Els fitxers Excel són àmpliament utilitzats en l'anàlisi de dades. R proporciona diversos paquets per llegir fitxers Excel, com readxl i openxlsx.

  • Utilitzant el paquet readxl: 
    library(readxl)
data <- read_excel("path/to/your/file.xlsx", sheet = "Sheet1")
    • La funció read_excel() llegeix un fitxer Excel.
    • El paràmetre sheet especifica el full a llegir.

Llegir dades de bases de dades

R pot connectar-se a diverses bases de dades utilitzant paquets com DBI i RSQLite.

  • Connectar-se a una base de dades SQLite: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "path/to/your/database.sqlite")
data <- dbGetQuery(con, "SELECT * FROM your_table")
dbDisconnect(con)
    • dbConnect() estableix una connexió amb la base de dades.
    • dbGetQuery() executa una consulta SQL i retorna el resultat com un data frame.
    • dbDisconnect() tanca la connexió.

Llegir dades des del web

R també pot llegir dades directament des del web utilitzant funcions com read.csv() per a fitxers CSV allotjats en línia.

  • Llegir CSV des d'una URL: 
    url <- "http://example.com/data.csv"
data <- read.csv(url)

Exportació de dades

Escriure dades a fitxers CSV

Exportar dades a CSV és senzill amb la funció write.csv().

  • Exportació bàsica de CSV: 
    write.csv(data, "path/to/your/output.csv")
    • write.csv() escriu un data frame a un fitxer CSV.
    • "path/to/your/output.csv" és la ruta on es guardarà el fitxer.

Escriure dades a fitxers Excel

Per escriure dades a fitxers Excel, pots utilitzar els paquets writexl o openxlsx.

  • Utilitzant el paquet writexl: 
    library(writexl)
write_xlsx(data, "path/to/your/output.xlsx")
    • write_xlsx() escriu un data frame a un fitxer Excel.

Escriure dades a bases de dades

També pots exportar dades a bases de dades utilitzant el paquet DBI.

  • Escriure a una base de dades SQLite: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "path/to/your/database.sqlite")
dbWriteTable(con, "your_table", data)
dbDisconnect(con)
    • dbWriteTable() escriu un data frame a una taula de la base de dades.

Escriure dades al web

Encara que R no suporta directament l'escriptura de dades a ubicacions web, pots utilitzar APIs o altres serveis web per pujar dades.

Resum

En aquesta secció, hem cobert les tècniques essencials per importar i exportar dades en R:

  • Importació de dades:

    • Llegir de fitxers CSV utilitzant read.csv().
    • Llegir de fitxers Excel utilitzant el paquet readxl.
    • Llegir de bases de dades utilitzant DBI i RSQLite.
    • Llegir des del web utilitzant URLs.

  • Exportació de dades:

    • Escriure a fitxers CSV utilitzant write.csv().
    • Escriure a fitxers Excel utilitzant el paquet writexl.
    • Escriure a bases de dades utilitzant DBI i RSQLite.

Dominar aquestes habilitats et permetrà manejar dades de diverses fonts i formats, fent el teu flux de treball d'anàlisi de dades més eficient i versàtil.
