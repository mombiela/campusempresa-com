Entendre els tipus de dades és fonamental per programar en R. Aquesta secció cobrirà els principals tipus de dades en R, com utilitzar-los i la seva importància en l'anàlisi de dades.

Tipus de dades bàsics

R té diversos tipus de dades bàsics que són essencials per a qualsevol programador de R. Aquests inclouen:

  • Numèric
  • Enter
  • Caràcter
  • Lògic
  • Complex

Numèric

Els tipus de dades numèrics s'utilitzen per representar nombres reals.

# Exemple de Numèric
num <- 42.5
print(num)  # Sortida: 42.5

Enter

Els enters són nombres sencers. En R, pots especificar un enter afegint una L al número.

# Exemple d'Enter
int <- 42L
print(int)  # Sortida: 42

Caràcter

Els tipus de dades de caràcter s'utilitzen per representar valors de text o cadenes.

# Exemple de Caràcter
char <- "Hola, R!"
print(char)  # Sortida: "Hola, R!"

Lògic

Els tipus de dades lògics representen valors booleans: TRUE o FALSE.

# Exemple de Lògic
logi <- TRUE
print(logi)  # Sortida: TRUE

Complex

Els nombres complexos tenen una part real i una part imaginària.

# Exemple de Complex
comp <- 3 + 2i
print(comp)  # Sortida: 3+2i

Estructures de dades

R també proporciona diverses estructures de dades per emmagatzemar col·leccions de dades. Aquestes inclouen:

  • Vectors
  • Matrius
  • Arrays
  • Data Frames
  • Llistes

Vectors

Els vectors són l'estructura de dades més bàsica en R i poden contenir elements del mateix tipus.

# Exemple de Vector
vec <- c(1, 2, 3, 4, 5)
print(vec)  # Sortida: 1 2 3 4 5

Matrius

Les matrius són estructures de dades bidimensionals i homogènies.

# Exemple de Matriu
mat <- matrix(1:9, nrow = 3, ncol = 3)
print(mat)
# Sortida:
#      [,1] [,2] [,3]
# [1,]    1    4    7
# [2,]    2    5    8
# [3,]    3    6    9

Arrays

Els arrays són estructures de dades multidimensionals i homogènies.

# Exemple d'Array
arr <- array(1:8, dim = c(2, 2, 2))
print(arr)
# Sortida:
# , , 1
#      [,1] [,2]
# [1,]    1    3
# [2,]    2    4
#
# , , 2
#      [,1] [,2]
# [1,]    5    7
# [2,]    6    8

Data Frames

Els data frames són estructures de dades bidimensionals i heterogènies, similars a les taules en una base de dades o full de càlcul d'Excel.

# Exemple de Data Frame
df <- data.frame(
  Name = c("John", "Jane", "Doe"),
  Age = c(23, 25, 31),
  Gender = c("Male", "Female", "Male")
)
print(df)
# Sortida:
#   Name Age Gender
# 1 John  23   Male
# 2 Jane  25 Female
# 3  Doe  31   Male

Llistes

Les llistes poden contenir elements de diferents tipus i estructures.

# Exemple de Llista
lst <- list(
  Name = "John",
  Age = 23,
  Scores = c(90, 85, 88)
)
print(lst)
# Sortida:
# $Name
# [1] "John"
#
# $Age
# [1] 23
#
# $Scores
# [1] 90 85 88

Resum

En aquesta secció, hem cobert els tipus de dades bàsics en R, incloent-hi els tipus numèric, enter, caràcter, lògic i complex. També hem explorat diverses estructures de dades com vectors, matrius, arrays, data frames i llistes. Entendre aquests tipus de dades i estructures és crucial per a una manipulació i anàlisi de dades efectiva en R. A mesura que avancis en el teu viatge de programació en R, aquests conceptes formaran la base per a temes i tècniques més avançades.
