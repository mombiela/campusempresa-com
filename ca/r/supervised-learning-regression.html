Introducció a la Regressió

La regressió és un concepte fonamental en l'aprenentatge supervisat, on l'objectiu és predir una variable objectiu contínua basada en una o més variables predictors. En aquesta secció, cobrirem els conceptes bàsics de la regressió, centrant-nos en com implementar i interpretar models de regressió en R.

Conceptes Clau

  • Variable Dependent (Objectiu): La variable que intentem predir.
  • Variables Independents (Predictors): Les variables utilitzades per fer prediccions.
  • Regressió Lineal: Un mètode per modelar la relació entre la variable dependent i una o més variables independents ajustant una equació lineal.
  • Residuals: Les diferències entre els valors observats i els valors previstos.

Regressió Lineal Simple

La regressió lineal simple implica predir una variable dependent utilitzant una sola variable independent.

Exemple

Comencem amb un exemple senzill per entendre com realitzar una regressió lineal en R.

# Carregar la biblioteca necessària
library(ggplot2)

# Dades de mostra
data <- data.frame(
  x = c(1, 2, 3, 4, 5),
  y = c(2, 4, 5, 4, 5)
)

# Representar les dades
ggplot(data, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  labs(title = "Regressió Lineal Simple", x = "Variable Independent (x)", y = "Variable Dependent (y)")

Explicació

  • Preparació de Dades: Creem un data frame amb dues variables x i y.
  • Visualització: Utilitzem ggplot2 per representar els punts de dades i la línia de regressió lineal.

Ajust del Model

# Ajustar el model lineal
model <- lm(y ~ x, data = data)

# Resum del model
summary(model)

Explicació

  • Ajust del Model: La funció lm s'utilitza per ajustar un model lineal.
  • Resum del Model: La funció summary proporciona informació detallada sobre el model, incloent-hi els coeficients, el valor R-quadrat i els valors p.

Regressió Lineal Múltiple

La regressió lineal múltiple implica predir una variable dependent utilitzant dues o més variables independents.

Exemple

Estenem el nostre exemple per incloure múltiples predictors.

# Dades de mostra
data <- data.frame(
  x1 = c(1, 2, 3, 4, 5),
  x2 = c(5, 4, 3, 2, 1),
  y = c(2, 3, 4, 5, 6)
)

# Ajustar el model lineal múltiple
model <- lm(y ~ x1 + x2, data = data)

# Resum del model
summary(model)

Explicació

  • Preparació de Dades: Creem un data frame amb tres variables x1, x2 i y.
  • Ajust del Model: La funció lm s'utilitza per ajustar un model de regressió lineal múltiple.
  • Resum del Model: La funció summary proporciona informació detallada sobre el model.

Avaluació del Model

Avaluar el rendiment d'un model de regressió és crucial per entendre el seu poder predictiu.

Mètriques Clau

  • R-quadrat: Indica la proporció de la variància en la variable dependent que és previsible a partir de les variables independents.
  • Error Quadràtic Mitjà (MSE): La mitjana dels quadrats dels residuals.
  • Arrel de l'Error Quadràtic Mitjà (RMSE): L'arrel quadrada del MSE, proporcionant una mesura de la magnitud mitjana dels residuals.

Exemple

# Calcular les prediccions
predictions <- predict(model, data)

# Calcular els residuals
residuals <- data$y - predictions

# Calcular MSE i RMSE
mse <- mean(residuals^2)
rmse <- sqrt(mse)

# Imprimir les mètriques
cat("MSE:", mse, "\n")
cat("RMSE:", rmse, "\n")

Explicació

  • Prediccions: La funció predict s'utilitza per generar prediccions a partir del model.
  • Residuals: Calculats com la diferència entre els valors observats i els valors previstos.
  • MSE i RMSE: Calculats per avaluar el rendiment del model.

Conclusió

En aquest tema, hem cobert els conceptes bàsics de l'aprenentatge supervisat amb un enfocament en la regressió. Hem explorat la regressió lineal simple i múltiple, hem après com ajustar models en R i hem discutit les mètriques clau per a l'avaluació del model. Entenent aquests conceptes, pots començar a construir i avaluar models de regressió per fer prediccions basades en dades contínues.
