Introducció

En aquest mòdul, explorarem dues proves estadístiques importants: ANOVA (Anàlisi de Variància) i proves de Chi-quadrat. Aquestes proves s'utilitzen àmpliament per a la verificació d'hipòtesis en diversos camps com la biologia, la psicologia i les ciències socials. Al final d'aquest mòdul, entendràs com realitzar aquestes proves en R i interpretar-ne els resultats.

ANOVA (Anàlisi de Variància)

Què és l'ANOVA?

L'ANOVA és un mètode estadístic utilitzat per comparar les mitjanes de tres o més grups per veure si almenys un d'ells és significativament diferent dels altres. Ajuda a determinar si les diferències observades entre les mitjanes de les mostres es deuen a diferències reals o a variació aleatòria.

Tipus d'ANOVA

  • ANOVA d'un factor: Compara les mitjanes de grups basats en una variable independent.
  • ANOVA de dos factors: Compara les mitjanes de grups basats en dues variables independents i també pot comprovar els efectes d'interacció.

Realitzar ANOVA d'un factor en R

Escenari d'exemple

Suposem que tenim tres dietes diferents i volem veure si hi ha una diferència significativa en la pèrdua de pes entre aquestes dietes.

Exemple de codi

# Dades de mostra
diet <- factor(c(rep("Diet1", 10), rep("Diet2", 10), rep("Diet3", 10)))
weight_loss <- c(2, 3, 2.5, 3.5, 4, 3, 2.8, 3.2, 3.1, 3.3, 
                 3, 3.5, 3.2, 3.8, 4.1, 3.6, 3.4, 3.7, 3.9, 4,
                 2.5, 2.8, 2.9, 3, 3.1, 2.7, 2.6, 2.9, 3.2, 3.1)

# Crear un data frame
data <- data.frame(diet, weight_loss)

# Realitzar ANOVA d'un factor
anova_result <- aov(weight_loss ~ diet, data = data)
summary(anova_result)

Explicació

  • aov(weight_loss ~ diet, data = data): Aquesta funció realitza la prova ANOVA.
  • summary(anova_result): Aquesta funció proporciona el resum de la prova ANOVA, incloent-hi el valor F i el valor p.

Interpretació dels resultats de l'ANOVA

  • Valor F: Indica la proporció de variància entre els grups a la variància dins dels grups.
  • Valor p: Si el valor p és menor que el nivell de significació (normalment 0.05), rebutgem la hipòtesi nul·la i concloem que hi ha una diferència significativa entre les mitjanes dels grups.

Proves de Chi-quadrat

Què és una prova de Chi-quadrat?

Una prova de Chi-quadrat s'utilitza per determinar si hi ha una associació significativa entre dues variables categòriques. Compara les freqüències observades en cada categoria amb les freqüències esperades si no hi hagués associació.

Tipus de proves de Chi-quadrat

  • Prova de Chi-quadrat d'independència: Comprova si dues variables categòriques són independents.
  • Prova de bondat d'ajust de Chi-quadrat: Comprova si una mostra coincideix amb la distribució esperada.

Realitzar la prova de Chi-quadrat d'independència en R

Escenari d'exemple

Suposem que volem comprovar si hi ha una associació entre el gènere i la preferència per un nou producte.

Exemple de codi

# Dades de mostra
gender <- c("Male", "Male", "Female", "Female", "Male", "Female", "Male", "Female", "Male", "Female")
preference <- c("Like", "Dislike", "Like", "Dislike", "Like", "Like", "Dislike", "Dislike", "Like", "Like")

# Crear una taula de contingència
table_data <- table(gender, preference)

# Realitzar la prova de Chi-quadrat
chi_square_result <- chisq.test(table_data)
chi_square_result

Explicació

  • table(gender, preference): Crea una taula de contingència de les freqüències observades.
  • chisq.test(table_data): Realitza la prova de Chi-quadrat d'independència.

Interpretació dels resultats de la prova de Chi-quadrat

  • Estadístic de Chi-quadrat: Mesura quant es desvien les freqüències observades de les freqüències esperades.
  • Valor p: Si el valor p és menor que el nivell de significació (normalment 0.05), rebutgem la hipòtesi nul·la i concloem que hi ha una associació significativa entre les variables.
