Introducció

R és un llenguatge potent per al càlcul estadístic i gràfics. Entendre la sintaxi bàsica és crucial per escriure codi R eficient i llegible. Aquesta secció cobrirà els elements fonamentals de la sintaxi de R, incloent variables, tipus de dades, operadors i funcions bàsiques.

Variables i assignació

Les variables en R s'utilitzen per emmagatzemar dades que es poden referenciar i manipular. L'operador d'assignació <- s'utilitza comunament per assignar valors a les variables.

# Assignar un valor a una variable
x <- 10
y <- 5
  • Operador d'assignació: <- és preferit, però = també es pot utilitzar.
  • Noms de variables: Han de començar amb una lletra i poden contenir lletres, números i guions baixos.

Tipus de dades

R admet diversos tipus de dades, incloent numèric, enter, caràcter, lògic i complex.

# Numèric
num <- 3.14

# Enter
int <- 42L

# Caràcter
char <- "Hola, R!"

# Lògic
bool <- TRUE

# Complex
comp <- 2 + 3i

| Tipus de dades | Exemple | Descripció | |----------------|---------------|------------------------------| | Numèric | 3.14 | Nombres decimals | | Enter | 42L | Nombres sencers amb sufix L| | Caràcter | "Hola, R!" | Cadenes de text | | Lògic | TRUE | Valors booleans (TRUE, FALSE) | | Complex | 2 + 3i | Nombres complexos |

Operadors

Els operadors en R s'utilitzen per realitzar operacions aritmètiques, de comparació i lògiques.

Operadors aritmètics

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • ^ : Exponenciació
# Operacions aritmètiques
sum <- x + y
diff <- x - y
prod <- x * y
quot <- x / y
exp <- x ^ y

Operadors de comparació

  • == : Igual a
  • != : Diferent de
  • < : Menor que
  • > : Major que
  • <= : Menor o igual que
  • >= : Major o igual que
# Operacions de comparació
is_equal <- x == y
is_not_equal <- x != y
is_less <- x < y
is_greater <- x > y

Operadors lògics

  • & : I
  • | : O
  • ! : NO
# Operacions lògiques
and_op <- (x > 5) & (y < 10)
or_op <- (x > 5) | (y < 10)
not_op <- !(x > 5)

Funcions bàsiques

Les funcions s'utilitzen per realitzar tasques específiques. R té moltes funcions integrades, i també pots crear les teves pròpies.

Funcions integrades

  • print(): Imprimeix la sortida a la consola.
  • sqrt(): Calcula l'arrel quadrada.
  • abs(): Calcula el valor absolut.
# Utilitzant funcions integrades
print("Hola, R!")
sqrt_val <- sqrt(16)
abs_val <- abs(-5)

Funcions definides per l'usuari

Pots crear les teves pròpies funcions utilitzant la paraula clau function.

# Definint una funció
add_numbers <- function(a, b) {
  result <- a + b
  return(result)
}

# Cridant la funció
sum_result <- add_numbers(3, 4)
print(sum_result)

Conclusió

Entendre la sintaxi bàsica de R és el primer pas per dominar el llenguatge. Aquesta secció ha cobert els elements essencials, incloent variables, tipus de dades, operadors i funcions bàsiques. Amb aquests fonaments, pots començar a escriure scripts simples en R i gradualment passar a tasques més complexes. Practica aquests conceptes amb exercicis per reforçar el teu aprenentatge i construir una base sòlida en la programació en R.
