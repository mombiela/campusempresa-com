El preprocessament de dades és un pas crucial en el pipeline d'anàlisi de dades. Implica transformar dades en brut en un format net i utilitzable. Aquesta secció et guiarà a través de diverses tècniques de preprocessament de dades en R, des de nivells bàsics fins a avançats.

Introducció al Preprocessament de Dades

El preprocessament de dades inclou diverses tasques:

  • Gestió de valors perduts
  • Transformació de dades
  • Normalització i escalat de dades
  • Codificació de variables categòriques
  • Selecció de característiques

Gestió de Valors Perduts

Els valors perduts poden afectar significativament els resultats de la teva anàlisi. R proporciona diverses maneres de gestionar les dades perdudes.

Identificació de Valors Perduts

Pots identificar valors perduts utilitzant la funció is.na().

# Dades de mostra
data <- c(1, 2, NA, 4, NA, 6)

# Identificar valors perduts
is.na(data)

Eliminació de Valors Perduts

Pots eliminar valors perduts utilitzant la funció na.omit().

# Eliminar valors perduts
clean_data <- na.omit(data)

Imputació de Valors Perduts

La imputació implica substituir els valors perduts amb un valor específic, com ara la mitjana o la mediana.

# Imputar valors perduts amb la mitjana
data[is.na(data)] <- mean(data, na.rm = TRUE)

Transformació de Dades

La transformació de dades implica convertir les dades en un format adequat per a l'anàlisi.

Transformació Logarítmica

La transformació logarítmica pot ajudar a tractar dades esbiaixades.

# Transformació logarítmica
data <- c(1, 10, 100, 1000)
log_data <- log(data)

Escalat i Normalització

L'escalat i la normalització s'utilitzen per estandarditzar el rang de les variables independents.

# Escalat de dades
scaled_data <- scale(data)

# Normalització de dades
normalized_data <- (data - min(data)) / (max(data) - min(data))

Codificació de Variables Categòriques

Les variables categòriques necessiten ser convertides en un format numèric per a la majoria d'algoritmes d'aprenentatge automàtic.

Codificació One-Hot

La codificació one-hot crea columnes binàries per a cada categoria.

# Data frame de mostra
df <- data.frame(
  color = c('red', 'blue', 'green', 'blue', 'red')
)

# Codificació one-hot
df_one_hot <- model.matrix(~ color - 1, data = df)

Selecció de Característiques

La selecció de característiques implica seleccionar les característiques més rellevants per al teu model.

Matriu de Correlació

Una matriu de correlació pot ajudar a identificar característiques altament correlacionades.

# Data frame de mostra
df <- data.frame(
  x1 = rnorm(100),
  x2 = rnorm(100),
  x3 = rnorm(100)
)

# Matriu de correlació
cor_matrix <- cor(df)

Eliminació Recursiva de Característiques

L'eliminació recursiva de característiques (RFE) és una tècnica per seleccionar característiques considerant recursivament conjunts més petits de característiques.

# Carregar la biblioteca necessària
library(caret)

# Dades de mostra
data(iris)
x <- iris[, -5]
y <- iris$Species

# Eliminació recursiva de característiques
control <- rfeControl(functions = rfFuncs, method = "cv", number = 10)
results <- rfe(x, y, sizes = c(1:4), rfeControl = control)

# Imprimir els resultats
print(results)

Conclusió

El preprocessament de dades és un pas essencial en el procés d'anàlisi de dades. Assegura que les teves dades siguin netes, consistents i llestes per a l'anàlisi. Gestionant els valors perduts, transformant les dades, codificant les variables categòriques i seleccionant les característiques rellevants, pots millorar significativament el rendiment dels teus models. Practica aquestes tècniques amb diferents conjunts de dades per convertir-te en un expert en preprocessament de dades utilitzant R.
