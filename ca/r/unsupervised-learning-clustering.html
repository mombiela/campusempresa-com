Introducció al Clustering

El clustering és un tipus d'aprenentatge no supervisat que implica agrupar un conjunt d'objectes de manera que els objectes del mateix grup (anomenat clúster) siguin més similars entre ells que amb els d'altres grups. El clustering s'utilitza àmpliament en diversos camps com la mineria de dades, el reconeixement de patrons i l'anàlisi d'imatges.

Conceptes Clau

  • Aprenentatge No Supervisat: Aprendre a partir de dades sense respostes etiquetades.
  • Clúster: Una col·lecció de punts de dades agregats junts a causa de certes similituds.
  • Centroid: El centre d'un clúster.
  • Mètriques de Distància: Mètodes per mesurar la similitud o dissimilitud entre punts de dades (per exemple, distància euclidiana).

Tipus d'Algoritmes de Clustering

Hi ha diversos algoritmes de clustering, cadascun amb els seus punts forts i febles. Aquí, ens centrarem en dos mètodes populars: K-Means i Clustering Jeràrquic.

Clustering K-Means

K-Means és un mètode de partició que divideix el conjunt de dades en K clústers, on cada clúster està representat pel seu centroid.

Passos de l'Algoritme K-Means

  1. Tria el nombre de clústers, K.
  2. Inicialitza els centroids aleatòriament.
  3. Assigna cada punt de dades al centroid més proper.
  4. Recalcula els centroids basant-se en els punts assignats.
  5. Repeteix els passos 3 i 4 fins a la convergència (és a dir, els centroids ja no canvien).

Exemple de Codi

# Carrega la biblioteca necessària
library(ggplot2)

# Genera dades de mostra
set.seed(123)
data <- data.frame(x = rnorm(100), y = rnorm(100))

# Realitza el clustering K-Means
kmeans_result <- kmeans(data, centers = 3)

# Afegeix l'assignació de clúster a les dades
data$cluster <- as.factor(kmeans_result$cluster)

# Dibuixa els clústers
ggplot(data, aes(x = x, y = y, color = cluster)) +
  geom_point() +
  labs(title = "Clustering K-Means", x = "Eix X", y = "Eix Y")

Clustering Jeràrquic

El clustering jeràrquic construeix un arbre de clústers. Pot ser aglomeratiu (de baix a dalt) o divisori (de dalt a baix).

Passos del Clustering Jeràrquic Aglomeratiu

  1. Comença amb cada punt de dades com el seu propi clúster.
  2. Fusiona els dos clústers més propers.
  3. Repeteix el pas 2 fins que només quedi un clúster.

Exemple de Codi

# Carrega la biblioteca necessària
library(ggplot2)

# Genera dades de mostra
set.seed(123)
data <- data.frame(x = rnorm(100), y = rnorm(100))

# Calcula la matriu de distàncies
dist_matrix <- dist(data)

# Realitza el clustering jeràrquic
hclust_result <- hclust(dist_matrix, method = "complete")

# Talla l'arbre per formar clústers
data$cluster <- cutree(hclust_result, k = 3)

# Dibuixa els clústers
ggplot(data, aes(x = x, y = y, color = as.factor(cluster))) +
  geom_point() +
  labs(title = "Clustering Jeràrquic", x = "Eix X", y = "Eix Y")

Comparació entre K-Means i Clustering Jeràrquic

| Característica | K-Means | Clustering Jeràrquic | |------------------------|--------------------------------|-------------------------------| | Nombre de Clústers | S'ha de especificar prèviament | Es pot determinar pel dendrograma | | Complexitat | O(n * K * I) | O(n^3) | | Escalabilitat | Adequat per a grans conjunts de dades | Adequat per a conjunts de dades més petits | | Forma del Clúster | Assumeix clústers esfèrics | Pot trobar clústers de forma arbitrària |

Consideracions Pràctiques

  • Triar el nombre de clústers: Mètodes com el Mètode del Colze o l'Anàlisi de Silueta poden ajudar a determinar el nombre òptim de clústers.
  • Escalar les dades: Estandarditzar o normalitzar les dades pot millorar el rendiment del clustering.
  • Mètriques de distància: La tria de la mètrica de distància (per exemple, Euclidiana, Manhattan) pot afectar els resultats del clustering.

Conclusió

El clustering és una eina poderosa en l'aprenentatge no supervisat per descobrir patrons i estructures en les dades. K-Means i el Clustering Jeràrquic són dos mètodes àmpliament utilitzats, cadascun amb els seus avantatges i limitacions. Entendre aquests algoritmes i les seves consideracions pràctiques et permetrà aplicar el clustering de manera efectiva en diverses situacions.
