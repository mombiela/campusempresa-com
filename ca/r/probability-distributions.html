Introducció

Les distribucions de probabilitat són fonamentals en estadística i anàlisi de dades. En R, hi ha nombroses funcions i paquets disponibles per treballar amb diverses distribucions de probabilitat. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de les distribucions de probabilitat en R.

Conceptes Clau

  • Distribució de Probabilitat: Una funció que descriu la probabilitat d'obtenir els valors possibles que una variable aleatòria pot prendre.
  • Variable Aleatòria: Una variable els valors de la qual depenen dels resultats d'un fenomen aleatori.
  • Distribucions Discretes vs. Contínues: Les distribucions discretes tracten amb resultats comptables, mentre que les distribucions contínues tracten amb resultats que poden prendre qualsevol valor dins d'un rang.

Treballant amb Distribucions en R

Distribució Binomial

La distribució binomial modela el nombre d'èxits en un nombre fix de proves de Bernoulli independents.

  • Funcions:
    • dbinom(x, size, prob): Funció de massa de probabilitat.
    • pbinom(q, size, prob): Funció de distribució acumulada.
    • qbinom(p, size, prob): Funció quantílica.
    • rbinom(n, size, prob): Generació aleatòria.
# Exemple: Distribució Binomial
# Paràmetres: size = 10 proves, prob = 0.5 probabilitat d'èxit
size <- 10
prob <- 0.5

# Probabilitat d'exactament 5 èxits
dbinom(5, size, prob)

# Probabilitat acumulada de 5 o menys èxits
pbinom(5, size, prob)

# Quantil per a 0.5 probabilitat
qbinom(0.5, size, prob)

# Generar 10 mostres aleatòries
rbinom(10, size, prob)

Distribució Normal

La distribució normal és una distribució de probabilitat contínua caracteritzada per la seva corba en forma de campana.

  • Funcions:
    • dnorm(x, mean, sd): Funció de densitat de probabilitat.
    • pnorm(q, mean, sd): Funció de distribució acumulada.
    • qnorm(p, mean, sd): Funció quantílica.
    • rnorm(n, mean, sd): Generació aleatòria.
# Exemple: Distribució Normal
# Paràmetres: mean = 0, sd = 1
mean <- 0
sd <- 1

# Densitat a x = 0
dnorm(0, mean, sd)

# Probabilitat acumulada fins a x = 1
pnorm(1, mean, sd)

# Quantil per a 0.95 probabilitat
qnorm(0.95, mean, sd)

# Generar 10 mostres aleatòries
rnorm(10, mean, sd)

Distribució de Poisson

La distribució de Poisson modela el nombre d'esdeveniments que ocorren dins d'un interval fix de temps o espai.

  • Funcions:
    • dpois(x, lambda): Funció de massa de probabilitat.
    • ppois(q, lambda): Funció de distribució acumulada.
    • qpois(p, lambda): Funció quantílica.
    • rpois(n, lambda): Generació aleatòria.
# Exemple: Distribució de Poisson
# Paràmetre: lambda = 3 (taxa mitjana d'ocurrència)
lambda <- 3

# Probabilitat d'exactament 2 esdeveniments
dpois(2, lambda)

# Probabilitat acumulada de 2 o menys esdeveniments
ppois(2, lambda)

# Quantil per a 0.8 probabilitat
qpois(0.8, lambda)

# Generar 10 mostres aleatòries
rpois(10, lambda)

Temes Avançats

Ajustar Distribucions a Dades

R proporciona funcions per ajustar distribucions a dades, permetent l'estimació de paràmetres i la prova de bondat d'ajust.

  • Exemple: Ajustar una distribució normal a dades
# Generar dades de mostra
data <- rnorm(100, mean = 5, sd = 2)

# Ajustar una distribució normal
fit <- fitdistr(data, "normal")

# Mostrar els paràmetres d'ajust
fit

Distribucions Personalitzades

Pots definir distribucions de probabilitat personalitzades utilitzant funcions d'R.

  • Exemple: Funció de densitat de probabilitat personalitzada
# Funció de densitat personalitzada
custom_density <- function(x) {
  ifelse(x >= 0 & x <= 1, 2 * x, 0)
}

# Integrar per comprovar si és una funció de densitat vàlida
integrate(custom_density, -Inf, Inf)

Resum

En aquesta secció, hem cobert els conceptes bàsics de les distribucions de probabilitat en R, incloent-hi les distribucions discretes i contínues més comunes. Hem explorat com utilitzar les funcions d'R per treballar amb aquestes distribucions i hem tocat temes avançats com l'ajust de distribucions a dades i la creació de distribucions personalitzades. Entendre aquests conceptes és crucial per a l'anàlisi estadística i les aplicacions de ciència de dades.
