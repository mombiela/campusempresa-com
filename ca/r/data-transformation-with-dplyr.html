Introducció a dplyr

dplyr és un paquet potent de R per a la manipulació i transformació de dades. Proporciona un conjunt consistent de verbs que t'ajuden a resoldre els desafiaments més comuns de manipulació de dades.

  • Conceptes clau:
    • Tibble: Una reimaginació moderna dels data frames.
    • Verbs: Funcions que realitzen tasques específiques de manipulació de dades.
    • Tubs (%>%): Una manera d'encadenar múltiples operacions juntes.

Instal·lació i càrrega de dplyr

Per començar amb dplyr, necessites instal·lar i carregar el paquet.

install.packages("dplyr")
library(dplyr)

Verbs principals en dplyr

dplyr proporciona un conjunt de verbs principals que cobreixen les tasques de manipulació de dades més comunes.

select()

La funció select() s'utilitza per triar columnes específiques d'un data frame.

# Exemple: Seleccionant columnes específiques d'un tibble
library(dplyr)
data <- tibble(
  name = c("Alice", "Bob", "Charlie"),
  age = c(25, 30, 35),
  score = c(85, 90, 95)
)

selected_data <- select(data, name, score)
print(selected_data)

filter()

La funció filter() s'utilitza per subestablir files basades en condicions.

# Exemple: Filtrant files basades en una condició
filtered_data <- filter(data, age > 25)
print(filtered_data)

mutate()

La funció mutate() afegeix noves variables o transforma les existents.

# Exemple: Afegint una nova columna
mutated_data <- mutate(data, age_in_5_years = age + 5)
print(mutated_data)

arrange()

La funció arrange() ordena les files d'un data frame.

# Exemple: Ordenant files per una columna
arranged_data <- arrange(data, desc(score))
print(arranged_data)

summarize() i group_by()

La funció summarize() crea un resum de les dades, sovint utilitzada amb group_by() per realitzar operacions per grup.

# Exemple: Resumint dades per grup
grouped_data <- data %>%
  group_by(age) %>%
  summarize(mean_score = mean(score))
print(grouped_data)

Transformació avançada de dades

A mesura que et familiaritzis més amb dplyr, pots començar a combinar aquests verbs per realitzar transformacions de dades més complexes.

Encadenant operacions amb tubs

L'operador de tubs (%>%) et permet encadenar múltiples funcions dplyr juntes, fent el teu codi més llegible.

# Exemple: Encadenant múltiples operacions
chained_data <- data %>%
  filter(age > 25) %>%
  select(name, score) %>%
  arrange(desc(score))
print(chained_data)

Unions

dplyr proporciona funcions per unir dos data frames: inner_join(), left_join(), right_join(), i full_join().

# Exemple: Unint dos data frames
data1 <- tibble(
  id = c(1, 2, 3),
  name = c("Alice", "Bob", "Charlie")
)

data2 <- tibble(
  id = c(1, 2, 4),
  score = c(85, 90, 95)
)

joined_data <- inner_join(data1, data2, by = "id")
print(joined_data)

Conclusió

dplyr és una eina essencial per a la manipulació de dades en R. Dominant els seus verbs principals i entenent com encadenar operacions amb tubs, pots realitzar transformacions de dades complexes de manera eficient. Practica utilitzant aquestes funcions amb conjunts de dades reals per convertir-te en un expert en manipulació de dades amb dplyr.
