Introducció

En aquesta secció, ens endinsarem en tècniques avançades per crear gràfics sofisticats i visualment atractius utilitzant ggplot2 en R. Això es basarà en els teus coneixements fonamentals i t'ajudarà a crear visualitzacions més complexes i personalitzades.

Personalització de l'aparença del gràfic

Temes

  • Temes et permeten controlar els elements no relacionats amb les dades dels teus gràfics, com el color de fons, les línies de la quadrícula i el text.
  • ggplot2 ve amb diversos temes integrats com theme_minimal(), theme_classic() i theme_dark().
library(ggplot2)

# Gràfic bàsic
p <- ggplot(mpg, aes(x=displ, y=hwy)) + geom_point()

# Aplicació d'un tema
p + theme_minimal()

Temes personalitzats

  • Pots crear el teu propi tema modificant els existents o començant des de zero utilitzant theme().
custom_theme <- theme(
  panel.background = element_rect(fill = "lightblue"),
  panel.grid.major = element_line(color = "white"),
  axis.text = element_text(size = 12, color = "blue"),
  axis.title = element_text(size = 14, face = "bold")
)

p + custom_theme

Geometries avançades

Geom_Smooth

  • geom_smooth() afegeix una mitjana condicional suavitzada, que és útil per visualitzar tendències.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE, color = "red")

Geom_Text i Geom_Label

  • geom_text() i geom_label() et permeten afegir anotacions de text als teus gràfics.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  geom_text(aes(label=manufacturer), vjust=-1, size=3)

Facetació

Facet Wrap

  • facet_wrap() crea panells separats per a cada nivell d'un factor.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  facet_wrap(~ class)

Facet Grid

  • facet_grid() permet una facetació més complexa especificant files i columnes.
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  facet_grid(drv ~ cyl)

Sistemes de coordenades

Coord_Flip

  • coord_flip() gira els eixos x i y, cosa que és útil per a diagrames de caixa o gràfics de barres horitzontals.
ggplot(mpg, aes(x=class, y=hwy)) +
  geom_boxplot() +
  coord_flip()

Coord_Polar

  • coord_polar() transforma el gràfic en un sistema de coordenades polars, útil per crear diagrames de sectors.
ggplot(mpg, aes(x=factor(1), fill=class)) +
  geom_bar(width=1) +
  coord_polar(theta="y")

Anotacions avançades

Afegir anotacions personalitzades

  • Pots afegir anotacions personalitzades utilitzant annotate().
ggplot(mpg, aes(x=displ, y=hwy)) +
  geom_point() +
  annotate("text", x=6, y=40, label="Alta eficiència", color="red", size=5, angle=45)

Combinació de múltiples gràfics

Utilitzant Patchwork

  • El paquet patchwork et permet combinar múltiples gràfics de ggplot2 en un.
library(patchwork)

p1 <- ggplot(mpg, aes(x=displ, y=hwy)) + geom_point()
p2 <- ggplot(mpg, aes(x=class, fill=class)) + geom_bar()

p1 + p2

Conclusió

En aquesta secció, hem explorat tècniques avançades de ggplot2 per millorar les teves habilitats de visualització de dades en R. Hem cobert la personalització de l'aparença del gràfic, l'ús de geometries avançades, la facetació, els sistemes de coordenades, l'afegiment d'anotacions i la combinació de múltiples gràfics. Aquestes tècniques t'ajudaran a crear gràfics més informatius i visualment atractius, fent que la teva anàlisi de dades sigui més efectiva i atractiva.
