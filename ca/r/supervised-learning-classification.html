Introducció a l'Aprenentatge Supervisat

L'aprenentatge supervisat és un tipus d'aprenentatge automàtic on el model es forma amb dades etiquetades. L'objectiu és aprendre una correspondència entre les entrades i les sortides basant-se en les etiquetes proporcionades.

  • Dades Etiquetades: Dades que inclouen tant les característiques d'entrada com les etiquetes de sortida corresponents.
  • Fase d'Entrenament: El model aprèn a partir de les dades etiquetades.
  • Fase de Predicció: El model prediu etiquetes per a noves dades no vistes.

Què és la Classificació?

La classificació és un tipus d'aprenentatge supervisat on la variable de sortida és categòrica. L'objectiu és predir la categoria o classe de noves observacions basant-se en observacions passades.

  • Classificació Binària: Dues possibles classes (per exemple, correu brossa o no correu brossa).
  • Classificació Multiclasse: Més de dues classes (per exemple, classificar tipus d'animals).

Conceptes Clau en Classificació

  • Característiques: Les variables d'entrada utilitzades per fer prediccions.
  • Etiquetes: Les categories de sortida que volem predir.
  • Conjunt d'Entrenament: El subconjunt de dades utilitzat per entrenar el model.
  • Conjunt de Prova: El subconjunt de dades utilitzat per avaluar el rendiment del model.

Algoritmes de Classificació Comuns

  • Regressió Logística
  • k-Nearest Neighbors (k-NN)
  • Arbres de Decisió
  • Boscos Aleatoris
  • Màquines de Suport Vectorial (SVM)
  • Naive Bayes

Implementació de la Classificació en R

Regressió Logística

La regressió logística s'utilitza per a problemes de classificació binària. Modela la probabilitat que una entrada donada pertanyi a una classe particular.

# Carregar la biblioteca necessària
library(caret)

# Carregar el conjunt de dades
data(iris)
# Utilitzar només dues classes per a la classificació binària
iris_binary <- subset(iris, Species != "setosa")

# Dividir les dades en conjunts d'entrenament i prova
set.seed(123)
trainIndex <- createDataPartition(iris_binary$Species, p = .8, 
                                  list = FALSE, 
                                  times = 1)
irisTrain <- iris_binary[ trainIndex,]
irisTest  <- iris_binary[-trainIndex,]

# Entrenar el model de regressió logística
model <- train(Species ~ ., data = irisTrain, method = "glm", family = "binomial")

# Fer prediccions
predictions <- predict(model, irisTest)

# Avaluar el model
confusionMatrix(predictions, irisTest$Species)

k-Nearest Neighbors (k-NN)

k-NN és un algoritme senzill basat en instàncies que classifica una nova mostra basant-se en la classe majoritària entre els seus k veïns més propers.

# Carregar la biblioteca necessària
library(class)

# Preparar les dades
trainData <- irisTrain[, -5]
trainLabels <- irisTrain$Species
testData <- irisTest[, -5]
testLabels <- irisTest$Species

# Entrenar i predir utilitzant k-NN
k <- 3
predictions <- knn(trainData, testData, trainLabels, k = k)

# Avaluar el model
confusionMatrix(predictions, testLabels)

Arbres de Decisió

Els arbres de decisió classifiquen les dades dividint-les en subconjunts basant-se en el valor de les característiques d'entrada.

# Carregar la biblioteca necessària
library(rpart)

# Entrenar el model d'arbre de decisió
model <- rpart(Species ~ ., data = irisTrain, method = "class")

# Fer prediccions
predictions <- predict(model, irisTest, type = "class")

# Avaluar el model
confusionMatrix(predictions, irisTest$Species)

Mètriques d'Avaluació del Model

  • Precisió: La proporció d'instàncies classificades correctament.
  • Precisió (Precision): La proporció d'instàncies positives veritables entre les instàncies classificades com a positives.
  • Sensibilitat (Recall): La proporció d'instàncies positives veritables entre totes les instàncies realment positives.
  • F1 Score: La mitjana harmònica de la precisió i la sensibilitat.

| Mètrica | Fórmula | Interpretació | |------------|----------------------------------------------|-----------------------------------------------------| | Precisió | (TP + TN) / (TP + TN + FP + FN) | Correcció global del model | | Precisió | TP / (TP + FP) | Correcció de les prediccions positives | | Sensibilitat| TP / (TP + FN) | Capacitat de trobar totes les instàncies positives | | F1 Score | 2 * (Precisió * Sensibilitat) / (Precisió + Sensibilitat) | Equilibri entre precisió i sensibilitat |
