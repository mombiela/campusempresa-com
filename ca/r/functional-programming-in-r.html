Programació funcional en R
Introducció a l'Aprenentatge Automàtic ►

La programació funcional és un paradigma de programació que tracta la computació com l'avaluació de funcions matemàtiques i evita l'estat canviant i les dades mutables. En R, la programació funcional pot conduir a un codi més concís, llegible i mantenible. Aquest tema et guiarà des dels conceptes bàsics fins als avançats de la programació funcional en R.

Introducció a la Programació Funcional

  • Definició: La programació funcional és un estil de programació on el mètode bàsic de computació és l'aplicació de funcions a arguments.
  • Conceptes Clau:
    • Funcions Pures: Funcions que no tenen efectes secundaris i retornen el mateix resultat per la mateixa entrada.
    • Funcions de Primera Classe: Funcions que poden ser tractades com a valors, passades com a arguments i retornades des d'altres funcions.
    • Funcions d'Ordre Superior: Funcions que prenen altres funcions com a arguments o les retornen com a resultats.
    • Immutabilitat: Els objectes de dades són immutables, és a dir, no es poden modificar després de ser creats.

Funcions Pures

Les funcions pures són els blocs de construcció de la programació funcional. Asseguren que la sortida de la funció es determini només pels seus valors d'entrada, sense efectes secundaris observables.

Exemple d'una Funció Pura

# Exemple de funció pura: quadrat d'un número
square <- function(x) {
  return(x * x)
}

# Prova de la funció pura
square(4)  # Retorna 16

Funcions de Primera Classe

En R, les funcions són ciutadans de primera classe. Això significa que pots assignar-les a variables, passar-les com a arguments i retornar-les des d'altres funcions.

Exemple de Funcions de Primera Classe

# Assignar una funció a una variable
add <- function(x, y) {
  return(x + y)
}

# Utilitzar la funció
result <- add(3, 5)  # Retorna 8

Funcions d'Ordre Superior

Les funcions d'ordre superior són funcions que poden prendre altres funcions com a arguments o retornar-les com a resultats. Són una eina poderosa en la programació funcional.

Exemple de Funcions d'Ordre Superior

# Funció d'ordre superior: aplicar una funció a una llista
apply_function <- function(func, values) {
  return(sapply(values, func))
}

# Definir una funció simple per utilitzar
double <- function(x) {
  return(x * 2)
}

# Utilitzar la funció d'ordre superior
result <- apply_function(double, c(1, 2, 3, 4))  # Retorna c(2, 4, 6, 8)

Immutabilitat

L'immutabilitat és un concepte clau en la programació funcional. Significa que un cop es crea un objecte de dades, no es pot canviar. En canvi, es creen nous objectes.

Exemple d'Immutabilitat

# Exemple de dades immutables
original_vector <- c(1, 2, 3)

# En lloc de modificar el vector original, crear-ne un de nou
new_vector <- c(original_vector, 4)

# original_vector roman inalterat
print(original_vector)  # Retorna c(1, 2, 3)
print(new_vector)       # Retorna c(1, 2, 3, 4)

Eines de Programació Funcional en R

R proporciona diverses funcions integrades i paquets que faciliten la programació funcional.

  • lapply i sapply: Aplicar una funció sobre una llista o vector.
  • apply: Aplicar una funció sobre els marges d'un array o matriu.
  • tapply: Aplicar una funció sobre subconjunts d'un vector.
  • Paquet purrr: Proporciona un conjunt complet i consistent d'eines per treballar amb funcions i vectors.

Exemple Usant `lapply` i `sapply`

# Usant lapply per aplicar una funció a cada element d'una llista
list_data <- list(a = 1, b = 2, c = 3)
result <- lapply(list_data, function(x) x * 2)
print(result)  # Retorna list(a = 2, b = 4, c = 6)

# Usant sapply per simplificar el resultat a un vector
result <- sapply(list_data, function(x) x * 2)
print(result)  # Retorna c(a = 2, b = 4, c = 6)

Exemple Usant el Paquet `purrr`

# Instal·lar i carregar el paquet purrr
install.packages("purrr")
library(purrr)

# Usant la funció map del paquet purrr
result <- map(list_data, ~ .x * 2)
print(result)  # Retorna list(a = 2, b = 4, c = 6)

Conclusió

La programació funcional en R ofereix un paradigma poderós per escriure codi net, eficient i mantenible. Entenent i aplicant conceptes com funcions pures, funcions de primera classe, funcions d'ordre superior i immutabilitat, pots aprofitar les capacitats de programació funcional de R per resoldre problemes complexos de manera més efectiva. A més, les funcions integrades de R i el paquet purrr proporcionen eines robustes per facilitar les pràctiques de programació funcional.
