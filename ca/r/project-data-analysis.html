Projecte: Visualització de Dades ►

Introducció

En aquest projecte, aplicarem les habilitats apreses al llarg del curs per realitzar anàlisis de dades utilitzant R. Aquest projecte et guiarà a través del procés de càrrega, neteja, anàlisi i visualització de dades. Al final d'aquest projecte, tindràs una comprensió completa de com manejar dades del món real utilitzant R.

Càrrega de Dades

Per començar qualsevol projecte d'anàlisi de dades, el primer pas és carregar les dades a R.

Conceptes Clau

  • Llegir dades de diverses fonts (CSV, Excel, bases de dades).
  • Comprendre els data frames i tibbles.

Exemple

# Carregar les biblioteques necessàries
library(readr)
library(readxl)

# Llegir dades d'un fitxer CSV
data_csv <- read_csv("path/to/your/data.csv")

# Llegir dades d'un fitxer Excel
data_excel <- read_excel("path/to/your/data.xlsx")

# Mostrar les primeres files de les dades
head(data_csv)
head(data_excel)

Neteja de Dades

La neteja de dades és un pas crucial per assegurar que les dades estiguin en el format adequat per a l'anàlisi.

Conceptes Clau

  • Manejar valors perduts.
  • Eliminar duplicats.
  • Convertir tipus de dades.

Exemple

# Carregar la biblioteca necessària
library(dplyr)

# Eliminar files amb valors perduts
cleaned_data <- data_csv %>%
  drop_na()

# Eliminar files duplicades
cleaned_data <- cleaned_data %>%
  distinct()

# Convertir una columna a un tipus de dada diferent
cleaned_data$column_name <- as.numeric(cleaned_data$column_name)

# Mostrar les dades netejades
head(cleaned_data)

Anàlisi Exploratori de Dades (EDA)

L'EDA implica resumir les característiques principals de les dades, sovint utilitzant mètodes visuals.

Conceptes Clau

  • Estadístiques resum.
  • Visualització de dades (histogrames, diagrames de dispersió, diagrames de caixa).

Exemple

# Carregar les biblioteques necessàries
library(ggplot2)

# Estadístiques resum
summary(cleaned_data)

# Histograma d'una columna numèrica
ggplot(cleaned_data, aes(x = numeric_column)) +
  geom_histogram(binwidth = 10, fill = "blue", color = "black") +
  labs(title = "Histograma de la Columna Numèrica", x = "Valor", y = "Freqüència")

# Diagrama de dispersió entre dues columnes numèriques
ggplot(cleaned_data, aes(x = numeric_column1, y = numeric_column2)) +
  geom_point(color = "red") +
  labs(title = "Diagrama de Dispersió", x = "Columna Numèrica 1", y = "Columna Numèrica 2")

Anàlisi de Dades

Realitzar anàlisis de dades per extreure informació significativa.

Conceptes Clau

  • Agrupar i resumir dades.
  • Aplicar proves estadístiques.

Exemple

# Agrupar i resumir dades
grouped_data <- cleaned_data %>%
  group_by(categorical_column) %>%
  summarize(mean_value = mean(numeric_column, na.rm = TRUE),
            count = n())

# Mostrar les dades agrupades
print(grouped_data)

# Aplicar una prova t
t_test_result <- t.test(numeric_column ~ categorical_column, data = cleaned_data)
print(t_test_result)

Visualització de Dades

Crear visualitzacions avançades per presentar els resultats.

Conceptes Clau

  • Tècniques avançades de traçat.
  • Personalitzar gràfics.

Exemple

# Diagrama de caixa amb personalitzacions
ggplot(cleaned_data, aes(x = categorical_column, y = numeric_column)) +
  geom_boxplot(fill = "orange", color = "black") +
  labs(title = "Diagrama de Caixa", x = "Categoria", y = "Valor") +
  theme_minimal()

# Diagrama de línies amb múltiples línies
ggplot(cleaned_data, aes(x = time_column, y = numeric_column, color = categorical_column)) +
  geom_line() +
  labs(title = "Diagrama de Línies", x = "Temps", y = "Valor") +
  theme_minimal()

Conclusió

En aquest projecte, hem cobert tot el flux de treball d'anàlisi de dades utilitzant R, des de la càrrega i neteja de dades fins a la realització d'anàlisi exploratori de dades, anàlisi estadística i creació de visualitzacions. Seguint aquests passos, pots manejar dades del món real i extreure informació significativa, la qual cosa és una habilitat valuosa en ciència de dades i anàlisi. Continua practicant amb diferents conjunts de dades per millorar encara més les teves habilitats d'anàlisi de dades en R.
