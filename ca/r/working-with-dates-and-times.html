Introducció

Treballar amb dates i hores és una habilitat crucial en l'anàlisi de dades i la programació. En R, hi ha diversos paquets i funcions que fan que la gestió de dates i hores sigui senzilla. Aquesta secció cobrirà els conceptes bàsics de la manipulació de dates i hores, avançant des de temes per a principiants fins a temes avançats.

Funcions Bàsiques de Dates i Hores

Obtenir la Data i Hora Actuals

  • Sys.Date(): Retorna la data actual.
  • Sys.time(): Retorna la data i hora actuals.
# Obtenir la data actual
current_date <- Sys.Date()
print(current_date)

# Obtenir la data i hora actuals
current_time <- Sys.time()
print(current_time)

Crear Dates

  • as.Date(): Converteix una cadena de caràcters a un objecte Date.
  • ISOdate(): Crea un objecte de data i hora.
# Convertir una cadena de caràcters a un objecte Date
date_string <- "2023-10-01"
date_object <- as.Date(date_string)
print(date_object)

# Crear un objecte de data i hora
datetime_object <- ISOdate(2023, 10, 1, 12, 0, 0)
print(datetime_object)

Formatant Dates

  • format(): Dona format a un objecte Date en una cadena.
# Donar format a un objecte Date
formatted_date <- format(date_object, "%B %d, %Y")
print(formatted_date)  # Output: "October 01, 2023"

Manipulació Intermèdia de Dates i Hores

Aritmètica de Dates

  • Afegir o restar dies a una data.
  • Calcular la diferència entre dues dates.
# Afegir dies a una data
new_date <- date_object + 10
print(new_date)  # Output: "2023-10-11"

# Calcular la diferència entre dues dates
date1 <- as.Date("2023-10-01")
date2 <- as.Date("2023-10-15")
date_diff <- date2 - date1
print(date_diff)  # Output: Time difference of 14 days

Treballar amb el Paquet `lubridate`

  • lubridate simplifica la manipulació de dates i hores.
# Instal·lar i carregar el paquet lubridate
install.packages("lubridate")
library(lubridate)

# Analitzar dates utilitzant lubridate
parsed_date <- ymd("2023-10-01")
print(parsed_date)

# Extreure components
year_component <- year(parsed_date)
month_component <- month(parsed_date)
day_component <- day(parsed_date)
print(year_component)  # Output: 2023
print(month_component)  # Output: 10
print(day_component)  # Output: 1

Manipulació Avançada de Dates i Hores

Fusos Horaris i Localització

  • Gestionar diferents fusos horaris.
  • Convertir entre fusos horaris.
# Crear un objecte de data i hora amb un fus horari específic
datetime_with_tz <- ymd_hms("2023-10-01 12:00:00", tz = "America/New_York")
print(datetime_with_tz)

# Convertir a un altre fus horari
datetime_utc <- with_tz(datetime_with_tz, tzone = "UTC")
print(datetime_utc)

Treballar amb Períodes i Duracions

  • periods representen intervals de temps amigables per a humans (per exemple, mesos, anys).
  • durations representen intervals de temps exactes (per exemple, segons, minuts).
# Crear períodes
one_month <- months(1)
new_date_period <- date_object + one_month
print(new_date_period)  # Output: "2023-11-01"

# Crear duracions
one_hour <- dhours(1)
new_time_duration <- current_time + one_hour
print(new_time_duration)

Resum

En aquesta secció, hem cobert els conceptes essencials per treballar amb dates i hores en R, començant per funcions bàsiques fins a manipulacions més avançades utilitzant el paquet lubridate. Entendre com gestionar dates i hores és vital per a l'anàlisi de dades, i R proporciona eines robustes per facilitar aquesta tasca. Practiqueu aquests conceptes amb dades del món real per esdevenir proficient en la manipulació de dates i hores en R.
