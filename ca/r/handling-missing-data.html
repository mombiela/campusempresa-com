El tractament de dades faltants és una habilitat crucial en l'anàlisi i manipulació de dades. En aquesta secció, cobrirem diverses tècniques per tractar les dades faltants en R, des de nivells bàsics fins a avançats.

Introducció a les Dades Faltants

  • Definició: Les dades faltants es produeixen quan no es guarda cap valor de dades per a una variable en una observació.
  • Causes Comunes: Errors d'entrada de dades, problemes de recollida de dades o omissió intencional.

Identificació de Dades Faltants

Comprovar Valors Faltants

  • Funció is.na(): Aquesta funció comprova si hi ha valors faltants en un conjunt de dades.
# Conjunt de dades d'exemple
data <- c(1, 2, NA, 4, NA, 6)

# Comprovar valors faltants
is.na(data)
  • Funció anyNA(): Aquesta funció comprova si hi ha algun valor faltant en el conjunt de dades.
# Comprovar si hi ha algun valor faltant
anyNA(data)

Tractament de Dades Faltants

Eliminar Dades Faltants

  • Funció na.omit(): Aquesta funció elimina les files amb valors faltants.
# Conjunt de dades d'exemple
data <- data.frame(
  x = c(1, 2, NA, 4),
  y = c(NA, 2, 3, 4)
)

# Eliminar files amb valors faltants
clean_data <- na.omit(data)

Reemplaçar Dades Faltants

  • Reemplaçar amb un Valor Específic: Utilitza la funció is.na() per reemplaçar els valors faltants amb un valor específic.
# Reemplaçar NA amb 0
data[is.na(data)] <- 0
  • Reemplaçar amb la Mitjana/Mediana/Moda: Calcula la mitjana, mediana o moda i reemplaça els valors faltants.
# Reemplaçar NA amb la mitjana
data <- c(1, 2, NA, 4, NA, 6)
data[is.na(data)] <- mean(data, na.rm = TRUE)

Tècniques Avançades

  • Utilitzant el paquet mice: La imputació múltiple per equacions en cadena (MICE) és un mètode potent per tractar les dades faltants.
# Instal·lar i carregar el paquet mice
install.packages("mice")
library(mice)

# Conjunt de dades d'exemple
data <- data.frame(
  x = c(1, 2, NA, 4),
  y = c(NA, 2, 3, 4)
)

# Imputar dades faltants
imputed_data <- mice(data, m = 5, method = 'pmm', maxit = 50, seed = 500)
complete_data <- complete(imputed_data)

Visualització de Dades Faltants

  • Utilitzant el paquet VIM: Visualitzar patrons de dades faltants.
# Instal·lar i carregar el paquet VIM
install.packages("VIM")
library(VIM)

# Conjunt de dades d'exemple
data <- data.frame(
  x = c(1, 2, NA, 4),
  y = c(NA, 2, 3, 4)
)

# Visualitzar dades faltants
aggr(data, col = c('navyblue', 'red'), numbers = TRUE, sortVars = TRUE, labels = names(data), cex.axis = .7, gap = 3, ylab = c("Dades faltants", "Patró"))

Resum

El tractament de dades faltants és essencial per a una anàlisi de dades precisa. En aquesta secció, hem cobert:

  • Identificació de dades faltants utilitzant is.na() i anyNA().
  • Eliminació de dades faltants amb na.omit().
  • Reemplaçament de dades faltants amb valors específics, mitjana, mediana o moda.
  • Tècniques avançades d'imputació utilitzant el paquet mice.
  • Visualització de patrons de dades faltants amb el paquet VIM.

En dominar aquestes tècniques, pots assegurar-te que la teva anàlisi de dades sigui robusta i fiable, fins i tot en presència de dades faltants.
