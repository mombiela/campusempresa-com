Estructures de control en R
Depuració i Gestió d'Errors ►

Les estructures de control són fonamentals en la programació ja que et permeten controlar el flux d'execució del teu codi. En R, les estructures de control inclouen condicionals, bucles i altres construccions que et permeten escriure programes més complexos i dinàmics. Aquesta secció cobrirà les estructures de control essencials en R, començant pels conceptes bàsics i avançant cap a conceptes més avançats.

Condicionals

Els condicionals et permeten executar diferents blocs de codi basats en certes condicions.

Instrucció if

La instrucció if s'utilitza per executar un bloc de codi només si una condició especificada és certa.

x <- 10

if (x > 5) {
  print("x és més gran que 5")
}

Instrucció if-else

La instrucció if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

x <- 3

if (x > 5) {
  print("x és més gran que 5")
} else {
  print("x no és més gran que 5")
}

Funció ifelse

La funció ifelse és una versió vectoritzada de la instrucció if-else, útil per aplicar lògica condicional a vectors.

x <- c(1, 2, 3, 4, 5)
result <- ifelse(x > 3, "Més gran", "No més gran")
print(result)

Bucles

Els bucles et permeten executar un bloc de codi múltiples vegades.

Bucle for

El bucle for s'utilitza per iterar sobre una seqüència d'elements.

for (i in 1:5) {
  print(i)
}

Bucle while

El bucle while continua executant un bloc de codi mentre una condició especificada sigui certa.

x <- 1

while (x <= 5) {
  print(x)
  x <- x + 1
}

Bucle repeat

El bucle repeat executa un bucle infinit fins que es troba una instrucció break.

x <- 1

repeat {
  print(x)
  x <- x + 1
  if (x > 5) {
    break
  }
}

Estructures de control avançades

break i next

La instrucció break s'utilitza per sortir d'un bucle, mentre que la instrucció next s'utilitza per saltar la iteració actual i procedir a la següent.

for (i in 1:10) {
  if (i == 5) {
    break
  }
  print(i)
}

for (i in 1:10) {
  if (i %% 2 == 0) {
    next
  }
  print(i)
}

Instrucció switch

La instrucció switch et permet executar un dels diversos blocs de codi basats en el valor d'una expressió.

x <- "b"

result <- switch(x,
  "a" = "Opció A",
  "b" = "Opció B",
  "c" = "Opció C",
  "Opció no vàlida"
)

print(result)

Resum

Les estructures de control són essencials per gestionar el flux d'execució en els teus programes R. Dominant els condicionals, els bucles i les estructures de control avançades, pots escriure codi més eficient i dinàmic. Aquí tens un breu resum:

  • Condicionals: Utilitza if, if-else i ifelse per executar codi basat en condicions.
  • Bucles: Utilitza for, while i repeat per executar codi múltiples vegades.
  • Estructures de control avançades: Utilitza break, next i switch per a un control de flux més complex.

Entendre i utilitzar efectivament aquestes estructures de control millorarà significativament la teva capacitat per escriure programes R robustos i flexibles.
