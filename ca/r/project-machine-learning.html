Introducció al Machine Learning en R

El Machine Learning (ML) és un camp de la intel·ligència artificial que utilitza tècniques estadístiques per donar als sistemes informàtics la capacitat de "aprendre" a partir de dades. En aquest projecte, explorarem com implementar algoritmes de machine learning utilitzant R, un llenguatge potent per al càlcul estadístic i gràfics.

Conceptes Clau

  • Aprenentatge Supervisat: Aprenentatge a partir de dades etiquetades (per exemple, classificació, regressió).
  • Aprenentatge No Supervisat: Aprenentatge a partir de dades no etiquetades (per exemple, agrupament, associació).
  • Avaluació del Model: Tècniques per avaluar el rendiment dels models de ML (per exemple, validació creuada, matriu de confusió).

Configuració de l'Entorn

Abans d'endinsar-nos en el machine learning, hem de configurar el nostre entorn R.

Instal·lació de Paquets Necessaris

install.packages("caret")
install.packages("randomForest")
install.packages("e1071")
install.packages("ggplot2")

Carregar els Paquets

library(caret)
library(randomForest)
library(e1071)
library(ggplot2)

Preprocessament de Dades

El preprocessament de dades és un pas crucial en el machine learning. Implica netejar i transformar dades en brut en un format adequat per al modelatge.

Carregar i Explorar el Conjunt de Dades

# Carregar el conjunt de dades
data(iris)
# Veure les primeres files
head(iris)

Tractament de Valors Perduts

# Comprovar valors perduts
sum(is.na(iris))

# Imputar valors perduts (si n'hi ha)
iris[is.na(iris)] <- median(iris, na.rm = TRUE)

Escalat de Característiques

# Normalitzar les característiques
normalize <- function(x) {
  return ((x - min(x)) / (max(x) - min(x)))
}

iris_norm <- as.data.frame(lapply(iris[1:4], normalize))
iris_norm$Species <- iris$Species

Aprenentatge Supervisat

Classificació amb Random Forest

Random Forest és un mètode d'aprenentatge en conjunt per a classificació i regressió.

Entrenament del Model

# Dividir les dades en conjunts d'entrenament i prova
set.seed(123)
trainIndex <- createDataPartition(iris_norm$Species, p = .8, 
                                  list = FALSE, 
                                  times = 1)
irisTrain <- iris_norm[ trainIndex,]
irisTest  <- iris_norm[-trainIndex,]

# Entrenar el model Random Forest
model_rf <- randomForest(Species ~ ., data = irisTrain)
print(model_rf)

Fer Prediccions

# Predir en el conjunt de prova
predictions <- predict(model_rf, irisTest)

# Avaluar el model
confusionMatrix(predictions, irisTest$Species)

Regressió amb Suport Vector Machines (SVM)

Suport Vector Machines també es poden utilitzar per a tasques de regressió.

Entrenament del Model

# Entrenar el model SVM
model_svm <- svm(Sepal.Length ~ ., data = irisTrain)
print(model_svm)

Fer Prediccions

# Predir en el conjunt de prova
predictions_svm <- predict(model_svm, irisTest)

# Avaluar el model
postResample(predictions_svm, irisTest$Sepal.Length)

Aprenentatge No Supervisat

Agrupament amb K-Means

K-Means és un algoritme popular d'agrupament.

Entrenament del Model

# Determinar el nombre òptim de clústers
wss <- (nrow(iris_norm)-1)*sum(apply(iris_norm[,1:4],2,var))
for (i in 2:15) wss[i] <- sum(kmeans(iris_norm[,1:4], centers=i)$withinss)

# Gràfic del mètode del colze
plot(1:15, wss, type="b", xlab="Nombre de Clústers", ylab="Suma de quadrats dins dels grups")

# Entrenar el model K-Means
set.seed(123)
kmeans_model <- kmeans(iris_norm[,1:4], centers=3)
print(kmeans_model)

Visualització dels Clústers

# Visualitzar els clústers
iris_norm$Cluster <- as.factor(kmeans_model$cluster)
ggplot(iris_norm, aes(Petal.Length, Petal.Width, color = Cluster)) + geom_point()

Avaluació del Model

Avaluar el rendiment dels models de machine learning és essencial per entendre la seva efectivitat.

Validació Creuada

# Realitzar validació creuada de 10 plecs
control <- trainControl(method="cv", number=10)
model_cv <- train(Species ~ ., data=irisTrain, method="rf", trControl=control)
print(model_cv)

Matriu de Confusió

# Matriu de confusió per al model Random Forest
confusionMatrix(predictions, irisTest$Species)
