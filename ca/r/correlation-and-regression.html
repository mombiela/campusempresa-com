Introducció

La correlació i la regressió són conceptes fonamentals en estadística i anàlisi de dades. Ens ajuden a entendre les relacions entre variables i fer prediccions. En aquesta secció, explorarem aquests conceptes utilitzant R, un llenguatge potent per al càlcul estadístic.

Correlació

Què és la Correlació?

  • Definició: La correlació mesura la força i la direcció de la relació lineal entre dues variables.
  • Tipus de Correlació:
    • Correlació Positiva: Quan una variable augmenta, l'altra variable també augmenta.
    • Correlació Negativa: Quan una variable augmenta, l'altra variable disminueix.
    • Sense Correlació: No hi ha relació lineal entre les variables.

Calculant la Correlació en R

R proporciona diverses funcions per calcular la correlació, la més comuna és cor().

# Dades de mostra
x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)

# Calcular la correlació
correlation <- cor(x, y)
print(correlation)

Interpretant els Coeficients de Correlació

  • Rang: -1 a 1
  • Interpretació:
    • 1: Correlació positiva perfecta
    • -1: Correlació negativa perfecta
    • 0: Sense correlació

Exemple: Matriu de Correlació

Una matriu de correlació mostra els coeficients de correlació entre múltiples variables.

# Dades de mostra
data <- data.frame(
  var1 = c(1, 2, 3, 4, 5),
  var2 = c(2, 4, 6, 8, 10),
  var3 = c(5, 4, 3, 2, 1)
)

# Calcular la matriu de correlació
cor_matrix <- cor(data)
print(cor_matrix)

Regressió

Què és la Regressió?

  • Definició: L'anàlisi de regressió estima les relacions entre variables. Ens permet predir el valor d'una variable dependent basant-nos en una o més variables independents.
  • Tipus de Regressió:
    • Regressió Lineal Simple: Una variable independent.
    • Regressió Lineal Múltiple: Més d'una variable independent.

Regressió Lineal Simple en R

La regressió lineal simple modela la relació entre dues variables ajustant una equació lineal a les dades observades.

# Dades de mostra
x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)

# Ajustar el model lineal
model <- lm(y ~ x)

# Resum del model
summary(model)

Interpretant els Resultats de la Regressió

  • Coeficients: Estimacions dels paràmetres de la regressió.
  • R-quadrat: Proporció de la variància explicada pel model.
  • p-valor: Significança dels predictors.

Exemple: Traçant la Línia de Regressió

Visualitzar la línia de regressió ajuda a entendre l'ajust del model.

# Dades de mostra
x <- c(1, 2, 3, 4, 5)
y <- c(2, 4, 6, 8, 10)

# Ajustar el model lineal
model <- lm(y ~ x)

# Traçar les dades
plot(x, y, main="Regressió Lineal Simple", xlab="X", ylab="Y")

# Afegir la línia de regressió
abline(model, col="red")

Regressió Lineal Múltiple en R

La regressió lineal múltiple modela la relació entre una variable dependent i múltiples variables independents.

# Dades de mostra
data <- data.frame(
  y = c(2, 4, 6, 8, 10),
  x1 = c(1, 2, 3, 4, 5),
  x2 = c(5, 4, 3, 2, 1)
)

# Ajustar el model de regressió lineal múltiple
model <- lm(y ~ x1 + x2, data=data)

# Resum del model
summary(model)

Conclusió

Entendre la correlació i la regressió és crucial per a l'anàlisi de dades i la modelització predictiva. La correlació ens ajuda a identificar relacions entre variables, mentre que la regressió ens permet modelar i predir resultats. R proporciona eines potents per realitzar aquestes anàlisis de manera eficient. Practica amb conjunts de dades reals per enfortir la teva comprensió i habilitats en aquestes àrees.
