Introducció

L'avaluació i l'ajust del model són passos crítics en el flux de treball de l'aprenentatge automàtic. Asseguren que el model que construeixes no només sigui precís, sinó que també es generalitzi bé a dades no vistes. En aquesta secció, cobrirem diverses tècniques i mètodes per avaluar i ajustar models en R.

Conceptes Clau

  • Avaluació del Model: Avaluar el rendiment d'un model utilitzant diverses mètriques.
  • Ajust del Model: Ajustar els hiperparàmetres d'un model per millorar el seu rendiment.
  • Validació Creuada: Una tècnica per avaluar el rendiment del model dividint les dades en múltiples conjunts d'entrenament i prova.
  • Hiperparàmetres: Paràmetres que no s'aprenen a partir de les dades, sinó que es fixen abans que comenci el procés d'entrenament.

Mètriques d'Avaluació del Model

Mètriques de Classificació

  • Precisió: La proporció d'instàncies correctament predites respecte al total d'instàncies.
  • Precisió (Precision): La proporció d'observacions positives correctament predites respecte al total de prediccions positives.
  • Record (Sensibilitat): La proporció d'observacions positives correctament predites respecte a totes les observacions en la classe real.
  • F1 Score: La mitjana ponderada de Precisió i Record.

Mètriques de Regressió

  • Error Absolut Mitjà (MAE): La mitjana dels errors absoluts.
  • Error Quadràtic Mitjà (MSE): La mitjana dels errors quadràtics.
  • Arrel de l'Error Quadràtic Mitjà (RMSE): L'arrel quadrada de la mitjana dels errors quadràtics.
  • R-quadrat (R²): La proporció de la variància en la variable dependent que és previsible a partir de les variables independents.

Exemple: Avaluació d'un Model de Classificació

# Carregar les biblioteques necessàries
library(caret)
library(e1071)

# Carregar el conjunt de dades
data(iris)

# Dividir les dades en conjunts d'entrenament i prova
set.seed(123)
trainIndex <- createDataPartition(iris$Species, p = .8, 
                                  list = FALSE, 
                                  times = 1)
irisTrain <- iris[ trainIndex,]
irisTest  <- iris[-trainIndex,]

# Entrenar un model
model <- train(Species ~ ., data = irisTrain, method = "svmLinear")

# Fer prediccions
predictions <- predict(model, irisTest)

# Avaluar el model
confusionMatrix(predictions, irisTest$Species)

Validació Creuada

La validació creuada és un mètode robust per avaluar el rendiment d'un model. El tipus més comú és la validació creuada k-fold.

Exemple: Validació Creuada k-Fold

# Realitzar validació creuada k-fold
control <- trainControl(method="cv", number=10)
model <- train(Species ~ ., data=iris, method="svmLinear", trControl=control)

# Imprimir els resultats
print(model)

Ajust del Model

L'ajust del model implica ajustar els hiperparàmetres per millorar el rendiment del model. El paquet caret en R proporciona una manera convenient de realitzar l'ajust d'hiperparàmetres.

Exemple: Ajust d'Hiperparàmetres

# Definir la graella de paràmetres
tuneGrid <- expand.grid(C = seq(0.1, 1, by = 0.1))

# Entrenar el model amb ajust
model <- train(Species ~ ., data=iris, method="svmLinear", 
               trControl=control, tuneGrid=tuneGrid)

# Imprimir el millor model
print(model$bestTune)

Conclusió

L'avaluació i l'ajust del model són passos essencials per assegurar que els teus models d'aprenentatge automàtic funcionin bé i es generalitzin a noves dades. Utilitzant tècniques com la validació creuada i l'ajust d'hiperparàmetres, pots millorar significativament el rendiment dels teus models. En aquesta secció, hem cobert els conceptes bàsics de les mètriques d'avaluació del model, la validació creuada i l'ajust d'hiperparàmetres en R. Amb aquestes eines, estàs ben equipat per construir models robustos i precisos.
