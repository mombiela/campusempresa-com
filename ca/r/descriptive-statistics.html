Les estadístiques descriptives són essencials per resumir i entendre les característiques principals d'un conjunt de dades. En aquesta secció, cobrirem diverses tècniques d'estadístiques descriptives utilitzant R, des de nivells bàsics fins a avançats.

Introducció a les Estadístiques Descriptives

Les estadístiques descriptives proporcionen resums simples sobre la mostra i les mesures. Aquests resums poden ser quantitatius (per exemple, mitjana, mediana) o visuals (per exemple, gràfics, diagrames).

Conceptes Clau

  • Mitjana: La mitjana d'un conjunt de números.
  • Mediana: El valor central en un conjunt de números.
  • Moda: El valor que es repeteix amb més freqüència en un conjunt de números.
  • Rang: La diferència entre els valors més alts i més baixos.
  • Variància: Una mesura de com els valors en un conjunt de dades difereixen de la mitjana.
  • Desviació Estàndard: L'arrel quadrada de la variància, indicant com de dispersos estan els valors.
  • Quartils: Valors que divideixen un conjunt de dades en quatre parts iguals.
  • Rang Interquartílic (IQR): El rang entre el primer quartil (Q1) i el tercer quartil (Q3).

Estadístiques Descriptives Bàsiques en R

Càlcul de la Mitjana, Mediana i Moda

# Dades de mostra
data <- c(10, 20, 20, 30, 40, 50, 60)

# Mitjana
mean_value <- mean(data)
print(paste("Mitjana:", mean_value))

# Mediana
median_value <- median(data)
print(paste("Mediana:", median_value))

# Moda (funció personalitzada ja que R no té una funció de moda incorporada)
get_mode <- function(v) {
  uniqv <- unique(v)
  uniqv[which.max(tabulate(match(v, uniqv)))]
}
mode_value <- get_mode(data)
print(paste("Moda:", mode_value))

Càlcul del Rang, Variància i Desviació Estàndard

# Rang
range_value <- range(data)
print(paste("Rang:", range_value))

# Variància
variance_value <- var(data)
print(paste("Variància:", variance_value))

# Desviació Estàndard
sd_value <- sd(data)
print(paste("Desviació Estàndard:", sd_value))

Càlcul dels Quartils i IQR

# Quartils
quartiles <- quantile(data)
print("Quartils:")
print(quartiles)

# Rang Interquartílic (IQR)
iqr_value <- IQR(data)
print(paste("Rang Interquartílic (IQR):", iqr_value))

Visualització de les Estadístiques Descriptives

Histogrames

# Histograma
hist(data, main="Histograma de les Dades", xlab="Valors", ylab="Freqüència", col="blue")

Diagrames de Caixa

# Diagrama de Caixa
boxplot(data, main="Diagrama de Caixa de les Dades", ylab="Valors", col="red")

Estadístiques Descriptives Avançades en R

Estadístiques Resum amb la Funció `summary()`

# Estadístiques resum
summary_stats <- summary(data)
print("Estadístiques Resum:")
print(summary_stats)

Utilitzant el Paquet `dplyr` per a Estadístiques Descriptives

# Instal·lar i carregar el paquet dplyr
install.packages("dplyr")
library(dplyr)

# Crear un data frame
df <- data.frame(values = data)

# Estadístiques resum utilitzant dplyr
summary_dplyr <- df %>%
  summarise(
    mean = mean(values),
    median = median(values),
    mode = get_mode(values),
    variance = var(values),
    sd = sd(values),
    range = diff(range(values)),
    IQR = IQR(values)
  )
print("Estadístiques Resum utilitzant dplyr:")
print(summary_dplyr)

Conclusió

Les estadístiques descriptives són fonamentals per entendre i resumir dades. R proporciona una varietat de funcions i paquets per calcular i visualitzar aquestes estadístiques. En dominar aquestes tècniques, pots obtenir informació valuosa sobre les teves dades, cosa que és crucial per a qualsevol tasca d'anàlisi de dades.

En aquesta secció, hem cobert:

  • Càlculs bàsics d'estadístiques descriptives.
  • Tècniques de visualització utilitzant histogrames i diagrames de caixa.
  • Estadístiques resum avançades utilitzant la funció summary() i el paquet dplyr.

Amb aquestes eines, estàs ben equipat per realitzar una anàlisi estadística descriptiva completa en R.
