Introducció als Data Frames

Els data frames són una estructura de dades crucial a R, utilitzada per emmagatzemar dades tabulars. Són similars a les taules en una base de dades o fulls de càlcul d'Excel. Cada columna en un data frame pot contenir diferents tipus de dades (numèric, caràcter, factor, etc.), fent-los versàtils per a l'anàlisi de dades.

Conceptes Clau

  • Estructura del Data Frame: Les files representen observacions, i les columnes representen variables.
  • Creació de Data Frames: Utilitzant la funció data.frame().
  • Accés a les Dades: Utilitzant indexació, noms de columnes i funcions com subset().

Creació de Data Frames

Per crear un data frame, pots utilitzar la funció data.frame(). Aquí tens un exemple senzill:

# Creació d'un data frame
df <- data.frame(
  Name = c("Alice", "Bob", "Charlie"),
  Age = c(25, 30, 35),
  Height = c(5.5, 6.0, 5.8)
)

# Mostrar el data frame
print(df)

Explicació

  • Name: Un vector de caràcters que representa noms.
  • Age: Un vector numèric que representa edats.
  • Height: Un vector numèric que representa alçades.

Accés a les Dades en Data Frames

Pots accedir a les dades en un data frame utilitzant diversos mètodes:

Per Índex

# Accés a la primera fila
print(df[1, ])

# Accés a la primera columna
print(df[, 1])

# Accés a un element específic
print(df[1, 2])

Per Nom de Columna

# Accés a la columna 'Name'
print(df$Name)

# Accés a la columna 'Age'
print(df[["Age"]])

Utilitzant la Funció `subset()`

# Subset de dades on Age és major que 25
subset_df <- subset(df, Age > 25)
print(subset_df)

Modificació de Data Frames

Pots modificar data frames afegint o eliminant columnes i files.

Afegir Columnes

# Afegir una nova columna 'Weight'
df$Weight <- c(120, 150, 140)
print(df)

Eliminar Columnes

# Eliminar la columna 'Height'
df$Height <- NULL
print(df)

Afegir Files

# Afegir una nova fila
new_row <- data.frame(Name = "David", Age = 40, Weight = 160)
df <- rbind(df, new_row)
print(df)

Eliminar Files

# Eliminar la primera fila
df <- df[-1, ]
print(df)

Operacions Avançades amb Data Frames

A mesura que et familiaritzis més amb els data frames, podràs realitzar operacions més complexes.

Fusió de Data Frames

Pots fusionar dos data frames utilitzant la funció merge().

# Creació d'un altre data frame
df2 <- data.frame(
  Name = c("Alice", "Bob", "David"),
  Salary = c(50000, 60000, 70000)
)

# Fusió de data frames per 'Name'
merged_df <- merge(df, df2, by = "Name")
print(merged_df)

Aplicació de Funcions a Data Frames

Pots aplicar funcions a data frames utilitzant apply(), lapply() i sapply().

# Aplicació d'una funció a cada columna
mean_values <- sapply(df[, -1], mean)
print(mean_values)

Resum

Els data frames són una estructura de dades fonamental a R, essencial per a la manipulació i anàlisi de dades. Et permeten emmagatzemar i operar amb dades tabulars de manera eficient. Comprenent com crear, accedir, modificar i realitzar operacions avançades amb data frames, podràs gestionar una àmplia gamma de tasques d'anàlisi de dades a R.

Punts Clau

  • Els data frames emmagatzemen dades tabulars amb files i columnes.
  • Pots crear data frames utilitzant la funció data.frame().
  • Accedeix a les dades utilitzant indexació, noms de columnes i la funció subset().
  • Modifica data frames afegint/eliminant columnes i files.
  • Realitza operacions avançades com la fusió i l'aplicació de funcions a data frames.

Dominant aquests conceptes, estaràs ben preparat per gestionar tasques d'anàlisi de dades a R.
