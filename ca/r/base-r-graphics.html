Introducció als Gràfics Base R

Els gràfics base R proporcionen un sistema potent i flexible per crear una gran varietat de gràfics i visualitzacions directament dins de R. Aquesta secció us introduirà als conceptes i funcions fonamentals utilitzats en els gràfics base R.

  • Conceptes Clau:
    • Dispositius Gràfics: El mitjà on es mostra el gràfic (per exemple, pantalla, fitxer).
    • Funcions de Gràfics: Funcions utilitzades per crear diferents tipus de gràfics.
    • Personalització de Gràfics: Modificar l'aparença dels gràfics utilitzant diversos paràmetres.

Funcions Bàsiques de Gràfics

Base R ofereix diverses funcions per crear gràfics bàsics. Aquestes funcions són fàcils d'utilitzar i proporcionen un bon punt de partida per als principiants.

  • Crear un Gràfic de Dispersió Simple:

    # Generar algunes dades de mostra
x <- 1:10
y <- x^2

# Crear un gràfic de dispersió
plot(x, y, main="Gràfic de Dispersió", xlab="Eix X", ylab="Eix Y")

    Explicació:

    • plot(x, y): Crea un gràfic de dispersió amb valors x i y.
    • main, xlab, ylab: Afegeix un títol i etiquetes als eixos.

  • Gràfic de Línies:

    # Crear un gràfic de línies
plot(x, y, type="l", main="Gràfic de Línies", xlab="Eix X", ylab="Eix Y")

    Explicació:

    • type="l": Especifica que el gràfic ha de ser un gràfic de línies.

  • Gràfic de Barres:

    # Crear un gràfic de barres
barplot(y, names.arg=x, main="Gràfic de Barres", xlab="Eix X", ylab="Eix Y")

    Explicació:

    • barplot(y): Crea un gràfic de barres amb valors y.
    • names.arg=x: Etiqueta les barres amb valors x.

Personalització de Gràfics

Personalitzar els gràfics us permet millorar la llegibilitat i l'estètica de les vostres visualitzacions.

  • Canviar Colors i Símbols del Gràfic:

    # Gràfic de dispersió amb colors i símbols personalitzats
plot(x, y, col="blue", pch=19, main="Gràfic de Dispersió Personalitzat", xlab="Eix X", ylab="Eix Y")

    Explicació:

    • col: Canvia el color dels punts.
    • pch: Canvia el símbol del gràfic.

  • Afegir Línies de Quadrícula:

    # Gràfic de dispersió amb línies de quadrícula
plot(x, y, main="Gràfic de Dispersió amb Quadrícula", xlab="Eix X", ylab="Eix Y")
grid()

    Explicació:

    • grid(): Afegeix línies de quadrícula al gràfic.

  • Afegir Llegendes:

    # Gràfic de dispersió amb una llegenda
plot(x, y, col="blue", pch=19, main="Gràfic de Dispersió amb Llegenda", xlab="Eix X", ylab="Eix Y")
legend("topleft", legend="Punts de Dades", col="blue", pch=19)

    Explicació:

    • legend(): Afegeix una llegenda al gràfic. La posició ("topleft") i altres paràmetres especifiquen l'aparença de la llegenda.

Tècniques Avançades de Gràfics

Per a visualitzacions més complexes, base R proporciona funcions i tècniques addicionals.

  • Diversos Gràfics en una Figura:

    # Configurar l'àrea de gràfics per tenir 2 files i 1 columna
par(mfrow=c(2,1))

# Primer gràfic
plot(x, y, main="Gràfic 1")

# Segon gràfic
plot(y, x, main="Gràfic 2")

    Explicació:

    • par(mfrow=c(2,1)): Divideix l'àrea de gràfics en una quadrícula de 2x1.

  • Superposar Gràfics:

    # Crear el primer gràfic
plot(x, y, type="l", col="blue", main="Superposar Gràfics", xlab="Eix X", ylab="Eix Y")

# Afegir un segon gràfic a sobre
lines(x, y/2, col="red")

    Explicació:

    • lines(): Afegeix un gràfic de línies a un gràfic existent.

Guardar Gràfics

Podeu guardar els vostres gràfics en diversos formats de fitxer per utilitzar-los en informes o presentacions.

  • Guardar en un Fitxer PNG:

    # Obrir un dispositiu PNG
png("plot.png")

# Crear un gràfic
plot(x, y, main="Gràfic Guardat", xlab="Eix X", ylab="Eix Y")

# Tancar el dispositiu
dev.off()

    Explicació:

    • png("plot.png"): Obre un dispositiu PNG per guardar el gràfic.
    • dev.off(): Tanca el dispositiu i escriu el fitxer.

  • Guardar en un Fitxer PDF:

    # Obrir un dispositiu PDF
pdf("plot.pdf")

# Crear un gràfic
plot(x, y, main="Gràfic Guardat", xlab="Eix X", ylab="Eix Y")

# Tancar el dispositiu
dev.off()

    Explicació:

    • pdf("plot.pdf"): Obre un dispositiu PDF per guardar el gràfic.

Conclusió

Els gràfics base R proporcionen un conjunt robust d'eines per crear una àmplia gamma de visualitzacions. Des de gràfics bàsics fins a personalitzacions avançades i opcions de guardat, entendre aquestes eines és essencial per a una visualització de dades efectiva en R. Dominant aquestes tècniques, podeu crear gràfics informatius i visualment atractius que millorin la vostra anàlisi de dades i presentació.
