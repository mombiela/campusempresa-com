Introducció

En aquesta secció, explorarem les operacions bàsiques en estructures de dades en R. Entendre aquestes operacions és crucial per a la manipulació i anàlisi de dades. Cobrirem vectors, llistes, matrius, data frames i factors, proporcionant exemples i exercicis per consolidar la teva comprensió.

Vectors

Els vectors són el tipus més simple d'estructura de dades en R. Són seqüències d'elements de dades del mateix tipus bàsic.

Creació de vectors

  • Utilitza la funció c() per crear un vector.
# Creació d'un vector numèric
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Creació d'un vector de caràcters
char_vector <- c("a", "b", "c")
print(char_vector)

Operacions bàsiques

  • Accedeix als elements utilitzant índexs.
  • Realitza operacions aritmètiques.
# Accés als elements
first_element <- numeric_vector[1]
print(first_element)

# Operacions aritmètiques
sum_vector <- numeric_vector + 2
print(sum_vector)

Llistes

Les llistes poden contenir elements de diferents tipus, incloent altres llistes.

Creació de llistes

  • Utilitza la funció list() per crear una llista.
# Creació d'una llista
my_list <- list(name = "John", age = 25, scores = c(90, 85, 88))
print(my_list)

Operacions bàsiques

  • Accedeix als elements utilitzant $ o [[ ]].
# Accés als elements
name <- my_list$name
print(name)

# Accés als elements niats
first_score <- my_list$scores[1]
print(first_score)

Matrius

Les matrius són estructures de dades bidimensionals i homogènies.

Creació de matrius

  • Utilitza la funció matrix() per crear una matriu.
# Creació d'una matriu
my_matrix <- matrix(1:9, nrow = 3, ncol = 3)
print(my_matrix)

Operacions bàsiques

  • Accedeix als elements utilitzant índexs de fila i columna.
  • Realitza operacions amb matrius.
# Accés als elements
element <- my_matrix[2, 3]
print(element)

# Multiplicació de matrius
result_matrix <- my_matrix %*% my_matrix
print(result_matrix)

Data Frames

Els data frames són estructures de dades bidimensionals i heterogènies.

Creació de Data Frames

  • Utilitza la funció data.frame() per crear un data frame.
# Creació d'un data frame
my_data_frame <- data.frame(
  Name = c("John", "Jane", "Doe"),
  Age = c(25, 30, 22),
  Score = c(90, 85, 88)
)
print(my_data_frame)

Operacions bàsiques

  • Accedeix als elements utilitzant $ o índexs.
  • Afegeix noves columnes.
# Accés als elements
age <- my_data_frame$Age
print(age)

# Afegeix una nova columna
my_data_frame$Gender <- c("M", "F", "M")
print(my_data_frame)

Factors

Els factors s'utilitzen per gestionar dades categòriques.

Creació de factors

  • Utilitza la funció factor() per crear un factor.
# Creació d'un factor
gender <- factor(c("Male", "Female", "Female", "Male"))
print(gender)

Operacions bàsiques

  • Accedeix als nivells i canvia els nivells.
# Accés als nivells
levels(gender)

# Canvi dels nivells
levels(gender) <- c("M", "F")
print(gender)

Resum

En aquesta secció, hem cobert les operacions bàsiques en diverses estructures de dades en R, incloent vectors, llistes, matrius, data frames i factors. Aquestes operacions formen la base per a tasques més avançades de manipulació i anàlisi de dades. Practica aquestes operacions amb diferents conjunts de dades per esdevenir proficient en la gestió d'estructures de dades en R.
