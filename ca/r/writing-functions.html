Introducció a les Funcions

Les funcions són un bloc fonamental en la programació en R. Et permeten encapsular codi en blocs reutilitzables, fent el teu codi més modular, llegible i fàcil de depurar.

Conceptes Clau

  • Definició de Funció: Com crear una funció.
  • Arguments de Funció: Paràmetres que les funcions poden prendre.
  • Valors de Retorn: El que una funció retorna.
  • Àmbit: Entendre les variables locals i globals dins de les funcions.

Definint una Funció

Per definir una funció en R, utilitzes la paraula clau function. Aquí tens la sintaxi bàsica:

nom_funcio <- function(arg1, arg2, ...) {
  # Cos de la funció
  # Codi a executar
  return(valor)
}

Exemple: Funció Simple

Creem una funció simple que suma dos números:

sumar_numeros <- function(a, b) {
  suma <- a + b
  return(suma)
}

Pots cridar aquesta funció passant dos números:

resultat <- sumar_numeros(5, 3)
print(resultat)  # Sortida: 8

Arguments de Funció

Les funcions poden prendre múltiples arguments. També pots establir valors per defecte per als arguments.

Exemple: Funció amb Arguments per Defecte

saludar <- function(nom = "Món") {
  missatge <- paste("Hola,", nom)
  return(missatge)
}

print(saludar())         # Sortida: "Hola, Món"
print(saludar("Alice"))  # Sortida: "Hola, Alice"

Valors de Retorn

La funció return() s'utilitza per especificar la sortida d'una funció. Si return() no s'utilitza explícitament, R retornarà l'última expressió avaluada.

Exemple: Retorn Implícit

multiplicar_numeros <- function(x, y) {
  producte <- x * y
  producte  # L'última expressió avaluada es retorna
}

print(multiplicar_numeros(4, 5))  # Sortida: 20

Àmbit de les Variables

Les variables definides dins d'una funció són locals a aquesta funció. No són accessibles fora de la funció.

Exemple: Variables Locals vs Globals

x <- 10  # Variable global

exemple_ambit <- function() {
  x <- 5  # Variable local
  return(x)
}

print(exemple_ambit())  # Sortida: 5
print(x)                # Sortida: 10 (la variable global roman inalterada)

Tècniques Avançades de Funcions

Funcions Anònimes

Les funcions anònimes són funcions sense nom, sovint utilitzades per a tasques curtes i d'un sol ús.

resultat <- (function(x, y) { x + y })(3, 4)
print(resultat)  # Sortida: 7

Funcions d'Ordre Superior

Funcions que prenen altres funcions com a arguments o que retornen funcions.

Exemple: Aplicar una Funció a una Llista

aplicar_funcio <- function(f, valors) {
  sapply(valors, f)
}

quadrar <- function(x) {
  return(x^2)
}

numeros <- c(1, 2, 3, 4)
print(aplicar_funcio(quadrar, numeros))  # Sortida: 1 4 9 16

Conclusió

Escriure funcions en R és una manera poderosa de fer el teu codi més modular i reutilitzable. Entenent com definir funcions, utilitzar arguments, valors de retorn i gestionar l'àmbit de les variables, pots escriure codi més eficient i organitzat. A mesura que avancis, podràs explorar conceptes més complexos com les funcions anònimes i les funcions d'ordre superior per millorar encara més les teves habilitats de programació en R.
