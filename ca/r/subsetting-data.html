Subsetting data és una habilitat fonamental en R que et permet extreure elements específics de les estructures de dades com vectors, matrius, data frames i llistes. Aquest tema et guiarà des de tècniques bàsiques fins a avançades de subsetting.

Subsetting Bàsic

Vectors

Els vectors són les estructures de dades més simples en R. Pots fer subsetting de vectors utilitzant índexs, vectors lògics o noms.

  • Utilitzant Índexs: Extreu elements especificant les seves posicions.

    vec <- c(10, 20, 30, 40, 50)
vec[1]  # Extreu el primer element: 10
vec[3:5]  # Extreu elements del 3r al 5è: 30, 40, 50

  • Utilitzant Vectors Lògics: Extreu elements que compleixen una condició.

    vec <- c(10, 20, 30, 40, 50)
vec[vec > 25]  # Extreu elements majors de 25: 30, 40, 50

  • Utilitzant Noms: Extreu elements pels seus noms.

    vec <- c(a = 10, b = 20, c = 30)
vec["b"]  # Extreu l'element anomenat 'b': 20

Matrius

Les matrius són arrays bidimensionals. Pots fer subsetting de matrius utilitzant índexs de fila i columna.

  • Utilitzant Índexs: Extreu elements especificant posicions de fila i columna. 
    mat <- matrix(1:9, nrow = 3)
mat[1, 2]  # Extreu l'element a la 1a fila, 2a columna: 2
mat[1, ]  # Extreu tota la 1a fila: 1, 2, 3
mat[, 2]  # Extreu tota la 2a columna: 2, 5, 8

Data Frames

Els data frames són similars a les matrius però poden contenir diferents tipus de dades. Pots fer subsetting de data frames utilitzant índexs de fila i columna, noms de columna o vectors lògics.

  • Utilitzant Índexs: Extreu elements especificant posicions de fila i columna.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[1, 2]  # Extreu l'element a la 1a fila, 2a columna: 4
df[1, ]  # Extreu tota la 1a fila
df[, 2]  # Extreu tota la 2a columna

  • Utilitzant Noms de Columna: Extreu columnes pels seus noms.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df["b"]  # Extreu la columna anomenada 'b'
df[c("a", "c")]  # Extreu les columnes 'a' i 'c'

  • Utilitzant Vectors Lògics: Extreu files que compleixen una condició.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[df$a > 1, ]  # Extreu files on la columna 'a' és major que 1

Subsetting Intermedi

Llistes

Les llistes poden contenir diferents tipus de dades i estructures. Pots fer subsetting de llistes utilitzant índexs o noms.

  • Utilitzant Índexs: Extreu elements especificant les seves posicions.

    lst <- list(a = 1:3, b = "hello", c = TRUE)
lst[[1]]  # Extreu el primer element: 1, 2, 3
lst[1]  # Extreu el primer element com a llista

  • Utilitzant Noms: Extreu elements pels seus noms.

    lst <- list(a = 1:3, b = "hello", c = TRUE)
lst[["b"]]  # Extreu l'element anomenat 'b': "hello"
lst["b"]  # Extreu l'element anomenat 'b' com a llista

Data Frames Avançats

Combinar múltiples tècniques de subsetting pot ser molt potent.

  • Combinant Condicions: Utilitza múltiples condicions per fer subsetting de data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df[df$a > 2 & df$b < 9, ]  # Extreu files on 'a' > 2 i 'b' < 9

  • Utilitzant la Funció subset: Una manera més llegible de fer subsetting de data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
subset(df, a > 2 & b < 9)  # Extreu files on 'a' > 2 i 'b' < 9

Subsetting Avançat

Utilitzant `dplyr` per a Data Frames

El paquet dplyr proporciona una manera més llegible i eficient de fer subsetting de data frames.

  • Utilitzant filter: Extreu files que compleixen una condició.

    library(dplyr)
df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2 & b < 9)  # Extreu files on 'a' > 2 i 'b' < 9

  • Utilitzant select: Extreu columnes específiques.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% select(a, c)  # Extreu les columnes 'a' i 'c'

  • Combinant filter i select: Combina múltiples operacions.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2) %>% select(a, c)  # Extreu files on 'a' > 2 i columnes 'a' i 'c'

Utilitzant `data.table` per a Grans Conjunts de Dades

El paquet data.table està optimitzat per a grans conjunts de dades i proporciona subsetting eficient.

  • Subsetting Bàsic: Extreu files i columnes.

    library(data.table)
dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2, .(a, c)]  # Extreu files on 'a' > 2 i columnes 'a' i 'c'

  • Subsetting Avançat: Utilitza condicions i operacions complexes.

    dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2 & b < 9, .(sum_a = sum(a), mean_c = mean(c))]  # Extreu files on 'a' > 2 i 'b' < 9, després calcula la suma de 'a' i la mitjana de 'c'

Conclusió

Fer subsetting de dades és una habilitat crucial en R que et permet manipular i analitzar dades de manera eficient. Començant per tècniques bàsiques amb vectors i matrius, pots avançar cap a operacions més complexes amb data frames i llistes. Utilitzar paquets com dplyr i data.table pot millorar encara més la teva capacitat per gestionar grans i complexos conjunts de dades. Practica aquestes tècniques per convertir-te en un expert en manipulació de dades en R.
