Introducció a la Prova d'Hipòtesis

La prova d'hipòtesis és un mètode estadístic utilitzat per prendre decisions sobre les propietats d'una població basada en una mostra. Implica fer una suposició (la hipòtesi) i després determinar si les dades recolzen o refuten aquesta suposició.

Conceptes Clau

  • Hipòtesi Nul·la (H0): La suposició per defecte que no hi ha efecte o diferència.
  • Hipòtesi Alternativa (H1 o Ha): La suposició que hi ha un efecte o una diferència.
  • p-valor: La probabilitat d'obtenir resultats de prova almenys tan extrems com els resultats observats, assumint que la hipòtesi nul·la és certa.
  • Nivell de Significació (α): Un llindar establert pel investigador (comunament 0.05) per decidir si rebutjar la hipòtesi nul·la.
  • Error de Tipus I: Rebutjar incorrectament la hipòtesi nul·la quan és certa.
  • Error de Tipus II: No rebutjar la hipòtesi nul·la quan és falsa.

Realització de la Prova d'Hipòtesis en R

Prova t d'una Mostra

Una prova t d'una mostra s'utilitza per determinar si la mitjana d'una sola mostra és diferent d'una mitjana coneguda.

Exemple

Suposem que tenim una mostra d'alçades de 10 individus, i volem provar si l'alçada mitjana és diferent de 170 cm.

# Dades de la mostra
heights <- c(172, 168, 175, 170, 169, 174, 171, 173, 167, 170)

# Realitzar la prova t d'una mostra
t_test_result <- t.test(heights, mu = 170)

# Imprimir el resultat
print(t_test_result)

Explicació:

  • heights: Un vector que conté les dades de la mostra.
  • t.test(): Funció per realitzar la prova t.
  • mu: El valor mitjà contra el qual es prova (170 cm en aquest cas).

Prova t de Dues Mostres

Una prova t de dues mostres compara les mitjanes de dos grups independents.

Exemple

Suposem que tenim dues mostres d'alçades de dos grups diferents, i volem provar si les seves mitjanes són diferents.

# Dades de la mostra
group1 <- c(172, 168, 175, 170, 169, 174, 171, 173, 167, 170)
group2 <- c(160, 162, 165, 163, 161, 164, 166, 167, 159, 162)

# Realitzar la prova t de dues mostres
t_test_result <- t.test(group1, group2)

# Imprimir el resultat
print(t_test_result)

Explicació:

  • group1 i group2: Vectors que contenen les dades de la mostra per a cada grup.
  • t.test(): Funció per realitzar la prova t.

Prova t Aparellada

Una prova t aparellada compara les mitjanes del mateix grup en diferents moments o sota diferents condicions.

Exemple

Suposem que tenim els pesos d'individus abans i després d'un programa de dieta, i volem provar si hi ha una pèrdua de pes significativa.

# Dades de la mostra
before <- c(80, 82, 78, 85, 90, 88, 84, 86, 87, 89)
after <- c(78, 80, 76, 83, 88, 86, 82, 84, 85, 87)

# Realitzar la prova t aparellada
t_test_result <- t.test(before, after, paired = TRUE)

# Imprimir el resultat
print(t_test_result)

Explicació:

  • before i after: Vectors que contenen les dades de la mostra abans i després del programa de dieta.
  • paired = TRUE: Indica que la prova és aparellada.

Prova de Chi-Quadrat

Una prova de chi-quadrat s'utilitza per determinar si hi ha una associació significativa entre dues variables categòriques.

Exemple

Suposem que tenim dades sobre la preferència per dos tipus de productes entre homes i dones.

# Dades de la mostra
data <- matrix(c(50, 30, 20, 40), nrow = 2, byrow = TRUE)
colnames(data) <- c("Producte A", "Producte B")
rownames(data) <- c("Home", "Dona")

# Realitzar la prova de chi-quadrat
chi_square_result <- chisq.test(data)

# Imprimir el resultat
print(chi_square_result)

Explicació:

  • data: Una matriu que conté els recomptes de freqüència.
  • chisq.test(): Funció per realitzar la prova de chi-quadrat.

Conclusió

La prova d'hipòtesis és un aspecte fonamental de l'anàlisi estadística, permetent als investigadors fer inferències sobre poblacions basades en dades de mostra. R proporciona una varietat de funcions per realitzar diferents tipus de proves d'hipòtesis, fent-lo una eina poderosa per a l'anàlisi estadística. Entenent i aplicant aquestes proves, pots prendre decisions basades en dades i validar les teves suposicions amb confiança.
