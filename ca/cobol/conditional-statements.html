Introducció

Les declaracions condicionals són fonamentals en qualsevol llenguatge de programació, incloent COBOL. Permeten que el programa prengui decisions i executi diferents blocs de codi basats en certes condicions. En aquesta secció, cobrirem els conceptes bàsics de les declaracions condicionals en COBOL, progressant cap a conceptes més avançats.

Declaracions condicionals bàsiques

Declaració IF

La declaració IF és la forma més bàsica de declaració condicional en COBOL. Permet que el programa executi un bloc de codi només si una condició especificada és certa.

Sintaxi

IF condition
    statement-1
    statement-2
    ...
END-IF.

Exemple

IF AGE > 18
    DISPLAY 'You are an adult.'
END-IF.

En aquest exemple, el missatge "You are an adult." es mostrarà només si el valor de AGE és superior a 18.

Declaració IF-ELSE

La declaració IF-ELSE amplia la declaració IF proporcionant un bloc de codi alternatiu per executar si la condició és falsa.

Sintaxi

IF condition
    statement-1
ELSE
    statement-2
END-IF.

Exemple

IF AGE > 18
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

En aquest exemple, el programa mostrarà "You are an adult." si AGE és superior a 18, en cas contrari, mostrarà "You are not an adult."

Declaracions IF niades

Es poden niar declaracions IF dins d'altres per comprovar múltiples condicions.

Exemple

IF AGE > 18
    IF AGE < 65
        DISPLAY 'You are an adult.'
    ELSE
        DISPLAY 'You are a senior.'
    END-IF
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

En aquest exemple, el programa comprova si AGE és superior a 18 i inferior a 65 per determinar si la persona és un adult o un sènior.

Declaracions condicionals avançades

Declaració EVALUATE

La declaració EVALUATE en COBOL és similar a la declaració SWITCH en altres llenguatges de programació. Permet que es puguin avaluar múltiples condicions d'una manera més llegible i organitzada.

Sintaxi

EVALUATE TRUE
    WHEN condition-1
        statement-1
    WHEN condition-2
        statement-2
    ...
    WHEN OTHER
        statement-n
END-EVALUATE.

Exemple

EVALUATE TRUE
    WHEN AGE < 13
        DISPLAY 'You are a child.'
    WHEN AGE < 18
        DISPLAY 'You are a teenager.'
    WHEN AGE < 65
        DISPLAY 'You are an adult.'
    WHEN OTHER
        DISPLAY 'You are a senior.'
END-EVALUATE.

En aquest exemple, el programa avalua el valor de AGE i mostra el missatge apropiat basat en el rang en què es troba AGE.

Condicions complexes

Es poden utilitzar operadors lògics com AND i OR per formar condicions complexes en declaracions IF i EVALUATE.

Exemple

IF AGE > 18 AND AGE < 65
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are either a minor or a senior.'
END-IF.

En aquest exemple, el programa comprova si AGE està entre 18 i 65 per determinar si la persona és un adult.

Conclusió

Les declaracions condicionals en COBOL són essencials per controlar el flux d'un programa basat en diferents condicions. Des de les declaracions IF bàsiques fins a les declaracions EVALUATE més avançades, entendre aquests constructes et permetrà escriure programes COBOL més dinàmics i flexibles. Practica escrivint diverses declaracions condicionals per convertir-te en un expert en fer que els teus programes COBOL responguin a diferents escenaris.
