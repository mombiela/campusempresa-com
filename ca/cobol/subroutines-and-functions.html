Subrutines i Funcions en COBOL
Taules i Arrays ►

Introducció

Les subrutines i funcions són components essencials en la programació COBOL, permetent un codi modular, reutilitzable i organitzat. Aquesta secció cobrirà els conceptes bàsics de les subrutines i funcions, la seva sintaxi, ús i millors pràctiques des de nivells bàsics fins a avançats.

Conceptes Clau

  • Subrutines: També conegudes com a paràgrafs o seccions, són blocs de codi que realitzen tasques específiques i es poden cridar des de diferents parts del programa.
  • Funcions: Operacions predefinides o definides per l'usuari que retornen un valor i es poden utilitzar dins d'expressions.

Subrutines en COBOL

Estructura Bàsica

Les subrutines en COBOL es defineixen a la PROCEDURE DIVISION. Es poden organitzar en paràgrafs o seccions.

Paràgrafs

Un paràgraf és un bloc de codi amb nom que es pot executar utilitzant la instrucció PERFORM.

IDENTIFICATION DIVISION.
PROGRAM-ID. SubroutineExample.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 NUM1 PIC 9(2) VALUE 10.
01 NUM2 PIC 9(2) VALUE 20.
01 RESULT PIC 9(3).

PROCEDURE DIVISION.
    PERFORM ADD-NUMBERS
    DISPLAY "El resultat és: " RESULT
    STOP RUN.

ADD-NUMBERS.
    ADD NUM1 TO NUM2 GIVING RESULT.

Seccions

Les seccions són blocs més grans que poden contenir múltiples paràgrafs.

IDENTIFICATION DIVISION.
PROGRAM-ID. SectionExample.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 NUM1 PIC 9(2) VALUE 10.
01 NUM2 PIC 9(2) VALUE 20.
01 RESULT PIC 9(3).

PROCEDURE DIVISION.
    PERFORM CALCULATION-SECTION
    DISPLAY "El resultat és: " RESULT
    STOP RUN.

CALCULATION-SECTION.
ADD-NUMBERS.
    ADD NUM1 TO NUM2 GIVING RESULT.

Cridar Subrutines

  • PERFORM: Executa un paràgraf o secció.
  • THRU: Executa un rang de paràgrafs.
PERFORM ADD-NUMBERS
PERFORM CALCULATION-SECTION THRU END-CALCULATION

Funcions en COBOL

Funcions Intrínseques

COBOL proporciona un conjunt de funcions integrades per a operacions comunes.

Exemple: Funció NUMVAL

Converteix una cadena numèrica a un valor numèric.

IDENTIFICATION DIVISION.
PROGRAM-ID. FunctionExample.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 NUM-STRING PIC X(5) VALUE '12345'.
01 NUM-VALUE PIC 9(5).

PROCEDURE DIVISION.
    COMPUTE NUM-VALUE = FUNCTION NUMVAL(NUM-STRING)
    DISPLAY "El valor numèric és: " NUM-VALUE
    STOP RUN.

Funcions Definides per l'Usuari

Les funcions definides per l'usuari es poden crear utilitzant les instruccions FUNCTION-ID i END FUNCTION.

IDENTIFICATION DIVISION.
FUNCTION-ID. MyFunction.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 INPUT-NUMBER PIC 9(2).
01 RESULT PIC 9(3).

PROCEDURE DIVISION.
    ENTRY "MyFunction" USING INPUT-NUMBER
    ADD INPUT-NUMBER TO INPUT-NUMBER GIVING RESULT
    EXIT FUNCTION RESULT.
END FUNCTION MyFunction.

Cridar Funcions Definides per l'Usuari

Les funcions definides per l'usuari es poden cridar utilitzant la paraula clau FUNCTION.

IDENTIFICATION DIVISION.
PROGRAM-ID. CallFunctionExample.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 INPUT-NUMBER PIC 9(2) VALUE 10.
01 RESULT PIC 9(3).

PROCEDURE DIVISION.
    COMPUTE RESULT = FUNCTION MyFunction(INPUT-NUMBER)
    DISPLAY "El resultat és: " RESULT
    STOP RUN.

Millors Pràctiques

  • Modularitat: Divideix tasques complexes en subrutines o funcions més petites.
  • Reutilització: Escriu subrutines i funcions que es puguin reutilitzar en diferents programes.
  • Claredat: Utilitza noms significatius per a les subrutines i funcions per millorar la llegibilitat del codi.
  • Documentació: Comenta el teu codi per explicar el propòsit i ús de les subrutines i funcions.

Resum

Les subrutines i funcions són eines poderoses en COBOL que ajuden a crear codi organitzat, modular i reutilitzable. Entendre la seva estructura, ús i millors pràctiques és crucial per a una programació COBOL eficient. En dominar aquests conceptes, pots escriure programes COBOL més mantenibles i escalables.
