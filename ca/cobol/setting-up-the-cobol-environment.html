Configurar l'entorn COBOL és el primer pas per escriure i executar programes COBOL. Aquesta secció et guiarà a través del procés de configuració d'un entorn de desenvolupament COBOL a la teva màquina, cobrint tant sistemes Windows com Unix/Linux.

Prerequisits

Abans de configurar l'entorn COBOL, assegura't de tenir els següents prerequisits:

  • Un ordinador amb sistema operatiu Windows, Unix o Linux.
  • Coneixements bàsics de la interfície de línia de comandes (CLI).

Instal·lant un Compilador COBOL

Un compilador COBOL és essencial per convertir el codi COBOL en programes executables. Hi ha diversos compiladors COBOL disponibles, tant gratuïts com comercials. Per a aquest curs, utilitzarem el compilador de codi obert GnuCOBOL.

Instal·lant GnuCOBOL a Windows

  1. Descarregar GnuCOBOL:

    • Visita la pàgina de GnuCOBOL a SourceForge i descarrega l'última versió de GnuCOBOL per a Windows.

  2. Instal·lar GnuCOBOL:

    • Executa l'instal·lador descarregat i segueix les instruccions en pantalla per completar la instal·lació.

  3. Verificar la Instal·lació:

    • Obre el Símbol del sistema i escriu la següent comanda per verificar la instal·lació: 
      cobc -v
    • Hauries de veure la informació de la versió de GnuCOBOL si la instal·lació ha estat exitosa.

Instal·lant GnuCOBOL a Unix/Linux

  1. Actualitzar la Llista de Paquets:

    • Obre una terminal i actualitza la teva llista de paquets: 
      sudo apt-get update

  2. Instal·lar GnuCOBOL:

    • Instal·la GnuCOBOL utilitzant el gestor de paquets: 
      sudo apt-get install open-cobol

  3. Verificar la Instal·lació:

    • Verifica la instal·lació comprovant la versió: 
      cobc -v
    • Hauries de veure la informació de la versió de GnuCOBOL.

Configurant l'Entorn de Desenvolupament

Un cop el compilador està instal·lat, necessites configurar el teu entorn de desenvolupament. Això inclou triar un editor de text o un Entorn de Desenvolupament Integrat (IDE) i configurar-lo per al desenvolupament COBOL.

Triant un Editor de Text o IDE

  • Editors de Text:

    • Notepad++ (Windows): Un editor de text lleuger i versàtil.
    • Vim (Unix/Linux): Un editor de text potent disponible a la majoria de sistemes Unix/Linux.
    • Sublime Text: Un editor de text multiplataforma amb suport per a diversos llenguatges de programació.

  • IDEs:

    • Eclipse amb Plugin COBOL: Un IDE popular que suporta el desenvolupament COBOL a través de plugins.
    • Visual Studio Code: Un editor de codi versàtil amb extensions per a COBOL.

Configurant l'Editor de Text o IDE

Exemple: Configurant Visual Studio Code per a COBOL

  1. Instal·lar Visual Studio Code:

    • Descarrega i instal·la Visual Studio Code des del lloc web oficial.

  2. Instal·lar l'Extensió COBOL:

    • Obre Visual Studio Code i ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat.
    • Cerca "COBOL" i instal·la l'extensió COBOL de bitlang.

  3. Configurar l'Extensió COBOL:

    • Obre la configuració (File > Preferences > Settings) i cerca "COBOL".
    • Configura els paràmetres de l'extensió COBOL segons sigui necessari, com especificar el camí al compilador GnuCOBOL.

Escriure i Executar el Teu Primer Programa COBOL

Amb l'entorn configurat, ara pots escriure i executar el teu primer programa COBOL.

Exemple de Programa COBOL

Crea un nou fitxer anomenat hello.cob i afegeix el següent codi:

       IDENTIFICATION DIVISION.
       PROGRAM-ID. HelloWorld.
       PROCEDURE DIVISION.
       DISPLAY 'Hello, World!'.
       STOP RUN.

Compilant i Executant el Programa

  1. Compilar el Programa:

    • Obre una terminal o el Símbol del sistema i navega al directori que conté hello.cob.
    • Compila el programa utilitzant la següent comanda: 
      cobc -x hello.cob

  2. Executar el Programa:

    • Executa el programa compilat: 
      ./hello
    • Hauries de veure la sortida: 
      Hello, World!

Conclusió

Configurar l'entorn COBOL implica instal·lar un compilador COBOL, triar i configurar un editor de text o IDE, i escriure i executar el teu primer programa COBOL. Amb l'entorn llest, ara estàs preparat per aprofundir en la programació COBOL. A les següents seccions, explorarem la sintaxi COBOL, els tipus de dades i conceptes de programació més avançats.
