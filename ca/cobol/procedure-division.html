La Divisió de Procediments en COBOL és on es realitza el processament real de les dades. Aquesta és la secció del programa on escrius les instruccions que diuen a l'ordinador què fer amb les dades definides a la Divisió de Dades. Entendre la Divisió de Procediments és crucial per escriure programes COBOL efectius.

Conceptes Clau

  • Paràgrafs i Seccions: La Divisió de Procediments està organitzada en paràgrafs i seccions. Una secció és una col·lecció de paràgrafs, i un paràgraf és una col·lecció d'instruccions.
  • Instruccions: Aquestes són les instruccions individuals que realitzen operacions com moure dades, fer càlculs i controlar el flux del programa.
  • Flux de Control: COBOL proporciona diversos constructes per controlar el flux del programa, incloent instruccions IF, bucles PERFORM i instruccions GO TO.

Estructura de la Divisió de Procediments

La Divisió de Procediments comença amb l'encapçalament PROCEDURE DIVISION i és seguida per seccions, paràgrafs i instruccions.

Estructura Bàsica

PROCEDURE DIVISION.
    SECTION-1.
    PARAGRAPH-1.
        DISPLAY 'Hello, World!'.
        STOP RUN.

Seccions i Paràgrafs

  • Seccions: Agrupen paràgrafs relacionats.
  • Paràgrafs: Contenen les instruccions executables.
PROCEDURE DIVISION.
    INITIALIZE-SECTION.
    INITIALIZE-PARAGRAPH.
        DISPLAY 'Initializing Program'.
    
    PROCESS-SECTION.
    PROCESS-PARAGRAPH.
        DISPLAY 'Processing Data'.
    
    TERMINATE-SECTION.
    TERMINATE-PARAGRAPH.
        DISPLAY 'Terminating Program'.
        STOP RUN.

Instruccions Comunes

Instrucció DISPLAY

La instrucció DISPLAY s'utilitza per mostrar text a la consola.

PROCEDURE DIVISION.
    DISPLAY 'Hello, World!'.
    STOP RUN.

Instrucció MOVE

La instrucció MOVE assigna valors a variables.

DATA DIVISION.
WORKING-STORAGE SECTION.
01  WS-NAME PIC X(20).

PROCEDURE DIVISION.
    MOVE 'John Doe' TO WS-NAME.
    DISPLAY WS-NAME.
    STOP RUN.

Instrucció IF

La instrucció IF s'utilitza per al processament condicional.

DATA DIVISION.
WORKING-STORAGE SECTION.
01  WS-AGE PIC 99.

PROCEDURE DIVISION.
    MOVE 25 TO WS-AGE.
    IF WS-AGE > 18
        DISPLAY 'Adult'
    ELSE
        DISPLAY 'Minor'.
    STOP RUN.

Instrucció PERFORM

La instrucció PERFORM s'utilitza per executar un paràgraf o secció.

PROCEDURE DIVISION.
    PERFORM INITIALIZE-PARAGRAPH.
    PERFORM PROCESS-PARAGRAPH.
    PERFORM TERMINATE-PARAGRAPH.
    STOP RUN.

INITIALIZE-PARAGRAPH.
    DISPLAY 'Initializing Program'.

PROCESS-PARAGRAPH.
    DISPLAY 'Processing Data'.

TERMINATE-PARAGRAPH.
    DISPLAY 'Terminating Program'.

Instrucció GO TO

La instrucció GO TO transfereix el control a una altra part del programa.

PROCEDURE DIVISION.
    DISPLAY 'Start'.
    GO TO END-PARAGRAPH.
    DISPLAY 'This will not be displayed'.
    
END-PARAGRAPH.
    DISPLAY 'End'.
    STOP RUN.

Constructes de Flux de Control

Instruccions IF Imbricades

DATA DIVISION.
WORKING-STORAGE SECTION.
01  WS-SCORE PIC 99.

PROCEDURE DIVISION.
    MOVE 85 TO WS-SCORE.
    IF WS-SCORE >= 90
        DISPLAY 'Grade: A'
    ELSE
        IF WS-SCORE >= 80
            DISPLAY 'Grade: B'
        ELSE
            DISPLAY 'Grade: C'.
    STOP RUN.

Bucle PERFORM UNTIL

DATA DIVISION.
WORKING-STORAGE SECTION.
01  WS-COUNTER PIC 99 VALUE 1.

PROCEDURE DIVISION.
    PERFORM UNTIL WS-COUNTER > 5
        DISPLAY 'Counter: ' WS-COUNTER
        ADD 1 TO WS-COUNTER
    END-PERFORM.
    STOP RUN.

Conclusió

La Divisió de Procediments és el cor d'un programa COBOL, on es realitza la lògica real i el processament de dades. Entenent com estructurar seccions, paràgrafs i instruccions, i dominant els constructes de flux de control com les instruccions IF i els bucles PERFORM, pots escriure programes COBOL eficients i efectius. Practica escrivint diferents tipus d'instruccions i estructures de control per convertir-te en un programador COBOL proficient.
