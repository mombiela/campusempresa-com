El processament de fitxers seqüencials és un concepte fonamental en la programació COBOL, essencial per gestionar dades emmagatzemades en fitxers. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades, assegurant que entenguis com llegir i escriure en fitxers seqüencials de manera efectiva.

Introducció als Fitxers Seqüencials

Els fitxers seqüencials són fitxers on les dades s'emmagatzemen en un ordre específic, i els registres s'accedeixen seqüencialment, un darrere l'altre. Aquesta és l'organització de fitxers més comuna en COBOL.

Conceptes Clau

  • Control de Fitxers: Especifica l'organització del fitxer i el mode d'accés.
  • Descripció de Fitxers (FD): Descriu l'estructura dels registres del fitxer.
  • Sentència OPEN: Obre el fitxer per llegir, escriure o ambdues coses.
  • Sentència READ: Llegeix registres del fitxer.
  • Sentència WRITE: Escriu registres al fitxer.
  • Sentència CLOSE: Tanca el fitxer després del processament.

Control de Fitxers i Descripció de Fitxers

Control de Fitxers

Les clàusules SELECT i ASSIGN en el paràgraf FILE-CONTROL especifiquen l'organització del fitxer i el nom extern del fitxer.

IDENTIFICATION DIVISION.
PROGRAM-ID. SeqFileExample.

ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
    SELECT InputFile ASSIGN TO 'input.dat'
        ORGANIZATION IS SEQUENTIAL.
    SELECT OutputFile ASSIGN TO 'output.dat'
        ORGANIZATION IS SEQUENTIAL.

Descripció de Fitxers

L'entrada FD en la DATA DIVISION descriu l'estructura dels registres del fitxer.

DATA DIVISION.
FILE SECTION.
FD  InputFile.
01  InputRecord.
    05  CustomerID    PIC X(10).
    05  CustomerName  PIC X(30).
    05  CustomerBalance PIC 9(7)V99.

FD  OutputFile.
01  OutputRecord.
    05  CustomerID    PIC X(10).
    05  CustomerName  PIC X(30).
    05  CustomerBalance PIC 9(7)V99.

Obrir i Tancar Fitxers

Sentència OPEN

La sentència OPEN s'utilitza per obrir fitxers per al processament.

PROCEDURE DIVISION.
    OPEN INPUT InputFile.
    OPEN OUTPUT OutputFile.

Sentència CLOSE

La sentència CLOSE s'utilitza per tancar fitxers després del processament.

    CLOSE InputFile.
    CLOSE OutputFile.

Llegir i Escriure Registres

Sentència READ

La sentència READ llegeix el següent registre del fitxer.

    READ InputFile INTO InputRecord
        AT END
            MOVE 'YES' TO EndOfFile
    END-READ.

Sentència WRITE

La sentència WRITE escriu un registre al fitxer.

    MOVE InputRecord TO OutputRecord.
    WRITE OutputRecord.

Programa Exemple

Aquí tens un exemple complet que llegeix registres d'un fitxer d'entrada i els escriu en un fitxer de sortida.

IDENTIFICATION DIVISION.
PROGRAM-ID. SeqFileExample.

ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
    SELECT InputFile ASSIGN TO 'input.dat'
        ORGANIZATION IS SEQUENTIAL.
    SELECT OutputFile ASSIGN TO 'output.dat'
        ORGANIZATION IS SEQUENTIAL.

DATA DIVISION.
FILE SECTION.
FD  InputFile.
01  InputRecord.
    05  CustomerID    PIC X(10).
    05  CustomerName  PIC X(30).
    05  CustomerBalance PIC 9(7)V99.

FD  OutputFile.
01  OutputRecord.
    05  CustomerID    PIC X(10).
    05  CustomerName  PIC X(30).
    05  CustomerBalance PIC 9(7)V99.

WORKING-STORAGE SECTION.
01  EndOfFile        PIC X VALUE 'NO'.

PROCEDURE DIVISION.
    OPEN INPUT InputFile.
    OPEN OUTPUT OutputFile.

    PERFORM UNTIL EndOfFile = 'YES'
        READ InputFile INTO InputRecord
            AT END
                MOVE 'YES' TO EndOfFile
        END-READ

        IF EndOfFile = 'NO'
            MOVE InputRecord TO OutputRecord
            WRITE OutputRecord
        END-IF
    END-PERFORM.

    CLOSE InputFile.
    CLOSE OutputFile.

    STOP RUN.

Tècniques Avançades

Gestió de l'Estat del Fitxer

Utilitzant la clàusula FILE STATUS per gestionar errors durant les operacions amb fitxers.

FILE-CONTROL.
    SELECT InputFile ASSIGN TO 'input.dat'
        ORGANIZATION IS SEQUENTIAL
        FILE STATUS IS WS-FileStatus.

WORKING-STORAGE SECTION.
01  WS-FileStatus    PIC XX.

Comprovar l'Estat del Fitxer

Comprovant l'estat del fitxer després de cada operació per assegurar una execució exitosa.

    READ InputFile INTO InputRecord
        AT END
            MOVE 'YES' TO EndOfFile
    END-READ.

    IF WS-FileStatus NOT = '00'
        DISPLAY 'Error reading file: ' WS-FileStatus
        STOP RUN
    END-IF.

Conclusió

El processament de fitxers seqüencials en COBOL implica entendre com definir, obrir, llegir, escriure i tancar fitxers. En dominar aquestes tècniques bàsiques i avançades, podràs gestionar eficientment les dades emmagatzemades en fitxers seqüencials, un requisit comú en moltes aplicacions COBOL. Practica amb diferents estructures de fitxers i mecanismes de gestió d'errors per convertir-te en un expert en el processament de fitxers seqüencials.
