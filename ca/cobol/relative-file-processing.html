Introducció

El processament de fitxers relatius en COBOL implica gestionar fitxers on els registres s'accedeixen basant-se en la seva posició relativa dins del fitxer. Aquest mètode és útil per a aplicacions que requereixen un accés ràpid als registres sense llegir seqüencialment tot el fitxer.

Conceptes Clau

  • Fitxer Relatiu: Un fitxer on els registres s'identifiquen pel seu número de registre relatiu (RRN).
  • RRN (Número de Registre Relatiu): Un número únic que representa la posició d'un registre dins d'un fitxer relatiu.
  • Modes d'Accés: Mètodes per accedir als registres en un fitxer relatiu, incloent-hi l'accés seqüencial i aleatori.

Definició d'un Fitxer Relatiu

Per treballar amb fitxers relatius en COBOL, cal definir el fitxer a la SECCIÓ DE FITXERS de la DIVISIÓ DE DADES i especificar la seva organització a la DIVISIÓ D'ENTORN.

Exemple

IDENTIFICATION DIVISION.
PROGRAM-ID. RelativeFileExample.

ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
    SELECT REL-FILE ASSIGN TO 'relative.dat'
    ORGANIZATION IS RELATIVE
    ACCESS MODE IS RANDOM
    RELATIVE KEY IS REL-KEY.

DATA DIVISION.
FILE SECTION.
FD  REL-FILE.
01  REL-RECORD.
    05  REL-KEY        PIC 9(4).
    05  REL-DATA       PIC X(20).

WORKING-STORAGE SECTION.
01  WS-REL-KEY         PIC 9(4).
01  WS-REL-DATA        PIC X(20).

PROCEDURE DIVISION.

Obrir i Tancar Fitxers Relatius

Els fitxers relatius s'han d'obrir abans que es puguin llegir o escriure, i tancar després que les operacions estiguin completes.

Exemple

OPEN INPUT REL-FILE.
CLOSE REL-FILE.

Llegir Registres

Per llegir un registre d'un fitxer relatiu, cal establir la clau relativa i utilitzar la instrucció READ.

Exemple

MOVE 1 TO WS-REL-KEY.
READ REL-FILE
    INVALID KEY DISPLAY 'Record not found.'
    NOT INVALID KEY DISPLAY 'Record found: ' REL-DATA.

Escriure Registres

Per escriure un registre en un fitxer relatiu, cal establir la clau relativa i les dades, i després utilitzar la instrucció WRITE.

Exemple

MOVE 2 TO WS-REL-KEY.
MOVE 'Sample Data' TO WS-REL-DATA.
WRITE REL-RECORD INVALID KEY DISPLAY 'Error writing record.'

Actualitzar Registres

Per actualitzar un registre, primer cal llegir-lo, modificar les dades i després reescriure'l.

Exemple

MOVE 1 TO WS-REL-KEY.
READ REL-FILE
    INVALID KEY DISPLAY 'Record not found.'
    NOT INVALID KEY
        MOVE 'Updated Data' TO REL-DATA
        REWRITE REL-RECORD INVALID KEY DISPLAY 'Error updating record.'

Esborrar Registres

Per esborrar un registre, utilitza la instrucció DELETE després d'establir la clau relativa.

Exemple

MOVE 1 TO WS-REL-KEY.
DELETE REL-FILE INVALID KEY DISPLAY 'Error deleting record.'

Resum

El processament de fitxers relatius en COBOL permet un accés eficient als registres basant-se en les seves posicions relatives. Les operacions clau inclouen llegir, escriure, actualitzar i esborrar registres utilitzant el número de registre relatiu. Entendre com definir, obrir, tancar i manipular fitxers relatius és essencial per a una programació efectiva en COBOL.

En dominar aquests conceptes, pots gestionar dades de manera més eficient en aplicacions que requereixen un accés ràpid i directe als registres.
