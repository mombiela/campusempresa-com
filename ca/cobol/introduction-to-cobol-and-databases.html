Visió General

En aquesta secció, introduirem COBOL (Common Business-Oriented Language) i la seva interacció amb les bases de dades. COBOL és un llenguatge de programació d'alt nivell utilitzat principalment en sistemes de negocis, finances i administració per a empreses i governs. Entendre com COBOL interactua amb les bases de dades és crucial per gestionar dades en aplicacions empresarials a gran escala.

Conceptes Clau

  • Conceptes Bàsics de COBOL: Entendre l'estructura i la sintaxi de COBOL.
  • Bases de Dades: Introducció a les bases de dades, centrant-se en les bases de dades relacionals.
  • COBOL i Bases de Dades: Com COBOL interactua amb les bases de dades, incloent SQL incrustat.

Conceptes Bàsics de COBOL

COBOL és conegut per la seva llegibilitat i sintaxi senzilla. Aquí hi ha alguns punts clau:

  • Estructura de Divisions: Els programes COBOL es divideixen en quatre divisions:
    1. Divisió d'Identificació: Metadades del programa.
    2. Divisió d'Entorn: Configuració.
    3. Divisió de Dades: Declaració de variables.
    4. Divisió de Procediments: Lògica de negoci i operacions.

Exemple: Programa COBOL Simple

IDENTIFICATION DIVISION.
PROGRAM-ID. HelloWorld.

PROCEDURE DIVISION.
    DISPLAY 'Hello, World!'.
    STOP RUN.

Explicació:

  • IDENTIFICATION DIVISION: Especifica el nom del programa.
  • PROCEDURE DIVISION: Conté el codi executable.

Introducció a les Bases de Dades

Les bases de dades són col·leccions organitzades de dades. Les bases de dades relacionals utilitzen taules per emmagatzemar dades en files i columnes.

  • Taules: Consisteixen en files (registres) i columnes (camps).
  • SQL (Structured Query Language): Utilitzat per interactuar amb bases de dades relacionals.

Exemple: Consulta SQL Simple

SELECT * FROM Employees WHERE Department = 'Sales';

Explicació:

  • SELECT: Recupera dades de la base de dades.
  • FROM Employees: Especifica la taula.
  • WHERE Department = 'Sales': Filtra els resultats.

COBOL i Bases de Dades

COBOL pot interactuar amb bases de dades utilitzant SQL incrustat, que permet incloure sentències SQL dins del codi COBOL.

Exemple: Programa COBOL amb SQL Incrustat

IDENTIFICATION DIVISION.
PROGRAM-ID. EmployeeReport.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 EMP-ID PIC 9(5).
01 EMP-NAME PIC X(20).
01 EMP-DEPT PIC X(10).

PROCEDURE DIVISION.
    EXEC SQL
        DECLARE C1 CURSOR FOR
        SELECT EMP_ID, EMP_NAME, EMP_DEPT
        FROM EMPLOYEES
        WHERE EMP_DEPT = 'Sales'
    END-EXEC.

    EXEC SQL
        OPEN C1
    END-EXEC.

    PERFORM UNTIL SQLCODE = 100
        EXEC SQL
            FETCH C1 INTO :EMP-ID, :EMP-NAME, :EMP-DEPT
        END-EXEC
        DISPLAY 'ID: ' EMP-ID ' Name: ' EMP-NAME ' Dept: ' EMP-DEPT
    END-PERFORM.

    EXEC SQL
        CLOSE C1
    END-EXEC.

    STOP RUN.

Explicació:

  • EXEC SQL: Marca l'inici d'una sentència SQL incrustada.
  • DECLARE C1 CURSOR: Declara un cursor per a la consulta SQL.
  • OPEN C1: Obre el cursor.
  • FETCH C1 INTO: Recupera dades del cursor a les variables COBOL.
  • PERFORM UNTIL SQLCODE = 100: Bucla fins que no es recuperin més files (SQLCODE 100 indica que no hi ha més dades).
  • CLOSE C1: Tanca el cursor.

Resum

En aquesta secció, hem cobert els conceptes bàsics de COBOL i la seva interacció amb les bases de dades. Hem après sobre l'estructura dels programes COBOL, els fonaments de les bases de dades relacionals i com utilitzar SQL incrustat dins de COBOL per interactuar amb una base de dades. Entendre aquests conceptes és essencial per desenvolupar i mantenir aplicacions a nivell empresarial que gestionen grans volums de dades.
