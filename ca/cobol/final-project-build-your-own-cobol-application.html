Introducció

En aquest projecte final, aplicaràs les habilitats i coneixements que has adquirit al llarg d'aquest curs per construir la teva pròpia aplicació COBOL. Aquest projecte t'ajudarà a consolidar la teva comprensió dels conceptes de programació COBOL i demostrar la teva capacitat per desenvolupar una aplicació funcional des de zero.

Requisits del Projecte

La teva aplicació COBOL hauria de complir els següents requisits:

  • Funcionalitat: L'aplicació hauria de realitzar una tasca significativa, com gestionar una petita base de dades, processar transaccions o generar informes.
  • Interfície d'Usuari: Implementa una interfície d'usuari senzilla que permeti als usuaris interactuar amb l'aplicació.
  • Gestió de Dades: Inclou la gestió d'arxius o operacions de base de dades per emmagatzemar i recuperar dades.
  • Gestió d'Errors: Implementa la gestió d'errors per gestionar situacions inesperades de manera elegant.
  • Documentació: Proporciona una documentació clara per al teu codi, incloent comentaris i un manual d'usuari.

Passos del Projecte

1. Defineix el Propòsit de l'Aplicació

Comença definint el propòsit de la teva aplicació. Considera el següent:

  • Quin problema resol la teva aplicació?
  • Qui és l'usuari objectiu?
  • Quines són les característiques i funcionalitats principals?

2. Dissenya l'Aplicació

Crea un disseny per a la teva aplicació, incloent:

  • Diagrama de Flux o Pseudocodi: Esbossa la lògica i el flux de la teva aplicació.
  • Estructures de Dades: Defineix les estructures de dades que utilitzaràs, com arxius o taules de base de dades.
  • Interfície d'Usuari: Dibuixa la interfície d'usuari i com els usuaris interactuaran amb l'aplicació.

3. Configura l'Entorn de Desenvolupament

Assegura't que el teu entorn de desenvolupament està llest:

  • Instal·la un compilador COBOL (per exemple, GnuCOBOL).
  • Configura qualsevol biblioteca o eina necessària.

4. Implementa l'Aplicació

Comença a codificar la teva aplicació. Aquí tens alguns conceptes clau a tenir en compte:

Gestió d'Arxius

Utilitza les capacitats de gestió d'arxius de COBOL per gestionar l'emmagatzematge de dades. Per exemple:

IDENTIFICATION DIVISION.
PROGRAM-ID. FileHandlingExample.

ENVIRONMENT DIVISION.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
    SELECT StudentFile ASSIGN TO 'students.dat'
    ORGANIZATION IS LINE SEQUENTIAL.

DATA DIVISION.
FILE SECTION.
FD StudentFile.
01 StudentRecord.
   05 StudentID     PIC 9(5).
   05 StudentName   PIC A(20).

WORKING-STORAGE SECTION.
01 WS-StudentID     PIC 9(5).
01 WS-StudentName   PIC A(20).

PROCEDURE DIVISION.
    OPEN OUTPUT StudentFile
    MOVE 12345 TO WS-StudentID
    MOVE 'John Doe' TO WS-StudentName
    WRITE StudentRecord FROM WS-StudentID WS-StudentName
    CLOSE StudentFile
    STOP RUN.

Interfície d'Usuari

Implementa una interfície d'usuari senzilla utilitzant les instruccions ACCEPT i DISPLAY:

DISPLAY "Enter Student ID: "
ACCEPT WS-StudentID
DISPLAY "Enter Student Name: "
ACCEPT WS-StudentName

Gestió d'Errors

Inclou la gestió d'errors per gestionar situacions inesperades:

IF WS-StudentID IS NUMERIC
    CONTINUE
ELSE
    DISPLAY "Error: Invalid Student ID"
    STOP RUN.

5. Prova l'Aplicació

Prova exhaustivament la teva aplicació per assegurar-te que funciona com s'espera:

  • Prova totes les funcionalitats.
  • Gestiona casos límit i entrades inesperades.
  • Corregeix qualsevol error o problema.

6. Documenta l'Aplicació

Proporciona una documentació clara per a la teva aplicació:

  • Comentaris en el Codi: Inclou comentaris en el teu codi per explicar la lògica.
  • Manual d'Usuari: Escriu un manual d'usuari que expliqui com utilitzar l'aplicació.

Conclusió

En completar aquest projecte final, hauràs demostrat la teva capacitat per dissenyar, implementar i documentar una aplicació COBOL. Aquest projecte no només solidificarà la teva comprensió de la programació COBOL, sinó que també et proporcionarà un exemple tangible de les teves habilitats que podràs mostrar a possibles ocupadors o clients. Felicitats per arribar a aquest punt en el teu viatge de programació COBOL!
