Gestió Avançada d'Arxius en COBOL
Tècniques de Gestió d'Errors ►

Introducció

En aquesta secció, aprofundirem en les tècniques avançades de gestió d'arxius en COBOL. Això inclou entendre els arxius indexats, els arxius relatius i les operacions avançades d'arxius com l'accés dinàmic i la gestió de l'estat dels arxius. Al final d'aquesta secció, hauríeu de ser capaços de gestionar i manipular arxius en COBOL de manera eficient per a aplicacions complexes.

Conceptes Clau

  • Arxius Indexats: Arxius que permeten l'accés als registres basats en valors de clau.
  • Arxius Relatius: Arxius que permeten l'accés als registres basats en la seva posició relativa.
  • Accés Dinàmic: La capacitat de llegir, escriure i actualitzar registres en un arxiu de manera dinàmica.
  • Codis d'Estat d'Arxiu: Codis que indiquen l'èxit o el fracàs de les operacions d'arxiu.

Arxius Indexats

Els arxius indexats en COBOL permeten accedir als registres utilitzant una clau. Això és particularment útil per a bases de dades i aplicacions on és necessari recuperar registres ràpidament.

Definició d'un Arxiu Indexat

Per definir un arxiu indexat, cal especificar la clàusula ORGANIZATION com a INDEXED i definir la clau.

FILE-CONTROL.
    SELECT EMPLOYEE-FILE ASSIGN TO 'employee.dat'
        ORGANIZATION IS INDEXED
        ACCESS MODE IS DYNAMIC
        RECORD KEY IS EMP-ID
        FILE STATUS IS WS-FILE-STATUS.

DATA DIVISION.
FILE SECTION.
FD  EMPLOYEE-FILE.
01  EMPLOYEE-RECORD.
    05  EMP-ID         PIC X(5).
    05  EMP-NAME       PIC X(20).
    05  EMP-DEPT       PIC X(10).

Accés als Registres

Podeu accedir als registres en un arxiu indexat utilitzant les instruccions READ, WRITE, REWRITE i DELETE.

PROCEDURE DIVISION.
    OPEN I-O EMPLOYEE-FILE.
    MOVE '12345' TO EMP-ID.
    READ EMPLOYEE-FILE
        INVALID KEY DISPLAY 'Record not found'.
    IF WS-FILE-STATUS = '00'
        DISPLAY 'Employee Name: ' EMP-NAME
        DISPLAY 'Employee Dept: ' EMP-DEPT.
    CLOSE EMPLOYEE-FILE.

Arxius Relatius

Els arxius relatius permeten accedir als registres basats en la seva posició relativa en l'arxiu.

Definició d'un Arxiu Relatiu

Per definir un arxiu relatiu, utilitzeu la clàusula ORGANIZATION com a RELATIVE i especifiqueu la clau relativa.

FILE-CONTROL.
    SELECT CUSTOMER-FILE ASSIGN TO 'customer.dat'
        ORGANIZATION IS RELATIVE
        ACCESS MODE IS DYNAMIC
        RELATIVE KEY IS CUST-REL-KEY
        FILE STATUS IS WS-FILE-STATUS.

DATA DIVISION.
FILE SECTION.
FD  CUSTOMER-FILE.
01  CUSTOMER-RECORD.
    05  CUST-REL-KEY   PIC 9(4).
    05  CUST-NAME      PIC X(20).
    05  CUST-ADDRESS   PIC X(50).

Accés als Registres

Podeu accedir als registres en un arxiu relatiu utilitzant les instruccions READ, WRITE, REWRITE i DELETE.

PROCEDURE DIVISION.
    OPEN I-O CUSTOMER-FILE.
    MOVE 1001 TO CUST-REL-KEY.
    READ CUSTOMER-FILE
        INVALID KEY DISPLAY 'Record not found'.
    IF WS-FILE-STATUS = '00'
        DISPLAY 'Customer Name: ' CUST-NAME
        DISPLAY 'Customer Address: ' CUST-ADDRESS.
    CLOSE CUSTOMER-FILE.

Accés Dinàmic

L'accés dinàmic permet llegir, escriure i actualitzar registres en un arxiu sense tancar i reobrir l'arxiu.

Exemple d'Accés Dinàmic

PROCEDURE DIVISION.
    OPEN I-O EMPLOYEE-FILE.
    MOVE '12345' TO EMP-ID.
    READ EMPLOYEE-FILE
        INVALID KEY DISPLAY 'Record not found'.
    IF WS-FILE-STATUS = '00'
        MOVE 'John Doe' TO EMP-NAME
        REWRITE EMPLOYEE-RECORD
        INVALID KEY DISPLAY 'Update failed'.
    CLOSE EMPLOYEE-FILE.

Codis d'Estat d'Arxiu

Els codis d'estat d'arxiu proporcionen informació sobre l'èxit o el fracàs de les operacions d'arxiu. És crucial gestionar aquests codis per assegurar una gestió robusta dels arxius.

Codis d'Estat d'Arxiu Comuns

| Codi d'Estat | Significat | |--------------|--------------------------| | 00 | Operació reeixida | | 10 | Final de l'arxiu | | 23 | Registre no trobat | | 34 | Error d'I/O permanent | | 92 | Error lògic |

Gestió dels Codis d'Estat d'Arxiu

PROCEDURE DIVISION.
    OPEN I-O EMPLOYEE-FILE.
    IF WS-FILE-STATUS NOT = '00'
        DISPLAY 'Error opening file: ' WS-FILE-STATUS
        STOP RUN.
    MOVE '12345' TO EMP-ID.
    READ EMPLOYEE-FILE
        INVALID KEY DISPLAY 'Record not found: ' WS-FILE-STATUS.
    CLOSE EMPLOYEE-FILE.

Conclusió

La gestió avançada d'arxius en COBOL implica entendre i utilitzar arxius indexats i relatius, accés dinàmic i gestionar adequadament els codis d'estat d'arxiu. Dominar aquests conceptes us permetrà crear aplicacions COBOL eficients i robustes que poden gestionar operacions d'arxiu complexes. Practiqueu aquestes tècniques amb diversos exemples per consolidar la vostra comprensió i millorar les vostres habilitats de gestió d'arxius en COBOL.
