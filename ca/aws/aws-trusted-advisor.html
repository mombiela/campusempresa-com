Introducció a AWS Trusted Advisor

AWS Trusted Advisor és un servei que proporciona orientació en temps real per ajudar-te a aprovisionar els teus recursos seguint les millors pràctiques d'AWS. Les comprovacions de Trusted Advisor cobreixen cinc categories: optimització de costos, rendiment, seguretat, tolerància a fallades i límits de servei.

Conceptes Clau

  • Optimització de Costos: Identifica oportunitats per reduir els teus costos generals d'AWS.
  • Rendiment: Ajuda a millorar el rendiment dels teus serveis d'AWS.
  • Seguretat: Millora la seguretat del teu entorn AWS.
  • Tolerància a Fallades: Augmenta la disponibilitat i la redundància de les teves aplicacions AWS.
  • Límits de Servei: Supervisa el teu ús en comparació amb els límits de servei per evitar interrupcions.

Accedir a AWS Trusted Advisor

Per accedir a AWS Trusted Advisor:

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega al tauler de Trusted Advisor des de la Consola de Gestió d'AWS.

Utilitzar AWS Trusted Advisor

Visió General del Tauler

El tauler de Trusted Advisor proporciona un resum dels resultats de les teves comprovacions, categoritzats en les cinc àrees clau. Cada categoria mostra el nombre de comprovacions que estan en estat "verd" (sense problemes), "groc" (investigació recomanada) o "vermell" (acció recomanada).

Exemple: Comprovació d'Optimització de Costos

Una de les comprovacions més comunes és la comprovació d'optimització de costos. Aquesta comprovació t'ajuda a identificar recursos infrautilitzats que pots aturar o reduir per estalviar diners.

# Exemple: Identificar Instàncies EC2 Infrautilitzades
import boto3

# Inicialitza una sessió utilitzant Amazon EC2
sessió = boto3.Session(
    aws_access_key_id='YOUR_ACCESS_KEY',
    aws_secret_access_key='YOUR_SECRET_KEY',
    region_name='us-west-2'
)

# Inicialitza el recurs EC2
ec2 = sessió.resource('ec2')

# Llista totes les instàncies
for instance in ec2.instances.all():
    print(instance.id, instance.instance_type, instance.state)

Aquest script llista totes les instàncies EC2, que pots analitzar per identificar instàncies infrautilitzades.

Exemple: Comprovació de Seguretat

Les comprovacions de seguretat t'ajuden a identificar possibles vulnerabilitats de seguretat en el teu entorn AWS. Per exemple, pot comprovar si hi ha cubs S3 amb accés públic.

# Exemple: Comprovar Cub S3 Públic
import boto3

# Inicialitza una sessió utilitzant Amazon S3
s3 = boto3.client('s3')

# Llista tots els cubs
resposta = s3.list_buckets()

# Comprova cada cub per accés públic
for bucket in resposta['Buckets']:
    bucket_name = bucket['Name']
    acl = s3.get_bucket_acl(Bucket=bucket_name)
    for grant in acl['Grants']:
        if grant['Grantee']['Type'] == 'Group' and 'AllUsers' in grant['Grantee']['URI']:
            print(f"Cub {bucket_name} és públic.")

Aquest script comprova si hi ha cubs S3 amb accés públic i imprimeix els seus noms.

Ús Avançat

Personalitzar les Comprovacions de Trusted Advisor

Per a usuaris avançats, AWS Trusted Advisor permet la personalització de comprovacions per adaptar-se millor a necessitats específiques. Això es pot fer a través de la Consola de Gestió d'AWS o programàticament utilitzant els SDKs d'AWS.

Integrar Trusted Advisor amb Eines d'Automatització

Pots integrar Trusted Advisor amb eines d'automatització com AWS Lambda per remediar automàticament els problemes. Per exemple, pots crear una funció Lambda que aturi instàncies EC2 infrautilitzades basant-se en les recomanacions de Trusted Advisor.

# Exemple: Funció Lambda per Aturar Instàncies EC2 Infrautilitzades
import boto3

def lambda_handler(event, context):
    ec2 = boto3.client('ec2')
    trusted_advisor = boto3.client('support')
    
    # Obtenir resultats de la comprovació de Trusted Advisor
    resposta = trusted_advisor.describe_trusted_advisor_check_result(
        checkId='eW7HH0l7J9',  # Exemple d'ID de comprovació per instàncies infrautilitzades
        language='en'
    )
    
    # Aturar instàncies infrautilitzades
    for resource in resposta['result']['flaggedResources']:
        instance_id = resource['resourceId']
        ec2.stop_instances(InstanceIds=[instance_id])
        print(f"Instància {instance_id} aturada")

# Nota: Assegura't que la funció Lambda tingui els permisos IAM necessaris.

Conclusió

AWS Trusted Advisor és una eina poderosa que t'ajuda a optimitzar el teu entorn AWS en termes de costos, rendiment, seguretat, tolerància a fallades i límits de servei. Revisant regularment les recomanacions de Trusted Advisor i integrant-les en les teves pràctiques operacionals, pots assegurar-te que els teus recursos AWS s'utilitzen de manera eficient i segura.
