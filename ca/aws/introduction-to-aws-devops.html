Visió general

Amazon Web Services (AWS) proporciona un conjunt d'eines i serveis que permeten pràctiques DevOps. Aquesta secció us introduirà als conceptes fonamentals de AWS DevOps, les eines disponibles i com es poden utilitzar per automatitzar i agilitzar els vostres processos de desenvolupament i operacions.

Conceptes clau

  • DevOps: Un conjunt de pràctiques que combinen el desenvolupament de programari (Dev) i les operacions de TI (Ops) per escurçar el cicle de vida del desenvolupament i lliurar programari d'alta qualitat de manera contínua.
  • Eines AWS DevOps: Una col·lecció de serveis proporcionats per AWS per facilitar les pràctiques DevOps, incloent CI/CD, infraestructura com a codi, monitorització i registre.

Eines AWS DevOps

AWS ofereix una varietat d'eines per ajudar a implementar pràctiques DevOps. Aquí teniu alguns dels serveis clau:

  • AWS CodePipeline: Un servei d'integració contínua i lliurament continu (CI/CD) per a actualitzacions ràpides i fiables d'aplicacions i infraestructures.
  • AWS CodeBuild: Un servei de compilació totalment gestionat que compila codi font, executa proves i produeix paquets de programari llestos per al desplegament.
  • AWS CodeDeploy: Un servei que automatitza els desplegaments de codi a qualsevol instància, incloent instàncies Amazon EC2 i servidors locals.
  • AWS CodeCommit: Un servei de control de codi font totalment gestionat que facilita als equips allotjar repositoris Git segurs i escalables.
  • AWS CloudFormation: Un servei que proporciona un llenguatge comú per descriure i aprovisionar tots els recursos d'infraestructura en el vostre entorn de núvol.
  • Amazon CloudWatch: Un servei de monitorització i observabilitat dissenyat per a enginyers DevOps, desenvolupadors, enginyers de fiabilitat del lloc (SREs) i gestors de TI.

Configuració d'un pipeline bàsic de CI/CD

Pas 1: Crear un repositori CodeCommit

Primer, creeu un repositori CodeCommit per emmagatzemar el vostre codi font.

aws codecommit create-repository --repository-name MyDemoRepo --repository-description "El meu repositori de demostració"

Pas 2: Configurar CodeBuild

Creeu un projecte de compilació a CodeBuild per compilar el vostre codi i executar proves.

{
  "name": "MyDemoBuildProject",
  "source": {
    "type": "CODECOMMIT",
    "location": "https://git-codecommit.us-east-1.amazonaws.com/v1/repos/MyDemoRepo"
  },
  "artifacts": {
    "type": "NO_ARTIFACTS"
  },
  "environment": {
    "type": "LINUX_CONTAINER",
    "image": "aws/codebuild/standard:4.0",
    "computeType": "BUILD_GENERAL1_SMALL"
  },
  "serviceRole": "arn:aws:iam::123456789012:role/service-role/codebuild-service-role"
}

Pas 3: Configurar CodeDeploy

Configureu CodeDeploy per automatitzar el desplegament de la vostra aplicació.

{
  "applicationName": "MyDemoApplication",
  "deploymentGroupName": "MyDemoDeploymentGroup",
  "serviceRoleArn": "arn:aws:iam::123456789012:role/CodeDeployDemoRole",
  "deploymentConfigName": "CodeDeployDefault.OneAtATime",
  "ec2TagFilters": [
    {
      "Key": "Name",
      "Value": "MyDemoInstance",
      "Type": "KEY_AND_VALUE"
    }
  ]
}

Pas 4: Crear un CodePipeline

Finalment, creeu un pipeline a CodePipeline per unir-ho tot.

{
  "pipeline": {
    "name": "MyDemoPipeline",
    "roleArn": "arn:aws:iam::123456789012:role/AWS-CodePipeline-Service",
    "artifactStore": {
      "type": "S3",
      "location": "my-demo-bucket"
    },
    "stages": [
      {
        "name": "Source",
        "actions": [
          {
            "name": "Source",
            "actionTypeId": {
              "category": "Source",
              "owner": "AWS",
              "provider": "CodeCommit",
              "version": "1"
            },
            "outputArtifacts": [
              {
                "name": "SourceArtifact"
              }
            ],
            "configuration": {
              "RepositoryName": "MyDemoRepo",
              "BranchName": "main"
            }
          }
        ]
      },
      {
        "name": "Build",
        "actions": [
          {
            "name": "Build",
            "actionTypeId": {
              "category": "Build",
              "owner": "AWS",
              "provider": "CodeBuild",
              "version": "1"
            },
            "inputArtifacts": [
              {
                "name": "SourceArtifact"
              }
            ],
            "outputArtifacts": [
              {
                "name": "BuildArtifact"
              }
            ],
            "configuration": {
              "ProjectName": "MyDemoBuildProject"
            }
          }
        ]
      },
      {
        "name": "Deploy",
        "actions": [
          {
            "name": "Deploy",
            "actionTypeId": {
              "category": "Deploy",
              "owner": "AWS",
              "provider": "CodeDeploy",
              "version": "1"
            },
            "inputArtifacts": [
              {
                "name": "BuildArtifact"
              }
            ],
            "configuration": {
              "ApplicationName": "MyDemoApplication",
              "DeploymentGroupName": "MyDemoDeploymentGroup"
            }
          }
        ]
      }
    ]
  }
}

Conclusió

En aquesta secció, hem introduït els conceptes bàsics de AWS DevOps i hem explorat alguns dels serveis clau d'AWS que donen suport a les pràctiques DevOps. També hem recorregut la configuració d'un pipeline bàsic de CI/CD utilitzant AWS CodeCommit, CodeBuild, CodeDeploy i CodePipeline. Mitjançant aquestes eines, podeu automatitzar el vostre cicle de vida de desenvolupament de programari, millorar l'eficiència i assegurar desplegaments consistents i fiables.
