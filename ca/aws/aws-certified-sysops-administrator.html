Introducció

El curs AWS Certified SysOps Administrator està dissenyat per ajudar-te a adquirir les habilitats i coneixements necessaris per gestionar, desplegar i operar sistemes escalables, altament disponibles i tolerants a fallades a AWS. Aquest curs et portarà des d'un nivell principiant fins a un nivell avançat, cobrint tots els temes essencials necessaris per aprovar l'examen AWS Certified SysOps Administrator.

Exemple: Configuració d'una alarma de CloudWatch

import boto3

# Crear client de CloudWatch
cloudwatch = boto3.client('cloudwatch')

# Crear alarma
cloudwatch.put_metric_alarm(
    AlarmName='CPU_Utilization_Alarm',
    ComparisonOperator='GreaterThanThreshold',
    EvaluationPeriods=1,
    MetricName='CPUUtilization',
    Namespace='AWS/EC2',
    Period=60,
    Statistic='Average',
    Threshold=70.0,
    ActionsEnabled=False,
    AlarmActions=[
        'arn:aws:sns:us-east-1:123456789012:MyTopic'
    ],
    AlarmDescription='Alarma quan la CPU del servidor supera el 70%',
    Dimensions=[
        {
          'Name': 'InstanceId',
          'Value': 'i-1234567890abcdef0'
        },
    ],
    Unit='Seconds'
)

Explicació: Aquest codi configura una alarma de CloudWatch per monitoritzar la utilització de la CPU d'una instància EC2 i activa una alarma si supera el 70%.
