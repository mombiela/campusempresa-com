Introducció a AWS Budgets

AWS Budgets és un servei que et permet establir pressupostos personalitzats de cost i ús per al teu compte d'AWS. T'ajuda a monitoritzar els teus costos i ús d'AWS, proporcionant alertes quan superes o es preveu que superis els llindars del teu pressupost.

Conceptes Clau

  • Tipus de Pressupost: Pressupostos de cost, pressupostos d'ús i pressupostos de reserva.
  • Llindars: Estableix límits per als teus pressupostos per activar alertes.
  • Notificacions: Alertes enviades per correu electrònic o SNS quan es superen els llindars.
  • Períodes de Temps: Mensuals, trimestrals o anuals.

Configuració de AWS Budgets

Crear un Pressupost

Per crear un pressupost, segueix aquests passos:

  1. Navega al Tauler de Control de AWS Budgets:

    • Inicia sessió a la Consola de Gestió d'AWS.
    • Obre el Tauler de Control de Facturació i Gestió de Costos.
    • Selecciona "Budgets" des del panell de navegació.

  2. Crea un Nou Pressupost:

    • Fes clic a "Create a budget".
    • Tria el tipus de pressupost: Cost, Ús o Reserva.

  3. Configura els Detalls del Pressupost:

    • Nom: Dona un nom descriptiu al teu pressupost.
    • Període: Selecciona el període de temps (mensual, trimestral, anual).
    • Quantitat: Estableix la quantitat del pressupost o límit d'ús.
    • Filtres: Aplica filtres per restringir l'abast (per exemple, serveis específics, comptes enllaçats).

  4. Estableix Alertes:

    • Defineix percentatges de llindar (per exemple, 80%, 100%).
    • Especifica canals de notificació (correu electrònic, SNS).

  5. Revisa i Crea:

    • Revisa la configuració del teu pressupost.
    • Fes clic a "Create budget".

Exemple: Crear un Pressupost Mensual de Cost

1. Navega al Tauler de Control de AWS Budgets.
2. Fes clic a "Create a budget".
3. Selecciona "Cost budget".
4. Nom: "Monthly EC2 Budget".
5. Període: Mensual.
6. Quantitat Pressupostada: $500.
7. Filtres: Servei - EC2.
8. Estableix Alertes: 
   - Llindar del 80%, notificar per correu electrònic.
   - Llindar del 100%, notificar per SNS.
9. Revisa i Crea.

Gestió i Monitorització de Pressupostos

Veure el Rendiment del Pressupost

  • Tauler de Control: El tauler de control de AWS Budgets proporciona una visió general de tots els teus pressupostos.
  • Detalls: Fes clic en un pressupost per veure el rendiment detallat, incloent costos o ús reals vs. previstos.

Modificar Pressupostos

  • Editar Pressupost: Pots modificar la configuració del pressupost, com la quantitat, període i filtres.
  • Ajustar Alertes: Canvia els nivells de llindar o els canals de notificació segons sigui necessari.

Exemple: Modificar un Pressupost Existent

1. Navega al Tauler de Control de AWS Budgets.
2. Selecciona el pressupost que vols modificar.
3. Fes clic a "Edit budget".
4. Ajusta la quantitat del pressupost o els filtres.
5. Actualitza els llindars d'alerta o la configuració de notificació.
6. Desa els canvis.

Funcions Avançades

Utilitzar AWS Budgets amb AWS Cost Explorer

  • Integració: AWS Budgets s'integra amb AWS Cost Explorer per proporcionar una anàlisi detallada de costos i ús.
  • Previsió: Utilitza Cost Explorer per preveure costos futurs i ajustar els pressupostos en conseqüència.

Automatitzar Respostes amb Accions de AWS Budgets

  • Accions: Activa automàticament accions com aturar instàncies o enviar notificacions quan es superen els llindars del pressupost.
  • Configuració: Configura accions al tauler de control de AWS Budgets.

Exemple: Configurar una Acció

1. Navega al Tauler de Control de AWS Budgets.
2. Selecciona el pressupost al qual vols afegir una acció.
3. Fes clic a "Edit budget".
4. Ves a la secció "Actions".
5. Defineix l'acció (per exemple, aturar instàncies EC2).
6. Estableix el llindar de desencadenament (per exemple, 100% del pressupost).
7. Desa els canvis.

Conclusió

AWS Budgets és una eina potent per gestionar i controlar els teus costos i ús d'AWS. Configurant pressupostos, configurant alertes i utilitzant funcions avançades com accions automatitzades, pots assegurar-te que la teva despesa en AWS es mantingui dins dels límits desitjats. Monitoritzar i ajustar regularment els teus pressupostos t'ajudarà a optimitzar les teves despeses en el núvol i evitar càrrecs inesperats.
