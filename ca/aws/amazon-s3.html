Amazon S3 (Simple Storage Service) és un servei d'emmagatzematge d'objectes escalable proporcionat per AWS. Està dissenyat per emmagatzemar i recuperar qualsevol quantitat de dades des de qualsevol lloc de la web. Aquest tema et guiarà des dels conceptes bàsics fins a les característiques avançades d'Amazon S3, ajudant-te a entendre com aprofitar-lo per a les teves necessitats d'emmagatzematge.

Introducció a Amazon S3

  • Què és Amazon S3?

    • Amazon S3 és un servei d'emmagatzematge d'objectes que ofereix escalabilitat líder en la indústria, disponibilitat de dades, seguretat i rendiment.
    • Està dissenyat per emmagatzemar i recuperar qualsevol quantitat de dades des de qualsevol lloc de la web.

  • Conceptes Clau:

    • Buckets: Contenidors per emmagatzemar objectes.
    • Objectes: Fitxers i metadades emmagatzemats en buckets.
    • Claus: Identificador únic per a un objecte dins d'un bucket.
    • Regions: Ubicacions geogràfiques on s'emmagatzemen els buckets.

Configuració d'Amazon S3

  • Creació d'un Bucket S3:
    1. Inicia sessió a la Consola de Gestió d'AWS.
    2. Navega al servei S3.
    3. Fes clic a "Create bucket."
    4. Introdueix un nom de bucket únic i selecciona una regió.
    5. Configura opcions com la versió, el registre i el xifratge.
    6. Revisa i crea el bucket.
import boto3

# Crear un client S3
s3 = boto3.client('s3')

# Crear un bucket
s3.create_bucket(Bucket='my-new-bucket', CreateBucketConfiguration={
    'LocationConstraint': 'us-west-1'})
  • Pujar Objectes a S3:
    • Pots pujar fitxers al teu bucket S3 utilitzant la Consola de Gestió d'AWS, AWS CLI, o SDKs.
# Pujar un fitxer al bucket
s3.upload_file('local_file.txt', 'my-new-bucket', 'remote_file.txt')

Gestió de Buckets i Objectes S3

  • Polítiques de Bucket i Permisos:
    • Controla l'accés als teus buckets i objectes S3 utilitzant polítiques de bucket, polítiques IAM i ACLs (Llistes de Control d'Accés).
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": "*",
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::my-new-bucket/*"
    }
  ]
}
  • Versió:
    • Activa la versió per mantenir múltiples versions d'un objecte en el mateix bucket.
# Activar la versió en un bucket
s3.put_bucket_versioning(Bucket='my-new-bucket', VersioningConfiguration={
    'Status': 'Enabled'})
  • Polítiques de Cicle de Vida:
    • Defineix regles per gestionar el cicle de vida dels objectes, com ara la transició d'objectes a diferents classes d'emmagatzematge o la seva expiració després d'un cert període.
{
  "Rules": [
    {
      "ID": "Move to Glacier after 30 days",
      "Prefix": "",
      "Status": "Enabled",
      "Transitions": [
        {
          "Days": 30,
          "StorageClass": "GLACIER"
        }
      ]
    }
  ]
}

Característiques Avançades d'Amazon S3

  • Classes d'Emmagatzematge S3:
    • Standard: Emmagatzematge de propòsit general per a dades accedides freqüentment.
    • Intelligent-Tiering: Mou automàticament les dades a la capa d'accés més rendible.
    • Standard-IA (Infrequent Access): Per a dades accedides menys freqüentment.
    • One Zone-IA: Opció de menor cost per a dades accedides infreqüentment emmagatzemades en una sola zona de disponibilitat.
    • Glacier: Emmagatzematge de baix cost per a dades d'arxiu.
    • Glacier Deep Archive: Emmagatzematge de menor cost per a la retenció a llarg termini.

| Classe d'Emmagatzematge | Cas d'Ús | Disponibilitat | Durabilitat | |-------------------------|-----------------------------------|----------------|---------------------| | Standard | Dades accedides freqüentment | 99.99% | 99.999999999% | | Intelligent-Tiering | Patrons d'accés desconeguts o canviants | 99.9% | 99.999999999% | | Standard-IA | Dades accedides infreqüentment | 99.9% | 99.999999999% | | One Zone-IA | Dades no crítiques accedides infreqüentment | 99.5% | 99.999999999% | | Glacier | Dades d'arxiu | 99.99% | 99.999999999% | | Glacier Deep Archive | Dades d'arxiu a llarg termini | 99.99% | 99.999999999% |

  • Replicació entre Regions:
    • Replícala automàticament objectes a diferents regions d'AWS per a redundància i recuperació de desastres.
# Activar la replicació entre regions
s3.put_bucket_replication(Bucket='my-new-bucket', ReplicationConfiguration={
    'Role': 'arn:aws:iam::account-id:role/replication-role',
    'Rules': [
        {
            'Status': 'Enabled',
            'Prefix': '',
            'Destination': {
                'Bucket': 'arn:aws:s3:::destination-bucket'
            }
        }
    ]
})
  • Notificacions d'Esdeveniments:
    • Configura S3 per enviar notificacions a AWS Lambda, SNS o SQS quan es produeixin certs esdeveniments, com ara la creació o eliminació d'objectes.
# Configurar notificacions d'esdeveniments
s3.put_bucket_notification_configuration(Bucket='my-new-bucket', NotificationConfiguration={
    'LambdaFunctionConfigurations': [
        {
            'LambdaFunctionArn': 'arn:aws:lambda:region:account-id:function:my-function',
            'Events': ['s3:ObjectCreated:*']
        }
    ]
})

Conclusió

Amazon S3 és un servei d'emmagatzematge versàtil i potent que pot gestionar una àmplia gamma de necessitats d'emmagatzematge, des de dades accedides freqüentment fins a arxius a llarg termini. Entenent els seus conceptes bàsics, configurant i gestionant buckets i objectes, i aprofitant les característiques avançades, pots optimitzar la teva estratègia d'emmagatzematge i assegurar-te que les teves dades siguin segures, disponibles i rendibles.
