Introducció

Amazon Web Services (AWS) proporciona un conjunt robust d'eines i serveis per ajudar-te a assegurar el teu entorn al núvol. Aquesta secció et guiarà a través de les millors pràctiques per assegurar la teva infraestructura AWS, des de nivells bàsics fins a avançats.

Pràctiques Bàsiques de Seguretat

Gestió d'Identitat i Accés (IAM)

  • Utilitza Rols IAM: Assigna rols a serveis i usuaris en lloc de compartir credencials.
  • Principi del Menor Privilegi: Concedeix només els permisos necessaris perquè els usuaris puguin realitzar les seves tasques.
  • Autenticació Multi-Factor (MFA): Activa MFA per a tots els usuaris, especialment per als comptes root.
# Exemple: Creant un rol IAM utilitzant Boto3 en Python
import boto3

iam = boto3.client('iam')

response = iam.create_role(
    RoleName='MyExampleRole',
    AssumeRolePolicyDocument='''{
        "Version": "2012-10-17",
        "Statement": [
            {
                "Effect": "Allow",
                "Principal": {
                    "Service": "ec2.amazonaws.com"
                },
                "Action": "sts:AssumeRole"
            }
        ]
    }'''
)

print(response)

Assegura el Teu Compte Root

  • Evita Utilitzar el Compte Root: Crea usuaris IAM individuals en lloc d'això.
  • Activa MFA: Afegeix una capa extra de seguretat al teu compte root.

Seguretat de Xarxa

  • Utilitza Grups de Seguretat: Controla el tràfic entrant i sortint cap a les teves instàncies.
  • Llistes de Control d'Accés de Xarxa (NACLs): Proporciona una capa addicional de seguretat a nivell de subxarxa.
# Exemple: Creant un grup de seguretat utilitzant Boto3 en Python
ec2 = boto3.client('ec2')

response = ec2.create_security_group(
    GroupName='MySecurityGroup',
    Description='My security group',
    VpcId='vpc-12345678'
)

print(response)

Pràctiques Intermèdies de Seguretat

Protecció de Dades

  • Xifrat: Utilitza AWS Key Management Service (KMS) per xifrar dades en repòs i en trànsit.
  • Polítiques de Bucket S3: Assegura't que els teus buckets S3 no siguin públicament accessibles a menys que sigui necessari.
# Exemple: Activant el xifrat en un bucket S3 utilitzant Boto3 en Python
s3 = boto3.client('s3')

response = s3.put_bucket_encryption(
    Bucket='my-bucket',
    ServerSideEncryptionConfiguration={
        'Rules': [
            {
                'ApplyServerSideEncryptionByDefault': {
                    'SSEAlgorithm': 'AES256'
                }
            }
        ]
    }
)

print(response)

Monitorització i Registre

  • AWS CloudTrail: Activa CloudTrail per registrar totes les crides API.
  • Amazon CloudWatch: Monitoritza els teus recursos i aplicacions AWS.
# Exemple: Activant CloudTrail utilitzant Boto3 en Python
cloudtrail = boto3.client('cloudtrail')

response = cloudtrail.create_trail(
    Name='MyTrail',
    S3BucketName='my-cloudtrail-bucket'
)

print(response)

Pràctiques Avançades de Seguretat

Seguretat Avançada de Xarxa

  • VPC Flow Logs: Captura informació sobre el tràfic IP que va i ve de les interfícies de xarxa a la teva VPC.
  • AWS WAF i Shield: Protegeix les teves aplicacions de vulnerabilitats web comunes i atacs DDoS.
# Exemple: Activant VPC Flow Logs utilitzant Boto3 en Python
ec2 = boto3.client('ec2')

response = ec2.create_flow_logs(
    ResourceIds=['vpc-12345678'],
    ResourceType='VPC',
    TrafficType='ALL',
    LogGroupName='my-vpc-flow-logs',
    DeliverLogsPermissionArn='arn:aws:iam::123456789012:role/flow-logs-role'
)

print(response)

Compliment i Governança

  • AWS Config: Monitoritza i registra contínuament les configuracions dels teus recursos AWS.
  • AWS Organizations: Gestiona i governa centralment el teu entorn a mesura que creixes i escales els teus recursos AWS.
# Exemple: Activant AWS Config utilitzant Boto3 en Python
config = boto3.client('config')

response = config.put_configuration_recorder(
    ConfigurationRecorder={
        'name': 'default',
        'roleARN': 'arn:aws:iam::123456789012:role/config-role',
        'recordingGroup': {
            'allSupported': True
        }
    }
)

print(response)

Resum

Assegurar el teu entorn AWS és un procés continu que implica múltiples capes de protecció. Seguint aquestes millors pràctiques, pots reduir significativament el risc de violacions de seguretat i assegurar que les teves dades i aplicacions estiguin segures. Des de configurar rols IAM i activar MFA fins a pràctiques avançades com utilitzar AWS WAF i AWS Config, cada pas és crucial per construir una infraestructura AWS segura.
