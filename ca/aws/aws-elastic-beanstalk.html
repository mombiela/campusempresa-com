Introducció a AWS Elastic Beanstalk

AWS Elastic Beanstalk és una Plataforma com a Servei (PaaS) que permet als desenvolupadors desplegar i gestionar aplicacions al núvol d'AWS sense preocupar-se per la infraestructura que executa aquestes aplicacions.

  • Què és AWS Elastic Beanstalk?
    • Un servei per desplegar i escalar aplicacions web i serveis.
    • Suporta múltiples llenguatges de programació incloent Java, .NET, PHP, Node.js, Python, Ruby, Go i Docker.
    • Gestiona automàticament el desplegament, des de la provisió de capacitat, equilibrament de càrrega i auto-escalat fins al monitoratge de la salut de l'aplicació.

Conceptes Clau

  • Entorn: Una col·lecció de recursos d'AWS que executen una versió d'aplicació.
  • Aplicació: Una col·lecció lògica de components d'Elastic Beanstalk, incloent entorns, versions i configuracions.
  • Versió d'Aplicació: Una iteració específica i etiquetada de codi desplegable per a una aplicació web.
  • Plantilla de Configuració: Un conjunt de paràmetres i configuracions que defineixen com es comporta un entorn i els seus recursos.

Configuració d'AWS Elastic Beanstalk

Prerequisits

  • Compte d'AWS
  • AWS CLI instal·lat i configurat
  • Comprensió bàsica d'aplicacions web

Passos per Crear una Aplicació Elastic Beanstalk

  1. Crear una Aplicació:

    eb init -p python-3.7 my-app

    Aquesta comanda inicialitza una nova aplicació Elastic Beanstalk utilitzant Python 3.7.

  2. Crear un Entorn:

    eb create my-env

    Aquesta comanda crea un nou entorn anomenat my-env.

  3. Desplegar l'Aplicació:

    eb deploy

    Aquesta comanda desplega l'aplicació a l'entorn creat.

Exemple: Desplegant una Aplicació Simple en Python

  1. Crear una Aplicació en Python:

    # application.py
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

  2. Inicialitzar Elastic Beanstalk:

    eb init -p python-3.7 my-flask-app

  3. Crear un Entorn i Desplegar:

    eb create flask-env
eb deploy

Gestió d'Entorns

  • Escalat: Elastic Beanstalk escala automàticament la teva aplicació amunt i avall segons la teva configuració.
  • Monitoratge: Utilitza el tauler de control d'Elastic Beanstalk per monitorar la salut i el rendiment de la teva aplicació.
  • Actualització: Desplega noves versions de la teva aplicació utilitzant la comanda eb deploy.

Temes Avançats

Personalització de la Configuració de l'Entorn

  • Fitxers de Configuració: Utilitza .ebextensions per personalitzar el teu entorn. 
    # .ebextensions/options.config
option_settings:
  aws:elasticbeanstalk:application:environment:
    DJANGO_SETTINGS_MODULE: "myapp.settings"

Utilitzant Docker amb Elastic Beanstalk

  • Dockerfile: Defineix el teu entorn Docker.

    # Dockerfile
FROM python:3.7-slim
WORKDIR /app
COPY . /app
RUN pip install -r requirements.txt
CMD ["python", "application.py"]

  • Desplegant Aplicació Docker:

    eb init -p docker my-docker-app
eb create docker-env
eb deploy

Conclusió

AWS Elastic Beanstalk simplifica el procés de desplegar i gestionar aplicacions web al núvol. Abstrau la infraestructura subjacent, permetent als desenvolupadors centrar-se en escriure codi. Comprenent els conceptes clau i seguint els passos de configuració, pots posar en marxa les teves aplicacions ràpidament a AWS. Les característiques avançades com la personalització de l'entorn i el suport per Docker proporcionen flexibilitat per a casos d'ús més complexos.
