Introducció a AWS Lex i Polly

AWS Lex i Polly són dos serveis potents proporcionats per Amazon Web Services que permeten als desenvolupadors crear interfícies conversacionals i convertir text en veu realista, respectivament.

  • AWS Lex: Un servei per crear interfícies conversacionals utilitzant veu i text.
  • AWS Polly: Un servei que converteix text en veu realista.

Començant amb AWS Lex

Què és AWS Lex?

AWS Lex és un servei per crear interfícies conversacionals en qualsevol aplicació utilitzant veu i text. Proporciona les funcionalitats d'aprenentatge profund de reconeixement automàtic de veu (ASR) per convertir veu en text, i comprensió del llenguatge natural (NLU) per reconèixer la intenció del text.

Conceptes Clau en AWS Lex

  • Intencions: Representa una acció que l'usuari vol realitzar.
  • Expressions: Frases que els usuaris poden dir per expressar una intenció.
  • Slots: Paràmetres que capturen informació específica de l'usuari.
  • Compliment: L'acció presa per satisfer la intenció.

Creant el teu Primer Bot Lex

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega al servei Lex.
  3. Crea un nou bot:
    • Tria una plantilla o comença des de zero.
    • Defineix les intencions, expressions i slots del bot.
  4. Construeix i prova el bot:
    • Utilitza la finestra de prova integrada per interactuar amb el teu bot.
  5. Desplega el bot:
    • Integra amb altres serveis d'AWS o plataformes de tercers.

Exemple: Bot Simple per Demanar Pizza

{
  "intents": [
    {
      "name": "OrderPizza",
      "sampleUtterances": [
        "Vull demanar una pizza",
        "Puc demanar una pizza?",
        "Demanar una pizza"
      ],
      "slots": [
        {
          "name": "PizzaType",
          "slotType": "PizzaType",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "PlainText",
                "content": "Quin tipus de pizza t'agradaria?"
              }
            ],
            "maxAttempts": 2
          }
        },
        {
          "name": "PizzaSize",
          "slotType": "PizzaSize",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "PlainText",
                "content": "Quina mida de pizza t'agradaria?"
              }
            ],
            "maxAttempts": 2
          }
        }
      ],
      "fulfillmentActivity": {
        "type": "ReturnIntent"
      }
    }
  ]
}

Començant amb AWS Polly

Què és AWS Polly?

AWS Polly és un servei que converteix text en veu realista, permetent-te crear aplicacions que parlen i construir categories completament noves de productes habilitats per veu.

Conceptes Clau en AWS Polly

  • Veus: Polly ofereix desenes de veus realistes en una varietat de llengües.
  • Llenguatge de Marcatge de Síntesi de Veu (SSML): Un estàndard per controlar aspectes de la veu com la pronunciació, el volum i el to.

Utilitzant AWS Polly

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega al servei Polly.
  3. Introdueix text i selecciona una veu.
  4. Sintetitza veu:
    • Escolta la veu generada.
    • Descarrega el fitxer d'àudio.

Exemple: Convertint Text a Veu

import boto3

# Crea un client Polly
polly = boto3.client('polly')

# Sol·licita síntesi de veu
response = polly.synthesize_speech(
    Text='Hola, benvingut a AWS Polly!',
    OutputFormat='mp3',
    VoiceId='Joanna'
)

# Desa el flux d'àudio retornat per Amazon Polly
with open('speech.mp3', 'wb') as file:
    file.write(response['AudioStream'].read())

Temes Avançats

Integrant Lex i Polly

  • Interacció per Veu: Utilitza Polly per proporcionar respostes de veu en un bot Lex.
  • SSML en Lex: Millora les respostes de Lex amb SSML per a interaccions més naturals.

Exemple: Utilitzant Polly amb Lex

{
  "intents": [
    {
      "name": "OrderPizza",
      "sampleUtterances": [
        "Vull demanar una pizza"
      ],
      "slots": [
        {
          "name": "PizzaType",
          "slotType": "PizzaType",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "SSML",
                "content": "<speak>Quin tipus de pizza t'agradaria?</speak>"
              }
            ],
            "maxAttempts": 2
          }
        }
      ],
      "fulfillmentActivity": {
        "type": "ReturnIntent"
      }
    }
  ]
}

Conclusió

AWS Lex i Polly són eines potents per crear interfícies conversacionals i convertir text en veu. Comprenent els seus conceptes clau i aprenent a integrar-los, pots construir aplicacions sofisticades habilitades per veu que proporcionen una experiència d'usuari natural i atractiva. Tant si acabes de començar com si vols aprofundir en els teus coneixements, dominar aquests serveis obrirà noves possibilitats en el camp de les interaccions per veu i text.
