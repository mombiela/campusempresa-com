Introducció a AWS

Amazon Web Services (AWS) és una plataforma de núvol completa i àmpliament adoptada oferta per Amazon. Proporciona una varietat de serveis de computació en núvol que ajuden les empreses a escalar i créixer. AWS ofereix serveis en diversos dominis com la potència de computació, l'emmagatzematge, les bases de dades, l'aprenentatge automàtic, i més.

Conceptes Clau d'AWS

  • Computació en Núvol: El lliurament de serveis de computació a través d'internet ("el núvol") per oferir una innovació més ràpida, recursos flexibles i economies d'escala.
  • Regions i Zones de Disponibilitat: La infraestructura d'AWS està organitzada en regions i zones de disponibilitat per garantir una alta disponibilitat i tolerància a fallades.
  • Preus de Pagament per Ús: AWS cobra segons l'ús, permetent a les empreses escalar recursos amunt o avall segons sigui necessari sense costos inicials.

Serveis Bàsics d'AWS

Computació

  • Amazon EC2 (Elastic Compute Cloud): Proporciona servidors virtuals escalables.
  • AWS Lambda: Et permet executar codi sense aprovisionar o gestionar servidors.

Emmagatzematge

  • Amazon S3 (Simple Storage Service): Emmagatzematge d'objectes escalable per a còpies de seguretat i arxiu de dades.
  • Amazon EBS (Elastic Block Store): Emmagatzematge de blocs per utilitzar amb Amazon EC2.

Bases de Dades

  • Amazon RDS (Relational Database Service): Servei de base de dades relacional gestionat.
  • Amazon DynamoDB: Servei de base de dades NoSQL completament gestionat.

Xarxes

  • Amazon VPC (Virtual Private Cloud): Et permet aprovisionar una secció lògicament aïllada del núvol d'AWS.
  • Amazon Route 53: Servei web escalable de Sistema de Noms de Domini (DNS).

Exemple: Llançament d'una Instància EC2

Llançar una instància EC2 és una de les tasques més comunes a AWS. Aquí tens un exemple senzill utilitzant la Consola de Gestió d'AWS:

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega al Tauler de Control d'EC2.
  3. Fes clic a "Llançar Instància".
  4. Tria una Imatge de Màquina d'Amazon (AMI): Selecciona una AMI, com Amazon Linux 2.
  5. Tria un Tipus d'Instància: Selecciona un tipus d'instància, com t2.micro (elegible per a la capa gratuïta).
  6. Configura els Detalls de la Instància: Configura el nombre d'instàncies, la configuració de xarxa, etc.
  7. Afegeix Emmagatzematge: Configura les opcions d'emmagatzematge, com els volums EBS.
  8. Afegeix Etiquetes: Opcionalment, afegeix etiquetes a la teva instància per a una gestió més fàcil.
  9. Configura el Grup de Seguretat: Configura les regles del tallafoc per permetre el trànsit a la teva instància.
  10. Revisa i Llança: Revisa la teva configuració i fes clic a "Llançar".
# Exemple de llançament d'una instància EC2 utilitzant AWS CLI
aws ec2 run-instances --image-id ami-0abcdef1234567890 --count 1 --instance-type t2.micro --key-name MyKeyPair --security-group-ids sg-0abc1234 --subnet-id subnet-0abc1234

Conclusió

AWS és una plataforma de núvol potent i flexible que ofereix una àmplia gamma de serveis per satisfer diverses necessitats empresarials. Tant si necessites potència de computació, solucions d'emmagatzematge o gestió de bases de dades, AWS proporciona opcions escalables i rendibles. Entendre els conceptes bàsics d'AWS i els seus serveis bàsics és el primer pas per aprofitar tot el potencial de la computació en núvol.
