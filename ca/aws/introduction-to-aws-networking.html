Introducció a la Xarxa d'AWS
Amazon Route 53 ►

Visió General

Amazon Web Services (AWS) proporciona una plataforma de computació en núvol robusta i escalable que inclou una varietat de serveis de xarxa. Aquesta secció t'introduirà als conceptes fonamentals de la xarxa d'AWS, incloent-hi Virtual Private Cloud (VPC), subxarxes, taules de rutes i grups de seguretat.

Conceptes Clau

Virtual Private Cloud (VPC)

  • Definició: Una VPC és una xarxa virtual dedicada al teu compte d'AWS. Està lògicament aïllada d'altres xarxes virtuals al núvol d'AWS.
  • Propòsit: Et permet llançar recursos d'AWS en una xarxa virtual que has definit.
  • Components:
    • Bloc CIDR: Defineix el rang d'adreces IP per a la VPC.
    • Subxarxes: Segments del rang d'adreces IP de la VPC on pots col·locar grups de recursos aïllats.

Subxarxes

  • Definició: Una subxarxa és un rang d'adreces IP dins de la teva VPC.
  • Tipus:
    • Subxarxa Pública: Accessible des d'internet.
    • Subxarxa Privada: No accessible des d'internet.
  • Propòsit: Organitzar i assegurar els teus recursos dins de la VPC.

Taules de Rutes

  • Definició: Un conjunt de regles, anomenades rutes, que s'utilitzen per determinar on es dirigeix el trànsit de xarxa.
  • Components:
    • Destinació: El rang d'adreces IP on es dirigeix el trànsit.
    • Objectiu: La passarel·la, interfície de xarxa o connexió a través de la qual s'envia el trànsit.

Passarel·la d'Internet

  • Definició: Un component de VPC escalat horitzontalment, redundant i altament disponible que permet la comunicació entre instàncies a la teva VPC i internet.
  • Propòsit: Permet que les instàncies en una VPC es connectin a internet.

Passarel·la NAT

  • Definició: Un servei de traducció d'adreces de xarxa (NAT) per a les teves instàncies en una subxarxa privada per accedir a internet.
  • Propòsit: Permet el trànsit de sortida a internet des de les instàncies en una subxarxa privada mentre es prevé el trànsit d'entrada.

Grups de Seguretat

  • Definició: Tallafocs virtuals que controlen el trànsit entrant i sortint cap als recursos d'AWS.
  • Components:
    • Regles d'Entrada: Defineixen el trànsit entrant permès.
    • Regles de Sortida: Defineixen el trànsit sortint permès.

Exemple: Crear una VPC amb Subxarxes Públiques i Privades

Guia Pas a Pas

  1. Crear una VPC:

    import boto3

ec2 = boto3.client('ec2')

response = ec2.create_vpc(
    CidrBlock='10.0.0.0/16'
)
vpc_id = response['Vpc']['VpcId']
print(f'Created VPC with ID: {vpc_id}')

  2. Crear Subxarxes:

    # Subxarxa Pública
public_subnet = ec2.create_subnet(
    CidrBlock='10.0.1.0/24',
    VpcId=vpc_id
)
public_subnet_id = public_subnet['Subnet']['SubnetId']
print(f'Created Public Subnet with ID: {public_subnet_id}')

# Subxarxa Privada
private_subnet = ec2.create_subnet(
    CidrBlock='10.0.2.0/24',
    VpcId=vpc_id
)
private_subnet_id = private_subnet['Subnet']['SubnetId']
print(f'Created Private Subnet with ID: {private_subnet_id}')

  3. Crear una Passarel·la d'Internet i Adjuntar-la a la VPC:

    igw = ec2.create_internet_gateway()
igw_id = igw['InternetGateway']['InternetGatewayId']
ec2.attach_internet_gateway(
    InternetGatewayId=igw_id,
    VpcId=vpc_id
)
print(f'Created and attached Internet Gateway with ID: {igw_id}')

  4. Crear una Taula de Rutes i Afegir Rutes:

    route_table = ec2.create_route_table(VpcId=vpc_id)
route_table_id = route_table['RouteTable']['RouteTableId']
ec2.create_route(
    RouteTableId=route_table_id,
    DestinationCidrBlock='0.0.0.0/0',
    GatewayId=igw_id
)
print(f'Created Route Table with ID: {route_table_id} and added route to Internet Gateway')

  5. Associar la Taula de Rutes amb la Subxarxa Pública:

    ec2.associate_route_table(
    RouteTableId=route_table_id,
    SubnetId=public_subnet_id
)
print(f'Associated Route Table with Public Subnet')

  6. Crear Grups de Seguretat:

    security_group = ec2.create_security_group(
    GroupName='my-security-group',
    Description='My security group',
    VpcId=vpc_id
)
security_group_id = security_group['GroupId']
print(f'Created Security Group with ID: {security_group_id}')

# Afegir Regla d'Entrada per permetre SSH
ec2.authorize_security_group_ingress(
    GroupId=security_group_id,
    IpPermissions=[
        {
            'IpProtocol': 'tcp',
            'FromPort': 22,
            'ToPort': 22,
            'IpRanges': [{'CidrIp': '0.0.0.0/0'}]
        }
    ]
)
print('Added inbound rule to allow SSH')

Resum

En aquesta secció, hem cobert els conceptes bàsics de la xarxa d'AWS, incloent-hi VPCs, subxarxes, taules de rutes, passarel·les d'internet, passarel·les NAT i grups de seguretat. També hem realitzat un exemple pràctic de creació d'una VPC amb subxarxes públiques i privades, adjuntant una passarel·la d'internet i configurant grups de seguretat. Entendre aquests conceptes fonamentals és crucial per dissenyar i gestionar entorns d'AWS segurs i eficients.
