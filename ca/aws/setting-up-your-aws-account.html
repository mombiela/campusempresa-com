Introducció

Configurar el teu compte d'AWS és el primer pas per aprofitar el poder d'Amazon Web Services per als teus projectes. Aquesta secció et guiarà a través del procés de creació i configuració del teu compte d'AWS, assegurant que tinguis una base sòlida sobre la qual construir.

Crear un compte d'AWS

Per començar a utilitzar AWS, necessites crear un compte. Segueix aquests passos:

  1. Visita el lloc web d'AWS: Ves a aws.amazon.com.
  2. Fes clic a "Create an AWS Account": Aquest botó normalment es troba a la cantonada superior dreta de la pàgina.
  3. Introdueix la teva adreça de correu electrònic: Proporciona una adreça de correu electrònic vàlida i tria una contrasenya.
  4. Tria un nom per al compte d'AWS: Pot ser el nom de la teva empresa o qualsevol nom que prefereixis.
  5. Completa el CAPTCHA: Verifica que no ets un robot.
  6. Fes clic a "Continue": Procedeix al següent pas.

Informació de facturació

AWS requereix informació de facturació per activar el teu compte. Segueix aquests passos:

  1. Introdueix la teva informació de contacte: Proporciona el teu nom complet, adreça i número de telèfon.
  2. Introdueix la teva informació de pagament: AWS accepta targetes de crèdit i dèbit.
  3. Verifica la teva identitat: AWS enviarà un codi de verificació al teu telèfon.
  4. Tria un pla de suport: Selecciona el pla que millor s'adapti a les teves necessitats. El pla Bàsic és gratuït i suficient per a la majoria de principiants.

Completar la configuració

Després d'introduir la teva informació de facturació i verificar la teva identitat, hauràs de:

  1. Revisar i confirmar: Comprova tota la informació que has introduït.
  2. Acceptar l'Acord de Client d'AWS: Llegeix i accepta els termes.
  3. Fes clic a "Create Account and Continue": El teu compte es crearà i seràs redirigit al Consola de Gestió d'AWS.

Configurar el teu compte d'AWS

Un cop creat el teu compte, necessites configurar-lo per a un ús òptim. Això inclou configurar usuaris IAM, configurar alertes de facturació i assegurar el teu compte.

Configurar usuaris IAM

IAM (Identity and Access Management) et permet gestionar l'accés als teus recursos d'AWS de manera segura. Segueix aquests passos per crear un usuari IAM:

  1. Inicia sessió a la Consola de Gestió d'AWS: Utilitza les teves credencials de compte root.
  2. Navega al Tauler de IAM: Cerca "IAM" a la Consola de Gestió d'AWS.
  3. Crea un nou usuari:
    • Fes clic a "Users" al menú de l'esquerra.
    • Fes clic a "Add user".
    • Introdueix un nom d'usuari i selecciona el tipus d'accés (Accés programàtic, Accés a la Consola de Gestió d'AWS, o ambdós).
  4. Configura els permisos:
    • Tria "Attach existing policies directly".
    • Selecciona les polítiques que coincideixin amb els permisos que vols concedir.
  5. Revisa i crea: Revisa els detalls de l'usuari i fes clic a "Create user".

Configurar alertes de facturació

Per evitar càrrecs inesperats, configura alertes de facturació:

  1. Navega al Tauler de Facturació: Cerca "Billing" a la Consola de Gestió d'AWS.
  2. Activa les alertes de facturació:
    • Fes clic a "Billing preferences".
    • Marca la casella per "Receive Billing Alerts".
    • Fes clic a "Save preferences".
  3. Crea una alarma de facturació:
    • Ves al tauler de CloudWatch.
    • Fes clic a "Alarms" i després a "Create Alarm".
    • Selecciona "Billing" com a mètrica i estableix un llindar per a l'alarma.

Assegurar el teu compte

La seguretat és crucial quan utilitzes AWS. Segueix aquestes millors pràctiques:

  • Activa l'Autenticació Multifactor (MFA): Afegeix una capa extra de seguretat al teu compte root i als usuaris IAM.
  • Utilitza contrasenyes fortes: Assegura't que tots els usuaris tinguin contrasenyes fortes i úniques.
  • Revisa regularment els permisos: Revisa i actualitza periòdicament els permisos dels usuaris per seguir el principi del mínim privilegi.

Conclusió

Configurar el teu compte d'AWS és un procés senzill que implica crear el compte, introduir la informació de facturació i configurar els ajustos essencials. Seguint els passos descrits en aquesta secció, asseguraràs que el teu compte d'AWS estigui segur i llest per a ser utilitzat. Recorda revisar i actualitzar regularment els ajustos del teu compte per mantenir la seguretat i controlar els costos.
