Introducció a l'Autenticació Multi-Factor (MFA)

L'Autenticació Multi-Factor (MFA) és un sistema de seguretat que requereix més d'un mètode d'autenticació per verificar la identitat de l'usuari per a un inici de sessió o una altra transacció. MFA combina dos o més credencials independents: el que l'usuari sap (contrasenya), el que l'usuari té (testimoni de seguretat) i el que l'usuari és (verificació biomètrica).

Conceptes Clau

  • Alguna Cosa que Saps: Normalment una contrasenya o PIN.
  • Alguna Cosa que Tens: Un dispositiu físic com un telèfon intel·ligent o un testimoni de maquinari.
  • Alguna Cosa que Ets: Verificació biomètrica com empremtes digitals o reconeixement facial.

Importància de MFA a AWS

Els entorns AWS sovint contenen dades sensibles i aplicacions crítiques. Implementar MFA afegeix una capa extra de seguretat, fent significativament més difícil que usuaris no autoritzats obtinguin accés.

Beneficis de MFA

  • Seguretat Millorada: Redueix el risc de comptes compromesos.
  • Compliment: Ajuda a complir amb els requisits reguladors.
  • Responsabilitat de l'Usuari: Assegura que només els usuaris autoritzats poden realitzar certes accions.

Configuració de MFA a AWS

Prerequisits

  • Un compte AWS.
  • Accés a la Consola de Gestió d'AWS.
  • Un dispositiu MFA compatible (per exemple, un telèfon intel·ligent amb una aplicació MFA com Google Authenticator o un dispositiu MFA de maquinari).

Passos per Activar MFA per a Usuaris IAM d'AWS

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega al Tauler de Control IAM:
    • Ves al servei IAM des de la Consola de Gestió d'AWS.
  3. Selecciona Usuaris:
    • Tria l'usuari per al qual vols activar MFA.
  4. Gestiona el Dispositiu MFA:
    • A la secció de detalls de l'usuari, fes clic a la pestanya "Credencials de seguretat".
    • Fes clic a "Gestiona" al costat de la secció "Dispositiu MFA assignat".
  5. Tria el Tipus de Dispositiu MFA:
    • Selecciona "Dispositiu MFA virtual" o "Dispositiu MFA de maquinari".
  6. Configura el Dispositiu MFA:
    • Per a un dispositiu MFA virtual, utilitza una aplicació MFA per escanejar el codi QR proporcionat per AWS.
    • Per a un dispositiu MFA de maquinari, introdueix el número de sèrie del dispositiu.
  7. Assigna el Dispositiu MFA:
    • Introdueix els codis d'autenticació generats pel dispositiu MFA per completar la configuració.

Exemple: Activació d'un Dispositiu MFA Virtual

1. Obre la consola IAM a https://console.aws.amazon.com/iam/.
2. Al panell de navegació, tria Usuaris.
3. Tria el nom de l'usuari per al qual vols activar MFA.
4. Tria la pestanya Credencials de seguretat.
5. A la secció Autenticació multi-factor (MFA), tria Gestiona.
6. Tria Dispositiu MFA virtual, i després tria Continua.
7. Obre la teva aplicació MFA al teu telèfon intel·ligent i escaneja el codi QR mostrat a la consola d'AWS.
8. Introdueix els dos codis d'autenticació consecutius generats per la teva aplicació MFA.
9. Tria Assigna MFA.

Configuracions Avançades de MFA

MFA per al Compte Root

  • Altament Recomanat: El compte root té accés il·limitat als teus recursos AWS.
  • Passos: Similar a l'activació de MFA per a usuaris IAM però fet a través de la configuració de seguretat del compte root.

MFA per a AWS CLI i SDK

  • Credencials de Seguretat Temporals: Utilitza AWS STS (Security Token Service) per generar credencials temporals que requereixen MFA.
  • Exemple: 
    aws sts get-session-token --serial-number arn-of-the-mfa-device --token-code code-from-token

Aplicació de MFA amb Polítiques IAM

  • Exemple de Política IAM: 
    {
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Deny",
            "Action": "*",
            "Resource": "*",
            "Condition": {
                "BoolIfExists": {
                    "aws:MultiFactorAuthPresent": "false"
                }
            }
        }
    ]
}

Resolució de Problemes de MFA

  • Dispositiu MFA Perdut: Utilitza el compte root per desactivar MFA per a l'usuari afectat i reconfigura-ho.
  • Problemes de Sincronització: Assegura't que l'hora del teu dispositiu MFA està sincronitzada amb els servidors d'AWS.
  • Codis de Respatller: AWS no proporciona codis de respatller; considera tenir un dispositiu MFA secundari.

Conclusió

L'Autenticació Multi-Factor (MFA) és una mesura de seguretat crítica per protegir el teu entorn AWS. Requerint múltiples formes de verificació, MFA millora significativament la seguretat dels teus recursos AWS. Configurar MFA és senzill i es pot fer a través de la Consola de Gestió d'AWS, CLI o SDK. Les configuracions avançades i les polítiques poden reforçar encara més l'ús de MFA, assegurant pràctiques de seguretat robustes a tota la teva infraestructura AWS.
