Introducció

Benvingut al curs de Desenvolupador Certificat d'AWS! Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat en desenvolupament d'AWS. Al final d'aquest curs, estaràs ben preparat per fer l'examen de Desenvolupador Certificat d'AWS i aplicar les teves habilitats en escenaris del món real.

Serveis bàsics d'AWS

  • Amazon EC2 (Elastic Compute Cloud)
    • Llançant i gestionant instàncies EC2
    • Tipus d'instàncies EC2 i preus
    • Grups de seguretat i parells de claus
    • Exemple: 
      # Llançant una instància EC2 utilitzant AWS CLI
aws ec2 run-instances --image-id ami-0abcdef1234567890 --count 1 --instance-type t2.micro --key-name MyKeyPair --security-groups MySecurityGroup
  • Amazon S3 (Simple Storage Service)
    • Creant i gestionant cubs S3
    • Classes d'emmagatzematge S3 i polítiques de cicle de vida
    • Exemple: 
      import boto3

# Crear un client S3
s3 = boto3.client('s3')

# Crear un nou cub
s3.create_bucket(Bucket='my-new-bucket')
  • Amazon RDS (Relational Database Service)
    • Configurant i gestionant instàncies RDS
    • Tipus d'instàncies RDS i preus
    • Exemple: 
      # Creant una instància RDS utilitzant AWS CLI
aws rds create-db-instance --db-instance-identifier mydbinstance --db-instance-class db.t2.micro --engine mysql --master-username admin --master-user-password password --allocated-storage 20

Serveis avançats d'AWS

  • AWS Lambda
    • Introducció a la computació sense servidor
    • Creant i desplegant funcions Lambda
    • Exemple: 
      import json

def lambda_handler(event, context):
    return {
        'statusCode': 200,
        'body': json.dumps('Hello from Lambda!')
    }
  • Amazon DynamoDB
    • Creant i gestionant taules DynamoDB
    • Modelatge de dades i indexació a DynamoDB
    • Exemple: 
      import boto3

# Crear un client DynamoDB
dynamodb = boto3.client('dynamodb')

# Crear una nova taula
dynamodb.create_table(
    TableName='MyTable',
    KeySchema=[
        {
            'AttributeName': 'ID',
            'KeyType': 'HASH'
        }
    ],
    AttributeDefinitions=[
        {
            'AttributeName': 'ID',
            'AttributeType': 'S'
        }
    ],
    ProvisionedThroughput={
        'ReadCapacityUnits': 5,
        'WriteCapacityUnits': 5
    }
)
