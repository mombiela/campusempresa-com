Configuració d'Amazon Redshift

  • Prerequisits
    • Compte AWS
    • Rols i Permisos IAM
  • Creació d'un Clúster Redshift
    • Guia Pas a Pas
    • Opcions de Configuració
    • Exemple: 
      import boto3

redshift = boto3.client('redshift')

response = redshift.create_cluster(
    ClusterIdentifier='my-redshift-cluster',
    NodeType='dc2.large',
    MasterUsername='awsuser',
    MasterUserPassword='YourPassword123',
    ClusterType='multi-node',
    NumberOfNodes=2
)

print(response)
  • Connexió al Clúster Redshift
    • Utilitzant Clients SQL (p. ex., SQL Workbench/J)
    • Exemple de Cadena de Connexió: 
      jdbc:redshift://<cluster-endpoint>:5439/dev

Consulta de Dades a Amazon Redshift

  • Consultes SQL Bàsiques
    • SELECT, INSERT, UPDATE, DELETE
    • Exemple: 
      SELECT * FROM my_table WHERE column1 = 'value';
  • Consultes SQL Avançades
    • Joins, Agregacions, Funcions de Finestra
    • Exemple: 
      SELECT column1, COUNT(*)
FROM my_table
GROUP BY column1
HAVING COUNT(*) > 1;
