Concurrència i Paral·lelisme en F#
Proves i Depuració ►

Introducció

La concurrència i el paral·lelisme són conceptes essencials en la programació moderna, especialment per a aplicacions que requereixen alt rendiment i capacitat de resposta. En aquesta secció, explorarem com F# maneja la concurrència i el paral·lelisme, proporcionant-te les eines per escriure codi eficient i escalable.

Conceptes Clau

  • Concurrència: La capacitat de gestionar múltiples tasques al mateix temps. Implica gestionar múltiples tasques que poden córrer de manera independent però no necessàriament simultània.
  • Paral·lelisme: La capacitat d'executar múltiples tasques simultàniament, típicament per millorar el rendiment utilitzant múltiples processadors o nuclis.

Concurrència en F#

F# proporciona diverses maneres de gestionar la concurrència, incloent-hi fluxos de treball asíncrons i el mòdul Async.

Fluxos de Treball Asíncrons

Els fluxos de treball asíncrons en F# et permeten escriure codi no bloquejant que pot realitzar operacions d'E/S sense congelar el fil principal.

Exemple: Flux de Treball Asíncron Bàsic

open System.Net

let fetchUrlAsync (url: string) =
    async {
        let request = WebRequest.Create(url)
        use! response = request.AsyncGetResponse()
        use stream = response.GetResponseStream()
        use reader = new System.IO.StreamReader(stream)
        let html = reader.ReadToEnd()
        return html
    }

let url = "http://example.com"
let html = fetchUrlAsync url |> Async.RunSynchronously
printfn "%s" html
  • async { ... }: Defineix un flux de treball asíncron.
  • use!: Espera el resultat d'una operació asíncrona.

El Mòdul `Async`

El mòdul Async proporciona diverses funcions per treballar amb fluxos de treball asíncrons.

Exemple: Fluxos de Treball Asíncrons Paral·lels

let urls = ["http://example.com"; "http://example.org"; "http://example.net"]

let fetchAllUrlsAsync urls =
    urls
    |> List.map fetchUrlAsync
    |> Async.Parallel
    |> Async.RunSynchronously

let results = fetchAllUrlsAsync urls
results |> Array.iter (printfn "%s")
  • Async.Parallel: Executa múltiples fluxos de treball asíncrons en paral·lel.
  • Async.RunSynchronously: Executa un flux de treball asíncron de manera síncrona.

Paral·lelisme en F#

F# aprofita la Task Parallel Library (TPL) de .NET per al paral·lelisme, permetent-te realitzar càlculs en paral·lel.

Càlculs Paral·lels

El mòdul Parallel en F# proporciona funcions per realitzar càlculs paral·lels.

Exemple: Processament Paral·lel d'Arrays

open System.Threading.Tasks

let numbers = [| 1 .. 1000000 |]

let sumOfSquaresParallel (nums: int[]) =
    nums
    |> Array.Parallel.map (fun x -> x * x)
    |> Array.sum

let result = sumOfSquaresParallel numbers
printfn "Sum of squares: %d" result
  • Array.Parallel.map: Aplica una funció a cada element de l'array en paral·lel.
  • Array.sum: Suma els elements de l'array.

Task Parallel Library (TPL)

La TPL proporciona més control sobre les tasques paral·leles, incloent-hi la creació de tasques, continuació i cancel·lació.

Exemple: Utilitzant Tasques

open System.Threading.Tasks

let task1 = Task.Run(fun () -> 
    // Simula una mica de treball
    System.Threading.Thread.Sleep(1000)
    42)

let task2 = Task.Run(fun () -> 
    // Simula una mica de treball
    System.Threading.Thread.Sleep(500)
    24)

let combinedTask = Task.WhenAll(task1, task2)

combinedTask.ContinueWith(fun t ->
    let results = t.Result
    printfn "Results: %A" results
) |> ignore

// Espera que la tasca combinada es completi
combinedTask.Wait()
  • Task.Run: Crea i inicia una nova tasca.
  • Task.WhenAll: Crea una tasca que es completarà quan totes les tasques subministrades hagin completat.
  • ContinueWith: Especifica una acció a realitzar quan la tasca es completa.

Resum

En aquesta secció, hem explorat els conceptes de concurrència i paral·lelisme en F#. Hem cobert:

  • Fluxos de treball asíncrons utilitzant la paraula clau async i el mòdul Async.
  • Càlculs paral·lels utilitzant el mòdul Parallel i la Task Parallel Library (TPL).

Entendre aquests conceptes i eines t'ajudarà a escriure aplicacions eficients i responsives en F#. Practica amb els exemples proporcionats i intenta implementar les teves pròpies solucions concurrents i paral·leles per reforçar el teu aprenentatge.
