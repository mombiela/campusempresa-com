Benvingut al teu viatge en la programació amb F#! En aquesta secció, cobrirem els conceptes bàsics per escriure el teu primer programa en F#: l'exemple clàssic "Hola, Món!". Això t'ajudarà a familiaritzar-te amb la sintaxi i l'estructura de F#.

Introducció a F#

F# és un llenguatge de programació funcional-primer que també suporta programació orientada a objectes i imperativa. Forma part de l'ecosistema .NET i és conegut per la seva sintaxi concisa i la seva potent inferència de tipus.

Configurant el Teu Entorn

Abans d'escriure el nostre primer programa en F#, necessitem configurar el nostre entorn de desenvolupament.

Instal·lant el SDK de .NET

  1. Descarrega i instal·la el SDK de .NET des del lloc web oficial de .NET.
  2. Verifica la instal·lació obrint una terminal o símbol del sistema i escrivint: 
    dotnet --version
    Hauries de veure el número de versió del SDK instal·lat.

Configurant un Projecte F#

  1. Obre una terminal o símbol del sistema.
  2. Crea una nova aplicació de consola executant: 
    dotnet new console -lang "F#" -o HelloWorld
  3. Navega al directori del projecte: 
    cd HelloWorld

Escriure el Teu Primer Programa en F#

Ara que tenim el nostre projecte configurat, escrivim el programa "Hola, Món!".

Entenent l'Estructura del Programa

Obre el fitxer Program.fs en el teu editor de text o IDE preferit. Veureu algun codi boilerplate que sembla així:

open System

[<EntryPoint>]
let main argv =
    printfn "Hola, Món!"
    0 // retorna un codi de sortida enter

Explicació del Codi

  • open System: Aquesta línia importa l'espai de noms System, que conté classes fonamentals i classes base que defineixen tipus de dades de valor i referència comunament utilitzats.
  • [<EntryPoint>]: Aquest atribut marca la funció main com el punt d'entrada del programa.
  • let main argv =: Això defineix la funció main, que pren una matriu de cadenes (argv) com a arguments.
  • printfn "Hola, Món!": Aquesta línia imprimeix "Hola, Món!" a la consola. printfn és una funció que formata i imprimeix una cadena seguida d'una nova línia.
  • 0: Aquest és el valor de retorn de la funció main, que és un codi de sortida enter. 0 típicament significa que el programa s'ha executat correctament.

Executant el Programa

Per executar el teu programa, utilitza la següent comanda en la teva terminal o símbol del sistema:

dotnet run

Hauries de veure la sortida:

Hola, Món!

Resum

En aquesta secció, hem cobert el següent:

  • Configurar el SDK de .NET i crear un nou projecte F#.
  • Escriure i entendre un simple programa "Hola, Món!" en F#.
  • Executar el programa per veure la sortida.

Felicitats! Has fet el teu primer pas en la programació amb F#. En les següents seccions, aprofundirem en la sintaxi de F#, els tipus de dades i els conceptes de programació funcional.
