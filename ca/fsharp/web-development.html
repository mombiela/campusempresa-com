Introducció al Desenvolupament Web amb F#

En aquesta secció, introduirem els conceptes bàsics del desenvolupament web utilitzant F#. Cobrirem els conceptes i eines fonamentals necessaris per començar.

  • Què és F#?

    • Un llenguatge de programació funcional-primer que també suporta programació orientada a objectes i imperativa.
    • Part de l'ecosistema .NET.

  • Per què utilitzar F# per al Desenvolupament Web?

    • Sistema de tipus fort i immutabilitat per defecte.
    • Sintaxi concisa i expressiva.
    • Interoperabilitat amb altres llenguatges i biblioteques .NET.

Configurant el Teu Entorn

Abans d'endinsar-nos en el desenvolupament web, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lant .NET SDK

    • Descarrega i instal·la el .NET SDK des del lloc web oficial de Microsoft.

  • Configurant un projecte F#

    dotnet new console -lang F# -o MyWebApp
cd MyWebApp
dotnet run

  • Instal·lant eines necessàries

    • Visual Studio Code o Visual Studio.
    • Extensió Ionide per a suport F# en Visual Studio Code.

Construint un Servidor Web Simple

Començarem construint un servidor web simple utilitzant Suave, una biblioteca de servidor web lleugera per a F#.

  • Instal·lant Suave

    dotnet add package Suave

  • Creant un servidor web bàsic

    open Suave
open Suave.Successful

let app = OK "Hello, World!"

startWebServer defaultConfig app

  • Executant el servidor

    dotnet run

Gestionant Rutes i Sol·licituds

En aquesta secció, aprendrem a gestionar diferents rutes i sol·licituds en la nostra aplicació web.

  • Definint rutes

    open Suave
open Suave.Operators
open Suave.Filters
open Suave.Successful

let app =
  choose [
    GET >=> path "/" >=> OK "Home Page"
    GET >=> path "/about" >=> OK "About Page"
  ]

startWebServer defaultConfig app

  • Gestionant paràmetres de consulta

    open Suave
open Suave.Filters
open Suave.Operators
open Suave.Successful

let app =
  choose [
    GET >=> pathScan "/hello/%s" (fun name -> OK (sprintf "Hello, %s!" name))
  ]

startWebServer defaultConfig app

Treballant amb Plantilles

Les plantilles són essencials per renderitzar contingut HTML dinàmic. Utilitzarem la biblioteca Giraffe per a aquest propòsit.

  • Instal·lant Giraffe

    dotnet add package Giraffe

  • Creant una plantilla simple

    open Giraffe
open Giraffe.ViewEngine

let layout (content: XmlNode list) =
  html [] [
    head [] [ title [] [ str "My Web App" ] ]
    body [] content
  ]

let indexView =
  layout [
    h1 [] [ str "Welcome to My Web App" ]
    p [] [ str "This is a simple web application using F# and Giraffe." ]
  ]

let app = choose [ GET >=> route "/" >=> htmlView indexView ]

startWebServer defaultConfig app

Interactuant amb Bases de Dades

En aquesta secció, cobrirem com interactuar amb bases de dades utilitzant F#.

  • Instal·lant SQLProvider

    dotnet add package SQLProvider

  • Connectant-se a una base de dades

    open FSharp.Data.Sql

type sql = SqlDataProvider< 
              Common.DatabaseProviderTypes.MSSQLSERVER, 
              "Server=myServerAddress;Database=myDataBase;User Id=myUsername;Password=myPassword;" >

let ctx = sql.GetDataContext()

let customers = ctx.Dbo.Customers |> Seq.toList

  • Realitzant operacions CRUD

    // Crear
let newCustomer = ctx.Dbo.Customers.Create()
newCustomer.Name <- "John Doe"
ctx.SubmitUpdates()

// Llegir
let customer = ctx.Dbo.Customers |> Seq.head

// Actualitzar
customer.Name <- "Jane Doe"
ctx.SubmitUpdates()

// Esborrar
ctx.Dbo.Customers.DeleteOnSubmit(customer)
ctx.SubmitUpdates()

Temes Avançats

En aquesta secció, cobrirem temes més avançats com l'autenticació, middleware i desplegament d'aplicacions web F#.

  • Autenticació i Autorització

    • Utilitzant OAuth o JWT per a una autenticació segura.
    • Implementant control d'accés basat en rols.

  • Middleware

    • Creant middleware personalitzat per a registre, gestió d'errors, etc.

  • Desplegant Aplicacions Web F#

    • Desplegant a Azure, AWS o altres plataformes de núvol.
    • Contenint la teva aplicació utilitzant Docker.

Conclusió

En aquest curs, hem cobert els aspectes essencials del desenvolupament web amb F#. Hem començat configurant l'entorn, construint un servidor web simple, gestionant rutes, treballant amb plantilles, interactuant amb bases de dades i, finalment, temes avançats com l'autenticació i el desplegament. Amb aquestes habilitats, ara estàs equipat per construir aplicacions web robustes utilitzant F#. Feliç programació!
