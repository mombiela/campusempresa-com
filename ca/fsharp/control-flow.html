Introducció

El flux de control és un concepte fonamental en la programació que et permet dictar l'ordre en què s'executen les instruccions. En F#, el flux de control es gestiona utilitzant condicionals i bucles. Aquesta secció cobrirà els conceptes bàsics d'aquests constructes, proporcionant exemples i exercicis per ajudar-te a entendre'ls i aplicar-los de manera efectiva.

Condicionals

Instruccions If-Else

La instrucció if-else s'utilitza per executar codi basat en una condició.

Sintaxi

if condition then
    // codi a executar si la condició és certa
else
    // codi a executar si la condició és falsa

Exemple

let number = 10

if number > 0 then
    printfn "El número és positiu."
else
    printfn "El número és no positiu."

Instruccions If-Else Imbricades

Pots imbricar instruccions if-else per gestionar múltiples condicions.

Exemple

let number = 0

if number > 0 then
    printfn "El número és positiu."
elif number < 0 then
    printfn "El número és negatiu."
else
    printfn "El número és zero."

Coincidència de Patrons

La coincidència de patrons és una característica poderosa en F# que et permet coincidir valors amb patrons i executar codi basat en la coincidència.

Sintaxi

match expression with
| pattern1 -> // codi a executar si pattern1 coincideix
| pattern2 -> // codi a executar si pattern2 coincideix
| _ -> // codi a executar si cap patró coincideix

Exemple

let number = 10

match number with
| n when n > 0 -> printfn "El número és positiu."
| n when n < 0 -> printfn "El número és negatiu."
| _ -> printfn "El número és zero."

Bucles

Bucles For

El bucle for s'utilitza per iterar sobre un rang de valors.

Sintaxi

for variable in start .. end do
    // codi a executar per a cada iteració

Exemple

for i in 1 .. 5 do
    printfn "Iteració %d" i

Bucles While

El bucle while s'utilitza per executar codi mentre una condició sigui certa.

Sintaxi

while condition do
    // codi a executar mentre la condició sigui certa

Exemple

let mutable count = 5

while count > 0 do
    printfn "El compte és %d" count
    count <- count - 1

Funcions Recursives

En F#, els bucles sovint poden ser reemplaçats per funcions recursives, que són funcions que es criden a si mateixes.

Exemple

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "El factorial de 5 és %d" (factorial 5)

Exercicis

Exercici 1: If-Else Bàsic

Escriu una funció que prengui un enter i retorni una cadena indicant si el número és positiu, negatiu o zero.

Exercici 2: Coincidència de Patrons

Escriu una funció que prengui un dia de la setmana (com a cadena) i retorni si és un dia laborable o un cap de setmana.

Exercici 3: Bucle For

Escriu una funció que imprimeixi els primers 10 números de la seqüència de Fibonacci utilitzant un bucle for.

Exercici 4: Bucle While

Escriu una funció que prengui un enter i imprimeixi tots els números parells des de 0 fins a aquest enter utilitzant un bucle while.

Exercici 5: Funció Recursiva

Escriu una funció recursiva per calcular el màxim comú divisor (MCD) de dos números.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics del flux de control en F#, incloent condicionals i bucles. Hem explorat les instruccions if-else, la coincidència de patrons, els bucles for, els bucles while i les funcions recursives. Entendre aquests constructes és crucial per escriure codi F# eficient i efectiu. Assegura't de practicar els exercicis per consolidar la teva comprensió.
