Introducció

Les funcions i els mètodes són blocs fonamentals en F#. Permeten encapsular la lògica, promoure la reutilització del codi i millorar la llegibilitat. Aquesta secció cobrirà els conceptes bàsics de definir i utilitzar funcions, així com temes més avançats com funcions d'ordre superior, recursió i mètodes en programació orientada a objectes.

Funcions Bàsiques

Les funcions en F# són ciutadans de primera classe, és a dir, poden ser passades com arguments, retornades d'altres funcions i assignades a variables.

Definició d'una Funció

Una funció simple es pot definir utilitzant la paraula clau let.

let add x y = x + y
  • let: Paraula clau utilitzada per definir una funció.
  • add: Nom de la funció.
  • x i y: Paràmetres de la funció.
  • x + y: Cos de la funció.

Crida a una Funció

Pots cridar una funció utilitzant el seu nom seguit dels arguments.

let result = add 3 5
printfn "El resultat és %d" result

Funció amb Múltiples Línies

Si una funció té múltiples línies, utilitza la indentació per definir l'abast.

let greet name =
    let message = "Hola, " + name
    printfn "%s" message

greet "Alice"

Funcions d'Ordre Superior

Les funcions d'ordre superior són funcions que prenen altres funcions com a paràmetres o retornen funcions com a resultats.

Exemple: Funció `map`

La funció List.map aplica una funció donada a cada element d'una llista.

let square x = x * x
let numbers = [1; 2; 3; 4]
let squaredNumbers = List.map square numbers
printfn "%A" squaredNumbers

Recursió

La recursió és una tècnica on una funció es crida a si mateixa per resoldre un problema.

Exemple: Funció Factorial

Un exemple clàssic de recursió és la funció factorial.

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "El factorial de 5 és %d" (factorial 5)
  • rec: Paraula clau utilitzada per definir una funció recursiva.
  • factorial: Nom de la funció.
  • n: Paràmetre de la funció.
  • if n <= 1 then 1: Cas base.
  • else n * factorial (n - 1): Cas recursiu.

Mètodes en Programació Orientada a Objectes

F# suporta la programació orientada a objectes, i els mètodes són funcions definides dins d'una classe o objecte.

Definició d'una Classe amb Mètodes

Aquí tens un exemple d'una classe simple amb mètodes.

type Person(name: string, age: int) =
    member this.Name = name
    member this.Age = age
    member this.Greet() = printfn "Hola, em dic %s i tinc %d anys." this.Name this.Age

let alice = Person("Alice", 30)
alice.Greet()
  • type: Paraula clau utilitzada per definir una classe.
  • Person: Nom de la classe.
  • name i age: Paràmetres del constructor.
  • member: Paraula clau utilitzada per definir un mètode o propietat.
  • this: Referència a la instància actual de la classe.

Resum

En aquesta secció, hem cobert els conceptes bàsics de funcions i mètodes en F#. Hem començat amb definicions i crides de funcions simples, hem passat a funcions d'ordre superior i recursió, i hem conclòs amb mètodes en programació orientada a objectes. Entendre aquests conceptes és crucial per escriure codi eficient i reutilitzable en F#. Practica aquests conceptes amb diversos exemples per consolidar el teu enteniment.
