Llistes i Arrays en F#

Introducció

En F#, les llistes i els arrays són estructures de dades fonamentals que et permeten emmagatzemar col·leccions d'elements. Entendre com utilitzar aquestes estructures de manera efectiva és crucial per a qualsevol programador de F#. Aquesta secció cobrirà els conceptes bàsics de les llistes i els arrays, les seves diferències i com manipular-los.

Llistes

Les llistes en F# són col·leccions immutables i ordenades d'elements del mateix tipus. Es defineixen utilitzant claudàtors i punts i coma per separar els elements.

Creació de Llistes

Per crear una llista, pots utilitzar la següent sintaxi:

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Operacions Comunes amb Llistes

  • Accedir a Elements: Utilitza la funció List.item o la sintaxi .[index].
  • Afegir Elements: Utilitza l'operador @ per concatenar llistes.
  • Mapeig: Utilitza la funció List.map per aplicar una funció a cada element.
let firstElement = List.item 0 numbers
let secondElement = numbers.[1]
let combinedList = numbers @ [6; 7; 8]
let squaredNumbers = List.map (fun x -> x * x) numbers

Patró de Coincidència amb Llistes

El patró de coincidència és una característica poderosa en F# que et permet descompondre llistes fàcilment.

let rec sumList lst =
    match lst with
    | [] -> 0
    | head :: tail -> head + sumList tail

let total = sumList numbers

Arrays

Els arrays en F# són col·leccions mutables i de mida fixa d'elements del mateix tipus. Es defineixen utilitzant claudàtors i barres verticals.

Creació d'Arrays

Per crear un array, pots utilitzar la següent sintaxi:

let numbersArray = [| 1; 2; 3; 4; 5 |]
let fruitsArray = [| "apple"; "banana"; "cherry" |]

Operacions Comunes amb Arrays

  • Accedir a Elements: Utilitza la sintaxi .[index].
  • Modificar Elements: Assigna directament un nou valor a un índex.
  • Mapeig: Utilitza la funció Array.map per aplicar una funció a cada element.
let firstElementArray = numbersArray.[0]
numbersArray.[1] <- 10
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray

Array vs Llista

| Característica | Llista (Immutable) | Array (Mutable) | |-----------------|--------------------|-----------------| | Mutabilitat | Immutable | Mutable | | Mida | Dinàmica | Fixa | | Rendiment | Més lent per a actualitzacions | Més ràpid per a actualitzacions | | Sintaxi | [1; 2; 3] | [| 1; 2; 3 |] |

Temes Avançats

Funcions d'Ordre Superior

Tant les llistes com els arrays suporten funcions d'ordre superior com map, filter i fold.

let evenNumbers = List.filter (fun x -> x % 2 = 0) numbers
let sumOfArray = Array.fold (+) 0 numbersArray

Processament Paral·lel amb Arrays

Els arrays es poden processar en paral·lel utilitzant el mòdul Array.Parallel per a millores de rendiment.

let squaredNumbersParallel = Array.Parallel.map (fun x -> x * x) numbersArray

Conclusió

Les llistes i els arrays són estructures de dades essencials en F#. Les llistes són immutables i de mida dinàmica, cosa que les fa adequades per a paradigmes de programació funcional. Els arrays són mutables i de mida fixa, oferint un millor rendiment per a certes operacions. Entendre quan i com utilitzar aquestes estructures millorarà enormement la teva capacitat per escriure codi F# eficient i efectiu.
