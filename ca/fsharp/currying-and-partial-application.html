Introducció

El currying i l'aplicació parcial són tècniques poderoses de programació funcional que et permeten crear noves funcions fixant alguns arguments de funcions existents. Aquests conceptes són particularment útils en F#, un llenguatge que abraça els paradigmes de programació funcional.

Currying

El currying és el procés de transformar una funció que pren múltiples arguments en una seqüència de funcions, cadascuna prenent un sol argument.

Conceptes Clau

  • Funció Curried: Una funció que retorna una altra funció fins que es proporcionen tots els arguments.
  • Funció No Curried: Una funció que pren tots els seus arguments alhora.

Exemple

Considera una funció simple que suma dos números:

let add x y = x + y

En la seva forma curried, add es pot veure com una funció que pren un argument x i retorna una altra funció que pren un argument y:

let add x = (fun y -> x + y)

Utilitzant la funció curried:

let add5 = add 5
let result = add5 10  // result és 15

Aplicació Parcial

L'aplicació parcial és el procés de fixar alguns arguments d'una funció, produint una altra funció de menor aritat (menys arguments).

Conceptes Clau

  • Aplicació Parcial: Aplicar una funció a alguns dels seus arguments, produint una altra funció que pren els arguments restants.
  • Aritat: El nombre d'arguments que pren una funció.

Exemple

Utilitzant la mateixa funció add:

let add x y = x + y

Podem aplicar-la parcialment:

let add5 = add 5
let result = add5 10  // result és 15

Aquí, add5 és una funció parcialment aplicada on el primer argument està fixat a 5.

Diferències Entre Currying i Aplicació Parcial

| Concepte | Descripció | Exemple | |--------------------|-----------------------------------------------------------------------------|----------------------------------------| | Currying | Transformar una funció en una seqüència de funcions, cadascuna prenent un argument | let add x = (fun y -> x + y) | | Aplicació Parcial | Fixar alguns arguments d'una funció, produint una altra funció amb menys arguments | let add5 = add 5 |

Aplicacions Pràctiques

Composició de Funcions

El currying i l'aplicació parcial sovint s'utilitzen en la composició de funcions, on combines funcions simples per construir-ne de més complexes.

Reutilització de Codi

Descomponent funcions en components més petits i reutilitzables, pots crear codi més modular i mantenible.

Funcions d'Ordre Superior

El currying i l'aplicació parcial són essencials quan treballes amb funcions d'ordre superior, que són funcions que prenen altres funcions com a arguments o les retornen com a resultats.

Exercicis

  1. Exercici de Currying:

    • Defineix una funció curried multiply que multiplica tres números.
    • Utilitza la funció curried per crear una nova funció multiplyBy2 que multiplica qualsevol número per 2.
    let multiply x y z = x * y * z
let multiplyBy2 = multiply 2
let result = multiplyBy2 3 4  // result és 24

  2. Exercici d'Aplicació Parcial:

    • Defineix una funció subtract que resta dos números.
    • Crea una funció parcialment aplicada subtract5 que resta 5 de qualsevol número.
    let subtract x y = x - y
let subtract5 = subtract 5
let result = subtract5 10  // result és -5

Conclusió

El currying i l'aplicació parcial són conceptes fonamentals en la programació funcional que milloren la flexibilitat i la reutilització del codi. En dominar aquestes tècniques, pots escriure codi F# més concís i expressiu, fent que els teus programes siguin més fàcils d'entendre i mantenir.
