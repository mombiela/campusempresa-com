Processament de JSON i XML en F#
Accés a Bases de Dades ►

Introducció

En aquesta secció, explorarem com gestionar dades JSON i XML en F#. JSON (JavaScript Object Notation) i XML (eXtensible Markup Language) són formats àmpliament utilitzats per a l'intercanvi de dades. Entendre com analitzar, manipular i generar aquests formats és crucial per a moltes tasques de programació, incloent-hi el desenvolupament web, l'anàlisi de dades i la integració d'API.

Processament de JSON

Conceptes Clau

  • Estructura JSON: JSON és un format lleuger d'intercanvi de dades que és fàcil de llegir i escriure per als humans, i fàcil de parsejar i generar per a les màquines. Es basa en dues estructures:
    • Una col·lecció de parells nom/valor (sovint anomenada objecte).
    • Una llista ordenada de valors (sovint anomenada array).
  • Biblioteques F# per a JSON: F# proporciona diverses biblioteques per treballar amb JSON, incloent Newtonsoft.Json (Json.NET) i FSharp.Data.

Parsejar JSON

Per parsejar JSON en F#, pots utilitzar la biblioteca FSharp.Data, que proporciona un proveïdor de tipus per a JSON.

Exemple

#r "nuget: FSharp.Data"
open FSharp.Data

let json = """{ "name": "John", "age": 30, "isStudent": false }"""
let parsedJson = JsonValue.Parse(json)

let name = parsedJson?name.AsString()
let age = parsedJson?age.AsInteger()
let isStudent = parsedJson?isStudent.AsBoolean()

printfn "Name: %s, Age: %d, Is Student: %b" name age isStudent

Explicació:

  • Referenciem la biblioteca FSharp.Data.
  • Definim una cadena JSON i la parsegem utilitzant JsonValue.Parse.
  • Extraiem valors utilitzant l'operador ? i els mètodes de conversió de tipus apropiats (AsString, AsInteger, AsBoolean).

Generar JSON

Generar JSON implica crear un objecte JSON i convertir-lo a una cadena.

Exemple

open FSharp.Data.JsonExtensions

let jsonObject = JsonValue.Record [|
    "name", JsonValue.String "John"
    "age", JsonValue.Number 30M
    "isStudent", JsonValue.Boolean false
|]

let jsonString = jsonObject.ToString()
printfn "Generated JSON: %s" jsonString

Explicació:

  • Utilitzem JsonValue.Record per crear un objecte JSON.
  • Convertim l'objecte JSON a una cadena utilitzant ToString.

Processament de XML

Conceptes Clau

  • Estructura XML: XML és un llenguatge de marques que defineix un conjunt de regles per codificar documents en un format que és llegible tant per a humans com per a màquines. Utilitza una estructura d'arbre amb elements, atributs i text.
  • Biblioteques F# per a XML: F# proporciona l'espai de noms System.Xml i la biblioteca FSharp.Data per treballar amb XML.

Parsejar XML

Per parsejar XML en F#, pots utilitzar l'espai de noms System.Xml.Linq.

Exemple

open System.Xml.Linq

let xml = "<person><name>John</name><age>30</age><isStudent>false</isStudent></person>"
let parsedXml = XDocument.Parse(xml)

let name = parsedXml.Root.Element(XName.Get("name")).Value
let age = parsedXml.Root.Element(XName.Get("age")).Value
let isStudent = parsedXml.Root.Element(XName.Get("isStudent")).Value

printfn "Name: %s, Age: %s, Is Student: %s" name age isStudent

Explicació:

  • Utilitzem XDocument.Parse per parsejar la cadena XML.
  • Naveguem per l'arbre XML utilitzant Root.Element i extraiem valors utilitzant Value.

Generar XML

Generar XML implica crear un arbre XML i convertir-lo a una cadena.

Exemple

let xmlTree = 
    XElement(XName.Get("person"),
        XElement(XName.Get("name"), "John"),
        XElement(XName.Get("age"), "30"),
        XElement(XName.Get("isStudent"), "false")
    )

let xmlString = xmlTree.ToString()
printfn "Generated XML: %s" xmlString

Explicació:

  • Utilitzem XElement per crear un arbre XML.
  • Convertim l'arbre XML a una cadena utilitzant ToString.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics del processament de JSON i XML en F#. Hem après com parsejar i generar JSON utilitzant la biblioteca FSharp.Data i com parsejar i generar XML utilitzant l'espai de noms System.Xml.Linq. Dominar aquestes habilitats et permetrà gestionar formats d'intercanvi de dades de manera efectiva en les teves aplicacions F#.
