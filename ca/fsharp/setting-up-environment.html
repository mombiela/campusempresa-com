Configuració de l'Entorn de Desenvolupament
Primers Passos: Hola Món ►

Configurar el teu entorn de desenvolupament és el primer pas per començar a programar en F#. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per escriure, executar i depurar codi F# de manera eficient.

Instal·lació del .NET SDK

El .NET SDK (Software Development Kit) és necessari per desenvolupar aplicacions amb F#. Inclou el runtime, les biblioteques i les eines necessàries per construir i executar aplicacions F#.

  1. Descarregar el .NET SDK:

    • Visita la pàgina de descàrrega de .NET.
    • Tria la versió més recent del .NET SDK i descarrega l'instal·lador per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·lar el .NET SDK:

    • Executa l'instal·lador i segueix les instruccions en pantalla per completar la instal·lació.

  3. Verificar la instal·lació:

    • Obre un terminal o símbol del sistema.
    • Escriu la següent comanda per comprovar si el .NET SDK està instal·lat correctament: 
      dotnet --version
    • Hauries de veure el número de versió del .NET SDK que has instal·lat.

Configuració d'un IDE

Un Entorn de Desenvolupament Integrat (IDE) proporciona un entorn complet per escriure, depurar i executar el teu codi F#. Aquí tens alguns IDEs populars per al desenvolupament en F#:

Visual Studio Code

Visual Studio Code (VS Code) és un editor lleuger i de codi obert que suporta F# a través d'extensions.

  1. Descarregar i instal·lar Visual Studio Code:

    • Visita la pàgina de descàrrega de Visual Studio Code.
    • Descarrega l'instal·lador per al teu sistema operatiu i segueix les instruccions d'instal·lació.

  2. Instal·lar l'extensió Ionide:

    • Obre Visual Studio Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca "Ionide-fsharp" i fes clic a "Install" per afegir suport per a F#.

  3. Verificar la configuració:

    • Crea un nou fitxer amb l'extensió .fs.
    • Escriu un programa F# senzill: 
      printfn "Hello, F#"
    • Desa el fitxer i executa'l utilitzant el terminal a VS Code: 
      dotnet fsi yourfile.fs

Visual Studio (Windows i macOS)

Visual Studio és un IDE complet que proporciona suport extensiu per al desenvolupament en F#.

  1. Descarregar i instal·lar Visual Studio:

    • Visita la pàgina de descàrrega de Visual Studio.
    • Tria l'edició Community, Professional o Enterprise segons les teves necessitats i descarrega l'instal·lador.

  2. Instal·lar suport per a F#:

    • Durant la instal·lació, selecciona la càrrega de treball ".NET desktop development".
    • Assegura't que "F# desktop language support" estigui marcat.

  3. Crear un nou projecte F#:

    • Obre Visual Studio.
    • Ves a File > New > Project.
    • Selecciona "F# Console App" i segueix les indicacions per crear el teu projecte.

  4. Escriure i executar el teu primer programa F#:

    • Substitueix el codi per defecte per: 
      printfn "Hello, F#"
    • Prem F5 per executar el programa.

Configuració de la Línia de Comandes

Per a aquells que prefereixen utilitzar la línia de comandes, el .NET CLI (Command Line Interface) proporciona eines potents per crear, construir i executar projectes F#.

  1. Crear un nou projecte F#:

    • Obre un terminal o símbol del sistema.
    • Executa la següent comanda per crear una nova aplicació de consola: 
      dotnet new console -lang F# -o MyFSharpApp
    • Navega al directori del projecte: 
      cd MyFSharpApp

  2. Escriure el teu codi F#:

    • Obre el fitxer Program.fs en el teu editor de text preferit.
    • Substitueix el codi per defecte per: 
      printfn "Hello, F#"

  3. Construir i executar el projecte:

    • Construeix el projecte utilitzant: 
      dotnet build
    • Executa el projecte utilitzant: 
      dotnet run

Conclusió

Configurar l'entorn de desenvolupament per a F# implica instal·lar el .NET SDK, triar un IDE o editor de text, i configurar el teu sistema per escriure i executar codi F#. Tant si prefereixes un IDE complet com Visual Studio, un editor lleuger com Visual Studio Code, o la línia de comandes, ara tens les eines i el coneixement per començar el teu viatge amb F#. Feliç programació!
