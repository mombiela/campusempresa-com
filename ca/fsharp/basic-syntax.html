Introducció

F# és un llenguatge de programació funcional-primer que també suporta programació orientada a objectes i imperativa. Entendre la sintaxi i l'estructura bàsica de F# és crucial per escriure codi eficient i llegible. Aquesta secció cobrirà els elements fonamentals de la sintaxi i l'estructura de F#, proporcionant una base sòlida per a un aprenentatge posterior.

Conceptes Clau

Comentaris

  • Comentaris d'una sola línia: Utilitza // per comentar una sola línia.
  • Comentaris de múltiples línies: Utilitza (* ... *) per comentar múltiples línies.
// Aquest és un comentari d'una sola línia

(* Aquest és un 
   comentari de múltiples línies *)

Variables i Valors

  • Valors immutables: Definits utilitzant la paraula clau let.
  • Valors mutables: Definits utilitzant la paraula clau let mutable.
let x = 10  // Valor immutable
let mutable y = 20  // Valor mutable
y <- 30  // Canviant el valor de y

Tipus de Dades Bàsiques

  • Enter: int
  • Punt flotant: float
  • Booleà: bool
  • Cadena: string
let a: int = 5
let b: float = 3.14
let c: bool = true
let d: string = "Hola, F#"

Funcions

  • Definides utilitzant la paraula clau let.
  • Poden ser niades i suporten la recursió.
let add x y = x + y  // Funció simple

let rec factorial n =  // Funció recursiva
    if n <= 1 then 1
    else n * factorial (n - 1)

Expressions Condicionals

  • Utilitza if ... then ... else per a la lògica condicional.
let isEven n =
    if n % 2 = 0 then
        true
    else
        false

Coincidència de Patrons

  • Característica potent per descompondre tipus de dades.
let describeNumber n =
    match n with
    | 0 -> "Zero"
    | 1 -> "Un"
    | _ -> "Altres"

Llistes

  • Col·leccions immutables d'elements.
  • Creats utilitzant claudàtors [].
let numbers = [1; 2; 3; 4; 5]
let firstNumber = List.head numbers
let remainingNumbers = List.tail numbers

Tuples

  • Agrupen múltiples valors en un sol valor compost.
  • Creats utilitzant parèntesis ().
let person = ("John", 30)
let name, age = person

Registres

  • Defineixen tipus de dades personalitzats amb camps anomenats.
type Person = { Name: string; Age: int }
let john = { Name = "John"; Age = 30 }

Unions Discriminades

  • Defineixen un tipus que pot ser un de diversos casos anomenats.
type Shape =
    | Circle of float
    | Rectangle of float * float

let area shape =
    match shape with
    | Circle radius -> 3.14 * radius * radius
    | Rectangle (width, height) -> width * height

Conclusió

Entendre la sintaxi i l'estructura bàsica de F# és essencial per escriure codi efectiu. Aquesta secció ha cobert els elements fonamentals, incloent comentaris, variables, tipus de dades, funcions, expressions condicionals, coincidència de patrons, llistes, tuples, registres i unions discriminades. Amb aquestes bases, estàs ben equipat per aprofundir en característiques i paradigmes més avançats de la programació en F#.
