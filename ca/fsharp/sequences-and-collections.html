Introducció

A F#, les seqüències i col·leccions són estructures de dades fonamentals que et permeten emmagatzemar i manipular grups de dades. Aquesta secció cobrirà els conceptes bàsics de seqüències, llistes, arrays i altres tipus de col·leccions, proporcionant-te el coneixement per utilitzar-les efectivament en els teus programes.

Llistes

Les llistes són col·leccions immutables d'elements del mateix tipus. Són un dels tipus de col·leccions més comunament utilitzats a F#.

Creació de Llistes

Pots crear llistes utilitzant claudàtors i separant els elements amb punts i coma.

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Operacions Bàsiques

  • Accedir a Elements: Utilitza la funció List.item o l'operador ..
  • Afegir Elements: Utilitza l'operador :: per afegir elements al principi.
  • Concatenar Llistes: Utilitza l'operador @.
let firstElement = List.item 0 numbers
let newList = 0 :: numbers
let combinedList = numbers @ [6; 7; 8]

Funcions Comunes

  • List.map: Aplica una funció a cada element.
  • List.filter: Filtra elements basant-se en un predicat.
  • List.fold: Agrega elements utilitzant una funció.
let squaredNumbers = List.map (fun x -> x * x) numbers
let evenNumbers = List.filter (fun x -> x % 2 = 0) numbers
let sum = List.fold (+) 0 numbers

Arrays

Els arrays són col·leccions mutables d'elements del mateix tipus, permetent accés i actualitzacions eficients.

Creació d'Arrays

Pots crear arrays utilitzant claudàtors i separant els elements amb punts i coma, o utilitzant la funció Array.init.

let numbersArray = [|1; 2; 3; 4; 5|]
let initializedArray = Array.init 5 (fun i -> i * 2)

Operacions Bàsiques

  • Accedir a Elements: Utilitza l'operador . amb un índex.
  • Actualitzar Elements: Utilitza l'operador <- amb un índex.
let firstElementArray = numbersArray.[0]
numbersArray.[0] <- 10

Funcions Comunes

  • Array.map: Aplica una funció a cada element.
  • Array.filter: Filtra elements basant-se en un predicat.
  • Array.fold: Agrega elements utilitzant una funció.
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray
let evenNumbersArray = Array.filter (fun x -> x % 2 = 0) numbersArray
let sumArray = Array.fold (+) 0 numbersArray

Seqüències

Les seqüències són col·leccions mandroses que s'avaluen sota demanda, fent-les adequades per treballar amb conjunts de dades grans o infinits.

Creació de Seqüències

Pots crear seqüències utilitzant la paraula clau seq o utilitzant expressions de seqüència.

let numbersSeq = seq {1 .. 5}
let infiniteSeq = Seq.initInfinite (fun i -> i * 2)

Operacions Bàsiques

  • Accedir a Elements: Utilitza la funció Seq.item.
  • Convertir a Altres Col·leccions: Utilitza funcions com Seq.toList o Seq.toArray.
let firstElementSeq = Seq.item 0 numbersSeq
let numbersList = Seq.toList numbersSeq
let numbersArrayFromSeq = Seq.toArray numbersSeq

Funcions Comunes

  • Seq.map: Aplica una funció a cada element.
  • Seq.filter: Filtra elements basant-se en un predicat.
  • Seq.fold: Agrega elements utilitzant una funció.
let squaredNumbersSeq = Seq.map (fun x -> x * x) numbersSeq
let evenNumbersSeq = Seq.filter (fun x -> x % 2 = 0) numbersSeq
let sumSeq = Seq.fold (+) 0 numbersSeq

Altres Col·leccions

F# també proporciona altres tipus de col·leccions com conjunts i mapes, que ofereixen propietats i casos d'ús únics.

Conjunts

Els conjunts són col·leccions d'elements únics.

let numbersSet = Set.ofList [1; 2; 3; 4; 5; 5]
let isMember = Set.contains 3 numbersSet

Mapes

Els mapes són col·leccions de parelles clau-valor.

let numbersMap = Map.ofList [(1, "one"); (2, "two"); (3, "three")]
let value = Map.find 2 numbersMap

Conclusió

En aquesta secció, hem explorat diverses seqüències i col·leccions a F#, incloent llistes, arrays, seqüències, conjunts i mapes. Entendre aquestes estructures de dades i les seves operacions és crucial per a una programació efectiva a F#. Practica utilitzant aquestes col·leccions en diferents escenaris per convertir-te en un expert en la gestió i manipulació de dades en les teves aplicacions F#.
