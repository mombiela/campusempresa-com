Classes i Objectes en F#
Herència i Interfícies ►

Introducció

Les classes i els objectes són conceptes fonamentals en la programació orientada a objectes (OOP). En F#, tot i que el llenguatge és principalment funcional, admet els principis de l'OOP, permetent-te definir i utilitzar classes i objectes. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat de classes i objectes en F#.

Conceptes Clau

  • Classe: Un plànol per crear objectes, definint propietats, mètodes i esdeveniments.
  • Objecte: Una instància d'una classe.
  • Constructor: Un mètode especial utilitzat per inicialitzar objectes.
  • Herència: Un mecanisme per crear una nova classe utilitzant les propietats i mètodes d'una classe existent.
  • Encapsulació: L'agrupació de dades amb mètodes que operen sobre aquestes dades.
  • Polimorfisme: La capacitat de processar objectes de manera diferent segons el seu tipus de dada o classe.

Definició d'una Classe

En F#, defineixes una classe utilitzant la paraula clau type seguida del nom de la classe i els seus membres.

type Person(name: string, age: int) =
    member this.Name = name
    member this.Age = age
    member this.Introduce() = 
        printfn "Hi, my name is %s and I am %d years old." this.Name this.Age

Explicació

  • type Person(name: string, age: int) defineix una classe anomenada Person amb dos paràmetres: name i age.
  • member this.Name = name i member this.Age = age defineixen propietats.
  • member this.Introduce() defineix un mètode que imprimeix un missatge de salutació.

Creació d'un Objecte

Per crear un objecte, utilitzes la paraula clau new seguida del nom de la classe i els paràmetres del constructor.

let person1 = new Person("Alice", 30)
person1.Introduce()

Explicació

  • let person1 = new Person("Alice", 30) crea una instància de la classe Person.
  • person1.Introduce() crida el mètode Introduce en l'objecte person1.

Herència

L'herència et permet crear una nova classe que reutilitza, amplia o modifica el comportament d'una altra classe.

type Employee(name: string, age: int, employeeId: string) =
    inherit Person(name, age)
    member this.EmployeeId = employeeId
    member this.Work() = 
        printfn "%s is working." this.Name

Explicació

  • inherit Person(name, age) indica que Employee hereta de Person.
  • member this.EmployeeId = employeeId afegeix una nova propietat específica per a Employee.
  • member this.Work() defineix un nou mètode per a Employee.

Encapsulació

L'encapsulació s'aconsegueix controlant l'accés als membres de la classe utilitzant modificadors d'accés com private i public.

type BankAccount(initialBalance: decimal) =
    let mutable balance = initialBalance
    member this.Deposit(amount: decimal) =
        balance <- balance + amount
    member this.Withdraw(amount: decimal) =
        if amount <= balance then
            balance <- balance - amount
        else
            printfn "Insufficient funds"
    member this.Balance = balance

Explicació

  • let mutable balance = initialBalance defineix un camp mutable privat.
  • member this.Deposit(amount: decimal) i member this.Withdraw(amount: decimal) són mètodes públics per modificar el saldo.
  • member this.Balance és una propietat pública per accedir al saldo.

Polimorfisme

El polimorfisme permet que els mètodes processin objectes de manera diferent segons la seva classe.

type Shape() =
    abstract member Area: unit -> float

type Circle(radius: float) =
    inherit Shape()
    override this.Area() = Math.PI * radius * radius

type Square(side: float) =
    inherit Shape()
    override this.Area() = side * side

let shapes: Shape list = [ Circle(5.0); Square(4.0) ]
shapes |> List.iter (fun shape -> printfn "Area: %f" (shape.Area()))

Explicació

  • abstract member Area: unit -> float defineix un mètode abstracte en la classe Shape.
  • override this.Area() proporciona implementacions específiques en Circle i Square.
  • shapes |> List.iter (fun shape -> printfn "Area: %f" (shape.Area())) itera sobre una llista de formes i imprimeix les seves àrees.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les classes i els objectes en F#, incloent la definició de classes, la creació d'objectes, l'herència, l'encapsulació i el polimorfisme. Aquests conceptes són crucials per construir aplicacions robustes i mantenibles. Practica creant les teves pròpies classes i experimentant amb aquestes característiques per aprofundir en la teva comprensió.
