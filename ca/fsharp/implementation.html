Introducció

En aquesta secció, ens endinsarem en els aspectes pràctics de la implementació de codi en F#. Cobriren els conceptes bàsics d'escriure i executar codi F#, explorarem conceptes clau i proporcionarem exemples per consolidar la vostra comprensió. Al final d'aquesta secció, hauríeu de sentir-vos còmodes implementant diverses funcionalitats en F#.

Escriure i Executar Codi F#

Per començar amb F#, necessiteu tenir un entorn configurat on pugueu escriure i executar el vostre codi. Eines comunes inclouen Visual Studio, Visual Studio Code amb l'extensió Ionide, i el .NET Core SDK.

Sintaxi Bàsica

Comencem amb un simple programa "Hola, Món!" per entendre la sintaxi bàsica.

// HelloWorld.fsx
printfn "Hola, Món!"
  • printfn és una funció que imprimeix una cadena formatada a la consola.
  • // denota un comentari en F#.

Per executar aquest script, podeu utilitzar l'interactiu de F# (FSI) o compilar-lo i executar-lo utilitzant la CLI de .NET Core:

dotnet fsi HelloWorld.fsx

Conceptes Clau

Variables i Immutabilitat

En F#, les variables són immutables per defecte. Això significa que un cop assignat un valor a una variable, no es pot canviar.

let x = 10
// x = 20 // Això causarà un error perquè x és immutable

Per crear una variable mutable, podeu utilitzar la paraula clau mutable.

let mutable y = 10
y <- 20 // Ara y és 20

Funcions

Les funcions són ciutadans de primera classe en F#. Podeu definir-les utilitzant la paraula clau let.

let add a b = a + b
printfn "%d" (add 2 3) // Mostra: 5

Coincidència de Patrons

La coincidència de patrons és una característica poderosa en F# que us permet coincidir valors amb patrons i descompondre estructures de dades.

let describeNumber x =
    match x with
    | 0 -> "Zero"
    | 1 -> "One"
    | _ -> "Other"

printfn "%s" (describeNumber 1) // Mostra: One

Llistes i Col·leccions

F# proporciona un conjunt ric de tipus de col·lecció, amb les llistes sent una de les més utilitzades.

let numbers = [1; 2; 3; 4; 5]
let sum = List.sum numbers
printfn "Sum: %d" sum // Mostra: Sum: 15

Tuples i Registres

Els tuples i els registres s'utilitzen per agrupar valors relacionats.

// Tuple
let person = ("John", 30)
let (name, age) = person
printfn "Nom: %s, Edat: %d" name age // Mostra: Nom: John, Edat: 30

// Registre
type Person = { Name: string; Age: int }
let john = { Name = "John"; Age = 30 }
printfn "Nom: %s, Edat: %d" john.Name john.Age // Mostra: Nom: John, Edat: 30

Mòduls i Espais de Noms

Els mòduls i els espais de noms ajuden a organitzar el codi en unitats lògiques.

module MathOperations =
    let add a b = a + b
    let subtract a b = a - b

open MathOperations

printfn "%d" (add 5 3) // Mostra: 8

Temes Avançats

Programació Asíncrona

F# suporta la programació asíncrona utilitzant fluxos de treball async.

open System.Net

let fetchUrl url =
    async {
        let request = WebRequest.Create(url)
        use! response = request.AsyncGetResponse()
        use stream = response.GetResponseStream()
        use reader = new System.IO.StreamReader(stream)
        return reader.ReadToEnd()
    }

let content = fetchUrl "http://example.com" |> Async.RunSynchronously
printfn "%s" content

Proveïdors de Tipus

Els proveïdors de tipus són una característica única en F# que proporcionen una manera d'accedir a dades de manera fortament tipada.

open FSharp.Data

type Weather = JsonProvider<"""{"temp": 23, "humidity": 80}""">
let sample = Weather.Parse("""{"temp": 25, "humidity": 70}""")
printfn "Temperatura: %d, Humitat: %d" sample.Temp sample.Humidity

Conclusió

En aquesta secció, hem cobert els aspectes essencials de la implementació de codi en F#. Hem començat amb els conceptes bàsics d'escriure i executar codi F#, hem explorat conceptes clau com variables, funcions, coincidència de patrons i col·leccions, i hem tocat temes avançats com la programació asíncrona i els proveïdors de tipus. Amb aquestes eines i conceptes, esteu ben equipats per començar a construir aplicacions funcionals i eficients en F#.
