Introducció

En aquesta secció, explorarem dues estructures de dades fonamentals en F#: Tuples i Registres. Aquestes estructures són essencials per organitzar i gestionar dades de manera eficient. Començarem amb els conceptes bàsics de les tuples i després passarem als registres, cobrint la seva sintaxi, ús i exemples pràctics.

Tuples

Les tuples són una manera senzilla d'agrupar múltiples valors en un sol valor compost. Són útils per retornar múltiples valors des d'una funció o per agrupar dades relacionades.

Conceptes Clau

  • Definició: Una tuple és una col·lecció ordenada d'elements, que poden ser de diferents tipus.
  • Immutabilitat: Les tuples són immutables, és a dir, els seus valors no es poden canviar després de ser creades.
  • Sintaxi: Les tuples es defineixen utilitzant parèntesis () i separant els elements amb comes ,.

Exemples

Crear i Accedir a Tuples

// Crear una tuple amb tres elements
let myTuple = (1, "hola", true)

// Accedir als elements d'una tuple
let firstElement = fst myTuple  // 1
let secondElement = snd myTuple // "hola"

// Utilitzar el patró de coincidència per extreure elements
let (a, b, c) = myTuple
printfn "a: %d, b: %s, c: %b" a b c

Tuples Imbricades

// Crear una tuple imbricada
let nestedTuple = ((1, 2), ("a", "b"))

// Accedir als elements d'una tuple imbricada
let ((x, y), (m, n)) = nestedTuple
printfn "x: %d, y: %d, m: %s, n: %s" x y m n

Usos Pràctics

  • Retornar Múltiples Valors d'una Funció: 
    let divide x y =
    if y = 0 then None
    else Some (x / y, x % y)

match divide 10 3 with
| Some (quotient, remainder) -> printfn "Quocient: %d, Resta: %d" quotient remainder
| None -> printfn "No es pot dividir per zero"

Registres

Els registres són una manera de definir un tipus amb camps anomenats. Són més estructurats que les tuples i són particularment útils quan necessites agrupar dades relacionades amb noms significatius.

Conceptes Clau

  • Definició: Un registre és una col·lecció de camps anomenats, cadascun amb un tipus específic.
  • Immutabilitat: Com les tuples, els registres són immutables per defecte, però pots crear camps mutables si és necessari.
  • Sintaxi: Els registres es defineixen utilitzant la paraula clau type seguida dels noms i tipus dels camps.

Exemples

Definir i Crear Registres

// Definir un tipus de registre
type Person = {
    Name: string
    Age: int
    IsStudent: bool
}

// Crear una instància d'un registre
let person1 = { Name = "Alice"; Age = 25; IsStudent = true }

// Accedir als camps d'un registre
printfn "Nom: %s, Edat: %d, És Estudiant: %b" person1.Name person1.Age person1.IsStudent

Actualitzar Registres

// Crear un nou registre amb camps actualitzats
let person2 = { person1 with Age = 26 }
printfn "Edat Actualitzada: %d" person2.Age

Usos Pràctics

  • Modelar Dades: 
    type Address = {
    Street: string
    City: string
    ZipCode: string
}

type Employee = {
    Name: string
    Age: int
    Address: Address
}

let employee1 = {
    Name = "Bob"
    Age = 30
    Address = { Street = "123 Main St"; City = "Somewhere"; ZipCode = "12345" }
}

printfn "Empleat: %s, Ciutat: %s" employee1.Name employee1.Address.City

Conclusió

Les tuples i els registres són eines poderoses en F# per organitzar i gestionar dades. Les tuples són ideals per agrupar un petit nombre de valors sense la necessitat de camps anomenats, mentre que els registres proporcionen una manera més estructurada de definir i treballar amb dades relacionades. Entendre aquestes estructures de dades t'ajudarà a escriure codi F# més eficient i llegible.
