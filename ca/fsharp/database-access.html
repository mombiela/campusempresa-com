Accés a bases de dades en F#
Visualització de Dades ►

Accedir a bases de dades és una habilitat crucial per a qualsevol programador. En aquesta secció, cobrirem com interactuar amb bases de dades utilitzant F#. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Introducció a l'accés a bases de dades

  • Què és una base de dades?
    • Una base de dades és una col·lecció estructurada de dades que es poden accedir, gestionar i actualitzar fàcilment.
  • Per què utilitzar bases de dades?
    • Les bases de dades proporcionen una manera sistemàtica d'emmagatzemar, recuperar i gestionar dades de manera eficient.
  • Tipus de bases de dades:
    • Bases de dades relacionals (per exemple, SQL Server, PostgreSQL)
    • Bases de dades NoSQL (per exemple, MongoDB, Redis)

Configuració del teu entorn

Abans de començar a codificar, assegura't de tenir el següent:

  • F# instal·lat a la teva màquina.
  • Un servidor de base de dades (per exemple, SQL Server, PostgreSQL) instal·lat i en funcionament.
  • Un IDE o editor de text (per exemple, Visual Studio, JetBrains Rider).

Connexió a una base de dades

Utilitzant ADO.NET

ADO.NET és un conjunt de classes que exposen serveis d'accés a dades per a programadors del .NET Framework. Aquí tens com connectar-te a una base de dades SQL Server utilitzant ADO.NET en F#.

Exemple: Connexió a SQL Server

open System.Data.SqlClient

let connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;"

let connectToDatabase() =
    use connection = new SqlConnection(connectionString)
    connection.Open()
    printfn "Connexió oberta amb èxit"
    // Realitza operacions de base de dades aquí
    connection.Close()
    printfn "Connexió tancada amb èxit"

connectToDatabase()
  • Explicació:
    • SqlConnection s'utilitza per establir una connexió amb la base de dades.
    • connectionString conté la informació necessària per connectar-se a la base de dades.
    • connection.Open() obre la connexió.
    • connection.Close() tanca la connexió.

Execució de comandes SQL

Execució de consultes

Un cop connectat, pots executar comandes SQL per interactuar amb la base de dades.

Exemple: Execució d'una consulta SELECT

let executeSelectQuery() =
    use connection = new SqlConnection(connectionString)
    connection.Open()
    let command = new SqlCommand("SELECT * FROM your_table", connection)
    use reader = command.ExecuteReader()
    while reader.Read() do
        printfn "Columna1: %s, Columna2: %s" (reader.GetString(0)) (reader.GetString(1))
    connection.Close()

executeSelectQuery()
  • Explicació:
    • SqlCommand s'utilitza per executar una consulta SQL.
    • ExecuteReader executa la comanda i retorna un SqlDataReader per llegir els resultats.
    • reader.Read() llegeix cada fila del conjunt de resultats.

Inserció de dades

Exemple: Execució d'una comanda INSERT

let executeInsertCommand() =
    use connection = new SqlConnection(connectionString)
    connection.Open()
    let command = new SqlCommand("INSERT INTO your_table (Column1, Column2) VALUES (@value1, @value2)", connection)
    command.Parameters.AddWithValue("@value1", "SampleValue1")
    command.Parameters.AddWithValue("@value2", "SampleValue2")
    let rowsAffected = command.ExecuteNonQuery()
    printfn "%d files inserides" rowsAffected
    connection.Close()

executeInsertCommand()
  • Explicació:
    • Parameters.AddWithValue s'utilitza per afegir paràmetres a la comanda SQL per prevenir la injecció SQL.
    • ExecuteNonQuery executa la comanda i retorna el nombre de files afectades.

Temes avançats

Utilitzant Entity Framework Core

Entity Framework Core (EF Core) és un ORM (Object-Relational Mapper) que simplifica l'accés a bases de dades permetent-te treballar amb dades com a objectes.

Exemple: Configuració d'EF Core

  1. Instal·la els paquets d'EF Core:

    dotnet add package Microsoft.EntityFrameworkCore
dotnet add package Microsoft.EntityFrameworkCore.SqlServer

  2. Defineix un model:

    open Microsoft.EntityFrameworkCore

type YourEntity() =
    [<Key>]
    member val Id = 0 with get, set
    member val Column1 = "" with get, set
    member val Column2 = "" with get, set

type YourDbContext() =
    inherit DbContext()
    [<DefaultValue>]
    val mutable yourEntities : DbSet<YourEntity>
    member this.YourEntities with get() = this.yourEntities and set v = this.yourEntities <- v

    override this.OnConfiguring(optionsBuilder: DbContextOptionsBuilder) =
        optionsBuilder.UseSqlServer("Server=your_server;Database=your_database;User Id=your_username;Password=your_password;") |> ignore

  3. Realització d'operacions CRUD:

    let addEntity() =
    use context = new YourDbContext()
    let entity = YourEntity(Column1 = "SampleValue1", Column2 = "SampleValue2")
    context.YourEntities.Add(entity) |> ignore
    context.SaveChanges() |> ignore
    printfn "Entitat afegida amb èxit"

addEntity()
  • Explicació:
    • DbContext és la classe principal per interactuar amb la base de dades utilitzant EF Core.
    • DbSet representa una col·lecció d'entitats.
    • El mètode OnConfiguring s'utilitza per configurar la connexió a la base de dades.
    • El mètode Add afegeix una nova entitat al context.
    • El mètode SaveChanges guarda els canvis a la base de dades.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de l'accés a bases de dades en F#, incloent la connexió a una base de dades, l'execució de comandes SQL i l'ús d'Entity Framework Core per a escenaris més avançats. Dominant aquestes tècniques, podràs interactuar eficientment amb bases de dades en les teves aplicacions F#. Practica aquests exemples i explora funcions més avançades per aprofundir en el teu coneixement.
