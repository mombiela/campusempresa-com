Unions Discriminats en F#
Tipus Opció i Gestió d'Errors ►

Els Unions Discriminats (DUs) són una característica poderosa en F# que et permeten definir un tipus que pot contenir un de diversos tipus de valors diferents però fixos. Són particularment útils per modelar dades que poden prendre múltiples formes. En aquest tema, explorarem què són els Unions Discriminats, com definir-los i utilitzar-los, i algunes tècniques avançades.

Què són els Unions Discriminats?

Els Unions Discriminats són una manera de definir un tipus que pot ser un de diversos casos anomenats, cadascun potencialment amb diferents tipus i valors. Són similars als enums en altres llenguatges però són més poderosos perquè cada cas pot portar dades addicionals.

Conceptes Clau

  • Cas: Cada valor possible del tipus d'unió.
  • Etiqueta: El nom del cas.
  • Càrrega: Les dades associades amb un cas (opcional).

Definint Unions Discriminats

Per definir un Union Discriminat en F#, utilitzes la paraula clau type seguida del nom de la unió i els casos.

type Shape =
    | Circle of float
    | Rectangle of float * float
    | Triangle of float * float * float

En aquest exemple:

  • Shape és el tipus d'unió.
  • Circle, Rectangle, i Triangle són els casos.
  • Cada cas té dades associades (per exemple, Circle té un float que representa el radi).

Utilitzant Unions Discriminats

Un cop definits, pots crear instàncies d'un Union Discriminat i utilitzar el patró de coincidència per treballar amb ells.

Creant Instàncies

let myCircle = Circle(5.0)
let myRectangle = Rectangle(4.0, 6.0)
let myTriangle = Triangle(3.0, 4.0, 5.0)

Patró de Coincidència

El patró de coincidència et permet descompondre i treballar amb els valors d'un Union Discriminat.

let describeShape shape =
    match shape with
    | Circle(radius) -> sprintf "Un cercle amb radi %f" radius
    | Rectangle(width, height) -> sprintf "Un rectangle amb amplada %f i alçada %f" width height
    | Triangle(a, b, c) -> sprintf "Un triangle amb costats %f, %f, %f" a b c

let description = describeShape myCircle
printfn "%s" description

Tècniques Avançades

Unions Discriminats Imbricats

Pots imbricar Unions Discriminats dins d'altres Unions Discriminats per modelar estructures de dades més complexes.

type Point = { X: float; Y: float }

type Shape =
    | Circle of float
    | Rectangle of float * float
    | Triangle of float * float * float
    | Polygon of Point list

Utilitzant Tipus Option

Els Unions Discriminats sovint s'utilitzen amb el tipus Option per representar valors que poden o no estar presents.

type Result =
    | Success of string
    | Error of string

let processResult result =
    match result with
    | Success(message) -> printfn "Èxit: %s" message
    | Error(errorMessage) -> printfn "Error: %s" errorMessage

let result = Success("Operació completada amb èxit")
processResult result

Unions Discriminats Recursius

Els Unions Discriminats poden ser recursius, cosa que és útil per representar estructures en forma d'arbre.

type BinaryTree =
    | Leaf of int
    | Node of BinaryTree * int * BinaryTree

let rec sumTree tree =
    match tree with
    | Leaf(value) -> value
    | Node(left, value, right) -> sumTree(left) + value + sumTree(right)

let myTree = Node(Node(Leaf(1), 2, Leaf(3)), 4, Leaf(5))
let total = sumTree myTree
printfn "Suma de l'arbre: %d" total

Conclusió

Els Unions Discriminats són una característica versàtil i poderosa en F# que et permeten definir tipus amb múltiples casos diferents i distintius. Són útils per modelar dades que poden prendre diferents formes i per implementar el patró de coincidència. Entenent i utilitzant els Unions Discriminats, pots escriure codi F# més expressiu i robust.
