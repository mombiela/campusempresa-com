Aplicacions d'escriptori en F#
Desenvolupament Mòbil ►

En aquesta secció, explorarem com crear aplicacions d'escriptori utilitzant F#. Cobrirarem els conceptes bàsics de la configuració d'un projecte, la creació d'una interfície d'usuari senzilla, la gestió d'esdeveniments i temes més avançats com el vincle de dades i l'arquitectura MVVM (Model-View-ViewModel).

Configurant el teu projecte

Abans de començar a construir aplicacions d'escriptori, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar .NET SDK: Assegura't de tenir instal·lat el .NET SDK. Pots descarregar-lo des del lloc web oficial de .NET.
  • Instal·lar un IDE: Es recomana Visual Studio o JetBrains Rider per al desenvolupament en F#.

Creant un nou projecte

Per crear un nou projecte d'aplicació d'escriptori en F#, pots utilitzar la .NET CLI o Visual Studio.

Utilitzant la .NET CLI:

dotnet new winforms -lang F# -o MyDesktopApp
cd MyDesktopApp

Utilitzant Visual Studio:

  1. Obre Visual Studio.
  2. Selecciona "Crear un nou projecte".
  3. Tria "Aplicació de Windows Forms (F#)".
  4. Nomena el teu projecte i fes clic a "Crear".

Construint una interfície d'usuari senzilla

Comencem creant una interfície d'usuari senzilla amb un botó i una etiqueta.

Afegint controls

En el fitxer Form1.fs, pots afegir controls així:

open System
open System.Windows.Forms

type Form1() as this =
    inherit Form(Text = "My First F# Desktop App")

    let label = new Label(Text = "Hello, F#!", Dock = DockStyle.Top)
    let button = new Button(Text = "Click Me", Dock = DockStyle.Bottom)

    do
        this.Controls.Add(label)
        this.Controls.Add(button)

Gestionant esdeveniments

Per gestionar esdeveniments, com ara clics de botó, pots afegir gestors d'esdeveniments en el constructor.

do
    button.Click.Add(fun _ -> label.Text <- "Button Clicked!")

Temes avançats

Vincle de dades

El vincle de dades és una característica poderosa que et permet sincronitzar la interfície d'usuari amb el teu model de dades.

open System.ComponentModel

type Person(name: string, age: int) =
    let mutable name = name
    let mutable age = age

    member this.Name
        with get() = name
        and set(value) = name <- value

    member this.Age
        with get() = age
        and set(value) = age <- value

    interface INotifyPropertyChanged with
        [<CLIEvent>]
        member this.PropertyChanged = new Event<PropertyChangedEventHandler, PropertyChangedEventArgs>()

let person = Person("John Doe", 30)

Arquitectura MVVM

El patró MVVM ajuda a separar la lògica de negoci de la interfície d'usuari. Aquí tens un exemple bàsic:

  • Model: Representa les dades.
  • Vista: Representa la interfície d'usuari.
  • ViewModel: Actua com a pont entre el Model i la Vista.
type MainViewModel() =
    inherit ViewModelBase()

    let mutable name = "John Doe"
    let mutable age = 30

    member this.Name
        with get() = name
        and set(value) =
            name <- value
            this.OnPropertyChanged("Name")

    member this.Age
        with get() = age
        and set(value) =
            age <- value
            this.OnPropertyChanged("Age")

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la creació d'aplicacions d'escriptori en F#. Hem començat amb la configuració del projecte, la creació d'una interfície d'usuari senzilla, la gestió d'esdeveniments i hem tocat temes més avançats com el vincle de dades i l'arquitectura MVVM. Amb aquestes bases, pots començar a construir aplicacions d'escriptori més complexes i amb més funcionalitats en F#.
