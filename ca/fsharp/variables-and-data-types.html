Variables i Tipus de Dades en F#
Immutabilitat i Mutabilitat ►

Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en F#. Entendre aquests conceptes bàsics és crucial per escriure programes F# efectius i eficients. Cobrirem:

  • Declaració de variables
  • Variables immutables i mutables
  • Tipus de dades bàsics
  • Inferència de tipus
  • Tipus de dades personalitzats

Declaració de Variables

En F#, les variables es declaren utilitzant la paraula clau let. Per defecte, les variables en F# són immutables, és a dir, els seus valors no es poden canviar un cop assignats.

Exemple:

let x = 10
let y = "Hola, F#"

En l'exemple anterior:

  • x és una variable enter amb un valor de 10.
  • y és una variable cadena amb un valor de "Hola, F#".

Variables Immutables i Mutables

Variables Immutables

Com s'ha esmentat, les variables declarades amb let són immutables.

Exemple:

let a = 5
// a <- 10  // Això causarà un error perquè 'a' és immutable

Variables Mutables

Si necessites una variable el valor de la qual pugui canviar, pots utilitzar la paraula clau mutable.

Exemple:

let mutable b = 5
b <- 10  // Això està permès perquè 'b' és mutable

Tipus de Dades Bàsics

F# admet diversos tipus de dades bàsics. Aquí hi ha alguns dels més utilitzats:

| Tipus de Dades | Descripció | Exemple | |----------------|-----------------------|------------------------| | int | Enter | let num = 42 | | float | Número amb coma flotant| let pi = 3.14 | | bool | Booleà | let isTrue = true | | string | Cadena | let greeting = "Hola"| | char | Caràcter | let letter = 'A' |

Exemple:

let age: int = 25
let price: float = 19.99
let isActive: bool = true
let name: string = "Alice"
let initial: char = 'A'

Inferència de Tipus

F# té un sistema d'inferència de tipus potent que sovint pot deduir el tipus d'una variable a partir del seu valor, de manera que no sempre cal especificar el tipus explícitament.

Exemple:

let inferredInt = 100  // F# infereix això com a int
let inferredString = "Inferència de Tipus"  // F# infereix això com a string

Tipus de Dades Personalitzats

F# et permet definir els teus propis tipus de dades utilitzant la paraula clau type. Pots crear tipus de registre, tipus d'unió i més.

Tipus de Registre

Els tipus de registre s'utilitzen per agrupar valors relacionats.

Exemple:

type Person = {
    Name: string
    Age: int
}

let person1 = { Name = "John"; Age = 30 }

Tipus d'Unió

Els tipus d'unió s'utilitzen per representar un valor que pot ser un de diversos tipus diferents.

Exemple:

type Shape =
    | Circle of float
    | Rectangle of float * float

let circle = Circle 5.0
let rectangle = Rectangle (4.0, 6.0)

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en F#. Hem après com declarar variables, la diferència entre variables immutables i mutables, i hem explorat tipus de dades bàsics i personalitzats. Entendre aquests conceptes és essencial per escriure programes F# robustos. En la següent secció, aprofundirem en les funcions i com utilitzar-les de manera efectiva en F#.
