Introducció

Les proves i la depuració són aspectes crucials del desenvolupament de programari. Asseguren que el teu codi funcioni com s'espera i ajuden a identificar i solucionar problemes. En aquesta secció, cobrirem els conceptes bàsics de les proves i la depuració en F#, des de nivells principiants fins a avançats.

Proves Unitàries en F#

Què són les Proves Unitàries?

  • Les proves unitàries impliquen provar components o funcions individuals del teu codi per assegurar-se que funcionen correctament.
  • Ajuda a detectar errors aviat i fa que el teu codi sigui més mantenible.

Configuració d'un Entorn de Proves

  • Utilitza el framework xUnit per a proves unitàries en F#.
  • Instal·la els paquets NuGet xUnit i xUnit.runner.visualstudio.

Escriure la Teva Primera Prova

open Xunit

module MyTests =

    [<Fact>]
    let ``Addition Test`` () =
        let result = 2 + 2
        Assert.Equal(4, result)
  • open Xunit importa la biblioteca xUnit.
  • L'atribut [<Fact>] marca el mètode com una prova.
  • Assert.Equal comprova si el resultat esperat coincideix amb el resultat real.

Executar Proves

  • Utilitza Visual Studio Test Explorer o la comanda dotnet test per executar les teves proves.

Proves Unitàries Avançades

Proves Parametritzades

  • Utilitza els atributs [<Theory>] i [<InlineData>] per a proves parametritzades.
open Xunit

module MyTests =

    [<Theory>]
    [<InlineData(2, 2, 4)>]
    [<InlineData(3, 3, 6)>]
    let ``Addition Test`` (a: int, b: int, expected: int) =
        let result = a + b
        Assert.Equal(expected, result)

Simulació de Dependències

  • Utilitza la biblioteca Foq per a la simulació de dependències.
  • Instal·la el paquet NuGet Foq.
open Foq
open Xunit

type ICalculator =
    abstract member Add: int -> int -> int

module MyTests =

    [<Fact>]
    let ``Mocking Test`` () =
        let mockCalculator = Mock<ICalculator>()
                              .Setup(fun x -> <@ x.Add(2, 2) @>)
                              .Returns(4)
                              .Create()
        let result = mockCalculator.Add(2, 2)
        Assert.Equal(4, result)

Depuració en F#

Utilitzant el Depurador de Visual Studio

  • Configura punts de ruptura fent clic al marge al costat del número de línia.
  • Utilitza F5 per iniciar la depuració i F10/F11 per avançar sobre/entrar en el codi.

Tècniques Comunes de Depuració

  • Declaracions de Print: Utilitza printfn per imprimir valors de variables.
  • Finestra de Supervisió: Monitoritza els valors de les variables en temps real.
  • Finestra Immediata: Executa fragments de codi durant la depuració.

Gestió d'Excepcions

  • Utilitza blocs try...with per gestionar excepcions.
try
    // Codi que podria llançar una excepció
    let result = 10 / 0
    printfn "Result: %d" result
with
| :? System.DivideByZeroException as ex ->
    printfn "Exception: %s" ex.Message

Conclusió

Les proves i la depuració són habilitats essencials per a qualsevol desenvolupador de F#. Les proves unitàries asseguren que el teu codi funcioni com s'espera, mentre que la depuració t'ajuda a identificar i solucionar problemes. En dominar aquestes tècniques, podràs escriure codi més fiable i mantenible.
