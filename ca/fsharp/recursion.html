Introducció a la Recursió

La recursió és un concepte fonamental en la informàtica i la programació funcional. Implica una funció que es crida a si mateixa per resoldre instàncies més petites del mateix problema. En F#, la recursió és una eina poderosa que permet solucions elegants i concises a molts problemes.

Conceptes Clau

  • Cas Base: La condició sota la qual la funció recursiva deixa de cridar-se a si mateixa.
  • Cas Recursiu: La part de la funció on la funció es crida a si mateixa amb una entrada més petita o més senzilla.

Recursió Bàsica

Exemple: Funció Factorial

El factorial d'un enter no negatiu n és el producte de tots els enters positius menors o iguals a n. Es pot definir recursivament com:

  • factorial(0) = 1 (Cas Base)
  • factorial(n) = n * factorial(n - 1) (Cas Recursiu)
let rec factorial n =
    if n = 0 then 1
    else n * factorial (n - 1)

// Exemple d'ús:
let result = factorial 5 // result és 120

Explicació

  • Cas Base: Quan n és 0, la funció retorna 1.
  • Cas Recursiu: La funció es crida a si mateixa amb n - 1 i multiplica el resultat per n.

Recursió Intermèdia

Exemple: Seqüència de Fibonacci

La seqüència de Fibonacci és una sèrie de nombres on cada nombre és la suma dels dos precedents. Es pot definir recursivament com:

  • fibonacci(0) = 0 (Cas Base)
  • fibonacci(1) = 1 (Cas Base)
  • fibonacci(n) = fibonacci(n - 1) + fibonacci(n - 2) (Cas Recursiu)
let rec fibonacci n =
    match n with
    | 0 -> 0
    | 1 -> 1
    | _ -> fibonacci (n - 1) + fibonacci (n - 2)

// Exemple d'ús:
let result = fibonacci 6 // result és 8

Explicació

  • Casos Base: Quan n és 0 o 1, la funció retorna 0 o 1, respectivament.
  • Cas Recursiu: La funció es crida a si mateixa amb n - 1 i n - 2 i suma els resultats.

Recursió Avançada

Exemple: Recursió de Cua

La recursió de cua és una forma especial de recursió on la crida recursiva és l'última operació de la funció. Això permet al compilador optimitzar la recursió, prevenint desbordaments de pila i millorant el rendiment.

Factorial Recursiu de Cua

let factorialTailRecursive n =
    let rec loop acc n =
        if n = 0 then acc
        else loop (acc * n) (n - 1)
    loop 1 n

// Exemple d'ús:
let result = factorialTailRecursive 5 // result és 120

Explicació

  • Acumulador: El paràmetre acc acumula el resultat a mesura que la recursió progressa.
  • Crida de Cua: La crida recursiva a loop és l'última operació, permetent l'optimització.

Exemple: Fibonacci Recursiu de Cua

let fibonacciTailRecursive n =
    let rec loop a b n =
        if n = 0 then a
        else loop b (a + b) (n - 1)
    loop 0 1 n

// Exemple d'ús:
let result = fibonacciTailRecursive 6 // result és 8

Explicació

  • Paràmetres: a i b contenen els dos nombres de Fibonacci precedents.
  • Crida de Cua: La crida recursiva a loop és l'última operació, permetent l'optimització.

Conclusió

La recursió és una tècnica poderosa en F# que permet solucions elegants a problemes complexos. Entendre el cas base i el cas recursiu és crucial per escriure funcions recursives correctes. La recursió de cua és una forma avançada de recursió que optimitza el rendiment i prevé desbordaments de pila. En dominar la recursió, pots resoldre una àmplia gamma de problemes de manera més efectiva en F#.
