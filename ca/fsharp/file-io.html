Introducció a l'Entrada/Sortida de Fitxers

L'entrada/sortida de fitxers (Input/Output) és un aspecte fonamental de la programació que et permet llegir i escriure fitxers. En F#, pots gestionar operacions de fitxers utilitzant l'espai de noms System.IO, que proporciona diverses classes i mètodes per treballar amb fitxers i directoris.

Llegir Fitxers

Llegir Tot el Fitxer

Per llegir tot el contingut d'un fitxer, pots utilitzar el mètode File.ReadAllText.

open System.IO

let readFile path =
    let content = File.ReadAllText(path)
    printfn "Contingut del Fitxer:\n%s" content

// Exemple d'ús
readFile "example.txt"

Llegir Fitxer Línia per Línia

Per llegir un fitxer línia per línia, pots utilitzar el mètode File.ReadLines.

open System.IO

let readFileLines path =
    let lines = File.ReadLines(path)
    for line in lines do
        printfn "%s" line

// Exemple d'ús
readFileLines "example.txt"

Escriure Fitxers

Escriure Tot el Contingut

Per escriure una cadena a un fitxer, pots utilitzar el mètode File.WriteAllText.

open System.IO

let writeFile path content =
    File.WriteAllText(path, content)
    printfn "Contingut escrit a %s" path

// Exemple d'ús
writeFile "example.txt" "Hola, món F#!"

Escriure Línia per Línia

Per escriure múltiples línies a un fitxer, pots utilitzar el mètode File.WriteAllLines.

open System.IO

let writeFileLines path lines =
    File.WriteAllLines(path, lines)
    printfn "Línies escrites a %s" path

// Exemple d'ús
writeFileLines "example.txt" [| "Línia 1"; "Línia 2"; "Línia 3" |]

Afegir a Fitxers

Per afegir contingut a un fitxer existent, pots utilitzar els mètodes File.AppendAllText o File.AppendAllLines.

Afegir Text

open System.IO

let appendToFile path content =
    File.AppendAllText(path, content)
    printfn "Contingut afegit a %s" path

// Exemple d'ús
appendToFile "example.txt" "\nLínia afegida"

Afegir Línies

open System.IO

let appendLinesToFile path lines =
    File.AppendAllLines(path, lines)
    printfn "Línies afegides a %s" path

// Exemple d'ús
appendLinesToFile "example.txt" [| "Línia afegida 1"; "Línia afegida 2" |]

Treballar amb Fitxers Binari

Per operacions de fitxers binaris, pots utilitzar FileStream juntament amb BinaryReader i BinaryWriter.

Escriure Dades Binàries

open System.IO

let writeBinaryFile path data =
    use fs = new FileStream(path, FileMode.Create, FileAccess.Write)
    use bw = new BinaryWriter(fs)
    bw.Write(data)
    printfn "Dades binàries escrites a %s" path

// Exemple d'ús
writeBinaryFile "example.bin" 12345

Llegir Dades Binàries

open System.IO

let readBinaryFile path =
    use fs = new FileStream(path, FileMode.Open, FileAccess.Read)
    use br = new BinaryReader(fs)
    let data = br.ReadInt32()
    printfn "Dades binàries llegides de %s: %d" path data

// Exemple d'ús
readBinaryFile "example.bin"

Gestionar Excepcions

Les operacions de fitxers poden fallar per diverses raons (per exemple, fitxer no trobat, accés denegat). És important gestionar les excepcions correctament.

open System.IO

let safeReadFile path =
    try
        let content = File.ReadAllText(path)
        printfn "Contingut del Fitxer:\n%s" content
    with
    | :? FileNotFoundException ->
        printfn "Fitxer no trobat: %s" path
    | :? UnauthorizedAccessException ->
        printfn "Accés denegat al fitxer: %s" path
    | ex ->
        printfn "S'ha produït un error: %s" (ex.Message)

// Exemple d'ús
safeReadFile "nonexistent.txt"

Resum

En aquesta secció, hem cobert els conceptes bàsics de l'entrada/sortida de fitxers en F#. Has après a:

  • Llegir fitxers completament o línia per línia.
  • Escriure i afegir text o línies a fitxers.
  • Treballar amb fitxers binaris utilitzant FileStream, BinaryReader i BinaryWriter.
  • Gestionar excepcions que poden ocórrer durant les operacions de fitxers.

Entendre l'entrada/sortida de fitxers és crucial per a moltes aplicacions, i dominar aquestes tècniques t'ajudarà a gestionar l'emmagatzematge i la recuperació de dades de manera eficient en els teus programes F#.
