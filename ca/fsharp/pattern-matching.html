Coincidència de Patrons en F#
Funcions d'Ordre Superior ►

La coincidència de patrons és una característica poderosa en F# que et permet comprovar un valor contra un patró i descompondre dades d'una manera concisa i llegible. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de la coincidència de patrons en F#.

Introducció a la Coincidència de Patrons

La coincidència de patrons en F# s'utilitza per a coincidir valors contra patrons i executar codi basat en la coincidència. És similar a les instruccions switch-case en altres llenguatges però més poderosa i expressiva.

Coincidència de Patrons Bàsica

En la seva forma més simple, la coincidència de patrons es pot utilitzar per a coincidir valors literals.

let describeNumber x =
    match x with
    | 1 -> "One"
    | 2 -> "Two"
    | 3 -> "Three"
    | _ -> "Unknown"

printfn "%s" (describeNumber 2)  // Output: Two
  • match x with inicia la coincidència de patrons.
  • | separa diferents patrons.
  • _ és un patró comodí que coincideix amb qualsevol valor no coincidit per patrons anteriors.

Coincidència de Patrons amb Tuples

La coincidència de patrons també es pot utilitzar amb tuples per descompondre valors.

let addTuples (a, b) (c, d) =
    match (a, b, c, d) with
    | (0, 0, 0, 0) -> "All zeros"
    | (x, y, 0, 0) -> sprintf "First tuple: (%d, %d)" x y
    | (0, 0, x, y) -> sprintf "Second tuple: (%d, %d)" x y
    | _ -> "Mixed values"

printfn "%s" (addTuples (1, 2) (0, 0))  // Output: First tuple: (1, 2)
  • Les tuples es descomponen en la coincidència de patrons.
  • Es poden coincidir patrons específics per manejar diferents casos.

Coincidència de Patrons amb Llistes

Les llistes també es poden coincidir per manejar diferents estructures.

let rec sumList lst =
    match lst with
    | [] -> 0
    | head :: tail -> head + sumList tail

printfn "%d" (sumList [1; 2; 3; 4])  // Output: 10
  • [] coincideix amb una llista buida.
  • head :: tail coincideix amb una llista no buida, on head és el primer element i tail és la resta de la llista.

Coincidència de Patrons amb Registres i Unions Discriminades

La coincidència de patrons és particularment poderosa quan s'utilitza amb registres i unions discriminades.

Registres

type Person = { Name: string; Age: int }

let describePerson person =
    match person with
    | { Name = "Alice"; Age = 30 } -> "Alice is 30 years old"
    | { Name = name; Age = age } -> sprintf "%s is %d years old" name age

let alice = { Name = "Alice"; Age = 30 }
printfn "%s" (describePerson alice)  // Output: Alice is 30 years old

Unions Discriminades

type Shape =
    | Circle of radius: float
    | Rectangle of width: float * height: float

let area shape =
    match shape with
    | Circle radius -> System.Math.PI * radius * radius
    | Rectangle (width, height) -> width * height

let myCircle = Circle 5.0
printfn "%f" (area myCircle)  // Output: 78.539816

Coincidència de Patrons Avançada

Patrons Actius

Els patrons actius et permeten definir la lògica de coincidència de patrons personalitzada.

let (|Even|Odd|) x =
    if x % 2 = 0 then Even else Odd

let describeNumber x =
    match x with
    | Even -> "Even"
    | Odd -> "Odd"

printfn "%s" (describeNumber 4)  // Output: Even

Patrons Actius Parcials

Els patrons actius parcials poden retornar un tipus d'opció, permetent una coincidència més complexa.

let (|Positive|_|) x =
    if x > 0 then Some Positive else None

let describeNumber x =
    match x with
    | Positive -> "Positive"
    | _ -> "Non-positive"

printfn "%s" (describeNumber 5)  // Output: Positive

Conclusió

La coincidència de patrons en F# és una característica versàtil i poderosa que simplifica la gestió d'estructures de dades complexes. Des de la coincidència de valors simples fins a la descomposició de tuples, llistes, registres i unions discriminades, la coincidència de patrons millora la llegibilitat i mantenibilitat del codi. Les tècniques avançades com els patrons actius amplien encara més les seves capacitats, fent de F# un llenguatge robust per a la programació funcional.

En dominar la coincidència de patrons, pots escriure codi més expressiu i concís, fent que la teva programació en F# sigui més efectiva i agradable.
