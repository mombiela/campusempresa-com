Configuracions d'Exemple

  • Configuració Bàsica del Servidor HTTP:

    server {
    listen 80;
    server_name example.com;

    location / {
        root /usr/share/nginx/html;
        index index.html index.htm;
    }
}
    • Aquesta configuració configura un servidor HTTP bàsic que escolta al port 80, servint fitxers des de /usr/share/nginx/html.

  • Configuració de Proxy Invers:

    server {
    listen 80;
    server_name example.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
    • Aquesta configuració configura NGINX com un proxy invers, reenviant les sol·licituds a un servidor backend que s'executa a localhost:3000.
