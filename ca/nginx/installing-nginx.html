Introducció

NGINX és un servidor HTTP d'alt rendiment, un proxy invers i un servidor proxy IMAP/POP3. Aquesta secció et guiarà a través del procés d'instal·lació de NGINX en diversos sistemes operatius. Al final d'aquesta secció, tindràs un servidor NGINX en funcionament a la teva màquina.

Prerequisits

Abans d'instal·lar NGINX, assegura't de tenir el següent:

  • Una màquina amb un sistema operatiu basat en Unix (Linux, macOS) o Windows.
  • Coneixements bàsics de la línia de comandes.
  • Accés administratiu (root) a la teva màquina.

Instal·lant NGINX a Linux

Utilitzant el gestor de paquets (Ubuntu/Debian)

  1. Actualitza la llista de paquets: 
    sudo apt update
  2. Instal·la NGINX: 
    sudo apt install nginx
  3. Inicia NGINX: 
    sudo systemctl start nginx
  4. Habilita NGINX per iniciar-se en arrencar: 
    sudo systemctl enable nginx
  5. Verifica la instal·lació: Obre el teu navegador web i navega a http://localhost. Hauries de veure la pàgina de benvinguda de NGINX.

Utilitzant el gestor de paquets (CentOS/RHEL)

  1. Instal·la el repositori EPEL: 
    sudo yum install epel-release
  2. Instal·la NGINX: 
    sudo yum install nginx
  3. Inicia NGINX: 
    sudo systemctl start nginx
  4. Habilita NGINX per iniciar-se en arrencar: 
    sudo systemctl enable nginx
  5. Verifica la instal·lació: Obre el teu navegador web i navega a http://localhost. Hauries de veure la pàgina de benvinguda de NGINX.

Instal·lant NGINX a macOS

Utilitzant Homebrew

  1. Instal·la Homebrew (si no està instal·lat): 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instal·la NGINX: 
    brew install nginx
  3. Inicia NGINX: 
    sudo nginx
  4. Verifica la instal·lació: Obre el teu navegador web i navega a http://localhost:8080. Hauries de veure la pàgina de benvinguda de NGINX.

Instal·lant NGINX a Windows

Utilitzant el binari de NGINX

  1. Descarrega el binari de NGINX: Ves a la pàgina de descàrrega de NGINX i descarrega la versió estable per a Windows.
  2. Extreu el fitxer descarregat: Extreu el contingut del fitxer zip a un directori de la teva elecció (per exemple, C:\nginx).
  3. Executa NGINX: Obre el Símbol del sistema i navega al directori de NGINX: 
    cd C:\nginx
    Inicia NGINX: 
    start nginx
  4. Verifica la instal·lació: Obre el teu navegador web i navega a http://localhost. Hauries de veure la pàgina de benvinguda de NGINX.

Passos comuns post-instal·lació

Comprovar l'estat de NGINX

  • Linux: 
    sudo systemctl status nginx
  • macOS: 
    sudo nginx -t
  • Windows: Obre el Símbol del sistema i navega al directori de NGINX, després executa: 
    nginx -t

Aturar i reiniciar NGINX

  • Linux: 
    sudo systemctl stop nginx
sudo systemctl restart nginx
  • macOS: 
    sudo nginx -s stop
sudo nginx -s reload
  • Windows: Obre el Símbol del sistema i navega al directori de NGINX, després executa: 
    nginx -s stop
nginx -s reload

Conclusió

En aquesta secció, has après com instal·lar NGINX en diversos sistemes operatius, incloent Linux, macOS i Windows. També has après com iniciar, aturar i verificar l'estat del teu servidor NGINX. Amb NGINX instal·lat, ara estàs preparat per configurar-lo i utilitzar-lo per a les teves necessitats de servidor web.
