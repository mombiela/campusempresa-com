Introducció a FastCGI

FastCGI és un protocol per interfacing programes interactius amb un servidor web. És una millora respecte a l'antic Common Gateway Interface (CGI) i proporciona millor rendiment i escalabilitat.

  • CGI vs FastCGI:
    • CGI: Cada sol·licitud genera un nou procés, cosa que comporta un alt cost.
    • FastCGI: Manté els processos vius per gestionar múltiples sol·licituds, reduint el cost.

Configuració de NGINX amb FastCGI

Per utilitzar FastCGI amb NGINX, cal configurar NGINX per passar les sol·licituds a un servidor FastCGI.

Instal·lació de FastCGI

Primer, assegura't que tens un servidor FastCGI instal·lat. Per a aplicacions PHP, pots utilitzar php-fpm (PHP FastCGI Process Manager).

sudo apt-get install php-fpm

Configuració de NGINX per utilitzar FastCGI

Edita el teu fitxer de configuració de NGINX per incloure els paràmetres de FastCGI.

server {
    listen 80;
    server_name example.com;

    root /var/www/html;

    location / {
        try_files $uri $uri/ =404;
    }

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
    }
}
  • Explicació:
    • server: Defineix un servidor virtual.
    • listen 80: Escolta al port 80.
    • server_name example.com: El nom de domini.
    • root /var/www/html: El directori arrel per al servidor.
    • location /: Gestiona les sol·licituds a l'arrel.
    • try_files $uri $uri/ =404: Intenta servir el fitxer, directori, o retorna un 404.
    • location ~ \.php$: Gestiona fitxers PHP.
    • include snippets/fastcgi-php.conf: Inclou la configuració de FastCGI.
    • fastcgi_pass unix:/var/run/php/php7.4-fpm.sock: Passa la sol·licitud al socket PHP-FPM.

Paràmetres de FastCGI

Pots ajustar els paràmetres de FastCGI per optimitzar el rendiment.

location ~ \.php$ {
    include fastcgi_params;
    fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
}
  • Explicació:
    • fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name: Estableix el paràmetre del nom del fitxer de script per a FastCGI.

Configuració avançada de FastCGI

Per a configuracions més avançades, pots ajustar les mides dels buffers de FastCGI i els temps d'espera.

Mides dels Buffers

Ajustar les mides dels buffers pot ajudar a gestionar respostes grans.

location ~ \.php$ {
    include fastcgi_params;
    fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
    fastcgi_buffers 16 16k;
    fastcgi_buffer_size 32k;
}
  • Explicació:
    • fastcgi_buffers 16 16k: Estableix el nombre i la mida dels buffers de FastCGI.
    • fastcgi_buffer_size 32k: Estableix la mida del buffer de FastCGI.

Temps d'Espera

Establir temps d'espera adequats pot evitar sol·licituds penjades.

location ~ \.php$ {
    include fastcgi_params;
    fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
    fastcgi_connect_timeout 60s;
    fastcgi_send_timeout 60s;
    fastcgi_read_timeout 60s;
}
  • Explicació:
    • fastcgi_connect_timeout 60s: Temps d'espera per connectar-se a FastCGI.
    • fastcgi_send_timeout 60s: Temps d'espera per enviar dades a FastCGI.
    • fastcgi_read_timeout 60s: Temps d'espera per llegir dades de FastCGI.

Conclusió

NGINX i FastCGI treballen junts per proporcionar una manera potent i eficient de servir contingut dinàmic. Entenent i configurant els paràmetres de FastCGI, pots optimitzar el rendiment del teu servidor i gestionar càrregues de trànsit elevades de manera efectiva. Aquesta secció ha cobert els conceptes bàsics de la configuració de NGINX amb FastCGI, així com configuracions avançades per a un millor rendiment.
