Introducció

Benvingut de nou! Ara que tens una base sòlida en la programació de scripts Bash, és hora d'aprofundir i explorar conceptes i tècniques més avançades. Aquesta secció t'ajudarà a millorar les teves habilitats i a ser més eficient en l'escriptura de scripts Bash potents.

Tècniques avançades de scripting

Funcions

Les funcions et permeten encapsular codi en blocs reutilitzables, fent que els teus scripts siguin més modulars i fàcils de mantenir.

Definint una funció

my_function() {
    echo "Hola des de my_function!"
}

Cridant una funció

my_function
  • Conceptes clau:
    • Les funcions es defineixen utilitzant la sintaxi function_name().
    • Les funcions es poden cridar simplement utilitzant el seu nom.

Arrays

Els arrays poden emmagatzemar múltiples valors en una sola variable, cosa que és útil per gestionar llistes de dades.

Declarant un array

my_array=("poma" "plàtan" "cirera")

Accedint als elements de l'array

echo ${my_array[1]}  # Mostra: plàtan
  • Conceptes clau:
    • Els arrays es declaren utilitzant parèntesis ().
    • Els elements de l'array s'accedeixen utilitzant la sintaxi ${array_name[index]}.

Constructs de bucles

Els constructs de bucles avançats et poden ajudar a iterar sobre arrays, fitxers i sortides de comandes de manera més eficient.

Bucle For

for fruit in "${my_array[@]}"; do
    echo "M'agrada $fruit"
done

Bucle While

counter=1
while [ $counter -le 5 ]; do
    echo "Comptador: $counter"
    ((counter++))
done
  • Conceptes clau:
    • El bucle for itera sobre una llista d'elements.
    • El bucle while continua mentre la condició sigui certa.

Gestió d'errors i depuració

Gestió d'errors

Una correcta gestió d'errors assegura que el teu script pugui gestionar situacions inesperades de manera elegant.

Utilitzant l'estat de sortida

if cp source.txt destination.txt; then
    echo "Fitxer copiat amb èxit."
else
    echo "No s'ha pogut copiar el fitxer."
fi
  • Conceptes clau:
    • Les comandes retornen un estat de sortida (0 per èxit, no zero per fallada).
    • Utilitza sentències if per comprovar l'estat de sortida i gestionar errors en conseqüència.

Depuració

La depuració t'ajuda a identificar i solucionar problemes en els teus scripts.

Activant la depuració

#!/bin/bash
set -x  # Activa la depuració
  • Conceptes clau:
    • Utilitza set -x per activar la depuració, que imprimeix cada comanda abans d'executar-la.
    • Utilitza set +x per desactivar la depuració.

Treballant amb fitxers i directoris

Manipulació de fitxers

Aprèn a crear, llegir i manipular fitxers en els teus scripts.

Creant un fitxer

echo "Hola, Món!" > hello.txt

Llegint un fitxer

while IFS= read -r line; do
    echo "$line"
done < hello.txt
  • Conceptes clau:
    • Utilitza > per crear o sobreescriure un fitxer.
    • Utilitza un bucle while amb read per llegir un fitxer línia per línia.

Operacions amb directoris

Realitza operacions en directoris, com crear, eliminar i navegar.

Creant un directori

mkdir my_directory

Canviant de directori

cd my_directory
  • Conceptes clau:
    • Utilitza mkdir per crear un directori.
    • Utilitza cd per canviar el directori actual.

Conclusió

En aquesta secció, hem cobert tècniques avançades de scripting Bash, incloent funcions, arrays i constructs de bucles. També hem explorat la gestió d'errors, la depuració i les operacions amb fitxers i directoris. En dominar aquests conceptes, estaràs ben preparat per escriure scripts Bash més eficients i potents. Continua practicant i experimentant amb aquestes tècniques per continuar el teu viatge amb Bash!
