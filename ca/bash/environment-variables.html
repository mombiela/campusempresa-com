Introducció a les variables d'entorn

Les variables d'entorn són valors dinàmics que afecten els processos o programes en un ordinador. Es poden utilitzar per emmagatzemar informació com la ubicació dels executables, l'editor per defecte o la configuració local del sistema.

Conceptes clau

  • Variable d'entorn: Una variable que està disponible a tot el sistema i pot afectar el comportament dels processos en execució.
  • Variable de shell: Una variable que només està disponible en la sessió de shell actual.
  • Exportar variables: Fer que una variable de shell estigui disponible per als processos fills.

Variables d'entorn comunes

Aquí hi ha algunes variables d'entorn comunament utilitzades en Bash:

| Variable | Descripció | |----------|-------------| | PATH | Especifica els directoris en què es troben els programes executables. | | HOME | El directori personal de l'usuari actual. | | USER | El nom de l'usuari actual. | | SHELL | El camí al shell de l'usuari actual. | | PWD | El directori de treball actual. | | LANG | La configuració local actual. |

Veure les variables d'entorn

Per veure les variables d'entorn, pots utilitzar la comanda printenv o env.

# Mostra totes les variables d'entorn
printenv

# Mostra una variable d'entorn específica
printenv PATH

# Una altra manera de mostrar totes les variables d'entorn
env

Configurar variables d'entorn

Pots configurar variables d'entorn en Bash utilitzant la comanda export.

# Configura una nova variable d'entorn
export MY_VAR="Hola, Món!"

# Verifica que la variable està configurada
echo $MY_VAR

Modificar variables d'entorn

Pots modificar les variables d'entorn existents reassignant-les.

# Afegeix un nou directori a la variable PATH
export PATH=$PATH:/new/directory

# Verifica que la variable PATH s'ha actualitzat
echo $PATH

Desconfigurar variables d'entorn

Per eliminar una variable d'entorn, utilitza la comanda unset.

# Desconfigura una variable d'entorn
unset MY_VAR

# Verifica que la variable està desconfigurada
echo $MY_VAR  # No hauria de mostrar res

Utilitzar variables d'entorn en scripts

Les variables d'entorn es poden utilitzar dins dels scripts de Bash per fer-los més dinàmics i flexibles.

#!/bin/bash

# Configura una variable d'entorn
export GREETING="Hola"

# Utilitza la variable d'entorn
echo "$GREETING, $USER!"

Millors pràctiques

  • Utilitza noms descriptius: Tria noms significatius per a les teves variables d'entorn per fer que els teus scripts siguin més fàcils d'entendre.
  • Evita sobreescriure variables del sistema: Sigues cautelós en modificar variables d'entorn del sistema com PATH o HOME.
  • Utilitza cometes: Envolta els valors de les variables amb cometes per gestionar correctament els espais i els caràcters especials.

Conclusió

Les variables d'entorn són una característica poderosa en Bash que et permet personalitzar i controlar el comportament del teu shell i scripts. Entenent com veure, configurar, modificar i desconfigurar aquestes variables, pots crear scripts més flexibles i dinàmics. Recorda seguir les millors pràctiques per evitar errors comuns i assegurar-te que els teus scripts siguin robustos i mantenibles.
