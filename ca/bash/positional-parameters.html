Introducció als Paràmetres Posicionals

Els paràmetres posicionals en Bash són una manera de passar arguments a un script o funció. S'accedeix a ells utilitzant variables especials com $1, $2, $3, i així successivament, on $1 és el primer argument, $2 és el segon, i així successivament. Aquests paràmetres et permeten crear scripts més dinàmics i flexibles.

Accés als Paràmetres Posicionals

Quan executes un script Bash, pots passar arguments a aquest. Aquests arguments són accessibles dins de l'script utilitzant paràmetres posicionals.

Exemple

#!/bin/bash
echo "Primer argument: $1"
echo "Segon argument: $2"

Si guardes aquest script com example.sh i l'executes amb:

./example.sh arg1 arg2

La sortida serà:

Primer argument: arg1
Segon argument: arg2

Paràmetres Posicionals Especials

Bash proporciona diversos paràmetres posicionals especials que tenen significats específics:

  • $0: El nom de l'script mateix.
  • $#: El nombre d'arguments passats a l'script.
  • $@: Tots els arguments passats a l'script.
  • $*: Tots els arguments passats a l'script com una sola paraula.
  • $$: El ID del procés de l'script.
  • $?: L'estat de sortida de l'última comanda executada.

Exemple

#!/bin/bash
echo "Nom de l'script: $0"
echo "Nombre d'arguments: $#"
echo "Tots els arguments (\$@): $@"
echo "Tots els arguments (\$*): $*"
echo "ID del procés: $$"
echo "Estat de sortida de l'última comanda: $?"

Si executes aquest script amb:

./example.sh arg1 arg2 arg3

La sortida serà:

Nom de l'script: ./example.sh
Nombre d'arguments: 3
Tots els arguments ($@): arg1 arg2 arg3
Tots els arguments ($*): arg1 arg2 arg3
ID del procés: <some_pid>
Estat de sortida de l'última comanda: 0

Desplaçament dels Paràmetres Posicionals

La comanda shift s'utilitza per desplaçar els paràmetres posicionals cap a l'esquerra. Això significa que $2 es converteix en $1, $3 es converteix en $2, i així successivament. Això és útil quan necessites processar arguments en un bucle.

Exemple

#!/bin/bash
while [ $# -gt 0 ]; do
  echo "Processant argument: $1"
  shift
done

Si executes aquest script amb:

./example.sh arg1 arg2 arg3

La sortida serà:

Processant argument: arg1
Processant argument: arg2
Processant argument: arg3

Ús dels Paràmetres Posicionals en Funcions

Els paràmetres posicionals també es poden utilitzar dins de funcions. Dins d'una funció, $1, $2, etc., es refereixen als arguments de la funció.

Exemple

#!/bin/bash
my_function() {
  echo "Primer argument: $1"
  echo "Segon argument: $2"
}

my_function arg1 arg2

La sortida serà:

Primer argument: arg1
Segon argument: arg2

Conclusió

Els paràmetres posicionals són un concepte fonamental en la programació d'scripts Bash que et permeten crear scripts flexibles i dinàmics. Entenent com accedir, manipular i utilitzar aquests paràmetres, pots gestionar una àmplia gamma de tasques d'scripting de manera més efectiva. Recorda utilitzar paràmetres especials com $@ i $* adequadament, i aprofita la comanda shift quan sigui necessari per processar arguments en bucles.
