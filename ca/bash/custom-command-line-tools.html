Introducció

Crear eines de línia de comandes personalitzades pot millorar significativament la teva productivitat i agilitzar tasques repetitives. En aquesta secció, explorarem com crear, gestionar i utilitzar eines de línia de comandes personalitzades utilitzant Bash.

Conceptes clau

  • Shebang (#!): La primera línia en un script que especifica l'intèrpret.
  • Permisos de l'script: Fer que els scripts siguin executables.
  • Arguments de línia de comandes: Passar arguments als scripts.
  • Variables d'entorn: Utilitzar i gestionar variables d'entorn.
  • Funcions: Crear blocs de codi reutilitzables dins dels scripts.
  • Gestió d'errors: Gestionar errors i codis de sortida.

Creació d'un script senzill

Exemple: Script de Hello World

#!/bin/bash
# Aquest és un script senzill de Hello World
echo "Hello, World!"
  • Explicació:
    • #!/bin/bash: Línia shebang que especifica que l'script s'ha d'executar amb Bash.
    • echo "Hello, World!": Imprimeix "Hello, World!" al terminal.

Fer l'script executable

chmod +x hello_world.sh
  • Explicació:
    • chmod +x hello_world.sh: Canvia els permisos del fitxer per fer l'script executable.

Arguments de línia de comandes

Exemple: Script de salutació amb arguments

#!/bin/bash
# Aquest script saluda l'usuari amb el seu nom
echo "Hello, $1!"
  • Explicació:
    • $1: Representa el primer argument passat a l'script.

Executar l'script amb arguments

./greeting.sh Alice
  • Sortida:
    • Hello, Alice!

Variables d'entorn

Exemple: Utilitzar variables d'entorn

#!/bin/bash
# Aquest script demostra l'ús de variables d'entorn
echo "L'usuari actual és $USER"
echo "El directori home és $HOME"
  • Explicació:
    • $USER: Variable d'entorn per a l'usuari actual.
    • $HOME: Variable d'entorn per al directori home.

Funcions en scripts

Exemple: Funció per sumar dos números

#!/bin/bash
# Funció per sumar dos números
add_numbers() {
  local sum=$(( $1 + $2 ))
  echo "La suma és: $sum"
}

# Cridar la funció
add_numbers 3 5
  • Explicació:
    • add_numbers() { ... }: Defineix una funció anomenada add_numbers.
    • local sum=$(( $1 + $2 )): Calcula la suma de dos arguments.
    • echo "La suma és: $sum": Imprimeix la suma.

Gestió d'errors

Exemple: Comprovar l'existència d'un fitxer

#!/bin/bash
# Aquest script comprova si un fitxer existeix
if [ -f "$1" ]; then
  echo "El fitxer $1 existeix."
else
  echo "El fitxer $1 no existeix."
  exit 1
fi
  • Explicació:
    • [ -f "$1" ]: Comprova si el fitxer especificat pel primer argument existeix.
    • exit 1: Surt de l'script amb un codi de sortida d'1 si el fitxer no existeix.

Conclusió

Crear eines de línia de comandes personalitzades en Bash pot millorar enormement el teu flux de treball automatitzant tasques repetitives i proporcionant solucions ràpides a problemes comuns. Entenent els conceptes bàsics de l'scripting, gestionant arguments de línia de comandes, utilitzant variables d'entorn, definint funcions i implementant la gestió d'errors, pots construir eines potents i eficients adaptades a les teves necessitats. Practica aquests conceptes amb diversos exemples per convertir-te en un expert en la creació de les teves pròpies eines de línia de comandes personalitzades.
