El Projecte Sobre nosaltres Contribuir Donacions Llicència

La prova de fitxers és un aspecte essencial de l'scripting en Bash. Et permet comprovar les propietats dels fitxers i directoris, com ara la seva existència, tipus i permisos. Aquest tema cobrirà els conceptes bàsics de la prova de fitxers, diversos operadors de prova de fitxers i exemples pràctics per ajudar-te a entendre com utilitzar aquestes proves de manera efectiva en els teus scripts.

Conceptes Clau

  • Operadors de Prova de Fitxers: Operadors especials utilitzats per provar diferents propietats dels fitxers i directoris.
  • Declaracions Condicionals: Utilitzades per realitzar accions basades en els resultats de les proves de fitxers.

Operadors de Prova de Fitxers

Bash proporciona diversos operadors de prova de fitxers que es poden utilitzar dins de declaracions condicionals per comprovar diverses propietats dels fitxers i directoris. Aquí tens alguns dels operadors de prova de fitxers més utilitzats:

| Operador | Descripció | |----------|--------------------------------------------| | -e | Comprova si el fitxer existeix | | -f | Comprova si el fitxer és un fitxer regular | | -d | Comprova si el fitxer és un directori | | -r | Comprova si el fitxer és llegible | | -w | Comprova si el fitxer és escripturable | | -x | Comprova si el fitxer és executable | | -s | Comprova si el fitxer no està buit | | -L | Comprova si el fitxer és un enllaç simbòlic| | -p | Comprova si el fitxer és una canonada amb nom| | -S | Comprova si el fitxer és un socket |

Ús dels Operadors de Prova de Fitxers

Els operadors de prova de fitxers s'utilitzen típicament dins de declaracions if per realitzar accions basades en els resultats de la prova. Aquí tens alguns exemples per il·lustrar el seu ús:

Exemple 1: Comprovar si un Fitxer Existeix

#!/bin/bash

FILE="example.txt"

if [ -e "$FILE" ]; then
    echo "El fitxer '$FILE' existeix."
else
    echo "El fitxer '$FILE' no existeix."
fi

Exemple 2: Comprovar si un Fitxer és un Fitxer Regular

#!/bin/bash

FILE="example.txt"

if [ -f "$FILE" ]; then
    echo "'$FILE' és un fitxer regular."
else
    echo "'$FILE' no és un fitxer regular."
fi

Exemple 3: Comprovar si un Directori Existeix

#!/bin/bash

DIR="example_dir"

if [ -d "$DIR" ]; then
    echo "El directori '$DIR' existeix."
else
    echo "El directori '$DIR' no existeix."
fi

Exemple 4: Comprovar els Permisos d'un Fitxer

#!/bin/bash

FILE="example.txt"

if [ -r "$FILE" ]; then
    echo "El fitxer '$FILE' és llegible."
else
    echo "El fitxer '$FILE' no és llegible."
fi

if [ -w "$FILE" ]; then
    echo "El fitxer '$FILE' és escripturable."
else
    echo "El fitxer '$FILE' no és escripturable."
fi

if [ -x "$FILE" ]; then
    echo "El fitxer '$FILE' és executable."
else
    echo "El fitxer '$FILE' no és executable."
fi

Exemple 5: Comprovar si un Fitxer no està Buit

#!/bin/bash

FILE="example.txt"

if [ -s "$FILE" ]; then
    echo "El fitxer '$FILE' no està buit."
else
    echo "El fitxer '$FILE' està buit."
fi

Exercicis Pràctics

Exercici 1: Comprovació de l'Existència d'un Fitxer

Escriu un script que comprovi si un fitxer anomenat data.txt existeix en el directori actual. Si existeix, imprimeix "El fitxer existeix." En cas contrari, imprimeix "El fitxer no existeix."

Exercici 2: Comprovació d'un Directori

Escriu un script que comprovi si un directori anomenat backup existeix. Si existeix, imprimeix "El directori existeix." En cas contrari, crea el directori i imprimeix "Directori creat."

Exercici 3: Comprovació de Permisos d'un Fitxer

Escriu un script que comprovi si un fitxer anomenat script.sh és llegible, escripturable i executable. Imprimeix missatges apropiats per a cada comprovació.

Conclusió

La prova de fitxers en Bash és una característica potent que et permet escriure scripts més robustos i flexibles. Utilitzant operadors de prova de fitxers, pots comprovar diverses propietats dels fitxers i directoris i realitzar accions basades en aquestes comprovacions. Practica utilitzant aquests operadors en els teus scripts per esdevenir més proficient en l'scripting en Bash.
