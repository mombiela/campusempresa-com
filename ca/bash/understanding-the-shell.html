Introducció a la Shell

La shell és una interfície de línia de comandes que permet als usuaris interactuar amb el sistema operatiu. És una eina poderosa per gestionar fitxers, executar programes i automatitzar tasques.

  • Què és una Shell?

    • Una shell és un programa que pren comandes del teclat i les dóna al sistema operatiu per executar-les.
    • Tipus comuns de shells inclouen Bash (Bourne Again SHell), Zsh i Fish.

  • Per què Utilitzar la Shell?

    • Gestió eficient de fitxers.
    • Automatització de tasques repetitives.
    • Capacitats de scripting poderoses.
    • Gestió remota de sistemes.

Comandes Bàsiques de la Shell

Entendre les comandes bàsiques de la shell és essencial per navegar i manipular el sistema de fitxers.

  • Navegar pel Sistema de Fitxers

    • pwd: Imprimir el Directori de Treball

      pwd

      Aquesta comanda mostra el directori actual en què et trobes.

    • ls: Llistar el Contingut del Directori

      ls

      Aquesta comanda llista els fitxers i directoris en el directori actual.

    • cd: Canviar de Directori

      cd /path/to/directory

      Aquesta comanda canvia el directori actual al camí especificat.

  • Operacions amb Fitxers i Directoris

    • touch: Crear un Nou Fitxer

      touch filename.txt

      Aquesta comanda crea un nou fitxer buit anomenat filename.txt.

    • mkdir: Crear un Nou Directori

      mkdir new_directory

      Aquesta comanda crea un nou directori anomenat new_directory.

    • rm: Eliminar Fitxers o Directoris

      rm filename.txt

      Aquesta comanda elimina el fitxer anomenat filename.txt.

      rm -r directory_name

      Aquesta comanda elimina el directori anomenat directory_name i el seu contingut.

    • cp: Copiar Fitxers o Directoris

      cp source_file.txt destination_file.txt

      Aquesta comanda copia source_file.txt a destination_file.txt.

    • mv: Moure o Reanomenar Fitxers o Directoris

      mv old_name.txt new_name.txt

      Aquesta comanda reanomena old_name.txt a new_name.txt.

Conceptes Bàsics de Scripting en la Shell

El scripting en la shell permet automatitzar tasques escrivint scripts.

  • Crear un Script de Shell

    • Un script de shell és un fitxer que conté una sèrie de comandes.
    • Comença amb un shebang (#!) seguit del camí a la shell. 
      #!/bin/bash
echo "Hola, Món!"

  • Executar un Script de Shell

    • Fes el script executable. 
      chmod +x script.sh
    • Executa el script. 
      ./script.sh

  • Variables i Comentaris

    • Les variables emmagatzemen dades que es poden utilitzar més tard en el script. 
      #!/bin/bash
name="Joan"
echo "Hola, $name"
    • Els comentaris són línies que no s'executen. 
      # Aquest és un comentari
echo "Això s'executarà"

Funcions Avançades de la Shell

A mesura que et sentis més còmode amb la shell, pots explorar funcions més avançades.

  • Pipelines i Redirecció

    • Els pipelines (|) permeten passar la sortida d'una comanda com a entrada a una altra. 
      ls | grep "patró"
    • La redirecció (>, >>, <) permet redirigir la sortida o entrada de les comandes. 
      echo "Hola, Món!" > output.txt

  • Sentències Condicionals

    • Utilitza sentències if per executar comandes basades en condicions. 
      #!/bin/bash
if [ -f "file.txt" ]; then
  echo "El fitxer existeix"
else
  echo "El fitxer no existeix"
fi

  • Bucles

    • Utilitza bucles per repetir comandes. 
      #!/bin/bash
for i in {1..5}; do
  echo "Iteració $i"
done

Conclusió

Entendre la shell és fonamental per a qualsevol programador o administrador de sistemes. Proporciona eines poderoses per gestionar fitxers, automatitzar tasques i escriure scripts. Dominant les comandes bàsiques, el scripting i les funcions avançades, pots millorar significativament la teva productivitat i eficiència en un entorn similar a Unix.
