La gestió de directoris és una habilitat fonamental quan es treballa amb Bash. Aquest tema cobrirà ordres essencials i tècniques per navegar i manipular directoris en un sistema operatiu similar a Unix.

Navegant per Directoris

Navegar a través de directoris és una de les primeres tasques que necessitaràs dominar en Bash.

`pwd` - Imprimir el Directori de Treball

L'ordre pwd mostra el directori actual en què et trobes.

pwd

Exemple de sortida:

/home/user

`cd` - Canviar de Directori

L'ordre cd s'utilitza per canviar el directori actual.

cd /path/to/directory

Exemple:

cd /home/user/Documents

`ls` - Llistar el Contingut del Directori

L'ordre ls llista el contingut d'un directori.

ls

Exemple:

ls /home/user/Documents

Opcions Comunes de `ls`

  • -l: Format de llista llarga
  • -a: Incloure fitxers ocults
  • -h: Mides de fitxers llegibles per humans

Exemple:

ls -lah /home/user/Documents

Creació i Eliminació de Directoris

`mkdir` - Crear Directori

L'ordre mkdir crea un nou directori.

mkdir new_directory

Exemple:

mkdir /home/user/new_folder

`rmdir` - Eliminar Directori

L'ordre rmdir elimina un directori buit.

rmdir empty_directory

Exemple:

rmdir /home/user/old_folder

`rm -r` - Eliminar Directori i Contingut

L'ordre rm -r elimina un directori i el seu contingut de manera recursiva.

rm -r directory_with_contents

Exemple:

rm -r /home/user/folder_to_delete

Moure i Canviar el Nom dels Directoris

`mv` - Moure o Canviar el Nom

L'ordre mv s'utilitza per moure o canviar el nom dels directoris.

mv old_directory new_directory

Exemple:

mv /home/user/old_folder /home/user/new_folder

Copiar Directoris

`cp -r` - Copiar Directori i Contingut

L'ordre cp -r copia un directori i el seu contingut de manera recursiva.

cp -r source_directory destination_directory

Exemple:

cp -r /home/user/source_folder /home/user/destination_folder

Veure l'Estructura del Directori

`tree` - Vista en Forma d'Arbre del Directori

L'ordre tree mostra els directoris i fitxers en un format semblant a un arbre. Tingues en compte que tree podria necessitar ser instal·lat al teu sistema.

tree

Exemple:

tree /home/user/Documents

Resum

En aquest tema, hem cobert les ordres essencials per a la gestió de directoris en Bash:

  • pwd per imprimir el directori de treball actual.
  • cd per canviar de directoris.
  • ls per llistar el contingut del directori.
  • mkdir per crear nous directoris.
  • rmdir i rm -r per eliminar directoris.
  • mv per moure o canviar el nom dels directoris.
  • cp -r per copiar directoris.
  • tree per veure estructures de directoris.

Dominar aquestes ordres millorarà significativament la teva capacitat per navegar i gestionar el sistema de fitxers de manera eficient. Practica aquestes ordres per sentir-te còmode amb la gestió de directoris en Bash.
