Introducció

La gestió d'errors i la depuració són habilitats crucials per a qualsevol programador de Bash. Aquesta secció cobrirà els conceptes bàsics de la gestió d'errors, tècniques comunes de depuració i mètodes avançats per assegurar que els teus scripts funcionin de manera suau i eficient.

Gestió d'Errors

Gestió Bàsica d'Errors

  • Estat de Sortida: Cada comanda retorna un estat de sortida (0 per èxit, no zero per fallada).

    ls /directori_inexistent
echo $?  # Mostra: 2 (o un altre valor no zero indicant un error)

  • Ús de if Statements: Comprova l'estat de sortida de les comandes.

    if ls /directori_inexistent; then
  echo "El directori existeix"
else
  echo "El directori no existeix"
fi

Gestió Avançada d'Errors

  • Comanda set: Modifica el comportament de la shell per gestionar errors de manera més efectiva.

    • set -e: Surt immediatament si una comanda surt amb un estat no zero.
    • set -u: Tracta les variables no establertes com un error.
    • set -o pipefail: Retorna l'estat de sortida de l'última comanda en la pipeline que ha fallat.
    set -euo pipefail

mkdir /algún_directori
cd /algún_directori

  • Comanda Trap: Executa una comanda quan l'script surt o rep un senyal.

    trap 'echo "S'ha produït un error. Sortint..."; exit 1' ERR

Tècniques de Depuració

Depuració Bàsica

  • Declaracions Echo: Imprimeix valors de variables i el progrés de l'script.

    echo "Iniciant l'script..."
echo "Valor de la variable: $variable"

  • set -x: Habilita el mode de depuració per imprimir cada comanda abans d'executar-la.

    set -x
ls /algún_directori
set +x  # Deshabilita el mode de depuració

Depuració Avançada

  • Ús de trap per a la Depuració: Captura i depura errors.

    trap 'echo "Error a la línia $LINENO"; exit 1' ERR

  • Variable PS4: Personalitza el prompt de depuració.

    export PS4='+ $BASH_SOURCE:$LINENO: '
set -x
ls /algún_directori
set +x

Errors Comuns i Solucions

Errors de Sintaxi

  • Cometes Faltants: Assegura't que les cadenes estiguin correctament entre cometes.

    echo "Hola, Món"  # Correcte
echo Hola, Món    # Incorrecte

  • Ús Incorrecte de Variables: Utilitza $ per referenciar variables.

    nom="Joan"
echo $nom  # Correcte
echo nom   # Incorrecte

Errors Lògics

  • Condicionals Incorrectes: Assegura't que les condicions estiguin correctament escrites. 
    if [ "$valor" -eq 10 ]; then
  echo "El valor és 10"
fi

Errors d'Execució

  • Fitxer No Trobat: Comprova l'existència del fitxer abans d'accedir-hi. 
    if [ -f "/camí/al/fitxer" ]; then
  cat /camí/al/fitxer
else
  echo "Fitxer no trobat"
fi

Conclusió

La gestió d'errors i la depuració són habilitats essencials per escriure scripts de Bash robustos. Comprenent els estats de sortida, utilitzant les comandes set i trap, i emprant tècniques de depuració com set -x i declaracions echo, pots gestionar i solucionar efectivament els errors en els teus scripts. Practica aquestes tècniques per millorar la teva competència en scripting i assegurar que els teus scripts funcionin sense problemes.
