Les variables especials a Bash són variables predefinides que proporcionen informació útil sobre l'entorn, el script o la línia de comandes. Són essencials per escriure scripts robustos i flexibles.

Variables especials clau

`$0`

  • Representa el nom del script que s'està executant.
  • Útil per mostrar el nom del script en missatges o registres.
#!/bin/bash
echo "El nom del script és: $0"

`$1, $2, ..., $N`

  • Representen els paràmetres posicionals passats al script.
  • $1 és el primer argument, $2 és el segon, i així successivament.
#!/bin/bash
echo "Primer argument: $1"
echo "Segon argument: $2"

`$#`

  • Representa el nombre de paràmetres posicionals passats al script.
  • Útil per comprovar si s'ha proporcionat el nombre correcte d'arguments.
#!/bin/bash
if [ $# -ne 2 ]; then
  echo "Ús: $0 arg1 arg2"
  exit 1
fi

`$@` i `$*`

  • Ambdues representen tots els paràmetres posicionals passats al script.
  • $@ tracta cada paràmetre com una paraula separada.
  • $* tracta tots els paràmetres com una sola paraula.
#!/bin/bash
echo "Tots els arguments usant \$@: $@"
echo "Tots els arguments usant \$*: $*"

`$?`

  • Representa l'estat de sortida de l'última comanda executada.
  • Un valor de 0 indica èxit, mentre que qualsevol altre valor indica un error.
#!/bin/bash
ls /directori_inexistent
echo "Estat de sortida de l'última comanda: $?"

`$$`

  • Representa l'identificador de procés (PID) del propi script.
  • Útil per crear fitxers temporals únics.
#!/bin/bash
tempfile="/tmp/tempfile_$$"
echo "Fitxer temporal: $tempfile"

`$!`

  • Representa l'identificador de procés de l'última comanda en segon pla.
  • Útil per fer el seguiment de processos en segon pla.
#!/bin/bash
sleep 60 &
echo "PID del procés en segon pla: $!"

`$-`

  • Representa les opcions actuals establertes per la shell.
  • Útil per depurar i entendre el comportament de la shell.
#!/bin/bash
echo "Opcions actuals de la shell: $-"

`$_`

  • Representa l'últim argument de la comanda anterior.
  • Útil per reutilitzar l'últim argument.
#!/bin/bash
echo "Primera comanda"
echo "Segona comanda amb l'últim argument: $_"

Comparació de `$@` i `$*`

| Variable | Descripció | Exemple de sortida | |----------|-------------|--------------------| | $@ | Tracta cada paràmetre com una paraula separada | "arg1" "arg2" "arg3" | | $* | Tracta tots els paràmetres com una sola paraula | "arg1 arg2 arg3" |

Conclusió

Les variables especials a Bash proporcionen maneres poderoses d'interactuar amb l'entorn del script i els arguments de la línia de comandes. Entendre i utilitzar aquestes variables de manera efectiva pot millorar significativament la flexibilitat i la robustesa dels teus scripts Bash. Practica utilitzant aquestes variables en diferents escenaris per convertir-te en un expert en l'escriptura de scripts Bash avançats.
