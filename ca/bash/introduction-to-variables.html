Què són les Variables en Bash?

Les variables en Bash s'utilitzen per emmagatzemar dades que es poden referenciar i manipular al llarg d'un script. Són fonamentals per a la creació de scripts i l'automatització, permetent crear scripts dinàmics i flexibles.

  • Definició: Una variable és una ubicació d'emmagatzematge amb nom que conté un valor.
  • Propòsit: Emmagatzemar dades que es poden utilitzar i manipular dins d'un script.

Declaració i Assignació de Variables

En Bash, declares i assignes valors a les variables utilitzant l'operador =. A diferència d'altres llenguatges de programació, no cal especificar el tipus de dades.

Sintaxi

variable_name=value
  • Sense espais: Assegura't que no hi hagi espais al voltant del signe =.
  • Sensibilitat a majúscules i minúscules: Els noms de les variables distingeixen entre majúscules i minúscules (VAR i var són diferents).

Exemples

# Manera correcta de declarar i assignar una variable
greeting="Hola, Món!"

# Manera incorrecta (causarà un error)
greeting = "Hola, Món!"

Accés als Valors de les Variables

Per accedir al valor emmagatzemat en una variable, utilitzes el símbol $ seguit del nom de la variable.

Exemples

# Declaració d'una variable
name="Alice"

# Accés a la variable
echo "Hola, $name!"

Sortida:

Hola, Alice!

Millors Pràctiques per Nomenar Variables

  • Noms descriptius: Utilitza noms significatius que descriguin les dades emmagatzemades.
  • Evita caràcters especials: Utilitza només lletres, números i guions baixos (_).
  • Comença amb una lletra: Els noms de les variables han de començar amb una lletra o un guió baix.

Exemples

# Bons noms de variables
user_name="Bob"
file_count=10

# Mals noms de variables
1user="Charlie"  # Comença amb un número
file-count=5     # Conté un guionet

Tipus de Variables

Variables Locals

Les variables locals només són accessibles dins del bloc de codi o funció on es declaren.

function greet {
    local message="Hola, $1"
    echo $message
}

greet "Alice"

Variables Globals

Les variables globals són accessibles des de qualsevol lloc de l'script després de ser declarades.

greeting="Hola, Món!"

function greet {
    echo $greeting
}

greet

Variables Especials

Bash proporciona diverses variables especials que tenen significats predefinits.

| Variable | Descripció | |----------|-------------------------------------| | $0 | El nom de l'script | | $1, $2, ... | Paràmetres posicionals (arguments) | | $# | Nombre de paràmetres posicionals | | $@ | Tots els paràmetres posicionals | | $? | Estat de sortida de l'última comanda| | $$ | Identificador de procés del shell actual |

Exemples

# Nom de l'script
echo "Nom de l'script: $0"

# Primer argument
echo "Primer argument: $1"

# Nombre d'arguments
echo "Nombre d'arguments: $#"

Conclusió

Les variables són un concepte fonamental en la creació de scripts en Bash, permetent emmagatzemar i manipular dades de manera eficient. Entenent com declarar, assignar i accedir a les variables, pots crear scripts més dinàmics i flexibles. Recorda seguir les millors pràctiques per nomenar variables i estar al corrent dels diferents tipus de variables i variables especials disponibles en Bash.

A la següent secció, explorarem com utilitzar variables en escenaris més complexos i introduirem estructures de control per fer els teus scripts encara més potents.
