Sentències Case en Bash
Funcions en Bash ►

Introducció

Les sentències case en Bash són una eina poderosa per gestionar múltiples condicions. Permeten executar diferents blocs de codi basats en el valor d'una variable, fent que els teus scripts siguin més llegibles i mantenibles.

Conceptes Clau

  • Sintaxi: L'estructura bàsica d'una sentència case.
  • Patrons: Com coincidir amb diferents patrons.
  • Caiguda: Entendre com les sentències case gestionen múltiples coincidències.
  • Cas per Defecte: Utilitzar el cas per defecte per gestionar patrons no coincidents.

Sintaxi

La sintaxi bàsica d'una sentència case en Bash és la següent:

case expression in
    pattern1)
        # commands
        ;;
    pattern2)
        # commands
        ;;
    *)
        # default commands
        ;;
esac
  • expression: La variable o valor que estàs provant.
  • pattern1, pattern2, etc.: Patrons per coincidir amb l'expressió.
  • ;;: Acaba cada bloc de comandes.
  • *): El cas per defecte, executat si cap patró coincideix.
  • esac: Acaba la sentència case.

Exemples

Exemple Bàsic

Aquí tens un exemple senzill per il·lustrar com funcionen les sentències case:

#!/bin/bash

echo "Introdueix un número entre 1 i 3:"
read number

case $number in
    1)
        echo "Has introduït un."
        ;;
    2)
        echo "Has introduït dos."
        ;;
    3)
        echo "Has introduït tres."
        ;;
    *)
        echo "Número invàlid."
        ;;
esac

Utilitzant Patrons

Pots utilitzar comodins i rangs en els patrons:

#!/bin/bash

echo "Introdueix un caràcter:"
read char

case $char in
    [a-z])
        echo "Has introduït una lletra minúscula."
        ;;
    [A-Z])
        echo "Has introduït una lletra majúscula."
        ;;
    [0-9])
        echo "Has introduït un dígit."
        ;;
    *)
        echo "Has introduït un caràcter especial."
        ;;
esac

Múltiples Patrons

Pots coincidir amb múltiples patrons amb un sol bloc de comandes:

#!/bin/bash

echo "Introdueix un dia de la setmana:"
read day

case $day in
    "Monday"|"Tuesday"|"Wednesday"|"Thursday"|"Friday")
        echo "És un dia laborable."
        ;;
    "Saturday"|"Sunday")
        echo "És el cap de setmana."
        ;;
    *)
        echo "Dia invàlid."
        ;;
esac

Ús Avançat

Sentències Case Imbricades

Pots imbricar sentències case dins d'altres per a una lògica més complexa:

#!/bin/bash

echo "Introdueix una fruita (poma, plàtan, cirera):"
read fruit

case $fruit in
    "poma")
        echo "Has triat poma."
        ;;
    "plàtan")
        echo "Has triat plàtan."
        ;;
    "cirera")
        echo "Has triat cirera."
        ;;
    *)
        echo "Fruita desconeguda."
        ;;
esac

Utilitzant Variables en Patrons

Pots utilitzar variables dins dels patrons per fer les teves sentències case més dinàmiques:

#!/bin/bash

echo "Introdueix una extensió de fitxer (per exemple, .txt, .sh):"
read extension

case $extension in
    *.txt)
        echo "Això és un fitxer de text."
        ;;
    *.sh)
        echo "Això és un script de shell."
        ;;
    *)
        echo "Tipus de fitxer desconegut."
        ;;
esac

Conclusió

Les sentències case en Bash proporcionen una manera flexible i llegible de gestionar múltiples condicions. Entenent la sintaxi, utilitzant patrons de manera efectiva i aprofitant funcions avançades com les sentències case imbricades i els patrons de variables, pots escriure scripts més eficients i mantenibles. Practica amb els exemples proporcionats i intenta crear les teves pròpies sentències case per consolidar la teva comprensió.
