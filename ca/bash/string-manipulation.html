Manipulació de Cadenes en Bash
Arrays en Bash ►

La manipulació de cadenes és una habilitat fonamental en la programació de scripts en Bash. Aquest tema et guiarà des d'operacions bàsiques fins a avançades amb cadenes, proporcionant explicacions clares, exemples i exercicis per consolidar la teva comprensió.

Operacions Bàsiques amb Cadenes

Concatenació

La concatenació és el procés d'unir dues o més cadenes.

str1="Hello"
str2="World"
result="$str1 $str2"
echo $result  # Output: Hello World

Longitud de la Cadena

Pots trobar la longitud d'una cadena utilitzant la sintaxi ${#string}.

str="Hello"
length=${#str}
echo $length  # Output: 5

Extracció de Subcadena

Extreu una subcadena utilitzant la sintaxi ${string:position:length}.

str="Hello World"
substr=${str:6:5}
echo $substr  # Output: World

Operacions Intermèdies amb Cadenes

Reemplaçament de Cadena

Reemplaça una subcadena dins d'una cadena utilitzant la sintaxi ${string/substring/replacement}.

str="Hello World"
new_str=${str/World/Bash}
echo $new_str  # Output: Hello Bash

Conversió de Majúscules i Minúscules

Converteix cadenes a majúscules o minúscules utilitzant tr.

str="Hello World"
uppercase=$(echo $str | tr '[:lower:]' '[:upper:]')
echo $uppercase  # Output: HELLO WORLD

lowercase=$(echo $str | tr '[:upper:]' '[:lower:]')
echo $lowercase  # Output: hello world

Divisió de Cadena

Divideix una cadena en un array basat en un delimitador.

str="apple,banana,cherry"
IFS=',' read -r -a array <<< "$str"
echo ${array[0]}  # Output: apple
echo ${array[1]}  # Output: banana
echo ${array[2]}  # Output: cherry

Operacions Avançades amb Cadenes

Expressions Regulars

Utilitza expressions regulars per a coincidències i manipulacions complexes de cadenes.

str="Hello123"
if [[ $str =~ [0-9]+ ]]; then
  echo "String contains numbers"
else
  echo "String does not contain numbers"
fi
# Output: String contains numbers

Comparació de Cadenes

Compara cadenes utilitzant expressions condicionals.

str1="Hello"
str2="World"

if [ "$str1" == "$str2" ]; then
  echo "Strings are equal"
else
  echo "Strings are not equal"
fi
# Output: Strings are not equal

Reemplaçament Avançat de Cadena

Utilitza l'expansió de paràmetres per a reemplaçaments més complexos.

str="Hello World World"
new_str=${str//World/Bash}
echo $new_str  # Output: Hello Bash Bash

Resum

La manipulació de cadenes en Bash és una eina poderosa per a qualsevol programador. Des d'operacions bàsiques com la concatenació i el càlcul de longitud fins a tècniques més avançades com les expressions regulars i els reemplaçaments complexos, dominar aquestes habilitats millorarà enormement les teves capacitats de scripting. Practica els exemples i exercicis proporcionats per esdevenir proficient en la gestió de cadenes en Bash.
