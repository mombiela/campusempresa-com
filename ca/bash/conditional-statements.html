Introducció

Les declaracions condicionals són fonamentals en qualsevol llenguatge de programació, ja que permeten executar diferents blocs de codi basats en certes condicions. En Bash, les declaracions condicionals ajuden a controlar el flux dels teus scripts, fent-los més dinàmics i sensibles a diferents entrades o estats.

Conceptes clau

  • Declaracions if-else: Estructura condicional bàsica.
  • Declaracions elif: Afegint múltiples condicions.
  • Declaracions case: Gestionant múltiples condicions de manera més eficient.
  • Comanda test: Utilitzada per avaluar condicions.
  • Operadors lògics: Combinació de múltiples condicions.

Declaracions if-else

La declaració if-else et permet executar un bloc de codi si una condició és certa, i un altre bloc si és falsa.

Sintaxi

if [ condition ]; then
    # Codi a executar si la condició és certa
else
    # Codi a executar si la condició és falsa
fi

Exemple

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 10 ]; then
    echo "The number is greater than 10."
else
    echo "The number is 10 or less."
fi

En aquest exemple, l'script llegeix un número de l'usuari i comprova si és més gran que 10.

Declaracions elif

La declaració elif et permet comprovar múltiples condicions seqüencialment.

Sintaxi

if [ condition1 ]; then
    # Codi a executar si la condició1 és certa
elif [ condition2 ]; then
    # Codi a executar si la condició2 és certa
else
    # Codi a executar si cap de les condicions és certa
fi

Exemple

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 10 ]; then
    echo "The number is greater than 10."
elif [ $number -eq 10 ]; then
    echo "The number is exactly 10."
else
    echo "The number is less than 10."
fi

Aquest script comprova si el número és més gran que, igual a, o menor que 10.

Declaracions case

La declaració case és útil per gestionar múltiples condicions de manera més eficient, especialment quan tens molts valors possibles a comprovar.

Sintaxi

case $variable in
    pattern1)
        # Codi a executar si la variable coincideix amb el pattern1
        ;;
    pattern2)
        # Codi a executar si la variable coincideix amb el pattern2
        ;;
    *)
        # Codi a executar si la variable no coincideix amb cap pattern
        ;;
esac

Exemple

#!/bin/bash

read -p "Enter a character: " char

case $char in
    [a-z])
        echo "You entered a lowercase letter."
        ;;
    [A-Z])
        echo "You entered an uppercase letter."
        ;;
    [0-9])
        echo "You entered a digit."
        ;;
    *)
        echo "You entered a special character."
        ;;
esac

Aquest script categoritza el caràcter d'entrada com una lletra minúscula, lletra majúscula, dígit o caràcter especial.

Comanda test

La comanda test (o la sintaxi [ ]) s'utilitza per avaluar condicions.

Expressions de test comunes

  • Comparacions de cadenes:
    • [ "$str1" = "$str2" ]: Cert si les cadenes són iguals.
    • [ "$str1" != "$str2" ]: Cert si les cadenes no són iguals.
  • Comparacions numèriques:
    • [ $num1 -eq $num2 ]: Cert si els números són iguals.
    • [ $num1 -ne $num2 ]: Cert si els números no són iguals.
    • [ $num1 -gt $num2 ]: Cert si el num1 és més gran que el num2.
    • [ $num1 -lt $num2 ]: Cert si el num1 és menor que el num2.
  • Tests de fitxers:
    • [ -e $file ]: Cert si el fitxer existeix.
    • [ -d $directory ]: Cert si el directori existeix.
    • [ -f $file ]: Cert si el fitxer és un fitxer regular.

Exemple

#!/bin/bash

read -p "Enter a filename: " filename

if [ -e $filename ]; then
    echo "The file exists."
else
    echo "The file does not exist."
fi

Aquest script comprova si un fitxer existeix.

Operadors lògics

Els operadors lògics et permeten combinar múltiples condicions.

Operadors lògics comuns

  • AND (&&): Ambdues condicions han de ser certes.
  • OR (||): Almenys una condició ha de ser certa.
  • NOT (!): Nega la condició.

Exemple

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 0 ] && [ $number -lt 100 ]; then
    echo "The number is between 1 and 99."
else
    echo "The number is not in the range 1-99."
fi

Aquest script comprova si el número està entre 1 i 99.

Conclusió

Les declaracions condicionals en Bash són essencials per controlar el flux dels teus scripts. Dominant if-else, elif, case i la comanda test, juntament amb els operadors lògics, pots crear scripts més dinàmics i sensibles. Practica aquests conceptes amb diversos exemples per convertir-te en un expert en escriure scripts Bash que gestionin diferents condicions de manera efectiva.
