Introducció a Bash

Bash, abreviatura de "Bourne Again SHell," és una shell de Unix i un llenguatge de comandes escrit per Brian Fox per al Projecte GNU com a substitut de programari lliure per a la shell Bourne (sh). Està àmpliament disponible en diversos sistemes operatius, incloent Linux i macOS, i es pot instal·lar en Windows.

Conceptes Clau

  • Shell: Una shell és un intèrpret de línia de comandes que proporciona una interfície d'usuari per al sistema operatiu Unix. Permet als usuaris executar comandes, executar scripts i gestionar processos del sistema.
  • Llenguatge de Comandes: Bash inclou un llenguatge de script que es pot utilitzar per automatitzar tasques, manipular fitxers i realitzar tasques d'administració del sistema.
  • Projecte GNU: Una iniciativa per desenvolupar un sistema operatiu similar a Unix lliure, del qual Bash en forma part.

Característiques de Bash

Bash ofereix diverses característiques que el fan una eina poderosa tant per a l'ús interactiu com per a l'escriptura de scripts:

  • Historial de Comandes: Permet als usuaris recordar i reutilitzar comandes executades prèviament.
  • Compleció de Tabulació: Ajuda a completar automàticament noms de fitxers i comandes.
  • Escriptura de Scripts: Suporta l'escriptura de scripts per automatitzar tasques repetitives.
  • Control de Tasques: Gestiona múltiples tasques o processos dins d'una sola sessió de shell.
  • Àlies: Permet als usuaris crear dreceres per a comandes més llargues.

Comandes Bàsiques de Bash

Aquí tens algunes comandes bàsiques de Bash per començar:

# Llistar fitxers i directoris
ls

# Canviar de directori
cd /path/to/directory

# Imprimir el directori de treball
pwd

# Copiar fitxers
cp source_file destination_file

# Moure o reanomenar fitxers
mv old_name new_name

# Eliminar fitxers o directoris
rm file_name
rm -r directory_name

# Mostrar el contingut d'un fitxer
cat file_name

# Crear un nou directori
mkdir directory_name

Exemple: Navegant pel Sistema de Fitxers

# Navegar al directori d'inici
cd ~

# Llistar tots els fitxers i directoris al directori d'inici
ls -a

# Crear un nou directori anomenat 'projects'
mkdir projects

# Canviar al directori 'projects'
cd projects

# Crear un nou fitxer anomenat 'example.txt'
touch example.txt

# Mostrar el camí del directori actual
pwd

Conceptes Bàsics de l'Escriptura de Scripts en Bash

Els scripts de Bash són fitxers de text que contenen una sèrie de comandes. Són executats per l'intèrpret de Bash.

Exemple: Un Script Senzill en Bash

#!/bin/bash
# Això és un comentari

# Imprimir un missatge al terminal
echo "Hola, Món!"

# Assignar un valor a una variable
name="Alice"

# Imprimir el valor de la variable
echo "Hola, $name!"

Executar un Script en Bash

  1. Crear un nou fitxer amb una extensió .sh, per exemple, hello.sh.

  2. Afegir el contingut de l'script al fitxer.

  3. Fer l'script executable:

    chmod +x hello.sh

  4. Executar l'script:

    ./hello.sh

Conclusió

Bash és una eina versàtil i poderosa per interactuar amb el sistema operatiu Unix. Proporciona un conjunt ric de característiques tant per a l'ús de línia de comandes com per a l'escriptura de scripts, fent-la una habilitat essencial per a administradors de sistemes, desenvolupadors i usuaris avançats. En dominar Bash, pots automatitzar tasques, gestionar fitxers i processos, i augmentar la teva productivitat en un entorn Unix.
