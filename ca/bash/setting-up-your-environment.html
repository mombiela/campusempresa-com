Configurar el teu entorn és el primer pas per convertir-te en un expert en la creació de scripts Bash. Aquesta secció et guiarà a través del procés de preparar el teu sistema, instal·lar les eines necessàries i configurar el teu entorn per a una experiència òptima en la creació de scripts Bash.

Prerequisits

Abans de començar amb la configuració, assegura't de tenir el següent:

  • Un ordinador amb un sistema operatiu tipus Unix (Linux, macOS o Windows amb WSL).
  • Coneixements bàsics d'interfícies de línia de comandes.

Instal·lant Bash

A Linux

La majoria de distribucions Linux vénen amb Bash preinstal·lat. Pots verificar-ho obrint una terminal i escrivint:

bash --version

Si Bash no està instal·lat, pots instal·lar-lo utilitzant el teu gestor de paquets. Per exemple, a Ubuntu:

sudo apt update
sudo apt install bash

A macOS

macOS també ve amb Bash preinstal·lat. Pots comprovar la versió obrint l'aplicació Terminal i escrivint:

bash --version

Si necessites actualitzar Bash, pots utilitzar Homebrew:

brew install bash

A Windows

Per als usuaris de Windows, la millor manera d'utilitzar Bash és a través del Subsistema Windows per a Linux (WSL). Segueix aquests passos:

  1. Activa WSL:

    • Obre PowerShell com a Administrador i executa:
    wsl --install

  2. Instal·la una distribució Linux:

    • Després d'activar WSL, instal·la una distribució Linux des de la Microsoft Store (per exemple, Ubuntu).

  3. Configura el teu entorn Linux:

    • Obre la distribució Linux instal·lada i segueix les instruccions en pantalla per completar la configuració.

Configurant el teu entorn

Configurant el teu editor de text

Escollir l'editor de text adequat és crucial per escriure i editar scripts Bash. Aquí tens algunes opcions populars:

  • Nano: Simple i fàcil d'utilitzar.
  • Vim: Potent i altament configurable, però amb una corba d'aprenentatge pronunciada.
  • VS Code: Ric en funcionalitats amb extensions per a la creació de scripts Bash.

Instal·lant Nano

Nano sol estar preinstal·lat en la majoria de sistemes tipus Unix. Si no, pots instal·lar-lo utilitzant:

sudo apt install nano  # Per a sistemes basats en Debian

Instal·lant Vim

Vim també sol estar preinstal·lat. Per instal·lar-lo:

sudo apt install vim  # Per a sistemes basats en Debian

Instal·lant VS Code

Per instal·lar Visual Studio Code:

  1. Descarrega i instal·la VS Code des de el lloc web oficial.
  2. Instal·la l'extensió "Bash IDE" per a ressaltat de sintaxi i altres funcionalitats.

Personalitzant la teva shell

Personalitzar la teva shell pot millorar la teva productivitat. Aquí tens algunes personalitzacions comunes:

  • Àlies: Dreceres per a comandes més llargues.
  • Personalització del prompt: Fes que el teu prompt sigui més informatiu.
  • Variables d'entorn: Configura variables que els teus scripts poden utilitzar.

Creant àlies

Pots crear àlies en el teu fitxer .bashrc. Per exemple:

alias ll='ls -la'

Després d'afegir l'àlies, recarrega el teu .bashrc:

source ~/.bashrc

Personalitzant el prompt

Pots personalitzar el teu prompt modificant la variable PS1 en el teu fitxer .bashrc. Per exemple:

PS1='\u@\h:\w\$ '

Això configura el prompt per mostrar el nom d'usuari, el nom de l'ordinador i el directori actual.

Configurant variables d'entorn

Pots configurar variables d'entorn en el teu fitxer .bashrc. Per exemple:

export PATH=$PATH:/your/custom/path

Conclusió

Configurar el teu entorn és un pas crucial per convertir-te en un expert en la creació de scripts Bash. Assegurant-te que tens Bash instal·lat, escollint l'editor de text adequat i personalitzant la teva shell, crearàs un espai de treball productiu i eficient. Amb el teu entorn preparat, ara pots passar a aprendre i dominar la creació de scripts Bash.
