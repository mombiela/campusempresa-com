Introducció

Els bucles són construccions fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En els scripts de Bash, hi ha tres tipus principals de bucles: for, while i until. Aquesta secció cobrirà cada tipus de bucle, proporcionarà exemples i inclourà exercicis per reforçar la teva comprensió.

Bucle For

El bucle for itera sobre una llista d'elements i executa un bloc de codi per a cada element.

Sintaxi

for item in list
do
    # Comandes a executar
done

Exemple

#!/bin/bash

# Llista d'elements
for fruit in apple banana cherry
do
    echo "M'agrada $fruit"
done

Explicació:

  • El bucle itera sobre la llista apple banana cherry.
  • Per a cada element de la llista, imprimeix "M'agrada [fruit]".

Exercici

Escriu un script que imprimeixi números de l'1 al 5 utilitzant un bucle for.

Bucle While

El bucle while executa un bloc de codi mentre una condició especificada sigui certa.

Sintaxi

while [ condition ]
do
    # Comandes a executar
done

Exemple

#!/bin/bash

count=1

while [ $count -le 5 ]
do
    echo "El compte és $count"
    ((count++))
done

Explicació:

  • El bucle continua mentre count sigui menor o igual a 5.
  • Imprimeix el valor actual de count i després incrementa count en 1.

Exercici

Escriu un script que imprimeixi números parells del 2 al 10 utilitzant un bucle while.

Bucle Until

El bucle until és similar al bucle while però executa un bloc de codi mentre una condició especificada sigui falsa.

Sintaxi

until [ condition ]
do
    # Comandes a executar
done

Exemple

#!/bin/bash

count=1

until [ $count -gt 5 ]
do
    echo "El compte és $count"
    ((count++))
done

Explicació:

  • El bucle continua fins que count sigui més gran que 5.
  • Imprimeix el valor actual de count i després incrementa count en 1.

Exercici

Escriu un script que imprimeixi números del 5 a l'1 utilitzant un bucle until.

Comparació de Bucles

| Tipus de Bucle | Comprovació de Condició | Cas d'Ús | |----------------|-------------------------|----------| | for | Itèra sobre una llista d'elements | Quan coneixes el nombre d'iteracions | | while | Executa mentre la condició sigui certa | Quan el nombre d'iteracions no es coneix per endavant | | until | Executa mentre la condició sigui falsa | Similar a while, però amb la condició invertida |

Conclusió

Els bucles són eines poderoses en els scripts de Bash que et permeten automatitzar tasques repetitives. El bucle for és ideal per iterar sobre una llista predefinida, mentre que els bucles while i until són més adequats per a situacions on el nombre d'iteracions depèn de condicions dinàmiques. Practica utilitzant aquests bucles amb els exercicis proporcionats per solidificar la teva comprensió.
