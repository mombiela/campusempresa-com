Introducció

En aquesta secció, cobrirem les comandes essencials de xarxa en Bash. Aquestes comandes són crucials per gestionar i solucionar problemes de connexions de xarxa en sistemes tipus Unix. Al final d'aquesta secció, estaràs familiaritzat amb comandes com ping, ifconfig, netstat, traceroute i més.

Comandes clau de xarxa

`ping`

La comanda ping s'utilitza per provar la connectivitat d'un host en una xarxa IP. Envia paquets de sol·licitud d'eco ICMP a l'host objectiu i espera una resposta d'eco ICMP.

Sintaxi

ping [opcions] destinació

Exemple

ping google.com

Aquesta comanda envia paquets a google.com i espera una resposta, ajudant-te a determinar si l'host és accessible.

`ifconfig`

La comanda ifconfig s'utilitza per configurar, controlar i consultar els paràmetres de la interfície de xarxa TCP/IP.

Sintaxi

ifconfig [interfície] [opcions]

Exemple

ifconfig

Aquesta comanda mostra totes les interfícies de xarxa actives i les seves configuracions.

`netstat`

La comanda netstat proporciona estadístiques de xarxa, incloent informació sobre connexions de xarxa, taules de routing, estadístiques d'interfície, connexions de màscara i membres de multicast.

Sintaxi

netstat [opcions]

Exemple

netstat -a

Aquesta comanda mostra totes les connexions actives i els ports d'escolta.

`traceroute`

La comanda traceroute mostra la ruta que prenen els paquets per arribar a un host de xarxa. Ajuda a diagnosticar problemes de routing de xarxa.

Sintaxi

traceroute [opcions] destinació

Exemple

traceroute google.com

Aquesta comanda mostra el camí que prenen els paquets per arribar a google.com.

`curl`

La comanda curl s'utilitza per transferir dades des d'un servidor o cap a un servidor utilitzant diversos protocols com HTTP, HTTPS, FTP i més.

Sintaxi

curl [opcions] [URL]

Exemple

curl http://example.com

Aquesta comanda obté el contingut de http://example.com.

`wget`

La comanda wget és un descarregador de xarxa no interactiu. Suporta protocols HTTP, HTTPS i FTP.

Sintaxi

wget [opcions] [URL]

Exemple

wget http://example.com/file.zip

Aquesta comanda descarrega el fitxer file.zip de http://example.com.

`nslookup`

La comanda nslookup consulta els servidors de noms de domini d'Internet per trobar adreces IP d'un domini o viceversa.

Sintaxi

nslookup [opcions] [domini]

Exemple

nslookup google.com

Aquesta comanda recupera l'adreça IP associada amb google.com.

`dig`

La comanda dig és una eina flexible per interrogar els servidors de noms DNS. Realitza consultes DNS i mostra les respostes que es retornen del servidor de noms consultat.

Sintaxi

dig [opcions] [domini]

Exemple

dig google.com

Aquesta comanda realitza una consulta DNS per google.com.

Comparació de `ping` i `traceroute`

| Característica | ping | traceroute | |----------------|-------------------------------|---------------------------------| | Propòsit | Provar la connectivitat d'un host | Mostrar la ruta que prenen els paquets | | Sortida | Temps de resposta i pèrdua de paquets | Llista de salts i temps de resposta | | Cas d'ús | Comprovar si un host està actiu | Diagnosticar problemes de routing |

Exercicis

  1. Prova bàsica de connectivitat

    • Utilitza la comanda ping per comprovar si example.com és accessible.
    ping example.com

  2. Informació de la interfície de xarxa

    • Mostra la configuració de totes les interfícies de xarxa utilitzant ifconfig.
    ifconfig

  3. Connexions actives

    • Llista totes les connexions de xarxa actives utilitzant netstat.
    netstat -a

  4. Traçar ruta

    • Utilitza traceroute per trobar el camí que prenen els paquets per arribar a example.com.
    traceroute example.com

  5. Descarregar un fitxer

    • Descarrega un fitxer d'internet utilitzant wget.
    wget http://example.com/file.zip

  6. Consulta DNS

    • Realitza una consulta DNS per example.com utilitzant nslookup.
    nslookup example.com

Conclusió

Entendre les comandes bàsiques de xarxa en Bash és essencial per gestionar i solucionar problemes de connexions de xarxa. Comandes com ping, ifconfig, netstat, traceroute, curl, wget, nslookup i dig proporcionen informació valuosa sobre l'estat i el rendiment de la xarxa. Practica aquestes comandes per convertir-te en un expert en la gestió i diagnòstic de xarxes.
