Introducció a les Funcions

Les funcions en Bash són una manera d'agrupar comandes per reutilitzar-les. Ajuden a organitzar el codi, fan els scripts més llegibles i redueixen la redundància.

Conceptes Clau

  • Definició: Una funció es defineix utilitzant la paraula clau function o simplement escrivint el nom de la funció seguit de parèntesis.
  • Invocació: Les funcions es criden pel seu nom seguit de parèntesis.
  • Àmbit: Les variables dins de les funcions són locals per defecte, tret que es declarin globals.
  • Valors de Retorn: Les funcions poden retornar valors utilitzant la paraula clau return o fent un echo del resultat.

Definint Funcions

Hi ha dues maneres principals de definir una funció en Bash:

Utilitzant la Paraula Clau `function`

function greet {
    echo "Hola, $1!"
}

Sense la Paraula Clau `function`

greet() {
    echo "Hola, $1!"
}

Ambdós mètodes són equivalents i es poden utilitzar indistintament.

Exemple: Funció Simple

greet() {
    echo "Hola, $1!"
}

# Cridant la funció
greet "Alice"

Sortida:

Hola, Alice!

Paràmetres de Funció

Les funcions poden acceptar paràmetres, que s'accedeixen utilitzant $1, $2, etc.

Exemple: Funció amb Paràmetres

add() {
    local sum=$(( $1 + $2 ))
    echo "Suma: $sum"
}

# Cridant la funció amb arguments
add 5 3

Sortida:

Suma: 8

Retornant Valors

Les funcions poden retornar valors utilitzant la paraula clau return o fent un echo del resultat.

Utilitzant `return`

multiply() {
    local product=$(( $1 * $2 ))
    return $product
}

# Cridant la funció
multiply 4 5
result=$?
echo "Producte: $result"

Nota: La paraula clau return només pot retornar un enter entre 0 i 255.

Utilitzant `echo`

multiply() {
    local product=$(( $1 * $2 ))
    echo $product
}

# Cridant la funció
result=$(multiply 4 5)
echo "Producte: $result"

Variables Locals i Globals

Les variables dins de les funcions són locals per defecte quan es declaren amb local.

Exemple: Variables Locals

calculate() {
    local result=$(( $1 + $2 ))
    echo $result
}

# Cridant la funció
sum=$(calculate 10 20)
echo "Suma: $sum"

Exemple: Variables Globals

result=0

calculate() {
    result=$(( $1 + $2 ))
}

# Cridant la funció
calculate 10 20
echo "Suma: $result"

Característiques Avançades de les Funcions

Funcions Recursives

Les funcions poden cridar-se a si mateixes recursivament.

Exemple: Funció Recursiva

factorial() {
    if [ $1 -le 1 ]; then
        echo 1
    else
        local temp=$(( $1 - 1 ))
        local result=$(factorial $temp)
        echo $(( $1 * result ))
    fi
}

# Cridant la funció
result=$(factorial 5)
echo "Factorial: $result"

Llibreries de Funcions

Les funcions es poden emmagatzemar en fitxers separats i incloure's en scripts.

Exemple: Llibreria de Funcions

Crea un fitxer mathlib.sh:

add() {
    echo $(( $1 + $2 ))
}

subtract() {
    echo $(( $1 - $2 ))
}

Inclou la llibreria en el teu script:

#!/bin/bash
source mathlib.sh

result=$(add 10 5)
echo "Addició: $result"

result=$(subtract 10 5)
echo "Subtracció: $result"

Conclusió

Les funcions en Bash són eines poderoses per organitzar i reutilitzar codi. Poden prendre paràmetres, retornar valors i fins i tot cridar-se a si mateixes recursivament. Dominant les funcions, pots escriure scripts Bash més eficients i mantenibles.
