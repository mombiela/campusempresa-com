Introducció als Arrays

Els arrays en Bash són una característica potent que et permet emmagatzemar múltiples valors en una sola variable. Això pot ser particularment útil per gestionar llistes de dades, com ara noms de fitxers, entrades d'usuari o opcions de configuració.

Creació d'Arrays

  • Arrays Indexats: Aquests arrays utilitzen índexs numèrics per accedir als seus elements.
  • Arrays Associatius: Aquests arrays utilitzen cadenes com a índexs.

Arrays Indexats

Per crear un array indexat, pots utilitzar la següent sintaxi:

# Mètode 1: Utilitzant parèntesis
fruits=("Poma" "Plàtan" "Cirera")

# Mètode 2: Utilitzant assignacions individuals
fruits[0]="Poma"
fruits[1]="Plàtan"
fruits[2]="Cirera"

Arrays Associatius

Per crear un array associatiu, primer cal declarar-lo i després assignar valors:

# Declarar un array associatiu
declare -A colors

# Assignar valors
colors[poma]="vermell"
colors[platan]="groc"
colors[cirera]="vermell"

Accés als Elements de l'Array

  • Arrays Indexats: Accedeix als elements utilitzant el seu índex numèric.
  • Arrays Associatius: Accedeix als elements utilitzant la seva clau de cadena.

Arrays Indexats

# Accés als elements
echo ${fruits[0]}  # Sortida: Poma
echo ${fruits[1]}  # Sortida: Plàtan
echo ${fruits[2]}  # Sortida: Cirera

Arrays Associatius

# Accés als elements
echo ${colors[poma]}  # Sortida: vermell
echo ${colors[platan]} # Sortida: groc
echo ${colors[cirera]} # Sortida: vermell

Modificació d'Arrays

  • Afegir Elements: Pots afegir elements assignant valors a nous índexs o claus.
  • Eliminar Elements: Utilitza la comanda unset per eliminar elements.

Afegir Elements

# Array Indexat
fruits[3]="Dàtil"

# Array Associatiu
colors[datil]="marró"

Eliminar Elements

# Array Indexat
unset fruits[1]  # Elimina "Plàtan"

# Array Associatiu
unset colors[platan]  # Elimina la clau "platan"

Iterar a través dels Arrays

Pots utilitzar bucles per iterar a través dels arrays i processar els seus elements.

Arrays Indexats

# Iterar a través d'un array indexat
for fruit in "${fruits[@]}"; do
  echo $fruit
done

Arrays Associatius

# Iterar a través d'un array associatiu
for key in "${!colors[@]}"; do
  echo "$key és ${colors[$key]}"
done

Longitud de l'Array

Pots obtenir la longitud d'un array utilitzant la sintaxi ${#array[@]}.

Arrays Indexats

# Obtenir la longitud d'un array indexat
echo ${#fruits[@]}  # Sortida: 4

Arrays Associatius

# Obtenir la longitud d'un array associatiu
echo ${#colors[@]}  # Sortida: 3

Resum

Els arrays en Bash proporcionen una manera flexible de gestionar múltiples valors dins d'una sola variable. Poden ser indexats o associatius, permetent tant la indexació numèrica com basada en cadenes. Les operacions clau inclouen la creació, accés, modificació i iteració a través dels arrays. Entendre aquests conceptes és essencial per a una programació eficient en Bash i la gestió de dades.

En dominar els arrays, pots escriure scripts Bash més eficients i llegibles, fent que el teu codi sigui més potent i versàtil.
