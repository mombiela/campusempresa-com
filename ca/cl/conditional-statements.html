Introducció

Les declaracions condicionals són fonamentals en la programació ja que permeten l'execució de codi basat en condicions específiques. En CL (Control Language), les declaracions condicionals ajuden a controlar el flux d'un programa prenent decisions. Aquesta secció cobrirà des dels conceptes bàsics fins a l'ús avançat de les declaracions condicionals en CL.

Declaracions condicionals bàsiques

Declaració IF

La declaració IF s'utilitza per executar un bloc de codi si una condició especificada és certa.

Sintaxi

IF COND(logical-expression) THEN(DO)
    /* Codi a executar si la condició és certa */
ENDDO

Exemple

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO

En aquest exemple, es mostra el missatge "You are an adult." si la variable &AGE té 18 anys o més.

Declaració ELSE

La declaració ELSE s'utilitza per executar un bloc de codi si la condició en la declaració IF és falsa.

Sintaxi

IF COND(logical-expression) THEN(DO)
    /* Codi a executar si la condició és certa */
ENDDO
ELSE DO
    /* Codi a executar si la condició és falsa */
ENDDO

Exemple

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(16)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are a minor.')
ENDDO

En aquest exemple, es mostra el missatge "You are a minor." si la variable &AGE té menys de 18 anys.

Declaracions condicionals avançades

Declaració ELSEIF

La declaració ELSEIF permet comprovar múltiples condicions en seqüència.

Sintaxi

IF COND(logical-expression1) THEN(DO)
    /* Codi a executar si la condició1 és certa */
ENDDO
ELSEIF COND(logical-expression2) THEN(DO)
    /* Codi a executar si la condició2 és certa */
ENDDO
ELSE DO
    /* Codi a executar si cap de les condicions anteriors és certa */
ENDDO

Exemple

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE < 13) THEN(DO)
    SNDPGMMSG MSG('You are a child.')
ENDDO
ELSEIF COND(&AGE < 18) THEN(DO)
    SNDPGMMSG MSG('You are a teenager.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are an adult.')
ENDDO

En aquest exemple, es mostra el missatge "You are an adult." si la variable &AGE té 18 anys o més.

Declaracions IF niades

Les declaracions IF niades són declaracions IF col·locades dins d'una altra declaració IF per comprovar múltiples condicions.

Sintaxi

IF COND(logical-expression1) THEN(DO)
    IF COND(logical-expression2) THEN(DO)
        /* Codi a executar si ambdues condicions són certes */
    ENDDO
ENDDO

Exemple

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
DCL VAR(&MEMBER) TYPE(*CHAR) LEN(1)
CHGVAR VAR(&AGE) VALUE(20)
CHGVAR VAR(&MEMBER) VALUE('Y')

IF COND(&AGE >= 18) THEN(DO)
    IF COND(&MEMBER = 'Y') THEN(DO)
        SNDPGMMSG MSG('You are an adult member.')
    ENDDO
ENDDO

En aquest exemple, es mostra el missatge "You are an adult member." si la variable &AGE té 18 anys o més i &MEMBER és 'Y'.

Resum

Les declaracions condicionals en CL són crucials per controlar el flux d'un programa. Començant amb les declaracions bàsiques IF i ELSE, pots crear estructures de presa de decisions complexes utilitzant ELSEIF i declaracions IF niades. Entendre i dominar aquests constructes millorarà significativament la teva capacitat per escriure programes CL eficients i efectius.
