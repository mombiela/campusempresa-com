Introducció

En aquesta secció, tractarem errors comuns trobats en escriure i executar scripts CL (Control Language), juntament amb les seves solucions. Entendre aquests errors i saber com resoldre'ls és crucial tant per a programadors principiants com avançats per assegurar una execució suau i eficient dels scripts.

Errors de sintaxi

Els errors de sintaxi es produeixen quan l'script CL no segueix les regles de sintaxi correctes. Aquests errors solen ser detectats durant la fase de compilació.

Errors de sintaxi comuns

  • Paraules clau mancants o mal col·locades: Assegureu-vos que totes les paraules clau necessàries estiguin presents i correctament col·locades.
  • Format de comanda incorrecte: Les comandes han de seguir el format específic definit per CL.
  • Parèntesis o cometes no aparellades: Assegureu-vos que tots els parèntesis i cometes estiguin correctament aparellats.

Exemple i solució

// Comanda CL incorrecta per falta de paraula clau
PGM
    DCL VAR(&VAR1) TYPE(*CHAR) LEN(10)
    CHGVAR VAR(&VAR1) VALUE('Hello)
ENDPGM

Error: Falta la cometa de tancament per a la cadena 'Hello'.

Solució:

PGM
    DCL VAR(&VAR1) TYPE(*CHAR) LEN(10)
    CHGVAR VAR(&VAR1) VALUE('Hello')
ENDPGM

Errors lògics

Els errors lògics es produeixen quan l'script s'executa sense errors de sintaxi però produeix resultats incorrectes a causa de falles en la lògica.

Errors lògics comuns

  • Inicialització incorrecta de variables: Assegureu-vos que les variables estiguin inicialitzades correctament.
  • Declaracions condicionals incorrectes: Verifiqueu que les declaracions condicionals (IF, ELSE) estiguin correctament implementades.
  • Errors en bucles: Assegureu-vos que els bucles (DO, DOWHILE) tinguin condicions d'inici i final correctes.

Exemple i solució

PGM
    DCL VAR(&COUNTER) TYPE(*INT) VALUE(0)
    DOWHILE COND(&COUNTER < 10)
        CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
    ENDDO
ENDPGM

Error: El bucle s'executarà indefinidament perquè la condició sempre és certa.

Solució:

PGM
    DCL VAR(&COUNTER) TYPE(*INT) VALUE(0)
    DOWHILE COND(&COUNTER < 10)
        CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
    ENDDO
ENDPGM

Assegureu-vos que la condició eventualment esdevingui falsa per sortir del bucle.

Errors d'execució

Els errors d'execució es produeixen mentre l'script s'està executant, sovint a causa d'operacions invàlides o indisponibilitat de recursos.

Errors d'execució comuns

  • Fitxer no trobat: Assegureu-vos que les rutes dels fitxers siguin correctes i que els fitxers existeixin.
  • Divisió per zero: Comproveu que no hi hagi zero abans de realitzar la divisió.
  • Permisos insuficients: Assegureu-vos que l'script tingui els permisos necessaris per realitzar operacions.

Exemple i solució

PGM
    DCL VAR(&NUM1) TYPE(*INT) VALUE(10)
    DCL VAR(&NUM2) TYPE(*INT) VALUE(0)
    DCL VAR(&RESULT) TYPE(*INT)
    CHGVAR VAR(&RESULT) VALUE(&NUM1 / &NUM2)
ENDPGM

Error: Divisió per zero.

Solució:

PGM
    DCL VAR(&NUM1) TYPE(*INT) VALUE(10)
    DCL VAR(&NUM2) TYPE(*INT) VALUE(0)
    DCL VAR(&RESULT) TYPE(*INT)
    IF COND(&NUM2 *NE 0) THEN(DO)
        CHGVAR VAR(&RESULT) VALUE(&NUM1 / &NUM2)
    ENDDO
    ELSE CMD(DO)
        /* Gestionar l'error */
    ENDDO
ENDPGM

Comproveu que no hi hagi zero abans de realitzar la divisió.

Tècniques de depuració

Les tècniques de depuració efectives poden ajudar a identificar i resoldre errors ràpidament.

Tècniques de depuració comunes

  • Ús de comandes de visualització: Utilitzeu DSPPGMVAR per mostrar els valors de les variables.
  • Punts de ruptura: Inseriu punts de ruptura per pausar l'execució i inspeccionar l'estat.
  • Registres d'errors: Reviseu els registres d'errors per obtenir informació detallada sobre els errors.

Exemple

PGM
    DCL VAR(&VAR1) TYPE(*CHAR) LEN(10)
    CHGVAR VAR(&VAR1) VALUE('Test')
    DSPPGMVAR VAR(&VAR1) /* Mostra el valor de VAR1 */
ENDPGM

Utilitzeu DSPPGMVAR per mostrar el valor de les variables en diferents punts de l'script.

Conclusió

Entendre i resoldre errors comuns en scripts CL és essencial per a una programació eficient. Reconèixer errors de sintaxi, lògics i d'execució, i emprar tècniques de depuració efectives, permet als programadors assegurar que els seus scripts s'executin sense problemes i produeixin els resultats desitjats. Practiqueu aquestes solucions i tècniques per esdevenir proficient en la gestió d'errors en CL.
