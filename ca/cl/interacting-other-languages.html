El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, explorarem com el Llenguatge de Control (CL) pot interactuar amb altres llenguatges de programació. Aquesta és una habilitat crucial per aprofitar els punts forts de diferents llenguatges dins d'una sola aplicació o sistema.

Conceptes Clau

  • Interoperabilitat: La capacitat del CL per treballar amb altres llenguatges de programació.
  • Cridar Programes: Com cridar programes escrits en altres llenguatges des d'un programa CL.
  • Intercanvi de Dades: Mètodes per intercanviar dades entre CL i altres llenguatges.
  • Gestió d'Errors: Gestionar errors quan s'interactua amb altres llenguatges.

Cridar Programes des de CL

Una de les principals maneres en què el CL interactua amb altres llenguatges és cridant programes escrits en aquests llenguatges. Això sovint es fa utilitzant la comanda CALL.

Exemple: Cridar un Programa RPG

PGM
    DCL VAR(&RESULT) TYPE(*CHAR) LEN(10)
    CALL PGM(MYLIB/MYPGM) PARM(&RESULT)
    /* Processar el resultat */
    SNDPGMMSG MSG('Resultat del programa RPG: ' *CAT &RESULT)
ENDPGM
  • Explicació:
    • DCL VAR(&RESULT) TYPE(*CHAR) LEN(10): Declara una variable per contenir el resultat.
    • CALL PGM(MYLIB/MYPGM) PARM(&RESULT): Crida un programa RPG i passa la variable &RESULT.
    • SNDPGMMSG MSG('Resultat del programa RPG: ' *CAT &RESULT): Envia un missatge de programa amb el resultat.

Exemple: Cridar un Programa COBOL

PGM
    DCL VAR(&STATUS) TYPE(*CHAR) LEN(1)
    CALL PGM(MYLIB/COBOLPGM) PARM(&STATUS)
    IF COND(&STATUS *EQ '0') THEN(DO)
        SNDPGMMSG MSG('El programa COBOL ha completat amb èxit.')
    ENDDO
    ELSE DO
        SNDPGMMSG MSG('El programa COBOL ha fallat.')
    ENDDO
ENDPGM
  • Explicació:
    • DCL VAR(&STATUS) TYPE(*CHAR) LEN(1): Declara una variable per contenir l'estat.
    • CALL PGM(MYLIB/COBOLPGM) PARM(&STATUS): Crida un programa COBOL i passa la variable &STATUS.
    • IF COND(&STATUS *EQ '0') THEN(DO): Comprova si l'estat és '0' (èxit).
    • SNDPGMMSG MSG('El programa COBOL ha completat amb èxit.'): Envia un missatge d'èxit.
    • ELSE DO: Gestiona el cas de fallada.
    • SNDPGMMSG MSG('El programa COBOL ha fallat.'): Envia un missatge de fallada.

Intercanvi de Dades

L'intercanvi de dades entre CL i altres llenguatges es pot fer a través de paràmetres, àrees de dades o fitxers.

Utilitzant Paràmetres

Els paràmetres són la manera més senzilla d'intercanviar dades.

PGM
    DCL VAR(&INPUT) TYPE(*CHAR) LEN(20)
    DCL VAR(&OUTPUT) TYPE(*CHAR) LEN(20)
    CHGVAR VAR(&INPUT) VALUE('Hola, Món!')
    CALL PGM(MYLIB/OTHERPGM) PARM(&INPUT &OUTPUT)
    SNDPGMMSG MSG('Sortida d'altre programa: ' *CAT &OUTPUT)
ENDPGM
  • Explicació:
    • DCL VAR(&INPUT) TYPE(*CHAR) LEN(20): Declara una variable d'entrada.
    • DCL VAR(&OUTPUT) TYPE(*CHAR) LEN(20): Declara una variable de sortida.
    • CHGVAR VAR(&INPUT) VALUE('Hola, Món!'): Estableix la variable d'entrada.
    • CALL PGM(MYLIB/OTHERPGM) PARM(&INPUT &OUTPUT): Crida un altre programa amb paràmetres d'entrada i sortida.
    • SNDPGMMSG MSG('Sortida d'altre programa: ' *CAT &OUTPUT): Envia un missatge amb la sortida.

Utilitzant Àrees de Dades

Les àrees de dades es poden utilitzar per a un intercanvi de dades més persistent.

PGM
    DCL VAR(&DATA) TYPE(*CHAR) LEN(50)
    RTVDTAARA DTAARA(MYLIB/MYDATA) RTNVAR(&DATA)
    CALL PGM(MYLIB/OTHERPGM)
    CHGDTAARA DTAARA(MYLIB/MYDATA) VALUE(&DATA)
ENDPGM
  • Explicació:
    • DCL VAR(&DATA) TYPE(*CHAR) LEN(50): Declara una variable per al contingut de l'àrea de dades.
    • RTVDTAARA DTAARA(MYLIB/MYDATA) RTNVAR(&DATA): Recupera dades d'una àrea de dades.
    • CALL PGM(MYLIB/OTHERPGM): Crida un altre programa.
    • CHGDTAARA DTAARA(MYLIB/MYDATA) VALUE(&DATA): Canvia el contingut de l'àrea de dades.

Gestió d'Errors

Una gestió adequada dels errors és essencial quan s'interactua amb altres llenguatges per garantir la robustesa.

PGM
    DCL VAR(&STATUS) TYPE(*CHAR) LEN(1)
    MONMSG MSGID(CPF0000) EXEC(DO)
        SNDPGMMSG MSG('S'ha produït un error en cridar el programa.')
        RETURN
    ENDDO
    CALL PGM(MYLIB/OTHERPGM) PARM(&STATUS)
    IF COND(&STATUS *NE '0') THEN(DO)
        SNDPGMMSG MSG('L'altre programa ha retornat un error.')
    ENDDO
ENDPGM
  • Explicació:
    • MONMSG MSGID(CPF0000) EXEC(DO): Monitora qualsevol missatge d'error.
    • SNDPGMMSG MSG('S'ha produït un error en cridar el programa.'): Envia un missatge d'error.
    • RETURN: Surt del programa.
    • IF COND(&STATUS *NE '0') THEN(DO): Comprova si l'estat indica un error.
    • SNDPGMMSG MSG('L'altre programa ha retornat un error.'): Envia un missatge d'error.

Conclusió

Interactuar amb altres llenguatges des de CL és una característica poderosa que et permet aprofitar els punts forts de diferents entorns de programació. Entenent com cridar programes, intercanviar dades i gestionar errors, pots crear aplicacions robustes i flexibles. Practica aquestes tècniques amb diversos llenguatges per convertir-te en un expert en interoperabilitat multillenguatge.
