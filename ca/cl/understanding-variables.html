Introducció a les Variables en CL

Les variables són fonamentals en qualsevol llenguatge de programació, incloent el Control Language (CL). S'utilitzen per emmagatzemar dades que es poden manipular i recuperar durant l'execució d'un programa.

Conceptes Clau

  • Definició: Una variable és una ubicació d'emmagatzematge amb nom en la memòria que conté un valor.
  • Tipus: CL admet diversos tipus de dades per a les variables, com ara caràcter, decimal i enter.
  • Abast: Les variables poden tenir diferents abasts, com ara local o global, determinant on es poden accedir dins del programa.

Declarar Variables

En CL, les variables es declaren utilitzant la comanda DCL (Declare). La sintaxi per declarar una variable inclou especificar el seu nom, tipus i longitud.

Sintaxi

DCL VAR(&NomVariable) TYPE(*DataType) LEN(longitud)

Exemple

DCL VAR(&Nom) TYPE(*CHAR) LEN(20)
DCL VAR(&Edat) TYPE(*DEC) LEN(3 0)
  • &Nom és una variable de caràcter amb una longitud de 20.
  • &Edat és una variable decimal amb una longitud de 3 dígits i 0 decimals.

Assignar Valors a les Variables

Els valors es poden assignar a les variables utilitzant la comanda CHGVAR (Change Variable).

Sintaxi

CHGVAR VAR(&NomVariable) VALUE(expressió)

Exemple

CHGVAR VAR(&Nom) VALUE('John Doe')
CHGVAR VAR(&Edat) VALUE(30)
  • &Nom s'assigna el valor 'John Doe'.
  • &Edat s'assigna el valor 30.

Utilitzar Variables en Expressions

Les variables es poden utilitzar en expressions per realitzar càlculs o manipular dades.

Exemple

DCL VAR(&Total) TYPE(*DEC) LEN(5 2)
DCL VAR(&Preu) TYPE(*DEC) LEN(5 2)
DCL VAR(&Quantitat) TYPE(*DEC) LEN(3 0)

CHGVAR VAR(&Preu) VALUE(19.99)
CHGVAR VAR(&Quantitat) VALUE(3)
CHGVAR VAR(&Total) VALUE(&Preu * &Quantitat)
  • &Total es calcula multiplicant &Preu i &Quantitat.

Abast de les Variables

Les variables en CL poden ser locals o globals. Les variables locals es declaren dins d'un procediment i només són accessibles dins d'aquest procediment. Les variables globals es declaren fora de qualsevol procediment i són accessibles a tot el programa.

Exemple de Variable Local

PGM
    DCL VAR(&VarLocal) TYPE(*CHAR) LEN(10)
    CHGVAR VAR(&VarLocal) VALUE('Local')
    /* &VarLocal només és accessible dins d'aquest bloc PGM */
ENDPGM

Exemple de Variable Global

DCL VAR(&VarGlobal) TYPE(*CHAR) LEN(10)

PGM
    CHGVAR VAR(&VarGlobal) VALUE('Global')
    /* &VarGlobal és accessible a tot el programa */
ENDPGM

Resum

Comprendre les variables en CL és crucial per a una programació efectiva. Les variables emmagatzemen dades que es poden manipular i recuperar, i tenen diversos tipus i abasts. Declarar, assignar valors i utilitzar variables en expressions són habilitats fonamentals. En dominar aquests conceptes, pots escriure programes CL més eficients i llegibles.

Punts Clau

  • Les variables són ubicacions d'emmagatzematge amb nom en la memòria.
  • Utilitza la comanda DCL per declarar variables.
  • Utilitza la comanda CHGVAR per assignar valors a les variables.
  • Les variables es poden utilitzar en expressions per a càlculs.
  • Comprèn l'abast de les variables per gestionar la seva accessibilitat dins dels teus programes.

En practicar aquests conceptes, obtindràs una base sòlida en la gestió de variables en CL, establint les bases per a tècniques de programació més avançades.
