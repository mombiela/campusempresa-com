Visió General

El Control Language (CL) és un llenguatge de script utilitzat en els sistemes de gamma mitjana d'IBM, com l'AS/400 (ara IBM i). Els comandaments CL s'utilitzen per controlar les operacions del sistema, gestionar treballs i manejar dades. Aquesta secció us introduirà als conceptes bàsics dels comandaments CL, la seva estructura i com utilitzar-los de manera efectiva.

Conceptes Clau

  • Estructura del Comandament: Els comandaments CL segueixen una sintaxi i estructura específica.
  • Paràmetres: Els comandaments sovint requereixen paràmetres per especificar detalls.
  • Execució: Com executar els comandaments CL en diferents entorns.
  • Gestió d'Errors: Gestió bàsica d'errors en els comandaments CL.

Estructura del Comandament

Els comandaments CL estan estructurats d'una manera específica per assegurar claredat i consistència. Un comandament CL típic es veu així:

COMMAND PARAMETER1(VALUE1) PARAMETER2(VALUE2)

Exemple

CRTUSRPRF USRPRF(USER1) PASSWORD(PASS123)
  • CRTUSRPRF: Comandament per crear un perfil d'usuari.
  • USRPRF: Paràmetre que especifica el nom del perfil d'usuari.
  • PASSWORD: Paràmetre que especifica la contrasenya per al perfil d'usuari.

Comandaments CL Comuns

Aquí hi ha alguns comandaments CL comuns i els seus propòsits:

| Comandament | Descripció | |-------------|--------------------------------------------| | CRTUSRPRF | Crear un perfil d'usuari | | DLTUSRPRF | Eliminar un perfil d'usuari | | WRKJOB | Treballar amb treballs | | DSPMSG | Mostrar missatges | | SBMJOB | Enviar un treball | | ENDJOB | Finalitzar un treball | | CHGJOB | Canviar atributs del treball | | SNDMSG | Enviar un missatge |

Exemple: Creant un Perfil d'Usuari

CRTUSRPRF USRPRF(USER1) PASSWORD(PASS123) TEXT('New User Profile')
  • TEXT: Un paràmetre opcional per afegir una descripció.

Paràmetres

Els paràmetres són components essencials dels comandaments CL. Proporcionen els detalls necessaris perquè el comandament s'executi correctament.

Exemple: Enviant un Treball

SBMJOB CMD(CALL PGM(MYPGM)) JOB(MYJOB) JOBQ(QBATCH)
  • CMD: Especifica el comandament a executar.
  • JOB: Nomena el treball.
  • JOBQ: Especifica la cua de treballs.

Execució

Els comandaments CL es poden executar en diversos entorns:

  • Línia de Comandaments: Introduir directament els comandaments a la interfície de línia de comandaments.
  • Programes CL: Inserir comandaments dins de programes CL per al processament per lots.
  • Planificador de Treballs: Programar comandaments per executar-se en moments específics.

Exemple: Executant un Comandament en un Programa CL

PGM
    CRTUSRPRF USRPRF(USER1) PASSWORD(PASS123)
    SBMJOB CMD(CALL PGM(MYPGM)) JOB(MYJOB) JOBQ(QBATCH)
ENDPGM

Gestió d'Errors

La gestió d'errors en els comandaments CL és crucial per a una execució robusta dels scripts. Les tècniques comunes inclouen:

  • Monitor de Missatges (MONMSG): Capturar i gestionar errors.
  • Gestió de Condicions: Utilitzar condicions per gestionar diferents escenaris.

Exemple: Utilitzant MONMSG

PGM
    CRTUSRPRF USRPRF(USER1) PASSWORD(PASS123)
    MONMSG MSGID(CPF0000) EXEC(DO)
        /* Codi de gestió d'errors */
    ENDDO
ENDPGM
  • MONMSG MSGID(CPF0000): Captura qualsevol missatge d'error que comenci amb CPF.

Conclusió

Aquesta introducció als comandaments CL proporciona una base per entendre com utilitzar el CL per gestionar i controlar els sistemes IBM i. En dominar l'estructura, els paràmetres, els mètodes d'execució i les tècniques de gestió d'errors, podeu automatitzar i optimitzar efectivament les operacions del sistema. A mesura que avanceu en aquest curs, aprofundireu en temes més avançats i escenaris complexos, millorant la vostra competència en els comandaments CL.
