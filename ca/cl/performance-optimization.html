Introducció a l'Optimització del Rendiment

L'optimització del rendiment en CL (Control Language) és crucial per assegurar que els teus programes s'executin de manera eficient, especialment quan es treballa amb grans conjunts de dades o operacions complexes. Aquesta secció et guiarà a través de diverses tècniques i millors pràctiques per optimitzar els teus programes CL des d'un nivell principiant fins a un nivell avançat.

Conceptes Clau de l'Optimització del Rendiment

  • Pràctiques de Codificació Eficients: Escriure codi net i eficient.
  • Gestió de Recursos: Ús adequat dels recursos del sistema.
  • Depuració i Perfilatge: Identificar i solucionar colls d'ampolla de rendiment.
  • Tècniques Avançades: Utilitzar característiques i comandes avançades per a l'optimització.

Pràctiques de Codificació Eficients

Ús de Funcions Incorporades

Les funcions incorporades estan optimitzades per al rendiment i s'han d'utilitzar sempre que sigui possible.

/* Exemple d'ús de funcions incorporades */
DCL VAR(&DATE) TYPE(*CHAR) LEN(6)
DCL VAR(&TIME) TYPE(*CHAR) LEN(6)

RTVDATE DATE(&DATE)
RTVTIME TIME(&TIME)

Minimitzar Operacions d'I/O

Redueix el nombre d'operacions d'entrada/sortida per millorar el rendiment.

/* Exemple de minimització d'operacions d'I/O */
DCLF FILE(MYFILE)
DCL VAR(&COUNT) TYPE(*DEC) LEN(5 0)

CHGVAR VAR(&COUNT) VALUE(0)
READ FILE(MYFILE)
DOUNTIL COND(&EOF)
    CHGVAR VAR(&COUNT) VALUE(&COUNT + 1)
    READ FILE(MYFILE)
ENDDO

Ús Eficient de Variables

Declara variables amb el tipus i la longitud apropiats per estalviar memòria i temps de processament.

/* Exemple d'ús eficient de variables */
DCL VAR(&SMALLVAR) TYPE(*DEC) LEN(3 0)
DCL VAR(&LARGEVAR) TYPE(*CHAR) LEN(100)

Gestió de Recursos

Prioritats de Treball

Estableix prioritats de treball adequades per assegurar que les tasques crítiques obtinguin més temps de CPU.

/* Exemple d'establiment de prioritat de treball */
SBMJOB CMD(CALL PGM(MYPGM)) JOB(MYJOB) JOBPTY(5)

Gestió de Memòria

Assigna i desassigna memòria de manera eficient per evitar fuites i ús excessiu.

/* Exemple de gestió de memòria */
DCL VAR(&LARGEARRAY) TYPE(*CHAR) LEN(1000)

Depuració i Perfilatge

Ús d'Eines de Depuració

Utilitza eines de depuració per identificar colls d'ampolla de rendiment.

/* Exemple d'ús de STRDBG per a la depuració */
STRDBG PGM(MYPGM)

Perfilatge del Teu Codi

Perfila el teu codi per entendre on es gasta més temps.

/* Exemple d'ús de dades de rendiment */
DSPJOB JOB(MYJOB) OPTION(*PERF)

Tècniques Avançades

Ús d'APIs

Aprofita les APIs per a tasques que requereixen alt rendiment.

/* Exemple d'ús d'una API del sistema */
CALL PGM(QSYS/QCMDEXC) PARM('DSPLIBL' 7)

Multi-threading

Implementa multi-threading per realitzar processament en paral·lel.

/* Exemple de submissió de múltiples treballs */
SBMJOB CMD(CALL PGM(MYPGM1)) JOB(MYJOB1)
SBMJOB CMD(CALL PGM(MYPGM2)) JOB(MYJOB2)

Conclusió

L'optimització del rendiment en CL implica una combinació de pràctiques de codificació eficients, gestió adequada de recursos, depuració i perfilatge efectius, i l'ús de tècniques avançades. Seguint aquestes directrius, pots assegurar que els teus programes CL s'executin de manera eficient i efectiva, fent el millor ús dels recursos del sistema i minimitzant el temps d'execució.
