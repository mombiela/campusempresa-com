El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el teu entorn és el primer pas per convertir-te en un expert en Control Language (CL). Aquesta secció et guiarà a través dels passos necessaris per preparar el teu entorn per a la programació en CL, des de la instal·lació del programari necessari fins a la configuració de les teves eines de desenvolupament.

Instal·lar el Programari Necessari

Abans de començar a escriure programes CL, necessites tenir el programari adequat instal·lat al teu sistema.

  • IBM i Access Client Solutions (ACS): Aquesta és l'eina principal per accedir als sistemes IBM i. Inclou un emulador 5250, eines de base de dades i altres utilitats.
  • Sistema IBM i: Assegura't de tenir accés a un sistema IBM i on puguis executar els teus programes CL. Aquest podria ser un sistema local o un de remot proporcionat per la teva organització.
  • Editor de Text o IDE: Tot i que pots escriure programes CL amb qualsevol editor de text, utilitzar un Entorn de Desenvolupament Integrat (IDE) com RDi (Rational Developer for i) pot augmentar significativament la teva productivitat.

Passos per Instal·lar IBM i Access Client Solutions (ACS)

  1. Descarregar ACS: Visita el lloc web d'IBM i descarrega l'última versió d'IBM i Access Client Solutions.
  2. Instal·lar ACS: Segueix les instruccions d'instal·lació proporcionades per IBM. Això normalment implica executar un instal·lador i seguir les instruccions a la pantalla.
  3. Configurar ACS: Un cop instal·lat, configura ACS per connectar-te al teu sistema IBM i. Necessitaràs l'adreça IP o el nom del sistema, i les teves credencials d'usuari.
# Exemple de configuració d'ACS
Nom del Sistema: myibmisystem
ID d'Usuari: myusername
Contrasenya: **

Configurar les Teves Eines de Desenvolupament

Després d'instal·lar el programari necessari, el següent pas és configurar les teves eines de desenvolupament.

  • Configurar RDi: Si estàs utilitzant Rational Developer for i (RDi), hauràs de configurar-lo per connectar-te al teu sistema IBM i.
  • Configurar un Editor de Text: Si prefereixes utilitzar un editor de text com VS Code, pots configurar-lo per treballar amb CL instal·lant extensions rellevants i configurant connexions remotes.

Passos per Configurar RDi

  1. Instal·lar RDi: Descarrega i instal·la Rational Developer for i des del lloc web d'IBM.
  2. Crear una Connexió: Obre RDi i crea una nova connexió al teu sistema IBM i.
  3. Configurar un Projecte: Crea un nou projecte a RDi on guardaràs els teus programes CL.
# Exemple de creació d'una connexió a RDi
Nom de la Connexió: MyIBMConnection
Nom del Host: myibmisystem
ID d'Usuari: myusername
Contrasenya: **

Configurar VS Code per a CL

  1. Instal·lar VS Code: Descarrega i instal·la Visual Studio Code des del lloc web oficial.
  2. Instal·lar Extensions: Instal·la extensions com IBM i Languages i IBM i Remote per millorar la teva experiència de desenvolupament en CL.
  3. Configurar Connexió Remota: Configura una connexió remota al teu sistema IBM i utilitzant les extensions instal·lades.
// Exemple de configuració de connexió remota a VS Code
{
    "name": "IBM i",
    "host": "myibmisystem",
    "username": "myusername",
    "password": "**"
}

Provar la Teva Configuració

Un cop hagis instal·lat i configurat les teves eines, és important provar la teva configuració per assegurar-te que tot funciona correctament.

  • Connectar-se a IBM i: Utilitza les teves eines configurades per connectar-te al teu sistema IBM i.
  • Crear un Programa CL Senzill: Escriu un programa CL senzill i executa'l per verificar que el teu entorn està configurat correctament.

Exemple d'un Programa CL Senzill

PGM
    DCL VAR(&MSG) TYPE(*CHAR) LEN(50)
    CHGVAR VAR(&MSG) VALUE('Hello, World!')
    SNDPGMMSG MSG(&MSG)
ENDPGM
  • Guardar el Programa: Guarda el codi anterior en un fitxer amb una extensió .CLP.
  • Compilar i Executar: Utilitza la teva eina de desenvolupament per compilar i executar el programa al teu sistema IBM i.

Conclusió

Configurar el teu entorn és un pas crucial en el teu camí per dominar el Control Language. Seguint els passos descrits en aquesta secció, hauràs de tenir un entorn de desenvolupament completament funcional preparat per escriure i executar programes CL. Recorda provar la teva configuració a fons per assegurar-te que tot funciona com s'espera. Amb el teu entorn preparat, ara pots passar a aprendre els conceptes bàsics de la programació en CL.
