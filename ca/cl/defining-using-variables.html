Introducció

En aquesta secció, cobrirem els conceptes bàsics de la definició i ús de variables en Control Language (CL). Les variables són essencials per emmagatzemar dades que es poden manipular i utilitzar al llarg dels teus programes CL. Començarem amb conceptes bàsics i gradualment passarem a temes més avançats.

Què és una Variable?

  • Una variable és una ubicació d'emmagatzematge identificada per un nom (o identificador) que conté dades que es poden modificar durant l'execució del programa.
  • Les variables en CL poden emmagatzemar diferents tipus de dades, com ara valors numèrics, cadenes de text, i més.

Definició de Variables

Per definir una variable en CL, utilitzes la comanda DCL (Declare). Aquí tens la sintaxi bàsica:

DCL VAR(&VariableName) TYPE(*DataType) LEN(length)

Exemple:

DCL VAR(&NAME) TYPE(*CHAR) LEN(20)
DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
  • &NAME és una variable de caràcter amb una longitud de 20.
  • &AGE és una variable decimal amb una longitud de 3 dígits i 0 decimals.

Conceptes Clau:

  • VAR: Especifica el nom de la variable.
  • TYPE: Especifica el tipus de dades que la variable contindrà. Tipus comuns inclouen:
    • *CHAR: Cadena de caràcters.
    • *DEC: Número decimal.
    • *INT: Enter.
  • LEN: Especifica la longitud de la variable. Per als tipus decimals, inclou el nombre total de dígits i el nombre de decimals.

Assignació de Valors a Variables

Un cop una variable està definida, pots assignar-li valors utilitzant la comanda CHGVAR (Change Variable).

Exemple:

CHGVAR VAR(&NAME) VALUE('John Doe')
CHGVAR VAR(&AGE) VALUE(30)
  • &NAME se li assigna el valor 'John Doe'.
  • &AGE se li assigna el valor 30.

Conceptes Clau:

  • CHGVAR: Comanda utilitzada per canviar el valor d'una variable.
  • VALUE: Especifica el nou valor que s'assignarà a la variable.

Ús de Variables en Expressions

Les variables es poden utilitzar en expressions per realitzar càlculs o concatenar cadenes.

Exemple:

DCL VAR(&SALARY) TYPE(*DEC) LEN(7 2)
DCL VAR(&BONUS) TYPE(*DEC) LEN(7 2)
DCL VAR(&TOTAL) TYPE(*DEC) LEN(7 2)

CHGVAR VAR(&SALARY) VALUE(50000.00)
CHGVAR VAR(&BONUS) VALUE(5000.00)
CHGVAR VAR(&TOTAL) VALUE(&SALARY + &BONUS)
  • &TOTAL es calcula sumant &SALARY i &BONUS.

Sentències Condicionals amb Variables

Les variables sovint s'utilitzen en sentències condicionals per controlar el flux del programa.

Exemple:

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)

CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE *LT 18) THEN(DO)
    /* Codi per a menors de 18 */
    SNDPGMMSG MSG('Menor de 18')
ENDDO
ELSE DO
    /* Codi per a 18 i majors */
    SNDPGMMSG MSG('18 o més gran')
ENDDO
  • La sentència IF comprova si &AGE és menor de 18 i executa el bloc de codi corresponent.

Ús Avançat de Variables

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors en una sola variable.

Exemple:

DCL VAR(&NAMES) TYPE(*CHAR) LEN(20) DIM(5)

CHGVAR VAR(&NAMES(1)) VALUE('Alice')
CHGVAR VAR(&NAMES(2)) VALUE('Bob')
CHGVAR VAR(&NAMES(3)) VALUE('Charlie')
CHGVAR VAR(&NAMES(4)) VALUE('Diana')
CHGVAR VAR(&NAMES(5)) VALUE('Eve')
  • &NAMES és un array de 5 cadenes de caràcters, cadascuna amb una longitud de 20.

Estructures de Dades

Les estructures de dades permeten agrupar variables relacionades.

Exemple:

DCL VAR(&EMPLOYEE) TYPE(*CHAR) LEN(50) 
DCL VAR(&EMPID) TYPE(*CHAR) LEN(10) STG(*DEFINED) DEFVAR(&EMPLOYEE 1)
DCL VAR(&EMPNAME) TYPE(*CHAR) LEN(40) STG(*DEFINED) DEFVAR(&EMPLOYEE 11)

CHGVAR VAR(&EMPID) VALUE('E12345')
CHGVAR VAR(&EMPNAME) VALUE('John Smith')
  • &EMPLOYEE és una estructura de dades que conté &EMPID i &EMPNAME.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la definició i ús de variables en CL, incloent:

  • Declaració de variables amb la comanda DCL.
  • Assignació de valors utilitzant la comanda CHGVAR.
  • Ús de variables en expressions i sentències condicionals.
  • Ús avançat amb arrays i estructures de dades.

Entendre aquests conceptes és crucial per escriure programes CL efectius. Practica creant i manipulant variables en els teus propis scripts CL per reforçar aquests conceptes.
