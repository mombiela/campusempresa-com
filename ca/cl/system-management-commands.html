Introducció

Les Comandes de Gestió del Sistema en Llenguatge de Control (CL) són essencials per gestionar i controlar el sistema operatiu IBM i. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat d'aquestes comandes, proporcionant-te el coneixement per gestionar eficaçment els recursos del sistema, els usuaris i les feines.

Comandes Bàsiques de Gestió del Sistema

Comandes de Visualització

  • DSPJOB (Visualitzar Feina): Mostra informació sobre una feina.
  • DSPUSRPRF (Visualitzar Perfil d'Usuari): Mostra informació sobre un perfil d'usuari.
  • DSPMSG (Visualitzar Missatge): Mostra missatges d'una cua de missatges.

Exemple: Visualitzar Informació de la Feina

DSPJOB JOB(123456/USER/JOBNAME)

Aquesta comanda mostra informació detallada sobre la feina amb el número de feina, usuari i nom de feina especificats.

Comandes de Treball

  • WRKACTJOB (Treballar amb Feines Actives): Mostra una llista de feines actives.
  • WRKSYSSTS (Treballar amb l'Estat del Sistema): Mostra l'estat actual del sistema.
  • WRKUSRPRF (Treballar amb Perfils d'Usuari): Mostra i permet gestionar perfils d'usuari.

Exemple: Treballar amb Feines Actives

WRKACTJOB

Aquesta comanda obre una pantalla que mostra totes les feines actives, permetent-te gestionar-les.

Comandes Intermèdies de Gestió del Sistema

Comandes de Control

  • ENDJOB (Finalitzar Feina): Finalitza una feina especificada.
  • ENDSYS (Finalitzar Sistema): Finalitza el sistema de manera controlada.
  • ENDSBS (Finalitzar Subsistema): Finalitza un subsistema especificat.

Exemple: Finalitzar una Feina

ENDJOB JOB(123456/USER/JOBNAME) OPTION(*IMMED)

Aquesta comanda finalitza immediatament la feina especificada.

Comandes de Gestió d'Usuaris

  • CRTUSRPRF (Crear Perfil d'Usuari): Crea un nou perfil d'usuari.
  • CHGUSRPRF (Canviar Perfil d'Usuari): Canvia atributs d'un perfil d'usuari existent.
  • DLTUSRPRF (Eliminar Perfil d'Usuari): Elimina un perfil d'usuari.

Exemple: Crear un Perfil d'Usuari

CRTUSRPRF USRPRF(NEWUSER) PASSWORD('password') USRCLS(*USER)

Aquesta comanda crea un nou perfil d'usuari anomenat NEWUSER amb la contrasenya especificada i la classe d'usuari.

Comandes Avançades de Gestió del Sistema

Comandes de Còpia de Seguretat i Restauració

  • SAVLIB (Guardar Biblioteca): Guarda una biblioteca en un dispositiu especificat.
  • RSTLIB (Restaurar Biblioteca): Restaura una biblioteca des d'un dispositiu especificat.
  • SAVSYS (Guardar Sistema): Guarda tot el sistema.

Exemple: Guardar una Biblioteca

SAVLIB LIB(MYLIB) DEV(TAP01)

Aquesta comanda guarda la biblioteca MYLIB en el dispositiu de cinta TAP01.

Comandes de Gestió del Rendiment

  • WRKSYSACT (Treballar amb l'Activitat del Sistema): Mostra i gestiona l'activitat del sistema.
  • WRKDSKSTS (Treballar amb l'Estat del Disc): Mostra i gestiona l'estat del disc.
  • WRKSHRPOOL (Treballar amb Pools Compartits): Gestiona pools de memòria compartida.

Exemple: Treballar amb l'Activitat del Sistema

WRKSYSACT

Aquesta comanda obre una pantalla que mostra l'activitat del sistema, permetent-te monitoritzar i gestionar el rendiment.

Resum

Les Comandes de Gestió del Sistema en CL són eines poderoses per gestionar el sistema operatiu IBM i. Des de visualitzar informació de feines fins a gestionar perfils d'usuari i el rendiment del sistema, aquestes comandes proporcionen un control complet sobre les operacions del sistema. En dominar aquestes comandes, pots assegurar una gestió eficient i efectiva del sistema.

Punts Clau

  • Les comandes bàsiques com DSPJOB i WRKACTJOB són essencials per monitoritzar l'estat del sistema.
  • Les comandes intermèdies com ENDJOB i CRTUSRPRF permeten un control més directe sobre les feines i els perfils d'usuari.
  • Les comandes avançades com SAVLIB i WRKSYSACT són crucials per a la còpia de seguretat, restauració i gestió del rendiment del sistema.

En practicar aquestes comandes i entendre les seves aplicacions, estaràs ben preparat per gestionar un sistema IBM i de manera efectiva.
