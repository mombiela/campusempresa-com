Llibres i publicacions

Per aprofundir en la teva comprensió del Control Language (CL), considera explorar els següents llibres i publicacions:

  • "IBM i Control Language (CL): The Basics" d'IBM Redbooks

    • Aquest llibre proporciona una introducció completa al CL, cobrint conceptes fonamentals i comandes bàsiques.

  • "Advanced Control Language Programming for IBM i" de Bryan Meyers

    • Aquesta publicació tracta temes i tècniques avançades, ideal per a aquells que volen dominar el CL.

  • "IBM i Control Language Programming" de Ted Holt

    • Una guia pràctica que inclou nombrosos exemples i exercicis per ajudar-te a aplicar el que has après.

Exemple de fragment de codi

Aquí tens un exemple senzill d'un programa CL que demostra la sintaxi i l'estructura bàsiques:

PGM        /* Inici del programa CL */
DCL        VAR(&MSG) TYPE(*CHAR) LEN(50) /* Declarar una variable */
CHGVAR     VAR(&MSG) VALUE('Hello, World!') /* Canviar el valor de la variable */
SNDPGMMSG  MSG(&MSG) /* Enviar un missatge de programa */
ENDPGM     /* Fi del programa CL */
  • Explicació:
    • PGM i ENDPGM marquen el començament i el final del programa CL.
    • DCL s'utilitza per declarar una variable anomenada &MSG de tipus caràcter amb una longitud de 50.
    • CHGVAR canvia el valor de &MSG a 'Hello, World!'.
    • SNDPGMMSG envia el missatge emmagatzemat a &MSG a la cua de missatges del programa.

Conclusió

La lectura addicional i la pràctica contínua són essencials per dominar el Control Language (CL). Els recursos llistats anteriorment, incloent llibres, tutorials en línia i repositoris de codi, proporcionen una gran quantitat d'informació per ajudar-te a avançar de principiant a expert. Recorda experimentar amb codi, participar en fòrums comunitaris i mantenir-te actualitzat amb els últims desenvolupaments en la programació IBM i. Feliç programació!
