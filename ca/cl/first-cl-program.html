Introducció

Control Language (CL) és un llenguatge de script utilitzat en els sistemes AS/400 d'IBM (ara coneguts com IBM i). S'utilitza principalment per a l'automatització i l'scripting a nivell de sistema. En aquesta secció, escriurem el nostre primer programa CL, entendrem la seva estructura i l'executarem.

Conceptes Clau

  • Programa CL: Una seqüència de comandes CL que realitzen una tasca específica.
  • Fitxer Físic de Font: Un fitxer on es guarda el codi font CL.
  • Compilació: El procés de convertir el codi font CL en un objecte executable.
  • Execució: Correr el programa CL compilat.

Configuració de l'Entorn

Abans d'escriure el teu primer programa CL, assegura't de tenir accés a un sistema IBM i i els permisos necessaris per crear i executar programes CL.

Escriure el Teu Primer Programa CL

  1. Crear un Fitxer Físic de Font:

    • Un fitxer físic de font és on guardaràs el teu codi font CL.
    • Utilitza la següent comanda per crear un fitxer físic de font:
    CRTSRCPF FILE(QGPL/QCLSRC) RCDLEN(112)
    • Aquesta comanda crea un fitxer físic de font anomenat QCLSRC a la biblioteca QGPL amb una longitud de registre de 112.

  2. Afegir un Membre al Fitxer Físic de Font:

    • Un membre és un lloc dins del fitxer físic de font on escriuràs el teu programa CL.
    • Utilitza la següent comanda per afegir un membre:
    ADDPFM FILE(QGPL/QCLSRC) MBR(MYCLPGM) SRCTYPE(CL)
    • Aquesta comanda afegeix un membre anomenat MYCLPGM al fitxer QCLSRC amb un tipus de font CL.

  3. Editar el Membre de Font:

    • Utilitza la Source Entry Utility (SEU) o qualsevol altre editor disponible al teu sistema per editar el membre de font.
    • Utilitza la següent comanda per començar a editar:
    STRSEU SRCFILE(QGPL/QCLSRC) SRCMBR(MYCLPGM)
    • Aquesta comanda obre el membre MYCLPGM al fitxer QCLSRC per editar-lo.

  4. Escriure el Programa CL:

    • Introdueix el següent codi a l'editor:
    PGM
DCL VAR(&MSG) TYPE(*CHAR) LEN(50)
CHGVAR VAR(&MSG) VALUE('Hello, World!')
SNDPGMMSG MSG(&MSG)
ENDPGM
    • Explicació del codi:
      • PGM: Indica l'inici del programa CL.
      • DCL VAR(&MSG) TYPE(*CHAR) LEN(50): Declara una variable &MSG de tipus caràcter amb una longitud de 50.
      • CHGVAR VAR(&MSG) VALUE('Hello, World!'): Canvia el valor de &MSG a 'Hello, World!'.
      • SNDPGMMSG MSG(&MSG): Envia un missatge de programa que conté el valor de &MSG.
      • ENDPGM: Indica el final del programa CL.

Compilar el Programa CL

  • Després d'escriure el programa CL, necessites compilar-lo per crear un objecte executable.
  • Utilitza la següent comanda per compilar el programa:
CRTCLPGM PGM(QGPL/MYCLPGM) SRCFILE(QGPL/QCLSRC) SRCMBR(MYCLPGM)
  • Aquesta comanda crea un objecte programa anomenat MYCLPGM a la biblioteca QGPL utilitzant el membre de font MYCLPGM del fitxer QCLSRC.

Executar el Programa CL

  • Per executar el programa CL compilat, utilitza la següent comanda:
CALL PGM(QGPL/MYCLPGM)
  • Aquesta comanda corre el programa MYCLPGM de la biblioteca QGPL, i hauries de veure el missatge 'Hello, World!' mostrat.

Conclusió

En aquesta secció, has après com crear, escriure, compilar i executar un programa CL senzill en un sistema IBM i. Aquest coneixement fonamental és crucial a mesura que avances cap a temes més avançats en la programació CL. Practica escrivint i corrent programes CL bàsics per familiaritzar-te amb la sintaxi i les comandes.
