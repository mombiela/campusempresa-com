Introducció al CL (Llenguatge de Control)

El Llenguatge de Control (CL) és un llenguatge de script utilitzat en els sistemes AS/400 d'IBM (ara coneguts com IBM i). S'utilitza principalment per controlar l'execució de programes i gestionar els recursos del sistema. Entendre la sintaxi i l'estructura bàsica del CL és essencial per a qualsevol persona que vulgui treballar amb sistemes IBM i.

Conceptes Clau

  • Comandes: Les unitats fonamentals del CL, utilitzades per realitzar tasques específiques.
  • Declaracions: Instruccions escrites en CL que poden incloure comandes, etiquetes i estructures de control.
  • Variables: Utilitzades per emmagatzemar dades que es poden manipular dins d'un programa CL.
  • Estructures de Control: Construccions com bucles i condicionals que controlen el flux del programa.

Sintaxi Bàsica

Comandes

Les comandes en CL són instruccions predefinides que realitzen tasques específiques. Cada comanda té una sintaxi específica i un conjunt de paràmetres.

Exemple:

SNDMSG MSG('Hola, Món!') TOUSR(USER1)

Explicació:

  • SNDMSG és la comanda per enviar un missatge.
  • MSG és un paràmetre que especifica el contingut del missatge.
  • TOUSR és un paràmetre que especifica el destinatari.

Declaracions

Un programa CL consisteix en una sèrie de declaracions. Cada declaració pot ser una comanda, una etiqueta o una estructura de control.

Exemple:

PGM
    SNDMSG MSG('Iniciant el procés') TOUSR(USER1)
    /* Comandes addicionals van aquí */
ENDPGM

Explicació:

  • PGM marca el començament d'un programa CL.
  • SNDMSG és una declaració de comanda.
  • ENDPGM marca el final del programa CL.

Variables

Les variables en CL s'utilitzen per emmagatzemar dades que es poden utilitzar i manipular dins del programa. Es defineixen utilitzant la comanda DCL (Declarar).

Exemple:

DCL VAR(&MSG) TYPE(*CHAR) LEN(50)
CHGVAR VAR(&MSG) VALUE('Hola, Món!')
SNDMSG MSG(&MSG) TOUSR(USER1)

Explicació:

  • DCL declara una variable anomenada &MSG de tipus caràcter amb una longitud de 50.
  • CHGVAR canvia el valor de la variable &MSG a 'Hola, Món!'.
  • SNDMSG envia el missatge emmagatzemat a &MSG a USER1.

Estructures de Control

Les estructures de control en CL inclouen bucles i declaracions condicionals que controlen el flux del programa.

Declaració IF

La declaració IF s'utilitza per executar un bloc de codi només si una condició especificada és certa.

Exemple:

DCL VAR(&STATUS) TYPE(*CHAR) LEN(10)
CHGVAR VAR(&STATUS) VALUE('SUCCESS')

IF COND(&STATUS *EQ 'SUCCESS') THEN(DO)
    SNDMSG MSG('Procés completat amb èxit') TOUSR(USER1)
ENDDO

Explicació:

  • IF COND comprova si la variable &STATUS és igual a 'SUCCESS'.
  • THEN(DO) comença un bloc de codi per executar si la condició és certa.
  • ENDDO marca el final del bloc.

Bucle DO

El bucle DO s'utilitza per repetir un bloc de codi un nombre especificat de vegades.

Exemple:

DCL VAR(&I) TYPE(*DEC) LEN(2 0)
DCL VAR(&MSG) TYPE(*CHAR) LEN(50)

DOFOR VAR(&I) FROM(1) TO(5)
    CHGVAR VAR(&MSG) VALUE('Iteració ' *CAT &I)
    SNDMSG MSG(&MSG) TOUSR(USER1)
ENDDO

Explicació:

  • DOFOR comença un bucle amb la variable &I que va de 1 a 5.
  • CHGVAR canvia el valor de &MSG a 'Iteració ' seguit del valor actual de &I.
  • SNDMSG envia el missatge emmagatzemat a &MSG a USER1.
  • ENDDO marca el final del bucle.

Conclusió

Entendre la sintaxi i l'estructura bàsica del CL és crucial per escriure i gestionar scripts de manera efectiva en sistemes IBM i. Això inclou saber com utilitzar comandes, declaracions, variables i estructures de control. Amb aquests fonaments, pots començar a crear programes CL més complexos i funcionals.
