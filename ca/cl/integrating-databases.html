Introducció

Integrar-se amb bases de dades és un aspecte crucial de treballar amb Control Language (CL) en sistemes IBM i. Aquest tema et guiarà a través dels conceptes bàsics de la integració de bases de dades, progressant cap a tècniques més avançades. Al final d'aquesta secció, seràs capaç de realitzar operacions de bases de dades utilitzant comandes CL de manera efectiva.

Conceptes Clau

  • Fitxers de Base de Dades: Entendre els tipus de fitxers de base de dades (físics i lògics) i els seus rols.
  • Comandes CL: Aprendre les comandes CL utilitzades per a operacions de bases de dades.
  • Gestió d'Errors: Implementar la gestió d'errors en operacions de bases de dades.
  • Tècniques Avançades: Explorar tècniques avançades d'integració de bases de dades.

Fitxers de Base de Dades

Fitxers Físics

Els fitxers físics són els fitxers de base de dades reals que emmagatzemen dades.

CRTPF FILE(MYLIB/MYFILE) RCDLEN(100)
  • CRTPF: Comanda per crear un fitxer físic.
  • FILE: Especifica la biblioteca i el nom del fitxer.
  • RCDLEN: Defineix la longitud del registre.

Fitxers Lògics

Els fitxers lògics proporcionen una vista diferent o un camí d'accés a les dades en fitxers físics.

CRTLF FILE(MYLIB/MYLOGICAL) SRCFILE(MYLIB/QDDSSRC)
  • CRTLF: Comanda per crear un fitxer lògic.
  • FILE: Especifica la biblioteca i el nom del fitxer lògic.
  • SRCFILE: Especifica el fitxer font que conté les DDS (Especificacions de Descripció de Dades).

Comandes CL per a Operacions de Base de Dades

Afegir Registres

Utilitza la comanda ADDPFTRG per afegir registres a un fitxer físic.

ADDPFTRG FILE(MYLIB/MYFILE) TRGTIME(*AFTER) TRGEVENT(*INSERT) PGM(MYLIB/MYPGM)
  • ADDPFTRG: Comanda per afegir un disparador a un fitxer físic.
  • FILE: Especifica la biblioteca i el nom del fitxer.
  • TRGTIME: Especifica quan s'ha d'activar el disparador.
  • TRGEVENT: Especifica l'esdeveniment que activa el disparador.
  • PGM: Especifica el programa que s'executarà quan s'activi el disparador.

Actualitzar Registres

Utilitza la comanda UPDPF per actualitzar registres en un fitxer físic.

UPDPF FILE(MYLIB/MYFILE) MBR(MYMBR) FROMFILE(MYLIB/NEWFILE)
  • UPDPF: Comanda per actualitzar un fitxer físic.
  • FILE: Especifica la biblioteca i el nom del fitxer.
  • MBR: Especifica el membre a actualitzar.
  • FROMFILE: Especifica el fitxer font per a l'actualització.

Eliminar Registres

Utilitza la comanda DLTPF per eliminar registres d'un fitxer físic.

DLTPF FILE(MYLIB/MYFILE)
  • DLTPF: Comanda per eliminar un fitxer físic.
  • FILE: Especifica la biblioteca i el nom del fitxer.

Gestió d'Errors

Implementar la gestió d'errors assegura que els teus programes CL puguin gestionar excepcions de manera elegant.

MONMSG MSGID(CPF0000) EXEC(DO)
  /* Codi de gestió d'errors aquí */
ENDDO
  • MONMSG: Comanda per monitoritzar missatges.
  • MSGID: Especifica l'ID del missatge a monitoritzar.
  • EXEC: Especifica la comanda a executar si es rep el missatge.

Tècniques Avançades

Utilitzar SQL en CL

Pots incrustar instruccions SQL dins de programes CL per a operacions de bases de dades més complexes.

RUNSQL SQL('INSERT INTO MYLIB.MYTABLE (COL1, COL2) VALUES (''Value1'', ''Value2'')')
  • RUNSQL: Comanda per executar una instrucció SQL.
  • SQL: Especifica la instrucció SQL a executar.

Utilitzar Àrees de Dades

Les àrees de dades es poden utilitzar per emmagatzemar i recuperar dades de manera eficient.

CRTDTAARA DTAARA(MYLIB/MYDATA) TYPE(*CHAR) LEN(100)
  • CRTDTAARA: Comanda per crear una àrea de dades.
  • DTAARA: Especifica la biblioteca i el nom de l'àrea de dades.
  • TYPE: Especifica el tipus de dades.
  • LEN: Especifica la longitud de l'àrea de dades.

Conclusió

Integrar-se amb bases de dades utilitzant CL implica entendre els tipus de fitxers de base de dades, dominar les comandes CL essencials per a operacions de bases de dades, implementar la gestió d'errors i explorar tècniques avançades com utilitzar SQL dins de programes CL. Amb aquestes habilitats, podràs gestionar i manipular dades de manera eficient en sistemes IBM i.
