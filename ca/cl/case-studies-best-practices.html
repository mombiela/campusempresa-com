Introducció

En aquesta secció, explorarem estudis de casos reals i millors pràctiques per utilitzar el Control Language (CL) en entorns IBM i. Això t'ajudarà a entendre com aplicar CL en escenaris pràctics i seguir els estàndards de la indústria per escriure codi eficient i mantenible.

Estudis de casos

Estudi de cas 1: Automatització de còpies de seguretat diàries

Escenari

Una empresa necessita automatitzar el seu procés diari de còpia de seguretat per garantir la integritat de les dades i minimitzar la intervenció manual.

Solució

Es pot escriure un programa CL per automatitzar el procés de còpia de seguretat, programar-lo per executar-se a una hora específica i gestionar qualsevol error que pugui ocórrer.

Exemple de codi

PGM
DCL VAR(&DATE) TYPE(*CHAR) LEN(8)
DCL VAR(&TIME) TYPE(*CHAR) LEN(6)
DCL VAR(&MSG) TYPE(*CHAR) LEN(100)

RTVSYSVAL SYSVAL(QDATE) RTNVAR(&DATE)
RTVSYSVAL SYSVAL(QTIME) RTNVAR(&TIME)

SAVLIB LIB(*ALLUSR) DEV(TAP01) ENDOPT(*LEAVE) +
    SAVACT(*LIB) SAVACTMSGQ(QSYSOPR)

IF COND(&LASTCC *NE 0) THEN(DO)
    CHGVAR VAR(&MSG) VALUE('Backup failed on ' *CAT &DATE *CAT ' at ' *CAT &TIME)
    SNDPGMMSG MSG(&MSG) TOUSR(*SYSOPR)
ENDDO

ENDPGM

Explicació

  • DCL VAR(&DATE) TYPE(*CHAR) LEN(8): Declara una variable per emmagatzemar la data actual.
  • RTVSYSVAL SYSVAL(QDATE) RTNVAR(&DATE): Recupera la data del sistema.
  • SAVLIB LIB(*ALLUSR) DEV(TAP01): Desa totes les biblioteques d'usuaris al dispositiu de cinta especificat.
  • IF COND(&LASTCC *NE 0): Comprova si l'última comanda s'ha completat amb èxit.
  • SNDPGMMSG MSG(&MSG) TOUSR(*SYSOPR): Envia un missatge de programa a l'operador del sistema si la còpia de seguretat falla.

Estudi de cas 2: Monitorització del rendiment del sistema

Escenari

Una organització necessita monitoritzar el rendiment del sistema i generar alertes si es superen certs llindars.

Solució

Es pot crear un programa CL per comprovar periòdicament les mètriques de rendiment del sistema i enviar alertes quan sigui necessari.

Exemple de codi

PGM
DCL VAR(&CPU) TYPE(*DEC) LEN(3 0)
DCL VAR(&MSG) TYPE(*CHAR) LEN(100)

RTVJOBA JOBTYPE(&JOBTYPE) CPUPCT(&CPU)

IF COND(&CPU *GT 80) THEN(DO)
    CHGVAR VAR(&MSG) VALUE('High CPU usage detected: ' *CAT &CPU *CAT '%')
    SNDPGMMSG MSG(&MSG) TOUSR(*SYSOPR)
ENDDO

ENDPGM

Explicació

  • DCL VAR(&CPU) TYPE(*DEC) LEN(3 0): Declara una variable per emmagatzemar el percentatge d'ús de la CPU.
  • RTVJOBA JOBTYPE(&JOBTYPE) CPUPCT(&CPU): Recupera el percentatge d'ús actual de la CPU.
  • IF COND(&CPU *GT 80): Comprova si l'ús de la CPU supera el 80%.
  • SNDPGMMSG MSG(&MSG) TOUSR(*SYSOPR): Envia un missatge de programa a l'operador del sistema si es detecta un alt ús de la CPU.

Millors pràctiques

Llegibilitat del codi

  • Utilitza noms de variables significatius: Tria noms de variables que descriguin clarament el seu propòsit.
  • Comenta el teu codi: Afegeix comentaris per explicar la lògica complexa o seccions importants.
  • Indentació consistent: Utilitza una indentació consistent per millorar la llegibilitat del codi.

Gestió d'errors

  • Comprova els codis de finalització de les comandes: Comprova sempre els codis de finalització de les comandes per gestionar els errors de manera elegant.
  • Utilitza la gestió de missatges: Envia missatges informatius als operadors o registra'ls per a una revisió posterior.

Optimització del rendiment

  • Minimitza l'ús de recursos: Evita el consum innecessari de recursos optimitzant bucles i declaracions condicionals.
  • Programa tasques pesades durant hores de baixa activitat: Programa tasques intensives en recursos durant períodes de baixa activitat per minimitzar l'impacte en els usuaris.

Seguretat

  • Limita els permisos dels usuaris: Assegura't que els programes CL s'executin amb els permisos mínims necessaris.
  • Valida les entrades: Valida sempre les entrades per evitar atacs d'injecció o altres activitats malicioses.

Conclusió

Estudiant aquests casos pràctics i adherint-te a les millors pràctiques, podràs escriure programes CL més eficients, mantenibles i segurs. Recorda sempre considerar les necessitats específiques del teu entorn i refinar contínuament les teves habilitats a través de la pràctica i la revisió.
