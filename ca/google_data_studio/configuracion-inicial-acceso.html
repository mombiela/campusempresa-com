Introducció

En aquesta secció, aprendràs com configurar Google Data Studio des de zero i com accedir a la plataforma. Aquest és el primer pas essencial per a qualsevol professional que desitgi utilitzar Google Data Studio per crear informes i dashboards interactius.

Requisits previs

Abans de començar, assegura't de complir amb els següents requisits:

  • Tenir un compte de Google.
  • Accés a dades que desitges visualitzar (per exemple, Google Sheets, Google Analytics, etc.).

Creació d'un compte a Google Data Studio

Per començar a usar Google Data Studio, segueix aquests passos:

  1. Accedeix a Google Data Studio:

    • Ves a Google Data Studio.
    • Inicia sessió amb el teu compte de Google.

  2. Accepta els termes i condicions:

    • La primera vegada que accedeixis, hauràs d'acceptar els termes i condicions de Google Data Studio.

  3. Configura el teu perfil:

    • Completa la configuració inicial del teu perfil, proporcionant la informació sol·licitada.

Navegació per la interfície de Google Data Studio

Un cop hagis accedit a Google Data Studio, familiaritza't amb la seva interfície:

  • Panell de control: Aquí pots veure tots els teus informes i fonts de dades.
  • Botó "Crear": Utilitza'l per crear nous informes o fonts de dades.
  • Menú de navegació: Accedeix als teus informes, fonts de dades, i altres configuracions.

Connexió a fonts de dades

Per crear informes a Google Data Studio, necessites connectar una o més fonts de dades. Segueix aquests passos per connectar una font de dades:

  1. Fes clic a "Crear":

    • Selecciona "Font de dades".

  2. Selecciona el connector de dades:

    • Google Data Studio ofereix diversos connectors, com Google Sheets, Google Analytics, BigQuery, etc.
    • Selecciona el connector adequat per a les teves dades.

  3. Autoritza l'accés:

    • Si és la primera vegada que utilitzes un connector, és possible que necessitis autoritzar l'accés a les teves dades.

  4. Configura la connexió:

    • Proporciona els detalls necessaris per connectar la font de dades (per exemple, URL de Google Sheets, ID de vista de Google Analytics).

  5. Afegeix la font de dades al teu informe:

    • Un cop configurada, pots afegir la font de dades al teu informe i començar a crear visualitzacions.

Exemple: Connexió a Google Sheets

Suposem que desitges connectar un Google Sheet a Google Data Studio. Aquí tens un exemple pas a pas:

1. Fes clic a "Crear" i selecciona "Font de dades".
2. Selecciona "Google Sheets" com a connector.
3. Autoritza l'accés al teu compte de Google Sheets.
4. Selecciona l'arxiu de Google Sheets que desitges connectar.
5. Configura les opcions addicionals (per exemple, seleccionar el full específic dins de l'arxiu).
6. Fes clic a "Connectar" per finalitzar la configuració.

Creació del teu primer informe

Un cop hagis connectat les teves fonts de dades, pots crear el teu primer informe:

  1. Fes clic a "Crear" i selecciona "Informe".

  2. Selecciona una plantilla o comença des de zero:

    • Google Data Studio ofereix diverses plantilles predefinides que pots utilitzar.
    • També pots començar amb un informe en blanc.

  3. Afegeix la teva font de dades:

    • Selecciona la font de dades que vas configurar prèviament.

  4. Dissenya el teu informe:

    • Utilitza les eines de disseny per afegir gràfics, taules, i altres elements visuals.
    • Personalitza els estils i formats segons les teves necessitats.

Conclusió

En aquesta secció, hem cobert els passos essencials per configurar Google Data Studio i accedir a la plataforma. Ara hauries d'estar familiaritzat amb la creació d'un compte, la navegació per la interfície, la connexió a fonts de dades i la creació del teu primer informe. Amb aquesta base, estaràs llest per explorar més funcionalitats avançades de Google Data Studio en les següents seccions del curs.
