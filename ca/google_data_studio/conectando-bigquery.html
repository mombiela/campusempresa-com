Introducció

En aquesta secció, aprendràs com connectar Google Data Studio amb BigQuery, una eina poderosa per a l'anàlisi de grans volums de dades. BigQuery és un magatzem de dades empresarial altament escalable i sense servidor, dissenyat per fer consultes SQL ultraràpides usant el poder de la infraestructura de Google.

Requisits Previs

Abans de començar, assegura't de tenir el següent:

  • Un compte de Google Cloud Platform (GCP).
  • Accés a BigQuery amb almenys un conjunt de dades disponible.
  • Accés a Google Data Studio.

Passos per Connectar BigQuery a Google Data Studio

1. Accedir a Google Data Studio

  • Obre Google Data Studio al teu navegador.
  • Inicia sessió amb el teu compte de Google.

2. Crear un Nou Informe

  • Fes clic al botó "Crear" a la cantonada superior esquerra.
  • Selecciona "Informe" per crear un nou informe.

3. Seleccionar BigQuery com a Font de Dades

  • A la finestra emergent de selecció de font de dades, busca i selecciona "BigQuery".
  • Si és la primera vegada que connectes BigQuery a Data Studio, és possible que necessitis autoritzar l'accés. Fes clic a "Autoritzar" i segueix les instruccions.

4. Configurar la Connexió a BigQuery

  • Un cop autoritzat, veuràs una llista dels teus projectes de GCP.
  • Selecciona el projecte que conté el conjunt de dades que desitges usar.
  • Navega a través dels conjunts de dades i taules disponibles i selecciona la taula que desitges connectar.

5. Personalitzar els Camps de Dades

  • Després de seleccionar la taula, Data Studio et permetrà personalitzar els camps de dades.
  • Pots canviar els noms dels camps, ajustar els tipus de dades i aplicar filtres si és necessari.

6. Afegir la Font de Dades a l'Informe

  • Fes clic a "Afegir" per incloure la font de dades al teu informe.
  • Ara pots començar a crear visualitzacions utilitzant les dades de BigQuery.

Exemple Pràctic

Suposem que tens un conjunt de dades a BigQuery anomenat sales_data que conté informació sobre vendes. Aquí hi ha un exemple de com podries connectar i visualitzar aquestes dades a Google Data Studio.

Pas a Pas

  1. Accedeix a Google Data Studio i crea un nou informe.
  2. Selecciona BigQuery com la teva font de dades.
  3. Autoritza l'accés si és necessari.
  4. Navega al teu projecte a GCP i selecciona el conjunt de dades sales_data.
  5. Personalitza els camps si és necessari, per exemple, canviant el nom del camp total_sales a Vendes Totals.
  6. Afegeix la font de dades a l'informe.

Exemple de Consulta SQL a BigQuery

Si necessites realitzar una consulta específica a BigQuery abans de connectar a Data Studio, aquí tens un exemple de com fer-ho:

SELECT 
  product_name,
  SUM(total_sales) as total_sales
FROM 
  `your_project.sales_data`
GROUP BY 
  product_name
ORDER BY 
  total_sales DESC

Aquesta consulta agrupa les vendes totals per nom de producte i les ordena de major a menor.

Conclusió

Connectar BigQuery a Google Data Studio et permet aprofitar el poder de BigQuery per realitzar anàlisis de dades avançats i crear visualitzacions interactives. Seguint els passos esmentats, pots integrar fàcilment les teves dades de BigQuery a Data Studio i començar a crear informes detallats i visualment atractius.

Recorda que la personalització dels camps i la capacitat de realitzar consultes SQL directament a BigQuery t'ofereixen una gran flexibilitat per adaptar les dades a les teves necessitats específiques. Comença a explorar i a visualitzar les teves dades avui mateix!
