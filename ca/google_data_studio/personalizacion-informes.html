Introducció

En aquesta secció del curs sobre Google Data Studio, aprendràs com personalitzar informes perquè s'adaptin a les teves necessitats específiques. La personalització d'informes és crucial per presentar dades de manera clara i efectiva, permetent als usuaris prendre decisions informades basades en la informació visualitzada.

Conceptes Clau

  • Temes i Estils: Configuració de colors, fonts i altres elements visuals.
  • Filtres i Controls: Eines per interactuar amb les dades.
  • Camps Calculats: Creació de mètriques personalitzades.
  • Gràfics i Taules: Selecció i personalització de diferents tipus de visualitzacions.
  • Interactivitat: Afegir elements interactius per millorar l'experiència de l'usuari.

Temes i Estils

Configuració de Temes

Google Data Studio permet aplicar temes predefinits o crear-ne un de personalitzat per mantenir la coherència visual en els teus informes.

  • Temes Predefinits: Selecciona un tema de la galeria de temes.
  • Temes Personalitzats: Ajusta colors, fonts i altres elements visuals.
1. Ves a la pestanya de "Tema i disseny".
2. Selecciona "Personalitzar" per ajustar els colors i fonts.
3. Guarda el tema per reutilitzar-lo en altres informes.

Exemple de Configuració de Tema

1. Obre el teu informe a Google Data Studio.
2. Fes clic a "Tema i disseny" a la barra lateral dreta.
3. Selecciona "Personalitzar".
4. Ajusta els colors primaris i secundaris.
5. Canvia les fonts segons la teva preferència.
6. Guarda els canvis.

Filtres i Controls

Afegir Filtres

Els filtres permeten als usuaris interactuar amb les dades i visualitzar només la informació rellevant.

  • Filtres de Data: Permeten seleccionar rangs de dates específics.
  • Filtres de Dimensions: Permeten filtrar dades basades en dimensions específiques com regió, producte, etc.
1. Selecciona el gràfic o taula al qual desitges afegir un filtre.
2. Fes clic a "Afegir un control" a la barra d'eines.
3. Selecciona el tipus de filtre (per exemple, "Filtre de data").
4. Configura el filtre segons les teves necessitats.

Exemple de Filtre de Data

1. Selecciona el gràfic de vendes mensuals.
2. Fes clic a "Afegir un control" i selecciona "Filtre de data".
3. Configura el filtre per permetre la selecció d'un rang de dates.
4. Aplica el filtre i verifica que les dades s'actualitzin correctament.

Camps Calculats

Creació de Camps Calculats

Els camps calculats et permeten crear mètriques personalitzades que no estan disponibles de manera predeterminada.

  • Fórmules Bàsiques: Suma, resta, multiplicació i divisió.
  • Funcions Avançades: Ús de funcions com CASE, IF, REGEXP_MATCH, etc.
1. Obre l'editor de dades.
2. Fes clic a "Afegir un camp".
3. Introdueix el nom del camp i la fórmula.
4. Guarda el camp calculat.

Exemple de Camp Calculat

1. Obre l'editor de dades.
2. Fes clic a "Afegir un camp".
3. Nomena el camp "Margen de Benefici".
4. Introdueix la fórmula: `Revenue - Cost`.
5. Guarda el camp calculat.

Gràfics i Taules

Selecció de Visualitzacions

Google Data Studio ofereix una varietat de gràfics i taules per representar les teves dades.

  • Gràfics de Barres: Útils per comparar diferents categories.
  • Gràfics de Línies: Ideals per mostrar tendències al llarg del temps.
  • Taules: Perfectes per mostrar dades detallades.
1. Selecciona "Afegir un gràfic" a la barra d'eines.
2. Tria el tipus de gràfic que millor s'adapti a les teves dades.
3. Configura les dimensions i mètriques.
4. Personalitza l'estil del gràfic.

Exemple de Gràfic de Barres

1. Selecciona "Afegir un gràfic" i tria "Gràfic de barres".
2. Configura la dimensió com "Producte" i la mètrica com "Vendes".
3. Personalitza els colors i etiquetes.
4. Guarda i verifica que el gràfic mostra les dades correctament.

Interactivitat

Afegir Elements Interactius

Els elements interactius milloren l'experiència de l'usuari i permeten explorar les dades de manera més dinàmica.

  • Controls de Data: Permeten seleccionar rangs de dates.
  • Filtres de Dimensions: Permeten filtrar dades per categories específiques.
  • Botons de Navegació: Faciliten la navegació entre diferents pàgines de l'informe.
1. Selecciona "Afegir un control" a la barra d'eines.
2. Tria el tipus de control (per exemple, "Control de data").
3. Configura el control segons les teves necessitats.
4. Afegeix botons de navegació si és necessari.

Exemple de Control de Data

1. Selecciona "Afegir un control" i tria "Control de data".
2. Configura el control per permetre la selecció d'un rang de dates.
3. Col·loca el control en una posició visible a l'informe.
4. Verifica que el control interactuï correctament amb els gràfics i taules.

Conclusió

La personalització d'informes a Google Data Studio és una habilitat essencial per presentar dades de manera clara i efectiva. En dominar temes i estils, filtres i controls, camps calculats, gràfics i taules, i elements interactius, podràs crear informes que no només siguin visualment atractius, sinó també altament funcionals i útils per a la presa de decisions. Practica aquests conceptes i experimenta amb diferents configuracions per trobar les que millor s'adaptin a les teves necessitats específiques.
