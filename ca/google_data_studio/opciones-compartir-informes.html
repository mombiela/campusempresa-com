En aquesta secció del curs sobre Google Data Studio, explorarem les diverses opcions disponibles per compartir informes. Aprendràs com col·laborar amb altres usuaris, controlar els permisos d'accés i utilitzar enllaços compartits per distribuir els teus informes de manera efectiva.

Opcions de Compartir

Google Data Studio ofereix diverses maneres de compartir els teus informes amb altres usuaris. Aquí estan les principals opcions:

  • Compartir amb usuaris específics: Pots convidar persones específiques a veure o editar el teu informe.
  • Enllaç compartit: Genera un enllaç que pots compartir amb qualsevol persona, permetent-los veure o editar l'informe segons els permisos que configuris.
  • Incorporar informes: Insereix els teus informes en llocs web o blogs utilitzant codi HTML.
  • Descarregar com a PDF: Exporta el teu informe a un fitxer PDF per compartir-lo fora de Google Data Studio.

Compartir amb Usuaris Específics

Per compartir un informe amb usuaris específics, segueix aquests passos:

  1. Obre l'informe que desitges compartir.
  2. Fes clic al botó "Compartir" a la cantonada superior dreta.
  3. Introdueix les adreces de correu electrònic de les persones amb les quals desitges compartir l'informe.
  4. Selecciona els permisos adequats (veure o editar).
  5. Fes clic a "Enviar".

Exemple

1. Obre l'informe "Vendes Mensuals".
2. Fes clic a "Compartir".
3. Introdueix el correu electrònic "exemple@domini.com".
4. Selecciona "Pot veure".
5. Fes clic a "Enviar".

Enllaç Compartit

Per generar un enllaç compartit:

  1. Obre l'informe.
  2. Fes clic a "Compartir".
  3. A la secció "Obtenir enllaç", fes clic a "Copiar enllaç".
  4. Configura els permisos de l'enllaç (veure o editar).
  5. Comparteix l'enllaç amb les persones desitjades.

Exemple

1. Obre l'informe "Anàlisi de Trànsit Web".
2. Fes clic a "Compartir".
3. A "Obtenir enllaç", fes clic a "Copiar enllaç".
4. Configura els permisos com "Qualsevol amb l'enllaç pot veure".
5. Comparteix l'enllaç: https://datastudio.google.com/reporting/exemple

Incorporar Informes

Per incorporar un informe en un lloc web:

  1. Obre l'informe.
  2. Fes clic a "Arxiu" > "Incorporar informe".
  3. Copia el codi HTML proporcionat.
  4. Enganxa el codi a l'HTML del teu lloc web o blog.

Exemple

<iframe width="600" height="450" src="https://datastudio.google.com/reporting/exemple" frameborder="0" style="border:0" allowfullscreen></iframe>

Descarregar com a PDF

Per descarregar un informe com a PDF:

  1. Obre l'informe.
  2. Fes clic a "Arxiu" > "Descarregar com a PDF".
  3. Configura les opcions de descàrrega (pàgines, orientació, etc.).
  4. Fes clic a "Descarregar".

Exemple

1. Obre l'informe "Resum Financer".
2. Fes clic a "Arxiu" > "Descarregar com a PDF".
3. Selecciona "Totes les pàgines".
4. Fes clic a "Descarregar".

Comparació d'Opcions

| Opció | Permisos Disponibles | Ideal Per | |---------------------|----------------------|----------------------------------------------| | Usuaris Específics | Veure, Editar | Col·laboració directa amb companys d'equip | | Enllaç Compartit | Veure, Editar | Distribució àmplia sense necessitat de comptes| | Incorporar Informes | Veure | Publicació en llocs web o blogs | | Descarregar com a PDF| N/A | Compartir fora de Google Data Studio |

Conclusió

Compartir informes en Google Data Studio és una funcionalitat poderosa que facilita la col·laboració i distribució de dades. Ja sigui que necessitis col·laborar amb col·legues, distribuir informes a una audiència més àmplia o incorporar les teves dades en un lloc web, Google Data Studio ofereix opcions flexibles per satisfer les teves necessitats. Assegura't de configurar els permisos adequadament per mantenir el control sobre qui pot veure o editar els teus informes.
