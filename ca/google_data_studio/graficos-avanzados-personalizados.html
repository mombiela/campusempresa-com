Introducció

En aquesta secció, aprendràs a crear gràfics avançats i personalitzats a Google Data Studio. Aquests gràfics et permetran visualitzar dades de manera més efectiva i adaptada a les teves necessitats específiques. Explorarem com utilitzar les opcions avançades de personalització i com integrar gràfics personalitzats mitjançant l'ús de JavaScript i CSS.

Conceptes Clau

  • Gràfics avançats: Gràfics que van més enllà de les opcions bàsiques ofertes per Google Data Studio, permetent una major personalització i funcionalitat.
  • Personalització: Ajustos i modificacions que es poden fer als gràfics perquè s'adaptin millor a les necessitats específiques de l'usuari.
  • JavaScript i CSS: Llenguatges de programació que es poden utilitzar per crear gràfics personalitzats i millorar la presentació visual de les dades.

Tipus de Gràfics Avançats

Gràfics de Sèries Temporals

  • Descripció: Utilitzats per mostrar dades al llarg del temps.
  • Personalització:
    • Ajust d'intervals de temps.
    • Aplicació de línies de tendència.
    • Ús de colors personalitzats per a diferents sèries de dades.
// Exemple de personalització d'un gràfic de sèries temporals
var options = {
  title: 'Vendes al llarg del temps',
  curveType: 'function',
  legend: { position: 'bottom' },
  colors: ['#1b9e77', '#d95f02', '#7570b3']
};

Gràfics de Barres Apilades

  • Descripció: Permeten comparar parts d'un tot al llarg de diferents categories.
  • Personalització:
    • Configuració de colors per a cada segment.
    • Ajust d'etiquetes i llegendes.
    • Aplicació de filtres per mostrar dades específiques.
// Exemple de personalització d'un gràfic de barres apilades
var options = {
  title: 'Distribució de Vendes per Regió',
  isStacked: true,
  colors: ['#8e44ad', '#3498db', '#e74c3c'],
  hAxis: { title: 'Regió' },
  vAxis: { title: 'Vendes' }
};

Gràfics de Dispersió

  • Descripció: Utilitzats per mostrar la relació entre dues variables.
  • Personalització:
    • Ajust de mida i color dels punts.
    • Inclusió de línies de regressió.
    • Configuració d'eixos personalitzats.
// Exemple de personalització d'un gràfic de dispersió
var options = {
  title: 'Relació entre Ingressos i Despeses',
  hAxis: { title: 'Ingressos' },
  vAxis: { title: 'Despeses' },
  bubble: { textStyle: { fontSize: 11 } },
  colors: ['#2ecc71']
};

Integració de Gràfics Personalitzats

Ús de JavaScript i CSS

  • JavaScript: Permet crear gràfics interactius i dinàmics.
  • CSS: Utilitzat per estilitzar els gràfics i millorar la seva aparença visual.
// Exemple d'integració d'un gràfic personalitzat amb JavaScript i CSS
google.charts.load('current', {'packages':['corechart']});
google.charts.setOnLoadCallback(drawChart);

function drawChart() {
  var data = google.visualization.arrayToDataTable([
    ['Any', 'Vendes', 'Despeses'],
    ['2014',  1000,      400],
    ['2015',  1170,      460],
    ['2016',  660,       1120],
    ['2017',  1030,      540]
  ]);

  var options = {
    title: 'Rendiment de l\'Empresa',
    curveType: 'function',
    legend: { position: 'bottom' },
    colors: ['#1abc9c', '#e74c3c']
  };

  var chart = new google.visualization.LineChart(document.getElementById('curve_chart'));
  chart.draw(data, options);
}
/* Exemple de CSS per estilitzar un gràfic */
#curve_chart {
  width: 100%;
  height: 500px;
  border: 1px solid #ddd;
  padding: 10px;
}

Exercicis Pràctics

  1. Crear un gràfic de sèries temporals: Utilitza les dades de vendes mensuals de la teva empresa per crear un gràfic de sèries temporals. Personalitza els colors i afegeix una línia de tendència.
  2. Configurar un gràfic de barres apilades: Mostra la distribució de vendes per producte i regió. Ajusta els colors i les etiquetes perquè siguin clars i fàcils d'entendre.
  3. Dissenyar un gràfic de dispersió: Visualitza la relació entre el pressupost de màrqueting i les vendes. Afegeix una línia de regressió i personalitza els eixos.

Conclusió

En aquesta secció, hem explorat com crear i personalitzar gràfics avançats a Google Data Studio. La capacitat de personalitzar gràfics utilitzant JavaScript i CSS et permet crear visualitzacions de dades més efectives i atractives. Practica amb els exercicis proporcionats per millorar les teves habilitats i aplicar aquests coneixements als teus propis projectes d'anàlisi de dades.
