Introducció

En aquest cas d'estudi, aprendrem a crear un dashboard de vendes utilitzant Google Data Studio. Aquest dashboard permetrà als usuaris visualitzar i analitzar dades de vendes de manera efectiva. Explorarem com connectar fonts de dades, crear gràfics i taules, i personalitzar el disseny del dashboard perquè sigui intuïtiu i útil.

Objectius del cas d'estudi

  • Connectar Google Data Studio amb una font de dades de vendes.
  • Crear gràfics i taules que representin les dades de vendes.
  • Personalitzar el disseny del dashboard per millorar la visualització de les dades.
  • Compartir el dashboard amb altres usuaris.

Connectant la font de dades

Per començar, necessitem connectar Google Data Studio amb la nostra font de dades de vendes. En aquest exemple, utilitzarem un full de càlcul de Google Sheets que conté dades de vendes.

  1. Obrir Google Data Studio: Navega a Google Data Studio.
  2. Crear un nou informe: Fes clic a "Crear" i selecciona "Informe".
  3. Connectar la font de dades:
    • Fes clic a "Afegir dades".
    • Selecciona "Google Sheets".
    • Tria el full de càlcul que conté les dades de vendes.
    • Selecciona el full específic dins de l'arxiu i fes clic a "Connectar".

Creació de gràfics i taules

Un cop connectada la font de dades, podem començar a crear gràfics i taules per visualitzar les dades de vendes.

Gràfic de barres per a vendes mensuals

  1. Afegir un gràfic de barres:
    • Fes clic a "Afegir un gràfic" i selecciona "Gràfic de barres".
    • Col·loca el gràfic a l'àrea de treball.
  2. Configurar el gràfic:
    • Al panell de configuració, selecciona "Data" com l'eix X.
    • Selecciona "Vendes" com l'eix Y.
    • Assegura't que l'interval de dates estigui configurat per mostrar dades mensuals.

Taula de vendes per producte

  1. Afegir una taula:
    • Fes clic a "Afegir un gràfic" i selecciona "Taula".
    • Col·loca la taula a l'àrea de treball.
  2. Configurar la taula:
    • Al panell de configuració, selecciona "Producte" com la dimensió.
    • Selecciona "Vendes" com la mètrica.
    • Ordena la taula per la columna de vendes en ordre descendent per veure els productes més venuts a la part superior.

Personalització del disseny del dashboard

Per fer el dashboard més intuïtiu i visualment atractiu, podem personalitzar el seu disseny.

  • Canviar colors i fonts: Al panell d'estil, pots canviar els colors dels gràfics i les fonts perquè coincideixin amb la identitat visual de la teva empresa.
  • Afegir filtres: Afegeix filtres per permetre als usuaris filtrar les dades per diferents criteris, com rang de dates o categories de productes.
  • Incloure títols i descripcions: Afegeix títols i descripcions als gràfics i taules per proporcionar context i facilitar la interpretació de les dades.

Compartir el dashboard

Un cop el dashboard estigui complet, pots compartir-lo amb altres usuaris.

  1. Compartir l'informe:

    • Fes clic a "Compartir" a la cantonada superior dreta.
    • Introdueix les adreces de correu electrònic dels usuaris amb els quals desitges compartir el dashboard.
    • Configura els permisos d'accés (poden veure o editar).

  2. Generar un enllaç compartible:

    • Fes clic a "Obtenir enllaç compartible".
    • Copia l'enllaç i comparteix-lo amb els usuaris.

Conclusió

En aquest cas d'estudi, hem après a crear un dashboard de vendes a Google Data Studio. Hem cobert com connectar una font de dades, crear gràfics i taules, personalitzar el disseny del dashboard i compartir-lo amb altres usuaris. Amb aquestes habilitats, pots crear dashboards efectius que ajudin el teu equip a prendre decisions informades basades en dades de vendes.
