Introducció

Google Data Studio és una eina poderosa per a la visualització de dades, i una de les seves característiques més avançades és la capacitat de crear visualitzacions personalitzades a través de Community Visualizations. Aquesta funcionalitat permet als usuaris desenvolupar i compartir gràfics personalitzats utilitzant tecnologies web com HTML, CSS i JavaScript.

Què són les Community Visualizations?

Les Community Visualizations són gràfics personalitzats que pots crear i compartir a Google Data Studio. Aquestes visualitzacions et permeten:

  • Ampliar les capacitats de visualització més enllà de les opcions predeterminades.
  • Utilitzar tecnologies web estàndard (HTML, CSS i JavaScript) per crear gràfics interactius i personalitzats.
  • Compartir les teves visualitzacions amb altres usuaris de Data Studio.

Requisits Previs

Abans de començar a crear Community Visualizations, assegura't de tenir:

  • Coneixements bàsics d'HTML, CSS i JavaScript.
  • Accés a Google Data Studio.
  • Un compte de Google Cloud Platform (GCP) per allotjar els teus arxius.

Configuració Inicial

Pas 1: Crear un Projecte a Google Cloud Platform

  1. Ves a Google Cloud Console.
  2. Crea un nou projecte.
  3. Habilita l'API de Google Data Studio.

Pas 2: Configurar l'Entorn de Desenvolupament

  1. Crea una carpeta a la teva màquina local per al teu projecte.
  2. Dins d'aquesta carpeta, crea els següents arxius:
    • manifest.json: Defineix les metadades de la teva visualització.
    • index.html: Conté l'estructura HTML de la teva visualització.
    • index.css: Defineix els estils CSS.
    • index.js: Conté la lògica JavaScript.

Exemple d'Estructura d'Arxius

my-visualization/
├── manifest.json
├── index.html
├── index.css
└── index.js

Creació de la Visualització

Definir el Manifest

L'arxiu manifest.json és crucial ja que defineix les metadades de la teva visualització.

{
  "dataStudio": {
    "name": "My Custom Visualization",
    "logoUrl": "https://example.com/logo.png",
    "company": "My Company",
    "description": "A custom visualization for Google Data Studio",
    "resources": {
      "js": ["index.js"],
      "css": ["index.css"],
      "html": "index.html"
    }
  }
}

Crear l'Estructura HTML

L'arxiu index.html defineix l'estructura bàsica de la teva visualització.

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="index.css">
</head>
<body>
  <div id="chart"></div>
  <script src="index.js"></script>
</body>
</html>

Estilitzar amb CSS

L'arxiu index.css defineix els estils de la teva visualització.

#chart {
  width: 100%;
  height: 100%;
}

Afegir la Lògica amb JavaScript

L'arxiu index.js conté la lògica per renderitzar la teva visualització.

function drawChart(data) {
  const chartDiv = document.getElementById('chart');
  chartDiv.innerHTML = ''; // Clear previous content

  // Example: Create a simple bar chart
  const chartData = data.tables.DEFAULT;
  const labels = chartData.map(row => row[0]);
  const values = chartData.map(row => row[1]);

  const chart = document.createElement('div');
  chart.style.display = 'flex';

  labels.forEach((label, index) => {
    const bar = document.createElement('div');
    bar.style.height = `${values[index]}px`;
    bar.style.width = '20px';
    bar.style.margin = '0 5px';
    bar.style.backgroundColor = 'blue';
    chart.appendChild(bar);
  });

  chartDiv.appendChild(chart);
}

// Listen for data updates
dscc.subscribeToData(drawChart);

Publicació i Ús a Data Studio

Publicar a Google Cloud Storage

  1. Puja els teus arxius a un bucket a Google Cloud Storage.
  2. Assegura't que els arxius siguin accessibles públicament.

Afegir la Visualització a Data Studio

  1. Obre Google Data Studio.
  2. Ves a "Community Visualizations" i selecciona "Manage".
  3. Afegeix la URL del teu manifest.json.

Conclusió

Integrar Community Visualizations a Google Data Studio et permet crear gràfics personalitzats que poden satisfer necessitats específiques de visualització de dades. Amb coneixements bàsics d'HTML, CSS i JavaScript, pots ampliar significativament les capacitats de Data Studio i compartir les teves creacions amb la comunitat.

Recorda seguir les millors pràctiques de desenvolupament web i mantenir les teves visualitzacions optimitzades per a un rendiment òptim. ¡Feliç visualització!
