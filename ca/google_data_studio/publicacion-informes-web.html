Introducció

En aquesta secció, aprendràs com publicar els teus informes de Google Data Studio al web. Publicar informes permet compartir les teves anàlisis i visualitzacions amb un públic més ampli, facilitant la presa de decisions basada en dades.

Passos per publicar un informe

Per publicar un informe a Google Data Studio, segueix aquests passos:

  1. Obre el teu informe a Google Data Studio:

    • Navega a Google Data Studio.
    • Selecciona l'informe que desitges publicar.

  2. Configura els permisos d'accés:

    • Fes clic al botó de "Compartir" a la cantonada superior dreta.
    • Configura els permisos d'accés segons les teves necessitats:
      • Privat: Només les persones específiques poden veure l'informe.
      • Qualsevol amb l'enllaç: Qualsevol persona amb l'enllaç pot veure l'informe.
      • Públic al web: Qualsevol persona al web pot trobar i veure l'informe.

  3. Publica l'informe:

    • Fes clic a "Arxiu" al menú superior.
    • Selecciona "Incorporar informe".
    • Copia el codi d'inserció proporcionat.

  4. Incorpora l'informe al teu lloc web:

    • Obre l'editor del teu lloc web.
    • Enganxa el codi d'inserció al lloc on desitges que aparegui l'informe.

Exemple de codi d'inserció

A continuació, es mostra un exemple de com es veuria el codi d'inserció d'un informe de Google Data Studio:

<iframe width="600" height="450" src="https://datastudio.google.com/embed/reporting/1A2B3C4D5E6F7G8H9I0J/page/1M" frameborder="0" style="border:0" allowfullscreen></iframe>

Explicació del codi

  • <iframe>: Aquest és l'element HTML utilitzat per incrustar contingut d'una altra font en una pàgina web.
  • width i height: Aquestes propietats defineixen la mida de l'iframe.
  • src: Aquesta propietat conté la URL de l'informe de Google Data Studio que desitges incrustar.
  • frameborder: Aquesta propietat defineix si l'iframe tindrà una vora. Un valor de "0" significa que no hi haurà vora.
  • style: Aquí pots afegir estils CSS addicionals. En aquest cas, s'estableix border:0 per assegurar-se que no hi hagi vora.
  • allowfullscreen: Aquesta propietat permet que l'iframe es vegi en pantalla completa.

Consideracions de seguretat

En publicar informes al web, és important tenir en compte les següents consideracions de seguretat:

  • Dades sensibles: Assegura't que l'informe no contingui dades sensibles o confidencials abans de fer-lo públic.
  • Permisos d'accés: Configura els permisos d'accés adequadament per controlar qui pot veure l'informe.
  • Actualitzacions: Tingues en compte que qualsevol canvi a l'informe es reflectirà automàticament a la versió publicada.

Conclusió

Publicar informes al web amb Google Data Studio és una excel·lent manera de compartir les teves anàlisis i visualitzacions amb un públic més ampli. Seguint els passos descrits, pots configurar els permisos d'accés i utilitzar el codi d'inserció per incorporar els teus informes a qualsevol lloc web. Recorda sempre considerar la seguretat i privacitat de les dades abans de fer un informe públic.
