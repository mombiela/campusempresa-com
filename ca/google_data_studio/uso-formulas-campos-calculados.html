Introducció

En aquesta secció del curs, aprendràs a utilitzar fórmules i camps calculats a Google Data Studio. Aquests elements són essencials per transformar i analitzar dades de manera efectiva, permetent-te crear informes més dinàmics i personalitzats.

Conceptes Clau

  • Camps Calculats: Són camps que es creen utilitzant fórmules per realitzar càlculs o transformacions en les dades existents.
  • Fórmules: Expressions matemàtiques o lògiques que s'utilitzen per manipular dades dins dels camps calculats.

Creació de Camps Calculats

Per crear un camp calculat a Google Data Studio, segueix aquests passos:

  1. Obre el teu informe a Google Data Studio.
  2. Selecciona la font de dades a la qual desitges afegir un camp calculat.
  3. Fes clic a "Afegir un camp" a la cantonada superior dreta.
  4. Introdueix un nom per al nou camp.
  5. Escriu la fórmula que desitges utilitzar.
  6. Fes clic a "Crear camp".

Exemple de Camp Calculat

Suposem que tens una font de dades amb les següents columnes: Vendes, Costos i Data. Vols crear un camp calculat per obtenir el benefici (Benefici = Vendes - Costos).

  1. Nom del camp: Benefici
  2. Fórmula: Vendes - Costos
Benefici = Vendes - Costos

Tipus de Fórmules

Fórmules Matemàtiques

  • Suma: Vendes + Impostos
  • Resta: Vendes - Costos
  • Multiplicació: Preu * Quantitat
  • Divisió: Ingressos / Nombre de Vendes

Fórmules Lògiques

  • IF: IF(Condició, Valor si veritable, Valor si fals)

Exemple:

IF(Vendes > 1000, "Alta", "Baixa")

Fórmules de Text

  • CONCAT: CONCAT(Camp1, Camp2)
  • LOWER: LOWER(Camp)
  • UPPER: UPPER(Camp)

Exemple:

CONCAT(LOWER(Nom), "@empresa.com")

Fórmules de Data

  • YEAR: YEAR(Data)
  • MONTH: MONTH(Data)
  • DAY: DAY(Data)

Exemple:

YEAR(Data)

Exemples Pràctics

Exemple 1: Calcular el Marge de Benefici

Suposem que tens les columnes Ingressos i Costos. Vols calcular el marge de benefici com un percentatge.

  1. Nom del camp: Marge de Benefici
  2. Fórmula: ((Ingressos - Costos) / Ingressos) * 100
Marge_de_Benefici = ((Ingressos - Costos) / Ingressos) * 100

Exemple 2: Classificar Vendes

Vols classificar les vendes en "Alta", "Mitjana" i "Baixa" segons el seu valor.

  1. Nom del camp: Classificació de Vendes
  2. Fórmula:
IF(Vendes > 1000, "Alta", IF(Vendes > 500, "Mitjana", "Baixa"))

Comparació de Fórmules i Camps Calculats

| Tipus de Fórmula | Exemple | Descripció | |------------------|---------|------------| | Matemàtica | Vendes - Costos | Realitza operacions aritmètiques. | | Lògica | IF(Vendes > 1000, "Alta", "Baixa") | Avalua condicions i retorna valors basats en el resultat. | | Text | CONCAT(Nom, Cognom) | Manipula cadenes de text. | | Data | YEAR(Data) | Extreu parts específiques d'una data. |

Conclusió

L'ús de fórmules i camps calculats a Google Data Studio és una habilitat essencial per a qualsevol professional que treballi amb dades. Aquestes eines et permeten transformar i analitzar dades de manera més efectiva, creant informes més dinàmics i personalitzats. Practica creant els teus propis camps calculats i experimenta amb diferents fórmules per veure com poden millorar els teus informes.
