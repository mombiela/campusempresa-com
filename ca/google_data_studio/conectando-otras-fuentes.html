Introducció

En aquesta secció del curs sobre Google Data Studio, aprendràs com connectar i utilitzar diverses fonts de dades externes. Google Data Studio permet integrar dades de múltiples orígens, cosa que facilita la creació d'informes i dashboards complets i personalitzats.

Fonts de Dades Compatibles

Google Data Studio és compatible amb una àmplia varietat de fonts de dades. Algunes de les més comunes inclouen:

  • Google Sheets
  • Google Analytics
  • BigQuery
  • MySQL
  • PostgreSQL
  • CSV Uploads
  • Facebook Ads
  • YouTube Analytics
  • Salesforce

Connectant Google Sheets

Google Sheets és una de les fonts de dades més populars a causa de la seva simplicitat i accessibilitat. Per connectar Google Sheets a Google Data Studio, segueix aquests passos:

  1. Obrir Google Data Studio: Inicia sessió al teu compte de Google Data Studio.
  2. Crear una Font de Dades: Fes clic al botó "Crear" i selecciona "Font de dades".
  3. Seleccionar Google Sheets: A la llista de connectors, selecciona "Google Sheets".
  4. Autoritzar l'Accés: Autoritza Google Data Studio per accedir a les teves fulles de càlcul de Google.
  5. Seleccionar l'Arxiu i la Fulla: Tria l'arxiu de Google Sheets i la fulla específica que vols connectar.
  6. Configurar els Camps: Revisa i configura els camps de dades segons sigui necessari.
# Exemple de configuració de camps
{
  "fieldName": "Data",
  "fieldType": "DATE"
},
{
  "fieldName": "Vendes",
  "fieldType": "NUMBER"
}

Connectant BigQuery

BigQuery és una poderosa eina d'anàlisi de dades de Google Cloud. Per connectar BigQuery a Google Data Studio:

  1. Obrir Google Data Studio: Inicia sessió al teu compte de Google Data Studio.
  2. Crear una Font de Dades: Fes clic al botó "Crear" i selecciona "Font de dades".
  3. Seleccionar BigQuery: A la llista de connectors, selecciona "BigQuery".
  4. Autoritzar l'Accés: Autoritza Google Data Studio per accedir als teus projectes de BigQuery.
  5. Seleccionar el Projecte i el Conjunt de Dades: Tria el projecte i el conjunt de dades que vols connectar.
  6. Configurar els Camps: Revisa i configura els camps de dades segons sigui necessari.
# Exemple de consulta SQL en BigQuery
SELECT
  data,
  vendes
FROM
  `el_meu_projecte.el_meu_dataset.la_meva_taula`
WHERE
  data BETWEEN '2023-01-01' AND '2023-12-31'

Connectant MySQL

Per connectar una base de dades MySQL a Google Data Studio:

  1. Obrir Google Data Studio: Inicia sessió al teu compte de Google Data Studio.
  2. Crear una Font de Dades: Fes clic al botó "Crear" i selecciona "Font de dades".
  3. Seleccionar MySQL: A la llista de connectors, selecciona "MySQL".
  4. Configurar la Connexió: Introdueix els detalls de la connexió, com l'adreça del servidor, el port, el nom d'usuari i la contrasenya.
  5. Seleccionar la Base de Dades i la Taula: Tria la base de dades i la taula que vols connectar.
  6. Configurar els Camps: Revisa i configura els camps de dades segons sigui necessari.
# Exemple de consulta SQL en MySQL
SELECT
  data,
  vendes
FROM
  la_meva_taula
WHERE
  data BETWEEN '2023-01-01' AND '2023-12-31'

Exercicis Pràctics

  1. Connectar una Fulla de Google Sheets: Crea una nova font de dades a Google Data Studio utilitzant una fulla de Google Sheets que contingui dades de vendes mensuals.
  2. Connectar a BigQuery: Configura una connexió a BigQuery i realitza una consulta per obtenir dades de vendes anuals.
  3. Connectar a MySQL: Estableix una connexió a una base de dades MySQL i extreu dades d'una taula específica.

Conclusió

Connectar diverses fonts de dades a Google Data Studio et permet crear informes i dashboards més complets i personalitzats. Ja sigui que utilitzis Google Sheets, BigQuery, MySQL o altres fonts, el procés de connexió és relativament senzill i et brinda la flexibilitat de treballar amb les dades que necessites. Practica connectant diferents fonts de dades per familiaritzar-te amb el procés i maximitzar el potencial dels teus informes a Google Data Studio.
