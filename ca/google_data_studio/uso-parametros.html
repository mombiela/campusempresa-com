Introducció

A Google Data Studio, els paràmetres són una eina poderosa que permet als usuaris interactuar amb les dades de manera dinàmica. Els paràmetres poden ser utilitzats per filtrar dades, canviar visualitzacions i personalitzar informes segons les necessitats de l'usuari final.

Conceptes Clau

  • Paràmetre: Una variable que pot ser utilitzada per canviar dinàmicament el contingut d'un informe.
  • Control de Paràmetre: Un component de la interfície d'usuari que permet als usuaris establir el valor d'un paràmetre.
  • Camp Calculat: Un camp que utilitza el valor del paràmetre per realitzar càlculs o filtrar dades.

Creació de Paràmetres

Per crear un paràmetre a Google Data Studio, segueix aquests passos:

  1. Obre el teu informe a Google Data Studio.
  2. Fes clic a "Recursos" a la barra de menú.
  3. Selecciona "Administrar paràmetres".
  4. Fes clic a "Afegir un paràmetre".
  5. Defineix el nom, tipus de dades i valor predeterminat del paràmetre.

Exemple de Creació de Paràmetre

Nom: Rang de Dates
Tipus de Dades: Data
Valor Predeterminat: 2023-01-01

Ús de Paràmetres en Camps Calculats

Un cop has creat un paràmetre, pots utilitzar-lo en camps calculats per filtrar o transformar dades.

Exemple de Camp Calculat

Suposem que tens un paràmetre anomenat Rang de Dates. Pots crear un camp calculat per filtrar les dades segons aquest paràmetre:

CASE
  WHEN Data >= @Rang_de_Dates THEN "Dins del Rang"
  ELSE "Fora del Rang"
END

En aquest exemple, el camp calculat classifica les dates com "Dins del Rang" o "Fora del Rang" segons el valor del paràmetre Rang de Dates.

Controls de Paràmetre

Per permetre que els usuaris canviïn el valor d'un paràmetre, has d'afegir un control de paràmetre al teu informe.

Passos per Afegir un Control de Paràmetre

  1. Selecciona l'informe on desitges afegir el control.
  2. Fes clic a "Afegir un control" a la barra d'eines.
  3. Selecciona "Control de Paràmetre".
  4. Configura el control perquè estigui vinculat al paràmetre que has creat.

Exemple de Control de Paràmetre

Si has creat un paràmetre Rang de Dates, pots afegir un control de paràmetre de tipus "Selector de Data" per permetre als usuaris seleccionar una data.

Exemples Pràctics

Exemple 1: Filtrat Dinàmic de Dades

Suposem que tens un informe de vendes i desitges permetre als usuaris filtrar les vendes per un rang de dates específic.

  1. Crea un paràmetre Data Inici i un altre Data Fi.
  2. Utilitza aquests paràmetres en un camp calculat per filtrar les vendes:
CASE
  WHEN Data >= @Data_Inici AND Data <= @Data_Fi THEN Vendes
  ELSE NULL
END
  1. Afegeix controls de paràmetre per Data Inici i Data Fi al teu informe.

Exemple 2: Canvi de Visualització

Pots utilitzar paràmetres per canviar dinàmicament el tipus de visualització al teu informe.

  1. Crea un paràmetre Tipus de Visualització amb valors possibles com "Gràfic de Barres" i "Gràfic de Línies".
  2. Utilitza aquest paràmetre en un camp calculat per canviar la visualització:
CASE
  WHEN @Tipus_de_Visualització = "Gràfic de Barres" THEN "Bar"
  WHEN @Tipus_de_Visualització = "Gràfic de Línies" THEN "Line"
  ELSE "Bar"
END
  1. Afegeix un control de paràmetre per Tipus de Visualització al teu informe.

Conclusió

L'ús de paràmetres a Google Data Studio permet als usuaris crear informes interactius i personalitzats. En comprendre com crear i utilitzar paràmetres, pots oferir als usuaris finals una experiència més dinàmica i flexible. Practica creant i utilitzant paràmetres en els teus propis informes per aprofitar al màxim aquesta funcionalitat.
