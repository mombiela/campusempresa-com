Introducció

En aquesta secció del curs, aprendràs com crear i visualitzar sèries temporals utilitzant Google Data Studio. Les sèries temporals són fonamentals per analitzar dades que canvien amb el temps, com vendes diàries, visites a un lloc web, o qualsevol altre tipus de dada que es registri de manera contínua.

Conceptes Clau

  • Sèrie Temporal: Conjunt de dades ordenades cronològicament.
  • Dimensió de Data: Camp que conté la informació temporal.
  • Gràfics de Sèrie Temporal: Visualitzacions que mostren com canvien les dades al llarg del temps.

Preparació de Dades

Abans de crear una sèrie temporal a Google Data Studio, és crucial assegurar-se que les dades estiguin correctament formatades i que continguin una dimensió de data.

  • Format de Data: Assegura't que el teu camp de data estigui en un format reconeixible per Google Data Studio (per exemple, YYYY-MM-DD).
  • Dades Contínues: Verifica que les dades no tinguin buits significatius que puguin afectar la visualització.

Creació d'una Sèrie Temporal

Pas 1: Connectar les Dades

  1. Obre Google Data Studio.
  2. Selecciona "Crear" i després "Informe".
  3. Connecta la teva font de dades (per exemple, Google Sheets, BigQuery, etc.).

Pas 2: Afegir un Gràfic de Sèrie Temporal

  1. Al panell d'eines, selecciona "Gràfic de Sèrie Temporal".
  2. Arrossega el gràfic a l'àrea de treball.

Pas 3: Configurar el Gràfic

  1. Dimensió de Data: Arrossega el camp de data a la secció de "Dimensió".
  2. Mètrica: Arrossega la mètrica que desitges analitzar (per exemple, vendes, visites, etc.) a la secció de "Mètrica".

Exemple de Configuració

Suposem que tens un conjunt de dades amb les següents columnes: Data, Vendes.

Data        | Vendes
------------|-------
2023-01-01  | 100
2023-01-02  | 150
2023-01-03  | 200

Per configurar el gràfic:

  • Dimensió: Data
  • Mètrica: Vendes

Personalització del Gràfic

  • Interval de Temps: Pots ajustar l'interval de temps (diari, setmanal, mensual) a la configuració del gràfic.
  • Línies de Tendència: Afegeix línies de tendència per identificar patrons a llarg termini.
  • Anotacions: Afegeix anotacions per marcar esdeveniments importants a la sèrie temporal.

Exemple de Codi

Encara que Google Data Studio no requereix codi per crear gràfics, aquí tens un exemple de com podries preparar les teves dades a Google Sheets abans d'importar-les:

Data        | Vendes
------------|-------
2023-01-01  | 100
2023-01-02  | 150
2023-01-03  | 200

Assegura't que la columna Data estigui en format de data i que les dades no tinguin buits.

Exercici Pràctic

  1. Connecta una font de dades que contingui una dimensió de data i una mètrica.
  2. Crea un gràfic de sèrie temporal a Google Data Studio.
  3. Configura el gràfic amb la dimensió de data i la mètrica.
  4. Personalitza el gràfic afegint línies de tendència i anotacions.

Conclusió

Crear sèries temporals a Google Data Studio és una habilitat essencial per analitzar dades que canvien amb el temps. Assegura't que les teves dades estiguin ben formatades i utilitza les eines de personalització per obtenir insights valuosos. Practica amb diferents conjunts de dades per familiaritzar-te amb les diverses opcions i configuracions disponibles.
