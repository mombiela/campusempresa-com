Executar Suites de Proves
Organitzar Tests ►

En aquesta secció, explorarem com executar suites de proves utilitzant JUnit. Les suites de proves et permeten agrupar múltiples classes de prova i executar-les juntes, cosa que és particularment útil per gestionar projectes grans. Cobrirem els conceptes bàsics de la creació i execució de suites de proves, així com algunes tècniques avançades.

Què és una Suite de Proves?

Una suite de proves és una col·lecció de casos de prova que es pretenen executar junts. Això és útil per a:

  • Organitzar proves en grups lògics.
  • Executar múltiples proves amb una sola comanda.
  • Assegurar-se que totes les proves passen abans de desplegar el codi.

Crear una Suite de Proves

Per crear una suite de proves en JUnit, necessites utilitzar l'anotació @Suite juntament amb les anotacions @SelectClasses o @SelectPackages.

Utilitzant `@SelectClasses`

L'anotació @SelectClasses et permet especificar classes de prova individuals per incloure a la suite.

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.SelectClasses;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectClasses({TestClass1.class, TestClass2.class})
public class MyTestSuite {
}

En aquest exemple:

  • @RunWith(JUnitPlatform.class) indica a JUnit que utilitzi el corredor de la Plataforma JUnit.
  • @Suite indica que aquesta classe és una suite de proves.
  • @SelectClasses especifica les classes de prova per incloure a la suite.

Utilitzant `@SelectPackages`

L'anotació @SelectPackages et permet incloure totes les classes de prova en un o més paquets.

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.SelectPackages;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectPackages("com.example.tests")
public class MyPackageTestSuite {
}

En aquest exemple:

  • @SelectPackages("com.example.tests") inclou totes les classes de prova en el paquet com.example.tests.

Executar la Suite de Proves

Un cop hagis creat la teva suite de proves, pots executar-la utilitzant el teu IDE, eina de construcció o línia de comandes.

Executar amb un IDE

La majoria dels IDEs com IntelliJ IDEA, Eclipse i NetBeans et permeten executar suites de proves directament:

  • Fes clic dret sobre la classe de la suite de proves.
  • Selecciona "Run" o "Run as JUnit Test".

Executar amb Maven

Si estàs utilitzant Maven, pots executar la teva suite de proves amb la següent comanda:

mvn test -Dtest=MyTestSuite

Executar amb Gradle

Si estàs utilitzant Gradle, pots executar la teva suite de proves amb la següent comanda:

gradle test --tests "MyTestSuite"

Tècniques Avançades

Filtrar Proves

Pots filtrar quines proves executar dins d'una suite utilitzant etiquetes o filtres personalitzats.

Utilitzant Etiquetes

Primer, etiqueta els teus mètodes de prova:

import org.junit.jupiter.api.Tag;
import org.junit.jupiter.api.Test;

public class TaggedTests {

    @Test
    @Tag("fast")
    void fastTest() {
        // codi de prova ràpida
    }

    @Test
    @Tag("slow")
    void slowTest() {
        // codi de prova lenta
    }
}

Després, configura la teva suite per incloure o excloure etiquetes específiques:

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.IncludeTags;
import org.junit.platform.suite.api.SelectClasses;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectClasses(TaggedTests.class)
@IncludeTags("fast")
public class FastTestsSuite {
}

Execució Paral·lela

JUnit 5 suporta l'execució paral·lela de proves, que es pot configurar en el fitxer junit-platform.properties:

junit.jupiter.execution.parallel.enabled = true
junit.jupiter.execution.parallel.mode.default = concurrent

Aquesta configuració habilitarà l'execució paral·lela per a totes les proves, incloses les de les suites de proves.

Conclusió

Executar suites de proves en JUnit et permet gestionar i executar múltiples proves de manera eficient. Agrupant proves en suites, pots assegurar-te d'una prova exhaustiva i agilitzar el teu flux de treball de desenvolupament. Tant si estàs executant proves des d'un IDE, Maven o Gradle, JUnit proporciona opcions flexibles per adaptar-se a les teves necessitats. Les tècniques avançades com el filtratge i l'execució paral·lela milloren encara més les teves capacitats de prova.
