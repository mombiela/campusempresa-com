Introducció a les Suites de Proves en JUnit

Una suite de proves és una col·lecció de casos de prova que es poden executar junts. En JUnit, les suites de proves permeten agrupar múltiples classes de prova i executar-les com una sola unitat. Això és particularment útil per organitzar proves i assegurar-se que les proves relacionades s'executen juntes.

Beneficis d'Utilitzar Suites de Proves

  • Organització: Agrupar proves relacionades per a una millor estructura.
  • Eficiència: Executar múltiples proves amb una sola comanda.
  • Manteniment: Més fàcil de gestionar i actualitzar proves.
  • Cobertura: Assegurar-se que totes les proves relacionades s'executen, millorant la cobertura de proves.

Creació d'una Suite de Proves Bàsica

Per crear una suite de proves en JUnit, cal utilitzar l'anotació @Suite juntament amb les anotacions @SelectClasses o @SelectPackages.

Exemple: Utilitzant `@SelectClasses`

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.SelectClasses;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectClasses({TestClass1.class, TestClass2.class})
public class BasicTestSuite {
    // Aquesta classe roman buida, s'utilitza només com a contenidor per a les anotacions anteriors
}
  • Explicació:
    • @RunWith(JUnitPlatform.class): Especifica que s'ha d'utilitzar la plataforma JUnit per executar la suite.
    • @Suite: Marca la classe com una suite de proves.
    • @SelectClasses: Especifica les classes de prova a incloure en la suite.

Exemple: Utilitzant `@SelectPackages`

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.SelectPackages;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectPackages("com.example.tests")
public class PackageTestSuite {
    // Aquesta classe roman buida, s'utilitza només com a contenidor per a les anotacions anteriors
}
  • Explicació:
    • @SelectPackages: Especifica els paquets que contenen les classes de prova a incloure en la suite.

Configuració Avançada de la Suite de Proves

JUnit permet configuracions més avançades, com incloure o excloure proves específiques basades en etiquetes.

Exemple: Utilitzant Etiquetes

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.IncludeTags;
import org.junit.platform.suite.api.SelectClasses;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectClasses({TestClass1.class, TestClass2.class})
@IncludeTags("fast")
public class TaggedTestSuite {
    // Aquesta classe roman buida, s'utilitza només com a contenidor per a les anotacions anteriors
}
  • Explicació:
    • @IncludeTags: Inclou només les proves amb l'etiqueta especificada.

Exemple: Excloent Etiquetes

import org.junit.platform.runner.JUnitPlatform;
import org.junit.platform.suite.api.ExcludeTags;
import org.junit.platform.suite.api.SelectClasses;
import org.junit.platform.suite.api.Suite;
import org.junit.runner.RunWith;

@RunWith(JUnitPlatform.class)
@Suite
@SelectClasses({TestClass1.class, TestClass2.class})
@ExcludeTags("slow")
public class ExcludedTaggedTestSuite {
    // Aquesta classe roman buida, s'utilitza només com a contenidor per a les anotacions anteriors
}
  • Explicació:
    • @ExcludeTags: Exclou les proves amb l'etiqueta especificada.

Executant Suites de Proves

Pots executar suites de proves utilitzant el teu IDE, eines de construcció com Maven o Gradle, o directament des de la línia de comandes.

Exemple: Executant amb Maven

Afegeix el següent al teu pom.xml:

<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter-engine</artifactId>
    <version>5.7.0</version>
    <scope>test</scope>
</dependency>

Després, executa les proves utilitzant:

mvn test

Exemple: Executant amb Gradle

Afegeix el següent al teu build.gradle:

testImplementation 'org.junit.jupiter:junit-jupiter-engine:5.7.0'

Després, executa les proves utilitzant:

gradle test

Conclusió

Crear suites de proves en JUnit és una manera poderosa d'organitzar i gestionar les teves proves. Agrupant proves relacionades, pots millorar l'eficiència i el manteniment del teu codi de prova. Tant si utilitzes suites de proves bàsiques com configuracions avançades amb etiquetes, JUnit proporciona les eines que necessites per assegurar una cobertura de proves completa.
