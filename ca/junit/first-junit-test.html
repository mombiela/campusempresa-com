Primer Test JUnit
Comprendre les Assertions ►

En aquesta secció, cobrirem els conceptes bàsics per escriure el teu primer test JUnit. JUnit és un marc de proves popular per a Java que ajuda els desenvolupadors a escriure i executar proves repetibles. Al final d'aquesta secció, seràs capaç de crear un cas de prova JUnit senzill i entendre els seus components bàsics.

Conceptes Clau

  • Marc JUnit: Un marc de proves per a Java.
  • Cas de Prova: Una unitat única de prova.
  • Afirmacions: Declaracions que comproven si una condició és certa.
  • Anotacions: Marcadors especials en el codi que proporcionen metadades sobre els mètodes de prova.

Configurant JUnit

Abans d'escriure el teu primer test JUnit, has de configurar JUnit en el teu projecte. Això es pot fer utilitzant una eina de construcció com Maven o Gradle, o afegint manualment la biblioteca JUnit al teu projecte.

Utilitzant Maven

Afegeix la següent dependència al teu fitxer pom.xml:

<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>

Utilitzant Gradle

Afegeix el següent al teu fitxer build.gradle:

dependencies {
    testImplementation 'junit:junit:4.13.2'
}

Escriure el Teu Primer Cas de Prova

Escrivim un cas de prova senzill per entendre l'estructura bàsica d'un test JUnit.

Codi d'Exemple

import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {

    @Test
    public void testAddition() {
        Calculator calculator = new Calculator();
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

Explicació

  1. Declaracions d'Importació:

    • import org.junit.Test; - Importa l'anotació @Test.
    • import static org.junit.Assert.assertEquals; - Importa el mètode assertEquals per a les afirmacions.

  2. Classe de Prova:

    • public class CalculatorTest - Defineix una classe de prova anomenada CalculatorTest.

  3. Mètode de Prova:

    • @Test - Marca el mètode testAddition com un mètode de prova.
    • public void testAddition() - Defineix un mètode de prova anomenat testAddition.

  4. Afirmacions:

    • assertEquals(5, result); - Comprova si el resultat de la suma és igual a 5.

Classe Calculator

Perquè la prova anterior funcioni, necessites una classe Calculator senzilla:

public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

Executant la Prova

Pots executar la prova utilitzant el corredor de proves integrat del teu IDE o des de la línia de comandes utilitzant Maven o Gradle.

Utilitzant IDE

  • Fes clic dret sobre la classe de prova i selecciona "Run" o "Run as JUnit Test".

Utilitzant Maven

Executa la següent comanda en el teu terminal:

mvn test

Utilitzant Gradle

Executa la següent comanda en el teu terminal:

gradle test

Conclusió

En aquesta secció, has après com configurar JUnit en el teu projecte i escriure el teu primer cas de prova senzill. També has après sobre els components bàsics d'un test JUnit, incloent mètodes de prova, afirmacions i anotacions. Escriure proves és una habilitat essencial per a qualsevol desenvolupador, i dominar JUnit t'ajudarà a assegurar que el teu codi sigui fiable i lliure d'errors.
