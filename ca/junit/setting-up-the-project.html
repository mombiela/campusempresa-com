El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, cobrirem com configurar un projecte per utilitzar JUnit, un popular marc de proves per a Java. Aquesta configuració és crucial tant per a principiants com per a usuaris avançats per assegurar un flux de treball suau quan s'escriuen i s'executen proves.

Prerequisits

Abans de començar, assegura't de tenir instal·lat el següent al teu ordinador:

  • Java Development Kit (JDK) 8 o superior
  • Un Entorn de Desenvolupament Integrat (IDE) com IntelliJ IDEA, Eclipse o NetBeans
  • Maven o Gradle (opcional però recomanat per a la gestió de dependències)

Creació d'un Nou Projecte

Utilitzant IntelliJ IDEA

  1. Obre IntelliJ IDEA i fes clic a New Project.
  2. Selecciona Java i fes clic a Next.
  3. Nomena el teu projecte i tria una ubicació per desar-lo.
  4. Selecciona el JDK que tens instal·lat i fes clic a Next.
  5. Finalitza la configuració fent clic a Finish.

Utilitzant Eclipse

  1. Obre Eclipse i ves a File > New > Java Project.
  2. Introdueix el nom del projecte i fes clic a Next.
  3. Configura el JRE (Java Runtime Environment) si és necessari.
  4. Fes clic a Finish per crear el projecte.

Afegir JUnit al Teu Projecte

Utilitzant Maven

  1. Obre el fitxer pom.xml del teu projecte.

  2. Afegeix la següent dependència dins de l'etiqueta <dependencies>:

    <dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>

  3. Desa el fitxer pom.xml i deixa que Maven descarregui les dependències.

Utilitzant Gradle

  1. Obre el fitxer build.gradle del teu projecte.

  2. Afegeix la següent dependència dins del bloc dependencies:

    testImplementation 'junit:junit:4.13.2'

  3. Desa el fitxer build.gradle i deixa que Gradle descarregui les dependències.

Afegir Manualment el JAR de JUnit

  1. Descarrega el fitxer JAR de JUnit des del lloc web oficial de JUnit.
  2. Afegeix el fitxer JAR al classpath del teu projecte:
    • A IntelliJ IDEA: Fes clic dret sobre el teu projecte > Open Module Settings > Libraries > + > Afegeix el fitxer JAR.
    • A Eclipse: Fes clic dret sobre el teu projecte > Build Path > Add External Archives > Afegeix el fitxer JAR.

Verificació de la Configuració

Per verificar que JUnit està correctament configurat, crea un cas de prova senzill.

Exemple de Cas de Prova

  1. Crea una nova classe Java anomenada Calculator:

    public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

  2. Crea una classe de prova anomenada CalculatorTest:

    import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {
    @Test
    public void testAdd() {
        Calculator calculator = new Calculator();
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

  3. Executa la prova:

    • A IntelliJ IDEA: Fes clic dret sobre la classe CalculatorTest > Run CalculatorTest.
    • A Eclipse: Fes clic dret sobre la classe CalculatorTest > Run As > JUnit Test.

Si la prova passa, la teva configuració de JUnit està completa!

Conclusió

Configurar un projecte per a JUnit implica crear un nou projecte Java, afegir JUnit com a dependència i verificar la configuració amb un cas de prova senzill. Tant si utilitzes Maven, Gradle, o afegeixes manualment el fitxer JAR, el procés és senzill i assegura que estàs preparat per escriure i executar proves de manera eficient. Aquesta configuració fonamental servirà com a base per a temes més avançats en les proves amb JUnit.
