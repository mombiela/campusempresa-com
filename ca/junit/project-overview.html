Introducció a JUnit

JUnit és un popular marc de proves per a Java que permet als desenvolupadors escriure i executar proves repetibles. És una eina essencial per garantir la qualitat i la fiabilitat del codi. Aquest curs et portarà des dels conceptes bàsics de JUnit fins a temes avançats, permetent-te escriure proves unitàries efectives per a les teves aplicacions Java.

Conceptes Clau

  • Proves Unitàries: Provar unitats o components individuals d'un programari.
  • Fixtures de Prova: Codi de configuració que s'executa abans i després de cada prova.
  • Afirmacions: Declaracions que comproven si una condició és certa.
  • Executors de Proves: Eines que executen proves i informen dels resultats.
  • Anotacions: Metadades que proporcionen informació sobre els mètodes de prova.

Exemples de Codi

Escriure el Teu Primer Cas de Prova

import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {

    @Test
    public void testAddition() {
        Calculator calculator = new Calculator();
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

Explicació:

  • Declaracions d'Importació: Importar les classes necessàries de JUnit.
  • Mètode de Prova: Anotar el mètode amb @Test.
  • Afirmacions: Utilitzar assertEquals per comprovar si el resultat és l'esperat.

Utilitzant `@Before` i `@After`

import org.junit.Before;
import org.junit.After;
import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {

    private Calculator calculator;

    @Before
    public void setUp() {
        calculator = new Calculator();
    }

    @After
    public void tearDown() {
        calculator = null;
    }

    @Test
    public void testAddition() {
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

Explicació:

  • @Before: S'executa abans de cada mètode de prova per configurar l'entorn de prova.
  • @After: S'executa després de cada mètode de prova per netejar l'entorn de prova.
