Introducció

Les afirmacions són una part fonamental de les proves unitàries en JUnit. Et permeten verificar que el teu codi es comporta com s'espera. En aquesta secció, cobrirem les afirmacions més comunes utilitzades en JUnit, proporcionant explicacions clares i exemples per ajudar-te a entendre com utilitzar-les de manera efectiva.

Conceptes Clau

  • Afirmacions: Declaracions que comproven si una condició és certa. Si la condició és falsa, l'afirmació falla i el cas de prova es marca com a fallit.
  • JUnit: Un marc de proves popular per a Java que proporciona diversos mètodes d'afirmació per facilitar les proves unitàries.

Afirmacions Comunes

assertEquals

El mètode assertEquals comprova si dos valors són iguals. És una de les afirmacions més utilitzades en JUnit.

import static org.junit.jupiter.api.Assertions.assertEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testAddition() {
        int result = 2 + 3;
        assertEquals(5, result, "2 + 3 hauria de ser igual a 5");
    }
}

En aquest exemple, assertEquals comprova si el resultat de 2 + 3 és igual a 5. Si no ho és, la prova fallarà amb el missatge "2 + 3 hauria de ser igual a 5".

assertNotEquals

El mètode assertNotEquals comprova si dos valors no són iguals.

import static org.junit.jupiter.api.Assertions.assertNotEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testSubtraction() {
        int result = 5 - 3;
        assertNotEquals(1, result, "5 - 3 no hauria de ser igual a 1");
    }
}

Aquí, assertNotEquals assegura que 5 - 3 no és igual a 1. Si ho és, la prova fallarà.

assertTrue

El mètode assertTrue comprova si una condició és certa.

import static org.junit.jupiter.api.Assertions.assertTrue;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testIsPositive() {
        int number = 5;
        assertTrue(number > 0, "El nombre hauria de ser positiu");
    }
}

En aquest cas, assertTrue verifica que el nombre és positiu. Si el nombre no és més gran que 0, la prova fallarà.

assertFalse

El mètode assertFalse comprova si una condició és falsa.

import static org.junit.jupiter.api.Assertions.assertFalse;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testIsNegative() {
        int number = -5;
        assertFalse(number >= 0, "El nombre hauria de ser negatiu");
    }
}

Aquí, assertFalse assegura que el nombre no és més gran o igual a 0. Si ho és, la prova fallarà.

assertNull

El mètode assertNull comprova si un objecte és nul.

import static org.junit.jupiter.api.Assertions.assertNull;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testObjectIsNull() {
        Object obj = null;
        assertNull(obj, "L'objecte hauria de ser nul");
    }
}

En aquest exemple, assertNull verifica que l'objecte és nul. Si no ho és, la prova fallarà.

assertNotNull

El mètode assertNotNull comprova si un objecte no és nul.

import static org.junit.jupiter.api.Assertions.assertNotNull;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testObjectIsNotNull() {
        Object obj = new Object();
        assertNotNull(obj, "L'objecte no hauria de ser nul");
    }
}

Aquí, assertNotNull assegura que l'objecte no és nul. Si ho és, la prova fallarà.

assertArrayEquals

El mètode assertArrayEquals comprova si dues matrius són iguals.

import static org.junit.jupiter.api.Assertions.assertArrayEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testArrayEquality() {
        int[] expected = {1, 2, 3};
        int[] actual = {1, 2, 3};
        assertArrayEquals(expected, actual, "Les matrius haurien de ser iguals");
    }
}

En aquest cas, assertArrayEquals verifica que les dues matrius són iguals. Si no ho són, la prova fallarà.

Resum

Les afirmacions són essencials per verificar la correcció del teu codi en les proves de JUnit. Les afirmacions més comunes inclouen assertEquals, assertNotEquals, assertTrue, assertFalse, assertNull, assertNotNull i assertArrayEquals. Cadascuna d'aquestes afirmacions té un propòsit específic i ajuda a assegurar que el teu codi es comporta com s'espera. Entenent i utilitzant aquestes afirmacions de manera efectiva, pots escriure proves més robustes i fiables.
