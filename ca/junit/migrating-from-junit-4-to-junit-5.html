Introducció

Migrar de JUnit 4 a JUnit 5 pot semblar descoratjador, però ofereix nombrosos beneficis com ara una arquitectura millorada, noves característiques i una millor extensibilitat. Aquesta secció et guiarà pel procés de migració pas a pas, assegurant una transició suau.

Diferències Clau Entre JUnit 4 i JUnit 5

Abans d'endinsar-nos en el procés de migració, és essencial entendre les diferències clau entre JUnit 4 i JUnit 5.

  • Arquitectura: JUnit 5 està compost per tres subprojectes: JUnit Platform, JUnit Jupiter i JUnit Vintage.
  • Anotacions: Algunes anotacions han canviat o s'han afegit.
  • Afirmacions i Suposicions: Millorades i més flexibles.
  • Model d'Extensió: Més potent i extensible.

Taula Comparativa

| Característica/Aspecte | JUnit 4 | JUnit 5 | |------------------------|----------------------------------|----------------------------------| | Arquitectura | Monolític | Modular (Platform, Jupiter, Vintage) | | Anotacions | @Test, @Before, @After | @Test, @BeforeEach, @AfterEach, @DisplayName | | Afirmacions | assertEquals, assertTrue | assertEquals, assertTrue, assertAll, assertThrows | | Suposicions | assumeTrue, assumeFalse | assumeTrue, assumeFalse, assumingThat | | Model d'Extensió | Regles | Extensions (més potent) |

Procés de Migració Pas a Pas

1. Actualitza les Dependències

Primer, actualitza les dependències del teu projecte per incloure JUnit 5. Si utilitzes Maven, actualitza el teu pom.xml:

<dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter-engine</artifactId>
    <version>5.8.1</version>
    <scope>test</scope>
</dependency>

Per a Gradle, actualitza el teu build.gradle:

testImplementation 'org.junit.jupiter:junit-jupiter-engine:5.8.1'

2. Reemplaça les Anotacions

Reemplaça les anotacions de JUnit 4 amb els seus equivalents de JUnit 5:

  • @Before -> @BeforeEach
  • @After -> @AfterEach
  • @BeforeClass -> @BeforeAll
  • @AfterClass -> @AfterAll
  • @Ignore -> @Disabled

Exemple

JUnit 4:

import org.junit.Before;
import org.junit.After;
import org.junit.Test;

public class ExampleTest {
    @Before
    public void setUp() {
        // codi de configuració
    }

    @After
    public void tearDown() {
        // codi de desconfiguració
    }

    @Test
    public void testSomething() {
        // codi de prova
    }
}

JUnit 5:

import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.AfterEach;
import org.junit.jupiter.api.Test;

public class ExampleTest {
    @BeforeEach
    public void setUp() {
        // codi de configuració
    }

    @AfterEach
    public void tearDown() {
        // codi de desconfiguració
    }

    @Test
    public void testSomething() {
        // codi de prova
    }
}

3. Actualitza les Afirmacions i Suposicions

JUnit 5 proporciona afirmacions i suposicions més flexibles.

Exemple

JUnit 4:

import static org.junit.Assert.assertEquals;
import static org.junit.Assume.assumeTrue;

public class ExampleTest {
    @Test
    public void testSomething() {
        assumeTrue(System.getProperty("os.name").startsWith("Windows"));
        assertEquals(4, 2 + 2);
    }
}

JUnit 5:

import static org.junit.jupiter.api.Assertions.assertEquals;
import static org.junit.jupiter.api.Assumptions.assumeTrue;

public class ExampleTest {
    @Test
    public void testSomething() {
        assumeTrue(System.getProperty("os.name").startsWith("Windows"));
        assertEquals(4, 2 + 2);
    }
}

4. Utilitza JUnit Vintage

Si tens una gran base de codi i no pots migrar totes les proves alhora, utilitza JUnit Vintage per executar proves de JUnit 4 al costat de les proves de JUnit 5.

Configuració Maven

<dependency>
    <groupId>org.junit.vintage</groupId>
    <artifactId>junit-vintage-engine</artifactId>
    <version>5.8.1</version>
    <scope>test</scope>
</dependency>

Configuració Gradle

testImplementation 'org.junit.vintage:junit-vintage-engine:5.8.1'

Conclusió

Migrar de JUnit 4 a JUnit 5 implica actualitzar les dependències, reemplaçar les anotacions i aprofitar les noves característiques i millores. Seguint els passos descrits en aquesta guia, pots assegurar una transició suau i aprofitar les capacitats potents que ofereix JUnit 5.
