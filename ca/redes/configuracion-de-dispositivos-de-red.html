Introducció

La configuració de dispositius de xarxa és una habilitat essencial per a qualsevol professional de TI. Aquest tema cobreix els conceptes fonamentals i les pràctiques recomanades per configurar routers, switches i altres dispositius de xarxa. Aprendràs a utilitzar interfícies de línia de comandaments (CLI), interfícies gràfiques d'usuari (GUI) i eines de gestió de xarxa per configurar i gestionar aquests dispositius.

Objectius del Tema

  • Comprendre els conceptes bàsics de la configuració de dispositius de xarxa.
  • Aprendre a configurar routers i switches utilitzant CLI i GUI.
  • Familiaritzar-se amb les eines de gestió de xarxa.
  • Aplicar configuracions de xarxa en un entorn pràctic.

Conceptes Clau

  • Router: Dispositiu que dirigeix el trànsit de xarxa entre diferents xarxes.
  • Switch: Dispositiu que connecta múltiples dispositius dins de la mateixa xarxa.
  • CLI (Command Line Interface): Interfície basada en text per configurar dispositius de xarxa.
  • GUI (Graphical User Interface): Interfície gràfica per configurar dispositius de xarxa.
  • VLAN (Virtual Local Area Network): Segmentació lògica d'una xarxa física en múltiples xarxes virtuals.

Configuració de Routers

Accés a la CLI del Router

Per accedir a la CLI d'un router, generalment s'utilitza una connexió sèrie o SSH. A continuació es mostra un exemple de com connectar-se a un router utilitzant SSH:

ssh admin@192.168.1.1

Configuració Bàsica del Router

Un cop connectat, pots començar a configurar el router. Aquí hi ha un exemple de configuració bàsica:

enable
configure terminal
hostname Router1
interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
exit

Configuració de Rutes Estàtiques

Les rutes estàtiques són essencials per dirigir el trànsit de xarxa. Aquí tens un exemple:

ip route 10.0.0.0 255.255.255.0 192.168.1.2

Configuració de Switches

Accés a la CLI del Switch

Similar als routers, pots accedir a la CLI d'un switch utilitzant una connexió sèrie o SSH:

ssh admin@192.168.1.2

Configuració Bàsica del Switch

Aquí hi ha un exemple de configuració bàsica per a un switch:

enable
configure terminal
hostname Switch1
interface GigabitEthernet0/1
switchport mode access
switchport access vlan 10
no shutdown
exit

Configuració de VLANs

Les VLANs permeten segmentar la xarxa física en múltiples xarxes virtuals. Aquí tens un exemple:

vlan 10
name Sales
exit
interface GigabitEthernet0/1
switchport mode access
switchport access vlan 10
exit

Eines de Gestió de Xarxa

  • Cisco Network Assistant: Eina GUI per gestionar dispositius Cisco.
  • SolarWinds Network Configuration Manager: Eina per automatitzar la configuració i gestió de dispositius de xarxa.
  • Wireshark: Eina d'anàlisi de trànsit de xarxa.

Exemple Pràctic

Configuració d'una Xarxa Simple

Imagina que tens una xarxa amb un router i dos switches. Aquí tens un exemple de com configurar-los:

  1. Router:

    enable
configure terminal
hostname Router1
interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
exit

  2. Switch 1:

    enable
configure terminal
hostname Switch1
vlan 10
name Sales
exit
interface GigabitEthernet0/1
switchport mode access
switchport access vlan 10
no shutdown
exit

  3. Switch 2:

    enable
configure terminal
hostname Switch2
vlan 20
name Marketing
exit
interface GigabitEthernet0/1
switchport mode access
switchport access vlan 20
no shutdown
exit

Conclusió
