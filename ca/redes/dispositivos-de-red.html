Introducció

En aquesta secció del curs, explorarem els dispositius de xarxa, que són components essencials en qualsevol infraestructura de xarxa. Aquests dispositius permeten la comunicació i l'intercanvi de dades entre diferents equips i xarxes. Comprendre el seu funcionament i característiques és crucial per a qualsevol professional de TI.

Tipus de Dispositius de Xarxa

Routers

  • Funció Principal: Dirigeixen el trànsit de dades entre diferents xarxes.
  • Característiques:
    • Capacitat per connectar múltiples xarxes.
    • Utilitzen taules d'encaminament per determinar la millor ruta per a les dades.
    • Poden realitzar funcions de seguretat com firewalls.
  • Exemple de Configuració Bàsica: 
    # Configuració d'una interfície en un router Cisco
Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# exit
Router# write memory

Switches

  • Funció Principal: Connecten dispositius dins de la mateixa xarxa local (LAN).
  • Característiques:
    • Operen en la capa 2 del model OSI (capa d'enllaç de dades).
    • Utilitzen adreces MAC per enviar dades al dispositiu correcte.
    • Poden ser gestionats o no gestionats.
  • Exemple de Configuració Bàsica: 
    # Configuració d'una VLAN en un switch Cisco
Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Vendes
Switch(config-vlan)# exit
Switch(config)# interface FastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config-if)# exit
Switch(config)# exit
Switch# write memory

Access Points (AP)

  • Funció Principal: Permeten la connexió sense fils de dispositius a una xarxa cablejada.
  • Característiques:
    • Operen en la capa 2 del model OSI.
    • Poden ser autònoms o controlats per un controlador central.
    • Suporten diferents estàndards Wi-Fi (802.11a/b/g/n/ac).
  • Exemple de Configuració Bàsica: 
    # Configuració d'un SSID en un AP Cisco
AP> enable
AP# configure terminal
AP(config)# interface Dot11Radio0
AP(config-if)# ssid LaMevaXarxaSenseFils
AP(config-if-ssid)# authentication open
AP(config-if-ssid)# authentication key-management wpa
AP(config-if-ssid)# wpa-psk ascii 0 LaMevaClauSecreta
AP(config-if-ssid)# exit
AP(config-if)# exit
AP(config)# exit
AP# write memory

Firewalls

  • Funció Principal: Proporcionen seguretat a la xarxa controlant el trànsit entrant i sortint.
  • Característiques:
    • Poden ser hardware o software.
    • Implementen polítiques de seguretat basades en regles.
    • Poden realitzar inspecció de paquets i filtratge de contingut.
  • Exemple de Configuració Bàsica: 
    # Configuració d'una regla de firewall en un dispositiu Cisco ASA
ASA> enable
ASA# configure terminal
ASA(config)# access-list OUTSIDE_IN extended permit tcp any host 192.168.1.10 eq 80
ASA(config)# access-group OUTSIDE_IN in interface outside
ASA(config)# exit
ASA# write memory

Comparació de Dispositius de Xarxa

| Dispositiu | Capa OSI | Funció Principal | Exemple d'Ús | |---------------|----------------|-----------------------------------------------|---------------------------------------| | Router | Capa 3 (Xarxa) | Dirigir trànsit entre diferents xarxes | Connectar una xarxa domèstica a Internet | | Switch | Capa 2 (Enllaç)| Connectar dispositius dins d'una mateixa LAN | Connectar PCs en una oficina | | Access Point | Capa 2 (Enllaç)| Proveir connectivitat sense fils | Connectar portàtils i smartphones | | Firewall | Capa 3 (Xarxa) | Proveir seguretat mitjançant control de trànsit | Protegir una xarxa empresarial |

Conclusió

Els dispositius de xarxa són fonamentals per a la construcció i manteniment de qualsevol infraestructura de xarxa. Des de routers i switches fins a access points i firewalls, cada dispositiu té un rol específic i característiques que el fan adequat per a diferents tasques. Comprendre com configurar i gestionar aquests dispositius és essencial per a qualsevol professional de TI que treballi en el camp de les xarxes.
