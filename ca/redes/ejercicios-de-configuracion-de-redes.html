Introducció

En aquesta secció del curs, ens enfocarem en exercicis pràctics per a la configuració de xarxes. Aquests exercicis estan dissenyats per proporcionar una comprensió profunda de com configurar i gestionar xarxes en un entorn professional. Els exercicis cobriran des de configuracions bàsiques fins a avançades, utilitzant tant interfícies gràfiques com línies de comandament.

Objectius d'Aprenentatge

  • Comprendre els conceptes bàsics de configuració de xarxes.
  • Configurar dispositius de xarxa com routers i switches.
  • Implementar i gestionar subxarxes.
  • Configurar serveis de xarxa essencials com DHCP i DNS.
  • Realitzar configuracions avançades de seguretat en xarxes.

Exercici 1: Configuració Bàsica d'un Router

Descripció

En aquest exercici, configurarem un router bàsic per permetre la comunicació entre dues xarxes diferents.

Passos

  1. Connectar el maquinari:

    • Connectar el router a dues xarxes diferents utilitzant cables Ethernet.
    • Assegurar-se que les interfícies del router estiguin correctament connectades.

  2. Accedir al router:

    • Utilitzar una consola o una interfície web per accedir al router.
    • Iniciar sessió amb les credencials proporcionades.

  3. Configurar les interfícies:

    • Assignar adreces IP a les interfícies del router.
    Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit

  4. Guardar la configuració:

    • Assegurar-se que la configuració es guarda correctament.
    Router# copy running-config startup-config

Verificació

  • Verificar la connectivitat entre les dues xarxes utilitzant comandaments de ping. 
    Router# ping 192.168.2.1

Exercici 2: Configuració d'un Switch

Descripció

En aquest exercici, configurarem un switch per segmentar una xarxa en múltiples VLANs.

Passos

  1. Accedir al switch:

    • Utilitzar una consola o una interfície web per accedir al switch.
    • Iniciar sessió amb les credencials proporcionades.

  2. Crear VLANs:

    • Crear dues VLANs en el switch.
    Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Sales
Switch(config-vlan)# exit
Switch(config)# vlan 20
Switch(config-vlan)# name Engineering
Switch(config-vlan)# exit

  3. Assignar ports a les VLANs:

    • Assignar ports específics a cada VLAN.
    Switch(config)# interface range GigabitEthernet0/1 - 2
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10
Switch(config-if-range)# exit
Switch(config)# interface range GigabitEthernet0/3 - 4
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 20
Switch(config-if-range)# exit

  4. Guardar la configuració:

    • Assegurar-se que la configuració es guarda correctament.
    Switch# copy running-config startup-config

Verificació

  • Verificar la configuració de les VLANs. 
    Switch# show vlan brief

Exercici 3: Configuració de DHCP

Descripció

En aquest exercici, configurarem un servidor DHCP en un router per assignar adreces IP automàticament als dispositius en la xarxa.

Passos

  1. Accedir al router:

    • Utilitzar una consola o una interfície web per accedir al router.
    • Iniciar sessió amb les credencials proporcionades.

  2. Configurar el pool DHCP:

    • Configurar un pool d'adreces IP per a la xarxa.
    Router> enable
Router# configure terminal
Router(config)# ip dhcp pool LAN
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# exit

  3. Excloure adreces IP:

    • Excloure adreces IP específiques del pool DHCP.
    Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10

  4. Guardar la configuració:

    • Assegurar-se que la configuració es guarda correctament.
    Router# copy running-config startup-config

Verificació

  • Verificar que els dispositius en la xarxa reben adreces IP automàticament. 
    Router# show ip dhcp binding

Conclusió

Aquests exercicis proporcionen una base sòlida per a la configuració de xarxes. A través de la pràctica, els estudiants aprendran a configurar routers, switches i serveis de xarxa essencials com DHCP. La comprensió i l'habilitat per realitzar aquestes configuracions són fonamentals per a qualsevol professional de TI que treballi amb xarxes.
