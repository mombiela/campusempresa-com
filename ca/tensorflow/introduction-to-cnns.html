Què són les Xarxes Neuronals Convolucionals (CNNs)?

Les Xarxes Neuronals Convolucionals (CNNs) són una classe de xarxes neuronals profundes, més comunament aplicades a l'anàlisi d'imatges visuals. Estan dissenyades per aprendre automàticament i adaptativament jerarquies espacials de característiques a partir d'imatges d'entrada.

Conceptes Clau

  • Capa Convolucional: Aplica una operació de convolució a l'entrada, passant el resultat a la següent capa.
  • Capa de Pooling: Redueix les dimensions espacials (amplada i alçada) del volum d'entrada.
  • Capa Totalment Connectada: Les neurones en una capa totalment connectada tenen connexions amb totes les activacions de la capa anterior.
  • Funció d'Activació: Introdueix no linealitat a la xarxa, permetent-li aprendre patrons més complexos.

Per què Utilitzar CNNs?

Les CNNs són particularment efectives per a tasques de reconeixement i classificació d'imatges degut a la seva capacitat per capturar jerarquies espacials en les dades. Són àmpliament utilitzades en diverses aplicacions com:

  • Reconeixement d'imatges i vídeos
  • Sistemes de recomanació
  • Classificació d'imatges
  • Anàlisi d'imatges mèdiques

Blocs de Construcció de les CNNs

Capa Convolucional

La capa convolucional és el bloc de construcció principal d'una CNN. Consisteix en un conjunt de filtres (o kernels) aprenables que s'apliquen a la imatge d'entrada.

import tensorflow as tf
from tensorflow.keras.layers import Conv2D

# Exemple d'una capa convolucional
conv_layer = Conv2D(filters=32, kernel_size=(3, 3), activation='relu', input_shape=(64, 64, 3))

Capa de Pooling

Les capes de pooling s'utilitzen per reduir les dimensions espacials de l'entrada, cosa que ajuda a reduir el cost computacional i controlar el sobreajustament.

from tensorflow.keras.layers import MaxPooling2D

# Exemple d'una capa de max pooling
pooling_layer = MaxPooling2D(pool_size=(2, 2))

Capa Totalment Connectada

Les capes totalment connectades s'utilitzen al final de la xarxa per fer prediccions. Són similars a les capes d'una xarxa neuronal tradicional.

from tensorflow.keras.layers import Dense, Flatten

# Exemple d'una capa totalment connectada
flatten_layer = Flatten()
dense_layer = Dense(units=128, activation='relu')

Funcions d'Activació

Les funcions d'activació introdueixen no linealitat a la xarxa, permetent-li aprendre patrons més complexos.

from tensorflow.keras.layers import Activation

# Exemple d'una funció d'activació
activation_layer = Activation('relu')

Exemple: Construir una CNN Simple amb TensorFlow

Construïm una CNN simple per a la classificació d'imatges utilitzant TensorFlow.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

# Inicialitzar el model
model = Sequential()

# Afegir una capa convolucional
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(64, 64, 3)))

# Afegir una capa de pooling
model.add(MaxPooling2D(pool_size=(2, 2)))

# Afegir una altra capa convolucional
model.add(Conv2D(64, (3, 3), activation='relu'))

# Afegir una altra capa de pooling
model.add(MaxPooling2D(pool_size=(2, 2)))

# Aplanar la sortida
model.add(Flatten())

# Afegir una capa totalment connectada
model.add(Dense(128, activation='relu'))

# Afegir la capa de sortida
model.add(Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

Entrenament de la CNN

Per entrenar la CNN, necessites un conjunt de dades. TensorFlow proporciona diversos conjunts de dades que pots utilitzar per practicar. Aquí, utilitzarem el conjunt de dades CIFAR-10.

from tensorflow.keras.datasets import cifar10
from tensorflow.keras.utils import to_categorical

# Carregar el conjunt de dades
(x_train, y_train), (x_test, y_test) = cifar10.load_data()

# Normalitzar les dades d'entrada
x_train = x_train.astype('float32') / 255.0
x_test = x_test.astype('float32') / 255.0

# Codificar les etiquetes en one-hot
y_train = to_categorical(y_train, 10)
y_test = to_categorical(y_test, 10)

# Entrenar el model
model.fit(x_train, y_train, epochs=10, batch_size=32, validation_data=(x_test, y_test))

Conclusió

En aquesta secció, hem introduït les Xarxes Neuronals Convolucionals (CNNs) i hem discutit els seus components clau, incloent capes convolucionals, capes de pooling, capes totalment connectades i funcions d'activació. També hem proporcionat un exemple simple de construcció i entrenament d'una CNN utilitzant TensorFlow. Entendre aquests conceptes bàsics t'ajudarà a construir models més complexos i abordar diverses tasques relacionades amb imatges de manera efectiva.
