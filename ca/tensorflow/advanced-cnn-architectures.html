Introducció

En aquesta secció, aprofundirem en les arquitectures avançades de Xarxes Neuronals Convolucionals (CNN) utilitzant TensorFlow. Aquestes arquitectures han estat dissenyades per abordar tasques complexes de reconeixement d'imatges i han establert referències en diverses competicions. Entendre aquestes arquitectures t'ajudarà a construir models més eficients i precisos.

Conceptes clau

  • Xarxes Residuals (ResNet)
  • Xarxes Inception
  • MobileNets
  • EfficientNet

Xarxes Residuals (ResNet)

ResNet introdueix el concepte d'aprenentatge residual per abordar el problema del gradient que es desvaneix en xarxes profundes.

Característiques clau

  • Connexions de salt: Permet que la xarxa aprengui funcions d'identitat, facilitant l'entrenament de xarxes molt profundes.
  • Blocs residuals: Consisteix en unes quantes capes convolucionals amb una connexió de salt que passa per alt una o més capes.

Exemple de codi

import tensorflow as tf
from tensorflow.keras.layers import Input, Conv2D, BatchNormalization, Activation, Add
from tensorflow.keras.models import Model

def residual_block(x, filters, kernel_size=3, stride=1):
    shortcut = x
    x = Conv2D(filters, kernel_size, strides=stride, padding='same')(x)
    x = BatchNormalization()(x)
    x = Activation('relu')(x)
    x = Conv2D(filters, kernel_size, strides=1, padding='same')(x)
    x = BatchNormalization()(x)
    x = Add()([x, shortcut])
    x = Activation('relu')(x)
    return x

input_tensor = Input(shape=(224, 224, 3))
x = Conv2D(64, 7, strides=2, padding='same')(input_tensor)
x = BatchNormalization()(x)
x = Activation('relu')(x)
x = residual_block(x, 64)
model = Model(inputs=input_tensor, outputs=x)
model.summary()

Explicació

  • Conv2D: Capa convolucional amb filtres i mida de nucli especificats.
  • BatchNormalization: Normalitza la sortida de la capa anterior.
  • Activation: Aplica la funció d'activació ReLU.
  • Add: Afegeix la connexió de salt a la sortida de les capes convolucionals.

Xarxes Inception

Les Xarxes Inception tenen com a objectiu millorar la utilització dels recursos de computació dins de la xarxa.

Característiques clau

  • Mòduls Inception: Combina múltiples operacions convolucionals amb diferents mides de nucli en paral·lel.
  • Reducció de dimensionalitat: Utilitza convolucions de 1x1 per reduir el nombre de paràmetres.

Exemple de codi

from tensorflow.keras.layers import concatenate

def inception_module(x, filters):
    branch1x1 = Conv2D(filters, 1, padding='same', activation='relu')(x)
    
    branch3x3 = Conv2D(filters, 1, padding='same', activation='relu')(x)
    branch3x3 = Conv2D(filters, 3, padding='same', activation='relu')(branch3x3)
    
    branch5x5 = Conv2D(filters, 1, padding='same', activation='relu')(x)
    branch5x5 = Conv2D(filters, 5, padding='same', activation='relu')(branch5x5)
    
    branch_pool = tf.keras.layers.MaxPooling2D(3, strides=1, padding='same')(x)
    branch_pool = Conv2D(filters, 1, padding='same', activation='relu')(branch_pool)
    
    x = concatenate([branch1x1, branch3x3, branch5x5, branch_pool], axis=-1)
    return x

input_tensor = Input(shape=(224, 224, 3))
x = inception_module(input_tensor, 32)
model = Model(inputs=input_tensor, outputs=x)
model.summary()

Explicació

  • concatenate: Fusiona les sortides de diferents branques.
  • Mòdul Inception: Combina múltiples capes convolucionals amb diferents mides de nucli i una capa de pooling.

MobileNets

MobileNets estan dissenyades per a aplicacions de visió mòbil i incrustada.

Característiques clau

  • Convolucions separables en profunditat: Redueix el nombre de paràmetres i el càlcul.
  • Multiplicador d'amplada: Controla l'amplada de la xarxa.
  • Multiplicador de resolució: Controla la resolució de la imatge d'entrada.

Exemple de codi

from tensorflow.keras.layers import DepthwiseConv2D

def depthwise_separable_conv_block(x, filters, stride=1):
    x = DepthwiseConv2D(kernel_size=3, strides=stride, padding='same')(x)
    x = BatchNormalization()(x)
    x = Activation('relu')(x)
    x = Conv2D(filters, kernel_size=1, strides=1, padding='same')(x)
    x = BatchNormalization()(x)
    x = Activation('relu')(x)
    return x

input_tensor = Input(shape=(224, 224, 3))
x = depthwise_separable_conv_block(input_tensor, 64)
model = Model(inputs=input_tensor, outputs=x)
model.summary()

Explicació

  • DepthwiseConv2D: Aplica un únic filtre convolucional per canal d'entrada.
  • Convolucions separables en profunditat: Combina convolucions en profunditat i punt a punt per reduir el càlcul.

EfficientNet

EfficientNet escala l'amplada, la profunditat i la resolució de la xarxa de manera equilibrada.

Característiques clau

  • Escalat compost: Equilibra la profunditat, l'amplada i la resolució de la xarxa.
  • EfficientNet-B0 a B7: Diferents versions amb complexitat i precisió creixents.

Exemple de codi

import tensorflow as tf
from tensorflow.keras.applications import EfficientNetB0

model = EfficientNetB0(weights='imagenet', include_top=False, input_shape=(224, 224, 3))
model.summary()

Explicació

  • EfficientNetB0: Model EfficientNet preentrenat.
  • include_top=False: Exclou la capa de classificació final.

Conclusió

En aquesta secció, hem explorat arquitectures avançades de CNN com ResNet, Xarxes Inception, MobileNets i EfficientNet. Cada arquitectura té les seves característiques úniques i avantatges, cosa que les fa adequades per a diferents tipus de tasques de reconeixement d'imatges. Entendre aquestes arquitectures et permetrà construir models més eficients i precisos utilitzant TensorFlow.
