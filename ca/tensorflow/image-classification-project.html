Introducció

La classificació d'imatges és una tasca fonamental en la visió per computador on l'objectiu és categoritzar imatges en classes predefinides. En aquest projecte, utilitzarem TensorFlow, una potent biblioteca de codi obert per a l'aprenentatge automàtic, per construir un model de classificació d'imatges. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en TensorFlow.

Prerequisits

Abans de començar aquest projecte, hauríeu de tenir:

  • Comprensió bàsica de la programació en Python.
  • Familiaritat amb els conceptes d'aprenentatge automàtic.
  • TensorFlow instal·lat a la vostra màquina.

Configuració de l'Entorn

Per començar, assegureu-vos que teniu TensorFlow instal·lat. Podeu instal·lar-lo utilitzant pip:

pip install tensorflow

Carregant i Preprocessant Dades

Utilitzarem el conjunt de dades CIFAR-10, que consta de 60.000 imatges en color de 32x32 en 10 classes.

Carregant el Conjunt de Dades

TensorFlow proporciona accés fàcil al conjunt de dades CIFAR-10:

import tensorflow as tf
from tensorflow.keras.datasets import cifar10

# Carregar el conjunt de dades
(x_train, y_train), (x_test, y_test) = cifar10.load_data()

# Normalitzar les imatges a un rang de 0 a 1
x_train, x_test = x_train / 255.0, x_test / 255.0

Exploració de Dades

És important entendre les dades amb les quals esteu treballant. Visualitzem algunes imatges:

import matplotlib.pyplot as plt

# Mostrar les primeres 25 imatges del conjunt d'entrenament
plt.figure(figsize=(10,10))
for i in range(25):
    plt.subplot(5,5,i+1)
    plt.xticks([])
    plt.yticks([])
    plt.grid(False)
    plt.imshow(x_train[i])
    plt.xlabel(y_train[i][0])
plt.show()

Construint el Model

Construirem una Xarxa Neuronal Convolucional (CNN) per a la classificació d'imatges.

Definint el Model

Aquí teniu un model CNN senzill:

from tensorflow.keras import layers, models

model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10))

Compilant el Model

A continuació, hem de compilar el model:

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

Entrenant el Model

Ara, entrenem el model:

history = model.fit(x_train, y_train, epochs=10, 
                    validation_data=(x_test, y_test))

Avaluant el Model

Després de l'entrenament, hem d'avaluar el rendiment del model en les dades de prova:

test_loss, test_acc = model.evaluate(x_test, y_test, verbose=2)
print(f'\nTest accuracy: {test_acc}')

Tècniques Avançades

Per aquells que busquen millorar el seu model, considereu les següents tècniques avançades:

Augmentació de Dades

L'augmentació de dades pot ajudar a millorar el rendiment del model augmentant artificialment la mida del conjunt d'entrenament:

from tensorflow.keras.preprocessing.image import ImageDataGenerator

datagen = ImageDataGenerator(
    rotation_range=20,
    width_shift_range=0.2,
    height_shift_range=0.2,
    horizontal_flip=True)

datagen.fit(x_train)

Aprenentatge per Transferència

Utilitzar un model preentrenat pot augmentar significativament el rendiment:

base_model = tf.keras.applications.VGG16(input_shape=(32, 32, 3),
                                         include_top=False,
                                         weights='imagenet')

base_model.trainable = False

model = models.Sequential([
    base_model,
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10)
])

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])
