Visió General

Les xarxes neuronals són un concepte fonamental en l'aprenentatge automàtic i la intel·ligència artificial. Aquesta secció us introduirà als conceptes bàsics de les xarxes neuronals, la seva estructura i com implementar-les utilitzant TensorFlow.

Què és una Xarxa Neuronal?

Una xarxa neuronal és una sèrie d'algoritmes que intenten reconèixer relacions subjacents en un conjunt de dades a través d'un procés que imita la manera com opera el cervell humà.

Conceptes Clau

  • Neurona: La unitat bàsica d'una xarxa neuronal, similar a una neurona biològica.
  • Capa: Una col·lecció de neurones. Les xarxes neuronals estan compostes per múltiples capes.
  • Pesos i Biasos: Paràmetres que s'ajusten durant l'entrenament per minimitzar l'error.
  • Funció d'Activació: Una funció aplicada a la sortida d'una neurona per introduir no linealitat.

Estructura d'una Xarxa Neuronal

Una xarxa neuronal típica consta de tres tipus de capes:

  1. Capa d'Entrada: La capa que rep les dades d'entrada.
  2. Capes Ocultes: Capes entre les capes d'entrada i sortida on es realitzen els càlculs.
  3. Capa de Sortida: La capa que produeix la sortida final.

Exemple d'Estructura

| Tipus de Capa | Descripció | |---------------|-----------------------------------| | Capa d'Entrada| Rep les dades d'entrada | | Capa Oculta | Realitza càlculs i extracció de característiques | | Capa de Sortida| Produeix la predicció final |

Implementació d'una Xarxa Neuronal en TensorFlow

Vegem els passos per crear una xarxa neuronal simple utilitzant TensorFlow.

Pas 1: Importar Llibreries

Primer, necessitem importar TensorFlow i altres llibreries necessàries.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

Pas 2: Definir el Model

Utilitzarem l'API Sequential per definir el nostre model.

model = Sequential([
    Dense(32, activation='relu', input_shape=(784,)),  # Capa d'entrada amb 784 neurones
    Dense(64, activation='relu'),                      # Capa oculta amb 64 neurones
    Dense(10, activation='softmax')                    # Capa de sortida amb 10 neurones (per classificació)
])

Pas 3: Compilar el Model

A continuació, compilem el model especificant l'optimitzador, la funció de pèrdua i les mètriques.

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

Pas 4: Entrenar el Model

Entrenem el model utilitzant el mètode fit.

# Suposant que X_train i y_train són les vostres dades d'entrenament i etiquetes
model.fit(X_train, y_train, epochs=10, batch_size=32)

Pas 5: Avaluar el Model

Finalment, avaluem el rendiment del model en dades de prova.

# Suposant que X_test i y_test són les vostres dades de prova i etiquetes
test_loss, test_acc = model.evaluate(X_test, y_test)
print(f'Precisió en la prova: {test_acc}')

Conclusió

En aquesta secció, hem introduït els conceptes bàsics de les xarxes neuronals i hem demostrat com implementar una xarxa neuronal simple utilitzant TensorFlow. Entendre aquests fonaments és crucial a mesura que avanceu cap a temes més avançats en xarxes neuronals i aprenentatge profund.
