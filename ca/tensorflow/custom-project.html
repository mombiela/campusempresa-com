Introducció

En aquesta secció, et guiarem a través de la creació d'un projecte personalitzat utilitzant TensorFlow. Aquest projecte consolidarà la teva comprensió dels conceptes de TensorFlow i et donarà experiència pràctica en la construcció i desplegament d'un model d'aprenentatge automàtic.

Visió General del Projecte

  • Objectiu: Construir una xarxa neuronal per classificar imatges del conjunt de dades CIFAR-10.
  • Conjunt de dades: CIFAR-10, que consisteix en 60.000 imatges en color de 32x32 en 10 classes.
  • Eines: TensorFlow, Keras i altres biblioteques essencials.

Guia Pas a Pas

Pas 1: Configuració de l'Entorn

Abans de començar, assegura't de tenir TensorFlow i altres biblioteques necessàries instal·lades.

pip install tensorflow numpy matplotlib

Pas 2: Importació de Biblioteques

Comença importat les biblioteques requerides.

import tensorflow as tf
from tensorflow.keras import datasets, layers, models
import matplotlib.pyplot as plt

Pas 3: Carregar i Preprocessar les Dades

Carrega el conjunt de dades CIFAR-10 i preprocessa'l.

# Carregar conjunt de dades
(train_images, train_labels), (test_images, test_labels) = datasets.cifar10.load_data()

# Normalitzar els valors dels píxels per estar entre 0 i 1
train_images, test_images = train_images / 255.0, test_images / 255.0

Pas 4: Visualitzar les Dades

Visualitza algunes imatges del conjunt de dades per entendre amb què estàs treballant.

class_names = ['airplane', 'automobile', 'bird', 'cat', 'deer', 'dog', 'frog', 'horse', 'ship', 'truck']

plt.figure(figsize=(10,10))
for i in range(25):
    plt.subplot(5,5,i+1)
    plt.xticks([])
    plt.yticks([])
    plt.grid(False)
    plt.imshow(train_images[i])
    plt.xlabel(class_names[train_labels[i][0]])
plt.show()

Pas 5: Construir el Model

Crea un model de Xarxa Neuronal Convolucional (CNN).

model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10))

Pas 6: Compilar el Model

Compila el model amb la funció de pèrdua, optimitzador i mètriques adequades.

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

Pas 7: Entrenar el Model

Entrena el model utilitzant les dades d'entrenament.

history = model.fit(train_images, train_labels, epochs=10, 
                    validation_data=(test_images, test_labels))

Pas 8: Avaluar el Model

Avalua el rendiment del model en el conjunt de dades de prova.

test_loss, test_acc = model.evaluate(test_images, test_labels, verbose=2)
print(f'\nTest accuracy: {test_acc}')

Pas 9: Fer Prediccions

Utilitza el model entrenat per fer prediccions en noves dades.

probability_model = tf.keras.Sequential([model, 
                                         tf.keras.layers.Softmax()])

predictions = probability_model.predict(test_images)
print(predictions[0])
print(f'Predicted label: {class_names[np.argmax(predictions[0])]}')

Pas 10: Visualitzar les Prediccions

Visualitza les prediccions per entendre el rendiment del model.

def plot_image(i, predictions_array, true_label, img):
    true_label, img = true_label[i], img[i]
    plt.grid(False)
    plt.xticks([])
    plt.yticks([])
    
    plt.imshow(img, cmap=plt.cm.binary)
    
    predicted_label = np.argmax(predictions_array)
    if predicted_label == true_label:
        color = 'blue'
    else:
        color = 'red'
    
    plt.xlabel(f"{class_names[predicted_label]} {100*np.max(predictions_array):2.0f}% ({class_names[true_label]})", color=color)

def plot_value_array(i, predictions_array, true_label):
    true_label = true_label[i]
    plt.grid(False)
    plt.xticks(range(10))
    plt.yticks([])
    thisplot = plt.bar(range(10), predictions_array, color="#777777")
    plt.ylim([0, 1])
    predicted_label = np.argmax(predictions_array)
    
    thisplot[predicted_label].set_color('red')
    thisplot[true_label].set_color('blue')

i = 0
plt.figure(figsize=(6,3))
plt.subplot(1,2,1)
plot_image(i, predictions[i], test_labels, test_images)
plt.subplot(1,2,2)
plot_value_array(i, predictions[i], test_labels)
plt.show()

Conclusió

En aquest projecte personalitzat, has après a:

  • Configurar un entorn TensorFlow.
  • Carregar i preprocessar el conjunt de dades CIFAR-10.
  • Construir, compilar i entrenar un model CNN.
  • Avaluar el rendiment del model.
  • Fer i visualitzar prediccions.

En completar aquest projecte, has adquirit experiència pràctica en l'ús de TensorFlow per a tasques de classificació d'imatges. Continua experimentant amb diferents arquitectures i hiperparàmetres per millorar encara més el rendiment del teu model.
