Introducció

En aquesta secció, explorarem els conceptes fonamentals de les funcions de pèrdua i els optimitzadors en TensorFlow. Entendre aquests conceptes és crucial per entrenar models d'aprenentatge automàtic de manera efectiva. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Funcions de pèrdua

Què és una funció de pèrdua?

  • Una funció de pèrdua mesura com de bé les prediccions d'un model coincideixen amb les dades reals.
  • Quantifica la diferència entre la sortida predita i la sortida real.
  • L'objectiu d'entrenar un model és minimitzar aquesta pèrdua.

Funcions de pèrdua comunes

  1. Error quadràtic mitjà (MSE)

    • Utilitzat per a tasques de regressió.
    • Mesura la diferència quadràtica mitjana entre els valors predits i els valors reals.
    import tensorflow as tf

y_true = [1.0, 2.0, 3.0]
y_pred = [1.5, 2.5, 3.5]
mse = tf.keras.losses.MeanSquaredError()
loss = mse(y_true, y_pred)
print('Error quadràtic mitjà:', loss.numpy())

  2. Entropia creuada binària

    • Utilitzat per a tasques de classificació binària.
    • Mesura la diferència entre dues distribucions de probabilitat.
    y_true = [0, 1, 0, 1]
y_pred = [0.1, 0.9, 0.2, 0.8]
bce = tf.keras.losses.BinaryCrossentropy()
loss = bce(y_true, y_pred)
print('Entropia creuada binària:', loss.numpy())

  3. Entropia creuada categòrica

    • Utilitzat per a tasques de classificació multiclasse.
    • Mesura la diferència entre la distribució de les etiquetes reals i la distribució de les etiquetes predites.
    y_true = [[0, 1, 0], [0, 0, 1]]
y_pred = [[0.05, 0.95, 0], [0.1, 0.8, 0.1]]
cce = tf.keras.losses.CategoricalCrossentropy()
loss = cce(y_true, y_pred)
print('Entropia creuada categòrica:', loss.numpy())

Funcions de pèrdua personalitzades

  • Pots crear funcions de pèrdua personalitzades en TensorFlow.
  • Les funcions de pèrdua personalitzades poden ser útils per a tasques específiques o quan les funcions de pèrdua predefinides no compleixen les teves necessitats.
def custom_loss(y_true, y_pred):
    return tf.reduce_mean(tf.square(y_true - y_pred))

y_true = [1.0, 2.0, 3.0]
y_pred = [1.5, 2.5, 3.5]
loss = custom_loss(y_true, y_pred)
print('Pèrdua personalitzada:', loss.numpy())

Optimitzadors

Què és un optimitzador?

  • Un optimitzador ajusta els paràmetres del model per minimitzar la funció de pèrdua.
  • Actualitza els pesos i els biaixos del model basant-se en els gradients calculats.

Optimitzadors comuns

  1. Descens de gradient estocàstic (SGD)

    • Actualitza els paràmetres utilitzant el gradient de la funció de pèrdua.
    model = tf.keras.Sequential([tf.keras.layers.Dense(1)])
model.compile(optimizer='sgd', loss='mean_squared_error')

  2. Adam

    • Combina els avantatges de dues altres extensions del descens de gradient estocàstic.
    • Taxa d'aprenentatge adaptativa i moment.
    model = tf.keras.Sequential([tf.keras.layers.Dense(1)])
model.compile(optimizer='adam', loss='mean_squared_error')

  3. RMSprop

    • Optimitzador que divideix la taxa d'aprenentatge per una mitjana exponencialment decreixent dels gradients quadrats.
    model = tf.keras.Sequential([tf.keras.layers.Dense(1)])
model.compile(optimizer='rmsprop', loss='mean_squared_error')

Optimitzadors personalitzats

  • TensorFlow permet la creació d'optimitzadors personalitzats.
  • Els optimitzadors personalitzats poden ser útils per a tasques específiques o quan els optimitzadors predefinits no compleixen les teves necessitats.
class CustomOptimizer(tf.keras.optimizers.Optimizer):
    def __init__(self, learning_rate=0.01, name="CustomOptimizer", **kwargs):
        super().__init__(name, **kwargs)
        self.learning_rate = learning_rate

    def _resource_apply_dense(self, grad, var, apply_state=None):
        var.assign_sub(self.learning_rate * grad)

model = tf.keras.Sequential([tf.keras.layers.Dense(1)])
model.compile(optimizer=CustomOptimizer(), loss='mean_squared_error')

Resum

  • Les funcions de pèrdua i els optimitzadors són components crítics de l'entrenament de models d'aprenentatge automàtic.
  • Les funcions de pèrdua mesuren la diferència entre els valors predits i els valors reals, guiant el procés d'optimització.
  • Els optimitzadors ajusten els paràmetres del model per minimitzar la funció de pèrdua.
  • TensorFlow proporciona una varietat de funcions de pèrdua i optimitzadors integrats, i també permet implementacions personalitzades.

Entenent i utilitzant efectivament les funcions de pèrdua i els optimitzadors, pots millorar significativament el rendiment dels teus models d'aprenentatge automàtic.
