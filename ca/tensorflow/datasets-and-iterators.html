Introducció

En aquesta secció, explorarem com gestionar dades de manera eficient utilitzant Datasets i Iteradors en TensorFlow. Això és crucial per construir models d'aprenentatge automàtic escalables i eficients. Cobrirem els conceptes bàsics de la creació de datasets, la seva transformació i la seva iteració.

Creació de Datasets

TensorFlow proporciona l'API tf.data per construir pipelines d'entrada complexes a partir de peces senzilles i reutilitzables.

Des de Tensores

Pots crear un dataset directament des de tensors utilitzant tf.data.Dataset.from_tensor_slices.

import tensorflow as tf

# Crear un dataset des de tensors
data = tf.data.Dataset.from_tensor_slices([1, 2, 3, 4, 5])
for element in data:
    print(element.numpy())

Des de Fitxers

També pots crear datasets des de fitxers, com ara fitxers CSV o fitxers TFRecord.

# Crear un dataset des d'un fitxer CSV
file_path = 'path/to/your/csvfile.csv'
dataset = tf.data.experimental.make_csv_dataset(
    file_path,
    batch_size=5,
    label_name='label_column_name',
    na_value="?",
    num_epochs=1,
    ignore_errors=True
)

Transformació de Datasets

Les transformacions et permeten mapar, agrupar en lots, barrejar i prefetch dades per optimitzar el pipeline d'entrada.

Mapeig

Utilitza el mètode map per aplicar una funció a cada element del dataset.

def add_one(x):
    return x + 1

mapped_data = data.map(add_one)
for element in mapped_data:
    print(element.numpy())

Agrupació en Lots

L'agrupació en lots combina elements consecutius del dataset en un sol element.

batched_data = data.batch(2)
for batch in batched_data:
    print(batch.numpy())

Barreja

La barreja randomitza l'ordre dels elements en el dataset.

shuffled_data = data.shuffle(buffer_size=5)
for element in shuffled_data:
    print(element.numpy())

Prefetch

El prefetch solapa el preprocessament i l'execució del model de dades.

prefetched_data = data.prefetch(buffer_size=tf.data.experimental.AUTOTUNE)

Iteració a Través de Datasets

Els iteradors proporcionen una manera d'accedir als elements del dataset.

Iterador One-shot

Un iterador one-shot és la forma més senzilla d'iterador.

iterator = iter(data)
print(next(iterator).numpy())

Iterador Inicialitzable

Un iterador inicialitzable requereix una inicialització explícita.

iterator = iter(data)
print(next(iterator).numpy())

Tècniques Avançades

Per pipelines d'entrada més complexos, pots utilitzar característiques com tf.data.Dataset.zip, tf.data.Dataset.concatenate i tf.data.Dataset.interleave.

Zipping Datasets

Combina elements de dos datasets en un sol dataset.

dataset1 = tf.data.Dataset.from_tensor_slices([1, 2, 3])
dataset2 = tf.data.Dataset.from_tensor_slices(['a', 'b', 'c'])
zipped_data = tf.data.Dataset.zip((dataset1, dataset2))

for element in zipped_data:
    print(element)

Concatenació de Datasets

Concatena dos datasets.

dataset3 = tf.data.Dataset.from_tensor_slices([4, 5, 6])
concatenated_data = dataset1.concatenate(dataset3)

for element in concatenated_data:
    print(element.numpy())

Interleaving Datasets

Intercala elements de múltiples datasets.

interleaved_data = tf.data.Dataset.from_tensor_slices([1, 2, 3]).interleave(
    lambda x: tf.data.Dataset.from_tensors(x).repeat(2),
    cycle_length=2
)

for element in interleaved_data:
    print(element.numpy())

Conclusió

En aquesta secció, hem cobert els conceptes bàsics i algunes tècniques avançades per gestionar datasets i iteradors en TensorFlow. La gestió eficient de dades és crucial per construir models d'aprenentatge automàtic escalables. En dominar aquestes tècniques, pots assegurar-te que els teus models siguin tant eficients com efectius.
