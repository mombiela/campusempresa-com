Introducció

Benvingut al tema "Hola TensorFlow"! En aquesta secció, et presentarem TensorFlow, una potent biblioteca de codi obert per a l'aprenentatge automàtic desenvolupada per Google. Aquest tema està dissenyat per a principiants i et guiarà a través dels conceptes bàsics de TensorFlow, incloent la instal·lació, operacions bàsiques i la creació del teu primer programa amb TensorFlow.

Què és TensorFlow?

TensorFlow és una plataforma de codi obert de punta a punta per a l'aprenentatge automàtic. Té un ecosistema complet i flexible d'eines, biblioteques i recursos comunitaris que permeten als investigadors avançar en l'estat de l'art en ML, i als desenvolupadors construir i desplegar fàcilment aplicacions impulsades per ML.

Conceptes Clau

  • Tensors: La unitat central de dades en TensorFlow, similar als arrays en NumPy.
  • Gràfics: TensorFlow utilitza gràfics computacionals per representar les teves computacions.
  • Sessions: Una sessió és un entorn per executar un gràfic.

Instal·lant TensorFlow

Abans de començar a utilitzar TensorFlow, necessitem instal·lar-lo. Pots instal·lar TensorFlow utilitzant pip, l'instal·lador de paquets de Python.

Passos d'Instal·lació

  1. Crear un Entorn Virtual (Opcional però Recomanat)

    python -m venv tensorflow_env
source tensorflow_env/bin/activate  # A Windows utilitza `tensorflow_env\Scripts\activate`

  2. Instal·lar TensorFlow

    pip install tensorflow

  3. Verificar la Instal·lació

    import tensorflow as tf
print(tf.__version__)

El Teu Primer Programa amb TensorFlow

Escrivim un programa senzill amb TensorFlow per familiaritzar-nos amb les operacions bàsiques.

Exemple: Hola TensorFlow

import tensorflow as tf

# Crear un tensor constant
hello = tf.constant('Hola, TensorFlow!')

# Iniciar una sessió de TensorFlow
with tf.Session() as sess:
    # Executar la sessió per obtenir la sortida del tensor
    output = sess.run(hello)
    print(output)

Explicació

  • Importar TensorFlow: Comencem importat la biblioteca de TensorFlow.
  • Crear un Tensor Constant: Creem un tensor constant amb el valor 'Hola, TensorFlow!'.
  • Iniciar una Sessió: Creem una sessió per executar el gràfic.
  • Executar la Sessió: Executem la sessió per avaluar el tensor i imprimir la sortida.

Operacions Bàsiques amb TensorFlow

TensorFlow suporta una varietat d'operacions sobre tensors. Vegem algunes operacions bàsiques.

Exemple: Operacions Bàsiques

import tensorflow as tf

# Crear tensors constants
a = tf.constant(2)
b = tf.constant(3)

# Realitzar operacions bàsiques
add = tf.add(a, b)
subtract = tf.subtract(a, b)
multiply = tf.multiply(a, b)
divide = tf.divide(a, b)

# Iniciar una sessió per executar les operacions
with tf.Session() as sess:
    print("Suma: ", sess.run(add))
    print("Resta: ", sess.run(subtract))
    print("Multiplicació: ", sess.run(multiply))
    print("Divisió: ", sess.run(divide))

Explicació

  • Crear Tensors Constants: Creem dos tensors constants a i b.
  • Realitzar Operacions: Realitzem sumes, restes, multiplicacions i divisions sobre els tensors.
  • Executar la Sessió: Executem la sessió per avaluar les operacions i imprimir els resultats.
