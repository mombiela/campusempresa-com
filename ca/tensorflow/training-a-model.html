Introducció

Entrenar un model és un pas fonamental en l'aprenentatge automàtic on el model aprèn a partir de dades. En TensorFlow, això implica definir un model, especificar una funció de pèrdua i utilitzar un optimitzador per ajustar els paràmetres del model per minimitzar la pèrdua. Aquesta secció et guiarà a través del procés d'entrenament d'un model en TensorFlow, des de conceptes bàsics fins a tècniques avançades.

Conceptes Clau

  • Model: Una representació matemàtica d'un procés del món real. En TensorFlow, els models es construeixen típicament utilitzant l'API tf.keras.
  • Funció de Pèrdua: Una funció que mesura com de bé les prediccions del model coincideixen amb les dades reals. Les funcions de pèrdua comunes inclouen l'Error Quadràtic Mitjà (MSE) per a la regressió i l'Entropia Creuada Categòrica per a la classificació.
  • Optimitzador: Un algoritme que ajusta els paràmetres del model per minimitzar la funció de pèrdua. Els optimitzadors populars inclouen el Descens de Gradient Estocàstic (SGD) i Adam.
  • Èpoques: El nombre de vegades que tot el conjunt de dades es passa pel model durant l'entrenament.
  • Mida del Lot: El nombre de mostres processades abans que es facin actualitzacions als paràmetres del model.

Construcció d'un Model Simple

Pas 1: Importar Biblioteques

Primer, importa les biblioteques necessàries.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

Pas 2: Definir el Model

Crea una xarxa neuronal feedforward simple utilitzant l'API Sequential.

model = Sequential([
    Dense(64, activation='relu', input_shape=(784,)),
    Dense(64, activation='relu'),
    Dense(10, activation='softmax')
])
  • Capa Densa: Una capa totalment connectada on cada neurona està connectada a cada neurona de la capa anterior.
  • Funció d'Activació: Una funció aplicada a la sortida de cada neurona. Les opcions comunes inclouen relu (Unitat Lineal Rectificada) i softmax.

Pas 3: Compilar el Model

Especifica la funció de pèrdua, l'optimitzador i les mètriques.

model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)
  • Optimitzador: Adam és una elecció popular per la seva eficiència i rendiment.
  • Funció de Pèrdua: sparse_categorical_crossentropy s'utilitza per a problemes de classificació multiclasse.
  • Mètriques: L'exactitud és una mètrica comuna per a tasques de classificació.

Pas 4: Entrenar el Model

Utilitza el mètode fit per entrenar el model.

history = model.fit(
    x_train, y_train,
    epochs=10,
    batch_size=32,
    validation_split=0.2
)
  • x_train i y_train: Dades d'entrenament i etiquetes.
  • èpoques: Nombre de vegades que el model iterarà sobre tot el conjunt de dades.
  • mida del lot: Nombre de mostres per actualització de gradient.
  • validation_split: Fracció de les dades d'entrenament que s'utilitzarà com a dades de validació.

Tècniques Avançades

Aturada Primerenca

Evita el sobreajustament aturant l'entrenament quan la pèrdua de validació deixi de millorar.

from tensorflow.keras.callbacks import EarlyStopping

early_stopping = EarlyStopping(
    monitor='val_loss',
    patience=3,
    restore_best_weights=True
)

history = model.fit(
    x_train, y_train,
    epochs=50,
    batch_size=32,
    validation_split=0.2,
    callbacks=[early_stopping]
)

Programes de Taxa d'Aprenentatge

Ajusta la taxa d'aprenentatge durant l'entrenament per millorar la convergència.

from tensorflow.keras.callbacks import LearningRateScheduler

def scheduler(epoch, lr):
    if epoch < 10:
        return lr
    else:
        return lr * tf.math.exp(-0.1)

lr_scheduler = LearningRateScheduler(scheduler)

history = model.fit(
    x_train, y_train,
    epochs=50,
    batch_size=32,
    validation_split=0.2,
    callbacks=[lr_scheduler]
)

Comprovació de Model

Guarda el model en diverses etapes durant l'entrenament.

from tensorflow.keras.callbacks import ModelCheckpoint

checkpoint = ModelCheckpoint(
    'best_model.h5',
    monitor='val_loss',
    save_best_only=True
)

history = model.fit(
    x_train, y_train,
    epochs=50,
    batch_size=32,
    validation_split=0.2,
    callbacks=[checkpoint]
)

Resum

Entrenar un model en TensorFlow implica definir l'arquitectura del model, compilar-lo amb una funció de pèrdua i un optimitzador, i després ajustar-lo a les dades. Les tècniques avançades com l'aturada primerenca, els programes de taxa d'aprenentatge i la comprovació de models poden ajudar a millorar el rendiment i evitar el sobreajustament. Dominant aquests conceptes, pots construir i entrenar models d'aprenentatge automàtic robustos utilitzant TensorFlow.
