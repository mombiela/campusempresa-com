Introducció a TensorFlow

TensorFlow és un marc de treball de codi obert per a l'aprenentatge automàtic desenvolupat per Google. S'utilitza àmpliament per construir i desplegar models d'aprenentatge automàtic, especialment models d'aprenentatge profund. TensorFlow proporciona un ecosistema complet d'eines, biblioteques i recursos comunitaris que permeten als desenvolupadors crear i desplegar aplicacions d'aprenentatge automàtic de manera eficient.

Conceptes clau

Tensor

  • Definició: Els tensors són les estructures de dades fonamentals a TensorFlow. Són arrays multidimensionals que poden contenir diversos tipus de dades.
  • Tipus de tensors:
    • Escalars (tensors 0-D)
    • Vectors (tensors 1-D)
    • Matrius (tensors 2-D)
    • Tensors de dimensions superiors (3-D, 4-D, etc.)
import tensorflow as tf

# Escalar
scalar = tf.constant(3)
print(scalar)

# Vector
vector = tf.constant([1, 2, 3])
print(vector)

# Matriu
matrix = tf.constant([[1, 2], [3, 4]])
print(matrix)

# Tensor 3-D
tensor_3d = tf.constant([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print(tensor_3d)

Grafs computacionals

  • Definició: Un graf computacional és una sèrie d'operacions de TensorFlow disposades en un graf de nodes. Cada node representa una operació, i les vores representen les dades (tensors) que flueixen entre aquestes operacions.
  • Execució immediata: TensorFlow 2.x utilitza l'execució immediata per defecte, la qual cosa permet que les operacions s'executin immediatament quan es criden dins de Python.
# Exemple d'execució immediata
a = tf.constant(2)
b = tf.constant(3)
c = a + b
print(c)  # Sortida: 5

Variables

  • Definició: Les variables són tensors especials que es poden actualitzar durant l'execució d'un programa. S'utilitzen per emmagatzemar i actualitzar paràmetres en un model d'aprenentatge automàtic.
  • Ús: Les variables es creen utilitzant tf.Variable.
# Creant una variable
var = tf.Variable(5)
print(var)

# Actualitzant una variable
var.assign(10)
print(var)

Operacions

  • Definició: Les operacions (o ops) són funcions que prenen un o més tensors com a entrada i produeixen un o més tensors com a sortida. Les operacions comunes inclouen l'addició, la multiplicació i la multiplicació de matrius.
# Operacions bàsiques
a = tf.constant(2)
b = tf.constant(3)

add = tf.add(a, b)
mul = tf.multiply(a, b)
matmul = tf.matmul([[1, 2]], [[3], [4]])

print("Addició:", add)
print("Multiplicació:", mul)
print("Multiplicació de matrius:", matmul)

Sessions (TensorFlow 1.x)

  • Definició: A TensorFlow 1.x, les sessions s'utilitzen per executar operacions en el graf computacional. No obstant això, a TensorFlow 2.x, l'execució immediata elimina la necessitat de sessions.
# Exemple de TensorFlow 1.x (per referència)
import tensorflow.compat.v1 as tf
tf.disable_v2_behavior()

a = tf.constant(2)
b = tf.constant(3)
c = a + b

with tf.Session() as sess:
    result = sess.run(c)
    print(result)

Resum

En aquesta secció, hem cobert els conceptes bàsics de TensorFlow, incloent tensors, grafs computacionals, variables i operacions. També hem esmentat les diferències entre TensorFlow 1.x i 2.x, especialment l'ús de l'execució immediata a TensorFlow 2.x. Entendre aquests conceptes fonamentals és crucial per construir i desplegar models d'aprenentatge automàtic utilitzant TensorFlow. En les properes seccions, aprofundirem en temes més avançats i aplicacions pràctiques de TensorFlow.
