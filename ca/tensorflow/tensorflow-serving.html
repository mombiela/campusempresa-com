Introducció a TensorFlow Serving

TensorFlow Serving és un sistema flexible i d'alt rendiment per servir models d'aprenentatge automàtic, dissenyat per a entorns de producció. Permet desplegar nous algorismes i experiments mantenint la mateixa arquitectura de servidor i APIs.

Conceptes Clau

  • Model Server: Un servidor dissenyat per servir models d'aprenentatge automàtic.
  • Model Versioning: Suporta múltiples versions de models, permetent actualitzacions sense interrupcions.
  • Batching: Combina múltiples sol·licituds per millorar el rendiment.
  • Monitoring: Proporciona eines per monitoritzar la salut i el rendiment dels models.

Configuració de TensorFlow Serving

Per començar amb TensorFlow Serving, cal instal·lar-lo i configurar el teu entorn.

Instal·lació

Pots instal·lar TensorFlow Serving utilitzant Docker o directament al teu sistema.

Utilitzant Docker

docker pull tensorflow/serving

Instal·lació Directa

Per a Ubuntu:

echo "deb [arch=amd64] http://storage.googleapis.com/tensorflow-serving-apt stable tensorflow-model-server tensorflow-model-server-universal" | sudo tee /etc/apt/sources.list.d/tensorflow-serving.list && \
curl https://storage.googleapis.com/tensorflow-serving-apt/tensorflow-serving.release.pub.gpg | sudo apt-key add -
sudo apt-get update && sudo apt-get install tensorflow-model-server

Servir un Model TensorFlow

Un cop TensorFlow Serving està instal·lat, pots servir un model TensorFlow.

Exportar un Model

Primer, cal exportar el teu model entrenat en un format que TensorFlow Serving pugui utilitzar.

import tensorflow as tf

# Defineix un model simple
model = tf.keras.Sequential([
    tf.keras.layers.Dense(10, activació='relu', input_shape=(784,)),
    tf.keras.layers.Dense(10, activació='softmax')
])

# Guarda el model
model.save('my_model/1')

Iniciar el Servidor de Models

Utilitza la següent comanda per iniciar el TensorFlow Model Server:

docker run -p 8501:8501 --name=tfserving_my_model --mount type=bind,source=$(pwd)/my_model,target=/models/my_model -e MODEL_NAME=my_model -t tensorflow/serving

Fer Prediccions

Pots fer prediccions enviant sol·licituds HTTP POST a l'API de TensorFlow Serving.

Exemple de Sol·licitud

import requests
import json

data = json.dumps({"signature_name": "serving_default", "instances": [[0.0]*784]})
headers = {"content-type": "application/json"}
json_response = requests.post('http://localhost:8501/v1/models/my_model:predict', data=data, headers=headers)
predictions = json.loads(json_response.text)['predictions']
print(predictions)

Funcionalitats Avançades

TensorFlow Serving ofereix diverses funcionalitats avançades per optimitzar i gestionar els teus models.

Versionat de Models

TensorFlow Serving suporta servir múltiples versions d'un model simultàniament. Això permet desplegar noves versions sense temps d'inactivitat.

Exemple d'Estructura de Directori

/models
    /my_model
        /1
        /2

Batching

Batching combina múltiples sol·licituds en un sol lot per millorar el rendiment.

Activar Batching

Crea un fitxer batching_parameters.txt:

max_batch_size { value: 32 }
batch_timeout_micros { value: 10000 }

Inicia el servidor amb batching activat:

docker run -p 8501:8501 --name=tfserving_my_model --mount type=bind,source=$(pwd)/my_model,target=/models/my_model -e MODEL_NAME=my_model -t tensorflow/serving --enable_batching=true --batching_parameters_file=/models/my_model/batching_parameters.txt

Monitorització

TensorFlow Serving proporciona capacitats de monitorització per seguir el rendiment i la salut dels teus models.

Integració amb Prometheus

TensorFlow Serving pot exportar mètriques en un format que Prometheus pot recollir.

Inicia el servidor amb monitorització activada:

docker run -p 8501:8501 -p 8502:8502 --name=tfserving_my_model --mount type=bind,source=$(pwd)/my_model,target=/models/my_model -e MODEL_NAME=my_model -t tensorflow/serving --monitoring_config_file=/models/my_model/monitoring_config.txt

Conclusió

TensorFlow Serving és una eina poderosa per desplegar models d'aprenentatge automàtic en producció. Suporta versionat de models, batching i monitorització, fent-lo adequat per a aplicacions d'aprenentatge automàtic d'alt rendiment i escalables. Seguint els passos descrits en aquest tema, pots configurar TensorFlow Serving, desplegar els teus models i aprofitar les seves funcionalitats avançades per optimitzar els teus fluxos de treball d'aprenentatge automàtic.
