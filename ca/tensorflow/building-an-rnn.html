Introducció a les Xarxes Neuronals Recurrents (RNNs)

Les Xarxes Neuronals Recurrents (RNNs) són una classe de xarxes neuronals que són particularment efectives per a dades seqüencials. A diferència de les xarxes neuronals feedforward tradicionals, les RNNs tenen connexions que formen cicles dirigits, permetent-los mantenir una 'memòria' de les entrades anteriors.

  • Dades Seqüencials: Dades on l'ordre dels punts de dades és important, com ara sèries temporals, text i veu.
  • Memòria: Les RNNs poden recordar entrades anteriors gràcies al seu estat intern, que s'actualitza a cada pas de temps.

Entendre les RNNs

Abans d'endinsar-nos en la construcció d'una RNN amb TensorFlow, és crucial entendre l'estructura bàsica i el funcionament de les RNNs.

  • Estat Ocult: L'estat ocult és un vector que captura informació dels passos de temps anteriors.
  • Pesos: Les RNNs utilitzen els mateixos pesos per a cada pas de temps, cosa que ajuda a capturar patrons al llarg del temps.
  • Retropropagació en el Temps (BPTT): Una variant de la retropropagació utilitzada per entrenar RNNs, que considera la naturalesa temporal de les dades.

Construir una RNN amb TensorFlow

En aquesta secció, construirem una RNN simple utilitzant TensorFlow. Utilitzarem l'API d'alt nivell de TensorFlow, Keras, per simplificar el procés.

Pas 1: Importar Llibreries

Primer, necessitem importar les llibreries necessàries.

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import SimpleRNN, Dense

Pas 2: Preparar les Dades

Per a aquest exemple, crearem algunes dades seqüencials sintètiques.

import numpy as np

# Generar dades seqüencials sintètiques
def generate_data(seq_length, num_samples):
    X = np.random.rand(num_samples, seq_length, 1)
    y = np.sum(X, axis=1)
    return X, y

seq_length = 10
num_samples = 1000
X, y = generate_data(seq_length, num_samples)

Pas 3: Construir el Model RNN

Ara construirem un model RNN simple utilitzant Keras.

model = Sequential()
model.add(SimpleRNN(50, activation='relu', input_shape=(seq_length, 1)))
model.add(Dense(1))

model.compile(optimizer='adam', loss='mse')
model.summary()

Pas 4: Entrenar el Model

A continuació, entrenem el model amb les nostres dades sintètiques.

model.fit(X, y, epochs=20, batch_size=32)

Pas 5: Fer Prediccions

Finalment, podem utilitzar el model entrenat per fer prediccions.

# Generar noves dades per a la predicció
X_new, y_true = generate_data(seq_length, 10)
y_pred = model.predict(X_new)

print("Valors reals:", y_true)
print("Valors predits:", y_pred)

Resum de Conceptes Clau

  • Dades Seqüencials: Dades on l'ordre és important.
  • Estat Ocult: Captura informació dels passos de temps anteriors.
  • Compartició de Pesos: Els mateixos pesos s'utilitzen per a cada pas de temps.
  • BPTT: Mètode d'entrenament per a RNNs que considera les dependències temporals.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les Xarxes Neuronals Recurrents i hem demostrat com construir una RNN simple utilitzant l'API Keras de TensorFlow. Hem recorregut els passos d'importar llibreries, preparar dades, construir el model, entrenar-lo i fer prediccions. Entendre aquests fonaments t'ajudarà a abordar problemes de dades seqüencials més complexos i a construir arquitectures RNN més avançades en el futur.
