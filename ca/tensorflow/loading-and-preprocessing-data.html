En aquesta secció, cobrirem com carregar i preprocessar dades utilitzant TensorFlow. Aquest és un pas crucial en qualsevol pipeline d'aprenentatge automàtic, ja que la qualitat i el format de les teves dades poden afectar significativament el rendiment dels teus models.

Conceptes Clau

  • Carregant Dades: Importar dades de diverses fonts com fitxers CSV, imatges o bases de dades.
  • Preprocessant Dades: Transformar dades en brut en un format adequat per entrenar models d'aprenentatge automàtic. Això inclou normalització, augmentació i divisió de dades en conjunts d'entrenament, validació i prova.

Carregant Dades

Carregant Dades des de Fitxers CSV

TensorFlow proporciona diverses utilitats per carregar dades des de fitxers CSV. L'API tf.data és particularment útil per a aquest propòsit.

import tensorflow as tf

# Defineix la ruta del fitxer
file_path = 'path/to/your/data.csv'

# Crea un dataset des del fitxer CSV
dataset = tf.data.experimental.make_csv_dataset(
    file_path,
    batch_size=32,  # Nombre de mostres per lot
    label_name='target_column',  # La columna a utilitzar com a etiquetes
    na_value="?",  # Valor a considerar com a faltant
    num_epochs=1,  # Nombre de vegades a iterar sobre el dataset
    ignore_errors=True  # Ignorar errors durant la càrrega
)

# Inspecciona el dataset
for features, label in dataset.take(1):
    print("Característiques: ", features)
    print("Etiqueta: ", label)

Carregant Dades d'Imatges

TensorFlow també proporciona utilitats per carregar i preprocessar dades d'imatges. La funció tf.keras.preprocessing.image_dataset_from_directory és molt útil per a això.

import tensorflow as tf

# Defineix la ruta del directori
directory_path = 'path/to/your/images'

# Crea un dataset des del directori
dataset = tf.keras.preprocessing.image_dataset_from_directory(
    directory_path,
    image_size=(256, 256),  # Redimensiona les imatges a 256x256
    batch_size=32,  # Nombre de mostres per lot
    label_mode='int'  # Les etiquetes es retornen com a enters
)

# Inspecciona el dataset
for images, labels in dataset.take(1):
    print("Imatges: ", images.shape)
    print("Etiquetes: ", labels)

Preprocessant Dades

Normalització

La normalització és un pas comú de preprocessament que escala les característiques a un rang estàndard, normalment [0, 1] o [-1, 1].

def normalize(image, label):
    image = tf.cast(image, tf.float32) / 255.0
    return image, label

# Aplica la normalització al dataset
normalized_dataset = dataset.map(normalize)

# Inspecciona el dataset normalitzat
for images, labels in normalized_dataset.take(1):
    print("Imatges Normalitzades: ", images[0])
    print("Etiquetes: ", labels)

Augmentació de Dades

L'augmentació de dades és una tècnica per augmentar artificialment la mida del conjunt d'entrenament creant versions modificades de les imatges en el dataset.

data_augmentation = tf.keras.Sequential([
    tf.keras.layers.RandomFlip("horizontal_and_vertical"),
    tf.keras.layers.RandomRotation(0.2),
])

# Aplica l'augmentació de dades al dataset
augmented_dataset = normalized_dataset.map(lambda x, y: (data_augmentation(x, training=True), y))

# Inspecciona el dataset augmentat
for images, labels in augmented_dataset.take(1):
    print("Imatges Augmentades: ", images[0])
    print("Etiquetes: ", labels)

Dividint Dades

Dividir el dataset en conjunts d'entrenament, validació i prova és essencial per avaluar el rendiment del teu model.

# Defineix les mides de les divisions
train_size = 0.7
val_size = 0.15
test_size = 0.15

# Calcula el nombre de mostres
total_samples = len(list(dataset))
train_samples = int(train_size * total_samples)
val_samples = int(val_size * total_samples)

# Divideix el dataset
train_dataset = dataset.take(train_samples)
val_dataset = dataset.skip(train_samples).take(val_samples)
test_dataset = dataset.skip(train_samples + val_samples)

# Inspecciona les divisions
print(f"Mostres d'entrenament: {len(list(train_dataset))}")
print(f"Mostres de validació: {len(list(val_dataset))}")
print(f"Mostres de prova: {len(list(test_dataset))}")

Conclusió

En aquesta secció, hem cobert els passos essencials per carregar i preprocessar dades en TensorFlow. Hem après a:

  • Carregar dades des de fitxers CSV i directoris d'imatges.
  • Normalitzar i augmentar dades.
  • Dividir dades en conjunts d'entrenament, validació i prova.

Aquests passos són fonamentals per preparar les teves dades per a tasques d'aprenentatge automàtic. Una correcta càrrega i preprocessament de dades pot millorar significativament el rendiment i la fiabilitat dels teus models.
