Les capes de pooling són un component crucial en les Xarxes Neuronals Convolucionals (CNNs). Ajuden a reduir les dimensions espacials de l'entrada, cosa que disminueix la càrrega computacional i ajuda a extreure característiques dominants que són invariants a petites translacions. En aquesta secció, explorarem el concepte de les capes de pooling, els seus tipus i com implementar-les utilitzant TensorFlow.

Conceptes Clau

  • Operació de Pooling: Una tècnica per reduir les dimensions espacials (amplada i alçada) del volum d'entrada.
  • Tipus de Pooling:
    • Max Pooling: Selecciona el valor màxim de cada fragment del mapa de característiques.
    • Average Pooling: Calcula el valor mitjà de cada fragment del mapa de característiques.
  • Stride: El nombre de píxels pels quals es desplaça la finestra de pooling a través de la matriu d'entrada.
  • Padding: Afegir píxels extra al voltant de la matriu d'entrada per controlar les dimensions espacials de la sortida.

Max Pooling

El max pooling és el tipus de pooling més comú. Funciona desplaçant una finestra (normalment de 2x2) sobre el mapa de característiques d'entrada i prenent el valor màxim en cada finestra.

Exemple

Considera una matriu d'entrada de 4x4:

Matriu d'Entrada:
[[1, 3, 2, 4],
 [5, 6, 1, 2],
 [7, 8, 3, 0],
 [2, 4, 6, 8]]

Aplicar un max pooling de 2x2 amb un stride de 2:

Matriu Max Pooling:
[[6, 4],
 [8, 8]]

Implementació amb TensorFlow

import tensorflow as tf

# Crear una matriu d'entrada de 4x4
input_matrix = tf.constant([[1, 3, 2, 4],
                            [5, 6, 1, 2],
                            [7, 8, 3, 0],
                            [2, 4, 6, 8]], dtype=tf.float32)

# Reestructurar a tensor 4D: [batch_size, height, width, channels]
input_tensor = tf.reshape(input_matrix, [1, 4, 4, 1])

# Aplicar max pooling
max_pooling = tf.nn.max_pool2d(input_tensor, ksize=2, strides=2, padding='VALID')

print("Sortida Max Pooling:\n", tf.squeeze(max_pooling).numpy())

Average Pooling

L'average pooling funciona de manera similar al max pooling però en lloc de prendre el valor màxim, calcula la mitjana de tots els valors en la finestra.

Exemple

Considera la mateixa matriu d'entrada de 4x4:

Matriu d'Entrada:
[[1, 3, 2, 4],
 [5, 6, 1, 2],
 [7, 8, 3, 0],
 [2, 4, 6, 8]]

Aplicar un average pooling de 2x2 amb un stride de 2:

Matriu Average Pooling:
[[3.75, 2.25],
 [5.25, 4.25]]

Implementació amb TensorFlow

import tensorflow as tf

# Crear una matriu d'entrada de 4x4
input_matrix = tf.constant([[1, 3, 2, 4],
                            [5, 6, 1, 2],
                            [7, 8, 3, 0],
                            [2, 4, 6, 8]], dtype=tf.float32)

# Reestructurar a tensor 4D: [batch_size, height, width, channels]
input_tensor = tf.reshape(input_matrix, [1, 4, 4, 1])

# Aplicar average pooling
avg_pooling = tf.nn.avg_pool2d(input_tensor, ksize=2, strides=2, padding='VALID')

print("Sortida Average Pooling:\n", tf.squeeze(avg_pooling).numpy())

Padding i Stride

  • Padding: Determina si l'operació de pooling ha de padding la matriu d'entrada per mantenir les dimensions espacials.
    • VALID: Sense padding. La mida de la sortida es redueix.
    • SAME: S'afegeix padding per assegurar que la mida de la sortida sigui la mateixa que la de l'entrada.
  • Stride: Determina la mida del pas de la finestra de pooling. Un stride de 2 significa que la finestra es mou 2 píxels a la vegada.

Exemple amb Padding i Stride

Considera una matriu d'entrada de 4x4 i una finestra de pooling de 2x2 amb un stride d'1 i padding SAME:

Matriu d'Entrada:
[[1, 3, 2, 4],
 [5, 6, 1, 2],
 [7, 8, 3, 0],
 [2, 4, 6, 8]]

Aplicar max pooling:

Matriu Max Pooling:
[[6, 6, 4, 4],
 [8, 8, 3, 2],
 [8, 8, 6, 8],
 [4, 6, 6, 8]]

Implementació amb TensorFlow

import tensorflow as tf

# Crear una matriu d'entrada de 4x4
input_matrix = tf.constant([[1, 3, 2, 4],
                            [5, 6, 1, 2],
                            [7, 8, 3, 0],
                            [2, 4, 6, 8]], dtype=tf.float32)

# Reestructurar a tensor 4D: [batch_size, height, width, channels]
input_tensor = tf.reshape(input_matrix, [1, 4, 4, 1])

# Aplicar max pooling amb stride 1 i padding SAME
max_pooling_same = tf.nn.max_pool2d(input_tensor, ksize=2, strides=1, padding='SAME')

print("Sortida Max Pooling amb Padding SAME:\n", tf.squeeze(max_pooling_same).numpy())

Conclusió

Les capes de pooling juguen un paper vital en la reducció de les dimensions espacials dels mapes de característiques, cosa que ajuda a disminuir la càrrega computacional i a extreure característiques dominants. El max pooling i l'average pooling són els dos tipus més comuns d'operacions de pooling. Entendre com implementar aquestes operacions en TensorFlow, juntament amb els conceptes de padding i stride, és essencial per construir CNNs eficients. En dominar aquests conceptes, pots crear xarxes neuronals més efectives i computacionalment eficients.
