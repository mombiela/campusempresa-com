Introducció a TensorFlow Hub

TensorFlow Hub és una biblioteca per a mòduls reutilitzables d'aprenentatge automàtic. Et permet utilitzar models preentrenats per a diverses tasques com la classificació d'imatges, l'embedding de text i més. Això pot accelerar significativament el teu procés de desenvolupament aprofitant models existents en lloc d'entrenar des de zero.

Conceptes Clau

  • Mòduls: Models preentrenats o components de models que es poden reutilitzar.
  • Embedding: Una manera de representar dades (com text o imatges) en un espai d'alta dimensió.
  • Ajust fi: Ajustar un model preentrenat en un nou conjunt de dades per millorar el rendiment en una tasca específica.

Instal·lant TensorFlow Hub

Per començar amb TensorFlow Hub, necessites instal·lar-lo. Pots fer-ho utilitzant pip.

pip install tensorflow-hub

Utilitzant Models Preentrenats

TensorFlow Hub proporciona una varietat de models preentrenats que pots utilitzar per a diferents tasques. Aquí, ens centrarem en la classificació d'imatges i l'embedding de text.

Classificació d'Imatges

Fem servir un model de classificació d'imatges preentrenat per classificar imatges.

import tensorflow as tf
import tensorflow_hub as hub
import numpy as np
import PIL.Image as Image

# Carrega el model preentrenat
model = hub.load("https://tfhub.dev/google/imagenet/mobilenet_v2_100_224/classification/4")

# Carrega i preprocessa la imatge
image = Image.open('path_to_image.jpg').resize((224, 224))
image = np.array(image) / 255.0
image = image[np.newaxis, ...]

# Fes prediccions
predictions = model(image)
predicted_class = np.argmax(predictions, axis=-1)

print(f'Classe predita: {predicted_class}')

Embedding de Text

Els models d'embedding de text poden convertir text en vectors numèrics que es poden utilitzar per a diverses tasques de PNL.

import tensorflow as tf
import tensorflow_hub as hub

# Carrega el model d'embedding de text preentrenat
embed = hub.load("https://tfhub.dev/google/universal-sentence-encoder/4")

# Frases d'exemple
sentences = [
    "TensorFlow és una gran eina per a l'aprenentatge automàtic.",
    "M'encanta utilitzar models preentrenats."
]

# Genera embeddings
embeddings = embed(sentences)

print(embeddings)

Ajust Fi de Models Preentrenats

L'ajust fi et permet adaptar un model preentrenat al teu conjunt de dades específic. Això és particularment útil quan tens un conjunt de dades petit.

Ajust Fi d'un Model de Classificació d'Imatges

Aquí tens com pots ajustar fi un model de classificació d'imatges preentrenat.

import tensorflow as tf
import tensorflow_hub as hub
from tensorflow.keras import layers

# Carrega el model preentrenat sense la capa superior
base_model = hub.KerasLayer("https://tfhub.dev/google/imagenet/mobilenet_v2_100_224/feature_vector/4", trainable=True)

# Construeix el teu propi model a sobre
model = tf.keras.Sequential([
    base_model,
    layers.Dense(10, activation='softmax')  # Suposant 10 classes en el teu conjunt de dades
])

# Compila el model
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Carrega el teu conjunt de dades
(train_images, train_labels), (test_images, test_labels) = tf.keras.datasets.cifar10.load_data()

# Preprocessa el conjunt de dades
train_images = train_images / 255.0
test_images = test_images / 255.0

# Entrena el model
model.fit(train_images, train_labels, epochs=5, validation_data=(test_images, test_labels))

Conclusió

TensorFlow Hub és una eina poderosa que et permet aprofitar models preentrenats per a diverses tasques d'aprenentatge automàtic. Tant si estàs treballant en classificació d'imatges, embedding de text o altres tasques, TensorFlow Hub pot estalviar-te temps i recursos computacionals. Comprenent com utilitzar i ajustar fi aquests models, pots millorar significativament els teus projectes d'aprenentatge automàtic.
