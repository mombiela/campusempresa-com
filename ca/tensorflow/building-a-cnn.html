Introducció

En aquesta secció, explorarem com construir una Xarxa Neuronal Convolucional (CNN) utilitzant TensorFlow. Les CNNs són particularment efectives per a tasques de reconeixement i classificació d'imatges. Començarem des dels conceptes bàsics i gradualment avançarem cap a conceptes més avançats.

Conceptes Clau

  • Capa Convolucional: Aplica operacions de convolució a les dades d'entrada.
  • Capa de Pooling: Redueix la dimensionalitat dels mapes de característiques.
  • Capa Totalment Connectada: Connecta cada neurona d'una capa amb cada neurona d'una altra capa.
  • Funció d'Activació: Introdueix no linealitat al model.
  • Funció de Pèrdua: Mesura com de bé les prediccions del model coincideixen amb les dades reals.
  • Optimitzador: Ajusta els pesos de la xarxa per minimitzar la funció de pèrdua.

Configuració de l'Entorn

Abans de començar a construir la nostra CNN, assegura't de tenir TensorFlow instal·lat. Pots instal·lar-lo utilitzant pip:

pip install tensorflow

Construir una CNN Simple

Importar Llibreries

Primer, necessitem importar les llibreries necessàries.

import tensorflow as tf
from tensorflow.keras import layers, models

Carregar i Preprocessar Dades

Per a aquest exemple, utilitzarem el conjunt de dades CIFAR-10, que conté 60.000 imatges en color de 32x32 en 10 classes.

# Carregar el conjunt de dades CIFAR-10
(train_images, train_labels), (test_images, test_labels) = tf.keras.datasets.cifar10.load_data()

# Normalitzar els valors dels píxels per estar entre 0 i 1
train_images, test_images = train_images / 255.0, test_images / 255.0

Definir l'Arquitectura de la CNN

Crearem una CNN simple amb la següent arquitectura:

  • Capa Convolucional: 32 filtres, mida del nucli 3x3, activació ReLU
  • Capa de Pooling: MaxPooling amb mida de pool 2x2
  • Capa Convolucional: 64 filtres, mida del nucli 3x3, activació ReLU
  • Capa de Pooling: MaxPooling amb mida de pool 2x2
  • Capa Totalment Connectada: 64 unitats, activació ReLU
  • Capa de Sortida: 10 unitats (una per cada classe), activació softmax
model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

Compilar el Model

A continuació, necessitem compilar el model especificant l'optimitzador, la funció de pèrdua i les mètriques.

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

Entrenar el Model

Ara podem entrenar el model utilitzant les dades d'entrenament.

history = model.fit(train_images, train_labels, epochs=10, 
                    validation_data=(test_images, test_labels))

Avaluar el Model

Finalment, avaluem el rendiment del model en les dades de prova.

test_loss, test_acc = model.evaluate(test_images, test_labels, verbose=2)
print(f'Precisió en el test: {test_acc}')

Conceptes Avançats

Augment de Dades

Per millorar el rendiment del model, podem utilitzar l'augment de dades per augmentar artificialment la mida del conjunt de dades d'entrenament.

from tensorflow.keras.preprocessing.image import ImageDataGenerator

datagen = ImageDataGenerator(
    rotation_range=20,
    width_shift_range=0.2,
    height_shift_range=0.2,
    horizontal_flip=True)

datagen.fit(train_images)

# Utilitzar el generador de dades per entrenar el model
history = model.fit(datagen.flow(train_images, train_labels, batch_size=32),
                    epochs=10,
                    validation_data=(test_images, test_labels))

Aprenentatge per Transferència

L'aprenentatge per transferència implica utilitzar un model preentrenat i ajustar-lo per a una tasca específica. TensorFlow proporciona diversos models preentrenats a través del mòdul tf.keras.applications.

from tensorflow.keras.applications import VGG16

# Carregar el model VGG16, excloent les capes superiors
base_model = VGG16(weights='imagenet', include_top=False, input_shape=(32, 32, 3))

# Congelar el model base
base_model.trainable = False

# Afegir capes personalitzades a sobre
model = models.Sequential([
    base_model,
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# Compilar i entrenar el model com abans
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

history = model.fit(train_images, train_labels, epochs=10, 
                    validation_data=(test_images, test_labels))
