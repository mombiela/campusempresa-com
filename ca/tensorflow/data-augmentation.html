Augmentació de Dades en TensorFlow
Introducció a les xarxes neuronals ►

L'augmentació de dades és una tècnica utilitzada per augmentar la diversitat del teu conjunt de dades d'entrenament sense haver de recollir dades noves. Això és particularment útil en l'aprenentatge profund, on sovint es requereixen grans quantitats de dades per entrenar models de manera efectiva. TensorFlow proporciona diverses eines i mètodes per realitzar l'augmentació de dades de manera eficient.

Introducció a l'Augmentació de Dades

  • Definició: L'augmentació de dades implica crear nous exemples d'entrenament aplicant transformacions aleatòries a les dades existents.
  • Propòsit: Millorar la capacitat de generalització dels models d'aprenentatge automàtic exposant-los a una varietat més àmplia de dades.
  • Tècniques Comunes: Inversió, rotació, escalat, retallada i ajustaments de color.

Per què Utilitzar l'Augmentació de Dades?

  • Prevenir el Sobreajustament: En augmentar la varietat de dades d'entrenament, els models tenen menys probabilitats de memoritzar les dades d'entrenament i més probabilitats de generalitzar bé a dades noves i no vistes.
  • Millorar la Robustesa del Model: Els models entrenats amb dades augmentades són típicament més robustos a les variacions en les dades d'entrada.

Augmentació de Dades en TensorFlow

TensorFlow proporciona diverses maneres de realitzar l'augmentació de dades, incloent el mòdul tf.image i la classe tf.keras.preprocessing.image.ImageDataGenerator.

Utilitzant el Mòdul `tf.image`

El mòdul tf.image conté una varietat de funcions per realitzar transformacions d'imatges comunes.

Exemple: Transformacions Bàsiques d'Imatges

import tensorflow as tf
import matplotlib.pyplot as plt

# Carregar una imatge d'exemple
image_path = tf.keras.utils.get_file('cat.jpg', 'https://storage.googleapis.com/download.tensorflow.org/example_images/320px-Felis_catus-cat_on_snow.jpg')
image = tf.io.read_file(image_path)
image = tf.image.decode_jpeg(image)

# Mostrar la imatge original
plt.figure(figsize=(6, 6))
plt.imshow(image)
plt.title("Imatge Original")
plt.axis('off')
plt.show()

# Aplicar inversió aleatòria
flipped_image = tf.image.random_flip_left_right(image)

# Aplicar rotació aleatòria
rotated_image = tf.image.rot90(image)

# Mostrar les imatges augmentades
fig, ax = plt.subplots(1, 2, figsize=(12, 6))
ax[0].imshow(flipped_image)
ax[0].set_title("Imatge Invertida")
ax[0].axis('off')

ax[1].imshow(rotated_image)
ax[1].set_title("Imatge Rotada")
ax[1].axis('off')

plt.show()

Utilitzant `tf.keras.preprocessing.image.ImageDataGenerator`

La classe ImageDataGenerator proporciona una manera més d'alt nivell i convenient de realitzar l'augmentació de dades.

Exemple: Utilitzant ImageDataGenerator

from tensorflow.keras.preprocessing.image import ImageDataGenerator, array_to_img, img_to_array, load_img

# Crear una instància de ImageDataGenerator amb opcions d'augmentació
datagen = ImageDataGenerator(
    rotation_range=40,
    width_shift_range=0.2,
    height_shift_range=0.2,
    shear_range=0.2,
    zoom_range=0.2,
    horizontal_flip=True,
    fill_mode='nearest'
)

# Carregar una imatge d'exemple
img = load_img(image_path)  # aquesta és una imatge PIL
x = img_to_array(img)  # això és un array de Numpy amb forma (3, 150, 150)
x = x.reshape((1,) + x.shape)  # això és un array de Numpy amb forma (1, 3, 150, 150)

# Generar lots d'imatges augmentades
i = 0
for batch in datagen.flow(x, batch_size=1):
    plt.figure(i)
    imgplot = plt.imshow(array_to_img(batch[0]))
    i += 1
    if i % 4 == 0:
        break

plt.show()

Tècniques Avançades d'Augmentació de Dades

Per a estratègies d'augmentació més complexes, TensorFlow proporciona l'API tf.data, que permet crear pipelines d'augmentació personalitzats.

Exemple: Pipeline d'Augmentació Personalitzat amb `tf.data`

def augment(image):
    image = tf.image.random_flip_left_right(image)
    image = tf.image.random_brightness(image, max_delta=0.3)
    image = tf.image.random_contrast(image, lower=0.8, upper=1.2)
    return image

# Crear un conjunt de dades
dataset = tf.data.Dataset.from_tensor_slices([image])
dataset = dataset.map(augment)

# Mostrar les imatges augmentades
for img in dataset:
    plt.figure()
    plt.imshow(img)
    plt.axis('off')
    plt.show()

Conclusió

L'augmentació de dades és una tècnica poderosa per millorar el rendiment i la robustesa dels models d'aprenentatge automàtic. TensorFlow proporciona múltiples mètodes per realitzar l'augmentació de dades, des de transformacions simples d'imatges utilitzant el mòdul tf.image fins a pipelines més avançats utilitzant tf.data. En incorporar l'augmentació de dades en el teu procés d'entrenament, pots millorar significativament la capacitat de generalització dels teus models.
