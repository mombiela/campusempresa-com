Xarxes Generatives Adversàries (GANs)
Autoencoders ►

Les Xarxes Generatives Adversàries (GANs) són una classe de marcs d'aprenentatge automàtic dissenyats per Ian Goodfellow i els seus col·legues el 2014. Les GANs consisteixen en dues xarxes neuronals, el generador i el discriminador, que competeixen entre elles en un joc de suma zero. Aquest tema et guiarà a través de la comprensió de les GANs, la seva implementació utilitzant TensorFlow, i l'exploració de tècniques avançades.

Introducció a les GANs

  • Definició: Les GANs estan compostes per dos models: el generador, que crea dades, i el discriminador, que avalua les dades.
  • Objectiu: El generador té com a objectiu produir dades que siguin indistingibles de les dades reals, mentre que el discriminador té com a objectiu distingir entre dades reals i generades.
  • Aplicacions: Generació d'imatges, augment de dades, transferència d'estil, i més.

Conceptes Bàsics

  • Generador: Una xarxa neuronal que genera dades falses a partir de soroll aleatori.
  • Discriminador: Una xarxa neuronal que distingeix entre dades reals i falses.
  • Funcions de Pèrdua:
    • Pèrdua del Generador: Mesura com de bé el generador enganya el discriminador.
    • Pèrdua del Discriminador: Mesura com de bé el discriminador distingeix entre dades reals i falses.

Implementació d'una GAN Bàsica en TensorFlow

Pas 1: Configuració de l'Entorn

import tensorflow as tf
from tensorflow.keras.layers import Dense, Flatten, Reshape, LeakyReLU
from tensorflow.keras.models import Sequential
import numpy as np
import matplotlib.pyplot as plt

Pas 2: Construcció del Generador

def build_generator():
    model = Sequential()
    model.add(Dense(256, input_dim=100))
    model.add(LeakyReLU(alpha=0.2))
    model.add(Dense(512))
    model.add(LeakyReLU(alpha=0.2))
    model.add(Dense(1024))
    model.add(LeakyReLU(alpha=0.2))
    model.add(Dense(28 * 28 * 1, activation='tanh'))
    model.add(Reshape((28, 28, 1)))
    return model

generator = build_generator()
generator.summary()

Pas 3: Construcció del Discriminador

def build_discriminator():
    model = Sequential()
    model.add(Flatten(input_shape=(28, 28, 1)))
    model.add(Dense(512))
    model.add(LeakyReLU(alpha=0.2))
    model.add(Dense(256))
    model.add(LeakyReLU(alpha=0.2))
    model.add(Dense(1, activation='sigmoid'))
    return model

discriminator = build_discriminator()
discriminator.summary()

Pas 4: Compilació dels Models

discriminator.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
discriminator.trainable = False

gan_input = tf.keras.Input(shape=(100,))
generated_image = generator(gan_input)
gan_output = discriminator(generated_image)

gan = tf.keras.Model(gan_input, gan_output)
gan.compile(loss='binary_crossentropy', optimizer='adam')

Pas 5: Entrenament de la GAN

def train_gan(epochs, batch_size, sample_interval):
    (X_train, _), (_, _) = tf.keras.datasets.mnist.load_data()
    X_train = X_train / 127.5 - 1.0
    X_train = np.expand_dims(X_train, axis=3)
    valid = np.ones((batch_size, 1))
    fake = np.zeros((batch_size, 1))

    for epoch in range(epochs):
        idx = np.random.randint(0, X_train.shape[0], batch_size)
        real_imgs = X_train[idx]

        noise = np.random.normal(0, 1, (batch_size, 100))
        gen_imgs = generator.predict(noise)

        d_loss_real = discriminator.train_on_batch(real_imgs, valid)
        d_loss_fake = discriminator.train_on_batch(gen_imgs, fake)
        d_loss = 0.5 * np.add(d_loss_real, d_loss_fake)

        noise = np.random.normal(0, 1, (batch_size, 100))
        g_loss = gan.train_on_batch(noise, valid)

        if epoch % sample_interval == 0:
            print(f"{epoch} [D loss: {d_loss[0]} | D accuracy: {100*d_loss[1]}] [G loss: {g_loss}]")
            sample_images(epoch)

def sample_images(epoch):
    noise = np.random.normal(0, 1, (25, 100))
    gen_imgs = generator.predict(noise)
    gen_imgs = 0.5 * gen_imgs + 0.5

    fig, axs = plt.subplots(5, 5)
    cnt = 0
    for i in range(5):
        for j in range(5):
            axs[i, j].imshow(gen_imgs[cnt, :, :, 0], cmap='gray')
            axs[i, j].axis('off')
            cnt += 1
    plt.show()

train_gan(epochs=10000, batch_size=64, sample_interval=1000)

Tècniques Avançades

  • GANs condicionals (cGANs): GANs condicionades a informació addicional, com ara etiquetes de classe.
  • GANs de Wasserstein (WGANs): GANs amb una funció de pèrdua diferent per millorar l'estabilitat de l'entrenament.
  • CycleGANs: GANs per a la traducció d'imatges a imatges sense exemples aparellats.

Conclusió

Les Xarxes Generatives Adversàries són una eina poderosa en el camp de l'aprenentatge automàtic, capaç de generar dades realistes. Comprenent els conceptes bàsics i implementant una GAN senzilla en TensorFlow, pots construir una base per explorar arquitectures i aplicacions de GANs més avançades. Continua experimentant amb diferents tipus de GANs per aprofundir en la teva comprensió i millorar les teves habilitats.
