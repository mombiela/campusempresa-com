Introducció als tensors

Els tensors són els blocs fonamentals en TensorFlow. Són arrays multidimensionals amb un tipus uniforme (anomenat dtype). Entendre els tensors és crucial per treballar eficaçment amb TensorFlow.

Conceptes clau

  • Escalars: Tensors de 0 dimensions (per exemple, un sol número).
  • Vectors: Tensors de 1 dimensió (per exemple, una llista de números).
  • Matrius: Tensors de 2 dimensions (per exemple, una taula de números).
  • Tensors de dimensions superiors: Tensors amb 3 o més dimensions.

Exemples

import tensorflow as tf

# Escalar
escalar = tf.constant(3)
print(escalar)

# Vector
vector = tf.constant([1, 2, 3])
print(vector)

# Matriu
matriu = tf.constant([[1, 2], [3, 4]])
print(matriu)

# Tensor de dimensions superiors
tensor = tf.constant([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print(tensor)

Operacions amb tensors

TensorFlow proporciona una varietat d'operacions que es poden realitzar sobre tensors. Aquestes operacions són essencials per construir i entrenar models d'aprenentatge automàtic.

Operacions bàsiques

  • Suma: Sumar dos tensors element a element.
  • Resta: Restar un tensor d'un altre element a element.
  • Multiplicació: Multiplicació element a element de dos tensors.
  • Divisió: Divisió element a element d'un tensor per un altre.

Exemples

# Definir tensors
a = tf.constant([1, 2, 3])
b = tf.constant([4, 5, 6])

# Suma
suma = tf.add(a, b)
print(suma)

# Resta
resta = tf.subtract(a, b)
print(resta)

# Multiplicació
multiplicacio = tf.multiply(a, b)
print(multiplicacio)

# Divisió
divisio = tf.divide(a, b)
print(divisio)

Operacions avançades

  • Multiplicació de matrius: Multiplicar dues matrius.
  • Transposició: Transposar una matriu.
  • Reformar: Canviar la forma d'un tensor sense canviar les seves dades.
  • Tall: Extreure un sub-tensor d'un tensor.

Exemples

# Definir matrius
matriu1 = tf.constant([[1, 2], [3, 4]])
matriu2 = tf.constant([[5, 6], [7, 8]])

# Multiplicació de matrius
matmul = tf.matmul(matriu1, matriu2)
print(matmul)

# Transposició
transposicio = tf.transpose(matriu1)
print(transposicio)

# Reformar
reformar = tf.reshape(matriu1, [1, 4])
print(reformar)

# Tall
tall = matriu1[:, 1]
print(tall)

Broadcasting

El broadcasting és un mecanisme poderós que permet a TensorFlow realitzar operacions sobre tensors de formes diferents.

Conceptes clau

  • Regles de broadcasting: TensorFlow expandeix automàticament el tensor més petit per coincidir amb la forma del tensor més gran.
  • Compatibilitat: Dos tensors són compatibles per al broadcasting si, per a cada dimensió, les mides de les dimensions són iguals o una d'elles és 1.

Exemples

# Definir tensors
a = tf.constant([1, 2, 3])
b = tf.constant([[1], [2], [3]])

# Suma amb broadcasting
broadcast_add = tf.add(a, b)
print(broadcast_add)

Resum

En aquesta secció, hem cobert els conceptes bàsics dels tensors i les operacions en TensorFlow. Hem après sobre diferents tipus de tensors, operacions bàsiques i avançades amb tensors, i el concepte de broadcasting. Entendre aquests fonaments és essencial per construir i entrenar models d'aprenentatge automàtic en TensorFlow.

Dominant aquests conceptes, estaràs ben preparat per manejar tasques i operacions més complexes en TensorFlow, obrint el camí per a projectes avançats d'aprenentatge automàtic i aprenentatge profund.
