Configurar l'entorn és un primer pas crucial per treballar amb TensorFlow. Aquesta secció et guiarà a través del procés d'instal·lació de TensorFlow i les seves dependències, configurant el teu entorn de desenvolupament i verificant la teva instal·lació.

Instal·lant TensorFlow

Per començar a utilitzar TensorFlow, necessites instal·lar-lo a la teva màquina. TensorFlow es pot instal·lar utilitzant pip, l'instal·lador de paquets de Python.

Prerequisits

Abans d'instal·lar TensorFlow, assegura't de tenir els següents prerequisits:

  • Python: TensorFlow suporta Python 3.7 a 3.10.
  • pip: Assegura't de tenir la versió més recent de pip.

Passos d'instal·lació

  1. Instal·la Python: Si no tens Python instal·lat, descarrega'l i instal·la'l des del lloc web oficial de Python.

  2. Actualitza pip: Obre el teu terminal o símbol del sistema i executa:

    pip install --upgrade pip

  3. Instal·la TensorFlow: Utilitza pip per instal·lar TensorFlow. Pots triar instal·lar la versió per a CPU o GPU. Es recomana la versió per a GPU si tens una GPU NVIDIA compatible.

    • Versió per a CPU:

      pip install tensorflow

    • Versió per a GPU:

      pip install tensorflow-gpu

Verificant la instal·lació

Per verificar que TensorFlow està instal·lat correctament, obre un intèrpret de Python i executa les següents comandes:

import tensorflow as tf
print(tf.__version__)

Si TensorFlow està instal·lat correctament, això imprimirà la versió de TensorFlow que tens instal·lada.

Configurant l'entorn de desenvolupament

Un cop TensorFlow està instal·lat, necessites configurar el teu entorn de desenvolupament. Això implica triar un Entorn de Desenvolupament Integrat (IDE) i configurar-lo per al desenvolupament amb TensorFlow.

Triant un IDE

Aquí tens alguns IDEs populars que funcionen bé amb TensorFlow:

  • PyCharm: Un IDE potent per a Python amb excel·lent suport per a TensorFlow.
  • Visual Studio Code (VS Code): Un editor de codi lleuger i versàtil amb extensions per a TensorFlow.
  • Jupyter Notebook: Un entorn interactiu que és ideal per experimentar amb codi de TensorFlow.

Configurant el teu IDE

PyCharm

  1. Instal·la PyCharm: Descarrega i instal·la PyCharm des del lloc web oficial.
  2. Crea un nou projecte: Obre PyCharm i crea un nou projecte.
  3. Configura l'intèrpret de Python: Ves a File > Settings > Project: <your_project> > Python Interpreter i selecciona l'intèrpret de Python on TensorFlow està instal·lat.

Visual Studio Code

  1. Instal·la VS Code: Descarrega i instal·la VS Code des del lloc web oficial.
  2. Instal·la l'extensió de Python: Obre VS Code i instal·la l'extensió de Python des del mercat d'Extensions.
  3. Selecciona l'intèrpret de Python: Prem Ctrl+Shift+P, escriu Python: Select Interpreter i tria l'intèrpret on TensorFlow està instal·lat.

Jupyter Notebook

  1. Instal·la Jupyter Notebook: Si no tens Jupyter Notebook instal·lat, pots instal·lar-lo utilitzant pip: 
    pip install notebook
  2. Inicia Jupyter Notebook: Executa la següent comanda per iniciar Jupyter Notebook: 
    jupyter notebook
  3. Crea un nou Notebook: A la interfície de Jupyter, crea un nou notebook de Python 3.

Verificant la configuració

Per assegurar-te que el teu entorn de desenvolupament està configurat correctament, crea un script senzill de TensorFlow al teu IDE triat o Jupyter Notebook:

import tensorflow as tf

# Crea un tensor constant
hello = tf.constant('Hello, TensorFlow!')

# Inicia una sessió de TensorFlow
with tf.Session() as sess:
    print(sess.run(hello))

Si tot està configurat correctament, aquest script hauria d'imprimir "Hello, TensorFlow!".

Conclusió

Configurar l'entorn per a TensorFlow implica instal·lar la biblioteca, configurar les teves eines de desenvolupament i verificar la instal·lació. Seguint els passos descrits en aquesta secció, estaràs preparat per començar a desenvolupar aplicacions amb TensorFlow. A les següents seccions, aprofundirem en les característiques i capacitats de TensorFlow, començant amb operacions bàsiques i avançant cap a temes avançats.
