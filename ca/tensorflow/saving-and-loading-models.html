En aquesta secció, cobrirem com desar i carregar models en TensorFlow. Aquesta és una habilitat crucial per a qualsevol practicant d'aprenentatge automàtic, ja que et permet preservar els teus models entrenats i reutilitzar-los més tard sense haver de tornar a entrenar-los.

Per què Desar i Carregar Models?

  • Preservació: Desa el model per evitar haver de tornar a entrenar-lo.
  • Desplegament: Carrega el model per al desplegament en entorns de producció.
  • Compartició: Comparteix el model amb altres persones.
  • Experimentació: Desa models intermedis durant l'entrenament per a l'experimentació.

Mètodes per Desar i Carregar Models

TensorFlow proporciona diversos mètodes per desar i carregar models:

  1. Checkpointing: Desa i carrega els pesos del model.
  2. SavedModel: Desa i carrega tota l'arquitectura del model, els pesos i l'estat de l'optimitzador.
  3. Format HDF5: Desa i carrega models utilitzant el format HDF5.

Checkpointing

El checkpointing és útil per desar i carregar només els pesos del model.

Desar Pesos

import tensorflow as tf

# Defineix un model senzill
model = tf.keras.Sequential([
    tf.keras.layers.Dense(10, activation='relu'),
    tf.keras.layers.Dense(1)
])

# Compila el model
model.compile(optimizer='adam', loss='mean_squared_error')

# Desa els pesos
model.save_weights('model_checkpoint')

Carregar Pesos

# Carrega els pesos
model.load_weights('model_checkpoint')

Format SavedModel

El format SavedModel és el format de serialització estàndard de TensorFlow. Inclou l'arquitectura, els pesos i l'estat de l'optimitzador.

Desar el Model

# Desa tot el model
model.save('saved_model/my_model')

Carregar el Model

# Carrega tot el model
loaded_model = tf.keras.models.load_model('saved_model/my_model')

Format HDF5

El format HDF5 és una altra manera de desar tot el model, incloent l'arquitectura, els pesos i l'estat de l'optimitzador.

Desar el Model

# Desa tot el model en format HDF5
model.save('my_model.h5')

Carregar el Model

# Carrega tot el model des del format HDF5
loaded_model = tf.keras.models.load_model('my_model.h5')

Comparació de Mètodes

| Mètode | Desa Arquitectura | Desa Pesos | Desa Estat de l'Optimitzador | Format | |--------------|--------------------|------------|------------------------------|--------------| | Checkpointing| No | Sí | No | TensorFlow | | SavedModel | Sí | Sí | Sí | TensorFlow | | HDF5 | Sí | Sí | Sí | HDF5 |

Exemple Pràctic

Fem un recorregut per un exemple pràctic d'entrenament d'un model, desant-lo i després carregant-lo per a inferència.

import tensorflow as tf
import numpy as np

# Genera dades fictícies
x_train = np.random.random((1000, 32))
y_train = np.random.random((1000, 1))

# Defineix un model senzill
model = tf.keras.Sequential([
    tf.keras.layers.Dense(64, activation='relu', input_shape=(32,)),
    tf.keras.layers.Dense(1)
])

# Compila el model
model.compile(optimizer='adam', loss='mean_squared_error')

# Entrena el model
model.fit(x_train, y_train, epochs=10)

# Desa el model
model.save('my_model.h5')

# Carrega el model
loaded_model = tf.keras.models.load_model('my_model.h5')

# Fes prediccions amb el model carregat
predictions = loaded_model.predict(x_train)
print(predictions)

Conclusió

En aquesta secció, hem cobert la importància de desar i carregar models en TensorFlow. Hem explorat diferents mètodes, incloent el checkpointing, el format SavedModel i el format HDF5. Cada mètode té els seus propis casos d'ús i beneficis. Dominant aquestes tècniques, pots assegurar-te que els teus models es preservin, siguin fàcilment desplegables i compartibles.
