Introducció

En aquesta secció, explorarem els conceptes de sobreajustament i subajustament, que són crucials en el camp de l'aprenentatge automàtic. Entendre aquests conceptes t'ajudarà a construir models més precisos i generalitzables utilitzant TensorFlow.

Conceptes Clau

Sobreajustament

  • Definició: El sobreajustament es produeix quan un model aprèn massa bé les dades d'entrenament, capturant soroll i detalls que no es generalitzen a noves dades.
  • Símptomes: Alta precisió en les dades d'entrenament però un rendiment pobre en les dades de validació/prova.
  • Causes:
    • Model massa complex (per exemple, massa capes o neurones).
    • Dades d'entrenament insuficients.
    • Massa èpoques d'entrenament.

Subajustament

  • Definició: El subajustament passa quan un model és massa simple per capturar els patrons subjacents en les dades.
  • Símptomes: Rendiment pobre tant en les dades d'entrenament com en les de validació/prova.
  • Causes:
    • Model massa simple (per exemple, massa poques capes o neurones).
    • Èpoques d'entrenament insuficients.
    • Mala selecció de característiques.

Estratègies per Mitigar el Sobreajustament i el Subajustament

Tècniques per Prevenir el Sobreajustament

  • Regularització: Afegir una penalització a la funció de pèrdua per desincentivar models complexos.
    • Regularització L1 (Lasso)
    • Regularització L2 (Ridge)
  • Dropout: Eliminar aleatòriament unitats (juntament amb les seves connexions) durant l'entrenament per prevenir la co-adaptació.
  • Aturada Primerenca: Monitoritzar la pèrdua de validació i aturar l'entrenament quan comença a augmentar.
  • Augment de Dades: Augmentar la diversitat de les dades d'entrenament aplicant transformacions aleatòries.

Tècniques per Prevenir el Subajustament

  • Augmentar la Complexitat del Model: Afegir més capes o neurones al model.
  • Entrenar Més Temps: Augmentar el nombre d'èpoques d'entrenament.
  • Enginyeria de Característiques: Crear característiques més rellevants a partir de les dades existents.

Exemples Pràctics

Exemple de Sobreajustament

import tensorflow as tf
from tensorflow.keras import layers, models
from tensorflow.keras.datasets import mnist

# Carregar dades
(train_images, train_labels), (test_images, test_labels) = mnist.load_data()
train_images = train_images.reshape((60000, 28, 28, 1)).astype('float32') / 255
test_images = test_images.reshape((10000, 28, 28, 1)).astype('float32') / 255

# Construir un model complex
model = models.Sequential([
    layers.Conv2D(64, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu'),
    layers.Flatten(),
    layers.Dense(512, activation='relu'),
    layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Entrenar el model
history = model.fit(train_images, train_labels, epochs=20, validation_data=(test_images, test_labels))

Exemple de Subajustament

# Construir un model simple
model = models.Sequential([
    layers.Flatten(input_shape=(28, 28, 1)),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Entrenar el model
history = model.fit(train_images, train_labels, epochs=5, validation_data=(test_images, test_labels))

Aplicar Regularització per Prevenir el Sobreajustament

# Construir un model amb regularització L2
model = models.Sequential([
    layers.Conv2D(64, (3, 3), activation='relu', input_shape=(28, 28, 1), kernel_regularizer=tf.keras.regularizers.l2(0.001)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu', kernel_regularizer=tf.keras.regularizers.l2(0.001)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(128, (3, 3), activation='relu', kernel_regularizer=tf.keras.regularizers.l2(0.001)),
    layers.Flatten(),
    layers.Dense(512, activation='relu', kernel_regularizer=tf.keras.regularizers.l2(0.001)),
    layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Entrenar el model
history = model.fit(train_images, train_labels, epochs=20, validation_data=(test_images, test_labels))

Conclusió

Entendre i abordar el sobreajustament i el subajustament són essencials per construir models d'aprenentatge automàtic efectius. Utilitzant tècniques com la regularització, el dropout i l'aturada primerenca, pots millorar la capacitat de generalització del teu model. Per contra, augmentar la complexitat del model i la durada de l'entrenament pot ajudar a mitigar el subajustament. Amb aquestes estratègies, pots crear models més robustos utilitzant TensorFlow.
