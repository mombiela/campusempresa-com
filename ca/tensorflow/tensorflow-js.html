TensorFlow.js és una biblioteca potent per a l'aprenentatge automàtic en JavaScript. Et permet construir i entrenar models directament al navegador o en Node.js. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de TensorFlow.js, proporcionant-te el coneixement per aprofitar aquesta tecnologia en les teves aplicacions web.

Introducció a TensorFlow.js

  • Què és TensorFlow.js?

    • Una biblioteca de JavaScript per entrenar i desplegar models d'aprenentatge automàtic al navegador i en Node.js.
    • Permet aplicacions d'aprenentatge automàtic en temps real i interactives.

  • Per què utilitzar TensorFlow.js?

    • Aprofita el poder de l'aprenentatge automàtic en aplicacions web.
    • Utilitza la GPU per a càlculs directament al navegador.
    • Construeix i entrena models en JavaScript, fent-lo accessible als desenvolupadors web.

Configuració de TensorFlow.js

  • Instal·lant TensorFlow.js

    • Utilitzant npm: 
      npm install @tensorflow/tfjs
    • Utilitzant un CDN: 
      <script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs"></script>

  • Exemple de Configuració Bàsica

    <!DOCTYPE html>
<html>
<head>

  <script>
    // El teu codi de TensorFlow.js anirà aquí

  </script>
</body>
</html>

Conceptes Bàsics

  • Tensors

    • La unitat central de dades en TensorFlow.js.
    • Arrays multidimensionals amb un tipus uniforme.
    • Exemple: 
      const tensor = tf.tensor([1, 2, 3, 4]);
tensor.print();  // Sortida: [1, 2, 3, 4]

  • Operacions amb Tensors

    • Operacions bàsiques com suma, multiplicació, etc.
    • Exemple: 
      const a = tf.tensor([1, 2, 3, 4]);
const b = tf.tensor([5, 6, 7, 8]);
const c = tf.add(a, b);
c.print();  // Sortida: [6, 8, 10, 12]

Construcció i Entrenament de Models

  • Model Seqüencial

    • Una pila lineal de capes.
    • Exemple: 
      const model = tf.sequential();
model.add(tf.layers.dense({units: 32, inputShape: [50]}));
model.add(tf.layers.dense({units: 1}));

  • Compilant el Model

    • Especifica l'optimitzador, la funció de pèrdua i les mètriques.
    • Exemple: 
      model.compile({
  optimizer: 'sgd',
  loss: 'meanSquaredError',
  metrics: ['mse']
});

  • Entrenant el Model

    • Utilitzant el mètode fit.
    • Exemple: 
      const xs = tf.tensor2d([[1], [2], [3], [4]], [4, 1]);
const ys = tf.tensor2d([[1], [3], [5], [7]], [4, 1]);

model.fit(xs, ys, {
  epochs: 10
}).then(() => {
  // Utilitza el model per a inferència
  model.predict(tf.tensor2d([[5]], [1, 1])).print();  // Sortida: [9]
});

Temes Avançats

  • Transfer Learning

    • Utilitzant models pre-entrenats i ajustant-los per a tasques específiques.
    • Exemple amb MobileNet: 
      const mobilenet = await tf.loadLayersModel('https://tfhub.dev/google/tfjs-model/imagenet/mobilenet_v2_100_224/classification/3/default/1', {fromTFHub: true});
const activation = mobilenet.predict(tf.zeros([1, 224, 224, 3]));

  • Capes i Models Personalitzats

    • Creant capes personalitzades estenent tf.layers.Layer.
    • Exemple: 
      class CustomLayer extends tf.layers.Layer {
  constructor() {
    super({});
  }

  computeOutputShape(inputShape) {
    return inputShape;
  }

  call(input) {
    return input.mul(tf.scalar(2));
  }

  static get className() {
    return 'CustomLayer';
  }
}
tf.serialization.registerClass(CustomLayer);

const model = tf.sequential();
model.add(new CustomLayer());

  • Utilitzant TensorFlow.js amb Node.js

    • Executant TensorFlow.js al costat del servidor.
    • Exemple: 
      const tf = require('@tensorflow/tfjs-node');

const tensor = tf.tensor([1, 2, 3, 4]);
tensor.print();  // Sortida: [1, 2, 3, 4]

Conclusió

TensorFlow.js porta el poder de l'aprenentatge automàtic a l'ecosistema de JavaScript, permetent als desenvolupadors construir, entrenar i desplegar models al navegador i en Node.js. Entenent els conceptes bàsics, configurant l'entorn i explorant temes avançats, pots crear aplicacions d'aprenentatge automàtic sofisticades que s'executen de manera eficient a la web. Tant si ets un principiant com un usuari avançat, TensorFlow.js ofereix un conjunt d'eines versàtil i potent per als teus projectes d'aprenentatge automàtic.
