A mesura que avances en el teu viatge d'aprenentatge de TensorFlow, és important construir sobre els coneixements fonamentals que has adquirit i explorar temes més avançats. Conceptes Avançats de TensorFlow

Per convertir-te en un expert en TensorFlow, hauries d'endinsar-te en conceptes i tècniques més complexes. Aquí tens algunes àrees clau en les quals centrar-te:

  • Capes i Models Personalitzats: Aprèn a crear capes i models personalitzats per adaptar TensorFlow a les teves necessitats específiques.
  • Entrenament Distribuït: Entén com aprofitar múltiples GPUs i TPUs per a un entrenament més ràpid.
  • Optimització de Models: Explora tècniques per optimitzar el rendiment del model, incloent quantització i poda.
  • TensorFlow Serving: Descobreix com desplegar els teus models en entorns de producció utilitzant TensorFlow Serving.

Capes i Models Personalitzats

Crear capes i models personalitzats et permetrà estendre les capacitats de TensorFlow. Aquí tens un exemple de com crear una capa personalitzada:

import tensorflow as tf

class MyCustomLayer(tf.keras.layers.Layer):
    def __init__(self, units=32):
        super(MyCustomLayer, self).__init__()
        self.units = units

    def build(self, input_shape):
        self.w = self.add_weight(shape=(input_shape[-1], self.units),
                                 initializer='random_normal',
                                 trainable=True)
        self.b = self.add_weight(shape=(self.units,),
                                 initializer='random_normal',
                                 trainable=True)

    def call(self, inputs):
        return tf.matmul(inputs, self.w) + self.b

# Exemple d'ús
model = tf.keras.Sequential([
    tf.keras.layers.Input(shape=(4,)),
    MyCustomLayer(10)
])

Entrenament Distribuït

L'entrenament distribuït pot accelerar significativament el procés d'entrenament. TensorFlow proporciona diverses estratègies per a l'entrenament distribuït:

  • MirroredStrategy: Entrenament sincrònic en múltiples GPUs en una màquina.
  • MultiWorkerMirroredStrategy: Entrenament sincrònic en múltiples màquines.
  • TPUStrategy: Entrenament en TPUs.

Exemple d'ús de MirroredStrategy:

strategy = tf.distribute.MirroredStrategy()

with strategy.scope():
    model = tf.keras.Sequential([
        tf.keras.layers.Dense(128, activation='relu'),
        tf.keras.layers.Dense(10)
    ])
    model.compile(loss='sparse_categorical_crossentropy', optimizer='adam')

Optimització de Models

Optimitzar models pot fer-los més eficients i adequats per al desplegament en diversos dispositius. Les tècniques inclouen:

  • Quantització: Reduir la precisió dels números utilitzats per representar els paràmetres del teu model.
  • Poda: Eliminar pesos que no són crítics per a les prediccions del model.

Exemple d'entrenament conscient de la quantització:

import tensorflow_model_optimization as tfmot

quantize_model = tfmot.quantization.keras.quantize_model

# Quantitzar el model
q_aware_model = quantize_model(model)

# Compilar i entrenar el model quantitzat
q_aware_model.compile(optimizer='adam', loss='sparse_categorical_crossentropy')
q_aware_model.fit(train_dataset, epochs=1)

TensorFlow Serving

Desplegar models en producció requereix una infraestructura robusta de servei. TensorFlow Serving és un sistema de servei flexible i d'alt rendiment per a models d'aprenentatge automàtic.

Passos bàsics per servir un model:

  1. Exportar el model:

    model.save('path/to/exported_model')

  2. Iniciar TensorFlow Serving:

    tensorflow_model_server --rest_api_port=8501 --model_name=my_model --model_base_path="path/to/exported_model"

  3. Fer prediccions:

    import requests
import json

data = json.dumps({"signature_name": "serving_default", "instances": [[1.0, 2.0, 5.0]]})
headers = {"content-type": "application/json"}
json_response = requests.post('http://localhost:8501/v1/models/my_model:predict', data=data, headers=headers)
predictions = json_response.json()['predictions']
