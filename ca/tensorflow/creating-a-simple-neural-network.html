Introducció

En aquesta secció, cobrirem els conceptes bàsics per crear una xarxa neuronal simple utilitzant TensorFlow. Això inclourà configurar l'entorn, entendre els conceptes bàsics i construir un model de xarxa neuronal bàsic.

Configuració de l'Entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar TensorFlow: Assegura't de tenir Python instal·lat, després instal·la TensorFlow utilitzant pip. 
    pip install tensorflow
  • Verificar la Instal·lació: Comprova si TensorFlow està instal·lat correctament. 
    import tensorflow as tf
print(tf.__version__)

Conceptes Bàsics

Abans d'endinsar-nos en el codi, entenguem alguns conceptes clau:

  • Xarxa Neuronal: Una sèrie d'algoritmes que intenten reconèixer relacions subjacents en un conjunt de dades mitjançant un procés que imita la manera com opera el cervell humà.
  • Capas: Els blocs de construcció d'una xarxa neuronal. Cada capa consisteix en nodes (neurones) que realitzen càlculs.
  • Funció d'Activació: Una funció aplicada a la sortida de cada node per introduir no linealitat en el model.
  • Funció de Pèrdua: Un mètode per mesurar com de bé està funcionant la xarxa neuronal.
  • Optimitzador: Un algoritme per ajustar els pesos de la xarxa per minimitzar la funció de pèrdua.

Construcció d'una Xarxa Neuronal Simple

Construïm una xarxa neuronal simple per classificar dígits escrits a mà del conjunt de dades MNIST.

Pas 1: Importar Llibreries

Primer, importa les llibreries necessàries.

import tensorflow as tf
from tensorflow.keras import layers, models
import numpy as np

Pas 2: Carregar i Preprocessar Dades

Carrega el conjunt de dades MNIST i preprocessa'l.

# Carregar conjunt de dades
mnist = tf.keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

# Normalitzar les dades
x_train, x_test = x_train / 255.0, x_test / 255.0

Pas 3: Definir el Model

Crea un model de xarxa neuronal simple.

model = models.Sequential([
    layers.Flatten(input_shape=(28, 28)),  # Aplanar l'entrada
    layers.Dense(128, activation='relu'),  # Capa oculta amb activació ReLU
    layers.Dense(10, activation='softmax') # Capa de sortida amb activació softmax
])

Pas 4: Compilar el Model

Compila el model amb un optimitzador, funció de pèrdua i mètriques.

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

Pas 5: Entrenar el Model

Entrena el model utilitzant les dades d'entrenament.

model.fit(x_train, y_train, epochs=5)

Pas 6: Avaluar el Model

Avalua el model utilitzant les dades de prova.

test_loss, test_acc = model.evaluate(x_test, y_test)
print(f'Precisió en el test: {test_acc}')

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per crear una xarxa neuronal simple utilitzant TensorFlow. Hem començat configurant l'entorn, entenent conceptes bàsics, i després construint, compilant, entrenant i avaluant un model de xarxa neuronal bàsic. Aquest coneixement fonamental serà crucial a mesura que avancem cap a temes més avançats en xarxes neuronals i TensorFlow.
