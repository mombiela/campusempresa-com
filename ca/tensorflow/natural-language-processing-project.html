Introducció al Processament del Llenguatge Natural (PLN)

El Processament del Llenguatge Natural (PLN) és un camp de la intel·ligència artificial que se centra en la interacció entre ordinadors i humans a través del llenguatge natural. L'objectiu és permetre que els ordinadors entenguin, interpretin i generin llenguatge humà d'una manera valuosa.

Conceptes Clau en PLN

  • Tokenització: Dividir el text en paraules o frases individuals.
  • Arrelament i Lematització: Reduir les paraules a la seva forma base o arrel.
  • Paraules de Parada: Paraules comunament utilitzades (per exemple, "i", "el") que sovint es treuen de les dades de text.
  • Bossa de Paraules: Una representació del text que descriu l'ocurrència de paraules dins d'un document.
  • TF-IDF: Term Frequency-Inverse Document Frequency, una mesura estadística utilitzada per avaluar la importància d'una paraula en un document en relació amb una col·lecció de documents.
  • Embeddings de Paraules: Representacions de paraules en un espai vectorial continu on paraules similars tenen vectors similars.

Preprocessament de Text

El preprocessament de text és un pas crucial en PLN per netejar i preparar les dades de text per a l'anàlisi.

Exemple: Preprocessament Bàsic de Text

import nltk
from nltk.corpus import stopwords
from nltk.tokenize import word_tokenize
from nltk.stem import WordNetLemmatizer

# Descarregar els fitxers de dades necessaris de NLTK
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('wordnet')

text = "Natural Language Processing with TensorFlow is exciting!"

# Tokenització
tokens = word_tokenize(text)

# Treure paraules de parada
stop_words = set(stopwords.words('english'))
filtered_tokens = [word for word in tokens if word.lower() not in stop_words]

# Lematització
lemmatizer = WordNetLemmatizer()
lemmatized_tokens = [lemmatizer.lemmatize(token) for token in filtered_tokens]

print(lemmatized_tokens)

Construcció d'un Model de PLN amb TensorFlow

TensorFlow proporciona eines potents per construir i entrenar models de PLN. Començarem amb un model senzill de classificació de text.

Exemple: Classificació de Text amb TensorFlow

import tensorflow as tf
from tensorflow.keras.preprocessing.text import Tokenizer
from tensorflow.keras.preprocessing.sequence import pad_sequences

# Dades de mostra
sentences = [
    'I love machine learning',
    'Natural language processing is fascinating',
    'TensorFlow makes it easy to build models'
]
labels = [1, 1, 0]  # 1 per sentiment positiu, 0 per sentiment neutre/negatiu

# Tokenització i padding
tokenizer = Tokenizer(num_words=100)
tokenizer.fit_on_texts(sentences)
sequences = tokenizer.texts_to_sequences(sentences)
padded_sequences = pad_sequences(sequences, padding='post')

# Construir el model
model = tf.keras.Sequential([
    tf.keras.layers.Embedding(input_dim=100, output_dim=16, input_length=10),
    tf.keras.layers.GlobalAveragePooling1D(),
    tf.keras.layers.Dense(16, activació='relu'),
    tf.keras.layers.Dense(1, activació='sigmoid')
])

model.compile(optimitzador='adam', pèrdua='binary_crossentropy', mètriques=['accuracy'])

# Entrenar el model
model.fit(padded_sequences, labels, epochs=10)

Tècniques Avançades de PLN

A mesura que progressis, pots explorar tècniques de PLN més avançades com:

  • Xarxes Neuronals Recurrents (RNNs): Adequades per a dades seqüencials.
  • Memòria a Llarg Termini (LSTM): Un tipus de RNN que pot aprendre dependències a llarg termini.
  • Transformers: Models d'avantguarda per a tasques de PLN.

Exemple: Utilitzant LSTM per a la Classificació de Text

# Construir el model LSTM
model = tf.keras.Sequential([
    tf.keras.layers.Embedding(input_dim=100, output_dim=16, input_length=10),
    tf.keras.layers.LSTM(32),
    tf.keras.layers.Dense(16, activació='relu'),
    tf.keras.layers.Dense(1, activació='sigmoid')
])

model.compile(optimitzador='adam', pèrdua='binary_crossentropy', mètriques=['accuracy'])

# Entrenar el model
model.fit(padded_sequences, labels, epochs=10)
