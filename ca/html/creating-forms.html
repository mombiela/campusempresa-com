Els formularis són una part crucial del desenvolupament web, permetent als usuaris interactuar amb el teu lloc web enviant dades. Aquest tema et guiarà des dels conceptes bàsics de la creació de formularis en HTML fins a tècniques més avançades.

Introducció als Formularis

Els formularis en HTML s'utilitzen per recollir dades dels usuaris. Poden contenir diversos tipus d'elements d'entrada com camps de text, caselles de selecció, botons de ràdio, botons d'enviament, i més.

Estructura Bàsica d'un Formulari

Un formulari es defineix utilitzant l'etiqueta <form>. Aquí tens un exemple senzill:

<form action="/submit-form" method="post">
  <label for="name">Nom:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Enviar">
</form>
  • <form>: El contenidor per a tots els elements del formulari.
  • action: La URL on s'enviaran les dades del formulari.
  • method: El mètode HTTP a utilitzar quan s'envien les dades del formulari (GET o POST).
  • <label>: Defineix una etiqueta per a un element d'entrada.
  • <input>: Defineix un camp d'entrada.

Elements del Formulari

Entrada de Text

Els camps d'entrada de text s'utilitzen per recollir dades dels usuaris. Es defineixen utilitzant l'etiqueta <input type="text">.

<label for="email">Correu electrònic:</label>
<input type="email" id="email" name="email">

Entrada de Contrasenya

Els camps de contrasenya són similars als camps de text però oculten l'entrada. Es defineixen utilitzant <input type="password">.

<label for="password">Contrasenya:</label>
<input type="password" id="password" name="password">

Botons de Ràdio

Els botons de ràdio permeten a l'usuari seleccionar una opció d'un conjunt. Es defineixen utilitzant <input type="radio">.

<label for="gender">Gènere:</label>
<input type="radio" id="male" name="gender" value="male">
<label for="male">Home</label>
<input type="radio" id="female" name="gender" value="female">
<label for="female">Dona</label>

Caselles de Selecció

Les caselles de selecció permeten a l'usuari seleccionar múltiples opcions. Es defineixen utilitzant <input type="checkbox">.

<label for="subscribe">Subscriure's al butlletí:</label>
<input type="checkbox" id="subscribe" name="subscribe" value="yes">

Botó d'Enviament

El botó d'enviament envia les dades del formulari al servidor. Es defineix utilitzant <input type="submit">.

<input type="submit" value="Enviar">

Elements Avançats del Formulari

Selecció Desplegable

L'element <select> crea una llista desplegable.

<label for="country">País:</label>
<select id="country" name="country">
  <option value="usa">EUA</option>
  <option value="canada">Canadà</option>
  <option value="mexico">Mèxic</option>
</select>

Àrea de Text

L'element <textarea> permet l'entrada de text en múltiples línies.

<label for="message">Missatge:</label>
<textarea id="message" name="message" rows="4" cols="50"></textarea>

Entrada de Fitxer

L'element <input type="file"> permet a l'usuari carregar fitxers.

<label for="file">Carregar un fitxer:</label>
<input type="file" id="file" name="file">

Validació de Formularis

Validació Bàsica

HTML5 proporciona validació de formularis integrada. Per exemple, pots utilitzar l'atribut required per assegurar-te que un camp no es deixi buit.

<label for="username">Nom d'usuari:</label>
<input type="text" id="username" name="username" required>

Validació Avançada

Per a una validació més complexa, pots utilitzar JavaScript.

<form id="myForm">
  <label for="age">Edat:</label>
  <input type="number" id="age" name="age">
  <input type="submit" value="Enviar">
</form>

<script>
document.getElementById('myForm').addEventListener('submit', function(event) {
  var age = document.getElementById('age').value;
  if (age < 18) {
    alert('Has de tenir almenys 18 anys.');
    event.preventDefault();
  }
});
</script>

Estilització de Formularis

Els formularis es poden estilitzar utilitzant CSS per millorar la seva aparença i usabilitat.

<style>
  form {
    max-width: 400px;
    margin: auto;
    padding: 1em;
    border: 1px solid #ccc;
    border-radius: 1em;
  }
  label {
    margin-top: 1em;
    display: block;
  }
  input, textarea, select {
    width: 100%;
    padding: 0.5em;
    margin-top: 0.5em;
  }
  input[type="submit"] {
    background: #007BFF;
    color: white;
    border: none;
    padding: 0.7em;
    border-radius: 0.3em;
    cursor: pointer;
  }
</style>

Conclusió

Crear formularis en HTML és una habilitat fonamental per als desenvolupadors web. Entenent els diversos elements del formulari i com utilitzar-los, pots crear formularis web interactius i fàcils d'utilitzar. Recorda validar les dades dels usuaris per assegurar la integritat de les dades i millorar l'experiència de l'usuari. Amb pràctica, podràs crear formularis complexos que satisfacin les necessitats de les teves aplicacions web.
