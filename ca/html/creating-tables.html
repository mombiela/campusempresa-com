Introducció a les Taules HTML

Les taules HTML s'utilitzen per mostrar dades en un format tabular, que és essencialment una graella de files i columnes. Les taules són útils per organitzar informació d'una manera fàcil de llegir i entendre.

Estructura Bàsica de la Taula

Per crear una taula en HTML, utilitzes l'element <table>. Dins de l'element <table>, defineixes files utilitzant l'element <tr>, i dins de cada fila, defineixes cel·les utilitzant l'element <td>.

Exemple:

<table>
  <tr>
    <td>Fila 1, Cel·la 1</td>
    <td>Fila 1, Cel·la 2</td>
  </tr>
  <tr>
    <td>Fila 2, Cel·la 1</td>
    <td>Fila 2, Cel·la 2</td>
  </tr>
</table>

Afegint Capçaleres a la Taula

Les capçaleres de la taula es defineixen utilitzant l'element <th>. Les capçaleres s'utilitzen típicament per definir el títol de cada columna.

Exemple:

<table>
  <tr>
    <th>Capçalera 1</th>
    <th>Capçalera 2</th>
  </tr>
  <tr>
    <td>Fila 1, Cel·la 1</td>
    <td>Fila 1, Cel·la 2</td>
  </tr>
  <tr>
    <td>Fila 2, Cel·la 1</td>
    <td>Fila 2, Cel·la 2</td>
  </tr>
</table>

Fusionant Files i Columnes

Pots fer que una cel·la abarqui múltiples files o columnes utilitzant els atributs rowspan i colspan.

Exemple:

<table>
  <tr>
    <th>Capçalera 1</th>
    <th>Capçalera 2</th>
    <th>Capçalera 3</th>
  </tr>
  <tr>
    <td rowspan="2">Fila 1, Cel·la 1 (abarca 2 files)</td>
    <td>Fila 1, Cel·la 2</td>
    <td>Fila 1, Cel·la 3</td>
  </tr>
  <tr>
    <td>Fila 2, Cel·la 2</td>
    <td>Fila 2, Cel·la 3</td>
  </tr>
  <tr>
    <td>Fila 3, Cel·la 1</td>
    <td colspan="2">Fila 3, Cel·la 2 (abarca 2 columnes)</td>
  </tr>
</table>

Estilitzant Taules

Les taules es poden estilitzar utilitzant CSS per millorar-ne l'aparença. Pots afegir vores, padding i altres estils per fer la taula més llegible.

Exemple:

<style>
  table {
    width: 100%;
    border-collapse: collapse;
  }
  th, td {
    border: 1px solid black;
    padding: 8px;
    text-align: left;
  }
  th {
    background-color: #f2f2f2;
  }
</style>

<table>
  <tr>
    <th>Capçalera 1</th>
    <th>Capçalera 2</th>
    <th>Capçalera 3</th>
  </tr>
  <tr>
    <td>Fila 1, Cel·la 1</td>
    <td>Fila 1, Cel·la 2</td>
    <td>Fila 1, Cel·la 3</td>
  </tr>
  <tr>
    <td>Fila 2, Cel·la 1</td>
    <td>Fila 2, Cel·la 2</td>
    <td>Fila 2, Cel·la 3</td>
  </tr>
</table>

Funcionalitats Avançades de les Taules

Per a funcionalitats més avançades de les taules, pots utilitzar elements i atributs addicionals d'HTML.

Títol de la Taula

L'element <caption> s'utilitza per afegir un títol a la taula.

Exemple:

<table>
  <caption>Dades de Vendes Mensuals</caption>
  <tr>
    <th>Mes</th>
    <th>Vendes</th>
  </tr>
  <tr>
    <td>Gener</td>
    <td>$1000</td>
  </tr>
  <tr>
    <td>Febrer</td>
    <td>$1200</td>
  </tr>
</table>

Agrupant Files de la Taula

Pots agrupar files utilitzant els elements <thead>, <tbody> i <tfoot>.

Exemple:

<table>
  <thead>
    <tr>
      <th>Capçalera 1</th>
      <th>Capçalera 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Fila 1, Cel·la 1</td>
      <td>Fila 1, Cel·la 2</td>
    </tr>
    <tr>
      <td>Fila 2, Cel·la 1</td>
      <td>Fila 2, Cel·la 2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Peu 1</td>
      <td>Peu 2</td>
    </tr>
  </tfoot>
</table>

Conclusió

Crear taules en HTML és una habilitat fonamental per organitzar i mostrar dades. Entenent l'estructura bàsica de les taules, afegint capçaleres, fusionant files i columnes, i aplicant estils, pots crear taules ben organitzades i visualment atractives. Les funcionalitats avançades com els títols i l'agrupació de files milloren encara més la funcionalitat i llegibilitat de les teves taules. Practica creant diferents tipus de taules per esdevenir expert en aquesta habilitat essencial d'HTML.
