En aquesta secció, explorarem com crear i manipular files i cel·les de taula en HTML. Les taules són una part fonamental del desenvolupament web, ja que et permeten organitzar dades en un format estructurat. Cobrirem les etiquetes bàsiques utilitzades per crear taules, files i cel·les, i proporcionarem exemples i exercicis per consolidar la teva comprensió.

Conceptes clau

  • Taula (<table>): L'element contenidor per a una taula.
  • Fila de taula (<tr>): Defineix una fila en una taula.
  • Capçalera de taula (<th>): Defineix una cel·la de capçalera en una taula.
  • Dada de taula (<td>): Defineix una cel·la estàndard en una taula.

Creació d'una taula bàsica

Per crear una taula, necessites utilitzar l'element <table>. Dins d'aquest element, pots definir files utilitzant l'element <tr>. Cada fila pot contenir cel·les de capçalera (<th>) i cel·les de dades (<td>).

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Taula bàsica</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Capçalera 1</th>
            <th>Capçalera 2</th>
            <th>Capçalera 3</th>
        </tr>
        <tr>
            <td>Dada 1</td>
            <td>Dada 2</td>
            <td>Dada 3</td>
        </tr>
        <tr>
            <td>Dada 4</td>
            <td>Dada 5</td>
            <td>Dada 6</td>
        </tr>
    </table>
</body>
</html>

Explicació

  • <table border="1">: Crea una taula amb una vora.
  • <tr>: Defineix una fila de taula.
  • <th>: Defineix una cel·la de capçalera, que és en negreta i centrada per defecte.
  • <td>: Defineix una cel·la estàndard.

Afegir més files i cel·les

Pots afegir tantes files i cel·les com necessitis. Cada fila (<tr>) pot contenir múltiples cel·les de capçalera (<th>) o de dades (<td>).

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Taula ampliada</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Capçalera 1</th>
            <th>Capçalera 2</th>
            <th>Capçalera 3</th>
        </tr>
        <tr>
            <td>Dada 1</td>
            <td>Dada 2</td>
            <td>Dada 3</td>
        </tr>
        <tr>
            <td>Dada 4</td>
            <td>Dada 5</td>
            <td>Dada 6</td>
        </tr>
        <tr>
            <td>Dada 7</td>
            <td>Dada 8</td>
            <td>Dada 9</td>
        </tr>
    </table>
</body>
</html>

Fusionar cel·les

Pots fusionar cel·les horitzontalment o verticalment utilitzant els atributs colspan i rowspan.

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Cel·les fusionades</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Capçalera 1</th>
            <th>Capçalera 2</th>
            <th>Capçalera 3</th>
        </tr>
        <tr>
            <td colspan="2">Cel·la fusionada 1</td>
            <td>Dada 3</td>
        </tr>
        <tr>
            <td>Dada 4</td>
            <td rowspan="2">Cel·la fusionada 2</td>
            <td>Dada 6</td>
        </tr>
        <tr>
            <td>Dada 7</td>
            <td>Dada 9</td>
        </tr>
    </table>
</body>
</html>

Explicació

  • colspan="2": Fusiona dues cel·les horitzontalment.
  • rowspan="2": Fusiona dues cel·les verticalment.

Estilitzar taules

Les taules es poden estilitzar utilitzant CSS per millorar-ne l'aparença.

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Taula estilitzada</title>
    <style>
        table {
            width: 100%;
            border-collapse: collapse;
        }
        th, td {
            border: 1px solid black;
            padding: 8px;
            text-align: left;
        }
        th {
            background-color: #f2f2f2;
        }
    </style>
</head>
<body>
    <table>
        <tr>
            <th>Capçalera 1</th>
            <th>Capçalera 2</th>
            <th>Capçalera 3</th>
        </tr>
        <tr>
            <td>Dada 1</td>
            <td>Dada 2</td>
            <td>Dada 3</td>
        </tr>
        <tr>
            <td>Dada 4</td>
            <td>Dada 5</td>
            <td>Dada 6</td>
        </tr>
    </table>
</body>
</html>

Explicació

  • border-collapse: collapse;: Col·lapsa les vores en una sola vora.
  • padding: 8px;: Afegeix farciment dins de les cel·les.
  • background-color: #f2f2f2;: Estableix el color de fons de les cel·les de capçalera.

Conclusió

En aquesta secció, hem cobert les bases de la creació de taules en HTML, incloent com definir files i cel·les, fusionar cel·les i aplicar estil bàsic. Les taules són una eina poderosa per organitzar dades, i entendre com utilitzar-les efectivament és essencial per a qualsevol desenvolupador web. Practica creant les teves pròpies taules i experimenta amb diferents estils per sentir-te més còmode amb aquests conceptes.
