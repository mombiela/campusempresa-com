Introducció

Benvingut al teu viatge en HTML! En aquesta secció, et guiarem a través del procés de crear la teva primera pàgina HTML. Al final d'aquesta lliçó, tindràs una comprensió bàsica de l'estructura HTML i seràs capaç de crear una pàgina web senzilla.

Conceptes clau

  • HTML (HyperText Markup Language): El llenguatge estàndard per crear pàgines web.
  • Etiquetes: HTML utilitza etiquetes per estructurar el contingut. Les etiquetes estan tancades entre claudàtors angulars, per exemple, <tagname>.
  • Elements: Un element consisteix en una etiqueta d'inici, contingut i una etiqueta de tancament, per exemple, <p>Això és un paràgraf.</p>.
  • Atributs: Les etiquetes poden tenir atributs que proporcionen informació addicional sobre els elements, per exemple, <a href="https://example.com">Enllaç</a>.

Estructura bàsica d'un document HTML

Cada document HTML té una estructura bàsica que inclou els següents elements:

<!DOCTYPE html>
<html>
<head>
    <title>La meva primera pàgina HTML</title>
</head>
<body>
    <h1>Benvingut a la meva primera pàgina HTML</h1>
    <p>Això és un paràgraf de text a la meva primera pàgina HTML.</p>
</body>
</html>

Explicació del codi

  • <!DOCTYPE html>: Aquesta declaració defineix el tipus de document i la versió d'HTML. Ajuda el navegador a entendre com renderitzar la pàgina.
  • <html>: L'element arrel que conté tots els altres elements HTML.
  • <head>: Conté metainformació sobre el document, com el títol i enllaços a fulls d'estil.
  • <title>: Defineix el títol de la pàgina web, que apareix a la pestanya del navegador.
  • <body>: Conté el contingut del document HTML, com text, imatges i enllaços.
  • <h1>: Una etiqueta de capçalera. Hi ha sis nivells de capçaleres, des de <h1> (més important) fins a <h6> (menys important).
  • <p>: Una etiqueta de paràgraf utilitzada per definir un bloc de text.

Creant la teva primera pàgina HTML

Segueix aquests passos per crear la teva primera pàgina HTML:

  1. Obre un editor de text:

    • Pots utilitzar qualsevol editor de text, com Notepad (Windows), TextEdit (Mac), o un editor de codi com Visual Studio Code o Sublime Text.

  2. Escriu el codi HTML:

    • Copia l'estructura bàsica proporcionada anteriorment al teu editor de text.

  3. Guarda el fitxer:

    • Guarda el fitxer amb una extensió .html, per exemple, index.html.

  4. Obre el fitxer HTML en un navegador:

    • Fes doble clic al fitxer HTML guardat o obre'l en un navegador web (Chrome, Firefox, Safari, etc.).

Exemple: Afegint més contingut

Afegim més contingut a la nostra pàgina HTML per fer-la més interessant.

<!DOCTYPE html>
<html>
<head>
    <title>La meva primera pàgina HTML</title>
</head>
<body>
    <h1>Benvingut a la meva primera pàgina HTML</h1>
    <p>Això és un paràgraf de text a la meva primera pàgina HTML.</p>
    <h2>Sobre mi</h2>
    <p>El meu nom és John Doe, i estic aprenent HTML.</p>
    <ul>
        <li>M'encanta programar.</li>
        <li>Gaudeixo aprenent noves tecnologies.</li>
        <li>Estic emocionat per construir la meva primera pàgina web.</li>
    </ul>
    <a href="https://example.com">Visita Example.com</a>
</body>
</html>

Explicació del codi addicional

  • <h2>: Una capçalera de segon nivell.
  • <ul>: Una llista desordenada.
  • <li>: Un element de llista dins d'una llista desordenada.
  • <a href="URL">: Una etiqueta d'ancoratge utilitzada per crear hipervincles. L'atribut href especifica la URL de la pàgina a la qual enllaça.

Conclusió

Felicitats! Has creat la teva primera pàgina HTML. Ara entens l'estructura bàsica d'un document HTML i com afegir capçaleres, paràgrafs, llistes i enllaços. Practica creant més pàgines HTML per sentir-te més còmode amb aquests conceptes. En les properes lliçons, aprofundirem en els elements i atributs HTML per millorar encara més les teves pàgines web. Feliç programació!
