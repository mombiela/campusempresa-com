Introducció als atributs HTML

Els atributs HTML proporcionen informació addicional sobre els elements HTML. Sempre s'inclouen a l'etiqueta d'obertura i normalment vénen en parelles nom/valor com name="value".

Conceptes clau

  • Els atributs sempre es especifiquen a l'etiqueta d'inici.
  • Els atributs normalment vénen en parelles nom/valor com name="value".
  • Els atributs comuns inclouen id, class, style, title i href.

Atributs HTML comuns

L'atribut `id`

L'atribut id especifica un id únic per a un element HTML. El valor de l'atribut id ha de ser únic dins del document HTML.

<p id="unique-paragraph">Aquest és un paràgraf amb un id únic.</p>

L'atribut `class`

L'atribut class s'utilitza per definir una classe per als elements HTML. Diversos elements poden compartir la mateixa classe.

<p class="intro">Aquest és un paràgraf introductori.</p>
<p class="intro">Aquest és un altre paràgraf introductori.</p>

L'atribut `style`

L'atribut style s'utilitza per afegir CSS en línia a un element.

<p style="color: blue; font-size: 20px;">Aquest és un paràgraf estilitzat.</p>

L'atribut `title`

L'atribut title proporciona informació addicional sobre un element. La informació sovint es mostra com una eina de suggeriment quan el ratolí es mou sobre l'element.

<p title="Aquesta és una eina de suggeriment">Passa el ratolí sobre aquest paràgraf per veure l'eina de suggeriment.</p>

L'atribut `href`

L'atribut href especifica l'URL de la pàgina a la qual enllaça.

<a href="https://www.example.com">Visita Example.com</a>

Atributs HTML avançats

Els atributs `data-*`

Els atributs data-* s'utilitzen per emmagatzemar dades personalitzades privades per a la pàgina o aplicació.

<div data-user-id="12345" data-role="admin">Informació de l'usuari</div>

L'atribut `src`

L'atribut src especifica l'URL d'un recurs extern, més comunament utilitzat amb imatges, scripts i iframes.

<img src="image.jpg" alt="Descripció de la imatge">

L'atribut `alt`

L'atribut alt proporciona text alternatiu per a una imatge si la imatge no es pot mostrar.

<img src="image.jpg" alt="Descripció de la imatge">

L'atribut `disabled`

L'atribut disabled es pot aplicar als elements de formulari per desactivar-los.

<input type="text" disabled>

L'atribut `readonly`

L'atribut readonly es pot aplicar als camps d'entrada per fer-los només de lectura.

<input type="text" value="No pots canviar això" readonly>

Comparació d'atributs comuns vs. avançats

| Atribut | Descripció | Comú/Avançat | |------------|--------------------------------------------------|-----------------| | id | Identificador únic per a un element | Comú | | class | Nom(s) de classe per a un element | Comú | | style | Estils CSS en línia | Comú | | title | Text de l'eina de suggeriment | Comú | | href | URL per a un enllaç | Comú | | data-* | Atributs de dades personalitzades | Avançat | | src | URL d'un recurs extern | Avançat | | alt | Text alternatiu per a una imatge | Avançat | | disabled | Desactiva un element de formulari | Avançat | | readonly | Fa que un camp d'entrada sigui només de lectura | Avançat |

Conclusió

Els atributs HTML són essencials per afegir informació i funcionalitat addicional als elements HTML. Entendre tant els atributs comuns com els avançats et permet crear pàgines web més dinàmiques i interactives. Practica utilitzant aquests atributs en els teus documents HTML per esdevenir més proficient en el desenvolupament web.
