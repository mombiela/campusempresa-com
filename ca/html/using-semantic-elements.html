Introducció als elements semàntics

Els elements semàntics en HTML són aquells que descriuen clarament el seu significat d'una manera llegible per humans i màquines. Aquests elements no només ajuden amb l'estructura i la llegibilitat del codi, sinó que també milloren l'accessibilitat i el SEO.

Conceptes clau

  • Elements semàntics: Elements que proporcionen significat al contingut que emboliquen.
  • Elements no semàntics: Elements que no proporcionen cap informació sobre el contingut que contenen.

Exemples d'elements semàntics

  • <header>: Defineix una capçalera per a un document o secció.
  • <nav>: Defineix un conjunt d'enllaços de navegació.
  • <section>: Defineix una secció en un document.
  • <article>: Defineix un contingut independent i autònom.
  • <aside>: Defineix contingut al marge del contingut en què es troba.
  • <footer>: Defineix un peu de pàgina per a un document o secció.

Exemples d'elements no semàntics

  • <div>: Defineix una divisió o una secció en un document HTML.
  • <span>: Defineix una secció en un document.

Per què utilitzar elements semàntics?

  • Millora de l'accessibilitat: Els lectors de pantalla i altres tecnologies assistives poden entendre i navegar millor pel contingut.
  • Millor SEO: Els motors de cerca poden entendre millor l'estructura i el contingut de les teves pàgines web.
  • Manteniment més fàcil: El codi és més llegible i més fàcil de mantenir.

Ús bàsic dels elements semàntics

Exemple: Creació d'una estructura de pàgina web senzilla

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pàgina Web Senzilla</title>
</head>
<body>
    <header>
        <h1>Benvingut al Meu Lloc Web</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#home">Inici</a></li>
            <li><a href="#about">Sobre</a></li>
            <li><a href="#contact">Contacte</a></li>
        </ul>
    </nav>
    <section id="home">
        <h2>Inici</h2>
        <p>Aquesta és la secció d'inici.</p>
    </section>
    <section id="about">
        <h2>Sobre</h2>
        <p>Aquesta és la secció sobre.</p>
    </section>
    <aside>
        <h2>Enllaços Relacionats</h2>
        <ul>
            <li><a href="#link1">Enllaç 1</a></li>
            <li><a href="#link2">Enllaç 2</a></li>
        </ul>
    </aside>
    <footer>
        <p>&copy; 2023 El Meu Lloc Web</p>
    </footer>
</body>
</html>
  • Explicació:
    • <header> conté el títol principal de la pàgina.
    • <nav> conté els enllaços de navegació.
    • Els elements <section> s'utilitzen per definir diferents seccions de la pàgina.
    • <aside> conté enllaços relacionats que no formen part del contingut principal.
    • <footer> conté la informació del peu de pàgina.

Ús avançat dels elements semàntics

Exemple: Elements semàntics niats

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Pàgina Web Avançada</title>
</head>
<body>
    <header>
        <h1>Lloc Web Avançat</h1>
        <nav>
            <ul>
                <li><a href="#home">Inici</a></li>
                <li><a href="#services">Serveis</a></li>
                <li><a href="#contact">Contacte</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <article>
            <header>
                <h2>Títol de l'Article</h2>
                <p>Publicat el <time datetime="2023-10-01">1 d'octubre de 2023</time></p>
            </header>
            <section>
                <h3>Introducció</h3>
                <p>Aquesta és la introducció de l'article.</p>
            </section>
            <section>
                <h3>Contingut Principal</h3>
                <p>Aquest és el contingut principal de l'article.</p>
            </section>
            <footer>
                <p>Autor: John Doe</p>
            </footer>
        </article>
    </main>
    <footer>
        <p>&copy; 2023 Lloc Web Avançat</p>
    </footer>
</body>
</html>
  • Explicació:
    • L'element <main> s'utilitza per encapsular el contingut principal del document.
    • L'element <article> conté una peça de contingut independent.
    • Els elements <header> i <footer> niats dins de l'<article> proporcionen estructura i metadades addicionals.

Conclusió

Utilitzar elements semàntics en HTML és crucial per crear pàgines web ben estructurades, accessibles i amigables amb el SEO. Entenent i implementant aquests elements, pots millorar significativament la qualitat i mantenibilitat dels teus projectes web.
