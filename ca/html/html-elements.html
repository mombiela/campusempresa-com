Introducció als elements HTML

HTML (HyperText Markup Language) és el llenguatge estàndard per crear pàgines web. Els elements HTML són els blocs de construcció de les pàgines HTML. Estan representats per etiquetes i poden contenir text, atributs, altres elements, o ser auto-tancats.

Conceptes clau

  • Etiquetes: Els elements HTML es defineixen per etiquetes, que estan tancades entre claudàtors angulats (< >).
  • Atributs: Les etiquetes poden tenir atributs que proporcionen informació addicional sobre l'element.
  • Niuament: Els elements HTML poden estar niuats dins d'altres elements.
  • Etiquetes auto-tancades: Alguns elements HTML no tenen una etiqueta de tancament.

Elements HTML bàsics

Paràgrafs i encapçalaments

  • Paràgraf (<p>): S'utilitza per definir un bloc de text.
  • Encapçalaments (<h1> a <h6>): S'utilitzen per definir encapçalaments, amb <h1> sent el nivell més alt i <h6> el més baix.
<h1>Això és un Encapçalament 1</h1>
<p>Això és un paràgraf.</p>
<h2>Això és un Encapçalament 2</h2>
<p>Això és un altre paràgraf.</p>

Llistes

  • Llista ordenada (<ol>): Crea una llista numerada.
  • Llista desordenada (<ul>): Crea una llista amb vinyetes.
  • Element de llista (<li>): Defineix un element en una llista.
<ol>
  <li>Primer element</li>
  <li>Segon element</li>
</ol>
<ul>
  <li>Primer element</li>
  <li>Segon element</li>
</ul>

Enllaços i imatges

  • Enllaç (<a>): Crea un hipervincle.
  • Imatge (<img>): Insereix una imatge.
<a href="https://www.example.com">Visita Example.com</a>
<img src="image.jpg" alt="Descripció de la imatge">

Taules

Les taules s'utilitzen per mostrar dades en un format tabular.

  • Taula (<table>): Defineix una taula.
  • Fila de taula (<tr>): Defineix una fila en una taula.
  • Capçalera de taula (<th>): Defineix una cel·la de capçalera en una taula.
  • Dada de taula (<td>): Defineix una cel·la estàndard en una taula.
<table>
  <tr>
    <th>Capçalera 1</th>
    <th>Capçalera 2</th>
  </tr>
  <tr>
    <td>Dada 1</td>
    <td>Dada 2</td>
  </tr>
</table>

Elements HTML intermedis

Formularis

Els formularis s'utilitzen per recollir dades de l'usuari.

  • Formulari (<form>): Defineix un formulari.
  • Entrada (<input>): Defineix un camp d'entrada.
  • Etiqueta (<label>): Defineix una etiqueta per a un element d'entrada.
  • Botó (<button>): Defineix un botó clicable.
<form action="/submit" method="post">
  <label for="name">Nom:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Enviar</button>
</form>

Elements semàntics

Els elements semàntics proporcionen significat a l'estructura de la pàgina web.

  • Capçalera (<header>): Defineix una capçalera per a un document o secció.
  • Peu de pàgina (<footer>): Defineix un peu de pàgina per a un document o secció.
  • Article (<article>): Defineix un contingut independent i autònom.
  • Secció (<section>): Defineix una secció en un document.
  • Navegació (<nav>): Defineix enllaços de navegació.
<header>
  <h1>Títol del lloc web</h1>
</header>
<nav>
  <a href="#home">Inici</a>
  <a href="#about">Sobre nosaltres</a>
</nav>
<section>
  <article>
    <h2>Títol de l'article</h2>
    <p>El contingut de l'article va aquí.</p>
  </article>
</section>
<footer>
  <p>El contingut del peu de pàgina va aquí.</p>
</footer>

Elements HTML avançats

Elements multimèdia

  • Àudio (<audio>): Insereix contingut d'àudio.
  • Vídeo (<video>): Insereix contingut de vídeo.
<audio controls>
  <source src="audiofile.mp3" type="audio/mpeg">
  El teu navegador no suporta l'element d'àudio.
</audio>
<video width="320" height="240" controls>
  <source src="videofile.mp4" type="video/mp4">
  El teu navegador no suporta l'etiqueta de vídeo.
</video>

Canvas i SVG

  • Canvas (<canvas>): S'utilitza per dibuixar gràfics mitjançant scripts (normalment JavaScript).
  • SVG (<svg>): S'utilitza per definir gràfics vectorials.
<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;">
  El teu navegador no suporta l'etiqueta canvas d'HTML.
</canvas>
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Conclusió

Els elements HTML són els components fonamentals de les pàgines web. Entendre com utilitzar aquests elements de manera efectiva és crucial per crear pàgines web ben estructurades i funcionals. Des dels elements bàsics com paràgrafs i encapçalaments fins als elements avançats com multimèdia i gràfics, dominar els elements HTML és el primer pas per convertir-se en un desenvolupador web competent.
