Introducció a l'Emmagatzematge Web

L'Emmagatzematge Web és una característica d'HTML5 que permet a les aplicacions web emmagatzemar dades localment dins del navegador de l'usuari. Això és útil per guardar preferències d'usuari, mantenir informació de sessió i més, sense la necessitat d'emmagatzematge al servidor.

Conceptes Clau

  • Emmagatzematge Local: Emmagatzema dades sense data de caducitat. Les dades persisteixen fins i tot quan el navegador es tanca i es torna a obrir.
  • Emmagatzematge de Sessió: Emmagatzema dades durant la durada de la sessió de la pàgina. Les dades es netegen quan la sessió de la pàgina acaba (és a dir, quan es tanca la pestanya del navegador).

Emmagatzematge Local

L'Emmagatzematge Local proporciona una manera d'emmagatzemar parells clau-valor en un navegador web sense data de caducitat.

Operacions Bàsiques

  • Set Item: Emmagatzemar dades.
  • Get Item: Recuperar dades.
  • Remove Item: Eliminar dades.
  • Clear: Netejar totes les dades emmagatzemades.

Exemples de Codi

Emmagatzemar un Element

// Emmagatzemar dades en l'emmagatzematge local
localStorage.setItem('username', 'JohnDoe');

Recuperar un Element

// Recuperar dades de l'emmagatzematge local
let username = localStorage.getItem('username');
console.log(username); // Sortida: JohnDoe

Eliminar un Element

// Eliminar dades de l'emmagatzematge local
localStorage.removeItem('username');

Netejar Tots els Elements

// Netejar totes les dades de l'emmagatzematge local
localStorage.clear();

Exemple Pràctic

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Emmagatzematge Local</title>
</head>
<body>
    <input type="text" id="nameInput" placeholder="Introdueix el teu nom">
    <button onclick="saveName()">Guardar Nom</button>
    <button onclick="showName()">Mostrar Nom</button>
    <p id="displayName"></p>

    <script>
        function saveName() {
            let name = document.getElementById('nameInput').value;
            localStorage.setItem('name', name);
        }

        function showName() {
            let name = localStorage.getItem('name');
            document.getElementById('displayName').innerText = name ? `Hola, ${name}!` : 'No s'ha trobat cap nom.';
        }
    </script>
</body>
</html>

Emmagatzematge de Sessió

L'Emmagatzematge de Sessió funciona de manera similar a l'Emmagatzematge Local però amb una vida útil més curta. Les dades només estan disponibles durant la durada de la sessió de la pàgina.

Operacions Bàsiques

  • Set Item: Emmagatzemar dades.
  • Get Item: Recuperar dades.
  • Remove Item: Eliminar dades.
  • Clear: Netejar totes les dades emmagatzemades.

Exemples de Codi

Emmagatzemar un Element

// Emmagatzemar dades en l'emmagatzematge de sessió
sessionStorage.setItem('sessionUser', 'JaneDoe');

Recuperar un Element

// Recuperar dades de l'emmagatzematge de sessió
let sessionUser = sessionStorage.getItem('sessionUser');
console.log(sessionUser); // Sortida: JaneDoe

Eliminar un Element

// Eliminar dades de l'emmagatzematge de sessió
sessionStorage.removeItem('sessionUser');

Netejar Tots els Elements

// Netejar totes les dades de l'emmagatzematge de sessió
sessionStorage.clear();

Exemple Pràctic

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Emmagatzematge de Sessió</title>
</head>
<body>
    <input type="text" id="sessionInput" placeholder="Introdueix dades de sessió">
    <button onclick="saveSessionData()">Guardar Dades de Sessió</button>
    <button onclick="showSessionData()">Mostrar Dades de Sessió</button>
    <p id="displaySessionData"></p>

    <script>
        function saveSessionData() {
            let data = document.getElementById('sessionInput').value;
            sessionStorage.setItem('sessionData', data);
        }

        function showSessionData() {
            let data = sessionStorage.getItem('sessionData');
            document.getElementById('displaySessionData').innerText = data ? `Dades de Sessió: ${data}` : 'No s'han trobat dades de sessió.';
        }
    </script>
</body>
</html>

Comparació entre Emmagatzematge Local i Emmagatzematge de Sessió

| Característica | Emmagatzematge Local | Emmagatzematge de Sessió | |-------------------|----------------------------------|----------------------------------| | Vida útil | Sense caducitat | Durada de la sessió de la pàgina | | Persistència de Dades | Persistent entre sessions del navegador | Netejat quan es tanca la pestanya | | Capacitat d'Emmagatzematge | Normalment 5-10 MB per origen | Normalment 5-10 MB per origen |

Conclusió

L'Emmagatzematge Web proporciona una manera poderosa d'emmagatzemar dades al costat del client, oferint tant Emmagatzematge Local per a dades persistents com Emmagatzematge de Sessió per a dades temporals. Entendre com utilitzar aquests mecanismes d'emmagatzematge pot millorar enormement la funcionalitat i l'experiència d'usuari de les aplicacions web.
