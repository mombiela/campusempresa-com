Introducció

Els formularis HTML són essencials per recollir dades dels usuaris a les pàgines web. Entendre els atributs dels formularis és crucial per crear formularis funcionals i fàcils d'usar. Aquesta secció cobrirà els diversos atributs que es poden aplicar als formularis HTML, des de nivells bàsics fins a avançats.

Atributs Bàsics de Formulari

`action`

  • Definició: Especifica la URL on s'enviaran les dades del formulari després de l'enviament.
  • Exemple: 
    <form action="submit_form.php">
    <!-- elements del formulari -->
</form>

`method`

  • Definició: Defineix el mètode HTTP que s'utilitzarà quan s'enviïn les dades del formulari. Els valors comuns són GET i POST.
  • Exemple: 
    <form action="submit_form.php" method="post">
    <!-- elements del formulari -->
</form>

`enctype`

  • Definició: Especifica com s'han d'encaixar les dades del formulari quan s'enviïn. Els valors comuns són application/x-www-form-urlencoded, multipart/form-data, i text/plain.
  • Exemple: 
    <form action="submit_form.php" method="post" enctype="multipart/form-data">
    <!-- elements del formulari -->
</form>

`name`

  • Definició: Assigna un nom al formulari, que pot ser útil per a la programació i la identificació del formulari.
  • Exemple: 
    <form name="userForm">
    <!-- elements del formulari -->
</form>

`target`

  • Definició: Especifica on mostrar la resposta després d'enviar el formulari. Els valors comuns són _self, _blank, _parent, i _top.
  • Exemple: 
    <form action="submit_form.php" target="_blank">
    <!-- elements del formulari -->
</form>

`autocomplete`

  • Definició: Activa o desactiva l'autocompletar per al formulari. Els valors poden ser on o off.
  • Exemple: 
    <form action="submit_form.php" autocomplete="off">
    <!-- elements del formulari -->
</form>

Atributs Intermedis de Formulari

`novalidate`

  • Definició: Quan està present, desactiva la validació predeterminada del navegador per al formulari.
  • Exemple: 
    <form action="submit_form.php" novalidate>
    <!-- elements del formulari -->
</form>

`accept-charset`

  • Definició: Especifica les codificacions de caràcters que accepta el servidor.
  • Exemple: 
    <form action="submit_form.php" accept-charset="UTF-8">
    <!-- elements del formulari -->
</form>

`rel`

  • Definició: Especifica la relació entre el document actual i el destí del formulari. S'utilitza comunament amb target="_blank".
  • Exemple: 
    <form action="submit_form.php" target="_blank" rel="noopener noreferrer">
    <!-- elements del formulari -->
</form>

Atributs Avançats de Formulari

`onsubmit`

  • Definició: Especifica una funció JavaScript que s'executarà quan s'enviï el formulari.
  • Exemple: 
    <form action="submit_form.php" onsubmit="return validateForm()">
    <!-- elements del formulari -->
</form>
<script>
    function validateForm() {
        // Lògica de validació personalitzada
        return true; // Retorna fals per evitar l'enviament del formulari
    }
</script>

`form`

  • Definició: Associa l'element amb un element de formulari. Això és útil per a elements de formulari que no estan dins de l'etiqueta del formulari.
  • Exemple: 
    <form id="myForm" action="submit_form.php">
    <!-- elements del formulari -->
</form>
<input type="text" form="myForm" name="username">

`formaction`

  • Definició: Sobreescriu l'atribut action del formulari per a un element d'entrada específic.
  • Exemple: 
    <form action="submit_form.php" method="post">
    <input type="submit" value="Enviar per Defecte">
    <input type="submit" formaction="submit_another_form.php" value="Enviar a un Altre">
</form>

`formenctype`

  • Definició: Sobreescriu l'atribut enctype del formulari per a un element d'entrada específic.
  • Exemple: 
    <form action="submit_form.php" method="post">
    <input type="file" name="fileUpload">
    <input type="submit" formenctype="multipart/form-data" value="Pujar Fitxer">
</form>

`formmethod`

  • Definició: Sobreescriu l'atribut method del formulari per a un element d'entrada específic.
  • Exemple: 
    <form action="submit_form.php" method="post">
    <input type="submit" value="Enviar POST">
    <input type="submit" formmethod="get" value="Enviar GET">
</form>

`formtarget`

  • Definició: Sobreescriu l'atribut target del formulari per a un element d'entrada específic.
  • Exemple: 
    <form action="submit_form.php" method="post">
    <input type="submit" value="Enviar Aquí">
    <input type="submit" formtarget="_blank" value="Enviar en una Nova Pestaña">
</form>

Conclusió

Entendre i utilitzar els atributs dels formularis en HTML és essencial per crear formularis robustos i fàcils d'usar. Des dels atributs bàsics com action i method fins als més avançats com onsubmit i formtarget, cada atribut serveix a un propòsit específic que millora la funcionalitat i usabilitat dels formularis. En dominar aquests atributs, pots assegurar-te que els teus formularis siguin funcionals i segurs, proporcionant una millor experiència d'usuari.
