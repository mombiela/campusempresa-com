Introducció

Els formularis són una part crucial del desenvolupament web, permetent als usuaris interactuar amb els llocs web enviant dades. En aquesta secció, explorarem diversos elements de formulari HTML, els seus atributs i com utilitzar-los de manera efectiva.

Estructura Bàsica del Formulari

Un formulari en HTML es defineix utilitzant l'etiqueta <form>. Aquí teniu un exemple senzill:

<form action="/submit" method="post">
  <!-- Els elements del formulari van aquí -->
</form>
  • action: Especifica la URL on s'han d'enviar les dades del formulari.
  • method: Especifica el mètode HTTP a utilitzar quan s'envien les dades del formulari (GET o POST).

Elements Comuns del Formulari

Entrada de Text

L'element <input> amb type="text" s'utilitza per a l'entrada de text d'una sola línia.

<form action="/submit" method="post">
  <label for="name">Nom:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Enviar">
</form>
  • label: Associa una etiqueta de text amb un element del formulari.
  • id: Identificador únic per a l'element del formulari.
  • name: Nom de l'element del formulari, utilitzat per identificar les dades quan s'envien.
  • type: Especifica el tipus d'entrada (text en aquest cas).

Entrada de Contrasenya

L'element <input> amb type="password" s'utilitza per a camps de contrasenya.

<form action="/submit" method="post">
  <label for="password">Contrasenya:</label>
  <input type="password" id="password" name="password">
  <input type="submit" value="Enviar">
</form>

Botons de Ràdio

Els botons de ràdio permeten als usuaris seleccionar una opció d'un conjunt.

<form action="/submit" method="post">
  <label for="gender">Gènere:</label>
  <input type="radio" id="male" name="gender" value="male">
  <label for="male">Home</label>
  <input type="radio" id="female" name="gender" value="female">
  <label for="female">Dona</label>
  <input type="submit" value="Enviar">
</form>

Caselles de Verificació

Les caselles de verificació permeten als usuaris seleccionar múltiples opcions.

<form action="/submit" method="post">
  <label for="hobbies">Aficions:</label>
  <input type="checkbox" id="reading" name="hobbies" value="reading">
  <label for="reading">Llegir</label>
  <input type="checkbox" id="traveling" name="hobbies" value="traveling">
  <label for="traveling">Viatjar</label>
  <input type="submit" value="Enviar">
</form>

Llista Desplegable

L'element <select> crea una llista desplegable.

<form action="/submit" method="post">
  <label for="country">País:</label>
  <select id="country" name="country">
    <option value="usa">EUA</option>
    <option value="canada">Canadà</option>
    <option value="mexico">Mèxic</option>
  </select>
  <input type="submit" value="Enviar">
</form>

Àrea de Text

L'element <textarea> s'utilitza per a l'entrada de text de múltiples línies.

<form action="/submit" method="post">
  <label for="message">Missatge:</label>
  <textarea id="message" name="message" rows="4" cols="50"></textarea>
  <input type="submit" value="Enviar">
</form>

Botons

Els botons es poden crear utilitzant l'element <button> o l'element <input> amb type="submit", type="reset" o type="button".

<form action="/submit" method="post">
  <button type="submit">Enviar</button>
  <button type="reset">Restablir</button>
</form>

Elements Avançats del Formulari

Entrada de Fitxer

L'element <input> amb type="file" permet als usuaris carregar fitxers.

<form action="/upload" method="post" enctype="multipart/form-data">
  <label for="file">Carregar un fitxer:</label>
  <input type="file" id="file" name="file">
  <input type="submit" value="Carregar">
</form>
  • enctype: Especifica com s'han d'encodear les dades del formulari quan s'envien al servidor. Per a la càrrega de fitxers, utilitzeu multipart/form-data.

Entrada de Data i Hora

HTML5 va introduir nous tipus d'entrada per a data i hora.

<form action="/submit" method="post">
  <label for="birthday">Aniversari:</label>
  <input type="date" id="birthday" name="birthday">
  <label for="appointment">Cita:</label>
  <input type="datetime-local" id="appointment" name="appointment">
  <input type="submit" value="Enviar">
</form>

Entrada de Rang

L'element <input> amb type="range" crea un control de lliscament.

<form action="/submit" method="post">
  <label for="volume">Volum:</label>
  <input type="range" id="volume" name="volume" min="0" max="100">
  <input type="submit" value="Enviar">
</form>

Validació del Formulari

HTML5 proporciona atributs de validació de formularis integrats.

  • required: Especifica que un camp d'entrada ha d'estar omplert abans d'enviar el formulari.
  • pattern: Especifica una expressió regular que el valor del camp d'entrada ha de coincidir.
  • min i max: Especifiquen els valors mínim i màxim per a camps d'entrada numèrica.
<form action="/submit" method="post">
  <label for="email">Correu electrònic:</label>
  <input type="email" id="email" name="email" required>
  <label for="age">Edat:</label>
  <input type="number" id="age" name="age" min="18" max="99">
  <input type="submit" value="Enviar">
</form>

Conclusió

Entendre i utilitzar els elements de formulari HTML de manera efectiva és essencial per crear aplicacions web interactives i fàcils d'usar. Des de les entrades de text bàsiques fins a les càrregues de fitxers avançades i la validació integrada, els formularis HTML proporcionen una àmplia gamma de funcionalitats per recollir i processar dades dels usuaris. Practiqueu utilitzant aquests elements en diversos escenaris per convertir-vos en experts en la gestió de formularis.
