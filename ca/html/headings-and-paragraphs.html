Introducció

En aquesta secció, cobrirem els conceptes bàsics de la creació d'encapçalaments i paràgrafs en HTML. Aquests elements són fonamentals per estructurar el contingut a les pàgines web, fent-lo més fàcil de llegir i entendre.

Encapçalaments

Els encapçalaments s'utilitzen per definir els títols i subtítols del teu contingut. HTML proporciona sis nivells d'encapçalaments, des de <h1> fins a <h6>, amb <h1> sent el nivell més alt (o més important) i <h6> el més baix.

Conceptes Clau

  • Jerarquia: Els encapçalaments s'han d'utilitzar de manera jeràrquica, començant amb <h1> i baixant fins a <h6>.
  • Importància per al SEO: L'ús adequat dels encapçalaments pot millorar l'optimització per a motors de cerca (SEO) de la teva pàgina web.
  • Accessibilitat: Els lectors de pantalla utilitzen els encapçalaments per ajudar els usuaris a navegar pel contingut.

Exemples de Codi

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'Encapçalaments</title>
</head>
<body>
    <h1>Títol Principal</h1>
    <h2>Subtítol Nivell 1</h2>
    <h3>Subtítol Nivell 2</h3>
    <h4>Subtítol Nivell 3</h4>
    <h5>Subtítol Nivell 4</h5>
    <h6>Subtítol Nivell 5</h6>
</body>
</html>

Explicació

  • <h1>: Utilitzat per al títol principal de la pàgina.
  • <h2>: Utilitzat per a seccions principals dins de la pàgina.
  • <h3> a <h6>: Utilitzats per a subseccions i subdivisions addicionals.

Paràgrafs

Els paràgrafs s'utilitzen per definir blocs de text. L'etiqueta <p> s'utilitza per crear un paràgraf en HTML.

Conceptes Clau

  • Element de Nivell de Bloc: Els paràgrafs són elements de nivell de bloc, el que significa que ocupen tota l'amplada disponible.
  • Espaiat Automàtic: Els navegadors afegeixen automàticament un espai abans i després dels paràgrafs per separar-los d'altres elements.

Exemples de Codi

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Paràgraf</title>
</head>
<body>
    <p>Aquest és un paràgraf senzill. Els paràgrafs s'utilitzen per agrupar frases juntes.</p>
    <p>Aquest és un altre paràgraf. Cada paràgraf està separat per un espai.</p>
</body>
</html>

Explicació

  • <p>: Defineix un paràgraf de text.
  • El text dins de les etiquetes <p> es formata automàticament amb un espai abans i després.

Combinació d'Encapçalaments i Paràgrafs

La combinació d'encapçalaments i paràgrafs et permet crear contingut ben estructurat i llegible.

Exemple de Codi

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Encapçalaments i Paràgrafs</title>
</head>
<body>
    <h1>Introducció a l'HTML</h1>
    <p>HTML significa HyperText Markup Language. És el llenguatge estàndard per crear pàgines web.</p>
    
    <h2>Elements Bàsics</h2>
    <p>HTML consisteix en una sèrie d'elements que pots utilitzar per estructurar el teu contingut web.</p>
    
    <h3>Encapçalaments</h3>
    <p>Els encapçalaments s'utilitzen per crear títols i subtítols. Van des de <code>&lt;h1&gt;</code> fins a <code>&lt;h6&gt;</code>.</p>
    
    <h3>Paràgrafs</h3>
    <p>Els paràgrafs s'utilitzen per agrupar frases juntes. Es defineixen utilitzant l'etiqueta <code>&lt;p&gt;</code>.</p>
</body>
</html>

Explicació

  • L'exemple combina encapçalaments i paràgrafs per crear un document estructurat.
  • <h1> s'utilitza per al títol principal, <h2> per a seccions principals, i <h3> per a subseccions.
  • Les etiquetes <p> s'utilitzen per afegir text descriptiu sota cada encapçalament.

Conclusió

Entendre com utilitzar encapçalaments i paràgrafs és essencial per crear pàgines web ben estructurades i llegibles. Els encapçalaments ajuden a organitzar el contingut jeràrquicament, facilitant la navegació per als usuaris i els motors de cerca. Els paràgrafs agrupen frases en blocs coherents de text, millorant la llegibilitat. En dominar aquests elements bàsics d'HTML, estableixes una base sòlida per a conceptes més avançats de desenvolupament web.
