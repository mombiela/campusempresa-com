Introducció

JavaScript és un llenguatge de script potent que et permet crear pàgines web dinàmiques i interactives. Mentre que l'HTML proporciona l'estructura i el CSS s'encarrega de l'estil, JavaScript dóna vida a les teves pàgines web permetent-te manipular el DOM (Model d'Objectes del Document), gestionar esdeveniments i realitzar diverses accions basades en les interaccions de l'usuari.

Conceptes Clau

  • Conceptes Bàsics de JavaScript: Variables, tipus de dades, funcions i estructures de control.
  • Manipulació del DOM: Seleccionar i modificar elements HTML.
  • Gestió d'Esdeveniments: Respondre a accions de l'usuari com clics, enviaments de formularis i esdeveniments del teclat.
  • Contingut Dinàmic: Crear i eliminar elements, actualitzar contingut dinàmicament.
  • Validació de Formularis: Assegurar que les entrades de l'usuari compleixin certs criteris abans de l'enviament.

Conceptes Bàsics de JavaScript

Abans d'endinsar-nos en la interactivitat, cobrim alguns conceptes fonamentals de JavaScript.

Variables i Tipus de Dades

JavaScript suporta diversos tipus de dades, incloent cadenes de text, números, booleans, arrays i objectes.

// Declaració de variables
let name = "John Doe"; // Cadena de text
let age = 30; // Número
let isStudent = true; // Booleà
let hobbies = ["reading", "gaming", "coding"]; // Array
let address = { city: "New York", zip: "10001" }; // Objecte

Funcions

Les funcions són blocs de codi reutilitzables que realitzen tasques específiques.

// Declaració de funció
function greet(userName) {
    return `Hello, ${userName}!`;
}

// Crida a la funció
console.log(greet("Alice")); // Sortida: Hello, Alice!

Estructures de Control

Les estructures de control com les instruccions if-else i els bucles t'ajuden a controlar el flux del teu programa.

// Instrucció if-else
let score = 85;
if (score >= 90) {
    console.log("A");
} else if (score >= 80) {
    console.log("B");
} else {
    console.log("C");
}

// Bucle for
for (let i = 0; i < 5; i++) {
    console.log(i); // Sortida: 0 1 2 3 4
}

Manipulació del DOM

El DOM representa l'estructura del teu document HTML. JavaScript et permet seleccionar i manipular aquests elements.

Selecció d'Elements

Pots seleccionar elements utilitzant mètodes com getElementById, getElementsByClassName, getElementsByTagName, querySelector i querySelectorAll.

// Selecció d'un element per ID
let header = document.getElementById("header");

// Selecció d'elements per nom de classe
let items = document.getElementsByClassName("item");

// Selecció d'elements per nom de etiqueta
let paragraphs = document.getElementsByTagName("p");

// Selecció d'elements utilitzant querySelector
let main = document.querySelector("#main");

// Selecció de múltiples elements utilitzant querySelectorAll
let links = document.querySelectorAll("a");

Modificació d'Elements

Un cop seleccionats, pots modificar el contingut, els atributs i els estils dels elements.

// Canvi de l'HTML intern
header.innerHTML = "Welcome to My Website";

// Canvi d'atributs
main.setAttribute("class", "active");

// Canvi d'estils
main.style.backgroundColor = "lightblue";

Gestió d'Esdeveniments

Els esdeveniments són accions que ocorren al navegador, com clics, enviaments de formularis i pulsacions de tecles. Pots utilitzar JavaScript per respondre a aquests esdeveniments.

Afegir Escoltadors d'Esdeveniments

Pots afegir escoltadors d'esdeveniments als elements per executar codi quan ocorri un esdeveniment.

// Afegir un escoltador d'esdeveniments de clic
let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button clicked!");
});

// Afegir un escoltador d'esdeveniments d'enviament de formulari
let form = document.getElementById("myForm");
form.addEventListener("submit", function(event) {
    event.preventDefault(); // Prevenir l'enviament del formulari
    console.log("Form submitted!");
});

Contingut Dinàmic

JavaScript et permet crear i eliminar elements dinàmicament, així com actualitzar el contingut basat en les interaccions de l'usuari.

Crear i Eliminar Elements

Pots crear nous elements i afegir-los al DOM o eliminar elements existents.

// Creació d'un nou element
let newParagraph = document.createElement("p");
newParagraph.textContent = "This is a new paragraph.";
document.body.appendChild(newParagraph);

// Eliminació d'un element
let oldParagraph = document.getElementById("oldParagraph");
oldParagraph.remove();

Actualització de Contingut Dinàmicament

Pots actualitzar el contingut dels elements basat en les interaccions de l'usuari o altres condicions.

// Actualització de contingut basat en l'entrada de l'usuari
let input = document.getElementById("userInput");
input.addEventListener("input", function() {
    let output = document.getElementById("output");
    output.textContent = input.value;
});

Validació de Formularis

La validació de formularis assegura que les entrades de l'usuari compleixin certs criteris abans de l'enviament. Això es pot fer utilitzant JavaScript.

Validació Bàsica de Formularis

Pots validar les entrades del formulari comprovant els seus valors i mostrant missatges d'error si és necessari.

let form = document.getElementById("registrationForm");
form.addEventListener("submit", function(event) {
    let username = document.getElementById("username").value;
    let password = document.getElementById("password").value;

    if (username === "" || password === "") {
        event.preventDefault(); // Prevenir l'enviament del formulari
        alert("All fields are required!");
    }
});

Conclusió

JavaScript és una eina essencial per fer que les teves pàgines web siguin interactives i dinàmiques. Entenent els conceptes bàsics de JavaScript, la manipulació del DOM, la gestió d'esdeveniments, la creació de contingut dinàmic i la validació de formularis, pots crear experiències web atractives i fàcils d'usar. Practica aquests conceptes amb exemples del món real per millorar les teves habilitats i construir aplicacions web més interactives.
