Afegir contingut és un dels aspectes fonamentals de la creació de pàgines web amb HTML. Aquest tema et guiarà a través dels conceptes bàsics d'afegir text, imatges, enllaços i altres elements multimèdia a les teves pàgines web, progressant des de nivells bàsics fins a avançats.

Contingut de Text

El text és la forma més bàsica de contingut que pots afegir a una pàgina web. HTML proporciona diverses etiquetes per estructurar i formatar el text.

Paràgrafs i Encapçalaments

  • Paràgrafs: Utilitza l'etiqueta <p> per definir un paràgraf.
  • Encapçalaments: Utilitza les etiquetes <h1> a <h6> per definir encapçalaments, amb <h1> sent el nivell més alt i <h6> el més baix.
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Contingut de Text</title>
</head>
<body>
    <h1>Aquest és un Encapçalament H1</h1>
    <p>Aquest és un paràgraf de text.</p>
    <h2>Aquest és un Encapçalament H2</h2>
    <p>Un altre paràgraf de text.</p>
</body>
</html>

Llistes

  • Llistes Ordenades: Utilitza l'etiqueta <ol> per a llistes ordenades (numerades).
  • Llistes Desordenades: Utilitza l'etiqueta <ul> per a llistes desordenades (amb punts).
  • Elements de Llista: Utilitza l'etiqueta <li> per definir elements de llista.
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Llistes</title>
</head>
<body>
    <h2>Llista Ordenada</h2>
    <ol>
        <li>Primer element</li>
        <li>Segon element</li>
        <li>Tercer element</li>
    </ol>
    <h2>Llista Desordenada</h2>
    <ul>
        <li>Primer element</li>
        <li>Segon element</li>
        <li>Tercer element</li>
    </ul>
</body>
</html>

Imatges

Les imatges es poden afegir a les teves pàgines web utilitzant l'etiqueta <img>. L'atribut src especifica el camí a la imatge, i l'atribut alt proporciona text alternatiu per a lectors de pantalla.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Imatges</title>
</head>
<body>
    <h2>Afegint una Imatge</h2>
    <img src="path/to/image.jpg" alt="Descripció de la imatge">
</body>
</html>

Enllaços

Els enllaços es creen utilitzant l'etiqueta <a>. L'atribut href especifica la URL de la pàgina a la qual va l'enllaç.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Enllaços</title>
</head>
<body>
    <h2>Afegint un Enllaç</h2>
    <a href="https://www.example.com">Visita Example.com</a>
</body>
</html>

Contingut Multimèdia

Àudio

Pots afegir àudio a les teves pàgines web utilitzant l'etiqueta <audio>. L'atribut controls afegeix controls de reproducció, pausa i volum.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Àudio</title>
</head>
<body>
    <h2>Afegint Àudio</h2>
    <audio controls>
        <source src="path/to/audio.mp3" type="audio/mpeg">
        El teu navegador no suporta l'element d'àudio.
    </audio>
</body>
</html>

Vídeo

L'etiqueta <video> s'utilitza per incrustar contingut de vídeo. Similar a l'etiqueta <audio>, pot incloure controls per a la interacció de l'usuari.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Vídeo</title>
</head>
<body>
    <h2>Afegint Vídeo</h2>
    <video controls>
        <source src="path/to/video.mp4" type="video/mp4">
        El teu navegador no suporta l'etiqueta de vídeo.
    </video>
</body>
</html>

Taules

Les taules s'utilitzen per mostrar dades en un format tabular. Utilitza l'etiqueta <table> per crear una taula, <tr> per a files de taula, <th> per a encapçalaments de taula, i <td> per a cel·les de dades de taula.

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple de Taules</title>
</head>
<body>
    <h2>Afegint una Taula</h2>
    <table border="1">
        <tr>
            <th>Encapçalament 1</th>
            <th>Encapçalament 2</th>
        </tr>
        <tr>
            <td>Dada 1</td>
            <td>Dada 2</td>
        </tr>
        <tr>
            <td>Dada 3</td>
            <td>Dada 4</td>
        </tr>
    </table>
</body>
</html>

Conclusió

En aquest tema, has après com afegir diversos tipus de contingut a les teves pàgines web utilitzant HTML. Des de text bàsic i llistes fins a imatges, enllaços, multimèdia i taules, aquests elements formen els blocs de construcció del contingut web. Practica utilitzant aquestes etiquetes per sentir-te còmode afegint i estructurant contingut a les teves pàgines web.
