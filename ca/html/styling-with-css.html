Introducció al CSS

CSS (Fulls d'Estil en Cascada) és un llenguatge utilitzat per descriure la presentació d'un document escrit en HTML. El CSS controla el disseny, els colors, les fonts i l'aparença visual general de les pàgines web.

  • Sintaxi del CSS: Les regles del CSS es componen de selectors i declaracions. 
    selector {
  property: value;
}
  • Selectors: Seleccionen els elements HTML que vols estilitzar.
  • Declaracions: Contenen propietats i valors que defineixen els estils.

Afegir CSS a HTML

Hi ha tres maneres principals d'afegir CSS als documents HTML:

  1. CSS en línia: Utilitzant l'atribut style dins de les etiquetes HTML.

    <p style="color: blue;">Aquest és un paràgraf blau.</p>

  2. CSS intern: Utilitzant l'etiqueta <style> dins de la secció <head> del document HTML.

    <head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>

  3. CSS extern: Enllaçant un fitxer CSS extern utilitzant l'etiqueta <link>.

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

Selectors bàsics de CSS

  • Selector d'element: Selecciona tots els elements d'un tipus donat.

    p {
  color: blue;
}

  • Selector de classe: Selecciona elements amb un atribut de classe específic, prefixat amb un punt (.).

    .myClass {
  color: green;
}

  • Selector d'ID: Selecciona un únic element amb un atribut d'ID específic, prefixat amb un coixinet (#).

    #myId {
  color: red;
}

  • Selector d'atribut: Selecciona elements amb un atribut específic.

    [type="text"] {
  border: 1px solid black;
}

Selectors avançats de CSS

  • Selector de descendent: Selecciona elements que són descendents d'un element especificat.

    div p {
  color: purple;
}

  • Selector de fill: Selecciona elements que són fills directes d'un element especificat.

    div > p {
  color: orange;
}

  • Pseudo-classes: Selecciona elements basats en el seu estat.

    a:hover {
  color: pink;
}

Model de caixa de CSS

El model de caixa de CSS descriu les caixes rectangulars generades per als elements en l'arbre del document i consisteix en:

  • Contingut: El contingut real de la caixa, on apareixen text i imatges.
  • Farcell: Aclareix una àrea al voltant del contingut. El farcell és transparent.
  • Vora: Una vora que envolta el farcell i el contingut.
  • Marge: Aclareix una àrea fora de la vora. El marge també és transparent.
div {
  width: 300px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Tècniques de disseny de CSS

  • Flexbox: Un model de disseny que permet als elements alinear-se i distribuir l'espai dins d'un contenidor.

    .container {
  display: flex;
}

  • Grid: Un sistema de disseny per crear dissenys web complexos fàcilment.

    .grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
}

Disseny responsiu

El disseny responsiu assegura que les pàgines web es vegin bé en tots els dispositius. Les tècniques clau inclouen:

  • Consultes de mitjans: Aplicar diferents estils per a diferents dispositius o mides de pantalla.

    @media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

  • Dissenys fluids: Utilitzar unitats relatives com percentatges en lloc d'unitats fixes com píxels.

    .container {
  width: 100%;
}

Animacions i transicions de CSS

  • Transicions: Canviar suaument les propietats durant una durada especificada.

    .box {
  transition: width 2s;
}
.box:hover {
  width: 300px;
}

  • Animacions: Crear animacions complexes utilitzant fotogrames clau.

    @keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}
.box {
  animation: example 5s infinite;
}

Conclusió

El CSS és una eina poderosa per controlar la presentació visual de les pàgines web. Comprenent i utilitzant diverses tècniques de CSS, des de selectors bàsics fins a sistemes de disseny avançats i disseny responsiu, pots crear pàgines web visualment atractives i funcionals. La pràctica i l'experimentació són clau per dominar el CSS i millorar les teves habilitats de desenvolupament web.
