Introducció a les Capçaleres de Taula

En HTML, les taules s'utilitzen per mostrar dades en un format estructurat utilitzant files i columnes. Les capçaleres de taula són una part essencial de les taules ja que ajuden a identificar el contingut de cada columna o fila, fent que les dades siguin més fàcils d'entendre.

Creació de Capçaleres de Taula Bàsiques

Per crear una capçalera de taula en HTML, s'utilitza l'element <th>. L'element <th> s'utilitza dins de l'element <tr> (fila de taula) i normalment es col·loca al començament d'una fila de taula per definir la capçalera per a aquesta fila o columna.

Exemple de Capçaleres de Taula Bàsiques

<table>
  <tr>
    <th>First Name</th>
    <th>Last Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Smith</td>
    <td>25</td>
  </tr>
</table>

En aquest exemple:

  • Els elements <th> defineixen les capçaleres per a les columnes: "First Name", "Last Name" i "Age".
  • Els elements <td> contenen les dades reals per a cada fila.

Estilització de les Capçaleres de Taula

Les capçaleres de taula es poden estilitzar utilitzant CSS per fer-les destacar de la resta de les dades de la taula. Per defecte, els navegadors renderitzen les capçaleres de taula en negreta i alineades al centre.

Exemple d'Estilització de les Capçaleres de Taula

<style>
  th {
    background-color: #f2f2f2;
    color: #333;
    padding: 10px;
    text-align: left;
  }
</style>

<table>
  <tr>
    <th>First Name</th>
    <th>Last Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Smith</td>
    <td>25</td>
  </tr>
</table>

En aquest exemple:

  • El bloc <style> conté regles CSS que estilitzen els elements <th>.
  • La propietat background-color estableix un fons gris clar per a les capçaleres.
  • La propietat color estableix el color del text a gris fosc.
  • La propietat padding afegeix espai al voltant del text de la capçalera.
  • La propietat text-align alinea el text a l'esquerra.

Capçaleres de Taula Avançades

Per a taules més complexes, pot ser necessari crear capçaleres de diversos nivells o capçaleres que abastin diverses files o columnes. Això es pot aconseguir utilitzant els atributs rowspan i colspan.

Exemple de Capçaleres de Taula Avançades

<table>
  <tr>
    <th rowspan="2">Name</th>
    <th colspan="2">Contact Info</th>
  </tr>
  <tr>
    <th>Email</th>
    <th>Phone</th>
  </tr>
  <tr>
    <td>John Doe</td>
    <td>john@example.com</td>
    <td>555-1234</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>jane@example.com</td>
    <td>555-5678</td>
  </tr>
</table>

En aquest exemple:

  • El primer element <th> utilitza rowspan="2" per abastar dues files.
  • El segon element <th> utilitza colspan="2" per abastar dues columnes.
  • Això crea una taula amb una estructura de capçalera de diversos nivells.

Consideracions d'Accessibilitat

Quan es creen capçaleres de taula, és important assegurar-se que les taules siguin accessibles per a tots els usuaris, inclosos aquells que utilitzen lectors de pantalla. Això es pot aconseguir utilitzant l'atribut scope.

Exemple de Capçaleres de Taula Accessibles

<table>
  <tr>
    <th scope="col">First Name</th>
    <th scope="col">Last Name</th>
    <th scope="col">Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>Smith</td>
    <td>25</td>
  </tr>
</table>

En aquest exemple:

  • L'atribut scope="col" indica que la capçalera s'aplica a tota la columna, millorant l'accessibilitat per als lectors de pantalla.

Conclusió

Les capçaleres de taula són una part fonamental de la creació de taules estructurades i llegibles en HTML. Utilitzant l'element <th>, estilitzant amb CSS i considerant l'accessibilitat, es poden crear taules que siguin tant visualment atractives com fàcils d'utilitzar. Les tècniques avançades com rowspan i colspan permeten estructures de taula més complexes, fent que la presentació de les dades sigui encara més versàtil.
