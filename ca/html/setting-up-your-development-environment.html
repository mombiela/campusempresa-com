◄ Què és l'HTML?
Estructura Bàsica de l'HTML ►

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert en HTML. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar.

Conceptes Clau

  • Editor de Text: Una aplicació de programari per escriure i editar codi.
  • Navegador Web: Una aplicació de programari per accedir a informació a la World Wide Web.
  • Servidor Local (Opcional): Un servidor que s'executa a la teva màquina local per provar pàgines web.

Triar un Editor de Text

Un editor de text és essencial per escriure codi HTML. Aquí tens algunes opcions populars:

  • Visual Studio Code (VS Code): Un editor gratuït i de codi obert amb funcions robustes.
  • Sublime Text: Un editor de text sofisticat per a codi, marcatge i prosa.
  • Atom: Un editor de text hackejable per al segle XXI.
  • Notepad++: Un editor de codi font gratuït i substitut del Notepad.

Instal·lar Visual Studio Code

  1. Descarregar: Ves al lloc web de Visual Studio Code i descarrega l'instal·lador per al teu sistema operatiu.
  2. Instal·lar: Executa l'instal·lador i segueix les instruccions en pantalla.
  3. Llançar: Obre Visual Studio Code després de la instal·lació.

Configurar Visual Studio Code

  • Extensions: Millora la funcionalitat amb extensions.
    • HTML Snippets: Proporciona un conjunt de fragments de codi HTML.
    • Live Server: Llança un servidor de desenvolupament local amb una funció de recàrrega en viu.

Instal·lar Extensions

  1. Fes clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
  2. Cerca l'extensió desitjada (per exemple, "Live Server").
  3. Fes clic al botó Instal·lar.

Configurar un Navegador Web

Un navegador web és necessari per veure i provar el teu codi HTML. Els navegadors populars inclouen:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Eines de Desenvolupador

Els navegadors moderns vénen amb eines de desenvolupador integrades que t'ajuden a inspeccionar i depurar el teu codi HTML.

  • Google Chrome: Fes clic dret en una pàgina web i selecciona "Inspect" o prem Ctrl+Shift+I.
  • Mozilla Firefox: Fes clic dret en una pàgina web i selecciona "Inspect Element" o prem Ctrl+Shift+I.

Opcional: Configurar un Servidor Local

Un servidor local és útil per provar aplicacions web més complexes. Per a HTML bàsic, aquest pas és opcional però recomanat a mesura que avancis.

Utilitzar l'Extensió Live Server a VS Code

  1. Instal·lar Live Server: Segueix els passos a la secció "Instal·lar Extensions".
  2. Obrir un Projecte: Obre el teu projecte HTML a VS Code.
  3. Iniciar Live Server: Fes clic dret al teu fitxer index.html i selecciona "Open with Live Server".

Exemple: Crear el teu Primer Fitxer HTML

  1. Obre VS Code.
  2. Crear un Nou Fitxer: Fes clic a File > New File.
  3. Desar el Fitxer: Desa el fitxer amb una extensió .html, per exemple, index.html.
  4. Escriure HTML Bàsic: Afegeix el següent codi al teu fitxer:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>
  1. Obrir amb Live Server: Fes clic dret a index.html i selecciona "Open with Live Server".

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu viatge per dominar l'HTML. Triant l'editor de text adequat, configurant-lo amb extensions útils i entenent com utilitzar els navegadors web i les seves eines de desenvolupador, estàs ben equipat per començar a codificar. Eines opcionals com un servidor local poden millorar encara més la teva experiència de desenvolupament a mesura que progressis cap a temes més avançats.
