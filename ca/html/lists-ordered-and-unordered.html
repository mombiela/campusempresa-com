Introducció

En HTML, les llistes són una manera fonamental d'organitzar i mostrar contingut. Hi ha dos tipus principals de llistes: llistes ordenades i llistes desordenades. Entendre com utilitzar aquestes llistes és crucial per estructurar el contingut web de manera efectiva.

Llistes Ordenades

Les llistes ordenades s'utilitzen quan l'ordre dels elements és important. Cada element en una llista ordenada està numerat.

Sintaxi

Una llista ordenada es crea utilitzant l'etiqueta <ol>, i cada element de la llista es defineix amb l'etiqueta <li>.

<ol>
  <li>Primer element</li>
  <li>Segon element</li>
  <li>Tercer element</li>
</ol>

Exemple Explicat

  • <ol>: Aquesta etiqueta defineix l'inici d'una llista ordenada.
  • <li>: Cada element dins de la llista està embolicat en aquesta etiqueta.
  • El navegador numera automàticament els elements.

Exemple Pràctic

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Exemple de Llista Ordenada</title>
</head>
<body>
  <h1>Passos per Fer un Entrepà</h1>
  <ol>
    <li>Agafa dues llesques de pa.</li>
    <li>Unta mantega de cacauet en una llesca.</li>
    <li>Unta melmelada en l'altra llesca.</li>
    <li>Posa les llesques juntes.</li>
    <li>Gaudeix del teu entrepà!</li>
  </ol>
</body>
</html>

Llistes Desordenades

Les llistes desordenades s'utilitzen quan l'ordre dels elements no importa. Cada element en una llista desordenada està marcat típicament amb un punt.

Sintaxi

Una llista desordenada es crea utilitzant l'etiqueta <ul>, i cada element de la llista es defineix amb l'etiqueta <li>.

<ul>
  <li>Primer element</li>
  <li>Segon element</li>
  <li>Tercer element</li>
</ul>

Exemple Explicat

  • <ul>: Aquesta etiqueta defineix l'inici d'una llista desordenada.
  • <li>: Cada element dins de la llista està embolicat en aquesta etiqueta.
  • El navegador afegeix automàticament punts als elements.

Exemple Pràctic

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Exemple de Llista Desordenada</title>
</head>
<body>
  <h1>Llista de la Compra</h1>
  <ul>
    <li>Llet</li>
    <li>Pa</li>
    <li>Ous</li>
    <li>Mantega</li>
  </ul>
</body>
</html>

Comparació de Llistes Ordenades i Desordenades

| Característica | Llista Ordenada (<ol>) | Llista Desordenada (<ul>) | |-----------------|--------------------------|-----------------------------| | Etiqueta | <ol> | <ul> | | Etiqueta d'Element | <li> | <li> | | Marcador per Defecte | Números | Punts | | Cas d'Ús | Passos seqüencials, classificacions | Agrupació d'elements sense un ordre específic |

Conclusió

Les llistes són una part versàtil i essencial de l'HTML per organitzar contingut. Les llistes ordenades (<ol>) són ideals per a seqüències on l'ordre importa, mentre que les llistes desordenades (<ul>) són perfectes per a col·leccions d'elements on l'ordre no és important. Dominant aquestes estructures bàsiques de llistes, pots crear pàgines web ben organitzades i fàcils de llegir.
