Les entitats HTML són una part crucial del desenvolupament web, permetent als desenvolupadors mostrar caràcters reservats, símbols especials i fins i tot caràcters no imprimibles en pàgines web. Aquest tema cobrirà els conceptes bàsics de les entitats HTML, el seu ús i proporcionarà exemples per ajudar-te a entendre com implementar-les en els teus documents HTML.

Què són les Entitats HTML?

Les entitats HTML són una manera de representar caràcters que tenen significats especials en HTML o que no es poden escriure fàcilment amb un teclat. S'utilitzen per assegurar que aquests caràcters es mostrin correctament al navegador.

Conceptes Clau

  • Caràcters Reservats: Caràcters com <, >, & i " tenen significats especials en HTML i necessiten ser representats per entitats per ser mostrats correctament.
  • Símbols Especials: Caràcters com ©, ®, i ™ no es troben típicament en un teclat i es representen per entitats.
  • Caràcters No Imprimibles: Caràcters com espais, tabulacions i salts de línia també poden ser representats per entitats.

Ús de les Entitats HTML

Per utilitzar una entitat HTML, simplement inclou-la en el teu codi HTML on vols que aparegui el caràcter. Aquí tens alguns exemples:

Exemple 1: Mostrar Caràcters Reservats

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Entitats HTML</title>
</head>
<body>
    <p>Utilitza &lt; i &gt; per mostrar els signes de menor que i major que.</p>
    <p>Utilitza &amp; per mostrar un ampersand.</p>
</body>
</html>

Exemple 2: Mostrar Símbols Especials

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Entitats HTML</title>
</head>
<body>
    <p>Símbol de copyright: &copy; 2023</p>
    <p>Símbol de marca registrada: &reg;</p>
    <p>Símbol de marca comercial: &trade;</p>
</body>
</html>

Exemple 3: Caràcters No Imprimibles

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Exemple d'Entitats HTML</title>
</head>
<body>
    <p>Aquesta&nbsp;és&nbsp;una&nbsp;frase&nbsp;amb&nbsp;espais&nbsp;no&nbsp;trencables.</p>
</body>
</html>

Conclusió

Les entitats HTML són essencials per mostrar caràcters que tenen significats especials en HTML o que no es poden escriure fàcilment amb un teclat. Utilitzant entitats, pots assegurar que el teu contingut web es mostri correctament i de manera consistent en diferents navegadors i dispositius. Recorda utilitzar els noms o números d'entitat apropiats per als caràcters que necessites mostrar, i practica incorporant-los en els teus documents HTML per sentir-te més còmode amb el seu ús.
