HTML5 va introduir una varietat de noves APIs que permeten als desenvolupadors crear aplicacions web més dinàmiques i interactives. Aquestes APIs proporcionen funcionalitats per a tot, des de dibuixar gràfics fins a emmagatzemar dades localment. Aquesta secció cobrirà algunes de les APIs HTML5 més importants, proporcionant explicacions, exemples i exercicis per ajudar-te a entendre-les i utilitzar-les de manera efectiva.

API de Geolocalització

L'API de Geolocalització permet a les aplicacions web accedir a la ubicació geogràfica d'un usuari. Això pot ser útil per proporcionar serveis basats en la ubicació.

Conceptes Clau

  • navigator.geolocation: L'objecte principal utilitzat per accedir a l'API de Geolocalització.
  • getCurrentPosition(): Mètode per obtenir la posició actual de l'usuari.
  • watchPosition(): Mètode per obtenir la posició actual de l'usuari i continuar monitoritzant-la.

Exemple

<!DOCTYPE html>
<html>
<head>
    <title>Exemple d'API de Geolocalització</title>
</head>
<body>
    <button onclick="getLocation()">Obtenir Ubicació</button>
    <p id="location"></p>

    <script>
        function getLocation() {
            if (navigator.geolocation) {
                navigator.geolocation.getCurrentPosition(showPosition);
            } else {
                document.getElementById("location").innerHTML = "La geolocalització no és compatible amb aquest navegador.";
            }
        }

        function showPosition(position) {
            document.getElementById("location").innerHTML = 
                "Latitud: " + position.coords.latitude + 
                "<br>Longitud: " + position.coords.longitude;
        }
    </script>
</body>
</html>

Exercici

  1. Modifica l'exemple per mostrar un missatge d'error si l'usuari denega el permís per accedir a la seva ubicació.
  2. Utilitza el mètode watchPosition() per actualitzar contínuament la ubicació de l'usuari a la pàgina.

API de Canvas

L'API de Canvas proporciona una manera de dibuixar gràfics utilitzant JavaScript i l'element <canvas>. Es pot utilitzar per renderitzar gràfics, gràfics de jocs o altres imatges visuals sobre la marxa.

Conceptes Clau

  • element : L'element HTML utilitzat per definir un canvas.
  • getContext(): Mètode per obtenir el context de dibuix al canvas.
  • fillRect(), strokeRect(): Mètodes per dibuixar rectangles.

Exemple

<!DOCTYPE html>
<html>
<head>
    <title>Exemple d'API de Canvas</title>
</head>
<body>
    <canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

    <script>
        var canvas = document.getElementById("myCanvas");
        var ctx = canvas.getContext("2d");
        ctx.fillStyle = "#FF0000";
        ctx.fillRect(20, 20, 150, 100);
    </script>
</body>
</html>

Exercici

  1. Dibuixa un cercle al canvas.
  2. Crea una animació simple que mogui un rectangle a través del canvas.

API d'Emmagatzematge Local

L'API d'Emmagatzematge Local permet a les aplicacions web emmagatzemar dades localment dins del navegador de l'usuari. Aquestes dades persisteixen fins i tot després que el navegador es tanqui i es torni a obrir.

Conceptes Clau

  • localStorage: L'objecte principal utilitzat per accedir a l'emmagatzematge local.
  • setItem(), getItem(): Mètodes per emmagatzemar i recuperar dades.

Exemple

<!DOCTYPE html>
<html>
<head>
    <title>Exemple d'API d'Emmagatzematge Local</title>
</head>
<body>
    <input type="text" id="name" placeholder="Introdueix el teu nom">
    <button onclick="saveName()">Desa el Nom</button>
    <button onclick="loadName()">Carrega el Nom</button>
    <p id="displayName"></p>

    <script>
        function saveName() {
            var name = document.getElementById("name").value;
            localStorage.setItem("name", name);
        }

        function loadName() {
            var name = localStorage.getItem("name");
            if (name) {
                document.getElementById("displayName").innerHTML = "Hola, " + name;
            } else {
                document.getElementById("displayName").innerHTML = "No s'ha trobat cap nom.";
            }
        }
    </script>
</body>
</html>

Exercici

  1. Modifica l'exemple per desar i carregar múltiples dades (per exemple, nom, edat i correu electrònic).
  2. Crea un formulari que desi les preferències de l'usuari (per exemple, color del tema) i les apliqui quan es carregui la pàgina.

Conclusió

Les APIs HTML5 proporcionen eines poderoses per crear aplicacions web dinàmiques i interactives. Entenent i utilitzant aquestes APIs, pots millorar la funcionalitat i l'experiència d'usuari dels teus projectes web. Practica utilitzant les APIs de Geolocalització, Canvas i Emmagatzematge Local a través dels exemples i exercicis proporcionats per solidificar la teva comprensió. A mesura que et sentis més còmode amb aquestes APIs, explora altres APIs HTML5 com l'API de Fitxers, Web Workers i WebSockets per ampliar encara més les teves habilitats de desenvolupament web.
