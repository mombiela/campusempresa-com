Introducció

En HTML, l'element <input> s'utilitza per crear controls interactius per a formularis web que accepten l'entrada de l'usuari. L'atribut type de l'element <input> especifica el tipus de control d'entrada que es mostrarà. Entendre els diferents tipus d'entrada és crucial per crear formularis efectius i fàcils d'utilitzar.

Tipus d'entrada bàsics

Entrada de text

El tipus d'entrada text s'utilitza per a l'entrada de text d'una sola línia.

<label for="name">Nom:</label>
<input type="text" id="name" name="name">
  • Ús: Recollir dades de text curtes com noms o títols.

Entrada de contrasenya

El tipus d'entrada password amaga els caràcters introduïts.

<label for="password">Contrasenya:</label>
<input type="password" id="password" name="password">
  • Ús: Recollir informació sensible com contrasenyes.

Entrada de correu electrònic

El tipus d'entrada email assegura que el valor introduït sigui una adreça de correu electrònic vàlida.

<label for="email">Correu electrònic:</label>
<input type="email" id="email" name="email">
  • Ús: Recollir adreces de correu electrònic.

Entrada de número

El tipus d'entrada number permet només valors numèrics.

<label for="age">Edat:</label>
<input type="number" id="age" name="age" min="0" max="100">
  • Ús: Recollir dades numèriques com l'edat o la quantitat.

Entrada de data

El tipus d'entrada date proporciona un selector de dates.

<label for="birthday">Aniversari:</label>
<input type="date" id="birthday" name="birthday">
  • Ús: Recollir dates com aniversaris o dates d'esdeveniments.

Tipus d'entrada avançats

Entrada de rang

El tipus d'entrada range permet a l'usuari seleccionar un valor d'un rang especificat.

<label for="volume">Volum:</label>
<input type="range" id="volume" name="volume" min="0" max="100">
  • Ús: Ajustar configuracions com el volum o la brillantor.

Entrada de color

El tipus d'entrada color proporciona un selector de colors.

<label for="favcolor">Color favorit:</label>
<input type="color" id="favcolor" name="favcolor">
  • Ús: Seleccionar colors per a disseny o personalització.

Entrada de fitxer

El tipus d'entrada file permet a l'usuari seleccionar fitxers.

<label for="resume">Carrega el teu currículum:</label>
<input type="file" id="resume" name="resume">
  • Ús: Carregar fitxers com documents o imatges.

Entrada de cerca

El tipus d'entrada search està optimitzat per a consultes de cerca.

<label for="search">Cerca:</label>
<input type="search" id="search" name="search">
  • Ús: Proporcionar una caixa de cerca per a consultes d'usuari.

Entrada de telèfon

El tipus d'entrada tel s'utilitza per a números de telèfon.

<label for="phone">Número de telèfon:</label>
<input type="tel" id="phone" name="phone">
  • Ús: Recollir números de telèfon.

Comparació de tipus d'entrada

| Tipus d'entrada | Descripció | Exemple d'ús | |-----------------|-------------|------------------| | text | Entrada de text d'una sola línia | Nom, Títol | | password | Entrada de text oculta | Contrasenyes | | email | Entrada d'adreça de correu electrònic | Adreces de correu electrònic | | number | Entrada numèrica | Edat, Quantitat | | date | Selector de dates | Aniversaris, Dates d'esdeveniments | | range | Selector de rang | Volum, Brillantor | | color | Selector de colors | Disseny, Personalització | | file | Selector de fitxers | Carregar documents | | search | Caixa de cerca | Consultes d'usuari | | tel | Entrada de número de telèfon | Números de telèfon |

Conclusió

Entendre i utilitzar els diferents tipus d'entrada en HTML és essencial per crear formularis efectius i fàcils d'utilitzar. Cada tipus d'entrada serveix per a un propòsit específic i millora l'experiència de l'usuari proporcionant controls adequats per a diferents tipus de dades. Dominant aquests tipus d'entrada, pots assegurar-te que els teus formularis siguin tant funcionals com intuïtius.
