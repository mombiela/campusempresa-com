En aquesta secció, explorarem com inserir àudio en documents HTML. Aquest tema cobrirà tècniques bàsiques i avançades, assegurant que puguis afegir i controlar elements d'àudio de manera efectiva en els teus projectes web.

Introducció a la Inserció d'Àudio

Inserir àudio en HTML et permet afegir so a les teves pàgines web, millorant l'experiència de l'usuari. HTML5 va introduir l'element <audio>, que facilita la inserció d'arxius d'àudio.

Inserció Bàsica d'Àudio

Per inserir àudio en HTML, utilitzes l'element <audio>. Aquí tens un exemple senzill:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Inserció Bàsica d'Àudio</title>
</head>
<body>
    <h1>Exemple d'Inserció d'Àudio</h1>
    <audio controls>
        <source src="audiofile.mp3" type="audio/mpeg">
        El teu navegador no suporta l'element d'àudio.
    </audio>
</body>
</html>
  • L'element <audio> s'utilitza per inserir contingut d'àudio.
  • L'atribut controls afegeix controls de reproducció, pausa i volum.
  • L'element <source> especifica l'arxiu d'àudio i el seu format.

Formats d'Àudio Suportats

Diferents navegadors suporten diferents formats d'àudio. Els formats més comuns són:

| Format | Tipus MIME | Suport del Navegador | |--------|-----------------|----------------------------| | MP3 | audio/mpeg | Tots els navegadors majors | | Ogg | audio/ogg | Firefox, Chrome, Opera | | WAV | audio/wav | Tots els navegadors majors |

Per assegurar la compatibilitat en tots els navegadors, és una bona pràctica proporcionar múltiples formats:

<audio controls>
    <source src="audiofile.mp3" type="audio/mpeg">
    <source src="audiofile.ogg" type="audio/ogg">
    El teu navegador no suporta l'element d'àudio.
</audio>

Inserció Avançada d'Àudio

Per tenir més control sobre l'element d'àudio, pots utilitzar JavaScript. Aquí tens un exemple de com reproduir, pausar i controlar el volum d'un arxiu d'àudio utilitzant JavaScript:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Control Avançat d'Àudio</title>
</head>
<body>
    <h1>Exemple de Control Avançat d'Àudio</h1>
    <audio id="myAudio">
        <source src="audiofile.mp3" type="audio/mpeg">
        El teu navegador no suporta l'element d'àudio.
    </audio>
    <br>
    <button onclick="playAudio()">Reproduir</button>
    <button onclick="pauseAudio()">Pausar</button>
    <button onclick="increaseVolume()">Augmentar Volum</button>
    <button onclick="decreaseVolume()">Disminuir Volum</button>

    <script>
        var audio = document.getElementById("myAudio");

        function playAudio() {
            audio.play();
        }

        function pauseAudio() {
            audio.pause();
        }

        function increaseVolume() {
            if (audio.volume < 1) audio.volume += 0.1;
        }

        function decreaseVolume() {
            if (audio.volume > 0) audio.volume -= 0.1;
        }
    </script>
</body>
</html>
  • playAudio(): Reprodueix l'àudio.
  • pauseAudio(): Pausa l'àudio.
  • increaseVolume(): Augmenta el volum en 0.1.
  • decreaseVolume(): Disminueix el volum en 0.1.

Conclusió

Inserir àudio en HTML és un procés senzill amb l'element <audio>. Proporcionant múltiples formats d'àudio, assegures la compatibilitat en diferents navegadors. Per a un control avançat, es pot utilitzar JavaScript per manipular la reproducció i el volum de l'àudio. Amb aquestes tècniques, pots enriquir les teves pàgines web amb contingut d'àudio de qualitat.
