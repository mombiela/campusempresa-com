Text preformatat en HTML
Afegir Imatges ►

Introducció

El text preformatat en HTML s'utilitza per mostrar el text exactament tal com està escrit al fitxer HTML, preservant els espais, salts de línia i altres caràcters d'espai en blanc. Això és particularment útil per mostrar fragments de codi, poesia o qualsevol text on el format sigui crucial.

L'etiqueta `
`

L'element HTML principal utilitzat per al text preformatat és l'etiqueta <pre>.

  • Sintaxi: 
    <pre>
  El teu text preformatat va aquí.
</pre>
  • Atributs: L'etiqueta <pre> admet atributs globals com id, class, style, etc.

Exemple

<pre>
  Aquest és un exemple de text preformatat.
  Observa com es preserven els espais i els salts de línia.
</pre>

Quan es renderitza en un navegador, el text apareixerà exactament tal com està escrit dins de les etiquetes <pre>, incloent tots els espais i salts de línia.

Combinació de `
` amb altres etiquetes

Pots combinar l'etiqueta <pre> amb altres etiquetes HTML per millorar la presentació del text preformatat.

Ús de `` dins de `
`

L'etiqueta <code> s'utilitza sovint dins de <pre> per denotar fragments de codi.

  • Sintaxi: 
    <pre>
  <code>
    // Aquest és un fragment de codi
    function helloWorld() {
      console.log("Hello, World!");
    }
  </code>
</pre>

Exemple

<pre>
  <code>
    // Aquesta és una funció JavaScript
    function greet(name) {
      console.log("Hello, " + name + "!");
    }
  </code>
</pre>

Estilització del text preformatat

Pots estilitzar el text preformatat utilitzant CSS per fer-lo més llegible o per adaptar-lo al disseny del teu lloc web.

Exemple

<style>
  pre {
    background-color: #f4f4f4;
    padding: 10px;
    border: 1px solid #ccc;
    overflow: auto;
  }
  code {
    color: #d63384;
  }
</style>

<pre>
  <code>
    // Fragment de codi estilitzat
    const add = (a, b) => a + b;
  </code>
</pre>

Exercicis pràctics

Exercici 1: Text preformatat bàsic

Crea un fitxer HTML senzill que utilitzi l'etiqueta <pre> per mostrar un poema amb un format específic.

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Exemple de text preformatat</title>
</head>
<body>
  <pre>
    Les roses són vermelles,
    Les violetes són blaves,
    El sucre és dolç,
    I tu també.
  </pre>
</body>
</html>

Exercici 2: Mostrar fragments de codi

Crea un fitxer HTML que utilitzi tant les etiquetes <pre> com <code> per mostrar un fragment de codi Python.

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Exemple de fragment de codi</title>
</head>
<body>
  <pre>
    <code>
      def greet(name):
          print(f"Hola, {name}!")
    </code>
  </pre>
</body>
</html>

Exercici 3: Estilització del text preformatat

Crea un fitxer HTML que estilitzí el text preformatat utilitzant CSS.

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Text preformatat estilitzat</title>
  <style>
    pre {
      background-color: #f0f0f0;
      padding: 15px;
      border-radius: 5px;
    }
    code {
      color: #2a9d8f;
    }
  </style>
</head>
<body>
  <pre>
    <code>
      // Funció JavaScript
      function add(a, b) {
        return a + b;
      }
    </code>
  </pre>
</body>
</html>

Conclusió

El text preformatat és una característica poderosa en HTML que et permet mostrar el text exactament tal com està escrit, preservant tots els espais en blanc i salts de línia. L'etiqueta <pre> és essencial per a aquest propòsit, i es pot combinar amb altres etiquetes com <code> per mostrar fragments de codi de manera efectiva. Estilitzar el text preformatat amb CSS pot millorar la seva llegibilitat i integració amb el disseny del teu lloc web. Entenent i practicant aquests conceptes, podràs utilitzar efectivament el text preformatat en els teus projectes web.
