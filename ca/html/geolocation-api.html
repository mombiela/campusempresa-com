Introducció a l'API de Geolocalització

L'API de Geolocalització permet que les aplicacions web accedeixin a la ubicació geogràfica d'un dispositiu. Això pot ser útil per a una varietat d'aplicacions, com ara mapes, serveis basats en la ubicació i més.

Conceptes Clau

  • API de Geolocalització: Una API web que proporciona accés a la ubicació geogràfica del dispositiu de l'usuari.
  • Navigator.geolocation: La interfície a través de la qual s'accedeix a l'API de Geolocalització.
  • Position: Un objecte que conté la latitud i la longitud del dispositiu.
  • PositionError: Un objecte que conté informació d'error si no es pot determinar la ubicació.

Accedint a l'API de Geolocalització

Per utilitzar l'API de Geolocalització, cal comprovar si el navegador la suporta i després sol·licitar la ubicació de l'usuari.

Comprovant el Suport

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'API de Geolocalització</title>
</head>
<body>
    <script>
        if ("geolocation" in navigator) {
            console.log("La geolocalització està disponible");
        } else {
            console.log("La geolocalització no és suportada per aquest navegador");
        }
    </script>
</body>
</html>
  • Explicació: Aquest codi comprova si l'objecte geolocation està disponible en l'objecte navigator, indicant que el navegador suporta l'API de Geolocalització.

Obtenint la Posició Actual

Per obtenir la posició actual del dispositiu, utilitza el mètode getCurrentPosition.

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'API de Geolocalització</title>
</head>
<body>
    <button onclick="getLocation()">Obtenir Ubicació</button>
    <p id="location"></p>

    <script>
        function getLocation() {
            if (navigator.geolocation) {
                navigator.geolocation.getCurrentPosition(showPosition, showError);
            } else {
                document.getElementById("location").innerHTML = "La geolocalització no és suportada per aquest navegador.";
            }
        }

        function showPosition(position) {
            document.getElementById("location").innerHTML = 
                "Latitud: " + position.coords.latitude + 
                "<br>Longitud: " + position.coords.longitude;
        }

        function showError(error) {
            switch(error.code) {
                case error.PERMISSION_DENIED:
                    document.getElementById("location").innerHTML = "L'usuari ha denegat la sol·licitud de geolocalització.";
                    break;
                case error.POSITION_UNAVAILABLE:
                    document.getElementById("location").innerHTML = "La informació de la ubicació no està disponible.";
                    break;
                case error.TIMEOUT:
                    document.getElementById("location").innerHTML = "La sol·licitud per obtenir la ubicació de l'usuari ha caducat.";
                    break;
                case error.UNKNOWN_ERROR:
                    document.getElementById("location").innerHTML = "S'ha produït un error desconegut.";
                    break;
            }
        }
    </script>
</body>
</html>
  • Explicació:
    • La funció getLocation comprova si la geolocalització és suportada i després crida getCurrentPosition.
    • La funció showPosition mostra la latitud i la longitud.
    • La funció showError maneja diferents tipus d'errors que poden ocórrer.

Monitoritzant la Posició

Per monitoritzar contínuament la posició, utilitza el mètode watchPosition.

Exemple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'API de Geolocalització</title>
</head>
<body>
    <button onclick="startWatching()">Començar a Monitoritzar</button>
    <button onclick="stopWatching()">Aturar la Monitorització</button>
    <p id="location"></p>

    <script>
        let watchID;

        function startWatching() {
            if (navigator.geolocation) {
                watchID = navigator.geolocation.watchPosition(showPosition, showError);
            } else {
                document.getElementById("location").innerHTML = "La geolocalització no és suportada per aquest navegador.";
            }
        }

        function stopWatching() {
            if (watchID) {
                navigator.geolocation.clearWatch(watchID);
            }
        }

        function showPosition(position) {
            document.getElementById("location").innerHTML = 
                "Latitud: " + position.coords.latitude + 
                "<br>Longitud: " + position.coords.longitude;
        }

        function showError(error) {
            switch(error.code) {
                case error.PERMISSION_DENIED:
                    document.getElementById("location").innerHTML = "L'usuari ha denegat la sol·licitud de geolocalització.";
                    break;
                case error.POSITION_UNAVAILABLE:
                    document.getElementById("location").innerHTML = "La informació de la ubicació no està disponible.";
                    break;
                case error.TIMEOUT:
                    document.getElementById("location").innerHTML = "La sol·licitud per obtenir la ubicació de l'usuari ha caducat.";
                    break;
                case error.UNKNOWN_ERROR:
                    document.getElementById("location").innerHTML = "S'ha produït un error desconegut.";
                    break;
            }
        }
    </script>
</body>
</html>
  • Explicació:
    • La funció startWatching comença a monitoritzar la posició utilitzant watchPosition.
    • La funció stopWatching atura la monitorització de la posició utilitzant clearWatch.

Opcions Avançades

Els mètodes getCurrentPosition i watchPosition accepten un tercer paràmetre per a opcions.

Objecte d'Opcions

  • enableHighAccuracy: Un booleà que indica si es desitja alta precisió.
  • timeout: El temps màxim (en mil·lisegons) per esperar una posició.
  • maximumAge: L'edat màxima (en mil·lisegons) d'una posició emmagatzemada en memòria cau.

Exemple amb Opcions

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple d'API de Geolocalització</title>
</head>
<body>
    <button onclick="getLocation()">Obtenir Ubicació</button>
    <p id="location"></p>

    <script>
        function getLocation() {
            if (navigator.geolocation) {
                const options = {
                    enableHighAccuracy: true,
                    timeout: 5000,
                    maximumAge: 0
                };
                navigator.geolocation.getCurrentPosition(showPosition, showError, options);
            } else {
                document.getElementById("location").innerHTML = "La geolocalització no és suportada per aquest navegador.";
            }
        }

        function showPosition(position) {
            document.getElementById("location").innerHTML = 
                "Latitud: " + position.coords.latitude + 
                "<br>Longitud: " + position.coords.longitude;
        }

        function showError(error) {
            switch(error.code) {
                case error.PERMISSION_DENIED:
                    document.getElementById("location").innerHTML = "L'usuari ha denegat la sol·licitud de geolocalització.";
                    break;
                case error.POSITION_UNAVAILABLE:
                    document.getElementById("location").innerHTML = "La informació de la ubicació no està disponible.";
                    break;
                case error.TIMEOUT:
                    document.getElementById("location").innerHTML = "La sol·licitud per obtenir la ubicació de l'usuari ha caducat.";
                    break;
                case error.UNKNOWN_ERROR:
                    document.getElementById("location").innerHTML = "S'ha produït un error desconegut.";
                    break;
            }
        }
    </script>
</body>
</html>
  • Explicació:
    • L'objecte options es passa a getCurrentPosition per especificar alta precisió, un temps d'espera de 5 segons i no utilitzar posicions emmagatzemades en memòria cau.

Conclusió

L'API de Geolocalització és una eina poderosa per accedir a la ubicació geogràfica del dispositiu d'un usuari. Comprenent com comprovar el suport, obtenir la posició actual, monitoritzar la posició i utilitzar opcions avançades, pots crear aplicacions web que proporcionin serveis basats en la ubicació. Sempre maneja els errors amb gràcia i respecta la privacitat de l'usuari sol·licitant permís per accedir a la seva ubicació.
