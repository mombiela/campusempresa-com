Introducció a l'Element Canvas

L'element <canvas> és una característica poderosa en HTML5 que permet el renderitzat dinàmic i scriptable de formes 2D i imatges bitmap. S'utilitza àmpliament per crear gràfics, animacions, jocs i visualitzacions de dades.

Conceptes Clau

  • Element: L'element <canvas> en si, que actua com un contenidor per als gràfics.
  • Context: El context de dibuix, que proporciona els mètodes i propietats per dibuixar al canvas.
  • Renderitzat: El procés de dibuixar formes, text, imatges i altres objectes al canvas.

Ús Bàsic

Per utilitzar l'element <canvas>, necessites:

  1. Definir l'element <canvas> en el teu HTML.
  2. Accedir al canvas a través de JavaScript per dibuixar-hi.

Estructura HTML

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Canvas</title>
</head>
<body>
    <canvas id="myCanvas" width="500" height="500" style="border:1px solid #000000;"></canvas>
</body>
</html>
  • L'atribut id s'utilitza per referenciar el canvas en JavaScript.
  • Els atributs width i height defineixen la mida del canvas.

Accedint al Canvas en JavaScript

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
  • El mètode getContext('2d') retorna un context de dibuix en el canvas.

Dibuixant Formes

Dibuixant Rectangles

// Dibuixar un rectangle ple
ctx.fillStyle = 'green';
ctx.fillRect(10, 10, 150, 100);

// Dibuixar un rectangle amb contorn
ctx.strokeStyle = 'blue';
ctx.strokeRect(200, 10, 150, 100);
  • fillStyle estableix el color per omplir el rectangle.
  • fillRect(x, y, width, height) dibuixa un rectangle ple.
  • strokeStyle estableix el color per al contorn del rectangle.
  • strokeRect(x, y, width, height) dibuixa el contorn d'un rectangle.

Dibuixant Camins

// Començar un nou camí
ctx.beginPath();
ctx.moveTo(50, 50); // Mou-te al punt d'inici
ctx.lineTo(200, 50); // Dibuixa una línia fins al següent punt
ctx.lineTo(200, 200); // Dibuixa una altra línia
ctx.closePath(); // Tanca el camí
ctx.stroke(); // Dibuixa el contorn del camí
  • beginPath() comença un nou camí.
  • moveTo(x, y) mou el punt d'inici.
  • lineTo(x, y) dibuixa una línia fins al punt especificat.
  • closePath() tanca el camí.
  • stroke() dibuixa el contorn del camí.

Tècniques Avançades

Dibuixant Cercles i Arcs

ctx.beginPath();
ctx.arc(150, 150, 75, 0, 2 * Math.PI); // Dibuixa un cercle
ctx.fillStyle = 'red';
ctx.fill();
ctx.stroke();
  • arc(x, y, radius, startAngle, endAngle) dibuixa un arc o cercle.

Dibuixant Text

ctx.font = '30px Arial';
ctx.fillStyle = 'purple';
ctx.fillText('Hola Canvas', 50, 250);
ctx.strokeText('Hola Canvas', 50, 300);
  • font estableix les propietats de la font.
  • fillText(text, x, y) dibuixa text ple.
  • strokeText(text, x, y) dibuixa el contorn del text.

Treballant amb Imatges

const img = new Image();
img.onload = function() {
    ctx.drawImage(img, 0, 0, 300, 200);
};
img.src = 'path/to/image.jpg';
  • drawImage(image, x, y, width, height) dibuixa una imatge en el canvas.

Conclusió

L'element <canvas> és una eina versàtil en HTML5 per crear gràfics dinàmics. Entenent els conceptes bàsics de l'element canvas, dibuixant formes, camins, text i imatges, pots crear una àmplia gamma de contingut visual. A mesura que avancis, pots explorar característiques més complexes com animacions, transformacions i gràfics interactius per millorar les teves aplicacions web.
