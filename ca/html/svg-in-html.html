Introducció a SVG

Scalable Vector Graphics (SVG) és un format basat en XML per crear imatges vectorials. A diferència de les imatges raster (per exemple, JPEG, PNG), les imatges SVG es poden escalar a qualsevol mida sense perdre qualitat, cosa que les fa ideals per a gràfics web.

Conceptes Clau

  • Gràfics Vectorials: Imatges definides per punts, línies, corbes i formes, que es poden escalar sense pèrdua de qualitat.
  • Basat en XML: Els fitxers SVG estan escrits en XML, cosa que els fa llegibles tant per humans com per màquines.
  • Interactiu i Animable: Els elements SVG es poden estilitzar, animar i fer interactius utilitzant CSS i JavaScript.

Estructura Bàsica de SVG

Els elements SVG es poden incrustar directament en documents HTML utilitzant l'etiqueta <svg>.

Exemple: Cercle SVG Simple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple SVG</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    </svg>
</body>
</html>

Explicació

  • <svg width="100" height="100">: Defineix el contenidor SVG amb una amplada i alçada de 100 unitats.
  • <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />: Dibuixa un cercle amb:
    • cx i cy: Coordenades del centre (50, 50).
    • r: Radi (40).
    • stroke: Color del contorn (negre).
    • stroke-width: Gruix del contorn (3 unitats).
    • fill: Color de farciment (vermell).

Elements Comuns de SVG

  • <rect>: Dibuixa rectangles.
  • <circle>: Dibuixa cercles.
  • <ellipse>: Dibuixa el·lipses.
  • <line>: Dibuixa línies rectes.
  • <polyline>: Dibuixa una sèrie de línies connectades.
  • <polygon>: Dibuixa una forma tancada amb múltiples costats.
  • <path>: Dibuixa formes complexes utilitzant una sèrie de comandes.

Exemple: Diversos Elements SVG

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Elements SVG</title>
</head>
<body>
    <svg width="200" height="200">
        <rect x="10" y="10" width="50" height="50" stroke="black" stroke-width="2" fill="blue" />
        <circle cx="100" cy="50" r="30" stroke="black" stroke-width="2" fill="green" />
        <ellipse cx="150" cy="150" rx="30" ry="20" stroke="black" stroke-width="2" fill="yellow" />
        <line x1="10" y1="150" x2="50" y2="190" stroke="black" stroke-width="2" />
        <polyline points="60,150 80,170 100,150" stroke="black" stroke-width="2" fill="none" />
        <polygon points="110,150 130,170 150,150" stroke="black" stroke-width="2" fill="purple" />
        <path d="M 160 150 Q 180 170 200 150" stroke="black" stroke-width="2" fill="none" />
    </svg>
</body>
</html>

Estilitzar SVG amb CSS

Els elements SVG es poden estilitzar utilitzant CSS, igual que els elements HTML.

Exemple: Estilitzar SVG amb CSS

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SVG Estilitzat</title>
    <style>
        .my-circle {
            fill: orange;
            stroke: blue;
            stroke-width: 4;
        }
    </style>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" class="my-circle" />
    </svg>
</body>
</html>

Explicació

  • La classe .my-circle està definida a la secció <style>.
  • L'atribut class="my-circle" s'afegeix a l'element <circle> per aplicar els estils.

Animar SVG

Els elements SVG es poden animar utilitzant l'etiqueta <animate> o animacions CSS.

Exemple: Animació SVG Simple

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Animació SVG</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red">
            <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />
        </circle>
    </svg>
</body>
</html>

Explicació

  • <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />: Anima l'atribut cx del cercle de 50 a 150 durant 2 segons, repetint indefinidament.

Conclusió

SVG és una eina poderosa per crear gràfics escalables, interactius i animats a HTML. Entenent l'estructura bàsica, els elements comuns, l'estilització i les tècniques d'animació, pots millorar els teus projectes web amb gràfics vectorials d'alta qualitat.
