Introducció a HTML

HTML (HyperText Markup Language) és el llenguatge estàndard utilitzat per crear i dissenyar documents a la World Wide Web. Estructura el contingut web i permet als navegadors mostrar text, imatges i altres elements en un format llegible.

Conceptes clau d'HTML

  • Elements: Els blocs de construcció d'HTML, definits per etiquetes.
  • Etiquetes: Paraules clau envoltades per claudàtors angulats, per exemple, <tagname>.
  • Atributs: Proporcionen informació addicional sobre els elements, normalment en forma de parelles nom/valor.
  • Document Object Model (DOM): Una estructura en forma d'arbre que representa el document HTML.

Estructura bàsica d'un document HTML

Un document HTML té una estructura específica que inclou diversos elements clau:

<!DOCTYPE html>
<html>
<head>
    <title>Títol de la pàgina</title>
</head>
<body>
    <h1>Això és un Encapçalament</h1>
    <p>Això és un paràgraf.</p>
</body>
</html>

Explicació de l'estructura bàsica

  • <!DOCTYPE html>: Declara el tipus de document i la versió d'HTML.
  • <html>: L'element arrel d'una pàgina HTML.
  • <head>: Conté meta-informació sobre el document HTML, com el títol i enllaços a fulls d'estil.
  • <title>: Estableix el títol de la pàgina web, que es mostra a la barra de títol o a la pestanya del navegador.
  • <body>: Conté el contingut del document HTML, com text, imatges i altres mitjans.

Elements comuns d'HTML

Aquí hi ha alguns elements HTML comunament utilitzats:

  • Encapçalaments: Definits amb les etiquetes <h1> a <h6>.
  • Paràgrafs: Definits amb l'etiqueta <p>.
  • Enllaços: Definits amb l'etiqueta <a>.
  • Imatges: Definides amb l'etiqueta <img>.
  • Llistes: Llistes ordenades (<ol>) i llistes desordenades (<ul>).

Exemples d'elements HTML comuns

Encapçalaments

<h1>Això és un Encapçalament 1</h1>
<h2>Això és un Encapçalament 2</h2>

Paràgrafs

<p>Això és un paràgraf de text.</p>

Enllaços

<a href="https://www.example.com">Això és un enllaç</a>

Imatges

<img src="image.jpg" alt="Descripció de la imatge">

Llistes

<ul>
    <li>Element 1</li>
    <li>Element 2</li>
    <li>Element 3</li>
</ul>

<ol>
    <li>Primer element</li>
    <li>Segon element</li>
    <li>Tercer element</li>
</ol>

Atributs en HTML

Els atributs proporcionen informació addicional sobre els elements HTML. Sempre s'inclouen a l'etiqueta d'obertura i normalment vénen en parelles nom/valor com name="value".

Exemple d'atributs

<a href="https://www.example.com" target="_blank">Visita Example</a>
<img src="image.jpg" alt="Descripció de la imatge" width="500" height="600">
  • href: Especifica l'URL de la pàgina a la qual va l'enllaç.
  • target: Especifica on obrir el document enllaçat.
  • src: Especifica el camí a la imatge.
  • alt: Proporciona text alternatiu per a la imatge.
  • width i height: Especifiquen les dimensions de la imatge.

Conclusió

HTML és el llenguatge fonamental per crear pàgines web. Entendre la seva estructura bàsica, elements i atributs és essencial per a qualsevol persona que vulgui construir o dissenyar llocs web. A mesura que avancis en aquest curs, aprendràs conceptes HTML més avançats i com integrar-los amb CSS i JavaScript per crear pàgines web dinàmiques i visualment atractives.
