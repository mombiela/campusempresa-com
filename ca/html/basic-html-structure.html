Introducció

HTML (HyperText Markup Language) és el llenguatge estàndard per crear pàgines web. Entendre l'estructura bàsica d'un document HTML és essencial per a qualsevol persona que vulgui construir llocs web. En aquesta secció, cobrirem els elements fonamentals que componen un document HTML.

Conceptes Clau

  • Estructura del Document HTML: L'esquelet bàsic d'un document HTML.
  • Etiquetes i Elements: Els blocs de construcció d'HTML.
  • Atributs: Informació addicional proporcionada dins de les etiquetes.
  • Niuament: Col·locar elements dins d'altres elements.

Estructura del Document HTML

Un document HTML està format per diversos components clau. Aquí teniu un exemple senzill d'un document HTML bàsic:

<!DOCTYPE html>
<html>
<head>
    <title>La Meva Primera Pàgina HTML</title>
</head>
<body>
    <h1>Benvingut al Meu Lloc Web</h1>
    <p>Aquest és un paràgraf de text a la meva primera pàgina HTML.</p>
</body>
</html>

Explicació

  • <!DOCTYPE html>: Aquesta declaració defineix el tipus de document i la versió d'HTML. Ajuda el navegador a entendre com renderitzar el contingut.
  • <html>: L'element arrel que embolcalla tot el contingut del document HTML.
  • <head>: Conté metainformació sobre el document, com ara el títol i enllaços a fulls d'estil.
  • <title>: Defineix el títol de la pàgina web, que apareix a la pestanya del navegador.
  • <body>: Conté el contingut del document HTML que és visible per a l'usuari, com ara text, imatges i enllaços.

Etiquetes i Elements

HTML utilitza etiquetes per crear elements. Les etiquetes estan tancades entre claudàtors angulars, i la majoria d'elements tenen una etiqueta d'obertura i una de tancament.

Etiquetes Comunes

  • <h1> a <h6>: Etiquetes de capçalera, amb <h1> sent el nivell més alt i <h6> el més baix.
  • <p>: Etiqueta de paràgraf per a blocs de text.
  • <a>: Etiqueta d'ancoratge per a hiperenllaços.
  • <img>: Etiqueta d'imatge per mostrar imatges.
  • <ul>, <ol>, <li>: Etiquetes per crear llistes (desordenades i ordenades).

Exemple

<h1>Això és una Capçalera</h1>
<p>Això és un paràgraf.</p>
<a href="https://www.example.com">Això és un enllaç</a>
<img src="image.jpg" alt="Descripció de la imatge">
<ul>
    <li>Primer element</li>
    <li>Segon element</li>
</ul>

Atributs

Els atributs proporcionen informació addicional sobre els elements HTML. Sempre s'inclouen a l'etiqueta d'obertura i normalment venen en parelles nom/valor com nom="valor".

Atributs Comuns

  • href: Utilitzat amb etiquetes <a> per especificar l'URL de l'enllaç.
  • src: Utilitzat amb etiquetes <img> per especificar el camí a la imatge.
  • alt: Utilitzat amb etiquetes <img> per proporcionar text alternatiu per a la imatge.
  • class: Utilitzat per definir una classe per a un element, que es pot utilitzar per estilitzar amb CSS.
  • id: Utilitzat per identificar de manera única un element.

Exemple

<a href="https://www.example.com" target="_blank">Visita Example</a>
<img src="image.jpg" alt="Un paisatge bell" width="500" height="300">
<p class="intro">Aquest és un paràgraf introductori.</p>
<div id="main-content">Aquesta és l'àrea de contingut principal.</div>

Niuament

El niuament es refereix a col·locar elements HTML dins d'altres elements. Un niuament adequat és crucial per crear documents HTML ben estructurats i llegibles.

Exemple

<div>
    <h2>Títol de la Secció</h2>
    <p>Aquest és un paràgraf dins d'un div.</p>
    <ul>
        <li>Element de llista niuat 1</li>
        <li>Element de llista niuat 2</li>
    </ul>
</div>

Resum

Entendre l'estructura bàsica d'un document HTML és el primer pas en el desenvolupament web. Dominant l'ús d'etiquetes, elements, atributs i niuament, podeu crear pàgines web ben estructurades i significatives. A mesura que avanceu, aquestes habilitats fonamentals seran essencials per a temes més avançats d'HTML i desenvolupament web.
