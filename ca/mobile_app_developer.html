El Projecte Sobre nosaltres Contribuir Donacions Llicència

Dissenya, construeix i manté aplicacions mòbils per a plataformes iOS i Android utilitzant tecnologies natives o multiplataforma.

Competències tècniques

Swift
Llenguatge de programació desenvolupat per Apple per a la creació d'aplicacions en iOS i macOS.
Kotlin
Llenguatge de programació modern i concís per al desenvolupament d'aplicacions Android, compatible amb Java.
Java
Llenguatge de programació àmpliament utilitzat per al desenvolupament d'aplicacions empresarials i mòbils.
Flutter
Framework de desenvolupament d'aplicacions multiplataforma creat per Google, que permet crear aplicacions per a iOS i Android amb un sol codi.
React Native
Framework basat en React per al desenvolupament d'aplicacions mòbils multiplataforma utilitzant JavaScript.
Xcode
Entorn de desenvolupament integrat (IDE) utilitzat per crear aplicacions en iOS i macOS, desenvolupat per Apple.
Android Studio
Entorn de desenvolupament integrat (IDE) oficial de Google per a la creació d'aplicacions Android.
Apache Cordova
Framework de desenvolupament mòbil que permet crear aplicacions multiplataforma utilitzant tecnologies web estàndard com HTML, CSS i JavaScript, empaquetant-les per a la seva execució en dispositius mòbils.
Ionic
Framework de desenvolupament d'aplicacions mòbils híbrides que utilitza tecnologies web com HTML, CSS i JavaScript, i permet el desplegament en múltiples plataformes com iOS i Android.
Firebase
Plataforma de desenvolupament d'aplicacions mòbils de Google que proporciona serveis backend com autenticació, emmagatzematge al núvol, bases de dades en temps real i notificacions push.

Coneixements

Fonaments de la Programació
Curs introductori sobre conceptes bàsics de la programació, com variables, estructures de control, i algorismes.
Algorismes
Curs sobre anàlisi i disseny d'algorismes, que cobreix l'eficiència dels algorismes i com millorar el rendiment del codi.
Estructures de Dades
Introducció a les estructures de dades comunes com llistes, piles, cues, arbres i grafs, essencials per a programadors.
Patrons de Disseny
Curs sobre patrons de disseny de programari que proporcionen solucions reutilitzables a problemes comuns de desenvolupament.
REST API
Principis de disseny i desenvolupament d'APIs RESTful, utilitzades per a la interacció entre aplicacions.
CI/CD
Integració i desplegament continu (CI/CD), pràctiques per automatitzar la construcció, proves i lliurament de programari.

Competències socials

Creativitat
Capacitat per proposar idees innovadores i solucions originals als problemes.
Resolució de problemes
Capacitat per identificar problemes i desenvolupar solucions eficients i efectives.
Adaptabilitat
Capacitat per ajustar-se ràpidament a noves circumstàncies, tecnologies o mètodes de treball.
