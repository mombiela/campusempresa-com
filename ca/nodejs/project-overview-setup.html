Introducció

En aquesta secció, cobrirem els passos essencials per configurar un projecte Node.js. Això inclou instal·lar Node.js, inicialitzar un projecte i entendre l'estructura bàsica d'una aplicació Node.js. Al final d'aquesta secció, tindràs un entorn Node.js funcionant i una estructura bàsica de projecte sobre la qual construir.

Conceptes clau

  • Node.js: Un entorn d'execució de JavaScript construït sobre el motor de JavaScript V8 de Chrome.
  • npm: Node Package Manager, utilitzat per gestionar les dependències del projecte.
  • package.json: Un fitxer que conté diverses metadades rellevants per al projecte i s'utilitza per gestionar les dependències, scripts, versió i més del projecte.

Instal·lació de Node.js

Per començar a treballar amb Node.js, necessites instal·lar-lo a la teva màquina. Segueix aquests passos:

  1. Descarrega Node.js: Visita el lloc web oficial de Node.js i descarrega la versió LTS (Long Term Support).
  2. Instal·la Node.js: Executa l'instal·lador i segueix les instruccions. Això també instal·larà npm.

Verificar la instal·lació

Obre el teu terminal o símbol del sistema i executa les següents comandes per verificar la instal·lació:

node -v

Això hauria de mostrar la versió de Node.js instal·lada.

npm -v

Això hauria de mostrar la versió de npm instal·lada.

Inicialització d'un projecte Node.js

Un cop Node.js està instal·lat, pots inicialitzar un nou projecte. Segueix aquests passos:

  1. Crea un directori de projecte: Navega al directori on vols crear el teu projecte i crea una nova carpeta.
mkdir my-node-project
cd my-node-project
  1. Inicialitza el projecte: Utilitza npm per inicialitzar el projecte. Això crearà un fitxer package.json.
npm init

Se't demanarà que introdueixis diversos detalls sobre el teu projecte. Pots prémer Enter per acceptar els valors per defecte o proporcionar els teus propis.

Exemple de `package.json`

Aquí tens un exemple d'un fitxer package.json bàsic:

{
  "name": "my-node-project",
  "version": "1.0.0",
  "description": "Un projecte Node.js simple",
  "main": "index.js",
  "scripts": {
    "start": "node index.js"
  },
  "author": "El teu nom",
  "license": "ISC"
}

Estructura del projecte

Una estructura típica de projecte Node.js podria semblar així:

my-node-project/
├── node_modules/
├── package.json
├── index.js
└── README.md
  • node_modules/: Aquesta carpeta conté totes les dependències instal·lades via npm.
  • package.json: Aquest fitxer conté metadades sobre el projecte i les seves dependències.
  • index.js: El punt d'entrada principal de l'aplicació.
  • README.md: Un fitxer markdown que proporciona una visió general del projecte.

Creació de `index.js`

Crea un fitxer index.js al directori del teu projecte. Aquest serà el punt d'entrada de la teva aplicació.

// index.js
console.log("Hola, Node.js!");

Executar l'aplicació

Per executar la teva aplicació Node.js, utilitza la següent comanda:

npm start

Això executarà el script start definit al teu package.json, que executa node index.js.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per configurar un projecte Node.js. Has après com instal·lar Node.js, inicialitzar un projecte amb npm i crear una estructura bàsica de projecte. Amb aquesta base, ara estàs preparat per començar a construir aplicacions Node.js més complexes. En les següents seccions, aprofundirem en les característiques de Node.js i les millors pràctiques.
