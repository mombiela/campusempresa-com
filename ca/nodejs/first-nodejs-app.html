Introducció

Benvingut a la teva primera aplicació Node.js! En aquesta secció, et guiarem a través de la configuració d'un entorn bàsic de Node.js i la creació d'una aplicació senzilla. Al final d'aquesta lliçó, tindràs una base sòlida per construir aplicacions Node.js més complexes.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Node.js instal·lat al teu ordinador. Pots descarregar-lo des de nodejs.org.
  • Un editor de codi com Visual Studio Code, Sublime Text, o qualsevol altre de la teva elecció.

Configurant el Teu Entorn

  1. Instal·la Node.js: Segueix les instruccions al lloc web de Node.js per descarregar i instal·lar Node.js.
  2. Verifica la Instal·lació: Obre el teu terminal o símbol del sistema i escriu: 
    node -v
    Aquesta comanda hauria de mostrar la versió de Node.js instal·lada al teu sistema.

Creant la Teva Primera Aplicació Node.js

Creem una aplicació senzilla de "Hola, Món!" per començar.

Pas 1: Crear un Directori de Projecte

Primer, crea un nou directori per al teu projecte i navega dins d'ell:

mkdir my-first-node-app
cd my-first-node-app

Pas 2: Inicialitzar un Projecte Node.js

Inicialitza el teu projecte amb npm (Node Package Manager):

npm init -y

Aquesta comanda crearà un fitxer package.json amb la configuració per defecte.

Pas 3: Crear el Fitxer de l'Aplicació

Crea un nou fitxer anomenat app.js al teu directori de projecte. Aquest serà el fitxer principal de la teva aplicació.

Pas 4: Escriure el Teu Primer Codi Node.js

Obre app.js al teu editor de codi i afegeix el següent codi:

// app.js
console.log('Hola, Món!');

Pas 5: Executar la Teva Aplicació

Torna al teu terminal i executa l'aplicació utilitzant la següent comanda:

node app.js

Hauries de veure la sortida:

Hola, Món!

Entenent el Codi

  • console.log('Hola, Món!');: Aquesta línia de codi imprimeix la cadena 'Hola, Món!' a la consola. L'objecte console en Node.js proporciona una consola de depuració senzilla que és similar a la consola de JavaScript en els navegadors web.

Afegint Més Funcionalitat

Anem a estendre la nostra aplicació per crear un servidor HTTP senzill que respongui amb "Hola, Món!" quan s'accedeixi a través d'un navegador web.

Pas 1: Importar el Mòdul HTTP

Actualitza el teu fitxer app.js per incloure el mòdul HTTP:

const http = require('http');

Pas 2: Crear el Servidor

Afegeix el següent codi per crear un servidor HTTP:

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hola, Món!\n');
});

Pas 3: Escoltar en un Port

Afegeix el següent codi per fer que el servidor escolti en el port 3000:

const port = 3000;
server.listen(port, () => {
  console.log(`Servidor funcionant a http://localhost:${port}/`);
});

Exemple de Codi Complet

Aquí tens el fitxer complet app.js:

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hola, Món!\n');
});

const port = 3000;
server.listen(port, () => {
  console.log(`Servidor funcionant a http://localhost:${port}/`);
});

Executant el Servidor

Executa la teva aplicació de nou:

node app.js

Obre el teu navegador web i navega a http://localhost:3000/. Hauries de veure "Hola, Món!" mostrat al teu navegador.

Resum

Felicitats! Has creat la teva primera aplicació Node.js. En aquesta lliçó, has après a:

  • Configurar un projecte Node.js.
  • Escriure una aplicació senzilla de "Hola, Món!".
  • Crear un servidor HTTP utilitzant Node.js.

Aquest coneixement fonamental serà crucial a mesura que avancis cap a temes més avançats en Node.js. Continua experimentant i construint sobre el que has après!
