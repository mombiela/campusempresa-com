Introducció a Docker

  • Què és Docker?

    • Docker és una plataforma per desenvolupar, enviar i executar aplicacions dins de contenidors.
    • Els contenidors són lleugers, portàtils i garanteixen la consistència a través de diferents entorns.

  • Per què utilitzar Docker amb Node.js?

    • Simplifica el procés de configuració.
    • Garanteix la consistència a través dels entorns de desenvolupament, proves i producció.
    • Facilita la gestió de dependències i versions.

Configurant Docker

  • Instal·lant Docker

    • Segueix la guia oficial d'instal·lació de Docker per al teu sistema operatiu: Docker Installation

  • Comandes bàsiques de Docker

    • docker --version: Comprovar la versió de Docker.
    • docker run: Executar un contenidor.
    • docker build: Construir una imatge a partir d'un Dockerfile.
    • docker ps: Llistar els contenidors en execució.
    • docker stop: Aturar un contenidor en execució.

Creant una Aplicació Simple de Node.js

  • Aplicació bàsica de Node.js

    // app.js
const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

  • Package.json

    {
  "name": "docker-nodejs-app",
  "version": "1.0.0",
  "description": "A simple Node.js app to demonstrate Docker",
  "main": "app.js",
  "scripts": {
    "start": "node app.js"
  },
  "dependencies": {
    "http": "^0.0.1-security"
  }
}

Dockeritzant l'Aplicació de Node.js

  • Creant un Dockerfile

    # Utilitza la imatge oficial de Node.js del Docker Hub
FROM node:14

# Crear i canviar al directori de l'aplicació
WORKDIR /usr/src/app

# Copiar package.json i package-lock.json
COPY package*.json ./

# Instal·lar dependències
RUN npm install

# Copiar la resta del codi de l'aplicació
COPY . .

# Exposar el port en què s'executa l'aplicació
EXPOSE 3000

# Definir la comanda per executar l'aplicació
CMD ["node", "app.js"]

  • Construint la Imatge de Docker

    docker build -t docker-nodejs-app .

  • Executant el Contenidor de Docker

    docker run -p 3000:3000 docker-nodejs-app

Gestionant Contenidors de Docker

  • Veient Contenidors en Execució

    docker ps

  • Aturant un Contenidor

    docker stop <container_id>

  • Eliminant un Contenidor

    docker rm <container_id>

Conceptes Avançats de Docker

  • Docker Compose

    • Docker Compose és una eina per definir i executar aplicacions Docker amb múltiples contenidors.

    • docker-compose.yml

      version: '3'
services:
  app:
    build: .
    ports:
      - "3000:3000"

    • Executant Docker Compose

      docker-compose up

  • Volums de Docker

    • Els volums s'utilitzen per persistir dades generades i utilitzades pels contenidors de Docker.

    • Creant un Volum

      docker volume create my_volume

    • Utilitzant un Volum en Docker Compose

      version: '3'
services:
  app:
    build: .
    ports:
      - "3000:3000"
    volumes:
      - my_volume:/usr/src/app

Conclusió

Utilitzar Docker amb Node.js simplifica el procés de desenvolupament i desplegament garantint la consistència a través de diferents entorns. Dockeritzant les teves aplicacions de Node.js, pots gestionar fàcilment les dependències, escalar les teves aplicacions i mantenir una configuració de desenvolupament neta i reproduïble. Docker Compose millora encara més això permetent-te gestionar aplicacions amb múltiples contenidors amb facilitat.
