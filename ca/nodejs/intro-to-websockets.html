Els WebSockets proporcionen un canal de comunicació de doble sentit sobre una connexió única i de llarga durada. Això és particularment útil per a aplicacions en temps real com aplicacions de xat, notificacions en viu i jocs en línia. En aquesta secció, explorarem els WebSockets en el context de Node.js, des dels conceptes bàsics fins als més avançats.

Què són els WebSockets?

Els WebSockets són un protocol que permet la comunicació bidireccional entre un client i un servidor. A diferència de l'HTTP, que es basa en petició-resposta, els WebSockets permeten connexions persistents, permetent que les dades es puguin enviar i rebre en qualsevol moment.

Conceptes Clau

  • Comunicació de Doble Sentit: Tant el client com el servidor poden enviar i rebre missatges simultàniament.
  • Connexió Persistent: La connexió roman oberta, reduint la sobrecàrrega d'establir múltiples connexions HTTP.
  • Baixa Latència: Ideal per a aplicacions en temps real gràcies a la latència reduïda.

Configuració dels WebSockets en Node.js

Per utilitzar WebSockets en Node.js, utilitzarem la biblioteca ws, una implementació popular de WebSocket.

Instal·lació de la Biblioteca `ws`

Primer, instal·la la biblioteca ws utilitzant npm:

npm install ws

Creació d'un Servidor WebSocket Bàsic

Creem un servidor WebSocket senzill que escolti connexions i retorni qualsevol missatge que rebi.

const WebSocket = require('ws');

const server = new WebSocket.Server({ port: 8080 });

server.on('connection', (socket) => {
  console.log('Client connected');

  socket.on('message', (message) => {
    console.log(`Received: ${message}`);
    socket.send(`Echo: ${message}`);
  });

  socket.on('close', () => {
    console.log('Client disconnected');
  });
});

console.log('WebSocket server is running on ws://localhost:8080');

Explicació

  • WebSocket.Server: Crea un nou servidor WebSocket escoltant al port 8080.
  • server.on('connection'): Oient d'esdeveniments per a noves connexions de clients.
  • socket.on('message'): Oient d'esdeveniments per a missatges del client.
  • socket.send(): Envia un missatge de tornada al client.
  • socket.on('close'): Oient d'esdeveniments per quan el client es desconnecta.

Creació d'un Client WebSocket

Ara, creem un client WebSocket senzill que es connecti al nostre servidor i enviï un missatge.

Codi del Client

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Client WebSocket</title>
</head>
<body>
  <script>
    const socket = new WebSocket('ws://localhost:8080');

    socket.onopen = () => {
      console.log('Connected to server');
      socket.send('Hello Server!');
    };

    socket.onmessage = (event) => {
      console.log(`Received from server: ${event.data}`);
    };

    socket.onclose = () => {
      console.log('Disconnected from server');
    };
  </script>
</body>
</html>

Explicació

  • new WebSocket('ws://localhost:8080'): Crea una nova connexió WebSocket al servidor.
  • socket.onopen: Oient d'esdeveniments per quan la connexió s'estableix.
  • socket.send(): Envia un missatge al servidor.
  • socket.onmessage: Oient d'esdeveniments per a missatges del servidor.
  • socket.onclose: Oient d'esdeveniments per quan la connexió es tanca.

Funcionalitats Avançades de WebSocket

Emissió de Missatges

Per emetre un missatge a tots els clients connectats, itera sobre els clients i envia el missatge.

server.on('connection', (socket) => {
  socket.on('message', (message) => {
    server.clients.forEach((client) => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(`Broadcast: ${message}`);
      }
    });
  });
});

Gestió de Dades Binàries

Els WebSockets també poden gestionar dades binàries. Aquí tens un exemple d'enviament i recepció de dades binàries.

// Costat del servidor
socket.on('message', (data) => {
  if (data instanceof Buffer) {
    console.log('Received binary data');
    // Processa les dades binàries
  }
});

// Costat del client
const buffer = new ArrayBuffer(8);
const view = new DataView(buffer);
view.setInt32(0, 12345678, true);
socket.send(buffer);
