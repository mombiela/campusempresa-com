Socket.io és una biblioteca potent que permet la comunicació en temps real, bidireccional i basada en esdeveniments entre clients web i servidors. Aquest tema et guiarà a través de l'ús de Socket.io en un entorn Node.js, des de la configuració bàsica fins a funcions avançades.

Introducció a Socket.io

  • Què és Socket.io?

    • Una biblioteca JavaScript per a aplicacions web en temps real.
    • Permet la comunicació en temps real, bidireccional entre clients web i servidors.
    • Construït sobre WebSockets, però proporciona funcions addicionals i alternatives per a navegadors antics.

  • Per què utilitzar Socket.io?

    • Actualitzacions en temps real (per exemple, aplicacions de xat, notificacions en viu).
    • Fàcil d'implementar i utilitzar.
    • Robusta i fiable amb reconnexió automàtica.

Configurant Socket.io

Instal·lant Socket.io

Per començar amb Socket.io, necessites instal·lar-lo via npm.

npm install socket.io

Configuració Bàsica del Servidor

Crea un servidor bàsic de Node.js amb Socket.io.

// server.js
const express = require('express');
const http = require('http');
const socketIo = require('socket.io');

const app = express();
const server = http.createServer(app);
const io = socketIo(server);

io.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat');
  
  socket.on('disconnect', () => {
    console.log('Usuari desconnectat');
  });
});

server.listen(3000, () => {
  console.log('El servidor està funcionant al port 3000');
});

Configuració Bàsica del Client

Crea un fitxer HTML senzill per connectar-te al servidor Socket.io.

<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
  <title>Exemple de Socket.io</title>
  <script src="/socket.io/socket.io.js"></script>
  <script>
    document.addEventListener('DOMContentLoaded', (event) => {
      const socket = io();

      socket.on('connect', () => {
        console.log('Connectat al servidor');
      });

      socket.on('disconnect', () => {
        console.log('Desconnectat del servidor');
      });
    });
  </script>
</head>
<body>
  <h1>Exemple de Socket.io</h1>
</body>
</html>

Emissió i Escolta d'Esdeveniments

Emissió des del Servidor

Emet esdeveniments des del servidor al client.

io.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat');

  // Emet un esdeveniment al client
  socket.emit('message', 'Hola des del servidor');

  socket.on('disconnect', () => {
    console.log('Usuari desconnectat');
  });
});

Escolta al Client

Escolta esdeveniments al costat del client.

<script>
  document.addEventListener('DOMContentLoaded', (event) => {
    const socket = io();

    socket.on('connect', () => {
      console.log('Connectat al servidor');
    });

    // Escolta l'esdeveniment 'message'
    socket.on('message', (data) => {
      console.log(data);
    });

    socket.on('disconnect', () => {
      console.log('Desconnectat del servidor');
    });
  });
</script>

Emissió des del Client

Emet esdeveniments des del client al servidor.

<script>
  document.addEventListener('DOMContentLoaded', (event) => {
    const socket = io();

    socket.on('connect', () => {
      console.log('Connectat al servidor');

      // Emet un esdeveniment al servidor
      socket.emit('clientMessage', 'Hola des del client');
    });

    socket.on('disconnect', () => {
      console.log('Desconnectat del servidor');
    });
  });
</script>

Escolta al Servidor

Escolta esdeveniments al costat del servidor.

io.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat');

  // Escolta l'esdeveniment 'clientMessage'
  socket.on('clientMessage', (data) => {
    console.log(data);
  });

  socket.on('disconnect', () => {
    console.log('Usuari desconnectat');
  });
});

Emissió a Tots els Clients

L'emissió permet enviar un missatge a tots els clients connectats excepte l'emissor.

Emissió des del Servidor

io.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat');

  socket.on('clientMessage', (data) => {
    // Emet el missatge a tots els altres clients
    socket.broadcast.emit('message', data);
  });

  socket.on('disconnect', () => {
    console.log('Usuari desconnectat');
  });
});

Sales i Espais de Noms

Utilitzant Sales

Les sales permeten unir clients en grups per a missatgeria dirigida.

io.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat');

  socket.join('room1');

  socket.on('clientMessage', (data) => {
    // Envia el missatge a tots els clients de 'room1'
    io.to('room1').emit('message', data);
  });

  socket.on('disconnect', () => {
    console.log('Usuari desconnectat');
  });
});

Utilitzant Espais de Noms

Els espais de noms permeten crear canals de comunicació separats.

const chat = io.of('/chat');

chat.on('connection', (socket) => {
  console.log('Un usuari s\'ha connectat a l\'espai de noms de xat');

  socket.on('clientMessage', (data) => {
    chat.emit('message', data);
  });

  socket.on('disconnect', () => {
    console.log('Usuari desconnectat de l\'espai de noms de xat');
  });
});

Funcions Avançades

Middleware

Les funcions middleware es poden utilitzar per gestionar l'autenticació i altres tasques de pre-processament.

io.use((socket, next) => {
  const token = socket.handshake.query.token;
  if (isValidToken(token)) {
    return next();
  }
  return next(new Error('error d\'autenticació'));
});

Gestió d'Errors

Gestiona els errors de manera elegant a la teva aplicació Socket.io.

io.on('connection', (socket) => {
  socket.on('error', (err) => {
    console.error('Error de socket:', err);
  });
});

Conclusió

Socket.io és una biblioteca versàtil i potent per afegir capacitats en temps real a les teves aplicacions Node.js. Entenent els conceptes bàsics de la configuració d'un servidor i client Socket.io, emetent i escoltant esdeveniments, emetent a tots els clients, i utilitzant sales i espais de noms, pots construir aplicacions en temps real complexes. Les funcions avançades com el middleware i la gestió d'errors milloren encara més la robustesa i seguretat de les teves aplicacions. Amb aquestes eines a la teva disposició, estàs ben preparat per crear aplicacions web dinàmiques i interactives.
