Introducció

Desplegar una aplicació Node.js implica moure la teva aplicació d'un entorn de desenvolupament a un entorn de producció on els usuaris hi puguin accedir. Aquesta secció et guiarà a través dels passos i les millors pràctiques per desplegar una aplicació Node.js.

Preparant la Teva Aplicació per al Desplegament

Abans de desplegar, assegura't que la teva aplicació està llesta per a la producció.

  • Variables d'Entorn: Utilitza variables d'entorn per gestionar les configuracions.
  • Gestió d'Errors: Implementa una gestió d'errors robusta per gestionar problemes inesperats.
  • Registre: Configura el registre per monitoritzar la teva aplicació.
  • Seguretat: Assegura la teva aplicació abordant vulnerabilitats comunes.

Exemple: Utilitzant Variables d'Entorn

// config.js
module.exports = {
  port: process.env.PORT || 3000,
  dbConnectionString: process.env.DB_CONNECTION_STRING,
};

// app.js
const config = require('./config');
const express = require('express');
const app = express();

app.listen(config.port, () => {
  console.log(`Servidor funcionant al port ${config.port}`);
});

Triant un Proveïdor d'Allotjament

Hi ha diversos proveïdors d'allotjament que pots triar, cadascun amb el seu conjunt de característiques.

  • Heroku: Fàcil d'utilitzar, ideal per a principiants.
  • AWS (Amazon Web Services): Altament escalable, adequat per a aplicacions grans.
  • DigitalOcean: Assequible i senzill.
  • Google Cloud Platform: Ofereix una àmplia gamma de serveis i integracions.

| Proveïdor | Facilitat d'Ús | Escalabilitat | Cost | |----------------|-----------------|---------------|------------| | Heroku | Alta | Mitjana | Moderat | | AWS | Mitjana | Alta | Variable | | DigitalOcean | Mitjana | Mitjana | Assequible | | Google Cloud | Mitjana | Alta | Variable |

Desplegant a Heroku

Heroku és una opció popular per desplegar aplicacions Node.js degut a la seva simplicitat.

Passos per Desplegar

  1. Instal·la Heroku CLI: Descarrega i instal·la la Interfície de Línia de Comandes de Heroku.
  2. Inicia sessió a Heroku: Autentica el teu compte de Heroku.
  3. Crea una Aplicació Heroku: Inicialitza una nova aplicació Heroku.
  4. Desplega el teu Codi: Puja el teu codi a Heroku.

Exemple: Desplegant a Heroku

# Instal·la Heroku CLI
$ curl https://cli-assets.heroku.com/install.sh | sh

# Inicia sessió a Heroku
$ heroku login

# Crea una nova aplicació Heroku
$ heroku create my-node-app

# Desplega el teu codi
$ git push heroku main

Desplegant a AWS

AWS proporciona una infraestructura robusta per desplegar aplicacions escalables.

Passos per Desplegar

  1. Configura una Instància EC2: Crea i configura una instància EC2.
  2. Instal·la Node.js: Instal·la Node.js a la teva instància EC2.
  3. Transfereix el teu Codi: Utilitza SCP o Git per transferir el teu codi.
  4. Executa la Teva Aplicació: Inicia la teva aplicació Node.js.

Exemple: Desplegant a AWS EC2

# Connecta't a la teva instància EC2
$ ssh -i "your-key.pem" ec2-user@your-ec2-ip

# Instal·la Node.js
$ sudo yum update -y
$ curl -sL https://rpm.nodesource.com/setup_14.x | sudo bash -
$ sudo yum install -y nodejs

# Transfereix el teu codi
$ scp -i "your-key.pem" -r ./your-app ec2-user@your-ec2-ip:~/your-app

# Inicia la teva aplicació
$ cd your-app
$ npm install
$ node app.js

Millors Pràctiques per a Producció

  • Utilitza un Gestor de Processos: Eines com PM2 poden gestionar els processos de la teva aplicació.
  • Activa la Memòria Cau: Utilitza mecanismes de memòria cau per millorar el rendiment.
  • Monitoritza el Rendiment: Utilitza eines de monitorització per mantenir el seguiment de la salut de la teva aplicació.
  • Automatitza els Desplegaments: Utilitza pipelines de CI/CD per a desplegaments automatitzats.

Exemple: Utilitzant PM2

# Instal·la PM2 globalment
$ npm install pm2 -g

# Inicia la teva aplicació amb PM2
$ pm2 start app.js

# Configura PM2 per iniciar-se en l'arrencada del sistema
$ pm2 startup
$ pm2 save

Conclusió

Desplegar una aplicació Node.js implica diversos passos, des de preparar la teva aplicació fins a triar un proveïdor d'allotjament i seguir les millors pràctiques per a producció. Entenent aquests passos i utilitzant les eines adequades, pots assegurar un procés de desplegament suau i eficient.
