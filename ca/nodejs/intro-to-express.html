Introducció a Express.js
Configuració d'una Aplicació Express ►

Express.js és un framework minimalista i flexible per a aplicacions web de Node.js que proporciona un conjunt robust de característiques per desenvolupar aplicacions web i mòbils. Facilita el desenvolupament ràpid d'aplicacions web basades en Node.

Què és Express.js?

Express.js és un framework d'aplicacions web per a Node.js, dissenyat per construir aplicacions web i APIs. És el framework de servidor estàndard de facto per a Node.js.

  • Minimalista: Express és un framework minimalista que proporciona una capa fina de característiques fonamentals per a aplicacions web.
  • Flexible: És altament flexible i permet als desenvolupadors estructurar les seves aplicacions com ho considerin oportú.
  • Robust: Express proporciona un conjunt robust de característiques per a aplicacions web i mòbils.

Configuració d'Express.js

Per començar a utilitzar Express.js, necessites tenir Node.js instal·lat a la teva màquina. Pots instal·lar Express.js utilitzant npm (Node Package Manager).

Passos d'Instal·lació

  1. Inicialitza un nou projecte de Node.js:

    mkdir myapp
cd myapp
npm init -y

  2. Instal·la Express.js:

    npm install express

Creació d'una Aplicació Bàsica amb Express.js

Creem una aplicació senzilla amb Express.js per entendre la seva estructura bàsica.

Estructura Bàsica de l'Aplicació

  1. Crea un fitxer anomenat app.js:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

  2. Executa l'aplicació:

    node app.js

  3. Accedeix a l'aplicació: Obre el teu navegador i navega a http://localhost:3000. Hauries de veure "Hello World!" mostrat.

Explicació del Codi

  • Importació d'Express:

    const express = require('express');

    Aquesta línia importa el mòdul Express.

  • Creació d'una aplicació Express:

    const app = express();

    Aquesta línia inicialitza una nova aplicació Express.

  • Definició d'una ruta:

    app.get('/', (req, res) => {
  res.send('Hello World!');
});

    Això defineix una ruta per a l'URL arrel (/). Quan es fa una sol·licitud GET a l'URL arrel, el servidor respon amb "Hello World!".

  • Inici del servidor:

    app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

    Això inicia el servidor i escolta al port 3000. Quan el servidor s'inicia, registra un missatge a la consola.

Característiques Clau d'Express.js

  • Middleware: Funcions que s'executen durant el cicle de vida d'una sol·licitud al servidor.
  • Routing: Defineix com una aplicació respon a una sol·licitud del client a un punt final particular.
  • Engines de Plantilles: Permet utilitzar fitxers de plantilles estàtiques a la teva aplicació.

Middleware

Les funcions middleware són funcions que tenen accés a l'objecte de sol·licitud (req), l'objecte de resposta (res), i la següent funció middleware en el cicle de sol·licitud-resposta de l'aplicació.

Exemple d'un middleware senzill:

app.use((req, res, next) => {
  console.log('Time:', Date.now());
  next();
});

Routing

El routing es refereix a com els punts finals (URIs) d'una aplicació responen a les sol·licituds dels clients.

Exemple de definició de rutes:

app.get('/about', (req, res) => {
  res.send('About Page');
});

app.post('/submit', (req, res) => {
  res.send('Form Submitted');
});

Engines de Plantilles

Express suporta diversos engines de plantilles com Pug, EJS, etc., per generar HTML dinàmicament.

Exemple utilitzant Pug:

  1. Instal·la Pug:

    npm install pug

  2. Configura Pug a Express:

    app.set('view engine', 'pug');
app.set('views', './views');

  3. Crea una plantilla Pug (views/index.pug):

    html
  head
    title= title
  body
    h1= message

  4. Renderitza la plantilla Pug:

    app.get('/', (req, res) => {
  res.render('index', { title: 'Hey', message: 'Hello there!' });
});

Conclusió

Express.js és un framework potent i flexible per construir aplicacions web amb Node.js. Proporciona un conjunt minimalista però robust de característiques que fan fàcil desenvolupar aplicacions web i mòbils. Entenent les bases de la configuració d'una aplicació Express, definint rutes, utilitzant middleware, i integrant engines de plantilles, pots començar a construir les teves pròpies aplicacions web de manera eficient.
