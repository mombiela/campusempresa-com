Introducció

En aquesta secció, aprendrem a crear un servidor web simple utilitzant Node.js. Això cobrirà els conceptes bàsics de la configuració d'un servidor, la gestió de sol·licituds i respostes, i la comprensió dels mòduls principals implicats.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Node.js instal·lat a la teva màquina.
  • Una comprensió bàsica de JavaScript.

Configuració del Teu Entorn

  1. Instal·la Node.js: Descarrega i instal·la Node.js des del lloc web oficial.
  2. Crea un Directori de Projecte: Obre el teu terminal i crea un nou directori per al teu projecte. 
    mkdir simple-web-server
cd simple-web-server
  3. Inicialitza un Projecte Node.js: Executa la següent comanda per crear un fitxer package.json. 
    npm init -y

Creació del Servidor

  1. Crea el Fitxer del Servidor: Crea un fitxer anomenat server.js al teu directori de projecte.

  2. Importa el Mòdul HTTP: Node.js té un mòdul integrat anomenat http que et permet crear un servidor.

    const http = require('http');

  3. Crea el Servidor: Utilitza el mètode http.createServer per crear un servidor.

    const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello, World!\n');
});

  4. Escolta en un Port: Fes que el servidor escolti en un port específic.

    const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}/`);
});

Exemple de Codi Complet

const http = require('http');

const server = http.createServer((req, res) => {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Hello, World!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}/`);
});

Executar el Servidor

  1. Inicia el Servidor: Executa la següent comanda al teu terminal. 
    node server.js
  2. Accedeix al Servidor: Obre el teu navegador web i navega a http://localhost:3000. Hauries de veure "Hello, World!" mostrat.

Gestió de Diferents Rutes

Per gestionar diferents rutes, pots inspeccionar la propietat req.url.

Exemple: Gestió de Múltiples Rutes

const http = require('http');

const server = http.createServer((req, res) => {
    if (req.url === '/') {
        res.statusCode = 200;
        res.setHeader('Content-Type', 'text/plain');
        res.end('Welcome to the Home Page!\n');
    } else if (req.url === '/about') {
        res.statusCode = 200;
        res.setHeader('Content-Type', 'text/plain');
        res.end('About Us\n');
    } else {
        res.statusCode = 404;
        res.setHeader('Content-Type', 'text/plain');
        res.end('Page Not Found\n');
    }
});

const PORT = 3000;
server.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}/`);
});

Conceptes Avançats

Utilitzant Express.js

Per a un enrutament més avançat i middleware, pots utilitzar el framework Express.js.

  1. Instal·la Express:

    npm install express

  2. Crea un Servidor Express:

    const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
    res.send('Welcome to the Home Page!');
});

app.get('/about', (req, res) => {
    res.send('About Us');
});

app.use((req, res) => {
    res.status(404).send('Page Not Found');
});

app.listen(PORT, () => {
    console.log(`Server running at http://localhost:${PORT}/`);
});

Comparació: Mòdul HTTP vs. Express.js

| Característica | Mòdul HTTP | Express.js | |-----------------------|-------------------------------------|------------------------------------------| | Enrutament | Gestió manual amb req.url | Enrutament integrat amb app.get, etc. | | Middleware | Implementació manual | Suport integrat | | Simplicitat | Bàsic i mínim | Abstraccions de nivell superior | | Rendiment | Lleugerament més ràpid per a tasques simples | Lleugerament més lent degut a funcions addicionals |

Conclusió

En aquest tema, hem cobert els conceptes bàsics de la creació d'un servidor web simple utilitzant Node.js. Hem començat amb el mòdul integrat http i després hem explorat un enrutament més avançat amb Express.js. Comprenent aquests fonaments, pots construir servidors web més complexos i rics en funcionalitats.
