Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert en Node.js. Aquesta secció et guiarà a través del procés d'instal·lació de Node.js, configuració d'un editor de codi i verificació de la teva instal·lació.

Instal·lació de Node.js

Node.js és un entorn d'execució de JavaScript construït sobre el motor de JavaScript V8 de Chrome. Per començar a desenvolupar amb Node.js, necessites instal·lar-lo a la teva màquina.

Passos per Instal·lar Node.js

  1. Descarregar Node.js:

    • Ves al lloc web oficial de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·lar Node.js:

    • Executa l'instal·lador i segueix les instruccions en pantalla.
    • L'instal·lador també instal·larà npm (Node Package Manager), que és essencial per gestionar els paquets de Node.js.

  3. Verificar la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa les següents comandes per comprovar si Node.js i npm estan instal·lats correctament: 
      node -v
npm -v
    • Hauries de veure els números de versió de Node.js i npm.

Configuració d'un Editor de Codi

Un bon editor de codi pot millorar significativament la teva productivitat. Per al desenvolupament de Node.js, Visual Studio Code (VS Code) és altament recomanat a causa de les seves potents característiques i extensions.

Instal·lació de Visual Studio Code

  1. Descarregar VS Code:

    • Visita el lloc web de Visual Studio Code.
    • Descarrega l'instal·lador per al teu sistema operatiu.

  2. Instal·lar VS Code:

    • Executa l'instal·lador i segueix les instruccions en pantalla.

  3. Instal·lar Extensions:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca i instal·la les següents extensions:
      • Node.js Extension Pack: Proporciona una col·lecció d'extensions per al desenvolupament de Node.js.
      • ESLint: Ajuda a identificar i corregir problemes en el teu codi JavaScript.
      • Prettier - Code formatter: Assegura un format de codi consistent.

Creació del Teu Primer Projecte Node.js

Ara que tens Node.js i un editor de codi configurats, creem un simple projecte Node.js per assegurar-nos que tot funciona correctament.

Passos per Crear un Projecte Node.js

  1. Crear un Directori de Projecte:

    • Obre el teu terminal o símbol del sistema.
    • Crea un nou directori per al teu projecte i navega dins d'ell: 
      mkdir my-node-project
cd my-node-project

  2. Inicialitzar el Projecte:

    • Executa la següent comanda per crear un fitxer package.json, que gestionarà les dependències del teu projecte: 
      npm init -y

  3. Crear un Fitxer d'Entrada:

    • Al teu directori de projecte, crea un fitxer anomenat index.js.
    • Obre index.js al teu editor de codi i afegeix el següent codi: 
      console.log('Hello, Node.js!');

  4. Executar la Teva Aplicació Node.js:

    • Al teu terminal, executa la següent comanda: 
      node index.js
    • Hauries de veure la sortida Hello, Node.js!.

Resum

En aquesta secció, has après com configurar el teu entorn de desenvolupament per a Node.js. Has instal·lat Node.js i npm, configurat Visual Studio Code amb extensions essencials, i creat un simple projecte Node.js per verificar la teva configuració. Amb el teu entorn preparat, ara estàs llest per aprofundir en el desenvolupament de Node.js.
