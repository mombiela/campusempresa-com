Els buffers són una part fonamental de Node.js, especialment quan es tracta de dades binàries. Aquest tema et guiarà des dels conceptes bàsics dels buffers fins a un ús més avançat, assegurant que entenguis com utilitzar-los de manera efectiva en les teves aplicacions Node.js.

Introducció als Buffers

Els buffers s'utilitzen per gestionar dades binàries a Node.js. Són particularment útils quan es treballa amb sistemes de fitxers, protocols de xarxa o qualsevol altre flux de dades binàries.

  • Dades Binàries: Dades que no estan en un format llegible per humans, com imatges, vídeos o qualsevol altre fitxer no de text.
  • Classe Buffer: Una classe global a Node.js que proporciona una manera de gestionar dades binàries directament.

Creació de Buffers

Hi ha diverses maneres de crear buffers a Node.js:

  1. Allocar un Buffer: Utilitza Buffer.alloc(size) per crear un buffer d'una mida especificada.
  2. Des d'un Array: Utilitza Buffer.from(array) per crear un buffer a partir d'un array de bytes.
  3. Des d'una Cadena: Utilitza Buffer.from(string, encoding) per crear un buffer a partir d'una cadena amb una codificació especificada.
// Allocant un buffer de 10 bytes
const buffer1 = Buffer.alloc(10);
console.log(buffer1);

// Creant un buffer a partir d'un array
const buffer2 = Buffer.from([1, 2, 3, 4]);
console.log(buffer2);

// Creant un buffer a partir d'una cadena
const buffer3 = Buffer.from('Hello, World!', 'utf-8');
console.log(buffer3);

Escriure en Buffers

Pots escriure dades en un buffer utilitzant el mètode write.

const buffer = Buffer.alloc(10);
buffer.write('Hello');
console.log(buffer.toString('utf-8')); // Sortida: Hello

Llegir de Buffers

Per llegir dades d'un buffer, pots utilitzar diversos mètodes com toString, readUInt8, readUInt16LE, etc.

const buffer = Buffer.from('Hello, World!', 'utf-8');
console.log(buffer.toString('utf-8')); // Sortida: Hello, World!
console.log(buffer.readUInt8(0)); // Sortida: 72 (codi ASCII per 'H')

Mètodes i Propietats del Buffer

Els buffers tenen una varietat de mètodes i propietats que et permeten manipular i inspeccionar les dades que contenen.

  • Length: La propietat length dóna la mida del buffer.
  • Slice: El mètode slice crea un nou buffer que fa referència a la mateixa memòria que el buffer original.
  • Copy: El mètode copy et permet copiar dades d'un buffer a un altre.
const buffer = Buffer.from('Hello, World!', 'utf-8');
console.log(buffer.length); // Sortida: 13

const slicedBuffer = buffer.slice(0, 5);
console.log(slicedBuffer.toString('utf-8')); // Sortida: Hello

const targetBuffer = Buffer.alloc(5);
buffer.copy(targetBuffer, 0, 0, 5);
console.log(targetBuffer.toString('utf-8')); // Sortida: Hello

Operacions Avançades amb Buffers

Concatenació de Buffers

Pots concatenar múltiples buffers utilitzant Buffer.concat.

const buffer1 = Buffer.from('Hello, ');
const buffer2 = Buffer.from('World!');
const combinedBuffer = Buffer.concat([buffer1, buffer2]);
console.log(combinedBuffer.toString('utf-8')); // Sortida: Hello, World!

Comparació de Buffers

Els buffers es poden comparar utilitzant el mètode compare.

const buffer1 = Buffer.from('ABC');
const buffer2 = Buffer.from('ABD');
const result = buffer1.compare(buffer2);

if (result < 0) {
  console.log('buffer1 ve abans de buffer2');
} else if (result > 0) {
  console.log('buffer1 ve després de buffer2');
} else {
  console.log('buffer1 és igual a buffer2');
}

Codificació i Decodificació de Buffers

Els buffers es poden codificar i decodificar utilitzant diverses codificacions de caràcters com utf-8, ascii, base64, etc.

const buffer = Buffer.from('Hello, World!', 'utf-8');
const base64String = buffer.toString('base64');
console.log(base64String); // Sortida: SGVsbG8sIFdvcmxkIQ==

const decodedBuffer = Buffer.from(base64String, 'base64');
console.log(decodedBuffer.toString('utf-8')); // Sortida: Hello, World!

Resum

Els buffers són una característica poderosa a Node.js per gestionar dades binàries. Proporcionen una varietat de mètodes per crear, escriure, llegir i manipular dades binàries. Entendre com utilitzar els buffers de manera efectiva pot millorar enormement la teva capacitat per treballar amb dades binàries en les teves aplicacions Node.js.

  • Creació de Buffers: Aprèn les diferents maneres de crear buffers.
  • Escriure i Llegir: Entén com escriure i llegir de buffers.
  • Mètodes i Propietats: Familiaritza't amb els diversos mètodes i propietats disponibles per als buffers.
  • Operacions Avançades: Explora operacions avançades com la concatenació, comparació i codificació/decodificació.

En dominar els buffers, estaràs ben preparat per gestionar dades binàries en els teus projectes Node.js.
