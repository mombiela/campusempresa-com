Introducció

Node.js és un potent entorn d'execució de codi obert i multiplataforma que permet als desenvolupadors executar codi JavaScript fora d'un navegador web. Està construït sobre el motor JavaScript V8 de Chrome i està dissenyat per construir aplicacions de xarxa escalables.

Conceptes Clau

  • Entorn d'Execució de JavaScript: Node.js permet executar JavaScript al costat del servidor, no només al navegador.
  • I/O No Bloquejant: Node.js utilitza un model d'I/O no bloquejant i basat en esdeveniments, cosa que el fa lleuger i eficient.
  • Bucle d'Esdeveniments Monofil: Node.js opera en un bucle d'esdeveniments monofil, que pot gestionar múltiples connexions simultàniament.
  • Gestor de Paquets (npm): Node.js ve amb npm, un gestor de paquets que permet als desenvolupadors compartir i reutilitzar codi.

Característiques de Node.js

  • Asíncron i Basat en Esdeveniments: Totes les API de la biblioteca de Node.js són asíncrones, és a dir, no bloquejants.
  • Execució Ràpida: Construït sobre el motor JavaScript V8, Node.js executa el codi ràpidament.
  • Un Sol Llenguatge de Programació: Els desenvolupadors poden escriure tant el codi del client com el del servidor en JavaScript.
  • Comunitat i Ecosistema: Una comunitat gran i activa contribueix a un ric ecosistema de mòduls i eines.

Exemple de Codi: Servidor HTTP Bàsic

Creem un servidor HTTP senzill utilitzant Node.js per entendre la seva funcionalitat bàsica.

// Carregar el mòdul http per crear un servidor HTTP.
const http = require('http');

// Configurar el servidor HTTP per respondre amb "Hello, World!" a totes les sol·licituds.
const server = http.createServer((req, res) => {
  res.statusCode = 200; // Codi d'estat HTTP: 200 (OK)
  res.setHeader('Content-Type', 'text/plain'); // Establir la capçalera de la resposta
  res.end('Hello, World!\n'); // Enviar el cos de la resposta
});

// Escoltar al port 3000, l'IP per defecte és 127.0.0.1
server.listen(3000, '127.0.0.1', () => {
  console.log('Servidor funcionant a http://127.0.0.1:3000/');
});

Explicació

  1. Carregar el Mòdul HTTP: Utilitzem require('http') per carregar el mòdul HTTP, que és part dels mòduls bàsics de Node.js.
  2. Crear el Servidor: El mètode http.createServer() crea un nou servidor HTTP i configura una funció de retorn de trucada per gestionar les sol·licituds entrants.
  3. Gestionar les Sol·licituds: La funció de retorn de trucada pren dos arguments, req (sol·licitud) i res (resposta). Establim el codi d'estat a 200 (OK), establim el tipus de contingut a text/plain, i enviem el cos de la resposta "Hello, World!".
  4. Escoltar al Port 3000: El servidor escolta al port 3000, i quan comença, registra un missatge a la consola.

Resum

Node.js és un entorn d'execució versàtil i eficient per executar JavaScript al costat del servidor. El seu model d'I/O no bloquejant, el bucle d'esdeveniments monofil i l'execució ràpida el fan ideal per construir aplicacions de xarxa escalables. Amb npm, els desenvolupadors tenen accés a un vast repositori de mòduls reutilitzables, millorant la productivitat i la col·laboració. L'exemple bàsic de servidor HTTP demostra com és de fàcil començar amb Node.js i destaca les seves característiques principals.
