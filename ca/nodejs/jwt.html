Introducció a JWT

JSON Web Tokens (JWT) són un mitjà compacte i segur per a representar afirmacions que es transferiran entre dues parts. Les afirmacions en un JWT s'encoden com un objecte JSON que s'utilitza com a càrrega útil d'una estructura JSON Web Signature (JWS) o com a text pla d'una estructura JSON Web Encryption (JWE), permetent que les afirmacions siguin signades digitalment o protegides per integritat amb un Codi d'Autenticació de Missatge (MAC) i/o encriptades.

Conceptes Clau

  • Estructura del Token: Els JWTs consten de tres parts: Capçalera, Càrrega útil i Signatura.
  • Capçalera: Conté metadades sobre el tipus de token i els algorismes criptogràfics utilitzats.
  • Càrrega útil: Conté les afirmacions. Les afirmacions són declaracions sobre una entitat (normalment, l'usuari) i dades addicionals.
  • Signatura: S'utilitza per verificar la integritat del token i assegurar-se que no ha estat manipulat.

Estructura de JWT

Un JWT normalment està estructurat de la següent manera:

header.payload.signature

Exemple

{
  "alg": "HS256",
  "typ": "JWT"
}
{
  "sub": "1234567890",
  "name": "John Doe",
  "iat": 1516239022
}

Creació de JWTs en Node.js

Per crear JWTs en Node.js, pots utilitzar la biblioteca jsonwebtoken.

Instal·lació

npm install jsonwebtoken

Creació d'un Token

const jwt = require('jsonwebtoken');

const payload = {
  sub: '1234567890',
  name: 'John Doe',
  iat: Math.floor(Date.now() / 1000) - 30
};

const secret = 'your-256-bit-secret';

const token = jwt.sign(payload, secret);
console.log(token);

Verificació d'un Token

const token = 'your.jwt.token.here';

jwt.verify(token, secret, (err, decoded) => {
  if (err) {
    console.log('Token no és vàlid:', err);
  } else {
    console.log('Token decodificat:', decoded);
  }
});

Ús de JWT per a l'Autenticació

Els JWTs s'utilitzen comunament per a l'autenticació en aplicacions web. Aquí tens un exemple senzill de com utilitzar JWTs per a l'autenticació d'usuaris en una aplicació Node.js.

Exemple: Inici de Sessió d'Usuari i Generació de Token

const express = require('express');
const jwt = require('jsonwebtoken');
const bodyParser = require('body-parser');

const app = express();
app.use(bodyParser.json());

const users = [
  { id: 1, username: 'john', password: 'password123' }
];

const secret = 'your-256-bit-secret';

app.post('/login', (req, res) => {
  const { username, password } = req.body;
  const user = users.find(u => u.username === username && u.password === password);

  if (user) {
    const token = jwt.sign({ sub: user.id }, secret, { expiresIn: '1h' });
    res.json({ token });
  } else {
    res.status(401).send('Nom d’usuari o contrasenya incorrectes');
  }
});

app.listen(3000, () => {
  console.log('Servidor iniciat al port 3000');
});

Exemple: Protecció de Rutes

const express = require('express');
const jwt = require('jsonwebtoken');

const app = express();
const secret = 'your-256-bit-secret';

const authenticateJWT = (req, res, next) => {
  const token = req.header('Authorization').split(' ')[1];

  if (token) {
    jwt.verify(token, secret, (err, user) => {
      if (err) {
        return res.sendStatus(403);
      }
      req.user = user;
      next();
    });
  } else {
    res.sendStatus(401);
  }
};

app.get('/protected', authenticateJWT, (req, res) => {
  res.send('Aquesta és una ruta protegida');
});

app.listen(3000, () => {
  console.log('Servidor iniciat al port 3000');
});

Conclusió

Els JWTs són una eina poderosa per a l'autenticació i l'intercanvi d'informació en aplicacions web. Proporcionen una manera segura i eficient de transmetre informació entre parts. Comprenent l'estructura i l'ús dels JWTs, pots implementar mecanismes d'autenticació robustos en les teves aplicacions Node.js. Recorda sempre mantenir les teves claus secretes segures i validar i verificar els tokens adequadament per assegurar la seguretat de la teva aplicació.
