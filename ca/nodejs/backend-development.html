Introducció a Node.js

  • Què és Node.js?
    • Node.js és un entorn d'execució de JavaScript construït sobre el motor de JavaScript V8 de Chrome.
    • Permet executar JavaScript al costat del servidor.
  • Per què utilitzar Node.js per al desenvolupament de backend?
    • Arquitectura no bloquejant i basada en esdeveniments.
    • Alt rendiment per a tasques d'E/S.
    • Gran ecosistema amb npm (Node Package Manager).

Configuració del teu entorn Node.js

  • Instal·lant Node.js i npm
    • Descarrega i instal·la des del lloc web oficial de Node.js.
    • Verifica la instal·lació: 
      node -v
npm -v
  • Configurant un projecte
    • Inicialitza un nou projecte Node.js: 
      mkdir my-node-app
cd my-node-app
npm init -y

Conceptes bàsics

  • Mòduls i require
    • Creant i utilitzant mòduls: 
      // math.js
function add(a, b) {
  return a + b;
}
module.exports = add;

// app.js
const add = require('./math');
console.log(add(2, 3)); // Output: 5
  • Programació asíncrona
    • Callbacks: 
      const fs = require('fs');
fs.readFile('example.txt', 'utf8', (err, data) => {
  si (err) throw err;
  console.log(data);
});
    • Promises: 
      const fs = require('fs').promises;
fs.readFile('example.txt', 'utf8')
  .then(data => console.log(data))
  .catch(err => console.error(err));
    • Async/Await: 
      const fs = require('fs').promises;
async function readFile() {
  try {
    const data = await fs.readFile('example.txt', 'utf8');
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
readFile();

Construint un servidor web senzill

  • Utilitzant el mòdul HTTP
    • Creant un servidor bàsic: 
      const http = require('http');
const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});
server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});

Framework Express.js

  • Introducció a Express.js
    • Simplifica la creació de servidors i el routing.
    • Instal·lació: 
      npm install express
  • Ús bàsic
    • Creant un servidor Express: 
      const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, World!');
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}/`);
});
  • Middleware
    • Utilitzant middleware per al processament de sol·licituds: 
      app.use((req, res, next) => {
  console.log('Time:', Date.now());
  next();
});

Treballant amb bases de dades

  • Connectant-se a una base de dades MongoDB utilitzant Mongoose
    • Instal·lació: 
      npm install mongoose
    • Ús bàsic: 
      const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost/mydatabase', { useNewUrlParser: true, useUnifiedTopology: true });

const db = mongoose.connection;
db.on('error', console.error.bind(console, 'connection error:'));
db.once('open', () => {
  console.log('Connected to the database');
});
  • Definint esquemes i models
    • Exemple d'esquema i model: 
      const userSchema = new mongoose.Schema({
  name: String,
  age: Number,
  email: String
});

const User = mongoose.model('User', userSchema);

const newUser = new User({ name: 'John Doe', age: 30, email: 'john@example.com' });
newUser.save((err) => {
  if (err) return console.error(err);
  console.log('User saved successfully');
});

Temes avançats

  • Autenticació amb JWT (JSON Web Tokens)
    • Instal·lació: 
      npm install jsonwebtoken
    • Generant i verificant tokens: 
      const jwt = require('jsonwebtoken');
const secretKey = 'your-secret-key';

// Generant un token
const token = jwt.sign({ username: 'john_doe' }, secretKey, { expiresIn: '1h' });
console.log('Token:', token);

// Verificant un token
jwt.verify(token, secretKey, (err, decoded) => {
  if (err) {
    console.error('Token verification failed:', err);
  } else {
    console.log('Decoded token:', decoded);
  }
});
  • Comunicació en temps real amb Socket.io
    • Instal·lació: 
      npm install socket.io
    • Ús bàsic: 
      const http = require('http');
const socketIo = require('socket.io');
const server = http.createServer();
const io = socketIo(server);

io.on('connection', (socket) => {
  console.log('A user connected');
  socket.on('disconnect', () => {
    console.log('User disconnected');
  });
});

server.listen(3000, () => {
  console.log('Server running at http://localhost:3000/');
});
