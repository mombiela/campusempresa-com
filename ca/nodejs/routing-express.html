L'enrutament és un concepte fonamental en Express, un popular framework d'aplicacions web per a Node.js. Es refereix a com els punts finals (URI) d'una aplicació responen a les sol·licituds dels clients. Aquest tema et guiarà des dels conceptes bàsics de l'enrutament fins a conceptes més avançats.

Introducció a l'Enrutament

L'enrutament en Express implica definir com la teva aplicació respon a diverses sol·licituds HTTP. Cada ruta pot tenir una o més funcions manejadores, que s'executen quan la ruta coincideix.

Enrutament Bàsic

  • Definició de Ruta: Una ruta es defineix per un mètode HTTP, un camí i una funció manejadora.
  • Mètodes HTTP: Els mètodes comuns inclouen GET, POST, PUT, DELETE, etc.

Exemple: Ruta GET Bàsica

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hola Món!');
});

app.listen(3000, () => {
  console.log('El servidor està funcionant al port 3000');
});

En aquest exemple:

  • app.get('/') defineix una ruta que escolta sol·licituds GET a l'URL arrel (/).
  • La funció manejadora envia una resposta "Hola Món!".

Paràmetres de Ruta

Els paràmetres de ruta són segments de l'URL amb nom que s'utilitzen per capturar valors en posicions específiques de l'URL.

Exemple: Paràmetres de Ruta

app.get('/users/:userId', (req, res) => {
  const userId = req.params.userId;
  res.send(`ID d'usuari: ${userId}`);
});

En aquest exemple:

  • :userId és un paràmetre de ruta.
  • req.params.userId captura el valor de userId de l'URL.

Paràmetres de Consulta

Els paràmetres de consulta s'utilitzen per passar dades al servidor com a part de l'URL.

Exemple: Paràmetres de Consulta

app.get('/search', (req, res) => {
  const query = req.query.q;
  res.send(`Consulta de cerca: ${query}`);
});

En aquest exemple:

  • req.query.q captura el valor del paràmetre de consulta q de l'URL.

Gestió de Diferents Mètodes HTTP

Express et permet gestionar diferents mètodes HTTP per a la mateixa ruta.

Exemple: Gestió de Diferents Mètodes

app.route('/book')
  .get((req, res) => {
    res.send('Obtenir un llibre aleatori');
  })
  .post((req, res) => {
    res.send('Afegir un llibre');
  })
  .put((req, res) => {
    res.send('Actualitzar el llibre');
  });

En aquest exemple:

  • app.route('/book') enllaça múltiples manejadors per a diferents mètodes HTTP en la mateixa ruta.

Middleware en Rutes

Les funcions middleware es poden utilitzar per gestionar sol·licituds abans que arribin al manejador de la ruta.

Exemple: Middleware en Rutes

const logRequest = (req, res, next) => {
  console.log(`${req.method} ${req.url}`);
  next();
};

app.use(logRequest);

app.get('/', (req, res) => {
  res.send('Hola Món!');
});

En aquest exemple:

  • logRequest és una funció middleware que registra el mètode de la sol·licitud i l'URL.
  • app.use(logRequest) aplica aquest middleware a totes les rutes.

Enrutament Avançat

Grups de Rutes

Els grups de rutes et permeten aplicar middleware a un grup de rutes.

Exemple: Grups de Rutes

const express = require('express');
const app = express();
const router = express.Router();

router.use((req, res, next) => {
  console.log('URL de la sol·licitud:', req.originalUrl);
  next();
});

router.get('/user/:id', (req, res) => {
  res.send(`ID d'usuari: ${req.params.id}`);
});

router.get('/user/:id/edit', (req, res) => {
  res.send(`Editar ID d'usuari: ${req.params.id}`);
});

app.use('/admin', router);

app.listen(3000, () => {
  console.log('El servidor està funcionant al port 3000');
});

En aquest exemple:

  • router.use aplica middleware a totes les rutes dins del router.
  • app.use('/admin', router) munta el router en el camí /admin.

Resum

L'enrutament en Express és una característica poderosa que et permet definir com la teva aplicació respon a diverses sol·licituds HTTP. Els conceptes clau inclouen:

  • Enrutament bàsic amb mètodes HTTP.
  • Ús de paràmetres de ruta i paràmetres de consulta.
  • Gestió de diferents mètodes HTTP en la mateixa ruta.
  • Aplicació de middleware a les rutes.
  • Agrupació de rutes amb routers.

En dominar aquests conceptes, pots crear aplicacions web robustes i escalables utilitzant Express.
