Visió General

En aquesta secció, explorarem els fonaments de les bases de dades i com interactuar amb elles utilitzant Node.js. Les bases de dades són essencials per emmagatzemar, recuperar i gestionar dades en aplicacions web. Cobrirem tant bases de dades SQL com NoSQL, les seves diferències i com integrar-les amb Node.js.

Conceptes Clau

  • Base de Dades: Una col·lecció estructurada de dades.
  • SQL (Structured Query Language): Un llenguatge utilitzat per gestionar i manipular bases de dades relacionals.
  • NoSQL (Not Only SQL): Una categoria de sistemes de gestió de bases de dades que no utilitzen SQL com a llenguatge de consulta principal.
  • Operacions CRUD: Les quatre operacions bàsiques per gestionar dades - Crear, Llegir, Actualitzar, Eliminar.

Tipus de Bases de Dades

Bases de Dades SQL

  • Exemples: MySQL, PostgreSQL, SQLite
  • Característiques:
    • Utilitzen taules per emmagatzemar dades.
    • Les dades estan estructurades en files i columnes.
    • Suporten transaccions ACID (Atomicitat, Consistència, Aïllament, Durabilitat).

Bases de Dades NoSQL

  • Exemples: MongoDB, CouchDB, Redis
  • Característiques:
    • Utilitzen diversos models de dades com document, clau-valor, gràfic o família de columnes.
    • Sense esquema o amb esquema flexible.
    • Dissenyades per a escalabilitat i rendiment.

Configuració d'una Base de Dades amb Node.js

Instal·lació de Paquets Necessaris

Per interactuar amb bases de dades en Node.js, cal instal·lar paquets específics. Aquí en teniu alguns de comuns:

  • Per MySQL:

    npm install mysql

  • Per MongoDB:

    npm install mongodb

Connexió a una Base de Dades

Exemple de MySQL

const mysql = require('mysql');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: 'password',
  database: 'test_db'
});

connection.connect((err) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected as id ' + connection.threadId);
});
  • Explicació:
    • mysql.createConnection: Crea una connexió a la base de dades MySQL.
    • connection.connect: Estableix la connexió.

Exemple de MongoDB

const { MongoClient } = require('mongodb');

const url = 'mongodb://localhost:27017';
const dbName = 'test_db';

MongoClient.connect(url, { useNewUrlParser: true, useUnifiedTopology: true }, (err, client) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected successfully to server');
  const db = client.db(dbName);
  client.close();
});
  • Explicació:
    • MongoClient.connect: Connecta al servidor MongoDB.
    • client.db: Accedeix a la base de dades específica.

Realització d'Operacions CRUD

Crear

MySQL

const sql = 'INSERT INTO users (name, age) VALUES (?, ?)';
const values = ['John Doe', 30];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertId);
});

MongoDB

const collection = db.collection('users');
collection.insertOne({ name: 'John Doe', age: 30 }, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertedId);
});

Llegir

MySQL

const sql = 'SELECT * FROM users WHERE age > ?';
const age = 25;

connection.query(sql, [age], (err, results) => {
  if (err) throw err;
  console.log(results);
});

MongoDB

const collection = db.collection('users');
collection.find({ age: { $gt: 25 } }).toArray((err, docs) => {
  if (err) throw err;
  console.log(docs);
});

Actualitzar

MySQL

const sql = 'UPDATE users SET age = ? WHERE name = ?';
const values = [35, 'John Doe'];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record updated, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.updateOne({ name: 'John Doe' }, { $set: { age: 35 } }, (err, result) => {
  if (err) throw err;
  console.log('Record updated, modified count: ' + result.modifiedCount);
});

Eliminar

MySQL

const sql = 'DELETE FROM users WHERE name = ?';
const name = 'John Doe';

connection.query(sql, [name], (err, result) => {
  if (err) throw err;
  console.log('Record deleted, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.deleteOne({ name: 'John Doe' }, (err, result) => {
  if (err) throw err;
  console.log('Record deleted, deleted count: ' + result.deletedCount);
});

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les bases de dades, incloent les diferències entre bases de dades SQL i NoSQL. També hem demostrat com configurar i connectar-se a ambdós tipus de bases de dades utilitzant Node.js, i com realitzar operacions CRUD. Entendre aquests fonaments és crucial per construir aplicacions web robustes i escalables.
