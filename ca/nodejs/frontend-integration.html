Introducció

Integrar Node.js amb tecnologies de frontend és una habilitat crucial per als desenvolupadors full-stack. Aquesta secció et guiarà a través del procés de connectar el teu backend de Node.js amb diversos frameworks i biblioteques de frontend. Al final d'aquest tema, entendràs com servir fitxers estàtics, utilitzar motors de plantilles i crear APIs que puguin ser consumides per aplicacions de frontend.

Servir Fitxers Estàtics

Node.js pot servir fitxers estàtics com HTML, CSS i JavaScript. Aquest és sovint el primer pas per integrar el teu backend amb el frontend.

Exemple: Servir Fitxers Estàtics amb Express

const express = require('express');
const path = require('path');
const app = express();

// Serveix fitxers estàtics des del directori 'public'
app.use(express.static(path.join(__dirname, 'public')));

app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, 'public', 'index.html'));
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
  console.log(`El servidor està funcionant al port ${PORT}`);
});
  • Explicació:
    • El middleware express.static s'utilitza per servir fitxers estàtics.
    • path.join ajuda a construir el camí cap als fitxers estàtics.
    • res.sendFile envia el fitxer HTML al client.

Utilitzar Motors de Plantilles

Els motors de plantilles permeten generar HTML dinàmicament. Els motors de plantilles populars inclouen EJS, Pug i Handlebars.

Exemple: Utilitzar EJS amb Express

const express = require('express');
const app = express();

// Configura EJS com a motor de plantilles
app.set('view engine', 'ejs');

app.get('/', (req, res) => {
  res.render('index', { title: 'Pàgina Principal', message: 'Benvingut a Node.js amb EJS!' });
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
  console.log(`El servidor està funcionant al port ${PORT}`);
});
  • Explicació:
    • app.set('view engine', 'ejs') configura EJS com a motor de plantilles.
    • res.render renderitza la plantilla EJS i l'envia al client.

Crear APIs per al Consum del Frontend

Les APIs són essencials per a la comunicació frontend-backend. Pots crear APIs RESTful utilitzant Node.js i Express.

Exemple: Crear una Simple API REST

const express = require('express');
const app = express();

app.use(express.json());

let items = [
  { id: 1, name: 'Item 1' },
  { id: 2, name: 'Item 2' }
];

// Obtenir tots els ítems
app.get('/api/items', (req, res) => {
  res.json(items);
});

// Obtenir un ítem per ID
app.get('/api/items/:id', (req, res) => {
  const item = items.find(i => i.id === parseInt(req.params.id));
  if (!item) return res.status(404).send('Ítem no trobat');
  res.json(item);
});

// Crear un nou ítem
app.post('/api/items', (req, res) => {
  const newItem = {
    id: items.length + 1,
    name: req.body.name
  };
  items.push(newItem);
  res.status(201).json(newItem);
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
  console.log(`El servidor està funcionant al port ${PORT}`);
});
  • Explicació:
    • El middleware express.json s'utilitza per analitzar cossos JSON.
    • app.get defineix rutes per obtenir dades.
    • app.post defineix una ruta per crear noves dades.

Integració amb Frameworks de Frontend

Els frameworks de frontend moderns com React, Angular i Vue.js poden consumir APIs creades amb Node.js.

Exemple: Obtenir Dades en React

import React, { useEffect, useState } from 'react';

function App() {
  const [items, setItems] = useState([]);

  useEffect(() => {
    fetch('/api/items')
      .then(response => response.json())
      .then(data => setItems(data));
  }, []);

  return (
    <div>
      <h1>Ítems</h1>
      <ul>
        {items.map(item => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default App;
  • Explicació:
    • useEffect s'utilitza per obtenir dades de l'API quan el component es munta.
    • fetch fa una sol·licitud HTTP a l'API de Node.js.
    • setItems actualitza l'estat amb les dades obtingudes.

Conclusió

Integrar Node.js amb tecnologies de frontend implica servir fitxers estàtics, utilitzar motors de plantilles, crear APIs i consumir aquestes APIs en aplicacions de frontend. En dominar aquestes habilitats, pots construir aplicacions full-stack robustes que proporcionen una experiència d'usuari fluida.
