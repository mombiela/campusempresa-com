Introducció a Postman

Postman és una eina popular utilitzada per a proves d'API. Permet als desenvolupadors enviar sol·licituds HTTP a APIs, inspeccionar respostes i automatitzar fluxos de treball de proves. Aquesta secció cobrirà els conceptes bàsics de Postman i com es pot utilitzar conjuntament amb aplicacions Node.js.

Conceptes Clau

  • Mètodes HTTP: GET, POST, PUT, DELETE, etc.
  • Endpoints: URLs on les APIs són accessibles.
  • Sol·licituds i Respostes: Enviar dades a i rebre dades de les APIs.
  • Col·leccions: Agrupar sol·licituds d'API relacionades.
  • Variables d'Entorn: Emmagatzemar i reutilitzar valors a través de sol·licituds.

Configurant Postman

Per començar amb Postman, segueix aquests passos:

  1. Descarrega i Instal·la Postman: Visita el lloc web de Postman i descarrega la versió adequada per al teu sistema operatiu.
  2. Crea un Compte: Registra't per a un compte gratuït per guardar el teu treball i accedir a funcions addicionals.
  3. Crea una Nova Sol·licitud: Obre Postman i fes clic al botó "New" per crear una nova sol·licitud.

Fent la Teva Primera Sol·licitud d'API

Fem una simple sol·licitud GET a una API pública.

  1. Obre Postman.
  2. Crea una Nova Sol·licitud:
    • Fes clic a "New" > "Request".
    • Dona un nom a la teva sol·licitud i afegeix-la a una col·lecció si ho desitges.
  3. Configura el Tipus de Sol·licitud i la URL:
    • Selecciona "GET" del menú desplegable.
    • Introdueix la URL: https://jsonplaceholder.typicode.com/posts/1.
  4. Envia la Sol·licitud:
    • Fes clic al botó "Send".
    • Inspecciona la resposta a la secció inferior de la interfície de Postman.

Exemple de Resposta

{
  "userId": 1,
  "id": 1,
  "title": "sunt aut facere repellat provident occaecati excepturi optio reprehenderit",
  "body": "quia et suscipit\nsuscipit..."
}

Provant una API de Node.js amb Postman

Ara, provem una API de Node.js. Suposem que tens un servidor Express simple executant-se localment.

Exemple de Servidor Node.js

const express = require('express');
const app = express();
const port = 3000;

app.use(express.json());

app.get('/api/users', (req, res) => {
  res.json([{ id: 1, name: 'John Doe' }]);
});

app.post('/api/users', (req, res) => {
  const newUser = req.body;
  res.status(201).json(newUser);
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}`);
});

Provant l'Endpoint GET

  1. Configura el Tipus de Sol·licitud i la URL:
    • Selecciona "GET" del menú desplegable.
    • Introdueix la URL: http://localhost:3000/api/users.
  2. Envia la Sol·licitud:
    • Fes clic al botó "Send".
    • Inspecciona la resposta.

Provant l'Endpoint POST

  1. Configura el Tipus de Sol·licitud i la URL:
    • Selecciona "POST" del menú desplegable.
    • Introdueix la URL: http://localhost:3000/api/users.
  2. Configura el Cos de la Sol·licitud:
    • Fes clic a la pestanya "Body".
    • Selecciona "raw" i tria "JSON" del menú desplegable.
    • Introdueix el següent JSON: 
      {
  "id": 2,
  "name": "Jane Doe"
}
  3. Envia la Sol·licitud:
    • Fes clic al botó "Send".
    • Inspecciona la resposta.

Utilitzant Col·leccions i Variables d'Entorn

Les col·leccions i les variables d'entorn de Postman poden ajudar a gestionar i organitzar les teves proves d'API.

Creant una Col·lecció

  1. Crea una Nova Col·lecció:
    • Fes clic a "New" > "Collection".
    • Dona un nom a la teva col·lecció i afegeix una descripció si ho desitges.
  2. Afegeix Sol·licituds a la Col·lecció:
    • Arrossega i deixa anar sol·licituds existents a la col·lecció o crea de noves dins d'aquesta.

Utilitzant Variables d'Entorn

  1. Crea un Entorn:
    • Fes clic a la icona d'engranatge a la cantonada superior dreta i selecciona "Manage Environments".
    • Fes clic a "Add" i dona un nom al teu entorn.
  2. Afegeix Variables:
    • Afegeix parells clau-valor, per exemple, baseUrl = http://localhost:3000.
  3. Utilitza Variables en Sol·licituds:
    • Substitueix parts de la teva URL amb variables, per exemple, {{baseUrl}}/api/users.

Automatitzant Proves amb Postman

Postman et permet escriure proves utilitzant JavaScript per automatitzar les teves proves d'API.

Escriure Proves

  1. Obre la Pestanya de Proves:
    • Fes clic a la pestanya "Tests" en la teva sol·licitud.
  2. Afegeix Scripts de Prova:
    • Escriu JavaScript per validar respostes.
    pm.test("Status code is 200", function () {
  pm.response.to.have.status(200);
});

pm.test("Response has userId", function () {
  const jsonData = pm.response.json();
  pm.expect(jsonData).to.have.property('userId');
});

Conclusió

Postman és una eina poderosa per provar APIs, i s'integra perfectament amb aplicacions Node.js. Entenent com configurar sol·licituds, utilitzar col·leccions i variables d'entorn, i automatitzar proves, pots millorar significativament el teu desenvolupament i flux de treball de proves d'API. Practica utilitzant Postman amb els teus projectes Node.js per convertir-te en un expert en proves d'API.
