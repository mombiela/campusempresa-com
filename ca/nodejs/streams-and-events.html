Introducció als Fluxos

Els fluxos són una característica potent a Node.js que et permeten gestionar dades de manera eficient. Són instàncies de EventEmitter i es poden utilitzar per llegir o escriure dades contínuament.

Conceptes Clau

  • Fluxos Llegibles: Utilitzats per llegir dades.
  • Fluxos Escriuribles: Utilitzats per escriure dades.
  • Fluxos Dúplex: Es poden utilitzar tant per llegir com per escriure.
  • Fluxos de Transformació: Un tipus de flux dúplex on la sortida es calcula en funció de l'entrada.

Exemple: Llegint d'un Flux Llegible

const fs = require('fs');
const readableStream = fs.createReadStream('example.txt', 'utf8');

readableStream.on('data', (chunk) => {
  console.log(`Rebuts ${chunk.length} bytes de dades.`);
  console.log(chunk);
});

readableStream.on('end', () => {
  console.log('No hi ha més dades.');
});
  • Explicació: Aquest codi llegeix dades de example.txt en fragments i les mostra a la consola. L'esdeveniment data s'emet quan un fragment de dades està disponible, i l'esdeveniment end s'emet quan no hi ha més dades per llegir.

Fluxos Escriuribles

Els fluxos escriuribles s'utilitzen per escriure dades a una destinació.

Exemple: Escriure a un Flux Escriurible

const fs = require('fs');
const writableStream = fs.createWriteStream('output.txt');

writableStream.write('Hola, món!\n');
writableStream.write('Escrivint dades a un fitxer.\n');
writableStream.end('Escritura finalitzada.');
  • Explicació: Aquest codi escriu dades a output.txt. El mètode write s'utilitza per escriure dades, i el mètode end s'utilitza per senyalitzar el final de l'escriptura.

Fluxos Dúplex i de Transformació

Els fluxos dúplex poden llegir i escriure dades, mentre que els fluxos de transformació poden modificar o transformar les dades a mesura que es llegeixen o escriuen.

Exemple: Flux de Transformació

const { Transform } = require('stream');

class UppercaseTransform extends Transform {
  _transform(chunk, encoding, callback) {
    this.push(chunk.toString().toUpperCase());
    callback();
  }
}

const uppercase = new UppercaseTransform();

process.stdin.pipe(uppercase).pipe(process.stdout);
  • Explicació: Aquest codi crea un flux de transformació que converteix les dades d'entrada a majúscules. El flux process.stdin es canalitza a través del flux de transformació i després a process.stdout.

Introducció als Esdeveniments

Node.js està construït al voltant d'una arquitectura basada en esdeveniments. La classe EventEmitter és al cor d'aquesta arquitectura.

Conceptes Clau

  • EventEmitter: Una classe que et permet crear, emetre i escoltar esdeveniments.
  • on(event, listener): Afegeix un oient per a l'esdeveniment especificat.
  • emit(event, [arg1], [arg2], [...]): Emet l'esdeveniment especificat.

Exemple: EventEmitter Bàsic

const EventEmitter = require('events');
const myEmitter = new EventEmitter();

myEmitter.on('event', () => {
  console.log('Ha ocorregut un esdeveniment!');
});

myEmitter.emit('event');
  • Explicació: Aquest codi crea una instància de EventEmitter, afegeix un oient per a l'esdeveniment event, i després emet l'esdeveniment.

Gestió Avançada d'Esdeveniments

Pots passar arguments als oients d'esdeveniments i gestionar múltiples esdeveniments.

Exemple: Passant Arguments als Oients d'Esdeveniments

const EventEmitter = require('events');
const myEmitter = new EventEmitter();

myEmitter.on('greet', (name) => {
  console.log(`Hola, ${name}!`);
});

myEmitter.emit('greet', 'Alice');
  • Explicació: Aquest codi emet l'esdeveniment greet amb un argument, i l'oient mostra un salut personalitzat.

Combinant Fluxos i Esdeveniments

Els fluxos i els esdeveniments sovint treballen junts en aplicacions Node.js.

Exemple: Utilitzant Fluxos amb EventEmitter

const fs = require('fs');
const EventEmitter = require('events');
const myEmitter = new EventEmitter();

const readableStream = fs.createReadStream('example.txt', 'utf8');

readableStream.on('data', (chunk) => {
  myEmitter.emit('dataReceived', chunk);
});

myEmitter.on('dataReceived', (data) => {
  console.log(`Dades rebudes: ${data}`);
});
  • Explicació: Aquest codi llegeix dades d'un fitxer i emet un esdeveniment personalitzat dataReceived cada vegada que es llegeix un fragment de dades. L'oient de l'esdeveniment mostra les dades rebudes.

Conclusió

Els fluxos i els esdeveniments són conceptes fonamentals a Node.js que permeten una gestió eficient de les dades i una programació basada en esdeveniments. Entendre com utilitzar fluxos llegibles, escriuribles, dúplex i de transformació, així com com treballar amb la classe EventEmitter, és crucial per construir aplicacions Node.js escalables i eficients.

En dominar aquests conceptes, podràs gestionar grans quantitats de dades i crear aplicacions responsives que reaccionen a diversos esdeveniments de manera fluida.
