Introducció

Django proporciona una potent utilitat de línia de comandes anomenada manage.py que permet als desenvolupadors realitzar diverses tasques administratives. Tot i que Django ve amb un conjunt de comandes integrades, també pots crear les teves pròpies comandes de gestió personalitzades per automatitzar tasques específiques del teu projecte.

Per què utilitzar comandes de gestió personalitzades?

  • Automatització: Automatitza tasques repetitives com la importació/exportació de dades, neteja o processament per lots.
  • Conveniencia: Simplifica operacions complexes en comandes úniques.
  • Integració: Integra fàcilment amb altres eines i scripts.

Creació d'una comanda de gestió personalitzada

Guia pas a pas

  1. Crea el directori de la comanda:

    • Navega al directori de la teva aplicació Django.
    • Crea un directori anomenat management/commands dins del directori de la teva aplicació.
    mkdir -p myapp/management/commands

  2. Crea el fitxer de la comanda:

    • Dins del directori commands, crea un fitxer Python per a la teva comanda. El nom del fitxer serà el nom de la comanda.
    touch myapp/management/commands/mycommand.py

  3. Escriu el codi de la comanda:

    • Obre el fitxer acabat de crear i defineix la teva comanda ampliant BaseCommand.
    from django.core.management.base import BaseCommand

class Command(BaseCommand):
    help = 'Descripció de la teva comanda'

    def handle(self, *args, **kwargs):
        self.stdout.write('Hola, aquesta és la meva comanda personalitzada!')

Afegir arguments a la teva comanda

Pots afegir arguments a la teva comanda per fer-la més flexible.

from django.core.management.base import BaseCommand

class Command(BaseCommand):
    help = 'Descripció de la teva comanda'

    def add_arguments(self, parser):
        parser.add_argument('name', type=str, help='Nom de la persona a saludar')

    def handle(self, *args, **kwargs):
        name = kwargs['name']
        self.stdout.write(f'Hola, {name}!')

Exemple: Una comanda per netejar dades antigues

Creem una comanda que esborri registres més antics d'un nombre especificat de dies.

  1. Crea el fitxer de la comanda:

    touch myapp/management/commands/cleanup_old_data.py

  2. Escriu el codi de la comanda:

    from django.core.management.base import BaseCommand
from myapp.models import MyModel
from datetime import datetime, timedelta

class Command(BaseCommand):
    help = 'Esborra registres més antics d'un nombre especificat de dies'

    def add_arguments(self, parser):
        parser.add_argument('days', type=int, help='Nombre de dies')

    def handle(self, *args, **kwargs):
        days = kwargs['days']
        cutoff_date = datetime.now() - timedelta(days=days)
        old_records = MyModel.objects.filter(created_at__lt=cutoff_date)
        count = old_records.count()
        old_records.delete()
        self.stdout.write(f'S'han esborrat {count} registres més antics de {days} dies')

Executar la teva comanda personalitzada

Per executar la teva comanda personalitzada, utilitza l'script manage.py seguit del nom de la comanda i qualsevol argument.

python manage.py mycommand

Per a l'exemple de neteja:

python manage.py cleanup_old_data 30

Conclusió

Les comandes de gestió personalitzades a Django proporcionen una manera potent d'automatitzar i simplificar tasques administratives. Seguint els passos descrits en aquest tema, pots crear les teves pròpies comandes adaptades a les necessitats específiques del teu projecte. Ja sigui per a la neteja de dades, processament per lots o integració amb altres eines, les comandes personalitzades poden millorar significativament la teva productivitat i agilitzar el teu flux de treball.
