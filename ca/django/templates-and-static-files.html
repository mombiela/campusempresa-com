En aquesta secció, explorarem com Django gestiona les plantilles i els fitxers estàtics. Les plantilles s'utilitzen per generar contingut HTML dinàmic, mentre que els fitxers estàtics s'utilitzen per servir CSS, JavaScript i imatges. Entendre aquests conceptes és crucial per construir una aplicació web completament funcional.

Plantilles

Les plantilles de Django són una manera poderosa de generar HTML dinàmicament. Permeten separar la capa de presentació de la lògica de negoci, fent que el teu codi sigui més net i mantenible.

Conceptes Clau

  • Motor de Plantilles: Django utilitza el seu propi motor de plantilles per defecte, però es pot configurar per utilitzar-ne d'altres com Jinja2.
  • Etiquetes de Plantilles: Sintaxi especial dins de les plantilles per realitzar lògica com bucles i condicionals.
  • Filtres de Plantilles: S'utilitzen per modificar variables per a la seva visualització.
  • Herència de Plantilles: Permet crear una plantilla base i estendre-la en altres plantilles.

Crear i Utilitzar Plantilles

  1. Crear un Directori de Plantilles: Per defecte, Django busca plantilles en un directori anomenat templates dins de cada aplicació.

    myapp/
├── templates/
│   └── myapp/
│       └── index.html

  2. Configurar la Configuració de Plantilles: Assegura't que el teu fitxer settings.py estigui configurat per trobar les teves plantilles.

    TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [os.path.join(BASE_DIR, 'templates')],
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]

  3. Crear una Vista per Renderitzar la Plantilla: Utilitza vistes de Django per renderitzar plantilles.

    from django.shortcuts import render

def index(request):
    return render(request, 'myapp/index.html', {'message': 'Hello, World!'})

  4. Utilitzar Etiquetes i Filtres de Plantilles: Exemple d'una plantilla utilitzant etiquetes i filtres.

    <!-- templates/myapp/index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>My Django App</title>
</head>
<body>
    <h1>{{ message|upper }}</h1>
    {% if user.is_authenticated %}
        <p>Welcome, {{ user.username }}!</p>
    {% else %}
        <p>Please log in.</p>
    {% endif %}
</body>
</html>

Herència de Plantilles

L'herència de plantilles permet crear una plantilla base que altres plantilles poden estendre.

  1. Crear una Plantilla Base:

    <!-- templates/base.html -->
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}My Site{% endblock %}</title>
</head>
<body>
    <header>
        <h1>My Site</h1>
    </header>
    <main>
        {% block content %}{% endblock %}
    </main>
    <footer>
        <p>&copy; 2023 My Site</p>
    </footer>
</body>
</html>

  2. Estendre la Plantilla Base:

    <!-- templates/myapp/index.html -->
{% extends "base.html" %}

{% block title %}Home{% endblock %}

{% block content %}
    <h2>Welcome to My Site</h2>
    <p>{{ message }}</p>
{% endblock %}

Fitxers Estàtics

Els fitxers estàtics inclouen CSS, JavaScript i imatges que es serveixen al client. Django proporciona una manera de gestionar aquests fitxers de manera eficient.

Conceptes Clau

  • STATIC_URL: La URL a utilitzar quan es fa referència a fitxers estàtics.
  • STATICFILES_DIRS: Una llista de directoris on Django també buscarà fitxers estàtics.
  • collectstatic: Una comanda de gestió per recollir fitxers estàtics en una ubicació única per al desplegament.

Configurar Fitxers Estàtics

  1. Configurar Fitxers Estàtics a settings.py:

    STATIC_URL = '/static/'
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]

  2. Organitzar Fitxers Estàtics:

    myapp/
├── static/
│   └── myapp/
│       ├── css/
│       │   └── styles.css
│       ├── js/
│       │   └── scripts.js
│       └── images/
│           └── logo.png

  3. Utilitzar Fitxers Estàtics en Plantilles:

    <!-- templates/myapp/index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>My Django App</title>
    <link rel="stylesheet" type="text/css" href="{% static 'myapp/css/styles.css' %}">
</head>
<body>
    <h1>{{ message }}</h1>
    <img src="{% static 'myapp/images/logo.png' %}" alt="Logo">
    <script src="{% static 'myapp/js/scripts.js' %}"></script>
</body>
</html>

  4. Recollir Fitxers Estàtics per al Desplegament:

    python manage.py collectstatic
