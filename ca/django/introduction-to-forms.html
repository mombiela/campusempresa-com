Els formularis són una part crucial del desenvolupament web, permetent als usuaris interactuar amb la teva aplicació enviant dades. A Django, els formularis s'utilitzen per gestionar l'entrada de l'usuari, validar dades i desar-les a la base de dades. Aquest tema cobrirà els conceptes bàsics dels formularis de Django, com crear-los i utilitzar-los, i algunes característiques avançades.

Què són els Formularis de Django?

Els formularis de Django són una manera de gestionar l'entrada de l'usuari i la validació d'una manera estructurada i segura. Proporcionen una abstracció d'alt nivell per crear formularis HTML, validar l'entrada de l'usuari i processar les dades.

Conceptes Clau

  • Classe de Formulari: Una classe de Python que defineix els camps del formulari i les seves regles de validació.
  • Camps del Formulari: Elements individuals del formulari, com ara entrades de text, caselles de verificació i menús desplegables.
  • Validació: El procés de comprovar si les dades enviades per l'usuari compleixen certs criteris.
  • Renderització del Formulari: El procés de convertir la classe del formulari en HTML.

Creació d'un Formulari Simple

Per crear un formulari a Django, cal definir una classe de formulari al teu fitxer forms.py.

Exemple

# forms.py
from django import forms

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

Explicació

  • forms.Form: Aquesta és la classe base per a tots els formularis a Django.
  • name: Un CharField amb una longitud màxima de 100 caràcters.
  • email: Un EmailField que assegura que l'entrada és una adreça de correu electrònic vàlida.
  • message: Un CharField amb un widget Textarea per a l'entrada de múltiples línies.

Renderització de Formularis en Plantilles

Un cop hagis definit el teu formulari, cal renderitzar-lo en una plantilla.

Exemple

<!-- contact.html -->
<form method="post">
    {% csrf_token %}
    {{ form.as_p }}
    <button type="submit">Submit</button>
</form>

Explicació

  • {% csrf_token %}: Una mesura de seguretat per prevenir atacs de Cross-Site Request Forgery (CSRF).
  • {{ form.as_p }}: Renderitza els camps del formulari com a elements HTML <p>.
  • : Un botó d'enviament per enviar les dades del formulari.

Gestió de l'Enviament del Formulari

Per gestionar l'enviament del formulari, cal processar les dades del formulari a la teva vista.

Exemple

# views.py
from django.shortcuts import render
from .forms import ContactForm

def contact_view(request):
    if request.method == 'POST':
        form = ContactForm(request.POST)
        if form.is_valid():
            # Processar les dades
            name = form.cleaned_data['name']
            email = form.cleaned_data['email']
            message = form.cleaned_data['message']
            # Fer alguna cosa amb les dades
    else:
        form = ContactForm()
    return render(request, 'contact.html', {'form': form})

Explicació

  • request.method == 'POST': Comprova si el formulari ha estat enviat.
  • form.is_valid(): Valida les dades del formulari.
  • form.cleaned_data: Un diccionari que conté les dades validades del formulari.

Característiques Avançades

Els formularis de Django ofereixen diverses característiques avançades, com ara validació personalitzada, formsets i formularis de models.

Validació Personalitzada

Pots afegir mètodes de validació personalitzats a la teva classe de formulari.

Exemple

# forms.py
from django import forms

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

    def clean_name(self):
        name = self.cleaned_data.get('name')
        if "badword" in name:
            raise forms.ValidationError("Paraula inapropiada en el nom!")
        return name

Formsets

Els formsets et permeten gestionar múltiples instàncies d'un formulari en una sola pàgina.

Exemple

# forms.py
from django import forms
from django.forms import formset_factory

class ItemForm(forms.Form):
    name = forms.CharField(max_length=100)
    quantity = forms.IntegerField()

ItemFormSet = formset_factory(ItemForm, extra=3)

Formularis de Models

Els formularis de models són una manera convenient de crear formularis que corresponen a models de Django.

Exemple

# forms.py
from django import forms
from .models import Contact

class ContactForm(forms.ModelForm):
    class Meta:
        model = Contact
        fields = ['name', 'email', 'message']

Conclusió

Els formularis de Django proporcionen una manera potent i flexible de gestionar l'entrada de l'usuari i la validació. Entenent els conceptes bàsics de la creació de formularis, la renderització i la gestió de l'enviament, pots crear aplicacions web robustes. Les característiques avançades com la validació personalitzada, els formsets i els formularis de models amplien encara més les capacitats dels formularis de Django, convertint-los en una eina essencial per a qualsevol desenvolupador de Django.
