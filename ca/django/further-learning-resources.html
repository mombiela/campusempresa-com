Mantenir-se Actualitzat amb Django ►

A mesura que continues el teu viatge amb Django, és essencial tenir accés a una varietat de recursos que et puguin ajudar a aprofundir el teu coneixement i mantenir-te actualitzat amb els últims desenvolupaments. A continuació es presenten alguns recursos recomanats, categoritzats per tipus.

Documentació Oficial

  • Documentació de Django: La documentació oficial de Django és el recurs més complet disponible. Cobreix tot, des de tutorials bàsics fins a temes avançats.
    • Documentació de Django

Llibres

  • "Django for Beginners" de William S. Vincent: Aquest llibre és perfecte per a aquells que són nous a Django i volen construir una base sòlida.
  • "Django for APIs" de William S. Vincent: Es centra en la construcció d'APIs amb Django i Django REST Framework.
  • "Two Scoops of Django" de Daniel Roy Greenfeld i Audrey Roy Greenfeld: Ofereix millors pràctiques i consells per treballar amb Django en projectes del món real.

    • Udemy Django Courses

Exemple de Codi

Aquí tens un exemple senzill d'una vista de Django que recupera i mostra una llista d'ítems d'una base de dades:

# views.py
from django.shortcuts import render
from .models import Item

def item_list(request):
    items = Item.objects.all()
    return render(request, 'item_list.html', {'items': items})

En la plantilla corresponent, pots recórrer els ítems i mostrar-los:

<!-- item_list.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Llista d'Ítems</title>
</head>
<body>
    <h1>Ítems</h1>
    <ul>
        {% for item in items %}
            <li>{{ item.name }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Conclusió

Continuar aprenent i creixent com a desenvolupador de Django implica aprofitar una varietat de recursos. Ja sigui a través de la documentació oficial, llibres, cursos en línia, tutorials, fòrums comunitaris o projectes de codi obert, hi ha una gran quantitat d'informació disponible per ajudar-te a avançar en les teves habilitats. Mantén la curiositat, segueix experimentant i no dubtis a contactar amb la comunitat per obtenir suport. Feliç codificació!
