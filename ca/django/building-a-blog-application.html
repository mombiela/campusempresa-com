Introducció

En aquest curs, construirem una aplicació de blog totalment funcional utilitzant Django, un framework web de Python d'alt nivell. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat, cobrint tots els conceptes i característiques essencials de Django.

Configuració de l'Entorn

Abans de començar a construir la nostra aplicació de blog, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar Python: Assegura't de tenir Python instal·lat a la teva màquina. Pots descarregar-lo des de python.org.
  • Instal·lar Django: Utilitza pip per instal·lar Django. 
    pip install django
  • Crear un Projecte Django: Inicia un nou projecte Django. 
    django-admin startproject myblog
cd myblog

Creació de l'Aplicació de Blog

A continuació, crearem una nova aplicació Django dins del nostre projecte.

  • Crear l'App de Blog: 
    python manage.py startapp blog
  • Afegir l'App de Blog a les Apps Instal·lades: Obre myblog/settings.py i afegeix 'blog' a la llista INSTALLED_APPS.

Models

Definirem l'estructura de dades per al nostre blog utilitzant models de Django.

  • Crear Models de Blog: Obre blog/models.py i defineix els models per al nostre blog.

    from django.db import models
from django.utils import timezone

class Post(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    created_at = models.DateTimeField(default=timezone.now)
    updated_at = models.DateTimeField(auto_now=True)

    def __str__(self):
        return self.title

  • Migrar els Models: Crea i aplica migracions per actualitzar l'esquema de la base de dades.

    python manage.py makemigrations
python manage.py migrate

Interfície d'Administració

Django proporciona una interfície d'administració integrada per gestionar els nostres models.

  • Registrar Models a l'Administració: Obre blog/admin.py i registra el model Post. 
    from django.contrib import admin
from .models import Post

admin.site.register(Post)

Vistes i URLs

Crearem vistes per gestionar la lògica per mostrar les entrades del blog i configurarem URLs per dirigir les sol·licituds a aquestes vistes.

  • Crear Vistes: Obre blog/views.py i defineix vistes per llistar i mostrar les entrades del blog.

    from django.shortcuts import render
from .models import Post

def post_list(request):
    posts = Post.objects.all()
    return render(request, 'blog/post_list.html', {'posts': posts})

def post_detail(request, pk):
    post = Post.objects.get(pk=pk)
    return render(request, 'blog/post_detail.html', {'post': post})

  • Configurar URLs: Crea un fitxer blog/urls.py i configura patrons de URL.

    from django.urls import path
from . import views

urlpatterns = [
    path('', views.post_list, name='post_list'),
    path('post/<int:pk>/', views.post_detail, name='post_detail'),
]

    Actualitza el urls.py principal per incloure les URLs del blog.

    from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('blog.urls')),
]

Plantilles

Crearem plantilles HTML per renderitzar les nostres entrades del blog.

  • Crear Directori de Plantilles: Crea un directori templates dins de l'app blog i afegeix subdirectoris per a cada vista.

    blog/
└── templates/
    └── blog/
        ├── post_list.html
        └── post_detail.html

  • Plantilla de Llista d'Entrades: Crea post_list.html.

    <!DOCTYPE html>
<html>
<head>
    <title>Blog</title>
</head>
<body>
    <h1>Entrades del Blog</h1>
    <ul>
        {% for post in posts %}
            <li>
                <a href="{% url 'post_detail' post.pk %}">{{ post.title }}</a>
                <p>{{ post.content|truncatewords:30 }}</p>
            </li>
        {% endfor %}
    </ul>
</body>
</html>

  • Plantilla de Detall d'Entrada: Crea post_detail.html.

    <!DOCTYPE html>
<html>
<head>
    <title>{{ post.title }}</title>
</head>
<body>
    <h1>{{ post.title }}</h1>
    <p>{{ post.content }}</p>
    <p><a href="{% url 'post_list' %}">Tornar al Blog</a></p>
</body>
</html>

Formularis i Entrada d'Usuari

Afegirem funcionalitat per crear i editar entrades del blog utilitzant formularis de Django.

  • Crear Formularis: Obre blog/forms.py i defineix un formulari per al model Post.

    from django import forms
from .models import Post

class PostForm(forms.ModelForm):
    class Meta:
        model = Post
        fields = ['title', 'content']

  • Crear i Editar Vistes: Actualitza blog/views.py per incloure vistes per crear i editar entrades.

    from django.shortcuts import redirect
from .forms import PostForm

def post_new(request):
    if request.method == "POST":
        form = PostForm(request.POST)
        if form.is_valid():
            post = form.save(commit=False)
            post.save()
            return redirect('post_detail', pk=post.pk)
    else:
        form = PostForm()
    return render(request, 'blog/post_edit.html', {'form': form})

def post_edit(request, pk):
    post = Post.objects.get(pk=pk)
    if request.method == "POST":
        form = PostForm(request.POST, instance=post)
        if form.is_valid():
            post = form.save(commit=False)
            post.save()
            return redirect('post_detail', pk=post.pk)
    else:
        form = PostForm(instance=post)
    return render(request, 'blog/post_edit.html', {'form': form})

  • Actualitzar URLs: Afegeix patrons de URL per a les noves vistes a blog/urls.py.

    urlpatterns = [
    path('', views.post_list, name='post_list'),
    path('post/<int:pk>/', views.post_detail, name='post_detail'),
    path('post/new/', views.post_new, name='post_new'),
    path('post/<int:pk>/edit/', views.post_edit, name='post_edit'),
]

  • Crear Plantilla de Formulari: Crea post_edit.html.

    <!DOCTYPE html>
<html>
<head>
    <title>Nova Entrada</title>
</head>
<body>
    <h1>Nova Entrada</h1>
    <form method="POST">
        {% csrf_token %}
        {{ form.as_p }}
        <button type="submit">Desar</button>
    </form>
</body>
</html>
