Introducció

A Django, les plantilles s'utilitzen per generar contingut HTML dinàmic. Les plantilles poden incloure variables i lògica per mostrar dades des de les teves vistes. Aquesta secció cobrirà com passar context a les plantilles i com utilitzar etiquetes de plantilla per afegir lògica a les teves plantilles.

Context en Plantilles de Django

El context és un diccionari que mapatge noms de variables a objectes de Python. Aquest diccionari es passa al motor de plantilles per renderitzar la plantilla amb dades dinàmiques.

Passar Context des de les Vistes

Per passar context a una plantilla, cal incloure'l en la funció render a la teva vista.

from django.shortcuts import render

def my_view(request):
    context = {
        'name': 'John Doe',
        'age': 30,
        'hobbies': ['Reading', 'Cycling', 'Hiking']
    }
    return render(request, 'my_template.html', context)

En l'exemple anterior, el diccionari context conté tres claus: name, age, i hobbies. Aquestes es poden accedir a la plantilla.

Accedir al Context en les Plantilles

A la teva plantilla, pots accedir a les variables de context utilitzant doble claudàtor {{ }}.

<!DOCTYPE html>
<html>
<head>
    <title>My Template</title>
</head>
<body>
    <h1>Hello, {{ name }}!</h1>
    <p>You are {{ age }} years old.</p>
    <h2>Your Hobbies:</h2>
    <ul>
        {% for hobby in hobbies %}
            <li>{{ hobby }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Etiquetes de Plantilla

Les etiquetes de plantilla proporcionen lògica a la plantilla. Estan tancades en {% %} i es poden utilitzar per a bucles, condicionals, i més.

Etiquetes de Plantilla Comunes

Aquí hi ha algunes etiquetes de plantilla comunament utilitzades:

  • {% for %}: Bucle sobre una seqüència.
  • {% if %}: Declaració condicional.
  • {% block %}: Defineix un bloc que es pot sobrescriure en plantilles filles.
  • {% extends %}: Estén una plantilla base.
  • {% include %}: Inclou una altra plantilla.

Exemples d'Etiquetes de Plantilla

Bucle amb {% for %}

<ul>
    {% for item in item_list %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

Declaracions Condicionals amb {% if %}

{% if user.is_authenticated %}
    <p>Welcome, {{ user.username }}!</p>
{% else %}
    <p>Please log in.</p>
{% endif %}

Estendre Plantilles amb {% extends %} i {% block %}

<!-- base.html -->
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}My Site{% endblock %}</title>
</head>
<body>
    <header>
        <h1>My Site</h1>
    </header>
    <main>
        {% block content %}{% endblock %}
    </main>
</body>
</html>
<!-- child.html -->
{% extends "base.html" %}

{% block title %}Home{% endblock %}

{% block content %}
    <p>Welcome to the homepage!</p>
{% endblock %}

Incloure Plantilles amb {% include %}

<!-- sidebar.html -->
<div class="sidebar">
    <h2>Sidebar</h2>
    <p>This is the sidebar content.</p>
</div>
<!-- main.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Main Page</title>
</head>
<body>
    <h1>Main Content</h1>
    {% include "sidebar.html" %}
</body>
</html>

Conclusió

Entendre com passar context i utilitzar etiquetes de plantilla és crucial per crear plantilles dinàmiques i reutilitzables a Django. En dominar aquests conceptes, pots construir aplicacions web més eficients i mantenibles. Practica creant diferents vistes i plantilles, experimentant amb diverses dades de context, i utilitzant etiquetes de plantilla per controlar la lògica de renderització.
