Gestió de Contrasenyes a Django
Middleware ►

La gestió de contrasenyes és un aspecte crític de la seguretat de les aplicacions web. En aquesta secció, cobrim com gestionar les contrasenyes de manera segura a Django, des de nivells bàsics fins a avançats.

Introducció a la Gestió de Contrasenyes

  • Importància de la Seguretat de les Contrasenyes: Les contrasenyes són la primera línia de defensa contra l'accés no autoritzat. Una gestió adequada assegura que les dades dels usuaris es mantinguin segures.
  • Eines Integrades de Django: Django proporciona eines robustes per al hashing, la validació i la gestió de contrasenyes.

Emmagatzematge Segur de Contrasenyes

Hashing de Contrasenyes

Django utilitza un algoritme de hashing segur per emmagatzemar contrasenyes. Això assegura que, fins i tot si la base de dades és compromesa, les contrasenyes reals no s'exposin.

  • Algoritmes de Hashing: Django utilitza PBKDF2 per defecte, però també suporta altres algoritmes com Argon2, BCrypt i SHA-1.
  • Configuració: Pots configurar l'algoritme de hashing al fitxer settings.py.
# settings.py
PASSWORD_HASHERS = [
    'django.contrib.auth.hashers.PBKDF2PasswordHasher',
    'django.contrib.auth.hashers.PBKDF2SHA1PasswordHasher',
    'django.contrib.auth.hashers.Argon2PasswordHasher',
    'django.contrib.auth.hashers.BCryptSHA256PasswordHasher',
]

Creació i Emmagatzematge de Contrasenyes

Quan creïs un usuari, mai has d'emmagatzemar la contrasenya en text pla. En canvi, utilitza els mètodes integrats de Django per fer el hashing de la contrasenya.

from django.contrib.auth.models import User

# Crear un nou usuari amb una contrasenya hashada
user = User.objects.create_user(username='john', password='mypassword')
user.save()

Validació de Contrasenyes

Django proporciona una manera de validar les contrasenyes per assegurar-se que compleixen certs criteris (per exemple, longitud, complexitat).

Ús de Validadores Integrades

Django ve amb diversos validadores de contrasenyes integrades que pots configurar al fitxer settings.py.

# settings.py
AUTH_PASSWORD_VALIDATORS = [
    {
        'NAME': 'django.contrib.auth.password_validation.UserAttributeSimilarityValidator',
    },
    {
        'NAME': 'django.contrib.auth.password_validation.MinimumLengthValidator',
        'OPTIONS': {
            'min_length': 8,
        }
    },
    {
        'NAME': 'django.contrib.auth.password_validation.CommonPasswordValidator',
    },
    {
        'NAME': 'django.contrib.auth.password_validation.NumericPasswordValidator',
    },
]

Validadores de Contrasenyes Personalitzades

També pots crear validadores de contrasenyes personalitzades implementant els mètodes validate i get_help_text.

from django.core.exceptions import ValidationError
from django.utils.translation import gettext as _

class MyCustomPasswordValidator:
    def validate(self, password, user=None):
        if not any(char.isdigit() for char in password):
            raise ValidationError(
                _("The password must contain at least one digit."),
                code='password_no_digit',
            )

    def get_help_text(self):
        return _("Your password must contain at least one digit.")

Restabliment i Canvi de Contrasenya

Restabliment de Contrasenya

Django proporciona una vista integrada per al restabliment de contrasenya, que inclou l'enviament d'un correu electrònic amb un enllaç de restabliment.

  • Configuració de l'URL: Afegeix les URLs de restabliment de contrasenya al teu urls.py.
# urls.py
from django.contrib.auth import views as auth_views

urlpatterns = [
    path('password_reset/', auth_views.PasswordResetView.as_view(), name='password_reset'),
    path('password_reset/done/', auth_views.PasswordResetDoneView.as_view(), name='password_reset_done'),
    path('reset/<uidb64>/<token>/', auth_views.PasswordResetConfirmView.as_view(), name='password_reset_confirm'),
    path('reset/done/', auth_views.PasswordResetCompleteView.as_view(), name='password_reset_complete'),
]
  • Configuració del Correu Electrònic: Assegura't que la configuració del correu electrònic està configurada al settings.py.
# settings.py
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.example.com'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'your-email@example.com'
EMAIL_HOST_PASSWORD = 'your-email-password'

Canvi de Contrasenya

Django també proporciona vistes perquè els usuaris autenticats puguin canviar les seves contrasenyes.

  • Configuració de l'URL: Afegeix les URLs de canvi de contrasenya al teu urls.py.
# urls.py
urlpatterns += [
    path('password_change/', auth_views.PasswordChangeView.as_view(), name='password_change'),
    path('password_change/done/', auth_views.PasswordChangeDoneView.as_view(), name='password_change_done'),
]

Temes Avançats

Integració amb Autenticació de Tercers

  • Autenticació Social: Utilitza biblioteques com django-allauth o python-social-auth per integrar l'inici de sessió social (per exemple, Google, Facebook).
  • Autenticació de Doble Factor (2FA): Implementa 2FA utilitzant paquets com django-otp o django-two-factor-auth.

Polítiques de Contrasenyes

  • Imposar Caducitat de Contrasenyes: Implementa polítiques per forçar els usuaris a canviar les seves contrasenyes periòdicament.
  • Bloqueig de Comptes: Implementa mecanismes de bloqueig de comptes per prevenir atacs de força bruta.

Conclusió

La gestió de contrasenyes és un aspecte crucial de la seguretat de les aplicacions web. Django proporciona un conjunt complet d'eines i millors pràctiques per gestionar les contrasenyes de manera segura. Seguint aquestes directrius, pots assegurar-te que la teva aplicació està ben protegida contra amenaces de seguretat comunes.

  • Punts Clau:
    • Sempre fes hashing de les contrasenyes abans d'emmagatzemar-les.
    • Utilitza els validadores integrats de Django per imposar polítiques de contrasenyes fortes.
    • Implementa funcionalitats de restabliment i canvi de contrasenya.
    • Considera mesures de seguretat avançades com 2FA i mecanismes de bloqueig de comptes.

Dominant aquests conceptes, estaràs ben preparat per gestionar les contrasenyes de manera segura a les teves aplicacions Django.
