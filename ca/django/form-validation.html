Introducció a la Validació de Formularis

La validació de formularis és un aspecte crucial del desenvolupament web que assegura que les dades enviades pels usuaris siguin netes, correctes i segures. En Django, la validació de formularis es gestiona mitjançant el mòdul forms, que proporciona un marc robust per crear i validar formularis.

Creació de Formularis en Django

Abans d'endinsar-nos en la validació, entenguem com crear formularis en Django.

Utilitzant Formularis de Django

Django proporciona una classe Form que pots utilitzar per crear formularis.

from django import forms

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)
  • CharField: Utilitzat per a l'entrada de text.
  • EmailField: Assegura que l'entrada sigui una adreça de correu electrònic vàlida.
  • Textarea: Widget per a l'entrada de text de múltiples línies.

Validació Incorporada

Els formularis de Django vénen amb validació incorporada per a cada tipus de camp. Per exemple, EmailField comprovarà automàticament si l'entrada és una adreça de correu electrònic vàlida.

Exemple de Validació Incorporada

form = ContactForm({'name': 'John Doe', 'email': 'invalid-email', 'message': 'Hello'})
if form.is_valid():
    # Processa les dades
    pass
else:
    print(form.errors)

Sortida:

{'email': ['Introdueix una adreça de correu electrònic vàlida.']}

Validació Personalitzada

De vegades, la validació incorporada no és suficient. Django et permet afegir mètodes de validació personalitzats.

Validació Específica del Camp

Pots afegir validació personalitzada a camps individuals definint un mètode clean_<fieldname>.

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

    def clean_name(self):
        name = self.cleaned_data.get('name')
        if "admin" in name.lower():
            raise forms.ValidationError("El nom no pot contenir 'admin'.")
        return name

Validació a Nivell de Formulari

Per a la validació que implica múltiples camps, sobreescriu el mètode clean.

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

    def clean(self):
        cleaned_data = super().clean()
        name = cleaned_data.get('name')
        email = cleaned_data.get('email')

        if name and email and "example.com" in email:
            raise forms.ValidationError("No es permeten adreces de correu electrònic de 'example.com'.")

Tècniques Avançades de Validació

Per a escenaris de validació més complexos, pots utilitzar els validadors de Django i missatges d'error personalitzats.

Utilitzant Validadors

Django proporciona un conjunt de validadors que pots utilitzar amb els camps del formulari.

from django.core.validators import MinLengthValidator

class ContactForm(forms.Form):
    name = forms.CharField(max_length=100, validators=[MinLengthValidator(5)])
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

Missatges d'Error Personalitzats

Pots personalitzar els missatges d'error per a cada camp.

class ContactForm(forms.Form):
    name = forms.CharField(
        max_length=100,
        error_messages={
            'required': 'Si us plau, introdueix el teu nom.',
            'max_length': 'El nom no pot superar els 100 caràcters.'
        }
    )
    email = forms.EmailField(
        error_messages={
            'invalid': 'Introdueix una adreça de correu electrònic vàlida.'
        }
    )
    message = forms.CharField(widget=forms.Textarea)

Resum

La validació de formularis en Django és una característica poderosa que assegura la integritat i seguretat de l'entrada d'usuaris. Mitjançant la utilització de validació incorporada, mètodes de validació personalitzats i tècniques avançades com validadors i missatges d'error personalitzats, pots crear formularis robustos que gestionen una àmplia gamma d'escenaris de validació.

  • Validació Incorporada: Proporcionada automàticament per a cada tipus de camp.
  • Validació Personalitzada: Afegeix lògica de validació específica utilitzant els mètodes clean_<fieldname> i clean.
  • Tècniques Avançades: Utilitza els validadors de Django i missatges d'error personalitzats per a necessitats de validació més complexes.

En dominar aquestes tècniques, pots assegurar-te que les teves aplicacions Django gestionin l'entrada d'usuaris de manera efectiva i segura.
