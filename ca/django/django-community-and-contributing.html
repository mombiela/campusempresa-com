Introducció

La comunitat Django és una de les més vibrants i acollidores del món del codi obert. Participar en la comunitat pot proporcionar nombrosos beneficis, des d'aprendre i fer xarxa fins a contribuir i retornar. Aquesta secció et guiarà a través dels diversos aspectes de la comunitat Django i com pots contribuir-hi.

Comunitat Django

Components Clau de la Comunitat Django

  • Llista de Correu d'Usuaris de Django: Un lloc per fer preguntes i compartir coneixements.
  • Llista de Correu de Desenvolupadors de Django: Es centra en el desenvolupament de Django mateix.
  • Fòrum de Django: Un fòrum web per a discussions.
  • Canal IRC de Django: Xat en temps real amb altres usuaris i desenvolupadors de Django.
  • DjangoCon: Conferències anuals celebrades en diverses ubicacions arreu del món.
  • Meetups: Trobades locals de Django per fer xarxa i aprendre.

Recursos en Línia

  • Documentació Oficial: Documentació completa i detallada disponible a docs.djangoproject.com.
  • Blog de Django: Blog oficial amb actualitzacions i notícies.
  • Blogs i Tutorials de Tercers: Nombrosos blogs i tutorials creats per membres de la comunitat.

Xarxes Socials

  • Twitter: Segueix @djangoproject per a actualitzacions.
  • Reddit: Participa en discussions a r/django.
  • GitHub: Marca amb una estrella i bifurca el repositori de Django.

Contribuir a Django

Per què Contribuir?

  • Aprenentatge: Millora les teves habilitats treballant en problemes del món real.
  • Xarxa: Connecta amb altres desenvolupadors.
  • Retornar: Ajuda a millorar una eina que utilitzes.
  • Reconeixement: Construeix el teu portafoli i guanya reconeixement a la comunitat.

Maneres de Contribuir

  • Contribucions de Codi: Corregeix errors, afegeix funcionalitats o millora la documentació.
  • Documentació: Ajuda a millorar la documentació oficial.
  • Proves: Prova noves versions i informa d'errors.
  • Suport Comunitari: Respon preguntes a les llistes de correu, fòrums i IRC.
  • Patrocini: Dona suport financerament a la Django Software Foundation.

Començar amb les Contribucions de Codi

Configurar el Teu Entorn

  1. Bifurca el Repositori: Bifurca el repositori de Django a GitHub.
  2. Clona la Teva Bifurcació: Clona la teva bifurcació a la teva màquina local. 
    git clone https://github.com/el-teu-usuari/django.git
  3. Crea un Entorn Virtual: Configura un entorn virtual. 
    python -m venv env
source env/bin/activate  # A Windows usa `env\Scripts\activate`
  4. Instal·la les Dependències: Instal·la les dependències necessàries. 
    pip install -r requirements.txt

Fer Canvis

  1. Crea una Branca: Crea una nova branca per als teus canvis. 
    git checkout -b la-meva-branca-de-funcionalitat
  2. Fes els Teus Canvis: Edita el codi o la documentació.
  3. Executa les Proves: Assegura't que totes les proves passen. 
    python manage.py test
  4. Confirma els Teus Canvis: Confirma els teus canvis amb un missatge descriptiu. 
    git add .
git commit -m "Descripció dels canvis"
  5. Empeny a GitHub: Empeny la teva branca a GitHub. 
    git push origin la-meva-branca-de-funcionalitat

Enviar una Sol·licitud d'Extracció

  1. Obre una Sol·licitud d'Extracció: Ves al repositori de Django a GitHub i obre una sol·licitud d'extracció.
  2. Descriu els Teus Canvis: Proporciona una descripció detallada dels teus canvis.
  3. Aborda els Comentaris: Estigues preparat per fer canvis addicionals basats en els comentaris dels mantenidors.

Conclusió

Participar en la comunitat Django i contribuir al projecte pot ser increïblement gratificant. Ja sigui corregint errors, escrivint documentació o ajudant altres, les teves contribucions marquen la diferència. Participant, no només millores les teves pròpies habilitats, sinó que també ajudes a millorar Django per a tothom. Així que, llença't i converteix-te en un membre actiu de la comunitat Django!
