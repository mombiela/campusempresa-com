El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, explorarem alguns dels paquets més útils de Django que poden ajudar-te a millorar els teus projectes de Django. Aquests paquets poden simplificar tasques comunes, millorar el rendiment i afegir noves funcionalitats a les teves aplicacions.

1. Django Rest Framework (DRF)

Django Rest Framework és un conjunt d'eines potent i flexible per construir APIs web. És àmpliament utilitzat per crear APIs RESTful amb Django.

  • Característiques clau:

    • Serialització que suporta tant fonts de dades ORM com no ORM.
    • Autenticació i permisos.
    • Conjunts de vistes i routers per a un desenvolupament ràpid d'APIs.
    • API navegable per a proves i interacció fàcils.

  • Instal·lació:

    pip install djangorestframework

  • Exemple bàsic:

    # serializers.py
from rest_framework import serializers
from .models import MyModel

class MyModelSerializer(serializers.ModelSerializer):
    class Meta:
        model = MyModel
        fields = '__all__'

# views.py
from rest_framework import viewsets
from .models import MyModel
from .serializers import MyModelSerializer

class MyModelViewSet(viewsets.ModelViewSet):
    queryset = MyModel.objects.all()
    serializer_class = MyModelSerializer

# urls.py
from django.urls import path, include
from rest_framework.routers import DefaultRouter
from .views import MyModelViewSet

router = DefaultRouter()
router.register(r'mymodel', MyModelViewSet)

urlpatterns = [
    path('', include(router.urls)),
]

2. Django Allauth

Django Allauth és un conjunt integrat d'aplicacions de Django que aborden l'autenticació, el registre, la gestió de comptes i l'autenticació de comptes de tercers (socials).

  • Característiques clau:

    • Registre i autenticació d'usuaris.
    • Autenticació de comptes socials (per exemple, Facebook, Google).
    • Verificació de correu electrònic i gestió de contrasenyes.

  • Instal·lació:

    pip install django-allauth

  • Configuració bàsica:

    # settings.py
INSTALLED_APPS = [
    ...
    'django.contrib.sites',
    'allauth',
    'allauth.account',
    'allauth.socialaccount',
    'allauth.socialaccount.providers.google',
    ...
]

SITE_ID = 1

AUTHENTICATION_BACKENDS = (
    'django.contrib.auth.backends.ModelBackend',
    'allauth.account.auth_backends.AuthenticationBackend',
)

# urls.py
from django.urls import path, include

urlpatterns = [
    ...
    path('accounts/', include('allauth.urls')),
    ...
]

3. Django Debug Toolbar

Django Debug Toolbar és un conjunt configurable de panells que mostren diverses informacions de depuració sobre la sol·licitud/resposta actual.

  • Característiques clau:

    • Consultes SQL i el seu temps d'execució.
    • Ús de la memòria cau.
    • Temps de renderització de plantilles.
    • Capçaleres de sol·licitud i resposta.

  • Instal·lació:

    pip install django-debug-toolbar

  • Configuració bàsica:

    # settings.py
INSTALLED_APPS = [
    ...
    'debug_toolbar',
    ...
]

MIDDLEWARE = [
    ...
    'debug_toolbar.middleware.DebugToolbarMiddleware',
    ...
]

INTERNAL_IPS = [
    '127.0.0.1',
]

# urls.py
from django.urls import path, include

urlpatterns = [
    ...
    path('__debug__/', include('debug_toolbar.urls')),
    ...
]

4. Django Crispy Forms

Django Crispy Forms és una aplicació de Django que t'ajuda a gestionar els formularis de Django. Et permet controlar el comportament de renderització dels teus formularis d'una manera molt elegant i DRY.

  • Característiques clau:

    • Formularis DRY.
    • Suport per a Bootstrap, Foundation i altres frameworks CSS.
    • Dissenys de formularis personalitzats.

  • Instal·lació:

    pip install django-crispy-forms

  • Exemple bàsic:

    # settings.py
INSTALLED_APPS = [
    ...
    'crispy_forms',
    ...
]

CRISPY_TEMPLATE_PACK = 'bootstrap4'

# forms.py
from django import forms
from crispy_forms.helper import FormHelper
from crispy_forms.layout import Submit

class MyForm(forms.Form):
    name = forms.CharField(max_length=100)
    email = forms.EmailField()

    def __init__(self, *args, **kwargs):
        super(MyForm, self).__init__(*args, **kwargs)
        self.helper = FormHelper()
        self.helper.form_method = 'post'
        self.helper.add_input(Submit('submit', 'Submit'))

5. Django Celery

Django Celery és una cua de tasques asíncrona basada en el pas de missatges distribuïts. Es centra en l'operació en temps real però també suporta la programació de tasques.

  • Característiques clau:

    • Execució de tasques asíncrones.
    • Programació de tasques.
    • Integració amb l'ORM de Django i la memòria cau.

  • Instal·lació:

    pip install celery

  • Configuració bàsica:

    # settings.py
CELERY_BROKER_URL = 'redis://localhost:6379/0'
CELERY_RESULT_BACKEND = 'redis://localhost:6379/0'

# celery.py
from __future__ import absolute_import, unicode_literals
import os
from celery import Celery

os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'myproject.settings')

app = Celery('myproject')
app.config_from_object('django.conf:settings', namespace='CELERY')
app.autodiscover_tasks()

# tasks.py
from celery import shared_task

@shared_task
def add(x, y):
    return x + y
