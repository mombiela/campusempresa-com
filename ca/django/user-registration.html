Introducció

El registre d'usuaris és una característica fonamental en les aplicacions web, permetent als usuaris crear comptes i accedir a contingut personalitzat. En aquesta secció, cobrirem com implementar el registre d'usuaris en Django, des de la configuració bàsica fins a la personalització avançada.

Configuració del Projecte

Abans de començar amb el registre d'usuaris, assegura't de tenir Django instal·lat i un projecte creat.

pip install django
django-admin startproject myproject
cd myproject
python manage.py startapp users

Registre Bàsic d'Usuaris

Creació del Model d'Usuari

Django ve amb un model d'usuari integrat, però pots estendre'l si cal. Per simplicitat, utilitzarem el model d'usuari per defecte de Django.

Creació del Formulari de Registre

Utilitzarem els formularis integrats de Django per crear un formulari de registre d'usuari.

# users/forms.py
from django import forms
from django.contrib.auth.models import User
from django.contrib.auth.forms import UserCreationForm

class UserRegisterForm(UserCreationForm):
    email = forms.EmailField()

    class Meta:
        model = User
        fields = ['username', 'email', 'password1', 'password2']

Creació de la Vista de Registre

Crea una vista per gestionar el formulari de registre.

# users/views.py
from django.shortcuts import render, redirect
from django.contrib import messages
from .forms import UserRegisterForm

def register(request):
    if request.method == 'POST':
        form = UserRegisterForm(request.POST)
        if form.is_valid():
            form.save()
            username = form.cleaned_data.get('username')
            messages.success(request, f'Compte creat per {username}!')
            return redirect('login')
    else:
        form = UserRegisterForm()
    return render(request, 'users/register.html', {'form': form})

Creació de la Plantilla de Registre

Crea una plantilla per mostrar el formulari de registre.

<!-- users/templates/users/register.html -->
{% extends "base_generic.html" %}

{% block content %}
  <div class="container">
    <h2>Registra't</h2>
    <form method="POST">
      {% csrf_token %}
      {{ form.as_p }}
      <button type="submit" class="btn btn-primary">Registra't</button>
    </form>
  </div>
{% endblock %}

Afegir Patrons d'URL

Afegeix el patró d'URL per a la vista de registre.

# users/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('register/', views.register, name='register'),
]

Inclou les URLs de l'aplicació users en les URLs principals del projecte.

# myproject/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('users/', include('users.urls')),
]

Registre Avançat d'Usuaris

Personalització del Model d'Usuari

Si necessites emmagatzemar informació addicional, pots estendre el model d'usuari.

# users/models.py
from django.contrib.auth.models import AbstractUser
from django.db import models

class CustomUser(AbstractUser):
    age = models.PositiveIntegerField(null=True, blank=True)

Actualitza la configuració per utilitzar el model d'usuari personalitzat.

# myproject/settings.py
AUTH_USER_MODEL = 'users.CustomUser'

Gestió de Perfils d'Usuari

Crea un model de perfil per emmagatzemar informació addicional de l'usuari.

# users/models.py
from django.db import models
from django.contrib.auth.models import User

class Profile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    image = models.ImageField(default='default.jpg', upload_to='profile_pics')

    def __str__(self):
        return f'{self.user.username} Profile'

Senyals per a la Creació de Perfils

Utilitza senyals de Django per crear automàticament un perfil quan un usuari es registra.

# users/signals.py
from django.db.models.signals import post_save
from django.contrib.auth.models import User
from django.dispatch import receiver
from .models import Profile

@receiver(post_save, sender=User)
def create_profile(sender, instance, created, **kwargs):
    if created:
        Profile.objects.create(user=instance)

@receiver(post_save, sender=User)
def save_profile(sender, instance, **kwargs):
    instance.profile.save()

Registra les senyals a apps.py.

# users/apps.py
from django.apps import AppConfig

class UsersConfig(AppConfig):
    name = 'users'

    def ready(self):
        import users.signals
