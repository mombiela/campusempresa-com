Introducció als Serialitzadors

Els serialitzadors a Django s'utilitzen per convertir tipus de dades complexos, com ara querysets i instàncies de models, en tipus de dades natius de Python que després es poden renderitzar fàcilment en JSON, XML o altres tipus de contingut. També proporcionen deserialització, permetent que les dades analitzades es converteixin de nou en tipus complexos, després de validar primer les dades entrants.

Per què utilitzar serialitzadors?

  • Conversió de dades: Convertir dades entre diferents formats (per exemple, JSON a objectes Python).
  • Validació: Assegurar que les dades compleixin el format i les restriccions requerides.
  • Simplificació: Simplificar el procés de crear APIs gestionant la serialització i deserialització de dades.

Ús bàsic dels serialitzadors

Crear un serialitzador simple

Per crear un serialitzador, cal definir una classe que hereti de serializers.Serializer i especificar els camps que es volen incloure.

from rest_framework import serializers

class SimpleSerializer(serializers.Serializer):
    name = serializers.CharField(max_length=100)
    age = serializers.IntegerField()

Serialitzar dades

Es pot crear una instància del serialitzador i passar-hi les dades que es volen serialitzar.

data = {'name': 'John Doe', 'age': 30}
serializer = SimpleSerializer(data)
print(serializer.data)

Deserialitzar dades

La deserialització implica validar les dades i després convertir-les de nou en un tipus complex.

data = {'name': 'John Doe', 'age': 30}
serializer = SimpleSerializer(data=data)
if serializer.is_valid():
    validated_data = serializer.validated_data
    print(validated_data)
else:
    print(serializer.errors)

Serialitzadors de models

Els serialitzadors de models proporcionen una drecera per crear serialitzadors que tracten amb instàncies de models i querysets. Generen automàticament un conjunt de camps basats en el model.

Crear un serialitzador de model

Primer, defineix un model de Django.

from django.db import models

class Person(models.Model):
    name = models.CharField(max_length=100)
    age = models.IntegerField()

Després, crea un serialitzador per a aquest model.

from rest_framework import serializers

class PersonSerializer(serializers.ModelSerializer):
    class Meta:
        model = Person
        fields = ['name', 'age']

Utilitzar el serialitzador de model

Es pot utilitzar el serialitzador de model per serialitzar i deserialitzar instàncies de models.

# Serialitzar una instància de model
person = Person(name='John Doe', age=30)
serializer = PersonSerializer(person)
print(serializer.data)

# Deserialitzar dades a una instància de model
data = {'name': 'John Doe', 'age': 30}
serializer = PersonSerializer(data=data)
if serializer.is_valid():
    person = serializer.save()
    print(person)
else:
    print(serializer.errors)

Funcionalitats avançades dels serialitzadors

Serialitzadors niats

Es poden incloure serialitzadors dins d'altres serialitzadors per gestionar estructures de dades niades.

class AddressSerializer(serializers.Serializer):
    street = serializers.CharField(max_length=100)
    city = serializers.CharField(max_length=100)

class PersonSerializer(serializers.Serializer):
    name = serializers.CharField(max_length=100)
    age = serializers.IntegerField()
    address = AddressSerializer()

Mètodes del serialitzador

Es poden afegir mètodes personalitzats per gestionar la lògica complexa de serialització i deserialització.

class PersonSerializer(serializers.ModelSerializer):
    full_name = serializers.SerializerMethodField()

    class Meta:
        model = Person
        fields = ['name', 'age', 'full_name']

    def get_full_name(self, obj):
        return f"{obj.name} (Age: {obj.age})"

Validació de dades

Es pot afegir lògica de validació personalitzada als serialitzadors.

class PersonSerializer(serializers.ModelSerializer):
    class Meta:
        model = Person
        fields = ['name', 'age']

    def validate_age(self, value):
        if value < 0:
            raise serializers.ValidationError("Age cannot be negative")
        return value

Conclusió

Els serialitzadors a Django REST Framework són eines potents que simplifiquen el procés de convertir tipus de dades complexos a i des de tipus de dades natius de Python. Gestionen la validació, serialització i deserialització, facilitant la creació d'APIs robustes. Entenent i utilitzant les funcionalitats bàsiques i avançades dels serialitzadors, es pot crear codi eficient i mantenible per a les aplicacions Django.
