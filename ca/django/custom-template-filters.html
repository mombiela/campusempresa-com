En aquesta secció, explorarem com crear filtres de plantilla personalitzats a Django. Els filtres de plantilla són una característica potent que et permet modificar la visualització de les variables a les teves plantilles. Al final d'aquesta secció, seràs capaç de crear i utilitzar els teus propis filtres personalitzats per millorar les teves aplicacions Django.

Què són els Filtres de Plantilla?

  • Definició: Els filtres de plantilla s'utilitzen per transformar el valor d'una variable abans que es rendeixi en una plantilla.
  • Ús: S'apliquen utilitzant el caràcter de tub (|) a les plantilles de Django.

Filtres Incorporats

Django ve amb una varietat de filtres incorporats com date, length, lower, i upper. Aquí tens un exemple d'ús d'un filtre incorporat:

{{ my_date|date:"Y-m-d" }}

En aquest exemple, el filtre date formata la variable my_date al format "Any-Mes-Dia".

Creant Filtres de Plantilla Personalitzats

Per crear un filtre de plantilla personalitzat, segueix aquests passos:

  1. Crea un Directori templatetags: Dins de la teva aplicació Django, crea un directori anomenat templatetags.
  2. Crea un Mòdul de Python: Dins del directori templatetags, crea un fitxer Python (per exemple, custom_filters.py).
  3. Registra el Filtre: Utilitza el decorador @register.filter de Django per registrar el teu filtre personalitzat.

Exemple: Creant un Filtre Personalitzat

Creem un filtre personalitzat que capitalitzi la primera lletra de cada paraula en una cadena.

  1. Crea el Directori templatetags i el Mòdul:

    myapp/
    templatetags/
        __init__.py
        custom_filters.py

  2. Defineix el Filtre Personalitzat:

    # myapp/templatetags/custom_filters.py
from django import template

register = template.Library()

@register.filter(name='capitalize_words')
def capitalize_words(value):
    """Capitalitza la primera lletra de cada paraula en la cadena."""
    return ' '.join(word.capitalize() for word in value.split())

  3. Carrega i Utilitza el Filtre Personalitzat en una Plantilla:

    {% load custom_filters %}

<p>{{ "hello world"|capitalize_words }}</p>

    Això es renderitzarà com:

    <p>Hello World</p>

Filtres Personalitzats Avançats

Els filtres personalitzats també poden acceptar arguments. Creem un filtre que truncarà una cadena a un nombre específic de caràcters.

Exemple: Filtre de Truncament amb Arguments

  1. Defineix el Filtre de Truncament:

    # myapp/templatetags/custom_filters.py
from django import template

register = template.Library()

@register.filter(name='truncate_chars')
def truncate_chars(value, num_chars):
    """Trunca la cadena al nombre especificat de caràcters."""
    si la longitud de la cadena és més gran que num_chars:
        return value[:num_chars] + '...'
    return value

  2. Utilitza el Filtre de Truncament en una Plantilla:

    {% load custom_filters %}

<p>{{ "This is a long sentence that needs to be truncated."|truncate_chars:20 }}</p>

    Això es renderitzarà com:

    <p>This is a long sent...</p>

Resum

  • Filtres de Plantilla: Modifiquen la visualització de les variables a les plantilles de Django.
  • Filtres Incorporats: Django proporciona molts filtres incorporats per a tasques comunes.
  • Filtres Personalitzats: Pots crear filtres personalitzats definint una funció i registrant-la amb el sistema de plantilles de Django.
  • Filtres Avançats: Els filtres personalitzats també poden acceptar arguments per proporcionar més funcionalitat.

Creant filtres de plantilla personalitzats, pots adaptar la presentació de les teves dades per satisfer les necessitats específiques de la teva aplicació, fent les teves plantilles més potents i flexibles.
