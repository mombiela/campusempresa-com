Introducció

En aquest curs, construirem una xarxa social utilitzant Django, un framework web de Python d'alt nivell. Aquest projecte et portarà des del nivell principiant fins a l'avançat, cobrint conceptes essencials i característiques de Django.

Configurant l'Entorn

  • Instal·lar Python i Django
  • Configurar un entorn virtual
  • Crear un nou projecte Django

Exemple de Codi: Configurant un Projecte Django

# Instal·lar Django
pip install django

# Crear un nou projecte Django
django-admin startproject social_network

# Navegar dins del directori del projecte
cd social_network

# Iniciar el servidor de desenvolupament
python manage.py runserver

Creant el Model d'Usuari

  • Utilitzant el model d'Usuari integrat de Django
  • Estenent el model d'Usuari amb un model de Perfil

Exemple de Codi: Estenent el Model d'Usuari

# models.py
from django.contrib.auth.models import User
from django.db import models

class Profile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    bio = models.TextField(max_length=500, blank=True)
    location = models.CharField(max_length=30, blank=True)
    birth_date = models.DateField(null=True, blank=True)

    def __str__(self):
        return self.user.username

Exemple de Codi: Formulari de Registre d'Usuari

# forms.py
from django import forms
from django.contrib.auth.models import User
from django.contrib.auth.forms import UserCreationForm

class SignUpForm(UserCreationForm):
    email = forms.EmailField(max_length=200, help_text='Required')

    class Meta:
        model = User
        fields = ('username', 'email', 'password1', 'password2')

Construint les Característiques de la Xarxa Social

  • Creant i gestionant publicacions
  • Implementant perfils d'usuaris
  • Afegint un sistema de seguiment

Exemple de Codi: Creant un Model de Publicació

# models.py
from django.db import models
from django.contrib.auth.models import User

class Post(models.Model):
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.content[:20]

Implementant el Sistema de Seguiment

  • Creant un model de Seguiment
  • Gestionant les relacions de seguiment
  • Mostrant llistes de seguidors i seguits

Exemple de Codi: Model de Seguiment

# models.py
class Follow(models.Model):
    follower = models.ForeignKey(User, related_name='following', on_delete=models.CASCADE)
    following = models.ForeignKey(User, related_name='followers', on_delete=models.CASCADE)
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.follower} follows {self.following}"

Exemple de Codi: Escriure una Prova Simple

# tests.py
from django.test import TestCase
from django.contrib.auth.models import User
from .models import Post

class PostModelTest(TestCase):
    def setUp(self):
        self.user = User.objects.create_user(username='testuser', password='12345')
        self.post = Post.objects.create(author=self.user, content='Just a test post')

    def test_post_content(self):
        self.assertEqual(self.post.content, 'Just a test post')
