Introducció

En aquesta secció, aprofundirem en la gestió avançada de formularis a Django. Aprendràs a crear formularis complexos, gestionar la validació de formularis i administrar formsets. Al final d'aquesta secció, hauràs de ser capaç de construir i gestionar formularis sofisticats en les teves aplicacions Django.

Conceptes Clau

  • Validació Personalitzada de Formularis
  • Formularis de Model
  • Formsets
  • Gestió de Càrregues de Fitxers
  • Formularis Dinàmics

Validació Personalitzada de Formularis

La validació personalitzada de formularis et permet imposar regles específiques per als camps del formulari.

Exemple

from django import forms

class CustomForm(forms.Form):
    name = forms.CharField(max_length=100)
    age = forms.IntegerField()

    def clean_age(self):
        age = self.cleaned_data.get('age')
        if age < 18:
            raise forms.ValidationError("Age must be at least 18.")
        return age

En aquest exemple, definim un mètode de validació personalitzat clean_age per assegurar que l'edat sigui almenys 18.

Formularis de Model

Els formularis de model són una manera convenient de crear formularis vinculats a models de Django.

Exemple

from django import forms
from .models import Person

class PersonForm(forms.ModelForm):
    class Meta:
        model = Person
        fields = ['name', 'age', 'email']

Aquest exemple mostra com crear un formulari basat en el model Person, generant automàticament camps per a name, age i email.

Formsets

Els formsets et permeten gestionar múltiples instàncies d'un formulari en una sola pàgina.

Exemple

from django import forms
from django.forms import formset_factory

class ItemForm(forms.Form):
    name = forms.CharField(max_length=100)
    quantity = forms.IntegerField()

ItemFormSet = formset_factory(ItemForm, extra=3)

Aquí, creem un formset amb tres instàncies de ItemForm.

Gestió de Càrregues de Fitxers

Django proporciona suport integrat per gestionar càrregues de fitxers.

Exemple

from django import forms

class UploadFileForm(forms.Form):
    title = forms.CharField(max_length=50)
    file = forms.FileField()

En aquest exemple, definim un formulari amb un camp de fitxer. La vista que gestiona aquest formulari hauria de guardar el fitxer carregat.

Exemple de Vista

from django.shortcuts import render
from django.http import HttpResponseRedirect
from .forms import UploadFileForm

def upload_file(request):
    if request.method == 'POST':
        form = UploadFileForm(request.POST, request.FILES)
        if form.is_valid():
            handle_uploaded_file(request.FILES['file'])
            return HttpResponseRedirect('/success/')
    else:
        form = UploadFileForm()
    return render(request, 'upload.html', {'form': form})

def handle_uploaded_file(f):
    with open('some/file/name.txt', 'wb+') as destination:
        for chunk in f.chunks():
            destination.write(chunk)

Aquesta vista gestiona la càrrega del fitxer i el guarda al servidor.

Formularis Dinàmics

Els formularis dinàmics et permeten afegir o eliminar camps del formulari basant-se en l'entrada de l'usuari o altres condicions.

Exemple

from django import forms

class DynamicForm(forms.Form):
    def __init__(self, *args, **kwargs):
        super(DynamicForm, self).__init__(*args, **kwargs)
        self.fields['dynamic_field'] = forms.CharField(max_length=100)

En aquest exemple, afegim dinàmicament un camp al formulari durant la inicialització.

Conclusió

La gestió avançada de formularis a Django implica validació personalitzada, formularis de model, formsets, càrregues de fitxers i formularis dinàmics. Dominar aquestes tècniques et permetrà crear formularis robustos i fàcils d'usar en les teves aplicacions Django. Practica aquests conceptes amb exemples del món real per consolidar la teva comprensió.
