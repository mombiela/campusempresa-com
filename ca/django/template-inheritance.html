El Projecte Sobre nosaltres Contribuir Donacions Llicència

L'herència de plantilles és una característica poderosa a Django que et permet crear una plantilla base i estendre-la per crear plantilles més específiques. Això ajuda a mantenir una aparença consistent a tot el teu lloc web i facilita la gestió i actualització de les teves plantilles.

Conceptes Clau

  • Plantilla Base: Una plantilla general que conté elements comuns com capçaleres, peus de pàgina i barres de navegació.
  • Plantilla Filla: Una plantilla més específica que estèn la plantilla base i sobreescriu certes seccions.
  • Blocs: Seccions a la plantilla base que poden ser sobreescrites per plantilles filles.

Creant una Plantilla Base

La plantilla base és la fundació de la teva estructura d'herència de plantilles. Conté el disseny i els elements comuns que es compartiran a través de múltiples pàgines.

Exemple: `base.html`

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}El Meu Lloc Web{% endblock %}</title>
</head>
<body>
    <header>
        <h1>Benvingut al Meu Lloc Web</h1>
        <nav>
            <ul>
                <li><a href="/">Inici</a></li>
                <li><a href="/about/">Sobre</a></li>
                <li><a href="/contact/">Contacte</a></li>
            </ul>
        </nav>
    </header>
    <main>
        {% block content %}
        <!-- El contingut per defecte va aquí -->
        {% endblock %}
    </main>
    <footer>
        <p>&copy; 2023 El Meu Lloc Web</p>
    </footer>
</body>
</html>

En aquest exemple:

  • {% block title %} i {% block content %} són espais reservats que poden ser sobreescrits per plantilles filles.

Creant una Plantilla Filla

Una plantilla filla estèn la plantilla base i proporciona contingut específic per als espais reservats definits a la plantilla base.

Exemple: `home.html`

{% extends "base.html" %}

{% block title %}Inici - El Meu Lloc Web{% endblock %}

{% block content %}
    <h2>Pàgina d'Inici</h2>
    <p>Benvingut a la pàgina d'inici del meu lloc web.</p>
{% endblock %}

En aquest exemple:

  • {% extends "base.html" %} diu a Django que aquesta plantilla ha d'utilitzar base.html com a base.
  • {% block title %} i {% block content %} sobreescriuen els blocs corresponents a la plantilla base.

Utilitzant l'Herència de Plantilles a les Vistes

Per renderitzar una plantilla que utilitza herència, simplement especifiques la plantilla filla a la teva vista.

Exemple: `views.py`

from django.shortcuts import render

def home(request):
    return render(request, 'home.html')

En aquest exemple:

  • La vista home renderitza la plantilla home.html, que al seu torn estèn base.html.

Ús Avançat

Blocs Niats

Pots niar blocs dins d'altres blocs per crear plantilles més complexes.

Exemple: Blocs Niats

{% extends "base.html" %}

{% block content %}
    <h2>Pàgina d'Inici</h2>
    <p>Benvingut a la pàgina d'inici del meu lloc web.</p>
    {% block extra_content %}
    <!-- Contingut addicional pot ser afegit aquí per plantilles filles -->
    {% endblock %}
{% endblock %}

Exemple: Sobreescrivint Blocs Niats

{% extends "home.html" %}

{% block extra_content %}
    <p>Aquest és un contingut extra afegit a la pàgina d'inici.</p>
{% endblock %}

Conclusió

L'herència de plantilles a Django és una eina poderosa per crear aplicacions web mantenibles i escalables. Utilitzant plantilles base i plantilles filles, pots assegurar una aparença consistent a tot el teu lloc mentre fas que sigui fàcil actualitzar i gestionar les teves plantilles. Entendre i utilitzar l'herència de plantilles millorarà significativament la teva eficiència i la qualitat dels teus projectes Django.
