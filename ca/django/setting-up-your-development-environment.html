Introducció

Configurar el teu entorn de desenvolupament és el primer pas per començar a treballar amb Django. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per al desenvolupament amb Django. Al final d'aquesta secció, tindràs un entorn de desenvolupament de Django completament funcional.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb un sistema operatiu (Windows, macOS o Linux).
  • Coneixements bàsics de programació en Python.
  • Connexió a Internet per descarregar el programari necessari.

Pas 1: Instal·lar Python

Django és un framework web de Python, així que necessites tenir Python instal·lat a la teva màquina.

Windows

  1. Descarrega l'última versió de Python des del lloc web oficial.
  2. Executa l'instal·lador i assegura't de marcar la casella que diu "Add Python to PATH".
  3. Verifica la instal·lació obrint el Símbol del sistema i escrivint: 
    python --version

macOS

  1. Obre Terminal.
  2. Utilitza Homebrew per instal·lar Python (si no tens Homebrew, instal·la'l des d'aquí): 
    brew install python
  3. Verifica la instal·lació escrivint: 
    python3 --version

Linux

  1. Obre Terminal.
  2. Utilitza el gestor de paquets per instal·lar Python: 
    sudo apt-get update
sudo apt-get install python3
  3. Verifica la instal·lació escrivint: 
    python3 --version

Pas 2: Instal·lar pip

pip és l'instal·lador de paquets per a Python. Normalment s'inclou amb Python, però pots instal·lar-lo manualment si cal.

Verificar la instal·lació de pip

pip --version

Instal·lar pip (si no està instal·lat)

python -m ensurepip --upgrade

Pas 3: Configurar un entorn virtual

Un entorn virtual ajuda a gestionar les dependències de diferents projectes per separat.

Crear un entorn virtual

  1. Navega al directori del teu projecte: 
    cd path/to/your/project
  2. Crea un entorn virtual: 
    python -m venv myenv

Activar l'entorn virtual

  • Windows: 
    myenv\Scripts\activate
  • macOS/Linux: 
    source myenv/bin/activate

Desactivar l'entorn virtual

Per desactivar l'entorn virtual, simplement escriu:

deactivate

Pas 4: Instal·lar Django

Amb l'entorn virtual activat, instal·la Django utilitzant pip.

pip install django

Verifica la instal·lació escrivint:

django-admin --version

Pas 5: Configurar un projecte Django

  1. Crea un nou projecte Django: 
    django-admin startproject myproject
  2. Navega al directori del projecte: 
    cd myproject
  3. Executa el servidor de desenvolupament per assegurar-te que tot està configurat correctament: 
    python manage.py runserver

Hauries de veure una sortida indicant que el servidor està en funcionament. Obre un navegador web i ves a http://127.0.0.1:8000/ per veure la pàgina de benvinguda de Django.

Conclusió

Felicitats! Has configurat amb èxit el teu entorn de desenvolupament de Django. Ara tens Python, pip, un entorn virtual i Django instal·lats. També has creat un nou projecte Django i has executat el servidor de desenvolupament. Aquesta configuració servirà com a base per a tots els teus projectes Django. A les següents seccions, aprofundirem en les característiques de Django i començarem a construir aplicacions web.
