Introducció

En aquesta secció, et guiarem a través del procés de creació del teu primer projecte Django. Django és un framework web de Python d'alt nivell que fomenta el desenvolupament ràpid i un disseny net i pragmàtic. Al final d'aquesta secció, tindràs un projecte bàsic de Django en funcionament.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Python instal·lat (versió 3.6 o superior).
  • Pip (instal·lador de paquets de Python).
  • Virtualenv (opcional però recomanat per gestionar les dependències).

Configurant el teu entorn

  1. Instal·la Django: Primer, has d'instal·lar Django. Pots fer-ho utilitzant pip. 
    pip install django
  2. Crea un entorn virtual: És una bona pràctica crear un entorn virtual per al teu projecte. 
    python -m venv myenv
source myenv/bin/activate  # A Windows usa `myenv\Scripts\activate`

Creant un projecte Django

  1. Inicia un nou projecte: Utilitza la comanda django-admin per iniciar un nou projecte.

    django-admin startproject myproject

    Aquesta comanda crea un nou directori anomenat myproject amb l'estructura següent:

    myproject/
    manage.py
    myproject/
        __init__.py
        settings.py
        urls.py
        wsgi.py

  2. Entenent l'estructura del projecte:

    • manage.py: Una utilitat de línia de comandes que et permet interactuar amb aquest projecte Django.
    • myproject/: El directori intern és el paquet de Python real per al teu projecte.
      • __init__.py: Un fitxer buit que indica a Python que aquest directori ha de ser considerat un paquet.
      • settings.py: Configuració del projecte.
      • urls.py: Declaracions d'URL per al projecte.
      • wsgi.py: Un punt d'entrada per als servidors web compatibles amb WSGI per servir el teu projecte.

Executant el servidor de desenvolupament

  1. Navega al directori del teu projecte: 
    cd myproject
  2. Executa el servidor: 
    python manage.py runserver
    Hauries de veure una sortida similar a: 
    Watching for file changes with StatReloader
Performing system checks...

System check identified no issues (0 silenced).
October 10, 2023 - 14:00:00
Django version 3.2, using settings 'myproject.settings'
Starting development server at http://127.0.0.1:8000/
Quit the server with CONTROL-C.
  3. Accedeix al servidor: Obre el teu navegador web i navega a http://127.0.0.1:8000/. Hauries de veure la pàgina de benvinguda de Django.

Creant una aplicació Django

  1. Crea una aplicació: Dins del directori del teu projecte, crea una nova aplicació.

    python manage.py startapp myapp

    Aquesta comanda crea un nou directori anomenat myapp amb l'estructura següent:

    myapp/
    __init__.py
    admin.py
    apps.py
    models.py
    tests.py
    views.py
    migrations/

  2. Entenent l'estructura de l'aplicació:

    • admin.py: Configuració per a la interfície d'administració de Django.
    • apps.py: Configuració per a l'aplicació mateixa.
    • models.py: Models de base de dades per a l'aplicació.
    • tests.py: Casos de prova per a l'aplicació.
    • views.py: Vistes per a l'aplicació.
    • migrations/: Migracions de base de dades per a l'aplicació.

Configurant l'aplicació

  1. Afegeix l'aplicació a la configuració: Obre myproject/settings.py i afegeix la teva aplicació a la llista INSTALLED_APPS. 
    INSTALLED_APPS = [
    ...
    'myapp',
]

Creant una vista simple

  1. Defineix una vista: Obre myapp/views.py i afegeix una vista simple.

    from django.http import HttpResponse

def home(request):
    return HttpResponse("Hello, world!")

  2. Mapeja la vista a una URL: Obre myapp/urls.py i crea un patró d'URL.

    from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]

  3. Inclou les URLs de l'aplicació en el projecte: Obre myproject/urls.py i inclou les URLs de l'aplicació.

    from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

Conclusió

Felicitats! Has creat amb èxit el teu primer projecte Django i una aplicació simple amb una vista bàsica. Ara tens una comprensió fonamental de com estan estructurats els projectes Django i com crear i configurar aplicacions dins d'un projecte. En les següents seccions, aprofundirem en les característiques de Django, incloent models, plantilles i vistes més avançades.
