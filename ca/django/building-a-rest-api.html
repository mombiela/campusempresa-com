Introducció

En aquest mòdul, aprendrem a construir una API REST utilitzant Django. Cobrirem els conceptes bàsics dels principis RESTful, la configuració de Django i la creació de punts finals. Al final d'aquest mòdul, seràs capaç de crear una API REST totalment funcional.

Prerequisits

Abans de començar, assegura't de tenir:

  • Coneixements bàsics de Python i Django.
  • Python instal·lat a la teva màquina.
  • Django instal·lat (pip install django).

Configuració del Projecte

  1. Crear un Projecte Django:

    django-admin startproject myapi
cd myapi

  2. Crear una App Django:

    python manage.py startapp api

  3. Afegir l'App a INSTALLED_APPS: Edita myapi/settings.py:

    INSTALLED_APPS = [
    ...
    'api',
    'rest_framework',  # Afegeix Django REST framework
]

Creació de Models

Defineix l'estructura de dades per a la teva API utilitzant models de Django.

  1. Crear un Model: Edita api/models.py:

    from django.db import models

class Item(models.Model):
    name = models.CharField(maxlength=100)
    description = models.TextField()
    price = models.DecimalField(max_digits=10, decimal_places=2)
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.name

  2. Aplicar Migracions:

    python manage.py makemigrations
python manage.py migrate

Serialitzadors

Els serialitzadors converteixen tipus de dades complexos, com ara querysets i instàncies de models, en tipus de dades natius de Python que després es poden renderitzar fàcilment en JSON o XML.

  1. Crear un Serialitzador: Edita api/serializers.py: 
    from rest_framework import serializers
from .models import Item

class ItemSerializer(serializers.ModelSerializer):
    class Meta:
        model = Item
        fields = '__all__'

Vistes

Les vistes gestionen la lògica dels punts finals de la teva API.

  1. Crear Vistes: Edita api/views.py: 
    from rest_framework import generics
from .models import Item
from .serializers import ItemSerializer

class ItemListCreate(generics.ListCreateAPIView):
    queryset = Item.objects.all()
    serializer_class = ItemSerializer

class ItemDetail(generics.RetrieveUpdateDestroyAPIView):
    queryset = Item.objects.all()
    serializer_class = ItemSerializer

URLs

Defineix els patrons de URL per als punts finals de la teva API.

  1. Crear Patrons de URL: Edita api/urls.py:

    from django.urls import path
from .views import ItemListCreate, ItemDetail

urlpatterns = [
    path('items/', ItemListCreate.as_view(), name='item-list-create'),
    path('items/<int:pk>/', ItemDetail.as_view(), name='item-detail'),
]

  2. Incloure URLs de l'API en les URLs del Projecte: Edita myapi/urls.py:

    from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include('api.urls')),
]

Provar l'API

Utilitza eines com Postman o curl per provar els punts finals de la teva API.

  1. Llistar Ítems:

    curl -X GET http://127.0.0.1:8000/api/items/

  2. Crear un Ítem:

    curl -X POST http://127.0.0.1:8000/api/items/ -d '{"name": "Item1", "description": "Description1", "price": "10.00"}' -H "Content-Type: application/json"

  3. Recuperar un Ítem:

    curl -X GET http://127.0.0.1:8000/api/items/1/

  4. Actualitzar un Ítem:

    curl -X PUT http://127.0.0.1:8000/api/items/1/ -d '{"name": "Updated Item", "description": "Updated Description", "price": "20.00"}' -H "Content-Type: application/json"

  5. Eliminar un Ítem:

    curl -X DELETE http://127.0.0.1:8000/api/items/1/

Conclusió

En aquest mòdul, hem cobert els conceptes bàsics de la construcció d'una API REST utilitzant Django. Hem passat per la configuració d'un projecte Django, la creació de models, serialitzadors, vistes i la definició de patrons de URL. També hem provat la nostra API utilitzant comandes curl. Amb aquests fonaments, ara pots expandir la teva API per incloure característiques més complexes i gestionar diversos tipus de dades.
