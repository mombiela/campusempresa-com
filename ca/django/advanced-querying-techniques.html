En aquesta secció, aprofundirem en tècniques avançades de consultes a Django. Això t'ajudarà a escriure consultes més eficients i potents, permetent-te recuperar exactament les dades que necessites de la teva base de dades.

Conceptes clau

  • QuerySet API: Comprendre tot el potencial de l'API QuerySet de Django.
  • Agregació: Realitzar càlculs complexos sobre les teves dades.
  • Anotacions: Afegir camps calculats als teus QuerySets.
  • Subconsultes: Utilitzar subconsultes per realitzar cerques més complexes.
  • SQL cru: Quan i com utilitzar SQL cru a Django.

QuerySet API

L'API QuerySet de Django és increïblement potent i et permet filtrar, ordenar i manipular les teves dades de diverses maneres.

Filtrant dades

Pots filtrar dades utilitzant el mètode filter(). Aquest mètode et permet especificar condicions que els objectes retornats han de complir.

# Exemple: Filtrant llibres publicats després del 2020
recent_books = Book.objects.filter(publication_date__year__gt=2020)

Enllaçant filtres

Els filtres es poden enllaçar per refinar encara més els resultats.

# Exemple: Filtrant llibres publicats després del 2020 i escrits per 'John Doe'
recent_books_by_john = Book.objects.filter(publication_date__year__gt=2020).filter(author__name='John Doe')

Excloent dades

El mètode exclude() et permet excloure objectes que compleixin certes condicions.

# Exemple: Excloent llibres publicats abans del 2000
modern_books = Book.objects.exclude(publication_date__year__lt=2000)

Agregació

L'agregació et permet realitzar càlculs sobre un QuerySet, com comptar, sumar o calcular la mitjana de valors.

Utilitzant funcions d'agregació

Django proporciona diverses funcions d'agregació com Count, Sum, Avg, Min i Max.

from django.db.models import Count, Avg

# Exemple: Comptant el nombre de llibres per cada autor
author_book_counts = Author.objects.annotate(num_books=Count('book'))

# Exemple: Calculant la mitjana de valoracions de tots els llibres
average_rating = Book.objects.aggregate(Avg('rating'))

Anotacions

Les anotacions són similars a les agregacions però afegeixen els valors calculats a cada objecte del QuerySet.

Afegint anotacions

Pots afegir anotacions utilitzant el mètode annotate().

from django.db.models import Count

# Exemple: Afegint el nombre de llibres que cada autor ha escrit
authors_with_book_counts = Author.objects.annotate(num_books=Count('book'))

Subconsultes

Les subconsultes et permeten realitzar cerques més complexes incrustant una consulta dins d'una altra.

Utilitzant subconsultes

Les subconsultes es poden utilitzar amb les classes Subquery i OuterRef.

from django.db.models import Subquery, OuterRef

# Exemple: Obtenint l'últim llibre per a cada autor
latest_books = Book.objects.filter(author=OuterRef('pk')).order_by('-publication_date')
authors_with_latest_books = Author.objects.annotate(latest_book=Subquery(latest_books.values('title')[:1]))

SQL cru

Tot i que l'ORM de Django és potent, hi ha moments en què podries necessitar escriure consultes SQL crues.

Executant SQL cru

Pots executar consultes SQL crues utilitzant el mètode raw().

# Exemple: Executant una consulta SQL crua
books = Book.objects.raw('SELECT * FROM myapp_book WHERE publication_date > %s', [2020])

Utilitzant SQL cru per a consultes complexes

L'SQL cru pot ser útil per a consultes complexes que són difícils d'expressar utilitzant l'ORM.

# Exemple: Utilitzant SQL cru per a una consulta complexa
complex_query = '''
SELECT author.name, COUNT(book.id) as book_count
FROM myapp_author as author
JOIN myapp_book as book ON author.id = book.author_id
GROUP BY author.name
HAVING COUNT(book.id) > 5
'''
authors_with_many_books = Author.objects.raw(complex_query)

Conclusió

Les tècniques avançades de consultes a Django et permeten realitzar tasques complexes de recuperació i manipulació de dades de manera eficient. Dominant l'API QuerySet, l'agregació, les anotacions, les subconsultes i l'SQL cru, podràs gestionar una àmplia gamma de desafiaments relacionats amb les dades a les teves aplicacions Django. Practica aquestes tècniques amb exemples del món real per aprofundir en la teva comprensió i millorar les teves habilitats.
