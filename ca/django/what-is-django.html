Introducció

Django és un framework web de Python d'alt nivell que fomenta el desenvolupament ràpid i un disseny net i pragmàtic. Va ser dissenyat per ajudar els desenvolupadors a portar aplicacions des del concepte fins a la finalització tan ràpidament com sigui possible. Django posa èmfasi en la reutilització i "plugabilitat" dels components, menys codi, baixa dependència i el principi de "no et repeteixis" (DRY).

Conceptes Clau

  • Framework Web: Una col·lecció de mòduls i eines que ajuden els desenvolupadors a construir aplicacions web.
  • Framework d'Alt Nivell: Proporciona abstraccions i eines per manejar tasques comunes de desenvolupament web, permetent als desenvolupadors centrar-se en escriure la lògica de la seva aplicació.
  • Basat en Python: Escrit en Python, aprofitant la seva simplicitat i llegibilitat.

Característiques de Django

  • ORM (Object-Relational Mapping): Permet als desenvolupadors interactuar amb bases de dades utilitzant codi Python en lloc de SQL.
  • Interfície d'Administració: Panell d'administració generat automàticament per gestionar les dades de l'aplicació.
  • Enrutament d'URL: Assigna URLs a vistes utilitzant una sintaxi neta i llegible.
  • Motor de Plantilles: Facilita la separació del disseny (HTML) del codi Python.
  • Gestió de Formularis: Simplifica la creació, validació i processament de formularis.
  • Seguretat: Inclou protecció integrada contra atacs web comuns com la injecció SQL, el cross-site scripting (XSS) i el cross-site request forgery (CSRF).
  • Escalabilitat: Dissenyat per manejar llocs web i aplicacions amb alt trànsit.

Exemple de Codi: Una Aplicació Simple de Django

Pas 1: Configurar Django

Primer, necessites instal·lar Django. Pots fer-ho utilitzant pip:

pip install django

Pas 2: Crear un Projecte Django

Crea un nou projecte Django utilitzant el comandament django-admin:

django-admin startproject myproject

Pas 3: Crear una Aplicació Django

Navega dins del directori del teu projecte i crea una nova aplicació:

cd myproject
python manage.py startapp myapp

Pas 4: Definir un Model

A myapp/models.py, defineix un model simple:

from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.title

Pas 5: Registrar el Model a l'Administració

A myapp/admin.py, registra el model per fer-lo disponible a la interfície d'administració:

from django.contrib import admin
from .models import Post

admin.site.register(Post)

Pas 6: Crear una Vista

A myapp/views.py, crea una vista simple per mostrar les publicacions:

from django.shortcuts import render
from .models import Post

def post_list(request):
    posts = Post.objects.all()
    return render(request, 'post_list.html', {'posts': posts})

Pas 7: Assignar l'URL

A myapp/urls.py, assigna l'URL a la vista:

from django.urls import path
from .views import post_list

urlpatterns = [
    path('', post_list, name='post_list'),
]

Pas 8: Crear una Plantilla

A myapp/templates/post_list.html, crea una plantilla simple per mostrar les publicacions:

<!DOCTYPE html>
<html>
<head>
    <title>Llista de Publicacions</title>
</head>
<body>
    <h1>Publicacions</h1>
    <ul>
        {% for post in posts %}
            <li>{{ post.title }} - {{ post.created_at }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Pas 9: Executar el Servidor

Executa el servidor de desenvolupament per veure la teva aplicació en acció:

python manage.py runserver

Visita http://127.0.0.1:8000/ al teu navegador per veure la llista de publicacions.

Conclusió

Django és un framework potent i flexible que simplifica el procés de construir aplicacions web. Les seves característiques integrades com l'ORM, la interfície d'administració i els mecanismes de seguretat el fan una gran elecció tant per a principiants com per a desenvolupadors avançats. Seguint els passos descrits anteriorment, pots configurar ràpidament una aplicació bàsica de Django i començar a explorar les seves capacitats. A mesura que avancis en aquest curs, aprendràs característiques més avançades i millors pràctiques per construir aplicacions web robustes i escalables amb Django.
