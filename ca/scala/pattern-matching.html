El Projecte Sobre nosaltres Contribuir Donacions Llicència

La coincidència de patrons és una característica potent a Scala que et permet fer coincidir valors amb patrons i descompondre estructures de dades. És similar a les instruccions switch-case en altres llenguatges però molt més potent i expressiva.

Introducció a la coincidència de patrons

  • Definició: La coincidència de patrons és un mecanisme per comprovar un valor contra un patró. És una versió més potent de la instrucció switch que es troba en molts altres llenguatges.
  • Usos: S'utilitza comunament per a:
    • Descompondre estructures de dades
    • Gestionar diferents tipus de dades
    • Implementar flux de control basat en la coincidència de patrons

Sintaxi bàsica

La sintaxi bàsica de la coincidència de patrons a Scala utilitza la paraula clau match seguida d'instruccions case.

val number = 3

number match {
  case 1 => println("One")
  case 2 => println("Two")
  case 3 => println("Three")
  case _ => println("Other number")
}
  • Explicació:
    • number match { ... }: La paraula clau match s'utilitza per començar la coincidència de patrons.
    • case 1 => println("One"): Cada paraula clau case introdueix un patró per fer coincidir amb el valor.
    • case _ => println("Other number"): El guió baix _ és un patró comodí que coincideix amb qualsevol valor.

Coincidència de tipus

La coincidència de patrons també es pot utilitzar per fer coincidir tipus, cosa que és útil per gestionar diferents tipus de dades de manera segura.

def process(value: Any): String = value match {
  case i: Int => s"Integer: $i"
  case s: String => s"String: $s"
  case _ => "Unknown type"
}

println(process(42))        // Output: Integer: 42
println(process("Scala"))   // Output: String: Scala
println(process(3.14))      // Output: Unknown type
  • Explicació:
    • case i: Int => ...: Coincideix si value és de tipus Int.
    • case s: String => ...: Coincideix si value és de tipus String.
    • case _ => ...: Coincideix amb qualsevol altre tipus.

Descomposició d'estructures de dades

La coincidència de patrons és particularment potent quan es treballa amb estructures de dades complexes com tuples, llistes i classes de cas.

Tuples

val tuple = (1, "Scala")

tuple match {
  case (1, lang) => println(s"Language: $lang")
  case (2, lang) => println(s"Second language: $lang")
  case _ => println("Other tuple")
}
  • Explicació:
    • case (1, lang) => ...: Coincideix amb una tupla on el primer element és 1 i vincula el segon element a lang.

Llistes

val list = List(1, 2, 3)

list match {
  case List(1, 2, 3) => println("Exact match")
  case List(1, _*) => println("Starts with 1")
  case _ => println("Other list")
}
  • Explicació:
    • case List(1, 2, 3) => ...: Coincideix amb una llista amb exactament tres elements: 1, 2 i 3.
    • case List(1, _*) => ...: Coincideix amb una llista que comença amb 1 seguit de qualsevol nombre d'elements.

Classes de cas

case class Person(name: String, age: Int)

val person = Person("Alice", 25)

person match {
  case Person("Alice", age) => println(s"Alice is $age years old")
  case Person(name, age) => println(s"$name is $age years old")
  case _ => println("Unknown person")
}
  • Explicació:
    • case Person("Alice", age) => ...: Coincideix amb un objecte Person amb el nom "Alice" i vincula l'edat a age.
    • case Person(name, age) => ...: Coincideix amb qualsevol objecte Person i vincula el nom i l'edat a name i age.

Coincidència de patrons avançada

Patrons niats

La coincidència de patrons es pot niar per gestionar estructures de dades més complexes.

val nestedList = List(List(1, 2), List(3, 4))

nestedList match {
  case List(List(1, 2), List(a, b)) => println(s"Nested match: $a, $b")
  case _ => println("Other nested list")
}
  • Explicació:
    • case List(List(1, 2), List(a, b)) => ...: Coincideix amb una llista que conté dues llistes, on la primera llista és exactament List(1, 2) i la segona llista té dos elements que es vinculen a a i b.

Guàrdies

Les guàrdies afegeixen condicions addicionals als patrons.

val number = 5

number match {
  case x if x % 2 == 0 => println("Even number")
  case x if x % 2 != 0 => println("Odd number")
  case _ => println("Unknown number")
}
  • Explicació:
    • case x if x % 2 == 0 => ...: Coincideix si x és un nombre parell.
    • case x if x % 2 != 0 => ...: Coincideix si x és un nombre senar.

Resum

La coincidència de patrons a Scala és una característica versàtil i potent que et permet:

  • Fer coincidir valors amb patrons.
  • Descompondre estructures de dades complexes.
  • Gestionar diferents tipus de dades de manera segura.
  • Implementar flux de control basat en la coincidència de patrons.

En dominar la coincidència de patrons, pots escriure codi Scala més expressiu i concís, facilitant la gestió d'una àmplia gamma d'escenaris de programació.
