Introducció

A Scala, les classes i els objectes són blocs de construcció fonamentals per crear codi estructurat i reutilitzable. Aquesta secció et guiarà a través dels conceptes bàsics de la definició de classes i objectes, i com utilitzar-los de manera efectiva en els teus programes Scala.

Classes a Scala

Una classe a Scala és un plànol per crear objectes. Pot contenir camps (variables) i mètodes (funcions) per definir el comportament i l'estat dels objectes.

Definició d'una classe

Per definir una classe a Scala, utilitzes la paraula clau class seguida del nom de la classe i una llista de paràmetres opcional.

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hola, em dic $name i tinc $age anys."
}
  • class Person(val name: String, val age: Int): Això defineix una classe anomenada Person amb dos paràmetres, name i age.
  • def greet(): String = ...: Això defineix un mètode anomenat greet que retorna una cadena de salutació.

Creació d'objectes a partir d'una classe

Per crear un objecte a partir d'una classe, utilitzes la paraula clau new.

val person = new Person("Alice", 30)
println(person.greet()) // Sortida: Hola, em dic Alice i tinc 30 anys.

Objectes a Scala

A Scala, un objecte és una instància singleton d'una classe. Els objectes es defineixen utilitzant la paraula clau object i poden contenir mètodes i valors.

Definició d'un objecte

Un objecte és similar a una classe però no pot prendre paràmetres.

object MathUtils {
  def add(a: Int, b: Int): Int = a + b
  def subtract(a: Int, b: Int): Int = a - b
}
  • object MathUtils: Això defineix un objecte singleton anomenat MathUtils.
  • def add(a: Int, b: Int): Int = ...: Això defineix un mètode anomenat add que retorna la suma de dos enters.

Ús d'un objecte

Pots cridar mètodes en un objecte directament sense crear una instància.

val sum = MathUtils.add(5, 3)
println(sum) // Sortida: 8

Objectes companys

A Scala, un objecte company és un objecte que comparteix el mateix nom que una classe i es defineix en el mateix fitxer. Els objectes companys s'utilitzen per definir mètodes i valors que estan relacionats amb la classe però no estan lligats a una instància específica.

Exemple d'objecte company

class Circle(val radius: Double) {
  def area: Double = Circle.pi * radius * radius
}

object Circle {
  val pi: Double = 3.14159
}
  • class Circle(val radius: Double): Això defineix una classe anomenada Circle.
  • object Circle: Això defineix un objecte company per a la classe Circle.
  • val pi: Double = 3.14159: Això defineix un valor constant pi en l'objecte company.

Ús d'objectes companys

Pots accedir als membres d'un objecte company utilitzant el nom de la classe.

val circle = new Circle(5.0)
println(circle.area) // Sortida: 78.53975

Classes de cas

Les classes de cas són un tipus especial de classe a Scala que són immutables i vénen amb mètodes integrats per a la comparació i la coincidència de patrons.

Definició d'una classe de cas

case class Point(x: Int, y: Int)
  • case class Point(x: Int, y: Int): Això defineix una classe de cas anomenada Point amb dos paràmetres, x i y.

Creació i ús de classes de cas

Pots crear instàncies de classes de cas sense utilitzar la paraula clau new.

val point1 = Point(1, 2)
val point2 = Point(1, 2)
println(point1 == point2) // Sortida: true

Resum

En aquesta secció, hem cobert els conceptes bàsics de les classes i els objectes a Scala, incloent:

  • Com definir i instanciar classes.
  • Com definir i utilitzar objectes singleton.
  • El concepte d'objectes companys i el seu ús.
  • Les característiques especials de les classes de cas.

Entendre aquests conceptes és crucial per escriure codi organitzat i reutilitzable a Scala. A mesura que progressis, veuràs com aquests blocs de construcció encaixen en aplicacions Scala més complexes.
