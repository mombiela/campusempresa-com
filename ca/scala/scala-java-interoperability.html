Introducció

Scala està dissenyat per ser totalment interoperable amb Java, permetent als desenvolupadors aprofitar les biblioteques i els marcs de treball existents de Java sense problemes. Aquesta secció cobrirà els conceptes bàsics de com Scala i Java poden treballar junts, incloent la crida de codi Java des de Scala i viceversa.

Conceptes Clau

  • Interoperabilitat: La capacitat de Scala i Java per interactuar i utilitzar les biblioteques i els marcs de treball de l'altre.
  • Classes Java en Scala: Com instanciar i utilitzar classes Java dins del codi Scala.
  • Classes Scala en Java: Com instanciar i utilitzar classes Scala dins del codi Java.
  • Conversions de Tipus: Gestionar les diferències en els tipus de dades entre Scala i Java.
  • Col·leccions: Utilitzar col·leccions Java en Scala i viceversa.

Utilitzant Classes Java en Scala

Scala pot utilitzar directament classes Java sense cap configuració especial. Aquí teniu com podeu instanciar i utilitzar una classe Java en Scala.

Exemple: Utilitzant una Classe Java en Scala

Suposem que tenim una classe Java simple Person:

// Person.java
public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }
}

Podeu utilitzar aquesta classe en Scala de la següent manera:

// Main.scala
object Main extends App {
    val person = new Person("John Doe", 30)
    println(s"Name: ${person.getName}, Age: ${person.getAge}")
}

Explicació

  • Instanciació: new Person("John Doe", 30) crea una nova instància de la classe Person.
  • Crides a Mètodes: person.getName i person.getAge criden els respectius mètodes de la classe Java.

Utilitzant Classes Scala en Java

Java també pot utilitzar classes Scala, però hi ha alguns matisos a tenir en compte, especialment pel que fa a les característiques de Scala com els objectes companys i els paràmetres per defecte.

Exemple: Utilitzant una Classe Scala en Java

Suposem que tenim una classe Scala simple Car:

// Car.scala
class Car(val make: String, val model: String) {
    def details(): String = s"Make: $make, Model: $model"
}

Podeu utilitzar aquesta classe en Java de la següent manera:

// Main.java
public class Main {
    public static void main(String[] args) {
        Car car = new Car("Toyota", "Corolla");
        System.out.println(car.details());
    }
}

Explicació

  • Instanciació: new Car("Toyota", "Corolla") crea una nova instància de la classe Car.
  • Crides a Mètodes: car.details() crida el mètode details de la classe Scala.

Conversions de Tipus

Scala i Java tenen sistemes de tipus diferents, però Scala proporciona conversions implícites per gestionar aquestes diferències.

Exemple: Conversió de Tipus

// Codi Scala
val javaList: java.util.List[String] = new java.util.ArrayList[String]()
javaList.add("Scala")
javaList.add("Java")

import scala.jdk.CollectionConverters._
val scalaList: List[String] = javaList.asScala.toList

Explicació

  • Llista Java: java.util.List[String] és una llista Java.
  • Conversió: javaList.asScala.toList converteix la llista Java a una llista Scala utilitzant la importació scala.jdk.CollectionConverters._.

Col·leccions

Utilitzar col·leccions entre Scala i Java pot ser complicat a causa de les diferències en les seves biblioteques de col·leccions.

Exemple: Utilitzant Col·leccions Java en Scala

import java.util.{ArrayList, List}

val javaList: List[String] = new ArrayList[String]()
javaList.add("Scala")
javaList.add("Java")

import scala.jdk.CollectionConverters._
val scalaList: Seq[String] = javaList.asScala.toSeq

Explicació

  • Llista Java: java.util.List[String] és una llista Java.
  • Conversió: javaList.asScala.toSeq converteix la llista Java a una seqüència Scala.

Conclusió

L'interoperabilitat entre Scala i Java permet als desenvolupadors aprofitar els punts forts de tots dos llenguatges. Entenent com utilitzar classes Java en Scala i viceversa, gestionar conversions de tipus i treballar amb col·leccions, podeu crear aplicacions potents que utilitzin les millors característiques de tots dos ecosistemes. Aquesta interoperabilitat assegura que podeu integrar Scala en projectes Java existents sense problemes i aprofitar la vasta gamma de biblioteques Java disponibles.
