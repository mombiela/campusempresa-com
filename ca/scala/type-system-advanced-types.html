Introducció al Sistema de Tipus de Scala

El sistema de tipus de Scala és una de les seves característiques més poderoses, proporcionant tant flexibilitat com seguretat. Entendre el sistema de tipus és crucial per escriure codi robust i mantenible en Scala.

  • Tipatge Estàtic: Scala és tipat estàticament, el que significa que la verificació de tipus es fa en temps de compilació.
  • Inferència de Tipus: Scala pot inferir tipus, reduint la verbositat del codi.
  • Seguretat de Tipus: Assegura que les operacions es realitzin en tipus compatibles, prevenint molts errors en temps d'execució.

Exemple: Inferència de Tipus

val x = 42  // Scala infereix que x és de tipus Int
val y = "Hola, Scala!"  // Scala infereix que y és de tipus String

Tipus Bàsics

Scala té diversos tipus bàsics que s'utilitzen comunament.

  • Tipus Numèrics: Byte, Short, Int, Long, Float, Double
  • Tipus de Caràcter: Char
  • Tipus Booleà: Boolean
  • Tipus de Cadena: String

Exemple: Tipus Bàsics

val a: Int = 10
val b: Double = 20.5
val c: Char = 'A'
val d: Boolean = true
val e: String = "Scala"

Àlies de Tipus

Els àlies de tipus permeten crear un nou nom per a un tipus existent, fent el codi més llegible.

Exemple: Àlies de Tipus

type Edat = Int
val laMevaEdat: Edat = 25

Tipus Parametritzats

Els tipus parametritzats (o genèrics) permeten crear classes, trets i mètodes que operen en tipus especificats pel cridant.

Exemple: Classe Parametritzada

class Caixa[T](val contingut: T)
val caixaInt = new Caixa[Int](42)
val caixaString = new Caixa[String]("Hola")

Variància

La variància defineix com la relació de subtipus entre tipus més complexos es relaciona amb la relació de subtipus entre els seus components.

  • Covariància (+T): Si B és un subtipo de A, llavors Caixa[B] és un subtipo de Caixa[A].
  • Contravariància (-T): Si A és un subtipo de B, llavors Caixa[A] és un subtipo de Caixa[B].
  • Invariància: No hi ha relació de subtipus.

Exemple: Variància

class Covariant[+A]
class Contravariant[-A]
class Invariant[A]

val covariant: Covariant[Any] = new Covariant[String]
val contravariant: Contravariant[String] = new Contravariant[Any]

Límits de Tipus Superiors i Inferiors

Els límits de tipus restringeixen els tipus que es poden utilitzar com a paràmetres de tipus.

  • Límit Superior (<:): Especifica que un tipus ha de ser un subtipo d'un altre tipus.
  • Límit Inferior (>:): Especifica que un tipus ha de ser un supertipo d'un altre tipus.

Exemple: Límits de Tipus

def exempleLímitSuperior[T <: Animal](animal: T): Unit = {
  println(animal.son)
}

def exempleLímitInferior[T >: Gos](animal: T): Unit = {
  println(animal.toString)
}

Tipus Abstractes

Els tipus abstractes són membres de tipus de classes i trets que no es especifiquen fins que la classe o tret s'estén.

Exemple: Tipus Abstractes

trait Animal {
  type Son
  def ferSon: Son
}

class Gos extends Animal {
  type Son = String
  def ferSon: Son = "Bup"
}

Tipus Propis

Els tipus propis especifiquen que un tret ha de ser barrejat en un altre tret o classe que satisfaci un cert tipus.

Exemple: Tipus Propis

trait Logger {
  def log(missatge: String): Unit
}

trait Servei {
  self: Logger =>
  def realitzarAcció(): Unit = {
    log("Acció realitzada")
  }
}

class LoggerConsola extends Logger {
  def log(missatge: String): Unit = println(missatge)
}

class ElMeuServei extends Servei amb LoggerConsola

Classes de Tipus

Les classes de tipus són una manera d'aconseguir polimorfisme ad-hoc. Permeten definir comportament per a tipus fora d'aquests tipus.

Exemple: Classe de Tipus

trait Mostra[T] {
  def mostra(valor: T): String
}

object Mostra {
  implicit val intMostra: Mostra[Int] = new Mostra[Int] {
    def mostra(valor: Int): String = valor.toString
  }

  implicit val stringMostra: Mostra[String] = new Mostra[String] {
    def mostra(valor: String): String = valor
  }
}

def imprimirValor[T](valor: T)(implicit mostra: Mostra[T]): Unit = {
  println(mostra.mostra(valor))
}

imprimirValor(42)
imprimirValor("Hola")

Conclusió

El sistema de tipus de Scala és ric i expressiu, proporcionant moltes eines per escriure codi segur i flexible. Des dels tipus bàsics i la inferència de tipus fins a conceptes avançats com la variància, els límits de tipus i les classes de tipus, dominar aquests conceptes et permetrà aprofitar tot el potencial de Scala. Entendre i utilitzar aquestes característiques t'ajudarà a escriure codi més robust, mantenible i expressiu.
