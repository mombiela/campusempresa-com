Introducció

A Scala, els objectes companys són una característica poderosa que et permet definir membres tant estàtics com a nivell d'instància per a una classe. Aquest tema et guiarà a través del concepte d'objectes companys, el seu ús i les millors pràctiques, des d'un nivell principiant fins a un nivell avançat.

Què és un objecte company?

Un objecte company a Scala és un objecte que:

  • Comparteix el mateix nom que una classe.
  • Està definit en el mateix fitxer que la classe.
  • Pot accedir als membres privats de la classe i viceversa.

Conceptes clau

  • Membres estàtics: A Scala, no hi ha membres estàtics com a Java. En canvi, utilitzes objectes companys per definir membres que es comparteixen entre totes les instàncies d'una classe.
  • Membres d'instància: Aquests són membres que pertanyen a una instància específica d'una classe.

Definint un objecte company

Un objecte company es defineix utilitzant la paraula clau object, i ha d'estar en el mateix fitxer que la classe amb la qual està associat.

Exemple

class MyClass(val name: String) {
  def greet(): Unit = {
    println(s"Hola, $name!")
  }
}

object MyClass {
  def apply(name: String): MyClass = new MyClass(name)
}

En aquest exemple:

  • MyClass és una classe amb un membre d'instància name i un mètode greet.
  • L'objecte company MyClass conté un mètode de fàbrica apply que crea una instància de MyClass.

Utilitzant objectes companys

Els objectes companys sovint s'utilitzen per a:

  • Mètodes de fàbrica.
  • Funcions d'utilitat.
  • Constants.

Exemple de mètode de fàbrica

class Circle(val radius: Double) {
  def area: Double = Math.PI * radius * radius
}

object Circle {
  def apply(radius: Double): Circle = new Circle(radius)
}

Ús:

val circle = Circle(5.0)  // Crida el mètode apply en l'objecte company
println(circle.area)      // Imprimeix l'àrea del cercle

Exemple de funcions d'utilitat

class MathUtils {
  def square(x: Int): Int = x * x
}

object MathUtils {
  def cube(x: Int): Int = x * x * x
}

Ús:

val utils = new MathUtils()
println(utils.square(4))  // Imprimeix 16

println(MathUtils.cube(3))  // Imprimeix 27

Ús avançat

Patró Singleton

Els objectes companys es poden utilitzar per implementar el patró Singleton, assegurant que només existeixi una instància d'una classe.

class Singleton private() {
  def show(): Unit = println("Instància Singleton")
}

object Singleton {
  private val instance = new Singleton()
  
  def getInstance: Singleton = instance
}

Ús:

val singleton = Singleton.getInstance
singleton.show()  // Imprimeix "Instància Singleton"

Accedint a membres privats

Els objectes companys poden accedir als membres privats de la seva classe companya i viceversa.

class Secret(val secretCode: String) {
  private def reveal(): String = secretCode
}

object Secret {
  def revealSecret(secret: Secret): String = secret.reveal()
}

Ús:

val secret = new Secret("1234")
println(Secret.revealSecret(secret))  // Imprimeix "1234"

Conclusió

Els objectes companys a Scala proporcionen una manera flexible de definir membres semblants als estàtics i funcions d'utilitat. Milloren les capacitats orientades a objectes del llenguatge permetent una interacció estreta entre les classes i els seus objectes companys. Entendre i utilitzar els objectes companys de manera efectiva pot conduir a un codi més net i fàcil de mantenir.
