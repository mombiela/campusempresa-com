Introducció

Les declaracions condicionals són fonamentals en la programació, ja que permeten als desenvolupadors executar diferents blocs de codi basats en certes condicions. A Scala, les declaracions condicionals són similars a les d'altres llenguatges de programació, però tenen algunes característiques i sintaxi úniques. Aquesta secció cobrirà els conceptes bàsics de les declaracions condicionals a Scala, progressant cap a conceptes més avançats.

Declaracions condicionals bàsiques

Declaració `if`

La declaració if s'utilitza per executar un bloc de codi només si una condició especificada és certa.

Sintaxi

if (condition) {
  // codi a executar si la condició és certa
}

Exemple

val x = 10
if (x > 5) {
  println("x és més gran que 5")
}

Declaració `if-else`

La declaració if-else et permet executar un bloc de codi si la condició és certa i un altre bloc si és falsa.

Sintaxi

if (condition) {
  // codi a executar si la condició és certa
} else {
  // codi a executar si la condició és falsa
}

Exemple

val x = 3
if (x > 5) {
  println("x és més gran que 5")
} else {
  println("x no és més gran que 5")
}

Escala `if-else if-else`

Quan tens múltiples condicions a comprovar, pots utilitzar una escala if-else if-else.

Sintaxi

if (condition1) {
  // codi a executar si la condició1 és certa
} else if (condition2) {
  // codi a executar si la condició2 és certa
} else {
  // codi a executar si ambdues condicions són falses
}

Exemple

val x = 7
if (x > 10) {
  println("x és més gran que 10")
} else if (x > 5) {
  println("x és més gran que 5 però menor o igual a 10")
} else {
  println("x és 5 o menys")
}

Declaracions condicionals avançades

Expressió `if` semblant a ternària

Scala no té un operador ternari com alguns altres llenguatges (per exemple, condition ? trueExpr : falseExpr), però pots aconseguir una funcionalitat similar utilitzant una expressió if.

Sintaxi

val result = if (condition) trueExpr else falseExpr

Exemple

val x = 8
val result = if (x > 5) "Més gran" else "Menor"
println(result) // Sortida: Més gran

Coincidència de patrons

La coincidència de patrons és una característica poderosa a Scala que es pot utilitzar com una alternativa a les declaracions condicionals complexes.

Sintaxi

value match {
  case pattern1 => // codi a executar si el valor coincideix amb el patró1
  case pattern2 => // codi a executar si el valor coincideix amb el patró2
  case _ => // codi a executar si el valor no coincideix amb cap dels patrons anteriors
}

Exemple

val x = 3
x match {
  case 1 => println("x és 1")
  case 2 => println("x és 2")
  case 3 => println("x és 3")
  case _ => println("x és una altra cosa")
}

Condicions de guarda en la coincidència de patrons

Les condicions de guarda es poden utilitzar en la coincidència de patrons per afegir condicions addicionals a un cas.

Sintaxi

value match {
  case pattern if condition => // codi a executar si el patró coincideix i la condició és certa
  case _ => // codi a executar si cap altre cas coincideix
}

Exemple

val x = 10
x match {
  case y if y > 5 => println("x és més gran que 5")
  case _ => println("x és 5 o menys")
}

Conclusió

Les declaracions condicionals a Scala proporcionen una manera robusta de controlar el flux del teu programa. Des de les declaracions if bàsiques fins a la coincidència de patrons més avançada, Scala ofereix una varietat d'eines per manejar diferents escenaris de manera efectiva. Entendre i dominar aquests conceptes millorarà significativament la teva capacitat per escriure codi Scala eficient i llegible.
