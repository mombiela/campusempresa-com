Introducció

En aquesta secció, cobrirem les operacions i expressions fonamentals en Scala. Entendre aquestes bases és crucial per escriure codi Scala efectiu. Explorarem operacions aritmètiques, operacions relacionals i lògiques, i com utilitzar expressions per crear una lògica més complexa.

Operacions aritmètiques

Les operacions aritmètiques són els blocs bàsics de qualsevol llenguatge de programació. En Scala, pots realitzar operacions aritmètiques estàndard com la suma, la resta, la multiplicació, la divisió i el mòdul.

Conceptes clau

  • Suma (+): Suma dos números.
  • Resta (-): Resta el segon número del primer.
  • Multiplicació (*): Multiplica dos números.
  • Divisió (/): Divideix el primer número pel segon.
  • Mòdul (%): Retorna el residu de la divisió del primer número pel segon.

Exemples de codi

val a = 10
val b = 3

val sum = a + b        // 13
val difference = a - b // 7
val product = a * b    // 30
val quotient = a / b   // 3
val remainder = a % b  // 1

Operacions relacionals

Les operacions relacionals s'utilitzen per comparar dos valors. El resultat d'una operació relacional és un valor booleà (true o false).

Conceptes clau

  • Igual a (==): Comprova si dos valors són iguals.
  • No igual a (!=): Comprova si dos valors no són iguals.
  • Major que (>): Comprova si el primer valor és més gran que el segon.
  • Menor que (<): Comprova si el primer valor és menor que el segon.
  • Major o igual a (>=): Comprova si el primer valor és més gran o igual que el segon.
  • Menor o igual a (<=): Comprova si el primer valor és menor o igual que el segon.

Exemples de codi

val x = 5
val y = 10

val isEqual = x == y       // false
val isNotEqual = x != y    // true
val isGreater = x > y      // false
val isLess = x < y         // true
val isGreaterOrEqual = x >= y // false
val isLessOrEqual = x <= y // true

Operacions lògiques

Les operacions lògiques s'utilitzen per combinar múltiples expressions booleanes. El resultat d'una operació lògica també és un valor booleà.

Conceptes clau

  • AND lògic (&&): Retorna true si ambdues expressions són true.
  • OR lògic (||): Retorna true si almenys una de les expressions és true.
  • NOT lògic (!): Inverteix el valor booleà.

Exemples de codi

val a = true
val b = false

val andResult = a && b // false
val orResult = a || b  // true
val notResult = !a     // false

Expressions

Les expressions són combinacions de variables, operadors i valors que produeixen un resultat. En Scala, gairebé tot és una expressió, incloent-hi les instruccions if i els blocs de codi.

Conceptes clau

  • Expressions simples: Càlculs o avaluacions directes.
  • Expressions complexes: Combinacions de múltiples operacions i funcions.

Exemples de codi

// Expressió simple
val simpleExpression = 5 * 2 + 3 // 13

// Expressió complexa
val complexExpression = {
  val x = 10
  val y = 20
  x + y * 2 // 50
}

// Utilitzant if com una expressió
val condition = true
val result = if (condition) "Sí" else "No" // "Sí"

Resum

En aquesta secció, hem cobert les operacions bàsiques i expressions en Scala. Hem explorat operacions aritmètiques, relacionals i lògiques, i com utilitzar expressions per crear una lògica més complexa. Entendre aquests fonaments és essencial per progressar cap a temes més avançats en la programació amb Scala. Practica aquests conceptes amb diversos exemples per solidificar la teva comprensió.
