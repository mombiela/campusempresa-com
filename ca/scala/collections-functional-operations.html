Introducció

Scala proporciona un conjunt ric de col·leccions i operacions funcionals que permeten als desenvolupadors escriure codi concís i expressiu. Aquesta secció cobrirà els conceptes bàsics de les col·leccions de Scala, incloent llistes, conjunts, mapes i seqüències, així com operacions funcionals com map, filter i reduce.

Col·leccions a Scala

Les col·leccions de Scala es divideixen en tres categories principals:

  • Seqüències: Col·leccions ordenades d'elements.
  • Conjunts: Col·leccions desordenades d'elements únics.
  • Mapes: Col·leccions de parelles clau-valor.

Seqüències

Les seqüències són col·leccions ordenades d'elements. Els tipus de seqüència més comuns a Scala són List, Array i Vector.

List

Una List és una seqüència immutable. Aquí tens com crear i utilitzar una llista:

val fruits = List("apple", "banana", "cherry")
println(fruits.head) // Output: apple
println(fruits.tail) // Output: List(banana, cherry)

Array

Un Array és una seqüència mutable. Aquí tens com crear i utilitzar un array:

val numbers = Array(1, 2, 3, 4, 5)
numbers(0) = 10
println(numbers.mkString(", ")) // Output: 10, 2, 3, 4, 5

Vector

Un Vector és una seqüència immutable que proporciona accés ràpid aleatori. Aquí tens com crear i utilitzar un vector:

val vector = Vector(1, 2, 3, 4, 5)
println(vector(2)) // Output: 3

Conjunts

Els conjunts són col·leccions d'elements únics. Scala proporciona tant conjunts mutables com immutables.

Conjunt Immutable

Aquí tens com crear i utilitzar un conjunt immutable:

val set = Set(1, 2, 3, 4, 5)
println(set.contains(3)) // Output: true
println(set.contains(6)) // Output: false

Conjunt Mutable

Aquí tens com crear i utilitzar un conjunt mutable:

import scala.collection.mutable

val mutableSet = mutable.Set(1, 2, 3)
mutableSet += 4
println(mutableSet) // Output: Set(1, 2, 3, 4)

Mapes

Els mapes són col·leccions de parelles clau-valor. Scala proporciona tant mapes mutables com immutables.

Mapa Immutable

Aquí tens com crear i utilitzar un mapa immutable:

val map = Map("a" -> 1, "b" -> 2, "c" -> 3)
println(map("a")) // Output: 1

Mapa Mutable

Aquí tens com crear i utilitzar un mapa mutable:

import scala.collection.mutable

val mutableMap = mutable.Map("a" -> 1, "b" -> 2)
mutableMap("c") = 3
println(mutableMap) // Output: Map(a -> 1, b -> 2, c -> 3)

Operacions Funcionals

Les operacions funcionals permeten manipular col·leccions en un estil de programació funcional. Les operacions funcionals més comunes són map, filter i reduce.

Map

L'operació map aplica una funció a cada element d'una col·lecció i retorna una nova col·lecció amb els resultats.

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)
println(doubled) // Output: List(2, 4, 6, 8, 10)

Filter

L'operació filter selecciona elements d'una col·lecció que satisfan un predicat.

val numbers = List(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter(_ % 2 == 0)
println(evenNumbers) // Output: List(2, 4)

Reduce

L'operació reduce combina els elements d'una col·lecció utilitzant una funció binària.

val numbers = List(1, 2, 3, 4, 5)
val sum = numbers.reduce(_ + _)
println(sum) // Output: 15

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les col·leccions i operacions funcionals de Scala. Hem explorat diferents tipus de col·leccions com llistes, arrays, vectors, conjunts i mapes, i hem après com realitzar operacions funcionals com map, filter i reduce. Entendre aquests conceptes és crucial per escriure codi eficient i expressiu en Scala. Practica utilitzant aquestes col·leccions i operacions per convertir-te en un programador més proficient en Scala.
