Introducció a Scala

Scala és un llenguatge de programació potent que combina paradigmes de programació orientada a objectes i funcional. Està dissenyat per ser concís, elegant i segur de tipus, fent-lo una excel·lent elecció tant per a principiants com per a desenvolupadors experimentats.

Començant amb Scala

  • Instal·lació: Com instal·lar Scala en diferents sistemes operatius.
  • Scala REPL: Introducció al bucle Read-Eval-Print de Scala (REPL) per a programació interactiva.
  • Hola Món: Escriure i executar el teu primer programa en Scala.
object HelloWorld {
  def main(args: Array[String]): Unit = {
    println("Hola, Món!")
  }
}

Sintaxi Bàsica i Tipus de Dades

  • Variables i Valors: Entendre var i val.
  • Tipus de Dades Bàsics: Int, Double, String, Boolean, etc.
  • Inferència de Tipus: Deixar que Scala infereixi els tipus per tu.
val immutableValue: Int = 10
var mutableVariable: String = "Hola"
mutableVariable = "Món"

Estructures de Control

  • Declaracions Condicionals: if, else if, else.
  • Bucles: for, while, do-while.
  • Coincidència de Patrons: Una característica poderosa per al flux de control.
val number = 10
if (number > 0) {
  println("Positiu")
} else if (number < 0) {
  println("Negatiu")
} else {
  println("Zero")
}

for (i <- 1 to 5) {
  println(i)
}

number match {
  case 1 => println("Un")
  case 2 => println("Dos")
  case _ => println("Altres")
}

Funcions i Mètodes

  • Definició de Funcions: Sintaxi i ús.
  • Funcions d'Ordre Superior: Funcions que prenen altres funcions com a paràmetres.
  • Funcions Anònimes: Expressions lambda.
def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)

val multiply = (x: Int, y: Int) => x * y
val product = multiply(4, 2)

Col·leccions

  • Llistes: Seqüències immutables.
  • Arrays: Seqüències mutables.
  • Mapes: Parells clau-valor.
  • Tuples: Col·leccions de mida fixa d'elements.
val list = List(1, 2, 3, 4, 5)
val array = Array(1, 2, 3, 4, 5)
val map = Map("a" -> 1, "b" -> 2)
val tuple = (1, "Scala", true)

Programació Orientada a Objectes

  • Classes i Objectes: Definició i ús de classes.
  • Herència: Extensió de classes.
  • Traits: Similar als interfícies en altres llenguatges.
class Animal {
  def speak(): Unit = {
    println("L'animal parla")
  }
}

class Dog extends Animal {
  override def speak(): Unit = {
    println("El gos borda")
  }
}

val dog = new Dog()
dog.speak()

trait Friendly {
  def greet(): Unit = {
    println("Hola!")
  }
}

class Person extends Friendly

val person = new Person()
person.greet()

Programació Funcional

  • Immutabilitat: Beneficis i ús.
  • Funcions Pures: Funcions sense efectes secundaris.
  • Monads: Entendre Option, Either, i Future.
val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)

val maybeNumber: Option[Int] = Some(5)
val result = maybeNumber match {
  case Some(value) => value * 2
  case None => 0
}

Temes Avançats

  • Conversions Implícites: Convertir tipus automàticament.
  • Classes de Tipus: Polimorfisme ad-hoc.
  • Concurrència: Utilitzar Futures i Promises per a programació asíncrona.
implicit def intToString(x: Int): String = x.toString

def printString(s: String): Unit = {
  println(s)
}

printString(123) // Converteix implícitament Int a String

import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

val futureResult = Future {
  Thread.sleep(1000)
  42
}

futureResult.map(result => println(s"Resultat: $result"))

Conclusió

Scala és un llenguatge versàtil que ofereix un conjunt ric de característiques tant per a la programació orientada a objectes com per a la funcional. Dominant els conceptes bàsics i explorant temes avançats, pots escriure codi eficient, concís i elegant. Practica regularment i explora les àmplies biblioteques i marcs de treball de Scala per convertir-te en un expert en aquest potent llenguatge.
