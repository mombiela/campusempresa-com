Introducció

Les funcions són blocs fonamentals en Scala, permetent encapsular la lògica i reutilitzar el codi de manera eficient. Aquesta secció et guiarà a través dels conceptes bàsics de la definició i invocació de funcions, progressant des de conceptes simples fins a més avançats.

Definició de Funcions

Definició Bàsica de Funció

En Scala, una funció es defineix utilitzant la paraula clau def seguida del nom de la funció, la llista de paràmetres i el tipus de retorn.

def add(a: Int, b: Int): Int = {
  a + b
}
  • def: Paraula clau per definir una funció.
  • add: Nom de la funció.
  • (a: Int, b: Int): Llista de paràmetres amb tipus.
  • : Int: Tipus de retorn.
  • { a + b }: Cos de la funció.

Funcions d'Una Sola Expressió

Per a funcions amb una sola expressió, pots ometre les claus.

def subtract(a: Int, b: Int): Int = a - b

Funcions Sense Paràmetres

Les funcions també es poden definir sense paràmetres.

def greet(): String = "Hola, Món!"

Funcions Sense Tipus de Retorn

Si una funció no retorna un valor, el seu tipus de retorn és Unit.

def printMessage(message: String): Unit = {
  println(message)
}

Invocació de Funcions

Invocació Bàsica

Per invocar una funció, utilitza el nom de la funció seguit dels arguments entre parèntesis.

val sum = add(5, 3) // sum serà 8

Ometre els Parèntesis

Per a funcions sense paràmetres, pots ometre els parèntesis en invocar-les.

val greeting = greet // greeting serà "Hola, Món!"

Arguments Nomenats

Pots utilitzar arguments nomenats per fer la invocació més clara.

val result = add(a = 10, b = 5) // result serà 15

Paràmetres per Defecte

Les funcions poden tenir valors de paràmetres per defecte.

def multiply(a: Int, b: Int = 2): Int = a * b

val product1 = multiply(4) // product1 serà 8
val product2 = multiply(4, 3) // product2 serà 12

Funcions d'Ordre Superior

Funcions com a Paràmetres

Scala suporta funcions d'ordre superior, que poden prendre altres funcions com a paràmetres.

def applyOperation(a: Int, b: Int, operation: (Int, Int) => Int): Int = {
  operation(a, b)
}

val sum = applyOperation(5, 3, add) // sum serà 8

Funcions Anònimes

Pots passar funcions anònimes (lambdes) com a arguments.

val result = applyOperation(5, 3, (x, y) => x * y) // result serà 15

Recursió

Funcions Recursives

Les funcions poden cridar-se a si mateixes, cosa que es coneix com a recursió.

def factorial(n: Int): Int = {
  if (n == 0) 1
  else n * factorial(n - 1)
}

val fact5 = factorial(5) // fact5 serà 120

Recursió de Cua

Les funcions de recursió de cua són optimitzades pel compilador per prevenir desbordaments de pila.

def factorialTailRec(n: Int, acc: Int = 1): Int = {
  if (n == 0) acc
  else factorialTailRec(n - 1, n * acc)
}

val fact5TailRec = factorialTailRec(5) // fact5TailRec serà 120

Conclusió

En aquesta secció, has après com definir i invocar funcions en Scala, incloent definicions bàsiques, funcions d'ordre superior i recursió. Entendre aquests conceptes és crucial per escriure codi eficient i reutilitzable. Practica definint les teves pròpies funcions i experimentant amb diferents tipus d'invocacions de funcions per consolidar la teva comprensió.
