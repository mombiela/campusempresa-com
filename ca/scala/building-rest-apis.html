Introducció

En aquesta secció, explorarem com construir APIs REST utilitzant Scala. Començarem amb els conceptes bàsics i gradualment avançarem cap a temes més avançats. Al final d'aquest tema, tindràs una comprensió sòlida de com crear, gestionar i desplegar APIs REST en Scala.

Prerequisits

Abans d'endinsar-nos en la construcció d'APIs REST, assegura't de tenir:

  • Comprensió bàsica de la sintaxi de Scala i dels conceptes de programació funcional.
  • Familiaritat amb els conceptes HTTP i els principis REST.
  • Instal·lats Scala i sbt (Scala Build Tool) a la teva màquina.

Configuració del Projecte

Per començar a construir una API REST, necessitem configurar un projecte Scala utilitzant sbt.

  1. Crea un nou directori per al teu projecte:

    mkdir scala-rest-api
cd scala-rest-api

  2. Inicialitza un nou projecte sbt:

    sbt new scala/scala-seed.g8

  3. Actualitza build.sbt per incloure les dependències necessàries:

    name := "scala-rest-api"

version := "0.1"

scalaVersion := "2.13.6"

libraryDependencies ++= Seq(
  "com.typesafe.akka" %% "akka-http" % "10.2.6",
  "com.typesafe.akka" %% "akka-stream" % "2.6.16",
  "de.heikoseeberger" %% "akka-http-circe" % "1.37.0",
  "io.circe" %% "circe-generic" % "0.14.1"
)

Creació d'una API REST Simple

Comencem creant una API REST simple que respongui amb un missatge de "Hello, World!".

  1. Crea un nou objecte Scala per al servidor:

    // src/main/scala/com/example/HelloWorldServer.scala
package com.example

import akka.actor.ActorSystem
import akka.http.scaladsl.Http
import akka.http.scaladsl.server.Directives._
import akka.stream.ActorMaterializer

object HelloWorldServer {
  def main(args: Array[String]): Unit = {
    implicit val system: ActorSystem = ActorSystem("hello-world-system")
    implicit val materializer: ActorMaterializer = ActorMaterializer()
    implicit val executionContext = system.dispatcher

    val route =
      path("hello") {
        get {
          complete("Hello, World!")
        }
      }

    Http().newServerAt("localhost", 8080).bind(route)
    println("Server online at http://localhost:8080/")
  }
}

  2. Executa el servidor:

    sbt run

  3. Prova l'API navegant a http://localhost:8080/hello en el teu navegador o utilitzant curl:

    curl http://localhost:8080/hello

Gestió de Dades JSON

Per gestionar dades JSON, utilitzarem Circe, una biblioteca JSON per a Scala.

  1. Actualitza la ruta per gestionar respostes JSON:

    // src/main/scala/com/example/JsonServer.scala
package com.example

import akka.actor.ActorSystem
import akka.http.scaladsl.Http
import akka.http.scaladsl.server.Directives._
import akka.stream.ActorMaterializer
import de.heikoseeberger.akkahttpcirce.FailFastCirceSupport._
import io.circe.generic.auto._

object JsonServer {
  case class Greeting(message: String)

  def main(args: Array[String]): Unit = {
    implicit val system: ActorSystem = ActorSystem("json-system")
    implicit val materializer: ActorMaterializer = ActorMaterializer()
    implicit val executionContext = system.dispatcher

    val route =
      path("greet") {
        get {
          complete(Greeting("Hello, World!"))
        }
      }

    Http().newServerAt("localhost", 8080).bind(route)
    println("Server online at http://localhost:8080/")
  }
}

  2. Prova l'API JSON:

    curl http://localhost:8080/greet

Operacions CRUD

Implementem una API CRUD (Create, Read, Update, Delete) simple per gestionar una llista d'elements.

  1. Defineix el model de dades i l'emmagatzematge en memòria:

    // src/main/scala/com/example/ItemServer.scala
package com.example

import akka.actor.ActorSystem
import akka.http.scaladsl.Http
import akka.http.scaladsl.model.StatusCodes
import akka.http.scaladsl.server.Directives._
import akka.stream.ActorMaterializer
import de.heikoseeberger.akkahttpcirce.FailFastCirceSupport._
import io.circe.generic.auto._

import scala.collection.mutable
import scala.concurrent.Future

object ItemServer {
  case class Item(id: Int, name: String, description: String)

  var items: mutable.Map[Int, Item] = mutable.Map()

  def main(args: Array[String]): Unit = {
    implicit val system: ActorSystem = ActorSystem("item-system")
    implicit val materializer: ActorMaterializer = ActorMaterializer()
    implicit val executionContext = system.dispatcher

    val route =
      pathPrefix("items") {
        concat(
          pathEnd {
            concat(
              get {
                complete(items.values.toList)
              },
              post {
                entity(as[Item]) { item =>
                  items += (item.id -> item)
                  complete(StatusCodes.Created, item)
                }
              }
            )
          },
          path(IntNumber) { id =>
            concat(
              get {
                items.get(id) match {
                  case Some(item) => complete(item)
                  case None => complete(StatusCodes.NotFound)
                }
              },
              put {
                entity(as[Item]) { item =>
                  items.update(id, item)
                  complete(StatusCodes.OK, item)
                }
              },
              delete {
                items.remove(id) match {
                  case Some(_) => complete(StatusCodes.NoContent)
                  case None => complete(StatusCodes.NotFound)
                }
              }
            )
          }
        )
      }

    Http().newServerAt("localhost", 8080).bind(route)
    println("Server online at http://localhost:8080/")
  }
}

  2. Prova l'API CRUD utilitzant curl:

    # Crea un element
curl -X POST -H "Content-Type: application/json" -d '{"id":1,"name":"Item1","description":"First item"}' http://localhost:8080/items

# Obté tots els elements
curl http://localhost:8080/items

# Obté un element específic
curl http://localhost:8080/items/1

# Actualitza un element
curl -X PUT -H "Content-Type: application/json" -d '{"id":1,"name":"Updated Item1","description":"Updated description"}' http://localhost:8080/items/1

# Elimina un element
curl -X DELETE http://localhost:8080/items/1

Temes Avançats

Autenticació i Autorització

Implementar l'autenticació i l'autorització és crucial per assegurar la teva API. Pots utilitzar biblioteques com akka-http amb OAuth2 o JWT per a aquest propòsit.

Gestió d'Errors

Una gestió adequada dels errors assegura que la teva API proporcioni missatges d'error i codis d'estat significatius. Utilitza ExceptionHandler i RejectionHandler en akka-http per gestionar els errors.

Proves

Provar la teva API és essencial per assegurar la seva fiabilitat. Utilitza biblioteques com akka-http-testkit i ScalaTest per escriure proves unitàries i d'integració per a la teva API.

Desplegament

El desplegament de la teva API REST en Scala es pot fer utilitzant diversos mètodes, com Docker, Kubernetes, o serveis al núvol com AWS i Heroku.

Conclusió

En aquest tema, hem cobert els conceptes bàsics de la construcció d'APIs REST en Scala utilitzant akka-http. Hem començat configurant un projecte, creant una API simple, gestionant dades JSON i implementant operacions CRUD. També hem tocat temes avançats com l'autenticació, la gestió d'errors, les proves i el desplegament. Amb aquest coneixement, ara estàs equipat per construir i desplegar APIs REST robustes en Scala.
