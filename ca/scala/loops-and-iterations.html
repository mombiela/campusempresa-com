Introducció

Els bucles i les iteracions són conceptes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En Scala, hi ha diverses maneres de realitzar bucles i iteracions, cadascuna amb els seus propis casos d'ús i avantatges. Aquesta secció cobrirà els constructes de bucles bàsics i avançats disponibles en Scala.

Constructes de bucles bàsics

Bucle `while`

El bucle while executa repetidament un bloc de codi mentre una condició donada sigui certa.

Sintaxi

while (condition) {
  // bloc de codi a executar
}

Exemple

var i = 0
while (i < 5) {
  println(s"i = $i")
  i += 1
}

En aquest exemple, el bucle imprimirà el valor de i de 0 a 4.

Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executi almenys una vegada.

Sintaxi

do {
  // bloc de codi a executar
} while (condition)

Exemple

var i = 0
do {
  println(s"i = $i")
  i += 1
} while (i < 5)

Aquest bucle també imprimirà el valor de i de 0 a 4, però el bloc de codi s'executa abans que es comprovi la condició.

Bucles `for`

Bucle `for` bàsic

El bucle for en Scala és més potent i flexible en comparació amb els bucles for tradicionals en altres llenguatges.

Sintaxi

for (i <- range) {
  // bloc de codi a executar
}

Exemple

for (i <- 0 until 5) {
  println(s"i = $i")
}

Aquest bucle imprimirà el valor de i de 0 a 4.

Bucle `for` niat

Pots niar bucles for per iterar sobre múltiples rangs.

Exemple

for (i <- 1 to 3; j <- 1 to 3) {
  println(s"i = $i, j = $j")
}

Això imprimirà totes les combinacions de i i j de 1 a 3.

Bucle `for` amb condicions (Guards)

Pots afegir condicions als bucles for per filtrar les iteracions.

Exemple

for (i <- 1 to 10 if i % 2 == 0) {
  println(s"Even number: $i")
}

Aquest bucle imprimirà només els nombres parells entre 1 i 10.

Constructes de bucles avançats

`for` Comprehensions

Les for comprehensions són una característica potent en Scala que et permet generar i transformar col·leccions.

Exemple

val numbers = List(1, 2, 3, 4, 5)
val doubled = for (n <- numbers) yield n * 2
println(doubled)  // Output: List(2, 4, 6, 8, 10)

En aquest exemple, la for comprehension crea una nova llista on cada element es duplica.

Ús de `foreach`

El mètode foreach s'utilitza per aplicar una funció a cada element d'una col·lecció.

Exemple

val numbers = List(1, 2, 3, 4, 5)
numbers.foreach(n => println(n))

Això imprimirà cada número de la llista.

Ús de `map`

El mètode map transforma cada element d'una col·lecció utilitzant una funció donada.

Exemple

val numbers = List(1, 2, 3, 4, 5)
val squared = numbers.map(n => n * n)
println(squared)  // Output: List(1, 4, 9, 16, 25)

Aquest exemple crea una nova llista on cada element es quadrat.

Conclusió

Els bucles i les iteracions són eines essencials en la caixa d'eines de qualsevol programador. Scala ofereix una varietat de constructes de bucles, des dels bucles while i for bàsics fins a les for comprehensions avançades i mètodes de col·lecció com foreach i map. Entendre aquests constructes t'ajudarà a escriure codi més eficient i llegible. Practica utilitzant aquests bucles en diferents escenaris per convertir-te en un programador proficient en Scala.
