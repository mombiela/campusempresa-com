La gestió d'excepcions és un aspecte crucial de la programació que permet als desenvolupadors gestionar i respondre als errors en temps d'execució de manera elegant. A Scala, la gestió d'excepcions és similar a Java però amb algunes millores específiques de Scala. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de la gestió d'excepcions a Scala.

Introducció a les Excepcions

  • Definició: Una excepció és un esdeveniment que interromp el flux normal de l'execució d'un programa.
  • Excepcions Comunes: NullPointerException, ArrayIndexOutOfBoundsException, ArithmeticException, etc.

Gestió Bàsica d'Excepcions

Bloc Try-Catch

Scala utilitza un bloc try-catch per gestionar excepcions.

try {
  // Codi que podria llançar una excepció
  val result = 10 / 0
} catch {
  case e: ArithmeticException => println("S'ha capturat una excepció aritmètica: " + e.getMessage)
}
  • Explicació: El bloc try conté codi que podria llançar una excepció. El bloc catch captura excepcions específiques i les gestiona.

Blocs Catch Múltiples

Pots capturar múltiples excepcions utilitzant múltiples declaracions case.

try {
  val arr = Array(1, 2, 3)
  println(arr(5))
} catch {
  case e: ArrayIndexOutOfBoundsException => println("Índex de l'array fora de límits: " + e.getMessage)
  case e: Exception => println("Excepció general: " + e.getMessage)
}
  • Explicació: El primer bloc case que coincideixi s'executa. Si no es troba cap coincidència, l'excepció es propaga.

Bloc Finally

El bloc finally conté codi que sempre s'executarà, independentment de si s'ha llançat una excepció o no.

try {
  val result = 10 / 0
} catch {
  case e: ArithmeticException => println("S'ha capturat una excepció aritmètica: " + e.getMessage)
} finally {
  println("Això sempre s'executarà.")
}
  • Explicació: El bloc finally és útil per a activitats de neteja com tancar fitxers o alliberar recursos.

Gestió Avançada d'Excepcions

Llançar Excepcions

Pots llançar excepcions utilitzant la paraula clau throw.

def validateAge(age: Int): Unit = {
  if (age < 18) {
    throw new IllegalArgumentException("L'edat ha de ser 18 o més")
  }
}

try {
  validateAge(16)
} catch {
  case e: IllegalArgumentException => println("S'ha capturat una excepció d'argument il·legal: " + e.getMessage)
}
  • Explicació: La paraula clau throw s'utilitza per llançar explícitament una excepció.

Excepcions Personalitzades

Pots definir les teves pròpies excepcions estenent la classe Exception.

class CustomException(message: String) extends Exception(message)

def riskyMethod(): Unit = {
  throw new CustomException("Aquesta és una excepció personalitzada")
}

try {
  riskyMethod()
} catch {
  case e: CustomException => println("S'ha capturat una excepció personalitzada: " + e.getMessage)
}
  • Explicació: Les excepcions personalitzades proporcionen informació d'error més específica.

Try, Success, and Failure

Scala proporciona un enfocament més funcional per a la gestió d'excepcions utilitzant Try, Success, i Failure.

import scala.util.{Try, Success, Failure}

def divide(a: Int, b: Int): Try[Int] = Try(a / b)

val result = divide(10, 0)

result match {
  case Success(value) => println("Resultat: " + value)
  case Failure(exception) => println("S'ha capturat una excepció: " + exception.getMessage)
}
  • Explicació: Try embolcalla un càlcul que podria llançar una excepció. Success i Failure representen el resultat.

Resum

  • Try-Catch-Finally: Estructura bàsica per gestionar excepcions.
  • Blocs Catch Múltiples: Captura diferents tipus d'excepcions.
  • Llançar Excepcions: Llança explícitament excepcions quan sigui necessari.
  • Excepcions Personalitzades: Crea els teus propis tipus d'excepcions per a una gestió d'errors específica.
  • Enfocament Funcional: Utilitza Try, Success, i Failure per a un estil més funcional de gestió d'excepcions.

Entendre i utilitzar efectivament la gestió d'excepcions a Scala farà que el teu codi sigui més robust i fàcil de mantenir. Practica aquests conceptes amb diversos exemples per obtenir una comprensió sòlida de la gestió d'excepcions a Scala.
