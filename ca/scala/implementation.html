Introducció a la Implementació en Scala

En aquesta secció, explorarem com implementar diversos constructes i conceptes de programació en Scala. Començarem amb implementacions bàsiques i gradualment passarem a temes més avançats. Al final d'aquesta secció, hauríeu de ser capaços d'escriure i entendre codi Scala que implementi una varietat de funcionalitats.

Implementacions Bàsiques

Variables i Tipus de Dades

Scala suporta diversos tipus de dades i permet definir variables utilitzant val (immutable) i var (mutable).

val immutableVariable: Int = 10
var mutableVariable: Int = 20

// Intentar reasignar immutableVariable causarà un error de compilació
// immutableVariable = 15 // Això no compilarà

mutableVariable = 25 // Això està permès
  • Conceptes Clau:
    • val per a variables immutables.
    • var per a variables mutables.
    • Tipus de dades comuns: Int, Double, String, Boolean.

Estructures de Control

Scala proporciona estructures de control estàndard com if-else, for i while.

val x = 10
if (x > 5) {
  println("x és més gran que 5")
} else {
  println("x és 5 o menys")
}

for (i <- 1 to 5) {
  println(i)
}

var count = 0
while (count < 5) {
  println(count)
  count += 1
}
  • Conceptes Clau:
    • Declaracions condicionals: if, else.
    • Constructs de bucle: for, while.

Implementacions Intermèdies

Funcions

Les funcions en Scala es poden definir utilitzant la paraula clau def.

def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)
println(sum) // Sortida: 8
  • Conceptes Clau:
    • Definició de funcions utilitzant def.
    • Paràmetres de funció i tipus de retorn.

Col·leccions

Scala proporciona un conjunt ric de tipus de col·lecció com List, Set i Map.

val numbers: List[Int] = List(1, 2, 3, 4, 5)
val uniqueNumbers: Set[Int] = Set(1, 2, 2, 3)
val nameToAge: Map[String, Int] = Map("Alice" -> 25, "Bob" -> 30)

println(numbers) // Sortida: List(1, 2, 3, 4, 5)
println(uniqueNumbers) // Sortida: Set(1, 2, 3)
println(nameToAge) // Sortida: Map(Alice -> 25, Bob -> 30)
  • Conceptes Clau:
    • Col·leccions immutables: List, Set, Map.
    • Operacions bàsiques sobre col·leccions.

Implementacions Avançades

Classes i Objectes

Scala és un llenguatge orientat a objectes i permet definir classes i objectes.

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hola, em dic $name i tinc $age anys."
}

val alice = new Person("Alice", 25)
println(alice.greet()) // Sortida: Hola, em dic Alice i tinc 25 anys.
  • Conceptes Clau:
    • Definició de classes utilitzant class.
    • Paràmetres del constructor.
    • Mètodes dins de les classes.

Traits i Mixins

Els traits s'utilitzen per compartir interfícies i camps entre classes.

trait Greeter {
  def greet(name: String): String = s"Hola, $name!"
}

class FriendlyPerson(val name: String) extends Greeter

val bob = new FriendlyPerson("Bob")
println(bob.greet(bob.name)) // Sortida: Hola, Bob!
  • Conceptes Clau:
    • Definició de traits utilitzant trait.
    • Barrejar traits en classes utilitzant extends i with.

Coincidència de Patrons

La coincidència de patrons és una característica poderosa en Scala per comprovar un valor contra un patró.

val number = 2
val numberType = number match {
  case 1 => "Un"
  case 2 => "Dos"
  case _ => "Altres"
}

println(numberType) // Sortida: Dos
  • Conceptes Clau:
    • Coincidència de patrons utilitzant match.
    • Declaracions de cas.

Resum

En aquesta secció, hem cobert la implementació de diversos constructes en Scala, començant per variables i estructures de control bàsiques fins a temes més avançats com classes, traits i coincidència de patrons. Entendre aquestes implementacions és crucial per escriure codi Scala efectiu. Practiqueu aquests conceptes amb els exemples i exercicis proporcionats per consolidar la vostra comprensió.
