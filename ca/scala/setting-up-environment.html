Configuració de l'Entorn de Desenvolupament

Configurar l'entorn de desenvolupament és el primer pas per començar a programar en Scala. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per escriure, compilar i executar programes Scala de manera eficient.

Prerequisits

Abans de configurar l'entorn de desenvolupament Scala, assegura't de tenir el següent:

  • Un ordinador amb un sistema operatiu (Windows, macOS o Linux).
  • Connexió a Internet per descarregar el programari necessari.

Instal·lació del Java Development Kit (JDK)

Scala s'executa a la Java Virtual Machine (JVM), així que necessites instal·lar el JDK.

  1. Descarregar JDK:

    • Visita la pàgina de descàrrega de l'Oracle JDK o utilitza una alternativa de codi obert com OpenJDK.
    • Tria la versió compatible amb el teu sistema operatiu.

  2. Instal·lar JDK:

    • Segueix les instruccions d'instal·lació proporcionades a la pàgina de descàrrega.
    • Configura la variable d'entorn JAVA_HOME per apuntar al directori d'instal·lació del JDK.

Verificació de la Instal·lació del JDK

Obre un terminal o símbol del sistema i escriu:

java -version

Hauries de veure una sortida similar a:

java version "1.8.0_231"
Java(TM) SE Runtime Environment (build 1.8.0_231-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.231-b11, mixed mode)

Instal·lació de Scala

Scala es pot instal·lar utilitzant diversos mètodes. Aquí, cobrirem els més comuns: utilitzant l'instal·lador oficial de Scala i utilitzant SDKMAN.

Mètode 1: Utilitzant l'Instal·lador Oficial de Scala

  1. Descarregar Scala:

    • Visita la pàgina de descàrrega de Scala.
    • Tria l'instal·lador adequat per al teu sistema operatiu.

  2. Instal·lar Scala:

    • Executa l'instal·lador i segueix les instruccions en pantalla.

Mètode 2: Utilitzant SDKMAN

SDKMAN és una eina per gestionar versions paral·leles de múltiples kits de desenvolupament de programari en la majoria de sistemes basats en Unix.

  1. Instal·lar SDKMAN:

    • Obre un terminal i executa la següent comanda: 
      curl -s "https://get.sdkman.io" | bash
    • Segueix les instruccions per completar la instal·lació.

  2. Instal·lar Scala utilitzant SDKMAN:

    • Obre un nou terminal i executa: 
      sdk install scala

Verificació de la Instal·lació de Scala

Obre un terminal o símbol del sistema i escriu:

scala -version

Hauries de veure una sortida similar a:

Scala code runner version 2.13.3 -- Copyright 2002-2020, LAMP/EPFL

Instal·lació d'un Entorn de Desenvolupament Integrat (IDE)

Un IDE pot augmentar molt la teva productivitat proporcionant funcions com l'autocompletat de codi, depuració i gestió de projectes. Aquí tens alguns IDEs populars per Scala:

  • IntelliJ IDEA:

    • Descarrega des del lloc web de JetBrains.
    • Instal·la el plugin de Scala mitjançant el gestor de plugins.

  • Visual Studio Code:

    • Descarrega des del lloc web de Visual Studio Code.
    • Instal·la l'extensió Metals per Scala.

Configuració d'IntelliJ IDEA per Scala

  1. Instal·lar IntelliJ IDEA:

    • Segueix les instruccions d'instal·lació al lloc web de JetBrains.

  2. Instal·lar el Plugin de Scala:

    • Obre IntelliJ IDEA.
    • Ves a File > Settings > Plugins.
    • Cerca "Scala" i instal·la el plugin.

  3. Crear un Projecte Scala:

    • Ves a File > New > Project.
    • Selecciona Scala de la llista i segueix les indicacions per configurar el teu projecte.

Configuració de Visual Studio Code per Scala

  1. Instal·lar Visual Studio Code:

    • Segueix les instruccions d'instal·lació al lloc web de Visual Studio Code.

  2. Instal·lar l'Extensió Metals:

    • Obre Visual Studio Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca "Metals" i instal·la l'extensió.

  3. Crear un Projecte Scala:

    • Obre el terminal a Visual Studio Code.
    • Utilitza sbt (Scala Build Tool) per crear un nou projecte: 
      sbt new scala/hello-world.g8

Conclusió

Configurar l'entorn de desenvolupament és un pas crucial en el teu viatge de programació en Scala. Seguint els passos descrits en aquesta secció, hauràs de tenir un entorn de desenvolupament Scala completament funcional. Tant si tries IntelliJ IDEA com Visual Studio Code, assegura't que les teves eines estiguin correctament configurades per maximitzar la teva productivitat. Ara que el teu entorn està llest, pots començar a escriure i executar programes Scala.
