Introducció

L'herència i el polimorfisme són conceptes fonamentals en la programació orientada a objectes (OOP). Permeten la creació de codi flexible i reutilitzable. En Scala, aquests conceptes s'implementen d'una manera que aprofita tant els paradigmes funcionals com els orientats a objectes.

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe. La classe que hereta s'anomena subclasse, i la classe de la qual s'hereta s'anomena superclasse.

Conceptes Clau

  • Superclasse: La classe les propietats i mètodes de la qual s'hereten.
  • Subclasse: La classe que hereta propietats i mètodes de la superclasse.
  • Paraula clau extends: Utilitzada per denotar l'herència.

Exemple

// Superclasse
class Animal {
  def eat(): Unit = {
    println("Aquest animal està menjant")
  }
}

// Subclasse
class Dog extends Animal {
  def bark(): Unit = {
    println("El gos està bordant")
  }
}

// Ús
val dog = new Dog
dog.eat()  // Mètode heretat
dog.bark() // Mètode específic de la subclasse

En aquest exemple:

  • Animal és la superclasse.
  • Dog és la subclasse que hereta de Animal.
  • La classe Dog pot utilitzar el mètode eat de la classe Animal.

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen, encara que comparteixin el mateix nom. En Scala, el polimorfisme s'aconsegueix a través de la sobrescriptura de mètodes i l'ús de traits.

Conceptes Clau

  • Sobrescriptura de Mètodes: Una subclasse proporciona una implementació específica d'un mètode que ja està definit en la seva superclasse.
  • Traits: Els traits són similars a les interfícies en Java i es poden utilitzar per aconseguir polimorfisme.

Exemple de Sobrescriptura de Mètodes

class Animal {
  def sound(): Unit = {
    println("Alguns sons genèrics d'animals")
  }
}

class Dog extends Animal {
  override def sound(): Unit = {
    println("Bordar")
  }
}

class Cat extends Animal {
  override def sound(): Unit = {
    println("Miolar")
  }
}

// Ús
val animals: List[Animal] = List(new Dog, new Cat)
animals.foreach(_.sound())

En aquest exemple:

  • El mètode sound es sobrescriu tant a les classes Dog com Cat.
  • Quan es crida sound a cada animal, s'executa la implementació específica per a aquell animal.

Exemple de Traits

trait Speaker {
  def speak(): Unit
}

class Human extends Speaker {
  def speak(): Unit = {
    println("Hola!")
  }
}

class Robot extends Speaker {
  def speak(): Unit = {
    println("Beep boop")
  }
}

// Ús
val speakers: List[Speaker] = List(new Human, new Robot)
speakers.foreach(_.speak())

En aquest exemple:

  • Speaker és un trait amb un mètode abstracte speak.
  • Tant les classes Human com Robot implementen el mètode speak.
  • El mètode speak es crida polimòrficament a cada instància de la llista.

Conclusió

L'herència i el polimorfisme són característiques poderoses en Scala que permeten la reutilització i flexibilitat del codi. En entendre com utilitzar aquests conceptes, pots crear aplicacions més mantenibles i escalables. Recorda:

  • Utilitza l'herència per compartir codi comú entre classes.
  • Utilitza el polimorfisme per permetre que els mètodes operin de manera diferent segons l'objecte sobre el qual actuen.
  • Aprofita els traits per aconseguir polimorfisme i herència múltiple en Scala.
