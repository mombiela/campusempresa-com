Introducció

A Scala, les variables i els tipus de dades són conceptes fonamentals que formen la base de qualsevol programa. Entendre com declarar variables i utilitzar diferents tipus de dades és essencial per escriure codi Scala efectiu i eficient. Aquesta secció cobrirà els conceptes bàsics de les variables, els diferents tipus de dades disponibles a Scala i com utilitzar-los.

Variables

Les variables a Scala es poden definir utilitzant dues paraules clau: var i val.

  • var: Defineix una variable mutable, el que significa que el seu valor pot canviar després de ser inicialment establert.
  • val: Defineix una variable immutable, el que significa que el seu valor no pot canviar un cop establert.

Exemples

// Variable mutable
var mutableVar: Int = 10
mutableVar = 20 // Això està permès

// Variable immutable
val immutableVal: Int = 10
// immutableVal = 20 // Això causarà un error de compilació

Conceptes Clau

  • Mutable vs Immutable: Utilitza var per a variables que necessiten canviar i val per a constants.
  • Inferència de Tipus: Scala sovint pot inferir el tipus d'una variable, així que les declaracions de tipus explícites són opcionals.
var inferredVar = 10 // Scala infereix això com un Int
val inferredVal = "Hola, Scala!" // Scala infereix això com un String

Tipus de Dades

Scala té un conjunt ric de tipus de dades, que es poden categoritzar àmpliament en:

  • Tipus de Dades Primitives
  • Tipus de Dades Compostes

Tipus de Dades Primitives

Els tipus de dades primitives són els blocs bàsics per a la manipulació de dades. Aquí hi ha els tipus de dades primitives comuns a Scala:

| Tipus de Dada | Descripció | Exemple | |---------------|-------------|---------| | Int | Enter signat de 32 bits | val age: Int = 25 | | Long | Enter signat de 64 bits | val distance: Long = 123456789L | | Float | Punt flotant de 32 bits | val pi: Float = 3.14f | | Double | Punt flotant de 64 bits | val e: Double = 2.71828 | | Char | Caràcter Unicode de 16 bits | val initial: Char = 'A' | | String | Seqüència de caràcters | val name: String = "Scala" | | Boolean | Valor vertader o fals | val isScalaFun: Boolean = true | | Unit | Representa cap valor | val result: Unit = () |

Exemples

val age: Int = 25
val distance: Long = 123456789L
val pi: Float = 3.14f
val e: Double = 2.71828
val initial: Char = 'A'
val name: String = "Scala"
val isScalaFun: Boolean = true
val result: Unit = ()

Tipus de Dades Compostes

Els tipus de dades compostes es construeixen a partir de tipus de dades primitives. Els tipus de dades compostes més comuns a Scala són:

  • Tuples: Agrupen múltiples valors de diferents tipus.
  • Arrays: Seqüències de mida fixa d'elements del mateix tipus.
  • Lists: Seqüències immutables d'elements del mateix tipus.
  • Maps: Col·leccions de parells clau-valor.

Exemples

// Tuple
val person: (String, Int) = ("Alice", 30)

// Array
val numbers: Array[Int] = Array(1, 2, 3, 4, 5)

// List
val fruits: List[String] = List("Poma", "Plàtan", "Cirera")

// Map
val ages: Map[String, Int] = Map("Alice" -> 30, "Bob" -> 25)

Conclusió

Entendre les variables i els tipus de dades és crucial per a qualsevol programador de Scala. Les variables poden ser mutables o immutables, i Scala proporciona una varietat de tipus de dades primitives i compostes per manejar diferents tipus de dades. En dominar aquests conceptes, estaràs ben preparat per escriure programes Scala robustos i eficients.
