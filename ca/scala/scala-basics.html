Introducció

Scala és un llenguatge de programació potent que combina paradigmes de programació orientada a objectes i funcional. Està dissenyat per ser concís, elegant i segur en termes de tipus. Aquesta secció t'introduirà a la sintaxi i estructura bàsiques de Scala, proporcionant una base per a temes més avançats.

Conceptes Clau

  • Variables Immutables: Variables que no es poden canviar un cop assignades.
  • Variables Mutables: Variables que es poden canviar després de l'assignació.
  • Expressions i Declaracions: Blocs bàsics de codi Scala.
  • Funcions: Blocs de codi reutilitzables que realitzen tasques específiques.
  • Classes i Objectes: Components principals de la programació orientada a objectes en Scala.

Variables Immutables i Mutables

A Scala, les variables es poden definir utilitzant val i var.

  • val: Defineix una variable immutable.
  • var: Defineix una variable mutable.

Exemple

val immutableVariable = 10  // Aquest valor no es pot canviar
var mutableVariable = 20    // Aquest valor es pot canviar

mutableVariable = 30        // Operació vàlida
// immutableVariable = 15   // Això causarà un error de compilació

Expressions i Declaracions

Scala és un llenguatge orientat a expressions, el que significa que la majoria de construccions són expressions que retornen un valor.

Exemple

val result = if (5 > 3) "Greater" else "Lesser"  // "Greater" s'assigna a result

Funcions

Les funcions a Scala són ciutadans de primera classe, el que significa que es poden assignar a variables, passar com a arguments i retornar d'altres funcions.

Definició de Funcions

Les funcions es poden definir utilitzant la paraula clau def.

Exemple

def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)  // sum és 8

Classes i Objectes

Scala suporta la programació orientada a objectes amb classes i objectes.

Definició d'una Classe

Una classe es defineix utilitzant la paraula clau class.

Exemple

class Person(val name: String, val age: Int) {
  def greet(): String = s"Hello, my name is $name and I am $age years old."
}

val person = new Person("Alice", 30)
println(person.greet())  // Sortida: Hello, my name is Alice and I am 30 years old.

Objectes Singleton

Scala té un concepte d'objectes singleton, que es defineixen utilitzant la paraula clau object.

Exemple

object Singleton {
  def greet(): String = "Hello from the singleton object!"
}

println(Singleton.greet())  // Sortida: Hello from the singleton object!

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la sintaxi i estructura de Scala, incloent variables immutables i mutables, expressions i declaracions, funcions, i classes i objectes. Entendre aquests fonaments és crucial per progressar cap a temes més avançats de Scala. Practica escrivint programes simples en Scala per reforçar aquests conceptes i construir una base sòlida.
