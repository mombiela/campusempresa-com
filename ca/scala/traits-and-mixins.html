Introducció als Traits

Els traits a Scala són un concepte fonamental que et permet definir mètodes i camps que poden ser reutilitzats per diferents classes. Són similars a les interfícies a Java, però són més potents perquè poden contenir mètodes concrets i camps.

Conceptes Clau

  • Definició de Trait: Els traits es defineixen utilitzant la paraula clau trait.
  • Barreja de Traits: Els traits es poden barrejar en classes utilitzant les paraules clau extends i with.
  • Múltiples Traits: Una classe pot barrejar múltiples traits.
  • Mètodes Concrets: Els traits poden tenir mètodes concrets amb implementacions.
  • Mètodes Abstractes: Els traits també poden tenir mètodes abstractes que han de ser implementats per la classe que barreja el trait.

Exemple: Definició i Ús de Traits

trait Greeter {
  def greet(name: String): Unit = {
    println(s"Hello, $name!")
  }
}

class Person(name: String) extends Greeter {
  def sayHello(): Unit = {
    greet(name)
  }
}

val john = new Person("John")
john.sayHello()  // Output: Hello, John!

En aquest exemple:

  • El trait Greeter defineix un mètode concret greet.
  • La classe Person extén el trait Greeter i utilitza el mètode greet.

Barreja de Múltiples Traits

Scala et permet barrejar múltiples traits en una sola classe, proporcionant una manera potent de compondre comportaments.

Exemple: Barreja de Múltiples Traits

trait A {
  def methodA(): Unit = println("Method A")
}

trait B {
  def methodB(): Unit = println("Method B")
}

class C extends A with B

val c = new C
c.methodA()  // Output: Method A
c.methodB()  // Output: Method B

En aquest exemple:

  • La classe C barreja els traits A i B.
  • C pot utilitzar mètodes de tots dos traits.

Traits amb Mètodes Abstractes

Els traits també poden definir mètodes abstractes que han de ser implementats per la classe que barreja el trait.

Exemple: Mètodes Abstractes en Traits

trait Animal {
  def sound(): String
}

class Dog extends Animal {
  def sound(): String = "Woof"
}

class Cat extends Animal {
  def sound(): String = "Meow"
}

val dog = new Dog
val cat = new Cat
println(dog.sound())  // Output: Woof
println(cat.sound())  // Output: Meow

En aquest exemple:

  • El trait Animal defineix un mètode abstracte sound.
  • Les classes Dog i Cat implementen el mètode sound.

Ús Avançat de Traits

Els traits es poden utilitzar de maneres més avançades, com ara apilant comportaments i utilitzant self-types.

Apilament de Traits

Els traits es poden apilar per crear una cadena de comportaments.

Exemple: Apilament de Traits

trait Logger {
  def log(message: String): Unit = println(message)
}

trait TimestampLogger extends Logger {
  override def log(message: String): Unit = {
    super.log(s"${java.time.Instant.now}: $message")
  }
}

trait ShortLogger extends Logger {
  val maxLength = 15
  override def log(message: String): Unit = {
    super.log(
      if (message.length <= maxLength) message
      else s"${message.substring(0, maxLength - 3)}..."
    )
  }
}

class ConsoleLogger extends Logger

val logger = new ConsoleLogger with TimestampLogger with ShortLogger
logger.log("This is a very long message that will be truncated.") 
// Output: 2023-10-01T12:34:56.789Z: This is a ver...

En aquest exemple:

  • Els traits TimestampLogger i ShortLogger extenen Logger i sobreescriuen el mètode log.
  • La classe ConsoleLogger barreja TimestampLogger i ShortLogger, apilant els seus comportaments.

Conclusió

Els traits a Scala proporcionen una manera flexible i potent de compondre i reutilitzar codi. Et permeten definir tant mètodes concrets com abstractes, barrejar múltiples traits en una sola classe i apilar comportaments. Entendre els traits i mixins és crucial per escriure codi modular i mantenible a Scala. En dominar aquests conceptes, pots aprofitar tot el potencial de Scala per construir aplicacions robustes.
