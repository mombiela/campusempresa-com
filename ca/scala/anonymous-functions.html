Introducció

Les funcions anònimes, també conegudes com a lambdas, són una característica poderosa a Scala que permet definir funcions sense donar-los un nom. Són particularment útils per a funcions curtes i d'un sol ús que s'utilitzen només una vegada o unes poques vegades. En aquesta secció, explorarem com crear i utilitzar funcions anònimes a Scala, des de l'ús bàsic fins a l'avançat.

Sintaxi Bàsica

  • Les funcions anònimes a Scala es defineixen utilitzant el símbol =>.
  • La sintaxi general és (paràmetres) => expressió.

Exemple

val add = (x: Int, y: Int) => x + y
println(add(3, 4))  // Output: 7

En aquest exemple, add és una funció anònima que pren dos enters i retorna la seva suma.

Ús de Funcions Anònimes amb Col·leccions

Les funcions anònimes sovint s'utilitzen amb mètodes de col·leccions com map, filter i reduce.

Exemple amb `map`

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(x => x * 2)
println(doubled)  // Output: List(2, 4, 6, 8, 10)

Aquí, la funció anònima x => x * 2 s'aplica a cada element de la llista numbers.

Exemple amb `filter`

val numbers = List(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter(x => x % 2 == 0)
println(evenNumbers)  // Output: List(2, 4)

En aquest exemple, la funció anònima x => x % 2 == 0 filtra els nombres parells de la llista.

Simplificant Funcions Anònimes

Scala proporciona diverses maneres de simplificar les funcions anònimes, fent-les més concises.

Sintaxi de Marcador de Posició

  • Es poden utilitzar guions baixos (_) com a marcadors de posició per als paràmetres.

Exemple

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)
println(doubled)  // Output: List(2, 4, 6, 8, 10)

Aquí, _ * 2 és una forma abreujada de x => x * 2.

Inferència de Tipus

  • Scala sovint pot inferir els tipus dels paràmetres, permetent ometre'ls.

Exemple

val add = (_: Int) + (_: Int)
println(add(3, 4))  // Output: 7

En aquest cas, els tipus dels paràmetres s'infereixen del context.

Ús Avançat

Les funcions anònimes es poden utilitzar en escenaris més complexos, com ara funcions d'ordre superior i currificació.

Funcions d'Ordre Superior

  • Funcions que prenen altres funcions com a paràmetres o retornen funcions com a resultats.

Exemple

def applyOperation(x: Int, y: Int, operation: (Int, Int) => Int): Int = {
  operation(x, y)
}

val sum = applyOperation(3, 4, (a, b) => a + b)
println(sum)  // Output: 7

Aquí, applyOperation és una funció d'ordre superior que pren una funció anònima com a paràmetre.

Currificació

  • Transformar una funció que pren múltiples arguments en una sèrie de funcions que cadascuna pren un sol argument.

Exemple

def curriedAdd(x: Int)(y: Int): Int = x + y
val addThree = curriedAdd(3) _
println(addThree(4))  // Output: 7

En aquest exemple, curriedAdd és una funció currificada, i addThree és una funció parcialment aplicada.

Conclusió

Les funcions anònimes (lambdas) a Scala són una eina versàtil que pot simplificar el teu codi i fer-lo més expressiu. Són particularment útils per a funcions curtes i d'un sol ús i s'utilitzen comunament amb mètodes de col·leccions. Entenent la sintaxi bàsica, les tècniques de simplificació i l'ús avançat, pots aprofitar tot el potencial de les funcions anònimes en els teus programes Scala.
