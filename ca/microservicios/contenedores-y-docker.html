Introducció als Contenidors

Els contenidors són una tecnologia que permet empaquetar i executar aplicacions de manera aïllada del sistema operatiu subjacent. A diferència de les màquines virtuals, els contenidors comparteixen el mateix nucli del sistema operatiu, cosa que els fa més lleugers i eficients.

Conceptes Clau

  • Aïllament: Els contenidors proporcionen un entorn aïllat per executar aplicacions, cosa que millora la seguretat i l'estabilitat.
  • Portabilitat: Els contenidors poden executar-se en qualsevol sistema que tingui un motor de contenidors compatible, com Docker.
  • Eficiència: Els contenidors són més lleugers que les màquines virtuals perquè comparteixen el nucli del sistema operatiu.

Introducció a Docker

Docker és una plataforma de contenidorització que permet als desenvolupadors empaquetar aplicacions i les seves dependències en contenidors. Docker facilita la creació, desplegament i execució d'aplicacions en contenidors.

Conceptes Clau

  • Docker Engine: El motor de Docker és el component principal que permet la creació i execució de contenidors.
  • Docker Images: Les imatges de Docker són plantilles de només lectura que contenen el sistema operatiu i les aplicacions necessàries per executar un contenidor.
  • Docker Containers: Els contenidors de Docker són instàncies en execució d'imatges de Docker.
  • Docker Hub: Un registre públic on els usuaris poden emmagatzemar i compartir imatges de Docker.

Instal·lació de Docker

Per instal·lar Docker en diferents sistemes operatius, segueixi els passos a continuació:

Instal·lació en Linux

sudo apt-get update
sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

Instal·lació en Windows

  1. Descarregueu Docker Desktop des de Docker Hub.
  2. Executeu l'instal·lador i seguiu les instruccions en pantalla.
  3. Reinicieu el vostre ordinador si és necessari.

Instal·lació en macOS

  1. Descarregueu Docker Desktop des de Docker Hub.
  2. Obriu el fitxer .dmg i arrossegueu Docker a la carpeta d'Aplicacions.
  3. Executeu Docker des de la carpeta d'Aplicacions.

Ús Bàsic de Docker

Comandes Essencials

  • docker run: Executa un contenidor a partir d'una imatge.
docker run hello-world
  • docker ps: Llista els contenidors en execució.
docker ps
  • docker images: Llista les imatges disponibles en el sistema.
docker images
  • docker stop: Atura un contenidor en execució.
docker stop <container_id>

Crear una Imatge de Docker

Per crear una imatge de Docker, s'utilitza un fitxer Dockerfile. A continuació es mostra un exemple d'un Dockerfile simple:

# Useu una imatge base de Python
FROM python:3.8-slim

# Establiu el directori de treball
WORKDIR /app

# Copieu els fitxers necessaris
COPY . /app

# Instaleu les dependències
RUN pip install --no-cache-dir -r requirements.txt

# Comanda per executar l'aplicació
CMD ["python", "app.py"]

Per construir la imatge, useu la següent comanda:

docker build -t my-python-app .

Exemple Pràctic: Desplegar una Aplicació Web

Pas 1: Crear el `Dockerfile`

FROM node:14

WORKDIR /usr/src/app

COPY package*.json ./

RUN npm install

COPY . .

EXPOSE 8080

CMD ["node", "server.js"]

Pas 2: Construir la Imatge

docker build -t my-node-app .

Pas 3: Executar el Contenidor

docker run -p 8080:8080 -d my-node-app

Conclusió

Docker és una eina poderosa per a la contenidorització d'aplicacions, proporcionant portabilitat, eficiència i aïllament. Amb Docker, els desenvolupadors poden empaquetar aplicacions i les seves dependències en contenidors que es poden executar en qualsevol entorn compatible. En aquest tema, hem cobert els conceptes bàsics dels contenidors i Docker, incloent la instal·lació, comandes essencials i un exemple pràctic de desplegament d'una aplicació web.
