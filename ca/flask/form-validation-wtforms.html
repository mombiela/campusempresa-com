Introducció a WTForms

WTForms és una biblioteca flexible de validació i renderització de formularis per al desenvolupament web en Python. És particularment útil en aplicacions Flask per gestionar dades de formularis i assegurar-se que les entrades dels usuaris compleixin criteris específics.

  • Conceptes Clau:
    • WTForms proporciona una manera de definir formularis com a classes de Python.
    • Inclou validadors integrats per assegurar la integritat de les dades.
    • WTForms es pot integrar fàcilment amb Flask.

Configuració de WTForms a Flask

Per utilitzar WTForms amb Flask, cal instal·lar l'extensió Flask-WTF, que fa de pont entre Flask i WTForms.

Instal·lació

pip install Flask-WTF

Configuració Bàsica

En la teva aplicació Flask, has de configurar la clau secreta i inicialitzar l'extensió Flask-WTF.

from flask import Flask
from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField
from wtforms.validators import DataRequired

app = Flask(__name__)
app.config['SECRET_KEY'] = 'your_secret_key'

class MyForm(FlaskForm):
    name = StringField('Nom', validators=[DataRequired()])
    submit = SubmitField('Enviar')

Creació de Formularis amb WTForms

Els formularis en WTForms es creen com a classes de Python, on cada camp és una instància d'una classe de camp de WTForms.

Exemple de Formulari

from wtforms import StringField, PasswordField, BooleanField, SubmitField
from wtforms.validators import DataRequired, Email, Length

class LoginForm(FlaskForm):
    email = StringField('Correu Electrònic', validators=[DataRequired(), Email()])
    password = PasswordField('Contrasenya', validators=[DataRequired(), Length(min=6)])
    remember_me = BooleanField('Recorda\'m')
    submit = SubmitField('Iniciar Sessió')
  • Explicació:
    • StringField, PasswordField i BooleanField són diferents tipus de camps.
    • Els validators asseguren que l'entrada compleixi certs criteris, com ser obligatòria o tenir una longitud mínima.

Gestió de l'Enviament de Formularis

Per gestionar l'enviament de formularis, cal renderitzar el formulari en la teva plantilla i processar les dades del formulari en la teva funció de vista.

Renderització del Formulari

En la teva plantilla HTML, utilitza la sintaxi de Jinja2 per renderitzar el formulari.

<form method="POST" action="">
    {{ form.hidden_tag() }}
    <p>
        {{ form.email.label }}<br>
        {{ form.email(size=32) }}<br>
        {{ form.email.errors }}
    </p>
    <p>
        {{ form.password.label }}<br>
        {{ form.password(size=32) }}<br>
        {{ form.password.errors }}
    </p>
    <p>
        {{ form.remember_me() }} {{ form.remember_me.label }}
    </p>
    <p>{{ form.submit() }}</p>
</form>

Processament de les Dades del Formulari

En la teva funció de vista Flask, comprova si el formulari s'ha enviat i és vàlid.

from flask import render_template, flash, redirect, url_for

@app.route('/login', methods=['GET', 'POST'])
def login():
    form = LoginForm()
    if form.validate_on_submit():
        flash(f'Sol·licitud d\'inici de sessió per a l\'usuari {form.email.data}, remember_me={form.remember_me.data}')
        return redirect(url_for('index'))
    return render_template('login.html', form=form)
  • Explicació:
    • validate_on_submit() comprova si el formulari s'ha enviat i tots els validadors passen.
    • flash() s'utilitza per mostrar un missatge a l'usuari.
    • redirect() i url_for() s'utilitzen per navegar a una altra ruta.

Validadors Personalitzats

WTForms et permet crear validadors personalitzats si els integrats no compleixen les teves necessitats.

Exemple de Validador Personalitzat

from wtforms.validators import ValidationError

def my_custom_validator(form, field):
    if "@" not in field.data:
        raise ValidationError('El camp ha de contenir un símbol @')

class CustomForm(FlaskForm):
    custom_field = StringField('Camp Personalitzat', validators=[my_custom_validator])
    submit = SubmitField('Enviar')
  • Explicació:
    • Els validadors personalitzats són funcions que llancen ValidationError si la validació falla.
    • Es poden afegir a la llista de validators de qualsevol camp.

Conclusió

WTForms és una eina poderosa per a la validació de formularis en aplicacions Flask. Simplifica el procés de creació, renderització i validació de formularis, assegurant que les entrades dels usuaris siguin netes i compleixin els criteris requerits. Aprofitant els validadors integrats i creant-ne de personalitzats, pots gestionar una àmplia gamma d'escenaris de validació de formularis de manera efectiva.

  • Resum:
    • WTForms s'integra perfectament amb Flask.
    • Els formularis es defineixen com a classes de Python.
    • Els validadors asseguren la integritat de les dades.
    • Es poden crear validadors personalitzats per a necessitats específiques.
    • La gestió de l'enviament de formularis implica renderitzar el formulari en plantilles i processar les dades en funcions de vista.
