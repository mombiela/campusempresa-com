Introducció als Mètodes HTTP

Els mètodes HTTP són una part fonamental del desenvolupament web, permetent que els clients es comuniquin amb els servidors. A Flask, entendre i utilitzar aquests mètodes és crucial per crear aplicacions web robustes.

Mètodes HTTP Clau

  • GET: Recuperar dades del servidor.
  • POST: Enviar dades al servidor per crear un nou recurs.
  • PUT: Actualitzar un recurs existent al servidor.
  • DELETE: Eliminar un recurs del servidor.

Utilitzant Mètodes HTTP a Flask

Configurant Flask

Primer, assegura't que tens Flask instal·lat. Pots instal·lar-lo utilitzant pip:

pip install Flask

Aplicació Bàsica de Flask

Comencem amb una aplicació bàsica de Flask per demostrar com manejar diferents mètodes HTTP.

from flask import Flask, request

app = Flask(__name__)

@app.route('/')
def home():
    return "Benvingut al tutorial de Mètodes HTTP amb Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Manejant Peticions GET

El mètode GET s'utilitza per recuperar dades del servidor. Per defecte, les rutes de Flask manegen peticions GET.

@app.route('/get-example', methods=['GET'])
def get_example():
    return "Aquesta és una petició GET"

Manejant Peticions POST

El mètode POST s'utilitza per enviar dades al servidor. Això és comunament utilitzat per enviaments de formularis.

@app.route('/post-example', methods=['POST'])
def post_example():
    data = request.form['data']
    return f"Rebudes dades POST: {data}"

Manejant Peticions PUT

El mètode PUT s'utilitza per actualitzar recursos existents. A Flask, pots manejar peticions PUT de manera similar a les peticions POST.

@app.route('/put-example', methods=['PUT'])
def put_example():
    data = request.form['data']
    return f"Rebudes dades PUT: {data}"

Manejant Peticions DELETE

El mètode DELETE s'utilitza per eliminar recursos del servidor.

@app.route('/delete-example', methods=['DELETE'])
def delete_example():
    data = request.form['data']
    return f"Rebuda petició DELETE per a les dades: {data}"

Aplicació d'Exemple

Creem una aplicació senzilla que demostri els quatre mètodes HTTP.

from flask import Flask, request, jsonify

app = Flask(__name__)

data_store = {}

@app.route('/data', methods=['GET', 'POST', 'PUT', 'DELETE'])
def data_handler():
    if request.method == 'GET':
        return jsonify(data_store)
    
    elif request.method == 'POST':
        new_data = request.json
        data_store.update(new_data)
        return jsonify({"message": "Dades afegides", "data": new_data})
    
    elif request.method == 'PUT':
        update_data = request.json
        data_store.update(update_data)
        return jsonify({"message": "Dades actualitzades", "data": update_data})
    
    elif request.method == 'DELETE':
        delete_data = request.json
        for key in delete_data.keys():
            data_store.pop(key, None)
        return jsonify({"message": "Dades eliminades", "data": delete_data})

if __name__ == '__main__':
    app.run(debug=True)

Resum

Entendre i utilitzar els mètodes HTTP a Flask és essencial per construir aplicacions web dinàmiques. Aquest tema ha cobert els conceptes bàsics dels mètodes GET, POST, PUT i DELETE, proporcionant exemples i una aplicació senzilla per il·lustrar el seu ús. En dominar aquests mètodes, podràs crear aplicacions web més interactives i funcionals amb Flask.
