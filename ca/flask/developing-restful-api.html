Introducció a les APIs RESTful

  • Definició: REST (Representational State Transfer) és un estil arquitectònic per dissenyar aplicacions en xarxa.
  • Conceptes Clau:
    • Recursos: Objectes o representacions de dades amb els quals els clients interactuen.
    • Mètodes HTTP: GET, POST, PUT, DELETE, etc.
    • Estatelessness: Cada sol·licitud d'un client conté tota la informació necessària per processar la sol·licitud.
    • Endpoints: URLs que representen recursos.

Configuració de Flask

  • Instal·lació: 
    pip install Flask
  • Aplicació Bàsica de Flask: 
    from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hola, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Creació d'una API RESTful Simple

  • Creació d'Endpoints: 
    from flask import Flask, jsonify, request

app = Flask(__name__)

@app.route('/api/resource', methods=['GET'])
def get_resource():
    return jsonify({"message": "Sol·licitud GET rebuda"})

@app.route('/api/resource', methods=['POST'])
def create_resource():
    data = request.get_json()
    return jsonify({"message": "Sol·licitud POST rebuda", "data": data}), 201

if __name__ == '__main__':
    app.run(debug=True)
  • Explicació:
    • @app.route('/api/resource', methods=['GET']): Defineix un endpoint GET.
    • @app.route('/api/resource', methods=['POST']): Defineix un endpoint POST.
    • request.get_json(): Analitza les dades de la sol·licitud JSON entrant.

Gestió d'Operacions CRUD

  • Operacions CRUD:

    • Crear: POST
    • Llegir: GET
    • Actualitzar: PUT
    • Eliminar: DELETE

  • Exemple:

    from flask import Flask, jsonify, request

app = Flask(__name__)

resources = []

@app.route('/api/resources', methods=['GET'])
def get_resources():
    return jsonify(resources)

@app.route('/api/resources', methods=['POST'])
def create_resource():
    data = request.get_json()
    resources.append(data)
    return jsonify(data), 201

@app.route('/api/resources/<int:id>', methods=['PUT'])
def update_resource(id):
    data = request.get_json()
    resources[id] = data
    return jsonify(data)

@app.route('/api/resources/<int:id>', methods=['DELETE'])
def delete_resource(id):
    resources.pop(id)
    return '', 204

if __name__ == '__main__':
    app.run(debug=True)

  • Explicació:

    • resources = []: Una llista simple en memòria per emmagatzemar recursos.
    • @app.route('/api/resources', methods=['GET']): Recupera tots els recursos.
    • @app.route('/api/resources', methods=['POST']): Crea un nou recurs.
    • @app.route('/api/resources/<int:id>', methods=['PUT']): Actualitza un recurs per ID.
    • @app.route('/api/resources/<int:id>', methods=['DELETE']): Elimina un recurs per ID.

Gestió d'Errors

  • Respostes d'Error Personalitzades: 
    from flask import Flask, jsonify

app = Flask(__name__)

@app.errorhandler(404)
def not_found(error):
    return jsonify({"error": "Recurs no trobat"}), 404

@app.errorhandler(500)
def internal_error(error):
    return jsonify({"error": "Error intern del servidor"}), 500

if __name__ == '__main__':
    app.run(debug=True)
  • Explicació:
    • @app.errorhandler(404): Gestor personalitzat per errors 404.
    • @app.errorhandler(500): Gestor personalitzat per errors 500.

Temes Avançats

  • Autenticació i Autorització:
    • JWT (JSON Web Tokens): 
      from flask import Flask, jsonify, request
import jwt
import datetime

app = Flask(__name__)
app.config['SECRET_KEY'] = 'your_secret_key'

@app.route('/login', methods=['POST'])
def login():
    auth_data = request.get_json()
    if auth_data['username'] == 'user' and auth_data['password'] == 'pass':
        token = jwt.encode({'user': auth_data['username'], 'exp': datetime.datetime.utcnow() + datetime.timedelta(minutes=30)}, app.config['SECRET_KEY'])
        return jsonify({'token': token})
    return jsonify({'message': 'Credencials invàlides'}), 401

@app.route('/protected', methods=['GET'])
def protected():
    token = request.headers.get('x-access-tokens')
    if not token:
        return jsonify({'message': 'Falta el token'}), 401
    try:
        data = jwt.decode(token, app.config['SECRET_KEY'], algorithms=["HS256"])
    except:
        return jsonify({'message': 'Token invàlid'}), 401
    return jsonify({'message': 'Ruta protegida accedida'})

if __name__ == '__main__':
    app.run(debug=True)
    • Explicació:
      • jwt.encode(): Codifica una càrrega útil en un token JWT.
      • jwt.decode(): Decodifica un token JWT.
      • @app.route('/login', methods=['POST']): Endpoint per autenticar i generar un token.
      • @app.route('/protected', methods=['GET']): Endpoint protegit que requereix un token vàlid.

Conclusió

Desenvolupar una API RESTful amb Flask implica entendre els conceptes bàsics de l'arquitectura REST, configurar Flask, crear endpoints, gestionar operacions CRUD i implementar la gestió d'errors. Temes avançats com l'autenticació i l'autorització es poden afegir per assegurar l'API. Amb aquests conceptes, pots construir APIs robustes i escalables utilitzant Flask.
