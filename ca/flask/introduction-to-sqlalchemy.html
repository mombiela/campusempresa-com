Visió general

SQLAlchemy és una potent eina SQL i una biblioteca de Mapatge Objecte-Relacional (ORM) per a Python. Proporciona un conjunt complet de patrons de persistència a nivell empresarial ben coneguts, dissenyats per a un accés eficient i d'alt rendiment a la base de dades. En aquesta secció, cobrirem els conceptes bàsics de SQLAlchemy i com integrar-lo amb una aplicació Flask.

Conceptes clau

  • SQLAlchemy Core: La capa fonamental que proporciona un llenguatge d'expressió SQL i un enfocament centrat en l'esquema per a la interacció amb la base de dades.
  • SQLAlchemy ORM: Una capa opcional que proporciona una interfície orientada a objectes de més alt nivell per a operacions amb bases de dades.
  • Engine: El punt de partida per a qualsevol aplicació SQLAlchemy, que representa la connexió amb la base de dades.
  • Session: Un espai de treball per a tots els objectes carregats des de la base de dades durant una transacció.
  • Model: Una classe Python que es mapatge a una taula de la base de dades.

Configuració de SQLAlchemy amb Flask

Instal·lació

Per començar, cal instal·lar Flask i SQLAlchemy. Pots fer-ho utilitzant pip:

pip install Flask SQLAlchemy

Configuració

Primer, configura la teva aplicació Flask per utilitzar SQLAlchemy. Aquí tens un exemple bàsic:

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///example.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

db = SQLAlchemy(app)
  • SQLALCHEMY_DATABASE_URI: Especifica l'URL de la base de dades.
  • SQLALCHEMY_TRACK_MODIFICATIONS: Desactiva una funció que senyala l'aplicació cada vegada que es fa un canvi a la base de dades, cosa que pot ser útil per a depuració però no es recomana per a producció.

Definició de Models

Els models són classes Python que representen taules de la base de dades. Aquí tens un exemple d'un model senzill:

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'
  • db.Column: Defineix una columna a la taula de la base de dades.
  • db.Integer, db.String: Especifica el tipus de dades de la columna.
  • primary_key=True: Indica que aquesta columna és la clau primària.
  • unique=True: Assegura que tots els valors d'aquesta columna siguin únics.
  • nullable=False: Assegura que aquesta columna no pugui ser nul·la.

Creació de la Base de Dades

Per crear la base de dades i les taules, cal executar les següents comandes en una shell de Python:

from your_application import db
db.create_all()

Això crearà les taules definides pels teus models a la base de dades especificada per SQLALCHEMY_DATABASE_URI.

Operacions CRUD

Crear

Per afegir un nou registre a la base de dades:

new_user = User(username='john_doe', email='john@example.com')
db.session.add(new_user)
db.session.commit()

Llegir

Per consultar registres de la base de dades:

users = User.query.all()  # Obtenir tots els usuaris
user = User.query.filter_by(username='john_doe').first()  # Obtenir un usuari específic

Actualitzar

Per actualitzar un registre existent:

user = User.query.filter_by(username='john_doe').first()
user.email = 'john_doe@example.com'
db.session.commit()

Esborrar

Per esborrar un registre:

user = User.query.filter_by(username='john_doe').first()
db.session.delete(user)
db.session.commit()

Conclusió

En aquesta secció, hem introduït SQLAlchemy i hem demostrat com integrar-lo amb una aplicació Flask. Hem cobert els conceptes bàsics de la configuració de SQLAlchemy, la definició de models i la realització d'operacions CRUD. SQLAlchemy és una eina potent que pot simplificar molt les interaccions amb la base de dades en les teves aplicacions Flask. A mesura que progressis, aprendràs funcions més avançades i millors pràctiques per utilitzar SQLAlchemy de manera efectiva.
