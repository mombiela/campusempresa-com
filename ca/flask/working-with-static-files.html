En aquesta secció, explorarem com gestionar fitxers estàtics a Flask. Els fitxers estàtics inclouen CSS, JavaScript, imatges i altres recursos que no canvien dinàmicament. Entendre com servir aquests fitxers és crucial per construir una aplicació web completa.

Introducció als Fitxers Estàtics

  • Definició: Els fitxers estàtics són fitxers que s'envien al client tal com són, sense cap processament del costat del servidor.
  • Exemples: Fitxers CSS, fitxers JavaScript, imatges, fonts.

Configuració dels Fitxers Estàtics a Flask

Per defecte, Flask espera que els fitxers estàtics estiguin situats en una carpeta anomenada static al directori arrel del teu projecte.

Exemple d'Estructura de Projecte

my_flask_app/
│
├── app.py
├── static/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── scripts.js
│   └── images/
│       └── logo.png
└── templates/
    └── index.html

Servir Fitxers Estàtics

Flask serveix automàticament fitxers des de la carpeta static. Pots referenciar aquests fitxers en les teves plantilles HTML.

Exemple: Enllaçar un Fitxer CSS

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>La Meva App Flask</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <h1>Benvingut a La Meva App Flask</h1>
</body>
</html>
  • Explicació: La funció url_for genera la URL per al fitxer estàtic. Això assegura que s'utilitzi el camí correcte, fins i tot si l'aplicació es desplega en un subdirectori.

Organització dels Fitxers Estàtics

  • CSS: Emmagatzema tots els teus fitxers CSS en una subcarpeta css.
  • JavaScript: Emmagatzema tots els teus fitxers JavaScript en una subcarpeta js.
  • Imatges: Emmagatzema totes les teves imatges en una subcarpeta images.

Exemple: Enllaçar un Fitxer JavaScript

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>La Meva App Flask</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <h1>Benvingut a La Meva App Flask</h1>
    <script src="{{ url_for('static', filename='js/scripts.js') }}"></script>
</body>
</html>

Personalització de la Carpeta Estàtica

Si vols utilitzar una carpeta diferent per als teus fitxers estàtics, pots personalitzar-la quan creïs l'aplicació Flask.

Exemple: Carpeta Estàtica Personalitzada

from flask import Flask

app = Flask(__name__, static_folder='assets')

@app.route('/')
def home():
    return 'Hola, Flask!'

if __name__ == '__main__':
    app.run(debug=True)
  • Explicació: En aquest exemple, els fitxers estàtics es serveixen des de la carpeta assets en lloc de la carpeta static per defecte.

Millors Pràctiques

  • Minimitzar i Comprimir: Minimitza i comprimeix els teus fitxers CSS i JavaScript per reduir els temps de càrrega.
  • Control de Caché: Utilitza capçaleres de control de caché per millorar el rendiment emmagatzemant fitxers estàtics al costat del client.
  • Organitzar: Mantingues els teus fitxers estàtics ben organitzats en subcarpetes per facilitar-ne el manteniment.

Conclusió

Gestionar fitxers estàtics a Flask és senzill gràcies al seu suport integrat. Seguint les millors pràctiques i organitzant correctament els teus fitxers, pots assegurar-te que la teva aplicació web sigui eficient i fàcil de mantenir. Recorda utilitzar la funció url_for per generar URLs per als teus fitxers estàtics, la qual cosa ajuda a evitar problemes amb camins incorrectes.
