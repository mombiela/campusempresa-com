L'enrutament dinàmic d'URL és una característica poderosa de Flask que et permet crear URL que poden canviar segons les dades que reben. Això és essencial per crear aplicacions web flexibles i escalables. En aquesta secció, cobrirem els conceptes bàsics de l'enrutament dinàmic d'URL, proporcionarem exemples i inclourem exercicis per consolidar la teva comprensió.

Què és l'enrutament dinàmic d'URL?

L'enrutament dinàmic d'URL et permet definir rutes que poden acceptar paràmetres. Aquests paràmetres es poden utilitzar per passar dades a les teves funcions de vista, fent que la teva aplicació web sigui més dinàmica i interactiva.

Conceptes clau

  • Paràmetres de ruta: Variables a l'URL que es poden capturar i utilitzar a les teves funcions de vista.
  • Convertidors d'URL: Especifiquen el tipus dels paràmetres de ruta (per exemple, enter, cadena).

Enrutament dinàmic d'URL bàsic

Comencem amb un exemple senzill per entendre com funciona l'enrutament dinàmic d'URL a Flask.

Exemple: Ruta bàsica amb un paràmetre

from flask import Flask

app = Flask(__name__)

@app.route('/user/<username>')
def show_user_profile(username):
    return f'User {username}'

if __name__ == '__main__':
    app.run(debug=True)

En aquest exemple:

  • La ruta /user/<username> captura el valor a l'URL i el passa a la funció show_user_profile.
  • Si navegues a /user/John, la funció retornarà User John.

Convertidors d'URL

Flask proporciona diversos convertidors per especificar el tipus dels paràmetres de ruta:

| Convertidor | Descripció | |-------------|-----------------------------| | <int> | Accepta enters | | <float> | Accepta números de punt flotant | | <path> | Accepta cadenes amb barres |

Exemple: Ús de convertidors d'URL

@app.route('/post/<int:post_id>')
def show_post(post_id):
    return f'Post {post_id}'

En aquest exemple:

  • La ruta /post/<int:post_id> assegura que post_id és un enter.
  • Si navegues a /post/123, la funció retornarà Post 123.

Enrutament dinàmic d'URL avançat

A mesura que et sentis més còmode amb l'enrutament dinàmic bàsic, pots començar a explorar característiques més avançades.

Múltiples paràmetres

Pots capturar múltiples paràmetres en una sola ruta.

@app.route('/user/<username>/post/<int:post_id>')
def show_user_post(username, post_id):
    return f'User {username}, Post {post_id}'

En aquest exemple:

  • La ruta /user/<username>/post/<int:post_id> captura tant username com post_id.
  • Si navegues a /user/John/post/123, la funció retornarà User John, Post 123.

Paràmetres opcionals

També pots definir rutes amb paràmetres opcionals utilitzant valors per defecte.

@app.route('/page/<int:page_num>')
@app.route('/page/', defaults={'page_num': 1})
def show_page(page_num):
    return f'Page {page_num}'

En aquest exemple:

  • La ruta /page/<int:page_num> captura page_num.
  • La ruta /page/ utilitza un valor per defecte de 1 per page_num.
  • Si navegues a /page/, la funció retornarà Page 1.

Exercicis

  1. Ruta dinàmica bàsica: Crea una ruta que capturi el nom i l'edat d'un usuari, i retorni un missatge de benvinguda.
  2. Convertidor d'URL: Crea una ruta que capturi un ID de producte (enter) i retorni els detalls del producte.
  3. Múltiples paràmetres: Crea una ruta que capturi una categoria i un ID d'element, i retorni els detalls de l'element dins d'aquesta categoria.
  4. Paràmetres opcionals: Crea una ruta que capturi un número de pàgina amb un valor per defecte de 1, i retorni el contingut de la pàgina.

Conclusió

L'enrutament dinàmic d'URL a Flask et permet crear aplicacions web flexibles i interactives capturant i utilitzant paràmetres d'URL. Entenent i utilitzant paràmetres de ruta, convertidors d'URL i paràmetres opcionals, pots construir aplicacions més dinàmiques i fàcils d'usar. Practica els exercicis proporcionats per consolidar la teva comprensió i explora tècniques d'enrutament més avançades a mesura que et sentis més còmode amb Flask.
