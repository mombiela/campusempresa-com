Introducció

L'autenticació d'usuaris és un aspecte crucial del desenvolupament web, assegurant que els usuaris puguin iniciar sessió de manera segura i accedir a les seves dades. En aquesta secció, cobrirem els conceptes bàsics de l'autenticació d'usuaris en Flask, progressant des de conceptes per a principiants fins a conceptes avançats.

Configuració de Flask

Abans d'entrar en l'autenticació d'usuaris, assegura't de tenir Flask instal·lat i configurat.

pip install Flask

Crea una aplicació bàsica de Flask:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hola, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Autenticació Bàsica d'Usuaris

Creació d'un Model d'Usuari

Per gestionar les dades dels usuaris, crearem un model d'Usuari. Per simplicitat, utilitzarem un diccionari per emmagatzemar la informació dels usuaris.

users = {
    "user1": {"username": "user1", "password": "password1"},
    "user2": {"username": "user2", "password": "password2"}
}

Ruta d'Inici de Sessió

Crea una ruta per gestionar l'inici de sessió dels usuaris.

from flask import request, jsonify

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        return jsonify({"message": "Inici de sessió reeixit!"}), 200
    return jsonify({"message": "Credencials invàlides"}), 401

Protecció de Rutes

Per protegir certes rutes, podem utilitzar un enfocament senzill basat en sessions.

from flask import session

app.secret_key = 'clausecreta'

@app.route('/protected')
def protected():
    if 'username' in session:
        return f"Hola, {session['username']}!"
    return "No has iniciat sessió!", 401

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        session['username'] = username
        return jsonify({"message": "Inici de sessió reeixit!"}), 200
    return jsonify({"message": "Credencials invàlides"}), 401

@app.route('/logout')
def logout():
    session.pop('username', None)
    return "Sessió tancada!"

Autenticació Intermèdia d'Usuaris

Ús de Flask-Login

Flask-Login proporciona gestió de sessions d'usuari per a Flask. Instal·la Flask-Login:

pip install Flask-Login

Configuració de Flask-Login

Integra Flask-Login a la teva aplicació.

from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin):
    def __init__(self, id):
        self.id = id

@login_manager.user_loader
def load_user(user_id):
    return User(user_id)

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "Inici de sessió reeixit!"}), 200
    return jsonify({"message": "Credencials invàlides"}), 401

@app.route('/protected')
@login_required
def protected():
    return f"Hola, {current_user.id}!"

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return "Sessió tancada!"

Autenticació Avançada d'Usuaris

Hashing de Contrasenyes

Emmagatzemar contrasenyes en text pla és insegur. Utilitza una biblioteca de hashing com werkzeug.security per fer hashing de les contrasenyes.

from werkzeug.security import generate_password_hash, check_password_hash

# Hashing de contrasenyes
users = {
    "user1": {"username": "user1", "password": generate_password_hash("password1")},
    "user2": {"username": "user2", "password": generate_password_hash("password2")}
}

# Verificació de contrasenyes
@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and check_password_hash(user['password'], password):
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "Inici de sessió reeixit!"}), 200
    return jsonify({"message": "Credencials invàlides"}), 401

Ús de Flask-SQLAlchemy

Per a una solució més robusta, integra Flask-SQLAlchemy per gestionar les dades dels usuaris en una base de dades.

pip install Flask-SQLAlchemy
from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///users.db'
db = SQLAlchemy(app)

class User(db.Model, UserMixin):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

# Crear la base de dades
db.create_all()

# Afegir un usuari
new_user = User(username='user1', password=generate_password_hash('password1'))
db.session.add(new_user)
db.session.commit()

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = User.query.filter_by(username=username).first()
    if user and check_password_hash(user.password, password):
        login_user(user)
        return jsonify({"message": "Inici de sessió reeixit!"}), 200
    return jsonify({"message": "Credencials invàlides"}), 401
