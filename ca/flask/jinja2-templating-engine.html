Introducció a Jinja2

Jinja2 és un motor de plantilles potent per a Python, utilitzat extensament amb el framework web Flask per crear pàgines web dinàmiques. Permet incrustar expressions semblants a Python en els teus fitxers HTML, facilitant la generació dinàmica d'HTML.

Conceptes clau

  • Plantilles: Fitxers HTML amb sintaxi Jinja2 incrustada.
  • Variables: Marcadors de posició per a contingut dinàmic.
  • Estructures de control: Lògica per a bucles i condicionals.
  • Filtres: Funcions per modificar variables.
  • Macros: Blocs de codi reutilitzables.

Configuració de Jinja2 a Flask

Per utilitzar Jinja2 amb Flask, cal configurar correctament l'estructura del teu projecte.

Estructura del projecte

my_flask_app/
│
├── app.py
├── templates/
│   ├── base.html
│   └── index.html
└── static/
    └── style.css

Aplicació bàsica de Flask

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html', title='Home Page')

if __name__ == '__main__':
    app.run(debug=True)

Ús de variables en plantilles

Les variables en Jinja2 estan tancades en doble claudàtors {{ }}.

Exemple

<!-- index.html -->
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
</head>
<body>
    <h1>Benvingut a {{ title }}</h1>
</body>
</html>

Explicació

  • {{ title }}: Això serà reemplaçat pel valor de la variable title passada des de la ruta de Flask.

Estructures de control

Jinja2 suporta estructures de control com bucles i condicionals.

Bucles

<!-- index.html -->
<ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

Condicionals

<!-- index.html -->
{% if user %}
    <p>Benvingut, {{ user }}!</p>
{% else %}
    <p>Benvingut, Convidat!</p>
{% endif %}

Explicació

  • {% for item in items %}: Recorre cada element de la llista items.
  • {% if user %}: Comprova si la variable user està definida.

Filtres

Els filtres s'utilitzen per modificar variables. S'apliquen utilitzant el símbol de tub |.

Exemple

<!-- index.html -->
<p>{{ name | upper }}</p>
<p>{{ date | dateformat('%Y-%m-%d') }}</p>

Explicació

  • {{ name | upper }}: Converteix la variable name a majúscules.
  • {{ date | dateformat('%Y-%m-%d') }}: Dona format a la variable date.

Macros

Les macros són blocs de codi reutilitzables que es poden definir i cridar dins de les plantilles.

Exemple

<!-- macros.html -->
{% macro render_item(item) %}
    <li>{{ item }}</li>
{% endmacro %}
<!-- index.html -->
{% from 'macros.html' import render_item %}
<ul>
    {% for item in items %}
        {{ render_item(item) }}
    {% endfor %}
</ul>

Explicació

  • {% macro render_item(item) %}: Defineix una macro anomenada render_item.
  • {% from 'macros.html' import render_item %}: Importa la macro render_item des de macros.html.

Conclusió

Jinja2 és una eina essencial per crear pàgines web dinàmiques a Flask. Comprenent els seus conceptes clau—plantilles, variables, estructures de control, filtres i macros—pots construir aplicacions web més interactives i amigables per a l'usuari. Practica utilitzant aquestes característiques per esdevenir proficient en Jinja2 i millorar els teus projectes Flask.
