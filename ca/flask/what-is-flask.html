Flask és un micro framework web escrit en Python. Està dissenyat per ser lleuger i modular, cosa que facilita escalar-lo a aplicacions complexes o reduir-lo a aplicacions senzilles. Flask és conegut per la seva simplicitat, flexibilitat i control detallat.

Característiques Clau de Flask

  • Lleuger i Modular: Flask és minimalista i no inclou moltes funcions integrades, permetent als desenvolupadors afegir només el que necessiten.
  • Servidor de Desenvolupament Integrat: Flask inclou un servidor integrat per al desenvolupament i les proves.
  • Despatx de Sol·licituds RESTful: Flask suporta el despatx de sol·licituds RESTful, facilitant la creació d'APIs.
  • Plantilles Jinja2: Flask utilitza Jinja2 per a la generació de plantilles, permetent la generació dinàmica d'HTML.
  • Extensible: Flask es pot ampliar amb diversos plugins i extensions per afegir funcionalitat.
  • Proves Unitàries Integrades: Flask suporta proves unitàries, facilitant la prova de les aplicacions.

Per què Utilitzar Flask?

  • Simplicitat: Flask és fàcil d'aprendre i utilitzar, cosa que el converteix en una gran elecció per a principiants.
  • Flexibilitat: Flask permet als desenvolupadors triar els components que volen utilitzar, proporcionant un major control sobre l'aplicació.
  • Comunitat i Documentació: Flask té una gran comunitat i una documentació extensa, que pot ser molt útil per a la resolució de problemes i l'aprenentatge.

Aplicació Bàsica de Flask

Comencem amb un exemple senzill per demostrar com funciona Flask. Aquest exemple crearà una aplicació web bàsica que mostra "Hello, World!" quan s'accedeix.

Guia Pas a Pas

  1. Instal·lar Flask: Primer, necessites instal·lar Flask. Pots fer-ho utilitzant pip:

    pip install Flask

  2. Crear una Aplicació Flask: Crea un nou fitxer Python, per exemple, app.py, i afegeix el següent codi:

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

  3. Executar l'Aplicació: Executa l'aplicació executant la següent comanda al teu terminal:

    python app.py

  4. Accedir a l'Aplicació: Obre el teu navegador web i ves a http://127.0.0.1:5000/. Hauries de veure "Hello, World!" mostrat a la pàgina.

Explicació del Codi

  • Importar Flask: La línia from flask import Flask importa la classe Flask del mòdul Flask.
  • Crear una Instància de l'Aplicació: app = Flask(__name__) crea una instància de la classe Flask. La variable __name__ es passa al constructor de Flask per ajudar Flask a determinar el camí arrel de l'aplicació.
  • Definir una Ruta: El decorador @app.route('/') defineix una ruta per a l'URL arrel (/). La funció hello_world es crida quan s'accedeix a aquest URL, i retorna la cadena "Hello, World!".
  • Executar l'Aplicació: El bloc if __name__ == '__main__': assegura que l'aplicació s'executi només si el script s'executa directament (no importat com a mòdul). app.run(debug=True) inicia el servidor de desenvolupament amb el mode de depuració activat.

Conclusió

Flask és un framework web potent però senzill per a Python que permet als desenvolupadors crear aplicacions web amb una configuració mínima. La seva naturalesa lleugera i flexibilitat el fan una excel·lent elecció tant per a principiants com per a desenvolupadors experimentats. En aquesta secció, hem cobert les bases de què és Flask, les seves característiques clau i com crear una aplicació senzilla amb Flask. A mesura que avancem en aquest curs, aprofundirem en temes i característiques més avançades de Flask, construint sobre aquest coneixement fonamental.
