Introducció

Flask és un micro framework web escrit en Python. Està dissenyat per ser simple i fàcil d'utilitzar, fent-lo una excel·lent elecció per a principiants que volen començar amb el desenvolupament web. En aquesta secció, et guiarem a través de la creació de la teva primera aplicació Flask.

Configurant el teu entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

Instal·lant Flask

Per instal·lar Flask, necessites tenir Python i pip (l'instal·lador de paquets de Python) instal·lats al teu sistema. Pots instal·lar Flask utilitzant pip amb la següent comanda:

pip install Flask

Creant un directori de projecte

Crea un directori per al teu projecte Flask. Això t'ajudarà a mantenir els teus arxius de projecte organitzats.

mkdir my_flask_app
cd my_flask_app

Escriure la teva primera aplicació Flask

Ara que tenim el nostre entorn configurat, escrivim la nostra primera aplicació Flask.

Creant l'arxiu de l'aplicació

Crea un nou arxiu Python anomenat app.py en el teu directori de projecte. Aquest arxiu contindrà el codi per a la teva aplicació Flask.

# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Explicació del codi

  • Importar Flask: Comencem importat la classe Flask del mòdul flask.
  • Crear una instància de Flask: Creem una instància de la classe Flask. Aquesta instància serà la nostra aplicació WSGI.
  • Definir una ruta: Utilitzem el decorador @app.route('/') per definir una ruta per a l'URL arrel (/). Quan un usuari visiti aquest URL, es cridarà la funció home.
  • Retornar una resposta: La funció home retorna una cadena simple "Hello, Flask!" que es mostrarà en el navegador web de l'usuari.
  • Executar l'aplicació: Finalment, comprovem si l'script s'executa directament (no importat com a mòdul) i cridem app.run(debug=True) per iniciar el servidor de desenvolupament. L'argument debug=True habilita el mode de depuració, que proporciona missatges d'error útils i recarrega automàticament el servidor quan es fan canvis en el codi.

Executar la teva aplicació Flask

Per executar la teva aplicació Flask, navega al teu directori de projecte i executa la següent comanda:

python app.py

Hauries de veure una sortida indicant que el servidor està en funcionament. Obre el teu navegador web i ves a http://127.0.0.1:5000/. Hauries de veure el missatge "Hello, Flask!".

Resum

En aquesta secció, hem cobert els conceptes bàsics de la creació d'una aplicació Flask simple. Hem configurat el nostre entorn de desenvolupament, hem escrit una aplicació Flask bàsica i l'hem executat en un servidor local. Aquest coneixement fonamental servirà com a base per a temes més avançats de Flask.
