Introducció als Plànols

Els plànols a Flask són una manera d'organitzar la teva aplicació en components més petits i reutilitzables. Ajuden a gestionar aplicacions grans permetent-te dividir l'aplicació en mòduls.

  • Definició: Un Plànol és una manera d'organitzar la teva aplicació Flask en mòduls, fent-la més fàcil de gestionar i escalar.
  • Propòsit: Separar diferents parts de l'aplicació, com ara l'autenticació, el lloc principal, el panell d'administració, etc.
  • Beneficis:
    • Millora de l'organització del codi
    • Manteniment més fàcil
    • Reutilització millorada
    • Simplificació de les proves

Configuració dels Plànols

Per començar a utilitzar Plànols, necessites crear un objecte Plànol i registrar-lo amb la teva aplicació Flask.

Creació d'un Plànol

Primer, crea un nou fitxer Python per al teu Plànol. Per exemple, creem un Plànol main.

# main.py
from flask import Blueprint

main = Blueprint('main', __name__)

@main.route('/')
def index():
    return "Benvingut a la pàgina principal!"

Registre d'un Plànol

A continuació, registra el Plànol amb la teva aplicació Flask en el fitxer principal de l'aplicació.

# app.py
from flask import Flask
from main import main

app = Flask(__name__)
app.register_blueprint(main)

if __name__ == '__main__':
    app.run(debug=True)

Organització dels Plànols

Quan la teva aplicació creix, potser voldràs organitzar els teus Plànols en un paquet.

Creació d'un Paquet per als Plànols

Crea una estructura de directoris com aquesta:

/myapp
    /blueprints
        /main
            __init__.py
            routes.py
        /auth
            __init__.py
            routes.py
    app.py

Definició de Plànols en un Paquet

En main/__init__.py:

from flask import Blueprint

main = Blueprint('main', __name__)

from . import routes

En main/routes.py:

from . import main

@main.route('/')
def index():
    return "Benvingut a la pàgina principal!"

Registre de Plànols des d'un Paquet

En app.py:

from flask import Flask
from blueprints.main import main as main_blueprint
from blueprints.auth import auth as auth_blueprint

app = Flask(__name__)
app.register_blueprint(main_blueprint)
app.register_blueprint(auth_blueprint, url_prefix='/auth')

if __name__ == '__main__':
    app.run(debug=True)

Ús Avançat dels Plànols

Els plànols també es poden utilitzar per gestionar escenaris més complexos, com ara plànols niats i configuracions específiques de plànols.

Plànols Niats

Pots niar Plànols dins d'altres Plànols per modularitzar encara més la teva aplicació.

# admin/__init__.py
from flask import Blueprint

admin = Blueprint('admin', __name__)

from . import routes

# admin/routes.py
from . import admin

@admin.route('/dashboard')
def dashboard():
    return "Benvingut al tauler d'administració!"

Configuracions Específiques de Plànols

També pots definir configuracions que són específiques per a un Plànol.

# auth/__init__.py
from flask import Blueprint

auth = Blueprint('auth', __name__)

@auth.before_request
def before_request():
    # Codi per executar abans de cada sol·licitud en el Plànol auth
    pass

from . import routes

Conclusió

Els plànols són una característica poderosa a Flask que t'ajuden a gestionar i escalar aplicacions grans organitzant el teu codi en components més petits i reutilitzables. Utilitzant Plànols, pots millorar la mantenibilitat, llegibilitat i testabilitat de la teva aplicació. Tant si estàs treballant en un projecte petit com en una aplicació a gran escala, entendre i utilitzar Plànols millorarà significativament el teu flux de treball de desenvolupament.
