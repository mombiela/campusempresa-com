Flask és un framework web lleuger i flexible per a Python. Una de les seves fortaleses és la seva extensibilitat a través de diverses extensions que afegeixen funcionalitat a les teves aplicacions. En aquesta secció, explorarem algunes de les extensions de Flask més utilitzades, com instal·lar-les i com integrar-les a les teves aplicacions Flask.

Què són les Extensions de Flask?

Les extensions de Flask són paquets que afegeixen funcions específiques a les aplicacions Flask. Estan dissenyades per treballar perfectament amb Flask i seguir les seves convencions. Les extensions poden ajudar-te amb tasques com la integració de bases de dades, la gestió de formularis, l'autenticació i més.

Extensions de Flask Més Utilitzades

Aquí tens algunes de les extensions de Flask més populars:

  • Flask-SQLAlchemy: Afegeix suport de SQLAlchemy a les aplicacions Flask.
  • Flask-WTF: Integra WTForms amb Flask per a la gestió de formularis.
  • Flask-Login: Gestiona sessions d'usuari i autenticació.
  • Flask-Migrate: Gestiona migracions de bases de dades per a aplicacions Flask utilitzant Alembic.
  • Flask-Mail: Proporciona capacitats d'enviament de correu electrònic.
  • Flask-Caching: Afegeix suport de memòria cau a les aplicacions Flask.

Instal·lació d'Extensions de Flask

Pots instal·lar extensions de Flask utilitzant pip. Per exemple, per instal·lar Flask-SQLAlchemy, hauries d'executar:

pip install Flask-SQLAlchemy

Pots instal·lar múltiples extensions alhora llistant-les a la comanda pip:

pip install Flask-SQLAlchemy Flask-WTF Flask-Login

Ús d'Extensions de Flask

Flask-SQLAlchemy

Flask-SQLAlchemy simplifica la integració de SQLAlchemy amb Flask. Aquí tens un exemple bàsic:

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///example.db'
db = SQLAlchemy(app)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'

@app.route('/')
def index():
    return 'Hola, Món!'

if __name__ == '__main__':
    app.run(debug=True)

Flask-WTF

Flask-WTF integra WTForms amb Flask per a la gestió de formularis. Aquí tens un exemple:

from flask import Flask, render_template, request
from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField
from wtforms.validators import DataRequired

app = Flask(__name__)
app.config['SECRET_KEY'] = 'mysecret'

class MyForm(FlaskForm):
    name = StringField('Nom', validators=[DataRequired()])
    submit = SubmitField('Enviar')

@app.route('/', methods=['GET', 'POST'])
def index():
    form = MyForm()
    if form.validate_on_submit():
        name = form.name.data
        return f'Hola, {name}!'
    return render_template('index.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)

Flask-Login

Flask-Login proporciona gestió de sessions d'usuari. Aquí tens un exemple:

from flask import Flask, render_template, redirect, url_for
from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

app = Flask(__name__)
app.config['SECRET_KEY'] = 'mysecret'
login_manager = LoginManager(app)

class User(UserMixin):
    def __init__(self, id):
        self.id = id

@login_manager.user_loader
def load_user(user_id):
    return User(user_id)

@app.route('/login')
def login():
    user = User(id=1)
    login_user(user)
    return redirect(url_for('protected'))

@app.route('/protected')
@login_required
def protected():
    return f'Iniciat sessió com: {current_user.id}'

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return 'Sessió tancada'

if __name__ == '__main__':
    app.run(debug=True)

Conclusió

Les extensions de Flask són eines poderoses que poden millorar significativament la funcionalitat de les teves aplicacions Flask. Aprofitant extensions com Flask-SQLAlchemy, Flask-WTF i Flask-Login, pots afegir funcions robustes com la integració de bases de dades, la gestió de formularis i l'autenticació d'usuaris amb un esforç mínim. Entendre com instal·lar i utilitzar aquestes extensions és crucial per desenvolupar aplicacions Flask avançades.
