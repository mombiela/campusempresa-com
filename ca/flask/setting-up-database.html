Introducció

En aquesta secció, aprendrem a configurar una base de dades per a una aplicació Flask. Cobrirà els conceptes bàsics de la connexió a una base de dades, la creació de taules i la realització d'operacions CRUD (Crear, Llegir, Actualitzar, Esborrar). Al final d'aquesta secció, tindràs una comprensió sòlida de com integrar una base de dades amb la teva aplicació Flask.

Triar una Base de Dades

Abans de començar, és important triar la base de dades adequada per a la teva aplicació. Flask suporta diverses bases de dades, però les més utilitzades són:

  • SQLite: Lleugera i fàcil de configurar, ideal per a aplicacions petites o desenvolupament.
  • PostgreSQL: Potent i escalable, adequada per a aplicacions més grans.
  • MySQL: Molt utilitzada i fiable, bona per a aplicacions mitjanes a grans.

Per a aquest curs, utilitzarem SQLite per la seva simplicitat i facilitat de configuració.

Configurar SQLite amb Flask

Instal·lar Flask i Flask-SQLAlchemy

Primer, assegura't de tenir instal·lats Flask i Flask-SQLAlchemy. Flask-SQLAlchemy és una extensió que simplifica la integració de SQLAlchemy amb Flask.

pip install Flask Flask-SQLAlchemy

Configurar la Base de Dades

A continuació, configura la teva aplicació Flask per utilitzar SQLite. Crea un fitxer anomenat app.py i afegeix el següent codi:

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///site.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False
db = SQLAlchemy(app)

Crear Models

Els models són classes que representen taules a la teva base de dades. Creem un model senzill per a una taula User.

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    password = db.Column(db.String(60), nullable=False)

    def __repr__(self):
        return f"User('{self.username}', '{self.email}')"

Crear la Base de Dades

Per crear la base de dades i les taules, utilitza les següents comandes en una consola de Python:

from app import db
db.create_all()

Això crearà un fitxer site.db amb una taula User.

Realitzar Operacions CRUD

Crear

Per afegir un nou usuari a la base de dades, utilitza el següent codi:

from app import db, User

new_user = User(username='john_doe', email='john@example.com', password='password123')
db.session.add(new_user)
db.session.commit()

Llegir

Per consultar usuaris de la base de dades:

users = User.query.all()
print(users)

user = User.query.filter_by(username='john_doe').first()
print(user)

Actualitzar

Per actualitzar la informació d'un usuari:

user = User.query.filter_by(username='john_doe').first()
user.email = 'john_doe@example.com'
db.session.commit()

Esborrar

Per esborrar un usuari de la base de dades:

user = User.query.filter_by(username='john_doe').first()
db.session.delete(user)
db.session.commit()

Resum

En aquesta secció, hem cobert els conceptes bàsics de configurar una base de dades en una aplicació Flask utilitzant SQLite. Hem après a instal·lar els paquets necessaris, configurar la base de dades, crear models i realitzar operacions CRUD. Aquest coneixement fonamental et permetrà construir aplicacions més complexes amb interaccions robustes amb la base de dades.
