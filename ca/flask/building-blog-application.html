Introducció

En aquest curs, construirem una aplicació de blog completa utilitzant Flask, un popular micro framework web per a Python. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat, cobrint tots els aspectes essencials del desenvolupament amb Flask.

Configurant l'Entorn

  • Instal·lar Flask: Utilitza pip per instal·lar Flask. 
    pip install Flask
  • Estructura del Projecte: Organitza els teus fitxers de projecte. 
    blog/
├── app.py
├── templates/
│   ├── base.html
│   ├── index.html
│   ├── post.html
├── static/
│   ├── style.css
└── models.py

Creant l'Aplicació Flask

  • Aplicació Flask Bàsica: Crea una aplicació Flask senzilla. 
    from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hola, Blog!"

if __name__ == '__main__':
    app.run(debug=True)

Plantilles i Fitxers Estàtics

  • Plantilles HTML: Utilitza el motor de plantilles Jinja2. 
    <!-- templates/base.html -->
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}Blog{% endblock %}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">
</head>
<body>
    {% block content %}{% endblock %}
</body>
</html>
  • Fitxers Estàtics: Serveix fitxers CSS i JavaScript. 
    /* static/style.css */
body {
    font-family: Arial, sans-serif;
}

Rutes i Vistes

  • Rutes Dinàmiques: Crea rutes amb segments dinàmics. 
    @app.route('/post/<int:post_id>')
def post(post_id):
    return f"Post {post_id}"

Models i Integració de Base de Dades

  • Configuració de SQLAlchemy: Integra SQLAlchemy per a la gestió de bases de dades.

    from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///blog.db'
db = SQLAlchemy(app)

  • Creant Models: Defineix models per als posts del blog.

    # models.py
from app import db

class Post(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    content = db.Column(db.Text, nullable=False)
    date_posted = db.Column(db.DateTime, nullable=False, default=datetime.utcnow)

Formularis i Entrada d'Usuari

  • WTForms: Utilitza Flask-WTF per a la gestió de formularis. 
    from flask_wtf import FlaskForm
from wtforms import StringField, TextAreaField, SubmitField
from wtforms.validators import DataRequired

class PostForm(FlaskForm):
    title = StringField('Títol', validators=[DataRequired()])
    content = TextAreaField('Contingut', validators=[DataRequired()])
    submit = SubmitField('Publicar')

Autenticació d'Usuaris

  • Flask-Login: Implementa l'autenticació d'usuaris. 
    from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin, db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

Funcionalitats Avançades

  • Paginació: Implementa la paginació per als posts del blog.

    @app.route('/posts')
def posts():
    page = request.args.get('page', 1, type=int)
    posts = Post.query.paginate(page=page, per_page=5)
    return render_template('posts.html', posts=posts)

  • Funcionalitat de Cerca: Afegeix funcionalitat de cerca al blog.

    @app.route('/search')
def search():
    query = request.args.get('query')
    results = Post.query.filter(Post.title.contains(query) | Post.content.contains(query)).all()
    return render_template('search_results.html', results=results)

Conclusió

En aquest curs, hem construït una aplicació de blog totalment funcional utilitzant Flask. Hem cobert els conceptes bàsics de la configuració d'un entorn Flask, creant rutes i vistes, integrant una base de dades, gestionant l'entrada d'usuaris, implementant l'autenticació i afegint funcionalitats avançades com la paginació i la cerca. Finalment, hem desplegat la nostra aplicació a Heroku. Aquesta guia completa hauria de donar-te una base sòlida per construir i expandir les teves pròpies aplicacions amb Flask.
