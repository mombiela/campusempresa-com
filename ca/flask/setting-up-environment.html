Configurar el teu entorn de desenvolupament és el primer pas per començar a construir aplicacions web amb Flask. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu entorn per al desenvolupament amb Flask.

Prerequisits

Abans de començar, assegura't de tenir els següents prerequisits instal·lats a la teva màquina:

  • Python: Flask és un framework web de Python, així que necessites tenir Python instal·lat. Flask suporta Python 3.6 i versions superiors.
  • pip: L'instal·lador de paquets de Python, que s'utilitza per instal·lar Flask i altres dependències.

Instal·lació de Python

Si no tens Python instal·lat, segueix aquests passos:

  1. Descarrega Python: Visita el lloc web oficial de Python i descarrega l'última versió de Python.
  2. Instal·la Python: Executa l'instal·lador i segueix les instruccions. Assegura't de marcar l'opció per afegir Python al teu PATH del sistema durant la instal·lació.

Configuració d'un entorn virtual

Un entorn virtual és un directori autònom que conté una instal·lació de Python per a una versió específica de Python, a més d'un nombre de paquets addicionals. Utilitzar un entorn virtual assegura que les dependències del teu projecte estiguin aïllades d'altres projectes.

  1. Crea un entorn virtual:

    python -m venv venv

    Aquesta comanda crea un directori anomenat venv a la carpeta del teu projecte.

  2. Activa l'entorn virtual:

    • A Windows: 
      venv\Scripts\activate
    • A macOS i Linux: 
      source venv/bin/activate

  3. Desactiva l'entorn virtual: Per desactivar l'entorn virtual, simplement executa:

    deactivate

Instal·lació de Flask

Amb el teu entorn virtual activat, ara pots instal·lar Flask utilitzant pip:

pip install Flask

Per verificar la instal·lació, pots executar:

python -m flask --version

Creació de la teva primera aplicació Flask

Ara que Flask està instal·lat, creem una simple aplicació Flask per assegurar-nos que tot està configurat correctament.

  1. Crea un nou fitxer anomenat app.py al directori del teu projecte.

  2. Afegeix el següent codi a app.py:

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

  3. Executa l'aplicació Flask:

    python app.py

    Hauries de veure una sortida similar a aquesta:

    * Serving Flask app "app" (lazy loading)
* Environment: production
  WARNING: This is a development server. Do not use it in a production deployment.
  Use a production WSGI server instead.
* Debug mode: on
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

  4. Obre el teu navegador web i navega a http://127.0.0.1:5000/. Hauries de veure "Hello, World!" mostrat a la pàgina.

Resum

En aquesta secció, has après a:

  • Instal·lar Python i pip.
  • Configurar un entorn virtual per gestionar les dependències del teu projecte.
  • Instal·lar Flask utilitzant pip.
  • Crear i executar una simple aplicació Flask.

Amb el teu entorn de desenvolupament configurat, ara estàs preparat per aprofundir en Flask i començar a construir aplicacions web més complexes.
