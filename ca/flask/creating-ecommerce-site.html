Introducció

En aquest curs, aprendrem a crear un lloc de comerç electrònic completament funcional utilitzant Flask, un framework web lleuger per a Python. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat, cobrint tots els aspectes essencials del desenvolupament web amb Flask.

Configurant l'entorn

Abans de començar a construir el nostre lloc de comerç electrònic, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar Python: Assegura't que tens Python instal·lat a la teva màquina.
  • Instal·lar Flask: Utilitza pip per instal·lar Flask. 
    pip install Flask
  • Crear un directori de projecte: Configura un directori per al teu projecte. 
    mkdir ecommerce_site
cd ecommerce_site

Aplicació bàsica de Flask

Comencem amb una aplicació simple de Flask per entendre els conceptes bàsics.

  • Crear una aplicació bàsica de Flask: Crea un fitxer anomenat app.py.

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Benvingut al lloc de comerç electrònic!"

if __name__ == '__main__':
    app.run(debug=True)

  • Executar l'aplicació: Executa l'aplicació.

    python app.py

Plantilles i fitxers estàtics

Flask utilitza plantilles Jinja2 per renderitzar HTML. Crearem plantilles per al nostre lloc de comerç electrònic.

  • Crear un directori de plantilles: Crea un directori anomenat templates.

    mkdir templates

  • Crear una plantilla base: Crea un fitxer anomenat base.html al directori templates.

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{% block title %}Lloc de comerç electrònic{% endblock %}</title>
</head>
<body>
    <header>
        <h1>Lloc de comerç electrònic</h1>
    </header>
    <main>
        {% block content %}{% endblock %}
    </main>
</body>
</html>

  • Crear una plantilla d'inici: Crea un fitxer anomenat home.html al directori templates.

    {% extends "base.html" %}

{% block title %}Inici{% endblock %}

{% block content %}
<h2>Benvingut al lloc de comerç electrònic!</h2>
{% endblock %}

  • Actualitzar app.py per utilitzar plantilles:

    from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('home.html')

if __name__ == '__main__':
    app.run(debug=True)

Integració de bases de dades

Utilitzarem SQLite per a la nostra base de dades. Flask-SQLAlchemy és una extensió que simplifica la integració de bases de dades.

  • Instal·lar Flask-SQLAlchemy:

    pip install Flask-SQLAlchemy

  • Configurar la base de dades:

    from flask import Flask, render_template
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///ecommerce.db'
db = SQLAlchemy(app)

@app.route('/')
def home():
    return render_template('home.html')

if __name__ == '__main__':
    app.run(debug=True)

  • Crear models: Defineix models per a productes i usuaris.

    class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

class Product(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    price = db.Column(db.Float, nullable=False)

  • Crear la base de dades:

    from app import db
db.create_all()

Autenticació d'usuaris

Implementa la funcionalitat de registre i inici de sessió d'usuaris.

  • Instal·lar Flask-Login:

    pip install Flask-Login

  • Configurar l'autenticació d'usuaris:

    from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin, db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    password = db.Column(db.String(120), nullable=False)

@login_manager.user_loader
def load_user(user_id):
    return User.query.get(int(user_id))

  • Crear formularis de registre i inici de sessió:

    from flask_wtf import FlaskForm
from wtforms import StringField, PasswordField, SubmitField
from wtforms.validators import DataRequired, Email, EqualTo

class RegistrationForm(FlaskForm):
    username = StringField('Nom d'usuari', validators=[DataRequired()])
    email = StringField('Correu electrònic', validators=[DataRequired(), Email()])
    password = PasswordField('Contrasenya', validators=[DataRequired()])
    confirm_password = PasswordField('Confirma la contrasenya', validators=[DataRequired(), EqualTo('password')])
    submit = SubmitField('Registrar-se')

class LoginForm(FlaskForm):
    email = StringField('Correu electrònic', validators=[DataRequired(), Email()])
    password = PasswordField('Contrasenya', validators=[DataRequired()])
    submit = SubmitField('Iniciar sessió')

  • Crear rutes de registre i inici de sessió:

    from flask import Flask, render_template, redirect, url_for, flash
from flask_login import login_user, login_required, logout_user, current_user
from werkzeug.security import generate_password_hash, check_password_hash

@app.route('/register', methods=['GET', 'POST'])
def register():
    form = RegistrationForm()
    if form.validate_on_submit():
        hashed_password = generate_password_hash(form.password.data, method='sha256')
        new_user = User(username=form.username.data, email=form.email.data, password=hashed_password)
        db.session.add(new_user)
        db.session.commit()
        flash('El teu compte ha estat creat!', 'success')
        return redirect(url_for('login'))
    return render_template('register.html', form=form)

@app.route('/login', methods=['GET', 'POST'])
def login():
    form = LoginForm()
    if form.validate_on_submit():
        user = User.query.filter_by(email=form.email.data).first()
        if user and check_password_hash(user.password, form.password.data):
            login_user(user)
            return redirect(url_for('home'))
        else:
            flash('Inici de sessió fallit. Si us plau, comprova el correu electrònic i la contrasenya', 'danger')
    return render_template('login.html', form=form)

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return redirect(url_for('home'))

Gestió de productes

Implementa la funcionalitat per afegir, veure i gestionar productes.

  • Crear formulari de producte:

    class ProductForm(FlaskForm):
    name = StringField('Nom del producte', validators=[DataRequired()])
    price = StringField('Preu', validators=[DataRequired()])
    submit = SubmitField('Afegir producte')

  • Crear rutes de producte:

    @app.route('/product/new', methods=['GET', 'POST'])
@login_required
def new_product():
    form = ProductForm()
    if form.validate_on_submit():
        product = Product(name=form.name.data, price=form.price.data)
        db.session.add(product)
        db.session.commit()
        flash('El producte ha estat afegit!', 'success')
        return redirect(url_for('home'))
    return render_template('create_product.html', form=form)

@app.route('/products')
def products():
    products = Product.query.all()
    return render_template('products.html', products=products)
