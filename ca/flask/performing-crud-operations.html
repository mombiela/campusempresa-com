Introducció

En aquesta secció, explorarem com realitzar operacions CRUD (Create, Read, Update, Delete) utilitzant Flask. Les operacions CRUD són fonamentals per a les aplicacions web, ja que permeten als usuaris interactuar amb la base de dades. Cobrirem cada operació pas a pas, proporcionant exemples de codi i explicacions per assegurar-nos que entens com implementar-les en la teva aplicació Flask.

Configuració de l'entorn

Abans d'endinsar-nos en les operacions CRUD, configurem el nostre entorn Flask.

  1. Instal·la Flask i SQLAlchemy:

    pip install Flask SQLAlchemy

  2. Crea una aplicació bàsica de Flask:

    from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///test.db'
db = SQLAlchemy(app)

if __name__ == '__main__':
    app.run(debug=True)

Creació del model de base de dades

Defineix un model per a la base de dades. Per a aquest exemple, crearem un model senzill User.

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.name}>'

Executa les següents comandes per crear la base de dades:

from your_application import db
db.create_all()

Operació de creació

Per afegir nous usuaris a la base de dades, necessitem crear una ruta que gestioni la creació de noves instàncies de User.

@app.route('/user', methods=['POST'])
def create_user():
    name = request.json['name']
    email = request.json['email']
    new_user = User(name=name, email=email)

    try:
        db.session.add(new_user)
        db.session.commit()
        return {"message": "User created successfully"}, 201
    except:
        return {"message": "There was an issue creating the user"}, 500

Operació de lectura

Per llegir o recuperar usuaris de la base de dades, podem crear rutes per obtenir tots els usuaris o un usuari específic per ID.

  • Obtenir tots els usuaris:

    @app.route('/users', methods=['GET'])
def get_users():
    users = User.query.all()
    return {"users": [{"id": user.id, "name": user.name, "email": user.email} for user in users]}

  • Obtenir usuari per ID:

    @app.route('/user/<int:id>', methods=['GET'])
def get_user(id):
    user = User.query.get_or_404(id)
    return {"id": user.id, "name": user.name, "email": user.email}

Operació d'actualització

Per actualitzar un usuari existent, necessitem crear una ruta que gestioni la lògica d'actualització.

@app.route('/user/<int:id>', methods=['PUT'])
def update_user(id):
    user = User.query.get_or_404(id)
    name = request.json['name']
    email = request.json['email']

    try:
        user.name = name
        user.email = email
        db.session.commit()
        return {"message": "User updated successfully"}
    except:
        return {"message": "There was an issue updating the user"}, 500

Operació d'eliminació

Per eliminar un usuari de la base de dades, necessitem crear una ruta que gestioni l'eliminació.

@app.route('/user/<int:id>', methods=['DELETE'])
def delete_user(id):
    user = User.query.get_or_404(id)

    try:
        db.session.delete(user)
        db.session.commit()
        return {"message": "User deleted successfully"}
    except:
        return {"message": "There was an issue deleting the user"}, 500

Conclusió

En aquesta secció, hem cobert com realitzar operacions CRUD en una aplicació Flask utilitzant SQLAlchemy. Hem après com:

  • Configurar l'entorn Flask i la base de dades.
  • Crear un model de base de dades.
  • Implementar rutes per crear, llegir, actualitzar i eliminar registres.

Entendre les operacions CRUD és essencial per construir aplicacions web dinàmiques. Practica aquests conceptes per consolidar la teva comprensió i aplicar-los als teus projectes.
