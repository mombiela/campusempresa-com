Les carregues de fitxers són una característica comuna en les aplicacions web, permetent als usuaris carregar fitxers com imatges, documents i altres mitjans. Flask proporciona una manera senzilla de gestionar les carregues de fitxers. Aquest tema et guiarà des dels conceptes bàsics fins a la gestió més avançada de les carregues de fitxers en Flask.

Introducció a les Carregues de Fitxers

  • Què són les carregues de fitxers?
    • Les carregues de fitxers permeten als usuaris enviar fitxers des de la seva màquina local a un servidor.
  • Per què són importants les carregues de fitxers?
    • Permeten funcionalitats com l'actualització de fotos de perfil, la presentació de documents i el compartir mitjans.

Configuració de la teva Aplicació Flask

Abans de gestionar les carregues de fitxers, assegura't de tenir Flask instal·lat i configurat. Si no, pots instal·lar Flask utilitzant pip:

pip install Flask

Crea una aplicació bàsica de Flask:

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Gestió de Carregues de Fitxers Bàsiques

Configuració de l'Aplicació

Per gestionar les carregues de fitxers, necessites configurar la teva aplicació Flask per especificar la carpeta de càrrega i les extensions de fitxer permeses.

import os

app.config['UPLOAD_FOLDER'] = 'uploads/'
app.config['ALLOWED_EXTENSIONS'] = {'png', 'jpg', 'jpeg', 'gif'}

def allowed_file(filename):
    return '.' in filename and filename.rsplit('.', 1)[1].lower() in app.config['ALLOWED_EXTENSIONS']

Creació del Formulari de Càrrega

Crea un formulari HTML per permetre als usuaris carregar fitxers. Desa això com templates/index.html:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Carrega de Fitxers</title>
  </head>
  <body>
    <h1>Carrega un Fitxer</h1>
    <form method="post" enctype="multipart/form-data" action="/upload">
      <input type="file" name="file">
      <input type="submit" value="Carregar">
    </form>
  </body>
</html>

Gestió de la Càrrega de Fitxers

Afegeix una ruta per gestionar la càrrega de fitxers en la teva aplicació Flask:

@app.route('/upload', methods=['POST'])
def upload_file():
    if 'file' not in request.files:
        return 'No hi ha cap fitxer'
    file = request.files['file']
    if file.filename == '':
        return 'No s\'ha seleccionat cap fitxer'
    if file and allowed_file(file.filename):
        filename = secure_filename(file.filename)
        file.save(os.path.join(app.config['UPLOAD_FOLDER'], filename))
        return 'Fitxer carregat correctament'
    return 'Tipus de fitxer no permès'

Explicació

  • request.files: Accedeix al fitxer carregat.
  • secure_filename: Assegura que el nom del fitxer és segur per utilitzar.
  • file.save: Desa el fitxer a la carpeta de càrrega especificada.

Gestió Avançada de Carregues de Fitxers

Validació de la Mida del Fitxer

Per evitar que els usuaris carreguin fitxers excessivament grans, pots validar la mida del fitxer:

app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024  # 16 MB

@app.errorhandler(413)
def request_entity_too_large(error):
    return 'Fitxer massa gran', 413

Emmagatzematge de Fitxers en una Base de Dades

En lloc de desar els fitxers al sistema de fitxers, pots desar-los en una base de dades. Aquí tens un exemple utilitzant SQLAlchemy:

from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///files.db'
db = SQLAlchemy(app)

class File(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    data = db.Column(db.LargeBinary, nullable=False)

@app.route('/upload', methods=['POST'])
def upload_file():
    file = request.files['file']
    if file and allowed_file(file.filename):
        new_file = File(name=secure_filename(file.filename), data=file.read())
        db.session.add(new_file)
        db.session.commit()
        return 'Fitxer carregat correctament'
    return 'Tipus de fitxer no permès'

Servir Fitxers Carregats

Per servir fitxers carregats, pots crear una ruta per enviar el fitxer al client:

from flask import send_from_directory

@app.route('/uploads/<filename>')
def uploaded_file(filename):
    return send_from_directory(app.config['UPLOAD_FOLDER'], filename)

Conclusió

Gestionar les carregues de fitxers en Flask implica configurar la teva aplicació, crear un formulari de càrrega i escriure rutes per gestionar el procés de càrrega de fitxers. Les tècniques avançades inclouen la validació de la mida del fitxer i l'emmagatzematge de fitxers en una base de dades. Amb aquestes habilitats, pots implementar una funcionalitat robusta de càrrega de fitxers en les teves aplicacions Flask.

Seguint aquesta guia, ara hauries de tenir una comprensió sòlida de com gestionar les carregues de fitxers en Flask, des de configuracions bàsiques fins a implementacions més avançades.
