Introducció

Flask és un micro framework web per a Python, dissenyat per ser simple però potent. Comprendre l'estructura d'una aplicació Flask és crucial per desenvolupar aplicacions web escalables i mantenibles. Aquesta secció et guiarà a través de l'estructura típica d'una aplicació Flask, explicant cada component i el seu propòsit.

Conceptes Clau

  • Aplicació Flask: El nucli de la teva aplicació web.
  • Rutes: Patrons d'URL que es mapegen a funcions específiques en la teva aplicació.
  • Plantilles: Fitxers HTML que són renderitzats per Flask per generar pàgines web dinàmiques.
  • Fitxers Estàtics: Fitxers com CSS, JavaScript i imatges que es serveixen directament al client.
  • Configuració: Configuracions que controlen el comportament de la teva aplicació Flask.

Estructura Bàsica d'una Aplicació Flask

Una aplicació Flask bàsica típicament consisteix en els següents fitxers i directoris:

my_flask_app/
│
├── app.py
├── templates/
│   └── index.html
├── static/
│   ├── css/
│   ├── js/
│   └── images/
└── config.py

app.py

Aquest és el fitxer principal que inicialitza l'aplicació Flask i defineix les rutes.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
  • Inicialització de Flask: app = Flask(__name__) inicialitza l'aplicació Flask.
  • Definició de Rutes: @app.route('/') defineix una ruta per a la pàgina principal.
  • Renderitzar Plantilla: render_template('index.html') renderitza la plantilla index.html.

templates/index.html

Aquest és un fitxer HTML que serà renderitzat per Flask.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Home</title>
</head>
<body>
    <h1>Welcome to My Flask App!</h1>
</body>
</html>

static/

Aquest directori conté fitxers estàtics com CSS, JavaScript i imatges. Aquests fitxers es serveixen directament al client.

  • css/: Conté fitxers CSS.
  • js/: Conté fitxers JavaScript.
  • images/: Conté fitxers d'imatges.

config.py

Aquest fitxer conté configuracions per a la teva aplicació Flask.

import os

class Config:
    SECRET_KEY = os.environ.get('SECRET_KEY') or 'you-will-never-guess'
    DEBUG = True

Estructura Avançada d'una Aplicació Flask

A mesura que la teva aplicació creix, potser voldràs organitzar-la en una estructura més modular. Aquí tens un exemple d'una estructura més avançada:

my_flask_app/
│
├── app/
│   ├── __init__.py
│   ├── routes.py
│   ├── models.py
│   ├── templates/
│   │   └── index.html
│   └── static/
│       ├── css/
│       ├── js/
│       └── images/
├── config.py
└── run.py

app/__init__.py

Aquest fitxer inicialitza l'aplicació Flask i configura la configuració.

from flask import Flask
from config import Config

app = Flask(__name__)
app.config.from_object(Config)

from app import routes

app/routes.py

Aquest fitxer defineix les rutes per a la teva aplicació.

from app import app
from flask import render_template

@app.route('/')
def home():
    return render_template('index.html')

app/models.py

Aquest fitxer conté els models de base de dades per a la teva aplicació. (Exemple amb SQLAlchemy)

from app import app
from flask_sqlalchemy import SQLAlchemy

db = SQLAlchemy(app)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(64), index=True, unique=True)
    email = db.Column(db.String(120), index=True, unique=True)

    def __repr__(self):
        return f'<User {self.username}>'

run.py

Aquest fitxer s'utilitza per executar l'aplicació Flask.

from app import app

if __name__ == '__main__':
    app.run(debug=True)

Conclusió

Comprendre l'estructura d'una aplicació Flask és essencial per construir aplicacions web eficients i mantenibles. Començar amb una estructura bàsica i gradualment moure's cap a una estructura més modular i avançada t'ajudarà a gestionar millor el teu codi a mesura que la teva aplicació creix. Organitzant correctament les teves rutes, plantilles, fitxers estàtics i configuracions, pots crear una aplicació Flask escalable i mantenible.
