Introducció

En aquesta secció, explorarem com utilitzar materials i textures a Unreal Engine. Aquest tema és essencial per crear entorns i personatges visualment atractius en el teu joc. Començarem amb els conceptes bàsics i gradualment passarem a conceptes més avançats.

Conceptes Bàsics de Materials i Textures

Què són els Materials i les Textures?

  • Materials: Defineixen les propietats de la superfície d'un objecte, incloent color, brillantor i textura.
  • Textures: Imatges aplicades a la superfície d'un material per donar-li detall i realisme.

Crear un Material Bàsic

  1. Obrir l'Editor de Materials:
    • Fes clic dret al Content Browser.
    • Selecciona Create Basic Asset > Material.
    • Posa un nom al teu material (per exemple, M_BasicMaterial).
  2. Editar el Material:
    • Fes doble clic al material per obrir l'Editor de Materials.
    • Veureu una interfície basada en nodes on podeu definir les propietats del vostre material.

Aplicar una Textura a un Material

  1. Importar una Textura:
    • Fes clic dret al Content Browser.
    • Selecciona Import to /Game/... i tria un fitxer d'imatge (per exemple, T_BrickTexture.jpg).
  2. Assignar la Textura al Material:
    • A l'Editor de Materials, arrossega la textura des del Content Browser fins al gràfic.
    • Connecta el node de la textura a l'entrada Base Color del node Material.
// Exemple d'una configuració bàsica de material a Unreal Engine
TextureSampleParameter2D(Texture) -> BaseColor(Material)

Aplicar el Material a un Objecte

  1. Seleccionar un Objecte:
    • Fes clic en un objecte a la teva escena.
  2. Assignar el Material:
    • Al panell de Detalls, troba la secció Materials.
    • Fes clic al desplegable i selecciona el teu material (per exemple, M_BasicMaterial).

Tècniques Intermèdies

Ús de Mapes Normals

  • Mapes Normals: Afegeixen detall a una superfície sense augmentar el recompte de polígons.
  1. Importar un Mapa Normal:
    • Importa una textura de mapa normal (per exemple, T_BrickNormal.jpg).
  2. Assignar el Mapa Normal:
    • A l'Editor de Materials, arrossega la textura del mapa normal fins al gràfic.
    • Connecta el node del mapa normal a l'entrada Normal del node Material.
// Exemple d'un material amb un mapa normal
TextureSampleParameter2D(BaseColorTexture) -> BaseColor(Material)
TextureSampleParameter2D(NormalMapTexture) -> Normal(Material)

Mapes d'Especularitat i Rugositat

  • Mapes d'Especularitat: Controlen la brillantor d'una superfície.
  • Mapes de Rugositat: Defineixen com de rugosa o suau apareix una superfície.
  1. Importar Mapes d'Especularitat i Rugositat:
    • Importa les textures respectives (per exemple, T_BrickSpecular.jpg i T_BrickRoughness.jpg).
  2. Assignar els Mapes:
    • Connecta el mapa d'especularitat a l'entrada Specular.
    • Connecta el mapa de rugositat a l'entrada Roughness.
// Exemple d'un material amb mapes d'especularitat i rugositat
TextureSampleParameter2D(BaseColorTexture) -> BaseColor(Material)
TextureSampleParameter2D(NormalMapTexture) -> Normal(Material)
TextureSampleParameter2D(SpecularMapTexture) -> Specular(Material)
TextureSampleParameter2D(RoughnessMapTexture) -> Roughness(Material)

Tècniques Avançades

Ús d'Instàncies de Material

  • Instàncies de Material: Et permeten crear variacions d'un material sense duplicar el material base.
  1. Crear una Instància de Material:
    • Fes clic dret sobre el teu material (per exemple, M_BasicMaterial).
    • Selecciona Create Material Instance.
  2. Editar la Instància de Material:
    • Fes doble clic a la instància de material per obrir l'Editor d'Instàncies.
    • Ajusta paràmetres com color, rugositat i valors d'especularitat.

Canvis Dinàmics de Material

  • Materials Dinàmics: Canvien les propietats del material en temps d'execució utilitzant Blueprints o C++.
  1. Configuració de Blueprint:
    • Crea un Blueprint i afegeix un component Static Mesh.
    • Al gràfic d'esdeveniments, utilitza nodes com Create Dynamic Material Instance i Set Vector Parameter Value.
// Exemple de nodes de Blueprint per a canvis dinàmics de material
Event BeginPlay
-> Create Dynamic Material Instance (Target: StaticMeshComponent, Material: M_BasicMaterial)
-> Set Vector Parameter Value (Parameter Name: "BaseColor", Value: (R=1,G=0,B=0))

Conclusió

Entendre i utilitzar materials i textures a Unreal Engine és crucial per crear jocs visualment impressionants. Dominant la creació bàsica de materials, aplicant textures i utilitzant tècniques avançades com instàncies de material i canvis dinàmics, pots millorar significativament la qualitat visual dels teus projectes. Practica aquests conceptes per convertir-te en un expert en la gestió de materials i textures a Unreal Engine.
