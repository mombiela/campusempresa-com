Introducció

Les proves i la depuració són habilitats crucials per a qualsevol desenvolupador de jocs. A Unreal Engine, aquests processos asseguren que el teu joc funcioni sense problemes i estigui lliure d'errors. Aquesta secció et guiarà a través dels conceptes bàsics de les proves i la depuració, des del nivell principiant fins a l'avançat.

Nivell Principiant

Comprendre els Conceptes Bàsics

  • Proves: El procés d'executar el teu joc per trobar i corregir problemes.
  • Depuració: El procés d'identificar, analitzar i eliminar errors o fallades en el teu codi.

Eines i Funcions

  • Play In Editor (PIE): Et permet jugar al teu joc directament a l'editor d'Unreal Engine.
  • Output Log: Mostra missatges, advertències i errors durant el joc.
  • Breakpoints: Pausa el joc en punts específics per inspeccionar variables i l'estat del joc.

Exemple Senzill de Depuració

// Exemple d'un missatge de depuració senzill a Unreal Engine utilitzant C++
UE_LOG(LogTemp, Warning, TEXT("Aquest és un missatge de depuració"));
  • Explicació: UE_LOG és una macro utilitzada per imprimir missatges al Output Log. LogTemp és una categoria de registre temporal, Warning és el nivell de registre, i TEXT("Aquest és un missatge de depuració") és el missatge.

Exercicis

  1. Exercici 1: Crea un projecte senzill i utilitza UE_LOG per imprimir un missatge quan el joc comenci.
  2. Exercici 2: Estableix un breakpoint en el teu codi i inspecciona els valors de diferents variables.

Nivell Intermedi

Tècniques Avançades de Depuració

  • Visual Logger: Captura i visualitza dades del joc.
  • Eines de Profiling: Ajuda a identificar colls d'ampolla de rendiment.
  • Proves d'Automatització: Escriure proves automatitzades per assegurar la funcionalitat del joc.

Utilitzant el Visual Logger

// Exemple d'ús del Visual Logger
UE_VLOG(this, LogTemp, Log, TEXT("Posició del jugador: %s"), *GetActorLocation().ToString());
  • Explicació: UE_VLOG és similar a UE_LOG però específicament per al Visual Logger. GetActorLocation().ToString() converteix la ubicació de l'actor en una cadena.

Exemple de Prova d'Automatització

// Exemple d'una prova d'automatització senzilla
IMPLEMENT_SIMPLE_AUTOMATION_TEST(FMyTest, "MyGame.MyTest", EAutomationTestFlags::EditorContext | EAutomationTestFlags::EngineFilter)

bool FMyTest::RunTest(const FString& Parameters)
{
    TestEqual("Comprova si 2 + 2 és igual a 4", 2 + 2, 4);
    return true;
}
  • Explicació: IMPLEMENT_SIMPLE_AUTOMATION_TEST crea una nova prova. TestEqual comprova si l'expressió 2 + 2 és igual a 4.

Exercicis

  1. Exercici 3: Utilitza el Visual Logger per registrar la posició del jugador cada segon.
  2. Exercici 4: Escriu una prova d'automatització per comprovar si un objecte específic del joc està present en el nivell.

Nivell Avançat

Escenaris Complexos de Depuració

  • Depuració de Memòria: Identificar i corregir fuites de memòria.
  • Problemes de Multithreading: Depurar problemes relacionats amb l'execució concurrent.
  • Depuració de Xarxa: Assegurar experiències multijugador suaus.

Exemple de Depuració de Memòria

// Exemple d'ús d'un profiler de memòria
#include "Runtime/Core/Public/HAL/MemoryMisc.h"

void CheckMemoryUsage()
{
    FPlatformMemoryStats MemoryStats = FPlatformMemory::GetStats();
    UE_LOG(LogTemp, Warning, TEXT("Memòria Física Total: %llu"), MemoryStats.TotalPhysical);
    UE_LOG(LogTemp, Warning, TEXT("Memòria Física Utilitzada: %llu"), MemoryStats.UsedPhysical);
}
  • Explicació: FPlatformMemory::GetStats() recupera estadístiques de memòria. MemoryStats.TotalPhysical i MemoryStats.UsedPhysical donen la memòria física total i utilitzada, respectivament.

Exemple de Depuració de Xarxa

// Exemple de depuració de xarxa utilitzant NetDebug
void AMyPlayerController::BeginPlay()
{
    Super::BeginPlay();
    if (IsLocalController())
    {
        GetWorld()->Exec(GetWorld(), TEXT("NetDebug"));
    }
}
  • Explicació: La comanda NetDebug proporciona estadístiques detallades de xarxa i ajuda a depurar problemes de xarxa.

Exercicis

  1. Exercici 5: Utilitza eines de profiling de memòria per identificar i corregir una fuita de memòria en el teu projecte.
  2. Exercici 6: Depura un joc multijugador per assegurar un rendiment de xarxa suau.

Conclusió

Les proves i la depuració són essencials per crear jocs d'alta qualitat a Unreal Engine. En dominar aquestes habilitats, pots assegurar que el teu joc sigui funcional i eficient. Des del registre bàsic i els breakpoints fins a la depuració avançada de memòria i xarxa, aquestes tècniques t'ajudaran a identificar i corregir problemes de manera eficient. Continua practicant i explorant les eines disponibles a Unreal Engine per convertir-te en un expert en proves i depuració.
