Introducció

En aquesta secció, aprofundirem en les eines avançades de terreny i paisatge disponibles a Unreal Engine. Aquestes eines et permeten crear entorns altament detallats i realistes per als teus jocs o simulacions. Al final d'aquesta secció, seràs capaç d'utilitzar aquestes eines per millorar significativament els teus paisatges i terrenys.

Conceptes clau

  • Mode Paisatge: El mode principal per crear i editar terrenys a Unreal Engine.
  • Eines d'Esculpir: Eines utilitzades per donar forma al terreny.
  • Eines de Pintura: Eines utilitzades per aplicar textures i materials al terreny.
  • Capes de Paisatge: Capes que permeten l'edició no destructiva dels paisatges.
  • Mapes d'Alçada: Imatges en escala de grisos utilitzades per definir l'elevació del terreny.
  • Generació Procedural: Tècniques per generar automàticament característiques del terreny.

Mode Paisatge

El Mode Paisatge és la interfície principal per crear i modificar terrenys a Unreal Engine. Pots accedir-hi fent clic al botó "Modes" i seleccionant "Paisatge".

Crear un nou paisatge

Per crear un nou paisatge:

  1. Obre el panell "Modes" i selecciona "Paisatge".
  2. Configura els paràmetres del paisatge, com ara la mida i la resolució.
  3. Fes clic a "Crear" per generar el paisatge.
// Exemple: Crear un paisatge bàsic
ULandscapeComponent* Landscape = NewObject<ULandscapeComponent>(this, ULandscapeComponent::StaticClass());
Landscape->SetWorldLocation(FVector(0, 0, 0));
Landscape->SetRelativeScale3D(FVector(100, 100, 100));
Landscape->RegisterComponent();

Eines d'Esculpir

Les eines d'esculpir s'utilitzen per modificar la forma del terreny. Algunes de les eines d'esculpir clau inclouen:

  • Esculpir: Eleva o abaixa el terreny.
  • Suavitzar: Suavitza les àrees rugoses.
  • Aplanar: Aplana el terreny a una alçada especificada.
  • Rampa: Crea una rampa entre dos punts.

Exemple: Utilitzar l'eina d'Esculpir

Per utilitzar l'eina d'Esculpir:

  1. Selecciona l'eina "Esculpir" del panell del Mode Paisatge.
  2. Ajusta la mida i la força del pinzell.
  3. Fes clic i arrossega sobre el terreny per elevar-lo o abaixar-lo.
// Exemple: Aplicar una operació d'esculpir
Landscape->Sculpt(FVector2D(100, 100), 50.0f, 1.0f);

Eines de Pintura

Les eines de pintura s'utilitzen per aplicar textures i materials al terreny. Les eines de pintura clau inclouen:

  • Pintar: Aplica un material seleccionat al terreny.
  • Esborrar: Elimina el material aplicat.
  • Suavitzar: Barreja diferents materials.

Exemple: Utilitzar l'eina de Pintura

Per utilitzar l'eina de Pintura:

  1. Selecciona l'eina "Pintar" del panell del Mode Paisatge.
  2. Tria un material del navegador de contingut.
  3. Ajusta la mida i la força del pinzell.
  4. Fes clic i arrossega sobre el terreny per aplicar el material.
// Exemple: Aplicar un material al paisatge
Landscape->Paint(FVector2D(100, 100), MyMaterial, 1.0f);

Capes de Paisatge

Les capes de paisatge permeten l'edició no destructiva dels paisatges. Pots crear múltiples capes i modificar-les independentment.

Exemple: Afegir una nova capa

Per afegir una nova capa:

  1. Obre el panell "Paisatge".
  2. Fes clic a la pestanya "Capes".
  3. Fes clic a "Afegir Capa" i posa-li un nom.
// Exemple: Afegir una nova capa de paisatge
Landscape->AddLayer("NewLayer");

Mapes d'Alçada

Els mapes d'alçada són imatges en escala de grisos on la brillantor de cada píxel representa l'alçada del terreny en aquell punt. Sovint s'utilitzen per importar dades de terreny de fonts externes.

Exemple: Importar un mapa d'alçada

Per importar un mapa d'alçada:

  1. Obre el panell "Paisatge".
  2. Fes clic al botó "Importar des d'Arxiu".
  3. Selecciona el fitxer del mapa d'alçada i ajusta els paràmetres d'importació.
  4. Fes clic a "Importar" per generar el terreny.
// Exemple: Importar un mapa d'alçada
Landscape->ImportHeightmap("path/to/heightmap.png");

Generació Procedural

Les tècniques de generació procedural es poden utilitzar per crear automàticament característiques del terreny, com ara muntanyes, valls i rius.

Exemple: Generació de terreny procedural

Per generar terreny proceduralment:

  1. Crea un blueprint o una classe C++ per a la generació procedural.
  2. Implementa la lògica de generació.
// Exemple: Generació de terreny procedural
void GenerateTerrain(ULandscapeComponent* Landscape)
{
    for (int x = 0; x < Landscape->GetWidth(); x++)
    {
        for (int y = 0; y < Landscape->GetHeight(); y++)
        {
            float Height = FMath::PerlinNoise2D(FVector2D(x, y) * 0.1f) * 100.0f;
            Landscape->SetHeightAt(x, y, Height);
        }
    }
}

Conclusió

En aquesta secció, hem cobert les eines avançades de terreny i paisatge disponibles a Unreal Engine. Has après a crear i modificar paisatges utilitzant eines d'esculpir i pintar, gestionar capes de paisatge, importar mapes d'alçada i generar terreny proceduralment. Aquestes eines són essencials per crear entorns detallats i realistes en els teus projectes. Continua experimentant amb aquestes eines per dominar la creació de terrenys i paisatges a Unreal Engine.
