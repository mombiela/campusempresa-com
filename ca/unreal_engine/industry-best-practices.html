Millors Pràctiques de la Indústria

Introducció

En aquesta secció, cobrim les millors pràctiques de la indústria per treballar amb Unreal Engine. Aquestes pràctiques són essencials per crear projectes eficients, mantenibles i escalables. Tant si ets un principiant com un usuari avançat, seguir aquestes directrius t'ajudarà a produir treballs d'alta qualitat i a col·laborar eficaçment amb altres.

Organització del Projecte

  • Estructura de Carpetes: Organitza els fitxers del teu projecte en una estructura de carpetes clara i lògica. Això facilita trobar actius i gestionar el teu projecte.
    • Exemple: 
      /Content
  /Blueprints
  /Materials
  /Meshes
  /Textures
  /UI
  • Convencions de Nomenclatura: Utilitza convencions de nomenclatura consistents per als teus actius. Això ajuda a identificar ràpidament el tipus i la finalitat de cada actiu.
    • Exemple: 
      BP_PlayerCharacter
M_Wood
SM_Tree
T_Grass

Estàndards de Codificació

  • Comentaris i Documentació: Sempre comenta el teu codi i proporciona documentació. Això és crucial per mantenir i entendre la base de codi.

    • Exemple: 
      // Aquesta funció gestiona el moviment del jugador
void APlayerCharacter::MoveForward(float Value)
{
    if (Controller && Value != 0.0f)
    {
        // Descobrir quina direcció és endavant
        const FRotator Rotation = Controller->GetControlRotation();
        const FRotator YawRotation(0, Rotation.Yaw, 0);

        // Obtenir vector endavant
        const FVector Direction = FRotationMatrix(YawRotation).GetUnitAxis(EAxis::X);
        AddMovementInput(Direction, Value);
    }
}

  • Estil de Codificació Consistent: Segueix un estil de codificació consistent al llarg del teu projecte. Això inclou la indentació, la col·locació de claus i les convencions de nomenclatura per a variables i funcions.

    • Exemple: 
      // Bé
void APlayerCharacter::Jump()
{
    Super::Jump();
}

// Malament
void APlayerCharacter::jump() {
    super::jump();
}

Optimització del Rendiment

  • Nivell de Detall (LOD): Utilitza LODs per als teus malles per reduir el nombre de polígons renderitzats a distància.
  • Transmissió de Textures: Activa la transmissió de textures per gestionar l'ús de memòria de manera eficient.
  • Eines de Perfilatge: Utilitza les eines de perfilatge d'Unreal Engine per identificar i abordar colls d'ampolla de rendiment.
    • Exemple:
      • Stat Unit: Mostra el temps de fotograma, el temps del fil del joc, el temps del fil de dibuix i el temps de la GPU.
      • Stat FPS: Mostra els fotogrames per segon.

Gestió d'Actius

  • Control de Versions: Utilitza sistemes de control de versions com Git o Perforce per gestionar els fitxers del teu projecte. Això és essencial per a la col·laboració i la versió.
  • Referenciació d'Actius: Evita referències dures als actius quan sigui possible. Utilitza referències suaus o punters d'actius per reduir l'ús de memòria i els temps de càrrega.

Millors Pràctiques de Blueprints

  • Blueprints Modulars: Crea Blueprints modulars que es puguin reutilitzar en diferents parts del teu projecte.
  • Arquitectura Basada en Esdeveniments: Utilitza esdeveniments i delegats per gestionar les interaccions entre Blueprints.
    • Exemple: 
      // Dispatcher d'Esdeveniments en Blueprint
Event OnPlayerHealthChanged

Proves i Depuració

  • Proves Automatitzades: Implementa proves automatitzades per assegurar-te que el teu joc funcioni correctament.
  • Eines de Depuració: Utilitza les eines de depuració d'Unreal Engine per solucionar problemes.
    • Exemple:
      • Breakpoints: Estableix breakpoints en el teu codi per pausar l'execució i inspeccionar variables.
      • Sortida de Registre: Utilitza UE_LOG per imprimir missatges al registre de sortida per a finalitats de depuració. 
        UE_LOG(LogTemp, Warning, TEXT("Player Health: %f"), PlayerHealth);

Conclusió

Seguir les millors pràctiques de la indústria en el desenvolupament d'Unreal Engine és crucial per crear projectes d'alta qualitat, mantenibles i eficients. Organitzant el teu projecte, seguint estàndards de codificació, optimitzant el rendiment, gestionant actius de manera efectiva i utilitzant eines de proves i depuració, pots assegurar-te que el teu treball compleixi els estàndards professionals i estigui preparat per a la col·laboració i l'escalabilitat.
