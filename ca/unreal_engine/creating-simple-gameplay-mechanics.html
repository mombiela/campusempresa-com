Introducció

En aquesta secció, explorarem com crear mecàniques de joc simples utilitzant Unreal Engine. Això inclourà entendre els conceptes bàsics dels Blueprints, crear controls per al jugador i implementar la lògica bàsica del joc. Al final d'aquesta secció, tindràs una comprensió fonamental de com crear elements de joc interactius.

Entendre els Blueprints

Els Blueprints són el sistema de script visual d'Unreal Engine, que et permet crear mecàniques de joc sense escriure codi. Això el fa accessible per a principiants, tot i que és prou potent per a usuaris avançats.

  • Conceptes Clau:
    • Nodes: Els blocs de construcció dels Blueprints, que representen funcions, esdeveniments i variables.
    • Pins: Connecten nodes per passar dades i controlar el flux.
    • Gràfics: L'espai de treball on organitzes i connectes nodes.

Exemple: Crear un Blueprint Simple

  1. Crear una Nova Classe de Blueprint:

    • Fes clic dret al Content Browser.
    • Selecciona Blueprint Class.
    • Tria Actor com a classe pare.
    • Anomena'l MySimpleActor.

  2. Afegir un Component de Malla Estàtica:

    • Obre MySimpleActor.
    • Al panell de Components, fes clic a Add Component i selecciona Static Mesh.
    • Assigna una malla al component Static Mesh.

  3. Afegir una Lògica de Moviment Simple:

    • Al Event Graph, fes clic dret i afegeix un node Event Tick.
    • Afegeix un node Add Actor Local Offset.
    • Connecta el node Event Tick al node Add Actor Local Offset.
    • Configura el Delta Location a (0, 0, 1) per moure l'actor cap amunt cada fotograma.
Event Tick -> Add Actor Local Offset (Delta Location: 0, 0, 1)

Crear Controls per al Jugador

Els controls per al jugador són essencials per a un joc interactiu. Unreal Engine proporciona diverses maneres de gestionar l'entrada del jugador, incloent l'ús de Blueprints i C++.

  • Conceptes Clau:
    • Mapatges d'Entrada: Defineixen com les entrades del jugador (teclat, ratolí, gamepad) es mapen a accions i eixos.
    • Player Controller: Gestiona l'entrada del jugador i controla el peó.

Exemple: Configurar l'Entrada del Jugador

  1. Configurar Mapatges d'Entrada:

    • Ves a Edit -> Project Settings -> Input.
    • A Bindings, afegeix un Action Mapping anomenat Jump i assigna la tecla Space Bar.

  2. Crear un Blueprint de Personatge del Jugador:

    • Fes clic dret al Content Browser.
    • Selecciona Blueprint Class.
    • Tria Character com a classe pare.
    • Anomena'l MyPlayerCharacter.

  3. Implementar la Lògica de Salt:

    • Obre MyPlayerCharacter.
    • Al Event Graph, fes clic dret i afegeix un node InputAction Jump.
    • Afegeix un node Jump i connecta'l al pin Pressed del node InputAction Jump.
InputAction Jump (Pressed) -> Jump

Implementar la Lògica Bàsica del Joc

La lògica del joc defineix les regles i mecàniques del teu joc. Això pot incloure puntuació, sistemes de salut i condicions de victòria/derrota.

  • Conceptes Clau:
    • Variables: Emmagatzemen dades com la salut del jugador, la puntuació, etc.
    • Funcions: Encapsulen lògica reutilitzable.
    • Esdeveniments: Desencadenen accions basades en condicions del joc.

Exemple: Crear un Sistema de Salut

  1. Afegir una Variable de Salut:

    • Obre MyPlayerCharacter.
    • Al panell de Variables, fes clic a + per afegir una nova variable.
    • Anomena-la Health i configura el seu tipus a Float.

  2. Implementar la Lògica de Dany:

    • Al Event Graph, afegeix un esdeveniment personalitzat anomenat TakeDamage.
    • Afegeix un node Float - Float per restar el dany de la salut.
    • Afegeix un node Set Health per actualitzar la variable de salut.
Event TakeDamage (Damage Amount: Float) -> Float - Float (Health, Damage Amount) -> Set Health
  1. Comprovar la Mort:
    • Després de configurar la salut, afegeix un node Branch per comprovar si la salut és menor o igual a zero.
    • Si és cert, desencadena un esdeveniment o lògica de Death.
Set Health -> Branch (Condition: Health <= 0) -> [True] Death Logic

Conclusió

En aquest tema, hem cobert els conceptes bàsics de la creació de mecàniques de joc simples a Unreal Engine. Hem explorat els Blueprints, els controls per al jugador i la lògica bàsica del joc. Entenent aquests elements fonamentals, pots començar a construir jocs més complexos i interactius. Continua experimentant amb diferents nodes i lògica per millorar les teves habilitats i crear experiències de joc úniques.
