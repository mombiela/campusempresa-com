Els Blueprints són un potent sistema de guió visual a Unreal Engine que et permet crear elements de joc sense escriure ni una sola línia de codi. Aquest tema et guiarà des dels conceptes bàsics fins als més avançats, assegurant que tinguis una comprensió sòlida de com utilitzar els Blueprints de manera efectiva.

Introducció als Blueprints

Els Blueprints són una manera visual de crear mecàniques de joc, interaccions i més a Unreal Engine. Es basen en el mateix sistema subjacent que el C++, però proporcionen una interfície més accessible per a dissenyadors i artistes.

  • Guió Visual: Els Blueprints utilitzen una interfície basada en nodes per crear lògica.
  • Tipus de Blueprints: Hi ha diversos tipus de Blueprints, incloent-hi Level Blueprints, Class Blueprints i Function Blueprints.
  • Avantatges: Fàcil d'entendre, iteració ràpida i eines de depuració potents.

Creant el teu Primer Blueprint

Configurant el Projecte

  1. Obre Unreal Engine: Comença llançant Unreal Engine i creant un nou projecte.
  2. Tria una Plantilla: Selecciona una plantilla que s'adapti a les teves necessitats (per exemple, First Person, Third Person).
  3. Crea una Nova Classe de Blueprint: Navega al Content Browser, fes clic dret i tria "Blueprint Class."

Nodes Bàsics de Blueprint

  • Event BeginPlay: Aquest node es crida quan el joc comença.
  • Event Tick: Aquest node es crida cada fotograma.
  • Print String: Útil per a la depuració, imprimeix un missatge a la pantalla.
Event BeginPlay -> Print String("Hello, Unreal Engine!")

Connectant Nodes

Els nodes es connecten arrossegant cables des de la sortida d'un node fins a l'entrada d'un altre node. Això crea un flux d'execució.

Variables i Tipus de Dades

Les variables emmagatzemen dades que es poden utilitzar i manipular dins dels Blueprints.

  • Creant Variables: Fes clic al botó "Variables" al panell My Blueprint.
  • Tipus de Dades: Els tipus de dades comuns inclouen Boolean, Integer, Float i String.
Event BeginPlay -> Set MyVariable (Value: 10)

Funcions i Esdeveniments

Les funcions i esdeveniments et permeten organitzar i reutilitzar la lògica.

  • Creant Funcions: Al panell My Blueprint, fes clic a "Functions" i després a "Add Function."
  • Esdeveniments Personalitzats: Es poden crear esdeveniments personalitzats per gestionar accions específiques.
Custom Event: MyCustomEvent
MyCustomEvent -> Print String("Custom Event Triggered")

Comunicació entre Blueprints

Els Blueprints poden comunicar-se entre ells per compartir dades i desencadenar esdeveniments.

  • Casting: S'utilitza per accedir a variables i funcions d'un altre Blueprint.
  • Event Dispatchers: Permeten que els Blueprints enviïn missatges entre ells.
Event BeginPlay -> Cast to MyOtherBlueprint -> Call Function in MyOtherBlueprint

Conceptes Avançats de Blueprint

Interfícies de Blueprint

Les interfícies de Blueprint permeten que diferents Blueprints es comuniquin sense necessitat de conèixer la implementació de l'altre.

  • Creant una Interfície: Al Content Browser, fes clic dret i tria "Blueprint Interface."
  • Implementant una Interfície: Al teu Blueprint, ves a Class Settings i afegeix la interfície.

Macros

Les macros són nodes reutilitzables que poden simplificar la lògica complexa.

  • Creant una Macro: Al panell My Blueprint, fes clic a "Macros" i després a "Add Macro."
  • Utilitzant Macros: Arrossega la macro al teu Blueprint per utilitzar-la.

Línies de Temps

Les línies de temps s'utilitzen per crear animacions i altres esdeveniments basats en el temps.

  • Creant una Línia de Temps: Al panell My Blueprint, fes clic a "Add Timeline."
  • Editant una Línia de Temps: Afegeix fotogrames clau i corbes per definir el comportament de la línia de temps.
Event BeginPlay -> Play Timeline
Timeline -> Update -> Set Actor Location

Depurant Blueprints

La depuració és una part essencial de treballar amb Blueprints.

  • Breakpoints: Configura breakpoints per pausar l'execució i inspeccionar variables.
  • Watch Variables: Monitoritza variables en temps real per veure com canvien.
  • Print String: Utilitza nodes Print String per mostrar missatges per a la depuració.

Conclusió

Els Blueprints són una eina versàtil i potent a Unreal Engine, que et permeten crear mecàniques de joc complexes sense escriure codi. Comprenent els conceptes bàsics dels Blueprints, variables, funcions i conceptes avançats com interfícies i línies de temps, pots crear experiències interactives i atractives. Practica regularment i experimenta amb diferents nodes per convertir-te en un expert en l'ús dels Blueprints.
