Introducció

Els Blueprints són un sistema de script visual a Unreal Engine que et permet crear elements de joc sense escriure codi. Aquest tema cobrirà les Funcions i Esdeveniments de Blueprint, components essencials per crear mecàniques de joc interactives i reutilitzables.

Funcions de Blueprint

Què són les Funcions de Blueprint?

  • Definició: Les Funcions de Blueprint són nodes reutilitzables que encapsulen un conjunt d'instruccions o lògica. Ajuden a organitzar i reutilitzar codi dins dels Blueprints.
  • Propòsit: Les funcions s'utilitzen per realitzar tasques específiques i es poden cridar múltiples vegades des de diferents llocs dins dels teus Blueprints.

Crear una Funció de Blueprint

  1. Obre el teu Blueprint: Navega fins a l'Editor de Blueprint.
  2. Afegeix una Funció: Fes clic a la pestanya "Functions" al panell My Blueprint i després fes clic al botó "+" per crear una nova funció.
  3. Nomena la Funció: Dóna-li a la teva funció un nom significatiu que descrigui el seu propòsit.
  4. Afegeix Nodes: Arrossega i deixa anar nodes al gràfic de la funció per definir el seu comportament.

Exemple: Funció Simple de Càlcul de Salut

// Nom de la Funció: CalculateHealth
// Entrades: CurrentHealth (float), Damage (float)
// Sortides: NewHealth (float)

1. Crea una nova funció anomenada "CalculateHealth".
2. Afegeix dos paràmetres d'entrada: CurrentHealth (float) i Damage (float).
3. Afegeix un paràmetre de sortida: NewHealth (float).
4. Dins del gràfic de la funció, resta Damage de CurrentHealth.
5. Assigna el resultat a NewHealth.

Gràfic:
[CurrentHealth] -- [Subtract] -- [Damage] --> [NewHealth]

Conceptes Clau

  • Paràmetres d'Entrada: Variables passades a la funció.
  • Paràmetres de Sortida: Variables retornades per la funció.
  • Variables Locals: Variables utilitzades dins de la funció que no són accessibles fora d'ella.

Esdeveniments de Blueprint

Què són els Esdeveniments de Blueprint?

  • Definició: Els esdeveniments són nodes que responen a accions o condicions específiques en el joc, com l'entrada del jugador, col·lisions o temporitzadors.
  • Propòsit: Els esdeveniments s'utilitzen per desencadenar peces específiques de lògica quan es compleixen certes condicions.

Tipus Comuns d'Esdeveniments

  • Begin Play: Desencadenat quan el joc comença.
  • Tick: Desencadenat cada fotograma.
  • Esdeveniments Personalitzats: Esdeveniments definits per l'usuari que es poden cridar des d'altres Blueprints o dins del mateix Blueprint.

Crear un Esdeveniment Personalitzat

  1. Obre el teu Blueprint: Navega fins a l'Editor de Blueprint.
  2. Afegeix un Esdeveniment: Fes clic amb el botó dret al Gràfic d'Esdeveniments i selecciona "Add Custom Event".
  3. Nomena l'Esdeveniment: Dóna-li a l'esdeveniment un nom significatiu.
  4. Afegeix Lògica: Arrossega i deixa anar nodes per definir què passa quan es desencadena l'esdeveniment.

Exemple: Esdeveniment Personalitzat per Imprimir un Missatge

// Nom de l'Esdeveniment: PrintMessage
// Acció: Imprimir "Hello, Unreal!" a la pantalla

1. Crea un nou esdeveniment personalitzat anomenat "PrintMessage".
2. Afegeix un node "Print String".
3. Connecta el pin d'execució de "PrintMessage" al node "Print String".
4. Assigna l'entrada de cadena del node "Print String" a "Hello, Unreal!".

Gràfic:
[PrintMessage] --> [Print String ("Hello, Unreal!")]

Conceptes Clau

  • Gràfic d'Esdeveniments: L'àrea principal on es defineixen els esdeveniments i la seva lògica corresponent.
  • Flux d'Execució: L'ordre en què s'executen els nodes, típicament controlat pels pins d'execució blancs.

Combinació de Funcions i Esdeveniments

Exemple: Utilitzar una Funció en un Esdeveniment

// Esdeveniment: OnTakeDamage
// Funció: CalculateHealth

1. Crea un esdeveniment personalitzat anomenat "OnTakeDamage".
2. Afegeix paràmetres d'entrada: CurrentHealth (float) i Damage (float).
3. Crida la funció "CalculateHealth" dins de l'esdeveniment.
4. Utilitza la sortida de "CalculateHealth" per actualitzar la salut del jugador.

Gràfic:
[OnTakeDamage] --> [CalculateHealth (CurrentHealth, Damage)] --> [Set Player Health]

Resum

  • Funcions de Blueprint: Encapsulen lògica reutilitzable, fent els teus Blueprints més organitzats i modulars.
  • Esdeveniments de Blueprint: Responen a accions o condicions específiques, permetent-te crear elements de joc interactius.
  • Combinació de Funcions i Esdeveniments: Millora la flexibilitat i reutilització dels teus Blueprints, permetent mecàniques de joc més complexes i interactives.

En dominar les Funcions i Esdeveniments de Blueprint, pots crear elements de joc eficients, reutilitzables i interactius a Unreal Engine, obrint el camí per a un desenvolupament de jocs més avançat.
