La il·luminació és un aspecte crucial per crear entorns realistes i visualment atractius a Unreal Engine. Aquest tema et guiarà a través dels conceptes bàsics de la il·luminació, els diferents tipus de llums i tècniques avançades d'il·luminació per millorar els teus projectes.

Conceptes Clau de la Il·luminació

  • Tipus de Llum: Entendre els diferents tipus de llums disponibles a Unreal Engine.
  • Propietats de la Llum: Aprendre sobre les propietats i configuracions que es poden ajustar per a cada tipus de llum.
  • Tècniques d'Il·luminació: Explorar diverses tècniques per aconseguir efectes d'il·luminació realistes.
  • Consideracions de Rendiment: Equilibrar la qualitat visual i el rendiment.

Tipus de Llum

Unreal Engine proporciona diversos tipus de llums, cadascun adequat per a diferents propòsits:

  1. Llum Direccional:

    • Simula la llum del sol o de la lluna.
    • Projecta raigs de llum paral·lels.
    • Ideal per a escenes a l'aire lliure.

  2. Llum Puntual:

    • Emissió de llum en totes direccions des d'un punt únic.
    • Similar a una bombeta.
    • Útil per a fonts de llum localitzades.

  3. Llum Focal:

    • Emissió de llum en forma de con.
    • Similar a una llanterna o un focus d'escenari.
    • Ideal per a efectes d'il·luminació enfocada.

  4. Llum Rectangular:

    • Emissió de llum des d'una àrea rectangular.
    • Útil per simular llums fluorescents o pantalles de televisió.

  5. Llum del Cel:

    • Captura les parts distants de l'escena i aplica aquesta il·luminació a tot el nivell.
    • Ideal per a il·luminació ambiental i reflexions.

Propietats de la Llum

Cada tipus de llum té diverses propietats que es poden ajustar per aconseguir l'efecte desitjat:

  • Intensitat: Controla la brillantor de la llum.
  • Color: Defineix el color de la llum.
  • Radi d'Atenuació: Defineix l'àrea afectada per la llum.
  • Projectar Ombres: Determina si la llum projecta ombres.
  • Radi/Longitud de la Font: Ajusta la mida de la font de llum per a ombres més suaus o més dures.

Exemple: Configuració d'una Llum Puntual

// Afegint una Llum Puntual en C++
// Suposant que tens una classe bàsica d'Actor

#include "Components/PointLightComponent.h"

APointLightActor::APointLightActor()
{
    // Crear un Component de Llum Puntual
    PointLight = CreateDefaultSubobject<UPointLightComponent>(TEXT("PointLight"));
    PointLight->Intensity = 3000.0f; // Configurar la intensitat
    PointLight->bVisible = true; // Fer-la visible
    PointLight->SetupAttachment(RootComponent); // Adjuntar al component arrel
}

En Blueprints, pots afegir un component de Llum Puntual a un actor i ajustar les seves propietats al panell de Detalls.

Tècniques d'Il·luminació

  1. Il·luminació Estàtica:

    • Il·luminació precomputada.
    • Millor per a objectes estàtics.
    • Ombres i il·luminació d'alta qualitat però sense canvis dinàmics.

  2. Il·luminació Estacionària:

    • Parcialment dinàmica.
    • Permet alguns canvis en les propietats de la llum en temps d'execució.
    • Adequada per a objectes que es mouen ocasionalment.

  3. Il·luminació Dinàmica:

    • Totalment dinàmica.
    • Il·luminació i ombres en temps real.
    • Ideal per a objectes que es mouen freqüentment però més intensiva en rendiment.

Exemple: Ús d'Il·luminació Dinàmica

// Activant Il·luminació Dinàmica per a una Llum Direccional en C++

#include "Components/DirectionalLightComponent.h"

ADirectionalLightActor::ADirectionalLightActor()
{
    // Crear un Component de Llum Direccional
    DirectionalLight = CreateDefaultSubobject<UDirectionalLightComponent>(TEXT("DirectionalLight"));
    DirectionalLight->Mobility = EComponentMobility::Movable; // Configurar com a Movable per a il·luminació dinàmica
    DirectionalLight->Intensity = 10.0f; // Configurar la intensitat
    DirectionalLight->SetupAttachment(RootComponent); // Adjuntar al component arrel
}

Consideracions de Rendiment

Equilibrar la qualitat visual i el rendiment és essencial:

  • Utilitza Il·luminació Estàtica per a objectes que no es mouen per estalviar en rendiment.
  • Limita el nombre de Llums Dinàmics per evitar impactes en el rendiment.
  • Optimitza la Configuració d'Ombres: Utilitza ombres de menor resolució per a objectes distants.
  • Utilitza Baking de Llum: Precomputa la il·luminació per a objectes estàtics per millorar el rendiment en temps d'execució.

Conclusió

La il·luminació a Unreal Engine és una eina poderosa que pot millorar significativament la qualitat visual dels teus projectes. Entenent els diferents tipus de llums, les seves propietats i diverses tècniques d'il·luminació, pots crear entorns realistes i immersius. Considera sempre les implicacions de rendiment i optimitza la teva configuració d'il·luminació per aconseguir el millor equilibri entre fidelitat visual i rendiment.
