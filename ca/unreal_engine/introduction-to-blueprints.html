Els Blueprints són un potent sistema de guió visual a Unreal Engine que et permet crear elements de joc sense escriure ni una sola línia de codi. Aquest tema et guiarà a través dels conceptes bàsics dels Blueprints, des d'entendre què són fins a crear el teu primer guió de Blueprint.

Què són els Blueprints?

Els Blueprints són un llenguatge de guió visual basat en el concepte de nodes i connexions. Et permeten crear mecàniques de joc complexes, interaccions i comportaments sense necessitat d'escriure codi tradicional.

  • Guió Visual: Els Blueprints utilitzen una interfície basada en nodes per crear lògica.
  • Accessibilitat: Dissenyat tant per a programadors com per a no programadors.
  • Integració: Totalment integrat amb Unreal Engine, permetent un prototipat i iteració ràpids.

Tipus de Blueprints

Hi ha diversos tipus de Blueprints a Unreal Engine, cadascun amb diferents propòsits:

  • Level Blueprints: Utilitzats per a escriptura d'esdeveniments específics del nivell.
  • Class Blueprints: Defineixen el comportament d'objectes i actors.
  • Function Blueprints: Conjunts reutilitzables de nodes que es poden cridar des d'altres Blueprints.
  • Macro Blueprints: Similar als funcions però poden contenir múltiples camins d'execució.

| Tipus de Blueprint | Descripció | |---------------------|--------------------------------------------------------| | Level Blueprints | Controlen esdeveniments i interaccions dins d'un nivell.| | Class Blueprints | Defineixen les propietats i comportaments dels objectes de joc.| | Function Blueprints | Encapsulen lògica reutilitzable. | | Macro Blueprints | Permeten estructures de nodes complexes i reutilitzables.|

Creant el teu Primer Blueprint

Creem un Blueprint senzill per fer que un objecte es mogui quan es prem una tecla.

Guia Pas a Pas

  1. Obre Unreal Engine: Comença un nou projecte o obre'n un d'existent.
  2. Crea una Nova Classe de Blueprint:
    • Ves al Content Browser.
    • Fes clic al botó "Add New".
    • Selecciona "Blueprint Class".
    • Tria "Actor" com a classe pare.
  3. Nomena el teu Blueprint: Dona-li un nom significatiu, com MovingActor.
  4. Obre l'Editor de Blueprint: Fes doble clic al teu nou Blueprint per obrir-lo.

Afegint Components

  1. Afegeix un Component de Malla Estàtica:
    • A la pestanya de Components, fes clic a "Add Component".
    • Selecciona "Static Mesh".
    • Tria una malla dels actius disponibles.

Creant la Lògica de Moviment

  1. Obre el Gràfic d'Esdeveniments: Aquí és on afegiràs els teus nodes.
  2. Afegeix un Esdeveniment d'Entrada:
    • Fes clic dret al Gràfic d'Esdeveniments.
    • Cerca "Input" i selecciona "InputAction Jump" (pots configurar les accions d'entrada a la Configuració del Projecte).
  3. Afegeix un Node de Moviment:
    • Arrossega des del pin "Pressed" del node InputAction Jump.
    • Cerca i selecciona "Add Actor Local Offset".
  4. Configura el Node de Moviment:
    • Configura el valor Z de l'Offset a 100 (això mourà l'actor cap amunt per 100 unitats).

Exemple de Guió de Blueprint

+--------------------+     +-------------------------+
| InputAction Jump   |     | Add Actor Local Offset  |
|--------------------|     |-------------------------|
| Pressed -----------|---->| Target: Self            |
|                    |     | Offset: (0, 0, 100)     |
+--------------------+     +-------------------------+

Provant el teu Blueprint

  1. Col·loca el Blueprint al Nivell:
    • Arrossega el teu Blueprint MovingActor des del Content Browser al nivell.
  2. Juga el Nivell:
    • Fes clic al botó "Play".
    • Prem la tecla de Salt (normalment la Barra Espaiadora) per veure com es mou l'actor.

Conclusió

Els Blueprints són una eina essencial a Unreal Engine, permetent tant a principiants com a desenvolupadors experimentats crear mecàniques de joc complexes de manera ràpida i eficient. Entenent els conceptes bàsics dels Blueprints, pots començar a construir elements interactius i dinàmics als teus jocs. A mesura que progressis, descobriràs funcions i tècniques més avançades que milloraran encara més les teves capacitats de desenvolupament.
