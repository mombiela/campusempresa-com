Creació i Ús de Variables a Unreal Engine
Funcions i Esdeveniments de Blueprints ►

Introducció

Les variables són fonamentals en la programació ja que emmagatzemen dades que es poden utilitzar i manipular al llarg del teu codi. A Unreal Engine, les variables s'utilitzen dins de Blueprints i C++ per controlar la lògica del joc, emmagatzemar dades del jugador i gestionar els estats del joc. Aquesta secció et guiarà a través dels conceptes bàsics de la creació i ús de variables a Unreal Engine, progressant des de conceptes per a principiants fins a conceptes avançats.

Nivell Principiant

Què és una Variable?

  • Una variable és una ubicació d'emmagatzematge identificada per un nom que conté dades que es poden modificar durant l'execució del programa.
  • Les variables tenen tipus, que defineixen el tipus de dades que poden contenir (per exemple, enters, floats, cadenes).

Creació de Variables en Blueprints

  1. Obre una Classe Blueprint: Comença obrint qualsevol classe Blueprint, com un Actor o un Personatge.
  2. Afegeix una Variable:
    • Al panell "My Blueprint", fes clic al botó "+" al costat de "Variables".
    • Nomena la teva variable (per exemple, Health).
    • Tria el tipus de variable del menú desplegable (per exemple, Float per punts de salut).
+-------------------+
| My Blueprint      |
|-------------------|
| Variables         |
| + Health (Float)  |
+-------------------+

Ús de Variables en Blueprints

  • Nodes Get i Set: Utilitza els nodes "Get" i "Set" per recuperar i modificar el valor d'una variable.
  • Exemple: Establir la salut del jugador a 100 quan el joc comença.
Event BeginPlay
    |
    V
Set Health (100)

Exemple de Codi Blueprint

Event BeginPlay
    |
    V
Set Health (100)

Creació de Variables en C++

  1. Obre la Teva Classe C++: Obre qualsevol classe C++, com una classe ACharacter personalitzada.
  2. Declara una Variable:
    • Al fitxer de capçalera (.h), declara la teva variable.
    • Exemple: float Health;
// MyCharacter.h
public:
    float Health;
  1. Inicialitza la Variable:
    • Al fitxer de codi (.cpp), inicialitza la teva variable al constructor o en un altre mètode apropiat.
// MyCharacter.cpp
AMyCharacter::AMyCharacter()
{
    Health = 100.0f;
}

Ús de Variables en C++

  • Accés i Modificació: Utilitza la variable directament dins dels mètodes de la teva classe.
  • Exemple: Imprimir la salut del jugador a la consola.
UE_LOG(LogTemp, Warning, TEXT("Player Health: %f"), Health);

Nivell Intermedi

Abast i Durada de les Variables

  • Variables Locals: Declarades dins d'una funció i només accessibles dins d'aquesta funció.
  • Variables de Membre: Declarades en una classe i accessibles per tots els mètodes de la classe.
  • Variables Globals: Declarades fora de qualsevol classe o funció i accessibles des de qualsevol lloc del codi (utilitza-les amb moderació).

Arrays i Col·leccions

  • Arrays: Emmagatzemen múltiples elements del mateix tipus.
  • Exemple en Blueprints: Crear un array d'enemics.
+-------------------+
| My Blueprint      |
|-------------------|
| Variables         |
| + Enemies (Array) |
+-------------------+
  • Exemple en C++:
TArray<AEnemy*> Enemies;

Structs

  • Definició: Un struct és un tipus de dada personalitzat que agrupa múltiples variables.
  • Exemple en Blueprints: Crear un struct per a les estadístiques del jugador.
+-------------------+
| My Blueprint      |
|-------------------|
| Structs           |
| + PlayerStats     |
+-------------------+
  • Exemple en C++:
USTRUCT(BlueprintType)
struct FPlayerStats
{
    GENERATED_BODY()

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float Health;

    UPROPERTY(EditAnywhere, BlueprintReadWrite)
    float Stamina;
};

Nivell Avançat

Estructures de Dades Avançades

  • Maps: Emmagatzemen parelles clau-valor.
  • Sets: Emmagatzemen elements únics.
  • Exemple en C++:
TMap<FString, int32> PlayerScores;
TSet<int32> UniqueIDs;

Enums

  • Definició: Els enums són tipus definits per l'usuari que consisteixen en valors amb nom.
  • Exemple en Blueprints: Crear un enum per a tipus d'armes.
+-------------------+
| My Blueprint      |
|-------------------|
| Enums             |
| + WeaponType      |
+-------------------+
  • Exemple en C++:
UENUM(BlueprintType)
enum class EWeaponType : uint8
{
    EWT_Sword UMETA(DisplayName = "Sword"),
    EWT_Bow UMETA(DisplayName = "Bow"),
    EWT_Axe UMETA(DisplayName = "Axe")
};

Ús de Punteres i Referències

  • Punteres: Variables que emmagatzemen adreces de memòria.
  • Referències: Àlies per a variables existents.
  • Exemple en C++:
AEnemy* EnemyPointer;
AEnemy& EnemyReference = *EnemyPointer;

Conclusió

Entendre i utilitzar efectivament les variables és crucial per al desenvolupament de jocs a Unreal Engine. Començant per la creació i manipulació bàsica de variables en Blueprints i C++, pots progressar a estructures de dades més complexes i conceptes avançats. El domini d'aquests temes et permetrà gestionar les dades del joc de manera eficient i crear experiències de joc més dinàmiques i interactives.
