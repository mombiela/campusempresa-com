Introducció

Embalatge i publicació del teu projecte d'Unreal Engine és el pas final en el desenvolupament de jocs. Aquest procés implica preparar el teu joc per a la distribució en diverses plataformes, com PC, consoles i dispositius mòbils. Aquesta secció et guiarà a través dels passos necessaris per embalar i publicar el teu joc, des de la configuració del projecte fins a la distribució al teu públic.

Configurant el Teu Projecte per a l'Embalatge

Abans de poder embalar el teu projecte, has d'assegurar-te que està correctament configurat.

  • Configuració del Projecte: Configura els paràmetres del teu projecte per a que coincideixin amb la plataforma objectiu.
  • Configuració de la Compilació: Tria la configuració de compilació adequada (per exemple, Desenvolupament, Distribució).
  • Configuració Específica de la Plataforma: Ajusta els paràmetres específics de la plataforma que estàs objectivant (per exemple, Android, iOS, Windows).

Configurant els Paràmetres del Projecte

  1. Obre el teu projecte a Unreal Engine.
  2. Navega a Edit > Project Settings.
  3. Ajusta els següents paràmetres:
    • Maps & Modes: Configura els teus mapes per defecte per al joc i l'editor.
    • Packaging: Configura paràmetres com Full Rebuild, For Distribution i Include Prerequisites.

Triant la Configuració de la Compilació

  • Desenvolupament: Utilitzat per a proves i depuració. Inclou símbols de depuració.
  • Distribució: Optimitzat per a rendiment i mida. No inclou símbols de depuració.

Configuració Específica de la Plataforma

  • Windows: Configura paràmetres com Minimum OS Version, Supported RHI.
  • Android: Configura el teu Android SDK, NDK i Java Development Kit (JDK).
  • iOS: Configura el teu Provisioning Profile i Certificate.

Embalant el Teu Projecte

Un cop el teu projecte està configurat, pots procedir a embalar-lo.

Embalant per a Windows

  1. Navega a File > Package Project > Windows.
  2. Tria 32-bit o 64-bit.
  3. Selecciona un directori per guardar el projecte embalat.
  4. Fes clic a OK per iniciar el procés d'embalatge.

Embalant per a Android

  1. Navega a File > Package Project > Android.
  2. Tria l'arquitectura d'Android adequada (per exemple, ARMv7, ARM64).
  3. Selecciona un directori per guardar l'APK embalat.
  4. Fes clic a OK per iniciar el procés d'embalatge.

Embalant per a iOS

  1. Navega a File > Package Project > iOS.
  2. Assegura't que el teu Mac està connectat si estàs embalant des d'una màquina Windows.
  3. Selecciona un directori per guardar l'IPA embalat.
  4. Fes clic a OK per iniciar el procés d'embalatge.

Publicant el Teu Joc

Després d'embalar, el següent pas és publicar el teu joc. Això implica distribuir-lo a través de diverses plataformes.

Distribuint a Steam

  1. Crea un Compte Steamworks: Registra't per a un compte Steamworks.
  2. Configura la Teva Aplicació: Crea una nova aplicació i configura els seus paràmetres.
  3. Carrega la Teva Compilació: Utilitza l'SDK de Steamworks per carregar la teva compilació embalada.
  4. Configura la Pàgina de la Botiga: Configura la teva pàgina de la botiga amb descripcions, imatges i vídeos.
  5. Envia per a Revisió: Envia el teu joc per al procés de revisió de Steam.

Distribuint a Google Play

  1. Crea un Compte de Desenvolupador de Google Play: Registra't i paga la quota de registre.
  2. Prepara el Teu APK: Assegura't que el teu APK està signat i compleix els requisits de Google Play.
  3. Carrega el Teu APK: Utilitza la Consola de Google Play per carregar el teu APK.
  4. Configura la Llista de la Botiga: Proporciona detalls, captures de pantalla i materials promocionals.
  5. Publica: Envia la teva aplicació per a revisió i publicació.

Distribuint a l'Apple App Store

  1. Crea un Compte de Desenvolupador d'Apple: Registra't i paga la quota anual.
  2. Prepara el Teu IPA: Assegura't que el teu IPA està signat i compleix els requisits d'Apple.
  3. Carrega el Teu IPA: Utilitza Xcode o Application Loader per carregar el teu IPA.
  4. Configura la Llista de l'App Store: Proporciona detalls, captures de pantalla i materials promocionals.
  5. Envia per a Revisió: Envia la teva aplicació per al procés de revisió d'Apple.

Conclusió

Embalatge i publicació del teu projecte d'Unreal Engine és un pas crític per portar el teu joc al teu públic. Seguint els passos descrits en aquesta secció, pots assegurar-te que el teu joc està correctament configurat, embalat i distribuït en diverses plataformes. Recorda provar a fons el teu joc en cada plataforma objectiu per assegurar una experiència d'usuari suau. Feliç desenvolupament!
