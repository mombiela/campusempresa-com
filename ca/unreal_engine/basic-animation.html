Introducció

En aquesta secció, cobrirem els conceptes bàsics de la creació i gestió d'animacions a Unreal Engine. Això inclou entendre el sistema d'animació, importar animacions i crear seqüències d'animació simples.

Conceptes clau

  • Sistema d'animació: El sistema d'animació d'Unreal Engine et permet crear i controlar animacions de personatges.
  • Malla esquelètica: Una malla que està equipada amb ossos i es pot animar.
  • Seqüència d'animació: Una sèrie de fotogrames que defineixen un moviment específic.
  • Plànol d'animació: Un plànol especialitzat que controla l'animació d'una malla esquelètica.

Configuració del teu projecte

Abans d'endinsar-te en les animacions, assegura't de tenir un projecte configurat amb un personatge que tingui una malla esquelètica.

  1. Crear un nou projecte:

    • Obre Unreal Engine.
    • Selecciona "Jocs" i fes clic a "Següent".
    • Tria una plantilla (per exemple, Tercera Persona) i fes clic a "Següent".
    • Configura els paràmetres del teu projecte i fes clic a "Crear".

  2. Importar una malla esquelètica:

    • Descarrega o crea una malla esquelètica en un programari de modelatge 3D.
    • Importa la malla a Unreal Engine arrossegant-la al Navegador de Contingut.

Importar animacions

Per animar el teu personatge, necessites seqüències d'animació. Aquestes es poden importar de fonts externes o crear dins d'Unreal Engine.

  1. Importar fitxers d'animació:

    • Obteniu fitxers d'animació en format FBX.
    • Arrossega i deixa anar els fitxers FBX al Navegador de Contingut.
    • A les opcions d'importació, assegura't que la casella "Importar animacions" estigui seleccionada.

  2. Crear seqüències d'animació:

    • Fes clic dret al Navegador de Contingut.
    • Navega a "Animació" -> "Seqüència d'animació".
    • Selecciona la malla esquelètica que vols animar.

Crear seqüències d'animació simples

Creem una animació de caminar simple per al nostre personatge.

  1. Obrir la seqüència d'animació:

    • Fes doble clic a la seqüència d'animació al Navegador de Contingut per obrir-la a l'Editor d'Animació.

  2. Editar l'animació:

    • Utilitza la línia de temps per desplaçar-te pels fotogrames de l'animació.
    • Ajusta els fotogrames clau per modificar l'animació.

  3. Reproduir l'animació:

    • Fes clic al botó "Reproduir" a l'Editor d'Animació per previsualitzar l'animació.

Utilitzar plànols d'animació

Els plànols d'animació et permeten controlar animacions complexes i barrejar diferents estats d'animació.

  1. Crear un plànol d'animació:

    • Fes clic dret al Navegador de Contingut.
    • Navega a "Animació" -> "Plànol d'animació".
    • Selecciona la malla esquelètica i fes clic a "Crear".

  2. Configurar el gràfic d'animació:

    • Obre el Plànol d'Animació.
    • Al Gràfic d'Animació, afegeix nodes per controlar el flux de l'animació.
    • Per exemple, afegeix un node de "Màquina d'Estats" per gestionar diferents estats d'animació (per exemple, inactiu, caminar, córrer).

  3. Crear una màquina d'estats:

    • Fes doble clic al node de la Màquina d'Estats per obrir-lo.
    • Afegeix estats (per exemple, Inactiu, Caminar, Córrer) i transicions entre ells.
    • Defineix les condicions per a cada transició (per exemple, velocitat > 0 per a la transició d'Inactiu a Caminar).

Exemple: Plànol d'animació de caminar simple

// Pseudo-codi per a un plànol d'animació de caminar simple

// Defineix els estats
Estat Inactiu {
    // Animació d'inactivitat
}

Estat Caminar {
    // Animació de caminar
}

// Defineix les transicions
Transició Inactiu -> Caminar {
    Condició: Velocitat > 0
}

Transició Caminar -> Inactiu {
    Condició: Velocitat == 0
}

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de l'animació a Unreal Engine, incloent la configuració del teu projecte, la importació d'animacions, la creació de seqüències d'animació simples i l'ús de plànols d'animació. Entenent aquests fonaments, pots començar a crear i controlar animacions per als teus personatges, donant-los vida en els teus projectes de jocs. Continua practicant i explorant funcions més avançades per millorar encara més les teves habilitats d'animació.
