Exportant Dades
Treballant amb Diferents Formats de Fitxers ►

Exportar dades és una habilitat crucial a MATLAB, que et permet guardar la teva feina i compartir resultats amb altres. Aquest tema et guiarà a través de diverses maneres d'exportar dades, des de tècniques bàsiques fins a avançades.

1. Exportació Bàsica de Dades

1.1 Exportant a Fitxers de Text

Els fitxers de text són un format senzill i àmpliament utilitzat per exportar dades.

  • Funció save: Guarda les variables de l'espai de treball en un fitxer.
  • Funció dlmwrite: Escriu dades de matriu en un fitxer de text amb un delimitador especificat.
  • Funció fprintf: Escriu dades formatades en un fitxer de text.

Exemple: Usant save

% Crear algunes dades
A = rand(5);
B = magic(5);

% Guardar dades en un fitxer .mat
save('myData.mat', 'A', 'B');

Exemple: Usant dlmwrite

% Crear algunes dades
C = rand(5);

% Guardar dades en un fitxer de text amb delimitador de coma
dlmwrite('myData.txt', C, 'delimiter', ',');

Exemple: Usant fprintf

% Crear algunes dades
D = rand(5);

% Obrir un fitxer per escriure
fileID = fopen('myDataFormatted.txt', 'w');

% Escriure dades amb sortida formatada
fprintf(fileID, '%6.2f %6.2f %6.2f %6.2f %6.2f\n', D');

% Tancar el fitxer
fclose(fileID);

1.2 Exportant a Fitxers Excel

Els fitxers Excel són útils per a l'anàlisi de dades i per compartir amb persones que no són programadors.

  • Funció xlswrite: Escriu dades en un fitxer Excel.

Exemple: Usant xlswrite

% Crear algunes dades
E = rand(5);

% Guardar dades en un fitxer Excel
xlswrite('myData.xlsx', E);

2. Exportació Intermèdia de Dades

2.1 Exportant a Fitxers CSV

Els fitxers CSV (Valors Separats per Coma) són un format comú per a l'intercanvi de dades.

  • Funció csvwrite: Escriu dades de matriu en un fitxer CSV.
  • Funció writematrix: Escriu dades de matriu en un fitxer, suportant diversos formats.

Exemple: Usant csvwrite

% Crear algunes dades
F = rand(5);

% Guardar dades en un fitxer CSV
csvwrite('myData.csv', F);

Exemple: Usant writematrix

% Crear algunes dades
G = rand(5);

% Guardar dades en un fitxer CSV
writematrix(G, 'myData.csv');

2.2 Exportant a Altres Formats de Fulls de Càlcul

MATLAB suporta diversos formats de fulls de càlcul més enllà d'Excel.

  • Funció writetable: Escriu dades de taula en un fitxer, suportant diferents formats com Excel, CSV i text.

Exemple: Usant writetable

% Crear una taula
T = table(rand(5,1), rand(5,1), 'VariableNames', {'Column1', 'Column2'});

% Guardar dades de taula en un fitxer Excel
writetable(T, 'myTableData.xlsx');

% Guardar dades de taula en un fitxer CSV
writetable(T, 'myTableData.csv');

3. Exportació Avançada de Dades

3.1 Exportant a Formats Personalitzats

Per a aplicacions especialitzades, pot ser necessari exportar dades en formats personalitzats.

  • Funció fwrite: Escriu dades binàries en un fitxer.
  • Funció jsonencode: Codifica dades de MATLAB en format JSON.
  • Funció xmlwrite: Escriu dades en un fitxer XML.

Exemple: Usant fwrite

% Crear algunes dades
H = rand(5);

% Obrir un fitxer per escriure
fileID = fopen('myData.bin', 'w');

% Escriure dades binàries
fwrite(fileID, H, 'double');

% Tancar el fitxer
fclose(fileID);

Exemple: Usant jsonencode

% Crear una estructura
dataStruct = struct('Field1', rand(5,1), 'Field2', magic(5));

% Codificar l'estructura a JSON
jsonData = jsonencode(dataStruct);

% Escriure dades JSON en un fitxer
fid = fopen('myData.json', 'w');
fwrite(fid, jsonData, 'char');
fclose(fid);

Exemple: Usant xmlwrite

% Crear un document XML
docNode = com.mathworks.xml.XMLUtils.createDocument('root_element');
docRootNode = docNode.getDocumentElement;
docRootNode.appendChild(docNode.createElement('child_element'));

% Escriure dades XML en un fitxer
xmlwrite('myData.xml', docNode);

Conclusió

Exportar dades a MATLAB és un procés versàtil que es pot adaptar a diverses necessitats, des de fitxers de text simples fins a formats personalitzats complexos. Dominant aquestes tècniques, pots assegurar-te que les teves dades es guarden, es comparteixen i s'utilitzen de manera efectiva en diferents entorns.
