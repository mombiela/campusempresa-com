Introducció a les Funcions

Les funcions són un bloc fonamental en la programació de MATLAB. Permeten encapsular codi en blocs reutilitzables, fent que els teus programes siguin més modulars, llegibles i mantenibles.

Conceptes Clau

  • Definició de Funció: Com definir una funció a MATLAB.
  • Arguments d'Entrada i Sortida: Gestionar les entrades i sortides en funcions.
  • Abast i Durada: Entendre l'abast de les variables dins de les funcions.
  • Funcions Anònimes: Crear funcions senzilles i sense nom.

Definició d'una Funció

A MATLAB, una funció es defineix en un fitxer separat amb el mateix nom que la funció. El fitxer ha de tenir una extensió .m.

Estructura Bàsica

function [output1, output2, ...] = functionName(input1, input2, ...)
    % Descripció de la funció
    % Explicació detallada aquí
    
    % Codi de la funció
    output1 = ...;
    output2 = ...;
end

Exemple

Creem una funció senzilla que sumi dos números.

function result = addNumbers(a, b)
    % addNumbers suma dos números a i b
    result = a + b;
end

Per utilitzar aquesta funció, guarda-la en un fitxer anomenat addNumbers.m i crida-la des de la finestra de comandes de MATLAB o un altre script:

sum = addNumbers(3, 5);
disp(sum);  % Sortida: 8

Arguments d'Entrada i Sortida

Les funcions poden tenir múltiples arguments d'entrada i sortida. El nombre d'entrades i sortides ha de coincidir amb la definició de la funció quan es crida.

Exemple amb Múltiples Sortides

function [sum, difference] = addAndSubtract(a, b)
    % addAndSubtract retorna la suma i la diferència de a i b
    sum = a + b;
    difference = a - b;
end

Cridant la funció:

[sum, diff] = addAndSubtract(10, 4);
disp(sum);  % Sortida: 14
disp(diff); % Sortida: 6

Abast i Durada

Les variables definides dins d'una funció són locals a aquesta funció i no es poden accedir des de fora. Això es coneix com l'abast de la variable.

Exemple

function result = multiplyNumbers(a, b)
    % multiplyNumbers multiplica dos números a i b
    product = a * b;  % 'product' és local a aquesta funció
    result = product;
end

Intentar accedir a product fora de la funció resultarà en un error.

Funcions Anònimes

Les funcions anònimes són funcions senzilles d'una línia que no requereixen un fitxer separat. Són útils per a operacions curtes.

Exemple

square = @(x) x^2;
result = square(5);
disp(result);  % Sortida: 25

Conclusió

Crear funcions a MATLAB és una manera poderosa d'organitzar i reutilitzar codi. Entenent com definir funcions, gestionar arguments d'entrada i sortida, i gestionar l'abast de les variables, pots escriure programes MATLAB més eficients i llegibles. A més, les funcions anònimes proporcionen una manera ràpida de definir operacions senzilles sense la necessitat de fitxers separats.
