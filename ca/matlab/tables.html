Introducció a les Taules

Les taules són un tipus de dades potent a MATLAB que et permeten emmagatzemar dades de diferents tipus i mides en un únic contenidor. Són especialment útils per organitzar i analitzar conjunts de dades que inclouen tant dades numèriques com no numèriques.

Conceptes Clau

  • Creació de Taules: Com crear taules a partir de matrius, cel·les o important dades.
  • Propietats de les Taules: Comprendre les dimensions de la taula, els noms de les variables i els noms de les files.
  • Accés a les Dades: Mètodes per accedir i modificar dades dins de les taules.
  • Operacions amb Taules: Operacions comunes com ordenar, filtrar i unir taules.

Creació de Taules

Pots crear taules a MATLAB utilitzant la funció table. Aquí tens algunes maneres bàsiques de crear taules:

A partir de Matrius

% Crear matrius
Age = [25; 30; 35];
Height = [5.5; 6.0; 5.8];
Weight = [150; 180; 165];

% Crear taula
T = table(Age, Height, Weight);
disp(T);

Aquest codi crea una taula T amb tres columnes: Age, Height i Weight.

A partir de Cel·les

% Crear cel·les
Names = {'John'; 'Jane'; 'Jim'};
Scores = {85, 90, 88};

% Crear taula
T = table(Names, Scores);
disp(T);

Aquest codi crea una taula T amb dues columnes: Names i Scores.

Important Dades

També pots crear taules important dades de fitxers com CSV, Excel o fitxers de text.

% Importar dades d'un fitxer CSV
T = readtable('data.csv');
disp(T);

Propietats de les Taules

Les taules tenen propietats que defineixen la seva estructura i metadades.

Dimensions

Pots obtenir la mida d'una taula utilitzant la funció size.

% Obtenir dimensions de la taula
[rows, cols] = size(T);
disp(['Files: ', num2str(rows), ', Columnes: ', num2str(cols)]);

Noms de les Variables

Els noms de les variables es poden accedir i modificar utilitzant la propietat Properties.VariableNames.

% Obtenir noms de les variables
varNames = T.Properties.VariableNames;
disp(varNames);

% Establir nous noms de les variables
T.Properties.VariableNames = {'Age', 'Height_in', 'Weight_lb'};
disp(T);

Noms de les Files

De manera similar, els noms de les files es poden accedir i modificar.

% Establir noms de les files
T.Properties.RowNames = {'Person1', 'Person2', 'Person3'};
disp(T);

Accés i Modificació de Dades

Pots accedir i modificar les dades de la taula utilitzant diversos mètodes d'indexació.

Per Nom de Variable

% Accedir a dades per nom de variable
heights = T.Height_in;
disp(heights);

% Modificar dades per nom de variable
T.Height_in = T.Height_in * 2.54; % Convertir polzades a cm
disp(T);

Per Nom de Fila i Variable

% Accedir a un element específic
height_person1 = T{'Person1', 'Height_in'};
disp(height_person1);

% Modificar un element específic
T{'Person1', 'Height_in'} = 170;
disp(T);

Operacions amb Taules

MATLAB proporciona diverses funcions per realitzar operacions amb taules.

Ordenar

% Ordenar taula per Age
T = sortrows(T, 'Age');
disp(T);

Filtrar

% Filtrar files on Age és major que 30
T_filtered = T(T.Age > 30, :);
disp(T_filtered);

Unir Taules

Pots unir dues taules utilitzant la funció join.

% Crear una altra taula
Gender = {'M'; 'F'; 'M'};
T2 = table(Gender, 'RowNames', {'Person1', 'Person2', 'Person3'});

% Unir taules
T_joined = join(T, T2);
disp(T_joined);

Conclusió

Les taules a MATLAB són versàtils i potents per gestionar dades heterogènies. Comprenent com crear, manipular i realitzar operacions amb taules, pots organitzar i analitzar efectivament conjunts de dades complexos. Practica creant taules a partir de diferents fonts de dades i realitzant diverses operacions per convertir-te en un expert en l'ús de taules a MATLAB.
