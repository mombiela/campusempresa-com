Introducció

Els arrays són estructures de dades fonamentals a MATLAB, utilitzades per emmagatzemar i manipular col·leccions de dades. Aquesta secció et guiarà a través dels conceptes bàsics de la creació d'arrays, l'accés als seus elements i la realització d'operacions comunes sobre ells. Al final d'aquesta secció, et sentiràs còmode treballant amb arrays a MATLAB, des de la creació bàsica fins a tècniques avançades de manipulació.

Creació d'Arrays

1. Creació de Vectors de Fila i Columna

  • Vector de Fila: Un vector de fila és un array de 1xN, on N és el nombre d'elements.
  • Vector de Columna: Un vector de columna és un array de Nx1.
% Creació d'un vector de fila
row_vector = [1, 2, 3, 4, 5];

% Creació d'un vector de columna
column_vector = [1; 2; 3; 4; 5];

2. Creació de Matrius

  • Matriu: Una matriu és un array 2D amb M files i N columnes.
% Creació d'una matriu de 2x3
matrix = [1, 2, 3; 4, 5, 6];

3. Ús de Funcions Incorporades

  • zeros: Crea un array de tots zeros.
  • ones: Crea un array de tots uns.
  • eye: Crea una matriu identitat.
  • rand: Crea un array amb valors aleatoris entre 0 i 1.
% Creació d'una matriu de 3x3 de zeros
zero_matrix = zeros(3, 3);

% Creació d'una matriu de 2x4 d'uns
one_matrix = ones(2, 4);

% Creació d'una matriu identitat de 4x4
identity_matrix = eye(4);

% Creació d'una matriu de 3x3 amb valors aleatoris
random_matrix = rand(3, 3);

Accés i Modificació d'Elements d'Array

1. Indexació

  • MATLAB utilitza indexació basada en 1.
  • Pots accedir als elements utilitzant parèntesis ().
% Accés a l'element a la fila 2, columna 3
element = matrix(2, 3);

% Accés a tota la segona fila
second_row = matrix(2, :);

% Accés a tota la tercera columna
third_column = matrix(:, 3);

2. Modificació d'Elements

  • Pots modificar els elements assignant nous valors.
% Canvi de l'element a la fila 1, columna 2 a 10
matrix(1, 2) = 10;

% Canvi de tota la primera fila
matrix(1, :) = [7, 8, 9];

Operacions Comunes amb Arrays

1. Operacions Aritmètiques

  • Pots realitzar operacions element a element utilitzant .*, ./, i .^.
% Multiplicació element a element
result = matrix .* 2;

% Divisió element a element
result = matrix ./ 2;

% Potència element a element
result = matrix .^ 2;

2. Operacions amb Matrius

  • MATLAB suporta operacions estàndard amb matrius com suma, resta, multiplicació i transposició.
% Suma de matrius
result = matrix + ones(2, 3);

% Multiplicació de matrius
result = matrix * eye(3);

% Transposició de matriu
transposed_matrix = matrix';

3. Remodelació d'Arrays

  • Pots canviar la forma d'un array utilitzant la funció reshape.
% Remodelació d'una matriu de 2x3 a una matriu de 3x2
reshaped_matrix = reshape(matrix, 3, 2);

Manipulació Avançada d'Arrays

1. Concatenació

  • Pots concatenar arrays horitzontalment o verticalment.
% Concatenació horitzontal
h_concat = [matrix, ones(2, 3)];

% Concatenació vertical
v_concat = [matrix; ones(2, 3)];

2. Indexació Lògica

  • Utilitza condicions lògiques per indexar i manipular arrays.
% Creació d'un array lògic
logical_array = matrix > 3;

% Ús de la indexació lògica per modificar elements
matrix(matrix > 3) = 0;

3. Cerca d'Elements

  • Utilitza funcions com find per localitzar elements que compleixin una condició.
% Cerca d'índexs d'elements majors que 3
indices = find(matrix > 3);

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la creació i manipulació d'arrays a MATLAB. Has après a crear vectors i matrius, accedir i modificar els seus elements, realitzar operacions aritmètiques i amb matrius comunes, remodelar arrays, i utilitzar tècniques avançades com la concatenació i la indexació lògica. Dominar aquestes habilitats et permetrà gestionar dades de manera eficient i realitzar càlculs complexos a MATLAB.
