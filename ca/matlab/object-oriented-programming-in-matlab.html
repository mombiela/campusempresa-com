Introducció a la Programació Orientada a Objectes (OOP)

La Programació Orientada a Objectes (OOP) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. MATLAB admet OOP, permetent-te crear classes, objectes, i utilitzar herència i polimorfisme.

Conceptes Clau de l'OOP

  • Classe: Un plànol per crear objectes (una estructura de dades particular).
  • Objecte: Una instància d'una classe.
  • Propietats: Variables que pertanyen a un objecte o classe.
  • Mètodes: Funcions que pertanyen a un objecte o classe.
  • Herència: Mecanisme per crear una nova classe utilitzant propietats i mètodes d'una classe existent.
  • Polimorfisme: Capacitat de processar objectes de manera diferent segons la seva classe.

Creació de Classes i Objectes en MATLAB

Definició d'una Classe

En MATLAB, una classe es defineix utilitzant la paraula clau classdef. A continuació es mostra un exemple d'una definició de classe simple:

classdef Circle
    properties
        Radius
    end
    methods
        function obj = Circle(r)
            if nargin > 0
                obj.Radius = r;
            end
        end
        function area = getArea(obj)
            area = pi * obj.Radius^2;
        end
    end
end

Creació d'un Objecte

Per crear un objecte de la classe Circle, utilitzes el constructor de la classe:

c = Circle(5);
disp(c.Radius); % Sortida: 5
disp(c.getArea()); % Sortida: 78.5398

Propietats i Mètodes

Propietats

Les propietats són variables que pertanyen a una classe. Es defineixen dins del bloc properties.

classdef Car
    properties
        Make
        Model
        Year
    end
end

Mètodes

Els mètodes són funcions que operen sobre objectes de la classe. Es defineixen dins del bloc methods.

classdef Car
    properties
        Make
        Model
        Year
    end
    methods
        function obj = Car(make, model, year)
            obj.Make = make;
            obj.Model = model;
            obj.Year = year;
        end
        function age = getCarAge(obj)
            currentYear = year(datetime('now'));
            age = currentYear - obj.Year;
        end
    end
end

Exemple d'Ús

myCar = Car('Toyota', 'Camry', 2015);
disp(myCar.Make); % Sortida: Toyota
disp(myCar.getCarAge()); % Sortida: (any actual - 2015)

Herència

Definició d'una Subclasse

L'herència permet que una nova classe hereti propietats i mètodes d'una classe existent.

classdef ElectricCar < Car
    properties
        BatteryCapacity
    end
    methods
        function obj = ElectricCar(make, model, year, batteryCapacity)
            obj@Car(make, model, year); % Crida al constructor de la superclasse
            obj.BatteryCapacity = batteryCapacity;
        end
        function range = getRange(obj)
            range = obj.BatteryCapacity * 5; % Càlcul d'exemple
        end
    end
end

Exemple d'Ús

myElectricCar = ElectricCar('Tesla', 'Model S', 2020, 100);
disp(myElectricCar.Make); % Sortida: Tesla
disp(myElectricCar.getRange()); % Sortida: 500

Polimorfisme

Sobreescriptura de Mètodes

El polimorfisme en MATLAB es pot aconseguir mitjançant la sobreescriptura de mètodes, on una subclasse proporciona una implementació específica d'un mètode que ja està definit en la seva superclasse.

classdef SportsCar < Car
    methods
        function speed = getTopSpeed(obj)
            speed = 200; % Velocitat d'exemple
        end
    end
end

classdef ElectricSportsCar < SportsCar
    methods
        function speed = getTopSpeed(obj)
            speed = 250; % Mètode sobreescrit
        end
    end
end

Exemple d'Ús

mySportsCar = SportsCar('Ferrari', '488', 2019);
disp(mySportsCar.getTopSpeed()); % Sortida: 200

myElectricSportsCar = ElectricSportsCar('Tesla', 'Roadster', 2022);
disp(myElectricSportsCar.getTopSpeed()); % Sortida: 250

Resum

En aquesta secció, hem explorat els conceptes bàsics de la Programació Orientada a Objectes en MATLAB. Hem cobert:

  • Els conceptes fonamentals de l'OOP.
  • Com definir classes i crear objectes.
  • L'ús de propietats i mètodes.
  • Implementar l'herència per crear subclasses.
  • Aconseguir el polimorfisme mitjançant la sobreescriptura de mètodes.

Entendre aquests conceptes et permetrà dissenyar i implementar programes més complexos i modulars en MATLAB, aprofitant el poder de l'OOP per crear codi reutilitzable i mantenible.
