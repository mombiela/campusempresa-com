Declaracions condicionals a MATLAB
Bucles: for i while ►

Les declaracions condicionals són fonamentals en la programació ja que permeten executar diferents blocs de codi basats en certes condicions. A MATLAB, les declaracions condicionals inclouen if, else, elseif, switch i case. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús més avançat d'aquestes declaracions.

Introducció a les declaracions condicionals

Les declaracions condicionals controlen el flux d'execució en un programa. Permeten que el programa prengui decisions i executi blocs de codi específics basats en l'avaluació de condicions.

Conceptes clau

  • Condició: Una expressió que s'avalua com a certa o falsa.
  • Bloc de codi: Un conjunt de declaracions que s'executen si la condició és certa.
  • Flux de control: L'ordre en què es realitzen o s'avaluen les declaracions individuals, instruccions o crides a funcions.

Declaracions `if`, `else` i `elseif`

La declaració if és la forma més bàsica de declaració condicional. Executa un bloc de codi si una condició especificada és certa. Les declaracions else i elseif proporcionen un control addicional permetent blocs de codi alternatius si la condició inicial és falsa.

Sintaxi

if condition
    % Codi a executar si la condició és certa
elseif another_condition
    % Codi a executar si another_condition és certa
else
    % Codi a executar si cap de les condicions anteriors és certa
end

Exemple

x = 10;

if x > 0
    disp('x és positiu');
elseif x < 0
    disp('x és negatiu');
else
    disp('x és zero');
end

En aquest exemple:

  • Si x és més gran que 0, mostra "x és positiu".
  • Si x és menor que 0, mostra "x és negatiu".
  • Si x és 0, mostra "x és zero".

Declaracions `if` niades

Pots niar declaracions if dins d'altres per comprovar múltiples condicions de manera jeràrquica.

Exemple

x = 5;
y = 10;

if x > 0
    if y > 0
        disp('Tant x com y són positius');
    else
        disp('x és positiu, però y no és positiu');
    end
else
    disp('x no és positiu');
end

En aquest exemple:

  • El if extern comprova si x és positiu.
  • El if intern comprova si y és positiu si la condició externa és certa.

Declaracions `switch` i `case`

La declaració switch s'utilitza per executar un dels diversos grups de declaracions. És una alternativa a utilitzar múltiples declaracions if i elseif.

Sintaxi

switch variable
    case value1
        % Codi a executar si variable és igual a value1
    case value2
        % Codi a executar si variable és igual a value2
    otherwise
        % Codi a executar si variable no coincideix amb cap cas
end

Exemple

day = 'Monday';

switch day
    case 'Monday'
        disp('Inici de la setmana laboral');
    case 'Friday'
        disp('Final de la setmana laboral');
    otherwise
        disp('Mitja setmana');
end

En aquest exemple:

  • Si day és 'Monday', mostra "Inici de la setmana laboral".
  • Si day és 'Friday', mostra "Final de la setmana laboral".
  • Per qualsevol altre valor de day, mostra "Mitja setmana".

Comparació de `if` i `switch`

| Característica | Declaració if | Declaració switch | |------------------|-----------------------------------------------------|---------------------------------------------------| | Cas d'ús | Múltiples condicions amb diferents expressions | Variable única amb múltiples valors possibles | | Llegibilitat | Pot esdevenir complexa amb moltes condicions elseif| Més llegible per comprovar una variable única | | Flexibilitat | Més flexible amb condicions complexes | Menys flexible, millor per a valors discrets |

Ús avançat

Operadors lògics

Els operadors lògics es poden utilitzar per combinar múltiples condicions en declaracions if.

  • AND (&&): Cert si ambdues condicions són certes.
  • OR (||): Cert si almenys una condició és certa.
  • NOT (~): Cert si la condició és falsa.

Exemple

a = 5;
b = 10;

if a > 0 && b > 0
    disp('Tant a com b són positius');
elseif a > 0 || b > 0
    disp('Almenys un de a o b és positiu');
else
    disp('Ni a ni b són positius');
end

Conclusió

Les declaracions condicionals són essencials per controlar el flux d'un programa basat en diferents condicions. A MATLAB, les declaracions if, else, elseif, switch i case proporcionen eines poderoses per a la presa de decisions. Entendre i utilitzar efectivament aquestes declaracions et permetrà escriure codi més dinàmic i sensible.

Resum

  • if, else, elseif: Utilitzats per comprovar múltiples condicions.
  • if niat: Permet la comprovació jeràrquica de condicions.
  • switch, case: Simplifica la comprovació d'una variable única contra múltiples valors.
  • Operadors lògics: Combinen múltiples condicions dins de declaracions if.

En dominar aquests conceptes, estaràs ben preparat per gestionar processos de presa de decisions complexos en els teus programes MATLAB.
