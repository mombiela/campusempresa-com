Introducció

El Projecte Capstone és la culminació del teu curs de MATLAB, on aplicaràs els coneixements i habilitats que has adquirit des de nivells bàsics fins a avançats. Aquest projecte et desafiarà a resoldre un problema del món real utilitzant MATLAB, demostrant la teva competència en el llenguatge i la teva capacitat de pensar de manera crítica i creativa.

Visió General del Projecte

En aquesta secció, obtindràs una visió general del Projecte Capstone, incloent-hi els objectius, requisits i lliuraments.

Objectius

  • Aplicar habilitats de MATLAB per resoldre un problema complex.
  • Demostrar competència en la programació amb MATLAB.
  • Integrar diverses funcionalitats i caixes d'eines de MATLAB.
  • Desenvolupar una solució integral que inclogui anàlisi de dades, visualització i possiblement simulació.

Requisits

  • El projecte ha de ser original i no una replicació de treballs existents.
  • Utilitzar almenys tres funcionalitats diferents de MATLAB (per exemple, importació/exportació de dades, gràfics, anàlisi estadística).
  • Incloure un informe detallat explicant el teu enfocament, codi i resultats.
  • Presentar els teus descobriments de manera clara i concisa.

Lliuraments

  • Fitxers de codi MATLAB (.m files).
  • Un informe del projecte (document PDF o Word).
  • Una presentació (PowerPoint o format similar).

Idees de Projecte

Aquí tens algunes idees de projecte per començar. Pots triar una d'aquestes o proposar-ne una de pròpia.

Anàlisi i Visualització de Dades

  • Objectiu: Analitzar un conjunt de dades i crear visualitzacions per presentar els teus descobriments.
  • Exemple: Utilitzar un conjunt de dades de temperatures globals per analitzar tendències i crear gràfics que mostrin els canvis de temperatura al llarg del temps.

Exemple de Codi

% Carregar conjunt de dades
data = readtable('global_temperatures.csv');

% Extreure columnes rellevants
years = data.Year;
temperatures = data.AverageTemperature;

% Gràfic de dades
figure;
plot(years, temperatures);
title('Canvis de Temperatura Global al Llarg del Temps');
xlabel('Any');
ylabel('Temperatura Mitjana');
grid on;

Processament d'Imatges

  • Objectiu: Desenvolupar una aplicació de processament d'imatges utilitzant la Caixa d'Eines de Processament d'Imatges de MATLAB.
  • Exemple: Crear un programa que detecti i ressalti les vores en una imatge.

Exemple de Codi

% Llegir imatge
img = imread('example.jpg');

% Convertir a escala de grisos
grayImg = rgb2gray(img);

% Detectar vores
edges = edge(grayImg, 'Canny');

% Mostrar imatges originals i processades
figure;
subplot(1, 2, 1);
imshow(img);
title('Imatge Original');

subplot(1, 2, 2);
imshow(edges);
title('Detecció de Vores');

Simulació i Modelatge

  • Objectiu: Crear un model de simulació per estudiar un sistema dinàmic.
  • Exemple: Simular el moviment d'un pèndol utilitzant els solvers d'ODE de MATLAB.

Exemple de Codi

% Definir paràmetres
g = 9.81; % acceleració deguda a la gravetat
L = 1.0; % longitud del pèndol
theta0 = pi/4; % angle inicial
omega0 = 0; % velocitat angular inicial

% Definir l'ODE
ode = @(t, y) [y(2); -g/L * sin(y(1))];

% Resoldre l'ODE
[t, y] = ode45(ode, [0, 10], [theta0, omega0]);

% Gràfic dels resultats
figure;
plot(t, y(:, 1));
title('Moviment del Pèndol');
xlabel('Temps (s)');
ylabel('Angle (rad)');
grid on;

Passos per al Desenvolupament del Projecte

Segueix aquests passos per desenvolupar el teu Projecte Capstone:

  1. Definir el Problema: Indica clarament el problema que estàs abordant.
  2. Investigació: Reuneix informació i recursos relacionats amb el teu problema.
  3. Planificació: Esbossa el teu enfocament i els passos que seguiràs per resoldre el problema.
  4. Implementació: Escriu el codi MATLAB per implementar la teva solució.
  5. Prova: Prova el teu codi per assegurar-te que funciona correctament.
  6. Anàlisi: Analitza els resultats i fes els ajustos necessaris.
  7. Documentació: Escriu un informe detallat explicant el teu enfocament, codi i resultats.
  8. Presentació: Prepara una presentació per mostrar el teu projecte.

Conclusió

El Projecte Capstone és una oportunitat per mostrar les teves habilitats i creativitat amb MATLAB. En completar aquest projecte, demostraràs la teva capacitat per aplicar MATLAB a la resolució de problemes complexos, integrar diverses funcionalitats i presentar els teus descobriments de manera efectiva. Bona sort i gaudeix del procés de crear alguna cosa significativa amb MATLAB!
