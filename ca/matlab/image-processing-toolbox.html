Introducció

La Caixa d'Eines de Processament d'Imatges a MATLAB proporciona un conjunt complet d'algoritmes i funcions de referència estàndard per al processament, anàlisi, visualització i desenvolupament d'algoritmes d'imatges. Aquest curs et guiarà des de nivells bàsics fins a avançats, cobrint conceptes essencials, exemples pràctics i exercicis pràctics.

Començant amb la Caixa d'Eines de Processament d'Imatges

  • Instal·lació: Assegura't que la Caixa d'Eines de Processament d'Imatges està instal·lada en el teu entorn MATLAB.
  • Funcions Bàsiques: Introducció a funcions bàsiques com imread, imshow, imwrite.

Exemple: Llegir i Mostrar una Imatge

% Llegir una imatge d'un fitxer
image = imread('example.jpg');

% Mostrar la imatge
imshow(image);

Explicació: La funció imread llegeix el fitxer d'imatge en una matriu, i imshow la mostra.

Millora d'Imatges

  • Equalització d'Histograma: Millorar el contrast de les imatges.
  • Filtratge: Aplicar filtres per eliminar soroll o millorar característiques.

Exemple: Equalització d'Histograma

% Llegir una imatge
image = imread('example.jpg');

% Convertir a escala de grisos
grayImage = rgb2gray(image);

% Aplicar equalització d'histograma
equalizedImage = histeq(grayImage);

% Mostrar les imatges original i equalitzada
subplot(1, 2, 1), imshow(grayImage), title('Imatge Original');
subplot(1, 2, 2), imshow(equalizedImage), title('Imatge Equalitzada');

Explicació: rgb2gray converteix la imatge a escala de grisos, i histeq realitza l'equalització d'histograma per millorar el contrast.

Transformacions Geomètriques

  • Redimensionament: Canviar la mida d'una imatge.
  • Rotació: Rotar una imatge per un angle especificat.
  • Translació: Desplaçar una imatge en les direccions x i y.

Exemple: Redimensionar una Imatge

% Llegir una imatge
image = imread('example.jpg');

% Redimensionar la imatge a la meitat de la seva mida original
resizedImage = imresize(image, 0.5);

% Mostrar les imatges original i redimensionada
subplot(1, 2, 1), imshow(image), title('Imatge Original');
subplot(1, 2, 2), imshow(resizedImage), title('Imatge Redimensionada');

Explicació: La funció imresize s'utilitza per redimensionar la imatge al 50% de les seves dimensions originals.

Segmentació d'Imatges

  • Llindarització: Segmentar una imatge basada en valors d'intensitat.
  • Detecció de vores: Detectar vores utilitzant mètodes com Sobel, Canny.
  • Segmentació basada en regions: Segmentar una imatge en regions basades en la similitud.

Exemple: Detecció de Vores utilitzant el Mètode Canny

% Llegir una imatge
image = imread('example.jpg');

% Convertir a escala de grisos
grayImage = rgb2gray(image);

% Aplicar detecció de vores Canny
edges = edge(grayImage, 'Canny');

% Mostrar les imatges original i amb vores detectades
subplot(1, 2, 1), imshow(grayImage), title('Imatge Original');
subplot(1, 2, 2), imshow(edges), title('Imatge amb Vores Detectades');

Explicació: La funció edge amb el mètode 'Canny' detecta les vores en la imatge en escala de grisos.

Operacions Morfològiques

  • Dilació i Erosió: Operacions bàsiques per expandir o reduir regions d'imatge.
  • Obertura i Tancament: Combinació de dilació i erosió per eliminar soroll.

Exemple: Dilació i Erosió

% Llegir una imatge binària
binaryImage = imread('binary_example.png');

% Crear un element estructurant
se = strel('disk', 5);

% Aplicar dilació
dilatedImage = imdilate(binaryImage, se);

% Aplicar erosió
erodedImage = imerode(binaryImage, se);

% Mostrar les imatges original, dilatada i erosionada
subplot(1, 3, 1), imshow(binaryImage), title('Imatge Original');
subplot(1, 3, 2), imshow(dilatedImage), title('Imatge Dilatada');
subplot(1, 3, 3), imshow(erodedImage), title('Imatge Erosionada');

Explicació: Les funcions imdilate i imerode s'utilitzen amb un element estructurant en forma de disc per realitzar dilació i erosió.

Temes Avançats

  • Registre d'Imatges: Alinear imatges de diferents moments, sensors o punts de vista.
  • Extracció de Característiques: Extreure característiques com cantonades, blobs i línies.
  • Aprenentatge Automàtic per al Processament d'Imatges: Utilitzar tècniques d'aprenentatge automàtic per a tasques com classificació i segmentació.

Exemple: Extracció de Característiques utilitzant el Detector de Cantonades de Harris

% Llegir una imatge
image = imread('example.jpg');

% Convertir a escala de grisos
grayImage = rgb2gray(image);

% Detectar cantonades de Harris
corners = detectHarrisFeatures(grayImage);

% Mostrar la imatge amb cantonades
imshow(grayImage); hold on;
plot(corners.selectStrongest(50));

Explicació: La funció detectHarrisFeatures detecta cantonades en la imatge en escala de grisos, i es plotegen les 50 cantonades més fortes.

Conclusió

La Caixa d'Eines de Processament d'Imatges a MATLAB proporciona eines potents per a una àmplia gamma de tasques de processament d'imatges. Des d'operacions bàsiques com llegir i mostrar imatges fins a tècniques avançades com l'extracció de característiques i l'aprenentatge automàtic, aquesta caixa d'eines és essencial per a qualsevol persona que treballi amb dades d'imatges. Seguint aquest curs, adquiriràs una sòlida comprensió i experiència pràctica en el processament d'imatges utilitzant MATLAB.
