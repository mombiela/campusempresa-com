Introducció a la Computació Paral·lela

La computació paral·lela és un tipus de càlcul en el qual molts càlculs o processos es duen a terme simultàniament. MATLAB proporciona diverses eines i funcions per ajudar-te a aprofitar la computació paral·lela per accelerar el teu codi.

Conceptes Clau

  • Paral·lelisme: Realitzar múltiples operacions simultàniament.
  • Concurrència: Estructurar un programa per realitzar múltiples tasques al mateix temps.
  • Multithreading: Utilitzar múltiples fils dins d'un sol procés per executar tasques de manera concurrent.
  • Computació Distribuïda: Utilitzar múltiples ordinadors per realitzar càlculs.

Començant amb la Computació Paral·lela a MATLAB

MATLAB fa que sigui fàcil començar a utilitzar la computació paral·lela amb canvis mínims en el teu codi.

Parallel Computing Toolbox

La Parallel Computing Toolbox a MATLAB proporciona construccions per a la computació paral·lela, incloent bucles parfor, tipus d'array especials i versions paral·lelitzades de moltes funcions estàndard.

Configurant el Teu Entorn

Abans de poder utilitzar les característiques de computació paral·lela, has de configurar el teu entorn.

% Comprova si la Parallel Computing Toolbox està instal·lada
ver('distcomp')

% Inicia un pool paral·lel
parpool

Bucle bàsic `parfor`

El bucle parfor és una versió paral·lela del bucle for estàndard. Distribueix les iteracions entre els treballadors disponibles en un pool paral·lel.

% Exemple: Bucle paral·lel
n = 1000;
A = zeros(n, 1);

parfor i = 1:n
    A(i) = i^2;
end
  • Explicació: En aquest exemple, el bucle parfor distribueix el càlcul de i^2 entre múltiples treballadors.

Tècniques Avançades de Computació Paral·lela

A mesura que et familiaritzis amb les construccions paral·leles bàsiques, pots explorar tècniques més avançades.

Avaluació de Funcions Paral·leles (`parfeval`)

La funció parfeval et permet executar funcions de manera asíncrona.

% Exemple: Avaluació de funcions asíncrona
futures = parfeval(@magic, 1, 3); % Sol·licita un quadrat màgic de 3x3
result = fetchOutputs(futures); % Recupera el resultat
  • Explicació: Aquest codi executa la funció magic de manera asíncrona i recupera el resultat un cop està llest.

Processament de Dades Paral·lel amb `spmd`

La construcció spmd (Single Program Multiple Data) et permet executar codi simultàniament en múltiples treballadors, cadascun amb el seu propi espai de treball.

% Exemple: Utilitzant spmd
spmd
    labindex % Retorna l'índex del treballador
end
  • Explicació: Aquest codi s'executa en tots els treballadors i retorna l'índex de cada treballador.

Arrays Distribuïts

Els arrays distribuïts et permeten treballar amb grans conjunts de dades que no caben en la memòria d'una sola màquina.

% Exemple: Creant un array distribuït
D = distributed.rand(10000, 10000);
  • Explicació: Aquest codi crea un gran array distribuït de nombres aleatoris.

Consideracions de Rendiment

La computació paral·lela pot accelerar significativament els teus càlculs, però és important considerar la sobrecàrrega i la naturalesa de les teves tasques.

Punts Clau

  • Sobrecàrrega: Iniciar un pool paral·lel i distribuir tasques pot introduir sobrecàrrega.
  • Granularitat de les Tasques: Les tasques de granulat fi poden no beneficiar-se gaire del paral·lelisme a causa de la sobrecàrrega.
  • Transferència de Dades: Transferir grans quantitats de dades entre treballadors pot ser lent.

Consells de Rendiment

  • Utilitza parfor per a bucles amb iteracions independents.
  • Utilitza parfeval per a tasques que poden executar-se de manera asíncrona.
  • Utilitza arrays distribuïts per a grans conjunts de dades.
  • Profila el teu codi per identificar colls d'ampolla.

Conclusió

La computació paral·lela a MATLAB pot millorar enormement el rendiment del teu codi aprofitant múltiples nuclis i màquines. Comprenent i utilitzant construccions com parfor, parfeval, spmd i arrays distribuïts, pots distribuir eficientment els teus càlculs i gestionar grans conjunts de dades. Considera sempre la sobrecàrrega i la naturalesa de les teves tasques per maximitzar els beneficis de la computació paral·lela.
