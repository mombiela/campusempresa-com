Introducció a les Estructures

Les estructures a MATLAB són tipus de dades que agrupen dades relacionades utilitzant contenidors de dades anomenats camps. Cada camp pot contenir qualsevol tipus de dades, incloent-hi arrays, cadenes de text, o fins i tot altres estructures. Això fa que les estructures siguin una manera versàtil de gestionar dades complexes.

Conceptes Clau

  • Noms de Camps: Identificadors per a les dades emmagatzemades en una estructura.
  • Notació de Punt: Utilitzada per accedir a les dades dels camps d'una estructura.
  • Estructures Imbricades: Estructures que contenen altres estructures com a camps.

Creació d'Estructures

Per crear una estructura, pots utilitzar la funció struct o l'assignació directa utilitzant la notació de punt.

Utilitzant la Funció `struct`

student = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);

En aquest exemple:

  • Name, Age, i GPA són noms de camps.
  • 'John Doe', 21, i 3.5 són els valors corresponents.

Utilitzant la Notació de Punt

student.Name = 'John Doe';
student.Age = 21;
student.GPA = 3.5;

Aquest mètode aconsegueix el mateix resultat que utilitzant la funció struct.

Accés i Modificació de Dades en Estructures

Pots accedir i modificar les dades en una estructura utilitzant la notació de punt.

Accés a les Dades

name = student.Name;  % Recupera 'John Doe'
age = student.Age;    % Recupera 21
gpa = student.GPA;    % Recupera 3.5

Modificació de Dades

student.GPA = 3.8;  % Actualitza el GPA a 3.8

Estructures Imbricades

Les estructures poden contenir altres estructures, permetent una organització de dades més complexa.

Creació d'Estructures Imbricades

student.Address = struct('Street', '123 Main St', 'City', 'Springfield', 'Zip', '12345');

Accés a les Dades en Estructures Imbricades

city = student.Address.City;  % Recupera 'Springfield'

Arrays d'Estructures

Pots crear arrays d'estructures per gestionar múltiples registres.

Creació d'un Array d'Estructures

students(1) = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);
students(2) = struct('Name', 'Jane Smith', 'Age', 22, 'GPA', 3.7);

Accés a les Dades en un Array d'Estructures

name1 = students(1).Name;  % Recupera 'John Doe'
gpa2 = students(2).GPA;    % Recupera 3.7

Operacions Avançades amb Estructures

MATLAB proporciona funcions per manipular estructures de manera més eficient.

Fieldnames i Structfun

  • fieldnames: Retorna un array de cel·les amb els noms dels camps.
  • structfun: Aplica una funció a cada camp d'una estructura.

Exemple

fields = fieldnames(student);  % Retorna {'Name', 'Age', 'GPA', 'Address'}
gpaSum = structfun(@(x) sum([students.GPA]), students);  % Suma tots els GPAs

Resum

Les estructures a MATLAB són eines poderoses per organitzar i gestionar dades complexes. Permeten un emmagatzematge flexible de dades amb camps anomenats, suport per a estructures imbricades, i la capacitat de crear arrays d'estructures. Entendre com crear, accedir i manipular estructures és essencial per a una gestió eficient de dades a MATLAB.
