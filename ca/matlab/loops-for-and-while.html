Bucles: for i while
Creació de Funcions ►

Introducció

Els bucles són constructes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. A MATLAB, els dos tipus principals de bucles són els bucles for i els bucles while. Aquesta secció et guiarà a través dels conceptes bàsics d'aquests bucles, proporcionarà exemples i oferirà exercicis per practicar.

Bucles for

Sintaxi Bàsica

Un bucle for a MATLAB repeteix un grup d'instruccions un nombre fix i predeterminat de vegades. La sintaxi és la següent:

for index = startValue:endValue
    % Instruccions a executar
end

Exemple

Comencem amb un exemple senzill on imprimim números de l'1 al 5:

for i = 1:5
    disp(i)
end

Explicació

  • i = 1:5 inicialitza la variable del bucle i per anar de l'1 al 5.
  • disp(i) mostra el valor actual de i en cada iteració.

Bucles for Niuats

També pots niuar bucles for per manejar tasques més complexes. Per exemple, crear una taula de multiplicar:

for i = 1:5
    for j = 1:5
        fprintf('%d * %d = %d\n', i, j, i*j);
    end
end

Explicació

  • El bucle extern va de l'1 al 5.
  • El bucle intern també va de l'1 al 5.
  • fprintf imprimeix el producte de i i j.

Bucles while

Sintaxi Bàsica

Un bucle while a MATLAB repeteix un grup d'instruccions un nombre indefinit de vegades mentre la condició es mantingui certa. La sintaxi és:

while condition
    % Instruccions a executar
end

Exemple

Vegem un exemple on imprimim números de l'1 al 5 utilitzant un bucle while:

i = 1;
while i <= 5
    disp(i)
    i = i + 1;
end

Explicació

  • i = 1 inicialitza la variable del bucle i.
  • while i <= 5 comprova si i és menor o igual a 5.
  • disp(i) mostra el valor actual de i.
  • i = i + 1 incrementa i en 1 en cada iteració.

Bucles Infinits

Tingues cura amb els bucles while ja que poden crear bucles infinits si la condició mai es torna falsa. Per exemple:

i = 1;
while true
    disp(i)
    i = i + 1;
    if i > 5
        break;
    end
end

Explicació

  • while true crea un bucle infinit.
  • if i > 5 comprova si i és més gran que 5.
  • break surt del bucle quan i és més gran que 5.

Comparació de bucles for i while

| Característica | Bucle for | Bucle while | |----------------|-----------------------------------|------------------------------------| | Cas d'Ús | Nombre conegut d'iteracions | Nombre desconegut d'iteracions | | Inicialització | Dins la definició del bucle | Fora del bucle | | Condició | Definida pel rang | Definida per una condició lògica | | Trampa Comuna | Cap | Risc de bucles infinits |

Exercicis

Exercici 1: Suma de Nombres

Escriu un bucle for per calcular la suma dels nombres de l'1 al 10.

sum = 0;
for i = 1:10
    sum = sum + i;
end
disp(sum)

Exercici 2: Càlcul del Factorial

Escriu un bucle while per calcular el factorial d'un nombre donat n.

n = 5; % Pots canviar aquest valor per provar amb diferents nombres
factorial = 1;
i = 1;
while i <= n
    factorial = factorial * i;
    i = i + 1;
end
disp(factorial)

Conclusió

Entendre i utilitzar els bucles de manera efectiva és crucial per a una programació eficient a MATLAB. Els bucles for són ideals quan el nombre d'iteracions es coneix per endavant, mentre que els bucles while són més adequats per a situacions on el nombre d'iteracions depèn d'una condició. Practica utilitzant ambdós tipus de bucles per convertir-te en un expert en la seva aplicació.
