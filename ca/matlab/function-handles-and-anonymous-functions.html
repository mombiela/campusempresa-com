Introducció

A MATLAB, les funcions manejadores i les funcions anònimes són eines poderoses que et permeten crear codi flexible i reutilitzable. Aquesta secció et guiarà a través dels conceptes bàsics de les funcions manejadores i les funcions anònimes, i com utilitzar-les de manera efectiva en els teus programes MATLAB.

Funcions manejadores

Què és una funció manejadora?

  • Una funció manejadora és un tipus de dada de MATLAB que representa una funció.
  • Et permet passar funcions com a arguments a altres funcions, emmagatzemar-les en variables i cridar-les dinàmicament.

Crear funcions manejadores

  • Utilitza el símbol @ seguit del nom de la funció per crear una funció manejadora.
% Exemple de creació d'una funció manejadora
f = @sin;

Utilitzar funcions manejadores

  • Pots cridar la funció utilitzant la manejadora igual que cridaries la funció directament.
% Utilitzar la funció manejadora per cridar la funció sin
result = f(pi/2);  % el resultat és 1

Passar funcions manejadores com a arguments

  • Les funcions manejadores es poden passar com a arguments a altres funcions, permetent funcions d'ordre superior.
% Exemple de funció que pren una funció manejadora com a argument
function y = applyFunction(f, x)
    y = f(x);
end

% Utilitzar la funció applyFunction amb una funció manejadora
result = applyFunction(@cos, pi);  % el resultat és -1

Funcions anònimes

Què és una funció anònima?

  • Una funció anònima és una funció d'una sola línia que no requereix un fitxer separat.
  • Es defineix utilitzant el símbol @ seguit d'una llista d'arguments d'entrada i l'expressió de la funció.

Crear funcions anònimes

  • Defineix una funció anònima utilitzant la següent sintaxi:
% Sintaxi per crear una funció anònima
f = @(arg1, arg2, ...) expressió;

% Exemple d'una funció anònima
square = @(x) x^2;

Utilitzar funcions anònimes

  • Crida una funció anònima igual que qualsevol altra funció.
% Utilitzar la funció anònima
result = square(4);  % el resultat és 16

Funcions anònimes amb múltiples entrades

  • Les funcions anònimes poden tenir múltiples entrades.
% Funció anònima amb múltiples entrades
add = @(x, y) x + y;

% Utilitzar la funció anònima amb múltiples entrades
result = add(3, 5);  % el resultat és 8

Funcions anònimes sense entrades

  • Les funcions anònimes també poden no tenir entrades.
% Funció anònima sense entrades
greet = @() disp('Hello, World!');

% Utilitzar la funció anònima sense entrades
greet();  % Mostra 'Hello, World!'

Combinació de funcions manejadores i funcions anònimes

Funcions d'ordre superior

  • Combina funcions manejadores i funcions anònimes per crear funcions d'ordre superior.
% Exemple de funció d'ordre superior
function result = operate(f, x, y)
    result = f(x, y);
end

% Utilitzar la funció d'ordre superior amb una funció anònima
result = operate(@(a, b) a * b, 3, 4);  % el resultat és 12

Emmagatzemar funcions anònimes en variables

  • Emmagatzema múltiples funcions anònimes en una cel·la o estructura per a un accés organitzat.
% Emmagatzemar funcions anònimes en una cel·la
funcs = {@(x) x^2, @(x) x^3, @(x) x^4};

% Accedir i utilitzar les funcions anònimes
result1 = funcs{1}(2);  % el resultat és 4
result2 = funcs{2}(2);  % el resultat és 8
result3 = funcs{3}(2);  % el resultat és 16

Conclusió

Les funcions manejadores i les funcions anònimes a MATLAB proporcionen una manera flexible de crear i gestionar funcions. Et permeten escriure codi més modular i reutilitzable, que és especialment útil en programes complexos. Comprenent i utilitzant aquestes característiques, pots millorar les teves habilitats de programació a MATLAB i crear codi més eficient i dinàmic.
