Introducció

La gestió d'errors i la depuració són habilitats essencials per a qualsevol programador de MATLAB. Aquesta secció et guiarà a través dels conceptes bàsics per identificar, gestionar i depurar errors en el teu codi MATLAB, progressant des de tècniques per a principiants fins a tècniques avançades.

Tipus d'Errors

Entendre els tipus d'errors és el primer pas per a una gestió i depuració d'errors efectiva.

  • Errors de Sintaxi: Errors en el codi que violen les regles del llenguatge MATLAB.
  • Errors d'Execució: Errors que ocorren durant l'execució del programa.
  • Errors Lògics: Errors que fan que el programa funcioni incorrectament però no necessàriament facin que el programa es bloquegi.

Identificació d'Errors

Errors de Sintaxi

Els errors de sintaxi són els més fàcils d'identificar, ja que MATLAB els destacarà i proporcionarà missatges d'error.

% Exemple d'un error de sintaxi
x = 5
y = 10;
z = x + y
disp(z)

En el codi anterior, la falta de punt i coma després de x = 5 causarà un error de sintaxi.

Errors d'Execució

Els errors d'execució ocorren mentre el programa s'està executant i es poden identificar utilitzant els missatges d'error de MATLAB.

% Exemple d'un error d'execució
a = 10;
b = 0;
c = a / b; % Error de divisió per zero

Errors Lògics

Els errors lògics són els més difícils d'identificar, ja que no produeixen missatges d'error però resulten en una sortida incorrecta.

% Exemple d'un error lògic
x = 5;
y = 10;
z = x - y; % Es pretenia sumar x i y, però s'ha restat per error
disp(z)

Tècniques de Depuració

Utilitzant l'Editor de MATLAB

L'Editor de MATLAB proporciona diverses eines per ajudar a depurar el teu codi.

  • Punts de Trencament: Estableix punts de trencament per pausar l'execució en línies específiques.
  • Passar pel Codi: Utilitza les funcions de passar dins, passar fora i passar per sobre per executar el teu codi línia per línia.
  • Supervisar Variables: Monitoritza els valors de les variables mentre passes pel codi.
% Exemple d'ús de punts de trencament
x = 5;
y = 10;
z = x + y; % Estableix un punt de trencament aquí
disp(z)

Funcions de la Finestra de Comandes

MATLAB proporciona diverses funcions per ajudar amb la depuració a la Finestra de Comandes.

  • dbstop: Estableix punts de trencament.
  • dbclear: Esborra punts de trencament.
  • dbstep: Passa pel codi.
  • dbcont: Continua l'execució.
  • dbquit: Surt del mode de depuració.
% Exemple d'ús de funcions de la Finestra de Comandes
dbstop if error
a = 10;
b = 0;
c = a / b; % Això activarà el punt de trencament

Gestió d'Errors

Utilitzant `try` i `catch`

Els blocs try i catch et permeten gestionar errors de manera elegant sense que el programa es bloquegi.

% Exemple de try i catch
try
    a = 10;
    b = 0;
    c = a / b; % Això causarà un error
catch ME
    disp('S'ha produït un error:');
    disp(ME.message);
end

Missatges d'Error Personalitzats

Pots crear missatges d'error personalitzats utilitzant la funció error.

% Exemple de missatge d'error personalitzat
x = -5;
if x < 0
    error('x ha de ser un nombre no negatiu');
end

Tècniques Avançades de Depuració

Utilitzant la Funció `debug`

La funció debug et permet executar un script en mode de depuració.

% Exemple d'ús de la funció debug
debug('myScript')

Perfilant el Codi

El perfilat t'ajuda a entendre el rendiment del teu codi i identificar colls d'ampolla.

% Exemple de perfilat de codi
profile on
myFunction();
profile viewer

Conclusió

La gestió d'errors i la depuració són crucials per escriure codi MATLAB robust i eficient. Entenent els tipus d'errors, utilitzant eines de depuració i implementant tècniques de gestió d'errors, pots millorar significativament les teves habilitats de programació i la qualitat del teu codi. Practica aquestes tècniques regularment per convertir-te en un expert en identificar i resoldre problemes en els teus programes MATLAB.
