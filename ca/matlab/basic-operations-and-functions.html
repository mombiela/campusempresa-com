Introducció

En aquesta secció, cobrirem les operacions i funcions fonamentals en MATLAB. Això inclou operacions aritmètiques bàsiques, manipulacions de matrius i la creació i ús de funcions. Al final d'aquesta secció, seràs capaç de realitzar càlculs essencials i escriure funcions simples en MATLAB.

Operacions aritmètiques bàsiques

MATLAB està dissenyat per a càlculs numèrics, i entendre les operacions aritmètiques bàsiques és crucial.

  • Suma (+)
  • Resta (-)
  • Multiplicació (*)
  • Divisió (/ i \)
  • Exponenciació (^)

Exemples

a = 5;
b = 3;

% Suma
c = a + b; % c = 8

% Resta
d = a - b; % d = 2

% Multiplicació
e = a * b; % e = 15

% Divisió
f = a / b; % f ≈ 1.6667
g = a \ b; % g = 0.6 (equivalent a b / a)

% Exponenciació
h = a ^ b; % h = 125

Operacions amb matrius

MATLAB excel·leix en operacions amb matrius, que són fonamentals en moltes aplicacions científiques i d'enginyeria.

  • Suma i resta de matrius
  • Multiplicació de matrius
  • Operacions element a element
  • Transposició

Exemples

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];

% Suma de matrius
C = A + B; % C = [6, 8; 10, 12]

% Resta de matrius
D = A - B; % D = [-4, -4; -4, -4]

% Multiplicació de matrius
E = A * B; % E = [19, 22; 43, 50]

% Multiplicació element a element
F = A .* B; % F = [5, 12; 21, 32]

% Transposició
G = A'; % G = [1, 3; 2, 4]

Funcions integrades

MATLAB proporciona una vasta biblioteca de funcions integrades per a diverses tasques, incloent càlculs matemàtics, anàlisi de dades i visualització.

  • Funcions matemàtiques: sum, mean, max, min, sqrt, abs
  • Funcions trigonomètriques: sin, cos, tan, asin, acos, atan
  • Funcions estadístiques: median, std, var

Exemples

% Funcions matemàtiques
x = [1, 2, 3, 4, 5];

total = sum(x); % total = 15
mitjana = mean(x); % mitjana = 3
màxim = max(x); % màxim = 5
mínim = min(x); % mínim = 1
arrel_quadrada = sqrt(x); % arrel_quadrada = [1, 1.4142, 1.7321, 2, 2.2361]
absolut = abs([-1, -2, -3]); % absolut = [1, 2, 3]

% Funcions trigonomètriques
angle = pi / 4; % 45 graus

seno = sin(angle); % seno ≈ 0.7071
coseno = cos(angle); % coseno ≈ 0.7071
tangent = tan(angle); % tangent = 1

Creació de funcions

Crear funcions personalitzades en MATLAB et permet encapsular codi per a la seva reutilització i millor organització.

Sintaxi

function [sortida1, sortida2, ...] = nomFuncio(entrada1, entrada2, ...)
    % Descripció de la funció
    % Explicació detallada aquí

    % Codi de la funció
    sortida1 = ...;
    sortida2 = ...;
end

Exemple

function [àrea, circumferència] = propietatsCercle(radi)
    % propietatsCercle calcula l'àrea i la circumferència d'un cercle
    % donat el seu radi.

    % Àrea del cercle
    àrea = pi * radi^2;

    % Circumferència del cercle
    circumferència = 2 * pi * radi;
end

% Ús
[àreaRadi, circumferènciaRadi] = propietatsCercle(5);
% àreaRadi = 78.5398
% circumferènciaRadi = 31.4159

Conclusió

En aquesta secció, hem cobert les bases de les operacions i funcions en MATLAB. Has après com realitzar operacions aritmètiques i amb matrius, utilitzar funcions integrades i crear funcions personalitzades. Aquestes habilitats fonamentals són essencials per a temes més avançats en la programació amb MATLAB. Practica aquests conceptes amb diversos exemples per solidificar la teva comprensió i preparar-te per a tasques més complexes.
