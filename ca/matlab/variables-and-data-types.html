Introducció

En aquesta secció, cobrirem els conceptes bàsics de les variables i els tipus de dades en MATLAB. Entendre aquests conceptes és fonamental per escriure codi MATLAB eficient i efectiu. Començarem amb els conceptes bàsics de les variables, passarem a diferents tipus de dades i després explorarem temes més avançats.

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden utilitzar i manipular al llarg del teu codi MATLAB.

Creació de Variables

  • Per crear una variable, simplement assigna-li un valor utilitzant l'operador =.
  • Els noms de les variables han de començar amb una lletra, seguida de lletres, números o guions baixos.
x = 5;          % Crea una variable x amb valor 5
y = 3.14;       % Crea una variable y amb valor 3.14
nom = 'John';   % Crea una variable nom amb la cadena 'John'

Convencions de Nomenclatura de Variables

  • Utilitza noms descriptius per a les variables.
  • Evita utilitzar paraules reservades de MATLAB (per exemple, if, else, end).
total_sum = 100;  % Nom de variable descriptiu

Tipus de Dades

MATLAB suporta diversos tipus de dades, cadascun adequat per a diferents tipus de dades i operacions.

Tipus de Dades Numèriques

  • Double: Tipus de dada numèrica per defecte en MATLAB.
  • Enter: MATLAB suporta diversos tipus d'enters (per exemple, int8, int16, int32, int64).
a = 10;        % Double
b = int32(10); % Enter de 32 bits

Tipus de Dades de Caràcter i Cadena

  • Arrays de Caràcters: S'utilitzen cometes simples per crear arrays de caràcters.
  • Arrays de Cadenes: S'utilitzen cometes dobles per crear arrays de cadenes.
charArray = 'Hello';  % Array de caràcters
strArray = "Hello";   % Array de cadenes

Tipus de Dades Lògiques

  • Els tipus de dades lògiques s'utilitzen per a valors vertader/fals.
isTrue = true;   % Lògic vertader
isFalse = false; % Lògic fals

Nombres Complexos

  • MATLAB suporta nombres complexos amb parts reals i imaginàries.
z = 3 + 4i;  % Nombre complex

Arrays i Matrius

  • Els arrays i les matrius són fonamentals en MATLAB, que significa "Laboratori de Matrius".
array = [1, 2, 3, 4];       % Array de 1x4
matriu = [1, 2; 3, 4];      % Matriu de 2x2

Arrays de Cel·les i Estructures

  • Arrays de Cel·les: Poden contenir diferents tipus de dades en cada cel·la.
  • Estructures: S'utilitzen per agrupar dades relacionades utilitzant camps amb nom.
cellArray = {1, 'text', [2, 3]};  % Array de cel·les
structVar = struct('field1', 1, 'field2', 'text');  % Estructura

Conversió de Tipus

MATLAB permet convertir entre diferents tipus de dades utilitzant funcions integrades.

num = 5.67;
intNum = int32(num);  % Converteix double a int32
strNum = num2str(num);  % Converteix nombre a cadena

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en MATLAB. Hem après a crear variables, hem explorat diferents tipus de dades incloent numèriques, de caràcter, lògiques, nombres complexos, arrays, matrius, arrays de cel·les i estructures. També hem discutit la conversió de tipus. Entendre aquests conceptes és essencial per a una programació efectiva en MATLAB i servirà com a base per a temes més avançats.

En dominar les variables i els tipus de dades, estaràs ben equipat per gestionar dades de manera eficient i escriure codi MATLAB robust.
