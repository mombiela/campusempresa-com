Introducció

MATLAB (Matrix Laboratory) és un llenguatge d'alt rendiment per al càlcul tècnic. Integra càlcul, visualització i programació en un entorn fàcil d'utilitzar. Aquesta secció cobrirà les comandes bàsiques i la sintaxi necessària per començar amb MATLAB.

Comandes bàsiques

Entendre les comandes bàsiques és essencial per navegar per MATLAB i realitzar operacions simples.

  • clc: Neteja la finestra de comandes.
  • clear: Elimina les variables de l'espai de treball.
  • close all: Tanca totes les finestres de figures.
  • who: Llista les variables actuals a l'espai de treball.
  • whos: Llista les variables actuals juntament amb els seus detalls.
  • help: Proporciona ajuda per a les funcions de MATLAB.
  • doc: Obre la documentació per a les funcions de MATLAB.

Exemple

clc;          % Neteja la finestra de comandes
clear;        % Neteja totes les variables de l'espai de treball
close all;    % Tanca totes les finestres de figures

Variables i tipus de dades

Les variables a MATLAB s'utilitzen per emmagatzemar dades, i MATLAB admet diversos tipus de dades.

  • Tipus numèrics: double, single, int8, int16, int32, int64, uint8, uint16, uint32, uint64
  • Caràcters i cadenes: char, string
  • Lògic: true, false
  • Arrays i matrius: Fonamentals per a MATLAB

Exemple

a = 5;                % Enter
b = 3.14;             % Número de punt flotant
c = 'Hello';          % Array de caràcters (cadena)
d = true;             % Lògic cert
e = [1, 2, 3; 4, 5, 6]; % Matriu de 2x3

Operacions bàsiques

MATLAB admet una varietat d'operacions bàsiques, incloent-hi aritmètiques, relacionals i lògiques.

Operacions aritmètiques

  • Suma: +
  • Resta: -
  • Multiplicació: *
  • Divisió: /
  • Exponenciació: ^

Exemple

x = 10;
y = 3;
sum = x + y;          % Suma
difference = x - y;   % Resta
product = x * y;      % Multiplicació
quotient = x / y;     % Divisió
power = x ^ y;        % Exponenciació

Operacions relacionals

  • Igual a: ==
  • No igual a: ~=
  • Major que: >
  • Menor que: <
  • Major o igual a: >=
  • Menor o igual a: <=

Exemple

a = 5;
b = 10;
isEqual = (a == b);          % Fals
isNotEqual = (a ~= b);       % Cert
isGreater = (a > b);         % Fals
isLess = (a < b);            % Cert
isGreaterOrEqual = (a >= b); % Fals
isLessOrEqual = (a <= b);    % Cert

Operacions lògiques

  • I: &
  • O: |
  • NO: ~

Exemple

p = true;
q = false;
andResult = p & q;   % Fals
orResult = p | q;    % Cert
notResult = ~p;      % Fals

Flux de control

Les instruccions de flux de control et permeten controlar l'execució del codi basant-te en condicions.

Instruccions If-Else

x = 10;
if x > 5
    disp('x és més gran que 5');
else
    disp('x no és més gran que 5');
end

Instruccions Switch-Case

choice = 'A';
switch choice
    case 'A'
        disp('La tria és A');
    case 'B'
        disp('La tria és B');
    otherwise
        disp('Tria desconeguda');
end

Bucles

Bucle For

for i = 1:5
    disp(['Iteració: ', num2str(i)]);
end

Bucle While

i = 1;
while i <= 5
    disp(['Iteració: ', num2str(i)]);
    i = i + 1;
end

Funcions

Les funcions a MATLAB et permeten encapsular codi per a la seva reutilització i millor organització.

Definició d'una funció

function result = addNumbers(a, b)
    result = a + b;
end

Crida a una funció

sum = addNumbers(3, 4);  % Crida a la funció i emmagatzema el resultat a 'sum'
disp(sum);               % Mostra 7

Conclusió

Aquesta secció ha introduït les comandes bàsiques i la sintaxi a MATLAB, incloent-hi la declaració de variables, operacions bàsiques, flux de control i funcions. Dominar aquests conceptes bàsics és crucial per avançar cap a temes més avançats a MATLAB. Practica aquestes comandes i intenta escriure scripts simples per consolidar la teva comprensió.
