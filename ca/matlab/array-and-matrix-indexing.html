Indexació d'Arrays i Matrius a MATLAB

Introducció

La indexació d'arrays i matrius és un concepte fonamental a MATLAB que et permet accedir i manipular elements dins d'arrays i matrius. Aquest tema cobreix els conceptes bàsics de la indexació, tècniques avançades de indexació i exemples pràctics per ajudar-te a ser competent en la gestió d'arrays i matrius a MATLAB.

Indexació Bàsica

  • Arrays Unidimensionals (Vectors):
    • Accedeix als elements utilitzant parèntesis ().
    • La indexació comença a 1 (no a 0 com en molts altres llenguatges de programació).
% Crear un vector fila
row_vector = [10, 20, 30, 40, 50];

% Accedir al tercer element
third_element = row_vector(3); % Sortida: 30
  • Arrays Bidimensionals (Matrius):
    • Utilitza índexs de fila i columna per accedir als elements.
    • Sintaxi: matrix(row, column).
% Crear una matriu 3x3
matrix = [1, 2, 3; 4, 5, 6; 7, 8, 9];

% Accedir a l'element a la segona fila, tercera columna
element = matrix(2, 3); % Sortida: 6

Tècniques Avançades de Indexació

  • Operador de Doble Punt (:):
    • Utilitzat per accedir a un rang d'elements o a files/columnes completes.
% Accedir a tots els elements de la segona fila
second_row = matrix(2, :); % Sortida: [4, 5, 6]

% Accedir a tots els elements de la tercera columna
third_column = matrix(:, 3); % Sortida: [3; 6; 9]
  • Indexació Lògica:
    • Utilitza condicions lògiques per indexar arrays.
% Crear un array lògic on els elements > 4
logical_index = matrix > 4;

% Utilitzar l'array lògic per indexar la matriu
filtered_elements = matrix(logical_index); % Sortida: [5, 6, 7, 8, 9]
  • Indexació Lineal:
    • Tracta la matriu com un únic vector columna.
% Accedir al cinquè element en ordre de columnes
linear_element = matrix(5); % Sortida: 5

Modificació d'Elements

  • Modificació d'un Sol Element:
% Modificar l'element a la primera fila, segona columna
matrix(1, 2) = 20; % la matriu es converteix en [1, 20, 3; 4, 5, 6; 7, 8, 9]
  • Modificació de Múltiples Elements:
% Modificar una fila completa
matrix(2, :) = [40, 50, 60]; % la matriu es converteix en [1, 20, 3; 40, 50, 60; 7, 8, 9]

Funcions Especials de Indexació

  • Paraula Clau end:
    • Representa l'últim índex en una dimensió.
% Accedir a l'últim element del vector
last_element = row_vector(end); % Sortida: 50

% Accedir a l'últim element de la primera fila
last_element_row = matrix(1, end); % Sortida: 3
  • sub2ind i ind2sub:
    • Convertir entre índexs de subscript i lineals.
% Convertir subscript a índex lineal
linear_index = sub2ind(size(matrix), 3, 2); % Sortida: 8

% Convertir índex lineal a subscript
[row, col] = ind2sub(size(matrix), 8); % Sortida: row = 3, col = 2

Exemples Pràctics

  • Exemple 1: Extracció d'una Submatriu
% Extreure una submatriu 2x2 de la cantonada superior esquerra
submatrix = matrix(1:2, 1:2); % Sortida: [1, 20; 40, 50]
  • Exemple 2: Reemplaçament d'Elements Basat en una Condició
% Reemplaçar tots els elements majors que 5 amb 0
matrix(matrix > 5) = 0; % la matriu es converteix en [1, 20, 3; 40, 50, 0; 7, 8, 9]

Conclusió

Entendre la indexació d'arrays i matrius a MATLAB és crucial per a una manipulació i anàlisi eficient de dades. Dominant les tècniques bàsiques i avançades de indexació, pots realitzar operacions complexes amb facilitat. Practica aquests conceptes amb diversos exemples per consolidar la teva comprensió i convertir-te en un expert en programació MATLAB.
