Arrays de cel·les a MATLAB
Estructures ►

Els arrays de cel·les són un tipus de dada versàtil a MATLAB que et permeten emmagatzemar arrays de diferents mides i tipus. Això els fa particularment útils per gestionar dades heterogènies. En aquesta secció, cobrirem els conceptes bàsics dels arrays de cel·les, com crear-los i manipular-los, i algunes tècniques avançades.

Introducció als Arrays de Cel·les

  • Definició: Un array de cel·les és un tipus de dada amb contenidors de dades indexats anomenats cel·les, on cada cel·la pot contenir qualsevol tipus de dada.
  • Usos: Útils per emmagatzemar una col·lecció de diferents tipus de dades, com ara cadenes de text, números i altres arrays.

Creació d'Arrays de Cel·les

Utilitzant la Funció `cell`

La funció cell crea un array de cel·les buit de la mida especificada.

C = cell(3, 2); % Crea un array de cel·les de 3x2

Assignació Directa

Pots assignar directament valors a les cel·les utilitzant claus {}.

C{1, 1} = 42;          % Assigna un número
C{1, 2} = 'Hola';      % Assigna una cadena de text
C{2, 1} = [1, 2, 3];   % Assigna un array numèric
C{2, 2} = {1, 'a', 3}; % Assigna un array de cel·les

Combinació de Diferents Tipus de Dades

C = {42, 'Hola'; [1, 2, 3], {1, 'a', 3}};

Accés a les Dades en Arrays de Cel·les

  • Utilitzant Claus {}: Per accedir al contingut d'una cel·la.
  • Utilitzant Parèntesis (): Per accedir a la cel·la mateixa.

Exemples

% Accés al contingut
value = C{1, 1}; % Retorna 42

% Accés a la cel·la
cellContent = C(1, 1); % Retorna {42}

Manipulació d'Arrays de Cel·les

Afegir Elements

Pots afegir elements a un array de cel·les assignant un valor a una nova cel·la.

C{3, 1} = 'Nou Element';

Eliminar Elements

Pots eliminar elements assignant un array buit [] a la cel·la.

C{1, 1} = [];

Redimensionar Arrays de Cel·les

Els arrays de cel·les es poden redimensionar dinàmicament assignant valors fora de les dimensions actuals.

C{4, 3} = 'Un altre Element'; % Expandeix l'array de cel·les a 4x3

Tècniques Avançades

Funcions de Cel·les

MATLAB proporciona diverses funcions per treballar amb arrays de cel·les:

  • cellfun: Aplica una funció a cada cel·la.
  • celldisp: Mostra el contingut d'un array de cel·les.
  • cell2mat: Converteix un array de cel·les de matrius en una única matriu.

Exemple: Utilitzant `cellfun`

C = {1, 2, 3; 4, 5, 6};
result = cellfun(@(x) x^2, C); % Eleva al quadrat cada element de l'array de cel·les

Exemple: Utilitzant `cell2mat`

C = {1, 2; 3, 4};
matrix = cell2mat(C); % Converteix a una matriu de 2x2

Resum

Els arrays de cel·les a MATLAB són una eina poderosa per gestionar dades heterogènies. Et permeten emmagatzemar diferents tipus i mides de dades en un únic contenidor. Entendre com crear, accedir i manipular arrays de cel·les és essencial per a la gestió avançada de dades a MATLAB. En dominar els arrays de cel·les, podràs escriure codi MATLAB més flexible i eficient.
