Project 4: Aplicació d'Aprenentatge Automàtic

Introducció

En aquest projecte, desenvoluparem una aplicació d'aprenentatge automàtic utilitzant MATLAB. Aquest projecte et guiarà a través del procés de preprocessament de dades, entrenament del model, avaluació i desplegament. Al final d'aquest projecte, tindràs una comprensió sòlida de com implementar algoritmes d'aprenentatge automàtic en MATLAB.

Conceptes Clau

  • Preprocessament de Dades: Neteja i preparació de dades per a l'anàlisi.
  • Enginyeria de Característiques: Creació de noves característiques o modificació de les existents per millorar el rendiment del model.
  • Entrenament del Model: Utilització d'algoritmes per aprendre a partir de les dades.
  • Avaluació del Model: Avaluació del rendiment del model entrenat.
  • Desplegament del Model: Implementació del model per a ús pràctic.

Preprocessament de Dades

El preprocessament de dades és un pas crucial en qualsevol projecte d'aprenentatge automàtic. Implica netejar les dades, gestionar els valors perduts i normalitzar les característiques.

Exemple: Carregar i Netejar Dades

% Carregar el conjunt de dades
data = readtable('data.csv');

% Mostrar les primeres files del conjunt de dades
head(data)

% Gestionar els valors perduts eliminant les files amb dades perdudes
data = rmmissing(data);

% Normalitzar les característiques
dataNorm = normalize(data);

Enginyeria de Característiques

L'enginyeria de característiques implica crear noves característiques o modificar les existents per millorar el rendiment del model d'aprenentatge automàtic.

Exemple: Crear Noves Característiques

% Crear una nova característica combinant característiques existents
data.NewFeature = data.Feature1 .* data.Feature2;

% Mostrar les primeres files amb la nova característica
head(data)

Entrenament del Model

L'entrenament del model implica seleccionar un algoritme adequat i utilitzar-lo per aprendre a partir de les dades.

Exemple: Entrenar un Classificador d'Arbre de Decisió

% Dividir les dades en conjunts d'entrenament i prova
cv = cvpartition(height(data), 'HoldOut', 0.3);
trainData = data(training(cv), :);
testData = data(test(cv), :);

% Entrenar un classificador d'arbre de decisió
treeModel = fitctree(trainData, 'ResponseVar', 'Class');

% Mostrar el model entrenat
view(treeModel, 'Mode', 'graph')

Avaluació del Model

L'avaluació del model és el procés d'avaluar el rendiment del model entrenat utilitzant diverses mètriques.

Exemple: Avaluar el Model

% Predir les etiquetes de classe per al conjunt de prova
predictions = predict(treeModel, testData);

% Calcular l'exactitud
accuracy = sum(predictions == testData.Class) / length(testData.Class);
fprintf('Exactitud del Model: %.2f%%\n', accuracy * 100);

% Matriu de confusió
confMat = confusionmat(testData.Class, predictions);
disp('Matriu de Confusió:');
disp(confMat);

Desplegament del Model

El desplegament del model implica implementar el model entrenat en una aplicació del món real.

Exemple: Guardar i Carregar el Model

% Guardar el model entrenat en un fitxer
save('treeModel.mat', 'treeModel');

% Carregar el model des del fitxer
loadedModel = load('treeModel.mat');
treeModel = loadedModel.treeModel;

% Utilitzar el model carregat per fer prediccions
newData = [1.5, 2.3, 3.1]; % Exemple de noves dades
newPrediction = predict(treeModel, newData);
fprintf('Classe Predita: %d\n', newPrediction);

Conclusió

En aquest projecte, hem cobert els passos essencials per desenvolupar una aplicació d'aprenentatge automàtic en MATLAB. Hem començat amb el preprocessament de dades, hem passat a l'enginyeria de característiques, després hem entrenat i avaluat un classificador d'arbre de decisió, i finalment hem discutit el desplegament del model. Seguint aquests passos, pots construir i desplegar els teus propis models d'aprenentatge automàtic utilitzant MATLAB.
