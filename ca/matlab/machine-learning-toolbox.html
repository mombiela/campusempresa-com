Introducció a la Caixa d'Eines d'Aprenentatge Automàtic

La Caixa d'Eines d'Aprenentatge Automàtic a MATLAB proporciona eines i funcions per realitzar diverses tasques d'aprenentatge automàtic. Aquesta secció cobrirà els conceptes bàsics de la caixa d'eines, incloent-hi la instal·lació i una visió general de les seves capacitats.

  • Instal·lació: Assegureu-vos que la Caixa d'Eines d'Aprenentatge Automàtic està instal·lada a través de l'Explorador d'Add-Ons de MATLAB.
  • Capacitats: Preprocessament de dades, classificació, regressió, agrupament i avaluació de models.

Preprocessament de Dades

El preprocessament de dades és un pas crucial en l'aprenentatge automàtic. Aquesta secció cobrirà tècniques per preparar les dades per a l'anàlisi.

  • Importació de Dades: Utilitzeu readtable, readmatrix i altres funcions per importar dades.
  • Neteja de Dades: Gestioneu valors perduts, valors atípics i normalització de dades.
  • Enginyeria de Característiques: Creeu noves característiques a partir de dades existents.

Exemple: Importació i Neteja de Dades

% Importar dades
data = readtable('data.csv');

% Mostrar les primeres files de la taula
head(data)

% Gestionar valors perduts omplint amb la mitjana
data = fillmissing(data, 'constant', mean(data, 'omitnan'));

% Normalitzar dades
data = normalize(data);

Classificació

La classificació és el procés de predir la classe de punts de dades donats. Aquesta secció cobrirà diversos algorismes de classificació disponibles a MATLAB.

  • Algorismes: Arbres de Decisió, Màquines de Suport Vectorial (SVM), k-Nearest Neighbors (k-NN) i Xarxes Neuronals.
  • Entrenament: Utilitzeu fitctree, fitcsvm, fitcknn i fitcnet per entrenar models.
  • Avaluació: Utilitzeu matrius de confusió i validació creuada per avaluar models.

Exemple: Entrenament d'un Arbre de Decisió

% Carregar dades de mostra
load fisheriris

% Dividir dades en conjunts d'entrenament i prova
cv = cvpartition(species, 'HoldOut', 0.3);
trainData = meas(training(cv), :);
trainLabels = species(training(cv));
testData = meas(test(cv), :);
testLabels = species(test(cv));

% Entrenar un classificador d'arbre de decisió
treeModel = fitctree(trainData, trainLabels);

% Predir sobre dades de prova
predictions = predict(treeModel, testData);

% Avaluar el model
confMat = confusionmat(testLabels, predictions);
disp('Matriu de Confusió:');
disp(confMat);

Regressió

L'anàlisi de regressió s'utilitza per predir una variable de sortida contínua basada en una o més característiques d'entrada. Aquesta secció cobrirà diverses tècniques de regressió.

  • Algorismes: Regressió Lineal, Regressió Polinòmica, Regressió de Suport Vectorial (SVR).
  • Entrenament: Utilitzeu fitlm, fitrsvm i altres funcions per entrenar models de regressió.
  • Avaluació: Utilitzeu mètriques com l'Error Quadràtic Mitjà (MSE) i el R-quadrat.

Exemple: Regressió Lineal

% Carregar dades de mostra
load carsmall

% Preparar dades
X = [Weight, Horsepower];
y = MPG;

% Entrenar un model de regressió lineal
lm = fitlm(X, y);

% Predir sobre noves dades
newData = [3500, 200]; % Exemple de noves dades
predictedMPG = predict(lm, newData);

% Mostrar el resultat
disp(['MPG Previst: ', num2str(predictedMPG)]);

Agrupament

L'agrupament és la tasca d'agrupament d'un conjunt d'objectes de manera que els objectes del mateix grup siguin més semblants entre ells que als d'altres grups.

  • Algorismes: k-Means, Agrupament Jeràrquic, Models de Mescla Gaussiana (GMM).
  • Entrenament: Utilitzeu kmeans, linkage i fitgmdist per a l'agrupament.
  • Avaluació: Utilitzeu gràfics de silueta i índexs de validesa de clústers.

Exemple: Agrupament k-Means

% Carregar dades de mostra
load fisheriris

% Realitzar agrupament k-means
numClusters = 3;
[idx, C] = kmeans(meas, numClusters);

% Gràfic dels clústers
figure;
gscatter(meas(:,1), meas(:,2), idx);
hold on;
plot(C(:,1), C(:,2), 'kx', 'MarkerSize', 15, 'LineWidth', 3);
title('Agrupament k-Means');
xlabel('Longitud del Sèpal');
ylabel('Amplada del Sèpal');
hold off;

Avaluació de Models i Ajust de Paràmetres

Avaluar i ajustar models d'aprenentatge automàtic és essencial per millorar el seu rendiment. Aquesta secció cobrirà diverses mètriques d'avaluació i tècniques per ajustar paràmetres.

  • Mètriques d'Avaluació: Precisió, Precisió, Record, F1-Score, MSE, R-quadrat.
  • Validació Creuada: Utilitzeu crossval i cvpartition per a la validació creuada.
  • Ajust de Paràmetres: Utilitzeu bayesopt, fitctree amb opcions de paràmetres.

Exemple: Validació Creuada i Ajust de Paràmetres

% Carregar dades de mostra
load fisheriris

% Definir el model i els paràmetres
treeTemplate = templateTree('MaxNumSplits', 1:10);

% Realitzar validació creuada
cvModel = fitcensemble(meas, species, 'Method', 'Bag', 'Learners', treeTemplate, 'CrossVal', 'on');

% Mostrar resultats de la validació creuada
cvLoss = kfoldLoss(cvModel);
disp(['Pèrdua de Validació Creuada: ', num2str(cvLoss)]);

Conclusió

La Caixa d'Eines d'Aprenentatge Automàtic a MATLAB proporciona un conjunt complet d'eines per al preprocessament de dades, classificació, regressió, agrupament i avaluació de models. Comprenent i utilitzant aquestes eines, podeu construir i avaluar efectivament models d'aprenentatge automàtic per a diverses aplicacions. Practiqueu amb els exemples proporcionats i exploreu més per aprofundir la vostra comprensió i competència en l'aprenentatge automàtic amb MATLAB.
