Llegir i Escriure Fitxers a MATLAB

Introducció

Llegir i escriure fitxers és una habilitat fonamental a MATLAB, essencial per a l'anàlisi de dades, guardar resultats i compartir informació. Aquesta secció et guiarà a través dels conceptes bàsics de l'E/S (Entrada/Sortida) de fitxers a MATLAB, progressant des de tècniques per a principiants fins a tècniques avançades.

Operacions Bàsiques amb Fitxers

Llegir Fitxers

MATLAB proporciona diverses funcions per llegir dades de fitxers. Les funcions més utilitzades són fopen, fclose, fread, fgets i fscanf.

Exemple: Llegir un Fitxer de Text

% Obrir el fitxer per llegir
fileID = fopen('example.txt', 'r');

% Comprovar si el fitxer s'ha obert correctament
if fileID == -1
    error('El fitxer no s\'ha pogut obrir.');
end

% Llegir el fitxer línia per línia
while ~feof(fileID)
    line = fgets(fileID);
    disp(line);
end

% Tancar el fitxer
fclose(fileID);
  • fopen: Obre el fitxer.
  • fgets: Llegeix una línia del fitxer.
  • feof: Comprova si s'ha arribat al final del fitxer.
  • fclose: Tanca el fitxer.

Escriure Fitxers

Escriure dades a fitxers és igualment important. Les funcions de MATLAB com fprintf, fwrite i fputc són comunament utilitzades per a aquest propòsit.

Exemple: Escriure a un Fitxer de Text

% Obrir el fitxer per escriure
fileID = fopen('output.txt', 'w');

% Comprovar si el fitxer s'ha obert correctament
if fileID == -1
    error('El fitxer no s\'ha pogut obrir.');
end

% Escriure dades al fitxer
fprintf(fileID, 'Hola, Món!\n');
fprintf(fileID, 'Això és una prova.\n');

% Tancar el fitxer
fclose(fileID);
  • fprintf: Escriu dades formatades al fitxer.

Operacions Intermèdies amb Fitxers

Llegir i Escriure Fitxers Binari

Els fitxers binaris s'utilitzen per a un emmagatzematge eficient i un accés ràpid. Les funcions de MATLAB fread i fwrite s'utilitzen per a operacions amb fitxers binaris.

Exemple: Escriure i Llegir un Fitxer Binari

% Crear algunes dades
data = rand(10, 1);

% Obrir el fitxer per escriure en mode binari
fileID = fopen('data.bin', 'w');

% Escriure les dades al fitxer
fwrite(fileID, data, 'double');

% Tancar el fitxer
fclose(fileID);

% Obrir el fitxer per llegir en mode binari
fileID = fopen('data.bin', 'r');

% Llegir les dades del fitxer
readData = fread(fileID, [10, 1], 'double');

% Tancar el fitxer
fclose(fileID);

% Mostrar les dades llegides
disp(readData);
  • fwrite: Escriu dades binàries al fitxer.
  • fread: Llegeix dades binàries del fitxer.

Utilitzar `textscan` per a Dades Estructurades

textscan és útil per llegir dades estructurades de fitxers de text.

Exemple: Llegir Dades Estructurades

% Obrir el fitxer per llegir
fileID = fopen('data.txt', 'r');

% Definir el format de les dades
formatSpec = '%f %s %d';

% Llegir les dades
data = textscan(fileID, formatSpec);

% Tancar el fitxer
fclose(fileID);

% Mostrar les dades
disp(data);
  • textscan: Llegeix dades formatades d'un fitxer de text.

Operacions Avançades amb Fitxers

Treballar amb Fitxers Grans

Per a fitxers grans, una lectura i escriptura eficient són crucials. MATLAB proporciona funcions com matfile per gestionar fitxers MAT grans.

Exemple: Utilitzar matfile per a Dades Grans

% Crear un objecte MAT-file
matObj = matfile('largeData.mat', 'Writable', true);

% Escriure dades grans al fitxer
matObj.largeData = rand(1e6, 1);

% Llegir una part de les dades
partialData = matObj.largeData(1:1000);

% Mostrar les dades parcials
disp(partialData);
  • matfile: Proporciona accés a variables en fitxers MAT sense carregar-les a la memòria.

Formats de Fitxer Personalitzats

Crear i llegir formats de fitxer personalitzats es pot fer utilitzant funcions d'E/S de fitxers de baix nivell.

Exemple: Format de Fitxer Binari Personalitzat

% Definir dades personalitzades
header = 'CustomHeader';
data = rand(10, 1);

% Obrir el fitxer per escriure
fileID = fopen('custom.bin', 'w');

% Escriure la capçalera
fwrite(fileID, header, 'char');

% Escriure les dades
fwrite(fileID, data, 'double');

% Tancar el fitxer
fclose(fileID);

% Obrir el fitxer per llegir
fileID = fopen('custom.bin', 'r');

% Llegir la capçalera
readHeader = fread(fileID, length(header), 'char=>char')';

% Llegir les dades
readData = fread(fileID, [10, 1], 'double');

% Tancar el fitxer
fclose(fileID);

% Mostrar les dades llegides
disp(readHeader);
disp(readData);
  • Els formats de fitxer personalitzats es poden gestionar combinant diferents operacions de lectura/escriptura.

Conclusió

Llegir i escriure fitxers a MATLAB és una habilitat versàtil que abasta des d'operacions simples amb fitxers de text fins a la gestió de grans conjunts de dades i formats de fitxer personalitzats. En dominar aquestes tècniques, pots gestionar dades de manera eficient dins de les teves aplicacions MATLAB, assegurant un codi robust i escalable.
