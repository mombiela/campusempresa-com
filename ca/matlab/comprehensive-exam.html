Visió General

L'examen comprensiu està dissenyat per avaluar la teva comprensió i competència en MATLAB, cobrint temes des de nivells bàsics fins a avançats. Aquest examen posarà a prova els teus coneixements sobre diversos aspectes de MATLAB, incloent sintaxi bàsica, manipulació de dades, funcions avançades i desenvolupament d'aplicacions.

Estructura de l'Examen

L'examen està dividit en tres seccions:

  1. Nivell Bàsic
  2. Nivell Intermedi
  3. Nivell Avançat

Cada secció contindrà preguntes de selecció múltiple, preguntes de resposta curta i exercicis pràctics de codificació.

Nivell Bàsic

Conceptes Clau

  • Entorn MATLAB
    • Finestra de Comandes
    • Espai de Treball
    • Editor
  • Sintaxi Bàsica
    • Variables i Tipus de Dades
    • Operacions Aritmètiques
    • Funcions Bàsiques
  • Estructures de Control
    • Sentències If-Else
    • Bucles For
    • Bucles While

Preguntes d'Exemple

  1. Selecció Múltiple:

    • Quin és el comandament per esborrar totes les variables de l'espai de treball?
      • a) clear all
      • b) clc
      • c) clearvars
      • d) reset

  2. Resposta Curta:

    • Explica la diferència entre = i == en MATLAB.

  3. Exercici de Codificació:

    • Escriu un script que calculi el factorial d'un número donat utilitzant un bucle for.
% Càlcul del Factorial
n = 5; % Número d'exemple
factorial = 1;
for i = 1:n
    factorial = factorial * i;
end
disp(['El factorial de ', num2str(n), ' és ', num2str(factorial)]);

Preguntes d'Exemple

  1. Selecció Múltiple:

    • Quina funció s'utilitza per llegir dades d'un fitxer de text?
      • a) fopen
      • b) fread
      • c) textread
      • d) readtable

  2. Resposta Curta:

    • Descriu com crear una matriu identitat de 3x3 en MATLAB.

  3. Exercici de Codificació:

    • Escriu un script per representar les funcions sinus i cosinus en el mateix gràfic amb colors diferents i afegeix una llegenda.
% Representació de Sinus i Cosinus
x = linspace(0, 2*pi, 100);
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'r', 'DisplayName', 'Sinus');
hold on;
plot(x, y2, 'b', 'DisplayName', 'Cosinus');
legend show;
title('Funcions Sinus i Cosinus');
xlabel('x');
ylabel('y');
hold off;

Preguntes d'Exemple

  1. Selecció Múltiple:

    • Quina funció s'utilitza per trobar el mínim d'una funció en MATLAB?
      • a) fminsearch
      • b) min
      • c) fminbnd
      • d) minimize

  2. Resposta Curta:

    • Explica l'ús dels manipuladors de funcions en MATLAB.

  3. Exercici de Codificació:

    • Crea una GUI senzilla que permeti a l'usuari introduir un número i mostri el seu quadrat.
% GUI Senzilla per Quadrar un Número
function simple_gui
    fig = uifigure('Position', [100 100 300 200]);
    lbl = uilabel(fig, 'Position', [100 150 100 22], 'Text', 'Introdueix un número:');
    edt = uieditfield(fig, 'numeric', 'Position', [100 120 100 22]);
    btn = uibutton(fig, 'Position', [100 80 100 22], 'Text', 'Calcular', ...
        'ButtonPushedFcn', @(btn,event) calculateSquare(edt.Value));
    lblResult = uilabel(fig, 'Position', [100 50 100 22], 'Text', '');
    
    function calculateSquare(num)
        result = num^2;
        lblResult.Text = ['Quadrat: ', num2str(result)];
    end
end

Preparant-te per aquest examen, consolidaràs la teva comprensió de MATLAB i estaràs ben preparat per afrontar problemes del món real utilitzant aquesta poderosa eina. Bona sort!
