Introducció

En aquesta secció, explorarem com treballar amb diversos formats de fitxer a MATLAB. Això inclou llegir i escriure diferents tipus de fitxers com ara fitxers de text, fitxers CSV, fitxers Excel i fitxers MAT. Al final d'aquesta secció, seràs capaç de gestionar dades emmagatzemades en diferents formats de manera eficient.

Fitxers de Text

Els fitxers de text són una de les formes més simples d'emmagatzematge de dades. MATLAB proporciona diverses funcions per llegir i escriure fitxers de text.

Llegir Fitxers de Text

Per llegir dades d'un fitxer de text, pots utilitzar les funcions fopen, fgets, fscanf i fclose.

% Obrir el fitxer
fileID = fopen('data.txt', 'r');

% Llegir cada línia del fitxer
line = fgets(fileID);
while ischar(line)
    disp(line);
    line = fgets(fileID);
end

% Tancar el fitxer
fclose(fileID);

Escriure Fitxers de Text

Per escriure dades a un fitxer de text, pots utilitzar les funcions fopen, fprintf i fclose.

% Obrir el fitxer
fileID = fopen('output.txt', 'w');

% Escriure dades al fitxer
fprintf(fileID, 'Hello, World!\n');
fprintf(fileID, 'This is a test.\n');

% Tancar el fitxer
fclose(fileID);

Fitxers CSV

Els fitxers CSV (Valors Separats per Comes) són comunament utilitzats per emmagatzemar dades tabulars. MATLAB proporciona funcions integrades per gestionar fitxers CSV fàcilment.

Llegir Fitxers CSV

Per llegir dades d'un fitxer CSV, pots utilitzar les funcions readtable o csvread.

% Utilitzant readtable
data = readtable('data.csv');

% Utilitzant csvread
data = csvread('data.csv', 1, 0); % Saltant la fila de capçalera

Escriure Fitxers CSV

Per escriure dades a un fitxer CSV, pots utilitzar les funcions writetable o csvwrite.

% Utilitzant writetable
data = table([1; 2; 3], [4; 5; 6], 'VariableNames', {'A', 'B'});
writetable(data, 'output.csv');

% Utilitzant csvwrite
csvwrite('output.csv', [1 4; 2 5; 3 6]);

Fitxers Excel

Els fitxers Excel són àmpliament utilitzats en l'anàlisi de dades i la creació d'informes. MATLAB proporciona funcions per llegir i escriure fitxers Excel.

Llegir Fitxers Excel

Per llegir dades d'un fitxer Excel, pots utilitzar les funcions readtable o xlsread.

% Utilitzant readtable
data = readtable('data.xlsx');

% Utilitzant xlsread
[num, txt, raw] = xlsread('data.xlsx');

Escriure Fitxers Excel

Per escriure dades a un fitxer Excel, pots utilitzar les funcions writetable o xlswrite.

% Utilitzant writetable
data = table([1; 2; 3], [4; 5; 6], 'VariableNames', {'A', 'B'});
writetable(data, 'output.xlsx');

% Utilitzant xlswrite
xlswrite('output.xlsx', [1 4; 2 5; 3 6]);

Fitxers MAT

Els fitxers MAT són el format natiu de MATLAB per emmagatzemar variables. Són particularment útils per guardar i carregar grans conjunts de dades i variables complexes.

Guardar a Fitxers MAT

Per guardar variables a un fitxer MAT, pots utilitzar la funció save.

% Definir algunes variables
A = [1, 2, 3];
B = {'Hello', 'World'};

% Guardar variables a un fitxer MAT
save('data.mat', 'A', 'B');

Carregar des de Fitxers MAT

Per carregar variables d'un fitxer MAT, pots utilitzar la funció load.

% Carregar variables d'un fitxer MAT
load('data.mat');

% Mostrar les variables carregades
disp(A);
disp(B);

Conclusió

En aquesta secció, hem cobert com treballar amb diferents formats de fitxer a MATLAB, incloent fitxers de text, fitxers CSV, fitxers Excel i fitxers MAT. Entendre com llegir i escriure aquests formats de fitxer és essencial per a l'anàlisi i manipulació de dades a MATLAB. Practica utilitzant aquestes funcions per convertir-te en un expert en la gestió de diversos tipus de fitxers de dades.
