Introducció als Gràfics 3D

Els gràfics 3D a MATLAB et permeten visualitzar dades en tres dimensions, proporcionant una comprensió més profunda de les relacions i estructures de les dades. Aquesta secció et guiarà des dels conceptes bàsics fins a tècniques avançades en gràfics 3D.

Gràfics 3D Bàsics

Creació d'un Gràfic 3D Simple

Per crear un gràfic 3D bàsic, pots utilitzar la funció plot3. Aquesta funció requereix tres vectors: X, Y i Z.

% Defineix les dades
X = 1:10;
Y = rand(1, 10);
Z = rand(1, 10);

% Crea el gràfic 3D
figure;
plot3(X, Y, Z, '-o');
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic 3D Simple');
grid on;

Gràfics de Dispersió

Per a punts de dades dispersos, utilitza la funció scatter3.

% Defineix les dades
X = rand(1, 50);
Y = rand(1, 50);
Z = rand(1, 50);

% Crea el gràfic de dispersió 3D
figure;
scatter3(X, Y, Z, 'filled');
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic de Dispersió 3D');
grid on;

Gràfics de Superfície

Els gràfics de superfície són útils per visualitzar funcions de dues variables.

Creació d'un Gràfic de Superfície

Utilitza la funció surf per crear un gràfic de superfície.

% Defineix les dades
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crea el gràfic de superfície
figure;
surf(X, Y, Z);
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic de Superfície');
colorbar;

Gràfics de Malla

Els gràfics de malla són similars als gràfics de superfície però només mostren les vores.

% Defineix les dades
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crea el gràfic de malla
figure;
mesh(X, Y, Z);
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic de Malla');

Tècniques Avançades de Gràfics 3D

Gràfics de Contorn

Els gràfics de contorn es poden afegir als gràfics de superfície per mostrar corbes de nivell.

% Defineix les dades
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crea el gràfic de superfície amb contorns
figure;
surf(X, Y, Z);
hold on;
contour3(X, Y, Z, 20, 'k');
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic de Superfície amb Contorns');
colorbar;

Personalització de Gràfics 3D

Pots personalitzar els gràfics 3D canviant propietats com el color, la il·luminació i els angles de visió.

% Defineix les dades
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crea el gràfic de superfície
figure;
surf(X, Y, Z);
xlabel('Eix X');
ylabel('Eix Y');
zlabel('Eix Z');
title('Gràfic de Superfície Personalitzat');
colorbar;

% Personalitza el gràfic
shading interp; % Transicions suaus de color
colormap jet; % Canvia la paleta de colors
lighting phong; % Afegeix il·luminació
camlight headlight; % Afegeix una font de llum
view([-30, 30]); % Canvia l'angle de visió

Conclusió

Els gràfics 3D a MATLAB proporcionen eines poderoses per visualitzar i analitzar dades en tres dimensions. Començant per gràfics bàsics com plot3 i scatter3, pots avançar cap a visualitzacions més complexes com els gràfics de superfície i de malla. Les opcions avançades de personalització et permeten millorar la llegibilitat i l'estètica dels teus gràfics. En dominar aquestes tècniques, podràs comunicar de manera efectiva les idees complexes de les dades.
