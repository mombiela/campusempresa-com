Introducció a les Matrius

Les matrius són fonamentals per a MATLAB, que significa "Laboratori de Matrius". Entendre com crear, manipular i realitzar operacions amb matrius és essencial per a qualsevol programador de MATLAB.

Creació de Matrius

  • Vector Fila: Una matriu de 1xN. 
    row_vector = [1, 2, 3, 4];
  • Vector Columna: Una matriu de Nx1. 
    column_vector = [1; 2; 3; 4];
  • Matriu 2D: Una matriu de MxN. 
    matrix_2d = [1, 2, 3; 4, 5, 6; 7, 8, 9];

Operacions Bàsiques amb Matrius

  • Suma i Resta: Operacions element a element. 
    A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Suma
D = A - B; % Resta
  • Multiplicació per Escalar: Multiplicar cada element per un escalar. 
    scalar = 2;
E = scalar * A;

Multiplicació de Matrius

La multiplicació de matrius segueix les regles de l'àlgebra lineal.

  • Multiplicació Element a Element: Utilitza l'operador .*. 
    F = A .* B;
  • Producte de Matrius: Utilitza l'operador *. 
    G = A * B;

Transposar una Matriu

  • Transposar: Girar una matriu sobre la seva diagonal. 
    H = A';

Operacions Avançades amb Matrius

Inversa i Determinant

  • Inversa: Només per a matrius quadrades. 
    I = inv(A);
  • Determinant: Un valor escalar que és una funció dels elements d'una matriu quadrada. 
    det_A = det(A);

Valors Propis i Vectors Propis

  • Valors Propis i Vectors Propis: Importants en moltes aplicacions com l'anàlisi de l'estabilitat. 
    [V, D] = eig(A);
    • V conté els vectors propis.
    • D és una matriu diagonal amb els valors propis.

Resolució de Sistemes Lineals

  • Sistema Lineal: Ax = b 
    A = [1, 2; 3, 4];
b = [5; 6];
x = A \ b; % Resol per x

Matrius Especials

  • Matriu Identitat: Una matriu quadrada amb uns a la diagonal i zeros a la resta. 
    I = eye(3); % Matriu identitat de 3x3
  • Matriu de Zeros: Una matriu plena de zeros. 
    Z = zeros(3, 2); % Matriu de 3x2 de zeros
  • Matriu de Uns: Una matriu plena d'uns. 
    O = ones(2, 3); % Matriu de 2x3 d'uns

Exemples Pràctics

Exemple 1: Suma i Multiplicació de Matrius

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Suma de matrius
D = A * B; % Multiplicació de matrius

Exemple 2: Resolució d'un Sistema Lineal

A = [2, -1; -3, 4];
b = [1; 7];
x = A \ b; % Resolent Ax = b

Exemple 3: Valors Propis i Vectors Propis

A = [4, -2; 1, 1];
[V, D] = eig(A);

Resum

Entendre les operacions amb matrius a MATLAB és crucial per realitzar una àmplia gamma de tasques matemàtiques i d'enginyeria. Des d'operacions bàsiques com la suma i la multiplicació fins a conceptes avançats com els valors propis i la resolució de sistemes lineals, dominar aquestes habilitats millorarà significativament la teva capacitat d'utilitzar MATLAB de manera efectiva. Practica aquestes operacions amb diferents matrius per convertir-te en un expert en la gestió de problemes relacionats amb matrius.
