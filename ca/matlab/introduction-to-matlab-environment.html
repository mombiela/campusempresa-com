Visió general

MATLAB (Matrix Laboratory) és un llenguatge d'alt rendiment per a la computació tècnica. Integra càlcul, visualització i programació en un entorn fàcil d'utilitzar on els problemes i solucions s'expressen en una notació matemàtica familiar.

Conceptes clau

  • Interfície MATLAB: Comprendre la disposició i els components de l'entorn MATLAB.
  • Comandes bàsiques: Aprendre les comandes essencials per navegar i utilitzar MATLAB.
  • Variables i tipus de dades: Introducció a les variables i els tipus de dades que MATLAB pot gestionar.
  • Scripts i funcions: Escriure i executar scripts i funcions en MATLAB.

Interfície MATLAB

Quan obres MATLAB, et trobes amb diversos components clau:

  • Finestra de comandes: L'àrea principal on pots introduir comandes i veure els resultats.
  • Espai de treball: Mostra les variables que has creat i els seus valors.
  • Historial de comandes: Manté un registre de les comandes que has introduït.
  • Carpeta actual: Mostra els fitxers del directori actual.
  • Editor: On pots escriure, editar i desar scripts i funcions.

Exemple

% Això és un comentari
disp('Hola, món!') % Mostra un missatge

A la Finestra de comandes, escriu el codi anterior per mostrar "Hola, món!".

Comandes bàsiques

Aquí tens algunes comandes bàsiques per començar:

  • clc: Neteja la Finestra de comandes.
  • clear: Elimina totes les variables de l'espai de treball.
  • who: Llista totes les variables a l'espai de treball.
  • whos: Proporciona informació detallada sobre les variables.

Exemple

a = 5; % Assigna el valor 5 a la variable a
b = 10; % Assigna el valor 10 a la variable b
sum = a + b; % Suma a i b, emmagatzema el resultat a sum
disp(sum) % Mostra el valor de sum

Variables i tipus de dades

MATLAB admet diversos tipus de dades, incloent:

  • Tipus numèrics: Enters, nombres de punt flotant.
  • Caràcters i cadenes: Dades de text.
  • Tipus lògics: Valors booleans (true o false).
  • Arrays i matrius: Fonamentals per a MATLAB, que significa Matrix Laboratory.

Exemple

x = 3.14; % Nombre de punt flotant
y = 'Hola'; % Cadena
z = true; % Valor lògic
A = [1, 2; 3, 4]; % Matriu 2x2

Scripts i funcions

Els scripts i funcions són essencials per automatitzar tasques i organitzar el codi.

  • Scripts: Fitxers que contenen una seqüència de comandes MATLAB.
  • Funcions: Fitxers que accepten arguments d'entrada i retornen sortides.

Exemple de script

Crea un fitxer anomenat myscript.m:

% myscript.m
a = 2;
b = 3;
c = a + b;
disp(c)

Executa l'script escrivint myscript a la Finestra de comandes.

Exemple de funció

Crea un fitxer anomenat myfunction.m:

function result = myfunction(x, y)
    result = x + y;
end

Crida la funció des de la Finestra de comandes:

sum = myfunction(5, 7);
disp(sum)

Conclusió

En aquesta introducció, hem cobert els conceptes bàsics de l'entorn MATLAB, incloent la interfície, comandes bàsiques, variables i tipus de dades, i com escriure scripts i funcions. Comprendre aquests fonaments proporcionarà una base sòlida per a temes més avançats en la programació MATLAB.
