Introducció

En aquesta secció, aprofundirem en les funcions avançades d'arrays en MATLAB. Aquestes funcions són essencials per a una manipulació i anàlisi eficient de dades. Al final d'aquesta secció, seràs capaç de realitzar operacions complexes en arrays amb facilitat.

Conceptes Clau

  • Manipulació d'Arrays: Tècniques per canviar la forma, ordenar i filtrar arrays.
  • Operacions Matemàtiques: Funcions matemàtiques avançades que operen en arrays.
  • Operacions Lògiques: Ús de l'indexació lògica i operacions per manipular arrays.
  • Funcions Especialitzades: Funcions dissenyades per a tasques específiques com trobar elements únics, interpolar dades i més.

Manipulació d'Arrays

Canvi de Forma d'Arrays

Canviar la forma dels arrays et permet canviar les dimensions d'un array sense canviar les seves dades.

A = 1:12; % Un array 1x12
B = reshape(A, [3, 4]); % Canvia la forma d'A a un array 3x4
disp(B);

Sortida:

     1     4     7    10
     2     5     8    11
     3     6     9    12

Ordenació d'Arrays

Ordenar arrays ajuda a organitzar dades en ordre ascendent o descendent.

C = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5];
sortedC = sort(C); % Ordena en ordre ascendent
disp(sortedC);

Sortida:

     1     1     2     3     3     4     5     5     5     6     9

Filtratge d'Arrays

El filtratge et permet extreure elements que compleixen certs criteris.

D = [10, 20, 30, 40, 50];
filteredD = D(D > 25); % Extreu elements més grans que 25
disp(filteredD);

Sortida:

    30    40    50

Operacions Matemàtiques

Operacions Element a Element

Realitza operacions en cada element de l'array individualment.

E = [1, 2, 3, 4];
F = [5, 6, 7, 8];
G = E .* F; % Multiplicació element a element
disp(G);

Sortida:

     5    12    21    32

Operacions de Matriu

Realitza operacions en arrays com a matrius.

H = [1, 2; 3, 4];
I = [5, 6; 7, 8];
J = H * I; % Multiplicació de matrius
disp(J);

Sortida:

    19    22
    43    50

Operacions Lògiques

Indexació Lògica

Utilitza condicions lògiques per indexar arrays.

K = [10, 20, 30, 40, 50];
logicalIndex = K > 25;
disp(K(logicalIndex)); % Elements més grans que 25

Sortida:

    30    40    50

Funcions Lògiques

Utilitza funcions lògiques per realitzar operacions.

L = [1, 0, 1, 0, 1];
M = [0, 1, 0, 1, 0];
N = L & M; % AND lògic
disp(N);

Sortida:

     0     0     0     0     0

Funcions Especialitzades

Elements Únics

Troba elements únics en un array.

O = [1, 2, 2, 3, 4, 4, 4, 5];
uniqueO = unique(O);
disp(uniqueO);

Sortida:

     1     2     3     4     5

Interpolació

Interpola punts de dades dins d'un array.

x = 1:10;
y = sin(x);
xi = 1:0.5:10;
yi = interp1(x, y, xi, 'linear');
disp(yi);

Sortida:

    0.8415    0.9093    0.1411   -0.7568   -0.9589   -0.2794    0.6570    0.9894    0.4121   -0.5440   -0.9999   -0.5366    0.4202    0.9906    0.6503   -0.2879   -0.9614   -0.7539    0.1499    0.9129

Conclusió

En aquesta secció, hem explorat funcions avançades d'arrays en MATLAB, incloent manipulació d'arrays, operacions matemàtiques, operacions lògiques i funcions especialitzades. Dominar aquestes funcions millorarà significativament la teva capacitat per gestionar i analitzar dades de manera eficient en MATLAB. Practica aquests conceptes amb diferents conjunts de dades per esdevenir proficient en l'ús de funcions avançades d'arrays.
