Introducció a la Criptografia

La criptografia és una disciplina fonamental en la seguretat informàtica que se centra en protegir la informació mitjançant tècniques de xifrat i desxifrat. En el context d'OWASP (Open Web Application Security Project), la criptografia és crucial per assegurar la integritat, confidencialitat i autenticitat de les dades en aplicacions web.

Conceptes Clau

  • Xifrat i Desxifrat: Procés de convertir dades llegibles en un format codificat (xifrat) i viceversa (desxifrat).
  • Clau Criptogràfica: Informació utilitzada en el procés de xifrat i desxifrat.
  • Algoritmes Simètrics: Utilitzen la mateixa clau per xifrar i desxifrar dades (e.g., AES).
  • Algoritmes Asimètrics: Utilitzen un parell de claus (pública i privada) per xifrar i desxifrar dades (e.g., RSA).
  • Hashing: Procés de convertir dades en una cadena de longitud fixa que representa les dades originals.
  • Firma Digital: Mètode per verificar l'autenticitat i integritat d'un missatge o document.

Algoritmes Criptogràfics

Algoritmes Simètrics

  • AES (Advanced Encryption Standard):
    • Utilitza claus de 128, 192 o 256 bits.
    • És àmpliament utilitzat degut a la seva eficiència i seguretat.
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clau aleatòria
key = get_random_bytes(16)  # AES-128

# Crear un objecte de xifrat
cipher = AES.new(key, AES.MODE_EAX)

# Xifrar dades
plaintext = b'Aquest és un missatge secret'
ciphertext, tag = cipher.encrypt_and_digest(plaintext)

print(f'Xifrat: {ciphertext}')

Algoritmes Asimètrics

  • RSA (Rivest-Shamir-Adleman):
    • Utilitza un parell de claus (pública i privada).
    • És comunament utilitzat per a l'intercanvi segur de claus i firmes digitals.
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generar un parell de claus RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objecte de xifrat amb la clau pública
cipher = PKCS1_OAEP.new(public_key)

# Xifrar dades
plaintext = b'Aquest és un missatge secret'
ciphertext = cipher.encrypt(plaintext)

print(f'Xifrat: {ciphertext}')

Hashing

  • SHA-256 (Secure Hash Algorithm 256-bit):
    • Genera un hash de 256 bits.
    • És àmpliament utilitzat per verificar la integritat de les dades.
import hashlib

# Crear un objecte de hash
hash_object = hashlib.sha256()

# Actualitzar l'objecte de hash amb les dades
hash_object.update(b'Aquest és un missatge secret')

# Obtenir el hash
hash_digest = hash_object.hexdigest()

print(f'Hash: {hash_digest}')

Firma Digital

  • DSA (Digital Signature Algorithm):
    • Utilitzat per firmar digitalment i verificar l'autenticitat dels missatges.
from Crypto.Signature import DSS
from Crypto.Hash import SHA256
from Crypto.PublicKey import DSA

# Generar un parell de claus DSA
key = DSA.generate(2048)

# Crear un objecte de hash
hash_object = SHA256.new(b'Aquest és un missatge secret')

# Firmar el hash
signer = DSS.new(key, 'fips-186-3')
signature = signer.sign(hash_object)

print(f'Firma: {signature}')

Bones Pràctiques en Criptografia

  • Ús d'Algoritmes Segurs: Utilitzar algoritmes criptogràfics que siguin considerats segurs per la comunitat de seguretat.
  • Gestió de Claus: Assegurar que les claus criptogràfiques siguin gestionades i emmagatzemades de manera segura.
  • Actualització Regular: Mantenir-se actualitzat amb les últimes recomanacions i actualitzacions de seguretat en criptografia.
  • Evitar Algoritmes Obsolets: No utilitzar algoritmes que hagin estat compromesos o considerats insegurs (e.g., DES, MD5).

Conclusió

La criptografia és una eina essencial en la seguretat d'aplicacions web, proporcionant mecanismes per protegir la confidencialitat, integritat i autenticitat de les dades. Comprendre i aplicar correctament els conceptes i algoritmes criptogràfics és crucial per a qualsevol professional de TI que treballi en el desenvolupament i seguretat d'aplicacions. A través d'aquest tema, hem explorat els fonaments de la criptografia, els principals algoritmes utilitzats i les bones pràctiques per a la seva implementació en el context d'OWASP.
