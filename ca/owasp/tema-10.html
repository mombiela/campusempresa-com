Introducció

En el context de la seguretat d'aplicacions web, les fallades criptogràfiques són una de les principals vulnerabilitats identificades per OWASP en la seva llista dels deu principals riscos de seguretat. Aquest tema aborda els errors comuns en la implementació de criptografia i com poden ser explotats per atacants per comprometre la seguretat d'una aplicació.

Conceptes Clau

  • Criptografia: Tècnica per protegir la informació mitjançant la transformació de dades llegibles en dades xifrades.
  • Xifrat: Procés de convertir dades en un format il·legible per protegir la seva confidencialitat.
  • Desxifrat: Procés de convertir dades xifrades de tornada al seu format original.
  • Algoritmes Criptogràfics: Conjunt de regles matemàtiques utilitzades per xifrar i desxifrar dades.
  • Claus Criptogràfiques: Valors secrets utilitzats en els algoritmes criptogràfics per xifrar i desxifrar dades.

Errors Comuns en la Criptografia

  1. Ús d'Algoritmes Obsolets o Insegurs:

    • Exemple: Ús de MD5 o SHA-1 per hashing.
    • Solució: Utilitzar algoritmes moderns com SHA-256 o SHA-3.

  2. Gestió Inadequada de Claus:

    • Exemple: Emmagatzematge de claus en text pla.
    • Solució: Utilitzar mòduls de seguretat de maquinari (HSM) o serveis de gestió de claus.

  3. Ús Incorrecte de Modes d'Operació de Xifrat:

    • Exemple: Ús d'ECB (Electronic Codebook) en lloc de CBC (Cipher Block Chaining).
    • Solució: Utilitzar modes d'operació segurs com CBC o GCM (Galois/Counter Mode).

  4. Manca d'Entropia en la Generació de Claus:

    • Exemple: Ús de generadors de nombres pseudoaleatoris insegurs.
    • Solució: Utilitzar generadors de nombres aleatoris criptogràficament segurs.

Exemples de Codi

Exemple d'Ús Incorrecte d'Algoritmes Obsolets

import hashlib

# Ús de MD5 (insegur)
data = "missatge_secret"
hash_md5 = hashlib.md5(data.encode()).hexdigest()
print(f"MD5: {hash_md5}")

Exemple d'Ús Correcte d'Algoritmes Moderns

import hashlib

# Ús de SHA-256 (segur)
data = "missatge_secret"
hash_sha256 = hashlib.sha256(data.encode()).hexdigest()
print(f"SHA-256: {hash_sha256}")

Exemple de Gestió Inadequada de Claus

# Emmagatzematge de clau en text pla (insegur)
key = "clau_secreta"

Exemple de Gestió Adequada de Claus

from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.backends import default_backend
import os

# Generació de clau segura usant PBKDF2HMAC
password = b"la_meva_contrasenya"
salt = os.urandom(16)
kdf = PBKDF2HMAC(
    algorithm=hashes.SHA256(),
    length=32,
    salt=salt,
    iterations=100000,
    backend=default_backend()
)
key = kdf.derive(password)
print(f"Clau segura: {key.hex()}")

Comparació de Modes d'Operació de Xifrat

| Mode d'Operació | Seguretat | Ús Recomanat | |-----------------|-----------|--------------| | ECB | Insegur | No | | CBC | Segur | Sí | | GCM | Molt Segur| Sí |

Conclusió

Les fallades criptogràfiques poden tenir conseqüències greus per a la seguretat d'una aplicació. És crucial utilitzar algoritmes moderns i segurs, gestionar adequadament les claus criptogràfiques i assegurar-se que els modes d'operació de xifrat siguin apropiats. En seguir aquestes pràctiques, es pot mitigar significativament el risc de vulnerabilitats criptogràfiques en les aplicacions web.
