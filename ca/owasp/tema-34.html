Cursos i Tallers d'OWASP

Introducció

El Projecte de Seguretat d'Aplicacions Web Obertes (OWASP, per les seves sigles en anglès) és una organització sense ànim de lucre que es dedica a millorar la seguretat del programari. OWASP proporciona una sèrie de recursos, incloent documentació, eines i estàndards, que són essencials per als professionals de TI. Aquest curs està dissenyat per oferir una comprensió profunda dels conceptes i pràctiques de seguretat promoguts per OWASP.

Objectius del Curs

  • Comprendre els principis fonamentals de la seguretat d'aplicacions web.
  • Familiaritzar-se amb les eines i recursos proporcionats per OWASP.
  • Aprendre a identificar i mitigar les vulnerabilitats comunes en aplicacions web.
  • Aplicar bones pràctiques de seguretat en el desenvolupament i manteniment de programari.

Contingut del Curs

1. Introducció a OWASP

  • Història i Missió d'OWASP
    • Fundada el 2001.
    • Missió: Millorar la seguretat del programari.
  • Principals Projectes d'OWASP
    • OWASP Top Ten.
    • OWASP ASVS (Application Security Verification Standard).
    • OWASP ZAP (Zed Attack Proxy).

2. OWASP Top Ten

  • Descripció General
    • Llista de les deu vulnerabilitats més crítiques en aplicacions web.
  • Vulnerabilitats Comunes
    • Injecció (SQL, NoSQL, OS, etc.).
    • Autenticació Trencada.
    • Exposició de Dades Sensibles.
    • Entitats Externes XML (XXE).
    • Control d'Accés Trencat.
    • Configuració Incorrecta de Seguretat.
    • Cross-Site Scripting (XSS).
    • Deserialització Insegura.
    • Ús de Components amb Vulnerabilitats Conegudes.
    • Registre i Monitoratge Insuficients.

3. Eines d'OWASP

  • OWASP ZAP (Zed Attack Proxy)
    • Eina de prova de penetració.
    • Exemple d'ús: 
      zap.sh -daemon -port 8080 -config api.key=12345
  • OWASP Dependency-Check
    • Eina per identificar vulnerabilitats en dependències.
    • Exemple d'ús: 
      dependency-check --project "My Project" --scan /path/to/project
  • OWASP ASVS (Application Security Verification Standard)
    • Estàndard per verificar la seguretat d'aplicacions.
    • Nivells de verificació: Nivell 1, Nivell 2, Nivell 3.

4. Tallers Pràctics

  • Taller d'Identificació de Vulnerabilitats
    • Ús d'OWASP ZAP per identificar i explotar vulnerabilitats.
  • Taller de Mitigació de Vulnerabilitats
    • Aplicació de pegats i configuracions de seguretat.
    • Exemple de mitigació de SQL Injection: 
      # Ús de consultes preparades en Python
cursor.execute("SELECT * FROM users WHERE username = %s", (username,))
  • Taller d'Implementació de Bones Pràctiques
    • Implementació de controls d'accés.
    • Exemple de control d'accés en una aplicació web: 
      @PreAuthorize("hasRole('ADMIN')")
public String adminPage() {
    return "admin";
}

Conclusió

Aquest curs proporciona una base sòlida en els principis i pràctiques de seguretat promoguts per OWASP. En finalitzar, els participants estaran equipats amb el coneixement i les habilitats necessàries per identificar i mitigar vulnerabilitats en aplicacions web, utilitzant les eines i recursos proporcionats per OWASP. La seguretat és un procés continu, i aquest curs és un pas important cap a la creació d'aplicacions més segures i robustes.
