Introducció a les Textures

Les textures són imatges aplicades a les superfícies de formes o models 3D per donar-los més detall i realisme. En OpenGL, les textures es poden utilitzar per afegir color, patrons o altres efectes visuals als objectes.

Conceptes Clau

  • Imatge de Textura: El fitxer d'imatge (per exemple, PNG, JPEG) que s'utilitzarà com a textura.
  • Coordenades de Textura: Coordenades que mapejen la imatge de textura a la geometria.
  • Filtratge de Textura: Tècniques per determinar com es mapejen els píxels de textura (texels) als píxels de pantalla.
  • Mipmaps: Seqüències d'imatges pre-calculades i optimitzades que acompanyen una textura principal, utilitzades per augmentar la velocitat de renderització i reduir artefactes.

Carregant Textures

Per utilitzar textures en OpenGL, primer has de carregar la imatge de textura al teu programa.

Exemple: Carregant una Textura

GLuint texture;
glGenTextures(1, &texture);
glBindTexture(GL_TEXTURE_2D, texture);

// Carregar imatge, crear textura i generar mipmaps
int width, height, nrChannels;
unsigned char *data = stbi_load("path/to/your/image.jpg", &width, &height, &nrChannels, 0);
if (data)
{
    glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, width, height, 0, GL_RGB, GL_UNSIGNED_BYTE, data);
    glGenerateMipmap(GL_TEXTURE_2D);
}
else
{
    std::cout << "Failed to load texture" << std::endl;
}
stbi_image_free(data);
  • stbi_load: Funció de la biblioteca d'imatges STB per carregar imatges.
  • glTexImage2D: Especifica una imatge de textura bidimensional.
  • glGenerateMipmap: Genera mipmaps per a la textura especificada.

Coordenades de Textura

Les coordenades de textura van de (0,0) a (1,1), on (0,0) és la cantonada inferior esquerra i (1,1) és la cantonada superior dreta de la textura.

Exemple: Aplicant Coordenades de Textura

float vertices[] = {
    // posicions          // coords de textura
     0.5f,  0.5f, 0.0f,   1.0f, 1.0f, // superior dreta
     0.5f, -0.5f, 0.0f,   1.0f, 0.0f, // inferior dreta
    -0.5f, -0.5f, 0.0f,   0.0f, 0.0f, // inferior esquerra
    -0.5f,  0.5f, 0.0f,   0.0f, 1.0f  // superior esquerra 
};
  • Coordenades de Textura: Els dos últims valors de cada vèrtex especifiquen les coordenades de textura.

Filtratge de Textura

El filtratge de textura determina com es mostra una textura quan es redimensiona.

Tipus de Filtratge de Textura

  • Filtratge de Veí Més Proper: Tria el texel més proper.
  • Filtratge Lineal: Interpola entre múltiples texels.

Exemple: Configurant el Filtratge de Textura

glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);
  • GL_TEXTURE_MIN_FILTER: Filtre utilitzat quan la textura es minimitza.
  • GL_TEXTURE_MAG_FILTER: Filtre utilitzat quan la textura es maximitza.

Mipmaps

Els mipmaps són una sèrie de textures pre-calculades, cadascuna de les quals és una representació de resolució progressivament inferior de la mateixa imatge.

Exemple: Generant Mipmaps

glGenerateMipmap(GL_TEXTURE_2D);
  • glGenerateMipmap: Genera automàticament tots els mipmaps necessaris per a la textura actualment vinculada.

Tècniques Avançades de Mapeig de Textures

A mesura que avancis, pots explorar tècniques de mapeig de textures més complexes com ara:

  • Multi-texturització: Utilitzar múltiples textures en un sol objecte.
  • Mapeig Normal: Afegir més detall a les superfícies simulant petits cops i abonyegadures.
  • Mapeig de Cub: Utilitzar una textura de cub per crear reflexos.

Exemple: Multi-texturització

glActiveTexture(GL_TEXTURE0);
glBindTexture(GL_TEXTURE_2D, texture1);
glUniform1i(glGetUniformLocation(shaderProgram, "texture1"), 0);

glActiveTexture(GL_TEXTURE1);
glBindTexture(GL_TEXTURE_2D, texture2);
glUniform1i(glGetUniformLocation(shaderProgram, "texture2"), 1);
  • glActiveTexture: Selecciona quina unitat de textura fer activa.
  • glUniform1i: Configura el sampler de textura a la unitat de textura correcta.

Conclusió

Les textures i el mapeig de textures són conceptes fonamentals en OpenGL que milloren significativament la qualitat visual dels objectes renderitzats. Comprenent com carregar textures, aplicar coordenades de textura i utilitzar el filtratge de textures i mipmaps, pots crear gràfics més realistes i visualment atractius. A mesura que avancis, explorar tècniques com la multi-texturització i el mapeig normal ampliarà encara més la teva capacitat per crear escenes complexes i detallades.
