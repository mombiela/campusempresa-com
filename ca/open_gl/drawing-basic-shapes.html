Introducció

En aquesta secció, explorarem com dibuixar formes bàsiques utilitzant OpenGL. Això és fonamental per a qualsevol persona que comenci amb OpenGL, ja que estableix les bases per a una programació gràfica més complexa. Cobrirà les següents formes:

  • Punts
  • Línies
  • Triangles
  • Quads (Quadrilàters)

Configuració d'OpenGL

Abans de començar a dibuixar formes, assegura't de tenir una configuració bàsica d'OpenGL. Això inclou inicialitzar el context d'OpenGL, crear una finestra i configurar un bucle de renderització. Aquí tens una configuració mínima utilitzant GLFW i GLEW:

#include <GL/glew.h>
#include <GLFW/glfw3.h>

int main() {
    // Inicialitzar GLFW
    if (!glfwInit()) {
        return -1;
    }

    // Crear una finestra en mode finestra i el seu context OpenGL
    GLFWwindow* window = glfwCreateWindow(640, 480, "OpenGL Setup", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return -1;
    }

    // Fer que el context de la finestra sigui actual
    glfwMakeContextCurrent(window);

    // Inicialitzar GLEW
    if (glewInit() != GLEW_OK) {
        return -1;
    }

    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        // Renderitzar aquí

        // Intercanviar els buffers frontal i posterior
        glfwSwapBuffers(window);

        // Poll per i processar esdeveniments
        glfwPollEvents();
    }

    glfwTerminate();
    return 0;
}

Dibuixar Punts

Els punts són el primitiu més senzill en OpenGL. Per dibuixar un punt, necessites especificar les seves coordenades.

Exemple

void drawPoint() {
    glBegin(GL_POINTS); // Començar a dibuixar punts
    glVertex2f(0.0f, 0.0f); // Especificar les coordenades del punt
    glEnd(); // Acabar de dibuixar punts
}

Explicació

  • glBegin(GL_POINTS); diu a OpenGL que comencem a dibuixar punts.
  • glVertex2f(0.0f, 0.0f); especifica les coordenades del punt.
  • glEnd(); acaba el procés de dibuix.

Dibuixar Línies

Les línies es defineixen per dos punts. OpenGL proporciona diversos tipus de línies, però començarem amb la línia bàsica.

Exemple

void drawLine() {
    glBegin(GL_LINES); // Començar a dibuixar línies
    glVertex2f(-0.5f, 0.0f); // Primer punt
    glVertex2f(0.5f, 0.0f); // Segon punt
    glEnd(); // Acabar de dibuixar línies
}

Explicació

  • glBegin(GL_LINES); diu a OpenGL que comencem a dibuixar línies.
  • glVertex2f(-0.5f, 0.0f); especifica el primer punt de la línia.
  • glVertex2f(0.5f, 0.0f); especifica el segon punt de la línia.
  • glEnd(); acaba el procés de dibuix.

Dibuixar Triangles

Els triangles són un dels primitius més importants en OpenGL, ja que s'utilitzen per construir formes més complexes.

Exemple

void drawTriangle() {
    glBegin(GL_TRIANGLES); // Començar a dibuixar triangles
    glVertex2f(0.0f, 0.5f); // Primer vèrtex
    glVertex2f(-0.5f, -0.5f); // Segon vèrtex
    glVertex2f(0.5f, -0.5f); // Tercer vèrtex
    glEnd(); // Acabar de dibuixar triangles
}

Explicació

  • glBegin(GL_TRIANGLES); diu a OpenGL que comencem a dibuixar triangles.
  • glVertex2f(0.0f, 0.5f); especifica el primer vèrtex del triangle.
  • glVertex2f(-0.5f, -0.5f); especifica el segon vèrtex del triangle.
  • glVertex2f(0.5f, -0.5f); especifica el tercer vèrtex del triangle.
  • glEnd(); acaba el procés de dibuix.

Dibuixar Quads

Els quads es defineixen per quatre vèrtexs. No obstant això, OpenGL modern ha deprecat els quads en favor d'utilitzar dos triangles per formar un quad.

Exemple

void drawQuad() {
    glBegin(GL_QUADS); // Començar a dibuixar quads
    glVertex2f(-0.5f, 0.5f); // Primer vèrtex
    glVertex2f(0.5f, 0.5f); // Segon vèrtex
    glVertex2f(0.5f, -0.5f); // Tercer vèrtex
    glVertex2f(-0.5f, -0.5f); // Quart vèrtex
    glEnd(); // Acabar de dibuixar quads
}

Explicació

  • glBegin(GL_QUADS); diu a OpenGL que comencem a dibuixar quads.
  • glVertex2f(-0.5f, 0.5f); especifica el primer vèrtex del quad.
  • glVertex2f(0.5f, 0.5f); especifica el segon vèrtex del quad.
  • glVertex2f(0.5f, -0.5f); especifica el tercer vèrtex del quad.
  • glVertex2f(-0.5f, -0.5f); especifica el quart vèrtex del quad.
  • glEnd(); acaba el procés de dibuix.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de dibuixar formes en OpenGL, incloent punts, línies, triangles i quads. Entendre aquests primitius és crucial ja que formen la base per a una programació gràfica més complexa. Practica dibuixant aquestes formes i experimenta amb diferents coordenades per familiaritzar-te amb el sistema de coordenades d'OpenGL i el pipeline de renderització.
