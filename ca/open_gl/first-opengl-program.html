Introducció

OpenGL (Open Graphics Library) és una potent API de gràfics multiplataforma que permet als desenvolupadors crear gràfics 2D i 3D. Aquesta secció et guiarà a través de la creació del teu primer programa OpenGL, cobrint els passos essencials i proporcionant exemples clars per ajudar-te a entendre els conceptes bàsics.

Configuració del teu entorn de desenvolupament

Abans de començar a codificar, necessites configurar el teu entorn de desenvolupament.

  • Instal·la un compilador de C++: Assegura't que tens un compilador de C++ instal·lat (per exemple, GCC per a Linux, MinGW per a Windows, o Xcode per a macOS).
  • Instal·la les biblioteques OpenGL: Necessitaràs les biblioteques OpenGL i un kit d'eines de finestres com GLFW o GLUT.
  • Configuració de l'IDE: Tria un Entorn de Desenvolupament Integrat (IDE) com Visual Studio, Code::Blocks, o CLion.

Exemple: Instal·lant GLFW a Windows

  1. Descarrega els binaris de GLFW des del lloc web de GLFW.
  2. Extreu els fitxers i col·loca'ls en un directori.
  3. Configura el teu IDE per incloure les capçaleres de GLFW i enllaçar contra els fitxers de la biblioteca GLFW.

Escriure el teu primer programa OpenGL

Escrivim un programa simple d'OpenGL que obre una finestra i mostra un fons de color.

Guia pas a pas

  1. Inclou les capçaleres necessàries:

    #include <GL/glew.h>
#include <GLFW/glfw3.h>

  2. Inicialitza GLFW:

    if (!glfwInit()) {
    return -1;
}

  3. Crea una finestra en mode finestra i el seu context OpenGL:

    GLFWwindow* window = glfwCreateWindow(640, 480, "Hello OpenGL", NULL, NULL);
if (!window) {
    glfwTerminate();
    return -1;
}

  4. Fes que el context de la finestra sigui actual:

    glfwMakeContextCurrent(window);

  5. Inicialitza GLEW:

    if (glewInit() != GLEW_OK) {
    return -1;
}

  6. Bucle principal:

    while (!glfwWindowShouldClose(window)) {
    // Renderitza aquí
    glClear(GL_COLOR_BUFFER_BIT);

    // Intercanvia els buffers frontal i posterior
    glfwSwapBuffers(window);

    // Consulta i processa els esdeveniments
    glfwPollEvents();
}

  7. Neteja i surt:

    glfwDestroyWindow(window);
glfwTerminate();
return 0;

Exemple complet de codi

#include <GL/glew.h>
#include <GLFW/glfw3.h>

int main(void) {
    // Inicialitza GLFW
    if (!glfwInit()) {
        return -1;
    }

    // Crea una finestra en mode finestra i el seu context OpenGL
    GLFWwindow* window = glfwCreateWindow(640, 480, "Hello OpenGL", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return -1;
    }

    // Fes que el context de la finestra sigui actual
    glfwMakeContextCurrent(window);

    // Inicialitza GLEW
    if (glewInit() != GLEW_OK) {
        return -1;
    }

    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        // Renderitza aquí
        glClear(GL_COLOR_BUFFER_BIT);

        // Intercanvia els buffers frontal i posterior
        glfwSwapBuffers(window);

        // Consulta i processa els esdeveniments
        glfwPollEvents();
    }

    // Neteja i surt
    glfwDestroyWindow(window);
    glfwTerminate();
    return 0;
}

Explicació dels conceptes clau

  • Inicialització de GLFW: glfwInit() inicialitza la biblioteca GLFW. Si falla, el programa retorna un error.
  • Creació de la finestra: glfwCreateWindow() crea una finestra amb l'amplada, alçada i títol especificats. Si falla, el programa es termina.
  • Gestió del context: glfwMakeContextCurrent(window) fa que el context OpenGL de la finestra especificada sigui actual en el fil de crida.
  • Inicialització de GLEW: glewInit() inicialitza la biblioteca GLEW, que ajuda a gestionar les extensions d'OpenGL.
  • Bucle de renderització: El bucle while continua fins que la finestra s'ha de tancar. glClear(GL_COLOR_BUFFER_BIT) esborra la pantalla, i glfwSwapBuffers(window) intercanvia els buffers frontal i posterior per actualitzar la pantalla.
  • Consulta d'esdeveniments: glfwPollEvents() processa esdeveniments com l'entrada del teclat i el ratolí.

Conclusió

Crear el teu primer programa OpenGL implica configurar el teu entorn de desenvolupament, inicialitzar les biblioteques necessàries, crear una finestra i escriure un bucle de renderització bàsic. Aquest coneixement fonamental t'ajudarà a construir aplicacions gràfiques més complexes a mesura que avancis en el curs.
