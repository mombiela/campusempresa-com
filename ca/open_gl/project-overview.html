Introducció

En aquesta secció, proporcionarem una visió general del projecte en el qual treballaràs al llarg d'aquest curs d'OpenGL. També esbossarem els requisits necessaris per completar amb èxit el projecte. Aquest projecte t'ajudarà a aplicar els conceptes apresos des de nivells bàsics fins a avançats en OpenGL.

Visió general del projecte

El projecte en el qual treballaràs és una aplicació de gràfics 3D que renderitza una escena simple amb diversos objectes, il·luminació i textures. Al final del curs, hauràs creat una escena 3D completament funcional amb elements interactius.

Característiques clau del projecte

  • Renderització d'objectes 3D: Aprèn a renderitzar formes bàsiques 3D com cubs, esferes i piràmides.
  • Il·luminació i ombrejat: Implementa diferents models d'il·luminació, incloent-hi il·luminació ambiental, difusa i especular.
  • Textures: Aplica textures als teus objectes 3D per fer-los més realistes.
  • Control de càmera: Implementa un sistema de càmera per navegar per l'escena 3D.
  • Interacció de l'usuari: Afegeix una interacció bàsica de l'usuari per manipular objectes i la càmera.

Requisits

Abans de començar el projecte, assegura't de tenir les següents eines i biblioteques instal·lades:

Requisits de programari

  • OpenGL: La biblioteca de gràfics principal que utilitzaràs.
  • GLFW: Una biblioteca per crear finestres i gestionar l'entrada.
  • GLEW: La biblioteca OpenGL Extension Wrangler, que ajuda a gestionar extensions d'OpenGL.
  • GLM: Una biblioteca de matemàtiques per a programari gràfic basada en les especificacions de l'OpenGL Shading Language (GLSL).
  • Compilador de C++: Un compilador modern de C++ com GCC o MSVC.

Requisits de maquinari

  • Targeta gràfica: Una GPU dedicada que suporti OpenGL 3.3 o superior.
  • Memòria: Almenys 4GB de RAM.
  • Processador: Es recomana un processador multi-nucli per a un millor rendiment.

Configuració del teu entorn de desenvolupament

Per començar, segueix aquests passos per configurar el teu entorn de desenvolupament:

  1. Instal·la OpenGL: La majoria dels sistemes operatius vénen amb OpenGL preinstal·lat. Verifica la teva instal·lació executant un programa simple d'OpenGL.
  2. Instal·la GLFW: Descarrega i instal·la GLFW des del seu lloc web oficial.
  3. Instal·la GLEW: Descarrega i instal·la GLEW des del seu lloc web oficial.
  4. Instal·la GLM: Descarrega i instal·la GLM des del seu repositori oficial.

Exemple: Configuració de GLFW i GLEW en C++

Aquí tens un exemple simple per inicialitzar GLFW i GLEW en un programa de C++:

#include <GL/glew.h>
#include <GLFW/glfw3.h>
#include <iostream>

int main() {
    // Inicialitza GLFW
    if (!glfwInit()) {
        std::cerr << "No s'ha pogut inicialitzar GLFW" << std::endl;
        return -1;
    }

    // Crea una finestra GLFW
    GLFWwindow* window = glfwCreateWindow(800, 600, "Projecte OpenGL", nullptr, nullptr);
    if (!window) {
        std::cerr << "No s'ha pogut crear la finestra GLFW" << std::endl;
        glfwTerminate();
        return -1;
    }

    // Fa que el context de la finestra sigui actual
    glfwMakeContextCurrent(window);

    // Inicialitza GLEW
    glewExperimental = GL_TRUE;
    if (glewInit() != GLEW_OK) {
        std::cerr << "No s'ha pogut inicialitzar GLEW" << std::endl;
        return -1;
    }

    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        // Renderitza aquí

        // Intercanvia els buffers frontal i posterior
        glfwSwapBuffers(window);

        // Espera i processa esdeveniments
        glfwPollEvents();
    }

    glfwTerminate();
    return 0;
}

Explicació

  • Inicialització de GLFW: La funció glfwInit() inicialitza la biblioteca GLFW.
  • Creació de la finestra: glfwCreateWindow() crea una finestra amb una amplada, alçada i títol especificats.
  • Context actual: glfwMakeContextCurrent() fa que el context d'OpenGL de la finestra especificada sigui actual en el fil de trucada.
  • Inicialització de GLEW: glewInit() inicialitza la biblioteca GLEW.
  • Bucle principal: El bucle principal continua executant-se fins que la finestra s'ha de tancar, renderitzant l'escena i gestionant esdeveniments.

Conclusió

En aquesta secció, hem esbossat el projecte en el qual treballaràs i els requisits necessaris per completar-lo. Configurant el teu entorn de desenvolupament i entenent les característiques clau del projecte, ara estàs preparat per endinsar-te en el món de la programació d'OpenGL. A mesura que avancis en el curs, construiràs i milloraràs gradualment la teva aplicació de gràfics 3D, aplicant els conceptes apresos en cada etapa.
