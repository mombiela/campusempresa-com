Dissenyar una aplicació OpenGL implica entendre l'estructura i el flux de com es renderitzen els gràfics. Aquest tema et guiarà a través dels components essencials i les millors pràctiques per dissenyar una aplicació OpenGL eficient i fàcil de mantenir, des de la configuració de l'entorn fins a la renderització d'escenes complexes.

Configuració del teu entorn

Abans d'entrar en el codi OpenGL, necessites configurar el teu entorn de desenvolupament.

  • Instal·la les biblioteques OpenGL: Assegura't de tenir les biblioteques necessàries instal·lades. Per a la majoria dels sistemes, això inclou la biblioteca OpenGL, GLU (OpenGL Utility Library) i GLEW (OpenGL Extension Wrangler Library).
  • Tria un kit d'eines de finestres: Les opcions comunes inclouen GLFW, SDL i GLUT. Aquests kits d'eines ajuden a crear finestres i gestionar l'entrada.
  • Eines de desenvolupament: Utilitza un IDE o editor de text que suporti el desenvolupament en C/C++, com Visual Studio, CLion o VSCode.

Exemple: Configuració amb GLFW i GLEW

#include <GL/glew.h>
#include <GLFW/glfw3.h>
#include <iostream>

int main() {
    // Inicialitza GLFW
    if (!glfwInit()) {
        std::cerr << "No s'ha pogut inicialitzar GLFW" << std::endl;
        return -1;
    }

    // Crea una finestra en mode finestra i el seu context OpenGL
    GLFWwindow* window = glfwCreateWindow(640, 480, "Hola OpenGL", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return -1;
    }

    // Fa que el context de la finestra sigui actual
    glfwMakeContextCurrent(window);

    // Inicialitza GLEW
    if (glewInit() != GLEW_OK) {
        std::cerr << "No s'ha pogut inicialitzar GLEW" << std::endl;
        return -1;
    }

    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        // Renderitza aquí

        // Intercanvia els buffers frontal i posterior
        glfwSwapBuffers(window);

        // Poll per i processa esdeveniments
        glfwPollEvents();
    }

    glfwDestroyWindow(window);
    glfwTerminate();
    return 0;
}

Estructura de l'aplicació

Una aplicació OpenGL ben estructurada segueix típicament un flux específic:

  1. Inicialització: Configura el context OpenGL, carrega shaders i inicialitza recursos.
  2. Bucle de joc: Renderitza contínuament fotogrames i gestiona l'entrada de l'usuari.
  3. Neteja: Allibera recursos i tanca correctament l'aplicació.

Inicialització

La inicialització implica configurar el teu context OpenGL i preparar els recursos necessaris per a la renderització.

  • Creació del context: Utilitza un kit d'eines de finestres per crear un context OpenGL.
  • Compilació de shaders: Carrega i compila shaders de vèrtex i fragment.
  • Inicialització de recursos: Carrega textures, crea buffers i configura arrays de vèrtex.

Exemple: Compilació de shaders

GLuint compileShader(const char* source, GLenum type) {
    GLuint shader = glCreateShader(type);
    glShaderSource(shader, 1, &source, NULL);
    glCompileShader(shader);

    // Comprova errors de compilació
    GLint success;
    glGetShaderiv(shader, GL_COMPILE_STATUS, &success);
    if (!success) {
        char infoLog[512];
        glGetShaderInfoLog(shader, 512, NULL, infoLog);
        std::cerr << "ERROR::SHADER::COMPILATION_FAILED\n" << infoLog << std::endl;
    }

    return shader;
}

Bucle de joc

El bucle de joc és on es produeix la renderització i la gestió de l'entrada. Típicament inclou:

  • Renderització: Neteja la pantalla, dibuixa objectes i intercanvia buffers.
  • Gestió de l'entrada: Processa l'entrada de l'usuari per interactuar amb l'aplicació.
  • Temporització: Manté una taxa de fotogrames consistent.

Exemple: Bucle de joc bàsic

while (!glfwWindowShouldClose(window)) {
    // Neteja la pantalla
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);

    // Renderitza objectes aquí

    // Intercanvia buffers
    glfwSwapBuffers(window);

    // Poll per i processa esdeveniments
    glfwPollEvents();
}

Tècniques de renderització

La renderització en OpenGL pot anar des de formes simples fins a escenes complexes amb il·luminació i textures.

  • Formes bàsiques: Comença renderitzant formes simples com triangles i quadrats.
  • Textures: Aplica textures a les teves formes per afegir detall.
  • Il·luminació: Implementa models d'il·luminació bàsics per afegir realisme.
  • Tècniques avançades: Explora tècniques avançades com el shadow mapping, normal mapping i efectes de post-processament.

Exemple: Renderització d'un quadrat texturat

// Dades de vèrtex
GLfloat vertices[] = {
    // Posicions        // Coordenades de textura
    0.5f,  0.5f, 0.0f,  1.0f, 1.0f,
    0.5f, -0.5f, 0.0f,  1.0f, 0.0f,
   -0.5f, -0.5f, 0.0f,  0.0f, 0.0f,
   -0.5f,  0.5f, 0.0f,  0.0f, 1.0f
};
GLuint indices[] = {
    0, 1, 3,
    1, 2, 3
};

// Configura buffers i arrays
GLuint VBO, VAO, EBO;
glGenVertexArrays(1, &VAO);
glGenBuffers(1, &VBO);
glGenBuffers(1, &EBO);

glBindVertexArray(VAO);

glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glBindBuffer(GL_ELEMENT_ARRAY_BUFFER, EBO);
glBufferData(GL_ELEMENT_ARRAY_BUFFER, sizeof(indices), indices, GL_STATIC_DRAW);

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 5 * sizeof(GLfloat), (GLvoid*)0);
glEnableVertexAttribArray(0);
glVertexAttribPointer(1, 2, GL_FLOAT, GL_FALSE, 5 * sizeof(GLfloat), (GLvoid*)(3 * sizeof(GLfloat)));
glEnableVertexAttribArray(1);

glBindVertexArray(0);

// Carrega i crea una textura
GLuint texture;
glGenTextures(1, &texture);
glBindTexture(GL_TEXTURE_2D, texture);
// Configura paràmetres de textura
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);
// Carrega imatge, crea textura i genera mipmaps
int width, height;
unsigned char* image = SOIL_load_image("path/to/texture.jpg", &width, &height, 0, SOIL_LOAD_RGB);
glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, width, height, 0, GL_RGB, GL_UNSIGNED_BYTE, image);
glGenerateMipmap(GL_TEXTURE_2D);
SOIL_free_image_data(image);
glBindTexture(GL_TEXTURE_2D, 0);

// Bucle de renderització
while (!glfwWindowShouldClose(window)) {
    // Neteja la pantalla
    glClear(GL_COLOR_BUFFER_BIT);

    // Vincula la textura
    glBindTexture(GL_TEXTURE_2D, texture);

    // Dibuixa el quadrat
    glBindVertexArray(VAO);
    glDrawElements(GL_TRIANGLES, 6, GL_UNSIGNED_INT, 0);
    glBindVertexArray(0);

    // Intercanvia buffers
    glfwSwapBuffers(window);

    // Poll per i processa esdeveniments
    glfwPollEvents();
}

Conclusió

Dissenyar una aplicació OpenGL implica una planificació i estructuració acurada del teu codi. Configurant correctament el teu entorn, seguint un flux d'aplicació estructurat i implementant diverses tècniques de renderització, pots crear aplicacions gràfiques eficients i visualment atractives. A mesura que progressis, continua explorant tècniques de renderització avançades i optimitza la teva aplicació per obtenir un millor rendiment i mantenibilitat.
