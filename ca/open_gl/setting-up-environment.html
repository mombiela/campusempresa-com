◄Què és OpenGL?
Configuració del teu entorn de desenvolupament
Creant el Teu Primer Programa OpenGL ►

Configurar el teu entorn de desenvolupament és el primer pas crucial per aprendre OpenGL. Aquesta secció et guiarà a través de les eines i biblioteques necessàries per començar a programar amb OpenGL, assegurant que tinguis una base sòlida tant per a temes de nivell inicial com avançat.

Prerequisits

Abans d'endinsar-te en la configuració, assegura't de tenir el següent:

  • Coneixement bàsic de programació en C o C++.
  • Un ordinador amb una GPU moderna que suporti OpenGL.

Triar un IDE

Un Entorn de Desenvolupament Integrat (IDE) pot millorar significativament la teva productivitat. Aquí tens algunes opcions populars:

  • Visual Studio (Windows): IDE complet amb excel·lents eines de depuració.
  • CLion (Multiplataforma): Potent IDE de JetBrains amb suport robust per a C++.
  • Code::Blocks (Multiplataforma): Lleuger i fàcil de configurar.
  • Xcode (macOS): L'IDE d'Apple per al desenvolupament en macOS.

Instal·lació de les eines necessàries

Compilador

Assegura't de tenir un compilador de C++ instal·lat. Aquí tens algunes opcions segons el teu sistema operatiu:

  • Windows: Instal·la Visual Studio, que inclou el compilador MSVC.
  • macOS: Instal·la les eines de línia de comandes de Xcode amb xcode-select --install.
  • Linux: Instal·la GCC utilitzant el teu gestor de paquets, per exemple, sudo apt-get install build-essential.

Biblioteques OpenGL

Per treballar amb OpenGL, necessites instal·lar les següents biblioteques:

  • GLFW: Una biblioteca per crear finestres i gestionar l'entrada.
  • GLAD: Un carregador d'OpenGL que gestiona els punters de funció.
  • GLM: Una biblioteca de matemàtiques per a programari gràfic.

Instal·lació de GLFW

  1. Windows:

    • Descarrega els binaris precompilats des del lloc web de GLFW.
    • Extreu els fitxers i col·loca'ls en un directori de la teva elecció.

  2. macOS:

    brew install glfw

  3. Linux:

    sudo apt-get install libglfw3-dev

Instal·lació de GLAD

  1. Ves al generador de GLAD.
  2. Selecciona les següents opcions:
    • API: OpenGL
    • Perfil: Core
    • Generar un carregador: Sí
  3. Fes clic a "Generate" i descarrega els fitxers generats.
  4. Extreu els fitxers i col·loca'ls en el directori del teu projecte.

Instal·lació de GLM

  1. Windows:

    • Descarrega la biblioteca des del repositori de GitHub de GLM.
    • Extreu els fitxers i col·loca'ls en un directori de la teva elecció.

  2. macOS:

    brew install glm

  3. Linux:

    sudo apt-get install libglm-dev

Configuració del teu projecte

Creació d'una estructura bàsica de projecte

Crea una estructura de directoris per al teu projecte:

MyOpenGLProject/
├── include/
│   ├── glad/
│   └── glm/
├── src/
│   ├── main.cpp
│   └── glad.c
├── lib/
└── CMakeLists.txt

Escriure el teu primer programa OpenGL

main.cpp

#include <GLFW/glfw3.h>
#include <iostream>

// Funció per inicialitzar GLFW i crear una finestra
GLFWwindow* initializeWindow(int width, int height, const char* title) {
    if (!glfwInit()) {
        std::cerr << "No s'ha pogut inicialitzar GLFW" << std::endl;
        return nullptr;
    }

    GLFWwindow* window = glfwCreateWindow(width, height, title, nullptr, nullptr);
    if (!window) {
        std::cerr << "No s'ha pogut crear la finestra de GLFW" << std::endl;
        glfwTerminate();
        return nullptr;
    }

    glfwMakeContextCurrent(window);
    return window;
}

int main() {
    GLFWwindow* window = initializeWindow(800, 600, "OpenGL Setup");

    if (!window) {
        return -1;
    }

    while (!glfwWindowShouldClose(window)) {
        glClear(GL_COLOR_BUFFER_BIT);

        // Intercanviar els buffers frontal i posterior
        glfwSwapBuffers(window);

        // Processar esdeveniments
        glfwPollEvents();
    }

    glfwDestroyWindow(window);
    glfwTerminate();
    return 0;
}

Compilar i executar el teu programa

Utilitzant CMake

Crea un fitxer CMakeLists.txt a l'arrel del teu projecte:

cmake_minimum_required(VERSION 3.10)
project(MyOpenGLProject)

set(CMAKE_CXX_STANDARD 11)

# Directoris d'inclusió
include_directories(include)

# Fitxers font
set(SOURCES src/main.cpp src/glad.c)

# Executable
add_executable(MyOpenGLProject ${SOURCES})

# Enllaçar biblioteques
target_link_libraries(MyOpenGLProject glfw GL)

Construir el projecte

  1. Crea un directori de construcció:

    mkdir build
cd build

  2. Genera els fitxers de construcció:

    cmake ..

  3. Construeix el projecte:

    make

  4. Executa l'executable:

    ./MyOpenGLProject

Conclusió

Configurar el teu entorn de desenvolupament per a OpenGL implica triar les eines adequades, instal·lar les biblioteques necessàries i configurar el teu projecte. Seguint aquesta guia, ara hauries de tenir un projecte bàsic d'OpenGL en funcionament. Aquesta configuració servirà com a base per a temes més avançats d'OpenGL que es tractaran en el curs.
