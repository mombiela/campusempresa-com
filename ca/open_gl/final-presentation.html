Visió general

En aquesta secció final del curs, consolidarem tot el que has après sobre OpenGL. Això implicarà preparar una presentació final del teu projecte, revisar conceptes clau i assegurar-te que tens una comprensió sòlida d'OpenGL des de nivells bàsics fins a avançats.

Presentació final

Requisits del projecte

  • Objectiu: Crear un projecte complet d'OpenGL que demostri la teva comprensió dels conceptes tractats en el curs.
  • Components:
    • Renderització bàsica: Incloure formes bàsiques i transformacions.
    • Shaders: Implementar shaders de vèrtex i fragment.
    • Textures: Aplicar textures als objectes.
    • Il·luminació: Incorporar almenys un tipus d'il·luminació (ambiental, difusa, especular).
    • Característiques avançades: Implementar almenys una característica avançada com el mapatge d'ombres, efectes de post-processament o animacions complexes.

Estructura de la presentació

  • Introducció:
    • Presenta breument el teu projecte.
    • Explica l'objectiu i les característiques clau.
  • Detalls tècnics:
    • Discuteix els conceptes d'OpenGL utilitzats.
    • Explica els shaders i com es van implementar.
    • Descriu les tècniques de mapatge de textures i il·luminació.
  • Demostració:
    • Mostra una demostració en viu del teu projecte.
    • Destaca les característiques avançades.
  • Reptes i solucions:
    • Discuteix qualsevol repte enfrontat durant el desenvolupament.
    • Explica com vas superar aquests reptes.
  • Conclusió:
    • Resumeix el projecte.
    • Reflexiona sobre el que has après.

Revisió de conceptes clau

Renderització bàsica

  • Vèrtexs i buffers:
    • Utilitza Vertex Buffer Objects (VBOs) i Vertex Array Objects (VAOs) per gestionar les dades de vèrtexs.
    // Exemple: Configuració d'un VBO i VAO
GLuint VBO, VAO;
glGenBuffers(1, &VBO);
glGenVertexArrays(1, &VAO);

glBindVertexArray(VAO);
glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);

Shaders

  • Shaders de vèrtex i fragment:
    • Escriu i compila shaders per controlar la pipeline de renderització.
    // Exemple: Shaders de vèrtex simple
const char* vertexShaderSource = "#version 330 core\n"
                                 "layout (location = 0) in vec3 aPos;\n"
                                 "void main()\n"
                                 "{\n"
                                 "   gl_Position = vec4(aPos, 1.0);\n"
                                 "}\0";

GLuint vertexShader = glCreateShader(GL_VERTEX_SHADER);
glShaderSource(vertexShader, 1, &vertexShaderSource, NULL);
glCompileShader(vertexShader);

Textures

  • Carregar i aplicar textures:
    • Utilitza biblioteques com SOIL o stb_image per carregar textures.
    // Exemple: Carregar una textura utilitzant stb_image
int width, height, nrChannels;
unsigned char *data = stbi_load("path/to/texture.jpg", &width, &height, &nrChannels, 0);
if (data)
{
    glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, width, height, 0, GL_RGB, GL_UNSIGNED_BYTE, data);
    glGenerateMipmap(GL_TEXTURE_2D);
}
stbi_image_free(data);

Il·luminació

  • Tipus d'il·luminació:
    • Implementa il·luminació ambiental, difusa i especular.
    // Exemple: Model d'il·luminació Phong en un shader de fragment
vec3 ambient = light.ambient * vec3(texture(material.diffuse, TexCoords));
vec3 norm = normalize(Normal);
vec3 lightDir = normalize(light.position - FragPos);
float diff = max(dot(norm, lightDir), 0.0);
vec3 diffuse = light.diffuse * diff * vec3(texture(material.diffuse, TexCoords));
vec3 viewDir = normalize(viewPos - FragPos);
vec3 reflectDir = reflect(-lightDir, norm);
float spec = pow(max(dot(viewDir, reflectDir), 0.0), material.shininess);
vec3 specular = light.specular * spec * vec3(texture(material.specular, TexCoords));
vec3 result = ambient + diffuse + specular;
FragColor = vec4(result, 1.0);

Característiques avançades

  • Mapatge d'ombres:
    • Implementa el mapatge d'ombres per afegir ombres realistes a la teva escena.
    // Exemple: Configuració del mapatge d'ombres
glGenFramebuffers(1, &depthMapFBO);
glGenTextures(1, &depthMap);
glBindTexture(GL_TEXTURE_2D, depthMap);
glTexImage2D(GL_TEXTURE_2D, 0, GL_DEPTH_COMPONENT, SHADOW_WIDTH, SHADOW_HEIGHT, 0, GL_DEPTH_COMPONENT, GL_FLOAT, NULL);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_NEAREST);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_NEAREST);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_CLAMP_TO_BORDER);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_CLAMP_TO_BORDER);
float borderColor[] = { 1.0, 1.0, 1.0, 1.0 };
glTexParameterfv(GL_TEXTURE_2D, GL_TEXTURE_BORDER_COLOR, borderColor);
glBindFramebuffer(GL_FRAMEBUFFER, depthMapFBO);
glFramebufferTexture2D(GL_FRAMEBUFFER, GL_DEPTH_ATTACHMENT, GL_TEXTURE_2D, depthMap, 0);
glDrawBuffer(GL_NONE);
glReadBuffer(GL_NONE);
glBindFramebuffer(GL_FRAMEBUFFER, 0);
