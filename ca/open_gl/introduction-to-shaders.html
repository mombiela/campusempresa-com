Els shaders són una part fonamental de la programació moderna d'OpenGL. Et permeten controlar la pipeline gràfica i crear efectes visuals impressionants. Aquesta secció t'introduirà als shaders, els seus tipus i com utilitzar-los en OpenGL.

Què són els Shaders?

Els shaders són petits programes que s'executen a la GPU. Estan escrits en GLSL (OpenGL Shading Language) i s'utilitzen per controlar la pipeline de renderització. Hi ha dos tipus principals de shaders en OpenGL:

  • Vertex Shaders: Processen els atributs de cada vèrtex.
  • Fragment Shaders: Processen el color de cada píxel.

Vertex Shaders

Els vertex shaders són el primer pas en la pipeline gràfica. Processen els atributs de cada vèrtex, com ara la posició, el color i les coordenades de textura.

Exemple de Vertex Shader

#version 330 core
layout(location = 0) in vec3 aPos; // La variable de posició té l'atribut de posició 0

void main()
{
    gl_Position = vec4(aPos, 1.0); // Estableix la variable de sortida a la posició d'entrada
}
  • #version 330 core: Especifica la versió de GLSL.
  • layout(location = 0) in vec3 aPos: Defineix una variable d'entrada aPos amb la ubicació 0.
  • gl_Position: Variable de sortida integrada que determina la posició del vèrtex.

Fragment Shaders

Els fragment shaders s'executen per a cada píxel i determinen el color final del píxel.

Exemple de Fragment Shader

#version 330 core
out vec4 FragColor; // Variable de sortida

void main()
{
    FragColor = vec4(1.0, 0.5, 0.2, 1.0); // Estableix el color de sortida a taronja
}
  • out vec4 FragColor: Defineix una variable de sortida FragColor.
  • vec4(1.0, 0.5, 0.2, 1.0): Estableix el color a taronja (format RGBA).

Compilació i Enllaç de Shaders

Per utilitzar shaders en OpenGL, cal compilar-los i enllaçar-los en un programa de shaders.

Exemple de Codi per Compilar i Enllaçar Shaders

// Codi font del vertex shader
const char* vertexShaderSource = "#version 330 core\n"
    "layout(location = 0) in vec3 aPos;\n"
    "void main()\n"
    "{\n"
    "   gl_Position = vec4(aPos, 1.0);\n"
    "}\0";

// Codi font del fragment shader
const char* fragmentShaderSource = "#version 330 core\n"
    "out vec4 FragColor;\n"
    "void main()\n"
    "{\n"
    "   FragColor = vec4(1.0, 0.5, 0.2, 1.0);\n"
    "}\0";

// Compilar el vertex shader
unsigned int vertexShader = glCreateShader(GL_VERTEX_SHADER);
glShaderSource(vertexShader, 1, &vertexShaderSource, NULL);
glCompileShader(vertexShader);

// Comprovar errors de compilació del vertex shader
int success;
char infoLog[512];
glGetShaderiv(vertexShader, GL_COMPILE_STATUS, &success);
if (!success)
{
    glGetShaderInfoLog(vertexShader, 512, NULL, infoLog);
    std::cout << "ERROR::SHADER::VERTEX::COMPILATION_FAILED\n" << infoLog << std::endl;
}

// Compilar el fragment shader
unsigned int fragmentShader = glCreateShader(GL_FRAGMENT_SHADER);
glShaderSource(fragmentShader, 1, &fragmentShaderSource, NULL);
glCompileShader(fragmentShader);

// Comprovar errors de compilació del fragment shader
glGetShaderiv(fragmentShader, GL_COMPILE_STATUS, &success);
if (!success)
{
    glGetShaderInfoLog(fragmentShader, 512, NULL, infoLog);
    std::cout << "ERROR::SHADER::FRAGMENT::COMPILATION_FAILED\n" << infoLog << std::endl;
}

// Enllaçar els shaders
unsigned int shaderProgram = glCreateProgram();
glAttachShader(shaderProgram, vertexShader);
glAttachShader(shaderProgram, fragmentShader);
glLinkProgram(shaderProgram);

// Comprovar errors d'enllaç
glGetProgramiv(shaderProgram, GL_LINK_STATUS, &success);
if (!success) {
    glGetProgramInfoLog(shaderProgram, 512, NULL, infoLog);
    std::cout << "ERROR::SHADER::PROGRAM::LINKING_FAILED\n" << infoLog << std::endl;
}

// Netejar els shaders ja que ja no són necessaris
glDeleteShader(vertexShader);
glDeleteShader(fragmentShader);
  • glCreateShader: Crea un objecte shader.
  • glShaderSource: Estableix el codi font del shader.
  • glCompileShader: Compila el shader.
  • glCreateProgram: Crea un programa de shaders.
  • glAttachShader: Adjunta un shader al programa.
  • glLinkProgram: Enllaça el programa de shaders.
  • glDeleteShader: Elimina l'objecte shader després de l'enllaç.

Utilitzar Shaders en OpenGL

Un cop els shaders estan compilats i enllaçats en un programa, els pots utilitzar en el teu bucle de renderització.

Exemple de Codi per Utilitzar el Programa de Shaders

// Utilitzar el programa de shaders
glUseProgram(shaderProgram);

// Bucle de renderització
while (!glfwWindowShouldClose(window))
{
    // Comandes de renderització aquí...

    // Intercanviar buffers i gestionar esdeveniments d'entrada/sortida
    glfwSwapBuffers(window);
    glfwPollEvents();
}
  • glUseProgram: Activa el programa de shaders per a la renderització.

Conclusió

Els shaders són una eina poderosa en OpenGL que et permeten controlar la pipeline gràfica i crear efectes visuals complexos. Entenent els vertex i fragment shaders, i aprenent a compilar-los i enllaçar-los, pots començar a crear els teus propis efectes gràfics personalitzats. Practica escrivint shaders simples i gradualment passa a shaders més complexos per aprofitar al màxim el poder dels shaders d'OpenGL.
