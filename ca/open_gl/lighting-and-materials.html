Introducció a la Il·luminació en OpenGL

La il·luminació és un aspecte crucial dels gràfics 3D que afegeix realisme a les escenes simulant com la llum interactua amb els objectes. En OpenGL, la il·luminació implica configurar fonts de llum i definir com els objectes responen a aquestes llums.

Conceptes Clau

  • Fonts de Llum: Punts a l'escena que emeten llum.
  • Propietats del Material: Característiques dels objectes que determinen com reflecteixen la llum.
  • Models d'Ombrejat: Mètodes per calcular el color dels píxels basats en la llum i les propietats del material.

Tipus de Fonts de Llum

OpenGL suporta diversos tipus de fonts de llum, cadascuna amb propietats i usos únics.

Llum Direccional

  • Descripció: Simula la llum que ve d'una direcció específica, com la llum solar.
  • Propietats: Direcció, color, intensitat.

Llum Puntual

  • Descripció: Emmet llum en totes les direccions des d'un punt únic, com una bombeta.
  • Propietats: Posició, color, intensitat, atenuació.

Llum de Punt

  • Descripció: Emmet llum en una direcció específica dins d'un con, com una llanterna.
  • Propietats: Posició, direcció, angle de tall, color, intensitat, atenuació.

Propietats del Material

Els materials defineixen com els objectes interactuen amb la llum. Les propietats clau inclouen:

  • Ambient: El color de l'objecte sota llum ambiental.
  • Difús: El color de l'objecte sota llum directa.
  • Especular: El color dels reflexos de l'objecte.
  • Brillantor: La mida i intensitat dels reflexos especulars.

Implementació de la Il·luminació en OpenGL

Per implementar la il·luminació en OpenGL, cal configurar shaders que calculin els efectes de la il·luminació basats en les fonts de llum i les propietats del material.

Configuració Bàsica del Shader

Aquí hi ha un exemple senzill de shaders de vèrtex i fragment que implementen una il·luminació bàsica.

Shader de Vèrtex

#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;

out vec3 FragPos;
out vec3 Normal;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

void main()
{
    FragPos = vec3(model * vec4(aPos, 1.0));
    Normal = mat3(transpose(inverse(model))) * aNormal;  
    
    gl_Position = projection * view * vec4(FragPos, 1.0);
}

Shader de Fragment

#version 330 core
out vec4 FragColor;

in vec3 FragPos;
in vec3 Normal;

uniform vec3 lightPos;
uniform vec3 viewPos;
uniform vec3 lightColor;
uniform vec3 objectColor;

void main()
{
    // Ambient
    float ambientStrength = 0.1;
    vec3 ambient = ambientStrength * lightColor;
    
    // Difús 
    vec3 norm = normalize(Normal);
    vec3 lightDir = normalize(lightPos - FragPos);
    float diff = max(dot(norm, lightDir), 0.0);
    vec3 diffuse = diff * lightColor;
    
    // Especular
    float specularStrength = 0.5;
    vec3 viewDir = normalize(viewPos - FragPos);
    vec3 reflectDir = reflect(-lightDir, norm);  
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), 32);
    vec3 specular = specularStrength * spec * lightColor;  
    
    vec3 result = (ambient + diffuse + specular) * objectColor;
    FragColor = vec4(result, 1.0);
}

Explicació

  • Shader de Vèrtex: Transforma les posicions dels vèrtexs i les normals a l'espai mundial.
  • Shader de Fragment: Calcula els components ambient, difús i especular basats en les propietats de la llum i del material.

Tècniques Avançades d'Il·luminació

A mesura que progressis, pots explorar tècniques d'il·luminació més avançades com:

  • Múltiples Fonts de Llum: Gestionar múltiples llums a l'escena.
  • Ombrejat Phong: Un model d'ombrejat més precís en comparació amb l'ombrejat Gouraud.
  • Mapatge de Normals: Millorar els detalls de la superfície sense augmentar la complexitat de la geometria.
  • Mapatge d'Ombres: Afegir ombres per augmentar el realisme.

Exercicis

  1. Il·luminació Bàsica: Implementa una escena amb una única llum direccional i un cub. Ajusta la direcció de la llum i observa els canvis.
  2. Múltiples Llums: Afegeix múltiples fonts de llum (direccional, puntual i de punt) a l'escena i configura les seves propietats.
  3. Propietats del Material: Experimenta amb diferents propietats del material (ambient, difús, especular, brillantor) i observa els seus efectes sobre l'objecte.

Conclusió

La il·luminació i els materials són fonamentals per crear escenes 3D realistes en OpenGL. Comprenent i implementant diferents tipus de fonts de llum i propietats del material, pots millorar significativament la qualitat visual dels teus gràfics. A mesura que avancis, explorar tècniques d'il·luminació més complexes millorarà encara més les teves habilitats i el realisme de les teves escenes.
