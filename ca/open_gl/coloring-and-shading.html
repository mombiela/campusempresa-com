Introducció a la Coloració en OpenGL

La coloració és un dels aspectes fonamentals del renderitzat en OpenGL. Implica assignar colors als vèrtexs, que després s'interpolen a través de la superfície dels primitius (triangles, línies, etc.).

Conceptes Clau

  • Colors de Vèrtex: Colors assignats a cada vèrtex.
  • Colors de Fragment: Colors calculats per a cada píxel (fragment) durant la rasterització.
  • Interpolació: El procés de barrejar els colors dels vèrtexs a través de la superfície d'un primitiu.

Exemple Bàsic de Coloració

Aquí teniu un exemple senzill de com assignar colors als vèrtexs en OpenGL:

// Vertex Shader
#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aColor;

out vec3 ourColor;

void main()
{
    gl_Position = vec4(aPos, 1.0);
    ourColor = aColor;
}

// Fragment Shader
#version 330 core
in vec3 ourColor;
out vec4 FragColor;

void main()
{
    FragColor = vec4(ourColor, 1.0);
}

En aquest exemple:

  • Les posicions i els colors dels vèrtexs es passen al vertex shader.
  • El vertex shader produeix el color que s'interpolarà a través del primitiu.
  • El fragment shader rep el color interpolat i el configura com el color de sortida.

Introducció a l'Ombrejat en OpenGL

L'ombrejat es refereix al procés de calcular el color de cada píxel basat en la il·luminació, les propietats del material i altres factors.

Conceptes Clau

  • Ombrejat Pla: Cada primitiu es pinta amb un sol color.
  • Ombrejat Gouraud: Els colors es calculen als vèrtexs i s'interpolen a través del primitiu.
  • Ombrejat Phong: Els colors es calculen per píxel, proporcionant efectes d'il·luminació més realistes.

Exemple Bàsic d'Ombrejat

Aquí teniu un exemple senzill d'ombrejat Phong en OpenGL:

// Vertex Shader
#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;

out vec3 FragPos;
out vec3 Normal;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

void main()
{
    FragPos = vec3(model * vec4(aPos, 1.0));
    Normal = mat3(transpose(inverse(model))) * aNormal;  
    gl_Position = projection * view * vec4(FragPos, 1.0);
}

// Fragment Shader
#version 330 core
in vec3 FragPos;
in vec3 Normal;

out vec4 FragColor;

uniform vec3 lightPos;
uniform vec3 viewPos;
uniform vec3 lightColor;
uniform vec3 objectColor;

void main()
{
    // Ambient
    float ambientStrength = 0.1;
    vec3 ambient = ambientStrength * lightColor;
    
    // Difús 
    vec3 norm = normalize(Normal);
    vec3 lightDir = normalize(lightPos - FragPos);
    float diff = max(dot(norm, lightDir), 0.0);
    vec3 diffuse = diff * lightColor;
    
    // Especular
    float specularStrength = 0.5;
    vec3 viewDir = normalize(viewPos - FragPos);
    vec3 reflectDir = reflect(-lightDir, norm);  
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), 32);
    vec3 specular = specularStrength * spec * lightColor;  
    
    vec3 result = (ambient + diffuse + specular) * objectColor;
    FragColor = vec4(result, 1.0);
}

En aquest exemple:

  • El vertex shader calcula la posició i la normal de cada vèrtex.
  • El fragment shader calcula els components d'il·luminació ambient, difusa i especular per determinar el color final de cada píxel.

Tècniques Avançades d'Ombrejat

A mesura que progressis, pots explorar tècniques d'ombrejat més avançades com:

  • Mapatge de Normals: Afegeix detall a les superfícies sense augmentar la complexitat de la geometria.
  • Mapatge de Parallax: Millora la il·lusió de profunditat en superfícies texturitzades.
  • Renderitzat Basat en Física (PBR): Simula les propietats físiques dels materials per a un renderitzat més realista.

Exemple de Mapatge de Normals

Aquí teniu un breu exemple de mapatge de normals:

// Vertex Shader
#version 330 core
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;
layout(location = 2) in vec2 aTexCoords;
layout(location = 3) in vec3 aTangent;

out vec2 TexCoords;
out vec3 FragPos;
out mat3 TBN;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

void main()
{
    FragPos = vec3(model * vec4(aPos, 1.0));
    TexCoords = aTexCoords;
    
    vec3 T = normalize(mat3(model) * aTangent);
    vec3 N = normalize(mat3(model) * aNormal);
    vec3 B = cross(N, T);
    TBN = mat3(T, B, N);
    
    gl_Position = projection * view * vec4(FragPos, 1.0);
}

// Fragment Shader
#version 330 core
in vec2 TexCoords;
in vec3 FragPos;
in mat3 TBN;

uniform vec3 lightColor;
uniform vec3 objectColor;

void main()
{
    vec3 normal = texture(normalMap, TexCoords).rgb;
    normal = normalize(normal * 2.0 - 1.0);
    normal = normalize(TBN * normal);

}

En aquest exemple:

  • El vertex shader calcula la matriu TBN per transformar les normals de l'espai tangent a l'espai mundial.
  • El fragment shader utilitza el mapa de normals per pertorbar les normals de la superfície per obtenir efectes d'il·luminació més detallats.

Conclusió

La coloració i l'ombrejat són tècniques essencials en OpenGL per crear gràfics visualment atractius. A partir de la coloració bàsica dels vèrtexs, pots progressar cap a tècniques d'ombrejat més avançades com l'ombrejat Phong i el mapatge de normals. Dominar aquestes tècniques millorarà significativament el realisme i la qualitat visual de les teves aplicacions OpenGL.
