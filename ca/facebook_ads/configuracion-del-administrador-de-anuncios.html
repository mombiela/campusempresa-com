Introducció

L'Administrador d'Anuncis de Facebook és una eina essencial per a la creació, gestió i anàlisi de campanyes publicitàries a la plataforma. En aquesta secció, aprendràs a configurar el teu compte d'Administrador d'Anuncis, crear campanyes i optimitzar els teus anuncis per assolir els teus objectius de màrqueting.

Creació d'un Compte d'Administrador d'Anuncis

Per començar a utilitzar l'Administrador d'Anuncis, primer necessites crear un compte. Segueix aquests passos:

  1. Accedeix a Facebook Business: Ves a Facebook Business.
  2. Inicia sessió: Usa el teu compte personal de Facebook per iniciar sessió.
  3. Crea un compte de Business Manager: Si encara no en tens una, segueix les instruccions per crear-ne una.
  4. Accedeix a l'Administrador d'Anuncis: Un cop dins de Business Manager, selecciona "Administrador d'Anuncis" en el menú principal.

Configuració Inicial

Configuració del Compte Publicitari

  1. Selecciona el compte publicitari: Si ja tens un compte publicitari, selecciona-la. Si no, crea'n una de nova.
  2. Configura la moneda i zona horària: Assegura't de configurar la moneda i la zona horària correctament, ja que no podràs canviar-les després.
  3. Mètodes de pagament: Afegeix un mètode de pagament vàlid per poder començar a publicar anuncis.

Configuració del Píxel de Facebook

El Píxel de Facebook és una eina d'anàlisi que et permet mesurar l'efectivitat de la teva publicitat en entendre les accions que les persones realitzen al teu lloc web.

  1. Crear un Píxel:
    • Ves a la secció de "Píxels" a l'Administrador d'Esdeveniments.
    • Fes clic a "Afegeix" i segueix les instruccions per crear el teu píxel.
  2. Instal·lar el Píxel:
    • Copia el codi del píxel i enganxa'l a l'encapçalament del teu lloc web.
    • Alternativament, usa una integració de soci com Google Tag Manager.

Configuració d'Audiències

Les audiències són grups de persones a les quals desitges mostrar els teus anuncis. Pots crear diferents tipus d'audiències:

  • Audiències Personalitzades: Basades en dades dels teus clients, trànsit del lloc web o activitat a l'app.
  • Audiències Similars: Persones similars als teus clients existents.
  • Audiències Guardades: Basades en dades demogràfiques, interessos i comportaments.

Creació de Campanyes

Estructura de la Campanya

Una campanya a Facebook Ads es compon de tres nivells:

  1. Campanya: Defineix l'objectiu de màrqueting (per exemple, trànsit, conversions, reconeixement de marca).
  2. Conjunt d'Anuncis: Configura l'audiència, el pressupost, la programació i la ubicació dels anuncis.
  3. Anunci: Crea el contingut visual i el text de l'anunci.

Exemple de Configuració d'una Campanya

{
  "campaign": {
    "objective": "TRAFFIC"
  },
  "ad_set": {
    "name": "Conjunt d'Anuncis 1",
    "daily_budget": 1000,
    "start_time": "2023-10-01T00:00:00-0700",
    "end_time": "2023-10-31T23:59:59-0700",
    "targeting": {
      "geo_locations": {
        "countries": ["US"]
      },
      "age_min": 18,
      "age_max": 65,
      "genders": [1, 2],
      "interests": [
        {
          "id": "6003139266461",
          "name": "Online shopping"
        }
      ]
    }
  },
  "ad": {
    "name": "Anunci 1",
    "creative": {
      "title": "Compra ara",
      "body": "Descomptes increïbles a la nostra botiga en línia",
      "image_url": "https://example.com/image.jpg"
    }
  }
}

Optimització d'Anuncis

Per maximitzar el rendiment dels teus anuncis, considera les següents pràctiques:

  • Proves A/B: Crea múltiples versions dels teus anuncis per veure quina funciona millor.
  • Monitoratge i Ajust: Revisa regularment el rendiment dels teus anuncis i ajusta l'audiència, el pressupost o el contingut segons sigui necessari.
  • Ús de Retargeting: Torna a dirigir-te a les persones que han interactuat amb el teu lloc web o els teus anuncis anteriors.

Conclusió

La configuració de l'Administrador d'Anuncis de Facebook és un pas crucial per a qualsevol estratègia de màrqueting digital. Assegura't de seguir cada pas acuradament per optimitzar les teves campanyes i assolir els teus objectius de negoci. Amb una configuració adequada, podràs aprofitar al màxim les eines i funcionalitats que Facebook Ads ofereix.
