Introducció

En aquesta secció del curs sobre Facebook Ads, ens enfocarem en exercicis pràctics per a la creació de campanyes publicitàries. Aquests exercicis estan dissenyats per ajudar-te a aplicar els conceptes teòrics apresos i a desenvolupar habilitats pràctiques en la plataforma de Facebook Ads.

Exercici 1: Creació d'una Campanya de Reconeixement de Marca

Objectiu

L'objectiu d'aquest exercici és crear una campanya de reconeixement de marca que augmenti la visibilitat del teu negoci.

Passos a seguir

  1. Accedeix a Facebook Ads Manager:

    • Inicia sessió al teu compte de Facebook.
    • Dirigeix-te a Ads Manager.

  2. Crear una nova campanya:

    • Fes clic al botó "Crear" per iniciar una nova campanya.
    • Selecciona l'objectiu de la campanya: Reconeixement de marca.

  3. Configurar la campanya:

    • Nom de la campanya: Assigna un nom descriptiu a la teva campanya, per exemple, "Reconeixement de Marca - Primavera 2023".
    • Pressupost: Estableix un pressupost diari o total per a la campanya.

  4. Definir l'audiència:

    • Ubicació: Selecciona les ubicacions geogràfiques on desitges que es mostrin els teus anuncis.
    • Edat i gènere: Defineix el rang d'edat i el gènere de la teva audiència objectiu.
    • Interessos i comportaments: Utilitza les opcions de segmentació detallada per seleccionar interessos i comportaments rellevants per a la teva audiència.

  5. Seleccionar ubicacions d'anuncis:

    • Tria si desitges que Facebook optimitzi automàticament les ubicacions dels teus anuncis o selecciona manualment les ubicacions específiques (Facebook, Instagram, Audience Network, Messenger).

  6. Crear l'anunci:

    • Format de l'anunci: Selecciona el format del teu anunci (imatge, vídeo, carrusel, col·lecció).
    • Contingut de l'anunci: Puja les imatges o vídeos, escriu el text de l'anunci i afegeix una crida a l'acció (CTA).

  7. Revisar i publicar:

    • Revisa tots els detalls de la teva campanya.
    • Fes clic a "Publicar" per llançar la teva campanya.

Exemple de configuració d'anunci

{
  "name": "Reconeixement de Marca - Primavera 2023",
  "objective": "BRAND_AWARENESS",
  "status": "ACTIVE",
  "daily_budget": 1000,
  "targeting": {
    "geo_locations": {
      "countries": ["US"]
    },
    "age_min": 18,
    "age_max": 35,
    "genders": [1, 2],
    "interests": [
      {
        "id": "6003139266461",
        "name": "Shopping"
      }
    ]
  },
  "creative": {
    "title": "Descobreix la nostra nova col·lecció de primavera",
    "body": "Explora les últimes tendències en moda.",
    "image_url": "https://example.com/image.jpg",
    "cta": "Shop Now"
  }
}

Exercici 2: Creació d'una Campanya de Conversió

Objectiu

L'objectiu d'aquest exercici és crear una campanya de conversió per augmentar les vendes al teu lloc web.

Passos a seguir

  1. Accedeix a Facebook Ads Manager:

    • Inicia sessió al teu compte de Facebook.
    • Dirigeix-te a Ads Manager.

  2. Crear una nova campanya:

    • Fes clic al botó "Crear" per iniciar una nova campanya.
    • Selecciona l'objectiu de la campanya: Conversió.

  3. Configurar la campanya:

    • Nom de la campanya: Assigna un nom descriptiu a la teva campanya, per exemple, "Conversió - Venda d'Estiu 2023".
    • Pressupost: Estableix un pressupost diari o total per a la campanya.

  4. Definir l'audiència:

Conclusió

Aquests exercicis pràctics et proporcionen una guia pas a pas per crear campanyes de reconeixement de marca i de conversió a Facebook Ads. Practicar amb aquests exercicis t'ajudarà a familiaritzar-te amb la plataforma i a desenvolupar habilitats essencials per gestionar campanyes publicitàries efectives. Recorda sempre revisar i optimitzar les teves campanyes basant-te en els resultats obtinguts per millorar contínuament la teva estratègia publicitària.
