Introducció

Crear un compte publicitari a Facebook és el primer pas per començar a utilitzar Facebook Ads i aprofitar les seves capacitats per arribar a la teva audiència objectiu. En aquesta secció, aprendràs com configurar un compte publicitari des de zero, els requisits necessaris i alguns consells per optimitzar la teva configuració inicial.

Requisits previs

Abans de crear un compte publicitari a Facebook, assegura't de complir amb els següents requisits:

  • Tenir un compte personal de Facebook.
  • Tenir una pàgina de Facebook per al teu negoci o projecte.
  • Accés a un mètode de pagament vàlid (targeta de crèdit, PayPal, etc.).

Passos per crear un compte publicitari

1. Accedir a l'Administrador d'Anuncis

L'Administrador d'Anuncis és l'eina principal per crear i gestionar les teves campanyes publicitàries a Facebook. Segueix aquests passos per accedir:

  1. Inicia sessió al teu compte personal de Facebook.
  2. Ves a la pàgina principal de Facebook i fes clic al menú desplegable a la cantonada superior dreta.
  3. Selecciona "Gestionar anuncis" o visita directament Facebook Ads Manager.

2. Crear un compte publicitari

Un cop a l'Administrador d'Anuncis, segueix aquests passos per crear el teu compte publicitari:

  1. Al menú de l'esquerra, fes clic a "Configuració del negoci".
  2. A la secció "Comptes", selecciona "Comptes publicitaris".
  3. Fes clic al botó "Afegir" i selecciona "Crear un nou compte publicitari".
  4. Completa la informació sol·licitada:
    • Nom del compte publicitari.
    • Zona horària.
    • Moneda.
  5. Assigna permisos a les persones que gestionaran el compte publicitari.
  6. Fes clic a "Crear compte publicitari".

3. Configurar el mètode de pagament

Per activar el teu compte publicitari, has d'afegir un mètode de pagament vàlid:

  1. A l'Administrador d'Anuncis, ves a "Configuració de pagaments".
  2. Fes clic a "Afegir mètode de pagament".
  3. Selecciona el teu mètode de pagament preferit (targeta de crèdit, PayPal, etc.).
  4. Introdueix la informació requerida i guarda els canvis.

4. Verificar el compte publicitari

En alguns casos, Facebook pot sol·licitar una verificació addicional per assegurar l'autenticitat del teu compte publicitari. Segueix les instruccions proporcionades per Facebook per completar aquest procés.

Consells per optimitzar la teva configuració inicial

  • Nom del compte publicitari: Tria un nom descriptiu que et permeti identificar fàcilment el compte, especialment si gestiones múltiples comptes.
  • Zona horària i moneda: Configura aquests paràmetres d'acord amb la teva ubicació i preferències per evitar confusions en la facturació i programació d'anuncis.
  • Permisos i rols: Assigna rols i permisos adequats a les persones que gestionaran el compte per mantenir un control adequat i evitar errors.

Conclusió

Crear un compte publicitari a Facebook és un procés senzill però crucial per començar a utilitzar Facebook Ads de manera efectiva. Assegura't de seguir els passos esmentats i configurar correctament el teu compte per evitar problemes futurs. Amb un compte publicitari ben configurat, estaràs llest per crear i gestionar campanyes publicitàries que t'ajudin a assolir els teus objectius de màrqueting.
