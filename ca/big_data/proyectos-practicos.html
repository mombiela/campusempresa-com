Introducció

En aquesta secció del curs, ens enfocarem en projectes pràctics que et permetran aplicar els conceptes i tècniques apreses en Big Data. Aquests projectes estan dissenyats per reforçar la teva comprensió i habilitats en la gestió de grans volums de dades, utilitzant eines i tecnologies populars en la indústria.

Projecte 1: Anàlisi de Sentiments en Xarxes Socials

Descripció del Projecte

L'objectiu d'aquest projecte és analitzar els sentiments expressats en publicacions de xarxes socials sobre un tema específic. Utilitzarem tècniques de processament de llenguatge natural (NLP) i eines de Big Data per recopilar, processar i analitzar les dades.

Passos del Projecte

  1. Recopilació de Dades:

    • Utilitzar l'API de Twitter per recopilar tweets relacionats amb un tema específic.
    • Emmagatzemar les dades en un sistema d'emmagatzematge distribuït com Hadoop HDFS o Amazon S3.

  2. Processament de Dades:

    • Netejar i preprocessar les dades utilitzant Apache Spark.
    • Tokenitzar i normalitzar el text per facilitar l'anàlisi.

  3. Anàlisi de Sentiments:

    • Aplicar un model d'anàlisi de sentiments preentrenat (per exemple, VADER o TextBlob).
    • Classificar els tweets en categories de sentiments (positiu, negatiu, neutral).

  4. Visualització de Resultats:

    • Utilitzar eines de visualització com Tableau o Matplotlib per presentar els resultats de l'anàlisi.

Exemple de Codi

from pyspark.sql import SparkSession
from textblob import TextBlob

# Crear una sessió de Spark
spark = SparkSession.builder.appName("TwitterSentimentAnalysis").getOrCreate()

# Carregar dades de tweets des d'HDFS
tweets_df = spark.read.json("hdfs://path/to/tweets.json")

# Funció per analitzar el sentiment d'un tweet
def analyze_sentiment(tweet):
    analysis = TextBlob(tweet)
    if analysis.sentiment.polarity > 0:
        return 'positive'
    elif analysis.sentiment.polarity == 0:
        return 'neutral'
    else:
        return 'negative'

# Aplicar la funció d'anàlisi de sentiments als tweets
tweets_df = tweets_df.withColumn("sentiment", analyze_sentiment(tweets_df.text))

# Mostrar els resultats
tweets_df.show()

Projecte 2: Recomanador de Productes

Descripció del Projecte

En aquest projecte, desenvoluparem un sistema de recomanació de productes utilitzant tècniques de filtratge col·laboratiu. L'objectiu és recomanar productes als usuaris basant-nos en les seves preferències i comportaments passats.

Passos del Projecte

  1. Recopilació de Dades:

    • Obtenir dades de transaccions d'usuaris d'una botiga en línia.
    • Emmagatzemar les dades en un sistema d'emmagatzematge distribuït.

  2. Processament de Dades:

    • Netejar i preprocessar les dades utilitzant Apache Spark.
    • Crear una matriu d'usuari-producte que representi les interaccions.

  3. Construcció del Model de Recomanació:

    • Utilitzar la biblioteca MLlib de Spark per construir un model de filtratge col·laboratiu.
    • Entrenar el model amb les dades de transaccions.

  4. Generació de Recomanacions:

    • Utilitzar el model entrenat per generar recomanacions personalitzades per a cada usuari.
    • Avaluar la precisió del model utilitzant mètriques com RMSE.

Exemple de Codi

from pyspark.sql import SparkSession
from pyspark.ml.recommendation import ALS

# Crear una sessió de Spark
spark = SparkSession.builder.appName("ProductRecommender").getOrCreate()

# Carregar dades de transaccions des d'HDFS
transactions_df = spark.read.csv("hdfs://path/to/transactions.csv", header=True, inferSchema=True)

# Crear una matriu d'usuari-producte
user_product_df = transactions_df.select("user_id", "product_id", "rating")

# Construir el model de recomanació utilitzant ALS
als = ALS(userCol="user_id", itemCol="product_id", ratingCol="rating", coldStartStrategy="drop")
model = als.fit(user_product_df)

# Generar recomanacions per a tots els usuaris
user_recommendations = model.recommendForAllUsers(10)

# Mostrar les recomanacions
user_recommendations.show()
