Introducció a MapReduce i Hadoop

En el món del Big Data, MapReduce i Hadoop són tecnologies fonamentals per al processament i anàlisi de grans volums de dades. Aquest tema cobrirà els conceptes bàsics, l'arquitectura i exemples pràctics de com utilitzar MapReduce i Hadoop.

Què és MapReduce?

MapReduce és un model de programació per al processament de grans conjunts de dades amb un algorisme paral·lel i distribuït. Va ser desenvolupat per Google i es compon de dues funcions principals:

  • Map: Processa i transforma les dades d'entrada en parells clau-valor.
  • Reduce: Agrega i combina els resultats del pas Map per produir el resultat final.

Exemple de MapReduce

Suposem que volem comptar la freqüència de paraules en un conjunt de documents. El procés es pot dividir en els següents passos:

  1. Map: Llegir cada document i emetre parells clau-valor (paraula, 1).
  2. Shuffle and Sort: Agrupar totes les claus iguals.
  3. Reduce: Sumar els valors per a cada clau.

Codi d'Exemple en Python

from collections import defaultdict

# Pas Map
def map_function(document):
    for word in document.split():
        yield (word, 1)

# Pas Reduce
def reduce_function(pairs):
    word_count = defaultdict(int)
    for word, count in pairs:
        word_count[word] += count
    return word_count

# Documents d'exemple
documents = ["hello world", "hello Hadoop", "hello MapReduce"]

# Aplicar Map
mapped = []
for doc in documents:
    mapped.extend(map_function(doc))

# Aplicar Reduce
reduced = reduce_function(mapped)
print(reduced)

Què és Hadoop?

Hadoop és un marc de treball de codi obert que permet el processament distribuït de grans conjunts de dades a través de clústers d'ordinadors utilitzant models de programació simples. Hadoop està compost per diversos mòduls:

  • Hadoop Common: Les utilitats comunes que suporten els altres mòduls de Hadoop.
  • HDFS (Hadoop Distributed File System): Un sistema de fitxers distribuït que proporciona un alt rendiment d'accés a les dades.
  • YARN (Yet Another Resource Negotiator): Un sistema de gestió de recursos per a clústers.
  • Hadoop MapReduce: Un sistema basat en YARN per al processament paral·lel de grans conjunts de dades.

Arquitectura de Hadoop

| Component | Descripció | |-----------|------------| | HDFS | Sistema de fitxers distribuït que emmagatzema dades en blocs distribuïts a través de múltiples nodes. | | YARN | Framework per a la gestió de recursos i la programació de tasques. | | MapReduce | Model de programació per al processament de dades distribuïdes. |

Exemple de Treball MapReduce en Hadoop

Per executar un treball MapReduce en Hadoop, s'han de seguir els següents passos:

  1. Escriure el codi MapReduce: Implementar les funcions Map i Reduce.
  2. Compilar i empaquetar el codi: Crear un fitxer JAR si s'utilitza Java.
  3. Pujar les dades a HDFS: Col·locar les dades d'entrada en el sistema de fitxers distribuït.
  4. Executar el treball MapReduce: Utilitzar el comandament hadoop jar per executar el treball.
  5. Recuperar els resultats: Obtenir els resultats del HDFS.

Exemple de Codi en Java

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

import java.io.IOException;
import java.util.StringTokenizer;

public class WordCount {

  public static class TokenizerMapper extends Mapper<Object, Text, Text, IntWritable> {
    private final static IntWritable one = new IntWritable(1);
    private Text word = new Text();

    public void map(Object key, Text value, Context context) throws IOException, InterruptedException {
      StringTokenizer itr = new StringTokenizer(value.toString());
      while (itr.hasMoreTokens()) {
        word.set(itr.nextToken());
        context.write(word, one);
      }
    }
  }

  public static class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
    private IntWritable result = new IntWritable();

    public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
      int sum = 0;
      for (IntWritable val : values) {
        sum += val.get();
      }
      result.set(sum);
      context.write(key, result);
    }
  }

  public static void main(String[] args) throws Exception {
    Configuration conf = new Configuration();
    Job job = Job.getInstance(conf, "word count");
    job.setJarByClass(WordCount.class);
    job.setMapperClass(TokenizerMapper.class);
    job.setCombinerClass(IntSumReducer.class);
    job.setReducerClass(IntSumReducer.class);
    job.setOutputKeyClass(Text.class);
    job.setOutputValueClass(IntWritable.class);
    FileInputFormat.addInputPath(job, new Path(args[0]));
    FileOutputFormat.setOutputPath(job, new Path(args[1]));
    System.exit(job.waitForCompletion(true) ? 0 : 1);
  }
}

Conclusió

MapReduce i Hadoop són eines poderoses per al processament de grans volums de dades de manera eficient i distribuïda. Comprendre com funcionen i com implementar-los és essencial per a qualsevol professional de TI que treballi en el camp del Big Data. A través d'exemples pràctics i exercicis, els estudiants poden adquirir habilitats valuoses per manejar dades a gran escala.
