Introducció

Crear funcions personalitzades és una habilitat fonamental en la programació RPG (Report Program Generator). Les funcions permeten encapsular codi reutilitzable, fent que els teus programes siguin més modulars, llegibles i mantenibles. Aquesta secció et guiarà a través del procés de creació de funcions personalitzades en RPG, des de nivells bàsics fins a avançats.

Conceptes Clau

  • Definició de Funció: Com definir una funció en RPG.
  • Paràmetres: Passar dades a les funcions.
  • Valors de Retorn: Retornar dades des de les funcions.
  • Àmbit: Comprendre les variables locals i globals dins de les funcions.
  • Modularitat: Descompondre tasques complexes en funcions més simples.

Definició Bàsica de Funció

En RPG, les funcions es defineixen utilitzant les paraules clau DCL-PROC i END-PROC. Aquí tens un exemple senzill d'una funció que suma dos números:

DCL-PROC AddNumbers;
  DCL-PI AddNumbers INT(10);
    num1 INT(10);
    num2 INT(10);
  END-PI;

  DCL-S result INT(10);

  result = num1 + num2;

  RETURN result;
END-PROC;

Explicació

  • DCL-PROC AddNumbers;: Declara l'inici d'un procediment anomenat AddNumbers.
  • DCL-PI AddNumbers INT(10);: Declara la interfície del procediment, especificant que AddNumbers retorna un enter de longitud 10.
  • num1 INT(10); i num2 INT(10);: Paràmetres de la funció, ambdós enters de longitud 10.
  • DCL-S result INT(10);: Declara una variable local result per emmagatzemar la suma.
  • result = num1 + num2;: Suma els dos paràmetres i emmagatzema el resultat.
  • RETURN result;: Retorna el resultat al cridant.
  • END-PROC;: Marca el final del procediment.

Paràmetres i Valors de Retorn

Les funcions poden prendre múltiples paràmetres i retornar un sol valor. Aquí tens un exemple d'una funció que calcula l'àrea d'un rectangle:

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

Explicació

  • La funció CalculateArea pren dos paràmetres: length i width.
  • Calcula l'àrea multiplicant length i width.
  • El resultat s'emmagatzema en la variable local area i es retorna al cridant.

Àmbit de les Variables

Comprendre l'àmbit de les variables és crucial quan es treballa amb funcions. Les variables declarades dins d'una funció són locals a aquesta funció i no es poden accedir fora d'ella.

DCL-PROC ExampleScope;
  DCL-PI ExampleScope;
  END-PI;

  DCL-S localVar INT(10);

  localVar = 100;

  RETURN;
END-PROC;

Explicació

  • localVar és una variable local i només es pot accedir dins de la funció ExampleScope.
  • Intentar accedir a localVar fora de ExampleScope resultarà en un error.

Modularitat

Descompondre tasques complexes en funcions més simples fa que el teu codi sigui més modular i fàcil de mantenir. Aquí tens un exemple d'un programa que utilitza múltiples funcions per realitzar una sèrie de càlculs:

DCL-PROC Main;
  DCL-PI Main;
  END-PI;

  DCL-S length INT(10) INZ(5);
  DCL-S width INT(10) INZ(10);
  DCL-S area INT(10);
  DCL-S perimeter INT(10);

  area = CalculateArea(length: width);
  perimeter = CalculatePerimeter(length: width);

  RETURN;
END-PROC;

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

DCL-PROC CalculatePerimeter;
  DCL-PI CalculatePerimeter INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S perimeter INT(10);

  perimeter = 2 * (length + width);

  RETURN perimeter;
END-PROC;

Explicació

  • El procediment Main inicialitza length i width.
  • Crida CalculateArea i CalculatePerimeter per calcular l'àrea i el perímetre d'un rectangle.
  • Cada funció realitza una tasca específica, fent que el codi sigui més modular i fàcil d'entendre.

Conclusió

Crear funcions personalitzades en RPG et permet escriure codi més modular, mantenible i reutilitzable. Comprenent com definir funcions, passar paràmetres, retornar valors i gestionar l'àmbit de les variables, pots descompondre tasques complexes en peces més simples i manejables. Practica creant les teves pròpies funcions per convertir-te en un expert en aquesta habilitat essencial de programació.
