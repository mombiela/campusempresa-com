La interfície amb altres llenguatges de programació és una habilitat crucial per als desenvolupadors d'RPG (Report Program Generator). Aquest tema et guiarà a través de les tècniques bàsiques i avançades d'integració d'RPG amb altres llenguatges, com ara C, Java i SQL.

Introducció a la Interfície

La interfície permet als programes RPG aprofitar les funcionalitats d'altres llenguatges, millorant les seves capacitats i rendiment. Aquesta secció cobreix els conceptes fonamentals i els beneficis de la interfície.

  • Conceptes Clau:
    • Interoperabilitat: La capacitat de diferents sistemes i organitzacions per treballar junts.
    • APIs (Application Programming Interfaces): Conjunts de rutines, protocols i eines per construir programari i aplicacions.
    • Intercanvi de Dades: Mètodes per transferir dades entre diferents entorns de programació.

Cridant Funcions C des d'RPG

RPG pot cridar funcions C per realitzar tasques que no són possibles o són ineficients en RPG. Aquesta secció explica com cridar funcions C des d'RPG.

  • Passos per Cridar Funcions C:

    1. Defineix el Prototip de la Funció C en RPG: 
      D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE
    2. Crida la Funció C en RPG: 
      C                   eval      result = myCFunction(10 : 20)

  • Exemple:

    // Definició de la funció C
int myCFunction(int a, int b) {
    return a + b;
}

    // Programa RPG que crida la funció C
D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = myCFunction(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Integrant Java amb RPG

Java ofereix una àmplia gamma de biblioteques i funcionalitats que es poden utilitzar en programes RPG. Aquesta secció cobreix la integració de Java amb RPG.

  • Passos per Cridar Mètodes Java:

    1. Defineix la Classe i el Mètode Java: 
      public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}
    2. Utilitza RPG per Cridar el Mètode Java: 
      D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

  • Exemple:

    // Definició de la classe Java
public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}

    // Programa RPG que crida el mètode Java
D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Utilitzant SQL en RPG

SQL es pot incrustar en programes RPG per realitzar operacions de base de dades de manera eficient. Aquesta secció explica com utilitzar SQL dins d'RPG.

  • Passos per Incrustar SQL en RPG:

    1. Inclou la Directiva del Precompilador SQL: 
      /EXEC SQL
    2. Escriu Sentències SQL dins del Programa RPG: 
      /EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

  • Exemple:

    // Programa RPG amb SQL incrustat
D recordCount     S               10I 0

/EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

C                   dsply     recordCount
C                   eval      *inlr = *on

Conclusió

La interfície d'RPG amb altres llenguatges com C, Java i SQL millora significativament la funcionalitat i el rendiment de les aplicacions RPG. Comprenent i aplicant aquestes tècniques d'interfície, els desenvolupadors d'RPG poden crear programes més robustos i versàtils. Practica aquests exemples i explora més per dominar l'art de la interfície en RPG.
