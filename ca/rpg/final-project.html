Introducció

En aquest projecte final, aplicaràs tot el que has après al llarg del curs per crear una aplicació completa de RPG (Role-Playing Game). Aquest projecte t'ajudarà a consolidar els teus coneixements i habilitats en desenvolupament de jocs, programació orientada a objectes, estructures de dades i més.

Visió General del Projecte

L'aplicació RPG inclourà les següents característiques:

  • Creació i gestió de personatges
  • Sistema d'inventari
  • Mecàniques de combat
  • Missions i progressió de la història
  • Funcionalitat de guardar i carregar el joc

Creació i Gestió de Personatges

Conceptes Clau

  • Classes i Objectes
  • Herència i Polimorfisme
  • Encapsulació

Exemple

class Character:
    def __init__(self, name, health, strength, defense):
        self.name = name
        self.health = health
        self.strength = strength
        self.defense = defense

    def attack(self, target):
        damage = self.strength - target.defense
        target.health -= damage
        return damage

class Warrior(Character):
    def __init__(self, name):
        super().__init__(name, health=100, strength=20, defense=10)

class Mage(Character):
    def __init__(self, name):
        super().__init__(name, health=70, strength=30, defense=5)

# Exemple d'ús
hero = Warrior("Aragorn")
villain = Mage("Saruman")
damage_dealt = hero.attack(villain)
print(f"{hero.name} ha causat {damage_dealt} de dany a {villain.name}.")

Sistema d'Inventari

Conceptes Clau

  • Llistes i Diccionaris
  • Mètodes i Funcions

Exemple

class Inventory:
    def __init__(self):
        self.items = {}

    def add_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] += quantity
        else:
            self.items[item_name] = quantity

    def remove_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] -= quantity
            if self.items[item_name] <= 0:
                del self.items[item_name]

# Exemple d'ús
inventory = Inventory()
inventory.add_item("Poció de Salut", 3)
inventory.add_item("Poció de Mana")
inventory.remove_item("Poció de Salut")
print(inventory.items)

Mecàniques de Combat

Conceptes Clau

  • Declaracions condicionals
  • Bucles

Exemple

def combat_round(attacker, defender):
    damage = attacker.attack(defender)
    print(f"{attacker.name} ataca {defender.name} per {damage} de dany.")
    si la salut del defensor <= 0:
        print(f"{defender.name} ha estat derrotat!")
        return True
    return False

# Exemple d'ús
hero = Warrior("Aragorn")
villain = Mage("Saruman")
while villain.health > 0 and hero.health > 0:
    if combat_round(hero, villain):
        break
    if combat_round(villain, hero):
        break

Missions i Progressió de la Història

Conceptes Clau

  • Classes i Objectes
  • Llistes i Diccionaris

Exemple

class Quest:
    def __init__(self, name, description, is_completed=False):
        self.name = name
        self.description = description
        self.is_completed = is_completed

    def complete_quest(self):
        self.is_completed = True

# Exemple d'ús
quest1 = Quest("Salvar el Poble", "Derrota els goblins que ataquen el poble.")
print(f"Missió: {quest1.name} - {quest1.description}")
quest1.complete_quest()
print(f"Missió Completa: {quest1.is_completed}")

Funcionalitat de Guardar i Carregar el Joc

Conceptes Clau

  • Entrada/Sortida de Fitxers
  • Serialització i Deserialització

Exemple

import json

class Game:
    def __init__(self, character, inventory):
        self.character = character
        self.inventory = inventory

    def save_game(self, filename):
        game_data = {
            'character': self.character.__dict__,
            'inventory': self.inventory.items
        }
        with open(filename, 'w') as file:
            json.dump(game_data, file)

    def load_game(self, filename):
        with open(filename, 'r') as file:
            game_data = json.load(file)
            self.character.__dict__.update(game_data['character'])
            self.inventory.items = game_data['inventory']

# Exemple d'ús
hero = Warrior("Aragorn")
inventory = Inventory()
game = Game(hero, inventory)
game.save_game("savefile.json")
game.load_game("savefile.json")

En completar aquest projecte, has demostrat la teva capacitat per dissenyar, implementar i gestionar una aplicació de programari complexa.
