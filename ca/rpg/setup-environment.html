Configurant el teu entorn de desenvolupament
Sintaxi i Estructura Bàsiques ►

Configurar el teu entorn de desenvolupament és un primer pas crucial per aprendre RPG (Report Program Generator). Aquesta guia et guiarà a través dels passos necessaris per preparar el teu entorn per a la programació RPG, des de la instal·lació del programari necessari fins a la configuració del teu IDE (Entorn de Desenvolupament Integrat).

Instal·lant el programari necessari

Abans de començar a escriure codi RPG, necessites instal·lar el programari necessari. Aquests són els components clau que necessitaràs:

  • IBM i Access Client Solutions (ACS): Aquesta és l'eina principal per accedir als sistemes IBM i.
  • RDi (Rational Developer for i): Un IDE dissenyat específicament per al desenvolupament RPG.
  • Sistema IBM i: El servidor on s'executaran els teus programes RPG.

IBM i Access Client Solutions (ACS)

IBM i Access Client Solutions és una interfície basada en Java que et permet interactuar amb els sistemes IBM i. Segueix aquests passos per instal·lar ACS:

  1. Descarrega ACS: Visita el lloc web d'IBM i descarrega l'última versió d'IBM i Access Client Solutions.
  2. Instal·la Java: Assegura't de tenir la darrera versió de Java instal·lada al teu ordinador, ja que ACS la requereix.
  3. Executa l'instal·lador: Executa l'instal·lador d'ACS descarregat i segueix les instruccions en pantalla.

Rational Developer for i (RDi)

RDi és un IDE basat en Eclipse dissenyat per al desenvolupament RPG. Aquí tens com configurar-lo:

  1. Descarrega RDi: Obté l'última versió de RDi des del lloc web d'IBM.
  2. Instal·la RDi: Executa l'instal·lador i segueix les instruccions per completar la instal·lació.
  3. Configura RDi: Obre RDi i configura'l per connectar-te al teu sistema IBM i.

Sistema IBM i

Un sistema IBM i és essencial per executar programes RPG. Si no tens accés a un, pots utilitzar un servei IBM i basat en el núvol o configurar una instància local si tens el maquinari necessari.

Configurant el teu IDE

Un cop hagis instal·lat el programari necessari, el següent pas és configurar el teu IDE per al desenvolupament RPG.

Connectant-se a IBM i

  1. Obre RDi: Llença Rational Developer for i.
  2. Crea una nova connexió:
    • Ves a File > New > Connection.
    • Introdueix el nom de l'amfitrió o l'adreça IP del teu sistema IBM i.
    • Proporciona les teves credencials per connectar-te.

Configurant el teu espai de treball

  1. Crea un nou projecte:
    • Ves a File > New > RPG Project.
    • Dona un nom al teu projecte i selecciona la configuració adequada.
  2. Configura els fitxers de codi font:
    • Configura les ubicacions dels teus fitxers de codi font.
    • Defineix la llista de biblioteques per al teu projecte.

Exemple: Creant un programa RPG senzill

Aquí tens un exemple senzill per assegurar-te que el teu entorn està configurat correctament:

**free
dcl-s message char(50) inz('Hello, RPG World!');
dsply message;
*inlr = *on;
  • Explicació:
    • **free: Indica l'inici del codi RPG en format lliure.
    • dcl-s: Declara una variable autònoma.
    • char(50): Especifica el tipus i la longitud de la variable.
    • inz('Hello, RPG World!'): Inicialitza la variable amb una cadena.
    • dsply message: Mostra el missatge a la pantalla.
    • *inlr = *on: Finalitza el programa.

Conclusió

Configurar el teu entorn de desenvolupament és la base del teu viatge en la programació RPG. Instal·lant IBM i Access Client Solutions, Rational Developer for i, i configurant el teu IDE, assegures un bon començament. Recorda provar la teva configuració amb un programa RPG senzill per confirmar que tot funciona correctament. Amb el teu entorn preparat, ara pots centrar-te en aprendre i dominar la programació RPG.
