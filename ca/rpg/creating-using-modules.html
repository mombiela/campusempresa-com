Introducció

Els mòduls en RPG (Report Program Generator) són una manera d'organitzar i reutilitzar codi. Ajuden a descompondre programes complexos en peces manejables, millorant la llegibilitat, mantenibilitat i reutilització. Aquesta secció et guiarà a través dels conceptes bàsics de la creació i ús de mòduls en RPG, des de nivells principiants fins a avançats.

Què és un Mòdul?

  • Un mòdul és una unitat de codi autònoma que realitza una tasca específica.
  • Els mòduls es poden compilar per separat i després enllaçar-se per formar un programa complet.
  • Promouen la reutilització de codi i el disseny modular.

Creació d'un Mòdul Simple

Guia Pas a Pas

  1. Defineix el Mòdul:

    • Utilitza l'especificació H per definir el mòdul.
    • Utilitza l'especificació P per definir procediments dins del mòdul.

  2. Escriu el Codi:

    • Escriu la lògica per al procediment dins del mòdul.

  3. Compila el Mòdul:

    • Utilitza les ordres de compilador adequades per compilar el mòdul.

Exemple

H NOMAIN

P AddNumbers       B                   EXPORT
D AddNumbers       PI            10I 0
D   num1           10I 0 VALUE
D   num2           10I 0 VALUE

 /FREE
     RETURN num1 + num2;
 /END-FREE
P AddNumbers       E
  • Explicació:
    • H NOMAIN: Indica que aquest és un mòdul i no un programa independent.
    • P AddNumbers B EXPORT: Comença la definició d'un procediment anomenat AddNumbers i el fa disponible per a altres programes.
    • D AddNumbers PI 10I 0: Defineix la interfície del procediment amb un tipus de retorn d'enter de 10 dígits.
    • D num1 10I 0 VALUE: Defineix el primer paràmetre del procediment.
    • D num2 10I 0 VALUE: Defineix el segon paràmetre del procediment.
    • La lògica dins de /FREE i /END-FREE calcula la suma de num1 i num2 i retorna el resultat.

Ús d'un Mòdul

Guia Pas a Pas

  1. Importa el Mòdul:

    • Utilitza l'especificació H per importar el mòdul.
    • Utilitza l'especificació D per declarar el procediment.

  2. Crida el Procediment:

    • Crida el procediment des del teu programa principal.

Exemple

H DFTACTGRP(*NO) ACTGRP(*NEW)

D AddNumbers       PR            10I 0
D   num1           10I 0 VALUE
D   num2           10I 0 VALUE

 /COPY QRPGLESRC,MYMODULE

D result           S             10I 0

 /FREE
     result = AddNumbers(5: 10);
     dsply result;
     *INLR = *ON;
 /END-FREE
  • Explicació:
    • H DFTACTGRP(*NO) ACTGRP(*NEW): Defineix el grup d'activació per al programa.
    • D AddNumbers PR 10I 0: Declara el prototip per al procediment AddNumbers.
    • /COPY QRPGLESRC,MYMODULE: Inclou el codi font del mòdul.
    • result = AddNumbers(5: 10): Crida el procediment AddNumbers amb els paràmetres 5 i 10.
    • dsply result: Mostra el resultat.
    • *INLR = *ON: Finalitza el programa.

Conceptes Avançats de Mòduls

Disseny Modular

  • Descompon tasques complexes en mòduls més petits i reutilitzables.
  • Utilitza mòduls per encapsular la lògica de negoci, l'accés a dades i les funcions d'utilitat.

Gestió d'Errors en Mòduls

  • Implementa la gestió d'errors dins dels mòduls per assegurar la robustesa.
  • Utilitza funcions integrades com MONITOR, ON-ERROR i ENDMON per gestionar excepcions.

Exemple

P SafeDivide       B                   EXPORT
D SafeDivide       PI            10I 0
D   num1           10I 0 VALUE
D   num2           10I 0 VALUE

D result           S             10I 0

 /FREE
     MONITOR;
         result = num1 / num2;
     ON-ERROR;
         result = 0;
     ENDMON;
     RETURN result;
 /END-FREE
P SafeDivide       E
  • Explicació:
    • MONITOR: Comença el bloc de gestió d'errors.
    • ON-ERROR: Especifica l'acció a prendre si es produeix un error.
    • ENDMON: Finalitza el bloc de gestió d'errors.

Conclusió

Els mòduls en RPG són essencials per crear codi organitzat, mantenible i reutilitzable. Descomponent programes complexos en mòduls més petits, pots millorar l'estructura i la llegibilitat del teu codi. Aquesta secció ha cobert els conceptes bàsics de la creació i ús de mòduls, així com alguns conceptes avançats com el disseny modular i la gestió d'errors. Practica la creació i ús de mòduls per convertir-te en un expert en programació modular amb RPG.
