Introducció

Les estructures de control són fonamentals en la programació ja que permeten alterar el flux d'execució basant-se en certes condicions. En RPG (Report Program Generator), estructures de control com IF, ELSE i SELECT s'utilitzen per prendre decisions dins del codi.

Instrucció IF

La instrucció IF s'utilitza per executar un bloc de codi només si una condició especificada és certa.

Sintaxi

IF condition;
   // Codi a executar si la condició és certa
ENDIF;

Exemple

DCL-S age INT(3);
age = 20;

IF age >= 18;
   DSPLY 'You are an adult.';
ENDIF;
  • Explicació: En aquest exemple, el programa comprova si la variable age és major o igual a 18. Si la condició és certa, mostra el missatge "You are an adult."

Instrucció ELSE

La instrucció ELSE s'utilitza conjuntament amb la instrucció IF per executar un bloc de codi si la condició és falsa.

Sintaxi

IF condition;
   // Codi a executar si la condició és certa
ELSE;
   // Codi a executar si la condició és falsa
ENDIF;

Exemple

DCL-S age INT(3);
age = 16;

IF age >= 18;
   DSPLY 'You are an adult.';
ELSE;
   DSPLY 'You are a minor.';
ENDIF;
  • Explicació: Aquí, el programa comprova si age és major o igual a 18. Si és cert, mostra "You are an adult." Si és fals, mostra "You are a minor."

Instrucció ELSEIF

La instrucció ELSEIF permet comprovar múltiples condicions en seqüència.

Sintaxi

IF condition1;
   // Codi a executar si la condició1 és certa
ELSEIF condition2;
   // Codi a executar si la condició2 és certa
ELSE;
   // Codi a executar si cap de les condicions és certa
ENDIF;

Exemple

DCL-S score INT(3);
score = 75;

IF score >= 90;
   DSPLY 'Grade: A';
ELSEIF score >= 80;
   DSPLY 'Grade: B';
ELSEIF score >= 70;
   DSPLY 'Grade: C';
ELSE;
   DSPLY 'Grade: F';
ENDIF;
  • Explicació: El programa avalua el score i assigna una nota basada en el valor. Es comproven múltiples condicions en seqüència.

Instrucció SELECT

La instrucció SELECT s'utilitza per executar un bloc de codi d'entre múltiples blocs possibles basant-se en el valor d'una variable.

Sintaxi

SELECT;
   WHEN condition1;
      // Codi a executar si la condició1 és certa
   WHEN condition2;
      // Codi a executar si la condició2 és certa
   OTHER;
      // Codi a executar si cap de les condicions és certa
ENDSL;

Exemple

DCL-S day CHAR(10);
day = 'Monday';

SELECT;
   WHEN day = 'Monday';
      DSPLY 'Start of the work week.';
   WHEN day = 'Friday';
      DSPLY 'End of the work week.';
   OTHER;
      DSPLY 'Midweek day.';
ENDSL;
  • Explicació: El programa comprova el valor de la variable day i mostra un missatge basat en el dia de la setmana.

Comparació d'IF i SELECT

| Característica | IF/ELSEIF/ELSE | SELECT/WHEN/OTHER | |------------------|--------------------------|-----------------------------| | Cas d'ús | Condicions simples | Múltiples condicions basades en una sola variable | | Llegibilitat | Pot esdevenir complex | Més llegible per a múltiples condicions | | Flexibilitat | Alta | Limitada a una sola variable |

Exercicis

  1. Escriu un programa RPG que comprovi si un número és positiu, negatiu o zero utilitzant les instruccions IF, ELSEIF i ELSE.
  2. Crea un programa RPG que assigni una nota (A, B, C, D, F) basada en una puntuació numèrica utilitzant la instrucció SELECT.

Conclusió

Entendre estructures de control com IF, ELSE i SELECT és crucial per prendre decisions en els teus programes RPG. Aquestes estructures et permeten controlar el flux del teu programa basant-se en diverses condicions, fent el teu codi més dinàmic i responsiu. Practica utilitzant aquestes estructures de control per esdevenir proficient en escriure lògica condicional en RPG.
