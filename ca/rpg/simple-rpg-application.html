Introducció

En aquest curs, et guiarem a través del procés de construir una aplicació RPG (Role-Playing Game) simple des de zero. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat, cobrint tots els conceptes i tècniques essencials necessaris per crear un joc RPG funcional.

Esquema del Curs

  1. Configuració de l'Entorn de Desenvolupament
  2. Mecàniques Bàsiques del Joc
  3. Creació i Gestió de Personatges
  4. Sistema d'Inventari
  5. Sistema de Combat
  6. Guardar i Carregar l'Estat del Joc
  7. Característiques Avançades i Millores

1. Configuració de l'Entorn de Desenvolupament

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

  • Tria un Llenguatge de Programació: Per aquest curs, utilitzarem Python per la seva simplicitat i llegibilitat.
  • Instal·la Python: Assegura't de tenir Python instal·lat a la teva màquina. Pots descarregar-lo des de python.org.
  • IDE/Editor de Codi: Utilitza un IDE o editor de codi com PyCharm, VSCode, o fins i tot un editor de text simple com Sublime Text.

Exemple: Instal·lant Python

# Per a Windows
python -m pip install --upgrade pip

# Per a macOS/Linux
sudo apt-get install python3

2. Mecàniques Bàsiques del Joc

Començarem creant l'estructura bàsica del nostre joc RPG.

  • Bucle del Joc: El bucle principal que manté el joc en funcionament.
  • Entrada d'Usuari: Gestionar les entrades del jugador.
  • Estats del Joc: Gestionar diferents estats del joc (per exemple, menú principal, jugant, joc acabat).

Exemple: Bucle Bàsic del Joc

def main():
    running = True
    while running:
        print("Benvingut al Joc RPG!")
        command = input("Introdueix una comanda: ")
        if command == "quit":
            running = False
        else:
            print(f"Comanda desconeguda: {command}")

if __name__ == "__main__":
    main()

3. Creació i Gestió de Personatges

Crear i gestionar personatges és una part fonamental de qualsevol RPG.

  • Atributs del Personatge: Salut, força, agilitat, etc.
  • Classes i Objectes: Utilitzar classes per definir les propietats i comportaments dels personatges.

Exemple: Classe de Personatge

class Character:
    def __init__(self, name, health, strength, agility):
        self.name = name
        self.health = health
        self.strength = strength
        self.agility = agility

    def display_stats(self):
        print(f"Nom: {self.name}")
        print(f"Salut: {self.health}")
        print(f"Força: {self.strength}")
        print(f"Agilitat: {self.agility}")

# Creant un personatge
hero = Character("Heroi", 100, 10, 5)
hero.display_stats()

4. Sistema d'Inventari

Un sistema d'inventari permet als personatges portar objectes.

  • Classe d'Objecte: Definir objectes amb atributs com nom, tipus i valor.
  • Gestió d'Inventari: Afegir, eliminar i mostrar objectes.

Exemple: Sistema d'Inventari

class Item:
    def __init__(self, name, item_type, value):
        self.name = name
        self.item_type = item_type
        self.value = value

class Inventory:
    def __init__(self):
        self.items = []

    def add_item(self, item):
        self.items.append(item)

    def remove_item(self, item):
        self.items.remove(item)

    def display_inventory(self):
        for item in self.items:
            print(f"{item.name} ({item.item_type}): {item.value}")

# Creant objectes i afegint-los a l'inventari
espasa = Item("Espasa", "Arma", 10)
pocio = Item("Poció de Salut", "Consumible", 5)

inventari = Inventory()
inventari.add_item(espasa)
inventari.add_item(pocio)
inventari.display_inventory()

5. Sistema de Combat

Implementant un sistema de combat on els personatges poden participar en batalles.

  • Combat per Torn: Jugadors i enemics es tornen per atacar.
  • Càlcul de Dany: Basat en els atributs del personatge i factors aleatoris.

Exemple: Sistema de Combat Simple

import random

def attack(attacker, defender):
    damage = attacker.strength + random.randint(-2, 2)
    defender.health -= damage
    print(f"{attacker.name} ataca {defender.name} per {damage} de dany!")

# Exemple de combat
heroi = Character("Heroi", 100, 10, 5)
monstre = Character("Monstre", 50, 8, 3)

while heroi.health > 0 and monstre.health > 0:
    attack(heroi, monstre)
    if monstre.health <= 0:
        print("El Monstre ha estat derrotat!")
        break
    attack(monstre, heroi)
    if heroi.health <= 0:
        print("L'Heroi ha estat derrotat!")

6. Guardar i Carregar l'Estat del Joc

Permetre als jugadors guardar el seu progrés i carregar-lo més tard.

  • Serialització: Utilitzant JSON per guardar i carregar dades del joc.
  • Gestió de Fitxers: Llegir i escriure fitxers.

Exemple: Guardar i Carregar l'Estat del Joc

import json

def save_game(character, filename="savegame.json"):
    with open(filename, 'w') as file:
        json.dump(character.__dict__, file)

def load_game(filename="savegame.json"):
    with open(filename, 'r') as file:
        data = json.load(file)
        return Character(**data)

# Guardant i carregant un personatge
heroi = Character("Heroi", 100, 10, 5)
save_game(heroi)

heroi_carregat = load_game()
heroi_carregat.display_stats()

7. Característiques Avançades i Millores

Millora el teu RPG amb característiques avançades.

  • Missions i Trames: Afegint missions i elements de la història.
  • NPCs i Diàlegs: Personatges no jugadors i sistemes d'interacció.
  • Gràfics i So: Integrant gràfics bàsics i efectes de so.

Exemple: Afegint un Sistema de Missions

class Quest:
    def __init__(self, name, description, reward):
        self.name = name
        self.description = description
        self.reward = reward
        self.completed = False

    def complete_quest(self):
        self.completed = True
        print(f"Missió '{self.name}' completada! Recompensa: {self.reward}")

# Creant i completant una missió
missio = Quest("Troba l'Espasa Perduda", "Recupera l'espasa perduda de la cova.", "100 or")
missio.complete_quest()
