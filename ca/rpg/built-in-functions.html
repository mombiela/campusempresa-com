Introducció

Les funcions integrades en RPG (Report Program Generator) són funcions predefinides proporcionades pel llenguatge per realitzar tasques comunes. Aquestes funcions poden simplificar el teu codi, fer-lo més llegible i reduir la probabilitat d'errors. En aquesta secció, explorarem diverses funcions integrades, entendrem el seu ús i veurem com es poden aplicar en escenaris del món real.

Conceptes Clau

  • Definició: Les funcions integrades són rutines predefinides disponibles en RPG que realitzen tasques específiques.
  • Categories: Aquestes funcions es poden categoritzar en manipulació de cadenes, operacions matemàtiques, gestió de dates i temps, i més.
  • Sintaxi: Entendre la sintaxi i els paràmetres de cada funció és crucial per a un ús efectiu.

Funcions Integrades Comunes

Funcions de Manipulació de Cadenes

Les funcions de manipulació de cadenes ajuden a gestionar i modificar dades de cadena.

  • %TRIM: Elimina els espais inicials i finals d'una cadena.

    Dcl-S myString Char(20) Inz('  Hola Món  ');
Dcl-S trimmedString Char(20);

trimmedString = %Trim(myString);
// trimmedString ara conté 'Hola Món'

  • %SUBST: Extreu una subcadena d'una cadena.

    Dcl-S myString Char(20) Inz('Hola Món');
Dcl-S subString Char(5);

subString = %Subst(myString: 1: 5);
// subString ara conté 'Hola'

  • %SCAN: Cerca una subcadena dins d'una cadena i retorna la seva posició.

    Dcl-S myString Char(20) Inz('Hola Món');
Dcl-S position Int(10);

position = %Scan('Món': myString);
// position ara conté 6

Funcions Matemàtiques

Les funcions matemàtiques realitzen operacions aritmètiques i altres tasques matemàtiques.

  • %ABS: Retorna el valor absolut d'un nombre.

    Dcl-S myNumber Int(10) Inz(-5);
Dcl-S absValue Int(10);

absValue = %Abs(myNumber);
// absValue ara conté 5

  • %REM: Retorna el residu d'una operació de divisió.

    Dcl-S dividend Int(10) Inz(10);
Dcl-S divisor Int(10) Inz(3);
Dcl-S remainder Int(10);

remainder = %Rem(dividend: divisor);
// remainder ara conté 1

Funcions de Data i Temps

Les funcions de data i temps ajuden a gestionar i manipular valors de data i temps.

  • %DATE: Converteix una cadena a una data.

    Dcl-S dateString Char(10) Inz('2023-10-01');
Dcl-S myDate Date;

myDate = %Date(dateString);
// myDate ara conté la data 2023-10-01

  • %DIFF: Calcula la diferència entre dues dates.

    Dcl-S startDate Date Inz(D'2023-01-01');
Dcl-S endDate Date Inz(D'2023-10-01');
Dcl-S dateDiff Int(10);

dateDiff = %Diff(endDate: startDate: *D);
// dateDiff ara conté 273 (dies)

Exemples i Exercicis

Exemple 1: Ús de Funcions de Cadena

Dcl-S fullName Char(50) Inz('  John Doe  ');
Dcl-S firstName Char(20);
Dcl-S lastName Char(20);
Dcl-S position Int(10);

// Retalla el nom complet
fullName = %Trim(fullName);

// Troba la posició de l'espai
position = %Scan(' ': fullName);

// Extreu els noms i cognoms
firstName = %Subst(fullName: 1: position - 1);
lastName = %Subst(fullName: position + 1);

// Sortida: firstName = 'John', lastName = 'Doe'

Exercici 1: Calcula l'Edat

Escriu un programa que calculi l'edat d'una persona donada la seva data de naixement.

Dcl-S birthDate Date Inz(D'1990-05-15');
Dcl-S currentDate Date Inz(%Date());
Dcl-S age Int(10);

age = %Diff(currentDate: birthDate: *Y);
// Sortida: age contindrà el nombre d'anys des de birthDate

Conclusió

Les funcions integrades en RPG són eines poderoses que poden millorar molt l'eficiència de la teva programació. Entenent i utilitzant aquestes funcions, pots escriure codi més net i eficient. Practica utilitzant aquestes funcions en diversos escenaris per convertir-te en un expert en la seva aplicació.
