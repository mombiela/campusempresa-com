Introducció

En aquest estudi de cas, explorarem el desenvolupament d'un Sistema de Processament de Nòmines utilitzant RPG (Report Program Generator). Aquest sistema gestionarà dades dels empleats, calcularà salaris, generarà fulls de nòmina i gestionarà registres de nòmines. L'estudi de cas es dividirà en seccions que cobreixen diferents aspectes del sistema, des de conceptes bàsics fins a funcions avançades.

Conceptes Clau

  • Gestió de Dades dels Empleats: Emmagatzemar i recuperar informació dels empleats.
  • Càlcul de Salaris: Computar salaris bruts i nets basats en diversos factors.
  • Generació de Fulls de Nòmina: Crear fulls de nòmina detallats per als empleats.
  • Gestió de Registres de Nòmines: Mantenir i actualitzar registres de nòmines.

Gestió de Dades dels Empleats

Per començar, necessitem crear una base de dades per emmagatzemar informació dels empleats. Això inclou camps com ID de l'Empleat, Nom, Departament, Salari Bàsic, i més.

Exemple: Creació de la Base de Dades d'Empleats

**free
ctl-opt dftactgrp(*no) actgrp('EMPDB');

dcl-f EmployeeFile keyed;

dcl-ds EmployeeData qualified;
  EmployeeID char(10);
  Name char(50);
  Department char(20);
  BasicSalary packed(7:2);
end-ds;

dcl-proc CreateEmployee;
  dcl-pi *n;
    EmployeeID char(10);
    Name char(50);
    Department char(20);
    BasicSalary packed(7:2);
  end-pi;

  EmployeeData.EmployeeID = EmployeeID;
  EmployeeData.Name = Name;
  EmployeeData.Department = Department;
  EmployeeData.BasicSalary = BasicSalary;

  write EmployeeFile EmployeeData;
end-proc;

En aquest exemple, definim una estructura de dades EmployeeData i un procediment CreateEmployee per afegir nous registres d'empleats al EmployeeFile.

Càlcul de Salaris

A continuació, necessitem calcular els salaris. Això implica computar el salari brut, les deduccions i el salari net.

Exemple: Càlcul de Salaris

**free
dcl-proc CalculateSalary;
  dcl-pi *n packed(7:2);
    BasicSalary packed(7:2);
    HRA packed(7:2);
    OtherAllowances packed(7:2);
    Deductions packed(7:2);
  end-pi;

  dcl-s GrossSalary packed(7:2);
  dcl-s NetSalary packed(7:2);

  GrossSalary = BasicSalary + HRA + OtherAllowances;
  NetSalary = GrossSalary - Deductions;

  return NetSalary;
end-proc;

Aquest procediment CalculateSalary pren el salari bàsic, HRA, altres assignacions i deduccions com a entrada i retorna el salari net.

Generació de Fulls de Nòmina

Generar fulls de nòmina implica crear una sortida formatada que inclogui tots els detalls del salari per a un empleat.

Exemple: Generació de Fulls de Nòmina

**free
dcl-proc GeneratePaySlip;
  dcl-pi *n;
    EmployeeID char(10);
  end-pi;

  dcl-ds EmployeeData qualified;
    EmployeeID char(10);
    Name char(50);
    Department char(20);
    BasicSalary packed(7:2);
  end-ds;

  chain EmployeeID EmployeeFile EmployeeData;

  if %found(EmployeeFile);
    dsply ('Full de Nòmina per a ' + EmployeeData.Name);
    dsply ('Departament: ' + EmployeeData.Department);
    dsply ('Salari Bàsic: ' + %char(EmployeeData.BasicSalary));
    // Detalls addicionals com HRA, assignacions, deduccions, salari net
  else;
    dsply ('Empleat no trobat');
  endif;
end-proc;

Aquest procediment GeneratePaySlip recupera dades de l'empleat i mostra un full de nòmina formatat.

Gestió de Registres de Nòmines

Finalment, necessitem gestionar els registres de nòmines, incloent-hi l'actualització i l'eliminació de registres segons sigui necessari.

Exemple: Actualització del Registre d'Empleat

**free
dcl-proc UpdateEmployee;
  dcl-pi *n;
    EmployeeID char(10);
    NewBasicSalary packed(7:2);
  end-pi;

  chain EmployeeID EmployeeFile EmployeeData;

  if %found(EmployeeFile);
    EmployeeData.BasicSalary = NewBasicSalary;
    update EmployeeFile EmployeeData;
  else;
    dsply ('Empleat no trobat');
  endif;
end-proc;

Aquest procediment UpdateEmployee actualitza el salari bàsic d'un empleat.

Conclusió

En aquest estudi de cas, hem cobert els components essencials d'un Sistema de Processament de Nòmines utilitzant RPG. Hem començat amb la gestió de dades dels empleats, hem passat al càlcul de salaris, la generació de fulls de nòmina, i finalment, la gestió de registres de nòmines. Seguint aquests exemples, hauríeu de tenir una base sòlida per construir i expandir el vostre propi sistema de processament de nòmines en RPG.
