Introducció

En RPG (Report Program Generator), els operadors i les expressions són components fonamentals que et permeten realitzar càlculs, prendre decisions i manipular dades. Aquesta secció cobrirà els diversos tipus d'operadors disponibles en RPG i com utilitzar-los de manera efectiva en expressions.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • Addició (+): Suma dos operands.
  • Subtracció (-): Resta el segon operand del primer.
  • Multiplicació (*): Multiplica dos operands.
  • Divisió (/): Divideix el primer operand pel segon.
  • Exponenciació (**): Eleva el primer operand a la potència del segon.

Exemple

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S result Int(10);

result = num1 + num2; // result és 15
result = num1 - num2; // result és 5
result = num1 * num2; // result és 50
result = num1 / num2; // result és 2
result = num1 ** num2; // result és 100000

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors i retornar un resultat booleà (True o False).

  • Igual a (= o EQ)
  • No igual a (<> o NE)
  • Major que (> o GT)
  • Menor que (< o LT)
  • Major o igual a (>= o GE)
  • Menor o igual a (<= o LE)

Exemple

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S isEqual Ind;

isEqual = (num1 = num2); // isEqual és *False
isEqual = (num1 <> num2); // isEqual és *True
isEqual = (num1 > num2); // isEqual és *True
isEqual = (num1 < num2); // isEqual és *False
isEqual = (num1 >= num2); // isEqual és *True
isEqual = (num1 <= num2); // isEqual és *False

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions.

  • AND (& o AND): Retorna True si ambdues condicions són True.
  • OR (| o OR): Retorna True si almenys una condició és True.
  • NOT (NOT): Reverteix el valor booleà de la condició.

Exemple

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S num3 Int(10) Inz(15);
Dcl-S condition Ind;

condition = (num1 > num2) AND (num1 < num3); // condition és *True
condition = (num1 > num2) OR (num1 > num3); // condition és *True
condition = NOT(num1 = num2); // condition és *True

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació simple (=): Assigna el valor de l'operand dret a l'operand esquerre.
  • Assignació d'addició (+=): Suma l'operand dret a l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Assignació de subtracció (-=): Resta l'operand dret de l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Assignació de multiplicació (*=): Multiplica l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.
  • Assignació de divisió (/=): Divideix l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.

Exemple

Dcl-S num Int(10) Inz(10);

num = 5; // num és 5
num += 3; // num és 8
num -= 2; // num és 6
num *= 4; // num és 24
num /= 3; // num és 8

Operadors de Cadenes

Els operadors de cadenes s'utilitzen per manipular cadenes de text.

  • Concatenació (+): Combina dues cadenes en una.

Exemple

Dcl-S firstName Char(20) Inz('John');
Dcl-S lastName Char(20) Inz('Doe');
Dcl-S fullName Char(40);

fullName = firstName + ' ' + lastName; // fullName és 'John Doe'

Resum

En aquesta secció, hem cobert els diversos tipus d'operadors disponibles en RPG, incloent operadors aritmètics, de comparació, lògics, d'assignació i de cadenes. Entendre com utilitzar aquests operadors de manera efectiva és crucial per realitzar càlculs, prendre decisions i manipular dades en els teus programes RPG. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en escriure expressions en RPG.
