Introducció

La manipulació i el processament de dades són habilitats fonamentals en la programació RPG (Report Program Generator). Aquest tema cobrirà els conceptes bàsics de la gestió de dades, incloent la lectura, escriptura i modificació de dades en diversos formats. Progressarem des del nivell principiant fins al nivell avançat, assegurant una comprensió completa de la manipulació de dades en RPG.

Nivell Principiant

Lectura de Dades

  • Lectura de Fitxers Seqüencials: Aprèn a llegir dades de fitxers seqüencials.
  • Lectura de Fitxers de Base de Dades: Entén com llegir dades de fitxers de base de dades.

Exemple: Lectura d'un Fitxer Seqüencial

FMYFILE   IF   E           K DISK
D Record        DS
D  Field1               10A
D  Field2               20A

C     MYFILE      IF   E
C                   READ      MYFILE
C                   DOW       NOT %EOF(MYFILE)
C                   EVAL      Field1 = %SUBST(Record:1:10)
C                   EVAL      Field2 = %SUBST(Record:11:20)
C                   READ      MYFILE
C                   ENDDO
C                   ENDIF
  • Explicació: Aquest codi llegeix registres d'un fitxer seqüencial anomenat MYFILE i processa cada registre fins que s'arriba al final del fitxer.

Escriptura de Dades

  • Escriptura de Fitxers Seqüencials: Aprèn a escriure dades en fitxers seqüencials.
  • Escriptura de Fitxers de Base de Dades: Entén com escriure dades en fitxers de base de dades.

Exemple: Escriptura en un Fitxer Seqüencial

FMYFILE   O    F   80        DISK
D Record        DS
D  Field1               10A
D  Field2               20A

C                   EVAL      Field1 = 'Example1'
C                   EVAL      Field2 = 'Example2'
C                   EVAL      Record = %TRIM(Field1) + %TRIM(Field2)
C                   WRITE     MYFILE
  • Explicació: Aquest codi escriu un registre en un fitxer seqüencial anomenat MYFILE.

Nivell Intermedi

Modificació de Dades

  • Actualització de Registres: Aprèn a actualitzar registres existents en fitxers.
  • Eliminació de Registres: Entén com eliminar registres de fitxers.

Exemple: Actualització d'un Registre en un Fitxer de Base de Dades

FMYDBFILE IF   E           K DISK
D Record        DS
D  Field1               10A
D  Field2               20A

C     MYDBFILE   CHAIN     KeyField
C                   IF       %FOUND(MYDBFILE)
C                   EVAL      Field1 = 'UpdatedValue'
C                   UPDATE    MYDBFILE
C                   ENDIF
  • Explicació: Aquest codi actualitza un registre en un fitxer de base de dades anomenat MYDBFILE si es troba el registre.

Filtratge i Ordenació de Dades

  • Filtratge de Dades: Aprèn a filtrar dades basant-se en criteris específics.
  • Ordenació de Dades: Entén com ordenar dades en ordre ascendent o descendent.

Exemple: Filtratge i Ordenació de Dades

FMYDBFILE IF   E           K DISK
D Record        DS
D  Field1               10A
D  Field2               20A

C     MYDBFILE   SETLL     'A'
C                   READE    'A'
C                   DOW       NOT %EOF(MYDBFILE)
C                   IF       Field1 = 'Criteria'
C                   // Processa el registre
C                   ENDIF
C                   READE    'A'
C                   ENDDO
  • Explicació: Aquest codi llegeix i processa registres d'un fitxer de base de dades anomenat MYDBFILE que compleixen un criteri específic.

Nivell Avançat

Manipulació Complexa de Dades

  • Unió de Fitxers: Aprèn a unir múltiples fitxers per combinar dades.
  • Agregació de Dades: Entén com realitzar operacions d'agregació de dades.

Exemple: Unió de Dos Fitxers

FMYDBFILE1 IF   E           K DISK
FMYDBFILE2 IF   E           K DISK
D Record1       DS
D  Field1               10A
D  Field2               20A
D Record2       DS
D  Field3               10A
D  Field4               20A

C     MYDBFILE1 CHAIN     KeyField1
C                   IF       %FOUND(MYDBFILE1)
C                   EVAL      KeyField2 = Field1
C                   CHAIN     KeyField2 MYDBFILE2
C                   IF       %FOUND(MYDBFILE2)
C                   // Processa els registres units
C                   ENDIF
C                   ENDIF
  • Explicació: Aquest codi uneix registres de dos fitxers de base de dades, MYDBFILE1 i MYDBFILE2, basant-se en un camp clau.

Tècniques Avançades de Processament de Dades

  • Ús d'APIs per al Processament de Dades: Aprèn a utilitzar APIs per a tasques de processament de dades més complexes.
  • Optimització del Rendiment: Entén tècniques per optimitzar el rendiment del processament de dades.

Exemple: Ús d'una API per al Processament de Dades

D APIResult      S             10I 0
D InputData      DS
D  Field1               10A
D  Field2               20A

C                   EVAL      Field1 = 'Input1'
C                   EVAL      Field2 = 'Input2'
C                   CALLP     MyAPI(InputData: APIResult)
C                   IF       APIResult = 0
C                   // Processa la crida API amb èxit
C                   ELSE
C                   // Gestiona l'error
C                   ENDIF
  • Explicació: Aquest codi demostra com cridar una API per al processament de dades i gestionar el resultat.

Conclusió

La manipulació i el processament de dades en RPG són habilitats essencials per a qualsevol programador que treballi amb sistemes IBM i. En dominar la lectura, escriptura, actualització i unió de dades, així com l'ús de tècniques avançades com APIs i optimització del rendiment, pots gestionar eficientment tasques complexes de processament de dades. Practica aquests conceptes amb exemples del món real per consolidar la teva comprensió i millorar les teves capacitats de programació.
