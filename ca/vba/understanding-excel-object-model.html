Introducció

El Model d'Objectes d'Excel és un concepte fonamental en la programació VBA. Et permet interactuar amb els components d'Excel, com ara llibres de treball, fulls de càlcul, cel·les i rangs, a través de codi VBA. Comprendre aquest model és crucial per automatitzar tasques i crear aplicacions d'Excel potents.

Conceptes Clau

  • Objectes: Els blocs bàsics en el Model d'Objectes d'Excel. Exemples inclouen Application, Workbook, Worksheet, Range, etc.
  • Propietats: Atributs dels objectes que defineixen les seves característiques. Per exemple, la propietat Name d'un Workbook.
  • Mètodes: Accions que es poden realitzar sobre els objectes. Per exemple, el mètode Save d'un Workbook.
  • Col·leccions: Grups d'objectes similars. Per exemple, la col·lecció Workbooks conté tots els objectes Workbook oberts.

La Jerarquia d'Objectes

El Model d'Objectes d'Excel és jeràrquic. A la part superior hi ha l'objecte Application, que representa l'aplicació Excel en si mateixa. A sota hi ha col·leccions i objectes individuals.

Exemple de Jerarquia

  1. Application
    • Workbooks
      • Workbook
        • Worksheets
          • Worksheet
            • Range

Accés als Objectes

Per accedir als objectes en VBA, normalment comences des de l'objecte Application i baixes a través de la jerarquia.

Exemple: Accedir a un Full de Càlcul

Dim ws As Worksheet
Set ws = Application.Workbooks("MyWorkbook.xlsx").Worksheets("Sheet1")

En aquest exemple:

  • Application és l'aplicació Excel.
  • Workbooks("MyWorkbook.xlsx") accedeix a un llibre de treball específic.
  • Worksheets("Sheet1") accedeix a un full de càlcul específic dins d'aquest llibre de treball.

Propietats i Mètodes

Propietats

Les propietats s'utilitzen per obtenir o establir informació sobre un objecte.

Exemple: Obtenir i Establir Propietats

Dim ws As Worksheet
Set ws = Application.Workbooks("MyWorkbook.xlsx").Worksheets("Sheet1")

' Obtenir el nom del full de càlcul
Dim sheetName As String
sheetName = ws.Name

' Establir el nom del full de càlcul
ws.Name = "NewSheetName"

Mètodes

Els mètodes realitzen accions sobre els objectes.

Exemple: Utilitzar Mètodes

Dim wb As Workbook
Set wb = Application.Workbooks("MyWorkbook.xlsx")

' Guardar el llibre de treball
wb.Save

' Tancar el llibre de treball
wb.Close

Col·leccions

Les col·leccions s'utilitzen per gestionar grups d'objectes similars. Pots recórrer les col·leccions per realitzar accions sobre cada objecte dins de la col·lecció.

Exemple: Recórrer els Fulls de Càlcul

Dim ws As Worksheet
For Each ws In Application.Workbooks("MyWorkbook.xlsx").Worksheets
    ' Realitzar una acció sobre cada full de càlcul
    ws.Cells(1, 1).Value = "Hello, World!"
Next ws

Exercicis Pràctics

Exercici 1: Accedir i Modificar un Rang

  1. Obre un nou llibre de treball i crea un nou mòdul.
  2. Escriu un script VBA per accedir a la cel·la A1 del primer full de càlcul i establir el seu valor a "Hello, VBA!".
Sub SetCellValue()
    Dim ws As Worksheet
    Set ws = Application.Workbooks(1).Worksheets(1)
    ws.Range("A1").Value = "Hello, VBA!"
End Sub

Exercici 2: Recórrer Tots els Fulls de Càlcul

  1. Escriu un script VBA per recórrer tots els fulls de càlcul en el llibre de treball actiu i establir el valor de la cel·la A1 al nom del full de càlcul.
Sub SetSheetNames()
    Dim ws As Worksheet
    For Each ws In ThisWorkbook.Worksheets
        ws.Range("A1").Value = ws.Name
    Next ws
End Sub

Conclusió

Comprendre el Model d'Objectes d'Excel és essencial per a una programació VBA efectiva. En dominar els objectes, propietats, mètodes i col·leccions, pots automatitzar tasques complexes i crear aplicacions d'Excel robustes. Practica accedint i manipulant diferents objectes per convertir-te en un expert en VBA.
