Introducció

En aquesta secció, tractarem els fonaments de les variables i els tipus de dades a VBA (Visual Basic for Applications). Entendre aquests conceptes és crucial per escriure codi VBA eficient i lliure d'errors.

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden manipular durant l'execució d'un programa. A VBA, has de declarar una variable abans d'utilitzar-la.

Declaració de variables

Per declarar una variable a VBA, utilitza la instrucció Dim seguida del nom de la variable i el seu tipus de dades.

Dim nomVariable As TipusDeDades

Exemple

Dim edat As Integer
Dim nom As String
Dim salari As Double

Conceptes clau

  • Regles de nomenclatura de variables:

    • Ha de començar amb una lletra.
    • Pot contenir lletres, números i guions baixos.
    • No pot superar els 255 caràcters.
    • No pot ser una paraula reservada (per exemple, Dim, If, End).

  • Àmbit de les variables:

    • Nivell de procediment: Declarada dins d'un procediment i accessible només dins d'aquest procediment.
    • Nivell de mòdul: Declarada a la part superior d'un mòdul i accessible a tots els procediments dins d'aquest mòdul.
    • Nivell públic: Declarada utilitzant la paraula clau Public i accessible a tots els mòduls.

Exemple d'àmbit

Sub ExempleProcediment()
    Dim variableLocal As Integer ' Variable de nivell de procediment
    variableLocal = 10
End Sub

Dim variableModul As Integer ' Variable de nivell de mòdul

Public variablePublica As Integer ' Variable de nivell públic

Tipus de dades

Els tipus de dades defineixen el tipus de dades que una variable pot contenir. Triar el tipus de dades adequat és essencial per optimitzar l'ús de la memòria i garantir la integritat de les dades.

Tipus de dades comuns a VBA

| Tipus de dades | Descripció | Exemple | |----------------|-------------|---------| | Integer | Nombres enters de -32,768 a 32,767 | Dim edat As Integer | | Long | Nombres enters més grans de -2,147,483,648 a 2,147,483,647 | Dim distancia As Long | | Single | Nombres de punt flotant de precisió simple | Dim temperatura As Single | | Double | Nombres de punt flotant de precisió doble | Dim salari As Double | | String | Dades de text | Dim nom As String | | Boolean | Valors True o False | Dim estaActiu As Boolean | | Date | Valors de data i hora | Dim dataNaixement As Date | | Variant | Pot contenir qualsevol tipus de dades | Dim qualsevolDada As Variant |

Exemple

Dim edat As Integer
Dim nom As String
Dim salari As Double
Dim estaActiu As Boolean
Dim dataNaixement As Date
Dim qualsevolDada As Variant

edat = 30
nom = "John Doe"
salari = 55000.75
estaActiu = True
dataNaixement = #1/1/1990#
qualsevolDada = "Pot ser qualsevol tipus"

Conceptes clau

  • Integer vs. Long: Utilitza Integer per a nombres enters més petits i Long per a nombres enters més grans.
  • Single vs. Double: Utilitza Single per a menys precisió i Double per a més precisió en nombres de punt flotant.
  • String: Utilitza per a dades de text. Envolta el text amb cometes dobles.
  • Boolean: Utilitza per a valors true/false.
  • Date: Utilitza per a valors de data i hora. Envolta els literals de data amb #.
  • Variant: Utilitza quan el tipus de dades no es coneix per avançat. No obstant això, és menys eficient i s'hauria d'utilitzar amb moderació.

Conclusió

Entendre les variables i els tipus de dades és fonamental per programar a VBA. Declarar i utilitzar correctament les variables assegura que el teu codi sigui eficient, llegible i menys propens a errors. Tria sempre el tipus de dades més adequat per a les teves variables per optimitzar el rendiment i mantenir la integritat de les dades.
