Introducció

En aquest mòdul, explorarem com treballar amb dades externes en VBA (Visual Basic for Applications). Això inclou llegir i escriure fitxers externs com fitxers de text, fitxers CSV i bases de dades. Al final d'aquest mòdul, seràs capaç d'automatitzar tasques d'importació/exportació de dades i interactuar amb diverses fonts de dades de manera eficient.

Llegir de Fitxers Externs

Llegir de Fitxers de Text

Llegir de fitxers de text és una tasca comuna en VBA. El següent exemple demostra com llegir dades d'un fitxer de text línia per línia.

Sub ReadFromTextFile()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim line As String
    
    filePath = "C:\path\to\your\file.txt"
    fileNumber = FreeFile
    
    Open filePath For Input As #fileNumber
    
    Do While Not EOF(fileNumber)
        Line Input #fileNumber, line
        Debug.Print line
    Loop
    
    Close #fileNumber
End Sub
  • filePath: El camí al fitxer de text.
  • fileNumber: Un número únic assignat al fitxer.
  • Line Input: Llegeix una línia del fitxer.

Llegir de Fitxers CSV

Els fitxers CSV s'utilitzen comunament per a l'intercanvi de dades. El següent exemple mostra com llegir dades d'un fitxer CSV i omplir-les en un full de càlcul d'Excel.

Sub ReadFromCSV()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim line As String
    Dim data As Variant
    Dim row As Integer
    
    filePath = "C:\path\to\your\file.csv"
    fileNumber = FreeFile
    row = 1
    
    Open filePath For Input As #fileNumber
    
    Do While Not EOF(fileNumber)
        Line Input #fileNumber, line
        data = Split(line, ",")
        For col = 0 To UBound(data)
            Cells(row, col + 1).Value = data(col)
        Next col
        row = row + 1
    Loop
    
    Close #fileNumber
End Sub
  • Split: Divideix la línia en una matriu basada en el delimitador de coma.

Escriure a Fitxers Externs

Escriure a Fitxers de Text

Escriure dades a un fitxer de text pot ser útil per a registrar o exportar dades. Aquí hi ha un exemple de com escriure dades a un fitxer de text.

Sub WriteToTextFile()
    Dim filePath As String
    Dim fileNumber As Integer
    
    filePath = "C:\path\to\your\output.txt"
    fileNumber = FreeFile
    
    Open filePath For Output As #fileNumber
    
    Print #fileNumber, "This is a sample text."
    Print #fileNumber, "Writing to a text file using VBA."
    
    Close #fileNumber
End Sub
  • Print: Escriu una línia al fitxer.

Escriure a Fitxers CSV

Exportar dades a un fitxer CSV és senzill. El següent exemple demostra com escriure dades d'un full de càlcul d'Excel a un fitxer CSV.

Sub WriteToCSV()
    Dim filePath As String
    Dim fileNumber As Integer
    Dim row As Integer
    Dim col As Integer
    Dim line As String
    
    filePath = "C:\path\to\your\output.csv"
    fileNumber = FreeFile
    
    Open filePath For Output As #fileNumber
    
    row = 1
    Do While Cells(row, 1).Value <> ""
        line = ""
        col = 1
        Do While Cells(row, col).Value <> ""
            If col > 1 Then line = line & ","
            line = line & Cells(row, col).Value
            col = col + 1
        Loop
        Print #fileNumber, line
        row = row + 1
    Loop
    
    Close #fileNumber
End Sub

Interactuar amb Bases de Dades

Connectar-se a una Base de Dades

Connectar-se a una base de dades et permet llegir i escriure dades directament. El següent exemple demostra com connectar-se a una base de dades Access utilitzant ADO (ActiveX Data Objects).

Sub ConnectToDatabase()
    Dim conn As Object
    Dim rs As Object
    Dim connStr As String
    Dim sql As String
    
    connStr = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\path\to\your\database.accdb;"
    sql = "SELECT * FROM TableName"
    
    Set conn = CreateObject("ADODB.Connection")
    conn.Open connStr
    
    Set rs = CreateObject("ADODB.Recordset")
    rs.Open sql, conn
    
    Do While Not rs.EOF
        Debug.Print rs.Fields("FieldName").Value
        rs.MoveNext
    Loop
    
    rs.Close
    conn.Close
End Sub
  • connStr: Cadena de connexió a la base de dades.
  • sql: Consulta SQL per executar.
  • ADODB.Connection: Objecte per gestionar la connexió.
  • ADODB.Recordset: Objecte per mantenir els resultats de la consulta.

Executar Comandes SQL

Executar comandes SQL et permet manipular dades a la base de dades. El següent exemple mostra com inserir un nou registre en una taula.

Sub InsertIntoDatabase()
    Dim conn As Object
    Dim connStr As String
    Dim sql As String
    
    connStr = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\path\to\your\database.accdb;"
    sql = "INSERT INTO TableName (FieldName) VALUES ('SampleData')"
    
    Set conn = CreateObject("ADODB.Connection")
    conn.Open connStr
    
    conn.Execute sql
    
    conn.Close
End Sub
  • INSERT INTO: Comanda SQL per inserir dades.

Conclusió

En aquest mòdul, hem cobert diversos mètodes per treballar amb dades externes en VBA, incloent llegir i escriure fitxers de text i CSV, així com interactuar amb bases de dades. Aquestes habilitats són essencials per automatitzar tasques de processament de dades i integrar Excel amb altres fonts de dades. Practica aquests exemples i intenta implementar-los en els teus projectes per obtenir una comprensió més profunda.
