Formatació de Cel·les en VBA
Procediments Sub ►

Introducció

Formatar cel·les a Excel utilitzant VBA (Visual Basic for Applications) et permet automatitzar el procés de canviar l'aparença de les teves dades. Això pot incloure alterar estils de font, colors, vores i formats de números. En aquesta secció, cobrirem els conceptes bàsics de la formatació de cel·les i gradualment passarem a tècniques més avançades.

Formatació Bàsica de Cel·les

Canviar l'Estil i la Mida de la Font

Per canviar l'estil i la mida de la font d'una cel·la, pots utilitzar la propietat Font.

Sub ChangeFontStyle()
    Range("A1").Font.Name = "Arial"
    Range("A1").Font.Size = 14
End Sub
  • Range("A1").Font.Name estableix la font a Arial.
  • Range("A1").Font.Size estableix la mida de la font a 14.

Aplicar Negreta i Cursiva

Pots fer el text en negreta o cursiva utilitzant les propietats Bold i Italic.

Sub ApplyBoldItalic()
    Range("A1").Font.Bold = True
    Range("A1").Font.Italic = True
End Sub
  • Range("A1").Font.Bold estableix el text en negreta.
  • Range("A1").Font.Italic estableix el text en cursiva.

Canviar el Color de la Font

Per canviar el color de la font, utilitza la propietat Color.

Sub ChangeFontColor()
    Range("A1").Font.Color = RGB(255, 0, 0) ' Color vermell
End Sub
  • RGB(255, 0, 0) especifica el color vermell.

Formatació Intermèdia de Cel·les

Establir el Color de Fons de la Cel·la

Pots canviar el color de fons d'una cel·la utilitzant la propietat Interior.Color.

Sub SetBackgroundColor()
    Range("A1").Interior.Color = RGB(0, 255, 0) ' Color verd
End Sub
  • Range("A1").Interior.Color estableix el color de fons a verd.

Afegir Vores

Per afegir vores a una cel·la, utilitza la propietat Borders.

Sub AddBorders()
    With Range("A1").Borders
        .LineStyle = xlContinuous
        .Weight = xlThin
        .Color = RGB(0, 0, 0) ' Color negre
    End With
End Sub
  • .LineStyle estableix l'estil de la vora.
  • .Weight estableix el gruix de la vora.
  • .Color estableix el color de la vora.

Formatació Avançada de Cel·les

Formatació de Números

Per formatar números, utilitza la propietat NumberFormat.

Sub FormatNumbers()
    Range("A1").NumberFormat = "0.00" ' Dos decimals
    Range("B1").NumberFormat = "$#,##0.00" ' Format de moneda
End Sub
  • "0.00" formatar el número a dos decimals.
  • "$#,##0.00" formatar el número com a moneda.

Formatació Condicional

La formatació condicional es pot aplicar utilitzant VBA per canviar l'aparença de les cel·les basant-se en certes condicions.

Sub ApplyConditionalFormatting()
    Dim rng As Range
    Set rng = Range("A1:A10")
    
    With rng.FormatConditions.Add(Type:=xlCellValue, Operator:=xlGreater, Formula1:="=5")
        .Interior.Color = RGB(255, 255, 0) ' Color groc
    End With
End Sub
  • Type:=xlCellValue especifica el tipus de condició.
  • Operator:=xlGreater especifica la condició (major que).
  • Formula1:="=5" especifica el valor per comparar.

Conclusió

Formatar cel·les en VBA és una manera poderosa de millorar la presentació de les teves dades a Excel. Des de canvis bàsics de font fins a formatació condicional avançada, VBA proporciona una àmplia gamma d'opcions per personalitzar l'aparença de les teves cel·les. En dominar aquestes tècniques, pots crear fulls de càlcul més llegibles i visualment atractius, fent que les teves dades siguin més fàcils d'entendre i analitzar.
