Introducció als Arrays

Els arrays són un concepte fonamental en la programació, ja que permeten emmagatzemar múltiples valors en una sola variable. En VBA, els arrays poden ser particularment útils per gestionar grans conjunts de dades de manera eficient.

Conceptes Clau

  • Definició: Un array és una col·lecció de variables que s'accedeixen amb un número d'índex.
  • Tipus d'Arrays:
    • Arrays Unidimensionals: Una llista d'elements.
    • Arrays Multidimensionals: Arrays amb més d'una dimensió, com una taula (array 2D).

Declarant Arrays

Per utilitzar un array en VBA, primer cal declarar-lo. Això implica especificar el tipus de dades que contindrà i la seva mida.

Arrays Unidimensionals

Dim arr(5) As Integer
  • Això declara un array anomenat arr que pot contenir 6 enters (índex 0 a 5).

Arrays Multidimensionals

Dim arr(2, 3) As Integer
  • Això declara un array de 2x4 (2 files i 4 columnes).

Arrays Dinàmics

Els arrays dinàmics es poden redimensionar durant el temps d'execució utilitzant la instrucció ReDim.

Dim arr() As Integer
ReDim arr(5)
  • Això declara un array dinàmic i després el redimensiona per contenir 6 enters.

Inicialitzant Arrays

Els arrays es poden inicialitzar en el moment de la declaració o més tard en el codi.

Arrays Estàtics

Dim arr(2) As String
arr(0) = "Poma"
arr(1) = "Plàtan"
arr(2) = "Cirera"

Arrays Dinàmics

Dim arr() As String
ReDim arr(2)
arr(0) = "Poma"
arr(1) = "Plàtan"
arr(2) = "Cirera"

Accedint als Elements de l'Array

Es pot accedir i manipular els elements d'un array utilitzant el seu índex.

Exemple

Dim fruits(2) As String
fruits(0) = "Poma"
fruits(1) = "Plàtan"
fruits(2) = "Cirera"

MsgBox fruits(1)  ' Mostra "Plàtan"

Iterant a Través dels Arrays

Recórrer els arrays és una operació comuna. Es poden utilitzar bucles For per iterar a través dels elements de l'array.

Exemple

Dim fruits(2) As String
fruits(0) = "Poma"
fruits(1) = "Plàtan"
fruits(2) = "Cirera"

Dim i As Integer
For i = 0 To UBound(fruits)
    MsgBox fruits(i)
Next i
  • La funció UBound retorna el límit superior de l'array.

Arrays Multidimensionals

Els arrays multidimensionals són útils per representar estructures de dades més complexes com taules.

Exemple

Dim matrix(2, 2) As Integer
matrix(0, 0) = 1
matrix(0, 1) = 2
matrix(0, 2) = 3
matrix(1, 0) = 4
matrix(1, 1) = 5
matrix(1, 2) = 6

Dim i As Integer, j As Integer
For i = 0 To 1
    For j = 0 To 2
        MsgBox matrix(i, j)
    Next j
Next i

Tècniques Avançades amb Arrays

Redimensionant Arrays

Es poden redimensionar arrays dinàmics utilitzant ReDim Preserve per mantenir les dades existents.

Dim arr() As Integer
ReDim arr(2)
arr(0) = 1
arr(1) = 2
arr(2) = 3

ReDim Preserve arr(4)
arr(3) = 4
arr(4) = 5

Ordenant Arrays

Ordenar arrays es pot fer utilitzant algorismes d'ordenació personalitzats o funcions integrades.

Sub SortArray()
    Dim arr(4) As Integer
    arr(0) = 5
    arr(1) = 2
    arr(2) = 9
    arr(3) = 1
    arr(4) = 6
    
    Dim i As Integer, j As Integer, temp As Integer
    For i = LBound(arr) To UBound(arr) - 1
        For j = i + 1 To UBound(arr)
            If arr(i) > arr(j) Then
                temp = arr(i)
                arr(i) = arr(j)
                arr(j) = temp
            End If
        Next j
    Next i
    
    For i = LBound(arr) To UBound(arr)
        MsgBox arr(i)
    Next i
End Sub

Resum

Els arrays en VBA són eines poderoses per gestionar col·leccions de dades. Entenent com declarar, inicialitzar, accedir i manipular arrays, es pot escriure codi VBA més eficient i efectiu. Tant si es treballa amb arrays unidimensionals com multidimensionals, dominar aquests conceptes millorarà significativament les teves habilitats de programació en VBA.
