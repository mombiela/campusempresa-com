Introducció

Els operadors són components fonamentals en qualsevol llenguatge de programació, incloent VBA (Visual Basic for Applications). Permeten realitzar diverses operacions sobre variables i valors. En aquesta secció, cobrirem diferents tipus d'operadors en VBA, incloent operadors aritmètics, de comparació, lògics i de concatenació.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • Addició (+): Afegeix dos números.
  • Subtracció (-): Resta un número d'un altre.
  • Multiplicació (*): Multiplica dos números.
  • Divisió (/): Divideix un número per un altre.
  • Divisió Entera (\): Divideix dos números i retorna la part entera de la divisió.
  • Mòdul (Mod): Retorna el residu d'una divisió.
  • Exponenciació (^): Eleva un número a la potència d'un altre número.

Exemple

Dim a As Integer
Dim b As Integer
Dim result As Integer

a = 10
b = 3

result = a + b ' Addició: result és 13
result = a - b ' Subtracció: result és 7
result = a * b ' Multiplicació: result és 30
result = a / b ' Divisió: result és 3.3333
result = a \ b ' Divisió Entera: result és 3
result = a Mod b ' Mòdul: result és 1
result = a ^ b ' Exponenciació: result és 1000

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors. Retornen un valor Boolean (True o False).

  • Igual a (=): Comprova si dos valors són iguals.
  • No igual a (<>): Comprova si dos valors no són iguals.
  • Major que (>): Comprova si el valor de l'esquerra és major que el valor de la dreta.
  • Menor que (<): Comprova si el valor de l'esquerra és menor que el valor de la dreta.
  • Major o igual a (>=): Comprova si el valor de l'esquerra és major o igual al valor de la dreta.
  • Menor o igual a (<=): Comprova si el valor de l'esquerra és menor o igual al valor de la dreta.

Exemple

Dim x As Integer
Dim y As Integer
Dim comparisonResult As Boolean

x = 5
y = 10

comparisonResult = (x = y) ' Igual a: comparisonResult és False
comparisonResult = (x <> y) ' No igual a: comparisonResult és True
comparisonResult = (x > y) ' Major que: comparisonResult és False
comparisonResult = (x < y) ' Menor que: comparisonResult és True
comparisonResult = (x >= y) ' Major o igual a: comparisonResult és False
comparisonResult = (x <= y) ' Menor o igual a: comparisonResult és True

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions Booleanes.

  • And: Retorna True si ambdues expressions són True.
  • Or: Retorna True si almenys una de les expressions és True.
  • Not: Inverteix el valor Boolean d'una expressió.

Exemple

Dim a As Boolean
Dim b As Boolean
Dim logicalResult As Boolean

a = True
b = False

logicalResult = a And b ' And: logicalResult és False
logicalResult = a Or b ' Or: logicalResult és True
logicalResult = Not a ' Not: logicalResult és False

Operador de Concatenació

L'operador de concatenació s'utilitza per unir dues o més cadenes de text.

  • Concatenació (&): Uneix dues cadenes de text.

Exemple

Dim firstName As String
Dim lastName As String
Dim fullName As String

firstName = "John"
lastName = "Doe"

fullName = firstName & " " & lastName ' Concatenació: fullName és "John Doe"

Resum

En aquesta secció, hem cobert els diversos tipus d'operadors disponibles en VBA:

  • Operadors Aritmètics: Per a càlculs matemàtics.
  • Operadors de Comparació: Per comparar valors.
  • Operadors Lògics: Per combinar expressions Booleanes.
  • Operador de Concatenació: Per unir cadenes de text.

Entendre aquests operadors és crucial per realitzar una àmplia gamma de tasques en VBA, des de càlculs simples fins a operacions lògiques complexes. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en programació VBA.
