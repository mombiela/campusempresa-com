Creació de Funcions Personalitzades
Interacció amb altres aplicacions d'Office ►

Crear funcions personalitzades en VBA et permet estendre les capacitats d'Excel i altres aplicacions d'Office escrivint el teu propi codi reutilitzable. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en la creació de funcions personalitzades.

Introducció a les Funcions Personalitzades

Les funcions personalitzades, també conegudes com a Funcions Definides per l'Usuari (UDFs), són funcions que crees per realitzar càlculs o tasques específiques que no estan disponibles en les funcions integrades d'Excel.

Conceptes Clau

  • Definició de Funció: L'estructura i la sintaxi d'una funció.
  • Arguments de Funció: Paràmetres que passes a la funció.
  • Valor de Retorn: El resultat que retorna la funció.

Escriure la Teva Primera Funció Personalitzada

Comencem escrivint una funció personalitzada simple en VBA.

Exemple: Una Funció Simple d'Addició

Function AddNumbers(a As Double, b As Double) As Double
    AddNumbers = a + b
End Function

Explicació

  • Funció AddNumbers: Això defineix una nova funció anomenada AddNumbers.
  • Paràmetres (a As Double, b As Double): La funció pren dos arguments, a i b, tots dos de tipus Double.
  • Valor de Retorn (As Double): La funció retorna un valor de tipus Double.
  • Cos de la Funció (AddNumbers = a + b): La funció suma els dos arguments i retorna el resultat.

Utilitzar la Funció Personalitzada a Excel

  1. Obre Excel i prem Alt + F11 per obrir l'editor de VBA.
  2. Insereix un nou mòdul fent clic a Insertar > Mòdul.
  3. Copia i enganxa la funció AddNumbers al mòdul.
  4. Torna a Excel i utilitza la funció com qualsevol altra funció d'Excel: 
    =AddNumbers(5, 10)

Afegir Gestió d'Errors

És important gestionar els errors a les teves funcions per fer-les robustes.

Exemple: Divisió amb Gestió d'Errors

Function SafeDivide(a As Double, b As Double) As Variant
    If b = 0 Then
        SafeDivide = "Error: Division by zero"
    Else
        SafeDivide = a / b
    End If
End Function

Explicació

  • Tipus de Retorn (As Variant): Utilitzar Variant permet que la funció retorni un número o un missatge d'error.
  • Comprovació d'Error (If b = 0): Comprova si el denominador és zero.
  • Missatge d'Error (SafeDivide = "Error: Division by zero"): Retorna un missatge d'error si el denominador és zero.
  • Divisió (SafeDivide = a / b): Realitza la divisió si el denominador no és zero.

Funcions Personalitzades Avançades

Les funcions personalitzades avançades poden incloure lògica més complexa, múltiples passos i interacció amb objectes d'Excel.

Exemple: Calculant la Mitjana d'un Rang

Function AverageRange(rng As Range) As Double
    Dim cell As Range
    Dim total As Double
    Dim count As Integer
    
    total = 0
    count = 0
    
    For Each cell In rng
        If IsNumeric(cell.Value) Then
            total = total + cell.Value
            count = count + 1
        End If
    Next cell
    
    If count > 0 Then
        AverageRange = total / count
    Else
        AverageRange = 0
    End If
End Function

Explicació

  • Paràmetre (rng As Range): La funció pren un rang de cel·les com a argument.
  • Variables (Dim total As Double, count As Integer): Inicialitza variables per emmagatzemar la suma total i el recompte de cel·les numèriques.
  • Bucle (For Each cell In rng): Itera a través de cada cel·la en el rang.
  • Comprovació (If IsNumeric(cell.Value)): Comprova si la cel·la conté un valor numèric.
  • Suma i Recompte (total = total + cell.Value, count = count + 1): Afegeix el valor de la cel·la al total i incrementa el recompte.
  • Càlcul de la Mitjana (AverageRange = total / count): Calcula la mitjana si hi ha cel·les numèriques, en cas contrari retorna 0.

Conclusió

Crear funcions personalitzades en VBA et permet adaptar Excel a les teves necessitats específiques, fent la teva feina més eficient i efectiva. Entenent els conceptes bàsics de la definició de funcions, gestionant errors i implementant lògica avançada, pots crear funcions poderoses i reutilitzables.

Resum

  • Les funcions personalitzades estenen les capacitats d'Excel.
  • Comença amb funcions simples i afegeix complexitat gradualment.
  • Gestiona errors per fer les teves funcions robustes.
  • Utilitza bucles i condicions per crear funcions avançades.

Dominant aquestes tècniques, podràs crear funcions personalitzades que estalvien temps i milloren la teva productivitat a Excel.
