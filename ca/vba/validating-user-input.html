Validant l'Entrada d'Usuari
Tècniques avançades de formularis d'usuari ►

Introducció

Validar l'entrada d'usuari és un aspecte crucial de la programació, assegurant que les dades introduïdes pels usuaris siguin correctes, completes i dins del format esperat. En VBA (Visual Basic for Applications), la validació de l'entrada ajuda a prevenir errors i millorar la robustesa de les teves aplicacions. Aquest tema et guiarà a través dels conceptes bàsics de la validació de l'entrada d'usuari en VBA, progressant cap a tècniques més avançades.

Per què validar l'entrada d'usuari?

  • Prevenir errors: Evitar errors en temps d'execució causats per dades no vàlides.
  • Millorar la qualitat de les dades: Assegurar la integritat i la consistència de les dades.
  • Millorar l'experiència de l'usuari: Proporcionar retroalimentació immediata als usuaris, guiant-los per introduir informació correcta.

Validació bàsica de l'entrada

Ús de la funció `IsNumeric`

La funció IsNumeric comprova si un valor pot ser avaluat com un número.

Dim userInput As String
userInput = InputBox("Introdueix un número:")

If IsNumeric(userInput) Then
    MsgBox "Número vàlid: " & userInput
Else
    MsgBox "Entrada no vàlida. Si us plau, introdueix un número."
End If

Comprovació d'entrada buida

Utilitza la funció Len per comprovar si l'entrada està buida.

Dim userInput As String
userInput = InputBox("Introdueix el teu nom:")

If Len(userInput) = 0 Then
    MsgBox "L'entrada no pot estar buida. Si us plau, introdueix el teu nom."
Else
    MsgBox "Hola, " & userInput
End If

Validació intermèdia de l'entrada

Ús de l'operador `Like` per a la coincidència de patrons

L'operador Like et permet comprovar si una cadena coincideix amb un patró específic.

Dim userInput As String
userInput = InputBox("Introdueix la teva adreça de correu electrònic:")

If userInput Like "*@*.*" Then
    MsgBox "Adreça de correu electrònic vàlida: " & userInput
Else
    MsgBox "Adreça de correu electrònic no vàlida. Si us plau, introdueix una adreça de correu electrònic vàlida."
End If

Combinació de múltiples comprovacions

Pots combinar múltiples comprovacions de validació per assegurar una validació completa de l'entrada.

Dim userInput As String
userInput = InputBox("Introdueix la teva edat:")

If Len(userInput) = 0 Then
    MsgBox "L'entrada no pot estar buida. Si us plau, introdueix la teva edat."
ElseIf Not IsNumeric(userInput) Then
    MsgBox "Entrada no vàlida. Si us plau, introdueix un valor numèric."
ElseIf CInt(userInput) < 0 Or CInt(userInput) > 120 Then
    MsgBox "Edat no vàlida. Si us plau, introdueix un valor entre 0 i 120."
Else
    MsgBox "Edat vàlida: " & userInput
End If

Validació avançada de l'entrada

Ús de funcions personalitzades per a la validació

Crea funcions personalitzades per encapsular la lògica de validació, fent el teu codi més modular i reutilitzable.

Function IsValidEmail(email As String) As Boolean
    IsValidEmail = email Like "*@*.*"
End Function

Sub ValidateEmail()
    Dim userInput As String
    userInput = InputBox("Introdueix la teva adreça de correu electrònic:")

    If IsValidEmail(userInput) Then
        MsgBox "Adreça de correu electrònic vàlida: " & userInput
    Else
        MsgBox "Adreça de correu electrònic no vàlida. Si us plau, introdueix una adreça de correu electrònic vàlida."
    End If
End Sub

Ús d'expressions regulars

Per a patrons més complexos, utilitza expressions regulars amb la biblioteca VbScript_RegExp_55.

Sub ValidatePhoneNumber()
    Dim userInput As String
    Dim regex As Object
    Set regex = CreateObject("VBScript.RegExp")
    
    regex.Pattern = "^\d{3}-\d{3}-\d{4}$"
    regex.IgnoreCase = True
    
    userInput = InputBox("Introdueix el teu número de telèfon (XXX-XXX-XXXX):")
    
    If regex.Test(userInput) Then
        MsgBox "Número de telèfon vàlid: " & userInput
    Else
        MsgBox "Número de telèfon no vàlid. Si us plau, introdueix-lo en el format XXX-XXX-XXXX."
    End If
End Sub

Conclusió

Validar l'entrada d'usuari és essencial per crear aplicacions VBA fiables i fàcils d'usar. Començant amb comprovacions bàsiques com IsNumeric i Len, pots progressar cap a tècniques més sofisticades com la coincidència de patrons amb l'operador Like, funcions de validació personalitzades i expressions regulars. Implementant una validació robusta de l'entrada, assegures que les teves aplicacions gestionin les dades dels usuaris de manera correcta i eficient.
