Estructures de control: If...Then...Else
Bucles: For, For Each, Do While, Do Until ►

Introducció

Les estructures de control són fonamentals en la programació, ja que et permeten dictar el flux del teu programa basant-te en certes condicions. En VBA (Visual Basic for Applications), la instrucció If...Then...Else és una eina poderosa per a la presa de decisions. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat de If...Then...Else en VBA.

Instrucció bàsica If...Then

La forma més simple d'estructura de control és la instrucció If...Then. Et permet executar un bloc de codi només si una condició especificada és certa.

Sintaxi

If condition Then
    ' Codi a executar si la condició és certa
End If

Exemple

Sub CheckValue()
    Dim value As Integer
    value = 10
    
    If value > 5 Then
        MsgBox "El valor és més gran que 5"
    End If
End Sub

En aquest exemple, el quadre de missatge mostrarà "El valor és més gran que 5" perquè la condició value > 5 és certa.

Instrucció If...Then...Else

Quan necessites executar un bloc de codi si una condició és certa i un altre bloc si és falsa, utilitzes la instrucció If...Then...Else.

Sintaxi

If condition Then
    ' Codi a executar si la condició és certa
Else
    ' Codi a executar si la condició és falsa
End If

Exemple

Sub CheckValue()
    Dim value As Integer
    value = 3
    
    If value > 5 Then
        MsgBox "El valor és més gran que 5"
    Else
        MsgBox "El valor és 5 o menys"
    End If
End Sub

En aquest exemple, el quadre de missatge mostrarà "El valor és 5 o menys" perquè la condició value > 5 és falsa.

Instrucció If...Then...ElseIf...Else

Per a múltiples condicions, pots utilitzar la paraula clau ElseIf per comprovar condicions addicionals.

Sintaxi

If condition1 Then
    ' Codi a executar si la condició1 és certa
ElseIf condition2 Then
    ' Codi a executar si la condició2 és certa
Else
    ' Codi a executar si cap de les condicions anteriors és certa
End If

Exemple

Sub CheckValue()
    Dim value As Integer
    value = 7
    
    If value > 10 Then
        MsgBox "El valor és més gran que 10"
    ElseIf value > 5 Then
        MsgBox "El valor és més gran que 5 però 10 o menys"
    Else
        MsgBox "El valor és 5 o menys"
    End If
End Sub

En aquest exemple, el quadre de missatge mostrarà "El valor és més gran que 5 però 10 o menys" perquè la condició value > 5 és certa.

Instruccions If imbricades

Pots imbricar instruccions If...Then...Else dins d'altres per gestionar la presa de decisions més complexes.

Exemple

Sub CheckValue()
    Dim value As Integer
    value = 12
    
    If value > 10 Then
        If value > 15 Then
            MsgBox "El valor és més gran que 15"
        Else
            MsgBox "El valor és més gran que 10 però 15 o menys"
        End If
    Else
        MsgBox "El valor és 10 o menys"
    End If
End Sub

En aquest exemple, el quadre de missatge mostrarà "El valor és més gran que 10 però 15 o menys" perquè la condició externa value > 10 és certa, però la condició interna value > 15 és falsa.

Ús avançat

Ús d'operadors lògics

Pots combinar múltiples condicions utilitzant operadors lògics com And, Or i Not.

Exemple

Sub CheckValue()
    Dim value As Integer
    value = 8
    
    If value > 5 And value < 10 Then
        MsgBox "El valor està entre 5 i 10"
    Else
        MsgBox "El valor no està entre 5 i 10"
    End If
End Sub

En aquest exemple, el quadre de missatge mostrarà "El valor està entre 5 i 10" perquè ambdues condicions value > 5 i value < 10 són certes.

Conclusió

La instrucció If...Then...Else és una estructura de control versàtil en VBA que et permet prendre decisions basades en condicions. Comprenent i utilitzant les seves diverses formes, des de simples instruccions If...Then fins a comprovacions de condicions lògiques i imbricades complexes, pots controlar el flux dels teus programes VBA de manera efectiva. Practica amb diferents escenaris per convertir-te en un expert en l'ús d'aquestes estructures de control.
