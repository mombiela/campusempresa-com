Introducció

En aquesta secció, explorarem com treballar amb Llibres i Fulls de Càlcul utilitzant VBA (Visual Basic for Applications). Aquest tema és essencial per automatitzar tasques a Excel, com l'entrada de dades, l'anàlisi de dades i la generació d'informes.

Conceptes Clau

  • Llibre: Un fitxer que conté un o més fulls de càlcul.
  • Full de càlcul: Un full individual dins d'un llibre, format per cel·les disposades en files i columnes.
  • VBA: Un llenguatge de programació utilitzat per automatitzar tasques en aplicacions de Microsoft Office.

Accedir als Llibres

Per treballar amb llibres a VBA, cal entendre com obrir, crear i referenciar-los.

Obrir un Llibre Existent

Per obrir un llibre existent, utilitza el mètode Workbooks.Open.

Sub OpenWorkbook()
    Dim wb As Workbook
    Set wb = Workbooks.Open("C:\Path\To\Your\Workbook.xlsx")
End Sub

Crear un Nou Llibre

Per crear un nou llibre, utilitza el mètode Workbooks.Add.

Sub CreateNewWorkbook()
    Dim wb As Workbook
    Set wb = Workbooks.Add
End Sub

Referenciar Llibres

Pots referenciar llibres pel seu nom o índex.

Sub ReferenceWorkbook()
    Dim wb As Workbook
    ' Per nom
    Set wb = Workbooks("WorkbookName.xlsx")
    ' Per índex
    Set wb = Workbooks(1)
End Sub

Accedir als Fulls de Càlcul

Un cop tens un llibre, pots accedir als seus fulls de càlcul.

Referenciar Fulls de Càlcul

Pots referenciar fulls de càlcul pel seu nom o índex.

Sub ReferenceWorksheet()
    Dim ws As Worksheet
    ' Per nom
    Set ws = ThisWorkbook.Sheets("Sheet1")
    ' Per índex
    Set ws = ThisWorkbook.Sheets(1)
End Sub

Afegir un Nou Full de Càlcul

Per afegir un nou full de càlcul, utilitza el mètode Sheets.Add.

Sub AddWorksheet()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets.Add
    ws.Name = "NewSheet"
End Sub

Eliminar un Full de Càlcul

Per eliminar un full de càlcul, utilitza el mètode Sheets("SheetName").Delete.

Sub DeleteWorksheet()
    Application.DisplayAlerts = False ' Suprimir el diàleg de confirmació
    ThisWorkbook.Sheets("Sheet1").Delete
    Application.DisplayAlerts = True
End Sub

Treballar amb Cel·les i Rangs

Les cel·les i els rangs són les unitats fonamentals de dades en els fulls de càlcul.

Escriure a les Cel·les

Per escriure dades a una cel·la, utilitza la propietat Range o Cells.

Sub WriteToCell()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("Sheet1")
    ws.Range("A1").Value = "Hello, World!"
    ws.Cells(2, 1).Value = "Hello, VBA!"
End Sub

Llegir de les Cel·les

Per llegir dades d'una cel·la, utilitza la propietat Range o Cells.

Sub ReadFromCell()
    Dim ws As Worksheet
    Dim cellValue As String
    Set ws = ThisWorkbook.Sheets("Sheet1")
    cellValue = ws.Range("A1").Value
    MsgBox cellValue
End Sub

Treballar amb Rangs

Pots realitzar operacions en rangs, com copiar, enganxar i formatar.

Sub CopyRange()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("Sheet1")
    ws.Range("A1:B2").Copy Destination:=ws.Range("C1")
End Sub

Resum

En aquesta secció, hem cobert els conceptes bàsics de treballar amb llibres i fulls de càlcul a VBA. Has après com:

  • Obrir, crear i referenciar llibres.
  • Referenciar, afegir i eliminar fulls de càlcul.
  • Escriure i llegir de les cel·les.
  • Realitzar operacions en rangs.

Entendre aquests conceptes és crucial per automatitzar tasques a Excel utilitzant VBA. Practica aquests exemples i experimenta amb els teus propis per convertir-te en més proficient en la programació VBA.
