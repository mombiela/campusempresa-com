Introducció

Els sistemes d'equacions lineals són conjunts de dues o més equacions lineals amb les mateixes variables. Resoldre aquests sistemes és fonamental en diverses àrees de les matemàtiques i les seves aplicacions en l'enginyeria, la física i l'economia, entre altres. En aquest tema, aprendrem a resoldre sistemes d'equacions lineals en el context d'espais tridimensionals (3D).

Conceptes Clau

  • Equació Lineal: Una equació de la forma \( ax + by + cz = d \), on \(a\), \(b\), \(c\) i \(d\) són constants i \(x\), \(y\), \(z\) són variables.
  • Sistema d'Equacions Lineals: Un conjunt de dues o més equacions lineals amb les mateixes variables.
  • Solució d'un Sistema: Un conjunt de valors per a les variables que satisfà totes les equacions del sistema simultàniament.
  • Mètodes de Resolució:
    • Mètode de Substitució
    • Mètode d'Eliminació (o Reducció)
    • Mètode de Matrius (o Mètode de Gauss-Jordan)

Representació de Sistemes d'Equacions Lineals

Un sistema d'equacions lineals en 3D pot representar-se de la següent manera:

\[ \begin{cases} a_1x + b_1y + c_1z = d_1
a_2x + b_2y + c_2z = d_2
a_3x + b_3y + c_3z = d_3 \end{cases} \]

On \(a_1, a_2, a_3, b_1, b_2, b_3, c_1, c_2, c_3, d_1, d_2, d_3\) són constants.

Mètodes de Resolució

Mètode de Substitució

  1. Despejar una variable en una de les equacions.
  2. Substituir aquesta variable en les altres equacions.
  3. Resoldre el sistema reduït.
  4. Substituir les solucions obtingudes en l'equació desplaçada inicialment.

Mètode d'Eliminació

  1. Multiplicar les equacions per constants adequades perquè en sumar-les o restar-les, una de les variables s'elimini.
  2. Resoldre el sistema reduït.
  3. Substituir les solucions obtingudes en les equacions originals per trobar les altres variables.

Mètode de Matrius

  1. Escriure el sistema en forma matricial \(AX = B\), on \(A\) és la matriu de coeficients, \(X\) és el vector de variables i \(B\) és el vector de termes independents.
  2. Aplicar el mètode de Gauss-Jordan per reduir la matriu \(A\) a la seva forma esglaonada reduïda.
  3. Resoldre el sistema resultant.

Exemple de Codi en Python

import numpy as np

# Coeficients de les equacions
A = np.array([[2, -1, 3],
              [4, 1, -2],
              [1, 1, 1]])

# Termes independents
B = np.array([5, 6, 0])

# Resoldre el sistema d'equacions
solucio = np.linalg.solve(A, B)

print("La solució del sistema és:", solucio)

En aquest exemple, utilitzem la biblioteca numpy de Python per resoldre un sistema d'equacions lineals. La funció np.linalg.solve s'encarrega de trobar la solució del sistema \(AX = B\).

Exercicis

  1. Resol el següent sistema d'equacions lineals utilitzant el mètode de substitució: \[ \begin{cases} x + 2y + 3z = 9
    2x - y + z = 8
    3x + y - z = 3 \end{cases} \]

  2. Utilitza el mètode d'eliminació per resoldre el següent sistema: \[ \begin{cases} 2x + 3y - z = 7
    4x - y + 5z = 3
    -x + 2y + 4z = 1 \end{cases} \]

  3. Resol el següent sistema d'equacions utilitzant el mètode de matrius: \[ \begin{cases} x - y + 2z = 4
    3x + y - z = 5
    2x - 3y + 4z = 6 \end{cases} \]

Conclusió

Els sistemes d'equacions lineals són eines poderoses per resoldre problemes en múltiples dimensions. Comprendre i aplicar els diferents mètodes de resolució és essencial per abordar problemes complexos en matemàtiques i altres disciplines. Practicar amb diferents mètodes i exemples t'ajudarà a dominar aquestes tècniques i a aplicar-les eficaçment en situacions reals.
