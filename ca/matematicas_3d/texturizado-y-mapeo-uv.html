Introducció

El texturitzat i el mapeig UV són tècniques fonamentals en la creació de gràfics 3D. Aquestes tècniques permeten aplicar imatges 2D (textures) a superfícies 3D, proporcionant detalls visuals que milloren l'aparença dels models. En aquesta secció, explorarem els conceptes bàsics, les matemàtiques darrere del mapeig UV i com implementar aquestes tècniques en un entorn de programació.

Conceptes Clau

  • Textura: Una imatge 2D que s'aplica a la superfície d'un model 3D.
  • Mapeig UV: El procés de projectar una textura 2D sobre una superfície 3D utilitzant coordenades UV.
  • Coordenades UV: Un sistema de coordenades bidimensional que s'utilitza per mapar textures a superfícies 3D. La 'U' representa la coordenada horitzontal i la 'V' la vertical.
  • Desenvolupament UV (UV Unwrapping): El procés d'aplanar una malla 3D en un pla 2D per facilitar el mapeig de textures.

Mapeig UV

El mapeig UV és essencial per aplicar textures correctament a un model 3D. Les coordenades UV s'assignen a cada vèrtex del model, determinant com es projecta la textura sobre la superfície.

Exemple de Coordenades UV

Imaginem un quadrat simple amb quatre vèrtexs. Les coordenades UV per a cada vèrtex podrien ser les següents:

| Vèrtex | Coordenada UV (U, V) | |--------|----------------------| | V0 | (0, 0) | | V1 | (1, 0) | | V2 | (1, 1) | | V3 | (0, 1) |

Aquestes coordenades indiquen que la textura s'aplicarà de manera uniforme sobre el quadrat.

Texturitzat

El texturitzat implica aplicar una imatge 2D a la superfície d'un model 3D utilitzant les coordenades UV. Aquest procés pot realitzar-se mitjançant shaders en un entorn de programació gràfica.

Exemple de Codi: Shader per a Texturitzat

A continuació, es presenta un exemple d'un shader simple en GLSL (OpenGL Shading Language) que aplica una textura a un model 3D.

#version 330 core

in vec2 TexCoords; // Coordenades UV d'entrada
out vec4 FragColor; // Color de sortida

uniform sampler2D texture1; // Textura d'entrada

void main()
{
    FragColor = texture(texture1, TexCoords); // Aplicar la textura usant les coordenades UV
}

En aquest shader, TexCoords són les coordenades UV que es passen des del vèrtex, i texture1 és la textura que s'aplica. La funció texture pren la textura i les coordenades UV per determinar el color final del fragment.

Exercici Pràctic

Per consolidar els conceptes apresos, realitzi el següent exercici:

  1. Model 3D Simple: Creï un model 3D simple, com un cub, en el seu programari de modelatge 3D preferit.
  2. Assignació de Coordenades UV: Assigni coordenades UV a cada vèrtex del cub.
  3. Aplicació de Textura: Utilitzi un shader similar a l'exemple proporcionat per aplicar una textura al cub.
  4. Visualització: Renderitzi el cub texturitzat i observi com s'aplica la textura a cada cara del cub.

Conclusió

El texturitzat i el mapeig UV són tècniques essencials en la creació de gràfics 3D realistes. Comprendre com funcionen les coordenades UV i com aplicar textures mitjançant shaders és fonamental per a qualsevol professional que treballi en el camp dels gràfics 3D. Amb la pràctica i l'experimentació, aquestes tècniques poden dominar-se i utilitzar-se per crear models 3D visualment impressionants.
