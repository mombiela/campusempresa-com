Introducció a DirectX

DirectX és una col·lecció d'APIs (Interfícies de Programació d'Aplicacions) desenvolupades per Microsoft per manejar tasques relacionades amb multimèdia, especialment programació de jocs i vídeo, en plataformes Microsoft. En el context de les matemàtiques 3D, DirectX proporciona eines essencials per a la creació i manipulació de gràfics tridimensionals.

Conceptes Clau de DirectX

  • Direct3D: És l'API de DirectX utilitzada per renderitzar gràfics en 3D. És fonamental per a la creació d'aplicacions gràfiques i jocs.
  • Shaders: Programes que s'executen a la GPU per realitzar operacions de gràfics avançades.
  • Buffers: Àrees de memòria que emmagatzemen dades que la GPU necessita per renderitzar gràfics.
  • Matrius de Transformació: Utilitzades per realitzar operacions de transformació en objectes 3D, com traslació, rotació i escalat.

Instal·lació i Configuració de DirectX

Per començar a utilitzar DirectX, és necessari instal·lar l'SDK de DirectX i configurar l'entorn de desenvolupament. Aquí hi ha un exemple de com fer-ho a Visual Studio:

  1. Instal·lar l'SDK de DirectX: Descarregueu i instal·leu l'SDK de DirectX des del lloc web de Microsoft.
  2. Configurar Visual Studio:
    • Obriu Visual Studio i creeu un nou projecte.
    • Aneu a Project > Properties.
    • A la secció VC++ Directories, afegiu les rutes dels directoris d'inclusió i biblioteques de l'SDK de DirectX.
    • A la secció Linker > Input, afegiu les biblioteques de DirectX necessàries (per exemple, d3d11.lib).

Creació d'una Finestra Bàsica amb DirectX

A continuació, es presenta un exemple de codi per crear una finestra bàsica utilitzant DirectX:

#include <windows.h>
#include <d3d11.h>

// Variables globals
HINSTANCE hInstance;
HWND hwnd;
ID3D11Device* device = nullptr;
ID3D11DeviceContext* deviceContext = nullptr;
IDXGISwapChain* swapChain = nullptr;
ID3D11RenderTargetView* renderTargetView = nullptr;

// Prototips de funcions
LRESULT CALLBACK WndProc(HWND, UINT, WPARAM, LPARAM);
void InitD3D(HWND hwnd);
void RenderFrame();
void CleanD3D();

int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow) {
    // Registre de la classe de finestra
    WNDCLASSEX wc;
    ZeroMemory(&wc, sizeof(WNDCLASSEX));
    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WndProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = "WindowClass";
    RegisterClassEx(&wc);

    // Creació de la finestra
    hwnd = CreateWindowEx(NULL, "WindowClass", "DirectX Window", WS_OVERLAPPEDWINDOW, 100, 100, 800, 600, NULL, NULL, hInstance, NULL);
    ShowWindow(hwnd, nCmdShow);

    // Inicialització de Direct3D
    InitD3D(hwnd);

    // Bucle de missatges
    MSG msg;
    while (TRUE) {
        if (PeekMessage(&msg, NULL, 0, 0, PM_REMOVE)) {
            if (msg.message == WM_QUIT) break;
            TranslateMessage(&msg);
            DispatchMessage(&msg);
        } else {
            RenderFrame();
        }
    }

    // Neteja de Direct3D
    CleanD3D();

    return msg.wParam;
}

LRESULT CALLBACK WndProc(HWND hwnd, UINT msg, WPARAM wParam, LPARAM lParam) {
    switch (msg) {
        case WM_DESTROY:
            PostQuitMessage(0);
            return 0;
    }
    return DefWindowProc(hwnd, msg, wParam, lParam);
}

void InitD3D(HWND hwnd) {
    // Creació del dispositiu i el context de dispositiu
    D3D_FEATURE_LEVEL featureLevel;
    D3D11CreateDevice(NULL, D3D_DRIVER_TYPE_HARDWARE, NULL, 0, NULL, 0, D3D11_SDK_VERSION, &device, &featureLevel, &deviceContext);

    // Creació de la cadena d'intercanvi
    DXGI_SWAP_CHAIN_DESC scd;
    ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));
    scd.BufferCount = 1;
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;
    scd.OutputWindow = hwnd;
    scd.SampleDesc.Count = 4;
    scd.Windowed = TRUE;
    IDXGIFactory* factory;
    CreateDXGIFactory(__uuidof(IDXGIFactory), (void**)&factory);
    factory->CreateSwapChain(device, &scd, &swapChain);

    // Creació de la vista de destí de renderitzat
    ID3D11Texture2D* backBuffer;
    swapChain->GetBuffer(0, __uuidof(ID3D11Texture2D), (void**)&backBuffer);
    device->CreateRenderTargetView(backBuffer, NULL, &renderTargetView);
    backBuffer->Release();
    deviceContext->OMSetRenderTargets(1, &renderTargetView, NULL);
}

void RenderFrame() {
    // Neteja del buffer
    float color[4] = {0.0f, 0.2f, 0.4f, 1.0f};
    deviceContext->ClearRenderTargetView(renderTargetView, color);

    // Presentació del buffer
    swapChain->Present(0, 0);
}

void CleanD3D() {
    swapChain->Release();
    renderTargetView->Release();
    deviceContext->Release();
    device->Release();
}

Transformacions en 3D amb DirectX

Les transformacions en 3D són essencials per manipular objectes en l'espai tridimensional. DirectX utilitza matrius per realitzar aquestes transformacions:

  • Traslació: Mou un objecte d'un lloc a un altre.
  • Rotació: Gira un objecte al voltant d'un eix.
  • Escalat: Canvia la mida d'un objecte.

Exemple de codi per aplicar una transformació de traslació:

#include <DirectXMath.h>

using namespace DirectX;

XMMATRIX translationMatrix = XMMatrixTranslation(3.0f, 4.0f, 5.0f);

Conclusió

DirectX és una eina poderosa per a la creació de gràfics en 3D, proporcionant una àmplia gamma de funcionalitats per manejar gràfics i multimèdia. En aquest tema, hem cobert els conceptes bàsics de DirectX, com configurar l'entorn de desenvolupament, i com crear una finestra bàsica i aplicar transformacions en 3D. Amb aquesta base, els estudiants poden explorar més a fons les capacitats de DirectX i la seva aplicació en projectes més complexos.
