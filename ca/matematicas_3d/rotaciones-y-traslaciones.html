Introducció

En l'estudi de les matemàtiques 3D, les rotacions i translacions són transformacions fonamentals que permeten manipular objectes en l'espai tridimensional. Aquestes transformacions són essencials en camps com la computació gràfica, la robòtica i la simulació física.

Translacions

La translació és una transformació que desplaça cada punt d'un objecte una distància constant en una direcció específica.

  • Definició: Una translació en 3D es defineix per un vector de translació \(\mathbf{t} = (t_x, t_y, t_z)\).
  • Fórmula: Si un punt \(\mathbf{P} = (x, y, z)\) es trasllada per \(\mathbf{t}\), el nou punt \(\mathbf{P'}\) serà: \[ \mathbf{P'} = \mathbf{P} + \mathbf{t} = (x + t_x, y + t_y, z + t_z) \]

Exemple de Translació

Suposem que tenim un punt \(\mathbf{P} = (1, 2, 3)\) i volem traslladar-lo pel vector \(\mathbf{t} = (4, -2, 1)\).

\[ \mathbf{P'} = (1 + 4, 2 - 2, 3 + 1) = (5, 0, 4) \]

En codi Python, això es pot realitzar de la següent manera:

import numpy as np

# Punt original
P = np.array([1, 2, 3])

# Vector de translació
t = np.array([4, -2, 1])

# Nou punt després de la translació
P_prime = P + t

print("Punt original:", P)
print("Vector de translació:", t)
print("Nou punt:", P_prime)

Rotacions

La rotació és una transformació que gira un objecte al voltant d'un eix fix.

  • Definició: Una rotació en 3D es defineix per un angle de rotació \(\theta\) i un eix de rotació \(\mathbf{u} = (u_x, u_y, u_z)\).
  • Matriu de Rotació: La matriu de rotació \(R\) per un angle \(\theta\) al voltant de l'eix \(\mathbf{u}\) es pot expressar utilitzant la fórmula de Rodrigues: \[ R = I + \sin(\theta)K + (1 - \cos(\theta))K^2 \] on \(I\) és la matriu identitat i \(K\) és la matriu de components creuats del vector \(\mathbf{u}\).

Exemple de Rotació

Suposem que volem girar un punt \(\mathbf{P} = (1, 0, 0)\) 90 graus (\(\pi/2\) radians) al voltant de l'eix \(z\).

La matriu de rotació \(R_z\) per un angle \(\theta\) al voltant de l'eix \(z\) és:

\[ R_z = \begin{bmatrix} \cos(\theta) & -\sin(\theta) & 0
\sin(\theta) & \cos(\theta) & 0
0 & 0 & 1 \end{bmatrix} \]

Per \(\theta = \pi/2\):

\[ R_z = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \]

Aplicant aquesta matriu al punt \(\mathbf{P}\):

\[ \mathbf{P'} = R_z \mathbf{P} = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \begin{bmatrix} 1
0
0 \end{bmatrix} = \begin{bmatrix} 0
1
0 \end{bmatrix} \]

En codi Python, això es pot realitzar de la següent manera:

import numpy as np

# Punt original
P = np.array([1, 0, 0])

# Angle de rotació (90 graus en radians)
theta = np.pi / 2

# Matriu de rotació al voltant de l'eix z
R_z = np.array([
    [np.cos(theta), -np.sin(theta), 0],
    [np.sin(theta), np.cos(theta), 0],
    [0, 0, 1]
])

# Nou punt després de la rotació
P_prime = R_z.dot(P)

print("Punt original:", P)
print("Matriu de rotació R_z:\n", R_z)
print("Nou punt:", P_prime)

Conclusió

Les rotacions i translacions són eines essencials per manipular objectes en l'espai tridimensional. Comprendre com aplicar aquestes transformacions és crucial per a qualsevol professional que treballi en camps relacionats amb les matemàtiques 3D. A través d'exemples i exercicis pràctics, es pot adquirir una comprensió sòlida d'aquests conceptes fonamentals.
