Introducció

Els extractes de dades a Tableau són una característica potent que et permet optimitzar el rendiment, treballar fora de línia i gestionar grans conjunts de dades de manera eficient. Aquesta secció et guiarà des dels conceptes bàsics de la creació d'extractes de dades fins a tècniques avançades per optimitzar i gestionar-los.

Què és un Extracte de Dades?

  • Un extracte de dades és una instantània de dades optimitzada per a l'agregació i carregada a la memòria del sistema per ser recordada ràpidament per a la visualització.
  • Els extractes es poden crear a partir de diverses fonts de dades, incloent-hi bases de dades, fulls de càlcul i serveis al núvol.
  • Milloren el rendiment reduint la càrrega sobre la font de dades original i permetent l'anàlisi fora de línia.

Creació d'un Extracte de Dades

Guia Pas a Pas

  1. Connecta't a la Font de Dades: Obre Tableau i connecta't a la font de dades desitjada.
  2. Selecciona Dades: Tria les taules o fulls específics que vols incloure a l'extracte.
  3. Crea l'Extracte:
    • Fes clic al botó "Extract" a la pestanya Font de Dades.
    • Opcionalment, pots filtrar les dades per incloure només els registres rellevants.
    • Fes clic a "Extract" per crear el fitxer.
1. Obre Tableau i connecta't a la teva font de dades.
2. A la pestanya Font de Dades, fes clic a "Extract".
3. Aplica els filtres necessaris.
4. Fes clic a "Extract" per guardar el fitxer.

Ús d'Extractes al Teu Llibre de Treball

  • Un cop creat l'extracte, apareixerà al panell de Dades.
  • Pots utilitzar-lo com qualsevol altra font de dades per crear visualitzacions.
  • Els extractes es poden actualitzar per incloure les dades més recents de la font original.

Filtres d'Extracte

  • Els filtres d'extracte et permeten limitar les dades incloses a l'extracte.
  • Això pot ser útil per millorar el rendiment i centrar-se en dades rellevants.

Exemple: Aplicació d'un Filtre d'Extracte

  1. Obre el Diàleg d'Extracte: Fes clic al botó "Extract".
  2. Afegeix Filtre: Fes clic a "Add" a la secció de Filtres.
  3. Selecciona Camp: Tria el camp pel qual vols filtrar.
  4. Defineix Criteris: Defineix els criteris del filtre (per exemple, rang de dates, valors específics).
1. Fes clic a "Extract" a la pestanya Font de Dades.
2. Al diàleg d'Extracte, fes clic a "Add" sota Filtres.
3. Selecciona el camp pel qual filtrar.
4. Defineix els criteris del filtre.

Actualització Incremental

  • L'actualització incremental et permet actualitzar l'extracte només amb dades noves, en lloc de recarregar tot el conjunt de dades.
  • Això és útil per a grans conjunts de dades on les actualitzacions completes són lentes.

Exemple: Configuració d'Actualització Incremental

  1. Obre el Diàleg d'Extracte: Fes clic al botó "Extract".
  2. Selecciona Actualització Incremental: Marca l'opció "Incremental refresh".
  3. Tria Camp: Selecciona el camp que s'utilitzarà per identificar les noves files (per exemple, una marca de temps o un ID únic).
1. Fes clic a "Extract" a la pestanya Font de Dades.
2. Marca l'opció "Incremental refresh".
3. Selecciona el camp per identificar les noves files.

Optimització d'Extractes

  • Agregació: Agrega dades per reduir el nombre de files.
  • Amaga Camps No Utilitzats: Exclou camps que no són necessaris en l'anàlisi.
  • Filtres d'Extracte: Utilitza filtres per limitar les dades incloses a l'extracte.

Exemple: Agregació de Dades

  1. Obre el Diàleg d'Extracte: Fes clic al botó "Extract".
  2. Agrega Dades: Marca l'opció "Aggregate data for visible dimensions".
1. Fes clic a "Extract" a la pestanya Font de Dades.
2. Marca "Aggregate data for visible dimensions".

Gestió d'Extractes

  • Actualització d'Extractes: Actualitza regularment els extractes per assegurar que les dades estiguin actualitzades.
  • Programació d'Actualitzacions: Utilitza Tableau Server o Tableau Online per programar actualitzacions automàtiques.
  • Emmagatzematge d'Extractes: Gestiona on s'emmagatzemen els extractes per optimitzar el rendiment i l'ús de l'emmagatzematge.

Exemple: Programació d'una Actualització

  1. Publica el Llibre de Treball: Publica el teu llibre de treball a Tableau Server o Tableau Online.
  2. Programa la Tasca: A la interfície del servidor, ves al llibre de treball i configura un horari d'actualització.
1. Publica el teu llibre de treball a Tableau Server o Tableau Online.
2. Navega fins al llibre de treball al servidor.
3. Configura un horari d'actualització a la interfície del servidor.

Conclusió

Els extractes de dades són una característica crucial a Tableau per millorar el rendiment, permetre l'anàlisi fora de línia i gestionar grans conjunts de dades. Entenent com crear, utilitzar i optimitzar extractes, pots millorar significativament l'eficiència i l'efectivitat de les teves visualitzacions a Tableau. Actualitzar i gestionar regularment els teus extractes assegura que les teves dades es mantinguin actualitzades i rellevants per a les teves necessitats d'anàlisi.
