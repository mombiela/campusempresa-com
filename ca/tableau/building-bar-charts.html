Introducció als Gràfics de Barres

Els gràfics de barres són un dels tipus de visualitzacions més comuns utilitzats en l'anàlisi de dades. Són particularment útils per comparar quantitats entre diferents categories. A Tableau, crear gràfics de barres és senzill, i es poden personalitzar de diverses maneres per millorar la seva llegibilitat i efectivitat.

Creació d'un Gràfic de Barres Bàsic

Per crear un gràfic de barres bàsic a Tableau, segueix aquests passos:

  1. Connecta't a la Font de Dades: Obre Tableau i connecta't a la teva font de dades.
  2. Arrossega i Deixa Caure Camps:
    • Arrossega una dimensió (per exemple, Categoria) a la prestatgeria de Columnes.
    • Arrossega una mesura (per exemple, Vendes) a la prestatgeria de Files.
  3. Selecciona el Tipus de Gràfic de Barres: Tableau crearà automàticament un gràfic de barres si les dades són adequades. Si no, fes clic al panell "Show Me" i selecciona la icona del gràfic de barres.

Exemple

Suposem que tens un conjunt de dades amb els camps Categoria i Vendes. Aquí tens com pots crear un gràfic de barres:

1. Arrossega `Categoria` a Columnes.
2. Arrossega `Vendes` a Files.
3. Tableau generarà un gràfic de barres mostrant les vendes per cada categoria.

Personalització dels Gràfics de Barres

Un cop tinguis un gràfic de barres bàsic, el pots personalitzar per adaptar-lo millor a les teves necessitats.

Canvi de Colors

  • Fes clic a la prestatgeria Color i tria una paleta de colors o colors específics per a les teves barres.

Afegir Etiquetes

  • Arrossega la mesura Vendes a la prestatgeria Etiqueta per mostrar els valors de les vendes a les barres.

Ordenar les Barres

  • Fes clic a la icona d'ordenació a la barra d'eines per ordenar les barres en ordre ascendent o descendent basat en els valors de les vendes.

Exemple

Per afegir etiquetes i canviar colors:

1. Arrossega `Vendes` a la prestatgeria `Etiqueta`.
2. Fes clic a la prestatgeria `Color` i selecciona una paleta de colors.

Tècniques Avançades de Gràfics de Barres

Per a usuaris més avançats, Tableau ofereix diverses tècniques per millorar els gràfics de barres.

Gràfics de Barres amb Doble Eix

Els gràfics de barres amb doble eix permeten comparar dues mesures diferents en el mateix gràfic.

  • Arrossega una segona mesura (per exemple, Benefici) a la prestatgeria de Files.
  • Fes clic amb el botó dret a l'eix de la segona mesura i selecciona "Dual-Axis".
  • Sincronitza els eixos si és necessari fent clic amb el botó dret a un dels eixos i seleccionant "Synchronize Axis".

Gràfics de Barres Apilades

Els gràfics de barres apilades són útils per mostrar la composició de categories dins d'una barra.

  • Arrossega una dimensió (per exemple, Sub-Categoria) a la prestatgeria Color.
  • Això apilarà les barres per les subcategories dins de cada categoria principal.

Exemple

Creació d'un gràfic de barres amb doble eix:

1. Arrossega `Categoria` a Columnes.
2. Arrossega `Vendes` a Files.
3. Arrossega `Benefici` a Files.
4. Fes clic amb el botó dret a l'eix de `Benefici` i selecciona "Dual-Axis".
5. Fes clic amb el botó dret a un dels eixos i selecciona "Synchronize Axis".

Millors Pràctiques per als Gràfics de Barres

  • Mantingues-ho Simple: Evita omplir el gràfic amb massa colors o etiquetes.
  • Utilitza Escales Consistents: Assegura't que les escales en els gràfics amb doble eix estiguin sincronitzades per a una comparació precisa.
  • Ressalta les Dades Clau: Utilitza colors o anotacions per ressaltar punts de dades importants.

Conclusió

Els gràfics de barres són una eina versàtil i poderosa a Tableau per visualitzar i comparar dades entre categories. Dominant tant les tècniques bàsiques com les avançades, pots crear gràfics de barres efectius i informatius que comuniquin les teves dades de manera clara i efectiva. Recorda seguir les millors pràctiques per assegurar-te que els teus gràfics siguin tant informatius com fàcils d'entendre.
