Connectar-se a fonts de dades és una habilitat fonamental a Tableau, que et permet visualitzar i analitzar dades de diversos orígens. Aquest tema et guiarà a través del procés de connexió a diferents fonts de dades, des de nivells bàsics fins a avançats.

Introducció a les connexions de dades

Abans d'endinsar-nos en els detalls, és essencial entendre què són les connexions de dades i per què són crucials a Tableau.

  • Connexions de dades: Els enllaços entre Tableau i les fonts de dades que vols analitzar.
  • Importància: Et permet importar dades a Tableau per a la seva visualització i anàlisi.

Tipus de fonts de dades

Tableau admet una àmplia gamma de fonts de dades. Aquí hi ha alguns tipus comuns:

  • Basades en fitxers: Excel, CSV, JSON, PDF.
  • Servidors de bases de dades: MySQL, PostgreSQL, SQL Server, Oracle.
  • Basades en el núvol: Google Sheets, Salesforce, Amazon Redshift.
  • Connectors de dades web: APIs, connectors personalitzats.

Connexió a fonts de dades basades en fitxers

Excel

  1. Obre Tableau.
  2. Fes clic a "Microsoft Excel" al panell de "Connect".
  3. Selecciona el fitxer Excel al qual vols connectar-te.
  4. Arrossega i deixa anar els fulls des del panell esquerre al llenç.
# Exemple: Connexió a un fitxer Excel
1. Obre Tableau.
2. Fes clic a "Microsoft Excel".
3. Selecciona "SalesData.xlsx".
4. Arrossega "Sheet1" al llenç.

CSV

  1. Obre Tableau.
  2. Fes clic a "Text File" al panell de "Connect".
  3. Selecciona el fitxer CSV al qual vols connectar-te.
  4. Arrossega i deixa anar el fitxer al llenç.
# Exemple: Connexió a un fitxer CSV
1. Obre Tableau.
2. Fes clic a "Text File".
3. Selecciona "SalesData.csv".
4. Arrossega el fitxer al llenç.

Connexió a servidors de bases de dades

MySQL

  1. Obre Tableau.
  2. Fes clic a "MySQL" al panell de "Connect".
  3. Introdueix els detalls del servidor (Servidor, Port, Base de dades, Nom d'usuari, Contrasenya).
  4. Fes clic a "Sign In".
# Exemple: Connexió a una base de dades MySQL
1. Obre Tableau.
2. Fes clic a "MySQL".
3. Introdueix els detalls del servidor:
   - Servidor: `localhost`
   - Port: `3306`
   - Base de dades: `sales_db`
   - Nom d'usuari: `root`
   - Contrasenya: `password`
4. Fes clic a "Sign In".

PostgreSQL

  1. Obre Tableau.
  2. Fes clic a "PostgreSQL" al panell de "Connect".
  3. Introdueix els detalls del servidor (Servidor, Port, Base de dades, Nom d'usuari, Contrasenya).
  4. Fes clic a "Sign In".
# Exemple: Connexió a una base de dades PostgreSQL
1. Obre Tableau.
2. Fes clic a "PostgreSQL".
3. Introdueix els detalls del servidor:
   - Servidor: `localhost`
   - Port: `5432`
   - Base de dades: `sales_db`
   - Nom d'usuari: `postgres`
   - Contrasenya: `password`
4. Fes clic a "Sign In".

Connexions de dades avançades

Connectors de dades web

  1. Obre Tableau.
  2. Fes clic a "Web Data Connector" al panell de "Connect".
  3. Introdueix la URL del connector de dades web.
  4. Segueix les instruccions per connectar-te a les dades.
# Exemple: Connexió a un connector de dades web
1. Obre Tableau.
2. Fes clic a "Web Data Connector".
3. Introdueix la URL: `https://example.com/webdataconnector`.
4. Segueix les instruccions per connectar-te a les dades.

SQL personalitzat

  1. Obre Tableau.
  2. Connecta't a una base de dades (per exemple, MySQL, PostgreSQL).
  3. Fes clic a "New Custom SQL".
  4. Introdueix la teva consulta SQL.
-- Exemple: Consulta SQL personalitzada
SELECT 
    customer_id,
    customer_name,
    total_sales
FROM 
    sales_data
WHERE 
    total_sales > 1000;

Conclusió

Connectar-se a fonts de dades a Tableau és un pas crític en el procés de visualització de dades. Tant si treballes amb dades basades en fitxers, bases de dades, serveis al núvol o connectors web personalitzats, Tableau proporciona un conjunt robust d'eines per ajudar-te a portar les teves dades a la plataforma. Dominar aquestes connexions et permetrà aprofitar tot el potencial de Tableau per a les teves necessitats d'anàlisi de dades.
