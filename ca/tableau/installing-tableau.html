Introducció

Tableau és una eina potent de visualització de dades utilitzada per convertir dades en brut en un format comprensible. És àmpliament utilitzada per a la intel·ligència empresarial i l'anàlisi de dades. Aquesta secció et guiarà a través del procés d'instal·lació de Tableau al teu sistema.

Requisits del Sistema

Abans d'instal·lar Tableau, assegura't que el teu sistema compleixi els següents requisits:

  • Sistema Operatiu: Windows 7 o posterior, macOS 10.12 o posterior
  • Processador: 1.5 GHz o més ràpid
  • Memòria: 4 GB de RAM mínim, 8 GB recomanats
  • Emmagatzematge: Almenys 1.5 GB d'espai lliure en disc
  • Internet: Una connexió a internet activa per descarregar i activar Tableau

Descarregant Tableau

Per descarregar Tableau, segueix aquests passos:

  1. Visita el Lloc Web de Tableau: Ves al lloc web oficial de Tableau a Tableau Download Page.
  2. Selecciona la Versió: Tria la versió adequada per al teu sistema operatiu (Windows o macOS).
  3. Omple les Dades: És possible que se't demani omplir algunes dades personals com el teu nom, adreça de correu electrònic i nom de l'empresa.
  4. Descarrega l'Instal·lador: Fes clic al botó de descàrrega per obtenir el fitxer de l'instal·lador.

Instal·lant Tableau a Windows

Segueix aquests passos per instal·lar Tableau en una màquina Windows:

  1. Executa l'Instal·lador: Localitza el fitxer de l'instal·lador descarregat (normalment a la carpeta de Descàrregues) i fes doble clic per executar-lo.
  2. Control de Compte d'Usuari: Si se't demana pel Control de Compte d'Usuari, fes clic a "Sí" per permetre que l'instal·lador faci canvis al teu dispositiu.
  3. Acord de Llicència: Llegeix i accepta l'acord de llicència marcant la casella i fent clic a "Instal·lar".
  4. Procés d'Instal·lació: L'instal·lador ara extraurà fitxers i instal·larà Tableau al teu sistema. Això pot trigar uns minuts.
  5. Llança Tableau: Un cop completada la instal·lació, fes clic a "Finalitzar" per llançar Tableau.

Instal·lant Tableau a macOS

Segueix aquests passos per instal·lar Tableau en una màquina macOS:

  1. Obre l'Instal·lador: Localitza el fitxer .dmg descarregat i fes doble clic per obrir-lo.
  2. Arrossega a Aplicacions: Arrossega la icona de Tableau a la carpeta d'Aplicacions.
  3. Llança Tableau: Obre la carpeta d'Aplicacions i fes doble clic a la icona de Tableau per llançar l'aplicació.
  4. Avís de Seguretat: Si veus un avís de seguretat, fes clic a "Obrir" per confirmar que vols executar l'aplicació.

Activant Tableau

Després de la instal·lació, necessites activar Tableau utilitzant una clau de producte o començant una prova:

  1. Clau de Producte: Si tens una clau de producte, introdueix-la quan se't demani i fes clic a "Activar".
  2. Comença la Prova: Si no tens una clau de producte, pots començar una prova gratuïta de 14 dies fent clic a "Comença la Prova".

Resolució de Problemes d'Instal·lació

Si trobes algun problema durant la instal·lació, considera els següents passos:

  • Comprova els Requisits del Sistema: Assegura't que el teu sistema compleixi els requisits mínims.
  • Connexió a Internet: Verifica que tens una connexió a internet estable.
  • Programari Antivirus: Desactiva temporalment el programari antivirus que podria estar bloquejant la instal·lació.
  • Drets d'Administrador: Assegura't que tens drets d'administrador al teu ordinador.

Conclusió

Instal·lar Tableau és un procés senzill, tant si utilitzes Windows com macOS. Assegura't que el teu sistema compleixi els requisits, segueix els passos per descarregar i instal·lar el programari, i actívalo utilitzant una clau de producte o una prova. Amb Tableau instal·lat, ara estàs llest per començar a explorar les seves potents capacitats de visualització de dades.
