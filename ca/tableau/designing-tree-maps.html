Introducció als Tree Maps

Els tree maps són una eina de visualització potent a Tableau que permet mostrar dades jeràrquiques utilitzant rectangles niats. Cada rectangle representa una categoria, i la seva mida i color es poden utilitzar per transmetre informació addicional.

Conceptes Clau

  • Dades Jeràrquiques: Dades que estan organitzades en una estructura en forma d'arbre amb relacions pare-fill.
  • Rectangles: Cada categoria o subcategoria està representada per un rectangle.
  • Mida i Color: La mida del rectangle pot representar una mesura (per exemple, vendes), mentre que el color pot representar una altra mesura (per exemple, benefici).

Creació del Teu Primer Tree Map

Per crear un tree map bàsic a Tableau, segueix aquests passos:

  1. Connecta't a les Dades: Obre Tableau i connecta't a la teva font de dades.
  2. Arrossega i Deixa Caure: Arrossega una dimensió a la prestatgeria de Files i una mesura a la prestatgeria de Columnes.
  3. Canvia a Tree Map: Fes clic al panell "Show Me" i selecciona l'opció Tree Map.

Exemple

Suposem que tens un conjunt de dades de vendes amb les següents columnes: Categoria, Subcategoria, Vendes i Benefici.

Categoria    Subcategoria    Vendes    Benefici
Mobles       Cadires         5000      200
Mobles       Taules          3000      150
Oficina      Paper           4000      100
Oficina      Carpetes        2000      50

Per crear un tree map:

  1. Arrossega Categoria a la prestatgeria de Files.
  2. Arrossega Vendes a la prestatgeria de Columnes.
  3. Fes clic a "Show Me" i selecciona l'opció Tree Map.

Personalització dels Tree Maps

Un cop hagis creat un tree map bàsic, el pots personalitzar per adaptar-lo millor a les teves necessitats.

Ajustar Mida i Color

  • Mida: Arrossega una mesura (per exemple, Vendes) a la prestatgeria de Mida per ajustar la mida dels rectangles.
  • Color: Arrossega una mesura (per exemple, Benefici) a la prestatgeria de Color per ajustar el color dels rectangles.

Afegir Etiquetes

  • Arrossega la dimensió (per exemple, Subcategoria) a la prestatgeria d'Etiqueta per afegir etiquetes a cada rectangle.

Exemple

Per millorar l'exemple anterior:

  1. Arrossega Vendes a la prestatgeria de Mida.
  2. Arrossega Benefici a la prestatgeria de Color.
  3. Arrossega Subcategoria a la prestatgeria d'Etiqueta.

Tècniques Avançades de Tree Map

Per a usuaris avançats, Tableau ofereix diverses tècniques per millorar encara més els tree maps.

Ús de Camps Calculats

Pots crear camps calculats per afegir lògica més complexa als teus tree maps.

Exemple

Crea un camp calculat per categoritzar els marges de benefici:

IF [Benefici] > 100 THEN "Alt"
ELSE "Baix"
END

Arrossega aquest camp calculat a la prestatgeria de Color per codificar els rectangles segons els marges de benefici.

Ús de Jerarquies

Tableau et permet crear jerarquies per organitzar millor les teves dades.

Exemple

  1. Fes clic amb el botó dret a Categoria i selecciona "Jerarquia" > "Crear Jerarquia".
  2. Afegeix Subcategoria a la jerarquia.
  3. Utilitza aquesta jerarquia en el teu tree map per mostrar rectangles niats.

Conclusió

Els tree maps són una manera versàtil i potent de visualitzar dades jeràrquiques a Tableau. Entenent els conceptes bàsics i explorant tècniques avançades, pots crear tree maps informatius i visualment atractius que comuniquin efectivament la història de les teves dades. Practica creant i personalitzant tree maps amb els teus conjunts de dades per convertir-te en un expert en aquesta tècnica de visualització.
