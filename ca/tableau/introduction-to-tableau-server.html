Visió general

Tableau Server és una eina potent que permet a les organitzacions compartir, distribuir i col·laborar en visualitzacions de dades creades a Tableau Desktop. Aquesta secció us introduirà als conceptes bàsics, característiques i funcionalitats de Tableau Server.

Conceptes clau

  • Tableau Server: Una plataforma per compartir i gestionar llibres de treball, quadres de comandament i fonts de dades de Tableau.
  • Llibres de treball: Col·leccions de quadres de comandament, fulls i històries creades a Tableau Desktop.
  • Quadres de comandament: Visualitzacions de dades interactives que combinen múltiples fulls.
  • Fonts de dades: Connexions a dades que es poden utilitzar en llibres de treball de Tableau.
  • Usuaris i grups: Individus i col·leccions d'individus que tenen accés a Tableau Server.
  • Permisos: Regles que defineixen què poden fer els usuaris i grups amb el contingut a Tableau Server.
  • Projectes: Unitats organitzatives dins de Tableau Server utilitzades per gestionar contingut.

Configuració de Tableau Server

Instal·lació

  1. Requisits del sistema: Assegureu-vos que el vostre sistema compleixi els requisits de maquinari i programari.
  2. Descarregar i instal·lar: Obteniu l'instal·lador de Tableau Server des del lloc web de Tableau i seguiu les instruccions d'instal·lació.
  3. Configuració: Configureu els paràmetres de Tableau Server com ports, mètodes d'autenticació i notificacions per correu electrònic.

Configuració inicial

  • Compte d'administrador: Creeu un compte d'administrador durant el procés d'instal·lació.
  • Llicència: Activeu la vostra llicència de Tableau Server.
  • Connexions de dades: Configureu connexions a les vostres fonts de dades.

Gestió d'usuaris i permisos

Afegir usuaris

  • Afegir manualment: Afegiu usuaris individualment a través de la interfície de Tableau Server.
  • Importació massiva: Utilitzeu fitxers CSV per importar múltiples usuaris alhora.
  • Integració amb Active Directory: Integreu-vos amb Active Directory per a la gestió d'usuaris.

Configuració de permisos

  • Permisos de contingut: Definiu quines accions poden realitzar els usuaris en llibres de treball, quadres de comandament i fonts de dades.
  • Permisos de projecte: Establiu permisos a nivell de projecte per gestionar l'accés a tot el contingut dins d'un projecte.

Publicació i compartició de contingut

Publicació de llibres de treball

  1. Des de Tableau Desktop: Publiqueu llibres de treball directament des de Tableau Desktop a Tableau Server.
  2. Edició web: Creeu i editeu llibres de treball directament a Tableau Server.

Compartició i col·laboració

  • Vistes: Compartiu vistes específiques de quadres de comandament amb usuaris.
  • Subscripcions: Permeteu que els usuaris es subscriguin a quadres de comandament i rebin actualitzacions per correu electrònic.
  • Comentaris: Permeteu que els usuaris comentin en els quadres de comandament per a discussions col·laboratives.

Monitorització i manteniment

Monitorització de l'estat del servidor

  • Estat de Tableau Server: Utilitzeu la pàgina d'estat de Tableau Server per monitoritzar la salut i el rendiment del servidor.
  • Registres: Reviseu els registres del servidor per a la resolució de problemes i l'anàlisi del rendiment.

Manteniment regular

  • Còpies de seguretat: Feu còpies de seguretat regulars de les dades de Tableau Server.
  • Actualitzacions: Mantingueu Tableau Server actualitzat amb els últims pegats i versions.

Conclusió

Tableau Server és una eina essencial per a les organitzacions que volen aprofitar tot el potencial de les seves visualitzacions de dades. Comprenent els conceptes clau, configurant el servidor, gestionant usuaris i permisos, i publicant i compartint contingut de manera efectiva, podeu assegurar-vos que la vostra organització maximitza el valor de les seves anàlisis de dades. La monitorització i el manteniment regulars ajudaran a mantenir el vostre Tableau Server funcionant de manera fluida i eficient.
