Garantir la Seguretat de les Dades
Utilitzant Extensions de Tableau ►

En aquest tema, tractarem els aspectes essencials per garantir la seguretat de les dades a Tableau. Això inclou entendre la importància de la seguretat de les dades, implementar permisos d'usuari i utilitzar les funcions de seguretat integrades de Tableau per protegir la informació sensible.

Importància de la Seguretat de les Dades

La seguretat de les dades és crucial en qualsevol eina de visualització de dades, incloent-hi Tableau. Garantir que la informació sensible estigui protegida contra l'accés no autoritzat i les violacions és essencial per mantenir la confiança i el compliment de les regulacions.

  • Confidencialitat: Garantir que les dades siguin accessibles només per aquells que estan autoritzats.
  • Integritat: Mantenir l'exactitud i la consistència de les dades durant el seu cicle de vida.
  • Disponibilitat: Garantir que les dades estiguin disponibles per als usuaris autoritzats quan es necessitin.

Permisos d'Usuari

Tableau proporciona configuracions robustes de permisos d'usuari que permeten als administradors controlar qui pot accedir i interactuar amb les dades.

Tipus de Permisos

  1. Permisos de Llibre de Treball: Controlen l'accés als llibres de treball.
  2. Permisos de Font de Dades: Controlen l'accés a les fonts de dades.
  3. Permisos de Projecte: Controlen l'accés als projectes, que poden contenir múltiples llibres de treball i fonts de dades.

Configuració de Permisos

Els permisos es poden establir a diferents nivells per garantir que només els usuaris autoritzats tinguin accés a dades específiques.

Exemple: Configuració de Permisos de Llibre de Treball

  1. Navegar al Llibre de Treball: Aneu al llibre de treball per al qual voleu establir permisos.
  2. Obrir Permisos: Feu clic a la pestanya 'Permissions'.
  3. Afegir Usuaris/Grups: Afegiu els usuaris o grups per als quals voleu establir permisos.
  4. Establir Permisos: Trieu els permisos adequats (per exemple, Veure, Editar, Eliminar).
| Tipus de Permís | Descripció                  |
|-----------------|-----------------------------|
| Veure           | Permet als usuaris veure el llibre de treball. |
| Editar          | Permet als usuaris editar el llibre de treball. |
| Eliminar        | Permet als usuaris eliminar el llibre de treball. |

Seguretat de la Font de Dades

Assegurar les fonts de dades és crític per garantir que les dades sensibles no estiguin exposades.

Credencials Incrustades

Quan publiqueu una font de dades, podeu triar incrustar les credencials. Això garanteix que la font de dades es pugui accedir sense que els usuaris hagin d'introduir credencials cada vegada.

Exemple: Incrustar Credencials

  1. Publicar Font de Dades: Quan publiqueu una font de dades, seleccioneu l'opció 'Embedded Credentials'.
  2. Introduir Credencials: Introduïu les credencials necessàries per a la font de dades.
  3. Guardar: Guardeu la font de dades amb les credencials incrustades.

Seguretat a Nivell de Fila

La seguretat a nivell de fila us permet restringir l'accés a files específiques en una font de dades basant-vos en els rols dels usuaris.

Exemple: Implementació de Seguretat a Nivell de Fila

  1. Crear un Filtre d'Usuari: Creeu un camp calculat que filtri les dades basant-se en el rol de l'usuari.
  2. Aplicar el Filtre: Apliqueu el filtre d'usuari a la font de dades.
IF [User Role] = 'Manager' THEN [Data]
ELSEIF [User Role] = 'Employee' THEN [Data] END

Funcions de Seguretat del Tableau Server

Tableau Server ofereix funcions de seguretat addicionals per protegir les vostres dades.

Xifratge SSL

El xifratge SSL (Secure Sockets Layer) garanteix que les dades transmeses entre Tableau Server i els clients siguin segures.

Activació del Xifratge SSL

  1. Obtenir un Certificat SSL: Compreu o genereu un certificat SSL.
  2. Configurar Tableau Server: Configureu Tableau Server per utilitzar el certificat SSL.
  3. Reiniciar Tableau Server: Reinicieu Tableau Server per aplicar els canvis.

Xifratge de Dades en Repòs

Tableau Server pot xifrar les dades en repòs per protegir-les contra l'accés no autoritzat.

Activació del Xifratge de Dades en Repòs

  1. Navegar a Configuració: Aneu a la configuració de Tableau Server.
  2. Activar el Xifratge: Activeu l'opció de xifratge de dades en repòs.
  3. Guardar Configuració: Guardeu la configuració per aplicar els canvis.

Conclusió

Garantir la seguretat de les dades a Tableau implica una combinació de permisos d'usuari, seguretat de la font de dades i aprofitar les funcions de seguretat integrades de Tableau Server. Entenent i implementant aquestes mesures de seguretat, podeu protegir la informació sensible i mantenir la integritat, confidencialitat i disponibilitat de les vostres dades.

En resum:

  • Entendre la importància de la seguretat de les dades.
  • Establir permisos d'usuari adequats per a llibres de treball, fonts de dades i projectes.
  • Utilitzar credencials incrustades i seguretat a nivell de fila per protegir les fonts de dades.
  • Utilitzar les funcions de xifratge SSL i xifratge de dades en repòs de Tableau Server.

Seguint aquestes millors pràctiques, podeu garantir que les vostres dades romanguin segures mentre utilitzeu Tableau.
