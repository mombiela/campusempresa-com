Ús de les Expressions LOD

Les Expressions de Nivell de Detall (LOD) a Tableau et permeten controlar la granularitat de les teves dades. Et permeten realitzar càlculs i agregacions complexes a diferents nivells de detall, independentment del nivell de detall de la vista. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de les Expressions LOD a Tableau.

Introducció a les Expressions LOD

  • Definició: Les Expressions LOD són una característica poderosa a Tableau que et permeten calcular valors al nivell de la font de dades i al nivell de la visualització.
  • Tipus d'Expressions LOD:
    • FIXED: Calcula valors utilitzant les dimensions especificades, ignorant el nivell de detall de la vista.
    • INCLUDE: Calcula valors utilitzant les dimensions especificades a més de les dimensions que hi ha a la vista.
    • EXCLUDE: Calcula valors utilitzant les dimensions especificades, excloent certes dimensions de la vista.

Expressions LOD Bàsiques

Expressions FIXED LOD

  • Sintaxi: { FIXED [Dimension1], [Dimension2], ... : Aggregate_Expression }
  • Exemple: Calcula les vendes totals per regió, independentment del nivell de detall de la vista.
{ FIXED [Region] : SUM([Sales]) }
  • Explicació: Aquesta expressió calcula la suma de les vendes per cada regió, ignorant qualsevol altra dimensió a la vista.

Expressions INCLUDE LOD

  • Sintaxi: { INCLUDE [Dimension1], [Dimension2], ... : Aggregate_Expression }
  • Exemple: Calcula la mitjana de vendes per client dins de cada regió.
{ INCLUDE [Customer Name] : AVG([Sales]) }
  • Explicació: Aquesta expressió calcula la mitjana de vendes per client, però inclou la dimensió del client a més de les dimensions que ja hi ha a la vista.

Expressions EXCLUDE LOD

  • Sintaxi: { EXCLUDE [Dimension1], [Dimension2], ... : Aggregate_Expression }
  • Exemple: Calcula les vendes totals excloent la categoria de producte.
{ EXCLUDE [Category] : SUM([Sales]) }
  • Explicació: Aquesta expressió calcula la suma de les vendes mentre exclou la dimensió de la categoria de producte del càlcul.

Expressions LOD Intermèdies

Combinació d'Expressions LOD

  • Exemple: Calcula la diferència entre les vendes totals per regió i la mitjana de vendes per regió.
{ FIXED [Region] : SUM([Sales]) } - { INCLUDE [Region] : AVG([Sales]) }
  • Explicació: Aquesta expressió primer calcula les vendes totals per regió i després resta la mitjana de vendes per regió, combinant expressions FIXED i INCLUDE LOD.

Expressions LOD Imbricades

  • Exemple: Calcula les vendes màximes per client dins de cada regió.
{ FIXED [Region] : MAX({ INCLUDE [Customer Name] : SUM([Sales]) }) }
  • Explicació: Aquesta expressió LOD imbricada primer calcula la suma de les vendes per client i després troba les vendes màximes per client dins de cada regió.

Expressions LOD Avançades

Ús d'Expressions LOD amb Filtres

  • Exemple: Calcula les vendes totals per regió, excloent certes categories.
{ FIXED [Region] : SUM(IF [Category] != 'Furniture' THEN [Sales] END) }
  • Explicació: Aquesta expressió calcula la suma de les vendes per regió, excloent les vendes de la categoria 'Furniture'.

Expressions LOD Dinàmiques

  • Exemple: Calcula les vendes totals per regió, ajustant-se dinàmicament segons les dimensions seleccionades per l'usuari.
{ FIXED [Region], [User Selected Dimension] : SUM([Sales]) }
  • Explicació: Aquesta expressió permet als usuaris seleccionar dinàmicament les dimensions que s'inclouran en el càlcul LOD.

Conclusió

Les Expressions LOD a Tableau són una eina versàtil que et pot ajudar a realitzar càlculs i agregacions complexes a diferents nivells de detall. En dominar les expressions FIXED, INCLUDE i EXCLUDE LOD, pots obtenir una comprensió més profunda de les teves dades i crear visualitzacions més sofisticades. Practica l'ús d'aquestes expressions amb diversos conjunts de dades per entendre completament el seu comportament i aplicacions.
