Introducció a les Extensions de Tableau

Les Extensions de Tableau són eines potents que permeten ampliar la funcionalitat de Tableau més enllà de les seves capacitats integrades. Et permeten integrar visualitzacions personalitzades, interaccions de dades i altres funcions avançades directament als teus quadres de comandament de Tableau.

Conceptes Clau

  • API d'Extensions: El conjunt de funcions i esdeveniments que permeten als desenvolupadors crear extensions personalitzades.
  • Galeria d'Extensions: Una col·lecció d'extensions preconstruïdes disponibles per al seu ús a Tableau.
  • Seguretat: Comprendre les implicacions de seguretat i els permisos necessaris per utilitzar extensions.

Configuració de les Extensions de Tableau

Abans de poder utilitzar les Extensions de Tableau, necessites configurar el teu entorn i entendre com afegir extensions als teus quadres de comandament.

Passos per Afegir una Extensió

  1. Obre Tableau Desktop: Comença obrint la teva aplicació Tableau Desktop.
  2. Crea o Obre un Quadre de Comandament: Crea un nou quadre de comandament o obre'n un d'existent.
  3. Arrossega l'Objecte d'Extensió: Des del panell d'Objectes, arrossega l'objecte "Extensió" al teu quadre de comandament.
  4. Tria una Extensió: Selecciona una extensió de la Galeria d'Extensions o carrega un fitxer d'extensió personalitzat.
  5. Configura l'Extensió: Segueix les instruccions per configurar l'extensió segons sigui necessari.

Exemple: Afegir una Extensió Preconstruïda

1. Obre Tableau Desktop i crea un nou quadre de comandament.
2. Arrossega l'objecte "Extensió" des del panell d'Objectes al quadre de comandament.
3. Al diàleg "Tria una Extensió", selecciona "Galeria d'Extensions".
4. Navega per la galeria i selecciona una extensió, com ara "DataTables".
5. Fes clic a "Afegeix al Quadre de Comandament" i configura els paràmetres de l'extensió segons les instruccions.

Creació d'Extensions Personalitzades de Tableau

Per als usuaris avançats, crear Extensions de Tableau personalitzades pot proporcionar solucions adaptades a necessitats específiques.

Prerequisits

  • Coneixements de JavaScript: Es requereix una comprensió bàsica de JavaScript.
  • Eines de Desenvolupament Web: Familiaritat amb HTML, CSS i eines de desenvolupament web.

Passos per Crear una Extensió Personalitzada

  1. Configura el Teu Entorn de Desenvolupament: Instal·la les eines necessàries com Node.js i un editor de codi.
  2. Crea els Fitxers de l'Extensió: Desenvolupa els fitxers HTML, CSS i JavaScript per a la teva extensió.
  3. Utilitza l'API d'Extensions: Implementa l'API d'Extensions de Tableau en el teu codi JavaScript.
  4. Prova la Teva Extensió: Prova la teva extensió a Tableau Desktop per assegurar-te que funciona com s'espera.
  5. Desplega l'Extensió: Un cop provada, desplega la teva extensió per al seu ús a Tableau.

Exemple: Codi Bàsic de JavaScript per a una Extensió

// Inicialitza l'API d'Extensions de Tableau
tableau.extensions.initializeAsync().then(() => {
  // El teu codi d'extensió aquí
  console.log("Extensió inicialitzada");
  // Exemple: Obtenir dades d'un full de treball
  let worksheet = tableau.extensions.dashboardContent.dashboard.worksheets[0];
  worksheet.getSummaryDataAsync().then((data) => {
    console.log(data);
  });
});

Millors Pràctiques per Utilitzar Extensions

  • Consideracions de Seguretat: Revisa sempre les implicacions de seguretat d'utilitzar extensions de tercers.
  • Impacte en el Rendiment: Tingues en compte l'impacte en el rendiment que les extensions poden tenir en els teus quadres de comandament.
  • Experiència d'Usuari: Assegura't que les extensions millorin l'experiència d'usuari i no sobrecarreguin el quadre de comandament.

Conclusió

Les Extensions de Tableau ofereixen una manera versàtil d'ampliar la funcionalitat dels teus quadres de comandament. Tant si utilitzes extensions preconstruïdes de la Galeria d'Extensions com si crees solucions personalitzades, entendre com implementar i gestionar efectivament les extensions pot millorar significativament els teus projectes de Tableau. Considera sempre la seguretat, el rendiment i l'experiència d'usuari quan treballis amb extensions per assegurar els millors resultats per als teus quadres de comandament.
