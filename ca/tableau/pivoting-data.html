Introducció a Pivotar Dades

Pivotar dades és una habilitat crucial a Tableau que et permet transformar les teves dades per a una millor anàlisi i visualització. Aquesta secció cobrirà els conceptes bàsics de pivotar dades, per què és important i com realitzar operacions de pivot a Tableau.

Què és Pivotar Dades?

  • Definició: Pivotar dades implica transformar files en columnes o columnes en files per reorganitzar l'estructura de les dades.
  • Propòsit: Ajuda a fer les dades més adequades per a l'anàlisi i la visualització alineant-les amb el format desitjat.

Per què Pivotar Dades?

  • Simplifica l'Anàlisi: Fa més fàcil realitzar càlculs i crear visualitzacions.
  • Millora la Claredat: Ajuda a entendre i interpretar millor les dades.
  • Augmenta la Flexibilitat: Permet una manipulació de dades més dinàmica i flexible.

Com Pivotar Dades a Tableau

Pivotar Columnes a Files

Per pivotar columnes a files a Tableau, segueix aquests passos:

  1. Connecta't a la Font de Dades: Carrega el teu conjunt de dades a Tableau.
  2. Selecciona Columnes: Ressalta les columnes que vols pivotar.
  3. Opció de Pivot: Fes clic dret i selecciona "Pivot".

Exemple

Suposem que tens un conjunt de dades amb dades de vendes mensuals per a diferents regions:

| Regió | Vendes Gen | Vendes Feb | Vendes Mar | |-------|------------|------------|------------| | Est | 1000 | 1500 | 2000 | | Oest | 1200 | 1600 | 2100 |

Després de pivotar les columnes a files, les dades es veuran així:

| Regió | Mes | Vendes | |-------|------|--------| | Est | Gen | 1000 | | Est | Feb | 1500 | | Est | Mar | 2000 | | Oest | Gen | 1200 | | Oest | Feb | 1600 | | Oest | Mar | 2100 |

Pivotar Files a Columnes

Pivotar files a columnes és menys comú però es pot fer utilitzant camps calculats o Tableau Prep.

  1. Utilitza Camps Calculats: Crea camps calculats per a cada nova columna.
  2. Tableau Prep: Utilitza Tableau Prep per a operacions de pivot més complexes.

Exemple

Si tens dades en el següent format:

| Regió | Mes | Vendes | |-------|------|--------| | Est | Gen | 1000 | | Est | Feb | 1500 | | Est | Mar | 2000 | | Oest | Gen | 1200 | | Oest | Feb | 1600 | | Oest | Mar | 2100 |

Pots pivotar files a columnes per obtenir:

| Regió | Vendes Gen | Vendes Feb | Vendes Mar | |-------|------------|------------|------------| | Est | 1000 | 1500 | 2000 | | Oest | 1200 | 1600 | 2100 |

Tècniques Avançades de Pivot

Utilitzar Tableau Prep per a Pivots Complexos

  • Tableau Prep: Una eina poderosa per a la preparació de dades que permet operacions de pivot més complexes.
  • Passos:
    1. Carrega les dades a Tableau Prep.
    2. Utilitza el pas "Pivot" per transformar les dades.
    3. Configura els ajustaments de pivot per aconseguir el format desitjat.

Pivotar amb Camps Calculats

  • Camps Calculats: Útils per crear noves columnes basades en dades existents.
  • Exemple: 
    IF [Mes] = 'Gen' THEN [Vendes] END
    Aquest camp calculat crearà una nova columna per a les vendes de gener.

Exercicis

Exercici 1: Pivot Bàsic

  1. Carrega un conjunt de dades amb dades de vendes mensuals.
  2. Pivota les columnes a files.
  3. Crea un gràfic de línies per visualitzar la tendència de vendes al llarg dels mesos.

Exercici 2: Pivot Avançat amb Tableau Prep

  1. Carrega un conjunt de dades complex a Tableau Prep.
  2. Realitza una operació de pivot per reorganitzar les dades.
  3. Exporta les dades i crea una visualització a Tableau.

Conclusió

Pivotar dades a Tableau és una habilitat essencial que millora la teva capacitat per analitzar i visualitzar dades de manera efectiva. En dominar tant les tècniques bàsiques com les avançades de pivot, pots transformar les teves dades en un format més útil, fent que la teva anàlisi sigui més perspicaz i les teves visualitzacions més impactants.
