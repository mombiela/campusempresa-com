Programació d'Extractes
Utilitzant Tableau Online ►

En aquesta secció, tractarem el concepte de programació d'extractes a Tableau, que és essencial per automatitzar les actualitzacions de dades i assegurar que les teves visualitzacions estiguin sempre actualitzades. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Introducció als Extractes

Els extractes són instantànies de dades optimitzades per a l'agregació i carregades a la memòria del sistema per ser recordades ràpidament per a la visualització. Poden millorar significativament el rendiment, especialment amb grans conjunts de dades.

  • Beneficis dels Extractes:
    • Millora del rendiment.
    • Accés a les dades fora de línia.
    • Reducció de la càrrega sobre la font de dades original.

Creació d'Extractes

Abans de programar extractes, necessites crear-los. Aquí tens com pots crear un extracte a Tableau:

  1. Connecta't a la teva font de dades.
  2. Selecciona les dades que vols extreure.
  3. Fes clic al menú Dades i selecciona Extreure Dades.
  4. Configura les opcions de l'extracte (per exemple, filtres, agregació).
  5. Fes clic a Extreure.
# Exemple: Creació d'un Extracte
1. Obre Tableau i connecta't a la teva font de dades.
2. Ves al menú Dades i selecciona 'Extreure Dades'.
3. Tria els camps i filtres que vols incloure a l'extracte.
4. Fes clic a 'Extreure' per crear el fitxer d'extracte.

Programació d'Extractes a Tableau Desktop

Programar extractes a Tableau Desktop implica configurar un horari per actualitzar les dades a intervals especificats.

  • Passos per Programar Extractes:
    1. Crea un extracte com es descriu més amunt.
    2. Publica el llibre de treball a Tableau Server o Tableau Online.
    3. Configura l'horari d'actualització de l'extracte durant el procés de publicació.
# Exemple: Programació d'un Extracte a Tableau Desktop
1. Crea i desa el teu extracte.
2. Ves a 'Servidor' -> 'Publicar Llibre de Treball'.
3. Al diàleg 'Publicar Llibre de Treball', selecciona 'Programació i Autenticació'.
4. Tria un horari existent o crea'n un de nou.
5. Publica el llibre de treball.

Programació d'Extractes a Tableau Server

Tableau Server proporciona opcions més robustes per programar i gestionar extractes.

  • Passos per Programar Extractes:
    1. Navega a la pestanya Fonts de Dades a Tableau Server.
    2. Selecciona la font de dades que vols programar.
    3. Fes clic a la pestanya Actualitzacions d'Extractes.
    4. Fes clic a Nova Actualització d'Extracte.
    5. Configura l'horari (per exemple, freqüència, hora).
# Exemple: Programació d'un Extracte a Tableau Server
1. Inicia sessió a Tableau Server i ves a la pestanya 'Fonts de Dades'.
2. Selecciona la font de dades que vols programar.
3. Fes clic a 'Actualitzacions d'Extractes' i després a 'Nova Actualització d'Extracte'.
4. Configura la freqüència (per exemple, diària, setmanal) i l'hora de l'actualització.
5. Desa l'horari.

Opcions Avançades de Programació

Per a usuaris avançats, Tableau proporciona opcions addicionals com actualitzacions incrementals i actualitzacions condicionals.

  • Actualització Incremental: Només s'afegeixen a l'extracte les dades noves o actualitzades.
  • Actualització Condicional: Les actualitzacions es desencadenen en funció de condicions o esdeveniments específics.
# Exemple: Actualització Incremental
1. Mentre crees l'extracte, selecciona 'Actualització incremental'.
2. Especifica la columna per seguir els canvis (per exemple, una columna de marca de temps).
3. Programa l'extracte com de costum.

# Exemple: Actualització Condicional
1. Utilitza l'API REST de Tableau per configurar actualitzacions condicionals.
2. Escriu un script per desencadenar l'actualització en funció de condicions específiques.
3. Programa l'script per executar-se a intervals desitjats.

Conclusió

Programar extractes a Tableau és una característica poderosa que assegura que les teves visualitzacions de dades estiguin sempre actualitzades sense intervenció manual. Entenent com crear, programar i gestionar extractes, pots millorar significativament l'eficiència i la fiabilitat dels teus quadres de comandament de Tableau.

  • Punts Clau:
    • Els extractes milloren el rendiment i proporcionen accés fora de línia.
    • Programar extractes automatitza les actualitzacions de dades.
    • Tableau Server ofereix opcions de programació més avançades.

Dominant aquests conceptes, pots assegurar que les teves visualitzacions de Tableau siguin tant eficients com precises, proporcionant informació en temps real als teus interessats.
