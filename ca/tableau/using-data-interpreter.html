Introducció

El Data Interpreter és una característica potent de Tableau que ajuda a netejar i preparar dades per a l'anàlisi. Detecta i corregeix automàticament problemes comuns en les teves dades, com ara encapçalaments, peus de pàgina i cel·les fusionades. Aquesta secció et guiarà a través de l'ús del Data Interpreter des d'un nivell principiant fins a un nivell avançat.

Per què utilitzar el Data Interpreter?

  • Automatitza la neteja de dades: Estalvia temps detectant i corregint automàticament problemes comuns de dades.
  • Millora la qualitat de les dades: Assegura que les teves dades estiguin en un format net i utilitzable.
  • Simplifica la preparació de dades: Facilita la preparació de dades per a l'anàlisi sense necessitat d'una intervenció manual extensa.

Activar el Data Interpreter

Per utilitzar el Data Interpreter, segueix aquests passos:

  1. Connecta't a les dades: Obre Tableau i connecta't a la teva font de dades.
  2. Activa el Data Interpreter: A la pestanya de la Font de Dades, marca la casella etiquetada "Utilitza el Data Interpreter."

Entendre la sortida del Data Interpreter

Quan el Data Interpreter està activat, Tableau proporciona un resum dels canvis fets a les teves dades. Aquest resum inclou:

  • Encapçalaments detectats: Identifica i corregeix les files d'encapçalament.
  • Peus de pàgina eliminats: Elimina les files de peu de pàgina innecessàries.
  • Cel·les fusionades: Gestiona les cel·les fusionades dividint-les adequadament.

Exemple: Utilitzant el Data Interpreter amb un fitxer Excel

Fem un recorregut per un exemple d'ús del Data Interpreter amb un fitxer Excel.

Exemple pas a pas

  1. Connecta't al fitxer Excel: Obre Tableau i connecta't a un fitxer Excel amb dades desordenades.
  2. Previsualitza les dades: Abans d'activar el Data Interpreter, previsualitza les dades per veure els problemes.
  3. Activa el Data Interpreter: Marca la casella "Utilitza el Data Interpreter."
  4. Revisa els canvis: Fes clic a "Revisa els resultats" per veure un resum dels canvis fets pel Data Interpreter.

Abans i després del Data Interpreter

| Problema | Abans del Data Interpreter | Després del Data Interpreter | |-----------------|----------------------------|------------------------------| | Encapçalaments | Col·locats incorrectament | Correctament identificats | | Peus de pàgina | Presents | Eliminats | | Cel·les fusionades | No gestionades | Dividides adequadament |

Ús avançat

Per a usuaris avançats, el Data Interpreter es pot personalitzar més:

  • Divisió personalitzada: Ajusta manualment com es divideixen les cel·les fusionades.
  • Reconeixement d'encapçalaments: Ajusta quines files es reconeixen com a encapçalaments.
  • Neteja de dades: Combina el Data Interpreter amb altres eines de neteja de dades a Tableau per a escenaris més complexos.

Exemple de divisió personalitzada

# Aquest és un exemple de pseudo-codi per il·lustrar la divisió personalitzada
data = load_data("example.xlsx")
data_interpreter = DataInterpreter(data)
data_interpreter.custom_split(column="A", split_char=";")
cleaned_data = data_interpreter.apply()

En aquest exemple, carreguem dades des d'un fitxer Excel, creem un objecte Data Interpreter, personalitzem la divisió per a la columna "A" utilitzant un punt i coma, i després apliquem els canvis.

Conclusió

Utilitzar el Data Interpreter a Tableau simplifica el procés de preparació de dades, facilitant la neteja i organització de les teves dades per a l'anàlisi. Tant si ets un principiant com un usuari avançat, entendre com aprofitar aquesta eina pot millorar significativament el teu flux de treball d'anàlisi de dades. Revisa sempre els canvis fets pel Data Interpreter per assegurar-te que les teves dades estan preparades amb precisió per a les teves necessitats específiques.
