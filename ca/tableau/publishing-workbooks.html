En aquesta secció, tractarem el procés de publicació de llibres de treball a Tableau. Publicar llibres de treball et permet compartir les teves visualitzacions i quadres de comandament amb altres persones, ja sigui dins de la teva organització o públicament a la web. Començarem amb els conceptes bàsics i avançarem cap a temes més avançats.

Introducció a la publicació de llibres de treball

Publicar un llibre de treball a Tableau implica carregar el teu llibre de treball a Tableau Server, Tableau Online o Tableau Public. Això fa que les teves visualitzacions siguin accessibles per a altres persones i permet la col·laboració i el compartir.

Conceptes clau

  • Tableau Server: Una solució local per compartir llibres de treball de Tableau dins d'una organització.
  • Tableau Online: Una solució basada en el núvol per compartir llibres de treball de Tableau.
  • Tableau Public: Una plataforma gratuïta per compartir llibres de treball de Tableau públicament a la web.

Passos per publicar un llibre de treball

1. Preparar el teu llibre de treball

Abans de publicar, assegura't que el teu llibre de treball està llest:

  • Netejar dades: Assegura't que les teves dades estan netes i són precises.
  • Finalitzar visualitzacions: Assegura't que totes les visualitzacions estan completes i polides.
  • Comprovar permisos: Verifica que tens els permisos necessaris per publicar.

2. Publicar a Tableau Server o Tableau Online

Per publicar un llibre de treball a Tableau Server o Tableau Online, segueix aquests passos:

  1. Obrir Tableau Desktop: Obre el llibre de treball que vols publicar.

  2. Iniciar sessió: Ves a Server > Sign In i introdueix les teves credencials de Tableau Server o Tableau Online.

  3. Publicar llibre de treball: Ves a Server > Publish Workbook.

    • Projecte: Selecciona el projecte on vols publicar el llibre de treball.
    • Nom: Introdueix un nom per al llibre de treball.
    • Fonts de dades: Tria si vols incrustar fonts de dades o utilitzar connexions en directe.
    • Permisos: Estableix permisos per a qui pot veure o editar el llibre de treball.
    • Extractes: Decideix si vols crear extractes per a un rendiment més ràpid.

  4. Publicar: Fes clic a Publish per carregar el llibre de treball.

3. Publicar a Tableau Public

Publicar a Tableau Public és similar però amb algunes diferències:

  1. Obrir Tableau Desktop: Obre el llibre de treball que vols publicar.
  2. Iniciar sessió: Ves a Server > Tableau Public > Save to Tableau Public i inicia sessió amb el teu compte de Tableau Public.
  3. Publicar llibre de treball: Introdueix un nom per al llibre de treball i fes clic a Save.

Exemple

Aquí tens un exemple de com publicar un llibre de treball a Tableau Server:

1. Obre Tableau Desktop i carrega el teu llibre de treball.
2. Ves a `Server` > `Sign In`.
3. Introdueix les teves credencials de Tableau Server.
4. Ves a `Server` > `Publish Workbook`.
5. Selecciona el projecte, posa un nom al llibre de treball i estableix els permisos.
6. Fes clic a `Publish`.

Gestió de llibres de treball publicats

Un cop un llibre de treball està publicat, el pots gestionar a través de Tableau Server o Tableau Online:

  • Editar: Fes canvis al llibre de treball directament al servidor.
  • Permisos: Ajusta qui pot veure o editar el llibre de treball.
  • Control de versions: Mantén un registre dels canvis i torna a versions anteriors si és necessari.

Exemple

Per editar un llibre de treball publicat:

  1. Iniciar sessió: Ves a Tableau Server o Tableau Online i inicia sessió.
  2. Navegar: Troba el llibre de treball que vols editar.
  3. Editar: Fes clic al llibre de treball i selecciona Edit.

Millors pràctiques per publicar llibres de treball

  • Optimitzar el rendiment: Utilitza extractes i optimitza les consultes per assegurar un rendiment ràpid.
  • Establir permisos adequats: Assegura't que només les persones adequades tenen accés per veure o editar el llibre de treball.
  • Actualitzacions regulars: Mantén els teus llibres de treball actualitzats amb les dades i visualitzacions més recents.

Conclusió

Publicar llibres de treball a Tableau és un pas crucial per compartir les teves idees i col·laborar amb altres persones. Tant si utilitzes Tableau Server, Tableau Online o Tableau Public, entendre el procés de publicació i les millors pràctiques t'ajudarà a compartir el teu treball de manera efectiva. Recorda preparar el teu llibre de treball, establir permisos adequats i gestionar els teus llibres de treball publicats per assegurar que siguin útils i rellevants.
