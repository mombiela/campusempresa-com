Introducció

Crear camps calculats a Tableau et permet realitzar càlculs sobre les teves dades, permetent obtenir insights més profunds i visualitzacions més dinàmiques. Aquest tema et guiarà a través dels conceptes bàsics dels camps calculats, des d'operacions aritmètiques simples fins a funcions més avançades.

Què és un Camp Calculat?

Un camp calculat és un nou camp creat aplicant una fórmula a les dades existents en la teva font de dades de Tableau. Això es pot utilitzar per a:

  • Realitzar operacions aritmètiques.
  • Aplicar lògica condicional.
  • Agregar dades.
  • Manipular cadenes i dates.

Creació d'un Camp Calculat Simple

Passos per Crear un Camp Calculat

  1. Obre Tableau i connecta't a la teva font de dades.
  2. Navega fins al panell de Dades.
  3. Fes clic amb el botó dret sobre la font de dades i selecciona "Crear Camp Calculat".
  4. Introdueix un nom per al teu camp calculat.
  5. Introdueix la fórmula per al teu càlcul.
  6. Fes clic a OK per crear el camp.

Exemple: Càlcul Aritmètic Bàsic

Suposem que tens un conjunt de dades amb els camps Vendes i Cost, i vols calcular el Benefici.

[Benefici] = [Vendes] - [Cost]
  1. Fes clic amb el botó dret sobre la font de dades.
  2. Selecciona "Crear Camp Calculat".
  3. Anomena el camp Benefici.
  4. Introdueix la fórmula [Vendes] - [Cost].
  5. Fes clic a OK.

Ús de Funcions en Camps Calculats

Tableau proporciona una àmplia gamma de funcions que es poden utilitzar en camps calculats. Aquestes inclouen:

  • Funcions de Cadena: p. ex., LEFT(), RIGHT(), MID()
  • Funcions de Data: p. ex., DATEADD(), DATEDIFF(), DATENAME()
  • Funcions Lògiques: p. ex., IF, THEN, ELSE, CASE
  • Funcions d'Agrupació: p. ex., SUM(), AVG(), MIN(), MAX()

Exemple: Ús de Funcions de Cadena

Suposem que tens un camp Nom Complet i vols extreure el primer nom.

[Primer Nom] = LEFT([Nom Complet], FIND([Nom Complet], ' ') - 1)
  1. Fes clic amb el botó dret sobre la font de dades.
  2. Selecciona "Crear Camp Calculat".
  3. Anomena el camp Primer Nom.
  4. Introdueix la fórmula LEFT([Nom Complet], FIND([Nom Complet], ' ') - 1).
  5. Fes clic a OK.

Exemple: Ús de Funcions de Data

Suposem que vols calcular el nombre de dies entre dues dates, Data de Comanda i Data d'Enviament.

[Dies per Enviar] = DATEDIFF('day', [Data de Comanda], [Data d'Enviament])
  1. Fes clic amb el botó dret sobre la font de dades.
  2. Selecciona "Crear Camp Calculat".
  3. Anomena el camp Dies per Enviar.
  4. Introdueix la fórmula DATEDIFF('day', [Data de Comanda], [Data d'Enviament]).
  5. Fes clic a OK.

Camps Calculats Avançats

Ús d'Expressions de Nivell de Detall (LOD)

Les expressions LOD et permeten calcular valors a nivell de la font de dades i a nivell de la visualització, proporcionant més control sobre la granularitat dels teus càlculs.

Exemple: Expressió LOD Fixa

Suposem que vols calcular la mitjana de vendes per regió, independentment del nivell de detall en la vista.

[Mitjana Vendes per Regió] = { FIXED [Regió] : AVG([Vendes]) }
  1. Fes clic amb el botó dret sobre la font de dades.
  2. Selecciona "Crear Camp Calculat".
  3. Anomena el camp Mitjana Vendes per Regió.
  4. Introdueix la fórmula { FIXED [Regió] : AVG([Vendes]) }.
  5. Fes clic a OK.

Ús de Càlculs de Taula

Els càlculs de taula són transformacions que apliques als valors en una visualització. Es calculen basant-se en les dades de la vista.

Exemple: Total Acumulat

Suposem que vols calcular un total acumulat de vendes.

  1. Arrossega Vendes a la prestatgeria de Files.
  2. Fes clic amb el botó dret sobre la pastilla de Vendes.
  3. Selecciona "Càlcul de Taula Ràpid" > "Total Acumulat".

Conclusió

Crear camps calculats a Tableau és una manera poderosa de millorar les teves capacitats d'anàlisi de dades. Entenent i utilitzant operacions aritmètiques bàsiques, funcions, expressions LOD i càlculs de taula, pots obtenir insights més significatius de les teves dades. Practica creant diferents tipus de camps calculats per convertir-te en un expert en transformar i analitzar les teves dades a Tableau.
