Introducció

La instal·lació i configuració de servidors és una habilitat essencial per a qualsevol professional de TI. Aquest tema cobreix els conceptes bàsics i avançats necessaris per instal·lar i configurar servidors de manera eficient i segura. Aprendràs a seleccionar el maquinari adequat, instal·lar sistemes operatius, configurar serveis i assegurar el teu servidor.

Selecció del Maquinari

Abans d'instal·lar un servidor, és crucial seleccionar el maquinari adequat per a les teves necessitats. Aquí hi ha alguns factors a considerar:

  • Processador (CPU): Assegura't que el processador tingui prou potència per manejar les càrregues de treball previstes.
  • Memòria (RAM): La quantitat de RAM ha de ser suficient per suportar les aplicacions i serveis que s'executaran en el servidor.
  • Emmagatzematge: Considera la capacitat i velocitat de l'emmagatzematge. Les unitats SSD són més ràpides que les HDD tradicionals.
  • Xarxa: Verifica que la targeta de xarxa sigui compatible i tingui la capacitat de manejar el trànsit esperat.
  • Redundància: Considera la redundància en components crítics com fonts d'alimentació i discos durs per augmentar la fiabilitat.

Instal·lació del Sistema Operatiu

La tria del sistema operatiu (SO) depèn dels requisits de les teves aplicacions i serveis. Els SO més comuns per a servidors són:

  • Linux: Distribucions com Ubuntu Server, CentOS, i Debian.
  • Windows Server: Versions com Windows Server 2019 i 2022.

Instal·lació de Linux

  1. Descarregar la ISO: Descarrega la imatge ISO de la distribució Linux que prefereixis.
  2. Crear un mitjà d'instal·lació: Usa eines com Rufus per crear un USB d'arrencada.
  3. Configurar BIOS/UEFI: Assegura't que el servidor arrenqui des del USB.
  4. Instal·lar el SO: Segueix les instruccions de l'instal·lador per completar la instal·lació.
# Exemple de comandes per instal·lar Apache en Ubuntu Server
sudo apt update
sudo apt install apache2

Instal·lació de Windows Server

  1. Descarregar la ISO: Descarrega la imatge ISO des del lloc oficial de Microsoft.
  2. Crear un mitjà d'instal·lació: Usa l'eina de creació de mitjans de Windows.
  3. Configurar BIOS/UEFI: Assegura't que el servidor arrenqui des del USB.
  4. Instal·lar el SO: Segueix les instruccions de l'instal·lador per completar la instal·lació.
# Exemple de comandes per instal·lar IIS en Windows Server
Install-WindowsFeature -name Web-Server -IncludeManagementTools

Configuració de Serveis

Un cop el sistema operatiu estigui instal·lat, el següent pas és configurar els serveis necessaris. Aquí es cobreixen alguns serveis comuns:

Servidor Web

  • Apache (Linux)
  • IIS (Windows)

Servidor de Base de Dades

  • MySQL/MariaDB (Linux)
  • SQL Server (Windows)

Servidor de Fitxers

  • Samba (Linux)
  • File and Storage Services (Windows)

Exemple de Configuració d'Apache en Ubuntu

  1. Editar el fitxer de configuració:
sudo nano /etc/apache2/sites-available/000-default.conf
  1. Reiniciar el servei:
sudo systemctl restart apache2

Exemple de Configuració d'IIS en Windows Server

  1. Obrir l'Administrador d'IIS.
  2. Configurar el lloc web: Afegir un nou lloc web i configurar les seves propietats.
  3. Reiniciar el servei:
Restart-WebAppPool -Name "DefaultAppPool"

Seguretat del Servidor

La seguretat és un aspecte crític en la configuració de servidors. Aquí hi ha algunes pràctiques recomanades:

  • Actualitzacions: Mantén el sistema operatiu i els serveis actualitzats.
  • Tallafocs: Configura regles de tallafocs per limitar l'accés.
  • Usuaris i Permisos: Configura usuaris i permisos de manera adequada.
  • Monitoratge: Implementa eines de monitoratge per detectar activitats sospitoses.

Exemple de Configuració de Tallafocs en Ubuntu

sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw enable

Exemple de Configuració de Tallafocs en Windows Server

New-NetFirewallRule -DisplayName "Allow HTTP" -Direction Inbound -Protocol TCP -LocalPort 80 -Action Allow
New-NetFirewallRule -DisplayName "Allow HTTPS" -Direction Inbound -Protocol TCP -LocalPort 443 -Action Allow

Conclusió

La instal·lació i configuració de servidors és una tasca fonamental per a qualsevol professional de TI. Des de la selecció del maquinari fins a la configuració de serveis i la seguretat, cada pas és crucial per assegurar que el servidor funcioni de manera eficient i segura. Practicar aquests passos t'ajudarà a desenvolupar les habilitats necessàries per gestionar infraestructures de TI complexes.
