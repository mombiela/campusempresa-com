Introducció a les Xarxes

Les xarxes de computadors són sistemes que permeten la comunicació entre diferents dispositius. Aquests sistemes són essencials per a la infraestructura de TI, ja que faciliten l'intercanvi de dades i recursos.

Conceptes Clau

  • Xarxa: Conjunt de dispositius interconnectats que poden comunicar-se entre si.
  • Node: Qualsevol dispositiu connectat a la xarxa, com ordinadors, impressores, routers, etc.
  • Protocol: Conjunt de regles que determinen com es comuniquen els dispositius en una xarxa.
  • Topologia de Xarxa: La disposició física o lògica dels nodes en una xarxa.

Tipus de Xarxes

Hi ha diversos tipus de xarxes, cadascuna amb característiques i usos específics.

Classificació per Abast

  • LAN (Local Area Network): Xarxa d'àrea local que cobreix una petita àrea geogràfica, com una oficina o un edifici.
  • WAN (Wide Area Network): Xarxa d'àrea àmplia que cobreix grans distàncies geogràfiques, com ciutats o països.
  • MAN (Metropolitan Area Network): Xarxa d'àrea metropolitana que cobreix una ciutat o una regió metropolitana.

Classificació per Propietat

  • Xarxa Privada: Xarxa operada per una sola organització.
  • Xarxa Pública: Xarxa accessible per qualsevol usuari, com Internet.

Components d'una Xarxa

Perquè una xarxa funcioni correctament, necessita diversos components essencials.

Dispositius de Xarxa

  • Router: Dispositiu que dirigeix el trànsit de dades entre diferents xarxes.
  • Switch: Dispositiu que connecta diversos dispositius en una xarxa i utilitza la commutació de paquets per enviar dades només al dispositiu destinatari.
  • Firewall: Dispositiu de seguretat que controla el trànsit de xarxa entrant i sortint basat en regles de seguretat predefinides.

Mitjans de Transmissió

  • Cablejat de Parell Trenat: Comunament utilitzat en xarxes LAN, com el cable Ethernet.
  • Fibra Òptica: Utilitzada per a llargues distàncies i alta velocitat de transmissió.
  • Sense Fil: Utilitza ones de ràdio per a la transmissió de dades, com Wi-Fi.

Models de Referència

Per entendre com funcionen les xarxes, és útil conèixer els models de referència.

Model OSI

El Model d'Interconnexió de Sistemes Oberts (OSI) té set capes:

  1. Capa Física: Transmissió de bits a través d'un mitjà físic.
  2. Capa d'Enllaç de Dades: Proporciona transferència de dades lliure d'errors entre dos nodes.
  3. Capa de Xarxa: Determina la ruta que les dades han de seguir.
  4. Capa de Transport: Proporciona transferència de dades fiable.
  5. Capa de Sessió: Gestiona i controla les connexions entre ordinadors.
  6. Capa de Presentació: Traduïx les dades entre el format de la xarxa i el format de l'aplicació.
  7. Capa d'Aplicació: Proporciona serveis de xarxa a les aplicacions de l'usuari.

Model TCP/IP

El Model TCP/IP té quatre capes:

  1. Capa d'Accés a la Xarxa: Equivalent a les capes física i d'enllaç de dades del model OSI.
  2. Capa d'Internet: Equivalent a la capa de xarxa del model OSI.
  3. Capa de Transport: Equivalent a la capa de transport del model OSI.
  4. Capa d'Aplicació: Combina les capes de sessió, presentació i aplicació del model OSI.

Protocols de Xarxa

Els protocols són essencials per a la comunicació en xarxes.

Protocols Comuns

  • TCP (Transmission Control Protocol): Protocol orientat a la connexió que garanteix el lliurament de dades.
  • UDP (User Datagram Protocol): Protocol no orientat a la connexió que no garanteix el lliurament de dades.
  • IP (Internet Protocol): Protocol que adreça i encamina els paquets de dades a través de la xarxa.
  • HTTP (Hypertext Transfer Protocol): Protocol utilitzat per a la transferència de pàgines web.
  • FTP (File Transfer Protocol): Protocol utilitzat per a la transferència d'arxius.

Exemple de Configuració de Xarxa

A continuació, es mostra un exemple bàsic de configuració d'una xarxa utilitzant comandaments en un sistema operatiu basat en Unix.

# Configuració d'una adreça IP estàtica
sudo ifconfig eth0 192.168.1.10 netmask 255.255.255.0

# Configuració de la porta d'enllaç predeterminada
sudo route add default gw 192.168.1.1

# Configuració dels servidors DNS
echo "nameserver 8.8.8.8" | sudo tee /etc/resolv.conf
echo "nameserver 8.8.4.4" | sudo tee -a /etc/resolv.conf

Conclusió

Comprendre els fonaments de les xarxes és crucial per a qualsevol professional de TI. Aquest coneixement no només permet la configuració i el manteniment de xarxes, sinó que també és essencial per a la resolució de problemes i la implementació de solucions de seguretat. Amb una base sòlida en els conceptes de xarxes, els professionals poden construir i gestionar infraestructures de TI eficients i segures.
