Introducció

La configuració d'equips de xarxa és una habilitat fonamental per a qualsevol professional de TI. Aquest tema abasta els conceptes essencials i les pràctiques recomanades per configurar routers, switches i altres dispositius de xarxa. Aprendràs a configurar aquests equips per assegurar una comunicació eficient i segura dins d'una infraestructura TI.

Conceptes Clau

  • Router: Dispositiu que dirigeix el trànsit de dades entre diferents xarxes.
  • Switch: Dispositiu que connecta múltiples dispositius dins de la mateixa xarxa i dirigeix el trànsit de dades entre ells.
  • Firewall: Dispositiu o programari que controla l'accés a la xarxa i protegeix contra amenaces externes.
  • VLAN (Virtual LAN): Segmentació lògica d'una xarxa física en múltiples xarxes virtuals.
  • IP Addressing: Assignació d'adreces IP a dispositius per identificar-los a la xarxa.
  • Subnetting: Tècnica per dividir una xarxa IP en subxarxes més petites.

Configuració de Routers

Configuració Bàsica

  1. Accés al Router:

    • Connecta't al router mitjançant consola, SSH o Telnet.
    • Exemple d'accés per consola: 
      console> enable
console# configure terminal

  2. Assignació d'Adreces IP:

    • Configura la interfície del router amb una adreça IP. 
      interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown

  3. Configuració de Rutes Estàtiques:

    • Afegeix rutes estàtiques per dirigir el trànsit a xarxes específiques. 
      ip route 192.168.2.0 255.255.255.0 192.168.1.2

Configuració Avançada

  1. Configuració de NAT (Network Address Translation):

    • Permet que múltiples dispositius en una xarxa local comparteixin una única adreça IP pública. 
      ip nat inside source list 1 interface GigabitEthernet0/0 overload
access-list 1 permit 192.168.1.0 0.0.0.255

  2. Configuració de DHCP (Dynamic Host Configuration Protocol):

    • Assigna automàticament adreces IP a dispositius a la xarxa. 
      ip dhcp pool LAN
network 192.168.1.0 255.255.255.0
default-router 192.168.1.1
dns-server 8.8.8.8

Configuració de Switches

Configuració Bàsica

  1. Accés al Switch:

    • Connecta't al switch mitjançant consola, SSH o Telnet.
    • Exemple d'accés per consola: 
      console> enable
console# configure terminal

  2. Configuració de VLANs:

    • Crea i assigna VLANs a ports específics. 
      vlan 10
name Sales
interface GigabitEthernet0/1
switchport mode access
switchport access vlan 10

  3. Configuració de Trunking:

    • Permet que múltiples VLANs passin a través d'un sol enllaç. 
      interface GigabitEthernet0/24
switchport mode trunk
switchport trunk allowed vlan 10,20

Configuració Avançada

  1. Configuració de STP (Spanning Tree Protocol):

    • Prevé bucles a la xarxa. 
      spanning-tree mode rapid-pvst
spanning-tree vlan 10 priority 4096

  2. Configuració de Port Security:

    • Limita el nombre d'adreces MAC que poden connectar-se a un port. 
      interface GigabitEthernet0/1
switchport port-security
switchport port-security maximum 2
switchport port-security violation restrict
switchport port-security mac-address sticky

Comparació de Dispositius de Xarxa

| Dispositiu | Funció Principal | Exemple de Configuració | |------------|------------------|-------------------------| | Router | Dirigeix trànsit entre xarxes | ip route 192.168.2.0 255.255.255.0 192.168.1.2 | | Switch | Connecta dispositius dins d'una xarxa | switchport access vlan 10 | | Firewall | Controla accés i protegeix la xarxa | access-list 100 permit ip any any |

Conclusió

La configuració d'equips de xarxa és essencial per assegurar una comunicació eficient i segura dins d'una infraestructura TI. Des de la configuració bàsica de routers i switches fins a la implementació de característiques avançades com NAT, DHCP, VLANs i STP, cada pas és crucial per mantenir una xarxa robusta i funcional. Practicar aquests conceptes i configuracions et permetrà gestionar i optimitzar xarxes de manera efectiva.
