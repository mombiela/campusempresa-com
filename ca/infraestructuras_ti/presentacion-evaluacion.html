Introducció

En aquesta secció, presentarem els objectius del curs sobre Infraestructures TI i com s'avaluarà el progrés dels estudiants. Aquest curs està dissenyat per proporcionar una comprensió profunda de les infraestructures de TI, abastant des dels conceptes bàsics fins a les pràctiques avançades.

Objectius del Curs

En finalitzar aquest curs, els estudiants seran capaços de:

  • Comprendre els components fonamentals d'una infraestructura TI.
  • Dissenyar i gestionar infraestructures TI eficients i segures.
  • Implementar solucions d'infraestructura en entorns empresarials.
  • Avaluar i optimitzar el rendiment de la infraestructura TI.

Estructura del Curs

El curs està dividit en diversos mòduls, cadascun enfocat en un aspecte específic de les infraestructures TI:

  1. Introducció a les Infraestructures TI
  2. Components de Maquinari i Programari
  3. Xarxes i Comunicacions
  4. Seguretat en Infraestructures TI
  5. Gestió i Manteniment
  6. Avaluació i Optimització

Metodologia d'Avaluació

Per assegurar que els estudiants han comprès i poden aplicar els coneixements adquirits, s'utilitzaran les següents eines d'avaluació:

  • Qüestionaris Setmanals: Al final de cada mòdul, es realitzarà un qüestionari per avaluar la comprensió dels conceptes clau.
  • Projectes Pràctics: Els estudiants hauran de completar projectes pràctics que simulin situacions reals en la gestió d'infraestructures TI.
  • Examen Final: Un examen comprensiu al final del curs per avaluar el coneixement global adquirit.

Exemple de Projecte Pràctic

Descripció del Projecte

Dissenyar una infraestructura TI per a una petita empresa que inclou:

  • Servidors
  • Equips de xarxa
  • Sistemes d'emmagatzematge
  • Mesures de seguretat

Requisits

  1. Servidors:

    • Seleccionar i justificar el tipus de servidors necessaris.
    • Configurar els servidors per a diferents rols (e.g., servidor web, servidor de base de dades).

  2. Equips de Xarxa:

    • Dissenyar la topologia de xarxa.
    • Seleccionar i justificar els dispositius de xarxa (e.g., routers, switches).

  3. Sistemes d'Emmagatzematge:

    • Triar solucions d'emmagatzematge adequades.
    • Configurar sistemes de còpia de seguretat i recuperació de dades.

  4. Mesures de Seguretat:

    • Implementar polítiques de seguretat.
    • Configurar firewalls i sistemes de detecció d'intrusos.

Exemple de Configuració d'un Servidor Web

# Actualitzar el sistema
sudo apt-get update && sudo apt-get upgrade

# Instal·lar Apache
sudo apt-get install apache2

# Iniciar el servei d'Apache
sudo systemctl start apache2

# Habilitar Apache perquè s'iniciï en arrencar el sistema
sudo systemctl enable apache2

# Verificar l'estat del servei
sudo systemctl status apache2

Explicació del Codi

  • Actualitzar el sistema: És important mantenir el sistema operatiu i els paquets actualitzats per assegurar l'estabilitat i seguretat.
  • Instal·lar Apache: Apache és un dels servidors web més utilitzats i la seva instal·lació és senzilla mitjançant el gestor de paquets apt.
  • Iniciar el servei d'Apache: Després de la instal·lació, s'ha d'iniciar el servei perquè el servidor web estigui operatiu.
  • Habilitar Apache a l'inici: Configurar Apache perquè s'iniciï automàticament en arrencar el sistema assegura que el servidor web estigui sempre disponible.
  • Verificar l'estat del servei: Comprovar que el servei està funcionant correctament és crucial per a la resolució de problemes.

Conclusió

La secció de Presentació i Avaluació estableix el marc per al curs sobre Infraestructures TI, definint clarament els objectius, l'estructura del curs i les metodologies d'avaluació. A través de qüestionaris, projectes pràctics i un examen final, els estudiants podran demostrar la seva comprensió i habilitats en la gestió d'infraestructures TI.
