Introducció

En l'àmbit de les infraestructures TI, els servidors juguen un paper crucial en proporcionar recursos i serveis a altres dispositius i usuaris en una xarxa. Aquest tema cobrirà els diferents tipus de servidors, les seves característiques i usos específics en un entorn empresarial.

Tipus de Servidors

1. Servidors de Fitxers

  • Funció Principal: Emmagatzemar i gestionar fitxers.
  • Usos Comuns:
    • Compartir documents i fitxers entre usuaris.
    • Realitzar còpies de seguretat de dades importants.
  • Exemple de Configuració: 
    # Configuració bàsica d'un servidor de fitxers en Linux usant Samba
sudo apt-get update
sudo apt-get install samba
sudo smbpasswd -a usuari
sudo nano /etc/samba/smb.conf
# Afegir la següent configuració al final del fitxer
[Compartit]
path = /home/usuari/compartit
available = yes
valid users = usuari
read only = no
browsable = yes
public = yes
writable = yes

2. Servidors Web

  • Funció Principal: Allotjar i servir pàgines web.
  • Usos Comuns:
    • Publicar llocs web i aplicacions web.
    • Proveir serveis d'API.
  • Exemple de Configuració: 
    # Configuració bàsica d'un servidor web en Linux usant Apache
sudo apt-get update
sudo apt-get install apache2
sudo systemctl start apache2
sudo systemctl enable apache2
# Col·locar fitxers HTML en /var/www/html

3. Servidors de Base de Dades

  • Funció Principal: Emmagatzemar i gestionar dades estructurades.
  • Usos Comuns:
    • Aplicacions empresarials que requereixen emmagatzematge de dades.
    • Sistemes de gestió de contingut (CMS).
  • Exemple de Configuració: 
    -- Configuració bàsica d'un servidor de base de dades MySQL
CREATE DATABASE la_meva_base_de_dades;
CREATE USER 'usuari'@'localhost' IDENTIFIED BY 'contrasenya';
GRANT ALL PRIVILEGES ON la_meva_base_de_dades.* TO 'usuari'@'localhost';
FLUSH PRIVILEGES;

4. Servidors de Correu

  • Funció Principal: Gestionar i enviar correus electrònics.
  • Usos Comuns:
    • Proveir serveis de correu electrònic a usuaris.
    • Filtrar i emmagatzemar correus electrònics.
  • Exemple de Configuració: 
    # Configuració bàsica d'un servidor de correu en Linux usant Postfix
sudo apt-get update
sudo apt-get install postfix
sudo dpkg-reconfigure postfix
# Seguir les instruccions per configurar el domini i el tipus de correu

5. Servidors d'Aplicacions

  • Funció Principal: Executar aplicacions empresarials.
  • Usos Comuns:
    • Servir aplicacions Java EE, .NET, etc.
    • Proveir serveis de backend per a aplicacions mòbils i web.
  • Exemple de Configuració: 
    # Configuració bàsica d'un servidor d'aplicacions usant Tomcat
sudo apt-get update
sudo apt-get install tomcat9
sudo systemctl start tomcat9
sudo systemctl enable tomcat9
# Desplegar aplicacions en /var/lib/tomcat9/webapps

Comparació de Tipus de Servidors

| Tipus de Servidor | Funció Principal | Usos Comuns | |------------------------|------------------------------------|--------------------------------------------------| | Servidor de Fitxers | Emmagatzemar i gestionar fitxers | Compartir documents, còpies de seguretat | | Servidor Web | Allotjar i servir pàgines web | Publicar llocs web, serveis d'API | | Servidor de Base de Dades | Emmagatzemar i gestionar dades estructurades | Aplicacions empresarials, CMS | | Servidor de Correu | Gestionar i enviar correus | Serveis de correu electrònic | | Servidor d'Aplicacions | Executar aplicacions empresarials | Aplicacions Java EE, .NET, backend per a apps |

Conclusió

Conèixer els diferents tipus de servidors i els seus usos és fonamental per dissenyar i mantenir una infraestructura TI eficient. Cada tipus de servidor té un propòsit específic i és crucial seleccionar el adequat segons les necessitats de l'organització. Aquest coneixement permet als professionals de TI optimitzar recursos i millorar la gestió de serveis i aplicacions en un entorn empresarial.
