El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquest tema, explorarem els components essencials que conformen una infraestructura de TI robusta i eficient. La infraestructura de TI és la columna vertebral de qualsevol organització moderna, proporcionant els recursos i serveis necessaris per suportar operacions diàries i estratègiques.

Servidors

Els servidors són ordinadors potents que proporcionen serveis a altres ordinadors (clients) en una xarxa. Hi ha diversos tipus de servidors, cadascun amb un propòsit específic:

  • Servidors de fitxers: Emmagatzemen i gestionen fitxers.
  • Servidors de bases de dades: Gestionen bases de dades i permeten l'accés a dades estructurades.
  • Servidors web: Allotgen llocs web i aplicacions web.
  • Servidors d'aplicacions: Executen aplicacions empresarials.

Exemple de configuració d'un servidor web amb Apache en Linux

# Actualitzar el sistema
sudo apt-get update

# Instal·lar Apache
sudo apt-get install apache2

# Iniciar el servei d'Apache
sudo systemctl start apache2

# Habilitar Apache perquè s'iniciï en arrencar el sistema
sudo systemctl enable apache2

# Verificar l'estat del servei
sudo systemctl status apache2

Emmagatzematge

L'emmagatzematge és crucial per guardar dades de manera segura i eficient. Els principals tipus d'emmagatzematge inclouen:

  • Emmagatzematge local: Discos durs interns i SSDs.
  • Emmagatzematge en xarxa (NAS): Dispositius d'emmagatzematge connectats a la xarxa.
  • Emmagatzematge al núvol: Serveis d'emmagatzematge proporcionats per proveïdors com AWS, Google Cloud i Azure.

Comparació de tipus d'emmagatzematge

| Tipus d'Emmagatzematge | Avantatges | Desavantatges | |------------------------|------------|---------------| | Local | Ràpid accés, control total | Limitat pel maquinari físic | | NAS | Compartit, fàcil d'expandir | Pot ser costós, requereix configuració | | Núvol | Escalable, accessible des de qualsevol lloc | Dependència de tercers, costos recurrents |

Xarxes

Les xarxes permeten la comunicació entre dispositius i sistemes. Els components clau d'una xarxa inclouen:

  • Routers: Dirigeixen el trànsit de xarxa entre diferents xarxes.
  • Switches: Connecten dispositius dins de la mateixa xarxa.
  • Firewalls: Protegeixen la xarxa d'accessos no autoritzats i amenaces.

Exemple de configuració bàsica d'un firewall amb UFW en Linux

# Instal·lar UFW
sudo apt-get install ufw

# Permetre trànsit SSH
sudo ufw allow ssh

# Permetre trànsit HTTP i HTTPS
sudo ufw allow http
sudo ufw allow https

# Habilitar el firewall
sudo ufw enable

# Verificar l'estat del firewall
sudo ufw status

Virtualització

La virtualització permet executar múltiples sistemes operatius i aplicacions en un sol servidor físic, millorant l'eficiència i reduint costos. Els principals tipus de virtualització inclouen:

  • Virtualització de servidors: Execució de múltiples servidors virtuals en un únic servidor físic.
  • Virtualització d'escriptori: Execució d'entorns d'escriptori virtuals.
  • Virtualització d'aplicacions: Execució d'aplicacions en entorns virtuals.

Exemple de creació d'una màquina virtual amb VirtualBox

# Crear una nova màquina virtual
VBoxManage createvm --name "MiVM" --ostype "Ubuntu_64" --register

# Configurar la memòria RAM
VBoxManage modifyvm "MiVM" --memory 2048

# Crear un disc dur virtual
VBoxManage createhd --filename "MiVM.vdi" --size 20000

# Adjuntar el disc dur a la màquina virtual
VBoxManage storagectl "MiVM" --name "SATA Controller" --add sata --controller IntelAhci
VBoxManage storageattach "MiVM" --storagectl "SATA Controller" --port 0 --device 0 --type hdd --medium "MiVM.vdi"

# Configurar l'adaptador de xarxa
VBoxManage modifyvm "MiVM" --nic1 nat

Seguretat

La seguretat és un component crític de la infraestructura de TI. Inclou mesures per protegir dades, xarxes i sistemes contra amenaces i atacs. Els elements clau de la seguretat inclouen:

  • Autenticació i autorització: Verificació d'identitats i control d'accés.
  • Xifrat: Protecció de dades mitjançant codificació.
  • Monitoratge i auditoria: Supervisió d'activitats i registres per detectar i respondre a incidents.

Exemple de configuració d'autenticació de dos factors (2FA) amb Google Authenticator en Linux

# Instal·lar Google Authenticator
sudo apt-get install libpam-google-authenticator

# Configurar Google Authenticator per a l'usuari actual
google-authenticator

# Editar el fitxer de configuració PAM per a SSH
sudo nano /etc/pam.d/sshd

# Afegir la següent línia al final del fitxer
auth required pam_google_authenticator.so

# Editar el fitxer de configuració SSH
sudo nano /etc/ssh/sshd_config

# Afegir o modificar les següents línies
ChallengeResponseAuthentication yes
AuthenticationMethods publickey,keyboard-interactive

# Reiniciar el servei SSH
sudo systemctl restart sshd

Conclusió

Comprendre els components clau de la infraestructura de TI és essencial per dissenyar, implementar i mantenir sistemes eficients i segurs. Des de servidors i emmagatzematge fins a xarxes, virtualització i seguretat, cada component juga un paper crucial en el suport de les operacions de TI. A mesura que avancem en el curs, aprofundirem en cadascun d'aquests components per proporcionar una comprensió més detallada i pràctica.
