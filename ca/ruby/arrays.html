Introducció als Arrays

Els arrays són col·leccions ordenades i indexades per enters de qualsevol objecte. Són una estructura de dades fonamental en Ruby, que et permet emmagatzemar i manipular llistes de dades.

Conceptes Clau

  • Col·lecció Ordenada: Els elements en un array s'emmagatzemen en un ordre específic.
  • Indexació: Els arrays utilitzen una indexació basada en zero, és a dir, el primer element està a l'índex 0.
  • Mida Dinàmica: Els arrays poden créixer o reduir-se segons sigui necessari.

Creació d'Arrays

Pots crear arrays utilitzant claudàtors [] o la classe Array.

# Utilitzant claudàtors
numbers = [1, 2, 3, 4, 5]

# Utilitzant la classe Array
words = Array.new
words = Array.new(3) # Crea un array amb 3 elements nil
words = Array.new(3, "hello") # Crea un array amb 3 elements "hello"

Accés als Elements

Accedeix als elements utilitzant el seu índex.

numbers = [1, 2, 3, 4, 5]
puts numbers[0] # Sortida: 1
puts numbers[4] # Sortida: 5

Modificació d'Arrays

Pots modificar arrays assignant valors a índexs específics o utilitzant mètodes com push, pop, shift i unshift.

numbers = [1, 2, 3, 4, 5]
numbers[2] = 10 # Modifica el tercer element
numbers.push(6) # Afegeix 6 al final de l'array
numbers.pop # Elimina l'últim element
numbers.shift # Elimina el primer element
numbers.unshift(0) # Afegeix 0 al començament de l'array

Mètodes d'Array

Ruby proporciona un conjunt ric de mètodes per manipular arrays.

Mètodes Comuns

  • length o size: Retorna el nombre d'elements.
  • empty?: Comprova si l'array està buit.
  • include?: Comprova si un element està present.
  • sort: Ordena l'array.
  • reverse: Inverteix l'array.
  • join: Converteix l'array en una cadena.
numbers = [3, 1, 4, 1, 5, 9]
puts numbers.length # Sortida: 6
puts numbers.empty? # Sortida: false
puts numbers.include?(4) # Sortida: true
puts numbers.sort # Sortida: [1, 1, 3, 4, 5, 9]
puts numbers.reverse # Sortida: [9, 5, 1, 4, 1, 3]
puts numbers.join(", ") # Sortida: "3, 1, 4, 1, 5, 9"

Iteració sobre Arrays

Utilitza bucles o iteradors per processar cada element en un array.

numbers = [1, 2, 3, 4, 5]

# Utilitzant un bucle for
for number in numbers
  puts number
end

# Utilitzant el mètode each
numbers.each do |number|
  puts number
end

Tècniques Avançades d'Array

Arrays Imbricats

Els arrays poden contenir altres arrays, creant arrays multidimensionals.

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
puts matrix[1][2] # Sortida: 6

Transformacions d'Array

Utilitza mètodes com map, select i reject per transformar arrays.

numbers = [1, 2, 3, 4, 5]

# Utilitzant map per crear un nou array amb cada element elevat al quadrat
squares = numbers.map { |number| number ** 2 }
puts squares # Sortida: [1, 4, 9, 16, 25]

# Utilitzant select per filtrar elements
evens = numbers.select { |number| number.even? }
puts evens # Sortida: [2, 4]

# Utilitzant reject per filtrar elements
odds = numbers.reject { |number| number.even? }
puts odds # Sortida: [1, 3, 5]

Operacions de Conjunt d'Array

Els arrays suporten operacions de conjunt com la unió, la intersecció i la diferència.

a = [1, 2, 3]
b = [3, 4, 5]

# Unió
puts (a | b) # Sortida: [1, 2, 3, 4, 5]

# Intersecció
puts (a & b) # Sortida: [3]

# Diferència
puts (a - b) # Sortida: [1, 2]

Conclusió

Els arrays són una estructura de dades versàtil i poderosa en Ruby, essencial per manejar col·leccions de dades. Des d'operacions bàsiques com accedir i modificar elements fins a tècniques avançades com transformacions i operacions de conjunt, dominar els arrays millorarà significativament les teves habilitats de programació en Ruby. Practica utilitzant arrays en diferents escenaris per esdevenir proficient en el seu ús.
