Introducció

Els operadors de comparació en Ruby s'utilitzen per comparar dos valors. Retornen un valor booleà (true o false) basat en la comparació. Entendre aquests operadors és crucial per controlar el flux dels teus programes a través d'instruccions condicionals i bucles.

Principals Operadors de Comparació

Operadors d'Igualtat

  • ==: Comprova si dos valors són iguals.
  • !=: Comprova si dos valors no són iguals.
a = 5
b = 5
c = 10

puts a == b  # Sortida: true
puts a != c  # Sortida: true

Operadors Relacionals

  • >: Major que.
  • <: Menor que.
  • >=: Major o igual que.
  • <=: Menor o igual que.
x = 7
y = 10

puts x > y   # Sortida: false
puts x < y   # Sortida: true
puts x >= 7  # Sortida: true
puts y <= 10 # Sortida: true

Operador de Comparació Combinada

  • <=>: També conegut com l'operador de la nau espacial, retorna:
    • -1 si l'operant esquerre és menor que l'operant dret.
    • 0 si l'operant esquerre és igual a l'operant dret.
    • 1 si l'operant esquerre és major que l'operant dret.
puts 5 <=> 10  # Sortida: -1
puts 10 <=> 10 # Sortida: 0
puts 15 <=> 10 # Sortida: 1

Operador d'Igualtat de Cas

  • ===: Utilitzat en instruccions de cas per provar la igualtat.
range = (1..10)

puts range === 5  # Sortida: true
puts range === 15 # Sortida: false

Identitat d'Objecte

  • equal?: Comprova si dos objectes són el mateix objecte.
str1 = "hello"
str2 = "hello"
str3 = str1

puts str1.equal?(str2) # Sortida: false
puts str1.equal?(str3) # Sortida: true

Exemples Pràctics

Utilitzant Operadors de Comparació en Instruccions Condicionals

age = 18

if age >= 18
  puts "Ets elegible per votar."
else
  puts "No ets elegible per votar."
end

Utilitzant l'Operador de la Nau Espacial per Ordenar

numbers = [5, 2, 9, 3, 8]

sorted_numbers = numbers.sort { |a, b| a <=> b }
puts sorted_numbers.inspect  # Sortida: [2, 3, 5, 8, 9]

Utilitzant Igualtat de Cas en Instruccions de Cas

grade = 'B'

case grade
when 'A'
  puts "Excel·lent!"
when 'B'
  puts "Bona feina!"
when 'C'
  puts "Ben fet!"
else
  puts "Segueix intentant-ho!"
end

Resum

Els operadors de comparació són fonamentals en Ruby per prendre decisions i controlar el flux dels teus programes. Et permeten comparar valors per igualtat, ordre relacional i identitat d'objecte. Entendre com utilitzar aquests operadors de manera efectiva et permetrà escriure codi més dinàmic i sensible.

En dominar aquests operadors, podràs gestionar una àmplia gamma de tasques de programació, des de comprovacions condicionals senzilles fins a algorismes de classificació complexos. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en la seva aplicació.
