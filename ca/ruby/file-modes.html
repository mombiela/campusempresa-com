Introducció

La gestió de fitxers és un aspecte crucial de la programació, ja que et permet llegir i escriure fitxers. En Ruby, els modes de fitxer determinen com s'obre un fitxer i quines operacions es poden realitzar. Aquesta secció cobrirà els diferents modes de fitxer disponibles en Ruby, proporcionant explicacions clares, exemples i exercicis per ajudar-te a entendre'ls i utilitzar-los de manera efectiva.

Conceptes clau

  • Modes de fitxer: Diferents maneres d'obrir un fitxer, determinant les operacions que es poden realitzar.
  • Modes de fitxer comuns: Modes com llegir, escriure, afegir i les seves combinacions.
  • Mètodes de la classe File: Mètodes proporcionats per la classe File de Ruby per gestionar fitxers.

Modes de fitxer comuns

Ruby proporciona diversos modes de fitxer que pots utilitzar per obrir fitxers. Aquí tens els més utilitzats:

| Mode | Descripció | |------|-------------| | r | Mode només lectura. El fitxer ha d'existir. | | w | Mode només escriptura. Crea un fitxer nou o trunca un fitxer existent a longitud zero. | | a | Mode afegir. Només escriptura. Crea un fitxer nou si no existeix. | | r+ | Mode lectura-escriptura. El fitxer ha d'existir. | | w+ | Mode lectura-escriptura. Crea un fitxer nou o trunca un fitxer existent a longitud zero. | | a+ | Mode lectura-escriptura. Crea un fitxer nou si no existeix. Les dades s'afegeixen al final del fitxer. |

Exemples

Mode només lectura (`r`)

# Obre un fitxer en mode només lectura
file = File.open("example.txt", "r")

# Llegeix el contingut del fitxer
content = file.read
puts content

# Tanca el fitxer
file.close

Explicació:

  • El fitxer s'obre en mode només lectura utilitzant "r".
  • El mètode read llegeix tot el contingut del fitxer.
  • El mètode close tanca el fitxer.

Mode només escriptura (`w`)

# Obre un fitxer en mode només escriptura
file = File.open("example.txt", "w")

# Escriu contingut al fitxer
file.write("Hello, World!")

# Tanca el fitxer
file.close

Explicació:

  • El fitxer s'obre en mode només escriptura utilitzant "w".
  • El mètode write escriu el contingut especificat al fitxer.
  • El mètode close tanca el fitxer.

Mode afegir (`a`)

# Obre un fitxer en mode afegir
file = File.open("example.txt", "a")

# Afegeix contingut al fitxer
file.write("\nAppended text.")

# Tanca el fitxer
file.close

Explicació:

  • El fitxer s'obre en mode afegir utilitzant "a".
  • El mètode write afegeix el contingut especificat al final del fitxer.
  • El mètode close tanca el fitxer.

Mode lectura-escriptura (`r+`)

# Obre un fitxer en mode lectura-escriptura
file = File.open("example.txt", "r+")

# Llegeix el contingut actual
content = file.read
puts "Current content: #{content}"

# Escriu nou contingut
file.write("\nNew content added.")

# Tanca el fitxer
file.close

Explicació:

  • El fitxer s'obre en mode lectura-escriptura utilitzant "r+".
  • El mètode read llegeix el contingut actual del fitxer.
  • El mètode write escriu nou contingut al fitxer.
  • El mètode close tanca el fitxer.

Exercicis

  1. Llegir un fitxer: Crea un fitxer anomenat data.txt amb algun text. Escriu un script en Ruby per llegir i imprimir el contingut del fitxer.
  2. Escriure a un fitxer: Escriu un script en Ruby per crear un nou fitxer anomenat output.txt i escriure el text "Ruby File Handling" en ell.
  3. Afegir a un fitxer: Modifica l'script de l'Exercici 2 per afegir el text "Appending more text." a output.txt.
  4. Operacions de lectura-escriptura: Escriu un script en Ruby per obrir un fitxer existent en mode lectura-escriptura, llegir el seu contingut i després afegir el text "End of file." a ell.

Conclusió

Entendre els modes de fitxer en Ruby és essencial per a una gestió efectiva dels fitxers. Dominant els diferents modes, podràs llegir, escriure i afegir a fitxers segons sigui necessari. Practica els exercicis proporcionats per consolidar la teva comprensió i convertir-te en un expert en la gestió de fitxers en Ruby.
