Introducció als Comentaris

Els comentaris són una part essencial de qualsevol llenguatge de programació, incloent Ruby. S'utilitzen per explicar el codi, fer-lo més llegible i ajudar altres desenvolupadors (o a tu mateix en el futur) a entendre la lògica darrere del codi. Els comentaris són ignorats per l'intèrpret de Ruby, així que no afecten l'execució del programa.

Tipus de Comentaris en Ruby

Ruby suporta dos tipus de comentaris:

  • Comentaris d'una sola línia
  • Comentaris de múltiples línies

Comentaris d'una sola línia

Els comentaris d'una sola línia comencen amb el símbol #. Tot el que segueix al # en aquesta línia es considera un comentari i és ignorat per l'intèrpret de Ruby.

Exemple:

# Aquest és un comentari d'una sola línia
puts "Hola, Món!" # Aquest comentari és ignorat per Ruby

Comentaris de múltiples línies

Els comentaris de múltiples línies, també coneguts com a comentaris de bloc, comencen amb =begin i acaben amb =end. Tot el que hi ha entre aquests marcadors es considera un comentari.

Exemple:

=begin
Aquest és un comentari de múltiples línies.
Pot abastar diverses línies.
=end

puts "Hola, Món!"

Millors Pràctiques per Utilitzar Comentaris

  • Mantingues els comentaris concisos i rellevants: Els comentaris han de ser curts i directes. Evita escriure comentaris innecessaris que no aportin valor.
  • Utilitza comentaris per explicar el perquè, no el què: El codi en si mateix hauria de ser autoexplicatiu en la majoria dels casos. Utilitza comentaris per explicar la raó darrere de lògiques o decisions complexes.
  • Actualitza els comentaris quan canviï el codi: Assegura't que els comentaris s'actualitzin sempre que es modifiqui el codi associat per evitar informació obsoleta o enganyosa.
  • Evita comentaris obvis: No escriguis comentaris que diguin l'obvi, ja que poden saturar el codi.

Exemple de Bons Comentaris:

# Calcula el factorial d'un número utilitzant recursió
def factorial(n)
  return 1 if n == 0 # Cas base: el factorial de 0 és 1
  n * factorial(n - 1) # Cas recursiu
end

Exemple de Mal Comentaris:

# Aquest és un mètode
def factorial(n)
  # Si n és 0, retorna 1
  return 1 if n == 0
  # Multiplica n pel factorial de n - 1
  n * factorial(n - 1)
end

Comentar Codi

De vegades, potser voldràs desactivar temporalment una part del codi sense eliminar-la. Això es pot fer comentant el codi.

Exemple:

# puts "Aquesta línia està comentada i no s'executarà"
puts "Aquesta línia s'executarà"

Conclusió

Els comentaris són una eina poderosa per fer que el teu codi Ruby sigui més comprensible i mantenible. Utilitzant comentaris d'una sola línia i de múltiples línies de manera efectiva, pots proporcionar context i explicacions valuoses per al teu codi. Recorda seguir les millors pràctiques per assegurar-te que els teus comentaris siguin útils i estiguin actualitzats. Feliç programació!
