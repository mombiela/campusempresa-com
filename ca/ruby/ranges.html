Introducció als Rangs

Els rangs en Ruby representen una seqüència de valors, que poden ser números, lletres o objectes. S'utilitzen comunament per a la iteració i la definició d'intervals.

Conceptes Clau

  • Operadors de Rang: .. (inclusiu) i ... (exclusiu).
  • Mètodes de Rang: Mètodes per manipular i consultar rangs.
  • Usos: Iteració, comprovació de condicions i segmentació.

Creació de Rangs

Els rangs es poden crear utilitzant els operadors .. i ....

Rangs Inclusius

Un rang inclusiu inclou el valor final.

inclusive_range = 1..5
puts inclusive_range.to_a  # Sortida: [1, 2, 3, 4, 5]

Rangs Exclusius

Un rang exclusiu exclou el valor final.

exclusive_range = 1...5
puts exclusive_range.to_a  # Sortida: [1, 2, 3, 4]

Mètodes de Rang

Ruby proporciona diversos mètodes per treballar amb rangs.

Mètode `to_a`

Converteix un rang en una matriu.

range = 1..3
array = range.to_a
puts array.inspect  # Sortida: [1, 2, 3]

Mètode `include?`

Comprova si un valor està dins del rang.

range = 1..5
puts range.include?(3)  # Sortida: true
puts range.include?(6)  # Sortida: false

Mètode `each`

Itera sobre cada element del rang.

(1..3).each do |number|
  puts number
end
# Sortida:
# 1
# 2
# 3

Característiques Avançades dels Rangs

Els rangs en Ruby es poden utilitzar de maneres més avançades, com amb objectes personalitzats i en declaracions condicionals.

Objectes Personalitzats en Rangs

Pots definir rangs amb objectes personalitzats si implementen l'operador <=> (nau espacial).

class Custom
  attr_reader :value

  def initialize(value)
    @value = value
  end

  def <=>(other)
    @value <=> other.value
  end
end

range = Custom.new(1)..Custom.new(3)
puts range.include?(Custom.new(2))  # Sortida: true

Rangs en Declaracions Condicionals

Els rangs es poden utilitzar en declaracions condicionals per comprovar si un valor es troba dins d'un rang específic.

age = 25

case age
when 0..12
  puts "Infant"
when 13..19
  puts "Adolescent"
when 20..64
  puts "Adult"
else
  puts "Gent Gran"
end
# Sortida: Adult

Resum

Els rangs en Ruby són una eina poderosa per representar seqüències i intervals. Es poden crear utilitzant els operadors inclusius (..) i exclusius (...), i ofereixen diversos mètodes per a la manipulació i la iteració. Entendre els rangs és essencial per a una programació efectiva en Ruby, especialment per a tasques que impliquen iteració, comprovació de condicions i maneig d'objectes personalitzats.
