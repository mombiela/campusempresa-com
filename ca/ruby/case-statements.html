Introducció a les Sentències Case

Les sentències case en Ruby són una eina poderosa per controlar el flux del teu programa basant-se en diferents condicions. Són similars a les sentències switch en altres llenguatges de programació i proporcionen una alternativa més llegible a múltiples sentències if-elsif-else.

Sintaxi Bàsica

La sintaxi bàsica d'una sentència case en Ruby és la següent:

case expression
when value1
  # codi a executar si expression == value1
when value2
  # codi a executar si expression == value2
else
  # codi a executar si cap de les condicions anteriors es compleix
end

Exemple

Aquí tens un exemple senzill per il·lustrar l'ús d'una sentència case:

day = "Saturday"

case day
when "Monday"
  puts "Start of the work week"
when "Friday"
  puts "End of the work week"
when "Saturday", "Sunday"
  puts "Weekend!"
else
  puts "Midweek day"
end

En aquest exemple:

  • Si day és "Monday", imprimeix "Start of the work week".
  • Si day és "Friday", imprimeix "End of the work week".
  • Si day és "Saturday" o "Sunday", imprimeix "Weekend!".
  • Per qualsevol altre valor de day, imprimeix "Midweek day".

Ús Avançat

Ús de Rangs

També pots utilitzar rangs en les sentències case per manejar un rang de valors.

score = 85

case score
when 90..100
  puts "Excellent"
when 75..89
  puts "Good"
when 50..74
  puts "Pass"
else
  puts "Fail"
end

Ús d'Expressions Regulars

Les sentències case també poden coincidir amb expressions regulars.

name = "Alice"

case name
when /Alic/
  puts "Name contains 'Alic'"
when /^A/
  puts "Name starts with 'A'"
else
  puts "Name does not match any pattern"
end

Ús de Múltiples Condicions

Pots especificar múltiples condicions en una sola clàusula when separant-les amb comes.

fruit = "Apple"

case fruit
when "Apple", "Banana", "Cherry"
  puts "This is a common fruit"
else
  puts "This is an uncommon fruit"
end

Millors Pràctiques

  • Llegibilitat: Utilitza les sentències case per millorar la llegibilitat del teu codi, especialment quan tractes amb múltiples condicions.
  • Simplicitat: Mantingues les condicions simples i evita lògiques excessivament complexes dins d'una sola sentència case.
  • Cas per Defecte: Inclou sempre una clàusula else per manejar valors inesperats.

Resum

Les sentències case en Ruby proporcionen una manera neta i llegible de manejar múltiples condicions. Es poden utilitzar amb valors simples, rangs, expressions regulars i múltiples condicions. Seguint les millors pràctiques, pots fer que el teu codi sigui més mantenible i fàcil d'entendre.

Punts Clau

  • Les sentències case són una alternativa a múltiples sentències if-elsif-else.
  • Milloren la llegibilitat i mantenibilitat del codi.
  • Poden manejar rangs, expressions regulars i múltiples condicions.
  • Inclou sempre una clàusula else per manejar valors inesperats.

Dominant les sentències case, pots escriure codi Ruby més eficient i llegible.
