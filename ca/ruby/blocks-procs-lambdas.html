Introducció

A Ruby, els blocs, procs i lambdas són eines poderoses per gestionar fragments de codi. Et permeten passar codi com si fossin variables, fent que els teus programes siguin més flexibles i concisos. Aquesta secció et guiarà per entendre i utilitzar aquestes característiques de manera efectiva.

Blocs

Què és un Bloc?

  • Un bloc és un fragment de codi anònim que es pot passar a mètodes.
  • Els blocs no són objectes, però es poden convertir en objectes utilitzant Proc.

Sintaxi

Els blocs es poden definir utilitzant do...end o claus {...}.

# Utilitzant do...end
[1, 2, 3].each do |number|
  puts number
end

# Utilitzant claus
[1, 2, 3].each { |number| puts number }

Cedir el control a un Bloc

  • Els mètodes poden cedir el control a un bloc utilitzant la paraula clau yield.
def greet
  yield "Hello"
end

greet { |greeting| puts greeting }  # Sortida: Hello

Paràmetres del Bloc

  • Els blocs poden tenir paràmetres, que es defineixen entre barres verticals | |.
def greet
  yield "Hello", "World"
end

greet { |greeting, name| puts "#{greeting}, #{name}!" }  # Sortida: Hello, World!

Procs

Què és un Proc?

  • Un Proc és un objecte que encapsula un bloc de codi, que es pot emmagatzemar en una variable, passar a mètodes o cridar.

Crear un Proc

  • Utilitza Proc.new o el mètode proc per crear un Proc.
# Utilitzant Proc.new
my_proc = Proc.new { |name| puts "Hello, #{name}!" }

# Utilitzant proc
my_proc = proc { |name| puts "Hello, #{name}!" }

Cridar un Proc

  • Crida un Proc utilitzant el mètode call.
my_proc.call("Alice")  # Sortida: Hello, Alice!

Proc vs Bloc

| Característica | Bloc | Proc | |-------------------|--------------------------------|--------------------------------| | Definició | Anònim, no és un objecte | Objecte, es pot emmagatzemar en una variable | | Passar a Mètodes | Implícitament | Explícitament | | Múltiples Blocs | No | Sí |

Lambdas

Què és una Lambda?

  • Una lambda és un tipus especial de Proc amb una comprovació d'arguments més estricta i un comportament diferent per a return.

Crear una Lambda

  • Utilitza el mètode lambda o la sintaxi ->.
# Utilitzant el mètode lambda
my_lambda = lambda { |name| puts "Hello, #{name}!" }

# Utilitzant la sintaxi ->
my_lambda = ->(name) { puts "Hello, #{name}!" }

Cridar una Lambda

  • Crida una lambda utilitzant el mètode call.
my_lambda.call("Bob")  # Sortida: Hello, Bob!

Lambda vs Proc

| Característica | Proc | Lambda | |-------------------|--------------------------------|--------------------------------| | Comprovació d'Arguments | Laxa | Estricta | | Comportament de Return | Retorna del mètode que l'envolta | Retorna de la lambda mateixa |

Conclusió

Entendre els blocs, procs i lambdas és essencial per escriure codi Ruby flexible i concís. Els blocs són útils per passar fragments de codi a mètodes, mentre que els procs i lambdas proporcionen més control i flexibilitat permetent emmagatzemar i reutilitzar codi. Les lambdas ofereixen una comprovació d'arguments més estricta i un comportament de retorn diferent en comparació amb els procs. Dominant aquests conceptes, podràs escriure programes Ruby més potents i mantenibles.
