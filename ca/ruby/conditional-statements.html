Introducció a les declaracions condicionals

Les declaracions condicionals són fonamentals en la programació, ja que permeten executar certs fragments de codi basats en condicions específiques. En Ruby, les declaracions condicionals ajuden a controlar el flux del teu programa prenent decisions.

Declaracions condicionals bàsiques

Declaració `if`

La declaració if és la forma més bàsica de declaració condicional en Ruby. Executa un bloc de codi si una condició especificada és certa.

if condition
  # codi a executar si la condició és certa
end

Exemple:

age = 18

if age >= 18
  puts "Ets un adult."
end

Declaració `else`

La declaració else es pot utilitzar juntament amb if per executar un bloc de codi si la condició és falsa.

if condition
  # codi a executar si la condició és certa
else
  # codi a executar si la condició és falsa
end

Exemple:

age = 16

if age >= 18
  puts "Ets un adult."
else
  puts "Ets menor d'edat."
end

Declaració `elsif`

La declaració elsif et permet comprovar múltiples condicions.

if condition1
  # codi a executar si la condició1 és certa
elsif condition2
  # codi a executar si la condició2 és certa
else
  # codi a executar si cap de les condicions anteriors és certa
end

Exemple:

age = 20

if age < 13
  puts "Ets un nen."
elsif age < 18
  puts "Ets un adolescent."
else
  puts "Ets un adult."
end

Declaracions condicionals avançades

Declaració `unless`

La declaració unless és l'oposat de if. Executa un bloc de codi si la condició és falsa.

unless condition
  # codi a executar si la condició és falsa
end

Exemple:

age = 15

unless age >= 18
  puts "No ets un adult."
end

Operador ternari

L'operador ternari és una forma abreujada de les declaracions if-else. És útil per a condicions simples.

condition ? true_expression : false_expression

Exemple:

age = 20
message = age >= 18 ? "Ets un adult." : "Ets menor d'edat."
puts message

Declaració `case`

La declaració case s'utilitza per comparar una variable amb múltiples valors. És similar a la declaració switch en altres llenguatges.

case variable
when value1
  # codi a executar si variable == value1
when value2
  # codi a executar si variable == value2
else
  # codi a executar si la variable no coincideix amb cap valor
end

Exemple:

day = "Saturday"

case day
when "Monday"
  puts "Inici de la setmana laboral."
when "Friday"
  puts "Final de la setmana laboral."
when "Saturday", "Sunday"
  puts "Cap de setmana!"
else
  puts "Dies entre setmana."
end

Combinació de condicions

Operadors lògics

Els operadors lògics es poden utilitzar per combinar múltiples condicions en una sola declaració if, elsif o unless.

  • && (i): Ambdues condicions han de ser certes.
  • || (o): Almenys una condició ha de ser certa.
  • ! (no): Inverteix la condició.

Exemple:

age = 25
has_id = true

if age >= 18 && has_id
  puts "Pots entrar al club."
else
  puts "No pots entrar al club."
end

Exercicis

  1. Escriu un programa en Ruby que comprovi si un número és positiu, negatiu o zero.
  2. Crea un script en Ruby que categoritzi el grup d'edat d'una persona (nen, adolescent, adult, ancià) basant-se en la seva edat.
  3. Implementa un programa en Ruby que utilitzi una declaració case per imprimir el dia de la setmana basant-se en un número d'entrada (1 per dilluns, 2 per dimarts, etc.).

Resum

Les declaracions condicionals en Ruby són essencials per controlar el flux del teu programa. Començant amb les declaracions bàsiques if, else i elsif, pots gestionar condicions simples. Per a escenaris més complexos, Ruby ofereix unless, operadors ternaris i declaracions case. La combinació de condicions amb operadors lògics permet un control encara més precís. Practica aquests conceptes per convertir-te en un expert en la gestió de la lògica del programa en Ruby.
