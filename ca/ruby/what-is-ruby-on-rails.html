Ruby on Rails, sovint anomenat simplement Rails, és un framework d'aplicacions web escrit en Ruby. Està dissenyat per fer que la programació d'aplicacions web sigui més fàcil fent suposicions sobre el que cada desenvolupador necessita per començar. Et permet escriure menys codi mentre aconsegueixes més que molts altres llenguatges i frameworks.

Conceptes Clau de Ruby on Rails

  • Arquitectura MVC: Rails segueix l'arquitectura Model-View-Controller (MVC), que separa l'aplicació en tres components interconnectats.

    • Model: Gestiona les dades i la lògica de negoci.
    • Vista: S'encarrega de la visualització de la informació.
    • Controlador: Gestiona l'entrada de l'usuari i interactua amb el model.

  • Convenció sobre Configuració: Rails fa suposicions sobre la millor manera de fer moltes coses, de manera que pots escriure menys codi.

  • Principi DRY: "Don't Repeat Yourself" (No et repeteixis) fomenta la reducció de la duplicació de codi.

  • Disseny RESTful: Rails fomenta l'ús de rutes i accions RESTful per a aplicacions web.

Per què Utilitzar Ruby on Rails?

  • Productivitat: Rails permet un desenvolupament i prototipat ràpid.
  • Comunitat i Ecosistema: Una comunitat gran i activa i un ric ecosistema de gemmes (biblioteques) que amplien la funcionalitat.
  • Escalabilitat: Adequat tant per a aplicacions petites com grans.

Estructura Bàsica d'una Aplicació Rails

Una aplicació Rails té una estructura de directoris específica. Aquí tens una vista simplificada:

myapp/
├── app/
│   ├── controllers/
│   ├── models/
│   ├── views/
│   └── ...
├── config/
├── db/
├── public/
└── ...
  • app/: Conté els components principals de l'aplicació.
    • controllers/: On es guarden els controladors.
    • models/: On es guarden els models.
    • views/: On es guarden les vistes.
  • config/: Fitxers de configuració per a l'aplicació.
  • db/: Fitxers relacionats amb la base de dades.
  • public/: Fitxers estàtics com imatges i fulls d'estil.

Exemple: Crear una Aplicació Rails Senzilla

Creem una aplicació Rails senzilla per entendre com funciona.

Pas 1: Instal·lar Rails

Primer, assegura't que tens Ruby instal·lat. Després, instal·la Rails utilitzant la següent comanda:

gem install rails

Pas 2: Crear una Nova Aplicació Rails

Crea una nova aplicació Rails anomenada blog:

rails new blog
cd blog

Pas 3: Generar un Scaffold

Genera un scaffold per a un recurs Post amb un títol i cos:

rails generate scaffold Post title:string body:text

Pas 4: Migrar la Base de Dades

Executa les migracions de la base de dades per crear les taules necessàries:

rails db:migrate

Pas 5: Iniciar el Servidor Rails

Inicia el servidor Rails per veure la teva aplicació en acció:

rails server

Visita http://localhost:3000/posts al teu navegador web per veure el recurs Post generat.

Conclusió

Ruby on Rails és un framework potent i productiu per construir aplicacions web. El seu èmfasi en la convenció sobre configuració, el principi DRY i el disseny RESTful el fan una elecció popular entre els desenvolupadors. Entenent l'estructura bàsica i els conceptes clau de Rails, pots començar ràpidament a construir aplicacions web robustes.
