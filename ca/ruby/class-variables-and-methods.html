Introducció

En Ruby, entendre les variables i mètodes de classe és crucial per dominar la programació orientada a objectes. Aquesta secció cobrirà els conceptes bàsics de les variables i mètodes de classe, el seu ús i com es diferencien de les variables i mètodes d'instància.

Variables de Classe

Les variables de classe es comparteixen entre totes les instàncies d'una classe. Es prefixen amb @@ i s'utilitzen per emmagatzemar informació que és comuna a tots els objectes d'una classe.

Conceptes Clau

  • Declaració de Variable de Classe: Les variables de classe es declaren dins del cos de la classe, prefixades amb @@.
  • Estat Compartit: Totes les instàncies d'una classe comparteixen la mateixa variable de classe.
  • Inicialització: Les variables de classe es poden inicialitzar directament dins del cos de la classe o dins dels mètodes de classe.

Exemple

class Car
  @@number_of_cars = 0

  def initialize(model)
    @model = model
    @@number_of_cars += 1
  end

  def self.total_cars
    @@number_of_cars
  end
end

car1 = Car.new("Toyota")
car2 = Car.new("Honda")

puts Car.total_cars # Output: 2

Explicació:

  • @@number_of_cars és una variable de classe.
  • S'incrementa cada vegada que es crea un nou objecte Car.
  • self.total_cars és un mètode de classe que retorna el valor de @@number_of_cars.

Mètodes de Classe

Els mètodes de classe són mètodes que es criden a la classe mateixa, en lloc de a les instàncies de la classe. Es defineixen utilitzant self. abans del nom del mètode.

Conceptes Clau

  • Declaració de Mètode de Classe: Utilitza self. abans del nom del mètode.
  • Accés a Variables de Classe: Els mètodes de classe poden accedir i modificar les variables de classe.
  • Funcions d'Utilitat: Sovint s'utilitzen per a funcions d'utilitat que estan relacionades amb la classe però no amb cap instància específica.

Exemple

class Calculator
  def self.add(a, b)
    a + b
  end

  def self.subtract(a, b)
    a - b
  end
end

puts Calculator.add(5, 3)      # Output: 8
puts Calculator.subtract(5, 3) # Output: 2

Explicació:

  • self.add i self.subtract són mètodes de classe.
  • Realitzen operacions que no depenen de dades específiques de la instància.

Comparació: Variables de Classe vs Variables d'Instància

| Aspecte | Variables de Classe (@@) | Variables d'Instància (@) | |-----------------------|----------------------------|-----------------------------| | Abast | Compartit entre totes les instàncies | Únic per a cada instància | | Inicialització | Dins del cos de la classe o mètodes de classe | Dins dels mètodes d'instància (p. ex., initialize) | | Accés | Accessible per mètodes de classe i d'instància | Accessible només per mètodes d'instància | | Cas d'Ús | Dades comunes per a totes les instàncies | Dades específiques per a cada instància |

Exercicis

Exercici 1: Variable de Classe

Crea una classe Library que faci un seguiment del nombre total de llibres a través de totes les instàncies.

class Library
  @@total_books = 0

  def initialize(books)
    @books = books
    @@total_books += books
  end

  def self.total_books
    @@total_books
  end
end

# Crea instàncies i prova la variable de classe
library1 = Library.new(100)
library2 = Library.new(200)

puts Library.total_books # Expected output: 300

Exercici 2: Mètode de Classe

Crea una classe TemperatureConverter amb mètodes de classe per convertir temperatures entre Celsius i Fahrenheit.

class TemperatureConverter
  def self.celsius_to_fahrenheit(celsius)
    (celsius * 9.0 / 5) + 32
  end

  def self.fahrenheit_to_celsius(fahrenheit)
    (fahrenheit - 32) * 5.0 / 9
  end
end

# Prova els mètodes de classe
puts TemperatureConverter.celsius_to_fahrenheit(0)    # Expected output: 32
puts TemperatureConverter.fahrenheit_to_celsius(32)   # Expected output: 0

Conclusió

Les variables i mètodes de classe són eines poderoses en Ruby que et permeten gestionar dades i funcionalitats compartides a nivell de classe. Entendre quan i com utilitzar-les t'ajudarà a escriure codi més eficient i organitzat. Recorda:

  • Utilitza variables de classe per a dades compartides entre totes les instàncies.
  • Utilitza mètodes de classe per a funcionalitats que pertanyen a la classe mateixa en lloc de a cap instància específica.

En dominar aquests conceptes, estaràs ben equipat per manejar tasques de programació orientada a objectes més complexes en Ruby.
