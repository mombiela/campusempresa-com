Introducció als Símbols

Els símbols són una part fonamental de Ruby, sovint utilitzats per a identificadors, claus en hashes, i més. Són immutables, el que significa que romanen constants al llarg del programa, cosa que els fa eficients en termes de memòria i més ràpids per a certes operacions en comparació amb les cadenes de text.

Què és un Símbol?

  • Un símbol és un identificador lleuger i immutable.
  • Els símbols es creen utilitzant un dos punts seguit d'un nom, per exemple, :my_symbol.
  • Són únics i reutilitzables, el que significa que el mateix símbol sempre farà referència al mateix objecte en memòria.
# Creant un símbol
:my_symbol

# Comprovant l'ID de l'objecte
puts :my_symbol.object_id
puts :my_symbol.object_id  # Mateix ID d'objecte que l'anterior

Símbols vs Cadenes de Text

  • Immutabilitat: Els símbols són immutables, mentre que les cadenes de text són mutables.
  • Ús de Memòria: Els símbols es guarden només una vegada en memòria, mentre que cada cadena de text es guarda per separat.
  • Rendiment: Els símbols poden ser més ràpids per a certes operacions, especialment quan s'utilitzen com a claus de hash.

| Característica | Símbol | Cadena de Text | |----------------|---------------------|---------------------| | Mutabilitat | Immutable | Mutable | | Ús de Memòria | Instància única | Múltiples instàncies| | Rendiment | Més ràpid per cerques| Més lent per cerques|

# Exemple de cadena de text
str1 = "hello"
str2 = "hello"
puts str1.object_id == str2.object_id  # fals

# Exemple de símbol
sym1 = :hello
sym2 = :hello
puts sym1.object_id == sym2.object_id  # cert

Ús dels Símbols

Símbols com a Claus de Hash

Els símbols es fan servir comunament com a claus en hashes degut a la seva immutabilitat i beneficis de rendiment.

# Utilitzant símbols com a claus de hash
person = {
  :name => "Alice",
  :age => 30,
  :city => "Wonderland"
}

# Accedint a valors
puts person[:name]  # Sortida: Alice

Símbols en Noms de Mètodes

Els símbols es poden utilitzar per referenciar noms de mètodes, cosa útil per a metaprogramació i crides dinàmiques a mètodes.

# Definint un mètode
def greet
  "Hello!"
end

# Utilitzant un símbol per cridar el mètode
method_name = :greet
puts send(method_name)  # Sortida: Hello!

Símbols en Ruby on Rails

En Ruby on Rails, els símbols s'utilitzen extensivament per a diverses finalitats, com ara definir rutes, paràmetres, i més.

# Exemple d'una ruta en Rails
get 'welcome/index', to: 'welcome#index'

Ús Avançat dels Símbols

Convertint entre Símbols i Cadenes de Text

Els símbols es poden convertir a cadenes de text i viceversa utilitzant els mètodes to_s i to_sym.

# Convertint símbol a cadena de text
sym = :ruby
str = sym.to_s
puts str  # Sortida: "ruby"

# Convertint cadena de text a símbol
str = "ruby"
sym = str.to_sym
puts sym  # Sortida: :ruby

Símbols Dinàmics

Els símbols es poden crear dinàmicament utilitzant interpolació.

# Creació dinàmica de símbols
dynamic_sym = :"symbol_#{1 + 1}"
puts dynamic_sym  # Sortida: :symbol_2

Conclusió

Els símbols són una característica poderosa i eficient en Ruby, proporcionant beneficis en termes de rendiment i ús de memòria, especialment quan s'utilitzen com a identificadors o claus de hash. Entendre les diferències entre símbols i cadenes de text, i saber quan utilitzar cadascun, és crucial per escriure codi Ruby optimitzat. A mesura que avancis en Ruby, trobaràs que els símbols són una part indispensable del teu conjunt d'eines de programació.
