Introducció

Ruby és un llenguatge de programació dinàmic i de codi obert amb un enfocament en la simplicitat i la productivitat. La seva sintaxi elegant és natural de llegir i fàcil d'escriure. En aquesta secció, cobrirem la sintaxi bàsica de Ruby, que és essencial per escriure i entendre el codi Ruby.

Comentaris

  • Els comentaris d'una sola línia comencen amb #.
  • Els comentaris de diverses línies estan tancats dins de =begin i =end.
# Aquest és un comentari d'una sola línia

=begin
Aquest és un
comentari de diverses línies
=end

Variables

  • Les variables en Ruby són de tipus dinàmic.
  • Els noms de les variables han de començar amb una lletra minúscula o un guió baix.
name = "Alice"
age = 30
is_student = true

Tipus de dades

Ruby admet diversos tipus de dades bàsics:

  • Nombres (Enter, Float)
  • Cadenes de text
  • Símbols
  • Arrays
  • Hashes
  • Booleans
# Enter
num = 42

# Float
pi = 3.14

# Cadena de text
greeting = "Hola, món!"

# Símbol
status = :active

# Array
numbers = [1, 2, 3, 4, 5]

# Hash
person = { name: "Alice", age: 30 }

# Booleà
is_ruby_fun = true

Operadors

Operadors aritmètics

  • + Suma
  • - Resta
  • * Multiplicació
  • / Divisió
  • % Mòdul
sum = 5 + 3       # 8
difference = 5 - 3 # 2
product = 5 * 3    # 15
quotient = 5 / 3   # 1
remainder = 5 % 3  # 2

Operadors de comparació

  • == Igual a
  • != Diferent de
  • > Més gran que
  • < Menys que
  • >= Més gran o igual que
  • <= Menys o igual que
a = 5
b = 3

puts a == b  # fals
puts a != b  # cert
puts a > b   # cert
puts a < b   # fals
puts a >= b  # cert
puts a <= b  # fals

Operadors lògics

  • && AND lògic
  • || OR lògic
  • ! NOT lògic
x = true
y = false

puts x && y  # fals
puts x || y  # cert
puts !x      # fals

Estructures de control

Declaracions condicionals

  • if, elsif, else
age = 18

if age < 18
  puts "Ets menor d'edat."
elsif age == 18
  puts "Acabes de convertir-te en adult."
else
  puts "Ets adult."
end

Bucles

  • Bucle while
  • Bucle for
  • Mètode each per iterar sobre col·leccions
# bucle while
i = 0
while i < 5
  puts i
  i += 1
end

# bucle for
for i in 0..4
  puts i
end

# mètode each
[0, 1, 2, 3, 4].each do |i|
  puts i
end

Mètodes

  • Defineix mètodes utilitzant la paraula clau def.
  • Els mètodes poden prendre arguments i retornar valors.
def greet(name)
  return "Hola, #{name}!"
end

puts greet("Alice")  # Sortida: Hola, Alice!

Conclusió

En aquesta secció, hem cobert la sintaxi bàsica de Ruby, incloent comentaris, variables, tipus de dades, operadors, estructures de control i mètodes. Entendre aquests fonaments és crucial per escriure i comprendre el codi Ruby. A mesura que avancis en el curs, aquestes bases serviran com a fonament per a temes més avançats.
