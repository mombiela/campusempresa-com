Introducció

Els operadors aritmètics són fonamentals en qualsevol llenguatge de programació, incloent Ruby. Et permeten realitzar operacions matemàtiques bàsiques com la suma, resta, multiplicació, divisió i més. Entendre aquests operadors és crucial per manipular dades numèriques i realitzar càlculs en els teus programes.

Operadors aritmètics bàsics

Ruby admet els següents operadors aritmètics bàsics:

  • Suma (+)
  • Resta (-)
  • Multiplicació (*)
  • Divisió (/)
  • Mòdul (%)
  • Exponenciació (**)

Suma (`+`)

L'operador de suma suma dos números.

a = 5
b = 3
sum = a + b
puts sum  # Output: 8

Resta (`-`)

L'operador de resta resta el segon número del primer.

a = 5
b = 3
difference = a - b
puts difference  # Output: 2

Multiplicació (`*`)

L'operador de multiplicació multiplica dos números.

a = 5
b = 3
product = a * b
puts product  # Output: 15

Divisió (`/`)

L'operador de divisió divideix el primer número pel segon. Tingues en compte que en Ruby, dividir dos enters donarà com a resultat un enter.

a = 5
b = 3
quotient = a / b
puts quotient  # Output: 1

Per obtenir un resultat de punt flotant, almenys un dels números ha de ser un float.

a = 5.0
b = 3
quotient = a / b
puts quotient  # Output: 1.6666666666666667

Mòdul (`%`)

L'operador de mòdul retorna el residu de la divisió del primer número pel segon.

a = 5
b = 3
remainder = a % b
puts remainder  # Output: 2

Exponenciació (`**`)

L'operador d'exponenciació eleva el primer número a la potència del segon número.

a = 5
b = 3
power = a ** b
puts power  # Output: 125

Ús avançat

Combinació d'operadors

Pots combinar operadors aritmètics per realitzar càlculs més complexos.

a = 5
b = 3
c = 2
result = (a + b) * c - (a / b)
puts result  # Output: 15

Precedència d'operadors

Ruby segueix les regles estàndard de precedència matemàtica (PEMDAS/BODMAS). Els parèntesis es poden utilitzar per definir explícitament l'ordre de les operacions.

a = 5
b = 3
c = 2
result = a + b * c  # Multiplicació abans de la suma
puts result  # Output: 11

result = (a + b) * c  # Els parèntesis canvien l'ordre
puts result  # Output: 16

Ús d'operadors aritmètics amb variables

Els operadors aritmètics es poden utilitzar directament amb variables per actualitzar els seus valors.

a = 5
a += 3  # Equivalent a a = a + 3
puts a  # Output: 8

a -= 2  # Equivalent a a = a - 2
puts a  # Output: 6

a *= 2  # Equivalent a a = a * 2
puts a  # Output: 12

a /= 4  # Equivalent a a = a / 4
puts a  # Output: 3

a %= 2  # Equivalent a a = a % 2
puts a  # Output: 1

a **= 3  # Equivalent a a = a ** 3
puts a  # Output: 1

Conclusió

Els operadors aritmètics són eines essencials en Ruby per realitzar operacions matemàtiques. Des de la suma i resta bàsiques fins a expressions més complexes que involucren múltiples operadors, entendre com utilitzar aquests operadors de manera efectiva és clau per escriure codi Ruby eficient i precís. Practica utilitzant aquests operadors en diferents escenaris per sentir-te més còmode amb el seu comportament i precedència.
