Introducció a l'Àmbit de les Variables

L'àmbit de les variables es refereix al context dins d'un programa on una variable és accessible. Entendre l'àmbit de les variables és crucial per escriure codi eficient i lliure d'errors. En Ruby, l'àmbit de les variables es determina per on es declara la variable i el tipus de variable.

Tipus de Variables en Ruby

Ruby té diversos tipus de variables, cadascun amb el seu propi àmbit:

  • Variables Locals: Accessibles només dins del bloc de codi on es declaren.
  • Variables d'Instància: Accessibles dins de la instància d'una classe.
  • Variables de Classe: Compartides entre totes les instàncies d'una classe.
  • Variables Globals: Accessibles des de qualsevol lloc del programa Ruby.
  • Constants: Accessibles a tota l'aplicació però no s'haurien de canviar un cop assignades.

Variables Locals

Les variables locals es defineixen dins d'un mètode, bloc o bucle i només són accessibles dins d'aquest context.

def exemple_metode
  variable_local = "Sóc una variable local"
  puts variable_local
end

exemple_metode
# Sortida: Sóc una variable local

puts variable_local
# Error: variable local o mètode `variable_local' no definit

Punts Clau:

  • Les variables locals comencen amb una lletra minúscula o un guió baix.
  • No són accessibles fora del mètode o bloc on es defineixen.

Variables d'Instància

Les variables d'instància es prefixen amb @ i són accessibles dins de la instància d'una classe.

class ExempleClasse
  def initialize(nom)
    @nom = nom
  end

  def mostrar_nom
    puts @nom
  end
end

exemple = ExempleClasse.new("Ruby")
exemple.mostrar_nom
# Sortida: Ruby

Punts Clau:

  • Les variables d'instància s'utilitzen per emmagatzemar dades específiques de l'objecte.
  • Són accessibles a través dels mètodes dins de la mateixa instància.

Variables de Classe

Les variables de classe es prefixen amb @@ i es comparteixen entre totes les instàncies d'una classe.

class ExempleClasse
  @@variable_classe = 0

  def initialize
    @@variable_classe += 1
  end

  def self.variable_classe
    @@variable_classe
  end
end

exemple1 = ExempleClasse.new
exemple2 = ExempleClasse.new

puts ExempleClasse.variable_classe
# Sortida: 2

Punts Clau:

  • Les variables de classe emmagatzemen dades que es comparteixen entre totes les instàncies d'una classe.
  • Poden ser accedides i modificades per qualsevol instància de la classe.

Variables Globals

Les variables globals es prefixen amb $ i són accessibles des de qualsevol lloc del programa Ruby.

$variable_global = "Sóc una variable global"

def mostrar_variable_global
  puts $variable_global
end

mostrar_variable_global
# Sortida: Sóc una variable global

$variable_global = "He estat canviada"
mostrar_variable_global
# Sortida: He estat canviada

Punts Clau:

  • Les variables globals poden ser accedides des de qualsevol part del programa.
  • S'haurien d'utilitzar amb moderació ja que poden dificultar la depuració.

Constants

Les constants es defineixen amb una lletra majúscula i es suposa que han de romandre inalterades.

CONSTANTE_VAR = "Sóc una constant"

def mostrar_constant
  puts CONSTANTE_VAR
end

mostrar_constant
# Sortida: Sóc una constant

# Intentar canviar una constant generarà un avís
CONSTANTE_VAR = "He estat canviada"
# Avís: constant CONSTANTE_VAR ja inicialitzada

Punts Clau:

  • Les constants són accessibles a tota l'aplicació.
  • No s'haurien de reasignar un cop establertes.

Resum

Entendre l'àmbit de les variables és essencial per gestionar dades i evitar errors en programes Ruby. Aquí teniu un resum ràpid:

  • Variables Locals: Limitades al mètode o bloc on es defineixen.
  • Variables d'Instància: Accessibles dins de la instància d'una classe.
  • Variables de Classe: Compartides entre totes les instàncies d'una classe.
  • Variables Globals: Accessibles des de qualsevol lloc del programa.
  • Constants: Han de romandre inalterades i són accessibles a tota l'aplicació.

Dominant l'àmbit de les variables, podeu escriure codi Ruby més eficient i mantenible.
