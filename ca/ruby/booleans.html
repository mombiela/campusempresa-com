Introducció als Booleans

Els booleans són un tipus de dada fonamental en Ruby, representant valors veritables o falsos. Són essencials per prendre decisions en el teu codi a través de declaracions condicionals i bucles.

  • True: Representa un valor veritable.
  • False: Representa un valor fals.

En Ruby, els booleans són instàncies de TrueClass i FalseClass.

Literals Booleans

En Ruby, els literals booleans són true i false.

is_ruby_fun = true
is_sky_green = false

puts is_ruby_fun  # Output: true
puts is_sky_green # Output: false

Expressions Booleans

Les expressions booleans s'avaluen com true o false. Sovint impliquen operadors de comparació.

Operadors de Comparació

  • == : Igual a
  • != : No igual a
  • > : Més gran que
  • < : Menys que
  • >= : Més gran o igual a
  • <= : Menys o igual a
a = 10
b = 20

puts a == b  # Output: false
puts a != b  # Output: true
puts a > b   # Output: false
puts a < b   # Output: true
puts a >= 10 # Output: true
puts b <= 15 # Output: false

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleans.

  • && : AND lògic
  • || : OR lògic
  • ! : NOT lògic
x = true
y = false

puts x && y  # Output: false
puts x || y  # Output: true
puts !x      # Output: false
puts !y      # Output: true

Declaracions condicionals

Les declaracions condicionals utilitzen expressions booleans per controlar el flux del programa.

Declaració if

age = 18

if age >= 18
  puts "Ets un adult."
else
  puts "Ets un menor."
end

Declaració unless

La declaració unless és l'oposat de if.

age = 15

unless age >= 18
  puts "Ets un menor."
else
  puts "Ets un adult."
end

Operador ternari

L'operador ternari és una forma abreujada de if-else.

age = 20
message = age >= 18 ? "Ets un adult." : "Ets un menor."
puts message  # Output: Ets un adult.

Mètodes Booleans

Ruby proporciona diversos mètodes que retornen valors booleans.

.nil?

Comprova si un objecte és nil.

name = nil
puts name.nil?  # Output: true

name = "Alice"
puts name.nil?  # Output: false

.empty?

Comprova si una cadena o una matriu està buida.

str = ""
puts str.empty?  # Output: true

arr = []
puts arr.empty?  # Output: true

.include?

Comprova si una cadena o una matriu inclou un element específic.

str = "Hola, món!"
puts str.include?("món")  # Output: true

arr = [1, 2, 3, 4, 5]
puts arr.include?(3)  # Output: true

Resum

Els booleans són una part crucial de la programació en Ruby, permetent la presa de decisions i el control del flux. Entendre els literals booleans, les expressions i els operadors lògics és essencial per escriure codi Ruby efectiu. Les declaracions condicionals com if, unless i l'operador ternari et permeten executar codi basat en condicions booleanes. A més, Ruby proporciona diversos mètodes que retornen valors booleans, ajudant-te a escriure codi més expressiu i llegible.
