Introducció

En Ruby, els mètodes poden prendre arguments per proporcionar dades d'entrada per a les seves operacions. Entendre com utilitzar els arguments de mètode de manera efectiva és crucial per escriure codi flexible i reutilitzable. Aquesta secció cobrirà els conceptes bàsics dels arguments de mètode, incloent diferents tipus d'arguments i com utilitzar-los.

Arguments Bàsics de Mètode

Definint Mètodes amb Arguments

Per definir un mètode que prengui arguments, has d'especificar els noms dels arguments entre parèntesis després del nom del mètode.

def greet(name)
  puts "Hello, #{name}!"
end

greet("Alice")  # Output: Hello, Alice!
  • Concepte Clau: El name dins dels parèntesis és un paràmetre, i "Alice" és un argument passat al mètode.

Arguments Múltiples

Pots definir mètodes que prenguin múltiples arguments separant-los amb comes.

def add(a, b)
  a + b
end

puts add(3, 4)  # Output: 7
  • Concepte Clau: a i b són paràmetres, i 3 i 4 són arguments passats al mètode.

Arguments per Defecte

Ruby permet establir valors per defecte per als arguments dels mètodes. Si no es proporciona un argument, s'utilitza el valor per defecte.

def greet(name = "World")
  puts "Hello, #{name}!"
end

greet()         # Output: Hello, World!
greet("Alice")  # Output: Hello, Alice!
  • Concepte Clau: name = "World" estableix un valor per defecte per al paràmetre name.

Arguments per Paraula Clau

Els arguments per paraula clau són una manera de definir mètodes amb paràmetres anomenats, fent el codi més llegible i flexible.

def print_address(street:, city:, zip:)
  puts "#{street}, #{city}, #{zip}"
end

print_address(street: "123 Main St", city: "Springfield", zip: "12345")
  • Concepte Clau: street:, city:, i zip: són arguments per paraula clau, i s'han de proporcionar quan es crida el mètode.

Arguments per Paraula Clau amb Valor per Defecte

També pots establir valors per defecte per als arguments per paraula clau.

def print_address(street:, city:, zip: "00000")
  puts "#{street}, #{city}, #{zip}"
end

print_address(street: "123 Main St", city: "Springfield")  # Output: 123 Main St, Springfield, 00000
  • Concepte Clau: zip: "00000" estableix un valor per defecte per a l'argument per paraula clau zip.

Arguments de Longitud Variable

Ruby permet que els mètodes acceptin un nombre variable d'arguments utilitzant l'operador splat (*).

def sum(*numbers)
  numbers.reduce(0) { |total, num| total + num }
end

puts sum(1, 2, 3)  # Output: 6
puts sum(4, 5)     # Output: 9
  • Concepte Clau: *numbers recull tots els arguments en un array anomenat numbers.

Arguments de Bloc

Els mètodes en Ruby també poden prendre blocs com arguments, que són trossos de codi que es poden executar dins del mètode.

def repeat(n)
  n.times { yield }
end

repeat(3) { puts "Hello!" }
# Output:
# Hello!
# Hello!
# Hello!
  • Concepte Clau: yield s'utilitza per cridar el bloc passat al mètode.

Conclusió

Entendre els arguments de mètode en Ruby és essencial per escriure codi clar i flexible. Aquesta secció ha cobert:

  • Arguments bàsics de mètode i arguments múltiples.
  • Arguments per defecte i arguments per paraula clau.
  • Arguments de longitud variable utilitzant l'operador splat.
  • Arguments de bloc per passar trossos de codi.

En dominar aquests conceptes, pots crear mètodes més dinàmics i reutilitzables en els teus programes Ruby. Practica utilitzant aquests diferents tipus d'arguments per esdevenir proficient en escriure mètodes versàtils en Ruby.
