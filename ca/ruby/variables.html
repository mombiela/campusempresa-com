Introducció a les Variables

Les variables són fonamentals en qualsevol llenguatge de programació, i Ruby no és una excepció. Et permeten emmagatzemar i manipular dades al llarg del teu programa. En aquesta secció, cobrirem els conceptes bàsics de les variables en Ruby, incloent com crear-les, diferents tipus de variables i millors pràctiques.

Creació de Variables

En Ruby, crees una variable simplement assignant un valor a un nom. Ruby és dinàmicament tipat, el que significa que no necessites declarar el tipus de la variable.

name = "Alice"
age = 30
is_student = true
  • name és una variable que emmagatzema una cadena.
  • age és una variable que emmagatzema un enter.
  • is_student és una variable que emmagatzema un booleà.

Tipus de Variables

Ruby té diversos tipus de variables, cadascun amb el seu propi abast i convencions de nomenclatura.

Variables Locals

Les variables locals són el tipus de variable més comú. Només són accessibles dins del context on es defineixen, com dins d'un mètode o un bloc.

def greet
  greeting = "Hello, World!"
  puts greeting
end

greet
# Sortida: Hello, World!
  • Les variables locals comencen amb una lletra minúscula o un guió baix (_).

Variables d'Instància

Les variables d'instància s'utilitzen en programació orientada a objectes. Són accessibles dins de la instància d'una classe.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_info
    puts "Name: #{@name}, Age: #{@age}"
  end
end

person = Person.new("Alice", 30)
person.display_info
# Sortida: Name: Alice, Age: 30
  • Les variables d'instància comencen amb un símbol @.

Variables de Classe

Les variables de classe es comparteixen entre totes les instàncies d'una classe. S'utilitzen quan vols emmagatzemar informació que és comuna a tots els objectes d'una classe.

class Person
  @@species = "Homo sapiens"

  def self.species
    @@species
  end
end

puts Person.species
# Sortida: Homo sapiens
  • Les variables de classe comencen amb @@.

Variables Globals

Les variables globals són accessibles des de qualsevol lloc del programa Ruby. S'han d'utilitzar amb moderació ja que poden dificultar la depuració.

$global_var = "I am a global variable"

def display_global
  puts $global_var
end

display_global
# Sortida: I am a global variable
  • Les variables globals comencen amb un símbol $.

Constants

Les constants són variables que, un cop assignades, no haurien de canviar. S'utilitzen per emmagatzemar valors que romanen constants al llarg del programa.

PI = 3.14159
puts PI
# Sortida: 3.14159
  • Les constants comencen amb una lletra majúscula.

Abast de les Variables

L'abast d'una variable determina on pot ser accessible dins del programa. Entendre l'abast és crucial per escriure codi net i mantenible.

  • Abast Local: Variables definides dins d'un mètode o bloc.
  • Abast d'Instància: Variables accessibles dins d'una instància d'una classe.
  • Abast de Classe: Variables compartides entre totes les instàncies d'una classe.
  • Abast Global: Variables accessibles des de qualsevol lloc del programa.

Millors Pràctiques

  • Utilitza noms de variables significatius per fer el teu codi més llegible.
  • Evita utilitzar variables globals tret que sigui absolutament necessari.
  • Segueix les convencions de nomenclatura de Ruby per als diferents tipus de variables.
  • Mantingues l'abast de les variables tan limitat com sigui possible per evitar efectes secundaris no desitjats.

Conclusió

Les variables són un concepte clau en Ruby, permetent-te emmagatzemar i manipular dades de manera eficient. Entenent els diferents tipus de variables i els seus abasts, pots escriure codi més efectiu i mantenible. Recorda seguir les millors pràctiques per assegurar que el teu codi sigui net i fàcil d'entendre.
