Els Enumerables són una part fonamental de Ruby, proporcionant un conjunt poderós de mètodes per recórrer, cercar, ordenar i manipular col·leccions. Aquest tema cobrirà els conceptes essencials dels Enumerables, des dels conceptes bàsics fins a l'ús avançat.

Introducció als Enumerables

  • Definició: Un Enumerable és un mòdul que proporciona un conjunt de mètodes per recórrer, cercar, ordenar i manipular col·leccions.
  • Classes Comunes: Arrays, Hashes i Ranges són classes comunes que inclouen el mòdul Enumerable.

Mètodes Bàsics

each

El mètode each és la manera més bàsica de recórrer una col·lecció.

[1, 2, 3].each do |number|
  puts number
end
  • Explicació: Aquest codi recorre l'array [1, 2, 3] i imprimeix cada número.

map

El mètode map crea un nou array que conté els valors retornats pel bloc.

squares = [1, 2, 3].map do |number|
  number * 2
end
puts squares.inspect
  • Explicació: Aquest codi multiplica cada número de l'array per 2 i retorna un nou array [2, 4, 6].

select

El mètode select retorna un nou array que conté tots els elements per als quals el bloc donat retorna un valor veritable.

evens = [1, 2, 3, 4].select do |number|
  number.even?
end
puts evens.inspect
  • Explicació: Aquest codi selecciona els números parells de l'array, resultant en [2, 4].

Mètodes Intermedis

`reduce`

El mètode reduce combina tots els elements d'un enumerable aplicant una operació binària, especificada per un bloc o un símbol.

sum = [1, 2, 3, 4].reduce(0) do |accumulator, number|
  accumulator + number
end
puts sum
  • Explicació: Aquest codi calcula la suma dels elements de l'array, resultant en 10.

`find`

El mètode find retorna el primer element per al qual el bloc retorna un valor veritable.

first_even = [1, 2, 3, 4].find do |number|
  number.even?
end
puts first_even
  • Explicació: Aquest codi troba el primer número parell de l'array, que és 2.

`all?` i `any?`

  • all?: Retorna veritable si el bloc mai retorna fals o nil per a tots els elements.
  • any?: Retorna veritable si el bloc retorna un valor diferent de fals o nil per a qualsevol element.
all_even = [2, 4, 6].all? { |number| number.even? }
any_even = [1, 3, 4].any? { |number| number.even? }

puts all_even  # true
puts any_even  # true

Mètodes Avançats

`group_by`

El mètode group_by agrupa la col·lecció pel resultat del bloc.

grouped = [1, 2, 3, 4, 5, 6].group_by do |number|
  number.even?
end
puts grouped.inspect
  • Explicació: Aquest codi agrupa els números en parells i senars, resultant en {false=>[1, 3, 5], true=>[2, 4, 6]}.

`partition`

El mètode partition divideix la col·lecció en dos arrays, el primer que conté els elements per als quals el bloc avalua com a veritable, el segon que conté la resta.

evens, odds = [1, 2, 3, 4, 5, 6].partition do |number|
  number.even?
end
puts evens.inspect  # [2, 4, 6]
puts odds.inspect   # [1, 3, 5]

`lazy`

El mètode lazy permet una enumeració mandrosa, que pot ser útil per manejar col·leccions grans o infinites.

infinite = (1..Float::INFINITY).lazy
first_ten = infinite.first(10)
puts first_ten.inspect
  • Explicació: Aquest codi crea un enumerador mandrós per a un rang infinit i recupera els primers 10 elements.

Conclusió

Els Enumerables a Ruby proporcionen un ric conjunt de mètodes per treballar amb col·leccions. Des de la iteració bàsica amb each fins a l'agrupació avançada amb group_by, entendre aquests mètodes pot millorar enormement la teva capacitat per escriure codi Ruby eficient i llegible. Practica utilitzant aquests mètodes per convertir-te en un expert en la manipulació de col·leccions a Ruby.
