Operadors d'assignació en Ruby
Precedència d'Operadors ►

Els operadors d'assignació s'utilitzen per assignar valors a variables. En Ruby, els operadors d'assignació no només inclouen l'operador bàsic = sinó també una varietat d'operadors compostos que realitzen una operació i una assignació en un sol pas.

Operador d'assignació bàsic

L'operador d'assignació més fonamental és l'operador =.

x = 10
y = "Hello, Ruby!"
  • x = 10 assigna el valor enter 10 a la variable x.
  • y = "Hello, Ruby!" assigna la cadena "Hello, Ruby!" a la variable y.

Operadors d'assignació compostos

Els operadors d'assignació compostos combinen una operació aritmètica amb l'assignació. Aquests operadors són una forma abreujada de realitzar una operació i després assignar el resultat a una variable.

Assignació d'addició (`+=`)

x = 5
x += 3  # Equivalent a x = x + 3
puts x  # Sortida: 8

Assignació de substracció (`-=`)

x = 5
x -= 2  # Equivalent a x = x - 2
puts x  # Sortida: 3

Assignació de multiplicació (`*=`)

x = 5
x *= 4  # Equivalent a x = x * 4
puts x  # Sortida: 20

Assignació de divisió (`/=`)

x = 20
x /= 4  # Equivalent a x = x / 4
puts x  # Sortida: 5

Assignació de mòdul (`%=`)

x = 20
x %= 3  # Equivalent a x = x % 3
puts x  # Sortida: 2

Assignació d'exponenciació (`**=`)

x = 2
x **= 3  # Equivalent a x = x ** 3
puts x  # Sortida: 8

Operadors d'assignació bit a bit

Ruby també admet operadors d'assignació bit a bit, que realitzen operacions bit a bit i després assignen el resultat a la variable.

Assignació AND bit a bit (`&=`)

x = 6  # Binari: 110
x &= 3  # Binari: 011, Resultat: 010 (2 en decimal)
puts x  # Sortida: 2

Assignació OR bit a bit (`|=`)

x = 6  # Binari: 110
x |= 3  # Binari: 011, Resultat: 111 (7 en decimal)
puts x  # Sortida: 7

Assignació XOR bit a bit (`^=`)

x = 6  # Binari: 110
x ^= 3  # Binari: 011, Resultat: 101 (5 en decimal)
puts x  # Sortida: 5

Assignació de desplaçament a l'esquerra bit a bit (`<<=`)

x = 3  # Binari: 11
x <<= 2  # Binari: 1100 (12 en decimal)
puts x  # Sortida: 12

Assignació de desplaçament a la dreta bit a bit (`>>=`)

x = 12  # Binari: 1100
x >>= 2  # Binari: 11 (3 en decimal)
puts x  # Sortida: 3

Comparació dels operadors d'assignació

| Operador | Descripció | Exemple | Equivalent a | |----------|----------------------------------|------------------|--------------------| | = | Assignació bàsica | x = 5 | x = 5 | | += | Assignació d'addició | x += 3 | x = x + 3 | | -= | Assignació de substracció | x -= 2 | x = x - 2 | | *= | Assignació de multiplicació | x *= 4 | x = x * 4 | | /= | Assignació de divisió | x /= 4 | x = x / 4 | | %= | Assignació de mòdul | x %= 3 | x = x % 3 | | **= | Assignació d'exponenciació | x **= 3 | x = x ** 3 | | &= | Assignació AND bit a bit | x &= 3 | x = x & 3 | | |= | Assignació OR bit a bit | x |= 3 | x = x | 3 | | ^= | Assignació XOR bit a bit | x ^= 3 | x = x ^ 3 | | <<= | Assignació de desplaçament a l'esquerra bit a bit | x <<= 2 | x = x << 2 | | >>= | Assignació de desplaçament a la dreta bit a bit | x >>= 2 | x = x >> 2 |

Conclusió

Els operadors d'assignació en Ruby són eines essencials per gestionar i manipular els valors de les variables de manera eficient. Entendre tant els operadors d'assignació bàsics com els compostos et permetrà escriure codi més concís i llegible. Practica l'ús d'aquests operadors en diferents escenaris per sentir-te còmode amb la seva funcionalitat i aplicacions.
