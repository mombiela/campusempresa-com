Introducció a Classes i Objectes

A Ruby, tot és un objecte. Els objectes són instàncies de classes, que defineixen el plànol per als objectes. Entendre les classes i els objectes és fonamental per dominar Ruby.

Conceptes Clau

  • Classe: Un plànol per crear objectes.
  • Objecte: Una instància d'una classe.
  • Variables d'Instància: Variables que pertanyen a una instància d'una classe.
  • Mètodes: Funcions definides dins d'una classe per realitzar accions sobre els objectes.

Definició d'una Classe

Per definir una classe a Ruby, utilitza la paraula clau class seguida del nom de la classe. Els noms de les classes s'escriuen en CamelCase.

class Person
  # Cos de la classe aquí
end

Exemple: Creació d'una Classe Simple

Creem una classe Person simple amb variables d'instància i mètodes.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_details
    puts "Name: #{@name}, Age: #{@age}"
  end
end

# Creació d'un objecte de la classe Person
person1 = Person.new("Alice", 30)
person1.display_details  # Sortida: Name: Alice, Age: 30

Explicació

  • Mètode initialize: Un mètode especial que es crida quan es crea un nou objecte. Inicialitza les variables d'instància de l'objecte.
  • @name i @age: Variables d'instància que emmagatzemen l'estat de l'objecte.
  • Mètode display_details: Un mètode per mostrar els detalls de l'objecte.

Variables d'Instància i Mètodes

Les variables d'instància es prefixen amb @ i s'utilitzen per emmagatzemar l'estat d'un objecte. Els mètodes s'utilitzen per definir el comportament dels objectes.

Exemple: Afegir Més Mètodes

Afegim més mètodes a la nostra classe Person.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_details
    puts "Name: #{@name}, Age: #{@age}"
  end

  def birthday
    @age += 1
  end
end

person1 = Person.new("Alice", 30)
person1.display_details  # Sortida: Name: Alice, Age: 30
person1.birthday
person1.display_details  # Sortida: Name: Alice, Age: 31

Explicació

  • Mètode birthday: Augmenta l'edat de la persona en 1.

Mètodes i Variables de Classe

Els mètodes i variables de classe estan associats amb la classe mateixa en lloc de qualsevol instància de la classe.

Exemple: Mètodes i Variables de Classe

Afegim una variable de classe i un mètode de classe a la nostra classe Person.

class Person
  @@count = 0  # Variable de classe

  def initialize(name, age)
    @name = name
    @age = age
    @@count += 1
  end

  def display_details
    puts "Name: #{@name}, Age: #{@age}"
  end

  def self.total_count
    @@count
  end
end

person1 = Person.new("Alice", 30)
person2 = Person.new("Bob", 25)
puts Person.total_count  # Sortida: 2

Explicació

  • @@count: Una variable de classe que manté el compte del nombre d'objectes Person creats.
  • self.total_count: Un mètode de classe que retorna el valor de la variable de classe @@count.

Herència

La herència permet que una classe hereti característiques d'una altra classe.

Exemple: Herència

Creem una classe Student que hereta de la classe Person.

class Student < Person
  def initialize(name, age, grade)
    super(name, age)
    @grade = grade
  end

  def display_details
    super
    puts "Grade: #{@grade}"
  end
end

student1 = Student.new("Charlie", 20, "A")
student1.display_details  # Sortida: Name: Charlie, Age: 20, Grade: A

Explicació

  • < Person: Indica que Student hereta de Person.
  • super: Crida el mètode initialize de la classe pare per inicialitzar name i age.

Conclusió

Entendre les classes i els objectes és crucial per a la programació en Ruby. Les classes defineixen el plànol per als objectes, i els objectes són instàncies de classes. Les variables d'instància emmagatzemen l'estat d'un objecte, mentre que els mètodes defineixen el comportament. Els mètodes i variables de classe estan associats amb la classe mateixa. La herència permet que una classe hereti característiques d'una altra classe, promovent la reutilització i organització del codi.

Dominant aquests conceptes, pots crear programes Ruby complexos i eficients.
