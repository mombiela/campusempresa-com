Introducció

En aquesta secció, recorrerem el procés de creació d'una aplicació simple amb Rails. Rails és un potent framework per a aplicacions web escrit en Ruby, i segueix l'arquitectura Model-Vista-Controlador (MVC). Al final d'aquesta secció, tindràs una comprensió bàsica de com configurar un projecte Rails, crear models, vistes i controladors, i executar la teva aplicació.

Prerequisits

Abans de començar, assegura't de tenir instal·lat el següent:

  • Ruby (versió 2.5 o superior)
  • Rails (versió 6 o superior)
  • SQLite3 (o un altre sistema de base de dades)

Configurant una Nova Aplicació Rails

  1. Instal·lar Rails: Si encara no has instal·lat Rails, pots fer-ho executant:

    gem install rails

  2. Crear una Nova Aplicació Rails: Utilitza l'eina de línia de comandes de Rails per crear una nova aplicació:

    rails new my_first_app

    Aquesta comanda generarà un nou projecte Rails en un directori anomenat my_first_app.

  3. Navegar al Directori de la Teva Aplicació:

    cd my_first_app

  4. Iniciar el Servidor Rails: Per veure la teva aplicació en acció, inicia el servidor Rails:

    rails server

    Obre el teu navegador web i navega a http://localhost:3000. Hauries de veure la pàgina de benvinguda de Rails.

Creant un Model

Els models en Rails s'utilitzen per interactuar amb la base de dades. Creem un model simple per a un Post.

  1. Generar el Model:

    rails generate model Post title:string body:text

    Aquesta comanda crea un fitxer de migració, un fitxer de model, i fitxers de prova per al model Post.

  2. Executar la Migració:

    rails db:migrate

    Aquesta comanda aplica la migració, creant la taula posts a la base de dades.

Creant un Controlador

Els controladors gestionen les sol·licituds i respostes web. Creem un controlador per al nostre model Post.

  1. Generar el Controlador:

    rails generate controller Posts

    Aquesta comanda crea un fitxer de controlador, plantilles de vista, i fitxers de prova per al controlador Posts.

  2. Definir Accions en el Controlador: Obre app/controllers/posts_controller.rb i defineix accions per a index, show, new, i create:

    class PostsController < ApplicationController
  def index
    @posts = Post.all
  end

  def show
    @post = Post.find(params[:id])
  end

  def new
    @post = Post.new
  end

  def create
    @post = Post.new(post_params)
    if @post.save
      redirect_to @post
    else
      render 'new'
    end
  end

  private

  def post_params
    params.require(:post).permit(:title, :body)
  end
end

Creant Vistes

Les vistes s'utilitzen per mostrar les dades a l'usuari. Creem vistes per a les accions del nostre Post.

  1. Crear la Vista index: Crea un fitxer app/views/posts/index.html.erb:

    <h1>Tots els Posts</h1>
<ul>
  <% @posts.each do |post| %>
    <li>
      <%= link_to post.title, post %>
    </li>
  <% end %>
</ul>
<%= link_to 'Nou Post', new_post_path %>

  2. Crear la Vista show: Crea un fitxer app/views/posts/show.html.erb:

    <h1><%= @post.title %></h1>
<p><%= @post.body %></p>
<%= link_to 'Enrere', posts_path %>

  3. Crear la Vista new: Crea un fitxer app/views/posts/new.html.erb:

    <h1>Nou Post</h1>
<%= form_with model: @post, local: true do |form| %>
  <div>
    <%= form.label :title %>
    <%= form.text_field :title %>
  </div>
  <div>
    <%= form.label :body %>
    <%= form.text_area :body %>
  </div>
  <div>
    <%= form.submit %>
  </div>
<% end %>
<%= link_to 'Enrere', posts_path %>

Routing

Per connectar les accions del controlador amb les URL, necessitem definir rutes.

  1. Definir Rutes de Recursos: Obre config/routes.rb i afegeix: 
    Rails.application.routes.draw do
  resources :posts
  root 'posts#index'
end

Executant l'Aplicació

  1. Iniciar el Servidor Rails: 
    rails server
  2. Accedir a l'Aplicació: Obre el teu navegador web i navega a http://localhost:3000. Hauries de veure la llista de posts i poder crear nous posts.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per crear una aplicació simple amb Rails. Hem configurat un nou projecte Rails, creat un model, un controlador, i vistes, i hem definit rutes per connectar-ho tot. Això és només el començament; Rails ofereix moltes més característiques i eines per construir aplicacions web complexes. Continua explorant Rails per aprofundir la teva comprensió i construir aplicacions més sofisticades.
