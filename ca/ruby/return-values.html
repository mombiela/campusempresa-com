Introducció als Valors de Retorn

En Ruby, els mètodes poden retornar valors. Entendre com funcionen els valors de retorn és crucial per escriure codi Ruby efectiu i eficient. Aquesta secció cobrirà els conceptes bàsics dels valors de retorn, com utilitzar-los i algunes tècniques avançades.

Valors de Retorn Bàsics

En Ruby, cada mètode retorna el valor de l'última expressió avaluada. No cal utilitzar explícitament la paraula clau return, però es pot fer si es vol.

Retorn Implícit

Els mètodes Ruby retornen per defecte l'última expressió avaluada.

def add(a, b)
  a + b
end

result = add(2, 3)
puts result  # Sortida: 5

Retorn Explícit

Es pot utilitzar la paraula clau return per retornar un valor explícitament des d'un mètode.

def multiply(a, b)
  return a * b
end

result = multiply(4, 5)
puts result  # Sortida: 20

Valors de Retorn Múltiples

Els mètodes Ruby poden retornar múltiples valors utilitzant arrays.

def divide_and_remainder(a, b)
  quotient = a / b
  remainder = a % b
  return quotient, remainder
end

quotient, remainder = divide_and_remainder(10, 3)
puts "Quotient: #{quotient}, Remainder: #{remainder}"
# Sortida: Quotient: 3, Remainder: 1

Valors de Retorn Avançats

Retornant des de Blocs

Els blocs en Ruby també poden retornar valors. El valor de l'última expressió en el bloc és el valor de retorn.

result = [1, 2, 3].map do |num|
  num * 2
end

puts result.inspect  # Sortida: [2, 4, 6]

Utilitzant `return` en Blocs

Utilitzar return dins d'un bloc retornarà des del mètode que envolta el bloc, no només des del bloc mateix.

def test_method
  [1, 2, 3].each do |num|
    return num if num == 2
  end
end

puts test_method  # Sortida: 2

Retornant Procs i Lambdas

Els procs i lambdas són objectes que es poden emmagatzemar en variables i passar al voltant. També poden retornar valors.

my_proc = Proc.new { |x| x * 2 }
puts my_proc.call(3)  # Sortida: 6

my_lambda = ->(x) { x * 2 }
puts my_lambda.call(3)  # Sortida: 6

Comparació de Procs i Lambdas

| Característica | Proc | Lambda | |----------------|-------------------------------|-------------------------------| | Valor de Retorn| Retorna des del mètode que l'envolta | Retorna des de la lambda mateixa | | Comptatge d'Arguments | Ignora arguments extra | Genera un error per nombre incorrecte d'arguments |

def proc_example
  my_proc = Proc.new { return "Returning from proc" }
  my_proc.call
  return "This won't be reached"
end

puts proc_example  # Sortida: Returning from proc

def lambda_example
  my_lambda = -> { return "Returning from lambda" }
  my_lambda.call
  return "This will be reached"
end

puts lambda_example  # Sortida: This will be reached

Conclusió

Entendre els valors de retorn en Ruby és essencial per escriure codi clar i eficient. Tant si utilitzes retorns implícits o explícits, gestionant múltiples valors de retorn, o treballant amb blocs, procs i lambdas, dominar els valors de retorn millorarà significativament les teves habilitats de programació en Ruby. Practica els exemples proporcionats i experimenta amb els teus propis mètodes per obtenir una comprensió sòlida dels valors de retorn en Ruby.
