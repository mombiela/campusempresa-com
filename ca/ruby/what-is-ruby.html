Introducció a Ruby

Ruby és un llenguatge de programació dinàmic i de codi obert amb un enfocament en la simplicitat i la productivitat. Té una sintaxi elegant que és natural de llegir i fàcil d'escriure. Ruby va ser creat a mitjans dels anys 90 per Yukihiro "Matz" Matsumoto al Japó.

Característiques Clau de Ruby

  • Orientat a Objectes: Tot en Ruby és un objecte, incloent-hi els tipus de dades primitius.
  • Tipatge Dinàmic: Les variables en Ruby no estan lligades a un tipus de dada específic.
  • Recollida d'Escombraries: Ruby gestiona automàticament l'assignació i la desassignació de memòria.
  • Biblioteca Estàndard Rica: Ruby ve amb una biblioteca estàndard completa que facilita la realització d'una àmplia gamma de tasques.
  • Comunitat i Ecosistema: Ruby té una comunitat vibrant i un ric ecosistema de biblioteques i frameworks, com Ruby on Rails.

Sintaxi Bàsica

Vegem una mica de sintaxi bàsica de Ruby per familiaritzar-nos amb el llenguatge.

Exemple de Hello World

puts 'Hello, World!'
  • puts és un mètode que mostra text a la consola.
  • 'Hello, World!' és un literal de cadena.

Variables i Tipus de Dades

name = "Alice"  # Cadena
age = 30        # Enter
height = 5.5    # Flotant
is_student = true  # Booleà

puts name       # Mostra: Alice
puts age        # Mostra: 30
puts height     # Mostra: 5.5
puts is_student # Mostra: true
  • Les variables són de tipus dinàmic.
  • Els tipus de dades comuns inclouen Cadena, Enter, Flotant i Booleà.

Estructures de Control

Sentència If-Else

age = 18

if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end
  • if i else s'utilitzen per a l'execució condicional.
  • La condició és avaluada, i el bloc de codi corresponent és executat.

Bucles

Bucle While
count = 0

while count < 5
  puts "Count is #{count}"
  count += 1
end
  • El bucle while s'executa mentre la condició sigui certa.
  • #{} s'utilitza per a la interpolació de cadenes.
Bucle Each
numbers = [1, 2, 3, 4, 5]

numbers.each do |number|
  puts number
end
  • El mètode each itera sobre els elements d'una col·lecció, com un array.
  • |number| és una variable de bloc que pren el valor de cada element de l'array.

Programació Orientada a Objectes en Ruby

Ruby és un llenguatge purament orientat a objectes, el que significa que tot és un objecte. Creem una classe senzilla per il·lustrar això.

Definint una Classe

class Person
  attr_accessor :name, :age

  def initialize(name, age)
    @name = name
    @age = age
  end

  def introduce
    puts "Hello, my name is #{@name} and I am #{@age} years old."
  end
end

person = Person.new("Alice", 30)
person.introduce
  • La paraula clau class s'utilitza per definir una classe.
  • attr_accessor crea mètodes getter i setter per a variables d'instància.
  • El mètode initialize és el constructor.
  • @name i @age són variables d'instància.
  • introduce és un mètode d'instància.

Resum

Ruby és un llenguatge de programació versàtil i amigable per a principiants conegut per la seva simplicitat i productivitat. Admet la programació orientada a objectes i té una rica biblioteca estàndard. Entenent la seva sintaxi bàsica, estructures de control i característiques orientades a objectes, pots començar a construir aplicacions potents. A mesura que progressis, descobriràs característiques més avançades i biblioteques que fan de Ruby una eina poderosa per als desenvolupadors.
