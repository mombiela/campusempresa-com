Introducció

En Ruby, les variables d'instància i els mètodes són conceptes fonamentals que ajuden a definir l'estat i el comportament dels objectes. Aquesta secció cobrirà què són les variables d'instància i els mètodes, com utilitzar-los, i proporcionarà exemples i exercicis per consolidar la teva comprensió.

Variables d'instància

Què són les variables d'instància?

  • Les variables d'instància són variables que pertanyen a una instància d'una classe.
  • S'utilitzen per emmagatzemar l'estat d'un objecte.
  • Les variables d'instància es prefixen amb un símbol @.

Definició de variables d'instància

Les variables d'instància es defineixen típicament dins dels mètodes d'instància. S'inicialitzen quan es crea un objecte i poden ser accedides per qualsevol mètode d'instància dins del mateix objecte.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_info
    puts "Nom: #{@name}, Edat: #{@age}"
  end
end

person = Person.new("Alice", 30)
person.display_info  # Sortida: Nom: Alice, Edat: 30

Accés a les variables d'instància

Les variables d'instància poden ser accedides i modificades dins dels mètodes d'instància. No obstant això, no són directament accessibles des de fora de la classe.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def set_name(new_name)
    @name = new_name
  end

  def get_name
    @name
  end
end

person = Person.new("Alice", 30)
puts person.get_name  # Sortida: Alice
person.set_name("Bob")
puts person.get_name  # Sortida: Bob

Mètodes d'instància

Què són els mètodes d'instància?

  • Els mètodes d'instància són mètodes que pertanyen a una instància d'una classe.
  • Defineixen el comportament d'un objecte.
  • Els mètodes d'instància poden accedir i modificar les variables d'instància.

Definició de mètodes d'instància

Els mètodes d'instància es defineixen utilitzant la paraula clau def dins d'una classe. Poden realitzar operacions utilitzant variables d'instància i altres mètodes.

class Calculator
  def initialize
    @result = 0
  end

  def add(number)
    @result += number
  end

  def subtract(number)
    @result -= number
  end

  def result
    @result
  end
end

calc = Calculator.new
calc.add(10)
calc.subtract(3)
puts calc.result  # Sortida: 7

Exemple: Un compte bancari simple

Creem una classe de compte bancari simple per demostrar les variables d'instància i els mètodes.

class BankAccount
  def initialize(owner, balance = 0)
    @owner = owner
    @balance = balance
  end

  def deposit(amount)
    @balance += amount
    puts "Ingressat #{amount}. Nou saldo: #{@balance}"
  end

  def withdraw(amount)
    if amount > @balance
      puts "Fons insuficients!"
    else
      @balance -= amount
      puts "Retirat #{amount}. Nou saldo: #{@balance}"
    end
  end

  def display_balance
    puts "Propietari del compte: #{@owner}, Saldo: #{@balance}"
  end
end

account = BankAccount.new("John Doe", 100)
account.deposit(50)
account.withdraw(30)
account.display_balance

Exercicis

  1. Crea una classe Car:

    • Defineix variables d'instància per make, model, i year.
    • Crea mètodes d'instància per engegar el cotxe, aturar el cotxe, i mostrar els detalls del cotxe.

  2. Millora la classe Person:

    • Afegeix un mètode d'instància per actualitzar l'edat.
    • Afegeix un mètode d'instància per mostrar un missatge de benvinguda que inclogui el nom i l'edat de la persona.

  3. Construeix una classe Library:

    • Defineix variables d'instància per name i books (una matriu de títols de llibres).
    • Crea mètodes d'instància per afegir un llibre, eliminar un llibre, i llistar tots els llibres.

Conclusió

Les variables d'instància i els mètodes són crucials per definir l'estat i el comportament dels objectes en Ruby. Comprenent com utilitzar-los, pots crear classes més complexes i funcionals. Practica creant les teves pròpies classes i experimentant amb variables d'instància i mètodes per aprofundir la teva comprensió.
