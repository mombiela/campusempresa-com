Introducció als Hashes

Els hashes, també coneguts com a diccionaris, mapes o arrays associatius, són una estructura de dades fonamental en Ruby. Emmagatzemen parells clau-valor, permetent-te recuperar valors basats en les seves claus corresponents.

Conceptes Clau

  • Parells Clau-Valor: Cada element en un hash és un parell format per una clau única i un valor.
  • Col·lecció Desordenada: A diferència dels arrays, els hashes no mantenen cap ordre específic per als seus elements.
  • Claus i Valors Flexibles: Les claus i els valors poden ser de qualsevol tipus de dada.

Creació de Hashes

Pots crear hashes utilitzant diferents sintaxis en Ruby.

Utilitzant Notació Literal

# Creant un hash buit
empty_hash = {}

# Creant un hash amb parells clau-valor inicials
person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

Utilitzant la Classe `Hash`

# Creant un hash buit
empty_hash = Hash.new

# Creant un hash amb parells clau-valor inicials
person = Hash.new
person["name"] = "Alice"
person["age"] = 30
person["city"] = "New York"

Accés i Modificació de Hashes

Accés als Valors

Pots accedir als valors en un hash utilitzant les seves claus.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

# Accés als valors
puts person["name"]  # Sortida: Alice
puts person["age"]   # Sortida: 30

Modificació de Valors

Pots modificar valors existents o afegir nous parells clau-valor.

person["age"] = 31  # Modificant un valor existent
person["country"] = "USA"  # Afegint un nou parell clau-valor

puts person
# Sortida: {"name"=>"Alice", "age"=>31, "city"=>"New York", "country"=>"USA"}

Mètodes Comuns de Hash

Ruby proporciona diversos mètodes útils per treballar amb hashes.

`keys` i `values`

Recupera totes les claus o valors d'un hash.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

puts person.keys   # Sortida: ["name", "age", "city"]
puts person.values # Sortida: ["Alice", 30, "New York"]

`each`

Itera sobre cada parell clau-valor en un hash.

person.each do |key, value|
  puts "#{key}: #{value}"
end
# Sortida:
# name: Alice
# age: 30
# city: New York

`delete`

Elimina un parell clau-valor d'un hash.

person.delete("age")
puts person
# Sortida: {"name"=>"Alice", "city"=>"New York"}

`merge`

Combina dos hashes en un.

person = {
  "name" => "Alice",
  "age" => 30
}

address = {
  "city" => "New York",
  "country" => "USA"
}

person.merge!(address)
puts person
# Sortida: {"name"=>"Alice", "age"=>30, "city"=>"New York", "country"=>"USA"}

Tècniques Avançades de Hash

Valors per Defecte

Pots establir un valor per defecte per a un hash, que serà retornat quan s'accedeixi a una clau inexistent.

hash_with_default = Hash.new("Not Found")
puts hash_with_default["missing_key"]  # Sortida: Not Found

Símbols com a Claus

Utilitzar símbols com a claus pot ser més eficient i llegible.

person = {
  name: "Alice",
  age: 30,
  city: "New York"
}

puts person[:name]  # Sortida: Alice

Hashes Imbricats

Els hashes poden contenir altres hashes, permetent estructures de dades més complexes.

person = {
  name: "Alice",
  age: 30,
  address: {
    city: "New York",
    country: "USA"
  }
}

puts person[:address][:city]  # Sortida: New York

Conclusió

Els hashes són una estructura de dades versàtil i poderosa en Ruby, adequada per a una àmplia gamma d'aplicacions. Entendre com crear, accedir i manipular hashes millorarà significativament la teva capacitat per escriure codi Ruby eficient i efectiu. Practica utilitzant els mètodes i tècniques discutits en aquesta secció per convertir-te en un expert treballant amb hashes.
