Introducció als Mòduls

Els mòduls a Ruby són una manera d'agrupar mètodes, constants i variables de classe relacionades. Tenen dos propòsits principals: espais de noms i mixins.

  • Espais de noms: Evita conflictes de noms agrupant classes, mètodes i constants relacionades.
  • Mixins: Permet compartir mètodes entre múltiples classes.

Creació d'un Mòdul

Per definir un mòdul, utilitza la paraula clau module seguida del nom del mòdul. Els noms dels mòduls són constants i han de començar amb una lletra majúscula.

module Greetings
  def say_hello
    "Hello!"
  end

  def say_goodbye
    "Goodbye!"
  end
end

Ús de Mòduls per a Espais de Noms

Els espais de noms ajuden a organitzar el codi i evitar conflictes amb noms de mètodes i classes.

module MathTools
  PI = 3.14159

  def self.square(x)
    x * x
  end
end

puts MathTools::PI          # => 3.14159
puts MathTools.square(3)    # => 9

Mixins amb Mòduls

Els mixins permeten incloure mètodes de mòduls a les classes. Això es fa utilitzant les paraules clau include o extend.

Incloure Mòduls

Quan inclous un mòdul, els seus mètodes esdevenen mètodes d'instància de la classe.

module Greetings
  def say_hello
    "Hello!"
  end
end

class Person
  include Greetings
end

person = Person.new
puts person.say_hello  # => "Hello!"

Estendre Mòduls

Quan estens un mòdul, els seus mètodes esdevenen mètodes de classe.

module Greetings
  def say_hello
    "Hello!"
  end
end

class Person
  extend Greetings
end

puts Person.say_hello  # => "Hello!"

Característiques Avançades dels Mòduls

Constants de Mòdul

Els mòduls poden contenir constants que es poden accedir utilitzant l'operador de resolució d'espai de noms ::.

module Config
  MAX_USERS = 100
end

puts Config::MAX_USERS  # => 100

Mètodes de Mòdul

Els mòduls poden tenir mètodes definits amb self. per fer-los mètodes de mòdul.

module MathTools
  def self.square(x)
    x * x
  end
end

puts MathTools.square(4)  # => 16

Incloure Múltiples Mòduls

Una classe pot incloure múltiples mòduls, permetent-li heretar mètodes de tots els mòduls inclosos.

module Greetings
  def say_hello
    "Hello!"
  end
end

module Farewells
  def say_goodbye
    "Goodbye!"
  end
end

class Person
  include Greetings
  include Farewells
end

person = Person.new
puts person.say_hello   # => "Hello!"
puts person.say_goodbye # => "Goodbye!"

Exercicis Pràctics

  1. Crea un mòdul anomenat MathOperations amb mètodes per a sumar, restar, multiplicar i dividir. Inclou aquest mòdul en una classe i demostra el seu ús.
module MathOperations
  def add(a, b)
    a + b
  end

  def subtract(a, b)
    a - b
  end

  def multiply(a, b)
    a * b
  end

  def divide(a, b)
    a / b
  end
end

class Calculator
  include MathOperations
end

calc = Calculator.new
puts calc.add(5, 3)        # => 8
puts calc.subtract(5, 3)   # => 2
puts calc.multiply(5, 3)   # => 15
puts calc.divide(6, 3)     # => 2
  1. Defineix un mòdul Trackable amb un mètode track que imprimeixi "Tracking...". Inclou aquest mòdul en una classe Vehicle i crida el mètode track.
module Trackable
  def track
    "Tracking..."
  end
end

class Vehicle
  include Trackable
end

car = Vehicle.new
puts car.track  # => "Tracking..."

Conclusió

Els mòduls i mixins a Ruby proporcionen una manera poderosa d'organitzar i reutilitzar codi. Utilitzant mòduls per a espais de noms, pots evitar conflictes de noms i mantenir el teu codi net. Els mixins permeten compartir mètodes entre múltiples classes, promovent la reutilització de codi i reduint la duplicació. Entendre i utilitzar efectivament els mòduls i mixins és essencial per escriure aplicacions Ruby mantenibles i escalables.
