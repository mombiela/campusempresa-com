Introducció

Els operadors lògics són fonamentals en la programació, ja que et permeten prendre decisions basades en múltiples condicions. En Ruby, els operadors lògics s'utilitzen per combinar dues o més condicions per produir un únic resultat booleà. Aquesta secció cobrirà els operadors lògics bàsics en Ruby, el seu ús i exemples per ajudar-te a entendre com aplicar-los en el teu codi.

Operadors lògics bàsics

Ruby proporciona tres operadors lògics principals:

  • && (I)
  • || (O)
  • ! (NO)

Operador I (`&&`)

L'operador && retorna true si ambdós operants són certs. Si algun dels operants és fals, retorna false.

Exemple:

a = true
b = false

result = a && b
puts result # Sortida: false

Explicació:

  • a && b avalua a false perquè b és false.

Operador O (`||`)

L'operador || retorna true si almenys un dels operants és cert. Retorna false només si ambdós operants són falsos.

Exemple:

a = true
b = false

result = a || b
puts result # Sortida: true

Explicació:

  • a || b avalua a true perquè a és true.

Operador NO (`!`)

L'operador ! nega el valor booleà del seu operant. Si l'operant és true, retorna false, i viceversa.

Exemple:

a = true

result = !a
puts result # Sortida: false

Explicació:

  • !a avalua a false perquè a és true.

Combinació d'operadors lògics

Pots combinar múltiples operadors lògics per formar condicions complexes. Els parèntesis es poden utilitzar per controlar l'ordre d'avaluació.

Exemple:

a = true
b = false
c = true

result = (a && b) || c
puts result # Sortida: true

Explicació:

  • (a && b) avalua a false perquè b és false.
  • false || c avalua a true perquè c és true.

Avaluació de curt circuit

Ruby utilitza l'avaluació de curt circuit per als operadors lògics. Això significa que l'avaluació s'atura tan aviat com es determina el resultat.

Avaluació de curt circuit de l'operador I

Si el primer operant de && és false, el segon operant no s'avalua perquè el resultat ja és false.

Exemple:

a = false
b = true

result = a && (b = false)
puts result # Sortida: false
puts b      # Sortida: true

Explicació:

  • a && (b = false) s'atura després d'avaluar a perquè a és false.
  • b es manté true perquè (b = false) no s'executa.

Avaluació de curt circuit de l'operador O

Si el primer operant de || és true, el segon operant no s'avalua perquè el resultat ja és true.

Exemple:

a = true
b = false

result = a || (b = true)
puts result # Sortida: true
puts b      # Sortida: false

Explicació:

  • a || (b = true) s'atura després d'avaluar a perquè a és true.
  • b es manté false perquè (b = true) no s'executa.

Valors veritables i falsos

En Ruby, cada valor és o "veritable" o "fals". Només nil i false es consideren falsos; tot el demés és veritable.

Exemple:

a = nil
b = 0
c = ""

result = a || b || c
puts result # Sortida: 0

Explicació:

  • a és nil (fals), així que l'avaluació continua.
  • b és 0 (veritable), així que l'avaluació s'atura, i 0 es retorna.

Exercicis

  1. Escriu un script en Ruby que comprovi si un número està dins d'un rang específic (per exemple, entre 10 i 20) utilitzant operadors lògics.
  2. Crea un mètode en Ruby que prengui dos arguments booleans i retorni true si almenys un d'ells és true.
  3. Escriu un script en Ruby que utilitzi operadors lògics per determinar si una cadena no està buida i no conté només espais en blanc.

Conclusió

Els operadors lògics són eines essencials per controlar el flux dels teus programes en Ruby. Entenent com utilitzar &&, || i !, pots crear condicions més complexes i dinàmiques. Recorda considerar l'avaluació de curt circuit i el concepte de valors veritables i falsos quan treballis amb operadors lògics. Practica amb els exercicis proporcionats per consolidar la teva comprensió.
