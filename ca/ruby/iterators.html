Introducció als Iteradors

Els iteradors són un concepte fonamental en Ruby, que et permeten recórrer col·leccions com ara arrays i hashes. Proporcionen una manera d'accedir a cada element d'una col·lecció sense necessitat de gestionar manualment l'indexació.

Conceptes Clau

  • Iterador: Un mètode que recorre una col·lecció, cedint cada element a un bloc.
  • Bloc: Un tros de codi tancat en do...end o claus {...} que es pot passar a mètodes com els iteradors.

Iteradors Bàsics

Ruby proporciona diversos mètodes d'iterador integrats, el més comú és each.

Ús de `each` amb Arrays

El mètode each s'utilitza per iterar sobre cada element d'un array.

numbers = [1, 2, 3, 4, 5]
numbers.each do |number|
  puts number
end
  • Explicació: El mètode each recorre l'array numbers, cedint cada element al bloc. La variable de bloc |number| representa l'element actual.

Ús de `each` amb Hashes

El mètode each també es pot utilitzar amb hashes, cedint parelles clau-valor.

person = {name: "Alice", age: 30, city: "New York"}
person.each do |key, value|
  puts "#{key}: #{value}"
end
  • Explicació: El mètode each recorre el hash person, cedint cada parella clau-valor al bloc. Les variables de bloc |key, value| representen la clau i el valor actuals.

Iteradors Avançats

Ruby ofereix iteradors més avançats per a casos d'ús específics.

`map`

El mètode map crea un nou array que conté els valors retornats pel bloc.

numbers = [1, 2, 3, 4, 5]
squares = numbers.map do |number|
  number * number
end
puts squares.inspect
  • Explicació: El mètode map itera sobre l'array numbers, aplicant el bloc a cada element i retornant un nou array amb els resultats.

`select`

El mètode select retorna un nou array que conté elements pels quals el bloc retorna true.

numbers = [1, 2, 3, 4, 5]
even_numbers = numbers.select do |number|
  number.even?
end
puts even_numbers.inspect
  • Explicació: El mètode select itera sobre l'array numbers, seleccionant elements que són parells.

`reduce`

El mètode reduce combina tots els elements d'un array aplicant una operació binària, especificada per un bloc o símbol.

numbers = [1, 2, 3, 4, 5]
sum = numbers.reduce(0) do |accumulator, number|
  accumulator + number
end
puts sum
  • Explicació: El mètode reduce itera sobre l'array numbers, acumulant la suma de tots els elements.

Iteradors Personalitzats

Pots crear iteradors personalitzats definint mètodes que cedeixin a un bloc.

def custom_each(array)
  for element in array
    yield(element)
  end
end

custom_each([1, 2, 3, 4, 5]) do |number|
  puts number
end
  • Explicació: El mètode custom_each pren un array i cedeix cada element al bloc, similar al mètode each integrat.

Conclusió

Els iteradors són una característica poderosa en Ruby, que et permeten recórrer i manipular col·leccions amb facilitat. Entenent i utilitzant iteradors com each, map, select i reduce, pots escriure codi més concís i llegible. Els iteradors personalitzats amplien encara més aquesta flexibilitat, permetent-te crear comportaments d'iteració especialitzats adaptats a les teves necessitats.
