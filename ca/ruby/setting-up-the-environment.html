Introducció

Abans d'endinsar-te en la programació amb Ruby, és essencial configurar correctament el teu entorn de desenvolupament. Això assegura que tens totes les eines i configuracions necessàries per escriure, executar i depurar codi Ruby de manera eficient.

Instal·lació de Ruby

Utilitzant un Gestor de Versions

Una de les millors maneres de gestionar les instal·lacions de Ruby és utilitzant un gestor de versions com rbenv o RVM. Aquestes eines et permeten instal·lar i canviar fàcilment entre diferents versions de Ruby.

Instal·lant rbenv

  1. Instal·la rbenv:

    # Utilitzant Homebrew a macOS
brew install rbenv

  2. Configura rbenv al teu shell:

    rbenv init

    Segueix les instruccions per afegir rbenv al teu fitxer de configuració del shell (per exemple, .bashrc, .zshrc).

  3. Instal·la Ruby:

    rbenv install 3.0.2
rbenv global 3.0.2

Instal·lant RVM

  1. Instal·la RVM: 
    \curl -sSL https://get.rvm.io | bash -s stable
  2. Carrega RVM: 
    source ~/.rvm/scripts/rvm
  3. Instal·la Ruby: 
    rvm install 3.0.2
rvm use 3.0.2 --default

Instal·lació Directa

Si prefereixes no utilitzar un gestor de versions, pots instal·lar Ruby directament.

A macOS

  1. Utilitzant Homebrew: 
    brew install ruby

A Ubuntu/Debian

  1. Utilitzant APT: 
    sudo apt update
sudo apt install ruby-full

A Windows

  1. Utilitzant RubyInstaller:
    • Descarrega el RubyInstaller des de rubyinstaller.org.
    • Executa l'instal·lador i segueix les instruccions en pantalla.

Configuració d'un Editor de Text o IDE

Escollir l'editor de text o l'entorn de desenvolupament integrat (IDE) adequat pot millorar significativament la teva productivitat.

Editors de Text Populars

  • VS Code: Un editor altament personalitzable amb un ric ecosistema d'extensions.
    • Instal·la l'extensió de Ruby per a ressaltat de sintaxi i fragments de codi.
  • Sublime Text: Lleuger i ràpid, amb suport per a Ruby mitjançant plugins.
    • Instal·la el paquet de Ruby des del Package Control.

Entorns de Desenvolupament Integrats (IDEs)

  • RubyMine: Un IDE completament funcional dissenyat específicament per al desenvolupament amb Ruby i Rails.
    • Ofereix funcions avançades com la completació de codi, depuració i integració amb el control de versions.

Instal·lació de Gems Essencials

Les gems són biblioteques de Ruby que amplien la funcionalitat de les teves aplicacions. Algunes gems essencials per a qualsevol desenvolupador de Ruby inclouen:

  • Bundler: Gestiona les dependències de la teva aplicació. 
    gem install bundler
  • RSpec: Una eina de proves per a Ruby. 
    gem install rspec

Configuració del Teu Entorn

Configuració del Teu Projecte

  1. Crea un nou directori per al teu projecte: 
    mkdir my_ruby_project
cd my_ruby_project
  2. Inicialitza un nou projecte Ruby: 
    bundle init

Creació d'un Script Ruby Simple

  1. Crea un nou fitxer Ruby: 
    touch hello_world.rb
  2. Escriu un script Ruby simple: 
    # hello_world.rb
puts 'Hello, world!'
  3. Executa l'script: 
    ruby hello_world.rb

Conclusió

Configurar el teu entorn de desenvolupament Ruby és el primer pas per convertir-te en un programador proficient en Ruby. Instal·lant Ruby, escollint l'editor de text o IDE adequat, i configurant les eines i biblioteques essencials, crearàs una base sòlida per al teu viatge de programació. Amb el teu entorn preparat, ara pots centrar-te en aprendre i construir aplicacions Ruby de manera eficient.
