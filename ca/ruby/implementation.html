Introducció a la Implementació en Ruby

En aquesta secció, explorarem com implementar diverses característiques i funcionalitats en Ruby. Començarem amb conceptes bàsics i gradualment passarem a temes més avançats. Al final d'aquesta secció, hauríeu de ser capaços d'escriure i entendre codi Ruby que implementi una varietat de tasques i algoritmes.

Conceptes Bàsics d'Implementació

Variables i Tipus de Dades

  • Variables: Contenidors per emmagatzemar valors de dades.
  • Tipus de Dades: Diferents tipus de dades que es poden emmagatzemar en variables, com ara enters, floats, cadenes, arrays i hashes.
# Exemple de variables i tipus de dades
name = "Alice"       # Cadena
age = 30             # Enter
height = 5.5         # Float
is_student = true    # Booleà

Estructures de Control

  • Declaracions Condicionals: if, else, elsif, unless
  • Bucles: while, until, for, each
# Exemple d'una declaració condicional
if age > 18
  puts "You are an adult."
else
  puts "You are a minor."
end

# Exemple d'un bucle
3.times do |i|
  puts "Iteration #{i}"
end

Mètodes

  • Definició de Mètodes: Utilitzant la paraula clau def.
  • Crida de Mètodes: Simplement utilitzant el nom del mètode seguit de parèntesis.
# Exemple de definició i crida d'un mètode
def greet(name)
  "Hello, #{name}!"
end

puts greet("Alice")

Conceptes Intermedis d'Implementació

Classes i Objectes

  • Classes: Plantilles per crear objectes.
  • Objectes: Instàncies de classes.
# Exemple d'una classe i un objecte
class Person
  attr_accessor :name, :age

  def initialize(name, age)
    @name = name
    @age = age
  end

  def introduce
    "Hi, I'm #{@name} and I'm #{@age} years old."
  end
end

person = Person.new("Alice", 30)
puts person.introduce

Mòduls i Mixins

  • Mòduls: Col·leccions de mètodes i constants.
  • Mixins: Incloure mòduls en classes per compartir funcionalitat.
# Exemple d'un mòdul i un mixin
module Greetable
  def greet
    "Hello!"
  end
end

class Person
  include Greetable
end

person = Person.new
puts person.greet

Gestió d'Errors

  • Begin-Rescue-End: Gestió d'excepcions en Ruby.
# Exemple de gestió d'errors
begin
  result = 10 / 0
rescue ZeroDivisionError
  puts "Cannot divide by zero!"
end

Conceptes Avançats d'Implementació

Herència

  • Herència: Crear una nova classe basada en una classe existent.
# Exemple d'herència
class Animal
  def speak
    "Hello!"
  end
end

class Dog < Animal
  def speak
    "Woof!"
  end
end

dog = Dog.new
puts dog.speak

Polimorfisme

  • Polimorfisme: Utilitzar una única interfície per representar diferents tipus de dades.
# Exemple de polimorfisme
class Animal
  def speak
    "Hello!"
  end
end

class Dog < Animal
  def speak
    "Woof!"
  end
end

class Cat < Animal
  def speak
    "Meow!"
  end
end

animals = [Dog.new, Cat.new]
animals.each { |animal| puts animal.speak }

Metaprogramació

  • Metaprogramació: Escriure codi que escriu codi.
# Exemple de metaprogramació
class Person
  attr_accessor :name, :age
end

person = Person.new
person.name = "Alice"
person.age = 30
puts person.name
puts person.age

Conclusió

En aquesta secció, hem cobert la implementació de diverses característiques en Ruby, començant per conceptes bàsics com variables i estructures de control fins a temes més avançats com herència i metaprogramació. Entendre aquests conceptes us permetrà escriure codi Ruby més eficient i efectiu. Practiqueu aquestes implementacions per consolidar la vostra comprensió i convertir-vos en experts en programació Ruby.
