Introducció a les cadenes

Les cadenes són seqüències de caràcters utilitzades per representar text en Ruby. Són un dels tipus de dades més utilitzats i són essencials per gestionar dades textuals.

Creació de cadenes

  • Cometes simples: Les cadenes es poden crear utilitzant cometes simples (').
  • Cometes dobles: Les cadenes també es poden crear utilitzant cometes dobles (").
single_quoted_string = 'Hello, World!'
double_quoted_string = "Hello, World!"

Diferències entre cometes simples i dobles

  • Interpolació: Les cadenes amb cometes dobles suporten la interpolació de cadenes, mentre que les cadenes amb cometes simples no.
  • Seqüències d'escapament: Les cadenes amb cometes dobles suporten més seqüències d'escapament que les cadenes amb cometes simples.
name = "Alice"
single_quoted = 'Hello, #{name}'  # Mostra: Hello, #{name}
double_quoted = "Hello, #{name}"  # Mostra: Hello, Alice

single_quoted_escape = 'It\'s a sunny day.'  # Mostra: It's a sunny day.
double_quoted_escape = "Line1\nLine2"       # Mostra: Line1
                                            #          Line2

Mètodes de cadenes

Ruby proporciona un conjunt ric de mètodes per manipular cadenes.

Mètodes comuns de cadenes

  • Length: Retorna el nombre de caràcters de la cadena.
str = "Hello"
puts str.length  # Mostra: 5
  • Upcase/Downcase: Converteix tots els caràcters a majúscules o minúscules.
puts str.upcase   # Mostra: HELLO
puts str.downcase # Mostra: hello
  • Substrings: Extreu una part de la cadena.
puts str[0, 2]  # Mostra: He
  • Concatenació: Combina cadenes juntes.
greeting = "Hello"
name = "Alice"
puts greeting + " " + name  # Mostra: Hello Alice
  • Interpolació: Insereix variables o expressions dins d'una cadena.
age = 25
puts "I am #{age} years old."  # Mostra: I am 25 years old.

Mètodes avançats de cadenes

  • Split: Divideix una cadena en un array basat en un delimitador.
sentence = "Hello, World!"
words = sentence.split(", ")
puts words  # Mostra: ["Hello", "World!"]
  • Join: Combina elements d'un array en una sola cadena.
words = ["Hello", "World!"]
sentence = words.join(", ")
puts sentence  # Mostra: Hello, World!
  • Gsub: Substitueix totes les ocurrències d'un patró per un reemplaçament.
text = "The quick brown fox"
new_text = text.gsub("quick", "slow")
puts new_text  # Mostra: The slow brown fox

Comparació de cadenes

Les cadenes es poden comparar utilitzant diversos operadors.

Operadors de comparació

  • Igualtat (==): Comprova si dues cadenes són idèntiques.
puts "hello" == "hello"  # Mostra: true
puts "hello" == "world"  # Mostra: false
  • Desigualtat (!=): Comprova si dues cadenes no són idèntiques.
puts "hello" != "world"  # Mostra: true
  • Comparació lexicogràfica (<, >, <=, >=): Compara cadenes basant-se en l'ordre alfabètic.
puts "apple" < "banana"  # Mostra: true
puts "apple" > "banana"  # Mostra: false

Codificació de cadenes

Ruby suporta múltiples codificacions de cadenes, cosa que és essencial per gestionar la internacionalització.

Comprovar i canviar la codificació

  • Codificació: Retorna la codificació de la cadena.
str = "Hello"
puts str.encoding  # Mostra: UTF-8
  • Forçar codificació: Canvia la codificació de la cadena.
str = "Hello"
str.force_encoding("ASCII-8BIT")
puts str.encoding  # Mostra: ASCII-8BIT

Conclusió

Les cadenes són una part fonamental de la programació en Ruby. Entendre com crear, manipular i comparar cadenes és crucial per gestionar dades textuals de manera efectiva. En dominar els diversos mètodes i tècniques discutits, podràs treballar eficientment amb cadenes en les teves aplicacions Ruby.
