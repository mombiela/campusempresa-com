Introducció als Mètodes

Els mètodes en Ruby són una manera d'agrupar codi reutilitzable en una sola unitat. Ajuden a fer el codi més modular, llegible i mantenible. Un mètode en Ruby es defineix utilitzant la paraula clau def seguida del nom del mètode i paràmetres opcionals.

Conceptes Clau

  • Definició de Mètode: La sintaxi per definir un mètode.
  • Invocació de Mètode: Com cridar o invocar un mètode.
  • Paràmetres i Arguments: Passar dades als mètodes.
  • Valors de Retorn: Què retorna un mètode després de l'execució.
  • Àmbit: Entendre l'àmbit de les variables dins dels mètodes.

Definició de Mètode

Per definir un mètode en Ruby, utilitzes la paraula clau def seguida del nom del mètode. Aquí tens la sintaxi bàsica:

def nom_del_metode
  # codi a executar
end

Exemple

def dir_hola
  puts "Hola, Món!"
end

En aquest exemple, dir_hola és un mètode que imprimeix "Hola, Món!" a la consola.

Invocació de Mètode

Per invocar o cridar un mètode, simplement utilitzes el nom del mètode seguit de parèntesis (opcional si no es passen arguments).

Exemple

dir_hola

Això mostrarà:

Hola, Món!

Paràmetres i Arguments

Els mètodes poden prendre paràmetres, que són variables que contenen els valors passats al mètode quan es crida.

Exemple

def saludar(nom)
  puts "Hola, #{nom}!"
end

saludar("Alice")

Això mostrarà:

Hola, Alice!

Múltiples Paràmetres

Pots definir mètodes amb múltiples paràmetres.

def sumar(a, b)
  a + b
end

puts sumar(2, 3)

Això mostrarà:

5

Valors de Retorn

Per defecte, un mètode Ruby retorna el valor de l'última expressió avaluada. També pots utilitzar la paraula clau return per retornar explícitament un valor.

Exemple

def multiplicar(a, b)
  return a * b
end

puts multiplicar(4, 5)

Això mostrarà:

20

Àmbit

Les variables definides dins d'un mètode no són accessibles fora d'aquest. Això es coneix com l'àmbit de les variables.

Exemple

def exemple_metode
  var_local = "Sóc dins del mètode"
  puts var_local
end

exemple_metode
# puts var_local # Això generarà un error

Això mostrarà:

Sóc dins del mètode

Intentar accedir a var_local fora del mètode resultarà en un error.

Conceptes Avançats de Mètodes

Paràmetres per Defecte

Pots assignar valors per defecte als paràmetres. Si no es passa cap argument, s'utilitza el valor per defecte.

def saludar(nom="Món")
  puts "Hola, #{nom}!"
end

saludar
saludar("Alice")

Això mostrarà:

Hola, Món!
Hola, Alice!

Nombre Variable d'Arguments

Ruby permet que els mètodes acceptin un nombre variable d'arguments utilitzant l'operador splat (*).

def sumar(*numeros)
  numeros.inject(0) { |suma, numero| suma + numero }
end

puts sumar(1, 2, 3, 4)

Això mostrarà:

10

Arguments per Paraula Clau

Ruby també suporta arguments per paraula clau, que són una manera de passar arguments utilitzant parelles clau-valor.

def imprimir_adreça(ciutat:, estat:)
  puts "Ciutat: #{ciutat}, Estat: #{estat}"
end

imprimir_adreça(ciutat: "San Francisco", estat: "CA")

Això mostrarà:

Ciutat: San Francisco, Estat: CA

Conclusió

Definir mètodes en Ruby és una habilitat fonamental que ajuda a escriure codi net, reutilitzable i mantenible. Entenent com definir mètodes, passar paràmetres, gestionar valors de retorn i administrar l'àmbit, pots crear programes Ruby més complexos i eficients. A mesura que avancis, també trobaràs característiques més sofisticades com paràmetres per defecte, arguments variables i arguments per paraula clau, que proporcionen flexibilitat i potència addicional en la definició de mètodes.
