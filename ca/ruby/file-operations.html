Operacions amb fitxers en Ruby
Introducció a les Excepcions ►

Introducció

Les operacions amb fitxers són essencials per a moltes aplicacions, ja que permeten llegir i escriure fitxers. Aquesta secció cobrirà els conceptes bàsics de la gestió de fitxers en Ruby, progressant des d'operacions simples fins a tècniques més avançades.

Operacions bàsiques amb fitxers

Obrir i tancar fitxers

  • Obrir un fitxer: Utilitza el mètode File.open per obrir un fitxer. Aquest mètode pot prendre un bloc, que assegura que el fitxer es tanqui correctament després que el bloc s'executi.
  • Tancar un fitxer: Si no utilitzes un bloc, has de tancar explícitament el fitxer utilitzant el mètode close.
# Obrir un fitxer amb un bloc
File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'
end
# El fitxer es tanca automàticament després del bloc

# Obrir un fitxer sense un bloc
file = File.open('example.txt', 'w')
file.puts 'Hello, world!'
file.close

Llegir fitxers

  • Llegir tot el fitxer: Utilitza File.read per llegir tot el contingut d'un fitxer.
  • Llegir línia per línia: Utilitza File.foreach o IO.readlines per llegir un fitxer línia per línia.
# Llegir tot el fitxer
content = File.read('example.txt')
puts content

# Llegir línia per línia
File.foreach('example.txt') do |line|
  puts line
end

Escriure fitxers

  • Modes d'escriptura: Utilitza diferents modes com 'w' per escriure, 'a' per afegir, i 'r+' per llegir/escriure.
  • Escriure dades: Utilitza mètodes com puts, write, i print per escriure dades a un fitxer.
# Escriure a un fitxer
File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'
end

# Afegir a un fitxer
File.open('example.txt', 'a') do |file|
  file.puts 'Appending this line.'
end

Operacions intermèdies amb fitxers

Modes i permisos de fitxers

  • Modes de fitxers: Entén els diferents modes de fitxers com 'r', 'w', 'a', 'r+', 'w+', i 'a+'.
  • Permisos: Utilitza File.chmod per canviar els permisos dels fitxers.
# Canviar els permisos del fitxer
File.chmod(0644, 'example.txt')

Posicionament de fitxers

  • Seek i Tell: Utilitza seek per moure el punter del fitxer i tell per obtenir la posició actual.
File.open('example.txt', 'r') do |file|
  file.seek(5, IO::SEEK_SET) # Moure's al 5è byte
  puts file.tell # Mostra: 5
end

Treballar amb fitxers binaris

  • Mode binari: Obre fitxers en mode binari utilitzant 'rb', 'wb', o 'ab'.
# Llegir un fitxer binari
File.open('example.bin', 'rb') do |file|
  data = file.read
  puts data
end

Operacions avançades amb fitxers

Manipulació de fitxers

  • Renombrar i eliminar fitxers: Utilitza File.rename i File.delete per renombrar i eliminar fitxers.
# Renombrar un fitxer
File.rename('example.txt', 'new_example.txt')

# Eliminar un fitxer
File.delete('new_example.txt')

Operacions amb directoris

  • Crear i eliminar directoris: Utilitza Dir.mkdir i Dir.rmdir per crear i eliminar directoris.
  • Llistar el contingut del directori: Utilitza Dir.entries o Dir.glob per llistar el contingut del directori.
# Crear un directori
Dir.mkdir('new_directory')

# Llistar el contingut del directori
puts Dir.entries('.')

# Eliminar un directori
Dir.rmdir('new_directory')

Informació de fitxers i directoris

  • Informació de fitxers: Utilitza mètodes com File.size, File.exist?, i File.mtime per obtenir informació sobre els fitxers.
  • Informació de directoris: Utilitza Dir.pwd per obtenir el directori actual i Dir.chdir per canviar el directori actual.
# Obtenir informació del fitxer
puts File.size('example.txt')
puts File.exist?('example.txt')
puts File.mtime('example.txt')

# Obtenir i canviar el directori actual
puts Dir.pwd
Dir.chdir('..')
puts Dir.pwd

Conclusió

Les operacions amb fitxers en Ruby són potents i versàtils, permetent gestionar fitxers i directoris de manera eficient. Des de la lectura i escriptura bàsiques fins a la manipulació avançada de fitxers i operacions amb directoris, dominar aquestes habilitats et permetrà construir aplicacions robustes que interactuen amb el sistema de fitxers. Practica aquests conceptes amb diversos exercicis per consolidar la teva comprensió i convertir-te en un expert en la gestió de fitxers amb Ruby.
