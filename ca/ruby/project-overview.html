Introducció

En aquest curs, et guiarem a través del viatge d'aprendre Ruby, des de nivells principiants fins a avançats. Ruby és un llenguatge de programació dinàmic i de codi obert amb un enfocament en la simplicitat i la productivitat. Té una sintaxi elegant que és natural de llegir i fàcil d'escriure. Al final d'aquest curs, seràs competent en Ruby i capaç de construir els teus propis projectes.

Estructura del Curs

El curs està dividit en tres seccions principals:

  1. Nivell Principiant
  2. Nivell Intermedi
  3. Nivell Avançat

Cada secció es construeix sobre l'anterior, assegurant una transició suau de conceptes bàsics a complexos.

Nivell Principiant

Conceptes Clau

  • Introducció a Ruby: Entendre què és Ruby i la seva història.
  • Configuració de l'Entorn: Instal·lar Ruby i configurar un entorn de desenvolupament.
  • Sintaxi Bàsica: Aprendre la sintaxi bàsica i l'estructura del codi Ruby.
  • Variables i Tipus de Dades: Entendre els diferents tipus de dades i com utilitzar variables.
  • Estructures de Control: Utilitzar declaracions if-else, bucles i iteradors.

Exemple de Codi

# Exemple de Sintaxi Bàsica de Ruby
puts "Hola, Món!"

# Variables i Tipus de Dades
nom = "Alice"
edat = 30
és_estudiant = true

# Estructures de Control
if edat > 18
  puts "#{nom} és un adult."
else
  puts "#{nom} no és un adult."
end

# Exemple de Bucle
3.times do |i|
  puts "Aquest és el bucle número #{i + 1}"
end

Nivell Intermedi

Conceptes Clau

  • Mètodes: Definir i utilitzar mètodes.
  • Classes i Objectes: Entendre la programació orientada a objectes en Ruby.
  • Mòduls i Mixins: Utilitzar mòduls per organitzar el codi i mixins per compartir funcionalitat.
  • Entrada/Sortida de Fitxers: Llegir i escriure fitxers.
  • Gestió d'Errors: Utilitzar excepcions per gestionar errors de manera elegant.

Exemple de Codi

# Mètodes
def saludar(nom)
  "Hola, #{nom}!"
end

puts saludar("Bob")

# Classes i Objectes
class Persona
  attr_accessor :nom, :edat

  def initialize(nom, edat)
    @nom = nom
    @edat = edat
  end

  def presentar
    "Hola, sóc #{@nom} i tinc #{@edat} anys."
  end
end

persona = Persona.new("Alice", 30)
puts persona.presentar

# Mòduls i Mixins
module Salutació
  def saludar
    "Hola!"
  end
end

class Robot
  include Salutació
end

robot = Robot.new
puts robot.saludar

# Entrada/Sortida de Fitxers
File.open("exemple.txt", "w") do |fitxer|
  fitxer.puts "Aquest és un fitxer d'exemple."
end

# Gestió d'Errors
begin
  resultat = 10 / 0
rescue ZeroDivisionError
  puts "No pots dividir per zero!"
end

Nivell Avançat

Conceptes Clau

  • Metaprogramació: Escriure codi que escriu codi.
  • Concurrència i Paral·lelisme: Utilitzar fils i processos.
  • Programació de Xarxes: Construir aplicacions en xarxa.
  • Proves: Escriure proves per assegurar la qualitat del codi.
  • Optimització del Rendiment: Tècniques per fer que el codi Ruby s'executi més ràpid.

Exemple de Codi

# Metaprogramació
class MètodeDinàmic
  define_method(:saludar) do |nom|
    "Hola, #{nom}!"
  end
end

dinàmic = MètodeDinàmic.new
puts dinàmic.saludar("Charlie")

# Concurrència i Paral·lelisme
fils = []
5.times do |i|
  fils << Thread.new do
    puts "El fil #{i} està funcionant"
  end
end
fils.each(&:join)

# Programació de Xarxes
require 'socket'

servidor = TCPServer.new(2345)
loop do
  client = servidor.accept
  client.puts "Hola des del servidor!"
  client.close
end

# Proves
require 'minitest/autorun'

class ProvaExemple < Minitest::Test
  def test_suma
    assert_equal 4, 2 + 2
  end
end

# Optimització del Rendiment
require 'benchmark'

array = (1..1000000).to_a
Benchmark.bm do |x|
  x.report { array.map { |i| i * 2 } }
  x.report { array.each { |i| i * 2 } }
end

Conclusió

Aquest curs t'equiparà amb els coneixements i habilitats per convertir-te en un expert en Ruby. Cada secció inclou exemples pràctics i exercicis per reforçar el teu aprenentatge. Feliç programació!
