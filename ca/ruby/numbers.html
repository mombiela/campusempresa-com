Introducció

Els nombres són una part fonamental de qualsevol llenguatge de programació, i Ruby no és una excepció. En aquesta secció, cobrirem els conceptes bàsics dels nombres en Ruby, incloent diferents tipus de nombres, operacions aritmètiques bàsiques, i alguns temes avançats com treballar amb nombres grans i nombres complexos.

Tipus de Nombres

Ruby suporta diversos tipus de nombres, cadascun amb les seves pròpies característiques i casos d'ús.

  • Enter: Nombres sencers sense punt decimal.
  • Float: Nombres amb punt decimal.
  • Racional: Nombres representats com a fraccions.
  • Complex: Nombres amb una part real i una part imaginària.

Enter

Els enters són nombres sencers, tant positius com negatius, incloent zero.

a = 42
b = -7
c = 0

Float

Els floats són nombres que contenen un punt decimal.

a = 3.14
b = -0.001
c = 2.0

Racional

Els nombres racionals es representen com a fraccions.

a = Rational(2, 3)  # 2/3
b = Rational(7, 4)  # 7/4

Complex

Els nombres complexos tenen una part real i una part imaginària.

a = Complex(2, 3)  # 2 + 3i
b = Complex(1, -4) # 1 - 4i

Operacions Aritmètiques Bàsiques

Ruby suporta operacions aritmètiques bàsiques com la suma, la resta, la multiplicació i la divisió.

# Suma
sum = 5 + 3  # 8

# Resta
difference = 10 - 4  # 6

# Multiplicació
product = 7 * 6  # 42

# Divisió
quotient = 20 / 4  # 5

# Mòdul (residu)
remainder = 10 % 3  # 1

# Exponenciació
power = 2 ** 3  # 8

Temes Avançats

Treballar amb Nombres Grans

Ruby pot manejar nombres molt grans amb facilitat. No cal preocupar-se pel desbordament d'enters.

large_number = 123456789012345678901234567890
puts large_number * large_number

Convertir Entre Tipus de Nombres

Pots convertir entre diferents tipus de nombres utilitzant els mètodes integrats de Ruby.

# Enter a Float
int_num = 10
float_num = int_num.to_f  # 10.0

# Float a Enter
float_num = 3.14
int_num = float_num.to_i  # 3

# Enter a Racional
int_num = 5
rational_num = int_num.to_r  # 5/1

# Float a Racional
float_num = 2.5
rational_num = float_num.to_r  # 5/2

# Enter a Complex
int_num = 4
complex_num = int_num.to_c  # 4+0i

Operacions amb Nombres Complexos

Ruby proporciona mètodes per realitzar operacions aritmètiques amb nombres complexos.

a = Complex(2, 3)
b = Complex(1, -4)

# Suma
sum = a + b  # 3 - 1i

# Resta
difference = a - b  # 1 + 7i

# Multiplicació
product = a * b  # 14 - 5i

# Divisió
quotient = a / b  # -0.6470588235294118 + 0.7058823529411765i

Resum

En aquesta secció, hem cobert els conceptes bàsics dels nombres en Ruby, incloent diferents tipus de nombres (Enter, Float, Racional, Complex), operacions aritmètiques bàsiques, i alguns temes avançats com treballar amb nombres grans i convertir entre tipus de nombres. Entendre aquests conceptes és crucial per realitzar operacions matemàtiques i manejar dades numèriques en els teus programes Ruby.
