Configurar Rails és un pas essencial per a qualsevol desenvolupador de Ruby que vulgui crear aplicacions web. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de Rails al teu sistema, assegurant que tinguis una base sòlida per començar a desenvolupar les teves aplicacions.

Prerequisits

Abans de configurar Rails, assegura't de tenir els següents prerequisits instal·lats al teu sistema:

  • Ruby: Rails és un framework de Ruby, així que necessites tenir Ruby instal·lat. Preferiblement, utilitza un gestor de versions com RVM o rbenv.
  • Node.js: Rails utilitza Node.js per gestionar les dependències de JavaScript.
  • Yarn: Un gestor de paquets per a JavaScript, utilitzat per Rails per gestionar les dependències del front-end.
  • SQLite3: Una base de dades lleugera utilitzada per defecte a Rails.

Instal·lant Ruby

Primer, instal·la Ruby utilitzant un gestor de versions. Aquí, utilitzarem rbenv:

  1. Instal·la rbenv i ruby-build:

    $ brew install rbenv
$ brew install ruby-build

  2. Configura rbenv al teu shell:

    $ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile
$ source ~/.bash_profile

  3. Instal·la Ruby:

    $ rbenv install 3.0.0
$ rbenv global 3.0.0

  4. Verifica la instal·lació:

    $ ruby -v

Instal·lant Node.js i Yarn

A continuació, instal·la Node.js i Yarn:

  1. Instal·la Node.js:

    $ brew install node

  2. Instal·la Yarn:

    $ brew install yarn

Instal·lant SQLite3

SQLite3 és la base de dades per defecte per al desenvolupament de Rails. Instal·la-la utilitzant Homebrew:

$ brew install sqlite3

Instal·lant Rails

Amb Ruby, Node.js, Yarn i SQLite3 instal·lats, ara pots instal·lar Rails:

  1. Instal·la Rails:

    $ gem install rails

  2. Verifica la instal·lació:

    $ rails -v

Creant una Nova Aplicació Rails

Un cop Rails està instal·lat, pots crear una nova aplicació Rails:

  1. Crea una nova aplicació Rails:

    $ rails new myapp

  2. Navega al directori de la teva aplicació:

    $ cd myapp

  3. Inicia el servidor de Rails:

    $ rails server

  4. Obre el teu navegador i navega a http://localhost:3000 per veure la teva nova aplicació Rails en funcionament.

Conclusió

Configurar Rails implica instal·lar diverses dependències i configurar el teu entorn de desenvolupament. Seguint els passos descrits en aquesta secció, hauràs de tenir una configuració de Rails completament funcional llesta per al desenvolupament. Aquesta base et permetrà començar a construir aplicacions web potents utilitzant el framework Rails.
