Introducció a les Constants

A Ruby, les constants s'utilitzen per emmagatzemar valors que no han de canviar durant l'execució d'un programa. Són una part fonamental de la programació en Ruby i ajuden a mantenir la integritat de les dades que han de romandre constants.

  • Definició: Les constants són variables els valors de les quals es pretenen mantenir inalterats.
  • Convenció de Nomenclatura: Les constants s'escriuen en majúscules, amb les paraules separades per guions baixos (_).

Declaració de Constants

Per declarar una constant a Ruby, simplement assigna un valor a un nom de variable que comenci amb una lletra majúscula.

PI = 3.14159
MAX_USERS = 100
WELCOME_MESSAGE = "Welcome to Ruby Programming!"
  • PI, MAX_USERS i WELCOME_MESSAGE són constants.
  • Per convenció, les constants s'escriuen en majúscules.

Accés a les Constants

Les constants es poden accedir directament pel seu nom dins del mateix àmbit.

puts PI  # Output: 3.14159
puts MAX_USERS  # Output: 100
puts WELCOME_MESSAGE  # Output: Welcome to Ruby Programming!

Constants en Classes i Mòduls

Les constants també es poden definir dins de classes i mòduls. S'accedeixen utilitzant l'operador de resolució d'àmbit ::.

class Circle
  PI = 3.14159
end

module Config
  MAX_USERS = 100
end

puts Circle::PI  # Output: 3.14159
puts Config::MAX_USERS  # Output: 100
  • Circle::PI accedeix a la constant PI dins de la classe Circle.
  • Config::MAX_USERS accedeix a la constant MAX_USERS dins del mòdul Config.

Modificació de Constants

Tot i que les constants es pretenen mantenir inalterades, Ruby permet modificar-les. No obstant això, fer-ho generarà un advertiment.

PI = 3.14159
PI = 3.14  # Warning: already initialized constant PI
  • Modificar una constant generalment es desaconsella ja que va en contra del propòsit d'utilitzar constants.

Constants i Herència

Quan una classe hereta d'una altra classe, també hereta les seves constants.

class Parent
  CONSTANT = "Parent Constant"
end

class Child < Parent
end

puts Child::CONSTANT  # Output: Parent Constant
  • La classe Child hereta la constant CONSTANT de la classe Parent.

Exercicis

  1. Declarar i Accedir a Constants: Crea una constant GRAVITY amb un valor de 9.8 i imprimeix-la.
  2. Constants en Classes: Defineix una classe MathConstants amb una constant E (nombre d'Euler) establerta a 2.718. Accedeix i imprimeix aquesta constant.
  3. Modificar una Constant: Declara una constant SPEED_OF_LIGHT amb un valor de 299792458. Intenta canviar el seu valor i observa l'advertiment.

Solucions Exemple

  1. Declarar i Accedir a Constants

    GRAVITY = 9.8
puts GRAVITY  # Output: 9.8

  2. Constants en Classes

    class MathConstants
  E = 2.718
end

puts MathConstants::E  # Output: 2.718

  3. Modificar una Constant

    SPEED_OF_LIGHT = 299792458
SPEED_OF_LIGHT = 300000000  # Warning: already initialized constant SPEED_OF_LIGHT

Conclusió

Les constants a Ruby són una característica poderosa que ajuda a mantenir la integritat dels valors que no han de canviar durant l'execució d'un programa. Seguint les convencions i entenent l'àmbit i l'herència de les constants, pots escriure codi Ruby més robust i mantenible. Recorda, tot i que Ruby permet modificar constants, és una bona pràctica evitar fer-ho per preservar la immutabilitat prevista d'aquests valors.
