Introducció a GHCi

GHCi (Glasgow Haskell Compiler interactive) és l'entorn interactiu per a Haskell. Et permet carregar i executar codi Haskell de manera interactiva, convertint-se en una eina inestimable per aprendre, depurar i experimentar amb programes Haskell.

  • REPL: Read-Eval-Print Loop, un entorn de programació interactiu.
  • GHCi: El REPL per a Haskell proporcionat pel Glasgow Haskell Compiler.

Iniciar GHCi

Per iniciar GHCi, obre el teu terminal i escriu:

ghci

Hauries de veure un prompt com aquest:

GHCi, version 8.10.4: https://www.haskell.org/ghc/  :? for help
Prelude>

Comandes bàsiques

Aquí tens algunes comandes bàsiques que pots utilitzar en GHCi:

  • :load o :l: Carregar un fitxer Haskell.
  • :reload o :r: Recarregar el fitxer actual.
  • :type o :t: Mostrar el tipus d'una expressió.
  • :info o :i: Mostrar informació sobre un nom.
  • :quit o :q: Sortir de GHCi.

Exemple d'ús

Prelude> :load MyModule.hs
[1 of 1] Compiling MyModule        ( MyModule.hs, interpreted )
Ok, one module loaded.

Prelude> :type 5
5 :: Num p => p

Prelude> :info Maybe
data Maybe a = Nothing | Just a
    -- Defined in ‘GHC.Base’
instance Eq a => Eq (Maybe a) -- Defined in ‘GHC.Base’
instance Monad Maybe -- Defined in ‘GHC.Base’
...

Avaluar expressions

Pots avaluar expressions Haskell directament en GHCi:

Prelude> 2 + 3
5

Prelude> map (+1) [1, 2, 3]
[2, 3, 4]

Definir funcions

Pots definir funcions directament en GHCi utilitzant la paraula clau let:

Prelude> let square x = x * x
Prelude> square 5
25

Carregar i recarregar mòduls

Quan treballes amb projectes més grans, sovint carregaràs i recarregaràs mòduls:

Prelude> :load MyModule.hs
Prelude> :reload

Inspeccionar tipus i informació

Entendre els tipus i les funcions disponibles és crucial en Haskell:

Prelude> :type map
map :: (a -> b) -> [a] -> [b]

Prelude> :info map
map :: (a -> b) -> [a] -> [b] 	-- Defined in ‘GHC.Base’

Utilitzar GHCi per depurar

GHCi pot ser una eina poderosa per depurar. Pots inspeccionar valors intermedis i provar petites peces de codi de manera interactiva.

Exemple de sessió de depuració

Suposa que tens una funció que no funciona com esperaves:

-- MyModule.hs
module MyModule where

factorial :: Integer -> Integer
factorial 0 = 1
factorial n = n * factorial (n - 1)

Pots carregar aquest mòdul i provar la funció:

Prelude> :load MyModule.hs
Prelude MyModule> factorial 5
120

Si la funció no funciona com esperaves, pots inspeccionar valors intermedis:

Prelude MyModule> let factorial n = if n == 0 then 1 else n * factorial (n - 1)
Prelude MyModule> factorial 5
120

Conclusió

GHCi és una eina essencial per als programadors Haskell, proporcionant un entorn interactiu per provar, depurar i explorar codi Haskell. Dominant les comandes i característiques de GHCi, pots millorar significativament la teva productivitat i comprensió de Haskell.

  • Practica: Utilitza regularment GHCi per provar petits fragments de codi i depurar els teus programes.
  • Explora: Utilitza comandes com :type i :info per aprendre més sobre els tipus i funcions de Haskell.
  • Experimenta: No dubtis a experimentar amb noves idees en GHCi abans d'integrar-les en el teu codi principal.

Incorporant GHCi en el teu flux de treball, trobaràs que aprendre i treballar amb Haskell és una experiència més interactiva i agradable.
