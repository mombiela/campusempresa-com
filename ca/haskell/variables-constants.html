Variables i constants en Haskell
Tipus de Dades Bàsics ►

Introducció

En Haskell, les variables i les constants són conceptes fonamentals que ajuden a emmagatzemar i manipular dades. A diferència dels llenguatges imperatius, Haskell tracta les variables de manera diferent a causa de la seva naturalesa funcional. Aquesta secció cobrirà els conceptes bàsics de les variables i les constants, les seves diferències i com utilitzar-les de manera efectiva en Haskell.

Variables en Haskell

En Haskell, les variables són immutables, és a dir, un cop assignat un valor a una variable, aquest no es pot canviar. Aquesta és una característica clau de la programació funcional.

Conceptes clau

  • Immutabilitat: Un cop una variable té assignat un valor, aquest no es pot canviar.
  • Associació: El procés d'associar una variable amb un valor.
  • Àmbit: La regió del codi on la variable és accessible.

Exemple

-- Associació de variables
x = 10

-- Ús de la variable en una funció
addTen :: Int -> Int
addTen y = x + y

-- Exemple d'ús
main = print (addTen 5)  -- Sortida: 15

Explicació:

  • x = 10 associa el valor 10 a la variable x.
  • addTen és una funció que pren un enter y i li suma x.
  • En main, addTen 5 resulta en 15 perquè x és 10 i y és 5.

Constants en Haskell

Les constants en Haskell són similars a les variables però s'utilitzen per representar valors fixos que no canvien al llarg del programa.

Conceptes clau

  • Valors literals: Valors escrits directament com 42, 3.14 o "Hola".
  • Constants amb nom: Constants a les quals se'ls dona un nom per a una millor llegibilitat i mantenibilitat.

Exemple

-- Constant amb nom
piValue = 3.14159

-- Ús de la constant en una funció
circumference :: Float -> Float
circumference radius = 2 * piValue * radius

-- Exemple d'ús
main = print (circumference 5)  -- Sortida: 31.4159

Explicació:

  • piValue = 3.14159 assigna el valor de Pi a la constant piValue.
  • circumference és una funció que calcula la circumferència d'un cercle utilitzant el radi i piValue.
  • En main, circumference 5 resulta en 31.4159 perquè el radi és 5 i piValue és 3.14159.

Diferències entre variables i constants

| Característica | Variables | Constants | |----------------|--------------------------|--------------------------| | Mutabilitat | Immutable | Immutable | | Propòsit | Emmagatzemar valors temporals | Representar valors fixos | | Ús | Es poden utilitzar en funcions | Utilitzades per a valors fixos | | Exemple | x = 10 | piValue = 3.14159 |

Exercicis

  1. Associació bàsica de variables: Defineix una variable a amb el valor 20 i crea una funció double que dobli el valor de a.

    a = 20

double :: Int -> Int
double y = a * 2

main = print (double 0)  -- La sortida hauria de ser 40

  2. Ús de constants: Defineix una constant gravity amb el valor 9.8 i crea una funció fallTime que calculi el temps que triga un objecte a caure des d'una alçada donada.

    gravity = 9.8

fallTime :: Float -> Float
fallTime height = sqrt (2 * height / gravity)

main = print (fallTime 20)  -- La sortida hauria de ser aproximadament 2.02

Conclusió

Entendre les variables i les constants en Haskell és crucial per escriure codi efectiu i eficient. Les variables en Haskell són immutables, cosa que s'alinea amb el paradigma de la programació funcional, i les constants s'utilitzen per representar valors fixos. En dominar aquests conceptes, pots escriure programes en Haskell més predictibles i mantenibles.
