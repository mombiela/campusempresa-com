Introducció

Les funcions de primera classe són un concepte fonamental en Haskell i en molts altres llenguatges de programació funcional. Aquest tema cobrirà què són les funcions de primera classe, com s'utilitzen en Haskell, i proporcionarà exemples i exercicis per consolidar la teva comprensió.

Què són les Funcions de Primera Classe?

Les funcions de primera classe són funcions que es tracten com a ciutadans de primera classe. Això significa que les funcions poden ser:

  • Assignades a variables.
  • Passades com a arguments a altres funcions.
  • Retornades com a valors d'altres funcions.
  • Emmagatzemades en estructures de dades.

En Haskell, les funcions són ciutadans de primera classe, cosa que permet patrons de programació potents i flexibles.

Assignar Funcions a Variables

En Haskell, pots assignar funcions a variables igual que qualsevol altre valor.

add :: Int -> Int -> Int
add x y = x + y

-- Assignar la funció a una variable
addition = add

-- Utilitzar la variable per cridar la funció
result = addition 5 3  -- result serà 8

Passar Funcions com a Arguments

Les funcions poden ser passades com a arguments a altres funcions. Això s'utilitza sovint en funcions d'ordre superior.

applyTwice :: (a -> a) -> a -> a
applyTwice f x = f (f x)

-- Exemple d'ús
double :: Int -> Int
double x = x * 2

result = applyTwice double 3  -- result serà 12

Retornar Funcions des de Funcions

Les funcions també poden retornar altres funcions.

createAdder :: Int -> (Int -> Int)
createAdder x = \y -> x + y

-- Exemple d'ús
addFive = createAdder 5
result = addFive 10  -- result serà 15

Emmagatzemar Funcions en Estructures de Dades

Les funcions poden ser emmagatzemades en llistes, tuples o altres estructures de dades.

functions :: [Int -> Int]
functions = [(*2), (+3), subtract 1]

-- Aplicar totes les funcions a un valor
applyAll :: Int -> [Int]
applyAll x = map ($ x) functions

result = applyAll 5  -- result serà [10, 8, 4]

Exemples i Exercicis

Exemple 1: Utilitzar Funcions d'Ordre Superior

Creem una funció que prengui una llista d'enters i una funció, i apliqui la funció a cada element de la llista.

mapFunction :: (Int -> Int) -> [Int] -> [Int]
mapFunction f xs = map f xs

-- Exemple d'ús
increment :: Int -> Int
increment x = x + 1

result = mapFunction increment [1, 2, 3, 4]  -- result serà [2, 3, 4, 5]

Exercici 1: Crear una Funció de Filtre

Crea una funció filterFunction que prengui un predicat (una funció que retorna un Bool) i una llista, i retorni una llista d'elements que satisfacin el predicat.

filterFunction :: (a -> Bool) -> [a] -> [a]
filterFunction pred xs = filter pred xs

-- Exemple d'ús
isEven :: Int -> Bool
isEven x = x `mod` 2 == 0

result = filterFunction isEven [1, 2, 3, 4, 5, 6]  -- result serà [2, 4, 6]

Exercici 2: Composar Funcions

Crea una funció compose que prengui dues funcions i retorni la seva composició.

compose :: (b -> c) -> (a -> b) -> (a -> c)
compose f g = \x -> f (g x)

-- Exemple d'ús
double :: Int -> Int
double x = x * 2

increment :: Int -> Int
increment x = x + 1

result = (compose double increment) 3  -- result serà 8

Conclusió

Les funcions de primera classe són una característica poderosa en Haskell que permeten codi flexible i expressiu. Entenent com assignar funcions a variables, passar-les com a arguments, retornar-les des d'altres funcions i emmagatzemar-les en estructures de dades, pots aprofitar tot el potencial de les capacitats de programació funcional de Haskell. Practica els exemples i exercicis proporcionats per consolidar la teva comprensió de les funcions de primera classe.
