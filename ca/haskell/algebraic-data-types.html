Introducció als Tipus de Dades Algebraiques (ADTs)

Els Tipus de Dades Algebraiques (ADTs) són un concepte fonamental en Haskell i la programació funcional. Permeten definir tipus de dades personalitzats que poden representar una combinació de diferents valors. Els ADTs es componen de dos tipus principals: tipus producte i tipus suma.

Conceptes Clau

  • Tipus Producte: Combinen múltiples valors en una sola estructura de dades (per exemple, tuples, registres).
  • Tipus Suma: Representen un valor que pot ser un de diversos tipus diferents (per exemple, Either, Maybe).

Tipus Producte

Els tipus producte s'utilitzen per agrupar múltiples valors junts. En Haskell, els tipus producte més comuns són les tuples i els registres.

Tuples

Les tuples són tipus producte simples que agrupen un nombre fix de valors junts.

-- Una tuple que conté un Int i un String
exampleTuple :: (Int, String)
exampleTuple = (42, "Hello")

Registres

Els registres són tipus producte més descriptius que utilitzen camps amb nom.

-- Definint un tipus de registre per a una Persona
data Person = Person {
    name :: String,
    age :: Int
}

-- Creant una instància de Persona
examplePerson :: Person
examplePerson = Person { name = "Alice", age = 30 }

Tipus Suma

Els tipus suma permeten que un valor sigui un de diversos tipus diferents. Es defineixen utilitzant la paraula clau data amb múltiples constructors.

Either

El tipus Either representa un valor que pot ser un de dos tipus.

-- Either pot ser Left o Right
exampleEither :: Either String Int
exampleEither = Right 42

Maybe

El tipus Maybe representa un valor opcional que pot ser Just un valor o Nothing.

-- Maybe pot ser Just un valor o Nothing
exampleMaybe :: Maybe Int
exampleMaybe = Just 42

Definint ADTs Personalitzats

Pots definir els teus propis ADTs utilitzant la paraula clau data.

Exemple: Shape

Anem a definir un ADT personalitzat per a formes geomètriques.

-- Definint un tipus Shape amb constructors Circle i Rectangle
data Shape = Circle Float
           | Rectangle Float Float

-- Funció per calcular l'àrea d'una Shape
area :: Shape -> Float
area (Circle radius) = pi * radius * radius
area (Rectangle width height) = width * height

-- Exemples d'ús del tipus Shape
exampleCircle :: Shape
exampleCircle = Circle 5.0

exampleRectangle :: Shape
exampleRectangle = Rectangle 4.0 6.0

Coincidència de Patrons

La coincidència de patrons és una característica poderosa en Haskell que permet descompondre ADTs i treballar amb els seus components.

Exemple: Coincidència de Patrons amb Shape

Utilitzant el tipus Shape definit anteriorment, podem fer coincidència de patrons per manejar diferents formes.

-- Funció per descriure una Shape
describeShape :: Shape -> String
describeShape (Circle radius) = "A circle with radius " ++ show radius
describeShape (Rectangle width height) = "A rectangle with width " ++ show width ++ " and height " ++ show height

-- Exemples d'ús de describeShape
exampleDescription1 :: String
exampleDescription1 = describeShape exampleCircle

exampleDescription2 :: String
exampleDescription2 = describeShape exampleRectangle

Temes Avançats

A mesura que et familiaritzis més amb els ADTs, pots explorar temes més avançats com:

  • Tipus de Dades Recursius: ADTs que es refereixen a si mateixos.
  • Tipus Parametritzats: ADTs que prenen paràmetres de tipus.
  • Classes de Tipus i ADTs: Utilitzant classes de tipus amb ADTs per a polimorfisme.

Tipus de Dades Recursius

Els tipus de dades recursius s'utilitzen per definir estructures com llistes i arbres.

-- Definint un tipus de Llista recursiu
data List a = Empty
            | Cons a (List a)

-- Funció per calcular la longitud d'una Llista
listLength :: List a -> Int
listLength Empty = 0
listLength (Cons _ tail) = 1 + listLength(tail)

-- Exemple d'ús del tipus List
exampleList :: List Int
exampleList = Cons 1 (Cons 2 (Cons 3 Empty))

Tipus Parametritzats

Els tipus parametritzats permeten que els ADTs siguin més flexibles i reutilitzables.

-- Definint un tipus Pair parametritzat
data Pair a b = Pair a b

-- Exemple d'ús del tipus Pair
examplePair :: Pair Int String
examplePair = Pair 42 "Answer"

Conclusió

Els Tipus de Dades Algebraiques són una característica poderosa en Haskell que permeten definir estructures de dades complexes de manera clara i concisa. Comprenent els tipus producte, els tipus suma i la coincidència de patrons, pots crear programes robustos i expressius. A mesura que avancis, explorar els tipus recursius i parametritzats millorarà encara més la teva capacitat per modelar i resoldre problemes de manera efectiva en Haskell.
