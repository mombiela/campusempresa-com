Introducció a Haskell

Haskell és un llenguatge de programació purament funcional amb tipatge estàtic fort i avaluació mandrosa. Rep el seu nom del lògic Haskell Curry. Haskell és conegut pel seu sistema de tipus expressiu, les seves abstraccions d'alt nivell i la seva sintaxi concisa, cosa que el converteix en una eina poderosa tant per a la investigació acadèmica com per al desenvolupament de programari pràctic.

Característiques Clau de Haskell

  • Purament Funcional: Les funcions en Haskell són pures, és a dir, no tenen efectes secundaris. Això condueix a un codi més previsible i mantenible.
  • Tipatge Estàtic Fort: Haskell utilitza un sistema de tipus robust per detectar errors en temps de compilació, reduint els errors en temps d'execució.
  • Avaluació Mandrosa: Haskell avalua expressions només quan els seus valors són necessaris, cosa que pot conduir a millores de rendiment i la capacitat de treballar amb estructures de dades infinites.
  • Immutabilitat: Les dades en Haskell són immutables per defecte, cosa que simplifica el raonament sobre el codi i millora la concurrència.

Sintaxi i Conceptes Bàsics de Haskell

Exemple de Hello World

Comencem amb un simple programa de "Hello, World!" en Haskell:

main :: IO ()
main = putStrLn "Hello, World!"
  • main és el punt d'entrada d'un programa Haskell.
  • putStrLn és una funció que mostra una cadena seguida d'un salt de línia.
  • IO () indica que main realitza una acció d'entrada/sortida i retorna un tipus unitari ().

Tipus de Dades Bàsics

Haskell té diversos tipus de dades bàsics:

  • Int: Enter de precisió fixa.
  • Integer: Enter de precisió arbitrària.
  • Float: Número de punt flotant de precisió simple.
  • Double: Número de punt flotant de precisió doble.
  • Char: Un sol caràcter.
  • Bool: Tipus booleà amb valors True o False.
  • String: Una llista de caràcters.

Exemple:

myInt :: Int
myInt = 42

myFloat :: Float
myFloat = 3.14

myChar :: Char
myChar = 'a'

myBool :: Bool
myBool = True

myString :: String
myString = "Hello, Haskell!"

Funcions

Les funcions són ciutadans de primera classe en Haskell. Aquí tens com definir i utilitzar funcions:

-- Funció per sumar dos números
add :: Int -> Int -> Int
add x y = x + y

-- Utilitzant la funció
result = add 5 3  -- result és 8
  • add és una funció que pren dos arguments Int i retorna un Int.
  • La signatura de tipus Int -> Int -> Int indica que add pren dos valors Int i retorna un Int.

Llistes

Les llistes són una estructura de dades fonamental en Haskell. Són homogènies, és a dir, tots els elements han de ser del mateix tipus.

-- Una llista d'enters
numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

-- Una llista de caràcters (una cadena)
chars :: [Char]
chars = ['H', 'a', 's', 'k', 'e', 'l', 'l']

Coincidència de Patrons

La coincidència de patrons és una característica poderosa en Haskell que permet desestructurar tipus de dades.

-- Funció per obtenir el cap d'una llista
headOfList :: [a] -> a
headOfList (x:_) = x
headOfList [] = error "Empty list"

-- Utilitzant la funció
firstElement = headOfList [1, 2, 3]  -- firstElement és 1
  • headOfList utilitza la coincidència de patrons per extreure el cap d'una llista.
  • (x:_) coincideix amb una llista no buida, vinculant x al cap.
  • [] coincideix amb una llista buida i genera un error.

Conclusió

Haskell és un llenguatge poderós i expressiu que emfatitza els principis de la programació funcional. El seu sistema de tipus fort, l'avaluació mandrosa i la immutabilitat el converteixen en una eina única i valuosa tant per a aplicacions acadèmiques com pràctiques. Entenent els conceptes bàsics de Haskell, com la seva sintaxi, tipus de dades i característiques clau, pots començar a apreciar l'elegància i el poder de la programació funcional.
