Anotacions de Tipus en Haskell
Tipus de Dades Algebraics ►

Les anotacions de tipus en Haskell són una característica poderosa que et permet especificar explícitament els tipus d'expressions. Això pot ajudar amb la llegibilitat del codi, la depuració i garantir la seguretat de tipus. En aquesta secció, cobrirem els conceptes bàsics de les anotacions de tipus, la seva sintaxi i com utilitzar-les de manera efectiva en els teus programes Haskell.

Introducció a les Anotacions de Tipus

  • Definició: Les anotacions de tipus són declaracions explícites dels tipus de variables o expressions.
  • Propòsit: Ajuden al compilador a entendre quins tipus esperes, cosa que pot detectar errors aviat i fer que el teu codi sigui més llegible.

Sintaxi Bàsica

En Haskell, les anotacions de tipus s'escriuen utilitzant l'operador ::. La sintaxi general és:

variableName :: Type
variableName = expression

Exemple

-- Sense anotació de tipus
add x y = x + y

-- Amb anotació de tipus
add :: Int -> Int -> Int
add x y = x + y

En aquest exemple, add és una funció que pren dos valors Int i retorna un Int.

Anotacions de Tipus Comuns

  • Tipus Bàsics: Int, Float, Double, Char, Bool
  • Llistes: [Type]
  • Tuples: (Type1, Type2, ...)
  • Funcions: Type1 -> Type2 -> ... -> TypeN

Exemples

-- Tipus bàsics
age :: Int
age = 25

piValue :: Double
piValue = 3.14159

isHaskellFun :: Bool
isHaskellFun = True

-- Llistes
numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

-- Tuples
person :: (String, Int)
person = ("Alice", 30)

-- Funcions
multiply :: Int -> Int -> Int
multiply x y = x * y

Anotacions de Tipus Avançades

Tipus Polimòrfics

Els tipus polimòrfics permeten que les funcions operin sobre qualsevol tipus. Es denoten utilitzant variables de tipus, típicament lletres minúscules.

-- Funció polimòrfica
identity :: a -> a
identity x = x

En aquest exemple, identity pot prendre qualsevol tipus a i retornar un valor del mateix tipus.

Funcions d'Ordre Superior

Les funcions d'ordre superior són funcions que prenen altres funcions com a arguments o retornen funcions com a resultats.

-- Funció d'ordre superior
applyTwice :: (a -> a) -> a -> a
applyTwice f x = f (f x)

Aquí, applyTwice pren una funció f i un valor x, i aplica f a x dues vegades.

Classes de Tipus

Les classes de tipus defineixen un conjunt de funcions que poden operar sobre múltiples tipus. Les classes de tipus comunes inclouen Eq, Ord, Show, i Read.

Exemple

-- Utilitzant classes de tipus
showValue :: Show a => a -> String
showValue x = show x

En aquest exemple, showValue pot prendre qualsevol tipus a que sigui una instància de la classe de tipus Show i convertir-lo en una String.

Exercicis Pràctics

  1. Anotació de Tipus Bàsica: Escriu una funció square que prengui un Int i retorni el seu quadrat. Afegeix una anotació de tipus a la funció.

    square :: Int -> Int
square x = x * x

  2. Funció Polimòrfica: Escriu una funció polimòrfica swap que prengui una tupla (a, b) i retorni (b, a).

    swap :: (a, b) -> (b, a)
swap (x, y) = (y, x)

  3. Funció d'Ordre Superior: Escriu una funció d'ordre superior mapTwice que apliqui una funció a cada element d'una llista dues vegades.

    mapTwice :: (a -> a) -> [a] -> [a]
mapTwice f xs = map (f . f) xs

Conclusió

Les anotacions de tipus en Haskell són una eina crucial per escriure codi clar, mantenible i lliure d'errors. En especificar explícitament els tipus, pots ajudar tant al compilador com a altres desenvolupadors a entendre millor el teu codi. A mesura que progressis en Haskell, trobaràs que les anotacions de tipus esdevenen una part integral de la teva pràctica de programació, especialment quan tractis amb tipus complexos i funcions d'ordre superior.
