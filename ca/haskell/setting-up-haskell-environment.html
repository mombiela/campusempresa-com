Configurar l'entorn Haskell
Sintaxi Bàsica i Hola Món ►

Configurar el teu entorn Haskell és el primer pas per aprendre i desenvolupar programes Haskell. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de les eines necessàries per començar amb Haskell.

Instal·lant GHC (Glasgow Haskell Compiler)

El Glasgow Haskell Compiler (GHC) és el compilador Haskell més utilitzat. És essencial per compilar i executar programes Haskell.

Passos per Instal·lar GHC

  1. Descarregar GHC:

    • Visita la pàgina de descàrrega de GHC.
    • Tria la versió adequada per al teu sistema operatiu (Windows, macOS, Linux).

  2. Instal·lar GHC:

    • Windows: Executa l'instal·lador i segueix les instruccions en pantalla.
    • macOS: Utilitza Homebrew per instal·lar GHC executant la comanda: 
      brew install ghc
    • Linux: Utilitza el teu gestor de paquets. Per exemple, a Ubuntu: 
      sudo apt-get install ghc

  3. Verificar la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa la comanda: 
      ghc --version
    • Hauries de veure la versió de GHC que has instal·lat.

Instal·lant Stack

Stack és una eina de construcció per a Haskell que simplifica el procés de gestionar dependències i construir projectes.

Passos per Instal·lar Stack

  1. Descarregar Stack:

    • Visita la pàgina d'instal·lació de Stack.
    • Segueix les instruccions per al teu sistema operatiu.

  2. Instal·lar Stack:

    • Windows: Executa l'instal·lador i segueix les instruccions en pantalla.
    • macOS: Utilitza Homebrew per instal·lar Stack executant la comanda: 
      brew install haskell-stack
    • Linux: Utilitza el teu gestor de paquets. Per exemple, a Ubuntu: 
      sudo apt-get install haskell-stack

  3. Verificar la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa la comanda: 
      stack --version
    • Hauries de veure la versió de Stack que has instal·lat.

Configurant el Teu Primer Projecte Haskell

Ara que tens GHC i Stack instal·lats, pots configurar el teu primer projecte Haskell.

Passos per Crear un Nou Projecte

  1. Crear un Nou Projecte:

    • Obre el teu terminal o símbol del sistema.
    • Executa la comanda: 
      stack new my-first-haskell-project
    • Aquesta comanda crea un nou directori anomenat my-first-haskell-project amb una estructura bàsica de projecte Haskell.

  2. Navegar al Directori del Projecte:

    • Executa la comanda: 
      cd my-first-haskell-project

  3. Construir el Projecte:

    • Executa la comanda: 
      stack build
    • Aquesta comanda descarrega les dependències necessàries i compila el projecte.

  4. Executar el Projecte:

    • Executa la comanda: 
      stack exec my-first-haskell-project-exe
    • Hauries de veure la sortida del programa Haskell per defecte.

Codi Exemple

Aquí tens el codi Haskell per defecte que ve amb el nou projecte:

module Main where

main :: IO ()
main = putStrLn "Hello, Haskell!"
  • Explicació:
    • module Main where: Defineix el mòdul principal del programa.
    • main :: IO (): Declara el tipus de la funció main, que realitza operacions d'entrada/sortida.
    • main = putStrLn "Hello, Haskell!": La funció main imprimeix "Hello, Haskell!" a la consola.

Conclusió

Configurar l'entorn Haskell implica instal·lar GHC i Stack, que són eines essencials per al desenvolupament en Haskell. Seguint els passos descrits en aquesta secció, ara hauries de tenir un entorn Haskell funcional i un projecte bàsic per començar a experimentar amb codi Haskell. Aquesta configuració servirà com a base per al teu viatge des de principiant fins a programador avançat en Haskell.
