Introducció a les Monades
Monads Comuns: Maybe, Either, List ►

Les monades són un concepte fonamental en Haskell que pot ser difícil d'entendre per als principiants. Aquest tema et guiarà a través dels conceptes bàsics de les monades, el seu propòsit i com utilitzar-les efectivament en la programació en Haskell.

Què és una Monada?

Les monades són una classe de tipus en Haskell utilitzada per manejar efectes secundaris i gestionar càlculs d'una manera funcional. Proporcionen una manera d'estructurar programes de manera genèrica.

Conceptes Clau

  • Classe de Tipus: Una classe de tipus defineix un conjunt de funcions que poden ser implementades per diferents tipus.
  • Bind (>>=): Una operació fonamental que aplica una funció a un valor embolicat.
  • Return: Embolica un valor en una monada.

Classe de Tipus Monad

La classe de tipus Monad es defineix en Haskell de la següent manera:

class Applicative m => Monad m where
    (>>=)  :: m a -> (a -> m b) -> m b
    return :: a -> m a

Explicació

  • Applicative m => Monad m: Indica que cada Monad també ha de ser un Applicative.
  • (>>=) :: m a -> (a -> m b) -> m b: La funció bind pren un valor monàdic i una funció que retorna un valor monàdic, i després aplica la funció al valor desempaquetat.
  • return :: a -> m a: La funció return pren un valor i l'embolica en una monada.

Monades Comunes

Maybe Monad

La monada Maybe s'utilitza per gestionar càlculs que poden fallar.

instance Monad Maybe where
    Nothing >>= _ = Nothing
    Just x  >>= f = f x
    return = Just

Exemple

safeDivide :: Double -> Double -> Maybe Double
safeDivide _ 0 = Nothing
safeDivide x y = Just (x / y)

result = Just 10 >>= (\x -> safeDivide x 2) >>= (\y -> safeDivide y 2)
-- resultarà en Just 2.5

List Monad

La monada List s'utilitza per a càlculs no deterministes.

instance Monad [] where
    xs >>= f = concat (map f xs)
    return x = [x]

Exemple

pairs :: [a] -> [b] -> [(a, b)]
pairs xs ys = xs >>= \x -> ys >>= \y -> return (x, y)

result = pairs [1, 2] ['a', 'b']
-- resultarà en [(1,'a'), (1,'b'), (2,'a'), (2,'b')]

IO Monad

La monada IO s'utilitza per gestionar operacions d'entrada/sortida.

instance Monad IO where
    (>>=)  = bindIO
    return = returnIO

Exemple

main :: IO ()
main = do
    putStrLn "Introdueix el teu nom:"
    name <- getLine
    putStrLn ("Hola, " ++ name)

Notació Do

Haskell proporciona sucre sintàctic per a operacions monàdiques anomenat "notació do", que fa que el codi monàdic sigui més llegible.

Exemple amb Maybe Monad

safeDivide :: Double -> Double -> Maybe Double
safeDivide _ 0 = Nothing
safeDivide x y = Just (x / y)

result = do
    x <- Just 10
    y <- safeDivide x 2
    safeDivide y 2
-- resultarà en Just 2.5

Exemple amb IO Monad

main :: IO ()
main = do
    putStrLn "Introdueix el teu nom:"
    name <- getLine
    putStrLn ("Hola, " ++ name)

Resum

Les monades són una abstracció poderosa en Haskell que ajuden a gestionar efectes secundaris i estructurar programes funcionals. Entenent la classe de tipus Monad, monades comunes com Maybe, List i IO, i utilitzant la notació do, pots escriure codi en Haskell més expressiu i mantenible. Practicar amb aquests conceptes aprofundirà la teva comprensió i competència en la programació en Haskell.
