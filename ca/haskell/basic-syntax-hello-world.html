Introducció

Benvingut al primer mòdul del nostre curs de Haskell! En aquesta secció, cobrirem la sintaxi bàsica de Haskell i et guiarem a través de l'escriptura del teu primer programa "Hola Món". Al final d'aquesta secció, tindràs una comprensió fonamental de la sintaxi de Haskell i de com executar un programa senzill.

Conceptes clau

  • Conceptes bàsics de Haskell: Comprendre l'estructura i la sintaxi del codi Haskell.
  • Comentaris: Com escriure comentaris en Haskell.
  • Programa Hola Món: Escriure i executar el teu primer programa Haskell.

Conceptes bàsics de Haskell

Haskell és un llenguatge de programació purament funcional amb un sistema de tipus estàtic fort. Aquí hi ha alguns aspectes fonamentals de la sintaxi de Haskell:

  • Mòduls: El codi Haskell està organitzat en mòduls.
  • Funcions: Les funcions són ciutadans de primera classe en Haskell.
  • Tipus: Haskell té un sistema de tipus estàtic fort.
  • Expressions: Tot en Haskell és una expressió.

Exemple: Estructura bàsica

-- Aquest és un comentari d'una sola línia

{-
Aquest és un comentari de múltiples línies
-}

module Main where

-- Definir una funció
add :: Int -> Int -> Int
add x y = x + y

-- Funció principal
main :: IO ()
main = do
    let result = add 5 3
    putStrLn ("El resultat és " ++ show result)

Explicació

  • Comentaris: Els comentaris d'una sola línia comencen amb --, i els comentaris de múltiples línies estan tancats entre {- -}.
  • Declaració de mòdul: module Main where declara un mòdul anomenat Main.
  • Definició de funció: add :: Int -> Int -> Int defineix una funció add que pren dos enters i retorna un enter.
  • Funció principal: main :: IO () és el punt d'entrada del programa. La notació do s'utilitza per seqüenciar accions d'entrada/sortida.

Escriure el teu primer programa "Hola Món"

Escrivim un senzill programa "Hola Món" en Haskell.

Exemple: Hola Món

module Main where

main :: IO ()
main = putStrLn "Hola, Món!"

Explicació

  • Declaració de mòdul: module Main where declara el mòdul principal.
  • Funció principal: main :: IO () és la funció principal que realitza accions d'entrada/sortida.
  • Sortida: putStrLn "Hola, Món!" imprimeix "Hola, Món!" a la consola.

Executar el programa

Per executar el teu programa Haskell, segueix aquests passos:

  1. Desa el codi en un fitxer anomenat Main.hs.
  2. Obre un terminal i navega al directori que conté Main.hs.
  3. Compila el programa utilitzant el compilador GHC: 
    ghc -o hello Main.hs
  4. Executa el programa compilat: 
    ./hello

Conclusió

En aquesta secció, hem cobert la sintaxi bàsica de Haskell i hem escrit un senzill programa "Hola Món". Has après sobre el sistema de mòduls de Haskell, les definicions de funcions i com realitzar operacions bàsiques d'entrada/sortida. Aquest coneixement fonamental serà essencial a mesura que avancem cap a temes més avançats en Haskell. Continua practicant, i aviat et sentiràs còmode amb la sintaxi de Haskell i el paradigma de programació funcional!
