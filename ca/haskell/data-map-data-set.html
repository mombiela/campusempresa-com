Data.Map i Data.Set en Haskell
Introducció als Functors ►

Introducció

En aquesta secció, explorarem dos mòduls essencials de la biblioteca estàndard de Haskell: Data.Map i Data.Set. Aquests mòduls proporcionen implementacions eficients de mapes (diccionaris) i conjunts, respectivament. Entendre com utilitzar aquestes estructures de dades és crucial per escriure codi Haskell eficient i idiomàtic.

Data.Map

Conceptes Clau

  • Mapa: Una estructura de dades associativa que mapeja claus a valors.
  • Eficiència: Data.Map està implementat utilitzant arbres binaris equilibrats, proporcionant una complexitat temporal logarítmica per a la majoria d'operacions.
  • Immutabilitat: Com totes les estructures de dades de Haskell, els mapes són immutables.

Operacions Bàsiques

Aquí hi ha algunes operacions bàsiques que pots realitzar amb Data.Map:

  • Crear un Mapa: Utilitza fromList per crear un mapa a partir d'una llista de parells clau-valor.
  • Inserir Elements: Utilitza insert per afegir un parell clau-valor a un mapa.
  • Buscar Valors: Utilitza lookup per recuperar un valor per la seva clau.
  • Eliminar Elements: Utilitza delete per eliminar un parell clau-valor d'un mapa.

Exemples de Codi

Crear i Manipular un Mapa

import qualified Data.Map as Map

-- Crear un mapa a partir d'una llista de parells clau-valor
let phoneBook = Map.fromList [("Alice", "1234"), ("Bob", "5678"), ("Charlie", "91011")]

-- Inserir una nova entrada
let updatedPhoneBook = Map.insert "David" "1213" phoneBook

-- Buscar un valor
case Map.lookup "Alice" updatedPhoneBook of
    Just number -> putStrLn $ "El número de l'Alice és " ++ number
    Nothing -> putStrLn "L'Alice no és a l'agenda"

-- Eliminar una entrada
let finalPhoneBook = Map.delete "Bob" updatedPhoneBook

Operacions Avançades

  • Unió i Intersecció: Combina mapes utilitzant union i intersection.
  • Map i Filtre: Aplica funcions a tots els valors utilitzant map i filtra entrades utilitzant filterWithKey.

Exemple de Codi Avançat

import qualified Data.Map as Map

let map1 = Map.fromList [(1, "one"), (2, "two")]
let map2 = Map.fromList [(2, "dos"), (3, "three")]

-- Unió de dos mapes
let unionMap = Map.union map1 map2

-- Intersecció de dos mapes
let intersectionMap = Map.intersection map1 map2

-- Map d'una funció sobre tots els valors
let upperCaseMap = Map.map (map toUpper) map1

-- Filtrar entrades
let filteredMap = Map.filterWithKey (\k _ -> k > 1) map1

Data.Set

Conceptes Clau

  • Conjunt: Una col·lecció d'elements únics.
  • Eficiència: Data.Set també està implementat utilitzant arbres binaris equilibrats, proporcionant una complexitat temporal logarítmica per a la majoria d'operacions.
  • Immutabilitat: Els conjunts en Haskell són immutables.

Operacions Bàsiques

Aquí hi ha algunes operacions bàsiques que pots realitzar amb Data.Set:

  • Crear un Conjunt: Utilitza fromList per crear un conjunt a partir d'una llista d'elements.
  • Inserir Elements: Utilitza insert per afegir un element a un conjunt.
  • Comprovar Membresia: Utilitza member per comprovar si un element és en un conjunt.
  • Eliminar Elements: Utilitza delete per eliminar un element d'un conjunt.

Exemples de Codi

Crear i Manipular un Conjunt

import qualified Data.Set as Set

-- Crear un conjunt a partir d'una llista d'elements
let mySet = Set.fromList [1, 2, 3, 4]

-- Inserir un nou element
let updatedSet = Set.insert 5 mySet

-- Comprovar membresia
if Set.member 3 updatedSet
    then putStrLn "3 és al conjunt"
    else putStrLn "3 no és al conjunt"

-- Eliminar un element
let finalSet = Set.delete 2 updatedSet

Operacions Avançades

  • Unió i Intersecció: Combina conjunts utilitzant union i intersection.
  • Diferència: Troba la diferència entre dos conjunts utilitzant difference.
  • Map i Filtre: Aplica funcions a tots els elements utilitzant map i filtra elements utilitzant filter.

Exemple de Codi Avançat

import qualified Data.Set as Set

let set1 = Set.fromList [1, 2, 3]
let set2 = Set.fromList [2, 3, 4]

-- Unió de dos conjunts
let unionSet = Set.union set1 set2

-- Intersecció de dos conjunts
let intersectionSet = Set.intersection set1 set2

-- Diferència entre dos conjunts
let differenceSet = Set.difference set1 set2

-- Map d'una funció sobre tots els elements
let squaredSet = Set.map (^2) set1

-- Filtrar elements
let filteredSet = Set.filter (>2) set1

Conclusió

En aquesta secció, hem cobert les bases i algunes característiques avançades de Data.Map i Data.Set en Haskell. Aquestes estructures de dades són eines poderoses per gestionar col·leccions de dades de manera eficient. Entenent com utilitzar mapes i conjunts, pots escriure codi Haskell més eficient i expressiu. Practica utilitzant aquests mòduls amb diferents tipus de dades per sentir-te més còmode amb les seves operacions i capacitats.
