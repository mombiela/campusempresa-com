Project: Construir una Aplicació Haskell
Cabal i Stack ►

Introducció

En aquesta secció, emprendrem un viatge per construir una aplicació Haskell completa des de zero. Aquest projecte t'ajudarà a consolidar els teus coneixements de Haskell i et donarà experiència pràctica en el desenvolupament d'una aplicació del món real.

Visió General del Projecte

  • Objectiu: Construir una aplicació de llista de tasques de línia de comandes simple.
  • Funcionalitats:
    • Afegir una nova tasca.
    • Llistar totes les tasques.
    • Marcar una tasca com a completada.
    • Esborrar una tasca.
  • Eines: GHC (Glasgow Haskell Compiler), Stack (eina de construcció de Haskell).

Configuració de l'Entorn

Abans de començar a codificar, configurem el nostre entorn de desenvolupament.

Instal·lació de GHC i Stack

  1. GHC: El Glasgow Haskell Compiler és el compilador Haskell més utilitzat.
  2. Stack: Un programa multiplataforma per desenvolupar projectes Haskell.

Passos d'Instal·lació

  • Windows: 
    choco install haskell-stack
  • macOS: 
    brew install haskell-stack
  • Linux: 
    curl -sSL https://get.haskellstack.org/ | sh

Creació d'un Nou Projecte

Utilitza Stack per crear un nou projecte Haskell:

stack new todo-app simple
cd todo-app
stack setup

Disseny de l'Aplicació

Anem a esbossar l'estructura de la nostra aplicació.

Model de Dades

Definirem un tipus de dades Task per representar cada element de la llista de tasques.

data Task = Task {
    taskId :: Int,
    taskDescription :: String,
    taskCompleted :: Bool
} deriving (Show, Read)

Funcions Principals

  • Afegir Tasca: Afegir una nova tasca a la llista.
  • Llistar Tasques: Mostrar totes les tasques.
  • Completar Tasca: Marcar una tasca com a completada.
  • Esborrar Tasca: Eliminar una tasca de la llista.

Implementació de l'Aplicació

Menú Principal

mainMenu :: IO ()
mainMenu = do
    putStrLn "1. Afegir Tasca"
    putStrLn "2. Llistar Tasques"
    putStrLn "3. Completar Tasca"
    putStrLn "4. Esborrar Tasca"
    putStrLn "5. Sortir"
    putStr "Tria una opció: "
    hFlush stdout
    option <- getLine
    handleOption option

Gestió de l'Entrada de l'Usuari

handleOption :: String -> IO ()
handleOption option = case option of
    "1" -> addTask
    "2" -> listTasks
    "3" -> completeTask
    "4" -> deleteTask
    "5" -> putStrLn "Adéu!"
    _   -> putStrLn "Opció invàlida!" >> mainMenu
