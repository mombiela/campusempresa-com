Introducció a la Implementació en Haskell

En aquesta secció, explorarem com implementar diversos conceptes i funcionalitats en Haskell. Començarem amb implementacions bàsiques i gradualment avançarem cap a temes més avançats. Al final d'aquesta secció, hauries de sentir-te còmode implementant funcions, estructures de dades i algorismes en Haskell.

Implementació Bàsica de Funcions

Haskell és un llenguatge de programació funcional, i les funcions són els blocs de construcció principals. Comencem amb algunes implementacions bàsiques de funcions.

Funcions Simples

Una funció simple en Haskell es pot definir utilitzant la següent sintaxi:

add :: Int -> Int -> Int
add x y = x + y
  • add és el nom de la funció.
  • :: especifica la signatura de tipus.
  • Int -> Int -> Int significa que la funció pren dos enters i retorna un enter.
  • add x y = x + y és la definició de la funció.

Funcions Recursives

La recursió és una tècnica comuna en Haskell per iterar sobre estructures de dades.

factorial :: Int -> Int
factorial 0 = 1
factorial n = n * factorial (n - 1)
  • factorial 0 = 1 és el cas base.
  • factorial n = n * factorial (n - 1) és el cas recursiu.

Funcions d'Ordre Superior

Les funcions que prenen altres funcions com a arguments o que retornen funcions es diuen funcions d'ordre superior.

applyTwice :: (a -> a) -> a -> a
applyTwice f x = f (f x)
  • applyTwice pren una funció f i un valor x, i aplica f a x dues vegades.

Estructures de Dades

Haskell proporciona diverses estructures de dades integrades, i també pots definir les teves pròpies.

Llistes

Les llistes són una estructura de dades fonamental en Haskell.

sumList :: [Int] -> Int
sumList [] = 0
sumList (x:xs) = x + sumList xs
  • sumList [] = 0 és el cas base per a una llista buida.
  • sumList (x:xs) = x + sumList xs suma el cap x amb la suma de la cua xs.

Tuples

Els tuples s'utilitzen per emmagatzemar col·leccions de mida fixa d'elements.

pairSum :: (Int, Int) -> Int
pairSum (x, y) = x + y
  • pairSum pren un tuple de dos enters i retorna la seva suma.

Tipus de Dades Personalitzats

Pots definir els teus propis tipus de dades utilitzant la paraula clau data.

data Shape = Circle Float | Rectangle Float Float

area :: Shape -> Float
area (Circle r) = pi * r * r
area (Rectangle w h) = w * h
  • Shape és un tipus de dades personalitzat amb dos constructors: Circle i Rectangle.
  • area calcula l'àrea d'una Shape donada.

Implementacions Avançades

A mesura que et sentis més còmode amb Haskell, pots començar a implementar funcionalitats més complexes.

Monads

Les monads són un concepte poderós en Haskell per gestionar efectes secundaris.

import Control.Monad

safeDivide :: Float -> Float -> Maybe Float
safeDivide _ 0 = Nothing
safeDivide x y = Just (x / y)

example :: Maybe Float
example = do
  a <- safeDivide 10 2
  b <- safeDivide a 2
  return b
  • safeDivide realitza una divisió i retorna Nothing si el denominador és zero.
  • example utilitza la notació do per encadenar operacions que retornen valors Maybe.

Avaluació Peresosa

Haskell utilitza l'avaluació peresosa, el que significa que les expressions no es calculen fins que els seus valors són necessaris.

infiniteList :: [Int]
infiniteList = [1..]

takeTen :: [Int]
takeTen = take 10 infiniteList
  • infiniteList és una llista infinita d'enters que comença des de l'1.
  • takeTen pren els primers 10 elements de infiniteList.

Resum

En aquesta secció, hem cobert els conceptes bàsics d'implementació de funcions i estructures de dades en Haskell. Hem començat amb funcions simples i recursió, hem passat a funcions d'ordre superior, i hem explorat estructures de dades integrades i personalitzades. Finalment, hem tocat temes avançats com les monads i l'avaluació peresosa. Amb aquestes eines, hauries d'estar ben equipat per implementar una àmplia gamma de funcionalitats en Haskell.
