Introducció a les Estructures de Dades Recursives

Les estructures de dades recursives són fonamentals en la programació funcional, especialment en Haskell. Ens permeten definir tipus de dades complexos utilitzant instàncies més simples del mateix tipus. Aquesta secció introduirà el concepte i proporcionarà una base per entendre temes més avançats.

Conceptes Clau

  • Recursió: Una funció que es crida a si mateixa.
  • Cas Base: La condició sota la qual la recursió s'atura.
  • Cas Recursiu: La part de la funció que inclou la crida recursiva.
  • Estructures de Dades: Maneres d'emmagatzemar i organitzar dades, com ara llistes, arbres i més.

Llistes com a Estructures de Dades Recursives

Les llistes són una de les estructures de dades recursives més simples i més utilitzades en Haskell.

Definició i Construcció

Una llista en Haskell és o bé buida o bé consisteix en un element cap seguit d'una cua, que és ella mateixa una llista.

data List a = Empty | Cons a (List a)

Exemple: Suma d'una Llista

Aquí hi ha una funció recursiva simple per calcular la suma d'una llista d'enters.

sumList :: [Int] -> Int
sumList [] = 0
sumList (x:xs) = x + sumList(xs)
  • Cas Base: sumList [] = 0 (La suma d'una llista buida és 0).
  • Cas Recursiu: sumList (x:xs) = x + sumList(xs) (La suma d'una llista és l'element cap més la suma de la cua).

Arbres com a Estructures de Dades Recursives

Els arbres són una altra estructura de dades recursiva comuna. Consisteixen en nodes, on cada node pot tenir zero o més nodes fills.

Definició i Construcció

Un arbre binari és un arbre on cada node té com a màxim dos fills.

data Tree a = EmptyTree | Node a (Tree a) (Tree a)

Exemple: Profunditat d'un Arbre

Aquí hi ha una funció recursiva per calcular la profunditat d'un arbre binari.

treeDepth :: Tree a -> Int
treeDepth EmptyTree = 0
treeDepth (Node _ left right) = 1 + max (treeDepth left) (treeDepth right)
  • Cas Base: treeDepth EmptyTree = 0 (La profunditat d'un arbre buit és 0).
  • Cas Recursiu: treeDepth (Node _ left right) = 1 + max (treeDepth left) (treeDepth right) (La profunditat d'un arbre és 1 més la profunditat màxima dels seus subarbres).

Estructures de Dades Recursives Avançades

A mesura que et sentis més còmode amb les estructures de dades recursives bàsiques, pots explorar-ne de més complexes com els gràfics i els tipus de dades personalitzats.

Exemple: Tipus de Dades Recursiu Personalitzat

Definim un tipus de dades recursiu personalitzat per a un arbre d'expressions simple.

data Expr = Val Int | Add Expr Expr | Mul Expr Expr

Exemple: Avaluació d'un Arbre d'Expressions

Aquí hi ha una funció per avaluar un arbre d'expressions.

eval :: Expr -> Int
eval (Val n) = n
eval (Add x y) = eval x + eval y
eval (Mul x y) = eval x * eval y
  • Cas Base: eval (Val n) = n (El valor d'un node fulla és l'enter que conté).
  • Cas Recursiu: eval (Add x y) = eval x + eval y i eval (Mul x y) = eval x * eval y (Avaluar els subarbres i aplicar l'operació).

Conclusió

Les estructures de dades recursives són una eina poderosa en Haskell, permetent definicions elegants i concises de tipus de dades i operacions complexes. Entenent les bases de les llistes i els arbres, i explorant estructures més avançades, pots aprofitar la recursió per resoldre una àmplia gamma de problemes en la programació funcional.

Resum

  • Llistes: Estructura recursiva simple amb un cas base (llista buida) i un cas recursiu (cap i cua).
  • Arbres: Estructura més complexa amb nodes i subarbres.
  • Tipus Personalitzats: Defineix les teves pròpies estructures de dades recursives per a problemes específics.
  • Recursió: Tècnica clau per processar estructures de dades recursives, amb casos base i recursius clars.

Dominant aquests conceptes, estaràs ben preparat per afrontar una varietat de desafiaments de programació en Haskell.
