Introducció al Currying i l'Aplicació Parcial

El currying i l'aplicació parcial són conceptes fonamentals en Haskell que permeten un codi més flexible i modular. Permeten que les funcions s'apliquin en etapes, cosa que pot conduir a un codi més llegible i reutilitzable.

Conceptes Clau

  • Currying: El procés de transformar una funció que pren múltiples arguments en una seqüència de funcions, cadascuna prenent un sol argument.
  • Aplicació Parcial: El procés de fixar alguns arguments d'una funció, produint una altra funció de menor aritat.

Entenent el Currying

El currying porta el nom del matemàtic Haskell Curry. En Haskell, totes les funcions són curried per defecte.

Exemple de Currying

Considera una funció simple que suma dos números:

add :: Int -> Int -> Int
add x y = x + y

Aquesta funció es pot currificar:

addCurried :: Int -> (Int -> Int)
addCurried x = \y -> x + y

En Haskell, add i addCurried són essencialment el mateix a causa del currying inherent de Haskell.

Utilitzant Funcions Curried

Pots aplicar arguments un a un:

let addFive = add 5
addFive 3  -- Resultat: 8

Aquí, add 5 retorna una nova funció que suma 5 al seu argument.

Aplicació Parcial

L'aplicació parcial està estretament relacionada amb el currying. Implica aplicar una funció a alguns dels seus arguments, produint una altra funció que pren els arguments restants.

Exemple d'Aplicació Parcial

Utilitzant la mateixa funció add:

let addTen = add 10
addTen 7  -- Resultat: 17

Aquí, add 10 és una funció parcialment aplicada que suma 10 a qualsevol número donat.

Usos Pràctics

L'aplicació parcial és útil per crear funcions especialitzades a partir de funcions més generals:

multiply :: Int -> Int -> Int
multiply x y = x * y

let double = multiply 2
double 8  -- Resultat: 16

let triple = multiply 3
triple 5  -- Resultat: 15

Comparació del Currying i l'Aplicació Parcial

| Concepte | Descripció | Exemple | |--------------------|-----------------------------------------------------------------------------|----------------------------------------------| | Currying | Transformar una funció amb múltiples arguments en una seqüència de funcions | addCurried x = \y -> x + y | | Aplicació Parcial | Aplicar una funció a alguns dels seus arguments per obtenir una nova funció | let addFive = add 5 |

Ús Avançat

Funcions d'Ordre Superior

El currying i l'aplicació parcial brillen quan s'utilitzen amb funcions d'ordre superior:

map (add 3) [1, 2, 3]  -- Resultat: [4, 5, 6]

Aquí, add 3 és parcialment aplicada i després mapejada sobre una llista.

Composició de Funcions

Pots composar funcions curried per construir operacions més complexes:

let increment = add 1
let double = multiply 2
let incrementThenDouble = double . increment

incrementThenDouble 3  -- Resultat: 8

Exercicis

  1. Pràctica de Currying: Escriu una funció curried que concateni tres cadenes. 
    concatThree :: String -> String -> String -> String
concatThree a b c = a ++ b ++ c
  2. Pràctica d'Aplicació Parcial: Crea una funció parcialment aplicada a partir d'una funció curried que multiplica tres números. 
    multiplyThree :: Int -> Int -> Int -> Int
multiplyThree x y z = x * y * z

let multiplyByTwoAndThree = multiplyThree 2 3
multiplyByTwoAndThree 4  -- Resultat: 24

Conclusió

El currying i l'aplicació parcial són tècniques poderoses en Haskell que milloren la modularitat i la reutilització del codi. Entenent i utilitzant aquests conceptes, pots escriure programes Haskell més concisos i flexibles. Practica aquestes tècniques amb diverses funcions per esdevenir proficient en el seu ús.
