Funcions d'ordre superior en Haskell
Expressions Lambda ►

Les funcions d'ordre superior són un concepte fonamental en Haskell i en la programació funcional en general. Són funcions que poden prendre altres funcions com a arguments i/o retornar funcions com a resultats. Aquest tema et guiarà des dels conceptes bàsics fins a usos més avançats de les funcions d'ordre superior en Haskell.

Introducció a les Funcions d'Ordre Superior

  • Definició: Una funció d'ordre superior és una funció que:

    • Pren una o més funcions com a arguments.
    • Retorna una funció com a resultat.

  • Importància: Les funcions d'ordre superior permeten un codi més abstracte i flexible. Permeten patrons poderosos com la composició de funcions, el currying, i més.

Exemples Bàsics

Comencem amb un exemple senzill per il·lustrar el concepte:

-- Una funció que pren una funció i un valor, i aplica la funció al valor
applyTwice :: (a -> a) -> a -> a
applyTwice f x = f (f x)

-- Exemple d'ús
double :: Int -> Int
double x = x * 2

main = print (applyTwice double 3)  -- Sortida: 12

En aquest exemple:

  • applyTwice és una funció d'ordre superior perquè pren una funció f com a argument.
  • double és una funció senzilla que multiplica la seva entrada per 2.
  • applyTwice double 3 aplica double a 3 dues vegades, resultant en 12.

Funcions d'Ordre Superior Comunes

`map`

  • Descripció: Aplica una funció a cada element d'una llista.
  • Signatura de Tipus: (a -> b) -> [a] -> [b]
-- Exemple d'ús de map
increment :: Int -> Int
increment x = x + 1

main = print (map increment [1, 2, 3, 4])  -- Sortida: [2, 3, 4, 5]

`filter`

  • Descripció: Selecciona elements d'una llista que satisfan un predicat.
  • Signatura de Tipus: (a -> Bool) -> [a] -> [a]
-- Exemple d'ús de filter
isEven :: Int -> Bool
isEven x = x `mod` 2 == 0

main = print (filter isEven [1, 2, 3, 4, 5, 6])  -- Sortida: [2, 4, 6]

`foldl` i `foldr`

  • Descripció: Redueix una llista a un sol valor combinant elements utilitzant una funció binària.
  • Signatura de Tipus:
    • foldl: (b -> a -> b) -> b -> [a] -> b
    • foldr: (a -> b -> b) -> b -> [a] -> b
-- Exemple d'ús de foldl
sumList :: [Int] -> Int
sumList = foldl (+) 0

main = print (sumList [1, 2, 3, 4])  -- Sortida: 10

Composició de Funcions

  • Descripció: Combinació de dues o més funcions per formar una nova funció.
  • Operador: (.)
  • Signatura de Tipus: (b -> c) -> (a -> b) -> a -> c
-- Exemple d'ús de la composició de funcions
addOne :: Int -> Int
addOne x = x + 1

multiplyByTwo :: Int -> Int
multiplyByTwo x = x * 2

-- Composa addOne i multiplyByTwo
composedFunction = addOne . multiplyByTwo

main = print (composedFunction 3)  -- Sortida: 7

Currying

  • Descripció: Transformació d'una funció que pren múltiples arguments en una sèrie de funcions que cadascuna pren un sol argument.
  • Exemple:
-- Una funció que suma dos números
add :: Int -> Int -> Int
add x y = x + y

-- Versió curried
addFive :: Int -> Int
addFive = add 5

main = print (addFive 3)  -- Sortida: 8

Ús Avançat

Creació de Funcions d'Ordre Superior Personalitzades

  • Exemple: Una funció que pren una funció i l'aplica tres vegades.
-- Una funció que aplica una funció donada tres vegades
applyThrice :: (a -> a) -> a -> a
applyThrice f x = f (f (f x))

-- Exemple d'ús
main = print (applyThrice double 2)  -- Sortida: 16

Ús de Funcions d'Ordre Superior amb Estructures de Dades

  • Exemple: Ús de map amb una estructura de dades personalitzada.
-- Definició d'una estructura de dades personalitzada
data Tree a = Leaf a | Node (Tree a) (Tree a) deriving Show

-- Definició d'una funció map per al Tree
treeMap :: (a -> b) -> Tree a -> Tree b
treeMap f (Leaf x) = Leaf (f x)
treeMap f (Node left right) = Node (treeMap f left) (treeMap f right)

-- Exemple d'ús
main = print (treeMap increment (Node (Leaf 1) (Leaf 2)))  -- Sortida: Node (Leaf 2) (Leaf 3)

Conclusió

Les funcions d'ordre superior són una característica poderosa de Haskell que permeten una programació més abstracta i flexible. Entenent i utilitzant les funcions d'ordre superior, pots escriure codi més concís, reutilitzable i expressiu. Aquest tema ha cobert els conceptes bàsics de les funcions d'ordre superior, funcions d'ordre superior comunes com map, filter, i fold, així com conceptes més avançats com la composició de funcions i el currying. Practica utilitzant aquests conceptes per convertir-te en un expert en programació funcional amb Haskell.
