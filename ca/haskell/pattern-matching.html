Coincidència de patrons en Haskell
Immutabilitat ►

La coincidència de patrons és una característica poderosa en Haskell que et permet comprovar un valor contra un patró. És un concepte fonamental que s'utilitza extensament en la programació en Haskell per descompondre estructures de dades, controlar el flux i més.

Introducció a la coincidència de patrons

La coincidència de patrons en Haskell s'utilitza per:

  • Descompondre estructures de dades.
  • Simplificar el codi gestionant diferents casos.
  • Millorar la llegibilitat i mantenibilitat.

Coincidència de patrons bàsica

En la seva forma més simple, la coincidència de patrons es pot utilitzar en definicions de funcions per coincidir amb valors específics.

-- Funció per comprovar si un número és zero
isZero :: Int -> Bool
isZero 0 = True
isZero _ = False

En aquest exemple:

  • isZero 0 = True coincideix amb l'entrada 0 i retorna True.
  • isZero _ = False utilitza el patró comodí _ per coincidir amb qualsevol altre valor i retorna False.

Coincidència de patrons amb llistes

Les llistes són una estructura de dades comuna en Haskell, i la coincidència de patrons es pot utilitzar per descompondre-les.

-- Funció per obtenir el cap d'una llista
head' :: [a] -> a
head' [] = error "Empty list"
head' (x:_) = x

En aquest exemple:

  • head' [] = error "Empty list" coincideix amb una llista buida i llança un error.
  • head' (x:_) = x coincideix amb una llista no buida, assigna el cap de la llista a x i el retorna.

Coincidència de patrons amb tuples

Els tuples també es poden descompondre utilitzant la coincidència de patrons.

-- Funció per sumar dos parells d'enters
addPairs :: (Int, Int) -> (Int, Int) -> (Int, Int)
addPairs (a, b) (c, d) = (a + c, b + d)

En aquest exemple:

  • addPairs (a, b) (c, d) = (a + c, b + d) coincideix amb dos parells d'enters i retorna un nou parell amb les seves sumes.

Coincidència de patrons avançada

Guàrdies

Les guàrdies proporcionen una manera d'afegir condicions més complexes a la coincidència de patrons.

-- Funció per classificar un número
classifyNumber :: Int -> String
classifyNumber n
  | n < 0     = "Negative"
  | n == 0    = "Zero"
  | otherwise = "Positive"

En aquest exemple:

  • Les guàrdies (|) s'utilitzen per comprovar múltiples condicions.
  • otherwise és una guàrdia de captura que coincideix amb qualsevol valor no capturat per les guàrdies anteriors.

Patrons As

Els patrons As et permeten mantenir una referència a tot el valor mentre coincideixes amb els seus components.

-- Funció per duplicar el primer element d'una llista
duplicateHead :: [a] -> [a]
duplicateHead xs@(x:_) = x : xs
duplicateHead [] = []

En aquest exemple:

  • xs@(x:_) coincideix amb una llista no buida, assigna tota la llista a xs i el cap a x.
  • La funció retorna una nova llista amb el cap duplicat.

Coincidència de patrons en expressions case

Les expressions case proporcionen una manera d'utilitzar la coincidència de patrons dins d'expressions.

-- Funció per descriure una llista
describeList :: [a] -> String
describeList xs = case xs of
  []      -> "Empty list"
  [x]     -> "Single element list"
  (x:xs)  -> "Longer list"

En aquest exemple:

  • case xs of s'utilitza per coincidir la llista xs contra diferents patrons.

Resum

La coincidència de patrons és una característica versàtil i poderosa en Haskell que simplifica el codi i millora la llegibilitat. En dominar la coincidència de patrons, pots escriure programes en Haskell més concisos i expressius. Els conceptes clau inclouen:

  • Coincidència de patrons bàsica en definicions de funcions.
  • Descomposició de llistes i tuples.
  • Ús de guàrdies per a condicions complexes.
  • Patrons As per referenciar valors complets.
  • Expressions case per a coincidència de patrons en línia.

Entendre i utilitzar efectivament la coincidència de patrons millorarà significativament les teves habilitats de programació en Haskell.
