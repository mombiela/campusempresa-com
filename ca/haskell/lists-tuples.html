Llistes i Tuples en Haskell
Coincidència de Patrons ►

Introducció

En Haskell, les llistes i tuples són estructures de dades fonamentals que et permeten emmagatzemar i manipular col·leccions de valors. Aquesta secció cobrirà els conceptes bàsics de les llistes i tuples, les seves diferències i com utilitzar-les de manera efectiva en la programació en Haskell.

Llistes

Les llistes en Haskell són col·leccions homogènies, és a dir, tots els elements han de ser del mateix tipus. Són una de les estructures de dades més comunament utilitzades en Haskell.

Creació de Llistes

Pots crear llistes utilitzant claudàtors [] i separant els elements amb comes.

-- Una llista buida
emptyList :: [Int]
emptyList = []

-- Una llista d'enters
intList :: [Int]
intList = [1, 2, 3, 4, 5]

-- Una llista de caràcters (String)
charList :: [Char]
charList = ['H', 'e', 'l', 'l', 'o']

Operacions amb Llistes

Haskell proporciona diverses funcions integrades per manipular llistes.

Concatenació

Pots concatenar dues llistes utilitzant l'operador ++.

list1 :: [Int]
list1 = [1, 2, 3]

list2 :: [Int]
list2 = [4, 5, 6]

combinedList :: [Int]
combinedList = list1 ++ list2  -- [1, 2, 3, 4, 5, 6]

Cap i Cua

La funció head retorna el primer element d'una llista, i la funció tail retorna la llista sense el seu primer element.

firstElement :: Int
firstElement = head [1, 2, 3, 4]  -- 1

restOfList :: [Int]
restOfList = tail [1, 2, 3, 4]  -- [2, 3, 4]

Comprensions de Llistes

Les comprensions de llistes proporcionen una manera concisa de crear llistes.

-- Una llista de quadrats de nombres de l'1 al 5
squares :: [Int]
squares = [x * x | x <- [1..5]]  -- [1, 4, 9, 16, 25]

-- Una llista de nombres parells de l'1 al 10
evens :: [Int]
evens = [x | x <- [1..10], even x]  -- [2, 4, 6, 8, 10]

Tuples

Les tuples en Haskell són col·leccions de mida fixa que poden contenir elements de diferents tipus. Són útils per agrupar valors relacionats.

Creació de Tuples

Les tuples es creen utilitzant parèntesis () i separant els elements amb comes.

-- Una tupla de dos enters
pair :: (Int, Int)
pair = (1, 2)

-- Una tupla d'un enter, un caràcter i una cadena
mixedTuple :: (Int, Char, String)
mixedTuple = (1, 'a', "Hello")

Accés als Elements de la Tupla

Pots accedir als elements de la tupla utilitzant el patró de coincidència.

-- Patró de coincidència per extreure elements d'una tupla
extractElements :: (Int, Char, String) -> (Int, Char, String)
extractElements (x, y, z) = (x, y, z)

-- Exemple d'ús
exampleTuple :: (Int, Char, String)
exampleTuple = (1, 'a', "Hello")

-- Elements extrets
(x, y, z) = extractElements exampleTuple  -- x = 1, y = 'a', z = "Hello"

Tuples Imbricades

Les tuples poden estar imbricades dins d'altres tuples.

-- Una tupla imbricada
nestedTuple :: ((Int, Char), String)
nestedTuple = ((1, 'a'), "Hello")

-- Accés als elements d'una tupla imbricada
extractNested :: ((Int, Char), String) -> (Int, Char, String)
extractNested ((x, y), z) = (x, y, z)

-- Exemple d'ús
(x, y, z) = extractNested nestedTuple  -- x = 1, y = 'a', z = "Hello"

Llistes vs. Tuples

| Característica | Llistes | Tuples | |------------------|-------------------------------|-------------------------------| | Homogeneïtat | Tots els elements han de ser del mateix tipus | Els elements poden ser de diferents tipus | | Mida | Dinàmica, pot créixer o disminuir | Mida fixa | | Sintaxi | Claudàtors [] | Parèntesis () | | Usos Comuns | Seqüències, col·leccions | Agrupació de valors relacionats |

Conclusió

Les llistes i tuples són estructures de dades essencials en Haskell. Les llistes són ideals per manejar seqüències d'elements del mateix tipus, mentre que les tuples són útils per agrupar un nombre fix d'elements de tipus potencialment diferents. Entendre com crear i manipular aquestes estructures millorarà significativament la teva capacitat per escriure programes efectius en Haskell.
