Introducció a la Gestió de Fitxers

La gestió de fitxers és un aspecte crucial de la programació que et permet llegir i escriure fitxers. En Haskell, la gestió de fitxers es realitza utilitzant el monad IO, que encapsula els efectes secundaris i assegura que es mantingui la puresa funcional del llenguatge.

Conceptes Bàsics

  • Camins de Fitxers: La ubicació d'un fitxer en el sistema de fitxers.
  • Modes de Fitxer: Modes en què es pot obrir un fitxer, com ara lectura, escriptura o afegir.
  • Monad IO: Un monad utilitzat per gestionar operacions d'entrada i sortida en Haskell.

Llegir Fitxers

Per llegir un fitxer en Haskell, pots utilitzar funcions com readFile i hGetContents.

Exemple: Llegir un Fitxer

import System.IO

main :: IO ()
main = do
    contents <- readFile "example.txt"
    putStrLn contents
  • readFile "example.txt": Llegeix tot el contingut de example.txt com una cadena.
  • putStrLn contents: Imprimeix el contingut a la consola.

Gestionar Fitxers amb Handles

Per tenir més control, pots utilitzar handles per llegir fitxers.

import System.IO

main :: IO ()
main = do
    handle <- openFile "example.txt" ReadMode
    contents <- hGetContents handle
    putStrLn contents
    hClose handle
  • openFile "example.txt" ReadMode: Obre el fitxer en mode lectura i retorna un handle.
  • hGetContents handle: Llegeix el contingut utilitzant el handle.
  • hClose handle: Tanca el handle del fitxer.

Escriure a Fitxers

Per escriure a un fitxer, pots utilitzar funcions com writeFile i hPutStr.

Exemple: Escriure a un Fitxer

import System.IO

main :: IO ()
main = do
    writeFile "output.txt" "Hello, Haskell!"
  • writeFile "output.txt" "Hello, Haskell!": Escriu la cadena "Hello, Haskell!" a output.txt.

Afegir a un Fitxer

Per afegir contingut a un fitxer existent, utilitza appendFile.

import System.IO

main :: IO ()
main = do
    appendFile "output.txt" "\nAppending this line."
  • appendFile "output.txt" "\nAppending this line.": Afegeix la cadena a output.txt.

Gestió Avançada de Fitxers

Per a una gestió més avançada de fitxers, pots utilitzar funcions com hPutStrLn i hGetLine.

Exemple: Escriure i Llegir Línia per Línia

import System.IO

main :: IO ()
main = do
    handle <- openFile "output.txt" AppendMode
    hPutStrLn handle "Another line."
    hClose handle

    handle <- openFile "output.txt" ReadMode
    contents <- hGetLine handle
    putStrLn contents
    hClose handle
  • hPutStrLn handle "Another line.": Escriu una línia al fitxer utilitzant el handle.
  • hGetLine handle: Llegeix una línia del fitxer utilitzant el handle.

Gestió d'Errors

Gestionar errors és important quan es treballa amb fitxers. Pots utilitzar catch de Control.Exception per gestionar excepcions.

Exemple: Gestionar Errors de Fitxer

import System.IO
import Control.Exception

main :: IO ()
main = do
    result <- try (readFile "nonexistent.txt") :: IO (Either IOException String)
    case result of
        Left ex  -> putStrLn $ "Caught exception: " ++ show ex
        Right contents -> putStrLn contents
  • try (readFile "nonexistent.txt"): Intenta llegir un fitxer i captura qualsevol IOException.
  • case result of ...: Fa un patró de coincidència sobre el resultat per gestionar l'èxit o el fracàs.

Resum

La gestió de fitxers en Haskell implica:

  • Entendre els camins i modes dels fitxers.
  • Utilitzar el monad IO per realitzar operacions de lectura i escriptura.
  • Utilitzar handles per tenir més control sobre les operacions de fitxers.
  • Gestionar errors de manera elegant utilitzant mecanismes de gestió d'excepcions.

En dominar aquests conceptes, pots gestionar de manera efectiva l'entrada i sortida de fitxers en els teus programes Haskell.
