Metaprogramació en Haskell

La metaprogramació és una tècnica de programació on els programes tenen la capacitat de tractar altres programes com les seves dades. Això permet que un programa sigui més flexible, reutilitzable i adaptable. En Haskell, la metaprogramació es pot aconseguir a través de diversos mitjans com Template Haskell, programació a nivell de tipus i programació genèrica.

Introducció a la Metaprogramació

  • Definició: La metaprogramació és escriure programes que escriuen o manipulen altres programes o a ells mateixos.
  • Propòsit: Permet un codi més abstracte i reutilitzable, permetent l'automatització de tasques repetitives i la generació de codi boilerplate.

Template Haskell

Template Haskell és una extensió de Haskell que suporta la metaprogramació en temps de compilació. Permet generar i manipular codi Haskell programàticament.

Activant Template Haskell

Per utilitzar Template Haskell, cal activar l'extensió a la part superior del teu fitxer Haskell:

{-# LANGUAGE TemplateHaskell #-}

Exemple Bàsic

Aquí tens un exemple senzill d'ús de Template Haskell per generar una funció:

{-# LANGUAGE TemplateHaskell #-}

module Main where

import Language.Haskell.TH

-- Una funció de Template Haskell per generar una funció simple
genFunction :: Q [Dec]
genFunction = [d| hello :: String -> String
                  hello name = "Hello, " ++ name |]

-- Inserir la funció generada al mòdul
$(genFunction)

main :: IO ()
main = putStrLn (hello "World")
  • Explicació:
    • genFunction és una funció de Template Haskell que genera una declaració (Dec) per a una funció hello.
    • $(genFunction) insereix el codi generat al mòdul.
    • La funció hello s'utilitza després a la funció main.

Quasi-Quoting

El quasi-quoting permet incrustar codi dins del teu codi Haskell utilitzant una sintaxi especial.

{-# LANGUAGE QuasiQuotes #-}

module Main where

import Language.Haskell.TH
import Language.Haskell.TH.Quote

-- Definir un quasi-quoter senzill
myQuoter :: QuasiQuoter
myQuoter = QuasiQuoter
  { quoteExp  = \str -> [| "Quoted: " ++ str |]
  , quotePat  = undefined
  , quoteType = undefined
  , quoteDec  = undefined
  }

main :: IO ()
main = putStrLn [myQuoter|Hello, Quasi-Quoting!|]
  • Explicació:
    • myQuoter és un quasi-quoter que afegeix "Quoted: " a la cadena donada.
    • [myQuoter|Hello, Quasi-Quoting!|] utilitza el quasi-quoter per transformar la cadena.

Programació a Nivell de Tipus

La programació a nivell de tipus en Haskell permet realitzar càlculs al nivell de tipus, permetent comprovacions de tipus més potents i garanties en temps de compilació.

Exemple Bàsic

Aquí tens un exemple d'ús de la programació a nivell de tipus per definir un nombre natural a nivell de tipus:

{-# LANGUAGE DataKinds #-}
{-# LANGUAGE TypeFamilies #-}
{-# LANGUAGE TypeOperators #-}

import GHC.TypeLits

-- Definir una suma a nivell de tipus
type family Add (a :: Nat) (b :: Nat) :: Nat where
  Add 0 b = b
  Add a b = 1 + Add (a - 1) b

-- Exemple d'ús
type Five = Add 2 3
  • Explicació:
    • Add és una família de tipus que realitza una suma a nivell de tipus.
    • Five és un sinònim de tipus per al resultat de sumar 2 i 3 a nivell de tipus.

Programació Genèrica

La programació genèrica permet escriure codi que pot operar sobre una gran varietat de tipus de dades.

Derivant Instàncies Genèriques

Haskell proporciona la classe de tipus Generic per facilitar la programació genèrica.

{-# LANGUAGE DeriveGeneric #-}

import GHC.Generics

-- Definir un tipus de dades senzill
data Person = Person { name :: String, age :: Int } deriving (Generic, Show)

-- Funció d'exemple que funciona amb qualsevol tipus Generic
genericShow :: (Generic a, Show (Rep a)) => a -> String
genericShow = show . from

main :: IO ()
main = putStrLn $ genericShow (Person "Alice" 30)
  • Explicació:
    • Person és un tipus de dades que deriva Generic.
    • genericShow és una funció que pot mostrar qualsevol tipus que tingui una instància Generic.

Conclusió

La metaprogramació en Haskell proporciona eines potents per generar i manipular codi, realitzar càlculs a nivell de tipus i escriure funcions genèriques. Mitjançant l'ús de Template Haskell, la programació a nivell de tipus i la programació genèrica, pots escriure codi més abstracte, reutilitzable i mantenible. A mesura que avancis en Haskell, dominar aquestes tècniques et permetrà abordar problemes més complexos amb solucions elegants.
