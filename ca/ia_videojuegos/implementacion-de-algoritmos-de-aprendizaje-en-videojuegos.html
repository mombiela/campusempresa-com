Introducció

En aquesta secció del curs, explorarem com implementar algoritmes d'aprenentatge en videojocs. Els algoritmes d'aprenentatge permeten als personatges i sistemes del joc adaptar-se i millorar el seu comportament amb el temps, proporcionant una experiència de joc més dinàmica i desafiant.

Conceptes Clau

  • Algoritmes d'Aprenentatge: Mètodes que permeten als sistemes aprendre i millorar a partir de dades.
  • Aprenentatge Supervisat: El sistema aprèn a partir d'un conjunt de dades etiquetades.
  • Aprenentatge No Supervisat: El sistema intenta trobar patrons en dades no etiquetades.
  • Aprenentatge per Reforç: El sistema aprèn a través de la interacció amb l'entorn, rebent recompenses o càstigs.

Algoritmes Comuns en Videojocs

Aprenentatge Supervisat

L'aprenentatge supervisat s'utilitza per a tasques com la classificació i la regressió. En videojocs, pot ser útil per a:

  • Reconeixement de patrons: Identificar comportaments de jugadors.
  • Predicció de moviments: Anticipar les accions d'un jugador.

Exemple de Codi: Classificació amb K-Nearest Neighbors (KNN)

from sklearn.neighbors import KNeighborsClassifier

# Dades d'entrenament (característiques i etiquetes)
X_train = [[0, 0], [1, 1], [2, 2], [3, 3]]
y_train = [0, 0, 1, 1]

# Crear el model KNN
knn = KNeighborsClassifier(n_neighbors=3)

# Entrenar el model
knn.fit(X_train, y_train)

# Predir la classe d'un nou punt
X_new = [[1.5, 1.5]]
prediction = knn.predict(X_new)
print(f'Predicció: {prediction}')

Aprenentatge No Supervisat

L'aprenentatge no supervisat s'utilitza per descobrir patrons ocults en les dades. En videojocs, pot ser útil per a:

  • Agrupació de jugadors: Identificar grups de jugadors amb comportaments similars.
  • Generació de contingut: Crear nivells o missions basats en patrons de joc.

Exemple de Codi: Clustering amb K-Means

from sklearn.cluster import KMeans

# Dades d'exemple
X = [[1, 2], [1, 4], [1, 0], [4, 2], [4, 4], [4, 0]]

# Crear el model K-Means
kmeans = KMeans(n_clusters=2, random_state=0)

# Ajustar el model
kmeans.fit(X)

# Predir els clusters
clusters = kmeans.predict(X)
print(f'Clusters: {clusters}')

Aprenentatge per Reforç

L'aprenentatge per reforç és especialment útil en videojocs per a:

  • Control de personatges no jugables (NPCs): NPCs que aprenen a millorar el seu comportament.
  • Optimització d'estratègies: Desenvolupar estratègies òptimes per guanyar el joc.

Exemple de Codi: Q-Learning

import numpy as np

# Paràmetres de l'entorn
states = [0, 1, 2, 3]
actions = [0, 1]
q_table = np.zeros((len(states), len(actions)))
learning_rate = 0.1
discount_factor = 0.9
epsilon = 0.1

# Funció de recompensa
def reward(state, action):
    if state == 3:
        return 1
    else:
        return 0

# Q-Learning
for episode in range(100):
    state = np.random.choice(states)
    while state != 3:
        if np.random.uniform(0, 1) < epsilon:
            action = np.random.choice(actions)
        else:
            action = np.argmax(q_table[state])
        
        next_state = (state + action) % len(states)
        r = reward(state, action)
        
        q_table[state, action] = q_table[state, action] + learning_rate * (r + discount_factor * np.max(q_table[next_state]) - q_table[state, action])
        
        state = next_state

print(f'Taula Q: \n{q_table}')

Comparació d'Algoritmes

| Algoritme | Tipus d'Aprenentatge | Aplicacions en Videojocs | |--------------------------|------------------------|-----------------------------------------| | K-Nearest Neighbors (KNN)| Supervisat | Predicció de moviments, reconeixement de patrons | | K-Means | No Supervisat | Agrupació de jugadors, generació de contingut | | Q-Learning | Reforç | Control de NPCs, optimització d'estratègies |

Conclusió

La implementació d'algoritmes d'aprenentatge en videojocs pot millorar significativament l'experiència del jugador en proporcionar comportaments més intel·ligents i adaptatius. Des de la predicció de moviments fins a l'optimització d'estratègies, aquests algoritmes permeten crear jocs més desafiants i atractius.
