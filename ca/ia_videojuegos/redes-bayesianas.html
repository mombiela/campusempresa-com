Introducció a les Xarxes Bayesianes

Les Xarxes Bayesianes són una eina poderosa en la intel·ligència artificial, especialment útil en la presa de decisions sota incertesa. En el context dels videojocs, poden ser emprades per modelar el comportament dels personatges no jugables (NPCs), la presa de decisions estratègiques i la predicció d'esdeveniments futurs.

Conceptes Clau

  • Xarxa Bayesiana: És un model probabilístic que representa un conjunt de variables i les seves dependències condicionals a través d'un graf dirigit acíclic.
  • Nodes: Representen les variables aleatòries.
  • Arcs: Indiquen la relació de dependència entre les variables.
  • Probabilitat Condicional: La probabilitat que ocorri un esdeveniment donat que un altre esdeveniment ha ocorregut.

Estructura d'una Xarxa Bayesiana

Una Xarxa Bayesiana es compon de dues parts principals:

  1. Estructura del Graf: Un graf dirigit acíclic on els nodes representen variables i els arcs representen dependències.
  2. Distribucions de Probabilitat Condicional: Per a cada node, s'especifica una distribució de probabilitat condicional donada el(s) seu(s) pare(s).

Exemple d'Estructura

Considerem un exemple simple en el context d'un videojoc d'estratègia:

  • Nodes:

    • Clima (assolellat, plujós)
    • Visibilitat (bona, dolenta)
    • Estratègia (agressiva, defensiva)

  • Arcs:

    • Clima -> Visibilitat
    • Visibilitat -> Estratègia

Taula de Probabilitats Condicionals

| Clima | Visibilitat | P(Visibilitat | Clima) | |---------|-------------|---------------| | Assollellat | Bona | 0.8 | | Assollellat | Dolenta | 0.2 | | Plujós | Bona | 0.3 | | Plujós | Dolenta | 0.7 |

| Visibilitat | Estratègia | P(Estratègia | Visibilitat) | |-------------|------------|-----------------------------| | Bona | Agressiva | 0.6 | | Bona | Defensiva | 0.4 | | Dolenta | Agressiva | 0.3 | | Dolenta | Defensiva | 0.7 |

Implementació en Codi

A continuació, es mostra un exemple de com implementar una Xarxa Bayesiana simple utilitzant Python i la biblioteca pgmpy.

from pgmpy.models import BayesianModel
from pgmpy.factors.discrete import TabularCPD
from pgmpy.inference import VariableElimination

# Definir l'estructura del graf
model = BayesianModel([('Clima', 'Visibilitat'), ('Visibilitat', 'Estratègia')])

# Definir les distribucions de probabilitat condicional
cpd_clima = TabularCPD(variable='Clima', variable_card=2, values=[[0.7], [0.3]])
cpd_visibilitat = TabularCPD(variable='Visibilitat', variable_card=2, 
                             values=[[0.8, 0.3], [0.2, 0.7]], 
                             evidence=['Clima'], evidence_card=[2])
cpd_estrategia = TabularCPD(variable='Estratègia', variable_card=2, 
                            values=[[0.6, 0.3], [0.4, 0.7]], 
                            evidence=['Visibilitat'], evidence_card=[2])

# Afegir les CPDs al model
model.add_cpds(cpd_clima, cpd_visibilitat, cpd_estrategia)

# Verificar el model
assert model.check_model()

# Inferència
infer = VariableElimination(model)
posterior_p = infer.query(variables=['Estratègia'], evidence={'Clima': 0, 'Visibilitat': 1})
print(posterior_p)

Aplicacions en Videojocs

  • Comportament de NPCs: Les Xarxes Bayesianes poden modelar com els NPCs prenen decisions basades en l'entorn i les accions del jugador.
  • Predicció d'Esdeveniments: Ajuden a predir esdeveniments futurs en el joc, com canvis en el clima o l'aparició d'enemics.
  • Presa de Decisions Estratègiques: Poden ser utilitzades per determinar la millor estratègia a seguir en jocs d'estratègia en temps real (RTS).

Conclusió

Les Xarxes Bayesianes són una eina valuosa en el desenvolupament de videojocs, proporcionant un marc per a la presa de decisions sota incertesa. La seva capacitat per modelar relacions probabilístiques entre variables les fa ideals per a aplicacions com el comportament de NPCs i la predicció d'esdeveniments. Amb una comprensió bàsica de la seva estructura i funcionament, els desenvolupadors poden implementar aquestes xarxes per millorar la intel·ligència i la jugabilitat dels seus videojocs.
