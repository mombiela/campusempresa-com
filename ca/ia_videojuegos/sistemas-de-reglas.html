Introducció

Els sistemes de regles són una tècnica fonamental en la intel·ligència artificial per a videojocs. Permeten definir comportaments i decisions dels personatges no jugadors (NPCs) mitjançant un conjunt de regles predefinides. Aquest enfocament és especialment útil per crear comportaments predictibles i controlables, el que és essencial en molts gèneres de videojocs.

Conceptes Clau

  • Regla: Una instrucció condicional que defineix una acció a prendre quan es compleix una condició específica.
  • Motor de Regles: El component que avalua les regles i executa les accions corresponents.
  • Condició: Una expressió que s'avalua com a verdadera o falsa.
  • Acció: L'operació que es realitza quan una condició es compleix.

Estructura d'una Regla

Una regla típica en un sistema de regles es pot expressar de la següent manera:

SI <condició> LLAVORS <acció>

Per exemple:

SI la salut del jugador és menor que 20% LLAVORS l'NPC ha de buscar un botiquí.

Implementació de Sistemes de Regles

Exemple en Pseudocodi

A continuació, es mostra un exemple bàsic de com es pot implementar un sistema de regles en pseudocodi:

regles = [
    {condicio: "jugador.salut < 20", accio: "NPC.buscarBotiqui()"},
    {condicio: "jugador.aProp(NPC)", accio: "NPC.atacar()"},
    {condicio: "jugador.lluny(NPC)", accio: "NPC.patrullar()"}
]

motorDeRegles = new MotorDeRegles(regles)

motorDeRegles.avaluar(jugador, NPC)

Exemple en Python

A continuació, es mostra un exemple de com es pot implementar un sistema de regles simple en Python:

class Regla:
    def __init__(self, condicio, accio):
        self.condicio = condicio
        self.accio = accio

class MotorDeRegles:
    def __init__(self, regles):
        self.regles = regles

    def avaluar(self, context):
        for regla in self.regles:
            if eval(regla.condicio, {}, context):
                exec(regla.accio, {}, context)

# Definir les regles
regles = [
    Regla("jugador['salut'] < 20", "NPC['accio'] = 'buscarBotiqui'"),
    Regla("jugador['distancia'] < 5", "NPC['accio'] = 'atacar'"),
    Regla("jugador['distancia'] >= 5", "NPC['accio'] = 'patrullar'")
]

# Crear el motor de regles
motor = MotorDeRegles(regles)

# Definir el context
context = {
    'jugador': {'salut': 15, 'distancia': 3},
    'NPC': {'accio': ''}
}

# Avaluar les regles
motor.avaluar(context)

# Veure l'acció de l'NPC
print(context['NPC']['accio'])  # Sortida: 'buscarBotiqui'

Exercicis Pràctics

  1. Exercici 1: Implementa un sistema de regles que permeti a un NPC decidir entre atacar, defensar-se o fugir basat en la salut del jugador i la distància a l'NPC.
  2. Exercici 2: Modifica l'exemple de Python per incloure una nova regla que faci que l'NPC busqui cobertura si hi ha un obstacle proper.
  3. Exercici 3: Compara el rendiment d'un sistema de regles amb una màquina d'estats finits per a un escenari de combat en un videojoc.

Conclusió

Els sistemes de regles són una eina poderosa i flexible per definir comportaments en videojocs. Permeten als desenvolupadors crear NPCs amb comportaments complexos de manera controlada i predictible. Tot i que tenen les seves limitacions, la seva facilitat d'implementació i control els fa una opció popular en el desenvolupament de videojocs.
