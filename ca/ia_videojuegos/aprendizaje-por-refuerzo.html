Introducció a l'Aprenentatge per Reforç

L'Aprenentatge per Reforç (Reinforcement Learning, RL) és una tècnica d'aprenentatge automàtic on un agent aprèn a prendre decisions mitjançant la interacció amb un entorn. En el context dels videojocs, l'agent pot ser un personatge o una entitat que aprèn a jugar o millorar el seu rendiment en el joc.

Conceptes Clau

  • Agent: L'entitat que pren decisions en l'entorn.
  • Entorn: El món amb el qual interactua l'agent.
  • Estat: Una representació de l'entorn en un moment donat.
  • Acció: Una decisió o moviment que l'agent pot prendre.
  • Recompensa: Retroalimentació rebuda per l'agent després de realitzar una acció.
  • Política: Estratègia que l'agent segueix per decidir quines accions prendre.
  • Funció de Valor: Estima la recompensa futura que es pot obtenir des d'un estat específic.
  • Q-Learning: Un algorisme de RL que busca aprendre la funció de valor d'acció.

Funcionament de l'Aprenentatge per Reforç

El procés de RL es pot descriure en els següents passos:

  1. Inicialització: L'agent comença en un estat inicial de l'entorn.
  2. Selecció d'Acció: L'agent selecciona una acció basada en la seva política actual.
  3. Transició d'Estat: L'entorn respon a l'acció i transita a un nou estat.
  4. Recompensa: L'agent rep una recompensa basada en l'acció presa.
  5. Actualització: L'agent actualitza la seva política i funció de valor basant-se en la recompensa rebuda.

Algorismes d'Aprenentatge per Reforç

Q-Learning

Q-Learning és un dels algorismes més populars en RL. Es basa en aprendre una funció de valor d'acció \( Q(s, a) \) que estima la recompensa esperada de prendre una acció \( a \) en un estat \( s \).

Fórmula d'Actualització de Q-Learning

\[ Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right] \]

  • \( \alpha \): Taxa d'aprenentatge.
  • \( \gamma \): Factor de descompte.
  • \( r \): Recompensa rebuda.
  • \( s' \): Nou estat després de prendre l'acció \( a \).

Exemple de Codi en Python

import numpy as np

# Paràmetres de l'entorn
states = ["A", "B", "C", "D"]
actions = ["left", "right"]
Q = np.zeros((len(states), len(actions)))  # Inicialització de la taula Q
alpha = 0.1  # Taxa d'aprenentatge
gamma = 0.9  # Factor de descompte

# Funció de recompensa (exemple simplificat)
def reward(state, action):
    if state == "A" and action == "right":
        return 1
    else:
        return 0

# Simulació d'un pas en l'entorn
def step(state, action):
    if state == "A" and action == "right":
        return "B", reward(state, action)
    else:
        return state, reward(state, action)

# Entrenament de l'agent
for episode in range(1000):
    state = "A"
    while state != "B":
        action = np.random.choice(actions)
        next_state, r = step(state, action)
        action_index = actions.index(action)
        state_index = states.index(state)
        next_state_index = states.index(next_state)
        Q[state_index, action_index] += alpha * (r + gamma * np.max(Q[next_state_index]) - Q[state_index, action_index])
        state = next_state

print("Taula Q final:")
print(Q)

Aplicacions en Videojocs

El RL es pot aplicar en videojocs per a:

  • Entrenament de NPCs (Non-Player Characters): Els NPCs poden aprendre a comportar-se de manera més realista i adaptativa.
  • Optimització d'Estratègies: Els agents poden aprendre estratègies òptimes per a jocs d'estratègia.
  • Generació de Continguts: Creació de nivells o desafiaments adaptatius basats en el rendiment del jugador.

Conclusió

L'Aprenentatge per Reforç és una tècnica poderosa per desenvolupar agents intel·ligents en videojocs. En comprendre i aplicar conceptes com estats, accions, recompenses i polítiques, els desenvolupadors poden crear experiències de joc més immersives i desafiants. L'algorisme Q-Learning és un bon punt de partida per a aquells que desitgen implementar RL en els seus projectes de videojocs.
