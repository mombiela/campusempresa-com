Introducció a Power Query ►

En aquesta secció, explorarem com connectar Power BI a diverses fonts de dades. Aquesta és una habilitat fonamental per a qualsevol persona que vulgui aprofitar Power BI per a l'anàlisi i visualització de dades. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Introducció a les Fonts de Dades en Power BI

Power BI admet una àmplia gamma de fonts de dades, des de simples fitxers Excel fins a bases de dades complexes i serveis al núvol. Entendre com connectar-se a aquestes fonts de dades és crucial per a una anàlisi de dades efectiva.

Conceptes Clau

  • Font de Dades: L'origen de les dades que vols analitzar (per exemple, fitxer Excel, base de dades SQL, servei web).
  • Connexió de Dades: L'enllaç establert entre Power BI i la font de dades.
  • Actualització de Dades: El procés d'actualitzar les dades en Power BI per reflectir els canvis en les dades d'origen.

Connexió a Fonts de Dades Comunes

Fitxers Excel

Excel és una de les fonts de dades més comunes per a Power BI. Vegem com connectar-se a un fitxer Excel.

  1. Obre Power BI Desktop.
  2. Fes clic a 'Obtenir Dades'.
  3. Selecciona 'Excel' de la llista de fonts de dades.
  4. Navega fins a la ubicació del teu fitxer Excel i selecciona'l.
  5. Fes clic a 'Obrir'.
  6. Selecciona les taules o fulls que vols carregar.
  7. Fes clic a 'Carregar' per importar les dades.

7. Fes clic a 'Carregar'

Bases de Dades SQL

Connectar-se a una base de dades SQL implica uns quants passos més, però és igualment senzill.

  1. Obre Power BI Desktop.
  2. Fes clic a 'Obtenir Dades'.
  3. Selecciona 'SQL Server' de la llista de fonts de dades.
  4. Introdueix el nom del servidor i el nom de la base de dades.
  5. Tria el mètode d'autenticació (Windows o SQL Server).
  6. Fes clic a 'Connectar'.
  7. Selecciona les taules o vistes que vols carregar.
  8. Fes clic a 'Carregar' per importar les dades.

Fonts de Dades Web

Power BI també pot connectar-se a fonts de dades web, com ara APIs o pàgines web.

  1. Obre Power BI Desktop.
  2. Fes clic a 'Obtenir Dades'.
  3. Selecciona 'Web' de la llista de fonts de dades.
  4. Introdueix l'URL de la font de dades web.
  5. Fes clic a 'D'acord'.
  6. Selecciona les taules de dades disponibles a la pàgina web.
  7. Fes clic a 'Carregar' per importar les dades.
# Exemple: Connexió a una font de dades web
1. Obre Power BI Desktop
2. Fes clic a 'Obtenir Dades'
3. Selecciona 'Web'
4. Introdueix l'URL de la font de dades web
5. Fes clic a 'D'acord'
6. Selecciona les taules de dades disponibles
7. Fes clic a 'Carregar'

Connexions de Dades Avançades

Connexió a Fonts de Dades d'Azure

Power BI pot connectar-se a diversos serveis d'Azure, com ara Azure SQL Database i Azure Blob Storage.

  1. Obre Power BI Desktop.
  2. Fes clic a 'Obtenir Dades'.
  3. Selecciona 'Azure' de la llista de fonts de dades.
  4. Tria el servei específic d'Azure (per exemple, Azure SQL Database).
  5. Introdueix les credencials i els detalls de connexió necessaris.
  6. Fes clic a 'Connectar'.
  7. Selecciona les taules o blobs que vols carregar.
  8. Fes clic a 'Carregar' per importar les dades.

Ús de DirectQuery

DirectQuery et permet consultar grans conjunts de dades directament des de la font sense importar-les a Power BI.

  • Avantatges:
    • Accés a dades en temps real.
    • Reducció de l'ús de memòria en Power BI.
  • Desavantatges:
    • Potencialment menor rendiment.
    • Capacitats limitades de transformació de dades.

Per utilitzar DirectQuery:

  1. Obre Power BI Desktop.
  2. Fes clic a 'Obtenir Dades'.
  3. Selecciona la teva font de dades (per exemple, SQL Server).
  4. Tria 'DirectQuery' en lloc de 'Importar'.
  5. Completa els passos de connexió com de costum.

3. Selecciona la teva font de dades (per exemple, SQL Server)
4. Tria 'DirectQuery'
5. Completa els passos de connexió

Actualització de Dades

Mantenir les teves dades actualitzades és essencial per a una anàlisi precisa. Power BI ofereix diverses opcions per a l'actualització de dades.

  • Actualització Manual: Activada per l'usuari.
  • Actualització Programada: Actualitza automàticament les dades a intervals especificats.
  • Actualització Incremental: Actualitza només les dades que han canviat.

Configuració de l'Actualització Programada

  1. Publica el teu informe al Servei Power BI.
  2. Ves a la configuració del conjunt de dades.
  3. Selecciona 'Actualització programada'.
  4. Configura la freqüència i l'hora de l'actualització.
  5. Introdueix les teves credencials si és necessari.
  6. Fes clic a 'Aplicar'.

Conclusió

Connectar-se a fonts de dades és una habilitat crítica en Power BI. Tant si treballes amb fitxers Excel, bases de dades SQL o dades web, entendre com establir i gestionar aquestes connexions et permetrà crear informes dinàmics i actualitzats. A mesura que avancis, podràs explorar fonts de dades i tècniques més complexes com DirectQuery i l'actualització incremental per optimitzar els teus fluxos de treball de dades.
