Visió general

Power BI Mobile permet als usuaris accedir i interactuar amb els seus informes i quadres de comandament de Power BI en dispositius mòbils. Aquesta secció cobrirà els conceptes bàsics de Power BI Mobile, les seves característiques i com començar a utilitzar-lo.

Conceptes clau

  • Aplicació Power BI Mobile: Una aplicació mòbil disponible per a dispositius iOS, Android i Windows que permet als usuaris veure i interactuar amb informes i quadres de comandament de Power BI.
  • Informes i quadres de comandament mòbils: Versions optimitzades dels informes i quadres de comandament de Power BI dissenyades per a la visualització mòbil.
  • Interactivitat: Característiques que permeten als usuaris interactuar amb les dades, com ara filtres, desglossaments i gestos tàctils.

Començar amb Power BI Mobile

Instal·lar l'aplicació Power BI Mobile

  1. iOS: Descarregar des de l'Apple App Store.
  2. Android: Descarregar des del Google Play Store.
  3. Windows: Descarregar des del Microsoft Store.

Iniciar sessió

  • Obre l'aplicació Power BI Mobile.
  • Inicia sessió amb les teves credencials del compte de Power BI.

Navegar per l'aplicació

  • Pantalla d'inici: Mostra informes i quadres de comandament recents.
  • Favorits: Accedeix ràpidament als teus informes i quadres de comandament favorits.
  • Compartit amb mi: Veure informes i quadres de comandament compartits per altres.
  • Espais de treball: Accedeix a diferents espais de treball dels quals formes part.

Veure i interactuar amb informes

Veure informes

  • Obre un informe des de la pantalla d'inici o l'espai de treball.
  • Desplaça't cap a l'esquerra o la dreta per navegar entre les pàgines de l'informe.

Interactuar amb les dades

  • Filtres: Toca les icones de filtre per aplicar filtres.
  • Desglossament: Toca els punts de dades per desglossar dades més detallades.
  • Gestos tàctils: Utilitza els gestos de pessic per fer zoom i desplaçament per navegar i explorar les dades.

Exemple: Aplicar un filtre

1. Obre un informe.
2. Toca la icona de filtre a la cantonada superior dreta.
3. Selecciona els criteris de filtre (per exemple, Rang de dates, Categoria de producte).
4. L'informe s'actualitzarà per reflectir el filtre seleccionat.

Personalitzar informes mòbils

Crear informes optimitzats per a mòbils

  • Utilitza Power BI Desktop per crear dissenys optimitzats per a mòbils.
  • Canvia a la vista de disseny mòbil a Power BI Desktop.
  • Organitza els visuals per adaptar-los a la pantalla mòbil.

Exemple: Crear un disseny mòbil

1. Obre el teu informe a Power BI Desktop.
2. Fes clic a la pestanya "Vista".
3. Selecciona "Disseny per a telèfon".
4. Arrossega i deixa anar els visuals per organitzar-los per a la visualització mòbil.
5. Desa i publica l'informe.

Notificacions i alertes

  • Alertes de dades: Configura alertes per rebre notificacions quan les dades canviïn.
  • Notificacions push: Rep notificacions al teu dispositiu mòbil per a actualitzacions importants.

Configurar una alerta de dades

1. Obre un quadre de comandament.
2. Toca una tile per a la qual vols configurar una alerta.
3. Toca els punts suspensius (...) i selecciona "Gestionar alertes".
4. Configura les condicions per a l'alerta (per exemple, quan les vendes superin una certa quantitat).
5. Desa l'alerta.

Conclusió

Power BI Mobile proporciona una manera convenient d'accedir i interactuar amb els teus informes i quadres de comandament de Power BI en moviment. Comprenent com navegar per l'aplicació, veure i interactuar amb informes, i personalitzar dissenys mòbils, pots treure el màxim profit de les capacitats mòbils de Power BI. Tant si estàs revisant dades sobre la marxa com si estàs configurant alertes per a mètriques crítiques, Power BI Mobile t'assegura que estiguis connectat amb les teves dades en qualsevol moment i lloc.
