Introducció

En aquesta secció, explorarem com millorar els teus informes de Power BI utilitzant visuals personalitzats i temes. Els visuals personalitzats et permeten crear visualitzacions de dades úniques i interactives, mentre que els temes et permeten aplicar un estil consistent a tots els teus informes.

Visuals personalitzats

Què són els visuals personalitzats?

Els visuals personalitzats són tipus addicionals de visualització que pots importar a Power BI per ampliar les seves capacitats per defecte. Aquests visuals poden ser creats per la comunitat de Power BI o per desenvolupadors utilitzant el Power BI Visuals SDK.

Importar visuals personalitzats

Per importar visuals personalitzats al teu informe de Power BI:

  1. Ves al panell de Visualitzacions.
  2. Fes clic a l'ellipse (...) i selecciona "Importar des del marketplace" o "Importar des d'un fitxer".
  3. Tria el visual personalitzat desitjat i afegeix-lo al teu informe.

Exemple: Utilitzant un visual personalitzat

Importem i utilitzem un visual personalitzat anomenat "Word Cloud".

  1. Importar el visual Word Cloud:

    - Ves al panell de Visualitzacions.
- Fes clic a l'ellipse (`...`) i selecciona "Importar des del marketplace".
- Cerca "Word Cloud" i fes clic a "Afegeix".

  2. Afegeix dades al Word Cloud:

    - Arrossega un camp de text (per exemple, Noms de Productes) al visual Word Cloud.
- Ajusta la configuració per personalitzar l'aparença.

Crear visuals personalitzats

Per a usuaris avançats, crear visuals personalitzats implica:

  • Configurar un entorn de desenvolupament amb Node.js i el Power BI Visuals SDK.
  • Escriure codi TypeScript per definir el comportament del visual.
  • Empaquetar el visual i importar-lo a Power BI.

Exemple: Codi bàsic d'un visual personalitzat

Aquí tens un exemple senzill d'un visual personalitzat utilitzant TypeScript:

module powerbi.extensibility.visual {
    export class SimpleVisual implements IVisual {
        private target: HTMLElement;

        constructor(options: VisualConstructorOptions) {
            this.target = options.element;
        }

        public update(options: VisualUpdateOptions) {
            this.target.innerHTML = `<div style="color:blue;">Hola, Power BI!</div>`;
        }
    }
}
  • Explicació:
    • SimpleVisual és una classe que implementa la interfície IVisual.
    • El constructor inicialitza el visual.
    • El mètode update actualitza el visual amb noves dades o configuracions.

Temes

Què són els temes?

Els temes a Power BI et permeten aplicar un aspecte consistent als teus informes definint colors, fonts i altres elements visuals.

Aplicar un tema

Per aplicar un tema al teu informe de Power BI:

  1. Ves a la pestanya Inici.
  2. Fes clic a "Canvia tema".
  3. Tria entre els temes disponibles o importa un fitxer de tema personalitzat.

Crear temes personalitzats

Els temes personalitzats es defineixen utilitzant fitxers JSON. Aquí tens un exemple d'un tema personalitzat senzill:

{
  "name": "Tema personalitzat",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#FFFFFF",
  "foreground": "#000000",
  "tableAccent": "#FF5733"
}
  • Explicació:
    • dataColors: Defineix els colors utilitzats en els gràfics.
    • background: Estableix el color de fons de l'informe.
    • foreground: Estableix el color del text.
    • tableAccent: Defineix el color d'accent per a les taules.

Exemple: Aplicar un tema personalitzat

  1. Crea un fitxer JSON amb la definició del tema:

    {
  "name": "El meu tema personalitzat",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#F0F0F0",
  "foreground": "#333333",
  "tableAccent": "#FF5733"
}

  2. Importa el tema personalitzat:

    - Ves a la pestanya Inici.
- Fes clic a "Canvia tema" i selecciona "Importa tema".
- Tria el fitxer JSON que has creat.

Conclusió

Els visuals personalitzats i els temes són eines poderoses a Power BI que et permeten crear informes més atractius i visualment agradables. Importar o crear visuals personalitzats et permet ampliar les capacitats de visualització per defecte de Power BI. Els temes ajuden a garantir un aspecte consistent i professional a tots els teus informes. Experimenta amb aquestes funcions per millorar els teus projectes de Power BI i oferir insights més impactants.
