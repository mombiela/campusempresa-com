Introducció

La neteja i transformació de dades són passos crucials en l'anàlisi i la creació d'informes de dades. A Power BI, aquests processos asseguren que les teves dades siguin precises, consistents i estiguin llestes per a l'anàlisi. Aquesta secció et guiarà des de les tècniques bàsiques fins a les avançades de neteja i transformació de dades a Power BI.

Començant amb l'Editor de Power Query

L'Editor de Power Query és l'eina principal a Power BI per a la neteja i transformació de dades.

Conceptes Clau

  • Editor de Power Query: Una eina dins de Power BI utilitzada per connectar, combinar i refinar dades de diferents fonts.
  • Consulta: Un conjunt de passos que Power BI utilitza per transformar dades.
  • Passos Aplicats: Cada transformació aplicada a les dades es registra com un pas.

Exemple: Carregar Dades a l'Editor de Power Query

1. Obre Power BI Desktop.
2. Fes clic a 'Inici' > 'Obtenir dades'.
3. Selecciona una font de dades (per exemple, Excel, SQL Server).
4. Fes clic a 'Carregar' per importar les dades.
5. Fes clic a 'Transformar dades' per obrir l'Editor de Power Query.

Tècniques Bàsiques de Neteja de Dades

Eliminar Duplicats

Els registres duplicats poden distorsionar la teva anàlisi. Eliminar-los és essencial.

1. A l'Editor de Power Query, selecciona la columna o columnes on podrien existir duplicats.
2. Ves a 'Inici' > 'Eliminar files' > 'Eliminar duplicats'.

Gestionar Valors Faltants

Els valors faltants es poden substituir, eliminar o omplir segons el context.

1. Selecciona la columna amb valors faltants.
2. Ves a 'Transformar' > 'Substituir valors' per substituir-los amb un valor específic.
3. Alternativament, utilitza 'Eliminar files' per eliminar files amb valors faltants.

Canviar Tipus de Dades

Assegurar-se que cada columna tingui el tipus de dades correcte és crucial per a una anàlisi precisa.

1. Selecciona la columna que vols canviar.
2. Ves a 'Transformar' > 'Tipus de dades' i selecciona el tipus apropiat (per exemple, Text, Número, Data).

Tècniques Intermèdies de Transformació de Dades

Fusionar Consultes

Combinar dades de múltiples fonts pot proporcionar una visió més completa.

1. A l'Editor de Power Query, ves a 'Inici' > 'Fusionar consultes'.
2. Selecciona les consultes que vols fusionar.
3. Tria les columnes coincidents i el tipus d'unió (per exemple, Interna, Externa).
4. Fes clic a 'D'acord' per fusionar les consultes.

Pivotejar i Despivotejar Columnes

Aquestes tècniques ajuden a reestructurar les teves dades per a una millor anàlisi.

  • Pivotejar: Converteix files en columnes.
  • Despivotejar: Converteix columnes en files.
1. Selecciona les columnes que vols pivotejar o despivotejar.
2. Ves a 'Transformar' > 'Pivotejar columna' o 'Despivotejar columnes'.

Agrupar Dades

Agrupar dades et permet agregar valors basant-te en criteris específics.

1. Selecciona la columna o columnes per les quals vols agrupar.
2. Ves a 'Transformar' > 'Agrupar per'.
3. Tria la funció d'agregació (per exemple, Suma, Mitjana).
4. Fes clic a 'D'acord' per aplicar l'agrupació.

Tècniques Avançades de Transformació de Dades

Utilitzar el Llenguatge M

M és el llenguatge de fórmules darrere de Power Query. Permet transformacions més complexes.

1. A l'Editor de Power Query, ves a 'Editor avançat'.
2. Escriu o modifica el codi M per realitzar transformacions avançades.

Exemple:

let Source = Excel.Workbook(File.Contents("C:\Data\SalesData.xlsx"), null, true), Sales_Sheet = Source{[Name="Sales"]}[Content], ChangedType = Table.TransformColumnTypes(Sales_Sheet,{{"Date", type date}, {"Sales", Int64.Type}}) in ChangedType

Funcions Personalitzades


Crear funcions personalitzades pot ajudar a reutilitzar lògica complexa en múltiples consultes.


1. Ves a 'Inici' > 'Editor avançat'.
2. Defineix una funció utilitzant el llenguatge M.

Exemple:
let
AddTax = (price as number) as number =>
let
taxRate = 0.1,
finalPrice = price * (1 + taxRate)
in
finalPrice
in
AddTax


Parametrització de Consultes


Els paràmetres permeten consultes dinàmiques i reutilitzables.


1. Ves a 'Inici' > 'Gestionar paràmetres' > 'Nou paràmetre'.
2. Defineix el paràmetre (per exemple, nom, tipus, valor per defecte).
3. Utilitza el paràmetre en les teves consultes.

Exemple:
let
Source = Sql.Database("ServerName", "DatabaseName"),
FilteredData = Table.SelectRows(Source, each [Date] >= StartDate and [Date] <= EndDate)
in
FilteredData


Conclusió


La neteja i transformació de dades són habilitats essencials per a qualsevol usuari de Power BI. En dominar aquestes tècniques, pots assegurar-te que les teves dades siguin precises, consistents i estiguin llestes per a una anàlisi informativa. Des de tasques bàsiques com eliminar duplicats i gestionar valors faltants fins a tècniques avançades com utilitzar el llenguatge M i funcions personalitzades, Power BI proporciona eines poderoses per transformar les teves dades de manera efectiva.
