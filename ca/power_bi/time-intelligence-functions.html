Introducció

Les funcions d'intel·ligència temporal a Power BI són essencials per analitzar i manipular dades al llarg del temps. Aquestes funcions et permeten crear càlculs que tenen en compte el temps, com ara totals acumulats fins a la data, mitjanes mòbils i comparacions en diferents períodes.

Conceptes Clau

  • Taula de Dates: Una taula que conté un rang continu de dates, que és crucial per als càlculs d'intel·ligència temporal.
  • DAX (Expressions d'Anàlisi de Dades): El llenguatge de fórmules utilitzat a Power BI per crear càlculs personalitzats.

Configuració d'una Taula de Dates

Abans d'utilitzar funcions d'intel·ligència temporal, necessites una Taula de Dates. Aquesta taula hauria d'incloure columnes per any, trimestre, mes, dia i altres períodes de temps rellevants.

DateTable = 
ADDCOLUMNS (
    CALENDAR (DATE(2020, 1, 1), DATE(2023, 12, 31)),
    "Year", YEAR([Date]),
    "Month", MONTH([Date]),
    "Day", DAY([Date]),
    "Quarter", QUARTER([Date]),
    "YearMonth", FORMAT([Date], "YYYY-MM")
)

Funcions Comunes d'Intel·ligència Temporal

TOTALYTD

Calcula el total acumulat fins a la data per a una mesura.

TotalSalesYTD = 
TOTALYTD (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

SAMEPERIODLASTYEAR

Retorna una taula que conté una columna de dates desplaçades un any enrere en el temps.

SalesLastYear = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    SAMEPERIODLASTYEAR(DateTable[Date])
)

DATESINPERIOD

Retorna una taula que conté una columna de dates que comença amb una data d'inici especificada i continua durant el nombre d'intervals especificat.

SalesLast30Days = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATESINPERIOD(DateTable[Date], MAX(DateTable[Date]), -30, DAY)
)

PARALLELPERIOD

Retorna una taula que conté una columna de dates que són paral·leles a les dates de la columna especificada, desplaçades pel nombre d'intervals especificat.

SalesParallelPeriod = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    PARALLELPERIOD(DateTable[Date], -1, MONTH)
)

Funcions Avançades d'Intel·ligència Temporal

DATEADD

Desplaça un conjunt de dates pel nombre d'intervals especificat.

SalesDateAdd = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATEADD(DateTable[Date], -1, YEAR)
)

CLOSINGBALANCEMONTH

Calcula el saldo de tancament per a una mesura al final del mes.

ClosingBalanceMonth = 
CLOSINGBALANCEMONTH (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

OPENINGBALANCEYEAR

Calcula el saldo d'obertura per a una mesura a l'inici de l'any.

OpeningBalanceYear = 
OPENINGBALANCEYEAR (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

Exemples Pràctics

Comparació de Vendes Acumulades Fins a la Data

Compara les vendes acumulades fins a la data de l'any actual amb l'any anterior.

YTD_Sales_CurrentYear = 
TOTALYTD (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

YTD_Sales_PreviousYear = 
CALCULATE (
    TOTALYTD (
        SUM(Sales[SalesAmount]),
        DateTable[Date]
    ),
    SAMEPERIODLASTYEAR(DateTable[Date])
)

Vendes Mòbils de 12 Mesos

Calcula el total de vendes mòbils de 12 mesos.

Rolling12MonthSales = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATESINPERIOD(DateTable[Date], MAX(DateTable[Date]), -12, MONTH)
)

Conclusió

Les funcions d'intel·ligència temporal a Power BI són eines poderoses per analitzar dades al llarg del temps. Configurant una Taula de Dates adequada i utilitzant funcions com TOTALYTD, SAMEPERIODLASTYEAR i DATESINPERIOD, pots crear càlculs basats en el temps molt útils. A mesura que avancis, funcions com DATEADD, CLOSINGBALANCEMONTH i OPENINGBALANCEYEAR t'ajudaran a realitzar anàlisis més complexes basades en el temps. Entendre i dominar aquestes funcions millorarà significativament les teves capacitats d'anàlisi de dades a Power BI.
