Introducció

Power BI és una eina potent d'analítica empresarial que permet als usuaris visualitzar i compartir informació a partir de les seves dades. Una de les seves fortaleses és la seva capacitat d'integrar-se perfectament amb altres serveis de Microsoft. Aquesta secció explorarà com es pot utilitzar Power BI conjuntament amb diversos serveis de Microsoft per millorar les capacitats d'anàlisi i informes de dades.

Integració de Power BI amb Excel

Excel és una eina àmpliament utilitzada per a l'anàlisi i manipulació de dades. Power BI pot importar dades d'Excel i també exportar dades a Excel per a una anàlisi posterior.

  • Importar Dades d'Excel:

    • Obre Power BI Desktop.
    • Fes clic a "Obtenir Dades" i selecciona "Excel".
    • Tria el fitxer Excel que vols importar.
    • Selecciona les taules o rangs que vols carregar a Power BI.

  • Exportar Dades a Excel:

    • A Power BI Service, obre l'informe que vols exportar.
    • Fes clic a l'ellipse (...) a la cantonada superior dreta del visual que vols exportar.
    • Selecciona "Exportar dades".
    • Tria el format (CSV o Excel) i fes clic a "Exportar".
# Exemple: Importar dades d'un fitxer Excel anomenat "SalesData.xlsx"
# Pas 1: Fes clic a "Obtenir Dades"
# Pas 2: Selecciona "Excel"
# Pas 3: Tria "SalesData.xlsx"
# Pas 4: Selecciona la taula "Sales" i fes clic a "Carregar"

Utilitzar Power BI amb SharePoint

SharePoint és una plataforma col·laborativa basada en web que s'integra amb Microsoft Office. Power BI pot connectar-se a llistes i biblioteques de SharePoint per crear informes dinàmics.

  • Connectar-se a Llistes de SharePoint:

    • Obre Power BI Desktop.
    • Fes clic a "Obtenir Dades" i selecciona "Llista de SharePoint Online".
    • Introdueix l'URL del lloc de SharePoint i fes clic a "D'acord".
    • Selecciona la llista que vols carregar a Power BI.

  • Incrustar Informes de Power BI a SharePoint:

    • Publica el teu informe de Power BI a Power BI Service.
    • Obre el teu lloc de SharePoint Online.
    • Edita la pàgina on vols incrustar l'informe.
    • Afegeix la part web "Power BI" i configura-la per mostrar el teu informe.
# Exemple: Connectar-se a una llista de SharePoint anomenada "EmployeeData"
# Pas 1: Fes clic a "Obtenir Dades"
# Pas 2: Selecciona "Llista de SharePoint Online"
# Pas 3: Introdueix l'URL "https://yourcompany.sharepoint.com/sites/HR"
# Pas 4: Selecciona la llista "EmployeeData" i fes clic a "Carregar"

Power BI i Microsoft Teams

Microsoft Teams és una plataforma de col·laboració que combina xat de treball, reunions i col·laboració de fitxers. Power BI es pot integrar a Teams per compartir informes i quadres de comandament dins del teu equip.

  • Afegir Informes de Power BI a Teams:

    • Obre Microsoft Teams.
    • Navega fins a l'equip i canal on vols afegir l'informe.
    • Fes clic a la icona "+" per afegir una nova pestanya.
    • Selecciona "Power BI" i tria l'informe que vols afegir.

  • Utilitzar l'Aplicació Power BI a Teams:

    • Instal·la l'aplicació Power BI des de la botiga d'aplicacions de Teams.
    • Accedeix als teus informes i quadres de comandament de Power BI directament dins de Teams.

Power BI i Azure

Azure és la plataforma de computació en núvol de Microsoft. Power BI pot connectar-se a diversos serveis d'Azure per analitzar i visualitzar dades emmagatzemades al núvol.

  • Connectar-se a la Base de Dades SQL d'Azure:

    • Obre Power BI Desktop.
    • Fes clic a "Obtenir Dades" i selecciona "Azure".
    • Tria "Base de Dades SQL d'Azure".
    • Introdueix el nom del servidor i de la base de dades, i proporciona les teves credencials.

  • Utilitzar l'Emmagatzematge de Llac de Dades d'Azure:

    • Obre Power BI Desktop.
    • Fes clic a "Obtenir Dades" i selecciona "Azure".
    • Tria "Emmagatzematge de Llac de Dades d'Azure".
    • Introdueix l'URL del teu Emmagatzematge de Llac de Dades i carrega les dades.

Conclusió

Integrar Power BI amb altres serveis de Microsoft pot millorar significativament les teves capacitats d'anàlisi i informes de dades. Tant si estàs important dades d'Excel, connectant-te a llistes de SharePoint, incrustant informes a Teams, o analitzant dades emmagatzemades a Azure, Power BI proporciona opcions d'integració sense problemes per ajudar-te a treure el màxim profit de les teves dades. Aprofitant aquestes integracions, pots crear informes més dinàmics i interactius, col·laborar de manera més efectiva i obtenir una comprensió més profunda de les teves dades.
