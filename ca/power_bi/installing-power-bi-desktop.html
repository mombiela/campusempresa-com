Power BI Desktop és una eina potent per a l'anàlisi i visualització de dades. Aquesta secció et guiarà a través del procés d'instal·lació de Power BI Desktop al teu ordinador. Al final d'aquesta secció, tindràs Power BI Desktop instal·lat i llest per utilitzar.

Requisits del Sistema

Abans d'instal·lar Power BI Desktop, assegura't que el teu sistema compleixi els següents requisits:

  • Sistema Operatiu: Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Server 2012 R2, Windows Server 2012, o Windows Server 2008 R2.
  • Memòria: Almenys 2 GB de RAM, tot i que es recomanen 4 GB o més.
  • Espai en Disc: Almenys 1 GB d'espai disponible.
  • .NET Framework: .NET 4.5 o posterior.

Descarregant Power BI Desktop

Per descarregar Power BI Desktop, segueix aquests passos:

  1. Obre el teu navegador web i ves a la pàgina de descàrrega de Power BI Desktop.
  2. Fes clic al botó Download Free.
  3. Seràs redirigit a la Microsoft Store o a un enllaç de descàrrega directa, depenent del teu navegador i configuració del sistema.

Instal·lant Power BI Desktop

Un cop hagis descarregat l'instal·lador, segueix aquests passos per instal·lar Power BI Desktop:

  1. Localitza el fitxer de l'instal·lador descarregat (normalment anomenat PBIDesktopSetup.exe) i fes doble clic per iniciar el procés d'instal·lació.
  2. Segueix les instruccions en pantalla:
    • Pantalla de Benvinguda: Fes clic a Next.
    • Acord de Llicència: Llegeix els termes, selecciona I accept the terms in the License Agreement, i fes clic a Next.
    • Carpeta d'Instal·lació: Tria la carpeta de destinació per a la instal·lació o deixa-la per defecte, després fes clic a Next.
    • Preparat per Instal·lar: Fes clic a Install per començar la instal·lació.
  3. Espera que la instal·lació es completi. Això pot trigar uns minuts.
  4. Un cop la instal·lació estigui completa, fes clic a Finish per sortir de l'instal·lador.

Iniciant Power BI Desktop

Després de la instal·lació, pots iniciar Power BI Desktop:

  1. Ves al menú d'Inici i cerca Power BI Desktop.
  2. Fes clic a l'aplicació Power BI Desktop per obrir-la.
  3. La primera vegada que obris Power BI Desktop, és possible que se't demani iniciar sessió amb el teu compte de Microsoft. Si no en tens un, pots crear-lo gratuïtament.

Resum

En aquesta secció, has après a:

  • Verificar que el teu sistema compleixi els requisits necessaris per instal·lar Power BI Desktop.
  • Descarregar l'instal·lador de Power BI Desktop des del lloc web oficial.
  • Instal·lar Power BI Desktop seguint les instruccions pas a pas.
  • Iniciar Power BI Desktop i iniciar sessió amb el teu compte de Microsoft.

Amb Power BI Desktop instal·lat, ara estàs llest per començar a explorar les seves característiques i capacitats. En les següents seccions, ens endinsarem en els conceptes bàsics d'ús de Power BI Desktop per crear potents visualitzacions de dades i informes.
