Introducció

L'optimització del rendiment a Power BI és crucial per assegurar que els teus informes i quadres de comandament siguin ràpids, responsius i eficients. Aquesta secció et guiarà a través de diverses tècniques i millors pràctiques per optimitzar les teves solucions de Power BI des de nivells bàsics fins a avançats.

Conceptes Clau

  • Optimització del Model de Dades: Estructurar el teu model de dades de manera eficient.
  • Optimització de Consultes: Escriure consultes eficients per obtenir dades.
  • Optimització de DAX: Escriure fórmules DAX (Expressions d'Anàlisi de Dades) eficients.
  • Optimització de Visualitzacions: Assegurar que els elements visuals no siguin intensius en recursos.
  • Actualització Incremental: Carregar només dades noves o canviades per millorar el rendiment.

Optimització del Model de Dades

Esquema Estrella vs. Esquema Floc de Neu

  • Esquema Estrella: Simplifica les consultes i millora el rendiment reduint el nombre d'unions.
  • Esquema Floc de Neu: Normalitza les dades però pot conduir a consultes complexes i rendiment més lent.

| Característica | Esquema Estrella | Esquema Floc de Neu | |------------------|---------------------|----------------------| | Complexitat | Simple | Complex | | Rendiment | Alt | Moderat a Baix | | Manteniment | Fàcil | Difícil |

Millors Pràctiques

  • Evitar Relacions Bidireccionals: Utilitza relacions unidireccionals sempre que sigui possible.
  • Reduir la Cardinalitat: Minimitza el nombre de valors únics en les columnes.
  • Utilitzar Tipus de Dades Adequats: Tria el tipus de dades més petit que pugui contenir les teves dades.

Optimització de Consultes

Plegament de Consultes

  • Definició: El procés on Power Query envia les transformacions a la font de dades.
  • Beneficis: Redueix la quantitat de dades processades localment, millorant el rendiment.

Millors Pràctiques

  • Filtrar Aviat: Aplica filtres tan aviat com sigui possible en les teves consultes.
  • Eliminar Columnes Innecessàries: Carrega només les columnes que necessites.
  • Combinar Passos: Fusiona passos per reduir el nombre de transformacions.
let
    Source = Sql.Database("NomServidor", "NomBaseDeDades"),
    FilteredRows = Table.SelectRows(Source, each [Columna] = "Valor"),
    RemovedColumns = Table.RemoveColumns(FilteredRows,{"ColumnaInnecessària"})
in
    RemovedColumns

Optimització de DAX

Mesures vs. Columnes Calculades

  • Mesures: Calculades en temps de consulta, generalment més eficients.
  • Columnes Calculades: Calculades en temps de càrrega de dades, poden augmentar la mida del model.

Millors Pràctiques

  • Evitar Funcions Iteratives: Utilitza funcions com SUMX, AVERAGEX amb moderació.
  • Utilitzar Variables: Emmagatzema resultats intermedis en variables per evitar càlculs repetits.
TotalSales = 
VAR SalesAmount = SUM(Sales[Amount])
RETURN
    SalesAmount

Optimització de Visualitzacions

Reduir la Càrrega Visual

  • Limitar Visuals: Minimitza el nombre de visuals en una sola pàgina.
  • Simplificar Visuals: Utilitza visuals més simples que requereixin menys potència de processament.

Millors Pràctiques

  • Utilitzar Agregacions: Agrega dades a la font per reduir la quantitat de dades processades.
  • Optimitzar Slicers: Utilitza slicers de manera eficient per filtrar dades sense sobrecarregar el model.

Actualització Incremental

Beneficis

  • Millora el Rendiment: Carrega només dades noves o canviades.
  • Redueix el Temps de Càrrega: Disminueix el temps necessari per actualitzar grans conjunts de dades.

Implementació

  1. Definir Paràmetres: Crea paràmetres per a les dates d'inici i final.
  2. Configurar l'Actualització Incremental: Configura polítiques d'actualització incremental a Power BI Desktop.
let
    Source = Sql.Database("NomServidor", "NomBaseDeDades"),
    FilteredRows = Table.SelectRows(Source, each [Data] >= IniciInterval i [Data] <= FinalInterval)
in
    FilteredRows

Conclusió

L'optimització del rendiment a Power BI implica una combinació de modelatge de dades eficient, escriptura de consultes, optimització de fórmules DAX i gestió de visualitzacions. Seguint les millors pràctiques i tècniques descrites en aquesta secció, pots assegurar que els teus informes i quadres de comandament de Power BI siguin ràpids i eficients, proporcionant una millor experiència d'usuari i informació més oportuna.
