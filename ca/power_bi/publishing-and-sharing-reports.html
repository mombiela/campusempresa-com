Introducció

En aquesta secció, explorarem com publicar i compartir informes a Power BI. Aquesta és una habilitat essencial per fer que les teves anàlisis de dades siguin accessibles per a altres, ja siguin membres de l'equip, parts interessades o el públic en general.

Publicació d'Informes

Passos per Publicar un Informe

  1. Prepara el Teu Informe: Assegura't que el teu informe està complet i totes les visualitzacions de dades són precises.
  2. Desa el Teu Informe: Desa el teu fitxer de Power BI Desktop (.pbix) localment.
  3. Inicia Sessió al Servei de Power BI: Obre Power BI Desktop i inicia sessió al teu compte de Power BI.
  4. Publica: Fes clic al botó 'Publica' a la pestanya Inici.
Inici -> Publica -> Selecciona Espai de Treball -> Publica

Selecció d'un Espai de Treball

  • El Meu Espai de Treball: Espai de treball personal per a ús individual.
  • Espais de Treball Compartits: Espais col·laboratius per a projectes d'equip.

Exemple

1. Fes clic a 'Publica' a Power BI Desktop.
2. Tria 'El Meu Espai de Treball' o un espai de treball compartit específic.
3. Fes clic a 'Selecciona' per publicar l'informe.

Compartició d'Informes

Opcions de Compartició

  1. Compartició Directa: Comparteix informes directament amb usuaris específics.
  2. Incrusta a SharePoint Online: Incrusta informes a SharePoint per a un accés fàcil.
  3. Publica a la Web: Fes que els informes siguin accessibles públicament mitjançant una URL.

Compartició Directa

  • Passos:
    1. Obre l'informe al Servei de Power BI.
    2. Fes clic al botó 'Comparteix'.
    3. Introdueix les adreces de correu electrònic dels destinataris.
    4. Estableix els permisos (visualitzar o editar).
    5. Fes clic a 'Envia'.
Informe -> Comparteix -> Introdueix Correus -> Estableix Permisos -> Envia

Exemple

1. Obre l'informe al Servei de Power BI.
2. Fes clic a 'Comparteix'.
3. Introdueix 'john.doe@example.com'.
4. Estableix el permís a 'Pot visualitzar'.
5. Fes clic a 'Envia'.

Incrusta a SharePoint Online

  • Passos:
    1. Obre l'informe al Servei de Power BI.
    2. Fes clic a 'Fitxer' -> 'Incrusta a SharePoint Online'.
    3. Copia el codi d'incrustació.
    4. Enganxa el codi d'incrustació en una pàgina de SharePoint Online.
Fitxer -> Incrusta a SharePoint Online -> Copia Codi d'Incrustació -> Enganxa a SharePoint

Publica a la Web

  • Passos:
    1. Obre l'informe al Servei de Power BI.
    2. Fes clic a 'Fitxer' -> 'Publica a la web'.
    3. Genera el codi d'incrustació o URL.
    4. Comparteix la URL o el codi d'incrustació segons sigui necessari.
Fitxer -> Publica a la web -> Genera Codi d'Incrustació/URL -> Comparteix

Gestió de Permisos

Establiment de Permisos

  • Només Visualitzar: Els usuaris poden veure l'informe però no poden fer canvis.
  • Editar: Els usuaris poden veure i fer canvis a l'informe.

Exemple

1. Obre l'informe al Servei de Power BI.
2. Fes clic a 'Comparteix'.
3. Introdueix 'jane.doe@example.com'.
4. Estableix el permís a 'Pot editar'.
5. Fes clic a 'Envia'.

Conclusió

Publicar i compartir informes a Power BI és un procés senzill que millora la col·laboració i l'accessibilitat de les dades. Seguint els passos descrits anteriorment, pots assegurar-te que els teus informes arribin al públic adequat amb els permisos corresponents. Ja sigui compartint directament, incrustant a SharePoint o publicant a la web, Power BI proporciona opcions flexibles per satisfer les teves necessitats.
