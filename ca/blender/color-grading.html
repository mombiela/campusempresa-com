El color grading és una part essencial del procés de postproducció en el renderitzat i l'animació 3D. Implica ajustar els colors i tons de les teves imatges o animacions renderitzades per aconseguir un aspecte o ambient desitjat. Aquest tema et guiarà des de les tècniques bàsiques fins a les avançades de color grading a Blender.

Introducció al Color Grading

El color grading pot canviar dràsticament la sensació del teu projecte. Pot fer que les teves escenes semblin més cinematogràfiques, destacar elements importants o crear una atmosfera específica.

Conceptes Clau

  • Correcció de Color: Ajustar els colors per fer-los semblar naturals i consistents.
  • Color Grading: Estilitzar els colors per aconseguir un efecte artístic específic.
  • LUTs (Look-Up Tables): Presets de color grading predefinits que es poden aplicar als teus renders.
  • Scopes: Eines com histogrames, formes d'ona i vectorscopes que ajuden a analitzar la informació del color.

Tècniques Bàsiques de Color Grading

Utilitzant el Node de Color Balance

El node de Color Balance a Blender et permet ajustar el lift, gamma i gain de la teva imatge, que corresponen a les ombres, mitjanes i altes llums, respectivament.

import bpy

# Crear un nou arbre de nodes per a la composició
bpy.context.scene.use_nodes = True
tree = bpy.context.scene.node_tree

# Esborrar els nodes per defecte
for node in tree.nodes:
    tree.nodes.remove(node)

# Afegir el node Render Layers
render_layers = tree.nodes.new(type='CompositorNodeRLayers')

# Afegir el node Color Balance
color_balance = tree.nodes.new(type='CompositorNodeColorBalance')

# Afegir el node Composite
composite = tree.nodes.new(type='CompositorNodeComposite')

# Enllaçar nodes
tree.links.new(render_layers.outputs['Image'], color_balance.inputs['Image'])
tree.links.new(color_balance.outputs['Image'], composite.inputs['Image'])

# Ajustar configuracions de Color Balance
color_balance.correction_method = 'LIFT_GAMMA_GAIN'
color_balance.lift = [0.8, 0.8, 0.8]
color_balance.gamma = [1.2, 1.2, 1.2]
color_balance.gain = [1.1, 1.1, 1.1]

Utilitzant el Node RGB Curves

El node RGB Curves proporciona un control més precís sobre els canals de color i la brillantor general.

import bpy

# Afegir el node RGB Curves
rgb_curves = tree.nodes.new(type='CompositorNodeCurveRGB')

# Enllaçar nodes
tree.links.new(render_layers.outputs['Image'], rgb_curves.inputs['Image'])
tree.links.new(rgb_curves.outputs['Image'], composite.inputs['Image'])

# Ajustar configuracions de RGB Curves
curve = rgb_curves.mapping.curves[3]  # Corba principal
curve.points.new(0.3, 0.2)
curve.points.new(0.7, 0.8)

Tècniques Intermèdies de Color Grading

Utilitzant LUTs (Look-Up Tables)

Els LUTs es poden utilitzar per aplicar presets de color grading complexos als teus renders.

import bpy

# Afegir un node LUT
lut_node = tree.nodes.new(type='CompositorNodeLUT')

# Enllaçar nodes
tree.links.new(render_layers.outputs['Image'], lut_node.inputs['Image'])
tree.links.new(lut_node.outputs['Image'], composite.inputs['Image'])

# Carregar el fitxer LUT
lut_node.lut.filepath = '/path/to/your/lut.cube'

Utilitzant Scopes per a l'Anàlisi

Els scopes t'ajuden a analitzar la informació del color a la teva imatge per fer ajustos més informats.

  • Histograma: Mostra la distribució de la brillantor dels píxels.
  • Forma d'ona: Mostra els nivells de brillantor a través de la imatge.
  • Vectorscope: Visualitza la informació del color.

Tècniques Avançades de Color Grading

Combinant Múltiples Nodes

Per a un color grading avançat, pots combinar múltiples nodes per aconseguir efectes complexos.

import bpy

# Afegir múltiples nodes
hue_saturation = tree.nodes.new(type='CompositorNodeHueSat')
brightness_contrast = tree.nodes.new(type='CompositorNodeBrightContrast')

# Enllaçar nodes
tree.links.new(render_layers.outputs['Image'], hue_saturation.inputs['Image'])
tree.links.new(hue_saturation.outputs['Image'], brightness_contrast.inputs['Image'])
tree.links.new(brightness_contrast.outputs['Image'], composite.inputs['Image'])

# Ajustar configuracions
hue_saturation.color_saturation = 1.2
brightness_contrast.inputs['Bright'].default_value = 0.1
brightness_contrast.inputs['Contrast'].default_value = 1.5

Conclusió

El color grading és una eina poderosa a Blender que pot millorar significativament la qualitat visual dels teus projectes. Entenent i utilitzant els diversos nodes i tècniques disponibles, pots aconseguir resultats de nivell professional.
