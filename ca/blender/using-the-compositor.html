Introducció al Compositor

El Compositor a Blender és una eina poderosa per al post-processament dels teus renders. Et permet combinar múltiples imatges, aplicar efectes i millorar el teu resultat final. Aquesta secció et guiarà des dels conceptes bàsics fins a tècniques avançades en l'ús del Compositor.

Començant amb el Compositor

Activant el Compositor

Per començar a utilitzar el Compositor:

  1. Obre Blender i canvia a l'espai de treball "Compositing".
  2. Marca la casella "Use Nodes" a la finestra del Compositor.

Configuració Bàsica de Nodes

  • Render Layers Node: Aquest node és el punt de partida per a la teva imatge renderitzada.
  • Composite Node: Aquest node és el punt final on es genera la imatge final.
# Exemple de Configuració Bàsica de Nodes
import bpy

# Canvia a l'espai de treball de Compositing
bpy.context.area.ui_type = 'CompositorNodeTree'

# Activa Use Nodes
bpy.context.scene.use_nodes = True

# Obté l'arbre de nodes
tree = bpy.context.scene.node_tree

# Elimina els nodes per defecte
for node in tree.nodes:
    tree.nodes.remove(node)

# Afegeix el node Render Layers
render_layers = tree.nodes.new(type='CompositorNodeRLayers')
render_layers.location = (0, 0)

# Afegeix el node Composite
composite = tree.nodes.new(type='CompositorNodeComposite')
composite.location = (400, 0)

# Enllaça els nodes
tree.links.new(render_layers.outputs[0], composite.inputs[0])

Tècniques Intermèdies de Compositing

Afegint Efectes

  • Blur Node: Afegeix un efecte de desenfocament a la teva imatge.
  • Color Balance Node: Ajusta l'equilibri de color de la teva imatge.
# Afegint un Efecte de Desenfocament
blur = tree.nodes.new(type='CompositorNodeBlur')
blur.location = (200, 0)
blur.size_x = 10
blur.size_y = 10

# Enllaça Render Layers amb Blur, després amb Composite
tree.links.new(render_layers.outputs[0], blur.inputs[0])
tree.links.new(blur.outputs[0], composite.inputs[0])

Mascarejant

  • Mask Node: Et permet crear i utilitzar màscares per a compositing selectiu.
  • Mix Node: Combina dues imatges utilitzant una màscara.
# Utilitzant una Màscara
mask = tree.nodes.new(type='CompositorNodeMask')
mask.location = (200, -200)
mask.name = "MyMask"

mix = tree.nodes.new(type='CompositorNodeMixRGB')
mix.location = (400, -200)
mix.blend_type = 'MIX'

# Enllaça els nodes
tree.links.new(render_layers.outputs[0], mix.inputs[1])
tree.links.new(mask.outputs[0], mix.inputs[0])
tree.links.new(mix.outputs[0], composite.inputs[0])

Tècniques Avançades de Compositing

Renderització Multi-Pass

  • Render Passes: Separa diferents aspectes del render (per exemple, Diffuse, Glossy, Shadow) per a més control en el compositing.
# Activant Render Passes
bpy.context.scene.view_layers["View Layer"].use_pass_diffuse_color = True
bpy.context.scene.view_layers["View Layer"].use_pass_glossy_direct = True

# Accedint a Render Passes en el Compositor
diffuse = render_layers.outputs['DiffCol']
glossy = render_layers.outputs['GlossDir']

# Exemple: Combinant Diffuse i Glossy Passes
add = tree.nodes.new(type='CompositorNodeMixRGB')
add.location = (600, 0)
add.blend_type = 'ADD'

tree.links.new(diffuse, add.inputs[1])
tree.links.new(glossy, add.inputs[2])
tree.links.new(add.outputs[0], composite.inputs[0])

Efectes Avançats

  • Lens Distortion Node: Simula efectes de distorsió de lent.
  • Glare Node: Afegeix efectes de resplendor a les àrees brillants de la imatge.
# Afegint Distorsió de Lent
lens_distortion = tree.nodes.new(type='CompositorNodeLensdist')
lens_distortion.location = (200, 200)
lens_distortion.inputs['Distort'].default_value = 0.5

# Afegint Resplendor
glare = tree.nodes.new(type='CompositorNodeGlare')
glare.location = (400, 200)
glare.glare_type = 'FOG_GLOW'

# Enllaça els nodes
tree.links.new(render_layers.outputs[0], lens_distortion.inputs[0])
tree.links.new(lens_distortion.outputs[0], glare.inputs[0])
tree.links.new(glare.outputs[0], composite.inputs[0])

Conclusió

El Compositor a Blender és una eina essencial per millorar els teus renders i aconseguir resultats de qualitat professional. Comprenent i utilitzant nodes, màscares, passes de render i efectes avançats, pots millorar significativament la qualitat visual dels teus projectes. Practica aquestes tècniques per convertir-te en un expert en compositing i elevar les teves habilitats a Blender al següent nivell.
