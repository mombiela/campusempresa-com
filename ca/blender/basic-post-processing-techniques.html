Introducció

El postprocessament a Blender implica aplicar diversos efectes i ajustos a les teves imatges o animacions renderitzades per millorar-ne la qualitat visual. Aquesta secció cobrirà tècniques bàsiques de postprocessament que són essencials perquè els principiants les entenguin i les apliquin als seus projectes.

Conceptes clau

  • Compositing: Combinar múltiples imatges o efectes per crear una imatge final.
  • Nodes: Blocs de construcció utilitzats en el compositor de Blender per aplicar efectes i ajustos.
  • Render Layers: Diferents capes d'una escena que es poden renderitzar per separat i després combinar.
  • Correcció de color: Ajustar els colors de la teva imatge per aconseguir un aspecte desitjat.
  • Filtres: Aplicar efectes com desenfocament, afilat i més per millorar la imatge.

Configuració del compositor

Per començar a utilitzar tècniques de postprocessament, cal configurar el compositor de Blender.

  1. Activar Nodes:

    • Ves a l'espai de treball Compositing.
    • Marca la casella Use Nodes.

  2. Afegir un Node de Render Layer:

    • Prem Shift + A per afegir un node.
    • Navega a Input > Render Layers.

  3. Afegir un Node Composite:

    • Prem Shift + A per afegir un node.
    • Navega a Output > Composite.

  4. Connectar Nodes:

    • Connecta la sortida Image del node Render Layers a l'entrada Image del node Composite.

Configuració bàsica de nodes

Aquí tens un exemple senzill d'una configuració bàsica de nodes per al postprocessament:

[Render Layers] --> [Composite]

Aquesta configuració assegura que la teva imatge renderitzada passi pel compositor i es produeixi com la imatge final.

Correcció de color

La correcció de color és crucial per aconseguir l'ambient i el to adequats en els teus renders. Blender proporciona diversos nodes per a la correcció de color:

  • RGB Curves: Ajustar els canals vermell, verd i blau individualment.
  • Color Balance: Ajustar el lift, gamma i gain de la imatge.
  • Hue Saturation Value: Ajustar el to, la saturació i el valor de la imatge.

Exemple: Ajustar les corbes RGB

  1. Afegir un Node RGB Curves:

    • Prem Shift + A per afegir un node.
    • Navega a Color > RGB Curves.

  2. Connectar els Nodes:

    • Connecta la sortida Image del node Render Layers a l'entrada Image del node RGB Curves.
    • Connecta la sortida Image del node RGB Curves a l'entrada Image del node Composite.

  3. Ajustar les Corbes:

    • Fes clic al node RGB Curves.
    • Ajusta les corbes per canviar l'equilibri de color de la imatge.
[Render Layers] --> [RGB Curves] --> [Composite]

Aplicar filtres

Els filtres es poden utilitzar per millorar la imatge afegint efectes com desenfocament, afilat i més.

Exemple: Afegir un efecte de desenfocament

  1. Afegir un Node Blur:

    • Prem Shift + A per afegir un node.
    • Navega a Filter > Blur.

  2. Connectar els Nodes:

    • Connecta la sortida Image del node Render Layers a l'entrada Image del node Blur.
    • Connecta la sortida Image del node Blur a l'entrada Image del node Composite.

  3. Ajustar els Paràmetres de Desenfocament:

    • Fes clic al node Blur.
    • Ajusta els valors X i Y per controlar la quantitat de desenfocament.
[Render Layers] --> [Blur] --> [Composite]

Render Layers i màscares

Les capes de render permeten separar diferents elements de la teva escena i compondre'ls junts. Les màscares es poden utilitzar per aïllar àrees específiques de la teva imatge per a ajustos dirigits.

Exemple: Utilitzar Render Layers

  1. Crear Múltiples Render Layers:

    • A la pestanya Render Layers, crea múltiples capes per a diferents elements (per exemple, fons, personatge).

  2. Afegir Nodes de Render Layer:

    • Afegeix un node Render Layers per a cada capa.

  3. Combinar Capaes:

    • Utilitza nodes com Alpha Over per combinar les capes.
[Render Layer 1] --> [Alpha Over] --> [Composite]
[Render Layer 2] --> [Alpha Over]

Conclusió

El postprocessament és una eina poderosa a Blender que pot millorar significativament la qualitat dels teus renders. Entenent i utilitzant tècniques bàsiques com el compositing, la correcció de color i els filtres, pots aconseguir resultats professionals. Experimenta amb diferents nodes i configuracions per trobar la millor combinació per als teus projectes.
