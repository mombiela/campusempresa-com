Introducció

La texturització i l'ombrejat són aspectes crucials del modelatge 3D que donen vida als teus models afegint color, textura i propietats materials. Aquesta secció et guiarà des dels conceptes bàsics fins a tècniques avançades a Blender.

Conceptes bàsics de la texturització

La texturització implica aplicar imatges o patrons a la superfície d'un model 3D per donar-li detall i realisme.

Mapeig UV

  • Definició: El mapeig UV és el procés de projectar una textura d'imatge 2D sobre un model 3D.
  • Passos:
    1. Selecciona el teu model a Blender.
    2. Entra en el mode d'edició (tecla Tab).
    3. Desembolica el model (tecla U) i tria un mètode de desembolicat (per exemple, Smart UV Project).
    4. Obre l'editor UV/Image per veure el mapa UV desembolicat.
    5. Carrega o crea una textura i aplica-la al mapa UV.

Aplicació de textures

  • Textures d'imatge:
    1. Ves a l'espai de treball Shading.
    2. Afegeix un nou material a la pestanya Material Properties.
    3. Utilitza l'Shader Editor per afegir un node Image Texture.
    4. Connecta el node Image Texture a l'entrada Base Color del shader Principled BSDF.
    5. Carrega la teva textura d'imatge al node Image Texture.
# Exemple d'una configuració bàsica de nodes per aplicar una textura d'imatge
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="MyMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes

# Esborrar els nodes per defecte
for node in nodes:
    nodes.remove(node)

# Afegir el node Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = 0,0

# Afegir el node Image Texture
texture = nodes.new(type='ShaderNodeTexImage')
texture.location = -300,0
texture.image = bpy.data.images.load("path/to/your/image.png")

# Enllaçar nodes
material.node_tree.links.new(bsdf.inputs['Base Color'], texture.outputs['Color'])

# Assignar material a l'objecte
obj = bpy.context.active_object
if obj.data.materials:
    obj.data.materials[0] = material
else:
    obj.data.materials.append(material)

Tècniques de texturització intermèdies

Textures procedurals

Les textures procedurals es generen mitjançant algoritmes matemàtics en lloc d'arxius d'imatge.

  • Texture de soroll:

    • Afegeix un node Noise Texture a l'Shader Editor.
    • Connecta'l a l'entrada Base Color del shader Principled BSDF.
    • Ajusta els paràmetres Scale, Detail i Roughness per aconseguir l'efecte desitjat.

  • Texture Voronoi:

    • Afegeix un node Voronoi Texture.
    • Connecta'l a l'entrada Base Color.
    • Experimenta amb els paràmetres Scale i Randomness.

Pintura de textures

  • Passos:
    1. Selecciona el teu model i entra en el mode Texture Paint.
    2. Crea una nova textura al panell Texture Slots.
    3. Utilitza les eines de pintura per pintar directament sobre el model.

Conceptes bàsics de l'ombrejat

L'ombrejat defineix com la llum interactua amb les superfícies del teu model.

Shader Principled BSDF

  • Visió general: El shader Principled BSDF és un shader versàtil que combina múltiples capes d'ombrejat en una sola.
  • Entrades clau:
    • Base Color: El color base del material.
    • Metallic: Determina si el material és metàl·lic.
    • Roughness: Controla la suavitat de la superfície.
    • Specular: Ajusta la intensitat de les reflexions especulars.

Afegir shaders

  • Shader difús:

    • Utilitza per a superfícies no reflectants i mat.
    • Connecta el node Diffuse BSDF a l'entrada Surface del node Material Output.

  • Shader brillant:

    • Utilitza per a superfícies reflectants.
    • Connecta el node Glossy BSDF a l'entrada Surface.
# Exemple d'una configuració bàsica de nodes per a un shader brillant
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="GlossyMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes

# Esborrar els nodes per defecte
for node in nodes:
    nodes.remove(node)

# Afegir el node Glossy BSDF
glossy = nodes.new(type='ShaderNodeBsdfGlossy')
glossy.location = 0,0

# Enllaçar nodes
material.node_tree.links.new(glossy.outputs['BSDF'], nodes.new(type='ShaderNodeOutputMaterial').inputs['Surface'])

# Assignar material a l'objecte
obj = bpy.context.active_object
if obj.data.materials:
    obj.data.materials[0] = material
else:
    obj.data.materials.append(material)

Tècniques avançades d'ombrejat

Ombrejat basat en nodes

  • Mescla de shaders:

    • Utilitza el node Mix Shader per barrejar dos shaders diferents.
    • Connecta la sortida del node Mix Shader a l'entrada Surface del node Material Output.

  • Ús de coordenades de textura:

    • Afegeix un node Texture Coordinate per controlar com es mapegen les textures sobre el model.
    • Connecta la sortida UV a l'entrada Vector dels nodes de textura.

Dispersió subsuperficial (SSS)

  • Definició: La SSS simula la manera com la llum penetra la superfície d'un material translúcid, es dispersa a l'interior i surt per un punt diferent.
  • Ús:
    • Activa Subsurface en el shader Principled BSDF.
    • Ajusta el Subsurface Radius i el Subsurface Color per aconseguir l'efecte desitjat.

Conclusió

La texturització i l'ombrejat són habilitats essencials per crear models 3D realistes i visualment atractius a Blender. En dominar el mapeig UV, les textures procedurals, la pintura de textures i les tècniques avançades d'ombrejat, pots millorar significativament la qualitat dels teus projectes 3D. Practica aquestes tècniques regularment per convertir-te en un expert en la creació de models detallats i realistes.
