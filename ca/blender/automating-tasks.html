Introducció a l'Automatització a Blender

Automatitzar tasques a Blender pot millorar significativament el teu flux de treball, permetent-te realitzar tasques repetitives de manera ràpida i eficient. Aquesta secció et guiarà a través dels conceptes bàsics de l'automatització, començant per accions simples fins a scripts més complexos.

Ús d'Accions i Macros

Blender proporciona eines integrades per automatitzar tasques repetitives sense necessitat d'escriure codi.

  • Accions: Grava i reprodueix una sèrie de passos.
  • Macros: Combina múltiples accions en una sola comanda.

Exemple: Creació d'una Macro Simple

  1. Grava Accions: Comença a gravar les teves accions.
  2. Realitza Tasques: Executa les tasques que vols automatitzar.
  3. Guarda la Macro: Guarda les accions gravades com una macro.
1. Obre Blender.
2. Ves a l'espai de treball 'Scripting'.
3. Fes clic a 'Record Macro'.
4. Realitza les tasques (per exemple, crear un cub, escalar-lo i aplicar-li un material).
5. Atura la gravació i guarda la macro.

Introducció a la Programació amb Python

Python és el llenguatge de programació utilitzat a Blender. Permet una automatització i personalització més avançada.

  • Blender API: Accedeix a la funcionalitat de Blender programàticament.
  • Scripts: Escriu scripts en Python per automatitzar tasques.

Exemple: Script Bàsic en Python

Aquest script crea un cub i l'escala.

import bpy

# Crear un cub
bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))

# Escalar el cub
bpy.ops.transform.resize(value=(2, 2, 2))

Executar l'Script

  1. Obre Blender.
  2. Ves a l'espai de treball 'Scripting'.
  3. Crea un nou bloc de text.
  4. Copia i enganxa l'script.
  5. Fes clic a 'Run Script'.

Programació Avançada amb Python

Per a tasques més complexes, pots utilitzar funcions avançades de l'API de Blender.

  • Operadors Personalitzats: Crea noves comandes.
  • Handlers: Executa scripts en resposta a esdeveniments.
  • Add-ons: Empaqueta scripts en mòduls reutilitzables.

Exemple: Operador Personalitzat

Aquest script defineix un operador personalitzat per crear i escalar un cub.

import bpy

class SimpleOperator(bpy.types.Operator):
    bl_idname = "object.simple_operator"
    bl_label = "Simple Object Operator"
    
    def execute(self, context):
        bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
        bpy.ops.transform.resize(value=(2, 2, 2))
        return {'FINISHED'}

# Registrar l'operador
bpy.utils.register_class(SimpleOperator)

# Executar l'operador
bpy.ops.object.simple_operator()

Exemple: Ús de Handlers

Aquest script utilitza un handler per executar una funció quan es canvia de fotograma.

import bpy

def frame_change_handler(scene):
    print("Canvi de Fotograma", scene.frame_current)

# Afegir el handler
bpy.app.handlers.frame_change_pre.append(frame_change_handler)

Creació d'Add-ons

Els add-ons són una manera de distribuir els teus scripts i fer-los fàcilment reutilitzables.

Exemple: Add-on Simple

Aquest add-on crea un panell amb un botó per crear i escalar un cub.

bl_info = {
    "name": "Simple Add-on",
    "blender": (2, 80, 0),
    "category": "Object",
}

import bpy

class SimpleOperator(bpy.types.Operator):
    bl_idname = "object.simple_operator"
    bl_label = "Simple Object Operator"
    
    def execute(self, context):
        bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
        bpy.ops.transform.resize(value=(2, 2, 2))
        return {'FINISHED'}

class SimplePanel(bpy.types.Panel):
    bl_label = "Simple Panel"
    bl_idname = "OBJECT_PT_simple_panel"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = 'Tool'
    
    def draw(self, context):
        layout = self.layout
        layout.operator("object.simple_operator")

def register():
    bpy.utils.register_class(SimpleOperator)
    bpy.utils.register_class(SimplePanel)

def unregister():
    bpy.utils.unregister_class(SimpleOperator)
    bpy.utils.unregister_class(SimplePanel)

if __name__ == "__main__":
    register()

Conclusió

Automatitzar tasques a Blender pot millorar enormement la teva eficiència i productivitat. Començant amb accions i macros simples, pots avançar gradualment cap a la programació més avançada amb Python i fins i tot crear els teus propis add-ons. En dominar aquestes tècniques, podràs adaptar Blender per satisfer millor el teu flux de treball i necessitats creatives.
