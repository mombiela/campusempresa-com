Les textures procedimentals a Blender són una eina poderosa per crear textures complexes, detallades i dinàmiques sense dependre d'arxius d'imatge. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades d'ús de textures procedimentals a Blender.

Introducció a les Textures Procedimentals

Les textures procedimentals es generen utilitzant algoritmes matemàtics en lloc d'imatges bitmap. Això permet un detall infinit i escalabilitat.

Conceptes Clau

  • Generació Procedimental: Crear textures utilitzant algoritmes.
  • Flux de Treball Basat en Nodes: Utilitzar l'Editor de Shaders de Blender per crear i manipular textures procedimentals.
  • Escalabilitat: Les textures procedimentals es poden escalar sense perdre detall.

Començant amb Textures Procedimentals

Nodes Bàsics i les Seves Funcions

L'Editor de Shaders de Blender proporciona diversos nodes per crear textures procedimentals. Aquí tens alguns nodes fonamentals:

  • Node de Coordenades de Textura: Proporciona diferents tipus de coordenades per al mapeig de textures.
  • Node de Textura de Soroll: Genera un patró de soroll.
  • Node de Textura Voronoi: Crea un patró cel·lular.
  • Node de Textura Musgrave: Produeix patrons de soroll fractal.
  • Node de ColorRamp: Assigna valors a colors.

Exemple: Creant una Textura de Soroll Simple

  1. Obre l'Editor de Shaders: Canvia a l'Editor de Shaders a Blender.
  2. Afegeix un Node de Textura de Soroll: Prem Shift + A, ves a Texture, i selecciona Noise Texture.
  3. Connecta al Principled BSDF: Connecta la sortida Color del node de Textura de Soroll a l'entrada Base Color del shader Principled BSDF.
  4. Ajusta els Paràmetres: Modifica l'Scale, Detail, i Roughness per veure com canvia la textura.
# Configuració d'Exemple de Nodes
# Node de Textura de Soroll
Noise Texture:
    Scale: 5.0
    Detail: 2.0
    Roughness: 0.5

# Connecta la Textura de Soroll al Principled BSDF
Noise Texture Color -> Principled BSDF Base Color

Tècniques Intermèdies

Combinant Múltiples Textures

Pots combinar múltiples textures procedimentals per crear materials més complexos.

  • Node de Mix Shader: Barreja dos shaders.
  • Node de MixRGB: Barreja dos colors o textures.

Exemple: Combinant Textures de Soroll i Voronoi

  1. Afegeix un Node de Textura Voronoi: Prem Shift + A, ves a Texture, i selecciona Voronoi Texture.
  2. Afegeix un Node de MixRGB: Prem Shift + A, ves a Color, i selecciona MixRGB.
  3. Connecta les Textures al MixRGB: Connecta la sortida Color de la Textura de Soroll a l'entrada Color1 del node MixRGB, i la sortida Color de la Textura Voronoi a l'entrada Color2.
  4. Connecta el MixRGB al Principled BSDF: Connecta la sortida Color del node MixRGB a l'entrada Base Color del shader Principled BSDF.
  5. Ajusta el Factor de Barreja: Modifica el valor Fac del node MixRGB per barrejar les textures.
# Configuració d'Exemple de Nodes
# Node de Textura de Soroll
Noise Texture:
    Scale: 5.0
    Detail: 2.0
    Roughness: 0.5

# Node de Textura Voronoi
Voronoi Texture:
    Scale: 10.0

# Node de MixRGB
MixRGB:
    Fac: 0.5
    Color1: Noise Texture Color
    Color2: Voronoi Texture Color

# Connecta el MixRGB al Principled BSDF
MixRGB Color -> Principled BSDF Base Color

Tècniques Avançades

Ús de Nodes Matemàtics per a Patrons Personalitzats

Els nodes matemàtics es poden utilitzar per crear patrons i efectes personalitzats manipulant les sortides de les textures.

  • Node Matemàtic: Realitza operacions matemàtiques sobre les entrades.

Exemple: Creant un Patró de Ratlles

  1. Afegeix un Node de Coordenades de Textura: Prem Shift + A, ves a Input, i selecciona Texture Coordinate.
  2. Afegeix un Node de Separar XYZ: Prem Shift + A, ves a Converter, i selecciona Separate XYZ.
  3. Afegeix un Node Matemàtic: Prem Shift + A, ves a Converter, i selecciona Math.
  4. Configura el Node Matemàtic a Sine: Canvia l'operació del node Matemàtic a Sine.
  5. Connecta els Nodes: Connecta la sortida Generated del node de Coordenades de Textura a l'entrada Vector del node de Separar XYZ. Connecta la sortida X del node de Separar XYZ a l'entrada Value del node Matemàtic.
  6. Connecta al Principled BSDF: Connecta la sortida Value del node Matemàtic a l'entrada Base Color del shader Principled BSDF.
# Configuració d'Exemple de Nodes
# Node de Coordenades de Textura
Texture Coordinate:
    Generated -> Separate XYZ Vector

# Node de Separar XYZ
Separate XYZ:
    X -> Math Value

# Node Matemàtic
Math:
    Operation: Sine

# Connecta el Node Matemàtic al Principled BSDF
Math Value -> Principled BSDF Base Color

Conclusió

Les textures procedimentals a Blender ofereixen una manera versàtil i poderosa de crear materials detallats i escalables. Entenent els nodes bàsics i aprenent a combinar-los i manipular-los, pots crear una gran varietat de textures per als teus projectes. Experimenta amb diferents nodes i paràmetres per descobrir patrons i efectes únics.
