Introducció al Mapeig UV

El mapeig UV és una tècnica utilitzada en la modelització 3D per projectar una textura d'imatge 2D sobre un model 3D. Aquest procés és essencial per afegir textures detallades als teus models en Blender.

Conceptes Clau

  • Coordenades UV: Aquestes són les coordenades que mapegen la textura 2D al model 3D.
  • Desenvolupament: El procés d'aplanar la superfície d'un model 3D en un pla 2D.
  • Costures: Vores marcades per definir on es 'tallarà' el model per al desenvolupament.
  • Textura: La imatge 2D aplicada al model 3D.

Mapeig UV Bàsic

Configuració del Teu Model

  1. Crear o Importar un Model: Comença amb un model senzill com un cub o una esfera.
  2. Canviar a l'Espai de Treball d'Edició UV: Ves a l'espai de treball d'Edició UV en Blender.

Desenvolupament del Teu Model

  1. Marcar Costures:
    • Selecciona les vores on vols crear costures.
    • Prem Ctrl + E i tria "Marcar Costura".
  2. Desenvolupar:
    • Selecciona tot el model en Mode d'Edició.
    • Prem U i tria "Desenvolupar".

Aplicació d'una Textura

  1. Carregar una Textura:
    • A l'Editor UV, obre un fitxer d'imatge per utilitzar com a textura.
  2. Assignar la Textura:
    • A l'Editor de Shaders, afegeix un node de Textura d'Imatge i connecta'l al Color Base del shader Principled BSDF.
    • Carrega el mateix fitxer d'imatge al node de Textura d'Imatge.

Exemple

# Aquest és un script de Python per automatitzar el desenvolupament UV en Blender
import bpy

# Selecciona l'objecte
obj = bpy.context.active_object

# Entra en Mode d'Edició
bpy.ops.object.mode_set(mode='EDIT')

# Marca costures
bpy.ops.mesh.mark_seam(clear=False)

# Desenvolupa
bpy.ops.uv.unwrap(method='ANGLE_BASED', margin=0.001)

# Torna al Mode Objecte
bpy.ops.object.mode_set(mode='OBJECT')

Mapeig UV Intermedi

Tècniques Avançades de Desenvolupament

  • Projecte UV Intel·ligent: Desenvolupa automàticament el model basant-se en un llindar d'angle.
  • Paquet de Mapes de Llum: Crea un mapa UV adequat per a mapes de llum.

Ajustament dels UVs

  • Escalat i Rotació dels UVs: Utilitza S per escalar i R per rotar les illes UV.
  • Fixació dels UVs: Fixa vèrtexs a l'Editor UV per mantenir-los fixos durant desenvolupaments posteriors.

Exemple

# Ajustament dels UVs utilitzant Python
import bpy

# Selecciona l'objecte
obj = bpy.context.active_object

# Entra en Mode d'Edició
bpy.ops.object.mode_set(mode='EDIT')

# Selecciona tots els vèrtexs UV
bpy.ops.uv.select_all(action='SELECT')

# Escala els UVs
bpy.ops.transform.resize(value=(1.5, 1.5, 1.5))

# Rota els UVs
bpy.ops.transform.rotate(value=0.785398, orient_axis='Z')

# Torna al Mode Objecte
bpy.ops.object.mode_set(mode='OBJECT')

Mapeig UV Avançat

Mapeig UV de Models Complexos

  • Ús de Múltiples Mapes UV: Crea i gestiona múltiples mapes UV per a diferents textures.
  • Disposicions UV Personalitzades: Ajusta manualment les illes UV per a un ús òptim de l'espai de textura.

Cocció de Textures

  • Oclusió Ambiental: Cou mapes d'oclusió ambiental per afegir ombrejat realista.
  • Mapes Normals: Cou mapes normals per afegir detall de superfície sense augmentar la geometria.

Exemple

# Cocció d'un mapa d'Oclusió Ambiental
import bpy

# Selecciona l'objecte
obj = bpy.context.active_object

# Assegura't que l'objecte té un mapa UV
if not obj.data.uv_layers:
    bpy.ops.uv.smart_project()

# Crea una nova imatge per a la cocció
bpy.ops.image.new(name="AO_Bake", width=1024, height=1024)

# Configura la imatge com a activa
bpy.data.images['AO_Bake'].use_fake_user = True

# Configura el tipus de cocció a Oclusió Ambiental
bpy.context.scene.render.bake.use_pass_ambient_occlusion = True

# Cou la textura
bpy.ops.object.bake(type='AO')

# Desa la imatge cuita
bpy.data.images['AO_Bake'].save_render(filepath="/path/to/save/AO_Bake.png")

Conclusió

El mapeig UV és una habilitat crítica per afegir textures detallades als teus models 3D en Blender. Començant amb el desenvolupament bàsic i l'aplicació de textures, pots progressar a tècniques més avançades com disposicions UV personalitzades i cocció de textures. El domini del mapeig UV millorarà significativament el realisme i el detall dels teus projectes 3D.
