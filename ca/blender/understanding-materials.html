Introducció als materials a Blender

Els materials a Blender defineixen l'aparença dels objectes. Controlen com les superfícies interactuen amb la llum, incloent-hi el color, la reflectivitat, la transparència i la textura.

Conceptes clau

  • Material: Un conjunt de propietats que defineixen l'aparença d'un objecte.
  • Shader: Un programa que calcula el color i l'aparença d'una superfície.
  • Textura: Una imatge o patró aplicat a un material per donar-li més detall.

Creació bàsica de materials

Creació d'un material simple

  1. Selecciona un objecte: Fes clic a l'objecte al qual vols aplicar un material.
  2. Obre les propietats del material: Al panell de propietats, fes clic a la pestanya de Material.
  3. Afegeix un nou material: Fes clic al botó "Nou" per crear un nou material.
# Exemple: Creació d'un material bàsic a Blender utilitzant Python
import bpy

# Selecciona l'objecte
obj = bpy.context.active_object

# Crea un nou material
material = bpy.data.materials.new(name="BasicMaterial")

# Assigna el material a l'objecte
if obj.data.materials:
    obj.data.materials[0] = material
else:
    obj.data.materials.append(material)

Ajustar les propietats del material

  • Color base: El color principal del material.
  • Especular: Controla la brillantor del material.
  • Rugositat: Determina com de suau o rugosa apareix la superfície.

Tècniques intermèdies de materials

Ús de nodes per a la creació de materials

L'Editor de Nodes de Blender permet la creació de materials més complexos utilitzant un flux de treball basat en nodes.

  1. Obre l'Editor de Shaders: Canvia a l'espai de treball de l'Editor de Shaders.
  2. Afegeix nodes: Utilitza nodes com Principled BSDF, Image Texture i Mix Shader per crear materials complexos.
# Exemple: Creació d'un material amb nodes
import bpy

# Crea un nou material
material = bpy.data.materials.new(name="NodeMaterial")
material.use_nodes = True

# Obté l'arbre de nodes del material
nodes = material.node_tree.nodes

# Esborra els nodes per defecte
nodes.clear()

# Afegeix el node Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = (0, 0)

# Afegeix el node de sortida de material
output = nodes.new(type='ShaderNodeOutputMaterial')
output.location = (200, 0)

# Enllaça els nodes
links = material.node_tree.links
links.new(bsdf.outputs['BSDF'], output.inputs['Surface'])

Afegir textures

Les textures afegeixen detall als materials. Poden ser imatges o patrons procedurals.

  1. Afegeix un node de textura d'imatge: A l'Editor de Shaders, afegeix un node Image Texture.
  2. Carrega una imatge: Obre un fitxer d'imatge per utilitzar-lo com a textura.
  3. Connecta la textura: Enllaça la sortida Color del node Image Texture a l'entrada Base Color del node Principled BSDF.
# Exemple: Afegir una textura d'imatge a un material
import bpy

# Crea un nou material
material = bpy.data.materials.new(name="TextureMaterial")
material.use_nodes = True

# Obté l'arbre de nodes del material
nodes = material.node_tree.nodes
links = material.node_tree.links

# Afegeix el node de textura d'imatge
texture_node = nodes.new(type='ShaderNodeTexImage')
texture_node.location = (-300, 0)

# Carrega una imatge
texture_node.image = bpy.data.images.load('/path/to/your/image.jpg')

# Afegeix el node Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = (0, 0)

# Afegeix el node de sortida de material
output = nodes.new(type='ShaderNodeOutputMaterial')
output.location = (200, 0)

# Enllaça els nodes
links.new(texture_node.outputs['Color'], bsdf.inputs['Base Color'])
links.new(bsdf.outputs['BSDF'], output.inputs['Surface'])

Tècniques avançades de materials

Textures procedurals

Les textures procedurals es generen matemàticament i poden ser altament personalitzables.

  • Textura de soroll: Crea un patró de soroll aleatori.
  • Textura Voronoi: Genera un patró cel·lular.

Ús de múltiples shaders

Combina múltiples shaders per crear materials complexos.

  • Mix Shader: Barreja dos shaders.
  • Add Shader: Afegeix els efectes de dos shaders.
# Exemple: Combinació de shaders
import bpy

# Crea un nou material
material = bpy.data.materials.new(name="ComplexMaterial")
material.use_nodes = True

# Obté l'arbre de nodes del material
nodes = material.node_tree.nodes
links = material.node_tree.links

# Afegeix nodes
bsdf1 = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf1.location = (0, 0)

bsdf2 = nodes.new(type='ShaderNodeBsdfGlossy')
bsdf2.location = (0, -200)

mix_shader = nodes.new(type='ShaderNodeMixShader')
mix_shader.location = (200, 0)

output = nodes.new(type='ShaderNodeOutputMaterial')
output.location = (400, 0)

# Enllaça els nodes
links.new(bsdf1.outputs['BSDF'], mix_shader.inputs[1])
links.new(bsdf2.outputs['BSDF'], mix_shader.inputs[2])
links.new(mix_shader.outputs['Shader'], output.inputs['Surface'])

Conclusió

Comprendre els materials a Blender és crucial per crear models 3D realistes i visualment atractius. Des de la creació bàsica de materials fins a tècniques avançades com l'ús de nodes i textures procedurals, dominar aquests conceptes millorarà significativament les teves habilitats de modelatge 3D. Practica experimentant amb diferents propietats i combinacions per veure com afecten l'aparença dels teus objectes.
