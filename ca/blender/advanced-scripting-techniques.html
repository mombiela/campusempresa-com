Introducció

En aquesta secció, ens endinsarem en tècniques avançades de scripting a Blender utilitzant Python. Això es basarà en els coneixements fonamentals que has adquirit en seccions anteriors i introduirà conceptes i mètodes més complexos per millorar les teves capacitats de scripting.

Conceptes Clau

  • Operadors Personalitzats: Crear i utilitzar operadors personalitzats per estendre la funcionalitat de Blender.
  • Operadors Modal: Implementar operadors que s'executen contínuament i interactuen amb l'entrada de l'usuari.
  • Panells de UI Personalitzats: Dissenyar i integrar panells de UI personalitzats dins de la interfície de Blender.
  • Gestió de Propietats: Gestionar propietats personalitzades i utilitzar-les de manera efectiva en scripts.
  • Manipulació d'Escenes: Tècniques avançades per manipular escenes, objectes i blocs de dades.
  • Add-ons: Crear i gestionar add-ons de Blender per a distribució i reutilització.

Operadors Personalitzats

Els operadors personalitzats et permeten estendre la funcionalitat de Blender creant noves comandes.

Exemple: Creant un Operador Personalitzat Simple

import bpy

class SimpleOperator(bpy.types.Operator):
    bl_idname = "object.simple_operator"
    bl_label = "Simple Operator"
    
    def execute(self, context):
        self.report({'INFO'}, "Simple Operator Executed")
        return {'FINISHED'}

def register():
    bpy.utils.register_class(SimpleOperator)

def unregister():
    bpy.utils.unregister_class(SimpleOperator)

if __name__ == "__main__":
    register()
  • Explicació:
    • bl_idname: Identificador únic per a l'operador.
    • bl_label: Nom de visualització per a l'operador.
    • execute(): Mètode que s'executa quan es crida l'operador.

Operadors Modal

Els operadors modal s'executen contínuament i poden interactuar amb l'entrada de l'usuari, fent-los adequats per a tasques que requereixen interacció contínua.

Exemple: Creant un Operador Modal

import bpy

class ModalOperator(bpy.types.Operator):
    bl_idname = "object.modal_operator"
    bl_label = "Modal Operator"
    
    def modal(self, context, event):
        if event.type in {'ESC'}:
            return {'CANCELLED'}
        return {'RUNNING_MODAL'}
    
    def invoke(self, context, event):
        context.window_manager.modal_handler_add(self)
        return {'RUNNING_MODAL'}

def register():
    bpy.utils.register_class(ModalOperator)

def unregister():
    bpy.utils.unregister_class(ModalOperator)

if __name__ == "__main__":
    register()
  • Explicació:
    • modal(): Mètode que gestiona els esdeveniments.
    • invoke(): Mètode que inicia l'operador modal.

Panells de UI Personalitzats

Els panells de UI personalitzats et permeten crear nous panells a la interfície de Blender per allotjar les teves eines i configuracions.

Exemple: Creant un Panell de UI Personalitzat

import bpy

class SimplePanel(bpy.types.Panel):
    bl_label = "Simple Panel"
    bl_idname = "OBJECT_PT_simple_panel"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = 'Tool'
    
    def draw(self, context):
        layout = self.layout
        layout.label(text="Hello, World!")

def register():
    bpy.utils.register_class(SimplePanel)

def unregister():
    bpy.utils.unregister_class(SimplePanel)

if __name__ == "__main__":
    register()
  • Explicació:
    • bl_space_type: Defineix on apareixerà el panell.
    • bl_region_type: Defineix la regió dins de l'espai.
    • draw(): Mètode per definir el disseny del panell.

Gestió de Propietats

Gestionar propietats personalitzades és crucial per crear scripts dinàmics i interactius.

Exemple: Afegint Propietats Personalitzades

import bpy

bpy.types.Scene.my_custom_float = bpy.props.FloatProperty(name="My Float", default=1.0)

class CustomPropertyPanel(bpy.types.Panel):
    bl_label = "Custom Property Panel"
    bl_idname = "OBJECT_PT_custom_property_panel"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = 'Tool'
    
    def draw(self, context):
        layout = self.layout
        layout.prop(context.scene, "my_custom_float")

def register():
    bpy.utils.register_class(CustomPropertyPanel)

def unregister():
    bpy.utils.unregister_class(CustomPropertyPanel)
    del bpy.types.Scene.my_custom_float

if __name__ == "__main__":
    register()
  • Explicació:
    • FloatProperty: Afegeix una propietat flotant a l'escena.
    • layout.prop(): Afegeix la propietat al panell.

Manipulació d'Escenes

La manipulació avançada d'escenes implica modificar objectes, blocs de dades i altres elements dins de l'escena de Blender.

Exemple: Duplicant Objectes

import bpy

class DuplicateObjectsOperator(bpy.types.Operator):
    bl_idname = "object.duplicate_objects"
    bl_label = "Duplicate Objects"
    
    def execute(self, context):
        for obj in context.selected_objects:
            new_obj = obj.copy()
            new_obj.data = obj.data.copy()
            context.collection.objects.link(new_obj)
        return {'FINISHED'}

def register():
    bpy.utils.register_class(DuplicateObjectsOperator)

def unregister():
    bpy.utils.unregister_class(DuplicateObjectsOperator)

if __name__ == "__main__":
    register()
  • Explicació:
    • obj.copy(): Duplica l'objecte.
    • context.collection.objects.link(): Afegeix l'objecte duplicat a l'escena.

Add-ons

Crear add-ons et permet empaquetar els teus scripts per a distribució i reutilització.

Exemple: Estructura Bàsica d'un Add-on

bl_info = {
    "name": "Simple Add-on",
    "blender": (2, 80, 0),
    "category": "Object",
}

import bpy

class SimpleAddonOperator(bpy.types.Operator):
    bl_idname = "object.simple_addon_operator"
    bl_label = "Simple Add-on Operator"
    
    def execute(self, context):
        self.report({'INFO'}, "Simple Add-on Operator Executed")
        return {'FINISHED'}

def menu_func(self, context):
    self.layout.operator(SimpleAddonOperator.bl_idname)

def register():
    bpy.utils.register_class(SimpleAddonOperator)
    bpy.types.VIEW3D_MT_object.append(menu_func)

def unregister():
    bpy.utils.unregister_class(SimpleAddonOperator)
    bpy.types.VIEW3D_MT_object.remove(menu_func)

if __name__ == "__main__":
    register()
  • Explicació:
    • bl_info: Metadades per a l'add-on.
    • menu_func(): Afegeix l'operador a un menú.
