Introducció

En aquesta secció, explorarem com crear i aplicar textures a Blender. La texturització és un aspecte crucial del modelatge 3D que afegeix realisme i detall als teus models. Començarem amb els conceptes bàsics i gradualment passarem a tècniques més avançades.

Comprensió de les Textures

  • Definició: Les textures són imatges o patrons aplicats a la superfície d'un model 3D per donar-li color, detall i realisme.
  • Tipus de Textures:
    • Difusa: Color bàsic de la superfície.
    • Especular: Reflectivitat de la superfície.
    • Normal: Simula petits detalls de la superfície.
    • Bump: Crea la il·lusió de profunditat i detall.
    • Desplaçament: Realment altera la geometria de la superfície.

Aplicació Bàsica de Textures

Afegir una Textura a un Model

  1. Selecciona l'Objecte: Fes clic a l'objecte que vols texturitzar.
  2. Canvia a l'Espai de Treball de Shading: Ves a la barra superior i selecciona 'Shading'.
  3. Afegeix un Material:
    • Al panell de Propietats, ves a la pestanya 'Material'.
    • Fes clic a 'Nou' per crear un nou material.
  4. Afegeix un Node de Textura:
    • Al Shader Editor, prem Shift + A per afegir un nou node.
    • Navega a 'Texture' i selecciona 'Image Texture'.
  5. Carrega una Imatge:
    • Fes clic a 'Open' al node de Image Texture i selecciona el teu fitxer d'imatge de textura.
  6. Connecta els Nodes:
    • Connecta la sortida 'Color' del node de Image Texture a l'entrada 'Base Color' del node Principled BSDF.
# Exemple d'una configuració bàsica de nodes a l'Editor de Shaders de Blender
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="TextureMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes

# Esborrar nodes per defecte
for node in nodes:
    nodes.remove(node)

# Afegir node Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = 0, 0

# Afegir node de Textura d'Imatge
texture = nodes.new(type='ShaderNodeTexImage')
texture.location = -300, 0

# Carregar una imatge
texture.image = bpy.data.images.load('/path/to/your/image.png')

# Connectar nodes
material.node_tree.links.new(bsdf.inputs['Base Color'], texture.outputs['Color'])

# Assignar material a l'objecte actiu
bpy.context.object.active_material = material

UV Mapping

  • Definició: El UV mapping és el procés de projectar una textura d'imatge 2D sobre un model 3D.
  • Passos per UV Map:
    1. Selecciona l'Objecte: Entra en Mode d'Edició (Tab).
    2. Desenvolupar el Model:
      • Selecciona totes les cares (A).
      • Prem U i tria un mètode de desenvolupament (per exemple, 'Smart UV Project').
    3. Ajustar els UVs:
      • Canvia a l'espai de treball de UV Editing.
      • Ajusta les illes UV per adaptar-les correctament a la textura.

Tècniques Avançades de Texturització

Ús de Múltiples Textures

  • Barrejar Textures: Combina diferents textures utilitzant nodes Mix Shader.
  • Capar Textures: Utilitza múltiples nodes de Image Texture i barreja'ls utilitzant diferents modes de barreja.

Textures Procedimentals

  • Definició: Les textures procedimentals es generen utilitzant algoritmes matemàtics en lloc de fitxers d'imatge.
  • Creació de Textures Procedimentals:
    • Afegeix nodes com 'Noise Texture', 'Voronoi Texture', etc.
    • Combina'ls utilitzant nodes com 'MixRGB', 'ColorRamp', etc.
# Exemple d'una configuració de textura procedural
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="ProceduralMaterial")
material.use_nodes = True
nodes = material.node_tree.nodes

# Esborrar nodes per defecte
for node in nodes:
    nodes.remove(node)

# Afegir node Principled BSDF
bsdf = nodes.new(type='ShaderNodeBsdfPrincipled')
bsdf.location = 0, 0

# Afegir node de Textura de Soroll
noise = nodes.new(type='ShaderNodeTexNoise')
noise.location = -300, 0

# Afegir node ColorRamp
color_ramp = nodes.new(type='ShaderNodeValToRGB')
color_ramp.location = -150, 0

# Connectar nodes
material.node_tree.links.new(color_ramp.inputs['Fac'], noise.outputs['Fac'])
material.node_tree.links.new(bsdf.inputs['Base Color'], color_ramp.outputs['Color'])

# Assignar material a l'objecte actiu
bpy.context.object.active_material = material

Baking de Textures

  • Definició: El baking de textures és el procés de transferir detalls d'un model a un altre o de textures procedimentals a textures d'imatge.
  • Passos per Bake Textures:
    1. Crear una Nova Imatge: Al UV/Image Editor, crea una nova imatge.
    2. Configurar els Ajustos de Bake: Al panell de Propietats de Render, configura el tipus de bake (per exemple, 'Diffuse', 'Normal').
    3. Bakejar la Textura: Selecciona l'objecte i fes clic a 'Bake'.

Conclusió

La texturització és una habilitat essencial en el modelatge 3D que millora l'atractiu visual dels teus models. Des de l'aplicació bàsica de textures fins a tècniques avançades com la texturització procedural i el baking de textures, dominar aquestes habilitats millorarà significativament la qualitat dels teus projectes 3D. Practica regularment i experimenta amb diferents textures i mètodes per convertir-te en un expert en texturització a Blender.
