Resum

Blender és una potent suite de creació 3D de codi obert que suporta la totalitat del pipeline 3D. Python és un llenguatge de programació versàtil que Blender utilitza per a scripting i automatització. Aquesta secció t'introduirà a l'ús de Python dins de Blender, cobrint els conceptes bàsics i proporcionant exemples per començar.

Per què utilitzar Python en Blender?

  • Automatització: Automatitza tasques repetitives per estalviar temps.
  • Personalització: Crea eines i complements personalitzats per ampliar la funcionalitat de Blender.
  • Eficiència: Millora l'eficiència del flux de treball scriptant operacions complexes.

Configuració de Python en Blender

Blender ve amb un intèrpret de Python integrat, així que no cal cap configuració addicional. Pots començar a scriptar directament dins de Blender.

Accés a la Consola de Python

  1. Obre Blender.
  2. Canvia a l'espai de treball "Scripting" des del menú superior.
  3. Veureu la Consola de Python on podeu escriure i executar comandes Python.

Comandes bàsiques de Python en Blender

Comencem amb algunes comandes bàsiques per familiaritzar-nos amb l'API de Python en Blender.

Imprimir un missatge

print("Hola, Blender!")
  • Aquesta comanda imprimirà "Hola, Blender!" a la consola.

Accés a les dades de Blender

import bpy

# Imprimir el nom de l'escena actual
print(bpy.context.scene.name)
  • bpy és el mòdul de Python de Blender.
  • bpy.context.scene et dóna accés a l'escena actual.
  • .name recupera el nom de l'escena.

Creació i manipulació d'objectes

Afegir un cub

import bpy

# Afegir un cub a l'escena
bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
  • bpy.ops.mesh.primitive_cube_add és l'operador per afegir un cub.
  • size estableix la mida del cub.
  • location estableix la posició del cub en l'espai 3D.

Moure un objecte

import bpy

# Seleccionar el cub
cube = bpy.context.selected_objects[0]

# Moure el cub
cube.location.x += 2
  • bpy.context.selected_objects retorna una llista d'objectes seleccionats.
  • cube.location.x accedeix a la coordenada x de la ubicació del cub i el mou 2 unitats.

Creació d'un script senzill

Creem un script senzill que afegeix un cub i el mou.

import bpy

def add_and_move_cube():
    # Afegir un cub
    bpy.ops.mesh.primitive_cube_add(size=2, location=(0, 0, 0))
    
    # Seleccionar el cub
    cube = bpy.context.selected_objects[0]
    
    # Moure el cub
    cube.location.x += 2
    cube.location.y += 3
    cube.location.z += 1

# Executar la funció
add_and_move_cube()
  • Aquest script defineix una funció add_and_move_cube que afegeix un cub i el mou a una nova ubicació.
  • La funció es crida després per executar les operacions.

Conclusió

En aquesta secció, hem introduït els conceptes bàsics de l'ús de Python en Blender. Has après com accedir a la consola de Python, executar comandes senzilles i crear un script bàsic per afegir i manipular objectes. El scripting en Python a Blender pot millorar significativament la teva productivitat i permetre't crear eines personalitzades adaptades al teu flux de treball. A mesura que avancis en el curs, aprofundiràs en tècniques de scripting més avançades i aplicacions.
