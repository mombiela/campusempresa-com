Introducció a la Il·luminació
Tipus de Llums a Blender ►

La il·luminació és un aspecte crucial del modelatge i renderització 3D a Blender. Pot afectar dràsticament l'ambient, el realisme i la qualitat general de la teva escena. Aquesta secció et guiarà a través dels conceptes bàsics de la il·luminació a Blender, des de la comprensió dels diferents tipus de llums fins a tècniques avançades per crear configuracions d'il·luminació realistes.

Tipus de Llums a Blender

Blender ofereix diversos tipus de llums, cadascun amb propietats i usos únics. Comprendre aquests tipus és el primer pas per dominar la il·luminació a Blender.

  • Llum Puntual: Emissió de llum uniformement en totes les direccions des d'un punt únic, similar a una bombeta.
  • Llum Solar: Simula la llum del sol, proporcionant raigs de llum paral·lels, ideal per a escenes exteriors.
  • Llum Focal: Emissió de llum en forma de con, útil per destacar àrees específiques.
  • Llum d'Àrea: Emissió de llum des d'una àrea rectangular o quadrada, proporcionant ombres suaus i una il·luminació més realista.
  • Llum Hemisfèrica: Simula la llum ambiental del cel, útil per a escenes exteriors.

Exemple: Afegir una Llum Puntual

Per afegir una Llum Puntual a Blender:

  1. Afegir Llum: Prem Shift + A > Llum > Puntual.
  2. Posicionar Llum: Mou la llum a la ubicació desitjada utilitzant la tecla G.
  3. Ajustar Propietats: Al panell de Propietats, ajusta la intensitat, el color i altres configuracions de la llum.
# Exemple d'afegir una Llum Puntual utilitzant script en Python a Blender
import bpy

# Crear una nova llum puntual
light_data = bpy.data.lights.new(name="Nova Llum Puntual", type='POINT')
light_object = bpy.data.objects.new(name="Nova Llum Puntual", object_data=light_data)

# Enllaçar l'objecte llum a l'escena
bpy.context.collection.objects.link(light_object)

# Establir la ubicació de la llum
light_object.location = (5.0, 5.0, 5.0)

# Ajustar propietats de la llum
light_data.energy = 1000  # Ajustar intensitat
light_data.color = (1.0, 0.8, 0.6)  # Ajustar color

Tècniques d'Il·luminació

Es poden utilitzar diferents tècniques per aconseguir diversos efectes d'il·luminació. Aquí hi ha algunes tècniques fonamentals:

  • Il·luminació de Tres Punts: Un mètode estàndard que utilitza tres llums (Llum Principal, Llum de Farciment i Llum de Fons) per il·luminar un subjecte.
  • Il·luminació Global: Simula la il·luminació indirecta, fent que les escenes semblin més realistes.
  • Il·luminació HDRI: Utilitza Imatges d'Alta Gamma Dinàmica per proporcionar una il·luminació ambiental realista.

Configuració d'Il·luminació de Tres Punts

  1. Llum Principal: La font de llum principal, normalment posicionada en un angle respecte al subjecte.
  2. Llum de Farciment: Redueix les ombres creades per la Llum Principal, col·locada al costat oposat.
  3. Llum de Fons: Afegeix profunditat il·luminant el subjecte des del darrere.
# Exemple de configuració d'Il·luminació de Tres Punts utilitzant script en Python a Blender
import bpy

# Funció per crear una llum
def create_light(name, light_type, location, energy):
    light_data = bpy.data.lights.new(name=name, type=light_type)
    light_object = bpy.data.objects.new(name=name, object_data=light_data)
    bpy.context.collection.objects.link(light_object)
    light_object.location = location
    light_data.energy = energy
    return light_object

# Llum Principal
create_light("Llum Principal", 'SPOT', (5.0, -5.0, 5.0), 1000)

# Llum de Farciment
create_light("Llum de Farciment", 'SPOT', (-5.0, -5.0, 5.0), 500)

# Llum de Fons
create_light("Llum de Fons", 'SPOT', (0.0, 5.0, 5.0), 750)

Tècniques d'Il·luminació Avançades

Per a usuaris més avançats, Blender ofereix eines i tècniques sofisticades per crear il·luminacions altament realistes:

  • Il·luminació Volumètrica: Afegeix profunditat i atmosfera simulant la dispersió de la llum a través d'un medi.
  • Probes de Llum: Capturen i reemeten la llum en una escena, útil per a il·luminació dinàmica en entorns complexos.
  • Textures de Llum Personalitzades: Utilitza textures per crear patrons i efectes de llum personalitzats.

Exemple: Il·luminació Volumètrica

  1. Afegir Dispersió de Volum: Al Shader Editor, afegeix un node de Dispersió de Volum al material del Món o de l'Objecte.
  2. Ajustar Densitat: Controla la densitat del volum per aconseguir l'efecte desitjat.
# Exemple de configuració d'Il·luminació Volumètrica utilitzant script en Python a Blender
import bpy

# Crear un nou material per a il·luminació volumètrica
material = bpy.data.materials.new(name="MaterialVolumètric")
material.use_nodes = True
nodes = material.node_tree.nodes

# Esborrar nodes per defecte
for node in nodes:
    nodes.remove(node)

# Afegir node de Dispersió de Volum
volume_scatter = nodes.new(type='ShaderNodeVolumeScatter')
volume_scatter.location = (0, 0)
volume_scatter.inputs['Density'].default_value = 0.1

# Afegir node de Sortida de Material
volume_output = nodes.new(type='ShaderNodeOutputMaterial')
volume_output.location = (200, 0)

# Enllaçar nodes
material.node_tree.links.new(volume_scatter.outputs['Volume'], volume_output.inputs['Volume'])

# Assignar material al món
bpy.context.scene.world.use_nodes = True
world_nodes = bpy.context.scene.world.node_tree.nodes
world_nodes.clear()
world_nodes.new(type='ShaderNodeBackground')
world_output = world_nodes.new(type='ShaderNodeOutputWorld')
world_nodes.new(type='ShaderNodeVolumeScatter')
bpy.context.scene.world.node_tree.links.new(world_nodes['Volume Scatter'].outputs['Volume'], world_output.inputs['Volume'])

Conclusió

La il·luminació és una habilitat essencial per crear escenes 3D atractives i realistes a Blender. Comprenent els diferents tipus de llums, tècniques bàsiques i avançades d'il·luminació, i com implementar-les, pots millorar significativament la qualitat dels teus renders. Practica aquestes tècniques, experimenta amb diferents configuracions, i estaràs en bon camí per dominar la il·luminació a Blender.
