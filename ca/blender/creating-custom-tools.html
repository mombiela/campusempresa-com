Creació d'eines personalitzades a Blender
Tècniques Avançades de Scripting ►

Introducció

La creació d'eines personalitzades a Blender pot millorar significativament el teu flux de treball automatitzant tasques repetitives i afegint noves funcionalitats. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en la creació d'eines personalitzades utilitzant les capacitats de scripting de Blender.

Prerequisits

Abans d'endinsar-te en la creació d'eines personalitzades, assegura't de tenir:

  • Comprensió bàsica de la interfície i les funcionalitats de Blender.
  • Familiaritat amb la programació en Python.

Configuració de l'entorn de scripting

Per començar a crear eines personalitzades, necessites configurar l'entorn de scripting de Blender.

  1. Accés a l'espai de treball de scripting:

    • Obre Blender.
    • Canvia a l'espai de treball "Scripting" des del menú superior.

  2. Creació d'un nou script:

    • A l'Editor de Text, fes clic a "New" per crear un nou script.
    • Desa el script amb un nom significatiu, per exemple, custom_tool.py.

Exemple bàsic d'eina personalitzada

Creem una eina personalitzada senzilla que afegeix un cub a l'escena.

  1. Estructura bàsica del script:

    import bpy

def add_cube():
    bpy.ops.mesh.primitive_cube_add()

# Registra la funció com a operador
class OBJECT_OT_add_cube(bpy.types.Operator):
    bl_idname = "object.add_cube"
    bl_label = "Add Cube"
    bl_options = {'REGISTER', 'UNDO'}

    def execute(self, context):
        add_cube()
        return {'FINISHED'}

# Registra l'operador
def register():
    bpy.utils.register_class(OBJECT_OT_add_cube)

def unregister():
    bpy.utils.unregister_class(OBJECT_OT_add_cube)

if __name__ == "__main__":
    register()

  2. Explicació:

    • import bpy: Importa l'API de Python de Blender.
    • add_cube(): Funció per afegir un cub a l'escena.
    • OBJECT_OT_add_cube: Defineix una nova classe d'operador per afegir un cub.
    • register() i unregister(): Funcions per registrar i desregistrar l'operador.

  3. Execució del script:

    • Fes clic a "Run Script" a l'Editor de Text.
    • Prem F3 i cerca "Add Cube" per executar l'eina personalitzada.

Exemple intermedi d'eina personalitzada

Millora l'eina personalitzada afegint paràmetres per controlar la mida del cub.

  1. Estructura millorada del script:

    import bpy

def add_cube(size):
    bpy.ops.mesh.primitive_cube_add(size=size)

class OBJECT_OT_add_custom_cube(bpy.types.Operator):
    bl_idname = "object.add_custom_cube"
    bl_label = "Add Custom Cube"
    bl_options = {'REGISTER', 'UNDO'}

    size: bpy.props.FloatProperty(name="Size", default=2.0)

    def execute(self, context):
        add_cube(self.size)
        return {'FINISHED'}

def register():
    bpy.utils.register_class(OBJECT_OT_add_custom_cube)

def unregister():
    bpy.utils.unregister_class(OBJECT_OT_add_custom_cube)

if __name__ == "__main__":
    register()

  2. Explicació:

    • size: bpy.props.FloatProperty: Afegeix una propietat per controlar la mida del cub.
    • add_cube(size): La funció ara accepta un paràmetre de mida.

  3. Execució del script millorada:

    • Fes clic a "Run Script".
    • Prem F3, cerca "Add Custom Cube" i estableix el paràmetre de mida.

Exemple avançat d'eina personalitzada

Crea una eina més complexa que afegeix una graella de cubs.

  1. Estructura avançada del script:

    import bpy

def add_cube_grid(rows, cols, size):
    for i in range(rows):
        for j in range(cols):
            bpy.ops.mesh.primitive_cube_add(size=size, location=(i*size*2, j*size*2, 0))

class OBJECT_OT_add_cube_grid(bpy.types.Operator):
    bl_idname = "object.add_cube_grid"
    bl_label = "Add Cube Grid"
    bl_options = {'REGISTER', 'UNDO'}

    rows: bpy.props.IntProperty(name="Rows", default=3)
    cols: bpy.props.IntProperty(name="Columns", default=3)
    size: bpy.props.FloatProperty(name="Size", default=2.0)

    def execute(self, context):
        add_cube_grid(self.rows, self.cols, self.size)
        return {'FINISHED'}

def register():
    bpy.utils.register_class(OBJECT_OT_add_cube_grid)

def unregister():
    bpy.utils.unregister_class(OBJECT_OT_add_cube_grid)

if __name__ == "__main__":
    register()

  2. Explicació:

    • add_cube_grid(rows, cols, size): Funció per afegir una graella de cubs.
    • rows i cols: Propietats per definir el nombre de files i columnes.
    • location=(i*size*2, j*size*2, 0): Calcula la posició de cada cub a la graella.

  3. Execució del script avançat:

    • Fes clic a "Run Script".
    • Prem F3, cerca "Add Cube Grid" i estableix els paràmetres de files, columnes i mida.

Conclusió

La creació d'eines personalitzades a Blender utilitzant scripting en Python pot millorar enormement la teva productivitat automatitzant tasques i afegint noves funcionalitats. Des de scripts bàsics fins a eines més avançades, pots adaptar Blender a les teves necessitats específiques. Practica creant diferents eines i explora l'extensa API de Blender per desbloquejar tot el seu potencial.
