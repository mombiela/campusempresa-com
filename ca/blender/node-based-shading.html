Creació de Materials Complexos ►

L'Ombrejat Basat en Nodes és una manera potent i flexible de crear materials i textures a Blender. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades, assegurant que entenguis com aprofitar el sistema de nodes de Blender per crear visuals impressionants.

Introducció a l'Ombrejat Basat en Nodes

L'Ombrejat Basat en Nodes et permet crear materials complexos connectant diferents nodes, cadascun representant una funció o operació específica. Aquest enfocament és no lineal i altament personalitzable.

Conceptes Clau

  • Nodes: Elements individuals que realitzen funcions específiques, com generar textures, barrejar colors o aplicar transformacions.
  • Sòcols: Punts de connexió en els nodes per a dades d'entrada i sortida.
  • Enllaços: Connexions entre nodes que permeten que les dades flueixin d'un node a un altre.
  • Editor de Shaders: L'espai de treball a Blender on crees i gestiones els teus materials basats en nodes.

Començant amb Nodes

Nodes Bàsics

  • Diffuse BSDF: Un shader simple que dispersa la llum de manera uniforme.
  • Glossy BSDF: Un shader que reflecteix la llum, creant una superfície brillant.
  • Mix Shader: Combina dos shaders basant-se en un factor.

Creant un Material Bàsic

  1. Obre l'Editor de Shaders.
  2. Afegeix un nou material.
  3. Connecta un node Diffuse BSDF al node de Sortida de Material.
  4. Ajusta el color del node Diffuse BSDF.
# Exemple d'una configuració bàsica de nodes a l'Editor de Shaders de Blender
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="BasicMaterial")
material.use_nodes = True

# Obtenir l'arbre de nodes del material
nodes = material.node_tree.nodes

# Netejar els nodes per defecte
nodes.clear()

# Afegir node Diffuse BSDF
diffuse = nodes.new(type='ShaderNodeBsdfDiffuse')
diffuse.location = (0, 0)

# Afegir node de Sortida de Material
material_output = nodes.new(type='ShaderNodeOutputMaterial')
material_output.location = (200, 0)

# Enllaçar Diffuse BSDF a Sortida de Material
links = material.node_tree.links
link = links.new(diffuse.outputs[0], material_output.inputs[0])

Tècniques Intermèdies

Mapeig de Textures

  • Node de Textura d'Imatge: Carrega un fitxer d'imatge per utilitzar-lo com a textura.
  • Node de Mapeig: Ajusta la posició, rotació i escala de les textures.
  • Node de Coordenades de Textura: Proporciona diferents maneres de mapar textures sobre objectes.

Creant un Material Texturat

  1. Afegeix un node de Textura d'Imatge.
  2. Connecta'l a un node Diffuse BSDF.
  3. Utilitza un node de Mapeig per ajustar la col·locació de la textura.
  4. Connecta un node de Coordenades de Textura al node de Mapeig.
# Exemple d'una configuració de material texturat
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="TexturedMaterial")
material.use_nodes = True

# Obtenir l'arbre de nodes del material
nodes = material.node_tree.nodes

# Netejar els nodes per defecte
nodes.clear()

# Afegir node de Textura d'Imatge
image_texture = nodes.new(type='ShaderNodeTexImage')
image_texture.location = (-300, 0)
image_texture.image = bpy.data.images.load("path/to/your/image.jpg")

# Afegir node Diffuse BSDF
diffuse = nodes.new(type='ShaderNodeBsdfDiffuse')
diffuse.location = (0, 0)

# Afegir node de Sortida de Material
material_output = nodes.new(type='ShaderNodeOutputMaterial')
material_output.location = (200, 0)

# Enllaçar Textura d'Imatge a Diffuse BSDF
links = material.node_tree.links
link = links.new(image_texture.outputs[0], diffuse.inputs[0])

# Enllaçar Diffuse BSDF a Sortida de Material
link = links.new(diffuse.outputs[0], material_output.inputs[0])

Tècniques Avançades

Textures Procedurals

  • Textura de Soroll: Genera un patró de soroll.
  • Textura Voronoi: Crea un patró cel·lular.
  • Textura Musgrave: Produeix diversos patrons fractals.

Creant un Material Procedural

  1. Afegeix un node de Textura de Soroll.
  2. Connecta'l a un node de ColorRamp per controlar la sortida de color.
  3. Utilitza la sortida de ColorRamp per conduir un node Diffuse BSDF.
# Exemple d'una configuració de material procedural
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="ProceduralMaterial")
material.use_nodes = True

# Obtenir l'arbre de nodes del material
nodes = material.node_tree.nodes

# Netejar els nodes per defecte
nodes.clear()

# Afegir node de Textura de Soroll
noise_texture = nodes.new(type='ShaderNodeTexNoise')
noise_texture.location = (-300, 0)

# Afegir node de ColorRamp
color_ramp = nodes.new(type='ShaderNodeValToRGB')
color_ramp.location = (-100, 0)

# Afegir node Diffuse BSDF
diffuse = nodes.new(type='ShaderNodeBsdfDiffuse')
diffuse.location = (100, 0)

# Afegir node de Sortida de Material
material_output = nodes.new(type='ShaderNodeOutputMaterial')
material_output.location = (300, 0)

# Enllaçar Textura de Soroll a ColorRamp
links = material.node_tree.links
link = links.new(noise_texture.outputs[0], color_ramp.inputs[0])

# Enllaçar ColorRamp a Diffuse BSDF
link = links.new(color_ramp.outputs[0], diffuse.inputs[0])

# Enllaçar Diffuse BSDF a Sortida de Material
link = links.new(diffuse.outputs[0], material_output.inputs[0])

Conclusió

L'Ombrejat Basat en Nodes a Blender ofereix una flexibilitat i control immensos sobre els teus materials. Entenent i utilitzant nodes, pots crear qualsevol cosa, des de colors simples fins a textures generades proceduralment complexes. Comença amb els conceptes bàsics, experimenta amb tècniques intermèdies i explora mètodes avançats per aprofitar completament el poder de l'Editor de Shaders de Blender.
