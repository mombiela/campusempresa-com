Introducció als Mètodes Numèrics

Els mètodes numèrics són tècniques utilitzades per resoldre problemes matemàtics numèricament, és a dir, utilitzant nombres i algorismes en lloc de manipulacions simbòliques. Aquesta secció us introduirà als conceptes bàsics dels mètodes numèrics i com es poden implementar en Fortran.

  • Definició: Els mètodes numèrics impliquen algorismes per resoldre problemes matemàtics.
  • Aplicacions: Enginyeria, física, finances i altres camps que requereixen solucions numèriques.

Sintaxi i Estructures Bàsiques de Fortran

Abans d'endinsar-nos en els mètodes numèrics, és essencial entendre la sintaxi i les estructures bàsiques de Fortran.

  • Variables i Tipus de Dades: INTEGER, REAL, DOUBLE PRECISION, COMPLEX, etc.
  • Estructures de Control: Sentències IF, bucles DO, etc.
  • Funcions i Subrutines: Modularitzar el codi per a la reutilització.

Exemple: Programa Bàsic en Fortran

PROGRAM HelloWorld
  PRINT *, 'Hello, World!'
END PROGRAM HelloWorld

Mètodes de Cerca de Ràdics

Els mètodes de cerca de ràdics s'utilitzen per trobar les arrels d'equacions. Els mètodes comuns inclouen el Mètode de Bisecció i el Mètode de Newton-Raphson.

Mètode de Bisecció

El Mètode de Bisecció és un mètode simple i robust per trobar arrels d'una funció.

  • Algorisme:
    1. Trieu dos punts \(a\) i \(b\) tal que \(f(a)\) i \(f(b)\) tinguin signes oposats.
    2. Calculeu el punt mig \(c = (a + b) / 2\).
    3. Comproveu el signe de \(f(c)\).
    4. Reemplaceu \(a\) o \(b\) amb \(c\) segons el signe.
    5. Repetiu fins que es trobi l'arrel dins d'una tolerància desitjada.

Exemple: Mètode de Bisecció en Fortran

PROGRAM BisectionMethod
  IMPLICIT NONE
  REAL :: a, b, c, tol
  INTEGER :: max_iter, iter

  a = 1.0
  b = 2.0
  tol = 1.0e-6
  max_iter = 100

  DO iter = 1, max_iter
    c = (a + b) / 2.0
    IF (ABS(f(c)) < tol) EXIT
    IF (f(a) * f(c) < 0.0) THEN
      b = c
    ELSE
      a = c
    END IF
  END DO

  PRINT *, 'L'arrel és aproximadament: ', c

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2 - 2.0
  END FUNCTION f
END PROGRAM BisectionMethod

Integració Numèrica

La integració numèrica implica aproximar la integral d'una funció. Els mètodes comuns inclouen la Regla del Trapezi i la Regla de Simpson.

Regla del Trapezi

La Regla del Trapezi aproxima la integral dividint l'àrea sota la corba en trapezis.

  • Algorisme:
    1. Divideix l'interval \([a, b]\) en \(n\) subintervals.
    2. Calcula la suma de les àrees dels trapezis.

Exemple: Regla del Trapezi en Fortran

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i * h)
  END DO

  integral = integral * h
  PRINT *, 'Integral aproximada: ', integral

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2
  END FUNCTION f
END PROGRAM TrapezoidalRule

Resolució de Sistemes d'Equacions Lineals

Resoldre sistemes d'equacions lineals és un problema fonamental en els mètodes numèrics. Les tècniques inclouen l'Eliminació Gaussiana i la Descomposició LU.

Eliminació Gaussiana

L'Eliminació Gaussiana és un mètode per resoldre sistemes lineals transformant la matriu del sistema en una forma triangular superior.

  • Algorisme:
    1. Eliminació cap endavant per formar una matriu triangular superior.
    2. Substitució cap enrere per resoldre les variables.

Exemple: Eliminació Gaussiana en Fortran

PROGRAM GaussianElimination
  IMPLICIT NONE
  REAL, DIMENSION(3,3) :: A
  REAL, DIMENSION(3) :: b, x
  INTEGER :: i, j, k

  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [1.0, 0.0, 4.0]

  CALL gauss_elimination(A, b, x)

  PRINT *, 'Solució: ', x

CONTAINS
  SUBROUTINE gauss_elimination(A, b, x)
    REAL, DIMENSION(:,:), INTENT(INOUT) :: A
    REAL, DIMENSION(:), INTENT(INOUT) :: b
    REAL, DIMENSION(:), INTENT(OUT) :: x
    INTEGER :: n, i, j, k
    REAL :: factor

    n = SIZE(b)
    DO k = 1, n-1
      DO i = k+1, n
        factor = A(i,k) / A(k,k)
        A(i,k:n) = A(i,k:n) - factor * A(k,k:n)
        b(i) = b(i) - factor * b(k)
      END DO
    END DO

    x(n) = b(n) / A(n,n)
    DO i = n-1, 1, -1
      x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
    END DO
  END SUBROUTINE gauss_elimination
END PROGRAM GaussianElimination

Conclusió

  • Sintaxi i estructures bàsiques de Fortran.
  • Mètodes de cerca de ràdics com el Mètode de Bisecció.
  • Integració numèrica utilitzant la Regla del Trapezi.
