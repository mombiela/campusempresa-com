Introducció

En Fortran, les estructures de control són essencials per dirigir el flux d'un programa. Una de les estructures de control més fonamentals és la sentència if, que et permet executar certes seccions de codi basades en condicions específiques. Aquest tema cobrirà els conceptes bàsics de les sentències if, incloent-hi la sintaxi, exemples i exercicis per ajudar-te a entendre i aplicar aquesta estructura de control de manera efectiva.

Sentència If bàsica

La forma més simple d'una sentència if en Fortran comprova una condició i executa un bloc de codi si la condició és certa.

Sintaxi

if (condition) then
    ! Codi a executar si la condició és certa
end if

Exemple

program basic_if
    implicit none
    integer :: a

    a = 5

    if (a > 3) then
        print *, "a és més gran que 3"
    end if
end program basic_if

En aquest exemple, el programa comprova si a és més gran que 3. Com que a és 5, la condició és certa, i es mostra el missatge "a és més gran que 3".

Sentència If-Else

De vegades, necessites executar un bloc de codi si una condició és certa i un altre bloc si la condició és falsa. Aquí és on entra la sentència if-else.

Sintaxi

if (condition) then
    ! Codi a executar si la condició és certa
else
    ! Codi a executar si la condició és falsa
end if

Exemple

program if_else
    implicit none
    integer :: a

    a = 2

    if (a > 3) then
        print *, "a és més gran que 3"
    else
        print *, "a no és més gran que 3"
    end if
end program if_else

En aquest exemple, el programa comprova si a és més gran que 3. Com que a és 2, la condició és falsa, i es mostra el missatge "a no és més gran que 3".

Sentència If-Else If-Else

Per a una presa de decisions més complexa, pots utilitzar múltiples condicions amb else if.

Sintaxi

if (condition1) then
    ! Codi a executar si la condició1 és certa
else if (condition2) then
    ! Codi a executar si la condició2 és certa
else
    ! Codi a executar si cap de les condicions anteriors és certa
end if

Exemple

program if_elseif_else
    implicit none
    integer :: a

    a = 5

    if (a > 10) then
        print *, "a és més gran que 10"
    else if (a > 3) then
        print *, "a és més gran que 3 però menor o igual a 10"
    else
        print *, "a és 3 o menys"
    end if
end program if_elseif_else

En aquest exemple, el programa comprova múltiples condicions. Com que a és 5, la segona condició és certa, i es mostra el missatge "a és més gran que 3 però menor o igual a 10".

Sentències If niades

També pots niar sentències if dins d'altres per manejar condicions més complexes.

Exemple

program nested_if
    implicit none
    integer :: a, b

    a = 5
    b = 7

    if (a > 3) then
        if (b > 6) then
            print *, "a és més gran que 3 i b és més gran que 6"
        else
            print *, "a és més gran que 3 però b no és més gran que 6"
        end if
    else
        print *, "a no és més gran que 3"
    end if
end program nested_if

En aquest exemple, el programa primer comprova si a és més gran que 3. Si és cert, després comprova si b és més gran que 6. Com que ambdues condicions són certes, es mostra el missatge "a és més gran que 3 i b és més gran que 6".

Exercicis

  1. Escriu un programa que comprovi si un número és positiu, negatiu o zero i mostri un missatge apropiat.
  2. Modifica l'exemple if_elseif_else per incloure una condició que comprovi si a és exactament 5 i mostri un missatge especial.
  3. Crea una sentència if niada que comprovi dues variables i mostri diferents missatges basats en els seus valors.

Conclusió

Entendre les sentències if en Fortran és crucial per controlar el flux dels teus programes. En dominar les sentències if bàsiques, if-else, if-else if-else i les sentències if niades, podràs manejar una àmplia gamma d'escenaris de presa de decisions en el teu codi. Practica amb els exemples i exercicis proporcionats per solidificar la teva comprensió i convertir-te en un expert en l'ús de les sentències if en Fortran.
