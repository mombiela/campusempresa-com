Introducció als Coarrays

Els coarrays són una característica de programació paral·lela en Fortran que permeten una computació paral·lela simple i eficient. Proporcionen una manera de distribuir dades entre múltiples processadors i realitzar operacions paral·leles.

Conceptes Clau

  • Imatge: Un fil d'execució independent amb la seva pròpia memòria local.
  • Coarray: Una estructura de dades que es distribueix entre múltiples imatges.
  • Sincronització: Mecanismes per coordinar operacions entre imatges.

Sintaxi Bàsica i Ús

Per utilitzar coarrays, cal declarar-los amb l'atribut codimension. Aquí teniu un exemple senzill:

program hello_coarray
  implicit none
  integer :: my_image, num_images
  integer, codimension[*] :: x

  ! Obtenir el número d'imatge actual i el nombre total d'imatges
  my_image = this_image()
  num_images = num_images()

  ! Inicialitzar el coarray
  x = my_image

  ! Sincronitzar totes les imatges
  sync all

  ! Imprimir el valor de x de cada imatge
  print *, 'Image ', my_image, ' has x = ', x

end program hello_coarray

Explicació

  • this_image(): Retorna el número d'imatge actual.
  • num_images(): Retorna el nombre total d'imatges.
  • codimension[*]: Declara x com un coarray.
  • sync all: Sincronitza totes les imatges per assegurar-se que han arribat al mateix punt del programa.

Característiques Avançades dels Coarrays

Comunicació amb Coarrays

Els coarrays permeten la comunicació directa entre imatges. Podeu accedir i modificar dades en altres imatges utilitzant claudàtors.

program coarray_communication
  implicit none
  integer :: my_image, num_images
  integer, codimension[*] :: x

  my_image = this_image()
  num_images = num_images()

  ! Inicialitzar el coarray
  x = my_image

  ! Sincronitzar totes les imatges
  sync all

  ! Exemple d'accés a dades d'una altra imatge
  if (my_image == 1) then
    x[2] = 10
  end if

  ! Sincronitzar totes les imatges
  sync all

  print *, 'Image ', my_image, ' has x = ', x

end program coarray_communication

Explicació

  • x[2] = 10: La Imatge 1 estableix el valor de x a la Imatge 2 a 10.
  • Això demostra com les imatges poden comunicar-se directament i modificar les dades de les altres.

Tècniques de Sincronització

Sync All

Sincronitza totes les imatges per assegurar-se que han arribat al mateix punt del programa.

Sync Images

Sincronitza un conjunt específic d'imatges.

program sync_images_example
  implicit none
  integer :: my_image, num_images
  integer, codimension[*] :: x

  my_image = this_image()
  num_images = num_images()

  x = my_image

  ! Sincronitzar només les imatges 1 i 2
  if (my_image <= 2) then
    sync images(1, 2)
  end if

  print *, 'Image ', my_image, ' has x = ', x

end program sync_images_example

Explicació

  • sync images(1, 2): Només les imatges 1 i 2 estan sincronitzades.

Consideracions de Rendiment

  • Equilibri de Càrrega: Assegureu-vos que el treball estigui distribuït uniformement entre les imatges.
  • Minimitzar la Comunicació: Una comunicació excessiva entre imatges pot degradar el rendiment.
  • Ús de Memòria: Tingueu en compte l'ús de memòria, especialment amb coarrays grans.

Conclusió

Els coarrays en Fortran proporcionen una manera potent i eficient de realitzar computacions paral·leles. Comprenent la sintaxi bàsica, els mètodes de comunicació i les tècniques de sincronització, podeu aprofitar els coarrays per escriure programes paral·lels d'alt rendiment. Recordeu considerar les implicacions de rendiment com l'equilibri de càrrega i la sobrecàrrega de comunicació per optimitzar les vostres aplicacions paral·leles.
