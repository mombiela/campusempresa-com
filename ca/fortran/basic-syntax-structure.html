Introducció al Fortran

Fortran, abreviatura de "Formula Translation," és un dels llenguatges de programació d'alt nivell més antics, utilitzat principalment per a la computació numèrica i científica. Entendre la seva sintaxi i estructura bàsica és crucial per escriure programes eficients i sense errors.

Conceptes Clau

  • Estructura del Programa: Els programes Fortran estan estructurats en unitats com programes, subrutines i funcions.
  • Comentaris: Línies que no s'executen, utilitzades per a la documentació.
  • Variables i Tipus de Dades: Fortran admet diversos tipus de dades, incloent enters, nombres reals i caràcters.
  • Operadors: Operadors aritmètics, relacionals i lògics.
  • Estructures de Control: Declaracions condicionals i bucles.
  • Entrada/Sortida: Llegir i escriure a fitxers o a la consola.

Estructura del Programa

Un programa Fortran bàsic consisteix en un nom de programa, declaracions, declaracions executables i una declaració de final.

program HelloWorld
  implicit none
  print *, 'Hello, World!'
end program HelloWorld
  • program HelloWorld: Declara el nom del programa.
  • implicit none: Assegura que totes les variables han de ser declarades explícitament.
  • print *, 'Hello, World!': Imprimeix text a la consola.
  • end program HelloWorld: Marca el final del programa.

Comentaris

Els comentaris s'utilitzen per explicar el codi i s'ignoren durant l'execució. En Fortran, els comentaris comencen amb un signe d'exclamació (!).

! Aquest és un comentari
print *, 'Hello, World!'  ! Això imprimeix Hello, World!

Variables i Tipus de Dades

Fortran admet diversos tipus de dades. Les variables han de ser declarades abans d'usar-les.

Tipus de Dades Comuns

  • Integer: Nombres enters.
  • Real: Nombres de punt flotant.
  • Character: Cadenes de text.
  • Logical: Valors booleans (cert o fals).
program VariablesExample
  implicit none
  integer :: i
  real :: x
  character(len=20) :: name
  logical :: flag

  i = 10
  x = 3.14
  name = 'Fortran'
  flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Character:', name
  print *, 'Logical:', flag
end program VariablesExample

Operadors

Fortran inclou diversos operadors per realitzar operacions aritmètiques, relacionals i lògiques.

Operadors Aritmètics

  • + Suma
  • - Resta
  • * Multiplicació
  • / Divisió
  • ** Exponenciació

Operadors Relacionals

  • == Igual a
  • /= Diferent de
  • > Major que
  • < Menor que
  • >= Major o igual que
  • <= Menor o igual que

Operadors Lògics

  • .and. Lògic AND
  • .or. Lògic OR
  • .not. Lògic NOT
program OperatorsExample
  implicit none
  integer :: a, b
  logical :: result

  a = 5
  b = 3

  print *, 'Suma:', a + b
  print *, 'Multiplicació:', a * b
  print *, 'Exponenciació:', a ** b

  result = (a > b) .and. (a /= 0)
  print *, 'Resultat Lògic AND:', result
end program OperatorsExample

Estructures de Control

Les estructures de control permeten l'execució condicional i la iteració.

Declaracions Condicionals

  • Declaració if
  • Declaració else if
  • Declaració else
program ConditionalExample
  implicit none
  integer :: num

  num = 10

  if (num > 0) then
    print *, 'El nombre és positiu'
  else if (num == 0) then
    print *, 'El nombre és zero'
  else
    print *, 'El nombre és negatiu'
  end if
end program ConditionalExample

Bucles

  • Bucle do
  • Bucle do while
program LoopExample
  implicit none
  integer :: i

  do i = 1, 5
    print *, 'Iteració:', i
  end do

  i = 1
  do while (i <= 5)
    print *, 'Iteració del bucle while:', i
    i = i + 1
  end do
end program LoopExample

Entrada/Sortida

Fortran proporciona maneres senzilles de gestionar operacions d'entrada i sortida.

Llegir Entrada

program InputExample
  implicit none
  integer :: num

  print *, 'Introdueix un nombre:'
  read *, num
  print *, 'Has introduït:', num
end program InputExample

Escriure Sortida

program OutputExample
  implicit none
  integer :: num

  num = 42
  print *, 'La resposta és:', num
end program OutputExample

Conclusió

Entendre la sintaxi i estructura bàsica del Fortran és essencial per escriure programes efectius. Això inclou saber com estructurar un programa, declarar variables, utilitzar operadors, controlar el flux del programa i gestionar l'entrada i sortida. Amb aquests fonaments, pots començar a explorar característiques i capacitats més avançades del Fortran.
