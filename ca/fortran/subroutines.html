Les subrutines són un concepte fonamental en Fortran, que et permeten encapsular i reutilitzar codi. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat de subrutines en Fortran.

Introducció a les Subrutines

Les subrutines en Fortran són similars a les funcions en altres llenguatges de programació. Et permeten descompondre el teu programa en peces més petites i manejables. Les subrutines poden prendre arguments, realitzar operacions i retornar resultats.

Conceptes Clau

  • Definició: Una subrutina es defineix utilitzant la paraula clau SUBROUTINE.
  • Arguments: Les subrutines poden acceptar arguments d'entrada i retornar resultats a través d'aquests arguments.
  • Crida: Les subrutines s'invoquen utilitzant la instrucció CALL.
  • Àmbit: Les variables dins d'una subrutina són locals a aquesta subrutina, tret que es comparteixin explícitament.

Estructura Bàsica d'una Subrutina

Aquí tens un exemple senzill per il·lustrar l'estructura bàsica d'una subrutina en Fortran.

PROGRAM Main
    CALL Greet
END PROGRAM Main

SUBROUTINE Greet
    PRINT *, "Hola, Món!"
END SUBROUTINE Greet

Explicació

  • PROGRAM Main: El bloc principal del programa.
  • CALL Greet: Involucra la subrutina Greet.
  • SUBROUTINE Greet: Defineix la subrutina anomenada Greet.
  • PRINT: Mostra "Hola, Món!" a la consola.
  • END SUBROUTINE Greet: Marca el final de la subrutina.

Subrutines amb Arguments

Les subrutines poden prendre arguments per realitzar operacions sobre dades d'entrada.

PROGRAM Main
    INTEGER :: num1, num2, result
    num1 = 10
    num2 = 20
    CALL Add(num1, num2, result)
    PRINT *, "La suma és: ", result
END PROGRAM Main

SUBROUTINE Add(a, b, sum)
    INTEGER, INTENT(IN) :: a, b
    INTEGER, INTENT(OUT) :: sum
    sum = a + b
END SUBROUTINE Add

Explicació

  • Arguments: a, b i sum són arguments de la subrutina Add.
  • INTENT(IN): Especifica que a i b són arguments d'entrada.
  • INTENT(OUT): Especifica que sum és un argument de sortida.
  • sum = a + b: Realitza la suma i emmagatzema el resultat en sum.

Característiques Avançades de les Subrutines

Arguments Opcional

Les subrutines poden tenir arguments opcionals, que no són necessaris quan es crida la subrutina.

PROGRAM Main
    CALL PrintMessage("Hola, Món!")
    CALL PrintMessage()
END PROGRAM Main

SUBROUTINE PrintMessage(msg)
    CHARACTER(LEN=*), OPTIONAL :: msg
    IF (PRESENT(msg)) THEN
        PRINT *, msg
    ELSE
        PRINT *, "No s'ha proporcionat cap missatge."
    END IF
END SUBROUTINE PrintMessage

Explicació

  • OPTIONAL: Indica que msg és un argument opcional.
  • PRESENT: Comprova si s'ha proporcionat l'argument opcional msg.

Subrutines Recursives

Les subrutines poden cridar-se a si mateixes de manera recursiva.

PROGRAM Main
    INTEGER :: result
    result = Factorial(5)
    PRINT *, "El factorial de 5 és: ", result
END PROGRAM Main

RECURSIVE SUBROUTINE Factorial(n, result)
    INTEGER, INTENT(IN) :: n
    INTEGER, INTENT(OUT) :: result
    IF (n <= 1) THEN
        result = 1
    ELSE
        CALL Factorial(n - 1, result)
        result = n * result
    END IF
END SUBROUTINE Factorial

Explicació

  • RECURSIVE: Indica que la subrutina pot cridar-se a si mateixa.
  • Factorial: Calcula el factorial de n utilitzant la recursió.

Resum

Les subrutines en Fortran són eines poderoses per organitzar i reutilitzar codi. Poden prendre arguments, retornar resultats i fins i tot cridar-se a si mateixes de manera recursiva. En dominar les subrutines, podràs escriure programes en Fortran més modulars, llegibles i mantenibles.

Punts Clau

  • Les subrutines ajuden a descompondre programes complexos en peces manejables.
  • Poden acceptar arguments d'entrada i retornar resultats.
  • Els arguments opcionals i la recursió afegeixen flexibilitat a l'ús de subrutines.

Practica escrivint i utilitzant subrutines per convertir-te en un expert en programació Fortran.
