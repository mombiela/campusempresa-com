Introducció

En aquesta secció, explorarem diversos estudis de cas i projectes que t'ajudaran a aplicar els conceptes que has après en Fortran. Aquests exemples pràctics aniran des de nivells principiants fins a avançats, permetent-te desenvolupar les teves habilitats progressivament. Al final d'aquesta secció, hauràs de ser capaç d'afrontar problemes del món real utilitzant Fortran.

Projectes de nivell principiant

1. Operacions matemàtiques simples

  • Objectiu: Implementar operacions aritmètiques bàsiques.
  • Conceptes coberts: Variables, E/S bàsica, operacions aritmètiques.

Codi d'exemple:

program simple_math
  implicit none
  real :: a, b, sum, difference, product, quotient

  ! Introduir valors
  print *, 'Enter first number:'
  read *, a
  print *, 'Enter second number:'
  read *, b

  ! Realitzar operacions
  sum = a + b
  difference = a - b
  product = a * b
  quotient = a / b

  ! Mostrar resultats
  print *, 'Sum:', sum
  print *, 'Difference:', difference
  print *, 'Product:', product
  print *, 'Quotient:', quotient
end program simple_math

2. Conversió de temperatura

  • Objectiu: Convertir temperatures entre Celsius i Fahrenheit.
  • Conceptes coberts: Declaracions condicionals, funcions.

Codi d'exemple:

program temp_conversion
  implicit none
  real :: celsius, fahrenheit

  ! Introduir temperatura en Celsius
  print *, 'Enter temperature in Celsius:'
  read *, celsius

  ! Convertir a Fahrenheit
  fahrenheit = celsius * 9.0 / 5.0 + 32.0

  ! Mostrar resultat
  print *, 'Temperature in Fahrenheit:', fahrenheit
end program temp_conversion

Projectes de nivell intermedi

1. Multiplicació de matrius

  • Objectiu: Multiplicar dues matrius.
  • Conceptes coberts: Arrays, bucles imbricats.

Codi d'exemple:

program matrix_multiplication
  implicit none
  integer, parameter :: n = 3
  real :: A(n, n), B(n, n), C(n, n)
  integer :: i, j, k

  ! Inicialitzar matrius A i B
  A = reshape([1, 2, 3, 4, 5, 6, 7, 8, 9], shape(A))
  B = reshape([9, 8, 7, 6, 5, 4, 3, 2, 1], shape(B))
  C = 0.0

  ! Multiplicació de matrius
  do i = 1, n
    do j = 1, n
      do k = 1, n
        C(i, j) = C(i, j) + A(i, k) * B(k, j)
      end do
    end do
  end do

  ! Mostrar resultat
  print *, 'Resultant Matrix C:'
  do i = 1, n
    print *, C(i, :)
  end do
end program matrix_multiplication

2. Integració numèrica

  • Objectiu: Implementar la regla trapezoïdal per a la integració numèrica.
  • Conceptes coberts: Funcions, bucles, mètodes numèrics.

Codi d'exemple:

program trapezoidal_rule
  implicit none
  real :: a, b, h, integral
  integer :: n, i

  ! Definir la funció a integrar
  contains
    real function f(x)
      real, intent(in) :: x
      f = x**2  ! Funció d'exemple: f(x) = x^2
    end function f

  ! Introduir límits i nombre d'intervals
  print *, 'Enter lower limit a:'
  read *, a
  print *, 'Enter upper limit b:'
  read *, b
  print *, 'Enter number of intervals n:'
  read *, n

  ! Calcular mida del pas
  h = (b - a) / n

  ! Aplicar regla trapezoïdal
  integral = 0.5 * (f(a) + f(b))
  do i = 1, n-1
    integral = integral + f(a + i * h)
  end do
  integral = integral * h

  ! Mostrar resultat
  print *, 'Integral:', integral
end program trapezoidal_rule

Projectes de nivell avançat

1. Resolució de sistemes lineals

  • Objectiu: Resoldre un sistema d'equacions lineals utilitzant eliminació gaussiana.
  • Conceptes coberts: Manipulacions avançades d'arrays, àlgebra lineal.

Codi d'exemple:

program gaussian_elimination
  implicit none
  integer, parameter :: n = 3
  real :: A(n, n+1)
  integer :: i, j, k
  real :: factor

  ! Inicialitzar matriu augmentada A
  A = reshape([2.0, -1.0, 1.0, 8.0, -3.0, -1.0, 2.0, -11.0, -2.0, 1.0, 2.0, -3.0], shape(A))

  ! Eliminació gaussiana
  do k = 1, n-1
    do i = k+1, n
      factor = A(i, k) / A(k, k)
      do j = k, n+1
        A(i, j) = A(i, j) - factor * A(k, j)
      end do
    end do
  end do

  ! Substitució cap enrere
  do i = n, 1, -1
    A(i, n+1) = A(i, n+1) / A(i, i)
    A(i, i) = 1.0
    do j = i-1, 1, -1
      A(j, n+1) = A(j, n+1) - A(j, i) * A(i, n+1)
      A(j, i) = 0.0
    end do
  end do

  ! Mostrar resultat
  print *, 'Solution:'
  do i = 1, n
    print *, 'x', i, '=', A(i, n+1)
  end do
end program gaussian_elimination

2. Simulació de sistemes físics

  • Objectiu: Simular el moviment d'un projectil.
  • Conceptes coberts: Equacions diferencials, simulacions físiques.

Codi d'exemple:

program projectile_motion
  implicit none
  real :: g, v0, theta, t, dt, x, y
  real, parameter :: pi = 3.141592653589793
  integer :: n, i

  ! Constants
  g = 9.81  ! Acceleració deguda a la gravetat (m/s^2)

  ! Introduir condicions inicials
  print *, 'Enter initial velocity (m/s):'
  read *, v0
  print *, 'Enter launch angle (degrees):'
  read *, theta

  ! Convertir angle a radians
  theta = theta * pi / 180.0

  ! Pas de temps i nombre de passos
  dt = 0.01
  n = 100

  ! Inicialitzar posició i temps
  x = 0.0
  y = 0.0
  t = 0.0

  ! Simular moviment
  do i = 1, n
    t = t + dt
    x = v0 * cos(theta) * t
    y = v0 * sin(theta) * t - 0.5 * g * t**2
    if (y < 0.0) exit
    print *, 'Time:', t, 's, Position: (', x, ',', y, ')'
  end do
end program projectile_motion
