Introducció a la Simulació i el Modelatge

La simulació i el modelatge són tècniques essencials en la computació científica, permetent als investigadors crear experiments virtuals i predir comportaments de sistemes complexos. Fortran, amb les seves potents capacitats de càlcul numèric, és àmpliament utilitzat en aquests camps.

Conceptes Clau

  • Modelatge: Crear una representació matemàtica d'un sistema del món real.
  • Simulació: Realitzar experiments sobre el model per estudiar el seu comportament sota diverses condicions.
  • Models Deterministes: Models on el resultat es determina precisament a través de relacions conegudes.
  • Models Estocàstics: Models que incorporen aleatorietat i comportament probabilístic.

Construir un Model Simple en Fortran

Exemple: Model de Creixement de la Població

Un exemple comú d'un model simple és el creixement exponencial d'una població.

Exemple de Codi

program population_growth
    implicit none
    integer :: i, n
    real :: P0, r, P

    ! Població inicial
    P0 = 100.0
    ! Taxa de creixement
    r = 0.05
    ! Nombre de passos de temps
    n = 10

    print *, 'Pas de Temps', 'Població'
    do i = 0, n
        P = P0 * exp(r * i)
        print *, i, P
    end do
end program population_growth

Explicació

  • P0: Població inicial.
  • r: Taxa de creixement.
  • n: Nombre de passos de temps.
  • exp: Funció intrínseca de Fortran per a l'exponencial.

Aquest programa calcula i imprimeix la població en cada pas de temps utilitzant la fórmula de creixement exponencial \( P = P0 \times e^{rt} \).

Tècniques Avançades de Modelatge

Equacions Diferencials

Molts sistemes del món real es modelen utilitzant equacions diferencials. Fortran pot resoldre aquestes utilitzant mètodes numèrics com el mètode d'Euler o els mètodes de Runge-Kutta.

Exemple: Resoldre una EDO utilitzant el Mètode d'Euler

Considerem l'equació diferencial \( \frac{dy}{dt} = -ky \).

Exemple de Codi

program euler_method
    implicit none
    integer :: i, n
    real :: y, t, dt, k

    ! Condicions inicials
    y = 1.0
    t = 0.0
    dt = 0.1
    k = 0.5
    n = 100

    print *, 'Temps', 'y'
    do i = 1, n
        y = y - k * y * dt
        t = t + dt
        print *, t, y
    end do
end program euler_method

Explicació

  • y: Variable dependent.
  • t: Variable independent (temps).
  • dt: Mida del pas de temps.
  • k: Constant de taxa.

Aquest programa utilitza el mètode d'Euler per resoldre l'equació diferencial i imprimeix el valor de \( y \) en cada pas de temps.

Modelatge Estocàstic

Simulacions de Monte Carlo

Els mètodes de Monte Carlo utilitzen l'aleatorietat per resoldre problemes que podrien ser deterministes en principi. Són particularment útils per a sistemes amb un gran nombre de graus de llibertat acoblats.

Exemple: Estimació de Pi

Un exemple clàssic de simulació de Monte Carlo és estimar el valor de Pi col·locant aleatòriament punts en un quadrat unitat i comptant quants cauen dins d'un quart de cercle.

Exemple de Codi

program monte_carlo_pi
    implicit none
    integer :: i, n, count
    real :: x, y, pi_estimate

    ! Nombre de punts aleatoris
    n = 1000000
    count = 0

    call random_seed()

    do i = 1, n
        call random_number(x)
        call random_number(y)
        if (x**2 + y**2 <= 1.0) count = count + 1
    end do

    pi_estimate = 4.0 * count / n
    print *, 'Pi Estimat = ', pi_estimate
end program monte_carlo_pi

Explicació

  • random_seed: Inicialitza el generador de nombres aleatoris.
  • random_number: Genera un nombre aleatori entre 0 i 1.
  • count: Compta el nombre de punts dins del quart de cercle.

Aquest programa estima Pi generant punts aleatoris i comprovant si cauen dins del quart de cercle.

Conclusió

La simulació i el modelatge són eines poderoses en la computació científica, i Fortran proporciona capacitats robustes per implementar aquestes tècniques. Des de models deterministes simples fins a simulacions estocàstiques complexes, la potència numèrica de Fortran el converteix en una excel·lent elecció per a investigadors i enginyers. En dominar aquests conceptes i tècniques, pots modelar i simular efectivament una àmplia gamma de sistemes del món real.
