Introducció a les Funcions

Les funcions en Fortran són blocs de construcció essencials que et permeten encapsular codi per a la seva reutilització, millorar la llegibilitat i gestionar la complexitat. En aquesta secció, cobrirem els conceptes bàsics de definir i utilitzar funcions en Fortran, progressant des d'exemples simples fins a conceptes més avançats.

Definició Bàsica de Funció

Una funció en Fortran és un bloc de codi que pren arguments d'entrada, realitza una tasca específica i retorna un valor. Aquí tens la sintaxi bàsica per definir una funció:

FUNCTION nom_de_la_funcio(arguments)
  ! Declaracions
  IMPLICIT NONE
  ! Cos de la funció
  nom_de_la_funcio = resultat
END FUNCTION nom_de_la_funcio

Exemple: Funció Simple

Comencem amb un exemple simple d'una funció que suma dos números:

PROGRAM main
  IMPLICIT NONE
  REAL :: resultat

  resultat = suma(2.0, 3.0)
  PRINT *, "La suma és: ", resultat

CONTAINS

  FUNCTION suma(a, b)
    REAL :: suma
    REAL, INTENT(IN) :: a, b

    suma = a + b
  END FUNCTION suma

END PROGRAM main
  • Explicació:
    • PROGRAM main: El bloc principal del programa.
    • IMPLICIT NONE: Assegura que totes les variables han de ser declarades explícitament.
    • REAL :: resultat: Declara una variable real per emmagatzemar el resultat.
    • resultat = suma(2.0, 3.0): Crida la funció suma amb els arguments 2.0 i 3.0.
    • CONTAINS: Indica que el bloc següent conté procediments interns.
    • FUNCTION suma(a, b): Defineix la funció suma.
    • REAL, INTENT(IN) :: a, b: Declara els arguments d'entrada a i b com a números reals.
    • suma = a + b: Calcula la suma i l'assigna al nom de la funció.

Arguments de Funció i Tipus de Retorn

Les funcions poden prendre diversos tipus d'arguments i retornar diferents tipus de valors. El tipus de retorn d'una funció es especifica en la seva declaració.

Exemple: Funció amb Arguments Enterers

Aquí tens un exemple d'una funció que multiplica dos enters:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: resultat

  resultat = multiplica(4, 5)
  PRINT *, "El producte és: ", resultat

CONTAINS

  FUNCTION multiplica(a, b)
    INTEGER :: multiplica
    INTEGER, INTENT(IN) :: a, b

    multiplica = a * b
  END FUNCTION multiplica

END PROGRAM main
  • Explicació:
    • INTEGER :: resultat: Declara una variable enter per emmagatzemar el resultat.
    • resultat = multiplica(4, 5): Crida la funció multiplica amb els arguments 4 i 5.
    • FUNCTION multiplica(a, b): Defineix la funció multiplica.
    • INTEGER, INTENT(IN) :: a, b: Declara els arguments d'entrada a i b com a enters.
    • multiplica = a * b: Calcula el producte i l'assigna al nom de la funció.

Conceptes Avançats de Funció

Funcions Recursives

Fortran suporta funcions recursives, que són funcions que es criden a elles mateixes. Per definir una funció recursiva, has d'utilitzar la paraula clau RECURSIVE.

Exemple: Funció Recursiva de Factorial

Aquí tens un exemple d'una funció recursiva per calcular el factorial d'un número:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: resultat

  resultat = factorial(5)
  PRINT *, "El factorial és: ", resultat

CONTAINS

  RECURSIVE FUNCTION factorial(n) RESULT(fact)
    INTEGER :: fact
    INTEGER, INTENT(IN) :: n

    IF (n <= 1) THEN
      fact = 1
    ELSE
      fact = n * factorial(n - 1)
    END IF
  END FUNCTION factorial

END PROGRAM main
  • Explicació:
    • RECURSIVE FUNCTION factorial(n) RESULT(fact): Defineix una funció recursiva factorial amb resultat fact.
    • IF (n <= 1) THEN fact = 1 ELSE fact = n * factorial(n - 1) END IF: Implementa la lògica recursiva per calcular el factorial.

Funcions amb Arguments de Matriu

Les funcions també poden prendre matrius com arguments i retornar matrius.

Exemple: Funció amb Arguments de Matriu

Aquí tens un exemple d'una funció que suma dues matrius element a element:

PROGRAM main
  IMPLICIT NONE
  REAL, DIMENSION(3) :: matriu1 = (/1.0, 2.0, 3.0/)
  REAL, DIMENSION(3) :: matriu2 = (/4.0, 5.0, 6.0/)
  REAL, DIMENSION(3) :: resultat

  resultat = suma_matrius(matriu1, matriu2)
  PRINT *, "La suma de les matrius és: ", resultat

CONTAINS

  FUNCTION suma_matrius(a, b) RESULT(suma)
    REAL, DIMENSION(3) :: suma
    REAL, DIMENSION(3), INTENT(IN) :: a, b

    suma = a + b
  END FUNCTION suma_matrius

END PROGRAM main
  • Explicació:
    • REAL, DIMENSION(3) :: matriu1, matriu2, resultat: Declara matrius amb tres elements.
    • resultat = suma_matrius(matriu1, matriu2): Crida la funció suma_matrius amb matriu1 i matriu2.
    • FUNCTION suma_matrius(a, b) RESULT(suma): Defineix la funció suma_matrius amb resultat suma.
    • REAL, DIMENSION(3), INTENT(IN) :: a, b: Declara els arguments d'entrada a i b com a matrius.
    • suma = a + b: Suma les matrius element a element i assigna el resultat a suma.

Conclusió

Les funcions en Fortran són eines poderoses que t'ajuden a organitzar i reutilitzar codi de manera eficient. Hem cobert els conceptes bàsics de definir funcions, utilitzar diferents tipus d'arguments i conceptes avançats com la recursió i els arguments de matriu. Dominant les funcions, podràs escriure programes en Fortran més modulars, llegibles i mantenibles.
