Introducció

Fortran, abreviatura de "Formula Translation", és un dels llenguatges de programació d'alt nivell més antics, utilitzat principalment per a càlculs numèrics i científics. Aquesta secció et guiarà a través de l'escriptura del teu primer programa en Fortran, cobrint els conceptes bàsics de sintaxi, estructura i execució.

Conceptes clau

  • Estructura del programa Fortran: Comprendre la disposició bàsica d'un programa Fortran.
  • Variables i tipus de dades: Introducció a la declaració i ús de variables.
  • Entrada i sortida bàsiques: Com llegir des de i escriure a la consola.
  • Compilar i executar: Passos per compilar i executar el teu codi Fortran.

Estructura del programa Fortran

Un programa Fortran senzill consta de les següents parts:

  1. Nom del programa: Declara el nom del programa.
  2. Implicit None: Assegura que totes les variables han de ser declarades explícitament.
  3. Declaracions de variables: Declara les variables utilitzades en el programa.
  4. Sentències executables: Conté la lògica i les operacions.
  5. End Program: Marca el final del programa.

Exemple

program HelloWorld
  implicit none
  ! Variable declarations go here

  ! Executable statements
  print *, 'Hello, World!'

end program HelloWorld

Explicació

  • program HelloWorld: Defineix l'inici del programa anomenat "HelloWorld".
  • implicit none: Assegura que totes les variables han de ser declarades explícitament, cosa que ajuda a prevenir errors.
  • print *, 'Hello, World!': Imprimeix la cadena "Hello, World!" a la consola.
  • end program HelloWorld: Marca el final del programa.

Variables i tipus de dades

Fortran admet diversos tipus de dades, incloent-hi enters, nombres reals i caràcters.

Exemple

program VariablesExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Assigning values to variables
  age = 25
  height = 5.9
  name = 'John Doe'

  ! Printing variables
  print *, 'Name:', name
  print *, 'Age:', age
  print *, 'Height:', height

end program VariablesExample

Explicació

  • integer :: age: Declara una variable enter anomenada age.
  • real :: height: Declara una variable real (de coma flotant) anomenada height.
  • character(len=20) :: name: Declara una variable de caràcter anomenada name amb una longitud de 20.
  • Les sentències print imprimeixen els valors de les variables a la consola.

Entrada i sortida bàsiques

Fortran utilitza les sentències read i print per a operacions d'entrada i sortida.

Exemple

program InputOutputExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Reading input from the user
  print *, 'Enter your name:'
  read *, name
  print *, 'Enter your age:'
  read *, age
  print *, 'Enter your height:'
  read *, height

  ! Printing the input values
  print *, 'Name:', name
  print *, 'Age:', age
  print *, 'Height:', height

end program InputOutputExample

Explicació

  • read *, name: Llegeix una entrada de cadena de l'usuari i l'emmagatzema a la variable name.
  • read *, age: Llegeix una entrada d'enter de l'usuari i l'emmagatzema a la variable age.
  • read *, height: Llegeix una entrada de nombre real de l'usuari i l'emmagatzema a la variable height.

Compilar i executar

Per compilar i executar un programa Fortran, normalment utilitzes un compilador Fortran com gfortran.

Passos

  1. Escriure el codi: Desa el teu codi Fortran en un fitxer amb una extensió .f90 (per exemple, HelloWorld.f90).
  2. Compilar el codi: Utilitza la comanda gfortran HelloWorld.f90 -o HelloWorld per compilar el codi.
  3. Executar l'executable: Executa el programa compilat amb ./HelloWorld.

Exemple

$ gfortran HelloWorld.f90 -o HelloWorld
$ ./HelloWorld
Hello, World!

Conclusió

En aquesta secció, has après com escriure un programa bàsic en Fortran, declarar variables, realitzar operacions d'entrada i sortida, i compilar i executar el teu codi. Aquestes habilitats fonamentals són essencials a mesura que avances cap a tasques de programació en Fortran més complexes. Practica escrivint programes senzills per reforçar aquests conceptes i construir la teva confiança en l'ús de Fortran.
