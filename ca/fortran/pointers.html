Introducció a les Punteres

Les punteres són una característica poderosa en Fortran que et permeten referenciar i manipular ubicacions de memòria directament. Això pot ser particularment útil per a l'assignació dinàmica de memòria, la creació d'estructures de dades complexes i l'optimització del rendiment.

Conceptes Clau

  • Declaració de Punteres: Com declarar una variable puntera.
  • Associació de Punteres: Associar una puntera amb un objectiu.
  • Operacions amb Punteres: Operacions bàsiques que pots realitzar amb punteres.
  • Assignació Dinàmica de Memòria: Utilitzar punteres per assignar i alliberar memòria dinàmicament.
  • Arrays de Punteres: Treballar amb arrays de punteres.

Declaració de Punteres

Per declarar una puntera en Fortran, utilitzes l'atribut POINTER en la declaració de tipus.

INTEGER, POINTER :: ptr
REAL, POINTER :: rptr

En aquest exemple:

  • ptr és una puntera a un enter.
  • rptr és una puntera a un número real.

Associació de Punteres

Les punteres necessiten ser associades amb un objectiu abans de poder ser utilitzades. Això es fa utilitzant l'operador =>.

INTEGER, TARGET :: target_var
INTEGER, POINTER :: ptr

ptr => target_var

En aquest exemple:

  • target_var és una variable enter que pot ser objectiu de punteres.
  • ptr està associada amb target_var.

Operacions amb Punteres

Un cop una puntera està associada amb un objectiu, pots realitzar diverses operacions sobre ella.

Exemple: Operacions Bàsiques amb Punteres

INTEGER, TARGET :: target_var = 10
INTEGER, POINTER :: ptr

ptr => target_var
PRINT *, "Valor de target_var: ", target_var
PRINT *, "Valor via puntera: ", ptr

ptr = 20
PRINT *, "Nou valor de target_var: ", target_var

Sortida:

Valor de target_var: 10
Valor via puntera: 10
Nou valor de target_var: 20

Assignació Dinàmica de Memòria

Les punteres són essencials per a l'assignació dinàmica de memòria en Fortran. Pots assignar i alliberar memòria utilitzant les instruccions ALLOCATE i DEALLOCATE.

Exemple: Assignació Dinàmica de Memòria

INTEGER, POINTER :: ptr
ALLOCATE(ptr)
ptr = 42
PRINT *, "Valor assignat dinàmicament: ", ptr
DEALLOCATE(ptr)

Sortida:

Valor assignat dinàmicament: 42

Arrays de Punteres

També pots crear arrays de punteres, que poden ser útils per gestionar col·leccions de memòria assignada dinàmicament.

Exemple: Arrays de Punteres

INTEGER, POINTER :: ptr(:)
ALLOCATE(ptr(5))

ptr(1) = 1
ptr(2) = 2
ptr(3) = 3
ptr(4) = 4
ptr(5) = 5

PRINT *, "Valors de l'array de punteres: ", ptr

DEALLOCATE(ptr)

Sortida:

Valors de l'array de punteres: 1 2 3 4 5

Resum

Les punteres en Fortran proporcionen una manera de referenciar i manipular ubicacions de memòria directament, cosa que és útil per a l'assignació dinàmica de memòria i la creació d'estructures de dades complexes. Els conceptes clau inclouen la declaració de punteres, l'associació, les operacions bàsiques, l'assignació dinàmica de memòria i els arrays de punteres. Entendre aquests conceptes et permetrà escriure programes en Fortran més flexibles i eficients.
