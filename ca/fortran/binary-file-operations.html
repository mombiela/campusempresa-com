Les operacions amb fitxers binaris són crucials per gestionar eficientment grans conjunts de dades i realitzar manipulacions de dades a baix nivell. Aquesta secció et guiarà des de les tècniques bàsiques fins a les avançades d'operacions amb fitxers binaris en Fortran.

Introducció als fitxers binaris

  • Fitxers binaris: A diferència dels fitxers de text, els fitxers binaris emmagatzemen dades en un format que no és llegible per humans. S'utilitzen per emmagatzemar dades com imatges, àudio i programes compilats.
  • Avantatges:
    • Operacions de lectura/escriptura més ràpides.
    • Emmagatzematge més compacte.
    • Accés directe a les dades.

Obrir i tancar fitxers binaris

Per treballar amb fitxers binaris en Fortran, cal obrir el fitxer en un mode específic i tancar-lo després de completar les operacions.

Obrir un fitxer binari

Utilitza la instrucció OPEN per obrir un fitxer binari.

OPEN(UNIT=10, FILE='data.bin', STATUS='UNKNOWN', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)
  • UNIT: Número d'unitat lògica.
  • FILE: Nom del fitxer.
  • STATUS: Estat del fitxer ('NEW', 'OLD', 'UNKNOWN').
  • ACCESS: Mode d'accés ('SEQUENTIAL', 'DIRECT').
  • FORM: Format del fitxer ('FORMATTED', 'UNFORMATTED').
  • RECL: Longitud del registre (en bytes).

Tancar un fitxer binari

Utilitza la instrucció CLOSE per tancar el fitxer.

CLOSE(UNIT=10)

Escriure en un fitxer binari

Per escriure dades en un fitxer binari, utilitza la instrucció WRITE amb el número d'unitat apropiat.

Exemple: Escriure enters en un fitxer binari

PROGRAM WriteBinary
  INTEGER :: i, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    WRITE(UNIT=unit_num, REC=i) i
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM WriteBinary
  • REC: Número de registre per accés directe.

Llegir d'un fitxer binari

Per llegir dades d'un fitxer binari, utilitza la instrucció READ amb el número d'unitat apropiat.

Exemple: Llegir enters d'un fitxer binari

PROGRAM ReadBinary
  INTEGER :: i, value, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i) value
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ReadBinary

Operacions avançades amb fitxers binaris

Gestionar diferents tipus de dades

Fortran permet escriure i llegir diversos tipus de dades a/de fitxers binaris.

Exemple: Escriure i llegir nombres reals

PROGRAM WriteReadReal
  REAL :: x, y
  INTEGER :: unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  x = 3.14
  WRITE(UNIT=unit_num, REC=1) x

  CLOSE(UNIT=unit_num)

  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  READ(UNIT=unit_num, REC=1) y
  PRINT *, 'Read value:', y

  CLOSE(UNIT=unit_num)
END PROGRAM WriteReadReal

Gestió d'errors

Una gestió d'errors adequada assegura que el teu programa pugui gestionar de manera elegant problemes com fitxers no trobats o errors de lectura/escriptura.

Exemple: Utilitzar IOSTAT per a la gestió d'errors

PROGRAM ErrorHandling
  INTEGER :: i, unit_num, io_status

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4, IOSTAT=io_status)

  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file. IOSTAT =', io_status
    STOP
  END IF

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i, IOSTAT=io_status) value
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading record', i, '. IOSTAT =', io_status
      EXIT
    END IF
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ErrorHandling
