Entendre l'àmbit i la durada de les variables és crucial per escriure programes Fortran eficients i sense errors. Aquest tema et guiarà a través dels conceptes d'àmbit i durada de les variables, des de nivells bàsics fins a avançats.

Àmbit de les Variables

L'àmbit d'una variable es refereix a la regió del programa on la variable és accessible.

Àmbit Local

  • Definició: Les variables declarades dins d'una subrutina, funció o bloc són locals a aquest àmbit.
  • Exemple: 
    program local_scope_example
    implicit none
    call my_subroutine()
contains
    subroutine my_subroutine()
        integer :: local_var
        local_var = 10
        print *, "Valor de la variable local: ", local_var
    end subroutine my_subroutine
end program local_scope_example
    En aquest exemple, local_var només és accessible dins de my_subroutine.

Àmbit Global

  • Definició: Les variables declarades en el programa principal o en un mòdul són accessibles a tot el programa o mòdul.
  • Exemple: 
    module global_scope_example
    implicit none
    integer :: global_var
end module global_scope_example

program use_global_scope
    use global_scope_example
    implicit none
    global_var = 20
    call print_global_var()
contains
    subroutine print_global_var()
        print *, "Valor de la variable global: ", global_var
    end subroutine print_global_var
end program use_global_scope
    Aquí, global_var és accessible tant en el programa principal com en la subrutina print_global_var.

Durada de les Variables

La durada d'una variable es refereix al temps durant el qual la variable reté el seu valor durant l'execució del programa.

Variables Automàtiques

  • Definició: Variables que es creen i es destrueixen automàticament en entrar i sortir d'un àmbit.
  • Exemple: 
    program automatic_variable_example
    implicit none
    call my_subroutine()
    call my_subroutine()
contains
    subroutine my_subroutine()
        integer :: auto_var
        auto_var = auto_var + 1
        print *, "Valor de la variable automàtica: ", auto_var
    end subroutine my_subroutine
end program automatic_variable_example
    Cada vegada que es crida my_subroutine, auto_var es reinicialitza, així que la sortida sempre serà 1.

Variables Estàtiques

  • Definició: Variables que retenen el seu valor entre crides a la subrutina o funció.
  • Exemple: 
    program static_variable_example
    implicit none
    call my_subroutine()
    call my_subroutine()
contains
    subroutine my_subroutine()
        integer, save :: static_var = 0
        static_var = static_var + 1
        print *, "Valor de la variable estàtica: ", static_var
    end subroutine my_subroutine
end program static_variable_example
    Aquí, static_var reté el seu valor entre crides, així que la sortida serà 1 i després 2.

Resum

  • Àmbit: Determina on es pot accedir a una variable.
    • Àmbit local: Les variables són accessibles només dins del bloc on es declaren.
    • Àmbit global: Les variables són accessibles a tot el programa o mòdul.
  • Durada: Determina quant temps una variable reté el seu valor.
    • Variables automàtiques: Es creen i es destrueixen amb l'àmbit.
    • Variables estàtiques: Retenen el seu valor entre crides.

Entendre aquests conceptes ajuda a gestionar la memòria de manera eficient i a evitar errors comuns de programació relacionats amb l'accessibilitat i la persistència de les variables.
