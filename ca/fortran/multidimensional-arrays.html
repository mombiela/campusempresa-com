Els arrays multidimensionals són un concepte fonamental en Fortran, que et permeten emmagatzemar i manipular dades en més d'una dimensió. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat dels arrays multidimensionals en Fortran.

Introducció als Arrays Multidimensionals

Els arrays multidimensionals en Fortran són arrays amb més d'un índex. S'utilitzen per representar dades en forma de graella o matriu, cosa que és comuna en càlculs científics i d'enginyeria.

Conceptes Clau

  • Dimensions: El nombre d'índexs necessaris per accedir a un element en l'array.
  • Rang: El nombre de dimensions que té un array.
  • Forma: La mida de cada dimensió.

Declaració d'Arrays Multidimensionals

Per declarar un array multidimensional en Fortran, especifiques les dimensions entre parèntesis després del nom de l'array.

Exemple: Declaració d'un Array 2D

PROGRAM TwoDArray
  IMPLICIT NONE
  INTEGER, DIMENSION(3, 4) :: matrix
END PROGRAM TwoDArray

En aquest exemple, matrix és un array 2D amb 3 files i 4 columnes.

Exemple: Declaració d'un Array 3D

PROGRAM ThreeDArray
  IMPLICIT NONE
  REAL, DIMENSION(2, 3, 4) :: tensor
END PROGRAM ThreeDArray

Aquí, tensor és un array 3D amb dimensions 2x3x4.

Inicialització d'Arrays Multidimensionals

Pots inicialitzar arrays multidimensionals en el moment de la declaració o més tard en el programa.

Exemple: Inicialització en la Declaració

PROGRAM InitArray
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArray

En aquest exemple, la funció RESHAPE s'utilitza per inicialitzar el matrix amb valors.

Exemple: Inicialització en el Programa

PROGRAM InitArrayInProgram
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArrayInProgram

Accés a Elements en Arrays Multidimensionals

Els elements en un array multidimensional s'accedeixen utilitzant els seus índexs.

Exemple: Accés a Elements

PROGRAM AccessElements
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
  PRINT *, 'Element at (1,1): ', matrix(1, 1)
  PRINT *, 'Element at (2,2): ', matrix(2, 2)
END PROGRAM AccessElements

Aquest programa imprimeix els elements en les posicions (1,1) i (2,2) del matrix.

Operacions en Arrays Multidimensionals

Pots realitzar diverses operacions en arrays multidimensionals, com ara suma, resta i multiplicació.

Exemple: Suma d'Arrays

PROGRAM ArrayAddition
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix1, matrix2, result
  matrix1 = RESHAPE([1, 2, 3, 4], SHAPE(matrix1))
  matrix2 = RESHAPE([5, 6, 7, 8], SHAPE(matrix2))
  result = matrix1 + matrix2
  PRINT *, 'Result of addition:'
  PRINT *, result
END PROGRAM ArrayAddition

Aquest programa suma dos arrays 2D i imprimeix el resultat.

Temes Avançats

Assignació Dinàmica

Fortran permet l'assignació dinàmica d'arrays, cosa que és útil quan la mida de l'array no es coneix en temps de compilació.

Exemple: Assignació Dinàmica

PROGRAM DynamicAllocation
  IMPLICIT NONE
  INTEGER, ALLOCATABLE, DIMENSION(:,:) :: matrix
  INTEGER :: rows, cols
  PRINT *, 'Enter number of rows and columns:'
  READ *, rows, cols
  ALLOCATE(matrix(rows, cols))
  matrix = 0  ! Initialize all elements to zero
  PRINT *, 'Matrix after allocation:'
  PRINT *, matrix
  DEALLOCATE(matrix)
END PROGRAM DynamicAllocation

Aquest programa assigna dinàmicament un array 2D basat en l'entrada de l'usuari.

Seccions d'Arrays

Pots treballar amb seccions d'arrays, cosa que és útil per manipular subarrays.

Exemple: Seccions d'Arrays

PROGRAM ArraySections
  IMPLICIT NONE
  INTEGER, DIMENSION(4, 4) :: matrix
  INTEGER :: i, j
  ! Initialize the matrix
  DO i = 1, 4
    DO j = 1, 4
      matrix(i, j) = i * 10 + j
    END DO
  END DO
  PRINT *, 'Original matrix:'
  PRINT *, matrix
  PRINT *, 'Submatrix (2:3, 2:3):'
  PRINT *, matrix(2:3, 2:3)
END PROGRAM ArraySections

Aquest programa demostra com accedir i imprimir una submatriu.

Conclusió

Els arrays multidimensionals són una característica poderosa en Fortran, que permeten una manipulació i emmagatzematge complex de dades. Des de declaracions i inicialitzacions bàsiques fins a temes avançats com l'assignació dinàmica i les seccions d'arrays, entendre els arrays multidimensionals és crucial per a una programació efectiva en Fortran. Practica amb els exemples i exercicis proporcionats per consolidar la teva comprensió.
