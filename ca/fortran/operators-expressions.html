Introducció

Els operadors i les expressions són components fonamentals en la programació en Fortran. Et permeten realitzar càlculs, prendre decisions i manipular dades. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en Fortran, com utilitzar-los i com construir expressions.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics. Fortran admet els següents operadors aritmètics:

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • ** : Exponenciació

Exemples

PROGRAM ArithmeticExample
  IMPLICIT NONE
  INTEGER :: a, b, c
  REAL :: x, y, z

  ! Aritmètica d'enters
  a = 10
  b = 3
  c = a + b  ! c = 13
  PRINT *, 'a + b = ', c

  c = a - b  ! c = 7
  PRINT *, 'a - b = ', c

  c = a * b  ! c = 30
  PRINT *, 'a * b = ', c

  c = a / b  ! c = 3 (divisió d'enters)
  PRINT *, 'a / b = ', c

  ! Aritmètica de reals
  x = 10.0
  y = 3.0
  z = x / y  ! z = 3.3333333
  PRINT *, 'x / y = ', z

  z = x ** y  ! z = 1000.0
  PRINT *, 'x ** y = ', z
END PROGRAM ArithmeticExample

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar valors. Fortran admet els següents operadors relacionals:

  • == : Igual a
  • /= : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual a
  • <= : Menor o igual a

Exemples

PROGRAM RelationalExample
  IMPLICIT NONE
  INTEGER :: a, b
  LOGICAL :: result

  a = 10
  b = 3

  result = (a == b)  ! result = .FALSE.
  PRINT *, 'a == b: ', result

  result = (a /= b)  ! result = .TRUE.
  PRINT *, 'a /= b: ', result

  result = (a > b)  ! result = .TRUE.
  PRINT *, 'a > b: ', result

  result = (a < b)  ! result = .FALSE.
  PRINT *, 'a < b: ', result

  result = (a >= b)  ! result = .TRUE.
  PRINT *, 'a >= b: ', result

  result = (a <= b)  ! result = .FALSE.
  PRINT *, 'a <= b: ', result
END PROGRAM RelationalExample

Operadors Lògics

Els operadors lògics s'utilitzen per realitzar operacions lògiques. Fortran admet els següents operadors lògics:

  • .AND. : I lògic
  • .OR. : O lògic
  • .NOT. : NO lògic

Exemples

PROGRAM LogicalExample
  IMPLICIT NONE
  LOGICAL :: p, q, result

  p = .TRUE.
  q = .FALSE.

  result = p .AND. q  ! result = .FALSE.
  PRINT *, 'p .AND. q: ', result

  result = p .OR. q  ! result = .TRUE.
  PRINT *, 'p .OR. q: ', result

  result = .NOT. p  ! result = .FALSE.
  PRINT *, '.NOT. p: ', result
END PROGRAM LogicalExample

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. L'operador d'assignació bàsic en Fortran és =.

Exemples

PROGRAM AssignmentExample
  IMPLICIT NONE
  INTEGER :: a
  REAL :: x

  a = 5  ! Assigna 5 a a
  PRINT *, 'a = ', a

  x = 3.14  ! Assigna 3.14 a x
  PRINT *, 'x = ', x
END PROGRAM AssignmentExample

Operadors Combinats

Fortran també admet operadors combinats per a operacions aritmètiques i assignació. Aquests inclouen:

  • += : Assignació de suma
  • -= : Assignació de resta
  • *= : Assignació de multiplicació
  • /= : Assignació de divisió

Exemples

PROGRAM CombinedOperatorsExample
  IMPLICIT NONE
  INTEGER :: a

  a = 10
  a += 5  ! Equivalent a a = a + 5
  PRINT *, 'a += 5: ', a  ! a = 15

  a -= 3  ! Equivalent a a = a - 3
  PRINT *, 'a -= 3: ', a  ! a = 12

  a *= 2  ! Equivalent a a = a * 2
  PRINT *, 'a *= 2: ', a  ! a = 24

  a /= 4  ! Equivalent a a = a / 4
  PRINT *, 'a /= 4: ', a  ! a = 6
END PROGRAM CombinedOperatorsExample

Precedència d'Operadors

La precedència d'operadors determina l'ordre en què es realitzen les operacions en una expressió. La següent taula resumeix la precedència dels operadors en Fortran, de més alta a més baixa:

| Operador | Descripció | |----------------|-----------------------| | ** | Exponenciació | | *, / | Multiplicació, Divisió | | +, - | Suma, Resta | | .NOT. | NO lògic | | .AND. | I lògic | | .OR. | O lògic | | ==, /=, >, <, >=, <= | Operadors Relacionals |

Exemples

PROGRAM PrecedenceExample
  IMPLICIT NONE
  INTEGER :: a, b, c, result

  a = 2
  b = 3
  c = 4

  result = a + b * c  ! Multiplicació abans de la suma
  PRINT *, 'a + b * c = ', result  ! result = 14

  result = (a + b) * c  ! Els parèntesis canvien l'ordre
  PRINT *, '(a + b) * c = ', result  ! result = 20

  result = a ** b + c  ! Exponenciació abans de la suma
  PRINT *, 'a ** b + c = ', result  ! result = 12
END PROGRAM PrecedenceExample

Conclusió

Entendre els operadors i les expressions és crucial per escriure programes efectius en Fortran. Aquesta secció ha cobert operadors aritmètics, relacionals, lògics, d'assignació i combinats, així com la precedència d'operadors. En dominar aquests conceptes, podràs realitzar càlculs complexos i prendre decisions informades en els teus programes Fortran. Practica utilitzant aquests operadors en diferents contextos per sentir-te més còmode amb el seu ús i comportament.
