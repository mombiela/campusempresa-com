Introducció

En aquesta secció, revisarem els conceptes i tècniques clau coberts al llarg del curs. Això ajudarà a reforçar la teva comprensió i assegurar que estàs ben preparat per aplicar Fortran en escenaris del món real. També presentarem alguns temes avançats per donar-te una ullada al que pots explorar més endavant.

Revisió de Conceptes Clau

Sintaxi i Estructura Bàsica

  • Estructura del Programa: Entendre l'estructura bàsica d'un programa Fortran. 
    program HelloWorld
    print *, "Hello, World!"
end program HelloWorld
  • Variables i Tipus de Dades: Declarar i utilitzar variables de diferents tipus de dades. 
    integer :: i
real :: x
character(len=20) :: name

Estructures de Control

  • Declaracions Condicionals: Utilitzar if, else if i else per a la presa de decisions. 
    if (x > 0) then
    print *, "x is positive"
else if (x == 0) then
    print *, "x is zero"
else
    print *, "x is negative"
end if
  • Bucles: Implementar bucles do per a la iteració. 
    do i = 1, 10
    print *, "i =", i
end do

Arrays i Matrius

  • Declaració i Inicialització d'Arrays: Crear i inicialitzar arrays. 
    integer, dimension(5) :: arr = (/1, 2, 3, 4, 5/)
  • Operacions amb Arrays: Realitzar operacions amb arrays. 
    arr = arr * 2

Funcions i Subrutines

  • Funcions: Definir i utilitzar funcions. 
    function add(a, b)
    integer :: add
    integer, intent(in) :: a, b
    add = a + b
end function add
  • Subrutines: Crear i cridar subrutines. 
    subroutine print_message(msg)
    character(len=*) :: msg
    print *, msg
end subroutine print_message

Visió General de Temes Avançats

Mòduls

  • Crear Mòduls: Organitzar el codi en mòduls reutilitzables. 
    module math_operations
    contains
    function add(a, b)
        integer :: add
        integer, intent(in) :: a, b
        add = a + b
    end function add
end module math_operations
  • Utilitzar Mòduls: Importar i utilitzar mòduls en programes. 
    program use_module
    use math_operations
    print *, add(3, 4)
end program use_module

Entrada/Sortida de Fitxers

  • Llegir de Fitxers: Llegir dades de fitxers externs. 
    open(unit=10, file='data.txt', status='old', action='read')
read(10, *) x
close(10)
  • Escriure a Fitxers: Escriure dades a fitxers externs. 
    open(unit=20, file='output.txt', status='new', action='write')
write(20, *) x
close(20)

Programació Paral·lela

  • Introducció als Coarrays: Utilitzar coarrays per a la programació paral·lela. 
    real, codimension[*] :: x
x = this_image()

Resum

En aquesta secció de 'Presentació i Revisió', hem revisitat els conceptes fonamentals de Fortran, incloent la sintaxi bàsica, estructures de control, arrays, funcions i subrutines. També hem introduït temes avançats com mòduls, entrada/sortida de fitxers i programació paral·lela amb coarrays. Aquesta revisió completa hauria de solidificar la teva comprensió i preparar-te per a tasques de programació Fortran més complexes. Continua practicant i explorant aquests conceptes per esdevenir proficient en Fortran.
