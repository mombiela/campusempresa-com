Introducció

L'estil de codi i la documentació són aspectes crucials de la programació que asseguren la llegibilitat, mantenibilitat i col·laboració del codi. En aquesta secció, cobrirem les millors pràctiques per escriure codi Fortran net i ben documentat, adequat tant per a principiants com per a programadors avançats.

Estil de Codi

Indentació i Espaiat

  • Indentació Consistent: Utilitza una indentació consistent al llarg del teu codi. La pràctica comuna és utilitzar 2 o 4 espais per nivell d'indentació.

  • Espaiat Al Voltant dels Operadors: Afegeix espais al voltant dels operadors per millorar la llegibilitat.

    ! Bon exemple
a = b + c * d

! Mal exemple
a=b+c*d

Longitud de la Línia

  • Límit de Longitud de la Línia: Mantingues les línies dins de 72 caràcters per a la font de forma fixa i 132 caràcters per a la font de forma lliure per assegurar la llegibilitat i compatibilitat amb diversos editors i eines.

Convencions de Nomenclatura

  • Variables i Funcions: Utilitza noms significatius per a les variables i funcions. Utilitza lletres minúscules amb guions baixos per als noms de les variables i camelCase per als noms de les funcions.

    ! Bon exemple
integer :: total_sum
real :: averageValue

! Mal exemple
integer :: ts
real :: avgval

Comentaris

  • Comentaris en Línia: Utilitza comentaris en línia per explicar la lògica complexa o passos importants en el codi.

    ! Calcula el valor mitjà
averageValue = total_sum / number_of_elements

  • Comentaris en Bloc: Utilitza comentaris en bloc per descriure la finalitat d'una secció de codi o una funció.

    !--------------------------------------------------
! Aquesta funció calcula el factorial d'un nombre
! Entrada: enter n
! Sortida: enter factorial
!--------------------------------------------------

Documentació

Capçaleres de Fitxer

  • Comentaris de Capçalera de Fitxer: Inclou un comentari de capçalera al començament de cada fitxer per descriure la finalitat del fitxer, l'autor, la data i qualsevol nota rellevant.

    !--------------------------------------------------
! Fitxer: factorial.f90
! Finalitat: Calcula el factorial d'un nombre
! Autor: Jane Doe
! Data: 2023-10-01
!--------------------------------------------------

Documentació de Funcions i Subrutines

  • Capçaleres de Funcions: Documenta cada funció i subrutina amb un comentari de capçalera que descrigui la seva finalitat, entrades, sortides i qualsevol efecte secundari.

    !--------------------------------------------------
! Funció: factorial
! Finalitat: Calcula el factorial d'un nombre
! Entrada: enter n
! Sortida: enter factorial
!--------------------------------------------------
function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  ! Lògica de la funció aquí
end function factorial

Documentació en Línia

  • Documentació en Línia: Utilitza comentaris dins de la funció o subrutina per explicar la lògica complexa o passos importants.

    function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  integer :: i

  factorial = 1
  do i = 1, n
    factorial = factorial * i
    ! Multiplica factorial pel valor actual de i
  end do
end function factorial

Exemple: Codi Fortran Ben Documentat

!--------------------------------------------------
! Fitxer: example.f90
! Finalitat: Demostrar codi Fortran ben documentat
! Autor: John Smith
! Data: 2023-10-01
!--------------------------------------------------

program example
  implicit none
  integer :: num, result

  ! Demana a l'usuari una entrada
  print *, 'Introdueix un nombre:'
  read *, num

  ! Calcula el factorial
  result = factorial(num)

  ! Mostra el resultat
  print *, 'El factorial de', num, 'és', result
end program example

!--------------------------------------------------
! Funció: factorial
! Finalitat: Calcula el factorial d'un nombre
! Entrada: enter n
  ! Sortida: enter factorial
  !--------------------------------------------------
function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  integer :: i

  factorial = 1
  do i = 1, n
    factorial = factorial * i
    ! Multiplica factorial pel valor actual de i
  end do
end function factorial

Conclusió

Adherir-se a bones pràctiques d'estil de codi i documentació en Fortran és essencial per crear codi llegible, mantenible i col·laboratiu. Seguint una indentació consistent, convencions de nomenclatura i una documentació exhaustiva, pots assegurar que el teu codi sigui fàcil d'entendre i mantenir, tant per a tu com per a altres que puguin treballar amb el teu codi en el futur.
