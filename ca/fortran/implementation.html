Introducció a la Implementació en Fortran

En aquesta secció, tractarem com implementar programes en Fortran, des d'escriure scripts simples fins a desenvolupar aplicacions més complexes. Començarem amb conceptes bàsics i gradualment passarem a temes avançats, assegurant-nos que entenguis cada pas al llarg del camí.

Escriure el teu Primer Programa en Fortran

Comencem amb un simple programa "Hola, Món!" per familiaritzar-te amb l'estructura bàsica d'un programa en Fortran.

program hello
    print *, 'Hola, Món!'
end program hello
  • program: Aquesta paraula clau inicia el programa.
  • print: Aquesta instrucció mostra text a la pantalla.
  • end program: Aquesta paraula clau finalitza el programa.

Exercici: Modificar el Programa

Prova de modificar el programa per imprimir el teu nom en lloc de "Hola, Món!".

program hello
    print *, 'Hola, [El Teu Nom]!'
end program hello

Variables i Tipus de Dades

Fortran admet diversos tipus de dades, incloent-hi enters, nombres reals i caràcters. Vegem com declarar i utilitzar aquestes variables.

program variables
    integer :: i
    real :: x
    character(len=20) :: name

    i = 10
    x = 3.14
    name = 'Fortran'

    print *, 'Enter:', i
    print *, 'Real:', x
    print *, 'Caràcter:', name
end program variables
  • integer: Declara una variable enter.
  • real: Declara una variable de punt flotant.
  • character: Declara una variable de cadena amb una longitud especificada.

Exercici: Declarar i Imprimir Variables

Declara variables de diferents tipus i imprimeix els seus valors.

program my_variables
    integer :: age
    real :: height
    character(len=30) :: hobby

    age = 25
    height = 5.9
    hobby = 'Programació'

    print *, 'Edat:', age
    print *, 'Alçada:', height
    print *, 'Afició:', hobby
end program my_variables

Estructures de Control

Les estructures de control com els bucles i les condicionals són essencials per implementar la lògica en els teus programes.

Instruccions If

program if_example
    integer :: num

    num = 10

    if (num > 5) then
        print *, 'El nombre és més gran que 5'
    else
        print *, 'El nombre és 5 o menys'
    end if
end program if_example

Bucles Do

program loop_example
    integer :: i

    do i = 1, 5
        print *, 'Iteració:', i
    end do
end program loop_example

Exercici: Implementar Estructures de Control

Escriu un programa que utilitzi tant instruccions if com bucles do.

program control_structures
    integer :: i

    do i = 1, 10
        if (mod(i, 2) == 0) then
            print *, i, 'és parell'
        else
            print *, i, 'és senar'
        end if
    end do
end program control_structures

Funcions i Subrutines

Les funcions i subrutines permeten modularitzar el teu codi, fent-lo més llegible i reutilitzable.

Funcions

program function_example
    real :: result

    result = square(4.0)
    print *, 'El quadrat de 4.0 és', result

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square
end program function_example

Subrutines

program subroutine_example
    real :: a, b, result

    a = 3.0
    b = 4.0
    call add(a, b, result)
    print *, 'La suma de', a, 'i', b, 'és', result

contains
    subroutine add(x, y, sum)
        real, intent(in) :: x, y
        real, intent(out) :: sum
        sum = x + y
    end subroutine add
end program subroutine_example

Exercici: Crear Funcions i Subrutines

Crea un programa que inclogui tant una funció com una subrutina.

program func_sub_example
    real :: num, sq, sum

    num = 5.0
    sq = square(num)
    call add(num, sq, sum)

    print *, 'El quadrat de', num, 'és', sq
    print *, 'La suma de', num, 'i el seu quadrat és', sum

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square

    subroutine add(x, y, result)
        real, intent(in) :: x, y
        real, intent(out) :: result
        result = x + y
    end subroutine add
end program func_sub_example

Temes Avançats

Mòduls

Els mòduls s'utilitzen per agrupar procediments i dades relacionades.

module math_operations
    contains
        real function square(x)
            real, intent(in) :: x
            square = x * x
        end function square

        subroutine add(x, y, sum)
            real, intent(in) :: x, y
            real, intent(out) :: sum
            sum = x + y
        end subroutine add
end module math_operations

program use_module
    use math_operations
    real :: a, b, result

    a = 2.0
    b = 3.0
    result = square(a)
    print *, 'El quadrat de', a, 'és', result

    call add(a, b, result)
    print *, 'La suma de', a, 'i', b, 'és', result
end program use_module

Exercici: Crear i Utilitzar Mòduls

Crea un mòdul amb almenys dues funcions i utilitza'l en un programa.

module my_module
    contains
        real function cube(x)
            real, intent(in) :: x
            cube = x * x * x
        end function cube

        subroutine multiply(x, y, product)
            real, intent(in) :: x, y
            real, intent(out) :: product
            product = x * y
        end subroutine multiply
end module my_module

program use_my_module
    use my_module
    real :: num, cubed, product

    num = 3.0
    cubed = cube(num)
    call multiply(num, cubed, product)

    print *, 'El cub de', num, 'és', cubed
    print *, 'El producte de', num, 'i el seu cub és', product
end program use_my_module
