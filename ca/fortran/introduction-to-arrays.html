Visió General

Els arrays són una estructura de dades fonamental en Fortran, que et permeten emmagatzemar i manipular col·leccions de dades de manera eficient. Aquesta secció t'introduirà als conceptes bàsics dels arrays, incloent la seva declaració, inicialització i operacions comunes.

Què és un Array?

  • Un array és una col·lecció d'elements, tots del mateix tipus, emmagatzemats en ubicacions de memòria contigües.
  • Els arrays poden ser unidimensionals (com una llista) o multidimensionals (com una matriu).

Declaració d'Arrays

Per declarar un array en Fortran, especifiques el tipus d'elements i les dimensions de l'array.

Arrays Unidimensionals

INTEGER :: array1D(10)
REAL :: array1D_real(5)
  • INTEGER :: array1D(10) declara un array d'enters amb 10 elements.
  • REAL :: array1D_real(5) declara un array de nombres reals amb 5 elements.

Arrays Multidimensionals

INTEGER :: array2D(3, 3)
REAL :: array3D(2, 2, 2)
  • INTEGER :: array2D(3, 3) declara un array d'enters de 3x3.
  • REAL :: array3D(2, 2, 2) declara un array de nombres reals de 2x2x2.

Inicialització d'Arrays

Els arrays es poden inicialitzar en el moment de la declaració o més tard en el codi.

Inicialització en la Declaració

INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/)
REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))
  • INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/) inicialitza un array unidimensional amb els valors de l'1 al 5.
  • REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/)) inicialitza un array de 2x2 amb els valors 1.0, 2.0, 3.0 i 4.0.

Inicialització Més Tard en el Codi

INTEGER :: array1D(5)
array1D = (/1, 2, 3, 4, 5/)

REAL :: array2D(2, 2)
array2D = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))

Accés als Elements de l'Array

Pots accedir i modificar els elements de l'array utilitzant els seus índexs.

Accés als Elements

INTEGER :: value
value = array1D(3)  ! Accedeix al tercer element de array1D

Modificació dels Elements

array1D(3) = 10  ! Assigna el valor 10 al tercer element de array1D

Operacions Comunes amb Arrays

  • Recorregut de l'Array: Recórrer els elements de l'array.
  • Tall d'Array: Accedir a subarrays.
  • Funcions Intrínseques d'Array: Funcions integrades per a la manipulació d'arrays.

Recorregut de l'Array

INTEGER :: i
DO i = 1, 5
    PRINT *, array1D(i)
END DO

Tall d'Array

INTEGER, DIMENSION(5) :: subArray
subArray = array1D(2:4)  ! Extreu els elements del 2 al 4

Funcions Intrínseques d'Array

REAL :: sum
sum = SUM(array1D_real)  ! Calcula la suma dels elements de array1D_real

Conclusió

Els arrays són una eina poderosa en Fortran per gestionar col·leccions de dades. Entendre com declarar, inicialitzar i manipular arrays és essencial per a una programació eficient. Practica aquests conceptes amb els exemples i exercicis proporcionats per consolidar la teva comprensió.

Exercicis

  1. Declara un array unidimensional d'enters amb 7 elements i inicialitza'l amb valors de l'1 al 7.
  2. Crea un array real de 3x3 i inicialitza'l amb valors de l'1.0 al 9.0.
  3. Escriu un bucle per imprimir tots els elements de l'array de 3x3.
  4. Utilitza el tall d'array per extreure un subarray de l'array unidimensional creat en l'exercici 1.
  5. Calcula la suma dels elements de l'array de 3x3 utilitzant una funció intrínseca.
