El posicionament de fitxers és un aspecte essencial de la gestió de fitxers en Fortran. Permet als programadors controlar des d'on es llegeixen o s'escriuen les dades en un fitxer. Aquest tema cobrirà els conceptes bàsics del posicionament de fitxers, incloent com moure el punter del fitxer, i proporcionarà exemples i exercicis per consolidar la teva comprensió.

Conceptes Clau

  • Punter de Fitxer: Un marcador que indica la posició actual en un fitxer on es produirà la següent operació de lectura o escriptura.
  • Accés Seqüencial: Llegir o escriure dades en una seqüència, des del començament fins al final del fitxer.
  • Accés Directe: Llegir o escriure dades en posicions específiques del fitxer, permetent un accés no seqüencial.

Accés Seqüencial

En l'accés seqüencial, el punter del fitxer es mou automàticament a mesura que es llegeixen o s'escriuen les dades. Aquesta és la forma més senzilla de posicionament de fitxers.

Exemple: Lectura Seqüencial

PROGRAM SequentialRead
  IMPLICIT NONE
  INTEGER :: i, io_status
  OPEN(UNIT=10, FILE='data.txt', STATUS='OLD', ACTION='READ')

  DO
    READ(10, *, IOSTAT=io_status) i
    IF (io_status /= 0) EXIT
    PRINT *, 'Read value:', i
  END DO

  CLOSE(10)
END PROGRAM SequentialRead

En aquest exemple:

  • El fitxer data.txt s'obre per a lectura.
  • Un bucle llegeix enters del fitxer fins que s'arriba al final.
  • El punter del fitxer es mou automàticament al següent enter després de cada operació de lectura.

Accés Directe

L'accés directe permet llegir o escriure dades en posicions específiques del fitxer. Això és útil per a l'accés aleatori a les dades.

Exemple: Escriptura i Lectura d'Accés Directe

PROGRAM DirectAccess
  IMPLICIT NONE
  INTEGER :: i, rec_num
  INTEGER, DIMENSION(5) :: data = (/1, 2, 3, 4, 5/)
  INTEGER :: io_status

  ! Obrir fitxer per a accés directe
  OPEN(UNIT=20, FILE='direct_data.dat', STATUS='NEW', ACCESS='DIRECT', RECL=4)

  ! Escriure dades en registres específics
  DO rec_num = 1, 5
    WRITE(20, REC=rec_num, IOSTAT=io_status) data(rec_num)
    IF (io_status /= 0) THEN
      PRINT *, 'Error writing to record', rec_num
      STOP
    END IF
  END DO

  ! Llegir dades de registres específics
  DO rec_num = 1, 5
    READ(20, REC=rec_num, IOSTAT=io_status) i
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading from record', rec_num
      STOP
    END IF
    PRINT *, 'Read value from record', rec_num, ':', i
  END DO

  CLOSE(20)
END PROGRAM DirectAccess

En aquest exemple:

  • El fitxer direct_data.dat s'obre per a accés directe amb una longitud de registre de 4 bytes.
  • Les dades s'escriuen en registres específics del fitxer.
  • Les dades es llegeixen després d'aquests registres, demostrant l'accés aleatori.

Resum

El posicionament de fitxers en Fortran és una habilitat crucial per gestionar operacions d'entrada/sortida de fitxers de manera efectiva. Entendre tant els mètodes d'accés seqüencial com directe permet una manipulació flexible i eficient de les dades. L'accés seqüencial és senzill i adequat per a operacions de lectura/escriptura simples, mentre que l'accés directe proporciona la capacitat de manipular dades en posicions específiques, cosa que és útil per a aplicacions més complexes.

En dominar aquestes tècniques, pots gestionar una àmplia gamma d'escenaris d'entrada/sortida de fitxers en Fortran, millorant la teva capacitat per desenvolupar programes robustos i eficients.
