Introducció a les llistes enllaçades

Les llistes enllaçades són una estructura de dades fonamental utilitzada en informàtica per emmagatzemar col·leccions d'elements. A diferència dels arrays, les llistes enllaçades consisteixen en nodes on cada node conté dades i una referència (o enllaç) al següent node en la seqüència. Aquesta estructura permet una inserció i eliminació eficient d'elements.

Conceptes clau

  • Node: La unitat bàsica d'una llista enllaçada, que conté dades i una referència al següent node.
  • Cap: El primer node d'una llista enllaçada.
  • Cua: L'últim node d'una llista enllaçada, que apunta a null o nil.
  • Punter: Una variable que conté l'adreça de memòria d'una altra variable, utilitzada per enllaçar nodes en una llista enllaçada.

Creació d'una llista enllaçada simple

Per crear una llista enllaçada en Fortran, necessitem definir una estructura de node i implementar operacions bàsiques com la inserció i el recorregut.

Definició d'un node

En Fortran, podem definir un node utilitzant una construcció type.

type :: Node
    integer :: data
    type(Node), pointer :: next => null()
end type Node

Inicialització de la llista enllaçada

Comencem creant el cap de la llista enllaçada.

program LinkedListExample
    implicit none
    type(Node), pointer :: head, newNode

    ! Inicialitzar el node cap
    allocate(head)
    head%data = 10
    head%next => null()

    ! Imprimir les dades del node cap
    print *, "Dades del node cap: ", head%data
end program LinkedListExample

Inserció de nodes

Podem inserir nous nodes al començament, al final o en qualsevol posició de la llista enllaçada.

Inserció al començament

Per inserir un node al començament, necessitem actualitzar el punter cap per apuntar al nou node.

subroutine insertAtBeginning(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode

    allocate(newNode)
    newNode%data = value
    newNode%next => head
    head => newNode
end subroutine insertAtBeginning

Inserció al final

Per inserir un node al final, recorrem la llista per trobar l'últim node i actualitzem el seu punter next.

subroutine insertAtEnd(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode, temp

    allocate(newNode)
    newNode%data = value
    newNode%next => null()

    if (.not. associated(head)) then
        head => newNode
    else
        temp => head
        do while (associated(temp%next))
            temp => temp%next
        end do
        temp%next => newNode
    end if
end subroutine insertAtEnd

Recorregut de la llista enllaçada

El recorregut implica visitar cada node de la llista enllaçada i realitzar alguna operació, com ara imprimir les dades.

subroutine traverse(head)
    type(Node), pointer :: head
    type(Node), pointer :: temp

    temp => head
    do while (associated(temp))
        print *, "Dades del node: ", temp%data
        temp => temp%next
    end do
end subroutine traverse

Operacions avançades

Les operacions avançades en llistes enllaçades inclouen l'eliminació, la cerca i la inversió de la llista.

Eliminació d'un node

Per eliminar un node, necessitem actualitzar els punters per passar per alt el node que s'ha d'eliminar.

subroutine deleteNode(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: temp, prev

    temp => head
    prev => null()

    do while (associated(temp))
        if (temp%data == value) then
            if (associated(prev)) then
                prev%next => temp%next
            else
                head => temp%next
            end if
            deallocate(temp)
            return
        end if
        prev => temp
        temp => temp%next
    end do
end subroutine deleteNode

Cerca d'un node

Per cercar un node, recorrem la llista i comprovem les dades de cada node.

function searchNode(head, value) result(found)
    type(Node), pointer :: head
    integer, intent(in) :: value
    logical :: found
    type(Node), pointer :: temp

    found = .false.
    temp => head
    do while (associated(temp))
        if (temp%data == value) then
            found = .true.
            return
        end if
        temp => temp%next
    end do
end function searchNode

Inversió de la llista enllaçada

Invertir una llista enllaçada implica canviar la direcció dels punters.

subroutine reverseList(head)
    type(Node), pointer :: head
    type(Node), pointer :: prev, current, next

    prev => null()
    current => head

    do while (associated(current))
        next => current%next
        current%next => prev
        prev => current
        current => next
    end do
    head => prev
end subroutine reverseList

Conclusió

Les llistes enllaçades són una estructura de dades versàtil i fonamental en la programació. En Fortran, les llistes enllaçades es poden implementar utilitzant punters i tipus personalitzats. Entendre com crear, manipular i recórrer llistes enllaçades és essencial per resoldre diversos problemes computacionals de manera eficient. En dominar aquests conceptes, podràs gestionar estructures de dades i algorismes més complexos en el teu viatge de programació.
