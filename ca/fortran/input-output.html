Introducció

Les operacions d'entrada i sortida (I/O) són fonamentals en qualsevol llenguatge de programació, incloent Fortran. Permeten que els programes interactuïn amb els usuaris i altres sistemes llegint dades de i escrivint dades a diverses fonts com la consola, fitxers, i més.

Entrada i Sortida Bàsiques

Llegir des de la Consola

Fortran utilitza la instrucció READ per prendre entrada de l'usuari. La forma més simple llegeix des de l'entrada estàndard (normalment el teclat).

PROGRAM ReadExample
  IMPLICIT NONE
  INTEGER :: num

  PRINT *, 'Enter a number:'
  READ *, num
  PRINT *, 'You entered:', num
END PROGRAM ReadExample
  • PRINT * s'utilitza per mostrar un missatge a la consola.
  • READ * llegeix l'entrada de la consola i l'emmagatzema a la variable num.

Escriure a la Consola

La instrucció PRINT s'utilitza per sortir dades a la consola.

PROGRAM PrintExample
  IMPLICIT NONE
  INTEGER :: num

  num = 10
  PRINT *, 'The number is:', num
END PROGRAM PrintExample
  • PRINT *, va seguit del missatge i les variables a imprimir.

Entrada i Sortida Avançades

Entrada i Sortida Formatades

Per tenir més control sobre el format de l'entrada i la sortida, Fortran proporciona I/O formatada utilitzant les instruccions READ i WRITE amb especificadors de format.

PROGRAM FormattedIO
  IMPLICIT NONE
  INTEGER :: num
  REAL :: value

  PRINT *, 'Enter an integer and a real number:'
  READ '(I5, F10.2)', num, value
  PRINT '(A, I5, A, F10.2)', 'Integer:', num, ' Real:', value
END PROGRAM FormattedIO
  • READ '(I5, F10.2)' especifica el format per llegir: un enter amb fins a 5 dígits i un número real amb fins a 10 dígits i 2 decimals.
  • PRINT '(A, I5, A, F10.2)' especifica el format per imprimir: una cadena, un enter amb fins a 5 dígits, una altra cadena, i un número real amb fins a 10 dígits i 2 decimals.

Entrada i Sortida de Fitxers

Fortran permet llegir de i escriure a fitxers utilitzant les instruccions OPEN, READ, WRITE, i CLOSE.

PROGRAM FileIO
  IMPLICIT NONE
  INTEGER :: num
  CHARACTER(LEN=100) :: line

  ! Obre un fitxer per escriure
  OPEN(UNIT=10, FILE='output.txt', STATUS='REPLACE')
  WRITE(10, *) 'This is a line of text.'
  WRITE(10, *) 12345
  CLOSE(10)

  ! Obre un fitxer per llegir
  OPEN(UNIT=10, FILE='output.txt', STATUS='OLD')
  READ(10, '(A)') line
  PRINT *, 'First line:', line
  READ(10, *) num
  PRINT *, 'Second line (number):', num
  CLOSE(10)
END PROGRAM FileIO
  • OPEN obre un fitxer amb un número d'unitat especificat.
  • WRITE escriu dades al fitxer associat amb el número d'unitat.
  • READ llegeix dades del fitxer associat amb el número d'unitat.
  • CLOSE tanca el fitxer associat amb el número d'unitat.

Gestió d'Errors en I/O

Fortran proporciona mecanismes per gestionar errors durant les operacions d'I/O utilitzant els especificadors IOSTAT i ERR.

PROGRAM IOErrorHandling
  IMPLICIT NONE
  INTEGER :: num, io_status

  OPEN(UNIT=10, FILE='nonexistent.txt', STATUS='OLD', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file. IOSTAT =', io_status
  ELSE
    READ(10, *, IOSTAT=io_status) num
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading file. IOSTAT =', io_status
    ELSE
      PRINT *, 'Number read from file:', num
    END IF
    CLOSE(10)
  END IF
END PROGRAM IOErrorHandling
  • IOSTAT captura l'estat de l'operació d'I/O.
  • ERR es pot utilitzar per especificar una etiqueta a la qual saltar en cas d'error.

Resum

En aquesta secció, hem cobert els aspectes bàsics i avançats de l'entrada i sortida en Fortran. Hem après com:

  • Llegir de i escriure a la consola.
  • Utilitzar I/O formatada per tenir un millor control sobre la representació de les dades.
  • Realitzar operacions d'I/O amb fitxers.
  • Gestionar errors durant les operacions d'I/O.

Entendre aquests conceptes és crucial per desenvolupar programes robustos en Fortran que puguin interactuar amb usuaris i altres sistemes de manera efectiva.
