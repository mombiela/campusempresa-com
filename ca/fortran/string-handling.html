Operacions amb Arrays i Cadenes ►

La gestió de cadenes és un aspecte crucial de la programació en Fortran, especialment quan es tracta de dades de text. Aquest tema cobrirà els conceptes bàsics de la gestió de cadenes, tècniques de manipulació i operacions avançades de cadenes en Fortran.

Introducció a les Cadenes en Fortran

En Fortran, les cadenes són seqüències de caràcters. S'utilitzen típicament per emmagatzemar i manipular dades de text.

Declaració de Cadenes

  • Les cadenes en Fortran es declaren utilitzant la paraula clau CHARACTER.
  • Es pot especificar la longitud de la cadena.
CHARACTER(LEN=20) :: str
str = 'Hello, Fortran!'

Operacions Bàsiques amb Cadenes

  • Assignació: Assignar un valor a una variable de cadena.
  • Concatenació: Combinar dues o més cadenes.
  • Subcadena: Extreure una part d'una cadena.
CHARACTER(LEN=20) :: str1, str2, result
str1 = 'Hello'
str2 = 'World'
result = str1 // ' ' // str2  ! Concatenació
PRINT *, result  ! Sortida: Hello World

result = str1(1:4)  ! Subcadena
PRINT *, result  ! Sortida: Hell

Tècniques de Manipulació de Cadenes

Longitud de la Cadena

  • La funció LEN retorna la longitud d'una cadena.
CHARACTER(LEN=20) :: str
str = 'Fortran'
PRINT *, LEN(str)  ! Sortida: 20 (incloent espais finals)
PRINT *, LEN_TRIM(str)  ! Sortida: 7 (excloent espais finals)

Retallar Cadenes

  • La funció TRIM elimina els espais finals d'una cadena.
CHARACTER(LEN=20) :: str
str = 'Fortran    '
PRINT *, TRIM(str)  ! Sortida: 'Fortran'

Comparació de Cadenes

  • Les cadenes es poden comparar utilitzant operadors relacionals (==, /=, <, >, <=, >=).
CHARACTER(LEN=10) :: str1, str2
str1 = 'abc'
str2 = 'def'
IF (str1 < str2) THEN
    PRINT *, 'str1 és menor que str2'
ENDIF

Operacions Avançades amb Cadenes

Funcions Intrínseques de Cadenes

Fortran proporciona diverses funcions intrínseques per a operacions avançades amb cadenes.

  • INDEX: Troba la posició d'una subcadena dins d'una cadena.
  • ADJUSTL: Justifica una cadena a l'esquerra.
  • ADJUSTR: Justifica una cadena a la dreta.
CHARACTER(LEN=20) :: str
str = 'Hello, Fortran!'
PRINT *, INDEX(str, 'Fortran')  ! Sortida: 8

str = '   Fortran'
PRINT *, ADJUSTL(str)  ! Sortida: 'Fortran   '
PRINT *, ADJUSTR(str)  ! Sortida: '   Fortran'

Assignació Dinàmica de Cadenes

  • Fortran 2003 va introduir la capacitat d'assignar cadenes dinàmicament.
CHARACTER(:), ALLOCATABLE :: dyn_str
ALLOCATE(CHARACTER(LEN=30) :: dyn_str)
dyn_str = 'Assignació Dinàmica de Cadenes'
PRINT *, dyn_str
DEALLOCATE(dyn_str)

Exercicis

  1. Operacions Bàsiques amb Cadenes: Declara dues cadenes, concatena-les i imprimeix el resultat.
  2. Extracció de Subcadena: Extreu i imprimeix una subcadena d'una cadena donada.
  3. Comparació de Cadenes: Escriu un programa per comparar dues cadenes i imprimeix quina és més gran.
  4. Funcions Avançades: Utilitza la funció INDEX per trobar la posició d'una subcadena dins d'una cadena.

Conclusió

La gestió de cadenes en Fortran implica diverses operacions com l'assignació, la concatenació i l'extracció de subcadenes. Entendre aquests conceptes bàsics, juntament amb tècniques avançades com l'assignació dinàmica i les funcions intrínseques, és essencial per a una manipulació efectiva de dades de text en Fortran. Practica aquests conceptes a través dels exercicis proporcionats per consolidar la teva comprensió.
