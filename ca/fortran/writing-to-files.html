Escriure a Fitxers en Fortran
Posicionament de Fitxers ►

Escriure a fitxers és una habilitat fonamental en la programació en Fortran, que et permet guardar dades per a un ús posterior, compartir resultats o registrar informació. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades d'escriptura a fitxers en Fortran.

Escriptura Bàsica a Fitxers

Obrir un Fitxer

Per escriure a un fitxer, primer has d'obrir-lo utilitzant la instrucció OPEN. Aquesta instrucció associa un número d'unitat amb un fitxer.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
END PROGRAM WriteToFile
  • UNIT: Un identificador únic per al fitxer.
  • FILE: El nom del fitxer.
  • STATUS: Especifica l'estat del fitxer (NEW, OLD, REPLACE, etc.).

Escriure Dades

Utilitza la instrucció WRITE per escriure dades al fitxer.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteToFile
  • WRITE(unit_number, *): Escriu al fitxer associat amb unit_number.
  • CLOSE(unit_number): Tanca el fitxer.

Exemple: Escriure Múltiples Línies

PROGRAM WriteMultipleLines
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Line 1: Hello, Fortran!'
  WRITE(unit_number, *) 'Line 2: Writing to files is easy.'
  CLOSE(unit_number)
END PROGRAM WriteMultipleLines

Escriptura Intermèdia a Fitxers

Formatant la Sortida

Pots formatar la sortida utilitzant especificadors de format.

PROGRAM WriteFormatted
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') 'Value: ', 42
  CLOSE(unit_number)
END PROGRAM WriteFormatted
  • '(A, I3)': Especificador de format on A és per a cadena i I3 és per a un enter amb una amplada de 3.

Escriure Arrays

També pots escriure arrays a fitxers.

PROGRAM WriteArray
  INTEGER :: unit_number
  INTEGER, DIMENSION(5) :: array = (/1, 2, 3, 4, 5/)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(5I3)') array
  CLOSE(unit_number)
END PROGRAM WriteArray
  • '(5I3)': Escriu 5 enters, cadascun amb una amplada de 3.

Escriptura Avançada a Fitxers

Gestionant Errors

Utilitza el paràmetre IOSTAT per gestionar errors durant les operacions de fitxer.

PROGRAM WriteWithErrorHandling
  INTEGER :: unit_number, io_status
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file!'
    STOP
  END IF
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteWithErrorHandling
  • IOSTAT=io_status: Emmagatzema l'estat de l'operació d'E/S.

Afegint a Fitxers

Per afegir dades a un fitxer existent, utilitza les opcions STATUS='OLD' i POSITION='APPEND'.

PROGRAM AppendToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='OLD', POSITION='APPEND')
  WRITE(unit_number, *) 'Appending this line.'
  CLOSE(unit_number)
END PROGRAM AppendToFile

Escriure Estructures de Dades Complexes

Per a estructures de dades més complexes, com tipus derivats, has d'escriure cada component individualment.

MODULE DataModule
  TYPE :: Person
    CHARACTER(LEN=20) :: name
    INTEGER :: age
  END TYPE Person
END MODULE DataModule

PROGRAM WriteComplexData
  USE DataModule
  INTEGER :: unit_number
  TYPE(Person) :: person
  person = Person('John Doe', 30)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') person%name, person%age
  CLOSE(unit_number)
END PROGRAM WriteComplexData

Conclusió

Escriure a fitxers en Fortran implica diversos passos: obrir el fitxer, escriure dades i tancar el fitxer. Pots formatar la sortida, gestionar errors, afegir a fitxers i escriure estructures de dades complexes. Dominar aquestes tècniques et permetrà gestionar eficaçment la sortida de dades en els teus programes Fortran.
