Els mòduls i paquets són components essencials en Fortran que ajuden a organitzar el codi, promoure la reutilització i mantenir un espai de noms net. Aquesta secció et guiarà a través dels conceptes bàsics dels mòduls, com crear-los i utilitzar-los, i com organitzar-los en paquets per a projectes més complexos.

Introducció als Mòduls

Els mòduls en Fortran s'utilitzen per agrupar procediments i dades relacionades. Ajuden a encapsular dades i procediments, fent que el codi sigui més modular i fàcil de gestionar.

Conceptes Clau

  • Definició de Mòdul: Un mòdul es defineix utilitzant les instruccions MODULE i END MODULE.
  • Accés Públic i Privat: Per defecte, totes les entitats en un mòdul són públiques, però es pot restringir l'accés utilitzant la paraula clau PRIVATE.
  • Instrucció Use: Per accedir al contingut d'un mòdul en un altre programa o mòdul, s'utilitza la instrucció USE.

Exemple: Mòdul Bàsic

MODULE MathModule
  IMPLICIT NONE
  PRIVATE
  PUBLIC :: add, subtract

  CONTAINS

  FUNCTION add(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: add
    add = a + b
  END FUNCTION add

  FUNCTION subtract(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: subtract
    subtract = a - b
  END FUNCTION subtract

END MODULE MathModule

En aquest exemple:

  • El mòdul MathModule conté dues funcions públiques, add i subtract.
  • La paraula clau PRIVATE assegura que qualsevol altra entitat en el mòdul no sigui accessible des de fora.

Utilitzant un Mòdul

PROGRAM Main
  USE MathModule
  IMPLICIT NONE

  INTEGER :: result

  result = add(5, 3)
  PRINT *, "5 + 3 =", result

  result = subtract(5, 3)
  PRINT *, "5 - 3 =", result

END PROGRAM Main

Aquí, la instrucció USE MathModule fa que les funcions add i subtract estiguin disponibles en el programa Main.

Característiques Avançades del Mòdul

Procediments de Mòdul

Els mòduls poden contenir procediments que es defineixen dins del mateix mòdul. Aquests es coneixen com a procediments de mòdul.

MODULE AdvancedMathModule
  IMPLICIT NONE
  PUBLIC :: multiply

  CONTAINS

  FUNCTION multiply(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: multiply
    multiply = a * b
  END FUNCTION multiply

END MODULE AdvancedMathModule

Variables de Mòdul

Els mòduls també poden contenir variables que es poden compartir entre procediments.

MODULE SharedDataModule
  IMPLICIT NONE
  INTEGER :: counter = 0
  PUBLIC :: counter

  CONTAINS

  SUBROUTINE increment_counter()
    counter = counter + 1
  END SUBROUTINE increment_counter

END MODULE SharedDataModule

Exemple: Utilitzant Variables de Mòdul

PROGRAM CounterProgram
  USE SharedDataModule
  IMPLICIT NONE

  CALL increment_counter()
  PRINT *, "Counter:", counter

  CALL increment_counter()
  PRINT *, "Counter:", counter

END PROGRAM CounterProgram

Paquets en Fortran

Fortran no té un concepte integrat de paquets com alguns altres llenguatges de programació, però pots organitzar els teus mòduls en directoris per crear una estructura semblant a un paquet.

Organitzant Mòduls en Paquets

  1. Estructura de Directori: Crea un directori per al teu paquet i col·loca els mòduls relacionats dins d'ell.
  2. Rutes d'Inclusió: Utilitza opcions del compilador per especificar les rutes als teus fitxers de mòdul.

Exemple d'Estructura de Directori

my_project/
  src/
    math/
      MathModule.f90
      AdvancedMathModule.f90
    main/
      MainProgram.f90

Compilant amb Paquets

Quan compiles, has d'especificar les rutes als fitxers de mòdul. Per exemple, utilitzant gfortran:

gfortran -I./src/math -o main_program ./src/main/MainProgram.f90 ./src/math/MathModule.f90 ./src/math/AdvancedMathModule.f90

Conclusió

Els mòduls i paquets en Fortran són eines poderoses per organitzar i gestionar el teu codi. En encapsular procediments i dades relacionades, els mòduls promouen la reutilització i el manteniment. Tot i que Fortran no té un sistema formal de paquets, pots aconseguir una funcionalitat similar organitzant els teus mòduls en directoris i gestionant les rutes d'inclusió durant la compilació. Entendre i utilitzar efectivament els mòduls millorarà significativament les teves habilitats de programació en Fortran.
