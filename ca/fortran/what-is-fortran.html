Fortran, abreviatura de "Formula Translation", és un llenguatge de programació d'alt nivell que va ser desenvolupat als anys 1950 per IBM. És un dels llenguatges de programació més antics i està especialment ben adaptat per al càlcul numèric i la computació científica.

Història de Fortran

  • Anys 1950: Desenvolupat per IBM, Fortran va ser el primer llenguatge de programació d'alt nivell.
  • Anys 1960-1970: Es van introduir Fortran II, Fortran IV i Fortran 66, afegint més característiques i millorant el llenguatge.
  • Anys 1980: Fortran 77 es va convertir en l'estàndard, introduint estructures de programació estructurada.
  • Anys 1990: Fortran 90 i Fortran 95 van afegir característiques modernes de programació com la programació amb arrays i la modularitat.
  • Anys 2000-Present: Fortran 2003, Fortran 2008 i Fortran 2018 han continuat evolucionant, afegint programació orientada a objectes i altres característiques avançades.

Característiques Clau de Fortran

  • Càlcul Numèric: Optimitzat per a càlculs matemàtics i científics.
  • Gestió d'Arrays: Manipulació eficient d'arrays.
  • Portabilitat: El codi es pot executar en diferents tipus de sistemes informàtics amb canvis mínims.
  • Rendiment: Alt rendiment per a tasques computacionals.
  • Codi Heretat: Extenses biblioteques i bases de codi que s'han desenvolupat durant dècades.

Sintaxi i Estructura Bàsica

El codi Fortran s'organitza en programes i subprogrames. Aquí hi ha un exemple senzill d'un programa Fortran:

program HelloWorld
  print *, 'Hello, World!'
end program HelloWorld

Explicació:

  • program HelloWorld: Declara l'inici d'un programa anomenat HelloWorld.
  • print *, 'Hello, World!': Imprimeix la cadena 'Hello, World!' a la consola.
  • end program HelloWorld: Marca el final del programa.

Tipus de Dades

Fortran admet diversos tipus de dades, incloent:

  • Integer: Nombres enters.
  • Real: Nombres de punt flotant.
  • Complex: Nombres complexos.
  • Character: Cadenes de text.
  • Logical: Valors booleans (cert o fals).

Exemple:

program DataTypes
  integer :: i = 10
  real :: x = 3.14
  complex :: z = (1.0, 2.0)
  character(len=20) :: str = 'Fortran'
  logical :: flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Complex:', z
  print *, 'Character:', str
  print *, 'Logical:', flag
end program DataTypes

Estructures de Control

Fortran inclou estructures de control estàndard com:

  • If Statements: Execució condicional.
  • Loops: Repetició de blocs de codi (per exemple, bucles do).

Exemple:

program ControlStructures
  integer :: i

  ! If statement
  if (i > 0) then
    print *, 'i is positive'
  else
    print *, 'i is non-positive'
  end if

  ! Do loop
  do i = 1, 5
    print *, 'Iteration:', i
  end do
end program ControlStructures

Arrays

Fortran excel·leix en la gestió d'arrays, permetent operacions eficients i concises sobre arrays.

Exemple:

program Arrays
  integer, dimension(5) :: arr = [1, 2, 3, 4, 5]
  integer :: i

  ! Print array elements
  do i = 1, 5
    print *, 'Element', i, ':', arr(i)
  end do
end program Arrays

Conclusió

Fortran és un llenguatge potent per a la computació numèrica i científica, amb una rica història i un fort enfocament en el rendiment i la gestió d'arrays. Entendre la seva sintaxi bàsica, tipus de dades, estructures de control i operacions amb arrays proporciona una base sòlida per a una exploració més profunda i una programació avançada en Fortran.
