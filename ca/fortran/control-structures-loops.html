Introducció als Bucles en Fortran

Els bucles són estructures de control fonamentals en la programació que permeten executar un bloc de codi múltiples vegades. En Fortran, hi ha diversos tipus de bucles, cadascun adequat per a diferents escenaris. Aquesta secció cobrirà les construccions bàsiques de bucles en Fortran, incloent-hi els bucles DO, DO WHILE i FORALL.

Bucle DO

El bucle DO és el bucle més comú en Fortran. Repeteix un bloc de codi un nombre especificat de vegades.

Sintaxi

DO variable = start, end, step
    ! Codi a executar
END DO
  • variable: La variable de control del bucle.
  • start: El valor inicial de la variable de control del bucle.
  • end: El valor final de la variable de control del bucle.
  • step: L'increment (opcional, per defecte és 1).

Exemple

PROGRAM DoLoopExample
    INTEGER :: i
    DO i = 1, 5
        PRINT *, "Iteració: ", i
    END DO
END PROGRAM DoLoopExample

En aquest exemple, el bucle imprimirà el número d'iteració de l'1 al 5.

Bucles DO Imbricats

Pots imbricar bucles DO per manejar iteracions més complexes.

Exemple

PROGRAM NestedDoLoopExample
    INTEGER :: i, j
    DO i = 1, 3
        DO j = 1, 2
            PRINT *, "i = ", i, " j = ", j
        END DO
    END DO
END PROGRAM NestedDoLoopExample

Aquest exemple demostra un bucle imbricat on el bucle intern s'executa dues vegades per a cada iteració del bucle extern.

Bucle DO WHILE

El bucle DO WHILE continua executant-se mentre una condició especificada sigui certa.

Sintaxi

DO WHILE (condition)
    ! Codi a executar
END DO

Exemple

PROGRAM DoWhileExample
    INTEGER :: i
    i = 1
    DO WHILE (i <= 5)
        PRINT *, "Iteració: ", i
        i = i + 1
    END DO
END PROGRAM DoWhileExample

En aquest exemple, el bucle s'executarà mentre i sigui menor o igual a 5.

Bucle FORALL

El bucle FORALL s'utilitza per a operacions amb arrays i pot ser més eficient per a certs tipus de càlculs.

Sintaxi

FORALL (index = start:end:step, condition)
    array(index) = expression
END FORALL

Exemple

PROGRAM ForallExample
    INTEGER, DIMENSION(5) :: array
    FORALL (i = 1:5)
        array(i) = i * i
    END FORALL
    PRINT *, array
END PROGRAM ForallExample

Aquest exemple inicialitza un array amb els quadrats dels seus índexs.

Resum

En aquesta secció, hem cobert les construccions bàsiques de bucles en Fortran:

  • Bucle DO: Repeteix un bloc de codi un nombre especificat de vegades.
  • Bucles DO Imbricats: Permet iteracions més complexes.
  • Bucle DO WHILE: Continua executant-se mentre una condició sigui certa.
  • Bucle FORALL: Utilitzat per a operacions eficients amb arrays.

Entendre aquests bucles t'ajudarà a escriure programes en Fortran més eficients i llegibles. Practica utilitzant aquests bucles amb diferents condicions i rangs per familiaritzar-te amb la seva sintaxi i comportament.
