Introducció

Fortran, abreviatura de "Formula Translation," és un dels llenguatges de programació d'alt nivell més antics, utilitzat principalment per a càlculs numèrics i científics. Entendre les variables i els tipus de dades és fonamental per programar en Fortran. Aquesta secció cobrirà els conceptes bàsics de la declaració de variables, la comprensió dels diferents tipus de dades i el seu ús en Fortran.

Variables en Fortran

Les variables s'utilitzen per emmagatzemar dades que es poden manipular al llarg d'un programa. En Fortran, les variables s'han de declarar abans de ser utilitzades.

Declaració de Variables

Per declarar una variable en Fortran, s'especifica el tipus seguit del nom de la variable. Opcionalment, també es pot inicialitzar la variable en el moment de la declaració.

INTEGER :: age
REAL :: height = 5.9
CHARACTER(LEN=20) :: name = "John Doe"
  • INTEGER declara una variable enter.
  • REAL declara una variable de punt flotant.
  • CHARACTER declara una variable de cadena amb una longitud especificada.

Regles de Nomenclatura de Variables

  • Els noms de les variables han de començar amb una lletra.
  • Poden contenir lletres, dígits i guions baixos.
  • No distingeixen entre majúscules i minúscules (per exemple, Age i age es consideren el mateix).

Tipus de Dades en Fortran

Fortran admet diversos tipus de dades, cadascun adequat per a diferents tipus de dades. Entendre aquests tipus és crucial per a una programació eficient.

Tipus de Dades Bàsics

  1. INTEGER: Utilitzat per a nombres enters. 
    INTEGER :: count = 10
  2. REAL: Utilitzat per a nombres de punt flotant. 
    REAL :: temperature = 36.6
  3. DOUBLE PRECISION: Utilitzat per a nombres de punt flotant de doble precisió. 
    DOUBLE PRECISION :: largeNumber = 1.0D+300
  4. COMPLEX: Utilitzat per a nombres complexos. 
    COMPLEX :: z = (3.0, 4.0)
  5. CHARACTER: Utilitzat per a cadenes. 
    CHARACTER(LEN=10) :: greeting = "Hello"
  6. LOGICAL: Utilitzat per a valors booleans. 
    LOGICAL :: isValid = .TRUE.

Tipus de Dades Derivats

Fortran permet la creació de tipus de dades personalitzats utilitzant la paraula clau TYPE. Això és útil per agrupar variables relacionades.

TYPE :: Person
    CHARACTER(LEN=20) :: name
    INTEGER :: age
    REAL :: height
END TYPE Person

TYPE(Person) :: student
student%name = "Alice"
student%age = 21
student%height = 5.7

Conversió de Tipus

Fortran proporciona funcions intrínseques per convertir entre diferents tipus de dades.

  • INT(): Converteix a enter.
  • REAL(): Converteix a real.
  • DBLE(): Converteix a doble precisió.
  • CMPLX(): Converteix a complex.
  • CHAR(): Converteix a caràcter.

Exemple

INTEGER :: i = 5
REAL :: r
r = REAL(i)  ! Converteix enter a real

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors del mateix tipus. Es poden declarar amb dimensions especificades.

Declaració d'Arrays

INTEGER, DIMENSION(5) :: numbers = (/1, 2, 3, 4, 5/)
REAL, DIMENSION(3, 3) :: matrix

Accés als Elements de l'Array

Els elements de l'array s'accedeixen utilitzant índexs.

numbers(1) = 10  ! Estableix el primer element a 10
matrix(2, 3) = 5.5  ! Estableix l'element a la fila 2, columna 3 a 5.5

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en Fortran. Hem après a declarar variables, entendre els diferents tipus de dades i utilitzar arrays. Dominar aquests conceptes és essencial per escriure programes en Fortran eficients i efectius. A mesura que progressis, trobaràs estructures de dades i operacions més complexes, però una comprensió sòlida d'aquests fonaments proporcionarà una base forta per al teu viatge en la programació en Fortran.
