Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre Fortran. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per escriure, compilar i executar programes Fortran.

1. Instal·lació d'un Compilador Fortran

Un compilador Fortran és essencial per convertir el teu codi Fortran en programes executables. Hi ha diversos compiladors Fortran populars disponibles:

  • GNU Fortran (gfortran): Part del GNU Compiler Collection (GCC), és de codi obert i àmpliament utilitzat.
  • Intel Fortran Compiler (ifort): Conegut per les seves optimitzacions de rendiment, especialment en maquinari Intel.
  • NAG Fortran Compiler: Conegut per la seva forta conformitat amb els estàndards i les seves característiques de depuració.

Instal·lació de gfortran

A Windows

  1. Descarregar i Instal·lar MinGW-w64:

    • Visita la pàgina de descàrrega de MinGW-w64.
    • Tria l'instal·lador adequat per al teu sistema (32-bit o 64-bit).
    • Segueix les instruccions d'instal·lació, assegurant-te de seleccionar el compilador Fortran durant la configuració.

  2. Verificar la Instal·lació:

    gfortran --version

    Aquesta comanda hauria de mostrar la versió de gfortran instal·lada.

A macOS

  1. Instal·lar Homebrew (si no està instal·lat):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·lar gfortran:

    brew install gcc

    Això instal·larà gfortran juntament amb altres compiladors GCC.

  3. Verificar la Instal·lació:

    gfortran --version

A Linux

  1. Instal·lar gfortran utilitzant el teu gestor de paquets:

    • Debian/Ubuntu: 
      sudo apt-get update
sudo apt-get install gfortran
    • Fedora: 
      sudo dnf install gcc-gfortran

  2. Verificar la Instal·lació:

    gfortran --version

2. Configuració d'un Entorn de Desenvolupament Integrat (IDE)

Un IDE pot millorar significativament la teva productivitat proporcionant característiques com la ressaltat de sintaxi, la completació de codi i les eines de depuració. Alguns IDEs populars per Fortran inclouen:

  • Visual Studio Code (VS Code): Un editor altament personalitzable i extensible.
  • Eclipse amb Photran: Un IDE potent amb un complement dedicat a Fortran.
  • Code::Blocks: Un IDE gratuït i de codi obert que suporta Fortran.

Configuració de Visual Studio Code per Fortran

  1. Instal·lar Visual Studio Code:

    • Descarrega i instal·la VS Code des del lloc web oficial.

  2. Instal·lar Extensions Fortran:

    • Obre VS Code i ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra.
    • Cerca "Modern Fortran" i instal·la'l.

  3. Configurar el Compilador Fortran:

    • Obre el Command Palette (Ctrl+Shift+P) i escriu "Preferences: Open Settings (JSON)".
    • Afegeix la següent configuració per especificar el camí al teu compilador gfortran: 
      {
  "fortran.compilerPath": "/path/to/gfortran"
}

  4. Crear i Executar un Programa Fortran Simple:

    • Crea un nou fitxer amb una extensió .f90 (per exemple, hello_world.f90).
    • Escriu un programa Fortran simple: 
      program hello
  print *, "Hello, Fortran!"
end program hello
    • Desa el fitxer i obre el terminal a VS Code.
    • Compila el programa: 
      gfortran -o hello_world hello_world.f90
    • Executa l'executable: 
      ./hello_world

3. Configuració de Variables d'Entorn

Configurar variables d'entorn pot ajudar a agilitzar el procés de desenvolupament fent que les eines i les biblioteques siguin fàcilment accessibles.

Afegir gfortran al PATH

A Windows

  1. Obrir Propietats del Sistema:

    • Fes clic amb el botó dret a "Aquest PC" o "Ordinador" a l'escriptori o a l'Explorador de Fitxers.
    • Selecciona "Propietats" i després "Configuració avançada del sistema".

  2. Editar Variables d'Entorn:

    • Fes clic a "Variables d'entorn".
    • Sota "Variables del sistema", troba la variable Path i fes clic a "Editar".
    • Afegeix el camí al directori que conté gfortran.exe (per exemple, C:\MinGW\bin).

  3. Verificar:

    • Obre un nou Símbol del Sistema i escriu: 
      gfortran --version

A macOS i Linux

  1. Editar el Fitxer de Configuració de la Shell:

    • Obre ~/.bashrc (o ~/.zshrc si utilitzes Zsh) en un editor de text.
    • Afegeix la següent línia: 
      export PATH="/path/to/gfortran:$PATH"

  2. Aplicar els Canvis:

    source ~/.bashrc

  3. Verificar:

    gfortran --version

Conclusió

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert en programació Fortran. Instal·lant un compilador Fortran, configurant un IDE i establint variables d'entorn, crearàs una base robusta per escriure, compilar i executar programes Fortran de manera eficient. Amb el teu entorn preparat, ara pots centrar-te en aprendre el llenguatge i desenvolupar les teves habilitats de programació.
