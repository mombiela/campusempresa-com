Introducció al Càlcul Científic

El càlcul científic implica l'ús de mètodes computacionals i programari per resoldre problemes científics i d'enginyeria complexos. Fortran, abreviatura de "Formula Translation," és un llenguatge de programació d'alt nivell que ha estat àmpliament utilitzat en el càlcul científic a causa de la seva eficiència en el càlcul numèric i la manipulació de matrius.

Començant amb Fortran

Abans d'endinsar-se en el càlcul científic, és essencial entendre els conceptes bàsics de Fortran.

Sintaxi i Estructura Bàsica

  • Estructura del Programa: Un programa Fortran típicament consta d'un nom de programa, declaracions, instruccions executables i una instrucció final.
  • Variables i Tipus de Dades: Els tipus de dades comuns inclouen INTEGER, REAL, DOUBLE PRECISION, COMPLEX, LOGICAL i CHARACTER.
  • Estructures de Control: Inclouen instruccions IF, bucles DO i instruccions SELECT CASE.

Exemple: Programa Simple en Fortran

PROGRAM HelloWorld
  PRINT *, 'Hello, World!'
END PROGRAM HelloWorld

Compilar i Executar Programes Fortran

  • Compilar: Utilitza un compilador Fortran com gfortran per compilar el teu codi. 
    gfortran HelloWorld.f90 -o HelloWorld
  • Executar: Executa el programa compilat. 
    ./HelloWorld

Mètodes Numèrics en Fortran

Els mètodes numèrics són algoritmes utilitzats per resoldre problemes matemàtics numèricament.

Resolució d'Equacions Lineals

  • Eliminació de Gauss: Un mètode per resoldre un sistema d'equacions lineals.
  • Descomposició LU: Descompon una matriu en una matriu triangular inferior (L) i una matriu triangular superior (U).

Exemple: Resolució d'Equacions Lineals utilitzant Eliminació de Gauss

PROGRAM GaussElimination
  IMPLICIT NONE
  INTEGER, PARAMETER :: n = 3
  REAL :: A(n,n), b(n), x(n)
  INTEGER :: i, j, k

  ! Inicialitzar matriu A i vector b
  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [8.0, 0.0, 6.0]

  ! Eliminació cap endavant
  DO k = 1, n-1
    DO i = k+1, n
      A(i,k:n) = A(i,k:n) - A(i,k)/A(k,k) * A(k,k:n)
      b(i) = b(i) - A(i,k)/A(k,k) * b(k)
    END DO
  END DO

  ! Substitució cap enrere
  x(n) = b(n) / A(n,n)
  DO i = n-1, 1, -1
    x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
  END DO

  PRINT *, 'Solució: ', x
END PROGRAM GaussElimination

Integració Numèrica

  • Regla del Trapezi: Aproxima la integral d'una funció dividint l'àrea sota la corba en trapezis.
  • Regla de Simpson: Proporciona una aproximació més precisa utilitzant segments parabòlics.

Exemple: Integració Numèrica utilitzant la Regla del Trapezi

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i
  REAL, EXTERNAL :: f

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i*h)
  END DO

  integral = integral * h
  PRINT *, 'Integral: ', integral
END PROGRAM TrapezoidalRule

REAL FUNCTION f(x)
  REAL, INTENT(IN) :: x
  f = x**2
END FUNCTION f

Temes Avançats en Fortran

A mesura que avancis, trobaràs temes més complexos i especialitzats.

Càlcul Paral·lel amb Fortran

  • OpenMP: Una API àmpliament utilitzada per a la programació paral·lela en entorns de memòria compartida.
  • MPI: Interfície de Pas de Missatges per a la programació paral·lela en sistemes de memòria distribuïda.

Exemple: Càlcul Paral·lel amb OpenMP

PROGRAM ParallelSum
  USE omp_lib
  IMPLICIT NONE
  INTEGER :: i, n
  REAL :: sum
  REAL, ALLOCATABLE :: a(:)

  n = 1000000
  ALLOCATE(a(n))
  a = 1.0

  sum = 0.0
  !$OMP PARALLEL DO REDUCTION(+:sum)
  DO i = 1, n
    sum = sum + a(i)
  END DO
  !$OMP END PARALLEL DO

  PRINT *, 'Sum: ', sum
END PROGRAM ParallelSum

Biblioteques Fortran per al Càlcul Científic

  • LAPACK: Linear Algebra PACKage per resoldre problemes d'àlgebra lineal.
  • BLAS: Basic Linear Algebra Subprograms per realitzar operacions bàsiques de vectors i matrius.
  • FFTW: Fastest Fourier Transform in the West per calcular transformades de Fourier.

Conclusió

Fortran continua sent una eina poderosa per al càlcul científic a causa de la seva eficiència i les seves àmplies biblioteques. En dominar Fortran, pots abordar una àmplia gamma de problemes numèrics i científics, des de l'àlgebra lineal bàsica fins al càlcul paral·lel avançat. Aquest curs ha proporcionat una base en la programació Fortran, mètodes numèrics i temes avançats, equipant-te amb les habilitats necessàries per al càlcul científic.
