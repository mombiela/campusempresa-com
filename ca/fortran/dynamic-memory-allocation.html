L'assignació dinàmica de memòria és un concepte crucial en Fortran, especialment per gestionar estructures de dades la mida de les quals pot no ser coneguda en temps de compilació. Aquest tema et guiarà des dels conceptes bàsics fins als avançats de l'assignació dinàmica de memòria en Fortran.

Introducció a l'Assignació Dinàmica de Memòria

L'assignació dinàmica de memòria et permet assignar memòria durant el temps d'execució del programa. Això és particularment útil quan es treballa amb arrays o altres estructures de dades on la mida pot variar.

Conceptes Clau

  • Assignació Estàtica vs. Dinàmica de Memòria: L'assignació estàtica de memòria es produeix en temps de compilació, mentre que l'assignació dinàmica de memòria passa en temps d'execució.
  • Instruccions ALLOCATE i DEALLOCATE: S'utilitzen per assignar i alliberar memòria dinàmicament.
  • Punteres: Variables que emmagatzemen l'adreça de la memòria assignada dinàmicament.

Assignació Dinàmica de Memòria Bàsica

Instruccions ALLOCATE i DEALLOCATE

La instrucció ALLOCATE s'utilitza per assignar memòria per a arrays o altres estructures de dades, i la instrucció DEALLOCATE s'utilitza per alliberar aquesta memòria.

Exemple: Assignació i Alliberament d'un Array

PROGRAM DynamicArray
  IMPLICIT NONE
  INTEGER, ALLOCATABLE :: array(:)
  INTEGER :: n, i

  ! Pregunta a l'usuari la mida de l'array
  PRINT *, 'Introdueix la mida de l''array:'
  READ *, n

  ! Assigna memòria per a l'array
  ALLOCATE(array(n))

  ! Inicialitza l'array
  DO i = 1, n
    array(i) = i * 2
  END DO

  ! Imprimeix l'array
  PRINT *, 'Elements de l''array:'
  PRINT *, array

  ! Allibera la memòria
  DEALLOCATE(array)
END PROGRAM DynamicArray

Gestió d'Errors en l'Assignació

És important gestionar els errors que poden ocórrer durant l'assignació de memòria. Això es pot fer utilitzant l'especificador STAT.

Exemple: Gestió d'Errors en l'Assignació

PROGRAM AllocationErrorHandling
  IMPLICIT NONE
  INTEGER, ALLOCATABLE :: array(:)
  INTEGER :: n, stat

  ! Pregunta a l'usuari la mida de l'array
  PRINT *, 'Introdueix la mida de l''array:'
  READ *, n

  ! Assigna memòria per a l'array amb gestió d'errors
  ALLOCATE(array(n), STAT=stat)
  IF (stat /= 0) THEN
    PRINT *, 'Error: No es pot assignar memòria.'
    STOP
  END IF

  ! Allibera la memòria
  DEALLOCATE(array)
END PROGRAM AllocationErrorHandling

Assignació Dinàmica de Memòria Avançada

Ús de Punteres

Les punteres en Fortran es poden utilitzar per assignar memòria dinàmicament i referenciar-la. Això és útil per crear estructures de dades complexes com llistes enllaçades.

Exemple: Ús de Punteres per a l'Assignació Dinàmica

PROGRAM PointerExample
  IMPLICIT NONE
  INTEGER, POINTER :: pArray(:)
  INTEGER :: n, i

  ! Pregunta a l'usuari la mida de l'array
  PRINT *, 'Introdueix la mida de l''array:'
  READ *, n

  ! Assigna memòria per a l'array utilitzant punteres
  ALLOCATE(pArray(n))

  ! Inicialitza l'array
  DO i = 1, n
    pArray(i) = i * 3
  END DO

  ! Imprimeix l'array
  PRINT *, 'Elements de l''array de punteres:'
  PRINT *, pArray

  ! Allibera la memòria
  DEALLOCATE(pArray)
END PROGRAM PointerExample

Arrays Dinàmics en Tipus Derivats

Fortran permet arrays dinàmics dins de tipus derivats, cosa que pot ser útil per crear estructures de dades més complexes.

Exemple: Arrays Dinàmics en Tipus Derivats

MODULE DynamicModule
  TYPE :: DynamicType
    INTEGER, ALLOCATABLE :: array(:)
  END TYPE DynamicType
END MODULE DynamicModule

PROGRAM DerivedTypeExample
  USE DynamicModule
  IMPLICIT NONE
  TYPE(DynamicType) :: myType
  INTEGER :: n, i

  ! Pregunta a l'usuari la mida de l'array
  PRINT *, 'Introdueix la mida de l''array:'
  READ *, n

  ! Assigna memòria per a l'array dins del tipus derivat
  ALLOCATE(myType%array(n))

  ! Inicialitza l'array
  DO i = 1, n
    myType%array(i) = i * 4
  END DO

  ! Imprimeix l'array
  PRINT *, 'Elements de l''array del tipus derivat:'
  PRINT *, myType%array

  ! Allibera la memòria
  DEALLOCATE(myType%array)
END PROGRAM DerivedTypeExample

Resum

L'assignació dinàmica de memòria en Fortran és una característica poderosa que permet una gestió de memòria flexible i eficient. Utilitzant les instruccions ALLOCATE i DEALLOCATE, gestionant errors, i utilitzant punteres i tipus derivats, pots crear estructures de dades complexes i dinàmiques. Entendre aquests conceptes és essencial per a la programació avançada en Fortran i pot millorar significativament el rendiment i la flexibilitat dels teus programes.
