Primera consulta amb GraphQL
Tipus de GraphQL ►

En aquesta secció, aprendrem a escriure i executar la nostra primera consulta utilitzant GraphQL. Cobrirà els conceptes bàsics de la consulta de dades, la comprensió de l'estructura d'una consulta GraphQL i com interpretar els resultats.

Conceptes clau

  • Consulta GraphQL: Una sol·licitud de dades que especifica exactament quina informació necessites.
  • Camps: Les peces específiques de dades que sol·licites en una consulta.
  • Arguments: Paràmetres que pots passar als camps per filtrar o modificar les dades retornades.
  • Resposta: Les dades retornades pel servidor en resposta a la teva consulta.

Escriure la teva primera consulta

Per començar amb GraphQL, necessites entendre l'estructura bàsica d'una consulta. Una consulta GraphQL s'escriu en una sintaxi específica que et permet especificar exactament quines dades necessites.

Estructura bàsica de la consulta

Una consulta GraphQL simple podria semblar així:

{
  user(id: "1") {
    id
    name
    email
  }
}
  • user(id: "1"): Aquest és el camp que estem consultant. Estem demanant un usuari amb l'ID "1".
  • id, name, email: Aquestes són les peces específiques de dades que volem recuperar per a l'usuari.

Explicació

  • Claudàtors {}: Envolten tota la consulta.
  • Nom del camp user: El tipus de dades que estem consultant.
  • Arguments (id: "1"): Paràmetres per filtrar les dades.
  • Subcamps id, name, email: Punts de dades específics que volem recuperar.

Consulta d'exemple

Escrivim una consulta per obtenir una llista de llibres amb els seus títols i autors:

{
  books {
    title
    author {
      name
    }
  }
}
  • books: El camp que estem consultant per obtenir una llista de llibres.
  • title: El títol de cada llibre.
  • author: Un camp niat per obtenir els detalls de l'autor.
  • name: El nom de l'autor.

Executar la consulta

Per executar una consulta GraphQL, normalment utilitzes un client GraphQL o un entorn de desenvolupament integrat (IDE) com GraphiQL o Apollo Studio. Aquí tens com fer-ho:

  1. Obre GraphiQL o Apollo Studio.
  2. Introdueix la consulta: Copia i enganxa la teva consulta a l'editor.
  3. Executa la consulta: Fes clic al botó "Run" per executar la consulta.

Resposta d'exemple

Quan executis la consulta anterior, podries obtenir una resposta com aquesta:

{
  "data": {
    "books": [
      {
        "title": "El gran Gatsby",
        "author": {
          "name": "F. Scott Fitzgerald"
        }
      },
      {
        "title": "1984",
        "author": {
          "name": "George Orwell"
        }
      }
    ]
  }
}
  • data: El camp arrel de la resposta.
  • books: La llista de llibres retornada per la consulta.
  • title i author.name: Els punts de dades específics que hem sol·licitat.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'escriure i executar una consulta GraphQL. Hem après sobre l'estructura d'una consulta, com especificar les dades que necessitem i com interpretar la resposta. Dominant aquests conceptes bàsics, ara estàs preparat per explorar funcions més avançades de GraphQL, com les mutacions i les subscripcions.

A la següent secció, aprofundirem en la consulta amb arguments i variables per fer les nostres consultes més dinàmiques i potents.
