Introducció

En aquesta secció, cobrirem els conceptes bàsics per configurar un servidor GraphQL. Utilitzarem Node.js i Express juntament amb les biblioteques graphql i express-graphql per crear un servidor GraphQL senzill. Al final d'aquesta secció, tindràs un servidor GraphQL funcional que podràs consultar.

Prerequisits

Abans de començar, assegura't de tenir instal·lat el següent:

  • Node.js i npm
  • Coneixements bàsics de JavaScript i Node.js

Pas 1: Inicialitzar un projecte Node.js

Primer, crea un nou directori per al teu projecte i navega-hi. Després, inicialitza un nou projecte Node.js.

mkdir graphql-server
cd graphql-server
npm init -y

Pas 2: Instal·lar els paquets necessaris

A continuació, instal·la els paquets necessaris: express, graphql i express-graphql.

npm install express graphql express-graphql

Pas 3: Crear el servidor

Crea un nou fitxer anomenat server.js i configura un servidor bàsic d'Express.

const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

const app = express();
const port = 4000;

// Definir un esquema
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// Definir un resolver
const root = {
  hello: () => {
    return 'Hello, world!';
  },
};

// Configurar el punt final de GraphQL
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(port, () => {
  console.log(`Server is running at http://localhost:${port}/graphql`);
});

Explicació

  • Configuració d'Express: Creem una aplicació d'Express i la configurem per escoltar al port 4000.
  • Esquema de GraphQL: Definim un esquema senzill amb una única consulta hello que retorna una String.
  • Resolver: Creem un resolver per a la consulta hello que retorna la cadena 'Hello, world!'.
  • Punt final de GraphQL: Configurem el punt final /graphql utilitzant express-graphql i habilitem la interfície GraphiQL per a proves.

Pas 4: Executar el servidor

Executa el servidor utilitzant Node.js.

node server.js

Hauries de veure el missatge Server is running at http://localhost:4000/graphql al teu terminal. Obre el teu navegador i navega a http://localhost:4000/graphql. Hauries de veure la interfície GraphiQL on pots executar la següent consulta:

{
  hello
}

Hauries d'obtenir la següent resposta:

{
  "data": {
    "hello": "Hello, world!"
  }
}

Conclusió

En aquesta secció, hem configurat amb èxit un servidor GraphQL bàsic utilitzant Node.js, Express i la biblioteca express-graphql. Hem definit un esquema i un resolver senzills, i hem provat la nostra configuració utilitzant la interfície GraphiQL. Aquesta configuració fonamental servirà com a base per a temes més avançats en GraphQL.
