En aquesta secció, veurem com configurar un projecte GraphQL des de zero. Això inclou instal·lar les eines necessàries, inicialitzar el projecte i configurar l'estructura bàsica. Al final d'aquesta secció, tindràs un servidor GraphQL funcional sobre el qual podràs construir en les lliçons següents.

Prerequisits

Abans de començar, assegura't de tenir instal·lat el següent al teu ordinador:

  • Node.js (versió 12.x o posterior)
  • npm (Node Package Manager) o yarn

Pots comprovar si tens instal·lats Node.js i npm executant les següents comandes al teu terminal:

node -v
npm -v

Inicialització del Projecte

  1. Crea un nou directori per al teu projecte:

    mkdir graphql-project
cd graphql-project

  2. Inicialitza un nou projecte Node.js:

    npm init -y

    Aquesta comanda crearà un fitxer package.json amb la configuració per defecte.

Instal·lació de Dependències

Necessitarem diversos paquets per configurar el nostre servidor GraphQL. Els paquets principals són express per al nostre servidor, graphql per a la funcionalitat GraphQL, i express-graphql per integrar GraphQL amb Express.

  1. Instal·la els paquets necessaris: 
    npm install express graphql express-graphql

Configuració del Servidor

  1. Crea un nou fitxer anomenat server.js al directori arrel:

    touch server.js

  2. Obre server.js i afegeix el següent codi:

    const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

// Construeix un esquema, utilitzant el llenguatge d'esquemes de GraphQL
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// El root proporciona una funció resolutora per a cada punt final de l'API
const root = {
  hello: () => {
    return 'Hello world!';
  },
};

const app = express();
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Now browse to localhost:4000/graphql'));

Explicació del Codi

  • Importació de Mòduls:

    const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

    Importem els mòduls necessaris per configurar el nostre servidor i la integració de GraphQL.

  • Definició de l'Esquema:

    const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

    Definim un esquema senzill amb una única consulta hello que retorna un String.

  • Creació del Root Resolver:

    const root = {
  hello: () => {
    return 'Hello world!';
  },
};

    La funció resolutora per a la consulta hello retorna la cadena "Hello world!".

  • Configuració del Servidor Express:

    const app = express();
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Now browse to localhost:4000/graphql'));

    Configurem un servidor Express i configurem el punt final /graphql per utilitzar express-graphql amb el nostre esquema i root resolver. També habilitem la interfície GraphiQL per provar les nostres consultes GraphQL.

Execució del Servidor

  1. Inicia el servidor executant:

    node server.js

  2. Obre el teu navegador i navega a:

    http://localhost:4000/graphql

    Hauries de veure la interfície GraphiQL on pots provar les teves consultes GraphQL.

Conclusió

En aquesta secció, hem configurat amb èxit un servidor GraphQL bàsic utilitzant Node.js, Express i el middleware express-graphql. Hem creat un esquema i un resolutor senzills, i hem configurat el nostre servidor per utilitzar GraphiQL per a les proves. Aquesta configuració fonamental ens permetrà construir APIs GraphQL més complexes en les seccions següents.
