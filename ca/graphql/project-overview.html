Introducció a GraphQL

GraphQL és un llenguatge de consulta per a APIs i un entorn d'execució per a executar aquestes consultes utilitzant un sistema de tipus que defineixes per a les teves dades. Proporciona una alternativa més eficient, potent i flexible a REST. Aquest curs et guiarà des dels conceptes bàsics de GraphQL fins a conceptes avançats, assegurant que tinguis una comprensió completa de com utilitzar GraphQL en els teus projectes.

Conceptes Clau

  • Esquema: Defineix l'estructura de les teves dades i els tipus de consultes que es poden fer.
  • Consultes: Sol·licituds fetes al servidor GraphQL per recuperar dades.
  • Mutacions: Sol·licituds fetes per modificar dades al servidor.
  • Resolvers: Funcions que gestionen la lògica per obtenir les dades per a una consulta o mutació.
  • Subscripcions: Permeten als clients rebre actualitzacions en temps real del servidor.
  • Tipus: Defineixen la forma de les dades i inclouen escalars, objectes, enums, interfícies, unions i tipus d'entrada.

Exemple Bàsic

Comencem amb un exemple senzill per il·lustrar els components bàsics d'una API GraphQL.

Definició de l'Esquema

type Query {
  hello: String
}

Aquest esquema defineix una única consulta hello que retorna un String.

Funció Resolver

const resolvers = {
  Query: {
    hello: () => 'Hello, world!',
  },
};

La funció resolver per a la consulta hello retorna la cadena "Hello, world!".

Configuració del Servidor

const { ApolloServer, gql } = require('apollo-server');

const typeDefs = gql`
  type Query {
    hello: String
  }
`;

const resolvers = {
  Query: {
    hello: () => 'Hello, world!',
  },
};

const server = new ApolloServer({ typeDefs, resolvers });

server.listen().then(({ url }) => {
  console.log(`🚀 Server ready at ${url}`);
});

Aquest codi configura un servidor Apollo bàsic amb l'esquema i el resolver definits.

Conceptes Avançats

A mesura que avancis, aprendràs sobre temes més avançats com:

  • Fragments: Unitats reutilitzables d'una consulta GraphQL.
  • Directives: Proporcionen una manera d'alterar dinàmicament l'execució d'una consulta.
  • Batching i Caching: Tècniques per optimitzar el rendiment del teu servidor GraphQL.
  • Autenticació i Autorització: Assegurant la teva API GraphQL.
  • Gestió d'Errors: Gestionar i retornar errors de manera estructurada.

Exemple d'una Consulta Avançada

query GetUserData($id: ID!) {
  user(id: $id) {
    name
    email
    posts {
      title
      content
    }
  }
}

Aquesta consulta recupera el nom, el correu electrònic d'un usuari i els títols i continguts de les seves publicacions basant-se en l'ID de l'usuari.

Conclusió

En aquest curs, tu:

  • Començaràs amb els conceptes bàsics de GraphQL, entenent els seus conceptes clau i sintaxi.
  • Construiràs un servidor i un client GraphQL senzills.
  • Gradualment passaràs a temes més avançats, incloent actualitzacions en temps real amb subscripcions, optimització del rendiment i assegurament de la teva API.
  • Treballaràs en projectes pràctics i exercicis per solidificar la teva comprensió i habilitats.

Al final d'aquest curs, estaràs ben preparat per implementar i gestionar APIs GraphQL en els teus propis projectes, des d'aplicacions senzilles fins a sistemes complexos del món real.
