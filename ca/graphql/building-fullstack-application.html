Introducció a GraphQL

  • Què és GraphQL?

    • Un llenguatge de consulta per a la teva API.
    • Proporciona una descripció completa i comprensible de les dades a la teva API.
    • Permet als clients sol·licitar exactament el que necessiten i res més.

  • Per què utilitzar GraphQL?

    • Extracció eficient de dades.
    • Esquema fortament tipat.
    • Dades en temps real amb subscripcions.
    • Millor experiència per als desenvolupadors.

Configurant l'entorn

  • Prerequisits

    • Coneixements bàsics de JavaScript/TypeScript.
    • Familiaritat amb Node.js i Express.
    • Comprensió de frameworks de front-end com React.

  • Eines i biblioteques

    • Node.js
    • Express.js
    • Apollo Server
    • GraphQL
    • React
    • Apollo Client

Creant el backend amb GraphQL

  • Configurant el servidor

    mkdir graphql-fullstack
cd graphql-fullstack
npm init -y
npm install express apollo-server-express graphql

  • Creant el servidor Express

    const express = require('express');
const { ApolloServer, gql } = require('apollo-server-express');

const app = express();

const typeDefs = gql`
  type Query {
    hello: String
  }
`;

const resolvers = {
  Query: {
    hello: () => 'Hello world!',
  },
};

const server = new ApolloServer({ typeDefs, resolvers });
server.applyMiddleware({ app });

app.listen({ port: 4000 }, () =>
  console.log(`Server ready at http://localhost:4000${server.graphqlPath}`)
);

  • Definint l'esquema

    type Query {
  hello: String
  users: [User]
}

type User {
  id: ID!
  name: String!
  email: String!
}

  • Creant resolutors

    const resolvers = {
  Query: {
    hello: () => 'Hello world!',
    users: () => [
      { id: 1, name: 'John Doe', email: 'john@example.com' },
      { id: 2, name: 'Jane Doe', email: 'jane@example.com' },
    ],
  },
};

Connectant a una base de dades

  • Utilitzant MongoDB amb Mongoose

    npm install mongoose

  • Configurant models de Mongoose

    const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/graphql', {
  useNewUrlParser: true,
  useUnifiedTopology: true,
});

const UserSchema = new mongoose.Schema({
  name: String,
  email: String,
});

const User = mongoose.model('User', UserSchema);

  • Actualitzant resolutors per utilitzar la base de dades

    const resolvers = {
  Query: {
    hello: () => 'Hello world!',
    users: async () => await User.find(),
  },
};

Construint el frontend amb React i Apollo Client

  • Configurant l'aplicació React

    npx create-react-app client
cd client
npm install @apollo/client graphql

  • Creant Apollo Client

    import React from 'react';
import { ApolloClient, InMemoryCache, ApolloProvider } from '@apollo/client';
import App from './App';

const client = new ApolloClient({
  uri: 'http://localhost:4000/graphql',
  cache: new InMemoryCache(),
});

const Root = () => (
  <ApolloProvider client={client}>
    <App />
  </ApolloProvider>
);

export default Root;

  • Extracció de dades amb Apollo Client

    import React from 'react';
import { useQuery, gql } from '@apollo/client';

const GET_USERS = gql`
  query GetUsers {
    users {
      id
      name
      email
    }
  }
`;

const Users = () => {
  const { loading, error, data } = useQuery(GET_USERS);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error :(</p>;

  return (
    <ul>
      {data.users.map(({ id, name, email }) => (
        <li key={id}>
          {name} - {email}
        </li>
      ))}
    </ul>
  );
};

export default Users;

Funcionalitats avançades

  • Mutacions

    type Mutation {
  addUser(name: String!, email: String!): User
}

    const resolvers = {
  Mutation: {
    addUser: async (_, { name, email }) => {
      const user = new User({ name, email });
      await user.save();
      return user;
    },
  },
};

    import React, { useState } from 'react';
import { useMutation, gql } from '@apollo/client';

const ADD_USER = gql`
  mutation AddUser($name: String!, $email: String!) {
    addUser(name: $name, email: $email) {
      id
      name
      email
    }
  }
`;

const AddUser = () => {
  let input;
  const [addUser, { data }] = useMutation(ADD_USER);

  return (
    <div>
      <form
        onSubmit={e => {
          e.preventDefault();
          addUser({ variables: { name: input.value, email: input.value } });
          input.value = '';
        }}
      >
        <input
          ref={node => {
            input = node;
          }}
        />
        <button type="submit">Add User</button>
      </form>
    </div>
  );
};

export default AddUser;

  • Subscripcions

    type Subscription {
  userAdded: User
}

    const { PubSub } = require('apollo-server');
const pubsub = new PubSub();

const resolvers = {
  Subscription: {
    userAdded: {
      subscribe: () => pubsub.asyncIterator(['USER_ADDED']),
    },
  },
  Mutation: {
    addUser: async (_, { name, email }) => {
      const user = new User({ name, email });
      await user.save();
      pubsub.publish('USER_ADDED', { userAdded: user });
      return user;
    },
  },
};

    import React from 'react';
import { useSubscription, gql } from '@apollo/client';

const USER_ADDED = gql`
  subscription OnUserAdded {
    userAdded {
      id
      name
      email
    }
  }
`;

const UserAdded = () => {
  const { data, loading } = useSubscription(USER_ADDED);

  if (loading) return <p>Loading...</p>;

  return (
    <div>
      <p>New User Added:</p>
      <p>{data.userAdded.name} - {data.userAdded.email}</p>
    </div>
  );
};

export default UserAdded;

Conclusió

Construir una aplicació full-stack amb GraphQL implica configurar un servidor backend amb un esquema i resolutors ben definits, connectar-se a una base de dades i crear una aplicació frontend que interactuï amb l'API GraphQL. Aprofitant el poder de GraphQL, els desenvolupadors poden crear aplicacions eficients, escalables i flexibles que proporcionen una gran experiència d'usuari. Aquest curs ha cobert des dels conceptes bàsics fins a les funcionalitats avançades, proporcionant una base sòlida per construir aplicacions full-stack robustes amb GraphQL.
