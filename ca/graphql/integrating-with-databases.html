Integrar GraphQL amb bases de dades és una habilitat crucial per construir aplicacions eficients i escalables. Aquest tema et guiarà a través del procés de connectar GraphQL a diversos tipus de bases de dades, des de configurar la connexió fins a realitzar operacions CRUD.

Introducció a la Integració de Bases de Dades

  • Entendre el Rol de les Bases de Dades: Les bases de dades emmagatzemen i gestionen dades per a aplicacions. Integrar GraphQL amb bases de dades et permet consultar i manipular aquestes dades de manera eficient.
  • Tipus de Bases de Dades:
    • Bases de Dades SQL: Bases de dades relacionals com PostgreSQL, MySQL.
    • Bases de Dades NoSQL: Bases de dades no relacionals com MongoDB, Redis.

Configurant l'Entorn

Prerequisits

  • Coneixements bàsics de GraphQL.
  • Familiaritat amb JavaScript (Node.js) o un altre llenguatge backend.
  • Base de dades instal·lada (per exemple, PostgreSQL, MongoDB).

Instal·lant els Paquets Necessaris

Per a Node.js, normalment utilitzaràs paquets com graphql, express, apollo-server-express, i un client de base de dades (per exemple, pg per a PostgreSQL, mongoose per a MongoDB).

npm install graphql express apollo-server-express pg mongoose

Connectant a una Base de Dades SQL

Configurant PostgreSQL

  1. Instal·la PostgreSQL: Segueix la guia d'instal·lació per al teu sistema operatiu.
  2. Crea una Base de Dades: Utilitza la línia de comandes de PostgreSQL o una eina GUI com pgAdmin.
CREATE DATABASE mydatabase;

Connectant GraphQL a PostgreSQL

  1. Crea una Connexió: Utilitza el paquet pg per connectar-te a PostgreSQL.
const { Pool } = require('pg');
const pool = new Pool({
  user: 'yourusername',
  host: 'localhost',
  database: 'mydatabase',
  password: 'yourpassword',
  port: 5432,
});
  1. Defineix l'Esquema i els Resolvers de GraphQL:
const { ApolloServer, gql } = require('apollo-server-express');
const express = require('express');

const typeDefs = gql`
  type Query {
    users: [User]
  }

  type User {
    id: ID!
    name: String!
    email: String!
  }
`;

const resolvers = {
  Query: {
    users: async () => {
      const res = await pool.query('SELECT * FROM users');
      return res.rows;
    },
  },
};

const server = new ApolloServer({ typeDefs, resolvers });
const app = express();
server.applyMiddleware({ app });

app.listen({ port: 4000 }, () =>
  console.log(`Server ready at http://localhost:4000${server.graphqlPath}`)
);

Connectant a una Base de Dades NoSQL

Configurant MongoDB

  1. Instal·la MongoDB: Segueix la guia d'instal·lació per al teu sistema operatiu.
  2. Crea una Base de Dades: Utilitza la shell de MongoDB o una eina GUI com MongoDB Compass.

Connectant GraphQL a MongoDB

  1. Crea una Connexió: Utilitza el paquet mongoose per connectar-te a MongoDB.
const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/mydatabase', {
  useNewUrlParser: true,
  useUnifiedTopology: true,
});
  1. Defineix l'Esquema i els Resolvers de GraphQL:
const { ApolloServer, gql } = require('apollo-server-express');
const express = require('express');

const UserSchema = new mongoose.Schema({
  name: String,
  email: String,
});

const User = mongoose.model('User', UserSchema);

const typeDefs = gql`
  type Query {
    users: [User]
  }

  type User {
    id: ID!
    name: String!
    email: String!
  }
`;

const resolvers = {
  Query: {
    users: async () => {
      return await User.find();
    },
  },
};

const server = new ApolloServer({ typeDefs, resolvers });
const app = express();
server.applyMiddleware({ app });

app.listen({ port: 4000 }, () =>
  console.log(`Server ready at http://localhost:4000${server.graphqlPath}`)
);

Realitzant Operacions CRUD

Crear (Insertar)

  • Exemple SQL:
const resolvers = {
  Mutation: {
    addUser: async (_, { name, email }) => {
      const res = await pool.query(
        'INSERT INTO users (name, email) VALUES ($1, $2) RETURNING *',
        [name, email]
      );
      return res.rows[0];
    },
  },
};
  • Exemple NoSQL:
const resolvers = {
  Mutation: {
    addUser: async (_, { name, email }) => {
      const user = new User({ name, email });
      await user.save();
      return user;
    },
  },
};

Llegir (Seleccionar)

  • Exemple SQL:
const resolvers = {
  Query: {
    users: async () => {
      const res = await pool.query('SELECT * FROM users');
      return res.rows;
    },
  },
};
  • Exemple NoSQL:
const resolvers = {
  Query: {
    users: async () => {
      return await User.find();
    },
  },
};

Actualitzar

  • Exemple SQL:
const resolvers = {
  Mutation: {
    updateUser: async (_, { id, name, email }) => {
      const res = await pool.query(
        'UPDATE users SET name = $1, email = $2 WHERE id = $3 RETURNING *',
        [name, email, id]
      );
      return res.rows[0];
    },
  },
};
  • Exemple NoSQL:
const resolvers = {
  Mutation: {
    updateUser: async (_, { id, name, email }) => {
      return await User.findByIdAndUpdate(id, { name, email }, { new: true });
    },
  },
};

Esborrar

  • Exemple SQL:
const resolvers = {
  Mutation: {
    deleteUser: async (_, { id }) => {
      const res = await pool.query('DELETE FROM users WHERE id = $1 RETURNING *', [id]);
      return res.rows[0];
    },
  },
};
  • Exemple NoSQL:
const resolvers = {
  Mutation: {
    deleteUser: async (_, { id }) => {
      return await User.findByIdAndDelete(id);
    },
  },
};

Conclusió

Integrar GraphQL amb bases de dades et permet aprofitar el poder de les consultes i mutacions de GraphQL per interactuar amb les teves dades de manera eficient. Tant si utilitzes bases de dades SQL com NoSQL, els principis són els mateixos: establir una connexió, definir el teu esquema i resolvers, i realitzar operacions CRUD. Dominant aquestes habilitats, podràs construir aplicacions robustes i escalables que poden gestionar interaccions de dades complexes.
