El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, explorarem com implementar un frontend que interactua amb una API GraphQL. Cobrirem els conceptes bàsics de la configuració d'un projecte frontend, la realització de consultes i mutacions GraphQL, i la gestió de respostes. Al final d'aquesta secció, seràs capaç de crear una aplicació frontend senzilla que es comuniqui amb un servidor GraphQL.

Configurant el Projecte Frontend

Per començar, necessitem configurar un projecte frontend. Utilitzarem React per a aquest exemple, però els conceptes es poden aplicar a altres frameworks frontend també.

Prerequisits

  • Node.js i npm instal·lats
  • Coneixements bàsics de React

Passos per Configurar

  1. Crea un nou projecte React:

    npx create-react-app graphql-frontend
cd graphql-frontend

  2. Instal·la Apollo Client:

    Apollo Client és una biblioteca popular per interactuar amb APIs GraphQL en una aplicació React.

    npm install @apollo/client graphql

  3. Configura Apollo Client:

    Crea un nou fitxer src/apolloClient.js i configura Apollo Client.

    import { ApolloClient, InMemoryCache, ApolloProvider } from '@apollo/client';

const client = new ApolloClient({
  uri: 'http://localhost:4000/graphql', // Reemplaça amb la URL del teu servidor GraphQL
  cache: new InMemoryCache()
});

export default client;

  4. Embolcalla la teva aplicació amb ApolloProvider:

    Modifica src/index.js per incloure ApolloProvider.

    import React from 'react';
import ReactDOM from 'react-dom';
import './index.css';
import App from './App';
import client from './apolloClient';
import { ApolloProvider } from '@apollo/client';

ReactDOM.render(
  <ApolloProvider client={client}>
    <App />
  </ApolloProvider>,
  document.getElementById('root')
);

Realitzant Consultes GraphQL

Ara que hem configurat el nostre projecte, fem una consulta GraphQL per obtenir dades del servidor.

Exemple de Consulta

  1. Defineix la consulta:

    Crea un nou fitxer src/queries.js i defineix la teva consulta GraphQL.

    import { gql } from '@apollo/client';

export const GET_USERS = gql`
  query GetUsers {
    users {
      id
      name
      email
    }
  }
`;

  2. Utilitza la consulta en un component:

    Modifica src/App.js per utilitzar la consulta.

    import React from 'react';
import { useQuery } from '@apollo/client';
import { GET_USERS } from './queries';

function App() {
  const { loading, error, data } = useQuery(GET_USERS);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error :(</p>;

  return (
    <div>
      <h1>Users</h1>
      <ul>
        {data.users.map(user => (
          <li key={user.id}>
            {user.name} - {user.email}
          </li>
        ))}
      </ul>
    </div>
  );
}

export default App;

Realitzant Mutacions GraphQL

A més de les consultes, també podem realitzar mutacions per modificar dades en el servidor.

Exemple de Mutació

  1. Defineix la mutació:

    Afegeix una mutació a src/queries.js.

    export const ADD_USER = gql`
  mutation AddUser($name: String!, $email: String!) {
    addUser(name: $name, email: $email) {
      id
      name
      email
    }
  }
`;

  2. Utilitza la mutació en un component:

    Modifica src/App.js per incloure un formulari per afegir un nou usuari.

    import React, { useState } from 'react';
import { useQuery, useMutation } from '@apollo/client';
import { GET_USERS, ADD_USER } from './queries';

function App() {
  const { loading, error, data } = useQuery(GET_USERS);
  const [addUser] = useMutation(ADD_USER);
  const [name, setName] = useState('');
  const [email, setEmail] = useState('');

  const handleSubmit = async (e) => {
    e.preventDefault();
    await addUser({ variables: { name, email } });
    setName('');
    setEmail('');
  };

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error :(</p>;

  return (
    <div>
      <h1>Users</h1>
      <ul>
        {data.users.map(user => (
          <li key={user.id}>
            {user.name} - {user.email}
          </li>
        ))}
      </ul>
      <h2>Add User</h2>
      <form onSubmit={handleSubmit}>
        <input
          type="text"
          placeholder="Name"
          value={name}
          onChange={(e) => setName(e.target.value)}
        />
        <input
          type="email"
          placeholder="Email"
          value={email}
          onChange={(e) => setEmail(e.target.value)}
        />
        <button type="submit">Add User</button>
      </form>
    </div>
  );
}

export default App;

Gestionant Respostes i Errors

Gestionar respostes i errors és crucial per a una aplicació robusta.

Exemple

  1. Estats de Càrrega i Error:

    Assegura't de gestionar els estats de càrrega i error en els teus components.

    if (loading) return <p>Loading...</p>;
if (error) return <p>Error :(</p>;

  2. Actualitzacions Optimistes de la UI:

    Per a una millor experiència d'usuari, pots utilitzar actualitzacions optimistes de la UI quan realitzis mutacions.

    const [addUser] = useMutation(ADD_USER, {
  optimisticResponse: {
    addUser: {
      id: -1, // ID temporal
      name,
      email,
      __typename: 'User'
    }
  },
  update: (cache, { data: { addUser } }) => {
    const { users } = cache.readQuery({ query: GET_USERS });
    cache.writeQuery({
      query: GET_USERS,
      data: { users: users.concat([addUser]) }
    });
  }
});
