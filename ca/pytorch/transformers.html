Introducció als Transformers

Els transformers són un tipus d'arquitectura de model que ha revolucionat el camp del processament del llenguatge natural (NLP). Estan dissenyats per manejar dades seqüencials i han tingut molt èxit en tasques com la traducció de llenguatges, el resum de textos i més.

Conceptes Clau

  • Mecanisme d'Atenció: Permet al model centrar-se en diferents parts de la seqüència d'entrada quan fa prediccions.
  • Auto-Atenció: Un tipus de mecanisme d'atenció on el model atén a diferents posicions de la mateixa seqüència.
  • Arquitectura Encoder-Decoder: El model transformer original consta d'un encoder per processar l'entrada i un decoder per generar la sortida.

Configuració de l'Entorn PyTorch

Abans d'endinsar-nos en els transformers, assegureu-vos que teniu PyTorch instal·lat. Podeu instal·lar-lo utilitzant pip:

pip install torch torchvision

Model Transformer Bàsic

PyTorch proporciona un mòdul nn.Transformer que es pot utilitzar per construir models transformer.

Exemple: Model Transformer Simple

Comencem amb un exemple simple per entendre l'estructura bàsica d'un model transformer.

import torch
import torch.nn as nn
import torch.optim as optim

# Definir el model transformer
class SimpleTransformer(nn.Module):
    def __init__(self, input_dim, model_dim, num_heads, num_layers, output_dim):
        super(SimpleTransformer, self).__init__()
        self.transformer = nn.Transformer(d_model=model_dim, nhead=num_heads, num_encoder_layers=num_layers, num_decoder_layers=num_layers)
        self.fc = nn.Linear(model_dim, output_dim)
    
    def forward(self, src, tgt):
        transformer_output = self.transformer(src, tgt)
        output = self.fc(transformer_output)
        return output

# Hiperparàmetres
input_dim = 10
model_dim = 512
num_heads = 8
num_layers = 6
output_dim = 10

# Inicialitzar el model, la funció de pèrdua i l'optimitzador
model = SimpleTransformer(input_dim, model_dim, num_heads, num_layers, output_dim)
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# Dades fictícies
src = torch.rand((10, 32, input_dim))  # (longitud_seqüència, mida_batch, input_dim)
tgt = torch.rand((20, 32, input_dim))  # (longitud_seqüència, mida_batch, input_dim)
output = model(src, tgt)

Explicació

  • Inicialització del Model: Definim una classe SimpleTransformer que hereta de nn.Module.
  • Capa Transformer: Utilitzem nn.Transformer per crear l'arquitectura transformer.
  • Capa Completament Connectada: Una capa lineal per mapar la sortida del transformer a la dimensió de sortida desitjada.
  • Mètode Forward: Defineix com les dades d'entrada flueixen a través del model.
  • Dades Fictícies: Creem tensors aleatoris per simular seqüències d'entrada i objectiu.

Tècniques Avançades de Transformers

A mesura que avanceu, podeu explorar tècniques més avançades i personalitzacions.

Mecanisme d'Atenció Personalitzat

Podeu crear mecanismes d'atenció personalitzats per adaptar-vos millor a tasques específiques.

class CustomAttention(nn.Module):
    def __init__(self, model_dim, num_heads):
        super(CustomAttention, self).__init__()
        self.attention = nn.MultiheadAttention(embed_dim=model_dim, num_heads=num_heads)
    
    def forward(self, query, key, value):
        attn_output, _ = self.attention(query, key, value)
        return attn_output

# Exemple d'ús
custom_attention = CustomAttention(model_dim=512, num_heads=8)
query = torch.rand((10, 32, 512))  # (longitud_seqüència, mida_batch, model_dim)
key = torch.rand((10, 32, 512))
value = torch.rand((10, 32, 512))
attn_output = custom_attention(query, key, value)

Ajust Fi de Transformers Preentrenats

PyTorch també admet l'ajust fi de models transformer preentrenats de biblioteques com transformers de Hugging Face.

from transformers import BertModel, BertTokenizer

# Carregar el model BERT preentrenat i el tokenitzador
model_name = 'bert-base-uncased'
tokenizer = BertTokenizer.from_pretrained(model_name)
model = BertModel.from_pretrained(model_name)

# Tokenitzar l'entrada
input_text = "Hello, how are you?"
inputs = tokenizer(input_text, return_tensors='pt')

# Passada endavant
outputs = model(**inputs)

Explicació

  • Atenció Personalitzada: Definim una classe CustomAttention per crear un mecanisme d'atenció multi-cap personalitzat.
  • Models Preentrenats: Utilitzant la biblioteca transformers, carreguem un model BERT preentrenat i un tokenitzador, tokenitzem el text d'entrada i realitzem una passada endavant.

Conclusió

Els transformers són una eina poderosa en el NLP modern, i PyTorch proporciona mòduls flexibles per construir i personalitzar aquests models. Des de models transformers bàsics fins a tècniques avançades com mecanismes d'atenció personalitzats i l'ajust fi de models preentrenats, podeu utilitzar PyTorch per implementar solucions NLP d'avantguarda. Continueu experimentant amb diferents arquitectures i conjunts de dades per aprofundir en la vostra comprensió i millorar els vostres models.
