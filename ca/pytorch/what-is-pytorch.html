Introducció

PyTorch és una biblioteca de codi obert per a l'aprenentatge automàtic desenvolupada pel laboratori de recerca en IA de Facebook (FAIR). És àmpliament utilitzada per a aplicacions com el processament del llenguatge natural i la visió per computador. PyTorch proporciona una plataforma flexible i intuïtiva per construir models d'aprenentatge profund, cosa que la fa una elecció popular entre investigadors i desenvolupadors.

Característiques Clau de PyTorch

  • Gràfics de Computació Dinàmics: PyTorch utilitza gràfics de computació dinàmics, el que significa que el gràfic es construeix sobre la marxa a mesura que es realitzen les operacions. Això proporciona una major flexibilitat i facilitat de depuració.
  • Computació de Tensor: PyTorch proporciona una robusta biblioteca de matrius N-dimensionals (tensor), similar a NumPy, però amb acceleració per GPU.
  • Diferenciació Automàtica: El mòdul autograd de PyTorch calcula automàticament els gradients, cosa essencial per entrenar xarxes neuronals.
  • Ric Ecosistema: PyTorch té un ric ecosistema d'eines i biblioteques per a diverses tasques d'aprenentatge automàtic, incloent TorchVision per a visió per computador, TorchText per a processament del llenguatge natural, i més.
  • Comunitat i Suport: PyTorch té una gran i activa comunitat, amb extensa documentació i nombrosos tutorials disponibles.

Instal·lació de PyTorch

Per començar amb PyTorch, cal instal·lar-lo. Pots instal·lar PyTorch utilitzant pip o conda. Aquí tens un exemple utilitzant pip:

pip install torch torchvision torchaudio

Per als usuaris de conda:

conda install pytorch torchvision torchaudio -c pytorch

Conceptes Bàsics

Tensors

Els tensors són els blocs fonamentals en PyTorch. Són similars a les matrius de NumPy però també es poden utilitzar en una GPU per accelerar el càlcul.

Creació de Tensors

Aquí tens alguns exemples de creació de tensors en PyTorch:

import torch

# Crear un tensor a partir d'una llista
tensor_from_list = torch.tensor([1, 2, 3, 4])
print(tensor_from_list)

# Crear un tensor amb valors aleatoris
random_tensor = torch.rand(3, 3)
print(random_tensor)

# Crear un tensor ple de zeros
zeros_tensor = torch.zeros(2, 2)
print(zeros_tensor)

Diferenciació Automàtica

El mòdul autograd de PyTorch proporciona diferenciació automàtica per a totes les operacions sobre tensors. Això és particularment útil per a la retropropagació en xarxes neuronals.

Exemple de Diferenciació Automàtica

import torch

# Crear un tensor i habilitar el càlcul de gradients
x = torch.tensor(2.0, requires_grad=True)

# Realitzar algunes operacions
y = x ** 2
z = y + 3

# Calcular els gradients
z.backward()

# Imprimir el gradient de x
print(x.grad)  # Sortida: tensor(4.0)

Construcció d'una Xarxa Neuronal Simple

Construïm una xarxa neuronal simple utilitzant PyTorch per entendre com tot es combina.

Exemple: Model de Regressió Lineal

import torch
import torch.nn as nn
import torch.optim as optim

# Definir el model
class LinearRegressionModel(nn.Module):
    def __init__(self):
        super(LinearRegressionModel, self).__init__()
        self.linear = nn.Linear(1, 1)

    def forward(self, x):
        return self.linear(x)

# Crear el model, definir la funció de pèrdua i l'optimitzador
model = LinearRegressionModel()
criterion = nn.MSELoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# Dades d'entrenament
x_train = torch.tensor([[1.0], [2.0], [3.0], [4.0]])
y_train = torch.tensor([[2.0], [4.0], [6.0], [8.0]])

# Bucle d'entrenament
for epoch in range(100):
    # Passada endavant
    outputs = model(x_train)
    loss = criterion(outputs, y_train)

    # Passada enrere i optimització
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

    if (epoch+1) % 10 == 0:
        print(f'Epoch [{epoch+1}/100], Loss: {loss.item():.4f}')

# Provar el model
with torch.no_grad():
    predicted = model(torch.tensor([[5.0]]))
    print(f'Prediction for input 5.0: {predicted.item():.4f}')

Conclusió

PyTorch és un marc d'aprenentatge profund potent i flexible que és fàcil d'utilitzar i entendre. El seu gràfic de computació dinàmic, capacitats de computació de tensors i diferenciació automàtica el fan una excel·lent elecció tant per a principiants com per a usuaris avançats. Entenent els conceptes bàsics de PyTorch, com els tensors i la diferenciació automàtica, pots començar a construir i entrenar les teves pròpies xarxes neuronals. A mesura que avancis en aquest curs, aprendràs tècniques i aplicacions més avançades de PyTorch.
