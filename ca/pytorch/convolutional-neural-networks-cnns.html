Introducció a les CNNs

Les Xarxes Neuronals Convolucionals (CNNs) són una classe de xarxes neuronals profundes que són particularment efectives per analitzar dades visuals. S'utilitzen àmpliament en el reconeixement d'imatges i vídeos, sistemes de recomanació i processament del llenguatge natural.

Conceptes Clau

  • Capes Convolucionals: Capes que apliquen una operació de convolució a l'entrada, passant el resultat a la següent capa.
  • Capes de Pooling: Capes que redueixen les dimensions espacials (amplada i alçada) del volum d'entrada.
  • Capes Totalment Connectades: Capes on cada neurona està connectada a cada neurona de la capa anterior.
  • Funcions d'Activació: Funcions aplicades a cada sortida de neurona per introduir no linealitat (per exemple, ReLU).
  • Dropout: Una tècnica per prevenir el sobreajustament establint aleatòriament una fracció d'unitats d'entrada a zero en cada actualització durant l'entrenament.

Construint una CNN Simple

Comencem construint una CNN simple per a la classificació d'imatges utilitzant el conjunt de dades MNIST.

Pas 1: Importar Llibreries

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Pas 2: Carregar i Preprocessar Dades

transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=32, shuffle=True)

testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=32, shuffle=False)

Pas 3: Definir el Model CNN

class SimpleCNN(nn.Module):
    def __init__(self):
        super(SimpleCNN, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, 1)
        self.conv2 = nn.Conv2d(32, 64, 3, 1)
        self.fc1 = nn.Linear(9216, 128)
        self.fc2 = nn.Linear(128, 10)
        self.dropout1 = nn.Dropout(0.25)
        self.dropout2 = nn.Dropout(0.5)

    def forward(self, x):
        x = self.conv1(x)
        x = nn.ReLU()(x)
        x = self.conv2(x)
        x = nn.ReLU()(x)
        x = nn.MaxPool2d(2)(x)
        x = self.dropout1(x)
        x = torch.flatten(x, 1)
        x = self.fc1(x)
        x = nn.ReLU()(x)
        x = self.dropout2(x)
        x = self.fc2(x)
        return nn.LogSoftmax(dim=1)(x)

Pas 4: Entrenar el Model

model = SimpleCNN()
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

for epoch in range(10):  # bucle sobre el conjunt de dades diverses vegades
    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        inputs, labels = data
        optimizer.zero_grad()
        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        running_loss += loss.item()
        if i % 100 == 99:    # imprimir cada 100 mini-lots
            print(f'[Epoch {epoch + 1}, Batch {i + 1}] loss: {running_loss / 100:.3f}')
            running_loss = 0.0

print('Finished Training')

Pas 5: Avaluar el Model

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = model(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Accuracy of the network on the 10000 test images: {100 * correct / total:.2f}%')

Temes Avançats en CNNs

Un cop estigueu còmodes amb la construcció d'una CNN simple, podeu explorar temes més avançats.

Transfer Learning

El transfer learning implica utilitzar un model preentrenat i ajustar-lo per a una tasca específica.

import torchvision.models as models

# Carregar un model ResNet preentrenat
model = models.resnet18(pretrained=True)

# Congelar totes les capes
for param in model.parameters():
    param.requires_grad = False

# Reemplaçar la capa totalment connectada final
model.fc = nn.Linear(model.fc.in_features, 10)

# Ara només s'actualitzaran els paràmetres de la capa final
optimizer = optim.Adam(model.fc.parameters(), lr=0.001)

Augment de Dades

Les tècniques d'augment de dades poden ajudar a millorar la robustesa del vostre model augmentant artificialment la mida del vostre conjunt de dades d'entrenament.

transform = transforms.Compose([
    transforms.RandomHorizontalFlip(),
    transforms.RandomRotation(10),
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

Ajust de Hiperparàmetres
