Configurar el teu entorn és el primer pas per començar a treballar amb PyTorch. Aquesta secció et guiarà a través del procés d'instal·lació de PyTorch i la configuració del teu entorn de desenvolupament per assegurar una experiència fluida.

Instal·lant PyTorch

Per començar amb PyTorch, necessites instal·lar-lo a la teva màquina. PyTorch es pot instal·lar utilitzant pip o conda, depenent de la teva preferència i del gestor de paquets que utilitzis.

Utilitzant pip

  1. Obre el teu terminal o símbol del sistema.

  2. Executa la següent comanda:

    pip install torch torchvision torchaudio
    • torch: La biblioteca principal de PyTorch.
    • torchvision: Conté conjunts de dades populars, arquitectures de models i transformacions d'imatges per a visió per computador.
    • torchaudio: Proporciona I/O d'àudio i transformacions.

Utilitzant conda

  1. Obre el teu terminal o símbol del sistema.

  2. Executa la següent comanda:

    conda install pytorch torchvision torchaudio -c pytorch
    • -c pytorch: Especifica el canal de PyTorch per assegurar que obtens la versió més recent.

Verificant la Instal·lació

Després d'instal·lar PyTorch, és important verificar que la instal·lació ha estat exitosa.

  1. Obre el teu intèrpret de Python o crea un nou script de Python.

  2. Executa el següent codi:

    import torch
print(torch.__version__)
    • Això hauria d'imprimir la versió de PyTorch que tens instal·lada.

  3. Comprova el suport per CUDA (si tens una GPU compatible):

    print(torch.cuda.is_available())
    • Això hauria de retornar True si CUDA està disponible i configurat correctament.

Configurant el teu IDE

Escollir l'Entorn de Desenvolupament Integrat (IDE) adequat pot millorar significativament la teva productivitat. Aquí tens algunes opcions populars:

  • PyCharm: Un IDE potent amb moltes característiques adaptades per al desenvolupament en Python.
  • Visual Studio Code (VS Code): Un editor lleuger, altament personalitzable amb un ric ecosistema d'extensions.
  • Jupyter Notebook: Un entorn interactiu ideal per a l'anàlisi de dades i l'experimentació.

Configurant PyCharm

  1. Instal·la PyCharm des del lloc web oficial.

  2. Crea un nou projecte i configura un entorn virtual:

    • Ves a File > New Project.
    • Tria una ubicació i selecciona New environment using: Virtualenv.
    • Fes clic a Create.

  3. Instal·la PyTorch a l'entorn virtual:

    • Obre el terminal dins de PyCharm.
    • Executa la comanda d'instal·lació de pip esmentada anteriorment.

Configurant Visual Studio Code

  1. Instal·la VS Code des del lloc web oficial.

  2. Instal·la l'extensió de Python:

    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca Python i instal·la l'extensió proporcionada per Microsoft.

  3. Configura un entorn virtual:

    • Obre el terminal a VS Code.

    • Executa les següents comandes:

      python -m venv venv
source venv/bin/activate  # A Windows utilitza `venv\Scripts\activate`

  4. Instal·la PyTorch a l'entorn virtual:

    • Executa la comanda d'instal·lació de pip esmentada anteriorment.

Utilitzant Jupyter Notebook

  1. Instal·la Jupyter Notebook:

    pip install notebook

  2. Inicia Jupyter Notebook:

    jupyter notebook

  3. Crea un nou quadern i instal·la PyTorch:

    • Obre un nou quadern.

    • Executa el següent codi en una cel·la:

      !pip install torch torchvision torchaudio

Conclusió

Configurar correctament el teu entorn és crucial per a una experiència de desenvolupament fluida amb PyTorch. Tant si tries utilitzar pip o conda per a la instal·lació, com si prefereixes PyCharm, VS Code o Jupyter Notebook, la clau és assegurar que PyTorch està instal·lat i configurat correctament. Seguint els passos descrits en aquesta secció, hauries d'estar preparat per començar el teu viatge amb PyTorch.
