Introducció

En aquesta secció, cobrirem les operacions fonamentals que podeu realitzar amb tensors utilitzant PyTorch. Els tensors són l'estructura de dades principal a PyTorch, similar als arrays en NumPy però amb capacitats addicionals per a l'acceleració GPU. Entendre aquestes operacions bàsiques és crucial per construir models d'aprenentatge automàtic més complexos.

Creació de tensors

Abans de realitzar operacions, heu de saber com crear tensors. PyTorch proporciona diverses maneres de crear tensors:

import torch

# Creació d'un tensor a partir d'una llista
tensor_from_list = torch.tensor([1, 2, 3, 4])

# Creació d'un tensor amb valors aleatoris
random_tensor = torch.rand(3, 3)

# Creació d'un tensor ple de zeros
zeros_tensor = torch.zeros(2, 2)

# Creació d'un tensor ple d'uns
ones_tensor = torch.ones(2, 2)
  • torch.tensor(data): Crea un tensor a partir de dades (llista, tupla, etc.).
  • torch.rand(size): Crea un tensor amb valors aleatoris distribuïts uniformement entre 0 i 1.
  • torch.zeros(size): Crea un tensor ple de zeros.
  • torch.ones(size): Crea un tensor ple d'uns.

Operacions bàsiques amb tensors

Operacions aritmètiques

Podeu realitzar operacions aritmètiques bàsiques amb tensors igual que ho faríeu amb arrays de NumPy.

a = torch.tensor([1, 2, 3])
b = torch.tensor([4, 5, 6])

# Suma
c = a + b  # o torch.add(a, b)

# Resta
d = a - b  # o torch.sub(a, b)

# Multiplicació
e = a * b  # o torch.mul(a, b)

# Divisió
f = a / b  # o torch.div(a, b)

Operacions amb matrius

Les operacions amb matrius són essencials per a molts algorismes d'aprenentatge automàtic.

# Multiplicació de matrius
matrix1 = torch.tensor([[1, 2], [3, 4]])
matrix2 = torch.tensor([[5, 6], [7, 8]])

result = torch.mm(matrix1, matrix2)  # o matrix1 @ matrix2

Operacions d'agregació

Les operacions d'agregació ajuden a resumir dades.

tensor = torch.tensor([1, 2, 3, 4, 5])

# Suma
sum_result = torch.sum(tensor)

# Mitjana
mean_result = torch.mean(tensor.float())  # Convertir a float per al càlcul de la mitjana

# Màxim
max_result = torch.max(tensor)

# Mínim
min_result = torch.min(tensor)

Reestructuració de tensors

Reestructurar tensors és sovint necessari per preparar dades per als models.

tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])

# Reestructurar a un tensor de 3x2
reshaped_tensor = tensor.view(3, 2)

# Aplanar el tensor
flattened_tensor = tensor.view(-1)

Tallar i indexar

Igual que els arrays de NumPy, podeu tallar i indexar tensors.

tensor = torch.tensor([[1, 2, 3], [4, 5, 6]])

# Indexar
element = tensor[0, 1]  # Accedir a l'element a la fila 0, columna 1

# Tallar
slice_tensor = tensor[:, 1]  # Accedir a totes les files, columna 1

Comparació d'operacions entre PyTorch i NumPy

| Operació | Sintaxi PyTorch | Sintaxi NumPy | |-----------------|-----------------------|-----------------------| | Suma | a + b o torch.add(a, b) | a + b o np.add(a, b) | | Resta | a - b o torch.sub(a, b) | a - b o np.subtract(a, b) | | Multiplicació | a * b o torch.mul(a, b) | a * b o np.multiply(a, b) | | Divisió | a / b o torch.div(a, b) | a / b o np.divide(a, b) | | Multiplicació de matrius | torch.mm(a, b) o a @ b | np.dot(a, b) o a @ b | | Reestructurar | a.view(new_shape) | a.reshape(new_shape) | | Suma | torch.sum(a) | np.sum(a) | | Mitjana | torch.mean(a) | np.mean(a) |

Conclusió

En aquesta secció, hem cobert les bases de les operacions amb tensors a PyTorch, incloent la creació de tensors, la realització d'operacions aritmètiques i amb matrius, l'agregació, la reestructuració i el tall. Aquestes operacions formen la base per a temes més avançats a PyTorch i són essencials per a qualsevol tasca d'aprenentatge automàtic o profund. Assegureu-vos de practicar aquestes operacions per familiaritzar-vos amb la manipulació de tensors, ja que són una part crítica del treball amb PyTorch.
