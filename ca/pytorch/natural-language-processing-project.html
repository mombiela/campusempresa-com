Introducció al Processament del Llenguatge Natural (PLN)

El Processament del Llenguatge Natural (PLN) és un camp de la intel·ligència artificial que se centra en la interacció entre els ordinadors i els humans a través del llenguatge natural. L'objectiu és permetre que els ordinadors entenguin, interpretin i generin el llenguatge humà d'una manera que sigui significativa i útil.

Conceptes Clau en PLN

  • Tokenització: Dividir el text en paraules o frases individuals.
  • Paraules Buida: Paraules comunes que normalment es filtren abans del processament.
  • Arrelament i Lematització: Reduir les paraules a la seva forma base o arrel.
  • Bossa de Paraules: Representar les dades de text com una col·lecció de freqüències de paraules.
  • TF-IDF: Freqüència de Termes-Inversa Freqüència de Documents, una mesura estadística utilitzada per avaluar la importància d'una paraula en un document.
  • Embutits de Paraules: Representar les paraules en un espai vectorial continu on les paraules semànticament similars estan més a prop.

Preprocessament de Dades

Tokenització i Eliminació de Paraules Buida

La tokenització és el procés de dividir el text en paraules o tokens individuals. Les paraules buida són paraules comunes que normalment es treuen per centrar-se en les paraules més significatives.

import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

nltk.download('punkt')
nltk.download('stopwords')

text = "Natural Language Processing with PyTorch is fun and educational."
tokens = word_tokenize(text)
stop_words = set(stopwords.words('english'))
filtered_tokens = [word for word in tokens if word.lower() not in stop_words]

print(filtered_tokens)

Arrelament i Lematització

L'arrelament i la lematització són tècniques utilitzades per reduir les paraules a la seva forma base o arrel.

from nltk.stem import PorterStemmer, WordNetLemmatizer

nltk.download('wordnet')

stemmer = PorterStemmer()
lemmatizer = WordNetLemmatizer()

stemmed_words = [stemmer.stem(word) for word in filtered_tokens]
lemmatized_words = [lemmatizer.lemmatize(word) for word in filtered_tokens]

print("Stemmed Words:", stemmed_words)
print("Lemmatized Words:", lemmatized_words)

Extracció de Característiques

Bossa de Paraules

El model de Bossa de Paraules representa les dades de text com una col·lecció de freqüències de paraules.

from sklearn.feature_extraction.text import CountVectorizer

corpus = [
    'Natural Language Processing with PyTorch is fun and educational.',
    'PyTorch makes building neural networks easy.',
    'NLP is a fascinating field of study.'
]

vectorizer = CountVectorizer()
X = vectorizer.fit_transform(corpus)

print(vectorizer.get_feature_names_out())
print(X.toarray())

TF-IDF

TF-IDF és una mesura estadística utilitzada per avaluar la importància d'una paraula en un document.

from sklearn.feature_extraction.text import TfidfVectorizer

tfidf_vectorizer = TfidfVectorizer()
X_tfidf = tfidf_vectorizer.fit_transform(corpus)

print(tfidf_vectorizer.get_feature_names_out())
print(X_tfidf.toarray())

Construcció d'un Model de PLN amb PyTorch

Preparació del Conjunt de Dades

Per a aquest projecte, utilitzarem un conjunt de dades senzill per a la classificació de text.

import torch
from torch.utils.data import Dataset, DataLoader

class TextDataset(Dataset):
    def __init__(self, texts, labels, vectorizer):
        self.texts = texts
        self.labels = labels
        self.vectorizer = vectorizer
        self.vectorized_texts = self.vectorizer.fit_transform(self.texts).toarray()
        
    def __len__(self):
        return len(self.texts)
    
    def __getitem__(self, idx):
        text_vector = torch.tensor(self.vectorized_texts[idx], dtype=torch.float32)
        label = torch.tensor(self.labels[idx], dtype=torch.long)
        return text_vector, label

texts = [
    'Natural Language Processing with PyTorch is fun and educational.',
    'PyTorch makes building neural networks easy.',
    'NLP is a fascinating field of study.'
]
labels = [0, 1, 0]  # Etiquetes d'exemple

dataset = TextDataset(texts, labels, vectorizer)
dataloader = DataLoader(dataset, batch_size=2, shuffle=True)

Definició del Model

Definirem una xarxa neuronal feedforward senzilla per a la classificació de text.

import torch.nn as nn

class TextClassifier(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(TextClassifier, self).__init__()
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(hidden_dim, output_dim)
    
    def forward(self, x):
        out = self.fc1(x)
        out = self.relu(out)
        out = self.fc2(out)
        return out

input_dim = len(vectorizer.get_feature_names_out())
hidden_dim = 10
output_dim = 2  # Nombre de classes

model = TextClassifier(input_dim, hidden_dim, output_dim)

Entrenament del Model

Entrenarem el model utilitzant un bucle d'entrenament senzill.

import torch.optim as optim

criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

num_epochs = 5

for epoch in range(num_epochs):
    for texts, labels in dataloader:
        outputs = model(texts)
        loss = criterion(outputs, labels)
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    
    print(f'Epoch [{epoch+1}/{num_epochs}], Loss: {loss.item():.4f}')
