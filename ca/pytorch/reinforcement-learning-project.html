Introducció a l'Aprenentatge per Reforç

L'Aprenentatge per Reforç (RL) és un tipus d'aprenentatge automàtic on un agent aprèn a prendre decisions realitzant accions en un entorn per maximitzar la recompensa acumulada.

Conceptes Clau

  • Agent: L'aprenent o el que pren decisions.
  • Entorn: El sistema extern amb el qual l'agent interactua.
  • Estat: Una representació de la situació actual de l'agent.
  • Acció: Un conjunt de tots els moviments possibles que l'agent pot fer.
  • Recompensa: Retroalimentació de l'entorn per avaluar l'acció.
  • Política: Una estratègia utilitzada per l'agent per decidir accions basades en l'estat.
  • Funció de Valor: Una predicció de recompenses futures.

Configuració de l'Entorn

Abans d'endinsar-nos en RL, assegureu-vos de tenir PyTorch instal·lat. Podeu instal·lar PyTorch utilitzant la següent comanda:

pip install torch torchvision

Construcció d'un Model RL Simple amb PyTorch

En aquesta secció, construirem un model RL simple utilitzant PyTorch. Utilitzarem l'entorn OpenAI Gym per a aquest propòsit.

Pas 1: Instal·lació d'OpenAI Gym

Primer, instal·leu la biblioteca OpenAI Gym:

pip install gym

Pas 2: Creació de l'Entorn

Creem un entorn simple utilitzant OpenAI Gym:

import gym

env = gym.make('CartPole-v1')
state = env.reset()
print(state)

Pas 3: Definició de la Xarxa Neuronal

Definirem una xarxa neuronal simple per aproximar la funció de valor Q.

import torch
import torch.nn as nn
import torch.optim as optim

class QNetwork(nn.Module):
    def __init__(self, state_size, action_size):
        super(QNetwork, self).__init__()
        self.fc1 = nn.Linear(state_size, 24)
        self.fc2 = nn.Linear(24, 24)
        self.fc3 = nn.Linear(24, action_size)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = self.fc3(x)
        return x

Pas 4: Entrenament de l'Agent

Ara entrenarem l'agent utilitzant l'algoritme Q-learning.

import numpy as np

def train(env, q_network, episodes=1000, gamma=0.99, epsilon=0.1, lr=0.001):
    optimizer = optim.Adam(q_network.parameters(), lr=lr)
    criterion = nn.MSELoss()

    for episode in range(episodes):
        state = env.reset()
        done = False
        total_reward = 0

        while not done:
            state_tensor = torch.FloatTensor(state).unsqueeze(0)
            q_values = q_network(state_tensor)
            if np.random.rand() < epsilon:
                action = env.action_space.sample()
            else:
                action = torch.argmax(q_values).item()

            next_state, reward, done, _ = env.step(action)
            total_reward += reward

            next_state_tensor = torch.FloatTensor(next_state).unsqueeze(0)
            next_q_values = q_network(next_state_tensor)
            max_next_q_value = torch.max(next_q_values).item()

            target_q_value = reward + (gamma * max_next_q_value * (1 - done))
            target_q_values = q_values.clone()
            target_q_values[0][action] = target_q_value

            loss = criterion(q_values, target_q_values)
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()

            state = next_state

        print(f"Episodi {episode + 1}: Recompensa Total: {total_reward}")

# Inicialitzar i entrenar la xarxa Q
state_size = env.observation_space.shape[0]
action_size = env.action_space.n
q_network = QNetwork(state_size, action_size)
train(env, q_network)
