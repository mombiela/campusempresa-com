En aquesta secció, explorarem com crear datasets personalitzats i dataloaders en PyTorch. Això és crucial per gestionar dades que no s'ajusten als datasets estàndard proporcionats per PyTorch. Al final d'aquesta secció, seràs capaç de crear els teus propis datasets i carregar-los eficientment utilitzant el DataLoader de PyTorch.

Conceptes clau

  • Classe Dataset: Una classe personalitzada que hereta de torch.utils.data.Dataset i implementa els mètodes __len__ i __getitem__.
  • Classe DataLoader: Una classe que proporciona un iterable sobre un dataset amb suport per a batching, shuffling i càrrega de dades multi-procés.

Creació d'un Dataset personalitzat

Per crear un dataset personalitzat, necessites subclassificar torch.utils.data.Dataset i sobreescriure dos mètodes:

  1. __len__: Retorna la mida del dataset.
  2. __getitem__: Recupera una mostra del dataset a l'índex donat.

Exemple: Dataset d'imatges personalitzat

Creem un dataset personalitzat per carregar imatges i les seves etiquetes corresponents.

import os
from PIL import Image
import torch
from torch.utils.data import Dataset

class CustomImageDataset(Dataset):
    def __init__(self, img_dir, transform=None):
        self.img_dir = img_dir
        self.transform = transform
        self.img_labels = [(img_name, int(img_name.split('_')[1])) per img_name in os.listdir(img_dir)]

    def __len__(self):
        return len(self.img_labels)

    def __getitem__(self, idx):
        img_name, label = self.img_labels[idx]
        img_path = os.path.join(self.img_dir, img_name)
        image = Image.open(img_path)
        
        if self.transform:
            image = self.transform(image)
        
        return image, label
  • Explicació:
    • __init__: Inicialitza el dataset amb el directori d'imatges i una transformació opcional.
    • __len__: Retorna el nombre d'imatges al directori.
    • __getitem__: Carrega una imatge i la seva etiqueta basant-se en l'índex.

Ús del DataLoader

La classe DataLoader proporciona una manera eficient d'iterar sobre el dataset. Suporta batching, shuffling i càrrega de dades en paral·lel.

Exemple: Ús del DataLoader amb Dataset personalitzat

from torchvision import transforms
from torch.utils.data import DataLoader

# Defineix una transformació per convertir imatges a tensors
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.ToTensor()
])

# Crea una instància del dataset personalitzat
dataset = CustomImageDataset(img_dir='path/to/images', transform=transform)

# Crea un DataLoader
dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

# Itera sobre el DataLoader
for images, labels in dataloader:
    print(images.size(), labels)
  • Explicació:
    • transform: Una sèrie de transformacions aplicades a cada imatge.
    • DataLoader: Crea un iterable sobre el dataset amb batching, shuffling i càrrega en paral·lel.
    • for images, labels in dataloader: Itera sobre el dataset en batches.

Ús avançat

Gestió de diferents tipus de dades

Pots crear datasets personalitzats per a diversos tipus de dades com text, àudio o fins i tot dades multi-modals complexes.

Exemple: Dataset de text personalitzat

class CustomTextDataset(Dataset):
    def __init__(self, text_file, transform=None):
        with open(text_file, 'r') as file:
            self.text_data = file.readlines()
        self.transform = transform

    def __len__(self):
        return len(self.text_data)

    def __getitem__(self, idx):
        text = self.text_data[idx].strip()
        
        if self.transform:
            text = self.transform(text)
        
        return text
  • Explicació:
    • __init__: Llegeix dades de text des d'un fitxer.
    • __len__: Retorna el nombre de línies al fitxer de text.
    • __getitem__: Recupera una línia de text i aplica qualsevol transformació.

Conclusió

Crear datasets i dataloaders personalitzats en PyTorch et permet gestionar una àmplia varietat de tipus i formats de dades. Subclassificant torch.utils.data.Dataset i utilitzant DataLoader, pots carregar i preprocessar les teves dades de manera eficient, fent-les llestes per entrenar models d'aprenentatge automàtic.

  • Punts clau:
    • Subclassifica torch.utils.data.Dataset per crear datasets personalitzats.
    • Implementa els mètodes __len__ i __getitem__.
    • Utilitza DataLoader per a una càrrega de dades eficient amb suport per a batching, shuffling i processament en paral·lel.

Amb aquestes eines, pots gestionar qualsevol dataset, per complex o únic que sigui, i preparar-lo per a les teves tasques d'aprenentatge automàtic.
