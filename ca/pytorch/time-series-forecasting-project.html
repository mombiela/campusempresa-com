Introducció

La predicció de sèries temporals és un aspecte crucial de moltes aplicacions del món real, com la predicció del mercat de valors, la predicció meteorològica i la predicció de la demanda. Aquest projecte et guiarà a través del procés de construcció d'un model de predicció de sèries temporals utilitzant PyTorch, començant pels conceptes bàsics i progressant cap a tècniques avançades.

Prerequisits

Abans de començar aquest projecte, hauries de tenir una comprensió bàsica de:

  • Programació en Python
  • Fonaments de PyTorch
  • Conceptes bàsics d'aprenentatge automàtic

Visió General del Projecte

En aquest projecte, farem:

  1. Entendre els conceptes bàsics de les dades de sèries temporals.
  2. Preprocessar les dades per a la modelització.
  3. Construir i entrenar un model RNN simple.
  4. Avaluar el rendiment del model.
  5. Implementar tècniques avançades per millorar el model.

1. Entendre les Dades de Sèries Temporals

Les dades de sèries temporals són una seqüència de punts de dades recollits o registrats a intervals de temps específics.

  • Conceptes Clau:
    • Tendència: El moviment a llarg termini en la sèrie temporal.
    • Estacionalitat: El cicle repetitiu a curt termini en la sèrie.
    • Soroll: Variacions aleatòries en les dades.

2. Preprocessament de Dades

El preprocessament de dades és un pas crític en la predicció de sèries temporals. Implica netejar les dades, gestionar els valors perduts i normalitzar les dades.

3. Construir un Model RNN Simple

Les Xarxes Neuronals Recurrents (RNNs) són adequades per a la predicció de sèries temporals a causa de la seva capacitat de retenir informació de passos de temps anteriors.

  • Passos:
    1. Definir el Model: 
      import torch
import torch.nn as nn

class SimpleRNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(SimpleRNN, self).__init__()
        self.rnn = nn.RNN(input_size, hidden_size, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)

    def forward(self, x):
        out, _ = self.rnn(x)
        out = self.fc(out[:, -1, :])
        return out
    2. Entrenar el Model: 
      model = SimpleRNN(input_size=1, hidden_size=50, output_size=1)
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

5. Tècniques Avançades

Per millorar el model, podem implementar tècniques avançades com les xarxes LSTM (Long Short-Term Memory) i l'ajust de hiperparàmetres.

  • Implementació de LSTM:

    class LSTM(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(LSTM, self).__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)

    def forward(self, x):

        out = self.fc(out[:, -1, :])
