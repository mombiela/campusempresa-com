Introducció

En aquesta secció, explorarem els conceptes de funcions de pèrdua i optimització en PyTorch. Entendre aquests conceptes és crucial per entrenar xarxes neuronals de manera efectiva. Cobrirem:

  • Què són les funcions de pèrdua i per què són importants.
  • Funcions de pèrdua comunament utilitzades en PyTorch.
  • El paper de l'optimització en l'entrenament de xarxes neuronals.
  • Com implementar algorismes d'optimització en PyTorch.

Funcions de pèrdua

Què són les Funcions de Pèrdua?

Les funcions de pèrdua, també conegudes com a funcions de cost o funcions objectiu, mesuren com de bé les prediccions d'un model coincideixen amb els valors objectiu reals. L'objectiu d'entrenar una xarxa neuronal és minimitzar la funció de pèrdua.

Funcions de Pèrdua Comunament Utilitzades en PyTorch

PyTorch proporciona diverses funcions de pèrdua integrades que s'utilitzen comunament en diverses tasques d'aprenentatge automàtic. Aquí en teniu algunes:

  • Error Quadràtic Mitjà (MSELoss): Utilitzat per a tasques de regressió.
  • Pèrdua d'Entropia Creuada (CrossEntropyLoss): Utilitzat per a tasques de classificació.
  • Pèrdua d'Entropia Creuada Binària (BCELoss): Utilitzat per a tasques de classificació binària.

Implementació de Funcions de Pèrdua en PyTorch

Vegem com implementar algunes d'aquestes funcions de pèrdua en PyTorch.

Error Quadràtic Mitjà (MSELoss)

import torch
import torch.nn as nn

# Dades d'exemple
predictions = torch.tensor([2.5, 0.0, 2.1, 7.8])
targets = torch.tensor([3.0, -0.5, 2.0, 7.5])

# Inicialitzar la funció de pèrdua
mse_loss = nn.MSELoss()

# Calcular la pèrdua
loss = mse_loss(predictions, targets)

Pèrdua d'Entropia Creuada (CrossEntropyLoss)

import torch

cross_entropy_loss = nn.CrossEntropyLoss()

Optimització

Què és l'Optimització?

L'optimització es refereix al procés d'ajustar els paràmetres del model per minimitzar la funció de pèrdua. Això es fa típicament utilitzant el descens de gradient i les seves variants.

Algorismes d'Optimització Comunament Utilitzats en PyTorch

PyTorch proporciona diversos algorismes d'optimització, incloent:

  • Descens de Gradient Estocàstic (SGD)
  • Adam
  • RMSprop

Implementació d'Algorismes d'Optimització en PyTorch

Vegem com implementar alguns d'aquests algorismes d'optimització en PyTorch.

Descens de Gradient Estocàstic (SGD)

import torch.optim as optim

# Paràmetres del model d'exemple
model_params = [torch.tensor([1.0, 2.0], requires_grad=True)]

# Inicialitzar l'optimitzador
optimizer = optim.SGD(model_params, lr=0.01)

# Bucle d'entrenament d'exemple
for epoch in range(100):
    # Netejar els gradients
    optimizer.zero_grad()
    
    # Passada endavant (càlcul de pèrdua fictícia)
    loss = (model_params[0] - torch.tensor([1.5, 2.5])).pow(2).sum()
    
    # Passada enrere
    loss.backward()
    
    # Actualitzar els paràmetres
    optimizer.step()
    
    print(f'Epoch {epoch+1}, Loss: {loss.item()}')

Adam

import torch.optim as optim

# Paràmetres del model d'exemple
model_params = [torch.tensor([1.0, 2.0], requires_grad=True)]

# Inicialitzar l'optimitzador
optimizer = optim.Adam(model_params, lr=0.01)

# Bucle d'entrenament d'exemple
for epoch in range(100):
    # Netejar els gradients
    optimizer.zero_grad()
    
    # Passada endavant (càlcul de pèrdua fictícia)
    loss = (model_params[0] - torch.tensor([1.5, 2.5])).pow(2).sum()
    
    # Passada enrere
    loss.backward()
    
    # Actualitzar els paràmetres
    optimizer.step()
    
    print(f'Epoch {epoch+1}, Loss: {loss.item()}')
