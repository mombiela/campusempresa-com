En aquesta secció, explorarem com crear capes i mòduls personalitzats en PyTorch. Aquesta és una habilitat crucial per desenvolupar arquitectures complexes de xarxes neuronals adaptades a tasques específiques. Començarem amb els conceptes bàsics i gradualment passarem a conceptes més avançats.

Introducció a Capes i Mòduls Personalitzats

Crear capes i mòduls personalitzats et permet estendre la funcionalitat de PyTorch i adaptar-la a les teves necessitats específiques. Això pot ser particularment útil quan les capes integrades no compleixen els teus requisits.

  • Capes Personalitzades: Aquests són blocs de construcció individuals d'una xarxa neuronal.
  • Mòduls Personalitzats: Aquests són estructures més complexes que poden contenir múltiples capes i altres mòduls.

Creació d'una Capa Personalitzada

Per crear una capa personalitzada en PyTorch, necessites subclassejar torch.nn.Module i definir els mètodes __init__ i forward.

Exemple: Capa Lineal Personalitzada

Creem una capa lineal personalitzada que realitza una transformació lineal en les dades d'entrada.

import torch
import torch.nn as nn

class CustomLinear(nn.Module):
    def __init__(self, input_dim, output_dim):
        super(CustomLinear, self).__init__()
        self.weight = nn.Parameter(torch.randn(input_dim, output_dim))
        self.bias = nn.Parameter(torch.randn(output_dim))

    def forward(self, x):
        return torch.matmul(x, self.weight) + self.bias

# Exemple d'ús
input_data = torch.randn(10, 5)  # Lot de 10 mostres, cadascuna amb 5 característiques
layer = CustomLinear(5, 3)
output_data = layer(input_data)
print(output_data)

Explicació

  • Mètode __init__: Inicialitza la capa amb pesos i biaixos.
  • Mètode forward: Defineix el càlcul realitzat en cada crida. En aquest cas, realitza una multiplicació de matrius seguida d'una suma.

Creació d'un Mòdul Personalitzat

Els mòduls personalitzats poden encapsular múltiples capes i altres mòduls, permetent arquitectures més complexes.

Exemple: Mòdul de Xarxa Neuronal Personalitzada

Creem un mòdul de xarxa neuronal personalitzat que consisteix en dues capes lineals amb una activació ReLU entremig.

class CustomNN(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(CustomNN, self).__init__()
        self.layer1 = CustomLinear(input_dim, hidden_dim)
        self.relu = nn.ReLU()
        self.layer2 = CustomLinear(hidden_dim, output_dim)

    def forward(self, x):
        x = self.layer1(x)
        x = self.relu(x)
        x = self.layer2(x)
        return x

# Exemple d'ús
input_data = torch.randn(10, 5)  # Lot de 10 mostres, cadascuna amb 5 característiques
model = CustomNN(5, 4, 3)
output_data = model(input_data)
print(output_data)

Explicació

  • Mètode __init__: Inicialitza les capes i funcions d'activació.
  • Mètode forward: Defineix el pas endavant a través de la xarxa, aplicant cada capa i activació en seqüència.

Mòduls Personalitzats Avançats

Per a casos d'ús més avançats, potser necessitis crear mòduls que incloguin bucles d'entrenament personalitzats, funcions de pèrdua o altres components especialitzats.

Exemple: Mòdul Personalitzat amb Dropout

Creem un mòdul personalitzat que inclou una capa de dropout per prevenir el sobreajustament.

class CustomNNWithDropout(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim, dropout_prob):
        super(CustomNNWithDropout, self).__init__()
        self.layer1 = CustomLinear(input_dim, hidden_dim)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(dropout_prob)
        self.layer2 = CustomLinear(hidden_dim, output_dim)

    def forward(self, x):
        x = self.layer1(x)
        x = self.relu(x)
        x = self.dropout(x)
        x = self.layer2(x)
        return x

# Exemple d'ús
input_data = torch.randn(10, 5)  # Lot de 10 mostres, cadascuna amb 5 característiques
model = CustomNNWithDropout(5, 4, 3, 0.5)
output_data = model(input_data)
print(output_data)

Explicació

  • Capa de Dropout: Estableix aleatòriament una fracció de les unitats d'entrada a 0 en cada actualització durant el temps d'entrenament, cosa que ajuda a prevenir el sobreajustament.

Resum

En aquesta secció, hem cobert com crear capes i mòduls personalitzats en PyTorch. Hem començat amb una simple capa lineal personalitzada i hem passat a mòduls personalitzats més complexos, incloent-hi aquells amb capes de dropout. Les capes i mòduls personalitzats són essencials per construir arquitectures de xarxes neuronals sofisticades adaptades a tasques específiques.

En dominar aquests conceptes, podràs estendre la funcionalitat de PyTorch i crear models altament personalitzats per als teus projectes de machine learning.
