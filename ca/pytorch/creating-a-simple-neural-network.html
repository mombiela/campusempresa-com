En aquesta secció, aprendrem a crear una xarxa neuronal simple utilitzant PyTorch. Això inclourà configurar l'entorn, entendre els blocs bàsics de les xarxes neuronals i implementar una xarxa neuronal simple des de zero.

Configuració de l'Entorn

Abans de començar a codificar, necessitem configurar el nostre entorn per utilitzar PyTorch.

  1. Instal·lar PyTorch: Seguiu les instruccions al lloc web oficial de PyTorch per instal·lar PyTorch.

  2. Verificar la Instal·lació: Obriu un intèrpret de Python i executeu les següents comandes per assegurar-vos que PyTorch està instal·lat correctament:

    import torch
print(torch.__version__)

Entendre els Blocs Bàsics

Abans de submergir-nos en el codi, entenguem els components bàsics d'una xarxa neuronal en PyTorch.

  • Tensors: El bloc fonamental en PyTorch, similar als arrays de NumPy però amb suport per a GPU.
  • Autograd: La biblioteca de diferenciació automàtica de PyTorch que ajuda a calcular gradients.
  • Mòdul de Xarxa Neuronal (torch.nn): Conté classes i funcions per construir xarxes neuronals.
  • Optimitzadors (torch.optim): Utilitzats per actualitzar els pesos de la xarxa basant-se en els gradients calculats.
  • Funcions de Pèrdua: Mesuren com de bé està funcionant la xarxa neuronal.

Implementació d'una Xarxa Neuronal Simple

Implementem una xarxa neuronal simple per classificar dígits manuscrits del conjunt de dades MNIST.

Pas 1: Importar Biblioteques

Primer, importem les biblioteques necessàries.

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Pas 2: Carregar i Preprocessar les Dades

Carreguem el conjunt de dades MNIST i apliquem les transformacions necessàries.

transform = transforms.Compose([transforms.ToTensor(), transforms.Normalize((0.5,), (0.5,))])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=64, shuffle=True)

testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=64, shuffle=False)

Pas 3: Definir la Xarxa Neuronal

Definim una xarxa neuronal simple amb una capa oculta.

class SimpleNN(nn.Module):
    def __init__(self):
        super(SimpleNN, self).__init__()
        self.fc1 = nn.Linear(28 * 28, 128)  # Capa d'entrada a capa oculta
        self.fc2 = nn.Linear(128, 10)       # Capa oculta a capa de sortida

    def forward(self, x):
        x = x.view(-1, 28 * 28)  # Aplanar l'entrada
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

net = SimpleNN()
print(net)

Pas 4: Definir la Funció de Pèrdua i l'Optimitzador

Utilitzem la pèrdua de Cross-Entropy i l'optimitzador Stochastic Gradient Descent (SGD).

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.01, momentum=0.9)

Pas 5: Entrenar la Xarxa Neuronal

Entrenem la xarxa durant uns quants epochs.

for epoch in range(5):  # Recorrem el conjunt de dades diverses vegades
    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        inputs, labels = data

        optimizer.zero_grad()  # Netejar els gradients dels paràmetres

        outputs = net(inputs)  # Passada endavant
        loss = criterion(outputs, labels)  # Calcular la pèrdua
        loss.backward()  # Passada enrere
        optimizer.step()  # Optimitzar els pesos

        running_loss += loss.item()
        if i % 100 == 99:  # Imprimir cada 100 mini-lots
            print(f'[Epoch {epoch + 1}, Batch {i + 1}] loss: {running_loss / 100:.3f}')
            running_loss = 0.0

print('Entrenament Finalitzat')

Pas 6: Avaluar la Xarxa Neuronal

Avaluem el rendiment de la xarxa entrenada en el conjunt de dades de prova.

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Precisió de la xarxa en les 10000 imatges de prova: {100 * correct / total:.2f}%')
