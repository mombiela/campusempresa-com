Introducció a les LSTMs

Les xarxes de Memòria a Curt i Llarg Termini (LSTM) són un tipus de xarxa neuronal recurrent (RNN) dissenyada per gestionar el problema del gradient que es desvaneix, fent-les efectives per aprendre dependències a llarg termini. S'utilitzen àmpliament en tasques com la predicció de sèries temporals, el processament del llenguatge natural i el reconeixement de veu.

Conceptes Clau

  • Xarxes Neuronals Recurrents (RNNs): Xarxes amb bucles que permeten que la informació persisteixi.
  • Problema del Gradient que es Desvaneix: Dificultat en l'entrenament de RNNs a causa dels gradients que es tornen molt petits.
  • Cèl·lula LSTM: Una unitat especial en les LSTMs dissenyada per mantenir dependències a llarg termini.

Comprensió de l'Arquitectura LSTM

Les LSTMs tenen una estructura única que inclou diverses portes per controlar el flux d'informació.

Components d'una Cèl·lula LSTM

  • Porta d'Oblit: Decideix quina informació descartar de l'estat de la cèl·lula.
  • Porta d'Entrada: Decideix quins valors de l'entrada actualitzaran l'estat de la cèl·lula.
  • Estat de la Cèl·lula: La memòria de la xarxa.
  • Porta de Sortida: Decideix quin hauria de ser el següent estat ocult.

Diagrama de la Cèl·lula LSTM

[Entrada] --> [Porta d'Oblit] --> [Estat de la Cèl·lula] --> [Porta de Sortida] --> [Estat Ocult]

Implementació de LSTMs en PyTorch

Anem a implementar les LSTMs utilitzant PyTorch.

Pas 1: Importació de Llibreries

import torch
import torch.nn as nn
import torch.optim as optim

Pas 2: Definició del Model LSTM

class LSTMModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(LSTMModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])
        return out

Pas 3: Entrenament del Model LSTM

# Hiperparàmetres
input_size = 10
hidden_size = 50
output_size = 1
num_layers = 2
num_epochs = 100
learning_rate = 0.001

# Model, Pèrdua, Optimitzador
model = LSTMModel(input_size, hidden_size, output_size, num_layers)
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=learning_rate)

# Dades Fictícies
x_train = torch.randn(100, 10, input_size)
y_train = torch.randn(100, output_size)

# Bucle d'Entrenament
for epoch in range(num_epochs):
    model.train()
    outputs = model(x_train)
    optimizer.zero_grad()
    loss = criterion(outputs, y_train)
    loss.backward()
    optimizer.step()
    
    if (epoch+1) % 10 == 0:
        print(f'Epoch [{epoch+1}/{num_epochs}], Loss: {loss.item():.4f}')

Tècniques Avançades de LSTM

LSTMs Bidireccionals

Les LSTMs bidireccionals processen dades en ambdues direccions, proporcionant context addicional.

class BiLSTMModel(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1):
        super(BiLSTMModel, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True, bidirectional=True)
        self.fc = nn.Linear(hidden_size*2, output_size)
    
    def forward(self, x):
        h0 = torch.zeros(self.num_layers*2, x.size(0), self.hidden_size).to(x.device)
        c0 = torch.zeros(self.num_layers*2, x.size(0), self.hidden_size).to(x.device)
        
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])
        return out

LSTM amb Dropout

Afegir dropout a les capes LSTM ajuda a prevenir el sobreajustament.

class LSTMWithDropout(nn.Module):
    def __init__(self, input_size, hidden_size, output_size, num_layers=1, dropout=0.5):
        super(LSTMWithDropout, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        self.lstm = nn.LSTM(input_size, hidden_size, num_layers, batch_first=True, dropout=dropout)
        self.fc = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size).to(x.device)
        
        out, _ = self.lstm(x, (h0, c0))
        out = self.fc(out[:, -1, :])
        return out

Conclusió

Les LSTMs són eines poderoses per a tasques de predicció de seqüències, superant les limitacions de les RNNs tradicionals. Comprenent la seva arquitectura i aprenent a implementar-les en PyTorch, pots aprofitar les seves capacitats per a diverses aplicacions. Tècniques avançades com les LSTMs bidireccionals i el dropout poden millorar encara més el rendiment i la robustesa del model.
