La càrrega i el preprocessament de dades són passos crucials en qualsevol pipeline d'aprenentatge automàtic. En aquesta secció, cobrirem com carregar i preprocessar dades de manera eficient utilitzant PyTorch. Això inclou entendre els datasets, els data loaders i diverses tècniques de preprocessament.

Introducció a la Càrrega de Dades

En PyTorch, la càrrega de dades es gestiona mitjançant el mòdul torch.utils.data. Aquest mòdul proporciona dues classes principals:

  • Dataset: Una classe que representa un dataset.
  • DataLoader: Una classe que carrega dades d'un Dataset i proporciona un iterable sobre el dataset.

Classe Dataset

La classe Dataset és una classe abstracta que representa un dataset. Per crear un dataset personalitzat, cal subclassejar Dataset i sobreescriure els següents mètodes:

  • __len__: Retorna la mida del dataset.
  • __getitem__: Recupera una mostra del dataset a l'índex donat.

Exemple: Dataset Personalitzat

import torch
from torch.utils.data import Dataset

class CustomDataset(Dataset):
    def __init__(self, data, labels):
        self.data = data
        self.labels = labels

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        sample = {'data': self.data[idx], 'label': self.labels[idx]}
        return sample

# Exemple d'ús
data = torch.randn(100, 3, 32, 32)  # 100 mostres d'imatges de 3x32x32
labels = torch.randint(0, 10, (100,))  # 100 etiquetes per a 10 classes

dataset = CustomDataset(data, labels)
print(len(dataset))  # Sortida: 100
print(dataset[0])  # Sortida: {'data': tensor(...), 'label': tensor(...)}

Classe DataLoader

La classe DataLoader proporciona un iterable sobre un dataset amb suport per a batching, shuffling i càrrega de dades en paral·lel.

Exemple: Utilitzant DataLoader

from torch.utils.data import DataLoader

dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

for batch in dataloader:
    print(batch['data'].shape, batch['label'])

Tècniques de Preprocessament

El preprocessament és essencial per assegurar que les dades estan en el format correcte i normalitzades per a l'entrenament. PyTorch proporciona diverses utilitats per al preprocessament, especialment per a dades d'imatges.

Passos Comuns de Preprocessament

  • Normalització: Ajustar el rang dels valors dels píxels.
  • Redimensionament: Canviar les dimensions de les imatges.
  • Augmentació de Dades: Tècniques com el flipping, la rotació i el cropping per augmentar artificialment la mida del dataset.

Utilitzant `torchvision.transforms`

El mòdul torchvision.transforms proporciona diverses operacions comunes de preprocessament.

Exemple: Preprocessament d'Imatges

from torchvision import transforms
from PIL import Image

# Definir una sèrie de transformacions
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.RandomHorizontalFlip(),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

# Carregar una imatge
image = Image.open('path/to/image.jpg')

# Aplicar transformacions
transformed_image = transform(image)
print(transformed_image.shape)  # Sortida: torch.Size([3, 128, 128])

Integrant Transforms amb Dataset

Podeu integrar aquestes transformacions directament en el vostre dataset personalitzat.

Exemple: Dataset Personalitzat amb Transforms

class CustomDataset(Dataset):
    def __init__(self, data, labels, transform=None):
        self.data = data
        self.labels = labels
        self.transform = transform

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        sample = {'data': self.data[idx], 'label': self.labels[idx]}
        if self.transform:
            sample['data'] = self.transform(sample['data'])
        return sample

# Exemple d'ús
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

dataset = CustomDataset(data, labels, transform=transform)
dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

for batch in dataloader:
    print(batch['data'].shape, batch['label'])

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la càrrega i el preprocessament de dades en PyTorch. Hem après com crear datasets personalitzats utilitzant la classe Dataset, com utilitzar la classe DataLoader per carregar dades de manera eficient, i com aplicar tècniques comunes de preprocessament utilitzant torchvision.transforms. Aquestes habilitats són fonamentals per preparar dades per a l'entrenament de models d'aprenentatge automàtic en PyTorch.
