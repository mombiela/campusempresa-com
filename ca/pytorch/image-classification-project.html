Introducció

En aquesta secció, explorarem els fonaments de la classificació d'imatges utilitzant PyTorch. Aquest projecte et guiarà des de la configuració del teu entorn fins a la construcció i entrenament d'un model d'aprenentatge profund per classificar imatges. Al final d'aquest projecte, tindràs una comprensió sòlida de com implementar models de classificació d'imatges utilitzant PyTorch.

Configuració de l'Entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

Instal·lació de PyTorch

Per instal·lar PyTorch, pots utilitzar pip. Obre el teu terminal i executa la següent comanda:

pip install torch torchvision

Importació de les Biblioteques Necessàries

Necessitarem diverses biblioteques per a aquest projecte. Importem-les:

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Preparació de les Dades

La preparació de les dades és un pas crucial en qualsevol projecte d'aprenentatge automàtic. Utilitzarem el conjunt de dades CIFAR-10 per a aquest projecte.

Carregant el Conjunt de Dades CIFAR-10

PyTorch proporciona una manera convenient de carregar el conjunt de dades CIFAR-10 utilitzant torchvision.datasets.

transform = transforms.Compose(
    [transforms.ToTensor(),
     transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))])

trainset = torchvision.datasets.CIFAR10(root='./data', train=True,
                                        download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=4,
                                          shuffle=True, num_workers=2)

testset = torchvision.datasets.CIFAR10(root='./data', train=False,
                                       download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4,
                                         shuffle=False, num_workers=2)

classes = ('plane', 'car', 'bird', 'cat',
           'deer', 'dog', 'frog', 'horse', 'ship', 'truck')

Construcció del Model

Construirem una Xarxa Neuronal Convolucional (CNN) per a la nostra tasca de classificació d'imatges.

Definint la CNN

Aquí tens un model CNN simple:

class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(3, 6, 5)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(6, 16, 5)
        self.fc1 = nn.Linear(16 * 5 * 5, 120)
        self.fc2 = nn.Linear(120, 84)
        self.fc3 = nn.Linear(84, 10)

    def forward(self, x):

        x = self.pool(F.relu(self.conv2(x)))
        x = x.view(-1, 16 * 5 * 5)
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

net = Net()

Entrenament del Model

Ara que tenim el nostre model, necessitem entrenar-lo.

Definint la Funció de Pèrdua i l'Optimitzador

Utilitzarem la Pèrdua d'Entropia Creuada i el Descens de Gradient Estocàstic (SGD) per a aquesta tasca.

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.001, momentum=0.9)

Bucle d'Entrenament

Aquí tens el codi per entrenar el model:

for epoch in range(2):  # loop over the dataset multiple times

    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        # get the inputs; data is a list of [inputs, labels]
        inputs, labels = data

        # zero the parameter gradients
        optimizer.zero_grad()

        # forward + backward + optimize
        outputs = net(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        # print statistics
        running_loss += loss.item()
        if i % 2000 == 1999:    # print every 2000 mini-batches
            print(f'[{epoch + 1}, {i + 1}] loss: {running_loss / 2000:.3f}')
            running_loss = 0.0

print('Finished Training')

Avaluació del Model

Després de l'entrenament, necessitem avaluar el rendiment del model en les dades de prova.

Provant el Model

Aquí tens el codi per provar el model:

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Accuracy of the network on the 10000 test images: {100 * correct / total}%')

Rendiment per Classe

També podem comprovar el rendiment del model per a cada classe.

class_correct = list(0. for i in range(10))
class_total = list(0. for i in range(10))
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs, 1)
        c = (predicted == labels).squeeze()
        for i in range(4):
            label = labels[i]
            class_correct[label] += c[i].item()
            class_total[label] += 1

for i in range(10):
    print(f'Accuracy of {classes[i]} : {100 * class_correct[i] / class_total[i]}%')
