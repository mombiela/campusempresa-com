Construint Xarxes Neuronals

  • Introducció a les Xarxes Neuronals
    • Què són les xarxes neuronals?
    • Components bàsics d'una xarxa neuronal
  • Utilitzant torch.nn per Construir Xarxes
    • Definint una xarxa neuronal simple
    • Passades endavant i enrere
import torch.nn as nn
import torch.nn.functional as F

# Definint una xarxa neuronal simple
class SimpleNN(nn.Module):
    def __init__(self):
        super(SimpleNN, self).__init__()
        self.fc1 = nn.Linear(10, 50)
        self.fc2 = nn.Linear(50, 1)

    def forward(self, x):
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# Creant una instància de la xarxa
net = SimpleNN()
print(net)

Entrenant Xarxes Neuronals

  • Preparant Dades
    • Carregant datasets utilitzant torchvision
    • Preprocessament i augmentació de dades
  • Bucle d'Entrenament
    • Definint funcions de pèrdua i optimitzadors
    • Implementant el bucle d'entrenament
import torch.optim as optim
from torchvision import datasets, transforms

# Carregant datasets
transform = transforms.Compose([transforms.ToTensor()])
train_dataset = datasets.MNIST(root='./data', train=True, download=True, transform=transform)
train_loader = torch.utils.data.DataLoader(dataset=train_dataset, batch_size=64, shuffle=True)

# Definint funció de pèrdua i optimitzador
criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.01)

# Bucle d'entrenament
for epoch in range(2):  # bucle sobre el dataset múltiples vegades
    for inputs, labels in train_loader:
        optimizer.zero_grad()  # zero els gradients dels paràmetres
        outputs = net(inputs.view(-1, 10))  # passada endavant
        loss = criterion(outputs, labels)  # calcular pèrdua
        loss.backward()  # passada enrere
        optimizer.step()  # optimitzar

print('Entrenament Finalitzat')
