Introducció a TorchVision

TorchVision és una biblioteca crucial en l'ecosistema PyTorch, específicament dissenyada per a tasques de visió per computador. Proporciona utilitats per al processament d'imatges, conjunts de dades i models preentrenats, facilitant el treball amb dades d'imatges.

Conceptes Clau

  • Datasets: Conjunts de dades predefinits com CIFAR-10, ImageNet i MNIST.
  • Transforms: Transformacions d'imatges per a la preprocessament i augment de dades.
  • Models: Models preentrenats per a tasques com classificació d'imatges, detecció d'objectes i segmentació.

Instal·lació de TorchVision

Per començar amb TorchVision, cal instal·lar-lo juntament amb PyTorch. Pots fer-ho utilitzant pip:

pip install torch torchvision

Treballant amb Datasets

TorchVision proporciona diversos conjunts de dades populars que es poden carregar i utilitzar fàcilment.

Carregant un Dataset

Aquí tens un exemple de com carregar el conjunt de dades CIFAR-10:

import torchvision
import torchvision.transforms as transforms

# Defineix una transformació per normalitzar les dades
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))
])

# Carrega els conjunts de dades d'entrenament i prova
trainset = torchvision.datasets.CIFAR10(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=4, shuffle=True, num_workers=2)

testset = torchvision.datasets.CIFAR10(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4, shuffle=False, num_workers=2)

Entenent les Transforms

Les transformacions s'utilitzen per preprocessar i augmentar les dades. Les transformacions comunes inclouen redimensionar, retallar, normalitzar i convertir imatges a tensors.

Exemple de Transforms Comuns

transform = transforms.Compose([
    transforms.Resize((256, 256)),
    transforms.RandomCrop(224),
    transforms.RandomHorizontalFlip(),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

Utilitzant Models Preentrenats

TorchVision proporciona diversos models preentrenats que es poden utilitzar per a diverses tasques. Aquests models estan entrenats en grans conjunts de dades i es poden ajustar per a tasques específiques.

Carregant un Model Preentrenat

Aquí tens un exemple de com carregar i utilitzar un model ResNet preentrenat per a la classificació d'imatges:

import torch
import torchvision.models as models

# Carrega un model ResNet preentrenat
model = models.resnet18(pretrained=True)

# Posa el model en mode d'avaluació
model.eval()

# Exemple d'ús del model per a inferència
from PIL import Image
from torchvision import transforms

# Carrega una imatge
img = Image.open("path_to_image.jpg")

# Defineix la transformació per complir amb els requisits d'entrada del model
preprocess = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]),
])

# Preprocessa la imatge
img_t = preprocess(img)
batch_t = torch.unsqueeze(img_t, 0)

# Realitza la inferència
with torch.no_grad():
    out = model(batch_t)

# Imprimeix les 5 classes més probables
_, indices = torch.topk(out, 5)
print(indices)

Tècniques d'Augment de Dades

L'augment de dades és una tècnica utilitzada per augmentar la diversitat de les teves dades d'entrenament sense recollir noves dades. TorchVision proporciona diverses tècniques d'augment.

Exemple d'Augment de Dades

transform = transforms.Compose([
    transforms.RandomResizedCrop(224),
    transforms.RandomHorizontalFlip(),
    transforms.RandomRotation(10),
    transforms.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2, hue=0.2),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
])

Conclusió

TorchVision és una biblioteca poderosa que simplifica molts aspectes de les tasques de visió per computador en PyTorch. Proporciona accés fàcil a conjunts de dades populars, una varietat de transformacions d'imatges per a preprocessament i augment, i models preentrenats per a un desplegament ràpid i ajustament. En dominar TorchVision, pots millorar significativament els teus projectes i fluxos de treball de visió per computador.
