El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, cobrirem els aspectes pràctics d'implementar un projecte utilitzant Hadoop. Això inclourà la configuració de l'entorn, l'escriptura i execució de treballs MapReduce, i l'ús d'altres eines de l'ecosistema Hadoop com Hive i Pig. Al final d'aquesta secció, hauríeu de ser capaços d'implementar una pipeline completa de processament de dades utilitzant Hadoop.

Configuració de l'Entorn

Abans de començar a escriure codi, necessitem configurar el nostre entorn Hadoop. Això implica instal·lar Hadoop i configurar-lo tant per a clústers de node únic com per a clústers de múltiples nodes.

Configuració de Node Únic

Una configuració de node únic és ideal per a finalitats de desenvolupament i proves.

  1. Descarregar Hadoop: Descarregueu la versió estable més recent de Hadoop des del lloc web oficial.
  2. Extreure Hadoop: Extreieu l'arxiu tarball descarregat a un directori de la vostra elecció.
  3. Configurar Hadoop: Editeu els arxius de configuració situats al directori etc/hadoop.
    • core-site.xml
    • hdfs-site.xml
    • mapred-site.xml
    • yarn-site.xml
  4. Iniciar Hadoop: Utilitzeu les següents comandes per iniciar els serveis de Hadoop: 
    start-dfs.sh
start-yarn.sh

Configuració de Múltiples Nodes

Una configuració de múltiples nodes s'utilitza per a entorns de producció.

  1. Prerequisits: Assegureu-vos que SSH està instal·lat i configurat per a l'inici de sessió sense contrasenya entre nodes.
  2. Instal·lar Hadoop: Instal·leu Hadoop a tots els nodes.
  3. Configurar Nodes Mestre i Esclaus: Editeu els arxius masters i slaves per especificar els nodes mestre i esclaus.
  4. Distribuir Configuració: Copieu els arxius de configuració des del node mestre a tots els nodes esclaus.
  5. Iniciar Hadoop: Inicieu els serveis de Hadoop a tots els nodes utilitzant les mateixes comandes que en la configuració de node únic.

Escriure i Executar Treballs MapReduce

MapReduce és el component central de Hadoop per processar grans conjunts de dades.

Escriure un Treball MapReduce

Un treball MapReduce consisteix en un Mapper i un Reducer.

  • Mapper: Processa dades d'entrada i produeix parells clau-valor intermedis.
  • Reducer: Processa parells clau-valor intermedis i produeix la sortida final.

Aquí teniu un exemple senzill d'un treball MapReduce en Java que compta el nombre d'ocurrències de cada paraula en un fitxer de text:

import java.io.IOException;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class WordCount {

    public static class TokenizerMapper extends Mapper<Object, Text, Text, IntWritable> {
        private final static IntWritable one = new IntWritable(1);
        private Text word = new Text();

        public void map(Object key, Text value, Context context) throws IOException, InterruptedException {
            String[] tokens = value.toString().split("\\s+");
            for (String token : tokens) {
                word.set(token);
                context.write(word, one);
            }
        }
    }

    public static class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
        public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            context.write(key, new IntWritable(sum));
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "word count");
        job.setJarByClass(WordCount.class);
        job.setMapperClass(TokenizerMapper.class);
        job.setCombinerClass(IntSumReducer.class);
        job.setReducerClass(IntSumReducer.class);
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);
        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));
        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

Executar un Treball MapReduce

  1. Compilar el Codi: Compileu el codi Java utilitzant javac. 
    javac -classpath `hadoop classpath` -d wordcount_classes WordCount.java
  2. Crear un Arxiu JAR: Empaqueteu les classes compilades en un arxiu JAR. 
    jar -cvf wordcount.jar -C wordcount_classes/ .
  3. Executar el Treball: Envieu el treball al clúster Hadoop. 
    hadoop jar wordcount.jar WordCount /input /output

Utilitzar Hive i Pig

Hive i Pig són eines d'alt nivell en l'ecosistema Hadoop que simplifiquen el procés d'escriure treballs de processament de dades.

Hive

Hive proporciona una interfície similar a SQL per consultar dades emmagatzemades a Hadoop.

  • Crear una Taula: 
    CREATE TABLE word_counts (word STRING, count INT) ROW FORMAT DELIMITED FIELDS TERMINATED BY '\t';
  • Carregar Dades: 
    LOAD DATA INPATH '/output' INTO TABLE word_counts;
  • Consultar Dades: 
    SELECT word, count FROM word_counts WHERE count > 10;

Pig

Pig utilitza un llenguatge de script anomenat Pig Latin per processar dades.

  • Carregar Dades: 
    word_counts = LOAD '/output' USING PigStorage('\t') AS (word:chararray, count:int);
  • Filtrar Dades: 
    filtered_word_counts = FILTER word_counts BY count > 10;
  • Emmagatzemar Dades: 
    STORE filtered_word_counts INTO '/filtered_output';
