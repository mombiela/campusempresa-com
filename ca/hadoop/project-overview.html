Introducció a Hadoop

Hadoop és un marc de codi obert que permet el processament distribuït de grans conjunts de dades a través de clústers d'ordinadors utilitzant models de programació simples. Està dissenyat per escalar des de servidors individuals fins a milers de màquines, cadascuna oferint computació i emmagatzematge local.

Conceptes clau

  • Emmagatzematge distribuït: Hadoop utilitza el Sistema de Fitxers Distribuït de Hadoop (HDFS) per emmagatzemar dades a través de múltiples màquines.
  • Processament distribuït: Hadoop processa dades utilitzant el model de programació MapReduce.
  • Escalabilitat: Hadoop pot escalar fàcilment des d'un sol node fins a milers de nodes.
  • Tolerància a fallades: Hadoop maneja automàticament les fallades de maquinari i assegura la integritat de les dades.

Components de Hadoop

Hadoop consta de diversos components bàsics que treballen junts per proporcionar un entorn de processament de dades robust i escalable.

Sistema de Fitxers Distribuït de Hadoop (HDFS)

HDFS està dissenyat per emmagatzemar conjunts de dades molt grans de manera fiable i per transmetre aquests conjunts de dades a gran amplada de banda a les aplicacions d'usuari.

  • NameNode: Gestiona les metadades i l'espai de noms del sistema de fitxers.
  • DataNode: Emmagatzema els blocs de dades reals.

MapReduce

MapReduce és un model de programació i una implementació associada per processar i generar grans conjunts de dades.

  • Funció Map: Processa dades d'entrada i produeix un conjunt de parells clau/valor intermedis.
  • Funció Reduce: Fusiona tots els valors intermedis associats amb la mateixa clau intermèdia.

YARN (Yet Another Resource Negotiator)

YARN és el sistema de gestió de recursos del clúster de Hadoop.

  • ResourceManager: Gestiona recursos i programa aplicacions.
  • NodeManager: Gestiona recursos en un sol node.

Ecosistema de Hadoop

L'ecosistema de Hadoop inclou una varietat d'eines i tecnologies que milloren les capacitats dels components bàsics de Hadoop.

  • Hive: Emmagatzematge de dades i llenguatge de consulta similar a SQL.
  • Pig: Plataforma d'alt nivell per crear programes MapReduce.
  • HBase: Emmagatzematge de dades grans, distribuït i escalable.
  • Spark: Sistema de computació de clúster ràpid i de propòsit general.
  • Sqoop: Eina per transferir dades entre Hadoop i bases de dades relacionals.
  • Flume: Servei per recollir, agregar i moure grans quantitats de dades de registre de manera eficient.

Exemple: Programa de recompte de paraules

Per il·lustrar com funciona Hadoop, vegem un exemple senzill: un programa de recompte de paraules utilitzant MapReduce.

Codi del Mapper

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

public class TokenizerMapper extends Mapper<LongWritable, Text, Text, IntWritable> {
    private final static IntWritable one = new IntWritable(1);
    private Text word = new Text();

    public void map(LongWritable key, Text value, Context context) throws IOException, InterruptedException {
        String[] tokens = value.toString().split("\\s+");
        for (String token : tokens) {
            word.set(token);
            context.write(word, one);
        }
    }
}

Codi del Reducer

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Reducer;

public class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
    public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
        int sum = 0;
        for (IntWritable val : values) {
            sum += val.get();
        }
        context.write(key, new IntWritable(sum));
    }
}

Explicació

  • Mapper: La classe TokenizerMapper pren una línia de text com a entrada i la divideix en paraules. Cada paraula es produeix com una clau amb un valor d'1.
  • Reducer: La classe IntSumReducer pren la llista de valors associats amb una paraula (clau) i els suma per obtenir el recompte total per a aquesta paraula.
