Execució d'un Clúster de Hadoop ►

Configurar un clúster Hadoop és un pas crucial per aprofitar el poder de la computació distribuïda per al processament de grans volums de dades. Aquesta secció et guiarà a través del procés de configuració d'un clúster Hadoop des de zero, cobrint tant configuracions de node únic com de múltiples nodes.

Prerequisits

Abans de configurar un clúster Hadoop, assegura't de tenir el següent:

  • Un coneixement bàsic dels comandaments de Linux i del sistema de fitxers.
  • Java Development Kit (JDK) instal·lat en tots els nodes.
  • Accés SSH entre nodes sense contrasenya.

Configuració del Clúster Hadoop de Node Únic

Una configuració de node únic és ideal per a finalitats d'aprenentatge i desenvolupament. Et permet executar Hadoop en una sola màquina.

Pas 1: Instal·lar Java

Hadoop requereix Java per funcionar. Assegura't que Java estigui instal·lat i configurat correctament.

sudo apt-get update
sudo apt-get install openjdk-8-jdk
java -version

Pas 2: Descarregar i Extreure Hadoop

Descarrega la versió estable més recent de Hadoop des del lloc web oficial i extreu-la.

wget https://downloads.apache.org/hadoop/common/hadoop-3.3.1/hadoop-3.3.1.tar.gz
tar -xzvf hadoop-3.3.1.tar.gz
mv hadoop-3.3.1 /usr/local/hadoop

Pas 3: Configurar les Variables d'Entorn de Hadoop

Afegeix les variables d'entorn de Hadoop al fitxer .bashrc.

export HADOOP_HOME=/usr/local/hadoop
export HADOOP_INSTALL=$HADOOP_HOME
export HADOOP_MAPRED_HOME=$HADOOP_HOME
export HADOOP_COMMON_HOME=$HADOOP_HOME
export HADOOP_HDFS_HOME=$HADOOP_HOME
export YARN_HOME=$HADOOP_HOME
export HADOOP_COMMON_LIB_NATIVE_DIR=$HADOOP_HOME/lib/native
export PATH=$PATH:$HADOOP_HOME/sbin:$HADOOP_HOME/bin

Aplica els canvis:

source ~/.bashrc

Pas 4: Configurar Hadoop

Edita els següents fitxers de configuració situats al directori $HADOOP_HOME/etc/hadoop:

  • core-site.xml
<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>
  • hdfs-site.xml
<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
</configuration>
  • mapred-site.xml
<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>
  • yarn-site.xml
<configuration>
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
</configuration>

Pas 5: Formatjar el Namenode

Formatja el sistema de fitxers Hadoop (HDFS) abans d'iniciar el clúster.

hdfs namenode -format

Pas 6: Iniciar els Dimonis de Hadoop

Inicia els dimonis de Hadoop (Namenode, Datanode, ResourceManager i NodeManager).

start-dfs.sh
start-yarn.sh

Pas 7: Verificar la Configuració

Comprova les interfícies web de Hadoop per assegurar-te que tot funciona correctament:

  • HDFS: http://localhost:9870
  • YARN: http://localhost:8088

Configuració del Clúster Hadoop de Múltiples Nodes

Una configuració de múltiples nodes s'utilitza per a entorns de producció per gestionar el processament de grans volums de dades.

Pas 1: Configurar l'Autenticació Basada en Clau SSH

Genera una clau SSH al node mestre i còpia-la a tots els nodes esclaus.

ssh-keygen -t rsa
ssh-copy-id user@slave1
ssh-copy-id user@slave2

Pas 2: Configurar Hadoop en Nodes Mestres i Esclaus

Repeteix els passos de configuració de node únic en tots els nodes. A més, configura el fitxer slaves al node mestre amb les adreces IP o noms d'host de tots els nodes esclaus.

slave1
slave2

Pas 3: Configurar els Fitxers Bàsics de Hadoop

Assegura't que els fitxers core-site.xml, hdfs-site.xml, mapred-site.xml i yarn-site.xml estiguin configurats correctament en tots els nodes, de manera similar a la configuració de node únic.

Pas 4: Formatjar el Namenode

Formatja el Namenode només al node mestre.

hdfs namenode -format

Pas 5: Iniciar els Dimonis de Hadoop

Inicia els dimonis de Hadoop al node mestre.

start-dfs.sh
start-yarn.sh

Pas 6: Verificar el Clúster

Comprova les interfícies web de Hadoop al node mestre per assegurar-te que tots els nodes estan connectats i funcionant correctament.

Conclusió

Configurar un clúster Hadoop implica diversos passos, des d'instal·lar els prerequisits fins a configurar diversos components de Hadoop. Una configuració de node únic és adequada per a l'aprenentatge i el desenvolupament, mentre que una configuració de múltiples nodes és essencial per a entorns de producció. Seguint els passos descrits en aquesta secció, pots configurar i verificar amb èxit un clúster Hadoop, permetent-te aprofitar el poder de la computació distribuïda per al processament de grans volums de dades.
