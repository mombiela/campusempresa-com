Introducció a Apache HBase

Apache HBase és un magatzem de dades distribuït, escalable i de grans dimensions modelat segons el Bigtable de Google. Està dissenyat per gestionar grans quantitats de dades a través de molts servidors de mercaderia. HBase és part de l'ecosistema Hadoop i s'executa a sobre del Hadoop Distributed File System (HDFS).

Conceptes Clau

  • Emmagatzematge Orientat a Columnes: A diferència de les bases de dades relacionals tradicionals, HBase emmagatzema dades en columnes en lloc de files.
  • Escalabilitat: HBase està dissenyat per escalar horitzontalment afegint més servidors.
  • Consistència: Proporciona una forta consistència per a operacions de lectura i escriptura.
  • Sense Esquema: Les taules d'HBase poden tenir un nombre variable de columnes, cosa que el fa flexible per a diferents tipus de dades.
  • Fragmentació Automàtica: Les dades es divideixen automàticament en regions i es distribueixen a través del clúster.

Configuració d'HBase

Prerequisits

  • Java Development Kit (JDK)
  • Hadoop instal·lat i configurat
  • Zookeeper (HBase utilitza Zookeeper per a la coordinació distribuïda)

Passos d'Instal·lació

  1. Descarregar HBase: Obteniu l'última versió des del lloc web d'Apache HBase.
  2. Extreure l'Arxiu: Descomprimiu el fitxer descarregat.
  3. Configurar HBase: Editeu el fitxer hbase-site.xml per configurar HBase.
  4. Iniciar HBase: Utilitzeu l'script start-hbase.sh per iniciar els serveis d'HBase.
# Exemple d'inici d'HBase
$ bin/start-hbase.sh

Model de Dades d'HBase

Taules i Famílies de Columnes

  • Taules: Similar a les taules en bases de dades relacionals però dissenyades per gestionar dades disperses.
  • Famílies de Columnes: Grup de columnes que s'emmagatzemen juntes. Cada família de columnes pot tenir múltiples columnes.

Exemple d'Estructura de Taula

| Clau de Fila | Família de Columnes: Columna | Valor | |--------------|------------------------------|-------| | fila1 | personal:nom | John | | fila1 | personal:edat | 30 | | fila2 | personal:nom | Jane | | fila2 | personal:edat | 25 |

Exemple de Codi: Creació d'una Taula

import org.apache.hadoop.hbase.HBaseConfiguration;
import org.apache.hadoop.hbase.client.Admin;
import org.apache.hadoop.hbase.client.Connection;
import org.apache.hadoop.hbase.client.ConnectionFactory;
import org.apache.hadoop.hbase.client.TableDescriptor;
import org.apache.hadoop.hbase.client.TableDescriptorBuilder;
import org.apache.hadoop.hbase.client.ColumnFamilyDescriptor;
import org.apache.hadoop.hbase.client.ColumnFamilyDescriptorBuilder;
import org.apache.hadoop.hbase.TableName;

public class CreateTableExample {
    public static void main(String[] args) throws Exception {
        // Crear configuració d'HBase
        org.apache.hadoop.conf.Configuration config = HBaseConfiguration.create();
        
        // Establir connexió
        try (Connection connection = ConnectionFactory.createConnection(config);
             Admin admin = connection.getAdmin()) {
            
            // Definir nom de la taula
            TableName tableName = TableName.valueOf("my_table");
            
            // Definir família de columnes
            ColumnFamilyDescriptor columnFamily = ColumnFamilyDescriptorBuilder.newBuilder("personal".getBytes()).build();
            
            // Definir descriptor de taula
            TableDescriptor tableDescriptor = TableDescriptorBuilder.newBuilder(tableName)
                    .setColumnFamily(columnFamily)
                    .build();
            
            // Crear taula
            admin.createTable(tableDescriptor);
            System.out.println("Taula creada amb èxit.");
        }
    }
}

Operacions CRUD a HBase

Inserir Dades

import org.apache.hadoop.hbase.client.Put;
import org.apache.hadoop.hbase.client.Table;

public class InsertDataExample {
    public static void main(String[] args) throws Exception {
        // Configuració i establiment de connexió (igual que anteriorment)
        
        // Obtenir taula
        Table table = connection.getTable(TableName.valueOf("my_table"));
        
        // Crear una nova fila
        Put p = new Put("row1".getBytes());
        p.addColumn("personal".getBytes(), "name".getBytes(), "John".getBytes());
        p.addColumn("personal".getBytes(), "age".getBytes(), "30".getBytes());
        
        // Inserir la fila
        table.put(p);
        System.out.println("Dades inserides amb èxit.");
        
        // Tancar taula
        table.close();
    }
}

Llegir Dades

import org.apache.hadoop.hbase.client.Get;
import org.apache.hadoop.hbase.client.Result;

public class ReadDataExample {
    public static void main(String[] args) throws Exception {
        // Configuració i establiment de connexió (igual que anteriorment)
        
        // Obtenir taula
        Table table = connection.getTable(TableName.valueOf("my_table"));
        
        // Crear una instància de Get
        Get g = new Get("row1".getBytes());
        
        // Llegir les dades
        Result result = table.get(g);
        byte[] name = result.getValue("personal".getBytes(), "name".getBytes());
        byte[] age = result.getValue("personal".getBytes(), "age".getBytes());
        
        // Imprimir el resultat
        System.out.println("Nom: " + new String(name));
        System.out.println("Edat: " + new String(age));
        
        // Tancar taula
        table.close();
    }
}

Actualitzar Dades

Actualitzar dades a HBase és similar a inserir dades. Utilitzeu la classe Put per sobreescriure els valors existents.

Esborrar Dades

import org.apache.hadoop.hbase.client.Delete;

public class DeleteDataExample {
    public static void main(String[] args) throws Exception {
        // Configuració i establiment de connexió (igual que anteriorment)
        
        // Obtenir taula
        Table table = connection.getTable(TableName.valueOf("my_table"));
        
        // Crear una instància de Delete
        Delete d = new Delete("row1".getBytes());
        
        // Esborrar la fila
        table.delete(d);
        System.out.println("Dades esborrades amb èxit.");
        
        // Tancar taula
        table.close();
    }
}

Funcionalitats Avançades d'HBase

Filtres

HBase proporciona un conjunt ric de filtres per recuperar dades específiques de la taula.

Coprocessadors

Els coprocessadors a HBase són similars als triggers i procediments emmagatzemats en bases de dades relacionals. Permeten executar codi personalitzat al servidor HBase.

Càrrega Massiva

Per carregar grans conjunts de dades, HBase proporciona mecanismes de càrrega massiva per millorar el rendiment.

Conclusió

Apache HBase és una base de dades NoSQL potent, escalable i flexible que s'integra perfectament amb l'ecosistema Hadoop. És ideal per a aplicacions que requereixen accés de lectura/escriptura en temps real a grans conjunts de dades. Comprenent els seus conceptes bàsics, model de dades i operacions CRUD, podeu aprofitar efectivament HBase per a les vostres aplicacions de grans dades. Les funcionalitats avançades com filtres, coprocessadors i càrrega massiva milloren encara més les seves capacitats, convertint-lo en una eina versàtil per gestionar diverses càrregues de treball de dades.
