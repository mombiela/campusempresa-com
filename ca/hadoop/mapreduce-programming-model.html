Introducció a MapReduce

MapReduce és un model de programació dissenyat per processar grans conjunts de dades amb un algorisme distribuït en un clúster. És un component central de l'ecosistema Hadoop.

  • Funció Map: Processa parells clau-valor d'entrada per generar un conjunt de parells clau-valor intermedis.
  • Funció Reduce: Fusiona tots els valors intermedis associats amb la mateixa clau intermèdia.

Conceptes Clau de MapReduce

Fase Map

  • Divisions d'Entrada: Les dades es divideixen en trossos més petits anomenats divisions.
  • Mapper: Cada divisió és processada per una funció map que genera parells clau-valor.

Fase de Shuffle i Sort

  • Shuffle: El procés de transferir dades de la fase map a la fase reduce.
  • Sort: Els parells clau-valor intermedis es classifiquen per clau.

Fase Reduce

  • Reducer: Processa els parells clau-valor ordenats i produeix la sortida final.

Exemple: Programa de Comptatge de Paraules

Exemple de Codi del Mapper

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Mapper;

public class WordCountMapper extends Mapper<LongWritable, Text, Text, IntWritable> {
    private final static IntWritable one = new IntWritable(1);
    private Text word = new Text();

    public void map(LongWritable key, Text value, Context context) throws IOException, InterruptedException {
        String line = value.toString();
        String[] words = line.split("\\s+");
        for (String str : words) {
            word.set(str);
            context.write(word, one);
        }
    }
}
  • Explicació:
    • El mètode map processa cada línia d'entrada.
    • Divideix la línia en paraules i emet cada paraula amb un compte d'un.

Exemple de Codi del Reducer

import java.io.IOException;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Reducer;

public class WordCountReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
    public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
        int sum = 0;
        for (IntWritable val : values) {
            sum += val.get();
        }
        context.write(key, new IntWritable(sum));
    }
}
  • Explicació:
    • El mètode reduce suma els comptes per a cada paraula.
    • Emmet la paraula i el seu compte total.

Conceptes Avançats de MapReduce

Combiner

  • Actua com un mini-reducer per realitzar l'agregació local de les sortides intermèdies, reduint la quantitat de dades transferides als reducers.

Partitioner

  • Controla la partició de les sortides intermèdies del map, assegurant que tots els valors per a una sola clau es dirigeixin al mateix reducer.

Comptadors

  • S'utilitzen per recollir estadístiques sobre la feina, com ara el nombre de registres processats.

Formats d'Entrada i Sortida Personalitzats

  • Permet processar diferents formats de dades i escriure sortides en diversos formats.

Optimització del Rendiment

Localitat de Dades

  • Assegurar que el càlcul es realitzi a prop d'on es guarden les dades per minimitzar la transferència de dades.

Execució Especulativa

  • Executar tasques duplicades per a tasques que s'executen lentament per reduir el temps de finalització de la feina.

Ajust de Paràmetres

  • Ajustar paràmetres com el nombre de mappers i reducers, l'assignació de memòria i la configuració d'I/O per optimitzar el rendiment.

Conclusió

El model de programació MapReduce és una eina poderosa per processar dades a gran escala en un entorn distribuït. Entenent els conceptes bàsics, escrivint funcions map i reduce eficients, i aprofitant les característiques avançades, pots utilitzar Hadoop de manera efectiva per a tasques de processament de grans dades.
