Introducció

En aquesta secció, cobrirem els passos necessaris per instal·lar Hadoop a la teva màquina local. Aquesta guia t'ajudarà a configurar un clúster Hadoop d'un sol node, que és ideal per a finalitats d'aprenentatge i desenvolupament.

Prerequisits

Abans d'instal·lar Hadoop, assegura't de tenir el següent:

  • Una màquina que executi un sistema operatiu basat en Unix (Linux, macOS).
  • Kit de Desenvolupament de Java (JDK) instal·lat (versió 8 o superior).
  • SSH instal·lat i configurat.

Guia d'Instal·lació Pas a Pas

1. Instal·lar Java

Hadoop requereix Java per funcionar. Pots comprovar si Java està instal·lat executant:

java -version

Si Java no està instal·lat, pots instal·lar-lo utilitzant les següents comandes:

Per a Ubuntu/Debian:

sudo apt update
sudo apt install openjdk-8-jdk

Per a CentOS/RHEL:

sudo yum install java-1.8.0-openjdk

2. Descarregar Hadoop

Descarrega l'última versió estable de Hadoop des del lloc web oficial d'Apache Hadoop:

wget https://downloads.apache.org/hadoop/common/hadoop-3.3.1/hadoop-3.3.1.tar.gz

3. Extreure Hadoop

Extreu el fitxer tarball descarregat a un directori de la teva elecció:

tar -xzvf hadoop-3.3.1.tar.gz
mv hadoop-3.3.1 /usr/local/hadoop

4. Configurar Variables d'Entorn

Afegeix Hadoop i Java al PATH del teu sistema editant el fitxer .bashrc:

nano ~/.bashrc

Afegeix les següents línies al final del fitxer:

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
export HADOOP_HOME=/usr/local/hadoop
export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin

Aplica els canvis:

source ~/.bashrc

5. Configurar Hadoop

Edita els fitxers de configuració de Hadoop situats al directori $HADOOP_HOME/etc/hadoop.

core-site.xml

<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>

hdfs-site.xml

<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
    <property>
        <name>dfs.namenode.name.dir</name>
        <value>file:///usr/local/hadoop/hadoop_data/hdfs/namenode</value>
    </property>
    <property>
        <name>dfs.datanode.data.dir</name>
        <value>file:///usr/local/hadoop/hadoop_data/hdfs/datanode</value>
    </property>
</configuration>

mapred-site.xml

<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>

yarn-site.xml

<configuration>
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
</configuration>

6. Formatjar el Sistema de Fitxers de Hadoop

Abans de començar Hadoop, formatja el sistema de fitxers de Hadoop (HDFS):

hdfs namenode -format

7. Iniciar Hadoop

Inicia els dimonis de Hadoop:

start-dfs.sh
start-yarn.sh

8. Verificar la Instal·lació

Comprova si els dimonis de Hadoop estan en funcionament:

jps

Hauries de veure els següents processos:

  • NameNode
  • DataNode
  • ResourceManager
  • NodeManager

Conclusió

Has instal·lat amb èxit Hadoop a la teva màquina local. Aquesta configuració d'un sol node és perfecta per aprendre i experimentar amb Hadoop. En les següents seccions, explorarem com utilitzar Hadoop per a diverses tasques de processament de dades.
