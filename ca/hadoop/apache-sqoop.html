Apache Sqoop és una eina dissenyada per transferir dades massives de manera eficient entre Apache Hadoop i magatzems de dades estructurats com bases de dades relacionals. Aquesta secció cobrirà els conceptes bàsics de Sqoop, la seva arquitectura, casos d'ús comuns i característiques avançades.

Introducció a Apache Sqoop

Apache Sqoop (SQL-to-Hadoop) és una eina potent que fa de pont entre Hadoop i bases de dades relacionals. Permet la importació i exportació de dades entre Hadoop i diversos sistemes RDBMS.

Conceptes Clau

  • Importació de Dades: Sqoop pot importar dades des de bases de dades relacionals al Hadoop Distributed File System (HDFS), Hive o HBase.
  • Exportació de Dades: Sqoop pot exportar dades des de HDFS, Hive o HBase de tornada a bases de dades relacionals.
  • Connectors: Sqoop utilitza connectors per interactuar amb diferents bases de dades. Connectors comuns inclouen MySQL, PostgreSQL, Oracle i Microsoft SQL Server.
  • Paral·lelisme: Sqoop pot realitzar transferències de dades en paral·lel, cosa que millora significativament el rendiment.

Instal·lació i Configuració de Sqoop

Prerequisits

  • Java Development Kit (JDK)
  • Configuració de Hadoop (HDFS, YARN)
  • Base de dades relacional (per exemple, MySQL)

Passos d'Instal·lació

  1. Descarregar Sqoop: Descarregar l'última versió de Sqoop des del lloc web d'Apache Sqoop.
  2. Extreure l'Arxiu: Extreure l'arxiu descarregat a un directori adequat.
  3. Configurar Variables d'Entorn: Afegir els binaris de Sqoop al PATH del sistema i configurar la variable d'entorn SQOOP_HOME.
export SQOOP_HOME=/path/to/sqoop
export PATH=$PATH:$SQOOP_HOME/bin
  1. Configurar Sqoop: Editar el fitxer sqoop-env.sh per configurar les configuracions de Hadoop i la base de dades.
export HADOOP_COMMON_HOME=/path/to/hadoop
export HADOOP_MAPRED_HOME=/path/to/hadoop
export HIVE_HOME=/path/to/hive

Comandes Bàsiques de Sqoop

Importació de Dades des de RDBMS a HDFS

sqoop import \
--connect jdbc:mysql://localhost/dbname \
--username root \
--password password \
--table tablename \
--target-dir /user/hadoop/tablename
  • --connect: Cadena de connexió JDBC a la base de dades.
  • --username: Nom d'usuari de la base de dades.
  • --password: Contrasenya de la base de dades.
  • --table: Nom de la taula a importar.
  • --target-dir: Directori HDFS on es desaran les dades.

Exportació de Dades des de HDFS a RDBMS

sqoop export \
--connect jdbc:mysql://localhost/dbname \
--username root \
--password password \
--table tablename \
--export-dir /user/hadoop/tablename
  • --export-dir: Directori HDFS que conté les dades a exportar.

Característiques Avançades de Sqoop

Importacions Incrementals

Sqoop suporta importacions incrementals, que permeten importar només les files noves o actualitzades d'una taula de base de dades.

sqoop import \
--connect jdbc:mysql://localhost/dbname \
--username root \
--password password \
--table tablename \
--incremental append \
--check-column id \
--last-value 100
  • --incremental: Mode d'importació incremental (append o lastmodified).
  • --check-column: Columna per verificar dades noves.
  • --last-value: Últim valor importat de la columna de verificació.

Importació de Dades a Hive

Sqoop pot importar dades directament a taules de Hive.

sqoop import \
--connect jdbc:mysql://localhost/dbname \
--username root \
--password password \
--table tablename \
--hive-import \
--create-hive-table \
--hive-table hive_db.hive_table
  • --hive-import: Importar dades directament a Hive.
  • --create-hive-table: Crear la taula de Hive si no existeix.
  • --hive-table: Nom de la taula de Hive.

Importació de Dades a HBase

Sqoop també pot importar dades a taules de HBase.

sqoop import \
--connect jdbc:mysql://localhost/dbname \
--username root \
--password password \
--table tablename \
--hbase-table hbase_table \
--column-family cf
  • --hbase-table: Nom de la taula de HBase.
  • --column-family: Família de columnes a HBase.

Casos d'Ús Comuns

  • Magatzem de Dades: Importació de dades des de bases de dades operacionals a Hadoop per a l'anàlisi.
  • Migració de Dades: Trasllat de dades entre diferents sistemes d'emmagatzematge.
  • Còpia de Seguretat i Recuperació: Exportació de dades des de Hadoop a bases de dades relacionals per a còpies de seguretat.

Conclusió

Apache Sqoop és una eina essencial per integrar Hadoop amb bases de dades relacionals. Simplifica el procés d'importació i exportació de dades, suporta importacions incrementals i pot interactuar directament amb Hive i HBase. Dominant Sqoop, pots gestionar eficientment les transferències de dades en un ecosistema Hadoop, fent que sigui una habilitat valuosa per a qualsevol enginyer de dades o administrador de Hadoop.
