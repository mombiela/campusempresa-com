Introducció a HDFS

Hadoop Distributed File System (HDFS) és el sistema d'emmagatzematge principal utilitzat per les aplicacions Hadoop. Proporciona accés d'alt rendiment a les dades de l'aplicació i està dissenyat per ser altament tolerant a fallades. Entendre les comandes HDFS és crucial per gestionar i interactuar amb el sistema de fitxers Hadoop.

Comandes bàsiques d'HDFS

Aquestes comandes són essencials per als principiants per començar amb HDFS.

Llistar fitxers i directoris

  • hdfs dfs -ls [path]: Llista fitxers i directoris en el camí especificat.
hdfs dfs -ls /user/hadoop

Aquesta comanda llista tots els fitxers i directoris sota /user/hadoop.

Crear directoris

  • hdfs dfs -mkdir [path]: Crea un nou directori en el camí especificat.
hdfs dfs -mkdir /user/hadoop/newdir

Aquesta comanda crea un nou directori anomenat newdir sota /user/hadoop.

Copiar fitxers a HDFS

  • hdfs dfs -put [local_path] [hdfs_path]: Copia fitxers del sistema de fitxers local a HDFS.
hdfs dfs -put /local/path/to/file.txt /user/hadoop/

Aquesta comanda copia file.txt del sistema de fitxers local al directori HDFS /user/hadoop/.

Veure el contingut dels fitxers

  • hdfs dfs -cat [path]: Mostra el contingut d'un fitxer.
hdfs dfs -cat /user/hadoop/file.txt

Aquesta comanda mostra el contingut de file.txt situat a /user/hadoop/.

Eliminar fitxers i directoris

  • hdfs dfs -rm [path]: Elimina un fitxer.
hdfs dfs -rm /user/hadoop/file.txt

Aquesta comanda elimina file.txt de /user/hadoop/.

  • hdfs dfs -rm -r [path]: Elimina un directori i el seu contingut de manera recursiva.
hdfs dfs -rm -r /user/hadoop/newdir

Aquesta comanda elimina el directori newdir i tot el seu contingut de /user/hadoop/.

Comandes intermèdies d'HDFS

Aquestes comandes proporcionen més control i són útils per gestionar HDFS de manera més efectiva.

Moure i canviar el nom dels fitxers

  • hdfs dfs -mv [source_path] [dest_path]: Mou o canvia el nom de fitxers i directoris.
hdfs dfs -mv /user/hadoop/file.txt /user/hadoop/renamed_file.txt

Aquesta comanda canvia el nom de file.txt a renamed_file.txt.

Copiar fitxers dins d'HDFS

  • hdfs dfs -cp [source_path] [dest_path]: Copia fitxers d'una ubicació HDFS a una altra.
hdfs dfs -cp /user/hadoop/file.txt /user/hadoop/copy_of_file.txt

Aquesta comanda copia file.txt a copy_of_file.txt dins del mateix directori.

Comprovar l'estat dels fitxers

  • hdfs dfs -stat [format] [path]: Mostra l'estat d'un fitxer o directori.
hdfs dfs -stat %b /user/hadoop/file.txt

Aquesta comanda mostra la mida en bytes de file.txt.

Canviar els permisos dels fitxers

  • hdfs dfs -chmod [permissions] [path]: Canvia els permisos d'un fitxer o directori.
hdfs dfs -chmod 755 /user/hadoop/file.txt

Aquesta comanda estableix els permisos de file.txt a 755.

Comandes avançades d'HDFS

Aquestes comandes són per a usuaris avançats que necessiten realitzar operacions complexes i optimitzacions.

Comprovar l'ús del disc

  • hdfs dfs -du [path]: Mostra l'ús del disc dels fitxers i directoris.
hdfs dfs -du /user/hadoop

Aquesta comanda mostra l'ús del disc de tots els fitxers i directoris sota /user/hadoop.

Balancejar el clúster HDFS

  • hdfs balancer: Balanceja les dades entre els DataNodes en el clúster HDFS.
hdfs balancer

Aquesta comanda inicia el procés de balanceig per assegurar una distribució uniforme de les dades entre els DataNodes.

Operacions de mode segur

  • hdfs dfsadmin -safemode [enter|leave|get]: Gestiona l'estat del mode segur d'HDFS.
hdfs dfsadmin -safemode enter

Aquesta comanda posa HDFS en mode segur, on no es permeten modificacions.

Gestió de snapshots

  • hdfs dfs -createSnapshot [path] [snapshotName]: Crea un snapshot d'un directori.
hdfs dfs -createSnapshot /user/hadoop mySnapshot

Aquesta comanda crea un snapshot anomenat mySnapshot del directori /user/hadoop.

  • hdfs dfs -deleteSnapshot [path] [snapshotName]: Elimina un snapshot.
hdfs dfs -deleteSnapshot /user/hadoop mySnapshot

Aquesta comanda elimina el snapshot anomenat mySnapshot.

Conclusió

Entendre i utilitzar efectivament les comandes HDFS és crucial per gestionar dades dins d'un ecosistema Hadoop. Des d'operacions bàsiques de fitxers fins a la gestió avançada del clúster, aquestes comandes proporcionen les eines necessàries per interactuar amb HDFS de manera eficient. Practiqueu aquestes comandes regularment per convertir-vos en experts en la gestió d'HDFS i aprofitar tot el seu potencial.
