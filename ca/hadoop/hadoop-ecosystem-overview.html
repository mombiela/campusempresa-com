Introducció

L'ecosistema Hadoop és un marc que permet el processament distribuït de grans conjunts de dades a través de clústers d'ordinadors utilitzant models de programació simples. Està dissenyat per escalar des de servidors únics fins a milers de màquines, cadascuna oferint computació i emmagatzematge locals.

Components clau de l'ecosistema Hadoop

Hadoop Common

  • Descripció: Les utilitats comunes que donen suport als altres mòduls de Hadoop.
  • Característiques clau:
    • Proporciona abstraccions de sistema de fitxers i nivell de sistema operatiu.
    • Conté les biblioteques i scripts Java necessaris.

Hadoop Distributed File System (HDFS)

  • Descripció: Un sistema de fitxers distribuït que proporciona accés d'alt rendiment a les dades de les aplicacions.
  • Característiques clau:
    • Emmagatzematge tolerant a fallades.
    • Dissenyat per funcionar amb maquinari de mercaderia.
    • Suporta grans conjunts de dades.

Hadoop YARN

  • Descripció: Una plataforma de gestió de recursos responsable de gestionar els recursos de computació en clústers i utilitzar-los per programar les aplicacions dels usuaris.
  • Característiques clau:
    • Assignació de recursos.
    • Programació de treballs.

Hadoop MapReduce

  • Descripció: Un model de programació per al processament de dades a gran escala.
  • Característiques clau:
    • Simplifica el processament de dades a través de grans clústers.
    • Gestiona la distribució de dades, la paral·lelització i la tolerància a fallades.

Components addicionals de l'ecosistema

Apache Hive

  • Descripció: Una infraestructura de magatzem de dades construïda sobre Hadoop per proporcionar resum de dades, consulta i anàlisi.
  • Característiques clau:
    • Llenguatge de consulta similar a SQL anomenat HiveQL.
    • Suporta l'anàlisi de grans conjunts de dades emmagatzemats a HDFS.

Apache HBase

  • Descripció: Un magatzem de dades distribuït, escalable i de grans dimensions.
  • Característiques clau:
    • Modelat després del Bigtable de Google.
    • Proporciona accés de lectura/escriptura en temps real i aleatori a les dades.

Apache Pig

  • Descripció: Una plataforma d'alt nivell per crear programes que s'executen a Hadoop.
  • Característiques clau:
    • Llenguatge de script Pig Latin.
    • Gestiona transformacions de dades complexes.

Apache Sqoop

  • Descripció: Una eina dissenyada per transferir dades massives de manera eficient entre Apache Hadoop i magatzems de dades estructurats com bases de dades relacionals.
  • Característiques clau:
    • Importació/exportació de dades entre Hadoop i RDBMS.
    • Suporta càrregues incrementals.

Apache Flume

  • Descripció: Un servei distribuït, fiable i disponible per recollir, agregar i moure grans quantitats de dades de registre de manera eficient.
  • Característiques clau:
    • Recollida de dades en streaming.
    • Alt rendiment.

Apache Oozie

  • Descripció: Un sistema de programació de flux de treball per gestionar treballs de Hadoop.
  • Característiques clau:
    • Programació de treballs.
    • Gestió de fluxos de treball.

Apache Zookeeper

  • Descripció: Un servei centralitzat per mantenir informació de configuració, nomenclatura, proporcionar sincronització distribuïda i serveis de grup.
  • Característiques clau:
    • Servei de coordinació.
    • Gestió de configuració.

Apache Kafka

  • Descripció: Una plataforma de streaming distribuïda.
  • Característiques clau:
    • Publicació i subscripció a fluxos de registres.
    • Emmagatzematge de fluxos de registres de manera tolerant a fallades.

Exemple: Programa simple de MapReduce

import java.io.IOException;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class WordCount {
    public static class TokenizerMapper extends Mapper<Object, Text, Text, IntWritable>{
        private final static IntWritable one = new IntWritable(1);
        private Text word = new Text();

        public void map(Object key, Text value, Context context) throws IOException, InterruptedException {
            String[] tokens = value.toString().split("\\s+");
            for (String token : tokens) {
                word.set(token);
                context.write(word, one);
            }
        }
    }

    public static class IntSumReducer extends Reducer<Text, IntWritable, Text, IntWritable> {
        private IntWritable result = new IntWritable();

        public void reduce(Text key, Iterable<IntWritable> values, Context context) throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            result.set(sum);
            context.write(key, result);
        }
    }

    public static void main(String[] args) throws Exception {
        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "word count");
        job.setJarByClass(WordCount.class);
        job.setMapperClass(TokenizerMapper.class);
        job.setCombinerClass(IntSumReducer.class);
        job.setReducerClass(IntSumReducer.class);
        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);
        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));
        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}

Explicació

  • Classe Mapper: Tokenitza el text d'entrada i emet cada paraula amb un recompte d'un.
  • Classe Reducer: Suma tots els recomptes per a cada paraula.
  • Mètode Main: Configura i executa el treball.

Conclusió

L'ecosistema Hadoop és un marc potent per gestionar grans dades. Inclou una varietat d'eines i components que treballen junts per proporcionar una solució integral per a l'emmagatzematge, el processament i l'anàlisi de dades. Entendre aquests components i com interactuen és crucial per aprofitar tot el potencial de Hadoop.
