Introducció

El disseny segur de sistemes és fonamental per protegir la integritat, confidencialitat i disponibilitat de la informació. Aquest tema aborda els principis bàsics que s'han de considerar en dissenyar sistemes segurs, proporcionant una base sòlida per a arquitectes de sistemes i altres professionals de TI.

Principis Fonamentals de Seguretat

Confidencialitat

  • Definició: Garantir que la informació només sigui accessible a les persones autoritzades.
  • Exemple: Ús de xifratge per protegir dades sensibles.
from cryptography.fernet import Fernet

# Generar una clau i xifrar un missatge
key = Fernet.generate_key()
cipher_suite = Fernet(key)
cipher_text = cipher_suite.encrypt(b"Missatge secret")
print(cipher_text)

Integritat

  • Definició: Assegurar que la informació no sigui alterada de manera no autoritzada.
  • Exemple: Ús de hashes per verificar la integritat de les dades.
import hashlib

# Crear un hash d'un missatge
message = b"Missatge important"
hash_object = hashlib.sha256(message)
hex_dig = hash_object.hexdigest()
print(hex_dig)

Disponibilitat

  • Definició: Garantir que els sistemes i dades estiguin disponibles per als usuaris autoritzats quan els necessitin.
  • Exemple: Implementació de redundància i balanceig de càrrega.
# Exemple conceptual de balanceig de càrrega
servers = ["Server1", "Server2", "Server3"]
def get_server(request):
    # Distribuir les sol·licituds de manera uniforme
    server = servers[hash(request) % len(servers)]
    return server

Principis de Disseny Segur

Principi de Privilegis Mínims

  • Definició: Els usuaris i sistemes han de tenir només els privilegis necessaris per realitzar les seves tasques.
  • Exemple: Configuració de permisos d'arxiu en un sistema Unix.
# Assignar permisos mínims necessaris a un arxiu
chmod 640 arxiu.txt

Defensa en Profunditat

  • Definició: Implementar múltiples capes de seguretat per protegir els sistemes.
  • Exemple: Ús combinat de firewalls, sistemes de detecció d'intrusos i autenticació multifactor.
# Exemple conceptual de configuració de firewall
sudo ufw allow 22/tcp  # Permetre SSH
sudo ufw allow 80/tcp  # Permetre HTTP
sudo ufw enable        # Habilitar el firewall

Seguretat per Disseny

  • Definició: Integrar la seguretat en totes les fases del cicle de vida del desenvolupament de sistemes.
  • Exemple: Realitzar revisions de seguretat en cada fase del desenvolupament.
| Fase del Desenvolupament | Activitat de Seguretat |
|--------------------------|------------------------|
| Requisits                | Anàlisi d'amenaces     |
| Disseny                  | Modelatge d'amenaces   |
| Implementació            | Revisions de codi      |
| Proves                   | Proves de penetració   |
| Desplegament             | Auditories de seguretat|

Principi de Falla Segura

  • Definició: Els sistemes han de fallar de manera segura, sense comprometre la seguretat.
  • Exemple: Configuració de polítiques de bloqueig de comptes després de múltiples intents fallits d'inici de sessió.
# Exemple conceptual de bloqueig de compte
class User:
    def __init__(self, username):
        self.username = username
        self.failed_attempts = 0
        self.locked = False

    def login(self, password):
        if self.locked:
            return "Compte bloquejada"
        if password == "correct_password":
            self.failed_attempts = 0
            return "Inici de sessió exitós"
        else:
            self.failed_attempts += 1
            if self.failed_attempts >= 3:
                self.locked = True
            return "Contrasenya incorrecta"

Conclusió

El disseny segur de sistemes és una pràctica essencial per protegir la informació i els recursos d'una organització. En aplicar principis com la confidencialitat, integritat, disponibilitat, privilegis mínims, defensa en profunditat, seguretat per disseny i falla segura, els arquitectes de sistemes poden crear entorns més segurs i resilients. Aquests principis han de ser considerats en cada fase del cicle de vida del desenvolupament per garantir una protecció integral i efectiva.
