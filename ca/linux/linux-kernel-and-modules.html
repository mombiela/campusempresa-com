Introducció al Nucli Linux

El nucli Linux és el component central del sistema operatiu Linux. Gestiona el maquinari i els recursos del sistema, permetent que el programari interactuï amb el maquinari de manera controlada.

  • Conceptes Clau:
    • Espai del Nucli vs. Espai d'Usuari: El nucli opera en un mode privilegiat anomenat espai del nucli, mentre que les aplicacions s'executen en l'espai d'usuari.
    • Nucli Monolític: El nucli Linux és monolític, el que significa que inclou tots els controladors i serveis del sistema necessaris dins d'un únic binari gran.
    • Crides al Sistema: Interfícies proporcionades pel nucli perquè les aplicacions en espai d'usuari puguin sol·licitar serveis.

Entendre els Mòduls del Nucli

Els mòduls del nucli són peces de codi que es poden carregar i descarregar al nucli segons la demanda. Amplien la funcionalitat del nucli sense la necessitat de reiniciar el sistema.

  • Conceptes Clau:
    • Càrrega Dinàmica: Els mòduls es poden carregar i descarregar dinàmicament utilitzant comandes com insmod i rmmod.
    • Tipus de Mòduls: Controladors de dispositius, controladors de sistemes de fitxers, controladors de xarxa, etc.
    • Dependències de Mòduls: Els mòduls poden dependre d'altres mòduls, que han de ser carregats primer.

Treballar amb Mòduls del Nucli

Carregar i Descarregar Mòduls

Els mòduls es poden gestionar utilitzant diverses comandes.

  • Carregar un Mòdul:

    sudo insmod mymodule.ko

    Aquesta comanda insereix el mòdul mymodule.ko al nucli.

  • Descarregar un Mòdul:

    sudo rmmod mymodule

    Aquesta comanda elimina el mòdul mymodule del nucli.

  • Llistar Mòduls Carregats:

    lsmod

    Aquesta comanda llista tots els mòduls actualment carregats.

Escriure un Mòdul de Nucli Simple

Escrivim un mòdul de nucli "Hola Món" simple.

  • hello.c:

    #include <linux/module.h>
#include <linux/kernel.h>

int init_module(void) {
    printk(KERN_INFO "Hola, món!\n");
    return 0;
}

void cleanup_module(void) {
    printk(KERN_INFO "Adéu, món!\n");
}

MODULE_LICENSE("GPL");
MODULE_DESCRIPTION("Un mòdul simple Hola Món");
MODULE_AUTHOR("El teu Nom");

  • Compilar el Mòdul: Crear un Makefile per compilar el mòdul.

    obj-m += hello.o

all:
    make -C /lib/modules/$(shell uname -r)/build M=$(PWD) modules

clean:
    make -C /lib/modules/$(shell uname -r)/build M=$(PWD) clean

    Executa make per compilar el mòdul.

  • Carregar i Provar el Mòdul:

    sudo insmod hello.ko
dmesg | tail
sudo rmmod hello
dmesg | tail

    La comanda dmesg mostra missatges del nucli, on hauríeu de veure els missatges "Hola, món!" i "Adéu, món!".

Temes Avançats

Paràmetres del Mòdul de Nucli

Els mòduls poden acceptar paràmetres en el moment de la càrrega.

  • Exemple amb Paràmetres:

    #include <linux/module.h>
#include <linux/kernel.h>
#include <linux/init.h>

static int myint = 0;
module_param(myint, int, 0);
MODULE_PARM_DESC(myint, "Un enter");

int init_module(void) {
    printk(KERN_INFO "Hola, món! myint = %d\n", myint);
    return 0;
}

void cleanup_module(void) {
    printk(KERN_INFO "Adéu, món!\n");
}

MODULE_LICENSE("GPL");
MODULE_DESCRIPTION("Un mòdul simple amb paràmetres");
MODULE_AUTHOR("El teu Nom");

    Carrega el mòdul amb un paràmetre:

    sudo insmod hello.ko myint=42
dmesg | tail

Depurar Mòduls del Nucli

Depurar mòduls del nucli pot ser un desafiament a causa de la manca d'eines de depuració estàndard.

  • Utilitzant printk: El mètode principal per depurar és utilitzar declaracions printk.
  • Depurador del Nucli (KGDB): Una eina més avançada per depurar codi del nucli.

Conclusió

Entendre el nucli Linux i els seus mòduls és crucial per a la programació avançada en Linux i l'administració de sistemes. Els mòduls del nucli proporcionen una manera flexible d'ampliar la funcionalitat del nucli sense reiniciar el sistema. Aprenent a escriure, carregar i depurar mòduls del nucli, obtens una comprensió més profunda del funcionament intern del sistema operatiu Linux.
