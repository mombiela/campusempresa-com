Introducció

La gestió de discs és un aspecte crucial de l'administració de sistemes en Linux. Inclou tasques com la partició, el format, el muntatge i el monitoratge de l'ús del disc. Aquesta secció et guiarà des de tècniques bàsiques fins a avançades de gestió de discs en Linux.

Conceptes Bàsics

Entendre les Particions

  • Partició: Una divisió d'un disc dur (HDD) o unitat d'estat sòlid (SSD) en seccions separades i independents.
  • Tipus de Particions:
    • Partició Primària: Pot ser utilitzada per arrencar el sistema operatiu.
    • Partició Estesa: Un tipus especial de partició primària que pot contenir múltiples particions lògiques.
    • Partició Lògica: Subdivisions dins d'una partició estesa.

Sistemes de Fitxers

  • Sistema de Fitxers: Un mètode per emmagatzemar i organitzar fitxers en un disc.
  • Sistemes de fitxers comuns en Linux:
    • ext4: Quart sistema de fitxers extès, àmpliament utilitzat.
    • XFS: Sistema de fitxers d'alt rendiment.
    • Btrfs: Sistema de fitxers amb funcions avançades com instantànies.

Dispositius de Disc

  • Els dispositius de disc es representen en Linux com a fitxers al directori /dev.
    • Exemple: /dev/sda (primer disc SATA), /dev/sdb (segon disc SATA).

Comandes Bàsiques de Gestió de Discs

Veure Informació del Disc

  • lsblk: Llista informació sobre tots els dispositius de bloc disponibles o especificats. 
    lsblk
  • fdisk -l: Llista totes les particions en tots els discs. 
    sudo fdisk -l

Crear Particions

  • fdisk: Una utilitat de línia de comandes per crear i manipular taules de particions. 
    sudo fdisk /dev/sda
    • Segueix les indicacions interactives per crear una nova partició.

Formatar Particions

  • mkfs: Utilitzat per crear un sistema de fitxers en una partició. 
    sudo mkfs.ext4 /dev/sda1

Muntar i Desmuntar Particions

  • mount: Munta un sistema de fitxers. 
    sudo mount /dev/sda1 /mnt
  • umount: Desmunta un sistema de fitxers. 
    sudo umount /mnt

Gestió de Discs Intermèdia

Muntatges Persistents

  • Per assegurar que un sistema de fitxers es munta a l'arrencada, afegeix una entrada a /etc/fstab. 
    sudo nano /etc/fstab
    • Exemple d'entrada: 
      /dev/sda1  /mnt  ext4  defaults  0  2

Quotes de Disc

  • Quota: Limita la quantitat d'espai de disc i el nombre de fitxers que un usuari o grup pot utilitzar.
  • Activa les quotes en un sistema de fitxers: 
    sudo mount -o remount,usrquota,grpquota /mnt
sudo quotacheck -cug /mnt
sudo quotaon /mnt

LVM (Gestor de Volums Lògics)

  • LVM: Permet una gestió flexible dels discs.
    • Crea un volum físic: 
      sudo pvcreate /dev/sda1
    • Crea un grup de volums: 
      sudo vgcreate myvg /dev/sda1
    • Crea un volum lògic: 
      sudo lvcreate -L 10G -n mylv myvg
    • Formata i munta el volum lògic: 
      sudo mkfs.ext4 /dev/myvg/mylv
sudo mount /dev/myvg/mylv /mnt

Gestió de Discs Avançada

RAID (Conjunt Redundant de Discs Independents)

  • RAID: Combina múltiples discs en una sola unitat per redundància o rendiment.
  • Crea un array RAID 1 utilitzant mdadm: 
    sudo mdadm --create --verbose /dev/md0 --level=1 --raid-devices=2 /dev/sda1 /dev/sdb1

Ajust de Sistemes de Fitxers

  • tune2fs: Ajusta paràmetres del sistema de fitxers en sistemes de fitxers ext2/ext3/ext4. 
    sudo tune2fs -m 1 /dev/sda1  # Estableix el percentatge de blocs reservats al 1%

Monitoratge de l'Ús del Disc

  • df: Informa sobre l'ús de l'espai del sistema de fitxers. 
    df -h
  • du: Estima l'ús de l'espai dels fitxers. 
    du -sh /path/to/directory

Conclusió

La gestió de discs en Linux abasta des de tasques bàsiques com la partició i el format fins a tècniques avançades com LVM i RAID. Dominar aquestes habilitats assegura una gestió eficient i fiable de l'emmagatzematge, que és crucial tant per a entorns personals com empresarials. Entenent i utilitzant les eines i comandes discutides, pots gestionar efectivament els recursos de disc en un sistema Linux.
