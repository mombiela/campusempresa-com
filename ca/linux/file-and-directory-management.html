Introducció

La gestió de fitxers i directoris és una habilitat fonamental per a qualsevol usuari de Linux. Aquesta secció cobrirà els conceptes bàsics de la navegació pel sistema de fitxers, la creació, eliminació i gestió de fitxers i directoris, i la comprensió dels permisos.

Navegant pel Sistema de Fitxers

  • pwd: Imprimir el Directori de Treball Actual
  • ls: Llistar el contingut del directori
  • cd: Canviar de directori

Exemples

# Imprimir el directori de treball actual
pwd

# Llistar el contingut del directori actual
ls

# Canviar al directori /home
cd /home

Gestió de Fitxers

  • touch: Crear un fitxer buit o actualitzar la marca de temps d'un fitxer existent
  • cp: Copiar fitxers o directoris
  • mv: Moure o canviar el nom de fitxers o directoris
  • rm: Eliminar fitxers o directoris

Exemples

# Crear un fitxer buit anomenat example.txt
touch example.txt

# Copiar example.txt a un nou fitxer anomenat example_copy.txt
cp example.txt example_copy.txt

# Moure example.txt a una nova ubicació o canviar-li el nom
mv example.txt /home/user/new_example.txt

# Eliminar example_copy.txt
rm example_copy.txt

Gestió de Directoris

  • mkdir: Crear un nou directori
  • rmdir: Eliminar un directori buit
  • rm -r: Eliminar un directori i el seu contingut de manera recursiva

Exemples

# Crear un nou directori anomenat new_directory
mkdir new_directory

# Eliminar un directori buit anomenat new_directory
rmdir new_directory

# Eliminar un directori i el seu contingut de manera recursiva
rm -r new_directory

Visualització i Edició de Fitxers

  • cat: Concatenar i mostrar el contingut del fitxer
  • less: Veure el contingut del fitxer una pantalla a la vegada
  • nano: Editor de text simple
  • vim: Editor de text avançat

Exemples

# Mostrar el contingut d'un fitxer anomenat example.txt
cat example.txt

# Veure el contingut d'un fitxer anomenat example.txt una pantalla a la vegada
less example.txt

# Obrir example.txt en l'editor nano
nano example.txt

# Obrir example.txt en l'editor vim
vim example.txt

Comprensió dels Permisos

  • chmod: Canviar el mode del fitxer (permisos)
  • chown: Canviar el propietari i el grup del fitxer
  • ls -l: Llistar fitxers amb informació detallada incloent permisos

Taula de Permisos

| Símbol | Descripció | |--------|-------------| | r | Llegir | | w | Escriure | | x | Executar |

Exemples

# Llistar fitxers amb informació detallada
ls -l

# Canviar els permisos de example.txt a llegir, escriure i executar per al propietari, i només llegir per al grup i altres
chmod 755 example.txt

# Canviar el propietari de example.txt a l'usuari 'john' i el grup a 'staff'
chown john:staff example.txt

Conclusió

Comprendre la gestió de fitxers i directoris és crucial per a un ús efectiu de Linux. Aquesta secció ha cobert els conceptes bàsics de la navegació pel sistema de fitxers, la gestió de fitxers i directoris, la visualització i edició de fitxers, i la comprensió dels permisos. Dominar aquests comandaments millorarà significativament la teva capacitat per treballar de manera eficient en un entorn Linux.
