Visió General

L'shell scripting és una manera poderosa d'automatitzar tasques en un entorn Linux. Aquest tema et guiarà a través dels conceptes bàsics de l'shell scripting, des d'entendre què és una shell fins a escriure el teu primer script i més enllà.

Què és una Shell?

  • Definició: Una shell és un intèrpret de línia de comandes que proporciona una interfície d'usuari per al sistema operatiu Unix/Linux.
  • Tipus de Shells:
    • Bourne Shell (sh)
    • Bourne Again Shell (bash)
    • C Shell (csh)
    • Korn Shell (ksh)
    • Z Shell (zsh)

Per què Utilitzar Shell Scripting?

  • Automatització: Automatitzar tasques repetitives.
  • Eficiència: Realitzar tasques complexes amb comandes simples.
  • Personalització: Adaptar el teu entorn a les teves necessitats.
  • Integració: Combinar múltiples comandes i utilitats.

Comandes Bàsiques de Shell

  • Navegació: cd, ls, pwd
  • Operacions amb Fitxers: cp, mv, rm, touch
  • Visualització de Fitxers: cat, more, less, head, tail
  • Permisos: chmod, chown

Escriure el Teu Primer Script de Shell

  1. Crear un Fitxer de Script: 
    touch myscript.sh
  2. Fer el Script Executable: 
    chmod +x myscript.sh
  3. Editar el Script: 
    nano myscript.sh
  4. Afegir la Línia Shebang: La línia shebang indica al sistema quin intèrpret utilitzar. 
    #!/bin/bash
  5. Afegir Comandes: 
    #!/bin/bash
echo "Hola, Món!"
  6. Executar el Script: 
    ./myscript.sh

Variables

  • Definició: Les variables emmagatzemen dades que es poden utilitzar i manipular dins d'un script.
  • Sintaxi: 
    VARIABLE_NAME="valor"
echo $VARIABLE_NAME

Declaracions Condicionals

  • If-Else: 
    #!/bin/bash
if [ $1 -gt 10 ]; then
    echo "El número és més gran que 10"
else
    echo "El número és 10 o menys"
fi
  • Case: 
    #!/bin/bash
case $1 in
    start)
        echo "Iniciant..."
        ;;
    stop)
        echo "Aturant..."
        ;;
    *)
        echo "Ús: $0 {start|stop}"
        ;;
esac

Bucles

  • Bucle For: 
    #!/bin/bash
for i in {1..5}; do
    echo "Benvingut $i vegades"
done
  • Bucle While: 
    #!/bin/bash
COUNTER=0
while [ $COUNTER -lt 5 ]; do
    echo "Comptador: $COUNTER"
    COUNTER=$((COUNTER + 1))
done

Funcions

  • Definició: Les funcions són blocs de codi reutilitzables.
  • Sintaxi: 
    #!/bin/bash
my_function() {
    echo "Això és una funció"
}
my_function

Entrada i Sortida

  • Llegir Entrada de l'Usuari: 
    #!/bin/bash
echo "Introdueix el teu nom:"
read NAME
echo "Hola, $NAME"
  • Redirigir Sortida: 
    #!/bin/bash
echo "Això és una prova" > output.txt

Gestió d'Errors

  • Estat de Sortida: Cada comanda retorna un estat de sortida (0 per èxit, no zero per fallada). 
    #!/bin/bash
cp file1.txt file2.txt
if [ $? -eq 0 ]; then
    echo "Còpia exitosa"
else
    echo "Còpia fallida"
fi

Temes Avançats

  • Arrays: 
    #!/bin/bash
NAMES=("Alice" "Bob" "Charlie")
echo "Primer nom: ${NAMES[0]}"
  • Expressions Regulars: 
    #!/bin/bash
if [[ "hello" =~ ^h.*o$ ]]; then
    echo "El patró coincideix"
else
    echo "El patró no coincideix"
fi
  • Depuració: Utilitza set -x per habilitar la depuració. 
    #!/bin/bash
set -x
echo "Depuració habilitada"

Conclusió

L'shell scripting és una habilitat essencial per a qualsevol persona que treballi en un entorn Linux. Et permet automatitzar tasques, gestionar operacions del sistema i personalitzar el teu flux de treball. Entenent els conceptes bàsics i practicant amb exemples, pots convertir-te en un expert en escriure i utilitzar scripts de shell per millorar la teva productivitat.
