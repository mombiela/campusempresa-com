Introducció

Instal·lar Linux és el primer pas per dominar aquest potent i versàtil sistema operatiu. Aquesta secció et guiarà a través del procés d'instal·lació de Linux al teu ordinador, proporcionant instruccions clares i exemples per assegurar una experiència d'instal·lació suau.

Triar una Distribució de Linux

Abans d'instal·lar Linux, necessites triar una distribució (distro) que s'adapti a les teves necessitats. Les distribucions populars inclouen:

  • Ubuntu: Fàcil d'usar, ideal per a principiants.
  • Fedora: Característiques avançades, ideal per a desenvolupadors.
  • Debian: Estable i fiable, preferida per a servidors.
  • Arch Linux: Altament personalitzable, per a usuaris avançats.

Preparació per a la Instal·lació

Requisits del Sistema

Assegura't que el teu sistema compleixi els requisits mínims per a la distribució escollida. Per exemple, els requisits d'Ubuntu són:

  • Processador: Processador de doble nucli de 2 GHz o millor.
  • Memòria: 4 GB de RAM.
  • Emmagatzematge: 25 GB d'espai lliure al disc dur.
  • Internet: L'accés a Internet és útil per a les actualitzacions.

Crear una Unitat USB d'Arrencada

Per instal·lar Linux, necessites crear una unitat USB d'arrencada. Segueix aquests passos:

  1. Descarrega el fitxer ISO: Visita el lloc web oficial de la distribució escollida i descarrega el fitxer ISO.
  2. Utilitza una eina per crear la unitat USB d'arrencada: Eines com Rufus (Windows), Etcher (multiplataforma) o la comanda dd (Linux) poden ser utilitzades.

Exemple: Utilitzant Rufus a Windows

  1. Insereix una unitat USB al teu ordinador.
  2. Obre Rufus.
  3. Selecciona la unitat USB sota "Dispositiu".
  4. Fes clic a "Seleccionar" i tria el fitxer ISO descarregat.
  5. Fes clic a "Iniciar" i espera que el procés es completi.
# Exemple: Utilitzant la comanda `dd` a Linux
sudo dd if=/path/to/your.iso of=/dev/sdX bs=4M status=progress && sync

Substitueix /path/to/your.iso amb el camí al teu fitxer ISO i /dev/sdX amb l'identificador de la teva unitat USB (per exemple, /dev/sdb).

Instal·lant Linux

Arrencant des de la Unitat USB

  1. Insereix la unitat USB d'arrencada al teu ordinador.
  2. Reinicia el teu ordinador i entra a la configuració del BIOS/UEFI (normalment prement una tecla com F2, F12, ESC o DEL durant l'inici).
  3. Canvia l'ordre d'arrencada per prioritzar la unitat USB.
  4. Desa i surt de la configuració del BIOS/UEFI.

Procés d'Instal·lació

  1. Inicia l'instal·lador: Un cop arrencat des de la unitat USB, veuràs l'instal·lador de Linux. Selecciona "Instal·lar" per començar.
  2. Tria l'idioma i la disposició del teclat: Segueix les indicacions per seleccionar el teu idioma preferit i la disposició del teclat.
  3. Prepara el disc: Tria com vols particionar el teu disc. Per a principiants, l'opció "Esborra el disc i instal·la Linux" és recomanada.
  4. Configura el compte d'usuari: Introdueix el teu nom, nom de l'ordinador, nom d'usuari i contrasenya.
  5. Instal·la: Fes clic a "Instal·lar Ara" i espera que la instal·lació es completi.

Configuració Post-Instal·lació

  1. Elimina la unitat USB: Després de la instal·lació, elimina la unitat USB i reinicia el teu ordinador.
  2. Actualitza el sistema: Obre una terminal i executa les següents comandes per actualitzar el teu sistema:
sudo apt update
sudo apt upgrade

Conclusió

Instal·lar Linux és un procés senzill que obre la porta a un sistema operatiu potent i flexible. Triant la distribució adequada, preparant el teu sistema i seguint els passos d'instal·lació, pots configurar Linux amb èxit al teu ordinador. Un cop instal·lat, recorda mantenir el teu sistema actualitzat i explorar la gran varietat de programari disponible per millorar la teva experiència amb Linux.
