Introducció

La gestió d'usuaris i grups és un aspecte fonamental de l'administració de sistemes Linux. Implica crear, modificar i eliminar comptes d'usuari i grups, així com gestionar permisos i controls d'accés.

Conceptes Clau

  • Usuaris: Comptes individuals que poden iniciar sessió al sistema.
  • Grups: Col·leccions d'usuaris que comparteixen permisos comuns.
  • UID (Identificador d'Usuari): Un identificador únic assignat a cada usuari.
  • GID (Identificador de Grup): Un identificador únic assignat a cada grup.
  • Directori Personal: El directori per defecte assignat a un usuari.
  • Shell: La interfície de línia de comandes assignada a un usuari.

Gestió d'Usuaris

Crear un Usuari

Per crear un nou usuari, utilitza la comanda useradd:

sudo useradd -m -s /bin/bash nomusuari
  • -m: Crea un directori personal per a l'usuari.
  • -s: Especifica el shell per defecte de l'usuari.

Establir una Contrasenya

Estableix una contrasenya per al nou usuari utilitzant la comanda passwd:

sudo passwd nomusuari

Modificar un Usuari

Per modificar un usuari existent, utilitza la comanda usermod:

sudo usermod -s /bin/zsh nomusuari
  • -s: Canvia el shell per defecte de l'usuari.

Eliminar un Usuari

Per eliminar un usuari, utilitza la comanda userdel:

sudo userdel -r nomusuari
  • -r: Elimina el directori personal i el correu de l'usuari.

Gestió de Grups

Crear un Grup

Per crear un nou grup, utilitza la comanda groupadd:

sudo groupadd nomgrup

Afegir un Usuari a un Grup

Per afegir un usuari a un grup, utilitza la comanda usermod amb l'opció -aG:

sudo usermod -aG nomgrup nomusuari
  • -aG: Afegeix l'usuari al grup especificat.

Eliminar un Usuari d'un Grup

Per eliminar un usuari d'un grup, utilitza la comanda gpasswd:

sudo gpasswd -d nomusuari nomgrup

Eliminar un Grup

Per eliminar un grup, utilitza la comanda groupdel:

sudo groupdel nomgrup

Veure Informació d'Usuaris i Grups

Llistar Usuaris

Per llistar tots els usuaris, visualitza el fitxer /etc/passwd:

cat /etc/passwd

Llistar Grups

Per llistar tots els grups, visualitza el fitxer /etc/group:

cat /etc/group

Comprovar la Pertinença a Grups

Per comprovar a quins grups pertany un usuari, utilitza la comanda groups:

groups nomusuari

Exemples Pràctics

Exemple: Crear un Nou Usuari i Afegir-lo a un Grup

  1. Crear un nou usuari: 
    sudo useradd -m -s /bin/bash alice
  2. Establir una contrasenya per al nou usuari: 
    sudo passwd alice
  3. Crear un nou grup: 
    sudo groupadd developers
  4. Afegir el nou usuari al grup: 
    sudo usermod -aG developers alice

Exemple: Modificar la Informació d'un Usuari

  1. Canviar el shell de l'usuari: 
    sudo usermod -s /bin/zsh alice

Exemple: Eliminar un Usuari i un Grup

  1. Eliminar l'usuari i el seu directori personal: 
    sudo userdel -r alice
  2. Eliminar el grup: 
    sudo groupdel developers

Resum

La gestió d'usuaris i grups a Linux és essencial per mantenir la seguretat i l'organització del sistema. Entenent com crear, modificar i eliminar usuaris i grups, així com gestionar els seus permisos, pots controlar efectivament l'accés als recursos del sistema. Practica aquestes comandes per convertir-te en un expert en la gestió d'usuaris i grups en un sistema Linux.
