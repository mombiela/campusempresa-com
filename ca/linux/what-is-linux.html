Introducció a Linux

Linux és un sistema operatiu potent i de codi obert que s'utilitza àmpliament en diversos entorns, des d'ordinadors personals fins a servidors i sistemes encastats. És conegut per la seva estabilitat, seguretat i flexibilitat.

Conceptes clau de Linux

  • Codi Obert: Linux és un programari de codi obert, el que significa que el seu codi font està disponible lliurement per a qualsevol persona per veure, modificar i distribuir.
  • Nucli: La part central del sistema operatiu Linux que gestiona els recursos de maquinari i permet l'execució de programari.
  • Distribucions (Distros): Variants de Linux que empaqueten el nucli amb diferents programes i eines. Exemples inclouen Ubuntu, Fedora i Debian.
  • Interfície de Línia de Comandes (CLI): Una interfície basada en text utilitzada per interactuar amb el sistema operatiu escrivint comandes.

Història de Linux

  • Creació: Linux va ser creat per Linus Torvalds el 1991 com una alternativa lliure i de codi obert al sistema operatiu MINIX.
  • Desenvolupament: Des de llavors ha crescut gràcies a les contribucions de desenvolupadors d'arreu del món, donant lloc a un sistema operatiu robust i versàtil.

Nucli de Linux

El nucli és el cor del sistema operatiu Linux. Gestiona els recursos del sistema i permet la comunicació entre el maquinari i el programari.

Funcions clau del Nucli

  • Gestió de Processos: Maneja la creació, planificació i terminació de processos.
  • Gestió de Memòria: Gestiona l'assignació i desassignació de memòria.
  • Controladors de Dispositius: Facilita la comunicació entre el sistema operatiu i els dispositius de maquinari.
  • Gestió del Sistema de Fitxers: Gestiona l'emmagatzematge, recuperació i organització de dades en unitats de disc.

Distribucions de Linux

Les distribucions de Linux són diferents versions del sistema operatiu Linux, cadascuna adaptada a necessitats i preferències específiques.

Distribucions Populars de Linux

| Distribució | Descripció | |--------------|-------------| | Ubuntu | Fàcil d'usar, popular per a escriptoris i principiants. | | Fedora | Funcions d'avantguarda, sovint utilitzada per desenvolupadors. | | Debian | Coneguda per la seva estabilitat i extens repositori de programari. | | CentOS | Versió suportada per la comunitat de Red Hat Enterprise Linux, utilitzada per a servidors. | | Arch Linux | Minimalista i altament personalitzable, preferida per usuaris avançats. |

Interfície de Línia de Comandes (CLI)

La CLI és una eina potent per interactuar amb Linux. Permet als usuaris executar comandes escrivint-les en un terminal.

Comandes Bàsiques

  • ls: Llista fitxers i directoris en el directori actual. 
    ls
  • cd: Canvia el directori actual. 
    cd /path/to/directory
  • pwd: Mostra el directori de treball actual. 
    pwd
  • mkdir: Crea un nou directori. 
    mkdir new_directory
  • rm: Elimina fitxers o directoris. 
    rm file_name

Conclusió

Linux és un sistema operatiu versàtil i potent que ofereix nombrosos beneficis, incloent estabilitat, seguretat i flexibilitat. Entendre els seus components bàsics, com el nucli i la interfície de línia de comandes, és essencial per utilitzar i gestionar efectivament els sistemes Linux. Tant si ets un principiant com un usuari avançat, Linux proporciona un entorn ric per a l'aprenentatge i el desenvolupament.
