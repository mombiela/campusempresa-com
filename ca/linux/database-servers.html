Introducció als Servidors de Bases de Dades

Els servidors de bases de dades són components crítics en moltes aplicacions, proporcionant una ubicació centralitzada per emmagatzemar, gestionar i recuperar dades. En aquesta secció, explorarem els conceptes bàsics dels servidors de bases de dades, la seva importància i com s'integren amb sistemes Linux.

Conceptes Clau

  • Servidor de Base de Dades: Un servidor que utilitza una aplicació de base de dades per proporcionar serveis de base de dades a altres programes o ordinadors.
  • RDBMS (Sistema de Gestió de Bases de Dades Relacionals): Un tipus de sistema de gestió de bases de dades que emmagatzema dades en taules i permet relacions entre taules.
  • SQL (Llenguatge de Consulta Estructurat): Un llenguatge estàndard per gestionar i manipular bases de dades.

Servidors de Bases de Dades Populars en Linux

Diversos servidors de bases de dades són àmpliament utilitzats en l'entorn Linux. Aquí, cobrirem alguns dels més populars.

MySQL

MySQL és un RDBMS de codi obert conegut per la seva fiabilitat i facilitat d'ús.

Instal·lació

Per instal·lar MySQL en un sistema Linux, podeu utilitzar les següents comandes:

sudo apt update
sudo apt install mysql-server

Comandes Bàsiques

  • Iniciar el Servei MySQL: 
    sudo systemctl start mysql
  • Assegurar la Instal·lació de MySQL: 
    sudo mysql_secure_installation
  • Iniciar Sessió a MySQL: 
    mysql -u root -p

PostgreSQL

PostgreSQL és un sistema de bases de dades objecte-relacional de codi obert i potent.

Instal·lació

Per instal·lar PostgreSQL en un sistema Linux, utilitzeu les següents comandes:

sudo apt update
sudo apt install postgresql postgresql-contrib

Comandes Bàsiques

  • Iniciar el Servei PostgreSQL: 
    sudo systemctl start postgresql
  • Iniciar Sessió a PostgreSQL: 
    sudo -i -u postgres
psql

SQLite

SQLite és una biblioteca en llenguatge C que implementa un motor de base de dades SQL petit, ràpid, autònom, d'alta fiabilitat i amb totes les funcions.

Ús

SQLite no requereix instal·lació ja que és una biblioteca. Podeu utilitzar-lo directament invocant la comanda sqlite3:

sqlite3 mydatabase.db

Gestió de Servidors de Bases de Dades

Gestionar servidors de bases de dades implica diverses tasques, incloent iniciar i aturar serveis, crear i gestionar bases de dades, i assegurar la seguretat.

Iniciar i Aturar Serveis

  • MySQL: 
    sudo systemctl start mysql
sudo systemctl stop mysql
  • PostgreSQL: 
    sudo systemctl start postgresql
sudo systemctl stop postgresql

Crear i Gestionar Bases de Dades

  • MySQL: 
    CREATE DATABASE mydatabase;
USE mydatabase;
CREATE TABLE users (id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(100));
INSERT INTO users (name) VALUES ('Alice'), ('Bob');
SELECT * FROM users;
  • PostgreSQL: 
    CREATE DATABASE mydatabase;
\c mydatabase
CREATE TABLE users (id SERIAL PRIMARY KEY, name VARCHAR(100));
INSERT INTO users (name) VALUES ('Alice'), ('Bob');
SELECT * FROM users;

Assegurar la Seguretat

  • MySQL:
    • Utilitzeu mysql_secure_installation per establir una contrasenya de root i eliminar usuaris anònims.
    • Actualitzeu regularment MySQL a l'última versió.
  • PostgreSQL:
    • Configureu pg_hba.conf per controlar l'autenticació dels clients.
    • Utilitzeu contrasenyes fortes i rols per als usuaris de la base de dades.

Conclusió

Els servidors de bases de dades són essencials per gestionar dades en aplicacions. Entendre com instal·lar, configurar i gestionar aquests servidors en un sistema Linux és crucial per a qualsevol administrador de sistemes o desenvolupador. Aquesta secció ha cobert els conceptes bàsics dels servidors de bases de dades populars com MySQL, PostgreSQL i SQLite, juntament amb tasques essencials de gestió. En dominar aquestes habilitats, podeu assegurar-vos que les vostres aplicacions tinguin una base de gestió de dades robusta i segura.
