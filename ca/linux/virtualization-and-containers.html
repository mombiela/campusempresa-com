Introducció a la Virtualització

La virtualització és el procés de crear una versió virtual d'alguna cosa, com ara plataformes de maquinari, dispositius d'emmagatzematge i recursos de xarxa. Permet que múltiples sistemes operatius s'executin en una única màquina física abstractant el maquinari.

Conceptes Clau

  • Hypervisor: Programari que crea i executa màquines virtuals (VMs). Hi ha dos tipus:
    • Tipus 1 (Bare-metal): S'executa directament en el maquinari de l'amfitrió (per exemple, VMware ESXi, Microsoft Hyper-V).
    • Tipus 2 (Hosted): S'executa en un sistema operatiu convencional (per exemple, VirtualBox, VMware Workstation).
  • Màquina Virtual (VM): Una emulació d'un sistema informàtic que s'executa en un hypervisor.
  • Guest OS: El sistema operatiu que s'executa dins d'una VM.
  • Host OS: El sistema operatiu que s'executa en el maquinari físic.

Exemple: Creació d'una VM amb VirtualBox

# Instal·lar VirtualBox en un sistema basat en Debian
sudo apt-get update
sudo apt-get install virtualbox

# Crear una nova VM
VBoxManage createvm --name "LinuxVM" --register

# Establir propietats de la VM
VBoxManage modifyvm "LinuxVM" --memory 2048 --acpi on --boot1 dvd --nic1 nat

# Crear un disc dur virtual
VBoxManage createhd --filename "LinuxVM.vdi" --size 20000

# Adjuntar el disc dur a la VM
VBoxManage storagectl "LinuxVM" --name "SATA Controller" --add sata --controller IntelAhci
VBoxManage storageattach "LinuxVM" --storagectl "SATA Controller" --port 0 --device 0 --type hdd --medium "LinuxVM.vdi"

# Adjuntar un fitxer ISO a la VM
VBoxManage storageattach "LinuxVM" --storagectl "SATA Controller" --port 1 --device 0 --type dvddrive --medium /path/to/iso

# Iniciar la VM
VBoxManage startvm "LinuxVM"

Introducció als Contenidors

Els contenidors són paquets de programari lleugers, autònoms i executables que inclouen tot el necessari per executar un tros de programari, incloent el codi, el temps d'execució, les eines del sistema, les biblioteques i les configuracions.

Conceptes Clau

  • Docker: Una plataforma popular per desenvolupar, enviar i executar aplicacions en contenidors.
  • Imatge: Un paquet de programari lleuger, autònom i executable que inclou tot el necessari per executar un tros de programari.
  • Contenidor: Una instància d'execució d'una imatge.
  • Dockerfile: Un document de text que conté totes les ordres per muntar una imatge.

Exemple: Creació i Execució d'un Contenidor Docker

# Exemple de Dockerfile
FROM ubuntu:latest
RUN apt-get update && apt-get install -y nginx
CMD ["nginx", "-g", "daemon off;"]
# Construir la imatge Docker
docker build -t my-nginx-image .

# Executar el contenidor Docker
docker run -d -p 80:80 my-nginx-image

Virtualització vs. Contenidors

| Característica | Màquines Virtuals (VMs) | Contenidors | |----------------------|---------------------------------------|-----------------------------------| | Aïllament | Aïllament fort mitjançant hypervisor | Aïllament a nivell de procés mitjançant namespaces i cgroups | | Rendiment | Sobrecàrrega degut a l'emulació completa del SO | Rendiment gairebé natiu | | Eficiència de Recursos | Requereix més recursos (CPU, RAM) | Lleuger i eficient | | Temps d'Arrencada | Minuts | Segons | | Cas d'Ús | Executar múltiples instàncies de SO | Microserveis, pipelines de CI/CD |

Temes Avançats

Orquestració amb Kubernetes

Kubernetes és una plataforma de codi obert per automatitzar el desplegament, l'escalat i les operacions de contenidors d'aplicacions a través de clústers d'amfitrions.

Exemple: Desplegament d'una Aplicació Simple amb Kubernetes

# deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:1.14.2
        ports:
        - containerPort: 80
# Aplicar el desplegament
kubectl apply -f deployment.yaml

# Comprovar l'estat del desplegament
kubectl get deployments

Conclusió

La virtualització i els contenidors són tecnologies essencials en la informàtica moderna, permetent una utilització eficient dels recursos i un desplegament flexible d'aplicacions. La virtualització proporciona un aïllament fort i és adequada per executar múltiples sistemes operatius, mentre que els contenidors ofereixen un desplegament lleuger, eficient i ràpid per a microserveis i pipelines de CI/CD. Entendre ambdues tecnologies i els seus casos d'ús és crucial per a qualsevol professional de Linux.
