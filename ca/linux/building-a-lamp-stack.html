Introducció

Una pila LAMP és una plataforma web de codi obert popular utilitzada per executar llocs web i servidors dinàmics. LAMP significa Linux, Apache, MySQL i PHP. Aquesta secció et guiarà a través del procés de configuració d'una pila LAMP des de zero en un servidor Linux.

Prerequisits

Abans de començar, assegura't de tenir:

  • Un servidor Linux (es recomanen distribucions basades en Ubuntu/Debian per a aquest tutorial).
  • Coneixements bàsics de la línia de comandes de Linux.
  • Accés root o sudo al servidor.

Pas 1: Actualitza el teu sistema

Primer, actualitza l'índex de paquets i actualitza els paquets del sistema a les últimes versions.

sudo apt update
sudo apt upgrade -y

Pas 2: Instal·la Apache

Apache és el component del servidor web de la pila LAMP.

Instal·lant Apache

sudo apt install apache2 -y

Iniciant i habilitant Apache

Inicia el servei Apache i habilita'l perquè s'iniciï en arrencar.

sudo systemctl start apache2
sudo systemctl enable apache2

Verificant la instal·lació d'Apache

Obre el teu navegador web i navega a l'adreça IP del teu servidor. Hauries de veure la pàgina per defecte d'Apache2 Ubuntu.

http://your_server_ip

Pas 3: Instal·la MySQL

MySQL és el sistema de gestió de bases de dades utilitzat en la pila LAMP.

Instal·lant MySQL

sudo apt install mysql-server -y

Assegurant la instal·lació de MySQL

Executa l'script de seguretat per configurar les opcions de seguretat de MySQL.

sudo mysql_secure_installation

Segueix les indicacions per establir una contrasenya de root i assegurar la teva instal·lació de MySQL.

Pas 4: Instal·la PHP

PHP és el llenguatge de script utilitzat per crear pàgines web dinàmiques.

Instal·lant PHP

sudo apt install php libapache2-mod-php php-mysql -y

Provant PHP

Crea un fitxer d'informació PHP per provar la teva instal·lació de PHP.

sudo nano /var/www/html/info.php

Afegeix el següent contingut al fitxer:

<?php
phpinfo();
?>

Guarda i tanca el fitxer. Després, navega a http://your_server_ip/info.php en el teu navegador web. Hauries de veure una pàgina que mostra informació de PHP.

Pas 5: Configura Apache per utilitzar PHP

Per defecte, Apache servirà fitxers index.html abans que fitxers index.php. Per canviar això, edita el fitxer dir.conf.

sudo nano /etc/apache2/mods-enabled/dir.conf

Mou la directiva index.php al començament de la llista.

<IfModule mod_dir.c>
    DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm
</IfModule>

Guarda i tanca el fitxer, després reinicia Apache.

sudo systemctl restart apache2

Pas 6: Provant la pila LAMP

Per assegurar-te que tot funciona correctament, crea un script PHP senzill per connectar-te a la base de dades MySQL.

Creant una base de dades de prova i un usuari

Inicia sessió a MySQL i crea una base de dades de prova i un usuari.

sudo mysql -u root -p

Dins de la consola de MySQL, executa les següents comandes:

CREATE DATABASE testdb;
CREATE USER 'testuser'@'localhost' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON testdb.* TO 'testuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Creant un script PHP per provar la connexió a la base de dades

Crea un nou fitxer PHP per provar la connexió a la base de dades.

sudo nano /var/www/html/testdb.php

Afegeix el següent contingut al fitxer:

<?php
$servername = "localhost";
$username = "testuser";
$password = "password";
$dbname = "testdb";

// Crear connexió
$conn = new mysqli($servername, $username, $password, $dbname);

// Comprovar connexió
if ($conn->connect_error) {
    die("Connexió fallida: " . $conn->connect_error);
}
echo "Connectat amb èxit";
?>

Guarda i tanca el fitxer. Navega a http://your_server_ip/testdb.php en el teu navegador web. Hauries de veure "Connectat amb èxit".

Conclusió

Felicitats! Has configurat amb èxit una pila LAMP en el teu servidor Linux. Aquesta pila proporciona un entorn robust per desenvolupar i desplegar aplicacions web dinàmiques. Ara pots procedir a desenvolupar les teves aplicacions web utilitzant PHP i MySQL, servits per Apache en el teu servidor Linux.
