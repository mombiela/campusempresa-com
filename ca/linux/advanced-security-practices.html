Introducció

En aquesta secció, aprofundirem en les pràctiques avançades de seguretat per a sistemes Linux. Això inclourà tècniques per assegurar el teu sistema, monitoritzar possibles amenaces i respondre a incidents de seguretat. Al final d'aquesta secció, hauràs de tenir una comprensió completa de com mantenir un entorn Linux segur.

Enduriment del Sistema

L'enduriment del sistema implica configurar el sistema per reduir la seva superfície d'atac i mitigar possibles vulnerabilitats.

Enduriment del Nucli

  • Desactivar Sistemes de Fitxers No Utilitzats: Prevenir la càrrega de sistemes de fitxers innecessaris per reduir la superfície d'atac.

    echo "install cramfs /bin/true" >> /etc/modprobe.d/disable-cramfs.conf
echo "install freevxfs /bin/true" >> /etc/modprobe.d/disable-freevxfs.conf

  • Activar SELinux o AppArmor: Utilitza Security-Enhanced Linux (SELinux) o AppArmor per fer complir les polítiques de seguretat.

    # Comprovar l'estat de SELinux
sestatus

# Activar SELinux
sudo setenforce 1

Enduriment de la Xarxa

  • Desactivar Serveis de Xarxa No Utilitzats: Apaga els serveis que no estan en ús per minimitzar possibles punts d'entrada.

    sudo systemctl disable telnet
sudo systemctl stop telnet

  • Configurar el Tallafocs: Utilitza iptables o firewalld per controlar el trànsit entrant i sortint.

    # Exemple de regla iptables per permetre SSH i bloquejar tot el demés
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
sudo iptables -A INPUT -j DROP

Monitorització i Registre

Una monitorització i registre efectius són crucials per detectar i respondre a incidents de seguretat.

Registres del Sistema

  • Configurar rsyslog: Assegura't que rsyslog està configurat per capturar i emmagatzemar registres.

    sudo systemctl enable rsyslog
sudo systemctl start rsyslog

  • Rotació de Registres: Utilitza logrotate per gestionar la mida i la retenció dels fitxers de registre.

    # Exemple de configuració de logrotate per /var/log/syslog
/var/log/syslog {
    daily
    rotate 7
    compress
    missingok
    notifempty
    create 0640 root utmp
}

Sistemes de Detecció d'Intrusions (IDS)

  • Instal·lar i Configurar AIDE: Advanced Intrusion Detection Environment (AIDE) ajuda a monitoritzar la integritat dels fitxers.

    sudo apt-get install aide
sudo aideinit
sudo cp /var/lib/aide/aide.db.new /var/lib/aide/aide.db

  • Utilitzar Snort per a la Monitorització de Xarxa: Snort és un sistema de detecció d'intrusions de xarxa que pot monitoritzar i registrar el trànsit de xarxa.

    sudo apt-get install snort
sudo snort -A console -i eth0 -c /etc/snort/snort.conf

Control d'Accés

Implementar mesures estrictes de control d'accés és essencial per mantenir la seguretat del sistema.

Gestió d'Usuaris i Grups

  • Principi de Menor Privilegi: Assegura't que els usuaris tenen els permisos mínims necessaris.

    # Afegir un usuari a un grup específic
sudo usermod -aG groupname username

  • Polítiques de Contrasenyes: Fer complir polítiques de contrasenyes fortes utilitzant pam_pwquality.

    # Editar /etc/security/pwquality.conf
minlen = 12
dcredit = -1
ucredit = -1
ocredit = -1

Enduriment de SSH

  • Desactivar l'Inici de Sessió com a Root: Prevenir l'inici de sessió com a root a través de SSH.

    # Editar /etc/ssh/sshd_config
PermitRootLogin no

  • Utilitzar Autenticació Basada en Claus: Desactivar l'autenticació per contrasenya i utilitzar claus SSH.

    # Editar /etc/ssh/sshd_config
PasswordAuthentication no

Resposta a Incidents

Estar preparat per respondre a incidents de seguretat és crucial per minimitzar els danys.

Pla de Resposta a Incidents

  • Preparació: Tenir un pla de resposta a incidents documentat.
  • Detecció i Anàlisi: Utilitzar eines de monitorització per detectar incidents.
  • Contenció, Erradicació i Recuperació: Passos per contenir i eliminar amenaces.
  • Activitat Post-Incident: Revisar i millorar les mesures de seguretat.

Eines per a la Resposta a Incidents

  • tcpdump: Capturar i analitzar el trànsit de xarxa.

    sudo tcpdump -i eth0 -w capture.pcap

  • chkrootkit: Comprovar la presència de rootkits al sistema.

    sudo apt-get install chkrootkit
sudo chkrootkit

Conclusió

En aquesta secció, hem cobert pràctiques avançades de seguretat per a sistemes Linux, incloent l'enduriment del sistema, la monitorització i el registre, el control d'accés i la resposta a incidents. Implementant aquestes pràctiques, pots millorar significativament la seguretat del teu entorn Linux i estar millor preparat per gestionar possibles amenaces de seguretat.
