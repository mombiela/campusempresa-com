El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar un servidor a casa pot ser un projecte gratificant que et permet allotjar llocs web, gestionar fitxers i executar diversos serveis des de casa teva. Aquest tema et guiarà a través del procés des d'un nivell principiant fins a un nivell avançat utilitzant Linux.

Introducció als Servidors a Casa

  • Què és un Servidor a Casa?

    • Un servidor a casa és un ordinador que proporciona serveis a altres dispositius dins d'una xarxa domèstica.
    • Els usos comuns inclouen l'emmagatzematge de fitxers, la transmissió de mitjans, l'allotjament web i l'execució d'aplicacions.

  • Beneficis d'un Servidor a Casa

    • Emmagatzematge de dades centralitzat.
    • Privacitat millorada i control sobre les teves dades.
    • Aprenentatge i experimentació amb la gestió de servidors.

Triar el Maquinari Adequat

  • Requisits Bàsics

    • Un ordinador amb un processador decent (per exemple, Intel i3 o millor).
    • Almenys 4GB de RAM.
    • Emmagatzematge suficient (HDD o SSD).
    • Connexió de xarxa fiable.

  • Maquinari Recomanat

    • Raspberry Pi per a tasques lleugeres.
    • Maquinari de servidor dedicat per a aplicacions més exigents.

Instal·lar Linux

  • Triar una Distribució de Linux

    • Ubuntu Server: Fàcil d'usar, àmpliament suportat.
    • CentOS: Estable, de grau empresarial.
    • Debian: Molt estable, bo per a ús a llarg termini.

  • Passos d'Instal·lació

    1. Descarrega la imatge ISO de la distribució escollida.
    2. Crea una unitat USB d'arrencada utilitzant eines com Rufus o Etcher.
    3. Arrenca des de la unitat USB i segueix les instruccions en pantalla per instal·lar Linux.
# Exemple: Crear una unitat USB d'arrencada amb dd (comanda de Linux)
sudo dd if=ubuntu-server.iso of=/dev/sdX bs=4M
sync

Configuració Bàsica del Servidor

  • Configurar SSH

    • Instal·la el servidor OpenSSH: 
      sudo apt update
sudo apt install openssh-server
    • Activa i inicia el servei SSH: 
      sudo systemctl enable ssh
sudo systemctl start ssh

  • Configurar una Adreça IP Estàtica

    • Edita el fitxer de configuració de la xarxa (exemple per a Ubuntu): 
      sudo nano /etc/netplan/01-netcfg.yaml
    • Configuració d'exemple: 
      network:
  version: 2
  ethernets:
    eth0:
      dhcp4: no
      addresses: [192.168.1.100/24]
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
    • Aplica la configuració: 
      sudo netplan apply

Configurar Serveis Comuns

  • Compartició de Fitxers amb Samba

    • Instal·la Samba: 
      sudo apt install samba
    • Configura una carpeta compartida: 
      sudo nano /etc/samba/smb.conf
    • Configuració d'exemple: 
      [shared]
  path = /srv/samba/shared
  read only = no
  browsable = yes
    • Reinicia el servei Samba: 
      sudo systemctl restart smbd

  • Allotjament Web amb Apache

    • Instal·la Apache: 
      sudo apt install apache2
    • Inicia i activa el servei Apache: 
      sudo systemctl start apache2
sudo systemctl enable apache2
    • Col·loca els fitxers del teu lloc web a /var/www/html.

Configuració Avançada

  • Configurar un Servidor de Base de Dades (MySQL)

    • Instal·la MySQL: 
      sudo apt install mysql-server
    • Assegura la instal·lació de MySQL: 
      sudo mysql_secure_installation

  • Executar Aplicacions amb Docker

    • Instal·la Docker: 
      sudo apt install docker.io
    • Descarrega i executa un contenidor Docker: 
      sudo docker run -d -p 80:80 --name webserver nginx

  • Automatitzar Tasques amb Cron Jobs

    • Edita el fitxer crontab: 
      crontab -e
    • Exemple de cron job (executa un script cada dia a mitjanit): 
      0 0 * * * /path/to/script.sh

Consideracions de Seguretat

  • Configuració del Tallafoc

    • Instal·la UFW (Uncomplicated Firewall): 
      sudo apt install ufw
    • Permet serveis essencials: 
      sudo ufw allow ssh
sudo ufw allow http
sudo ufw enable

  • Actualitzacions Regulars

    • Mantén el teu sistema actualitzat: 
      sudo apt update && sudo apt upgrade -y

Conclusió

Configurar un servidor a casa utilitzant Linux pot millorar significativament les teves habilitats tècniques i proporcionar nombrosos beneficis pràctics. Seguint aquesta guia, ara hauries de tenir una comprensió bàsica de com triar el maquinari, instal·lar Linux, configurar serveis essencials i assegurar el teu servidor. A mesura que adquireixis més experiència, podràs explorar configuracions avançades i serveis addicionals per ampliar encara més les capacitats del teu servidor.
