Introducció a la Seguretat en Linux

La seguretat en Linux és un aspecte crític de l'administració del sistema. Entendre com gestionar els permisos i assegurar el teu sistema és essencial per protegir les dades i garantir la integritat del sistema.

Permisos de Fitxers

Linux utilitza un model de permisos per controlar l'accés a fitxers i directoris. Cada fitxer i directori té un conjunt de permisos associats que determinen qui pot llegir, escriure o executar-lo.

Entendre els Tipus de Permisos

  • Llegir (r): Permet veure el contingut d'un fitxer o llistar el contingut d'un directori.
  • Escriure (w): Permet modificar el contingut d'un fitxer o afegir/eliminar fitxers en un directori.
  • Executar (x): Permet executar un fitxer com a programa o entrar en un directori.

Categories de Permisos

Els permisos es divideixen en tres categories:

  • Propietari: L'usuari que posseeix el fitxer.
  • Grup: Un grup d'usuaris que comparteixen drets d'accés.
  • Altres: Tots els altres usuaris.

Veure Permisos

Utilitza la comanda ls -l per veure els permisos dels fitxers.

$ ls -l
-rw-r--r-- 1 user group 1234 Jan 1 12:34 example.txt

Explicació:

  • -rw-r--r--: Cadena de permisos.
    • -: Tipus de fitxer (per exemple, d per directori).
    • rw-: Permisos del propietari (llegir i escriure).
    • r--: Permisos del grup (només llegir).
    • r--: Permisos dels altres (només llegir).

Canviar Permisos

Utilitza la comanda chmod per canviar els permisos dels fitxers.

# Concedeix permisos de lectura, escriptura i execució al propietari, i permisos de lectura i execució al grup i altres.
$ chmod 755 example.txt

Modes Simbòlics i Numèrics

Els permisos es poden establir utilitzant modes simbòlics o numèrics.

  • Mode Simbòlic: Utilitza lletres per representar els permisos. 
    $ chmod u+rwx,g+rx,o+rx example.txt
  • Mode Numèric: Utilitza números per representar els permisos. 
    $ chmod 755 example.txt

Gestió d'Usuaris i Grups

Gestionar usuaris i grups és essencial per controlar l'accés als recursos del sistema.

Crear i Gestionar Usuaris

  • Afegir un usuari: useradd 
    $ sudo useradd newuser
  • Establir una contrasenya: passwd 
    $ sudo passwd newuser
  • Eliminar un usuari: userdel 
    $ sudo userdel newuser

Crear i Gestionar Grups

  • Afegir un grup: groupadd 
    $ sudo groupadd newgroup
  • Afegir un usuari a un grup: usermod -aG 
    $ sudo usermod -aG newgroup newuser
  • Eliminar un grup: groupdel 
    $ sudo groupdel newgroup

Permisos Avançats

Els permisos avançats proporcionen un control més granular sobre l'accés als fitxers.

Setuid, Setgid i Sticky Bit

  • Setuid (Set User ID): Permet que un fitxer s'executi amb els permisos del propietari del fitxer. 
    $ chmod u+s example.txt
  • Setgid (Set Group ID): Permet que un fitxer s'executi amb els permisos del grup del fitxer. 
    $ chmod g+s example.txt
  • Sticky Bit: Assegura que només el propietari del fitxer pot eliminar o canviar el nom dels fitxers dins d'un directori. 
    $ chmod +t /example_directory

Llistes de Control d'Accés (ACLs)

Les ACLs proporcionen un mecanisme de permisos més flexible.

  • Veure ACLs: getfacl 
    $ getfacl example.txt
  • Establir ACLs: setfacl 
    $ setfacl -m u:newuser:rwx example.txt

Millors Pràctiques de Seguretat

Implementar les millors pràctiques de seguretat ajuda a protegir el teu sistema Linux contra accessos no autoritzats i vulnerabilitats.

Actualitzacions Regulars

  • Mantingues el teu sistema i programari actualitzats per protegir-te contra vulnerabilitats de seguretat. 
    $ sudo apt update && sudo apt upgrade

Tallafocs

  • Utilitza tallafocs per controlar el trànsit de xarxa entrant i sortint. 
    $ sudo ufw enable
$ sudo ufw allow ssh

Shell Segura (SSH)

  • Utilitza SSH per a un accés remot segur. 
    $ sudo systemctl start ssh
$ sudo systemctl enable ssh

Polítiques de Contrasenyes Fortes

  • Aplica polítiques de contrasenyes fortes per millorar la seguretat. 
    $ sudo apt install libpam-pwquality

Conclusió

Entendre i gestionar la seguretat i els permisos en Linux és crucial per mantenir un sistema segur i eficient. Dominant els permisos de fitxers, la gestió d'usuaris i grups, els permisos avançats i les millors pràctiques de seguretat, pots protegir el teu sistema contra accessos no autoritzats i possibles amenaces. Actualitzar regularment els teus coneixements i mantenir-te informat sobre noves pràctiques de seguretat millorarà encara més la teva capacitat per assegurar el teu entorn Linux.
