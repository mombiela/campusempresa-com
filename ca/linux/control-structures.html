Estructures de control
Funcions i Llibreries ►

Les estructures de control són conceptes fonamentals en la programació que et permeten controlar el flux d'execució en els teus scripts. En Linux, les estructures de control són essencials per escriure scripts de shell eficients i efectius. Aquest tema cobrirà els conceptes bàsics de les estructures de control, incloent-hi condicionals, bucles i declaracions case, progressant des de nivells principiants fins a avançats.

Condicionals

Els condicionals et permeten executar certes parts del teu codi basant-te en condicions específiques.

Declaracions if

La declaració if s'utilitza per provar una condició i executar un bloc de codi si la condició és certa.

Sintaxi:

if [ condició ]; then
    # Codi a executar si la condició és certa
fi

Exemple:

#!/bin/bash

number=10

if [ $number -gt 5 ]; then
    echo "El número és més gran que 5"
fi

Declaracions if-else

La declaració if-else et permet executar un bloc de codi si la condició és certa i un altre bloc si la condició és falsa.

Sintaxi:

if [ condició ]; then
    # Codi a executar si la condició és certa
else
    # Codi a executar si la condició és falsa
fi

Exemple:

#!/bin/bash

number=3

if [ $number -gt 5 ]; then
    echo "El número és més gran que 5"
else
    echo "El número no és més gran que 5"
fi

Declaracions if-elif-else

La declaració if-elif-else et permet provar múltiples condicions.

Sintaxi:

if [ condició1 ]; then
    # Codi a executar si la condició1 és certa
elif [ condició2 ]; then
    # Codi a executar si la condició2 és certa
else
    # Codi a executar si cap de les condicions és certa
fi

Exemple:

#!/bin/bash

number=7

if [ $number -gt 10 ]; then
    echo "El número és més gran que 10"
elif [ $number -gt 5 ]; then
    echo "El número és més gran que 5 però menor o igual a 10"
else
    echo "El número és 5 o menys"
fi

Bucles

Els bucles et permeten executar un bloc de codi múltiples vegades.

Bucle for

El bucle for itera sobre una llista d'elements i executa un bloc de codi per a cada element.

Sintaxi:

for item in list; do
    # Codi a executar per a cada element
done

Exemple:

#!/bin/bash

for i in 1 2 3 4 5; do
    echo "Número: $i"
done

Bucle while

El bucle while executa un bloc de codi mentre una condició sigui certa.

Sintaxi:

while [ condició ]; do
    # Codi a executar mentre la condició sigui certa
done

Exemple:

#!/bin/bash

counter=1

while [ $counter -le 5 ]; do
    echo "Comptador: $counter"
    ((counter++))
done

Bucle until

El bucle until executa un bloc de codi fins que una condició esdevingui certa.

Sintaxi:

until [ condició ]; do
    # Codi a executar fins que la condició sigui certa
done

Exemple:

#!/bin/bash

counter=1

until [ $counter -gt 5 ]; do
    echo "Comptador: $counter"
    ((counter++))
done

Declaracions Case

La declaració case et permet executar diferents blocs de codi basant-te en el valor d'una variable.

Sintaxi:

case $variable in
    pattern1)
        # Codi a executar si la variable coincideix amb pattern1
        ;;
    pattern2)
        # Codi a executar si la variable coincideix amb pattern2
        ;;
    *)
        # Codi a executar si la variable no coincideix amb cap patró
        ;;
esac

Exemple:

#!/bin/bash

day="Dilluns"

case $day in
    "Dilluns")
        echo "Inici de la setmana laboral"
        ;;
    "Divendres")
        echo "Final de la setmana laboral"
        ;;
    "Dissabte" | "Diumenge")
        echo "Cap de setmana!"
        ;;
    *)
        echo "Dia entre setmana"
        ;;
esac

Conclusió

Les estructures de control són essencials per escriure scripts de shell efectius en Linux. En dominar els condicionals, bucles i declaracions case, pots crear scripts que prenguin decisions, repeteixin tasques i gestionin múltiples escenaris de manera eficient. Practica aquests conceptes amb diversos exemples per convertir-te en un expert en escriure scripts de shell robustos.
