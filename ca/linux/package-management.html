La gestió de paquets és un aspecte crític del manteniment d'un sistema Linux. Implica la instal·lació, actualització i eliminació de paquets de programari. Aquest tema cobrirà els conceptes bàsics de la gestió de paquets, diferents sistemes de gestió de paquets i tècniques avançades per gestionar paquets en un sistema Linux.

Introducció a la Gestió de Paquets

  • Definició: La gestió de paquets es refereix al procés de manejar paquets de programari, incloent la seva instal·lació, actualització, configuració i eliminació.
  • Importància: Una gestió eficient de paquets assegura que les dependències de programari es manegin correctament, la integritat del sistema es mantingui i el programari es mantingui actualitzat.

Tipus de Sistemes de Gestió de Paquets

Les distribucions Linux utilitzen diferents sistemes de gestió de paquets. Els dos tipus més comuns són:

  • Sistemes basats en Debian: Utilitzen dpkg i APT (Advanced Package Tool).
  • Sistemes basats en Red Hat: Utilitzen RPM (Red Hat Package Manager) i YUM (Yellowdog Updater, Modified) o DNF (Dandified YUM).

Sistemes basats en Debian

dpkg

  • dpkg: El sistema de gestió de paquets base per a distribucions basades en Debian.
  • Ús: Gestiona directament els paquets .deb.
# Instal·lar un paquet
sudo dpkg -i package_name.deb

# Eliminar un paquet
sudo dpkg -r package_name

# Llistar paquets instal·lats
dpkg -l

APT

  • APT: Una eina de nivell superior que treballa amb dpkg per manejar la instal·lació i gestió de paquets des de repositoris.
  • Ús: Simplifica la gestió de paquets resolent les dependències automàticament.
# Actualitzar la llista de paquets
sudo apt update

# Actualitzar tots els paquets instal·lats
sudo apt upgrade

# Instal·lar un paquet
sudo apt install package_name

# Eliminar un paquet
sudo apt remove package_name

# Cercar un paquet
apt search package_name

Sistemes basats en Red Hat

RPM

  • RPM: El sistema de gestió de paquets base per a distribucions basades en Red Hat.
  • Ús: Gestiona directament els paquets .rpm.
# Instal·lar un paquet
sudo rpm -i package_name.rpm

# Eliminar un paquet
sudo rpm -e package_name

# Llistar paquets instal·lats
rpm -qa

YUM i DNF

  • YUM: Una eina de nivell superior que treballa amb RPM per manejar la instal·lació i gestió de paquets des de repositoris.
  • DNF: La versió de nova generació de YUM, utilitzada en distribucions basades en Red Hat més noves.
# Actualitzar la llista de paquets (YUM)
sudo yum check-update

# Actualitzar tots els paquets instal·lats (YUM)
sudo yum update

# Instal·lar un paquet (YUM)
sudo yum install package_name

# Eliminar un paquet (YUM)
sudo yum remove package_name

# Cercar un paquet (YUM)
yum search package_name

# Actualitzar la llista de paquets (DNF)
sudo dnf check-update

# Actualitzar tots els paquets instal·lats (DNF)
sudo dnf upgrade

# Instal·lar un paquet (DNF)
sudo dnf install package_name

# Eliminar un paquet (DNF)
sudo dnf remove package_name

# Cercar un paquet (DNF)
dnf search package_name

Tècniques Avançades de Gestió de Paquets

Gestió de Repositoris

  • Repositoris: Ubicacions en línia on es guarden els paquets i des d'on es poden descarregar i instal·lar.
  • Afegir Repositoris: De vegades, cal afegir repositoris de tercers per obtenir accés a programari addicional.

APT

# Afegir un nou repositori
sudo add-apt-repository ppa:repository_name

# Actualitzar la llista de paquets després d'afegir un repositori
sudo apt update

YUM/DNF

# Afegir un nou repositori (YUM)
sudo yum-config-manager --add-repo http://repository_url/repo_name.repo

# Afegir un nou repositori (DNF)
sudo dnf config-manager --add-repo http://repository_url/repo_name.repo

Mantenir i Fixar Paquets

  • Mantenir Paquets: Evita que un paquet s'actualitzi.
  • Fixar Paquets: Permet especificar quina versió d'un paquet s'ha d'instal·lar.

APT

# Mantenir un paquet
sudo apt-mark hold package_name

# Desmantenir un paquet
sudo apt-mark unhold package_name

YUM/DNF

# Mantenir un paquet (YUM)
echo "exclude=package_name" | sudo tee -a /etc/yum.conf

# Mantenir un paquet (DNF)
sudo dnf versionlock add package_name

Conclusió

La gestió de paquets és una habilitat fonamental per a qualsevol usuari de Linux. Entendre com instal·lar, actualitzar i eliminar paquets de programari, així com gestionar repositoris i manejar tasques avançades de gestió de paquets, és essencial per mantenir un sistema Linux saludable i eficient. En dominar tant els sistemes de gestió de paquets basats en Debian com en Red Hat, estaràs ben preparat per manejar la gestió de programari en una àmplia gamma de distribucions Linux.
