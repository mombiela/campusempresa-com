Introducció

Entendre els permisos i la propietat dels fitxers és crucial per gestionar fitxers i directoris a Linux. Aquesta secció cobrirà els conceptes bàsics dels permisos dels fitxers, com canviar-los i com funciona la propietat.

Permisos dels Fitxers

Conceptes Clau

  • Llegir (r): Permís per llegir el contingut del fitxer.
  • Escriure (w): Permís per modificar el contingut del fitxer.
  • Executar (x): Permís per executar el fitxer com a programa.

Tipus de Permisos

Els permisos es divideixen en tres categories:

  1. Usuari (u): El propietari del fitxer.
  2. Grup (g): Usuaris que formen part del grup del fitxer.
  3. Altres (o): Tots els altres usuaris.

Veure els Permisos

Per veure els permisos d'un fitxer o directori, utilitza la comanda ls -l:

$ ls -l filename
-rw-r--r-- 1 user group 1234 Jan 1 12:34 filename

Explicació:

  • -rw-r--r--: Cadena de permisos.
    • -: Tipus de fitxer (per exemple, - per fitxer regular, d per directori).
    • rw-: Permisos de l'usuari (llegir i escriure).
    • r--: Permisos del grup (només llegir).
    • r--: Permisos dels altres (només llegir).
  • 1: Nombre d'enllaços durs.
  • user: Propietari del fitxer.
  • group: Grup associat al fitxer.
  • 1234: Mida del fitxer en bytes.
  • Jan 1 12:34: Data i hora de l'última modificació.
  • filename: Nom del fitxer.

Canviar els Permisos

Utilitzant `chmod`

La comanda chmod s'utilitza per canviar els permisos dels fitxers.

Mode Simbòlic

$ chmod u+x filename

Explicació:

  • u+x: Afegir permís d'execució per a l'usuari.

Mode Numèric

Els permisos també es poden establir utilitzant números octals:

  • r = 4
  • w = 2
  • x = 1

Exemple:

$ chmod 755 filename

Explicació:

  • 7 (usuari): rwx (4+2+1)
  • 5 (grup): r-x (4+1)
  • 5 (altres): r-x (4+1)

Propietat dels Fitxers

Conceptes Clau

  • Propietari: L'usuari que posseeix el fitxer.
  • Grup: El grup que posseeix el fitxer.

Veure la Propietat

La propietat es pot veure utilitzant la comanda ls -l (com s'ha mostrat anteriorment).

Canviar la Propietat

La comanda chown s'utilitza per canviar el propietari i el grup d'un fitxer.

Canviar el Propietari

$ sudo chown newuser filename

Explicació:

  • newuser: Nou propietari del fitxer.

Canviar el Grup

$ sudo chown :newgroup filename

Explicació:

  • :newgroup: Nou grup del fitxer.

Canviar Tant el Propietari com el Grup

$ sudo chown newuser:newgroup filename

Explicació:

  • newuser:newgroup: Nou propietari i grup del fitxer.

Exemples Pràctics

Exemple 1: Establir Permisos

$ touch example.txt
$ ls -l example.txt
-rw-r--r-- 1 user group 0 Jan 1 12:34 example.txt

$ chmod 744 example.txt
$ ls -l example.txt
-rwxr--r-- 1 user group 0 Jan 1 12:34 example.txt

Explicació:

  • S'ha creat un nou fitxer example.txt.
  • S'han canviat els permisos a 744 (usuari: rwx, grup: r--, altres: r--).

Exemple 2: Canviar la Propietat

$ sudo chown newuser:newgroup example.txt
$ ls -l example.txt
-rwxr--r-- 1 newuser newgroup 0 Jan 1 12:34 example.txt

Explicació:

  • S'ha canviat el propietari a newuser i el grup a newgroup.

Resum

  • Els permisos dels fitxers a Linux es divideixen en llegir, escriure i executar.
  • Els permisos es categoritzen per a l'usuari, el grup i els altres.
  • Utilitza chmod per canviar els permisos i chown per canviar la propietat.
  • Entendre i gestionar els permisos i la propietat dels fitxers és essencial per a la seguretat del sistema i una correcta gestió dels fitxers.

En dominar aquests conceptes, seràs capaç de controlar efectivament l'accés als fitxers i directoris en un entorn Linux.
