Introducció

En aquesta secció, cobrirem els conceptes fonamentals de les variables i els tipus de dades en el context de la programació en Linux. Entendre aquests conceptes és crucial per escriure scripts i programes eficients i efectius.

Variables

Què és una Variable?

  • Una variable és un nom simbòlic associat amb un valor i el valor associat pot canviar.
  • En Linux, les variables s'utilitzen en la programació de scripts de shell per emmagatzemar dades que es poden utilitzar i manipular al llarg del script.

Declaració de Variables

  • Les variables en els scripts de shell de Linux no requereixen una declaració explícita dels seus tipus de dades.
  • Per declarar una variable, simplement assigna-li un valor utilitzant l'operador =.
#!/bin/bash
# Declaració d'una variable
my_variable="Hola, Món!"
echo $my_variable

Regles de Nomenclatura de Variables

  • Els noms de les variables poden contenir lletres (a-z, A-Z), números (0-9) i guions baixos (_).
  • Els noms de les variables han de començar amb una lletra o un guió baix.
  • Els noms de les variables són sensibles a majúscules i minúscules.

Ús de Variables

  • Per utilitzar el valor d'una variable, prefixa el nom de la variable amb un símbol $.
#!/bin/bash
# Ús d'una variable
greeting="Hola"
name="Alice"
echo "$greeting, $name!"

Tipus de Dades

Visió General

  • A diferència de molts llenguatges de programació, la programació de scripts de shell en Linux no requereix declaracions explícites de tipus de dades.
  • El shell interpreta el tipus de dades basant-se en el context en què s'utilitza la variable.

Tipus de Dades Comuns

  1. Cadenes
    • Una seqüència de caràcters.
    • Envoltat amb cometes dobles (") o cometes simples (').
#!/bin/bash
# Variable de cadena
string_var="Això és una cadena"
echo $string_var
  1. Enters
    • Nombres enters sense component fraccional.
    • Utilitzats per a operacions aritmètiques.
#!/bin/bash
# Variable enter
num1=5
num2=10
sum=$((num1 + num2))
echo "Suma: $sum"
  1. Arrays
    • Una col·lecció de valors indexats per números.
    • Declarats utilitzant parèntesis ().
#!/bin/bash
# Variable array
my_array=(poma plàtan cirera)
echo "Primer element: ${my_array[0]}"
echo "Tots els elements: ${my_array[@]}"

Conversió de Tipus

  • La programació de scripts de shell no suporta la conversió explícita de tipus.
  • El shell converteix automàticament entre tipus basant-se en el context.

Comparació de Tipus de Dades

| Tipus de Dades | Descripció | Exemple | |----------------|--------------------------------|--------------------------| | Cadena | Seqüència de caràcters | name="John Doe" | | Enter | Nombres enters | age=30 | | Array | Col·lecció indexada de valors | fruits=(poma plàtan) |

Exemples i Exercicis

Exemple 1: Manipulació de Cadenes

#!/bin/bash
# Concatenació de cadenes
first_name="John"
last_name="Doe"
full_name="$first_name $last_name"
echo "Nom Complet: $full_name"

Exemple 2: Operacions Aritmètiques

#!/bin/bash
# Realització d'operacions aritmètiques
a=15
b=5
sum=$((a + b))
difference=$((a - b))
product=$((a * b))
quotient=$((a / b))
echo "Suma: $sum"
echo "Diferència: $difference"
echo "Producte: $product"
echo "Quocient: $quotient"

Exercici 1: Crear i Utilitzar Variables

  • Crea un script que declari tres variables: name, age i city.
  • Assigna valors apropiats a aquestes variables.
  • Imprimeix una frase utilitzant aquestes variables, per exemple, "Em dic John, tinc 25 anys i visc a Nova York."

Exercici 2: Manipulació d'Arrays

  • Crea un script que declari un array amb les teves fruites preferides.
  • Imprimeix la primera fruita de l'array.
  • Imprimeix totes les fruites de l'array.

Conclusió

Entendre les variables i els tipus de dades és essencial per a una programació efectiva de scripts de shell en Linux. Les variables et permeten emmagatzemar i manipular dades, mentre que els tipus de dades t'ajuden a entendre com treballar amb diferents tipus de dades. Practica utilitzant variables i diferents tipus de dades per convertir-te en un expert en l'escriptura de scripts de shell.
