L'Inventari Dinàmic és una característica poderosa d'Ansible que et permet gestionar i configurar hosts de manera dinàmica, en lloc de dependre de fitxers d'inventari estàtics. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de l'inventari dinàmic a Ansible.

Introducció a l'Inventari Dinàmic

  • Definició: L'Inventari Dinàmic es refereix a la capacitat d'obtenir dades d'inventari de fonts externes, com ara proveïdors de núvol, bases de dades o altres APIs, en lloc d'utilitzar fitxers estàtics.
  • Casos d'Ús: Ideal per a entorns on la infraestructura està canviant constantment, com ara entorns de núvol (AWS, Azure, GCP), o per a desplegaments a gran escala on mantenir un inventari estàtic és poc pràctic.

Configuració de l'Inventari Dinàmic

Prerequisits

  • Comprensió bàsica d'Ansible i inventari estàtic.
  • Ansible instal·lat a la teva màquina local.
  • Accés a una font externa (per exemple, AWS, Azure) si tens previst utilitzar-la per a l'inventari dinàmic.

Exemple Bàsic: Utilitzant AWS EC2 com a Inventari Dinàmic

  1. Instal·la Boto3 i el Plugin EC2 d'Ansible:

    pip install boto3
ansible-galaxy collection install amazon.aws

  2. Configura les Credencials d'AWS: Assegura't de tenir les teves credencials d'AWS configurades. Això es pot fer utilitzant l'AWS CLI:

    aws configure

  3. Crea un Fitxer de Configuració d'Inventari: Crea un fitxer anomenat aws_ec2.yaml amb el següent contingut:

    plugin: amazon.aws.aws_ec2
regions:
  - us-east-1
filters:
  instance-state-name: running
keyed_groups:
  - key: tags.Name
    prefix: tag

  4. Executa Ansible amb Inventari Dinàmic:

    ansible-inventory -i aws_ec2.yaml --graph

Explicació

  • plugin: Especifica el plugin a utilitzar, en aquest cas, amazon.aws.aws_ec2.
  • regions: Llista de regions d'AWS a consultar.
  • filters: Filtres a aplicar quan es consulten instàncies, com ara instance-state-name: running.
  • keyed_groups: Agrupa instàncies basant-se en les seves etiquetes.

Inventari Dinàmic Avançat

Utilitzant Scripts Personalitzats

  • Scripts d'Inventari Personalitzats: Pots escriure els teus propis scripts per generar inventari dinàmic. Aquests scripts han de produir JSON que Ansible pugui entendre.

Exemple: Script d'Inventari Personalitzat

  1. Crea un Script:

    #!/usr/bin/env python3
import json

def get_inventory():
    inventory = {
        "all": {
            "hosts": ["host1.example.com", "host2.example.com"],
            "vars": {
                "ansible_user": "admin"
            }
        },
        "_meta": {
            "hostvars": {
                "host1.example.com": {
                    "ansible_host": "192.168.1.1"
                },
                "host2.example.com": {
                    "ansible_host": "192.168.1.2"
                }
            }
        }
    }
    return inventory

if __name__ == "__main__":
    print(json.dumps(get_inventory()))

  2. Fes el Script Executable:

    chmod +x custom_inventory.py

  3. Executa Ansible amb el Script Personalitzat:

    ansible-inventory -i custom_inventory.py --graph

Utilitzant Plugins d'Inventari

  • Plugins d'Inventari: Ansible proporciona diversos plugins integrats per a inventari dinàmic, com ara azure_rm, gcp_compute i openstack.

Exemple: Utilitzant el Plugin d'Inventari d'Azure

  1. Instal·la la Col·lecció d'Azure:

    ansible-galaxy collection install azure.azcollection

  2. Crea un Fitxer de Configuració d'Inventari:

    plugin: azure.azcollection.azure_rm
auth_source: auto
include_vm_resource_groups:
  - myResourceGroup

  3. Executa Ansible amb l'Inventari d'Azure:

    ansible-inventory -i azure_rm.yaml --graph

Millors Pràctiques

  • Seguretat: Assegura't que els teus scripts d'inventari dinàmic i fitxers de configuració no exposin informació sensible.
  • Rendiment: Optimitza els teus scripts d'inventari per minimitzar les crides a l'API i reduir la latència.
  • Proves: Prova regularment la teva configuració d'inventari dinàmic per assegurar-te que funciona com s'espera, especialment després de canvis a la teva infraestructura.

Conclusió

L'Inventari Dinàmic a Ansible proporciona una manera flexible i poderosa de gestionar hosts i entorns que estan canviant constantment. Aprofitant fonts externes i scripts personalitzats, pots mantenir un inventari actualitzat sense intervenció manual. Tant si utilitzes proveïdors de núvol com AWS, Azure, o escrius els teus propis scripts, l'inventari dinàmic pot agilitzar significativament els teus fluxos de treball amb Ansible.
