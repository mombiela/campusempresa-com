Introducció

Ansible és una eina d'automatització de codi obert utilitzada per a la gestió de configuracions, el desplegament d'aplicacions i l'automatització de tasques. Aquesta secció et guiarà a través del procés d'instal·lació d'Ansible en diversos sistemes operatius. Al final d'aquesta secció, tindràs Ansible instal·lat i llest per utilitzar.

Prerequisits

Abans d'instal·lar Ansible, assegura't de tenir el següent:

  • Un sistema operatiu compatible (Linux, macOS o Windows Subsystem for Linux).
  • Python 3.8 o posterior instal·lat.
  • Privilegis administratius per instal·lar programari.

Instal·lant Ansible a Linux

Utilitzant el Gestor de Paquets (Recomanat)

Ansible es pot instal·lar utilitzant el gestor de paquets específic per a la teva distribució de Linux.

Ubuntu/Debian

  1. Actualitza l'índex de paquets: 
    sudo apt update
  2. Instal·la el paquet software-properties-common: 
    sudo apt install software-properties-common
  3. Afegeix el PPA (Personal Package Archive) d'Ansible: 
    sudo add-apt-repository --yes --update ppa:ansible/ansible
  4. Instal·la Ansible: 
    sudo apt install ansible

CentOS/RHEL

  1. Instal·la el repositori EPEL: 
    sudo yum install epel-release
  2. Instal·la Ansible: 
    sudo yum install ansible

Utilitzant Pip (Instal·lador de Paquets de Python)

Si prefereixes utilitzar pip, segueix aquests passos:

  1. Assegura't que pip està instal·lat: 
    sudo apt install python3-pip  # Per a Ubuntu/Debian
sudo yum install python3-pip  # Per a CentOS/RHEL
  2. Instal·la Ansible utilitzant pip: 
    pip3 install ansible

Instal·lant Ansible a macOS

Pots instal·lar Ansible a macOS utilitzant Homebrew, un gestor de paquets popular per a macOS.

  1. Instal·la Homebrew si encara no ho has fet: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instal·la Ansible: 
    brew install ansible

Instal·lant Ansible a Windows

Ansible no és compatible nativament amb Windows, però pots utilitzar el Windows Subsystem for Linux (WSL) per executar una distribució de Linux i instal·lar Ansible.

  1. Activa WSL i instal·la una distribució de Linux des de la Microsoft Store (per exemple, Ubuntu).
  2. Obre la distribució de Linux instal·lada i actualitza l'índex de paquets: 
    sudo apt update
  3. Instal·la Ansible utilitzant els mateixos passos que per a Ubuntu/Debian: 
    sudo apt install software-properties-common
sudo add-apt-repository --yes --update ppa:ansible/ansible
sudo apt install ansible

Verificant la Instal·lació

Després d'instal·lar Ansible, verifica la instal·lació comprovant la versió:

ansible --version

Hauries de veure una sortida similar a:

ansible [core 2.10.7]
  config file = /etc/ansible/ansible.cfg
  configured module search path = ['/home/user/.ansible/plugins/modules', '/usr/share/ansible/plugins/modules']
  ansible python module location = /usr/local/lib/python3.8/dist-packages/ansible
  executable location = /usr/local/bin/ansible
  python version = 3.8.5 (default, Jul 28 2020, 12:59:40) [GCC 9.3.0]

Conclusió

En aquesta secció, has après com instal·lar Ansible en diversos sistemes operatius, incloent Linux, macOS i Windows (via WSL). Assegurar-te que Ansible està correctament instal·lat és el primer pas per aprofitar les seves potents capacitats d'automatització. En les següents seccions, explorarem com configurar i utilitzar Ansible per a diferents tasques d'automatització.
