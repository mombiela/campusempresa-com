Introducció a l'Inventari

L'inventari d'Ansible és un component crucial que defineix els hosts i grups de hosts sobre els quals operaran les teves tasques d'Ansible. Entendre com configurar i gestionar l'inventari és essencial per a una automatització efectiva.

Conceptes Clau

  • Fitxer d'Inventari: Un fitxer que llista els hosts i grups de hosts.
  • Hosts: Màquines individuals o nodes que Ansible gestionarà.
  • Grups: Col·leccions de hosts que es poden gestionar conjuntament.
  • Inventari Dinàmic: Inventari que es genera dinàmicament a partir de fonts externes.
  • Inventari Estàtic: Inventari que es defineix en un fitxer estàtic.

Inventari Estàtic

Un inventari estàtic és un fitxer de text senzill que llista els hosts i grups. Es pot escriure en format INI o YAML.

Format INI

El format INI és senzill i fàcil d'entendre. Aquí tens un exemple:

[webservers]
web1.example.com
web2.example.com

[databases]
db1.example.com
db2.example.com

Format YAML

El format YAML és més flexible i llegible, especialment per a inventaris complexos.

all:
  hosts:
    web1.example.com:
    web2.example.com:
  children:
    databases:
      hosts:
        db1.example.com:
        db2.example.com:

Inventari Dinàmic

L'inventari dinàmic et permet obtenir dades d'inventari de fonts externes, com ara proveïdors de núvol, bases de dades o altres serveis. Això és útil per a entorns on la infraestructura és dinàmica i canvia freqüentment.

Exemple: Inventari Dinàmic d'AWS EC2

Per utilitzar AWS EC2 com a font d'inventari dinàmic, cal configurar el plugin d'AWS.

  1. Instal·la les dependències necessàries:

    pip install boto boto3

  2. Crea un fitxer de configuració aws_ec2.yaml:

    plugin: aws_ec2
regions:
  - us-east-1
filters:
  instance-state-name: running

  3. Executa una comanda d'Ansible utilitzant l'inventari dinàmic:

    ansible-inventory -i aws_ec2.yaml --graph

Variables de Grup i Variables de Host

Ansible et permet definir variables tant a nivell de grup com de host. Aquestes variables es poden utilitzar per personalitzar el comportament dels teus playbooks.

Variables de Grup

Les variables de grup es defineixen en una estructura de directoris sota el directori group_vars.

# group_vars/webservers.yaml
---
http_port: 80
max_clients: 200

Variables de Host

Les variables de host es defineixen en una estructura de directoris sota el directori host_vars.

# host_vars/web1.example.com.yaml
---
ansible_host: 192.168.1.10

Millors Pràctiques

  • Utilitza Noms de Grup Descriptius: Utilitza noms significatius per als grups per fer el teu inventari més llegible.
  • Organitza els Fitxers d'Inventari: Mantingues els teus fitxers d'inventari organitzats, especialment quan tractis amb entorns grans.
  • Aprofita l'Inventari Dinàmic: Utilitza l'inventari dinàmic per a entorns que canvien freqüentment per reduir les actualitzacions manuals.
  • Protegeix les Dades Sensibles: Utilitza Ansible Vault per xifrar les dades sensibles en els teus fitxers d'inventari.

Conclusió

Entendre i gestionar l'inventari és fonamental per utilitzar Ansible de manera efectiva. Tant si treballes amb inventaris estàtics com dinàmics, saber com organitzar i configurar el teu inventari t'ajudarà a automatitzar la teva infraestructura de manera més eficient. Seguint les millors pràctiques i aprofitant la flexibilitat del sistema d'inventari d'Ansible, pots assegurar-te que les teves tasques d'automatització siguin tant escalables com mantenibles.
