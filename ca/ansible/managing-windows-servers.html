Introducció

Ansible és una eina d'automatització potent que pot gestionar tant servidors Linux com Windows. Aquesta secció et guiarà a través del procés de gestió de servidors Windows utilitzant Ansible, des de la configuració bàsica fins a configuracions avançades.

Prerequisits

Abans de començar a gestionar servidors Windows amb Ansible, assegura't de tenir el següent:

  • Una màquina de control amb Ansible instal·lat (Linux o macOS).
  • Servidors Windows amb WinRM habilitat.
  • Coneixement bàsic d'Ansible i YAML.

Configuració de l'entorn

Instal·lació de paquets necessaris

Per gestionar servidors Windows, necessites instal·lar el paquet pywinrm a la teva màquina de control.

pip install "pywinrm>=0.3.0"

Configuració de WinRM en servidors Windows

WinRM (Windows Remote Management) ha de ser configurat als teus servidors Windows. Pots utilitzar el següent script de PowerShell per habilitar i configurar WinRM:

# Habilitar WinRM
winrm quickconfig -force

# Configurar el servei WinRM per iniciar-se automàticament
Set-Service -Name WinRM -StartupType Automatic

# Permetre trànsit no xifrat (opcional, per a proves)
winrm set winrm/config/service '@{AllowUnencrypted="true"}'

# Permetre autenticació bàsica (opcional, per a proves)
winrm set winrm/config/service/auth '@{Basic="true"}'

# Crear un listener de WinRM
winrm create winrm/config/Listener?Address=*+Transport=HTTP

# Verificar la configuració
winrm get winrm/config

Configuració de l'inventari

Definició d'hostes Windows

Al teu fitxer d'inventari d'Ansible, defineix els teus hostes Windows i especifica els detalls de connexió.

[windows]
windows_server ansible_host=192.168.1.10

[windows:vars]
ansible_user=Administrator
ansible_password=YourPassword
ansible_connection=winrm
ansible_winrm_transport=basic
ansible_port=5985
ansible_winrm_server_cert_validation=ignore

Playbook bàsic per a Windows

Creació d'un Playbook senzill

Crea un playbook d'Ansible senzill per provar la connexió amb el teu servidor Windows.

---
- name: Provar la connexió amb el servidor Windows
  hosts: windows
  tasks:
    - name: Fer ping al servidor Windows
      win_ping:

Executa el playbook amb la següent comanda:

ansible-playbook test_connection.yml

Gestió de característiques de Windows

Instal·lació de característiques de Windows

Pots utilitzar el mòdul win_feature per instal·lar característiques de Windows.

---
- name: Instal·lar IIS al servidor Windows
  hosts: windows
  tasks:
    - name: Instal·lar IIS
      win_feature:
        name: Web-Server
        state: present

Eliminació de característiques de Windows

De manera similar, pots eliminar característiques de Windows utilitzant el mòdul win_feature.

---
- name: Eliminar IIS del servidor Windows
  hosts: windows
  tasks:
    - name: Eliminar IIS
      win_feature:
        name: Web-Server
        state: absent

Gestió d'arxius i directoris

Copiar arxius

Utilitza el mòdul win_copy per copiar arxius al servidor Windows.

---
- name: Copiar arxiu al servidor Windows
  hosts: windows
  tasks:
    - name: Copiar un arxiu
      win_copy:
        src: /path/to/local/file.txt
        dest: C:\path\to\remote\file.txt

Gestió de directoris

Utilitza el mòdul win_file per crear o eliminar directoris.

---
- name: Gestionar directoris al servidor Windows
  hosts: windows
  tasks:
    - name: Crear un directori
      win_file:
        path: C:\path\to\directory
        state: directory

    - name: Eliminar un directori
      win_file:
        path: C:\path\to\directory
        state: absent

Configuracions avançades

Gestió de serveis de Windows

Utilitza el mòdul win_service per gestionar serveis de Windows.

---
- name: Gestionar serveis de Windows
  hosts: windows
  tasks:
    - name: Assegurar que el servei està en execució
      win_service:
        name: wuauserv
        start_mode: auto
        state: started

    - name: Aturar un servei
      win_service:
        name: wuauserv
        state: stopped

Execució de scripts de PowerShell

Utilitza el mòdul win_shell o win_command per executar scripts de PowerShell.

---
- name: Executar script de PowerShell al servidor Windows
  hosts: windows
  tasks:
    - name: Executar script de PowerShell
      win_shell: |
        Get-Process | Where-Object { $_.CPU -gt 100 }

Resum

Gestionar servidors Windows amb Ansible implica configurar WinRM, configurar l'inventari i utilitzar diversos mòduls d'Ansible per realitzar tasques. Seguint aquesta guia, pots automatitzar la gestió de servidors Windows, des de tasques bàsiques com copiar arxius fins a configuracions avançades com gestionar serveis i executar scripts de PowerShell. Amb pràctica, podràs aprofitar tot el potencial d'Ansible per optimitzar la teva gestió de servidors Windows.
