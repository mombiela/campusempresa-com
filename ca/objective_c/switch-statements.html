Sentències Switch en Objective-C
Treballant amb Blocs ►

Introducció

Les sentències switch són una construcció de control de flux que permet executar diferents blocs de codi basats en el valor d'una variable o expressió. Proporcionen una manera més llegible i eficient de gestionar múltiples condicions en comparació amb l'ús de múltiples sentències if-else.

Sintaxi Bàsica

La sintaxi bàsica d'una sentència switch en Objective-C és la següent:

switch (expression) {
    case constant1:
        // Codi a executar si l'expressió és igual a constant1
        break;
    case constant2:
        // Codi a executar si l'expressió és igual a constant2
        break;
    // Més casos...
    default:
        // Codi a executar si l'expressió no coincideix amb cap cas
        break;
}

Conceptes Clau

  • Expressió: La variable o valor que s'està avaluant.
  • Cas: Cada valor possible que l'expressió pot prendre.
  • Break: Termina el cas actual i surt de la sentència switch.
  • Default: Opcional; s'executa si cap dels casos coincideix amb l'expressió.

Exemple: Sentència Switch Bàsica

Aquí teniu un exemple senzill per il·lustrar una sentència switch en Objective-C:

int day = 3;

switch (day) {
    case 1:
        NSLog(@"Dilluns");
        break;
    case 2:
        NSLog(@"Dimarts");
        break;
    case 3:
        NSLog(@"Dimecres");
        break;
    case 4:
        NSLog(@"Dijous");
        break;
    case 5:
        NSLog(@"Divendres");
        break;
    case 6:
        NSLog(@"Dissabte");
        break;
    case 7:
        NSLog(@"Diumenge");
        break;
    default:
        NSLog(@"Dia invàlid");
        break;
}

En aquest exemple:

  • La variable day s'avalua.
  • El cas que coincideix amb el valor de day (que és 3) s'executa, imprimint "Dimecres".
  • La sentència break evita que l'execució continuï amb els casos següents.

Ús Avançat

Comportament de Caiguda

En Objective-C, si omets la sentència break, l'execució "caurà" al següent cas. Això pot ser útil en certs escenaris.

int number = 2;

switch (number) {
    case 1:
    case 2:
    case 3:
        NSLog(@"El número està entre 1 i 3");
        break;
    case 4:
    case 5:
        NSLog(@"El número està entre 4 i 5");
        break;
    default:
        NSLog(@"El número està fora de rang");
        break;
}

En aquest exemple:

  • Si number és 1, 2 o 3, es imprimirà el missatge "El número està entre 1 i 3".
  • Si number és 4 o 5, es imprimirà el missatge "El número està entre 4 i 5".
  • El cas default gestiona qualsevol altre valor.

Ús d'Enums amb Sentències Switch

Els enums sovint s'utilitzen amb sentències switch per fer el codi més llegible i mantenible.

typedef NS_ENUM(NSInteger, DayOfWeek) {
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday,
    Sunday
};

DayOfWeek today = Wednesday;

switch (today) {
    case Monday:
        NSLog(@"És dilluns");
        break;
    case Tuesday:
        NSLog(@"És dimarts");
        break;
    case Wednesday:
        NSLog(@"És dimecres");
        break;
    case Thursday:
        NSLog(@"És dijous");
        break;
    case Friday:
        NSLog(@"És divendres");
        break;
    case Saturday:
        NSLog(@"És dissabte");
        break;
    case Sunday:
        NSLog(@"És diumenge");
        break;
}

En aquest exemple:

  • Es defineix un enum DayOfWeek per representar els dies de la setmana.
  • La sentència switch utilitza la variable today de tipus DayOfWeek per determinar quin missatge imprimir.

Resum

Les sentències switch en Objective-C proporcionen una manera clara i eficient de gestionar múltiples condicions basades en el valor d'una expressió. Són especialment útils quan es tracta d'una variable que pot prendre un conjunt limitat de valors. Entenent la sintaxi bàsica, el comportament de caiguda i com utilitzar enums amb sentències switch, pots escriure codi més llegible i mantenible.

Exercicis

  1. Escriu una sentència switch que imprimeixi el nom d'un mes basat en un valor enter (1 per gener, 2 per febrer, etc.).
  2. Modifica l'exercici anterior per gestionar valors de mes invàlids imprimint "Mes invàlid".
  3. Crea un enum per a diferents tipus de fruites i escriu una sentència switch que imprimeixi un missatge basat en el tipus de fruita.

Practicant aquests exercicis, obtindràs una comprensió més profunda de com utilitzar efectivament les sentències switch en Objective-C.
