Sintaxi de Mètodes en Objective-C
Passatge de Paràmetres ►

Introducció

Objective-C és un llenguatge potent que amplia C amb capacitats orientades a objectes. Entendre la sintaxi dels mètodes és crucial per escriure codi efectiu en Objective-C. Aquesta secció cobrirà els conceptes bàsics de la sintaxi dels mètodes, incloent la declaració, implementació i ús.

Declaració de Mètodes

En Objective-C, els mètodes es declaren a la secció d'interfície d'una classe. Les declaracions de mètodes especifiquen el nom del mètode, el tipus de retorn i els paràmetres.

Mètodes d'Instància

Els mètodes d'instància es criden en instàncies d'una classe.

@interface MyClass : NSObject
- (void)sayHello;
- (int)addNumber:(int)a toNumber:(int)b;
@end
  • - indica un mètode d'instància.
  • (void) especifica el tipus de retorn.
  • sayHello és el nom del mètode.
  • addNumber:toNumber: és un mètode amb dos paràmetres.

Mètodes de Classe

Els mètodes de classe es criden en la classe mateixa.

@interface MyClass : NSObject
+ (void)classMethodExample;
@end
  • + indica un mètode de classe.

Implementació de Mètodes

Els mètodes s'implementen a la secció d'implementació d'una classe.

Mètodes d'Instància

@implementation MyClass
- (void)sayHello {
    NSLog(@"Hello, World!");
}

- (int)addNumber:(int)a toNumber:(int)b {
    return a + b;
}
@end
  • El cos del mètode està tancat en {}.
  • NSLog s'utilitza per registrar la sortida a la consola.

Mètodes de Classe

@implementation MyClass
+ (void)classMethodExample {
    NSLog(@"This is a class method.");
}
@end

Ús de Mètodes

Els mètodes es criden utilitzant la següent sintaxi:

Cridar Mètodes d'Instància

MyClass *myObject = [[MyClass alloc] init];
[myObject sayHello];
int result = [myObject addNumber:5 toNumber:10];
NSLog(@"Result: %d", result);
  • [[MyClass alloc] init] crea una instància de MyClass.
  • [myObject sayHello] crida el mètode sayHello en myObject.
  • [myObject addNumber:5 toNumber:10] crida el mètode addNumber:toNumber: i emmagatzema el resultat.

Cridar Mètodes de Classe

[MyClass classMethodExample];
  • [MyClass classMethodExample] crida el mètode de classe classMethodExample en MyClass.

Convencions de Nomenclatura de Mètodes

Objective-C utilitza noms de mètodes descriptius que inclouen els noms dels paràmetres.

Exemple

- (void)setFirstName:(NSString *)firstName lastName:(NSString *)lastName;
  • El nom del mètode és setFirstName:lastName:.
  • Aquesta convenció de nomenclatura millora la llegibilitat del codi.

Resum

Entendre la sintaxi dels mètodes en Objective-C és essencial per escriure codi clar i efectiu. Aquesta secció ha cobert:

  • Declaració de mètodes tant per a mètodes d'instància com de classe.
  • Implementació de mètodes.
  • Com cridar mètodes.
  • Convencions de nomenclatura per a mètodes.

En dominar aquests conceptes, estaràs ben preparat per escriure i entendre mètodes en Objective-C, obrint el camí per a temes més avançats en el llenguatge.
