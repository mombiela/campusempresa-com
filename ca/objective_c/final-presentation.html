Presentació Final

Visió General

La presentació final és una part essencial d'aquest curs d'Objective-C, on mostraràs la teva comprensió i habilitats adquirides al llarg del curs. Aquesta presentació implicarà la creació d'un projecte que demostri la teva competència en Objective-C, incloent l'ús de diversos conceptes i tècniques coberts a les lliçons.

Objectius

  • Demostrar la teva comprensió dels conceptes d'Objective-C.
  • Mostrar la teva capacitat per desenvolupar una aplicació funcional.
  • Presentar el teu projecte de manera clara i organitzada.
  • Rebre comentaris per millorar les teves habilitats de codificació i presentació.

Requisits del Projecte

El teu projecte final hauria d'incloure els següents elements:

  • Conceptes Bàsics d'Objective-C: Ús de la sintaxi bàsica, variables, tipus de dades i estructures de control.
  • Programació Orientada a Objectes: Implementació de classes, objectes, herència i polimorfisme.
  • Gestió de Memòria: Ús adequat de tècniques de gestió de memòria com ARC (Automatic Reference Counting).
  • Collections: Ús de arrays, diccionaris i altres tipus de col·leccions.
  • Protocols i Delegats: Implementació de protocols i delegats per gestionar la comunicació entre objectes.
  • Gestió d'Errors: Gestió adequada d'errors utilitzant NSError i tècniques relacionades.
  • Interfície d'Usuari: Elements bàsics de la UI utilitzant UIKit, incloent vistes, controladors de vista i navegació.
  • Networking: Tasques bàsiques de xarxa com obtenir dades d'un servei web.

Idees de Projecte

Aquí tens algunes idees de projectes per començar:

  • App de Llista de Tasques: Una app senzilla per gestionar tasques amb funcions com afegir, editar i eliminar tasques.
  • App del Temps: Una app que obté i mostra informació meteorològica d'un servei web.
  • App de Notes: Una app per crear, editar i organitzar notes amb diferents categories.
  • App de Calculadora: Una calculadora bàsica amb operacions aritmètiques estàndard.
  • App de Receptes: Una app per navegar, cercar i desar receptes amb ingredients i instruccions.

Directrius per a la Presentació

  • Introducció: Presenta't breument i presenta el teu projecte.
  • Visió General del Projecte: Explica el propòsit i la funcionalitat del teu projecte.
  • Revisió del Codi: Destaca parts clau del teu codi, explicant com has implementat diversos conceptes d'Objective-C.
  • Demostració: Mostra una demostració en viu de la teva aplicació.
  • Reptes i Solucions: Discuteix qualsevol repte que hagis enfrontat durant el desenvolupament i com els has superat.
  • Conclusió: Resumeix el teu projecte i el que has après de l'experiència.

Exemple de Codi

Aquí tens un exemple d'una classe senzilla d'Objective-C que podria formar part del teu projecte:

#import <Foundation/Foundation.h>

@interface Task : NSObject

@property (nonatomic, strong) NSString *title;
@property (nonatomic, assign) BOOL completed;

- (instancetype)initWithTitle:(NSString *)title;
- (void)markAsCompleted;

@end

@implementation Task

- (instancetype)initWithTitle:(NSString *)title {
    self = [super init];
    if (self) {
        _title = title;
        _completed = NO;
    }
    return self;
}

- (void)markAsCompleted {
    self.completed = YES;
}

@end

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        Task *task = [[Task alloc] initWithTitle:@"Finish Objective-C project"];
        NSLog(@"Task: %@, Completed: %d", task.title, task.completed);
        [task markAsCompleted];
        NSLog(@"Task: %@, Completed: %d", task.title, task.completed);
    }
    return 0;
}

Explicació

  • Classe Task: Aquesta classe representa una tasca amb un títol i un estat de completat.
  • Propietats: title és una cadena que representa el títol de la tasca, i completed és un booleà que indica si la tasca està completada.
  • Mètodes: El mètode initWithTitle: inicialitza una nova tasca amb un títol donat, i el mètode markAsCompleted marca la tasca com a completada.
  • Funció Principal: Crea una nova tasca, imprimeix el seu estat, la marca com a completada i imprimeix l'estat de nou.

Conclusió

La presentació final és la teva oportunitat per demostrar la teva mestria en Objective-C. En completar un projecte i presentar-lo, no només consolidaràs la teva comprensió, sinó que també guanyaràs experiència valuosa en explicar el teu treball als altres. Recorda centrar-te en la claredat, l'organització i mostrar els conceptes clau que has après. Bona sort!
