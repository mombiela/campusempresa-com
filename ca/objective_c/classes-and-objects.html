Introducció a les Classes i Objectes

En Objective-C, les classes i els objectes són conceptes fonamentals que formen la base de la programació orientada a objectes. Una classe defineix un plànol per als objectes, encapsulant dades i comportament. Un objecte és una instància d'una classe.

Conceptes Clau

  • Classe: Un plànol per crear objectes. Defineix propietats (dades) i mètodes (comportament).
  • Objecte: Una instància d'una classe. Conté valors reals i pot realitzar accions definides per la seva classe.
  • Variables d'Instància: Variables que contenen dades específiques d'un objecte.
  • Mètodes: Funcions definides dins d'una classe que descriuen els comportaments dels objectes.

Definició d'una Classe

Per definir una classe en Objective-C, utilitzes les directives @interface i @implementation.

Exemple: Definició d'una Classe Simple

// Person.h
#import <Foundation/Foundation.h>

@interface Person : NSObject {
    NSString *name;
    int age;
}

- (void)setName:(NSString *)newName;
- (NSString *)name;
- (void)setAge:(int)newAge;
- (int)age;

@end

// Person.m
#import "Person.h"

@implementation Person

- (void)setName:(NSString *)newName {
    name = newName;
}

- (NSString *)name {
    return name;
}

- (void)setAge:(int)newAge {
    age = newAge;
}

- (int)age {
    return age;
}

@end

Explicació

  • Fitxer de Capçalera (Person.h): Defineix la interfície de la classe, incloent variables d'instància i declaracions de mètodes.
  • Fitxer d'Implementació (Person.m): Conté la implementació real dels mètodes declarats en el fitxer de capçalera.

Creació i Ús d'Objectes

Per crear i utilitzar objectes, has d'assignar memòria per a l'objecte i inicialitzar-lo.

Exemple: Creació i Ús d'un Objecte `Person`

#import <Foundation/Foundation.h>
#import "Person.h"

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        Person *person = [[Person alloc] init];
        [person setName:@"John Doe"];
        [person setAge:30];
        
        NSLog(@"Name: %@", [person name]);
        NSLog(@"Age: %d", [person age]);
    }
    return 0;
}

Explicació

  • Assignació i Inicialització: [[Person alloc] init] assigna memòria per a un nou objecte Person i l'inicialitza.
  • Establiment de Propietats: [person setName:@"John Doe"] estableix la propietat name de l'objecte person.
  • Obtenció de Propietats: [person name] recupera la propietat name de l'objecte person.
  • Registre: NSLog s'utilitza per imprimir els valors a la consola.

Propietats i Accessors Sintetitzats

Objective-C proporciona una manera més convenient de declarar propietats i generar automàticament mètodes getter i setter utilitzant les directives @property i @synthesize.

Exemple: Ús de Propietats

// Person.h
#import <Foundation/Foundation.h>

@interface Person : NSObject

@property (nonatomic, strong) NSString *name;
@property (nonatomic, assign) int age;

@end

// Person.m
#import "Person.h"

@implementation Person

@synthesize name;
@synthesize age;

@end

Explicació

  • @property: Declara una propietat amb atributs especificats (nonatomic, strong, assign).
  • @synthesize: Genera automàticament mètodes getter i setter per a les propietats.

Gestió de Memòria

Objective-C utilitza el comptatge de referències per a la gestió de memòria. Entendre retain, release i autorelease és crucial per gestionar els cicles de vida dels objectes.

Exemple: Comptatge Manual de Referències

Person *person = [[Person alloc] init];
[person retain]; // Incrementa el comptador de referències
[person release]; // Decrementa el comptador de referències
[person autorelease]; // Afegeix a la piscina d'autorelease

Explicació

  • retain: Augmenta el comptador de referències de l'objecte.
  • release: Disminueix el comptador de referències. Quan arriba a zero, l'objecte es desallotja.
  • autorelease: Afegeix l'objecte a la piscina d'autorelease, que alliberarà l'objecte més tard.

Conclusió

Entendre les classes i els objectes és essencial per dominar Objective-C. Aquest tema ha cobert les bases de la definició de classes, la creació d'objectes i l'ús de propietats. També s'han introduït temes avançats com els accessors sintetitzats i la gestió de memòria. Practicant aquests conceptes, estaràs ben encaminat per convertir-te en un expert en programació Objective-C.
