Strings i Text en Objective-C
Col·leccions: Arrays, Diccionaris i Conjunts ►

Introducció a les Cadenes

Les cadenes són seqüències de caràcters utilitzades per representar text en programació. En Objective-C, les cadenes es gestionen comunament utilitzant la classe NSString, que proporciona una àmplia gamma de mètodes per crear, manipular i consultar dades de cadena.

Conceptes Clau

  • NSString: La classe principal per a la manipulació de cadenes.
  • NSMutableString: Una subclasse de NSString que permet operacions de cadena mutables.
  • Literals de Cadena: Com crear cadenes utilitzant sintaxi literal.
  • Mètodes de Cadena: Mètodes comuns per a la manipulació de cadenes.

Creació de Cadenes

Utilitzant Literals de Cadena

Els literals de cadena són la manera més senzilla de crear cadenes en Objective-C. Estan tancats entre cometes dobles i prefixats amb un símbol @.

NSString *greeting = @"Hello, World!";

Utilitzant Mètodes NSString

També pots crear cadenes utilitzant diversos mètodes de NSString.

NSString *greeting = [NSString stringWithFormat:@"Hello, %@!", @"World"];

Mètodes Comuns de NSString

Longitud d'una Cadena

Per obtenir la longitud d'una cadena, utilitza el mètode length.

NSString *greeting = @"Hello, World!";
NSUInteger length = [greeting length];
NSLog(@"Length: %lu", (unsigned long)length); // Output: Length: 13

Comparant Cadenes

Per comparar cadenes, utilitza el mètode isEqualToString:.

NSString *string1 = @"Hello";
NSString *string2 = @"Hello";
BOOL areEqual = [string1 isEqualToString:string2];
NSLog(@"Are equal: %d", areEqual); // Output: Are equal: 1

Concatenant Cadenes

Per concatenar cadenes, utilitza el mètode stringByAppendingString:.

NSString *greeting = @"Hello";
NSString *name = @"World";
NSString *fullGreeting = [greeting stringByAppendingString:name];
NSLog(@"%@", fullGreeting); // Output: HelloWorld

Cadenes Mutables

Introducció a NSMutableString

NSMutableString és una subclasse de NSString que et permet modificar la cadena després que hagi estat creada.

NSMutableString *mutableGreeting = [NSMutableString stringWithString:@"Hello"];
[mutableGreeting appendString:@", World!"];
NSLog(@"%@", mutableGreeting); // Output: Hello, World!

Mètodes Comuns de NSMutableString

  • appendString: Afegeix una cadena al final del receptor.
  • insertString:atIndex: Insereix una cadena en un índex especificat.
  • deleteCharactersInRange: Elimina caràcters en un rang especificat.
  • replaceCharactersInRange:withString: Reemplaça caràcters en un rang especificat amb una altra cadena.
NSMutableString *mutableString = [NSMutableString stringWithString:@"Hello"];
[mutableString appendString:@" World"];
[mutableString insertString:@" Objective-C" atIndex:5];
[mutableString deleteCharactersInRange:NSMakeRange(5, 12)];
[mutableString replaceCharactersInRange:NSMakeRange(5, 5) withString:@" Universe"];
NSLog(@"%@", mutableString); // Output: Hello Universe

Manipulació Avançada de Cadenes

Subcadenes

Extreure subcadenes es pot fer utilitzant els mètodes substringFromIndex:, substringToIndex: i substringWithRange:.

NSString *greeting = @"Hello, World!";
NSString *substring1 = [greeting substringFromIndex:7]; // Output: World!
NSString *substring2 = [greeting substringToIndex:5]; // Output: Hello
NSString *substring3 = [greeting substringWithRange:NSMakeRange(7, 5)]; // Output: World

Cercant dins de les Cadenes

Per cercar subcadenes, utilitza mètodes com rangeOfString:.

NSString *greeting = @"Hello, World!";
NSRange range = [greeting rangeOfString:@"World"];
if (range.location != NSNotFound) {
    NSLog(@"Found 'World' at index %lu", (unsigned long)range.location); // Output: Found 'World' at index 7
}

Formatant Cadenes

Utilitza stringWithFormat: per crear cadenes formatades.

NSString *name = @"John";
int age = 30;
NSString *formattedString = [NSString stringWithFormat:@"%@ is %d years old.", name, age];
NSLog(@"%@", formattedString); // Output: John is 30 years old.

Conclusió

Les cadenes són una part fonamental de la programació en Objective-C, permetent-te gestionar i manipular text de manera eficient. Entenent i utilitzant NSString i NSMutableString, pots realitzar una àmplia gamma d'operacions, des de la creació bàsica de cadenes fins al processament avançat de text. Practica aquests conceptes amb els exemples i exercicis proporcionats per convertir-te en un expert en la gestió de cadenes en Objective-C.
