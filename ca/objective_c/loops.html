Introducció als Bucles

Els bucles són constructes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En Objective-C, com en molts altres llenguatges, els bucles t'ajuden a gestionar tasques repetitives de manera eficient.

Tipus de Bucles en Objective-C

Objective-C suporta diversos tipus de bucles:

  • Bucle for
  • Bucle while
  • Bucle do-while
  • Bucle for-in (utilitzat amb col·leccions)

El Bucle `for`

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi

for (inicialització; condició; increment) {
    // Codi a executar
}

Exemple

for (int i = 0; i < 5; i++) {
    NSLog(@"Iteració %d", i);
}
  • Inicialització: Estableix el punt de partida del bucle (per exemple, int i = 0).
  • Condició: El bucle continua mentre aquesta condició sigui certa (per exemple, i < 5).
  • Increment: Actualitza la variable de control del bucle (per exemple, i++).

El Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix per endavant. El bucle continua mentre la condició especificada sigui certa.

Sintaxi

while (condició) {
    // Codi a executar
}

Exemple

int i = 0;
while (i < 5) {
    NSLog(@"Iteració %d", i);
    i++;
}
  • Condició: El bucle continua mentre aquesta condició sigui certa (per exemple, i < 5).

El Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el cos del bucle s'executarà almenys una vegada.

Sintaxi

do {
    // Codi a executar
} while (condició);

Exemple

int i = 0;
do {
    NSLog(@"Iteració %d", i);
    i++;
} while (i < 5);
  • Condició: El bucle continua mentre aquesta condició sigui certa (per exemple, i < 5).

El Bucle `for-in`

El bucle for-in s'utilitza per iterar sobre tots els elements d'una col·lecció, com ara una matriu o un diccionari.

Sintaxi

for (Tipus variable en col·lecció) {
    // Codi a executar
}

Exemple

NSArray *array = @[@"Poma", @"Plàtan", @"Cirera"];
for (NSString *fruit in array) {
    NSLog(@"Fruit: %@", fruit);
}
  • Tipus: El tipus de la variable que contindrà cada element (per exemple, NSString *).
  • Variable: La variable que contindrà cada element (per exemple, fruit).
  • Col·lecció: La col·lecció a iterar (per exemple, array).

Conceptes Avançats de Bucles

Bucles Imbricats

Els bucles es poden imbricar dins d'altres bucles. Això és útil per a tasques que requereixen múltiples nivells d'iteració, com ara treballar amb matrius multidimensionals.

Exemple

for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        NSLog(@"i = %d, j = %d", i, j);
    }
}

Instruccions de Control de Bucles

Objective-C proporciona diverses instruccions per controlar l'execució dels bucles:

  • break: Surt immediatament del bucle.
  • continue: Salta l'iteració actual i procedeix a la següent iteració.

Exemple amb break

for (int i = 0; i < 5; i++) {
    if (i == 3) {
        break;
    }
    NSLog(@"Iteració %d", i);
}

Exemple amb continue

for (int i = 0; i < 5; i++) {
    if (i == 3) {
        continue;
    }
    NSLog(@"Iteració %d", i);
}

Resum

Els bucles són constructes essencials en Objective-C que et permeten executar codi repetidament. Entendre els diferents tipus de bucles (for, while, do-while i for-in) i els seus casos d'ús adequats és crucial per a una programació eficient. A més, dominar conceptes avançats com els bucles imbricats i les instruccions de control de bucles (break i continue) et permetrà escriure codi més complex i eficient. Practica aquests conceptes amb diversos exemples per consolidar la teva comprensió.
