Introducció a les declaracions condicionals

Les declaracions condicionals són fonamentals en la programació, ja que permeten executar diferents blocs de codi basats en certes condicions. En Objective-C, les principals declaracions condicionals inclouen if, else if, else i switch.

La declaració `if`

La declaració if avalua una condició i executa un bloc de codi si la condició és certa.

Sintaxi

if (condition) {
    // Codi a executar si la condició és certa
}

Exemple

int number = 10;
if (number > 5) {
    NSLog(@"El número és més gran que 5");
}

En aquest exemple, es mostrarà el missatge "El número és més gran que 5" perquè la condició number > 5 és certa.

La declaració `else`

La declaració else segueix una declaració if i executa un bloc de codi si la condició if és falsa.

Sintaxi

if (condition) {
    // Codi a executar si la condició és certa
} else {
    // Codi a executar si la condició és falsa
}

Exemple

int number = 3;
if (number > 5) {
    NSLog(@"El número és més gran que 5");
} else {
    NSLog(@"El número no és més gran que 5");
}

En aquest exemple, es mostrarà el missatge "El número no és més gran que 5" perquè la condició number > 5 és falsa.

La declaració `else if`

La declaració else if permet comprovar múltiples condicions seqüencialment.

Sintaxi

if (condition1) {
    // Codi a executar si la condició1 és certa
} else if (condition2) {
    // Codi a executar si la condició2 és certa
} else {
    // Codi a executar si cap de les condicions anteriors és certa
}

Exemple

int number = 7;
if (number > 10) {
    NSLog(@"El número és més gran que 10");
} else if (number > 5) {
    NSLog(@"El número és més gran que 5 però menor o igual a 10");
} else {
    NSLog(@"El número és 5 o menys");
}

En aquest exemple, es mostrarà el missatge "El número és més gran que 5 però menor o igual a 10" perquè la condició number > 5 és certa.

La declaració `switch`

La declaració switch proporciona una manera d'executar un bloc de codi d'entre molts basat en el valor d'una variable.

Sintaxi

switch (variable) {
    case value1:
        // Codi a executar si variable == value1
        break;
    case value2:
        // Codi a executar si variable == value2
        break;
    // Més casos...
    default:
        // Codi a executar si cap dels casos anteriors és cert
        break;
}

Exemple

int day = 3;
switch (day) {
    case 1:
        NSLog(@"Dilluns");
        break;
    case 2:
        NSLog(@"Dimarts");
        break;
    case 3:
        NSLog(@"Dimecres");
        break;
    case 4:
        NSLog(@"Dijous");
        break;
    case 5:
        NSLog(@"Divendres");
        break;
    case 6:
        NSLog(@"Dissabte");
        break;
    case 7:
        NSLog(@"Diumenge");
        break;
    default:
        NSLog(@"Dia invàlid");
        break;
}

En aquest exemple, es mostrarà el missatge "Dimecres" perquè el valor de day és 3.

Declaracions condicionals niades

Pots niar declaracions condicionals dins d'altres per gestionar una lògica més complexa.

Exemple

int number = 15;
if (number > 10) {
    if (number < 20) {
        NSLog(@"El número està entre 10 i 20");
    } else {
        NSLog(@"El número és 20 o més");
    }
} else {
    NSLog(@"El número és 10 o menys");
}

En aquest exemple, es mostrarà el missatge "El número està entre 10 i 20" perquè el número 15 satisfà ambdues condicions.

Conclusió

Les declaracions condicionals són essencials per controlar el flux del teu programa basat en diferents condicions. Entendre com utilitzar les declaracions if, else if, else i switch et permetrà escriure codi més dinàmic i sensible. Practica utilitzant aquestes declaracions en diverses situacions per convertir-te en un expert en la gestió de la lògica condicional en Objective-C.
