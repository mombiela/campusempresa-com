Introducció

En aquesta secció, esbossarem el projecte en què treballaràs al llarg d'aquest curs. Aquest projecte t'ajudarà a aplicar els conceptes que aprens en un entorn pràctic, reforçant la teva comprensió d'Objective-C des de nivells inicials fins a avançats.

Descripció del projecte

Desenvoluparàs una Aplicació de Llista de Tasques simple però completa. Aquesta aplicació permetrà als usuaris:

  • Afegir noves tasques.
  • Marcar tasques com a completades.
  • Eliminar tasques.
  • Veure una llista de totes les tasques.

Requisits del projecte

Per completar amb èxit aquest projecte, hauràs de complir els següents requisits:

Requisits funcionals

  • Gestió de tasques: Els usuaris haurien de poder afegir, editar, eliminar i marcar tasques com a completades.
  • Interfície d'usuari: L'aplicació hauria de tenir una interfície fàcil d'usar que permeti una navegació i gestió de tasques senzilles.
  • Persistència de dades: Les tasques haurien de guardar-se entre sessions. Això significa que quan l'usuari tanqui i torni a obrir l'aplicació, les seves tasques haurien de seguir allà.

Requisits no funcionals

  • Rendiment: L'aplicació hauria de ser responsiva i gestionar les entrades de l'usuari de manera eficient.
  • Escalabilitat: L'aplicació hauria de poder gestionar un nombre creixent de tasques sense degradació del rendiment.
  • Mantenibilitat: El codi hauria d'estar ben organitzat i documentat per facilitar futures actualitzacions i manteniment.

Eines i tecnologies

Per construir aquest projecte, necessitaràs les següents eines i tecnologies:

  • Xcode: L'entorn de desenvolupament integrat (IDE) per a macOS que conté un conjunt d'eines de desenvolupament de programari.
  • Objective-C: El llenguatge de programació principal utilitzat en aquest curs.
  • Cocoa Touch Framework: Un marc d'interfície d'usuari per construir aplicacions iOS.

Conceptes clau

Aquí tens alguns conceptes clau que necessitaràs entendre per completar aquest projecte:

  • Classes i objectes: Entendre com definir i utilitzar classes i objectes en Objective-C.
  • Gestió de memòria: Utilitzar el Comptatge Automàtic de Referències (ARC) per gestionar la memòria.
  • Disseny d'interfície d'usuari: Utilitzar l'Interface Builder en Xcode per dissenyar la interfície d'usuari.
  • Model-Vista-Controlador (MVC): Aplicar el patró de disseny MVC per estructurar el teu codi.
  • Persistència de dades: Utilitzar llistes de propietats, Core Data o altres mètodes per guardar dades entre sessions.

Exemple de codi

A continuació tens un exemple de com podries definir una classe Task simple en Objective-C:

#import <Foundation/Foundation.h>

@interface Task : NSObject

@property (nonatomic, strong) NSString *title;
@property (nonatomic, assign) BOOL isCompleted;

- (instancetype)initWithTitle:(NSString *)title;
- (void)markAsCompleted;

@end

@implementation Task

- (instancetype)initWithTitle:(NSString *)title {
    self = [super init];
    if (self) {
        _title = title;
        _isCompleted = NO;
    }
    return self;
}

- (void)markAsCompleted {
    self.isCompleted = YES;
}

@end

Explicació

  • @interface i @implementation: Aquestes paraules clau s'utilitzen per declarar i definir la classe Task.
  • Propietats: title i isCompleted són propietats de la classe Task.
  • Mètodes: initWithTitle: és un mètode inicialitzador, i markAsCompleted és un mètode per marcar la tasca com a completada.

Conclusió

En aquesta secció, hem esbossat el projecte en què treballaràs, incloent-hi els requisits funcionals i no funcionals, eines i tecnologies, i conceptes clau. Al final d'aquest curs, tindràs una aplicació de Llista de Tasques completament funcional i una sòlida comprensió de la programació en Objective-C. Aquest projecte servirà com una aplicació pràctica dels conceptes que aprens, ajudant-te a reforçar les teves habilitats i coneixements.
