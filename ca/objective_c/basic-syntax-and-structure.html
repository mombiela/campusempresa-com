Introducció

Objective-C és un llenguatge de programació orientat a objectes d'ús general que afegeix missatgeria d'estil Smalltalk al llenguatge de programació C. S'utilitza principalment per al desenvolupament de macOS i iOS. Aquesta secció cobrirà la sintaxi i l'estructura bàsiques d'Objective-C, proporcionant una base per a un aprenentatge posterior.

Conceptes clau

  • Fitxers de capçalera i fitxers d'implementació: Objective-C separa la interfície i la implementació de les classes en fitxers de capçalera (.h) i fitxers d'implementació (.m).
  • Classes i objectes: Objective-C és un llenguatge orientat a objectes, per tant, entendre les classes i els objectes és crucial.
  • Mètodes: Funcions definides dins d'una classe.
  • Propietats: Variables associades a una classe.
  • Missatgeria: La manera com es criden els mètodes en Objective-C.

Fitxers de capçalera i fitxers d'implementació

En Objective-C, la interfície d'una classe es declara en un fitxer de capçalera (.h), i la implementació es defineix en un fitxer d'implementació (.m).

Exemple

Fitxer de capçalera (MyClass.h):

#import <Foundation/Foundation.h>

@interface MyClass : NSObject

@property (nonatomic, strong) NSString *name;

- (void)printName;

@end

Fitxer d'implementació (MyClass.m):

#import "MyClass.h"

@implementation MyClass

- (void)printName {
    NSLog(@"%@", self.name);
}

@end

Explicació

  • Les paraules clau @interface i @end s'utilitzen per declarar la interfície de la classe.
  • Les paraules clau @implementation i @end s'utilitzen per definir la implementació.
  • #import s'utilitza per incloure els fitxers necessaris.
  • @property declara una propietat per a la classe.

Classes i objectes

Les classes són plànols per crear objectes. Un objecte és una instància d'una classe.

Exemple

MyClass *myObject = [[MyClass alloc] init];
myObject.name = @"Objective-C";
[myObject printName];

Explicació

  • MyClass *myObject declara un punter a una instància de MyClass.
  • [[MyClass alloc] init] assigna memòria i inicialitza l'objecte.
  • myObject.name = @"Objective-C" estableix la propietat name.
  • [myObject printName] envia un missatge a myObject per cridar el mètode printName.

Mètodes

Els mètodes són funcions definides dins d'una classe. Poden ser mètodes d'instància o mètodes de classe.

Exemple

- (void)instanceMethod {
    // Codi del mètode d'instància
}

+ (void)classMethod {
    // Codi del mètode de classe
}

Explicació

  • - denota un mètode d'instància.
  • + denota un mètode de classe.
  • (void) especifica el tipus de retorn.
  • Els noms dels mètodes segueixen el tipus de retorn.

Propietats

Les propietats són variables associades a una classe. Es poden accedir i modificar utilitzant la notació de punt.

Exemple

@property (nonatomic, strong) NSString *name;

Explicació

  • @property declara una propietat.
  • (nonatomic, strong) especifica els atributs de la propietat.
  • NSString *name és el tipus i el nom de la propietat.

Missatgeria

La missatgeria és la manera com es criden els mètodes en Objective-C. Utilitza claudàtors.

Exemple

[myObject printName];

Explicació

  • [myObject printName] envia un missatge a myObject per cridar el mètode printName.

Conclusió

Entendre la sintaxi i l'estructura bàsiques d'Objective-C és essencial per escriure codi efectiu. Això inclou saber com separar la interfície i la implementació, crear i utilitzar classes i objectes, definir mètodes, declarar propietats i enviar missatges. Amb aquests fonaments, estàs ben preparat per aprofundir en la programació en Objective-C.
