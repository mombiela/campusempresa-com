Introducció a Objective-C
Configuració de l'Entorn de Desenvolupament ►

Objective-C és un llenguatge de programació d'ús general i orientat a objectes que afegeix missatgeria d'estil Smalltalk al llenguatge de programació C. Va ser el principal llenguatge de programació utilitzat per Apple per al desenvolupament de macOS i iOS abans que es presentés Swift. Aquesta secció proporcionarà una visió general d'Objective-C, la seva sintaxi i els seus conceptes bàsics.

Què és Objective-C?

Objective-C és una extensió del llenguatge de programació C amb capacitats orientades a objectes i un temps d'execució dinàmic. Va ser creat a principis dels anys 80 i ha estat utilitzat extensivament en l'ecosistema d'Apple.

Característiques Clau d'Objective-C

  • Orientat a Objectes: Suporta classes i objectes, herència i polimorfisme.
  • Tipatge Dinàmic: Permet un codi més flexible i dinàmic.
  • Passatge de Missatges: Utilitza missatgeria d'estil Smalltalk per a les crides a mètodes.
  • Compatibilitat amb C: Pot utilitzar biblioteques i funcions de C directament.

Sintaxi Bàsica

Programa Hello World

Comencem amb un simple programa "Hello, World!" en Objective-C.

#import <Foundation/Foundation.h>

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        NSLog(@"Hello, World!");
    }
    return 0;
}
  • #import <Foundation/Foundation.h>: Importa el framework Foundation, que proporciona classes i funcions essencials.
  • @autoreleasepool: Gestiona la memòria per als objectes d'Objective-C.
  • NSLog: Una funció utilitzada per imprimir missatges a la consola.

Variables i Tipus de Dades

Objective-C suporta diversos tipus de dades, inclosos els heretats de C.

int anInteger = 42;
float aFloat = 3.14;
double aDouble = 3.14159;
char aChar = 'A';
NSString *aString = @"Hello, Objective-C!";
  • int, float, double, char: Tipus de dades bàsics de C.
  • NSString *: Un punter a un objecte NSString, utilitzat per a la manipulació de cadenes.

Comentaris

Objective-C suporta tant comentaris d'una sola línia com comentaris de múltiples línies.

// Aquest és un comentari d'una sola línia.

/*
 Aquest és un comentari de múltiples línies.
 Pot abastar diverses línies.
*/

Programació Orientada a Objectes en Objective-C

Classes i Objectes

Objective-C utilitza classes i objectes per implementar la programació orientada a objectes.

Definició d'una Classe

#import <Foundation/Foundation.h>

@interface Person : NSObject
@property NSString *name;
@property int age;

- (void)printDetails;
@end

@implementation Person
- (void)printDetails {
    NSLog(@"Name: %@, Age: %d", self.name, self.age);
}
@end
  • @interface: Declara la interfície de la classe.
  • @property: Declara propietats de la classe.
  • @implementation: Implementa els mètodes de la classe.
  • - (void)printDetails: Declara un mètode d'instància.

Creació i Ús d'Objectes

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        Person *person = [[Person alloc] init];
        person.name = @"John Doe";
        person.age = 30;
        [person printDetails];
    }
    return 0;
}
  • Person *person = [[Person alloc] init]: Crea una nova instància de la classe Person.
  • person.name = @"John Doe": Estableix la propietat name.
  • [person printDetails]: Crida el mètode printDetails.

Gestió de Memòria

Objective-C utilitza el recompte de referències per a la gestió de memòria. El bloc @autoreleasepool ajuda a gestionar la memòria dels objectes.

Recompte Automàtic de Referències (ARC)

ARC gestiona automàticament el recompte de referències dels objectes, fent la gestió de memòria més fàcil.

Person *person = [[Person alloc] init];
// ARC gestiona automàticament la retenció i alliberament de l'objecte.

Conclusió

En aquesta introducció a Objective-C, hem cobert els conceptes bàsics del llenguatge, incloent la seva sintaxi, tipus de dades i característiques orientades a objectes. També hem tocat la gestió de memòria amb ARC. Objective-C és un llenguatge potent que, tot i estar una mica eclipsat per Swift, continua sent una part important de l'ecosistema de desenvolupament d'Apple.

En les següents seccions, aprofundirem en temes més avançats, com protocols, categories i blocs, per millorar encara més la teva comprensió i competència en Objective-C.
