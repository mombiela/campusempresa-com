Introducció

En aquesta secció, explorarem els conceptes fonamentals dels tipus de dades i les variables en Objective-C. Entendre aquests conceptes és crucial per escriure codi eficient i lliure d'errors. Cobrirem els tipus de dades bàsics, com declarar i inicialitzar variables, i alguns temes avançats relacionats amb els tipus de dades.

Tipus de Dades Bàsics

Objective-C, com moltes altres llenguatges de programació, té diversos tipus de dades integrats. Aquests tipus de dades s'utilitzen per definir el tipus de dades que una variable pot contenir.

Tipus Enter

  • int: Utilitzat per emmagatzemar valors enters.
  • short: Utilitzat per emmagatzemar valors enters més petits.
  • long: Utilitzat per emmagatzemar valors enters més grans.
  • long long: Utilitzat per emmagatzemar valors enters encara més grans.
int age = 25;
short year = 2023;
long population = 7800000000;
long long distance = 123456789012345;

Tipus de Punt Flotant

  • float: Utilitzat per emmagatzemar nombres de punt flotant de precisió simple.
  • double: Utilitzat per emmagatzemar nombres de punt flotant de doble precisió.
float pi = 3.14f;
double e = 2.718281828459;

Tipus Caràcter

  • char: Utilitzat per emmagatzemar un únic caràcter.
char initial = 'A';

Tipus Booleà

  • BOOL: Utilitzat per emmagatzemar valors booleans (YES o NO).
BOOL isObjectiveCFun = YES;

Declarar i Inicialitzar Variables

Les variables en Objective-C es declaren especificant el tipus de dades seguit del nom de la variable. La inicialització es pot fer en el moment de la declaració o més tard en el codi.

int number; // Declaració
number = 10; // Inicialització

float temperature = 36.6; // Declaració i Inicialització

Constants

Les constants són variables els valors de les quals no es poden canviar un cop inicialitzades. Es declaren utilitzant la paraula clau const.

const int daysInWeek = 7;
const float pi = 3.14159;

Tipus de Dades Avançats

Objective-C també suporta tipus de dades més complexos com ara arrays, estructures i punters.

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors del mateix tipus.

int numbers[5] = {1, 2, 3, 4, 5};
char letters[3] = {'A', 'B', 'C'};

Estructures

Les estructures s'utilitzen per agrupar diferents tipus de variables sota un mateix nom.

struct Person {
    char name[50];
    int age;
    float height;
};

struct Person person1;
strcpy(person1.name, "John Doe");
person1.age = 30;
person1.height = 5.9;

Punters

Els punters són variables que emmagatzemen l'adreça de memòria d'una altra variable.

int value = 10;
int *pointer = &value;

Conversió de Tipus

La conversió de tipus és el procés de convertir un tipus de dades a un altre. Això es pot fer implícitament o explícitament.

Conversió Implícita

El compilador converteix automàticament un tipus de dades a un altre.

int a = 10;
float b = a; // Conversió implícita de int a float

Conversió Explícita

El programador converteix manualment un tipus de dades a un altre utilitzant el càsting.

float x = 9.99;
int y = (int)x; // Conversió explícita de float a int

Resum

En aquesta secció, hem cobert els tipus de dades bàsics i avançats en Objective-C, com declarar i inicialitzar variables, i el concepte de constants. També hem tocat els arrays, estructures, punters i la conversió de tipus. Entendre aquests conceptes és essencial per escriure codi robust i eficient en Objective-C. Practiqueu aquests conceptes amb els exemples i exercicis proporcionats per consolidar la vostra comprensió.
