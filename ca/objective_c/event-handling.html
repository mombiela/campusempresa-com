Introducció a la Gestió d'Esdeveniments

La gestió d'esdeveniments és un aspecte crucial del desenvolupament d'aplicacions interactives. En Objective-C, la gestió d'esdeveniments permet que la teva aplicació respongui a accions de l'usuari com tocs, lliscaments i altres gestos.

Conceptes Clau

  • Bucle d'Esdeveniments: El mecanisme central que espera i distribueix esdeveniments als gestors adequats.
  • Cadena de Resposta: Una jerarquia d'objectes que poden respondre a esdeveniments.
  • UIControl: Una classe base per a controls que poden enviar missatges d'acció a objectes objectiu en resposta a interaccions de l'usuari.

Bucle d'Esdeveniments

El bucle d'esdeveniments és l'eix vertebrador de la gestió d'esdeveniments en qualsevol aplicació. Espera contínuament esdeveniments i els distribueix als objectes adequats.

// Main.m
#import <UIKit/UIKit.h>
#import "AppDelegate.h"

int main(int argc, char * argv[]) {
    @autoreleasepool {
        return UIApplicationMain(argc, argv, nil, NSStringFromClass([AppDelegate class]));
    }
}
  • La funció UIApplicationMain configura el bucle d'esdeveniments i comença a processar esdeveniments.

Cadena de Resposta

La cadena de resposta és una sèrie d'objectes que tenen l'oportunitat de respondre a un esdeveniment. Si un objecte no pot gestionar l'esdeveniment, el passa al següent objecte de la cadena.

Exemple

// ViewController.m
#import "ViewController.h"

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];
    
    UITapGestureRecognizer *tapGesture = [[UITapGestureRecognizer alloc] initWithTarget:self action:@selector(handleTap:)];
    [self.view addGestureRecognizer:tapGesture];
}

- (void)handleTap:(UITapGestureRecognizer *)sender {
    NSLog(@"Vista tocada!");
}

@end
  • UITapGestureRecognizer s'afegeix a la vista.
  • El mètode handleTap: es crida quan es toca la vista.

UIControl i Mecanisme Target-Action

UIControl és una classe base per a controls com botons i barres de desplaçament. Utilitza el mecanisme target-action per enviar missatges a objectes objectiu en resposta a interaccions de l'usuari.

Exemple

// ViewController.m
#import "ViewController.h"

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];
    
    UIButton *button = [UIButton buttonWithType:UIButtonTypeSystem];
    [button setTitle:@"Toca'm" forState:UIControlStateNormal];
    button.frame = CGRectMake(100, 100, 100, 50);
    [button addTarget:self action:@selector(buttonTapped:) forControlEvents:UIControlEventTouchUpInside];
    [self.view addSubview:button];
}

- (void)buttonTapped:(UIButton *)sender {
    NSLog(@"Botó tocat!");
}

@end
  • Es crea un botó i s'afegeix a la vista.
  • El mètode buttonTapped: es crida quan es toca el botó.

Gestió d'Esdeveniments Avançada

La gestió d'esdeveniments avançada implica gestos personalitzats i interaccions més complexes.

Reconeixedors de Gestos Personalitzats

Pots crear reconeixedors de gestos personalitzats subclassejant UIGestureRecognizer.

// CustomGestureRecognizer.h
#import <UIKit/UIKit.h>

@interface CustomGestureRecognizer : UIGestureRecognizer
@end

// CustomGestureRecognizer.m
#import "CustomGestureRecognizer.h"

@implementation CustomGestureRecognizer

- (void)touchesBegan:(NSSet<UITouch *> *)touches withEvent:(UIEvent *)event {
    // Lògica personalitzada per al reconeixement de gestos
    [self setState:UIGestureRecognizerStateBegan];
}

- (void)touchesMoved:(NSSet<UITouch *> *)touches withEvent:(UIEvent *)event {
    // Lògica personalitzada per al reconeixement de gestos
    [self setState:UIGestureRecognizerStateChanged];
}

- (void)touchesEnded:(NSSet<UITouch *> *)touches withEvent:(UIEvent *)event {
    // Lògica personalitzada per al reconeixement de gestos
    [self setState:UIGestureRecognizerStateEnded];
}

@end
  • Subclasseja UIGestureRecognizer per crear un reconeixedor de gestos personalitzat.
  • Sobreescriu els mètodes de toc per implementar la lògica personalitzada.

Conclusió

La gestió d'esdeveniments en Objective-C és una característica potent que permet que la teva aplicació respongui a interaccions de l'usuari. Entendre el bucle d'esdeveniments, la cadena de resposta i el mecanisme target-action de UIControl és essencial per crear aplicacions interactives. Per a escenaris avançats, es poden implementar reconeixedors de gestos personalitzats per gestionar interaccions complexes.

Dominant aquests conceptes, pots crear aplicacions responsives i fàcils d'usar que proporcionen una experiència d'usuari fluida.
