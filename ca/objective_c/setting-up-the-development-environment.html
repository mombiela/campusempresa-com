Introducció

Configurar l'entorn de desenvolupament és el primer pas per començar a programar en Objective-C. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar.

Prerequisits

Abans d'endinsar-te en la configuració, assegura't de tenir el següent:

  • Un ordinador Mac amb macOS.
  • Un ID d'Apple (necessari per descarregar Xcode).

Instal·lació de Xcode

Xcode és l'entorn de desenvolupament integrat (IDE) per a macOS que inclou tot el que necessites per crear aplicacions en Objective-C.

Passos per instal·lar Xcode

  1. Obre l'App Store: Llença l'aplicació App Store al teu Mac.
  2. Cerca Xcode: Escriu "Xcode" a la barra de cerca i prem Enter.
  3. Descarrega Xcode: Fes clic al botó "Get" al costat de Xcode, després fes clic a "Install".
  4. Espera la instal·lació: Xcode és una aplicació gran, així que el procés de descàrrega i instal·lació pot trigar una estona.
  5. Obre Xcode: Un cop instal·lat, obre Xcode des de la carpeta Aplicacions o Launchpad.

Configuració inicial

Quan obris Xcode per primera vegada:

  • Accepta l'acord de llicència: Se't demanarà que acceptis els termes i condicions.
  • Instal·la components addicionals: Xcode pot demanar-te que instal·lis components addicionals. Fes clic a "Install" i introdueix la contrasenya del sistema si és necessari.

Creació del teu primer projecte

Creem un projecte senzill de "Hello, World!" per assegurar-nos que tot està configurat correctament.

Passos per crear un nou projecte

  1. Obre Xcode: Llença Xcode des de la teva carpeta d'Aplicacions.
  2. Crea un nou projecte: Selecciona "Create a new Xcode project" des de la pantalla de benvinguda.
  3. Tria una plantilla: Selecciona "macOS" de les opcions de plataforma, després tria "Command Line Tool" i fes clic a "Next".
  4. Configura el projecte:
    • Nom del producte: Introdueix "HelloWorld".
    • Equip: Si tens un compte de desenvolupador d'Apple, selecciona el teu equip. Si no, tria "None".
    • Nom de l'organització: Introdueix el teu nom o el nom de la teva organització.
    • Identificador de l'organització: Normalment un nom de domini invers, per exemple, "com.example".
    • Llenguatge: Selecciona "Objective-C".
    • Fes clic a "Next".
  5. Desa el projecte: Tria una ubicació per desar el teu projecte i fes clic a "Create".

Escriure el teu primer codi en Objective-C

  1. Navega a main.m: Al navegador de projectes, troba i fes clic a main.m.
  2. Reemplaça el codi: Reemplaça el codi existent amb el següent:
#import <Foundation/Foundation.h>

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        // insert code here...
        NSLog(@"Hello, World!");
    }
    return 0;
}
  1. Executa el projecte: Fes clic al botó "Run" (una icona de reproducció) a la barra d'eines o prem Cmd + R.

Explicació del codi

  • #import <Foundation/Foundation.h>: Importa el framework Foundation, que proporciona classes i funcions essencials.
  • int main(int argc, const char * argv[]): La funció principal on comença l'execució del programa.
  • @autoreleasepool { ... }: Gestiona la memòria per als objectes creats dins del seu àmbit.
  • NSLog(@"Hello, World!");: Imprimeix "Hello, World!" a la consola.

Configuració de les preferències de Xcode

Per millorar la teva experiència de desenvolupament, pots configurar les preferències de Xcode.

Preferències clau per configurar

  • Edició de text: Ves a Xcode > Preferences > Text Editing per personalitzar la mida de la font, els temes de colors i les configuracions d'autocompleció de codi.
  • Control de versions: Activa o configura Git per al control de versions a Xcode > Preferences > Source Control.
  • Comportaments: Configura comportaments personalitzats per a diverses accions (per exemple, executar proves, construir projectes) a Xcode > Preferences > Behaviors.

Conclusió

Configurar l'entorn de desenvolupament és crucial per a una experiència de programació fluida. En instal·lar Xcode, crear el teu primer projecte i configurar les preferències, ja estàs preparat per començar a codificar en Objective-C. A les següents seccions, ens endinsarem en els fonaments del llenguatge i les característiques avançades.
