Introducció

En Objective-C, els operadors són símbols especials que realitzen operacions sobre variables i valors. Les expressions combinen variables, constants i operadors per produir un nou valor. Entendre els operadors i les expressions és fonamental per escriure codi Objective-C efectiu i eficient.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques com la suma, la resta, la multiplicació i la divisió.

  • Suma (+): Suma dos operands.
  • Resta (-): Resta el segon operand del primer.
  • Multiplicació (*): Multiplica dos operands.
  • Divisió (/): Divideix el primer operand pel segon.
  • Mòdul (%): Retorna el residu d'una operació de divisió.

Exemple

int a = 10;
int b = 3;

int sum = a + b;        // sum és 13
int difference = a - b; // difference és 7
int product = a * b;    // product és 30
int quotient = a / b;   // quotient és 3
int remainder = a % b;  // remainder és 1

Operadors Relacionals

Els operadors relacionals comparen dos valors i retornen un resultat booleà (YES o NO en Objective-C).

  • Igual a (==): Comprova si dos operands són iguals.
  • No igual a (!=): Comprova si dos operands no són iguals.
  • Més gran que (>): Comprova si el primer operand és més gran que el segon.
  • Menys que (<): Comprova si el primer operand és menys que el segon.
  • Més gran o igual a (>=): Comprova si el primer operand és més gran o igual que el segon.
  • Menys o igual a (<=): Comprova si el primer operand és menys o igual que el segon.

Exemple

int a = 10;
int b = 20;

BOOL isEqual = (a == b);       // isEqual és NO
BOOL isNotEqual = (a != b);    // isNotEqual és YES
BOOL isGreater = (a > b);      // isGreater és NO
BOOL isLess = (a < b);         // isLess és YES
BOOL isGreaterOrEqual = (a >= b); // isGreaterOrEqual és NO
BOOL isLessOrEqual = (a <= b);    // isLessOrEqual és YES

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions o valors booleans.

  • AND lògic (&&): Retorna YES si ambdós operands són YES.
  • OR lògic (||): Retorna YES si almenys un operand és YES.
  • NOT lògic (!): Retorna YES si l'operand és NO.

Exemple

BOOL a = YES;
BOOL b = NO;

BOOL resultAND = (a && b); // resultAND és NO
BOOL resultOR = (a || b);  // resultOR és YES
BOOL resultNOT = !a;       // resultNOT és NO

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació simple (=): Assigna l'operand de la dreta a l'operand de l'esquerra.
  • Suma i assigna (+=): Suma l'operand de la dreta a l'operand de l'esquerra i assigna el resultat a l'operand de l'esquerra.
  • Resta i assigna (-=): Resta l'operand de la dreta a l'operand de l'esquerra i assigna el resultat a l'operand de l'esquerra.
  • Multiplica i assigna (*=): Multiplica l'operand de la dreta amb l'operand de l'esquerra i assigna el resultat a l'operand de l'esquerra.
  • Divideix i assigna (/=): Divideix l'operand de l'esquerra per l'operand de la dreta i assigna el resultat a l'operand de l'esquerra.
  • Mòdul i assigna (%=): Pren el mòdul utilitzant l'operand de l'esquerra i l'operand de la dreta i assigna el resultat a l'operand de l'esquerra.

Exemple

int a = 10;
a += 5;  // a és ara 15
a -= 3;  // a és ara 12
a *= 2;  // a és ara 24
a /= 4;  // a és ara 6
a %= 4;  // a és ara 2

Operador Condicional (Ternari)

L'operador condicional és una forma abreujada de l'instrucció if-else. Pren tres operands.

  • Sintaxi: condition ? expression1 : expression2

Exemple

int a = 10;
int b = 20;

int max = (a > b) ? a : b; // max és 20

Operadors a Nivell de Bit

Els operadors a nivell de bit realitzen operacions sobre les representacions binàries dels enters.

  • AND (&): Realitza una operació AND a nivell de bit.
  • OR (|): Realitza una operació OR a nivell de bit.
  • XOR (^): Realitza una operació XOR a nivell de bit.
  • NOT (~): Realitza una operació NOT a nivell de bit.
  • Desplaçament a l'esquerra (<<): Desplaça bits a l'esquerra.
  • Desplaçament a la dreta (>>): Desplaça bits a la dreta.

Exemple

int a = 5;  // Binari: 0101
int b = 3;  // Binari: 0011

int andResult = a & b; // andResult és 1 (Binari: 0001)
int orResult = a | b;  // orResult és 7 (Binari: 0111)
int xorResult = a ^ b; // xorResult és 6 (Binari: 0110)
int notResult = ~a;    // notResult és -6 (Binari: 1010 en complement a 2)
int leftShift = a << 1; // leftShift és 10 (Binari: 1010)
int rightShift = a >> 1; // rightShift és 2 (Binari: 0010)

Resum

Entendre els operadors i les expressions és crucial per manipular dades i controlar el flux dels teus programes Objective-C. Aquest tema ha cobert:

  • Operadors aritmètics per a operacions matemàtiques bàsiques.
  • Operadors relacionals per comparar valors.
  • Operadors lògics per combinar expressions booleans.
  • Operadors d'assignació per assignar i actualitzar valors de variables.
  • L'operador condicional (ternari) per expressions condicionals concises.
  • Operadors a nivell de bit per realitzar operacions sobre representacions binàries.

En dominar aquests operadors, podràs escriure codi Objective-C més eficient i llegible.
