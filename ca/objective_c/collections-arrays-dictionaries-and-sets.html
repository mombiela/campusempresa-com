Col·leccions: Arrays, Diccionaris i Conjunts
Treballant amb Fitxers ►

Introducció

En Objective-C, les col·leccions són estructures de dades fonamentals que et permeten emmagatzemar i gestionar grups d'objectes. Els tipus de col·leccions principals són Arrays, Diccionaris i Conjunts. Cadascun d'aquests té característiques i casos d'ús únics.

Arrays

Els arrays en Objective-C són col·leccions ordenades d'objectes. Poden emmagatzemar múltiples objectes de qualsevol tipus, però tots els objectes han de ser instàncies de NSObject o les seves subclasses.

Creació i Inicialització d'Arrays

Pots crear i inicialitzar arrays utilitzant diversos mètodes:

// Creació d'un array buit
NSArray *emptyArray = [NSArray array];

// Creació d'un array amb objectes
NSArray *fruits = [NSArray arrayWithObjects:@"Apple", @"Banana", @"Cherry", nil];

// Usant literals moderns d'Objective-C
NSArray *colors = @[@"Red", @"Green", @"Blue"];

Accés als Elements de l'Array

Pots accedir als elements d'un array utilitzant el seu índex:

NSString *firstFruit = fruits[0]; // Accés al primer element
NSLog(@"First fruit: %@", firstFruit);

Iteració sobre Arrays

Pots iterar sobre arrays utilitzant un bucle for o enumeració ràpida:

// Usant un bucle for
for (int i = 0; i < [fruits count]; i++) {
    NSLog(@"Fruit: %@", fruits[i]);
}

// Usant enumeració ràpida
for (NSString *fruit in fruits) {
    NSLog(@"Fruit: %@", fruit);
}

Diccionaris

Els diccionaris en Objective-C són col·leccions que emmagatzemen parelles clau-valor. Cada clau ha de ser única i està associada amb un valor.

Creació i Inicialització de Diccionaris

Pots crear i inicialitzar diccionaris utilitzant diversos mètodes:

// Creació d'un diccionari buit
NSDictionary *emptyDict = [NSDictionary dictionary];

// Creació d'un diccionari amb parelles clau-valor
NSDictionary *person = [NSDictionary dictionaryWithObjectsAndKeys:
                        @"John", @"firstName",
                        @"Doe", @"lastName",
                        @30, @"age", nil];

// Usant literals moderns d'Objective-C
NSDictionary *car = @{@"make": @"Toyota", @"model": @"Camry", @"year": @2020};

Accés als Valors del Diccionari

Pots accedir als valors d'un diccionari utilitzant les seves claus:

NSString *firstName = person[@"firstName"];
NSLog(@"First name: %@", firstName);

Iteració sobre Diccionaris

Pots iterar sobre diccionaris utilitzant enumeració ràpida:

for (NSString *key in person) {
    NSLog(@"%@: %@", key, person[key]);
}

Conjunts

Els conjunts en Objective-C són col·leccions no ordenades d'objectes únics. Són útils quan necessites assegurar-te que no hi hagi duplicats.

Creació i Inicialització de Conjunts

Pots crear i inicialitzar conjunts utilitzant diversos mètodes:

// Creació d'un conjunt buit
NSSet *emptySet = [NSSet set];

// Creació d'un conjunt amb objectes
NSSet *uniqueNumbers = [NSSet setWithObjects:@1, @2, @3, @4, nil];

// Usant literals moderns d'Objective-C
NSSet *letters = [NSSet setWithArray:@[@"A", @"B", @"C"]];

Accés als Elements del Conjunt

Com que els conjunts no estan ordenats, no pots accedir als elements per índex. En canvi, pots comprovar la seva pertinença:

if ([uniqueNumbers containsObject:@2]) {
    NSLog(@"The set contains the number 2");
}

Iteració sobre Conjunts

Pots iterar sobre conjunts utilitzant enumeració ràpida:

for (NSNumber *number in uniqueNumbers) {
    NSLog(@"Number: %@", number);
}

Resum

En aquest tema, hem cobert els tres tipus principals de col·leccions en Objective-C: Arrays, Diccionaris i Conjunts. Els arrays són col·leccions ordenades, els diccionaris emmagatzemen parelles clau-valor, i els conjunts són col·leccions no ordenades d'objectes únics. Entendre aquestes col·leccions i com utilitzar-les eficaçment és crucial per gestionar dades en els teus programes Objective-C. Practica creant, accedint i iterant sobre aquestes col·leccions per convertir-te en un expert en el seu ús.
