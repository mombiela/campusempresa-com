Introducció

La gestió de comptes d'usuari és un aspecte crític de l'administració del sistema a OpenVMS. Aquest tema et guiarà a través dels aspectes essencials de la creació, gestió i seguretat dels comptes d'usuari des d'un nivell principiant fins a un nivell avançat.

Nivell Principiant

Comprensió dels Comptes d'Usuari

  • Compte d'Usuari: Un compte d'usuari a OpenVMS és un identificador únic que permet a una persona iniciar sessió i utilitzar el sistema.
  • Nom d'Usuari: El nom assignat a un compte d'usuari.
  • Contrasenya: Una paraula o frase secreta utilitzada per autenticar l'usuari.

Creació d'un Compte d'Usuari

Per crear un compte d'usuari, cal utilitzar la utilitat AUTHORIZE. A continuació es mostra una guia pas a pas:

  1. Accedir a la Utilitat AUTHORIZE:

    $ RUN SYS$SYSTEM:AUTHORIZE

  2. Afegir un Nou Usuari:

    UAF> ADD username /PASSWORD=password /UIC=[group,member] /DEVICE=device /DIRECTORY=directory
    • username: El nom del nou usuari.
    • password: La contrasenya inicial per a l'usuari.
    • UIC: Codi d'Identificació d'Usuari, que és únic per a cada usuari.
    • device: El dispositiu de disc on residirà el directori de l'usuari.
    • directory: El camí del directori per a l'usuari.

  3. Exemple:

    UAF> ADD JOHNDOE /PASSWORD=welcome123 /UIC=[200,100] /DEVICE=SYS$SYSDEVICE /DIRECTORY=[JOHNDOE]

Modificació d'un Compte d'Usuari

Per modificar un compte d'usuari existent, utilitza la comanda MODIFY a la utilitat AUTHORIZE:

UAF> MODIFY username /parameter=value
  • Exemple: Canviar la contrasenya per a l'usuari JOHNDOE: 
    UAF> MODIFY JOHNDOE /PASSWORD=newpassword

Eliminació d'un Compte d'Usuari

Per eliminar un compte d'usuari, utilitza la comanda REMOVE:

UAF> REMOVE username
  • Exemple: Eliminar l'usuari JOHNDOE: 
    UAF> REMOVE JOHNDOE

Nivell Intermedi

Atributs del Compte d'Usuari

  • Privilegis: Drets especials assignats a un compte d'usuari per realitzar tasques específiques.
  • Quotes: Límits sobre els recursos que un usuari pot consumir.
  • Flags de Sessió: Controlen diversos aspectes del procés d'inici de sessió.

Gestió de Privilegis

Per assignar o modificar privilegis, utilitza la comanda MODIFY:

UAF> MODIFY username /PRIVILEGES=(priv1,priv2,...)
  • Exemple: Concedir privilegis SYSPRV i OPER a JOHNDOE: 
    UAF> MODIFY JOHNDOE /PRIVILEGES=(SYSPRV,OPER)

Establiment de Quotes

Les quotes controlen la quantitat de recursos del sistema que un usuari pot utilitzar:

UAF> MODIFY username /WSQUOTA=value /PGFLQUOTA=value /BYTLM=value
  • Exemple: Establir quotes per a JOHNDOE: 
    UAF> MODIFY JOHNDOE /WSQUOTA=1024 /PGFLQUOTA=50000 /BYTLM=100000

Flags de Sessió

Els flags de sessió es poden establir per controlar el comportament de l'inici de sessió de l'usuari:

UAF> MODIFY username /FLAGS=(flag1,flag2,...)
  • Exemple: Establir el flag DISUSER per desactivar el compte de JOHNDOE: 
    UAF> MODIFY JOHNDOE /FLAGS=DISUSER

Nivell Avançat

Funcions Avançades de Seguretat

  • ACLs (Llistes de Control d'Accés): Defineixen permisos d'accés detallats.
  • Auditoria: Monitoritza i registra les activitats dels usuaris per a finalitats de seguretat.

Gestió d'ACLs

Per gestionar les ACLs, utilitza la comanda SET ACL:

$ SET ACL /ACL=(IDENTIFIER=username,ACCESS=access) object
  • Exemple: Concedir a JOHNDOE accés de lectura i escriptura a un fitxer: 
    $ SET ACL /ACL=(IDENTIFIER=JOHNDOE,ACCESS=READ+WRITE) MYFILE.TXT

Configuració de l'Auditoria

Per habilitar l'auditoria, modifica la configuració d'auditoria del sistema:

  1. Editar la Configuració del Servidor d'Auditoria:

    $ RUN SYS$SYSTEM:SYSMAN
SYSMAN> SET PROFILE /AUDIT

  2. Habilitar l'Auditoria per a Esdeveniments Específics:

    $ SET AUDIT /ENABLE=SECURITY /CLASS=LOGIN

Automatització de la Gestió de Comptes d'Usuari

  • Scripts DCL: Utilitza scripts de Digital Command Language (DCL) per automatitzar tasques repetitives.

  • Exemple d'Script: Crear un script per afegir múltiples usuaris:

    $! Afegir múltiples usuaris
$ OPEN/READ userfile USERS.DAT
$ LOOP:
$   READ/END=EOF userfile line
$   username = F$ELEMENT(0, " ", line)
$   password = F$ELEMENT(1, " ", line)
$   UIC = F$ELEMENT(2, " ", line)
$   device = F$ELEMENT(3, " ", line)
$   directory = F$ELEMENT(4, " ", line)
$   RUN SYS$SYSTEM:AUTHORIZE
UAF> ADD 'username' /PASSWORD='password' /UIC='UIC' /DEVICE='device' /DIRECTORY='directory'
UAF> EXIT
$   GOTO LOOP
$ EOF:
$ CLOSE userfile
$ EXIT

Conclusió

La gestió de comptes d'usuari a OpenVMS implica la creació, modificació i eliminació de comptes d'usuari, la gestió de privilegis i quotes, i la implementació de funcions avançades de seguretat com les ACLs i l'auditoria. En dominar aquestes tasques, pots assegurar-te que el teu sistema OpenVMS es mantingui segur i eficient.
