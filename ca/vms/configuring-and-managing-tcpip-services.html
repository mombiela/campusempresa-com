Introducció

En aquest mòdul, explorarem com configurar i gestionar serveis TCP/IP a OpenVMS. Això inclou entendre els conceptes bàsics de TCP/IP, configurar interfícies de xarxa, gestionar serveis i resoldre problemes comuns. Al final d'aquest mòdul, tindràs una comprensió sòlida de com treballar amb TCP/IP a OpenVMS des d'un nivell principiant fins a un nivell avançat.

Entendre els Conceptes Bàsics de TCP/IP

Abans d'entrar en la configuració i gestió, és crucial entendre els conceptes bàsics de TCP/IP.

  • Suite de Protocols TCP/IP: Un conjunt de protocols de comunicació utilitzats per a internet i xarxes similars.
  • Adreça IP: Un identificador únic per a un dispositiu en una xarxa.
  • Màscara de Subxarxa: Utilitzada per dividir una adreça IP en parts de xarxa i host.
  • Passarel·la Per Defecte: El dispositiu que dirigeix el tràfic d'una xarxa local a altres xarxes.

Configuració d'Interfícies de Xarxa

Configurar interfícies de xarxa és el primer pas per configurar serveis TCP/IP a OpenVMS.

Veure Interfícies de Xarxa

Per veure les interfícies de xarxa actuals i les seves configuracions, utilitza la següent comanda:

$ TCPIP SHOW INTERFACE

Configurar una Adreça IP

Per configurar una adreça IP per a una interfície de xarxa, utilitza la següent comanda:

$ TCPIP SET INTERFACE nom-interfície /HOST=adreça-ip /NETWORK_MASK=màscara-subxarxa

Exemple:

$ TCPIP SET INTERFACE WE0 /HOST=192.168.1.10 /NETWORK_MASK=255.255.255.0

Configurar la Passarel·la Per Defecte

Per configurar la passarel·la per defecte, utilitza la següent comanda:

$ TCPIP SET ROUTE /DEFAULT /GATEWAY=adreça-ip

Exemple:

$ TCPIP SET ROUTE /DEFAULT /GATEWAY=192.168.1.1

Gestió de Serveis TCP/IP

OpenVMS proporciona diversos serveis TCP/IP que es poden gestionar a través de la utilitat de gestió TCP/IP.

Activar i Desactivar Serveis

Per activar o desactivar un servei, utilitza les següents comandes:

  • Activar un Servei:
$ TCPIP ENABLE SERVICE nom-servei
  • Desactivar un Servei:
$ TCPIP DISABLE SERVICE nom-servei

Exemple:

$ TCPIP ENABLE SERVICE FTP
$ TCPIP DISABLE SERVICE TELNET

Veure l'Estat del Servei

Per veure l'estat d'un servei, utilitza la següent comanda:

$ TCPIP SHOW SERVICE nom-servei

Exemple:

$ TCPIP SHOW SERVICE FTP

Configuració Avançada

Per a usuaris avançats, OpenVMS permet un control més granular sobre els serveis TCP/IP.

Configurar DNS

Per configurar els paràmetres DNS, utilitza les següents comandes:

  • Configurar el Resolutor DNS:
$ TCPIP SET NAME_SERVICE /DOMAIN=nom-domini /SERVERS=adreça-ip

Exemple:

$ TCPIP SET NAME_SERVICE /DOMAIN=example.com /SERVERS=8.8.8.8,8.8.4.4

Configurar Regles de Tallafocs

Per configurar regles de tallafocs, utilitza les següents comandes:

  • Afegir una Regla de Tallafocs:
$ TCPIP SET FIREWALL /ADD /RULE=nom-regla /ACTION=acció /PROTOCOL=protocol /SRC=origen /DST=destinació

Exemple:

$ TCPIP SET FIREWALL /ADD /RULE=ALLOW_HTTP /ACTION=ALLOW /PROTOCOL=TCP /SRC=ANY /DST=192.168.1.10:80

Resolució de Problemes Comuns

La resolució de problemes és una habilitat essencial per gestionar serveis TCP/IP. Aquí tens alguns problemes comuns i les seves solucions:

  • No es Pot Arribar a la Passarel·la Per Defecte:

    • Comprova la configuració de la interfície de xarxa.
    • Verifica la configuració de la passarel·la per defecte.

  • El Servei No Respon:

    • Assegura't que el servei està activat.
    • Comprova les regles del tallafocs.
    • Verifica la configuració del servei.

  • La Resolució DNS Falla:

    • Comprova la configuració del servidor DNS.
    • Verifica la connectivitat de xarxa amb el servidor DNS.

Conclusió

Configurar i gestionar serveis TCP/IP a OpenVMS implica entendre els conceptes bàsics de TCP/IP, configurar interfícies de xarxa, gestionar serveis i resoldre problemes. Seguint els passos descrits en aquest mòdul, hauràs de ser capaç de configurar i mantenir efectivament serveis TCP/IP a OpenVMS, assegurant una comunicació de xarxa fiable per als teus sistemes.
