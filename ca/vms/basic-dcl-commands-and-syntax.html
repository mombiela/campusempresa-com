Introducció a DCL (Digital Command Language)

Digital Command Language (DCL) és el llenguatge de comandes utilitzat en el sistema operatiu OpenVMS. És un llenguatge de script potent que permet als usuaris interactuar amb el sistema, gestionar fitxers i executar programes.

Conceptes clau de DCL

  • Indicador de comanda: L'indicador de comanda de DCL es representa típicament amb un signe de dòlar ($).
  • Insensibilitat a majúscules i minúscules: Les comandes de DCL no distingeixen entre majúscules i minúscules.
  • Estructura de la comanda: Les comandes segueixen típicament l'estructura: COMMAND [qualifiers] [parameters].

Comandes comunes de DCL

Aquí teniu algunes de les comandes de DCL més utilitzades:

Llistat de directoris

  • Comanda: DIRECTORY
  • Descripció: Llista els fitxers del directori actual.
  • Exemple: 
    $ DIRECTORY
    Aquesta comanda mostrarà una llista de fitxers del directori actual.

Còpia de fitxers

  • Comanda: COPY
  • Descripció: Copia fitxers d'una ubicació a una altra.
  • Exemple: 
    $ COPY source_file.txt destination_file.txt
    Aquesta comanda copia source_file.txt a destination_file.txt.

Eliminació de fitxers

  • Comanda: DELETE
  • Descripció: Elimina els fitxers especificats.
  • Exemple: 
    $ DELETE unwanted_file.txt
    Aquesta comanda elimina unwanted_file.txt.

Canvi de nom de fitxers

  • Comanda: RENAME
  • Descripció: Canvia el nom d'un fitxer.
  • Exemple: 
    $ RENAME old_name.txt new_name.txt
    Aquesta comanda canvia el nom de old_name.txt a new_name.txt.

Mostra el contingut del fitxer

  • Comanda: TYPE
  • Descripció: Mostra el contingut d'un fitxer.
  • Exemple: 
    $ TYPE file.txt
    Aquesta comanda mostra el contingut de file.txt.

Ús de qualificadors

Els qualificadors modifiquen el comportament de les comandes de DCL. Van precedits d'una barra inclinada (/).

Exemple de qualificadors

  • Comanda: DIRECTORY
  • Qualificador: /FULL
  • Descripció: Proporciona informació detallada sobre els fitxers.
  • Exemple: 
    $ DIRECTORY/FULL
    Aquesta comanda llista els fitxers amb informació detallada.

Combinació de comandes

DCL permet combinar múltiples comandes en una sola línia utilitzant el punt i coma (;).

Exemple de combinació de comandes

$ COPY file1.txt file2.txt; DELETE file1.txt

Aquesta comanda copia file1.txt a file2.txt i després elimina file1.txt.

Resum

En aquesta secció, hem cobert els conceptes bàsics de les comandes i la sintaxi de DCL, incloent:

  • L'estructura i la insensibilitat a majúscules i minúscules de les comandes de DCL.
  • Comandes comunament utilitzades com DIRECTORY, COPY, DELETE, RENAME i TYPE.
  • L'ús de qualificadors per modificar el comportament de les comandes.
  • La combinació de múltiples comandes en una sola línia.

Entendre aquestes comandes bàsiques de DCL i la seva sintaxi és essencial per gestionar efectivament els fitxers i interactuar amb el sistema operatiu OpenVMS. Practiqueu aquestes comandes per sentir-vos més còmodes amb DCL i preparar-vos per a temes més avançats.
