Visió General

OpenVMS (Open Virtual Memory System) és un sistema operatiu multiusuari i multiprocés basat en memòria virtual dissenyat per a ús en temps compartit, processament per lots i processament de transaccions. Aquest curs et guiarà a través dels conceptes bàsics del desenvolupament d'aplicacions OpenVMS, des de nivells principiants fins a avançats.

Començant amb OpenVMS

Què és OpenVMS?

  • Multiusuari i multiprocés: Suporta múltiples usuaris i processos simultàniament.
  • Memòria virtual: Gestió eficient de la memòria.
  • Alta disponibilitat: Dissenyat per a aplicacions crítiques.
  • Seguretat: Funcions de seguretat robustes.

Configurant el Teu Entorn

  • Accedint a OpenVMS: Utilitzant emuladors de terminal o SSH.
  • Comandes Bàsiques: 
    $ SHOW SYSTEM
$ DIRECTORY
$ TYPE filename.txt

Operacions Bàsiques de Fitxers

  • Creant Fitxers: 
    $ CREATE filename.txt
  • Editant Fitxers: Utilitzant editors com EDT o EVE. 
    $ EDIT/TPU filename.txt
  • Eliminant Fitxers: 
    $ DELETE filename.txt

Programant en OpenVMS

Llenguatges Suportats

  • C/C++
  • Fortran
  • Pascal
  • COBOL
  • BASIC

Escriure el Teu Primer Programa en C

  • Creant un Fitxer Font: 
    $ CREATE hello.c
  • Programa C de Mostra: 
    #include <stdio.h>

int main() {
    printf("Hello, OpenVMS!\n");
    return 0;
}
  • Compilant i Executant: 
    $ CC hello.c
$ LINK hello
$ RUN hello

Temes Avançats

Serveis del Sistema

  • Visió General: Els serveis del sistema són rutines cridables proporcionades per OpenVMS per a diverses operacions a nivell de sistema.
  • Exemple: 
    #include <lib$routines.h>
#include <starlet.h>

int main() {
    unsigned long status;
    status = lib$put_output(&"Hello, OpenVMS System Service!\n");
    return status;
}

Comunicació Interprocessos

  • Busties: Utilitzades per a la comunicació entre processos. 
    $ CREATE/MAILBOX MBX1
$ WRITE MBX1 "Message to another process"

Utilitzant RMS (Record Management Services)

  • Visió General: RMS s'utilitza per a la gestió de fitxers i registres.
  • Exemple: 
    #include <rms.h>

struct FAB fab;
struct RAB rab;
char record[100];

int main() {
    fab = cc$rms_fab;
    fab.fab$l_fna = "example.dat";
    fab.fab$b_fns = strlen(fab.fab$l_fna);
    fab.fab$b_fac = FAB$M_BIO | FAB$M_GET;
    fab.fab$b_shr = FAB$M_SHRGET | FAB$M_SHRPUT;

    rab = cc$rms_rab;
    rab.rab$l_fab = &fab;
    rab.rab$l_rop = RAB$M_RAH | RAB$M_RRL;
    rab.rab$l_ubf = record;
    rab.rab$w_usz = sizeof(record);

    sys$open(&fab);
    sys$connect(&rab);
    sys$get(&rab);
    printf("Record: %s\n", record);
    sys$disconnect(&rab);
    sys$close(&fab);

    return 0;
}

Conclusió

OpenVMS és un sistema operatiu potent i versàtil adequat per a diversos tipus d'aplicacions. Aquest curs t'ha introduït als conceptes bàsics del desenvolupament d'aplicacions OpenVMS, des de la configuració del teu entorn fins a escriure i executar programes. A mesura que progressis, exploraràs temes més avançats com els serveis del sistema, la comunicació interprocessos i RMS. Amb pràctica i exploració, et convertiràs en un desenvolupador competent d'aplicacions robustes en OpenVMS.
