Introducció al DCL (Digital Command Language)

Digital Command Language (DCL) és el llenguatge de script utilitzat en el sistema operatiu OpenVMS. S'utilitza per a l'automatització de tasques, la gestió del sistema i l'execució de comandes en mode batch.

Conceptes Clau

  • Procediments de Comanda: Els scripts escrits en DCL s'anomenen procediments de comanda.
  • Extensió de Fitxer: Els scripts DCL típicament tenen una extensió de fitxer .COM.
  • Execució: Els scripts DCL es poden executar de manera interactiva o en mode batch.

Escriure el teu Primer Script DCL

Per començar amb l'escriptura de scripts DCL, escrivim un script senzill que imprimeixi "Hello, World!" al terminal.

Exemple de Script

$! This is a comment
$ WRITE SYS$OUTPUT "Hello, World!"

Explicació

  • $! denota una línia de comentari.
  • $ WRITE SYS$OUTPUT "Hello, World!" és la comanda per imprimir text al terminal.

Guardar el Script

  1. Obre un editor de text.
  2. Escriu el script tal com es mostra a dalt.
  3. Guarda el fitxer amb una extensió .COM, per exemple, HELLO.COM.

Executar Scripts DCL

Execució Interactiva

Per executar el script de manera interactiva, utilitza la següent comanda al terminal:

$ @HELLO.COM

Execució en Batch

Per executar el script en mode batch, envia'l a la cua de batch:

$ SUBMIT HELLO.COM

Scripts DCL Avançats

A mesura que et familiaritzis amb DCL, pots començar a utilitzar funcions més avançades com variables, bucles i declaracions condicionals.

Variables

Les variables en DCL es defineixen utilitzant l'operador =.

Exemple

$ NAME = "OpenVMS"
$ WRITE SYS$OUTPUT "Welcome to ''NAME'"

Declaracions Condicionals

DCL suporta declaracions condicionals utilitzant IF i ENDIF.

Exemple

$ IF NAME .EQS. "OpenVMS" THEN WRITE SYS$OUTPUT "Correct!"
$ ELSE WRITE SYS$OUTPUT "Incorrect!"
$ ENDIF

Bucles

DCL suporta bucles utilitzant GOTO i etiquetes.

Exemple

$ COUNT = 1
$ LOOP:
$ IF COUNT .GT. 5 THEN GOTO ENDLOOP
$ WRITE SYS$OUTPUT "Count = ''COUNT'"
$ COUNT = COUNT + 1
$ GOTO LOOP
$ ENDLOOP:

Gestió d'Errors

La gestió d'errors en scripts DCL és crucial per a una execució robusta dels scripts. Utilitza la comanda ON per gestionar errors.

Exemple

$ ON ERROR THEN GOTO ERROR_HANDLER
$ WRITE SYS$OUTPUT "This is a test."
$ EXIT
$ ERROR_HANDLER:
$ WRITE SYS$OUTPUT "An error occurred."
$ EXIT

Resum

En aquest tema, hem cobert els conceptes bàsics d'escriure i executar scripts DCL en OpenVMS. Hem començat amb un script senzill de "Hello, World!", hem après com executar scripts de manera interactiva i en mode batch, i hem explorat tècniques avançades d'scripting com variables, declaracions condicionals, bucles i gestió d'errors. Amb aquestes habilitats, pots automatitzar tasques i gestionar el teu sistema OpenVMS de manera més eficient.
