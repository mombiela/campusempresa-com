Introducció

OpenVMS (Open Virtual Memory System) és un sistema operatiu de memòria virtual multiusuari i multiprocés dissenyat per a ús en temps compartit, processament per lots i processament de transaccions. Entendre les comandes bàsiques d'OpenVMS és essencial per navegar i gestionar el sistema de manera efectiva.

Conceptes clau

  • Interfície de línia de comandes (CLI): OpenVMS utilitza una CLI per a la interacció amb l'usuari.
  • DCL (Digital Command Language): El llenguatge de comandes principal utilitzat a OpenVMS.
  • Sistema de fitxers: Entendre l'estructura de fitxers i com navegar-hi és crucial.

Comandes bàsiques de navegació

Mostrar el directori actual

Per mostrar el directori actual, utilitza la comanda SHOW DEFAULT:

$ SHOW DEFAULT

Aquesta comanda mostrarà la ruta del directori actual.

Canviar de directori

Per canviar de directori, utilitza la comanda SET DEFAULT seguida de la ruta del directori:

$ SET DEFAULT [directory_path]

Exemple:

$ SET DEFAULT [USER.DOCUMENTS]

Llistar el contingut del directori

Per llistar el contingut d'un directori, utilitza la comanda DIRECTORY:

$ DIRECTORY

També pots especificar un directori per llistar-ne el contingut:

$ DIRECTORY [USER.DOCUMENTS]

Comandes de gestió de fitxers

Crear un fitxer

Per crear un nou fitxer, utilitza la comanda CREATE:

$ CREATE filename.ext

Exemple:

$ CREATE MYFILE.TXT

Aquesta comanda obrirà un editor on podràs introduir el contingut del fitxer.

Esborrar un fitxer

Per esborrar un fitxer, utilitza la comanda DELETE:

$ DELETE filename.ext

Exemple:

$ DELETE MYFILE.TXT

Canviar el nom d'un fitxer

Per canviar el nom d'un fitxer, utilitza la comanda RENAME:

$ RENAME oldfilename.ext newfilename.ext

Exemple:

$ RENAME MYFILE.TXT YOURFILE.TXT

Comandes d'informació del sistema

Mostrar informació del sistema

Per mostrar informació del sistema, utilitza la comanda SHOW SYSTEM:

$ SHOW SYSTEM

Aquesta comanda proporciona informació sobre l'estat del sistema, incloent-hi els processos actius.

Mostrar informació de l'usuari

Per mostrar informació sobre l'usuari actual, utilitza la comanda SHOW USER:

$ SHOW USER

Comandes de gestió de processos

Mostrar processos

Per mostrar una llista de processos, utilitza la comanda SHOW PROCESS:

$ SHOW PROCESS

També pots especificar un procés en particular:

$ SHOW PROCESS /ID=process_id

Aturar un procés

Per aturar un procés, utilitza la comanda STOP seguida de l'ID del procés:

$ STOP process_id

Exemple:

$ STOP 12345

Resum

Entendre i utilitzar les comandes bàsiques d'OpenVMS és essencial per a una navegació i gestió efectiva del sistema. Aquest tema ha cobert les comandes fonamentals per navegar per directoris, gestionar fitxers i obtenir informació del sistema i dels processos. El domini d'aquestes comandes proporcionarà una base sòlida per a operacions més avançades d'OpenVMS.
