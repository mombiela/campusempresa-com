Introducció

OpenVMS és un sistema operatiu versàtil que admet diversos llenguatges de programació d'alt nivell, inclosos C i Fortran. Aquesta secció us guiarà des dels conceptes bàsics fins a l'ús avançat d'aquests llenguatges a OpenVMS, proporcionant explicacions clares, exemples i exercicis.

Començant amb llenguatges d'alt nivell a OpenVMS

Configuració del vostre entorn

  • Instal·leu els compiladors necessaris: Assegureu-vos que els compiladors de C i Fortran estiguin instal·lats al vostre sistema OpenVMS.
  • Configureu les variables d'entorn: Configureu el vostre entorn per utilitzar els compiladors de manera efectiva. 
    $ SET DEFAULT SYS$SYSTEM
$ DEFINE CC$COMPILER CC
$ DEFINE FORTRAN$COMPILER FORTRAN

Escriure el vostre primer programa

Exemple en C

  1. Creeu un programa senzill en C: 
    #include <stdio.h>

int main() {
    printf("Hola, OpenVMS!\n");
    return 0;
}
  2. Guardeu el fitxer com hello.c.

Exemple en Fortran

  1. Creeu un programa senzill en Fortran: 
    PROGRAM HELLO
PRINT *, 'Hola, OpenVMS!'
END PROGRAM HELLO
  2. Guardeu el fitxer com hello.f90.

Compilar i executar programes

  • Compileu el programa en C: 
    $ CC hello.c
$ LINK hello
$ RUN hello
  • Compileu el programa en Fortran: 
    $ FORTRAN hello.f90
$ LINK hello
$ RUN hello

Conceptes intermedis

Gestió d'entrada i sortida

Exemple en C

  • Llegir entrada de l'usuari: 
    #include <stdio.h>

int main() {
    char name[50];
    printf("Introdueix el teu nom: ");
    scanf("%s", name);
    printf("Hola, %s!\n", name);
    return 0;
}

Exemple en Fortran

  • Llegir entrada de l'usuari: 
    PROGRAM GREET
CHARACTER(LEN=50) :: name
PRINT *, 'Introdueix el teu nom:'
READ *, name
PRINT *, 'Hola, ', TRIM(name), '!'
END PROGRAM GREET

Operacions amb fitxers

Exemple en C

  • Llegir i escriure en un fitxer: 
    #include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w");
    fprintf(file, "Hola, OpenVMS!\n");
    fclose(file);

    file = fopen("example.txt", "r");
    char line[100];
    fgets(line, sizeof(line), file);
    printf("%s", line);
    fclose(file);
    return 0;
}

Exemple en Fortran

  • Llegir i escriure en un fitxer: 
    PROGRAM FILE_IO
INTEGER :: unit = 10
CHARACTER(LEN=100) :: line

OPEN(UNIT=unit, FILE='example.txt', STATUS='NEW')
WRITE(unit, *) 'Hola, OpenVMS!'
CLOSE(unit)

OPEN(UNIT=unit, FILE='example.txt', STATUS='OLD')
READ(unit, *) line
PRINT *, TRIM(line)
CLOSE(unit)
END PROGRAM FILE_IO

Temes avançats

Interfície entre C i Fortran

  • Cridar Fortran des de C: 
    extern void fortran_subroutine();

int main() {
    fortran_subroutine();
    return 0;
}
  • Subrutina Fortran: 
    SUBROUTINE FORTRAN_SUBROUTINE
PRINT *, 'Cridat des de C!'
END SUBROUTINE FORTRAN_SUBROUTINE

Optimització del rendiment

  • Optimitzacions del compilador: Utilitzeu flags del compilador per optimitzar el rendiment.
    • C: CC /OPTIMIZE=LEVEL=4 hello.c
    • Fortran: FORTRAN /OPTIMIZE=LEVEL=4 hello.f90

Tècniques de depuració

  • Ús del depurador d'OpenVMS:
    • Iniciar el depurador: 
      $ DEBUG hello
    • Establir punts de ruptura i avançar pel codi: 
      DBG> SET BREAK main
DBG> GO
DBG> STEP

Conclusió

En aquesta secció, heu après a utilitzar llenguatges d'alt nivell com C i Fortran a OpenVMS, des de la configuració del vostre entorn fins a escriure, compilar i executar programes. També heu explorat temes intermedis i avançats, incloent operacions amb fitxers, interfície entre llenguatges, optimització del rendiment i tècniques de depuració. Practicant aquests conceptes, us convertireu en experts en el desenvolupament d'aplicacions robustes a la plataforma OpenVMS.
