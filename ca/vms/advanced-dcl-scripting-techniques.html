Introducció

En aquesta secció, ens endinsarem en tècniques avançades per fer scripts amb el Digital Command Language (DCL) a OpenVMS. Això es basarà en els coneixements fonamentals i cobrirà escenaris de scripting més complexos, estratègies d'optimització i millors pràctiques.

Conceptes Clau

  • Estructures de Control Avançades: Aprèn a utilitzar estructures de control complexes per gestionar el flux dels scripts.
  • Gestió d'Errors: Implementa una gestió d'errors robusta per fer que els scripts siguin més fiables.
  • Optimització del Rendiment: Tècniques per millorar el rendiment dels scripts DCL.
  • Scripting Modular: Divideix els scripts en mòduls reutilitzables.
  • Interfície amb Altres Llenguatges: Integra scripts DCL amb altres llenguatges de programació i eines.

Estructures de Control Avançades

Declaracions Condicionals

DCL admet lògica condicional complexa utilitzant IF, THEN, ELSE i ENDIF.

$ IF F$SEARCH("MYFILE.TXT") .EQS. "" THEN -
$    WRITE SYS$OUTPUT "Fitxer no trobat" -
$ ELSE -
$    WRITE SYS$OUTPUT "Fitxer trobat"
  • Explicació: Aquest script comprova si MYFILE.TXT existeix. Si existeix, imprimeix "Fitxer trobat"; en cas contrari, imprimeix "Fitxer no trobat".

Construccions de Bucle

Utilitza FOR, WHILE i GOTO per fer bucles.

$ COUNT = 1
$ WHILE COUNT .LE. 10 DO
$ BEGIN
$    WRITE SYS$OUTPUT "Compte: ''COUNT'"
$    COUNT = COUNT + 1
$ END
  • Explicació: Aquest bucle imprimeix els números de l'1 al 10.

Gestió d'Errors

Utilitzant `ON` i `SET NOON`

Gestiona els errors de manera elegant utilitzant ON i SET NOON.

$ ON WARNING THEN GOTO ERROR_HANDLER
$ SET NOON
$ ! La teva lògica de script aquí
$ EXIT
$ ERROR_HANDLER:
$ WRITE SYS$OUTPUT "S'ha produït un error"
$ EXIT
  • Explicació: Aquest script configura un gestor d'errors que imprimeix un missatge d'error si es produeix algun avís.

Optimització del Rendiment

Operacions de Fitxer Eficients

Minimitza les operacions d'E/S per millorar el rendiment.

$ OPEN/READ FILE MYFILE.TXT
$ READ_LOOP:
$ READ/END_OF_FILE=EOF FILE LINE
$ WRITE SYS$OUTPUT LINE
$ GOTO READ_LOOP
$ EOF:
$ CLOSE FILE
$ EXIT
  • Explicació: Aquest script llegeix i imprimeix cada línia de MYFILE.TXT de manera eficient minimitzant el nombre d'operacions d'obertura i tancament.

Scripting Modular

Utilitzant Procediments i Funcions

Divideix els scripts en procediments més petits i reutilitzables.

$ @MYPROCEDURE
$ EXIT

$! MYPROCEDURE.COM
$ PROCEDURE:
$ WRITE SYS$OUTPUT "Aquest és un procediment reutilitzable"
$ EXIT
  • Explicació: Aquest exemple mostra com cridar un script de procediment separat (MYPROCEDURE.COM) des de l'script principal.

Interfície amb Altres Llenguatges

Cridant Programes en C des de DCL

Integra scripts DCL amb programes en C per a tasques més complexes.

$ RUN MYCPROGRAM.EXE
$ EXIT
  • Explicació: Aquest script executa un programa en C compilat (MYCPROGRAM.EXE) des d'un script DCL.

Conclusió

En aquesta secció, hem cobert tècniques avançades de scripting DCL incloent estructures de control complexes, gestió d'errors, optimització del rendiment, scripting modular i interfície amb altres llenguatges. Dominant aquestes tècniques, podràs crear scripts DCL més eficients, fiables i mantenibles a OpenVMS. Continua practicant aquests conceptes per convertir-te en un expert en scripting DCL avançat.
