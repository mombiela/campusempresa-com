Les funcions lèxiques a OpenVMS són funcions integrades que proporcionen una manera d'obtenir informació del sistema, manipular cadenes i realitzar diverses altres tasques. Són una part essencial de la programació en DCL (Digital Command Language) i poden millorar molt la funcionalitat i l'eficiència dels teus scripts.

Introducció a les Funcions Lèxiques

  • Definició: Les funcions lèxiques són funcions especials a OpenVMS que retornen informació sobre el sistema, l'entorn o realitzen operacions específiques.
  • Sintaxi: Les funcions lèxiques normalment es prefixen amb F$ i s'utilitzen dins dels scripts DCL.

Funcions Lèxiques Bàsiques

Comencem amb algunes funcions lèxiques bàsiques que s'utilitzen comunament:

  1. F$GETSYI: Recupera informació del sistema.
  2. F$GETJPI: Recupera informació de treball/procés.
  3. F$TRNLNM: Traduïx noms lògics.
  4. F$FAO: Formatja la sortida.

Exemple: Ús de F$GETSYI

La funció F$GETSYI s'utilitza per obtenir informació del sistema. Aquí tens un exemple senzill:

$ SYSTEM_NAME = F$GETSYI("NODENAME")
$ WRITE SYS$OUTPUT "System Name: ''SYSTEM_NAME'"
  • Explicació:
    • F$GETSYI("NODENAME"): Recupera el nom del node del sistema.
    • WRITE SYS$OUTPUT "System Name: ''SYSTEM_NAME'": Mostra el nom del sistema a la consola.

Funcions de Manipulació de Cadenes

Les funcions lèxiques també es poden utilitzar per manipular cadenes. Algunes funcions comunes de manipulació de cadenes inclouen:

  1. F$EXTRACT: Extreu una subcadena d'una cadena.
  2. F$LENGTH: Retorna la longitud d'una cadena.
  3. F$LOCATE: Troba la posició d'una subcadena dins d'una cadena.
  4. F$EDIT: Edita una cadena segons les regles especificades.

Exemple: Ús de F$EXTRACT i F$LENGTH

$ FULL_NAME = "OpenVMS User"
$ FIRST_NAME = F$EXTRACT(0, 7, FULL_NAME)
$ NAME_LENGTH = F$LENGTH(FULL_NAME)
$ WRITE SYS$OUTPUT "First Name: ''FIRST_NAME'"
$ WRITE SYS$OUTPUT "Name Length: ''NAME_LENGTH'"
  • Explicació:
    • F$EXTRACT(0, 7, FULL_NAME): Extreu els primers 7 caràcters de FULL_NAME.
    • F$LENGTH(FULL_NAME): Retorna la longitud de la cadena FULL_NAME.
    • WRITE SYS$OUTPUT "First Name: ''FIRST_NAME'": Mostra el primer nom.
    • WRITE SYS$OUTPUT "Name Length: ''NAME_LENGTH'": Mostra la longitud del nom complet.

Funcions Lèxiques Avançades

Per a un ús més avançat, les funcions lèxiques poden interactuar amb processos del sistema i noms lògics.

Exemple: Ús de F$GETJPI i F$TRNLNM

$ PROCESS_ID = F$GETJPI("","PID")
$ LOGICAL_NAME = F$TRNLNM("SYS$LOGIN")
$ WRITE SYS$OUTPUT "Process ID: ''PROCESS_ID'"
$ WRITE SYS$OUTPUT "Logical Name SYS$LOGIN: ''LOGICAL_NAME'"
  • Explicació:
    • F$GETJPI("","PID"): Recupera l'identificador del procés del procés actual.
    • F$TRNLNM("SYS$LOGIN"): Traduïx el nom lògic SYS$LOGIN a la seva equivalència.
    • WRITE SYS$OUTPUT "Process ID: ''PROCESS_ID'": Mostra l'identificador del procés.
    • WRITE SYS$OUTPUT "Logical Name SYS$LOGIN: ''LOGICAL_NAME'": Mostra el nom lògic traduït.

Exercicis Pràctics

  1. Recuperar i Mostrar el Temps d'Uptime del Sistema:

    • Utilitza F$GETSYI per recuperar el temps d'uptime del sistema i mostrar-lo.
    $ UPTIME = F$GETSYI("BOOTTIME")
$ WRITE SYS$OUTPUT "System Uptime: ''UPTIME'"

  2. Extreure i Mostrar Informació de l'Usuari:

    • Utilitza F$GETJPI per recuperar el nom d'usuari actual i mostrar-lo.
    $ USERNAME = F$GETJPI("","USERNAME")
$ WRITE SYS$OUTPUT "Current User: ''USERNAME'"

  3. Traduir i Mostrar Noms Lògics:

    • Utilitza F$TRNLNM per traduir un nom lògic de la teva elecció i mostrar-lo.
    $ LOGICAL = F$TRNLNM("SYS$SCRATCH")
$ WRITE SYS$OUTPUT "SYS$SCRATCH: ''LOGICAL'"

Conclusió

Les funcions lèxiques a OpenVMS són eines poderoses que poden millorar molt els teus scripts DCL. Et permeten recuperar informació del sistema i dels processos, manipular cadenes i treballar amb noms lògics. En dominar aquestes funcions, podràs escriure scripts més eficients i efectius per automatitzar tasques i gestionar el teu entorn OpenVMS. Practica l'ús d'aquestes funcions amb els exercicis proporcionats per consolidar la teva comprensió i millorar les teves habilitats de programació.
