Introducció

OpenVMS és un sistema operatiu versàtil i potent utilitzat en diversos entorns empresarials. Personalitzar i estendre OpenVMS permet als usuaris adaptar el sistema a les seves necessitats específiques, millorant la funcionalitat i l'eficiència. Aquesta secció us guiarà a través dels conceptes bàsics de la personalització i l'extensió, des de nivells principiants fins a avançats.

Personalització Bàsica

Paràmetres del Sistema

  • Utilitat SYSGEN: Utilitzada per modificar els paràmetres del sistema.
  • MODPARAMS.DAT: Un fitxer on es registren els canvis de paràmetres.

Exemple:

$ RUN SYS$SYSTEM:SYSGEN
SYSGEN> USE CURRENT
SYSGEN> SET MAXPROCESSCNT 256
SYSGEN> WRITE CURRENT
SYSGEN> EXIT

Explicació: Aquest exemple mostra com utilitzar la utilitat SYSGEN per establir el paràmetre MAXPROCESSCNT a 256.

Procediments de Comandes

  • DCL (Digital Command Language): Utilitzat per escriure procediments de comandes.
  • LOGIN.COM: Un script que s'executa a l'inici de sessió de l'usuari.

Exemple:

$! LOGIN.COM
$ SET PROMPT="User>"
$ SHOW TIME

Explicació: Aquest script estableix un prompt personalitzat i mostra l'hora actual quan un usuari inicia sessió.

Personalització Intermèdia

Noms Lògics

  • Definició de Noms Lògics: Utilitzats per crear àlies per a fitxers, dispositius o directoris.
  • Noms Lògics de Sistema: Definits a SYLOGICALS.COM.

Exemple:

$ DEFINE MYDISK DKA0:[USERFILES]
$ SHOW LOGICAL MYDISK

Explicació: Aquest exemple defineix un nom lògic MYDISK per al directori DKA0:[USERFILES] i després el mostra.

Personalització de l'Entorn d'Usuari

  • SYLOGIN.COM: Un script d'inici de sessió de sistema.
  • Personalització de Perfils d'Usuari: Modificar configuracions específiques de l'usuari.

Exemple:

$! SYLOGIN.COM
$ SET TERMINAL /INQUIRE
$ DEFINE SYS$SCRATCH DKA0:[SCRATCH]

Explicació: Aquest script estableix les característiques del terminal i defineix un directori de treball per a fitxers temporals.

Personalització Avançada

Escriptura de Serveis del Sistema

  • Crides a Serveis del Sistema: Estendre la funcionalitat d'OpenVMS escrivint serveis personalitzats.
  • Ús de les Biblioteques LIB$ i SYS$: Biblioteques que proporcionen rutines de serveis del sistema.

Exemple:

#include <lib$routines.h>
#include <ssdef.h>

int main() {
    int status;
    status = lib$signal(SS$_NORMAL);
    return status;
}

Explicació: Aquest programa en C utilitza la rutina lib$signal per enviar un estat de finalització normal.

Desenvolupament d'Utilitats Personalitzades

  • Escriptura d'Utilitats Personalitzades: Crear utilitats per automatitzar tasques.
  • Ús de Rutines del Sistema VMS: Aprofitar les rutines integrades per al desenvolupament d'utilitats.

Exemple:

$! CUSTOM_UTILITY.COM
$! Aquest script automatitza tasques de còpia de seguretat
$ BACKUP DKA0:[USERFILES]*.*;* MKA500:[BACKUP] /LOG
$ EXIT

Explicació: Aquest procediment de comandes automatitza la còpia de seguretat dels fitxers d'usuari a una unitat de cinta.

Extensió d'OpenVMS

Instal·lació de Programari de Tercers

  • Utilitat VMSINSTAL: Utilitzada per instal·lar programari de tercers.
  • Gestió de Parches de Programari: Aplicar parches per estendre la funcionalitat.

Exemple:

$ @SYS$UPDATE:VMSINSTAL MYAPP032 DKA0:[KITS]

Explicació: Aquesta comanda instal·la el programari MYAPP des del directori DKA0:[KITS].

Desenvolupament d'Aplicacions Personalitzades

  • Ús de DECwindows: Desenvolupar aplicacions gràfiques.
  • Integració amb Altres Sistemes: Utilitzar protocols de xarxa i APIs.

Exemple:

#include <decw$include:Xm.h>

int main(int argc, char **argv) {
    XtAppContext app;
    Widget top_wid;

    top_wid = XtVaAppInitialize(&app, "HelloWorld", NULL, 0, &argc, argv, NULL, NULL);
    XtRealizeWidget(top_wid);
    XtAppMainLoop(app);
    return 0;
}

Explicació: Aquest programa en C inicialitza una aplicació bàsica de DECwindows.

Conclusió

Personalitzar i estendre OpenVMS pot millorar significativament les seves capacitats i adaptar-lo a necessitats específiques. Comprenent els paràmetres del sistema, els procediments de comandes, els noms lògics i desenvolupant utilitats i aplicacions personalitzades, els usuaris poden aprofitar al màxim el poder d'OpenVMS. Tant si sou un usuari principiant com avançat, aquestes habilitats són essencials per optimitzar el vostre entorn OpenVMS.
