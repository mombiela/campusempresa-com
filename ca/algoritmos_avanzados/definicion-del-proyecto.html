El Projecte Sobre nosaltres Contribuir Donacions Llicència

Introducció

En aquest curs sobre Algoritmes Avançats, ens enfocarem en la comprensió i aplicació de tècniques avançades en el disseny i anàlisi d'algoritmes. L'objectiu principal és proporcionar als professionals les eines necessàries per abordar problemes complexos de manera eficient i efectiva. En aquesta primera secció, definirem el projecte que guiarà el nostre aprenentatge al llarg del curs.

Objectius del Projecte

El projecte té com a objectiu principal aplicar els conceptes avançats d'algoritmes en un problema real. Al llarg del curs, desenvoluparem un algoritme que resolgui un problema específic, optimitzant tant el temps com l'espai d'execució.

Objectius Específics

  • Comprendre els fonaments dels algoritmes avançats: Revisar conceptes clau com complexitat temporal i espacial, estructures de dades avançades, i tècniques d'optimització.
  • Aplicar tècniques avançades en problemes reals: Implementar algoritmes que resolguin problemes complexos utilitzant tècniques com programació dinàmica, algoritmes de grafs, i algoritmes de cerca i classificació avançats.
  • Avaluar i optimitzar algoritmes: Analitzar l'eficiència dels algoritmes implementats i realitzar optimitzacions per millorar el seu rendiment.

Descripció del Problema

El problema que abordarem en aquest projecte és el "Problema del Viatjant" (Traveling Salesman Problem, TSP). Aquest problema consisteix a trobar la ruta més curta que permeti a un viatjant visitar un conjunt de ciutats, passant per cadascuna d'elles una sola vegada i tornant a la ciutat d'origen.

Exemple del Problema

Suposem que tenim les següents ciutats i distàncies entre elles:

| Ciutat A | Ciutat B | Distància | |----------|----------|-----------| | A | B | 10 | | A | C | 15 | | A | D | 20 | | B | C | 35 | | B | D | 25 | | C | D | 30 |

L'objectiu és trobar la ruta més curta que passi per totes les ciutats i torni al punt de partida.

Metodologia

Per resoldre el TSP, utilitzarem diferents tècniques avançades d'algoritmes. A continuació, es presenten algunes de les tècniques que explorarem:

  • Algoritmes de Força Bruta: Avaluar totes les possibles rutes i seleccionar la més curta.
  • Programació Dinàmica: Utilitzar la tècnica de memoització per reduir la complexitat temporal.
  • Algoritmes Genètics: Aplicar tècniques d'intel·ligència artificial per trobar solucions aproximades.
  • Algoritmes de Grafs: Utilitzar algoritmes com el de Dijkstra o el de Floyd-Warshall per trobar camins mínims.

Exemple de Codi: Força Bruta

A continuació, es presenta un exemple d'implementació en Python per resoldre el TSP utilitzant un enfocament de força bruta:

from itertools import permutations

def calculate_distance(route, distance_matrix):
    total_distance = 0
    for i in range(len(route) - 1):
        total_distance += distance_matrix[route[i]][route[i+1]]
    total_distance += distance_matrix[route[-1]][route[0]]  # Return to start
    return total_distance

def tsp_brute_force(distance_matrix):
    cities = list(range(len(distance_matrix)))
    min_distance = float('inf')
    best_route = None
    for route in permutations(cities):
        current_distance = calculate_distance(route, distance_matrix)
        if current_distance < min_distance:
            min_distance = current_distance
            best_route = route
    return best_route, min_distance

# Exemple de matriu de distàncies
distance_matrix = [
    [0, 10, 15, 20],
    [10, 0, 35, 25],
    [15, 35, 0, 30],
    [20, 25, 30, 0]
]

best_route, min_distance = tsp_brute_force(distance_matrix)
print(f"La millor ruta és: {best_route} amb una distància de: {min_distance}")
