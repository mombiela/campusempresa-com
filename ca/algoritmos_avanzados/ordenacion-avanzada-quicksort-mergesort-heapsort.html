Ordenació Avançada: QuickSort, MergeSort, HeapSort

Introducció

En aquesta secció del curs sobre Algorismes Avançats, explorarem tres algorismes d'ordenació fonamentals: QuickSort, MergeSort i HeapSort. Aquests algorismes són essencials per a qualsevol professional en el camp de la informàtica i la programació a causa de la seva eficiència i aplicabilitat en una àmplia varietat de contextos.

QuickSort

QuickSort és un algorisme d'ordenació basat en el paradigma de "divideix i venceràs". És conegut per la seva eficiència i simplicitat en la implementació.

Conceptes Clau

  • Pivote: Un element seleccionat de la llista que s'utilitza per dividir la llista en subllistes.
  • Partició: El procés de reorganitzar els elements de la llista de manera que tots els elements menors que el pivote quedin a la seva esquerra i tots els elements majors quedin a la seva dreta.
  • Recursió: QuickSort s'aplica recursivament a les subllistes generades per la partició.

Exemple de Codi

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
print(quicksort(arr))

Explicació del Codi

  1. Condició Base: Si la llista té un o zero elements, ja està ordenada.
  2. Selecció del Pivote: Es tria l'element central de la llista.
  3. Partició: Es creen tres subllistes: left per als elements menors que el pivote, middle per als elements iguals al pivote, i right per als elements majors que el pivote.
  4. Recursió: S'aplica QuickSort recursivament a les subllistes left i right, i es combinen els resultats.

MergeSort

MergeSort és un altre algorisme d'ordenació basat en "divideix i venceràs". És conegut per la seva estabilitat i eficiència en llistes grans.

Conceptes Clau

  • Divisió: La llista es divideix en dues subllistes d'aproximadament la mateixa longitud.
  • Fusió: Les subllistes ordenades es combinen en una sola llista ordenada.
  • Recursió: MergeSort s'aplica recursivament a les subllistes.

Exemple de Codi

def mergesort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = mergesort(arr[:mid])
    right = mergesort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
print(mergesort(arr))

Explicació del Codi

  1. Condició Base: Si la llista té un o zero elements, ja està ordenada.
  2. Divisió: La llista es divideix en dues subllistes left i right.
  3. Recursió: S'aplica MergeSort recursivament a les subllistes.
  4. Fusió: Es combinen les subllistes ordenades utilitzant la funció merge.

HeapSort

HeapSort és un algorisme d'ordenació basat en l'estructura de dades de munt (heap). És eficient i té un rendiment garantit en el pitjor dels casos.

Conceptes Clau

  • Munt (Heap): Una estructura de dades que satisfà la propietat del munt, on cada node és major o igual (o menor o igual) que els seus fills.
  • Heapify: El procés d'ajustar un arbre binari perquè satisfaci la propietat del munt.
  • Extracció del Màxim: El procés d'eliminar l'element màxim (o mínim) del munt i ajustar el munt resultant.

Exemple de Codi

def heapsort(arr):
    def heapify(arr, n, i):
        largest = i
        l = 2 * i + 1
        r = 2 * i + 2
        if l < n and arr[i] < arr[l]:
            largest = l
        if r < n and arr[largest] < arr[r]:
            largest = r
        if largest != i:
            arr[i], arr[largest] = arr[largest], arr[i]
            heapify(arr, n, largest)

    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n-1, 0, -1):
        arr[i], arr[0] = arr[0], arr[i]
        heapify(arr, i, 0)

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
heapsort(arr)
print(arr)

Explicació del Codi

  1. Heapify: Ajusta el subarbre amb arrel en l'índex i perquè satisfaci la propietat del munt.
  2. Construcció del Munt: Es construeix un munt a partir de la llista.
  3. Extracció del Màxim: S'intercanvia el primer element (màxim) amb l'últim i s'ajusta el munt resultant.

Comparació d'Algorismes

| Algorisme | Complexitat Promig | Complexitat Pitjor Cas | Estabilitat | Ús de Memòria | |------------|--------------------|------------------------|-------------|---------------| | QuickSort | O(n log n) | O(n^2) | No | O(log n) | | MergeSort | O(n log n) | O(n log n) | Sí | O(n) | | HeapSort | O(n log n) | O(n log n) | No | O(1) |

Conclusió

En aquesta secció, hem explorat tres algorismes d'ordenació avançats: QuickSort, MergeSort i HeapSort. Cadascun té els seus propis avantatges i desavantatges en termes de complexitat, estabilitat i ús de memòria. Comprendre aquests algorismes i les seves implementacions és crucial per a qualsevol professional en el camp de la programació i la informàtica.
