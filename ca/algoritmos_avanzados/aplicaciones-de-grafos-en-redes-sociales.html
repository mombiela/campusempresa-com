Introducció

Les xarxes socials són un exemple clàssic de grafs en la vida real. En aquest tema, explorarem com els grafs s'apliquen en l'anàlisi i l'optimització de xarxes socials. Veurem conceptes clau, algorismes utilitzats i exemples pràctics que il·lustren el seu ús.

Conceptes Clau

  • Node (Vèrtex): Representa un usuari en la xarxa social.
  • Aresta (Enllaç): Representa una relació o connexió entre dos usuaris.
  • Graf Dirigit: Un graf on les connexions tenen una direcció, per exemple, seguidors a Twitter.
  • Graf No Dirigit: Un graf on les connexions no tenen direcció, per exemple, amistats a Facebook.
  • Pes de l'Aresta: Pot representar la força de la connexió, com la freqüència d'interacció entre dos usuaris.

Algorismes Comuns en Xarxes Socials

Algorisme de Dijkstra

Utilitzat per trobar la ruta més curta entre dos nodes en un graf ponderat.

import heapq

def dijkstra(graph, start):
    queue = [(0, start)]
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0

    while queue:
        current_distance, current_node = heapq.heappop(queue)

        if current_distance > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))

    return distances

Algorisme de PageRank

Utilitzat per Google per classificar pàgines web, també es pot aplicar per determinar la influència d'usuaris en una xarxa social.

import numpy as np

def pagerank(graph, num_iterations: int = 100, d: float = 0.85):
    num_nodes = len(graph)
    M = np.zeros((num_nodes, num_nodes))

    for i in range(num_nodes):
        for j in graph[i]:
            M[j, i] = 1 / len(graph[i])

    v = np.ones(num_nodes) / num_nodes

    for _ in range(num_iterations):
        v = (1 - d) / num_nodes + d * M @ v

    return v

Exemples Pràctics

Exemple 1: Trobar Amics en Comú

Donat un graf no dirigit, trobar els amics en comú entre dos usuaris.

def common_friends(graph, user1, user2):
    return set(graph[user1]) & set(graph[user2])

# Exemple d'ús
graph = {
    'Alice': ['Bob', 'Charlie', 'David'],
    'Bob': ['Alice', 'Charlie'],
    'Charlie': ['Alice', 'Bob', 'David'],
    'David': ['Alice', 'Charlie']
}

print(common_friends(graph, 'Alice', 'Charlie'))  # Output: {'Bob', 'David'}

Exemple 2: Suggeriment d'Amics

Utilitzant l'algorisme de Dijkstra per suggerir amics basats en la distància més curta en un graf ponderat.

def suggest_friends(graph, user, max_distance):
    distances = dijkstra(graph, user)
    return [node for node, distance in distances.items() if distance <= max_distance and node != user]

# Exemple d'ús
graph = {
    'Alice': {'Bob': 1, 'Charlie': 2},
    'Bob': {'Alice': 1, 'Charlie': 1},
    'Charlie': {'Alice': 2, 'Bob': 1}
}

print(suggest_friends(graph, 'Alice', 1))  # Output: ['Bob']

Conclusió

Els grafs són una eina poderosa per modelar i analitzar xarxes socials. Des de trobar rutes més curtes fins a determinar la influència d'usuaris, els algorismes de grafs permeten optimitzar i entendre millor les dinàmiques de les xarxes socials. Amb els exemples i algorismes presentats, els professionals poden aplicar aquests conceptes per millorar la funcionalitat i l'experiència de l'usuari en plataformes socials.
