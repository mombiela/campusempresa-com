Algoritmes de Camins Mínims

Introducció

Els algorismes de camins mínims són fonamentals en la teoria de grafs i tenen aplicacions en diverses àrees com l'optimització de xarxes, la planificació de rutes i la intel·ligència artificial. Aquest tema cobreix els algorismes més importants per trobar camins mínims en grafs ponderats i no ponderats.

Conceptes Clau

  • Graf: Una estructura composta per nodes (o vèrtexs) i arestes (o arcs) que connecten parells de nodes.
  • Pes: Un valor associat a una aresta que pot representar distància, cost, temps, etc.
  • Camí Mínim: La ruta entre dos nodes en un graf que té el menor pes total.

Algorismes Principals

Algorisme de Dijkstra

L'algorisme de Dijkstra és un dels més coneguts per trobar el camí mínim des d'un node origen a tots els altres nodes en un graf amb pesos no negatius.

Passos de l'Algorisme

  1. Inicialitzar la distància al node origen com 0 i a tots els altres nodes com infinit.
  2. Marcar tots els nodes com no visitats.
  3. Seleccionar el node no visitat amb la menor distància.
  4. Actualitzar la distància dels nodes adjacents.
  5. Marcar el node actual com visitat.
  6. Repetir els passos 3-5 fins que tots els nodes hagin estat visitats.

Exemple de Codi

import heapq

def dijkstra(graph, start):
    # Inicialitzar distàncies i cua de prioritat
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    priority_queue = [(0, start)]
    
    while priority_queue:
        current_distance, current_vertex = heapq.heappop(priority_queue)
        
        if current_distance > distances[current_vertex]:
            continue
        
        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight
            
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))
    
    return distances

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

print(dijkstra(graph, 'A'))

Algorisme de Bellman-Ford

L'algorisme de Bellman-Ford és capaç de manejar grafs amb pesos negatius i també pot detectar cicles negatius.

Passos de l'Algorisme

  1. Inicialitzar la distància al node origen com 0 i a tots els altres nodes com infinit.
  2. Relaxar totes les arestes del graf repetidament.
  3. Verificar si hi ha cicles negatius.

Exemple de Codi

def bellman_ford(graph, start):
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    
    for _ in range(len(graph) - 1):
        for vertex in graph:
            for neighbor, weight in graph[vertex].items():
                if distances[vertex] + weight < distances[neighbor]:
                    distances[neighbor] = distances[vertex] + weight
    
    # Verificació de cicles negatius
    for vertex in graph:
        for neighbor, weight in graph[vertex].items():
            if distances[vertex] + weight < distances[neighbor]:
                raise ValueError("Graph contains a negative-weight cycle")
    
    return distances

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'C': -3, 'D': 2},
    'C': {'D': 3},
    'D': {}
}

print(bellman_ford(graph, 'A'))

Algorisme de Floyd-Warshall

L'algorisme de Floyd-Warshall troba camins mínims entre tots els parells de nodes en un graf.

Passos de l'Algorisme

  1. Crear una matriu de distàncies inicialitzada amb els pesos de les arestes.
  2. Iterar sobre tots els parells de nodes i actualitzar les distàncies mínimes.

Exemple de Codi

def floyd_warshall(graph):
    nodes = list(graph.keys())
    dist = {node: {node: float('infinity') for node in nodes} for node in nodes}
    
    for node in nodes:
        dist[node][node] = 0
        for neighbor, weight in graph[node].items():
            dist[node][neighbor] = weight
    
    for k in nodes:
        for i in nodes:
            for j in nodes:
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    
    return dist

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'C': 2, 'D': 5},
    'C': {'D': 1},
    'D': {}
}

print(floyd_warshall(graph))

Comparació d'Algorismes

| Algorisme | Complexitat Temporal | Maneja Pesos Negatius | Detecta Cicles Negatius | |-----------------|-----------------------|------------------------|--------------------------| | Dijkstra | O(V^2) o O(E + V log V) amb heap | No | No | | Bellman-Ford | O(VE) | Sí | Sí | | Floyd-Warshall | O(V^3) | Sí | No |

Conclusió

Els algorismes de camins mínims són eines essencials en la teoria de grafs i tenen aplicacions pràctiques en diverses àrees. Dijkstra és eficient per a grafs amb pesos no negatius, Bellman-Ford maneja pesos negatius i detecta cicles negatius, i Floyd-Warshall és útil per trobar camins mínims entre tots els parells de nodes. L'elecció de l'algorisme adequat depèn de les característiques del graf i els requisits específics del problema a resoldre.
