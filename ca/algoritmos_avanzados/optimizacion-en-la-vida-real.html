Introducció

L'optimització és una branca fonamental de la matemàtica aplicada i la informàtica que se centra a trobar la millor solució possible per a un problema donat, sota certes restriccions. En la vida real, l'optimització s'aplica en una varietat de camps com la logística, l'enginyeria, l'economia, i la intel·ligència artificial. Aquest tema explorarà com els algorismes avançats d'optimització poden ser utilitzats per resoldre problemes del món real.

Conceptes Clau

  • Funció Objectiu: La funció que es desitja maximitzar o minimitzar.
  • Variables de Decisió: Les variables que poden ser ajustades per optimitzar la funció objectiu.
  • Restriccions: Condicions que han de ser satisfetes per les variables de decisió.
  • Espai de Solucions: El conjunt de totes les possibles solucions que satisfan les restriccions.
  • Algorismes d'Optimització: Mètodes per trobar la millor solució en l'espai de solucions.

Tipus de Problemes d'Optimització

Optimització Lineal

L'optimització lineal s'utilitza quan la funció objectiu i les restriccions són lineals. Un exemple clàssic és el problema de la dieta, on es busca minimitzar el cost d'una dieta que compleixi amb certs requisits nutricionals.

Exemple de Codi: Problema de la Dieta

from scipy.optimize import linprog

# Coeficients de la funció objectiu (minimitzar cost)
c = [2, 3, 1.5]

# Coeficients de les restriccions (nutrients)
A = [
    [1, 1, 1],  # Calories
    [0.5, 0.2, 0.3],  # Proteïnes
    [0.2, 0.3, 0.4]  # Greixos
]

# Límit inferior de les restriccions
b = [2000, 50, 30]

# Resoldre el problema d'optimització
result = linprog(c, A_ub=A, b_ub=b, method='simplex')

print('Quantitat de cada aliment:', result.x)
print('Cost mínim:', result.fun)

Optimització No Lineal

En l'optimització no lineal, la funció objectiu o les restriccions són no lineals. Un exemple és l'optimització de portafolis en finances, on es busca maximitzar el retorn esperat i minimitzar el risc.

Exemple de Codi: Optimització de Portafolis

import numpy as np
from scipy.optimize import minimize

# Dades fictícies de retorns esperats i matriu de covariància
returns = np.array([0.1, 0.2, 0.15])
cov_matrix = np.array([
    [0.005, -0.010, 0.004],
    [-0.010, 0.040, -0.002],
    [0.004, -0.002, 0.023]
])

# Funció objectiu (minimitzar risc)
def portfolio_variance(weights):
    return np.dot(weights.T, np.dot(cov_matrix, weights))

# Restriccions
constraints = ({'type': 'eq', 'fun': lambda x: np.sum(x) - 1})
bounds = tuple((0, 1) for _ in range(len(returns)))

# Resoldre el problema d'optimització
result = minimize(portfolio_variance, len(returns) * [1. / len(returns)], bounds=bounds, constraints=constraints)

print('Pesos òptims:', result.x)
print('Variància mínima:', result.fun)

Algorismes d'Optimització

Algorismes Exactes

  • Mètode Simplex: Utilitzat principalment en optimització lineal.
  • Programació Dinàmica: Utilitzada per a problemes que poden ser descompostos en subproblemes més petits.

Algorismes Heurístics

  • Algorismes Genètics: Basats en l'evolució natural, útils per a problemes complexos i no lineals.
  • Recuit Simulat: Inspirat en el procés de refredament de metalls, útil per evitar òptims locals.

Algorismes Metaheurístics

  • Optimització per Eixam de Partícules (PSO): Basat en el comportament social dels animals.
  • Algorismes de Colònia de Formigues (ACO): Inspirat en el comportament de les formigues en la cerca d'aliments.

Comparació d'Algorismes

| Algorisme | Tipus | Avantatges | Desavantatges | |--------------------------|---------------|-------------------------------------|------------------------------------| | Mètode Simplex | Exacte | Eficient per a problemes lineals | No apte per a problemes no lineals | | Algorismes Genètics | Heurístic | Flexible i robust | Pot ser computacionalment costós | | Recuit Simulat | Heurístic | Evita òptims locals | Paràmetres difícils d'ajustar | | PSO | Metaheurístic | Fàcil d'implementar | Pot convergir prematurament | | ACO | Metaheurístic | Eficient per a problemes de ruta | Requereix ajust de paràmetres |

Conclusió

L'optimització és una eina poderosa que pot ser aplicada en una varietat de contextos del món real. Des de la logística fins a l'enginyeria i les finances, els algorismes avançats d'optimització permeten trobar solucions eficients i efectives a problemes complexos. Comprendre els diferents tipus de problemes d'optimització i els algorismes disponibles és crucial per seleccionar la millor estratègia per a cada situació específica.
