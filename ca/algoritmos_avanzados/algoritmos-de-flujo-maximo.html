Algoritmes de Flux Màxim

Introducció

Els algoritmes de flux màxim són fonamentals en la teoria de grafs i tenen aplicacions en diverses àrees com l'optimització de xarxes, la planificació de projectes i la logística. Aquest tema cobreix els conceptes bàsics, els algoritmes més importants i exemples pràctics per entendre el seu funcionament.

Conceptes Bàsics

  • Graf de Flux: Un graf dirigit on cada aresta té una capacitat que representa el màxim flux que pot passar a través d'ella.
  • Flux: La quantitat de "material" que passa a través d'una aresta.
  • Font (s): El node on s'origina el flux.
  • Embornal (t): El node on el flux es consumeix.
  • Tall: Una partició del graf en dos subconjunts que separa la font de l'embornal.
  • Capacitat de Tall: La suma de les capacitats de les arestes que creuen el tall.

Algoritmes Principals

Algoritme de Ford-Fulkerson

L'algoritme de Ford-Fulkerson és un dels mètodes més coneguts per trobar el flux màxim en un graf de flux. Utilitza el concepte de camins augmentants.

Passos de l'Algoritme

  1. Inicialització: Començar amb un flux de 0.
  2. Cerca de Camins Augmentants: Trobar un camí des de la font a l'embornal on el flux pot ser incrementat.
  3. Augmentar el Flux: Incrementar el flux al llarg del camí trobat.
  4. Repetir: Repetir els passos 2 i 3 fins que no hi hagi més camins augmentants.

Exemple de Codi

def bfs(C, F, source, sink):
    queue = [source]
    paths = {source: []}
    if source == sink:
        return paths[source]
    while queue:
        u = queue.pop(0)
        for v in range(len(C)):
            if C[u][v] - F[u][v] > 0 and v not in paths:
                paths[v] = paths[u] + [(u, v)]
                if v == sink:
                    return paths[v]
                queue.append(v)
    return None

def ford_fulkerson(C, source, sink):
    n = len(C)
    F = [[0] * n for _ in range(n)]
    path = bfs(C, F, source, sink)
    while path:
        flow = min(C[u][v] - F[u][v] for u, v in path)
        for u, v in path:
            F[u][v] += flow
            F[v][u] -= flow
        path = bfs(C, F, source, sink)
    return sum(F[source][i] for i in range(n))

# Exemple d'ús
C = [[0, 16, 13, 0, 0, 0],
     [0, 0, 10, 12, 0, 0],
     [0, 4, 0, 0, 14, 0],
     [0, 0, 9, 0, 0, 20],
     [0, 0, 0, 7, 0, 4],
     [0, 0, 0, 0, 0, 0]]
source = 0
sink = 5
print(f"El flux màxim és: {ford_fulkerson(C, source, sink)}")

Algoritme d'Edmonds-Karp

L'algoritme d'Edmonds-Karp és una implementació específica de l'algoritme de Ford-Fulkerson que utilitza la cerca en amplada (BFS) per trobar camins augmentants.

Diferències Clau

  • Utilitza BFS en lloc de DFS per trobar camins augmentants.
  • Garanteix un temps d'execució polinomial de O(VE^2), on V és el nombre de vèrtexs i E és el nombre d'arestes.

Exemple de Codi

El codi de l'algoritme d'Edmonds-Karp és similar al de Ford-Fulkerson, però amb BFS garantit per a la cerca de camins augmentants.

Comparació d'Algoritmes

| Algoritme | Mètode de Cerca | Complexitat Temporal | |-------------------|--------------------|----------------------| | Ford-Fulkerson | DFS/BFS | O(max_flow * E) | | Edmonds-Karp | BFS | O(VE^2) |

Aplicacions Pràctiques

  • Xarxes de Transport: Optimització de rutes i fluxos de trànsit.
  • Planificació de Projectes: Assignació de recursos i tasques.
  • Logística: Optimització de fluxos en cadenes de subministrament.

Conclusió

Els algoritmes de flux màxim són eines poderoses per resoldre problemes d'optimització en grafs. Comprendre els conceptes bàsics i els algoritmes principals com Ford-Fulkerson i Edmonds-Karp és essencial per aplicar aquestes tècniques en problemes del món real. Practicar amb exemples i exercicis ajudarà a solidificar aquest coneixement i a desenvolupar habilitats per abordar problemes complexos de flux en xarxes.
