Cerca Binària i Variants

Introducció a la Cerca Binària

La cerca binària és un algorisme eficient per trobar un element en una llista ordenada. La seva eficiència rau en dividir repetidament el rang de cerca a la meitat, cosa que redueix significativament el nombre de comparacions necessàries.

  • Propòsit: Trobar la posició d'un element en una llista ordenada.
  • Complexitat Temporal: O(log n), on n és el nombre d'elements a la llista.
  • Requisits: La llista ha d'estar ordenada prèviament.

Algorisme Bàsic de Cerca Binària

L'algorisme de cerca binària segueix aquests passos:

  1. Inicialitzar dos índexs, low i high, que representen els límits del rang de cerca.
  2. Calcular l'índex mitjà (mid) del rang actual.
  3. Comparar l'element en mid amb el valor buscat (target).
  4. Si el target és igual a l'element en mid, retornar mid.
  5. Si el target és menor que l'element en mid, ajustar high a mid - 1.
  6. Si el target és més gran que l'element en mid, ajustar low a mid + 1.
  7. Repetir els passos 2-6 fins que low sigui més gran que high.

Exemple de Codi en Python

def cerca_binaria(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1  # Element no trobat

# Exemple d'ús
arr = [1, 3, 5, 7, 9, 11]
target = 7
resultat = cerca_binaria(arr, target)
print(f'Element trobat a l\'índex: {resultat}')

Variants de la Cerca Binària

Cerca Binària Recursiva

La cerca binària també es pot implementar de manera recursiva. La lògica és similar a la iterativa, però s'utilitza una funció que es crida a si mateixa amb paràmetres actualitzats.

Exemple de Codi en Python

def cerca_binaria_recursiva(arr, target, low, high):
    if low > high:
        return -1  # Element no trobat

    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return cerca_binaria_recursiva(arr, target, mid + 1, high)
    else:
        return cerca_binaria_recursiva(arr, target, low, mid - 1)

# Exemple d'ús
arr = [1, 3, 5, 7, 9, 11]
target = 7
resultat = cerca_binaria_recursiva(arr, target, 0, len(arr) - 1)
print(f'Element trobat a l\'índex: {resultat}')

Cerca Binària per al Primer i Últim Element

En alguns casos, és necessari trobar la primera o l'última aparició d'un element en una llista ordenada que pot contenir elements duplicats.

Cerca del Primer Element

def cerca_binaria_primer(arr, target):
    low = 0
    high = len(arr) - 1
    resultat = -1

    mentre low <= high:
        mid = (low + high) // 2
        si arr[mid] == target:
            resultat = mid
            high = mid - 1  # Seguir buscant a la meitat esquerra
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return resultat

# Exemple d'ús
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
resultat = cerca_binaria_primer(arr, target)
print(f'Primera aparició de l\'element a l\'índex: {resultat}')

Cerca de l'Últim Element

def cerca_binaria_ultim(arr, target):
    low = 0
    high = len(arr) - 1
    resultat = -1

    mentre low <= high:
        mid = (low + high) // 2
        si arr[mid] == target:
            resultat = mid
            low = mid + 1  # Seguir buscant a la meitat dreta
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return resultat

# Exemple d'ús
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
resultat = cerca_binaria_ultim(arr, target)
print(f'Última aparició de l\'element a l\'índex: {resultat}')

Conclusió

La cerca binària és un algorisme fonamental en la informàtica a causa de la seva eficiència i simplicitat. Comprendre tant la implementació iterativa com la recursiva, així com les seves variants per trobar el primer i últim element, és crucial per resoldre problemes de cerca en llistes ordenades. Practicar aquests algorismes amb diferents tipus de dades i escenaris ajudarà a consolidar la seva comprensió i aplicació en situacions del món real.
