Introducció a la Programació Entera

La Programació Entera (PE) és una branca de la programació matemàtica que se centra en l'optimització de problemes on algunes o totes les variables de decisió han de ser enteres. Aquest tipus de programació és crucial en situacions on les solucions fraccionàries no són viables, com en l'assignació de recursos, planificació d'horaris i disseny de xarxes.

Conceptes Clau

  • Variables Enteres: Variables que només poden prendre valors enters.
  • Funció Objectiu: La funció que es desitja maximitzar o minimitzar.
  • Restriccions: Condicions que han de ser satisfetes per les variables.
  • Problemes de Programació Entera Mixta (MIP): Problemes que inclouen tant variables enteres com contínues.
  • Problemes de Programació Entera Pura: Problemes on totes les variables són enteres.

Formulació de Problemes de Programació Entera

Per formular un problema de PE, s'han de seguir aquests passos:

  1. Definir les Variables de Decisió: Identificar les variables que representen les decisions a prendre.
  2. Establir la Funció Objectiu: Determinar la funció que es desitja optimitzar.
  3. Especificar les Restriccions: Llistar les condicions que han de complir les variables.

Exemple de Formulació

Suposem que volem maximitzar el benefici de la producció de dos productes, A i B, amb les següents característiques:

  • Benefici per unitat d'A: 40 €
  • Benefici per unitat de B: 30 €
  • Màxim de 50 hores de treball disponibles.
  • Producte A requereix 1 hora de treball per unitat.
  • Producte B requereix 2 hores de treball per unitat.

Variables de Decisió:

  • \( x_1 \): Nombre d'unitats d'A a produir.
  • \( x_2 \): Nombre d'unitats de B a produir.

Funció Objectiu: \[ \text{Maximitzar } Z = 40x_1 + 30x_2 \]

Restriccions: \[ x_1 + 2x_2 \leq 50 \] \[ x_1, x_2 \geq 0 \] \[ x_1, x_2 \in \mathbb{Z} \]

Mètodes de Solució

Mètode de Ramificació i Acotament (Branch and Bound)

Aquest mètode divideix el problema en subproblemes més petits i els resol de manera recursiva. S'utilitza una estructura d'arbre per explorar les possibles solucions.

Mètode de Plans de Tall (Cutting Planes)

Aquest mètode afegeix restriccions addicionals (plans de tall) per eliminar parts de la regió factible que no contenen solucions enteres.

Programació Dinàmica

S'utilitza per a problemes que poden ser descompostos en subproblemes més petits i resolts de manera òptima.

Exemple de Codi

A continuació, es presenta un exemple de com resoldre un problema de PE utilitzant Python i la biblioteca PuLP:

import pulp

# Definir el problema
prob = pulp.LpProblem("Maximitzar_Benefici", pulp.LpMaximize)

# Definir les variables de decisió
x1 = pulp.LpVariable('x1', lowBound=0, cat='Integer')
x2 = pulp.LpVariable('x2', lowBound=0, cat='Integer')

# Definir la funció objectiu
prob += 40 * x1 + 30 * x2, "Benefici_Total"

# Definir les restriccions
prob += x1 + 2 * x2 <= 50, "Restriccio_Hores_Treball"

# Resoldre el problema
prob.solve()

# Imprimir els resultats
print(f"Estat: {pulp.LpStatus[prob.status]}")
print(f"Unitats d'A a produir: {x1.varValue}")
print(f"Unitats de B a produir: {x2.varValue}")
print(f"Benefici Total: {pulp.value(prob.objective)}")

Conclusió

La Programació Entera és una eina poderosa per resoldre problemes d'optimització on les solucions han de ser enteres. Comprendre com formular aquests problemes i els mètodes per resoldre'ls és essencial per abordar una àmplia gamma d'aplicacions pràctiques en la indústria i la recerca. Amb la pràctica i l'ús d'eines adequades, els professionals poden desenvolupar solucions eficients i efectives per a problemes complexos.
