Introducció

El clustering és una tècnica d'aprenentatge no supervisat que s'utilitza per agrupar dades en subconjunts o "clusters" de tal manera que les dades en el mateix cluster siguin més similars entre si que amb les dades d'altres clusters. Aquesta tècnica és fonamental en diverses aplicacions com segmentació de mercat, reconeixement de patrons, anàlisi d'imatges i més.

Conceptes Clau

  • Cluster: Un grup de dades que són similars entre si.
  • Centroid: El centre d'un cluster, utilitzat en algorismes com K-means.
  • Distància: Mesura de similitud entre dos punts de dades, comunament utilitzant la distància euclidiana.
  • Algorisme de Clustering: Mètode per agrupar dades en clusters.

Algorismes de Clustering

K-means

K-means és un dels algorismes de clustering més populars i fàcils d'entendre. Funciona de la següent manera:

  1. Selecciona K punts aleatoris com a centroides inicials.
  2. Assigna cada punt de dades al centroide més proper.
  3. Recalcula els centroides com la mitjana de tots els punts assignats a cada centroide.
  4. Repeteix els passos 2 i 3 fins que els centroides no canviïn significativament.

Exemple de Codi

from sklearn.cluster import KMeans
import numpy as np

# Dades d'exemple
X = np.array([[1, 2], [1, 4], [1, 0], 
              [10, 2], [10, 4], [10, 0]])

# Crear el model K-means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Prediccions
labels = kmeans.labels_
centroids = kmeans.cluster_centers_

print("Labels:", labels)
print("Centroids:", centroids)

DBSCAN

DBSCAN (Density-Based Spatial Clustering of Applications with Noise) és un algorisme que agrupa punts que estan densament empaquetats i marca com a soroll els punts que estan en regions de baixa densitat.

  1. Selecciona un punt aleatori.
  2. Troba tots els punts en la seva veïnat (dins d'un radi ε).
  3. Si hi ha prou punts en el veïnat, es forma un cluster.
  4. Repeteix per a tots els punts en el cluster.
  5. Marca els punts que no pertanyen a cap cluster com a soroll.

Exemple de Codi

from sklearn.cluster import DBSCAN
import numpy as np

# Dades d'exemple
X = np.array([[1, 2], [2, 2], [2, 3], 
              [8, 7], [8, 8], [25, 80]])

# Crear el model DBSCAN
dbscan = DBSCAN(eps=3, min_samples=2).fit(X)

# Prediccions
labels = dbscan.labels_

print("Labels:", labels)

Comparació d'Algorismes

| Algorisme | Avantatges | Desavantatges | |-----------|------------|---------------| | K-means | Fàcil d'implementar, eficient per a grans datasets | Sensible a l'elecció de K, no maneja bé clusters de formes irregulars | | DBSCAN | No requereix especificar el nombre de clusters, maneja bé el soroll i clusters de formes irregulars | Sensible a l'elecció de paràmetres (ε i min_samples), no funciona bé amb dades d'alta dimensió |

Conclusió

Els algorismes de clustering són eines poderoses en l'anàlisi de dades. K-means i DBSCAN són dos dels algorismes més utilitzats, cadascun amb els seus propis avantatges i desavantatges. L'elecció de l'algorisme adequat depèn de la naturalesa de les dades i del problema específic que es desitja resoldre. És important comprendre bé els conceptes i les tècniques per aplicar el clustering de manera efectiva en diferents contextos.
