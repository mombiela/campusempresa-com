Introducció

En el món actual, la quantitat de dades generades i emmagatzemades és immensa. La capacitat de cercar i ordenar eficientment grans volums de dades és crucial per a moltes aplicacions, des de motors de cerca fins a anàlisis de dades en temps real. Aquest tema abordarà els algorismes avançats utilitzats per a la cerca i ordenació en grans volums de dades, proporcionant exemples pràctics i exercicis per consolidar l'aprenentatge.

Algorismes de Cerca

Cerca Binària

  • Descripció: La cerca binària és un algorisme eficient per trobar un element en una llista ordenada. Divideix repetidament el rang de cerca a la meitat.
  • Complexitat: O(log n)
  • Exemple de codi: 
    def cerca_binaria(arr, x):
    esquerra, dreta = 0, len(arr) - 1
    mentre esquerra <= dreta:
        mig = (esquerra + dreta) // 2
        si arr[mig] == x:
            retorna mig
        elif arr[mig] < x:
            esquerra = mig + 1
        else:
            dreta = mig - 1
    retorna -1

# Exemple d'ús
arr = [1, 3, 5, 7, 9, 11]
x = 7
resultat = cerca_binaria(arr, x)
print("Element trobat a l'índex:", resultat)

Cerca en Arbres B

  • Descripció: Els arbres B són estructures de dades autoequilibrades que mantenen dades ordenades i permeten cerques, insercions i eliminacions en temps logarítmic.
  • Aplicacions: Bases de dades i sistemes de fitxers.
  • Complexitat: O(log n) per cerca, inserció i eliminació.

Algorismes d'Ordenació

Ordenació Ràpida (QuickSort)

  • Descripció: QuickSort és un algorisme d'ordenació eficient que utilitza la tècnica de divideix i venceràs. Selecciona un 'pivot' i particiona l'array en dos subarrays, un amb elements menors que el pivot i un altre amb elements majors.
  • Complexitat: O(n log n) de mitjana, O(n^2) en el pitjor cas.
  • Exemple de codi: 
    def quicksort(arr):
    si len(arr) <= 1:
        retorna arr
    pivot = arr[len(arr) // 2]
    esquerra = [x per x en arr si x < pivot]
    mig = [x per x en arr si x == pivot]
    dreta = [x per x en arr si x > pivot]
    retorna quicksort(esquerra) + mig + quicksort(dreta)

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
print("Array ordenat:", quicksort(arr))

Ordenació per Mescla (MergeSort)

  • Descripció: MergeSort és un algorisme d'ordenació que també utilitza la tècnica de divideix i venceràs. Divideix l'array en dues meitats, les ordena recursivament i després les combina.
  • Complexitat: O(n log n) en tots els casos.
  • Exemple de codi: 
    def mergesort(arr):
    si len(arr) <= 1:
        retorna arr
    mig = len(arr) // 2
    esquerra = mergesort(arr[:mig])
    dreta = mergesort(arr[mig:])
    retorna merge(esquerra, dreta)

def merge(esquerra, dreta):
    resultat = []
    i = j = 0
    mentre i < len(esquerra) i j < len(dreta):
        si esquerra[i] < dreta[j]:
            resultat.append(esquerra[i])
            i += 1
        else:
            resultat.append(dreta[j])
            j += 1
    resultat.extend(esquerra[i:])
    resultat.extend(dreta[j:])
    retorna resultat

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
print("Array ordenat:", mergesort(arr))

Comparació d'Algorismes d'Ordenació

| Algorisme | Complexitat Mitjana | Complexitat Pitjor Cas | Espai Addicional | |---------------|----------------------|------------------------|------------------| | QuickSort | O(n log n) | O(n^2) | O(log n) | | MergeSort | O(n log n) | O(n log n) | O(n) |

Conclusió

La cerca i ordenació en grans volums de dades són tasques fonamentals en la informàtica moderna. L'elecció de l'algorisme adequat depèn de les característiques específiques del problema i del conjunt de dades. La cerca binària i els arbres B són útils per a cerques ràpides en dades ordenades, mentre que QuickSort i MergeSort són algorismes d'ordenació eficients amb diferents avantatges i desavantatges. Comprendre aquests algorismes i les seves complexitats és essencial per dissenyar sistemes eficients i escalables.
