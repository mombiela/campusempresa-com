Introducció a la Regressió

La regressió és una tècnica d'aprenentatge supervisat utilitzada per predir valors continus. A diferència de la classificació, que prediu categories, la regressió se centra en predir una variable dependent basada en una o més variables independents.

Tipus d'Algoritmes de Regressió

Hi ha diversos tipus d'algoritmes de regressió, cadascun amb les seves pròpies característiques i aplicacions. A continuació, es descriuen alguns dels més comuns:

Regressió Lineal

La regressió lineal és un dels mètodes més simples i àmpliament utilitzats per modelar la relació entre una variable dependent i una o més variables independents.

  • Fórmula: \( y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n + \epsilon \)
  • Objectiu: Minimitzar la suma dels errors quadrats (MSE).
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió lineal
model = LinearRegression()
model.fit(X, y)

# Prediccions
y_pred = model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_pred, color='red')
plt.xlabel('X')
plt.ylabel('y')
plt.title('Regressió Lineal')
plt.show()

Regressió Polinòmica

La regressió polinòmica és una extensió de la regressió lineal que permet modelar relacions no lineals entre les variables.

  • Fórmula: \( y = \beta_0 + \beta_1 x + \beta_2 x^2 + \ldots + \beta_n x^n + \epsilon \)
  • Objectiu: Minimitzar la suma dels errors quadrats (MSE).
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió polinòmica
poly_model = make_pipeline(PolynomialFeatures(degree=2), LinearRegression())
poly_model.fit(X, y)

# Prediccions
y_poly_pred = poly_model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_poly_pred, color='red')
plt.xlabel('X')
plt.ylabel('y')
plt.title('Regressió Polinòmica')
plt.show()

Regressió Ridge

La regressió Ridge és una tècnica de regularització que introdueix una penalització en els coeficients de la regressió lineal per evitar el sobreajustament.

  • Fórmula: \( y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n + \epsilon \)
  • Penalització: \( \lambda \sum_{i=1}^{n} \beta_i^2 \)
  • Objectiu: Minimitzar la suma dels errors quadrats (MSE) més la penalització.
from sklearn.linear_model import Ridge

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió Ridge
ridge_model = Ridge(alpha=1.0)
ridge_model.fit(X, y)

# Prediccions
y_ridge_pred = ridge_model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_ridge_pred, color='red')
plt.xlabel('X')
plt.ylabel('y')
plt.title('Regressió Ridge')
plt.show()

Regressió Lasso

La regressió Lasso és una altra tècnica de regularització que introdueix una penalització en els coeficients de la regressió lineal, però utilitza la norma L1 en lloc de la norma L2.

  • Fórmula: \( y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n + \epsilon \)
  • Penalització: \( \lambda \sum_{i=1}^{n} |\beta_i| \)
  • Objectiu: Minimitzar la suma dels errors quadrats (MSE) més la penalització.
from sklearn.linear_model import Lasso

# Dades d'exemple
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([1, 3, 2, 5, 4])

# Crear el model de regressió Lasso
lasso_model = Lasso(alpha=0.1)
lasso_model.fit(X, y)

# Prediccions
y_lasso_pred = lasso_model.predict(X)

# Visualització
plt.scatter(X, y, color='blue')
plt.plot(X, y_lasso_pred, color='red')
plt.xlabel('X')
plt.ylabel('y')
plt.title('Regressió Lasso')
plt.show()

Comparació d'Algoritmes de Regressió

A continuació, es presenta una taula comparativa dels diferents algoritmes de regressió discutits:

| Algoritme | Penalització | Avantatges | Desavantatges | |---------------------|--------------|-------------------------------------------------|---------------------------------------------| | Regressió Lineal | Cap | Simple, fàcil d'interpretar | Pot sobreajustar si hi ha moltes variables | | Regressió Polinòmica| Cap | Captura relacions no lineals | Pot sobreajustar fàcilment | | Regressió Ridge | L2 | Redueix el sobreajustament, útil per moltes variables | No realitza selecció de variables | | Regressió Lasso | L1 | Realitza selecció de variables, redueix el sobreajustament | Pot eliminar variables importants |

Conclusió

Els algoritmes de regressió són eines poderoses per predir valors continus. Cada tipus de regressió té els seus propis avantatges i desavantatges, i l'elecció de l'algoritme adequat depèn del problema específic i de les dades disponibles. La comprensió d'aquests algoritmes i la seva correcta aplicació és essencial per a qualsevol professional que treballi en el camp de l'anàlisi de dades i l'aprenentatge automàtic.
