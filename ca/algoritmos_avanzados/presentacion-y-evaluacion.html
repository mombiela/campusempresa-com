Introducció

En aquesta secció, presentarem els objectius del curs i els criteris d'avaluació. Aquest curs està dissenyat per proporcionar una comprensió profunda dels algorismes avançats, incloent-hi el seu disseny, anàlisi i aplicació en problemes complexos.

Objectius del Curs

En finalitzar aquest curs, els estudiants seran capaços de:

  • Comprendre i aplicar algorismes avançats en la resolució de problemes complexos.
  • Analitzar l'eficiència i complexitat de diferents algorismes.
  • Implementar algorismes avançats en un llenguatge de programació.
  • Avaluar i comparar diferents enfocaments algorísmics per a un mateix problema.

Criteris d'Avaluació

L'avaluació del curs es basarà en els següents components:

  • Participació a classe i fòrums: 10%
  • Tasques i exercicis pràctics: 40%
  • Exàmens parcials: 20%
  • Projecte final: 30%

Exemples d'Avaluació

Exemple de Tasca Pràctica

Problema: Implementar l'algorisme de Dijkstra per trobar el camí més curt en un graf dirigit amb pesos no negatius.

import heapq

def dijkstra(graph, start):
    # Inicialització
    queue = []
    heapq.heappush(queue, (0, start))
    distances = {vertex: float('infinity') for vertex in graph}
    distances[start] = 0
    visited = set()

    while queue:
        current_distance, current_vertex = heapq.heappop(queue)

        if current_vertex in visited:
            continue

        visited.add(current_vertex)

        for neighbor, weight in graph[current_vertex].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(queue, (distance, neighbor))

    return distances

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

print(dijkstra(graph, 'A'))

Explicació:

  • S'utilitza una cua de prioritat (heapq) per seleccionar el vèrtex amb la distància més curta.
  • S'inicialitzen les distàncies de tots els vèrtexs a infinit, excepte el vèrtex d'inici.
  • S'actualitzen les distàncies a mesura que s'exploren els veïns de cada vèrtex.

Exemple de Projecte Final

Descripció: Desenvolupar un sistema de recomanació basat en l'algorisme de filtratge col·laboratiu. El projecte ha d'incloure:

  • Anàlisi del problema i disseny de l'algorisme.
  • Implementació de l'algorisme en Python.
  • Avaluació del rendiment de l'algorisme utilitzant un conjunt de dades real.
  • Comparació amb altres algorismes de recomanació.

Criteris d'Avaluació:

  • Correctitud de l'algorisme: 30%
  • Eficiència del codi: 20%
  • Qualitat de l'anàlisi i disseny: 20%
  • Documentació i presentació: 30%

Conclusió

La secció de presentació i avaluació estableix les expectatives i els objectius del curs, proporcionant una guia clara sobre com s'avaluarà el progrés dels estudiants. A través de tasques pràctiques i un projecte final, els estudiants tindran l'oportunitat d'aplicar els conceptes apresos i demostrar la seva comprensió dels algorismes avançats.
