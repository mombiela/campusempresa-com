Què és l'Aprenentatge Automàtic?

L'Aprenentatge Automàtic (Machine Learning) és una branca de la intel·ligència artificial que permet a les màquines aprendre i fer prediccions o decisions basades en dades. En lloc de ser explícitament programades per realitzar una tasca, les màquines utilitzen algoritmes per identificar patrons en les dades i millorar el seu rendiment amb el temps.

Conceptes Clau

  • Dades d'Entrenament: Conjunt de dades utilitzades per entrenar el model.
  • Model: La representació matemàtica que s'ajusta a les dades d'entrenament.
  • Algoritme d'Aprenentatge: Procediment que ajusta el model a les dades.
  • Predicció: Resultat generat pel model quan se li proporcionen noves dades.
  • Validació i Prova: Mètodes per avaluar la precisió del model.

Tipus d'Aprenentatge Automàtic

Aprenentatge Supervisat

En l'aprenentatge supervisat, el model s'entrena amb un conjunt de dades etiquetades. Això significa que cada entrada de dades ve amb una etiqueta o resultat esperat.

  • Exemple: Classificació de correus electrònics com a "spam" o "no spam".
  • Algoritmes Comuns:
    • Regressió Lineal
    • Màquines de Suport Vectorial (SVM)
    • Xarxes Neuronals
from sklearn.linear_model import LinearRegression

# Dades d'entrenament
X = [[1], [2], [3], [4]]
y = [2, 3, 4, 5]

# Crear el model
model = LinearRegression()

# Entrenar el model
model.fit(X, y)

# Fer una predicció
prediction = model.predict([[6]])
print(prediction)  # Sortida esperada: [7]

Aprenentatge No Supervisat

En l'aprenentatge no supervisat, el model s'entrena amb dades que no tenen etiquetes. L'objectiu és trobar patrons o estructures ocultes en les dades.

  • Exemple: Agrupació de clients en segments de mercat.
  • Algoritmes Comuns:
    • K-means
    • Anàlisi de Components Principals (PCA)
    • Xarxes Neuronals Autoencoders
from sklearn.cluster import KMeans

# Dades d'entrenament
X = [[1, 2], [1, 4], [1, 0], [10, 2], [10, 4], [10, 0]]

# Crear el model
kmeans = KMeans(n_clusters=2)

# Entrenar el model
kmeans.fit(X)

# Fer una predicció
labels = kmeans.predict([[0, 0], [12, 3]])
print(labels)  # Sortida esperada: [1, 0]

Aprenentatge per Reforç

En l'aprenentatge per reforç, un agent aprèn a prendre decisions mitjançant la interacció amb un entorn. L'agent rep recompenses o càstigs en funció de les accions que realitza.

  • Exemple: Un robot aprenent a navegar en una habitació.
  • Algoritmes Comuns:
    • Q-Learning
    • Deep Q-Networks (DQN)
    • Algoritmes de Política (Policy Gradient)
# Exemple simplificat de Q-Learning
import numpy as np

# Definir l'entorn
states = ["A", "B", "C"]
actions = ["left", "right"]
rewards = {"A": {"left": 0, "right": 1}, "B": {"left": 1, "right": 0}, "C": {"left": 0, "right": 0}}

# Inicialitzar la taula Q
Q = np.zeros((len(states), len(actions)))

# Paràmetres d'aprenentatge
alpha = 0.1  # Taxa d'aprenentatge
gamma = 0.9  # Factor de descompte

# Simulació d'aprenentatge
for episode in range(100):
    state = np.random.choice(states)
    action = np.random.choice(actions)
    reward = rewards[state][action]
    next_state = np.random.choice(states)
    Q[states.index(state), actions.index(action)] = Q[states.index(state), actions.index(action)] + alpha * (reward + gamma * np.max(Q[states.index(next_state), :]) - Q[states.index(state), actions.index(action)])

print(Q)

Conclusió

L'aprenentatge automàtic és una eina poderosa que permet a les màquines aprendre de les dades i millorar amb el temps. Comprendre els diferents tipus d'aprenentatge automàtic i els seus algoritmes és fonamental per aplicar aquestes tècniques de manera efectiva en problemes del món real.
