Introducció

Un Arbre d'Expansió Mínima (MST, per les seves sigles en anglès) és un subgraf d'un graf no dirigit i ponderat que connecta tots els vèrtexs amb el menor pes total possible. Els MST són fonamentals en diverses aplicacions, com el disseny de xarxes, l'optimització de rutes i la teoria de grafs.

Conceptes Clau

  • Graf No Dirigit: Un conjunt de nodes connectats per arestes, on les arestes no tenen una direcció específica.
  • Pes de l'Aresta: Un valor assignat a cada aresta que representa el cost o la distància entre dos nodes.
  • Subgraf: Un graf format per un subconjunt dels nodes i arestes del graf original.
  • Connectivitat: Un graf és connectat si existeix un camí entre qualsevol parell de nodes.
  • Arbre: Un graf connectat sense cicles.

Algoritmes Principals

Algoritme de Kruskal

L'algoritme de Kruskal és un enfocament de "greedy" que construeix el MST seleccionant les arestes de menor pes i assegurant que no es formin cicles.

Passos de l'Algoritme de Kruskal

  1. Ordenar totes les arestes del graf en ordre ascendent segons el seu pes.
  2. Inicialitzar un subgraf buit.
  3. Iterar sobre les arestes ordenades i afegir l'aresta al subgraf si no forma un cicle.
  4. Repetir fins que el subgraf tingui \(V-1\) arestes (on \(V\) és el nombre de vèrtexs).

Exemple de Codi en Python

class Graf:
    def __init__(self, vertices):
        self.V = vertices
        self.arestes = []

    def afegir_aresta(self, u, v, w):
        self.arestes.append([u, v, w])

    def trobar(self, parent, i):
        if parent[i] == i:
            return i
        return self.trobar(parent, parent[i])

    def unio(self, parent, rank, x, y):
        arrel_x = self.trobar(parent, x)
        arrel_y = self.trobar(parent, y)
        if rank[arrel_x] < rank[arrel_y]:
            parent[arrel_x] = arrel_y
        elif rank[arrel_x] > rank[arrel_y]:
            parent[arrel_y] = arrel_x
        else:
            parent[arrel_y] = arrel_x
            rank[arrel_x] += 1

    def kruskal(self):
        resultat = []
        i, e = 0, 0
        self.arestes = sorted(self.arestes, key=lambda item: item[2])
        parent = []
        rank = []
        per node in range(self.V):
            parent.append(node)
            rank.append(0)
        while e < self.V - 1:
            u, v, w = self.arestes[i]
            i = i + 1
            x = self.trobar(parent, u)
            y = self.trobar(parent, v)
            if x != y:
                e = e + 1
                resultat.append([u, v, w])
                self.unio(parent, rank, x, y)
        return resultat

# Exemple d'ús
g = Graf(4)
g.afegir_aresta(0, 1, 10)
g.afegir_aresta(0, 2, 6)
g.afegir_aresta(0, 3, 5)
g.afegir_aresta(1, 3, 15)
g.afegir_aresta(2, 3, 4)

mst = g.kruskal()
print("Arestes en el MST:")
for u, v, pes in mst:
    print(f"{u} -- {v} == {pes}")

Algoritme de Prim

L'algoritme de Prim també és un enfocament de "greedy" que construeix el MST seleccionant el vèrtex més proper no inclòs en el MST.

Passos de l'Algoritme de Prim

  1. Inicialitzar un conjunt buit per als vèrtexs inclosos en el MST.
  2. Inicialitzar un conjunt amb un vèrtex arbitrari.
  3. Repetir fins que tots els vèrtexs estiguin en el MST:
    • Seleccionar l'aresta de menor pes que connecta un vèrtex en el MST amb un vèrtex fora del MST.
    • Afegir el vèrtex i l'aresta al MST.

Exemple de Codi en Python

import heapq

def prim(graf, inici):
    mst = []
    visitats = set()
    min_heap = [(0, inici, None)]
    while min_heap:
        pes, u, prev = heapq.heappop(min_heap)
        if u no in visitats:
            visitats.add(u)
            if prev is not None:
                mst.append((prev, u, pes))
            for v, pes in graf[u]:
                if v no in visitats:
                    heapq.heappush(min_heap, (pes, v, u))
    return mst

# Exemple d'ús
graf = {
    0: [(1, 10), (2, 6), (3, 5)],
    1: [(0, 10), (3, 15)],
    2: [(0, 6), (3, 4)],
    3: [(0, 5), (1, 15), (2, 4)]
}

mst = prim(graf, 0)
print("Arestes en el MST:")
for u, v, pes in mst:
    print(f"{u} -- {v} == {pes}")

Comparació d'Algoritmes

| Algoritme | Complexitat Temporal | Millor per a | |-----------|-----------------------|--------------| | Kruskal | \(O(E \log E)\) | Grafs dispersos | | Prim | \(O(E + V \log V)\) | Grafs densos |

  • Kruskal: És més eficient en grafs dispersos a causa del seu enfocament en les arestes.
  • Prim: És més eficient en grafs densos a causa del seu enfocament en els vèrtexs.

Conclusió

Els Arbres d'Expansió Mínima són una eina essencial en la teoria de grafs i tenen aplicacions pràctiques en diverses àrees. Els algoritmes de Kruskal i Prim són dos mètodes eficients per trobar el MST, cadascun amb els seus avantatges depenent de l'estructura del graf. Entendre aquests algoritmes i les seves implementacions és crucial per resoldre problemes d'optimització en xarxes i altres aplicacions.
