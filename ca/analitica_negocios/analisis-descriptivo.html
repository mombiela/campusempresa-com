L'anàlisi descriptiu és una de les primeres etapes en el procés d'analítica de negocis. El seu objectiu principal és resumir i descriure les característiques d'un conjunt de dades, proporcionant una visió clara i comprensible del que les dades representen. Aquest tipus d'anàlisi és fonamental per entendre el context i les tendències abans de procedir a anàlisis més complexos.

Objectius de l'Anàlisi Descriptiu

  • Resumir dades: Proporcionar una visió general de les dades mitjançant estadístiques resumides.
  • Identificar patrons: Detectar tendències i patrons en les dades.
  • Visualització de dades: Utilitzar gràfics i taules per representar les dades de manera visual.
  • Preparació per a anàlisis avançats: Establir una base sòlida per a anàlisis predictius i prescriptius.

Conceptes Clau

  • Mesures de Tendència Central:

    • Mitjana: El promig de tots els valors.
    • Mediana: El valor central quan les dades estan ordenades.
    • Moda: El valor que apareix amb més freqüència.

  • Mesures de Dispersió:

    • Rang: La diferència entre el valor màxim i el mínim.
    • Variància: La mesura de quant varien les dades respecte a la mitjana.
    • Desviació estàndard: L'arrel quadrada de la variància, que indica la dispersió de les dades.

  • Distribució de Freqüència:

    • Taules de freqüència: Taules que mostren la freqüència de cada valor o rang de valors.
    • Histogrames: Gràfics de barres que representen la distribució de freqüències.

Eines i Tècniques

Estadístiques Descriptives

Les estadístiques descriptives són fonamentals per resumir i descriure les dades. A continuació, es mostra un exemple de com calcular algunes estadístiques descriptives bàsiques utilitzant Python i la biblioteca pandas.

import pandas as pd

# Crear un DataFrame d'exemple
data = {'Vendes': [200, 220, 250, 270, 300, 320, 350]}
df = pd.DataFrame(data)

# Calcular estadístiques descriptives
mitjana = df['Vendes'].mean()
mediana = df['Vendes'].median()
moda = df['Vendes'].mode()[0]
rang = df['Vendes'].max() - df['Vendes'].min()
variancia = df['Vendes'].var()
desviacio_estandard = df['Vendes'].std()

# Mostrar resultats
print(f"Mitjana: {mitjana}")
print(f"Mediana: {mediana}")
print(f"Moda: {moda}")
print(f"Rang: {rang}")
print(f"Variància: {variancia}")
print(f"Desviació Estàndard: {desviacio_estandard}")

Visualització de Dades

La visualització de dades és crucial per interpretar i comunicar els resultats de l'anàlisi descriptiu. A continuació, es mostra un exemple de com crear un histograma utilitzant matplotlib.

import matplotlib.pyplot as plt

# Crear un histograma
plt.hist(df['Vendes'], bins=5, edgecolor='black')
plt.title('Distribució de Vendes')
plt.xlabel('Vendes')
plt.ylabel('Freqüència')
plt.show()

Conclusió

L'anàlisi descriptiu és una etapa essencial en l'analítica de negocis, ja que proporciona una base sòlida per a qualsevol anàlisi posterior. En resumir i visualitzar les dades, podem identificar patrons i tendències que són crucials per a la presa de decisions informades. A mesura que avancem en el curs, construirem sobre aquests conceptes per explorar anàlisis més avançats i tècniques predictives.
