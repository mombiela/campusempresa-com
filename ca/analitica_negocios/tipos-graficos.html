Introducció

En l'analítica de negocis, la visualització de dades és crucial per interpretar i comunicar informació de manera efectiva. Hi ha diversos tipus de gràfics, cadascun adequat per a diferents tipus de dades i propòsits analítics. En aquesta secció, explorarem els tipus de gràfics més comuns i quan és apropiat utilitzar-los.

Tipus de Gràfics

Gràfics de Barres

  • Descripció: Els gràfics de barres són ideals per comparar quantitats entre diferents categories.
  • Quan Utilitzar-los:
    • Comparar vendes de diferents productes.
    • Comparar ingressos de diferents regions.
  • Exemple:
import matplotlib.pyplot as plt

categories = ['Producte A', 'Producte B', 'Producte C']
values = [100, 150, 80]

plt.bar(categories, values)
plt.xlabel('Productes')
plt.ylabel('Vendes')
plt.title('Vendes per Producte')
plt.show()

Gràfics de Línies

  • Descripció: Els gràfics de línies són útils per mostrar tendències al llarg del temps.
  • Quan Utilitzar-los:
    • Seguiment de vendes mensuals.
    • Anàlisi de tendències de mercat.
  • Exemple:
import matplotlib.pyplot as plt

months = ['Gener', 'Febrer', 'Març', 'Abril']
sales = [200, 250, 300, 400]

plt.plot(months, sales, marker='o')
plt.xlabel('Mesos')
plt.ylabel('Vendes')
plt.title('Tendència de Vendes Mensuals')
plt.show()

Gràfics de Pastís

  • Descripció: Els gràfics de pastís mostren la proporció de diferents categories en un tot.
  • Quan Utilitzar-los:
    • Distribució del pressupost.
    • Participació de mercat.
  • Exemple:
import matplotlib.pyplot as plt

labels = ['Màrqueting', 'Desenvolupament', 'Operacions']
sizes = [40, 35, 25]

plt.pie(sizes, labels=labels, autopct='%1.1f%%')
plt.title('Distribució del Pressupost')
plt.show()

Gràfics de Dispersió

  • Descripció: Els gràfics de dispersió són útils per mostrar la relació entre dues variables.
  • Quan Utilitzar-los:
    • Anàlisi de correlació.
    • Identificació de patrons.
  • Exemple:
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

plt.scatter(x, y)
plt.xlabel('Variable X')
plt.ylabel('Variable Y')
plt.title('Gràfic de Dispersió')
plt.show()

Gràfics d'Àrea

  • Descripció: Els gràfics d'àrea són similars als gràfics de línies, però l'àrea sota la línia està omplerta.
  • Quan Utilitzar-los:
    • Mostrar acumulació de dades al llarg del temps.
    • Comparar diferents categories en un context temporal.
  • Exemple:
import matplotlib.pyplot as plt

months = ['Gener', 'Febrer', 'Març', 'Abril']
sales_A = [200, 250, 300, 400]
sales_B = [150, 200, 250, 300]

plt.fill_between(months, sales_A, label='Producte A', alpha=0.5)
plt.fill_between(months, sales_B, label='Producte B', alpha=0.5)
plt.xlabel('Mesos')
plt.ylabel('Vendes')
plt.title('Vendes Acumulades per Producte')
plt.legend()
plt.show()

Conclusió

Escollir el tipus de gràfic adequat és essencial per a una visualització de dades efectiva. Cada gràfic té les seves fortaleses i és més adequat per a certs tipus d'anàlisi. En comprendre quan i com utilitzar cada tipus de gràfic, pots comunicar els teus descobriments de manera més clara i persuasiva. Practica amb diferents tipus de gràfics i dades per familiaritzar-te amb els seus usos i aplicacions.
