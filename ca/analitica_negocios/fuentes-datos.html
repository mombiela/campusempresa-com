Introducció

En l'analítica de negocis, les fonts de dades són fonamentals per obtenir informació valuosa que permeti prendre decisions informades. Aquest tema cobrirà els diferents tipus de fonts de dades, com es poden utilitzar i exemples pràctics per il·lustrar la seva aplicació en el món real.

Tipus de Fonts de Dades

Fonts de Dades Internes

Les fonts de dades internes són aquelles que provenen de dins de l'organització. Aquestes poden incloure:

  • Bases de dades transaccionals: Dades generades per les operacions diàries de l'empresa, com vendes, inventaris i registres de clients.
  • Sistemes de gestió empresarial (ERP): Dades integrades de diferents departaments com finances, recursos humans i producció.
  • Sistemes de gestió de relacions amb clients (CRM): Informació sobre interaccions amb clients, historial de compres i preferències.

Fonts de Dades Externes

Les fonts de dades externes són aquelles que provenen de fora de l'organització. Aquestes poden incloure:

  • Dades de mercat: Informació sobre tendències del mercat, competidors i anàlisis sectorials.
  • Xarxes socials: Dades obtingudes de plataformes com Twitter, Facebook i LinkedIn.
  • Dades obertes (Open Data): Informació pública disponible a través de governs i organitzacions no governamentals.

Fonts de Dades Semi-estructurades i No Estructurades

A més de les fonts estructurades, també existeixen dades semi-estructurades i no estructurades:

  • Dades semi-estructurades: Dades que no segueixen un esquema rígid però tenen una estructura definida, com arxius XML i JSON.
  • Dades no estructurades: Dades sense una estructura predefinida, com correus electrònics, documents de text i contingut multimèdia.

Exemples Pràctics

Exemple 1: Ús de Dades Internes

Suposem que una empresa de retail vol analitzar el comportament de compra dels seus clients. Per a això, pot utilitzar dades del seu sistema CRM i bases de dades transaccionals.

SELECT customer_id, purchase_date, amount
FROM sales
WHERE purchase_date BETWEEN '2023-01-01' AND '2023-12-31';

Exemple 2: Ús de Dades Externes

Una empresa de màrqueting digital pot utilitzar dades de xarxes socials per analitzar la percepció de la seva marca.

import tweepy

# Autenticació amb l'API de Twitter
auth = tweepy.OAuthHandler('CONSUMER_KEY', 'CONSUMER_SECRET')
auth.set_access_token('ACCESS_TOKEN', 'ACCESS_TOKEN_SECRET')
api = tweepy.API(auth)

# Cerca de tweets que mencionen la marca
tweets = api.search(q='la_teva_marca', count=100)

for tweet in tweets:
    print(tweet.text)

Exemple 3: Ús de Dades Semi-estructurades

Una empresa de logística pot utilitzar dades en format JSON per analitzar el rendiment de les seves entregues.

import json

# Carregar dades des d'un arxiu JSON
with open('entregues.json') as file:
    data = json.load(file)

# Analitzar dades
for entrega in data['entregues']:
    print(f"ID: {entrega['id']}, Estat: {entrega['estat']}")

Conclusió

Les fonts de dades són essencials per a l'analítica de negocis, proporcionant la matèria primera necessària per generar insights i prendre decisions informades. És crucial entendre les diferents fonts disponibles, tant internes com externes, i com es poden utilitzar de manera efectiva. Amb exemples pràctics, hem vist com extreure i utilitzar aquestes dades per obtenir informació valuosa.

En el següent tema, explorarem com netejar i preparar aquestes dades per a la seva anàlisi, assegurant que siguin precises i rellevants.
