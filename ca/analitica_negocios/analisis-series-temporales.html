Introducció

L'anàlisi de sèries temporals és una tècnica crucial en l'analítica de negocis que permet a les empreses comprendre i predir comportaments futurs basats en dades històriques. Aquest mòdul cobrirà els conceptes fonamentals, mètodes i eines per a l'anàlisi de sèries temporals, proporcionant exemples pràctics i exercicis per consolidar l'aprenentatge.

Conceptes Clau

  • Sèrie Temporal: Una seqüència de dades punts, típicament mesurats en intervals de temps successius.
  • Estacionarietat: Una propietat d'una sèrie temporal on les seves estadístiques (mitjana, variància) no canvien amb el temps.
  • Descomposició: Separar una sèrie temporal en components com tendència, estacionalitat i soroll.
  • Models ARIMA: Models autoregressius integrats de mitjana mòbil, utilitzats per predir valors futurs en una sèrie temporal.
  • Suavitzat Exponencial: Tècniques que apliquen una mitjana ponderada d'observacions passades per predir valors futurs.

Mètodes d'Anàlisi

Descomposició de Sèries Temporals

La descomposició d'una sèrie temporal implica dividir-la en components més simples:

  • Tendència (T): La direcció general en què es mou la sèrie temporal.
  • Estacionalitat (S): Fluctuacions periòdiques que ocorren en intervals regulars.
  • Soroll (R): Variacions aleatòries que no segueixen un patró específic.

Exemple en Python utilitzant statsmodels:

import pandas as pd
import statsmodels.api as sm
import matplotlib.pyplot as plt

# Carregar dades d'exemple
data = sm.datasets.co2.load_pandas().data
data = data.resample('M').mean()

# Descomposició de la sèrie temporal
decomposition = sm.tsa.seasonal_decompose(data['co2'], model='additive')
decomposition.plot()
plt.show()

Models ARIMA

Els models ARIMA són una classe de models que expliquen una sèrie temporal en termes dels seus propis valors passats (autoregressius) i l'error passat (mitjana mòbil).

  • AR (Autoregressiu): Relació entre una observació i un nombre d'observacions anteriors.
  • I (Integrat): Diferenciació de les observacions per fer la sèrie estacionària.
  • MA (Mitjana Mòbil): Modela l'error de la sèrie com una combinació lineal d'errors passats.

Exemple en Python utilitzant statsmodels:

from statsmodels.tsa.arima.model import ARIMA

# Ajustar el model ARIMA
model = ARIMA(data['co2'], order=(5,1,0))
model_fit = model.fit()

# Resum del model
print(model_fit.summary())

# Prediccions
forecast = model_fit.forecast(steps=12)
print(forecast)

Suavitzat Exponencial

El suavitzat exponencial és una tècnica que aplica una mitjana ponderada d'observacions passades, on els pesos decreixen exponencialment amb el temps.

Exemple en Python utilitzant statsmodels:

from statsmodels.tsa.holtwinters import ExponentialSmoothing

# Ajustar el model de suavitzat exponencial
model = ExponentialSmoothing(data['co2'], trend='add', seasonal='add', seasonal_periods=12)
model_fit = model.fit()

# Prediccions
forecast = model_fit.forecast(steps=12)
print(forecast)

Exercicis Pràctics

  1. Descomposició de Sèries Temporals:

    • Utilitza una sèrie temporal de vendes mensuals i descompon-la en els seus components.
    • Identifica la tendència i l'estacionalitat.

  2. Models ARIMA:

    • Ajusta un model ARIMA a una sèrie temporal de dades financeres.
    • Avalua el model utilitzant mètriques com AIC i BIC.

  3. Suavitzat Exponencial:

    • Aplica el suavitzat exponencial a una sèrie temporal de demanda de productes.
    • Compara les prediccions amb els valors reals.

Conclusió

L'anàlisi de sèries temporals és una eina poderosa en l'analítica de negocis, permetent a les empreses prendre decisions informades basades en patrons històrics. A través de la descomposició, els models ARIMA i el suavitzat exponencial, els professionals poden obtenir insights valuosos i realitzar prediccions precises. Practicar amb dades reals i aplicar aquests mètodes enfortirà la teva capacitat per analitzar i predir sèries temporals en diversos contextos empresarials.
