Introducció

La neteja i transformació de dades són passos crítics en el procés d'analítica de negocis. Aquests processos asseguren que les dades siguin precises, completes i estiguin en un format adequat per a l'anàlisi. En aquesta secció, aprendràs tècniques i eines per netejar i transformar dades de manera efectiva.

Importància de la Neteja de Dades

La neteja de dades és essencial per:

  • Eliminar errors i duplicats.
  • Gestionar valors faltants.
  • Corregir inconsistències.
  • Millorar la qualitat de les dades per a anàlisis precises.

Tècniques de Neteja de Dades

Eliminació de Duplicats

Els duplicats poden distorsionar els resultats de l'anàlisi. Per eliminar-los:

import pandas as pd

# Crear un DataFrame d'exemple
data = {'ID': [1, 2, 2, 3], 'Nom': ['Ana', 'Luis', 'Luis', 'Carlos']}
df = pd.DataFrame(data)

# Eliminar duplicats
df_cleaned = df.drop_duplicates()
print(df_cleaned)

Gestió de Valors Faltants

Els valors faltants poden ser tractats de diverses maneres:

  • Eliminació de files o columnes amb valors faltants.
  • Imputació de valors faltants amb la mitjana, mediana o moda.
  • Ús de tècniques avançades com la imputació múltiple.

Exemple d'imputació amb la mitjana:

import pandas as pd
import numpy as np

# Crear un DataFrame d'exemple amb valors faltants
data = {'ID': [1, 2, 3], 'Edat': [25, np.nan, 30]}
df = pd.DataFrame(data)

# Imputar valors faltants amb la mitjana
df['Edat'].fillna(df['Edat'].mean(), inplace=True)
print(df)

Correcció d'Inconsistències

Les inconsistències poden sorgir de diferents formats o errors tipogràfics. Exemple de correcció d'inconsistències:

import pandas as pd

# Crear un DataFrame d'exemple amb inconsistències
data = {'ID': [1, 2, 3], 'Gènere': ['M', 'F', 'm']}
df = pd.DataFrame(data)

# Corregir inconsistències
df['Gènere'] = df['Gènere'].str.upper()
print(df)

Transformació de Dades

La transformació de dades implica convertir les dades en un format adequat per a l'anàlisi. Això pot incloure:

  • Normalització i estandardització.
  • Creació de noves variables.
  • Agrupació i agregació de dades.

Normalització i Estandardització

Aquestes tècniques són útils per posar les dades en una escala comuna.

from sklearn.preprocessing import StandardScaler, MinMaxScaler
import pandas as pd

# Crear un DataFrame d'exemple
data = {'ID': [1, 2, 3], 'Ingressos': [50000, 60000, 55000]}
df = pd.DataFrame(data)

# Normalització
scaler = MinMaxScaler()
df['Ingressos_Normalitzats'] = scaler.fit_transform(df[['Ingressos']])

# Estandardització
scaler = StandardScaler()
df['Ingressos_Estandarditzats'] = scaler.fit_transform(df[['Ingressos']])
print(df)

Creació de Noves Variables

De vegades és útil crear noves variables a partir de les existents.

import pandas as pd

# Crear un DataFrame d'exemple
data = {'ID': [1, 2, 3], 'Ingressos': [50000, 60000, 55000]}
df = pd.DataFrame(data)

# Crear una nova variable 'Ingressos_Anuals' a partir de 'Ingressos'
df['Ingressos_Anuals'] = df['Ingressos'] * 12
print(df)

Agrupació i Agregació de Dades

Aquestes tècniques són útils per resumir les dades.

import pandas as pd

# Crear un DataFrame d'exemple
data = {'ID': [1, 2, 3, 4], 'Departament': ['Vendes', 'Vendes', 'IT', 'IT'], 'Ingressos': [50000, 60000, 55000, 70000]}
df = pd.DataFrame(data)

# Agrupar i agregar dades
df_grouped = df.groupby('Departament').agg({'Ingressos': 'mean'})
print(df_grouped)

Conclusió

La neteja i transformació de dades són passos fonamentals en l'analítica de negocis. A través de tècniques com l'eliminació de duplicats, gestió de valors faltants, correcció d'inconsistències, normalització, creació de noves variables i agrupació de dades, pots assegurar que les teves dades estiguin en la millor forma possible per a l'anàlisi. Practicar aquestes tècniques et permetrà millorar la qualitat de les teves anàlisis i obtenir insights més precisos i valuosos.
