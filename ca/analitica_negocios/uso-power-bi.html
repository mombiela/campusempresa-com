Introducció a Microsoft Power BI

Microsoft Power BI és una eina d'anàlisi de negocis que permet als usuaris visualitzar i compartir dades de manera interactiva. És una solució poderosa per transformar dades sense processar en informació significativa i accionable.

Conceptes Clau

  • Power BI Desktop: Aplicació d'escriptori per crear informes i visualitzacions.
  • Power BI Service: Servei al núvol per compartir i col·laborar en informes.
  • Power BI Mobile: Aplicació mòbil per accedir a informes i panells sobre la marxa.
  • Datasets: Conjunts de dades que s'utilitzen per crear informes.
  • Reports: Informes que contenen una o més visualitzacions.
  • Dashboards: Panells que combinen visualitzacions de diferents informes.

Instal·lació i Configuració

Requisits del Sistema

  • Sistema Operatiu: Windows 7/8/10 (64 bits)
  • RAM: Mínim 4 GB (8 GB recomanat)
  • Espai en Disc: Almenys 1 GB d'espai disponible

Instal·lació de Power BI Desktop

  1. Visita el lloc web de Power BI.
  2. Descarrega l'instal·lador de Power BI Desktop.
  3. Executa l'instal·lador i segueix les instruccions en pantalla.

Importació de Dades

Fonts de Dades Suportades

  • Arxius: Excel, CSV, XML, JSON
  • Bases de Dades: SQL Server, Oracle, MySQL
  • Serveis al Núvol: Azure, Google Analytics, Salesforce

Exemple: Importació des d'un Arxiu Excel

  1. Obre Power BI Desktop.
  2. Selecciona Obtenir dades a la cinta d'opcions.
  3. Tria Excel i selecciona l'arxiu que desitges importar.
  4. Selecciona les taules o rangs de dades que desitges carregar.
  5. Fes clic a Carregar.
# Exemple de codi per importar dades des d'Excel
let
    Source = Excel.Workbook(File.Contents("C:\\ruta\\a\\tu\\archivo.xlsx"), null, true),
    Sheet1 = Source{[Item="Hoja1",Kind="Sheet"]}[Data]
in
    Sheet1

Creació de Visualitzacions

Tipus de Visualitzacions

  • Gràfics de Barres i Columnes: Comparació de valors entre diferents categories.
  • Gràfics de Línies: Tendències al llarg del temps.
  • Mapes: Visualització geogràfica de dades.
  • Gràfics de Pastís: Distribució de parts d'un tot.
  • Taules i Matrius: Dades tabulars detallades.

Exemple: Creació d'un Gràfic de Barres

  1. Selecciona Gràfic de barres al panell de visualitzacions.
  2. Arrossega el camp Categoria a l'eix X.
  3. Arrossega el camp Valor a l'eix Y.
# Exemple de configuració d'un gràfic de barres
let
    Source = Excel.Workbook(File.Contents("C:\\ruta\\a\\tu\\archivo.xlsx"), null, true),
    Sheet1 = Source{[Item="Hoja1",Kind="Sheet"]}[Data],
    #"Changed Type" = Table.TransformColumnTypes(Sheet1,{{"Categoria", type text}, {"Valor", Int64.Type}})
in
    #"Changed Type"

Publicació i Compartició

Publicació a Power BI Service

  1. Fes clic a Publicar a la cinta d'opcions.
  2. Inicia sessió al teu compte de Power BI.
  3. Selecciona l'àrea de treball on desitges publicar l'informe.

Compartició d'Informes

  • Compartir Enllaços: Genera un enllaç compartit perquè altres usuaris puguin veure l'informe.
  • Col·laboració en Equips: Utilitza àrees de treball compartides per col·laborar amb el teu equip.

Conclusió

Microsoft Power BI és una eina integral per a l'analítica de negocis que permet als usuaris importar, transformar, visualitzar i compartir dades de manera eficient. Amb la seva interfície intuïtiva i capacitats robustes, Power BI facilita la presa de decisions basada en dades. Aquest curs ha cobert els aspectes bàsics de la instal·lació, importació de dades, creació de visualitzacions i publicació d'informes, proporcionant una base sòlida per explorar més a fons les capacitats avançades de Power BI.
