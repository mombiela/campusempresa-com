Introducció

Les bases de dades són una part fonamental de la tecnologia de la informació, permetent l'emmagatzematge, organització i recuperació eficient de dades. Aquest tema explora l'evolució de les bases de dades des dels seus inicis fins a les tecnologies modernes.

Orígens de les Bases de Dades

Sistemes de Fitxers

Abans de l'existència de les bases de dades, les dades s'emmagatzemaven en sistemes de fitxers simples. Aquests sistemes presentaven diverses limitacions:

  • Redundància de dades: Les mateixes dades s'emmagatzemaven en múltiples fitxers.
  • Inconsistència de dades: Les actualitzacions en un fitxer no es reflectien en altres.
  • Dificultat en l'accés: No hi havia una manera eficient de consultar i manipular les dades.

Primers Sistemes de Gestió de Bases de Dades (DBMS)

A la dècada de 1960, van sorgir els primers sistemes de gestió de bases de dades (DBMS) per superar les limitacions dels sistemes de fitxers. Els primers DBMS van ser:

  • Model Jeràrquic: Organitza les dades en una estructura d'arbre. Exemple: IMS (Information Management System) d'IBM.
  • Model de Xarxa: Permet relacions més complexes entre les dades mitjançant un graf. Exemple: IDMS (Integrated Database Management System).

Bases de Dades Relacionals

Introducció del Model Relacional

El 1970, Edgar F. Codd va proposar el model relacional, que va revolucionar la manera en què es gestionaven les bases de dades. Les característiques clau del model relacional inclouen:

  • Taules (Relacions): Les dades s'organitzen en taules.
  • Claus Primàries i Foranes: Per identificar de manera única les files i establir relacions entre taules.
  • SQL (Structured Query Language): Un llenguatge estàndard per consultar i manipular dades.

Exemple de Codi SQL

-- Crear una taula d'exemple
CREATE TABLE empleats (
    id INT PRIMARY KEY,
    nom VARCHAR(100),
    lloc VARCHAR(50)
);

-- Inserir dades a la taula
INSERT INTO empleats (id, nom, lloc) VALUES (1, 'Joan Pérez', 'Desenvolupador');
INSERT INTO empleats (id, nom, lloc) VALUES (2, 'Anna Gómez', 'Analista');

-- Consultar dades de la taula
SELECT * FROM empleats;

Avantatges del Model Relacional

  • Integritat de Dades: Assegura la consistència i precisió de les dades.
  • Flexibilitat: Facilita la modificació i consulta de dades.
  • Normalització: Minimitza la redundància i dependència de dades.

Bases de Dades NoSQL

Aparició de NoSQL

Amb l'auge de les aplicacions web i la necessitat de manejar grans volums de dades no estructurades, van sorgir les bases de dades NoSQL. Aquestes bases de dades ofereixen:

  • Escalabilitat Horitzontal: Capacitat de distribuir dades en múltiples servidors.
  • Flexibilitat en el Model de Dades: No requereixen un esquema fix.

Tipus de Bases de Dades NoSQL

  • Documentals: Emmagatzemen dades en documents JSON o BSON. Exemple: MongoDB.
  • Clau-Valor: Emmagatzemen dades com a parells clau-valor. Exemple: Redis.
  • Columnar: Emmagatzemen dades en columnes en lloc de files. Exemple: Apache Cassandra.
  • Grafs: Emmagatzemen dades en nodes i arestes. Exemple: Neo4j.

Exemple de Codi MongoDB

// Inserir un document en una col·lecció
db.empleats.insertOne({
    id: 1,
    nom: "Joan Pérez",
    lloc: "Desenvolupador"
});

// Consultar documents en una col·lecció
db.empleats.find();

Bases de Dades al Núvol

Serveis de Bases de Dades al Núvol

Amb l'adopció de la computació al núvol, molts proveïdors ofereixen serveis de bases de dades gestionats. Exemples inclouen:

  • Amazon RDS: Servei de base de dades relacional.
  • Google Cloud Firestore: Base de dades NoSQL documental.
  • Azure Cosmos DB: Base de dades multimodel.

Avantatges de les Bases de Dades al Núvol

  • Escalabilitat: Ajust automàtic de recursos segons la demanda.
  • Alta Disponibilitat: Redundància i recuperació davant desastres.
  • Gestió Simplificada: Reducció de la càrrega operativa.

Conclusió

L'evolució de les bases de dades ha estat impulsada per la necessitat de manejar dades de manera més eficient i flexible. Des dels sistemes de fitxers fins a les modernes bases de dades al núvol, cada avenç ha permès superar les limitacions anteriors i ha obert noves possibilitats per al maneig de dades en l'àmbit de la tecnologia de la informació.
