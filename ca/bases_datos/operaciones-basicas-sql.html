Introducció

En aquesta secció, aprendràs les operacions bàsiques en SQL que són fonamentals per interactuar amb bases de dades relacionals. Aquestes operacions inclouen la creació, lectura, actualització i eliminació de dades, comunament conegudes per les seves sigles en anglès com CRUD (Create, Read, Update, Delete).

Creació de Dades (INSERT)

L'operació INSERT s'utilitza per afegir nous registres a una taula en la base de dades.

Sintaxi Bàsica

INSERT INTO nom_taula (columna1, columna2, columna3, ...)
VALUES (valor1, valor2, valor3, ...);

Exemple

Suposem que tenim una taula anomenada empleats amb les columnes id, nom, edat i departament.

INSERT INTO empleats (id, nom, edat, departament)
VALUES (1, 'Joan Pérez', 30, 'Recursos Humans');

Lectura de Dades (SELECT)

L'operació SELECT s'utilitza per consultar i recuperar dades d'una taula.

Sintaxi Bàsica

SELECT columna1, columna2, ...
FROM nom_taula
WHERE condició;

Exemple

Per seleccionar tots els empleats del departament de "Recursos Humans":

SELECT nom, edat
FROM empleats
WHERE departament = 'Recursos Humans';

Actualització de Dades (UPDATE)

L'operació UPDATE s'utilitza per modificar les dades existents en una taula.

Sintaxi Bàsica

UPDATE nom_taula
SET columna1 = valor1, columna2 = valor2, ...
WHERE condició;

Exemple

Per actualitzar l'edat de "Joan Pérez" a 31:

UPDATE empleats
SET edat = 31
WHERE nom = 'Joan Pérez';

Eliminació de Dades (DELETE)

L'operació DELETE s'utilitza per eliminar registres d'una taula.

Sintaxi Bàsica

DELETE FROM nom_taula
WHERE condició;

Exemple

Per eliminar a "Joan Pérez" de la taula empleats:

DELETE FROM empleats
WHERE nom = 'Joan Pérez';

Comparació d'Operacions CRUD

| Operació | Descripció | Sintaxi Bàsica | |-----------|-------------|-----------------| | INSERT | Afegeix nous registres a una taula | INSERT INTO nom_taula (col1, col2, ...) VALUES (val1, val2, ...); | | SELECT | Recupera dades d'una taula | SELECT col1, col2, ... FROM nom_taula WHERE condició; | | UPDATE | Modifica dades existents en una taula | UPDATE nom_taula SET col1 = val1, col2 = val2, ... WHERE condició; | | DELETE | Elimina registres d'una taula | DELETE FROM nom_taula WHERE condició; |

Conclusió

Les operacions bàsiques en SQL són essencials per a qualsevol professional de TI que treballi amb bases de dades. Dominar INSERT, SELECT, UPDATE i DELETE et permetrà gestionar i manipular dades de manera efectiva. Practica aquests comandaments amb diferents escenaris i condicions per enfortir la teva comprensió i habilitats en SQL.
