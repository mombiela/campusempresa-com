Introducció al Llenguatge SQL

El Llenguatge de Consulta Estructurat (SQL, per les seves sigles en anglès) és un llenguatge estàndard utilitzat per gestionar i manipular bases de dades relacionals. SQL permet realitzar diverses operacions com la creació de bases de dades i taules, la inserció de dades, l'actualització i eliminació de registres, i la consulta d'informació emmagatzemada.

Conceptes Clau

  • Base de Dades: Conjunt organitzat de dades emmagatzemades i gestionades electrònicament.
  • Taula: Estructura dins d'una base de dades que organitza les dades en files i columnes.
  • Registre: Fila en una taula que representa una única entrada de dades.
  • Camp: Columna en una taula que representa un atribut específic de les dades.

Operacions Bàsiques en SQL

Creació de Bases de Dades i Taules

Per començar a treballar amb SQL, primer necessitem crear una base de dades i després definir les taules que emmagatzemaran les nostres dades.

-- Crear una base de dades
CREATE DATABASE LaMevaBaseDeDades;

-- Seleccionar la base de dades
USE LaMevaBaseDeDades;

-- Crear una taula
CREATE TABLE Empleats (
    ID INT PRIMARY KEY,
    Nom VARCHAR(50),
    Cognom VARCHAR(50),
    Edat INT,
    Departament VARCHAR(50)
);

Inserció de Dades

Un cop tenim les nostres taules, podem inserir dades en elles.

-- Inserir un registre en la taula Empleats
INSERT INTO Empleats (ID, Nom, Cognom, Edat, Departament)
VALUES (1, 'Joan', 'Pérez', 30, 'Vendes');

Consulta de Dades

La consulta de dades és una de les operacions més comunes en SQL. Utilitzem la clàusula SELECT per recuperar informació d'una o més taules.

-- Consultar tots els registres de la taula Empleats
SELECT * FROM Empleats;

-- Consultar registres específics
SELECT Nom, Cognom FROM Empleats WHERE Edat > 25;

Actualització de Dades

Podem actualitzar les dades existents en una taula utilitzant la clàusula UPDATE.

-- Actualitzar el departament d'un empleat
UPDATE Empleats
SET Departament = 'Màrqueting'
WHERE ID = 1;

Eliminació de Dades

Per eliminar registres d'una taula, utilitzem la clàusula DELETE.

-- Eliminar un registre de la taula Empleats
DELETE FROM Empleats WHERE ID = 1;

Funcions i Operadors en SQL

Funcions Agregades

SQL proporciona diverses funcions agregades que permeten realitzar càlculs sobre un conjunt de valors.

  • COUNT: Compta el nombre de files.
  • SUM: Suma els valors d'una columna.
  • AVG: Calcula la mitjana dels valors d'una columna.
  • MAX: Troba el valor màxim.
  • MIN: Troba el valor mínim.
-- Comptar el nombre d'empleats
SELECT COUNT(*) FROM Empleats;

-- Calcular l'edat mitjana dels empleats
SELECT AVG(Edat) FROM Empleats;

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors.

  • =: Igual a
  • <>: Diferent de
  • >: Major que
  • <: Menor que
  • >=: Major o igual que
  • <=: Menor o igual que
-- Seleccionar empleats majors de 25 anys
SELECT * FROM Empleats WHERE Edat > 25;

Conclusió

El llenguatge SQL és una eina poderosa i essencial per a qualsevol professional que treballi amb bases de dades. A través d'aquest tema, hem cobert les operacions bàsiques de SQL, incloent la creació de bases de dades i taules, la inserció, consulta, actualització i eliminació de dades. A més, hem explorat funcions agregades i operadors de comparació que permeten realitzar consultes més complexes i obtenir informació valuosa de les nostres bases de dades. Practicar aquests conceptes és fonamental per dominar SQL i aplicar-lo eficaçment en la gestió de dades.
