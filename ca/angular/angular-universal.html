Introducció a Angular Universal

Angular Universal és una tecnologia que permet que les aplicacions Angular es renderitzin al costat del servidor. Això pot millorar el rendiment, el SEO i l'experiència de l'usuari en lliurar pàgines completament renderitzades al client.

Conceptes Clau

  • Renderitzat del costat del servidor (SSR): El procés de renderitzar pàgines web al servidor en lloc del client.
  • Angular Universal: Una eina que permet SSR per a aplicacions Angular.
  • Beneficis SEO: Els motors de cerca poden rastrejar pàgines completament renderitzades, millorant els rànquings dels motors de cerca.
  • Rendiment: Temps de càrrega inicial més ràpids ja que el servidor envia una pàgina completament renderitzada al client.

Configuració d'Angular Universal

Per començar amb Angular Universal, cal afegir-lo al teu projecte Angular existent.

Passos per Afegir Angular Universal

  1. Instal·la Angular Universal:

    ng add @nguniversal/express-engine

    Aquesta comanda configurarà la teva aplicació Angular per utilitzar Angular Universal amb un servidor Express.

  2. Actualitza angular.json: Assegura't que les configuracions de build i server estiguin correctament configurades al teu fitxer angular.json.

  3. Modifica server.ts: El fitxer server.ts és el punt d'entrada per a la teva aplicació del costat del servidor. Hauria de semblar-se a això:

    import 'zone.js/dist/zone-node';
import { ngExpressEngine } from '@nguniversal/express-engine';
import * as express from 'express';
import { join } from 'path';

import { AppServerModule } from './src/main.server';
import { APP_BASE_HREF } from '@angular/common';
import { existsSync } from 'fs';

// L'app d'Express s'exporta perquè pugui ser utilitzada per Funcions sense servidor.
export function app() {
  const server = express();
  const distFolder = join(process.cwd(), 'dist/your-project-name/browser');
  const indexHtml = existsSync(join(distFolder, 'index.original.html')) ? 'index.original.html' : 'index';

  server.engine('html', ngExpressEngine({
    bootstrap: AppServerModule,
  }));

  server.set('view engine', 'html');
  server.set('views', distFolder);

  server.get('*.*', express.static(distFolder, {
    maxAge: '1y'
  }));

  server.get('*', (req, res) => {
    res.render(indexHtml, { req, providers: [{ provide: APP_BASE_HREF, useValue: req.baseUrl }] });
  });

  return server;
}

function run() {
  const port = process.env.PORT || 4000;

  const server = app();
  server.listen(port, () => {
    console.log(`Servidor Node Express escoltant a http://localhost:${port}`);
  });
}

declare const __non_webpack_require__: NodeRequire;

if (require.main === module) {
  run();
}

export * from './src/main.server';

  4. Actualitza main.server.ts: Assegura't que el teu fitxer main.server.ts inicialitzi el mòdul del servidor:

    import { enableProdMode } from '@angular/core';
import { environment } from './environments/environment';

if (environment.production) {
  enableProdMode();
}

export { AppServerModule } from './app/app.server.module';
export { renderModule, renderModuleFactory } from '@angular/platform-server';

Executar l'Aplicació

Per executar la teva aplicació Angular Universal, cal construir tant el codi del client com el del servidor i després servir-lo.

Construir i Servir

  1. Construeix el Client i el Servidor:

    npm run build:ssr

  2. Serveix l'Aplicació:

    npm run serve:ssr

Beneficis d'Angular Universal

  • Millora del SEO: Els motors de cerca poden indexar el teu lloc de manera més efectiva.
  • Càrrega Inicial més Ràpida: Els usuaris reben una pàgina completament renderitzada des del servidor, reduint el temps fins a la primera pintura significativa.
  • Millor Experiència d'Usuari: Els usuaris poden veure el contingut més ràpidament, fins i tot en xarxes lentes.

Conclusió

Angular Universal és una eina poderosa per millorar el rendiment i el SEO de les aplicacions Angular mitjançant el renderitzat del costat del servidor. Seguint els passos de configuració i entenent els conceptes clau, pots aprofitar Angular Universal per oferir una millor experiència d'usuari i millorar la visibilitat de la teva aplicació en els motors de cerca.
