Introducció als Formularis Reactius

Els Formularis Reactius a Angular proporcionen un enfocament basat en models per gestionar les entrades de formulari els valors dels quals canvien amb el temps. Aquest enfocament ofereix més control i flexibilitat en comparació amb els formularis basats en plantilles.

  • Enfocament basat en models: Els formularis es creen i es gestionen a la classe del component.
  • Estructures de dades immutables: Cada canvi a l'estat del formulari retorna un nou estat.
  • Accés sincrònic a les dades del formulari: Les dades del formulari estan disponibles immediatament.

Configuració dels Formularis Reactius

Per utilitzar els Formularis Reactius, cal importar el ReactiveFormsModule al vostre mòdul Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // altres imports
  ],
  // altres configuracions
})
export class AppModule { }

Creació d'un Formulari Reactiu Bàsic

Pas 1: Importar Classes de Formulari

Importeu les classes necessàries de @angular/forms.

import { FormGroup, FormControl } from '@angular/forms';

Pas 2: Definir el Model del Formulari

Creeu un model de formulari a la vostra classe de component.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }
}

Pas 3: Vincular el Formulari a la Plantilla

Vinculeu el model del formulari a la plantilla utilitzant directives d'Angular.

<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <label for="name">Nom:</label>
  <input id="name" formControlName="name">
  
  <label for="email">Correu electrònic:</label>
  <input id="email" formControlName="email">
  
  <button type="submit">Enviar</button>
</form>

Pas 4: Gestionar l'Enviament del Formulari

Gestioneu l'enviament del formulari a la vostra classe de component.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }

  onSubmit() {
    console.log(this.myForm.value);
  }
}

Controls de Formulari Avançats

FormBuilder

FormBuilder és un servei que proporciona mètodes convenients per crear controls de formulari.

import { FormBuilder, FormGroup } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: [''],
      email: [''],
    });
  }
}

FormArray

FormArray s'utilitza per gestionar una matriu de controls de formulari.

import { FormBuilder, FormArray } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      users: this.fb.array([]),
    });
  }

  get users() {
    return this.myForm.get('users') as FormArray;
  }

  addUser() {
    this.users.push(this.fb.group({
      name: [''],
      email: [''],
    }));
  }
}

FormGroups Imbricats

Podeu imbricar FormGroups per crear formularis complexos.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      userDetails: this.fb.group({
        name: [''],
        email: [''],
      }),
      address: this.fb.group({
        street: [''],
        city: [''],
      }),
    });
  }
}

Validació de Formularis

Validators Incorporats

Angular proporciona diversos validators incorporats.

import { Validators } from '@angular/forms;

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Validators Personalitzats

Podeu crear validators personalitzats per a una lògica de validació més complexa.

import { AbstractControl, ValidationErrors } from '@angular/forms';

export function forbiddenNameValidator(control: AbstractControl): ValidationErrors | null {
  const forbidden = /admin/.test(control.value);
  return forbidden ? { forbiddenName: { value: control.value } } : null;
}

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', [Validators.required, forbiddenNameValidator]],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Formularis Dinàmics

Els formularis dinàmics us permeten crear formularis basats en dades en temps d'execució.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({});
  }

  createForm(fields: any[]) {
    fields.forEach(field => {
      this.myForm.addControl(field.name, new FormControl(''));
    });
  }
}

Conclusió

Els Formularis Reactius a Angular proporcionen una manera robusta i flexible de gestionar les entrades de formulari i la validació. Aprofitant el poder de FormGroup, FormControl, FormArray i FormBuilder, podeu crear formularis complexos i dinàmics que són fàcils de mantenir i ampliar. Entendre aquests conceptes us ajudarà a construir aplicacions Angular més eficients i escalables.
