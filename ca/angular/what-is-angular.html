Introducció a Angular

Angular és una plataforma i un framework per construir aplicacions de client d'una sola pàgina utilitzant HTML i TypeScript. Desenvolupat i mantingut per Google, Angular proporciona un conjunt robust d'eines i característiques per construir aplicacions web dinàmiques i responsives.

Característiques Clau d'Angular

  • Arquitectura Basada en Components: Les aplicacions Angular es construeixen utilitzant components, que són els blocs bàsics de la interfície d'usuari.
  • Enllaç de Dades Bidireccional: Aquesta característica permet la sincronització automàtica de dades entre el model i la vista.
  • Injecció de Dependències: El sistema d'injecció de dependències integrat d'Angular ajuda a gestionar les dependències dels diferents components de manera eficient.
  • Directives: Aquests són marcadors especials en el DOM que indiquen a Angular que faci alguna cosa a un element del DOM (per exemple, mostrar/amagar elements, repetir elements).
  • Serveis i Injecció de Dependències: Els serveis en Angular s'utilitzen per compartir dades i lògica entre components. La Injecció de Dependències (DI) és un patró de disseny utilitzat per implementar IoC (Inversió de Control) per resoldre dependències.
  • Routing: El potent router d'Angular permet la navegació entre diferents vistes o components en una aplicació d'una sola pàgina.
  • RxJS i Programació Reactiva: Angular utilitza RxJS (Extensions Reactives per a JavaScript) per gestionar operacions asíncrones i programes basats en esdeveniments.

Estructura Bàsica d'una Aplicació Angular

Una aplicació Angular està estructurada en mòduls, components i serveis. Aquí tens un exemple senzill d'un component Angular:

Exemple: Component Bàsic d'Angular

// app.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `
    <h1>{{ title }}</h1>
    <p>Benvingut a Angular!</p>
  `,
  styles: [`
    h1 {
      color: blue;
    }
  `]
})
export class AppComponent {
  title = 'Hola Angular';
}

Explicació

  • Decorador de Component: El decorador @Component defineix les metadades per al component, incloent el selector, la plantilla i els estils.
  • Selector: El selector és l'etiqueta HTML que s'utilitzarà per inserir aquest component al DOM.
  • Plantilla: La template defineix l'estructura HTML del component.
  • Estils: L'array styles conté els estils CSS que s'apliquen a aquest component.
  • Classe: La classe AppComponent conté la lògica i les dades per al component. En aquest exemple, té una única propietat title.

Conclusió

Angular és un framework potent i complet per construir aplicacions web modernes. La seva arquitectura basada en components, l'enllaç de dades bidireccional i el sistema d'injecció de dependències el fan una elecció popular entre els desenvolupadors. Entenent l'estructura bàsica i les característiques d'Angular, pots començar a construir aplicacions web dinàmiques i responsives.

En les següents seccions d'aquest curs, aprofundirem en cadascuna d'aquestes característiques, proporcionant explicacions més detallades, exemples i exercicis per ajudar-te a dominar Angular des de nivells bàsics fins a avançats.
