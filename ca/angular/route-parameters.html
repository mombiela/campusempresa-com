En aquesta secció, explorarem els Paràmetres de Ruta en Angular, un concepte essencial per construir aplicacions web dinàmiques i navegables. Els paràmetres de ruta permeten passar dades a través de l'URL, permetent que la teva aplicació gestioni diferents entrades de dades sense problemes.

Introducció als Paràmetres de Ruta

Els paràmetres de ruta s'utilitzen per capturar valors de l'URL i utilitzar-los dins de la teva aplicació Angular. Són essencials per crear rutes dinàmiques que poden gestionar diverses entrades, com ara identificadors d'usuari, identificadors de producte o qualsevol altra dada dinàmica.

  • Rutes Dinàmiques: Rutes que canvien en funció dels paràmetres proporcionats a l'URL.
  • Rutes Estàtiques: Rutes que romanen constants i no canvien en funció dels paràmetres de l'URL.

Configuració dels Paràmetres de Ruta

Per configurar els paràmetres de ruta, cal definir-los a la configuració de la teva ruta. Això es fa utilitzant el mòdul Angular Router.

Definició dels Paràmetres de Ruta

Primer, definim una ruta amb paràmetres al fitxer app-routing.module.ts.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { ProductDetailComponent } from './product-detail/product-detail.component';

const routes: Routes = [
  { path: 'product/:id', component: ProductDetailComponent }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

En aquest exemple, la part :id del camí és un paràmetre de ruta. Capturarà el valor proporcionat a l'URL i el passarà al ProductDetailComponent.

Accés als Paràmetres de Ruta

Per accedir als paràmetres de ruta al teu component, pots utilitzar el servei ActivatedRoute.

import { Component, OnInit } from '@angular/core';
import { ActivatedRoute } from '@angular/router';

@Component({
  selector: 'app-product-detail',
  templateUrl: './product-detail.component.html',
  styleUrls: ['./product-detail.component.css']
})
export class ProductDetailComponent implements OnInit {
  productId: string;

  constructor(private route: ActivatedRoute) { }

  ngOnInit(): void {
    this.productId = this.route.snapshot.paramMap.get('id');
  }
}

En aquest exemple, el servei ActivatedRoute s'utilitza per accedir als paràmetres de ruta. El mètode paramMap.get('id') recupera el valor del paràmetre id de l'URL.

Ús dels Paràmetres de Ruta en Plantilles

Un cop tinguis el paràmetre de ruta al teu component, pots utilitzar-lo a la teva plantilla per mostrar contingut dinàmic.

<h1>Detalls del Producte</h1>
<p>ID del Producte: {{ productId }}</p>

Aquesta plantilla mostrarà l'identificador del producte capturat de l'URL.

Navegació amb Paràmetres de Ruta

Per navegar a una ruta amb paràmetres programàticament, pots utilitzar el servei Router.

import { Component } from '@angular/core';
import { Router } from '@angular/router';

@Component({
  selector: 'app-product-list',
  templateUrl: './product-list.component.html',
  styleUrls: ['./product-list.component.css']
})
export class ProductListComponent {
  constructor(private router: Router) { }

  viewProduct(productId: string): void {
    this.router.navigate(['/product', productId]);
  }
}

En aquest exemple, el mètode viewProduct navega al ProductDetailComponent amb l'identificador de producte especificat.

Resum

Els paràmetres de ruta són una característica poderosa en Angular que et permeten crear aplicacions web dinàmiques i navegables. Definint paràmetres de ruta a la teva configuració de rutes, accedint-hi als teus components i utilitzant-los a les teves plantilles, pots construir aplicacions que responen a diferents entrades sense problemes.

  • Defineix Paràmetres de Ruta: Utilitza la sintaxi :parameter a la teva configuració de rutes.
  • Accedeix als Paràmetres de Ruta: Utilitza el servei ActivatedRoute al teu component.
  • Utilitza'ls en Plantilles: Enllaça el valor del paràmetre a la teva plantilla per a contingut dinàmic.
  • Navega Programàticament: Utilitza el servei Router per navegar amb paràmetres.

Entendre i utilitzar els paràmetres de ruta millorarà la teva capacitat per crear aplicacions Angular robustes i fàcils d'usar.
