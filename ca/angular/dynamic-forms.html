Els formularis dinàmics en Angular et permeten crear formularis que poden canviar en funció de la interacció de l'usuari o altres condicions. Aquesta és una característica potent que pot fer que les teves aplicacions siguin més flexibles i fàcils d'usar.

Conceptes clau

  • Formularis reactius: Un enfocament basat en models per gestionar les entrades del formulari els valors dels quals canvien amb el temps.
  • FormBuilder: Un servei que proporciona mètodes convenients per crear controls de formulari.
  • FormGroup i FormControl: Classes que representen un grup de controls de formulari i controls de formulari individuals, respectivament.
  • FormArray: Una classe que representa una matriu de controls de formulari, útil per a formularis dinàmics.
  • Validators: Funcions que comproven la validesa dels controls del formulari.

Configuració dels formularis reactius

Per començar a utilitzar formularis reactius, cal importar els mòduls necessaris a la teva aplicació Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // altres imports
  ],
  // altres configuracions
})
export class AppModule { }

Creació d'un formulari reactiu bàsic

Creem un formulari senzill amb un camp de nom i un de correu electrònic.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-basic-form',
  template: `
    <form [formGroup]="basicForm" (ngSubmit)="onSubmit()">
      <label for="name">Nom:</label>
      <input id="name" formControlName="name">
      <div *ngIf="basicForm.get('name').invalid && basicForm.get('name').touched">
        El nom és obligatori.
      </div>

      <label for="email">Correu electrònic:</label>
      <input id="email" formControlName="email">
      <div *ngIf="basicForm.get('email').invalid && basicForm.get('email').touched">
        Introdueix un correu electrònic vàlid.
      </div>

      <button type="submit" [disabled]="basicForm.invalid">Enviar</button>
    </form>
  `
})
export class BasicFormComponent {
  basicForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.basicForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.basicForm.valid) {
      console.log(this.basicForm.value);
    }
  }
}

Afegint controls dinàmics

Per afegir controls dinàmics, pots utilitzar FormArray. Ampliem el nostre formulari per incloure una llista de números de telèfon.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, FormArray, Validators } from '@angular/forms';

@Component({
  selector: 'app-dynamic-form',
  template: `
    <form [formGroup]="dynamicForm" (ngSubmit)="onSubmit()">
      <label for="name">Nom:</label>
      <input id="name" formControlName="name">
      <div *ngIf="dynamicForm.get('name').invalid && dynamicForm.get('name').touched">
        El nom és obligatori.
      </div>

      <label for="email">Correu electrònic:</label>
      <input id="email" formControlName="email">
      <div *ngIf="dynamicForm.get('email').invalid && dynamicForm.get('email').touched">
        Introdueix un correu electrònic vàlid.
      </div>

      <div formArrayName="phones">
        <div *ngFor="let phone of phones.controls; let i=index">
          <label for="phone{{i}}">Telèfon {{i + 1}}:</label>
          <input [id]="'phone' + i" [formControlName]="i">
          <button type="button" (click)="removePhone(i)">Eliminar</button>
        </div>
      </div>
      <button type="button" (click)="addPhone()">Afegir telèfon</button>

      <button type="submit" [disabled]="dynamicForm.invalid">Enviar</button>
    </form>
  `
})
export class DynamicFormComponent {
  dynamicForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.dynamicForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]],
      phones: this.fb.array([])
    });
  }

  get phones() {
    return this.dynamicForm.get('phones') as FormArray;
  }

  addPhone() {
    this.phones.push(this.fb.control('', Validators.required));
  }

  removePhone(index: number) {
    this.phones.removeAt(index);
  }

  onSubmit() {
    if (this.dynamicForm.valid) {
      console.log(this.dynamicForm.value);
    }
  }
}

Tècniques avançades

Validació condicional

Pots afegir o eliminar validadors en funció de certes condicions.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-conditional-validation',
  template: `
    <form [formGroup]="conditionalForm" (ngSubmit)="onSubmit()">
      <label for="username">Nom d'usuari:</label>
      <input id="username" formControlName="username">
      <div *ngIf="conditionalForm.get('username').invalid && conditionalForm.get('username').touched">
        El nom d'usuari és obligatori.
      </div>

      <label for="password">Contrasenya:</label>
      <input id="password" formControlName="password" type="password">
      <div *ngIf="conditionalForm.get('password').invalid && conditionalForm.get('password').touched">
        La contrasenya és obligatòria.
      </div>

      <label for="confirmPassword">Confirma la contrasenya:</label>
      <input id="confirmPassword" formControlName="confirmPassword" type="password">
      <div *ngIf="conditionalForm.get('confirmPassword').invalid && conditionalForm.get('confirmPassword').touched">
        Les contrasenyes han de coincidir.
      </div>

      <button type="submit" [disabled]="conditionalForm.invalid">Enviar</button>
    </form>
  `
})
export class ConditionalValidationComponent {
  conditionalForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.conditionalForm = this.fb.group({
      username: ['', Validators.required],
      password: ['', Validators.required],
      confirmPassword: ['']
    });

    this.conditionalForm.get('confirmPassword').setValidators([
      Validators.required,
      this.passwordMatchValidator.bind(this)
    ]);
  }

  passwordMatchValidator(control: FormControl): { [key: string]: boolean } | null {
    if (this.conditionalForm) {
      return control.value === this.conditionalForm.get('password').value ? null : { mismatch: true };
    }
    return null;
  }

  onSubmit() {
    if (this.conditionalForm.valid) {
      console.log(this.conditionalForm.value);
    }
  }
}

Resum

Els formularis dinàmics en Angular proporcionen una manera potent de crear formularis flexibles i fàcils d'usar. Aprofitant els formularis reactius, FormBuilder, FormGroup, FormControl i FormArray, pots crear formularis que s'adaptin a les entrades de l'usuari i altres condicions. Les tècniques avançades com la validació condicional milloren encara més les capacitats dels teus formularis. Entendre i utilitzar aquests conceptes et permetrà construir formularis robustos i dinàmics a les teves aplicacions Angular.
