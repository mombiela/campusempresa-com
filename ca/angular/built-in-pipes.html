Introducció als Pipes

Els pipes a Angular són una característica potent que et permet transformar dades directament en les teves plantilles. Són funcions simples que accepten un valor d'entrada i retornen un valor transformat. Angular proporciona diversos pipes integrats que cobreixen necessitats comunes de transformació de dades.

Pipes integrats clau

Angular ve amb una varietat de pipes integrats que pots utilitzar directament. Aquí tens alguns dels més utilitzats:

  • DatePipe: Formata un valor de data segons les regles de la localització.
  • UpperCasePipe: Transforma text a majúscules.
  • LowerCasePipe: Transforma text a minúscules.
  • CurrencyPipe: Transforma un número en una cadena de moneda.
  • DecimalPipe: Transforma un número en una cadena amb un format decimal especificat.
  • PercentPipe: Transforma un número en una cadena de percentatge.
  • JsonPipe: Converteix un valor en una cadena JSON.
  • SlicePipe: Crea un nou array o cadena que conté un subconjunt (slice) dels elements.

Ús dels pipes integrats

DatePipe

El DatePipe s'utilitza per formatar valors de data. Pots especificar el format en què vols que es mostri la data.

<p>{{ today | date:'fullDate' }}</p>

En aquest exemple, today és un objecte de data, i fullDate és un format predefinit que mostra la data completa.

UpperCasePipe i LowerCasePipe

Aquests pipes s'utilitzen per transformar text a majúscules o minúscules.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

El CurrencyPipe s'utilitza per formatar números com a moneda.

<p>{{ price | currency:'USD' }}</p>

Aquí, price és un número, i 'USD' especifica el format de la moneda.

DecimalPipe

El DecimalPipe formata números amb un format decimal especificat.

<p>{{ 3.14159 | number:'1.2-2' }}</p>

En aquest exemple, 1.2-2 especifica que el número ha de tenir almenys una xifra abans del punt decimal, i entre dues i dues xifres després del punt decimal.

PercentPipe

El PercentPipe formata números com a percentatges.

<p>{{ 0.25 | percent }}</p>

JsonPipe

El JsonPipe converteix un valor en una cadena JSON, que és útil per depurar.

<p>{{ myObject | json }}</p>

SlicePipe

El SlicePipe crea un nou array o cadena que conté un subconjunt dels elements.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

| Pipe | Propòsit | Exemple d'ús | |---------------|----------------------------------------------|------------------------------------------| | DatePipe | Formata valors de data | {{ today | date:'shortDate' }} | | UpperCasePipe | Transforma text a majúscules | {{ 'hello' | uppercase }} | | LowerCasePipe | Transforma text a minúscules | {{ 'HELLO' | lowercase }} | | CurrencyPipe | Formata números com a moneda | {{ price | currency:'USD' }} | | DecimalPipe | Formata números amb decimals especificats | {{ 3.14159 | number:'1.2-2' }} | | PercentPipe | Formata números com a percentatges | {{ 0.25 | percent }} | | JsonPipe | Converteix un valor en una cadena JSON | {{ myObject | json }} | | SlicePipe | Crea un subconjunt d'un array o cadena | {{ 'Hello' | slice:0:2 }} |

Conclusió

Els pipes integrats a Angular proporcionen una manera convenient de transformar dades directament en les teves plantilles. Ajuden a mantenir el teu codi net i llegible gestionant tasques comunes de transformació de dades. Entendre i utilitzar aquests pipes pot millorar significativament les teves aplicacions Angular fent que les teves plantilles siguin més expressives i fàcils de mantenir.
