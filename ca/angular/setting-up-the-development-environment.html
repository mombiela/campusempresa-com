Configuració de l'Entorn de Desenvolupament
Arquitectura d'Angular ►

Configurar l'entorn de desenvolupament és el primer pas per començar a treballar amb Angular. Aquesta secció et guiarà a través de les eines i configuracions necessàries per posar en marxa els teus projectes Angular.

Prerequisits

Abans de configurar Angular, assegura't de tenir els següents prerequisits instal·lats a la teva màquina:

  • Node.js i npm: Angular requereix Node.js i npm (Node Package Manager) per gestionar dependències i executar scripts.
  • Editor de Text o IDE: Un bon editor de text o Entorn de Desenvolupament Integrat (IDE) com Visual Studio Code, WebStorm o Sublime Text.

Instal·lant Node.js i npm

  1. Descarrega i Instal·la Node.js:

    • Visita el lloc web oficial de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Verifica la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa els següents comandaments per comprovar les versions instal·lades: 
      node -v
npm -v
    • Hauries de veure els números de versió de Node.js i npm.

Instal·lant Angular CLI

L'Angular CLI (Interfície de Línia de Comandes) és una eina potent que t'ajuda a inicialitzar, desenvolupar, esbossar i mantenir aplicacions Angular.

  1. Instal·la Angular CLI Globalment:

    • Obre el teu terminal o símbol del sistema.
    • Executa el següent comandament per instal·lar Angular CLI globalment: 
      npm install -g @angular/cli
    • Aquest comandament instal·la l'Angular CLI globalment, fent-lo accessible des de qualsevol directori de la teva màquina.

  2. Verifica la Instal·lació:

    • Executa el següent comandament per comprovar la versió instal·lada de l'Angular CLI: 
      ng version
    • Hauries de veure els detalls de la versió de l'Angular CLI i les seves dependències.

Creant un Nou Projecte Angular

Un cop l'Angular CLI està instal·lat, pots crear un nou projecte Angular utilitzant un comandament senzill.

  1. Crea un Nou Projecte:

    • Navega al directori on vols crear el teu projecte.
    • Executa el següent comandament: 
      ng new my-angular-app
    • Substitueix my-angular-app pel nom desitjat del teu projecte.
    • El CLI et demanarà algunes opcions de configuració. Pots triar les opcions per defecte prement Enter.

  2. Navega al Directori del Projecte:

    • Després de crear el projecte, navega al directori del projecte: 
      cd my-angular-app

  3. Serveix l'Aplicació:

    • Executa el següent comandament per servir l'aplicació: 
      ng serve
    • Obre el teu navegador i navega a http://localhost:4200/. Hauries de veure la pàgina de benvinguda per defecte d'Angular.

Configurant l'Entorn de Desenvolupament

Per millorar la teva experiència de desenvolupament, considera configurar el teu editor de text o IDE amb les següents eines i extensions:

  • Extensions de Visual Studio Code:

    • Angular Language Service: Proporciona capacitats riques d'edició i navegació.
    • TSLint: Ajuda a identificar i corregir problemes de codi TypeScript.
    • Prettier: Formata automàticament el teu codi per a la consistència.

  • Plugins de WebStorm:

    • Angular i AngularJS: Proporciona suport per a projectes Angular.
    • TypeScript: Millora el desenvolupament de TypeScript.

Exemple de Configuració per a Visual Studio Code

  1. Instal·la Extensions:

    • Obre Visual Studio Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca i instal·la les següents extensions:
      • Angular Language Service
      • TSLint
      • Prettier

  2. Configura Prettier:

    • Crea un fitxer .prettierrc a l'arrel del teu projecte amb el següent contingut: 
      {
  "singleQuote": true,
  "trailingComma": "all"
}
    • Aquesta configuració assegura que s'utilitzin cometes simples i s'afegeixin comes finals on sigui vàlid en ES5 (objectes, arrays, etc.).

Conclusió

Configurar l'entorn de desenvolupament és un pas crucial per començar el teu viatge amb Angular. Assegurant-te que tens Node.js, npm i Angular CLI instal·lats, i configurant el teu editor de text o IDE amb extensions útils, crees un espai de treball productiu i eficient. Amb el teu entorn llest, ara pots passar a construir i desenvolupar aplicacions Angular amb facilitat.
