Introducció als Pipes

Els pipes a Angular són una manera de transformar dades en plantilles. Són funcions simples que accepten un valor d'entrada i retornen un valor transformat. Angular proporciona diversos pipes integrats, com DatePipe, UpperCasePipe, LowerCasePipe i CurrencyPipe.

Per què utilitzar pipes personalitzats?

Tot i que els pipes integrats cobreixen molts casos d'ús comuns, de vegades necessites transformar dades d'una manera específica per a la teva aplicació. Els pipes personalitzats et permeten crear lògica de transformació de dades reutilitzable que es pot aplicar directament a les teves plantilles.

Crear un pipe personalitzat

Crear un pipe personalitzat a Angular implica els següents passos:

  1. Generar un nou pipe utilitzant Angular CLI.
  2. Implementar la interfície PipeTransform.
  3. Definir la lògica de transformació.
  4. Utilitzar el pipe personalitzat a les teves plantilles.

Pas 1: Generar un nou pipe

Utilitza Angular CLI per generar un nou pipe:

ng generate pipe custom

Aquesta comanda crearà dos fitxers: custom.pipe.ts i custom.pipe.spec.ts.

Pas 2: Implementar la interfície `PipeTransform`

Obre el fitxer custom.pipe.ts i implementa la interfície PipeTransform:

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'custom'
})
export class CustomPipe implements PipeTransform {
  transform(value: any, ...args: any[]): any {
    // La lògica de transformació va aquí
    return value;
  }
}

Pas 3: Definir la lògica de transformació

Afegeix la teva lògica de transformació personalitzada dins del mètode transform. Per exemple, creem un pipe que converteix una cadena a majúscules de títol:

export class CustomPipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.replace(/\w\S*/g, (txt) => {
      return txt.charAt(0).toUpperCase() + txt.substr(1).toLowerCase();
    });
  }
}

Pas 4: Utilitzar el pipe personalitzat a les plantilles

Un cop definit el teu pipe personalitzat, pots utilitzar-lo a les teves plantilles com qualsevol altre pipe:

<p>{{ 'hello world' | custom }}</p>

Això mostrarà:

Hello World

Passar paràmetres als pipes personalitzats

Els pipes personalitzats també poden acceptar paràmetres. Per exemple, creem un pipe que formata un número a un nombre especificat de decimals:

@Pipe({
  name: 'decimal'
})
export class DecimalPipe implements PipeTransform {
  transform(value: number, decimalPlaces: number): string {
    return value.toFixed(decimalPlaces);
  }
}

Ús a la plantilla:

<p>{{ 123.4567 | decimal:2 }}</p>

Això mostrarà:

123.46

Pipes purs vs impurs

  • Pipes purs: Aquests pipes només es criden quan Angular detecta un canvi en el valor d'entrada. Són més eficients i s'han d'utilitzar en la majoria dels casos.
  • Pipes impurs: Aquests pipes es criden en cada cicle de detecció de canvis, cosa que pot afectar el rendiment. Utilitza'ls només quan sigui necessari.

Per crear un pipe impur, estableix la propietat pure a false en el decorador @Pipe:

@Pipe({
  name: 'impurePipe',
  pure: false
})
export class ImpurePipe implements PipeTransform {
  transform(value: any): any {
    // Lògica de transformació
    return value;
  }
}

Conclusió

Els pipes personalitzats a Angular són una manera poderosa d'encapsular i reutilitzar la lògica de transformació de dades a les teves plantilles. Seguint els passos descrits en aquest tema, pots crear els teus propis pipes personalitzats per satisfer les necessitats específiques de la teva aplicació. Recorda considerar les implicacions de rendiment quan decideixis entre pipes purs i impurs.
