Introducció a NgRx

NgRx és una potent biblioteca de gestió d'estat per a aplicacions Angular, inspirada en Redux. Ajuda a gestionar l'estat de la teva aplicació de manera previsible, fent que sigui més fàcil de depurar i provar.

Conceptes clau de NgRx

  • Store: Una estructura de dades única i immutable que conté l'estat de la teva aplicació.
  • Actions: Objectes simples que representen una intenció de canviar l'estat.
  • Reducers: Funcions pures que prenen l'estat actual i una acció, i retornen un nou estat.
  • Selectors: Funcions que seleccionen una part de l'estat.
  • Effects: Gestió d'efectes secundaris per a manejar operacions asíncrones.

Configuració de NgRx

Per començar amb NgRx, necessites instal·lar els paquets necessaris:

npm install @ngrx/store @ngrx/effects @ngrx/store-devtools

Configuració del Store

Crea un reductor arrel i configura el store al teu AppModule.

// app.reducer.ts
import { ActionReducerMap } from '@ngrx/store';
import { counterReducer } from './counter.reducer';

export interface AppState {
  counter: number;
}

export const reducers: ActionReducerMap<AppState> = {
  counter: counterReducer,
};
// app.module.ts
import { NgModule } from '@angular/core';
import { StoreModule } from '@ngrx/store';
import { reducers } from './app.reducer';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    StoreModule.forRoot(reducers),
  ],
  providers: [],
  bootstrap: [AppComponent],
})
export class AppModule {}

Accions

Les accions es despatxen per desencadenar canvis d'estat. Defineix accions utilitzant createAction.

// counter.actions.ts
import { createAction } from '@ngrx/store';

export const increment = createAction('[Counter] Increment');
export const decrement = createAction('[Counter] Decrement');
export const reset = createAction('[Counter] Reset');

Reductors

Els reductors gestionen les transicions d'estat basades en accions.

// counter.reducer.ts
import { createReducer, on } from '@ngrx/store';
import { increment, decrement, reset } from './counter.actions';

export const initialState = 0;

const _counterReducer = createReducer(
  initialState,
  on(increment, (state) => state + 1),
  on(decrement, (state) => state - 1),
  on(reset, () => 0)
);

export function counterReducer(state, action) {
  return _counterReducer(state, action);
}

Selectors

Els selectors s'utilitzen per consultar l'estat.

// counter.selectors.ts
import { createSelector, createFeatureSelector } from '@ngrx/store';
import { AppState } from './app.reducer';

export const selectCounter = createFeatureSelector<AppState, number>('counter');

export const selectCounterValue = createSelector(
  selectCounter,
  (counter) => counter
);

Effects

Els effects gestionen efectes secundaris com les crides a API.

// counter.effects.ts
import { Injectable } from '@angular/core';
import { Actions, createEffect, ofType } from '@ngrx/effects';
import { increment, decrement, reset } from './counter.actions';
import { map, mergeMap, catchError } from 'rxjs/operators';
import { of } from 'rxjs';

@Injectable()
export class CounterEffects {
  constructor(private actions$: Actions) {}

  increment$ = createEffect(() =>
    this.actions$.pipe(
      ofType(increment),
      mergeMap(() => {
        // Simula una crida a l'API
        return of({ type: '[Counter API] Increment Success' }).pipe(
          catchError(() => of({ type: '[Counter API] Increment Failure' }))
        );
      })
    )
  );
}

Registre d'Efectes

Registra els efectes al teu AppModule.

// app.module.ts
import { EffectsModule } from '@ngrx/effects';
import { CounterEffects } from './counter.effects';

@NgModule({
  imports: [
    EffectsModule.forRoot([CounterEffects]),
  ],
})
export class AppModule {}

Conclusió

NgRx proporciona un marc robust per gestionar l'estat en aplicacions Angular. Entenent i implementant stores, accions, reductors, selectors i efectes, pots crear aplicacions escalables i mantenibles. Practica construint petites aplicacions i integrant gradualment una lògica de gestió d'estat més complexa per dominar NgRx.
