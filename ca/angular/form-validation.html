Introducció a la Validació de Formularis

La validació de formularis és un aspecte crucial del desenvolupament web, assegurant que les dades introduïdes pels usuaris siguin correctes i utilitzables. Angular proporciona eines robustes tant per a formularis basats en plantilles com per a formularis reactius, facilitant la gestió eficient de la validació de formularis.

Formularis Basats en Plantilles

Els formularis basats en plantilles es basen en directives d'Angular per gestionar la validació de formularis. Aquests formularis són més senzills de configurar i són adequats per a casos d'ús més simples.

Conceptes Clau

  • ngModel: Enllaça els controls del formulari a propietats del component.
  • ngModelGroup: Agrupa múltiples controls del formulari.
  • Validators: Validadors integrats com required, minlength, maxlength, pattern.

Exemple

<form #userForm="ngForm">
  <div>
    <label for="name">Nom:</label>
    <input type="text" id="name" name="name" ngModel required minlength="3">
    <div *ngIf="userForm.controls.name?.invalid && userForm.controls.name?.touched">
      <small *ngIf="userForm.controls.name?.errors?.required">El nom és obligatori.</small>
      <small *ngIf="userForm.controls.name?.errors?.minlength">El nom ha de tenir almenys 3 caràcters.</small>
    </div>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Explicació

  • La directiva ngModel enllaça el camp d'entrada a la propietat name.
  • Els atributs required i minlength són validadors integrats.
  • La directiva *ngIf s'utilitza per mostrar missatges de validació de manera condicional.

Formularis Reactius

Els formularis reactius proporcionen més control i flexibilitat, fent-los adequats per a escenaris complexos. Es defineixen a la classe del component i s'enllacen a la plantilla.

Conceptes Clau

  • FormControl: Fa el seguiment del valor i l'estat de validació d'un control de formulari individual.
  • FormGroup: Fa el seguiment del mateix per a un grup de controls de formulari.
  • FormBuilder: Un servei per ajudar a crear controls i grups de formulari.

Exemple

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-user-form',
  template: `
    <form [formGroup]="userForm" (ngSubmit)="onSubmit()">
      <div>
        <label for="name">Nom:</label>
        <input id="name" formControlName="name">
        <div *ngIf="userForm.get('name').invalid && userForm.get('name').touched">
          <small *ngIf="userForm.get('name').errors.required">El nom és obligatori.</small>
          <small *ngIf="userForm.get('name').errors.minlength">El nom ha de tenir almenys 3 caràcters.</small>
        </div>
      </div>
      <button type="submit" [disabled]="userForm.invalid">Enviar</button>
    </form>
  `
})
export class UserFormComponent {
  userForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.userForm = this.fb.group({
      name: ['', [Validators.required, Validators.minLength(3)]]
    });
  }

  onSubmit() {
    if (this.userForm.valid) {
      console.log(this.userForm.value);
    }
  }
}

Explicació

  • El servei FormBuilder s'utilitza per crear el grup de formulari i els controls.
  • La matriu Validators inclou els validadors required i minlength.
  • La plantilla enllaça el grup de formulari al formulari utilitzant [formGroup] i els controls individuals utilitzant formControlName.

Validadors Personalitzats

Angular permet crear validadors personalitzats per a lògiques de validació més complexes.

Exemple

import { AbstractControl, ValidatorFn } from '@angular/forms';

export function forbiddenNameValidator(nameRe: RegExp): ValidatorFn {
  return (control: AbstractControl): { [key: string]: any } | null => {
    const forbidden = nameRe.test(control.value);
    return forbidden ? { 'forbiddenName': { value: control.value } } : null;
  };
}

Ús en Formulari Reactiu

this.userForm = this.fb.group({
  name: ['', [Validators.required, Validators.minLength(3), forbiddenNameValidator(/admin/i)]]
});

Explicació

  • La funció forbiddenNameValidator retorna una funció de validador que comprova si el valor del control coincideix amb un patró prohibit.
  • Aquest validador personalitzat s'afegeix a la matriu Validators en la definició del control del formulari.

Resum

La validació de formularis en Angular es pot gestionar utilitzant formularis basats en plantilles o formularis reactius, cadascun amb els seus propis avantatges. Els formularis basats en plantilles són més senzills i adequats per a casos d'ús bàsics, mentre que els formularis reactius ofereixen més control i flexibilitat per a escenaris complexos. Angular també admet validadors personalitzats, permetent implementar lògiques de validació específiques segons sigui necessari. Entendre aquests conceptes i eines et permetrà crear formularis robustos i fàcils d'usar en les teves aplicacions Angular.
