A Angular, els serveis són un concepte fonamental utilitzat per compartir dades, lògica i funcions entre diferents components d'una aplicació. Aquest tema et guiarà a través de la creació i ús de serveis, des de nivells bàsics fins a avançats.

Què és un Servei?

  • Definició: Un servei és una classe amb un propòsit focalitzat. Normalment s'utilitza per realitzar tasques com obtenir dades d'un servidor, validar l'entrada de l'usuari o registrar esdeveniments.
  • Propòsit: Els serveis ajuden a mantenir els teus components lleugers i centrats en les seves tasques principals, com mostrar dades i gestionar la interacció de l'usuari.

Creació d'un Servei Bàsic

Guia Pas a Pas

  1. Generar un Servei: Utilitza Angular CLI per generar un servei.

    ng generate service my-service

    Aquesta comanda crea dos fitxers: my-service.service.ts i my-service.service.spec.ts.

  2. Definir el Servei: Obre my-service.service.ts i defineix el servei.

    import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MyService {
  constructor() { }

  getData() {
    return 'Hola des de MyService!';
  }
}

Explicació

  • @Injectable({ providedIn: 'root' }): Aquest decorador fa que el servei estigui disponible a tota l'aplicació.
  • getData(): Un mètode senzill que retorna una cadena de text.

Ús d'un Servei en un Component

Guia Pas a Pas

  1. Injectar el Servei: Obre un fitxer de component (per exemple, app.component.ts) i injecta el servei.

    import { Component, OnInit } from '@angular/core';
import { MyService } from './my-service.service';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent implements OnInit {
  message: string;

  constructor(private myService: MyService) {}

  ngOnInit() {
    this.message = this.myService.getData();
  }
}

  2. Mostrar les Dades: Enllaça les dades en la plantilla del component (per exemple, app.component.html).

    <div>
  {{ message }}
</div>

Explicació

  • constructor(private myService: MyService): El servei s'injecta en el component.
  • ngOnInit(): El mètode getData() del servei es crida quan el component s'inicialitza.

Característiques Avançades del Servei

Injecció de Dependències (DI)

  • Injeccions Jeràrquiques: El sistema DI d'Angular permet que els serveis es proporcionin a diferents nivells (arrel, mòdul, component).
  • Exemple: Proporcionar un servei a nivell de component. 
    @Component({
  selector: 'app-child',
  templateUrl: './child.component.html',
  styleUrls: ['./child.component.css'],
  providers: [MyService]
})
export class ChildComponent {
  constructor(private myService: MyService) {}
}

Serveis HTTP

  • Obtenir Dades d'una API: Utilitza HttpClient d'Angular per obtenir dades d'un servidor. 
    import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  fetchData(): Observable<any> {
    return this.http.get<any>(this.apiUrl);
  }
}

Explicació

  • HttpClient: Servei integrat d'Angular per fer sol·licituds HTTP.
  • fetchData(): Mètode per obtenir dades de l'URL de l'API especificada.

Ús del Servei HTTP en un Component

  1. Injectar el Servei: Injecta el DataService en un component.

    import { Component, OnInit } from '@angular/core';
import { DataService } from './data.service';

@Component({
  selector: 'app-data',
  templateUrl: './data.component.html',
  styleUrls: ['./data.component.css']
})
export class DataComponent implements OnInit {
  data: any;

  constructor(private dataService: DataService) {}

  ngOnInit() {
    this.dataService.fetchData().subscribe(response => {
      this.data = response;
    });
  }
}

  2. Mostrar les Dades: Enllaça les dades obtingudes en la plantilla del component.

    <div *ngIf="data">
  <pre>{{ data | json }}</pre>
</div>

Explicació

  • fetchData().subscribe(response => { this.data = response; }): Subscriu a l'observable retornat per fetchData() i assigna la resposta a la propietat data.

Resum

  • Serveis: Essencials per compartir dades i lògica entre components.
  • Creació de Serveis: Utilitza Angular CLI i el decorador @Injectable.
  • Ús de Serveis: Injecta serveis en components i crida els seus mètodes.
  • Característiques Avançades: Injecció de dependències a diferents nivells, ús de HttpClient per a crides a API.

Dominant els serveis a Angular, pots crear aplicacions més modulars, mantenibles i testejables.
