Introducció a les Directives

Les directives són una característica central d'Angular que et permeten adjuntar comportament als elements del DOM. Poden canviar l'aparença o el comportament d'un element, component o una altra directiva. Angular proporciona diverses directives incorporades que pots utilitzar per gestionar formularis, llistes, estils i més.

Tipus de Directives Incorporades

Les directives incorporades d'Angular es poden categoritzar en tres tipus principals:

  • Directives d'Atribut: Canvien l'aparença o el comportament d'un element, component o una altra directiva.
  • Directives Estructurals: Canvien el disseny del DOM afegint o eliminant elements.
  • Directives de Component: Aquestes són directives amb una plantilla.

Directives d'Atribut

Les directives d'atribut canvien l'aparença o el comportament d'un element. Aquí hi ha algunes directives d'atribut comunament utilitzades:

`ngClass`

La directiva ngClass et permet establir dinàmicament classes CSS en un element.

Exemple:

<div [ngClass]="{'active': isActive, 'disabled': isDisabled}">
  Les classes d'aquest div canvien segons l'estat del component.
</div>

En l'exemple anterior, la classe active s'aplicarà si isActive és true, i la classe disabled s'aplicarà si isDisabled és true.

`ngStyle`

La directiva ngStyle et permet establir múltiples estils en línia de manera dinàmica.

Exemple:

<div [ngStyle]="{'color': textColor, 'font-size': fontSize + 'px'}">
  Els estils d'aquest div canvien segons l'estat del component.
</div>

En aquest exemple, els estils color i font-size s'estableixen dinàmicament segons les propietats textColor i fontSize del component.

Directives Estructurals

Les directives estructurals canvien el disseny del DOM afegint o eliminant elements. Aquí hi ha algunes directives estructurals comunament utilitzades:

`ngIf`

La directiva ngIf inclou condicionalment una plantilla segons el valor d'una expressió.

Exemple:

<div *ngIf="isVisible">
  Aquest div només és visible si isVisible és true.
</div>

En aquest exemple, el div només s'inclourà en el DOM si isVisible és true.

`ngFor`

La directiva ngFor s'utilitza per repetir una plantilla per a cada element d'una llista.

Exemple:

<ul>
  <li *ngFor="let item of items">
    {{ item }}
  </li>
</ul>

En aquest exemple, es crea un element <li> per a cada element de l'array items.

`ngSwitch`

La directiva ngSwitch s'utilitza per incloure condicionalment una de diverses plantilles possibles, segons el valor d'una expressió.

Exemple:

<div [ngSwitch]="viewMode">
  <div *ngSwitchCase="'map'">Vista de Mapa</div>
  <div *ngSwitchCase="'list'">Vista de Llista</div>
  <div *ngSwitchDefault>Vista per Defecte</div>
</div>

En aquest exemple, la plantilla que coincideix amb el valor de viewMode s'inclou en el DOM.

Directives de Component

Les directives de component són essencialment components d'Angular. Tenen una plantilla, estils i lògica encapsulada.

Exemple:

@Component({
  selector: 'app-hello',
  template: `<h1>Hello, {{name}}!</h1>`,
  styles: [`h1 { font-family: Lato; }`]
})
export class HelloComponent  {
  @Input() name: string;
}

En aquest exemple, HelloComponent és una directiva de component que mostra un missatge de salutació.

Conclusió

Les directives incorporades en Angular són eines poderoses que et permeten manipular el DOM de manera declarativa. Entendre com utilitzar directives d'atribut com ngClass i ngStyle, directives estructurals com ngIf, ngFor i ngSwitch, i directives de component et permetrà crear aplicacions web dinàmiques i responsives. Practica utilitzant aquestes directives en els teus projectes per obtenir una millor comprensió de les seves capacitats i matisos.
