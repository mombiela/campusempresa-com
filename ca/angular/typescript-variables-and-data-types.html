Introducció

TypeScript és un superconjunt de JavaScript amb tipatge estàtic que afegeix tipus opcionals, interfícies i altres característiques al llenguatge. Entendre les variables i els tipus de dades en TypeScript és crucial per desenvolupar aplicacions Angular robustes. Aquesta secció cobrirà els conceptes bàsics de la declaració de variables i l'ús de diferents tipus de dades en TypeScript.

Variables en TypeScript

Declaració de variables

En TypeScript, les variables es poden declarar utilitzant let, const i var. Cada una té el seu propi àmbit i ús.

  • let: Declaració de variables amb àmbit de bloc.
  • const: Declaració de constants amb àmbit de bloc.
  • var: Declaració de variables amb àmbit de funció (no recomanat per a TypeScript modern).

Exemples:

let message: string = "Hola, TypeScript!";
const pi: number = 3.14;
var globalVar: boolean = true;

Àmbit de les variables

  • Àmbit de bloc: Les variables declarades amb let i const estan limitades al bloc en què es defineixen.
  • Àmbit de funció: Les variables declarades amb var estan limitades a la funció en què es defineixen.

Exemple:

function exampleFunction() {
    if (true) {
        let blockScoped = "Sóc d'àmbit de bloc";
        var functionScoped = "Sóc d'àmbit de funció";
    }
    console.log(functionScoped); // Funciona
    // console.log(blockScoped); // Error: blockScoped no està definit
}

Tipus de dades en TypeScript

Tipus de dades bàsics

TypeScript suporta diversos tipus de dades bàsics, incloent:

  • string
  • number
  • boolean
  • any
  • void
  • null
  • undefined

Exemples:

let name: string = "John Doe";
let age: number = 30;
let isStudent: boolean = true;
let anything: any = "Podria ser qualsevol cosa";
let noReturnValue: void = undefined;
let empty: null = null;
let notAssigned: undefined = undefined;

Arrays i tuples

  • Arrays: Es poden declarar utilitzant la sintaxi type[] o Array<type>.
  • Tuples: Permeten expressar un array amb un nombre fix d'elements els tipus dels quals són coneguts.

Exemples:

let numbers: number[] = [1, 2, 3];
let strings: Array<string> = ["one", "two", "three"];
let tuple: [string, number] = ["John Doe", 30];

Enums

Els enums permeten definir un conjunt de constants amb nom. TypeScript proporciona enums numèrics i basats en cadenes.

Exemple:

enum Color {
    Red,
    Green,
    Blue
}
let color: Color = Color.Green;

Interfícies

Les interfícies defineixen l'estructura d'un objecte. S'utilitzen per a la comprovació de tipus i per assegurar que els objectes s'adhereixen a una forma específica.

Exemple:

interface Person {
    name: string;
    age: number;
    isStudent: boolean;
}

let person: Person = {
    name: "Jane Doe",
    age: 25,
    isStudent: false
};

Assertions de tipus

Les assertions de tipus són una manera de dir al compilador quin és el tipus d'una variable quan tu saps més sobre el tipus que TypeScript.

Exemple:

let someValue: any = "Això és una cadena";
let strLength: number = (someValue as string).length;

Conclusió

Entendre les variables i els tipus de dades en TypeScript és fonamental per desenvolupar aplicacions Angular. Dominant aquests conceptes, pots escriure codi més robust i amb tipatge segur. Aquesta secció ha cobert la declaració de variables, àmbits, tipus de dades bàsics, arrays, tuples, enums, interfícies i assertions de tipus. Practica aquests conceptes amb els exemples proporcionats per consolidar la teva comprensió.
