Què és TypeScript?

TypeScript és un superconjunt de JavaScript amb tipatge estàtic que es compila a JavaScript pla. Afegeix tipus estàtics opcionals, classes i interfícies a JavaScript, fent més fàcil escriure i mantenir aplicacions a gran escala.

  • Tipatge Estàtic: TypeScript et permet definir tipus per a variables, paràmetres de funcions i valors de retorn.
  • Superconjunt de JavaScript: Tot el codi JavaScript vàlid també és codi TypeScript vàlid.
  • Es compila a JavaScript: El codi TypeScript es transpila a JavaScript, que pot executar-se en qualsevol entorn que suporti JavaScript.

Per què utilitzar TypeScript en Angular?

Angular està construït amb TypeScript, i utilitzar TypeScript en els teus projectes Angular ofereix diversos beneficis:

  • Seguretat de Tipus: Ajuda a detectar errors en temps de compilació en lloc de temps d'execució.
  • Suport Millorat a l'IDE: Millor autocompleció, navegació i eines de refactorització.
  • Millora de la Llegibilitat del Codi: Codi més clar i mantenible amb tipus i interfícies.
  • Funcionalitats Modernes de JavaScript: Accés a les funcionalitats d'ES6/ES7 i més enllà.

Configuració de TypeScript

Per començar a utilitzar TypeScript, necessites instal·lar-lo globalment a la teva màquina i configurar un fitxer de configuració.

Instal·lant TypeScript

npm install -g typescript

Creant un tsconfig.json

El fitxer tsconfig.json especifica els fitxers arrel i les opcions del compilador necessàries per compilar el projecte.

{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src"]
}

Sintaxi Bàsica de TypeScript

Variables i Tipus

TypeScript et permet definir tipus per a variables.

let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];

Funcions

Pots especificar tipus per als paràmetres de funcions i els valors de retorn.

function add(x: number, y: number): number {
  return x + y;
}

Interfícies

Les interfícies defineixen l'estructura d'un objecte.

interface Person {
  firstName: string;
  lastName: string;
}

function greet(person: Person) {
  return "Hello, " + person.firstName + " " + person.lastName;
}

let user = { firstName: "John", lastName: "Doe" };
console.log(greet(user));

Classes

TypeScript suporta classes d'ES6 amb funcionalitats addicionals com modificadors d'accés.

class Animal {
  private name: string;

  constructor(name: string) {
    this.name = name;
  }

  public move(distanceInMeters: number): void {
    console.log(`${this.name} moved ${distanceInMeters}m.`);
  }
}

let dog = new Animal("Dog");
dog.move(10);

TypeScript en Angular

Angular CLI i TypeScript

Quan crees un nou projecte Angular utilitzant l'Angular CLI, TypeScript està configurat per defecte.

ng new my-angular-app

Utilitzant Tipus en Components Angular

A Angular, pots utilitzar TypeScript per definir els tipus per a les propietats i mètodes del component.

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title: string = 'my-angular-app';

  changeTitle(newTitle: string): void {
    this.title = newTitle;
  }
}

Conclusió

TypeScript és una eina poderosa que millora JavaScript afegint tipus estàtics, interfícies i funcionalitats modernes de JavaScript. És una part integral del desenvolupament d'Angular, proporcionant seguretat de tipus, millor eines i millora de la mantenibilitat del codi. Entenent els conceptes bàsics de TypeScript, pots escriure aplicacions Angular més robustes i escalables.
