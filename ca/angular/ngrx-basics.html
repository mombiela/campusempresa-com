NgRx avançat ►

NgRx és una biblioteca potent per gestionar l'estat en aplicacions Angular utilitzant el patró Redux. Aquesta secció t'introduirà als conceptes bàsics de NgRx, els seus beneficis i com implementar-lo en els teus projectes Angular.

Introducció a NgRx

NgRx és una biblioteca de gestió d'estat per a aplicacions Angular, inspirada en Redux. Ajuda a gestionar l'estat d'una aplicació de manera previsible, fent que sigui més fàcil de depurar i provar.

Conceptes clau

  • Estat: Representa l'estat de l'aplicació en un moment donat.
  • Accions: Objectes simples que descriuen què ha passat a l'aplicació.
  • Reductors: Funcions que prenen l'estat actual i una acció, i retornen un nou estat.
  • Selectors: Funcions que seleccionen una part de l'estat de la botiga.
  • Efectes: Efectes secundaris que interactuen amb serveis externs i despatxen accions.

Configuració de NgRx

Per començar a utilitzar NgRx en el teu projecte Angular, necessites instal·lar els paquets necessaris i configurar la botiga.

Instal·lació

Executa la següent comanda per instal·lar els paquets de NgRx:

npm install @ngrx/store @ngrx/effects @ngrx/entity @ngrx/store-devtools

Configuració de la botiga

Crea un nou fitxer app.reducer.ts per definir l'estat arrel i els reductors.

import { ActionReducerMap } from '@ngrx/store';
import { counterReducer } from './counter.reducer';

export interface AppState {
  counter: number;
}

export const reducers: ActionReducerMap<AppState> = {
  counter: counterReducer
};

En el teu app.module.ts, importa el StoreModule i proporciona els reductors.

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { StoreModule } from '@ngrx/store';
import { reducers } from './app.reducer';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    StoreModule.forRoot(reducers)
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Accions

Les accions es despatxen per senyalitzar que alguna cosa ha passat a l'aplicació.

Definició d'Accions

Crea un nou fitxer counter.actions.ts per definir accions per al comptador.

import { createAction } from '@ngrx/store';

export const increment = createAction('[Counter] Increment');
export const decrement = createAction('[Counter] Decrement');
export const reset = createAction('[Counter] Reset');

Reductors

Els reductors especifiquen com canvia l'estat de l'aplicació en resposta a les accions.

Creació d'un Reductor

Crea un nou fitxer counter.reducer.ts per definir el reductor del comptador.

import { createReducer, on } from '@ngrx/store';
import { increment, decrement, reset } from './counter.actions';

export const initialState = 0;

const _counterReducer = createReducer(
  initialState,
  on(increment, state => state + 1),
  on(decrement, state => state - 1),
  on(reset, state => 0)
);

export function counterReducer(state, action) {
  return _counterReducer(state, action);
}

Selectors

Els selectors s'utilitzen per consultar la botiga per obtenir peces específiques de l'estat.

Creació de Selectors

Crea un nou fitxer counter.selectors.ts per definir selectors per al comptador.

import { createSelector, createFeatureSelector } from '@ngrx/store';
import { AppState } from './app.reducer';

export const selectCounter = createFeatureSelector<AppState, number>('counter');

Efectes

Els efectes gestionen efectes secundaris com trucades a API i despatxen noves accions basades en els resultats.

Creació d'Efectes

Crea un nou fitxer counter.effects.ts per definir efectes per al comptador.

import { Injectable } from '@angular/core';
import { Actions, createEffect, ofType } from '@ngrx/effects';
import { of } from 'rxjs';
import { map, mergeMap, catchError } from 'rxjs/operators';
import { increment, decrement, reset } from './counter.actions';

@Injectable()
export class CounterEffects {
  constructor(private actions$: Actions) {}

  increment$ = createEffect(() =>
    this.actions$.pipe(
      ofType(increment),
      mergeMap(() => {
        // Simular una trucada a l'API
        return of({}).pipe(
          map(() => increment()),
          catchError(() => of(reset()))
        );
      })
    )
  );
}

Conclusió

NgRx proporciona un marc robust per gestionar l'estat en aplicacions Angular. Entenent i implementant accions, reductors, selectors i efectes, pots crear aplicacions escalables i mantenibles. Practica creant una aplicació de comptador simple i afegeix gradualment més complexitat a mesura que et familiaritzis amb els conceptes.
