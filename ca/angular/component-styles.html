A Angular, els estils de components són un aspecte crucial per crear aplicacions visualment atractives i mantenibles. Aquest tema cobrirà els conceptes bàsics d'estilitzar components Angular, les diferents maneres d'aplicar estils i tècniques avançades per gestionar estils en aplicacions grans.

Introducció als Estils de Components

Angular et permet definir estils directament dins dels teus components, fent més fàcil encapsular i gestionar l'aparença de cada part de la teva aplicació.

Conceptes Clau

  • Encapsulació: Angular proporciona diferents estratègies d'encapsulació per controlar com s'apliquen els estils als components.
  • Estils Inline: Els estils es poden definir directament dins del fitxer TypeScript del component.
  • Fulls d'Estil Externs: Els estils es poden enllaçar des de fitxers CSS externs.
  • Estils Escopats: Els estils definits en un component estan escopats a aquest component per defecte, evitant que es filtrin a altres parts de l'aplicació.

Definint Estils en Components Angular

Estils Inline

Pots definir estils directament dins del fitxer TypeScript del component utilitzant la propietat styles.

import { Component } from '@angular/core';

@Component({
  selector: 'app-inline-style',
  template: `<h1>Hello, Inline Styles!</h1>`,
  styles: [`
    h1 {
      color: blue;
    }
  `]
})
export class InlineStyleComponent {}

Fulls d'Estil Externs

Per una millor mantenibilitat, pots definir estils en fitxers CSS externs i enllaçar-los utilitzant la propietat styleUrls.

import { Component } from '@angular/core';

@Component({
  selector: 'app-external-style',
  template: `<h1>Hello, External Styles!</h1>`,
  styleUrls: ['./external-style.component.css']
})
export class ExternalStyleComponent {}

external-style.component.css:

h1 {
  color: green;
}

Encapsulació d'Estil

Angular proporciona tres estratègies d'encapsulació per controlar com s'apliquen els estils als components:

  • Emulated (per defecte): Els estils estan escopats al component utilitzant un selector d'atribut únic.
  • None: Els estils s'apliquen globalment.
  • Shadow DOM: Els estils estan escopats al component utilitzant el Shadow DOM.

Exemple d'Encapsulació

import { Component, ViewEncapsulation } from '@angular/core';

@Component({
  selector: 'app-encapsulation',
  template: `<h1>Hello, Encapsulation!</h1>`,
  styles: [`
    h1 {
      color: red;
    }
  `],
  encapsulation: ViewEncapsulation.Emulated // Per defecte
})
export class EncapsulationComponent {}

Tècniques Avançades d'Estilització

Selectors Host i Host Context

  • :host: Apunta a l'element host del component.
  • :host-context: Apunta a l'element host basat en una condició en l'arbre d'ancestres.
import { Component } from '@angular/core';

@Component({
  selector: 'app-host-selector',
  template: `<h1>Hello, Host Selector!</h1>`,
  styles: [`
    :host {
      display: block;
      border: 1px solid black;
    }
    :host-context(.dark-theme) h1 {
      color: white;
    }
  `]
})
export class HostSelectorComponent {}

Comparació de Modes d'Encapsulació de Vista

| Mode | Descripció | Cas d'Ús | |---------------|--------------------------------------------------|----------------------------------------------| | Emulated | Els estils estan escopats al component utilitzant atributs | Per defecte, el cas d'ús més comú | | None | Els estils s'apliquen globalment | Quan vols que els estils afectin tota l'aplicació | | Shadow DOM | Els estils estan escopats utilitzant Shadow DOM | Quan s'utilitzen components web o aïllament estricte |

Conclusió

Entendre i utilitzar efectivament els estils de components a Angular és essencial per crear aplicacions mantenibles i visualment atractives. Utilitzant diferents tècniques d'estilització i estratègies d'encapsulació, pots assegurar-te que els teus estils estan ben organitzats i no interfereixen entre ells.

En aquest tema, hem cobert:

  • Definir estils utilitzant estils inline i fulls d'estil externs.
  • Utilitzar diferents estratègies d'encapsulació per controlar l'abast dels estils.
  • Tècniques avançades com selectors host i host-context.

Amb aquestes eines, pots crear estils robustos i mantenibles per les teves aplicacions Angular.
