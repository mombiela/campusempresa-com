Introducció

En aquesta secció, et guiarem a través de la creació de la teva primera aplicació Angular. Cobrirem els conceptes bàsics de la configuració de l'entorn Angular, la creació d'un nou projecte i la comprensió de l'estructura del projecte. Al final d'aquesta secció, tindràs una aplicació Angular simple funcionant a la teva màquina local.

Configuració de l'Entorn Angular

Abans de poder crear una aplicació Angular, necessites configurar el teu entorn de desenvolupament.

Prerequisits

  • Node.js i npm: Angular requereix Node.js i npm (Node Package Manager). Pots descarregar i instal·lar-los des de nodejs.org.
  • Angular CLI: La Interfície de Línia de Comandes d'Angular (CLI) és una eina poderosa per inicialitzar, desenvolupar, esbossar i mantenir aplicacions Angular.

Instal·lació de l'Angular CLI

Per instal·lar l'Angular CLI, obre el teu terminal o símbol del sistema i executa la següent comanda:

npm install -g @angular/cli

Aquesta comanda instal·la l'Angular CLI globalment al teu sistema.

Creació d'un Nou Projecte Angular

Un cop l'Angular CLI està instal·lat, pots crear un nou projecte Angular.

Guia Pas a Pas

  1. Obre el teu terminal o símbol del sistema.

  2. Navega al directori on vols crear el teu projecte.

  3. Executa la següent comanda per crear un nou projecte Angular:

    ng new my-first-angular-app
    • Se't demanarà si vols afegir el routing d'Angular. Escriu y per sí o n per no.
    • També se't demanarà que seleccionis un format de full d'estils (CSS, SCSS, SASS, LESS, Stylus). Tria la teva opció preferida.

  4. Navega al directori del teu projecte:

    cd my-first-angular-app

  5. Serveix l'aplicació:

    ng serve
    • Obre el teu navegador i navega a http://localhost:4200/. Hauries de veure la pàgina de benvinguda predeterminada d'Angular.

Comprensió de l'Estructura del Projecte

Quan crees un nou projecte Angular, l'Angular CLI genera una estructura de projecte per a tu. Aquests són els components clau:

  • e2e/: Proves de punta a punta per a l'aplicació.
  • node_modules/: Conté els paquets npm instal·lats per al projecte.
  • src/: El codi font de l'aplicació.
    • app/: Conté el mòdul principal de l'aplicació i el component.
    • assets/: Conté actius estàtics com imatges i estils.
    • environments/: Conté fitxers de configuració específics per a l'entorn.
  • angular.json: Fitxer de configuració per a l'Angular CLI.
  • package.json: Llista els paquets npm dels quals depèn el projecte.
  • tsconfig.json: Fitxer de configuració de TypeScript.

Creació d'un Component Simple

Els components són els blocs de construcció d'una aplicació Angular. Creem un component simple.

Guia Pas a Pas

  1. Genera un nou component:

    ng generate component hello-world

    Aquesta comanda crea un nou directori hello-world dins del directori app amb quatre fitxers:

    • hello-world.component.ts: El fitxer TypeScript per a la lògica del component.
    • hello-world.component.html: La plantilla HTML per al component.
    • hello-world.component.css: El fitxer CSS per als estils del component.
    • hello-world.component.spec.ts: El fitxer de proves per al component.

  2. Modifica la plantilla del component: Obre hello-world.component.html i substitueix el seu contingut per:

    <h1>Hello, World!</h1>

  3. Utilitza el component en l'aplicació principal: Obre app.component.html i afegeix la següent línia:

    <app-hello-world></app-hello-world>

    Això renderitzarà el missatge Hello, World! a la teva aplicació principal.

  4. Serveix l'aplicació:

    ng serve

    Obre el teu navegador i navega a http://localhost:4200/. Hauries de veure el missatge Hello, World!.

Conclusió

Felicitats! Has creat amb èxit la teva primera aplicació Angular. Has après com configurar l'entorn Angular, crear un nou projecte, entendre l'estructura del projecte i crear un component simple. Aquests són els passos fonamentals per construir aplicacions Angular més complexes. A les següents seccions, aprofundirem en les característiques d'Angular i les millors pràctiques.
