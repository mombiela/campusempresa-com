Introducció a les plantilles de components

Les plantilles de components a Angular defineixen la vista per a un component. Estan escrites en HTML i poden incloure sintaxi específica d'Angular per vincular dades i gestionar interaccions amb l'usuari.

  • Component: Un bloc de construcció d'aplicacions Angular, que consisteix en una classe TypeScript, una plantilla HTML i estils CSS opcionals.
  • Plantilla: La part HTML d'un component que defineix el que veu l'usuari.

Sintaxi bàsica de plantilles

Les plantilles Angular utilitzen una combinació d'HTML estàndard i sintaxi de plantilla Angular per crear vistes dinàmiques.

Interpolació

La interpolació permet incrustar expressions en el teu HTML.

<p>{{ title }}</p>

En aquest exemple, {{ title }} serà reemplaçat pel valor de la propietat title de la classe del component.

Vinculació de propietats

La vinculació de propietats permet establir propietats d'elements HTML o directives.

<img [src]="imageUrl">

Aquí, [src] vincula l'atribut src de l'element img a la propietat imageUrl del component.

Vinculació d'esdeveniments

La vinculació d'esdeveniments permet escoltar i respondre a accions de l'usuari.

<button (click)="onClick()">Click me</button>

En aquest exemple, (click) vincula l'esdeveniment de clic del botó al mètode onClick del component.

Directives estructurals

Les directives estructurals alteren el disseny afegint o eliminant elements del DOM.

*ngIf

La directiva *ngIf inclou condicionalment una plantilla basada en el valor d'una expressió.

<p *ngIf="isVisible">This paragraph is visible.</p>

El paràgraf només es mostrarà si isVisible és true.

*ngFor

La directiva *ngFor repeteix una plantilla per a cada element d'una llista.

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

Això crearà un element de llista per a cada element de l'array items.

Directives d'atributs

Les directives d'atributs canvien l'aparença o el comportament d'un element.

ngClass

La directiva ngClass afegeix i elimina classes CSS.

<div [ngClass]="{ 'active': isActive }">Content</div>

La classe active s'afegirà al div si isActive és true.

ngStyle

La directiva ngStyle afegeix i elimina estils en línia.

<div [ngStyle]="{ 'color': textColor }">Styled text</div>

L'estil color es configurarà amb el valor de textColor.

Variables de referència de plantilla

Les variables de referència de plantilla permeten referenciar un element del DOM dins d'una plantilla.

<input #inputElement type="text">
<button (click)="logValue(inputElement.value)">Log Value</button>

En aquest exemple, #inputElement és una variable de referència de plantilla que fa referència a l'element input. El mètode logValue pot accedir al valor de l'input.

Conclusió

Les plantilles de components són una part fonamental d'Angular, permetent-te crear interfícies d'usuari dinàmiques i interactives. Entenent i utilitzant la interpolació, la vinculació de propietats, la vinculació d'esdeveniments, les directives estructurals, les directives d'atributs i les variables de referència de plantilla, pots construir aplicacions Angular potents i responsives.
