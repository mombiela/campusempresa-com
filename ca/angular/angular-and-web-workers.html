Introducció als Web Workers

Els Web Workers proporcionen una manera d'executar scripts en fils de fons. Poden executar tasques sense interferir amb la interfície d'usuari, fent-los ideals per manejar operacions computacionalment intensives.

Conceptes Clau

  • Web Workers: Scripts de fons que s'executen independentment del fil principal de la UI.
  • Fil Principal: El fil d'execució principal on es maneja la UI i les interaccions de l'usuari.
  • Comunicació: Els Web Workers es comuniquen amb el fil principal utilitzant missatges.

Configuració dels Web Workers en Angular

Angular proporciona una manera convenient d'integrar Web Workers a la teva aplicació. Aquesta secció et guiarà a través del procés de configuració.

Passos per Crear un Web Worker en Angular

  1. Generar un Web Worker: Utilitza Angular CLI per generar un Web Worker. 
    ng generate web-worker worker-name
  2. Modificar la Configuració d'Angular: Assegura't que el teu fitxer angular.json estigui configurat per suportar Web Workers. 
    "architect": {
  "build": {
    "options": {
      "webWorkerTsConfig": "tsconfig.worker.json"
    }
  }
}
  3. Implementar la Lògica del Worker: Edita el fitxer generat del worker (per exemple, src/app/worker-name.worker.ts) per incloure la lògica que vols executar en segon pla. 
    addEventListener('message', ({ data }) => {
  const response = `Worker response to ${data}`;
  postMessage(response);
});

Exemple: Utilitzant un Web Worker per a Càlculs Pesats

Creem un Web Worker per realitzar un càlcul pesat, com calcular nombres de Fibonacci.

Fitxer del Worker: src/app/fibonacci.worker.ts

addEventListener('message', ({ data }) => {
  const result = fibonacci(data);
  postMessage(result);
});

function fibonacci(num: number): number {
  if (num <= 1) return num;
  return fibonacci(num - 1) + fibonacci(num - 2);
}

Fil Principal: src/app/app.component.ts

import { Component, OnInit } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `
    <div>
      <h1>Calculadora de Fibonacci</h1>
      <input type="number" [(ngModel)]="inputValue" />
      <button (click)="calculateFibonacci()">Calcular</button>
      <p>Resultat: {{ result }}</p>
    </div>
  `,
  styles: []
})
export class AppComponent implements OnInit {
  inputValue: number;
  result: number;
  worker: Worker;

  ngOnInit() {
    if (typeof Worker !== 'undefined') {
      this.worker = new Worker(new URL('./fibonacci.worker', import.meta.url));
      this.worker.onmessage = ({ data }) => {
        this.result = data;
      };
    } else {
      console.warn('Els Web Workers no són suportats en aquest entorn.');
    }
  }

  calculateFibonacci() {
    if (this.worker) {
      this.worker.postMessage(this.inputValue);
    }
  }
}

Millors Pràctiques per Utilitzar Web Workers en Angular

  • Descarregar Càlculs Pesats: Utilitza Web Workers per a tasques que són computacionalment intensives per mantenir la UI responsiva.
  • Limitar la Transferència de Dades: Minimitza la quantitat de dades enviades entre el fil principal i els Web Workers per reduir la sobrecàrrega.
  • Gestió d'Errors: Implementa una gestió d'errors adequada tant en el fil principal com en el worker per gestionar problemes inesperats.

Conclusió

Els Web Workers són una característica poderosa per millorar el rendiment de les aplicacions Angular descarregant càlculs pesats a fils de fons. Seguint els passos de configuració i les millors pràctiques descrites en aquest tema, pots integrar efectivament els Web Workers en els teus projectes Angular, assegurant una experiència d'usuari fluida i responsiva.
