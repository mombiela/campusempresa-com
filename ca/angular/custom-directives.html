Introducció a les directives

Les directives a Angular són classes que afegeixen comportament addicional als elements de les teves aplicacions Angular. Amb directives, pots gestionar formularis, llistes, estils i el que els usuaris veuen. Hi ha tres tipus de directives a Angular:

  • Directives de component: Aquestes són les directives més comunes i s'utilitzen per crear components.
  • Directives d'atribut: Aquestes canvien l'aparença o el comportament d'un element, component o una altra directiva.
  • Directives estructurals: Aquestes canvien el disseny del DOM afegint i eliminant elements del DOM.

En aquest tema, ens centrarem en crear directives d'atribut i estructurals personalitzades.

Creació de directives d'atribut personalitzades

Les directives d'atribut s'utilitzen per canviar l'aparença o el comportament dels elements del DOM. Creem una simple directiva d'atribut que canvia el color de fons d'un element quan es passa el ratolí per sobre.

Guia pas a pas

  1. Generar la directiva: Utilitza Angular CLI per generar una nova directiva.

    ng generate directive highlight

  2. Implementar la directiva: Modifica la directiva generada per canviar el color de fons en passar el ratolí per sobre.

    import { Directive, ElementRef, HostListener, Input } from '@angular/core';

@Directive({
  selector: '[appHighlight]'
})
export class HighlightDirective {
  @Input() defaultColor: string = 'yellow';
  @Input('appHighlight') highlightColor: string;

  constructor(private el: ElementRef) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.highlight(this.highlightColor || this.defaultColor);
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.highlight(null);
  }

  private highlight(color: string) {
    this.el.nativeElement.style.backgroundColor = color;
  }
}

  3. Utilitzar la directiva en un component: Aplica la directiva a un element en la teva plantilla.

    <p appHighlight="lightblue">Passa el ratolí per sobre d'aquest text per veure un efecte de ressaltat.</p>

Explicació

  • ElementRef: Un servei que proporciona accés directe a l'element del DOM.
  • @HostListener: Decorador que escolta esdeveniments del DOM i desencadena el mètode especificat.
  • @Input: Decorador que permet que la directiva accepti valors d'entrada.

Creació de directives estructurals personalitzades

Les directives estructurals canvien el disseny del DOM afegint o eliminant elements. Creem una directiva estructural personalitzada que inclou condicionalment una plantilla basada en un valor booleà.

Guia pas a pas

  1. Generar la directiva: Utilitza Angular CLI per generar una nova directiva.

    ng generate directive unless

  2. Implementar la directiva: Modifica la directiva generada per incloure condicionalment una plantilla.

    import { Directive, Input, TemplateRef, ViewContainerRef } from '@angular/core';

@Directive({
  selector: '[appUnless]'
})
export class UnlessDirective {
  private hasView = false;

  constructor(
    private templateRef: TemplateRef<any>,
    private viewContainer: ViewContainerRef
  ) {}

  @Input() set appUnless(condition: boolean) {
    if (!condition && !this.hasView) {
      this.viewContainer.createEmbeddedView(this.templateRef);
      this.hasView = true;
    } else if (condition && this.hasView) {
      this.viewContainer.clear();
      this.hasView = false;
    }
  }
}

  3. Utilitzar la directiva en un component: Aplica la directiva a un element en la teva plantilla.

    <div *appUnless="isVisible">Aquest text es mostra només quan isVisible és fals.</div>

Explicació

  • TemplateRef: Representa una plantilla incrustada que es pot utilitzar per instanciar vistes incrustades.
  • ViewContainerRef: Representa un contenidor on es poden adjuntar una o més vistes.
  • @Input: Decorador que permet que la directiva accepti valors d'entrada.

Resum

En aquest tema, hem cobert com crear directives d'atribut i estructurals personalitzades a Angular. Les directives d'atribut s'utilitzen per canviar l'aparença o el comportament dels elements, mentre que les directives estructurals s'utilitzen per canviar el disseny del DOM afegint o eliminant elements. Entenent i utilitzant aquestes directives, pots crear aplicacions Angular més dinàmiques i interactives.

Punts clau

  • Les directives són classes que afegeixen comportament als elements en aplicacions Angular.
  • Les directives d'atribut canvien l'aparença o el comportament dels elements.
  • Les directives estructurals canvien el disseny del DOM afegint o eliminant elements.
  • Utilitza Angular CLI per generar directives i modificar-les segons les teves necessitats.
  • Decoradors com @Input i @HostListener són essencials per crear directives personalitzades.

Dominant les directives personalitzades, pots millorar significativament la funcionalitat i l'experiència d'usuari de les teves aplicacions Angular.
