Introducció

En aquesta secció, cobrirem els passos essencials i les millors pràctiques per construir i desplegar aplicacions Angular. Això inclou configurar l'entorn, construir l'aplicació i desplegar-la en diverses plataformes.

Configuració de l'Entorn

Abans de poder construir i desplegar una aplicació Angular, necessitem configurar el nostre entorn de desenvolupament.

Prerequisits

  • Node.js i npm: Assegureu-vos que teniu Node.js i npm instal·lats. Podeu descarregar-los de nodejs.org.
  • Angular CLI: Instal·leu l'Angular CLI globalment utilitzant npm. 
    npm install -g @angular/cli

Creació d'un Nou Projecte Angular

Utilitzeu l'Angular CLI per crear un nou projecte.

ng new my-angular-app
  • Seguiu les indicacions per configurar el vostre projecte. Podeu triar el routing i el format d'estil (CSS, SCSS, etc.).

Executar el Servidor de Desenvolupament

Navegueu al directori del vostre projecte i inicieu el servidor de desenvolupament.

cd my-angular-app
ng serve
  • Obriu el vostre navegador i aneu a http://localhost:4200 per veure la vostra aplicació en funcionament.

Construcció de l'Aplicació

Un cop la vostra aplicació estigui llesta, necessiteu construir-la per a producció.

Construcció per a Producció

Utilitzeu l'Angular CLI per crear una construcció de producció.

ng build --prod
  • Aquesta comanda crearà una construcció optimitzada de la vostra aplicació al directori dist/.

Configuració de la Construcció

Podeu personalitzar el procés de construcció utilitzant el fitxer angular.json. Aquí hi ha algunes configuracions comunes:

  • Output Path: Canvieu el directori de sortida.
  • File Replacements: Reemplaceu fitxers d'entorn per a diferents configuracions (per exemple, desenvolupament vs. producció).
  • Optimization: Activeu o desactiveu optimitzacions com la minificació i el tree-shaking.

Desplegament de l'Aplicació

Hi ha diverses maneres de desplegar una aplicació Angular. Cobrirem el desplegament en plataformes populars com GitHub Pages, Firebase Hosting i AWS S3.

Desplegament a GitHub Pages

  1. Instal·leu el desplegador de GitHub Pages de l'Angular CLI. 
    npm install -g angular-cli-ghpages
  2. Construïu el vostre projecte. 
    ng build --prod --base-href "https://<username>.github.io/<repository>/"
  3. Desplegueu a GitHub Pages. 
    ngh --dir=dist/<project-name>

Desplegament a Firebase Hosting

  1. Instal·leu el Firebase CLI. 
    npm install -g firebase-tools
  2. Inicieu sessió a Firebase. 
    firebase login
  3. Inicialitzeu Firebase en el vostre projecte. 
    firebase init
    • Trieu Hosting i seguiu les indicacions.
  4. Construïu el vostre projecte. 
    ng build --prod
  5. Desplegueu a Firebase. 
    firebase deploy

Desplegament a AWS S3

  1. Instal·leu l'AWS CLI i configureu-lo amb les vostres credencials.
  2. Construïu el vostre projecte. 
    ng build --prod
  3. Sincronitzeu la sortida de la construcció amb el vostre bucket S3. 
    aws s3 sync dist/<project-name> s3://<your-bucket-name> --acl public-read

Conclusió

Construir i desplegar aplicacions Angular implica configurar l'entorn de desenvolupament, crear una construcció de producció i desplegar-la en una plataforma d'allotjament. Seguint els passos descrits en aquesta secció, podeu assegurar-vos que la vostra aplicació Angular està optimitzada i accessible per als usuaris. Tant si trieu desplegar a GitHub Pages, Firebase Hosting o AWS S3, l'Angular CLI proporciona les eines necessàries per agilitzar el procés.
