La gestió d'estat és un aspecte crucial per construir aplicacions robustes i escalables en Angular. Aquest tema et guiarà a través dels conceptes fonamentals de la gestió d'estat, la seva importància i com implementar-la de manera efectiva en aplicacions Angular.

Què és la Gestió d'Estat?

La gestió d'estat es refereix a la pràctica de gestionar l'estat d'una aplicació. L'estat inclou totes les dades que l'aplicació necessita per funcionar correctament, com les entrades de l'usuari, la configuració de l'aplicació i les dades obtingudes d'APIs.

Conceptes Clau:

  • Estat: L'estat actual de l'aplicació, incloent totes les dades i elements de la interfície d'usuari.
  • Store: Un lloc centralitzat per gestionar l'estat.
  • Accions: Esdeveniments que desencadenen canvis d'estat.
  • Reducers: Funcions que determinen com canvia l'estat en resposta a les accions.
  • Selectors: Funcions que extreuen peces específiques de l'estat.

Per què és Important la Gestió d'Estat?

La gestió d'estat és essencial per a:

  • Consistència: Assegura que l'aplicació es comporti de manera previsible.
  • Escalabilitat: Facilita la gestió d'aplicacions complexes.
  • Mantenibilitat: Simplifica el depurament i les proves.

Gestió d'Estat en Angular

Angular proporciona diverses maneres de gestionar l'estat, incloent serveis, RxJS i biblioteques de tercers com NgRx. Ens centrarem en l'ús de NgRx, una biblioteca popular de gestió d'estat per a Angular.

Ús de Serveis per a la Gestió d'Estat

Els serveis en Angular es poden utilitzar per gestionar l'estat compartint dades entre components.

// counter.service.ts
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class CounterService {
  private count = 0;

  getCount() {
    return this.count;
  }

  increment() {
    this.count++;
  }

  decrement() {
    this.count--;
  }
}
// counter.component.ts
import { Component } from '@angular/core';
import { CounterService } from './counter.service';

@Component({
  selector: 'app-counter',
  template: `
    <div>
      <p>Count: {{ counterService.getCount() }}</p>
      <button (click)="counterService.increment()">Increment</button>
      <button (click)="counterService.decrement()">Decrement</button>
    </div>
  `
})
export class CounterComponent {
  constructor(public counterService: CounterService) {}
}

Ús de NgRx per a la Gestió d'Estat

NgRx és una potent biblioteca que segueix el patró Redux per gestionar l'estat en aplicacions Angular.

Configuració de NgRx

  1. Instal·la NgRx:

    npm install @ngrx/store @ngrx/effects @ngrx/store-devtools

  2. Defineix Accions:

    // counter.actions.ts
import { createAction } from '@ngrx/store';

export const increment = createAction('[Counter] Increment');
export const decrement = createAction('[Counter] Decrement');

  3. Crea Reducers:

    // counter.reducer.ts
import { createReducer, on } from '@ngrx/store';
import { increment, decrement } from './counter.actions';

export const initialState = 0;

const _counterReducer = createReducer(
  initialState,
  on(increment, state => state + 1),
  on(decrement, state => state - 1)
);

export function counterReducer(state, action) {
  return _counterReducer(state, action);
}

  4. Registra el Store:

    // app.module.ts
import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';
import { StoreModule } from '@ngrx/store';
import { counterReducer } from './counter.reducer';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    StoreModule.forRoot({ count: counterReducer })
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule {}

  5. Utilitza Selectors i Despatxa Accions:

    // counter.component.ts
import { Component } from '@angular/core';
import { Store } from '@ngrx/store';
import { increment, decrement } from './counter.actions';

@Component({
  selector: 'app-counter',
  template: `
    <div>
      <p>Count: {{ count$ | async }}</p>
      <button (click)="increment()">Increment</button>
      <button (click)="decrement()">Decrement</button>
    </div>
  `
})
export class CounterComponent {
  count$ = this.store.select('count');

  constructor(private store: Store<{ count: number }>) {}

  increment() {
    this.store.dispatch(increment());
  }

  decrement() {
    this.store.dispatch(decrement());
  }
}

Conclusió

La gestió d'estat és un concepte fonamental en el desenvolupament d'Angular. Comprenent i implementant tècniques de gestió d'estat, pots construir aplicacions més predictibles, escalables i mantenibles. Tant si utilitzes serveis com una biblioteca robusta com NgRx, dominar la gestió d'estat millorarà significativament les teves habilitats de desenvolupament en Angular.
