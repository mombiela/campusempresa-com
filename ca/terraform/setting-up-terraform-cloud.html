Introducció

Terraform Cloud és un servei gestionat que proporciona un entorn col·laboratiu per a equips per gestionar la seva infraestructura com a codi. Ofereix funcions com la gestió d'estat remot, integració amb VCS i aplicació de polítiques. Aquesta secció et guiarà a través de la configuració de Terraform Cloud des de zero.

Prerequisits

Abans de configurar Terraform Cloud, assegura't de tenir el següent:

  • Un compte de Terraform Cloud (registra't a Terraform Cloud)
  • Comprensió bàsica de Terraform i infraestructura com a codi
  • Un compte de sistema de control de versions (VCS) com GitHub, GitLab o Bitbucket

Passos per Configurar Terraform Cloud

1. Crear un Compte de Terraform Cloud

  1. Visita el lloc web de Terraform Cloud.
  2. Registra't per obtenir un nou compte o inicia sessió si ja en tens un.
  3. Segueix les instruccions en pantalla per completar la configuració.

2. Crear una Organització

Les organitzacions a Terraform Cloud s'utilitzen per agrupar espais de treball relacionats i gestionar el control d'accés.

  1. Un cop iniciada la sessió, fes clic a "New Organization".
  2. Introdueix un nom per a la teva organització.
  3. Fes clic a "Create Organization".

3. Crear un Espai de Treball

Els espais de treball a Terraform Cloud s'utilitzen per gestionar diferents conjunts d'infraestructura.

  1. Navega a la teva organització.
  2. Fes clic a "New Workspace".
  3. Tria un tipus d'espai de treball:
    • Version Control Workflow: Es connecta a un repositori VCS.
    • CLI-driven Workflow: Utilitza el CLI de Terraform per gestionar les execucions.
  4. Segueix les indicacions per configurar l'espai de treball.

4. Connectar-se a un Sistema de Control de Versions (VCS)

Si has triat el Version Control Workflow:

  1. Selecciona el teu proveïdor de VCS (per exemple, GitHub, GitLab).
  2. Autentica't i autoritza Terraform Cloud per accedir al teu compte de VCS.
  3. Selecciona el repositori que conté els teus fitxers de configuració de Terraform.

5. Configurar la Configuració de l'Espai de Treball

  1. Variables: Defineix qualsevol variable necessària per a la teva configuració de Terraform.
    • Navega a la pestanya "Variables".
    • Afegeix variables com a parells clau-valor.
  2. Versió de Terraform: Especifica la versió de Terraform a utilitzar.
    • Navega a la pestanya "Settings".
    • Selecciona la versió desitjada de Terraform des del menú desplegable.

6. Inicialitzar i Aplicar la Configuració

  1. Puja els teus fitxers de configuració de Terraform al repositori VCS connectat.
  2. Terraform Cloud detectarà automàticament els canvis i desencadenarà un pla.
  3. Revisa el pla i confirma l'aplicació per provisionar la teva infraestructura.

Exemple de Configuració

Aquí tens un exemple senzill d'un fitxer de configuració de Terraform (main.tf) que pots utilitzar per provar la teva configuració:

provider "aws" {
  region = "us-west-2"
}

resource "aws_instance" "example" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
}
  1. Desa aquest fitxer al teu repositori VCS.
  2. Puja els canvis per desencadenar una execució de Terraform Cloud.

Gestió de l'Estat

Terraform Cloud gestiona automàticament els teus fitxers d'estat, assegurant-se que es desen de manera segura i consistent. Pots veure i gestionar les versions de l'estat des de la pestanya "States" al teu espai de treball.

Aplicació de Polítiques

Terraform Cloud et permet aplicar polítiques utilitzant Sentinel. Això assegura que la teva infraestructura compleixi amb els estàndards organitzacionals i els requisits de compliment.

Conclusió

Configurar Terraform Cloud implica crear un compte, configurar una organització, crear espais de treball i connectar-se a un VCS. Terraform Cloud simplifica la col·laboració i la gestió de l'estat, facilitant la gestió de la infraestructura com a codi. Seguint els passos descrits en aquesta secció, pots configurar i començar a utilitzar Terraform Cloud de manera eficient per a les teves necessitats d'infraestructura.
