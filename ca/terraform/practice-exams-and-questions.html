Introducció

Els exàmens i les preguntes de pràctica són essencials per reforçar la teva comprensió dels conceptes de Terraform i assegurar-te que estàs ben preparat per a escenaris del món real. Aquesta secció proporciona una varietat de preguntes de pràctica i exàmens simulats, que van des de nivells principiants fins a avançats, per ajudar-te a provar els teus coneixements i habilitats.

Nivell Principiant

Conceptes Clau

  • Comandes bàsiques de Terraform
  • Comprensió dels proveïdors i recursos
  • Escriure i aplicar configuracions simples de Terraform

Preguntes de Pràctica

  1. Què és Terraform?

    • Una eina per construir, canviar i versionar infraestructura de manera segura i eficient.

  2. Escriu una configuració simple de Terraform per crear un bucket S3 d'AWS.

    provider "aws" {
  region = "us-west-2"
}

resource "aws_s3_bucket" "example" {
  bucket = "my-unique-bucket-name"
  acl    = "private"
}

  3. Explica la finalitat de la comanda terraform init.

    • Inicialitza un directori de treball de Terraform descarregant i instal·lant els plugins de proveïdor necessaris.

Examen Simulat

  1. Elecció Múltiple: Quina comanda s'utilitza per aplicar els canvis necessaris per arribar a l'estat desitjat de la configuració?

    • A) terraform plan
    • B) terraform apply
    • C) terraform init
    • D) terraform destroy
    • Resposta: B) terraform apply

  2. Cert o Fals: L'estat de Terraform s'utilitza per mapar recursos del món real a la teva configuració.

    • Resposta: Cert

Nivell Intermedi

Conceptes Clau

  • Mòduls i el seu ús
  • Gestió de l'estat
  • Fonts de dades i sortides

Preguntes de Pràctica

  1. Què és un mòdul de Terraform i per què és útil?

    • Un mòdul és un contenidor per a múltiples recursos que s'utilitzen junts. Els mòduls permeten organitzar i reutilitzar codi.

  2. Escriu una configuració de Terraform que utilitzi un mòdul per crear una instància EC2 d'AWS.

    module "ec2_instance" {
  source = "./modules/ec2_instance"
  instance_type = "t2.micro"
  ami           = "ami-0c55b159cbfafe1f0"
}

  3. Explica la finalitat de la comanda terraform state.

    • La comanda terraform state s'utilitza per gestionar el fitxer d'estat, que fa un seguiment dels recursos gestionats per Terraform.

Examen Simulat

  1. Elecció Múltiple: Quina comanda s'utilitza per eliminar un recurs de l'estat de Terraform?

    • A) terraform destroy
    • B) terraform rm
    • C) terraform state rm
    • D) terraform delete
    • Resposta: C) terraform state rm

  2. Cert o Fals: Les fonts de dades en Terraform s'utilitzen per obtenir informació de fonts externes.

    • Resposta: Cert

Nivell Avançat

Conceptes Clau

  • Gestió avançada de l'estat (per exemple, estat remot)
  • Funcionalitats de Terraform Cloud i Enterprise
  • Escriure proveïdors personalitzats

Preguntes de Pràctica

  1. Què és l'estat remot i per què és important?

    • L'estat remot permet emmagatzemar el fitxer d'estat en una ubicació remota, cosa que és essencial per a la col·laboració i per assegurar-se que el fitxer d'estat no es perdi.

  2. Escriu una configuració de Terraform per emmagatzemar l'estat en un bucket S3 d'AWS.

    terraform {
  backend "s3" {
    bucket = "my-terraform-state"
    key    = "path/to/my/key"
    region = "us-west-2"
  }
}

  3. Explica com escriure un proveïdor personalitzat en Terraform.

    • Escriure un proveïdor personalitzat implica utilitzar el SDK de plugins de Terraform per crear un nou proveïdor que pugui gestionar recursos no suportats pels proveïdors existents. Això inclou definir l'esquema del proveïdor, l'esquema de recursos i implementar operacions CRUD.

Examen Simulat

  1. Elecció Múltiple: Quin dels següents és una funcionalitat de Terraform Enterprise?

    • A) Emmagatzematge remot de l'estat
    • B) Polítiques com a codi
    • C) Integració amb control de versions
    • D) Totes les anteriors
    • Resposta: D) Totes les anteriors

  2. Cert o Fals: Terraform Cloud proporciona un entorn col·laboratiu per a equips per gestionar la infraestructura.

    • Resposta: Cert

Conclusió

Els exàmens i les preguntes de pràctica són una part vital per dominar Terraform. Treballant en aquests exercicis, pots consolidar la teva comprensió dels conceptes clau, millorar les teves habilitats de resolució de problemes i preparar-te per als desafiaments reals d'infraestructura. Tant si ets un usuari principiant, intermedi o avançat, la pràctica contínua t'ajudarà a convertir-te en un expert en l'ús de Terraform per gestionar la infraestructura com a codi.
