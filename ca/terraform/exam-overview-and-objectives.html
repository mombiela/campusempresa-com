Introducció

En aquesta secció, proporcionarem una visió general de l'estructura i els objectius de l'examen per al curs de Terraform. Això t'ajudarà a entendre què esperar i com preparar-te de manera efectiva.

Estructura de l'examen

L'examen està dissenyat per avaluar els teus coneixements i habilitats a través de diversos nivells de Terraform, des de principiant fins a avançat. L'examen es divideix en tres seccions principals:

  1. Nivell Principiant
  2. Nivell Intermedi
  3. Nivell Avançat

Cada secció inclourà preguntes de selecció múltiple, exercicis pràctics i tasques d'anàlisi de codi.

Objectius per nivell

Nivell Principiant

Al nivell principiant, l'examen se centrarà en conceptes fonamentals i l'ús bàsic de Terraform. Els objectius inclouen:

  • Comprendre els principis de la Infraestructura com a Codi (IaC).
  • Instal·lar i configurar Terraform.
  • Escriure configuracions bàsiques de Terraform.
  • Gestionar fitxers d'estat.
  • Aplicar i destruir infraestructura.

Conceptes clau:

  • Infraestructura com a Codi (IaC): La pràctica de gestionar i aprovisionar infraestructura informàtica a través de fitxers de configuració llegibles per màquina.
  • Configuració de Terraform: El conjunt de fitxers utilitzats per descriure la infraestructura.
  • Fitxer d'estat: Un fitxer que fa un seguiment dels recursos gestionats per Terraform.

Exemple:

provider "aws" {
  region = "us-west-2"
}

resource "aws_instance" "example" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
}

Aquest exemple mostra una configuració bàsica per crear una instància EC2 d'AWS.

Nivell Intermedi

Al nivell intermedi, l'examen provarà la teva capacitat per gestionar infraestructures més complexes i utilitzar funcions avançades de Terraform. Els objectius inclouen:

  • Utilitzar mòduls per organitzar configuracions.
  • Gestionar múltiples entorns.
  • Implementar emmagatzematge d'estat remot.
  • Utilitzar fonts de dades.
  • Escriure i utilitzar provisionadors.

Conceptes clau:

  • Mòduls: Paquets reutilitzables i autònoms de configuracions de Terraform.
  • Estat remot: Emmagatzemar el fitxer d'estat en una ubicació remota per a la col·laboració.
  • Fonts de dades: Permetre que Terraform obtingui dades de fonts externes.

Exemple:

module "vpc" {
  source = "terraform-aws-modules/vpc/aws"
  version = "2.21.0"

  name = "my-vpc"
  cidr = "10.0.0.0/16"
  azs  = ["us-west-2a", "us-west-2b", "us-west-2c"]
}

Aquest exemple demostra l'ús d'un mòdul per crear una VPC.

Nivell Avançat

Al nivell avançat, l'examen avaluarà la teva capacitat per gestionar escenaris complexos i optimitzar l'ús de Terraform. Els objectius inclouen:

  • Escriure mòduls personalitzats.
  • Implementar dependències complexes.
  • Utilitzar espais de treball per a la gestió d'entorns.
  • Optimitzar i refactoritzar el codi de Terraform.
  • Integrar Terraform amb pipelines de CI/CD.

Conceptes clau:

  • Mòduls personalitzats: Crear els teus propis mòduls reutilitzables.
  • Espais de treball: Aïllar fitxers d'estat per a diferents entorns.
  • Integració CI/CD: Automatitzar els fluxos de treball de Terraform dins de pipelines de CI/CD.

Exemple:

resource "aws_s3_bucket" "example" {
  bucket = "my-unique-bucket-name"
  acl    = "private"

  versioning {
    enabled = true
  }

  lifecycle_rule {
    id      = "log"
    enabled = true

    transition {
      days          = 30
      storage_class = "GLACIER"
    }

    expiration {
      days = 365
    }
  }
}

Aquest exemple mostra una configuració avançada per a un bucket S3 amb versionat i regles de cicle de vida.

Conclusió

Entendre l'estructura i els objectius de l'examen és crucial per a una preparació efectiva. Cada nivell es construeix sobre l'anterior, assegurant una comprensió completa de Terraform. En dominar els conceptes clau i practicar amb exemples del món real, estaràs ben preparat per tenir èxit en l'examen i aplicar Terraform en els teus projectes.
