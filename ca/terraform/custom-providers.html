En aquesta secció, explorarem com crear i utilitzar proveïdors personalitzats a Terraform. Aquest tema és essencial per als usuaris avançats de Terraform que necessiten ampliar les capacitats de Terraform més enllà dels proveïdors integrats.

Introducció als Proveïdors Personalitzats

Els proveïdors personalitzats a Terraform et permeten interactuar amb APIs i serveis que no són compatibles amb els proveïdors oficials de Terraform. Això pot ser particularment útil per a sistemes propietaris o serveis de nínxol.

Conceptes Clau

  • Proveïdor: Un proveïdor és un complement que Terraform utilitza per interactuar amb APIs de proveïdors de núvol, proveïdors SaaS i altres serveis.
  • Proveïdor Personalitzat: Un proveïdor personalitzat és un proveïdor definit per l'usuari que permet a Terraform gestionar recursos per a serveis no compatibles amb els proveïdors oficials.
  • Terraform Plugin SDK: Un conjunt d'eines i biblioteques proporcionades per HashiCorp per ajudar-te a construir proveïdors personalitzats.

Configuració del Teu Entorn de Desenvolupament

Abans de començar a crear un proveïdor personalitzat, necessites configurar el teu entorn de desenvolupament.

Prerequisits

  • Llenguatge de Programació Go: Els proveïdors de Terraform estan escrits en Go. Assegura't de tenir Go instal·lat a la teva màquina.
  • Terraform Plugin SDK: Necessitaràs el Terraform Plugin SDK per desenvolupar el teu proveïdor personalitzat.

Passos d'Instal·lació

  1. Instal·la Go: Segueix les instruccions al lloc web oficial de Go per instal·lar Go.
  2. Configura l'Espai de Treball de Go: Configura el teu espai de treball de Go establint la variable d'entorn GOPATH.
  3. Instal·la Terraform Plugin SDK: Utilitza la següent comanda per instal·lar l'SDK: 
    go get github.com/hashicorp/terraform-plugin-sdk/v2

Creació d'un Proveïdor Personalitzat

En aquesta secció, crearem un proveïdor personalitzat senzill per gestionar un servei fictici anomenat "ExampleService".

Guia Pas a Pas

  1. Inicialitza el Teu Projecte: Crea un nou directori per al teu proveïdor i inicialitza un mòdul de Go.

    mkdir terraform-provider-exampleservice
cd terraform-provider-exampleservice
go mod init github.com/yourusername/terraform-provider-exampleservice

  2. Crea el Fitxer del Proveïdor: Crea un fitxer anomenat main.go i defineix el proveïdor.

    package main

import (
    "github.com/hashicorp/terraform-plugin-sdk/v2/helper/schema"
    "github.com/hashicorp/terraform-plugin-sdk/v2/plugin"
)

func main() {
    plugin.Serve(&plugin.ServeOpts{
        ProviderFunc: func() *schema.Provider {
            return &schema.Provider{
                ResourcesMap: map[string]*schema.Resource{
                    "exampleservice_item": resourceItem(),
                },
            }
        },
    })
}

  3. Defineix un Recurs: Crea un fitxer anomenat resource_item.go i defineix un recurs per a l'ExampleService.

    package main

import (
    "github.com/hashicorp/terraform-plugin-sdk/v2/helper/schema"
)

func resourceItem() *schema.Resource {
    return &schema.Resource{
        Create: resourceItemCreate,
        Read:   resourceItemRead,
        Update: resourceItemUpdate,
        Delete: resourceItemDelete,

        Schema: map[string]*schema.Schema{
            "name": {
                Type:     schema.TypeString,
                Required: true,
            },
            "value": {
                Type:     schema.TypeString,
                Required: true,
            },
        },
    }
}

func resourceItemCreate(d *schema.ResourceData, m interface{}) error {
    // Implementa la lògica per crear un element a ExampleService
    d.SetId("example_id")
    return resourceItemRead(d, m)
}

func resourceItemRead(d *schema.ResourceData, m interface{}) error {
    // Implementa la lògica per llegir un element de ExampleService
    return nil
}

func resourceItemUpdate(d *schema.ResourceData, m interface{}) error {
    // Implementa la lògica per actualitzar un element a ExampleService
    return resourceItemRead(d, m)
}

func resourceItemDelete(d *schema.ResourceData, m interface{}) error {
    // Implementa la lògica per eliminar un element de ExampleService
    d.SetId("")
    return nil
}

  4. Construeix el Proveïdor: Compila el teu proveïdor utilitzant l'eina de construcció de Go.

    go build -o terraform-provider-exampleservice

Utilització del Teu Proveïdor Personalitzat

Un cop hagis construït el teu proveïdor personalitzat, pots utilitzar-lo en les teves configuracions de Terraform.

Configuració d'Exemple

Crea un fitxer de configuració de Terraform anomenat main.tf per utilitzar el teu proveïdor personalitzat.

provider "exampleservice" {}

resource "exampleservice_item" "example" {
  name  = "example_name"
  value = "example_value"
}

Inicialitza i Aplica

  1. Inicialitza Terraform: Executa la següent comanda per inicialitzar Terraform i descarregar el proveïdor.

    terraform init

  2. Aplica la Configuració: Aplica la configuració per crear el recurs.

    terraform apply

Conclusió

Crear proveïdors personalitzats a Terraform et permet ampliar la funcionalitat de Terraform per gestionar recursos per a serveis que no són compatibles de manera nativa. Seguint els passos descrits en aquesta secció, pots configurar el teu entorn de desenvolupament, crear un proveïdor personalitzat i utilitzar-lo en les teves configuracions de Terraform. Aquesta habilitat és particularment valuosa per als usuaris avançats que necessiten integrar Terraform amb serveis propietaris o de nínxol.
