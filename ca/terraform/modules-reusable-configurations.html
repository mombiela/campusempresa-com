Introducció als Mòduls

Els mòduls a Terraform són una manera poderosa d'organitzar i reutilitzar configuracions. Permeten encapsular un conjunt de recursos i reutilitzar aquesta configuració a diferents parts de la teva infraestructura.

Conceptes Clau

  • Mòdul: Un contenidor per a múltiples recursos que s'utilitzen junts.
  • Mòdul Arrel: El directori principal on s'executa Terraform.
  • Mòdul Fill: Qualsevol mòdul que és cridat per un altre mòdul.
  • Registre de Mòduls: Un repositori on els mòduls es poden compartir i reutilitzar.

Creació d'un Mòdul Bàsic

Per crear un mòdul, necessites organitzar els teus fitxers de configuració en una estructura de directoris. Cada mòdul ha de tenir el seu propi directori.

Exemple d'Estructura de Directoris

my-terraform-project/
├── main.tf
├── variables.tf
├── outputs.tf
└── modules/
    └── s3_bucket/
        ├── main.tf
        ├── variables.tf
        └── outputs.tf

Exemple de Configuració de Mòdul

Creem un mòdul senzill per gestionar un bucket S3.

modules/s3_bucket/main.tf

resource "aws_s3_bucket" "example" {
  bucket = var.bucket_name
  acl    = var.acl
}

modules/s3_bucket/variables.tf

variable "bucket_name" {
  description = "El nom del bucket S3"
  type        = string
}

variable "acl" {
  description = "L'ACL per al bucket S3"
  type        = string
  default     = "private"
}

modules/s3_bucket/outputs.tf

output "bucket_arn" {
  description = "L'ARN del bucket S3"
  value       = aws_s3_bucket.example.arn
}

Ús d'un Mòdul

Per utilitzar un mòdul, necessites cridar-lo des del teu mòdul arrel.

Exemple de Configuració del Mòdul Arrel

main.tf

module "s3_bucket" {
  source      = "./modules/s3_bucket"
  bucket_name = "my-unique-bucket-name"
  acl         = "public-read"
}

output "bucket_arn" {
  value = module.s3_bucket.bucket_arn
}

Explicació

  • source: Especifica el camí al mòdul.
  • bucket_name i acl: Aquests són variables definides al mòdul que necessites proporcionar valors.
  • output "bucket_arn": Això retorna l'ARN del bucket S3 creat pel mòdul.

Millors Pràctiques per als Mòduls

  • Encapsulació: Mantingues els mòduls enfocats en una sola responsabilitat.
  • Versionat: Utilitza control de versions per gestionar els canvis als teus mòduls.
  • Documentació: Documenta el propòsit i l'ús dels teus mòduls.
  • Proves: Prova els teus mòduls de manera independent abans d'integrar-los en configuracions més grans.

Exemple de Documentació del Mòdul

modules/s3_bucket/README.md

# Mòdul de Bucket S3

Aquest mòdul crea un bucket S3 amb el nom i l'ACL especificats.

## Variables

- `bucket_name`: El nom del bucket S3 (requerit).
- `acl`: L'ACL per al bucket S3 (opcional, el valor per defecte és "private").

## Sortides

- `bucket_arn`: L'ARN del bucket S3.

Conclusió

Els mòduls són una característica fonamental a Terraform que et permeten escriure codi d'infraestructura net, reutilitzable i mantenible. Encapsulant recursos en mòduls, pots gestionar i escalar fàcilment la teva infraestructura. Recorda seguir les millors pràctiques com l'encapsulació, el versionat, la documentació i les proves per assegurar que els teus mòduls siguin robustos i fiables.
