El flux de control és un concepte fonamental en la programació que dicta l'ordre en què s'executen les instruccions. A Xcode, que utilitza principalment Swift, entendre el flux de control és essencial per escriure codi eficient i efectiu. Aquest tema cobrirà les bases del flux de control, incloent-hi declaracions condicionals, bucles i tècniques avançades de flux de control.

Declaracions Condicionals

Les declaracions condicionals permeten que el teu programa prengui decisions basades en certes condicions. Les declaracions condicionals més comunes en Swift són if, else if, else i switch.

Declaracions If-Else

  • Declaració If: Executa un bloc de codi si una condició especificada és certa.
  • Declaració Else: Executa un bloc de codi si la condició en la declaració if és falsa.
  • Declaració Else If: Proporciona una condició addicional si la condició inicial if és falsa.
let temperature = 30

if temperature > 25 {
    print("Fa calor fora!")
} else if temperature < 15 {
    print("Fa fred fora!")
} else {
    print("El temps és moderat.")
}

Declaracions Switch

  • Declaració Switch: Avalua un valor contra múltiples patrons possibles coincidents i executa el bloc de codi corresponent.
let dayOfWeek = 3

switch dayOfWeek {
case 1:
    print("Dilluns")
case 2:
    print("Dimarts")
case 3:
    print("Dimecres")
case 4:
    print("Dijous")
case 5:
    print("Divendres")
case 6:
    print("Dissabte")
case 7:
    print("Diumenge")
default:
    print("Dia invàlid")
}

Bucles

Els bucles permeten executar un bloc de codi múltiples vegades. Swift proporciona diversos tipus de bucles, incloent-hi for, while i repeat-while.

Bucles For

  • Bucle For-In: Itera sobre una seqüència, com ara elements en una matriu o un rang de números.
let fruits = ["Poma", "Plàtan", "Cirera"]

for fruit in fruits {
    print(fruit)
}

Bucles While

  • Bucle While: Executa repetidament un bloc de codi mentre una condició especificada sigui certa.
var count = 5

while count > 0 {
    print(count)
    count -= 1
}

Bucles Repeat-While

  • Bucle Repeat-While: Similar al bucle while, però executa el bloc de codi almenys una vegada abans de comprovar la condició.
var number = 0

repeat {
    print("El número és \(number)")
    number += 1
} while number < 3

Flux de Control Avançat

Les tècniques avançades de flux de control poden fer que el teu codi sigui més eficient i llegible. Aquestes inclouen les declaracions guard i les declaracions break/continue.

Declaracions Guard

  • Declaració Guard: S'utilitza per transferir el control del programa fora d'un àmbit si una o més condicions no es compleixen. Sovint s'utilitza per a sortides anticipades en funcions.
func greet(person: [String: String]) {
    guard let name = person["name"] else {
        return
    }
    print("Hola, \(name)!")
}

let personInfo = ["name": "John"]
greet(person: personInfo)

Declaracions Break i Continue

  • Declaració Break: Termina el bucle actual o la declaració switch.
  • Declaració Continue: Salta la iteració actual del bucle i procedeix a la següent iteració.
for number in 1...5 {
    if number == 3 {
        continue // Salta la resta del codi en aquesta iteració
    }
    if number == 5 {
        break // Surt del bucle
    }
    print(number)
}

Conclusió

Entendre el flux de control és crucial per escriure codi efectiu en Swift a Xcode. En dominar les declaracions condicionals, els bucles i les tècniques avançades de flux de control, pots crear aplicacions més dinàmiques i responsives. Practica aquests conceptes amb exemples del món real per solidificar la teva comprensió i millorar les teves habilitats de programació.
