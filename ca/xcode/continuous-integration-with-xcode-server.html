Introducció a la Integració Contínua (CI)

  • Definició: La Integració Contínua (CI) és una pràctica de desenvolupament on els desenvolupadors integren codi en un repositori compartit freqüentment, idealment diverses vegades al dia.
  • Beneficis:
    • Detecció precoç d'errors.
    • Millora de la qualitat del codi.
    • Cicles de desenvolupament més ràpids.
    • Proves i desplegament automatitzats.

Configuració de Xcode Server

  • Requisits:
    • Un Mac amb macOS.
    • Xcode instal·lat.
    • Accés a un repositori Git.

Passos per Configurar Xcode Server

  1. Habilitar Xcode Server:

    • Obre Xcode.
    • Ves a Xcode > Preferències > Comptes.
    • Fes clic al botó + i selecciona Afegeix Servidor.
    • Segueix les instruccions per configurar el teu servidor.

  2. Crear un Bot:

    • A Xcode, ves a Producte > Crea Bot.
    • Selecciona el projecte i l'esquema que vols construir.
    • Configura els paràmetres del bot, com ara els desencadenants i les opcions de prova.

  3. Configurar Desencadenants del Bot:

    • Els desencadenants determinen quan s'executa el bot.
    • Desencadenants comuns inclouen:
      • En cada commit.
      • A una hora programada.
      • Manualment.

Exemple: Crear un Bot Simple

// Guia pas a pas per crear un bot
1. Obre el teu projecte Xcode.
2. Ves a `Producte` > `Crea Bot`.
3. Selecciona el teu projecte i esquema.
4. Configura el bot:
   - Nom: MyFirstBot
   - Desencadenants d'Integració: En cada commit
   - Proves: Habilita proves unitàries
5. Fes clic a `Crea` per acabar.

Configuració de Xcode Server per a Proves Automatitzades

  • Proves Unitàries:
    • Assegura't que el teu projecte tingui proves unitàries.
    • Xcode Server pot executar aquestes proves automàticament.
  • Proves d'Interfície d'Usuari (UI):
    • Crea proves d'interfície d'usuari per verificar la interfície.
    • Xcode Server també pot executar aquestes proves.

Exemple: Afegir Proves Unitàries

import XCTest

class MyFirstTests: XCTestCase {
    func testExample() {
        let value = 2 + 2
        XCTAssertEqual(value, 4, "Les matemàtiques estan trencades!")
    }
}

Monitorització i Gestió de Bots

  • Panell de Control:
    • Xcode Server proporciona un panell de control per monitoritzar l'activitat del bot.
    • Visualitza l'estat de les integracions, els resultats de les proves i els registres.
  • Notificacions per Correu Electrònic:
    • Configura notificacions per correu electrònic per rebre actualitzacions sobre l'activitat del bot.
    • Útil per mantenir l'equip informat sobre l'estat de la construcció.

Exemple: Configuració de Notificacions per Correu Electrònic

  1. Ves a la configuració del teu bot.
  2. Navega a la pestanya Notificacions.
  3. Afegeix adreces de correu electrònic per rebre notificacions.
  4. Tria els esdeveniments per desencadenar notificacions (per exemple, fallades de construcció).

Funcions Avançades de CI

  • Scripts Personalitzats:
    • Executa scripts personalitzats abans o després de les integracions.
    • Útil per a tasques com la revisió de codi o el desplegament.
  • Proves en Paral·lel:
    • Executa proves en paral·lel per accelerar el procés de CI.
    • Requereix múltiples dispositius o simuladors.

Exemple: Afegir un Script Personalitzat

#!/bin/bash
# Script per executar revisió de codi
swiftlint
  1. Desa l'script com lint.sh.
  2. A la configuració del teu bot, ves a la pestanya Scripts.
  3. Afegeix lint.sh com a script pre-integració.

Conclusió

  • Resum:
    • La Integració Contínua amb Xcode Server ajuda a agilitzar el procés de desenvolupament.
    • Automatitza la construcció, les proves i el desplegament d'aplicacions iOS.
    • Millora la qualitat del codi i redueix els problemes d'integració.
  • Pròxims Passos:
    • Experimenta amb diferents configuracions de bot.
    • Integra eines addicionals com Fastlane per a fluxos de treball més avançats.
    • Monitoritza i millora contínuament la teva configuració de CI per adaptar-la a les necessitats del teu equip.
