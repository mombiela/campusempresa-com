Introducció als Storyboards

Els storyboards són una representació visual de la interfície d'usuari de la teva aplicació a Xcode. Et permeten dissenyar i organitzar els elements de la UI i els controladors de vista de la teva aplicació de manera visual.

  • Què és un Storyboard?

    • Un fitxer que conté la representació visual de la UI de la teva aplicació.
    • Et permet dissenyar el flux de la teva aplicació utilitzant escenes i segues.
    • Proporciona una manera de veure tota la UI de la teva aplicació en un sol lloc.

  • Avantatges d'Utilitzar Storyboards

    • Representació visual de la UI de l'aplicació.
    • Més fàcil de gestionar i entendre el flux de l'aplicació.
    • Simplifica el procés de disseny de UIs complexes.

Creació i Configuració de Storyboards

Crear un Nou Storyboard

  1. Obre Xcode i crea un nou projecte.
  2. Selecciona la plantilla "Single View App".
  3. Al navegador de projectes, fes clic dret sobre el fitxer "Main.storyboard" i selecciona "New File".
  4. Tria "Storyboard" de les opcions de plantilla i fes clic a "Next".
  5. Nomena el teu fitxer storyboard i fes clic a "Create".

Afegir Controladors de Vista

  • Arrossega i deixa anar un controlador de vista des de la Biblioteca d'Objectes al storyboard.
  • Configura el controlador de vista inicial seleccionant el controlador de vista i marcant la casella "Is Initial View Controller" a l'Inspector d'Atributs.

Configurar Controladors de Vista

  • Establir la Classe: Assigna una classe personalitzada al teu controlador de vista.

    class MyViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        // Codi d'inicialització personalitzat
    }
}
    • A l'Inspector d'Identitat, estableix la classe del controlador de vista a MyViewController.

  • Afegir Elements de la UI: Arrossega i deixa anar elements de la UI (per exemple, botons, etiquetes) des de la Biblioteca d'Objectes al controlador de vista.

    • Personalitza les propietats utilitzant l'Inspector d'Atributs.

Treballant amb Segues

Els segues defineixen les transicions entre controladors de vista en un storyboard.

Crear un Segue

  1. Control-arrossega des d'un element de la UI (per exemple, botó) a un altre controlador de vista.
  2. Selecciona el tipus de segue (per exemple, Show, Present Modally).

Configurar Segues

  • Identificador: Estableix un identificador per al segue a l'Inspector d'Atributs.
  • Executar un Segue Programàticament: 
    performSegue(withIdentifier: "showDetail", sender: self)

Passar Dades Entre Controladors de Vista

  • Prepare for Segue: Sobreescriu el mètode prepare(for:sender:) per passar dades. 
    override func prepare(for segue: UIStoryboardSegue, sender: Any?) {
    if segue.identifier == "showDetail" {
        let destinationVC = segue.destination as! DetailViewController
        destinationVC.data = "Algunes dades"
    }
}

Auto Layout i Constraints

Auto Layout et permet crear UIs responsives i adaptatives.

Afegir Constraints

  • Selecciona un element de la UI i fes clic al botó "Add New Constraints" a l'Interface Builder.
  • Estableix les constraints desitjades (per exemple, leading, trailing, top, bottom).

Utilitzar Stack Views

  • Les stack views simplifiquen el procés de creació de dissenys complexos.
  • Arrossega i deixa anar una Stack View des de la Biblioteca d'Objectes.
  • Afegeix elements de la UI a la stack view i configura les seves propietats (per exemple, axis, distribution).

Tècniques Avançades de Storyboard

Segues Personalitzats

  • Crea classes de segue personalitzades subclassejant UIStoryboardSegue. 
    class CustomSegue: UIStoryboardSegue {
    override func perform() {
        // Codi de transició personalitzat
    }
}

Referències de Storyboard

  • Utilitza referències de storyboard per dividir storyboards grans en parts més petites i manejables.
  • Arrossega i deixa anar una Referència de Storyboard des de la Biblioteca d'Objectes.
  • Estableix el storyboard i el controlador de vista referenciat a l'Inspector d'Atributs.

Conclusió

Els storyboards són una eina poderosa a Xcode per dissenyar i gestionar la interfície d'usuari de la teva aplicació. Proporcionen una manera visual de dissenyar la UI, definir transicions i gestionar el flux de la teva aplicació. Entenent com crear i configurar storyboards, afegir controladors de vista, treballar amb segues i utilitzar Auto Layout, pots construir UIs complexes i responsives de manera eficient. Les tècniques avançades com els segues personalitzats i les referències de storyboard milloren encara més la teva capacitat per gestionar projectes grans. Amb pràctica, et convertiràs en un expert en l'ús de storyboards per crear aplicacions intuïtives i visualment atractives.
