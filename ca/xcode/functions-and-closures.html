Introducció a les funcions

Les funcions són blocs de codi autònoms que realitzen una tasca específica. A Swift, les funcions són ciutadans de primera classe, el que significa que es poden passar i utilitzar en el teu codi com qualsevol altre valor.

Definició i crida de funcions

Per definir una funció a Swift, utilitzes la paraula clau func seguida del nom de la funció, un parell de parèntesis i un conjunt de claus que contenen el codi de la funció.

func greet(person: String) -> String {
    return "Hello, \(person)!"
}

let greeting = greet(person: "Alice")
print(greeting) // Output: Hello, Alice!
  • func: Paraula clau per definir una funció.
  • greet(person: String) -> String: Signatura de la funció amb un paràmetre i tipus de retorn.
  • return "Hello, \\(person)!": El cos de la funció que retorna un missatge de salutació.

Paràmetres de funció i valors de retorn

Les funcions poden tenir múltiples paràmetres i retornar múltiples valors utilitzant tuples.

func add(a: Int, b: Int) -> Int {
    return a + b
}

let sum = add(a: 3, b: 5)
print(sum) // Output: 8
  • add(a: Int, b: Int) -> Int: Funció amb dos paràmetres i un tipus de retorn.

Valors de paràmetre per defecte

Pots proporcionar valors per defecte per a qualsevol paràmetre en una funció assignant un valor al paràmetre després del seu tipus.

func greet(person: String, from hometown: String = "Cupertino") -> String {
    return "Hello \(person)! Glad you could visit from \(hometown)."
}

print(greet(person: "Alice")) // Output: Hello Alice! Glad you could visit from Cupertino.
print(greet(person: "Bob", from: "New York")) // Output: Hello Bob! Glad you could visit from New York.

Conceptes avançats de funcions

Paràmetres in-out

Els paràmetres in-out permeten que una funció modifiqui una variable passada des de fora de la funció.

func swapTwoInts(_ a: inout Int, _ b: inout Int) {
    let temporaryA = a
    a = b
    b = temporaryA
}

var x = 3
var y = 5
swapTwoInts(&x, &y)
print("x: \(x), y: \(y)") // Output: x: 5, y: 3
  • inout: Paraula clau per indicar un paràmetre in-out.
  • &: Prefix utilitzat quan es passa una variable com a paràmetre in-out.

Tipus de funció

Cada funció té un tipus de funció específic, format pels tipus de paràmetres i el tipus de retorn.

func multiply(a: Int, b: Int) -> Int {
    return a * b
}

let mathFunction: (Int, Int) -> Int = multiply
print(mathFunction(2, 3)) // Output: 6
  • (Int, Int) -> Int: Tipus de funció per a una funció que pren dos paràmetres Int i retorna un Int.

Introducció a les clausures

Les clausures són blocs de funcionalitat autònoms que es poden passar i utilitzar en el teu codi. Les clausures a Swift són similars als blocs en C i Objective-C i a les lambdas en altres llenguatges de programació.

Definició i ús de clausures

Les clausures poden capturar i emmagatzemar referències a qualsevol constant i variable del context en què es defineixen.

let names = ["Chris", "Alex", "Ewa", "Barry", "Daniella"]

let sortedNames = names.sorted(by: { (s1: String, s2: String) -> Bool in
    return s1 < s2
})

print(sortedNames) // Output: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]
  • { (s1: String, s2: String) -> Bool in return s1 < s2 }: Sintaxi de l'expressió de clausura.

Noms de paràmetres abreujats

Swift proporciona noms de paràmetres abreujats per fer les clausures més concises.

let sortedNames = names.sorted(by: { $0 < $1 })
print(sortedNames) // Output: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]
  • $0, $1: Noms de paràmetres abreujats per al primer i segon paràmetre.

Clausures de tancament

Si una clausura és l'últim argument d'una funció, pots utilitzar la sintaxi de clausura de tancament.

let sortedNames = names.sorted { $0 < $1 }
print(sortedNames) // Output: ["Alex", "Barry", "Chris", "Daniella", "Ewa"]

Captura de valors

Les clausures poden capturar constants i variables del context circumdant en què es defineixen.

func makeIncrementer(forIncrement amount: Int) -> () -> Int {
    var total = 0
    let incrementer: () -> Int = {
        total += amount
        return total
    }
    return incrementer
}

let incrementByTwo = makeIncrementer(forIncrement: 2)
print(incrementByTwo()) // Output: 2
print(incrementByTwo()) // Output: 4
  • total += amount: Captura i modificació de la variable total del context circumdant.

Conclusió

Les funcions i les clausures són conceptes fonamentals en la programació amb Swift. Les funcions permeten encapsular codi per a la seva reutilització i claredat, mentre que les clausures proporcionen una manera poderosa de capturar i manipular valors del seu context circumdant. Dominar aquests conceptes et permetrà escriure codi més modular, flexible i concís a Swift.
