Introducció a Swift
Escrivint el Teu Primer Programa en Swift ►

Swift és un llenguatge de programació potent i intuïtiu desenvolupat per Apple per crear aplicacions per a iOS, macOS, watchOS i tvOS. Aquesta secció t'introduirà als conceptes bàsics de Swift, establint una base sòlida per desenvolupar aplicacions a Xcode.

Què és Swift?

Swift és un llenguatge de programació modern que és:

  • Segur: Swift elimina classes senceres de codi insegur.
  • Ràpid: Swift està dissenyat per ser ràpid i eficient.
  • Expressiu: Swift té una sintaxi neta que fa que el codi sigui fàcil de llegir i escriure.

Configuració del Teu Entorn

Abans de començar a codificar en Swift, necessites configurar el teu entorn de desenvolupament:

  1. Instal·la Xcode: Xcode és l'entorn de desenvolupament integrat (IDE) per a macOS que conté un conjunt d'eines de desenvolupament de programari.
  2. Crea un Nou Projecte: Obre Xcode i crea un nou projecte per començar a escriure codi Swift.

Sintaxi Bàsica

Variables i Constants

En Swift, utilitzes var per declarar variables i let per declarar constants.

var myVariable = 42
myVariable = 50
let myConstant = 42
  • Variables: Es poden canviar després de ser establertes.
  • Constants: No es poden canviar un cop establertes.

Tipus de Dades

Swift és un llenguatge segur en termes de tipus, el que significa que t'ajuda a ser clar sobre els tipus de valors amb els quals el teu codi pot treballar.

let explicitDouble: Double = 70
let explicitFloat: Float = 4.0
let explicitInt: Int = 42

Cadenes i Caràcters

Les cadenes s'utilitzen per emmagatzemar text. Pots crear una cadena tancant el text entre cometes dobles.

let greeting = "Hola, Món!"

Arrays i Diccionaris

Els arrays i diccionaris s'utilitzen per emmagatzemar col·leccions de valors.

var shoppingList = ["Ous", "Llet"]
var occupations = [
    "Malcolm": "Capità",
    "Kaylee": "Mecànica",
]

Flux de Control

Swift proporciona totes les estructures de flux de control estàndard com for, while i if.

let individualScores = [75, 43, 103, 87, 12]
var teamScore = 0
for score in individualScores {
    if score > 50 {
        teamScore += 3
    } else {
        teamScore += 1
    }
}
print(teamScore)

Funcions

Les funcions són blocs de codi autònoms que realitzen una tasca específica.

func greet(person: String, day: String) -> String {
    return "Hola \(person), avui és \(day)."
}
print(greet(person: "John", day: "Dimecres"))
  • Paràmetres: Entrades per a la funció.
  • Tipus de Retorn: El tipus de valor que la funció retorna.

Classes i Estructures

Swift proporciona construccions potents per crear tipus de dades personalitzats utilitzant classes i estructures.

Classes

Les classes suporten l'herència, el que significa que una classe pot heretar les característiques d'una altra.

class Shape {
    var numberOfSides = 0
    func simpleDescription() -> String {
        return "Una forma amb \(numberOfSides) costats."
    }
}

Estructures

Les estructures són tipus de valor i no suporten l'herència.

struct Card {
    var rank: Int
    var suit: String
    func simpleDescription() -> String {
        return "El \(rank) de \(suit)"
    }
}

Optionals

Els optionals s'utilitzen per gestionar l'absència d'un valor.

var optionalString: String? = "Hola"
print(optionalString == nil)

var optionalName: String? = "John Appleseed"
var greeting = "Hola!"
if let name = optionalName {
    greeting = "Hola, \(name)"
}

Conclusió

En aquesta introducció a Swift, hem cobert:

  • Els conceptes bàsics de la sintaxi de Swift.
  • Com declarar variables i constants.
  • Tipus de dades bàsics i col·leccions.
  • Estructures de flux de control.
  • Funcions, classes i estructures.
  • Optionals i el seu ús.

Entendre aquests fonaments et prepararà per a temes més avançats en Swift i el desenvolupament amb Xcode. Continua practicant aquests conceptes per convertir-te en un programador proficient en Swift.
