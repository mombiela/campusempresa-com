Introducció

Les proves i la depuració són aspectes crucials del desenvolupament de programari. En aquesta secció, explorarem com utilitzar efectivament les eines d'Xcode per provar i depurar les teves aplicacions. Això inclourà proves unitàries, proves d'interfície d'usuari i l'ús de les eines de depuració d'Xcode per identificar i solucionar problemes en el teu codi.

Proves Unitàries

Les proves unitàries impliquen provar components individuals del teu codi per assegurar-te que funcionen com s'espera.

  • Creació de Proves Unitàries:
    • Xcode proporciona un marc integrat anomenat XCTest per escriure proves unitàries.
    • Per crear una nova prova unitària, ves a File > New > Target i selecciona Unit Testing Bundle.
import XCTest
@testable import YourApp

class YourAppTests: XCTestCase {

    func testExample() {
        let result = YourClass().yourMethod()
        XCTAssertEqual(result, expectedValue, "The method did not return the expected value")
    }
}
  • Execució de Proves Unitàries:
    • Pots executar les teves proves prement Cmd + U o seleccionant Product > Test.

Proves d'Interfície d'Usuari

Les proves d'interfície d'usuari asseguren que la interfície d'usuari es comporti com s'espera.

  • Creació de Proves d'Interfície d'Usuari:
    • Similar a les proves unitàries, pots crear proves d'interfície d'usuari anant a File > New > Target i seleccionant UI Testing Bundle.
import XCTest

class YourAppUITests: XCTestCase {

    func testExample() {
        let app = XCUIApplication()
        app.launch()
        
        let button = app.buttons["YourButtonIdentifier"]
        XCTAssertTrue(button.exists, "The button should exist")
        button.tap()
        
        let label = app.staticTexts["YourLabelIdentifier"]
        XCTAssertEqual(label.label, "Expected Text", "The label text is incorrect")
    }
}
  • Execució de Proves d'Interfície d'Usuari:
    • Executa les teves proves d'interfície d'usuari prement Cmd + U o seleccionant Product > Test.

Depuració

La depuració és el procés d'identificar i solucionar problemes en el teu codi.

  • Punts de Trencament:

    • Els punts de trencament et permeten pausar l'execució del teu codi en una línia específica.
    • Per establir un punt de trencament, fes clic al número de línia al marge de l'editor de codi.

  • LLDB (Depurador de Baix Nivell):

    • Xcode utilitza LLDB per a la depuració. Pots interactuar amb LLDB a través de la consola de depuració.
    • Comandes comunes de LLDB:
      • po: Imprimeix la descripció de l'objecte.
      • p: Imprimeix el valor d'una variable.
      • bt: Imprimeix la traça de la pila del fil actual.
// Exemple d'ús de LLDB a la consola de depuració
(lldb) po someVariable
(lldb) p someVariable
(lldb) bt
  • Depuració de Vista:
    • Xcode proporciona un depurador visual de vista per inspeccionar la jerarquia de vistes.
    • Per utilitzar el depurador de vista, executa la teva aplicació i fes clic al botó Debug View Hierarchy a la barra d'eines de depuració.

Conclusió

Les proves i la depuració són habilitats essencials per a qualsevol desenvolupador. Xcode proporciona eines potents per ajudar-te a escriure proves i depurar les teves aplicacions de manera efectiva. Dominant aquestes eines, pots assegurar-te que les teves aplicacions siguin fiables i funcionin com s'espera.
