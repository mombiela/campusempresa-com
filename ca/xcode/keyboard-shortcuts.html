Introducció

Les dreceres de teclat són essencials per millorar la productivitat i l'eficiència quan es treballa a Xcode. Aquesta secció cobrirà les dreceres més útils des de nivells bàsics fins a avançats, ajudant-te a navegar i codificar de manera més efectiva.

Dreceres per a principiants

Aquestes dreceres són fonamentals i t'ajudaran a començar amb la navegació bàsica i l'edició a Xcode.

  • Command + N: Crear un nou fitxer.
  • Command + Shift + N: Crear un nou projecte.
  • Command + S: Desar el fitxer actual.
  • Command + Shift + S: Desar tots els fitxers.
  • Command + W: Tancar la finestra actual.
  • Command + Q: Sortir de Xcode.
  • Command + 1 a Command + 8: Canviar entre diferents navegadors (Projecte, Símbol, Cerca, etc.).

Exemple

// Utilitza Command + N per crear un nou fitxer Swift i escriure una funció simple
func greet(name: String) -> String {
    return "Hello, \(name)!"
}

Dreceres intermèdies

Aquestes dreceres t'ajudaran amb tasques de navegació i edició de codi més avançades.

  • Command + Control + E: Editar totes les ocurrències del símbol seleccionat.
  • Command + Option + /: Afegir un comentari de documentació.
  • Command + Option + .: Saltar al següent problema.
  • Command + Option + 0: Mostrar o amagar el panell d'Utilitats.
  • Command + Option + Enter: Mostrar l'Editor d'Assistent.
  • Command + Shift + O: Obrir ràpidament qualsevol fitxer pel nom.

Exemple

// Utilitza Command + Option + / per afegir documentació
/// Aquesta funció saluda una persona pel seu nom.
/// - Parameter name: El nom de la persona a saludar.
/// - Returns: Una cadena de salutació.
func greet(name: String) -> String {
    return "Hello, \(name)!"
}

Dreceres avançades

Aquestes dreceres són per a usuaris avançats que volen maximitzar la seva eficiència a Xcode.

  • Command + Shift + J: Revelar el fitxer actual al Navegador de Projectes.
  • Command + Control + Fletxa Amunt/Avall: Canviar entre fitxers .h i .m (Objective-C) o fitxers relacionats (Swift).
  • Command + Option + Shift + Fletxa Esquerra/Dreta: Navegar entre pestanyes.
  • Command + Option + Shift + Enter: Obrir l'Editor d'Assistent en una nova finestra.
  • Command + Option + Control + G: Obrir el menú de control de fonts Git.
  • Command + Option + Control + F: Formatjar el codi seleccionat.

Exemple

// Utilitza Command + Option + Control + F per formatar el codi
func greet(name: String) -> String {
    let greeting = "Hello, \(name)!"
    return greeting
}

Resum

Dominar les dreceres de teclat a Xcode pot millorar significativament la teva eficiència en la codificació i el teu flux de treball general. Començant amb dreceres bàsiques per a la gestió de fitxers i la navegació, progressant a dreceres intermèdies per a l'edició de codi i la documentació, i finalment, utilitzant dreceres avançades per a la gestió de projectes i el control de fonts, pots optimitzar el teu procés de desenvolupament. Practica aquestes dreceres regularment per esdevenir més proficient i estalviar temps valuós en les teves tasques de desenvolupament.
