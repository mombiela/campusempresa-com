La gestió de l'entrada d'usuari és un aspecte fonamental del desenvolupament d'aplicacions interactives. En aquesta secció, explorarem com gestionar l'entrada d'usuari a Xcode, des de camps de text bàsics fins al reconeixement avançat de gestos.

Entrada de Text Bàsica

UITextField

UITextField és un component bàsic de la interfície d'usuari que permet als usuaris introduir text. S'utilitza comunament per a formularis i barres de cerca.

  • Crear un UITextField: Pots afegir un UITextField al teu storyboard o crear-lo programàticament.
  • Configurar UITextField: Personalitza propietats com el text de marcador de posició, el tipus de teclat i el tipus de tecla de retorn.

Exemple: Crear un UITextField Programàticament

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Introdueix el teu nom"
        textField.borderStyle = .roundedRect
        self.view.addSubview(textField)
    }
}

Gestió de l'Entrada de Text

Per gestionar l'entrada de text, pots utilitzar el protocol UITextFieldDelegate.

  • Implementar UITextFieldDelegate: Conformar-se al protocol UITextFieldDelegate i implementar els seus mètodes.
  • Mètodes Comuns:
    • textFieldShouldReturn(_:): Es crida quan es prem el botó de retorn.
    • textField(_:shouldChangeCharactersIn:replacementString:): Es crida quan l'usuari escriu o esborra un caràcter.

Exemple: Gestió de la Tecla de Retorn

import UIKit

class ViewController: UIViewController, UITextFieldDelegate {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Introdueix el teu nom"
        textField.borderStyle = .roundedRect
        textField.delegate = self
        self.view.addSubview(textField)
    }
    
    func textFieldShouldReturn(_ textField: UITextField) -> Bool {
        textField.resignFirstResponder() // Amagar el teclat
        print("L'usuari ha introduït: \(textField.text ?? "")")
        return true
    }
}

Gestió Avançada de l'Entrada

Reconeixedors de Gestos

Els reconeixedors de gestos permeten detectar i respondre a esdeveniments tàctils com tocs, lliscaments i pessics.

  • Tipus de Reconeixedors de Gestos:
    • UITapGestureRecognizer
    • UIPinchGestureRecognizer
    • UISwipeGestureRecognizer
    • UILongPressGestureRecognizer

Exemple: Afegir un Reconeixedor de Toc

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let tapGesture = UITapGestureRecognizer(target: self, action: #selector(handleTap(_:)))
        self.view.addGestureRecognizer(tapGesture)
    }
    
    @objc func handleTap(_ sender: UITapGestureRecognizer) {
        print("Vista tocada")
    }
}

Vistes d'Entrada Personalitzades

Per a escenaris d'entrada més complexos, pots crear vistes d'entrada personalitzades.

  • Crear Vistes d'Entrada Personalitzades: Dissenya una vista personalitzada que es pugui utilitzar com a vista d'entrada.
  • Assignar Vistes d'Entrada Personalitzades: Assigna la vista personalitzada a la propietat inputView d'un responsable, com un UITextField.

Exemple: Selector de Data Personalitzat

import UIKit

class ViewController: UIViewController, UITextFieldDelegate {
    let datePicker = UIDatePicker()
    
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let textField = UITextField(frame: CGRect(x: 20, y: 100, width: 280, height: 40))
        textField.placeholder = "Selecciona una data"
        textField.borderStyle = .roundedRect
        textField.delegate = self
        textField.inputView = datePicker
        self.view.addSubview(textField)
        
        datePicker.datePickerMode = .date
        datePicker.addTarget(self, action: #selector(dateChanged(_:)), for: .valueChanged)
    }
    
    @objc func dateChanged(_ sender: UIDatePicker) {
        print("Data seleccionada: \(sender.date)")
    }
}

Conclusió

La gestió de l'entrada d'usuari és una habilitat crucial per a qualsevol desenvolupador d'iOS. Començant amb camps de text bàsics i progressant cap a reconeixedors de gestos avançats i vistes d'entrada personalitzades, pots crear aplicacions interactives i fàcils d'utilitzar. Practica aquests conceptes i explora les diverses tècniques de gestió d'entrada per millorar les teves habilitats de desenvolupament d'aplicacions a Xcode.
