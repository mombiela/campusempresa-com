Introducció

En aquesta secció, proporcionarem una visió general de com crear i gestionar projectes a Xcode, l'entorn de desenvolupament integrat (IDE) d'Apple per a macOS. Tant si ets un principiant com un usuari avançat, entendre l'estructura i els components d'un projecte Xcode és crucial per a un desenvolupament eficient.

Conceptes clau

  • Espai de treball Xcode: Un espai de treball pot contenir múltiples projectes i recursos compartits. És útil per gestionar aplicacions complexes amb múltiples objectius.
  • Projecte Xcode: La unitat fonamental de desenvolupament a Xcode. Cada projecte conté tots els fitxers i configuracions necessaris per construir la teva aplicació.
  • Objectius: Els objectius especifiquen el producte de sortida (per exemple, una aplicació, un marc) i les configuracions utilitzades per construir aquest producte.
  • Esquemes: Els esquemes defineixen una col·lecció d'objectius per construir, una configuració per utilitzar quan es construeix, i una col·lecció de proves per executar.
  • Configuracions de construcció: Opcions de configuració que controlen com es compila i enllaça el projecte.
  • Fitxers de codi: Els fitxers de codi actuals, incloent Swift, Objective-C, i altres recursos com imatges i fitxers de dades.

Crear un nou projecte

Per crear un nou projecte a Xcode, segueix aquests passos:

  1. Obre Xcode: Llença Xcode des de la teva carpeta d'Aplicacions.
  2. Crea un nou projecte: Selecciona "Create a new Xcode project" des de la pantalla de benvinguda o ves a File > New > Project.
  3. Tria una plantilla: Selecciona una plantilla que s'adapti a les necessitats del teu projecte. Per als principiants, la plantilla "App" sota la pestanya iOS és un bon punt de partida.
  4. Configura les opcions del projecte: Introdueix el nom del projecte, el nom de l'organització i l'identificador. Tria el llenguatge (Swift o Objective-C) i la interfície d'usuari (Storyboard o SwiftUI).
  5. Desa el projecte: Tria una ubicació per desar el teu projecte i fes clic a "Create".

Exemple: Crear una aplicació simple per a iOS

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Crear una etiqueta
        let label = UILabel()
        label.text = "Hello, Xcode!"
        label.textAlignment = .center
        label.frame = self.view.bounds
        
        // Afegir l'etiqueta a la vista
        self.view.addSubview(label)
    }
}

En aquest exemple, creem una aplicació simple per a iOS que mostra una etiqueta "Hello, Xcode!" al centre de la pantalla.

Gestionar els fitxers del projecte

  • Àrea de navegació: Utilitza l'àrea de navegació per gestionar els fitxers del teu projecte. El Navegador de Projectes (⌘1) és particularment útil per veure i organitzar fitxers.
  • Grups i carpetes: Organitza els teus fitxers en grups i carpetes per mantenir el teu projecte ordenat.
  • Afegir fitxers: Afegeix nous fitxers fent clic dret sobre un grup o carpeta i seleccionant New File.

Exemple: Afegir un nou fitxer Swift

  1. Fes clic dret sobre un grup: Al Navegador de Projectes, fes clic dret sobre el grup on vols afegir el nou fitxer.
  2. Selecciona Nou Fitxer: Tria New File des del menú contextual.
  3. Tria una plantilla: Selecciona Swift File i fes clic a Next.
  4. Nomena el fitxer: Introdueix un nom per al teu fitxer i fes clic a Create.

Configurar les configuracions de construcció

Les configuracions de construcció controlen com es compila i enllaça el teu projecte. Es poden configurar a nivell de projecte o d'objectiu.

  • Configuracions de projecte vs. objectiu: Les configuracions de projecte s'apliquen a tots els objectius, mentre que les configuracions d'objectiu sobreescriuen les configuracions de projecte per a un objectiu específic.
  • Configuracions de construcció comunes:
    • Base SDK: La versió de l'SDK utilitzada per construir el projecte.
    • Objectiu de desplegament: La versió mínima del sistema operatiu que l'aplicació suporta.
    • Nivell d'optimització: Controla el nivell d'optimització realitzat pel compilador.

Exemple: Configurar l'objectiu de desplegament

  1. Selecciona el projecte: Al Navegador de Projectes, selecciona el teu projecte.
  2. Selecciona l'objectiu: Tria l'objectiu que vols configurar.
  3. Informació de desplegament: A la pestanya General, estableix l'Objectiu de Desplegament a la versió del sistema operatiu desitjada.

Resum

En aquesta secció, hem cobert els conceptes bàsics de crear i gestionar projectes a Xcode. Hem discutit conceptes clau com espais de treball, projectes, objectius, esquemes i configuracions de construcció. També hem recorregut els passos per crear un nou projecte, afegir fitxers i configurar les configuracions de construcció. Entendre aquests fonaments t'ajudarà a navegar per Xcode de manera més eficient i establir una base sòlida per als teus projectes de desenvolupament.
