Introducció

Connectar la interfície d'usuari (UI) al codi subjacent és una habilitat fonamental en el desenvolupament d'iOS utilitzant Xcode. Aquest tema et guiarà a través del procés d'enllaçar elements de la UI creats a Interface Builder amb el teu codi Swift, permetent-te interactuar i manipular aquests elements programàticament.

Conceptes Clau

  • Interface Builder (IB): Una eina visual dins d'Xcode per dissenyar la UI de la teva aplicació.
  • IBOutlet: Una connexió d'un element de la UI a una propietat en el teu codi.
  • IBAction: Una connexió d'un element de la UI a un mètode en el teu codi, típicament utilitzat per gestionar interaccions d'usuari com tocs de botons.
  • ViewController: Una classe que gestiona una jerarquia de vistes per a la teva aplicació.

Configurant el Teu Projecte

  1. Crear un Nou Projecte:

    • Obre Xcode i crea un nou projecte de Single View App.
    • Dona un nom al teu projecte i selecciona Swift com a llenguatge.

  2. Dissenyant la UI:

    • Obre Main.storyboard.
    • Arrossega i deixa anar elements de la UI (per exemple, UILabel, UIButton) des de la Biblioteca d'Objectes fins al View Controller.

Connectant Elements de la UI al Codi

IBOutlet

Un IBOutlet s'utilitza per crear una referència a un element de la UI en el teu codi, permetent-te accedir-hi i modificar-lo programàticament.

  1. Crear un IBOutlet:
    • Obre l'Editor d'Assistent fent clic als dos cercles superposats a la part superior dreta d'Xcode.
    • Assegura't que Main.storyboard estigui obert en un costat i ViewController.swift a l'altre.
    • Control-arrossega des de l'element de la UI (per exemple, UILabel) en el storyboard fins al fitxer ViewController.swift.
    • Deixa anar l'arrossegament i dona un nom al teu IBOutlet (per exemple, myLabel).
@IBOutlet weak var myLabel: UILabel!
  1. Utilitzant l'IBOutlet:
    • Ara pots accedir i modificar el UILabel en el teu codi.
override func viewDidLoad() {
    super.viewDidLoad()
    myLabel.text = "Hola, Món!"
}

IBAction

Un IBAction s'utilitza per connectar un element de la UI a un mètode en el teu codi, típicament per gestionar interaccions d'usuari.

  1. Crear un IBAction:
    • Control-arrossega des de l'element de la UI (per exemple, UIButton) en el storyboard fins al fitxer ViewController.swift.
    • Deixa anar l'arrossegament i selecciona "Action" des del menú desplegable de Connexió.
    • Dona un nom al teu IBAction (per exemple, buttonTapped).
@IBAction func buttonTapped(_ sender: UIButton) {
    myLabel.text = "S'ha tocat el botó!"
}
  1. Connectant l'IBAction:
    • Assegura't que el UIButton estigui connectat al mètode buttonTapped.
    • Quan es toqui el botó, es cridarà el mètode i el text del UILabel canviarà.

Exemple: Connectant un Camp de Text i un Botó

  1. Afegeix Elements de la UI:

    • Arrossega un UITextField i un UIButton al View Controller en Main.storyboard.

  2. Crea IBOutlets i IBActions:

    • Control-arrossega des del UITextField fins a ViewController.swift i crea un IBOutlet anomenat myTextField.
    • Control-arrossega des del UIButton fins a ViewController.swift i crea un IBAction anomenat submitButtonTapped.
@IBOutlet weak var myTextField: UITextField!

@IBAction func submitButtonTapped(_ sender: UIButton) {
    if let text = myTextField.text {
        myLabel.text = text
    }
}
  1. Executa l'Aplicació:
    • Introdueix text al UITextField i toca el UIButton.
    • El UILabel s'actualitzarà per mostrar el text introduït.

Conclusió

Connectar elements de la UI al codi en Xcode és una habilitat crucial per als desenvolupadors d'iOS. Utilitzant IBOutlets i IBActions, pots interactuar i manipular elements de la UI programàticament, creant aplicacions dinàmiques i responsives. Practica aquests passos per convertir-te en un expert en enllaçar la teva UI al teu codi, i explora interaccions més avançades a mesura que progressis.
