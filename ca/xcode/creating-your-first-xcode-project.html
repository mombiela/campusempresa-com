Introducció

Crear el teu primer projecte Xcode és un pas essencial per convertir-te en un expert en desenvolupament iOS. Aquesta secció et guiarà a través del procés de configuració d'un nou projecte, entendre l'estructura del projecte i executar la teva primera aplicació en el simulador.

Passos per crear un nou projecte Xcode

1. Obre Xcode

  • Llença Xcode des de la teva carpeta d'Aplicacions o Dock.

2. Comença un nou projecte

  • Selecciona "Create a new Xcode project" des de la pantalla de benvinguda.
  • Alternativament, pots anar a File > New > Project.

3. Tria una plantilla

  • Xcode proporciona diverses plantilles per a diferents tipus d'aplicacions. Per a aquest tutorial, selecciona la plantilla "App" sota la pestanya iOS.
  • Fes clic a "Next".

4. Configura el teu projecte

  • Product Name: Introdueix un nom per al teu projecte. Aquest serà el nom de la teva aplicació.
  • Team: Selecciona el teu equip de desenvolupament. Si no en tens un, pots crear un Apple ID gratuït.
  • Organization Name: Introdueix el nom de la teva organització o el teu nom.
  • Organization Identifier: Normalment és un nom de domini invers (per exemple, com.example).
  • Bundle Identifier: Aquest es genera automàticament basat en el nom del producte i l'identificador de l'organització.
  • Language: Tria Swift.
  • User Interface: Tria SwiftUI o UIKit. Per a aquest tutorial, utilitzarem SwiftUI.
  • Include Tests: Opcionalment, pots incloure proves unitàries i de UI.
  • Fes clic a "Next".

5. Tria una ubicació

  • Selecciona una ubicació al teu ordinador per desar el projecte.
  • Fes clic a "Create".

Entendre l'estructura del projecte

Project Navigator

  • Situat al costat esquerre de Xcode, mostra tots els fitxers del teu projecte.
  • Fitxers clau inclouen:
    • AppDelegate.swift: Gestiona els esdeveniments del cicle de vida de l'aplicació.
    • SceneDelegate.swift: Gestiona les escenes (finestres de la UI).
    • ContentView.swift: La vista principal de la teva aplicació.

Editor Area

  • L'àrea central on escrius i edites el teu codi.
  • Mostra el contingut del fitxer seleccionat al Project Navigator.

Inspectors

  • Situats al costat dret, aquests panells proporcionen informació addicional i configuracions per al fitxer o element de la UI seleccionat.

Toolbar

  • Conté botons per executar la teva aplicació, aturar-la i accedir a altres eines.

Debug Area

  • Situada a la part inferior, mostra registres, errors i et permet interactuar amb el depurador.

Executar la teva aplicació

Selecciona un simulador

  • A la barra d'eines, fes clic al selector de dispositius i tria un simulador (per exemple, iPhone 14).

Executa l'aplicació

  • Fes clic al botó "Run" (una icona de reproducció) a la barra d'eines.
  • Xcode construirà el teu projecte i el llençarà en el simulador seleccionat.

Veure la sortida

  • El simulador s'obrirà i hauràs de veure la teva aplicació en execució.
  • Qualsevol declaració de print o registres apareixeran a la Debug Area.

Exemple de codi

Aquí tens un exemple senzill de com podria ser el teu ContentView.swift:

import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hello, World!")
            .padding()
    }
}

struct ContentView_Previews: PreviewProvider {
    static var previews: some View {
        ContentView()
    }
}

Explicació

  • import SwiftUI: Importa el framework SwiftUI.
  • struct ContentView: View: Defineix una nova vista.
  • var body: some View: La propietat body descriu el contingut i el disseny de la vista.
  • Text("Hello, World!"): Mostra una etiqueta de text.
  • .padding(): Afegeix espai al voltant del text.

Conclusió

Felicitats! Has creat amb èxit el teu primer projecte Xcode, has explorat la seva estructura i has executat una aplicació senzilla en el simulador. Aquest coneixement fonamental serà crucial a mesura que avancis en temes més avançats de desenvolupament iOS. Continua experimentant amb diferents plantilles i funcions per familiaritzar-te més amb Xcode.
