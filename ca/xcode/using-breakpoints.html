Introducció als Punts de Trencament

Els punts de trencament són una eina essencial en el depurament, ja que et permeten pausar l'execució del teu codi en punts específics per inspeccionar l'estat de la teva aplicació. A Xcode, els punts de trencament t'ajuden a entendre el flux del teu programa i identificar problemes.

Conceptes Clau

  • Punt de Trencament: Un marcador que estableixes en una línia de codi on vols que l'execució es pausi.
  • Depurador: Una eina que et permet executar el teu programa pas a pas, inspeccionar variables i avaluar expressions.
  • Pila de Crides: Un informe que mostra les subrutines o funcions actives en el teu programa en un moment específic.

Establir Punts de Trencament

Per establir un punt de trencament a Xcode:

  1. Obre el teu projecte a Xcode.
  2. Navega fins a la línia de codi on vols establir el punt de trencament.
  3. Fes clic a la canaleta (l'àrea a l'esquerra dels números de línia) al costat de la línia de codi.

Exemple

func calculateSum(a: Int, b: Int) -> Int {
    let sum = a + b // Estableix un punt de trencament aquí
    return sum
}

En aquest exemple, fent clic a la canaleta al costat de let sum = a + b establirà un punt de trencament en aquesta línia.

Gestionar Punts de Trencament

Xcode proporciona diverses opcions per gestionar els punts de trencament:

  • Habilitar/Deshabilitar Punts de Trencament: Fes clic dret sobre el punt de trencament i selecciona "Enable Breakpoint" o "Disable Breakpoint".
  • Eliminar Punts de Trencament: Arrossega el punt de trencament fora de la canaleta o fes clic dret i selecciona "Delete Breakpoint".
  • Navegador de Punts de Trencament: Utilitza el Navegador de Punts de Trencament (⌘8) per veure i gestionar tots els punts de trencament del teu projecte.

Funcions Avançades de Punts de Trencament

Punts de Trencament Condicionals

Els punts de trencament condicionals pausen l'execució només quan una condició especificada és certa. Això és útil per depurar bucles o quan necessites aturar l'execució sota circumstàncies específiques.

Exemple

for i in 0..<10 {
    print(i) // Estableix un punt de trencament condicional aquí amb la condició: i == 5
}

Per establir un punt de trencament condicional:

  1. Fes clic dret sobre el punt de trencament.
  2. Selecciona "Edit Breakpoint".
  3. Introdueix la condició (per exemple, i == 5).

Punts de Trencament d'Acció

Els punts de trencament d'acció realitzen una acció quan es troben, com ara registrar un missatge o executar un script, sense necessàriament pausar l'execució.

Exemple

func fetchData() {
    // Estableix un punt de trencament d'acció aquí per registrar un missatge
    print("Fetching data...")
}

Per establir un punt de trencament d'acció:

  1. Fes clic dret sobre el punt de trencament.
  2. Selecciona "Edit Breakpoint".
  3. Fes clic al botó "+" per afegir una acció (per exemple, "Log Message").

Ús del Depurador

Quan el teu programa troba un punt de trencament, Xcode obrirà el depurador. Aquí pots:

  • Inspeccionar Variables: Passa el cursor sobre les variables per veure els seus valors o utilitza la Vista de Variables.
  • Pas Sobre: Executa la següent línia de codi (F6).
  • Pas a l'Interior: Entra en la crida de funció de la línia actual (F7).
  • Pas a l'Exterior: Completa la funció actual i torna al cridant (F8).

Exemple

func multiply(a: Int, b: Int) -> Int {
    let result = a * b // Inspecciona 'result' aquí
    return result
}

Quan el punt de trencament es trobi, pots inspeccionar el valor de result en el depurador.

Conclusió

Els punts de trencament són una característica poderosa a Xcode que t'ajuden a depurar les teves aplicacions de manera efectiva. Establint, gestionant i utilitzant funcions avançades de punts de trencament, pots obtenir una comprensió més profunda del comportament del teu codi i identificar i solucionar problemes ràpidament. Practica l'ús de punts de trencament en diferents escenaris per convertir-te en un expert en depuració amb Xcode.
