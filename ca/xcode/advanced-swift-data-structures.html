Introducció

En aquesta secció, ens endinsarem en estructures de dades avançades en Swift. Entendre aquestes estructures t'ajudarà a escriure codi més eficient i optimitzat. Cobrirem:

  • Llistes Enllaçades
  • Arbres
  • Grafs
  • Taules de Hash
  • Heaps

Llistes Enllaçades

Una llista enllaçada és una estructura de dades lineal on cada element és un objecte separat, conegut com a node. Cada node conté dades i una referència (o enllaç) al següent node en la seqüència.

Conceptes Clau

  • Node: La unitat bàsica d'una llista enllaçada, que conté dades i una referència al següent node.
  • Cap: El primer node de la llista enllaçada.
  • Cua: L'últim node de la llista enllaçada, que apunta a nil.

Exemple de Codi

class Node<T> {
    var value: T
    var next: Node?

    init(value: T) {
        self.value = value
    }
}

class LinkedList<T> {
    var head: Node<T>?

    func append(value: T) {
        let newNode = Node(value: value)
        if let tailNode = head {
            var currentNode = tailNode
            while let nextNode = currentNode.next {
                currentNode = nextNode
            }
            currentNode.next = newNode
        } else {
            head = newNode
        }
    }
}

Explicació

  • Classe Node: Defineix un node amb un valor i una referència al següent node.
  • Classe LinkedList: Gestiona la llista enllaçada, amb mètodes per afegir nous nodes.

Arbres

Un arbre és una estructura de dades jeràrquica que consisteix en nodes, amb un sol node com a arrel i subnodes com a fills.

Conceptes Clau

  • Arrel: El node superior de l'arbre.
  • Fulla: Un node sense fills.
  • Arbre Binari: Un arbre on cada node té com a màxim dos fills.

Exemple de Codi

class TreeNode<T> {
    var value: T
    var left: TreeNode?
    var right: TreeNode?

    init(value: T) {
        self.value = value
    }
}

class BinaryTree<T> {
    var root: TreeNode<T>?

    func insert(value: T) {
        let newNode = TreeNode(value: value)
        if let rootNode = root {
            self.insertNode(rootNode, newNode)
        } else {
            root = newNode
        }
    }

    private func insertNode(_ node: TreeNode<T>, _ newNode: TreeNode<T>) {
        if newNode.value < node.value {
            if let leftNode = node.left {
                insertNode(leftNode, newNode)
            } else {
                node.left = newNode
            }
        } else {
            if let rightNode = node.right {
                insertNode(rightNode, newNode)
            } else {
                node.right = newNode
            }
        }
    }
}

Explicació

  • Classe TreeNode: Defineix un node amb un valor i referències als fills esquerre i dret.
  • Classe BinaryTree: Gestiona l'arbre binari, amb mètodes per inserir nous nodes.

Grafs

Un graf és una col·lecció de nodes (vèrtexs) i arestes que els connecten. Els grafs poden ser dirigits o no dirigits.

Conceptes Clau

  • Vèrtex: Un node en el graf.
  • Aresta: Una connexió entre dos vèrtexs.
  • Graf Dirigit: Les arestes tenen una direcció.
  • Graf No Dirigit: Les arestes no tenen una direcció.

Exemple de Codi

class Vertex<T> {
    var value: T
    var edges: [Edge<T>] = []

    init(value: T) {
        self.value = value
    }
}

class Edge<T> {
    var from: Vertex<T>
    var to: Vertex<T>

    init(from: Vertex<T>, to: Vertex<T>) {
        self.from = from
        self.to = to
    }
}

class Graph<T> {
    var vertices: [Vertex<T>] = []

    func addVertex(value: T) -> Vertex<T> {
        let vertex = Vertex(value: value)
        vertices.append(vertex)
        return vertex
    }

    func addEdge(from: Vertex<T>, to: Vertex<T>) {
        let edge = Edge(from: from, to: to)
        from.edges.append(edge)
    }
}

Explicació

  • Classe Vertex: Defineix un vèrtex amb un valor i una llista d'arestes.
  • Classe Edge: Defineix una aresta amb referències als vèrtexs d'inici i final.
  • Classe Graph: Gestiona el graf, amb mètodes per afegir vèrtexs i arestes.

Taules de Hash

Una taula de hash és una estructura de dades que associa claus a valors per a una cerca eficient.

Conceptes Clau

  • Funció de Hash: Converteix una clau en un índex a la taula de hash.
  • Col·lisió: Quan dues claus es transformen en el mateix índex.
  • Encadenament: Un mètode per gestionar col·lisions emmagatzemant múltiples elements en el mateix índex.

Exemple de Codi

class HashTable<Key: Hashable, Value> {
    private var buckets: [[(key: Key, value: Value)]]
    private let capacity: Int

    init(capacity: Int) {
        self.capacity = capacity
        self.buckets = Array(repeating: [], count: capacity)
    }

    func insert(key: Key, value: Value) {
        let index = self.index(for: key)
        buckets[index].append((key: key, value: value))
    }

    func value(for key: Key) -> Value? {
        let index = self.index(for: key)
        for element in buckets[index] {
            if element.key == key {
                return element.value
            }
        }
        return nil
    }

    private func index(for key: Key) -> Int {
        return abs(key.hashValue) % capacity
    }
}

Explicació

  • Classe HashTable: Gestiona la taula de hash amb mètodes per inserir i recuperar valors.
  • Mètode index(for:): Calcula l'índex per a una clau donada utilitzant una funció de hash.

Heaps

Un heap és una estructura de dades basada en arbres especialitzada que satisfà la propietat de heap.

Conceptes Clau

  • Max-Heap: El node pare és sempre més gran o igual que els nodes fills.
  • Min-Heap: El node pare és sempre més petit o igual que els nodes fills.

Exemple de Codi

class Heap<T: Comparable> {
    var elements: [T] = []

    func insert(_ value: T) {
        elements.append(value)
        siftUp(from: elements.count - 1)
    }

    private func siftUp(from index: Int) {
        var childIndex = index
        let childValue = elements[childIndex]
        var parentIndex = (childIndex - 1) / 2

        while childIndex > 0 && childValue > elements[parentIndex] {
            elements[childIndex] = elements[parentIndex]
            childIndex = parentIndex
            parentIndex = (childIndex - 1) / 2
        }
        elements[childIndex] = childValue
    }
}

Explicació

  • Classe Heap: Gestiona el heap amb mètodes per inserir elements.
  • Mètode siftUp(from:): Assegura que la propietat de heap es mantingui després de la inserció.
