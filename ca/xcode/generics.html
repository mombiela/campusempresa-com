Els genèrics són una característica poderosa a Swift que et permeten escriure codi flexible i reutilitzable. Et permeten escriure funcions i tipus que poden treballar amb qualsevol tipus, subjecte als requisits que defineixis. Aquesta secció et guiarà des dels conceptes bàsics dels genèrics fins a l'ús avançat a Xcode.

Introducció als Genèrics

Els genèrics et permeten escriure codi que funciona amb qualsevol tipus. Aquí tens un exemple senzill per il·lustrar el concepte:

func swapTwoValues<T>(_ a: inout T, _ b: inout T) {
    let temporaryA = a
    a = b
    b = temporaryA
}
  • T és un nom de tipus de marcador de posició, indicant que swapTwoValues pot treballar amb qualsevol tipus T.
  • La funció intercanvia els valors de a i b, independentment del seu tipus.

Conceptes Clau

  • Paràmetres de Tipus: Marcadors de posició per a tipus que es especifiquen quan s'utilitza la funció o el tipus.
  • Funcions Genèriques: Funcions que poden operar amb qualsevol tipus.
  • Tipus Genèrics: Tipus (com classes, structs i enums) que poden treballar amb qualsevol tipus.

Funcions Genèriques

Les funcions genèriques poden operar amb qualsevol tipus. Aquí tens un exemple més detallat:

func findIndex<T: Equatable>(of valueToFind: T, in array: [T]) -> Int? {
    for (index, value) in array.enumerated() {
        if value == valueToFind {
            return index
        }
    }
    return nil
}
  • T: Equatable especifica que el tipus T ha de conformar-se al protocol Equatable.
  • Aquesta funció busca un valor en un array i retorna el seu índex.

Exercici

Escriu una funció genèrica que retorni el valor màxim en un array:

func findMaximumValue<T: Comparable>(in array: [T]) -> T? {
    guard !array.isEmpty else { return nil }
    var maxValue = array[0]
    for value in array {
        if value > maxValue {
            maxValue = value
        }
    }
    return maxValue
}

Tipus Genèrics

Pots definir els teus propis tipus genèrics per crear components flexibles i reutilitzables. Aquí tens un exemple amb una pila genèrica:

struct Stack<Element> {
    var items = [Element]()
    
    mutating func push(_ item: Element) {
        items.append(item)
    }
    
    mutating func pop() -> Element? {
        return items.popLast()
    }
}
  • Element és un nom de tipus de marcador de posició.
  • El struct Stack pot emmagatzemar elements de qualsevol tipus.

Exercici

Crea un tipus de cua genèrica:

struct Queue<Element> {
    var items = [Element]()
    
    mutating func enqueue(_ item: Element) {
        items.append(item)
    }
    
    mutating func dequeue() -> Element? {
        guard !items.isEmpty else { return nil }
        return items.removeFirst()
    }
}

Restriccions de Tipus

Les restriccions de tipus especifiquen que un paràmetre de tipus ha d'heretar d'una classe específica o conformar-se a un protocol particular. Això assegura que la funció o tipus genèric només treballi amb tipus que compleixin certs criteris.

Exemple amb Restriccions de Tipus

func allItemsMatch<C1: Container, C2: Container>(_ container1: C1, _ container2: C2) -> Bool
where C1.Item == C2.Item, C1.Item: Equatable {
    if container1.count != container2.count {
        return false
    }
    
    for i in 0..<container1.count {
        if container1[i] != container2[i] {
            return false
        }
    }
    
    return true
}
  • C1 i C2 són tipus genèrics que es conformen al protocol Container.
  • C1.Item == C2.Item assegura que ambdós contenidors continguin el mateix tipus d'elements.
  • C1.Item: Equatable assegura que els elements es puguin comparar per igualtat.

Genèrics Avançats

Tipus Associats

Els protocols poden tenir tipus associats, que són marcadors de posició per a tipus utilitzats en el protocol. Aquí tens un exemple:

protocol Container {
    associatedtype Item
    var count: Int { get }
    subscript(i: Int) -> Item { get }
}
  • associatedtype Item és un marcador de posició per al tipus d'elements en el contenidor.

Subscripts Genèrics

Pots definir subscripts genèrics en els teus tipus. Aquí tens un exemple:

extension Array {
    subscript<Indices: Sequence>(indices: Indices) -> [Element]
    where Indices.Iterator.Element == Int {
        var result = [Element]()
        for index in indices {
            result.append(self[index])
        }
        return result
    }
}
  • Aquest subscript pren una seqüència d'índexs i retorna un array d'elements en aquests índexs.

Conclusió

Els genèrics són una característica poderosa a Swift que et permeten escriure codi flexible i reutilitzable. Entenent i utilitzant els genèrics, pots crear funcions i tipus que treballin amb qualsevol tipus, subjecte a les restriccions que defineixis. Això fa que el teu codi sigui més robust i fàcil de mantenir.

Resum

  • Els genèrics et permeten escriure codi que funciona amb qualsevol tipus.
  • Les funcions i tipus genèrics poden operar amb qualsevol tipus.
  • Les restriccions de tipus asseguren que les funcions i tipus genèrics només treballin amb tipus que compleixin certs criteris.
  • Els genèrics avançats inclouen tipus associats i subscripts genèrics.

Dominant els genèrics, pots millorar significativament les teves habilitats de programació en Swift i escriure codi més eficient i reutilitzable.
