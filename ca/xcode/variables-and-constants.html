Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i constants a Xcode utilitzant Swift. Entendre com declarar i utilitzar variables i constants és crucial per a qualsevol programador, ja que formen la base per emmagatzemar i manipular dades en les teves aplicacions.

Variables

Les variables s'utilitzen per emmagatzemar dades que poden canviar durant l'execució d'un programa. En Swift, les variables es declaren utilitzant la paraula clau var.

Declaració de Variables

Per declarar una variable, utilitzes la paraula clau var seguida del nom de la variable i una anotació de tipus opcional.

var greeting: String = "Hello, world!"
  • var és la paraula clau utilitzada per declarar una variable.
  • greeting és el nom de la variable.
  • String és el tipus de la variable.
  • "Hello, world!" és el valor inicial assignat a la variable.

Inferència de Tipus

Swift pot inferir el tipus de la variable basant-se en el valor inicial, així que pots ometre l'anotació de tipus.

var age = 25

En aquest exemple, Swift infereix que age és un Int perquè el valor inicial és un enter.

Canvi de Valors de Variables

Pots canviar el valor d'una variable després que hagi estat declarada.

var score = 10
score = 20

Constants

Les constants s'utilitzen per emmagatzemar dades que no haurien de canviar durant l'execució d'un programa. En Swift, les constants es declaren utilitzant la paraula clau let.

Declaració de Constants

Per declarar una constant, utilitzes la paraula clau let seguida del nom de la constant i una anotació de tipus opcional.

let pi: Double = 3.14159
  • let és la paraula clau utilitzada per declarar una constant.
  • pi és el nom de la constant.
  • Double és el tipus de la constant.
  • 3.14159 és el valor inicial assignat a la constant.

Inferència de Tipus

Igual que amb les variables, Swift pot inferir el tipus de la constant basant-se en el valor inicial.

let name = "Alice"

En aquest exemple, Swift infereix que name és un String perquè el valor inicial és una cadena de text.

Intentar Canviar Valors de Constants

Si intentes canviar el valor d'una constant després que hagi estat declarada, Swift produirà un error en temps de compilació.

let birthYear = 1990
// birthYear = 1991 // Això causarà un error

Diferències Clau Entre Variables i Constants

| Característica | Variables (var) | Constants (let) | |----------------|-------------------|-------------------| | Mutabilitat | Mutable | Immutable | | Declaració | Paraula clau var| Paraula clau let| | Cas d'Ús | Dades que canvien | Dades que romanen constants |

Millors Pràctiques

  • Utilitza constants (let) per defecte a menys que sàpigues que un valor necessita canviar.
  • Tria noms descriptius per a les teves variables i constants per fer el teu codi més llegible.
  • Utilitza anotacions de tipus quan el tipus no és clar des del context o quan vols ser explícit sobre el tipus.

Exercicis

  1. Declara una variable anomenada temperature i assigna-li el valor 72. Després, canvia el seu valor a 68.
  2. Declara una constant anomenada maximumSpeed i assigna-li el valor 120. Intenta canviar el seu valor i observa l'error.
  3. Declara una variable sense una anotació de tipus i assigna-li un nombre de punt flotant. Deixa que Swift infereixi el tipus.

Conclusió

Entendre les variables i constants és essencial per gestionar dades en les teves aplicacions Swift. Les variables et permeten emmagatzemar i modificar dades, mentre que les constants asseguren que certs valors romanen inalterats. Seguint les millors pràctiques i utilitzant el tipus adequat per a les teves dades, pots escriure codi més robust i mantenible.
