Introducció a les Stack Views

Les Stack Views són una eina poderosa a Xcode per crear interfícies d'usuari flexibles i adaptatives. Et permeten gestionar un grup de vistes en una pila horitzontal o vertical, gestionant automàticament el disseny i l'espai entre elles.

Conceptes Clau

  • Stack View: Una vista contenidora que disposa les seves vistes filles en una línia horitzontal o vertical.
  • Eix: Determina la direcció de la pila (horitzontal o vertical).
  • Distribució: Defineix com la stack view distribueix la mida de les seves vistes filles.
  • Alineació: Especifica com s'alineen les vistes filles dins de la stack view.
  • Espaiat: L'espai entre les vistes filles.

Creació d'una Stack View

Guia Pas a Pas

  1. Obre Xcode i Crea un Nou Projecte:

    • Selecciona "App" sota la pestanya iOS.
    • Tria "Storyboard" com a interfície.

  2. Afegeix una Stack View al Teu Storyboard:

    • Obre la Biblioteca d'Objectes (⌘ + Shift + L).
    • Arrossega una Stack View al teu View Controller.

  3. Configura la Stack View:

    • Configura l'eix a horitzontal o vertical.
    • Ajusta les propietats de distribució, alineació i espaiat.

Exemple de Codi

Aquí tens un exemple senzill de creació d'una stack view vertical programàticament:

import UIKit

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Crear una stack view
        let stackView = UIStackView()
        stackView.axis = .vertical
        stackView.distribution = .fillEqually
        stackView.alignment = .center
        stackView.spacing = 10
        
        // Afegir la stack view a la vista principal
        view.addSubview(stackView)
        
        // Establir les restriccions per a la stack view
        stackView.translatesAutoresizingMaskIntoConstraints = false
        NSLayoutConstraint.activate([
            stackView.centerXAnchor.constraint(equalTo: view.centerXAnchor),
            stackView.centerYAnchor.constraint(equalTo: view.centerYAnchor),
            stackView.widthAnchor.constraint(equalTo: view.widthAnchor, multiplier: 0.8),
            stackView.heightAnchor.constraint(equalTo: view.heightAnchor, multiplier: 0.5)
        ])
        
        // Afegir subviews ordenades
        for i in 1...3 {
            let label = UILabel()
            label.text = "Etiqueta \(i)"
            label.textAlignment = .center
            stackView.addArrangedSubview(label)
        }
    }
}

Explicació

  • UIStackView: Inicialitza una nova stack view.
  • axis: Configura la stack view per disposar les seves subviews verticalment.
  • distribution: Assegura que cada subview tingui la mateixa mida.
  • alignment: Centra les subviews dins de la stack view.
  • spacing: Configura l'espai entre cada subview a 10 punts.
  • translatesAutoresizingMaskIntoConstraints: Desactiva les restriccions de màscara de redimensionament automàtic.
  • NSLayoutConstraint.activate: Activa les restriccions per posicionar la stack view.

Personalització Avançada de Stack Views

Stack Views Niuades

Pots niuar stack views per crear dissenys complexos. Per exemple, una stack view vertical que conté stack views horitzontals.

let outerStackView = UIStackView()
outerStackView.axis = .vertical
outerStackView.spacing = 20

let innerStackView1 = UIStackView()
innerStackView1.axis = .horizontal
innerStackView1.spacing = 10

let innerStackView2 = UIStackView()
innerStackView2.axis = .horizontal
innerStackView2.spacing = 10

outerStackView.addArrangedSubview(innerStackView1)
outerStackView.addArrangedSubview(innerStackView2)

// Afegir etiquetes a les stack views internes
per i en 1...2 {
    let label1 = UILabel()
    label1.text = "Interna 1 - Etiqueta \(i)"
    innerStackView1.addArrangedSubview(label1)
    
    let label2 = UILabel()
    label2.text = "Interna 2 - Etiqueta \(i)"
    innerStackView2.addArrangedSubview(label2)
}

// Afegir la stack view externa a la vista principal
view.addSubview(outerStackView)
outerStackView.translatesAutoresizingMaskIntoConstraints = false
NSLayoutConstraint.activate([
    outerStackView.centerXAnchor.constraint(equalTo: view.centerXAnchor),
    outerStackView.centerYAnchor.constraint(equalTo: view.centerYAnchor),
    outerStackView.widthAnchor.constraint(equalTo: view.widthAnchor, multiplier: 0.8)
])

Stack Views Dinàmiques

Pots afegir o eliminar vistes d'una stack view dinàmicament en temps d'execució.

let newLabel = UILabel()
newLabel.text = "Nova Etiqueta"
stackView.addArrangedSubview(newLabel)

// Per eliminar una vista
stackView.removeArrangedSubview(newLabel)
newLabel.removeFromSuperview()

Conclusió

Les Stack Views a Xcode proporcionen una manera flexible i poderosa de gestionar el disseny de la teva aplicació. Entenent les propietats bàsiques i com configurar-les, pots crear interfícies d'usuari complexes i adaptatives amb facilitat. Practica creant i personalitzant stack views per convertir-te en un expert en la construcció de dissenys responsius.
