La refactorització és el procés de reestructurar el codi existent sense canviar-ne el comportament extern. Millora el disseny, l'estructura i la implementació del programari, fent-lo més fàcil d'entendre i mantenir. En aquest tema, cobrirem els conceptes bàsics de la refactorització a Xcode, progressant cap a tècniques més avançades.

Introducció a la Refactorització

  • Definició: La refactorització és el procés de millorar l'estructura interna del codi sense alterar-ne el comportament extern.
  • Propòsit: Millora la llegibilitat del codi, redueix la complexitat i fa que la base de codi sigui més fàcil de mantenir i ampliar.
  • Quan Refactoritzar: Regularment durant el desenvolupament, quan s'afegeixen noves funcionalitats, es corregeixen errors o es millora el rendiment.

Tècniques Bàsiques de Refactorització

Canviar el Nom

Canviar el nom de variables, funcions, classes i altres identificadors a noms més significatius.

// Abans de canviar el nom
var a = 10
func calc() {
    // alguns càlculs
}

// Després de canviar el nom
var itemCount = 10
func calculateTotal() {
    // alguns càlculs
}
  • Com Canviar el Nom a Xcode:
    1. Selecciona l'identificador que vols canviar.
    2. Fes clic dret i tria "Refactor" > "Rename".
    3. Introdueix el nou nom i prem Enter.

Extreure Mètodes

Descompondre mètodes grans en altres més petits i manejables.

// Abans d'extreure
func processOrder() {
    validateOrder()
    calculateTotal()
    applyDiscount()
    printReceipt()
}

// Després d'extreure
func processOrder() {
    validateOrder()
    calculateTotal()
    applyDiscount()
    printReceipt()
}

func validateOrder() {
    // lògica de validació
}

func calculateTotal() {
    // lògica de càlcul
}

func applyDiscount() {
    // lògica de descompte
}

func printReceipt() {
    // lògica d'impressió
}
  • Com Extreure Mètodes a Xcode:
    1. Selecciona el bloc de codi que vols extreure.
    2. Fes clic dret i tria "Refactor" > "Extract to Method".
    3. Dona un nom al nou mètode i prem Enter.

Tècniques Intermèdies de Refactorització

Moure Codi

Reorganitzar el codi movent mètodes o classes a ubicacions més apropiades.

// Abans de moure
class Order {
    func validateOrder() {
        // lògica de validació
    }
}

class OrderValidator {
    // Classe buida
}

// Després de moure
class Order {
    // Mètodes relacionats amb Order
}

class OrderValidator {
    func validateOrder() {
        // lògica de validació
    }
}
  • Com Moure Codi a Xcode:
    1. Selecciona el mètode o classe que vols moure.
    2. Talla el codi seleccionat.
    3. Navega a la nova ubicació i enganxa el codi.

Inlining de Mètodes

Substituir les crides a mètodes pel cos del mètode per simplificar el codi.

// Abans d'inlining
func calculateTotal() {
    let total = subtotal + tax
    return total
}

let orderTotal = calculateTotal()

// Després d'inlining
let orderTotal = subtotal + tax
  • Com Fer Inlining de Mètodes a Xcode:
    1. Selecciona la crida al mètode que vols fer inline.
    2. Fes clic dret i tria "Refactor" > "Inline".
    3. Confirma l'operació d'inlining.

Tècniques Avançades de Refactorització

Descomposar Condicionals

Simplificar declaracions condicionals complexes descomponent-les en mètodes més petits.

// Abans de descomposar
if (order.isValid && order.total > 100 && customer.isPremium) {
    applyDiscount()
}

// Després de descomposar
if (isEligibleForDiscount(order, customer)) {
    applyDiscount()
}

func isEligibleForDiscount(order: Order, customer: Customer) -> Bool {
    return order.isValid && order.total > 100 && customer.isPremium
}
  • Com Descomposar Condicionals a Xcode:
    1. Identifica la declaració condicional complexa.
    2. Extreu la condició en un mètode separat.

Encapsular Camps

Restringir l'accés directe als camps utilitzant mètodes getter i setter.

// Abans d'encapsular
class Customer {
    var name: String
}

// Després d'encapsular
class Customer {
    private var _name: String

    var name: String {
        get {
            return _name
        }
        set {
            _name = newValue
        }
    }
}
  • Com Encapsular Camps a Xcode:
    1. Canvia el nivell d'accés del camp a privat.
    2. Crea mètodes getter i setter per al camp.

Conclusió

La refactorització és una habilitat crucial per mantenir una base de codi saludable. Aplicant regularment tècniques de refactorització, pots assegurar-te que el teu codi es mantingui net, comprensible i fàcil de mantenir. Xcode proporciona diverses eines integrades per ajudar-te a refactoritzar el teu codi de manera eficient, des de canviar noms i extreure mètodes fins a tècniques més avançades com descomposar condicionals i encapsular camps. Practica aquestes tècniques regularment per convertir-te en un expert en escriure codi d'alta qualitat.
