Configuració del Projecte
Implementació de Funcionalitats ►

Configurar correctament el teu projecte Groovy és el primer pas per a una experiència de desenvolupament fluida. Aquesta secció et guiarà a través del procés de configuració d'un projecte Groovy des de zero, assegurant que tinguis totes les eines i configuracions necessàries en lloc.

Prerequisits

Abans de començar, assegura't de tenir instal·lat al teu sistema:

  • Java Development Kit (JDK) 8 o superior
  • Groovy SDK
  • Un Entorn de Desenvolupament Integrat (IDE) com IntelliJ IDEA, Eclipse, o Visual Studio Code

Instal·lació de Groovy

Utilitzant SDKMAN

SDKMAN és una eina per gestionar versions paral·leles de múltiples Kits de Desenvolupament de Programari en la majoria de sistemes basats en Unix. És la manera més fàcil d'instal·lar Groovy.

  1. Instal·lar SDKMAN:

    curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"

  2. Instal·lar Groovy:

    sdk install groovy

  3. Verificar la Instal·lació:

    groovy -version

Instal·lació Manual

Si prefereixes instal·lar Groovy manualment:

  1. Descarregar Groovy:

    • Ves a la Pàgina de Descàrregues de Groovy i descarrega la distribució binària.

  2. Extreure l'Arxiu:

    tar -xvf groovy-<version>.zip

  3. Configurar Variables d'Entorn:

    • Afegeix Groovy al teu PATH i configura GROOVY_HOME al teu .bashrc, .zshrc, o fitxer equivalent:
    export GROOVY_HOME=/path/to/groovy
export PATH=$PATH:$GROOVY_HOME/bin

  4. Verificar la Instal·lació:

    groovy -version

Configuració de l'IDE

IntelliJ IDEA

  1. Instal·lar IntelliJ IDEA:

    • Descarrega i instal·la IntelliJ IDEA des del lloc web de JetBrains.

  2. Instal·lar el Plugin de Groovy:

    • Ves a File > Settings > Plugins, cerca "Groovy" i instal·la el plugin.

  3. Crear un Nou Projecte:

    • Ves a File > New > Project, selecciona "Groovy" de la llista de tipus de projectes, i segueix les indicacions per configurar el teu projecte.

Eclipse

  1. Instal·lar Eclipse:

    • Descarrega i instal·la Eclipse des del lloc web d'Eclipse.

  2. Instal·lar el Plugin de Groovy:

    • Ves a Help > Eclipse Marketplace, cerca "Groovy Development Tools" i instal·la el plugin.

  3. Crear un Nou Projecte:

    • Ves a File > New > Project, selecciona "Groovy Project", i segueix les indicacions per configurar el teu projecte.

Visual Studio Code

  1. Instal·lar Visual Studio Code:

    • Descarrega i instal·la Visual Studio Code des del lloc web de VS Code.

  2. Instal·lar l'Extensió de Groovy:

    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra, cerca "Groovy" i instal·la l'extensió.

  3. Crear un Nou Projecte:

    • Obre una nova carpeta per al teu projecte i crea un nou fitxer amb una extensió .groovy.

Creació del Teu Primer Script Groovy

Un cop el teu entorn estigui configurat, pots crear i executar el teu primer script Groovy.

  1. Crear un Nou Fitxer:

    • Crea un nou fitxer anomenat HelloWorld.groovy.

  2. Escriure el Teu Script:

    println 'Hello, Groovy!'

  3. Executar el Teu Script:

    • Obre un terminal, navega al directori que conté HelloWorld.groovy, i executa:
    groovy HelloWorld.groovy
    • Hauries de veure la sortida:
    Hello, Groovy!

Conclusió

Configurar un projecte Groovy implica instal·lar les eines necessàries, configurar el teu IDE, i escriure el teu primer script. Amb aquests passos completats, ara estàs llest per aprofundir en la programació amb Groovy. En les següents seccions, explorarem la sintaxi de Groovy, els tipus de dades, i funcions més avançades per millorar les teves habilitats de programació.
