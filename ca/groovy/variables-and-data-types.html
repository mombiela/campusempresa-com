Variables i Tipus de Dades en Groovy
Operadors ►

Introducció

En aquesta secció, cobrirem els conceptes bàsics de les variables i els tipus de dades en Groovy. Entendre aquests conceptes és crucial ja que formen la base de qualsevol llenguatge de programació. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden manipular i recuperar al llarg del teu programa.

Declarar Variables

  • En Groovy, pots declarar una variable utilitzant la paraula clau def o especificant el tipus explícitament.
  • Groovy és un llenguatge de tipus dinàmic, el que significa que no sempre necessites especificar el tipus d'una variable.

Utilitzant la Paraula Clau def

def name = "John Doe"
def age = 30
def isStudent = true

Especificant el Tipus

String name = "John Doe"
int age = 30
boolean isStudent = true

Regles de Nomenclatura de Variables

  • Els noms de les variables han de començar amb una lletra, un signe de dòlar $, o un guió baix _.
  • Els caràcters següents poden ser lletres, dígits, signes de dòlar o guions baixos.
  • Els noms de les variables són sensibles a majúscules i minúscules.

Tipus de Dades

Groovy admet diversos tipus de dades, que es poden categoritzar àmpliament en tipus primitius i tipus d'objecte.

Tipus de Dades Primitives

  • byte: Enter amb signe de 8 bits
  • short: Enter amb signe de 16 bits
  • int: Enter amb signe de 32 bits
  • long: Enter amb signe de 64 bits
  • float: Número de coma flotant de 32 bits
  • double: Número de coma flotant de 64 bits
  • char: Caràcter Unicode de 16 bits
  • boolean: true o false

Exemples

byte b = 10
short s = 100
int i = 1000
long l = 10000L
float f = 10.5F
double d = 20.5
char c = 'A'
boolean bool = true

Tipus de Dades d'Objecte

  • String: Una seqüència de caràcters
  • List: Una col·lecció ordenada d'elements
  • Map: Una col·lecció de parelles clau-valor
  • Range: Una seqüència de números o caràcters

Exemples

String str = "Hola, Groovy!"
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ["John": 30, "Jane": 25]
Range range = 1..10

Inferència de Tipus

Groovy pot inferir el tipus d'una variable basant-se en el valor assignat. Això és particularment útil per simplificar el codi.

Exemple

def message = "Hola, Món!"  // Groovy infereix això com una String
def count = 42              // Groovy infereix això com un Integer
def price = 19.99           // Groovy infereix això com un Double

Conversió de Tipus

Groovy permet tant conversions de tipus implícites com explícites.

Conversió de Tipus Implícita

Groovy pot convertir automàticament entre tipus compatibles.

Exemple

int num = 10
double result = num + 5.5  // num es converteix implícitament a double

Conversió de Tipus Explícita

Pots convertir tipus explícitament utilitzant el càsting de tipus.

Exemple

double num = 10.5
int result = (int) num  // Es converteix explícitament double a int

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en Groovy. Hem après com declarar variables utilitzant la paraula clau def i especificant els tipus explícitament. També hem explorat els diversos tipus de dades primitives i d'objecte admesos per Groovy, amb exemples. A més, hem discutit la inferència de tipus i la conversió de tipus, tant implícita com explícita.

Entendre aquests conceptes fonamentals és essencial ja que formen els blocs de construcció per a temes de programació més avançats. Assegura't de practicar escrivint els teus propis scripts Groovy per reforçar aquests conceptes.
