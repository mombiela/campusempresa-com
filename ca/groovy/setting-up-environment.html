Configuració de l'Entorn de Desenvolupament
Groovy vs Java ►

Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre Groovy. Aquesta secció et guiarà a través del procés d'instal·lació de Groovy, la configuració d'un Entorn de Desenvolupament Integrat (IDE) i la verificació de la teva configuració.

Instal·lant Groovy

Per començar a codificar en Groovy, necessites instal·lar el llenguatge Groovy a la teva màquina. Segueix aquests passos per instal·lar Groovy:

  1. Descarrega Groovy:

    • Visita el lloc web oficial de Groovy.
    • Descarrega l'última versió estable adequada per al teu sistema operatiu.

  2. Instal·la Groovy:

    • Windows: Executa l'instal·lador descarregat i segueix les instruccions en pantalla.
    • macOS: Utilitza Homebrew per instal·lar Groovy executant la següent comanda al teu terminal: 
      brew install groovy
    • Linux: Utilitza SDKMAN per instal·lar Groovy executant les següents comandes: 
      curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install groovy

  3. Verifica la Instal·lació:

    • Obre un terminal o símbol del sistema.
    • Escriu groovy -version i prem Enter.
    • Hauries de veure la versió de Groovy instal·lada mostrada.

Configurant un Entorn de Desenvolupament Integrat (IDE)

Tot i que pots escriure codi Groovy en qualsevol editor de text, utilitzar un IDE pot millorar significativament la teva productivitat. Aquí tens alguns IDEs populars que suporten Groovy:

  • IntelliJ IDEA:

    • Descarrega i instal·la IntelliJ IDEA des del lloc web oficial.
    • Obre IntelliJ IDEA i configura'l per suportar Groovy instal·lant el plugin de Groovy des del repositori de plugins.

  • Eclipse:

    • Descarrega i instal·la Eclipse des del lloc web oficial.
    • Instal·la el plugin Groovy Development Tools (GDT) des del Marketplace d'Eclipse.

  • Visual Studio Code:

    • Descarrega i instal·la Visual Studio Code des del lloc web oficial.
    • Instal·la l'extensió de Groovy des del Marketplace d'Extensions.

Exemple: Configurant Groovy a IntelliJ IDEA

  1. Instal·la IntelliJ IDEA:

    • Descarrega i instal·la IntelliJ IDEA des del lloc web oficial.

  2. Configura el SDK de Groovy:

    • Obre IntelliJ IDEA.
    • Ves a File > Project Structure > SDKs.
    • Fes clic al botó + i selecciona Groovy SDK.
    • Navega fins al directori on està instal·lat Groovy i selecciona'l.

  3. Crea un Projecte Groovy:

    • Ves a File > New > Project.
    • Selecciona Groovy de la llista de tipus de projecte.
    • Segueix les instruccions per crear un nou projecte Groovy.

  4. Escriu i Executa un Script Groovy:

    • Crea un nou fitxer Groovy fent clic dret a la carpeta src i seleccionant New > Groovy Class.
    • Anomena el fitxer HelloWorld.
    • Afegeix el següent codi al fitxer: 
      class HelloWorld {
    static void main(String[] args) {
        println 'Hello, Groovy!'
    }
}
    • Executa el script fent clic dret al fitxer i seleccionant Run 'HelloWorld.main()'.

Verificant la Teva Configuració

Per assegurar-te que tot està configurat correctament, crea i executa un simple script Groovy:

  1. Crea un Script Groovy:

    • Obre el teu IDE o editor de text.
    • Crea un nou fitxer anomenat HelloGroovy.groovy.

  2. Escriu el Script:

    • Afegeix el següent codi al fitxer: 
      println 'Hello, Groovy!'

  3. Executa el Script:

    • Obre un terminal o símbol del sistema.
    • Navega fins al directori on està guardat el script.
    • Executa el script utilitzant la següent comanda: 
      groovy HelloGroovy.groovy
    • Hauries de veure la sortida Hello, Groovy!.

Conclusió

Configurar l'entorn de desenvolupament per Groovy implica instal·lar el llenguatge Groovy, escollir i configurar un IDE, i verificar la configuració amb un simple script. Seguint aquests passos, t'assegures que estàs preparat per començar a codificar en Groovy de manera eficient. A les següents seccions, aprofundirem en la sintaxi i les característiques de Groovy, construint sobre aquesta sòlida base.
