Introducció a Classes i Objectes

En Groovy, com en molts llenguatges de programació orientats a objectes, les classes i els objectes són conceptes fonamentals. Una classe és un plànol per crear objectes, proporcionant valors inicials per a l'estat (variables membres o propietats) i implementacions de comportament (funcions membres o mètodes). Un objecte és una instància d'una classe.

Definició d'una Classe

Per definir una classe en Groovy, utilitzes la paraula clau class seguida del nom de la classe. Aquí tens un exemple senzill:

class Person {
    String name
    int age

    void displayInfo() {
        println "Nom: $name, Edat: $age"
    }
}
  • class Person: Defineix una nova classe anomenada Person.
  • String name: Declara una propietat name de tipus String.
  • int age: Declara una propietat age de tipus int.
  • void displayInfo(): Declara un mètode que imprimeix la informació de la persona.

Creació d'Objectes

Per crear un objecte (o instància) d'una classe, utilitzes la paraula clau new seguida del nom de la classe i parèntesis. Aquí tens com pots crear un objecte de la classe Person:

def person = new Person()
person.name = "John Doe"
person.age = 30
person.displayInfo()
  • new Person(): Crea una nova instància de la classe Person.
  • person.name = "John Doe": Estableix la propietat name de l'objecte person.
  • person.age = 30: Estableix la propietat age de l'objecte person.
  • person.displayInfo(): Crida el mètode displayInfo de l'objecte person.

Constructors

Els constructors són mètodes especials utilitzats per inicialitzar objectes. En Groovy, pots definir constructors per establir valors inicials per a les propietats. Aquí tens un exemple:

class Person {
    String name
    int age

    Person(String name, int age) {
        this.name = name
        this.age = age
    }

    void displayInfo() {
        println "Nom: $name, Edat: $age"
    }
}

def person = new Person("Jane Doe", 25)
person.displayInfo()
  • Person(String name, int age): Constructor que inicialitza les propietats name i age.
  • this.name = name: Assigna el paràmetre del constructor name a la propietat name.
  • this.age = age: Assigna el paràmetre del constructor age a la propietat age.

Mètodes

Els mètodes defineixen el comportament dels objectes. Són funcions definides dins d'una classe. Aquí tens un exemple d'una classe amb múltiples mètodes:

class Calculator {
    int add(int a, int b) {
        return a + b
    }

    int subtract(int a, int b) {
        return a - b
    }
}

def calc = new Calculator()
println calc.add(5, 3)       // Sortida: 8
println calc.subtract(5, 3)  // Sortida: 2
  • int add(int a, int b): Mètode que suma dos enters.
  • int subtract(int a, int b): Mètode que resta un enter d'un altre.

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe. En Groovy, utilitzes la paraula clau extends per aconseguir l'herència. Aquí tens un exemple:

class Animal {
    String name

    void eat() {
        println "$name està menjant"
    }
}

class Dog extends Animal {
    void bark() {
        println "$name està bordant"
    }
}

def dog = new Dog()
dog.name = "Buddy"
dog.eat()   // Sortida: Buddy està menjant
dog.bark()  // Sortida: Buddy està bordant
  • class Dog extends Animal: La classe Dog hereta de la classe Animal.
  • dog.eat(): Crida el mètode heretat eat de la classe Animal.
  • dog.bark(): Crida el mètode bark definit a la classe Dog.

Polimorfisme

El polimorfisme permet que els objectes siguin tractats com a instàncies de la seva classe pare en lloc de la seva classe real. Això és útil per escriure codi més genèric i reutilitzable. Aquí tens un exemple:

class Animal {
    void makeSound() {
        println "Alguns sons genèrics d'animals"
    }
}

class Dog extends Animal {
    void makeSound() {
        println "Bup"
    }
}

class Cat extends Animal {
    void makeSound() {
        println "Mèu"
    }
}

def animals = [new Dog(), new Cat()]
animals.each { it.makeSound() }
// Sortida:
// Bup
// Mèu
  • void makeSound(): Mètode sobreescrit a les classes Dog i Cat.
  • animals.each { it.makeSound() }: Crida el mètode makeSound a cada animal, demostrant el polimorfisme.

Resum

En aquest tema, hem cobert els conceptes bàsics de les classes i objectes en Groovy, incloent:

  • Definició de classes i creació d'objectes.
  • Ús de constructors per inicialitzar objectes.
  • Definició i ús de mètodes.
  • Implementació de l'herència per crear una jerarquia de classes.
  • Ús del polimorfisme per escriure codi flexible i reutilitzable.

Entendre aquests conceptes és crucial per dominar la programació orientada a objectes en Groovy. Practica creant les teves pròpies classes i objectes per reforçar aquests conceptes.
