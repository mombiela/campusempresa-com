Introducció

Els operadors són símbols especials o paraules clau que s'utilitzen per realitzar operacions sobre operands. En Groovy, els operadors són similars als de Java, però amb algunes millores i característiques addicionals. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en Groovy, des dels operadors aritmètics bàsics fins als més avançats.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • Addició (+)
  • Subtracció (-)
  • Multiplicació (*)
  • Divisió (/)
  • Mòdul (%)

Exemples

def a = 10
def b = 5

println a + b  // Sortida: 15
println a - b  // Sortida: 5
println a * b  // Sortida: 50
println a / b  // Sortida: 2
println a % b  // Sortida: 0

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors.

  • Igual a (==)
  • No igual a (!=)
  • Més gran que (>)
  • Menys que (<)
  • Més gran o igual a (>=)
  • Menys o igual a (<=)

Exemples

def a = 10
def b = 5

println a == b  // Sortida: fals
println a != b  // Sortida: cert
println a > b   // Sortida: cert
println a < b   // Sortida: fals
println a >= b  // Sortida: cert
println a <= b  // Sortida: fals

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleanes.

  • I (&&)
  • O (||)
  • NO (!)

Exemples

def a = cert
def b = fals

println a && b  // Sortida: fals
println a || b  // Sortida: cert
println !a      // Sortida: fals

Operadors a Nivell de Bit

Els operadors a nivell de bit s'utilitzen per realitzar operacions sobre bits individuals de valors enters.

  • I (&)
  • O (|)
  • XOR (^)
  • NO (~)
  • Desplaçament a l'esquerra (<<)
  • Desplaçament a la dreta (>>)
  • Desplaçament a la dreta sense signe (>>>)

Exemples

def a = 5  // Binari: 0101
def b = 3  // Binari: 0011

println a & b  // Sortida: 1  (Binari: 0001)
println a | b  // Sortida: 7  (Binari: 0111)
println a ^ b  // Sortida: 6  (Binari: 0110)
println ~a     // Sortida: -6 (Binari: 1010, complement a dos)
println a << 1 // Sortida: 10 (Binari: 1010)
println a >> 1 // Sortida: 2  (Binari: 0010)
println a >>> 1 // Sortida: 2 (Binari: 0010)

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació simple (=)
  • Assignació d'addició (+=)
  • Assignació de subtracció (-=)
  • Assignació de multiplicació (*=)
  • Assignació de divisió (/=)
  • Assignació de mòdul (%=)

Exemples

def a = 10

a += 5  // Equivalent a a = a + 5
println a  // Sortida: 15

a -= 3  // Equivalent a a = a - 3
println a  // Sortida: 12

a *= 2  // Equivalent a a = a * 2
println a  // Sortida: 24

a /= 4  // Equivalent a a = a / 4
println a  // Sortida: 6

a %= 5  // Equivalent a a = a % 5
println a  // Sortida: 1

Conclusió

Els operadors són fonamentals per realitzar diverses operacions en Groovy. Entendre com utilitzar els operadors aritmètics, relacionals, lògics, a nivell de bit i d'assignació et permetrà escriure codi més eficient i efectiu. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un programador més competent en Groovy.
