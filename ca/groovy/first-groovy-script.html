Introducció

Groovy és un llenguatge de programació orientat a objectes per a la plataforma Java. És un llenguatge potent i dinàmic amb característiques similars a les de Python, Ruby i Smalltalk. Groovy s'integra perfectament amb tots els objectes i biblioteques Java existents, fent-lo una excel·lent elecció per als desenvolupadors Java que busquen augmentar la seva productivitat.

En aquesta secció, escriurem el nostre primer script Groovy. Això t'ajudarà a familiaritzar-te amb la sintaxi bàsica i l'estructura de Groovy.

Configuració de l'Entorn

Abans d'escriure el nostre primer script, necessitem configurar l'entorn:

  1. Instal·lar el Java Development Kit (JDK): Groovy s'executa a la plataforma Java, així que necessites tenir el JDK instal·lat.
  2. Instal·lar Groovy: Pots descarregar i instal·lar Groovy des del lloc web oficial o utilitzar un gestor de paquets com SDKMAN.

Escriure el Teu Primer Script

Comencem amb un simple script de "Hola, Món!" per familiaritzar-nos amb la sintaxi de Groovy.

Hola, Món!

Crea un nou fitxer anomenat HelloWorld.groovy i afegeix el següent codi:

println 'Hello, World!'

Explicació

  • println: Aquest és un mètode de Groovy que imprimeix el missatge especificat a la consola, seguit d'un caràcter de nova línia.
  • 'Hello, World!': Aquesta és una cadena literal en Groovy.

Per executar el script, obre el teu terminal o símbol del sistema, navega al directori que conté HelloWorld.groovy, i executa la següent comanda:

groovy HelloWorld.groovy

Hauries de veure la sortida:

Hello, World!

Sintaxi i Estructura Bàsiques

Explorem algunes sintaxis i estructures bàsiques de Groovy a través d'exemples.

Variables

Groovy és dinàmicament tipat, el que significa que no has de declarar explícitament el tipus d'una variable.

def name = 'Groovy'
println name
  • def: Utilitzat per declarar una variable.
  • name: El nom de la variable.
  • 'Groovy': El valor assignat a la variable.

Tipus de Dades

Groovy suporta diversos tipus de dades, incloent números, cadenes, llistes i mapes.

def number = 42
def text = 'Groovy is cool'
def list = [1, 2, 3, 4]
def map = [name: 'Groovy', type: 'language']

Estructures de Control

Groovy suporta estructures de control estàndard com if-else, bucles for i bucles while.

If-Else

def age = 18
if (age >= 18) {
    println 'Adult'
} else {
    println 'Minor'
}

Bucle For

for (i in 1..5) {
    println i
}

Bucle While

def count = 0
while (count < 5) {
    println count
    count++
}

Exercicis

Prova els següents exercicis per practicar l'escriptura de scripts Groovy:

  1. Imprimeix el Teu Nom: Escriu un script que imprimeixi el teu nom.
  2. Suma de Nombres: Escriu un script que calculi i imprimeixi la suma dels nombres de l'1 al 10.
  3. Manipulació de Llistes: Crea una llista de les teves fruites preferides i imprimeix cada fruita utilitzant un bucle.

Conclusió

En aquesta secció, hem escrit el nostre primer script Groovy i hem explorat algunes sintaxis i estructures bàsiques. Hem cobert variables, tipus de dades i estructures de control. Aquests fonaments són essencials a mesura que avancem cap a temes més avançats en Groovy.

Practicant els exercicis, guanyaràs confiança en l'escriptura de scripts Groovy simples. A la següent secció, aprofundirem en les característiques de Groovy i explorarem conceptes més complexos.
