Introducció a les Llistes

Les llistes són una estructura de dades fonamental en Groovy, utilitzades per emmagatzemar col·leccions ordenades d'elements. Són dinàmiques, és a dir, la seva mida pot canviar, i poden contenir elements de diferents tipus.

Conceptes Clau

  • Mida Dinàmica: Les llistes poden créixer i disminuir segons sigui necessari.
  • Col·lecció Ordenada: Els elements en una llista mantenen el seu ordre.
  • Elements Heterogenis: Les llistes poden contenir elements de diferents tipus.

Creació de Llistes

Crear llistes en Groovy és senzill. Pots utilitzar claudàtors [] per definir una llista.

def emptyList = []
def numberList = [1, 2, 3, 4, 5]
def mixedList = [1, "Groovy", true, 3.14]

Explicació

  • emptyList és una llista buida.
  • numberList conté enters.
  • mixedList conté un enter, una cadena, un booleà i un nombre de coma flotant.

Accés als Elements de la Llista

Pots accedir als elements d'una llista utilitzant el seu índex, començant des de 0.

def numberList = [1, 2, 3, 4, 5]
println(numberList[0])  // Mostra: 1
println(numberList[2])  // Mostra: 3

Explicació

  • numberList[0] accedeix al primer element.
  • numberList[2] accedeix al tercer element.

Modificació de Llistes

Les llistes en Groovy són mutables, és a dir, pots canviar el seu contingut.

Afegir Elements

Pots afegir elements utilitzant l'operador << o el mètode add().

def numberList = [1, 2, 3]
numberList << 4
numberList.add(5)
println(numberList)  // Mostra: [1, 2, 3, 4, 5]

Eliminar Elements

Pots eliminar elements utilitzant el mètode remove() o especificant l'índex.

def numberList = [1, 2, 3, 4, 5]
numberList.remove(2)  // Elimina l'element a l'índex 2
println(numberList)   // Mostra: [1, 2, 4, 5]
numberList.remove((Integer) 4)  // Elimina l'element amb valor 4
println(numberList)   // Mostra: [1, 2, 5]

Actualitzar Elements

Pots actualitzar elements assignant un nou valor a un índex específic.

def numberList = [1, 2, 3, 4, 5]
numberList[1] = 10
println(numberList)  // Mostra: [1, 10, 3, 4, 5]

Iterar Sobre les Llistes

Pots iterar sobre les llistes utilitzant bucles o el mètode each de Groovy.

Utilitzant un Bucle `for`

def numberList = [1, 2, 3, 4, 5]
for (num in numberList) {
    println(num)
}

Utilitzant el Mètode `each`

def numberList = [1, 2, 3, 4, 5]
numberList.each { num ->
    println(num)
}

Operacions Avançades amb Llistes

Groovy proporciona diverses operacions avançades per a les llistes, com ara filtratge, mapeig i reducció.

Filtratge

Pots filtrar llistes utilitzant el mètode findAll.

def numberList = [1, 2, 3, 4, 5]
def evenNumbers = numberList.findAll { it % 2 == 0 }
println(evenNumbers)  // Mostra: [2, 4]

Mapeig

Pots transformar llistes utilitzant el mètode collect.

def numberList = [1, 2, 3, 4, 5]
def squaredNumbers = numberList.collect { it * it }
println(squaredNumbers)  // Mostra: [1, 4, 9, 16, 25]

Reducció

Pots reduir llistes a un sol valor utilitzant el mètode inject.

def numberList = [1, 2, 3, 4, 5]
def sum = numberList.inject(0) { acc, num -> acc + num }
println(sum)  // Mostra: 15

Conclusió

Les llistes són una estructura de dades versàtil i poderosa en Groovy. Et permeten emmagatzemar, accedir, modificar i iterar sobre col·leccions d'elements de manera eficient. Entenent els conceptes bàsics i les operacions avançades de les llistes, pots aprofitar tot el seu potencial en els teus programes Groovy.
