Estructures de control en Groovy
Llistes ►

Les estructures de control són fonamentals en la programació ja que et permeten dictar el flux del teu programa. En Groovy, les estructures de control són similars a les de Java però amb algunes millores i sucre sintàctic que les fan més concises i expressives.

Declaracions condicionals

`if` i `else`

Les declaracions if i else s'utilitzen per executar blocs de codi basats en una condició.

def age = 18

if (age >= 18) {
    println "You are an adult."
} else {
    println "You are a minor."
}
  • Conceptes clau:
    • La condició dins de la declaració if ha d'avaluar-se a un booleà.
    • El bloc else és opcional i s'executa si la condició if és falsa.

`else if`

La declaració else if et permet comprovar múltiples condicions.

def score = 85

if (score >= 90) {
    println "Grade: A"
} else if (score >= 80) {
    println "Grade: B"
} else if (score >= 70) {
    println "Grade: C"
} else {
    println "Grade: F"
}
  • Conceptes clau:
    • Es poden encadenar múltiples blocs else if.
    • La primera condició que s'avaluï com a certa executarà el seu bloc.

Declaració Switch

La declaració switch en Groovy és més potent que en Java, permetent valors de cas més flexibles.

def fruit = "Apple"

switch (fruit) {
    case "Apple":
        println "It's an apple."
        break
    case "Banana":
        println "It's a banana."
        break
    default:
        println "Unknown fruit."
}
  • Conceptes clau:
    • La declaració switch avalua l'expressió i la compara amb els valors dels casos.
    • La declaració break s'utilitza per sortir del bloc switch.
    • El cas default és opcional i s'executa si cap altre cas coincideix.

Estructures de bucle

Bucle `for`

El bucle for en Groovy pot iterar sobre un rang, una llista o qualsevol objecte iterable.

for (i in 1..5) {
    println "Iteration: $i"
}
  • Conceptes clau:
    • La paraula clau in s'utilitza per iterar sobre un rang o col·lecció.
    • La variable del bucle (i en aquest cas) pren cada valor del rang o col·lecció.

Bucle `while`

El bucle while continua executant-se mentre la seva condició sigui certa.

def count = 1

while (count <= 5) {
    println "Count: $count"
    count++
}
  • Conceptes clau:
    • La condició s'avalua abans de cada iteració.
    • Si la condició és falsa inicialment, el cos del bucle no s'executarà.

Bucle `do-while`

El bucle do-while és similar al bucle while però garanteix que el cos del bucle s'executarà almenys una vegada.

def count = 1

do {
    println "Count: $count"
    count++
} while (count <= 5)
  • Conceptes clau:
    • El cos del bucle s'executa abans de comprovar la condició.
    • Útil quan el cos del bucle necessita executar-se almenys una vegada independentment de la condició.

Bucle millorat amb Groovy

Mètode `each`

Groovy proporciona el mètode each per iterar sobre col·leccions d'una manera més expressiva.

def list = [1, 2, 3, 4, 5]

list.each { item ->
    println "Item: $item"
}
  • Conceptes clau:
    • El mètode each pren un tancament que s'executa per a cada element de la col·lecció.
    • La variable del bucle (item en aquest cas) es defineix dins del tancament.

Mètode `collect`

El mètode collect transforma cada element d'una col·lecció i retorna una nova col·lecció.

def list = [1, 2, 3, 4, 5]
def squaredList = list.collect { it * it }

println squaredList  // Output: [1, 4, 9, 16, 25]
  • Conceptes clau:
    • El mètode collect pren un tancament que transforma cada element.
    • El resultat és una nova col·lecció amb els elements transformats.

Conclusió

Les estructures de control en Groovy proporcionen maneres potents i flexibles per controlar el flux del teu programa. Des de declaracions condicionals bàsiques i bucles fins a mètodes millorats com each i collect, Groovy ofereix un conjunt ric d'eines per fer el teu codi més expressiu i concís. Entendre aquestes estructures de control és essencial per escriure programes efectius en Groovy.
