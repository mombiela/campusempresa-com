Introducció al JSON

  • JSON (JavaScript Object Notation) és un format lleuger d'intercanvi de dades.
  • És fàcil de llegir i escriure per als humans i fàcil de parsejar i generar per a les màquines.
  • JSON s'utilitza sovint per transmetre dades en aplicacions web.

Exemple de JSON

{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}

Treballant amb JSON en Groovy

Groovy proporciona suport integrat per a JSON a través del paquet groovy.json. Aquest paquet inclou classes com JsonSlurper i JsonOutput per parsejar i generar dades JSON.

Parsejant JSON amb JsonSlurper

  • JsonSlurper s'utilitza per parsejar cadenes JSON en objectes Groovy.

Exemple: Parsejant JSON

import groovy.json.JsonSlurper

def jsonString = '''
{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.name  // Output: John Doe
println jsonObject.age   // Output: 30
println jsonObject.isStudent  // Output: false
println jsonObject.courses  // Output: [Math, Science, History]

Generant JSON amb JsonOutput

  • JsonOutput s'utilitza per convertir objectes Groovy en cadenes JSON.

Exemple: Generant JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
println jsonString

Pretty Printing JSON

  • JsonOutput.prettyPrint es pot utilitzar per formatar cadenes JSON per a una millor llegibilitat.

Exemple: Pretty Printing JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
def prettyJsonString = JsonOutput.prettyPrint(jsonString)
println prettyJsonString

Gestionant Estructures JSON Complexes

  • Groovy pot gestionar estructures JSON complexes i nidificades amb facilitat.

Exemple: Parsejant JSON Complex

import groovy.json.JsonSlurper

def jsonString = '''
{
  "person": {
    "name": "John Doe",
    "age": 30,
    "address": {
      "street": "123 Main St",
      "city": "Springfield",
      "zip": "12345"
    },
    "courses": ["Math", "Science", "History"]
  }
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.person.name  // Output: John Doe
println jsonObject.person.address.city  // Output: Springfield
println jsonObject.person.courses  // Output: [Math, Science, History]

Resum

  • JSON és un format àmpliament utilitzat per a l'intercanvi de dades.
  • Groovy proporciona un suport robust per a JSON a través del paquet groovy.json.
  • JsonSlurper s'utilitza per parsejar cadenes JSON en objectes Groovy.
  • JsonOutput s'utilitza per generar cadenes JSON a partir d'objectes Groovy.
  • Groovy pot gestionar tant estructures JSON simples com complexes de manera eficient.

Dominant aquestes eines, pots treballar efectivament amb dades JSON en les teves aplicacions Groovy, fent que l'intercanvi de dades sigui fluid i eficient.
