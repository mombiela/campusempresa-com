Introducció a les Clausures

Les clausures són una de les característiques més poderoses de Groovy. Són blocs de codi que es poden assignar a variables, passar com a paràmetres i executar més tard. Les clausures poden capturar variables del seu context circumdant, fent-les molt flexibles i útils per a diverses tasques de programació.

Conceptes Clau

  • Definició: Una clausura és una instància de la classe groovy.lang.Closure.
  • Sintaxi: Les clausures es defineixen utilitzant claus {}.
  • Paràmetres: Les clausures poden tenir paràmetres, que es defineixen dins de les claus.
  • Valor de Retorn: L'última expressió en una clausura és el valor de retorn.

Sintaxi Bàsica i Ús

Definint una Clausura

Es pot definir una clausura simple de la següent manera:

def myClosure = { println "Hello, Groovy!" }
myClosure() // Sortida: Hello, Groovy!

Clausures amb Paràmetres

Les clausures poden acceptar paràmetres, que es especifiquen abans del símbol ->:

def greet = { name -> println "Hello, $name!" }
greet("Alice") // Sortida: Hello, Alice!

Paràmetre Implícit `it`

Si una clausura pren un únic paràmetre, Groovy proporciona un paràmetre implícit anomenat it:

def printIt = { println it }
printIt("Implicit parameter example") // Sortida: Implicit parameter example

Característiques Avançades

Capturant Variables

Les clausures poden capturar variables del seu context circumdant, també conegudes com a "variables lliures":

def greeting = "Hello"
def greeter = { name -> println "$greeting, $name!" }
greeter("Bob") // Sortida: Hello, Bob!

Retornant Valors

L'última expressió en una clausura és el valor de retorn:

def square = { it * it }
println square(4) // Sortida: 16

Clausures com a Paràmetres de Mètode

Les clausures es poden passar com a paràmetres a mètodes, permetent funcions d'ordre superior:

def processList(list, closure) {
    list.each { closure(it) }
}

def numbers = [1, 2, 3, 4, 5]
processList(numbers, { println it * 2 }) // Sortida: 2, 4, 6, 8, 10

Clausures amb Múltiples Paràmetres

Les clausures poden tenir múltiples paràmetres, separats per comes:

def sum = { a, b -> a + b }
println sum(3, 7) // Sortida: 10

Exemples Pràctics

Filtrant una Llista

Utilitzant clausures per filtrar una llista:

def numbers = [1, 2, 3, 4, 5, 6]
def evenNumbers = numbers.findAll { it % 2 == 0 }
println evenNumbers // Sortida: [2, 4, 6]

Transformant una Llista

Utilitzant clausures per transformar una llista:

def numbers = [1, 2, 3, 4, 5]
def squares = numbers.collect { it * it }
println squares // Sortida: [1, 4, 9, 16, 25]

Ordenant una Llista

Utilitzant clausures per ordenar una llista:

def names = ["Charlie", "Alice", "Bob"]
names.sort { a, b -> a <=> b }
println names // Sortida: [Alice, Bob, Charlie]

Conclusió

Les clausures en Groovy proporcionen una manera poderosa i flexible de manejar blocs de codi. Poden capturar variables del seu context circumdant, fent-les molt útils per a tasques com filtrar, transformar i ordenar col·leccions. Entendre les clausures és essencial per escriure codi Groovy concís i expressiu. Dominant les clausures, pots aprofitar tot el potencial de Groovy per crear programes més llegibles i mantenibles.
