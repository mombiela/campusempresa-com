Introducció a XML

  • XML (eXtensible Markup Language) és un llenguatge de marques dissenyat per emmagatzemar i transportar dades.
  • XML és llegible tant per humans com per màquines.
  • S'utilitza àmpliament en serveis web, fitxers de configuració i intercanvi de dades entre sistemes.

Treballant amb XML a Groovy

Groovy proporciona maneres potents i flexibles de treballar amb XML. Simplifica l'anàlisi, la manipulació i la generació d'XML.

Analitzant XML

Groovy ofereix diverses maneres d'analitzar XML, incloent XmlSlurper i XmlParser.

Utilitzant XmlSlurper

XmlSlurper és una manera convenient d'analitzar XML en una estructura d'arbre que es pot navegar fàcilment.

def xmlContent = '''
<books>
    <book id="1">
        <title>Groovy in Action</title>
        <author>Dierk Koenig</author>
    </book>
    <book id="2">
        <title>Programming Groovy 2</title>
        <author>Venkat Subramaniam</author>
    </book>
</books>
'''

def books = new XmlSlurper().parseText(xmlContent)
books.book.each { book ->
    println "Title: ${book.title}, Author: ${book.author}"
}
  • Explicació:
    • XmlSlurper analitza el contingut XML en una estructura d'arbre.
    • El mètode parseText s'utilitza per analitzar la cadena XML.
    • Iterem sobre cada node book i imprimim el títol i l'autor.

Utilitzant XmlParser

XmlParser és una altra manera d'analitzar XML, proporcionant una estructura més semblant a DOM.

def xmlContent = '''
<books>
    <book id="1">
        <title>Groovy in Action</title>
        <author>Dierk Koenig</author>
    </book>
    <book id="2">
        <title>Programming Groovy 2</title>
        <author>Venkat Subramaniam</author>
    </book>
</books>
'''

def books = new XmlParser().parseText(xmlContent)
books.book.each { book ->
    println "Title: ${book.title.text()}, Author: ${book.author.text()}"
}
  • Explicació:
    • XmlParser analitza el contingut XML en una estructura semblant a DOM.
    • El mètode parseText s'utilitza per analitzar la cadena XML.
    • Accedim al contingut de text dels nodes title i author utilitzant el mètode text().

Generant XML

Groovy facilita la generació d'XML utilitzant MarkupBuilder.

import groovy.xml.MarkupBuilder

def writer = new StringWriter()
def xml = new MarkupBuilder(writer)

xml.books {
    book(id: 1) {
        title 'Groovy in Action'
        author 'Dierk Koenig'
    }
    book(id: 2) {
        title 'Programming Groovy 2'
        author 'Venkat Subramaniam'
    }
}

println writer.toString()
  • Explicació:
    • MarkupBuilder s'utilitza per crear contingut XML.
    • Definim l'estructura de l'XML utilitzant crides a mètodes i closures.
    • L'XML generat s'escriu a un StringWriter i s'imprimeix.

Manipulant XML

Groovy permet una manipulació fàcil dels nodes XML.

def xmlContent = '''
<books>
    <book id="1">
        <title>Groovy in Action</title>
        <author>Dierk Koenig</author>
    </book>
    <book id="2">
        <title>Programming Groovy 2</title>
        <author>Venkat Subramaniam</author>
    </book>
</books>
'''

def books = new XmlSlurper().parseText(xmlContent)
books.book.find { it.@id == '1' }.title = 'Groovy in Action, 2nd Edition'

println groovy.xml.XmlUtil.serialize(books)
  • Explicació:
    • Analitzem l'XML utilitzant XmlSlurper.
    • Trobem el node book amb id 1 i actualitzem el seu title.
    • L'XML actualitzat es serialitza i s'imprimeix.

Conclusió

Groovy proporciona eines robustes i fàcils d'utilitzar per treballar amb XML. Tant si necessites analitzar, generar o manipular XML, XmlSlurper, XmlParser i MarkupBuilder de Groovy fan que aquestes tasques siguin senzilles i eficients. Entendre aquestes eines millorarà la teva capacitat per gestionar dades XML en les teves aplicacions Groovy.
