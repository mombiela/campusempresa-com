Groovy és un llenguatge potent, opcionalment tipat i dinàmic, amb capacitats de tipatge estàtic i compilació estàtica, per a la plataforma Java destinat a millorar la productivitat dels desenvolupadors. Es pot utilitzar tant com a llenguatge de scripting com per escriure aplicacions completes. Aquesta secció se centrarà en l'ús de Groovy per a scripting, cobrint els conceptes bàsics fins a tècniques de scripting més avançades.

Introducció al Scripting amb Groovy

Els scripts de Groovy s'utilitzen típicament per automatitzar tasques, manipular fitxers i realitzar càlculs ràpids. Sovint s'utilitzen en eines d'automatització de compilació com Gradle.

Conceptes Clau

  • Tipatge Dinàmic: Groovy et permet escriure scripts sense definir explícitament els tipus.
  • Syntactic Sugar: Groovy proporciona una sintaxi més concisa i llegible en comparació amb Java.
  • Interoperabilitat: Groovy es pot integrar perfectament amb biblioteques i marcs de treball de Java.

Script de Hello World

Comencem amb un simple script de "Hello World" en Groovy.

println 'Hello, World!'

Aquest script demostra la simplicitat de la sintaxi de Groovy. La funció println imprimeix la cadena a la consola.

Variables i Tipus de Dades

Groovy admet diversos tipus de dades i permet el tipatge dinàmic, fent-lo flexible per a scripting.

Declaració de Variables

Pots declarar variables sense especificar el seu tipus.

def name = 'Groovy'
def age = 10
println "Nom: $name, Edat: $age"

Tipus de Dades Comuns

  • String: Dades textuals.
  • Integer: Nombres enters.
  • Float/Double: Nombres decimals.
  • Boolean: Valors veritable o fals.

Exemple: Tipus de Dades

def message = 'Hello, Groovy!'
def number = 42
def price = 19.99
def isAvailable = true

println "Missatge: $message"
println "Nombre: $number"
println "Preu: $price"
println "Disponible: $isAvailable"

Estructures de Control

Groovy admet estructures de control estàndard com bucles i condicionals.

Declaracions If-Else

def number = 10

if (number > 0) {
    println 'Nombre positiu'
} else {
    println 'Nombre no positiu'
}

Bucles For

for (i in 1..5) {
    println "Iteració: $i"
}

Bucles While

def count = 5

while (count > 0) {
    println "Compte: $count"
    count--
}

Funcions i Closures

Les funcions i closures són essencials per escriure codi reutilitzable en scripts de Groovy.

Definició de Funcions

def greet(name) {
    return "Hola, $name!"
}

println greet('Groovy')

Ús de Closures

Les closures són blocs de codi anònims que es poden assignar a variables i passar.

def square = { number -> number * number }

println square(4)  // Sortida: 16

Operacions amb Fitxers

Groovy facilita el treball amb fitxers, que és una tasca comuna en scripting.

Lectura de Fitxers

def file = new File('example.txt')
file.eachLine { line ->
    println line
}

Escriptura de Fitxers

def file = new File('output.txt')
file.withWriter { writer ->
    writer.writeLine('Hola, Groovy!')
}

Tècniques de Scripting Avançades

A mesura que et sentis més còmode amb el scripting en Groovy, pots explorar tècniques més avançades.

Ús de Groovy Shell

El Groovy Shell (groovysh) és una shell interactiva de línia de comandes per avaluar expressions Groovy.

groovy:000> println 'Hola des de Groovy Shell!'

Integració amb Biblioteques de Java

Groovy pot utilitzar fàcilment biblioteques de Java, fent-lo un llenguatge de scripting versàtil.

import java.util.Date

def now = new Date()
println "Data i hora actual: $now"

Conclusió

Groovy és un llenguatge de scripting versàtil i potent que pot millorar significativament la teva productivitat. El seu tipatge dinàmic, sintaxi concisa i integració perfecta amb Java el fan una excel·lent opció per a diverses tasques de scripting. En dominar els conceptes bàsics i explorar tècniques avançades, pots aprofitar Groovy per automatitzar tasques, manipular fitxers i integrar-te amb biblioteques de Java de manera efectiva.
