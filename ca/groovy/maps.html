Mapes en Groovy
Rangs ►

Els mapes en Groovy són una estructura de dades potent i flexible que et permet emmagatzemar parelles clau-valor. Són similars als diccionaris en Python o als hash maps en Java. Aquesta secció et guiarà des dels conceptes bàsics de creació i ús de mapes fins a tècniques més avançades i millors pràctiques.

Introducció als Mapes

  • Definició: Un mapa és una col·lecció de parelles clau-valor on cada clau és única.
  • Sintaxi: Els mapes en Groovy es defineixen utilitzant la notació [:].

Creació de Mapes

Pots crear un mapa en Groovy utilitzant diferents mètodes:

// Utilitzant la notació literal
def map = [key1: 'value1', key2: 'value2']

// Utilitzant el constructor de mapes
def map = new HashMap()
map.put('key1', 'value1')
map.put('key2', 'value2')

Accés als Elements del Mapa

Pots accedir als elements d'un mapa utilitzant la clau:

def map = [name: 'John', age: 30]

// Accés als valors
println map['name']  // Sortida: John
println map.age      // Sortida: 30

Modificació de Mapes

Pots afegir, actualitzar o eliminar elements d'un mapa:

def map = [name: 'John', age: 30]

// Afegir una nova parella clau-valor
map['city'] = 'New York'

// Actualitzar una parella clau-valor existent
map.age = 31

// Eliminar una parella clau-valor
map.remove('name')

println map  // Sortida: [age:31, city:New York]

Operacions Avançades amb Mapes

Iteració sobre Mapes

Pots iterar sobre les claus, valors o entrades d'un mapa:

def map = [name: 'John', age: 30, city: 'New York']

// Iteració sobre les claus
map.keySet().each { key ->
    println key
}

// Iteració sobre els valors
map.values().each { value ->
    println value
}

// Iteració sobre les entrades
map.each { key, value ->
    println "$key: $value"
}

Filtratge de Mapes

Pots filtrar mapes basant-te en certes condicions:

def map = [name: 'John', age: 30, city: 'New York']

// Filtratge d'entrades on el valor és una cadena
def filteredMap = map.findAll { key, value -> value instanceof String }

println filteredMap  // Sortida: [name:John, city:New York]

Transformació de Mapes

Pots transformar mapes utilitzant el mètode collectEntries:

def map = [name: 'John', age: 30, city: 'New York']

// Transformació del mapa a claus en majúscules
def transformedMap = map.collectEntries { key, value ->
    [(key.toUpperCase()): value]
}

println transformedMap  // Sortida: [NAME:John, AGE:30, CITY:New York]

Mètodes Comuns de Mapes

Aquí tens alguns mètodes comuns que pots utilitzar amb mapes:

| Mètode | Descripció | Exemple d'Ús | |-----------------|---------------------------------------------|----------------------------------------| | put | Afegeix una parella clau-valor al mapa | map.put('key', 'value') | | get | Recupera el valor per a una clau donada | map.get('key') | | remove | Elimina una parella clau-valor del mapa | map.remove('key') | | containsKey | Comprova si el mapa conté una clau específica | map.containsKey('key') | | containsValue | Comprova si el mapa conté un valor específic | map.containsValue('value') | | size | Retorna el nombre de parelles clau-valor | map.size() | | clear | Elimina totes les parelles clau-valor del mapa | map.clear() |

Conclusió

Els mapes en Groovy són una estructura de dades versàtil i essencial que et permet emmagatzemar i manipular parelles clau-valor de manera eficient. Des de la creació i accés bàsics fins a operacions avançades com el filtratge i la transformació, Groovy proporciona un conjunt ric de característiques per treballar amb mapes. Entendre aquests conceptes millorarà enormement la teva capacitat per gestionar i manipular dades en les teves aplicacions Groovy.
