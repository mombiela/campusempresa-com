Introducció

Groovy és un llenguatge de programació orientat a objectes per a la plataforma Java. Es compila dinàmicament a bytecode de Java i s'integra perfectament amb Java. Entendre la sintaxi bàsica de Groovy és essencial per escriure codi efectiu i eficient. Aquesta secció cobrirà els elements fonamentals de la sintaxi de Groovy, incloent variables, tipus de dades, operadors i estructures de control.

Variables i Tipus de Dades

Groovy és tipat dinàmicament, el que significa que no cal declarar explícitament el tipus d'una variable. No obstant això, encara pots utilitzar el tipatge estàtic si ho prefereixes.

Declaració de Variables

  • Tipatge Dinàmic: Pots declarar una variable sense especificar el seu tipus.
  • Tipatge Estàtic: Pots declarar una variable amb un tipus específic.
// Tipatge Dinàmic
def name = "Groovy"
def age = 25

// Tipatge Estàtic
String language = "Groovy"
int year = 2023

Tipus de Dades

Groovy suporta diversos tipus de dades, incloent:

  • Nombres: Integer, Float, Double, BigDecimal
  • Cadenes: Cometes simples, Cometes dobles, Cometes triples
  • Booleans: true, false
  • Col·leccions: Llistes, Mapes, Rangs
// Nombres
int num = 10
float pi = 3.14
BigDecimal bigNum = 12345678901234567890.12345678901234567890

// Cadenes
String singleQuote = 'Hola'
String doubleQuote = "Món"
String tripleQuote = """Això és una
cadena de múltiples línies"""

// Booleans
boolean isGroovy = true

// Col·leccions
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ['Alice': 30, 'Bob': 25]
Range<Integer> range = 1..5

Operadors

Groovy suporta una àmplia gamma d'operadors, incloent operadors aritmètics, relacionals, lògics i de bits.

Operadors Aritmètics

  • Addició (+)
  • Subtracció (-)
  • Multiplicació (*)
  • Divisió (/)
  • Mòdul (%)
int a = 10
int b = 5

println a + b  // 15
println a - b  // 5
println a * b  // 50
println a / b  // 2
println a % b  // 0

Operadors Relacionals

  • Igual a (==)
  • Diferent de (!=)
  • Major que (>)
  • Menor que (<)
  • Major o igual que (>=)
  • Menor o igual que (<=)
int x = 10
int y = 20

println x == y  // fals
println x != y  // cert
println x > y   // fals
println x < y   // cert
println x >= y  // fals
println x <= y  // cert

Operadors Lògics

  • I (&&)
  • O (||)
  • NO (!)
boolean isTrue = true
boolean isFalse = false

println isTrue && isFalse  // fals
println isTrue || isFalse  // cert
println !isTrue            // fals

Estructures de Control

Groovy proporciona diverses estructures de control per gestionar el flux del programa, incloent declaracions condicionals i bucles.

Declaracions Condicionals

  • if-else: Executa un bloc de codi si una condició és certa, en cas contrari executa un altre bloc de codi.
int score = 85

if (score >= 90) {
    println "A"
} else if (score >= 80) {
    println "B"
} else {
    println "C"
}

Bucles

  • for loop: Itera sobre un rang, llista o mapa.
  • while loop: Repeteix un bloc de codi mentre una condició sigui certa.
// for loop
for (int i = 1; i <= 5; i++) {
    println i
}

// while loop
int count = 1
while (count <= 5) {
    println count
    count++
}

Conclusió

Entendre la sintaxi bàsica de Groovy és crucial per escriure codi efectiu. Aquesta secció ha cobert la declaració de variables, tipus de dades, operadors i estructures de control. Amb aquests fonaments, pots començar a escriure programes simples en Groovy i gradualment passar a tasques més complexes. En les següents seccions, aprofundirem en característiques i capacitats més avançades de Groovy.
