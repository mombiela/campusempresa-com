Rangs en Groovy
Tancaments ►

Els rangs són una característica potent i versàtil en Groovy, que et permeten crear seqüències de valors amb facilitat. S'utilitzen comunament per a la iteració, el tall de col·leccions i més. Aquest tema cobrirà els conceptes bàsics dels rangs, el seu ús i característiques avançades.

Què és un Rang?

Un rang en Groovy és una seqüència de valors definida per un valor inicial i un valor final. Els rangs poden ser inclusius o exclusius, i es poden utilitzar amb números, caràcters i fins i tot objectes personalitzats.

Conceptes Clau

  • Rangs Inclusius: Inclouen tant el valor inicial com el valor final.
  • Rangs Exclusius: Exclouen el valor final.
  • Valors de Pas: Defineixen l'increment entre valors en el rang.
  • Rangs Inversos: Defineixen rangs en ordre descendent.

Creació de Rangs

Rangs Inclusius

Un rang inclusiu inclou tant el valor inicial com el valor final. Pots crear un rang inclusiu utilitzant l'operador ...

def inclusiveRange = 1..5
println inclusiveRange // Output: [1, 2, 3, 4, 5]

Rangs Exclusius

Un rang exclusiu exclou el valor final. Pots crear un rang exclusiu utilitzant l'operador ..<.

def exclusiveRange = 1..<5
println exclusiveRange // Output: [1, 2, 3, 4]

Rangs de Caràcters

Els rangs també es poden crear amb caràcters.

def charRange = 'a'..'e'
println charRange // Output: [a, b, c, d, e]

Rangs Inversos

Els rangs es poden definir en ordre invers.

def reverseRange = 5..1
println reverseRange // Output: [5, 4, 3, 2, 1]

Ús de Rangs

Iteració

Els rangs s'utilitzen sovint en bucles per a la iteració.

for (i in 1..5) {
    println i
}
// Output: 
// 1
// 2
// 3
// 4
// 5

Tall de Col·leccions

Els rangs es poden utilitzar per tallar llistes i arrays.

def list = [10, 20, 30, 40, 50]
def sublist = list[1..3]
println sublist // Output: [20, 30, 40]

Valors de Pas

Pots definir el valor de pas per a un rang utilitzant el mètode step.

def steppedRange = 1..10
println steppedRange.step(2) // Output: [1, 3, 5, 7, 9]

Característiques Avançades

Rangs d'Objectes Personalitzats

Pots crear rangs amb objectes personalitzats implementant els mètodes next i previous.

class CustomRange {
    int value

    CustomRange next() {
        return new CustomRange(value: value + 1)
    }

    CustomRange previous() {
        return new CustomRange(value: value - 1)
    }

    String toString() {
        return value.toString()
    }
}

def customRange = new CustomRange(value: 1)..new CustomRange(value: 5)
println customRange // Output: [1, 2, 3, 4, 5]

Comprovació de Pertenència

Pots comprovar si un valor està dins d'un rang utilitzant el mètode contains o la paraula clau in.

def range = 1..5
println range.contains(3) // Output: true
println 3 in range        // Output: true
println 6 in range        // Output: false

Conclusió

Els rangs en Groovy són una eina potent per crear seqüències de valors, iterar i tallar col·leccions. Poden ser inclusius o exclusius, i suporten objectes personalitzats i valors de pas. Entendre com utilitzar els rangs de manera efectiva pot millorar enormement les teves habilitats de programació en Groovy. Practica creant i utilitzant rangs en diferents contextos per sentir-te més còmode amb aquesta característica.
