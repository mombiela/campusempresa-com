Introducció a l'Automatització a Google Ads

L'automatització a Google Ads permet als anunciants optimitzar i gestionar les seves campanyes de manera més eficient mitjançant l'ús de scripts i eines automatitzades. En aquesta secció, aprendràs els conceptes bàsics i la importància de l'automatització en SEM.

  • Beneficis de l'automatització:
    • Estalvi de temps.
    • Millora en la precisió de les campanyes.
    • Optimització contínua.
    • Reducció d'errors humans.

Scripts a Google Ads

Els scripts a Google Ads són fragments de codi que permeten automatitzar tasques repetitives i complexes. Aquests scripts s'escriuen en JavaScript i poden interactuar amb l'API de Google Ads per realitzar diverses accions.

  • Conceptes clau:
    • JavaScript: Llenguatge de programació utilitzat per escriure scripts a Google Ads.
    • API de Google Ads: Interfície que permet als scripts interactuar amb la plataforma de Google Ads.
    • Automatització de tasques: Ús de scripts per realitzar tasques com ajustos de licitacions, generació d'informes, i pausa d'anuncis.

Configuració Inicial

Abans de començar a escriure scripts, és necessari configurar l'entorn adequat a Google Ads.

  • Passos per a la configuració:
    1. Accedeix al teu compte de Google Ads.
    2. Navega a la secció de "Eines i configuració".
    3. Selecciona "Scripts" al menú desplegable.
    4. Crea un nou script i atorga els permisos necessaris.

Exemples de Scripts Bàsics

A continuació, es presenten alguns exemples de scripts bàsics que poden ser útils per a principiants.

Exemple 1: Pausar Anuncis amb Baix Rendiment

Aquest script pausa anuncis que tenen un CTR (Click-Through Rate) inferior a l'1%.

function main() {
  var adIterator = AdsApp.ads()
    .withCondition("Status = ENABLED")
    .withCondition("Clicks > 100")
    .withCondition("Ctr < 1.0")
    .get();

  while (adIterator.hasNext()) {
    var ad = adIterator.next();
    ad.pause();
    Logger.log("Paused ad with ID: " + ad.getId());
  }
}
  • Explicació:
    • AdsApp.ads(): Obté tots els anuncis del compte.
    • withCondition(): Filtra els anuncis segons les condicions especificades.
    • get(): Obté un iterador per recórrer els anuncis filtrats.
    • pause(): Pausa l'anunci actual.
    • Logger.log(): Registra un missatge al registre de scripts.

Exemple 2: Ajust de Licitació Automàtic

Aquest script ajusta les licitacions de les paraules clau segons el seu rendiment.

function main() {
  var keywordIterator = AdsApp.keywords()
    .withCondition("Status = ENABLED")
    .withCondition("Clicks > 50")
    .get();

  while (keywordIterator.hasNext()) {
    var keyword = keywordIterator.next();
    var stats = keyword.getStatsFor("LAST_30_DAYS");
    var ctr = stats.getCtr();

    if (ctr > 2.0) {
      keyword.setMaxCpc(keyword.getMaxCpc() * 1.1);
      Logger.log("Increased bid for keyword: " + keyword.getText());
    } else if (ctr < 1.0) {
      keyword.setMaxCpc(keyword.getMaxCpc() * 0.9);
      Logger.log("Decreased bid for keyword: " + keyword.getText());
    }
  }
}
  • Explicació:
    • AdsApp.keywords(): Obté totes les paraules clau del compte.
    • getStatsFor("LAST_30_DAYS"): Obté les estadístiques dels últims 30 dies.
    • getCtr(): Obté el CTR de la paraula clau.
    • setMaxCpc(): Ajusta la licitació màxima de CPC (Cost per Clic).

Exemples Avançats de Scripts

Per als usuaris avançats, els scripts poden ser més complexos i realitzar tasques més sofisticades.

Exemple 3: Generació d'Informes Personalitzats

Aquest script genera un informe personalitzat i l'envia per correu electrònic.

function main() {
  var report = AdsApp.report(
    "SELECT CampaignName, Impressions, Clicks, Cost " +
    "FROM CAMPAIGN_PERFORMANCE_REPORT " +
    "DURING LAST_7_DAYS"
  );

  var rows = report.rows();
  var csv = "CampaignName,Impressions,Clicks,Cost\n";

  while (rows.hasNext()) {
    var row = rows.next();
    csv += row['CampaignName'] + "," + row['Impressions'] + "," + row['Clicks'] + "," + row['Cost'] + "\n";
  }

  MailApp.sendEmail("your-email@example.com", "Weekly Campaign Report", "Please find the attached report.", {
    attachments: [
      {
        fileName: "report.csv",
        mimeType: "text/csv",
        content: csv
      }
    ]
  });
}
  • Explicació:
    • AdsApp.report(): Executa una consulta d'informe a Google Ads.
    • rows(): Obté un iterador per recórrer les files de l'informe.
    • MailApp.sendEmail(): Envia un correu electrònic amb l'informe adjunt.

Bones Pràctiques i Consells

Per assegurar que els teus scripts funcionin correctament i de manera eficient, segueix aquestes bones pràctiques:

  • Prova els teus scripts: Sempre prova els teus scripts en un compte de prova abans d'implementar-los en un compte real.
  • Registres i alertes: Utilitza Logger.log() per registrar accions i MailApp.sendEmail() per enviar alertes en cas d'errors.
  • Manteniment regular: Revisa i actualitza els teus scripts regularment per adaptar-los a canvis en les teves campanyes o en l'API de Google Ads.

Conclusió

L'automatització i els scripts a Google Ads són eines poderoses que poden millorar significativament l'eficiència i efectivitat de les teves campanyes de SEM. Des de tasques bàsiques com pausar anuncis de baix rendiment fins a la generació d'informes personalitzats, els scripts et permeten optimitzar les teves campanyes de manera contínua i precisa. En seguir les bones pràctiques i consells proporcionats, podràs aprofitar al màxim aquestes eines i portar les teves campanyes de Google Ads al següent nivell.
