Introducció

En aquest projecte final, aplicaràs tot el que has après sobre SEM (Màrqueting en Motors de Cerca) per crear i optimitzar una campanya completa. Aquest projecte et permetrà consolidar els teus coneixements i adquirir experiència pràctica en la gestió de campanyes SEM.

Objectius del Projecte

  • Crear una campanya SEM des de zero.
  • Optimitzar la campanya per millorar el seu rendiment.
  • Analitzar els resultats i realitzar ajustos basats en dades.

Pas 1: Investigació de Paraules Clau

La investigació de paraules clau és el primer pas crucial en la creació d'una campanya SEM efectiva.

  • Eines d'Investigació:

    • Google Keyword Planner
    • SEMrush
    • Ahrefs

  • Tipus de Paraules Clau:

    • Paraules clau de cua curta: Tenen un alt volum de cerca però també alta competència.
    • Paraules clau de cua llarga: Tenen un menor volum de cerca però són més específiques i tenen menor competència.
# Exemple d'ús de l'API de Google Keyword Planner per obtenir idees de paraules clau
from googleads import adwords

def obtenir_idees_paraules_clau(client, paraules_clau):
    targeting_idea_service = client.GetService('TargetingIdeaService', version='v201809')
    selector = {
        'ideaType': 'KEYWORD',
        'requestType': 'IDEAS',
        'requestedAttributeTypes': ['KEYWORD_TEXT', 'SEARCH_VOLUME'],
        'searchParameters': [
            {
                'xsi_type': 'RelatedToQuerySearchParameter',
                'queries': paraules_clau
            }
        ],
        'paging': {
            'startIndex': 0,
            'numberResults': 10
        }
    }
    page = targeting_idea_service.get(selector)
    for result in page['entries']:
        data = result['data']
        print(f"Paraula clau: {data['KEYWORD_TEXT']['value']}, Volum de cerca: {data['SEARCH_VOLUME']['value']}")

# Exemple d'ús
client = adwords.AdWordsClient.LoadFromStorage()
obtenir_idees_paraules_clau(client, ['màrqueting digital'])

Pas 2: Creació de la Campanya

Un cop tens les teves paraules clau, el següent pas és crear la campanya a Google Ads.

  • Configuració de la Campanya:
    • Objectiu de la campanya: Vendes, leads, trànsit del lloc web, etc.
    • Pressupost diari: Defineix quant estàs disposat a gastar per dia.
    • Segmentació geogràfica: Selecciona les ubicacions on vols que es mostrin els teus anuncis.
    • Segmentació per dispositiu: Decideix si vols mostrar els teus anuncis en mòbils, ordinadors, o ambdós.
# Exemple de creació d'una campanya a Google Ads
from googleads import adwords

def crear_campanya(client, nom_campanya, pressupost_diari, ubicacions):
    campaign_service = client.GetService('CampaignService', version='v201809')
    operations = [{
        'operator': 'ADD',
        'operand': {
            'name': nom_campanya,
            'status': 'PAUSED',
            'advertisingChannelType': 'SEARCH',
            'biddingStrategyConfiguration': {
                'biddingStrategyType': 'MANUAL_CPC'
            },
            'budget': {
                'budgetId': pressupost_diari
            },
            'settings': [
                {
                    'xsi_type': 'GeoTargetTypeSetting',
                    'positiveGeoTargetType': 'DONT_CARE',
                    'negativeGeoTargetType': 'DONT_CARE'
                }
            ],
            'networkSetting': {
                'targetGoogleSearch': True,
                'targetSearchNetwork': True,
                'targetContentNetwork': False,
                'targetPartnerSearchNetwork': False
            },
            'startDate': '20230101',
            'endDate': '20231231',
            'adServingOptimizationStatus': 'ROTATE'
        }
    }]
    campaign_service.mutate(operations)

# Exemple d'ús
client = adwords.AdWordsClient.LoadFromStorage()
crear_campanya(client, 'La Meva Primera Campanya', 5000000, ['ES'])

Pas 3: Creació de Grups d'Anuncis i Anuncis

Dins de la teva campanya, crearàs grups d'anuncis que continguin anuncis específics i paraules clau relacionades.

  • Grups d'Anuncis:

    • Nom del grup d'anuncis: Ha de ser descriptiu i reflectir el tema del grup.
    • Paraules clau: Assigna les paraules clau rellevants a cada grup d'anuncis.

  • Anuncis:

    • Títol: Atraure l'atenció de l'usuari.
    • Descripció: Explica clarament el que ofereixes.
    • URL visible: La URL que es mostra a l'anunci.
    • URL de destinació: La URL a la qual es dirigeix l'usuari en fer clic.
# Exemple de creació d'un grup d'anuncis i anuncis a Google Ads
from googleads import adwords

def crear_grup_anuncis(client, campaign_id, nom_grup, paraules_clau, anuncis):
    ad_group_service = client.GetService('AdGroupService', version='v201809')
    operations = [{
        'operator': 'ADD',
        'operand': {
            'campaignId': campaign_id,
            'name': nom_grup,
            'status': 'ENABLED',
            'biddingStrategyConfiguration': {
                'bids': [{
                    'xsi_type': 'CpcBid',
                    'bid': {
                        'microAmount': 1000000
                    }
                }]
            }
        }
    }]
    ad_group_service.mutate(operations)

    ad_group_criterion_service = client.GetService('AdGroupCriterionService', version='v201809')
    for paraula_clau in paraules_clau:
        operations = [{
            'operator': 'ADD',
            'operand': {
                'xsi_type': 'BiddableAdGroupCriterion',
                'adGroupId': ad_group_id,
                'criterion': {
                    'xsi_type': 'Keyword',
                    'text': paraula_clau,
                    'matchType': 'BROAD'
                }
            }
        }]
        ad_group_criterion_service.mutate(operations)

    ad_group_ad_service = client.GetService('AdGroupAdService', version='v201809')
    for anunci in anuncis:
        operations = [{
            'operator': 'ADD',
            'operand': {
                'adGroupId': ad_group_id,
                'ad': {
                    'xsi_type': 'ExpandedTextAd',
                    'headlinePart1': anunci['titol1'],
                    'headlinePart2': anunci['titol2'],
                    'description': anunci['descripcio'],
                    'finalUrls': [anunci['url_desti']]
                },
                'status': 'ENABLED'
            }
        }]
        ad_group_ad_service.mutate(operations)

# Exemple d'ús
client = adwords.AdWordsClient.LoadFromStorage()
crear_grup_anuncis(client, '1234567890', 'Grup d\'Anuncis 1', ['paraula clau 1', 'paraula clau 2'], [{'titol1': 'Títol 1', 'titol2': 'Títol 2', 'descripcio': 'Descripció de l\'anunci', 'url_desti': 'http://www.exemple.com'}])

Pas 4: Optimització de la Campanya

L'optimització és un procés continu que implica ajustar i millorar la campanya per obtenir millors resultats.

  • Ajustos d'Oferta:

    • Per dispositiu: Augmenta o disminueix les ofertes segons el rendiment en diferents dispositius.
    • Per ubicació: Ajusta les ofertes segons el rendiment en diferents ubicacions geogràfiques.

  • Anàlisi de Rendiment:

    • CTR (Click-Through Rate): Percentatge de clics en relació amb les impressions.
    • CPC (Cost-Per-Click): Cost mitjà per clic.
    • Conversions: Nombre d'accions desitjades realitzades pels usuaris.
# Exemple d'ajust d'oferta per dispositiu a Google Ads
from googleads import adwords

def ajustar_oferta_dispositiu(client, campaign_id, dispositiu, ajust_percentatge):
    campaign_criterion_service = client.GetService('CampaignCriterionService', version='v201809')
    operations = [{
        'operator': 'SET',
        'operand': {
            'campaignId': campaign_id,
            'criterion': {
                'xsi_type': 'Platform',
                'id': dispositiu
            },
            'bidModifier': ajust_percentatge
        }
    }]
    campaign_criterion_service.mutate(operations)

# Exemple d'ús
client = adwords.AdWordsClient.LoadFromStorage()
ajustar_oferta_dispositiu(client, '1234567890', '30001', 1.2)  # 30001 és l'ID per a mòbils
