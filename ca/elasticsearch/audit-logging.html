Introducció al Registre d'Auditoria

El registre d'auditoria és una característica crítica per monitoritzar i rastrejar les activitats dels usuaris dins d'un clúster d'Elasticsearch. Ajuda a mantenir la seguretat, el compliment i la supervisió operativa registrant esdeveniments com ara inici de sessió d'usuaris, accés a dades i canvis de configuració.

Conceptes Clau

  • Registres d'Auditoria: Registres d'esdeveniments que capturen accions d'usuaris i canvis del sistema.
  • Compliment: Assegurar que el sistema s'adhereix als requisits normatius.
  • Seguretat: Monitoritzar l'accés no autoritzat i possibles violacions de seguretat.
  • Supervisió Operativa: Rastrejar canvis i activitats per a la resolució de problemes i la gestió del sistema.

Configuració del Registre d'Auditoria

Per habilitar el registre d'auditoria a Elasticsearch, cal configurar el fitxer elasticsearch.yml. Aquí tens una guia pas a pas:

Passos de Configuració

  1. Habilitar el Registre d'Auditoria:

    xpack.security.audit.enabled: true

  2. Especificar la Destinació de Sortida:

    • Fitxer de Registre: 
      xpack.security.audit.outputs: [file]
xpack.security.audit.logfile.events.emit_request_body: true
xpack.security.audit.logfile.prefix: "audit"
xpack.security.audit.logfile.events.include: ["access_granted", "access_denied"]
    • Índex: 
      xpack.security.audit.outputs: [index]
xpack.security.audit.index.events.emit_request_body: true
xpack.security.audit.index.events.include: ["access_granted", "access_denied"]

  3. Filtrar Esdeveniments:

    • Incloure esdeveniments específics: 
      xpack.security.audit.logfile.events.include: ["access_granted", "access_denied"]
    • Excloure esdeveniments específics: 
      xpack.security.audit.logfile.events.exclude: ["authentication_success"]

Exemple de Configuració

Aquí tens un exemple complet d'una configuració elasticsearch.yml per al registre d'auditoria:

xpack.security.audit.enabled: true
xpack.security.audit.outputs: [file, index]
xpack.security.audit.logfile.events.emit_request_body: true
xpack.security.audit.logfile.prefix: "audit"
xpack.security.audit.logfile.events.include: ["access_granted", "access_denied"]
xpack.security.audit.index.events.emit_request_body: true
xpack.security.audit.index.events.include: ["access_granted", "access_denied"]

Anàlisi dels Registres d'Auditoria

Un cop habilitat el registre d'auditoria, pots analitzar els registres per monitoritzar les activitats dels usuaris i els canvis del sistema.

Accés als Fitxers de Registre

Els registres d'auditoria es guarden al fitxer de registre o índex especificat. Per exemple, si els registres es guarden en un fitxer, pots accedir-hi a la ubicació configurada:

/var/log/elasticsearch/audit.log

Consultar els Registres d'Auditoria a Kibana

Si els registres d'auditoria es guarden en un índex, pots utilitzar Kibana per consultar i visualitzar els registres:

GET /_search
{
  "query": {
    "match": {
      "event.type": "access_granted"
    }
  }
}

Exemple d'Entrada de Registre

Aquí tens un exemple d'una entrada de registre d'auditoria:

{
  "timestamp": "2023-10-01T12:34:56Z",
  "principal": "user1",
  "action": "access_granted",
  "indices": ["index1"],
  "request": {
    "method": "GET",
    "uri": "/index1/_search"
  }
}

Millors Pràctiques

  • Monitorització Regular: Revisa regularment els registres d'auditoria per detectar qualsevol activitat inusual.
  • Polítiques de Retenció: Implementa polítiques de retenció de registres per gestionar l'emmagatzematge i el compliment.
  • Alertes: Configura alertes per a esdeveniments crítics com ara access_denied o authentication_failures.
  • Emmagatzematge Segur: Assegura't que els registres d'auditoria es guarden de manera segura per evitar manipulacions.

Conclusió

El registre d'auditoria a Elasticsearch és una característica essencial per mantenir la seguretat, el compliment i la supervisió operativa. En habilitar i configurar correctament el registre d'auditoria, pots monitoritzar les activitats dels usuaris, detectar accessos no autoritzats i assegurar-te que el teu clúster d'Elasticsearch opera dins dels requisits normatius. L'anàlisi regular i les millors pràctiques t'ajudaran a treure el màxim profit de les capacitats del registre d'auditoria.
