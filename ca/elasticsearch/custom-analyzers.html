En aquesta secció, explorarem els analitzadors personalitzats a Elasticsearch. Els analitzadors són components essencials a Elasticsearch que processen el text durant la indexació i la cerca. Els analitzadors personalitzats et permeten adaptar el procés d'anàlisi de text a les teves necessitats específiques.

Què és un Analitzador?

Un analitzador a Elasticsearch és una combinació de:

  • Filtres de caràcters: Modifiquen el flux de caràcters abans de la tokenització.
  • Tokenitzador: Divideix el flux de caràcters en tokens.
  • Filtres de tokens: Modifiquen, afegeixen o eliminen tokens.

Analitzadors Integrats vs. Personalitzats

Elasticsearch proporciona diversos analitzadors integrats, però de vegades poden no complir els teus requisits específics. Els analitzadors personalitzats et permeten definir la teva pròpia combinació de filtres de caràcters, tokenitzadors i filtres de tokens.

Analitzadors Integrats

Alguns analitzadors integrats comuns inclouen:

  • Analitzador Estàndard: L'analitzador per defecte que proporciona un bon equilibri per a la majoria dels casos d'ús.
  • Analitzador d'Espais en Blanc: Tokenitza el text basant-se en espais en blanc.
  • Analitzador de Paraules Clau: Tracta tota l'entrada com un únic token.

Analitzadors Personalitzats

Els analitzadors personalitzats es defineixen en la configuració de l'índex i permeten un control més granular sobre el procés d'anàlisi de text.

Creació d'un Analitzador Personalitzat

Per crear un analitzador personalitzat, cal definir-lo en la configuració de l'índex. Aquí tens una guia pas a pas:

Pas 1: Definir Filtres de Caràcters

Els filtres de caràcters preprocessen el text abans de la tokenització. Per exemple, podries voler eliminar etiquetes HTML o normalitzar certs caràcters.

"char_filter": {
  "html_strip": {
    "type": "html_strip"
  }
}

Pas 2: Definir un Tokenitzador

El tokenitzador divideix el text en tokens. Els tokenitzadors comuns inclouen standard, whitespace i keyword.

"tokenizer": {
  "my_tokenizer": {
    "type": "standard"
  }
}

Pas 3: Definir Filtres de Tokens

Els filtres de tokens processen els tokens generats pel tokenitzador. Poden modificar, afegir o eliminar tokens.

"filter": {
  "lowercase_filter": {
    "type": "lowercase"
  },
  "stop_filter": {
    "type": "stop",
    "stopwords": ["the", "a", "an"]
  }
}

Pas 4: Combinar-los en un Analitzador

Finalment, combina els filtres de caràcters, el tokenitzador i els filtres de tokens en un analitzador personalitzat.

"analysis": {
  "char_filter": {
    "html_strip": {
      "type": "html_strip"
    }
  },
  "tokenizer": {
    "my_tokenizer": {
      "type": "standard"
    }
  },
  "filter": {
    "lowercase_filter": {
      "type": "lowercase"
    },
    "stop_filter": {
      "type": "stop",
      "stopwords": ["the", "a", "an"]
    }
  },
  "analyzer": {
    "custom_analyzer": {
      "type": "custom",
      "char_filter": ["html_strip"],
      "tokenizer": "my_tokenizer",
      "filter": ["lowercase_filter", "stop_filter"]
    }
  }
}

Exemple Complet

Aquí tens un exemple complet de creació d'un índex amb un analitzador personalitzat:

PUT /my_index
{
  "settings": {
    "analysis": {
      "char_filter": {
        "html_strip": {
          "type": "html_strip"
        }
      },
      "tokenizer": {
        "my_tokenizer": {
          "type": "standard"
        }
      },
      "filter": {
        "lowercase_filter": {
          "type": "lowercase"
        },
        "stop_filter": {
          "type": "stop",
          "stopwords": ["the", "a", "an"]
        }
      },
      "analyzer": {
        "custom_analyzer": {
          "type": "custom",
          "char_filter": ["html_strip"],
          "tokenizer": "my_tokenizer",
          "filter": ["lowercase_filter", "stop_filter"]
        }
      }
    }
  },
  "mappings": {
    "properties": {
      "content": {
        "type": "text",
        "analyzer": "custom_analyzer"
      }
    }
  }
}

Pots provar el teu analitzador personalitzat utilitzant l'API _analyze per assegurar-te que funciona com s'espera.

Conclusió

Els analitzadors personalitzats a Elasticsearch proporcionen eines poderoses per adaptar l'anàlisi de text a les teves necessitats específiques. Combinant filtres de caràcters, tokenitzadors i filtres de tokens, pots crear analitzadors sofisticats que millorin la precisió i la rellevància dels teus resultats de cerca. Entendre i utilitzar analitzadors personalitzats pot millorar significativament el rendiment i les capacitats de les teves aplicacions Elasticsearch.
