Introducció a Kibana

Kibana és una eina de visualització i exploració de dades de codi obert utilitzada per a l'anàlisi de registres i sèries temporals, el monitoratge d'aplicacions i els casos d'ús d'intel·ligència operativa. Proporciona funcions potents i fàcils d'utilitzar per cercar, visualitzar i interactuar amb les dades emmagatzemades en els índexs d'Elasticsearch.

Conceptes Clau

  • Quadre de comandament de Kibana: Una col·lecció de visualitzacions, cerques i mapes, típicament utilitzada per monitoritzar i analitzar dades.
  • Visualitzacions: Representacions gràfiques de dades, com ara gràfics de barres, gràfics de línies, gràfics de sectors, etc.
  • Descobrir: Una funció que et permet explorar i cercar les teves dades.
  • Canvas: Un espai creatiu per construir pantalles personalitzades i perfectes al píxel.
  • Timelion: Un visualitzador de dades de sèries temporals.
  • Eines de desenvolupament: Eines per interactuar amb l'API d'Elasticsearch directament des de Kibana.

Configuració de Kibana

Per utilitzar Kibana, necessites tenir Elasticsearch en funcionament. Kibana es connecta a Elasticsearch i proporciona una interfície web per interactuar amb les teves dades.

Instal·lació

  1. Descarregar Kibana: Visita la pàgina de descàrrega de Kibana i descarrega la versió adequada per al teu sistema operatiu.
  2. Extreure el paquet: Descomprimeix el fitxer descarregat.
  3. Configurar Kibana: Edita el fitxer kibana.yml per establir elasticsearch.hosts apuntant a la teva instància d'Elasticsearch.
  4. Iniciar Kibana: Executa el script bin/kibana per iniciar Kibana.
# Exemple d'inici de Kibana
./bin/kibana

Exploració de dades amb Kibana

Kibana proporciona diverses eines per explorar i visualitzar les teves dades.

Descobrir

La funció Descobrir et permet cercar i filtrar les teves dades. Pots desar cerques i utilitzar-les en visualitzacions i quadres de comandament.

  • Barra de cerca: Utilitza la barra de cerca per introduir consultes.
  • Selector de temps: Filtra dades per rang de temps.
  • Llista de camps: Visualitza i filtra per camps en les teves dades.
# Exemple d'una consulta simple en el Descobrir de Kibana
{
  "query": {
    "match": {
      "message": "error"
    }
  }
}

Visualitzacions

Les visualitzacions són el nucli de les capacitats de representació de dades de Kibana. Pots crear diversos tipus de visualitzacions, com ara gràfics de barres, gràfics de línies, gràfics de sectors i més.

  • Creació de visualitzacions: Utilitza la pestanya "Visualize" per crear noves visualitzacions.
  • Tipus de visualitzacions:
    • Gràfic de barres: Útil per comparar diferents categories.
    • Gràfic de línies: Ideal per a dades de sèries temporals.
    • Gràfic de sectors: Bo per mostrar proporcions.
# Exemple d'una configuració simple de gràfic de barres
{
  "type": "bar",
  "aggs": {
    "x": {
      "terms": {
        "field": "status.keyword"
      }
    },
    "y": {
      "count": {}
    }
  }
}

Quadres de comandament

Els quadres de comandament són col·leccions de visualitzacions i cerques que proporcionen una visió completa de les teves dades.

  • Creació de quadres de comandament: Utilitza la pestanya "Dashboard" per crear i gestionar quadres de comandament.
  • Afegir visualitzacions: Afegeix visualitzacions existents al teu quadre de comandament.
  • Personalització del disseny: Organitza i redimensiona les visualitzacions per adaptar-les a les teves necessitats.
# Exemple d'una configuració de quadre de comandament
{
  "title": "Monitoratge de Servidors",
  "panels": [
    {
      "type": "visualization",
      "id": "cpu-usage"
    },
    {
      "type": "visualization",
      "id": "memory-usage"
    }
  ]
}

Funcions Avançades

Kibana també ofereix funcions avançades per a una anàlisi i visualització de dades més complexes.

Canvas

Canvas és una eina potent per crear pantalles personalitzades i perfectes al píxel. Et permet combinar dades de múltiples fonts i crear visualitzacions altament personalitzades.

  • Elements: Afegeix text, imatges i visualitzacions al teu canvas.
  • Fonts de dades: Utilitza consultes d'Elasticsearch per obtenir dades al teu canvas.
  • Estilització: Personalitza l'aparença dels teus elements.

Timelion

Timelion és un visualitzador de dades de sèries temporals que et permet realitzar càlculs i transformacions complexes sobre les teves dades.

  • Expressions de Timelion: Utilitza el llenguatge d'expressions de Timelion per crear consultes complexes.
  • Funcions en cadena: Combina múltiples funcions per transformar les teves dades.
# Exemple d'una expressió de Timelion
.es(index="logstash-*", metric="avg:response_time").label("Temps de Resposta Mitjà")

Eines de Desenvolupament

Les Eines de Desenvolupament proporcionen una consola per interactuar amb l'API d'Elasticsearch directament des de Kibana. Això és útil per executar consultes complexes i gestionar el teu clúster d'Elasticsearch.

# Exemple d'una consulta en Eines de Desenvolupament
GET /_search
{
  "query": {
    "match_all": {}
  }
}

Conclusió

Kibana és una eina potent per visualitzar i explorar dades emmagatzemades en Elasticsearch. Comprenent les seves funcions clau i capacitats, pots crear quadres de comandament i visualitzacions perspicaces per monitoritzar i analitzar les teves dades de manera efectiva. Tant si ets un usuari principiant com avançat, Kibana proporciona les eines que necessites per treure el màxim profit de les teves dades d'Elasticsearch.
