Introducció a l'Anàlisi de Dades a Elasticsearch

Elasticsearch és un potent motor de cerca i anàlisi que et permet emmagatzemar, cercar i analitzar grans volums de dades ràpidament i gairebé en temps real. Aquesta secció et guiarà a través dels conceptes bàsics d'analitzar dades utilitzant Elasticsearch, des de consultes simples fins a agregacions avançades.

Conceptes Clau

  • Índex: Una col·lecció de documents que comparteixen característiques similars.
  • Document: Un objecte JSON que s'emmagatzema a Elasticsearch.
  • Camp: Un parell clau-valor dins d'un document.
  • Consulta: Una sol·licitud per recuperar dades de Elasticsearch.
  • Agregació: Una eina poderosa per resumir i analitzar dades.

Consultes Bàsiques

Elasticsearch utilitza un llenguatge de consultes anomenat Query DSL (Domain Specific Language). Aquí tens algunes consultes bàsiques per començar:

Consulta Match

Una consulta match s'utilitza per cercar documents que coincideixin amb un text, número, data o valor booleà donat.

GET /my_index/_search
{
  "query": {
    "match": {
      "field_name": "search_text"
    }
  }
}

Consulta Term

Una consulta term s'utilitza per cercar coincidències exactes d'un terme.

GET /my_index/_search
{
  "query": {
    "term": {
      "field_name": "exact_term"
    }
  }
}

Consultes Avançades

A mesura que et familiaritzis amb les consultes bàsiques, pots començar a explorar consultes més avançades.

Consulta Bool

Una consulta bool et permet combinar múltiples consultes utilitzant lògica booleana.

GET /my_index/_search
{
  "query": {
    "bool": {
      "must": [
        { "match": { "field1": "value1" }},
        { "match": { "field2": "value2" }}
      ],
      "filter": [
        { "term": { "status": "active" }}
      ]
    }
  }
}

Consulta Range

Una consulta range s'utilitza per trobar documents amb valors dins d'un rang especificat.

GET /my_index/_search
{
  "query": {
    "range": {
      "age": {
        "gte": 30,
        "lte": 40
      }
    }
  }
}

Agregacions

Les agregacions s'utilitzen per analitzar i resumir dades. Es poden utilitzar per calcular mètriques, construir histogrames, i més.

Agregacions de Mètriques

Les agregacions de mètriques calculen mètriques com suma, mitjana, mínim i màxim.

GET /my_index/_search
{
  "aggs": {
    "average_age": {
      "avg": {
        "field": "age"
      }
    }
  }
}

Agregacions de Cubetes

Les agregacions de cubetes agrupen documents en cubetes basades en valors de camp.

Agregació de Termes

Una agregació de termes agrupa documents per valors únics d'un camp.

GET /my_index/_search
{
  "aggs": {
    "group_by_status": {
      "terms": {
        "field": "status"
      }
    }
  }
}

Agregació d'Histograma de Dates

Una agregació d'histograma de dates agrupa documents per intervals de dates.

GET /my_index/_search
{
  "aggs": {
    "sales_over_time": {
      "date_histogram": {
        "field": "sale_date",
        "interval": "month"
      }
    }
  }
}

Combinant Agregacions

Pots combinar múltiples agregacions per realitzar anàlisis complexes.

GET /my_index/_search
{
  "aggs": {
    "sales_over_time": {
      "date_histogram": {
        "field": "sale_date",
        "interval": "month"
      },
      "aggs": {
        "total_sales": {
          "sum": {
            "field": "amount"
          }
        }
      }
    }
  }
}

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'analitzar dades a Elasticsearch. Hem començat amb consultes simples i hem avançat cap a consultes i agregacions més avançades. Entenent aquests conceptes, pots aprofitar Elasticsearch per realitzar anàlisis de dades poderoses i obtenir informació valuosa de les teves dades. Practica aquestes consultes i agregacions per esdevenir proficient en l'ús d'Elasticsearch per a l'anàlisi de dades.
