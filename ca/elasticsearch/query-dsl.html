Introducció a Query DSL

Elasticsearch Query DSL (Domain Specific Language) és una manera potent i flexible de consultar dades emmagatzemades a Elasticsearch. Et permet construir consultes complexes utilitzant sintaxi JSON.

  • Què és Query DSL?
    • Un llenguatge de consulta basat en JSON.
    • Permet consultes complexes i niades.
    • Suporta filtratge, agregació i cerca de text complet.

Consultes Bàsiques

Consulta Match

La consulta match s'utilitza per cercar un terme específic en un camp.

{
  "query": {
    "match": {
      "field_name": "search_term"
    }
  }
}

Consulta Term

La consulta term s'utilitza per a coincidències exactes.

{
  "query": {
    "term": {
      "field_name": "exact_term"
    }
  }
}

Consulta Range

La consulta range s'utilitza per trobar documents dins d'un rang específic.

{
  "query": {
    "range": {
      "field_name": {
        "gte": 10,
        "lte": 20
      }
    }
  }
}

Consultes Compostes

Consulta Bool

La consulta bool et permet combinar múltiples consultes utilitzant lògica booleana.

{
  "query": {
    "bool": {
      "must": [
        { "match": { "field_name": "search_term" } }
      ],
      "filter": [
        { "range": { "field_name": { "gte": 10, "lte": 20 } } }
      ]
    }
  }
}
  • must: Totes les condicions han de ser certes.
  • filter: Les condicions han de ser certes, però no afecten la puntuació.
  • should: Almenys una condició hauria de ser certa.
  • must_not: Les condicions no han de ser certes.

Agregacions

Les agregacions et permeten realitzar càlculs complexos i anàlisi de dades.

Agregació Terms

L'agregació terms s'utilitza per agrupar documents per un camp específic.

{
  "aggs": {
    "group_by_field": {
      "terms": {
        "field": "field_name"
      }
    }
  }
}

Agregació Date Histogram

L'agregació date histogram s'utilitza per agrupar documents per intervals de data.

{
  "aggs": {
    "sales_over_time": {
      "date_histogram": {
        "field": "date_field",
        "interval": "month"
      }
    }
  }
}

Consultes Avançades

Consulta Nested

La consulta nested s'utilitza per cercar dins d'objectes niats.

{
  "query": {
    "nested": {
      "path": "nested_path",
      "query": {
        "bool": {
          "must": [
            { "match": { "nested_field": "search_term" } }
          ]
        }
      }
    }
  }
}

Consulta Script

La consulta script et permet utilitzar scripts personalitzats per a consultes més complexes.

{
  "query": {
    "script": {
      "script": {
        "source": "doc['field_name'].value > 10",
        "lang": "painless"
      }
    }
  }
}

Conclusió

Elasticsearch Query DSL és una eina versàtil i potent per a consultar i analitzar dades. Entenent les consultes bàsiques i avançades, pots aprofitar Elasticsearch al màxim. Practica amb diferents tipus de consultes i agregacions per convertir-te en un expert en l'ús de Elasticsearch Query DSL.

  • Punts Clau:
    • Query DSL està basat en JSON i suporta consultes complexes.
    • Les consultes bàsiques inclouen match, term i range.
    • Les consultes compostes com les bool permeten combinar múltiples condicions.
    • Les agregacions permeten una anàlisi de dades avançada.
    • Les consultes avançades inclouen nested i script.

Dominant aquests conceptes, estaràs ben preparat per gestionar una àmplia gamma de tasques de cerca i anàlisi a Elasticsearch.
