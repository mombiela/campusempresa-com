Instal·lant Elasticsearch ►

Elasticsearch és un motor de cerca i analítica potent i de codi obert dissenyat per a l'escalabilitat horitzontal, la fiabilitat i les capacitats de cerca en temps real. Està construït sobre Apache Lucene i forma part de l'Elastic Stack, que inclou eines com Kibana, Logstash i Beats.

Conceptes Clau

  • Motor de Cerca: Elasticsearch s'utilitza principalment per a funcionalitats de cerca, permetent buscar a través de grans volums de dades ràpidament.
  • Temps Real: Elasticsearch proporciona capacitats de cerca i analítica en temps real.
  • Escalabilitat: Està dissenyat per escalar horitzontalment, el que significa que pots afegir més nodes al teu clúster per manejar més dades i consultes.
  • API RESTful: Elasticsearch utilitza una API RESTful, fent que sigui fàcil d'interactuar amb mètodes HTTP com GET, POST, PUT i DELETE.
  • Sense Esquema: Elasticsearch és sense esquema, el que significa que pots indexar dades sense definir un esquema prèviament.

Components Bàsics

  • Índex: Un índex és una col·lecció de documents. És similar a una base de dades en un sistema de bases de dades relacionals.
  • Document: Un document és una unitat bàsica d'informació que es pot indexar. És similar a una fila en una taula en una base de dades relacional.
  • Fragment: Un índex es pot dividir en múltiples fragments, que es distribueixen entre els nodes en un clúster.
  • Clúster: Un clúster és una col·lecció d'un o més nodes (servidors) que junts contenen totes les teves dades i proporcionen capacitats d'indexació i cerca federades.

Ús Bàsic

Creació d'un Índex

Per crear un índex a Elasticsearch, pots utilitzar la següent petició HTTP:

PUT /my_first_index

Indexació d'un Document

Per indexar un document, pots utilitzar la següent petició HTTP:

POST /my_first_index/_doc/1
{
  "name": "John Doe",
  "age": 30,
  "occupation": "Software Engineer"
}

Cerca d'un Document

Per cercar un document, pots utilitzar la següent petició HTTP:

GET /my_first_index/_search
{
  "query": {
    "match": {
      "occupation": "Software Engineer"
    }
  }
}

Eliminació d'un Índex

Per eliminar un índex, pots utilitzar la següent petició HTTP:

DELETE /my_first_index

Conclusió

Elasticsearch és una eina versàtil i potent per a la cerca i l'analítica, oferint capacitats en temps real i escalabilitat horitzontal. La seva API RESTful i la seva naturalesa sense esquema el fan accessible i flexible per a diversos casos d'ús, des de funcionalitats de cerca simples fins a analítiques de dades complexes. Entendre els seus components bàsics i operacions és essencial per aprofitar tot el seu potencial en les teves aplicacions.
