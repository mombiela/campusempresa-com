La cerca de text complet és una de les característiques més potents d'Elasticsearch, permetent-te cercar a través de grans volums de text de manera ràpida i eficient. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en la cerca de text complet utilitzant Elasticsearch.

Introducció a la cerca de text complet

La cerca de text complet es refereix a tècniques per cercar una col·lecció de documents emmagatzemats en una base de dades. A diferència de les cerques simples per paraules clau, la cerca de text complet pot gestionar consultes complexes i retornar resultats més rellevants.

Conceptes clau

  • Document: La unitat bàsica d'informació a Elasticsearch, similar a una fila en una base de dades relacional.
  • Índex: Una col·lecció de documents, similar a una taula en una base de dades relacional.
  • Camp: Un parell clau-valor en un document, similar a una columna en una base de dades relacional.
  • Analitzador: Un component que processa el text durant la indexació i la consulta, descomponent-lo en tokens o termes.

Configuració de la cerca de text complet

Abans d'endinsar-te en la cerca de text complet, assegura't de tenir Elasticsearch instal·lat i en funcionament. Pots descarregar-lo des del lloc web oficial d'Elasticsearch.

Creació d'un índex i afegir documents

Primer, crea un índex i afegeix alguns documents. Aquí tens un exemple utilitzant cURL:

# Crear un índex
curl -X PUT "localhost:9200/my_index"

# Afegir un document
curl -X POST "localhost:9200/my_index/_doc/1" -H 'Content-Type: application/json' -d'
{
  "title": "Elasticsearch Basics",
  "content": "Elasticsearch is a distributed, RESTful search and analytics engine."
}
'

# Afegir un altre document
curl -X POST "localhost:9200/my_index/_doc/2" -H 'Content-Type: application/json' -d'
{
  "title": "Advanced Elasticsearch",
  "content": "This course covers advanced topics in Elasticsearch."
}
'

Cerca de text complet bàsica

Elasticsearch proporciona una manera senzilla de realitzar cerques de text complet utilitzant la consulta match.

Exemple: Consulta bàsica de coincidència

curl -X GET "localhost:9200/my_index/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "match": {
      "content": "Elasticsearch"
    }
  }
}
'

Aquesta consulta cerca documents a l'índex my_index on el camp content conté el terme "Elasticsearch".

Tècniques avançades de cerca de text complet

A mesura que et familiaritzis amb la cerca de text complet bàsica, pots explorar tècniques més avançades per refinar les teves cerques.

Consulta de coincidència múltiple

La consulta multi_match et permet cercar a través de múltiples camps.

curl -X GET "localhost:9200/my_index/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "multi_match": {
      "query": "Elasticsearch",
      "fields": ["title", "content"]
    }
  }
}
'

Consulta de coincidència de frase

La consulta match_phrase cerca frases exactes en lloc de termes individuals.

curl -X GET "localhost:9200/my_index/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "match_phrase": {
      "content": "distributed, RESTful search"
    }
  }
}
'

Consultes booleanes

Les consultes booleanes combinen múltiples consultes utilitzant operadors lògics com must, should, i must_not.

curl -X GET "localhost:9200/my_index/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "bool": {
      "must": [
        { "match": { "content": "Elasticsearch" } }
      ],
      "should": [
        { "match": { "title": "Advanced" } }
      ],
      "must_not": [
        { "match": { "content": "basic" } }
      ]
    }
  }
}
'

Analitzadors i tokenitzadors

Els analitzadors i tokenitzadors juguen un paper crucial en com es processa i s'indexa el text a Elasticsearch.

Analitzadors integrats

Elasticsearch inclou diversos analitzadors integrats, com standard, simple, i whitespace.

Analitzadors personalitzats

També pots crear analitzadors personalitzats per satisfer necessitats específiques.

curl -X PUT "localhost:9200/my_index" -H 'Content-Type: application/json' -d'
{
  "settings": {
    "analysis": {
      "analyzer": {
        "my_custom_analyzer": {
          "type": "custom",
          "tokenizer": "whitespace",
          "filter": ["lowercase"]
        }
      }
    }
  }
}
'

Conclusió

La cerca de text complet a Elasticsearch és una eina potent que et permet cercar a través de grans volums de text de manera eficient. Entenent els conceptes bàsics de creació d'índexs i documents, realitzant cerques simples i avançades, i utilitzant analitzadors, pots aprofitar Elasticsearch al màxim. Continua experimentant amb diferents consultes i analitzadors per dominar la cerca de text complet a Elasticsearch.
