Les agregacions a Elasticsearch et permeten realitzar anàlisis de dades complexes i operacions de resum sobre les teves dades indexades. Aquest tema et guiarà des dels conceptes bàsics d'agregacions fins a conceptes més avançats, proporcionant explicacions clares, exemples i exercicis.

Introducció a les Agregacions

Les agregacions són una característica poderosa a Elasticsearch que et permeten extreure informació de les teves dades. Et permeten agrupar, filtrar i realitzar càlculs sobre les teves dades.

Conceptes Clau

  • Buckets: Contenidors que contenen documents que coincideixen amb un criteri específic.
  • Mètriques: Càlculs realitzats sobre els documents en un bucket.
  • Agregacions de Pipeline: Agregacions que prenen la sortida d'altres agregacions com a entrada.

Tipus d'Agregacions

Elasticsearch ofereix diversos tipus d'agregacions, cadascuna amb diferents propòsits. Cobrirem les més utilitzades.

Agregacions de Bucket

Les agregacions de bucket creen buckets de documents basats en certs criteris.

Agregació de Termes

Agrupa documents per un camp específic.

{
  "aggs": {
    "genres": {
      "terms": {
        "field": "genre.keyword"
      }
    }
  }
}

Agregació de Rang

Agrupa documents en rangs.

{
  "aggs": {
    "price_ranges": {
      "range": {
        "field": "price",
        "ranges": [
          { "to": 50 },
          { "from": 50, "to": 100 },
          { "from": 100 }
        ]
      }
    }
  }
}

Agregacions de Mètriques

Les agregacions de mètriques calculen mètriques, com ara mitjanes i sumes, dels documents en un bucket.

Agregació de Mitjana

Calcula el valor mitjà d'un camp numèric.

{
  "aggs": {
    "average_price": {
      "avg": {
        "field": "price"
      }
    }
  }
}

Agregació de Suma

Calcula la suma d'un camp numèric.

{
  "aggs": {
    "total_sales": {
      "sum": {
        "field": "sales"
      }
    }
  }
}

Agregacions de Pipeline

Les agregacions de pipeline prenen la sortida d'altres agregacions com a entrada.

Agregació de Derivada

Calcula la derivada d'una mètrica especificada.

{
  "aggs": {
    "sales_over_time": {
      "date_histogram": {
        "field": "date",
        "interval": "month"
      },
      "aggs": {
        "sales": {
          "sum": {
            "field": "sales"
          }
        },
        "sales_derivative": {
          "derivative": {
            "buckets_path": "sales"
          }
        }
      }
    }
  }
}

Agregacions Avançades

Agregacions Imbricades

Les agregacions imbricades et permeten realitzar agregacions dins de documents imbricats.

{
  "aggs": {
    "nested_comments": {
      "nested": {
        "path": "comments"
      },
      "aggs": {
        "avg_rating": {
          "avg": {
            "field": "comments.rating"
          }
        }
      }
    }
  }
}

Combinació d'Agregacions

Combinant múltiples agregacions es poden obtenir coneixements més profunds.

{
  "aggs": {
    "genres": {
      "terms": {
        "field": "genre.keyword"
      },
      "aggs": {
        "average_price": {
          "avg": {
            "field": "price"
          }
        }
      }
    }
  }
}

Exercicis

  1. Agregació Bàsica de Termes: Crea una agregació de termes per agrupar documents pel camp category.
  2. Agregació de Rang: Crea una agregació de rang per agrupar documents per rangs de preu: 0-50, 50-100 i 100+.
  3. Combinació d'Agregacions: Combina una agregació de termes en category amb una agregació de mitjana en price.

Conclusió

Les agregacions a Elasticsearch són una eina versàtil per a l'anàlisi de dades, permetent-te agrupar, filtrar i calcular mètriques sobre les teves dades. Entenent i utilitzant diferents tipus d'agregacions, des de les bàsques de bucket i mètriques fins a les més avançades imbricades i de pipeline, pots extreure informació valuosa de les teves dades. Practica amb els exercicis proporcionats per consolidar la teva comprensió i convertir-te en un expert en l'ús d'agregacions a Elasticsearch.
