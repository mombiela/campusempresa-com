Introducció als Beats

Els Beats són enviadors de dades lleugers que instal·les als teus servidors per enviar dades operacionals a Elasticsearch. Estan dissenyats per ser simples i eficients, facilitant la recollida i l'enviament de dades des de diverses fonts.

  • Filebeat: Envia fitxers de registre.
  • Metricbeat: Recull mètriques del sistema operatiu i serveis.
  • Packetbeat: Monitora el trànsit de xarxa.
  • Winlogbeat: Envia registres d'esdeveniments de Windows.
  • Auditbeat: Recull dades d'auditoria del marc d'auditoria de Linux.
  • Heartbeat: Monitora la disponibilitat dels serveis.

Instal·lació i Configuració dels Beats

Instal·lació de Filebeat

  1. Descarrega i instal·la Filebeat:

    curl -L -O https://artifacts.elastic.co/downloads/beats/filebeat/filebeat-7.10.0-amd64.deb
sudo dpkg -i filebeat-7.10.0-amd64.deb

  2. Configura Filebeat: Edita el fitxer de configuració filebeat.yml per especificar els fitxers de registre a monitorar i la instància d'Elasticsearch a la qual enviar les dades:

    filebeat.inputs:
- type: log
  enabled: true
  paths:
    - /var/log/*.log

output.elasticsearch:
  hosts: ["localhost:9200"]

  3. Inicia Filebeat:

    sudo service filebeat start

Instal·lació de Metricbeat

  1. Descarrega i instal·la Metricbeat:

    curl -L -O https://artifacts.elastic.co/downloads/beats/metricbeat/metricbeat-7.10.0-amd64.deb
sudo dpkg -i metricbeat-7.10.0-amd64.deb

  2. Configura Metricbeat: Edita el fitxer de configuració metricbeat.yml per especificar els mòduls a habilitar i la instància d'Elasticsearch a la qual enviar les dades:

    metricbeat.modules:
- module: system
  metricsets:
    - cpu
    - memory
    - network
  enabled: true
  period: 10s
  processes: ['.*']

output.elasticsearch:
  hosts: ["localhost:9200"]

  3. Inicia Metricbeat:

    sudo service metricbeat start

Ús dels Beats amb Elasticsearch

Enviament de Dades a Elasticsearch

Els Beats es poden configurar per enviar dades directament a Elasticsearch o a través de Logstash per a un processament addicional.

  • Directament a Elasticsearch: Aquesta és la configuració més senzilla on els Beats envien dades directament a Elasticsearch.
  • A través de Logstash: Aquesta configuració permet un processament i transformació de dades més complex abans que les dades siguin indexades a Elasticsearch.

Exemple: Filebeat a Elasticsearch

  1. Configura Filebeat:

    filebeat.inputs:
- type: log
  enabled: true
  paths:
    - /var/log/*.log

output.elasticsearch:
  hosts: ["localhost:9200"]

  2. Inicia Filebeat:

    sudo service filebeat start

Exemple: Metricbeat a Elasticsearch

  1. Configura Metricbeat:

    metricbeat.modules:
- module: system
  metricsets:
    - cpu
    - memory
    - network
  enabled: true
  period: 10s
  processes: ['.*']

output.elasticsearch:
  hosts: ["localhost:9200"]

  2. Inicia Metricbeat:

    sudo service metricbeat start

Monitorització i Visualització de Dades

Ús de Kibana

Kibana és l'eina de visualització per a Elasticsearch. Et permet crear quadres de comandament i visualitzacions basades en les dades recollides pels Beats.

  1. Accedeix a Kibana: Obre el teu navegador i navega a http://localhost:5601.

  2. Crea Patrons d'Índex:

    • Ves a la secció "Management".
    • Crea un patró d'índex per a les dades recollides pels Beats (per exemple, filebeat-*, metricbeat-*).

  3. Crea Visualitzacions:

    • Utilitza la secció "Visualize" per crear visualitzacions basades en les dades.
    • Combina visualitzacions en quadres de comandament per a una monitorització completa.

Exemple: Creació d'un Quadre de Comandament

  1. Crea Visualitzacions:

    • Crea visualitzacions per a l'ús de la CPU, l'ús de la memòria i el trànsit de xarxa utilitzant dades de Metricbeat.

  2. Combina Visualitzacions en un Quadre de Comandament:

    • Ves a la secció "Dashboard".
    • Afegeix les visualitzacions creades en el pas anterior a un nou quadre de comandament.

Conclusió

Els Beats són eines potents per recollir i enviar dades a Elasticsearch. Són lleugers i fàcils de configurar, cosa que els fa ideals per monitoritzar diversos aspectes de la teva infraestructura. Integrant Beats amb Elasticsearch i Kibana, pots crear solucions de monitorització completes que proporcionen informació valuosa sobre els teus sistemes i aplicacions.
