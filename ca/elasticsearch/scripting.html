Introducció a l'Scripting

L'scripting a Elasticsearch et permet personalitzar el comportament de les teves consultes i operacions. Es pot utilitzar per a una varietat de propòsits, com ara puntuació personalitzada, filtratge i transformació de dades.

  • Què és l'Scripting?

    • L'scripting és el procés d'escriure codi per automatitzar tasques o personalitzar operacions.
    • A Elasticsearch, els scripts es poden utilitzar en consultes, agregacions i actualitzacions.

  • Per què Utilitzar l'Scripting?

    • Per realitzar càlculs complexos.
    • Per personalitzar els algorismes de puntuació.
    • Per manipular dades durant l'indexació o la consulta.

Tipus de Scripts

Elasticsearch admet diversos llenguatges de scripting, cadascun amb els seus propis casos d'ús i avantatges.

  • Painless

    • El llenguatge de scripting predeterminat i recomanat.
    • Dissenyat per ser segur i eficient.
    • Sintaxi similar a Java.

  • Expressions

    • Un llenguatge simple, segur i ràpid per a operacions aritmètiques i lògiques bàsiques.
    • Funcionalitat limitada en comparació amb Painless.

  • Mustache

    • Un llenguatge de plantilles utilitzat per crear consultes dinàmiques.
    • Útil per a la parametrització de consultes.

| Llenguatge | Cas d'Ús | Rendiment | Seguretat | |------------|-------------------------------------------|-------------|-----------| | Painless | Scripting de propòsit general | Alt | Alt | | Expressions| Operacions aritmètiques i lògiques bàsiques| Molt Alt | Molt Alt | | Mustache | Plantilles de consultes dinàmiques | Moderat | Alt |

Ús de Scripts Painless

Painless és el llenguatge de scripting més versàtil i potent a Elasticsearch. Aquí tens com utilitzar-lo:

Scripts Inline

Els scripts inline s'escriuen directament dins de la consulta.

{
  "query": {
    "script_score": {
      "query": {
        "match": {
          "message": "elasticsearch"
        }
      },
      "script": {
        "source": "doc['likes'].value * 2"
      }
    }
  }
}
  • Explicació: Aquest script multiplica el camp likes per 2 per a finalitats de puntuació.

Scripts Emmagatzemats

Els scripts emmagatzemats estan predefinits i emmagatzemats a Elasticsearch, permetent la seva reutilització i millor seguretat.

PUT _scripts/calculate-score
{
  "script": {
    "lang": "painless",
    "source": "doc['likes'].value * params.factor"
  }
}
  • Explicació: Aquest script calcula una puntuació multiplicant el camp likes per un paràmetre factor.

Ús de Paràmetres

Es poden passar paràmetres als scripts per fer-los més flexibles i segurs.

{
  "query": {
    "script_score": {
      "query": {
        "match": {
          "message": "elasticsearch"
        }
      },
      "script": {
        "source": "doc['likes'].value * params.factor",
        "params": {
          "factor": 2
        }
      }
    }
  }
}
  • Explicació: Aquest script utilitza un paràmetre factor per multiplicar el camp likes.

Tècniques Avançades de Scripting

A mesura que et familiaritzis amb l'scripting, pots explorar tècniques avançades.

  • Puntuació Personalitzada

    • Crear algorismes de puntuació complexos basats en múltiples camps i condicions.

  • Transformació de Dades

    • Modificar documents durant la indexació o la consulta per adaptar-los a necessitats específiques.

  • Agregacions

    • Utilitzar scripts per crear agregacions personalitzades per a una anàlisi més detallada.

Exemple: Puntuació Personalitzada

{
  "query": {
    "function_score": {
      "query": {
        "match_all": {}
      },
      "script_score": {
        "script": {
          "source": "Math.log(2 + doc['likes'].value)"
        }
      }
    }
  }
}
  • Explicació: Aquest script utilitza el logaritme del camp likes més 2 per calcular la puntuació.

Conclusió

L'scripting a Elasticsearch és una característica potent que permet una personalització i optimització extensiva de les teves operacions de cerca i indexació. Comprenent els conceptes bàsics dels llenguatges de scripting com Painless, i aprenent a utilitzar scripts inline i emmagatzemats, pots desbloquejar tot el potencial d'Elasticsearch per als teus casos d'ús específics. A mesura que avancis, podràs explorar escenaris i tècniques més complexes per millorar encara més les teves aplicacions d'Elasticsearch.
