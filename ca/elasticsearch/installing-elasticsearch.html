Introducció

Elasticsearch és un potent motor de cerca i anàlisi que s'utilitza àmpliament per a diverses aplicacions, incloent l'anàlisi de dades de registres i esdeveniments, la cerca de text complet, i més. Aquesta secció et guiarà a través del procés d'instal·lació d'Elasticsearch en diferents sistemes operatius.

Prerequisits

Abans d'instal·lar Elasticsearch, assegura't de tenir els següents prerequisits:

  • Java Development Kit (JDK) 8 o superior instal·lat.
  • Recursos suficients del sistema (RAM, CPU, espai en disc).

Instal·lant Elasticsearch en Diferents Sistemes Operatius

Instal·lant en Windows

  1. Descarrega Elasticsearch:

    • Visita la pàgina de descàrrega d'Elasticsearch.
    • Descarrega el fitxer ZIP per a Windows.

  2. Extreu el fitxer ZIP:

    • Extreu el fitxer ZIP descarregat a un directori de la teva elecció, per exemple, C:\elasticsearch.

  3. Executa Elasticsearch:

    • Obre una finestra de Command Prompt.
    • Navega al directori bin dins de la carpeta extreta.
    • Executa la següent comanda: 
      elasticsearch.bat
    • Elasticsearch s'iniciarà, i pots accedir-hi navegant a http://localhost:9200 en el teu navegador web.

Instal·lant en macOS

  1. Descarrega Elasticsearch:

    • Visita la pàgina de descàrrega d'Elasticsearch.
    • Descarrega el fitxer TAR per a macOS.

  2. Extreu el fitxer TAR:

    • Obre una finestra de Terminal.
    • Navega al directori on es va descarregar el fitxer TAR.
    • Executa la següent comanda per extreure el fitxer: 
      tar -xzf elasticsearch-<version>.tar.gz

  3. Executa Elasticsearch:

    • Navega al directori bin dins de la carpeta extreta.
    • Executa la següent comanda: 
      ./elasticsearch
    • Elasticsearch s'iniciarà, i pots accedir-hi navegant a http://localhost:9200 en el teu navegador web.

Instal·lant en Linux

  1. Descarrega Elasticsearch:

Instal·lant via Gestors de Paquets

Utilitzant Homebrew en macOS

  1. Instal·la Homebrew:

    • Si no tens Homebrew instal·lat, instal·la'l executant la següent comanda en Terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·la Elasticsearch:

    • Executa la següent comanda per instal·lar Elasticsearch: 
      brew tap elastic/tap
brew install elastic/tap/elasticsearch-full

  3. Executa Elasticsearch:

    • Inicia Elasticsearch executant: 
      brew services start elastic/tap/elasticsearch-full

Utilitzant APT en Linux basat en Debian

  1. Instal·la APT Transport HTTPS:

    • Executa la següent comanda: 
      sudo apt-get install apt-transport-https

  2. Afegeix la clau GPG d'Elasticsearch:

    • Executa la següent comanda: 
      wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -

  3. Afegeix el repositori d'Elasticsearch:

    • Executa la següent comanda: 
      echo "deb https://artifacts.elastic.co/packages/7.x/apt stable main" | sudo tee -a /etc/apt/sources.list.d/elastic-7.x.list

  4. Instal·la Elasticsearch:

    • Actualitza la llista de paquets i instal·la Elasticsearch: 
      sudo apt-get update
sudo apt-get install elasticsearch

  5. Executa Elasticsearch:

    • Inicia Elasticsearch executant: 
      sudo systemctl start elasticsearch

Verificant la Instal·lació

Després d'instal·lar Elasticsearch, verifica que estigui funcionant correctament navegant a http://localhost:9200 en el teu navegador web. Hauries de veure una resposta JSON similar a la següent:

  "name" : "your-node-name",
  "cluster_name" : "elasticsearch",
  "cluster_uuid" : "your-cluster-uuid",
