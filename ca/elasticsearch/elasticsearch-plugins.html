Introducció als Plugins d'Elasticsearch

Els plugins d'Elasticsearch són mòduls que es poden afegir a un clúster d'Elasticsearch per ampliar la seva funcionalitat. Els plugins poden proporcionar funcions addicionals, millorar el rendiment o integrar-se amb altres sistemes.

Conceptes Clau

  • Tipus de Plugins: Elasticsearch admet diversos tipus de plugins, incloent-hi plugins d'anàlisi, plugins de descobriment i plugins de seguretat.
  • Instal·lació: Els plugins es poden instal·lar utilitzant la interfície de línia de comandes d'Elasticsearch.
  • Gestió: Els plugins es poden gestionar (instal·lar, eliminar, actualitzar) utilitzant comandes d'Elasticsearch.

Tipus de Plugins

Plugins d'Anàlisi

Els plugins d'anàlisi amplien les capacitats de les funcions d'anàlisi de text d'Elasticsearch. Poden proporcionar analitzadors, tokenitzadors i filtres addicionals.

  • Exemple: El plugin analysis-icu proporciona anàlisi de text avançada per a diferents idiomes.
# Instal·lant el plugin analysis-icu
bin/elasticsearch-plugin install analysis-icu

Plugins de Descobriment

Els plugins de descobriment ajuden els nodes d'Elasticsearch a descobrir-se entre ells en un clúster, especialment en entorns de núvol.

  • Exemple: El plugin discovery-ec2 permet que Elasticsearch utilitzi AWS EC2 per al descobriment de nodes.
# Instal·lant el plugin discovery-ec2
bin/elasticsearch-plugin install discovery-ec2

Plugins de Seguretat

Els plugins de seguretat afegeixen funcions com l'autenticació, l'autorització i el xifratge a Elasticsearch.

  • Exemple: El plugin x-pack proporciona funcions de seguretat, monitoratge i alertes.
# Instal·lant el plugin x-pack
bin/elasticsearch-plugin install x-pack

Instal·lació i Gestió de Plugins

Instal·lació

Per instal·lar un plugin, utilitza la comanda elasticsearch-plugin seguida del subcomandament install i el nom del plugin.

# Sintaxi general per instal·lar un plugin
bin/elasticsearch-plugin install <plugin-name>

Llistar Plugins Instal·lats

Per llistar tots els plugins instal·lats, utilitza el subcomandament list.

# Llistant tots els plugins instal·lats
bin/elasticsearch-plugin list

Eliminació de Plugins

Per eliminar un plugin, utilitza el subcomandament remove seguit del nom del plugin.

# Eliminant un plugin
bin/elasticsearch-plugin remove <plugin-name>

Exemple: Instal·lant i Utilitzant el Plugin Ingest Attachment

Instal·lació

El plugin ingest-attachment et permet indexar i cercar adjunts com PDFs i documents de Word.

# Instal·lant el plugin ingest-attachment
bin/elasticsearch-plugin install ingest-attachment

Utilitzant el Plugin

Després d'instal·lar el plugin, pots utilitzar-lo en les teves pipelines d'ingestió.

PUT _ingest/pipeline/attachment
{
  "description": "Extract attachment information",
  "processors": [
    {
      "attachment": {
        "field": "data"
      }
    }
  ]
}
PUT my_index/_doc/1?pipeline=attachment
{
  "data": "<base64 encoded attachment>"
}

Conclusió

Els plugins d'Elasticsearch són eines poderoses que amplien la funcionalitat del teu clúster d'Elasticsearch. Tant si necessites anàlisi de text avançada, descobriment de nodes basat en el núvol, o funcions de seguretat millorades, probablement hi ha un plugin que s'ajusta a les teves necessitats. Entendre com instal·lar, gestionar i utilitzar aquests plugins és essencial per aprofitar tot el potencial d'Elasticsearch.
