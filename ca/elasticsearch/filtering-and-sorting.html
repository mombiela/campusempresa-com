Introducció

En aquesta secció, explorarem com filtrar i ordenar dades a Elasticsearch. El filtratge i l'ordenació són operacions essencials per consultar grans conjunts de dades de manera eficient. Cobrirem els conceptes bàsics i gradualment passarem a conceptes més avançats.

Filtratge de Dades

El filtratge et permet restringir els resultats de la cerca basant-te en criteris específics. Elasticsearch proporciona diverses maneres de filtrar dades.

Filtres Bàsics

Els filtres bàsics són senzills i fàcils d'utilitzar. Inclouen filtres de terme, rang i existència.

  • Filtre de Terme: Filtra documents que contenen un terme exacte en un camp proporcionat.

    {
  "query": {
    "term": {
      "status": "active"
    }
  }
}

  • Filtre de Rang: Filtra documents que tenen camps dins d'un cert rang.

    {
  "query": {
    "range": {
      "age": {
        "gte": 30,
        "lte": 40
      }
    }
  }
}

  • Filtre d'Existència: Filtra documents que contenen un camp específic.

    {
  "query": {
    "exists": {
      "field": "email"
    }
  }
}

Filtres Compostos

Els filtres compostos combinen múltiples filtres utilitzant operadors lògics com must, should i must_not.

  • Consulta Bool: Combina múltiples filtres utilitzant lògica booleana.

    {
  "query": {
    "bool": {
      "must": [
        { "term": { "status": "active" } },
        { "range": { "age": { "gte": 30, "lte": 40 } } }
      ],
      "must_not": [
        { "term": { "role": "admin" } }
      ]
    }
  }
}

Filtres Avançats

Els filtres avançats proporcionen més flexibilitat i potència.

  • Filtre Anidat: Filtra documents que contenen objectes o arrays anidats.

    {
  "query": {
    "nested": {
      "path": "comments",
      "query": {
        "bool": {
          "must": [
            { "match": { "comments.author": "John" } },
            { "range": { "comments.date": { "gte": "2022-01-01" } } }
          ]
        }
      }
    }
  }
}

Ordenació de Dades

L'ordenació et permet ordenar els resultats de la cerca basant-te en camps específics.

Ordenació Bàsica

L'ordenació bàsica és simple i implica ordenar per un sol camp.

  • Ordenar per Camp: Ordena documents per un camp especificat en ordre ascendent o descendent.

    {
  "sort": [
    { "age": { "order": "asc" } }
  ]
}

Ordenació Multinivell

L'ordenació multinivell et permet ordenar per múltiples camps.

  • Ordenar per Múltiples Camps: Ordena documents per múltiples camps en un ordre especificat.

    {
  "sort": [
    { "age": { "order": "asc" } },
    { "name": { "order": "desc" } }
  ]
}

Ordenació de Camps Anidats

L'ordenació també es pot aplicar a camps anidats dins dels documents.

  • Ordenar per Camp Anidat: Ordena documents basant-se en camps dins d'objectes anidats.

    {
  "sort": [
    {
      "comments.date": {
        "order": "desc",
        "nested": {
          "path": "comments"
        }
      }
    }
  ]
}

Combinació de Filtratge i Ordenació

Sovint, necessitaràs filtrar i ordenar dades simultàniament. Elasticsearch et permet combinar aquestes operacions sense problemes.

  • Filtrar i Ordenar: Combina filtratge i ordenació en una sola consulta.

    {
  "query": {
    "bool": {
      "must": [
        { "term": { "status": "active" } },
        { "range": { "age": { "gte": 30, "lte": 40 } } }
      ]
    }
  },
  "sort": [
    { "age": { "order": "asc" } }
  ]
}

Conclusió

El filtratge i l'ordenació són funcions poderoses a Elasticsearch que et permeten consultar i organitzar les teves dades de manera eficient. Comprenent i utilitzant filtres bàsics, compostos i avançats, així com tècniques d'ordenació, pots realitzar consultes complexes i recuperar dades en un ordre significatiu. Practica aquests conceptes amb diversos conjunts de dades per convertir-te en un expert en la gestió de consultes a Elasticsearch.
