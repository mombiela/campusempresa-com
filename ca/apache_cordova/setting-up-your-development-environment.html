Configurar el teu entorn de desenvolupament és el primer pas per començar a desenvolupar aplicacions utilitzant Apache Cordova. Aquesta secció et guiarà a través de les eines i configuracions necessàries per preparar el teu entorn per al desenvolupament amb Cordova.

Prerequisits

Abans d'entrar en la configuració, assegura't de tenir els següents prerequisits instal·lats a la teva màquina:

  • Node.js i npm: Cordova està construït sobre Node.js, i npm (Node Package Manager) s'utilitza per instal·lar Cordova i els seus plugins.
  • Java Development Kit (JDK): Necessari per al desenvolupament d'Android.
  • Android Studio: Necessari per construir i executar aplicacions Android.
  • Xcode: Necessari per construir i executar aplicacions iOS (només macOS).

Instal·lant Node.js i npm

Node.js i npm són essencials per al desenvolupament amb Cordova. Segueix aquests passos per instal·lar-los:

  1. Descarrega Node.js: Visita el lloc web de Node.js i descarrega la versió LTS per al teu sistema operatiu.
  2. Instal·la Node.js: Executa l'instal·lador i segueix les instruccions.
  3. Verifica la Instal·lació: Obre un terminal o símbol del sistema i executa les següents comandes per verificar la instal·lació: 
    node -v
npm -v
    Hauries de veure els números de versió de Node.js i npm.

Instal·lant Apache Cordova

Un cop Node.js i npm estan instal·lats, pots instal·lar Apache Cordova utilitzant npm:

  1. Instal·la Cordova: Obre un terminal o símbol del sistema i executa:

    npm install -g cordova

    El flag -g instal·la Cordova globalment al teu sistema.

  2. Verifica la Instal·lació: Executa la següent comanda per verificar la instal·lació:

    cordova -v

    Hauries de veure el número de versió de Cordova.

Configurant l'Entorn de Desenvolupament d'Android

Per desenvolupar aplicacions Android amb Cordova, necessites configurar l'entorn de desenvolupament d'Android:

  1. Instal·la Java Development Kit (JDK):

    • Descarrega el JDK des del lloc web d'Oracle.
    • Instal·la el JDK seguint les instruccions per al teu sistema operatiu.
    • Configura la variable d'entorn JAVA_HOME per apuntar al directori d'instal·lació del JDK.

  2. Instal·la Android Studio:

    • Descarrega Android Studio des del lloc web oficial.
    • Instal·la Android Studio i segueix l'assistent de configuració per instal·lar els components necessaris del SDK.

  3. Configura l'SDK d'Android:

    • Obre Android Studio i ves a File > Settings > Appearance & Behavior > System Settings > Android SDK.
    • Assegura't que les plataformes i eines necessàries del SDK estan instal·lades.

  4. Configura les Variables d'Entorn:

    • Configura la variable d'entorn ANDROID_HOME per apuntar al directori del SDK d'Android.
    • Afegeix els directoris platform-tools i tools al PATH del teu sistema.

Configurant l'Entorn de Desenvolupament d'iOS (només macOS)

Per desenvolupar aplicacions iOS amb Cordova, necessites configurar l'entorn de desenvolupament d'iOS:

  1. Instal·la Xcode:

    • Descarrega Xcode des de la Mac App Store.
    • Instal·la Xcode i obre'l per completar la configuració inicial.

  2. Instal·la les Eines de Línia de Comandes de Xcode:

    • Obre un terminal i executa: 
      xcode-select --install

  3. Configura l'Simulador d'iOS:

    • Obre Xcode i ves a Preferences > Components.
    • Descarrega i instal·la els simuladors d'iOS necessaris.

Creant el teu Primer Projecte Cordova

Amb el teu entorn de desenvolupament configurat, ara pots crear el teu primer projecte Cordova:

  1. Crea un Nou Projecte:

    • Obre un terminal o símbol del sistema i executa: 
      cordova create MyApp com.example.myapp MyApp
      Aquesta comanda crea un nou projecte Cordova anomenat MyApp amb el nom de paquet com.example.myapp.

  2. Afegeix Plataformes:

    • Navega al directori del projecte: 
      cd MyApp
    • Afegeix la plataforma Android: 
      cordova platform add android
    • (Opcional) Afegeix la plataforma iOS (només macOS): 
      cordova platform add ios

  3. Construeix el Projecte:

    • Construeix el projecte per Android: 
      cordova build android
    • (Opcional) Construeix el projecte per iOS (només macOS): 
      cordova build ios

Conclusió

Configurar el teu entorn de desenvolupament per Apache Cordova implica instal·lar eines necessàries com Node.js, npm, JDK, Android Studio i Xcode. Un cop aquestes eines estan instal·lades i configurades, pots crear i construir el teu primer projecte Cordova. Aquesta configuració forma la base per desenvolupar aplicacions mòbils multiplataforma utilitzant Cordova.
