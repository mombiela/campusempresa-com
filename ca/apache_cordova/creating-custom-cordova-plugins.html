Introducció

Crear plugins personalitzats de Cordova permet als desenvolupadors ampliar la funcionalitat de les seves aplicacions Cordova integrant codi natiu. Aquest tema et guiarà a través del procés de creació, implementació i ús de plugins personalitzats de Cordova des de nivells principiants fins a avançats.

Prerequisits

Abans de submergir-te en la creació de plugins personalitzats de Cordova, assegura't de tenir:

  • Coneixements bàsics d'Apache Cordova.
  • Familiaritat amb JavaScript.
  • Comprensió bàsica del desenvolupament natiu (Java per a Android, Swift/Objective-C per a iOS).

Configuració del teu entorn

Per crear un plugin personalitzat de Cordova, necessites configurar el teu entorn de desenvolupament.

  1. Instal·la Node.js i npm: Cordova requereix Node.js i npm.

    # Instal·la Node.js i npm
sudo apt-get install nodejs npm

  2. Instal·la Cordova CLI:

    # Instal·la Cordova globalment
npm install -g cordova

  3. Crea un projecte Cordova:

    # Crea un nou projecte Cordova
cordova create MyCordovaApp com.example.mycordovaapp MyCordovaApp
cd MyCordovaApp

Creació d'un plugin personalitzat

Pas 1: Estructura del plugin

Un plugin de Cordova típicament té la següent estructura:

my-plugin/
├── src/
│   ├── android/
│   │   └── MyPlugin.java
│   ├── ios/
│   │   └── MyPlugin.swift
├── www/
│   └── myplugin.js
├── plugin.xml
└── package.json

Pas 2: Defineix el plugin a `plugin.xml`

El fitxer plugin.xml és el fitxer de configuració del teu plugin. Defineix les metadades del plugin, el codi específic de la plataforma i la interfície JavaScript.

<plugin id="com.example.myplugin" version="1.0.0" xmlns="http://apache.org/cordova/ns/plugins/1.0" xmlns:android="http://schemas.android.com/apk/res/android">
    <name>MyPlugin</name>
    <description>Un plugin d'exemple senzill</description>
    <license>Apache 2.0</license>
    <keywords>cordova,plugin</keywords>
    <engines>
        <engine name="cordova" version=">=9.0.0"/>
    </engines>
    <js-module src="www/myplugin.js" name="MyPlugin">
        <clobbers target="cordova.plugins.MyPlugin"/>
    </js-module>
    <platform name="android">
        <source-file src="src/android/MyPlugin.java" target-dir="src/com/example/myplugin"/>
    </platform>
    <platform name="ios">
        <source-file src="src/ios/MyPlugin.swift" />
    </platform>
</plugin>

Pas 3: Implementa la interfície JavaScript

Crea el fitxer www/myplugin.js per definir la interfície JavaScript del teu plugin.

var exec = require('cordova/exec');

var MyPlugin = {
    coolMethod: function(arg0, success, error) {
        exec(success, error, 'MyPlugin', 'coolMethod', [arg0]);
    }
};

module.exports = MyPlugin;

Pas 4: Implementa el codi natiu

Android (Java)

Crea el fitxer src/android/MyPlugin.java per implementar la funcionalitat nativa per a Android.

package com.example.myplugin;

import org.apache.cordova.CordovaPlugin;
import org.apache.cordova.CallbackContext;
import org.json.JSONArray;
import org.json.JSONException;

public class MyPlugin extends CordovaPlugin {
    @Override
    public boolean execute(String action, JSONArray args, CallbackContext callbackContext) throws JSONException {
        if (action.equals("coolMethod")) {
            String message = args.getString(0);
            this.coolMethod(message, callbackContext);
            return true;
        }
        return false;
    }

    private void coolMethod(String message, CallbackContext callbackContext) {
        if (message != null && message.length() > 0) {
            callbackContext.success(message);
        } else {
            callbackContext.error("Expected one non-empty string argument.");
        }
    }
}

iOS (Swift)

Crea el fitxer src/ios/MyPlugin.swift per implementar la funcionalitat nativa per a iOS.

import Foundation
import Cordova

@objc(MyPlugin) class MyPlugin: CDVPlugin {
    @objc(coolMethod:)
    func coolMethod(command: CDVInvokedUrlCommand) {
        let message = command.arguments[0] as? String ?? ""
        let pluginResult = CDVPluginResult(status: CDVCommandStatus_OK, messageAs: message)
        self.commandDelegate.send(pluginResult, callbackId: command.callbackId)
    }
}

Ús del plugin personalitzat

  1. Afegeix el plugin al teu projecte:

    # Afegeix el plugin personalitzat al teu projecte Cordova
cordova plugin add path/to/my-plugin

  2. Crida el plugin des de JavaScript:

    document.addEventListener('deviceready', function() {
    cordova.plugins.MyPlugin.coolMethod('Hello, Cordova!', function(success) {
        console.log('Success:', success);
    }, function(error) {
        console.error('Error:', error);
    });
}, false);

Conclusió

Crear plugins personalitzats de Cordova et permet aprofitar les capacitats natives del dispositiu i ampliar la funcionalitat de les teves aplicacions Cordova. Seguint els passos descrits en aquest tema, pots crear, implementar i utilitzar plugins personalitzats de manera efectiva. Recorda provar els teus plugins a fons tant en plataformes Android com iOS per assegurar la compatibilitat i el rendiment.
