Introducció

Apache Cordova permet als desenvolupadors crear aplicacions mòbils utilitzant HTML, CSS i JavaScript. Una de les característiques potents de Cordova és la seva capacitat per accedir a les funcionalitats natives del dispositiu a través de plugins. En aquesta secció, explorarem com utilitzar el Plugin de la Càmera per capturar fotos i recuperar imatges de la galeria del dispositiu.

Configuració del Plugin de la Càmera

Instal·lació del Plugin

Per utilitzar el Plugin de la Càmera, primer has d'instal·lar-lo en el teu projecte Cordova. Obre el terminal i navega fins al directori del teu projecte, després executa la següent comanda:

cordova plugin add cordova-plugin-camera

Aquesta comanda afegirà el Plugin de la Càmera al teu projecte, permetent-te accedir a la càmera del dispositiu i a la biblioteca de fotos.

Configuració de Permisos

Per a Android i iOS, has de configurar els permisos necessaris en els fitxers de configuració respectius.

  • Android: Assegura't que el fitxer AndroidManifest.xml inclogui els següents permisos:
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
  • iOS: Actualitza el fitxer config.xml per incloure les següents preferències:
<edit-config target="NSCameraUsageDescription" file="*-Info.plist" mode="merge">
    <string>Necessitem accedir a la teva càmera per fer fotos.</string>
</edit-config>
<edit-config target="NSPhotoLibraryUsageDescription" file="*-Info.plist" mode="merge">
    <string>Necessitem accedir a la teva biblioteca de fotos per seleccionar fotos.</string>
</edit-config>

Captura de Fotos

Ús Bàsic

Per capturar una foto utilitzant el Plugin de la Càmera, pots utilitzar el mètode navigator.camera.getPicture. Aquí tens un exemple bàsic:

document.getElementById('capturePhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 50,
        destinationType: Camera.DestinationType.DATA_URL
    });

    function onSuccess(imageData) {
        var image = document.getElementById('myImage');
        image.src = "data:image/jpeg;base64," + imageData;
    }

    function onFail(message) {
        alert('Ha fallat perquè: ' + message);
    }
});
  • Paràmetres:
    • quality: Especifica la qualitat de la imatge guardada (0-100).
    • destinationType: Determina el format de la imatge retornada (per exemple, DATA_URL retorna una cadena codificada en base64).

Opcions Avançades

El mètode navigator.camera.getPicture accepta diverses opcions per personalitzar el comportament de la càmera:

  • sourceType: Especifica la font de la imatge (per exemple, Camera.PictureSourceType.CAMERA o Camera.PictureSourceType.PHOTOLIBRARY).
  • allowEdit: Permet una edició simple de la imatge abans de la selecció.
  • targetWidth i targetHeight: Especifica l'amplada i l'alçada de la imatge.

Exemple amb opcions avançades:

document.getElementById('capturePhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 75,
        destinationType: Camera.DestinationType.FILE_URI,
        sourceType: Camera.PictureSourceType.CAMERA,
        allowEdit: true,
        targetWidth: 1000,
        targetHeight: 1000,
        saveToPhotoAlbum: true
    });

    function onSuccess(imageURI) {
        var image = document.getElementById('myImage');
        image.src = imageURI;
    }

    function onFail(message) {
        alert('Ha fallat perquè: ' + message);
    }
});

Recuperació d'Imatges de la Galeria

Per recuperar imatges de la biblioteca de fotos del dispositiu, pots establir el sourceType a Camera.PictureSourceType.PHOTOLIBRARY:

document.getElementById('getPhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 50,
        destinationType: Camera.DestinationType.FILE_URI,
        sourceType: Camera.PictureSourceType.PHOTOLIBRARY
    });

    function onSuccess(imageURI) {
        var image = document.getElementById('myImage');
        image.src = imageURI;
    }

    function onFail(message) {
        alert('Ha fallat perquè: ' + message);
    }
});

Gestió d'Errors

És important gestionar els errors de manera adequada. La funció de callback onFail proporciona un missatge d'error que es pot utilitzar per informar l'usuari del que ha anat malament.

function onFail(message) {
    alert('Ha fallat perquè: ' + message);
}

Conclusió

El Plugin de la Càmera en Apache Cordova és una eina potent que et permet capturar fotos i recuperar imatges de la galeria del dispositiu. Entenent les opcions bàsiques i avançades disponibles, pots crear una experiència d'usuari rica en les teves aplicacions mòbils. Recorda sempre gestionar els errors de manera adequada per assegurar una experiència d'usuari fluida.
