El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, et guiarem a través del procés de creació del teu primer projecte Apache Cordova. Apache Cordova és un popular marc de desenvolupament d'aplicacions mòbils que et permet construir aplicacions mòbils utilitzant HTML, CSS i JavaScript. Al final d'aquesta secció, tindràs un projecte bàsic de Cordova configurat i llest per a un desenvolupament posterior.

Prerequisits

Abans de començar, assegura't de tenir instal·lat al teu sistema el següent:

  • Node.js i npm (Node Package Manager)
  • Cordova CLI (Command Line Interface)

Instal·lant Node.js i npm

Si encara no has instal·lat Node.js i npm, pots descarregar-los i instal·lar-los des del lloc web oficial de Node.js.

Instal·lant Cordova CLI

Un cop Node.js i npm estan instal·lats, pots instal·lar Cordova CLI executant la següent comanda al teu terminal o símbol del sistema:

npm install -g cordova

El flag -g instal·la Cordova globalment, fent-lo accessible des de qualsevol directori del teu sistema.

Creant un nou projecte Cordova

Per crear un nou projecte Cordova, segueix aquests passos:

  1. Obre el teu terminal o símbol del sistema.
  2. Navega al directori on vols crear el teu projecte.
  3. Executa la següent comanda:
cordova create MyApp com.example.myapp MyApp

Aquesta comanda crea un nou projecte Cordova amb els següents paràmetres:

  • MyApp: El nom del directori per al teu projecte.
  • com.example.myapp: L'identificador d'estil de domini invers per al teu projecte.
  • MyApp: El nom de visualització de la teva aplicació.

Estructura del projecte

Després d'executar la comanda cordova create, tindràs un nou directori anomenat MyApp amb l'estructura següent:

MyApp/
├── config.xml
├── hooks/
├── platforms/
├── plugins/
└── www/
  • config.xml: El fitxer de configuració per al teu projecte Cordova.
  • hooks/: Scripts que es poden activar en diferents etapes del procés de construcció de Cordova.
  • platforms/: Conté codi específic de la plataforma (per exemple, Android, iOS).
  • plugins/: Conté els plugins de Cordova utilitzats en el teu projecte.
  • www/: Conté els teus actius web (HTML, CSS, JavaScript).

Afegint plataformes

Per construir el teu projecte Cordova per a diferents plataformes, necessites afegir les plataformes desitjades. Per exemple, per afegir la plataforma Android, executa la següent comanda:

cordova platform add android

De la mateixa manera, per afegir la plataforma iOS, executa:

cordova platform add ios

Llistant les plataformes instal·lades

Pots llistar les plataformes afegides al teu projecte executant:

cordova platform ls

Construint el projecte

Un cop hagis afegit les plataformes necessàries, pots construir el teu projecte utilitzant la següent comanda:

cordova build

Aquesta comanda compila el teu projecte per a totes les plataformes afegides. També pots construir per a una plataforma específica especificant el nom de la plataforma:

cordova build android

Executant el projecte

Per executar el teu projecte Cordova en un emulador o un dispositiu connectat, utilitza la següent comanda:

cordova run android

Substitueix android per ios si vols executar el projecte en un dispositiu o emulador iOS.

Conclusió

En aquesta secció, has après com crear el teu primer projecte Cordova, afegir plataformes, construir el projecte i executar-lo en un emulador o dispositiu. Aquesta és la base per desenvolupar aplicacions mòbils utilitzant Apache Cordova. En les següents seccions, aprofundirem en la personalització del teu projecte, l'ús de plugins i l'optimització de la teva aplicació per a diferents plataformes.
