Introducció a Cordova CLI

Apache Cordova és un popular marc de desenvolupament d'aplicacions mòbils que et permet utilitzar tecnologies web estàndard com HTML5, CSS3 i JavaScript per al desenvolupament multiplataforma. La Interfície de Línia de Comandes (CLI) de Cordova és una eina potent que simplifica el procés de crear, construir i desplegar aplicacions Cordova.

Instal·lació de Cordova CLI

Per començar amb Cordova CLI, necessites tenir Node.js i npm (Node Package Manager) instal·lats al teu sistema.

  1. Instal·la Node.js i npm:

    • Descarrega i instal·la Node.js des de nodejs.org.
    • Verifica la instal·lació executant les següents comandes al teu terminal: 
      node -v
npm -v

  2. Instal·la Cordova CLI:

    • Utilitza npm per instal·lar Cordova globalment: 
      npm install -g cordova
    • Verifica la instal·lació comprovant la versió de Cordova: 
      cordova -v

Creació d'un nou projecte Cordova

Crear un nou projecte Cordova és senzill amb la CLI.

  1. Crea un Projecte:

    • Utilitza la comanda cordova create seguida del nom del directori, l'identificador del paquet i el nom del projecte: 
      cordova create MyApp com.example.myapp MyApp
    • Aquesta comanda crea un nou directori anomenat MyApp amb l'estructura necessària del projecte.

  2. Estructura del Projecte:

    • El projecte creat tindrà la següent estructura: 
      MyApp/
├── config.xml
├── hooks/
├── platforms/
├── plugins/
└── www/
    • config.xml: Fitxer de configuració del teu projecte Cordova.
    • hooks/: Scripts que s'executen en diverses etapes del procés de construcció de Cordova.
    • platforms/: Conté el codi específic de la plataforma (per exemple, Android, iOS).
    • plugins/: Conté els plugins de Cordova.
    • www/: Conté els recursos web (HTML, CSS, JavaScript) de la teva aplicació.

Afegir plataformes

Cordova et permet afegir múltiples plataformes al teu projecte, com Android i iOS.

  1. Afegeix una Plataforma:

    • Utilitza la comanda cordova platform add seguida del nom de la plataforma: 
      cordova platform add android
cordova platform add ios
    • Aquesta comanda afegeix la plataforma especificada al directori platforms.

  2. Llista de Plataformes:

    • Per veure les plataformes afegides al teu projecte, utilitza: 
      cordova platform ls

  3. Elimina una Plataforma:

    • Si necessites eliminar una plataforma, utilitza: 
      cordova platform rm android

Construcció del Projecte

Un cop hagis afegit les plataformes necessàries, pots construir el teu projecte.

  1. Construeix per a una Plataforma Específica:

    • Utilitza la comanda cordova build seguida del nom de la plataforma: 
      cordova build android
cordova build ios

  2. Construeix per a Totes les Plataformes:

    • Per construir el projecte per a totes les plataformes afegides, simplement utilitza: 
      cordova build

Execució del Projecte

Pots executar el teu projecte Cordova en un emulador o en un dispositiu físic.

  1. Executa en un Emulador:

    • Utilitza la comanda cordova emulate seguida del nom de la plataforma: 
      cordova emulate android
cordova emulate ios

  2. Executa en un Dispositiu Físic:

    • Utilitza la comanda cordova run seguida del nom de la plataforma: 
      cordova run android
cordova run ios

Gestió de Plugins

Els plugins amplien la funcionalitat de la teva aplicació Cordova proporcionant accés a capacitats natives del dispositiu.

  1. Afegeix un Plugin:

    • Utilitza la comanda cordova plugin add seguida de l'identificador del plugin: 
      cordova plugin add cordova-plugin-camera

  2. Llista de Plugins:

    • Per veure els plugins afegits al teu projecte, utilitza: 
      cordova plugin ls

  3. Elimina un Plugin:

    • Si necessites eliminar un plugin, utilitza: 
      cordova plugin rm cordova-plugin-camera

Conclusió

La CLI de Cordova és una eina potent que simplifica el procés de desenvolupament d'aplicacions mòbils multiplataforma. Dominant els conceptes bàsics de la CLI de Cordova, pots crear, construir i desplegar aplicacions de manera eficient. A mesura que progressis, podràs explorar funcions més avançades i plugins per millorar encara més les teves aplicacions. Feliç programació!
