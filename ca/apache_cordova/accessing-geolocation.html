Introducció

Apache Cordova proporciona una manera senzilla d'accedir a les capacitats de geolocalització d'un dispositiu. Això et permet recuperar la ubicació geogràfica del dispositiu, la qual cosa pot ser útil per a una varietat d'aplicacions, com ara mapatge, navegació i serveis basats en la ubicació.

Configurant la Geolocalització a Cordova

Instal·lant el Plugin de Geolocalització

Per utilitzar la geolocalització en el teu projecte Cordova, necessites instal·lar el plugin de Geolocalització de Cordova. Això es pot fer amb la següent comanda:

cordova plugin add cordova-plugin-geolocation

Afegint Permisos

Assegura't que la teva aplicació té els permisos necessaris per accedir a la ubicació del dispositiu. Per a Android, això implica modificar el fitxer AndroidManifest.xml, i per a iOS, actualitzar el fitxer Info.plist.

Android

Afegeix els següents permisos al teu AndroidManifest.xml:

<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

iOS

Afegeix les següents claus al teu Info.plist:

<key>NSLocationWhenInUseUsageDescription</key>
<string>Necessitem la teva ubicació per proporcionar millors serveis.</string>
<key>NSLocationAlwaysUsageDescription</key>
<string>Necessitem la teva ubicació per proporcionar millors serveis.</string>

Ús Bàsic de la Geolocalització

Obtenint la Posició Actual

Per obtenir la posició actual del dispositiu, utilitza el mètode navigator.geolocation.getCurrentPosition. Aquí tens un exemple bàsic:

document.addEventListener("deviceready", onDeviceReady, false);

function onDeviceReady() {
    navigator.geolocation.getCurrentPosition(onSuccess, onError);
}

function onSuccess(position) {
    console.log('Latitud: ' + position.coords.latitude);
    console.log('Longitud: ' + position.coords.longitude);
}

function onError(error) {
    console.error('Codi d\'error: ' + error.code + '\nMissatge: ' + error.message);
}

Gestionant Errors

La funció de callback onError gestiona qualsevol error que es produeixi mentre s'intenta recuperar la geolocalització. L'objecte d'error conté un codi i un missatge.

  • error.code pot ser:
    • 1: Permís denegat
    • 2: Posició no disponible
    • 3: Temps d'espera esgotat

Ús Avançat de la Geolocalització

Vigilant la Posició

Per monitoritzar contínuament la posició del dispositiu, utilitza el mètode navigator.geolocation.watchPosition. Això és útil per a aplicacions que necessiten seguir el moviment de l'usuari.

var watchID;

document.addEventListener("deviceready", onDeviceReady, false);

function onDeviceReady() {
    watchID = navigator.geolocation.watchPosition(onSuccess, onError, { timeout: 30000 });
}

function onSuccess(position) {
    console.log('Latitud: ' + position.coords.latitude);
    console.log('Longitud: ' + position.coords.longitude);
}

function onError(error) {
    console.error('Codi d\'error: ' + error.code + '\nMissatge: ' + error.message);
}

function stopWatching() {
    if (watchID) {
        navigator.geolocation.clearWatch(watchID);
        watchID = null;
    }
}

Opcions de Geolocalització

Pots personalitzar el comportament dels mètodes de geolocalització passant un objecte d'opcions. Les opcions inclouen:

  • enableHighAccuracy: Booleà, indica si l'aplicació vol els millors resultats possibles.
  • timeout: Temps màxim (en mil·lisegons) que el dispositiu pot trigar a retornar una posició.
  • maximumAge: Edat màxima (en mil·lisegons) d'una posició emmagatzemada que és acceptable retornar.

Exemple amb opcions:

var options = {
    enableHighAccuracy: true,
    timeout: 5000,
    maximumAge: 0
};

navigator.geolocation.getCurrentPosition(onSuccess, onError, options);

Conclusió

Accedir a la geolocalització en Apache Cordova és senzill amb l'ajuda del cordova-plugin-geolocation. Entenent l'ús bàsic i avançat dels mètodes de geolocalització, pots crear aplicacions potents basades en la ubicació. Recorda sempre gestionar els errors de manera adequada i respectar la privacitat de l'usuari sol·licitant els permisos de manera apropiada.
