Introducció

Apache Cordova et permet construir aplicacions mòbils utilitzant HTML, CSS i JavaScript. Una de les característiques essencials de qualsevol aplicació mòbil és la capacitat d'interactuar amb el sistema de fitxers. Aquest tema et guiarà a través dels conceptes bàsics de treballar amb el sistema de fitxers en Cordova, des de nivells principiants fins a avançats.

Configuració

Abans de començar a treballar amb el sistema de fitxers, assegura't de tenir l'entorn Cordova configurat. Si no, segueix aquests passos:

  1. Instal·la Node.js i npm.
  2. Instal·la Cordova globalment utilitzant npm: 
    npm install -g cordova
  3. Crea un nou projecte Cordova: 
    cordova create myApp
cd myApp
  4. Afegeix una plataforma (per exemple, Android): 
    cordova platform add android

Instal·lant el Plugin de Fitxers

Per treballar amb el sistema de fitxers, necessites instal·lar el plugin de fitxers de Cordova:

cordova plugin add cordova-plugin-file

Conceptes Bàsics

  • Sistema de Fitxers: L'estructura que emmagatzema fitxers i directoris.
  • Fitxer: Una peça individual de dades emmagatzemada en el sistema de fitxers.
  • Directori: Un contenidor que conté fitxers i altres directoris.

Accedint al Sistema de Fitxers

Per accedir al sistema de fitxers, necessites utilitzar l'objecte cordova.file, que proporciona diverses propietats que representen diferents ubicacions del sistema de fitxers.

Exemple: Accedint al Directori Arrel

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        console.log("Directori arrel:", directoryEntry);
    }, function(error) {
        console.error("Error accedint al sistema de fitxers:", error);
    });
});
  • cordova.file.dataDirectory: Representa el directori de dades de l'aplicació.
  • resolveLocalFileSystemURL: Resol una URL del sistema de fitxers a una entrada de directori.

Llegint i Escrivent Fitxers

Escriure un Fitxer

Per escriure un fitxer, necessites crear o obtenir una entrada de fitxer i després escriure dades en ell.

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("example.txt", { create: true, exclusive: false }, function(fileEntry) {
            fileEntry.createWriter(function(fileWriter) {
                var dataObj = new Blob(['Hello, world!'], { type: 'text/plain' });

                fileWriter.write(dataObj);

                fileWriter.onwriteend = function() {
                    console.log("Fitxer escrit amb èxit.");
                };

                fileWriter.onerror = function(e) {
                    console.error("Error en escriure el fitxer:", e);
                };
            });
        });
    });
});
  • getFile: Obté una entrada de fitxer, creant-la si no existeix.
  • createWriter: Crea un escriptor de fitxers per escriure dades en el fitxer.

Llegir un Fitxer

Per llegir un fitxer, necessites obtenir una entrada de fitxer i després llegir-ne el contingut.

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("example.txt", { create: false }, function(fileEntry) {
            fileEntry.file(function(file) {
                var reader = new FileReader();

                reader.onloadend = function() {
                    console.log("Contingut del fitxer:", this.result);
                };

                reader.readAsText(file);
            });
        });
    });
});
  • FileReader: Llegeix el contingut d'un fitxer.

Temes Avançats

Gestionant Directoris

Pots crear, llegir i eliminar directoris de manera similar als fitxers.

Crear un Directori

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getDirectory("newDir", { create: true }, function(newDirEntry) {
            console.log("Directori creat:", newDirEntry);
        });
    });
});

Eliminar un Directori

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getDirectory("newDir", { create: false }, function(newDirEntry) {
            newDirEntry.removeRecursively(function() {
                console.log("Directori eliminat.");
            });
        });
    });
});

Gestionant Errors

Sempre gestiona els errors per assegurar que la teva aplicació pugui recuperar-se de manera elegant dels problemes.

function onError(error) {
    console.error("Codi d'error:", error.code);
}

document.addEventListener('deviceready', function() {
    window.resolveLocalFileSystemURL(cordova.file.dataDirectory, function(directoryEntry) {
        directoryEntry.getFile("nonexistent.txt", { create: false }, null, onError);
    });
});

Resum

Treballar amb el sistema de fitxers en Apache Cordova implica entendre com accedir a diferents ubicacions del sistema de fitxers, llegir i escriure fitxers, i gestionar directoris. En dominar aquests conceptes, pots crear aplicacions mòbils robustes que gestionin eficientment les operacions de fitxers.

  • Punts Clau:
    • Instal·la el plugin de fitxers de Cordova.
    • Utilitza cordova.file per accedir a diferents ubicacions del sistema de fitxers.
    • Llegeix i escriu fitxers utilitzant FileReader i FileWriter.
    • Gestiona directoris creant-los, llegint-los i eliminant-los.
    • Sempre gestiona els errors per assegurar una experiència d'usuari suau.

Seguint aquestes directrius i exemples, hauràs d'estar ben preparat per gestionar operacions del sistema de fitxers en les teves aplicacions Cordova.
