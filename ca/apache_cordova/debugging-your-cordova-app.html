La depuració és una part crucial del procés de desenvolupament, permetent-te identificar i solucionar problemes en la teva aplicació Cordova. Aquesta secció et guiarà a través de diverses tècniques i eines de depuració, des del registre bàsic de la consola fins a la depuració avançada amb eines de desenvolupador del navegador i la depuració remota.

Tècniques bàsiques de depuració

Registre de la consola

  • Console.log(): La manera més senzilla de depurar és utilitzant console.log() per imprimir missatges a la consola. 
    console.log('Aquest és un missatge de depuració');
  • Gestió d'errors: Utilitza console.error() per registrar errors. 
    try {
  // Codi que pot generar un error
} catch (error) {
  console.error('S'ha produït un error:', error);
}

Alertes

  • Diàlegs d'alerta: Utilitza alert() per mostrar un quadre de missatge. 
    alert('Aquest és un missatge d'alerta');

Ús de les eines de desenvolupador del navegador

Inspecció d'elements

  • Inspector d'elements: Fes clic dret sobre qualsevol element de la teva aplicació i selecciona "Inspect" per veure i editar HTML/CSS.
  • Consola: Utilitza la pestanya de la consola per executar comandes JavaScript i veure registres.

Monitorització de la xarxa

  • Pestanya de xarxa: Monitoritza les sol·licituds de xarxa per assegurar-te que la teva aplicació es comunica correctament amb els servidors.
    • Comprova les sol·licituds fallides.
    • Inspecciona les capçaleres de sol·licitud i resposta.

Depuració de JavaScript

  • Punts de trencament: Configura punts de trencament en el teu codi per pausar l'execució i inspeccionar variables.
    • Fes clic al número de línia a la pestanya del codi font per configurar un punt de trencament.
    • Utilitza el panell de "Watch" per monitoritzar variables específiques.

Depuració remota

Depuració en Android

  • Chrome DevTools: Utilitza Chrome DevTools per depurar la teva aplicació Cordova executant-se en un dispositiu Android.
    • Connecta el teu dispositiu Android via USB.
    • Obre Chrome i navega a chrome://inspect.
    • Fes clic a "Inspect" sota el teu dispositiu per començar a depurar.

Depuració en iOS

  • Safari Web Inspector: Utilitza el Web Inspector de Safari per depurar la teva aplicació Cordova executant-se en un dispositiu iOS.
    • Connecta el teu dispositiu iOS via USB.
    • Obre Safari i navega a Develop > [El teu dispositiu] > [La teva aplicació].

Tècniques avançades de depuració

Ús de plugins de depuració

  • Plugin Cordova per a depuració: Instal·la plugins que proporcionen capacitats addicionals de depuració. 
    cordova plugin add cordova-plugin-console

Depuració amb Visual Studio Code

  • Depurador de VS Code: Utilitza el depurador integrat de Visual Studio Code per a una experiència més integrada.
    • Instal·la l'extensió Cordova Tools.
    • Configura el teu fitxer launch.json per configurar la depuració.
    {
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Attach to Android",
      "type": "cordova",
      "request": "attach",
      "platform": "android",
      "target": "device",
      "port": 9222,
      "sourceMaps": true,
      "cwd": "${workspaceFolder}"
    }
  ]
}

Resum

Depurar la teva aplicació Cordova de manera efectiva implica una combinació de tècniques bàsiques com el registre de la consola i les alertes, l'ús d'eines de desenvolupador del navegador per inspeccionar elements i monitoritzar sol·licituds de xarxa, i mètodes avançats com la depuració remota i l'ús de plugins especialitzats. En dominar aquestes tècniques, podràs identificar i resoldre problemes de manera eficient, assegurant un procés de desenvolupament més suau i una aplicació més robusta.
