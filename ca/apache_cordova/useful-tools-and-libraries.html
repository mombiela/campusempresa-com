En aquesta secció, explorarem diverses eines i biblioteques que poden millorar la teva experiència de desenvolupament amb Apache Cordova. Aquestes eines i biblioteques poden ajudar-te a agilitzar el teu flux de treball, depurar de manera més eficient i afegir funcionalitats avançades a les teves aplicacions Cordova.

Cordova CLI

La Interfície de Línia de Comandes de Cordova (CLI) és l'eina principal per crear, construir i gestionar aplicacions Cordova.

  • Instal·lació: Pots instal·lar el Cordova CLI utilitzant npm (Node Package Manager). 
    npm install -g cordova
  • Creació d'un Projecte: Per crear un nou projecte Cordova, utilitza la següent comanda: 
    cordova create MyApp
  • Afegir Plataformes: Afegeix plataformes com Android o iOS al teu projecte. 
    cordova platform add android
cordova platform add ios
  • Construir el Projecte: Per construir el teu projecte per a una plataforma específica. 
    cordova build android
cordova build ios
  • Executar el Projecte: Per executar el teu projecte en un emulador o un dispositiu connectat. 
    cordova run android
cordova run ios

Visual Studio Code

Visual Studio Code (VS Code) és un editor de codi potent i de codi obert que suporta una àmplia gamma de llenguatges de programació i frameworks, incloent Cordova.

  • Extensions: Hi ha diverses extensions disponibles per al desenvolupament amb Cordova.
    • Cordova Tools: Proporciona IntelliSense, depuració i altres característiques per a Cordova.
    • Debugger for Chrome: Et permet depurar la teva aplicació Cordova en el navegador Chrome.
  • Configuració: Pots configurar VS Code per treballar perfectament amb Cordova configurant les configuracions de llançament en el fitxer launch.json.

Plugins de Cordova

Els plugins de Cordova són essencials per accedir a les funcionalitats natives del dispositiu. Aquí tens alguns plugins comunament utilitzats:

  • Plugin de Dispositiu: Proporciona informació sobre el maquinari i el programari del dispositiu. 
    cordova plugin add cordova-plugin-device

    document.addEventListener("deviceready", function() {
  console.log(device.cordova);
  console.log(device.model);
  console.log(device.platform);
  console.log(device.version);
}, false);
  • Plugin de Càmera: Et permet capturar fotos utilitzant la càmera del dispositiu. 
    cordova plugin add cordova-plugin-camera

    navigator.camera.getPicture(onSuccess, onFail, { 
  quality: 50,
  destinationType: Camera.DestinationType.DATA_URL
});

function onSuccess(imageData) {
  var image = document.getElementById('myImage');
  image.src = "data:image/jpeg;base64," + imageData;
}

function onFail(message) {
  alert('Failed because: ' + message);
}
  • Plugin de Geolocalització: Proporciona accés al GPS del dispositiu per obtenir la ubicació actual. 
    cordova plugin add cordova-plugin-geolocation

    navigator.geolocation.getCurrentPosition(onSuccess, onError);

function onSuccess(position) {
  console.log('Latitude: ' + position.coords.latitude);
  console.log('Longitude: ' + position.coords.longitude);
}

function onError(error) {
  console.log('Error: ' + error.message);
}

Eines de Depuració

Una depuració efectiva és crucial per desenvolupar aplicacions Cordova robustes. Aquí tens algunes eines que poden ajudar-te:

  • Chrome DevTools: Utilitza Chrome DevTools per depurar aplicacions Cordova que s'executen en el navegador o en dispositius Android.
    • Inspeccionar Elements: Inspecciona i modifica el DOM i el CSS.
    • Consola: Registra missatges i executa codi JavaScript.
    • Xarxa: Monitoritza les sol·licituds i respostes de la xarxa.
  • Safari Web Inspector: Per depurar aplicacions Cordova en dispositius iOS.
    • Inspeccionar Elements: Similar a Chrome DevTools, inspecciona i modifica el DOM i el CSS.
    • Consola: Registra missatges i executa codi JavaScript.
    • Xarxa: Monitoritza les sol·licituds i respostes de la xarxa.

Frameworks de Proves

Les proves són una part crítica del procés de desenvolupament. Aquí tens alguns frameworks que poden ajudar-te a provar les teves aplicacions Cordova:

  • Jasmine: Un framework de desenvolupament dirigit per comportament per provar codi JavaScript. 
    npm install jasmine

    describe("A suite", function() {
  it("contains spec with an expectation", function() {
    expect(true).toBe(true);
  });
});
  • Karma: Un corredor de proves que treballa amb Jasmine per executar proves en múltiples navegadors. 
    npm install karma --save-dev
npm install karma-jasmine karma-chrome-launcher --save-dev

    module.exports = function(config) {
  config.set({
    frameworks: ['jasmine'],
    browsers: ['Chrome'],
    files: [
      'src/**/*.js',
      'test/**/*.spec.js'
    ]
  });
};

Resum

En aquesta secció, hem cobert una sèrie d'eines i biblioteques que poden millorar significativament la teva experiència de desenvolupament amb Cordova. Des del Cordova CLI per gestionar els teus projectes fins a Visual Studio Code per a un entorn de codificació potent, i des de plugins essencials de Cordova fins a eines de depuració i proves, aquests recursos t'ajudaran a construir, depurar i provar les teves aplicacions Cordova de manera més eficient. Aprofitant aquestes eines i biblioteques, pots agilitzar el teu flux de treball i centrar-te en crear aplicacions mòbils d'alta qualitat.
