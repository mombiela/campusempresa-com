Introducció

En aquest estudi de cas, recorrerem el procés de construcció d'una aplicació mòbil del món real utilitzant Apache Cordova. Això t'ajudarà a entendre com aplicar els conceptes que has après en un escenari pràctic. Cobrirem els següents passos:

  • Configuració del projecte
  • Disseny de la interfície d'usuari
  • Implementació de la funcionalitat principal
  • Proves i depuració
  • Paquetització i desplegament de l'aplicació

Configuració del Projecte

Abans de començar a construir la nostra aplicació, necessitem configurar l'entorn de desenvolupament i crear un nou projecte Cordova.

Prerequisits

  • Node.js i npm instal·lats
  • Apache Cordova instal·lat globalment via npm

Creació d'un Nou Projecte

Primer, creem un nou projecte Cordova:

cordova create MyApp com.example.myapp MyApp
cd MyApp
cordova platform add android
cordova platform add ios

Això crearà un nou projecte Cordova anomenat MyApp amb les plataformes Android i iOS afegides.

Disseny de la Interfície d'Usuari

La interfície d'usuari (UI) és una part crucial de qualsevol aplicació. En aquesta secció, dissenyarem una UI senzilla per a la nostra aplicació.

Utilitzant HTML i CSS

Les aplicacions Cordova utilitzen tecnologies web estàndard com HTML, CSS i JavaScript per a la UI. Aquí tens un exemple bàsic:

<!DOCTYPE html>
<html>
<head>
    <title>MyApp</title>
    <link rel="stylesheet" href="css/index.css">
</head>
<body>
    <div class="app">
        <h1>Welcome to MyApp</h1>
        <button id="myButton">Click Me</button>
    </div>
    <script src="js/index.js"></script>
</body>
</html>

Estilitzant la UI

Afegim alguns estils bàsics a la nostra aplicació:

/* css/index.css */
body {
    font-family: Arial, sans-serif;
    text-align: center;
    margin-top: 50px;
}

button {
    padding: 10px 20px;
    font-size: 16px;
}

Implementació de la Funcionalitat Principal

Ara que tenim una UI bàsica, afegim alguna funcionalitat a la nostra aplicació.

Afegint Event Listeners

Afegirem un event listener al botó per mostrar una alerta quan es faci clic:

// js/index.js
document.addEventListener('deviceready', function() {
    document.getElementById('myButton').addEventListener('click', function() {
        alert('Button clicked!');
    });
}, false);

Utilitzant Plugins de Cordova

Els plugins de Cordova et permeten accedir a funcions natives del dispositiu. Utilitzem el plugin Cordova Device per mostrar informació del dispositiu:

cordova plugin add cordova-plugin-device

Actualitzem el JavaScript per mostrar informació del dispositiu:

// js/index.js
document.addEventListener('deviceready', function() {
    document.getElementById('myButton').addEventListener('click', function() {
        var deviceInfo = 'Device Model: ' + device.model + '\n' +
                         'Device Platform: ' + device.platform + '\n' +
                         'Device UUID: ' + device.uuid;
        alert(deviceInfo);
    });
}, false);

Proves i Depuració

Les proves i la depuració són passos essencials en el procés de desenvolupament per assegurar que la teva aplicació funcioni com s'espera.

Utilitzant l'Emulador

Pots utilitzar l'emulador d'Android o iOS per provar la teva aplicació:

cordova emulate android
cordova emulate ios

Depurant amb Chrome DevTools

Per a Android, pots utilitzar Chrome DevTools per depurar la teva aplicació:

  1. Obre Chrome i navega a chrome://inspect.
  2. Fes clic en el teu dispositiu sota "Remote Target".
  3. Utilitza DevTools per inspeccionar i depurar la teva aplicació.

Paquetització i Desplegament de l'Aplicació

Un cop la teva aplicació estigui llesta, necessites empaquetar-la i desplegar-la a les botigues d'aplicacions.

Construint l'Aplicació

Construeix l'aplicació per a producció:

cordova build android --release
cordova build ios --release

Signant l'Aplicació

Per a Android, necessites signar l'APK:

jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore my-release-key.jks platforms/android/app/build/outputs/apk/release/app-release-unsigned.apk alias_name

Per a iOS, necessites utilitzar Xcode per signar i pujar l'aplicació a l'App Store.

Conclusió

En aquest estudi de cas, hem cobert tot el procés de construcció d'una aplicació del món real utilitzant Apache Cordova. Hem començat amb la configuració del projecte, el disseny de la UI, la implementació de la funcionalitat principal, les proves i la depuració, i finalment la paquetització i el desplegament de l'aplicació. Seguint aquests passos, ara hauries de tenir una bona comprensió de com desenvolupar i desplegar una aplicació Cordova.
