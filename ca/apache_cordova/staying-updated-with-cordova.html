Introducció

Mantenir-se actualitzat amb Apache Cordova és crucial per aprofitar les últimes característiques, pegats de seguretat i millores de rendiment. Aquesta secció et guiarà a través de les millors pràctiques i recursos per mantenir els teus projectes Cordova al dia.

Conceptes Clau

  • Versionat: Entendre el sistema de versionat de Cordova.
  • Notes de Lliurament: On trobar i com interpretar les notes de lliurament.
  • Estratègies d'Actualització: Millors pràctiques per actualitzar Cordova i els seus plugins.
  • Comunitat i Recursos: Aprofitar la comunitat i els recursos oficials per a les actualitzacions.

Versionat

Cordova segueix el versionat semàntic, que és un número de versió de tres parts: MAJOR.MINOR.PATCH.

  • MAJOR: Canvis d'API incompatibles.
  • MINOR: Funcionalitat compatible amb versions anteriors.
  • PATCH: Correccions d'errors compatibles amb versions anteriors.

Exemple

{
  "cordova": {
    "version": "10.0.0"
  }
}

En aquest exemple, 10.0.0 indica una versió major amb possibles canvis trencadors.

Notes de Lliurament

Les notes de lliurament són essencials per entendre quins canvis s'han fet en cada versió. Normalment inclouen:

  • Noves característiques
  • Correccions d'errors
  • Desaprovacions
  • Canvis trencadors

On Trobar les Notes de Lliurament

  • Blog Oficial de Cordova: Cordova Blog
  • Lliuraments a GitHub: Cordova GitHub

Estratègies d'Actualització

Actualitzar Cordova i els seus plugins es pot fer a través de la línia de comandes. Aquí tens algunes millors pràctiques:

  • Còpia de Seguretat: Sempre fes una còpia de seguretat del teu projecte abans d'actualitzar.
  • Llegeix les Notes de Lliurament: Entén els canvis i el seu impacte.
  • Prova: Prova a fons la teva aplicació després d'actualitzar.

Actualitzar el CLI de Cordova

Per actualitzar el CLI de Cordova, utilitza la següent comanda:

npm install -g cordova

Actualitzar Plugins

Per actualitzar un plugin específic, utilitza:

cordova plugin update <plugin-name>

Per exemple:

cordova plugin update cordova-plugin-camera

Conclusió

Mantenir-se actualitzat amb Cordova implica entendre el seu sistema de versionat, revisar regularment les notes de lliurament, seguir les millors pràctiques per a les actualitzacions i aprofitar la comunitat i els recursos oficials. D'aquesta manera, pots assegurar-te que els teus projectes Cordova es mantinguin segurs, amb bon rendiment i rics en característiques.
