Desplegar la teva aplicació Apache Cordova a un dispositiu físic és un pas crucial en el procés de desenvolupament. Et permet provar la teva aplicació en un entorn real, assegurant-te que funcioni bé i es comporti com s'espera en maquinari real. Aquesta secció et guiarà a través del procés des de nivells principiants fins a avançats.

Prerequisits

Abans de començar a desplegar la teva aplicació Cordova a un dispositiu físic, assegura't de tenir els següents prerequisits:

  • Apache Cordova instal·lat a la teva màquina de desenvolupament.
  • Un dispositiu físic (Android o iOS) amb el mode desenvolupador habilitat.
  • Cable USB per connectar el teu dispositiu a la teva màquina de desenvolupament.
  • SDKs i eines necessàries instal·lades (Android Studio per a Android, Xcode per a iOS).

Configurant el Teu Entorn de Desenvolupament

Android

  1. Instal·la Android Studio: Descarrega i instal·la Android Studio des del lloc web oficial.
  2. Habilita el Mode Desenvolupador al Teu Dispositiu:
    • Ves a Configuració > Quant al telèfon.
    • Toca Número de compilació set vegades per habilitar les Opcions de Desenvolupador.
    • Ves a Configuració > Opcions de desenvolupador i habilita Depuració USB.
  3. Connecta el Teu Dispositiu: Utilitza un cable USB per connectar el teu dispositiu Android a la teva màquina de desenvolupament.

iOS

  1. Instal·la Xcode: Descarrega i instal·la Xcode des de la Mac App Store.
  2. Habilita el Mode Desenvolupador al Teu Dispositiu:
    • Ves a Configuració > General > Gestió de dispositius.
    • Confia en la teva màquina de desenvolupament.
  3. Connecta el Teu Dispositiu: Utilitza un cable USB per connectar el teu dispositiu iOS al teu Mac.

Desplegant a un Dispositiu Android

Guia Pas a Pas

  1. Obre el Teu Projecte Cordova: Navega al directori del teu projecte Cordova.
  2. Afegeix la Plataforma Android: Si encara no has afegit la plataforma Android, executa: 
    cordova platform add android
  3. Construeix el Projecte: Compila el teu projecte per a Android: 
    cordova build android
  4. Desplega al Dispositiu: Assegura't que el teu dispositiu està connectat i reconegut executant: 
    adb devices
    Si el teu dispositiu està a la llista, desplega la teva aplicació amb: 
    cordova run android --device

Problemes Comuns i Resolució de Problemes

  • Dispositiu No Reconegut: Assegura't que la depuració USB està habilitada i prova de reconnectar el teu dispositiu.
  • Errors de Construcció: Comprova si falten components de l'SDK o variables d'entorn mal configurades.

Desplegant a un Dispositiu iOS

Guia Pas a Pas

  1. Obre el Teu Projecte Cordova: Navega al directori del teu projecte Cordova.
  2. Afegeix la Plataforma iOS: Si encara no has afegit la plataforma iOS, executa: 
    cordova platform add ios
  3. Obre el Projecte a Xcode: Obre el fitxer .xcworkspace generat a Xcode: 
    open platforms/ios/YourApp.xcworkspace
  4. Configura la Signatura: A Xcode, ves a la pestanya Signing & Capabilities i selecciona el teu equip de desenvolupament.
  5. Construeix i Desplega: Selecciona el teu dispositiu connectat de la llista de dispositius i fes clic al botó Run a Xcode.

Problemes Comuns i Resolució de Problemes

  • Errors de Perfil de Provisió: Assegura't que el teu compte de desenvolupador d'Apple està configurat correctament i el teu dispositiu està registrat.
  • Errors de Construcció: Comprova si falten dependències o configuracions incorrectes del projecte a Xcode.

Tècniques Avançades de Desplegament

Utilitzant Live Reload

Live reload et permet veure els canvis en temps real sense reconstruir tot el projecte. Això pot accelerar significativament el teu procés de desenvolupament.

  1. Instal·la el Plugin Live Reload: 
    cordova plugin add cordova-plugin-livereload
  2. Executa amb Live Reload: 
    cordova run android --livereload
    o per a iOS: 
    cordova run ios --livereload

Depurant en un Dispositiu Físic

  • Android: Utilitza Chrome DevTools per inspeccionar i depurar la teva aplicació. Obre Chrome i navega a chrome://inspect.
  • iOS: Utilitza el Web Inspector de Safari. Habilita Web Inspector al teu dispositiu sota Configuració > Safari > Avançat.

Conclusió

Desplegar la teva aplicació Apache Cordova a un dispositiu físic és una habilitat essencial per a qualsevol desenvolupador d'aplicacions mòbils. Seguint els passos descrits en aquesta secció, pots assegurar-te que la teva aplicació està provada i funciona bé en maquinari real. Tant si estàs desplegant a Android com a iOS, entendre el procés de desplegament i resoldre problemes comuns t'ajudarà a lliurar una aplicació robusta i fiable.
