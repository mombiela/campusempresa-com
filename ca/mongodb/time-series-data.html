Introducció a les Dades de Sèries Temporals

Les dades de sèries temporals són una seqüència de punts de dades recollits o registrats a intervals de temps específics. Aquest tipus de dades és crucial en diversos camps com les finances, la salut i l'IoT (Internet de les Coses) per fer un seguiment dels canvis al llarg del temps.

  • Característiques Clau de les Dades de Sèries Temporals:
    • Ordre Temporal: Els punts de dades estan ordenats per temps.
    • Intervals de Temps: Les dades es recullen a intervals regulars o irregulars.
    • Tendències i Patrons: L'anàlisi sovint se centra en identificar tendències, patrons estacionals i anomalies.

Emmagatzematge de Dades de Sèries Temporals a MongoDB

MongoDB és molt adequat per emmagatzemar dades de sèries temporals gràcies al seu disseny d'esquema flexible i les seves potents capacitats de consulta.

Disseny Bàsic de l'Esquema

Un esquema senzill per a dades de sèries temporals podria incloure camps per al segell de temps, el valor que s'està mesurant i qualsevol metadada rellevant.

{
  "timestamp": "2023-10-01T00:00:00Z",
  "value": 23.5,
  "metadata": {
    "sensorId": "sensor_1",
    "location": "Room 101"
  }
}

Creació d'una Col·lecció per a Dades de Sèries Temporals

Per crear una col·lecció per emmagatzemar dades de sèries temporals, pots utilitzar la següent comanda:

db.createCollection("time_series_data", {
  timeseries: {
    timeField: "timestamp",
    metaField: "metadata",
    granularity: "seconds"
  }
})
  • Paràmetres:
    • timeField: El camp que conté el segell de temps.
    • metaField: El camp que conté metadades.
    • granularity: La granularitat de les dades de sèries temporals (per exemple, "seconds", "minutes", "hours").

Inserció de Dades de Sèries Temporals

Inserir dades de sèries temporals a la col·lecció és senzill:

db.time_series_data.insertOne({
  "timestamp": ISODate("2023-10-01T00:00:00Z"),
  "value": 23.5,
  "metadata": {
    "sensorId": "sensor_1",
    "location": "Room 101"
  }
})

Consulta de Dades de Sèries Temporals

MongoDB proporciona potents capacitats de consulta per recuperar i analitzar dades de sèries temporals.

Consultes Bàsiques

Per recuperar tots els punts de dades per a un sensor específic:

db.time_series_data.find({"metadata.sensorId": "sensor_1"})

Per recuperar punts de dades dins d'un interval de temps específic:

db.time_series_data.find({
  "timestamp": {
    $gte: ISODate("2023-10-01T00:00:00Z"),
    $lte: ISODate("2023-10-02T00:00:00Z")
  }
})

Marc d'Agregació

El marc d'agregació és útil per a consultes més complexes, com calcular el valor mitjà durant un període.

db.time_series_data.aggregate([
  {
    $match: {
      "metadata.sensorId": "sensor_1",
      "timestamp": {
        $gte: ISODate("2023-10-01T00:00:00Z"),
        $lte: ISODate("2023-10-02T00:00:00Z")
      }
    }
  },
  {
    $group: {
      _id: null,
      averageValue: { $avg: "$value" }
    }
  }
])

Indexació de Dades de Sèries Temporals

La indexació és crucial per optimitzar el rendiment de les consultes, especialment per a dades de sèries temporals.

Creació d'Índexs

Crea un índex al camp timestamp per accelerar les consultes basades en el temps:

db.time_series_data.createIndex({ "timestamp": 1 })

Crea un índex compost als camps metadata.sensorId i timestamp per a consultes que filtren per sensor i temps:

db.time_series_data.createIndex({ "metadata.sensorId": 1, "timestamp": 1 })

Temes Avançats

Fragmentació de Dades de Sèries Temporals

La fragmentació pot ajudar a distribuir les dades de sèries temporals a través de múltiples servidors per gestionar grans volums de dades.

  • Clau de Fragmentació: Tria una clau de fragmentació que distribueixi les dades de manera uniforme, com una combinació de sensorId i timestamp.

Processament de Dades en Temps Real

MongoDB es pot integrar amb marcs de processament de dades en temps real com Apache Kafka i Apache Spark per a analítiques en temps real.

Conclusió

Les dades de sèries temporals són una eina poderosa per fer un seguiment i analitzar els canvis al llarg del temps. El disseny d'esquema flexible de MongoDB, les seves potents capacitats de consulta i el suport per a la indexació i la fragmentació el converteixen en una excel·lent opció per emmagatzemar i processar dades de sèries temporals. Comprenent els conceptes bàsics del disseny d'esquema, la consulta i la indexació, pots gestionar i analitzar efectivament les dades de sèries temporals a MongoDB.
