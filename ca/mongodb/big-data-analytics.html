Introducció al Big Data i MongoDB

  • Big Data: Es refereix a grans volums de dades que poden ser estructurades, semi-estructurades o no estructurades. Es caracteritza per les 3 V: Volum, Velocitat i Varietat.
  • MongoDB: Una base de dades NoSQL dissenyada per gestionar grans volums de dades. Emmagatzema dades en documents flexibles, semblants a JSON.

Configuració de MongoDB

  • Instal·lació: Instruccions per instal·lar MongoDB en diversos sistemes operatius.
  • Comandes bàsiques: Iniciar el servidor MongoDB, connectar-se a la shell de MongoDB i operacions bàsiques CRUD.

Exemple: Operacions bàsiques CRUD

// Inserir un document
db.users.insertOne({ name: "Alice", age: 25, city: "New York" });

// Llegir un document
db.users.find({ name: "Alice" });

// Actualitzar un document
db.users.updateOne({ name: "Alice" }, { $set: { age: 26 } });

// Esborrar un document
db.users.deleteOne({ name: "Alice" });

Modelatge de Dades a MongoDB

  • Disseny d'Esquemes: Entendre com dissenyar esquemes per a MongoDB.
  • Normalització vs. Desnormalització: Quan utilitzar documents incrustats vs. referències.

Exemple: Documents Incrustats vs. Referències

// Document Incrustat
db.orders.insertOne({
  order_id: 1,
  product: "Laptop",
  customer: { name: "John Doe", address: "123 Main St" }
});

// Referència
db.customers.insertOne({ customer_id: 1, name: "John Doe", address: "123 Main St" });
db.orders.insertOne({ order_id: 1, product: "Laptop", customer_id: 1 });

Indexació i Optimització de Rendiment

  • Índexs: Com crear i utilitzar índexs per millorar el rendiment de les consultes.
  • Optimització de Consultes: Tècniques per optimitzar les consultes de MongoDB.

Exemple: Creació d'Índexs

// Crear un índex en el camp 'name'
db.users.createIndex({ name: 1 });

// Crear un índex compost en 'name' i 'age'
db.users.createIndex({ name: 1, age: -1 });

Marc d'Agrupació

  • Pipeline d'Agrupació: Utilitzar etapes com $match, $group, $sort i $project per processar dades.
  • Map-Reduce: Una alternativa al pipeline d'agrupació per al processament complex de dades.

Exemple: Pipeline d'Agrupació

// Agrupació per trobar l'edat mitjana dels usuaris a cada ciutat
db.users.aggregate([
  { $group: { _id: "$city", averageAge: { $avg: "$age" } } }
]);

Gestió de Big Data amb MongoDB

  • Sharding: Distribuir dades a través de múltiples servidors per gestionar grans conjunts de dades.
  • Replicació: Assegurar la disponibilitat i redundància de les dades.

Exemple: Sharding

// Activar el sharding a la base de dades
sh.enableSharding("myDatabase");

// Fragmentar una col·lecció
sh.shardCollection("myDatabase.users", { _id: "hashed" });

Analítica en Temps Real

  • Fluxos de Canvis: Processament de dades en temps real utilitzant fluxos de canvis.
  • Integració amb Eines de Big Data: Utilitzar MongoDB amb eines com Apache Spark per a l'analítica en temps real.

Exemple: Fluxos de Canvis

// Vigilar els canvis en la col·lecció 'users'
const changeStream = db.users.watch();
changeStream.on("change", (change) => {
  printjson(change);
});

Temes Avançats

  • Consultes Geoespacials: Treballar amb dades basades en la ubicació.
  • Cerca de Text: Implementar la cerca de text complet a MongoDB.
  • Dades de Sèries Temporals: Gestionar i analitzar dades de sèries temporals.

Exemple: Consulta Geoespacial

// Crear un índex geoespacial
db.places.createIndex({ location: "2dsphere" });

// Trobar llocs dins d'1 km d'un punt
db.places.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [ -73.9667, 40.78 ] },
      $maxDistance: 1000
    }
  }
});

Conclusió

En aquest curs, hem cobert els aspectes essencials de l'ús de MongoDB per a l'Analítica de Big Data. Hem començat amb les bases de la configuració de MongoDB i la realització d'operacions CRUD, després hem passat a temes més avançats com el modelatge de dades, la indexació i el marc d'agrupació. També hem explorat com gestionar big data amb sharding i replicació, i com realitzar analítica en temps real. Finalment, hem tocat temes avançats com les consultes geoespacials i la cerca de text.

Al final d'aquest curs, hauríeu de tenir una comprensió sòlida de com aprofitar MongoDB per a l'Analítica de Big Data, permetent-vos gestionar grans conjunts de dades de manera eficient i realitzar anàlisis de dades complexes.
