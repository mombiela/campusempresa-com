Connexió a MongoDB amb un Llenguatge de Programació
Consultant Documents ►

Connectar-se a MongoDB és una habilitat fonamental per a qualsevol desenvolupador que treballi amb aquesta base de dades NoSQL. Aquesta secció et guiarà a través del procés de connexió a MongoDB utilitzant diferents llenguatges de programació, centrant-se en Python i JavaScript (Node.js). Cobrirem els passos necessaris, proporcionarem exemples de codi i explicarem cada part del procés.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • MongoDB instal·lat i en funcionament a la teva màquina local o a un servidor remot.
  • Coneixements bàsics del llenguatge de programació que estàs utilitzant.
  • El controlador de MongoDB adequat instal·lat per al teu llenguatge de programació.

Connexió a MongoDB amb Python

Python és un llenguatge popular per a la manipulació i anàlisi de dades. Utilitzarem la biblioteca pymongo per connectar-nos a MongoDB.

Instal·lació de pymongo

Primer, necessites instal·lar la biblioteca pymongo. Pots fer-ho utilitzant pip:

pip install pymongo

Connexió a MongoDB

Aquí tens un exemple senzill de com connectar-te a MongoDB utilitzant Python:

from pymongo import MongoClient

# Crear un MongoClient per a la instància de MongoDB en funcionament
client = MongoClient('localhost', 27017)

# Accedir a la base de dades 'test'
db = client.test

# Imprimir la llista de col·leccions a la base de dades 'test'
print(db.list_collection_names())

Explicació:

  • MongoClient('localhost', 27017): Això crea una connexió al servidor de MongoDB que s'executa a localhost al port 27017.
  • client.test: Accedeix a la base de dades test.
  • db.list_collection_names(): Llista totes les col·leccions a la base de dades test.

Realització d'Operacions Bàsiques

Inserim un document en una col·lecció i el recuperem:

# Accedir a la base de dades 'test'
db = client.test

# Accedir a la col·lecció 'example'
collection = db.example

# Inserir un document a la col·lecció 'example'
document = {"name": "John Doe", "age": 30}
collection.insert_one(document)

# Recuperar el document
retrieved_document = collection.find_one({"name": "John Doe"})
print(retrieved_document)

Explicació:

  • db.example: Accedeix a la col·lecció example.
  • collection.insert_one(document): Insereix un document a la col·lecció example.
  • collection.find_one({"name": "John Doe"}): Troba un document on el camp name és "John Doe".

Connexió a MongoDB amb Node.js

Node.js és àmpliament utilitzat per construir aplicacions de xarxa escalables. Utilitzarem el paquet mongodb per connectar-nos a MongoDB.

Instal·lació de mongodb

Primer, necessites instal·lar el paquet mongodb. Pots fer-ho utilitzant npm:

npm install mongodb

Connexió a MongoDB

Aquí tens un exemple senzill de com connectar-te a MongoDB utilitzant Node.js:

const { MongoClient } = require('mongodb');

// URL de Connexió
const url = 'mongodb://localhost:27017';
const client = new MongoClient(url);

// Nom de la Base de Dades
const dbName = 'test';

async function main() {
  // Utilitza el mètode connect per connectar-te al servidor
  await client.connect();
  console.log('Connected successfully to server');
  
  const db = client.db(dbName);

  // Imprimir la llista de col·leccions a la base de dades 'test'
  const collections = await db.listCollections().toArray();
  console.log(collections);

  // Tancar la connexió
  await client.close();
}

main().catch(console.error);

Explicació:

  • const client = new MongoClient(url): Crea una nova instància de MongoClient.
  • await client.connect(): Es connecta al servidor de MongoDB.
  • const db = client.db(dbName): Accedeix a la base de dades test.
  • await db.listCollections().toArray(): Llista totes les col·leccions a la base de dades test.

Realització d'Operacions Bàsiques

Inserim un document en una col·lecció i el recuperem:

async function main() {
  await client.connect();
  console.log('Connected successfully to server');
  
  const db = client.db(dbName);
  const collection = db.collection('example');

  // Inserir un document a la col·lecció 'example'
  const document = { name: 'John Doe', age: 30 };
  await collection.insertOne(document);

  // Recuperar el document
  const retrievedDocument = await collection.findOne({ name: 'John Doe' });
  console.log(retrievedDocument);

  await client.close();
}

main().catch(console.error);

Explicació:

  • const collection = db.collection('example'): Accedeix a la col·lecció example.
  • await collection.insertOne(document): Insereix un document a la col·lecció example.
  • await collection.findOne({ name: 'John Doe' }): Troba un document on el camp name és "John Doe".

Conclusió

Connectar-se a MongoDB utilitzant un llenguatge de programació és un procés senzill un cop entens els conceptes bàsics. En aquesta secció, hem cobert com connectar-se a MongoDB utilitzant Python i Node.js, incloent la instal·lació de les biblioteques necessàries, l'establiment d'una connexió i la realització d'operacions bàsiques com inserir i recuperar documents. En dominar aquestes habilitats, podràs interactuar eficientment amb MongoDB en les teves aplicacions.
