Operacions d'escriptura massiva
Índexs a MongoDB ►

Les operacions d'escriptura massiva a MongoDB permeten realitzar múltiples operacions d'escriptura (insercions, actualitzacions, eliminacions) en una sola sol·licitud. Això pot millorar significativament el rendiment i l'eficiència de les operacions de la teva base de dades, especialment quan es treballa amb grans conjunts de dades.

Conceptes clau

  • Escriptura massiva: Un mètode per executar múltiples operacions d'escriptura en una sola comanda.
  • Escriptura massiva ordenada: Les operacions s'executen en l'ordre en què s'afegeixen. Si es produeix un error, l'execució s'atura.
  • Escriptura massiva desordenada: Les operacions s'executen en paral·lel, i els errors no aturen l'execució de les operacions restants.
  • Nivell d'escriptura: Especifica el nivell de reconeixement sol·licitat a MongoDB per a les operacions d'escriptura.

Per què utilitzar operacions d'escriptura massiva?

  • Rendiment: Redueix el nombre de viatges de xarxa entre el client i el servidor.
  • Atomicitat: Assegura que totes les operacions tinguin èxit o cap, mantenint la integritat de les dades.
  • Gestió d'errors: Proporciona informació detallada dels errors per a cada operació.

Sintaxi bàsica

El mètode bulkWrite s'utilitza per realitzar operacions d'escriptura massiva. Aquí tens la sintaxi bàsica:

db.collection.bulkWrite([
  { insertOne: { "document": { /* document a inserir */ } } },
  { updateOne: { "filter": { /* criteris de consulta */ }, "update": { /* operacions d'actualització */ } } },
  { deleteOne: { "filter": { /* criteris de consulta */ } } },
  // Més operacions...
], { ordered: true/false, writeConcern: { /* opcions de nivell d'escriptura */ } })

Exemples

Inserir múltiples documents

db.students.bulkWrite([
  { insertOne: { "document": { name: "Alice", age: 21, major: "Computer Science" } } },
  { insertOne: { "document": { name: "Bob", age: 22, major: "Mathematics" } } },
  { insertOne: { "document": { name: "Charlie", age: 23, major: "Physics" } } }
])

Actualitzar múltiples documents

db.students.bulkWrite([
  { updateOne: { "filter": { name: "Alice" }, "update": { $set: { age: 22 } } } },
  { updateOne: { "filter": { name: "Bob" }, "update": { $set: { major: "Statistics" } } } }
])

Eliminar múltiples documents

db.students.bulkWrite([
  { deleteOne: { "filter": { name: "Charlie" } } },
  { deleteOne: { "filter": { age: { $lt: 22 } } } }
])

Operacions mixtes

db.students.bulkWrite([
  { insertOne: { "document": { name: "David", age: 24, major: "Biology" } } },
  { updateOne: { "filter": { name: "Alice" }, "update": { $set: { major: "Data Science" } } } },
  { deleteOne: { "filter": { name: "Bob" } } }
], { ordered: false })

Escriptures massives ordenades vs desordenades

| Característica | Escriptura massiva ordenada | Escriptura massiva desordenada | |------------------------|-----------------------------|--------------------------------| | Ordre d'execució | Seqüencial | Paral·lel | | S'atura en cas d'error | Sí | No | | Rendiment | Més lent | Més ràpid | | Cas d'ús | Quan l'ordre importa | Quan l'ordre no importa |

Nivell d'escriptura

El nivell d'escriptura descriu el nivell de reconeixement sol·licitat a MongoDB per a les operacions d'escriptura. Pots especificar el nivell d'escriptura en les operacions d'escriptura massiva per assegurar la durabilitat i consistència de les dades.

Exemple amb nivell d'escriptura

db.students.bulkWrite([
  { insertOne: { "document": { name: "Eve", age: 25, major: "Chemistry" } } },
  { updateOne: { "filter": { name: "David" }, "update": { $set: { age: 25 } } } }
], { writeConcern: { w: "majority", j: true, wtimeout: 5000 } })

Conclusió

Les operacions d'escriptura massiva a MongoDB són una eina poderosa per realitzar múltiples operacions d'escriptura de manera eficient. Entenent les diferències entre operacions ordenades i desordenades i com utilitzar els nivells d'escriptura, pots optimitzar les teves interaccions amb la base de dades per obtenir un millor rendiment i fiabilitat. Practica utilitzant operacions d'escriptura massiva amb diferents escenaris per convertir-te en un expert en la gestió de manipulacions de dades a gran escala a MongoDB.
