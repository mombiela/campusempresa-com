Introducció a Mongoose

Mongoose és una biblioteca de modelatge de dades d'objectes (ODM) per a MongoDB i Node.js. Proporciona una solució senzilla basada en esquemes per modelar les dades de la teva aplicació. Inclou conversió de tipus integrada, validació, construcció de consultes, hooks de lògica de negoci, i més.

Conceptes Clau

  • ODM (Modelatge de Dades d'Objectes): Mongoose et permet definir esquemes per a les teves dades, que després es poden utilitzar per crear models.
  • Esquema: Un esquema defineix l'estructura dels documents dins d'una col·lecció.
  • Model: Un model és una versió compilada de l'esquema, que proporciona una interfície per interactuar amb la base de dades.

Configuració de Mongoose

Per començar amb Mongoose, necessites instal·lar-lo via npm i connectar-lo a la teva base de dades MongoDB.

Instal·lació

npm install mongoose

Connexió a MongoDB

const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/mydatabase', {
  useNewUrlParser: true,
  useUnifiedTopology: true
})
.then(() => console.log('MongoDB connected'))
.catch(err => console.log(err));

Definició d'Esquemes i Models

Els esquemes i models són el nucli de Mongoose. Defineixen l'estructura i el comportament de les dades a la teva aplicació.

Creació d'un Esquema

const mongoose = require('mongoose');
const Schema = mongoose.Schema;

const userSchema = new Schema({
  name: String,
  age: Number,
  email: String,
  createdAt: {
    type: Date,
    default: Date.now
  }
});

Creació d'un Model

const User = mongoose.model('User', userSchema);

Operacions CRUD

Mongoose facilita la realització d'operacions CRUD (Crear, Llegir, Actualitzar, Esborrar).

Crear

const newUser = new User({
  name: 'John Doe',
  age: 30,
  email: 'john.doe@example.com'
});

newUser.save()
  .then(user => console.log(user))
  .catch(err => console.log(err));

Llegir

User.find()
  .then(users => console.log(users))
  .catch(err => console.log(err));

Actualitzar

User.findOneAndUpdate({ name: 'John Doe' }, { age: 31 }, { new: true })
  .then(user => console.log(user))
  .catch(err => console.log(err));

Esborrar

User.findOneAndDelete({ name: 'John Doe' })
  .then(user => console.log(user))
  .catch(err => console.log(err));

Funcionalitats Avançades

Mongoose ofereix diverses funcionalitats avançades per millorar la teva aplicació.

Validació

Mongoose proporciona validadors integrats i la capacitat de crear lògica de validació personalitzada.

const userSchema = new Schema({
  name: {
    type: String,
    required: true
  },
  age: {
    type: Number,
    min: 0,
    max: 120
  },
  email: {
    type: String,
    match: /.+\@.+\..+/
  }
});

Middleware (Hooks)

El middleware de Mongoose (també conegut com hooks) són funcions que s'executen en etapes específiques del cicle de vida d'un model.

userSchema.pre('save', function(next) {
  this.createdAt = Date.now();
  next();
});

Població

La població és el procés de reemplaçar automàticament els camins especificats en el document amb document(s) d'altres col·leccions.

const postSchema = new Schema({
  title: String,
  author: { type: Schema.Types.ObjectId, ref: 'User' }
});

const Post = mongoose.model('Post', postSchema);

Post.find()
  .populate('author')
  .then(posts => console.log(posts))
  .catch(err => console.log(err));

Conclusió

Mongoose és una eina potent per treballar amb MongoDB en Node.js. Simplifica el procés de definir esquemes, realitzar operacions CRUD, i implementar funcionalitats avançades com la validació, middleware, i població. Dominant Mongoose, pots gestionar eficientment les dades de la teva aplicació i assegurar una manipulació de dades robusta i íntegra.
