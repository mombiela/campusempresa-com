Les operacions d'agrupació a MongoDB processen registres de dades i retornen resultats calculats. Sovint s'utilitzen per agrupar valors de múltiples documents, realitzar operacions sobre les dades agrupades i retornar un únic resultat. Aquesta secció cobrirà les etapes d'agrupació més comunes que trobaràs, des de nivells principiants fins a avançats.

Introducció al Marc d'Agrupació

El marc d'agrupació a MongoDB és una eina poderosa per a l'anàlisi i transformació de dades. Consisteix en una pipeline d'etapes, cadascuna realitzant una operació específica sobre els documents d'entrada.

Etapes Comunes d'Agrupació

$match

  • Propòsit: Filtra els documents per passar només aquells que compleixin la condició especificada a la següent etapa de la pipeline.
  • Ús: Similar a la consulta find, però utilitzat dins de la pipeline d'agrupació.
db.collection.aggregate([
  { $match: { status: "A" } }
])

$group

  • Propòsit: Agrupa els documents d'entrada per una expressió d'identificador especificada i aplica les expressions d'acumulador a cada grup.
  • Ús: Sovint s'utilitza per calcular agregats com sumes, mitjanes, recomptes, etc.
db.collection.aggregate([
  { $group: { _id: "$status", total: { $sum: "$amount" } } }
])

$project

  • Propòsit: Reestructura cada document en el flux, com ara afegint nous camps o eliminant camps existents.
  • Ús: Útil per incloure, excloure o calcular nous camps.
db.collection.aggregate([
  { $project: { item: 1, total: { $multiply: ["$price", "$quantity"] } } }
])

$sort

  • Propòsit: Ordena tots els documents d'entrada i els retorna en ordre ordenat.
  • Ús: Similar al mètode sort però utilitzat dins de la pipeline d'agrupació.
db.collection.aggregate([
  { $sort: { total: -1 } }
])

$limit

  • Propòsit: Limita el nombre de documents passats a la següent etapa de la pipeline.
  • Ús: Útil per a la paginació o per restringir la mida de la sortida.
db.collection.aggregate([
  { $limit: 5 }
])

$skip

  • Propòsit: Salta els primers N documents i passa els documents restants a la següent etapa.
  • Ús: Sovint s'utilitza conjuntament amb $limit per a la paginació.
db.collection.aggregate([
  { $skip: 10 }
])

$unwind

  • Propòsit: Descompon un camp de matriu dels documents d'entrada per produir un document per a cada element.
  • Ús: Útil per normalitzar dades emmagatzemades en matrius.
db.collection.aggregate([
  { $unwind: "$items" }
])

$lookup

  • Propòsit: Realitza una unió externa esquerra a una altra col·lecció en la mateixa base de dades per filtrar documents de la col·lecció "unida" per al processament.
  • Ús: Útil per combinar dades de múltiples col·leccions.
db.collection.aggregate([
  {
    $lookup: {
      from: "otherCollection",
      localField: "foreignKey",
      foreignField: "_id",
      as: "joinedData"
    }
  }
])

$addFields

  • Propòsit: Afegeix nous camps als documents.
  • Ús: Útil per afegir camps calculats.
db.collection.aggregate([
  { $addFields: { totalPrice: { $multiply: ["$price", "$quantity"] } } }
])

$out

  • Propòsit: Escriu els documents resultants de la pipeline d'agrupació a una col·lecció especificada.
  • Ús: Útil per emmagatzemar els resultats d'una agrupació per a un ús futur.
db.collection.aggregate([
  { $match: { status: "A" } },
  { $out: "newCollection" }
])

Resum

Entendre i utilitzar les etapes comunes d'agrupació a MongoDB és crucial per a una anàlisi i transformació de dades efectiva. Cada etapa serveix a un propòsit específic, des de filtrar i agrupar fins a reestructurar i unir dades. Dominant aquestes etapes, pots realitzar consultes complexes i obtenir una comprensió més profunda de les teves dades.

  • $match: Filtra documents.
  • $group: Agrupa documents i realitza agregacions.
  • $project: Reestructura documents.
  • $sort: Ordena documents.
  • $limit: Limita el nombre de documents.
  • $skip: Salta documents.
  • $unwind: Descompon matrius.
  • $lookup: Uneix col·leccions.
  • $addFields: Afegeix nous camps.
  • $out: Escriu resultats a una col·lecció.

Aquestes etapes formen els blocs de construcció del marc d'agrupació de MongoDB, permetent-te crear pipelines de processament de dades poderoses i eficients.
