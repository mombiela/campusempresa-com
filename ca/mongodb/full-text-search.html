La cerca de text complet és una característica potent de MongoDB que et permet cercar text dins dels teus documents. Això és particularment útil per a aplicacions que necessiten proporcionar funcionalitat de cerca similar a la que podries trobar en un motor de cerca.

Introducció a la Cerca de Text Complet

  • Definició: La cerca de text complet et permet cercar text dins dels teus documents utilitzant índexs que MongoDB crea.
  • Casos d'ús: Comunament utilitzat en aplicacions com blogs, llocs de comerç electrònic i qualsevol aplicació que requereixi funcionalitat de cerca.

Creació d'un Índex de Text

Per realitzar una cerca de text complet, primer has de crear un índex de text en els camps que vols cercar.

Exemple

db.articles.createIndex({ content: "text" });
  • Explicació: Aquesta comanda crea un índex de text en el camp content de la col·lecció articles.

Realització d'una Cerca de Text Complet

Un cop tens un índex de text, pots realitzar cerques utilitzant l'operador $text.

Exemple

db.articles.find({ $text: { $search: "MongoDB" } });
  • Explicació: Aquesta comanda cerca el terme "MongoDB" en el camp content de la col·lecció articles.

Opcions de Cerca de Text

MongoDB proporciona diverses opcions per personalitzar la teva cerca de text.

Sensibilitat a Majúscules i Minúscules

Per defecte, la cerca de text no distingeix entre majúscules i minúscules. Pots canviar aquest comportament utilitzant l'opció caseSensitive.

Exemple

db.articles.find({ $text: { $search: "MongoDB", $caseSensitive: true } });
  • Explicació: Aquesta comanda realitza una cerca sensible a majúscules i minúscules per al terme "MongoDB".

Opcions de Llengua

Pots especificar la llengua per a la cerca de text per millorar els resultats de la cerca.

Exemple

db.articles.find({ $text: { $search: "MongoDB", $language: "english" } });
  • Explicació: Aquesta comanda realitza una cerca de text utilitzant les regles de la llengua anglesa.

Ordenació per Puntuació de Cerca de Text

Pots ordenar els resultats de la cerca per la seva puntuació de rellevància.

Exemple

db.articles.find({ $text: { $search: "MongoDB" } }, { score: { $meta: "textScore" } }).sort({ score: { $meta: "textScore" } });
  • Explicació: Aquesta comanda cerca el terme "MongoDB" i ordena els resultats per la seva puntuació de rellevància.

Combinació de Cerca de Text amb Altres Consultes

Pots combinar la cerca de text amb altres consultes de MongoDB per filtrar més els teus resultats.

Exemple

db.articles.find({ $text: { $search: "MongoDB" }, author: "John Doe" });
  • Explicació: Aquesta comanda cerca el terme "MongoDB" en articles escrits per "John Doe".

Limitacions i Consideracions

  • Mida de l'Índex: Els índexs de text poden ser grans i afectar el rendiment.
  • Consultes Complexes: Combinar la cerca de text amb altres consultes complexes pot ser desafiant i pot requerir optimització.

Conclusió

La cerca de text complet en MongoDB és una característica potent que et permet cercar text dins dels teus documents de manera eficient. Creant índexs de text i utilitzant l'operador $text, pots implementar funcionalitats de cerca robustes en les teves aplicacions. Recorda considerar les limitacions i optimitzar les teves consultes per obtenir el millor rendiment.
