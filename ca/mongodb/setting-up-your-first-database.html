Introducció

En aquesta secció, et guiarem a través del procés de configuració de la teva primera base de dades MongoDB. MongoDB és una base de dades NoSQL popular coneguda per la seva flexibilitat i escalabilitat. Al final d'aquesta secció, tindràs una base de dades MongoDB funcional i entendràs els conceptes bàsics per interactuar amb ella.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb accés a internet
  • Coneixements bàsics de la interfície de línia de comandes (CLI)
  • MongoDB instal·lat a la teva màquina (cobrirem la instal·lació breument)

Instal·lació de MongoDB

A Windows

  1. Descarrega MongoDB: Visita el Centre de Descàrregues de MongoDB i descarrega l'instal·lador per a Windows.
  2. Executa l'Instal·lador: Segueix les indicacions de la instal·lació. Assegura't de seleccionar l'opció per instal·lar MongoDB com a servei.
  3. Verifica la Instal·lació: Obre el Símbol del Sistema i escriu: 
    mongod --version
    Hauries de veure la informació de la versió si MongoDB està instal·lat correctament.

A macOS

  1. Instal·la Homebrew: Si no tens Homebrew instal·lat, obre Terminal i executa: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instal·la MongoDB: Utilitza Homebrew per instal·lar MongoDB: 
    brew tap mongodb/brew
brew install mongodb-community
  3. Inicia MongoDB: Inicia el servei de MongoDB: 
    brew services start mongodb/brew/mongodb-community
  4. Verifica la Instal·lació: Comprova la instal·lació executant: 
    mongod --version

A Linux

  1. Importa la Clau Pública: Obre Terminal i executa: 
    wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -
  2. Crea un Fitxer de Llista: Crea el fitxer /etc/apt/sources.list.d/mongodb-org-4.4.list per a Ubuntu: 
    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list
  3. Instal·la MongoDB: Actualitza la llista de paquets i instal·la MongoDB: 
    sudo apt-get update
sudo apt-get install -y mongodb-org
  4. Inicia MongoDB: Inicia el servei de MongoDB: 
    sudo systemctl start mongod
  5. Verifica la Instal·lació: Comprova la instal·lació executant: 
    mongod --version

Creació de la teva primera base de dades

Iniciant el Shell de MongoDB

Per interactuar amb MongoDB, utilitzaràs el shell de MongoDB. Obre el teu terminal o símbol del sistema i escriu:

mongo

Aquesta comanda inicia el shell de MongoDB, on pots executar comandes per interactuar amb la teva base de dades.

Creant una Base de Dades

A MongoDB, les bases de dades es creen implícitament quan primer emmagatzemes dades en elles. Per crear una base de dades, canvia a un nou context de base de dades:

use myFirstDatabase

Aquesta comanda canvia a myFirstDatabase. Si no existeix, MongoDB la crearà quan primer emmagatzemis dades.

Inserint Dades

Per inserir dades a la teva nova base de dades, utilitza el mètode insertOne:

db.myCollection.insertOne({ name: "John Doe", age: 30, profession: "Software Engineer" })

Aquesta comanda crea una col·lecció anomenada myCollection i hi insereix un document.

Consultar Dades

Per recuperar dades de la teva col·lecció, utilitza el mètode find:

db.myCollection.find()

Aquesta comanda retorna tots els documents a la col·lecció myCollection.

Actualitzant Dades

Per actualitzar dades a la teva col·lecció, utilitza el mètode updateOne:

db.myCollection.updateOne({ name: "John Doe" }, { $set: { age: 31 } })

Aquesta comanda actualitza el camp age del document on name és "John Doe".

Eliminant Dades

Per eliminar dades de la teva col·lecció, utilitza el mètode deleteOne:

db.myCollection.deleteOne({ name: "John Doe" })

Aquesta comanda elimina el document on name és "John Doe".

Resum

En aquesta secció, has après com configurar MongoDB en diferents sistemes operatius, crear la teva primera base de dades i realitzar operacions bàsiques de CRUD (Crear, Llegir, Actualitzar, Eliminar). Aquestes habilitats fonamentals són essencials per treballar amb MongoDB i serviran com a base per a temes més avançats en aquest curs.
