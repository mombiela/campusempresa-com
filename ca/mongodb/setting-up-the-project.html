Configurar un projecte MongoDB és el primer pas per construir aplicacions que aprofitin les potents capacitats de bases de dades NoSQL de MongoDB. Aquesta secció et guiarà a través del procés d'instal·lació de MongoDB, configuració del teu entorn de desenvolupament i creació del teu primer projecte MongoDB.

Instal·lació de MongoDB

Abans de poder començar a utilitzar MongoDB, necessites instal·lar-lo a la teva màquina. Segueix aquests passos per instal·lar MongoDB en diferents sistemes operatius:

Windows

  1. Descarrega MongoDB:

    • Visita el Centre de Descàrregues de MongoDB i descarrega l'instal·lador per a Windows.

  2. Executa l'Instal·lador:

    • Fes doble clic al fitxer .msi descarregat i segueix les instruccions d'instal·lació.
    • Tria el tipus de configuració "Complete".

  3. Configura la Variable d'Entorn:

    • Afegeix el directori bin de MongoDB (C:\Program Files\MongoDB\Server\<version>\bin) a la variable d'entorn PATH del teu sistema.

  4. Verifica la Instal·lació:

    • Obre el Símbol del Sistema i escriu mongo --version per verificar la instal·lació.

macOS

  1. Instal·la Homebrew (si no està instal·lat):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·la MongoDB:

    brew tap mongodb/brew
brew install mongodb-community@<version>

  3. Inicia MongoDB:

    brew services start mongodb/brew/mongodb-community

  4. Verifica la Instal·lació:

    • Obre el Terminal i escriu mongo --version per verificar la instal·lació.

Linux

  1. Importa la Clau Pública:

    wget -qO - https://www.mongodb.org/static/pgp/server-<version>.asc | sudo apt-key add -

  2. Crea un Fitxer de Llista:

    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/<version> multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-<version>.list

  3. Instal·la MongoDB:

    sudo apt-get update
sudo apt-get install -y mongodb-org

  4. Inicia MongoDB:

    sudo systemctl start mongod

  5. Verifica la Instal·lació:

    • Obre el Terminal i escriu mongo --version per verificar la instal·lació.

Configuració de l'Entorn de Desenvolupament

Un cop MongoDB està instal·lat, necessites configurar el teu entorn de desenvolupament. Això implica triar un llenguatge de programació, instal·lar els controladors necessaris i configurar un Entorn de Desenvolupament Integrat (IDE).

Triar un Llenguatge de Programació

MongoDB suporta diversos llenguatges de programació. Algunes opcions populars inclouen:

  • JavaScript (Node.js)
  • Python
  • Java
  • C#

Per a aquest curs, utilitzarem Node.js per la seva popularitat i facilitat d'ús.

Instal·lació de Node.js i npm

  1. Descarrega Node.js:

    • Visita el lloc web de Node.js i descarrega la versió LTS.

  2. Instal·la Node.js:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  3. Verifica la Instal·lació:

    node --version
npm --version

Configuració d'un IDE

Tria un IDE que suporti el desenvolupament en JavaScript. Algunes opcions populars inclouen:

  • Visual Studio Code
  • WebStorm
  • Atom

Per a aquest curs, utilitzarem Visual Studio Code.

  1. Descarrega Visual Studio Code:

    • Visita el lloc web de Visual Studio Code i descarrega l'instal·lador.

  2. Instal·la Visual Studio Code:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

Creació del teu Primer Projecte MongoDB

Ara que el teu entorn de desenvolupament està configurat, creem el teu primer projecte MongoDB.

Inicialitza un Projecte Node.js

  1. Crea un Directori de Projecte:

    mkdir my-mongodb-project
cd my-mongodb-project

  2. Inicialitza el Projecte:

    npm init -y

Instal·la el Driver de MongoDB

  1. Instal·la el Driver de MongoDB: 
    npm install mongodb

Connecta't a MongoDB

Crea un fitxer anomenat app.js i afegeix el següent codi per connectar-te a MongoDB:

const { MongoClient } = require('mongodb');

// URL de Connexió
const url = 'mongodb://localhost:27017';
const client = new MongoClient(url);

// Nom de la Base de Dades
const dbName = 'myDatabase';

async function main() {
  // Utilitza el mètode connect per connectar-te al servidor
  await client.connect();
  console.log('Connectat correctament al servidor');
  const db = client.db(dbName);

  // Tanca la connexió
  await client.close();
}

main().catch(console.error);

Executa el Projecte

  1. Executa l'Aplicació:

    node app.js

  2. Verifica la Connexió:

    • Hauries de veure el missatge "Connectat correctament al servidor" a la consola.

Conclusió

En aquesta secció, hem cobert els passos essencials per configurar un projecte MongoDB. Has après com instal·lar MongoDB en diferents sistemes operatius, configurar el teu entorn de desenvolupament amb Node.js i Visual Studio Code, i crear una aplicació simple en Node.js que es connecta a una base de dades MongoDB. Amb el teu entorn preparat, ara estàs llest per aprofundir en MongoDB i començar a construir aplicacions potents.
