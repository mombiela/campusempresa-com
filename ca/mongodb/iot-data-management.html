Introducció a IoT i MongoDB

  • IoT (Internet de les Coses): Una xarxa de dispositius físics que recullen i intercanvien dades.
  • MongoDB: Una base de dades NoSQL que emmagatzema dades en documents flexibles semblants a JSON.

Per què MongoDB per a IoT?

  • Escalabilitat: Gestiona grans volums de dades de manera eficient.
  • Flexibilitat: El disseny sense esquema acomoda diversos formats de dades IoT.
  • Anàlisi en temps real: Suporta el processament i la consulta de dades en temps real.

Configuració de MongoDB per a Dades IoT

Instal·lació de MongoDB

  • Segueix la guia d'instal·lació oficial de MongoDB per al teu sistema operatiu.
  • Inicia el servidor MongoDB amb la comanda: 
    mongod

Connexió a MongoDB

  • Utilitza MongoDB Compass o la shell de MongoDB per connectar-te a la teva base de dades.
  • Exemple de cadena de connexió: 
    mongo "mongodb://localhost:27017"

Disseny de l'Esquema de la Base de Dades

Comprensió de les Dades IoT

  • Dades del Dispositiu: Informació recollida dels dispositius IoT (per exemple, sensors, actuadors).
  • Metadades: Informació addicional sobre els dispositius (per exemple, ubicació, tipus de dispositiu).

Exemple de Disseny d'Esquema

  • Col·lecció de Dispositius: Emmagatzema metadades sobre els dispositius IoT.
  • Col·lecció de Lectures: Emmagatzema les lectures de dades dels sensors.
// Col·lecció de Dispositius
{
  "_id": "device1",
  "type": "temperature_sensor",
  "location": "warehouse_1",
  "installed_at": "2023-01-01T00:00:00Z"
}

// Col·lecció de Lectures
{
  "_id": "reading1",
  "device_id": "device1",
  "timestamp": "2023-10-01T12:00:00Z",
  "value": 22.5
}

Inserció de Dades IoT

Utilitzant la Shell de MongoDB

  • Inserir un document de dispositiu: 
    db.devices.insertOne({
  _id: "device1",
  type: "temperature_sensor",
  location: "warehouse_1",
  installed_at: new Date("2023-01-01T00:00:00Z")
});
  • Inserir un document de lectura: 
    db.readings.insertOne({
  _id: "reading1",
  device_id: "device1",
  timestamp: new Date("2023-10-01T12:00:00Z"),
  value: 22.5
});

Utilitzant un Llenguatge de Programació (Exemple en Python)

  • Instal·lar el controlador de MongoDB per a Python: 
    pip install pymongo
  • Inserir dades utilitzant Python: 
    from pymongo import MongoClient
from datetime import datetime

client = MongoClient("mongodb://localhost:27017")
db = client.iot_database

device = {
    "_id": "device1",
    "type": "temperature_sensor",
    "location": "warehouse_1",
    "installed_at": datetime(2023, 1, 1)
}
db.devices.insert_one(device)

reading = {
    "_id": "reading1",
    "device_id": "device1",
    "timestamp": datetime(2023, 10, 1, 12, 0),
    "value": 22.5
}
db.readings.insert_one(reading)

Consulta de Dades IoT

Consultes Bàsiques

  • Trobar un dispositiu per ID: 
    db.devices.findOne({ _id: "device1" });
  • Trobar lectures per a un dispositiu específic: 
    db.readings.find({ device_id: "device1" });

Consultes Avançades

  • Trobar lectures dins d'un rang de dates:

    db.readings.find({
  device_id: "device1",
  timestamp: { $gte: new Date("2023-10-01T00:00:00Z"), $lte: new Date("2023-10-01T23:59:59Z") }
});

  • Agregar dades per trobar lectures mitjanes:

    db.readings.aggregate([
  { $match: { device_id: "device1" } },
  { $group: { _id: "$device_id", averageValue: { $avg: "$value" } } }
]);

Indexació i Optimització del Rendiment

Creació d'Índexs

  • Crear un índex en el camp timestamp per accelerar les consultes: 
    db.readings.createIndex({ timestamp: 1 });

Consells de Rendiment

  • Utilitza índexs adequats per optimitzar el rendiment de les consultes.
  • Monitoritza i analitza regularment el rendiment de les consultes utilitzant les eines de MongoDB.

Processament de Dades en Temps Real

Fluxos de Canvi

  • Utilitza fluxos de canvi per escoltar canvis en temps real a la base de dades: 
    const changeStream = db.readings.watch();
changeStream.on("change", (change) => {
  console.log("Canvi detectat:", change);
});

Integració amb Frameworks de Processament de Dades

  • Integra MongoDB amb frameworks com Apache Kafka o Apache Spark per a un processament avançat de dades en temps real.

Conclusió

  • MongoDB proporciona una solució robusta i flexible per gestionar dades IoT.
  • La seva escalabilitat, capacitats en temps real i facilitat d'ús el fan una elecció ideal per a aplicacions IoT.
  • Seguint les millors pràctiques en el disseny d'esquemes, indexació i processament de dades en temps real, pots gestionar i analitzar dades IoT de manera eficient amb MongoDB.
