Introducció

Actualitzar documents a MongoDB és una habilitat fonamental per gestionar i manipular dades. Aquesta secció et guiarà a través dels conceptes bàsics de l'actualització de documents, diversos operadors d'actualització i tècniques avançades per a actualitzacions més complexes.

Actualitzacions Bàsiques

MongoDB proporciona diversos mètodes per actualitzar documents en una col·lecció. Els mètodes més utilitzats són updateOne(), updateMany() i replaceOne().

`updateOne()`

El mètode updateOne() actualitza un únic document que coincideix amb un filtre especificat.

db.collection.updateOne(
  { _id: 1 }, // Filtre
  { $set: { name: "John Doe" } } // Operació d'actualització
);
  • Filtre: Especifica els criteris per coincidir amb els documents.
  • Operació d'actualització: Defineix les modificacions a aplicar.

`updateMany()`

El mètode updateMany() actualitza tots els documents que coincideixen amb un filtre especificat.

db.collection.updateMany(
  { status: "active" }, // Filtre
  { $set: { status: "inactive" } } // Operació d'actualització
);

`replaceOne()`

El mètode replaceOne() reemplaça un únic document que coincideix amb un filtre especificat amb un nou document.

db.collection.replaceOne(
  { _id: 1 }, // Filtre
  { name: "Jane Doe", age: 30 } // Nou document
);

Operadors d'Actualització

MongoDB proporciona una varietat d'operadors d'actualització per modificar documents. Aquí hi ha alguns dels operadors més utilitzats:

  • $set: Estableix el valor d'un camp.
  • $unset: Elimina un camp d'un document.
  • $inc: Incrementa el valor d'un camp per una quantitat especificada.
  • $push: Afegeix un element a una matriu.
  • $pull: Elimina un element d'una matriu.

Exemples

$set

db.collection.updateOne(
  { _id: 1 },
  { $set: { age: 25 } }
);

$unset

db.collection.updateOne(
  { _id: 1 },
  { $unset: { age: "" } }
);

$inc

db.collection.updateOne(
  { _id: 1 },
  { $inc: { age: 1 } }
);

$push

db.collection.updateOne(
  { _id: 1 },
  { $push: { hobbies: "reading" } }
);

$pull

db.collection.updateOne(
  { _id: 1 },
  { $pull: { hobbies: "reading" } }
);

Upserts

Una operació upsert actualitza un document si existeix, o insereix un nou document si no existeix. Això s'aconsegueix establint l'opció upsert a true.

db.collection.updateOne(
  { _id: 2 },
  { $set: { name: "Alice" } },
  { upsert: true }
);

Actualitzacions de Matrius

Actualitzar matrius a MongoDB pot ser més complex. Aquí hi ha algunes operacions comunes:

Actualització d'Elements Específics de la Matriu

Per actualitzar un element específic en una matriu, utilitza l'operador posicional $.

db.collection.updateOne(
  { _id: 1, "grades.grade": 85 },
  { $set: { "grades.$.grade": 90 } }
);

Afegir Elements a les Matrius

Utilitza l'operador $addToSet per afegir un element a una matriu només si no existeix ja.

db.collection.updateOne(
  { _id: 1 },
  { $addToSet: { hobbies: "swimming" } }
);

Conclusió

Actualitzar documents a MongoDB és una característica poderosa que et permet modificar les teves dades de manera eficient. Comprenent els mètodes bàsics d'actualització, diversos operadors d'actualització i tècniques avançades com els upserts i les actualitzacions de matrius, pots gestionar les teves col·leccions de MongoDB de manera efectiva. Practica aquests conceptes amb diferents escenaris per convertir-te en un expert en l'actualització de documents a MongoDB.
