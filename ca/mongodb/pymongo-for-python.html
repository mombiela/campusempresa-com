Introducció a PyMongo

PyMongo és una distribució de Python que conté eines per treballar amb MongoDB, una base de dades NoSQL. Aquesta secció cobrirà els conceptes bàsics de PyMongo, incloent la instal·lació i la configuració inicial.

Instal·lació

Per instal·lar PyMongo, pots utilitzar pip, l'instal·lador de paquets de Python:

pip install pymongo

Connexió a MongoDB

Per connectar-te a un servidor MongoDB, necessites crear una instància de MongoClient:

from pymongo import MongoClient

client = MongoClient('mongodb://localhost:27017/')
  • MongoClient és el punt d'entrada principal al servidor MongoDB.
  • La cadena de connexió 'mongodb://localhost:27017/' especifica que MongoDB s'està executant a la màquina local en el port per defecte 27017.

Operacions bàsiques

Aquesta secció cobrirà les operacions bàsiques CRUD (Crear, Llegir, Actualitzar, Esborrar) utilitzant PyMongo.

Creació d'una base de dades i una col·lecció

A MongoDB, les bases de dades contenen col·leccions, i les col·leccions contenen documents.

db = client['mydatabase']
collection = db['mycollection']
  • db és una referència a la base de dades anomenada mydatabase.
  • collection és una referència a la col·lecció anomenada mycollection.

Inserció de documents

Pots inserir documents en una col·lecció utilitzant els mètodes insert_one o insert_many.

document = {"name": "Alice", "age": 25, "city": "New York"}
collection.insert_one(document)
  • insert_one insereix un sol document.
  • El document és un diccionari de Python.

Consulta de documents

Per recuperar documents, utilitza el mètode find.

for doc in collection.find({"name": "Alice"}):
    print(doc)
  • find retorna un cursor als documents que coincideixen amb la consulta.
  • La consulta és un diccionari que especifica els criteris.

Actualització de documents

Per actualitzar documents, utilitza els mètodes update_one o update_many.

collection.update_one({"name": "Alice"}, {"$set": {"age": 26}})
  • El primer argument és la consulta per coincidir amb els documents.
  • El segon argument especifica l'operació d'actualització.

Esborrat de documents

Per esborrar documents, utilitza els mètodes delete_one o delete_many.

collection.delete_one({"name": "Alice"})
  • L'argument és la consulta per coincidir amb els documents a esborrar.

Operacions avançades

Aquesta secció cobrirà característiques més avançades de PyMongo, com ara indexació, agregació i transaccions.

Indexació

Els índexs milloren el rendiment de les consultes.

collection.create_index([("name", pymongo.ASCENDING)])
  • create_index crea un índex en el camp especificat.
  • pymongo.ASCENDING especifica l'ordre de classificació.

Agregació

Les operacions d'agregació processen registres de dades i retornen resultats calculats.

pipeline = [
    {"$match": {"city": "New York"}},
    {"$group": {"_id": "$city", "average_age": {"$avg": "$age"}}}
]
result = collection.aggregate(pipeline)
for doc in result:
    print(doc)
  • pipeline és una llista d'etapes d'agregació.
  • $match filtra documents.
  • $group agrupa documents per un camp especificat i realitza càlculs.

Transaccions

Les transaccions permeten que múltiples operacions s'executin de manera tot-o-res.

with client.start_session() as session:
    with session.start_transaction():
        collection.insert_one({"name": "Bob", "age": 30}, session=session)
        collection.update_one({"name": "Bob"}, {"$set": {"city": "San Francisco"}}, session=session)
  • start_session inicia una nova sessió.
  • start_transaction inicia una nova transacció dins de la sessió.

Conclusió

PyMongo és una eina poderosa per interactuar amb MongoDB des de Python. Suporta una àmplia gamma d'operacions, des de les bàsiques CRUD fins a característiques avançades com la indexació, l'agregació i les transaccions. Dominant PyMongo, pots gestionar i manipular eficientment les teves bases de dades MongoDB dins de les teves aplicacions Python.
