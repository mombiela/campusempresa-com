Introducció

MongoDB és una base de dades NoSQL popular coneguda per la seva flexibilitat i escalabilitat. Aquesta secció et guiarà a través del procés d'instal·lació de MongoDB en diferents sistemes operatius. Al final d'aquesta secció, tindràs MongoDB funcionant a la teva màquina.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb connexió a internet.
  • Coneixements bàsics d'ús de la línia de comandes o terminal.

Instal·lant MongoDB a Windows

Pas 1: Descarregar MongoDB

  1. Visita el Centre de Descàrregues de MongoDB.
  2. Selecciona la versió adequada per al teu sistema operatiu Windows.
  3. Fes clic al botó de descàrrega per obtenir l'instal·lador .msi.

Pas 2: Executar l'Instal·lador

  1. Localitza el fitxer .msi descarregat i fes doble clic per executar-lo.
  2. Segueix les indicacions d'instal·lació:
    • Tria el tipus de configuració "Complete".
    • Assegura't que "Install MongoDB as a Service" estigui marcat.
    • Opcionalment, pots triar instal·lar MongoDB Compass, una interfície gràfica per a MongoDB.

Pas 3: Verificar la Instal·lació

  1. Obre el Command Prompt.
  2. Escriu mongo --version i prem Enter.
  3. Hauries de veure la informació de la versió de MongoDB, confirmant la instal·lació.
C:\> mongo --version
Versió de la shell de MongoDB v4.4.6

Instal·lant MongoDB a macOS

Pas 1: Instal·lar Homebrew

Si no tens Homebrew instal·lat, obre el Terminal i executa la següent comanda:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Pas 2: Instal·lar MongoDB

  1. Obre el Terminal.
  2. Executa les següents comandes per afegir la fórmula de MongoDB i instal·lar-la:
brew tap mongodb/brew
brew install mongodb-community@4.4

Pas 3: Iniciar MongoDB

  1. Per iniciar MongoDB, executa:
brew services start mongodb/brew/mongodb-community
  1. Verifica la instal·lació executant:
mongo --version

Hauries de veure la informació de la versió de MongoDB.

Instal·lant MongoDB a Linux

Pas 1: Importar la Clau Pública

  1. Obre el Terminal.
  2. Executa la següent comanda per importar la clau pública GPG de MongoDB:
wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -

Pas 2: Crear un Fitxer de Llista per a MongoDB

  1. Crea el fitxer de llista utilitzant la següent comanda:
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list

Pas 3: Instal·lar els Paquets de MongoDB

  1. Actualitza la llista de paquets i instal·la MongoDB:
sudo apt-get update
sudo apt-get install -y mongodb-org

Pas 4: Iniciar MongoDB

  1. Inicia MongoDB utilitzant la següent comanda:
sudo systemctl start mongod
  1. Verifica la instal·lació executant:
mongo --version

Hauries de veure la informació de la versió de MongoDB.

Conclusió

En aquesta secció, has après com instal·lar MongoDB a Windows, macOS i Linux. Assegurar-te que MongoDB està correctament instal·lat és el primer pas per utilitzar les seves potents característiques per a la gestió de bases de dades. En les següents seccions, explorarem com configurar i utilitzar MongoDB per a diverses aplicacions.
