Introducció

En aquest tema, explorarem com construir una aplicació de comerç electrònic utilitzant MongoDB. Cobrirem tot, des de la configuració de la base de dades fins a la implementació de funcions avançades com la indexació i l'agregació. Al final d'aquest tema, tindràs una comprensió sòlida de com utilitzar MongoDB per gestionar i consultar dades per a una aplicació de comerç electrònic.

Configuració de MongoDB

  • Instal·lació: Guia sobre com instal·lar MongoDB en diverses plataformes.
  • Connexió a MongoDB: Utilitzant la shell de MongoDB i drivers per a diferents llenguatges de programació.
# Iniciant el servidor de MongoDB
mongod --dbpath /path/to/your/db
// Connexió a MongoDB utilitzant Node.js
const { MongoClient } = require('mongodb');
const uri = "mongodb://localhost:27017";
const client = new MongoClient(uri, { useNewUrlParser: true, useUnifiedTopology: true });

async function run() {
    try {
        await client.connect();
        console.log("Connectat a MongoDB");
    } finally {
        await client.close();
    }
}
run().catch(console.dir);

Operacions CRUD

  • Crear: Inserir documents a les col·leccions.
  • Llegir: Consultar documents de les col·leccions.
  • Actualitzar: Modificar documents existents.
  • Eliminar: Eliminar documents de les col·leccions.
// Inserir un document a la col·lecció d'Usuaris
const user = {
    _id: "user123",
    name: "John Doe",
    email: "john.doe@example.com",
    password: "hashed_password",
    address: {
        street: "123 Main St",
        city: "Anytown",
        state: "CA",
        zip: "12345"
    }
};

await client.db("ecommerce").collection("users").insertOne(user);
// Consultar documents de la col·lecció de Productes
const products = await client.db("ecommerce").collection("products").find({ category: "Electronics" }).toArray();
console.log(products);
// Actualitzar un document a la col·lecció de Comandes
await client.db("ecommerce").collection("orders").updateOne(
    { _id: "order123" },
    { $set: { status: "Shipped" } }
);
// Eliminar un document de la col·lecció d'Usuaris
await client.db("ecommerce").collection("users").deleteOne({ _id: "user123" });

Indexació

  • Propòsit dels Indexos: Millorar el rendiment de les consultes.
  • Crear Indexos: Com crear indexos a MongoDB.
  • Tipus d'Indexos: Camp únic, compost, text i geoespacial.
// Crear un index al camp email de la col·lecció d'Usuaris
await client.db("ecommerce").collection("users").createIndex({ email: 1 });

Marc d'Agregació

  • Pipeline d'Agregació: Utilitzar etapes per transformar i analitzar dades.
  • Etapes Comunes: $match, $group, $sort, $project.
// Agregar vendes totals per categoria de producte
const salesByCategory = await client.db("ecommerce").collection("orders").aggregate([
    { $unwind: "$products" },
    { $lookup: {
        from: "products",
        localField: "products.product_id",
        foreignField: "_id",
        as: "product_details"
    }},
    { $unwind: "$product_details" },
    { $group: {
        _id: "$product_details.category",
        total_sales: { $sum: "$products.quantity" }
    }},
    { $sort: { total_sales: -1 } }
]).toArray();
console.log(salesByCategory);

Temes Avançats

  • Transaccions: Assegurar la consistència de les dades amb transaccions multi-document.
  • Sharding: Distribuir dades a través de múltiples servidors per a l'escalabilitat.
  • Replicació: Assegurar alta disponibilitat amb conjunts de rèpliques.
// Utilitzar una transacció per assegurar l'atomicitat
const session = client.startSession();
session.startTransaction();
try {
    await client.db("ecommerce").collection("users").updateOne(
        { _id: "user123" },
        { $set: { email: "new.email@example.com" } },
        { session }
    );
    await client.db("ecommerce").collection("orders").updateOne(
        { user_id: "user123" },
        { $set: { status: "Updated" } },
        { session }
    );
    await session.commitTransaction();
} catch (error) {
    await session.abortTransaction();
    throw error;
} finally {
    session.endSession();
}
