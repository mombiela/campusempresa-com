En aquesta secció, explorarem com implementar diverses funcionalitats a MongoDB, des de funcionalitats bàsiques fins a avançades. Al final d'aquest tema, seràs capaç de crear, llegir, actualitzar i eliminar documents, així com implementar funcionalitats més complexes com indexació, agregació i transaccions.

Operacions CRUD Bàsiques

CRUD significa Crear, Llegir, Actualitzar i Eliminar. Aquestes són les operacions bàsiques que realitzaràs en les teves col·leccions de MongoDB.

Crear

Per inserir un document en una col·lecció, utilitzes els mètodes insertOne o insertMany.

// Inserir un únic document
db.collection('users').insertOne({
  name: "John Doe",
  age: 30,
  email: "john.doe@example.com"
});

// Inserir múltiples documents
db.collection('users').insertMany([
  { name: "Jane Doe", age: 25, email: "jane.doe@example.com" },
  { name: "Alice Smith", age: 28, email: "alice.smith@example.com" }
]);

Llegir

Per recuperar documents d'una col·lecció, utilitzes el mètode find.

// Trobar tots els documents a la col·lecció
db.collection('users').find({});

// Trobar documents amb una condició específica
db.collection('users').find({ age: { $gt: 25 } });

Actualitzar

Per modificar documents existents, utilitzes els mètodes updateOne o updateMany.

// Actualitzar un únic document
db.collection('users').updateOne(
  { name: "John Doe" },
  { $set: { age: 31 } }
);

// Actualitzar múltiples documents
db.collection('users').updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "active" } }
);

Eliminar

Per eliminar documents d'una col·lecció, utilitzes els mètodes deleteOne o deleteMany.

// Eliminar un únic document
db.collection('users').deleteOne({ name: "John Doe" });

// Eliminar múltiples documents
db.collection('users').deleteMany({ age: { $lt: 25 } });

Indexació

Els índexs milloren el rendiment de les consultes de cerca. MongoDB admet diversos tipus d'índexs.

  • Índex de Camp Únic: Indexa un únic camp en un document.
  • Índex Compost: Indexa múltiples camps en un document.
  • Índex Multiclau: Indexa camps que contenen arrays.
  • Índex de Text: Indexa contingut de text per a la cerca de text.
  • Índex Geoespacial: Indexa per a consultes basades en la ubicació.

Crear un Índex

// Crear un índex de camp únic
db.collection('users').createIndex({ email: 1 });

// Crear un índex compost
db.collection('users').createIndex({ name: 1, age: -1 });

Utilitzar Índexs

Els índexs són utilitzats automàticament per MongoDB per optimitzar el rendiment de les consultes.

// Consulta que es beneficia d'un índex
db.collection('users').find({ email: "john.doe@example.com" });

Agregació

Les operacions d'agregació processen registres de dades i retornen resultats computats. MongoDB proporciona el marc d'agregació, que és una eina poderosa per a l'anàlisi de dades.

Pipeline d'Agregació Bàsica

db.collection('orders').aggregate([
  { $match: { status: "completed" } },
  { $group: { _id: "$customerId", totalAmount: { $sum: "$amount" } } },
  { $sort: { totalAmount: -1 } }
]);
  • $match: Filtra els documents.
  • $group: Agrupa documents per un identificador especificat.
  • $sort: Ordena els documents.

Transaccions

Les transaccions et permeten executar múltiples operacions en aïllament i garantir la consistència de les dades.

Utilitzar Transaccions

const session = client.startSession();

session.startTransaction();

try {
  db.collection('accounts').updateOne(
    { accountId: 1 },
    { $inc: { balance: -100 } },
    { session }
  );

  db.collection('accounts').updateOne(
    { accountId: 2 },
    { $inc: { balance: 100 } },
    { session }
  );

  session.commitTransaction();
} catch (error) {
  session.abortTransaction();
  throw error;
} finally {
  session.endSession();
}
