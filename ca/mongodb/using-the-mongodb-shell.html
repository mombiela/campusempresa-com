Introducció al Shell de MongoDB

El Shell de MongoDB, també conegut com a mongosh, és una interfície interactiva de JavaScript per a MongoDB. Et permet interactuar amb la teva instància de MongoDB, realitzar tasques administratives i executar consultes.

Conceptes Clau

  • Shell de MongoDB (mongosh): La interfície de línia de comandes per interactuar amb MongoDB.
  • Sintaxi de JavaScript: El shell utilitza la sintaxi de JavaScript per a comandes i consultes.
  • Connexió: Connectar-se a una instància de MongoDB utilitzant el shell.

Instal·lació del Shell de MongoDB

Abans d'utilitzar el Shell de MongoDB, cal instal·lar-lo. El procés d'instal·lació varia segons el teu sistema operatiu.

Passos d'Instal·lació

  1. Descarregar: Visita el centre de descàrregues de MongoDB i descarrega el Shell de MongoDB per al teu sistema operatiu.
  2. Instal·lar: Segueix les instruccions d'instal·lació específiques per al teu sistema operatiu.
  3. Verificar la Instal·lació: Obre el teu terminal o símbol del sistema i escriu mongosh per verificar la instal·lació.
$ mongosh

Si està instal·lat correctament, hauries de veure un missatge similar a aquest:

Current Mongosh Log ID: 1234567890abcdef12345678
Connecting to: mongodb://localhost:27017
Using MongoDB: 4.4.0
Using Mongosh: 1.0.0

Connectar-se a una Instància de MongoDB

Per començar a utilitzar el Shell de MongoDB, cal connectar-se a una instància de MongoDB.

Connectar-se a una Instància Local

Per defecte, mongosh es connecta a una instància de MongoDB que s'executa a localhost al port 27017.

$ mongosh

Connectar-se a una Instància Remota

Per connectar-se a una instància remota de MongoDB, cal especificar la cadena de connexió.

$ mongosh "mongodb+srv://<username>:<password>@cluster0.mongodb.net/test"

Substitueix <username>, <password>, i cluster0.mongodb.net amb les teves credencials i adreça del clúster reals.

Comandes Bàsiques en el Shell de MongoDB

Un cop connectat, pots començar a executar comandes. Aquí tens algunes comandes bàsiques per començar.

Mostrar Bases de Dades

Llista totes les bases de dades en la instància de MongoDB.

> show dbs

Canviar de Base de Dades

Canvia a una base de dades específica.

> use myDatabase

Mostrar Col·leccions

Llista totes les col·leccions en la base de dades actual.

> show collections

Inserir un Document

Insereix un document en una col·lecció.

> db.myCollection.insertOne({ name: "Alice", age: 30 })

Trobar Documents

Recupera documents d'una col·lecció.

> db.myCollection.find()

Actualitzar un Document

Actualitza un document en una col·lecció.

> db.myCollection.updateOne({ name: "Alice" }, { $set: { age: 31 } })

Esborrar un Document

Esborra un document d'una col·lecció.

> db.myCollection.deleteOne({ name: "Alice" })

Funcions Avançades del Shell

A mesura que et familiaritzis amb el Shell de MongoDB, pots començar a utilitzar funcions més avançades.

Marc d'Agrupació

Realitza processament i anàlisi de dades complexes utilitzant el marc d'agrupació.

> db.myCollection.aggregate([
    { $match: { age: { $gt: 25 } } },
    { $group: { _id: "$age", count: { $sum: 1 } } }
])

Gestió d'Índexs

Crea i gestiona índexs per optimitzar el rendiment de les consultes.

> db.myCollection.createIndex({ name: 1 })

Comandes Administratives

Realitza tasques administratives com ara monitorització i còpies de seguretat.

> db.serverStatus()
> db.fsyncLock()

Conclusió

El Shell de MongoDB és una eina potent per interactuar amb la teva instància de MongoDB. Et permet realitzar una àmplia gamma de tasques, des d'operacions bàsiques de CRUD fins a processament avançat de dades i tasques administratives. En dominar el Shell de MongoDB, pots gestionar i interactuar eficientment amb les teves bases de dades de MongoDB.

Resum

  • Instal·lació: Instal·la el Shell de MongoDB (mongosh) per al teu sistema operatiu.
  • Connexió: Connecta't a instàncies locals o remotes de MongoDB.
  • Comandes Bàsiques: Executa comandes bàsiques com show dbs, use, insertOne, find, updateOne, i deleteOne.
  • Funcions Avançades: Utilitza funcions avançades com el marc d'agrupació, la gestió d'índexs i les comandes administratives.

Practica aquestes comandes i explora el Shell de MongoDB per obtenir una comprensió més profunda de com treballar efectivament amb MongoDB.
