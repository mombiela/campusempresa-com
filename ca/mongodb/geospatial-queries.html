Introducció a les consultes geoespacials

Les consultes geoespacials et permeten realitzar cerques i operacions basades en la ubicació en les teves dades. MongoDB admet consultes geoespacials mitjançant l'ús d'índexs geoespacials i operadors de consulta específics.

Conceptes clau

  • Tipus de dades geoespacials: MongoDB admet dos tipus principals de dades geoespacials: Point i Polygon.
  • Índexs geoespacials: Índexs que permeten la consulta eficient de dades geoespacials.
  • Operadors de consulta geoespacials: Operadors que et permeten realitzar diverses consultes geoespacials.

Tipus de dades geoespacials

MongoDB admet objectes GeoJSON i parells de coordenades heretats per a dades geoespacials.

Objectes GeoJSON

GeoJSON és un format per codificar una varietat d'estructures de dades geogràfiques.

  • Point: Representa una ubicació única. 
    { "type": "Point", "coordinates": [longitud, latitud] }
  • Polygon: Representa una àrea tancada per una sèrie de punts. 
    { 
  "type": "Polygon", 
  "coordinates": [
    [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ]
  ] 
}

Parells de coordenades heretats

Els parells de coordenades heretats són arrays simples de [longitud, latitud].

Índexs geoespacials

Per realitzar consultes geoespacials de manera eficient, cal crear índexs geoespacials.

Índex 2dsphere

L'índex 2dsphere admet consultes que interpreten geometries en una esfera semblant a la Terra.

db.collection.createIndex({ location: "2dsphere" })

Índex 2d

L'índex 2d admet consultes en un pla cartesià pla.

db.collection.createIndex({ location: "2d" })

Operadors de consulta geoespacials

MongoDB proporciona diversos operadors per realitzar consultes geoespacials.

$near

Troba documents que estan a prop d'un punt especificat.

db.collection.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [longitud, latitud] },
      $maxDistance: distància_en_metres
    }
  }
})

$geoWithin

Troba documents dins d'una geometria especificada.

db.collection.find({
  location: {
    $geoWithin: {
      $geometry: {
        type: "Polygon",
        coordinates: [ [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ] ]
      }
    }
  }
})

$geoIntersects

Troba documents que s'intersecten amb una geometria especificada.

db.collection.find({
  location: {
    $geoIntersects: {
      $geometry: {
        type: "Polygon",
        coordinates: [ [ [longitud1, latitud1], [longitud2, latitud2], [longitud3, latitud3], [longitud1, latitud1] ] ]
      }
    }
  }
})

Exemples i exercicis

Exemple 1: Creació d'un índex geoespacial

Crea un índex 2dsphere en una col·lecció anomenada places.

db.places.createIndex({ location: "2dsphere" })

Exemple 2: Trobar ubicacions properes

Troba tots els llocs dins de 500 metres d'un punt específic.

db.places.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [-73.97, 40.77] },
      $maxDistance: 500
    }
  }
})

Exercici 1: Creació i consulta de dades geoespacials

  1. Insereix alguns documents amb dades geoespacials en una col·lecció anomenada restaurants.
  2. Crea un índex 2dsphere en el camp location.
  3. Escriu una consulta per trobar tots els restaurants dins d'un quilòmetre d'un punt donat.
// Pas 1: Insereix documents
db.restaurants.insertMany([
  { name: "Restaurant A", location: { type: "Point", coordinates: [-73.97, 40.77] } },
  { name: "Restaurant B", location: { type: "Point", coordinates: [-73.98, 40.78] } },
  { name: "Restaurant C", location: { type: "Point", coordinates: [-73.99, 40.79] } }
])

// Pas 2: Crea un índex 2dsphere
db.restaurants.createIndex({ location: "2dsphere" })

// Pas 3: Consulta per restaurants propers
db.restaurants.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [-73.97, 40.77] },
      $maxDistance: 1000
    }
  }
})

Conclusió

Les consultes geoespacials a MongoDB et permeten realitzar cerques i operacions basades en la ubicació de manera potent. Comprenent els tipus de dades geoespacials, els índexs i els operadors de consulta, pots gestionar i consultar dades geoespacials de manera eficient en les teves aplicacions. Practica creant índexs i executant consultes per convertir-te en un expert en la gestió de dades geoespacials amb MongoDB.
