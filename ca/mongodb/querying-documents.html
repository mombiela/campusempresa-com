Introducció

Consultar documents és un aspecte fonamental de treballar amb MongoDB. Aquesta secció et guiarà a través dels conceptes bàsics de la consulta de documents, avançant cap a consultes més complexes. Al final d'aquesta secció, seràs capaç de recuperar i manipular dades emmagatzemades en col·leccions de MongoDB de manera eficient.

Consultes Bàsiques

Trobar Tots els Documents

Per recuperar tots els documents d'una col·lecció, utilitza el mètode find sense cap paràmetre.

db.collection.find()

Trobar Documents amb un Filtre

Per trobar documents que coincideixin amb criteris específics, passa un document de consulta al mètode find.

db.collection.find({ "field": "value" })

Exemple:

db.users.find({ "age": 25 })

Aquesta consulta recupera tots els documents de la col·lecció users on el camp age és 25.

Operadors de Consulta

MongoDB proporciona diversos operadors per construir consultes complexes.

Operadors de Comparació

  • $eq: Coincideix amb valors que són iguals a un valor especificat.
  • $ne: Coincideix amb tots els valors que no són iguals a un valor especificat.
  • $gt: Coincideix amb valors que són més grans que un valor especificat.
  • $gte: Coincideix amb valors que són més grans o iguals a un valor especificat.
  • $lt: Coincideix amb valors que són menors que un valor especificat.
  • $lte: Coincideix amb valors que són menors o iguals a un valor especificat.

Exemple:

db.users.find({ "age": { "$gt": 20, "$lt": 30 } })

Aquesta consulta recupera tots els documents on el camp age és més gran que 20 i menor que 30.

Operadors Lògics

  • $and: Uneix clàusules de consulta amb un AND lògic.
  • $or: Uneix clàusules de consulta amb un OR lògic.
  • $not: Inverteix l'efecte d'una expressió de consulta.
  • $nor: Uneix clàusules de consulta amb un NOR lògic.

Exemple:

db.users.find({
  "$or": [
    { "age": { "$lt": 20 } },
    { "age": { "$gt": 30 } }
  ]
})

Aquesta consulta recupera tots els documents on el camp age és menor que 20 o més gran que 30.

Projecció

La projecció s'utilitza per especificar o restringir els camps a retornar en els resultats de la consulta.

Incloure Camps

Per incloure camps específics en el resultat, utilitza un document de projecció amb el mètode find.

db.collection.find({}, { "field1": 1, "field2": 1 })

Exemple:

db.users.find({}, { "name": 1, "age": 1 })

Aquesta consulta recupera tots els documents de la col·lecció users, però només inclou els camps name i age en els resultats.

Excloure Camps

Per excloure camps específics, estableix el camp a 0 en el document de projecció.

db.collection.find({}, { "field": 0 })

Exemple:

db.users.find({}, { "password": 0 })

Aquesta consulta recupera tots els documents de la col·lecció users, excloent el camp password dels resultats.

Ordenació

Per ordenar els resultats d'una consulta, utilitza el mètode sort.

db.collection.find().sort({ "field": 1 }) // Ordre ascendent
db.collection.find().sort({ "field": -1 }) // Ordre descendent

Exemple:

db.users.find().sort({ "age": 1 })

Aquesta consulta recupera tots els documents de la col·lecció users, ordenats pel camp age en ordre ascendent.

Limitar i Saltar

Per limitar el nombre de documents retornats per una consulta, utilitza el mètode limit. Per saltar un nombre de documents, utilitza el mètode skip.

db.collection.find().limit(number)
db.collection.find().skip(number)

Exemple:

db.users.find().limit(5)
db.users.find().skip(5)

La primera consulta recupera els primers 5 documents de la col·lecció users. La segona consulta salta els primers 5 documents i recupera la resta.

Consultes Avançades

Documents Imbricats

Per consultar documents imbricats, utilitza la notació de punts.

db.collection.find({ "field.subfield": "value" })

Exemple:

db.users.find({ "address.city": "New York" })

Aquesta consulta recupera tots els documents de la col·lecció users on el camp city en el subdocument address és "New York".

Consultes d'Array

MongoDB proporciona operadors per consultar arrays.

  • $elemMatch: Coincideix amb documents que contenen un camp d'array amb almenys un element que coincideix amb tots els criteris de consulta especificats.
  • $size: Coincideix amb qualsevol array amb el nombre especificat d'elements.

Exemple:

db.users.find({ "hobbies": { "$elemMatch": { "name": "reading", "frequency": "daily" } } })

Aquesta consulta recupera tots els documents on l'array hobbies conté un element amb name "reading" i frequency "daily".

Conclusió

Consultar documents en MongoDB és una funció poderosa que et permet recuperar i manipular dades de manera eficient. Entenent i utilitzant diversos operadors de consulta, projecció, ordenació i tècniques de consulta avançades, pots realitzar tasques complexes de recuperació de dades amb facilitat. Practica aquests conceptes amb dades del món real per convertir-te en un expert en la consulta de col·leccions de MongoDB.
