Operacions CRUD a MongoDB
Tipus de Dades a MongoDB ►

Introducció

CRUD significa Crear, Llegir, Actualitzar i Eliminar. Aquestes són les quatre operacions bàsiques que es poden realitzar en una base de dades. En aquesta secció, explorarem com realitzar operacions CRUD utilitzant MongoDB, una popular base de dades NoSQL.

Prerequisits

Abans d'endinsar-nos en les operacions CRUD, assegura't de tenir:

  • MongoDB instal·lat a la teva màquina.
  • MongoDB Compass o qualsevol altre client de MongoDB.
  • Comprensió bàsica de les col·leccions i documents de MongoDB.

Operacions de Creació

Crear dades a MongoDB implica inserir documents en una col·lecció.

Inserir un Document Únic

Per inserir un document únic en una col·lecció, utilitza el mètode insertOne.

db.collectionName.insertOne({
  name: "John Doe",
  age: 29,
  email: "john.doe@example.com"
});

Inserir Múltiples Documents

Per inserir múltiples documents alhora, utilitza el mètode insertMany.

db.collectionName.insertMany([
  { name: "Jane Doe", age: 25, email: "jane.doe@example.com" },
  { name: "Alice Smith", age: 30, email: "alice.smith@example.com" }
]);

Operacions de Lectura

Llegir dades de MongoDB implica fer consultes a la base de dades per recuperar documents.

Trobar un Document Únic

Per trobar un document únic que coincideixi amb una consulta, utilitza el mètode findOne.

db.collectionName.findOne({ name: "John Doe" });

Trobar Múltiples Documents

Per trobar múltiples documents que coincideixin amb una consulta, utilitza el mètode find.

db.collectionName.find({ age: { $gt: 25 } });

Projecció

Per limitar els camps retornats per una consulta, utilitza la projecció.

db.collectionName.find(
  { age: { $gt: 25 } },
  { name: 1, email: 1, _id: 0 }
);

Operacions d'Actualització

Actualitzar dades a MongoDB implica modificar documents existents.

Actualitzar un Document Únic

Per actualitzar un document únic, utilitza el mètode updateOne.

db.collectionName.updateOne(
  { name: "John Doe" },
  { $set: { age: 30 } }
);

Actualitzar Múltiples Documents

Per actualitzar múltiples documents, utilitza el mètode updateMany.

db.collectionName.updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "young" } }
);

Reemplaçar un Document

Per reemplaçar un document sencer, utilitza el mètode replaceOne.

db.collectionName.replaceOne(
  { name: "John Doe" },
  { name: "John Doe", age: 30, email: "john.doe@newdomain.com" }
);

Operacions d'Eliminació

Eliminar dades a MongoDB implica eliminar documents d'una col·lecció.

Eliminar un Document Únic

Per eliminar un document únic, utilitza el mètode deleteOne.

db.collectionName.deleteOne({ name: "John Doe" });

Eliminar Múltiples Documents

Per eliminar múltiples documents, utilitza el mètode deleteMany.

db.collectionName.deleteMany({ age: { $lt: 25 } });

Conclusió

En aquesta secció, hem cobert les operacions bàsiques CRUD a MongoDB:

  • Crear: Inserir documents únics o múltiples.
  • Llegir: Consultar documents únics o múltiples amb projeccions opcionals.
  • Actualitzar: Modificar documents únics o múltiples, o reemplaçar documents sencers.
  • Eliminar: Eliminar documents únics o múltiples.

Entendre aquestes operacions és fonamental per treballar amb MongoDB, ja que formen la base de la interacció amb la base de dades. Practica aquestes operacions per convertir-te en un expert en la gestió de col·leccions i documents de MongoDB.
