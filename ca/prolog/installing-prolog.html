Introducció

Abans d'endinsar-nos en la programació en Prolog, és essencial tenir l'entorn de Prolog configurat a la teva màquina. Aquesta secció et guiarà a través del procés d'instal·lació de Prolog en diversos sistemes operatius, assegurant que estiguis preparat per començar a codificar.

Instal·lant SWI-Prolog

SWI-Prolog és un dels entorns de Prolog més populars i àmpliament utilitzats. És de codi obert i suporta moltes característiques que són útils tant per a principiants com per a usuaris avançats.

Windows

  1. Descarrega l'Instal·lador:

    • Visita la pàgina de descàrrega de SWI-Prolog.
    • Tria l'instal·lador per a Windows (normalment un fitxer .exe).

  2. Executa l'Instal·lador:

    • Obre el fitxer .exe descarregat.
    • Segueix les instruccions en pantalla per completar la instal·lació.
    • Per defecte, SWI-Prolog s'instal·larà a C:\Program Files\swipl.

  3. Verifica la Instal·lació:

    • Obre el Símbol del sistema.
    • Escriu swipl i prem Enter.
    • Hauries de veure el prompt de SWI-Prolog (?-).
C:\> swipl
Welcome to SWI-Prolog (threaded, 64 bits, version 8.4.2)
...
?-

macOS

  1. Utilitzant Homebrew:

    • Si no tens Homebrew instal·lat, instal·la'l executant la següent comanda al Terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la SWI-Prolog utilitzant Homebrew: 
      brew install swi-prolog

  2. Verifica la Instal·lació:

    • Obre el Terminal.
    • Escriu swipl i prem Enter.
    • Hauries de veure el prompt de SWI-Prolog (?-).
$ swipl
Welcome to SWI-Prolog (threaded, 64 bits, version 8.4.2)
...
?-

Linux

  1. Utilitzant el Gestor de Paquets:

    • Per a distribucions basades en Debian (com Ubuntu), utilitza apt: 
      sudo apt-get update
sudo apt-get install swi-prolog
    • Per a distribucions basades en Red Hat (com Fedora), utilitza dnf: 
      sudo dnf install swi-prolog

  2. Verifica la Instal·lació:

    • Obre el teu terminal.
    • Escriu swipl i prem Enter.
    • Hauries de veure el prompt de SWI-Prolog (?-).
$ swipl
Welcome to SWI-Prolog (threaded, 64 bits, version 8.4.2)
...
?-

Implementacions Alternatives de Prolog

Tot i que SWI-Prolog és altament recomanat, hi ha altres implementacions de Prolog que podries considerar:

  • GNU Prolog:

    • Visita la pàgina de descàrrega de GNU Prolog.
    • Segueix les instruccions específiques per al teu sistema operatiu.

  • SICStus Prolog:

    • Visita el lloc web de SICStus Prolog.
    • Tingues en compte que SICStus Prolog és un producte comercial i requereix una llicència.

Conclusió

Seguint els passos descrits anteriorment, ara hauries de tenir un entorn de Prolog funcionant a la teva màquina. Tant si has triat SWI-Prolog com una altra implementació, estàs preparat per començar a explorar el fascinant món de la programació en Prolog. En les properes seccions, ens endinsarem en els conceptes bàsics de la sintaxi i la semàntica de Prolog, ajudant-te a escriure els teus primers programes en Prolog.
