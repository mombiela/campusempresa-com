Predicats Integrats

Introducció

Prolog, un llenguatge de programació lògica, ve amb un ric conjunt de predicats integrats que faciliten diverses operacions com l'aritmètica, la manipulació de llistes, l'entrada/sortida, i més. Entendre aquests predicats integrats és crucial per escriure programes Prolog eficients i efectius.

Conceptes Clau

Què són els Predicats Integrats?

  • Definició: Els predicats integrats són funcions predefinides en Prolog que realitzen operacions específiques.
  • Propòsit: Simplifiquen tasques comunes i milloren la funcionalitat del llenguatge sense requerir que els usuaris defineixin aquestes operacions des de zero.

Categories de Predicats Integrats

  • Predicats Aritmètics: Realitzen operacions matemàtiques.
  • Predicats de Manipulació de Llistes: Gestionen operacions sobre llistes.
  • Predicats d'Entrada/Sortida: Gestionen la lectura i escriptura de fluxos.
  • Predicats de Control: Controlen el flux del programa.
  • Predicats de Comparació: Comparen termes i valors.

Predicats Aritmètics

Els predicats aritmètics et permeten realitzar operacions matemàtiques. Aquí en tens alguns de comuns:

  • is/2: Avalua una expressió aritmètica.
  • =/2: Comprova si dos termes són iguals.
  • >/2, >=/2, </2, =</2: Operadors de comparació.

Exemples

% Exemple de `is/2`
X is 3 + 4. % X = 7

% Exemple de comparació
7 > 5. % true
3 =< 4. % true

Predicats de Manipulació de Llistes

Prolog proporciona diversos predicats per treballar amb llistes:

  • member/2: Comprova si un element és membre d'una llista.
  • append/3: Concatena dues llistes.
  • length/2: Troba la longitud d'una llista.
  • reverse/2: Inverteix una llista.

Exemples

% Exemple de `member/2`
member(3, [1, 2, 3, 4]). % true

% Exemple de `append/3`
append([1, 2], [3, 4], X). % X = [1, 2, 3, 4]

% Exemple de `length/2`
length([1, 2, 3, 4], L). % L = 4

% Exemple de `reverse/2`
reverse([1, 2, 3, 4], R). % R = [4, 3, 2, 1]

Predicats d'Entrada/Sortida

Aquests predicats gestionen la lectura i escriptura de fluxos:

  • write/1: Escriu un terme al flux de sortida actual.
  • read/1: Llegeix un terme del flux d'entrada actual.
  • nl/0: Escriu un caràcter de nova línia.

Exemples

% Exemple de `write/1` i `nl/0`
write('Hello, World!'), nl. % Hello, World!

% Exemple de `read/1`
% Suposant que l'usuari introdueix: hello.
read(X). % X = hello

Predicats de Control

Els predicats de control gestionen el flux del programa:

  • fail/0: Sempre falla.
  • true/0: Sempre té èxit.
  • repeat/0: Sempre té èxit i es pot utilitzar per crear bucles.
  • cut (!)/0: Poda punts de tria per controlar el backtracking.

Exemples

% Exemple de `fail/0` i `true/0`
true. % true
fail. % false

% Exemple de `repeat/0`
repeat, write('Hello'), nl, fail. % Bucle infinit imprimint 'Hello'

% Exemple de `cut (!)/0`
member(X, [1, 2, 3]), !, write(X). % Imprimeix el primer element i atura el backtracking

Predicats de Comparació

Els predicats de comparació s'utilitzen per comparar termes:

  • ==/2: Comprova si dos termes són idèntics.
  • \==/2: Comprova si dos termes no són idèntics.
  • @</2, @>/2, @=</2, @>=/2: Comparació lexicogràfica de termes.

Exemples

% Exemple de `==/2` i `\==/2`
1 == 1. % true
1 \== 2. % true

% Exemple de comparació lexicogràfica
apple @< banana. % true
banana @> apple. % true

Conclusió

Els predicats integrats en Prolog són eines poderoses que simplifiquen moltes tasques comunes de programació. Dominant aquests predicats, pots escriure codi Prolog més eficient i llegible. Aquest tema ha cobert els predicats integrats essencials en diverses categories, proporcionant exemples per il·lustrar el seu ús. Practica utilitzant aquests predicats en diferents escenaris per convertir-te en un expert en programació Prolog.
