Llistes en Prolog
Recursió ►

Les llistes són una estructura de dades fonamental en Prolog, utilitzada per emmagatzemar seqüències d'elements. Entendre com treballar amb llistes és crucial per a una programació efectiva en Prolog.

Conceptes Clau

  • Estructura de la Llista: Les llistes en Prolog estan tancades entre claudàtors [] i poden contenir qualsevol nombre d'elements, incloent altres llistes.
  • Cap i Cua: El primer element d'una llista es diu cap, i la resta de la llista es diu cua.
  • Llista Buida: La llista buida es representa per [].
  • Operacions amb Llistes: Les operacions comunes inclouen comprovar la pertinença, la concatenació i el càlcul de la longitud.

Estructura de la Llista

Una llista en Prolog s'escriu com [Element1, Element2, ..., ElementN]. Per exemple:

[1, 2, 3, 4, 5]

Cap i Cua

El cap d'una llista és el seu primer element, i la cua és la llista dels elements restants. Per exemple, en la llista [1, 2, 3, 4, 5], el cap és 1 i la cua és [2, 3, 4, 5].

Pots descompondre una llista en el seu cap i cua utilitzant l'operador |:

[Head | Tail] = [1, 2, 3, 4, 5].
% Head = 1
% Tail = [2, 3, 4, 5]

Operacions Comunes amb Llistes

Comprovar la Pertinença

Per comprovar si un element és membre d'una llista, pots utilitzar el predicat incorporat member/2:

member(X, [1, 2, 3, 4, 5]).
% X = 1;
% X = 2;
% X = 3;
% X = 4;
% X = 5;
% false.

Concatenació

Per concatenar dues llistes, pots utilitzar el predicat incorporat append/3:

append([1, 2, 3], [4, 5], Result).
% Result = [1, 2, 3, 4, 5].

Càlcul de la Longitud

Per calcular la longitud d'una llista, utilitza el predicat incorporat length/2:

length([1, 2, 3, 4, 5], Length).
% Length = 5.

Exemples i Exercicis

Exemple 1: Descompondre una Llista

% Descompondre la llista [a, b, c, d] en cap i cua
[a, b, c, d] = [Head | Tail].
% Head = a
% Tail = [b, c, d]

Exemple 2: Comprovar la Pertinença

% Comprovar si 'c' és membre de la llista [a, b, c, d]
member(c, [a, b, c, d]).
% true.

Exercici 1: Concatenar Llistes

Escriu una consulta per concatenar les llistes [1, 2] i [3, 4, 5].

% La teva consulta aquí
append([1, 2], [3, 4, 5], Result).
% Result = [1, 2, 3, 4, 5].

Exercici 2: Calcular la Longitud

Escriu una consulta per trobar la longitud de la llista [apple, banana, cherry].

% La teva consulta aquí
length([apple, banana, cherry], Length).
% Length = 3.

Conclusió

Les llistes són una estructura de dades versàtil i essencial en Prolog. Entenent com crear, descompondre i manipular llistes, pots realitzar una àmplia gamma d'operacions de manera eficient. Practica els exemples i exercicis proporcionats per consolidar la teva comprensió de les llistes en Prolog.
