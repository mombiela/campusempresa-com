Operacions aritmètiques en Prolog
Estructures de Control ►

Introducció

Les operacions aritmètiques són fonamentals en qualsevol llenguatge de programació, incloent Prolog. Aquesta secció cobrirà els conceptes bàsics de les operacions aritmètiques en Prolog, incloent com realitzar càlculs, utilitzar operadors aritmètics i gestionar expressions aritmètiques.

Operadors aritmètics bàsics

Prolog suporta una varietat d'operadors aritmètics. Aquests són els més utilitzats:

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió (punt flotant)
  • // : Divisió entera
  • mod : Mòdul (resta)
  • ^ : Exponenciació

Exemples

Vegem alguns exemples bàsics d'operacions aritmètiques en Prolog:

% Suma
Result is 3 + 2. % Result = 5

% Resta
Result is 5 - 3. % Result = 2

% Multiplicació
Result is 4 * 2. % Result = 8

% Divisió (punt flotant)
Result is 7 / 2. % Result = 3.5

% Divisió entera
Result is 7 // 2. % Result = 3

% Mòdul
Result is 7 mod 2. % Result = 1

% Exponenciació
Result is 2 ^ 3. % Result = 8

Utilitzant expressions aritmètiques

En Prolog, les expressions aritmètiques es calculen utilitzant l'operador is. L'operador is s'utilitza per assignar el resultat d'una expressió aritmètica a una variable.

Exemple

% Assignant el resultat d'una expressió aritmètica a una variable
X is 5 + 3. % X = 8

Nota important

L'operador is requereix que el costat dret de l'expressió estigui completament instanciat (és a dir, totes les variables han de tenir valors) abans de l'avaluació.

% Això funcionarà
A = 5, B = 3, X is A + B. % X = 8

% Això no funcionarà i resultarà en un error
X is A + B, A = 5, B = 3. % Error: Arguments are not sufficiently instantiated

Operadors de comparació

Prolog també proporciona operadors de comparació per a expressions aritmètiques:

  • =:= : Igual a
  • =\= : Diferent de
  • < : Menor que
  • > : Major que
  • =< : Menor o igual a
  • >= : Major o igual a

Exemples

% Igual a
5 =:= 5. % true

% Diferent de
5 =\= 3. % true

% Menor que
3 < 5. % true

% Major que
5 > 3. % true

% Menor o igual a
3 =< 5. % true

% Major o igual a
5 >= 3. % true

Exercicis pràctics

Aquí tens alguns exercicis per practicar les operacions aritmètiques en Prolog:

  1. Escriu un predicat Prolog sum/3 que prengui dos números i retorni la seva suma.

    sum(A, B, Sum) :-
    Sum is A + B.

  2. Escriu un predicat Prolog factorial/2 que calculi el factorial d'un número donat.

    factorial(0, 1).
factorial(N, Result) :-
    N > 0,
    N1 is N - 1,
    factorial(N1, R1),
    Result is N * R1.

  3. Escriu un predicat Prolog max/3 que prengui tres números i retorni el màxim dels tres.

    max(A, B, C, Max) :-
    Max1 is max(A, B),
    Max is max(Max1, C).

Conclusió

Les operacions aritmètiques en Prolog són senzilles un cop entens els operadors bàsics i com utilitzar l'operador is per avaluar expressions. Recorda assegurar-te que totes les variables en una expressió aritmètica estiguin instanciades abans de l'avaluació. Practica amb els exercicis proporcionats per consolidar la teva comprensió de les operacions aritmètiques en Prolog.
