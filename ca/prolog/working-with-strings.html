Introducció

En Prolog, les cadenes són seqüències de caràcters tancades entre cometes dobles. Sovint s'utilitzen per representar dades de text. Aquesta secció cobrirà els conceptes bàsics de treballar amb cadenes en Prolog, incloent com crear, manipular i consultar cadenes.

Creació de Cadenes

  • Les cadenes en Prolog estan tancades entre cometes dobles.
  • Exemple: "Hola, Món!"
% Exemple d'una cadena en Prolog
my_string("Hola, Món!").

Operacions Bàsiques amb Cadenes

Concatenació

  • La concatenació de cadenes es pot fer utilitzant el predicat incorporat atom_concat/3.
% Concatenant dues cadenes
concatenate_strings(Str1, Str2, Result) :-
    atom_concat(Str1, Str2, Result).

% Exemple d'ús
% ?- concatenate_strings("Hola, ", "Món!", Result).
% Result = "Hola, Món!".

Subcadena

  • Per comprovar si una cadena és una subcadena d'una altra, utilitza el predicat sub_atom/5.
% Comprovant si una cadena és una subcadena
is_substring(Sub, Str) :-
    sub_atom(Str, _, _, _, Sub).

% Exemple d'ús
% ?- is_substring("Món", "Hola, Món!").
% true.

Longitud de la Cadena

  • La longitud d'una cadena es pot determinar utilitzant el predicat atom_length/2.
% Obtenint la longitud d'una cadena
string_length(Str, Length) :-
    atom_length(Str, Length).

% Exemple d'ús
% ?- string_length("Hola, Món!", Length).
% Length = 11.

Manipulacions Avançades de Cadenes

Divisió de Cadenes

  • La divisió de cadenes es pot aconseguir utilitzant el predicat split_string/4.
% Dividint una cadena per un delimitador
split_string_example(Str, Delimiter, Substrings) :-
    split_string(Str, Delimiter, "", Substrings).

% Exemple d'ús
% ?- split_string_example("poma,plàtan,cirera", ",", Substrings).
% Substrings = ["poma", "plàtan", "cirera"].

Reemplaçament de Subcadenes

  • El reemplaçament de subcadenes es pot fer utilitzant una combinació de sub_atom/5 i atom_concat/3.
% Reemplaçant una subcadena en una cadena
replace_substring(Str, Sub, Replacement, Result) :-
    sub_atom(Str, Before, Length, After, Sub),
    sub_atom(Str, 0, Before, _, Start),
    sub_atom(Str, _, After, 0, End),
    atom_concat(Start, Replacement, Temp),
    atom_concat(Temp, End, Result).

% Exemple d'ús
% ?- replace_substring("Hola, Món!", "Món", "Prolog", Result).
% Result = "Hola, Prolog!".

Exercicis Pràctics

  1. Concatenar Tres Cadenes: Escriu un predicat concatenate_three/4 que concateni tres cadenes.
  2. Trobar la Posició de la Subcadena: Escriu un predicat substring_position/3 que trobi la posició inicial d'una subcadena dins d'una cadena.
  3. Cadena a Llista de Caràcters: Escriu un predicat string_to_char_list/2 que converteixi una cadena en una llista dels seus caràcters.
% Exercici 1: Concatenar Tres Cadenes
concatenate_three(Str1, Str2, Str3, Result) :-
    atom_concat(Str1, Str2, Temp),
    atom_concat(Temp, Str3, Result).

% Exercici 2: Trobar la Posició de la Subcadena
substring_position(Sub, Str, Position) :-
    sub_atom(Str, Position, _, _, Sub).

% Exercici 3: Cadena a Llista de Caràcters
string_to_char_list(Str, CharList) :-
    atom_chars(Str, CharList).

Conclusió

Treballar amb cadenes en Prolog implica entendre com crear, manipular i consultar-les utilitzant predicats incorporats. En dominar aquestes operacions, pots gestionar dades de text de manera efectiva en els teus programes Prolog. Practica els exercicis proporcionats per reforçar la teva comprensió i convertir-te en un expert en la manipulació de cadenes en Prolog.
