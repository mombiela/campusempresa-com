Unificació en Prolog
Retrocessió ►

La unificació és un concepte fonamental en Prolog, essencial per entendre com el llenguatge processa consultes i coincideix patrons. Aquest tema et guiarà des dels conceptes bàsics fins als aspectes avançats de la unificació, proporcionant explicacions clares, exemples i exercicis.

Què és la Unificació?

La unificació és el procés de fer que dos termes siguin idèntics trobant una substitució adequada per a les variables. És el mecanisme central que Prolog utilitza per coincidir objectius amb fets i regles a la base de dades.

Conceptes Clau

  • Terme: El bloc de construcció bàsic en Prolog, que pot ser un àtom, número, variable o terme compost.
  • Variable: Un marcador de posició que es pot unificar amb diferents termes.
  • Substitució: El procés de reemplaçar una variable amb un terme.
  • Coincidència: El procés de comparar dos termes per veure si es poden fer idèntics mitjançant la substitució.

Unificació Bàsica

En Prolog, la unificació s'utilitza per determinar si dos termes es poden fer idèntics. Comencem amb alguns exemples simples.

Exemple 1: Unificant Àtoms

?- apple = apple.
true.

En aquest exemple, els dos àtoms apple i apple són idèntics, així que la unificació té èxit.

Exemple 2: Unificant Variables amb Àtoms

?- X = apple.
X = apple.

Aquí, la variable X s'unifica amb l'àtom apple. La variable X ara té el valor apple.

Exemple 3: Unificant Variables amb Variables

?- X = Y.
X = Y.

En aquest cas, tant X com Y són variables. Es poden unificar entre elles, és a dir, poden prendre el mateix valor.

Unificació amb Termes Compostos

La unificació també funciona amb termes compostos, que són termes que tenen un functor i arguments.

Exemple 4: Unificant Termes Compostos

?- point(X, Y) = point(1, 2).
X = 1,
Y = 2.

Aquí, point(X, Y) s'unifica amb point(1, 2). Les variables X i Y s'unifiquen amb 1 i 2, respectivament.

Exemple 5: Fallada en la Unificació

?- point(X, Y) = point(1, 2, 3).
false.

En aquest exemple, la unificació falla perquè els termes tenen estructures diferents. point(X, Y) té dos arguments, mentre que point(1, 2, 3) en té tres.

Unificació Avançada

Explorem alguns escenaris més complexos que involucren llistes i estructures niades.

Exemple 6: Unificant Llistes

?- [H|T] = [1, 2, 3].
H = 1,
T = [2, 3].

En aquest exemple, la llista [H|T] s'unifica amb [1, 2, 3]. El cap H s'unifica amb 1, i la cua T s'unifica amb [2, 3].

Exemple 7: Estructures Niades

?- person(name(First, Last), age(Age)) = person(name(john, doe), age(30)).
First = john,
Last = doe,
Age = 30.

Aquí, l'estructura niada person(name(First, Last), age(Age)) s'unifica amb person(name(john, doe), age(30)). Les variables First, Last i Age s'unifiquen amb john, doe i 30, respectivament.

Exercicis

  1. Unifica els següents termes i proporciona les substitucions resultants:

    • ?- X = Y, Y = 3.
    • ?- f(a, X) = f(Y, b).
    • ?- [A, B | T] = [1, 2, 3, 4].

  2. Determina si les següents unificacions tenen èxit o fallen:

    • ?- g(X, Y) = g(a, b).
    • ?- h(X, X) = h(a, b).
    • ?- [H|T] = [1].

Conclusió

La unificació és un concepte poderós i essencial en Prolog, que permet al llenguatge coincidir patrons i resoldre consultes. Entenent i practicant la unificació, pots aprofitar tot el potencial de les capacitats de programació lògica de Prolog. Recorda experimentar amb diferents termes i estructures per aprofundir en la teva comprensió de com funciona la unificació en diversos escenaris.
