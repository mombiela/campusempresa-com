Variables i Constants en Prolog
Unificació ►

Introducció

En Prolog, entendre les variables i les constants és crucial per escriure programes efectius i eficients. Aquesta secció cobrirà els conceptes bàsics de les variables i les constants, el seu ús i com es diferencien entre elles.

Variables

Les variables en Prolog són espais reservats per a termes (que poden ser àtoms, nombres o estructures complexes). S'utilitzen per emmagatzemar valors que poden canviar durant l'execució d'un programa.

Conceptes Clau

  • Convencions de Nomenclatura: Les variables han de començar amb una lletra majúscula o un guió baix (_).
  • Variables Anònimes: Representades per un guió baix (_), s'utilitzen quan el valor específic no és important.

Exemples

% Exemple d'una variable
X = 5.

% Exemple d'una variable anònima
member(_, [1, 2, 3]).

Explicació

  • En el primer exemple, X és una variable que se li assigna el valor 5.
  • En el segon exemple, _ és una variable anònima utilitzada en el predicat member, indicant que no ens importa el valor específic.

Constants

Les constants en Prolog són valors fixos que no canvien. Poden ser àtoms o nombres.

Conceptes Clau

  • Àtoms: Cadenes de caràcters que comencen amb una lletra minúscula o estan tancades entre cometes simples.
  • Nombres: Valors enters o de punt flotant.

Exemples

% Exemple d'un àtom
color(red).

% Exemple d'un nombre
age(30).

Explicació

  • En el primer exemple, red és un àtom que representa un color.
  • En el segon exemple, 30 és un nombre que representa una edat.

Diferències Entre Variables i Constants

| Característica | Variables | Constants | |----------------|------------------------------------|------------------------------------| | Nomenclatura | Comencen amb majúscula o guió baix | Comencen amb minúscula o són nombres| | Mutabilitat | Poden canviar durant l'execució | Valors fixos | | Ús | Espais reservats per a termes | Representen valors fixos |

Exemples Pràctics

Exemple 1: Ús de Variables

% Definir un predicat per comprovar si algú és adult
is_adult(Age) :- Age >= 18.

% Consulta
?- is_adult(20).
% Sortida: true

Explicació

  • Age és una variable que pren el valor 20 durant la consulta. El predicat comprova si Age és major o igual a 18.

Exemple 2: Ús de Constants

% Definir un predicat per comprovar si el color és vermell
is_red(Color) :- Color = red.

% Consulta
?- is_red(red).
% Sortida: true

Explicació

  • red és un àtom constant. El predicat comprova si el Color és red.

Conclusió

Entendre la distinció entre variables i constants és fonamental en la programació en Prolog. Les variables són dinàmiques i poden canviar durant l'execució, mentre que les constants són estàtiques i representen valors fixos. Dominar aquests conceptes t'ajudarà a escriure programes en Prolog més efectius.
