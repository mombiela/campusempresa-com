Introducció a Prolog i Bases de Dades

Prolog és un llenguatge potent per treballar amb bases de dades gràcies al seu suport inherent per a consultes lògiques i coincidència de patrons. Aquesta secció introduirà els conceptes bàsics d'ús de Prolog amb bases de dades, incloent com representar i consultar dades.

Representació de Dades en Prolog

En Prolog, les dades es representen típicament utilitzant fets i regles. Aquí, cobrirem com representar dades relacionals utilitzant fets de Prolog.

  • Fets: Afirmacions bàsiques sobre el món.
  • Regles: Declaracions lògiques que defineixen relacions entre fets.

Exemple: Representant una Base de Dades Simple

Considerem una base de dades simple d'empleats i departaments.

% Fets
employee(john, sales).
employee(mary, marketing).
employee(susan, sales).
employee(james, it).

department(sales, 101).
department(marketing, 102).
department(it, 103).

En aquest exemple:

  • employee(Name, Department) representa que un empleat treballa en un departament específic.
  • department(Name, Code) representa que un departament té un codi específic.

Consultar Dades en Prolog

Les consultes de Prolog permeten recuperar informació de la base de dades coincidint patrons contra fets i regles.

Consultes Bàsiques

Per trobar tots els empleats en el departament de vendes, pots utilitzar la següent consulta:

?- employee(Name, sales).

Aquesta consulta retornarà:

Name = john ;
Name = susan.

Consultes Complexes

També pots realitzar consultes més complexes combinant múltiples fets i regles. Per exemple, per trobar el codi del departament per als empleats en el departament de vendes:

?- employee(Name, sales), department(sales, Code).

Aquesta consulta retornarà:

Name = john,
Code = 101 ;

Name = susan,
Code = 101.

Operacions Avançades de Base de Dades

A mesura que avances, pots realitzar operacions més sofisticades com unions, agregacions i actualitzacions.

Unions

Unir dades de múltiples relacions és senzill en Prolog. Per exemple, per trobar els codis de departament per a tots els empleats:

?- employee(Name, Department), department(Department, Code).

Agregacions

Prolog també pot realitzar agregacions utilitzant predicats integrats com findall/3. Per exemple, per trobar tots els empleats en una llista:

?- findall(Name, employee(Name, _), Employees).

Aquesta consulta retornarà:

Employees = [john, mary, susan, james].

Actualitzacions

Actualitzar dades en Prolog implica retractar fets antics i afirmar-ne de nous. Per exemple, per actualitzar el departament de John a IT:

?- retract(employee(john, sales)), assert(employee(john, it)).

Exercicis Pràctics

Per consolidar la teva comprensió, prova els següents exercicis:

  1. Exercici 1: Afegeix nous empleats i departaments a la base de dades i consulta'ls.
  2. Exercici 2: Escriu una consulta per trobar tots els empleats que treballen en un departament amb un codi específic.
  3. Exercici 3: Actualitza el departament d'un empleat i verifica el canvi amb una consulta.

Conclusió

La base lògica de Prolog el converteix en una excel·lent elecció per treballar amb bases de dades. Comprenent com representar, consultar i manipular dades, pots aprofitar els punts forts de Prolog per construir aplicacions de bases de dades potents i flexibles. Practica amb els exercicis proporcionats per aprofundir la teva comprensió i preparar-te per a temes més avançats.
