Gestió Eficient de Datasets
Ús d'Utilitats del Sistema ►

La gestió eficient de datasets és crucial per optimitzar el rendiment i la utilització de recursos dels teus treballs JCL. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades per gestionar datasets de manera efectiva utilitzant JCL.

Introducció als Datasets en JCL

  • Definició: Un dataset en JCL és una col·lecció de registres de dades emmagatzemats en un disc o cinta.
  • Tipus de Datasets:
    • Datasets Seqüencials: Les dades s'emmagatzemen de manera seqüencial.
    • Datasets Particionats (PDS): Conté múltiples membres, cadascun dels quals pot ser un dataset separat.
    • Datasets VSAM: Utilitzats per a l'accés d'alt rendiment a les dades.

Operacions Bàsiques amb Datasets

  • Creació de Datasets: Utilitza la instrucció DD per definir datasets.
  • Eliminació de Datasets: Utilitza la utilitat IEFBR14 per eliminar datasets.

Exemple: Creació d'un Dataset Seqüencial

//CREATE   JOB (ACCT),'CREATE DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//DD1      DD  DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE),
//             SPACE=(TRK,(1,1)),UNIT=SYSDA,
//             DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)
  • Explicació:
    • DSN=MY.DATASET: Especifica el nom del dataset.
    • DISP=(NEW,CATLG,DELETE): Indica que el dataset és nou, s'ha de catalogar si el treball es completa amb èxit, i eliminar si falla.
    • SPACE=(TRK,(1,1)): Assigna espai en pistes.
    • UNIT=SYSDA: Especifica el tipus de dispositiu.
    • DCB=(RECFM=FB,LRECL=80,BLKSIZE=800): Defineix els paràmetres del bloc de control de dades.

Exemple: Eliminació d'un Dataset

//DELETE   JOB (ACCT),'DELETE DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//DD1      DD  DSN=MY.DATASET,DISP=(OLD,DELETE)
  • Explicació:
    • DISP=(OLD,DELETE): Especifica que el dataset s'ha d'eliminar.

Tècniques Avançades de Gestió de Datasets

  • Ús de GDGs (Generation Data Groups): Gestiona datasets que estan lògicament relacionats.
  • Assignació Eficient d'Espai: Optimitza l'assignació d'espai per evitar el malbaratament i assegurar un ús eficient de l'emmagatzematge.
  • Compressió de Datasets: Utilitza utilitats per comprimir datasets i estalviar espai.

Generation Data Groups (GDGs)

  • Definició: Els GDGs són una col·lecció de datasets que estan històricament relacionats.
  • Creació de GDGs:
    • Defineix una base GDG.
    • Crea noves generacions.

Exemple: Definició d'una Base GDG

//DEFINE   JOB (ACCT),'DEFINE GDG',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD  SYSOUT=A
//SYSIN    DD  *
  DEFINE GDG (NAME(MY.GDG.BASE) LIMIT(5) SCRATCH)
/*
  • Explicació:
    • DEFINE GDG: Comanda IDCAMS per definir un GDG.
    • NAME(MY.GDG.BASE): Especifica el nom de la base GDG.
    • LIMIT(5): Limita el nombre de generacions a 5.
    • SCRATCH: Elimina la generació més antiga quan es supera el límit.

Exemple: Creació d'una Nova Generació GDG

//CREATE   JOB (ACCT),'CREATE GDG',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//DD1      DD  DSN=MY.GDG.BASE(+1),DISP=(NEW,CATLG,DELETE),
//             SPACE=(TRK,(1,1)),UNIT=SYSDA,
//             DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)
  • Explicació:
    • DSN=MY.GDG.BASE(+1): Especifica la creació d'una nova generació.

Assignació Eficient d'Espai

  • Espai Primari i Secundari: Assigna espai primari per a l'emmagatzematge inicial i espai secundari per a l'excés.
  • Millors Pràctiques:
    • Estima els requisits d'espai amb precisió.
    • Utilitza el paràmetre SPACE amb saviesa.

Exemple: Optimització de l'Assignació d'Espai

//ALLOCATE JOB (ACCT),'ALLOCATE SPACE',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//DD1      DD  DSN=MY.EFFICIENT.DATASET,DISP=(NEW,CATLG,DELETE),
//             SPACE=(CYL,(5,2)),UNIT=SYSDA,
//             DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)
  • Explicació:
    • SPACE=(CYL,(5,2)): Assigna 5 cilindres inicialment i 2 cilindres com a espai secundari.

Compressió de Datasets

  • Utilitats: Utilitza utilitats com IEBGENER i IDCAMS per a la compressió de datasets.
  • Beneficis: Redueix l'espai d'emmagatzematge i millora el rendiment.

Exemple: Compressió d'un Dataset

//COMPRESS JOB (ACCT),'COMPRESS DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD  SYSOUT=A
//SYSIN    DD  *
  REPRO INFILE(INPUT) OUTFILE(OUTPUT) COMPRESS
/*
  • Explicació:
    • REPRO INFILE(INPUT) OUTFILE(OUTPUT) COMPRESS: Copia i comprimeix el dataset.

Conclusió

La gestió eficient de datasets en JCL implica entendre els tipus de datasets, dominar les operacions bàsiques i emprar tècniques avançades com els GDGs, l'assignació optimitzada d'espai i la compressió de datasets. Seguint les millors pràctiques i utilitzant les utilitats adequades, pots assegurar-te que els teus datasets es gestionen de manera efectiva, conduint a un millor rendiment i utilització de recursos en els teus treballs JCL.
