Introducció

En Job Control Language (JCL), els datasets són crucials per gestionar i processar dades. Entendre com definir i manipular els paràmetres de dataset és essencial per a una execució eficient de les tasques. Aquesta secció cobrirà els conceptes bàsics dels paràmetres de dataset, els seus tipus i com utilitzar-los en JCL.

Conceptes Clau

  • Dataset: Una col·lecció de registres de dades emmagatzemats en un disc o cinta.
  • Paràmetres de Dataset: Atributs que defineixen les propietats i el comportament dels datasets en JCL.

Tipus de Paràmetres de Dataset

1. Nom del Dataset (DSN)

  • Definició: L'identificador únic per a un dataset.
  • Sintaxi: DSN=dataset.name
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR

2. Disposició (DISP)

  • Definició: Especifica l'estat del dataset i què fer amb ell després que el pas de la tasca es completi.
  • Sintaxi: DISP=(status,normal-termination,abnormal-termination)
  • Valors d'Estat:
    • NEW: Crear un nou dataset.
    • OLD: Utilitzar un dataset existent.
    • SHR: Compartir un dataset existent.
    • MOD: Modificar un dataset existent.
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE)

3. Assignació d'Espai (SPACE)

  • Definició: Defineix la quantitat d'espai a assignar per a un dataset.
  • Sintaxi: SPACE=(unit,(primary,secondary,dir),RLSE)
  • Paràmetres:
    • unit: La unitat d'assignació (per exemple, CYL, TRK, BLK).
    • primary: Assignació d'espai primari.
    • secondary: Assignació d'espai secundari.
    • dir: Nombre de blocs de directori (per a datasets particionats).
    • RLSE: Alliberar espai no utilitzat.
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE),
//             SPACE=(CYL,(10,5),RLSE)

4. Bloc de Control de Dades (DCB)

  • Definició: Especifica les característiques físiques del dataset.
  • Sintaxi: DCB=(parameter1,parameter2,...)
  • Paràmetres Comuns:
    • RECFM: Format de registre (per exemple, FB, VB).
    • LRECL: Longitud del registre lògic.
    • BLKSIZE: Mida del bloc.
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE),
//             DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)

5. Número de Sèrie del Volum (VOL)

  • Definició: Especifica el número de sèrie del volum on resideix el dataset.
  • Sintaxi: VOL=SER=volume
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR,
//             VOL=SER=123456

6. Unitat (UNIT)

  • Definició: Especifica el tipus de dispositiu o grup on s'emmagatzema el dataset.
  • Sintaxi: UNIT=device
  • Exemple: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR,
//             UNIT=SYSDA

Comparació de Paràmetres de Dataset

| Paràmetre | Descripció | Exemple de Sintaxi | |-----------|-------------|--------------------| | DSN | Nom del Dataset | DSN=MY.DATASET | | DISP | Disposició | DISP=(NEW,CATLG,DELETE) | | SPACE | Assignació d'Espai | SPACE=(CYL,(10,5),RLSE) | | DCB | Bloc de Control de Dades | DCB=(RECFM=FB,LRECL=80,BLKSIZE=800) | | VOL | Número de Sèrie del Volum | VOL=SER=123456 | | UNIT | Tipus d'Unitat | UNIT=SYSDA |

Conclusió

Entendre els paràmetres de dataset en JCL és fonamental per gestionar datasets de manera efectiva. Dominant aquests paràmetres, pots controlar com es creen, accedeixen i gestionen els datasets durant l'execució de les tasques. Practica utilitzant aquests paràmetres en diverses combinacions per convertir-te en un expert en la gestió de datasets en JCL.
