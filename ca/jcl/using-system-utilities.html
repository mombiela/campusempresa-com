Introducció

En aquesta secció, explorarem com utilitzar les utilitats del sistema en JCL (Job Control Language). Les utilitats del sistema són eines essencials que ajuden a gestionar i manipular conjunts de dades, realitzar manteniment del sistema i automatitzar diverses tasques. Entendre aquestes utilitats és crucial tant per a usuaris principiants com avançats de JCL.

Utilitats Clau del Sistema

IEBGENER

IEBGENER és una utilitat utilitzada per copiar conjunts de dades, crear còpies de seguretat i imprimir conjunts de dades.

  • Propòsit: Copiar dades d'un conjunt de dades a un altre.
  • Usos Comuns:
    • Copiar conjunts de dades seqüencials.
    • Crear còpies de seguretat de conjunts de dades.
    • Imprimir conjunts de dades.

Exemple

//COPYJOB  JOB (ACCT),'COPY DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEBGENER
//SYSPRINT DD  SYSOUT=A
//SYSUT1   DD  DSN=INPUT.DATASET,DISP=SHR
//SYSUT2   DD  DSN=OUTPUT.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=*.SYSUT1
//SYSIN    DD  DUMMY
  • Explicació:
    • SYSUT1 especifica el conjunt de dades d'entrada.
    • SYSUT2 especifica el conjunt de dades de sortida.
    • SYSIN està configurat a DUMMY perquè no es necessiten declaracions de control.

IEFBR14

IEFBR14 és un programa de utilitat simple que no fa res més que retornar el control al sistema operatiu. Sovint s'utilitza per assignar o eliminar conjunts de dades.

  • Propòsit: Assignar o eliminar conjunts de dades.
  • Usos Comuns:
    • Assignar nous conjunts de dades.
    • Eliminar conjunts de dades existents.

Exemple

//ALLOCJOB JOB (ACCT),'ALLOCATE DATASET',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IEFBR14
//NEWDS    DD  DSN=NEW.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)
  • Explicació:
    • NEWDS especifica el nou conjunt de dades a assignar.
    • DISP=(NEW,CATLG,DELETE) indica que el conjunt de dades s'ha de catalogar si el treball es completa amb èxit i eliminar si falla.

IDCAMS

IDCAMS (Integrated Data Cluster Access Method Services) és una utilitat versàtil utilitzada per gestionar conjunts de dades VSAM i realitzar tasques relacionades amb el catàleg.

  • Propòsit: Gestionar conjunts de dades VSAM i catàlegs.
  • Usos Comuns:
    • Definir conjunts de dades VSAM.
    • Eliminar conjunts de dades VSAM.
    • Llistar entrades del catàleg.

Exemple

//IDCAMSJOB JOB (ACCT),'IDCAMS EXAMPLE',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD  SYSOUT=A
//SYSIN    DD  *
  DELETE MY.VSAM.DATASET CLUSTER
  DEFINE CLUSTER (NAME(MY.VSAM.DATASET) -
                  INDEXED -
                  RECORDSIZE(80 80) -
                  KEYS(8 0) -
                  TRACKS(1 1) -
                  FREESPACE(3 3))
/*
  • Explicació:
    • La comanda DELETE elimina el conjunt de dades VSAM especificat.
    • La comanda DEFINE CLUSTER crea un nou conjunt de dades VSAM amb els atributs especificats.

SORT

SORT és una utilitat utilitzada per ordenar, fusionar i copiar conjunts de dades. És altament flexible i pot gestionar tasques complexes de manipulació de dades.

  • Propòsit: Ordenar, fusionar i copiar conjunts de dades.
  • Usos Comuns:
    • Ordenar conjunts de dades basats en camps especificats.
    • Fusionar múltiples conjunts de dades.
    • Filtrar i copiar conjunts de dades.

Exemple

//SORTJOB  JOB (ACCT),'SORT EXAMPLE',CLASS=A,MSGCLASS=A
//STEP1    EXEC PGM=SORT
//SORTIN   DD  DSN=INPUT.DATASET,DISP=SHR
//SORTOUT  DD  DSN=OUTPUT.DATASET,DISP=(NEW,CATLG,DELETE),
//             UNIT=SYSDA,SPACE=(TRK,(1,1)),DCB=*.SORTIN
//SYSIN    DD  *
  SORT FIELDS=(1,8,CH,A)
/*
  • Explicació:
    • SORTIN especifica el conjunt de dades d'entrada.
    • SORTOUT especifica el conjunt de dades de sortida.
    • SORT FIELDS=(1,8,CH,A) ordena el conjunt de dades basat en els primers 8 caràcters en ordre ascendent.

Conclusió

Entendre i utilitzar les utilitats del sistema en JCL és essencial per a una gestió eficient dels conjunts de dades i l'automatització de treballs. Dominant utilitats com IEBGENER, IEFBR14, IDCAMS i SORT, pots realitzar una àmplia gamma de tasques, des de la simple còpia de conjunts de dades fins a la manipulació complexa de dades. Practica utilitzant aquestes utilitats amb diferents conjunts de dades i escenaris per convertir-te en un expert en JCL.
