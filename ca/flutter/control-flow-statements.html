Les sentències de control de flux són fonamentals en qualsevol llenguatge de programació, incloent Dart, que s'utilitza a Flutter. Aquestes sentències et permeten controlar el flux d'execució del teu codi, fent possible realitzar diferents accions basades en condicions, repetir accions i gestionar excepcions.

Sentències condicionals

Les sentències condicionals et permeten executar certes parts del codi basant-se en si una condició és certa o falsa.

if i else

La sentència if s'utilitza per executar un bloc de codi si una condició és certa. La sentència else es pot utilitzar per executar un bloc de codi si la condició és falsa.

void main() {
  int number = 10;

  if (number > 0) {
    print('El número és positiu.');
  } else {
    print('El número no és positiu.');
  }
}

else if

La sentència else if et permet comprovar múltiples condicions.

void main() {
  int number = 0;

  if (number > 0) {
    print('El número és positiu.');
  } else if (number < 0) {
    print('El número és negatiu.');
  } else {
    print('El número és zero.');
  }
}

Sentència switch

La sentència switch et permet executar un bloc de codi entre molts basant-se en el valor d'una variable.

void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print('Excel·lent!');
      break;
    case 'B':
      print('Bé!');
      break;
    case 'C':
      print('Regular!');
      break;
    case 'D':
      print('Insuficient!');
      break;
    default:
      print('Nota invàlida.');
  }
}

Sentències de bucle

Les sentències de bucle et permeten executar un bloc de codi múltiples vegades.

Bucle for

El bucle for s'utilitza per executar un bloc de codi un nombre específic de vegades.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteració $i');
  }
}

Bucle while

El bucle while s'utilitza per executar un bloc de codi mentre una condició sigui certa.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteració $i');
    i++;
  }
}

Bucle do-while

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

void main() {
  int i = 0;

  do {
    print('Iteració $i');
    i++;
  } while (i < 5);
}

Gestió d'excepcions

La gestió d'excepcions et permet gestionar errors i altres esdeveniments excepcionals en el teu codi.

try i catch

El bloc try et permet provar un bloc de codi per detectar errors, mentre que el bloc catch et permet gestionar l'error.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('S'ha produït un error: $e');
  }
}

finally

El bloc finally et permet executar codi, independentment de si s'ha produït una excepció o no.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('S'ha produït un error: $e');
  } finally {
    print('Aquest codi s'executa passi el que passi.');
  }
}

Conclusió

Les sentències de control de flux són essencials per gestionar l'execució del teu codi. Comprenent i utilitzant aquestes sentències de manera efectiva, pots crear aplicacions Flutter més dinàmiques i responsives. Practica utilitzant aquestes sentències en diferents escenaris per convertir-te en un expert en el control del flux del teu codi Dart.
