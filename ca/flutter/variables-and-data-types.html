Introducció

En aquesta secció, cobrirem els conceptes fonamentals de les variables i els tipus de dades en Flutter. Entendre aquests conceptes bàsics és crucial per construir aplicacions Flutter robustes i eficients. Començarem amb les bases de les variables i després passarem als diferents tipus de dades disponibles en Dart, el llenguatge de programació utilitzat en Flutter.

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden referenciar i manipular en un programa. En Dart, les variables es declaren utilitzant les paraules clau var, final o const.

Declaració de Variables

  • var: El tipus de la variable és inferit pel compilador.
  • final: Una variable el valor de la qual no es pot canviar un cop establert.
  • const: Una constant de temps de compilació.
void main() {
  var name = 'Flutter'; // Tipus String inferit
  final age = 5; // Tipus int inferit, no es pot canviar
  const pi = 3.14; // Tipus double, constant de temps de compilació

  print(name); // Sortida: Flutter
  print(age); // Sortida: 5
  print(pi); // Sortida: 3.14
}

Tipus de Dades

Dart suporta diversos tipus de dades, que es poden categoritzar àmpliament en els següents:

  • Nombres
  • Cadenes
  • Booleans
  • Llistes
  • Mapes

Nombres

Dart té dos tipus de nombres: int i double.

void main() {
  int age = 30;
  double height = 5.9;

  print(age); // Sortida: 30
  print(height); // Sortida: 5.9
}

Cadenes

Les cadenes són una seqüència de caràcters. Es poden crear utilitzant cometes simples o dobles.

void main() {
  String greeting = 'Hola, Flutter!';
  String name = "John Doe";

  print(greeting); // Sortida: Hola, Flutter!
  print(name); // Sortida: John Doe
}

Booleans

Els booleans representen valors vertaders o falsos.

void main() {
  bool isFlutterAwesome = true;
  bool isLearningFun = true;

  print(isFlutterAwesome); // Sortida: true
  print(isLearningFun); // Sortida: true
}

Llistes

Les llistes són col·leccions ordenades d'objectes. Són similars a les matrius en altres llenguatges de programació.

void main() {
  List<String> fruits = ['Poma', 'Plàtan', 'Mango'];
  
  print(fruits); // Sortida: [Poma, Plàtan, Mango]
  print(fruits[0]); // Sortida: Poma
}

Mapes

Els mapes són col·leccions de parells clau-valor. Cada clau en un mapa ha de ser única.

void main() {
  Map<String, String> capitals = {
    'EUA': 'Washington, D.C.',
    'Índia': 'Nova Delhi',
    'Japó': 'Tòquio'
  };

  print(capitals); // Sortida: {EUA: Washington, D.C., Índia: Nova Delhi, Japó: Tòquio}
  print(capitals['Índia']); // Sortida: Nova Delhi
}

Resum

En aquesta secció, hem cobert les bases de les variables i els tipus de dades en Flutter. Hem après com declarar variables utilitzant var, final i const. També hem explorat diferents tipus de dades disponibles en Dart, incloent nombres, cadenes, booleans, llistes i mapes. Entendre aquests fonaments és essencial per construir qualsevol aplicació Flutter. En la propera secció, aprofundirem en temes més avançats i com utilitzar efectivament aquests tipus de dades en aplicacions del món real.
