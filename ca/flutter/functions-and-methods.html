Introducció

Les funcions i els mètodes són blocs fonamentals en qualsevol llenguatge de programació, i Flutter no és una excepció. Entendre com definir, utilitzar i manipular funcions i mètodes és crucial per construir aplicacions Flutter eficients i mantenibles.

Funcions en Dart

Les funcions en Dart són objectes de primera classe, el que significa que poden ser assignades a variables, passades com a arguments i retornades des d'altres funcions.

Definint Funcions

Una funció en Dart es defineix utilitzant la paraula clau void si no retorna un valor, o un tipus de retorn específic si ho fa.

// Funció que retorna una cadena
String greet(String name) {
  return 'Hello, $name!';
}

// Funció que no retorna un valor
void printGreeting(String name) {
  print('Hello, $name!');
}

Cridant Funcions

Per cridar una funció, simplement utilitzes el seu nom seguit de parèntesis.

void main() {
  String message = greet('Alice');
  print(message); // Output: Hello, Alice!

  printGreeting('Bob'); // Output: Hello, Bob!
}

Paràmetres Opcional

Dart suporta paràmetres opcionals, que poden ser posicionals o amb nom.

  • Paràmetres Opcional Posicionals: Enclosos entre claudàtors [].
String greet([String name = 'Guest']) {
  return 'Hello, $name!';
}

void main() {
  print(greet()); // Output: Hello, Guest!
  print(greet('Alice')); // Output: Hello, Alice!
}
  • Paràmetres Opcional amb Nom: Enclosos entre claus {} i poden ser marcats com a requerits utilitzant l'anotació @required.
String greet({String name = 'Guest'}) {
  return 'Hello, $name!';
}

void main() {
  print(greet()); // Output: Hello, Guest!
  print(greet(name: 'Alice')); // Output: Hello, Alice!
}

Mètodes en Dart

Els mètodes són funcions que estan associades amb un objecte. En Dart, els mètodes es defineixen dins de classes.

Definint Mètodes

Els mètodes es defineixen de manera similar a les funcions però dins del context d'una classe.

class Greeter {
  String name;

  Greeter(this.name);

  String greet() {
    return 'Hello, $name!';
  }

  void printGreeting() {
    print('Hello, $name!');
  }
}

Cridant Mètodes

Per cridar un mètode, necessites crear una instància de la classe i després utilitzar la notació de punt.

void main() {
  Greeter greeter = Greeter('Alice');
  print(greeter.greet()); // Output: Hello, Alice!
  greeter.printGreeting(); // Output: Hello, Alice!
}

Mètodes Estàtics

Els mètodes estàtics pertanyen a la classe en si mateixa en lloc d'una instància de la classe. Es defineixen utilitzant la paraula clau static.

class MathUtils {
  static int add(int a, int b) {
    return a + b;
  }
}

void main() {
  print(MathUtils.add(2, 3)); // Output: 5
}

Funcions d'Ordre Superior

Dart suporta funcions d'ordre superior, que són funcions que prenen altres funcions com a paràmetres o les retornen.

void printResult(int a, int b, int Function(int, int) operation) {
  print(operation(a, b));
}

int add(int a, int b) => a + b;
int multiply(int a, int b) => a * b;

void main() {
  printResult(2, 3, add); // Output: 5
  printResult(2, 3, multiply); // Output: 6
}

Conclusió

Les funcions i els mètodes són essencials per estructurar el teu codi i fer-lo reutilitzable i modular. En Flutter, entendre com definir i utilitzar funcions i mètodes et permetrà construir aplicacions més complexes i eficients. Tant si treballes amb funcions simples com amb mètodes més complexos dins de classes, dominar aquests conceptes és crucial per a qualsevol desenvolupador de Flutter.
