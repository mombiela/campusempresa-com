Gestió de l'Entrada d'Usuari
Conceptes Bàsics de Gestió d'Estat ►

Introducció

La gestió de l'entrada d'usuari és un aspecte fonamental de qualsevol aplicació interactiva. En Flutter, l'entrada d'usuari pot provenir de diverses fonts com ara camps de text, botons i gestos. Aquesta secció et guiarà des de les tècniques bàsiques fins a les avançades per gestionar l'entrada d'usuari en Flutter.

Entrada de Text

Widget TextField

El widget TextField és la manera principal d'obtenir entrada de text dels usuaris en Flutter.

  • Ús Bàsic:

    TextField(
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: 'Introdueix el teu nom',
  ),
)

  • Gestió de l'Entrada: Per gestionar l'entrada, pots utilitzar un TextEditingController.

    final TextEditingController _controller = TextEditingController();

TextField(
  controller: _controller,
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: 'Introdueix el teu nom',
  ),
)

// Accedir al valor de l'entrada
String inputValue = _controller.text;

Validació de Formularis

Per a formularis més complexos, Flutter proporciona el widget Form i els widgets FormField.

  • Exemple Bàsic de Formulari: 
    final _formKey = GlobalKey<FormState>();

Form(
  key: _formKey,
  child: Column(
    children: <Widget>[
      TextFormField(
        decoration: InputDecoration(labelText: 'Introdueix el teu nom'),
        validator: (value) {
          if (value == null || value.isEmpty) {
            return 'Si us plau, introdueix algun text';
          }
          return null;
        },
      ),
      ElevatedButton(
        onPressed: () {
          if (_formKey.currentState!.validate()) {
            // Processar dades
          }
        },
        child: Text('Enviar'),
      ),
    ],
  ),
)

Entrada de Botons

ElevatedButton

El widget ElevatedButton s'utilitza per crear un botó que pot desencadenar accions.

  • Ús Bàsic: 
    ElevatedButton(
  onPressed: () {
    print('Botó premut');
  },
  child: Text('Prem-me'),
)

Gestió dels Estats del Botó

Pots gestionar l'estat d'un botó per habilitar-lo o deshabilitar-lo segons certes condicions.

  • Exemple: 
    bool _isButtonDisabled = true;

ElevatedButton(
  onPressed: _isButtonDisabled ? null : () {
    print('Botó premut');
  },
  child: Text('Prem-me'),
)

Detecció de Gestos

GestureDetector

El widget GestureDetector et permet detectar diversos gestos com ara tocs, lliscaments i premses llargues.

  • Detecció Bàsica de Toc: 
    GestureDetector(
  onTap: () {
    print('Tocat');
  },
  child: Container(
    color: Colors.blue,
    width: 100,
    height: 100,
  ),
)

Gestos Avançats

Pots detectar gestos més complexos com el doble toc, la premsa llarga i l'arrossegament.

  • Exemple: 
    GestureDetector(
  onDoubleTap: () {
    print('Doble Tocat');
  },
  onLongPress: () {
    print('Premsa Llarga');
  },
  onPanUpdate: (details) {
    print('Arrossegat: ${details.delta}');
  },
  child: Container(
    color: Colors.blue,
    width: 100,
    height: 100,
  ),
)

Conclusió

La gestió de l'entrada d'usuari en Flutter és senzilla amb l'ús de widgets com TextField, ElevatedButton i GestureDetector. Entenent aquestes tècniques bàsiques i avançades, pots crear aplicacions interactives i responsives. Practica aquests conceptes amb diversos exemples per obtenir una sòlida comprensió de la gestió de l'entrada d'usuari en Flutter.
