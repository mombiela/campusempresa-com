Què és Dart?

Dart és un llenguatge de programació optimitzat per a clients per a aplicacions ràpides en qualsevol plataforma. És desenvolupat per Google i s'utilitza per construir aplicacions mòbils, d'escriptori, de servidor i web. Dart és el llenguatge darrere de Flutter, un marc popular per construir aplicacions mòbils multiplataforma.

Característiques Clau de Dart

  • Optimitzat per a UI: Dart està dissenyat per crear interfícies d'usuari suaus i responsives.
  • Desenvolupament Productiu: Funcions com la recàrrega en calent faciliten veure els canvis en temps real.
  • Rendiment Ràpid: Dart es compila a codi màquina ARM i x64 per a mòbils, escriptori i backend.
  • Eines Flexibles: Dart suporta una àmplia gamma d'eines i biblioteques.

Configuració de Dart

Abans de començar a codificar en Dart, necessites configurar el teu entorn de desenvolupament.

Instal·lant Dart

  1. Descarrega el SDK de Dart: Visita la pàgina del SDK de Dart i descarrega el SDK per al teu sistema operatiu.
  2. Instal·la Dart: Segueix les instruccions d'instal·lació per al teu sistema operatiu específic.
  3. Verifica la Instal·lació: Obre el teu terminal o símbol del sistema i escriu: 
    dart --version
    Hauries de veure la versió de Dart instal·lada a la teva màquina.

Sintaxi i Estructura Bàsica

Entendre la sintaxi i estructura bàsica de Dart és crucial per escriure codi efectiu.

Exemple de Hello World

Comencem amb un simple programa de "Hello, World!" per familiaritzar-nos amb la sintaxi de Dart.

void main() {
  print('Hello, World!');
}
  • void main(): El punt d'entrada de cada aplicació Dart.
  • print(): Una funció per mostrar text a la consola.

Variables i Tipus de Dades

Dart és un llenguatge tipat estàticament, el que significa que necessites especificar el tipus d'una variable quan la declares.

Declarant Variables

void main() {
  int age = 25;
  double height = 5.9;
  String name = 'John Doe';
  bool isStudent = true;
}
  • int: Valors enters.
  • double: Nombres de punt flotant.
  • String: Dades textuals.
  • bool: Valors booleans (true o false).

Sentències de Control de Flux

Les sentències de control de flux et permeten gestionar el flux del teu programa.

Sentència If-Else

void main() {
  int score = 85;
  if (score >= 90) {
    print('Grade: A');
  } else if (score >= 80) {
    print('Grade: B');
  } else {
    print('Grade: C');
  }
}

Bucle For

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteration $i');
  }
}

Funcions

Les funcions són una part central de Dart, permetent encapsular codi per a la seva reutilització.

Definint una Funció

void greet(String name) {
  print('Hello, $name!');
}

void main() {
  greet('Alice');
}
  • void: Indica que la funció no retorna cap valor.
  • greet(String name): Una funció anomenada greet que pren un paràmetre String.

Programació Orientada a Objectes en Dart

Dart és un llenguatge orientat a objectes, el que significa que suporta conceptes com classes i objectes.

Classes i Objectes

Definint una Classe

class Person {
  String name;
  int age;

  Person(this.name, this.age);

  void introduce() {
    print('Hi, I am $name and I am $age years old.');
  }
}

void main() {
  Person person = Person('Alice', 30);
  person.introduce();
}
  • class Person: Defineix una classe anomenada Person.
  • Person(this.name, this.age): Un constructor que inicialitza name i age.
  • introduce(): Un mètode que imprimeix un missatge.

Conclusió

En aquesta introducció a Dart, hem cobert els conceptes bàsics del llenguatge, incloent les seves característiques clau, com configurar el teu entorn, sintaxi bàsica, sentències de control de flux, funcions i conceptes de programació orientada a objectes. Entendre aquests fonaments proporcionarà una base sòlida a mesura que progressis en temes més avançats en el desenvolupament amb Dart i Flutter.
