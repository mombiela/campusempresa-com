Introducció

Configurar i planificar un projecte Flutter és crucial per assegurar un procés de desenvolupament fluid. Aquesta secció et guiarà a través dels passos inicials necessaris per configurar un projecte Flutter i planificar la seva estructura de manera efectiva.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Flutter SDK instal·lat
  • Dart SDK instal·lat
  • Un IDE (Entorn de Desenvolupament Integrat) com Android Studio, IntelliJ IDEA o Visual Studio Code
  • Coneixement bàsic del llenguatge de programació Dart

Configuració de l'Entorn Flutter

Instal·lació del Flutter SDK

  1. Descarrega el Flutter SDK: Visita el lloc web de Flutter i descarrega el SDK per al teu sistema operatiu.
  2. Extreu el SDK: Descomprimeix el fitxer descarregat i col·loca'l en una ubicació desitjada al teu sistema.
  3. Actualitza el Path: Afegeix el directori bin de Flutter al PATH del teu sistema.

Exemple (Windows):

set PATH=%PATH%;C:\path\to\flutter\bin

Verificació de la Instal·lació

Executa la següent comanda per verificar que Flutter està instal·lat correctament:

flutter doctor

Aquesta comanda comprova el teu entorn i mostra un informe de l'estat de la teva instal·lació de Flutter.

Configuració d'un IDE

  • Visual Studio Code: Instal·la els plugins de Flutter i Dart des del mercat d'Extensions.
  • Android Studio: Instal·la els plugins de Flutter i Dart des de les preferències de Plugins.

Creació d'un Nou Projecte Flutter

  1. Utilitzant la Línia de Comandes: 
    flutter create my_first_app
cd my_first_app
  2. Utilitzant l'IDE:
    • Obre el teu IDE.
    • Selecciona Create New Flutter Project.
    • Segueix les indicacions per configurar el teu projecte.

Estructura del Projecte

Una estructura típica de projecte Flutter es veu així:

my_first_app/
  android/
  ios/
  lib/
    main.dart
  test/
  pubspec.yaml

Directories i Fitxers Clau

  • android/ i ios/: Codi específic de la plataforma.
  • lib/: Conté el codi Dart per a la teva aplicació.
  • main.dart: Punt d'entrada de l'aplicació Flutter.
  • test/: Conté fitxers de prova.
  • pubspec.yaml: Gestiona les dependències i recursos del projecte.

Planificació del Teu Projecte Flutter

Definició dels Requisits del Projecte

  • Identificar Funcionalitats Clau: Llista les principals funcionalitats que la teva aplicació proporcionarà.
  • Històries d'Usuari: Escriu històries d'usuari per entendre les interaccions dels usuaris amb l'aplicació.
  • Wireframes i Mockups: Crea representacions visuals de la interfície d'usuari de la teva aplicació.

Elecció de la Gestió d'Estat

La gestió d'estat és crucial per gestionar l'estat de la teva aplicació. Algunes opcions populars inclouen:

  • Provider: Simple i eficient.
  • Bloc: Business Logic Component, per a una gestió d'estat més complexa.
  • Riverpod: Una alternativa moderna a Provider.

Exemple: Configuració de Provider

  1. Afegeix la Dependència: Actualitza pubspec.yaml per incloure el paquet provider. 
    dependencies:
  flutter:
    sdk: flutter
  provider: ^5.0.0
  2. Instal·la els Paquets: 
    flutter pub get
  3. Utilitzant Provider: 
    import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

void main() {
  runApp(
    ChangeNotifierProvider(
      create: (context) => Counter(),
      child: MyApp(),
    ),
  );
}

class Counter with ChangeNotifier {
  int _count = 0;
  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Provider Example')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              Text('You have pushed the button this many times:'),
              Consumer<Counter>(
                builder: (context, counter, child) {
                  return Text('${counter.count}', style: Theme.of(context).textTheme.headline4);
                },
              ),
            ],
          ),
        ),
        floatingActionButton: FloatingActionButton(
          onPressed: () => context.read<Counter>().increment(),
          tooltip: 'Increment',
          child: Icon(Icons.add),
        ),
      ),
    );
  }
}

Conclusió

Configurar i planificar un projecte Flutter implica diversos passos, des d'instal·lar les eines necessàries fins a definir l'estructura i els requisits del projecte. Una planificació adequada, incloent l'elecció de la solució de gestió d'estat adequada, és essencial per a una aplicació Flutter exitosa. Seguint aquestes directrius, pots assegurar un procés de desenvolupament fluid i crear una aplicació Flutter ben estructurada.
