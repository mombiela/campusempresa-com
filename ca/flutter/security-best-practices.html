Introducció

La seguretat és un aspecte crític de qualsevol aplicació, i Flutter no és una excepció. Aquesta secció et guiarà a través de les millors pràctiques per assegurar les teves aplicacions Flutter, des de nivells bàsics fins a avançats.

Pràctiques Bàsiques de Seguretat

Emmagatzematge Segur

  • Evita Emmagatzemar Dades Sensibles en Text Pla: Utilitza solucions d'emmagatzematge segur per emmagatzemar dades sensibles com claus API, tokens i contrasenyes.
  • Utilitza Flutter Secure Storage: Aquest paquet proporciona una manera segura d'emmagatzemar parells clau-valor.
import 'package:flutter_secure_storage/flutter_secure_storage.dart';

final storage = new FlutterSecureStorage();

// Escriure valor
await storage.write(key: 'api_token', value: 'your_api_token');

// Llegir valor
String value = await storage.read(key: 'api_token');

Validació d'Entrades

  • Sanititzar les Entrades de l'Usuari: Sempre valida i sanititza les entrades de l'usuari per prevenir atacs d'injecció.
  • Utilitza la Validació de Formularis: Flutter proporciona mecanismes de validació de formularis per assegurar que les dades d'entrada siguin correctes.
final _formKey = GlobalKey<FormState>();

Form(
  key: _formKey,
  child: Column(
    children: <Widget>[
      TextFormField(
        validator: (value) {
          if (value.isEmpty) {
            return 'Si us plau, introdueix algun text';
          }
          return null;
        },
      ),
      RaisedButton(
        onPressed: () {
          if (_formKey.currentState.validate()) {
            // Processar dades
          }
        },
        child: Text('Enviar'),
      ),
    ],
  ),
);

Pràctiques Intermèdies de Seguretat

Seguretat de la Xarxa

  • Utilitza HTTPS: Sempre utilitza HTTPS per encriptar les dades transmeses per la xarxa.
  • Pinning de Certificats: Assegura't que el certificat del servidor sigui vàlid i de confiança.
import 'dart:io';

class MyHttpOverrides extends HttpOverrides {
  @override
  HttpClient createHttpClient(SecurityContext context) {
    return super.createHttpClient(context)
      ..badCertificateCallback = (X509Certificate cert, String host, int port) => true;
  }
}

void main() {
  HttpOverrides.global = new MyHttpOverrides();
  runApp(MyApp());
}

Autenticació i Autorització

  • Utilitza OAuth2: Implementa OAuth2 per a una autenticació segura.
  • Autenticació de Firebase: Utilitza l'Autenticació de Firebase per a un sistema d'autenticació robust i segur.
import 'package:firebase_auth/firebase_auth.dart';

final FirebaseAuth _auth = FirebaseAuth.instance;

Future<void> signInWithEmailAndPassword(String email, String password) async {
  try {
    UserCredential userCredential = await _auth.signInWithEmailAndPassword(
      email: email,
      password: password,
    );
    // Inici de sessió amb èxit
  } on FirebaseAuthException catch (e) {
    if (e.code == 'user-not-found') {
      print('No s\'ha trobat cap usuari amb aquest correu electrònic.');
    } else if (e.code == 'wrong-password') {
      print('Contrasenya incorrecta per a aquest usuari.');
    }
  }
}

Pràctiques Avançades de Seguretat

Ofuscació de Codi

  • Ofusca el Codi Dart: Utilitza l'ofuscació per dificultar als atacants la tasca de revertir l'enginyeria de la teva aplicació.
flutter build apk --obfuscate --split-debug-info=/<project-name>/debug-info

Crides API Segures

  • Utilitza Autenticació Basada en Tokens: Assegura les teves crides API amb mecanismes d'autenticació basats en tokens com JWT.
  • Tokens de Refresc: Implementa mecanismes de refresc de tokens per mantenir sessions segures.
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(
    'https://example.com/data',
    headers: {
      'Authorization': 'Bearer your_jwt_token',
    },
  );

  if (response.statusCode == 200) {
    // Processar dades
  } else {
    // Gestionar error
  }
}

Encriptació de Dades

  • Encripta Dades Sensibles: Utilitza biblioteques d'encriptació per encriptar dades sensibles abans d'emmagatzemar-les o transmetre-les.
import 'package:encrypt/encrypt.dart';

final key = Key.fromUtf8('my 32 length key................');
final iv = IV.fromLength(16);

final encrypter = Encrypter(AES(key));

final encrypted = encrypter.encrypt('Sensitive data', iv: iv);
final decrypted = encrypter.decrypt(encrypted, iv: iv);

print(decrypted); // Sensitive data

Conclusió

Assegurar la teva aplicació Flutter és un procés continu que implica múltiples capes de protecció. Seguint aquestes millors pràctiques, pots millorar significativament la seguretat de la teva aplicació. Mantén-te sempre actualitzat amb les últimes tendències en seguretat i audita contínuament el teu codi per detectar possibles vulnerabilitats.
