Introducció

L'emmagatzematge de dades locals és un aspecte crucial del desenvolupament d'aplicacions mòbils. Permet que les aplicacions emmagatzemin dades al dispositiu, assegurant que els usuaris puguin accedir a la seva informació fins i tot sense connexió a Internet. En Flutter, hi ha diverses maneres de gestionar l'emmagatzematge de dades locals, cadascuna amb els seus propis casos d'ús i beneficis.

Conceptes Clau

  • Persistència: Assegurar que les dades romanguin disponibles entre sessions de l'aplicació.
  • Formats de Dades: Entendre diferents formats com JSON, SQLite i parells clau-valor.
  • Seguretat: Protegir dades sensibles emmagatzemades localment.

Mètodes d'Emmagatzematge de Dades Locals en Flutter

1. Shared Preferences

Shared Preferences és una manera senzilla d'emmagatzemar petites quantitats de dades com a parells clau-valor. És ideal per emmagatzemar configuracions d'usuari o informació d'estat simple.

Conceptes Clau

  • Emmagatzematge Clau-Valor: Les dades s'emmagatzemen com a parells de claus i valors.
  • Tipus de Dades Primitives: Només suporta tipus de dades bàsics com cadenes, enters i booleans.

Exemple

import 'package:shared_preferences/shared_preferences.dart';

void saveData() async {
  final prefs = await SharedPreferences.getInstance();
  await prefs.setString('username', 'JohnDoe');
}

void loadData() async {
  final prefs = await SharedPreferences.getInstance();
  String? username = prefs.getString('username');
  print(username); // Output: JohnDoe
}

2. SQLite

SQLite és un motor de base de dades potent que permet emmagatzemar estructures de dades complexes. És adequat per a aplicacions que requereixen una base de dades relacional.

Conceptes Clau

  • Base de Dades Relacional: Les dades s'emmagatzemen en taules amb files i columnes.
  • Consultes SQL: Les dades es manipulen utilitzant comandes SQL.

Exemple

import 'package:sqflite/sqflite.dart';
import 'package:path/path.dart';

Future<Database> initializeDb() async {
  String path = join(await getDatabasesPath(), 'my_database.db');
  return openDatabase(
    path,
    onCreate: (db, version) {
      return db.execute(
        "CREATE TABLE users(id INTEGER PRIMARY KEY, name TEXT, age INTEGER)",
      );
    },
    version: 1,
  );
}

Future<void> insertUser(Database db, User user) async {
  await db.insert(
    'users',
    user.toMap(),
    conflictAlgorithm: ConflictAlgorithm.replace,
  );
}

class User {
  final int id;
  final String name;
  final int age;

  User({required this.id, required this.name, required this.age});

  Map<String, dynamic> toMap() {
    return {
      'id': id,
      'name': name,
      'age': age,
    };
  }
}

3. Hive

Hive és una base de dades clau-valor lleugera i ràpida escrita en Dart pur. És adequada per emmagatzemar grans quantitats de dades amb alt rendiment.

Conceptes Clau

  • Base de Dades NoSQL: Les dades s'emmagatzemen en caixes, que són similars a les taules en SQL.
  • Emmagatzematge de Dades Binàries: Les dades s'emmagatzemen en un format binari per a operacions de lectura/escriptura ràpides.

Exemple

import 'package:hive/hive.dart';

void main() async {
  await Hive.initFlutter();
  var box = await Hive.openBox('myBox');

  // Emmagatzemar dades
  await box.put('name', 'John Doe');

  // Recuperar dades
  var name = box.get('name');
  print(name); // Output: John Doe
}

Taula Comparativa

| Mètode | Suport de Tipus de Dades | Complexitat | Cas d'Ús | |--------------------|--------------------------|-------------|------------------------------------| | Shared Preferences | Tipus de Dades Primitives| Baixa | Configuracions d'usuari, dades simples | | SQLite | Estructures de Dades Complexes | Alta | Dades relacionals, consultes complexes | | Hive | Parells Clau-Valor | Mitjana | Grans conjunts de dades, alt rendiment |

Conclusió

L'emmagatzematge de dades locals és una part essencial del desenvolupament d'aplicacions Flutter. Depenent dels requisits de la teva aplicació, pots triar entre Shared Preferences per a emmagatzematge simple de clau-valor, SQLite per a bases de dades relacionals, o Hive per a emmagatzematge de clau-valor d'alt rendiment. Entendre aquests mètodes i els seus casos d'ús t'ajudarà a prendre decisions informades sobre com gestionar les dades en les teves aplicacions Flutter.
