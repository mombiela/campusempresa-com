Introducció a Flutter

Flutter és un kit de desenvolupament de programari d'interfície d'usuari de codi obert creat per Google. S'utilitza per desenvolupar aplicacions per a Android, iOS, Linux, Mac, Windows, Google Fuchsia i la web des d'una única base de codi. Flutter és conegut pel seu desenvolupament ràpid, interfície d'usuari expressiva i flexible, i rendiment natiu.

Conceptes Clau de Flutter

  • Codi Obert: Flutter és gratuït i té una gran comunitat de desenvolupadors que contribueixen al seu creixement.
  • Única Base de Codi: Escriu una vegada, executa en qualsevol lloc. Flutter et permet crear aplicacions per a múltiples plataformes utilitzant una única base de codi.
  • Widgets: Tot a Flutter és un widget. Els widgets són els blocs de construcció de la interfície d'usuari d'una aplicació Flutter.
  • Hot Reload: La funció de hot reload de Flutter et permet veure els canvis que fas en el teu codi gairebé instantàniament sense reiniciar l'aplicació.
  • Llenguatge Dart: Les aplicacions Flutter estan escrites en Dart, un llenguatge desenvolupat per Google que és fàcil d'aprendre i optimitzat per al desenvolupament d'interfícies d'usuari.

Per què Escollir Flutter?

  • Desenvolupament Ràpid: Amb funcions com el hot reload, els desenvolupadors poden iterar ràpidament en el seu codi.
  • Interfície d'Usuari Expressiva i Flexible: Flutter proporciona un conjunt ric de widgets personalitzables per construir interfícies natives.
  • Rendiment Natiu: Flutter es compila en codi ARM natiu tant per a iOS com per a Android, assegurant un alt rendiment en ambdues plataformes.
  • Comunitat i Suport Forts: Estar recolzat per Google i tenir una comunitat forta assegura una millora contínua i suport.

Arquitectura de Flutter

L'arquitectura de Flutter està dissenyada per ser en capes i composable. Aquí tens una vista simplificada de la seva arquitectura:

| Capa | Descripció | |----------------|-----------------------------------------------------------------------------| | Framework | La capa superior, escrita en Dart, inclou widgets, renderització, animació, etc.| | Motor | Escrit en C++, proporciona suport de renderització de baix nivell utilitzant Skia. | | Embedder | Codi específic de la plataforma que permet que Flutter s'executi en diferents plataformes OS.|

Exemple Bàsic de Codi Flutter

Aquí tens una aplicació Flutter simple que mostra "Hola, Món!" a la pantalla:

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Benvingut a Flutter'),
        ),
        body: Center(
          child: Text('Hola, Món!'),
        ),
      ),
    );
  }
}

Explicació

  • import 'package:flutter/material.dart';: Importa la biblioteca de disseny material de Flutter.
  • void main() { runApp(MyApp()); }: La funció principal és el punt d'entrada de l'aplicació. runApp inicia l'aplicació Flutter.
  • MyApp: Un widget sense estat que representa l'aplicació.
  • MaterialApp: Un widget de conveniència que embolcalla diversos widgets comunament requerits per a aplicacions de disseny material.
  • Scaffold: Implementa l'estructura visual bàsica del disseny material.
  • AppBar: Una barra d'aplicació de disseny material.
  • Center: Centra el seu widget fill.
  • Text: Un widget que mostra una cadena de text.

Conclusió

Flutter és un potent kit d'eines per construir aplicacions multiplataforma amb una única base de codi. El seu ric conjunt de widgets, cicle de desenvolupament ràpid i rendiment natiu el fan una excel·lent elecció per al desenvolupament d'aplicacions modernes. Entenent els conceptes clau i l'arquitectura de Flutter, pots començar a construir aplicacions belles i de gran rendiment per a múltiples plataformes.
