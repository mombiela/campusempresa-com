Introducció

En aquesta secció, explorarem com treballar amb llistes i graelles a Flutter. Les llistes i les graelles són components essencials per mostrar col·leccions de dades de manera estructurada i visualment atractiva. Al final d'aquesta secció, seràs capaç de crear llistes i graelles dinàmiques, personalitzar-ne l'aparença i gestionar les interaccions dels usuaris.

Llistes a Flutter

ListView Bàsic

El widget ListView és el widget més utilitzat per mostrar una llista d'elements a Flutter. Es pot utilitzar per crear llistes tant estàtiques com dinàmiques.

ListView Estàtic

Un ListView estàtic és útil quan tens un nombre fix d'elements per mostrar.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('ListView Estàtic')),
        body: ListView(
          children: <Widget>[
            ListTile(
              leading: Icon(Icons.map),
              title: Text('Mapa'),
            ),
            ListTile(
              leading: Icon(Icons.photo_album),
              title: Text('Àlbum'),
            ),
            ListTile(
              leading: Icon(Icons.phone),
              title: Text('Telèfon'),
            ),
          ],
        ),
      ),
    );
  }
}

ListView Dinàmic

Un ListView dinàmic és útil quan el nombre d'elements no és fix i pot canviar durant l'execució.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Element $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('ListView Dinàmic')),
        body: ListView.builder(
          itemCount: items.length,
          itemBuilder: (context, index) {
            return ListTile(
              title: Text('${items[index]}'),
            );
          },
        ),
      ),
    );
  }
}

Graelles a Flutter

GridView Bàsic

El widget GridView s'utilitza per mostrar elements en un disseny de graella. Es pot utilitzar per crear graelles tant estàtiques com dinàmiques.

GridView Estàtic

Un GridView estàtic és útil quan tens un nombre fix d'elements per mostrar.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('GridView Estàtic')),
        body: GridView.count(
          crossAxisCount: 2,
          children: <Widget>[
            Container(
              color: Colors.red,
              child: Center(child: Text('Element 1')),
            ),
            Container(
              color: Colors.blue,
              child: Center(child: Text('Element 2')),
            ),
            Container(
              color: Colors.green,
              child: Center(child: Text('Element 3')),
            ),
            Container(
              color: Colors.yellow,
              child: Center(child: Text('Element 4')),
            ),
          ],
        ),
      ),
    );
  }
}

GridView Dinàmic

Un GridView dinàmic és útil quan el nombre d'elements no és fix i pot canviar durant l'execució.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Element $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('GridView Dinàmic')),
        body: GridView.builder(
          gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
            crossAxisCount: 2,
          ),
          itemCount: items.length,
          itemBuilder: (context, index) {
            return Card(
              child: Center(child: Text('${items[index]}')),
            );
          },
        ),
      ),
    );
  }
}

Personalitzant Elements de la Graella

Pots personalitzar l'aparença dels elements de la graella utilitzant diversos widgets i propietats.

Card(
  child: Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: <Widget>[
      Icon(Icons.map, size: 50),

  ),
