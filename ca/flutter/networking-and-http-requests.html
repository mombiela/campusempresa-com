Introducció

En aquesta secció, explorarem com gestionar les xarxes i les sol·licituds HTTP a Flutter. Això és crucial per construir aplicacions que necessiten comunicar-se amb serveis web o APIs. Cobrirem els conceptes bàsics de fer sol·licituds HTTP, gestionar respostes i analitzar dades JSON. Al final d'aquesta secció, seràs capaç d'integrar trucades de xarxa a les teves aplicacions Flutter.

Conceptes Clau

  • Mètodes HTTP: GET, POST, PUT, DELETE
  • Programació Asíncrona: Futures, async, await
  • Anàlisi de JSON: Convertir dades JSON a objectes Dart i viceversa
  • Gestió d'Errors: Gestionar errors de xarxa i excepcions

Fer Sol·licituds HTTP

Utilitzant el paquet `http`

Flutter proporciona el paquet http per gestionar sol·licituds HTTP. Primer, afegeix el paquet http al teu fitxer pubspec.yaml:

dependencies:
  http: ^0.13.3

Després, importa el paquet al teu fitxer Dart:

import 'package:http/http.dart' as http;

Sol·licitud GET

Una sol·licitud GET s'utilitza per recuperar dades d'un servidor. Aquí tens un exemple de com fer una sol·licitud GET:

import 'dart:convert';
import 'package:http/http.dart' as http;

Future<void> fetchData() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

  if (response.statusCode == 200) {
    List<dynamic> data = jsonDecode(response.body);
    print(data);
  } else {
    throw Exception('No s\'han pogut carregar les dades');
  }
}

Sol·licitud POST

Una sol·licitud POST s'utilitza per enviar dades a un servidor. Aquí tens un exemple de com fer una sol·licitud POST:

import 'dart:convert';
import 'package:http/http.dart' as http;

Future<void> sendData() async {
  final response = await http.post(
    Uri.parse('https://jsonplaceholder.typicode.com/posts'),
    headers: <String, String>{
      'Content-Type': 'application/json; charset=UTF-8',
    },
    body: jsonEncode(<String, String>{
      'title': 'foo',
      'body': 'bar',
      'userId': '1',
    }),
  );

  if (response.statusCode == 201) {
    print('Dades enviades correctament');
  } else {
    throw Exception('No s\'han pogut enviar les dades');
  }
}

Anàlisi de Dades JSON

Quan reps dades d'un servidor, sovint estan en format JSON. Necessites analitzar aquestes dades JSON en objectes Dart.

Exemple de Dades JSON

[
  {
    "userId": 1,
    "id": 1,
    "title": "Sample Title",
    "body": "Sample Body"
  }
]

Analitzar JSON en Dart

Primer, crea una classe Dart per representar les dades:

class Post {
  final int userId;
  final int id;
  final String title;
  final String body;

  Post({required this.userId, required this.id, required this.title, required this.body});

  factory Post.fromJson(Map<String, dynamic> json) {
    return Post(
      userId: json['userId'],
      id: json['id'],
      title: json['title'],
      body: json['body'],
    );
  }
}

Després, analitza les dades JSON:

Future<List<Post>> fetchPosts() async {
  final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

  if (response.statusCode == 200) {
    List<dynamic> jsonData = jsonDecode(response.body);
    return jsonData.map((json) => Post.fromJson(json)).toList();
  } else {
    throw Exception('No s\'han pogut carregar les publicacions');
  }
}

Gestió d'Errors

Gestionar errors és crucial per a una experiència d'usuari suau. Aquí tens un exemple de com gestionar errors en les teves sol·licituds de xarxa:

Future<void> fetchDataWithErrorHandling() async {
  try {
    final response = await http.get(Uri.parse('https://jsonplaceholder.typicode.com/posts'));

    si (response.statusCode == 200) {
      List<dynamic> data = jsonDecode(response.body);
      print(data);
    } else {
      print('Error del servidor: ${response.statusCode}');
    }
  } catch (e) {
    print('Error de xarxa: $e');
  }
}
