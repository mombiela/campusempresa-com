Introducció

Benvingut a la teva primera aplicació Flutter! En aquesta secció, et guiarem a través del procés de configuració del teu entorn de desenvolupament, creació d'un nou projecte Flutter i construcció d'una aplicació senzilla. Al final d'aquesta secció, tindràs una comprensió bàsica de com funciona Flutter i estaràs preparat per aprofundir en temes més avançats.

Configuració del teu entorn de desenvolupament

Abans de començar a construir la nostra aplicació, necessitem configurar el nostre entorn de desenvolupament.

  1. Instal·lar Flutter SDK:

    • Descarrega el Flutter SDK des del lloc web oficial de Flutter.
    • Extreu el fitxer descarregat i afegeix el directori bin de Flutter al PATH del teu sistema.

  2. Instal·lar un IDE:

    • Flutter suporta múltiples IDEs, però els més populars són Visual Studio Code i Android Studio.
    • Instal·la qualsevol d'aquests IDEs i afegeix els plugins de Flutter i Dart.

  3. Configurar un emulador o dispositiu físic:

    • Per a Android, configura un emulador d'Android mitjançant Android Studio.
    • Per a iOS, utilitza el simulador d'iOS disponible a Xcode.
    • Alternativament, pots connectar un dispositiu físic via USB i habilitar el mode desenvolupador.

  4. Verificar la instal·lació:

    • Obre un terminal i executa flutter doctor per comprovar si tot està configurat correctament.

Creació d'un nou projecte Flutter

Creem un nou projecte Flutter per començar.

  1. Crear un nou projecte:

    • Obre el teu terminal i executa: 
      flutter create my_first_app
    • Navega al directori del projecte: 
      cd my_first_app

  2. Obrir el projecte al teu IDE:

    • Obre el projecte acabat de crear al teu IDE preferit.

Comprensió de l'estructura del projecte

Aquí tens una breu visió general dels fitxers i directoris clau en un projecte Flutter:

  • lib/main.dart: El punt d'entrada principal per a l'aplicació.
  • pubspec.yaml: Fitxer de configuració per a dependències i recursos.
  • android/, ios/: Codi i configuracions específiques de la plataforma.
  • test/: Directori per a proves unitàries i de widgets.

Construcció de la teva primera aplicació Flutter

Construïm una aplicació senzilla de "Hola, món!".

  1. Editar lib/main.dart:

    • Substitueix el contingut de lib/main.dart pel següent codi: 
      import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('My First Flutter App'),
        ),
        body: Center(
          child: Text('Hello, World!'),
        ),
      ),
    );
  }
}

  2. Executar l'aplicació:

    • Executa l'aplicació utilitzant la següent comanda al terminal: 
      flutter run
    • Alternativament, pots utilitzar la funció d'execució/depuració al teu IDE.

Explicació del codi

  • Funció main(): El punt d'entrada de l'aplicació. Crida runApp() per iniciar l'aplicació.
  • Classe MyApp: Un widget sense estat que construeix la interfície d'usuari de l'aplicació.
  • MaterialApp: Un widget que embolcalla diversos widgets comunament requerits per a aplicacions de Material Design.
  • Scaffold: Proporciona una estructura bàsica per a l'aplicació, incloent una barra d'aplicació i un cos.
  • AppBar: Una barra d'aplicació de disseny material.
  • Center: Un widget que centra el seu fill dins d'ell mateix.
  • Text: Un widget que mostra una cadena de text.
