Introducció a Firebase i Flutter

  • Firebase: Una plataforma desenvolupada per Google per crear aplicacions mòbils i web. Proporciona una varietat d'eines i serveis per ajudar els desenvolupadors a construir aplicacions d'alta qualitat, incloent autenticació, bases de dades, analítiques i més.
  • Flutter: Un kit de desenvolupament de programari d'interfície d'usuari de codi obert creat per Google. S'utilitza per desenvolupar aplicacions per a Android, iOS, Linux, Mac, Windows, Google Fuchsia i la web des d'una única base de codi.

Configurant Firebase amb Flutter

Prerequisits

  • Coneixements bàsics de Flutter i Dart.
  • Un compte de Google per accedir als serveis de Firebase.
  • SDK de Flutter instal·lat a la teva màquina.

Creant un Projecte Firebase

  1. Ves a la Consola de Firebase.
  2. Fes clic a "Afegeix projecte" i segueix les instruccions per crear un nou projecte.
  3. Un cop creat el projecte, afegeix una aplicació Android i/o iOS al projecte.

Configurant Firebase a Flutter

Configuració d'Android

  1. Descarrega el fitxer google-services.json des de la Consola de Firebase i col·loca'l al directori android/app.
  2. Modifica el fitxer android/build.gradle: 
    buildscript {
    dependencies {
        // Afegeix aquesta línia
        classpath 'com.google.gms:google-services:4.3.3'
    }
}
  3. Modifica el fitxer android/app/build.gradle: 
    apply plugin: 'com.android.application'
apply plugin: 'com.google.gms.google-services' // Afegeix aquesta línia

Configuració d'iOS

  1. Descarrega el fitxer GoogleService-Info.plist des de la Consola de Firebase i col·loca'l al directori ios/Runner.
  2. Obre ios/Runner.xcworkspace a Xcode.
  3. Afegeix el fitxer GoogleService-Info.plist al projecte a Xcode.

Afegint Dependències de Firebase

Afegeix les dependències necessàries de Firebase al teu fitxer pubspec.yaml:

dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^1.10.0
  firebase_auth: ^3.3.3
  cloud_firestore: ^3.1.5

Executa flutter pub get per instal·lar les dependències.

Autenticació de Firebase

Configurant l'Autenticació

  1. A la Consola de Firebase, navega a la secció "Autenticació".
  2. Activa els mètodes d'autenticació desitjats (per exemple, Correu electrònic/Contrasenya, Inici de sessió amb Google).

Implementant l'Autenticació a Flutter

Exemple d'Autenticació amb Correu electrònic/Contrasenya

import 'package:firebase_auth/firebase_auth.dart';

class AuthService {
  final FirebaseAuth _auth = FirebaseAuth.instance;

  // Iniciar sessió amb correu electrònic i contrasenya
  Future<User?> signInWithEmailAndPassword(String email, String password) async {
    try {
      UserCredential result = await _auth.signInWithEmailAndPassword(email: email, password: password);
      User? user = result.user;
      return user;
    } catch (e) {
      print(e.toString());
      return null;
    }
  }

  // Registrar-se amb correu electrònic i contrasenya
  Future<User?> registerWithEmailAndPassword(String email, String password) async {
    try {
      UserCredential result = await _auth.createUserWithEmailAndPassword(email: email, password: password);
      User? user = result.user;
      return user;
    } catch (e) {
      print(e.toString());
      return null;
    }
  }

  // Tancar sessió
  Future<void> signOut() async {
    try {
      return await _auth.signOut();
    } catch (e) {
      print(e.toString());
      return null;
    }
  }
}

Cloud Firestore

Configurant Firestore

  1. A la Consola de Firebase, navega a la secció "Firestore Database".
  2. Fes clic a "Crear base de dades" i segueix les instruccions per configurar Firestore.

Implementant Firestore a Flutter

Afegint Dades a Firestore

import 'package:cloud_firestore/cloud_firestore.dart';

class DatabaseService {
  final CollectionReference userCollection = FirebaseFirestore.instance.collection('users');

  Future<void> addUser(String uid, String name, String email) async {
    return await userCollection.doc(uid).set({
      'name': name,
      'email': email,
    });
  }
}

Recuperant Dades de Firestore

import 'package:cloud_firestore/cloud_firestore.dart';

class DatabaseService {
  final CollectionReference userCollection = FirebaseFirestore.instance.collection('users');

  // Obtenir dades de l'usuari
  Future<DocumentSnapshot> getUserData(String uid) async {
    return await userCollection.doc(uid).get();
  }

  // Flux de dades de l'usuari
  Stream<DocumentSnapshot> getUserStream(String uid) {
    return userCollection.doc(uid).snapshots();
  }
}

Emmagatzematge de Firebase

Configurant l'Emmagatzematge de Firebase

  1. A la Consola de Firebase, navega a la secció "Emmagatzematge".
  2. Fes clic a "Comença" i segueix les instruccions per configurar l'Emmagatzematge de Firebase.

Implementant l'Emmagatzematge de Firebase a Flutter

Pujant Fitxers a l'Emmagatzematge de Firebase

import 'package:firebase_storage/firebase_storage.dart';
import 'dart:io';

class StorageService {
  final FirebaseStorage _storage = FirebaseStorage.instance;

  Future<String> uploadFile(File file, String path) async {
    try {
      TaskSnapshot snapshot = await _storage.ref().child(path).putFile(file);
      String downloadUrl = await snapshot.ref.getDownloadURL();
      return downloadUrl;
    } catch (e) {
      print(e.toString());
      return '';
    }
  }
}

Descarregant Fitxers de l'Emmagatzematge de Firebase

import 'package:firebase_storage/firebase_storage.dart';

class StorageService {
  final FirebaseStorage _storage = FirebaseStorage.instance;

  Future<String> getFileDownloadUrl(String path) async {
    try {
      String downloadUrl = await _storage.ref().child(path).getDownloadURL();
      return downloadUrl;
    } catch (e) {
      print(e.toString());
      return '';
    }
  }
}
