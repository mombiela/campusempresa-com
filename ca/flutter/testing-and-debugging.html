Introducció

Les proves i la depuració són aspectes crucials del desenvolupament de programari, assegurant que les teves aplicacions Flutter siguin fiables, eficients i lliures d'errors. Aquesta secció et guiarà a través dels conceptes bàsics de les proves i la depuració en Flutter, progressant cap a tècniques més avançades.

Proves Unitàries

Les proves unitàries se centren en provar components individuals de la teva aplicació de manera aïllada.

  • Conceptes Clau:

    • Desenvolupament Guiat per Proves (TDD): Escriure proves abans del codi.
    • Afirmacions: Declaracions que comproven si una condició és certa.
    • Simulació: Crear objectes falsos per simular els reals.

  • Exemple:

    import 'package:flutter_test/flutter_test.dart';

void main() {
  test('Prova de divisió de cadena', () {
    var string = 'Hola Flutter';
    var result = string.split(' ');

    expect(result, ['Hola', 'Flutter']);
  });
}

Proves de Widgets

Les proves de widgets verifiquen el comportament dels widgets individuals i les seves interaccions.

  • Conceptes Clau:

    • WidgetTester: Una utilitat per interactuar amb widgets en proves.
    • Finder: Localitza widgets en l'arbre de widgets.
    • Pump: Reconstrueix l'arbre de widgets.

  • Exemple:

    import 'package:flutter/material.dart';
import 'package:flutter_test/flutter_test.dart';

void main() {
  testWidgets('Prova de fum d'increment del comptador', (WidgetTester tester) async {
    await tester.pumpWidget(MyApp());

    expect(find.text('0'), findsOneWidget);
    expect(find.text('1'), findsNothing);

    await tester.tap(find.byIcon(Icons.add));
    await tester.pump();

    expect(find.text('0'), findsNothing);
    expect(find.text('1'), findsOneWidget);
  });
}

Proves d'Integració

Les proves d'integració asseguren que totes les parts de l'aplicació funcionin juntes com s'espera.

  • Conceptes Clau:

    • Driver: Automatitza les interaccions amb l'aplicació.
    • Servidor de Proves: Executa l'aplicació i escolta comandes.
    • Proves de Rendiment: Mesura el rendiment de l'aplicació sota diverses condicions.

  • Exemple:

    import 'package:flutter_driver/flutter_driver.dart';
import 'package:test/test.dart';

void main() {
  group('Aplicació de Comptador', () {
    FlutterDriver driver;

    setUpAll(() async {
      driver = await FlutterDriver.connect();
    });

    tearDownAll(() async {
      if (driver != null) {
        driver.close();
      }
    });

    test('comença a 0', () async {
      expect(await driver.getText(find.byValueKey('counter')), "0");
    });

    test('incrementa el comptador', () async {
      await driver.tap(find.byValueKey('increment'));
      expect(await driver.getText(find.byValueKey('counter')), "1");
    });
  });
}

Tècniques de Depuració

Una depuració efectiva ajuda a identificar i solucionar problemes ràpidament.

  • Conceptes Clau:

    • Punts de Trencament: Pausa l'execució en una línia específica.
    • Registre: Imprimeix missatges a la consola.
    • Inspector: Visualitza i interactua amb l'arbre de widgets.

  • Exemple:

    void main() {
  debugPrint('Aplicació iniciada');
  runApp(MyApp());
}

Eines Comunes de Depuració

  • Flutter DevTools: Un conjunt d'eines de rendiment i depuració.
  • Dart Analyzer: Proporciona anàlisi estàtica per detectar errors aviat.
  • Depurador de VS Code/Android Studio: Eines de depuració integrades dins dels IDEs populars.

Resum

Les proves i la depuració són pràctiques essencials en el desenvolupament de Flutter. Les proves unitàries asseguren que els components individuals funcionin correctament, les proves de widgets verifiquen el comportament de la interfície d'usuari i les proves d'integració confirmen que tota l'aplicació funcioni com s'espera. Les tècniques i eines de depuració efectives ajuden a identificar i resoldre problemes ràpidament, conduint a aplicacions més robustes i fiables. En dominar aquestes habilitats, pots millorar significativament la qualitat i mantenibilitat dels teus projectes Flutter.
