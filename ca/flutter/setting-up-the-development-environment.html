El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar l'entorn de desenvolupament és un primer pas crucial en el teu viatge per convertir-te en un expert en Flutter. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de les eines necessàries per començar a construir aplicacions Flutter.

Prerequisits

Abans d'endinsar-te en la configuració, assegura't de tenir els següents prerequisits:

  • Un ordinador amb macOS, Windows o Linux.
  • Comprensió bàsica de les operacions de línia de comandes.
  • Connexió a internet per descarregar els fitxers necessaris.

Instal·lació del Flutter SDK

El Flutter SDK és el component principal necessari per desenvolupar aplicacions Flutter. Segueix aquests passos per instal·lar-lo:

  1. Descarrega el Flutter SDK:

    • Visita el lloc web oficial de Flutter i descarrega la versió adequada per al teu sistema operatiu.

  2. Extreu el SDK:

    • Extreu el fitxer descarregat a una ubicació desitjada al teu sistema. Per exemple, en macOS o Linux, pots extreure'l al teu directori personal: 
      cd ~/development
unzip ~/Downloads/flutter_macos_vX.X.X-stable.zip

  3. Afegeix Flutter al PATH:

    • Actualitza la variable PATH del teu sistema per incloure el Flutter SDK. Això et permetrà executar comandes de Flutter des de qualsevol finestra de terminal.
      • En macOS i Linux, afegeix la següent línia al teu fitxer .bashrc, .zshrc o .bash_profile: 
        export PATH="$PATH:`pwd`/flutter/bin"
      • En Windows, segueix aquests passos:
        1. Obre la Cerca d'Inici, escriu "env" i selecciona "Edita les variables d'entorn del sistema".
        2. A la finestra de Propietats del Sistema, fes clic al botó "Variables d'entorn".
        3. Sota "Variables del sistema", troba la variable PATH, selecciona-la i fes clic a "Edita".
        4. Afegeix el camí al directori bin del Flutter SDK.

  4. Verifica la instal·lació:

    • Obre una nova finestra de terminal i executa: 
      flutter doctor
    • Aquesta comanda comprova el teu entorn i mostra un informe de l'estat de la teva instal·lació de Flutter. Segueix qualsevol instrucció addicional proporcionada per completar la configuració.

Configuració d'un editor

Flutter suporta diversos editors, però els més utilitzats són Visual Studio Code (VS Code) i Android Studio. A continuació es mostren les instruccions de configuració per a tots dos:

Visual Studio Code

  1. Instal·la VS Code:

    • Descarrega i instal·la VS Code des del lloc web oficial.

  2. Instal·la els plugins de Flutter i Dart:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca "Flutter" i instal·la l'extensió de Flutter.
    • L'extensió de Dart s'instal·larà automàticament juntament amb l'extensió de Flutter.

  3. Configura VS Code:

    • Obre el Paleta de Comandes (Ctrl+Shift+P) i executa la comanda Flutter: New Project per crear un nou projecte Flutter.
    • Segueix les indicacions per donar nom al teu projecte i triar una ubicació per desar-lo.

Android Studio

  1. Instal·la Android Studio:

    • Descarrega i instal·la Android Studio des del lloc web oficial.

  2. Instal·la els plugins de Flutter i Dart:

    • Obre Android Studio.
    • Ves a File > Settings (en macOS, Android Studio > Preferences).
    • Selecciona Plugins i cerca "Flutter".
    • Instal·la el plugin de Flutter, que també instal·larà el plugin de Dart.

  3. Configura Android Studio:

    • Reinicia Android Studio.
    • Obre la finestra Welcome to Android Studio i fes clic a Start a new Flutter project.
    • Segueix les indicacions per crear un nou projecte Flutter.

Configuració d'un emulador

Per provar les teves aplicacions Flutter, necessites un emulador o un dispositiu físic. Aquí tens com configurar un emulador d'Android:

  1. Instal·la l'emulador d'Android:

    • Obre Android Studio.
    • Ves a Tools > AVD Manager.
    • Fes clic a Create Virtual Device.
    • Tria una definició de dispositiu i fes clic a Next.
    • Selecciona una imatge de sistema i fes clic a Next.
    • Verifica la configuració i fes clic a Finish.

  2. Inicia l'emulador:

    • A l'AVD Manager, fes clic al botó Play al costat del teu dispositiu virtual per iniciar l'emulador.

  3. Verifica la configuració de l'emulador:

    • Executa la següent comanda al teu terminal per assegurar-te que l'emulador és reconegut: 
      flutter devices
    • Hauries de veure el teu emulador llistat com un dispositiu connectat.

Conclusió

Configurar l'entorn de desenvolupament per a Flutter implica instal·lar el Flutter SDK, configurar un editor com VS Code o Android Studio, i configurar un emulador per a proves. Seguint aquests passos, assegures que el teu entorn està llest per al desenvolupament amb Flutter, permetent-te centrar-te en construir aplicacions increïbles. Si trobes algun problema, consulta la documentació de Flutter o els fòrums de la comunitat per obtenir suport addicional.
