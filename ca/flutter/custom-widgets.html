Widgets Personalitzats
Gestió d'Estat Avançada ►

En aquesta secció, explorarem com crear widgets personalitzats en Flutter. Els widgets personalitzats són essencials per construir codi reutilitzable i mantenible. Al final d'aquesta secció, entendràs com crear i utilitzar widgets personalitzats, i com fer-los flexibles i adaptables a diferents casos d'ús.

Introducció als Widgets Personalitzats

Els widgets personalitzats són widgets definits per l'usuari que encapsulen una peça de la interfície d'usuari i el seu comportament. Ajuden a descompondre interfícies d'usuari complexes en peces més petites i manejables.

Conceptes Clau

  • StatelessWidget: Un widget que no manté cap estat.
  • StatefulWidget: Un widget que manté estat i pot canviar amb el temps.
  • Composició: Construir widgets complexos combinant-ne de més simples.

Creació d'un Widget Personalitzat Sense Estat

Un StatelessWidget és un widget que no canvia el seu estat un cop construït. És ideal per a contingut estàtic.

Exemple: Botó Personalitzat Simple

import 'package:flutter/material.dart';

class CustomButton extends StatelessWidget {
  final String text;
  final VoidCallback onPressed;

  CustomButton({required this.text, required this.onPressed});

  @override
  Widget build(BuildContext context) {
    return ElevatedButton(
      onPressed: onPressed,
      child: Text(text),
    );
  }
}

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Custom Button Example')),
        body: Center(
          child: CustomButton(
            text: 'Click Me',
            onPressed: () {
              print('Button Pressed');
            },
          ),
        ),
      ),
    );
  }
}

Explicació

  • Classe CustomButton: Aquesta classe extén StatelessWidget i pren dos paràmetres: text i onPressed.
  • Mètode build: Retorna un ElevatedButton amb el text proporcionat i la funció de callback onPressed.

Creació d'un Widget Personalitzat amb Estat

Un StatefulWidget és un widget que pot canviar el seu estat durant la seva vida útil. És útil per a contingut dinàmic.

Exemple: Widget de Comptador Personalitzat

import 'package:flutter/material.dart';

class CustomCounter extends StatefulWidget {
  @override
  _CustomCounterState createState() => _CustomCounterState();
}

class _CustomCounterState extends State<CustomCounter> {
  int _count = 0;

  void _increment() {
    setState(() {
      _count++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Text('Count: $_count', style: TextStyle(fontSize: 24)),
        SizedBox(height: 20),
        ElevatedButton(
          onPressed: _increment,
          child: Text('Increment'),
        ),
      ],
    );
  }
}

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Custom Counter Example')),
        body: Center(
          child: CustomCounter(),
        ),
      ),
    );
  }
}

Explicació

  • Classe CustomCounter: Extén StatefulWidget i crea el seu estat utilitzant _CustomCounterState.
  • Classe _CustomCounterState: Gestiona l'estat de CustomCounter. Inclou una variable _count i un mètode _increment que actualitza l'estat.
  • Mètode build: Retorna una Column amb un widget Text que mostra el comptador i un ElevatedButton per incrementar el comptador.

Fer Widgets Personalitzats Flexibles

Els widgets personalitzats han de ser flexibles i reutilitzables. Això es pot aconseguir utilitzant paràmetres i valors per defecte.

Exemple: Botó Personalitzat Flexible

import 'package:flutter/material.dart';

class FlexibleButton extends StatelessWidget {
  final String text;
  final VoidCallback onPressed;
  final Color color;
  final double fontSize;

  FlexibleButton({
    required this.text,
    required this.onPressed,
    this.color = Colors.blue,
    this.fontSize = 16.0,
  });

  @override
  Widget build(BuildContext context) {
    return ElevatedButton(
      style: ElevatedButton.styleFrom(primary: color),
      onPressed: onPressed,
      child: Text(
        text,
        style: TextStyle(fontSize: fontSize),
      ),
    );
  }
}

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Flexible Button Example')),
        body: Center(
          child: FlexibleButton(
            text: 'Click Me',
            onPressed: () {
              print('Button Pressed');
            },
            color: Colors.red,
            fontSize: 20.0,
          ),
        ),
      ),
    );
  }
}

Explicació

  • Classe FlexibleButton: Extén StatelessWidget i inclou paràmetres addicionals per a color i fontSize amb valors per defecte.
  • Mètode build: Utilitza els paràmetres proporcionats per estilitzar el ElevatedButton.

Conclusió

Els widgets personalitzats són una característica poderosa en Flutter que et permet crear components de la interfície d'usuari reutilitzables i mantenibles. Entenent la diferència entre StatelessWidget i StatefulWidget, i aprenent a fer els teus widgets flexibles, pots construir interfícies d'usuari complexes de manera més eficient. Practica creant els teus propis widgets personalitzats per convertir-te en un expert en el desenvolupament amb Flutter.
