Introducció a la Programació Orientada a Objectes (OOP)

La Programació Orientada a Objectes (OOP) és un paradigma de programació basat en el concepte d'"objectes", que poden contenir dades i codi: dades en forma de camps (sovint coneguts com a atributs o propietats), i codi en forma de procediments (sovint coneguts com a mètodes).

Conceptes Clau de l'OOP

  • Classes i Objectes
  • Herència
  • Polimorfisme
  • Encapsulació
  • Abstracció

Classes i Objectes en Dart

Una classe és un plànol per crear objectes. Un objecte és una instància d'una classe.

Definició d'una Classe

class Animal {
  String name;
  int age;

  // Constructor
  Animal(this.name, this.age);

  // Mètode
  void makeSound() {
    print('$name makes a sound.');
  }
}

Creació d'un Objecte

void main() {
  Animal cat = Animal('Whiskers', 2);
  cat.makeSound(); // Output: Whiskers makes a sound.
}

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

Exemple d'Herència

class Animal {
  String name;
  int age;

  Animal(this.name, this.age);

  void makeSound() {
    print('$name makes a sound.');
  }
}

class Dog extends Animal {
  String breed;

  Dog(String name, int age, this.breed) : super(name, age);

  void bark() {
    print('$name barks.');
  }
}

void main() {
  Dog dog = Dog('Buddy', 3, 'Golden Retriever');
  dog.makeSound(); // Output: Buddy makes a sound.
  dog.bark();      // Output: Buddy barks.
}

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen.

Exemple de Polimorfisme

class Animal {
  void makeSound() {
    print('Some generic animal sound');
  }
}

class Dog extends Animal {
  @override
  void makeSound() {
    print('Bark');
  }
}

class Cat extends Animal {
  @override
  void makeSound() {
    print('Meow');
  }
}

void main() {
  Animal myDog = Dog();
  Animal myCat = Cat();

  myDog.makeSound(); // Output: Bark
  myCat.makeSound(); // Output: Meow
}

Encapsulació

L'encapsulació és el concepte d'envoltar dades i mètodes que treballen sobre les dades dins d'una unitat, per exemple, una classe en Dart.

Exemple d'Encapsulació

class Person {
  String _name; // Propietat privada
  int _age;    // Propietat privada

  Person(this._name, this._age);

  // Getter per name
  String get name => _name;

  // Setter per name
  set name(String name) {
    _name = name;
  }

  // Getter per age
  int get age => _age;

  // Setter per age
  set age(int age) {
    if (age > 0) {
      _age = age;
    }
  }
}

void main() {
  Person person = Person('John', 30);
  print(person.name); // Output: John
  person.age = 35;
  print(person.age);  // Output: 35
}

Abstracció

L'abstracció és el concepte d'amagar els detalls complexos de la implementació i mostrar només les característiques necessàries d'un objecte.

Exemple d'Abstracció

abstract class Shape {
  void draw(); // Mètode abstracte
}

class Circle extends Shape {
  @override
  void draw() {
    print('Drawing a Circle');
  }
}

class Square extends Shape {
  @override
  void draw() {
    print('Drawing a Square');
  }
}

void main() {
  Shape circle = Circle();
  Shape square = Square();

  circle.draw(); // Output: Drawing a Circle
  square.draw(); // Output: Drawing a Square
}

Conclusió

La Programació Orientada a Objectes en Dart és un paradigma poderós que ajuda a estructurar el codi d'una manera més modular, reutilitzable i mantenible. Comprenent i aplicant els principis de classes i objectes, herència, polimorfisme, encapsulació i abstracció, pots crear aplicacions robustes i escalables en Flutter.
