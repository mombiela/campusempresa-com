Introducció

L'optimització del rendiment és crucial per crear aplicacions Flutter suaus, responsives i eficients. Aquesta secció et guiarà a través de diverses tècniques i millors pràctiques per optimitzar el rendiment de les teves aplicacions Flutter, des de nivells bàsics fins a avançats.

Entendre el Rendiment de Flutter

  • Arquitectura de Flutter: Entén l'arquitectura bàsica de Flutter, incloent-hi el llenguatge Dart, el motor de Flutter i el framework.
  • Pipeline de Renderització: Aprèn com Flutter renderitza widgets i la importància del pipeline de renderització en el rendiment.

  • DevTools: Utilitza Flutter DevTools per fer el perfil de la teva aplicació.

  • Registre: Utilitza declaracions print i paquets de registre per seguir problemes de rendiment.

Optimitzar Mètodes de Construcció

  • Evitar Reconstruir Widgets: Utilitza constructors const i widgets const per evitar reconstruccions innecessàries.
  • Claus: Utilitza claus per preservar l'estat dels widgets quan són reordenats o eliminats de l'arbre de widgets.
class MyWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        const Text('Aquest és un widget constant'),
        Text('Aquest és un widget no constant'),
      ],
    );
  }
}

Gestió Eficient de l'Estat

  • Widgets amb Estat: Utilitza StatefulWidget sàviament per gestionar l'estat localment.
  • Paquet Provider: Utilitza el paquet Provider per a una gestió eficient de l'estat a tota l'aplicació.
  • Riverpod: Explora Riverpod per a necessitats més avançades de gestió de l'estat.
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

void main() {
  runApp(
    ChangeNotifierProvider(
      create: (context) => Counter(),
      child: MyApp(),
    ),
  );
}

class Counter with ChangeNotifier {
  int _count = 0;

  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Exemple de Provider'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              Text('Has premut el botó aquest nombre de vegades:'),
              Consumer<Counter>(
                builder: (context, counter, child) {
                  return Text(
                    '${counter.count}',
                    style: Theme.of(context).textTheme.headline4,
                  );
                },
              ),
            ],
          ),
        ),
        floatingActionButton: FloatingActionButton(
          onPressed: () {
            Provider.of<Counter>(context, listen: false).increment();
          },
          tooltip: 'Incrementar',
          child: Icon(Icons.add),
        ),
      ),
    );
  }
}

Reduir la Reconstrucció de Widgets

  • Memoització: Emmagatzema en memòria cau operacions costoses per evitar càlculs redundants.
  • Selector i Consumer: Utilitza widgets Selector i Consumer del paquet Provider per minimitzar reconstruccions.

Renderització Eficient de Llistes

  • ListView.builder: Utilitza ListView.builder per a llistes grans per construir elements sota demanda.
  • ReorderableListView: Utilitza ReorderableListView per a llistes que necessiten capacitats de reordenació.
class MyListView extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return ListView.builder(
      itemCount: 1000,

  }
}

Optimització d'Imatges

  • Utilitza Formats d'Imatge Correctes: Utilitza formats d'imatge apropiats (per exemple, PNG, JPEG, WebP) segons el cas d'ús.
  • Emmagatzema Imatges en Memòria Cau: Utilitza el paquet cached_network_image per emmagatzemar imatges en memòria cau i reduir les trucades de xarxa.
import 'package:cached_network_image/cached_network_image.dart';

class MyImageWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return CachedNetworkImage(
      imageUrl: 'https://example.com/image.jpg',
      placeholder: (context, url) => CircularProgressIndicator(),
      errorWidget: (context, url, error) => Icon(Icons.error),
    );
  }
}

Optimització de Xarxa

  • Trucades API Eficients: Utilitza trucades API eficients i gestiona els errors de manera elegant.
  • Emmagatzematge de Dades en Memòria Cau: Emmagatzema dades localment per reduir les trucades de xarxa i millorar el rendiment.

Tècniques Avançades

  • Isolates: Utilitza isolates per a càlculs pesats per evitar bloquejar el fil principal.
  • Pintors Personalitzats: Utilitza pintors personalitzats per a gràfics i animacions complexes per optimitzar la renderització.
import 'dart:isolate';

void main() async {
  final receivePort = ReceivePort();
  await Isolate.spawn(isolateFunction, receivePort.sendPort);

  receivePort.listen((message) {
    print('Rebut: $message');
  });
}

void isolateFunction(SendPort sendPort) {
  sendPort.send('Hola des de l\'isolate!');
}
