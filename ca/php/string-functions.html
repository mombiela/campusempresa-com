Introducció a les funcions de cadena

Les funcions de cadena en PHP són eines essencials per manipular i gestionar text. Et permeten realitzar una varietat d'operacions com ara cercar, substituir, formatar i dividir cadenes. Aquesta secció et guiarà a través de les funcions de cadena més utilitzades en PHP, des de nivells bàsics fins a avançats.

Funcions bàsiques de cadena

strlen()

  • Descripció: Retorna la longitud d'una cadena.
  • Sintaxi: strlen(string $string): int
  • Exemple: 
    $text = "Hello, World!";
echo strlen($text); // Outputs: 13

strpos()

  • Descripció: Troba la posició de la primera ocurrència d'una subcadena en una cadena.
  • Sintaxi: strpos(string $haystack, string $needle, int $offset = 0): int|false
  • Exemple: 
    $text = "Hello, World!";
echo strpos($text, "World"); // Outputs: 7

substr()

  • Descripció: Retorna una part d'una cadena.
  • Sintaxi: substr(string $string, int $start, int $length = null): string
  • Exemple: 
    $text = "Hello, World!";
echo substr($text, 7, 5); // Outputs: World

strtoupper() i strtolower()

  • Descripció: Converteix una cadena a majúscules o minúscules.
  • Sintaxi:
    • strtoupper(string $string): string
    • strtolower(string $string): string
  • Exemple: 
    $text = "Hello, World!";
echo strtoupper($text); // Outputs: HELLO, WORLD!
echo strtolower($text); // Outputs: hello, world!

Funcions intermèdies de cadena

str_replace()

  • Descripció: Substitueix totes les ocurrències de la cadena de cerca amb la cadena de reemplaçament.
  • Sintaxi: str_replace(mixed $search, mixed $replace, mixed $subject, int &$count = null): mixed
  • Exemple: 
    $text = "Hello, World!";
echo str_replace("World", "PHP", $text); // Outputs: Hello, PHP!

explode() i implode()

  • Descripció:
    • explode(): Divideix una cadena per un delimitador especificat en un array.
    • implode(): Uneix elements d'un array en una cadena amb un delimitador especificat.
  • Sintaxi:
    • explode(string $delimiter, string $string, int $limit = PHP_INT_MAX): array
    • implode(string $glue, array $pieces): string
  • Exemple: 
    $text = "apple,banana,cherry";
$array = explode(",", $text);
print_r($array); // Outputs: Array ( [0] => apple [1] => banana [2] => cherry )

$array = ["apple", "banana", "cherry"];
echo implode(", ", $array); // Outputs: apple, banana, cherry

trim()

  • Descripció: Elimina espais en blanc (o altres caràcters) del començament i del final d'una cadena.
  • Sintaxi: trim(string $string, string $characters = " \n\r\t\v\x00"): string
  • Exemple: 
    $text = "  Hello, World!  ";
echo trim($text); // Outputs: Hello, World!

Funcions avançades de cadena

preg_match() i preg_replace()

  • Descripció:
    • preg_match(): Realitza una coincidència d'expressió regular.
    • preg_replace(): Realitza una cerca i substitució d'expressió regular.
  • Sintaxi:
    • preg_match(string $pattern, string $subject, array &$matches = null, int $flags = 0, int $offset = 0): int
    • preg_replace(mixed $pattern, mixed $replacement, mixed $subject, int $limit = -1, int &$count = null): mixed
  • Exemple: 
    $text = "The quick brown fox jumps over the lazy dog.";
if (preg_match("/quick/", $text)) {
    echo "Match found!"; // Outputs: Match found!
}

$text = "The quick brown fox jumps over the lazy dog.";
echo preg_replace("/quick/", "slow", $text); // Outputs: The slow brown fox jumps over the lazy dog.

sprintf()

  • Descripció: Retorna una cadena formatada.
  • Sintaxi: sprintf(string $format, mixed ...$values): string
  • Exemple: 
    $number = 9;
$text = "Beetlejuice";
echo sprintf("There are %d %s movies.", $number, $text); // Outputs: There are 9 Beetlejuice movies.

htmlentities() i htmlspecialchars()

  • Descripció: Converteix caràcters a entitats HTML.
  • Sintaxi:
    • htmlentities(string $string, int $flags = ENT_COMPAT, string $encoding = ini_get("default_charset"), bool $double_encode = true): string
    • htmlspecialchars(string $string, int $flags = ENT_COMPAT, string $encoding = ini_get("default_charset"), bool $double_encode = true): string
  • Exemple: 
    $text = "<a href='test'>Test</a>";
echo htmlentities($text); // Outputs: &lt;a href=&#039;test&#039;&gt;Test&lt;/a&gt;
echo htmlspecialchars($text); // Outputs: &lt;a href='test'&gt;Test&lt;/a&gt;

Conclusió

Les funcions de cadena en PHP són eines poderoses que et permeten manipular i gestionar text de manera eficient. Des de funcions bàsiques com strlen() i strpos() fins a funcions més avançades com preg_match() i sprintf(), entendre aquestes funcions et permetrà gestionar cadenes de manera efectiva en les teves aplicacions PHP. Practica utilitzant aquestes funcions amb diferents exemples per esdevenir proficient en la manipulació de cadenes en PHP.
