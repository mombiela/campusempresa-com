Introducció

La gestió d'arxius és un aspecte crucial de la programació en PHP, que et permet llegir i escriure arxius al servidor. Aquesta secció cobrirà els conceptes bàsics de la gestió d'arxius en PHP, incloent l'obertura, lectura, escriptura i tancament d'arxius.

Conceptes Clau

  • Modes d'Arxiu: Diferents modes per obrir arxius (per exemple, lectura, escriptura, afegir).
  • Funcions d'Arxiu: Funcions de PHP utilitzades per a la gestió d'arxius.
  • Gestió d'Errors: Gestionar errors durant les operacions amb arxius.

Modes d'Arxiu

PHP proporciona diversos modes per obrir arxius. Aquests són els més utilitzats:

| Mode | Descripció | |------|-------------| | r | Obrir només per llegir; comença al principi de l'arxiu. | | w | Obrir només per escriure; trunca l'arxiu a longitud zero o crea un nou arxiu si no existeix. | | a | Obrir només per escriure; afegeix dades al final de l'arxiu o crea un nou arxiu si no existeix. | | r+ | Obrir per llegir i escriure; comença al principi de l'arxiu. | | w+ | Obrir per llegir i escriure; trunca l'arxiu a longitud zero o crea un nou arxiu si no existeix. | | a+ | Obrir per llegir i escriure; afegeix dades al final de l'arxiu o crea un nou arxiu si no existeix. |

Funcions d'Arxiu

Obrir un Arxiu

Per obrir un arxiu, utilitza la funció fopen(). Aquesta funció requereix dos paràmetres: el nom de l'arxiu i el mode.

<?php
$file = fopen("example.txt", "r");
if ($file) {
    echo "Arxiu obert amb èxit.";
} else {
    echo "No s'ha pogut obrir l'arxiu.";
}
?>

Llegir d'un Arxiu

Per llegir d'un arxiu, pots utilitzar funcions com fread(), fgets(), o file_get_contents().

Utilitzant fread()

<?php
$file = fopen("example.txt", "r");
if ($file) {
    $content = fread($file, filesize("example.txt"));
    echo $content;
    fclose($file);
} else {
    echo "No s'ha pogut obrir l'arxiu.";
}
?>

Utilitzant fgets()

<?php
$file = fopen("example.txt", "r");
if ($file) {
    while (($line = fgets($file)) !== false) {
        echo $line . "<br>";
    }
    fclose($file);
} else {
    echo "No s'ha pogut obrir l'arxiu.";
}
?>

Utilitzant file_get_contents()

<?php
$content = file_get_contents("example.txt");
if ($content !== false) {
    echo $content;
} else {
    echo "No s'ha pogut llegir l'arxiu.";
}
?>

Escriure a un Arxiu

Per escriure a un arxiu, utilitza la funció fwrite().

<?php
$file = fopen("example.txt", "w");
if ($file) {
    $text = "Hola, Món!";
    fwrite($file, $text);
    fclose($file);
    echo "Dades escrites a l'arxiu.";
} else {
    echo "No s'ha pogut obrir l'arxiu.";
}
?>

Afegir a un Arxiu

Per afegir dades a un arxiu, obre'l en mode d'afegir (a o a+).

<?php
$file = fopen("example.txt", "a");
if ($file) {
    $text = "Afegint aquest text.";
    fwrite($file, $text);
    fclose($file);
    echo "Dades afegides a l'arxiu.";
} else {
    echo "No s'ha pogut obrir l'arxiu.";
}
?>

Tancar un Arxiu

Sempre tanca un arxiu després de completar les operacions per alliberar recursos.

<?php
$file = fopen("example.txt", "r");
// Realitzar operacions amb l'arxiu
fclose($file);
?>

Gestió d'Errors

Una gestió d'errors adequada és essencial per a operacions robustes amb arxius. Utilitza sentències condicionals per comprovar errors i gestionar-los de manera adequada.

<?php
$file = fopen("example.txt", "r");
if (!$file) {
    die("No s'ha pogut obrir l'arxiu.");
}
// Realitzar operacions amb l'arxiu
fclose($file);
?>

Resum

En aquesta secció, has après els conceptes bàsics de la gestió d'arxius en PHP, incloent:

  • Diferents modes d'arxiu per obrir arxius.
  • Funcions clau de PHP per a operacions amb arxius com fopen(), fread(), fwrite(), i fclose().
  • Tècniques per llegir, escriure i afegir dades a arxius.
  • La importància de la gestió d'errors en operacions amb arxius.

Entendre aquests conceptes bàsics t'ajudarà a gestionar arxius de manera efectiva en les teves aplicacions PHP.
