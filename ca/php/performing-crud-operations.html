Realització d'operacions CRUD
Ús de Sentències Preparades ►

CRUD significa Crear, Llegir, Actualitzar i Esborrar. Aquestes són les quatre operacions bàsiques per gestionar dades en una base de dades. En aquesta secció, aprendrem com realitzar aquestes operacions utilitzant PHP i MySQL.

Prerequisits

Abans d'endinsar-nos en les operacions CRUD, assegura't de tenir:

  • Una comprensió bàsica de la sintaxi de PHP.
  • Un entorn de servidor local (per exemple, XAMPP, WAMP o MAMP).
  • MySQL instal·lat i en funcionament.
  • Una base de dades i una taula per treballar.

Configuració de la base de dades

Primer, creem una base de dades i una taula per realitzar les nostres operacions CRUD.

CREATE DATABASE crud_example;
USE crud_example;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Connexió a la base de dades

Per interactuar amb la base de dades, necessitem establir una connexió utilitzant PHP.

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "crud_example";

// Crear connexió
$conn = new mysqli($servername, $username, $password, $dbname);

// Comprovar connexió
if ($conn->connect_error) {
    die("Connexió fallida: " . $conn->connect_error);
}
echo "Connexió establerta amb èxit";
?>

Operació de Creació

Per inserir dades a la base de dades, utilitzem la sentència SQL INSERT INTO.

<?php
$name = "John Doe";
$email = "john.doe@example.com";

$sql = "INSERT INTO users (name, email) VALUES ('$name', '$email')";

if ($conn->query($sql) === TRUE) {
    echo "Nou registre creat amb èxit";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}
?>

Explicació

  • $name i $email són les dades que volem inserir.
  • La sentència INSERT INTO especifica la taula (users) i les columnes (name, email) on es col·locaran les dades.
  • $conn->query($sql) executa la sentència SQL.

Operació de Lectura

Per recuperar dades de la base de dades, utilitzem la sentència SQL SELECT.

<?php
$sql = "SELECT id, name, email, created_at FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    // Sortida de dades de cada fila
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Nom: " . $row["name"]. " - Correu electrònic: " . $row["email"]. " - Creat el: " . $row["created_at"]. "<br>";
    }
} else {
    echo "0 resultats";
}
?>

Explicació

  • La sentència SELECT especifica les columnes que volem recuperar.
  • $result = $conn->query($sql) executa la sentència SQL.
  • $result->num_rows comprova si hi ha alguna fila retornada.
  • while($row = $result->fetch_assoc()) recorre el conjunt de resultats i obté cada fila com un array associatiu.

Operació d'Actualització

Per actualitzar dades existents, utilitzem la sentència SQL UPDATE.

<?php
$id = 1;
$newEmail = "new.email@example.com";

$sql = "UPDATE users SET email='$newEmail' WHERE id=$id";

if ($conn->query($sql) === TRUE) {
    echo "Registre actualitzat amb èxit";
} else {
    echo "Error en actualitzar el registre: " . $conn->error;
}
?>

Explicació

  • $id és l'identificador del registre que volem actualitzar.
  • $newEmail és el nou valor per a la columna de correu electrònic.
  • La sentència UPDATE especifica la taula (users), el nou valor (email='$newEmail') i la condició (WHERE id=$id).

Operació d'Esborrat

Per esborrar dades de la base de dades, utilitzem la sentència SQL DELETE.

<?php
$id = 1;

$sql = "DELETE FROM users WHERE id=$id";

if ($conn->query($sql) === TRUE) {
    echo "Registre esborrat amb èxit";
} else {
    echo "Error en esborrar el registre: " . $conn->error;
}
?>

Explicació

  • $id és l'identificador del registre que volem esborrar.
  • La sentència DELETE especifica la taula (users) i la condició (WHERE id=$id).

Conclusió

En aquesta secció, hem cobert els conceptes bàsics per realitzar operacions CRUD utilitzant PHP i MySQL:

  • Crear: Inserir nous registres a la base de dades.
  • Llegir: Recuperar registres de la base de dades.
  • Actualitzar: Modificar registres existents a la base de dades.
  • Esborrar: Eliminar registres de la base de dades.

Entendre aquestes operacions és fonamental per gestionar dades en qualsevol aplicació web. Practica aquests exemples i experimenta amb diferents consultes per aprofundir en la teva comprensió.
