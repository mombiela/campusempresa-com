Què és una Base de Dades?

Una base de dades és una col·lecció organitzada de dades, generalment emmagatzemada i accedida electrònicament des d'un sistema informàtic. Les bases de dades són essencials per emmagatzemar, gestionar i recuperar dades de manera eficient.

Tipus de Bases de Dades

  • Bases de Dades Relacionals: Emmagatzemen dades en taules amb files i columnes. Exemples inclouen MySQL, PostgreSQL i SQLite.
  • Bases de Dades NoSQL: Emmagatzemen dades en diversos formats com parelles clau-valor, documents, gràfics o magatzems de columnes amples. Exemples inclouen MongoDB, Redis i Cassandra.

Per què Utilitzar Bases de Dades amb PHP?

  • Persistència de Dades: Emmagatzemar dades de manera permanent.
  • Gestió de Dades: Gestionar grans quantitats de dades de manera eficient.
  • Recuperació de Dades: Recuperar dades ràpidament mitjançant consultes.
  • Seguretat: Emmagatzemar informació sensible de manera segura.

Configuració d'una Base de Dades

Instal·lant MySQL

  1. Windows: Descarrega i instal·la MySQL des del lloc web oficial.
  2. macOS: Utilitza Homebrew: brew install mysql
  3. Linux: Utilitza el teu gestor de paquets, per exemple, sudo apt-get install mysql-server

Creant una Base de Dades

Un cop MySQL està instal·lat, pots crear una base de dades utilitzant les següents comandes:

CREATE DATABASE my_database;

Connectant PHP a una Base de Dades

Utilitzant MySQLi

MySQLi (MySQL Improved) és una extensió de PHP dissenyada per treballar amb bases de dades MySQL.

Exemple: Connectant a una Base de Dades MySQL

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "my_database";

// Crear connexió
$conn = new mysqli($servername, $username, $password, $dbname);

// Comprovar connexió
if ($conn->connect_error) {
    die("Connexió fallida: " . $conn->connect_error);
}
echo "Connexió exitosa";
?>

Utilitzant PDO (PHP Data Objects)

PDO és una manera més flexible i segura de connectar-se a bases de dades en PHP.

Exemple: Connectant a una Base de Dades MySQL amb PDO

<?php
$dsn = 'mysql:host=localhost;dbname=my_database';
$username = 'username';
$password = 'password';

try {
    $pdo = new PDO($dsn, $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connexió exitosa";
} catch (PDOException $e) {
    echo "Connexió fallida: " . $e->getMessage();
}
?>

Realitzant Operacions CRUD

CRUD significa Crear, Llegir, Actualitzar i Esborrar. Aquestes són les operacions bàsiques que pots realitzar en una base de dades.

Crear

Exemple: Inserint Dades en una Taula

<?php
$sql = "INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com')";
if ($conn->query($sql) === TRUE) {
    echo "Nou registre creat amb èxit";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}
?>

Llegir

Exemple: Recuperant Dades d'una Taula

<?php
$sql = "SELECT id, username, email FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Nom: " . $row["username"]. " - Correu: " . $row["email"]. "<br>";
    }
} else {
    echo "0 resultats";
}
?>

Actualitzar

Exemple: Actualitzant Dades en una Taula

<?php
$sql = "UPDATE users SET email='new_email@example.com' WHERE username='john_doe'";
if ($conn->query($sql) === TRUE) {
    echo "Registre actualitzat amb èxit";
} else {
    echo "Error actualitzant el registre: " . $conn->error;
}
?>

Esborrar

Exemple: Esborrant Dades d'una Taula

<?php
$sql = "DELETE FROM users WHERE username='john_doe'";
if ($conn->query($sql) === TRUE) {
    echo "Registre esborrat amb èxit";
} else {
    echo "Error esborrant el registre: " . $conn->error;
}
?>

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les bases de dades, incloent què són, per què són importants i com connectar-s'hi utilitzant PHP. També hem explorat com realitzar operacions bàsiques CRUD. Entendre aquests fonaments és crucial per a qualsevol desenvolupador PHP, ja que les bases de dades són integrals per a la majoria d'aplicacions web.
