Introducció a les APIs

  • Definició: API significa Application Programming Interface. Permet que diferents aplicacions de programari es comuniquin entre elles.
  • Tipus d'APIs:
    • REST (Representational State Transfer): Utilitza sol·licituds HTTP per obtenir (GET), posar (PUT), enviar (POST) i eliminar (DELETE) dades.
    • SOAP (Simple Object Access Protocol): Un protocol per intercanviar informació estructurada en la implementació de serveis web.
  • Usos Comuns:
    • Obtenir dades d'un servei de tercers.
    • Enviar dades a un servidor remot.
    • Integrar-se amb altres aplicacions.

Configuració del Teu Entorn

  • Instal·lació de PHP: Assegura't que PHP està instal·lat al teu sistema.
  • Extensió cURL: cURL és una biblioteca utilitzada en PHP per fer sol·licituds HTTP. Assegura't que està habilitada al teu fitxer php.ini.
; Descomenta la següent línia al teu fitxer php.ini
extension=curl

Fer Sol·licituds HTTP amb cURL

  • Sol·licitud GET Bàsica: Obtenir dades d'una API.
<?php
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.example.com/data");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
print_r($data);
?>

  • Explicació:

    • curl_init(): Inicialitza una nova sessió cURL.
    • curl_setopt(): Estableix opcions per a la sessió cURL.
    • curl_exec(): Executa la sessió cURL i retorna la resposta.
    • curl_close(): Tanca la sessió cURL.
    • json_decode(): Converteix la resposta JSON a un array de PHP.

  • Sol·licitud POST Bàsica: Enviar dades a una API.

<?php
$data = array("name" => "John", "email" => "john@example.com");

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.example.com/create");
curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);
curl_close($ch);

$result = json_decode($response, true);
print_r($result);
?>
  • Explicació:
    • curl_setopt($ch, CURLOPT_POST, 1): Especifica que aquesta és una sol·licitud POST.
    • curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data)): Envia les dades com una cadena codificada en URL.

Gestionar Respostes d'API

  • Respostes JSON: La majoria d'APIs retornen dades en format JSON.
  • Gestió d'Errors: Comprova si hi ha errors en la resposta.
<?php
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.example.com/data");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);

if(curl_errno($ch)) {
    echo 'Error:' . curl_error($ch);
} else {
    $data = json_decode($response, true);
    if (isset($data['error'])) {
        echo 'Error de l\'API: ' . $data['error'];
    } else {
        print_r($data);
    }
}

curl_close($ch);
?>

Temes Avançats

  • Autenticació: Moltes APIs requereixen autenticació, com ara claus d'API o tokens OAuth.
  • Limitació de Rati: Les APIs sovint limiten el nombre de sol·licituds que pots fer en un període de temps determinat.
  • Paginació: Gestionar grans conjunts de dades que estan dividits en múltiples pàgines.

Exemple d'Autenticació

  • Utilitzant una Clau d'API:
<?php
$apiKey = "your_api_key";

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.example.com/data");
curl_setopt($ch, CURLOPT_HTTPHEADER, array(
    "Authorization: Bearer $apiKey"
));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
print_r($data);
?>

Conclusió

  • Resum:
    • Les APIs són essencials per integrar diferents sistemes de programari.
    • La biblioteca cURL de PHP és una eina poderosa per fer sol·licituds HTTP.
    • Entendre com gestionar respostes i errors és crucial per treballar amb APIs.
    • Temes avançats com l'autenticació, la limitació de rati i la paginació són importants per a aplicacions del món real.

En dominar aquests conceptes, estaràs ben preparat per treballar amb una àmplia gamma d'APIs en els teus projectes PHP.
