Introducció a PHP

PHP (Hypertext Preprocessor) és un llenguatge de script del costat del servidor popular dissenyat per al desenvolupament web, però també utilitzat com a llenguatge de programació de propòsit general. Aquesta secció cobrirà la sintaxi fonamental i els conceptes bàsics de PHP per començar a escriure scripts en PHP.

Sintaxi Bàsica de PHP

  • Etiquetes PHP: El codi PHP s'incrusta dins d'HTML utilitzant etiquetes PHP. 
    <?php
// El codi PHP va aquí
?>
  • Instrucció Echo: S'utilitza per mostrar text o variables. 
    <?php
echo "Hola, Món!";
?>

Variables i Tipus de Dades

  • Variables: Es declaren utilitzant el símbol $ seguit del nom de la variable. 
    <?php
$salutació = "Hola, Món!";
echo $salutació;
?>
  • Tipus de Dades:
    • String
    • Enter
    • Float (Double)
    • Booleà
    • Array
    • Objecte
    • NULL

Exemple de Diferents Tipus de Dades

<?php
$string = "Hola, Món!";
$enter = 42;
$float = 3.14;
$booleà = true;
$array = array("poma", "plàtan", "cirera");
$objecte = (object) ['propietat' => 'valor'];
$null = NULL;

echo $string;
echo $enter;
echo $float;
echo $booleà;
print_r($array);
echo $objecte->propietat;
echo $null;
?>

Operadors

  • Operadors Aritmètics: +, -, *, /, % 
    <?php
$suma = 5 + 3; // 8
$diferència = 5 - 3; // 2
$producte = 5 * 3; // 15
$quocient = 5 / 3; // 1.6667
$resta = 5 % 3; // 2
?>
  • Operadors d'Assignació: =, +=, -=, *=, /=, %= 
    <?php
$x = 10;
$x += 5; // 15
$x -= 3; // 12
$x *= 2; // 24
$x /= 4; // 6
$x %= 5; // 1
?>
  • Operadors de Comparació: ==, ===, !=, !==, >, <, >=, <= 
    <?php
$a = 5;
$b = 10;
var_dump($a == $b); // false
var_dump($a === $b); // false
var_dump($a != $b); // true
var_dump($a !== $b); // true
var_dump($a > $b); // false
var_dump($a < $b); // true
var_dump($a >= $b); // false
var_dump($a <= $b); // true
?>

Estructures de Control

  • Instrucció If-Else: Execució condicional de blocs de codi. 
    <?php
$edat = 20;
if ($edat >= 18) {
    echo "Ets un adult.";
} else {
    echo "Ets un menor.";
}
?>
  • Instrucció Switch: Selecciona un de molts blocs de codi per ser executat. 
    <?php
$dia = "Dilluns";
switch ($dia) {
    case "Dilluns":
        echo "Inici de la setmana laboral.";
        break;
    case "Divendres":
        echo "Final de la setmana laboral.";
        break;
    default:
        echo "Dies de mitja setmana.";
}
?>

Bucles

  • Bucle While: Repeteix un bloc de codi mentre la condició especificada sigui certa. 
    <?php
$i = 0;
while ($i < 5) {
    echo $i;
    $i++;
}
?>
  • Bucle For: Recorre un bloc de codi un nombre especificat de vegades. 
    <?php
for ($i = 0; $i < 5; $i++) {
    echo $i;
}
?>
  • Bucle Foreach: Recorre cada parell clau/valor en un array. 
    <?php
$fruites = array("poma", "plàtan", "cirera");
foreach ($fruites com a fruita) {
    echo $fruita;
}
?>

Funcions

  • Definició de Funcions: Les funcions són blocs de codi que es poden cridar repetidament. 
    <?php
function saludar($nom) {
    return "Hola, " . $nom;
}
echo saludar("Alice");
?>

Conclusió

En aquesta secció, hem cobert la sintaxi bàsica i els conceptes de PHP, incloent variables, tipus de dades, operadors, estructures de control, bucles i funcions. Aquests fonaments formen els blocs de construcció per a una programació PHP més avançada. Practiqueu aquests conceptes amb els exemples proporcionats per consolidar la vostra comprensió abans de passar a temes més complexos.
