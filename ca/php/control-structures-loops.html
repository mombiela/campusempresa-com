Introducció als Bucles

Els bucles són estructures de control fonamentals en PHP que et permeten executar un bloc de codi múltiples vegades. Són essencials per a tasques que requereixen repetició, com iterar a través d'arrays o realitzar càlculs repetitius.

Tipus de Bucles en PHP

PHP suporta diversos tipus de bucles:

  • Bucle while
  • Bucle do...while
  • Bucle for
  • Bucle foreach

Cada tipus de bucle és adequat per a diferents escenaris, i entendre quan utilitzar cadascun és crucial per a una programació eficient.

Bucle `while`

El bucle while executa un bloc de codi mentre una condició especificada sigui certa.

Sintaxi:

while (condició) {
    // Codi a executar
}

Exemple:

$count = 1;
while ($count <= 5) {
    echo "El compte és: $count\n";
    $count++;
}

En aquest exemple, el bucle imprimirà els números de l'1 al 5.

Bucle `do...while`

El bucle do...while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

Sintaxi:

do {
    // Codi a executar
} while (condició);

Exemple:

$count = 1;
do {
    echo "El compte és: $count\n";
    $count++;
} while ($count <= 5);

Aquest bucle també imprimirà els números de l'1 al 5, però assegura que el bloc de codi s'executi almenys una vegada, fins i tot si la condició és falsa inicialment.

Bucle `for`

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi:

for (inicialització; condició; increment) {
    // Codi a executar
}

Exemple:

for ($count = 1; $count <= 5; $count++) {
    echo "El compte és: $count\n";
}

Aquest bucle imprimirà els números de l'1 al 5, similar als exemples anteriors.

Bucle `foreach`

El bucle foreach està dissenyat específicament per iterar sobre arrays.

Sintaxi:

foreach ($array as $valor) {
    // Codi a executar
}

Exemple:

$colors = array("vermell", "verd", "blau", "groc");
foreach ($colors as $color) {
    echo "El color és: $color\n";
}

Aquest bucle imprimirà cada color de l'array.

Comparació de les Estructures de Bucle

| Tipus de Bucle | Cas d'Ús | Garanteix Execució? | |----------------|-----------------------------------------------|---------------------| | while | Quan el nombre d'iteracions no és conegut | No | | do...while | Quan el bloc ha d'executar-se almenys una vegada | Sí | | for | Quan el nombre d'iteracions és conegut | No | | foreach | Quan s'itera sobre arrays | No |

Errors Comuns

  • Bucles Infinits: Assegura't que la condició del bucle esdevindrà falsa eventualment. En cas contrari, el bucle s'executarà indefinidament.
  • Errors de Desfasament d'Un: Tingues cura amb les condicions del bucle per evitar executar el bucle una vegada de més o de menys.

Exemple d'un Bucle Infinit:

$count = 1;
while ($count <= 5) {
    echo "El compte és: $count\n";
    // Falta la instrucció d'increment
}

En aquest exemple, el bucle no acabarà mai perquè $count mai s'incrementa.

Exercicis

  1. Bucle while Bàsic: Escriu un bucle while que imprimeixi els números del 10 a l'1.

  2. Bucle do...while: Escriu un bucle do...while que imprimeixi els números de l'1 al 10.

  3. Bucle for: Escriu un bucle for que imprimeixi els números parells entre l'1 i el 20.

  4. Bucle foreach: Donat un array de fruites, escriu un bucle foreach que imprimeixi cada fruita.

Array Exemple:

$fruits = array("poma", "plàtan", "cirera", "dàtil");

Conclusió

Els bucles són una eina poderosa en PHP que et permeten realitzar tasques repetitives de manera eficient. Entendre els diferents tipus de bucles i els seus casos d'ús adequats farà que el teu codi sigui més efectiu i fàcil de llegir. Practica utilitzant cada tipus de bucle per familiaritzar-te amb la seva sintaxi i comportament.
