Introducció

Configurar un projecte PHP és el primer pas per desenvolupar una aplicació web. Aquesta secció et guiarà a través del procés de configuració d'un entorn de desenvolupament PHP, creació d'una estructura de projecte i execució del teu primer script PHP.

Conceptes Clau

  • Instal·lació de PHP: Instal·lar PHP a la teva màquina local.
  • Servidor Web: Configurar un servidor web com Apache o Nginx.
  • Base de Dades: Configurar un servidor de base de dades com MySQL o MariaDB.
  • IDE/Editor de Codi: Triar un Entorn de Desenvolupament Integrat (IDE) o editor de codi.
  • Estructura del Projecte: Organitzar els teus fitxers i directoris del projecte.

Instal·lació de PHP

Per executar scripts PHP, necessites tenir PHP instal·lat a la teva màquina. Segueix aquests passos per instal·lar PHP:

Windows

  1. Descarrega l'última versió de PHP des del lloc web oficial de PHP.
  2. Extreu el fitxer descarregat a un directori, per exemple, C:\php.
  3. Afegeix el directori de PHP a la variable d'entorn PATH del teu sistema.

macOS

  1. Obre Terminal.
  2. Instal·la PHP utilitzant Homebrew: 
    brew install php

Linux (Ubuntu)

  1. Obre Terminal.
  2. Actualitza la teva llista de paquets: 
    sudo apt update
  3. Instal·la PHP: 
    sudo apt install php

Configuració d'un Servidor Web

Es requereix un servidor web per servir els teus fitxers PHP. Pots utilitzar Apache o Nginx.

Apache

  • Windows: Instal·la XAMPP o WAMP, que inclouen Apache, PHP i MySQL.
  • macOS: Instal·la MAMP, que inclou Apache, PHP i MySQL.
  • Linux: 
    sudo apt install apache2
sudo apt install libapache2-mod-php

Nginx

  • Windows: Instal·la Nginx des del lloc web oficial.
  • macOS: Instal·la Nginx utilitzant Homebrew: 
    brew install nginx
  • Linux: 
    sudo apt install nginx

Configuració d'una Base de Dades

La majoria d'aplicacions PHP requereixen una base de dades. MySQL és una opció popular.

Instal·lació de MySQL

  • Windows: Instal·la MySQL des del lloc web oficial.
  • macOS: Instal·la MySQL utilitzant Homebrew: 
    brew install mysql
  • Linux: 
    sudo apt install mysql-server

Triar un IDE/Editor de Codi

Triar el IDE o editor de codi adequat pot millorar la teva productivitat. Aquí tens algunes opcions populars:

  • Visual Studio Code: Un editor de codi gratuït i de codi obert amb extensions per a PHP.
  • PHPStorm: Un IDE comercial dissenyat específicament per al desenvolupament en PHP.
  • Sublime Text: Un editor de codi lleuger i ràpid amb suport per a PHP.

Creació d'una Estructura de Projecte

Una estructura de projecte ben organitzada fa que el teu codi sigui més mantenible. Aquí tens una estructura bàsica per a un projecte PHP:

my_project/
├── public/
│   └── index.php
├── src/
│   └── main.php
├── tests/
│   └── main_test.php
├── vendor/
├── composer.json
└── README.md
  • public/: Conté fitxers accessibles públicament com index.php.
  • src/: Conté el teu codi font PHP.
  • tests/: Conté els teus scripts de prova.
  • vendor/: Conté biblioteques de tercers (gestionades per Composer).
  • composer.json: Fitxer de configuració de Composer.
  • README.md: Documentació del projecte.

Executar el Teu Primer Script PHP

Creem un simple script PHP per assegurar-nos que tot està configurat correctament.

  1. Crea un fitxer anomenat index.php al directori public/.
  2. Afegeix el següent codi: 
    <?php
echo "Hola, món!";
?>
  3. Inicia el teu servidor web i navega a http://localhost/index.php al teu navegador web. Hauries de veure "Hola, món!" mostrat.

Conclusió

Configurar un projecte PHP implica instal·lar PHP, configurar un servidor web, configurar una base de dades, triar un IDE i organitzar l'estructura del projecte. Seguint aquests passos, tindràs una base sòlida per començar a desenvolupar aplicacions PHP.
