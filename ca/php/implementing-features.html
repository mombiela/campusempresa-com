Introducció

En aquesta secció, explorarem com implementar diverses funcionalitats en PHP, des de nivells bàsics fins a avançats. Al final d'aquesta secció, tindràs una comprensió sòlida de com crear aplicacions PHP funcionals i dinàmiques.

Funcionalitats Bàsiques

1. Gestió de Formularis

La gestió de formularis és un aspecte fonamental del desenvolupament web. PHP facilita la recollida, validació i processament de dades de formularis.

Exemple: Formulari de Contacte Simple

<!DOCTYPE html>
<html>
<head>
    <title>Formulari de Contacte</title>
</head>
<body>
    <form method="post" action="process_form.php">
        Nom: <input type="text" name="name"><br>
        Correu electrònic: <input type="email" name="email"><br>
        Missatge: <textarea name="message"></textarea><br>
        <input type="submit" value="Enviar">
    </form>
</body>
</html>

Processament de les Dades del Formulari

<?php
// process_form.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = htmlspecialchars($_POST['name']);
    $email = htmlspecialchars($_POST['email']);
    $message = htmlspecialchars($_POST['message']);

    // Validació bàsica
    si (!empty($name) && !empty($email) && !empty($message)) {
        echo "Gràcies, $name. Hem rebut el teu missatge.";
    } else {
        echo "Tots els camps són obligatoris.";
    }
}
?>

2. Pujada d'Arxius

Pujar arxius és una altra funcionalitat comuna en aplicacions web. PHP proporciona funcions integrades per gestionar la pujada d'arxius de manera segura.

Exemple: Formulari de Pujada d'Arxius

<!DOCTYPE html>
<html>
<head>
    <title>Pujada d'Arxius</title>
</head>
<body>
    <form method="post" enctype="multipart/form-data" action="upload.php">
        Selecciona arxiu: <input type="file" name="fileToUpload"><br>
        <input type="submit" value="Pujar Arxiu">
    </form>
</body>
</html>

Processament de la Pujada d'Arxius

<?php
// upload.php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$fileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

// Comprovar si l'arxiu ja existeix
if (file_exists($target_file)) {
    echo "Ho sentim, l'arxiu ja existeix.";
    $uploadOk = 0;
}

// Comprovar la mida de l'arxiu
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Ho sentim, el teu arxiu és massa gran.";
    $uploadOk = 0;
}

// Permetre certs formats d'arxiu
if ($fileType != "jpg" && $fileType != "png" && $fileType != "jpeg" && $fileType != "gif") {
    echo "Ho sentim, només es permeten arxius JPG, JPEG, PNG i GIF.";
    $uploadOk = 0;
}

// Comprovar si $uploadOk està establert a 0 per un error
if ($uploadOk == 0) {
    echo "Ho sentim, el teu arxiu no s'ha pujat.";
} else {
    if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
        echo "L'arxiu ". htmlspecialchars(basename($_FILES["fileToUpload"]["name"])). " s'ha pujat.";
    } else {
        echo "Ho sentim, hi ha hagut un error en pujar el teu arxiu.";
    }
}
?>

Funcionalitats Intermèdies

1. Autenticació d'Usuaris

L'autenticació d'usuaris és crucial per assegurar aplicacions web. PHP es pot utilitzar per crear un sistema de login simple.

Exemple: Formulari de Login

<!DOCTYPE html>
<html>
<head>
    <title>Login</title>
</head>
<body>
    <form method="post" action="login.php">
        Nom d'usuari: <input type="text" name="username"><br>
        Contrasenya: <input type="password" name="password"><br>
        <input type="submit" value="Login">
    </form>
</body>
</html>

Processament del Login

<?php
// login.php
session_start();
$users = [
    'admin' => 'password123',
    'user' => 'userpass'
];

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    $password = $_POST['password'];

    if (isset($users[$username]) && $users[$username] == $password) {
        $_SESSION['username'] = $username;
        echo "Benvingut, $username!";
    } else {
        echo "Nom d'usuari o contrasenya incorrectes.";
    }
}
?>

2. Interacció amb Bases de Dades

Interaccionar amb una base de dades és essencial per a aplicacions web dinàmiques. PHP proporciona diverses maneres de connectar i interactuar amb bases de dades, com MySQL.

Exemple: Connexió a una Base de Dades MySQL

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Crear connexió
$conn = new mysqli($servername, $username, $password, $dbname);

// Comprovar connexió
if ($conn->connect_error) {
    die("Connexió fallida: " . $conn->connect_error);
}
echo "Connexió establerta correctament";
?>

Exemple: Inserir Dades en una Base de Dades

<?php
// Suposant que la connexió està establerta com es mostra anteriorment
$sql = "INSERT INTO Users (username, email, password) VALUES ('john_doe', 'john@example.com', 'securepassword')";

if ($conn->query($sql) === TRUE) {
    echo "Nou registre creat correctament";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>

Funcionalitats Avançades

1. Implementació d'APIs RESTful

Les APIs RESTful permeten que diferents aplicacions es comuniquin entre elles. PHP es pot utilitzar per crear i consumir APIs RESTful.

Exemple: API RESTful Simple

<?php
header("Content-Type: application/json");

$method = $_SERVER['REQUEST_METHOD'];

switch ($method) {
    case 'GET':
        echo json_encode(["message" => "S'ha rebut una sol·licitud GET"]);
        break;
    case 'POST':
        echo json_encode(["message" => "S'ha rebut una sol·licitud POST"]);
        break;
    default:
        echo json_encode(["message" => "Mètode de sol·licitud no suportat"]);
        break;
}
?>

2. Ús de Frameworks

Els frameworks de PHP com Laravel i Symfony poden accelerar significativament el desenvolupament i aplicar les millors pràctiques.

Exemple: Ruta Bàsica en Laravel

// routes/web.php
Route::get('/', function () {
    return view('welcome');
});

Exemple: Controlador Bàsic en Laravel

// app/Http/Controllers/ExampleController.php
namespace App\Http\Controllers;

use Illuminate\Http\Request;

class ExampleController extends Controller
{
    public function index()
    {
        return view('example');
    }
}

Conclusió

Implementar funcionalitats en PHP va des de gestionar formularis simples fins a crear APIs RESTful complexes i utilitzar frameworks. En dominar aquestes funcionalitats, pots construir aplicacions web robustes i dinàmiques. Practica cada exemple i experimenta amb variacions per aprofundir en la teva comprensió.
