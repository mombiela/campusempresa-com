Introducció

En aquesta secció, introduirem el concepte de l'arquitectura MVC (Model-Vista-Controlador) i explicarem per què és beneficiós per construir aplicacions web. També proporcionarem una visió general del que aprendràs en aquest tema.

  • Arquitectura MVC: Un patró de disseny que separa l'aplicació en tres components principals: Model, Vista i Controlador.
  • Beneficis: Millora de l'organització del codi, manteniment més fàcil i millor separació de responsabilitats.

Configurant l'Entorn

Abans de començar a construir el nostre marc MVC, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lació de PHP: Assegura't que PHP està instal·lat al teu sistema.
  • Servidor Web: Utilitza Apache o Nginx.
  • Composer: Gestor de dependències de PHP.

Exemple: Instal·lant Composer

# A Windows
php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
php composer-setup.php
php -r "unlink('composer-setup.php');"
# A macOS/Linux
curl -sS https://getcomposer.org/installer | php

Creant l'Estructura de Directoris

Crearem una estructura de directoris que segueixi el patró MVC.

  • Directori Arrel: Conté el punt d'entrada i els fitxers de configuració.
  • Directori App: Conté els components MVC.
    • Models: Lògica de negoci i manipulació de dades.
    • Vistes: Capa de presentació.
    • Controladors: Gestionen l'entrada i les interaccions de l'usuari.

Exemple d'Estructura de Directoris

/my_mvc_framework
    /app
        /controllers
        /models
        /views
    /public
        index.php
    /config
        config.php
    /vendor
    composer.json

Configurant l'Aplicació

Configurarem fitxers de configuració per gestionar ajustos com connexions a bases de dades i configuracions de l'aplicació.

Exemple: config/config.php

<?php
return [
    'db' => [
        'host' => 'localhost',
        'dbname' => 'my_database',
        'user' => 'root',
        'password' => '',
    ],
];

Construint el Model

El Model representa les dades i la lògica de negoci de l'aplicació.

  • Connexió a la Base de Dades: Establir una connexió amb la base de dades.
  • Operacions CRUD: Crear, Llegir, Actualitzar, Eliminar operacions.

Exemple: app/models/User.php

<?php
class User {
    private $db;

    public function __construct($db) {
        $this->db = $db;
    }

    public function getAllUsers() {
        $stmt = $this->db->query("SELECT * FROM users");
        return $stmt->fetchAll(PDO::FETCH_ASSOC);
    }
}

Construint la Vista

La Vista és responsable de renderitzar la interfície d'usuari.

  • Plantilles: Fitxers HTML amb PHP incrustat per contingut dinàmic.
  • Separació de la Lògica: Mantenir la lògica de negoci fora de les vistes.

Exemple: app/views/users.php

<!DOCTYPE html>
<html>
<head>
    <title>Users</title>
</head>
<body>
    <h1>Users List</h1>
    <ul>
        <?php foreach ($users as $user): ?>
            <li><?php echo htmlspecialchars($user['name']); ?></li>
        <?php endforeach; ?>
    </ul>
</body>
</html>

Construint el Controlador

El Controlador gestiona l'entrada de l'usuari i actualitza el Model i la Vista en conseqüència.

  • Enrutament: Mapar URLs a controladors i accions específiques.
  • Gestió de Peticions: Processar l'entrada de l'usuari i interactuar amb el Model.

Exemple: app/controllers/UserController.php

<?php
class UserController {
    private $userModel;

    public function __construct($userModel) {
        $this->userModel = $userModel;
    }

    public function index() {
        $users = $this->userModel->getAllUsers();
        include '../app/views/users.php';
    }
}

Configurant l'Enrutament

L'enrutament és essencial per dirigir les peticions a les accions del controlador apropiades.

Exemple: public/index.php

<?php
require '../config/config.php';
require '../app/models/User.php';
require '../app/controllers/UserController.php';

$config = require '../config/config.php';
$db = new PDO('mysql:host=' . $config['db']['host'] . ';dbname=' . $config['db']['dbname'], $config['db']['user'], $config['db']['password']);

$userModel = new User($db);
$userController = new UserController($userModel);

if ($_SERVER['REQUEST_URI'] == '/users') {
    $userController->index();
} else {
    echo "404 Not Found";
}

Conclusió

En aquest tema, hem construït un marc MVC simple des de zero. Hem cobert els conceptes bàsics de la configuració de l'entorn, la creació de l'estructura de directoris i la implementació dels components Model, Vista i Controlador. Aquest marc es pot expandir i personalitzar per adaptar-se a requisits més complexos a mesura que avancis en les teves habilitats de programació en PHP.

  • Pròxims Passos: Considera afegir funcions com la gestió de formularis, validació i autenticació per millorar el teu marc MVC.
