Introducció als Arrays

Els arrays són una de les estructures de dades més fonamentals en PHP. Et permeten emmagatzemar múltiples valors en una sola variable, facilitant la gestió i manipulació de col·leccions de dades.

Conceptes Clau

  • Arrays Indexats: Arrays amb un índex numèric.
  • Arrays Associatius: Arrays amb claus nominatives.
  • Arrays Multidimensionals: Arrays que contenen un o més arrays.

Creació d'Arrays

Arrays Indexats

Els arrays indexats utilitzen claus numèriques per emmagatzemar i accedir als valors.

// Creació d'un array indexat
$fruits = array("Poma", "Plàtan", "Cirera");

// Accés als elements
echo $fruits[0]; // Mostra: Poma

Arrays Associatius

Els arrays associatius utilitzen claus nominatives per emmagatzemar i accedir als valors.

// Creació d'un array associatiu
$ages = array("Pere" => 35, "Ben" => 37, "Josep" => 43);

// Accés als elements
echo $ages["Pere"]; // Mostra: 35

Arrays Multidimensionals

Els arrays multidimensionals contenen un o més arrays.

// Creació d'un array multidimensional
$contacts = array(
    "Joan" => array("telèfon" => "123-4567", "correu" => "joan@example.com"),
    "Jana" => array("telèfon" => "987-6543", "correu" => "jana@example.com")
);

// Accés als elements
echo $contacts["Joan"]["telèfon"]; // Mostra: 123-4567

Funcions d'Array

PHP proporciona una àmplia gamma de funcions integrades per treballar amb arrays.

Funcions Comunes d'Array

  • count(): Compta el nombre d'elements en un array.
  • array_push(): Afegeix un o més elements al final d'un array.
  • array_pop(): Elimina l'últim element d'un array.
  • array_merge(): Fusiona un o més arrays.
  • array_keys(): Retorna totes les claus d'un array.
  • array_values(): Retorna tots els valors d'un array.
// Exemple d'ús de funcions d'array
$numbers = array(1, 2, 3);
array_push($numbers, 4); // Afegeix 4 al final de l'array
print_r($numbers); // Mostra: Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 4 )

$last = array_pop($numbers); // Elimina l'últim element
echo $last; // Mostra: 4

$merged = array_merge($numbers, array(5, 6));
print_r($merged); // Mostra: Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 5 [4] => 6 )

Iteració sobre Arrays

Ús del Bucle `foreach`

El bucle foreach és la manera més comuna d'iterar sobre arrays.

// Iteració sobre un array indexat
$fruits = array("Poma", "Plàtan", "Cirera");
foreach ($fruits as $fruit) {
    echo $fruit . "\n";
}

// Iteració sobre un array associatiu
$ages = array("Pere" => 35, "Ben" => 37, "Josep" => 43);
foreach ($ages as $name => $age) {
    echo "$name té $age anys.\n";
}

Ús del Bucle `for`

També pots utilitzar un bucle for per iterar sobre arrays indexats.

$fruits = array("Poma", "Plàtan", "Cirera");
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "\n";
}

Tècniques Avançades d'Array

Ordenació d'Arrays

PHP proporciona diverses funcions per ordenar arrays.

  • sort(): Ordena arrays indexats en ordre ascendent.
  • rsort(): Ordena arrays indexats en ordre descendent.
  • asort(): Ordena arrays associatius en ordre ascendent, segons el valor.
  • ksort(): Ordena arrays associatius en ordre ascendent, segons la clau.
// Ordenació d'un array indexat
$numbers = array(4, 2, 8, 6);
sort($numbers);
print_r($numbers); // Mostra: Array ( [0] => 2 [1] => 4 [2] => 6 [3] => 8 )

// Ordenació d'un array associatiu per valor
$ages = array("Pere" => 35, "Ben" => 37, "Josep" => 43);
asort($ages);
print_r($ages); // Mostra: Array ( [Pere] => 35 [Ben] => 37 [Josep] => 43 )

Filtratge d'Arrays

Pots filtrar arrays utilitzant la funció array_filter().

// Filtratge d'un array
$numbers = array(1, 2, 3, 4, 5, 6);
$even = array_filter($numbers, function($num) {
    return $num % 2 == 0;
});
print_r($even); // Mostra: Array ( [1] => 2 [3] => 4 [5] => 6 )

Conclusió

Els arrays són una estructura de dades potent i versàtil en PHP. Et permeten emmagatzemar i manipular col·leccions de dades de manera eficient. Entenent com crear, accedir i manipular arrays, pots gestionar una àmplia gamma de tasques de programació de manera més efectiva. Practica l'ús d'arrays i les seves funcions associades per convertir-te en un expert en la gestió de dades en les teves aplicacions PHP.
