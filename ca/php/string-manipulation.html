Manipulació de Cadenes en PHP
Funcions de Cadena ►

La manipulació de cadenes és un aspecte fonamental de la programació en PHP. Aquest tema cobrirà diverses tècniques i funcions per manejar i manipular cadenes de manera efectiva, des d'operacions bàsiques fins a manipulacions avançades.

Introducció a les Cadenes en PHP

  • Definició: Una cadena és una seqüència de caràcters.
  • Sintaxi: Les cadenes es poden definir utilitzant cometes simples (') o cometes dobles (").
$singleQuoteString = 'Hola, Món!';
$doubleQuoteString = "Hola, Món!";
  • Diferència entre cometes simples i dobles:
    • Cometes simples: No es fa anàlisi de variables o caràcters especials.
    • Cometes dobles: Analitza variables i caràcters especials com \n (nova línia).
$name = "Joan";
echo 'Hola, $name'; // Mostra: Hola, $name
echo "Hola, $name"; // Mostra: Hola, Joan

Funcions Bàsiques de Cadenes

  • strlen(): Retorna la longitud d'una cadena.
$string = "Hola, Món!";
echo strlen($string); // Mostra: 13
  • strtolower() i strtoupper(): Converteix una cadena a minúscules o majúscules.
echo strtolower($string); // Mostra: hola, món!
echo strtoupper($string); // Mostra: HOLA, MÓN!
  • ucfirst() i ucwords(): Capitalitza la primera lletra d'una cadena o cada paraula d'una cadena.
echo ucfirst("hola món"); // Mostra: Hola món
echo ucwords("hola món"); // Mostra: Hola Món
  • trim(): Elimina els espais en blanc del començament i el final d'una cadena.
$string = "  Hola, Món!  ";
echo trim($string); // Mostra: Hola, Món!

Concatenació de Cadenes

  • Operador de Concatenació (.): S'utilitza per unir dues o més cadenes.
$greeting = "Hola";
$name = "Joan";
echo $greeting . ", " . $name . "!"; // Mostra: Hola, Joan!

Funcions Avançades de Cadenes

  • substr(): Retorna una part d'una cadena.
$string = "Hola, Món!";
echo substr($string, 7, 5); // Mostra: Món!
  • strpos(): Troba la posició de la primera ocurrència d'una subcadena.
echo strpos($string, "Món"); // Mostra: 7
  • str_replace(): Substitueix totes les ocurrències d'una cadena de cerca per una cadena de reemplaçament.
echo str_replace("Món", "PHP", $string); // Mostra: Hola, PHP!
  • preg_match(): Realitza una coincidència d'expressió regular.
$pattern = "/món/i"; // 'i' ho fa insensible a majúscules i minúscules
echo preg_match($pattern, $string); // Mostra: 1 (cert)

Formatació de Cadenes

  • printf() i sprintf(): Format i sortida de cadenes.
$number = 123.456;
printf("El número és %.2f", $number); // Mostra: El número és 123.46

$formattedString = sprintf("El número és %.2f", $number);
echo $formattedString; // Mostra: El número és 123.46

Funcions de Cadenes Multibyte

  • mb_strlen(): Retorna la longitud d'una cadena en caràcters, tenint en compte els caràcters multibyte.
$multibyteString = "こんにちは";
echo mb_strlen($multibyteString); // Mostra: 5
  • mb_substr(): Retorna una part d'una cadena, tenint en compte els caràcters multibyte.
echo mb_substr($multibyteString, 0, 2); // Mostra: こん

Conclusió

La manipulació de cadenes és una habilitat crucial en la programació PHP. Entendre i utilitzar les diverses funcions de cadenes pot millorar enormement la teva capacitat per processar i gestionar dades de text. Des d'operacions bàsiques com la concatenació i el retallat fins a tècniques avançades com les expressions regulars i la gestió de cadenes multibyte, dominar aquestes eines et farà un desenvolupador PHP més efectiu i eficient.
