Introducció

Els operadors són caràcters o símbols especials en PHP que s'utilitzen per realitzar operacions sobre variables i valors. Són essencials per manipular dades i controlar el flux d'un programa. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en PHP, proporcionant explicacions clares, exemples i exercicis per ajudar-te a entendre el seu ús.

Tipus d'Operadors

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques comunes.

  • Addició (+): Afegeix dos operands.
  • Subtracció (-): Resta el segon operand del primer.
  • Multiplicació (*): Multiplica dos operands.
  • Divisió (/): Divideix el primer operand pel segon.
  • Mòdul (%): Retorna el residu de la divisió del primer operand pel segon.
  • Exponenciació (**): Eleva el primer operand a la potència del segon operand.
<?php
$a = 10;
$b = 3;

echo $a + $b; // Sortida: 13
echo $a - $b; // Sortida: 7
echo $a * $b; // Sortida: 30
echo $a / $b; // Sortida: 3.3333333333333
echo $a % $b; // Sortida: 1
echo $a ** $b; // Sortida: 1000
?>

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació Bàsica (=): Assigna el valor de l'operand dret a l'operand esquerre.
  • Assignació d'Addició (+=): Afegeix l'operand dret a l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Assignació de Subtracció (-=): Resta l'operand dret de l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Assignació de Multiplicació (*=): Multiplica l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.
  • Assignació de Divisió (/=): Divideix l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.
  • Assignació de Mòdul (%=): Pren el mòdul utilitzant dos operands i assigna el resultat a l'operand esquerre.
<?php
$a = 10;
$b = 5;

$a += $b; // $a ara és 15
$a -= $b; // $a ara és 10
$a *= $b; // $a ara és 50
$a /= $b; // $a ara és 10
$a %= $b; // $a ara és 0
?>

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors.

  • Igual (==): Retorna cert si ambdós operands són iguals.
  • Idèntic (===): Retorna cert si ambdós operands són iguals i del mateix tipus.
  • No Igual (!= o <>): Retorna cert si els operands no són iguals.
  • No Idèntic (!==): Retorna cert si els operands no són iguals o no són del mateix tipus.
  • Més Gran Que (>): Retorna cert si l'operand esquerre és més gran que l'operand dret.
  • Menys Que (<): Retorna cert si l'operand esquerre és menys que l'operand dret.
  • Més Gran o Igual Que (>=): Retorna cert si l'operand esquerre és més gran o igual que l'operand dret.
  • Menys Que o Igual Que (<=): Retorna cert si l'operand esquerre és menys o igual que l'operand dret.
<?php
$a = 10;
$b = 20;

var_dump($a == $b); // Sortida: bool(false)
var_dump($a === $b); // Sortida: bool(false)
var_dump($a != $b); // Sortida: bool(true)
var_dump($a !== $b); // Sortida: bool(true)
var_dump($a > $b); // Sortida: bool(false)
var_dump($a < $b); // Sortida: bool(true)
var_dump($a >= $b); // Sortida: bool(false)
var_dump($a <= $b); // Sortida: bool(true)
?>

Operadors Lògics

Els operadors lògics s'utilitzen per combinar declaracions condicionals.

  • I (&& o and): Retorna cert si ambdós operands són certs.
  • O (|| o or): Retorna cert si almenys un dels operands és cert.
  • No (!): Retorna cert si l'operand és fals.
<?php
$a = true;
$b = false;

var_dump($a && $b); // Sortida: bool(false)
var_dump($a || $b); // Sortida: bool(true)
var_dump(!$a); // Sortida: bool(false)
?>

Operadors d'Increment/Decrement

Els operadors d'increment i decrement s'utilitzen per augmentar o disminuir el valor d'una variable en un.

  • Increment (++): Augmenta el valor de la variable en un.
  • Decrement (--): Disminueix el valor de la variable en un.
<?php
$a = 10;

echo ++$a; // Sortida: 11 (Pre-increment)
echo $a++; // Sortida: 11 (Post-increment)
echo $a;   // Sortida: 12

echo --$a; // Sortida: 11 (Pre-decrement)
echo $a--; // Sortida: 11 (Post-decrement)
echo $a;   // Sortida: 10
?>

Operadors de Cadena

Els operadors de cadena s'utilitzen per manipular valors de cadena.

  • Concatenació (.): Concatena dues cadenes.
  • Assignació de Concatenació (.=): Afegeix l'operand dret a l'operand esquerre i assigna el resultat a l'operand esquerre.
<?php
$a = "Hola";
$b = " Món";

echo $a . $b; // Sortida: Hola Món

$a .= $b;
echo $a; // Sortida: Hola Món
?>

Operadors d'Array

Els operadors d'array s'utilitzen per comparar arrays.

  • Unió (+): Fusiona dos arrays.
  • Igualtat (==): Retorna cert si dos arrays tenen les mateixes parelles clau/valor.
  • Identitat (===): Retorna cert si dos arrays tenen les mateixes parelles clau/valor en el mateix ordre i del mateix tipus.
  • Desigualtat (!= o <>): Retorna cert si dos arrays no són iguals.
  • No Identitat (!==): Retorna cert si dos arrays no són idèntics.
<?php
$a = array("a" => "poma", "b" => "plàtan");
$b = array("c" => "cirera", "d" => "dàtil");

$c = $a + $b; // Unió de $a i $b
var_dump($c); // Sortida: array(4) { ["a"]=> string(5) "poma" ["b"]=> string(6) "plàtan" ["c"]=> string(6) "cirera" ["d"]=> string(4) "dàtil" }

var_dump($a == $b); // Sortida: bool(false)
var_dump($a === $b); // Sortida: bool(false)
var_dump($a != $b); // Sortida: bool(true)
var_dump($a !== $b); // Sortida: bool(true)
?>

Conclusió

Els operadors són fonamentals en PHP per realitzar diverses operacions sobre dades. Entendre com utilitzar operadors aritmètics, d'assignació, de comparació, lògics, d'increment/decrement, de cadena i d'array millorarà significativament la teva capacitat per escriure codi PHP eficient i efectiu. Practica l'ús d'aquests operadors en diferents escenaris per sentir-te més còmode amb la seva sintaxi i funcionalitat.
