Variables i Tipus de Dades en PHP

Introducció

En aquesta secció, cobrim els conceptes bàsics de les variables i els tipus de dades en PHP. Entendre aquests conceptes és crucial per escriure codi PHP eficient i efectiu. Començarem amb els fonaments de les variables, i després passarem als diferents tipus de dades disponibles en PHP.

Variables

Què és una Variable?

  • Una variable és una ubicació d'emmagatzematge identificada per un nom que conté dades que poden ser modificades durant l'execució del programa.
  • En PHP, les variables es denoten amb un signe de dòlar ($) seguit del nom de la variable.

Declaració de Variables

  • Les variables en PHP es declaren assignant-los un valor utilitzant l'operador d'assignació (=).
  • PHP és un llenguatge de tipus lliure, el que significa que no cal declarar el tipus de dades d'una variable.
<?php
$variableName = "Hola, Món!";
$number = 42;
?>

Regles de Nomenclatura de Variables

  • Els noms de les variables han de començar amb una lletra o un guió baix (_).
  • Els noms de les variables només poden contenir caràcters alfanumèrics i guions baixos (A-z, 0-9, i _).
  • Els noms de les variables són sensibles a majúscules i minúscules ($Variable i $variable són diferents).

Exemples

<?php
$name = "Joan";
$age = 30;
$isStudent = true;
?>

Tipus de Dades

Visió General

PHP suporta diversos tipus de dades, que es poden categoritzar en tipus escalars, tipus compostos i tipus especials.

Tipus Escalars

  • Enter: Nombres sencers, positius o negatius.
  • Float: Nombres amb un punt decimal o en forma exponencial.
  • String: Una seqüència de caràcters.
  • Boolean: Representa dos estats possibles: TRUE o FALSE.

Tipus Compostos

  • Array: Una col·lecció de valors.
  • Objecte: Una instància d'una classe.

Tipus Especials

  • NULL: Una variable sense valor.
  • Recurs: Una variable especial que conté una referència a un recurs extern (per exemple, connexió a una base de dades).

Enter

<?php
$intVar = 100;
echo $intVar; // Sortida: 100
?>

Float

<?php
$floatVar = 10.5;
echo $floatVar; // Sortida: 10.5
?>

String

<?php
$stringVar = "Hola, PHP!";
echo $stringVar; // Sortida: Hola, PHP!
?>

Boolean

<?php
$boolVar = true;
echo $boolVar; // Sortida: 1 (true es representa com 1)
?>

Array

<?php
$arrayVar = array("poma", "plàtan", "cirera");
echo $arrayVar[0]; // Sortida: poma
?>

Objecte

<?php
class Car {
    public $color;
    public function __construct($color) {
        $this->color = $color;
    }
}

$myCar = new Car("vermell");
echo $myCar->color; // Sortida: vermell
?>

NULL

<?php
$nullVar = NULL;
echo $nullVar; // Sortida: (res)
?>

Recurs

  • Els recursos són variables especials que contenen referències a recursos externs.
  • Comunament utilitzats amb connexions a bases de dades, manejadors de fitxers, etc.
<?php
$file = fopen("test.txt", "r");
echo $file; // Sortida: Resource id #3
fclose($file);
?>

Manipulació de Tipus i Conversió de Tipus

Manipulació de Tipus

  • PHP converteix automàticament les variables al tipus de dades correcte basant-se en el seu valor.
  • Això es coneix com a manipulació de tipus.
<?php
$var = "10";
$var = $var + 2;
echo $var; // Sortida: 12 (la cadena "10" es converteix a enter 10)
?>

Conversió de Tipus

  • Pots convertir explícitament una variable a un tipus de dades diferent utilitzant la conversió de tipus.
<?php
$var = "10";
$intVar = (int)$var;
echo $intVar; // Sortida: 10
?>

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en PHP. Hem après com declarar variables, les regles per nomenar-les, i els diversos tipus de dades disponibles en PHP. Entendre aquests conceptes és fonamental per escriure codi PHP efectiu. A mesura que progressis, veuràs com aquests blocs de construcció bàsics s'utilitzen per crear aplicacions web més complexes i dinàmiques.
