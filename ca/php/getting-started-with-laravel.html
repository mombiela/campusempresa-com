Introducció a Laravel

Laravel és un potent i flexible framework PHP dissenyat per fer el desenvolupament web més fàcil i agradable. Segueix el patró arquitectònic Model-Vista-Controlador (MVC) i proporciona un ric conjunt de característiques per construir aplicacions web modernes.

Conceptes Clau

  • Arquitectura MVC: Separa la lògica de l'aplicació en tres components principals: Model, Vista i Controlador.
  • Enrutament: Defineix com l'aplicació respon a diverses sol·licituds HTTP.
  • Eloquent ORM: Proporciona una manera simple i elegant d'interactuar amb la base de dades.
  • Motor de Plantilles Blade: Un potent motor de plantilles per crear vistes dinàmiques.
  • Artisan CLI: Una interfície de línia de comandes que proporciona nombroses ordres útils per al desenvolupament d'aplicacions.

Configurant Laravel

Per començar amb Laravel, necessites configurar el teu entorn de desenvolupament.

Prerequisits

  • PHP >= 7.3
  • Composer (Gestor de dependències per a PHP)
  • Un servidor web com Apache o Nginx
  • Una base de dades com MySQL o SQLite

Passos d'Instal·lació

  1. Instal·lar Composer: Si encara no has instal·lat Composer, descarrega'l i instal·la'l des de getcomposer.org.
  2. Crear un Nou Projecte Laravel: 
    composer create-project --prefer-dist laravel/laravel myLaravelApp
  3. Navegar al Directori del Projecte: 
    cd myLaravelApp
  4. Servir l'Aplicació: 
    php artisan serve
    Aquesta comanda iniciarà un servidor de desenvolupament local a http://localhost:8000.

Estructura de Directori de Laravel

Entendre l'estructura de directoris és crucial per a un desenvolupament eficient.

  • app/: Conté el codi principal de la teva aplicació.
  • bootstrap/: Conté l'script d'inici de l'aplicació.
  • config/: Conté els fitxers de configuració.
  • database/: Conté les migracions de la base de dades, fàbriques de models i llavors.
  • public/: El punt d'entrada per a l'aplicació, conté el fitxer index.php.
  • resources/: Conté vistes, recursos en brut (CSS, JS) i fitxers de llenguatge.
  • routes/: Conté totes les definicions de rutes.
  • storage/: Conté plantilles Blade compilades, sessions basades en fitxers, memòries cau de fitxers i altres fitxers generats pel framework.
  • tests/: Conté proves automatitzades.
  • vendor/: Conté les dependències de Composer.

Enrutament Bàsic

L'enrutament en Laravel és senzill i potent. Les rutes es defineixen al fitxer routes/web.php.

Exemple de Ruta

Route::get('/', function () {
    return view('welcome');
});

Aquesta ruta escolta les sol·licituds GET a l'URL arrel (/) i retorna la vista welcome.

Paràmetres de Ruta

Route::get('/user/{id}', function ($id) {
    return 'User '.$id;
});

Aquesta ruta captura un paràmetre ({id}) de l'URL i el passa al tancament.

Controladors

Els controladors són responsables de gestionar les sol·licituds entrants i retornar respostes.

Creant un Controlador

Utilitza l'Artisan CLI per crear un controlador:

php artisan make:controller UserController

Aquesta comanda crea un nou fitxer de controlador al directori app/Http/Controllers.

Exemple de Mètode de Controlador

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class UserController extends Controller
{
    public function show($id)
    {
        return 'User '.$id;
    }
}

Defineix una ruta que utilitza aquest mètode de controlador:

Route::get('/user/{id}', [UserController::class, 'show']);

Eloquent ORM

Eloquent proporciona una sintaxi elegant per interactuar amb la base de dades.

Definint un Model

Crea un model utilitzant Artisan:

php artisan make:model User

Aquesta comanda crea un nou fitxer de model al directori app/Models.

Ús Bàsic d'Eloquent

use App\Models\User;

// Recuperar tots els usuaris
$users = User::all();

// Recuperar un usuari per ID
$user = User::find(1);

// Crear un nou usuari
$user = new User;
$user->name = 'John Doe';
$user->email = 'john@example.com';
$user->save();

Motor de Plantilles Blade

Blade és el potent motor de plantilles de Laravel.

Sintaxi Bàsica de Blade

Crea un fitxer de vista Blade a resources/views/welcome.blade.php:

<!DOCTYPE html>
<html>
<head>
    <title>Laravel</title>
</head>
<body>
    <h1>Hello, {{ $name }}</h1>
</body>
</html>

Passa dades a la vista des d'una ruta o controlador:

Route::get('/', function () {
    return view('welcome', ['name' => 'John Doe']);
});

Conclusió

Laravel és un framework robust i flexible que simplifica el desenvolupament web amb PHP. Entenent els seus conceptes bàsics, configurant l'entorn i explorant les seves característiques com l'enrutament, els controladors, Eloquent ORM i el motor de plantilles Blade, pots construir aplicacions web potents de manera eficient. Continua explorant la documentació extensa de Laravel i els recursos de la comunitat per aprofundir en els teus coneixements i habilitats.
