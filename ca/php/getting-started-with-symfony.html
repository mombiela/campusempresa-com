Symfony és un potent framework PHP que ajuda els desenvolupadors a construir aplicacions web robustes i escalables. Aquesta secció et guiarà a través dels conceptes bàsics de Symfony, des de la instal·lació fins a la creació del teu primer projecte.

Introducció a Symfony

Symfony és un conjunt de components PHP reutilitzables i un framework d'aplicacions web. Té com a objectiu accelerar la creació i el manteniment d'aplicacions web i substituir les tasques de codificació repetitives.

Característiques Clau de Symfony

  • Sistema de Components Modulars: Symfony està compost per components desacoblats i reutilitzables.
  • Flexibilitat: Es pot utilitzar tant per a aplicacions petites com a gran escala.
  • Comunitat i Suport: Symfony té una gran comunitat i una documentació extensa.
  • Rendiment: Optimitzat per a rendiment i escalabilitat.

Instal·lant Symfony

Abans de començar, assegura't de tenir PHP i Composer instal·lats a la teva màquina.

Instal·lació Pas a Pas

  1. Instal·la Symfony CLI: El Symfony CLI és una eina per ajudar-te a gestionar aplicacions Symfony.

    curl -sS https://get.symfony.com/cli/installer | bash

    Afegeix el directori binari de Symfony al teu PATH:

    export PATH="$HOME/.symfony/bin:$PATH"

  2. Crea un Nou Projecte Symfony:

    symfony new my_project --full

    Aquesta comanda crea un nou projecte Symfony en un directori anomenat my_project.

  3. Navega al Teu Directori de Projecte:

    cd my_project

  4. Inicia el Servidor Symfony:

    symfony server:start

    Obre el teu navegador i navega a http://127.0.0.1:8000 per veure la teva nova aplicació Symfony en funcionament.

Estructura de Directoris de Symfony

Entendre l'estructura de directoris és crucial per treballar eficientment amb Symfony.

Directoris Clau

  • src/: Conté el teu codi PHP (controladors, entitats, serveis, etc.).
  • config/: Fitxers de configuració per a la teva aplicació.
  • templates/: Plantilles Twig per a les teves vistes.
  • public/: L'arrel del document del servidor web.
  • var/: Fitxers de memòria cau i registres.
  • vendor/: Dependències de tercers gestionades per Composer.

Creant la Teva Primera Pàgina Symfony

Creem una pàgina senzilla per entendre els conceptes bàsics de routing, controladors i plantilles.

Guia Pas a Pas

  1. Crea un Controlador:

    // src/Controller/DefaultController.php
namespace App\Controller;

use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
use Symfony\Component\HttpFoundation\Response;
use Symfony\Component\Routing\Annotation\Route;

class DefaultController extends AbstractController
{
    /**
     * @Route("/", name="home")
     */
    public function index(): Response
    {
        return $this->render('default/index.html.twig', [
            'message' => 'Hello, Symfony!',
        ]);
    }
}

  2. Crea una Plantilla:

    {# templates/default/index.html.twig #}
<!DOCTYPE html>
<html>
<head>
    <title>Welcome to Symfony</title>
</head>
<body>
    <h1>{{ message }}</h1>
</body>
</html>

  3. Visualitza la Teva Pàgina: Navega a http://127.0.0.1:8000 en el teu navegador. Hauries de veure "Hello, Symfony!".

Conclusió

En aquesta secció, has après els conceptes bàsics per començar amb Symfony, incloent la instal·lació, entendre l'estructura de directoris i crear la teva primera pàgina. Symfony és un framework potent que pot accelerar significativament el teu procés de desenvolupament. A mesura que continuïs explorant Symfony, descobriràs característiques més avançades i millors pràctiques que t'ajudaran a construir aplicacions robustes i mantenibles.
