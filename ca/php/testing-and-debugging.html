Proves i Depuració en PHP
Control de Versions amb Git ►

Les proves i la depuració són habilitats crucials per a qualsevol desenvolupador de PHP. Aquesta secció et guiarà a través dels conceptes bàsics de proves i depuració, des de nivells principiants fins a avançats.

Introducció a les Proves

Les proves són el procés d'avaluar el teu codi per assegurar-te que es comporta com s'espera. Ajuda a identificar errors i verificar que la teva aplicació compleix els seus requisits.

Conceptes Clau en les Proves

  • Proves Unitàries: Provar unitats o components individuals d'un programari.
  • Proves d'Integració: Provar la interacció entre diferents unitats o components.
  • Proves Funcionals: Provar la funcionalitat completa d'una aplicació.
  • Desenvolupament Guiat per Proves (TDD): Escriure proves abans d'escriure el codi real.

Per què les Proves són Importants

  • Assegura la fiabilitat i qualitat del codi.
  • Ajuda a identificar errors aviat.
  • Facilita el manteniment i la refactorització del codi.

Introducció a la Depuració

La depuració és el procés d'identificar, analitzar i eliminar errors del teu codi. És una habilitat essencial per a qualsevol desenvolupador per assegurar el funcionament suau d'una aplicació.

Conceptes Clau en la Depuració

  • Informes d'Errors: Mecanisme integrat de PHP per informar d'errors.
  • Registre: Registrar missatges d'error i altra informació en un fitxer de registre.
  • Punts de Trencament: Pausar l'execució del codi en un punt específic per inspeccionar l'estat.
  • Execució Pas a Pas: Executar el codi línia per línia per entendre el seu comportament.

Nivell Principiant

Proves Bàsiques amb Afirmacions

PHP proporciona maneres senzilles de realitzar proves bàsiques utilitzant afirmacions.

<?php
// Afirmació bàsica
assert(1 + 1 == 2);

// Afirmació personalitzada
function test_addition($a, $b, $expected) {
    assert($a + $b == $expected);
}

test_addition(2, 3, 5);
?>

Depuració Bàsica amb `var_dump` i `print_r`

Utilitzar var_dump i print_r per inspeccionar variables.

<?php
$variable = array("foo" => "bar", 12 => true);
var_dump($variable);
print_r($variable);
?>

Informes d'Errors

Activa els informes d'errors per veure errors en el teu codi PHP.

<?php
error_reporting(E_ALL);
ini_set('display_errors', 1);

// Error intencionat
echo $undefined_variable;
?>

Nivell Intermedi

Proves Unitàries amb PHPUnit

PHPUnit és un marc de proves popular per a PHP.

Instal·lació

composer require --dev phpunit/phpunit

Cas de Prova Bàsic

<?php
use PHPUnit\Framework\TestCase;

class MathTest extends TestCase {
    public function testAddition() {
        $this->assertEquals(4, 2 + 2);
    }
}
?>

Depuració amb Xdebug

Xdebug és una eina poderosa per depurar aplicacions PHP.

Instal·lació

pecl install xdebug

Configuració

Afegeix les següents línies al teu fitxer php.ini:

zend_extension=xdebug.so
xdebug.remote_enable=1
xdebug.remote_host=127.0.0.1
xdebug.remote_port=9000

Ús de Punts de Trencament

Estableix punts de trencament al teu IDE (per exemple, PhpStorm) i executa la teva aplicació. L'execució es pausarà als punts de trencament, permetent-te inspeccionar variables i la pila de trucades.

Nivell Avançat

Desenvolupament Guiat per Proves (TDD)

El TDD és un procés de desenvolupament on escrius proves abans d'escriure el codi real.

Exemple de Flux de Treball

  1. Escriu una prova que falla.
  2. Escriu el codi mínim per passar la prova.
  3. Refactoritza el codi assegurant-te que la prova encara passa.
<?php
use PHPUnit\Framework\TestCase;

class CalculatorTest extends TestCase {
    public function testAddition() {
        $calculator = new Calculator();
        $this->assertEquals(5, $calculator->add(2, 3));
    }
}

class Calculator {
    public function add($a, $b) {
        return $a + $b;
    }
}
?>

Tècniques Avançades de Depuració

Perfilat amb Xdebug

El perfilat ajuda a identificar colls d'ampolla de rendiment en la teva aplicació.

xdebug.profiler_enable=1
xdebug.profiler_output_dir=/tmp

Depuració Remota

Configura Xdebug per a la depuració remota per depurar aplicacions que s'executen en un servidor.

xdebug.remote_enable=1
xdebug.remote_host=your.remote.host
xdebug.remote_port=9000

Conclusió

Les proves i la depuració són habilitats essencials per als desenvolupadors de PHP. En dominar aquestes tècniques, pots assegurar-te que el teu codi és fiable, mantenible i té un bon rendiment. Comença amb proves i depuració bàsiques, i gradualment avança cap a tècniques més avançades com el TDD i el perfilat per convertir-te en un desenvolupador de PHP proficient.
