Connectar-se a una Base de Dades MySQL
Realització d'Operacions CRUD ►

Connectar-se a una base de dades MySQL és una habilitat fonamental per a qualsevol desenvolupador PHP. Aquest tema et guiarà a través del procés, des de la configuració del teu entorn fins a l'execució de consultes bàsiques.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Una instal·lació funcional de PHP.
  • Un servidor MySQL en funcionament i accessible.
  • Coneixements bàsics de la sintaxi de PHP.

Configuració de l'Entorn

Per connectar-te a una base de dades MySQL utilitzant PHP, necessites tenir l'extensió mysqli habilitada. Aquesta extensió està inclosa en PHP per defecte, però hauries de verificar que està habilitada en el teu fitxer de configuració php.ini.

; Assegura't que la següent línia no estigui comentada en php.ini
extension=mysqli

Establir una Connexió

Utilitzant `mysqli`

L'extensió mysqli proporciona una manera senzilla de connectar-se a una base de dades MySQL. Aquí tens com fer-ho:

  1. Crear una Connexió: Utilitza la funció mysqli_connect per establir una connexió.
  2. Verificar la Connexió: Verifica si la connexió ha estat exitosa.
<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "test_db";

// Crear connexió
$conn = mysqli_connect($servername, $username, $password, $dbname);

// Verificar connexió
if (!$conn) {
    die("Connexió fallida: " . mysqli_connect_error());
}
echo "Connexió exitosa";
?>

Utilitzant PDO (PHP Data Objects)

PDO és una manera més flexible i segura de connectar-se a una base de dades. Suporta múltiples tipus de bases de dades i proporciona una interfície consistent.

  1. Crear una Instància de PDO: Utilitza el constructor new PDO.
  2. Gestionar Excepcions: Utilitza blocs try-catch per gestionar errors de connexió.
<?php
$dsn = 'mysql:host=localhost;dbname=test_db';
$username = 'root';
$password = '';

try {
    $pdo = new PDO($dsn, $username, $password);
    // Establir el mode d'error de PDO a excepció
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connexió exitosa";
} catch (PDOException $e) {
    echo "Connexió fallida: " . $e->getMessage();
}
?>

Executar Consultes

Utilitzant `mysqli`

Un cop connectat, pots executar consultes SQL utilitzant la funció mysqli_query.

<?php
$sql = "SELECT id, firstname, lastname FROM Users";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    // Sortir dades de cada fila
    while($row = mysqli_fetch_assoc($result)) {
        echo "id: " . $row["id"]. " - Nom: " . $row["firstname"]. " " . $row["lastname"]. "<br>";
    }
} else {
    echo "0 resultats";
}
mysqli_close($conn);
?>

Utilitzant PDO

Amb PDO, pots preparar i executar declaracions, cosa que és més segura i eficient.

<?php
$sql = 'SELECT id, firstname, lastname FROM Users';
$stmt = $pdo->query($sql);

while ($row = $stmt->fetch(PDO::FETCH_ASSOC)) {
    echo "id: " . $row['id'] . " - Nom: " . $row['firstname'] . " " . $row['lastname'] . "<br>";
}
?>

Gestió d'Errors

`mysqli`

Comprova si hi ha errors després d'executar consultes.

<?php
if (!mysqli_query($conn, $sql)) {
    echo "Error: " . $sql . "<br>" . mysqli_error($conn);
}
?>

PDO

Utilitza excepcions per gestionar errors.

<?php
try {
    $stmt = $pdo->query($sql);
} catch (PDOException $e) {
    echo "Consulta fallida: " . $e->getMessage();
}
?>

Tancar la Connexió

`mysqli`

Sempre tanca la connexió quan hagis acabat.

<?php
mysqli_close($conn);
?>

PDO

Les connexions PDO es tanquen automàticament quan l'objecte és destruït, però pots establir explícitament l'objecte PDO a null.

<?php
$pdo = null;
?>

Resum

Connectar-se a una base de dades MySQL en PHP es pot fer utilitzant l'extensió mysqli o PDO. Mentre que mysqli és senzill i fàcil d'utilitzar, PDO ofereix més flexibilitat i seguretat, especialment per a projectes més grans. Assegura't sempre de gestionar els errors de manera adequada i tancar les connexions quan ja no siguin necessàries.
