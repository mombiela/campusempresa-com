Introducció a la POO

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Simplifica problemes complexos descomponent-los en objectes més petits i manejables.

Conceptes Clau de la POO

  • Classe: Un plànol per crear objectes (una estructura de dades particular).
  • Objecte: Una instància d'una classe.
  • Herència: Un mecanisme on una nova classe hereta propietats i comportaments (mètodes) d'una altra classe.
  • Encapsulació: Agrupar les dades (variables) i els mètodes (funcions) que operen sobre les dades en una sola unitat o classe.
  • Polimorfisme: La capacitat de presentar la mateixa interfície per a diferents tipus de dades subjacents.
  • Abstracció: Amagar els detalls complexos de la implementació i mostrar només les característiques necessàries d'un objecte.

Classes i Objectes

Definició d'una Classe

Una classe en PHP es defineix utilitzant la paraula clau class.

<?php
class Car {
    // Propietats
    public $color;
    public $model;

    // Mètodes
    public function __construct($color, $model) {
        $this->color = $color;
        $this->model = $model;
    }

    public function message() {
        return "El meu cotxe és un " . $this->color . " " . $this->model . ".";
    }
}
?>

Creació d'un Objecte

Un objecte es crea utilitzant la paraula clau new.

<?php
$myCar = new Car("negre", "Volvo");
echo $myCar->message();
?>

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

<?php
class ElectricCar extends Car {
    public $batteryCapacity;

    public function __construct($color, $model, $batteryCapacity) {
        parent::__construct($color, $model);
        $this->batteryCapacity = $batteryCapacity;
    }

    public function batteryStatus() {
        return "La capacitat de la bateria és de " . $this->batteryCapacity . " kWh.";
    }
}

$myElectricCar = new ElectricCar("vermell", "Tesla", 100);
echo $myElectricCar->message();
echo $myElectricCar->batteryStatus();
?>

Encapsulació

L'encapsulació s'aconsegueix utilitzant modificadors d'accés: public, protected i private.

<?php
class Person {
    private $name;
    private $age;

    public function setName($name) {
        $this->name = $name;
    }

    public function getName() {
        return $this->name;
    }

    public function setAge($age) {
        if($age > 0) {
            $this->age = $age;
        }
    }

    public function getAge() {
        return $this->age;
    }
}

$person = new Person();
$person->setName("John");
$person->setAge(30);
echo $person->getName();
echo $person->getAge();
?>

Polimorfisme

El polimorfisme permet que els mètodes facin diferents coses segons l'objecte sobre el qual actuen.

<?php
class Animal {
    public function makeSound() {
        echo "Algún so genèric d'animal";
    }
}

class Dog extends Animal {
    public function makeSound() {
        echo "Bup";
    }
}

class Cat extends Animal {
    public function makeSound() {
        echo "Mèu";
    }
}

$animals = [new Dog(), new Cat(), new Animal()];

foreach ($animals as $animal) {
    $animal->makeSound();
    echo "\n";
}
?>

Abstracció

L'abstracció s'aconsegueix utilitzant classes abstractes i interfícies.

Classe Abstracta

<?php
abstract class Shape {
    abstract protected function getArea();
}

class Circle extends Shape {
    private $radius;

    public function __construct($radius) {
        $this->radius = $radius;
    }

    public function getArea() {
        return pi() * pow($this->radius, 2);
    }
}

$circle = new Circle(5);
echo $circle->getArea();
?>

Interfície

<?php
interface Logger {
    public function log($message);
}

class FileLogger implements Logger {
    public function log($message) {
        echo "Registrant missatge en un fitxer: " . $message;
    }
}

$logger = new FileLogger();
$logger->log("S'ha produït un error.");
?>

Conclusió

La Programació Orientada a Objectes en PHP proporciona un marc robust per construir aplicacions escalables i mantenibles. Entenent i aplicant els principis de classes, objectes, herència, encapsulació, polimorfisme i abstracció, pots crear aplicacions complexes que siguin més fàcils de gestionar i ampliar. Practica aquests conceptes amb exemples del món real per aprofundir en la teva comprensió i competència en POO amb PHP.
