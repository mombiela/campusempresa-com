Càrregues de Fitxers en PHP
Introducció a les Bases de Dades ►

Introducció

Les càrregues de fitxers són una característica comuna en les aplicacions web, permetent als usuaris carregar fitxers com imatges, documents i vídeos al servidor. PHP proporciona un suport robust per gestionar les càrregues de fitxers, fent que sigui fàcil implementar aquesta funcionalitat en les teves aplicacions web.

Conceptes Bàsics

Abans de submergir-nos en el codi, cobrim alguns conceptes clau relacionats amb les càrregues de fitxers en PHP:

  • Formulari HTML: L'element form en HTML s'utilitza per recollir l'entrada de l'usuari. Per a les càrregues de fitxers, el formulari ha d'incloure l'atribut enctype="multipart/form-data".
  • Superglobal $_FILES: PHP utilitza l'array superglobal $_FILES per emmagatzemar informació sobre els fitxers carregats.
  • Emmagatzematge Temporal: Els fitxers carregats s'emmagatzemen temporalment al servidor abans de ser moguts a una ubicació permanent.

Creació d'un Formulari HTML per a Càrregues de Fitxers

Per permetre als usuaris carregar fitxers, necessites crear un formulari HTML amb els atributs apropiats.

<form action="upload.php" method="post" enctype="multipart/form-data">
    <label for="fileToUpload">Selecciona el fitxer a carregar:</label>
    <input type="file" name="fileToUpload" id="fileToUpload">
    <input type="submit" value="Carregar Fitxer" name="submit">
</form>
  • action="upload.php": Especifica el script PHP que gestionarà la càrrega del fitxer.
  • method="post": Utilitza el mètode POST per enviar les dades del formulari.
  • enctype="multipart/form-data": Assegura que el formulari pugui gestionar càrregues de fitxers.

Gestió de Càrregues de Fitxers en PHP

En el script upload.php, gestionaràs el procés de càrrega del fitxer. Això implica comprovar errors, validar el fitxer i moure'l a una ubicació permanent.

Procés Pas a Pas

  1. Comprovar Errors: Assegurar-se que no hi hagi errors durant la càrrega del fitxer.
  2. Validar el Fitxer: Comprovar el tipus i la mida del fitxer per assegurar-se que compleix els teus criteris.
  3. Moure el Fitxer: Moure el fitxer de la ubicació temporal a un directori permanent.

Exemple de Codi

<?php
if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $target_dir = "uploads/";
    $target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
    $uploadOk = 1;
    $fileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

    // Comprovar si el fitxer es va carregar sense errors
    if ($_FILES["fileToUpload"]["error"] != 0) {
        echo "Error: " . $_FILES["fileToUpload"]["error"] . "<br>";
        $uploadOk = 0;
    }

    // Comprovar la mida del fitxer (límit de 5MB)
    if ($_FILES["fileToUpload"]["size"] > 5000000) {
        echo "Ho sentim, el teu fitxer és massa gran.";
        $uploadOk = 0;
    }

    // Permetre certs formats de fitxer
    $allowedTypes = array("jpg", "png", "jpeg", "gif");
    if (!in_array($fileType, $allowedTypes)) {
        echo "Ho sentim, només es permeten fitxers JPG, JPEG, PNG i GIF.";
        $uploadOk = 0;
    }

    // Comprovar si $uploadOk està establert a 0 per un error
    if ($uploadOk == 0) {
        echo "Ho sentim, el teu fitxer no es va carregar.";
    } else {
        if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
            echo "El fitxer " . htmlspecialchars(basename($_FILES["fileToUpload"]["name"])) . " s'ha carregat.";
        } else {
            echo "Ho sentim, hi va haver un error en carregar el teu fitxer.";
        }
    }
}
?>

Explicació

  • Comprovar Errors: Es comprova la propietat $_FILES["fileToUpload"]["error"] per assegurar-se que no hi hagi errors.
  • Validar el Fitxer: Es valida la mida i el tipus del fitxer per assegurar-se que compleixen els criteris especificats.
  • Moure el Fitxer: Es fa servir la funció move_uploaded_file() per moure el fitxer de la ubicació temporal al directori de destinació.

Temes Avançats

Càrregues de Fitxers Múltiples

Per permetre als usuaris carregar múltiples fitxers, modifica el formulari HTML i gestiona els fitxers en el script PHP corresponent.

Formulari HTML per a Fitxers Múltiples

<form action="upload.php" method="post" enctype="multipart/form-data">
    <label for="filesToUpload">Selecciona els fitxers a carregar:</label>
    <input type="file" name="filesToUpload[]" id="filesToUpload" multiple>
    <input type="submit" value="Carregar Fitxers" name="submit">
</form>

Script PHP per Gestionar Fitxers Múltiples

<?php
if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $target_dir = "uploads/";
    $uploadOk = 1;
    $allowedTypes = array("jpg", "png", "jpeg", "gif");

    foreach ($_FILES["filesToUpload"]["name"] as $key => $name) {
        $target_file = $target_dir . basename($name);
        $fileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

        // Comprovar si el fitxer es va carregar sense errors
        if ($_FILES["filesToUpload"]["error"][$key] != 0) {
            echo "Error: " . $_FILES["filesToUpload"]["error"][$key] . "<br>";
            $uploadOk = 0;
        }

        // Comprovar la mida del fitxer (límit de 5MB)
        if ($_FILES["filesToUpload"]["size"][$key] > 5000000) {
            echo "Ho sentim, el teu fitxer " . $name . " és massa gran.<br>";
            $uploadOk = 0;
        }

        // Permetre certs formats de fitxer
        if (!in_array($fileType, $allowedTypes)) {
            echo "Ho sentim, només es permeten fitxers JPG, JPEG, PNG i GIF per al fitxer " . $name . ".<br>";
            $uploadOk = 0;
        }

        // Comprovar si $uploadOk està establert a 0 per un error
        if ($uploadOk == 0) {
            echo "Ho sentim, el teu fitxer " . $name . " no es va carregar.<br>";
        } else {
            if (move_uploaded_file($_FILES["filesToUpload"]["tmp_name"][$key], $target_file)) {
                echo "El fitxer " . htmlspecialchars(basename($name)) . " s'ha carregat.<br>";
            } else {
                echo "Ho sentim, hi va haver un error en carregar el teu fitxer " . $name . ".<br>";
            }
        }
    }
}
?>

Consideracions de Seguretat

Quan es gestionen càrregues de fitxers, és crucial considerar la seguretat per prevenir que es carreguin fitxers maliciosos.

  • Validació del Tipus de Fitxer: Sempre valida el tipus de fitxer per assegurar-te que només es carreguin els tipus permesos.
  • Límit de Mida del Fitxer: Estableix una mida màxima per evitar que fitxers grans consumeixin recursos del servidor.
  • Neteja de Noms de Fitxer: Utilitza funcions com basename() i htmlspecialchars() per netejar els noms de fitxer i prevenir atacs de travessia de directoris.
  • Emmagatzema Fitxers Fora de l'Arrel Web: Emmagatzema els fitxers carregats fora de l'arrel web per prevenir l'accés directe via URL.

Conclusió

Les càrregues de fitxers són una característica poderosa en PHP que permet als usuaris carregar fitxers al teu servidor. Entenent els conceptes bàsics i implementant mesures de validació i seguretat adequades, pots gestionar efectivament les càrregues de fitxers en les teves aplicacions web. Tant si estàs tractant amb càrregues de fitxers individuals com múltiples, PHP proporciona les eines necessàries per gestionar aquesta funcionalitat de manera eficient.
