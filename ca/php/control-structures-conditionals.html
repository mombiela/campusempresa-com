Introducció

Les estructures de control són fonamentals en la programació, ja que permeten als desenvolupadors dictar el flux d'execució basat en certes condicions. En PHP, els condicionals et permeten executar diferents blocs de codi basats en criteris específics. Aquesta secció cobrirà des dels conceptes bàsics fins a l'ús avançat dels condicionals en PHP.

Condicionals Bàsics

Instrucció if

La instrucció if és la forma més simple de condicional. Executa un bloc de codi només si una condició especificada és certa.

$age = 20;

if ($age >= 18) {
    echo "You are an adult.";
}
  • Explicació: Si $age és major o igual a 18, es mostrarà el missatge "You are an adult.".

Instrucció if-else

La instrucció if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

$age = 16;

if ($age >= 18) {
    echo "You are an adult.";
} else {
    echo "You are not an adult.";
}
  • Explicació: Si $age és menor de 18, es mostrarà el missatge "You are not an adult.".

Instrucció if-elseif-else

La instrucció if-elseif-else permet comprovar múltiples condicions en seqüència.

$score = 85;

if ($score >= 90) {
    echo "Grade: A";
} elseif ($score >= 80) {
    echo "Grade: B";
} elseif ($score >= 70) {
    echo "Grade: C";
} else {
    echo "Grade: F";
}
  • Explicació: Depenent del valor de $score, es mostrarà una nota diferent.

Condicionals Avançats

Condicionals Niats

Els condicionals es poden niar dins d'altres per gestionar lògiques més complexes.

$age = 20;
$hasID = true;

if ($age >= 18) {
    if ($hasID) {
        echo "You can enter the club.";
    } else {
        echo "You need an ID to enter.";
    }
} else {
    echo "You are not old enough to enter.";
}
  • Explicació: El codi comprova si la persona és prou gran i té un ID abans de permetre l'entrada.

Operador Ternari

L'operador ternari és una forma abreujada de la instrucció if-else.

$age = 20;
$message = ($age >= 18) ? "You are an adult." : "You are not an adult.";
echo $message;
  • Explicació: L'operador ternari comprova la condició i assigna el missatge apropiat a $message.

Instrucció Switch

La instrucció switch és útil per comparar una variable amb múltiples valors.

$day = "Tuesday";

switch ($day) {
    case "Monday":
        echo "Start of the work week.";
        break;
    case "Tuesday":
        echo "Second day of the work week.";
        break;
    case "Wednesday":
        echo "Midweek.";
        break;
    case "Thursday":
        echo "Almost the weekend.";
        break;
    case "Friday":
        echo "Last workday of the week.";
        break;
    default:
        echo "Weekend!";
}
  • Explicació: La instrucció switch comprova el valor de $day i mostra el missatge corresponent.

Conclusió

Entendre i utilitzar els condicionals de manera efectiva és crucial per controlar el flux dels teus programes en PHP. Des de les instruccions if bàsiques fins a estructures més complexes com els condicionals niats i les instruccions switch, dominar aquests conceptes et permetrà escriure codi més dinàmic i responsiu. Practica aquests exemples i intenta crear les teves pròpies instruccions condicionals per consolidar el teu enteniment.
