Ús de Sentències Preparades
Programació Orientada a Objectes (OOP) ►

Les sentències preparades són una característica poderosa en PHP que ajuda a prevenir atacs d'injecció SQL i millorar l'eficiència de les operacions de base de dades. Aquest tema et guiarà a través dels conceptes bàsics de l'ús de sentències preparades en PHP, des de nivell principiant fins a avançat.

Què són les Sentències Preparades?

Les sentències preparades són una manera d'executar consultes SQL de manera segura i eficient. Consten de dues etapes principals:

  1. Preparació: La plantilla de la consulta SQL s'envia a la base de dades amb espais reservats per als paràmetres.
  2. Execució: Els valors reals s'envien a la base de dades, substituint els espais reservats.

Beneficis Clau

  • Seguretat: Prevé atacs d'injecció SQL separant la lògica SQL de les dades.
  • Eficiència: La base de dades pot optimitzar el pla d'execució de la consulta.

Ús Bàsic de les Sentències Preparades

Connexió a la Base de Dades

Abans d'utilitzar sentències preparades, necessites establir una connexió amb la base de dades. Aquí tens un exemple senzill utilitzant PDO (PHP Data Objects):

<?php
$dsn = 'mysql:host=localhost;dbname=testdb';
$username = 'root';
$password = '';

try {
    $pdo = new PDO($dsn, $username, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    echo "Connexió establerta correctament";
} catch (PDOException $e) {
    echo "Connexió fallida: " . $e->getMessage();
}
?>

Preparació i Execució d'una Sentència

Aquí tens com pots preparar i executar una consulta SQL senzilla utilitzant espais reservats:

<?php
$sql = "SELECT * FROM users WHERE email = :email";
$stmt = $pdo->prepare($sql);
$email = 'example@example.com';
$stmt->bindParam(':email', $email);
$stmt->execute();
$result = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($result as $row) {
    echo $row['name'] . "<br>";
}
?>

Explicació

  • Preparació: La consulta SQL amb un espai reservat (:email) es prepara.
  • Assignació: El valor real s'assigna a l'espai reservat.
  • Execució: La consulta s'executa amb el valor assignat.

Ús Avançat de les Sentències Preparades

Ús de Múltiples Espais Reservats

Pots utilitzar múltiples espais reservats en una sola consulta:

<?php
$sql = "SELECT * FROM users WHERE email = :email AND status = :status";
$stmt = $pdo->prepare($sql);
$email = 'example@example.com';
$status = 'active';
$stmt->bindParam(':email', $email);
$stmt->bindParam(':status', $status);
$stmt->execute();
$result = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($result as $row) {
    echo $row['name'] . "<br>";
}
?>

Ús d'Espais Reservats Nominals vs. Posicionals

  • Espais Reservats Nominals: Utilitza espais reservats nominals com :email.
  • Espais Reservats Posicionals: Utilitza signes d'interrogació (?) com espais reservats.

Exemple amb espais reservats posicionals:

<?php
$sql = "SELECT * FROM users WHERE email = ? AND status = ?";
$stmt = $pdo->prepare($sql);
$email = 'example@example.com';
status = 'active';
$stmt->execute([$email, $status]);
$result = $stmt->fetchAll(PDO::FETCH_ASSOC);

foreach ($result as $row) {
    echo $row['name'] . "<br>";
}
?>

Gestió d'Errors

Sempre gestiona els possibles errors quan treballis amb bases de dades:

<?php
try {
    $sql = "SELECT * FROM users WHERE email = :email";
    $stmt = $pdo->prepare($sql);
    $email = 'example@example.com';
    $stmt->bindParam(':email', $email);
    $stmt->execute();
    $result = $stmt->fetchAll(PDO::FETCH_ASSOC);

    foreach ($result as $row) {
        echo $row['name'] . "<br>";
    }
} catch (PDOException $e) {
    echo "Error: " . $e->getMessage();
}
?>

Resum

Les sentències preparades en PHP ofereixen una manera segura i eficient d'executar consultes SQL. Separant la lògica SQL de les dades, ajuden a prevenir atacs d'injecció SQL i permeten que la base de dades optimitzi l'execució de les consultes. Tant si utilitzes espais reservats nominals com posicionals, els passos clau impliquen preparar la sentència, assignar els paràmetres i executar la consulta. Recorda sempre gestionar els possibles errors per assegurar que la teva aplicació funcioni sense problemes.

Dominant les sentències preparades, pots millorar significativament la seguretat i el rendiment de les teves aplicacions PHP.
