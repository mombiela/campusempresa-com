Configuració de l'Entorn de Desenvolupament
El teu Primer Script PHP ►

Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre PHP. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de les eines necessàries per començar a programar en PHP.

Prerequisits

Abans de configurar el teu entorn de desenvolupament PHP, assegura't de tenir el següent:

  • Un ordinador amb un sistema operatiu (Windows, macOS o Linux).
  • Una connexió a internet per descarregar el programari necessari.

Instal·lació de PHP

Windows

  1. Descarregar PHP:

    • Visita el lloc web oficial de PHP i descarrega l'última versió de PHP per a Windows.

  2. Instal·lar PHP:

    • Extreu el fitxer ZIP descarregat a un directori de la teva elecció (per exemple, C:\php).
    • Afegeix el directori de PHP a la variable d'entorn PATH del teu sistema:
      • Obre el Menú d'Inici, cerca "Variables d'entorn" i selecciona "Edita les variables d'entorn del sistema."
      • Fes clic a "Variables d'entorn" i troba la variable "Path" sota Variables del sistema.
      • Fes clic a "Edita," després "Nou," i afegeix el camí al teu directori de PHP (per exemple, C:\php).
    • Obre el Símbol del sistema i escriu php -v per verificar la instal·lació.

macOS

  1. Utilitzant Homebrew:
    • Obre Terminal i instal·la Homebrew si encara no ho has fet: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la PHP utilitzant Homebrew: 
      brew install php
    • Verifica la instal·lació escrivint php -v a Terminal.

Linux

  1. Utilitzant el Gestor de Paquets:
    • Obre Terminal i actualitza la teva llista de paquets: 
      sudo apt update
    • Instal·la PHP: 
      sudo apt install php
    • Verifica la instal·lació escrivint php -v a Terminal.

Instal·lació d'un Servidor Web

PHP funciona millor amb un servidor web. L'elecció més comuna és Apache, però també pots utilitzar Nginx o altres servidors. Per simplicitat, utilitzarem XAMPP, que inclou Apache, MySQL i PHP.

Instal·lació de XAMPP

  1. Descarregar XAMPP:

    • Visita el lloc web de XAMPP i descarrega la versió adequada per al teu sistema operatiu.

  2. Instal·lar XAMPP:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.
    • Inicia el Panell de Control de XAMPP i inicia els mòduls Apache i MySQL.

  3. Verificar la Instal·lació:

    • Obre el teu navegador web i ves a http://localhost. Hauries de veure la pàgina de benvinguda de XAMPP.

Configuració d'un Editor de Codi

Un bon editor de codi pot millorar significativament la teva productivitat. Algunes opcions populars per al desenvolupament en PHP són:

  • Visual Studio Code:

    • Descarrega des de Visual Studio Code.
    • Instal·la extensions de PHP com "PHP Intelephense" per a una millor autocompleció i linting.

  • PHPStorm:

    • Un IDE potent dissenyat específicament per al desenvolupament en PHP.
    • Descarrega des de JetBrains.

  • Sublime Text:

    • Descarrega des de Sublime Text.
    • Instal·la paquets de PHP via Package Control per a una funcionalitat millorada.

Configuració de l'Entorn de Desenvolupament

Configuració de PHP

  1. Fitxer php.ini:
    • Localitza el teu fitxer php.ini (normalment al directori de PHP o al directori d'instal·lació de XAMPP).
    • Obre'l en el teu editor de codi i configura ajustos com:
      • error_reporting = E_ALL
      • display_errors = On
      • max_execution_time = 30

Configuració d'Apache

  1. Fitxer httpd.conf:
    • Localitza el fitxer httpd.conf al directori d'Apache (per exemple, C:\xampp\apache\conf).
    • Obre'l en el teu editor de codi i assegura't que les següents línies estiguin descomentades: 
      LoadModule php_module "C:/php/php7apache2_4.dll"
AddType application/x-httpd-php .php
PHPIniDir "C:/php"

Provar la Teva Configuració

  1. Crear un Fitxer PHP:
    • Al directori arrel del teu servidor web (per exemple, C:\xampp\htdocs), crea un fitxer anomenat info.php.
    • Afegeix el següent codi: 
      <?php
phpinfo();
?>
    • Desa el fitxer i obre el teu navegador web. Navega a http://localhost/info.php. Hauries de veure la pàgina d'informació de PHP.

Conclusió

Configurar el teu entorn de desenvolupament PHP implica instal·lar PHP, un servidor web i un editor de codi. Una configuració adequada assegura que puguis escriure, provar i depurar el teu codi PHP de manera eficient. Amb el teu entorn llest, ara estàs preparat per submergir-te en la programació en PHP i començar a construir aplicacions web dinàmiques.
