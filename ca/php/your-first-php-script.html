Introducció al PHP

PHP (Hypertext Preprocessor) és un llenguatge de script del costat del servidor popular dissenyat per al desenvolupament web, però també utilitzat com a llenguatge de programació de propòsit general. En aquesta secció, et guiarem a través de l'escriptura del teu primer script PHP.

Configurant el teu entorn

Abans d'escriure el teu primer script PHP, assegura't de tenir el següent:

  • Un servidor web (p. ex., Apache, Nginx)
  • PHP instal·lat al teu servidor
  • Un editor de text (p. ex., VS Code, Sublime Text)

Instal·lant PHP

Si encara no has instal·lat PHP, pots descarregar-lo des del lloc web oficial de PHP. Segueix les instruccions d'instal·lació per al teu sistema operatiu.

Escriure el teu primer script PHP

Anem a escriure un simple script PHP.

Creant un fitxer PHP

  1. Obre el teu editor de text.
  2. Crea un nou fitxer i desa'l com index.php.

Sintaxi bàsica de PHP

El codi PHP s'integra dins de l'HTML utilitzant les etiquetes <?php ... ?>.

<!DOCTYPE html>
<html>
<head>
    <title>El meu primer script PHP</title>
</head>
<body>
    <?php
        echo "Hola, món!";
    ?>
</body>
</html>

Explicació

  • <!DOCTYPE html>: Declara el tipus de document com HTML5.
  • <html>, <head>, <title>, <body>: Etiquetes HTML estàndard.
  • <?php ... ?>: Bloc de codi PHP.
  • echo "Hola, món!";: Sentència PHP per mostrar text.

Executant el teu script PHP

  1. Col·loca index.php al directori arrel del teu servidor web (p. ex., htdocs per a XAMPP, www per a WAMP).
  2. Inicia el teu servidor web.
  3. Obre un navegador web i navega a http://localhost/index.php.

Hauries de veure "Hola, món!" mostrat a la pàgina.

Conceptes clau

  • Etiquetes PHP: El codi PHP s'escriu dins de les etiquetes <?php ... ?>.
  • Sentència Echo: Utilitzada per mostrar text o variables.
  • Comentaris: Utilitza // per a comentaris d'una sola línia i /* ... */ per a comentaris de múltiples línies.

Exemple amb comentaris

<!DOCTYPE html>
<html>
<head>
    <title>El meu primer script PHP amb comentaris</title>
</head>
<body>
    <?php
        // Aquest és un comentari d'una sola línia
        echo "Hola, món!"; // Això mostrarà Hola, món!
        
        /* Aquest és un comentari de múltiples línies
           Pot abastar diverses línies */
        echo "Aprendre PHP és divertit!";
    ?>
</body>
</html>

Errors comuns i resolució de problemes

  • Errors de sintaxi: Assegura't que totes les sentències PHP acabin amb un punt i coma (;).
  • Configuració del servidor: Assegura't que el teu servidor web estigui en funcionament i que PHP estigui correctament instal·lat.
  • Col·locació del fitxer: Assegura't que el teu fitxer PHP estigui al directori correcte per al teu servidor web.

Resum

En aquesta secció, has après a:

  • Configurar el teu entorn per al desenvolupament en PHP.
  • Escriure i executar un script PHP bàsic.
  • Entendre la sintaxi bàsica de PHP i els conceptes clau.

En dominar aquests conceptes bàsics, estàs en bon camí per convertir-te en un expert en PHP. A continuació, explorarem temes i funcionalitats més avançades en PHP.
