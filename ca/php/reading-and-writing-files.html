Introducció

Llegir i escriure fitxers és una habilitat fonamental en PHP que et permet gestionar l'emmagatzematge de dades, fitxers de configuració i més. Aquesta secció cobrirà els conceptes bàsics de la gestió de fitxers en PHP, incloent l'obertura, lectura, escriptura i tancament de fitxers.

Obrir i Tancar Fitxers

Per treballar amb fitxers en PHP, primer has d'obrir-los. PHP proporciona diverses funcions per a aquest propòsit.

Obrir un Fitxer

  • fopen(): Obre un fitxer o URL.
$file = fopen("example.txt", "r"); // Obre el fitxer en mode lectura

Tancar un Fitxer

  • fclose(): Tanca un punter de fitxer obert.
fclose($file); // Tanca el fitxer

Llegir Fitxers

PHP ofereix múltiples funcions per llegir fitxers, depenent de les teves necessitats.

Llegir un Fitxer Línia per Línia

  • fgets(): Llegeix una sola línia d'un fitxer.
$file = fopen("example.txt", "r");
while (!feof($file)) {
    $line = fgets($file);
    echo $line;
}
fclose($file);

Llegir el Fitxer Sencer

  • file_get_contents(): Llegeix tot el fitxer en una cadena.
$content = file_get_contents("example.txt");
echo $content;

Llegir un Fitxer en un Array

  • file(): Llegeix tot el fitxer en un array, amb cada línia com un element de l'array.
$lines = file("example.txt");
foreach ($lines as $line) {
    echo $line;
}

Escriure Fitxers

Escriure fitxers és tan important com llegir-los. PHP proporciona diverses funcions per escriure dades en fitxers.

Escriure en un Fitxer

  • fwrite(): Escriu dades en un fitxer.
$file = fopen("example.txt", "w"); // Obre el fitxer en mode escriptura
fwrite($file, "Hello, World!");
fclose($file);

Afegir a un Fitxer

  • fwrite(): També es pot utilitzar per afegir dades si el fitxer s'obre en mode afegir.
$file = fopen("example.txt", "a"); // Obre el fitxer en mode afegir
fwrite($file, "Appending this text.");
fclose($file);

Escriure el Fitxer Sencer

  • file_put_contents(): Escriu dades en un fitxer, reemplaçant el seu contingut.
file_put_contents("example.txt", "New content");

Modes de Fitxer

Quan obres un fitxer, pots especificar el mode en què vols obrir-lo. Aquí tens alguns modes comuns:

| Mode | Descripció | |------|------------------------------| | r | Només lectura | | w | Només escriptura; trunca el fitxer | | a | Només escriptura; afegeix al fitxer | | r+ | Lectura i escriptura | | w+ | Lectura i escriptura; trunca el fitxer | | a+ | Lectura i escriptura; afegeix al fitxer |

Gestió d'Errors

És important gestionar els errors quan treballes amb fitxers per assegurar-te que la teva aplicació funcioni sense problemes.

Comprovar si un Fitxer Existeix

  • file_exists(): Comprova si un fitxer o directori existeix.
if (file_exists("example.txt")) {
    echo "El fitxer existeix.";
} else {
    echo "El fitxer no existeix.";
}

Gestionar Errors d'Obertura de Fitxers

  • Utilitzar @ per suprimir errors i die() per gestionar-los.
$file = @fopen("example.txt", "r") or die("No es pot obrir el fitxer!");

Conclusió

Llegir i escriure fitxers en PHP és una habilitat crucial per a qualsevol desenvolupador. Entenent com obrir, llegir, escriure i tancar fitxers, pots gestionar l'emmagatzematge de dades i la configuració de manera més efectiva. Recorda sempre gestionar els errors de manera adequada per assegurar-te que la teva aplicació sigui robusta.

Amb aquests conceptes bàsics, estàs ben preparat per començar a treballar amb fitxers en PHP. A mesura que avancis, pots explorar operacions de fitxers més complexes i tècniques de gestió d'errors.
