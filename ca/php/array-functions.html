Introducció als Arrays en PHP

Els arrays són un dels tipus de dades més potents en PHP. Et permeten emmagatzemar múltiples valors en una sola variable, que es poden accedir utilitzant índexs o claus.

Conceptes Clau

  • Arrays Indexats: Arrays amb índexs numèrics.
  • Arrays Associatius: Arrays amb claus nominals.
  • Arrays Multidimensionals: Arrays que contenen un o més arrays.

Exemple

// Array Indexat
$fruits = array("Poma", "Plàtan", "Cirera");

// Array Associatiu
$ages = array("Pere" => 35, "Ben" => 37, "Josep" => 43);

// Array Multidimensional
$contacts = array(
    array("nom" => "Pere", "correu" => "pere@example.com"),
    array("nom" => "Ben", "correu" => "ben@example.com")
);

Funcions Bàsiques d'Array

`count()`

La funció count() retorna el nombre d'elements en un array.

$fruits = array("Poma", "Plàtan", "Cirera");
echo count($fruits); // Sortida: 3

`array_push()`

La funció array_push() afegeix un o més elements al final d'un array.

$fruits = array("Poma", "Plàtan");
array_push($fruits, "Cirera", "Dàtil");
print_r($fruits); // Sortida: Array ( [0] => Poma [1] => Plàtan [2] => Cirera [3] => Dàtil )

`array_pop()`

La funció array_pop() elimina l'últim element d'un array.

$fruits = array("Poma", "Plàtan", "Cirera");
array_pop($fruits);
print_r($fruits); // Sortida: Array ( [0] => Poma [1] => Plàtan )

Funcions Intermèdies d'Array

`array_merge()`

La funció array_merge() combina un o més arrays en un sol array.

$array1 = array("color" => "vermell", 2, 4);
$array2 = array("a", "b", "color" => "verd", "forma" => "trapezi", 4);
$result = array_merge($array1, $array2);
print_r($result);
// Sortida: Array ( [color] => verd [0] => 2 [1] => 4 [2] => a [3] => b [forma] => trapezi [4] => 4 )

`array_diff()`

La funció array_diff() compara arrays i retorna les diferències.

$array1 = array("a" => "verd", "vermell", "blau", "vermell");
$array2 = array("b" => "verd", "groc", "vermell");
$result = array_diff($array1, $array2);
print_r($result); // Sortida: Array ( [1] => blau )

`array_keys()`

La funció array_keys() retorna totes les claus d'un array.

$array = array("color" => "blau", "mida" => "mitjà", "forma" => "cercle");
$keys = array_keys($array);
print_r($keys); // Sortida: Array ( [0] => color [1] => mida [2] => forma )

Funcions Avançades d'Array

`array_map()`

La funció array_map() aplica una funció de retorn de trucada als elements dels arrays donats.

function square($n) {
    return $n * $n;
}
$numbers = array(1, 2, 3, 4, 5);
$squaredNumbers = array_map("square", $numbers);
print_r($squaredNumbers); // Sortida: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

`array_filter()`

La funció array_filter() filtra els elements d'un array utilitzant una funció de retorn de trucada.

function isEven($n) {
    return $n % 2 == 0;
}
$numbers = array(1, 2, 3, 4, 5, 6);
$evenNumbers = array_filter($numbers, "isEven");
print_r($evenNumbers); // Sortida: Array ( [1] => 2 [3] => 4 [5] => 6 )

`array_reduce()`

La funció array_reduce() redueix un array a un sol valor utilitzant una funció de retorn de trucada.

function sum($carry, $item) {
    $carry += $item;
    return $carry;
}
$numbers = array(1, 2, 3, 4, 5);
$total = array_reduce($numbers, "sum");
echo $total; // Sortida: 15

Conclusió

PHP proporciona un conjunt ric de funcions per manipular arrays, des d'operacions bàsiques com comptar elements i afegir/eliminar elements, fins a operacions més avançades com mapar, filtrar i reduir arrays. Entendre aquestes funcions millorarà significativament la teva capacitat per treballar amb arrays de manera efectiva en PHP. Practica utilitzant aquestes funcions amb diferents tipus d'arrays per convertir-te en un expert en la gestió de estructures de dades d'array.
