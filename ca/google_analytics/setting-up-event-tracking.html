Introducció al Seguiment d'Esdeveniments

El seguiment d'esdeveniments a Google Analytics et permet mesurar les interaccions dels usuaris amb els elements del teu lloc web que no desencadenen automàticament una visualització de pàgina. Aquestes interaccions poden incloure clics en botons, descàrregues, reproduccions de vídeos i més.

Conceptes Clau

  • Esdeveniments: Interaccions dels usuaris amb el contingut que es poden seguir de manera independent d'una pàgina web o una càrrega de pantalla.
  • Categoria d'Esdeveniment: El nom que proporciones per al grup d'objectes que vols seguir.
  • Acció d'Esdeveniment: El tipus d'interacció (per exemple, 'clic').
  • Etiqueta d'Esdeveniment: Informació addicional sobre l'esdeveniment (per exemple, el nom del botó clicat).
  • Valor d'Esdeveniment: Un valor numèric associat amb l'esdeveniment (per exemple, el temps dedicat a veure un vídeo).

Configuració del Seguiment Bàsic d'Esdeveniments

Per configurar el seguiment d'esdeveniments, has de modificar el codi del teu lloc web per enviar dades d'esdeveniments a Google Analytics.

Utilitzant Google Tag Manager (GTM)

Google Tag Manager simplifica el procés de configuració del seguiment d'esdeveniments sense necessitat de modificar directament el codi del teu lloc web.

Passos:

  1. Crear una Nova Etiqueta:

    • Ves al teu compte de GTM.
    • Fes clic a "Etiquetes" i després a "Nova".
    • Tria "Configuració de l'Etiqueta" i selecciona "Google Analytics: Universal Analytics".
    • Configura el "Tipus de Seguiment" a "Esdeveniment".

  2. Configurar l'Etiqueta:

    • Omple els Paràmetres de Seguiment d'Esdeveniments:
      • Categoria: per exemple, 'Botó'
      • Acció: per exemple, 'Clic'
      • Etiqueta: per exemple, 'Botó de Subscripció'
      • Valor: Opcional, per exemple, '1'

  3. Configurar el Desencadenador:

    • Fes clic a "Desencadenament" i després a "Nou".
    • Tria "Clic - Tots els Elements" o un tipus de desencadenador específic.
    • Configura el desencadenador perquè s'activi en la interacció desitjada (per exemple, clics en un botó específic).

  4. Publicar el Contenidor:

    • Envia i publica el teu contenidor per fer els canvis en directe.

Exemple de Codi per a la Implementació Manual

Si no utilitzes GTM, pots afegir manualment el codi de seguiment d'esdeveniments al teu lloc web.

<a href="#" onclick="ga('send', 'event', 'Button', 'Click', 'Subscribe Button');">Subscriu-te</a>

Explicació:

  • ga('send', 'event', 'Button', 'Click', 'Subscribe Button'); envia un esdeveniment a Google Analytics.
    • 'Button' és la Categoria d'Esdeveniment.
    • 'Click' és l'Acció d'Esdeveniment.
    • 'Subscribe Button' és l'Etiqueta d'Esdeveniment.

Seguiment Avançat d'Esdeveniments

El seguiment avançat d'esdeveniments implica interaccions més complexes i paràmetres addicionals.

Seguiment d'Enviaments de Formularis

Per seguir els enviaments de formularis, pots utilitzar el següent codi:

<form id="contactForm" onsubmit="ga('send', 'event', 'Form', 'Submit', 'Contact Form');">
  <!-- camps del formulari -->
  <button type="submit">Envia</button>
</form>

Seguiment de Reproduccions de Vídeos

Per seguir les reproduccions de vídeos, pots utilitzar l'API de YouTube o mètodes similars per a altres plataformes de vídeo.

Exemple amb l'API de YouTube:

  1. Carregar l'API IFrame de YouTube:

    <script src="https://www.youtube.com/iframe_api"></script>

  2. Inicialitzar el Reproductor i Seguir els Esdeveniments:

    <div id="player"></div>
<script>
  var tag = document.createElement('script');
  tag.src = "https://www.youtube.com/iframe_api";
  var firstScriptTag = document.getElementsByTagName('script')[0];
  firstScriptTag.parentNode.insertBefore(tag, firstScriptTag);

  var player;
  function onYouTubeIframeAPIReady() {
    player = new YT.Player('player', {
      height: '390',
      width: '640',
      videoId: 'M7lc1UVf-VE',
      events: {
        'onStateChange': onPlayerStateChange
      }
    });
  }

  function onPlayerStateChange(event) {
    if (event.data == YT.PlayerState.PLAYING) {
      ga('send', 'event', 'Video', 'Play', 'Demo Video');
    }
  }
</script>

Resum

Configurar el seguiment d'esdeveniments a Google Analytics et permet mesurar interaccions específiques dels usuaris al teu lloc web. Tant si utilitzes Google Tag Manager per a una configuració simplificada com si afegeixes manualment el codi de seguiment, pots capturar dades valuoses sobre com els usuaris interactuen amb el teu contingut. El seguiment avançat pot incloure enviaments de formularis i reproduccions de vídeos, proporcionant una visió més profunda del comportament dels usuaris. Entenent i implementant el seguiment d'esdeveniments, pots prendre decisions basades en dades per millorar el rendiment del teu lloc web i l'experiència de l'usuari.
