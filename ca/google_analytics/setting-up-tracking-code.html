El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el codi de seguiment és un pas fonamental per utilitzar Google Analytics per monitoritzar i analitzar el trànsit i el comportament dels usuaris del teu lloc web. Aquest tema et guiarà a través del procés des d'un nivell principiant fins a un nivell avançat.

Introducció al Codi de Seguiment de Google Analytics

  • Què és el Codi de Seguiment de Google Analytics?

    • Un fragment de codi JavaScript que recull dades sobre les interaccions dels usuaris al teu lloc web.
    • Envia aquestes dades a Google Analytics per al seu processament i informes.

  • Per què és important?

    • Et permet fer un seguiment del comportament dels usuaris, mesurar l'eficàcia dels teus esforços de màrqueting i prendre decisions basades en dades.

Començant amb el Codi de Seguiment de Google Analytics

Crear un Compte de Google Analytics

  1. Registra't a Google Analytics:

    • Visita el lloc web de Google Analytics.
    • Inicia sessió amb el teu compte de Google o crea'n un de nou.
    • Segueix les instruccions per configurar el teu compte, propietat i vista.

  2. Obteniu l'ID de Seguiment:

    • Després de configurar la teva propietat, rebràs un ID de Seguiment únic (per exemple, UA-XXXXXXXXX-X).

Afegir el Codi de Seguiment al Teu Lloc Web

  1. Localitza el Codi de Seguiment:

    • Al teu compte de Google Analytics, navega a Admin > Property > Tracking Info > Tracking Code.
    • Copia el fragment de codi JavaScript proporcionat.

  2. Insereix el Codi de Seguiment:

    • Enganxa el fragment de codi de seguiment a la secció <head> de cada pàgina que vulguis seguir.
<!DOCTYPE html>
<html>
<head>
  <title>El Meu Lloc Web</title>
  <!-- Etiqueta global del lloc (gtag.js) - Google Analytics -->
  <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
  <script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'UA-XXXXXXXXX-X');
  </script>
</head>
<body>
  <!-- El contingut del teu lloc web -->
</body>
</html>

Verificant la Instal·lació

  • Informes en Temps Real:
    • Comprova els informes en temps real a Google Analytics per veure si el teu codi de seguiment està funcionant correctament.
    • Visita el teu lloc web i busca la teva visita a la visió general en temps real.

Personalització Avançada del Codi de Seguiment

Seguiment de Múltiples Dominis

  • Seguiment entre Dominis:
    • Útil si tens múltiples dominis i vols fer un seguiment de les interaccions dels usuaris entre ells.
    • Modifica el codi de seguiment per incloure el plugin linker.
<script>
  gtag('config', 'UA-XXXXXXXXX-X', {
    'linker': {
      'domains': ['example1.com', 'example2.com']
    }
  });
</script>

Seguiment de Comerç Electrònic Avançat

  • Seguiment de Comerç Electrònic:
    • Fes un seguiment de transaccions, visualitzacions de productes i altres interaccions de comerç electrònic.
    • Requereix codi addicional que s'ha d'afegir a les teves pàgines de comerç electrònic.
<script>
  gtag('event', 'purchase', {
    "transaction_id": "24.031608523954162",
    "affiliation": "Botiga en Línia",
    "value": 23.07,
    "currency": "USD",
    "tax": 1.24,
    "shipping": 0,
    "items": [
      {
        "id": "P12345",
        "name": "Samarreta Android Warhol",
        "category": "Roba/Samarretes",
        "quantity": 1,
        "price": 23.07
      }
    ]
  });
</script>

Dimensions i Mètriques Personalitzades

  • Dimensions i Mètriques Personalitzades:
    • Et permeten recollir dades que Google Analytics no segueix automàticament.
    • Defineix dimensions i mètriques personalitzades a la configuració de la teva propietat de Google Analytics.
<script>
  gtag('event', 'custom_event', {
    'custom_dimension': 'dimension_value',
    'custom_metric': 123
  });
</script>

Conclusió

Configurar el codi de seguiment de Google Analytics és un pas crucial per aprofitar el poder de Google Analytics. Seguint els passos descrits en aquest tema, pots començar a recollir dades valuoses sobre el rendiment del teu lloc web i el comportament dels usuaris. A mesura que et familiaritzis amb els conceptes bàsics, podràs explorar tècniques de seguiment avançades per obtenir una comprensió més profunda i prendre decisions més informades.
