Introducció al Seguiment d'Esdeveniments GA4

Google Analytics 4 (GA4) introdueix un sistema de seguiment d'esdeveniments més flexible i potent en comparació amb Universal Analytics. Comprendre el seguiment d'esdeveniments GA4 és crucial per capturar les interaccions dels usuaris i obtenir informació sobre el comportament dels usuaris al teu lloc web o aplicació.

Conceptes Clau

  • Esdeveniments: Interaccions dels usuaris amb el contingut que es poden mesurar independentment de la càrrega d'una pàgina web o pantalla.
  • Paràmetres: Informació addicional que proporciona context a l'esdeveniment.
  • Tipus d'Esdeveniments: Esdeveniments recollits automàticament, esdeveniments de mesura millorada, esdeveniments recomanats i esdeveniments personalitzats.

Configuració del Seguiment d'Esdeveniments GA4

Abans de començar a fer el seguiment d'esdeveniments, has de configurar GA4 al teu lloc web o aplicació.

Creació d'una Propietat GA4

  1. Ves al tauler de Google Analytics.
  2. Fes clic a la icona d'engranatge "Admin" a la part inferior esquerra.
  3. A la columna "Propietat", fes clic a "Crear Propietat".
  4. Segueix l'assistent de configuració per crear la teva propietat GA4.

Afegir GA4 al Teu Lloc Web

  1. Ves a la teva propietat GA4.
  2. Fes clic a "Fluxos de dades" i selecciona el teu flux web.
  3. Copia l'ID de Mesura.
  4. Afegeix el següent fragment de codi a l'etiqueta <head> del teu lloc web, substituint G-XXXXXXXXXX pel teu ID de Mesura:
<!-- Etiqueta global del lloc (gtag.js) - Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', 'G-XXXXXXXXXX');
</script>

Esdeveniments Recollits Automàticament

GA4 fa el seguiment automàtic de diversos esdeveniments sense configuració addicional.

Exemples d'Esdeveniments Recollits Automàticament

  • page_view: Es desencadena quan es visualitza una pàgina.
  • first_visit: Es desencadena quan un usuari visita el teu lloc per primera vegada.
  • session_start: Es desencadena quan comença una nova sessió.

Esdeveniments de Mesura Millorada

Els esdeveniments de mesura millorada són esdeveniments addicionals que es poden habilitar amb un commutador a la interfície de GA4.

Habilitar la Mesura Millorada

  1. Ves a la teva propietat GA4.
  2. Fes clic a "Fluxos de dades" i selecciona el teu flux web.
  3. Activa les opcions de mesura millorada que vols habilitar, com ara el seguiment de desplaçaments, clics sortints i cerca al lloc.

Exemples d'Esdeveniments de Mesura Millorada

  • scroll: Es desencadena quan un usuari es desplaça fins al final de la pàgina.
  • click: Es desencadena quan un usuari fa clic en un enllaç sortint.
  • view_search_results: Es desencadena quan un usuari realitza una cerca al teu lloc.

Esdeveniments Recomanats

Google proporciona una llista d'esdeveniments recomanats per a sectors empresarials específics.

Exemples d'Esdeveniments Recomanats

  • purchase: Es desencadena quan un usuari completa una compra.
  • sign_up: Es desencadena quan un usuari es registra per a un compte.

Implementació d'Esdeveniments Recomanats

Per implementar un esdeveniment recomanat, utilitza la funció gtag amb el nom de l'esdeveniment i els paràmetres:

<script>
  gtag('event', 'purchase', {
    transaction_id: 'T12345',
    value: 23.07,
    currency: 'USD',
    items: [
      {
        item_id: 'P12345',
        item_name: 'Nom del Producte',
        quantity: 1,
        price: 23.07
      }
    ]
  });
</script>

Esdeveniments Personalitzats

Els esdeveniments personalitzats et permeten fer el seguiment d'interaccions específiques que són úniques per al teu lloc web o aplicació.

Creació d'Esdeveniments Personalitzats

  1. Identifica la interacció que vols fer el seguiment.
  2. Utilitza la funció gtag per enviar l'esdeveniment a GA4:
<script>
  gtag('event', 'custom_event_name', {
    parameter1: 'value1',
    parameter2: 'value2'
  });
</script>

Exemple: Fer el Seguiment d'un Clic en un Botó

Per fer el seguiment quan un usuari fa clic en un botó específic, pots afegir un oient d'esdeveniments al botó:

<button id="myButton">Fes Clic Aquí</button>
<script>
  document.getElementById('myButton').addEventListener('click', function() {
    gtag('event', 'button_click', {
      button_id: 'myButton'
    });
  });
</script>

Visualització d'Esdeveniments a GA4

Un cop configurats els esdeveniments, els pots veure a la interfície de GA4.

Navegació als Esdeveniments

  1. Ves a la teva propietat GA4.
  2. Fes clic a "Esdeveniments" al menú de l'esquerra.
  3. Aquí, pots veure una llista de tots els esdeveniments, els seus paràmetres i mètriques.

Conclusió

El seguiment d'esdeveniments GA4 proporciona un marc robust per capturar les interaccions dels usuaris al teu lloc web o aplicació. Comprenent i implementant esdeveniments recollits automàticament, esdeveniments de mesura millorada, esdeveniments recomanats i esdeveniments personalitzats, pots obtenir informació valuosa sobre el comportament dels usuaris i millorar la teva estratègia digital.
