El seguiment d'ID d'usuari és una característica potent de Google Analytics que et permet fer un seguiment d'usuaris individuals a través de múltiples dispositius i sessions. Això ajuda a entendre el comportament dels usuaris de manera més precisa i proporciona una visió més holística de les interaccions dels usuaris amb el teu lloc web o aplicació.

Introducció al Seguiment d'ID d'Usuari

  • Què és el Seguiment d'ID d'Usuari?

    • El seguiment d'ID d'usuari és una característica de Google Analytics que et permet associar múltiples sessions (i qualsevol activitat dins d'aquestes sessions) amb un ID únic i persistent per a cada usuari.

  • Per què Utilitzar el Seguiment d'ID d'Usuari?

    • Proporciona un recompte d'usuaris més precís.
    • Ajuda a entendre el comportament entre dispositius.
    • Millora la segmentació d'usuaris i la personalització.

Configuració del Seguiment d'ID d'Usuari

Prerequisits

  • Compte de Google Analytics: Assegura't de tenir un compte de Google Analytics configurat.
  • Sistema d'Autenticació d'Usuaris: Necessites un sistema per generar i gestionar IDs d'usuari únics.

Passos per Activar el Seguiment d'ID d'Usuari

  1. Activa la Funció d'ID d'Usuari a Google Analytics:

    • Navega a la secció d'Administració del teu compte de Google Analytics.
    • Sota la columna de Propietat, fes clic a "Informació de Seguiment" i després a "User-ID."
    • Segueix les instruccions en pantalla per activar la funció d'User-ID.

  2. Genera IDs d'Usuari Únics:

    • Assegura't que el teu lloc web o aplicació generi un ID d'usuari únic per a cada usuari. Aquest podria ser un ID d'usuari del teu sistema d'autenticació.

  3. Envia l'ID d'Usuari amb les Dades d'Analytics:

    • Modifica el teu codi de seguiment per incloure l'ID d'usuari. A continuació es mostra un exemple de com fer-ho utilitzant gtag.js de Google Analytics:
// Exemple de configuració de l'ID d'Usuari amb gtag.js
gtag('config', 'GA_TRACKING_ID', {
  'user_id': 'USER_ID'
});

Substitueix GA_TRACKING_ID pel teu ID de seguiment de Google Analytics i USER_ID per l'ID únic generat pel teu sistema.

Anàlisi de les Dades d'ID d'Usuari

  • Informe d'Explorador d'Usuaris:

    • Navega a Públic > Explorador d'Usuaris a Google Analytics.
    • Aquest informe et permet veure les interaccions i el comportament d'usuaris individuals.

  • Informes entre Dispositius:

    • Navega a Públic > Entre Dispositius.
    • Aquests informes t'ajuden a entendre com els usuaris interactuen amb el teu lloc a través de diferents dispositius.

Tècniques Avançades de Seguiment d'ID d'Usuari

Dimensions i Mètriques Personalitzades

  • Dimensions Personalitzades:
    • Utilitza dimensions personalitzades per emmagatzemar dades addicionals relacionades amb els usuaris.
    • Exemple: Emmagatzemar el nivell de membresia de l'usuari.
gtag('event', 'login', {
  'user_id': 'USER_ID',
  'membership_level': 'gold'
});
  • Mètriques Personalitzades:
    • Fes un seguiment d'accions o atributs específics dels usuaris com a mètriques personalitzades.
    • Exemple: Fer un seguiment del nombre d'inicis de sessió.
gtag('event', 'login', {
  'user_id': 'USER_ID',
  'login_count': 1
});

Integració amb Sistemes CRM

  • Integració CRM:
    • Sincronitza les teves dades CRM amb Google Analytics per enriquir els perfils d'usuari.
    • Exemple: Enviar dades CRM a Google Analytics utilitzant el Protocol de Mesurament.
// Exemple d'enviament de dades CRM utilitzant el Protocol de Mesurament
const measurementProtocolUrl = 'https://www.google-analytics.com/collect';
const params = new URLSearchParams({
  v: '1',
  tid: 'GA_TRACKING_ID',
  cid: 'CLIENT_ID',
  uid: 'USER_ID',
  t: 'event',
  ec: 'CRM',
  ea: 'update',
  el: 'user_data',
  ev: '1'
});

fetch(measurementProtocolUrl, {
  method: 'POST',
  body: params
});

Conclusió

El seguiment d'ID d'usuari a Google Analytics és una característica robusta que proporciona una visió més profunda del comportament dels usuaris en vincular múltiples sessions i dispositius a un sol usuari. En configurar el seguiment d'ID d'usuari, pots obtenir una comprensió més precisa de la teva audiència, millorar la segmentació d'usuaris i potenciar els esforços de personalització. Tant si estàs començant com si busques integrar tècniques avançades, el seguiment d'ID d'usuari pot elevar significativament les teves capacitats d'analítica.
