Introducció a l'API de Google Analytics

  • Què és l'API de Google Analytics?

    • Un conjunt d'eines proporcionades per Google per accedir a les dades de Google Analytics de manera programàtica.
    • Permet l'automatització de la recuperació de dades, informes personalitzats i integració amb altres aplicacions.

  • Per què utilitzar l'API de Google Analytics?

    • Automatitzar tasques repetitives.
    • Recuperar grans conjunts de dades de manera eficient.
    • Integrar les dades de Google Analytics amb altres sistemes empresarials.
    • Crear quadres de comandament i informes personalitzats.

Configuració de l'API de Google Analytics

  • Prerequisits:

    • Un compte de Google.
    • Accés a un compte de Google Analytics.
    • Coneixements bàsics de programació (preferiblement en Python o JavaScript).

  • Passos per habilitar l'API:

    1. Crear un projecte a la Consola del Núvol de Google:
      • Visita la Consola del Núvol de Google.
      • Crea un nou projecte o selecciona'n un d'existent.
    2. Habilitar l'API de Google Analytics:
      • Navega a la Biblioteca d'APIs.
      • Cerca "Google Analytics API" i habilita-la.
    3. Configurar les credencials OAuth 2.0:
      • Ves a la pàgina de "Credencials".
      • Crea identificadors de client OAuth 2.0.
      • Configura la pantalla de consentiment i descarrega el fitxer JSON de credencials.

Autenticació i autorització de sol·licituds

  • Autenticació OAuth 2.0:
    • Utilitza el fitxer JSON de credencials per autenticar la teva aplicació.
    • Exemple en Python utilitzant la biblioteca google-auth:
from google.oauth2 import service_account
from googleapiclient.discovery import build

# Carrega les credencials des del fitxer JSON
credentials = service_account.Credentials.from_service_account_file(
    'path/to/credentials.json',
    scopes=['https://www.googleapis.com/auth/analytics.readonly']
)

# Crea l'objecte del servei de l'API de Reporting de Google Analytics V4
analytics = build('analyticsreporting', 'v4', credentials=credentials)

Realització de sol·licituds a l'API

  • Estructura bàsica de la sol·licitud a l'API:
    • Defineix l'ID de la vista (perfil).
    • Especifica el rang de dates.
    • Estableix les mètriques i dimensions.
    • Exemple de sol·licitud per obtenir el nombre de sessions:
response = analytics.reports().batchGet(
    body={
        'reportRequests': [
            {
                'viewId': 'YOUR_VIEW_ID',
                'dateRanges': [{'startDate': '7daysAgo', 'endDate': 'today'}],
                'metrics': [{'expression': 'ga:sessions'}]
            }
        ]
    }
).execute()

# Imprimeix la resposta
print(response)

Anàlisi i ús de la resposta de l'API

  • Comprensió de l'estructura de la resposta:

    • La resposta és un objecte JSON que conté dades de l'informe.
    • Els components clau inclouen columnHeader, data i rows.

  • Extracció de dades de la resposta:

    • Exemple d'anàlisi de la resposta per obtenir el nombre de sessions:
for report in response.get('reports', []):
    columnHeader = report.get('columnHeader', {})
    metricHeaders = columnHeader.get('metricHeader', {}).get('metricHeaderEntries', [])
    rows = report.get('data', {}).get('rows', [])

    for row in rows:
        metrics = row.get('metrics', [])
        for metric in metrics:
            for i, value in enumerate(metric.get('values')):
                print(f"{metricHeaders[i].get('name')}: {value}")

Ús avançat

  • Combinació de múltiples mètriques i dimensions:
    • Exemple d'una sol·licitud més complexa:
response = analytics.reports().batchGet(
    body={
        'reportRequests': [
            {
                'viewId': 'YOUR_VIEW_ID',
                'dateRanges': [{'startDate': '30daysAgo', 'endDate': 'today'}],
                'metrics': [{'expression': 'ga:sessions'}, {'expression': 'ga:pageviews'}],
                'dimensions': [{'name': 'ga:country'}, {'name': 'ga:browser'}]
            }
        ]
    }
).execute()

# Imprimeix la resposta
print(response)
  • Gestió de grans conjunts de dades amb paginació:
    • Utilitza el pageToken per recuperar pàgines addicionals de dades.
    • Exemple:
page_token = None
while True:
    response = analytics.reports().batchGet(
        body={
            'reportRequests': [
                {
                    'viewId': 'YOUR_VIEW_ID',
                    'dateRanges': [{'startDate': '30daysAgo', 'endDate': 'today'}],
                    'metrics': [{'expression': 'ga:sessions'}],
                    'pageToken': page_token
                }
            ]
        }
    ).execute()

    # Processa la resposta
    print(response)

    # Comprova si hi ha més pàgines
    page_token = response.get('reports', [])[0].get('nextPageToken')
    if not page_token:
        break

Conclusió

  • Resum:

    • L'API de Google Analytics és una eina potent per automatitzar la recuperació de dades i crear informes personalitzats.
    • La configuració implica crear un projecte, habilitar l'API i configurar les credencials OAuth 2.0.
    • Realitzar sol·licituds requereix especificar l'ID de la vista, el rang de dates, les mètriques i les dimensions.
    • L'ús avançat inclou la combinació de múltiples mètriques i dimensions i la gestió de grans conjunts de dades amb paginació.

  • Pròxims passos:

    • Experimenta amb diferents mètriques i dimensions.
    • Integra l'API amb altres fonts de dades i eines de visualització.
    • Explora funcions i capacitats addicionals de l'API de Google Analytics.
