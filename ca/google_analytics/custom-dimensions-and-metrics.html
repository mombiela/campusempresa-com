Dimensions i Mètriques Personalitzades
Seguiment d'ID d'usuari ►

Introducció

Les Dimensions i Mètriques Personalitzades a Google Analytics et permeten recollir i analitzar dades específiques per a les necessitats del teu negoci. Amplien les dades predeterminades proporcionades per Google Analytics, permetent-te fer un seguiment d'informació addicional que no es captura automàticament.

Conceptes Clau

  • Dimensions Personalitzades: Són dimensions definides per l'usuari que pots utilitzar per recollir dades que Google Analytics no fa un seguiment automàticament. Per exemple, podries voler fer un seguiment del nivell de membres dels usuaris al teu lloc.
  • Mètriques Personalitzades: Són mètriques definides per l'usuari que et permeten quantificar dades que no es fan un seguiment automàticament a Google Analytics. Per exemple, podries voler fer un seguiment del nombre de vegades que es fa clic en un botó específic.

Configuració de Dimensions i Mètriques Personalitzades

Per configurar Dimensions i Mètriques Personalitzades, segueix aquests passos:

  1. Navega a la Configuració d'Administració:

    • Ves al teu compte de Google Analytics.
    • Selecciona la propietat on vols crear Dimensions i Mètriques Personalitzades.
    • Sota la columna de propietat, fes clic a "Definicions Personalitzades" i després a "Dimensions Personalitzades" o "Mètriques Personalitzades".

  2. Crea una Dimensió Personalitzada:

    • Fes clic al botó "+ Nova Dimensió Personalitzada".
    • Introdueix un nom per a la dimensió (per exemple, "Nivell de Membres").
    • Selecciona l'abast (Hit, Sessió, Usuari o Producte).
    • Fes clic a "Crear".

  3. Crea una Mètrica Personalitzada:

    • Fes clic al botó "+ Nova Mètrica Personalitzada".
    • Introdueix un nom per a la mètrica (per exemple, "Clics de Botó").
    • Defineix el tipus (Enter, Moneda, Temps).
    • Estableix els valors mínim i màxim.
    • Fes clic a "Crear".

Implementació de Dimensions i Mètriques Personalitzades en el Codi

Per enviar dades a Google Analytics utilitzant Dimensions i Mètriques Personalitzades, has de modificar el teu codi de seguiment. Aquí tens exemples tant per a Universal Analytics (analytics.js) com per a Google Analytics 4 (gtag.js).

Universal Analytics (analytics.js)

// Configuració d'una Dimensió Personalitzada
ga('create', 'UA-XXXXX-Y', 'auto');
ga('set', 'dimension1', 'Premium'); // Substitueix 'dimension1' pel teu índex de Dimensió Personalitzada i 'Premium' pel valor

// Configuració d'una Mètrica Personalitzada
ga('set', 'metric1', 42); // Substitueix 'metric1' pel teu índex de Mètrica Personalitzada i 42 pel valor

ga('send', 'pageview');

Google Analytics 4 (gtag.js)

// Configuració d'una Dimensió Personalitzada
gtag('config', 'G-XXXXXXX', {
  'custom_map': {'dimension1': 'nivell_de_membres'}
});
gtag('event', 'page_view', {'nivell_de_membres': 'Premium'}); // Substitueix 'nivell_de_membres' pel teu nom de Dimensió Personalitzada i 'Premium' pel valor

// Configuració d'una Mètrica Personalitzada
gtag('config', 'G-XXXXXXX', {
  'custom_map': {'metric1': 'clics_de_botó'}
});
gtag('event', 'button_click', {'clics_de_botó': 42}); // Substitueix 'clics_de_botó' pel teu nom de Mètrica Personalitzada i 42 pel valor

Ús de Dimensions i Mètriques Personalitzades en Informes

Un cop hagis configurat i implementat Dimensions i Mètriques Personalitzades, pots utilitzar-les en els teus informes:

  • Informes Personalitzats: Crea informes personalitzats a Google Analytics per incloure les teves Dimensions i Mètriques Personalitzades.
  • Segments: Utilitza Dimensions Personalitzades per crear segments per a una anàlisi més detallada.
  • Quadres de Comandament: Afegeix ginys als teus quadres de comandament que mostrin dades de les teves Dimensions i Mètriques Personalitzades.

Exemples d'Usos

  • Comerç Electrònic: Fes un seguiment de categories de productes, nivells de membres dels usuaris o el nombre de vegades que un producte s'afegeix al carret.
  • Llocs de Contingut: Fes un seguiment dels noms dels autors, categories d'articles o el nombre de comentaris per article.
  • Aplicacions: Fes un seguiment de rols d'usuari, ús de funcions o ocurrències d'errors.

Conclusió

Les Dimensions i Mètriques Personalitzades són eines poderoses a Google Analytics que et permeten adaptar la teva recollida de dades a les necessitats específiques del teu negoci. Configurant i implementant aquestes definicions personalitzades, pots obtenir una visió més profunda del comportament dels usuaris i prendre decisions més informades.

Recorda:

  • Defineix quines dades addicionals necessites.
  • Configura Dimensions i Mètriques Personalitzades a l'Administració de Google Analytics.
  • Implementa els canvis necessaris en el codi.
  • Utilitza les dades en els teus informes per a una millor anàlisi.

Dominant les Dimensions i Mètriques Personalitzades, pots millorar significativament el valor de les teves dades de Google Analytics.
