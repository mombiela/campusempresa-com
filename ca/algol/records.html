Introducció als Registres

Els registres en ALGOL són una manera d'agrupar diferents tipus de dades sota un sol nom. Són similars a les estructures en C o als objectes en llenguatges de programació moderns. Els registres permeten crear un tipus de dada compost que pot contenir múltiples valors de diferents tipus.

Conceptes Clau

  • Definició de Registre: Com definir un tipus de registre.
  • Variables de Registre: Com declarar variables d'un tipus de registre.
  • Accés als Camps del Registre: Com accedir i modificar els camps dins d'un registre.
  • Registres Imbricats: Com treballar amb registres que contenen altres registres.

Definició d'un Registre

Per definir un registre en ALGOL, s'utilitza la paraula clau record seguida dels camps i els seus tipus.

Exemple

record Person(
    string name;
    integer age;
    real height;
);

En aquest exemple, Person és un tipus de registre amb tres camps: name (una cadena), age (un enter) i height (un número real).

Declaració de Variables de Registre

Un cop has definit un tipus de registre, pots declarar variables d'aquest tipus.

Exemple

Person john;
Person jane;

Aquí, john i jane són variables del tipus de registre Person.

Accés i Modificació dels Camps del Registre

Pots accedir i modificar els camps d'un registre utilitzant la notació de punt.

Exemple

john.name := "John Doe";
john.age := 30;
john.height := 5.9;

print(john.name);  // Mostra: John Doe
print(john.age);   // Mostra: 30
print(john.height); // Mostra: 5.9

En aquest exemple, assignem valors als camps del registre john i després els imprimim.

Registres Imbricats

Els registres poden contenir altres registres, permetent estructures de dades més complexes.

Exemple

record Address(
    string street;
    string city;
    string country;
);

record Person(
    string name;
    integer age;
    real height;
    Address address;
);

Person john;
john.name := "John Doe";
john.age := 30;
john.height := 5.9;
john.address.street := "123 Main St";
john.address.city := "Anytown";
john.address.country := "USA";

print(john.address.city);  // Mostra: Anytown

En aquest exemple, el registre Person conté un registre Address, i accedim als camps imbricats utilitzant la notació de punt.

Resum

Els registres en ALGOL proporcionen una manera poderosa d'agrupar dades relacionades de diferents tipus. Definint tipus de registre, declarant variables de registre i utilitzant la notació de punt per accedir i modificar camps, pots crear estructures de dades complexes que són fàcils de gestionar i entendre. Els registres imbricats milloren encara més aquesta capacitat, permetent una representació jeràrquica de les dades. Entendre els registres és essencial per escriure programes organitzats i eficients en ALGOL.
