Introducció

En aquesta secció, explorarem els conceptes fonamentals dels operadors i expressions en ALGOL. Entendre aquests conceptes és crucial per escriure programes ALGOL eficients i efectius. Cobrirem diversos tipus d'operadors, la seva precedència i com construir expressions.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics. ALGOL admet els següents operadors aritmètics:

  • + : Addició
  • - : Subtracció
  • * : Multiplicació
  • / : Divisió
  • := : Assignació

Exemples

begin
  integer a, b, c;
  a := 10;
  b := 5;
  c := a + b;  ! c ara és 15
  c := a - b;  ! c ara és 5
  c := a * b;  ! c ara és 50
  c := a / b;  ! c ara és 2
end

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors. El resultat d'una operació relacional és un valor booleà (true o false). ALGOL admet els següents operadors relacionals:

  • = : Igual a
  • ≠ : Diferent de
  • < : Menor que
  • <= : Menor o igual que
  • > : Major que
  • >= : Major o igual que

Exemples

begin
  integer x, y;
  x := 10;
  y := 5;
  if x = y then
    print("x és igual a y")
  else
    print("x no és igual a y");  ! Això imprimirà "x no és igual a y"
end

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleanes. ALGOL admet els següents operadors lògics:

  • and : Lògic AND
  • or : Lògic OR
  • not : Lògic NOT

Exemples

begin
  boolean a, b, c;
  a := true;
  b := false;
  c := a and b;  ! c ara és false
  c := a or b;   ! c ara és true
  c := not a;    ! c ara és false
end

Precedència dels Operadors

La precedència dels operadors determina l'ordre en què els operadors són avaluats en una expressió. La següent taula resumeix la precedència dels operadors en ALGOL, de més alta a més baixa:

| Nivell de Precedència | Operadors | |-----------------------|--------------------------| | 1 | not | | 2 | *, / | | 3 | +, - | | 4 | <, <=, >, >= | | 5 | =, ≠ | | 6 | and | | 7 | or | | 8 | := (assignació) |

Exemples

begin
  integer a, b, c, d;
  a := 10;
  b := 5;
  c := 2;
  d := a + b * c;  ! d ara és 20 perquè * té més precedència que +
  d := (a + b) * c;  ! d ara és 30 perquè els parèntesis canvien la precedència
end

Expressions Compostes

Les expressions compostes impliquen múltiples operadors i operands. Els parèntesis es poden utilitzar per definir explícitament l'ordre d'avaluació.

Exemples

begin
  integer x, y, z;
  x := 10;
  y := 5;
  z := (x + y) * (x - y);  ! z ara és 75
end

Conclusió

Entendre els operadors i expressions és fonamental per programar en ALGOL. En dominar els operadors aritmètics, relacionals i lògics, així com la seva precedència, podràs escriure programes més complexos i eficients. Practica la construcció i avaluació d'expressions per convertir-te en un expert en l'ús efectiu d'aquests operadors.
