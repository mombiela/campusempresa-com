La gestió de fitxers és un aspecte essencial de la programació que et permet llegir i escriure fitxers. Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de la gestió de fitxers en ALGOL.

Introducció a la Gestió de Fitxers

La gestió de fitxers en ALGOL implica diverses operacions clau:

  • Obrir un fitxer
  • Llegir d'un fitxer
  • Escriure en un fitxer
  • Tancar un fitxer

Entendre aquestes operacions bàsiques et permetrà gestionar l'emmagatzematge i la recuperació de dades de manera efectiva.

Obrir un Fitxer

Per realitzar qualsevol operació en un fitxer, primer cal obrir-lo. En ALGOL, els fitxers s'obren típicament utilitzant la instrucció OPEN.

Sintaxi

OPEN(file_variable, mode);
  • file_variable: La variable que representa el fitxer.
  • mode: El mode en què s'obre el fitxer (per exemple, llegir, escriure).

Exemple

BEGIN
  FILE F;
  OPEN(F, 'READ');
  ...
END;

En aquest exemple, F és una variable de fitxer oberta en mode de lectura.

Llegir d'un Fitxer

Un cop obert un fitxer, pots llegir dades d'ell utilitzant la instrucció READ.

Sintaxi

READ(file_variable, variable);
  • file_variable: El fitxer del qual llegir.
  • variable: La variable on s'emmagatzemaran les dades llegides.

Exemple

BEGIN
  FILE F;
  INTEGER X;
  OPEN(F, 'READ');
  READ(F, X);
  ...
END;

Aquí, el valor enter del fitxer F es llegeix a la variable X.

Escriure en un Fitxer

Escriure dades en un fitxer es fa utilitzant la instrucció WRITE.

Sintaxi

WRITE(file_variable, variable);
  • file_variable: El fitxer on s'escriuran les dades.
  • variable: La variable que conté les dades a escriure.

Exemple

BEGIN
  FILE F;
  INTEGER X;
  X := 42;
  OPEN(F, 'WRITE');
  WRITE(F, X);
  ...
END;

En aquest exemple, el valor enter 42 s'escriu al fitxer F.

Tancar un Fitxer

Després de completar les operacions de fitxer, és crucial tancar el fitxer utilitzant la instrucció CLOSE.

Sintaxi

CLOSE(file_variable);
  • file_variable: El fitxer a tancar.

Exemple

BEGIN
  FILE F;
  OPEN(F, 'READ');
  ...
  CLOSE(F);
END;

Això assegura que tots els recursos associats amb el fitxer es tanquin correctament.

Gestió Avançada de Fitxers

Gestió d'Errors

La gestió d'errors és crucial quan es treballa amb fitxers per gestionar situacions inesperades com fitxers no trobats o errors de lectura/escriptura.

Exemple

BEGIN
  FILE F;
  INTEGER X;
  OPEN(F, 'READ');
  IF NOT FILE_EXISTS(F) THEN
    PRINT("Error: Fitxer no trobat");
  ELSE
    READ(F, X);
  ENDIF;
  CLOSE(F);
END;

En aquest exemple, comprovem si el fitxer existeix abans d'intentar llegir d'ell.

Modes de Fitxer

ALGOL suporta diversos modes de fitxer:

  • READ: Obre el fitxer per a lectura.
  • WRITE: Obre el fitxer per a escriptura.
  • APPEND: Obre el fitxer per a afegir dades.

Exemple

BEGIN
  FILE F;
  OPEN(F, 'APPEND');
  ...
  CLOSE(F);
END;

Això obre el fitxer en mode d'afegir, permetent afegir dades al final del fitxer.

Resum

La gestió de fitxers en ALGOL implica:

  • Obrir fitxers en diferents modes (llegir, escriure, afegir).
  • Llegir i escriure fitxers.
  • Tancar correctament els fitxers per alliberar recursos.
  • Implementar la gestió d'errors per gestionar errors relacionats amb fitxers.

En dominar aquests conceptes, podràs gestionar eficientment les operacions de fitxer en els teus programes ALGOL.
