Introducció a ALGOL

ALGOL (Llenguatge Algorísmic) és una família de llenguatges de programació imperatius desenvolupats originalment a mitjans del segle XX. Va ser dissenyat per ser un llenguatge universal per expressar algoritmes i ha influenciat molts llenguatges de programació moderns.

Començant amb ALGOL

Abans de submergir-nos en la implementació d'algoritmes, és essencial entendre els conceptes bàsics de la sintaxi i l'estructura d'ALGOL.

Sintaxi bàsica

  • Estructura del programa: Un programa ALGOL típicament consisteix en una seqüència de declaracions seguida d'una seqüència d'instruccions.
  • Variables: Les variables s'han de declarar abans de ser utilitzades.
  • Estructures de control: ALGOL suporta estructures de control estàndard com bucles i condicionals.

Exemple:

begin
  integer i;
  for i := 1 step 1 until 10 do
    print(i);
end

Explicació:

  • begin i end marquen l'inici i el final del programa.
  • integer i; declara una variable i de tipus enter.
  • El bucle for itera de l'1 al 10, imprimint cada valor de i.

Algoritmes bàsics

Cerca seqüencial

Un algoritme senzill per cercar un element en una matriu.

Exemple:

begin
  integer array[1:10];
  integer i, n, key, position;
  position := 0;
  key := 5;  // Element a cercar
  for i := 1 step 1 until 10 do
    if array[i] = key then
      position := i;
  if position = 0 then
    print("Element no trobat")
  else
    print("Element trobat a la posició: ", position);
end

Explicació:

  • Es declara una matriu d'enters.
  • El bucle for itera a través de la matriu per trobar la key.
  • Si es troba la key, es guarda la seva posició; en cas contrari, es mostra un missatge.

Ordenació per bombolla

Un algoritme bàsic d'ordenació que recorre repetidament la llista, compara elements adjacents i els intercanvia si estan en l'ordre incorrecte.

Exemple:

begin
  integer array[1:5];
  integer i, j, temp;
  for i := 1 step 1 until 4 do
    for j := 1 step 1 until 5 - i do
      if array[j] > array[j + 1] then
        temp := array[j];
        array[j] := array[j + 1];
        array[j + 1] := temp;
end

Explicació:

  • El bucle extern s'executa des del primer fins al penúltim element.
  • El bucle intern compara elements adjacents i els intercanvia si és necessari.
  • Aquest procés es repeteix fins que la matriu està ordenada.

Algoritmes intermedis

Cerca binària

Un algoritme de cerca més eficient per a matrius ordenades.

Exemple:

begin
  integer array[1:10];
  integer low, high, mid, key;
  low := 1;
  high := 10;
  key := 5;  // Element a cercar
  while low <= high do
    mid := (low + high) div 2;
    if array[mid] = key then
      print("Element trobat a la posició: ", mid);
      exit;
    else if array[mid] < key then
      low := mid + 1;
    else
      high := mid - 1;
  print("Element no trobat");
end

Explicació:

  • low i high marquen el rang de cerca.
  • mid és l'índex mitjà del rang actual.
  • L'algoritme redueix el rang de cerca basant-se en la comparació de key i array[mid].

Ordenació per fusió

Un algoritme de divideix i venceràs que divideix la matriu en meitats, les ordena i després les fusiona.

Exemple:

procedure mergeSort(array, left, right);
begin
  integer mid;
  if left < right then
    mid := (left + right) div 2;
    mergeSort(array, left, mid);
    mergeSort(array, mid + 1, right);
    merge(array, left, mid, right);
end

procedure merge(array, left, mid, right);
begin
  integer i, j, k;
  integer n1, n2;
  n1 := mid - left + 1;
  n2 := right - mid;
  integer L[1:n1], R[1:n2];
  for i := 1 step 1 until n1 do
    L[i] := array[left + i - 1];
  for j := 1 step 1 until n2 do
    R[j] := array[mid + j];
  i := 1; j := 1; k := left;
  while i <= n1 and j <= n2 do
    if L[i] <= R[j] then
      array[k] := L[i];
      i := i + 1;
    else
      array[k] := R[j];
      j := j + 1;
    k := k + 1;
  while i <= n1 do
    array[k] := L[i];
    i := i + 1;
    k := k + 1;
  while j <= n2 do
    array[k] := R[j];
    j := j + 1;
    k := k + 1;
end

begin
  integer array[1:10];
  mergeSort(array, 1, 10);
end

Explicació:

  • mergeSort divideix recursivament la matriu en meitats.
  • merge combina dues meitats ordenades en una sola matriu ordenada.
  • Les matrius temporals L i R emmagatzemen les meitats esquerra i dreta.

Algoritmes avançats

Programació dinàmica: Seqüència de Fibonacci

Un enfocament optimitzat per calcular la seqüència de Fibonacci utilitzant memoització.

Exemple:

integer array[1:100];

procedure fibonacci(n);
begin
  if n <= 1 then
    return n;
  if array[n] != 0 then
    return array[n];
  array[n] := fibonacci(n - 1) + fibonacci(n - 2);
  return array[n];
end

begin
  integer n;
  n := 10;  // Calcular el desè nombre de Fibonacci
  print(fibonacci(n));
end

Explicació:

  • El procediment fibonacci utilitza memoització per emmagatzemar valors calculats prèviament.
  • Això redueix la complexitat temporal d'exponencial a lineal.

Algoritmes de graf: Algoritme de Dijkstra

Un algoritme per trobar el camí més curt en un graf.

Exemple:

procedure dijkstra(graph, source, n);
begin
  integer dist[1:n], visited[1:n], i, j, minDist, nextNode;
  for i := 1 step 1 until n do
    dist[i] := infinity;
    visited[i] := 0;
  dist[source] := 0;
  for i := 1 step 1 until n - 1 do
    minDist := infinity;
    for j := 1 step 1 until n do
      if dist[j] < minDist and visited[j] = 0 then
        minDist := dist[j];
        nextNode := j;
    visited[nextNode] := 1;
    for j := 1 step 1 until n do
      if visited[j] = 0 and dist[nextNode] + graph[nextNode][j] < dist[j] then
        dist[j] := dist[nextNode] + graph[nextNode][j];
  for i := 1 step 1 until n do
    print("Distància des de la font fins al node ", i, ": ", dist[i]);
end

begin
  integer graph[1:5][1:5];
  dijkstra(graph, 1, 5);
end

Explicació:

  • La matriu dist emmagatzema la distància més curta des de la font fins a cada node.
  • La matriu visited fa un seguiment dels nodes visitats.
  • L'algoritme selecciona iterativament el node no visitat amb la distància més petita i actualitza les distàncies dels seus veïns.
