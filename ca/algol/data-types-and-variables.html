Introducció

En aquesta secció, explorarem els conceptes fonamentals dels tipus de dades i les variables en ALGOL. Entendre aquests conceptes és crucial per escriure programes eficients i sense errors. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Tipus de Dades

Els tipus de dades defineixen el tipus de dades que es poden emmagatzemar i manipular dins d'un programa. ALGOL admet diversos tipus de dades bàsics:

  • Enter: Representa nombres sencers.
  • Real: Representa nombres de punt flotant.
  • Booleà: Representa valors de veritat (true o false).
  • Caràcter: Representa caràcters individuals.
  • Cadena: Representa seqüències de caràcters (no és un tipus primitiu en ALGOL però es pot gestionar a través de matrius de caràcters).

Enter

Els enters s'utilitzen per emmagatzemar nombres sencers. En ALGOL, declares una variable enter utilitzant la paraula clau integer.

integer x;
x := 10;

Real

Els nombres reals s'utilitzen per emmagatzemar nombres de punt flotant. Declara una variable real utilitzant la paraula clau real.

real y;
y := 3.14;

Booleà

Els tipus de dades booleans s'utilitzen per emmagatzemar valors de veritat. Declara una variable booleana utilitzant la paraula clau boolean.

boolean flag;
flag := true;

Caràcter

Els caràcters s'utilitzen per emmagatzemar caràcters individuals. Declara una variable caràcter utilitzant la paraula clau char.

char letter;
letter := 'A';

Variables

Les variables són ubicacions d'emmagatzematge amb nom que poden contenir dades. En ALGOL, has de declarar una variable abans d'utilitzar-la. La declaració especifica el nom de la variable i el tipus de dades.

Declaració de Variables

Per declarar una variable, utilitza la següent sintaxi:

<type> <variable_name>;

Per exemple:

integer age;
real salary;
boolean isEmployed;
char initial;

Inicialització de Variables

Pots inicialitzar una variable en el moment de la declaració o més tard en el codi.

integer age := 25;
real salary := 50000.50;
boolean isEmployed := true;
char initial := 'J';

Assignació de Variables

Pots assignar un valor a una variable utilitzant l'operador d'assignació :=.

age := 30;
salary := 60000.75;
isEmployed := false;
initial := 'K';

Temes Avançats

Matrius

Les matrius s'utilitzen per emmagatzemar múltiples valors del mateix tipus de dades. En ALGOL, declares una matriu especificant el tipus, el nom i la mida.

integer array numbers[10];
real array temperatures[5];

Pots accedir i modificar els elements de la matriu utilitzant el seu índex.

numbers[1] := 5;
temperatures[2] := 36.6;

Registres

Els registres s'utilitzen per agrupar dades relacionades de diferents tipus. Són similars als structs en altres llenguatges de programació.

record Person:
    integer age;
    real salary;
    boolean isEmployed;
    char initial;
end;

Pots crear una variable de registre i accedir als seus camps de la següent manera:

Person john;
john.age := 30;
john.salary := 60000.75;
john.isEmployed := true;
john.initial := 'J';

Resum

En aquesta secció, hem cobert els conceptes bàsics dels tipus de dades i les variables en ALGOL. Hem après com declarar i inicialitzar variables, assignar valors i utilitzar estructures més avançades com matrius i registres. Entendre aquests conceptes és essencial per escriure programes efectius en ALGOL. En la següent secció, aprofundirem en les estructures de control i com gestionar el flux del teu programa.
