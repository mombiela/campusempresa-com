Introducció a les Sentències Switch Case

Les sentències switch case en ALGOL s'utilitzen per executar un bloc de codi entre molts basant-se en el valor d'una expressió. Proporcionen una manera més llegible i organitzada de gestionar múltiples branques condicionals en comparació amb l'ús de múltiples sentències if-else.

Sintaxi Bàsica

La sintaxi bàsica d'una sentència switch case en ALGOL és la següent:

switch expression
begin
    case value1: 
        // bloc de codi per a value1
    case value2: 
        // bloc de codi per a value2
    ...
    default: 
        // bloc de codi per defecte si no coincideix cap cas
end switch
  • expression: El valor que s'està avaluant.
  • case value: El valor específic per coincidir amb l'expressió.
  • default: El bloc de codi a executar si cap cas coincideix amb l'expressió.

Exemple: Switch Case Simple

Aquí teniu un exemple simple per demostrar la sentència switch case en ALGOL:

begin
    integer x;
    x := 2;
    
    switch x
    begin
        case 1: 
            print("x és 1");
        case 2: 
            print("x és 2");
        case 3: 
            print("x és 3");
        default: 
            print("x no és 1, 2, o 3");
    end switch
end

Explicació

  • La variable x s'assigna el valor 2.
  • La sentència switch avalua el valor de x.
  • Com que x és 2, s'executa el bloc de codi sota case 2, imprimint "x és 2".

Ús Avançat

Les sentències switch case també poden gestionar escenaris més complexos, com ara switch cases niats o l'ús d'expressions en els casos.

Exemple de Switch Case Niat

begin
    integer x, y;
    x := 1;
    y := 2;
    
    switch x
    begin
        case 1: 
            switch y
            begin
                case 1: 
                    print("x és 1 i y és 1");
                case 2: 
                    print("x és 1 i y és 2");
                default: 
                    print("x és 1 i y és una altra cosa");
            end switch
        case 2: 
            print("x és 2");
        default: 
            print("x no és 1 o 2");
    end switch
end

Explicació

  • El switch exterior avalua x.
  • Com que x és 1, entra al switch niat que avalua y.
  • Com que y és 2, imprimeix "x és 1 i y és 2".

Comparació amb les Sentències If-Else

Les sentències switch case sovint poden ser més llegibles i eficients que múltiples sentències if-else. Aquí teniu una comparació:

Exemple If-Else

begin
    integer x;
    x := 2;
    
    if x = 1 then
        print("x és 1")
    else if x = 2 then
        print("x és 2")
    else if x = 3 then
        print("x és 3")
    else
        print("x no és 1, 2, o 3")
end

Exemple Switch Case

begin
    integer x;
    x := 2;
    
    switch x
    begin
        case 1: 
            print("x és 1");
        case 2: 
            print("x és 2");
        case 3: 
            print("x és 3");
        default: 
            print("x no és 1, 2, o 3");
    end switch
end

Taula de Comparació

| Característica | Sentències If-Else | Sentències Switch Case | |--------------------|--------------------------------------|-----------------------------------| | Llegibilitat | Pot esdevenir desordenat amb molts casos | Més llegible amb múltiples casos | | Eficiència | Pot avaluar múltiples condicions | Avalua l'expressió una vegada | | Cas d'Ús | Condicions simples o poques | Múltiples condicions discretes |

Conclusió

Les sentències switch case en ALGOL proporcionen una manera estructurada i llegible de gestionar múltiples branques condicionals. Són particularment útils quan es tracta de múltiples valors discrets i poden millorar tant la llegibilitat com l'eficiència del vostre codi en comparació amb l'ús de múltiples sentències if-else. Dominant les sentències switch case, podeu escriure codi més organitzat i mantenible.
