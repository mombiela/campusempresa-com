Introducció a les Funcions Recursives

Les funcions recursives són funcions que es criden a si mateixes per resoldre un problema. Són una eina poderosa en la programació, permetent solucions elegants a problemes complexos descomponent-los en subproblemes més simples.

Conceptes Clau

  • Cas Base: La condició sota la qual la recursió acaba.
  • Cas Recursiu: La part de la funció on la funció es crida a si mateixa.
  • Desbordament de Pila: Un risc potencial quan la recursió no es gestiona adequadament, conduint a un ús excessiu de memòria.

Estructura Bàsica de les Funcions Recursives en ALGOL

En ALGOL, les funcions recursives es defineixen de manera similar a altres funcions, però inclouen una crida a si mateixes dins del seu cos.

Exemple: Funció Factorial

El factorial d'un enter no negatiu n és el producte de tots els enters positius menors o iguals a n. Es defineix com:

  • factorial(0) = 1 (Cas Base)
  • factorial(n) = n * factorial(n-1) (Cas Recursiu)

Aquí tens com pots implementar la funció factorial en ALGOL:

begin
    integer procedure factorial(n);
    value n;
    integer n;
    begin
        if n = 0 then
            factorial := 1
        else
            factorial := n * factorial(n - 1)
    end;

    integer n;
    n := 5;
    print(factorial(n))
end

Explicació

  • Cas Base: if n = 0 then factorial := 1
  • Cas Recursiu: factorial := n * factorial(n - 1)

Funcions Recursives Avançades

A mesura que et familiaritzes amb la recursió, pots abordar problemes més complexos. Aquí tens alguns exemples:

Exemple: Seqüència de Fibonacci

La seqüència de Fibonacci és una sèrie de nombres on cada nombre és la suma dels dos anteriors, normalment començant amb 0 i 1. Es defineix com:

  • fibonacci(0) = 0
  • fibonacci(1) = 1
  • fibonacci(n) = fibonacci(n-1) + fibonacci(n-2)

Aquí tens la implementació en ALGOL:

begin
    integer procedure fibonacci(n);
    value n;
    integer n;
    begin
        if n = 0 then
            fibonacci := 0
        else if n = 1 then
            fibonacci := 1
        else
            fibonacci := fibonacci(n - 1) + fibonacci(n - 2)
    end;

    integer n;
    n := 10;
    print(fibonacci(n))
end

Explicació

  • Casos Base: if n = 0 then fibonacci := 0 i else if n = 1 then fibonacci := 1
  • Cas Recursiu: fibonacci := fibonacci(n - 1) + fibonacci(n - 2)

Recursió de Cua

La recursió de cua és una forma especial de recursió on la crida recursiva és l'última operació en la funció. Això pot ser optimitzat pel compilador per evitar el desbordament de pila.

Exemple: Factorial Recursiu de Cua

Per convertir la funció factorial a una versió recursiva de cua, introduïm un acumulador per mantenir els resultats intermedis.

begin
    integer procedure factorial(n, acc);
    value n, acc;
    integer n, acc;
    begin
        if n = 0 then
            factorial := acc
        else
            factorial := factorial(n - 1, n * acc)
    end;

    integer n;
    n := 5;
    print(factorial(n, 1))
end

Explicació

  • Cas Base: if n = 0 then factorial := acc
  • Cas Recursiu: factorial := factorial(n - 1, n * acc)

Conclusió

Les funcions recursives són un concepte fonamental en la programació, permetent solucions elegants i eficients a problemes complexos. Entenent els casos base i recursius, i sent conscient dels possibles perills com el desbordament de pila, pots utilitzar efectivament la recursió en els teus programes ALGOL. La recursió de cua és una tècnica avançada que pot ajudar a optimitzar les funcions recursives, fent-les més eficients i menys propenses a errors.
