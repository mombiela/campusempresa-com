Introducció

El retargeting a Facebook Ads és una tècnica de màrqueting digital que permet als anunciants tornar a connectar amb usuaris que han interactuat prèviament amb el seu lloc web o contingut. Aquest procés és crucial per augmentar la conversió i maximitzar el retorn de la inversió (ROI).

Conceptes Clau

  • Pixel de Facebook: Un fragment de codi que es col·loca al teu lloc web per rastrejar les accions dels visitants.
  • Audiències Personalitzades: Grups d'usuaris que han interactuat amb el teu lloc web, aplicació o contingut a Facebook.
  • Esdeveniments: Accions específiques que els usuaris realitzen al teu lloc web, com afegir productes al carret o completar una compra.

Passos per Configurar Retargeting a Facebook Ads

1. Crear i Configurar el Pixel de Facebook

El primer pas és crear i configurar el Pixel de Facebook al teu lloc web.

Crear el Pixel

  1. Ves al teu Administrador d'Esdeveniments.
  2. Fes clic a "Connectar fonts de dades" i selecciona "Web".
  3. Tria "Facebook Pixel" i fes clic a "Connectar".
  4. Assigna un nom al teu Pixel i fes clic a "Continuar".

Instal·lar el Pixel al teu Lloc Web

  1. Copia el codi del Pixel proporcionat per Facebook.
  2. Enganxa el codi al <head> de cada pàgina del teu lloc web.

2. Definir Esdeveniments i Conversiones

Configura esdeveniments específics per rastrejar accions clau al teu lloc web.

Exemples d'Esdeveniments

  • PageView: S'activa quan un usuari visita una pàgina.
  • AddToCart: S'activa quan un usuari afegeix un producte al carret.
  • Purchase: S'activa quan un usuari completa una compra.
<script>
  fbq('track', 'AddToCart');
</script>

3. Crear Audiències Personalitzades

Utilitza les dades recopilades pel Pixel per crear audiències personalitzades.

  1. Ves al teu Administrador d'Audiències.
  2. Fes clic a "Crear audiència" i selecciona "Audiència personalitzada".
  3. Tria "Trànsit del lloc web".
  4. Configura les regles per incloure els usuaris que han realitzat accions específiques (per exemple, visitar una pàgina de producte).

4. Configurar Campanyes de Retargeting

Crea campanyes de retargeting dirigides a les teves audiències personalitzades.

  1. Ves al teu Administrador d'Anuncis.
  2. Fes clic a "Crear" i selecciona l'objectiu de la teva campanya (per exemple, "Conversiones").
  3. A la secció de "Audiència", selecciona l'audiència personalitzada que has creat.
  4. Configura la resta dels paràmetres de la teva campanya (pressupost, calendari, etc.).

Conclusió

La configuració de retargeting a Facebook Ads és una estratègia poderosa per reconnectar amb usuaris que han mostrat interès en el teu producte o servei. En seguir aquests passos, pots maximitzar les teves oportunitats de conversió i millorar significativament el teu ROI. Assegura't de monitoritzar i ajustar les teves campanyes regularment per obtenir els millors resultats.
