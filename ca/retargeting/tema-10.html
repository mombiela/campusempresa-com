Introducció

En aquesta secció del curs sobre Retargeting, aprendrem a crear anuncis personalitzats que s'adaptin a les necessitats i comportaments específics dels usuaris. La personalització d'anuncis és crucial per augmentar l'efectivitat de les campanyes de retargeting, ja que permet oferir contingut rellevant i atractiu per a cada segment d'audiència.

Conceptes Clau

  • Segmentació d'Audiència: Dividir els usuaris en grups basats en el seu comportament, interessos i dades demogràfiques.
  • Creativitat Dinàmica: Utilitzar elements variables en els anuncis que canvien segons el perfil de l'usuari.
  • A/B Testing: Provar diferents versions d'anuncis per determinar quina és més efectiva.
  • Píxel de Seguiment: Codi inserit al lloc web per rastrejar les accions dels usuaris i recopilar dades.

Segmentació d'Audiència

La segmentació d'audiència és el primer pas per crear anuncis personalitzats. Aquí separem els usuaris en diferents grups segons diversos criteris.

Exemple de Segmentació

  • Usuaris que van abandonar el carret: Persones que van afegir productes al carret però no van completar la compra.
  • Visitants recurrents: Usuaris que han visitat el lloc web diverses vegades.
  • Nous visitants: Persones que visiten el lloc web per primera vegada.
# Exemple de segmentació d'audiència en Python
usuaris = [
    {"id": 1, "visites": 5, "carret": True},
    {"id": 2, "visites": 1, "carret": False},
    {"id": 3, "visites": 3, "carret": False},
    {"id": 4, "visites": 2, "carret": True}
]

segments = {
    "van_abandonar_carret": [u for u in usuaris if u["carret"]],
    "visitants_recurrents": [u for u in usuaris if u["visites"] > 1],
    "nous_visitants": [u for u in usuaris si u["visites"] == 1]
}

print(segments)

Creativitat Dinàmica

La creativitat dinàmica permet personalitzar els anuncis en temps real segons el perfil de l'usuari. Això s'aconsegueix utilitzant plantilles amb elements variables.

Exemple de Creativitat Dinàmica

  • Plantilla d'Anunci: "Hola, [Nom]! Encara estàs interessat en [Producte]?"
  • Variables: Nom de l'usuari, Producte visitat
<!-- Exemple de plantilla HTML per a creativitat dinàmica -->
<div>
    <p>Hola, {{nom}}! Encara estàs interessat en {{producte}}?</p>
</div>
// Exemple de JavaScript per inserir variables en la plantilla
const nom = "Joan";
const producte = "Sabates Esportives";

const anunci = document.querySelector('div p');
anunci.innerHTML = `Hola, ${nom}! Encara estàs interessat en ${producte}?`;

A/B Testing

L'A/B testing és una tècnica per comparar dues versions d'un anunci i determinar quina és més efectiva. Això es fa mostrant les dues versions a diferents segments de l'audiència i mesurant el seu rendiment.

Exemple d'A/B Testing

  • Versió A: "Compra ara i obtén un 10% de descompte!"
  • Versió B: "No et perdis el nostre descompte exclusiu del 10%!"
# Exemple d'A/B testing en Python
import random

usuaris = ["usuari1", "usuari2", "usuari3", "usuari4"]

# Dividir usuaris en dos grups
grup_a = random.sample(usuaris, len(usuaris)//2)
grup_b = [u for u in usuaris si u no està en grup_a]

# Mostrar diferents versions d'anuncis
for usuari in grup_a:
    print(f"{usuari}: Compra ara i obtén un 10% de descompte!")

for usuari in grup_b:
    print(f"{usuari}: No et perdis el nostre descompte exclusiu del 10%!")

Píxel de Seguiment

El píxel de seguiment és un petit fragment de codi que s'insereix al lloc web per rastrejar les accions dels usuaris. Això permet recopilar dades valuoses per a la personalització d'anuncis.

Exemple de Píxel de Seguiment

<!-- Exemple de píxel de seguiment en HTML -->
<img src="https://example.com/pixel?user_id=12345&action=visit" style="display:none;" />

Conclusió

La creació d'anuncis personalitzats és una tècnica poderosa en les campanyes de retargeting. En segmentar l'audiència, utilitzar creativitat dinàmica, realitzar A/B testing i emprar píxels de seguiment, podem crear anuncis altament rellevants i efectius. Aquests mètodes no només milloren l'experiència de l'usuari, sinó que també augmenten les taxes de conversió i el retorn de la inversió en publicitat.
