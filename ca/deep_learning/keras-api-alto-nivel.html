Introducció a Keras

Keras és una API d'alt nivell per construir i entrenar models d'aprenentatge profund. Va ser desenvolupada amb l'objectiu de permetre una experimentació ràpida i fàcil, i és coneguda per la seva simplicitat i facilitat d'ús. Keras es pot executar sobre diverses biblioteques de backend, com TensorFlow, Theano i Microsoft Cognitive Toolkit (CNTK).

Característiques Principals de Keras

  • Simplicitat i facilitat d'ús: Keras està dissenyat per ser intuïtiu i fàcil d'usar, la qual cosa permet als desenvolupadors construir models de manera ràpida i eficient.
  • Modularitat: Els models en Keras estan construïts a partir de components modulars que es poden combinar de diverses maneres.
  • Extensibilitat: Keras és altament extensible, la qual cosa permet als investigadors implementar noves capes, funcions de pèrdua i mètriques personalitzades.
  • Compatibilitat amb múltiples backends: Keras pot executar-se sobre TensorFlow, Theano i CNTK, la qual cosa proporciona flexibilitat en l'elecció del backend.

Construcció de Models en Keras

Keras proporciona dues formes principals de construir models: el model seqüencial i el model funcional.

Model Seqüencial

El model seqüencial és una pila lineal de capes. És adequat per a la majoria dels problemes d'aprenentatge profund.

Exemple de Codi: Model Seqüencial

from keras.models import Sequential
from keras.layers import Dense

# Crear un model seqüencial
model = Sequential()

# Afegir capes al model
model.add(Dense(32, input_shape=(784,), activation='relu'))
model.add(Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

En aquest exemple:

  • Es crea un model seqüencial.
  • S'afegeixen dues capes denses (fully connected).
  • Es compila el model especificant l'optimitzador, la funció de pèrdua i les mètriques.
  • S'imprimeix un resum del model.

Model Funcional

El model funcional és més flexible i permet la construcció de models més complexos, com xarxes neuronals amb múltiples entrades i sortides.

Exemple de Codi: Model Funcional

from keras.models import Model
from keras.layers import Input, Dense

# Definir les entrades
inputs = Input(shape=(784,))

# Afegir capes
x = Dense(32, activation='relu')(inputs)
outputs = Dense(10, activation='softmax')(x)

# Crear el model
model = Model(inputs=inputs, outputs=outputs)

# Compilar el model
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

En aquest exemple:

  • Es defineix una entrada usant Input.
  • S'afegeixen capes usant la sintaxi funcional.
  • Es crea el model especificant les entrades i sortides.
  • Es compila el model i s'imprimeix un resum.

Entrenament i Avaluació de Models

Una vegada que el model està construït i compilat, es pot entrenar i avaluar utilitzant els mètodes fit i evaluate.

Entrenament del Model

# Entrenar el model
model.fit(x_train, y_train, epochs=10, batch_size=32, validation_split=0.2)

En aquest exemple:

  • x_train i y_train són les dades d'entrenament.
  • epochs és el nombre de vegades que el model veurà el conjunt de dades complet.
  • batch_size és el nombre de mostres per actualització de gradient.
  • validation_split és la fracció de dades d'entrenament que s'utilitzarà com a dades de validació.

Avaluació del Model

# Avaluar el model
loss, accuracy = model.evaluate(x_test, y_test)
print(f'Loss: {loss}, Accuracy: {accuracy}')

En aquest exemple:

  • x_test i y_test són les dades de prova.
  • El mètode evaluate retorna la pèrdua i la precisió del model en les dades de prova.

Conclusió

Keras és una eina poderosa i fàcil d'usar per construir i entrenar models d'aprenentatge profund. La seva simplicitat i modularitat la fan ideal tant per a principiants com per a investigadors avançats. Amb Keras, pots construir models seqüencials simples o models funcionals complexos, i entrenar-los i avaluar-los amb facilitat.
