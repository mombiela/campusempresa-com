Què són les Xarxes Neuronals Convolucionals (CNN)?

Les Xarxes Neuronals Convolucionals (CNN, per les seves sigles en anglès) són un tipus de xarxa neuronal profunda que s'utilitza principalment per a l'anàlisi d'imatges i dades visuals. Les CNN són especialment efectives per a tasques com la classificació d'imatges, la detecció d'objectes i la segmentació d'imatges.

Conceptes Clau

  • Convolució: Operació matemàtica que combina dos conjunts d'informació. En el context de les CNN, s'utilitza per extreure característiques de les imatges.
  • Filtres/Kernels: Petites matrius que es desplacen sobre la imatge d'entrada per realitzar l'operació de convolució.
  • Mapes de Característiques: Resultats d'aplicar filtres a la imatge d'entrada.
  • Pooling: Tècnica per reduir la dimensionalitat dels mapes de característiques, mantenint la informació més rellevant.
  • Capes Convolucionals: Capes en les quals s'apliquen els filtres per extreure característiques.
  • Capes de Pooling: Capes que redueixen la dimensionalitat dels mapes de característiques.
  • Capes Completament Connectades: Capes finals de la xarxa on cada neurona està connectada a totes les neurones de la capa anterior.

Arquitectura d'una CNN

Una CNN típica està composta per diverses capes, cadascuna de les quals té un propòsit específic. A continuació es mostra una taula que resumeix les capes més comunes en una CNN:

| Tipus de Capa | Descripció | |-------------------------|-----------------------------------------------------------------------------| | Capa d'Entrada | La imatge d'entrada, generalment de mida fixa (per exemple, 224x224x3). | | Capa Convolucional | Aplica filtres per extreure característiques de la imatge. | | Capa d'Activació | Aplica una funció d'activació (per exemple, ReLU) per introduir no linealitat. | | Capa de Pooling | Redueix la dimensionalitat dels mapes de característiques (per exemple, Max Pooling). | | Capa Completament Connectada | Connecta totes les neurones de la capa anterior a cada neurona d'aquesta capa. | | Capa de Sortida | Produeix la sortida final, com una probabilitat de classificació. |

Exemple de Codi

A continuació es mostra un exemple de com construir una CNN simple utilitzant TensorFlow i Keras:

import tensorflow as tf
from tensorflow.keras import layers, models

# Definir el model
model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

# Afegir capes completament connectades
model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Resum del model
model.summary()

Explicació del Codi

  1. Importació de Llibreries: S'importen TensorFlow i Keras.
  2. Definició del Model: S'utilitza Sequential per definir una seqüència de capes.
    • Capes Convolucionals: Tres capes convolucionals amb diferents nombres de filtres.
    • Capes de Pooling: Dues capes de Max Pooling per reduir la dimensionalitat.
    • Capes Completament Connectades: Una capa Flatten per aplanar els mapes de característiques i dues capes denses.
  3. Compilació del Model: S'especifica l'optimitzador, la funció de pèrdua i les mètriques.
  4. Resum del Model: S'imprimeix un resum del model per verificar la seva estructura.

Conclusió

Les Xarxes Neuronals Convolucionals són una eina poderosa per a l'anàlisi d'imatges i dades visuals. Comprendre la seva arquitectura i com implementar una CNN bàsica és fonamental per a qualsevol professional que desitgi treballar en el camp del Deep Learning. En aquest tema, hem cobert els conceptes clau, l'arquitectura típica d'una CNN i un exemple pràctic d'implementació utilitzant TensorFlow i Keras.
