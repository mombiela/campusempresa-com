Introducció

En aquest projecte, aprendrem a construir i entrenar una Xarxa Neuronal Convolucional (CNN) per a la tasca de classificació d'imatges. Les CNNs són una classe de xarxes neuronals profundes que han demostrat ser molt efectives per al processament de dades amb una estructura de quadrícula, com les imatges.

Objectius del Projecte

  • Comprendre l'arquitectura bàsica d'una CNN.
  • Implementar una CNN utilitzant una biblioteca de deep learning com TensorFlow o PyTorch.
  • Entrenar la CNN en un conjunt de dades d'imatges.
  • Avaluar el rendiment del model i realitzar ajustos per millorar la seva precisió.

Conceptes Clau

  • Convolució: Operació matemàtica que combina dos conjunts d'informació.
  • Filtre/Kernels: Matrius petites que es desplacen sobre la imatge per detectar característiques.
  • Pooling: Tècnica de reducció de dimensionalitat que redueix la quantitat de paràmetres i el cost computacional.
  • Capes de Convolució: Capes que apliquen operacions de convolució a l'entrada.
  • Capes de Pooling: Capes que apliquen operacions de pooling a l'entrada.
  • Capes Fully Connected: Capes on cada neurona està connectada a totes les neurones de la capa anterior.

Arquitectura d'una CNN

Una CNN típica per a classificació d'imatges pot tenir la següent estructura:

| Tipus de Capa | Descripció | |-------------------|--------------------------------------------------| | Input Layer | Imatge d'entrada (e.g., 32x32x3 per CIFAR-10) | | Convolution Layer | Aplicació de filtres per extreure característiques | | Pooling Layer | Reducció de dimensionalitat | | Convolution Layer | Aplicació de filtres addicionals | | Pooling Layer | Reducció de dimensionalitat | | Fully Connected | Classificació basada en característiques extretes| | Output Layer | Prediccions de classe |

Implementació en TensorFlow

Instal·lació de TensorFlow

Abans de començar, assegura't de tenir TensorFlow instal·lat. Pots instal·lar-lo usant pip:

pip install tensorflow

Importació de Biblioteques

import tensorflow as tf
from tensorflow.keras import datasets, layers, models
import matplotlib.pyplot as plt

Càrrega i Preprocessament de Dades

Farem servir el conjunt de dades CIFAR-10 per a aquest projecte.

# Carregar el conjunt de dades CIFAR-10
(train_images, train_labels), (test_images, test_labels) = datasets.cifar10.load_data()

# Normalitzar les imatges a valors entre 0 i 1
train_images, test_images = train_images / 255.0, test_images / 255.0

Construcció del Model

model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10))

Compilació i Entrenament del Model

model.compile(optimizer='adam',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True),
              metrics=['accuracy'])

history = model.fit(train_images, train_labels, epochs=10, 
                    validation_data=(test_images, test_labels))

Avaluació del Model

test_loss, test_acc = model.evaluate(test_images, test_labels, verbose=2)
print(f'\nPrecisió en el conjunt de prova: {test_acc}')

Visualització de Resultats

plt.plot(history.history['accuracy'], label='Precisió en el conjunt d'entrenament')
plt.plot(history.history['val_accuracy'], label='Precisió en el conjunt de validació')
plt.xlabel('Època')
plt.ylabel('Precisió')
plt.ylim([0, 1])
plt.legend(loc='lower right')
plt.show()

Conclusió

En aquest projecte, hem construït i entrenat una CNN bàsica per a la classificació d'imatges utilitzant TensorFlow. Hem cobert els conceptes clau de les CNNs, l'arquitectura típica d'una CNN, i els passos necessaris per implementar i avaluar el model. Aquest projecte proporciona una base sòlida per explorar arquitectures més complexes i tècniques avançades en deep learning.
