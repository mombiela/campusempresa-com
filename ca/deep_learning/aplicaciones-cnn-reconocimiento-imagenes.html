Introducció a les Xarxes Neuronals Convolucionals (CNN)

Les Xarxes Neuronals Convolucionals (CNN) són un tipus de xarxa neuronal profunda que ha demostrat ser extremadament eficaç en tasques de reconeixement d'imatges. Les CNN estan dissenyades per processar dades amb una estructura de quadrícula, com les imatges, i són capaces de capturar característiques espacials i temporals a través d'operacions de convolució.

Arquitectura d'una CNN

Una CNN típica està composta per diverses capes, cadascuna amb un propòsit específic:

  • Capes Convolucionals: Apliquen filtres per extreure característiques de la imatge.
  • Capes de Pooling: Redueixen la dimensionalitat de les característiques extretes, mantenint la informació rellevant.
  • Capes de Normalització: Normalitzen les dades per accelerar l'entrenament.
  • Capes Completament Connectades: Actuen com un classificador final.

Exemple d'una Arquitectura CNN Simple

import tensorflow as tf
from tensorflow.keras import layers, models

model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

Aplicacions de CNN en Reconeixement d'Imatges

Les CNN s'utilitzen en una àmplia varietat d'aplicacions de reconeixement d'imatges. A continuació, es presenten algunes de les aplicacions més comunes:

Classificació d'Imatges

La classificació d'imatges és una de les aplicacions més bàsiques i comunes de les CNN. En aquesta tasca, l'objectiu és assignar una etiqueta a una imatge d'un conjunt predefinit de categories.

  • Exemple: Classificació de dígits escrits a mà (MNIST).

Detecció d'Objectes

La detecció d'objectes implica no només classificar objectes dins d'una imatge, sinó també localitzar cada objecte mitjançant un quadre delimitador.

  • Exemple: Detecció de vianants en imatges de trànsit.

Segmentació d'Imatges

La segmentació d'imatges és el procés de dividir una imatge en segments significatius, sovint per identificar objectes i límits dins d'una imatge.

  • Exemple: Segmentació de cèl·lules en imatges mèdiques.

Reconeixement Facial

El reconeixement facial és una aplicació específica de la classificació d'imatges que se centra en identificar i verificar persones a partir d'imatges de les seves cares.

  • Exemple: Sistemes de seguretat basats en reconeixement facial.

Comparació d'Aplicacions

| Aplicació | Descripció | Exemple d'Ús | |--------------------------|----------------------------------------------------------------------------|-----------------------------------| | Classificació d'Imatges | Assignar una etiqueta a una imatge d'un conjunt predefinit de categories. | Classificació de dígits (MNIST). | | Detecció d'Objectes | Classificar i localitzar objectes dins d'una imatge. | Detecció de vianants en trànsit. | | Segmentació d'Imatges | Dividir una imatge en segments significatius. | Segmentació de cèl·lules mèdiques.| | Reconeixement Facial | Identificar i verificar persones a partir d'imatges de les seves cares. | Sistemes de seguretat facial. |

Exemple de Codi: Classificació d'Imatges amb CNN

A continuació, es presenta un exemple de codi per a la classificació d'imatges utilitzant una CNN en el conjunt de dades MNIST.

import tensorflow as tf
from tensorflow.keras import datasets, layers, models

# Carregar el conjunt de dades MNIST
(train_images, train_labels), (test_images, test_labels) = datasets.mnist.load_data()

# Preprocessar les dades
train_images = train_images.reshape((60000, 28, 28, 1)).astype('float32') / 255
test_images = test_images.reshape((10000, 28, 28, 1)).astype('float32') / 255

# Construir el model
model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

# Compilar el model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Entrenar el model
model.fit(train_images, train_labels, epochs=5, batch_size=64)

# Avaluar el model
test_loss, test_acc = model.evaluate(test_images, test_labels)
print(f'Test accuracy: {test_acc}')

Conclusió

Les Xarxes Neuronals Convolucionals (CNN) han revolucionat el camp del reconeixement d'imatges, permetent avenços significatius en diverses aplicacions com la classificació d'imatges, la detecció d'objectes, la segmentació d'imatges i el reconeixement facial. La seva capacitat per extreure característiques rellevants de les imatges i la seva flexibilitat per adaptar-se a diferents tasques les converteixen en una eina essencial en l'arsenal de qualsevol professional de Deep Learning.
