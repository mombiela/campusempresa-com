En aquest tema, explorarem dues tècniques essencials en el Deep Learning per prevenir el sobreajustament: la regularització i el dropout. Aquestes tècniques són fonamentals per millorar la capacitat de generalització dels models de xarxes neuronals.

Regularització

La regularització és una tècnica utilitzada per reduir el sobreajustament en els models d'aprenentatge automàtic. El sobreajustament ocorre quan un model s'ajusta massa a les dades d'entrenament i no generalitza bé a les dades noves. Hi ha diverses formes de regularització, però les més comunes són L1 i L2.

L1 Regularització (Lasso)

  • Definició: La regularització L1 afegeix una penalització igual a la suma dels valors absoluts dels coeficients del model.
  • Fórmula: \[ \text{Loss} = \text{Loss original} + \lambda \sum_{i} |w_i| \]
  • Avantatges:
    • Pot portar a solucions esparses, és a dir, alguns coeficients es tornen exactament zero.
  • Exemple de codi: 
    from keras.regularizers import l1
model.add(Dense(64, input_dim=64, kernel_regularizer=l1(0.01)))

L2 Regularització (Ridge)

  • Definició: La regularització L2 afegeix una penalització igual a la suma dels quadrats dels coeficients del model.
  • Fórmula: \[ \text{Loss} = \text{Loss original} + \lambda \sum_{i} w_i^2 \]
  • Avantatges:
    • No condueix a solucions esparses, però pot reduir la magnitud dels coeficients.
  • Exemple de codi: 
    from keras.regularizers import l2
model.add(Dense(64, input_dim=64, kernel_regularizer=l2(0.01)))

Comparació entre L1 i L2

| Característica | L1 (Lasso) | L2 (Ridge) | |----------------|------------|------------| | Penalització | Suma de valors absoluts | Suma de quadrats | | Solucions esparses | Sí | No | | Ús comú | Selecció de característiques | Reducció de la magnitud dels coeficients |

Dropout

El dropout és una tècnica de regularització que consisteix a "apagar" aleatòriament un conjunt de neurones durant l'entrenament. Això ajuda a prevenir el sobreajustament en reduir la dependència dels models en neurones específiques.

Funcionament del Dropout

  • Definició: Durant cada pas d'entrenament, cada neurona té una probabilitat \( p \) de ser "apagada" (és a dir, la seva sortida s'estableix en zero).
  • Avantatges:
    • Redueix el sobreajustament.
    • Millora la capacitat de generalització del model.
  • Exemple de codi: 
    from keras.layers import Dropout
model.add(Dense(64, input_dim=64, activation='relu'))
model.add(Dropout(0.5))

Exemple Pràctic

A continuació, es mostra un exemple complet de com implementar la regularització i el dropout en un model de xarxa neuronal utilitzant Keras:

from keras.models import Sequential
from keras.layers import Dense, Dropout
from keras.regularizers import l2

# Crear el model
model = Sequential()

# Afegir capes amb regularització L2 i dropout
model.add(Dense(64, input_dim=64, activation='relu', kernel_regularizer=l2(0.01)))
model.add(Dropout(0.5))
model.add(Dense(64, activation='relu', kernel_regularizer=l2(0.01)))
model.add(Dropout(0.5))
model.add(Dense(10, activation='softmax'))

# Compilar el model
model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])

# Resum del model
model.summary()

Conclusió

La regularització i el dropout són tècniques crucials per millorar la capacitat de generalització dels models de xarxes neuronals i prevenir el sobreajustament. La regularització L1 i L2 ajuden a controlar la magnitud dels coeficients del model, mentre que el dropout redueix la dependència de neurones específiques. En combinar aquestes tècniques, podem construir models més robustos i eficients en Deep Learning.
