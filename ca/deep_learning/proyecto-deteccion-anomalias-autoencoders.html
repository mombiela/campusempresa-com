Introducció

En aquest projecte, explorarem com utilitzar autoencoders, una classe de xarxes neuronals, per a la detecció d'anomalies en dades. Els autoencoders són especialment útils per a aquesta tasca a causa de la seva capacitat per aprendre representacions comprimides de les dades i reconstruir-les. Les anomalies poden ser identificades com aquelles dades que l'autoencoder no pot reconstruir adequadament.

Objectius del Projecte

  • Comprendre el concepte d'autoencoders i la seva arquitectura.
  • Implementar un autoencoder utilitzant una biblioteca de deep learning (TensorFlow/Keras).
  • Aplicar l'autoencoder per detectar anomalies en un conjunt de dades.
  • Avaluar el rendiment del model de detecció d'anomalies.

Conceptes Clau

  • Autoencoders: Xarxes neuronals dissenyades per aprendre una representació (codificació) eficient d'un conjunt de dades, típicament per reducció de dimensionalitat o detecció d'anomalies.
  • Codificador (Encoder): Part de l'autoencoder que redueix la dimensionalitat de les dades d'entrada.
  • Decodificador (Decoder): Part de l'autoencoder que reconstrueix les dades a partir de la representació comprimida.
  • Pèrdua de Reconstrucció: Mètrica que mesura la diferència entre les dades originals i les dades reconstruïdes per l'autoencoder.

Arquitectura d'un Autoencoder

Un autoencoder típic consta de dues parts principals:

  1. Codificador (Encoder):

    • Redueix la dimensionalitat de les dades d'entrada.
    • Compost per capes denses (fully connected) o convolucionals.

  2. Decodificador (Decoder):

    • Reconstrueix les dades a partir de la representació comprimida.
    • Compost per capes denses o convolucionals que "desfan" la compressió realitzada pel codificador.

Exemple d'Arquitectura

| Part | Tipus de Capa | Dimensions d'Entrada | Dimensions de Sortida | |-------------|---------------------|----------------------|-----------------------| | Codificador | Dense | 784 | 128 | | Codificador | Dense | 128 | 64 | | Codificador | Dense | 64 | 32 | | Decodificador| Dense | 32 | 64 | | Decodificador| Dense | 64 | 128 | | Decodificador| Dense | 128 | 784 |

Implementació de l'Autoencoder

Pas 1: Importar Biblioteques Necessàries

import numpy as np
import tensorflow as tf
from tensorflow.keras.models import Model
from tensorflow.keras.layers import Input, Dense
from tensorflow.keras.datasets import mnist

Pas 2: Preparar les Dades

# Carregar el conjunt de dades MNIST
(x_train, _), (x_test, _) = mnist.load_data()

# Normalitzar les dades
x_train = x_train.astype('float32') / 255.
x_test = x_test.astype('float32') / 255.

# Aplanar les imatges (28x28) a vectors de 784 elements
x_train = x_train.reshape((len(x_train), np.prod(x_train.shape[1:])))
x_test = x_test.reshape((len(x_test), np.prod(x_test.shape[1:])))

Pas 3: Definir l'Arquitectura de l'Autoencoder

# Definir la mida de l'entrada
input_dim = x_train.shape[1]

# Definir el codificador
input_layer = Input(shape=(input_dim,))
encoder = Dense(128, activation='relu')(input_layer)
encoder = Dense(64, activation='relu')(encoder)
encoder = Dense(32, activation='relu')(encoder)

# Definir el decodificador
decoder = Dense(64, activation='relu')(encoder)
decoder = Dense(128, activation='relu')(decoder)
decoder = Dense(input_dim, activation='sigmoid')(decoder)

# Crear el model autoencoder
autoencoder = Model(inputs=input_layer, outputs=decoder)
autoencoder.compile(optimizer='adam', loss='binary_crossentropy')

Pas 4: Entrenar l'Autoencoder

autoencoder.fit(x_train, x_train,
                epochs=50,
                batch_size=256,
                shuffle=True,
                validation_data=(x_test, x_test))

Pas 5: Avaluar el Model i Detectar Anomalies

# Obtenir les reconstruccions del conjunt de prova
reconstructions = autoencoder.predict(x_test)

# Calcular la pèrdua de reconstrucció
reconstruction_loss = np.mean(np.square(x_test - reconstructions), axis=1)

# Definir un llindar per a la detecció d'anomalies
threshold = np.percentile(reconstruction_loss, 95)

# Identificar les anomalies
anomalies = reconstruction_loss > threshold

print(f'Nombre d'anomalies detectades: {np.sum(anomalies)}')
