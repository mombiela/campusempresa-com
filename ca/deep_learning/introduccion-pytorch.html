Què és PyTorch?

PyTorch és una biblioteca d'aprenentatge automàtic de codi obert desenvolupada pel laboratori de recerca en IA de Facebook (FAIR). És àmpliament utilitzada en la recerca i la indústria a causa de la seva flexibilitat i facilitat d'ús. PyTorch permet als desenvolupadors construir i entrenar models d'aprenentatge profund de manera eficient.

Característiques clau de PyTorch

  • Tensor Computation: Similar a NumPy, però amb suport per a acceleració en GPU.
  • Autograd: Sistema de diferenciació automàtica que facilita la construcció i l'entrenament de xarxes neuronals.
  • Models Dinàmics: Permet la creació de gràfics computacionals dinàmics, la qual cosa és útil per a tasques complexes i models personalitzats.
  • Biblioteca Extensa: Inclou una àmplia gamma d'eines i biblioteques per facilitar el desenvolupament de models d'aprenentatge profund.

Instal·lació de PyTorch

Per instal·lar PyTorch, pots utilitzar pip o conda. Aquí et mostrem com fer-ho amb pip:

pip install torch torchvision

I amb conda:

conda install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch

Tensors en PyTorch

Els tensors són l'estructura de dades fonamental en PyTorch. Són similars als arrays de NumPy, però amb la capacitat de ser processats en una GPU per accelerar les operacions.

Creació de Tensors

import torch

# Crear un tensor buit
x = torch.empty(5, 3)
print(x)

# Crear un tensor amb valors aleatoris
x = torch.rand(5, 3)
print(x)

# Crear un tensor amb valors zero
x = torch.zeros(5, 3, dtype=torch.long)
print(x)

# Crear un tensor directament des de dades
x = torch.tensor([5.5, 3])
print(x)

Operacions amb Tensors

# Suma de dos tensors
y = torch.rand(5, 3)
print(x + y)

# Multiplicació de tensors
print(x * y)

# Slicing de tensors
print(x[:, 1])

Autograd: Diferenciació Automàtica

El mòdul autograd de PyTorch proporciona diferenciació automàtica per a totes les operacions en tensors. Això és essencial per a l'entrenament de xarxes neuronals.

Exemple d'ús d'Autograd

import torch

# Crear un tensor amb l'opció de calcular gradients
x = torch.ones(2, 2, requires_grad=True)
print(x)

# Realitzar una operació
y = x + 2
print(y)

# Una altra operació
z = y * y * 3
out = z.mean()
print(out)

# Calcular gradients
out.backward()
print(x.grad)

Construcció de Xarxes Neuronals

PyTorch proporciona el mòdul torch.nn per ajudar en la construcció de xarxes neuronals. Aquest mòdul conté totes les capes i funcions d'activació necessàries per construir models complexos.

Exemple d'una Xarxa Neuronal Simple

import torch
import torch.nn as nn
import torch.optim as optim

# Definir una xarxa neuronal simple
class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

net = Net()
print(net)

# Definir una funció de pèrdua i un optimitzador
criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.01)

# Exemple d'entrenament
for epoch in range(2):  # loop over the dataset multiple times
    inputs = torch.randn(64, 784)
    labels = torch.randint(0, 10, (64,))

    optimizer.zero_grad()  # zero the parameter gradients
    outputs = net(inputs)
    loss = criterion(outputs, labels)
    loss.backward()
    optimizer.step()

    print(f'Epoch {epoch+1}, Loss: {loss.item()}')

Conclusió

PyTorch és una eina poderosa i flexible per al desenvolupament de models d'aprenentatge profund. La seva facilitat d'ús i la seva capacitat per manejar gràfics computacionals dinàmics el fan ideal tant per a la recerca com per a aplicacions industrials. En aquesta introducció, hem cobert els conceptes bàsics de PyTorch, incloent la creació i manipulació de tensors, l'ús d'autograd per a la diferenciació automàtica i la construcció de xarxes neuronals simples. Amb aquesta base, estaràs preparat per explorar més a fons les capacitats avançades de PyTorch en l'aprenentatge profund.
