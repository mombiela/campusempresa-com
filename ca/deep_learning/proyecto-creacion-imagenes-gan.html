Introducció a les Xarxes Generatives Antagòniques (GAN)

Les Xarxes Generatives Antagòniques (GAN) són un tipus de model d'aprenentatge profund que pot generar dades noves i realistes a partir d'un conjunt de dades d'entrenament. Van ser introduïdes per Ian Goodfellow el 2014 i han revolucionat la generació d'imatges, vídeos i altres tipus de dades.

Conceptes Clau

  • Generador: Una xarxa neuronal que genera dades falses a partir de soroll aleatori.
  • Discriminador: Una xarxa neuronal que intenta distingir entre dades reals i falses.
  • Joc Minimax: El generador i el discriminador competeixen entre si en un joc de suma zero.
  • Funció de Pèrdua: Mesura com de bé el generador i el discriminador estan actuant.

Arquitectura d'una GAN

Una GAN consta de dos components principals:

  1. Generador (G):

    • Pren un vector de soroll com a entrada.
    • Produeix una imatge falsa que intenta assemblar-se a les imatges reals del conjunt de dades.

  2. Discriminador (D):

    • Pren una imatge com a entrada.
    • Classifica la imatge com a real (provinent del conjunt de dades) o falsa (generada pel generador).

Exemple de Codi

A continuació, es mostra un exemple bàsic de com implementar una GAN utilitzant TensorFlow i Keras.

import tensorflow as tf
from tensorflow.keras import layers

# Definir el generador
def build_generator():
    model = tf.keras.Sequential()
    model.add(layers.Dense(256, input_dim=100, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(512, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(1024, activation='relu'))
    model.add(layers.BatchNormalization())
    model.add(layers.Dense(28 * 28 * 1, activation='tanh'))
    model.add(layers.Reshape((28, 28, 1)))
    return model

# Definir el discriminador
def build_discriminator():
    model = tf.keras.Sequential()
    model.add(layers.Flatten(input_shape=(28, 28, 1)))
    model.add(layers.Dense(512, activation='relu'))
    model.add(layers.Dense(256, activation='relu'))
    model.add(layers.Dense(1, activation='sigmoid'))
    return model

# Crear el generador i el discriminador
generator = build_generator()
discriminator = build_discriminator()

# Compilar el discriminador
discriminator.compile(optimizer='adam', loss='binary_crossentropy', metrics=['accuracy'])

# Crear la GAN combinada
discriminator.trainable = False
gan_input = layers.Input(shape=(100,))
generated_image = generator(gan_input)
gan_output = discriminator(generated_image)
gan = tf.keras.Model(gan_input, gan_output)

# Compilar la GAN
gan.compile(optimizer='adam', loss='binary_crossentropy')

Entrenament de la GAN

L'entrenament d'una GAN implica entrenar tant el generador com el discriminador en un bucle alternat.

Passos d'Entrenament

  1. Entrenar el Discriminador:

    • Utilitzar un lot d'imatges reals i un lot d'imatges generades.
    • Calcular la pèrdua i actualitzar els pesos del discriminador.

  2. Entrenar el Generador:

    • Generar un lot d'imatges falses.
    • Passar aquestes imatges al discriminador.
    • Calcular la pèrdua basada en la capacitat del discriminador per distingir les imatges falses.
    • Actualitzar els pesos del generador per enganyar millor el discriminador.

Exemple de Codi d'Entrenament

import numpy as np

# Paràmetres d'entrenament
epochs = 10000
batch_size = 64
sample_interval = 1000

# Carregar i preprocessar el conjunt de dades MNIST
(X_train, _), (_, _) = tf.keras.datasets.mnist.load_data()
X_train = X_train / 127.5 - 1.0
X_train = np.expandir(X_train, axis=3)

# Etiquetes per a imatges reals i falses
real = np.ones((batch_size, 1))
fake = np.zeros((batch_size, 1))

for epoch in range(epochs):
    # Entrenar el discriminador
    idx = np.random.randint(0, X_train.shape[0], batch_size)
    real_images = X_train[idx]

    noise = np.random.normal(0, 1, (batch_size, 100))
    generated_images = generator.predict(noise)

    d_loss_real = discriminator.train_on_batch(real_images, real)
    d_loss_fake = discriminator.train_on_batch(generated_images, fake)
    d_loss = 0.5 * np.add(d_loss_real, d_loss_fake)

    # Entrenar el generador
    noise = np.random.normal(0, 1, (batch_size, 100))
    g_loss = gan.train_on_batch(noise, real)

    # Imprimir el progrés
    if epoch % sample_interval == 0:
        print(f"{epoch} [D loss: {d_loss[0]}, acc.: {100*d_loss[1]}%] [G loss: {g_loss}]")
