Introducció a Transfer Learning

Transfer Learning és una tècnica en Deep Learning on un model desenvolupat per a una tasca es reutilitza com a punt de partida per a un model en una segona tasca. Aquesta tècnica és especialment útil quan es disposa d'una quantitat limitada de dades per a la tasca específica, però es té accés a un model preentrenat en una tasca similar amb una gran quantitat de dades.

Conceptes Clau

  • Model Preentrenat: Un model que ha estat entrenat en una tasca gran i general, com la classificació d'imatges en ImageNet.
  • Fine-Tuning: Ajustar un model preentrenat a una nova tasca específica, generalment amb una taxa d'aprenentatge més baixa.
  • Feature Extraction: Utilitzar les capes d'un model preentrenat per extreure característiques rellevants de les dades i entrenar només les capes finals del model.

Avantatges de Transfer Learning

  • Reducció del Temps d'Entrenament: Aprofitar un model preentrenat pot reduir significativament el temps necessari per entrenar un nou model.
  • Millora del Rendiment: Els models preentrenats solen tenir un rendiment superior en tasques específiques a causa del seu entrenament previ en grans conjunts de dades.
  • Menor Necessitat de Dades: Transfer Learning permet obtenir bons resultats amb menys dades etiquetades, la qual cosa és beneficiosa en situacions on les dades són escasses o costoses d'obtenir.

Exemple de Transfer Learning en Keras

A continuació, es presenta un exemple de com implementar Transfer Learning utilitzant Keras i un model preentrenat com VGG16.

import tensorflow as tf
from tensorflow.keras.applications import VGG16
from tensorflow.keras.models import Model
from tensorflow.keras.layers import Dense, Flatten
from tensorflow.keras.optimizers import Adam

# Carregar el model VGG16 preentrenat sense la capa superior (classificació)
base_model = VGG16(weights='imagenet', include_top=False, input_shape=(224, 224, 3))

# Congelar les capes del model base
for layer in base_model.layers:
    layer.trainable = False

# Afegir capes personalitzades per a la nova tasca
x = base_model.output
x = Flatten()(x)
x = Dense(1024, activation='relu')(x)
predictions = Dense(10, activation='softmax')(x)  # Suposem que tenim 10 classes

# Crear el model final
model = Model(inputs=base_model.input, outputs=predictions)

# Compilar el model
model.compile(optimizer=Adam(learning_rate=0.0001), loss='categorical_crossentropy', metrics=['accuracy'])

# Resum del model
model.summary()

Explicació del Codi

  1. Carregar el Model Preentrenat: Utilitzem el model VGG16 preentrenat en ImageNet, excloent la capa superior de classificació (include_top=False).
  2. Congelar les Capaes del Model Base: Establim trainable=False per a totes les capes del model base, de manera que els seus pesos no s'actualitzin durant l'entrenament.
  3. Afegir Capaes Personalitzades: Afegim noves capes Flatten, Dense i Dense per adaptar el model a la nostra tasca específica.
  4. Compilar el Model: Utilitzem l'optimitzador Adam amb una taxa d'aprenentatge baixa i la funció de pèrdua categorical_crossentropy.
  5. Entrenar el Model: Entrenem el model amb les nostres dades d'entrenament i validació.

Conclusió

Transfer Learning és una tècnica poderosa en Deep Learning que permet reutilitzar models preentrenats per a noves tasques, reduint el temps d'entrenament i millorant el rendiment, especialment quan es disposa de poques dades. A través d'exemples pràctics, hem vist com implementar Transfer Learning utilitzant Keras, la qual cosa facilita la seva aplicació en projectes del món real.
