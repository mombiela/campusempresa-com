Configuració de l'Entorn de Desenvolupament
Hola Món en Assembly ►

Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre llenguatge Assembly. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar amb la programació en Assembly.

Conceptes Clau

  • Assembler: Un programa que converteix el codi en llenguatge Assembly a codi màquina.
  • IDE (Entorn de Desenvolupament Integrat): Programari que proporciona instal·lacions completes als programadors per al desenvolupament de programari.
  • Depurador: Una eina utilitzada per provar i depurar programes.
  • Emulador: Programari que permet que un sistema informàtic es comporti com un altre.

Eines Necessàries

Per començar a programar en Assembly, necessitaràs les següents eines:

  1. Assembler: NASM (Netwide Assembler) és una opció popular.
  2. IDE/Editor de Text: Visual Studio Code, Sublime Text, o qualsevol editor de text de la teva elecció.
  3. Depurador: GDB (GNU Debugger) és comunament utilitzat.
  4. Emulador: QEMU o Bochs es poden utilitzar per emular maquinari per executar codi Assembly.

Instal·lació de l'Assembler NASM

A Windows

  1. Descarrega l'instal·lador de NASM des del lloc web oficial de NASM.
  2. Executa l'instal·lador i segueix les instruccions en pantalla.
  3. Afegeix NASM al PATH del sistema:
    • Obre el Panell de Control.
    • Ves a Sistema i Seguretat > Sistema > Configuració avançada del sistema.
    • Fes clic a Variables d'entorn.
    • A la secció de Variables del sistema, troba la variable PATH i fes clic a Editar.
    • Afegeix el camí a l'executable de NASM a la variable PATH.

A macOS

  1. Obre Terminal.
  2. Instal·la NASM utilitzant Homebrew: 
    brew install nasm

A Linux

  1. Obre Terminal.
  2. Instal·la NASM utilitzant el gestor de paquets:
    • Per a sistemes basats en Debian (p. ex., Ubuntu): 
      sudo apt-get install nasm
    • Per a sistemes basats en Red Hat (p. ex., Fedora): 
      sudo dnf install nasm

Configuració d'un IDE/Editor de Text

Visual Studio Code

  1. Descarrega i instal·la Visual Studio Code des del lloc web oficial.
  2. Instal·la l'extensió Assembly x86:
    • Obre Visual Studio Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca "Assembly x86" i instal·la l'extensió.

Sublime Text

  1. Descarrega i instal·la Sublime Text des del lloc web oficial.
  2. Instal·la el paquet Assembly:
    • Obre Sublime Text.
    • Ves a Preferències > Control de Paquets.
    • Selecciona "Instal·lar Paquet" i cerca "Assembly".

Instal·lació i Ús d'un Depurador

GDB (GNU Debugger)

  1. Instal·la GDB:

    • A Windows: Utilitza MinGW per instal·lar GDB.
    • A macOS: Utilitza Homebrew: 
      brew install gdb
    • A Linux:
      • Per a sistemes basats en Debian: 
        sudo apt-get install gdb
      • Per a sistemes basats en Red Hat: 
        sudo dnf install gdb

  2. Comandes bàsiques de GDB:

    • Inicia GDB amb un executable: 
      gdb ./your_program
    • Estableix un punt de ruptura: 
      break main
    • Executa el programa: 
      run
    • Passa pel codi: 
      step
    • Imprimeix el valor d'una variable: 
      print variable_name

Ús d'un Emulador

QEMU

  1. Instal·la QEMU:

    • A Windows: Descarrega l'instal·lador des del lloc web oficial de QEMU.
    • A macOS: Utilitza Homebrew: 
      brew install qemu
    • A Linux:
      • Per a sistemes basats en Debian: 
        sudo apt-get install qemu
      • Per a sistemes basats en Red Hat: 
        sudo dnf install qemu

  2. Executar un programa Assembly a QEMU:

    • Compila el teu codi Assembly a un binari: 
      nasm -f bin -o program.bin program.asm
    • Executa el binari a QEMU: 
      qemu-system-x86_64 -drive format=raw,file=program.bin

Conclusió

Configurar l'entorn de desenvolupament per a la programació en Assembly implica instal·lar un assembler, triar un editor de text o IDE adequat, configurar un depurador i, opcionalment, utilitzar un emulador. Seguint els passos descrits en aquesta secció, tindràs un entorn completament funcional preparat per escriure, provar i depurar codi Assembly. Aquesta configuració servirà com a base per al teu viatge des de principiant fins a programador avançat en Assembly.
