Introducció al llenguatge d'assemblador

El llenguatge d'assemblador és un llenguatge de programació de baix nivell que està estretament relacionat amb el codi màquina. Proporciona una manera d'escriure programes que són executats directament per la CPU. Entendre les instruccions bàsiques d'assemblador és crucial per escriure codi d'assemblador eficient i efectiu.

Conceptes clau

  • Arquitectura del conjunt d'instruccions (ISA): El conjunt d'instruccions que una CPU particular pot executar.
  • Registres: Petits llocs d'emmagatzematge ràpids dins de la CPU utilitzats per contenir dades i adreces.
  • Operands: Les dades o adreces sobre les quals operen les instruccions.
  • Mnemònics: Representacions llegibles per humans de les instruccions de màquina.

Instruccions bàsiques d'assemblador

Instruccions de moviment de dades

Aquestes instruccions s'utilitzen per moure dades entre registres, memòria i ports d'E/S.

  • MOV: Mou dades d'una ubicació a una altra.

    MOV AX, BX  ; Mou el contingut del registre BX al registre AX
MOV [1234H], AX  ; Mou el contingut del registre AX a la ubicació de memòria 1234H

  • PUSH/POP: Empeny dades a la pila i treu dades de la pila.

    PUSH AX  ; Empeny el contingut d'AX a la pila
POP BX   ; Treu el cim de la pila a BX

Instruccions aritmètiques

Aquestes instruccions realitzen operacions aritmètiques com la suma, la resta, la multiplicació i la divisió.

  • ADD: Suma dos operands.

    ADD AX, BX  ; Suma el contingut de BX a AX i emmagatzema el resultat a AX

  • SUB: Resta un operand d'un altre.

    SUB AX, BX  ; Resta el contingut de BX d'AX i emmagatzema el resultat a AX

  • MUL: Multiplica dos operands.

    MUL BX  ; Multiplica AX per BX, resultat emmagatzemat a AX (per operands de 16 bits)

  • DIV: Divideix un operand per un altre.

    DIV BX  ; Divideix AX per BX, quocient a AX, residu a DX (per operands de 16 bits)

Instruccions lògiques

Aquestes instruccions realitzen operacions lògiques a nivell de bits.

  • AND: Realitza una operació AND a nivell de bits.

    AND AX, BX  ; Realitza AND a nivell de bits entre AX i BX, emmagatzema el resultat a AX

  • OR: Realitza una operació OR a nivell de bits.

    OR AX, BX  ; Realitza OR a nivell de bits entre AX i BX, emmagatzema el resultat a AX

  • XOR: Realitza una operació XOR a nivell de bits.

    XOR AX, BX  ; Realitza XOR a nivell de bits entre AX i BX, emmagatzema el resultat a AX

  • NOT: Realitza una operació NOT a nivell de bits.

    NOT AX  ; Realitza NOT a nivell de bits a AX, emmagatzema el resultat a AX

Instruccions de control de flux

Aquestes instruccions alteren el flux d'execució en un programa.

  • JMP: Salt incondicional a una adreça especificada.

    JMP 1234H  ; Salta a la ubicació de memòria 1234H

  • Sals condicionals: Salta a una adreça especificada si es compleix una condició.

    JE 1234H  ; Salta a 1234H si el flag de zero (ZF) està establert (igual)
JNE 1234H ; Salta a 1234H si el flag de zero (ZF) no està establert (diferent)

  • CALL/RET: Crida una subrutina i retorna d'una subrutina.

    CALL 1234H  ; Crida la subrutina a la ubicació de memòria 1234H
RET         ; Retorna de la subrutina

Programa d'exemple

Aquí hi ha un programa d'exemple senzill que demostra algunes de les instruccions bàsiques d'assemblador discutides anteriorment:

section .data
    num1 dw 10
    num2 dw 20
    result dw 0

section .text
    global _start

_start:
    ; Carrega num1 i num2 als registres
    MOV AX, [num1]
    MOV BX, [num2]

    ; Suma num1 i num2
    ADD AX, BX

    ; Emmagatzema el resultat
    MOV [result], AX

    ; Surt del programa
    MOV AX, 1
    INT 0x80

Conclusió

Entendre les instruccions bàsiques d'assemblador és la base per escriure programes d'assemblador efectius. Dominant les instruccions de moviment de dades, aritmètiques, lògiques i de control de flux, pots començar a escriure codi més complex i eficient. Practica aquestes instruccions amb programes senzills per construir una comprensió sòlida abans de passar a temes més avançats.
