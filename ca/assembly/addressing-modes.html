Introducció als Modes d'Adreçament

Els modes d'adreçament són tècniques utilitzades en llenguatge assemblador per especificar l'operand(s) per a les instruccions. Entendre els modes d'adreçament és crucial per escriure codi assemblador eficient i efectiu.

Conceptes Clau

  • Operand: Les dades sobre les quals opera la instrucció.
  • Mode d'Adreçament: La manera en què s'especifica l'operand.

Tipus de Modes d'Adreçament

Mode d'Adreçament Immediat

En el mode d'adreçament immediat, l'operand s'especifica directament en la instrucció.

Exemple

MOV AL, 5  ; Mou el valor immediat 5 al registre AL
  • Explicació: El valor 5 es proporciona directament en la instrucció.

Mode d'Adreçament de Registre

En el mode d'adreçament de registre, l'operand es troba en un registre.

Exemple

MOV AX, BX  ; Mou el valor del registre BX al registre AX
  • Explicació: El valor en el registre BX es mou al registre AX.

Mode d'Adreçament Directe

En el mode d'adreçament directe, l'adreça de l'operand s'especifica en la instrucció.

Exemple

MOV AX, [1234h]  ; Mou el valor de l'adreça de memòria 1234h al registre AX
  • Explicació: El valor de l'adreça de memòria 1234h es mou al registre AX.

Mode d'Adreçament Indirecte

En el mode d'adreçament indirecte, l'adreça de l'operand es troba en un registre.

Exemple

MOV AX, [BX]  ; Mou el valor de l'adreça continguda en el registre BX al registre AX
  • Explicació: El valor de l'adreça de memòria continguda en BX es mou a AX.

Mode d'Adreçament Indexat

En el mode d'adreçament indexat, l'adreça de l'operand es determina utilitzant una adreça base i un índex.

Exemple

MOV AX, [BX+SI]  ; Mou el valor de l'adreça (BX + SI) al registre AX
  • Explicació: El valor de l'adreça calculada sumant BX i SI es mou a AX.

Mode d'Adreçament Basat Indexat

En el mode d'adreçament basat indexat, l'adreça de l'operand es determina utilitzant una adreça base, un índex i un desplaçament opcional.

Exemple

MOV AX, [BX+SI+10h]  ; Mou el valor de l'adreça (BX + SI + 10h) al registre AX
  • Explicació: El valor de l'adreça calculada sumant BX, SI i 10h es mou a AX.

Mode d'Adreçament Relatiu

En el mode d'adreçament relatiu, l'adreça de l'operand es determina relativa a l'adreça de la instrucció actual.

Exemple

JMP SHORT LABEL  ; Salta a l'adreça relativa a la instrucció actual
LABEL:
  • Explicació: El salt es fa a l'adreça especificada per LABEL, relativa a la instrucció actual.

Comparació dels Modes d'Adreçament

| Mode d'Adreçament | Especificació de l'Operand | Exemple | |-------------------------|------------------------------------------------|--------------------------| | Immediat | L'operand s'especifica directament | MOV AL, 5 | | De Registre | L'operand està en un registre | MOV AX, BX | | Directe | S'especifica l'adreça de l'operand | MOV AX, [1234h] | | Indirecte | L'adreça de l'operand està en un registre | MOV AX, [BX] | | Indexat | L'adreça és adreça base + índex | MOV AX, [BX+SI] | | Basat Indexat | L'adreça és adreça base + índex + desplaçament | MOV AX, [BX+SI+10h] | | Relatiu | L'adreça és relativa a la instrucció actual | JMP SHORT LABEL |

Exercicis

Exercici 1: Adreçament Immediat

Escriu una instrucció d'assemblador per moure el valor 10 al registre CL.

Exercici 2: Adreçament de Registre

Escriu una instrucció d'assemblador per moure el valor del registre DX al registre CX.

Exercici 3: Adreçament Directe

Escriu una instrucció d'assemblador per moure el valor de l'adreça de memòria 2000h al registre AX.

Exercici 4: Adreçament Indirecte

Escriu una instrucció d'assemblador per moure el valor de l'adreça continguda en el registre SI al registre BX.

Exercici 5: Adreçament Indexat

Escriu una instrucció d'assemblador per moure el valor de l'adreça calculada sumant DI a BP al registre DX.

Exercici 6: Adreçament Basat Indexat

Escriu una instrucció d'assemblador per moure el valor de l'adreça calculada sumant BX, DI i 20h al registre AX.

Exercici 7: Adreçament Relatiu

Escriu una instrucció d'assemblador per saltar a una etiqueta NEXT que està 5 bytes per davant de la instrucció actual.

Conclusió

Els modes d'adreçament són fonamentals per entendre i escriure programes en llenguatge assemblador. Cada mode proporciona una manera diferent d'especificar els operands, permetent flexibilitat i eficiència en la codificació. En dominar aquests modes, pots escriure codi assemblador més efectiu i optimitzat.
