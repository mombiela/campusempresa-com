Introducció

En aquesta secció, explorarem una varietat de projectes de mostra que t'ajudaran a aplicar els conceptes que has après en la programació en Assembly. Aquests projectes van des de nivells per a principiants fins a avançats, permetent-te desenvolupar gradualment les teves habilitats i confiança.

Projectes per a Principiants

1. Hola Món

El clàssic programa "Hola Món" és un bon punt de partida per als principiants. T'introdueix a la sintaxi bàsica d'Assembly i al procés d'escriure, assemblar i executar un programa senzill.

section .data
    hello db 'Hola, Món!',0

section .text
    global _start

_start:
    ; Escriu la cadena a stdout
    mov eax, 4          ; número de syscall per a sys_write
    mov ebx, 1          ; descriptor de fitxer 1 és stdout
    mov ecx, hello      ; punter a la cadena
    mov edx, 13         ; longitud de la cadena
    int 0x80            ; crida al kernel

    ; Surt del programa
    mov eax, 1          ; número de syscall per a sys_exit
    xor ebx, ebx        ; codi de retorn 0
    int 0x80            ; crida al kernel
  • Conceptes Clau:
    • Secció de dades i secció de text.
    • Crides de sistema per escriure a stdout i sortir del programa.
    • Ús bàsic de registres.

2. Calculadora Simple

Crea una calculadora simple que pugui realitzar operacions aritmètiques bàsiques com suma, resta, multiplicació i divisió.

section .data
    num1 db 10
    num2 db 5
    result db 0

section .text
    global _start

_start:
    ; Suma
    mov al, [num1]
    add al, [num2]
    mov [result], al

    ; Surt del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptes Clau:
    • Operacions aritmètiques bàsiques.
    • Manipulació de registres.
    • Emmagatzematge i recuperació de dades de la memòria.

Projectes Intermedis

1. Reversió de Cadena

Escriu un programa que reverteixi una cadena donada.

section .data
    input db 'Assembly',0
    length equ $ - input
    output db length dup(0)

section .text
    global _start

_start:
    mov esi, input
    mov edi, output + length - 1

reverse_loop:
    lodsb
    stosb
    dec edi
    cmp esi, input + length
    jne reverse_loop

    ; Surt del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptes Clau:
    • Manipulació de cadenes.
    • Construccions de bucles.
    • Ús de les instruccions lodsb i stosb.

2. Seqüència de Fibonacci

Genera els primers N números de la seqüència de Fibonacci.

section .data
    N equ 10
    fib db N dup(0)

section .text
    global _start

_start:
    mov byte [fib], 0
    mov byte [fib+1], 1

    mov ecx, N-2
    mov esi, fib
    add esi, 2

fib_loop:
    mov al, [esi-1]
    add al, [esi-2]
    mov [esi], al
    inc esi
    loop fib_loop

    ; Surt del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptes Clau:
    • Bucles amb la instrucció loop.
    • Manipulació d'arrays.
    • Implementació d'algoritmes bàsics.

Projectes Avançats

1. Nucli de Sistema Operatiu Simple

Desenvolupa un nucli simple que pugui arrencar i mostrar un missatge.

section .text
    global start

start:
    mov ax, 0x07C0
    mov ds, ax
    mov es, ax

    mov si, msg
    call print_string

hang:
    jmp hang

print_string:
    mov ah, 0x0E
.next_char:
    lodsb
    cmp al, 0
    je .done
    int 0x10
    jmp .next_char
.done:
    ret

section .data
msg db 'Hola des del nucli!', 0
  • Conceptes Clau:
    • Conceptes bàsics de bootloader.
    • Interrupcions del BIOS.
    • Desenvolupament de nuclis.

2. Gestió de Memòria

Implementa un gestor de memòria simple que pugui assignar i alliberar blocs de memòria.

section .bss
    heap resb 1024

section .text
    global _start

_start:
    ; Inicialitza el punter del heap
    mov esi, heap

    ; Assigna memòria
    mov eax, 100
    call malloc

    ; Allibera memòria
    call free

    ; Surt del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80

malloc:
    ; Implementació de malloc
    ret

free:
    ; Implementació de free
    ret
  • Conceptes Clau:
    • Assignació dinàmica de memòria.
    • Aritmètica de punters.
    • Algoritmes de gestió de memòria.
