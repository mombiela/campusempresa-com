Introducció a les Macros

Les macros en llenguatge ensamblador són una característica poderosa que et permet definir una seqüència d'instruccions que es poden reutilitzar múltiples vegades al llarg del teu codi. Ajuden a reduir la duplicació de codi i a millorar la llegibilitat.

Conceptes Clau

  • Definició: Una macro es defineix utilitzant les directives MACRO i ENDM.
  • Invocació: Un cop definida, una macro es pot invocar simplement utilitzant el seu nom.
  • Paràmetres: Les macros poden acceptar paràmetres per fer-les més flexibles.

Exemple

; Defineix una macro per sumar dos números
ADD_NUMBERS MACRO num1, num2
    MOV AX, num1
    ADD AX, num2
ENDM

; Utilitza la macro
ADD_NUMBERS 5, 10

En aquest exemple:

  • La macro ADD_NUMBERS pren dos paràmetres, num1 i num2.
  • Quan s'invoca, mou num1 al registre AX i afegeix num2 a AX.

Característiques Avançades de les Macros

Les macros també poden incloure lògica condicional i bucles per fer-les encara més poderoses.

Lògica Condicional en Macros

Pots utilitzar directives d'assemblatge condicional dins de les macros per incloure o excloure codi basat en certes condicions.

Exemple

; Defineix una macro amb lògica condicional
CHECK_POSITIVE MACRO num
    IF num GT 0
        MOV AX, num
    ELSE
        MOV AX, 0
    ENDIF
ENDM

; Utilitza la macro
CHECK_POSITIVE 5
CHECK_POSITIVE -3

En aquest exemple:

  • La macro CHECK_POSITIVE comprova si el número és més gran que 0.
  • Si és cert, mou el número al registre AX; en cas contrari, mou 0 a AX.

Introducció a l'Ensamblatge Condicional

L'assemblatge condicional et permet incloure o excloure parts del teu codi basat en certes condicions. Això és útil per crear codi que es pugui compilar en diferents entorns o configuracions.

Conceptes Clau

  • Directives: Les directives d'assemblatge condicional comunes inclouen IF, ELSE, ENDIF, IFDEF i IFNDEF.
  • Ús: Aquestes directives s'utilitzen per controlar el procés d'assemblatge basat en condicions.

Exemple

; Defineix una constant
DEBUG EQU 1

; Utilitza l'assemblatge condicional
IF DEBUG
    MOV AX, 0x1234
    ; Codi de depuració aquí
ELSE
    MOV AX, 0x5678
    ; Codi de producció aquí
ENDIF

En aquest exemple:

  • La constant DEBUG es defineix amb un valor d'1.
  • La directiva IF comprova el valor de DEBUG i inclou el bloc de codi corresponent.

Combinació de Macros i Ensamblatge Condicional

Pots combinar macros i assemblatge condicional per crear codi altament flexible i reutilitzable.

Exemple

; Defineix una macro amb assemblatge condicional
DEBUG_PRINT MACRO msg
    IF DEBUG
        ; Simula la impressió d'un missatge de depuració
        MOV AX, msg
    ENDIF
ENDM

; Utilitza la macro
DEBUG_PRINT "Depuració activada"

En aquest exemple:

  • La macro DEBUG_PRINT utilitza l'assemblatge condicional per incloure el missatge de depuració només si DEBUG està definit.

Conclusió

Les macros i l'assemblatge condicional són eines essencials en la programació en llenguatge ensamblador. Ajuden a:

  • Reduir la duplicació de codi.
  • Millorar la llegibilitat i mantenibilitat del codi.
  • Crear codi flexible i configurable.

En dominar aquests conceptes, pots escriure programes en ensamblador més eficients i adaptables.
