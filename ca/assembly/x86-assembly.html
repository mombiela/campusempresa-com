Introducció a l'Assembly x86

  • Visió general del llenguatge Assembly
  • Importància d'aprendre Assembly
  • Conceptes bàsics: Registres, Memòria i Instruccions

Exemple de codi: Hello World en Assembly x86

section .data
    hello db 'Hello, World!', 0

section .text
    global _start

_start:
    ; Escriu la cadena a stdout
    mov eax, 4          ; número de syscall per a sys_write
    mov ebx, 1          ; descriptor de fitxer 1 és stdout
    mov ecx, hello      ; punter a la cadena
    mov edx, 13         ; longitud de la cadena
    int 0x80            ; crida al kernel

    ; Surt del programa
    mov eax, 1          ; número de syscall per a sys_exit
    xor ebx, ebx        ; codi de sortida 0
    int 0x80            ; crida al kernel
  • Explicació:
    • section .data: Defineix una secció de dades on es declaren variables.
    • section .text: Defineix una secció de codi on resideix el codi executable.
    • global _start: Punt d'entrada per al programa.
    • Instrucció mov: Mou dades d'un lloc a un altre.
    • int 0x80: Interrupció per cridar al kernel.

Instruccions bàsiques d'Assembly x86

  • Instruccions de moviment de dades: mov, push, pop
  • Instruccions aritmètiques: add, sub, mul, div
  • Instruccions lògiques: and, or, xor, not
  • Instruccions de control de flux: jmp, je, jne, call, ret

Exemple de codi: Operacions aritmètiques bàsiques

section .text
    global _start

_start:
    mov eax, 5          ; Carrega 5 a eax
    add eax, 3          ; Afegeix 3 a eax (eax = 8)
    sub eax, 2          ; Resta 2 de eax (eax = 6)
    ; Surt del programa
    mov eax, 1          ; número de syscall per a sys_exit
    xor ebx, ebx        ; codi de sortida 0
    int 0x80            ; crida al kernel
  • Explicació:
    • add eax, 3: Afegeix 3 al valor del registre eax.
    • sub eax, 2: Resta 2 del valor del registre eax.

Treballant amb Memòria

  • Comprensió dels modes d'adreçament: Immediat, Directe, Indirecte, Indexat
  • Ús de punters i arrays
  • Operacions amb la pila: push, pop, call, ret

Exemple de codi: Ús de punters

section .data
    num1 dd 10          ; Defineix una paraula doble amb valor 10
    num2 dd 20          ; Defineix una paraula doble amb valor 20

section .text
    global _start

_start:
    mov eax, [num1]     ; Carrega el valor de num1 a eax
    add eax, [num2]     ; Afegeix el valor de num2 a eax (eax = 30)
    ; Surt del programa
    mov eax, 1          ; número de syscall per a sys_exit
    xor ebx, ebx        ; codi de sortida 0
    int 0x80            ; crida al kernel
  • Explicació:
    • [num1]: Accedeix al valor emmagatzemat a la ubicació de memòria etiquetada com num1.
    • [num2]: Accedeix al valor emmagatzemat a la ubicació de memòria etiquetada com num2.

Control de Flux i Procediments

  • Salts condicionals: je, jne, jg, jl
  • Construccions de bucles: loop, loopz, loopnz
  • Definició i crida de procediments

Exemple de codi: Bucle simple

section .data
    counter dd 5        ; Defineix un comptador amb valor inicial 5

section .text
    global _start

_start:
    mov ecx, [counter]  ; Carrega el valor del comptador a ecx

loop_start:
    dec ecx             ; Decrementa ecx en 1
    cmp ecx, 0          ; Compara ecx amb 0
    jne loop_start      ; Salta a loop_start si ecx no és zero

    ; Surt del programa
    mov eax, 1          ; número de syscall per a sys_exit
    xor ebx, ebx        ; codi de sortida 0
    int 0x80            ; crida al kernel
  • Explicació:
    • dec ecx: Decrementa el valor del registre ecx en 1.
    • cmp ecx, 0: Compara el valor del registre ecx amb 0.
    • jne loop_start: Salta a loop_start si la comparació no és igual.

Temes Avançats

  • Macros i Assembly Condicional
  • Assembly en línia en llenguatges d'alt nivell
  • Optimització de codi Assembly
  • Interfície amb programes en C/C++

Exemple de codi: Assembly en línia en C

#include <stdio.h>

int main() {
    int a = 5, b = 3, result;

    __asm__ (
        "movl %1, %%eax;"
        "addl %2, %%eax;"
        "movl %%eax, %0;"
        : "=r" (result)
        : "r" (a), "r" (b)
        : "%eax"
    );

    printf("Resultat: %d\n", result);
    return 0;
}
  • Explicació:
    • __asm__: Bloc d'assembly en línia en C.
    • "movl %1, %%eax;": Mou el valor de a al registre eax.
    • "addl %2, %%eax;": Afegeix el valor de b al registre eax.
    • "movl %%eax, %0;": Mou el resultat del registre eax a result.
