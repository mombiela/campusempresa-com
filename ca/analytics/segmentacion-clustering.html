Introducció

La segmentació i el clustering són tècniques fonamentals en l'anàlisi de dades que permeten agrupar elements similars en categories. Aquestes tècniques són essencials per entendre patrons en les dades i prendre decisions informades en àrees com màrqueting, finances i ciències socials.

Conceptes Clau

  • Segmentació: Procés de dividir un conjunt de dades en subgrups més petits i homogènics.
  • Clustering: Tècnica d'aprenentatge no supervisat que agrupa dades en clusters o grups basats en similituds.
  • Distància Euclidiana: Mètrica comunament utilitzada per mesurar la similitud entre punts en un espai multidimensional.
  • K-means: Algoritme de clustering que particiona les dades en K clusters.
  • Jeràrquic: Algoritme de clustering que construeix una jerarquia de clusters.

Algoritmes de Clustering

K-means

L'algoritme K-means és un dels mètodes més populars per al clustering. Funciona de la següent manera:

  1. Seleccionar K punts aleatoris com a centroides inicials.
  2. Assignar cada punt de dades al centroide més proper.
  3. Recalcular els centroides com la mitjana dels punts assignats a cada cluster.
  4. Repetir els passos 2 i 3 fins que els centroides no canviïn significativament.

Exemple de Codi en Python

import numpy as np
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt

# Generar dades d'exemple
X = np.array([[1, 2], [1, 4], [1, 0], 
              [10, 2], [10, 4], [10, 0]])

# Aplicar K-means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Obtenir etiquetes i centroides
labels = kmeans.labels_
centroids = kmeans.cluster_centers_

# Visualitzar els clusters
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.scatter(centroids[:, 0], centroids[:, 1], s=300, c='red')
plt.show()

Clustering Jeràrquic

El clustering jeràrquic pot ser aglomeratiu o divisiu. L'aglomeratiu comença amb cada punt com un cluster individual i fusiona els clusters més propers fins que tots els punts estan en un sol cluster.

Exemple de Codi en Python

import numpy as np
import matplotlib.pyplot as plt
from scipy.cluster.hierarchy import dendrogram, linkage

# Generar dades d'exemple
X = np.array([[1, 2], [1, 4], [1, 0], 
              [10, 2], [10, 4], [10, 0]])

# Aplicar clustering jeràrquic
Z = linkage(X, 'ward')

# Visualitzar el dendrograma
plt.figure()
dendrogram(Z)
plt.show()

Comparació d'Algoritmes

| Característica | K-means | Jeràrquic | |----------------------|----------------------------|--------------------------| | Tipus d'algoritme | Particional | Jeràrquic | | Complexitat | O(n * k * i) | O(n^3) | | Nombre de clusters | Ha de ser especificat | Determinat pel tall | | Escalabilitat | Alta | Baixa |

Aplicacions Pràctiques

  • Màrqueting: Segmentació de clients per a campanyes personalitzades.
  • Finances: Identificació de patrons de frau.
  • Ciències Socials: Agrupació d'enquestes i estudis d'opinió.

Conclusió

La segmentació i el clustering són tècniques poderoses en l'anàlisi de dades que permeten descobrir patrons ocults i prendre decisions basades en dades. Comprendre i aplicar algoritmes com K-means i clustering jeràrquic és essencial per a qualsevol professional en el camp de l'analítica.

Aquests mètodes no només ajuden a simplificar dades complexes, sinó que també proporcionen una base sòlida per a l'anàlisi predictiu i la presa de decisions estratègiques.
