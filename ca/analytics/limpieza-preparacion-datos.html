Introducció

La neteja i preparació de dades és una etapa crítica en qualsevol projecte d'anàlisi de dades. Aquest procés garanteix que les dades siguin precises, completes i estiguin en un format adequat per a l'anàlisi. En aquesta secció, aprendrem les tècniques i eines essencials per netejar i preparar dades de manera efectiva.

Importància de la Neteja de Dades

  • Qualitat de les Dades: Dades netes i ben preparades són fonamentals per obtenir resultats precisos i fiables.
  • Eficiència: La neteja de dades redueix el temps i esforç necessaris per a l'anàlisi posterior.
  • Prevenció d'Errors: Minimitza la possibilitat d'errors i biaixos en els resultats.

Passos en la Neteja de Dades

1. Inspecció de Dades

Abans de començar amb la neteja, és crucial inspeccionar les dades per identificar problemes comuns.

  • Valors Faltants: Dades absents que poden afectar l'anàlisi.
  • Valors Atípics: Dades que es desvien significativament de la resta del conjunt.
  • Duplicats: Registres repetits que poden distorsionar els resultats.

2. Tractament de Valors Faltants

Hi ha diverses estratègies per manejar els valors faltants:

  • Eliminació: Treure files o columnes amb valors faltants.
  • Imputació: Omplir els valors faltants amb estimacions, com la mitjana, mediana o moda.
  • Models Predictius: Utilitzar algoritmes de machine learning per predir els valors faltants.

Exemple en Python:

import pandas as pd

# Crear un DataFrame d'exemple
data = {'A': [1, 2, None, 4], 'B': [None, 2, 3, 4]}
df = pd.DataFrame(data)

# Imputar valors faltants amb la mitjana de la columna
df.fillna(df.mean(), inplace=True)
print(df)

3. Detecció i Tractament de Valors Atípics

Els valors atípics poden ser identificats i tractats de diverses maneres:

  • Mètodes Estadístics: Utilitzar desviació estàndard o percentils.
  • Visualització: Gràfics com boxplots per identificar visualment els atípics.
  • Transformacions: Aplicar transformacions logarítmiques o d'arrel quadrada.

Exemple en Python:

import numpy as np

# Crear un DataFrame d'exemple
data = {'A': [1, 2, 3, 100, 5], 'B': [2, 3, 4, 5, 6]}
df = pd.DataFrame(data)

# Identificar valors atípics usant el mètode del rang interquartílic (IQR)
Q1 = df.quantile(0.25)
Q3 = df.quantile(0.75)
IQR = Q3 - Q1

# Filtrar valors atípics
df_outliers = df[~((df < (Q1 - 1.5 * IQR)) | (df > (Q3 + 1.5 * IQR))).any(axis=1)]
print(df_outliers)

4. Eliminació de Duplicats

Els duplicats poden ser eliminats fàcilment utilitzant funcions específiques en biblioteques d'anàlisi de dades.

Exemple en Python:

# Crear un DataFrame d'exemple amb duplicats
data = {'A': [1, 2, 2, 4], 'B': [2, 3, 3, 4]}
df = pd.DataFrame(data)

# Eliminar duplicats
df.drop_duplicates(inplace=True)
print(df)

5. Transformació de Dades

La transformació de dades inclou la normalització, estandardització i conversió de tipus de dades.

  • Normalització: Escalar les dades perquè estiguin en un rang específic.
  • Estandardització: Ajustar les dades perquè tinguin una mitjana de 0 i una desviació estàndard d'1.
  • Conversió de Tipus: Assegurar-se que les dades estiguin en el tipus correcte (per exemple, convertir cadenes a dates).

Exemple en Python:

from sklearn.preprocessing import StandardScaler

# Crear un DataFrame d'exemple
data = {'A': [1, 2, 3, 4], 'B': [2, 3, 4, 5]}
df = pd.DataFrame(data)

# Estandarditzar les dades
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df)
print(df_scaled)

Conclusió

La neteja i preparació de dades és un pas essencial en l'anàlisi de dades. A través de la inspecció, tractament de valors faltants i atípics, eliminació de duplicats i transformació de dades, podem assegurar que les nostres dades estiguin en la millor forma possible per a l'anàlisi. Practicar aquestes tècniques amb exemples i exercicis ajudarà a consolidar aquests conceptes i millorar la qualitat de les anàlisis futures.
