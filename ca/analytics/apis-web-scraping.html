Introducció

En aquesta secció del curs sobre Analytics, explorarem dues eines fonamentals per a l'obtenció de dades: les APIs i el Web Scraping. Ambdues tècniques són essencials per als analistes de dades que necessiten accedir a informació en línia de manera eficient i estructurada.

APIs (Application Programming Interfaces)

Les APIs permeten a les aplicacions comunicar-se entre si. Són interfícies que permeten l'accés a les funcionalitats d'un programari des d'un altre programari.

Conceptes Clau

  • Endpoint: Punt d'accés a una API on es pot realitzar una sol·licitud.
  • Mètodes HTTP: Accions que es poden realitzar en els recursos d'una API (GET, POST, PUT, DELETE).
  • Autenticació: Procés de verificació d'identitat per accedir a una API (API keys, OAuth).
  • Format de Dades: Estructura en la qual s'envien les dades (JSON, XML).

Exemple de Codi

A continuació, es mostra un exemple de com realitzar una sol·licitud GET a una API utilitzant Python i la biblioteca requests.

import requests

# URL del endpoint de la API
url = "https://api.example.com/data"

# Paràmetres de la sol·licitud
params = {
    "param1": "value1",
    "param2": "value2"
}

# Realitzar la sol·licitud GET
response = requests.get(url, params=params)

# Verificar l'estat de la resposta
if response.status_code == 200:
    data = response.json()
    print(data)
else:
    print(f"Error: {response.status_code}")

Exercici

  1. Registra't en una API pública (per exemple, OpenWeatherMap).
  2. Obté una API key.
  3. Realitza una sol·licitud GET per obtenir el clima actual d'una ciutat específica.
  4. Imprimeix les dades obtingudes en format JSON.

Web Scraping

El Web Scraping és la tècnica d'extreure informació de llocs web de manera automatitzada. És útil quan les dades no estan disponibles a través d'una API.

Conceptes Clau

  • HTML: Llenguatge de marcatge utilitzat per estructurar pàgines web.
  • DOM (Document Object Model): Model d'objectes que representa l'estructura d'un document HTML.
  • Selectors CSS: Eines per seleccionar elements específics en un document HTML.
  • Biblioteques de Scraping: Eines com BeautifulSoup i Scrapy que faciliten el procés de scraping.

Exemple de Codi

A continuació, es mostra un exemple de com realitzar web scraping utilitzant Python i la biblioteca BeautifulSoup.

import requests
from bs4 import BeautifulSoup

# URL de la pàgina web
url = "https://www.example.com"

# Realitzar la sol·licitud GET
response = requests.get(url)

# Verificar l'estat de la resposta
if response.status_code == 200:
    # Parsejar el contingut HTML
    soup = BeautifulSoup(response.content, 'html.parser')
    
    # Trobar i extreure dades específiques
    title = soup.find('title').get_text()
    print(f"Title: {title}")
else:
    print(f"Error: {response.status_code}")

Exercici

  1. Tria un lloc web que t'interessi.
  2. Identifica els elements HTML que contenen les dades que desitges extreure.
  3. Utilitza BeautifulSoup per extreure i mostrar aquestes dades.

Comparació entre APIs i Web Scraping

| Característica | APIs | Web Scraping | |----------------------|-------------------------------------------|------------------------------------------| | Facilitat d'Ús | Generalment més fàcil i estructurat | Pot ser més complex i propens a errors | | Estructura de Dades | Dades estructurades (JSON, XML) | Dades no estructurades (HTML) | | Taxa d'Actualització | Alta (dades en temps real) | Depèn de la freqüència de scraping | | Legalitat | Generalment permès amb restriccions | Pot ser legalment ambigu |

Conclusió

Tant les APIs com el Web Scraping són eines poderoses per obtenir dades en línia. Les APIs són preferibles quan estan disponibles a causa de la seva facilitat d'ús i estructura de dades clara. No obstant això, el Web Scraping és una alternativa valuosa quan les dades no estan accessibles a través d'una API. És important comprendre els avantatges i limitacions de cada tècnica per triar la més adequada segons les necessitats del projecte.
