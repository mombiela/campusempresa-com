Introducció

L'anàlisi de tendències i patrons és una part fonamental de l'anàlisi de dades. Permet als analistes identificar comportaments recurrents i predir futurs esdeveniments basant-se en dades històriques. Aquest tema és crucial per prendre decisions informades i estratègiques en diversos camps com màrqueting, finances i operacions.

Conceptes Clau

  • Tendències: Direccions generals en les quals alguna cosa està canviant o desenvolupant-se.
  • Patrons: Seqüències o estructures recurrents en les dades.
  • Sèries Temporals: Dades recollides en intervals de temps regulars.
  • Estacionalitat: Fluctuacions periòdiques i previsibles en les dades.
  • Sorra: Variabilitat aleatòria en les dades que no segueix un patró discernible.

Mètodes d'Anàlisi

Anàlisi de Tendències

L'anàlisi de tendències implica identificar la direcció general en què es mouen les dades al llarg del temps. Algunes tècniques comunes inclouen:

  • Mitjana Mòbil: Suavitza les dades per identificar la tendència subjacent.
  • Regressió Lineal: Ajusta una línia recta a les dades per modelar la relació entre variables.

Exemple de Mitjana Mòbil

import pandas as pd
import matplotlib.pyplot as plt

# Dades d'exemple
data = {'Data': pd.date_range(start='1/1/2020', periods=10, freq='M'),
        'Vendes': [200, 220, 250, 270, 300, 320, 350, 370, 400, 420]}
df = pd.DataFrame(data)

# Calcular la mitjana mòbil
df['Mitjana_Mòbil'] = df['Vendes'].rolling(window=3).mean()

# Gràfic
plt.plot(df['Data'], df['Vendes'], label='Vendes')
plt.plot(df['Data'], df['Mitjana_Mòbil'], label='Mitjana Mòbil', linestyle='--')
plt.legend()
plt.show()

Anàlisi de Patrons

L'anàlisi de patrons se centra en identificar estructures recurrents en les dades. Algunes tècniques inclouen:

  • Descomposició de Sèries Temporals: Separa una sèrie temporal en tendència, estacionalitat i sorra.
  • Anàlisi de Fourier: Transforma les dades per identificar freqüències dominants.

Exemple de Descomposició de Sèries Temporals

import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose

# Dades d'exemple
data = {'Data': pd.date_range(start='1/1/2020', periods=24, freq='M'),
        'Vendes': [200, 220, 250, 270, 300, 320, 350, 370, 400, 420, 450, 470,
                   500, 520, 550, 570, 600, 620, 650, 670, 700, 720, 750, 770]}
df = pd.DataFrame(data)
df.set_index('Data', inplace=True)

# Descomposició
result = seasonal_decompose(df['Vendes'], model='additive')
result.plot()
plt.show()

Comparació de Mètodes

| Mètode | Avantatges | Desavantatges | |--------------------------|---------------------------------------------|----------------------------------------------| | Mitjana Mòbil | Fàcil d'implementar i entendre | Pot retardar-se en detectar canvis ràpids | | Regressió Lineal | Bona per a tendències lineals | No captura bé les no lineals | | Descomposició de Sèries | Identifica components individuals | Requereix dades estacionals | | Anàlisi de Fourier | Detecta freqüències dominants | Complex d'interpretar |

Exercicis Pràctics

  1. Exercici de Mitjana Mòbil: Utilitza un conjunt de dades de vendes mensuals i calcula la mitjana mòbil amb diferents finestres (3, 6, 12 mesos). Gràfica els resultats i analitza les diferències.
  2. Exercici de Descomposició: Pren una sèrie temporal amb estacionalitat clara (per exemple, temperatures mensuals) i realitza una descomposició. Identifica i discuteix les components de tendència, estacionalitat i sorra.

Conclusió

L'anàlisi de tendències i patrons és essencial per comprendre i predir comportaments en les dades. Utilitzant tècniques com la mitjana mòbil, la regressió lineal i la descomposició de sèries temporals, els analistes poden extreure informació valuosa i prendre decisions informades. Practicar amb diferents mètodes i conjunts de dades ajudarà a desenvolupar una comprensió més profunda i habilitats analítiques robustes.
