Introducció a la Regressió Múltiple

La regressió múltiple és una tècnica estadística utilitzada per predir el valor d'una variable dependent basada en el valor de dues o més variables independents. És una extensió de la regressió lineal simple, que només utilitza una variable independent.

  • Variable dependent (Y): La variable que s'intenta predir o explicar.
  • Variables independents (X1, X2, ..., Xn): Les variables que s'utilitzen per predir la variable dependent.

Conceptes Clau

  • Coeficients de regressió: Representen el canvi en la variable dependent per una unitat de canvi en la variable independent, mantenint constants les altres variables.
  • Intercepció (Intercept): El valor de Y quan totes les variables independents són zero.
  • R-quadrat (R²): Indica la proporció de la variabilitat en la variable dependent que pot ser explicada per les variables independents.
  • P-valor: Indica la significança estadística de cada coeficient de regressió.

Fórmula de la Regressió Múltiple

L'equació general de la regressió múltiple és:

\[ Y = \beta_0 + \beta_1X_1 + \beta_2X_2 + ... + \beta_nX_n + \epsilon \]

  • \( Y \): Variable dependent.
  • \( \beta_0 \): Intercepció.
  • \( \beta_1, \beta_2, ..., \beta_n \): Coeficients de regressió.
  • \( X_1, X_2, ..., X_n \): Variables independents.
  • \( \epsilon \): Terme d'error.

Exemple Pràctic

Suposem que volem predir el preu d'una casa (Y) basat en la seva mida en peus quadrats (X1) i el nombre d'habitacions (X2).

Pas 1: Importar les Biblioteques Necessàries

import pandas as pd
import numpy as np
import statsmodels.api as sm

Pas 2: Carregar i Preparar les Dades

# Crear un DataFrame d'exemple
data = {
    'Mida': [1500, 1600, 1700, 1800, 1900],
    'Habitacions': [3, 3, 4, 4, 5],
    'Preu': [300000, 320000, 340000, 360000, 380000]
}

df = pd.DataFrame(data)

Pas 3: Definir les Variables Independents i Dependents

X = df[['Mida', 'Habitacions']]
Y = df['Preu']

Pas 4: Afegir una Constant a les Variables Independents

X = sm.add_constant(X)

Pas 5: Ajustar el Model de Regressió Múltiple

model = sm.OLS(Y, X).fit()

Pas 6: Resumir els Resultats del Model

print(model.summary())

Interpretació dels Resultats

  • Coeficients: Indiquen quant canvia el preu de la casa per cada unitat addicional de mida o habitació.
  • P-valors: Ajuden a determinar si els coeficients són estadísticament significatius.
  • R-quadrat: Mostra com de bé les variables independents expliquen la variabilitat del preu de la casa.

Conclusió

La regressió múltiple és una eina poderosa per entendre i predir el comportament d'una variable dependent basada en múltiples variables independents. En comprendre els coeficients, els p-valors i el R-quadrat, els professionals poden prendre decisions informades basades en les dades.

  • Pràctica recomanada: Aplicar la regressió múltiple a un conjunt de dades real per reforçar la comprensió.
  • Recursos addicionals: Revisar documentació i tutorials de biblioteques estadístiques en Python com statsmodels i scikit-learn.

Aquest tema proporciona una base sòlida per entendre la regressió múltiple i la seva aplicació pràctica en l'anàlisi de dades.
