Introducció

En estadística, les variables aleatòries i les distribucions de probabilitat són conceptes fonamentals que permeten modelar i analitzar fenòmens incerts. Aquest tema cobreix els conceptes bàsics, tipus de variables aleatòries, i les distribucions de probabilitat més comunes.

Variables Aleatòries

Una variable aleatòria és una funció que assigna un valor numèric a cada resultat possible d'un experiment aleatori.

Tipus de Variables Aleatòries

  • Variable Aleatòria Discreta: Pren un nombre finit o comptable de valors. Exemple: Nombre de cares en llançar una moneda tres vegades.
  • Variable Aleatòria Contínua: Pren un nombre infinit de valors dins d'un interval. Exemple: Temps d'espera en una cua.

Exemple de Variable Aleatòria Discreta

Suposem que llancem un dau just. La variable aleatòria \(X\) representa el resultat del llançament del dau.

  • Possibles valors de \(X\): {1, 2, 3, 4, 5, 6}

Exemple de Variable Aleatòria Contínua

Considerem el temps \(T\) que un client espera en una cua. Aquesta variable pot prendre qualsevol valor positiu.

  • Possibles valors de \(T\): {0, 0.1, 0.2, ..., ∞}

Distribucions de Probabilitat

Una distribució de probabilitat descriu com es distribueixen els valors d'una variable aleatòria.

Distribució de Probabilitat per a Variables Discretes

  • Funció de Probabilitat (PMF): Assigna una probabilitat a cada valor possible de la variable aleatòria discreta.

    Exemple: Llançament d'un dau just. \[ P(X = x) = \frac{1}{6} \quad \text{per a } x \in {1, 2, 3, 4, 5, 6} \]

Distribució de Probabilitat per a Variables Contínues

  • Funció de Densitat de Probabilitat (PDF): Descriu la probabilitat que una variable aleatòria contínua prengui un valor dins d'un interval.

    Exemple: Distribució normal. \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \] on \(\mu\) és la mitjana i \(\sigma\) és la desviació estàndard.

Distribucions de Probabilitat Comunes

Distribució Binomial

  • Descripció: Modela el nombre d'èxits en una sèrie d'assajos independents amb probabilitat constant d'èxit.
  • Paràmetres: \(n\) (nombre d'assajos), \(p\) (probabilitat d'èxit).
  • PMF: \[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \] on \(\binom{n}{k}\) és el coeficient binomial.

Distribució Normal

  • Descripció: Modela fenòmens naturals i és simètrica al voltant de la mitjana.
  • Paràmetres: \(\mu\) (mitjana), \(\sigma\) (desviació estàndard).
  • PDF: \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

Distribució Poisson

  • Descripció: Modela el nombre d'esdeveniments que ocorren en un interval de temps o espai.
  • Paràmetres: \(\lambda\) (taxa d'esdeveniments).
  • PMF: \[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \]

Exemples de Codi

Exemple en Python: Distribució Binomial

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom

# Paràmetres
n = 10  # nombre d'assajos
p = 0.5  # probabilitat d'èxit

# Valors possibles de X
x = np.arange(0, n+1)

# PMF
pmf = binom.pmf(x, n, p)

# Gràfica
plt.bar(x, pmf)
plt.xlabel('Nombre d\'èxits')
plt.ylabel('Probabilitat')
plt.title('Distribució Binomial (n=10, p=0.5)')
plt.show()

Exemple en Python: Distribució Normal

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# Paràmetres
mu = 0  # mitjana
sigma = 1  # desviació estàndard

# Valors possibles de X
x = np.linspace(mu - 3*sigma, mu + 3*sigma, 100)

# PDF
pdf = norm.pdf(x, mu, sigma)

# Gràfica
plt.plot(x, pdf)
plt.xlabel('Valor')
plt.ylabel('Densitat de probabilitat')
plt.title('Distribució Normal (μ=0, σ=1)')
plt.show()

Conclusió

Les variables aleatòries i les distribucions de probabilitat són eines essencials en estadística per modelar i analitzar dades. Comprendre aquests conceptes permet als professionals prendre decisions informades basades en dades i manejar la incertesa de manera efectiva. En aquest tema, hem cobert els tipus de variables aleatòries, les distribucions de probabilitat més comunes i proporcionat exemples pràctics per il·lustrar aquests conceptes.
