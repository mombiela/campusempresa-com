Introducció a l'ANOVA

L'Anàlisi de Variància (ANOVA) és una tècnica estadística utilitzada per comparar les mitjanes de tres o més grups independents per determinar si almenys una de les mitjanes és significativament diferent de les altres. És una extensió del test t de Student per a més de dos grups.

Objectius de l'ANOVA

  • Comparar les mitjanes de tres o més grups.
  • Determinar si existeixen diferències significatives entre les mitjanes dels grups.
  • Avaluar la variabilitat dins dels grups i entre els grups.

Conceptes Clau

  • Hipòtesi Nul·la (H0): Totes les mitjanes dels grups són iguals.
  • Hipòtesi Alternativa (H1): Almenys una de les mitjanes dels grups és diferent.
  • Variabilitat Entre Grups: Variabilitat deguda a les diferències entre les mitjanes dels grups.
  • Variabilitat Dins dels Grups: Variabilitat deguda a les diferències dins de cada grup.
  • F-Estadístic: Quocient de la variabilitat entre grups i la variabilitat dins dels grups.

Tipus d'ANOVA

  • ANOVA d'un factor: Compara les mitjanes de tres o més grups basats en un sol factor.
  • ANOVA de dos factors: Compara les mitjanes de grups basats en dos factors.
  • ANOVA de mesures repetides: Compara les mitjanes de grups on les mateixes unitats experimentals són mesurades en diferents condicions.

Procediment de l'ANOVA d'un Factor

  1. Formulació d'Hipòtesis:

    • H0: μ1 = μ2 = μ3 = ... = μk
    • H1: Almenys una mitjana és diferent.

  2. Càlcul de la Variabilitat:

    • Suma de Quadrats Total (SST): Variabilitat total en les dades.
    • Suma de Quadrats Entre Grups (SSB): Variabilitat deguda a les diferències entre les mitjanes dels grups.
    • Suma de Quadrats Dins dels Grups (SSW): Variabilitat deguda a les diferències dins dels grups.

  3. Càlcul del F-Estadístic:

    • \( F = \frac{MSB}{MSW} \)
    • On \( MSB = \frac{SSB}{k-1} \) i \( MSW = \frac{SSW}{N-k} \).

  4. Comparació del F-Estadístic:

    • Comparar el valor calculat de F amb el valor crític de F de la taula de distribució F per determinar si es rebutja H0.

Exemple Pràctic

Suposem que volem comparar el rendiment de tres mètodes d'ensenyament diferents en estudiants. Les dades són les següents:

| Mètode 1 | Mètode 2 | Mètode 3 | |----------|----------|----------| | 85 | 78 | 92 | | 90 | 82 | 88 | | 78 | 85 | 94 | | 92 | 80 | 90 |

Codi en Python per ANOVA

import scipy.stats as stats

# Dades
metode1 = [85, 90, 78, 92]
metode2 = [78, 82, 85, 80]
metode3 = [92, 88, 94, 90]

# ANOVA d'un factor
f_statistic, p_value = stats.f_oneway(metode1, metode2, metode3)

print(f'Estadístic F: {f_statistic}')
print(f'Valor p: {p_value}')

Interpretació de Resultats

  • Estadístic F: Indica la relació entre la variabilitat entre els grups i la variabilitat dins dels grups.
  • Valor p: Si el valor p és menor que el nivell de significança (per exemple, 0.05), es rebutja la hipòtesi nul·la, indicant que almenys una de les mitjanes és significativament diferent.

Conclusió

L'ANOVA és una eina poderosa per comparar les mitjanes de múltiples grups i determinar si existeixen diferències significatives entre ells. A través del càlcul del F-Estadístic i la interpretació del valor p, podem prendre decisions informades sobre les nostres hipòtesis. És fonamental entendre els conceptes clau i el procediment per aplicar correctament l'ANOVA en anàlisis estadístiques.
