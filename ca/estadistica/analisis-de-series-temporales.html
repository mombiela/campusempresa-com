Introducció

L'anàlisi de sèries temporals és una tècnica estadística que s'utilitza per analitzar dades que es recullen al llarg del temps. Aquest tipus d'anàlisi és crucial en diverses disciplines com l'economia, la meteorologia, l'enginyeria i la biologia, entre altres. En aquest tema, explorarem els conceptes bàsics, mètodes i aplicacions de l'anàlisi de sèries temporals.

Conceptes Clau

  • Sèrie Temporal: Una seqüència de dades punts, típicament consistents en mesuraments successius fets a intervals de temps uniformes.
  • Estacionarietat: Una propietat d'una sèrie temporal que indica que les seves propietats estadístiques, com la mitjana i la variància, no canvien amb el temps.
  • Autocorrelació: La correlació d'una sèrie temporal amb una versió retardada d'ella mateixa.
  • Descomposició de Sèries Temporals: Separar una sèrie temporal en components com tendència, estacionalitat i soroll.

Mètodes d'Anàlisi

Descomposició de Sèries Temporals

La descomposició de sèries temporals implica dividir la sèrie en tres components principals:

  1. Tendència (T): Representa la direcció general de les dades al llarg del temps.
  2. Estacionalitat (S): Captura els patrons que es repeteixen a intervals regulars.
  3. Soroll (R): La variabilitat aleatòria en les dades.

Exemple en Python

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.seasonal import seasonal_decompose

# Crear una sèrie temporal d'exemple
np.random.seed(0)
date_rng = pd.date_range(start='1/1/2020', end='1/1/2021', freq='D')
data = np.random.randn(len(date_rng)) + np.linspace(0, 10, len(date_rng))
ts = pd.Series(data, index=date_rng)

# Descomposició de la sèrie temporal
decomposition = seasonal_decompose(ts, model='additive')
trend = decomposition.trend
seasonal = decomposition.seasonal
residual = decomposition.resid

# Gràficar els components
plt.figure(figsize=(12, 8))
plt.subplot(411)
plt.plot(ts, label='Original')
plt.legend(loc='best')
plt.subplot(412)
plt.plot(trend, label='Tendència')
plt.legend(loc='best')
plt.subplot(413)
plt.plot(seasonal, label='Estacionalitat')
plt.legend(loc='best')
plt.subplot(414)
plt.plot(residual, label='Soroll')
plt.legend(loc='best')
plt.tight_layout()
plt.show()

Models ARIMA

El model ARIMA (AutoRegressive Integrated Moving Average) és un dels mètodes més utilitzats per a l'anàlisi de sèries temporals. Es compon de tres parts:

  1. AR (AutoRegressive): Utilitza la dependència entre una observació i un nombre de retards (valors anteriors).
  2. I (Integrated): Utilitza la diferència de les observacions per fer la sèrie estacionària.
  3. MA (Moving Average): Utilitza la dependència entre una observació i un terme d'error retardat.

Exemple en Python

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.arima.model import ARIMA

# Crear una sèrie temporal d'exemple
np.random.seed(0)
date_rng = pd.date_range(start='1/1/2020', end='1/1/2021', freq='D')
data = np.random.randn(len(date_rng)) + np.linspace(0, 10, len(date_rng))
ts = pd.Series(data, index=date_rng)

# Ajustar el model ARIMA
model = ARIMA(ts, order=(5, 1, 0))
model_fit = model.fit()

# Resum del model
print(model_fit.summary())

# Prediccions
forecast = model_fit.forecast(steps=10)
plt.figure(figsize=(12, 6))
plt.plot(ts, label='Original')
plt.plot(forecast, label='Predicció', color='red')
plt.legend(loc='best')
plt.show()

Aplicacions

  • Economia: Predicció de preus d'accions, taxes d'interès, i altres indicadors econòmics.
  • Meteorologia: Predicció del clima i anàlisi de patrons climàtics.
  • Enginyeria: Monitoratge i predicció de fallades en sistemes i maquinària.
  • Biologia: Anàlisi de dades de creixement poblacional i altres fenòmens biològics.

Conclusió

L'anàlisi de sèries temporals és una eina poderosa per entendre i predir comportaments en dades que varien amb el temps. Des de la descomposició de sèries fins a l'aplicació de models ARIMA, aquestes tècniques permeten als professionals prendre decisions informades basades en patrons i tendències observades en les dades històriques.
