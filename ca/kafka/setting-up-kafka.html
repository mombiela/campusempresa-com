Configuració de Kafka
Temes i particions de Kafka ►

Configurar Kafka implica diversos passos, des de descarregar i instal·lar Kafka fins a configurar-lo per a les teves necessitats específiques. Aquesta secció et guiarà a través del procés, assegurant que tinguis un entorn Kafka funcional al final.

Prerequisits

Abans de configurar Kafka, assegura't de tenir el següent:

  • Java Development Kit (JDK): Kafka requereix Java per funcionar. Assegura't de tenir JDK 8 o posterior instal·lat.
  • Zookeeper: Kafka utilitza Zookeeper per gestionar el seu clúster. Zookeeper ha d'estar instal·lat i en funcionament abans d'iniciar Kafka.

Descarregar Kafka

  1. Visita la Pàgina de Descàrrega de Kafka: Ves a la pàgina de descàrrega d'Apache Kafka.
  2. Selecciona una Versió: Tria la versió de Kafka que vols descarregar. Es recomana utilitzar l'última versió estable.
  3. Descarrega el Binari: Descarrega els fitxers binaris per al teu sistema operatiu.

Instal·lar Kafka

  1. Extreu l'Arxiu: Un cop descarregat, extreu l'arxiu de Kafka al directori desitjat.

    tar -xzf kafka_2.13-2.8.0.tgz
cd kafka_2.13-2.8.0

  2. Estructura del Directori: Després de l'extracció, hauries de veure la següent estructura de directoris:

    kafka_2.13-2.8.0/
├── bin
├── config
├── libs
├── logs
└── site-docs

Configurar Zookeeper

Kafka requereix Zookeeper per gestionar el seu clúster. Zookeeper està inclòs en la distribució de Kafka.

  1. Inicia Zookeeper: Utilitza el script proporcionat per iniciar Zookeeper.

    bin/zookeeper-server-start.sh config/zookeeper.properties

  2. Verifica Zookeeper: Assegura't que Zookeeper està en funcionament comprovant els logs o utilitzant un client de Zookeeper.

Configurar Kafka

  1. Edita la Configuració de Kafka: Obre el fitxer config/server.properties per configurar Kafka.

    • Broker ID: Identificador únic per a cada broker de Kafka. 
      broker.id=0
    • Directori de Logs: Directori on Kafka emmagatzemarà els seus logs. 
      log.dirs=/tmp/kafka-logs
    • Connexió a Zookeeper: Adreça de la instància de Zookeeper. 
      zookeeper.connect=localhost:2181

  2. Inicia Kafka: Utilitza el script proporcionat per iniciar Kafka.

    bin/kafka-server-start.sh config/server.properties

Verificar la Instal·lació

  1. Crea un Tema: Utilitza el script de Kafka per crear un nou tema.

    bin/kafka-topics.sh --create --topic test --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1

  2. Llista els Temes: Verifica que el tema s'ha creat.

    bin/kafka-topics.sh --list --bootstrap-server localhost:9092

  3. Envia Missatges: Utilitza el productor de consola de Kafka per enviar missatges al tema.

    bin/kafka-console-producer.sh --topic test --bootstrap-server localhost:9092
>Hola Kafka
>Aquest és un missatge de prova

  4. Consumeix Missatges: Utilitza el consumidor de consola de Kafka per llegir missatges del tema.

    bin/kafka-console-consumer.sh --topic test --from-beginning --bootstrap-server localhost:9092

Resum

Configurar Kafka implica descarregar els binaris de Kafka, configurar Zookeeper i iniciar tant els servidors de Zookeeper com de Kafka. Seguint els passos descrits anteriorment, hauries de tenir un entorn Kafka funcional. Aquesta configuració et permet crear temes, produir missatges i consumir missatges, formant la base per a operacions més avançades amb Kafka.
