Introducció

Apache Kafka és una eina poderosa per construir canals de dades en temps real i aplicacions de streaming. Una de les característiques clau que fan Kafka tan versàtil és la seva capacitat per transformar i convertir dades a mesura que flueixen pel sistema. Aquesta secció cobrirà els conceptes bàsics de les transformacions i els convertidors en Kafka, des de nivells bàsics fins a avançats.

Conceptes Clau

  • Transformacions: Operacions que modifiquen les dades a mesura que es mouen per Kafka.
  • Convertidors: Eines que canvien el format de les dades per fer-les compatibles amb diferents sistemes.

Transformacions

Les transformacions en Kafka són típicament gestionades per Kafka Streams o Kafka Connect. Aquestes transformacions poden anar des de modificacions simples de dades fins a agregacions i unions complexes.

Transformacions de Kafka Streams

Kafka Streams és una biblioteca client per construir aplicacions i microserveis, on les dades d'entrada i sortida s'emmagatzemen en clústers de Kafka. Permet el processament i la transformació de dades en temps real.

Transformacions Bàsiques

  • mapValues: Transforma els valors dels registres.
  • filter: Filtra registres basats en una condició.
  • flatMap: Transforma cada registre en zero o més registres.
KStream<String, String> source = builder.stream("input-topic");

// Exemple de mapValues
KStream<String, String> uppercased = source.mapValues(value -> value.toUpperCase());

// Exemple de filter
KStream<String, String> filtered = source.filter((key, value) -> value.length() > 5);

// Exemple de flatMap
KStream<String, String> expanded = source.flatMap((key, value) -> {
    List<KeyValue<String, String>> result = new ArrayList<>();
    for (String word : value.split(" ")) {
        result.add(new KeyValue<>(key, word));
    }
    return result;
});

Transformacions Avançades

  • groupByKey: Agrupa registres per clau.
  • aggregate: Agrega registres per clau.
  • join: Uneix dos fluxos basats en una clau.
KStream<String, String> source1 = builder.stream("input-topic1");
KStream<String, String> source2 = builder.stream("input-topic2");

// Exemple de groupByKey i aggregate
KGroupedStream<String, String> grouped = source.groupByKey();
KTable<String, Long> counts = grouped.count();

// Exemple de join
KStream<String, String> joined = source1.join(source2, 
    (value1, value2) -> value1 + "-" + value2,
    JoinWindows.of(Duration.ofMinutes(5))
);

Convertidors

Els convertidors s'utilitzen per canviar el format de les dades. Això és particularment útil quan s'integra Kafka amb altres sistemes que poden utilitzar diferents formats de dades.

Convertidors de Kafka Connect

Kafka Connect proporciona convertidors integrats per a formats de dades comuns com JSON i Avro.

Convertidor JSON

El convertidor JSON s'utilitza per convertir dades a i des del format JSON.

key.converter=org.apache.kafka.connect.json.JsonConverter
value.converter=org.apache.kafka.connect.json.JsonConverter

Convertidor Avro

El convertidor Avro s'utilitza per convertir dades a i des del format Avro. Requereix un registre d'esquemes per gestionar els esquemes Avro.

key.converter=io.confluent.connect.avro.AvroConverter
value.converter=io.confluent.connect.avro.AvroConverter
key.converter.schema.registry.url=http://localhost:8081
value.converter.schema.registry.url=http://localhost:8081

Convertidors Personalitzats

En alguns casos, pot ser necessari implementar convertidors personalitzats per gestionar formats de dades específics o transformacions.

public class CustomConverter implements Converter {
    @Override
    public byte[] fromConnectData(String topic, Schema schema, Object value) {
        // Lògica de serialització personalitzada
    }

    @Override
    public SchemaAndValue toConnectData(String topic, byte[] value) {
        // Lògica de deserialització personalitzada
    }
}

Conclusió

Les transformacions i els convertidors són components essencials de qualsevol canal de dades basat en Kafka. Permeten modificar i formatar dades a mesura que flueixen pel sistema, fent possible integrar Kafka amb una gran varietat d'altres sistemes i aplicacions. Entenent i utilitzant aquestes eines, pots construir solucions de processament de dades més flexibles i poderoses.
