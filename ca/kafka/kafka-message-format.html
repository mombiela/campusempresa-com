Apache Kafka és una plataforma de streaming distribuïda que et permet publicar i subscriure't a fluxos de registres. Entendre el format de missatge de Kafka és crucial per produir i consumir missatges de manera eficient. Aquesta secció cobrirà l'estructura dels missatges de Kafka, conceptes clau, i proporcionarà exemples per ajudar-te a comprendre els fonaments.

Conceptes Clau

  • Missatge: La unitat bàsica de dades a Kafka, també coneguda com a registre.
  • Productor: L'aplicació client que envia registres a un tema de Kafka.
  • Consumidor: L'aplicació client que llegeix registres d'un tema de Kafka.
  • Tema: Una categoria o nom de feed al qual els productors envien registres.
  • Partició: Una divisió d'un tema, que permet el processament en paral·lel.

Estructura del Missatge de Kafka

Un missatge de Kafka consta de diversos components:

  1. Clau: Un identificador opcional per al missatge, utilitzat per a la partició.
  2. Valor: La càrrega útil de dades real del missatge.
  3. Capçaleres: Metadades opcionals per al missatge.
  4. Marca de temps: El moment en què es va produir el missatge.
  5. Desplaçament: Un identificador únic per al missatge dins d'una partició.

Components del Missatge

  • Clau: Ajuda a determinar la partició a la qual s'enviarà el missatge. Si no es proporciona cap clau, Kafka utilitza un enfocament round-robin.
  • Valor: El contingut principal del missatge, que pot ser de qualsevol tipus de dades (per exemple, JSON, XML, Avro).
  • Capçaleres: Parells clau-valor que proporcionen context o metadades addicionals.
  • Marca de temps: Pot ser establerta pel productor o per Kafka mateix.
  • Desplaçament: Assignat automàticament per Kafka, assegurant que cada missatge tingui un identificador únic dins de la seva partició.

Exemple: Produir un Missatge de Kafka

A continuació es mostra un exemple de producció d'un missatge de Kafka en Python utilitzant la biblioteca kafka-python.

from kafka import KafkaProducer
import json

# Inicialitza el productor de Kafka
producer = KafkaProducer(
    bootstrap_servers='localhost:9092',
    value_serializer=lambda v: json.dumps(v).encode('utf-8')
)

# Defineix el missatge
message = {
    'key': 'user123',
    'value': {'event': 'login', 'timestamp': '2023-10-01T12:00:00Z'}
}

# Envia el missatge al tema 'user-events'
producer.send('user-events', key=message['key'].encode('utf-8'), value=message['value'])

# Assegura't que tots els missatges s'enviïn abans de tancar el productor
producer.flush()
producer.close()

Explicació

  • KafkaProducer: Inicialitza el productor amb l'adreça del broker de Kafka i un serialitzador per al valor del missatge.
  • message: Defineix el missatge amb una clau i un valor.
  • producer.send: Envia el missatge al tema especificat.
  • producer.flush: Assegura que tots els missatges en memòria intermèdia s'enviïn.
  • producer.close: Tanca la connexió del productor.

Exemple: Consumir un Missatge de Kafka

A continuació es mostra un exemple de consum d'un missatge de Kafka en Python utilitzant la biblioteca kafka-python.

from kafka import KafkaConsumer
import json

# Inicialitza el consumidor de Kafka
consumer = KafkaConsumer(
    'user-events',
    bootstrap_servers='localhost:9092',
    auto_offset_reset='earliest',
    value_deserializer=lambda v: json.loads(v.decode('utf-8'))
)

# Consumeix missatges del tema 'user-events'
for message in consumer:
    print(f"Key: {message.key.decode('utf-8')}, Value: {message.value}")

# Tanca la connexió del consumidor
consumer.close()

Explicació

  • KafkaConsumer: Inicialitza el consumidor amb l'adreça del broker de Kafka i un deserialitzador per al valor del missatge.
  • consumer: Es subscriu al tema 'user-events'.
  • for message in consumer: Itera sobre els missatges del tema.
  • print: Mostra la clau i el valor de cada missatge.
  • consumer.close: Tanca la connexió del consumidor.

Resum

Entendre el format de missatge de Kafka és essencial per produir i consumir missatges de manera efectiva. Un missatge de Kafka consta d'una clau, valor, capçaleres, marca de temps i desplaçament. Els productors envien missatges als temes, i els consumidors llegeixen missatges dels temes. Dominant aquests conceptes i practicant amb exemples de codi, podràs treballar eficientment amb Kafka a les teves aplicacions.
