Introducció

Apache Kafka és una eina poderosa per construir canals de dades en temps real i aplicacions de streaming. No obstant això, el seu veritable potencial es realitza quan s'integra amb altres sistemes. Aquesta secció cobrirà com integrar Kafka amb diversos sistemes, incloent bases de dades, plataformes de big data i microserveis.

Conceptes Clau

  • Productors i Consumidors: Entendre el paper dels productors i consumidors en Kafka.
  • Connectors: Utilitzar Kafka Connect per integrar-se amb sistemes externs.
  • Streams API: Aprofitar Kafka Streams per al processament de dades en temps real.
  • Schema Registry: Gestionar esquemes per a la serialització i deserialització de dades.

Integrar Kafka amb Bases de Dades

Utilitzant Kafka Connect

Kafka Connect és una eina per transmetre dades de manera escalable i fiable entre Kafka i altres sistemes. Ve amb una varietat de connectors per a diferents bases de dades.

Exemple: Connectar-se a una Base de Dades MySQL

  1. Instal·lar el Connector MySQL:

    confluent-hub install debezium/debezium-connector-mysql:latest

  2. Configurar el Connector:

    {
  "name": "mysql-connector",
  "config": {
    "connector.class": "io.debezium.connector.mysql.MySqlConnector",
    "tasks.max": "1",
    "database.hostname": "localhost",
    "database.port": "3306",
    "database.user": "root",
    "database.password": "password",
    "database.server.id": "184054",
    "database.server.name": "dbserver1",
    "database.whitelist": "inventory",
    "database.history.kafka.bootstrap.servers": "localhost:9092",
    "database.history.kafka.topic": "schema-changes.inventory"
  }
}

  3. Desplegar el Connector:

    curl -X POST -H "Content-Type: application/json" --data @mysql-connector.json http://localhost:8083/connectors

Utilitzant Kafka Streams

Kafka Streams es pot utilitzar per processar dades de bases de dades en temps real.

Exemple: Processament de Stream Simple

Properties props = new Properties();
props.put(StreamsConfig.APPLICATION_ID_CONFIG, "streams-example");
props.put(StreamsConfig.BOOTSTRAP_SERVERS_CONFIG, "localhost:9092");
props.put(StreamsConfig.DEFAULT_KEY_SERDE_CLASS_CONFIG, Serdes.String().getClass());
props.put(StreamsConfig.DEFAULT_VALUE_SERDE_CLASS_CONFIG, Serdes.String().getClass());

StreamsBuilder builder = new StreamsBuilder();
KStream<String, String> source = builder.stream("input-topic");
KStream<String, String> transformed = source.mapValues(value -> value.toUpperCase());
transformed.to("output-topic");

KafkaStreams streams = new KafkaStreams(builder.build(), props);
streams.start();

Integrar Kafka amb Plataformes de Big Data

Hadoop

Kafka es pot integrar amb Hadoop per al processament per lots.

Exemple: Utilitzant Kafka Connect amb HDFS

  1. Instal·lar el Connector HDFS:

    confluent-hub install confluentinc/kafka-connect-hdfs:latest

  2. Configurar el Connector:

    {
  "name": "hdfs-sink-connector",
  "config": {
    "connector.class": "io.confluent.connect.hdfs.HdfsSinkConnector",
    "tasks.max": "1",
    "topics": "input-topic",
    "hdfs.url": "hdfs://localhost:9000",
    "flush.size": "3"
  }
}

  3. Desplegar el Connector:

    curl -X POST -H "Content-Type: application/json" --data @hdfs-sink-connector.json http://localhost:8083/connectors

Spark

Kafka es pot integrar amb Apache Spark per a analítiques en temps real.

Exemple: Utilitzant Spark Structured Streaming amb Kafka

from pyspark.sql import SparkSession

spark = SparkSession.builder \
    .appName("KafkaSparkIntegration") \
    .getOrCreate()

df = spark \
    .readStream \
    .format("kafka") \
    .option("kafka.bootstrap.servers", "localhost:9092") \
    .option("subscribe", "input-topic") \
    .load()

df.selectExpr("CAST(key AS STRING)", "CAST(value AS STRING)") \
    .writeStream \
    .format("console") \
    .start() \
    .awaitTermination()

Integrar Kafka amb Microserveis

Kafka sovint s'utilitza com a columna vertebral per a la comunicació de microserveis a causa del seu alt rendiment i tolerància a fallades.

Exemple: Comunicació de Microserveis

  1. Servei Productor:

    Properties props = new Properties();
props.put("bootstrap.servers", "localhost:9092");
props.put("key.serializer", "org.apache.kafka.common.serialization.StringSerializer");
props.put("value.serializer", "org.apache.kafka.common.serialization.StringSerializer");

Producer<String, String> producer = new KafkaProducer<>(props);
producer.send(new ProducerRecord<>("input-topic", "key", "value"));
producer.close();

  2. Servei Consumidor:

    Properties props = new Properties();
props.put("bootstrap.servers", "localhost:9092");
props.put("group.id", "test-group");
props.put("key.deserializer", "org.apache.kafka.common.serialization.StringDeserializer");
props.put("value.deserializer", "org.apache.kafka.common.serialization.StringDeserializer");

KafkaConsumer<String, String> consumer = new KafkaConsumer<>(props);
consumer.subscribe(Arrays.asList("input-topic"));

while (true) {
    ConsumerRecords<String, String> records = consumer.poll(Duration.ofMillis(100));
    for (ConsumerRecord<String, String> record : records) {
        System.out.printf("offset = %d, key = %s, value = %s%n", record.offset(), record.key(), record.value());
    }
}

Conclusió

Integrar Kafka amb altres sistemes pot millorar significativament les teves capacitats de processament de dades. Ja sigui que estiguis connectant-te a bases de dades, plataformes de big data o microserveis, Kafka proporciona eines i API robustes per facilitar una integració sense problemes. En dominar aquestes integracions, pots construir canals de dades escalables i en temps real que impulsin les teves aplicacions i serveis.
