◄ Definició i funcions d'un sistema operatiu
Components principals d'un sistema operatiu ►

En aquesta secció del curs sobre Fonaments de Sistemes Operatius, explorarem els diferents tipus de sistemes operatius, les seves característiques i exemples comuns. Aquest coneixement és fonamental per entendre com funcionen els sistemes informàtics i com triar el sistema operatiu adequat per a diferents aplicacions i entorns.

Sistemes Operatius Monotasca

Els sistemes operatius monotasca estan dissenyats per executar una sola tasca a la vegada. Són simples i tenen un ús limitat en aplicacions modernes, però són importants per entendre l'evolució dels sistemes operatius.

  • Característiques:

    • Executen una sola tasca a la vegada.
    • Simplicitat en disseny i ús.
    • Menor consum de recursos.

  • Exemples:

    • MS-DOS
    • CP/M

Sistemes Operatius Multitasca

Els sistemes operatius multitasca permeten l'execució de múltiples tasques simultàniament. Això és essencial per a l'eficiència i productivitat en entorns moderns.

  • Característiques:

    • Capacitat per executar múltiples tasques al mateix temps.
    • Millor utilització de recursos del sistema.
    • Major complexitat en disseny i gestió.

  • Exemples:

    • Windows
    • Linux
    • macOS

Sistemes Operatius de Temps Real (RTOS)

Els sistemes operatius de temps real estan dissenyats per a aplicacions que requereixen una resposta immediata i previsible a esdeveniments externs.

  • Característiques:

    • Resposta ràpida i previsible a esdeveniments.
    • Alta fiabilitat i estabilitat.
    • Usats en sistemes embeguts i aplicacions crítiques.

  • Exemples:

    • VxWorks
    • FreeRTOS
    • QNX

Sistemes Operatius Distribuïts

Els sistemes operatius distribuïts gestionen un grup d'ordinadors independents i els presenten com un únic sistema coherent.

  • Característiques:

    • Coordinació i gestió de recursos distribuïts.
    • Transparència en la distribució de tasques.
    • Escalabilitat i tolerància a fallades.

  • Exemples:

    • Apache Hadoop
    • Google File System (GFS)
    • Microsoft Azure

Sistemes Operatius de Xarxa

Els sistemes operatius de xarxa proporcionen serveis i recursos a través d'una xarxa, permetent la comunicació i l'intercanvi de dades entre múltiples dispositius.

  • Característiques:

    • Gestió de recursos i serveis en xarxa.
    • Suport per a múltiples usuaris i dispositius.
    • Seguretat i administració de xarxa.

  • Exemples:

    • Novell NetWare
    • Windows Server
    • UNIX

Comparació de Tipus de Sistemes Operatius

| Tipus de Sistema Operatiu | Característiques Principals | Exemples | |---------------------------|-----------------------------|----------| | Monotasca | Executa una sola tasca a la vegada | MS-DOS, CP/M | | Multitasca | Executa múltiples tasques simultàniament | Windows, Linux, macOS | | Temps Real (RTOS) | Resposta ràpida i previsible a esdeveniments | VxWorks, FreeRTOS, QNX | | Distribuït | Gestiona recursos distribuïts com un únic sistema | Apache Hadoop, GFS, Azure | | De Xarxa | Proporciona serveis i recursos en xarxa | Novell NetWare, Windows Server, UNIX |

Conclusió

Comprendre els diferents tipus de sistemes operatius i les seves característiques és crucial per seleccionar el sistema adequat per a diverses aplicacions i entorns. Des de sistemes simples monotasca fins a complexos sistemes distribuïts, cada tipus té el seu propi conjunt d'avantatges i desavantatges que han de ser considerats en funció dels requisits específics del projecte o aplicació.
