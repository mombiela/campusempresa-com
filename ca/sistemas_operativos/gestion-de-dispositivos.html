Introducció

La gestió de dispositius d'entrada/sortida (E/S) és una funció crucial dels sistemes operatius. Aquesta secció del curs sobre Fonaments de Sistemes Operatius cobrirà els conceptes bàsics, la importància i les tècniques utilitzades per gestionar els dispositius d'E/S.

Conceptes Clau

  • Dispositius d'E/S: Maquinari que permet la comunicació entre el sistema informàtic i el món exterior. Exemples inclouen teclats, ratolins, impressores, discos durs i monitors.
  • Controladors de Dispositius: Programari que permet al sistema operatiu comunicar-se amb el maquinari d'E/S.
  • Interrupcions: Senyals enviades pels dispositius d'E/S per indicar que necessiten atenció del processador.
  • Buffering: Tècnica que utilitza memòria intermèdia per gestionar la transferència de dades entre dispositius d'E/S i la memòria principal.
  • Spooling: Tècnica que emmagatzema temporalment les dades en el disc per gestionar la impressió i altres operacions d'E/S.
  • DMA (Accés Directe a Memòria): Permet als dispositius d'E/S transferir dades directament a la memòria principal sense la intervenció de la CPU.

Tipus de Dispositius d'E/S

| Tipus de Dispositiu | Exemples | Característiques | |---------------------|----------|------------------| | Entrada | Teclat, Ratolí | Permeten a l'usuari introduir dades al sistema. | | Sortida | Monitor, Impressora | Permeten al sistema mostrar dades a l'usuari. | | Emmagatzematge | Disc Dur, SSD | Permeten emmagatzemar dades de manera persistent. | | Comunicació | Targeta de Xarxa, Mòdem | Permeten la comunicació entre sistemes. |

Tècniques de Gestió d'E/S

Interrupcions

Les interrupcions són senyals enviades pels dispositius d'E/S per indicar que necessiten atenció del processador. Hi ha dos tipus principals d'interrupcions:

  • Interrupcions de Maquinari: Generades per dispositius de maquinari.
  • Interrupcions de Programari: Generades per programes.

Exemple de Codi: Gestió d'Interrupcions en C

#include <stdio.h>
#include <signal.h>

void handle_interrupt(int signal) {
    printf("Interrupció rebuda: %d\n", signal);
}

int main() {
    signal(SIGINT, handle_interrupt); // Captura el senyal d'interrupció (Ctrl+C)
    while (1) {
        printf("Esperant interrupció...\n");
        sleep(1);
    }
    return 0;
}

Buffering

El buffering utilitza memòria intermèdia per gestionar la transferència de dades entre dispositius d'E/S i la memòria principal. Això ajuda a millorar l'eficiència i la velocitat de les operacions d'E/S.

  • Buffer d'Entrada: Emmagatzema dades que es processaran.
  • Buffer de Sortida: Emmagatzema dades que s'enviaran a un dispositiu de sortida.

Spooling

El spooling emmagatzema temporalment les dades en el disc per gestionar la impressió i altres operacions d'E/S. És especialment útil en sistemes multitasca.

DMA (Accés Directe a Memòria)

El DMA permet als dispositius d'E/S transferir dades directament a la memòria principal sense la intervenció de la CPU, la qual cosa allibera el processador per realitzar altres tasques.

Exemple de Gestió d'E/S en un Sistema Operatiu

Exemple de Codi: Lectura d'un Fitxer en C

#include <stdio.h>

int main() {
    FILE *file;
    char buffer[256];

    file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error en obrir el fitxer.\n");
        return 1;
    }

    while (fgets(buffer, sizeof(buffer), file) != NULL) {
        printf("%s", buffer);
    }

    fclose(file);
    return 0;
}

Conclusió

La gestió de dispositius d'entrada/sortida és una part essencial dels sistemes operatius. Comprendre com funcionen els dispositius d'E/S, els controladors de dispositius, les interrupcions, el buffering, el spooling i el DMA és crucial per a qualsevol professional de TI. Aquestes tècniques i conceptes permeten que els sistemes operatius gestionin eficientment les operacions d'E/S, millorant el rendiment i la usabilitat del sistema.

En la propera secció, explorarem la gestió de memòria en els sistemes operatius, un altre component fonamental per al funcionament eficient dels sistemes informàtics.
