Introducció a les Transformacions

Les transformacions CSS permeten modificar l'espai de coordenades del model de format visual CSS. Això significa que pots rotar, escalar, inclinar o traslladar elements en espai 2D o 3D.

  • Funcions de Transformació: Les funcions principals utilitzades en les transformacions CSS són translate, rotate, scale i skew.
  • Propietat Transform: La propietat transform aplica una transformació 2D o 3D a un element.

Transformacions 2D

Translate

La funció translate mou un element des de la seva posició actual.

.element {
  transform: translate(50px, 100px);
}
  • Explicació: Aquest codi mou l'element 50 píxels a la dreta i 100 píxels cap avall des de la seva posició original.

Rotate

La funció rotate rota un element al voltant d'un punt fix.

.element {
  transform: rotate(45deg);
}
  • Explicació: Aquest codi rota l'element 45 graus en sentit horari.

Scale

La funció scale redimensiona un element.

.element {
  transform: scale(1.5, 2);
}
  • Explicació: Aquest codi escala l'element a 1.5 vegades la seva amplada i 2 vegades la seva alçada.

Skew

La funció skew inclina un element al llarg dels eixos X i Y.

.element {
  transform: skew(30deg, 20deg);
}
  • Explicació: Aquest codi inclina l'element 30 graus al llarg de l'eix X i 20 graus al llarg de l'eix Y.

Transformacions 3D

Perspective

La funció perspective defineix la perspectiva des de la qual es veuen els elements transformats en 3D.

.container {
  perspective: 500px;
}
  • Explicació: Aquest codi estableix la distància de perspectiva a 500 píxels.

RotateX, RotateY, RotateZ

Aquestes funcions roden un element al voltant dels eixos X, Y i Z, respectivament.

.element {
  transform: rotateX(45deg);
}
  • Explicació: Aquest codi rota l'element 45 graus al voltant de l'eix X.

Scale3d

La funció scale3d escala un element en espai 3D.

.element {
  transform: scale3d(1.5, 2, 1);
}
  • Explicació: Aquest codi escala l'element a 1.5 vegades la seva amplada, 2 vegades la seva alçada i 1 vegada la seva profunditat.

Translate3d

La funció translate3d mou un element en espai 3D.

.element {
  transform: translate3d(50px, 100px, 200px);
}
  • Explicació: Aquest codi mou l'element 50 píxels al llarg de l'eix X, 100 píxels al llarg de l'eix Y i 200 píxels al llarg de l'eix Z.

Combinació de Transformacions

Pots combinar múltiples transformacions utilitzant una llista separada per espais.

.element {
  transform: translate(50px, 100px) rotate(45deg) scale(1.5, 2);
}
  • Explicació: Aquest codi mou l'element 50 píxels a la dreta i 100 píxels cap avall, el rota 45 graus en sentit horari i l'escala a 1.5 vegades la seva amplada i 2 vegades la seva alçada.

Origen de la Transformació

La propietat transform-origin permet canviar el punt al voltant del qual s'aplica una transformació.

.element {
  transform-origin: top left;
  transform: rotate(45deg);
}
  • Explicació: Aquest codi estableix l'origen de la transformació a la cantonada superior esquerra de l'element i després el rota 45 graus.

Exemples Pràctics

Exemple 1: Efecte de Botó en Passar el Ratolí

Crea un botó que s'escala lleugerament quan es passa el ratolí per sobre.

.button {
  transition: transform 0.3s;
}

.button:hover {
  transform: scale(1.1);
}
  • Explicació: Aquest codi escala el botó a 1.1 vegades la seva mida original quan es passa el ratolí per sobre, amb un efecte de transició suau.

Exemple 2: Efecte de Volteig de Targeta 3D

Crea un efecte de volteig de targeta 3D en passar el ratolí per sobre.

.card {
  perspective: 1000px;
}

.card-inner {
  transition: transform 0.6s;
  transform-style: preserve-3d;
}

.card:hover .card-inner {
  transform: rotateY(180deg);
}

.card-front, .card-back {
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}
  • Explicació: Aquest codi crea un efecte de volteig de targeta 3D. Quan es passa el ratolí per sobre de la targeta, aquesta rota 180 graus al voltant de l'eix Y, revelant la part posterior.

Conclusió

Les transformacions CSS són eines poderoses que permeten manipular elements tant en espai 2D com 3D. Dominant les funcions translate, rotate, scale i skew, juntament amb la comprensió de com combinar transformacions i establir orígens de transformació, pots crear dissenys web dinàmics i visualment atractius. Practica amb els exemples proporcionats i experimenta amb diferents combinacions de transformacions per millorar les teves habilitats en CSS.
