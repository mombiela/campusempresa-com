Introducció a Flexbox

Flexbox, o el Flexible Box Layout, és un mòdul de disseny CSS dissenyat per proporcionar una manera més eficient de dissenyar, alinear i distribuir l'espai entre els elements d'un contenidor. És particularment útil per crear dissenys web responsius.

Conceptes Clau

  • Flex Container: L'element pare que conté els elements flexibles.
  • Flex Items: Els elements fills dins d'un contenidor flexible.
  • Main Axis: L'eix principal al llarg del qual es disposen els elements flexibles (horitzontal per defecte).
  • Cross Axis: L'eix perpendicular a l'eix principal (vertical per defecte).

Configuració de Flexbox

Per començar a utilitzar Flexbox, cal definir un contenidor flexible establint la propietat display a flex o inline-flex.

.container {
  display: flex;
}

Propietats del Contenidor Flexible

`flex-direction`

Defineix la direcció en què es col·loquen els elements flexibles dins del contenidor flexible.

  • row (per defecte): Els elements es col·loquen horitzontalment.
  • row-reverse: Els elements es col·loquen horitzontalment en ordre invers.
  • column: Els elements es col·loquen verticalment.
  • column-reverse: Els elements es col·loquen verticalment en ordre invers.
.container {
  display: flex;
  flex-direction: row;
}

`flex-wrap`

Controla si els elements flexibles han de fer un salt de línia o no quan desborden el contenidor.

  • nowrap (per defecte): Tots els elements flexibles estaran en una sola línia.
  • wrap: Els elements flexibles faran un salt de línia en múltiples línies.
  • wrap-reverse: Els elements flexibles faran un salt de línia en múltiples línies en ordre invers.
.container {
  display: flex;
  flex-wrap: wrap;
}

`justify-content`

Alinea els elements flexibles al llarg de l'eix principal.

  • flex-start (per defecte): Els elements es col·loquen cap al començament del contenidor flexible.
  • flex-end: Els elements es col·loquen cap al final del contenidor flexible.
  • center: Els elements es centren al llarg de l'eix principal.
  • space-between: Els elements es distribueixen uniformement amb el primer element al començament i l'últim element al final.
  • space-around: Els elements es distribueixen uniformement amb espai igual al voltant d'ells.
.container {
  display: flex;
  justify-content: center;
}

`align-items`

Alinea els elements flexibles al llarg de l'eix transversal.

  • stretch (per defecte): Els elements s'estiren per omplir el contenidor.
  • flex-start: Els elements s'alineen al començament de l'eix transversal.
  • flex-end: Els elements s'alineen al final de l'eix transversal.
  • center: Els elements es centren al llarg de l'eix transversal.
  • baseline: Els elements s'alineen de manera que les seves línies base coincideixin.
.container {
  display: flex;
  align-items: center;
}

`align-content`

Alinea les línies flexibles quan hi ha espai addicional a l'eix transversal.

  • stretch (per defecte): Les línies s'estiren per ocupar l'espai restant.
  • flex-start: Les línies es col·loquen cap al començament del contenidor flexible.
  • flex-end: Les línies es col·loquen cap al final del contenidor flexible.
  • center: Les línies es centren dins del contenidor flexible.
  • space-between: Les línies es distribueixen uniformement amb la primera línia al començament i l'última línia al final.
  • space-around: Les línies es distribueixen uniformement amb espai igual al voltant d'elles.
.container {
  display: flex;
  align-content: space-around;
}

Propietats dels Elements Flexibles

`order`

Defineix l'ordre dels elements flexibles. El valor per defecte és 0.

.item {
  order: 1;
}

`flex-grow`

Defineix la capacitat d'un element flexible per créixer si és necessari. El valor per defecte és 0.

.item {
  flex-grow: 1;
}

`flex-shrink`

Defineix la capacitat d'un element flexible per reduir-se si és necessari. El valor per defecte és 1.

.item {
  flex-shrink: 1;
}

`flex-basis`

Defineix la mida per defecte d'un element abans que es distribueixi l'espai restant. Pot ser una longitud (per exemple, 20%, 5rem) o auto.

.item {
  flex-basis: 100px;
}

`align-self`

Sobreescriu la propietat align-items per a elements flexibles individuals.

  • auto (per defecte): Hereta el valor de align-items del pare.
  • flex-start: Alinea l'element al començament de l'eix transversal.
  • flex-end: Alinea l'element al final de l'eix transversal.
  • center: Centra l'element al llarg de l'eix transversal.
  • baseline: Alinea l'element de manera que la seva línia base coincideixi amb la línia base del pare.
  • stretch: Estira l'element per omplir el contenidor.
.item {
  align-self: flex-end;
}

Exemples Pràctics

Disseny Bàsic amb Flexbox

<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  background-color: lightblue;
  padding: 10px;
  margin: 5px;
}

Barra de Navegació Responsiva

<nav class="navbar">
  <div class="logo">Logo</div>
  <div class="menu">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Services</a>
    <a href="#">Contact</a>
  </div>
</nav>
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 10px;
  background-color: #333;
}

.logo {
  color: white;
  font-size: 1.5rem;
}

.menu a {
  color: white;
  margin: 0 10px;
  text-decoration: none;
}

Conclusió

Flexbox és una eina poderosa per crear dissenys flexibles i responsius amb facilitat. Comprenent i utilitzant les diverses propietats de Flexbox, pots controlar l'alineació, l'espaiat i l'ordre dels elements dins d'un contenidor, convertint-lo en una habilitat essencial per al desenvolupament web modern. Practica l'ús de Flexbox en diferents escenaris per convertir-te en un expert i crear dissenys web més dinàmics.
