CSS (Fulls d'Estil en Cascada) és una eina poderosa utilitzada per controlar el disseny i la presentació de pàgines web. Entendre el disseny CSS és crucial per crear llocs web visualment atractius i fàcils d'usar. Aquest tema cobrirà els conceptes bàsics del disseny CSS, incloent conceptes clau, propietats i exemples pràctics.

Conceptes Clau del Disseny CSS

  • Model de Caixa: La base del disseny CSS, que inclou marges, vores, farciment i l'àrea de contingut.
  • Propietat Display: Determina com es mostra un element a la pàgina web (per exemple, block, inline, inline-block).
  • Posicionament: Controla la posició dels elements a la pàgina (per exemple, static, relative, absolute, fixed, sticky).
  • Flexbox: Un model de disseny que proporciona una manera eficient d'alinear i distribuir espai entre elements en un contenidor.
  • Disseny de Graella: Un sistema de disseny potent per crear dissenys web complexos i responsius.

Model de Caixa

El model de caixa CSS és essencial per entendre com es renderitzen els elements a la pàgina web. Consisteix en:

  • Contingut: El contingut real de la caixa, on apareixen text i imatges.
  • Farciment: L'espai entre el contingut i la vora.
  • Vora: La línia que envolta el farciment (si n'hi ha) i el contingut.
  • Marge: L'espai fora de la vora, separant l'element d'altres elements.

Exemple

.box {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

En aquest exemple, l'amplada total de l'element .box és de 270px (200px de contingut + 20px de farciment a cada costat + 5px de vora a cada costat).

Propietat Display

La propietat display especifica el comportament de visualització d'un element. Els valors comuns inclouen:

  • block: L'element ocupa tota l'amplada disponible i comença en una nova línia.
  • inline: L'element ocupa només l'amplada necessària i no comença en una nova línia.
  • inline-block: L'element es formata com un element en línia però pot tenir amplada i alçada.
  • none: L'element no es mostra.

Exemple

.block-element {
  display: block;
}

.inline-element {
  display: inline;
}

.inline-block-element {
  display: inline-block;
}

Posicionament

El posicionament CSS permet controlar la col·locació dels elements a la pàgina. Els principals tipus de posicionament són:

  • static: El posicionament per defecte; els elements es posicionen segons el flux normal del document.
  • relative: L'element es posiciona en relació a la seva posició normal.
  • absolute: L'element es posiciona en relació al seu ancestre posicionat més proper.
  • fixed: L'element es posiciona en relació a la finestra de visualització i no es mou quan es desplaça la pàgina.
  • sticky: L'element alterna entre posicionament relatiu i fix en funció de la posició de desplaçament de l'usuari.

Exemple

.relative-element {
  position: relative;
  top: 10px;
  left: 20px;
}

.absolute-element {
  position: absolute;
  top: 50px;
  left: 100px;
}

.fixed-element {
  position: fixed;
  bottom: 0;
  right: 0;
}

Flexbox

Flexbox és un model de disseny que permet dissenyar dissenys complexos amb facilitat. És particularment útil per crear dissenys responsius.

Propietats Clau

  • display: flex: Defineix un contenidor flex.
  • flex-direction: Defineix la direcció dels elements flex (row, row-reverse, column, column-reverse).
  • justify-content: Alinea els elements flex al llarg de l'eix principal (flex-start, flex-end, center, space-between, space-around).
  • align-items: Alinea els elements flex al llarg de l'eix transversal (flex-start, flex-end, center, baseline, stretch).

Exemple

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

.flex-item {
  width: 100px;
  height: 100px;
  background-color: lightblue;
}

Disseny de Graella

El disseny de graella CSS és un sistema de disseny bidimensional per a la web. Permet crear dissenys complexos amb files i columnes.

Propietats Clau

  • display: grid: Defineix un contenidor de graella.
  • grid-template-columns: Defineix les columnes de la graella.
  • grid-template-rows: Defineix les files de la graella.
  • grid-gap: Defineix l'espai entre els elements de la graella.

Exemple

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px;
  grid-gap: 10px;
}

.grid-item {
  background-color: lightgreen;
  border: 1px solid black;
}

Conclusió

Entendre el disseny CSS és fonamental per crear pàgines web ben estructurades i visualment atractives. Dominant el model de caixa, la propietat display, el posicionament, Flexbox i el disseny de graella, pots dissenyar dissenys responsius i complexos amb facilitat. Practica aquests conceptes amb exemples del món real per consolidar els teus coneixements i millorar les teves habilitats de disseny web.
