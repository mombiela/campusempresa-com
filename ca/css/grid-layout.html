Introducció al Disseny de Graella en CSS

El Disseny de Graella en CSS és un sistema de disseny potent disponible en CSS. Et permet crear dissenys web complexos amb facilitat definint files i columnes, i col·locant elements precisament dins d'una graella.

Conceptes Clau

  • Contenidor de Graella: L'element al qual s'aplica display: grid. És el pare de tots els elements de la graella.
  • Element de Graella: Els fills del contenidor de la graella.
  • Línies de Graella: Les línies divisòries que formen l'estructura de la graella.
  • Pistes de Graella: L'espai entre dues línies de graella. Poden ser files o columnes.
  • Cel·la de Graella: L'espai entre quatre línies de graella, essencialment una "caixa" única dins de la graella.
  • Àrea de Graella: Una àrea rectangular que abasta múltiples cel·les de graella.

Creant un Contenidor de Graella

Per començar a utilitzar la Graella en CSS, necessites definir un contenidor de graella aplicant display: grid a un element.

.container {
  display: grid;
}

Definint Files i Columnes

Pots definir l'estructura de la teva graella utilitzant les propietats grid-template-rows i grid-template-columns.

.container {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 50px 100px;
}

Això crea una graella amb tres columnes i dues files.

Col·locant Elements de Graella

Pots col·locar elements de graella en cel·les específiques utilitzant les propietats grid-column i grid-row.

.item1 {
  grid-column: 1 / 3; /* abasta des de la línia de columna 1 fins a la 3 */
  grid-row: 1 / 2; /* abasta des de la línia de fila 1 fins a la 2 */
}

Espai entre Graelles

La propietat grid-gap (o column-gap i row-gap) et permet afegir espai entre els elements de la graella.

.container {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 50px 100px;
  grid-gap: 10px;
}

Propietats Avançades de la Graella

Àrees de Plantilla de Graella

Pots definir àrees de graella amb nom i col·locar elements dins d'aquestes àrees.

.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
  grid-template-rows: auto;
  grid-template-columns: 1fr 2fr 1fr;
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.content {
  grid-area: content;
}

.footer {
  grid-area: footer;
}

Graelles Implícites vs. Explícites

  • Graella Explícita: Definida utilitzant grid-template-rows i grid-template-columns.
  • Graella Implícita: Creada automàticament pel navegador quan els elements es col·loquen fora de la graella explícita.

Col·locació Automàtica

La Graella en CSS pot col·locar automàticament els elements utilitzant les propietats grid-auto-rows, grid-auto-columns i grid-auto-flow.

.container {
  display: grid;
  grid-template-columns: 100px 200px;
  grid-auto-rows: 50px;
  grid-auto-flow: row;
}

Disseny Responsiu amb Graella

La Graella en CSS facilita la creació de dissenys responsius. Pots utilitzar media queries per canviar l'estructura de la graella segons la mida de la finestra.

.container {
  display: grid;
  grid-template-columns: 1fr;
}

@media (min-width: 600px) {
  .container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 900px) {
  .container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}

Conclusió

El Disseny de Graella en CSS és una eina versàtil i potent per crear dissenys web complexos. Entenent els conceptes clau i les propietats, pots crear dissenys responsius i flexibles amb facilitat. Practica utilitzant el disseny de graella en diversos projectes per convertir-te en un expert en aquesta característica essencial de CSS.
