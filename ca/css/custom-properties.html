Introducció a les propietats personalitzades

Les propietats personalitzades, també conegudes com a variables CSS, et permeten emmagatzemar valors que pots reutilitzar al llarg de la teva fulla d'estil. Això fa que el teu CSS sigui més mantenible i fàcil de llegir.

  • Definició: Les propietats personalitzades són entitats definides pels autors de CSS que contenen valors específics per ser reutilitzats al llarg d'un document.
  • Sintaxi: Les propietats personalitzades es defineixen utilitzant el prefix -- i s'accedeix a elles utilitzant la funció var().

Sintaxi bàsica

Per definir una propietat personalitzada, utilitzes el prefix -- seguit del nom de la propietat. Després pots utilitzar la funció var() per accedir al valor de la propietat personalitzada.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
}

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}
  • :root: Aquesta és una pseudo-classe que coincideix amb l'element arrel del document. Sovint s'utilitza per definir propietats personalitzades globals.
  • --main-bg-color: Aquesta és una propietat personalitzada per al color de fons principal.
  • var(--main-bg-color): Aquesta funció recupera el valor de la propietat personalitzada.

Avantatges d'utilitzar propietats personalitzades

  • Reutilització: Defineix un valor una vegada i reutilitza'l al llarg de la teva fulla d'estil.
  • Mantenibilitat: Canvia el valor en un sol lloc, i s'actualitza a tot arreu on s'utilitza.
  • Canvis dinàmics: Les propietats personalitzades es poden actualitzar dinàmicament utilitzant JavaScript.

Exemple: Reutilització i mantenibilitat

:root {
  --primary-color: #e74c3c;
  --secondary-color: #2ecc71;
}

.header {
  background-color: var(--primary-color);
}

.footer {
  background-color: var(--secondary-color);
}

.button {
  background-color: var(--primary-color);
  color: var(--secondary-color);
}

En aquest exemple, canviar --primary-color o --secondary-color en el :root actualitzarà l'esquema de colors a tots els elements que utilitzen aquestes propietats personalitzades.

Ús avançat

Valors de reserva

Pots proporcionar un valor de reserva en cas que la propietat personalitzada no estigui definida.

.button {
  background-color: var(--nonexistent-color, #333);
}

En aquest cas, si --nonexistent-color no està definit, el color de fons del botó serà per defecte #333.

Media Queries i propietats personalitzades

Les propietats personalitzades es poden utilitzar dins de les media queries per crear dissenys responsius.

:root {
  --font-size: 16px;
}

@media (min-width: 768px) {
  :root {
    --font-size: 18px;
  }
}

body {
  font-size: var(--font-size);
}

Aquí, la mida de la font canvia en funció de l'amplada de la finestra, demostrant com les propietats personalitzades poden adaptar-se a diferents mides de pantalla.

JavaScript i propietats personalitzades

Pots manipular les propietats personalitzades utilitzant JavaScript per crear estils dinàmics.

<!DOCTYPE html>
<html lang="ca">
<head>
  <style>
    :root {
      --dynamic-color: #3498db;
    }
    .dynamic {
      color: var(--dynamic-color);
    }
  </style>
</head>
<body>
  <p class="dynamic">Aquest text canviarà de color dinàmicament.</p>
  <button onclick="changeColor()">Canvia el color</button>

  <script>
    function changeColor() {
      document.documentElement.style.setProperty('--dynamic-color', '#e74c3c');
    }
  </script>
</body>
</html>

En aquest exemple, fer clic al botó canvia el color del text actualitzant la propietat personalitzada --dynamic-color.

Conclusió

Les propietats personalitzades en CSS proporcionen una manera poderosa de gestionar i reutilitzar valors a les teves fulles d'estil. Milloren la mantenibilitat, la reutilització i permeten canvis dinàmics, fent-les una eina essencial per al desenvolupament web modern. Entenent i utilitzant les propietats personalitzades, pots crear fulles d'estil més eficients i adaptables.
