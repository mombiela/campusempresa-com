CSS-in-JS és un enfocament modern per estilitzar aplicacions web on el CSS s'escriu utilitzant JavaScript. Aquesta tècnica permet estils més dinàmics i mantenibles, especialment en aplicacions a gran escala.

Introducció al CSS-in-JS

  • Definició: CSS-in-JS es refereix a un patró on el CSS es compon utilitzant JavaScript en lloc d'arxius CSS externs tradicionals.
  • Beneficis:
    • Estils encapsulats: Els estils estan encapsulats en components, reduint el risc de conflictes.
    • Estilització dinàmica: Els estils es poden generar dinàmicament basant-se en l'estat del component o les seves propietats.
    • Mantenibilitat millorada: Els estils estan col·locats amb els components, fent més fàcil gestionar i mantenir.

Biblioteques populars de CSS-in-JS

  • Styled-components: Una biblioteca per a React i React Native que et permet utilitzar ES6 i CSS per estilitzar components.
  • Emotion: Una biblioteca de CSS-in-JS rendible i flexible.
  • JSS: Una biblioteca per generar CSS amb JavaScript.

Començant amb Styled-components

Instal·lació

Per utilitzar styled-components, necessites instal·lar-la via npm o yarn:

npm install styled-components

o

yarn add styled-components

Ús bàsic

Aquí tens un exemple senzill de com utilitzar styled-components en una aplicació React:

import React from 'react';
import styled from 'styled-components';

// Defineix un component de botó estilitzat
const Button = styled.button`
  background-color: #4CAF50;
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
`;

const App = () => (
  <div>
    <Button>Click Me</Button>
  </div>
);

export default App;

Estilització dinàmica

Pots crear estils dinàmics basant-te en les propietats:

const Button = styled.button`
  background-color: ${props => props.primary ? '#4CAF50' : '#008CBA'};
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
`;

const App = () => (
  <div>
    <Button primary>Primary Button</Button>
    <Button>Secondary Button</Button>
  </div>
);

export default App;

Conceptes avançats

Tematització

Styled-components suporta tematització, la qual cosa et permet definir un conjunt d'estils que es poden aplicar a tota la teva aplicació.

import { ThemeProvider } from 'styled-components';

const theme = {
  primaryColor: '#4CAF50',
  secondaryColor: '#008CBA',
};

const Button = styled.button`
  background-color: ${props => props.theme.primaryColor};
  border: none;
  color: white;
  padding: 15px 32px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
`;

const App = () => (
  <ThemeProvider theme={theme}>
    <Button>Click Me</Button>
  </ThemeProvider>
);

export default App;

Estils globals

Pots definir estils globals utilitzant el helper createGlobalStyle:

import { createGlobalStyle } from 'styled-components';

const GlobalStyle = createGlobalStyle`
  body {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
    font-family: 'Arial', sans-serif;
  }
`;

const App = () => (
  <>
    <GlobalStyle />
    <div>
      <h1>Hello World</h1>
    </div>
  </>
);

export default App;

Conclusió

CSS-in-JS ofereix una manera potent de gestionar els estils en aplicacions web modernes, especialment quan s'utilitzen biblioteques basades en components com React. Aprofitant JavaScript, proporciona estils dinàmics, encapsulats i mantenibles que poden millorar significativament l'experiència de desenvolupament. Tot i que pot introduir una certa sobrecàrrega de rendiment, els beneficis sovint superen els costos, particularment en aplicacions a gran escala.
