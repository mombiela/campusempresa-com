Bootstrap és un potent framework de front-end per a un desenvolupament web més ràpid i fàcil. Inclou plantilles de disseny basades en HTML i CSS per a tipografia, formularis, botons, taules, navegació, modals, carrusels d'imatges i molts altres components, així com plugins de JavaScript opcionals.

Introducció a Bootstrap

  • Què és Bootstrap?
    • Un framework CSS gratuït i de codi obert dirigit al desenvolupament web de front-end, primer per a mòbils i amb capacitat de resposta.
    • Conté plantilles de disseny basades en CSS i JavaScript.
  • Per què utilitzar Bootstrap?
    • Accelera el desenvolupament.
    • Garanteix la consistència entre diferents navegadors i dispositius.
    • Proporciona un sistema de graella responsiu.
    • Ofereix components preestilitzats extensius.

Començant amb Bootstrap

  • Incloure Bootstrap en el teu projecte
    • Utilitzant CDN (Xarxa de Distribució de Continguts): 
      <!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemple de Bootstrap</title>
  <link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <h1>Hola, Bootstrap!</h1>
  <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.5.4/dist/umd/popper.min.js"></script>
  <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
</body>
</html>
    • Descarregant Bootstrap:
      • Descarrega des de getbootstrap.com.
      • Inclou els fitxers descarregats al directori del teu projecte.

Sistema de graella de Bootstrap

  • Entenent el sistema de graella
    • El sistema de graella de Bootstrap utilitza una sèrie de contenidors, files i columnes per disposar i alinear contingut.
    • Està construït amb flexbox i és completament responsiu.
  • Estructura bàsica: 
    <div class="container">
  <div class="row">
    <div class="col">
      Columna 1
    </div>
    <div class="col">
      Columna 2
    </div>
    <div class="col">
      Columna 3
    </div>
  </div>
</div>
  • Columnes responsives: 
    <div class="container">
  <div class="row">
    <div class="col-sm">
      Columna petita
    </div>
    <div class="col-md">
      Columna mitjana
    </div>
    <div class="col-lg">
      Columna gran
    </div>
  </div>
</div>

Components de Bootstrap

  • Botons: 
    <button type="button" class="btn btn-primary">Botó primari</button>
<button type="button" class="btn btn-secondary">Botó secundari</button>
  • Barres de navegació: 
    <nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#">Inici <span class="sr-only">(actual)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Característiques</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Preus</a>
      </li>
    </ul>
  </div>
</nav>
  • Targetes: 
    <div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Títol de la targeta</h5>
    <p class="card-text">Alguns exemples de text ràpid per construir sobre el títol de la targeta i formar la major part del contingut de la targeta.</p>
    <a href="#" class="btn btn-primary">Ves a algun lloc</a>
  </div>
</div>

Utilitats de Bootstrap

  • Espaiat: 
    <div class="mt-3">Marge superior 3</div>
<div class="mb-3">Marge inferior 3</div>
<div class="pt-3">Farcil superior 3</div>
<div class="pb-3">Farcil inferior 3</div>
  • Alineació de text: 
    <div class="text-left">Text alineat a l'esquerra</div>
<div class="text-center">Text alineat al centre</div>
<div class="text-right">Text alineat a la dreta</div>

Conclusió

Bootstrap és una eina essencial per al desenvolupament web modern, proporcionant un framework robust que simplifica la creació de llocs web responsius i visualment atractius. En dominar Bootstrap, pots accelerar significativament el teu procés de desenvolupament i garantir una aparença consistent entre diferents dispositius i navegadors. Comença amb els conceptes bàsics, practica amb els components i explora les utilitats per aprofitar al màxim el poder de Bootstrap en els teus projectes.
