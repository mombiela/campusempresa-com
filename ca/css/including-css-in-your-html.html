Incloure CSS en el teu HTML
Sintaxi i selectors de CSS ►

CSS (Cascading Style Sheets) és un llenguatge utilitzat per descriure la presentació d'un document escrit en HTML. Incloure CSS en el teu HTML és essencial per crear pàgines web visualment atractives. Aquesta secció cobrirà els diferents mètodes per incloure CSS en els teus documents HTML.

Mètodes per Incloure CSS

Hi ha tres maneres principals d'incloure CSS en el teu HTML:

  1. CSS en línia
  2. CSS intern
  3. CSS extern

CSS en línia

El CSS en línia s'utilitza per aplicar un estil únic a un sol element HTML. Per utilitzar CSS en línia, afegeixes l'atribut style directament a l'etiqueta HTML.

Exemple:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de CSS en línia</title>
</head>
<body>
    <h1 style="color: blue; text-align: center;">Hola, món!</h1>
</body>
</html>

  • Avantatges:

    • Ràpid i fàcil per a petits canvis.
    • Útil per a proves i depuració.

  • Desavantatges:

    • No reutilitzable.
    • Pot fer que l'HTML sigui desordenat i més difícil de mantenir.

CSS intern

El CSS intern s'utilitza per definir estils per a un sol document HTML. Inclous CSS intern a la secció <head> del fitxer HTML utilitzant l'etiqueta <style>.

Exemple:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de CSS intern</title>
    <style>
        body {
            background-color: lightgray;
        }
        h1 {
            color: blue;
            text-align: center;
        }
    </style>
</head>
<body>
    <h1>Hola, món!</h1>
</body>
</html>

  • Avantatges:

    • Els estils estan centralitzats en un sol lloc dins del document.
    • Més fàcil de gestionar que els estils en línia per a documents més grans.

  • Desavantatges:

    • Els estils no són reutilitzables en múltiples documents.
    • Pot fer que el fitxer HTML sigui gran i més difícil de llegir.

CSS extern

El CSS extern s'utilitza per aplicar estils a múltiples documents HTML. Crees un fitxer CSS separat i l'enllaces als teus fitxers HTML utilitzant l'etiqueta <link>.

Exemple:

Fitxer HTML (index.html):

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de CSS extern</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Hola, món!</h1>
</body>
</html>

Fitxer CSS (styles.css):

body {
    background-color: lightgray;
}

h1 {
    color: blue;
    text-align: center;
}

  • Avantatges:

    • Els estils són reutilitzables en múltiples documents HTML.
    • Manté els fitxers HTML nets i fàcils de llegir.
    • Més fàcil de mantenir i actualitzar els estils.

  • Desavantatges:

    • Requereix una sol·licitud HTTP addicional per carregar el fitxer CSS.
    • Configuració lleugerament més complexa en comparació amb el CSS en línia i intern.

Comparació dels Mètodes d'Inclusió de CSS

| Mètode | Reutilitzabilitat | Facilitat de Manteniment | Mida del Fitxer HTML | Cas d'Ús | |--------------|-------------------|--------------------------|----------------------|-----------------------------------| | CSS en línia | No | Baixa | Més gran | Correccions ràpides, proves, depuració | | CSS intern | No | Mitjana | Mitjana | Estils per a una sola pàgina | | CSS extern | Sí | Alta | Més petita | Estils per a múltiples pàgines, projectes grans |

Conclusió

Entendre els diferents mètodes per incloure CSS en el teu HTML és crucial per a un desenvolupament web efectiu. Cada mètode té els seus propis avantatges i desavantatges, i l'elecció del mètode depèn dels requisits específics del teu projecte. Per a petits canvis ràpids, el CSS en línia pot ser suficient. Per a aplicacions d'una sola pàgina, el CSS intern pot ser útil. No obstant això, per a projectes més grans i llocs web de múltiples pàgines, el CSS extern és l'enfocament més eficient i fàcil de mantenir.
