La propietat display en CSS és una de les propietats més fonamentals i poderoses. Determina com es mostra un element a la pàgina web. Entendre la propietat display és crucial per crear dissenys web ben estructurats i visualment atractius.

Conceptes Clau

  • Elements de nivell de bloc: Aquests elements ocupen tota l'amplada disponible i comencen en una nova línia.
  • Elements en línia: Aquests elements ocupen només l'amplada necessària i no comencen en una nova línia.
  • Elements en línia-bloc: Aquests elements són com els elements en línia però poden tenir amplada i alçada establertes.
  • Cap: Aquest valor amaga l'element; no es mostrarà a la pàgina.

Valors Comuns de Display

Bloc

Els elements de nivell de bloc ocupen tota l'amplada del seu contenidor i comencen en una nova línia. Exemples inclouen <div>, <h1>, <p>, etc.

<div style="display: block;">
  Aquest és un element de nivell de bloc.
</div>

En línia

Els elements en línia ocupen només l'amplada necessària i no comencen en una nova línia. Exemples inclouen <span>, <a>, <strong>, etc.

<span style="display: inline;">
  Aquest és un element en línia.
</span>

En línia-bloc

Els elements en línia-bloc són similars als elements en línia però poden tenir amplada i alçada establertes.

<div style="display: inline-block; width: 100px; height: 50px; background-color: lightblue;">
  Aquest és un element en línia-bloc.
</div>

Cap

El valor none amaga l'element; no es mostrarà a la pàgina.

<div style="display: none;">
  Aquest element està amagat.
</div>

Valors Avançats de Display

Flex

El valor flex habilita un contenidor flex, permetent dissenys més complexos amb disseny flexible i responsiu.

<div style="display: flex;">
  <div style="flex: 1;">Element Flex 1</div>
  <div style="flex: 2;">Element Flex 2</div>
</div>

Grid

El valor grid habilita un contenidor de graella, proporcionant un sistema poderós per crear dissenys complexos amb files i columnes.

<div style="display: grid; grid-template-columns: 1fr 1fr;">
  <div>Element de Graella 1</div>
  <div>Element de Graella 2</div>
</div>

Taula Comparativa

| Valor de Display | Descripció | Pot Establir Amplada/Alçada | Comença Nova Línia | |------------------|------------|-----------------------------|--------------------| | block | Element de nivell de bloc | Sí | Sí | | inline | Element en línia | No | No | | inline-block | Element en línia-bloc | Sí | No | | none | Amaga l'element | N/A | N/A | | flex | Contenidor flex | Sí | No | | grid | Contenidor de graella | Sí | No |

Exercicis

  1. Exercici Bàsic: Crea una pàgina HTML senzilla amb una barreja d'elements de bloc, en línia i en línia-bloc. Experimenta amb diferents valors de display i observa els canvis.
  2. Exercici Intermedi: Crea un contenidor flex amb tres elements. Ajusta les propietats flex per veure com afecten el disseny.
  3. Exercici Avançat: Crea un disseny de graella amb múltiples files i columnes. Experimenta amb diferents propietats de graella per entendre com funcionen conjuntament.

Conclusió

La propietat display és una pedra angular de les tècniques de disseny CSS. Dominant els seus diversos valors, des del bloc i en línia bàsics fins al flex i graella avançats, pots crear dissenys web sofisticats i responsius. Practica utilitzant aquestes propietats en diferents escenaris per convertir-te en un expert en la creació de pàgines web ben estructurades.
