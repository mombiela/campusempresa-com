El Model de Caixa de CSS és un concepte fonamental que defineix com es estructuren i es mostren els elements en una pàgina web. Entendre el Model de Caixa és crucial per a un disseny i una disposició efectius en el desenvolupament web.

Conceptes Clau

  • Contingut: El contingut real de la caixa, on apareixen el text i les imatges.
  • Emplenat: L'espai entre el contingut i la vora. L'emplenat és transparent.
  • Vora: L'àrea que envolta l'emplenat (si n'hi ha) i el contingut. Les vores poden tenir diferents estils, amplades i colors.
  • Marge: L'espai fora de la vora. Els marges també són transparents i es poden utilitzar per crear espai entre elements.

Propietats de CSS

Contingut

L'àrea de contingut es defineix per les propietats d'amplada i alçada.

div {
  width: 200px;
  height: 100px;
}

Emplenat

L'emplenat afegeix espai dins de l'element, al voltant del contingut.

div {
  padding: 20px; /* Afegeix 20px d'emplenat a tots els costats */
}

També podeu especificar l'emplenat per a cada costat individualment:

div {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;
}

Vora

Les vores envolten l'emplenat i el contingut. Podeu establir l'amplada, l'estil i el color de la vora.

div {
  border: 2px solid black; /* Vora sòlida negra de 2px d'amplada */
}

També podeu especificar les vores per a cada costat individualment:

div {
  border-top: 2px solid black;
  border-right: 2px solid red;
  border-bottom: 2px solid green;
  border-left: 2px solid blue;
}

Marge

Els marges creen espai fora de la vora. També es poden establir per a cada costat individualment.

div {
  margin: 20px; /* Afegeix 20px de marge a tots els costats */
}

Marges individuals:

div {
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;
}

Mida de la Caixa

Per defecte, l'amplada i l'alçada d'un element només inclouen l'àrea de contingut. Per incloure l'emplenat i la vora en l'amplada i l'alçada totals de l'element, utilitzeu la propietat box-sizing.

div {
  box-sizing: border-box;
}

Exemple

Aquí teniu un exemple complet que demostra el Model de Caixa:

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .box {
      width: 200px;
      height: 100px;
      padding: 20px;
      border: 5px solid black;
      margin: 30px;
      box-sizing: border-box;
    }
  </style>
</head>
<body>
  <div class="box">Aquest és un exemple del model de caixa.</div>
</body>
</html>

Resum

El Model de Caixa de CSS és essencial per entendre com es mostren i es distribueixen els elements en una pàgina web. Consisteix en el contingut, l'emplenat, la vora i el marge. Dominant el Model de Caixa, podeu crear dissenys més precisos i visualment atractius. Recordeu utilitzar la propietat box-sizing per controlar com es calculen l'amplada i l'alçada dels elements, cosa que pot simplificar el vostre disseny de disposició.
