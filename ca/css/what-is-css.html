Introducció al CSS

CSS, o Fulls d'Estil en Cascada, és un llenguatge de fulls d'estil utilitzat per descriure la presentació d'un document escrit en HTML o XML. El CSS és una tecnologia fonamental de la World Wide Web, juntament amb l'HTML i el JavaScript. Et permet controlar el disseny, els colors, les fonts i l'aparença visual general de les pàgines web.

Conceptes Clau del CSS

  • Selectors: Patrons utilitzats per seleccionar els elements que vols estilitzar.
  • Propietats: Aspectes dels elements que vols canviar (per exemple, color, mida de la font).
  • Valors: Configuracions específiques per a les propietats (per exemple, vermell, 16px).
  • Conjunts de Regles: Combinacions de selectors, propietats i valors.

Exemple d'un Conjunt de Regles CSS Simple

/* Aquest és un comentari CSS */
p {
  color: blue; /* Això estableix el color del text de tots els elements <p> a blau */
  font-size: 16px; /* Això estableix la mida de la font de tots els elements <p> a 16 píxels */
}

En aquest exemple:

  • p és el selector, que apunta a tots els elements <p>.
  • color i font-size són propietats.
  • blue i 16px són valors.

Com Funciona el CSS

El CSS funciona associant regles amb elements HTML. Aquestes regles governen com s'ha de mostrar el contingut dels elements especificats. El CSS es pot aplicar de tres maneres diferents:

  1. CSS en línia: Directament dins d'un element HTML utilitzant l'atribut style.
  2. CSS intern: Dins d'una etiqueta <style> a la secció <head> d'un document HTML.
  3. CSS extern: En un fitxer extern .css enllaçat al document HTML.

Exemple de CSS en línia

<p style="color: blue; font-size: 16px;">Aquest és un paràgraf amb CSS en línia.</p>

Exemple de CSS intern

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemple de CSS Intern</title>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>
<body>
  <p>Aquest és un paràgraf amb CSS intern.</p>
</body>
</html>

Exemple de CSS extern

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemple de CSS Extern</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>Aquest és un paràgraf amb CSS extern.</p>
</body>
</html>

En el fitxer styles.css:

p {
  color: blue;
  font-size: 16px;
}

Avantatges d'Utilitzar el CSS

  • Separació de Responsabilitats: El CSS separa el contingut (HTML) de la presentació (CSS), fent-lo més fàcil de mantenir i actualitzar.
  • Reutilització: Les regles CSS es poden reutilitzar en diverses pàgines HTML.
  • Consistència: Assegura un aspecte i sensació consistents a tot el lloc web.
  • Eficiència: Redueix la quantitat de codi duplicat i fa que el lloc web es carregui més ràpidament.

Conclusió

El CSS és una tecnologia essencial per al desenvolupament web, permetent als desenvolupadors crear llocs web visualment atractius i fàcils d'usar. Comprenent els conceptes bàsics del CSS, incloent selectors, propietats i valors, i com aplicar el CSS a través de mètodes en línia, interns i externs, pots començar a controlar la presentació de les teves pàgines web de manera efectiva. A mesura que avancis en aquest curs, aprendràs tècniques més avançades i millors pràctiques per dominar el CSS.
