Dissenys fluids ►

El disseny responsiu és un aspecte crucial del desenvolupament web modern. Assegura que les pàgines web es vegin bé i funcionin correctament en una varietat de dispositius i mides de pantalla. En CSS, les unitats responsives ajuden a aconseguir aquesta flexibilitat. Aquesta secció cobrirà els diferents tipus d'unitats responsives, el seu ús i exemples pràctics.

Conceptes Clau

  • Unitats Absolutes: Unitats fixes que no canvien amb la mida de la finestra.
    • Exemples: px (píxels), pt (punts), cm (centímetres), in (polzades)
  • Unitats Relatives: Unitats que són relatives a una altra propietat de longitud.
    • Exemples: em, rem, %, vw, vh, vmin, vmax

Unitats Absolutes

Les unitats absolutes són fixes i no es redimensionen amb la finestra. Rarament s'utilitzen en disseny responsiu perquè no s'adapten a diferents mides de pantalla.

Exemple

div {
  width: 300px;
  height: 200px;
}

En aquest exemple, el div sempre tindrà 300 píxels d'amplada i 200 píxels d'alçada, independentment de la mida de la pantalla.

Unitats Relatives

Les unitats relatives són més flexibles i s'utilitzen comunament en disseny responsiu. S'adapten segons el context en què s'utilitzen.

`em` i `rem`

  • em: Relativa a la mida de la font de l'element.
  • rem: Relativa a la mida de la font de l'element arrel (<html>).

Exemple

html {
  font-size: 16px;
}

div {
  font-size: 2em; /* 32px */
  padding: 1em; /* 16px */
}

p {
  font-size: 1.5rem; /* 24px */
}

En aquest exemple:

  • La mida de la font del div és 2 vegades la mida de la font del seu element pare.
  • La mida de la font del p és 1.5 vegades la mida de la font de l'element arrel.

Percentatge (`%`)

Les unitats de percentatge són relatives a les dimensions de l'element pare.

Exemple

div {
  width: 50%;
  height: 50%;
}

En aquest exemple, el div ocuparà el 50% de l'amplada i l'alçada del seu element pare.

Unitats de la Finestra (`vw`, `vh`, `vmin`, `vmax`)

  • vw: 1% de l'amplada de la finestra.
  • vh: 1% de l'alçada de la finestra.
  • vmin: 1% de la dimensió més petita de la finestra (amplada o alçada).
  • vmax: 1% de la dimensió més gran de la finestra (amplada o alçada).

Exemple

div {
  width: 50vw; /* 50% de l'amplada de la finestra */
  height: 50vh; /* 50% de l'alçada de la finestra */
}

En aquest exemple, el div sempre tindrà la meitat de l'amplada i l'alçada de la finestra.

Taula Comparativa

| Unitat | Descripció | Exemple d'Ús | |--------|-------------------------------------|--------------------------------| | px | Píxels (unitat absoluta) | width: 300px; | | em | Relativa a la mida de la font | font-size: 2em; | | rem | Relativa a la mida de la font arrel | font-size: 1.5rem; | | % | Percentatge de l'element pare | width: 50%; | | vw | 1% de l'amplada de la finestra | width: 50vw; | | vh | 1% de l'alçada de la finestra | height: 50vh; | | vmin | 1% de la dimensió més petita | width: 50vmin; | | vmax | 1% de la dimensió més gran | width: 50vmax; |

Exercicis Pràctics

  1. Exercici 1: Crea un div responsiu que sempre sigui el 50% de l'amplada i l'alçada de la finestra.
  2. Exercici 2: Utilitza unitats em i rem per establir la mida de la font d'un paràgraf dins d'un div de manera que es redimensioni amb la mida de la font arrel.
  3. Exercici 3: Crea un disseny on un div ocupi el 75% de l'amplada de l'element pare i el 50% de la seva alçada.

Solucions d'Exemple

Solució de l'Exercici 1:

div {
  width: 50vw;
  height: 50vh;
}

Solució de l'Exercici 2:

html {
  font-size: 16px;
}

div {
  font-size: 2em; /* 32px */
}

p {
  font-size: 1.5rem; /* 24px */
}

Solució de l'Exercici 3:

.parent {
  width: 100%;
  height: 100%;
}

.child {
  width: 75%;
  height: 50%;
}

Conclusió

Entendre i utilitzar unitats responsives en CSS és essencial per crear dissenys web flexibles i adaptatius. Dominant unitats com em, rem, %, vw i vh, pots assegurar-te que les teves pàgines web es vegin genials en qualsevol dispositiu. Practica utilitzant aquestes unitats en diferents escenaris per convertir-te en un expert en disseny responsiu.
