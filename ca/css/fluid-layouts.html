Els dissenys fluids són un concepte fonamental en el disseny web modern, permetent que les pàgines web s'adaptin a diferents mides de pantalla i dispositius. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades per crear dissenys fluids utilitzant CSS.

Introducció als Dissenys Fluids

Els dissenys fluids, també coneguts com a dissenys líquids, estan dissenyats per estirar-se i contraure's a mesura que la finestra del navegador canvia de mida. A diferència dels dissenys fixos, que tenen una amplada establerta, els dissenys fluids utilitzen unitats relatives com percentatges per crear un disseny més flexible i responsiu.

Conceptes Clau

  • Unitats Relatives: Unitats com percentatges (%) i amplada de la finestra (vw) i alçada (vh) s'utilitzen en lloc d'unitats fixes com píxels (px).
  • Consultes de Mitjans: Regles CSS que apliquen estils basats en la mida de la finestra, permetent diferents dissenys en diferents dispositius.
  • Flexbox i Grid: Mòduls de disseny CSS moderns que proporcionen més control sobre els dissenys fluids.

Disseny Fluid Bàsic amb Percentatges

Utilitzar percentatges és la manera més senzilla de crear un disseny fluid. Aquí tens un exemple d'un disseny fluid bàsic utilitzant percentatges:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple de Disseny Fluid</title>
    <style>
        body {
            margin: 0;
            padding: 0;
        }
        .container {
            width: 100%;
            padding: 10px;
        }
        .header, .footer {
            background-color: #f1f1f1;
            text-align: center;
            padding: 20px;
        }
        .content {
            background-color: #e2e2e2;
            padding: 20px;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Capçalera</div>
        <div class="content">Contingut</div>
        <div class="footer">Peu de pàgina</div>
    </div>
</body>
</html>

Explicació

  • La classe .container té una amplada del 100%, fent que s'estiri a tota l'amplada de la finestra.
  • Les classes .header, .content, i .footer tenen padding i colors de fons per distingir-les visualment.

Dissenys Fluids Avançats amb Flexbox

Flexbox proporciona una manera més sofisticada de crear dissenys fluids. Permet disposicions més complexes i millor control sobre l'alineació i distribució de l'espai.

Exemple: Disseny Fluid amb Flexbox

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Disseny Fluid amb Flexbox</title>
    <style>
        body {
            margin: 0;
            padding: 0;
            display: flex;
            flex-direction: column;
            min-height: 100vh;
        }
        .header, .footer {
            background-color: #f1f1f1;
            text-align: center;
            padding: 20px;
        }
        .content {
            background-color: #e2e2e2;
            flex: 1;
            padding: 20px;
        }
    </style>
</head>
<body>
    <div class="header">Capçalera</div>
    <div class="content">Contingut</div>
    <div class="footer">Peu de pàgina</div>
</body>
</html>

Explicació

  • L'element body està configurat amb display: flex i flex-direction: column per crear un disseny vertical.
  • La classe .content té flex: 1, fent que s'expandeixi per omplir l'espai disponible entre la capçalera i el peu de pàgina.

Dissenys Fluids Responsius amb Consultes de Mitjans

Les consultes de mitjans permeten aplicar diferents estils basats en la mida de la finestra, fent que el teu disseny fluid sigui responsiu.

Exemple: Disseny Fluid Responsiu

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Disseny Fluid Responsiu</title>
    <style>
        body {
            margin: 0;
            padding: 0;
        }
        .container {
            width: 100%;
            padding: 10px;
        }
        .header, .footer {
            background-color: #f1f1f1;
            text-align: center;
            padding: 20px;
        }
        .content {
            background-color: #e2e2e2;
            padding: 20px;
        }
        @media (min-width: 600px) {
            .container {
                display: flex;
                flex-direction: row;
            }
            .content {
                flex: 1;
                padding: 20px;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="header">Capçalera</div>
        <div class="content">Contingut</div>
        <div class="footer">Peu de pàgina</div>
    </div>
</body>
</html>

Explicació

  • La consulta de mitjans @media (min-width: 600px) aplica estils quan la finestra té almenys 600px d'amplada.
  • Dins de la consulta de mitjans, la classe .container està configurada amb display: flex i flex-direction: row, canviant el disseny de vertical a horitzontal.

Conclusió

Els dissenys fluids són essencials per crear dissenys web responsius i adaptables. Utilitzant unitats relatives, Flexbox i consultes de mitjans, pots assegurar-te que les teves pàgines web es vegin bé en qualsevol dispositiu. Practica creant dissenys fluids amb diferents tècniques per dominar aquest aspecte crucial del desenvolupament web modern.
