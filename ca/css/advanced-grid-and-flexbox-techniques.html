Introducció

En aquesta secció, aprofundirem en tècniques avançades per utilitzar CSS Grid i Flexbox. Aquests poderosos sistemes de disseny poden ajudar-te a crear dissenys web complexos i responsius amb facilitat. Al final d'aquesta secció, seràs capaç d'aprofitar les funcions avançades de Grid i Flexbox per construir dissenys sofisticats.

Tècniques Avançades de Grid

Àrees de Plantilla de Grid

  • Concepte: Defineix àrees de grid amb nom per simplificar la gestió del disseny.
  • Exemple: 
    .container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
    grid-template-rows: auto 1fr auto;
    grid-template-columns: 200px 1fr;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Col·locació Automàtica de Grid

  • Concepte: Permet que el grid col·loqui automàticament els elements basant-se en l'estructura de grid definida.
  • Exemple: 
    .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: minmax(100px, auto);
}

.item {
    background: lightblue;
    border: 1px solid #333;
}

Alineació de Grid

  • Concepte: Alinea els elements del grid utilitzant propietats com align-items, justify-items, align-self i justify-self.
  • Exemple: 
    .container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    align-items: center;
    justify-items: center;
}

.item {
    background: lightcoral;
    border: 1px solid #333;
}

Grids Imbricats

  • Concepte: Crea grids dins d'elements de grid per a dissenys més complexos.
  • Exemple: 
    .outer-container {
    display: grid;
    grid-template-columns: 1fr 1fr;
}

.inner-container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
}

.item {
    background: lightgreen;
    border: 1px solid #333;
}

Tècniques Avançades de Flexbox

Alineació de Flexbox

  • Concepte: Utilitza align-items, justify-content, align-self i justify-self per controlar l'alineació dels elements flex.
  • Exemple: 
    .container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    background: lightblue;
    border: 1px solid #333;
}

Ordre de Flexbox

  • Concepte: Canvia l'ordre dels elements flex utilitzant la propietat order.
  • Exemple: 
    .container {
    display: flex;
}

.item1 {
    order: 2;
}

.item2 {
    order: 1;
}

.item3 {
    order: 3;
}

Envolupament de Flexbox

  • Concepte: Permet que els elements flex es distribueixin en múltiples línies utilitzant la propietat flex-wrap.
  • Exemple: 
    .container {
    display: flex;
    flex-wrap: wrap;
}

.item {
    background: lightcoral;
    border: 1px solid #333;
    flex: 1 1 200px;
}

Flexbox Imbricat

  • Concepte: Crea contenidors flex dins d'elements flex per a dissenys més complexos.
  • Exemple: 
    .outer-container {
    display: flex;
}

.inner-container {
    display: flex;
    flex-direction: column;
}

.item {
    background: lightgreen;
    border: 1px solid #333;
}

Comparació de Grid i Flexbox

| Característica | CSS Grid | Flexbox | |------------------------|----------------------------------|----------------------------------| | Tipus de Disseny | Bidimensional | Unidimensional | | Alineació | align-items, justify-items | align-items, justify-content | | Col·locació d'Elements | Línies i àrees de grid | Propietat d'ordre | | Envolupament | No aplicable | flex-wrap | | Cas d'Ús | Dissenys complexos | Dissenys simples i lineals |

Conclusió

En dominar les tècniques avançades de Grid i Flexbox, pots crear dissenys web altament responsius i complexos. Grid és ideal per a dissenys bidimensionals, mentre que Flexbox excel·leix en dissenys unidimensionals. Entendre quan i com utilitzar aquestes eines millorarà enormement les teves habilitats en CSS i et permetrà construir dissenys web sofisticats.
