Introducció a les Variables CSS

Les Variables CSS, també conegudes com a Propietats Personalitzades, et permeten emmagatzemar valors que pots reutilitzar al llarg de la teva fulla d'estil. Són una característica poderosa que pot fer que el teu CSS sigui més mantenible i flexible.

Conceptes Clau

  • Definició: Les Variables CSS són entitats definides pels autors de CSS que contenen valors específics per ser reutilitzats al llarg d'un document.
  • Sintaxi: Les Variables CSS es defineixen utilitzant la notació de propietat personalitzada (--nom-propietat) i s'accedeix a elles utilitzant la funció var().
  • Abast: Les variables poden tenir un abast global (disponibles a tot el document) o local (disponibles dins d'un selector específic).

Definició i Ús de les Variables CSS

Definició de Variables

Les Variables CSS es defineixen dins d'una regla CSS. Normalment es defineixen en el selector :root per a un abast global.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
  --padding: 10px;
}

Ús de Variables

Per utilitzar una Variable CSS, utilitzes la funció var().

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
  padding: var(--padding);
}

Exemple

Aquí tens un exemple senzill que demostra la definició i l'ús de Variables CSS:

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemple de Variables CSS</title>
  <style>
    :root {
      --main-bg-color: #3498db;
      --main-text-color: #ffffff;
      --padding: 10px;
    }

    body {
      background-color: var(--main-bg-color);
      color: var(--main-text-color);
      padding: var(--padding);
    }

    .container {
      background-color: var(--main-text-color);
      color: var(--main-bg-color);
      padding: var(--padding);
    }
  </style>
</head>
<body>
  <div class="container">
    <h1>Hola, Món!</h1>
    <p>Aquest és un exemple de Variables CSS.</p>
  </div>
</body>
</html>

Ús Avançat de les Variables CSS

Valors de Reserva

Les Variables CSS poden tenir valors de reserva, que s'utilitzen si la variable no està definida.

body {
  background-color: var(--main-bg-color, #2980b9); /* Reserva a #2980b9 si --main-bg-color no està definida */
}

Actualitzacions Dinàmiques

Les Variables CSS es poden actualitzar dinàmicament utilitzant JavaScript.

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Variables CSS Dinàmiques</title>
  <style>
    :root {
      --main-bg-color: #3498db;
    }

    body {
      background-color: var(--main-bg-color);
    }
  </style>
</head>
<body>
  <button onclick="changeColor()">Canvia el Color de Fons</button>

  <script>
    function changeColor() {
      document.documentElement.style.setProperty('--main-bg-color', '#e74c3c');
    }
  </script>
</body>
</html>

Tematització amb Variables CSS

Les Variables CSS es poden utilitzar per crear temes per a la teva aplicació.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
}

.dark-theme {
  --main-bg-color: #2c3e50;
  --main-text-color: #ecf0f1;
}

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

Per aplicar el tema fosc:

<body class="dark-theme">
  <!-- Contingut aquí -->
</body>

Conclusió

Les Variables CSS són una eina poderosa per fer que el teu CSS sigui més mantenible i flexible. Et permeten definir valors una vegada i reutilitzar-los al llarg de la teva fulla d'estil, fent més fàcil gestionar i actualitzar els teus estils. Entenent i utilitzant les Variables CSS, pots crear dissenys web més dinàmics i adaptables.
