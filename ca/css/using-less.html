Introducció a Less

Less (Leaner Style Sheets) és un pre-processador de CSS que amplia les capacitats de CSS amb comportaments dinàmics com variables, mixins, operacions i funcions. Ajuda a escriure fulls d'estil més mantenibles i reutilitzables.

Conceptes Clau

  • Variables: Emmagatzemen valors que es poden reutilitzar al llarg del full d'estil.
  • Mixins: Fragments reutilitzables de CSS que es poden incloure en altres regles.
  • Nidificació: Nidificació de selectors CSS per reflectir l'estructura HTML.
  • Operacions: Realitzen càlculs i manipulen valors directament dins del full d'estil.
  • Funcions: Funcions integrades per manipular colors, cadenes i altres valors.

Configuració de Less

Per utilitzar Less, cal instal·lar-lo i compilar els fitxers .less en fitxers .css estàndard.

Instal·lació

Pots instal·lar Less utilitzant npm (Node Package Manager):

npm install -g less

Compilant Less a CSS

Compila un fitxer Less a CSS utilitzant la línia de comandes:

lessc styles.less styles.css

Variables

Les variables en Less permeten emmagatzemar valors i reutilitzar-los al llarg del teu full d'estil.

Exemple

@primary-color: #4D926F;

header {
  color: @primary-color;
}

footer {
  color: @primary-color;
}

En aquest exemple, @primary-color es defineix una vegada i s'utilitza en diversos llocs, facilitant l'actualització de l'esquema de colors.

Mixins

Els mixins són grups reutilitzables de propietats CSS que es poden incloure en altres regles.

Exemple

.border-radius(@radius) {
  -webkit-border-radius: @radius;
     -moz-border-radius: @radius;
      -ms-border-radius: @radius;
          border-radius: @radius;
}

button {
  .border-radius(5px);
}

input {
  .border-radius(10px);
}

Aquí, el mixin .border-radius es defineix una vegada i s'utilitza tant en els elements button com input.

Nidificació

La nidificació permet nidificar els selectors CSS de manera que segueixin la mateixa jerarquia visual del teu HTML.

Exemple

nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;

    li {
      display: inline-block;

      a {
        text-decoration: none;
        color: @primary-color;
      }
    }
  }
}

Aquest exemple mostra com es poden nidificar els selectors ul, li i a dins d'un selector nav.

Operacions

Less permet realitzar operacions com suma, resta, multiplicació i divisió directament dins del teu full d'estil.

Exemple

@base: 5%;
@width: @base * 2; // 10%

.container {
  width: @width;
}

En aquest exemple, l'amplada del .container es calcula utilitzant una variable i una operació.

Funcions

Less proporciona funcions integrades per manipular valors com colors, cadenes i més.

Exemple

@base-color: #111;
@light-color: lighten(@base-color, 20%);

header {
  color: @light-color;
}

Aquí, la funció lighten s'utilitza per crear una versió més clara de @base-color.

Conclusió

Less és una eina poderosa que millora CSS afegint funcions com variables, mixins, nidificació, operacions i funcions. Aquestes funcions ajuden a escriure fulls d'estil més mantenibles, reutilitzables i eficients. En dominar Less, pots millorar significativament el teu flux de treball i la mantenibilitat del CSS.
