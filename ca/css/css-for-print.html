Introducció al CSS per a Impressió

El CSS per a impressió és una àrea especialitzada del CSS que se centra en l'estilització del contingut web per a mitjans impresos. Mentre que la major part del CSS està dissenyat per a la visualització en pantalla, els fulls d'estil per a impressió asseguren que el teu contingut es vegi bé quan es imprimeix en paper.

Conceptes Clau

  • Media Queries per a Impressió: Utilitza media queries per aplicar estils específicament per a impressió.
  • Salts de Pàgina: Controla on es trenca el contingut entre pàgines.
  • Amagar Elements: Amaga elements que són innecessaris o irrellevants en impressió.
  • Estilització per a Llegibilitat: Ajusta fonts, colors i disseny per a una millor llegibilitat en paper.

Media Queries per a Impressió

Les media queries et permeten aplicar regles CSS basades en el tipus de mitjà. Per als estils d'impressió, utilitzes la regla @media print.

@media print {
  body {
    font-size: 12pt;
    line-height: 1.5;
  }
}

En aquest exemple:

  • La regla @media print està dirigida a mitjans d'impressió.
  • La mida del text del cos es fixa en 12pt, i l'altura de línia es augmenta a 1.5 per a una millor llegibilitat.

Salts de Pàgina

Els salts de pàgina són crucials per assegurar que el contingut es divideixi lògicament entre pàgines. Utilitza les propietats break-before, break-after i break-inside.

@media print {
  h1 {
    break-before: page;
  }
  p {
    break-inside: avoid;
  }
}

En aquest exemple:

  • break-before: page assegura que cada h1 comenci en una nova pàgina.
  • break-inside: avoid evita que els paràgrafs es divideixin entre pàgines.

Amagar Elements

Alguns elements, com les barres de navegació o el contingut interactiu, són innecessaris en impressió. Utilitza la propietat display: none per amagar aquests elements.

@media print {
  .navbar, .footer, .interactive-content {
    display: none;
  }
}

En aquest exemple:

  • Les classes .navbar, .footer i .interactive-content es amaguen en la versió impresa.

Estilització per a Llegibilitat

El contingut imprès ha de ser fàcil de llegir. Ajusta fonts, colors i disseny per millorar la llegibilitat.

@media print {
  body {
    font-family: serif;
    color: black;
    background: white;
  }
  a {
    color: black;
    text-decoration: none;
  }
}

En aquest exemple:

  • La font del cos es canvia a una font serif, que generalment és més fàcil de llegir en paper.
  • El color del text es fixa en negre, i el fons es fixa en blanc.
  • Els enllaços es dissenyen per semblar text normal, ja que no són clicables en impressió.

Resum

El CSS per a impressió és essencial per assegurar que el contingut web es presenti de manera efectiva en paper. Utilitzant media queries, controlant els salts de pàgina, amagant elements innecessaris i estilitzant per a llegibilitat, pots crear pàgines web amigables per a impressió. Aquestes tècniques asseguren que el teu contingut mantingui la seva integritat i llegibilitat, ja sigui vist en una pantalla o imprès.
