Introducció a Sass

Sass (Syntactically Awesome Style Sheets) és un preprocesador de CSS que afegeix poder i elegància al llenguatge bàsic. Et permet utilitzar característiques que encara no existeixen en CSS, com ara variables, regles niades, mixins i funcions.

  • Preprocesador de CSS: Una eina que amplia CSS amb variables, regles niades i més.
  • Sintaxi: Sass té dues sintaxis: SCSS (Sassy CSS) i la sintaxi amb indentació.

Instal·lant Sass

Per començar a utilitzar Sass, necessites instal·lar-lo a la teva màquina. La manera més fàcil d'instal·lar Sass és a través de npm (Node Package Manager).

npm install -g sass

Després de la instal·lació, pots compilar Sass a CSS utilitzant la següent comanda:

sass input.scss output.css

Variables

Les variables en Sass et permeten emmagatzemar valors (com colors, fonts o qualsevol valor de CSS) en una variable, que pots reutilitzar al llarg de la teva fulla d'estils.

$primary-color: #333;

body {
  color: $primary-color;
}

Niament

Sass et permet niar els teus selectors de CSS d'una manera que segueix la mateixa jerarquia visual del teu HTML.

nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }

  li { display: inline-block; }

  a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
  }
}

Partials i Importació

Els partials en Sass s'utilitzen per crear petits fitxers CSS que pots incloure en altres fitxers Sass. Això ajuda a organitzar el teu CSS.

  • Fitxer Partial: Un fitxer Sass anomenat amb un guió baix al principi (per exemple, _partial.scss).
  • Importació de Partials: Utilitza la directiva @import per incloure partials.
// _reset.scss
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

// styles.scss
@import 'reset';

body {
  font-family: Arial, sans-serif;
}

Mixins

Els mixins et permeten crear blocs reutilitzables de CSS. Pots passar arguments als mixins per fer-los més flexibles.

@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }

Funcions

Les funcions de Sass et permeten crear peces de codi reutilitzables que retornen un valor. Són similars als mixins però s'utilitzen per a càlculs i retorn de valors.

@function calculate-rem($size) {
  @return $size / 16px * 1rem;
}

body {
  font-size: calculate-rem(16px);
}

Extensió/Herència

La directiva @extend et permet compartir un conjunt de propietats CSS d'un selector a un altre.

.message {
  border: 1px solid #ccc;
  padding: 10px;
  color: #333;
}

.success {
  @extend .message;
  border-color: green;
}

.error {
  @extend .message;
  border-color: red;
}

Directives de Control i Expressions

SassScript suporta directives de control per escriure fulles d'estil complexes.

  • @if: Declaracions condicionals.
  • @for: Iteració a través d'un rang de números.
  • @each: Iteració a través d'una llista o mapa.
  • @while: Iteració mentre una condició sigui certa.
@for $i from 1 through 3 {
  .item-#{$i} { width: 2em * $i; }
}

$colors: (primary: #333, secondary: #666);

@each $name, $color in $colors {
  .#{$name}-color { color: $color; }
}

Conclusió

Sass és una eina poderosa que millora les capacitats de CSS. Utilitzant variables, niament, mixins, funcions i directives de control, pots escriure fulles d'estil més mantenibles i escalables. A mesura que et familiaritzis amb Sass, descobriràs que millora significativament el teu flux de treball i la qualitat del teu CSS.

Recorda practicar convertint alguns dels teus projectes CSS existents a Sass i explorar les seves característiques més profundament. Feliç codificació!
