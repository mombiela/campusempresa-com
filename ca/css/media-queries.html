Introducció a les Media Queries

Les Media Queries són una part fonamental del disseny web responsiu, permetent aplicar regles CSS basades en les característiques del dispositiu que està renderitzant el contingut, com ara la seva amplada, alçada o orientació.

Conceptes Clau

  • Disseny Responsiu: Crear pàgines web que es vegin bé en tots els dispositius.
  • Viewport: L'àrea visible d'una pàgina web en un dispositiu.
  • Breakpoints: Punts específics on el disseny de la pàgina web canvia per adaptar-se a diferents mides de pantalla.

Sintaxi Bàsica

Les Media Queries utilitzen la regla @media per aplicar estils de manera condicional.

@media (condition) {
  /* Regles CSS */
}

Exemple

/* Estils per defecte */
body {
  background-color: white;
  color: black;
}

/* Estils per a pantalles més amples de 600px */
@media (min-width: 600px) {
  body {
    background-color: lightblue;
  }
}

En aquest exemple:

  • El color de fons per defecte és blanc.
  • Quan l'amplada de la pantalla és de 600px o més, el color de fons canvia a blau clar.

Característiques Comunes de les Media Queries

  • Amplada i Alçada: min-width, max-width, min-height, max-height
  • Orientació: orientation: portrait o orientation: landscape
  • Resolució: min-resolution, max-resolution

Exemple amb Múltiples Condicions

@media (min-width: 600px) and (orientation: landscape) {
  body {
    background-color: lightgreen;
  }
}

Aquesta regla s'aplica quan la pantalla té almenys 600px d'amplada i està en orientació horitzontal.

Ús de les Media Queries per al Disseny Responsiu

Breakpoints

Els breakpoints són amplades de pantalla específiques on el disseny canvia. Els breakpoints comuns inclouen:

  • 480px (dispositius petits)
  • 768px (tauletes)
  • 1024px (ordinadors de sobretaula)

Exemple amb Múltiples Breakpoints

/* Estils per defecte */
body {
  font-size: 16px;
}

/* Dispositius petits */
@media (max-width: 480px) {
  body {
    font-size: 14px;
  }
}

/* Tauletes */
@media (min-width: 481px) and (max-width: 768px) {
  body {
    font-size: 15px;
  }
}

/* Ordinadors de sobretaula */
@media (min-width: 769px) {
  body {
    font-size: 18px;
  }
}

Media Queries Avançades

Combinació de Característiques de les Media Queries

Pots combinar múltiples característiques de les media queries utilitzant operadors lògics com and, not, i only.

Exemple amb Operadors Lògics

/* Aplica estils només si el dispositiu està en mode vertical i l'amplada és almenys de 600px */
@media only screen and (min-width: 600px) and (orientation: portrait) {
  body {
    background-color: lightcoral;
  }
}

Media Queries per a Pantalles d'Alta Resolució

Les pantalles d'alta resolució, com les pantalles Retina, requereixen estils diferents per a una millor claredat.

@media (min-resolution: 2dppx) {
  body {
    font-size: 20px;
  }
}

Exercicis Pràctics

  1. Media Query Bàsica: Crea una pàgina web senzilla amb un color de fons que canviï quan l'amplada de la pantalla sigui de 600px o més.
  2. Text Responsiu: Ajusta la mida de la font d'un paràgraf basant-se en diferents amplades de pantalla (480px, 768px, 1024px).
  3. Canvi d'Orientació: Canvia el disseny d'una pàgina web quan l'orientació del dispositiu canviï entre vertical i horitzontal.

Conclusió

Les Media Queries són una eina poderosa en CSS per crear dissenys responsius que s'adaptin a diverses mides de pantalla i orientacions. Comprenent i utilitzant les media queries de manera efectiva, pots assegurar-te que les teves pàgines web proporcionin una gran experiència d'usuari en tots els dispositius. Practica amb diferents breakpoints i condicions per convertir-te en un expert en disseny web responsiu.
