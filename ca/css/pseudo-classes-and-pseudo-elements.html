Introducció

Les pseudo-classes i els pseudo-elements són eines poderoses en CSS que et permeten estilitzar elements basant-se en el seu estat o posició en l'arbre del document, sense necessitat d'afegir classes o IDs addicionals al teu HTML. Aquesta secció cobrirà els conceptes bàsics i l'ús avançat de les pseudo-classes i els pseudo-elements.

Pseudo-classes

Les pseudo-classes són paraules clau afegides als selectors que especifiquen un estat especial dels elements seleccionats. S'utilitzen per estilitzar un element quan es troba en un cert estat, com ara quan es passa el ratolí per sobre, o quan és el primer fill del seu pare.

Pseudo-classes comunes

  • :hover: S'aplica quan l'usuari designa un element (amb algun dispositiu de senyalització), però no l'activa.
  • :active: S'aplica mentre un element està sent activat per l'usuari.
  • :focus: S'aplica mentre un element té el focus (accepta esdeveniments de teclat o ratolí, o altres formes d'entrada).
  • :first-child: S'aplica al primer fill d'un element especificat.
  • :last-child: S'aplica a l'últim fill d'un element especificat.
  • :nth-child(n): S'aplica al nè fill d'un element especificat.

Exemple

/* Canvia el color de fons d'un botó quan es passa el ratolí per sobre */
button:hover {
  background-color: lightblue;
}

/* Canvia el color del primer paràgraf en un contenidor */
.container p:first-child {
  color: red;
}

/* Estilitza cada segon element de la llista */
ul li:nth-child(2n) {
  background-color: #f2f2f2;
}

Explicació

  • El selector button:hover canvia el color de fons d'un botó quan el punter del ratolí està sobre ell.
  • El selector .container p:first-child canvia el color del text del primer paràgraf dins d'un contenidor amb la classe container.
  • El selector ul li:nth-child(2n) canvia el color de fons de cada segon element de la llista en una llista desordenada.

Pseudo-elements

Els pseudo-elements són paraules clau afegides als selectors que et permeten estilitzar parts específiques d'un element. S'utilitzen per estilitzar una part d'un element, com ara la primera lletra o línia d'un paràgraf, o per inserir contingut abans o després del contingut d'un element.

Pseudo-elements comuns

  • ::before: Insereix contingut abans del contingut d'un element.
  • ::after: Insereix contingut després del contingut d'un element.
  • ::first-letter: Estilitza la primera lletra de la primera línia d'un element de bloc.
  • ::first-line: Estilitza la primera línia d'un element de bloc.

Exemple

/* Insereix contingut abans d'un paràgraf */
p::before {
  content: "Nota: ";
  font-weight: bold;
}

/* Insereix contingut després d'un paràgraf */
p::after {
  content: " [Llegeix més]";
  color: blue;
}

/* Estilitza la primera lletra d'un paràgraf */
p::first-letter {
  font-size: 2em;
  color: red;
}

/* Estilitza la primera línia d'un paràgraf */
p::first-line {
  font-weight: bold;
}

Explicació

  • El selector p::before insereix el text "Nota: " abans del contingut de cada paràgraf i el fa en negreta.
  • El selector p::after insereix el text " [Llegeix més]" després del contingut de cada paràgraf i el colora de blau.
  • El selector p::first-letter augmenta la mida de la lletra i canvia el color de la primera lletra de cada paràgraf.
  • El selector p::first-line fa que la primera línia de cada paràgraf sigui en negreta.

Combinació de Pseudo-classes i Pseudo-elements

Pots combinar pseudo-classes i pseudo-elements per crear estils més complexos.

Exemple

/* Estilitza la primera lletra d'un paràgraf quan es passa el ratolí per sobre */
p:hover::first-letter {
  color: green;
  font-size: 3em;
}

Explicació

  • El selector p:hover::first-letter canvia el color i la mida de la lletra de la primera lletra d'un paràgraf quan es passa el ratolí per sobre del paràgraf.

Conclusió

Les pseudo-classes i els pseudo-elements són eines essencials en CSS que et permeten aplicar estils basats en l'estat o parts específiques dels elements. Entendre com utilitzar-los de manera efectiva pot millorar molt la interactivitat i l'atractiu visual de les teves pàgines web. Practica utilitzant aquests selectors en diferents escenaris per convertir-te en un expert en el seu ús.
