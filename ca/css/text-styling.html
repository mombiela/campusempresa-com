Introducció

L'estilització de text és un aspecte fonamental del disseny web que millora la llegibilitat i l'atractiu visual del teu contingut. En aquesta secció, cobrirem diverses propietats de CSS que et permeten estilitzar el text de manera efectiva, des de tècniques bàsiques fins a avançades.

Estilització Bàsica del Text

Família de Fonts

  • La propietat font-family especifica la font per a un element.
  • Pots llistar múltiples fonts com a sistema de reserva.
p {
  font-family: Arial, Helvetica, sans-serif;
}

Mida de la Font

  • La propietat font-size estableix la mida del text.
  • Les unitats comunes inclouen px, em, rem i %.
h1 {
  font-size: 24px;
}

Pes de la Font

  • La propietat font-weight estableix el gruix del text.
  • Els valors van des de 100 (prim) fins a 900 (negreta), o pots utilitzar paraules clau com normal i bold.
strong {
  font-weight: bold;
}

Estil de la Font

  • La propietat font-style et permet fer el text en cursiva o oblic.
em {
  font-style: italic;
}

Color del Text

  • La propietat color estableix el color del text.
  • Pots utilitzar noms de colors, valors HEX, RGB o HSL.
a {
  color: #3498db;
}

Estilització Intermèdia del Text

Alineació del Text

  • La propietat text-align alinea el text dins del seu contenidor.
  • Els valors inclouen left, right, center i justify.
p {
  text-align: justify;
}

Decoració del Text

  • La propietat text-decoration afegeix decoracions com subratllats, sobrelínies i línies a través.
a {
  text-decoration: underline;
}

Transformació del Text

  • La propietat text-transform controla la capitalització del text.
  • Els valors inclouen uppercase, lowercase i capitalize.
h2 {
  text-transform: uppercase;
}

Alçada de la Línia

  • La propietat line-height estableix l'espai entre línies de text.
  • Es pot establir utilitzant nombres sense unitat, percentatges o valors de longitud.
p {
  line-height: 1.5;
}

Espaiat de Lletres

  • La propietat letter-spacing ajusta l'espai entre caràcters.
h1 {
  letter-spacing: 2px;
}

Espaiat de Paraules

  • La propietat word-spacing ajusta l'espai entre paraules.
p {
  word-spacing: 4px;
}

Estilització Avançada del Text

Ombra del Text

  • La propietat text-shadow afegeix ombra al text.
  • Sintaxi: text-shadow: h-shadow v-shadow blur-radius color;
h1 {
  text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

Fonts Web

  • Utilitzant @font-face per incloure fonts personalitzades.
  • Exemple d'importació d'una Font de Google.
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}

Efectes Avançats de Text

  • Utilitzant background-clip i color per a text amb degradat.
h1 {
  background: linear-gradient(to right, #30CFD0 0%, #330867 100%);
  -webkit-background-clip: text;
  color: transparent;
}

Text Responsiu

  • Utilitzant unitats vw (amplada de la finestra) per a la mida de text responsiu.
h1 {
  font-size: 5vw;
}

Conclusió

L'estilització de text en CSS va des de propietats bàsiques com la mida de la font i el color fins a tècniques avançades com les ombres de text i el text responsiu. Dominar aquestes propietats et permetrà crear pàgines web visualment atractives i llegibles. Practica utilitzant aquestes propietats en diferents combinacions per veure els seus efectes i entendre millor el seu comportament.
