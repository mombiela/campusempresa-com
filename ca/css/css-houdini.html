CSS Houdini és un conjunt d'APIs de baix nivell que donen als desenvolupadors el poder d'ampliar CSS connectant-se al procés d'estil i disseny del motor de renderització del navegador. Això permet la creació de propietats CSS personalitzades, algoritmes de disseny personalitzats i més. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de CSS Houdini.

Introducció a CSS Houdini

  • Què és CSS Houdini?

    • Una col·lecció d'APIs que permeten als desenvolupadors escriure codi que pot interactuar directament amb el Model d'Objectes CSS (CSSOM).
    • Permet la creació de característiques i propietats CSS personalitzades.

  • Per què utilitzar CSS Houdini?

    • Proporciona més control sobre el procés d'estil i disseny.
    • Permet optimitzacions de rendiment reduint la necessitat de JavaScript per a certes tasques.
    • Permet la creació de característiques CSS reutilitzables i compartibles.

Conceptes Clau de CSS Houdini

  • Worklets

    • Petits scripts que s'executen en un fil separat de l'execució principal de JavaScript.
    • Tipus de worklets inclouen Paint Worklet, Layout Worklet, Animation Worklet, i més.

  • API de Propietats i Valors CSS

    • Permet la definició de propietats personalitzades i els seus tipus.
    • Exemple: 
      CSS.registerProperty({
  name: '--my-custom-property',
  syntax: '<color>',
  inherits: false,
  initialValue: 'black'
});

  • API de Pintura

    • Permet la creació d'efectes de pintura personalitzats utilitzant el Paint Worklet.
    • Exemple: 
      if ('paintWorklet' in CSS) {
  CSS.paintWorklet.addModule('paint.js');
}

  • API de Disseny

    • Permet la creació d'algoritmes de disseny personalitzats.
    • Exemple: 
      if ('layoutWorklet' in CSS) {
  CSS.layoutWorklet.addModule('layout.js');
}

Començant amb CSS Houdini

Configurant el Teu Entorn

  • Assegura't que el teu navegador suporta les APIs de CSS Houdini.
  • Utilitza un navegador modern com Chrome o Edge amb les característiques experimentals de la plataforma web habilitades.

Exemple Bàsic: Paint Worklet Personalitzat

  1. Crea un Script de Paint Worklet (paint.js)

    class CheckerboardPainter {
  static get inputProperties() {
    return ['--checkerboard-size', '--checkerboard-color'];
  }

  paint(ctx, size, properties) {
    const checkerSize = parseInt(properties.get('--checkerboard-size').toString());
    const color = properties.get('--checkerboard-color').toString();

    ctx.fillStyle = color;
    for (let y = 0; y < size.height; y += checkerSize) {
      for (let x = 0; x < size.width; x += checkerSize) {
        if ((x / checkerSize + y / checkerSize) % 2 === 0) {
          ctx.fillRect(x, y, checkerSize, checkerSize);
        }
      }
    }
  }
}

registerPaint('checkerboard', CheckerboardPainter);

  2. Registra el Paint Worklet en el Teu HTML

    <style>
  .checkerboard {
    --checkerboard-size: 20px;
    --checkerboard-color: red;
    background: paint(checkerboard);
  }
</style>
<script>
  if ('paintWorklet' in CSS) {
    CSS.paintWorklet.addModule('paint.js');
  }
</script>

  3. Aplica la Pintura Personalitzada a un Element

    <div class="checkerboard" style="width: 200px; height: 200px;"></div>

Ús Avançat de CSS Houdini

Creant Dissenys Personalitzats amb l'API de Disseny

  1. Crea un Script de Layout Worklet (layout.js)

    class MasonryLayout {
  static get inputProperties() {
    return ['--column-count'];
  }

  *intrinsicSizes(children, edges, styleMap) {
    // Defineix mides intrínseques si cal
  }

  *layout(children, edges, constraints, styleMap) {
    const columnCount = parseInt(styleMap.get('--column-count').toString());
    const columnWidth = constraints.fixedInlineSize / columnCount;
    let yOffsets = Array(columnCount).fill(0);

    for (const child of children) {
      const column = yOffsets.indexOf(Math.min(...yOffsets));
      const x = column * columnWidth;
      const y = yOffsets[column];
      const childFragment = yield child.layoutNextFragment({
        fixedInlineSize: columnWidth
      });

      childFragment.inlineOffset = x;
      childFragment.blockOffset = y;
      yOffsets[column] += childFragment.blockSize;
    }
  }
}

registerLayout('masonry', MasonryLayout);

  2. Registra el Layout Worklet en el Teu HTML

    <style>
  .masonry {
    display: layout(masonry);
    --column-count: 3;
  }
</style>
<script>
  if ('layoutWorklet' in CSS) {
    CSS.layoutWorklet.addModule('layout.js');
  }
</script>

  3. Aplica el Disseny Personalitzat a un Element

    <div class="masonry">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <!-- Més elements -->
</div>

Conclusió

CSS Houdini obre un nou món de possibilitats per als desenvolupadors web proporcionant accés de baix nivell al motor de renderització del navegador. Entenent i utilitzant Worklets, l'API de Propietats i Valors CSS, l'API de Pintura i l'API de Disseny, els desenvolupadors poden crear experiències web altament personalitzades i amb un alt rendiment. A mesura que el suport dels navegadors continuï creixent, el potencial de CSS Houdini només s'ampliarà, convertint-lo en una eina essencial per al desenvolupament web modern.
