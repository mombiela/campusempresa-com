Les transicions i animacions CSS són eines poderoses que permeten als desenvolupadors crear experiències web dinàmiques i atractives. Aquest tema cobrirà els conceptes bàsics de les transicions i animacions CSS, progressant cap a tècniques més avançades.

Introducció a les Transicions

Les transicions CSS permeten canviar els valors de les propietats de manera suau (durant una durada determinada).

Conceptes Clau

  • Propietats de Transició: transition-property, transition-duration, transition-timing-function, transition-delay.
  • Propietat Abreujada: transition.

Exemple Bàsic

/* HTML */
<button class="btn">Hover me!</button>

/* CSS */
.btn {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.btn:hover {
  background-color: green;
}

En aquest exemple:

  • La propietat transition s'utilitza per definir l'efecte de transició.
  • Quan es passa el cursor sobre el botó, el color de fons canvia de blau a verd durant 0,3 segons.

Propietats de Transició Explicades

  • transition-property: Especifica la propietat CSS a la qual aplicar la transició.
  • transition-duration: Especifica la durada de la transició.
  • transition-timing-function: Especifica la corba de velocitat de la transició.
  • transition-delay: Especifica un retard abans que comenci la transició.

Funcions de Temporització de Transició

| Funció de Temporització | Descripció | |-------------------------|-------------| | ease | Per defecte. Comença lent, després ràpid, després acaba lent. | | linear | Mateixa velocitat de principi a fi. | | ease-in | Comença lent. | | ease-out | Acaba lent. | | ease-in-out | Comença i acaba lent. |

Introducció a les Animacions

Les animacions CSS fan possible animar transicions d'una configuració d'estil CSS a una altra.

Conceptes Clau

  • Propietats d'Animació: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, animation-play-state.
  • Fotogrames Clau: Defineixen els estils per a l'element en diversos punts durant la seqüència d'animació.

Exemple Bàsic

/* HTML */
<div class="box"></div>

/* CSS */
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation: move 2s infinite;
}

@keyframes move {
  0% { left: 0; }
  50% { left: 200px; }
  100% { left: 0; }
}

En aquest exemple:

  • La propietat animation s'utilitza per definir l'animació.
  • La regla @keyframes especifica la seqüència d'animació.

Propietats d'Animació Explicades

  • animation-name: Especifica el nom de l'animació @keyframes.
  • animation-duration: Especifica la durada de l'animació.
  • animation-timing-function: Especifica la corba de velocitat de l'animació.
  • animation-delay: Especifica un retard abans que comenci l'animació.
  • animation-iteration-count: Especifica el nombre de vegades que s'ha de reproduir l'animació.
  • animation-direction: Especifica si l'animació s'ha de reproduir en sentit invers en cicles alterns.
  • animation-fill-mode: Especifica com una animació CSS ha d'aplicar estils al seu objectiu abans i després d'executar-se.
  • animation-play-state: Especifica si l'animació està en execució o en pausa.

Exemple Avançat

/* HTML */
<div class="circle"></div>

/* CSS */
.circle {
  width: 50px;
  height: 50px;
  background-color: blue;
  border-radius: 50%;
  position: relative;
  animation: bounce 2s infinite alternate ease-in-out;
}

@keyframes bounce {
  0% { top: 0; }
  100% { top: 200px; }
}

En aquest exemple:

  • La propietat animation inclou múltiples valors: bounce (nom), 2s (durada), infinite (compte d'iteració), alternate (direcció) i ease-in-out (funció de temporització).
  • La regla @keyframes defineix l'efecte de rebot.

Conclusió

Les transicions i animacions CSS són essencials per crear pàgines web interactives i visualment atractives. Entenent i utilitzant aquestes propietats, pots millorar l'experiència de l'usuari i donar vida als teus dissenys web. Practica experimentant amb diferents propietats i fotogrames clau per veure els efectes que produeixen.
