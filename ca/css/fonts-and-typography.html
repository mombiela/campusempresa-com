La tipografia és un aspecte crucial del disseny web, i CSS proporciona una varietat de propietats per controlar l'aparença del text a les teves pàgines web. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en tipografia CSS.

Propietats bàsiques de les fonts

Família de Fonts

  • Definició: La propietat font-family especifica la font per a un element.
  • Ús: Pots llistar múltiples fonts com a sistema de reserva.
body {
    font-family: "Arial", "Helvetica", sans-serif;
}

Mida de la Font

  • Definició: La propietat font-size estableix la mida del text.
  • Unitats: Les unitats comunes inclouen px, em, rem, %, i vw.
p {
    font-size: 16px;
}

Pes de la Font

  • Definició: La propietat font-weight estableix el gruix de la font.
  • Valors: Els valors comuns inclouen normal, bold, bolder, lighter, i valors numèrics (100-900).
h1 {
    font-weight: bold;
}

Estil de la Font

  • Definició: La propietat font-style especifica l'estil de la font.
  • Valors: normal, italic, i oblique.
em {
    font-style: italic;
}

Propietats avançades de les fonts

Alçada de la Línia

  • Definició: La propietat line-height estableix l'espai entre línies de text.
  • Ús: Es pot establir utilitzant nombres sense unitat, percentatges o valors de longitud.
p {
    line-height: 1.5;
}

Transformació del Text

  • Definició: La propietat text-transform controla la capitalització del text.
  • Valors: none, capitalize, uppercase, i lowercase.
h2 {
    text-transform: uppercase;
}

Decoració del Text

  • Definició: La propietat text-decoration afegeix decoració al text.
  • Valors: none, underline, overline, line-through, i combinacions.
a {
    text-decoration: underline;
}

Espaiat de les Lletres

  • Definició: La propietat letter-spacing controla l'espai entre caràcters.
  • Ús: Es pot establir utilitzant valors de longitud.
h1 {
    letter-spacing: 2px;
}

Espaiat de les Paraules

  • Definició: La propietat word-spacing estableix l'espai entre paraules.
  • Ús: Es pot establir utilitzant valors de longitud.
p {
    word-spacing: 4px;
}

Fonts web

Ús de Google Fonts

  • Definició: Google Fonts és una biblioteca de famílies de fonts amb llicència gratuïta.
  • Ús: Importa fonts mitjançant una etiqueta <link> en HTML o utilitzant @import en CSS.
<!-- HTML -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
/* CSS */
body {
    font-family: 'Roboto', sans-serif;
}

Regla @font-face

  • Definició: La regla @font-face et permet definir fonts personalitzades.
  • Ús: Especifica el nom de la font i els fitxers font.
@font-face {
    font-family: 'MyCustomFont';
    src: url('mycustomfont.woff2') format('woff2'),
         url('mycustomfont.woff') format('woff');
}

body {
    font-family: 'MyCustomFont', sans-serif;
}

Tipografia responsiva

Tipografia fluida

  • Definició: La tipografia fluida s'ajusta suaument entre diferents mides de visualització.
  • Ús: Utilitza funcions CSS com calc() i unitats de visualització.
h1 {
    font-size: calc(1.5rem + 2vw);
}

Consultes de Mitjans

  • Definició: Les consultes de mitjans et permeten aplicar estils basats en la mida de la visualització.
  • Ús: Ajusta les mides de les fonts per a diferents dispositius.
@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

@media (min-width: 601px) {
    body {
        font-size: 16px;
    }
}

Conclusió

La tipografia en CSS és una eina poderosa que pot millorar enormement la llegibilitat i l'estètica de les teves pàgines web. Des de propietats bàsiques com font-family i font-size fins a tècniques avançades com @font-face i tipografia responsiva, dominar aquestes propietats et permetrà crear dissenys web visualment atractius i accessibles. Practica utilitzant aquestes propietats en diverses combinacions per veure els seus efectes i desenvolupar un ull crític per a una bona tipografia.
