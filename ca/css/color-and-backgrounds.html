Introducció

En aquesta secció, explorarem com utilitzar CSS per estilitzar el color i el fons dels elements HTML. Entendre aquestes propietats és essencial per crear pàgines web visualment atractives. Cobrirem propietats bàsiques de color, propietats de fons i tècniques avançades per crear fons complexos.

Propietats de Color

CSS proporciona diverses propietats per controlar el color del text i altres elements.

Propietat Color

La propietat color estableix el color del text dins d'un element.

p {
  color: blue;
}
  • Colors Hexadecimals: Representats per un # seguit de sis dígits hexadecimals. 
    p {
  color: #ff5733;
}
  • Colors RGB: Representats per rgb() seguit de tres valors separats per comes (0-255). 
    p {
  color: rgb(255, 87, 51);
}
  • Colors RGBA: Similar a RGB però inclou un valor alfa per a la transparència. 
    p {
  color: rgba(255, 87, 51, 0.5);
}
  • Colors HSL: Representats per hsl() seguit de valors de to, saturació i lluminositat. 
    p {
  color: hsl(9, 100%, 60%);
}
  • Colors HSLA: Similar a HSL però inclou un valor alfa per a la transparència. 
    p {
  color: hsla(9, 100%, 60%, 0.5);
}

Color de Fons

La propietat background-color estableix el color de fons d'un element.

div {
  background-color: yellow;
}
  • Exemple amb Color Hexadecimal: 
    div {
  background-color: #ff5733;
}
  • Exemple amb Color RGB: 
    div {
  background-color: rgb(255, 87, 51);
}
  • Exemple amb Color RGBA: 
    div {
  background-color: rgba(255, 87, 51, 0.5);
}

Propietats de Fons

CSS ofereix una varietat de propietats per controlar el fons dels elements.

Imatge de Fons

La propietat background-image estableix una imatge com a fons d'un element.

body {
  background-image: url('background.jpg');
}

Repetició de Fons

La propietat background-repeat controla com es repeteix la imatge de fons.

  • repeat: La imatge de fons es repeteix tant verticalment com horitzontalment. 
    body {
  background-repeat: repeat;
}
  • no-repeat: La imatge de fons no es repeteix. 
    body {
  background-repeat: no-repeat;
}
  • repeat-x: La imatge de fons es repeteix només horitzontalment. 
    body {
  background-repeat: repeat-x;
}
  • repeat-y: La imatge de fons es repeteix només verticalment. 
    body {
  background-repeat: repeat-y;
}

Posició de Fons

La propietat background-position estableix la posició inicial de la imatge de fons.

body {
  background-position: center;
}
  • Part Superior Esquerra: 
    body {
  background-position: top left;
}
  • Part Inferior Dreta: 
    body {
  background-position: bottom right;
}

Mida de Fons

La propietat background-size especifica la mida de la imatge de fons.

  • Cover: Escala la imatge per cobrir tot l'element. 
    body {
  background-size: cover;
}
  • Contain: Escala la imatge per ser completament visible dins de l'element. 
    body {
  background-size: contain;
}

Adjunció de Fons

La propietat background-attachment estableix si una imatge de fons és fixa o es desplaça amb la resta de la pàgina.

  • Scroll: La imatge de fons es desplaça amb la pàgina. 
    body {
  background-attachment: scroll;
}
  • Fixed: La imatge de fons és fixa i no es desplaça. 
    body {
  background-attachment: fixed;
}

Abreviatura de Fons

La propietat abreujada background permet establir totes les propietats de fons en una sola declaració.

body {
  background: url('background.jpg') no-repeat center/cover fixed;
}

Tècniques Avançades

Gradients

Els gradients CSS permeten crear transicions suaus entre dos o més colors.

  • Gradient Lineal: 
    div {
  background: linear-gradient(to right, red, yellow);
}
  • Gradient Radial: 
    div {
  background: radial-gradient(circle, red, yellow);
}

Fons Múltiples

Pots aplicar múltiples imatges de fons a un element.

div {
  background: url('image1.jpg'), url('image2.png');
  background-position: left top, right bottom;
  background-repeat: no-repeat, no-repeat;
}

Conclusió

Entendre com manipular colors i fons en CSS és crucial per crear pàgines web visualment atractives. Des de propietats bàsiques de color fins a tècniques avançades de fons, dominar aquestes habilitats millorarà significativament les teves capacitats de disseny web. Practica utilitzant aquestes propietats i experimenta amb diferents combinacions per veure què funciona millor per als teus projectes.
