Introducció

En CSS, les marges, el farciment i les fronteres són propietats essencials utilitzades per crear espai i definir la disposició dels elements en una pàgina web. Entendre com manipular aquestes propietats et permet controlar l'espai al voltant i dins dels elements, millorant el disseny general i l'experiència de l'usuari.

Marges

Les marges creen espai al voltant dels elements, fora de qualsevol frontera definida. Són transparents i no tenen color de fons.

Conceptes Clau

  • Propietats de Marges: margin-top, margin-right, margin-bottom, margin-left, i margin (abreviatura).
  • Marges Automàtiques: Centrar elements horitzontalment.
  • Col·lapse de Marges: Quan les marges verticals dels elements adjacents es combinen.

Exemples de Codi

/* Propietats de marges individuals */
.element {
    margin-top: 10px;
    margin-right: 15px;
    margin-bottom: 20px;
    margin-left: 25px;
}

/* Propietat abreviada de marges */
.element {
    margin: 10px 15px 20px 25px; /* dalt dreta baix esquerra */
}

/* Marge automàtic per centrar */
.centered-element {
    margin: 0 auto;
}

Explicació

  • El primer exemple estableix diferents marges per a cada costat de l'element.
  • El segon exemple utilitza l'abreviatura per aconseguir el mateix resultat.
  • El tercer exemple centra un element horitzontalment dins del seu contenidor utilitzant margin: 0 auto;.

Farciment

El farciment crea espai dins d'un element, entre el contingut i la frontera. També és transparent però pot afectar el color de fons de l'element.

Conceptes Clau

  • Propietats de Farciment: padding-top, padding-right, padding-bottom, padding-left, i padding (abreviatura).
  • Impacte del Model de Caixa: El farciment augmenta la mida de la caixa de l'element.

Exemples de Codi

/* Propietats de farciment individuals */
.element {
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 20px;
    padding-left: 25px;
}

/* Propietat abreviada de farciment */
.element {
    padding: 10px 15px 20px 25px; /* dalt dreta baix esquerra */
}

Explicació

  • El primer exemple estableix diferents valors de farciment per a cada costat de l'element.
  • El segon exemple utilitza l'abreviatura per aconseguir el mateix resultat.

Fronteres

Les fronteres són línies que envolten el farciment i el contingut d'un element. Poden tenir diversos estils, amplades i colors.

Conceptes Clau

  • Propietats de Frontera: border-width, border-style, border-color, i border (abreviatura).
  • Fronteres Individuals: border-top, border-right, border-bottom, border-left.
  • Radi de Frontera: Arrodonir les cantonades de les fronteres.

Exemples de Codi

/* Propietats de frontera individuals */
.element {
    border-width: 2px;
    border-style: solid;
    border-color: #000;
}

/* Propietat abreviada de frontera */
.element {
    border: 2px solid #000;
}

/* Fronteres de costat individual */
.element {
    border-top: 2px solid #000;
    border-right: 2px dashed #000;
    border-bottom: 2px dotted #000;
    border-left: 2px double #000;
}

/* Radi de frontera */
.element {
    border: 2px solid #000;
    border-radius: 10px;
}

Explicació

  • El primer exemple estableix l'amplada, l'estil i el color de la frontera individualment.
  • El segon exemple utilitza l'abreviatura per aconseguir el mateix resultat.
  • El tercer exemple demostra diferents estils de frontera per a cada costat.
  • El quart exemple mostra com arrodonir les cantonades d'una frontera utilitzant border-radius.

Taula Comparativa

| Propietat | Afecta | Abreviatura | Costats Individuals | Exemple | |-----------|--------|-------------|---------------------|---------| | Marge | Fora de la frontera | margin | margin-top, margin-right, margin-bottom, margin-left | margin: 10px 15px 20px 25px; | | Farciment | Dins de la frontera | padding | padding-top, padding-right, padding-bottom, padding-left | padding: 10px 15px 20px 25px; | | Frontera | Envolta el farciment i el contingut | border | border-top, border-right, border-bottom, border-left | border: 2px solid #000; |

Conclusió

Entendre les marges, el farciment i les fronteres és crucial per crear dissenys web ben estructurats i visualment atractius. Les marges controlen l'espai fora dels elements, el farciment gestiona l'espai dins dels elements, i les fronteres defineixen els límits al voltant dels elements. Dominar aquestes propietats millorarà significativament la teva capacitat per dissenyar i disposar pàgines web de manera efectiva.
