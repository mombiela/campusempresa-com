Introducció al CSS

CSS (Fulls d'Estil en Cascada) és un llenguatge utilitzat per descriure la presentació d'un document escrit en HTML o XML. El CSS controla el disseny, els colors, les fonts i l'aparença visual general de les pàgines web.

Conceptes Clau del CSS

Selectors

Els selectors s'utilitzen per apuntar elements HTML que vols estilitzar.

  • Selector d'Element: Apunta a totes les instàncies d'un element HTML específic. 
    p {
  color: blue;
}
  • Selector de Classe: Apunta a elements amb un atribut de classe específic. 
    .example {
  color: red;
}
  • Selector d'ID: Apunta a un únic element amb un atribut d'id específic. 
    #unique {
  color: green;
}
  • Selector d'Atribut: Apunta a elements amb un atribut específic. 
    [type="text"] {
  border: 1px solid black;
}

Propietats i Valors

Les propietats CSS s'utilitzen per definir els estils aplicats als elements seleccionats.

  • Color: Defineix el color del text. 
    color: blue;
  • Font-size: Defineix la mida del text. 
    font-size: 16px;
  • Margin: Defineix l'espai fora de l'element. 
    margin: 10px;
  • Padding: Defineix l'espai dins de l'element. 
    padding: 10px;

Cascada i Especificitat

El CSS segueix un ordre en cascada per determinar quins estils s'apliquen quan múltiples regles coincideixen amb el mateix element.

  • Especificitat: Determina quina regla s'aplica donant pes a diferents tipus de selectors.
    • Els estils en línia tenen la màxima especificitat.
    • Els selectors d'ID tenen més especificitat que els selectors de classe.
    • Els selectors de classe tenen més especificitat que els selectors d'element.
  • Herència: Algunes propietats s'hereten dels elements pares als elements fills, com color i font-family.

Model de Caixa

El model de caixa del CSS descriu les caixes rectangulars generades per als elements en l'arbre del document.

  • Contingut: El contingut real de l'element.
  • Padding: Espai entre el contingut i la vora.
  • Border: La vora que envolta el padding (si n'hi ha).
  • Margin: Espai fora de la vora.
div {
  width: 100px;
  height: 100px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Posicionament

El CSS proporciona diverses maneres de posicionar elements a la pàgina web.

  • Static: Posicionament per defecte, els elements es posicionen segons el flux normal del document.
  • Relative: Posicionat relatiu a la seva posició normal. 
    .relative {
  position: relative;
  top: 10px;
  left: 10px;
}
  • Absolute: Posicionat relatiu a l'ancestre posicionat més proper. 
    .absolute {
  position: absolute;
  top: 50px;
  left: 50px;
}
  • Fixed: Posicionat relatiu a la finestra de visualització, es manté en el mateix lloc fins i tot quan es desplaça la pàgina. 
    .fixed {
  position: fixed;
  top: 0;
  left: 0;
}
  • Sticky: Canvia entre posicionament relatiu i fix depenent de la posició de desplaçament. 
    .sticky {
  position: sticky;
  top: 0;
}

Conceptes Avançats de CSS

Flexbox

Flexbox és un model de disseny que permet l'organització d'elements de manera previsible.

  • Propietats del Contenidor:
    • display: flex;
    • flex-direction: row | column;
    • justify-content: flex-start | center | space-between;
    • align-items: flex-start | center | stretch;
  • Propietats dels Elements:
    • flex-grow: 1;
    • flex-shrink: 1;
    • flex-basis: auto;
.container {
  display: flex;
  justify-content: space-between;
}

.item {
  flex-grow: 1;
}

Disseny de Graella

El Disseny de Graella CSS és un sistema de disseny bidimensional per a la web.

  • Propietats del Contenidor:
    • display: grid;
    • grid-template-columns: repeat(3, 1fr);
    • grid-template-rows: auto;
    • gap: 10px;
  • Propietats dels Elements:
    • grid-column: 1 / span 2;
    • grid-row: 1 / 2;
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  grid-column: 1 / span 2;
}

Consultes de Mitjans

Les consultes de mitjans permeten aplicar estils basats en les característiques del dispositiu, com ara la seva amplada.

@media (max-width: 600px) {
  .responsive {
    font-size: 12px;
  }
}

Conclusió

Entendre com funciona el CSS és fonamental per crear pàgines web visualment atractives i responsives. Dominant els selectors, les propietats, el model de caixa, el posicionament, Flexbox, el Disseny de Graella i les consultes de mitjans, pots controlar el disseny i la presentació dels teus projectes web de manera efectiva. Practica aquests conceptes amb exemples del món real per consolidar els teus coneixements i convertir-te en un expert en CSS.
