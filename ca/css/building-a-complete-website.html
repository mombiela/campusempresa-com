Introducció al CSS

  • Què és el CSS?
    • CSS significa Fulls d'Estil en Cascada.
    • S'utilitza per estilitzar i dissenyar pàgines web.
  • Com Funciona el CSS
    • El CSS s'utilitza per controlar la presentació dels elements HTML.
    • El CSS es pot incloure en HTML de tres maneres: Inline, Intern i Extern.

Exemple: CSS Inline

<p style="color: blue;">Aquest és un paràgraf blau.</p>

Exemple: CSS Intern

<!DOCTYPE html>
<html>
<head>
<style>
  p {
    color: blue;
  }
</style>
</head>
<body>
  <p>Aquest és un paràgraf blau.</p>
</body>
</html>

Exemple: CSS Extern

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
  <p>Aquest és un paràgraf blau.</p>
</body>
</html>
/* styles.css */
p {
  color: blue;
}

Conceptes Bàsics de CSS

  • Selectors
    • Selector d'Element
    • Selector de Classe
    • Selector d'ID
  • Propietats i Valors
    • Color
    • Font
    • Màrgin
    • Farciment

Exemple: Selectors

<!DOCTYPE html>
<html>
<head>
<style>
  /* Selector d'Element */
  p {
    color: red;
  }
  /* Selector de Classe */
  .blue-text {
    color: blue;
  }
  /* Selector d'ID */
  #green-text {
    color: green;
  }
</style>
</head>
<body>
  <p>Aquest és un paràgraf vermell.</p>
  <p class="blue-text">Aquest és un paràgraf blau.</p>
  <p id="green-text">Aquest és un paràgraf verd.</p>
</body>
</html>

Conceptes Intermedis de CSS

  • Model de Caixa
    • Contingut
    • Farciment
    • Vora
    • Màrgin
  • Posicionament
    • Estàtic
    • Relatiu
    • Absolut
    • Fix
    • Adhesiu

Exemple: Model de Caixa

<!DOCTYPE html>
<html>
<head>
<style>
  .box {
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
    background-color: lightblue;
  }
</style>
</head>
<body>
  <div class="box">Exemple de Model de Caixa</div>
</body>
</html>

Exemple: Posicionament

<!DOCTYPE html>
<html>
<head>
<style>
  .relative {
    position: relative;
    top: 20px;
    left: 20px;
    background-color: yellow;
  }
  .absolute {
    position: absolute;
    top: 50px;
    left: 50px;
    background-color: green;
  }
</style>
</head>
<body>
  <div class="relative">Posicionament Relatiu</div>
  <div class="absolute">Posicionament Absolut</div>
</body>
</html>

Conceptes Avançats de CSS

  • Flexbox
    • Contenidor Flex
    • Elements Flex
    • Propietats Flex (justify-content, align-items, etc.)
  • Disseny de Graella
    • Contenidor de Graella
    • Elements de Graella
    • Propietats de Graella (grid-template-columns, grid-template-rows, etc.)

Exemple: Flexbox

<!DOCTYPE html>
<html>
<head>
<style>
  .container {
    display: flex;
    justify-content: space-between;
  }
  .item {
    background-color: lightcoral;
    padding: 20px;
    margin: 10px;
  }
</style>
</head>
<body>
  <div class="container">
    <div class="item">Element 1</div>
    <div class="item">Element 2</div>
    <div class="item">Element 3</div>
  </div>
</body>
</html>

Exemple: Disseny de Graella

<!DOCTYPE html>
<html>
<head>
<style>
  .grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
  }
  .grid-item {
    background-color: lightgreen;
    padding: 20px;
    text-align: center;
  }
</style>
</head>
<body>
  <div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
    <div class="grid-item">4</div>
    <div class="grid-item">5</div>
    <div class="grid-item">6</div>
  </div>
</body>
</html>

Construir un Lloc Web Complet

  • Estructura del Projecte
    • Fitxer HTML
    • Fitxer CSS
    • Recursos (imatges, fonts, etc.)
  • Estilitzar el Disseny
    • Capçalera
    • Navegació
    • Contingut Principal
    • Barra Lateral
    • Peu de Pàgina
  • Disseny Responsive
    • Media Queries
    • Enfocament Mobile-First

Exemple: Estructura del Projecte

/project
  /assets
    /images
      logo.png
  index.html
  styles.css

Exemple: Lloc Web Complet

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
  <header>
    <img src="assets/images/logo.png" alt="Logo">
    <nav>
      <ul>
        <li><a href="#home">Inici</a></li>
        <li><a href="#about">Sobre Nosaltres</a></li>
        <li><a href="#contact">Contacte</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <section>
      <h1>Benvinguts al Nostre Lloc Web</h1>
      <p>Aquesta és l'àrea de contingut principal.</p>
    </section>
    <aside>
      <h2>Barra Lateral</h2>
      <p>Aquest és el contingut de la barra lateral.</p>
    </aside>
  </main>
  <footer>
    <p>&copy; 2023 El Nostre Lloc Web</p>
  </footer>
</body>
</html>
/* styles.css */
body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

header {
  background-color: #333;
  color: white;
  padding: 10px 0;
  text-align: center;
}

header img {
  width: 50px;
}

nav ul {
  list-style-type: none;
  padding: 0;
}

nav ul li {
  display: inline;
  margin: 0 10px;
}

nav ul li a {
  color: white;
  text-decoration: none;
}

main {
  display: flex;
  justify-content: space-between;
  padding: 20px;
}

section {
  flex: 3;
}

aside {
  flex: 1;
  background-color: #f4f4f4;
  padding: 10px;
}

footer {
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px 0;
  position: fixed;
  width: 100%;
  bottom: 0;
}

Conclusió

En aquest tema, hem cobert els conceptes essencials del CSS des dels bàsics fins als avançats. Hem començat amb una introducció al CSS i les seves diverses maneres d'inclusió en HTML. Després hem explorat conceptes bàsics com els selectors i les propietats, seguit de conceptes intermedis com el model de caixa i el posicionament. Finalment, hem aprofundit en conceptes avançats com Flexbox i Disseny de Graella, i hem demostrat com construir un lloc web complet amb una estructura de projecte. En dominar aquests conceptes, estaràs ben preparat per crear llocs web visualment atractius i responsius.
