Introducció a la Sintaxi CSS

CSS (Cascading Style Sheets) és un llenguatge de fulls d'estil utilitzat per descriure la presentació d'un document escrit en HTML o XML. Entendre la sintaxi bàsica del CSS és crucial per estilitzar pàgines web de manera efectiva.

Sintaxi Bàsica del CSS

  • Selector: Indica a quins elements HTML s'aplica l'estil.
  • Bloc de Declaració: Conté una o més declaracions separades per punts i coma.
  • Declaració: Consisteix en una propietat i un valor, separats per dos punts.
selector {
  property: value;
}

Exemple

p {
  color: blue;
  font-size: 16px;
}

En aquest exemple:

  • p és el selector, que apunta a tots els elements <p>.
  • color i font-size són propietats.
  • blue i 16px són els valors corresponents.

Tipus de Selectors CSS

Els selectors CSS s'utilitzen per seleccionar els elements HTML que vols estilitzar. Hi ha diversos tipus de selectors, cadascun amb el seu propi cas d'ús.

Selectors Bàsics

  • Selector Universal (*): Selecciona tots els elements.
  • Selector de Tipus: Selecciona tots els elements d'un tipus donat.
  • Selector de Classe (.classname): Selecciona tots els elements amb una classe específica.
  • Selector d'ID (#idname): Selecciona un únic element amb un ID específic.
  • Selector d'Atribut: Selecciona elements basats en un atribut o valor d'atribut.

Exemples

* {
  margin: 0;
  padding: 0;
}

h1 {
  color: red;
}

.container {
  width: 100%;
}

#header {
  background-color: #f1f1f1;
}

input[type="text"] {
  border: 1px solid #ccc;
}

Selectors de Combinació

  • Selector Descendent (ancestor descendant): Selecciona tots els descendents d'un element especificat.
  • Selector de Fill (parent > child): Selecciona tots els fills directes d'un element especificat.
  • Selector de Germà Adjacent (element + adjacent): Selecciona un element que és immediatament precedit per un element especificat.
  • Selector de Germà General (element ~ siblings): Selecciona tots els germans d'un element especificat.

Exemples

div p {
  color: green;
}

ul > li {
  list-style-type: none;
}

h1 + p {
  margin-top: 0;
}

h1 ~ p {
  color: gray;
}

Pseudo-classes i Pseudo-elements

  • Pseudo-classes: S'utilitzen per definir un estat especial d'un element.
  • Pseudo-elements: S'utilitzen per estilitzar parts específiques d'un element.

Exemples

/* Pseudo-classes */
a:hover {
  color: orange;
}

input:focus {
  border-color: blue;
}

/* Pseudo-elements */
p::first-line {
  font-weight: bold;
}

div::before {
  content: "Nota: ";
  color: red;
}

Agrupació i Nesting de Selectors

  • Agrupació de Selectors: Diversos selectors poden compartir els mateixos estils separant-los amb comes.
  • Nesting de Selectors: El CSS no suporta un nesting veritable, però pots aconseguir resultats similars utilitzant combinadors.

Exemples

/* Agrupació de Selectors */
h1, h2, h3 {
  font-family: Arial, sans-serif;
}

/* Nesting de Selectors */
nav ul {
  list-style-type: none;
}

nav ul li {
  display: inline;
}

Especificitat i Herència

  • Especificitat: Determina quins estils s'apliquen quan diverses regles podrien aplicar-se al mateix element.
  • Herència: Algunes propietats CSS s'hereten dels elements pare als fills, mentre que altres no.

Càlcul de l'Especificitat

  • Els estils en línia tenen la major especificitat.
  • Els selectors d'ID són més específics que els selectors de classe.
  • Els selectors de classe són més específics que els selectors de tipus.

Exemple

/* Exemple d'Especificitat */
#unique {
  color: red; /* Major especificitat */
}

.container {
  color: blue; /* Especificitat mitjana */
}

p {
  color: green; /* Menor especificitat */
}

Si un element <p> està dins d'un element amb la classe container i té l'ID unique, serà vermell.

Conclusió

Entendre la sintaxi i els selectors CSS és fonamental per crear pàgines web ben estilitzades. Dominant els diferents tipus de selectors i la seva especificitat, pots aplicar estils de manera precisa i eficient. Practica utilitzant aquests selectors en diverses situacions per convertir-te en un expert en CSS.

Resum

  • La sintaxi CSS consisteix en selectors i blocs de declaració.
  • Els selectors bàsics inclouen selectors universals, de tipus, de classe, d'ID i d'atribut.
  • Els selectors de combinació ajuden a apuntar elements basats en les seves relacions.
  • Les pseudo-classes i pseudo-elements proporcionen capacitats d'estil addicionals.
  • L'agrupació i el nesting de selectors poden simplificar el teu CSS.
  • L'especificitat i l'herència determinen com s'apliquen els estils.

Comprenent aquests conceptes, estaràs ben encaminat per convertir-te en un expert en CSS.
