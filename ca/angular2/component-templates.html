A Angular, un component és un bloc fonamental de l'aplicació. Cada component consisteix en una plantilla HTML que defineix la vista, una classe TypeScript que defineix el comportament, i metadades que uneixen la vista i el comportament. Aquesta secció se centra en les Plantilles de Components, que són essencials per definir la interfície d'usuari d'una aplicació Angular.

Conceptes Clau

  • Plantilla: L'HTML que defineix la vista per a un component.
  • Interpolació: Inserir expressions en la plantilla per mostrar dades dinàmiques.
  • Directives: Tokens especials en la plantilla que canvien l'aparença o el comportament d'un element.
  • Enllaç: Connectar les dades del component i la plantilla.

Crear una Plantilla de Component Bàsica

Una plantilla de component es defineix utilitzant la propietat template o templateUrl en les metadades del component. Aquí hi ha un exemple d'un component senzill:

import { Component } from '@angular/core';

@Component({
  selector: 'app-hello-world',
  template: `<h1>Hello, {{name}}!</h1>`,
})
export class HelloWorldComponent {
  name: string = 'Angular';
}

En aquest exemple:

  • El selector defineix l'etiqueta HTML personalitzada (<app-hello-world>) per a aquest component.
  • La propietat template conté la plantilla HTML.
  • La propietat name s'interpola en la plantilla utilitzant {{name}}.

Interpolació

La interpolació et permet inserir expressions en la plantilla. Aquestes expressions s'avaluen i el resultat s'insereix en el DOM.

Exemple:

@Component({
  selector: 'app-user',
  template: `<p>Welcome, {{user.name}}!</p>`,
})
export class UserComponent {
  user = { name: 'John Doe' };
}

En aquest exemple, {{user.name}} es reemplaça per John Doe en l'HTML renderitzat.

Directives

Les directives són tokens especials en la plantilla que poden canviar l'aparença o el comportament d'un element. Hi ha tres tipus de directives a Angular:

  • Directives de Component: Directives amb una plantilla.
  • Directives Estructurals: Canvien el disseny del DOM afegint o eliminant elements (per exemple, *ngIf, *ngFor).
  • Directives d'Atribut: Canvien l'aparença o el comportament d'un element (per exemple, ngClass, ngStyle).

Directives Estructurals

*ngIf

La directiva *ngIf inclou o exclou condicionalment un element en el DOM.

Exemple:

@Component({
  selector: 'app-conditional',
  template: `
    <div *ngIf="isVisible">This is visible</div>
    <button (click)="toggleVisibility()">Toggle Visibility</button>
  `,
})
export class ConditionalComponent {
  isVisible: boolean = true;

  toggleVisibility() {
    this.isVisible = !this.isVisible;
  }
}

*ngFor

La directiva *ngFor repeteix un element per a cada element d'una llista.

Exemple:

@Component({
  selector: 'app-item-list',
  template: `
    <ul>
      <li *ngFor="let item of items">{{item}}</li>
    </ul>
  `,
})
export class ItemListComponent {
  items: string[] = ['Item 1', 'Item 2', 'Item 3'];
}

Directives d'Atribut

ngClass

La directiva ngClass afegeix i elimina classes CSS.

Exemple:

@Component({
  selector: 'app-highlight',
  template: `
    <p [ngClass]="{'highlight': isHighlighted}">Highlight me!</p>
    <button (click)="toggleHighlight()">Toggle Highlight</button>
  `,
  styles: [`
    .highlight {
      background-color: yellow;
    }
  `]
})
export class HighlightComponent {
  isHighlighted: boolean = false;

  toggleHighlight() {
    this.isHighlighted = !this.isHighlighted;
  }
}

Enllaç

L'enllaç connecta les dades del component i la plantilla. Hi ha quatre tipus d'enllaç a Angular:

  • Interpolació: {{expression}}
  • Enllaç de Propietat: [property]="expression"
  • Enllaç d'Esdeveniment: (event)="handler"
  • Enllaç Bidireccional: [(ngModel)]="property"

Enllaç de Propietat

Exemple:

@Component({
  selector: 'app-image',
  template: `<img [src]="imageUrl" alt="Image">`,
})
export class ImageComponent {
  imageUrl: string = 'https://example.com/image.jpg';
}

Enllaç d'Esdeveniment

Exemple:

@Component({
  selector: 'app-click',
  template: `<button (click)="handleClick()">Click me</button>`,
})
export class ClickComponent {
  handleClick() {
    alert('Button clicked!');
  }
}

Enllaç Bidireccional

L'enllaç bidireccional combina l'enllaç de propietat i d'esdeveniment utilitzant la directiva ngModel.

Exemple:

import { Component } from '@angular/core';

@Component({
  selector: 'app-input',
  template: `
    <input [(ngModel)]="name">
    <p>Hello, {{name}}!</p>
  `,
})
export class InputComponent {
  name: string = '';
}

Conclusió

Les plantilles de components a Angular són eines poderoses per definir la interfície d'usuari de la teva aplicació. Entenent i utilitzant la interpolació, les directives i l'enllaç, pots crear aplicacions web dinàmiques i interactives. Practica aquests conceptes amb els exemples i exercicis proporcionats per consolidar la teva comprensió i construir components més complexos.
