Introducció

Els formularis basats en plantilles són una característica potent d'Angular que permet als desenvolupadors crear i gestionar formularis utilitzant la sintaxi de plantilles d'Angular. Aquest enfocament és particularment útil per a formularis més senzills i proporciona una manera més declarativa de construir formularis en comparació amb els formularis reactius.

Conceptes clau

  • FormsModule: El mòdul d'Angular que cal importar per utilitzar formularis basats en plantilles.
  • Directiva ngModel: Vincula els controls del formulari a propietats de la classe del component.
  • Validació de formularis: Validadors integrats i validadors personalitzats per garantir la integritat de les dades del formulari.
  • Enviament de formularis: Gestionar l'enviament del formulari i accedir a les dades del formulari.

Configuració dels formularis basats en plantilles

Per començar a utilitzar formularis basats en plantilles, cal importar el FormsModule al vostre mòdul d'Angular.

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { FormsModule } from '@angular/forms';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, FormsModule],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Creació d'un formulari bàsic

Aquí teniu un exemple senzill d'un formulari basat en plantilles:

Plantilla HTML

<form #userForm="ngForm" (ngSubmit)="onSubmit(userForm)">
  <div>
    <label for="name">Nom:</label>
    <input type="text" id="name" name="name" ngModel required>
  </div>
  <div>
    <label for="email">Correu electrònic:</label>
    <input type="email" id="email" name="email" ngModel required email>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Classe del component

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  onSubmit(form: any) {
    console.log('Formulari enviat!', form);
  }
}

Validació de formularis

Angular proporciona diversos validadors integrats per garantir la integritat de les dades del formulari. Aquests inclouen required, minlength, maxlength, pattern, i email.

Exemple amb validació

<form #userForm="ngForm" (ngSubmit)="onSubmit(userForm)">
  <div>
    <label for="name">Nom:</label>
    <input type="text" id="name" name="name" ngModel required minlength="3">
    <div *ngIf="userForm.controls.name?.invalid && userForm.controls.name?.touched">
      <small *ngIf="userForm.controls.name?.errors?.required">El nom és obligatori.</small>
      <small *ngIf="userForm.controls.name?.errors?.minlength">El nom ha de tenir almenys 3 caràcters.</small>
    </div>
  </div>
  <div>
    <label for="email">Correu electrònic:</label>
    <input type="email" id="email" name="email" ngModel required email>
    <div *ngIf="userForm.controls.email?.invalid && userForm.controls.email?.touched">
      <small *ngIf="userForm.controls.email?.errors?.required">El correu electrònic és obligatori.</small>
      <small *ngIf="userForm.controls.email?.errors?.email">Adreça de correu electrònic no vàlida.</small>
    </div>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Validadors personalitzats

També podeu crear validadors personalitzats per gestionar una lògica de validació més complexa.

Exemple de validador personalitzat

import { Directive } from '@angular/core';
import { NG_VALIDATORS, Validator, AbstractControl, ValidationErrors } from '@angular/forms';

@Directive({
  selector: '[appCustomValidator]',
  providers: [{ provide: NG_VALIDATORS, useExisting: CustomValidatorDirective, multi: true }]
})
export class CustomValidatorDirective implements Validator {
  validate(control: AbstractControl): ValidationErrors | null {
    const isValid = /* la vostra lògica de validació personalitzada */;
    return isValid ? null : { customError: true };
  }
}

Ús del validador personalitzat a la plantilla

<form #userForm="ngForm" (ngSubmit)="onSubmit(userForm)">
  <div>
    <label for="customField">Camp personalitzat:</label>
    <input type="text" id="customField" name="customField" ngModel appCustomValidator>
    <div *ngIf="userForm.controls.customField?.invalid && userForm.controls.customField?.touched">
      <small *ngIf="userForm.controls.customField?.errors?.customError">Error de validació personalitzada.</small>
    </div>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Enviament de formularis

Gestionar l'enviament del formulari implica capturar les dades del formulari i processar-les segons sigui necessari.

Exemple d'enviament de formulari

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  onSubmit(form: any) {
    console.log('Dades del formulari:', form.value);
    // Processar les dades del formulari aquí
  }
}

Conclusió

Els formularis basats en plantilles a Angular proporcionen una manera senzilla de crear i gestionar formularis utilitzant la sintaxi de plantilles d'Angular. Són ideals per a formularis més senzills i ofereixen funcions de validació integrades. Comprenent els conceptes clau i seguint els exemples proporcionats, podeu utilitzar eficaçment els formularis basats en plantilles a les vostres aplicacions Angular.
