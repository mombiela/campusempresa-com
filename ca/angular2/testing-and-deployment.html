Introducció

En aquesta secció, cobrirem els aspectes essencials de provar i desplegar aplicacions Angular. Les proves asseguren que la teva aplicació funcioni com s'espera, mentre que el desplegament és el procés de fer que la teva aplicació estigui disponible per als usuaris. Aquesta secció està dividida en dues parts principals: Proves i Desplegament.

Proves

Per què són importants les proves

  • Assegura la Qualitat del Codi: Ajuda a detectar errors i fallades aviat.
  • Facilita la Refactorització: Fa més fàcil canviar el codi sense trencar la funcionalitat.
  • Millora la Mantenibilitat: El codi ben provat és més fàcil de mantenir i estendre.

Tipus de Proves

  • Proves Unitàries: Prova components o serveis individuals.
  • Proves d'Integració: Prova la interacció entre diferents parts de l'aplicació.
  • Proves de Cap a Cap (E2E): Prova tot el flux de l'aplicació de principi a fi.

Configuració de l'Entorn de Proves

Angular ve amb suport integrat per a proves utilitzant Jasmine i Karma.

Instal·lació de Paquets Necessaris

npm install --save-dev jasmine-core karma karma-chrome-launcher karma-jasmine karma-jasmine-html-reporter

Escriure Proves Unitàries

Les proves unitàries se centren en provar components o serveis individuals.

Exemple: Provar un Component

import { ComponentFixture, TestBed } from '@angular/core/testing';
import { MyComponent } from './my-component.component';

describe('MyComponent', () => {
  let component: MyComponent;
  let fixture: ComponentFixture<MyComponent>;

  beforeEach(async () => {
    await TestBed.configureTestingModule({
      declarations: [ MyComponent ]
    })
    .compileComponents();
  });

  beforeEach(() => {
    fixture = TestBed.createComponent(MyComponent);
    component = fixture.componentInstance;
    fixture.detectChanges();
  });

  it('should create', () => {
    expect(component).toBeTruthy();
  });

  it('should have a default title', () => {
    expect(component.title).toBe('Default Title');
  });
});

Executar Proves

Per executar les teves proves, utilitza la següent comanda:

ng test

Escriure Proves E2E

Les proves E2E es fan servir amb Protractor.

Exemple: Prova E2E

import { browser, by, element } from 'protractor';

describe('My App', () => {
  it('should display the welcome message', () => {
    browser.get('/');
    expect(element(by.css('h1')).getText()).toEqual('Welcome to My App!');
  });
});

Executar Proves E2E

Per executar les teves proves E2E, utilitza la següent comanda:

ng e2e

Desplegament

Preparació per al Desplegament

  • Optimització de la Construcció: Utilitza Angular CLI per construir una versió optimitzada de la teva aplicació.
  • Configuració d'Entorns: Configura diferents entorns (desenvolupament, producció).

Construir l'Aplicació

Per construir la teva aplicació per a producció, utilitza la següent comanda:

ng build --prod

Opcions de Desplegament

  • Allotjament Estàtic: Desplega la teva aplicació Angular a serveis d'allotjament estàtic com GitHub Pages, Firebase Hosting, o Netlify.
  • Allotjament en Servidor: Desplega la teva aplicació a un servidor utilitzant serveis com AWS, Azure, o Heroku.

Desplegar a GitHub Pages

  1. Instal·lar Angular CLI GitHub Pages Deploy: 
    npm install -g angular-cli-ghpages
  2. Construir el Projecte: 
    ng build --prod --base-href "https://<username>.github.io/<repository-name>/"
  3. Desplegar: 
    ngh --dir=dist/<project-name>

Desplegar a Firebase Hosting

  1. Instal·lar Firebase Tools: 
    npm install -g firebase-tools
  2. Iniciar Sessió a Firebase: 
    firebase login
  3. Inicialitzar Firebase en el Teu Projecte: 
    firebase init
  4. Construir el Projecte: 
    ng build --prod
  5. Desplegar: 
    firebase deploy

Conclusió

Les proves i el desplegament són aspectes crítics per desenvolupar aplicacions Angular robustes i fiables. Escriure proves exhaustives et permet assegurar que la teva aplicació es comporti com s'espera i sigui mantenible. Desplegar correctament la teva aplicació assegura que sigui accessible per als teus usuaris. Amb les eines i tècniques cobertes en aquesta secció, estàs ben equipat per provar i desplegar les teves aplicacions Angular de manera efectiva.
