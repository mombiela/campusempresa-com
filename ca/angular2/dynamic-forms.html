Els formularis dinàmics són una característica poderosa en Angular que permet als desenvolupadors crear formularis que poden canviar dinàmicament basats en l'entrada de l'usuari o altres condicions. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades per crear i gestionar formularis dinàmics en Angular.

Introducció als Formularis Dinàmics

Els formularis dinàmics són formularis la estructura dels quals pot canviar en temps d'execució. Són útils en escenaris on els camps del formulari necessiten ser generats basats en les interaccions de l'usuari o dades obtingudes d'un servidor.

Conceptes Clau

  • Formularis Reactius: L'enfocament d'Angular per gestionar formularis utilitzant principis de programació reactiva.
  • FormBuilder: Un servei que proporciona mètodes convenients per crear controls de formulari.
  • FormGroup: Una col·lecció de controls de formulari.
  • FormArray: Una matriu dinàmica de controls de formulari.

Configuració dels Formularis Reactius

Abans d'endinsar-nos en els formularis dinàmics, és essencial entendre els conceptes bàsics dels formularis reactius en Angular.

Configuració Bàsica

  1. Instal·lar Angular CLI: Assegura't de tenir Angular CLI instal·lat. 
    npm install -g @angular/cli
  2. Crear un Nou Projecte Angular: 
    ng new dynamic-forms-demo
cd dynamic-forms-demo
  3. Instal·lar el Mòdul de Formularis Reactius: 
    ng add @angular/forms

Creant un Formulari Reactiu Simple

// app.module.ts
import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // altres imports
  ],
  // altres configuracions
})
export class AppModule { }
// app.component.ts
import { Component } from '@angular/core';
import { FormBuilder, FormGroup } from '@angular/forms';

@Component({
  selector: 'app-root',
  template: `
    <form [formGroup]="form">
      <label>
        Nom:
        <input formControlName="name">
      </label>
    </form>
  `
})
export class AppComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      name: ['']
    });
  }
}

Creant Formularis Dinàmics

Utilitzant FormArray per a Controls Dinàmics

FormArray és una característica clau per crear formularis dinàmics. Permet gestionar una matriu de controls de formulari, que poden ser afegits o eliminats dinàmicament.

Exemple: Llista Dinàmica de Correus Electrònics

// app.component.ts
import { Component } from '@angular/core';
import { FormBuilder, FormGroup, FormArray } from '@angular/forms';

@Component({
  selector: 'app-root',
  template: `
    <form [formGroup]="form">
      <div formArrayName="emails">
        <div *ngFor="let email of emails.controls; let i=index">
          <label>
            Correu Electrònic {{i + 1}}:
            <input [formControlName]="i">
          </label>
          <button (click)="removeEmail(i)">Eliminar</button>
        </div>
      </div>
      <button (click)="addEmail()">Afegir Correu Electrònic</button>
    </form>
  `
})
export class AppComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      emails: this.fb.array([])
    });
  }

  get emails() {
    return this.form.get('emails') as FormArray;
  }

  addEmail() {
    this.emails.push(this.fb.control(''));
  }

  removeEmail(index: number) {
    this.emails.removeAt(index);
  }
}

Controls de Formulari Condicionals

De vegades, necessites mostrar o amagar controls de formulari basats en certes condicions.

Exemple: Camps d'Adreça Condicionals

// app.component.ts
import { Component } from '@angular/core';
import { FormBuilder, FormGroup } from '@angular/forms';

@Component({
  selector: 'app-root',
  template: `
    <form [formGroup]="form">
      <label>
        Incloure Adreça:
        <input type="checkbox" formControlName="includeAddress">
      </label>

      <div *ngIf="form.get('includeAddress').value">
        <label>
          Adreça:
          <input formControlName="address">
        </label>
      </div>
    </form>
  `
})
export class AppComponent {
  form: FormGroup;

  constructor(private fb: FormBuilder) {
    this.form = this.fb.group({
      includeAddress: [false],
      address: ['']
    });
  }
}

Tècniques Avançades

Configuració Dinàmica del Formulari des de JSON

En alguns casos, podries voler generar un formulari basat en un objecte de configuració o dades JSON.

Exemple: Formulari des de Configuració JSON

// form-config.json
{
  "fields": [
    { "type": "text", "name": "firstName", "label": "Nom" },
    { "type": "text", "name": "lastName", "label": "Cognom" },
    { "type": "email", "name": "email", "label": "Correu Electrònic" }
  ]
}
// app.component.ts
import { Component, OnInit } from '@angular/core';
import { FormBuilder, FormGroup, FormControl } from '@angular/forms';
import formConfig from './form-config.json';

@Component({
  selector: 'app-root',
  template: `
    <form [formGroup]="form">
      <div *ngFor="let field of fields">
        <label>
          {{field.label}}:
          <input [type]="field.type" [formControlName]="field.name">
        </label>
      </div>
    </form>
  `
})
export class AppComponent implements OnInit {
  form: FormGroup;
  fields = formConfig.fields;

  constructor(private fb: FormBuilder) {}

  ngOnInit() {
    const formControls = this.fields.reduce((acc, field) => {
      acc[field.name] = new FormControl('');
      return acc;
    }, {});

    this.form = this.fb.group(formControls);
  }
}

Conclusió

Els formularis dinàmics en Angular proporcionen una manera flexible i poderosa de crear formularis que poden adaptar-se a diverses condicions i entrades de l'usuari. Aprofitant els formularis reactius, FormArray i controls condicionals, pots construir formularis complexos i dinàmics amb facilitat. Entendre aquests conceptes et permetrà crear aplicacions més interactives i fàcils d'usar.
