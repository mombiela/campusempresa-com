NgRx és una potent biblioteca de gestió d'estat per a aplicacions Angular, que proporciona una manera de gestionar l'estat utilitzant el patró Redux. Aquesta secció aprofundirà en conceptes i tècniques avançades per ajudar-te a dominar NgRx en els teus projectes Angular.

Conceptes clau

  • Selectors: Consultar i derivar l'estat de manera eficient.
  • Effects: Gestionar efectes secundaris com les sol·licituds HTTP.
  • Entity State: Gestionar col·leccions d'entitats.
  • Router State: Integrar NgRx amb Angular Router.
  • Testing NgRx: Millors pràctiques per provar la gestió d'estat.

Selectors

Els selectors són funcions pures utilitzades per obtenir fragments d'estat. Ajuden a optimitzar el rendiment memorizant l'estat.

Exemple

import { createSelector, createFeatureSelector } from '@ngrx/store';
import { AppState } from '../state/app.state';
import { UserState } from '../state/user.state';

const selectUserState = createFeatureSelector<AppState, UserState>('user');

export const selectUser = createSelector(
  selectUserState,
  (state: UserState) => state.user
);

En aquest exemple:

  • createFeatureSelector selecciona el fragment user de l'estat.
  • createSelector deriva les dades de user de UserState.

Effects

Els efectes gestionen efectes secundaris, com operacions asíncrones. Escolten accions i realitzen tasques com sol·licituds HTTP.

Exemple

import { Injectable } from '@angular/core';
import { Actions, createEffect, ofType } from '@ngrx/effects';
import { of } from 'rxjs';
import { catchError, map, mergeMap } from 'rxjs/operators';
import { UserService } from '../services/user.service';
import * as UserActions from '../actions/user.actions';

@Injectable()
export class UserEffects {
  loadUsers$ = createEffect(() =>
    this.actions$.pipe(
      ofType(UserActions.loadUsers),
      mergeMap(() =>
        this.userService.getAll().pipe(
          map(users => UserActions.loadUsersSuccess({ users })),
          catchError(error => of(UserActions.loadUsersFailure({ error })))
        )
      )
    )
  );

  constructor(
    private actions$: Actions,
    private userService: UserService
  ) {}
}

En aquest exemple:

  • ofType filtra accions de tipus loadUsers.
  • mergeMap realitza la sol·licitud HTTP.
  • map envia loadUsersSuccess en cas d'èxit.
  • catchError envia loadUsersFailure en cas d'error.

Entity State

NgRx Entity proporciona una manera de gestionar col·leccions d'entitats, reduint el codi redundant.

Exemple

import { EntityState, EntityAdapter, createEntityAdapter } from '@ngrx/entity';
import { User } from '../models/user.model';

export interface UserState extends EntityState<User> {
  selectedUserId: number | null;
}

export const adapter: EntityAdapter<User> = createEntityAdapter<User>();

export const initialState: UserState = adapter.getInitialState({
  selectedUserId: null,
});

En aquest exemple:

  • EntityState i EntityAdapter s'utilitzen per gestionar una col·lecció d'entitats User.
  • createEntityAdapter crea un adaptador per a User.
  • getInitialState inicialitza l'estat amb una col·lecció buida i un selectedUserId.

Router State

Integrar NgRx amb Angular Router et permet gestionar l'estat de les rutes dins de la botiga.

Exemple

import { routerReducer, RouterReducerState } from '@ngrx/router-store';

export interface AppState {
  router: RouterReducerState;
}

export const reducers = {
  router: routerReducer,
};

En aquest exemple:

  • routerReducer s'afegeix a l'estat arrel.
  • RouterReducerState fa un seguiment de l'estat del router.

Testing NgRx

Provar NgRx implica provar reducers, selectors i effects per assegurar-se que funcionen com s'espera.

Exemple

Provar un Reducer

import { userReducer, initialState } from '../reducers/user.reducer';
import * as UserActions from '../actions/user.actions';

describe('UserReducer', () => {
  it('hauria de retornar l'estat inicial', () => {
    const action = {} as any;
    const state = userReducer(undefined, action);

    expect(state).toBe(initialState);
  });

  it('hauria de carregar usuaris', () => {
    const users = [{ id: 1, name: 'John' }];
    const action = UserActions.loadUsersSuccess({ users });
    const state = userReducer(initialState, action);

    expect(state.entities[1]).toEqual(users[0]);
  });
});

En aquest exemple:

  • Es prova l'estat inicial.
  • Es prova l'acció loadUsersSuccess per assegurar-se que actualitza l'estat correctament.

Conclusió

Dominar conceptes avançats de NgRx com selectors, effects, entity state, router state i testing millorarà significativament la teva capacitat per gestionar l'estat en aplicacions Angular. Entenent i aplicant aquestes tècniques, podràs crear aplicacions més eficients, escalables i mantenibles.
