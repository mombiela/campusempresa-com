Introducció als Paràmetres de Ruta

Els paràmetres de ruta són una manera de passar informació a una ruta a Angular. Són essencials per crear aplicacions dinàmiques i flexibles on el mateix component pot mostrar dades diferents basades en els paràmetres de ruta.

Configuració dels Paràmetres de Ruta

Per utilitzar paràmetres de ruta, cal definir-los en la configuració de la teva ruta.

Definició dels Paràmetres de Ruta

A continuació es mostra com definir paràmetres de ruta en el teu app-routing.module.ts:

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { UserComponent } from './user/user.component';

const routes: Routes = [
  { path: 'user/:id', component: UserComponent }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

En aquest exemple:

  • path: 'user/:id' defineix una ruta amb un paràmetre id.

Accés als Paràmetres de Ruta

Un cop hagis definit els paràmetres de ruta, pots accedir-hi en el teu component.

Ús de ActivatedRoute

Per accedir als paràmetres de ruta, utilitzes el servei ActivatedRoute.

import { Component, OnInit } from '@angular/core';
import { ActivatedRoute } from '@angular/router';

@Component({
  selector: 'app-user',
  template: `<h1>User ID: {{ userId }}</h1>`
})
export class UserComponent implements OnInit {
  userId: string;

  constructor(private route: ActivatedRoute) {}

  ngOnInit(): void {
    this.userId = this.route.snapshot.paramMap.get('id');
  }
}

En aquest exemple:

  • ActivatedRoute s'injecta en el component.
  • this.route.snapshot.paramMap.get('id') recupera el paràmetre id de la ruta.

Ús dels Paràmetres de Ruta en Plantilles

Pots utilitzar els paràmetres de ruta recuperats directament en les teves plantilles.

<h1>User ID: {{ userId }}</h1>

Això mostrarà l'ID de l'usuari a la pàgina.

Reacció als Canvis de Paràmetres

De vegades, els paràmetres de ruta poden canviar mentre el component encara està actiu. Per gestionar aquests casos, cal subscriure's als canvis de paràmetres.

Subscripció als Canvis de Paràmetres

A continuació es mostra com pots subscriure't als canvis de paràmetres de ruta:

import { Component, OnInit, OnDestroy } from '@angular/core';
import { ActivatedRoute } from '@angular/router';
import { Subscription } from 'rxjs';

@Component({
  selector: 'app-user',
  template: `<h1>User ID: {{ userId }}</h1>`
})
export class UserComponent implements OnInit, OnDestroy {
  userId: string;
  private routeSub: Subscription;

  constructor(private route: ActivatedRoute) {}

  ngOnInit(): void {
    this.routeSub = this.route.paramMap.subscribe(params => {
      this.userId = params.get('id');
    });
  }

  ngOnDestroy(): void {
    this.routeSub.unsubscribe();
  }
}

En aquest exemple:

  • this.route.paramMap.subscribe s'utilitza per reaccionar als canvis de paràmetres.
  • ngOnDestroy assegura que la subscripció es neteja quan el component es destrueix.

Paràmetres de Ruta Opcional

Angular també suporta paràmetres de ruta opcionals.

Definició dels Paràmetres de Ruta Opcional

Els paràmetres opcionals es defineixen utilitzant la propietat queryParams.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { UserComponent } from './user/user.component';

const routes: Routes = [
  { path: 'user', component: UserComponent }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Accés als Paràmetres de Ruta Opcional

Pots accedir als paràmetres opcionals utilitzant la propietat queryParams de ActivatedRoute.

import { Component, OnInit } from '@angular/core';
import { ActivatedRoute } from '@angular/router';

@Component({
  selector: 'app-user',
  template: `<h1>User ID: {{ userId }}</h1>`
})
export class UserComponent implements OnInit {
  userId: string;

  constructor(private route: ActivatedRoute) {}

  ngOnInit(): void {
    this.route.queryParams.subscribe(params => {
      this.userId = params['id'];
    });
  }
}

En aquest exemple:

  • this.route.queryParams.subscribe s'utilitza per reaccionar als canvis de paràmetres opcionals.

Resum

  • Paràmetres de Ruta: Permeten passar dades dinàmiques a les rutes.
  • Definició de Paràmetres: Utilitza :param en el camí de la ruta.
  • Accés als Paràmetres: Utilitza el servei ActivatedRoute.
  • Reacció als Canvis: Subscriu-te a paramMap o queryParams.
  • Paràmetres Opcional: Accedits a través de queryParams.

Entendre i utilitzar els paràmetres de ruta de manera efectiva pot millorar significativament la flexibilitat i dinamisme de les teves aplicacions Angular.
