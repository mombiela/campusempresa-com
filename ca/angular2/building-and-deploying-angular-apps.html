Introducció

En aquesta secció, cobrirem els passos essencials i les millors pràctiques per construir i desplegar aplicacions Angular. Això inclou configurar l'entorn de desenvolupament, construir l'aplicació per a producció i desplegar-la en diverses plataformes.

Configuració de l'Entorn de Desenvolupament

Abans de començar a construir i desplegar aplicacions Angular, necessitem configurar el nostre entorn de desenvolupament.

  • Node.js i npm: Angular requereix Node.js i npm (Node Package Manager) per gestionar les dependències.
  • Angular CLI: La Interfície de Línia de Comandes d'Angular (CLI) és una eina potent que ajuda a crear, construir i gestionar aplicacions Angular.

Instal·lació de Node.js i npm

  1. Descarrega i instal·la Node.js des del lloc web oficial.
  2. Verifica la instal·lació executant les següents comandes al teu terminal: 
    node -v
npm -v

Instal·lació d'Angular CLI

  1. Instal·la Angular CLI globalment utilitzant npm: 
    npm install -g @angular/cli
  2. Verifica la instal·lació executant: 
    ng version

Creació d'una Aplicació Angular

Creem una nova aplicació Angular utilitzant Angular CLI.

  1. Obre el teu terminal i executa la següent comanda: 
    ng new my-angular-app
  2. Navega al directori del projecte: 
    cd my-angular-app
  3. Serveix l'aplicació localment: 
    ng serve
  4. Obre el teu navegador i navega a http://localhost:4200 per veure la teva aplicació en funcionament.

Construcció de l'Aplicació per a Producció

Per desplegar una aplicació Angular, necessitem construir-la per a producció. Aquest procés optimitza l'aplicació per a rendiment i crea els fitxers necessaris per al desplegament.

  1. Executa la següent comanda per construir l'aplicació: 
    ng build --prod
  2. Els artefactes de la construcció es guardaran al directori dist/.

Entenent la Sortida de la Construcció

El directori dist/ conté els següents fitxers:

  • index.html: El fitxer HTML principal.
  • main.js: El paquet principal de JavaScript.
  • polyfills.js: Polyfills per a compatibilitat amb navegadors.
  • styles.css: Els estils globals per a l'aplicació.
  • vendor.js: Biblioteques i dependències de tercers.

Desplegament de l'Aplicació

Hi ha diverses maneres de desplegar una aplicació Angular. Cobrirem el desplegament a plataformes populars com GitHub Pages, Firebase Hosting i un servidor personalitzat.

Desplegament a GitHub Pages

  1. Instal·la el desplegador de GitHub Pages per Angular CLI: 
    npm install -g angular-cli-ghpages
  2. Construeix l'aplicació: 
    ng build --prod --base-href "https://<nom-usuari>.github.io/<nom-repositori>/"
  3. Desplega l'aplicació: 
    ngh --dir=dist/<nom-projecte>

Desplegament a Firebase Hosting

  1. Instal·la Firebase CLI: 
    npm install -g firebase-tools
  2. Inicia sessió a Firebase: 
    firebase login
  3. Inicialitza Firebase en el teu projecte: 
    firebase init
  4. Construeix l'aplicació: 
    ng build --prod
  5. Desplega l'aplicació: 
    firebase deploy

Desplegament a un Servidor Personalitzat

  1. Construeix l'aplicació: 
    ng build --prod
  2. Copia el contingut del directori dist/ al directori públic del teu servidor.
  3. Configura el teu servidor per servir el fitxer index.html per a totes les rutes.

Conclusió

En aquest tema, hem cobert els passos essencials per construir i desplegar aplicacions Angular. Hem començat configurant l'entorn de desenvolupament, creant una nova aplicació Angular i construint-la per a producció. Finalment, hem explorat diferents opcions de desplegament, incloent GitHub Pages, Firebase Hosting i servidors personalitzats. Seguint aquests passos, pots assegurar-te que les teves aplicacions Angular estiguin optimitzades i llestes per al desplegament en producció.
