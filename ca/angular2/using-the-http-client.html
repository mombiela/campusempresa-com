Introducció

A Angular, el client HTTP és una eina poderosa per fer sol·licituds HTTP per comunicar-se amb serveis de back-end. Aquesta secció cobrirà com utilitzar el client HTTP en aplicacions Angular, des de configurar el client fins a fer diversos tipus de sol·licituds HTTP.

Configurant el Client HTTP

Per utilitzar el client HTTP en la teva aplicació Angular, necessites importar el HttpClientModule en el teu mòdul principal de l'aplicació.

import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';
import { HttpClientModule } from '@angular/common/http';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,
    HttpClientModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Fent Sol·licituds HTTP

Sol·licitud GET

Una sol·licitud GET s'utilitza per recuperar dades d'un servidor. Aquí tens com pots fer una sol·licitud GET utilitzant el client HTTP d'Angular.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) { }

  getData(): Observable<any> {
    return this.http.get<any>(this.apiUrl);
  }
}

Sol·licitud POST

Una sol·licitud POST s'utilitza per enviar dades a un servidor. Aquí tens un exemple de com fer una sol·licitud POST.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) { }

  postData(data: any): Observable<any> {
    return this.http.post<any>(this.apiUrl, data);
  }
}

Sol·licitud PUT

Una sol·licitud PUT s'utilitza per actualitzar dades existents en el servidor. Aquí tens un exemple de com fer una sol·licitud PUT.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) { }

  updateData(id: number, data: any): Observable<any> {
    return this.http.put<any>(`${this.apiUrl}/${id}`, data);
  }
}

Sol·licitud DELETE

Una sol·licitud DELETE s'utilitza per eliminar dades del servidor. Aquí tens un exemple de com fer una sol·licitud DELETE.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) { }

  deleteData(id: number): Observable<any> {
    return this.http.delete<any>(`${this.apiUrl}/${id}`);
  }
}

Gestionant Respostes HTTP

Utilitzant Observables

El client HTTP d'Angular retorna observables, als quals pots subscriure't per gestionar la resposta.

import { Component, OnInit } from '@angular/core';
import { DataService } from './data.service';

@Component({
  selector: 'app-data',
  template: `
    <div *ngIf="data">
      {{ data | json }}
    </div>
  `
})
export class DataComponent implements OnInit {
  data: any;

  constructor(private dataService: DataService) { }

  ngOnInit() {
    this.dataService.getData().subscribe(response => {
      this.data = response;
    });
  }
}

Gestió d'Errors

Gestionar errors és crucial per a una aplicació robusta. Pots utilitzar l'operador catchError de RxJS per gestionar errors.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';
import { Observable, throwError } from 'rxjs';
import { catchError } from 'rxjs/operators';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) { }

  getData(): Observable<any> {
    return this.http.get<any>(this.apiUrl).pipe(
      catchError(this.handleError)
    );
  }

  private handleError(error: any) {
    console.error('An error occurred', error);
    return throwError('Something went wrong; please try again later.');
  }
}

Conclusió

Utilitzant el client HTTP d'Angular, pots fer fàcilment sol·licituds HTTP per interactuar amb serveis de back-end. Aquesta secció ha cobert els conceptes bàsics de configurar el client HTTP, fer sol·licituds GET, POST, PUT i DELETE, gestionar respostes i gestionar errors. Dominant aquests conceptes, pots construir aplicacions Angular robustes que es comuniquin eficaçment amb APIs de back-end.
