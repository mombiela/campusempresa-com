Introducció als Pipes

Els pipes a Angular són una característica poderosa que et permet transformar dades directament en les teves plantilles. Es poden utilitzar per formatar dades, com ara dates, números i cadenes de text, fent que les teves plantilles siguin més netes i llegibles.

Conceptes clau

  • Pipe: Una funció que pren dades com a entrada i les transforma en una sortida desitjada.
  • Pipes integrats: Angular proporciona diversos pipes integrats per a transformacions de dades comunes.
  • Pipes personalitzats: Pots crear els teus propis pipes per a transformacions de dades personalitzades (cobert en una altra secció).

Pipes integrats comuns

Angular proporciona diversos pipes integrats que pots utilitzar de manera immediata:

  1. DatePipe: Formatja un valor de data segons les regles de la localitat.
  2. UpperCasePipe: Transforma text a majúscules.
  3. LowerCasePipe: Transforma text a minúscules.
  4. CurrencyPipe: Transforma un número en una cadena de text de moneda.
  5. DecimalPipe: Transforma un número en una cadena de text amb un format decimal especificat.
  6. PercentPipe: Transforma un número en una cadena de text de percentatge.
  7. JsonPipe: Converteix un valor en una cadena de text JSON.
  8. SlicePipe: Crea un nou array o cadena de text que conté un subconjunt (slice) dels elements.

Ús dels Pipes integrats

DatePipe

El DatePipe s'utilitza per formatar valors de data.

<p>{{ today | date }}</p>
<p>{{ today | date:'short' }}</p>
<p>{{ today | date:'fullDate' }}</p>
<p>{{ today | date:'yyyy-MM-dd' }}</p>

UpperCasePipe i LowerCasePipe

Aquests pipes transformen text a majúscules o minúscules, respectivament.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

El CurrencyPipe formatja un número com a moneda.

<p>{{ 1234.56 | currency }}</p>
<p>{{ 1234.56 | currency:'USD' }}</p>
<p>{{ 1234.56 | currency:'EUR':'symbol':'4.2-2' }}</p>

DecimalPipe

El DecimalPipe formatja un número amb un format decimal especificat.

<p>{{ 1234.56 | number }}</p>
<p>{{ 1234.56 | number:'3.1-2' }}</p>

PercentPipe

El PercentPipe formatja un número com a percentatge.

<p>{{ 0.1234 | percent }}</p>
<p>{{ 0.1234 | percent:'2.2-2' }}</p>

JsonPipe

El JsonPipe converteix un objecte en una cadena de text JSON.

<p>{{ { name: 'John', age: 30 } | json }}</p>

SlicePipe

El SlicePipe crea un nou array o cadena de text que conté un subconjunt (slice) dels elements.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

Exemples i exercicis

Exemple 1: Formatant dates

Crea un component que mostri la data actual en diversos formats utilitzant el DatePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-date-example',
  template: `
    <p>Default: {{ today | date }}</p>
    <p>Short: {{ today | date:'short' }}</p>
    <p>Full Date: {{ today | date:'fullDate' }}</p>
    <p>Custom: {{ today | date:'yyyy-MM-dd' }}</p>
  `
})
export class DateExampleComponent {
  today: number = Date.now();
}

Exercici 1: Utilitzant CurrencyPipe

Crea un component que mostri el preu d'un producte en diferents monedes utilitzant el CurrencyPipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-currency-example',
  template: `
    <p>USD: {{ price | currency:'USD' }}</p>
    <p>EUR: {{ price | currency:'EUR' }}</p>
    <p>GBP: {{ price | currency:'GBP' }}</p>
  `
})
export class CurrencyExampleComponent {
  price: number = 1234.56;
}

Exercici 2: Transformant text

Crea un component que mostri una cadena de text en majúscules i minúscules utilitzant UpperCasePipe i LowerCasePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-text-transform-example',
  template: `
    <p>Original: {{ text }}</p>
    <p>Uppercase: {{ text | uppercase }}</p>
    <p>Lowercase: {{ text | lowercase }}</p>
  `
})
export class TextTransformExampleComponent {
  text: string = 'Hello Angular';
}

Conclusió

Els pipes integrats a Angular proporcionen una manera convenient de formatar i transformar dades directament en les teves plantilles. Entenent i utilitzant aquests pipes, pots fer que les teves plantilles siguin més netes i llegibles. Practica utilitzant aquests pipes en diferents escenaris per sentir-te més còmode amb la seva sintaxi i capacitats.
