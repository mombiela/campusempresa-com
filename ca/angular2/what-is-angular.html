Què és Angular?
Configuració de l'entorn de desenvolupament ►

Angular és una plataforma i un framework per construir aplicacions de client d'una sola pàgina utilitzant HTML i TypeScript. Desenvolupat i mantingut per Google, Angular és una reescriptura completa de l'equip que va construir AngularJS. Angular 2+ es refereix a totes les versions d'Angular a partir de la versió 2 i més enllà.

Característiques Clau d'Angular

  • Arquitectura Basada en Components: Les aplicacions Angular es construeixen utilitzant components, que són els blocs fonamentals de la interfície d'usuari.
  • TypeScript: Angular està escrit en TypeScript, que és un superconjunt de JavaScript que afegeix tipus estàtics.
  • Injecció de Dependències: Angular té un sistema d'injecció de dependències potent que facilita la gestió i la injecció de dependències.
  • Programació Reactiva amb RxJS: Angular utilitza RxJS per gestionar operacions asíncrones i programes basats en esdeveniments.
  • Desenvolupament Modular: Les aplicacions Angular són modulars, permetent als desenvolupadors dividir l'aplicació en peces més petites i reutilitzables.
  • Enllaç de Dades Bidireccional: Angular suporta l'enllaç de dades bidireccional, el que significa que qualsevol canvi en el model es reflecteix en la vista i viceversa.
  • Ruting: Angular proporciona un mecanisme de ruting robust per navegar entre diferents vistes o components.

Conceptes Bàsics en Angular

Components

Els components són els blocs de construcció d'una aplicació Angular. Cada component consta de:

  • Plantilla: Defineix la vista per al component utilitzant HTML.
  • Classe: Conté la lògica i les dades per al component, escrit en TypeScript.
  • Estils: Defineix els estils CSS per al component.

Exemple d'un component Angular simple:

import { Component } from '@angular/core';

@Component({
  selector: 'app-hello-world',
  template: `<h1>Hello, World!</h1>`,
  styles: [`h1 { font-family: Lato; }`]
})
export class HelloWorldComponent {
}

Mòduls

Els mòduls s'utilitzen per agrupar components, directives, pipes i serveis relacionats. Cada aplicació Angular té almenys un mòdul, el mòdul arrel.

Exemple d'un mòdul Angular bàsic:

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';
import { HelloWorldComponent } from './hello-world.component';

@NgModule({
  declarations: [
    AppComponent,
    HelloWorldComponent
  ],
  imports: [
    BrowserModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Serveis i Injecció de Dependències

Els serveis s'utilitzen per compartir dades i lògica entre múltiples components. El sistema d'injecció de dependències d'Angular facilita la injecció de serveis en components.

Exemple d'un servei simple:

import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class DataService {
  private data: string[] = [];

  addData(item: string) {
    this.data.push(item);
  }

  getData(): string[] {
    return this.data;
  }
}

Injectant el servei en un component:

import { Component } from '@angular/core';
import { DataService } from './data.service';

@Component({
  selector: 'app-data',
  template: `
    <div>
      <input #newItem>
      <button (click)="addItem(newItem.value)">Add</button>
      <ul>
        <li *ngFor="let item of items">{{ item }}</li>
      </ul>
    </div>
  `,
})
export class DataComponent {
  items: string[] = [];

  constructor(private dataService: DataService) {}

  addItem(item: string) {
    this.dataService.addData(item);
    this.items = this.dataService.getData();
  }
}

Conclusió

Angular és un framework potent i versàtil per construir aplicacions web modernes. La seva arquitectura basada en components, l'ús de TypeScript i les seves característiques robustes com la injecció de dependències i la programació reactiva el fan una elecció popular entre els desenvolupadors. Entendre els conceptes bàsics d'Angular, com els components, mòduls i serveis, proporciona una base sòlida per construir aplicacions escalables i mantenibles. A mesura que avancis en aquest curs, adquiriràs una comprensió més profunda de les característiques avançades d'Angular i les millors pràctiques.
