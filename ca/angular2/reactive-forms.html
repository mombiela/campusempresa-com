Els formularis reactius en Angular proporcionen un enfocament basat en models per gestionar les entrades de formulari els valors dels quals canvien amb el temps. Aquest enfocament és més potent i escalable en comparació amb els formularis basats en plantilles, especialment per a formularis complexos.

Introducció als Formularis Reactius

  • Enfocament basat en models: Els formularis reactius es construeixen al voltant de fluxos observables, cosa que facilita la gestió d'interaccions complexes de formulari.
  • FormControl, FormGroup i FormArray: Blocs de construcció bàsics per crear formularis reactius.
  • Validació: Validadors integrats i personalitzats per als controls de formulari.

Configuració dels Formularis Reactius

Per utilitzar els formularis reactius, cal importar el ReactiveFormsModule al teu mòdul d'Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // altres imports
  ],
  // altres configuracions
})
export class AppModule { }

Creació d'un Formulari Reactiu Bàsic

Pas 1: Importar les Classes Necessàries

import { Component } from '@angular/core';
import { FormControl, FormGroup } from '@angular/forms';

Pas 2: Definir el Model del Formulari

@Component({
  selector: 'app-basic-form',
  template: `
    <form [formGroup]="profileForm" (ngSubmit)="onSubmit()">
      <label>
        First Name:
        <input formControlName="firstName">
      </label>
      <label>
        Last Name:
        <input formControlName="lastName">
      </label>
      <button type="submit">Submit</button>
    </form>
  `
})
export class BasicFormComponent {
  profileForm = new FormGroup({
    firstName: new FormControl(''),
    lastName: new FormControl('')
  });

  onSubmit() {
    console.log(this.profileForm.value);
  }
}

Explicació

  • FormGroup: Representa el formulari en conjunt.
  • FormControl: Representa camps individuals del formulari.
  • formControlName: Vincula el control del formulari a l'element d'entrada.

Funcionalitats Avançades del Formulari

FormGroups Imbricats

Pots imbricar instàncies de FormGroup per crear formularis complexos.

this.profileForm = new FormGroup({
  firstName: new FormControl(''),
  lastName: new FormControl(''),
  address: new FormGroup({
    street: new FormControl(''),
    city: new FormControl(''),
    state: new FormControl(''),
    zip: new FormControl('')
  })
});

FormArray

FormArray s'utilitza per gestionar una matriu de controls de formulari.

this.profileForm = new FormGroup({
  firstName: new FormControl(''),
  lastName: new FormControl(''),
  aliases: new FormArray([
    new FormControl('')
  ])
});

get aliases() {
  return this.profileForm.get('aliases') as FormArray;
}

addAlias() {
  this.aliases.push(new FormControl(''));
}

Controls de Formulari Dinàmics

Pots afegir o eliminar controls de formulari dinàmicament.

addAlias() {
  this.aliases.push(new FormControl(''));
}

removeAlias(index: number) {
  this.aliases.removeAt(index);
}

Validació

Validadors Integrats

Angular proporciona diversos validadors integrats com Validators.required, Validators.minLength, etc.

import { Validators } from '@angular/forms';

this.profileForm = new FormGroup({
  firstName: new FormControl('', Validators.required),
  lastName: new FormControl('', Validators.minLength(2))
});

Validadors Personalitzats

Pots crear validadors personalitzats per adaptar-te a les teves necessitats específiques.

import { AbstractControl, ValidatorFn } from '@angular/forms';

export function forbiddenNameValidator(nameRe: RegExp): ValidatorFn {
  return (control: AbstractControl): {[key: string]: any} | null => {
    const forbidden = nameRe.test(control.value);
    return forbidden ? {'forbiddenName': {value: control.value}} : null;
  };
}

this.profileForm = new FormGroup({
  firstName: new FormControl('', [Validators.required, forbiddenNameValidator(/bob/i)]),
  lastName: new FormControl('')
});

Formularis Reactius vs Formularis Basats en Plantilles

| Característica | Formularis Reactius | Formularis Basats en Plantilles | |------------------------|-------------------------------------|-------------------------------------| | Configuració | Més complexa, requereix més configuració | Més simple, menys configuració | | Model de Formulari | Definit a la classe del component | Definit a la plantilla | | Validació | Definida a la classe del component | Definida a la plantilla | | Escalabilitat | Més escalable per a formularis complexos | Menys escalable per a formularis complexos | | Proves | Més fàcil de provar | Més difícil de provar |

Conclusió

Els formularis reactius en Angular proporcionen una manera robusta i escalable de gestionar les entrades de formulari i les validacions. Aprofitant el poder dels observables, els formularis reactius ofereixen un enfocament més flexible i provable en comparació amb els formularis basats en plantilles. Tant si estàs tractant amb formularis simples com amb formularis complexos imbricats, els formularis reactius poden gestionar-ho tot de manera eficient.
