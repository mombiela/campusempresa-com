Introducció als Pipes

Els pipes en Angular són una manera de transformar dades en plantilles. S'utilitzen per formatar dades mostrades a l'usuari. Angular proporciona diversos pipes integrats, com DatePipe, UpperCasePipe, LowerCasePipe, CurrencyPipe i PercentPipe.

Per què Pipes personalitzats?

Tot i que els pipes integrats cobreixen molts escenaris comuns, hi ha moments en què necessites crear pipes personalitzats per gestionar transformacions de dades específiques que són úniques per a la teva aplicació.

Crear un Pipe personalitzat

Crear un pipe personalitzat implica els següents passos:

  1. Generar el Pipe: Utilitza Angular CLI per generar un nou pipe.
  2. Implementar la lògica del Pipe: Defineix la lògica de transformació a la classe del pipe.
  3. Registrar el Pipe: Assegura't que el pipe està declarat en un mòdul d'Angular.

Pas 1: Generar el Pipe

Utilitza Angular CLI per generar un nou pipe:

ng generate pipe custom

Aquesta comanda crea un nou fitxer custom.pipe.ts amb una estructura bàsica.

Pas 2: Implementar la lògica del Pipe

Edita el fitxer generat del pipe per implementar la lògica de transformació. Per exemple, creem un pipe que converteix una cadena a majúscules (la primera lletra de cada paraula està en majúscula).

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'titleCase'
})
export class TitleCasePipe implements PipeTransform {
  transform(value: string): string {
    if (!value) return value;
    return value.replace(/\w\S*/g, (txt) => {
      return txt.charAt(0).toUpperCase() + txt.substr(1).toLowerCase();
    });
  }
}

Pas 3: Registrar el Pipe

Assegura't que el pipe està declarat en un mòdul d'Angular, típicament en app.module.ts.

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';
import { TitleCasePipe } from './title-case.pipe';

@NgModule({
  declarations: [
    AppComponent,
    TitleCasePipe
  ],
  imports: [
    BrowserModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Utilitzar el Pipe personalitzat

Un cop el pipe personalitzat està creat i registrat, pots utilitzar-lo en les teves plantilles com qualsevol altre pipe.

<p>{{ 'hello world' | titleCase }}</p>

Això mostrarà:

Hello World

Pipes amb paràmetres

Els pipes personalitzats també poden acceptar paràmetres. Per exemple, creem un pipe que formata un número amb un nombre especificat de decimals.

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'decimalFormat'
})
export class DecimalFormatPipe implements PipeTransform {
  transform(value: number, decimalPlaces: number): string {
    return value.toFixed(decimalPlaces);
  }
}

Ús en una plantilla:

<p>{{ 1234.5678 | decimalFormat:2 }}</p>

Això mostrarà:

1234.57

Pipes purs vs impurs

  • Pipes purs: Aquests són sense estat i només depenen dels seus arguments d'entrada. Només es criden quan Angular detecta un canvi en el valor d'entrada.
  • Pipes impurs: Aquests poden tenir efectes secundaris i es criden en cada cicle de detecció de canvis.

Per crear un pipe impur, estableix la propietat pure a false en el decorador @Pipe.

@Pipe({
  name: 'impurePipe',
  pure: false
})
export class ImpurePipe implements PipeTransform {
  // Implementació
}

Resum

  • Pipes personalitzats: Et permeten crear lògica de transformació de dades reutilitzable.
  • Passos per crear: Generar, implementar i registrar el pipe.
  • Ús: Utilitza pipes personalitzats en plantilles com els pipes integrats.
  • Pipes amb paràmetres: Accepten arguments addicionals per a transformacions més flexibles.
  • Pipes purs vs impurs: Els pipes purs són sense estat, mentre que els pipes impurs poden tenir efectes secundaris i es criden més freqüentment.

Dominant els pipes personalitzats, pots millorar significativament les capacitats de presentació de dades de les teves aplicacions Angular.
