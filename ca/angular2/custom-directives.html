Introducció

Les directives personalitzades a Angular 2+ et permeten estendre la funcionalitat dels elements HTML i crear components reutilitzables. Les directives es poden utilitzar per manipular el DOM, aplicar estils o afegir comportament als elements.

Tipus de directives

Angular 2+ proporciona tres tipus de directives:

  • Directives de component: Aquestes són les directives més comunes i s'utilitzen per crear components.
  • Directives estructurals: Aquestes directives canvien el disseny del DOM afegint o eliminant elements (per exemple, *ngIf, *ngFor).
  • Directives d'atribut: Aquestes directives canvien l'aparença o el comportament d'un element, component o una altra directiva (per exemple, ngClass, ngStyle).

Creació d'una directiva d'atribut personalitzada

Les directives d'atribut s'utilitzen per canviar el comportament o l'aparença d'un element. Creem una simple directiva d'atribut personalitzada que canviï el color de fons d'un element quan es passa el ratolí per sobre.

Guia pas a pas

  1. Generar la directiva: Utilitza Angular CLI per generar una nova directiva.

    ng generate directive highlight

  2. Implementar la directiva: Obre el fitxer generat highlight.directive.ts i implementa la lògica de la directiva.

    import { Directive, ElementRef, HostListener, Input } from '@angular/core';

@Directive({
  selector: '[appHighlight]'
})
export class HighlightDirective {
  @Input() defaultColor: string = 'yellow';
  @Input('appHighlight') highlightColor: string;

  constructor(private el: ElementRef) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.highlight(this.highlightColor || this.defaultColor);
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.highlight(null);
  }

  private highlight(color: string) {
    this.el.nativeElement.style.backgroundColor = color;
  }
}

  3. Utilitzar la directiva en un component: Aplica la directiva a un element en la plantilla del teu component.

    <p appHighlight="lightblue" defaultColor="lightgreen">
  Passa el ratolí per sobre d'aquest text per veure l'efecte de ressaltat.
</p>

Explicació

  • @Directive: Decorador que marca una classe com una directiva d'Angular.
  • selector: El selector CSS que identifica la directiva en una plantilla.
  • ElementRef: Servei que proporciona accés directe a l'element del DOM.
  • @HostListener: Decorador que escolta esdeveniments del DOM i desencadena el mètode especificat.
  • @Input: Decorador que permet passar dades a la directiva des del component pare.

Creació d'una directiva estructural personalitzada

Les directives estructurals canvien el disseny del DOM afegint o eliminant elements. Creem una directiva estructural personalitzada que inclogui condicionalment una plantilla basada en una expressió booleana.

Guia pas a pas

  1. Generar la directiva: Utilitza Angular CLI per generar una nova directiva.

    ng generate directive unless

  2. Implementar la directiva: Obre el fitxer generat unless.directive.ts i implementa la lògica de la directiva.

    import { Directive, Input, TemplateRef, ViewContainerRef } from '@angular/core';

@Directive({
  selector: '[appUnless]'
})
export class UnlessDirective {
  private hasView = false;

  constructor(
    private templateRef: TemplateRef<any>,
    private viewContainer: ViewContainerRef
  ) {}

  @Input() set appUnless(condition: boolean) {
    if (!condition && !this.hasView) {
      this.viewContainer.createEmbeddedView(this.templateRef);
      this.hasView = true;
    } else if (condition && this.hasView) {
      this.viewContainer.clear();
      this.hasView = false;
    }
  }
}

  3. Utilitzar la directiva en un component: Aplica la directiva a un element en la plantilla del teu component.

    <div *appUnless="isVisible">
  Aquest text es mostra només quan isVisible és fals.
</div>

Explicació

  • TemplateRef: Representa una plantilla incrustada que es pot utilitzar per instanciar vistes incrustades.
  • ViewContainerRef: Representa un contenidor on es poden adjuntar una o més vistes.
  • @Input: Decorador que permet passar dades a la directiva des del component pare.

Conclusió

Les directives personalitzades a Angular 2+ són eines poderoses que et permeten estendre la funcionalitat dels elements HTML i crear components reutilitzables. Comprenent i implementant directives d'atribut i estructurals, pots millorar la interactivitat i mantenibilitat de la teva aplicació. Practica creant les teves pròpies directives personalitzades per convertir-te en un desenvolupador expert en Angular.
