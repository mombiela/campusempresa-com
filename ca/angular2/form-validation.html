Introducció a la Validació de Formularis

La validació de formularis és un aspecte crucial del desenvolupament web, assegurant que les entrades dels usuaris siguin correctes i segures abans de processar-les. Angular 2+ proporciona eines poderoses per gestionar la validació de formularis de manera eficient.

Tipus de Formularis en Angular

Angular suporta dos tipus de formularis:

  • Formularis basats en plantilles: Més fàcils d'utilitzar, adequats per a formularis simples.
  • Formularis reactius: Més flexibles i escalables, adequats per a formularis complexos.

Formularis basats en plantilles

Els formularis basats en plantilles depenen molt de les directives d'Angular i es defineixen en la plantilla.

Creació d'un Formulari Bàsic Basat en Plantilles

<form #userForm="ngForm" (ngSubmit)="onSubmit(userForm)">
  <div>
    <label for="name">Nom:</label>
    <input type="text" id="name" name="name" ngModel required>
  </div>
  <div>
    <label for="email">Correu electrònic:</label>
    <input type="email" id="email" name="email" ngModel required email>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Validació de Formularis Basats en Plantilles

  • Requerit: Assegura que el camp no estigui buit.
  • Correu electrònic: Assegura que el camp contingui una adreça de correu electrònic vàlida.

Mostrant Errors de Validació

<div *ngIf="userForm.controls.name?.invalid && userForm.controls.name?.touched">
  <small *ngIf="userForm.controls.name?.errors?.required">El nom és requerit.</small>
</div>
<div *ngIf="userForm.controls.email?.invalid && userForm.controls.email?.touched">
  <small *ngIf="userForm.controls.email?.errors?.required">El correu electrònic és requerit.</small>
  <small *ngIf="userForm.controls.email?.errors?.email">Adreça de correu electrònic no vàlida.</small>
</div>

Formularis Reactius

Els formularis reactius són més robustos i proporcionen un major control sobre la validació dels formularis.

Creació d'un Formulari Bàsic Reactiu

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-user-form',
  templateUrl: './user-form.component.html'
})
export class UserFormComponent {
  userForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.userForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.userForm.valid) {
      console.log(this.userForm.value);
    }
  }
}

Plantilla per a Formulari Reactiu

<form [formGroup]="userForm" (ngSubmit)="onSubmit()">
  <div>
    <label for="name">Nom:</label>
    <input type="text" id="name" formControlName="name">
    <div *ngIf="userForm.controls.name.invalid && userForm.controls.name.touched">
      <small *ngIf="userForm.controls.name.errors.required">El nom és requerit.</small>
    </div>
  </div>
  <div>
    <label for="email">Correu electrònic:</label>
    <input type="email" id="email" formControlName="email">
    <div *ngIf="userForm.controls.email.invalid && userForm.controls.email.touched">
      <small *ngIf="userForm.controls.email.errors.required">El correu electrònic és requerit.</small>
      <small *ngIf="userForm.controls.email.errors.email">Adreça de correu electrònic no vàlida.</small>
    </div>
  </div>
  <button type="submit" [disabled]="userForm.invalid">Enviar</button>
</form>

Validadors Personalitzats

Els validadors personalitzats permeten crear una lògica de validació específica.

Creació d'un Validador Personalitzat

import { AbstractControl, ValidationErrors } from '@angular/forms';

export function forbiddenNameValidator(control: AbstractControl): ValidationErrors | null {
  const forbidden = /admin/.test(control.value);
  return forbidden ? { forbiddenName: { value: control.value } } : null;
}

Ús de Validadors Personalitzats

this.userForm = this.fb.group({
  name: ['', [Validators.required, forbiddenNameValidator]],
  email: ['', [Validators.required, Validators.email]]
});

Mostrant Errors de Validació Personalitzats

<div *ngIf="userForm.controls.name.invalid && userForm.controls.name.touched">
  <small *ngIf="userForm.controls.name.errors.required">El nom és requerit.</small>
  <small *ngIf="userForm.controls.name.errors.forbiddenName">El nom no pot ser 'admin'.</small>
</div>

Resum

La validació de formularis en Angular 2+ es pot gestionar utilitzant formularis basats en plantilles o formularis reactius. Els formularis basats en plantilles són més simples i adequats per a casos d'ús bàsics, mentre que els formularis reactius ofereixen més control i escalabilitat per a escenaris complexos. Els validadors personalitzats proporcionen flexibilitat per implementar una lògica de validació específica. Entendre aquests conceptes t'ajudarà a crear formularis robustos i fàcils d'usar en les teves aplicacions Angular.
