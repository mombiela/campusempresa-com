La teva primera aplicació Angular
Arquitectura d'Angular ►

Introducció

En aquesta secció, aprendràs a crear la teva primera aplicació Angular. Cobrirem els següents conceptes clau:

  • Configuració de l'entorn Angular
  • Creació d'un nou projecte Angular
  • Comprensió de l'estructura del projecte
  • Executar l'aplicació Angular
  • Components bàsics d'Angular

Configuració de l'entorn Angular

Abans de poder crear una aplicació Angular, has de configurar el teu entorn de desenvolupament.

Prerequisits

  • Node.js i npm: Angular requereix Node.js i npm (Node Package Manager). Pots descarregar i instal·lar-los des de nodejs.org.
  • Angular CLI: La Interfície de Línia de Comandes d'Angular (CLI) és una eina potent per inicialitzar, desenvolupar, esbossar i mantenir aplicacions Angular.

Instal·lació de l'Angular CLI

Per instal·lar l'Angular CLI, obre el teu terminal o símbol del sistema i executa:

npm install -g @angular/cli

Creació d'un nou projecte Angular

Un cop instal·lat l'Angular CLI, pots crear un nou projecte Angular.

Passos per crear un nou projecte

  1. Obre el teu terminal i navega al directori on vols crear el teu projecte.
  2. Executa la següent comanda per crear un nou projecte Angular: 
    ng new my-first-angular-app
  3. Navega al directori del projecte: 
    cd my-first-angular-app

Comprensió de l'estructura del projecte

Quan crees un nou projecte Angular, l'Angular CLI genera un conjunt d'arxius i carpetes. Aquí tens una breu visió general dels components principals:

  • e2e/: Proves de punta a punta per a l'aplicació.
  • node_modules/: Conté els paquets npm instal·lats per al projecte.
  • src/: El codi font de l'aplicació.
    • app/: Conté el codi principal de l'aplicació.
    • assets/: Conté actius estàtics com imatges i estils.
    • environments/: Conté arxius de configuració específics per a l'entorn.
  • angular.json: Arxiu de configuració per a l'Angular CLI.
  • package.json: Llista els paquets npm dels quals depèn el projecte.
  • tsconfig.json: Arxiu de configuració de TypeScript.

Executar l'aplicació Angular

Per executar la teva aplicació Angular, utilitza l'Angular CLI:

ng serve

Aquesta comanda compilarà l'aplicació i iniciarà un servidor de desenvolupament. Per defecte, l'aplicació estarà disponible a http://localhost:4200/.

Components bàsics d'Angular

Les aplicacions Angular es construeixen utilitzant components. Un component en Angular és una classe TypeScript amb una plantilla HTML i estils.

Creació d'un nou component

Pots crear un nou component utilitzant l'Angular CLI:

ng generate component my-component

Aquesta comanda generarà un nou component amb els següents arxius:

  • my-component.component.ts: L'arxiu TypeScript que conté la lògica del component.
  • my-component.component.html: La plantilla HTML per al component.
  • my-component.component.css: Els estils per al component.
  • my-component.component.spec.ts: L'arxiu de proves per al component.

Exemple de component

Aquí tens un exemple d'un component Angular senzill:

app.component.ts

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'my-first-angular-app';
}

app.component.html

<div style="text-align:center">
  <h1>
    Welcome to {{ title }}!
  </h1>
</div>

Explicació

  • @Component: Aquest decorador marca la classe com un component Angular i proporciona metadades sobre el component.
  • selector: L'etiqueta HTML personalitzada que representa aquest component.
  • templateUrl: La ruta a l'arxiu de la plantilla HTML.
  • styleUrls: La ruta a l'arxiu(s) CSS.

Conclusió

Felicitats! Has creat amb èxit la teva primera aplicació Angular. Has après a configurar l'entorn Angular, crear un nou projecte, comprendre l'estructura del projecte, executar l'aplicació i crear un component Angular bàsic. Aquest coneixement fonamental t'ajudarà a mesura que continuïs construint aplicacions Angular més complexes.
