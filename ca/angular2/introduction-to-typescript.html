Què és TypeScript?

TypeScript és un superconjunt de JavaScript que afegeix tipus estàtics. És desenvolupat i mantingut per Microsoft i ha guanyat una popularitat significativa en la comunitat de desenvolupament, especialment per a aplicacions a gran escala. TypeScript ajuda a detectar errors d'hora mitjançant un sistema de tipus i fa que el desenvolupament de JavaScript sigui més robust i mantenible.

Característiques Clau de TypeScript

  • Tipatge Estàtic: Et permet definir tipus per a variables, paràmetres de funcions i valors de retorn.
  • Inferència de Tipus: Infereix automàticament els tipus quan no estan definits explícitament.
  • Interfícies: Defineixen contractes dins del teu codi.
  • Classes i Herència: Suport per a la programació orientada a objectes.
  • Mòduls: Organitzen el codi en components reutilitzables.
  • Eines: Suport millorat en IDEs per a la navegació de codi, refactorització i depuració.

Configuració de TypeScript

Per començar a utilitzar TypeScript, cal instal·lar-lo a través de npm (Node Package Manager):

npm install -g typescript

Després pots compilar fitxers TypeScript a JavaScript utilitzant la comanda tsc:

tsc filename.ts

Sintaxi Bàsica de TypeScript

Variables i Tipus

TypeScript et permet definir tipus per a les variables:

let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];

Funcions

Pots especificar tipus per als paràmetres de funció i els valors de retorn:

function add(x: number, y: number): number {
    return x + y;
}

Interfícies

Les interfícies defineixen l'estructura d'un objecte:

interface Person {
    firstName: string;
    lastName: string;
}

function greet(person: Person) {
    return "Hello, " + person.firstName + " " + person.lastName;
}

let user = { firstName: "John", lastName: "Doe" };
console.log(greet(user));

Classes

TypeScript suporta classes i herència:

class Animal {
    name: string;
    constructor(name: string) {
        this.name = name;
    }
    move(distance: number = 0) {
        console.log(`${this.name} moved ${distance}m.`);
    }
}

class Dog extends Animal {
    bark() {
        console.log('Woof! Woof!');
    }
}

let dog = new Dog("Rex");
dog.bark();
dog.move(10);

Mòduls

Els mòduls ajuden a organitzar el codi:

// math.ts
export function add(x: number, y: number): number {
    return x + y;
}

// app.ts
import { add } from './math';
console.log(add(5, 3));

TypeScript en Angular

Angular està construït amb TypeScript, i entendre TypeScript és crucial per desenvolupar aplicacions Angular. Aquí hi ha alguns punts clau on TypeScript s'utilitza en Angular:

  • Components: Defineix classes de components amb TypeScript.
  • Serveis: Utilitza TypeScript per definir serveis i els seus mètodes.
  • Injecció de Dependències: TypeScript ajuda a definir tipus per a les dependències.
  • Rutes: Defineix rutes i els seus paràmetres utilitzant TypeScript.

Exemple: Component Angular amb TypeScript

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `<h1>{{title}}</h1>`,
})
export class AppComponent {
  title: string = 'Hello, Angular!';
}

Conclusió

TypeScript és una eina poderosa que millora JavaScript afegint tipus estàtics, interfícies, classes i mòduls. És essencial per desenvolupar aplicacions Angular robustes i mantenibles. Entenent els conceptes bàsics de TypeScript, estaràs millor preparat per treballar amb Angular i aprofitar tot el seu potencial.
