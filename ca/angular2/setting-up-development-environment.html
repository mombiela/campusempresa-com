La teva primera aplicació Angular ►

Configurar l'entorn de desenvolupament és el primer pas crucial en el teu viatge per dominar Angular 2+. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar amb el desenvolupament d'Angular.

Prerequisits

Abans de configurar Angular, assegura't de tenir els següents prerequisits instal·lats al teu sistema:

  • Node.js i npm: Angular requereix Node.js i npm (Node Package Manager). Pots descarregar i instal·lar-los des del lloc web oficial de Node.js.

Verificant la Instal·lació de Node.js i npm

Per verificar si Node.js i npm estan instal·lats, obre el teu terminal o símbol del sistema i executa les següents comandes:

node -v
npm -v

Hauries de veure els números de versió de Node.js i npm, respectivament.

Instal·lant Angular CLI

Angular CLI (Command Line Interface) és una eina poderosa que simplifica el procés de crear i gestionar aplicacions Angular. Per instal·lar Angular CLI globalment al teu sistema, utilitza la següent comanda:

npm install -g @angular/cli

Verificant la Instal·lació d'Angular CLI

Després de la instal·lació, verifica que Angular CLI està instal·lat correctament executant:

ng version

Aquesta comanda mostrarà la versió d'Angular CLI i altres paquets relacionats.

Creant un Nou Projecte Angular

Amb Angular CLI instal·lat, ara pots crear un nou projecte Angular. Utilitza la següent comanda per crear un nou projecte anomenat my-angular-app:

ng new my-angular-app

Opcions de Configuració del Projecte

Durant el procés de creació del projecte, Angular CLI et demanarà que triïs algunes opcions de configuració:

  • Would you like to add Angular routing?: Tria Yes o No depenent de si necessites routing a la teva aplicació.
  • Which stylesheet format would you like to use?: Selecciona el teu format de full d'estil preferit (CSS, SCSS, SASS, LESS, Stylus).

Executant l'Aplicació Angular

Navega al directori del teu projecte i inicia el servidor de desenvolupament utilitzant les següents comandes:

cd my-angular-app
ng serve

Accedint a l'Aplicació

Un cop el servidor estigui en funcionament, obre el teu navegador web i navega a http://localhost:4200/. Hauries de veure la pàgina de benvinguda predeterminada d'Angular.

Eines i Extensions de Desenvolupament

Per millorar la teva experiència de desenvolupament, considera instal·lar les següents eines i extensions:

  • Visual Studio Code (VS Code): Un editor de codi popular amb un excel·lent suport per Angular.
  • Angular Language Service: Una extensió per VS Code que proporciona edició rica i IntelliSense per a plantilles Angular.

Instal·lant Angular Language Service a VS Code

  1. Obre VS Code.
  2. Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
  3. Cerca "Angular Language Service" i fes clic a Instal·lar.

Resum

En aquesta secció, has après com configurar l'entorn de desenvolupament per Angular 2+. Has instal·lat Node.js i npm, configurat Angular CLI, creat un nou projecte Angular i executat el servidor de desenvolupament. A més, has explorat algunes eines i extensions útils per millorar la teva experiència de codificació.

Assegurant que el teu entorn de desenvolupament està correctament configurat, ara estàs preparat per aprofundir en Angular i començar a construir aplicacions web poderoses.
