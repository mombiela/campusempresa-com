Introducció

La Interfície de Línia de Comandes d'Angular (CLI) és una eina poderosa que simplifica el procés de desenvolupament d'aplicacions Angular. Proporciona un conjunt de comandes per generar, construir i gestionar projectes Angular de manera eficient. Aquesta secció cobrirà les millors pràctiques per utilitzar Angular CLI per assegurar que els teus projectes estiguin ben estructurats, siguin mantenibles i escalables.

Configuració del Teu Projecte Angular

  • Instal·la Angular CLI: Assegura't de tenir la darrera versió d'Angular CLI instal·lada. 
    npm install -g @angular/cli
  • Crea un Nou Projecte: Utilitza el CLI per generar un nou projecte Angular. 
    ng new my-angular-app
    • Utilitza noms de projectes significatius.
    • Segueix les indicacions per incloure el routing i seleccionar fulls d'estil (CSS, SCSS, etc.).

Estructura del Projecte

  • Organitza el Teu Codi: Segueix una estructura de carpetes consistent.
    • Mòdul Core: Per a serveis singleton, components universals i guardes.
    • Mòdul Shared: Per a components compartits, directives i pipes.
    • Mòduls de Funcionalitats: Per a diferents funcionalitats o seccions de la teva aplicació.
  • Exemple d'Estructura: 
    src/
  app/
    core/
    shared/
    features/
      feature1/
      feature2/

Generació de Components, Serveis i Mòduls

  • Utilitza Comandes CLI: Genera components, serveis i mòduls utilitzant comandes CLI per assegurar la consistència. 
    ng generate component my-component
ng generate service my-service
ng generate module my-module
  • Convencions de Nomenclatura: Utilitza noms consistents i descriptius per a fitxers i classes.

Configuració i Gestió d'Entorns

  • Fitxers d'Entorn: Utilitza fitxers d'entorn per gestionar diferents configuracions per a desenvolupament, staging i producció. 
    src/environments/
  environment.ts
  environment.prod.ts
  • Exemple de Configuració d'Entorn: 
    // environment.ts
export const environment = {
  production: false,
  apiUrl: 'http://localhost:3000/api'
};

// environment.prod.ts
export const environment = {
  production: true,
  apiUrl: 'https://api.myapp.com'
};

Linting i Formatació

  • Utilitza Eines de Linting: Integra eines com TSLint o ESLint per fer complir els estàndards de codificació. 
    ng lint
  • Prettier: Utilitza Prettier per a una formatació de codi consistent. 
    npm install --save-dev prettier

Proves

  • Proves Unitàries: Escriu proves unitàries per a components, serveis i pipes. 
    ng test
  • Proves de Cap a Cap: Utilitza Protractor o Cypress per a proves de cap a cap. 
    ng e2e

Construcció i Desplegament

  • Construeix per a Producció: Utilitza el CLI per construir la teva aplicació per a producció. 
    ng build --prod
  • Optimitza les Construccions: Activa la compilació Ahead-of-Time (AOT) i altres optimitzacions. 
    ng build --prod --aot

Resum

Utilitzar Angular CLI de manera efectiva pot millorar significativament el teu flux de treball de desenvolupament. Seguint les millors pràctiques com organitzar l'estructura del projecte, utilitzar fitxers d'entorn, fer complir els estàndards de codificació i escriure proves, pots crear aplicacions Angular mantenibles i escalables. Recorda mantenir les teves eines CLI actualitzades i aprofitar les potents comandes que ofereix per agilitzar el teu procés de desenvolupament.
