La càrrega peresosa és un patró de disseny comunament utilitzat en aplicacions Angular per carregar components JavaScript de manera asíncrona quan s'activa una ruta específica. Això pot millorar significativament el rendiment de la teva aplicació reduint el temps de càrrega inicial.

Conceptes Clau

  • Càrrega Peresosa: Carregar mòduls només quan són necessaris, en lloc de fer-ho al començament de l'aplicació.
  • Mòduls: Els mòduls d'Angular (NgModule) s'utilitzen per agrupar components, directives, pipes i serveis relacionats.
  • Rutes: Defineixen els camins de navegació en la teva aplicació. La càrrega peresosa sovint s'implementa utilitzant el router d'Angular.

Per què Utilitzar la Càrrega Peresosa?

  • Rendiment: Redueix el temps de càrrega inicial de l'aplicació carregant només els mòduls necessaris.
  • Escalabilitat: Ajuda a gestionar aplicacions grans dividint-les en peces més petites i manejables.
  • Optimització de Recursos: Carrega recursos només quan són necessaris, cosa que pot estalviar ample de banda i millorar l'experiència de l'usuari.

Configuració de la Càrrega Peresosa

Pas 1: Crear un Mòdul de Funcionalitat

Primer, crea un nou mòdul de funcionalitat que vulguis carregar de manera peresosa.

ng generate module feature --route feature --module app.module

Aquesta comanda crearà un nou mòdul i configurarà el router per carregar-lo de manera peresosa.

Pas 2: Configurar Rutes per a la Càrrega Peresosa

En el teu app-routing.module.ts, configura les rutes per carregar de manera peresosa el mòdul de funcionalitat.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: 'feature',
    loadChildren: () => import('./feature/feature.module').then(m => m.FeatureModule)
  }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Pas 3: Definir Rutes en el Mòdul de Funcionalitat

En el teu feature-routing.module.ts, defineix les rutes per al mòdul de funcionalitat.

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { FeatureComponent } from './feature.component';

const routes: Routes = [
  { path: '', component: FeatureComponent }
];

@NgModule({
  imports: [RouterModule.forChild(routes)],
  exports: [RouterModule]
})
export class FeatureRoutingModule { }

Pas 4: Crear el Component de Funcionalitat

Crea el component que formarà part del mòdul de funcionalitat.

ng generate component feature/feature

Pas 5: Actualitzar el Mòdul de Funcionalitat

Actualitza el feature.module.ts per incloure el nou component i el mòdul de rutes.

import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { FeatureComponent } from './feature.component';
import { FeatureRoutingModule } from './feature-routing.module';

@NgModule({
  declarations: [FeatureComponent],
  imports: [
    CommonModule,
    FeatureRoutingModule
  ]
})
export class FeatureModule { }

Exemple: Càrrega Peresosa d'un Mòdul de Tauler de Control

Creem un exemple més concret carregant de manera peresosa un DashboardModule.

Pas 1: Generar el Mòdul de Tauler de Control

ng generate module dashboard --route dashboard --module app.module

Pas 2: Configurar Rutes per a la Càrrega Peresosa

Actualitza app-routing.module.ts:

const routes: Routes = [
  {
    path: 'dashboard',
    loadChildren: () => import('./dashboard/dashboard.module').then(m => m.DashboardModule)
  }
];

Pas 3: Definir Rutes en el Mòdul de Tauler de Control

Actualitza dashboard-routing.module.ts:

const routes: Routes = [
  { path: '', component: DashboardComponent }
];

Pas 4: Crear el Component de Tauler de Control

ng generate component dashboard/dashboard

Pas 5: Actualitzar el Mòdul de Tauler de Control

Actualitza dashboard.module.ts:

@NgModule({
  declarations: [DashboardComponent],
  imports: [
    CommonModule,
    DashboardRoutingModule
  ]
})
export class DashboardModule { }

Conclusió

La càrrega peresosa és una característica poderosa en Angular que pot ajudar a millorar el rendiment i l'escalabilitat de la teva aplicació. Carregant mòduls només quan són necessaris, pots reduir el temps de càrrega inicial i optimitzar l'ús de recursos. Aquesta guia ha proporcionat un enfocament pas a pas per implementar la càrrega peresosa en Angular, des de la creació d'un mòdul de funcionalitat fins a la configuració de rutes i l'actualització de mòduls. Amb aquestes tècniques, pots construir aplicacions Angular més eficients i responsives.
