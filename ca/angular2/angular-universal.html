Introducció a Angular Universal

Angular Universal és una tecnologia que et permet renderitzar aplicacions Angular al costat del servidor. Això pot millorar el rendiment, el SEO i l'experiència de l'usuari en lliurar pàgines completament renderitzades al client.

Conceptes Clau

  • Renderitzat del costat del servidor (SSR): El procés de renderitzar pàgines web al servidor en lloc del client.
  • Angular Universal: Un conjunt d'eines i biblioteques que permeten SSR per a aplicacions Angular.
  • Beneficis SEO: Els motors de cerca poden indexar pàgines renderitzades pel servidor de manera més efectiva.
  • Millores de Rendiment: Temps de càrrega inicial més ràpids i millor rendiment percebut.

Configuració d'Angular Universal

Per començar amb Angular Universal, necessites afegir els paquets necessaris i configurar la teva aplicació Angular.

Configuració Pas a Pas

  1. Instal·la els Paquets d'Angular Universal:

    ng add @nguniversal/express-engine

    Aquesta comanda instal·larà els paquets necessaris i farà els canvis necessaris al teu projecte Angular.

  2. Actualitza la Configuració d'Angular: La comanda ng add actualitzarà automàticament el teu fitxer angular.json per incloure configuracions de renderitzat del costat del servidor.

  3. Crea el Mòdul del Servidor: Angular Universal requereix un mòdul del servidor per gestionar el renderitzat del costat del servidor.

    // src/app/app.server.module.ts
import { NgModule } from '@angular/core';
import { ServerModule } from '@angular/platform-server';
import { AppModule } from './app.module';
import { AppComponent } from './app.component';

@NgModule({
  imports: [
    AppModule,
    ServerModule,
  ],
  bootstrap: [AppComponent],
})
export class AppServerModule {}

  4. Crea el Fitxer del Servidor: Crea un fitxer del servidor per inicialitzar l'aplicació Angular al servidor.

    // server.ts
import 'zone.js/dist/zone-node';
import { ngExpressEngine } from '@nguniversal/express-engine';
import * as express from 'express';
import { join } from 'path';
import { APP_BASE_HREF } from '@angular/common';
import { existsSync } from 'fs';

import { AppServerModule } from './src/main.server';
import { APP_BASE_HREF } from '@angular/common;

const app = express();
const PORT = process.env.PORT || 4000;
const DIST_FOLDER = join(process.cwd(), 'dist/browser');

app.engine('html', ngExpressEngine({
  bootstrap: AppServerModule,
}));

app.set('view engine', 'html');
app.set('views', DIST_FOLDER);

app.get('*.*', express.static(DIST_FOLDER, {
  maxAge: '1y'
}));

app.get('*', (req, res) => {
  res.render('index', { req, providers: [{ provide: APP_BASE_HREF, useValue: req.baseUrl }] });
});

app.listen(PORT, () => {
  console.log(`Node server listening on http://localhost:${PORT}`);
});

  5. Construeix i Serveix: Construeix l'aplicació tant per al client com per al servidor:

    npm run build:ssr
npm run serve:ssr

Configuració Avançada

Un cop tinguis la configuració bàsica, pots optimitzar i configurar Angular Universal per a les teves necessitats específiques.

Càrrega Per Mandat de Mòduls

La càrrega per mandat es pot configurar per treballar amb Angular Universal per millorar el rendiment.

// app-routing.module.ts
const routes: Routes = [
  {
    path: '',
    loadChildren: () => import('./home/home.module').then(m => m.HomeModule)
  }
];

Gestió de Peticions API

El renderitzat del costat del servidor pot gestionar les peticions API de manera diferent. Assegura't que el teu codi del servidor pugui gestionar aquestes peticions adequadament.

// server.ts
app.get('/api/data', (req, res) => {
  res.json({ data: 'Server-side data' });
});

Estratègies de Caché

Implementa estratègies de caché per millorar el rendiment i reduir la càrrega del servidor.

// server.ts
app.get('*', (req, res) => {
  const cacheKey = req.url;
  if (cache.has(cacheKey)) {
    res.send(cache.get(cacheKey));
  } else {
    res.render('index', { req }, (err, html) => {
      cache.set(cacheKey, html);
      res.send(html);
    });
  }
});

Conclusió

Angular Universal proporciona una manera poderosa de millorar el rendiment i el SEO de les teves aplicacions Angular a través del renderitzat del costat del servidor. Seguint els passos de configuració, pots aprofitar SSR per lliurar aplicacions web més ràpides i eficients. Les configuracions avançades com la càrrega per mandat, la gestió de peticions API i la implementació d'estratègies de caché poden optimitzar encara més la teva aplicació per a un millor rendiment i escalabilitat.
