Introducció

TypeScript és un superconjunt de JavaScript amb tipatge estàtic que aporta tipatge estàtic opcional i altres característiques al llenguatge. Entendre les variables i els tipus de dades en TypeScript és crucial per desenvolupar aplicacions Angular robustes. Aquesta secció cobrirà els conceptes bàsics de la declaració de variables i els diversos tipus de dades disponibles en TypeScript.

Declaració de variables

En TypeScript, les variables es poden declarar utilitzant let, const i var. Cadascun té el seu propi àmbit i consideracions d'ús.

`let`

  • Àmbit de bloc.
  • Es pot reasignar.
let message: string = "Hola, TypeScript!";
message = "Hola, Angular!";
console.log(message); // Sortida: Hola, Angular!

`const`

  • Àmbit de bloc.
  • No es pot reasignar.
const pi: number = 3.14;
// pi = 3.14159; // Error: Assignació a una variable constant.
console.log(pi); // Sortida: 3.14

`var`

  • Àmbit de funció.
  • Es pot reasignar.
var greeting: string = "Hola!";
greeting = "Hola de nou!";
console.log(greeting); // Sortida: Hola de nou!

Tipus de dades

TypeScript suporta diversos tipus de dades, incloent-hi tipus primitius i tipus més complexos.

Tipus primitius

  • Number: Representa tant nombres enters com de punt flotant.
  • String: Representa dades de text.
  • Boolean: Representa valors veritat/fals.
  • Null: Representa una absència intencional de qualsevol valor d'objecte.
  • Undefined: Representa una variable no inicialitzada.
  • Symbol: Representa un valor primitiu únic i immutable.
let age: number = 25;
let name: string = "John Doe";
let isStudent: boolean = true;
let notAssigned: undefined = undefined;
let empty: null = null;

Array

Els arrays es poden definir de dues maneres: utilitzant el tipus seguit de [] o utilitzant la sintaxi Array<type>.

let numbers: number[] = [1, 2, 3, 4, 5];
let fruits: Array<string> = ["Poma", "Plàtan", "Cirera"];

Tuple

Les tuples permeten expressar un array amb un nombre fix d'elements els tipus dels quals són coneguts.

let person: [string, number];
person = ["Alice", 30]; // Correcte
// person = [30, "Alice"]; // Error: El tipus 'number' no es pot assignar al tipus 'string'.

Enum

Els enums permeten definir un conjunt de constants amb nom.

enum Color {
    Red,
    Green,
    Blue
}
let c: Color = Color.Green;
console.log(c); // Sortida: 1

Any

El tipus any permet desactivar la comprovació de tipus.

let randomValue: any = 10;
randomValue = "Hola";
randomValue = true;

Void

El tipus void s'utilitza per indicar l'absència d'un valor, comunament utilitzat com el tipus de retorn de funcions que no retornen cap valor.

function logMessage(message: string): void {
    console.log(message);
}

Object

El tipus object representa tipus no primitius.

let user: { name: string; age: number } = {
    name: "John",
    age: 30
};

Inferència de tipus

TypeScript pot inferir el tipus d'una variable basant-se en el valor assignat.

let inferredString = "Això és una cadena"; // inferredString és de tipus string
let inferredNumber = 42; // inferredNumber és de tipus number

Assertions de tipus

Les assertions de tipus s'utilitzen per dir al compilador que tracti una variable com un tipus específic.

let someValue: any = "Hola, TypeScript!";
let strLength: number = (someValue as string).length;
console.log(strLength); // Sortida: 17

Conclusió

Entendre les variables i els tipus de dades en TypeScript és fonamental per construir aplicacions Angular. Aprofitant el tipatge estàtic de TypeScript, pots escriure codi més predictible i mantenible. Aquesta secció ha cobert els conceptes bàsics de la declaració de variables, els diversos tipus de dades disponibles i com utilitzar la inferència de tipus i les assertions de tipus. Amb aquest coneixement, estàs ben preparat per gestionar dades de manera efectiva en els teus projectes Angular.
