Introducció a les Guàrdies de Ruta

Les guàrdies de ruta a Angular s'utilitzen per controlar l'accés a les rutes dins d'una aplicació Angular. Et permeten definir condicions que s'han de complir abans que una ruta es pugui activar o desactivar, assegurant que els usuaris només puguin accedir a certes parts de la teva aplicació sota condicions específiques.

Tipus de Guàrdies de Ruta

Angular proporciona diversos tipus de guàrdies de ruta, cadascuna amb un propòsit diferent:

  • CanActivate: Determina si una ruta es pot activar.
  • CanActivateChild: Determina si les rutes filles es poden activar.
  • CanDeactivate: Determina si una ruta es pot desactivar.
  • CanLoad: Determina si un mòdul es pot carregar.
  • Resolve: Precarrega dades abans d'activar una ruta.

Implementació de Guàrdies de Ruta

Guàrdia CanActivate

La guàrdia CanActivate s'utilitza per evitar que usuaris no autoritzats accedeixin a certes rutes.

Exemple

import { Injectable } from '@angular/core';
import { CanActivate, Router } from '@angular/router';
import { AuthService } from './auth.service';

@Injectable({
  providedIn: 'root'
})
export class AuthGuard implements CanActivate {

  constructor(private authService: AuthService, private router: Router) {}

  canActivate(): boolean {
    if (this.authService.isLoggedIn()) {
      return true;
    } else {
      this.router.navigate(['/login']);
      return false;
    }
  }
}

En aquest exemple, la AuthGuard comprova si l'usuari ha iniciat sessió utilitzant el AuthService. Si l'usuari no ha iniciat sessió, es redirigeix a la pàgina d'inici de sessió.

Guàrdia CanActivateChild

La guàrdia CanActivateChild s'utilitza per controlar l'accés a les rutes filles.

Exemple

import { Injectable } from '@angular/core';
import { CanActivateChild, Router } from '@angular/router';
import { AuthService } from './auth.service';

@Injectable({
  providedIn: 'root'
})
export class AdminGuard implements CanActivateChild {

  constructor(private authService: AuthService, private router: Router) {}

  canActivateChild(): boolean {
    if (this.authService.isAdmin()) {
      return true;
    } else {
      this.router.navigate(['/not-authorized']);
      return false;
    }
  }
}

En aquest exemple, la AdminGuard comprova si l'usuari té privilegis d'administrador abans de permetre l'accés a les rutes filles.

Guàrdia CanDeactivate

La guàrdia CanDeactivate s'utilitza per evitar que els usuaris abandonin una ruta si no es compleixen certes condicions.

Exemple

import { Injectable } from '@angular/core';
import { CanDeactivate } from '@angular/router';
import { Observable } from 'rxjs';

export interface CanComponentDeactivate {
  canDeactivate: () => Observable<boolean> | Promise<boolean> | boolean;
}

@Injectable({
  providedIn: 'root'
})
export class UnsavedChangesGuard implements CanDeactivate<CanComponentDeactivate> {

  canDeactivate(component: CanComponentDeactivate): Observable<boolean> | Promise<boolean> | boolean {
    return component.canDeactivate ? component.canDeactivate() : true;
  }
}

En aquest exemple, la UnsavedChangesGuard comprova si hi ha canvis no desats abans de permetre que l'usuari navegui fora de la ruta.

Guàrdia CanLoad

La guàrdia CanLoad s'utilitza per evitar la càrrega de mòduls de funcions fins que es compleixin certes condicions.

Exemple

import { Injectable } from '@angular/core';
import { CanLoad, Route, UrlSegment, Router } from '@angular/router';
import { AuthService } from './auth.service';

@Injectable({
  providedIn: 'root'
})
export class AuthLoadGuard implements CanLoad {

  constructor(private authService: AuthService, private router: Router) {}

  canLoad(route: Route, segments: UrlSegment[]): boolean {
    if (this.authService.isLoggedIn()) {
      return true;
    } else {
      this.router.navigate(['/login']);
      return false;
    }
  }
}

En aquest exemple, la AuthLoadGuard comprova si l'usuari ha iniciat sessió abans de carregar un mòdul de funcions.

Guàrdia Resolve

La guàrdia Resolve s'utilitza per precarregar dades abans d'activar una ruta.

Exemple

import { Injectable } from '@angular/core';
import { Resolve, ActivatedRouteSnapshot, RouterStateSnapshot } from '@angular/router';
import { Observable } from 'rxjs';
import { DataService } from './data.service';

@Injectable({
  providedIn: 'root'
})
export class DataResolver implements Resolve<any> {

  constructor(private dataService: DataService) {}

  resolve(route: ActivatedRouteSnapshot, state: RouterStateSnapshot): Observable<any> {
    return this.dataService.getData();
  }
}

En aquest exemple, la DataResolver obté dades utilitzant el DataService abans que la ruta s'activi.

Aplicació de Guàrdies de Ruta a les Rutes

Per aplicar guàrdies de ruta a les rutes, cal configurar-les en el teu mòdul de rutes.

Exemple

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { HomeComponent } from './home/home.component';
import { AdminComponent } from './admin/admin.component';
import { LoginComponent } from './login/login.component';
import { AuthGuard } from './auth.guard';
import { AdminGuard } from './admin.guard';
import { UnsavedChangesGuard } from './unsaved-changes.guard';
import { DataResolver } from './data.resolver';

const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'admin', component: AdminComponent, canActivate: [AuthGuard, AdminGuard] },
  { path: 'login', component: LoginComponent },
  { path: 'profile', component: ProfileComponent, canDeactivate: [UnsavedChangesGuard] },
  { path: 'data', component: DataComponent, resolve: { data: DataResolver } }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

En aquest exemple, s'apliquen diferents guàrdies de ruta a diferents rutes per controlar l'accés i la càrrega de dades.

Conclusió

Les guàrdies de ruta a Angular són eines poderoses per controlar l'accés a les rutes i assegurar que els usuaris només puguin accedir a certes parts de la teva aplicació sota condicions específiques. Entenent i implementant els diversos tipus de guàrdies de ruta, pots millorar la seguretat i l'experiència d'usuari de les teves aplicacions Angular.
