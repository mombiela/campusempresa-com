El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar un projecte és el primer pas en el desenvolupament d'una aplicació Angular. Aquesta secció et guiarà a través del procés de configuració d'un projecte Angular des de zero, assegurant que tinguis totes les eines i configuracions necessàries per començar a construir la teva aplicació.

Prerequisits

Abans de configurar un projecte Angular, assegura't de tenir instal·lat el següent al teu ordinador:

  • Node.js: Angular requereix Node.js per a les seves eines de compilació i desenvolupament.
  • npm (Node Package Manager): Ve amb Node.js i s'utilitza per instal·lar Angular CLI i altres dependències.
  • Angular CLI: Una eina d'interfície de línia de comandes que ajuda a crear, gestionar i construir aplicacions Angular.

Instal·lació de Node.js i npm

  1. Descarregar i Instal·lar Node.js:

    • Visita el lloc web oficial de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Verificar la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa les següents comandes per comprovar les versions instal·lades: 
      node -v
npm -v

Instal·lació d'Angular CLI

  1. Instal·lar Angular CLI Globalment:

    • Obre el teu terminal o símbol del sistema.
    • Executa la següent comanda per instal·lar Angular CLI globalment: 
      npm install -g @angular/cli

  2. Verificar la Instal·lació:

    • Executa la següent comanda per comprovar la versió instal·lada d'Angular CLI: 
      ng version

Creació d'un Nou Projecte Angular

Amb Angular CLI instal·lat, ara pots crear un nou projecte Angular.

  1. Crear un Nou Projecte:

    • Navega al directori on vols crear el teu projecte.
    • Executa la següent comanda, substituint my-angular-app pel nom desitjat del teu projecte: 
      ng new my-angular-app

  2. Prompts de Configuració del Projecte:

    • Se't demanarà que triïs algunes opcions de configuració:
      • Would you like to add Angular routing?: Tria Yes o No depenent de si necessites routing.
      • Which stylesheet format would you like to use?: Tria el teu format de full d'estils preferit (CSS, SCSS, SASS, LESS, Stylus).

  3. Navegar al Directori del Projecte:

    • Després de crear el projecte, navega al directori del projecte: 
      cd my-angular-app

  4. Servir l'Aplicació:

    • Executa la següent comanda per iniciar el servidor de desenvolupament: 
      ng serve
    • Obre el teu navegador i navega a http://localhost:4200/ per veure la teva nova aplicació Angular en funcionament.

Estructura del Projecte

Entendre l'estructura d'un projecte Angular és crucial per a un desenvolupament efectiu. Aquí tens una visió general dels fitxers i directoris clau:

  • e2e/: Proves end-to-end per a l'aplicació.
  • node_modules/: Conté tots els paquets npm instal·lats per al projecte.
  • src/: Carpeta principal de codi font que conté el codi de l'aplicació.
    • app/: Conté els mòduls i components principals de l'aplicació.
    • assets/: Conté actius estàtics com imatges i estils.
    • environments/: Conté fitxers de configuració específics per a cada entorn.
    • index.html: El fitxer HTML principal.
    • main.ts: El punt d'entrada principal per a l'aplicació.
    • styles.css: Estils globals per a l'aplicació.
  • angular.json: Fitxer de configuració per a Angular CLI.
  • package.json: Llista les dependències i scripts del projecte.
  • tsconfig.json: Fitxer de configuració de TypeScript.

Conclusió

Configurar un projecte Angular implica instal·lar les eines necessàries, crear un nou projecte utilitzant Angular CLI i entendre l'estructura del projecte. Amb aquests passos, ara estàs preparat per començar a desenvolupar la teva aplicació Angular. En les següents seccions, aprofundirem en la construcció de components, serveis i altres parts essencials d'una aplicació Angular.
