Introducció

En aquesta secció, explorarem com construir diverses funcionalitats en Angular 2+. Cobrirem els conceptes essencials, proporcionarem exemples de codi i inclourem exercicis per ajudar-vos a entendre i implementar aquestes funcionalitats en les vostres aplicacions.

Components

Els components són els blocs de construcció de les aplicacions Angular. Controlen una part de la pantalla anomenada vista.

  • Creació d'un Component: Utilitzeu l'Angular CLI per generar un nou component.

    ng generate component my-component

  • Decorador de Component: El decorador @Component s'utilitza per definir metadades per al component.

    import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent {
  title = 'My Angular Component';
}

  • Plantilla i Estils: Definiu l'HTML i el CSS per al component.

    <!-- my-component.component.html -->
<h1>{{ title }}</h1>

    /* my-component.component.css */
h1 {
  color: blue;
}

Serveis i Injecció de Dependències

Els serveis s'utilitzen per compartir dades i funcionalitats entre components.

  • Creació d'un Servei: Utilitzeu l'Angular CLI per generar un nou servei.

    ng generate service my-service

  • Decorador de Servei: El decorador @Injectable s'utilitza per definir un servei.

    import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class MyService {
  getData() {
    return 'Service Data';
  }
}

  • Injecció d'un Servei: Utilitzeu la injecció de dependències per utilitzar el servei en un component.

    import { Component, OnInit } from '@angular/core';
import { MyService } from './my-service.service';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  data: string;

  constructor(private myService: MyService) {}

  ngOnInit(): void {
    this.data = this.myService.getData();
  }
}

Enrutament

L'enrutament us permet navegar entre diferents vistes o components.

  • Configuració de Rutes: Definiu rutes a app-routing.module.ts.

    import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { MyComponent } from './my-component/my-component.component';

const routes: Routes = [
  { path: 'my-component', component: MyComponent },
  { path: '', redirectTo: '/my-component', pathMatch: 'full' }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

  • Sortida del Router: Utilitzeu <router-outlet> a la plantilla principal per mostrar components enrutats.

    <!-- app.component.html -->
<nav>
  <a routerLink="/my-component">My Component</a>
</nav>
<router-outlet></router-outlet>

Formularis

Angular proporciona dues aproximacions per gestionar l'entrada de l'usuari a través de formularis: Formularis basats en plantilles i Formularis reactius.

  • Formularis basats en plantilles: Utilitzeu directives Angular per crear formularis.

    <!-- my-component.component.html -->
<form #myForm="ngForm" (ngSubmit)="onSubmit(myForm)">
  <input name="name" ngModel required>
  <button type="submit">Submit</button>
</form>

    // my-component.component.ts
export class MyComponent {
  onSubmit(form: any): void {
    console.log('Form Data: ', form.value);
  }
}

  • Formularis reactius: Utilitzeu FormBuilder per crear formularis programàticament.

    import { Component, OnInit } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit(): void {
    this.myForm = this.fb.group({
      name: ['', Validators.required]
    });
  }

  onSubmit(): void {
    console.log('Form Data: ', this.myForm.value);
  }
}

    <!-- my-component.component.html -->
<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <input formControlName="name" required>
  <button type="submit">Submit</button>
</form>

Client HTTP

El mòdul HttpClient d'Angular us permet comunicar-vos amb serveis de backend.

  • Configuració de HttpClient: Importeu HttpClientModule al vostre mòdul d'aplicació.

    import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule],
  // altres imports
})
export class AppModule { }

  • Realització de Peticions HTTP: Utilitzeu HttpClient per fer peticions HTTP.

    import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class MyService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  getData() {
    return this.http.get(this.apiUrl);
  }
}

    import { Component, OnInit } from '@angular/core';
import { MyService } from './my-service.service';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  data: any;

  constructor(private myService: MyService) {}

  ngOnInit(): void {
    this.myService.getData().subscribe(response => {
      this.data = response;
    });
  }
}
