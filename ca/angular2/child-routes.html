Introducció a les Rutes Filles

Les rutes filles a Angular et permeten crear rutes niades dins d'una ruta pare. Això és útil per construir aplicacions complexes amb múltiples nivells de navegació. Entendre les rutes filles és essencial per crear aplicacions Angular modulars i fàcils de mantenir.

Conceptes Clau

  • Ruta Pare: La ruta principal que conté una o més rutes filles.
  • Ruta Filla: Una ruta niada dins d'una ruta pare.
  • Mòdul de Rutes: Mòdul d'Angular que proporciona capacitats de navegació i manipulació d'URL.
  • Configuració de Rutes: Una matriu de definicions de rutes que especifiquen els camins i els components.

Configuració de les Rutes Filles

Pas 1: Definir Components Pare i Fill

Primer, crea els components que serviran com a pare i fill en la teva estructura de rutes.

// parent.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-parent',
  template: `
    <h2>Parent Component</h2>
    <router-outlet></router-outlet>
  `
})
export class ParentComponent {}

// child.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-child',
  template: `<h3>Child Component</h3>`
})
export class ChildComponent {}

Pas 2: Configurar les Rutes

A continuació, configura les rutes en el teu AppRoutingModule.

// app-routing.module.ts
import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { ParentComponent } from './parent.component';
import { ChildComponent } from './child.component';

const routes: Routes = [
  {
    path: 'parent',
    component: ParentComponent,
    children: [
      {
        path: 'child',
        component: ChildComponent
      }
    ]
  }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule {}

Pas 3: Actualitzar el Mòdul de l'Aplicació

Assegura't que els teus components i el mòdul de rutes estan declarats en el teu AppModule.

// app.module.ts
import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppRoutingModule } from './app-routing.module';
import { AppComponent } from './app.component';
import { ParentComponent } from './parent.component';
import { ChildComponent } from './child.component';

@NgModule({
  declarations: [
    AppComponent,
    ParentComponent,
    ChildComponent
  ],
  imports: [
    BrowserModule,
    AppRoutingModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule {}

Pas 4: Navegar a les Rutes Filles

Per navegar a la ruta filla, pots utilitzar la directiva de l'enllaç del router d'Angular.

<!-- app.component.html -->
<nav>
  <a routerLink="/parent">Parent</a>
  <a routerLink="/parent/child">Child</a>
</nav>
<router-outlet></router-outlet>

Resum

Les rutes filles a Angular et permeten crear estructures de navegació niades, fent la teva aplicació més modular i fàcil de gestionar. Definint components pare i fill, configurant rutes i utilitzant la directiva de l'enllaç del router, pots implementar efectivament rutes filles en la teva aplicació Angular. Entendre i utilitzar les rutes filles és crucial per construir aplicacions Angular escalables i fàcils de mantenir.
