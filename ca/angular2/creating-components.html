Introducció als Components

Els components són els blocs de construcció de qualsevol aplicació Angular. Encapsulen l'HTML, CSS i la lògica per a una part de la interfície d'usuari. Entendre com crear i utilitzar components és fonamental per dominar Angular.

Conceptes Clau

  • Classe del Component: Defineix la lògica i les dades per al component.
  • Plantilla del Component: Defineix la vista HTML per al component.
  • Estils del Component: Defineix els estils CSS per al component.
  • Metadades: Decora la classe per definir com s'ha de processar, instanciar i utilitzar en temps d'execució.

Creació d'un Component Bàsic

Guia Pas a Pas

  1. Generar un Component: Utilitza Angular CLI per generar un nou component.

    ng generate component my-component

    Aquesta comanda crea una nova carpeta amb quatre fitxers: my-component.component.ts, my-component.component.html, my-component.component.css i my-component.component.spec.ts.

  2. Classe del Component: Situada a my-component.component.ts.

    import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent {
  title = 'Hello World';
}
    • Decorador @Component: Defineix les metadades per al component.
    • selector: L'etiqueta HTML utilitzada per incloure aquest component en altres plantilles.
    • templateUrl: Ruta al fitxer de plantilla HTML.
    • styleUrls: Ruta al(s) fitxer(s) CSS.

  3. Plantilla del Component: Situada a my-component.component.html.

    <h1>{{ title }}</h1>
    • Interpolació: {{ title }} vincula la propietat title de la classe del component a l'HTML.

  4. Estils del Component: Situats a my-component.component.css.

    h1 {
  color: blue;
}

  5. Utilitzar el Component: Afegeix el selector del component a la plantilla d'un component pare.

    <app-my-component></app-my-component>

Funcionalitats Avançades del Component

Propietats d'Entrada i Sortida

Els components poden comunicar-se entre ells utilitzant els decoradors @Input i @Output.

Propietat d'Entrada

Permet que un component pare passi dades a un component fill.

// child.component.ts
import { Component, Input } from '@angular/core';

@Component({
  selector: 'app-child',
  template: `<p>{{ childMessage }}</p>`
})
export class ChildComponent {
  @Input() childMessage: string;
}
<!-- parent.component.html -->
<app-child [childMessage]="'Hello from Parent'"></app-child>

Propietat de Sortida

Permet que un component fill enviï dades a un component pare.

// child.component.ts
import { Component, Output, EventEmitter } from '@angular/core';

@Component({
  selector: 'app-child',
  template: `<button (click)="sendMessage()">Send Message</button>`
})
export class ChildComponent {
  @Output() messageEvent = new EventEmitter<string>();

  sendMessage() {
    this.messageEvent.emit('Hello Parent');
  }
}
<!-- parent.component.html -->
<app-child (messageEvent)="receiveMessage($event)"></app-child>
<p>{{ message }}</p>
// parent.component.ts
export class ParentComponent {
  message: string;

  receiveMessage($event: string) {
    this.message = $event;
  }
}

Hooks de Cicle de Vida

Angular proporciona hooks de cicle de vida que et permeten accedir a esdeveniments clau en el cicle de vida d'un component.

Hooks de Cicle de Vida Comuns

  • ngOnInit: Es crida una vegada que el component està inicialitzat.
  • ngOnChanges: Es crida quan una propietat d'entrada canvia.
  • ngOnDestroy: Es crida just abans que el component sigui destruït.
import { Component, OnInit, OnChanges, OnDestroy, Input } from '@angular/core';

@Component({
  selector: 'app-lifecycle',
  template: `<p>Lifecycle Component</p>`
})
export class LifecycleComponent implements OnInit, OnChanges, OnDestroy {
  @Input() data: string;

  ngOnInit() {
    console.log('Component Initialized');
  }

  ngOnChanges() {
    console.log('Input Property Changed');
  }

  ngOnDestroy() {
    console.log('Component Destroyed');
  }
}

Resum

Crear components a Angular implica definir una classe de component, plantilla i estils. Els components poden comunicar-se entre ells utilitzant propietats d'entrada i sortida. Entendre els hooks de cicle de vida et permet gestionar el cicle de vida del component de manera efectiva. Dominar aquests conceptes és essencial per construir aplicacions Angular robustes i mantenibles.
