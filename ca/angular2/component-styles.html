A Angular, els estils de components són una part crucial per crear aplicacions visualment atractives i mantenibles. Aquest tema cobrirà com definir i gestionar els estils per als components d'Angular, des de conceptes bàsics fins a tècniques avançades.

Introducció als Estils de Components

A Angular, cada component pot tenir els seus propis estils, cosa que ajuda a encapsular l'aspecte i la sensació del component. Això assegura que els estils no es filtrin i afectin altres parts de l'aplicació.

  • Encapsulació: Angular proporciona mecanismes per encapsular estils dins d'un component.
  • URLs d'estil: Pots enllaçar fulls d'estil externs a un component.
  • Estils en línia: Pots definir estils directament dins del component.

Definició d'Estils de Components

Estils en Línia

Els estils en línia es defineixen directament dins de les metadades del component. Això és útil per a petites quantitats d'estilització.

import { Component } from '@angular/core';

@Component({
  selector: 'app-inline-style',
  template: `<h1>Hello, Inline Styles!</h1>`,
  styles: [`
    h1 {
      color: blue;
    }
  `]
})
export class InlineStyleComponent {}

Fulls d'Estil Externs

Per a projectes més grans, és més mantenible mantenir els estils en fitxers separats. Pots enllaçar aquests fulls d'estil a les metadades del component.

import { Component } from '@angular/core';

@Component({
  selector: 'app-external-style',
  template: `<h1>Hello, External Styles!</h1>`,
  styleUrls: ['./external-style.component.css']
})
export class ExternalStyleComponent {}

En external-style.component.css:

h1 {
  color: green;
}

Encapsulació d'Estils

Angular proporciona tres modes d'encapsulació:

  • Emulated (per defecte): Els estils es delimiten al component mitjançant una combinació d'atributs.
  • None: Els estils no es delimiten i afectaran tota l'aplicació.
  • Shadow DOM: Utilitza el Shadow DOM natiu del navegador per encapsular els estils.

Encapsulació Emulada

Aquest és el mode per defecte i és adequat per a la majoria dels casos d'ús.

import { Component, ViewEncapsulation } from '@angular/core';

@Component({
  selector: 'app-emulated-style',
  template: `<h1>Hello, Emulated Styles!</h1>`,
  styles: [`
    h1 {
      color: red;
    }
  `],
  encapsulation: ViewEncapsulation.Emulated
})
export class EmulatedStyleComponent {}

Encapsulació None

Utilitza aquest mode si vols que els estils siguin globals.

import { Component, ViewEncapsulation } from '@angular/core';

@Component({
  selector: 'app-none-style',
  template: `<h1>Hello, No Encapsulation!</h1>`,
  styles: [`
    h1 {
      color: purple;
    }
  `],
  encapsulation: ViewEncapsulation.None
})
export class NoneStyleComponent {}

Encapsulació Shadow DOM

Aquest mode aprofita l'API Shadow DOM del navegador per a una veritable encapsulació.

import { Component, ViewEncapsulation } from '@angular/core';

@Component({
  selector: 'app-shadow-dom-style',
  template: `<h1>Hello, Shadow DOM!</h1>`,
  styles: [`
    h1 {
      color: orange;
    }
  `],
  encapsulation: ViewEncapsulation.ShadowDom
})
export class ShadowDomStyleComponent {}

Tècniques Avançades d'Estilització

Selector Host

El selector :host et permet estilitzar l'element host del component.

import { Component } from '@angular/core';

@Component({
  selector: 'app-host-style',
  template: `<h1>Hello, Host Selector!</h1>`,
  styles: [`
    :host {
      display: block;
      border: 1px solid black;
    }
  `]
})
export class HostStyleComponent {}

Context Host

El selector :host-context s'utilitza per aplicar estils basats en el context en què s'utilitza el component.

import { Component } from '@angular/core';

@Component({
  selector: 'app-host-context-style',
  template: `<h1>Hello, Host Context!</h1>`,
  styles: [`
    :host-context(.theme-dark) h1 {
      color: white;
      background-color: black;
    }
  `]
})
export class HostContextStyleComponent {}

Resum

En aquest tema, hem cobert els conceptes bàsics i les tècniques avançades d'estilització de components Angular. Hem après com definir estils en línia i mitjançant fulls d'estil externs, i hem explorat els diferents modes d'encapsulació proporcionats per Angular. A més, hem aprofundit en tècniques avançades d'estilització utilitzant els selectors :host i :host-context.

Entendre i utilitzar efectivament els estils de components és essencial per crear aplicacions Angular mantenibles i visualment atractives. Encapsulant els estils dins dels components, pots assegurar-te que l'aspecte i la sensació de la teva aplicació siguin consistents i modulars.
