Introducció

En aquesta secció, explorarem com es pot utilitzar Dart conjuntament amb CSS per estilitzar aplicacions web. Dart, un llenguatge de programació desenvolupat per Google, s'utilitza sovint per construir aplicacions web, de servidor i mòbils. Quan es construeixen aplicacions web, l'estilització és crucial per crear interfícies visualment atractives i fàcils d'usar. CSS (Cascading Style Sheets) és el llenguatge estàndard per descriure la presentació de pàgines web. Combinant Dart i CSS, els desenvolupadors poden crear aplicacions web dinàmiques i ben estilitzades.

Conceptes Clau

  • Dart per al Desenvolupament Web: Dart es pot compilar a JavaScript, permetent que s'executi al navegador.
  • Conceptes Bàsics de CSS: Comprendre selectors, propietats i valors en CSS.
  • Integració de CSS amb Dart: Mètodes per aplicar estils CSS dins d'aplicacions Dart.
  • Estilització Dinàmica: Utilitzar Dart per manipular estils CSS dinàmicament.

Dart per al Desenvolupament Web

Dart es pot utilitzar per construir aplicacions web compilant el codi Dart a JavaScript. Això permet que les aplicacions Dart s'executin en qualsevol navegador web modern.

Exemple: Aplicació Web Bàsica amb Dart

import 'dart:html';

void main() {
  querySelector('#output').text = 'Hola, Dart!';
}

En aquest exemple, el codi Dart s'utilitza per seleccionar un element HTML amb l'id output i canviar el seu contingut de text a "Hola, Dart!".

Conceptes Bàsics de CSS

CSS s'utilitza per estilitzar elements HTML. Consisteix en selectors i declaracions. Un bloc de declaració conté una o més declaracions separades per punts i coma. Cada declaració inclou un nom de propietat CSS i un valor, separats per dos punts.

Exemple: CSS Bàsic

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

En aquest exemple, l'element body té un color de fons blau clar, i els elements h1 estan estilitzats amb un color blau marí i un marge esquerre de 20 píxels.

Integració de CSS amb Dart

Hi ha diverses maneres d'integrar CSS amb Dart:

  1. Estils Inline: Afegir estils directament als elements HTML.
  2. Fulls d'Estil Externs: Enllaçar a un fitxer CSS extern.
  3. Codi Dart: Utilitzar Dart per manipular estils.

Exemple: Estils Inline

<!DOCTYPE html>
<html>
  <head>
    <title>Dart i CSS</title>
  </head>
  <body>
    <h1 style="color: navy; margin-left: 20px;">Hola, Dart!</h1>
    <script defer src="main.dart.js"></script>
  </body>
</html>

En aquest exemple, l'element h1 està estilitzat directament utilitzant l'atribut style.

Exemple: Fulls d'Estil Externs

<!DOCTYPE html>
<html>
  <head>
    <title>Dart i CSS</title>
    <link rel="stylesheet" type="text/css" href="styles.css">
  </head>
  <body>
    <h1>Hola, Dart!</h1>
    <script defer src="main.dart.js"></script>
  </body>
</html>
/* styles.css */
h1 {
  color: navy;
  margin-left: 20px;
}

En aquest exemple, l'element h1 està estilitzat utilitzant un fitxer CSS extern.

Exemple: Codi Dart

import 'dart:html';

void main() {
  querySelector('h1').style
    ..color = 'navy'
    ..marginLeft = '20px';
}

En aquest exemple, el codi Dart s'utilitza per aplicar estils a l'element h1.

Estilització Dinàmica

Dart es pot utilitzar per canviar dinàmicament els estils CSS basant-se en interaccions de l'usuari o altres esdeveniments.

Exemple: Estilització Dinàmica amb Dart

import 'dart:html';

void main() {
  ButtonElement button = querySelector('#changeColor');
  button.onClick.listen((event) {
    querySelector('h1').style.color = 'red';
  });
}
<!DOCTYPE html>
<html>
  <head>
    <title>Dart i CSS</title>
    <link rel="stylesheet" type="text/css" href="styles.css">
  </head>
  <body>
    <h1>Hola, Dart!</h1>
    <button id="changeColor">Canvia el Color</button>
    <script defer src="main.dart.js"></script>
  </body>
</html>

En aquest exemple, fer clic al botó canvia el color de l'element h1 a vermell.

Conclusió

Combinant Dart i CSS permet als desenvolupadors crear aplicacions web dinàmiques i ben estilitzades. Comprenent els conceptes bàsics de CSS i com integrar-lo amb Dart, pots millorar l'atractiu visual i la interactivitat de les teves aplicacions web. Ja sigui utilitzant estils inline, fulls d'estil externs o manipulant estils a través de codi Dart, les possibilitats són vastes per crear experiències d'usuari atractives.
