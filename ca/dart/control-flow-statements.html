Sentències de control de flux en Dart
Funcions ►

Les sentències de control de flux són fonamentals en la programació ja que et permeten dictar l'ordre en què s'executen les sentències. En Dart, les sentències de control de flux inclouen condicionals, bucles i sentències de ramificació. Aquesta secció cobrirà aquests conceptes des d'un nivell principiant fins a un nivell avançat.

Condicionals

Els condicionals s'utilitzen per executar codi basat en certes condicions.

Sentències if i else

La sentència if et permet executar un bloc de codi si una condició és certa. La sentència else es pot utilitzar per executar un bloc de codi si la condició és falsa.

void main() {
  int number = 10;

  if (number > 0) {
    print('El número és positiu.');
  } else {
    print('El número no és positiu.');
  }
}

Sentència else if

La sentència else if et permet comprovar múltiples condicions.

void main() {
  int number = 0;

  if (number > 0) {
    print('El número és positiu.');
  } else if (number == 0) {
    print('El número és zero.');
  } else {
    print('El número és negatiu.');
  }
}

Operador ternari

L'operador ternari és una forma abreujada per a les sentències if-else.

void main() {
  int number = 10;
  String result = number > 0 ? 'Positiu' : 'No positiu';
  print(result);
}

Switch Case

La sentència switch et permet executar un bloc de codi entre moltes opcions.

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excel·lent');
      break;
    case 'B':
      print('Bo');
      break;
    case 'C':
      print('Regular');
      break;
    case 'D':
      print('Pobre');
      break;
    default:
      print('Suspès');
  }
}

Bucles

Els bucles s'utilitzen per executar un bloc de codi múltiples vegades.

Bucle for

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una sentència o un bloc de sentències.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteració $i');
  }
}

Bucle while

El bucle while s'utilitza quan vols executar un bloc de codi mentre una condició sigui certa.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteració $i');
    i++;
  }
}

Bucle do-while

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

void main() {
  int i = 0;

  do {
    print('Iteració $i');
    i++;
  } while (i < 5);
}

Sentències de ramificació

Les sentències de ramificació et permeten canviar el flux d'execució.

Sentència break

La sentència break s'utilitza per sortir d'un bucle o d'una sentència switch.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      break;
    }
    print('Iteració $i');
  }
}

Sentència continue

La sentència continue salta l'iteració actual d'un bucle i procedeix a la següent iteració.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      continue;
    }
    print('Iteració $i');
  }
}

Conclusió

Les sentències de control de flux són essencials per dirigir l'execució del teu codi. Entendre com utilitzar condicionals, bucles i sentències de ramificació et permetrà escriure programes en Dart més complexos i eficients. Practica aquests conceptes amb diversos exemples per convertir-te en un expert en el control del flux de les teves aplicacions en Dart.
