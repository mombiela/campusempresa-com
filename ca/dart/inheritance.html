Herència a Dart
Mixins ►

L'herència és un concepte fonamental en la programació orientada a objectes que permet a una classe heretar propietats i mètodes d'una altra classe. Això ajuda a la reutilització del codi i a establir una jerarquia natural entre classes.

Conceptes Clau

  • Classe Pare (Superclass): La classe les propietats i mètodes de la qual són heretats.
  • Classe Filla (Subclass): La classe que hereta d'una altra classe.
  • Paraula clau extends: Utilitzada per indicar que una classe està heretant d'una altra classe.
  • Sobreescriptura de Mètodes: La capacitat d'una subclasse per proporcionar una implementació específica d'un mètode que ja està definit a la seva classe pare.
  • Paraula clau super: Utilitzada per referir-se a la classe pare i accedir a les seves propietats i mètodes.

Herència Bàsica

A Dart, pots crear una subclasse utilitzant la paraula clau extends. Aquí tens un exemple senzill:

class Animal {
  void makeSound() {
    print('Animal fa un so');
  }
}

class Dog extends Animal {
  void bark() {
    print('El gos borda');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); // Mètode heretat
  dog.bark(); // Mètode de la subclasse
}

Explicació

  • Classe Animal: Aquesta és la classe pare amb un mètode makeSound.
  • Classe Dog: Aquesta és la subclasse que extén Animal i afegeix un nou mètode bark.
  • Funció main: Crea una instància de Dog i crida tant el mètode heretat makeSound com el mètode bark.

Sobreescriptura de Mètodes

Una subclasse pot sobreescriure un mètode de la seva classe pare per proporcionar una implementació específica. Utilitza l'anotació @override per indicar que estàs sobreescrivint un mètode.

class Animal {
  void makeSound() {
    print('Animal fa un so');
  }
}

class Dog extends Animal {
  @override
  void makeSound() {
    print('El gos borda');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); // Resultat: El gos borda
}

Explicació

  • Anotació @override: Indica que el mètode makeSound a Dog està sobreescrivint el mètode a Animal.
  • Resultat: Quan es crida makeSound en un objecte Dog, s'executa el mètode sobreescrit a Dog.

Utilitzant la Paraula Clau `super`

La paraula clau super s'utilitza per referir-se a la classe pare i es pot utilitzar per cridar els seus mètodes i constructors.

class Animal {
  void makeSound() {
    print('Animal fa un so');
  }
}

class Dog extends Animal {
  @override
  void makeSound() {
    super.makeSound(); // Crida el mètode de la classe pare
    print('El gos borda');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); 
  // Resultat:
  // Animal fa un so
  // El gos borda
}

Explicació

  • super.makeSound(): Crida el mètode makeSound de la classe pare Animal abans d'executar el mètode makeSound a Dog.

Constructors i Herència

Quan es crea una instància d'una subclasse, el constructor de la classe pare es crida primer. Pots cridar explícitament un constructor de la classe pare utilitzant la paraula clau super.

class Animal {
  String name;
  
  Animal(this.name);
  
  void display() {
    print('Animal: $name');
  }
}

class Dog extends Animal {
  String breed;
  
  Dog(String name, this.breed) : super(name);
  
  @override
  void display() {
    super.display();
    print('Raça: $breed');
  }
}

void main() {
  Dog dog = Dog('Buddy', 'Golden Retriever');
  dog.display();
  // Resultat:
  // Animal: Buddy
  // Raça: Golden Retriever
}

Explicació

  • Constructor d'Animal: Pren un paràmetre name.
  • Constructor de Dog: Pren els paràmetres name i breed i crida el constructor de la classe pare amb super(name).
  • Mètode display: Sobreescrit a Dog per incloure informació addicional sobre la raça.

Resum

L'herència a Dart et permet crear una jerarquia de classes on les subclasses hereten propietats i mètodes de les seves classes pare. Els conceptes clau inclouen la paraula clau extends per definir l'herència, la sobreescriptura de mètodes, i la paraula clau super per accedir als mètodes i constructors de la classe pare. Entendre aquests conceptes és crucial per escriure codi reutilitzable i mantenible a Dart.
