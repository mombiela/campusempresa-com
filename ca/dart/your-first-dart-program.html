Introducció

Benvingut a la teva primera lliçó de programació en Dart! Dart és un llenguatge optimitzat per a clients per a aplicacions ràpides en qualsevol plataforma. En aquesta secció, aprendràs a escriure, executar i entendre un programa bàsic en Dart. Al final d'aquesta lliçó, tindràs una base sòlida per construir a mesura que progressis en el curs.

Configuració del teu entorn

Abans d'escriure el teu primer programa Dart, necessites configurar el teu entorn de desenvolupament. Segueix aquests passos:

  1. Instal·la el Dart SDK: Descarrega i instal·la el Dart SDK des del lloc web oficial de Dart.
  2. Tria un IDE: Pots utilitzar qualsevol editor de text, però IDEs com Visual Studio Code o IntelliJ IDEA ofereixen un gran suport per a Dart.
  3. Instal·la el plugin de Dart: Si estàs utilitzant un IDE, assegura't d'instal·lar el plugin de Dart per a la ressaltació de sintaxi i altres característiques.

Escriure el teu primer programa Dart

Escrivim un programa simple en Dart que imprimeixi "Hello, World!" a la consola.

Exemple de codi

Crea un nou fitxer anomenat main.dart i afegeix el següent codi:

void main() {
  print('Hello, World!');
}

Explicació

  • void main() { ... }: Aquesta és la funció principal, el punt d'entrada de cada aplicació Dart. La paraula clau void indica que la funció no retorna cap valor.
  • print('Hello, World!');: Aquesta línia imprimeix la cadena 'Hello, World!' a la consola. La funció print és una funció incorporada en Dart utilitzada per a la sortida.

Executar el teu programa Dart

Per executar el teu programa Dart, obre un terminal o símbol del sistema i navega al directori on es troba el teu fitxer main.dart. A continuació, executa la següent comanda:

dart run main.dart

Hauries de veure la sortida:

Hello, World!

Conceptes clau

  • Funció principal: El punt de partida d'un programa Dart.
  • Funció Print: Utilitzada per a mostrar text a la consola.
  • Comentaris: Utilitza // per a comentaris d'una sola línia i /* ... */ per a comentaris de múltiples línies.

Exemple amb comentaris

// Aquest és un comentari d'una sola línia

/*
 Aquest és un comentari de múltiples línies
 que s'estén per diverses línies.
*/

void main() {
  // Imprimeix una salutació a la consola
  print('Hello, World!');
}

Resum

En aquesta lliçó, has après a:

  • Configurar el teu entorn de desenvolupament Dart.
  • Escriure un programa simple en Dart.
  • Entendre l'estructura d'un programa bàsic en Dart.
  • Executar el teu programa Dart i veure la sortida.

Felicitats per escriure el teu primer programa Dart! Aquest coneixement fonamental serà crucial a mesura que avancis cap a temes més complexos. Continua practicant, i aviat estaràs construint aplicacions més avançades amb Dart.
