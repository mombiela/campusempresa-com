Generadors en Dart
Gestió d'Excepcions ►

Els generadors són una característica potent en Dart que permet produir una seqüència de valors de manera mandrosa, és a dir, els valors es generen sota demanda en lloc de tots alhora. Això pot ser particularment útil quan es treballa amb grans conjunts de dades o fluxos de dades.

Conceptes Clau

  • Generadors: Funcions que produeixen una seqüència de valors.
  • Generadors Sincrònics: Utilitzen sync* i yield per produir valors.
  • Generadors Asincrònics: Utilitzen async* i yield per produir valors de manera asincrònica.
  • yield: Paraula clau utilitzada per produir un valor en una funció generadora.
  • *yield`: Paraula clau utilitzada per delegar part del procés de generació a un altre generador.

Generadors Sincrònics

Els generadors sincrònics s'utilitzen per produir valors en una seqüència que es pot iterar de manera sincrònica.

Exemple

Iterable<int> syncGenerator(int n) sync* {
  for (int i = 0; i < n; i++) {
    yield i;
  }
}

void main() {
  var numbers = syncGenerator(5);
  for (var number in numbers) {
    print(number);
  }
}

Explicació

  • La funció syncGenerator és un generador sincrònic que produeix enters des de 0 fins a n-1.
  • La paraula clau sync* indica que aquesta funció és un generador sincrònic.
  • La paraula clau yield s'utilitza per produir cada valor en la seqüència.
  • En la funció main, cridem syncGenerator i iterem sobre els valors produïts utilitzant un bucle for.

Generadors Asincrònics

Els generadors asincrònics s'utilitzen per produir valors en una seqüència que es pot iterar de manera asincrònica, cosa que és útil per gestionar fluxos de dades o realitzar operacions asincròniques.

Exemple

Stream<int> asyncGenerator(int n) async* {
  for (int i = 0; i < n; i++) {
    await Future.delayed(Duration(seconds: 1)); // Simular operació asincrònica
    yield i;
  }
}

void main() async {
  var numbers = asyncGenerator(5);
  await for (var number in numbers) {
    print(number);
  }
}

Explicació

  • La funció asyncGenerator és un generador asincrònic que produeix enters des de 0 fins a n-1.
  • La paraula clau async* indica que aquesta funció és un generador asincrònic.
  • La paraula clau yield s'utilitza per produir cada valor en la seqüència.
  • La paraula clau await s'utilitza per simular una operació asincrònica (per exemple, un retard).
  • En la funció main, cridem asyncGenerator i iterem sobre els valors produïts utilitzant un bucle await for.

Paraula Clau yield*

La paraula clau yield* s'utilitza per delegar part del procés de generació a un altre generador.

Exemple

Iterable<int> subGenerator(int n) sync* {
  for (int i = 0; i < n; i++) {
    yield i;
  }
}

Iterable<int> mainGenerator(int n) sync* {
  yield* subGenerator(n);
  yield n;
}

void main() {
  var numbers = mainGenerator(5);
  for (var number in numbers) {
    print(number);
  }
}

Explicació

  • La funció subGenerator produeix enters des de 0 fins a n-1.
  • La funció mainGenerator utilitza yield* per delegar la generació de valors a subGenerator i després produeix un valor addicional n.
  • En la funció main, cridem mainGenerator i iterem sobre els valors produïts utilitzant un bucle for.

Conclusió

Els generadors en Dart proporcionen una manera flexible i eficient de produir seqüències de valors, ja sigui de manera sincrònica o asincrònica. Utilitzant sync* i yield per a generadors sincrònics, i async* i yield per a generadors asincrònics, es poden gestionar una àmplia gamma d'escenaris de generació de dades. La paraula clau yield* millora encara més aquesta capacitat permetent la delegació a altres generadors, fent que el teu codi sigui més modular i fàcil de gestionar. Entendre i utilitzar generadors pot millorar molt el rendiment i la llegibilitat dels teus programes en Dart, especialment quan es tracta de grans conjunts de dades o fluxos de dades asincrònics.
