Fluxos en Dart
Generadors ►

Els fluxos són una part fonamental de la programació asíncrona en Dart. Et permeten treballar amb seqüències d'esdeveniments asíncrons, com ara dades rebudes d'un servidor web o l'entrada de l'usuari.

Què són els fluxos?

  • Definició: Un flux és una seqüència d'esdeveniments asíncrons. Cada esdeveniment és o bé un esdeveniment de dades o un esdeveniment d'error.
  • Usos: Els fluxos són útils per gestionar múltiples esdeveniments asíncrons al llarg del temps, com ara:
    • Rebre dades d'un servidor web.
    • Gestionar l'entrada de l'usuari.
    • Escoltar actualitzacions en temps real.

Tipus de fluxos

Dart proporciona dos tipus principals de fluxos:

  • Fluxos de subscripció única: Aquests fluxos només es poden escoltar una vegada. Són adequats per a esdeveniments que només ocorren una vegada o que necessiten ser processats en un ordre específic.
  • Fluxos de difusió: Aquests fluxos es poden escoltar múltiples vegades. Són adequats per a esdeveniments que poden ser processats per múltiples oients de manera independent.

Crear fluxos

Flux de subscripció única

Pots crear un flux de subscripció única utilitzant la classe Stream.

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() async {
  await for (var value in countStream(5)) {
    print(value); // Imprimeix 1, 2, 3, 4, 5
  }
}

Flux de difusió

Pots crear un flux de difusió utilitzant la classe StreamController.

import 'dart:async';

void main() {
  final controller = StreamController<int>.broadcast();

  controller.stream.listen((data) {
    print('Oient 1: $data');
  });

  controller.stream.listen((data) {
    print('Oient 2: $data');
  });

  for (int i = 1; i <= 3; i++) {
    controller.add(i); // Afegeix dades al flux
  }

  controller.close();
}

Escoltar fluxos

  • await for: Utilitza aquesta sintaxi per escoltar un flux de manera asíncrona.
  • listen(): Utilitza aquest mètode per gestionar esdeveniments del flux amb callbacks.

Utilitzant `await for`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() async {
  await for (var value in countStream(5)) {
    print(value); // Imprimeix 1, 2, 3, 4, 5
  }
}

Utilitzant `listen()`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() {
  countStream(5).listen((value) {
    print(value); // Imprimeix 1, 2, 3, 4, 5
  });
}

Transformar fluxos

Els fluxos es poden transformar utilitzant diversos mètodes com map(), where(), expand(), i reduce().

Exemple: Utilitzant `map()`

Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    yield i;
  }
}

void main() {
  countStream(5).map((value) => value * 2).listen((value) {
    print(value); // Imprimeix 2, 4, 6, 8, 10
  });
}

Gestionar errors en fluxos

Els fluxos poden emetre esdeveniments d'error. Pots gestionar aquests errors utilitzant el paràmetre onError en el mètode listen().

Stream<int> errorStream() async* {
  yield 1;
  throw Exception('S'ha produït un error');
  yield 2;
}

void main() {
  errorStream().listen(
    (value) {
      print(value); // Imprimeix 1
    },
    onError: (error) {
      print('Error: $error'); // Imprimeix 'Error: Exception: S'ha produït un error'
    },
  );
}

Resum

Els fluxos en Dart són una eina poderosa per gestionar seqüències asíncrones d'esdeveniments. Hi ha dos tipus principals: fluxos de subscripció única i fluxos de difusió. Pots crear fluxos utilitzant la classe Stream o StreamController. Escoltar fluxos es pot fer utilitzant await for o listen(). Els fluxos es poden transformar i els errors es poden gestionar de manera elegant. Entendre els fluxos és essencial per a una programació asíncrona efectiva en Dart.
