Introducció a les Llistes

Les llistes són una estructura de dades fonamental a Dart, utilitzada per emmagatzemar col·leccions ordenades d'elements. Són similars als arrays en altres llenguatges de programació però ofereixen més flexibilitat i funcionalitat.

Conceptes Clau

  • Inicialització de Llistes: Crear llistes utilitzant diferents mètodes.
  • Accés als Elements: Recuperar elements d'una llista.
  • Modificació de Llistes: Afegir, eliminar i actualitzar elements.
  • Iteració sobre Llistes: Utilitzar bucles per recórrer llistes.
  • Mètodes Comuns de Llistes: Mètodes integrats útils per a la manipulació de llistes.

Inicialització de Llistes

A Dart, les llistes es poden crear de diverses maneres. Aquí hi ha alguns mètodes comuns:

Utilitzant Literals de Llista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  print(numbers); // Output: [1, 2, 3, 4, 5]
}
  • Explicació: Això crea una llista d'enters utilitzant un literal de llista.

Utilitzant el Constructor de Llista

void main() {
  List<String> fruits = List<String>.filled(3, 'Apple');
  print(fruits); // Output: [Apple, Apple, Apple]
}
  • Explicació: Això crea una llista de cadenes amb una longitud fixa de 3, totes inicialitzades a 'Apple'.

Accés als Elements

Pots accedir als elements d'una llista utilitzant el seu índex, que comença a 0.

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits[1]); // Output: Banana
}
  • Explicació: Això recupera l'element a l'índex 1 de la llista.

Modificació de Llistes

Afegir Elements

  • Utilitzant el Mètode add:
void main() {
  List<int> numbers = [1, 2, 3];
  numbers.add(4);
  print(numbers); // Output: [1, 2, 3, 4]
}

  • Explicació: Afegeix un sol element al final de la llista.

  • Utilitzant el Mètode addAll:

void main() {
  List<int> numbers = [1, 2, 3];
  numbers.addAll([4, 5, 6]);
  print(numbers); // Output: [1, 2, 3, 4, 5, 6]
}
  • Explicació: Afegeix múltiples elements al final de la llista.

Eliminar Elements

  • Utilitzant el Mètode remove:
void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits.remove('Banana');
  print(fruits); // Output: [Apple, Cherry]
}

  • Explicació: Elimina la primera ocurrència de l'element especificat.

  • Utilitzant el Mètode removeAt:

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits.removeAt(1);
  print(fruits); // Output: [Apple, Cherry]
}
  • Explicació: Elimina l'element a l'índex especificat.

Actualitzar Elements

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  fruits[1] = 'Blueberry';
  print(fruits); // Output: [Apple, Blueberry, Cherry]
}
  • Explicació: Actualitza l'element a l'índex 1 a 'Blueberry'.

Iteració sobre Llistes

Utilitzant un Bucle `for`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  for (int i = 0; i < fruits.length; i++) {
    print(fruits[i]);
  }
}
  • Explicació: Itera sobre la llista utilitzant un bucle for tradicional.

Utilitzant un Bucle `for-in`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  for (var fruit in fruits) {
    print(fruit);
  }
}
  • Explicació: Itera sobre la llista utilitzant un bucle for-in, que és més concís.

Mètodes Comuns de Llistes

`contains`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits.contains('Banana')); // Output: true
}
  • Explicació: Comprova si la llista conté l'element especificat.

`indexOf`

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  print(fruits.indexOf('Cherry')); // Output: 2
}
  • Explicació: Retorna l'índex de la primera ocurrència de l'element especificat.

`sort`

void main() {
  List<int> numbers = [3, 1, 4, 1, 5, 9];
  numbers.sort();
  print(numbers); // Output: [1, 1, 3, 4, 5, 9]
}
  • Explicació: Ordena la llista en ordre ascendent.

`reversed`

void main() {
  List<int> numbers = [1, 2, 3];
  var reversedNumbers = numbers.reversed;
  print(reversedNumbers); // Output: (3, 2, 1)
}
  • Explicació: Retorna un iterable amb els elements en ordre invers.

Conclusió

Les llistes són una part versàtil i essencial de la programació a Dart. Permeten emmagatzemar i manipular col·leccions ordenades d'elements de manera eficient. Entenent com inicialitzar, accedir, modificar i iterar sobre les llistes, així com utilitzant els mètodes comuns de llista, pots manejar una àmplia gamma de tasques de programació de manera més efectiva. Practica aquests conceptes amb diversos exemples per esdevenir proficient en l'ús de llistes a Dart.
