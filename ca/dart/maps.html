Introducció als Mapes

Els mapes són una col·lecció de parells clau-valor on cada clau és única. Són similars als diccionaris en altres llenguatges de programació. Els mapes són útils quan necessites associar valors amb claus i recuperar-los de manera eficient.

  • Conceptes Clau:
    • Clau: Un identificador únic per a un valor.
    • Valor: Les dades associades amb una clau.
    • Mapa: Una col·lecció de parells clau-valor.

Creació de Mapes

Pots crear mapes en Dart utilitzant la classe Map o la sintaxi literal de mapes.

Utilitzant la Classe Map

void main() {
  Map<String, int> edats = Map();
  edats['Alice'] = 25;
  edats['Bob'] = 30;
  print(edats); // Sortida: {Alice: 25, Bob: 30}
}

Utilitzant Literals de Mapes

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  print(edats); // Sortida: {Alice: 25, Bob: 30}
}

Accés als Elements del Mapa

Pots accedir als elements del mapa utilitzant les seves claus.

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  print(edats['Alice']); // Sortida: 25
}

Modificació de Mapes

Pots afegir, actualitzar o eliminar elements d'un mapa.

Afegir Elements

void main() {
  var edats = {
    'Alice': 25,
  };
  edats['Bob'] = 30;
  print(edats); // Sortida: {Alice: 25, Bob: 30}
}

Actualitzar Elements

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  edats['Alice'] = 26;
  print(edats); // Sortida: {Alice: 26, Bob: 30}
}

Eliminar Elements

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  edats.remove('Bob');
  print(edats); // Sortida: {Alice: 25}
}

Iteració sobre Mapes

Pots iterar sobre les claus, valors o parells clau-valor en un mapa.

Iteració sobre Claus

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var clau in edats.keys) {
    print(clau); // Sortida: Alice Bob
  }
}

Iteració sobre Valors

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var valor in edats.values) {
    print(valor); // Sortida: 25 30
  }
}

Iteració sobre Parells Clau-Valor

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var entrada in edats.entries) {
    print('${entrada.key}: ${entrada.value}'); // Sortida: Alice: 25 Bob: 30
  }
}

Operacions Avançades amb Mapes

Comprovar si hi ha una Clau o Valor

Pots comprovar si un mapa conté una clau o valor específic.

void main() {
  var edats = {
    'Alice': 25,
    'Bob': 30,
  };
  print(edats.containsKey('Alice')); // Sortida: true
  print(edats.containsValue(30)); // Sortida: true
}

Fusionar Mapes

Pots fusionar dos mapes utilitzant el mètode addAll.

void main() {
  var edats1 = {
    'Alice': 25,
  };
  var edats2 = {
    'Bob': 30,
  };
  edats1.addAll(edats2);
  print(edats1); // Sortida: {Alice: 25, Bob: 30}
}

Conclusió

Els mapes són una manera potent i flexible de gestionar parells clau-valor en Dart. Permeten una recuperació i manipulació eficient de dades. Entendre com crear, accedir, modificar i iterar sobre mapes és essencial per a una programació efectiva en Dart. En dominar aquests conceptes, pots gestionar estructures de dades complexes i millorar el rendiment de les teves aplicacions.
