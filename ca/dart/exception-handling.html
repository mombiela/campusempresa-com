Introducció a la Gestió d'Excepcions

La gestió d'excepcions és un aspecte crucial del desenvolupament de programari robust. Permet als desenvolupadors gestionar i respondre a esdeveniments inesperats o errors que es produeixen durant l'execució d'un programa. A Dart, la gestió d'excepcions es realitza utilitzant els blocs try, catch i finally.

Conceptes Clau

  • Excepció: Un esdeveniment que interromp el flux normal d'un programa.
  • bloc try: Conté codi que podria llançar una excepció.
  • bloc catch: Gestiona l'excepció.
  • bloc finally: Executa codi independentment de si s'ha llançat una excepció o no.
  • throw: Utilitzat per llançar una excepció.

Gestió Bàsica d'Excepcions

A Dart, pots gestionar excepcions utilitzant els blocs try i catch.

Exemple

void main() {
  try {
    int result = 12 ~/ 0; // Això llançarà una excepció
    print(result);
  } catch (e) {
    print('S'ha produït un error: $e');
  }
}
  • Explicació:
    • El bloc try conté codi que podria llançar una excepció.
    • El bloc catch captura l'excepció i imprimeix un missatge d'error.

Captura d'Excepcions Específiques

Pots capturar tipus específics d'excepcions per gestionar-les de manera diferent.

Exemple

void main() {
  try {
    int result = 12 ~/ 0; // Això llançarà una IntegerDivisionByZeroException
    print(result);
  } on IntegerDivisionByZeroException {
    print('No es pot dividir per zero!');
  } catch (e) {
    print('S'ha produït un error: $e');
  }
}
  • Explicació:
    • La paraula clau on s'utilitza per capturar excepcions específiques.
    • Si l'excepció és del tipus IntegerDivisionByZeroException, es imprimeix un missatge específic.
    • El bloc catch gestiona qualsevol altre tipus d'excepcions.

Ús del Bloc finally

El bloc finally conté codi que sempre s'executarà, independentment de si s'ha llançat una excepció o no.

Exemple

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('S'ha produït un error: $e');
  } finally {
    print('Aquest codi sempre s'executa.');
  }
}
  • Explicació:
    • El bloc finally s'executa després dels blocs try i catch, assegurant-se que el codi dins d'ell sempre s'executi.

Llançament d'Excepcions

Pots llançar excepcions utilitzant la paraula clau throw.

Exemple

void checkAge(int age) {
  if (age < 18) {
    throw Exception('L'edat ha de ser 18 o més.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('S'ha produït un error: $e');
  }
}
  • Explicació:
    • La funció checkAge llança una excepció si l'edat és menor de 18.
    • El bloc try crida a checkAge, i el bloc catch gestiona l'excepció.

Excepcions Personalitzades

Pots crear excepcions personalitzades estenent la classe Exception.

Exemple

class AgeException implements Exception {
  String cause;
  AgeException(this.cause);
}

void checkAge(int age) {
  if (age < 18) {
    throw AgeException('L'edat ha de ser 18 o més.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('S'ha produït un error: $e');
  }
}
  • Explicació:
    • La classe AgeException implementa la interfície Exception.
    • La funció checkAge llança una AgeException si l'edat és menor de 18.
    • El bloc catch gestiona l'excepció personalitzada.

Resum

La gestió d'excepcions a Dart és una característica poderosa que ajuda a gestionar errors i esdeveniments inesperats en el teu codi. Utilitzant try, catch, finally i throw, pots assegurar-te que els teus programes siguin robustos i puguin gestionar diverses condicions d'error de manera elegant. A més, crear excepcions personalitzades permet una gestió d'errors més específica i adaptada a les necessitats de la teva aplicació.
