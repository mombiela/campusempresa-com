Introducció

Les proves i la depuració són aspectes crucials del desenvolupament de programari. Asseguren que les teves aplicacions Dart funcionin sense problemes i estiguin lliures d'errors. Aquesta secció cobrirà els conceptes bàsics de les proves i la depuració en Dart, progressant des de nivells per a principiants fins a nivells avançats.

Proves en Dart

Per què són importants les proves

  • Asseguren la fiabilitat i correcció del codi.
  • Faciliten el manteniment i la refactorització del codi.
  • Ajuden a detectar errors en les primeres etapes del procés de desenvolupament.

Tipus de proves

  • Proves d'Unitat: Prova unitats individuals de codi (funcions, mètodes).
  • Proves de Widget: Prova els components de la interfície d'usuari en aplicacions Flutter.
  • Proves d'Integració: Prova una aplicació completa o una gran part d'aquesta.

Configuració d'un entorn de proves

Per començar a fer proves en Dart, has d'afegir el paquet test al teu fitxer pubspec.yaml:

dev_dependencies:
  test: ^1.16.0

Escriure la teva primera prova d'unitat

Aquí tens un exemple senzill d'una prova d'unitat en Dart:

import 'package:test/test.dart';

int add(int a, int b) {
  return a + b;
}

void main() {
  test('adds two numbers', () {
    expect(add(1, 2), 3);
  });
}
  • Explicació:
    • import 'package:test/test.dart';: Importa el paquet de proves.
    • int add(int a, int b): Una funció senzilla per sumar dos números.
    • test('adds two numbers', () { ... }): Defineix un cas de prova.
    • expect(add(1, 2), 3): Comprova si la funció add retorna 3 quan se li donen 1 i 2.

Executar proves

Per executar les teves proves, utilitza la següent comanda al teu terminal:

dart test

Proves de Widget en Flutter

Per a aplicacions Flutter, les proves de widget són essencials per assegurar que els components de la interfície d'usuari es comportin com s'espera.

import 'package:flutter_test/flutter_test.dart';
import 'package:my_app/my_widget.dart';

void main() {
  testWidgets('MyWidget has a title and message', (WidgetTester tester) async {
    await tester.pumpWidget(MyWidget(title: 'T', message: 'M'));

    final titleFinder = find.text('T');
    final messageFinder = find.text('M');

    expect(titleFinder, findsOneWidget);
    expect(messageFinder, findsOneWidget);
  });
}
  • Explicació:
    • import 'package:flutter_test/flutter_test.dart';: Importa el paquet de proves de Flutter.
    • testWidgets('MyWidget has a title and message', (WidgetTester tester) async { ... }): Defineix una prova de widget.
    • await tester.pumpWidget(MyWidget(title: 'T', message: 'M'));: Construeix el widget.
    • final titleFinder = find.text('T');: Troba el text del títol.
    • expect(titleFinder, findsOneWidget);: Verifica que el títol estigui present.

Depuració en Dart

Tècniques comunes de depuració

  • Declaracions Print: Simple però efectiu per a problemes petits.
  • Depurador: Utilitza IDEs com VSCode o IntelliJ que tenen depuradors integrats.
  • Registre: Utilitza biblioteques de registre per a aplicacions més complexes.

Utilitzar declaracions Print

void main() {
  int a = 5;
  int b = 10;
  print('a: $a, b: $b'); // Sortida de depuració
  int sum = a + b;
  print('Sum: $sum'); // Sortida de depuració
}
  • Explicació:
    • print('a: $a, b: $b');: Mostra els valors de a i b.
    • print('Sum: $sum');: Mostra el resultat de la suma.

Utilitzar el depurador

  • Establir Punts de Trencament: Fes clic a la vora al costat del número de línia al teu IDE.
  • Passar pel Codi: Utilitza les funcions de passar per sobre, entrar i sortir per navegar pel teu codi.
  • Inspeccionar Variables: Passa el ratolí per sobre de les variables per veure els seus valors actuals.

Registre

Afegeix el paquet logging al teu fitxer pubspec.yaml:

dependencies:
  logging: ^0.11.4

Exemple d'ús del paquet de registre:

import 'package:logging/logging.dart';

void main() {
  final Logger logger = Logger('MyLogger');
  logger.onRecord.listen((record) {
    print('${record.level.name}: ${record.time}: ${record.message}');
  });

  logger.info('This is an info message');
  logger.warning('This is a warning message');
  logger.severe('This is a severe message');
}
  • Explicació:
    • import 'package:logging/logging.dart';: Importa el paquet de registre.
    • Logger logger = Logger('MyLogger');: Crea una instància de logger.
    • logger.onRecord.listen((record) { ... });: Escolta els registres i els mostra.
    • logger.info('This is an info message');: Registra un missatge d'informació.

Conclusió

Les proves i la depuració són habilitats essencials per a qualsevol desenvolupador de Dart. Escriure proves assegura que el teu codi sigui fiable i fàcil de mantenir. La depuració t'ajuda a identificar i solucionar problemes de manera eficient. A mesura que progressis des de nivells per a principiants fins a nivells avançats, aquestes pràctiques es convertiran en una segona naturalesa, millorant significativament el teu flux de treball de desenvolupament.
