Introducció

Iterar a través de col·leccions és un concepte fonamental en programació. En Dart, col·leccions com llistes, conjunts i mapes són comunament utilitzades, i saber com iterar a través d'elles de manera eficient és crucial. Aquesta secció cobrirà diverses maneres d'iterar a través de col·leccions en Dart, des de bucles bàsics fins a tècniques més avançades.

Iteració bàsica amb `for` Loop

El bucle for és una manera bàsica i directa d'iterar a través de col·leccions.

Exemple: Iterant a través d'una llista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  
  for (int i = 0; i < numbers.length; i++) {
    print(numbers[i]);
  }
}
  • Explicació: Aquest exemple utilitza un bucle for tradicional per iterar a través d'una llista d'enters. El bucle s'executa des de 0 fins a numbers.length - 1, imprimint cada element.

Iteració amb `for-in` Loop

El bucle for-in proporciona una manera més concisa d'iterar a través de col·leccions.

Exemple: Iterant a través d'una llista

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Cherry'];
  
  for (var fruit in fruits) {
    print(fruit);
  }
}
  • Explicació: El bucle for-in itera directament sobre els elements de la col·lecció, fent el codi més net i fàcil de llegir.

Iterant a través de conjunts

Els conjunts en Dart són col·leccions desordenades d'elements únics. Pots iterar a través d'ells de manera similar a les llistes.

Exemple: Iterant a través d'un conjunt

void main() {
  Set<String> animals = {'Dog', 'Cat', 'Bird'};
  
  for (var animal in animals) {
    print(animal);
  }
}
  • Explicació: El bucle for-in funciona perfectament amb els conjunts, iterant sobre cada element únic.

Iterant a través de mapes

Els mapes són col·leccions de parells clau-valor. Iterar a través de mapes es pot fer de diverses maneres, depenent de si necessites claus, valors, o ambdós.

Exemple: Iterant a través d'entrades de mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var entry in scores.entries) {
    print('${entry.key}: ${entry.value}');
  }
}
  • Explicació: Utilitzant entries, pots accedir tant a les claus com als valors en cada iteració.

Exemple: Iterant a través de claus de mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var key in scores.keys) {
    print('Key: $key');
  }
}
  • Explicació: Utilitzant keys, pots iterar a través de totes les claus en el mapa.

Exemple: Iterant a través de valors de mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  for (var value in scores.values) {
    print('Value: $value');
  }
}
  • Explicació: Utilitzant values, pots iterar a través de tots els valors en el mapa.

Iteració avançada amb `forEach`

El mètode forEach proporciona un enfocament funcional per iterar a través de col·leccions.

Exemple: Utilitzant `forEach` amb una llista

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  
  numbers.forEach((number) {
    print(number);
  });
}
  • Explicació: El mètode forEach pren una funció com a argument i l'aplica a cada element de la llista.

Exemple: Utilitzant `forEach` amb un mapa

void main() {
  Map<String, int> scores = {'Alice': 90, 'Bob': 85, 'Charlie': 95};
  
  scores.forEach((key, value) {
    print('$key: $value');
  });
}
  • Explicació: El mètode forEach per a mapes pren una funció amb dos paràmetres (clau i valor) i l'aplica a cada entrada.

Conclusió

Iterar a través de col·leccions és una habilitat versàtil en la programació en Dart. Ja sigui utilitzant bucles bàsics, el bucle for-in, o el mètode forEach, entendre aquestes tècniques t'ajudarà a manejar col·leccions de manera eficient. Practica aquests mètodes per convertir-te en un expert en iterar a través de diferents tipus de col·leccions en Dart.
