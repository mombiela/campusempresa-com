Variables i Tipus de Dades
Operadors ►

Introducció

En Dart, les variables s'utilitzen per emmagatzemar dades que es poden manipular i referenciar al llarg del teu programa. Entendre les variables i els tipus de dades és fonamental per programar en Dart, ja que et permet gestionar diferents tipus de dades de manera efectiva.

Variables

Les variables en Dart s'utilitzen per emmagatzemar valors. Pots declarar una variable utilitzant les paraules clau var, final o const.

Declaració de Variables

  • var: El tipus de la variable és inferit pel compilador.
  • final: Una variable que només es pot assignar una vegada.
  • const: Una constant de temps de compilació.
void main() {
  var name = 'Alice'; // Tipus String inferit
  final age = 30; // Tipus int inferit, no es pot canviar
  const pi = 3.14; // Tipus double, constant de temps de compilació
}

Regles de Nomenclatura de Variables

  • Han de començar amb una lletra o un guió baix (_).
  • Poden contenir lletres, dígits i guions baixos.
  • Són sensibles a majúscules i minúscules.
void main() {
  var firstName = 'Alice';
  var _lastName = 'Smith';
  var age1 = 25;
}

Tipus de Dades

Dart és un llenguatge de tipus estàtic, el que significa que cada variable té un tipus. Els tipus de dades comuns en Dart inclouen:

  • Nombres: int i double
  • Cadenes: String
  • Booleans: bool
  • Llistes: List
  • Mapes: Map

Nombres

Dart té dos tipus de nombres: int per a enters i double per a nombres de punt flotant.

void main() {
  int age = 25;
  double height = 5.9;
}

Cadenes

Les cadenes representen una seqüència de caràcters. Estan tancades en cometes simples o dobles.

void main() {
  String greeting = 'Hello, World!';
  String name = "Alice";
}

Booleans

Els booleans representen valors veritables o falsos.

void main() {
  bool isStudent = true;
  bool isEmployed = false;
}

Llistes

Les llistes són col·leccions ordenades d'objectes. Poden ser de longitud fixa o creixent.

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  List<String> names = ['Alice', 'Bob', 'Charlie'];
}

Mapes

Els mapes són col·leccions de parells clau-valor. Les claus i els valors poden ser de qualsevol tipus.

void main() {
  Map<String, int> phoneBook = {
    'Alice': 12345,
    'Bob': 67890,
  };
}

Inferència de Tipus

Dart pot inferir el tipus d'una variable basant-se en el valor assignat. Això es fa utilitzant la paraula clau var.

void main() {
  var name = 'Alice'; // Inferit com String
  var age = 30; // Inferit com int
  var height = 5.9; // Inferit com double
}

Anotació de Tipus

Pots especificar explícitament el tipus d'una variable.

void main() {
  String name = 'Alice';
  int age = 30;
  double height = 5.9;
}

Conversió de Tipus

De vegades necessites convertir un tipus de dada a un altre. Dart proporciona mètodes per a la conversió de tipus.

String a int

void main() {
  var one = int.parse('1'); // 1
}

String a double

void main() {
  var onePointOne = double.parse('1.1'); // 1.1
}

int a String

void main() {
  String oneAsString = 1.toString(); // '1'
}

double a String

void main() {
  String piAsString = 3.14159.toStringAsFixed(2); // '3.14'
}

Resum

Entendre les variables i els tipus de dades és crucial per a una programació efectiva en Dart. Les variables emmagatzemen dades, i els tipus de dades defineixen el tipus de dades que una variable pot contenir. Dart suporta diversos tipus de dades, incloent nombres, cadenes, booleans, llistes i mapes. Pots declarar variables utilitzant var, final i const, i Dart pot inferir tipus o pots anotar-los explícitament. A més, Dart proporciona mètodes per a la conversió de tipus per gestionar diferents tipus de dades sense problemes.

Dominant aquests conceptes, estaràs ben equipat per gestionar dades en els teus programes Dart de manera eficient.
