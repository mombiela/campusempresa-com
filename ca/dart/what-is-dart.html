Introducció a Dart

Dart és un llenguatge de programació optimitzat per a clients per a aplicacions ràpides en qualsevol plataforma. És desenvolupat per Google i s'utilitza per construir aplicacions mòbils, d'escriptori, de servidor i web. Dart està dissenyat tant per al desenvolupament del costat del client com del servidor, fent-lo un llenguatge versàtil per al desenvolupament d'aplicacions modernes.

Característiques Clau de Dart

  • Optimitzat per a la UI: Dart està dissenyat per crear interfícies d'usuari suaus i responsives.
  • Desenvolupament Productiu: Dart ofereix un conjunt ric de biblioteques i eines que milloren la productivitat del desenvolupador.
  • Ràpid en Totes les Plataformes: Dart es compila a codi màquina ARM i x64 per a mòbils, escriptori i backend. Per a web, es compila a JavaScript.
  • Fortament Tipat: Dart és un llenguatge tipat estàticament, cosa que ajuda a detectar errors a l'inici del procés de desenvolupament.
  • Programació Asíncrona: Dart té suport integrat per a la programació asíncrona amb característiques com async i await.

Visió General de la Sintaxi de Dart

La sintaxi de Dart és fàcil d'aprendre per a desenvolupadors familiaritzats amb llenguatges com JavaScript, Java o C#. A continuació es mostra un programa simple en Dart per il·lustrar la seva sintaxi:

void main() {
  print('Hello, Dart!');
}

Explicació:

  • void main() {}: La funció main és el punt d'entrada de cada aplicació Dart.
  • print('Hello, Dart!');: Aquesta línia imprimeix la cadena 'Hello, Dart!' a la consola.

Conceptes Bàsics de Dart

Variables

Les variables en Dart es poden declarar utilitzant var, final o const.

void main() {
  var name = 'John Doe'; // Variable que pot canviar
  final age = 30;        // Variable que no pot canviar
  const pi = 3.14;       // Constant de temps de compilació
}

Funcions

Les funcions en Dart són objectes de primera classe i es poden passar com a paràmetres o assignar a variables.

int add(int a, int b) {
  return a + b;
}

void main() {
  print(add(2, 3)); // Resultat: 5
}

Classes i Objectes

Dart és un llenguatge orientat a objectes amb classes i objectes.

class Person {
  String name;
  int age;

  Person(this.name, this.age);

  void greet() {
    print('Hello, my name is $name and I am $age years old.');
  }
}

void main() {
  var person = Person('Alice', 25);
  person.greet(); // Resultat: Hello, my name is Alice and I am 25 years old.
}

Conclusió

Dart és un llenguatge potent i versàtil que és particularment adequat per construir aplicacions modernes i d'alt rendiment. La seva sintaxi és fàcil d'aprendre, especialment per a aquells familiaritzats amb altres llenguatges d'estil C. Amb característiques com el tipatge fort, la programació asíncrona i un conjunt ric de biblioteques, Dart proporciona un entorn robust tant per al desenvolupament del costat del client com del servidor. Tant si estàs construint aplicacions mòbils amb Flutter com aplicacions del costat del servidor, Dart ofereix les eines i el rendiment necessaris per crear programari eficient i escalable.
