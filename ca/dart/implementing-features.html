Introducció

En aquesta secció, explorarem com implementar diverses funcionalitats en Dart. Això inclourà entendre els conceptes bàsics de la sintaxi de Dart, crear funcions, gestionar excepcions i treballar amb programació asíncrona. Al final d'aquesta secció, seràs capaç d'implementar funcionalitats de manera efectiva en les teves aplicacions Dart.

Sintaxi i Estructura Bàsica

Abans d'endinsar-nos en la implementació de funcionalitats, és essencial entendre la sintaxi i l'estructura bàsica de Dart.

Exemple de Hello World

void main() {
  print('Hello, World!');
}
  • void main() {}: El punt d'entrada d'una aplicació Dart.
  • print('Hello, World!');: Imprimeix text a la consola.

Variables i Tipus de Dades

void main() {
  int age = 25;
  double height = 5.9;
  String name = 'John Doe';
  bool isStudent = true;

  print('Name: $name, Age: $age, Height: $height, Is Student: $isStudent');
}
  • int, double, String, bool: Tipus de dades bàsics en Dart.
  • Les variables es declaren utilitzant var, final o tipus específics.

Funcions

Les funcions són essencials per implementar funcionalitats ja que permeten la reutilització de codi i la modularitat.

Definició i Crida de Funcions

void greet(String name) {
  print('Hello, $name!');
}

void main() {
  greet('Alice');
}
  • void greet(String name) {}: Defineix una funció anomenada greet que pren un paràmetre String.
  • greet('Alice');: Crida la funció greet amb l'argument 'Alice'.

Retorn de Valors des de Funcions

int add(int a, int b) {
  return a + b;
}

void main() {
  int result = add(3, 4);
  print('Sum: $result');
}
  • int add(int a, int b) {}: Defineix una funció que retorna un int.
  • return a + b;: Retorna la suma de a i b.

Gestió d'Excepcions

Gestionar excepcions és crucial per construir aplicacions robustes.

Blocs Try-Catch

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('Error: $e');
  }
}
  • try {}: Codi que podria llançar una excepció.
  • catch (e) {}: Gestiona l'excepció.

Llançament d'Excepcions

void checkAge(int age) {
  if (age < 18) {
    throw Exception('Age must be at least 18.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('Error: $e');
  }
}
  • throw Exception('message');: Llança una excepció amb un missatge personalitzat.

Programació Asíncrona

Dart suporta la programació asíncrona utilitzant Future i async/await.

Ús de Future

Future<String> fetchUserOrder() {
  return Future.delayed(Duration(seconds: 2), () => 'Large Latte');
}

void main() {
  fetchUserOrder().then((order) {
    print('Order: $order');
  });
}
  • Future.delayed(Duration(seconds: 2), () => 'Large Latte');: Simula una operació retardada.
  • .then((order) {}: Gestiona el resultat del Future.

Async i Await

Future<String> fetchUserOrder() async {
  return await Future.delayed(Duration(seconds: 2), () => 'Large Latte');
}

void main() async {
  String order = await fetchUserOrder();
  print('Order: $order');
}
  • async: Marca una funció com a asíncrona.
  • await: Espera que el Future es completi.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de la implementació de funcionalitats en Dart, incloent entendre la sintaxi, crear i utilitzar funcions, gestionar excepcions i treballar amb programació asíncrona. En dominar aquests conceptes, estaràs ben preparat per construir aplicacions Dart robustes i eficients. Continua practicant aquests conceptes per consolidar la teva comprensió i millorar les teves habilitats de programació.
