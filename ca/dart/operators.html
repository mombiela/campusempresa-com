Operadors en Dart
Sentències de Flux de Control ►

Introducció

Els operadors són símbols especials que realitzen operacions sobre variables i valors. Dart admet una àmplia gamma d'operadors, des de l'aritmètica bàsica fins a operadors més avançats de bits i lògics. Entendre aquests operadors és crucial per escriure codi Dart eficient i efectiu.

Operadors aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques comunes.

  • + Suma
  • - Resta
  • * Multiplicació
  • / Divisió
  • % Mòdul (resta)
  • ~/ Divisió entera

Exemples

void main() {
  int a = 10;
  int b = 3;

  print(a + b);  // Output: 13
  print(a - b);  // Output: 7
  print(a * b);  // Output: 30
  print(a / b);  // Output: 3.3333333333333335
  print(a % b);  // Output: 1
  print(a ~/ b); // Output: 3
}

Operadors d'assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • = Assignar
  • += Sumar i assignar
  • -= Restar i assignar
  • *= Multiplicar i assignar
  • /= Dividir i assignar
  • %= Mòdul i assignar

Exemples

void main() {
  int a = 5;
  a += 3; // Equivalent a a = a + 3
  print(a); // Output: 8

  a -= 2; // Equivalent a a = a - 2
  print(a); // Output: 6

  a *= 2; // Equivalent a a = a * 2
  print(a); // Output: 12

  a /= 3; // Equivalent a a = a / 3
  print(a); // Output: 4.0

  a %= 2; // Equivalent a a = a % 2
  print(a); // Output: 0.0
}

Operadors de comparació

Els operadors de comparació s'utilitzen per comparar dos valors i retornar un resultat booleà.

  • == Igual a
  • != Diferent de
  • > Major que
  • < Menor que
  • >= Major o igual que
  • <= Menor o igual que

Exemples

void main() {
  int a = 5;
  int b = 3;

  print(a == b); // Output: false
  print(a != b); // Output: true
  print(a > b);  // Output: true
  print(a < b);  // Output: false
  print(a >= b); // Output: true
  print(a <= b); // Output: false
}

Operadors lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleanes.

  • && Lògic AND
  • || Lògic OR
  • ! Lògic NOT

Exemples

void main() {
  bool a = true;
  bool b = false;

  print(a && b); // Output: false
  print(a || b); // Output: true
  print(!a);     // Output: false
}

Operadors de bits

Els operadors de bits s'utilitzen per realitzar operacions a nivell de bits.

  • & AND a nivell de bits
  • | OR a nivell de bits
  • ^ XOR a nivell de bits
  • ~ NOT a nivell de bits
  • << Desplaçament a l'esquerra
  • >> Desplaçament a la dreta

Exemples

void main() {
  int a = 5;  // 0101 en binari
  int b = 3;  // 0011 en binari

  print(a & b);  // Output: 1 (0001 en binari)
  print(a | b);  // Output: 7 (0111 en binari)
  print(a ^ b);  // Output: 6 (0110 en binari)
  print(~a);     // Output: -6 (inverteix tots els bits)
  print(a << 1); // Output: 10 (1010 en binari)
  print(a >> 1); // Output: 2 (0010 en binari)
}

Operador condicional (ternari)

L'operador condicional és una forma abreujada d'una instrucció if-else.

  • condició ? expr1 : expr2

Exemple

void main() {
  int a = 5;
  int b = 3;

  String result = a > b ? 'a és més gran' : 'b és més gran';
  print(result); // Output: a és més gran
}

Operadors de consciència de nul

Els operadors de consciència de nul s'utilitzen per gestionar valors nuls de manera més elegant.

  • ?? Si és nul
  • ??= Assignar si és nul
  • ?. Accés condicional a membres

Exemples

void main() {
  int? a;
  int b = 3;

  print(a ?? b); // Output: 3

  a ??= 10;
  print(a); // Output: 10

  String? name;
  print(name?.length); // Output: null
}

Resum

Els operadors en Dart són eines essencials per realitzar una àmplia gamma d'operacions, des de l'aritmètica bàsica fins a manipulacions lògiques i de bits complexes. Entendre com utilitzar aquests operadors de manera efectiva t'ajudarà a escriure codi Dart més eficient i llegible. Practica utilitzant aquests operadors en diferents escenaris per sentir-te més còmode amb la seva sintaxi i funcionalitat.
