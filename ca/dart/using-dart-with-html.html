Introducció

Dart és un llenguatge de programació versàtil que es pot utilitzar tant per al desenvolupament del costat del client com del servidor. Una de les seves característiques més potents és la capacitat d'interactuar amb HTML, cosa que el converteix en una excel·lent opció per al desenvolupament web. En aquesta secció, explorarem com utilitzar Dart amb HTML, des de conceptes bàsics fins a tècniques avançades.

Configuració de Dart per al Desenvolupament Web

Abans de submergir-nos en l'ús de Dart amb HTML, configurem el nostre entorn.

  • Instal·la Dart SDK: Assegura't de tenir el Dart SDK instal·lat a la teva màquina.
  • Instal·la Stagehand: Una eina de bastiment de projectes Dart. 
    pub global activate stagehand
  • Crea un nou projecte web Dart: 
    stagehand web-simple
cd <your_project_name>
pub get

Integració Bàsica amb HTML

Comencem amb un exemple senzill d'integració de Dart amb HTML.

Estructura HTML

Crea un fitxer index.html amb el següent contingut:

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <h1 id="title">Hello, Dart!</h1>
</body>
</html>

Codi Dart

Crea un fitxer main.dart amb el següent contingut:

import 'dart:html';

void main() {
  querySelector('#title')?.text = 'Hello, Dart and HTML!';
}

Explicació

  • HTML: El fitxer HTML inclou una etiqueta script que apunta al codi Dart compilat (main.dart.js).
  • Dart: El codi Dart utilitza querySelector per seleccionar l'element HTML amb l'id title i canvia el seu contingut de text.

Interactuant amb Elements HTML

Dart proporciona un conjunt ric d'APIs per interactuar amb elements HTML.

Seleccionant Elements

  • Per ID: 
    var element = querySelector('#elementId');
  • Per Classe: 
    var elements = querySelectorAll('.className');

Modificant Elements

  • Canviant Text: 
    querySelector('#elementId')?.text = 'New Text';
  • Canviant Atributs: 
    querySelector('#elementId')?.setAttribute('attributeName', 'value');

Exemple

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = 'Button clicked!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <button id="myButton">Click Me</button>
  <div id="output"></div>
</body>
</html>

Maneig d'Esdeveniments

El maneig d'esdeveniments en Dart és senzill i similar a JavaScript.

Exemple: Esdeveniment de Clic

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = 'Button clicked!';
  });
}

Explicació

  • Listener d'Esdeveniments: El mètode onClick.listen adjunta un listener d'esdeveniments al botó, que actualitza el text del div output quan es fa clic al botó.

Tècniques Avançades

Utilitzant Dart amb Formularis

El maneig de formularis és una tasca comuna en el desenvolupament web.

Exemple: Enviament de Formulari

import 'dart:html';

void main() {
  var form = querySelector('#myForm');
  form?.onSubmit.listen((event) {
    event.preventDefault(); // Prevenir l'enviament per defecte del formulari
    var input = (querySelector('#name') as InputElement).value;
    querySelector('#output')?.text = 'Hello, $input!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <form id="myForm">
    <input type="text" id="name" placeholder="Enter your name">
    <button type="submit">Submit</button>
  </form>
  <div id="output"></div>
</body>
</html>

Explicació

  • Maneig de Formularis: El mètode onSubmit.listen adjunta un listener d'esdeveniments al formulari, que prevé l'enviament per defecte del formulari i actualitza el div output amb el valor de l'input.

Conclusió

Utilitzar Dart amb HTML et permet crear aplicacions web dinàmiques i interactives. Entenent com seleccionar i manipular elements HTML, manejar esdeveniments i treballar amb formularis, pots aprofitar les capacitats de Dart per construir aplicacions web robustes. A mesura que progressis, podràs explorar característiques i biblioteques més avançades per millorar les teves habilitats de desenvolupament web amb Dart.
