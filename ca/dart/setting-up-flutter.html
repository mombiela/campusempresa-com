Introducció

Flutter és un kit d'eines de desenvolupament de programari d'interfície d'usuari de codi obert creat per Google. S'utilitza per desenvolupar aplicacions multiplataforma per a Android, iOS, Linux, Mac, Windows, Google Fuchsia i la web des d'una única base de codi. En aquesta secció, et guiarem a través de la configuració de Flutter a la teva màquina.

Prerequisits

Abans de configurar Flutter, assegura't de tenir el següent:

  • Un ordinador amb macOS, Windows o Linux.
  • Coneixement bàsic del llenguatge de programació Dart.
  • Un IDE (Entorn de Desenvolupament Integrat) com Android Studio, IntelliJ IDEA o Visual Studio Code.

Configuració Pas a Pas

1. Descarregar Flutter SDK

  • Visita el lloc web oficial de Flutter.
  • Tria el teu sistema operatiu (Windows, macOS o Linux).
  • Descarrega el fitxer zip de Flutter SDK.

2. Extreure Flutter SDK

  • Extreu el fitxer zip descarregat a una ubicació desitjada al teu sistema.
  • Windows: Extreu a C:\src\flutter (o qualsevol altra ubicació).
  • macOS/Linux: Extreu al teu directori d'inici o qualsevol altra ubicació.

3. Actualitzar el teu Path

Per executar comandes de Flutter al terminal, necessites afegir Flutter al teu PATH del sistema.

Windows

  1. Obre el Cercador d'Inici, escriu "env" i selecciona "Edita les variables d'entorn del sistema".
  2. A la finestra de Propietats del Sistema, fes clic al botó "Variables d'entorn".
  3. Sota "Variables del sistema", troba la variable Path, selecciona-la i fes clic a "Edita".
  4. Fes clic a "Nou" i afegeix el camí al directori flutter\bin (per exemple, C:\src\flutter\bin).
  5. Fes clic a OK per guardar els canvis.

macOS/Linux

  1. Obre un terminal.
  2. Obre el teu perfil bash en un editor de text: 
    nano ~/.bash_profile
  3. Afegeix la següent línia al fitxer: 
    export PATH="$PATH:`pwd`/flutter/bin"
  4. Guarda el fitxer i executa: 
    source ~/.bash_profile

4. Verificar la Instal·lació

Obre un terminal i executa la següent comanda per verificar que Flutter està instal·lat correctament:

flutter doctor

Aquesta comanda comprova el teu entorn i mostra un informe de l'estat de la teva instal·lació de Flutter. La sortida hauria de semblar-se a això:

Doctor summary (to see all details, run flutter doctor -v):
[✓] Flutter (Channel stable, 2.5.3, on macOS 11.6 20G165 darwin-x64, locale en-US)
[✓] Android toolchain - develop for Android devices (Android SDK version 31.0.0)
[✓] Xcode - develop for iOS and macOS (Xcode 13.1)
[✓] Chrome - develop for the web
[✓] Android Studio (version 2020.3)
[✓] VS Code (version 1.61.2)
[✓] Connected device (2 available)

Assegura't que totes les comprovacions estan marcades amb una marca verda. Si hi ha algun problema, segueix les instruccions proporcionades per resoldre'ls.

5. Configurar un Editor

Flutter suporta diversos editors. Aquí, cobrirem la configuració de Visual Studio Code i Android Studio.

Visual Studio Code

  1. Instal·la Visual Studio Code des de aquí.
  2. Obre Visual Studio Code.
  3. Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra o prement Ctrl+Shift+X.
  4. Cerca "Flutter" i instal·la l'extensió de Flutter per Dart Code.

Android Studio

  1. Descarrega i instal·la Android Studio des de aquí.
  2. Obre Android Studio.
  3. Ves a File > Settings > Plugins.
  4. Cerca "Flutter" i fes clic a "Instal·la".
  5. Segueix les instruccions per instal·lar també el plugin de Dart.

6. Crear un Nou Projecte Flutter

Un cop el teu editor estigui configurat, pots crear un nou projecte Flutter.

Visual Studio Code

  1. Obre Visual Studio Code.
  2. Prem Ctrl+Shift+P per obrir la paleta de comandes.
  3. Escriu Flutter: New Project i selecciona-ho.
  4. Segueix les instruccions per donar nom al teu projecte i triar una ubicació per guardar-lo.

Android Studio

  1. Obre Android Studio.
  2. Fes clic a Start a new Flutter project.
  3. Selecciona Flutter Application i fes clic a Next.
  4. Segueix les instruccions per configurar el teu nou projecte Flutter.

7. Executar la Teva Primera App Flutter

Per executar la teva app Flutter, segueix aquests passos:

Visual Studio Code

  1. Obre el fitxer principal de Dart (lib/main.dart).
  2. Prem F5 per iniciar el depurador.
  3. Selecciona el teu dispositiu objectiu (per exemple, un emulador d'Android o un dispositiu connectat).

Android Studio

  1. Obre el fitxer principal de Dart (lib/main.dart).
  2. Fes clic al botó verd de reproducció a la barra d'eines.
  3. Selecciona el teu dispositiu objectiu (per exemple, un emulador d'Android o un dispositiu connectat).

Conclusió

Configurar Flutter implica descarregar el SDK, actualitzar el teu PATH del sistema, verificar la instal·lació, configurar un editor, crear un nou projecte i executar la teva primera app. Seguint aquests passos, ara hauries de tenir un entorn de desenvolupament Flutter completament funcional. Aquesta configuració et permetrà construir i executar aplicacions Flutter sense problemes.
