Introducció al Sistema de Tipus de Dart

Dart és un llenguatge tipat estàticament, el que significa que el tipus d'una variable es coneix en temps de compilació. Això ajuda a detectar errors aviat i millora la qualitat i el rendiment del codi.

Conceptes Clau

  • Tipatge Estàtic: Els tipus es comproven en temps de compilació.
  • Inferència de Tipus: Dart pot inferir tipus, així que no sempre cal declarar-los explícitament.
  • Tipus Dinàmic: Dart també admet el tipatge dinàmic utilitzant la paraula clau dynamic.

Tipus Bàsics

Dart proporciona diversos tipus integrats que són comunament utilitzats en programació.

Tipus Comuns

  • int: Representa valors enters.
  • double: Representa valors de punt flotant.
  • String: Representa una seqüència de caràcters.
  • bool: Representa valors booleans (true o false).
  • List: Representa una col·lecció d'objectes ordenats.
  • Map: Representa una col·lecció de parells clau-valor.

Exemples de Codi

void main() {
  int age = 25;
  double height = 5.9;
  String name = 'Alice';
  bool isStudent = true;
  List<int> numbers = [1, 2, 3, 4, 5];
  Map<String, String> capitals = {
    'USA': 'Washington, D.C.',
    'France': 'Paris',
    'Japan': 'Tokyo'
  };

  print('Nom: $name, Edat: $age, Alçada: $height, Estudiant: $isStudent');
  print('Nombres: $numbers');
  print('Capitals: $capitals');
}

Inferència de Tipus

Dart pot inferir el tipus d'una variable basant-se en el valor assignat, el que fa que el codi sigui més net i fàcil de llegir.

Exemples de Codi

void main() {
  var age = 25; // Dart infereix que age és de tipus int
  var height = 5.9; // Dart infereix que height és de tipus double
  var name = 'Alice'; // Dart infereix que name és de tipus String
  var isStudent = true; // Dart infereix que isStudent és de tipus bool

  print('Nom: $name, Edat: $age, Alçada: $height, Estudiant: $isStudent');
}

Tipus Dinàmic

La paraula clau dynamic permet que una variable contingui valors de qualsevol tipus. Això és útil quan el tipus de la variable no es coneix en temps de compilació.

Exemples de Codi

void main() {
  dynamic variable = 25;
  print('Variable: $variable, Tipus: ${variable.runtimeType}');

  variable = 'Hola';
  print('Variable: $variable, Tipus: ${variable.runtimeType}');

  variable = true;
  print('Variable: $variable, Tipus: ${variable.runtimeType}');
}

Seguretat de Tipus

Dart assegura la seguretat de tipus evitant operacions que no estan permeses en certs tipus.

Exemples de Codi

void main() {
  int age = 25;
  // age = 'twenty-five'; // Això causarà un error en temps de compilació

  String name = 'Alice';
  // name = 123; // Això causarà un error en temps de compilació

  print('Nom: $name, Edat: $age');
}

Conversió de Tipus

De vegades, pot ser necessari convertir una variable d'un tipus a un altre. Dart proporciona diverses maneres de realitzar conversions de tipus.

Exemples de Codi

void main() {
  // String a int
  String ageStr = '25';
  int age = int.parse(ageStr);
  print('Edat: $age');

  // int a String
  int year = 2023;
  String yearStr = year.toString();
  print('Any: $yearStr');

  // double a int
  double height = 5.9;
  int roundedHeight = height.toInt();
  print('Alçada Arrodonida: $roundedHeight');
}

Seguretat de Null

Dart té una característica de seguretat de null sòlida que ajuda a prevenir errors de referència null. Per defecte, les variables no poden ser nulles a menys que es declarin explícitament com a nul·les.

Exemples de Codi

void main() {
  int? nullableAge;
  nullableAge = null; // Això està permès

  int nonNullableAge = 25;
  // nonNullableAge = null; // Això causarà un error en temps de compilació

  print('Edat Nullable: $nullableAge, Edat No Nullable: $nonNullableAge');
}

Resum

En aquesta secció, hem cobert els aspectes essencials del sistema de tipus de Dart, incloent:

  • Tipus bàsics i el seu ús.
  • Inferència de tipus per a un codi més net.
  • El tipus dynamic per a flexibilitat.
  • Seguretat de tipus per a prevenir errors.
  • Conversió de tipus per a convertir entre tipus.
  • Seguretat de null per a evitar errors de referència null.

Entendre el sistema de tipus de Dart és crucial per escriure codi robust i lliure d'errors. A mesura que progressis, trobaràs que aquests conceptes formen la base per a tècniques de programació en Dart més avançades.
