Introducció

Dart és un llenguatge optimitzat per a clients per a aplicacions ràpides en qualsevol plataforma. Ha estat desenvolupat per Google i s'utilitza per crear aplicacions mòbils, d'escriptori, de servidor i web. Entendre la sintaxi i l'estructura bàsiques de Dart és essencial per escriure codi eficient i llegible.

Conceptes Clau

  • Variables i Tipus de Dades
  • Funcions
  • Sentències de Control de Flux
  • Comentaris
  • Imports

Variables i Tipus de Dades

A Dart, les variables s'utilitzen per emmagatzemar dades. Dart és un llenguatge tipat estàticament, el que significa que cal especificar el tipus d'una variable quan es declara.

Declaració de Variables

void main() {
  int age = 25; // Enter
  double height = 5.9; // Doble
  String name = 'John Doe'; // Cadena
  bool isStudent = true; // Booleà
}

Inferència de Tipus

Dart pot inferir el tipus d'una variable basant-se en el valor assignat.

void main() {
  var age = 25; // Dart infereix que age és un int
  var height = 5.9; // Dart infereix que height és un double
  var name = 'John Doe'; // Dart infereix que name és una String
  var isStudent = true; // Dart infereix que isStudent és un bool
}

Funcions

Les funcions són els blocs de construcció d'un programa Dart. Permeten encapsular codi en blocs reutilitzables.

Definició de Funcions

void main() {
  printGreeting();
  print(add(5, 3));
}

void printGreeting() {
  print('Hello, World!');
}

int add(int a, int b) {
  return a + b;
}

Funcions Fletxa

Per a funcions simples, Dart proporciona una sintaxi abreujada coneguda com a funcions fletxa.

int add(int a, int b) => a + b;

Sentències de Control de Flux

Les sentències de control de flux s'utilitzen per controlar el flux d'execució del programa.

Sentències If-Else

void main() {
  int age = 20;

  if (age >= 18) {
    print('You are an adult.');
  } else {
    print('You are a minor.');
  }
}

Sentències Switch

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excellent');
      break;
    case 'B':
      print('Good');
      break;
    case 'C':
      print('Fair');
      break;
    case 'D':
      print('Poor');
      break;
    default:
      print('Fail');
  }
}

Bucles

Bucle For

void main() {
  for (int i = 0; i < 5; i++) {
    print(i);
  }
}

Bucle While

void main() {
  int i = 0;

  while (i < 5) {
    print(i);
    i++;
  }
}

Imports

Els imports s'utilitzen per incloure biblioteques i altres fitxers Dart en el teu programa.

Importar Biblioteques Bàsiques

import 'dart:math';

Importar Fitxers Personalitzats

import 'my_custom_file.dart';

Conclusió

Entendre la sintaxi i l'estructura bàsiques de Dart és crucial per escriure codi eficient i llegible. Això inclou saber com declarar variables, definir funcions, utilitzar sentències de control de flux, escriure comentaris i importar biblioteques. Dominar aquests conceptes bàsics proporcionarà una base sòlida per a conceptes més avançats de programació en Dart.
