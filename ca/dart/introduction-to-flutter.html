Què és Flutter?

Flutter és un kit d'eines de desenvolupament de programari d'interfície d'usuari de codi obert creat per Google. S'utilitza per desenvolupar aplicacions per a Android, iOS, Linux, Mac, Windows, Google Fuchsia i la web des d'una única base de codi.

Característiques Clau de Flutter

  • Única Base de Codi: Escriu una vegada, executa en qualsevol lloc.
  • Desenvolupament Ràpid: Hot reload et permet veure els canvis instantàniament.
  • Interfície d'Usuari Expressiva i Flexible: Conjunt ric de widgets personalitzables.
  • Rendiment Natiu: Compila a codi ARM natiu per a una execució ràpida.

Configuració de Flutter

Abans de començar a desenvolupar amb Flutter, necessites configurar el teu entorn de desenvolupament.

Passos d'Instal·lació

  1. Descarrega el SDK de Flutter: Visita el lloc web oficial de Flutter i descarrega el SDK per al teu sistema operatiu.
  2. Extreu el SDK: Descomprimeix el fitxer descarregat i col·loca'l en una ubicació desitjada.
  3. Actualitza el Path: Afegeix el directori bin de Flutter al PATH del teu sistema.
  4. Executa Flutter Doctor: Obre un terminal i executa flutter doctor per comprovar si hi ha alguna dependència que necessitis instal·lar.
$ flutter doctor

Exemple: Configuració a Windows

  1. Descarrega el SDK de Flutter des del lloc web oficial.
  2. Extreu el fitxer zip i col·loca'l a C:\src\flutter.
  3. Afegeix C:\src\flutter\bin a la variable d'entorn PATH.
  4. Obre el Símbol del Sistema i executa flutter doctor.

Creació de la Teva Primera Aplicació Flutter

Creem una aplicació Flutter senzilla per entendre els conceptes bàsics.

Passos per Crear una Aplicació Flutter

  1. Crea un Nou Projecte: Utilitza la CLI de Flutter per crear un nou projecte. 
    $ flutter create my_first_app
  2. Navega al Directori del Projecte: 
    $ cd my_first_app
  3. Executa l'Aplicació: Utilitza la següent comanda per executar la teva aplicació en un emulador o dispositiu connectat. 
    $ flutter run

Comprensió de l'Estructura del Projecte

  • lib/main.dart: El punt d'entrada principal de l'aplicació.
  • pubspec.yaml: Fitxer de configuració del projecte, incloent-hi les dependències.

Exemple: Aplicació Flutter Bàsica

Aquí tens una aplicació Flutter senzilla que mostra "Hello, World!".

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Welcome to Flutter'),
        ),
        body: Center(
          child: Text('Hello, World!'),
        ),
      ),
    );
  }
}

Explicació

  • main(): El punt d'entrada de l'aplicació.
  • MyApp: Un widget sense estat que representa l'aplicació.
  • MaterialApp: Un widget de conveniència que embolcalla diversos widgets comunament requerits per a aplicacions de disseny material.
  • Scaffold: Implementa l'estructura visual bàsica del disseny material.
  • AppBar: Una barra d'aplicació de disseny material.
  • Center: Centra el seu widget fill.
  • Text: Un widget que mostra una cadena de text.

Conclusió

En aquesta secció, hem introduït Flutter, hem configurat l'entorn de desenvolupament i hem creat una aplicació Flutter bàsica. La capacitat de Flutter per utilitzar una única base de codi per a múltiples plataformes, combinada amb la seva interfície d'usuari expressiva i el seu cicle de desenvolupament ràpid, el converteixen en una eina poderosa per al desenvolupament d'aplicacions modernes. A mesura que avancis en aquest curs, aprofundiràs en les capacitats de Flutter i aprendràs a construir aplicacions complexes i d'alt rendiment.
