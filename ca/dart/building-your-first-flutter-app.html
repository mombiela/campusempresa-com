Introducció a Flutter

Flutter és un kit d'eines de desenvolupament de programari d'interfície d'usuari de codi obert creat per Google. S'utilitza per desenvolupar aplicacions per a Android, iOS, Linux, Mac, Windows, Google Fuchsia i la web des d'una única base de codi.

  • Per què Flutter?
    • Única base de codi per a múltiples plataformes.
    • Ric conjunt de widgets pre-dissenyats.
    • Alt rendiment gràcies a la compilació directa a codi natiu.
    • Comunitat forta i documentació extensa.

Configurant el teu entorn

Abans de començar a construir la teva primera aplicació Flutter, has de configurar el teu entorn de desenvolupament.

  • Instal·lar Flutter SDK

    • Descarrega el Flutter SDK des del lloc web oficial.
    • Extreu el fitxer zip i afegeix el directori flutter/bin al PATH del teu sistema.

  • Instal·lar un IDE

    • Recomanat: Visual Studio Code o Android Studio.
    • Instal·la els plugins de Flutter i Dart per al teu IDE escollit.

  • Verificar la instal·lació

    • Executa flutter doctor al teu terminal per comprovar si hi ha dependències que falten.

Creant un nou projecte Flutter

Un cop el teu entorn estigui configurat, pots crear un nou projecte Flutter.

  • Utilitzant la línia de comandes

    flutter create my_first_app
cd my_first_app

  • Utilitzant un IDE

    • Obre el teu IDE i selecciona "Create New Flutter Project".
    • Segueix les indicacions per configurar el teu projecte.

Entenent l'estructura del projecte

Un projecte Flutter té una estructura específica que has d'entendre.

  • Directoris i fitxers clau
    • lib/: Conté el codi Dart principal de la teva aplicació.
    • pubspec.yaml: Gestiona les dependències del projecte.
    • android/ i ios/: Codi específic de la plataforma per a Android i iOS.
    • test/: Conté proves unitàries i de widgets.

Escriure la teva primera aplicació Flutter

Comencem escrivint una simple aplicació Flutter que mostri "Hello, World!".

  • main.dart

    import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Hello, World!'),
        ),
        body: Center(
          child: Text('Hello, World!'),
        ),
      ),
    );
  }
}

  • Explicació

    • main(): El punt d'entrada de l'aplicació.
    • runApp(): Infla el widget donat i l'enganxa a la pantalla.
    • MyApp: Un widget sense estat que construeix la interfície d'usuari de l'aplicació.
    • MaterialApp: Un widget de conveniència que embolcalla diversos widgets comunament requerits per a aplicacions de disseny material.
    • Scaffold: Implementa l'estructura visual bàsica del disseny material.
    • AppBar: Una barra d'aplicació de disseny material.
    • Center: Centra el seu widget fill.
    • Text: Un widget que mostra una cadena de text.

Executant la teva aplicació

Pots executar la teva aplicació Flutter en un emulador o un dispositiu físic.

  • Utilitzant la línia de comandes

    flutter run

  • Utilitzant un IDE

    • Fes clic al botó "Run" al teu IDE.

Afegint interactivitat

Afegim un botó que canviï el text quan es premi.

  • main.dart actualitzat

    import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  String _displayText = 'Hello, World!';

  void _updateText() {
    setState(() {
      _displayText = 'You pressed the button!';
    });
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Interactive App'),
        ),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: <Widget>[
              Text(_displayText),
              SizedBox(height: 20),
              ElevatedButton(
                onPressed: _updateText,
                child: Text('Press me'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

  • Explicació

    • StatefulWidget: Un widget que té estat mutable.
    • _MyAppState: La classe d'estat per a MyApp.
    • _displayText: Una cadena que conté el text a mostrar.
    • _updateText(): Un mètode que actualitza _displayText i crida setState() per reconstruir la interfície d'usuari.
    • Column: Un widget que disposa els seus fills en una columna vertical.
    • ElevatedButton: Un botó elevat de disseny material.

Conclusió

En aquest tema, has après a configurar el teu entorn de desenvolupament Flutter, crear un nou projecte Flutter, entendre l'estructura del projecte, escriure una simple aplicació Flutter, executar l'aplicació i afegir interactivitat. Aquest coneixement fonamental t'ajudarà a mesura que continuïs explorant i construint aplicacions Flutter més complexes.
