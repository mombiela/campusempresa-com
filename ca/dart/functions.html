Introducció a les Funcions

Les funcions són blocs fonamentals en la programació en Dart. Et permeten encapsular un bloc de codi que realitza una tasca específica, que després es pot reutilitzar al llarg del teu programa.

Conceptes Clau

  • Definició de Funció: Com definir una funció.
  • Invocació de Funció: Com cridar una funció.
  • Paràmetres i Arguments: Com passar dades a les funcions.
  • Valors de Retorn: Com les funcions poden retornar dades.
  • Funcions Anònimes: Funcions sense nom.
  • Funcions Fletxa: Una manera concisa d'escriure funcions.
  • Funcions d'Ordre Superior: Funcions que prenen altres funcions com a paràmetres o les retornen.

Definició i Invocació de Funció

Per definir una funció en Dart, utilitzes la paraula clau void si la funció no retorna un valor, o especifiques un tipus de retorn si ho fa.

// Funció que no retorna un valor
void greet() {
  print('Hola, Món!');
}

// Funció que retorna un valor
int add(int a, int b) {
  return a + b;
}

Per cridar una funció, simplement utilitza el seu nom seguit de parèntesis.

void main() {
  greet(); // Sortida: Hola, Món!
  int sum = add(3, 4);
  print(sum); // Sortida: 7
}

Paràmetres i Arguments

Les funcions poden prendre paràmetres, que es especifiquen en la definició de la funció. Quan crides la funció, proporciones arguments.

void printMessage(String message) {
  print(message);
}

void main() {
  printMessage('Hola, Dart!'); // Sortida: Hola, Dart!
}

Paràmetres Opcional

Dart suporta paràmetres opcionals, que poden ser posicionals o amb nom.

Paràmetres Opcionals Posicionals

Utilitza claudàtors [] per definir paràmetres opcionals posicionals.

void greet([String name = 'Convidat']) {
  print('Hola, $name!');
}

void main() {
  greet(); // Sortida: Hola, Convidat!
  greet('Alice'); // Sortida: Hola, Alice!
}

Paràmetres Opcionals amb Nom

Utilitza claus {} per definir paràmetres opcionals amb nom.

void greet({String name = 'Convidat'}) {
  print('Hola, $name!');
}

void main() {
  greet(); // Sortida: Hola, Convidat!
  greet(name: 'Alice'); // Sortida: Hola, Alice!
}

Valors de Retorn

Les funcions poden retornar valors utilitzant la paraula clau return.

double multiply(double a, double b) {
  return a * b;
}

void main() {
  double result = multiply(2.5, 4.0);
  print(result); // Sortida: 10.0
}

Funcions Anònimes

Les funcions anònimes, també conegudes com a funcions lambda o en línia, són funcions sense nom. Sovint s'utilitzen com a arguments per a funcions d'ordre superior.

void main() {
  var list = ['poma', 'plàtan', 'cirera'];
  list.forEach((item) {
    print(item);
  });
}

Funcions Fletxa

Les funcions fletxa proporcionen una sintaxi concisa per a funcions que contenen una sola expressió.

int square(int num) => num * num;

void main() {
  print(square(4)); // Sortida: 16
}

Funcions d'Ordre Superior

Les funcions d'ordre superior són funcions que prenen altres funcions com a paràmetres o les retornen.

void printElement(int element) {
  print(element);
}

void main() {
  var list = [1, 2, 3, 4];
  list.forEach(printElement); // Sortida: 1 2 3 4
}

Conclusió

Les funcions en Dart són eines versàtils i poderoses que t'ajuden a organitzar i reutilitzar codi de manera eficient. Des de definicions bàsiques de funcions fins a conceptes avançats com les funcions d'ordre superior, entendre com treballar amb funcions és crucial per a una programació efectiva en Dart. Practica definint, invocant i utilitzant diferents tipus de funcions per convertir-te en un expert en Dart.
