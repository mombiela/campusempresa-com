Configuració de l'Entorn de Desenvolupament
El Teu Primer Programa en Dart ►

Configurar l'entorn de desenvolupament és el primer pas per començar a programar en Dart. Aquesta secció et guiarà a través del procés d'instal·lació de Dart, configuració d'un IDE i execució del teu primer programa en Dart.

Instal·lant Dart

Per començar a codificar en Dart, necessites instal·lar el Dart SDK (Software Development Kit).

Passos per Instal·lar Dart SDK

  1. Descarregar Dart SDK:

    • Visita la pàgina de descàrrega de Dart SDK.
    • Tria la versió adequada per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·lar Dart SDK:

    • Windows:
      • Descarrega el fitxer zip del Dart SDK.
      • Extreu el fitxer zip a una ubicació desitjada.
      • Afegeix el directori bin a la variable d'entorn PATH del teu sistema.
    • macOS:
      • Utilitza Homebrew per instal·lar Dart executant la comanda: 
        brew tap dart-lang/dart
brew install dart
    • Linux:
      • Utilitza APT per instal·lar Dart executant la comanda: 
        sudo apt update
sudo apt install dart

  3. Verificar la Instal·lació:

    • Obre un terminal o símbol del sistema.
    • Executa la comanda: 
      dart --version
    • Hauries de veure la versió de Dart instal·lada.

Configurant un Entorn de Desenvolupament Integrat (IDE)

Un IDE t'ajuda a escriure, depurar i gestionar el teu codi Dart de manera eficient. Els IDEs populars per a Dart inclouen Visual Studio Code i IntelliJ IDEA.

Configurant Visual Studio Code (VS Code)

  1. Instal·lar VS Code:

    • Descarrega i instal·la VS Code des del lloc web oficial.

  2. Instal·lar l'Extensió de Dart:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra.
    • Cerca "Dart" i instal·la l'extensió de Dart.

  3. Configurar Dart SDK:

    • Obre el Command Palette (Ctrl+Shift+P o Cmd+Shift+P).
    • Escriu Dart: Use Recommended Settings i selecciona-ho.
    • Això configurarà VS Code per utilitzar el Dart SDK.

Configurant IntelliJ IDEA

  1. Instal·lar IntelliJ IDEA:

    • Descarrega i instal·la IntelliJ IDEA des del lloc web oficial.

  2. Instal·lar el Plugin de Dart:

    • Obre IntelliJ IDEA.
    • Ves a File > Settings (o IntelliJ IDEA > Preferences en macOS).
    • Navega a Plugins i cerca "Dart".
    • Instal·la el plugin de Dart i reinicia IntelliJ IDEA si se't demana.

  3. Configurar Dart SDK:

    • Ves a File > Settings (o IntelliJ IDEA > Preferences en macOS).
    • Navega a Languages & Frameworks > Dart.
    • Configura el camí del Dart SDK a la ubicació on vas instal·lar el Dart SDK.

Executant el Teu Primer Programa en Dart

Ara que el teu entorn de desenvolupament està configurat, anem a executar un programa simple en Dart.

Escriure un Programa Simple en Dart

  1. Crear un Nou Fitxer Dart:

    • En el teu IDE, crea un nou fitxer anomenat hello.dart.

  2. Escriure el Codi:

    • Obre hello.dart i escriu el següent codi: 
      void main() {
  print('Hello, Dart!');
}

  3. Executar el Programa:

    • En VS Code:
      • Obre el terminal dins de VS Code (Ctrl+ o Cmd+).
      • Navega al directori que conté hello.dart.
      • Executa la comanda: 
        dart hello.dart
    • En IntelliJ IDEA:
      • Fes clic dret sobre hello.dart i selecciona Run 'hello.dart'.

Explicació del Codi

  • void main() { ... }: Aquesta és la funció principal, el punt d'entrada de cada aplicació Dart.
  • print('Hello, Dart!');: Aquesta línia imprimeix la cadena 'Hello, Dart!' a la consola.

Conclusió

Configurar l'entorn de desenvolupament per a Dart implica instal·lar el Dart SDK, configurar un IDE i executar un programa simple en Dart. Seguint aquests passos, ara estàs preparat per començar a explorar la programació en Dart. En les següents seccions, aprofundirem en la sintaxi de Dart, les seves característiques i temes avançats.
