Introducció a les Classes i Objectes

A Dart, les classes són el plànol per crear objectes. Un objecte és una instància d'una classe, i pot tenir propietats (variables) i mètodes (funcions) que defineixen el seu comportament.

Conceptes Clau

  • Classe: Una plantilla per crear objectes.
  • Objecte: Una instància d'una classe.
  • Constructor: Un mètode especial utilitzat per inicialitzar objectes.
  • Propietats: Variables que pertanyen a un objecte.
  • Mètodes: Funcions que pertanyen a un objecte.

Definició d'una Classe

Per definir una classe a Dart, utilitza la paraula clau class seguida del nom de la classe i un parell de claus {}.

class Animal {
  // Propietats
  String name;
  int age;

  // Constructor
  Animal(this.name, this.age);

  // Mètode
  void displayInfo() {
    print('Name: $name, Age: $age');
  }
}

Explicació

  • Propietats: name i age són propietats de la classe Animal.
  • Constructor: Animal(this.name, this.age); inicialitza les propietats.
  • Mètode: displayInfo() és un mètode que imprimeix la informació de l'objecte.

Creació d'Objectes

Per crear un objecte, utilitza la paraula clau new seguida del nom de la classe i els paràmetres requerits.

void main() {
  // Creació d'un objecte
  Animal dog = new Animal('Buddy', 3);

  // Accés a les propietats
  print(dog.name); // Sortida: Buddy

  // Crida als mètodes
  dog.displayInfo(); // Sortida: Name: Buddy, Age: 3
}

Explicació

  • Creació d'Objectes: Animal dog = new Animal('Buddy', 3); crea un objecte anomenat dog.
  • Accés a les Propietats: dog.name accedeix a la propietat name.
  • Crida als Mètodes: dog.displayInfo() crida el mètode displayInfo.

Conceptes Avançats

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

class Dog extends Animal {
  String breed;

  Dog(String name, int age, this.breed) : super(name, age);

  void displayBreed() {
    print('Breed: $breed');
  }
}

Explicació

  • Herència: class Dog extends Animal significa que Dog hereta de Animal.
  • Constructor: Dog(String name, int age, this.breed) : super(name, age); crida el constructor de la classe pare.

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen.

void main() {
  Animal myDog = new Dog('Buddy', 3, 'Golden Retriever');
  myDog.displayInfo(); // Sortida: Name: Buddy, Age: 3
  // myDog.displayBreed(); // Això causarà un error
}

Explicació

  • Polimorfisme: Animal myDog = new Dog('Buddy', 3, 'Golden Retriever'); crea un objecte Dog però el tracta com un Animal.

Encapsulació

L'encapsulació restringeix l'accés directe a alguns components d'un objecte.

class Cat {
  String _name; // Propietat privada

  Cat(this._name);

  String get name => _name; // Getter

  set name(String name) {
    _name = name;
  }
}

Explicació

  • Propietat Privada: _name és privada a la classe Cat.
  • Getter i Setter: get name i set name proporcionen accés controlat a _name.

Resum

  • Classes són plànols per crear objectes.
  • Objectes són instàncies de classes amb propietats i mètodes.
  • Constructors inicialitzen objectes.
  • Herència permet que una classe hereti d'una altra classe.
  • Polimorfisme permet que els mètodes actuïn de manera diferent segons l'objecte.
  • Encapsulació restringeix l'accés directe a les propietats dels objectes.

Entendre aquests conceptes és crucial per dominar la programació orientada a objectes a Dart. Practica creant classes, objectes i utilitzant l'herència per construir una base sòlida.
