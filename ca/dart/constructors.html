Introducció als Constructors

Els constructors són mètodes especials a Dart que s'utilitzen per crear i inicialitzar objectes. Es criden quan es crea una instància d'una classe. Entendre els constructors és essencial per gestionar la programació orientada a objectes a Dart.

Constructors per Defecte

Quan crees una classe a Dart, es proporciona automàticament un constructor per defecte si no defineixes cap constructor. Aquest constructor per defecte és un constructor sense arguments que inicialitza l'objecte.

class Person {
  String name;
  int age;
}

void main() {
  var person = Person(); // Es crida el constructor per defecte
  person.name = 'Alice';
  person.age = 30;
  print('Name: ${person.name}, Age: ${person.age}');
}

Constructors Personalitzats

Pots definir els teus propis constructors per inicialitzar objectes amb valors específics.

Constructor Personalitzat Bàsic

Un constructor personalitzat bàsic es pot definir creant un mètode amb el mateix nom que la classe.

class Person {
  String name;
  int age;

  // Constructor personalitzat
  Person(String name, int age) {
    this.name = name;
    this.age = age;
  }
}

void main() {
  var person = Person('Bob', 25);
  print('Name: ${person.name}, Age: ${person.age}');
}

Utilitzant la Paraula Clau `this`

Dart proporciona una forma abreujada per assignar els paràmetres del constructor a les variables d'instància utilitzant la paraula clau this.

class Person {
  String name;
  int age;

  // Constructor personalitzat utilitzant `this`
  Person(this.name, this.age);
}

void main() {
  var person = Person('Charlie', 40);
  print('Name: ${person.name}, Age: ${person.age}');
}

Constructors Anomenats

Dart et permet crear múltiples constructors per a una classe utilitzant constructors anomenats. Això és útil per proporcionar diferents maneres d'inicialitzar un objecte.

class Person {
  String name;
  int age;

  // Constructor per defecte
  Person(this.name, this.age);

  // Constructor anomenat
  Person.withName(this.name) {
    age = 0; // Edat per defecte
  }
}

void main() {
  var person1 = Person('Dave', 35);
  var person2 = Person.withName('Eve');
  print('Person1 - Name: ${person1.name}, Age: ${person1.age}');
  print('Person2 - Name: ${person2.name}, Age: ${person2.age}');
}

Constructors de Redirecció

Un constructor de redirecció crida un altre constructor de la mateixa classe. Això ajuda a evitar la duplicació de codi.

class Person {
  String name;
  int age;

  // Constructor principal
  Person(this.name, this.age);

  // Constructor de redirecció
  Person.withName(String name) : this(name, 0);
}

void main() {
  var person = Person.withName('Frank');
  print('Name: ${person.name}, Age: ${person.age}');
}

Constructors Constants

Si la teva classe produeix objectes que són immutables i idèntics, pots definir un constructor constant utilitzant la paraula clau const.

class Point {
  final int x;
  final int y;

  // Constructor constant
  const Point(this.x, this.y);
}

void main() {
  var point1 = const Point(2, 3);
  var point2 = const Point(2, 3);

  print(point1 == point2); // true, perquè són idèntics
}

Constructors de Fàbrica

Els constructors de fàbrica s'utilitzen quan necessites més control sobre el procés de creació d'objectes. Poden retornar una instància de la classe o una subclasse, o fins i tot una instància existent.

class Logger {
  final String name;
  static final Map<String, Logger> _cache = <String, Logger>{};

  // Constructor de fàbrica
  factory Logger(String name) {
    if (_cache.containsKey(name)) {
      return _cache[name];
    } else {
      final logger = Logger._internal(name);
      _cache[name] = logger;
      return logger;
    }
  }

  Logger._internal(this.name);
}

void main() {
  var logger1 = Logger('UI');
  var logger2 = Logger('UI');

  print(logger1 == logger2); // true, perquè són la mateixa instància
}

Resum

  • Constructors per Defecte: Proporcionats automàticament si no es defineixen constructors.
  • Constructors Personalitzats: Permeten inicialitzar objectes amb valors específics.
  • Constructors Anomenats: Proporcionen múltiples maneres d'inicialitzar un objecte.
  • Constructors de Redirecció: Criden un altre constructor de la mateixa classe.
  • Constructors Constants: Utilitzats per a objectes immutables i idèntics.
  • Constructors de Fàbrica: Proporcionen més control sobre la creació d'objectes.

Entendre i utilitzar els constructors de manera efectiva pot millorar enormement la teva capacitat per gestionar i inicialitzar objectes a Dart, fent el teu codi més robust i mantenible.
