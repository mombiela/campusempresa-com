Visió general

Dart és un llenguatge optimitzat per a clients per a aplicacions ràpides en qualsevol plataforma. És especialment adequat per al desenvolupament web gràcies al seu tipatge fort, sintaxi moderna i eines potents. En aquesta secció, us presentarem Dart per al desenvolupament web, cobrint els conceptes bàsics i proporcionant-vos els coneixements per començar a construir aplicacions web.

Conceptes clau

  • Conceptes bàsics del llenguatge Dart: Comprendre la sintaxi i l'estructura de Dart.
  • Dart per a Web: Com s'utilitza Dart en el desenvolupament web.
  • Configuració de Dart per al desenvolupament web: Eines i configuració de l'entorn.
  • Construcció d'una aplicació web senzilla: Guia pas a pas per crear una aplicació web bàsica.

Conceptes bàsics del llenguatge Dart

Abans d'endinsar-nos en el desenvolupament web amb Dart, és essencial entendre les característiques bàsiques del llenguatge.

Variables i tipus de dades

Dart és un llenguatge de tipatge estàtic, el que significa que cal declarar el tipus d'una variable.

void main() {
  int age = 25;
  double height = 5.9;
  String name = 'John Doe';
  bool isStudent = true;

  print('Nom: $name, Edat: $age, Alçada: $height, Estudiant: $isStudent');
}

Funcions

Les funcions en Dart són objectes de primera classe, el que significa que es poden assignar a variables o passar com a arguments a altres funcions.

void greet(String name) {
  print('Hola, $name!');
}

void main() {
  greet('Alice');
}

Flux de control

Dart admet les declaracions de flux de control típiques com if, else, for, while i switch.

void main() {
  int score = 85;

  if (score >= 90) {
    print('Nota: A');
  } else if (score >= 80) {
    print('Nota: B');
  } else {
    print('Nota: C');
  }
}

Dart per a Web

Dart es pot compilar a JavaScript, el que el converteix en una eina potent per al desenvolupament web. El marc principal per construir aplicacions web amb Dart és AngularDart, però també podeu utilitzar Dart senzill amb HTML.

Configuració de Dart per al desenvolupament web

Per començar a desenvolupar aplicacions web amb Dart, cal configurar l'entorn.

  1. Instal·lar Dart SDK: Descarregueu i instal·leu el Dart SDK des del lloc web oficial de Dart.
  2. Configurar un IDE: Utilitzeu un IDE com Visual Studio Code o IntelliJ IDEA amb plugins de Dart.
  3. Crear un nou projecte Dart: Utilitzeu l'eina de línia de comandes de Dart per crear un nou projecte web.
dart create -t web-simple my_web_app
cd my_web_app

Construcció d'una aplicació web senzilla

Construïm una aplicació web senzilla que mostri "Hello, Dart!" en una pàgina web.

  1. Estructura del projecte: El vostre projecte tindrà una estructura similar a aquesta:

    my_web_app/
├── web/
│   └── index.html
│   └── main.dart
└── pubspec.yaml

  2. index.html: Aquest fitxer conté l'estructura HTML de la vostra pàgina web.

    <!DOCTYPE html>
<html>
<head>
  <title>La meva aplicació web Dart</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <h1 id="output">Carregant...</h1>
</body>
</html>

  3. main.dart: Aquest fitxer conté el codi Dart que manipularà el DOM.

    import 'dart:html';

void main() {
  querySelector('#output')?.text = 'Hola, Dart!';
}

  4. Executar l'aplicació: Utilitzeu el servidor de desenvolupament de Dart per executar la vostra aplicació.

    webdev serve

    Obriu el vostre navegador i navegueu a http://localhost:8080 per veure la vostra aplicació web en acció.

Conclusió

En aquesta secció, us hem presentat Dart per al desenvolupament web. Hem cobert els conceptes bàsics del llenguatge Dart, com configurar el vostre entorn de desenvolupament i com construir una aplicació web senzilla. Amb aquesta base, ara esteu preparats per explorar temes més avançats en el desenvolupament web amb Dart, com l'ús de marcs com AngularDart, la gestió de l'estat i l'optimització del rendiment.
