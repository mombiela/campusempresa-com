Introducció a la Gestió de Paquets

La gestió de paquets és un aspecte crucial del desenvolupament de programari modern. Permet als desenvolupadors compartir i reutilitzar codi de manera eficient. En Dart, la gestió de paquets es maneja a través del Gestor de Paquets de Dart, pub. Aquesta secció et guiarà pels conceptes bàsics d'ús de pub per gestionar paquets en els teus projectes de Dart.

Entenent `pub`

pub és el gestor de paquets oficial per a Dart. T'ajuda a:

  • Instal·lar paquets.
  • Gestionar dependències.
  • Publicar els teus propis paquets.

Conceptes Clau

  • Paquet: Un conjunt de codi Dart que es pot compartir i reutilitzar.
  • Dependència: Un paquet del qual depèn el teu projecte.
  • pubspec.yaml: Un fitxer de configuració que especifica les dependències i altres metadades per al teu projecte Dart.

Ús de `pub` per Gestionar Paquets

Creant un Nou Projecte Dart

Per crear un nou projecte Dart, pots utilitzar la comanda dart create:

dart create my_project

Aquesta comanda genera un nou projecte Dart amb un fitxer pubspec.yaml.

Afegint Dependències

Per afegir una dependència, has de modificar el fitxer pubspec.yaml. Per exemple, per afegir el paquet http:

dependencies:
  http: ^0.13.3

Després d'afegir la dependència, executa:

dart pub get

Aquesta comanda obté els paquets llistats en el fitxer pubspec.yaml.

Exemple: Ús del Paquet `http`

Aquí tens un exemple senzill d'ús del paquet http per fer una sol·licitud GET:

import 'package:http/http.dart' as http;

void main() async {
  var url = Uri.parse('https://jsonplaceholder.typicode.com/posts/1');
  var response = await http.get(url);

  if (response.statusCode == 200) {
    print('Response data: ${response.body}');
  } else {
    print('Request failed with status: ${response.statusCode}');
  }
}

En aquest exemple:

  • Importem el paquet http.
  • Definim una URL i fem una sol·licitud GET.
  • Comprovem l'estat de la resposta i imprimim les dades de la resposta si la sol·licitud té èxit.

Gestionant Dependències

Actualitzant Dependències

Per actualitzar les teves dependències a les versions compatibles més recents, executa:

dart pub upgrade

Per actualitzar a les versions més recents independentment de la compatibilitat, utilitza:

dart pub upgrade --major-versions

Eliminant Dependències

Per eliminar una dependència, simplement esborra-la del fitxer pubspec.yaml i executa:

dart pub get

Publicant el Teu Propi Paquet

Creant un Paquet

Per crear un nou paquet, utilitza:

dart create -t package-simple my_package

Aquesta comanda crea un nou paquet amb una estructura bàsica.

Preparant per a la Publicació

Assegura't que el teu paquet compleixi els següents criteris:

  • Té un fitxer pubspec.yaml vàlid.
  • Inclou documentació.
  • Passa totes les proves.

Publicant el Paquet

Per publicar el teu paquet, executa:

dart pub publish

Segueix les instruccions per completar el procés de publicació.

Conclusió

La gestió de paquets en Dart, facilitada per pub, és essencial per a un desenvolupament eficient. Entenent com crear, gestionar i publicar paquets, pots aprofitar l'extens ecosistema de paquets de Dart per millorar els teus projectes. Ja sigui afegint dependències, actualitzant-les o compartint els teus propis paquets, pub proporciona les eines necessàries per agilitzar el teu flux de treball.
