Introducció

En aquesta secció, proporcionarem una visió general del projecte en el qual treballaràs al llarg d'aquest curs de Dart. Aquest projecte està dissenyat per ajudar-te a aplicar els conceptes que aprens d'una manera pràctica i pràctica. Al final del curs, tindràs una aplicació completament funcional construïda amb Dart.

Descripció del Projecte

El projecte en el qual treballaràs és una Aplicació de Gestió de Tasques. Aquesta aplicació permetrà als usuaris crear, llegir, actualitzar i eliminar tasques. També inclourà funcions com la categorització de tasques, dates de venciment i nivells de prioritat.

Característiques Clau

  • Autenticació d'Usuaris: Els usuaris podran registrar-se, iniciar sessió i tancar sessió.
  • Operacions CRUD de Tasques: Els usuaris poden crear, llegir, actualitzar i eliminar tasques.
  • Categorització de Tasques: Les tasques es poden categoritzar en diferents grups.
  • Dates de Venciment i Nivells de Prioritat: Les tasques poden tenir dates de venciment i nivells de prioritat assignats.
  • Disseny Responsiu: L'aplicació serà responsiva i funcionarà en diferents mides de pantalla.

Estructura del Projecte

El projecte es dividirà en diversos mòduls, cadascun centrat en diferents aspectes de la programació en Dart i el desenvolupament d'aplicacions. Aquí tens una descripció dels mòduls:

  1. Mòdul 1: Introducció a Dart

    • Sintaxi bàsica
    • Variables i tipus de dades
    • Sentències de control de flux

  2. Mòdul 2: Programació Orientada a Objectes en Dart

    • Classes i objectes
    • Herència i polimorfisme
    • Interfícies i classes abstractes

  3. Mòdul 3: Col·leccions i Genèrics en Dart

    • Llistes, conjunts i mapes
    • Genèrics i seguretat de tipus

  4. Mòdul 4: Programació Asíncrona

    • Futures i async/await
    • Streams

  5. Mòdul 5: Construcció de la Interfície d'Usuari

    • Introducció a Flutter (si escau)
    • Widgets i disseny
    • Gestió d'estat

  6. Mòdul 6: Xarxes i Emmagatzematge de Dades

    • Sol·licituds HTTP
    • Anàlisi de JSON
    • Emmagatzematge local

  7. Mòdul 7: Autenticació d'Usuaris

    • Implementació de la funcionalitat de registre i inici de sessió
    • Emmagatzematge segur de les credencials dels usuaris

  8. Mòdul 8: Funcions Avançades

    • Categorització de tasques
    • Dates de venciment i nivells de prioritat
    • Tècniques de disseny responsiu

Codi d'Exemple

Aquí tens un exemple senzill per donar-te una idea del que construiràs. Aquest fragment de codi demostra com crear una classe bàsica de Dart per a una Tasca:

class Task {
  String title;
  String description;
  DateTime dueDate;
  int priority;

  Task(this.title, this.description, this.dueDate, this.priority);

  void displayTask() {
    print('Title: $title');
    print('Description: $description');
    print('Due Date: $dueDate');
    print('Priority: $priority');
  }
}

void main() {
  Task task = Task('Complete Dart Course', 'Finish all modules and exercises', DateTime(2023, 12, 31), 1);
  task.displayTask();
}

Explicació

  • Definició de la Classe: Definim una classe Task amb propietats per al títol, descripció, data de venciment i prioritat.
  • Constructor: El constructor inicialitza aquestes propietats.
  • Mètode: El mètode displayTask imprimeix els detalls de la tasca.
  • Funció Principal: Creem una instància de la classe Task i cridem el mètode displayTask per mostrar els detalls de la tasca.

Conclusió

Al final d'aquest curs, tindràs una comprensió completa de la programació en Dart i seràs capaç de construir una Aplicació de Gestió de Tasques completament funcional. Cada mòdul està dissenyat per construir sobre l'anterior, assegurant que adquireixis una base sòlida abans de passar a temes més avançats. Feliç programació!
