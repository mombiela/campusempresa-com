Configurar el teu projecte Dart és el primer pas per construir aplicacions utilitzant el llenguatge de programació Dart. Aquesta secció et guiarà a través del procés d'instal·lació de Dart, configuració del teu entorn de desenvolupament i creació del teu primer projecte Dart.

Instal·lació de Dart

Abans de començar a escriure codi Dart, necessites instal·lar el Dart SDK (Software Development Kit) a la teva màquina. El Dart SDK inclou la Dart VM, les biblioteques bàsiques de Dart i diverses eines de línia de comandes.

Passos per Instal·lar Dart

  1. Visita la Pàgina de Descàrrega del Dart SDK:

    • Ves a la pàgina oficial de descàrrega del Dart SDK: Dart SDK.

  2. Tria el Teu Sistema Operatiu:

    • Selecciona el mètode d'instal·lació adequat per al teu sistema operatiu (Windows, macOS o Linux).

  3. Segueix les Instruccions d'Instal·lació:

    • Windows: Descarrega el fitxer zip del Dart SDK, extreu-lo i afegeix el directori bin al PATH del teu sistema.
    • macOS: Utilitza Homebrew per instal·lar Dart executant la comanda: 
      brew tap dart-lang/dart
brew install dart
    • Linux: Utilitza el gestor de paquets apt per instal·lar Dart executant les comandes: 
      sudo apt update
sudo apt install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt update
sudo apt install dart

  4. Verifica la Instal·lació:

    • Obre un terminal o símbol del sistema i executa: 
      dart --version
    • Hauries de veure la versió de Dart instal·lada.

Configuració del Teu Entorn de Desenvolupament

Per escriure i executar codi Dart de manera eficient, necessites un bon entorn de desenvolupament. Visual Studio Code (VS Code) és una elecció popular a causa de les seves potents característiques i el seu extens suport de plugins.

Instal·lació de Visual Studio Code

  1. Descarrega i Instal·la VS Code:

    • Visita la pàgina de descàrrega de Visual Studio Code i descarrega l'instal·lador per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació per instal·lar VS Code.

  2. Instal·la el Plugin de Dart:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca "Dart" i fes clic al botó d'Instal·lar per al plugin de Dart.

Creació del Teu Primer Projecte Dart

Ara que tens Dart i VS Code instal·lats, pots crear el teu primer projecte Dart.

Passos per Crear un Projecte Dart

  1. Obre VS Code:

    • Llança Visual Studio Code.

  2. Obre la Paleta de Comandes:

    • Prem Ctrl+Shift+P per obrir la Paleta de Comandes.

  3. Crea un Nou Projecte Dart:

    • Escriu Dart: New Project i selecciona-ho de la llista.
    • Tria Console Application quan se't demani.
    • Selecciona una ubicació per al teu nou projecte i proporciona un nom per a aquest.

  4. Explora l'Estructura del Projecte:

    • VS Code crearà un nou projecte Dart amb la següent estructura: 
      my_project/
├── bin/
│   └── main.dart
├── lib/
├── test/
└── pubspec.yaml
    • bin/main.dart: El punt d'entrada principal de la teva aplicació.
    • lib/: Directori per a biblioteques reutilitzables.
    • test/: Directori per a proves unitàries.
    • pubspec.yaml: Fitxer de configuració per a les dependències del teu projecte.

Executar la Teva Aplicació Dart

  1. Obre main.dart:

    • Navega a bin/main.dart i obre'l.

  2. Escriu el Teu Primer Codi Dart:

    • Substitueix el codi existent pel següent: 
      void main() {
  print('Hello, Dart!');
}

  3. Executa l'Aplicació:

    • Obre el terminal a VS Code prement `Ctrl+``.
    • Assegura't que estàs al directori del projecte i executa: 
      dart run bin/main.dart
    • Hauries de veure la sortida: 
      Hello, Dart!

Conclusió

Configurar un projecte Dart implica instal·lar el Dart SDK, configurar un entorn de desenvolupament com Visual Studio Code i crear un nou projecte Dart. Seguint aquests passos, has configurat amb èxit el teu primer projecte Dart i has executat una simple aplicació Dart. Aquesta configuració fonamental et permetrà començar a explorar i construir aplicacions Dart més complexes a mesura que progressis en el curs.
