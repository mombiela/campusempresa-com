Introducció als Conjunts

Els conjunts són una col·lecció d'elements únics. A diferència de les llistes, els conjunts no permeten elements duplicats. En Dart, els conjunts estan representats per la classe Set.

Conceptes Clau

  • Unicitat: Cada element en un conjunt ha de ser únic.
  • Desordenat: Els elements en un conjunt no s'emmagatzemen en cap ordre particular.
  • Literals de Conjunt: Els conjunts es poden crear utilitzant literals de conjunt, similars als literals de llista però amb claus.

Creació de Conjunts

Pots crear un conjunt en Dart utilitzant el constructor Set o literals de conjunt.

Utilitzant el Constructor Set

void main() {
  Set<int> numbers = Set();
  numbers.add(1);
  numbers.add(2);
  numbers.add(3);
  print(numbers); // Output: {1, 2, 3}
}

Utilitzant Literals de Conjunt

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits); // Output: {apple, banana, orange}
}

Operacions Bàsiques amb Conjunts

Afegir Elements

Pots afegir elements a un conjunt utilitzant el mètode add.

void main() {
  Set<int> numbers = {1, 2, 3};
  numbers.add(4);
  print(numbers); // Output: {1, 2, 3, 4}
}

Eliminar Elements

Pots eliminar elements utilitzant el mètode remove.

void main() {
  Set<int> numbers = {1, 2, 3, 4};
  numbers.remove(3);
  print(numbers); // Output: {1, 2, 4}
}

Comprovar Elements

Pots comprovar si un conjunt conté un element utilitzant el mètode contains.

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits.contains('banana')); // Output: true
  print(fruits.contains('grape')); // Output: false
}

Operacions Avançades amb Conjunts

Unió

La unió de dos conjunts és un conjunt que conté tots els elements de tots dos conjunts.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> unionSet = setA.union(setB);
  print(unionSet); // Output: {1, 2, 3, 4, 5}
}

Intersecció

La intersecció de dos conjunts és un conjunt que conté només els elements que estan presents en tots dos conjunts.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> intersectionSet = setA.intersection(setB);
  print(intersectionSet); // Output: {3}
}

Diferència

La diferència entre dos conjunts és un conjunt que conté els elements que estan en el primer conjunt però no en el segon conjunt.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> differenceSet = setA.difference(setB);
  print(differenceSet); // Output: {1, 2}
}

Iterar sobre Conjunts

Pots iterar sobre un conjunt utilitzant un bucle for o un mètode forEach.

Utilitzant un Bucle For

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  for (var fruit in fruits) {
    print(fruit);
  }
}

Utilitzant el Mètode forEach

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  fruits.forEach((fruit) => print(fruit));
}

Conclusió

Els conjunts en Dart són una manera poderosa de gestionar col·leccions d'elements únics. Proporcionen diversos mètodes per afegir, eliminar i comprovar elements, així com per realitzar operacions avançades com la unió, la intersecció i la diferència. Entendre els conjunts i les seves operacions pot ajudar-te a escriure codi Dart més eficient i efectiu.

En dominar els conjunts, pots gestionar col·leccions úniques de dades amb facilitat, assegurant que les teves aplicacions siguin tant eficients com lògicament sòlides.
