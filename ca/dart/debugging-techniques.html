Tècniques de Depuració
Organització del Codi ►

La depuració és una habilitat essencial per a qualsevol programador. Aquesta secció et guiarà a través de diverses tècniques de depuració en Dart, des de nivells bàsics fins a avançats. Al final d'aquesta secció, estaràs equipat amb el coneixement per identificar i solucionar errors de manera eficient en les teves aplicacions Dart.

Introducció a la Depuració

La depuració és el procés d'identificar i eliminar errors dels programes informàtics. En Dart, com en qualsevol altre llenguatge de programació, la depuració és crucial per assegurar que el teu codi s'executi correctament i eficientment.

Conceptes Clau

  • Error: Un error, defecte o fallada en un programa que fa que produeixi resultats incorrectes o inesperats.
  • Depuració: El procés de trobar i solucionar errors.
  • Punt de trencament: Un marcador establert en el codi on el depurador pausarà l'execució, permetent-te inspeccionar l'estat del programa.
  • Traça de la pila: Un informe que proporciona informació sobre els marcs de pila actius en un moment determinat durant l'execució d'un programa.

Tècniques Bàsiques de Depuració

Declaracions Print

Una de les tècniques de depuració més simples i comunes és utilitzar declaracions print per mostrar els valors de les variables i el flux d'execució.

void main() {
  int a = 5;
  int b = 10;
  int sum = add(a, b);
  print('Sum: $sum'); // Depurant amb declaracions print
}

int add(int x, int y) {
  print('Adding $x and $y'); // Depurant amb declaracions print
  return x + y;
}

Utilitzant les Dart DevTools

Dart DevTools és un conjunt d'eines de rendiment i depuració per a Dart i Flutter. Proporciona un conjunt ric de funcions per ajudar-te a depurar les teves aplicacions.

  • Inspeccionar Variables: Veure els valors de les variables en diferents punts del teu codi.
  • Establir Punts de Trencament: Pausar l'execució en línies específiques del codi per inspeccionar l'estat del programa.
  • Passar a Través del Codi: Executar el teu codi línia per línia per veure com es comporta.

Tècniques Intermèdies de Depuració

Punts de Trencament i Passar a Través del Codi

Utilitzar punts de trencament i passar a través del codi et permet pausar l'execució i inspeccionar l'estat de la teva aplicació en punts específics.

void main() {
  int a = 5;
  int b = 10;
  int sum = add(a, b);
  print('Sum: $sum');
}

int add(int x, int y) {
  // Estableix un punt de trencament aquí
  return x + y;
}
  • Passar per Sobre: Executar la següent línia de codi, però no entrar en les crides de funció.
  • Entrar: Entrar en la crida de funció per veure la seva execució.
  • Sortir: Completar l'execució de la funció actual i tornar al cridant.

Analitzant Traces de la Pila

Les traces de la pila proporcionen informació valuosa sobre la seqüència de crides de funció que han portat a un error. Entendre com llegir les traces de la pila pot ajudar-te a identificar ràpidament la font d'un error.

void main() {
  try {
    int result = divide(10, 0);
    print('Result: $result');
  } catch (e, stackTrace) {
    print('Error: $e');
    print('Stack Trace: $stackTrace');
  }
}

int divide(int a, int b) {
  return a ~/ b; // Això llançarà una excepció si b és 0
}

Tècniques Avançades de Depuració

Utilitzant Llibreries de Registre

Per a una depuració més avançada, pots utilitzar llibreries de registre per proporcionar un registre més estructurat i configurable.

import 'package:logging/logging.dart';

void main() {
  _setupLogging();
  int a = 5;
  int b = 10;
  int sum = add(a, b);
  Logger('Main').info('Sum: $sum');
}

void _setupLogging() {
  Logger.root.level = Level.ALL;
  Logger.root.onRecord.listen((record) {
    print('${record.level.name}: ${record.time}: ${record.message}');
  });
}

int add(int x, int y) {
  Logger('Add').fine('Adding $x and $y');
  return x + y;
}

Depurant Codi Asíncron

Depurar codi asíncron pot ser un repte a causa del flux d'execució no lineal. Dart proporciona eines per ajudar-te a depurar operacions asíncrones.

import 'dart:async';

void main() async {
  try {
    await fetchData();
  } catch (e, stackTrace) {
    print('Error: $e');
    print('Stack Trace: $stackTrace');
  }
}

Future<void> fetchData() async {
  await Future.delayed(Duration(seconds: 2));
  throw Exception('Data fetch failed');
}

Conclusió

La depuració és una habilitat crítica per a qualsevol programador Dart. En dominar les tècniques cobertes en aquesta secció, estaràs millor equipat per identificar i solucionar errors en el teu codi. Recorda començar amb tècniques simples com les declaracions print i avançar gradualment cap a eines més avançades com les Dart DevTools i les llibreries de registre segons sigui necessari. Depurar codi asíncron i entendre les traces de la pila també són habilitats essencials per a la depuració avançada. Feliç codificació!
