Les expressions lambda són una característica potent en C# que et permeten escriure expressions o funcions en línia. Són particularment útils per escriure codi concís i llegible, especialment quan es treballa amb col·leccions i LINQ (Language Integrated Query). Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de les expressions lambda en C#.

Què són les expressions lambda?

  • Definició: Una expressió lambda és una funció anònima que pots utilitzar per crear delegats o tipus d'arbres d'expressions.
  • Sintaxi: La sintaxi bàsica d'una expressió lambda és (paràmetres d'entrada) => expressió o bloc d'instruccions.

Sintaxi bàsica

(paràmetres) => expressió
(paràmetres) => { instruccions }

Exemple

Func<int, int> square = x => x * x;
Console.WriteLine(square(5)); // Sortida: 25

Per què utilitzar expressions lambda?

  • Concisión: Les expressions lambda et permeten escriure menys codi.
  • Llegibilitat: Poden fer que el teu codi sigui més llegible eliminant codi redundant.
  • Flexibilitat: Són versàtils i es poden utilitzar en diversos escenaris, com ara consultes LINQ, maneig d'esdeveniments, i més.

Ús bàsic

Un sol paràmetre

Func<int, int> increment = x => x + 1;
Console.WriteLine(increment(5)); // Sortida: 6

Múltiples paràmetres

Func<int, int, int> add = (x, y) => x + y;
Console.WriteLine(add(3, 4)); // Sortida: 7

Sense paràmetres

Action greet = () => Console.WriteLine("Hello, World!");
greet(); // Sortida: Hello, World!

Ús avançat

Utilitzant expressions lambda amb LINQ

Les expressions lambda s'utilitzen extensament amb LINQ per realitzar operacions sobre col·leccions.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };
var evenNumbers = numbers.Where(n => n % 2 == 0).ToList();
evenNumbers.ForEach(n => Console.WriteLine(n)); // Sortida: 2 4

Arbres d'expressions

Les expressions lambda es poden utilitzar per crear arbres d'expressions, que són estructures de dades que representen codi.

Expression<Func<int, bool>> isEven = x => x % 2 == 0;
Console.WriteLine(isEven); // Sortida: x => (x % 2 == 0)

Capturant variables

Les expressions lambda poden capturar variables del seu àmbit envoltant.

int factor = 2;
Func<int, int> multiply = x => x * factor;
Console.WriteLine(multiply(3)); // Sortida: 6

Usos comuns

Maneig d'esdeveniments

Les expressions lambda es poden utilitzar per manejar esdeveniments de manera concisa.

button.Click += (sender, e) => { Console.WriteLine("Button clicked!"); };

Ordenació de col·leccions

Pots utilitzar expressions lambda per definir lògica d'ordenació personalitzada.

List<string> names = new List<string> { "Alice", "Bob", "Charlie" };
names.Sort((a, b) => a.Length.CompareTo(b.Length));
names.ForEach(name => Console.WriteLine(name)); // Sortida: Bob Alice Charlie

Conclusió

Les expressions lambda són una característica versàtil i potent en C#. Et permeten escriure codi concís i llegible, especialment quan es treballa amb col·leccions i LINQ. Comprenent i utilitzant expressions lambda, pots millorar significativament l'eficiència i la llegibilitat del teu codi.

Resum

  • Les expressions lambda són funcions anònimes que es poden utilitzar per crear delegats o tipus d'arbres d'expressions.
  • Proporcionen una manera concisa i llegible d'escriure expressions o funcions en línia.
  • Les expressions lambda s'utilitzen extensament amb LINQ, maneig d'esdeveniments, i ordenació personalitzada.
  • Comprendre les expressions lambda pot millorar enormement les teves habilitats de programació en C#.

Dominant les expressions lambda, estaràs ben equipat per escriure codi en C# més eficient i mantenible.
