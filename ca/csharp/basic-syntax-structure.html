Introducció

C# és un llenguatge de programació modern i orientat a objectes desenvolupat per Microsoft. S'utilitza àmpliament per desenvolupar aplicacions d'escriptori, aplicacions web i jocs. Entendre la sintaxi i l'estructura bàsiques de C# és essencial per escriure codi eficient i lliure d'errors.

Conceptes Clau

1. Programa Hola Món

La millor manera de començar a aprendre qualsevol llenguatge de programació és escrivint un simple programa "Hola, Món!". Aquest programa t'introduirà a l'estructura bàsica d'un programa en C#.

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
  • using System;: Aquesta línia importa l'espai de noms System, que conté classes fonamentals per a operacions bàsiques d'entrada i sortida.
  • namespace HelloWorld: Els espais de noms s'utilitzen per organitzar el codi i evitar conflictes de noms.
  • class Program: Les classes són els blocs de construcció dels programes en C#.
  • static void Main(string[] args): El mètode Main és el punt d'entrada d'una aplicació en C#. És on el programa comença l'execució.
  • Console.WriteLine("Hello, World!");: Aquesta línia imprimeix "Hello, World!" a la consola.

2. Variables i Tipus de Dades

Les variables s'utilitzen per emmagatzemar dades, i els tipus de dades especifiquen el tipus de dades que es poden emmagatzemar en una variable.

Tipus de Dades Comuns

| Tipus de Dades | Descripció | Exemple | |----------------|-----------------------------|---------------| | int | Nombres enters | 1, 2, 3 | | double | Nombres de punt flotant | 1.23, 4.56 | | char | Caràcter únic | 'a', 'b' | | string | Seqüència de caràcters | "Hola" | | bool | Valors booleans (true/false)| true, false |

Exemple

int age = 25;
double salary = 50000.50;
char grade = 'A';
string name = "John";
bool isEmployed = true;

3. Operadors

Els operadors s'utilitzen per realitzar operacions sobre variables i valors.

Operadors Comuns

  • Operadors Aritmètics: +, -, *, /, %
  • Operadors de Comparació: ==, !=, >, <, >=, <=
  • Operadors Lògics: &&, ||, !

Exemple

int a = 10;
int b = 20;
int sum = a + b; // 30
bool isEqual = (a == b); // false
bool isGreater = (a > b); // false
bool logicalAnd = (a < b && b > 15); // true

4. Estructures de Control

Les estructures de control s'utilitzen per controlar el flux d'execució en un programa.

Instrucció If-Else

int number = 10;
if (number > 0)
{
    Console.WriteLine("Positive number");
}
else
{
    Console.WriteLine("Non-positive number");
}

Instrucció Switch

int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    default:
        Console.WriteLine("Other day");
        break;
}

Bucles

  • Bucle For
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}
  • Bucle While
int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}
  • Bucle Do-While
int i = 0;
do
{
    Console.WriteLine(i);
    i++;
} while (i < 5);

5. Mètodes

Els mètodes són blocs de codi que realitzen una tasca específica i es poden cridar quan sigui necessari.

Exemple

class Program
{
    static void Main(string[] args)
    {
        Greet("John");
    }

    static void Greet(string name)
    {
        Console.WriteLine("Hello, " + name);
    }
}

Conclusió

Entendre la sintaxi i l'estructura bàsiques de C# és crucial per a qualsevol principiant. Això inclou saber com escriure un programa simple, declarar variables, utilitzar operadors, controlar el flux del programa amb estructures de control i definir mètodes. El domini d'aquests conceptes bàsics proporcionarà una base sòlida per a temes més avançats en C#.
