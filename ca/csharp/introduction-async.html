La programació asíncrona és una tècnica poderosa que permet als teus programes realitzar tasques de manera concurrent, millorant l'eficiència i la capacitat de resposta. Aquest tema et guiarà des dels conceptes bàsics fins als avançats de la programació asíncrona en C#.

Què és la Programació Asíncrona?

La programació asíncrona permet a un programa realitzar tasques sense bloquejar el fil principal. Això és particularment útil per a operacions d'E/S, com llegir d'un fitxer o fer sol·licituds de xarxa, on esperar que l'operació es completi bloquejaria l'execució d'altres tasques.

Conceptes Clau

  • Síncron vs Asíncron: Les operacions síncrones bloquegen l'execució fins que la tasca es completa, mentre que les operacions asíncrones permeten que el programa continuï executant altres tasques.
  • Concurrència: La capacitat de realitzar múltiples tasques simultàniament.
  • Filat: Utilitzar múltiples fils per aconseguir la concurrència.
  • Patró Asíncron Basat en Tasques (TAP): Un model en C# per escriure codi asíncron utilitzant els tipus Task i Task<T>.

Programació Asíncrona en C#

C# proporciona diverses maneres d'implementar la programació asíncrona, principalment a través de les paraules clau async i await, i la classe Task.

Les Paraules Clau `async` i `await`

La paraula clau async s'utilitza per definir un mètode asíncron, i la paraula clau await s'utilitza per pausar l'execució del mètode fins que la tasca esperada es completi.

Exemple

using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Començant...");
        await PerformTaskAsync();
        Console.WriteLine("Acabat.");
    }

    static async Task PerformTaskAsync()
    {
        Console.WriteLine("Tasca començada.");
        await Task.Delay(2000); // Simula un retard
        Console.WriteLine("Tasca completada.");
    }
}

Explicació:

  • El mètode Main està marcat amb async i retorna una Task.
  • await Task.Delay(2000) pausa el mètode durant 2 segons sense bloquejar el fil principal.

Patró Asíncron Basat en Tasques (TAP)

TAP és el patró recomanat per a la programació asíncrona en C#. Utilitza els tipus Task i Task<T> per representar operacions asíncrones.

Exemple

using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        string url = "https://jsonplaceholder.typicode.com/posts/1";
        string result = await FetchDataAsync(url);
        Console.WriteLine(result);
    }

    static async Task<string> FetchDataAsync(string url)
    {
        using HttpClient client = new HttpClient();
        HttpResponseMessage response = await client.GetAsync(url);
        response.EnsureSuccessStatusCode();
        return await response.Content.ReadAsStringAsync();
    }
}

Explicació:

  • El mètode FetchDataAsync obté dades d'una URL de manera asíncrona.
  • await client.GetAsync(url) envia una sol·licitud HTTP GET i espera la resposta.
  • await response.Content.ReadAsStringAsync() llegeix el contingut de la resposta com una cadena.

Conceptes Avançats

Gestió d'Excepcions

Els mètodes asíncrons poden llençar excepcions, que cal gestionar correctament.

Exemple

using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        try
        {
            string url = "https://jsonplaceholder.typicode.com/posts/1";
            string result = await FetchDataAsync(url);
            Console.WriteLine(result);
        }
        catch (HttpRequestException e)
        {
            Console.WriteLine($"Error de sol·licitud: {e.Message}");
        }
    }

    static async Task<string> FetchDataAsync(string url)
    {
        using HttpClient client = new HttpClient();
        HttpResponseMessage response = await client.GetAsync(url);
        response.EnsureSuccessStatusCode();
        return await response.Content.ReadAsStringAsync();
    }
}

Explicació:

  • El bloc try-catch en el mètode Main captura qualsevol HttpRequestException llençada per FetchDataAsync.

Programació Paral·lela

La programació paral·lela implica executar múltiples operacions simultàniament. La classe Parallel en C# proporciona mètodes per a l'execució paral·lela.

Exemple

using System;
using System.Threading.Tasks;

class Program
{
    static void Main(string[] args)
    {
        Parallel.For(0, 10, i =>
        {
            Console.WriteLine($"Processant {i}");
        });
    }
}

Explicació:

  • Parallel.For executa el cos del bucle en paral·lel, processant múltiples iteracions de manera concurrent.

Conclusió

La programació asíncrona en C# és essencial per construir aplicacions responsives i eficients. Entenent i utilitzant les paraules clau async i await, el Patró Asíncron Basat en Tasques, i gestionant les excepcions correctament, pots escriure codi asíncron robust. Els conceptes avançats com la programació paral·lela milloren encara més el rendiment de les teves aplicacions. Practica aquests conceptes amb escenaris del món real per dominar la programació asíncrona en C#.
