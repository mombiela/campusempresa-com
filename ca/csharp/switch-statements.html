Introducció

Les instruccions switch en C# proporcionen una manera d'executar diferents blocs de codi basats en el valor d'una variable o expressió. Són una alternativa a l'ús de múltiples instruccions if-else i poden fer que el teu codi sigui més net i llegible.

Sintaxi Bàsica

La sintaxi bàsica d'una instrucció switch en C# és la següent:

switch (expression)
{
    case value1:
        // Codi a executar si expression == value1
        break;
    case value2:
        // Codi a executar si expression == value2
        break;
    // Casos addicionals...
    default:
        // Codi a executar si cap dels casos anteriors coincideix
        break;
}

Exemple

Aquí tens un exemple senzill per il·lustrar l'ús d'una instrucció switch:

int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Dilluns");
        break;
    case 2:
        Console.WriteLine("Dimarts");
        break;
    case 3:
        Console.WriteLine("Dimecres");
        break;
    case 4:
        Console.WriteLine("Dijous");
        break;
    case 5:
        Console.WriteLine("Divendres");
        break;
    case 6:
        Console.WriteLine("Dissabte");
        break;
    case 7:
        Console.WriteLine("Diumenge");
        break;
    default:
        Console.WriteLine("Dia invàlid");
        break;
}

Conceptes Clau

  • Expressió: El valor o variable que s'està avaluant.
  • Cas: Cada valor possible de l'expressió que vols gestionar.
  • Break: Termina el bloc de codi del cas actual.
  • Default: Opcional; s'executa si cap dels casos coincideix amb l'expressió.

Ús Avançat

Coincidència de Patrons

C# 7.0 va introduir la coincidència de patrons en les instruccions switch, permetent condicions més complexes.

object obj = "Hola";

switch (obj)
{
    case int i:
        Console.WriteLine($"Enter: {i}");
        break;
    case string s:
        Console.WriteLine($"Cadena: {s}");
        break;
    default:
        Console.WriteLine("Tipus desconegut");
        break;
}

Expressions Switch

C# 8.0 va introduir les expressions switch, que proporcionen una sintaxi més concisa.

int number = 3;
string result = number switch
{
    1 => "Un",
    2 => "Dos",
    3 => "Tres",
    _ => "Desconegut"
};

Console.WriteLine(result); // Sortida: Tres

Errors Comuns

  • Oblidar la instrucció break: Això pot portar a un comportament de caiguda, on s'executen múltiples casos.
  • No gestionar tots els valors possibles: Sempre inclou un cas default per gestionar valors inesperats.

Exercicis

  1. Instrucció Switch Bàsica: Escriu una instrucció switch que imprimeixi el nom d'un mes basat en una entrada d'enter (1-12).
  2. Coincidència de Patrons: Crea una instrucció switch que gestioni diferents tipus d'objectes (per exemple, int, string, bool) i imprimeixi un missatge per a cada tipus.
  3. Expressió Switch: Utilitza una expressió switch per convertir una nota (A, B, C, D, F) a una descripció corresponent (Excel·lent, Bo, Mitjà, Pobre, Suspès).

Conclusió

Les instruccions switch són una eina poderosa en C# per gestionar múltiples condicions d'una manera neta i llegible. Entenent la sintaxi bàsica i les característiques avançades com la coincidència de patrons i les expressions switch, pots escriure codi més eficient i mantenible. Recorda sempre gestionar tots els casos possibles i utilitzar el cas default per cobrir valors inesperats.
