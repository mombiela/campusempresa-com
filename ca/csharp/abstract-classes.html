Introducció

Les classes abstractes en C# són un concepte fonamental en la programació orientada a objectes. Proporcionen una manera de definir una classe base que no es pot instanciar per si sola, però que es pot utilitzar com a fonament per a altres classes. Aquest tema cobrirà els conceptes bàsics de les classes abstractes, el seu propòsit i com utilitzar-les de manera efectiva en els teus programes en C#.

Conceptes clau

  • Classe abstracta: Una classe que no es pot instanciar i està destinada a ser una classe base per a altres classes.
  • Mètode abstracte: Un mètode declarat en una classe abstracta sense una implementació, que ha de ser implementat per les classes derivades.
  • Herència: El mecanisme pel qual una classe pot heretar les propietats i mètodes d'una altra classe.

Definició d'una classe abstracta

Una classe abstracta es defineix utilitzant la paraula clau abstract. Aquí hi ha un exemple senzill:

public abstract class Animal
{
    public abstract void MakeSound(); // Mètode abstracte
    public void Sleep()
    {
        Console.WriteLine("Sleeping...");
    }
}

En aquest exemple:

  • Animal és una classe abstracta.
  • MakeSound és un mètode abstracte que ha de ser implementat per qualsevol classe no abstracta que hereti d'Animal.
  • Sleep és un mètode regular amb una implementació.

Implementació d'una classe abstracta

Per utilitzar una classe abstracta, cal crear una classe derivada que hereti d'ella i proporcioni implementacions per als mètodes abstractes.

public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Bark");
    }
}

En aquest exemple:

  • Dog és una classe que hereta d'Animal.
  • MakeSound està implementat a la classe Dog.

Exemple d'ús

Aquí hi ha un exemple complet que demostra com utilitzar classes i mètodes abstractes:

using System;

public abstract class Animal
{
    public abstract void MakeSound();
    public void Sleep()
    {
        Console.WriteLine("Sleeping...");
    }
}

public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Bark");
    }
}

public class Cat : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Meow");
    }
}

public class Program
{
    public static void Main()
    {
        Animal myDog = new Dog();
        Animal myCat = new Cat();

        myDog.MakeSound(); // Sortida: Bark
        myDog.Sleep();     // Sortida: Sleeping...

        myCat.MakeSound(); // Sortida: Meow
        myCat.Sleep();     // Sortida: Sleeping...
    }
}

Punts clau a recordar

  • Les classes abstractes no es poden instanciar directament.
  • Els mètodes abstractes no tenen cos i han de ser implementats a les classes derivades.
  • Una classe que hereti d'una classe abstracta ha d'implementar tots els mètodes abstractes o ser declarada abstracta ella mateixa.
  • Les classes abstractes poden contenir tant mètodes abstractes com no abstractes.

Conclusió

Les classes abstractes són una característica poderosa en C# que et permeten definir una classe base comuna amb funcionalitat compartida mentre s'obliga a implementar certs mètodes a les classes derivades. Això promou la reutilització de codi i assegura una interfície consistent a través de diferents implementacions. Entendre i utilitzar les classes abstractes de manera efectiva pot millorar enormement el disseny i la mantenibilitat de les teves aplicacions en C#.
