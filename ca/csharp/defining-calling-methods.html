Introducció

Els mètodes són blocs fonamentals en C#. Permeten encapsular codi en blocs reutilitzables, fent que els teus programes siguin més modulars i fàcils de mantenir. En aquesta secció, cobrim com definir i cridar mètodes, des de conceptes bàsics fins a avançats.

Definició de Mètodes

Definició Bàsica de Mètodes

Un mètode en C# es defineix dins d'una classe i consisteix en una signatura i un cos. La signatura inclou el modificador d'accés, el tipus de retorn, el nom del mètode i els paràmetres (si n'hi ha).

public class Program
{
    // Definició del mètode
    public void SayHello()
    {
        Console.WriteLine("Hello, World!");
    }
}
  • Modificador d'Accés: Determina la visibilitat del mètode (per exemple, public, private).
  • Tipus de Retorn: Especifica el tipus de valor que retorna el mètode (per exemple, void si no es retorna cap valor).
  • Nom del Mètode: Un identificador únic per al mètode.
  • Paràmetres: Opcional; s'utilitzen per passar dades al mètode.

Mètode amb Paràmetres

Els mètodes poden prendre paràmetres per realitzar operacions sobre dades d'entrada.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"Hello, {name}!");
    }
}
  • Paràmetres: Definits dins dels parèntesis i separats per comes si n'hi ha múltiples.

Mètode amb Valor de Retorn

Els mètodes poden retornar valors al cridant.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }
}
  • Sentència de Retorn: S'utilitza per retornar un valor des del mètode.

Crida de Mètodes

Crida d'un Mètode sense Paràmetres

Per cridar un mètode, utilitzes el nom del mètode seguit de parèntesis.

public class Program
{
    public void SayHello()
    {
        Console.WriteLine("Hello, World!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.SayHello(); // Crida del mètode
    }
}

Crida d'un Mètode amb Paràmetres

Quan crides un mètode amb paràmetres, passes els arguments dins dels parèntesis.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"Hello, {name}!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.GreetUser("Alice"); // Crida del mètode amb argument
    }
}

Crida d'un Mètode amb Valor de Retorn

Quan crides un mètode que retorna un valor, pots emmagatzemar el resultat en una variable.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }

    public static void Main()
    {
        Program program = new Program();
        int result = program.Add(5, 3); // Crida del mètode amb valor de retorn
        Console.WriteLine(result); // Sortida: 8
    }
}

Conceptes Avançats

Sobrecàrrega de Mètodes

La sobrecàrrega de mètodes et permet definir múltiples mètodes amb el mateix nom però diferents paràmetres.

public class Program
{
    public void Display(int number)
    {
        Console.WriteLine($"Number: {number}");
    }

    public void Display(string text)
    {
        Console.WriteLine($"Text: {text}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.Display(10); // Crida el primer mètode Display
        program.Display("Hello"); // Crida el segon mètode Display
    }
}

Paràmetres Opcional

C# et permet definir mètodes amb paràmetres opcionals, proporcionant valors per defecte si no es subministren arguments.

public class Program
{
    public void DisplayMessage(string message = "Default Message")
    {
        Console.WriteLine(message);
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayMessage(); // Sortida: Default Message
        program.DisplayMessage("Custom Message"); // Sortida: Custom Message
    }
}

Arguments Nominals

Els arguments nominals et permeten especificar el nom del paràmetre quan crides un mètode, millorant la llegibilitat.

public class Program
{
    public void DisplayInfo(string name, int age)
    {
        Console.WriteLine($"Name: {name}, Age: {age}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayInfo(name: "Alice", age: 30); // Arguments nominals
    }
}

Resum

  • Definicions i crides bàsiques de mètodes.

Entendre com definir i cridar mètodes és crucial per escriure codi net, eficient i fàcil de mantenir. Practica aquests conceptes per convertir-te en un expert en l'ús de mètodes en C#.
