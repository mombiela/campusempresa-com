Constructors i Destructors en C#
Herència ►

Introducció

En C#, els constructors i destructors són mètodes especials utilitzats per inicialitzar i netejar objectes, respectivament. Entendre com utilitzar-los de manera efectiva és crucial per gestionar recursos i assegurar l'estabilitat de les teves aplicacions.

Constructors

Què és un Constructor?

  • Un constructor és un mètode especial que es crida quan es crea una instància d'una classe.
  • S'utilitza per inicialitzar l'objecte.
  • Els constructors tenen el mateix nom que la classe i no tenen un tipus de retorn.

Tipus de Constructors

  1. Constructor per Defecte

    • Proporcionat automàticament per C# si no es defineixen constructors.
    • Inicialitza els camps als seus valors per defecte.
    public class Person
{
    public string Name;
    public int Age;

    // Constructor per defecte
    public Person()
    {
        Name = "Unknown";
        Age = 0;
    }
}

  2. Constructor Parametritzat

    • Permet passar paràmetres per inicialitzar camps amb valors específics.
    public class Person
{
    public string Name;
    public int Age;

    // Constructor parametritzat
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }
}

  3. Constructor de Còpia

    • Inicialitza un objecte utilitzant un altre objecte de la mateixa classe.
    public class Person
{
    public string Name;
    public int Age;

    // Constructor de còpia
    public Person(Person other)
    {
        Name = other.Name;
        Age = other.Age;
    }
}

Sobrecàrrega de Constructors

  • Es poden definir múltiples constructors en una classe, cadascun amb diferents paràmetres.
  • Això es coneix com a sobrecàrrega de constructors.
public class Person
{
    public string Name;
    public int Age;
    
    // Constructor per defecte
    public Person()
    {
        Name = "Unknown";
        Age = 0;
    }
    
    // Constructor parametritzat
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }
    
    // Constructor de còpia
    public Person(Person other)
    {
        Name = other.Name;
        Age = other.Age;
    }
}

Destructors

Què és un Destructor?

  • Un destructor és un mètode especial que es crida quan es destrueix una instància d'una classe.
  • S'utilitza per netejar recursos abans que l'objecte sigui eliminat de la memòria.
  • Els destructors tenen el mateix nom que la classe, precedit per una tilde (~), i no tenen paràmetres ni un tipus de retorn.

Utilitzant Destructors

  • Els destructors són menys utilitzats en C# degut a que el col·lector d'escombraries gestiona la major part de les tasques de gestió de memòria.
  • S'utilitzen típicament per alliberar recursos no gestionats com manetes de fitxers o connexions a bases de dades.
public class ResourceHolder
{
    // Destructor
    ~ResourceHolder()
    {
        // Codi de neteja aquí
        Console.WriteLine("Destructor cridat, netejant recursos.");
    }
}

Exemple de Destructor

public class FileManager
{
    private FileStream fileStream;

    public FileManager(string filePath)
    {
        fileStream = new FileStream(filePath, FileMode.OpenOrCreate);
    }

    // Destructor
    ~FileManager()
    {
        if (fileStream != null)
        {
            fileStream.Close();
            Console.WriteLine("Flux de fitxer tancat.");
        }
    }
}

Resum

  • Els constructors s'utilitzen per inicialitzar objectes, i els destructors s'utilitzen per netejar recursos.
  • C# proporciona constructors per defecte si no se'n defineixen, però també pots crear constructors parametritzats i de còpia.
  • Els destructors són menys utilitzats degut al col·lector d'escombraries però són essencials per gestionar recursos no gestionats.
  • L'ús adequat de constructors i destructors assegura una gestió eficient dels recursos i l'estabilitat de l'aplicació.

Entendre i utilitzar constructors i destructors de manera efectiva t'ajudarà a escriure codi C# més robust i mantenible.
