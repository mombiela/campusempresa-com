La serialització i deserialització són conceptes essencials en C# per convertir objectes en un format que es pugui emmagatzemar o transmetre fàcilment i després reconstruir-los de nou en objectes. Aquest tema cobrirà els aspectes bàsics fins als avançats de la serialització i deserialització en C#.

Què és la Serialització?

La serialització és el procés de convertir un objecte en un flux de bytes per emmagatzemar l'objecte o transmetre'l a la memòria, una base de dades o un fitxer. El seu principal propòsit és guardar l'estat d'un objecte perquè es pugui recrear quan sigui necessari.

Conceptes Clau

  • Estat de l'Objecte: Les dades contingudes dins d'un objecte.
  • Flux de Bytes: Una seqüència de bytes que representa l'estat de l'objecte.
  • Emmagatzematge/Transmissió: El mitjà on es guarda o transmeten les dades serialitzades.

Exemple

using System;
using System.IO;
using System.Runtime.Serialization.Formatters.Binary;

[Serializable]
public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void Main()
    {
        Person person = new Person { Name = "John", Age = 30 };

        // Serialitzar
        BinaryFormatter formatter = new BinaryFormatter();
        using (FileStream stream = new FileStream("person.dat", FileMode.Create))
        {
            formatter.Serialize(stream, person);
        }

        Console.WriteLine("Objecte Person serialitzat.");
    }
}

Què és la Deserialització?

La deserialització és el procés invers de la serialització. Implica convertir un flux de bytes de nou en un objecte. Això és útil per reconstruir l'objecte des del seu estat emmagatzemat o transmès.

Conceptes Clau

  • Reconstrucció: El procés de crear un objecte a partir d'un flux de bytes.
  • Flux de Bytes: Les dades serialitzades que representen l'estat de l'objecte.

Exemple

using System;
using System.IO;
using System.Runtime.Serialization.Formatters.Binary;

[Serializable]
public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void Main()
    {
        Person person;

        // Deserialitzar
        BinaryFormatter formatter = new BinaryFormatter();
        using (FileStream stream = new FileStream("person.dat", FileMode.Open))
        {
            person = (Person)formatter.Deserialize(stream);
        }

        Console.WriteLine($"Objecte Person deserialitzat. Nom: {person.Name}, Edat: {person.Age}");
    }
}

Tipus de Serialització

Serialització Binària

  • Descripció: Converteix un objecte en un format binari.
  • Cas d'Ús: Eficient per a l'emmagatzematge i la transmissió però no llegible per humans.
  • Exemple: Utilitzant BinaryFormatter.

Serialització XML

  • Descripció: Converteix un objecte en un format XML.
  • Cas d'Ús: Llegible per humans i útil per a serveis web.
  • Exemple: Utilitzant XmlSerializer.
using System;
using System.IO;
using System.Xml.Serialization;

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void Main()
    {
        Person person = new Person { Name = "John", Age = 30 };

        // Serialitzar
        XmlSerializer serializer = new XmlSerializer(typeof(Person));
        using (FileStream stream = new FileStream("person.xml", FileMode.Create))
        {
            serializer.Serialize(stream, person);
        }

        Console.WriteLine("Objecte Person serialitzat a XML.");
    }
}

Serialització JSON

  • Descripció: Converteix un objecte en un format JSON.
  • Cas d'Ús: Llegible per humans, àmpliament utilitzat en aplicacions web.
  • Exemple: Utilitzant JsonSerializer.
using System;
using System.Text.Json;

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void Main()
    {
        Person person = new Person { Name = "John", Age = 30 };

        // Serialitzar
        string jsonString = JsonSerializer.Serialize(person);
        Console.WriteLine("Objecte Person serialitzat a JSON:");
        Console.WriteLine(jsonString);
    }
}

Temes Avançats

Serialització Personalitzada

  • Descripció: Implementar lògica personalitzada per a la serialització i deserialització.
  • Cas d'Ús: Quan la serialització per defecte no compleix els requisits.
  • Exemple: Implementant la interfície ISerializable.
using System;
using System.IO;
using System.Runtime.Serialization;
using System.Runtime.Serialization.Formatters.Binary;

[Serializable]
public class Person : ISerializable
{
    public string Name { get; set; }
    public int Age { get; set; }

    public Person() { }

    protected Person(SerializationInfo info, StreamingContext context)
    {
        Name = info.GetString("Name");
        Age = info.GetInt32("Age");
    }

    public void GetObjectData(SerializationInfo info, StreamingContext context)
    {
        info.AddValue("Name", Name);
        info.AddValue("Age", Age);
    }
}

class Program
{
    static void Main()
    {
        Person person = new Person { Name = "John", Age = 30 };

        // Serialitzar
        BinaryFormatter formatter = new BinaryFormatter();
        using (FileStream stream = new FileStream("person_custom.dat", FileMode.Create))
        {
            formatter.Serialize(stream, person);
        }

        // Deserialitzar
        using (FileStream stream = new FileStream("person_custom.dat", FileMode.Open))
        {
            person = (Person)formatter.Deserialize(stream);
        }

        Console.WriteLine($"Objecte Person deserialitzat. Nom: {person.Name}, Edat: {person.Age}");
    }
}

Resum

La serialització i deserialització són tècniques poderoses en C# per convertir objectes a i des d'un format emmagatzemable o transmissible. Entendre els diferents tipus de serialització (binària, XML, JSON) i com implementar la serialització personalitzada pot millorar molt la teva capacitat de gestionar dades en les teves aplicacions. En dominar aquests conceptes, pots assegurar-te que les teves aplicacions poden emmagatzemar i transmetre dades de manera eficient mentre es manté la integritat i l'estructura dels teus objectes.
