Introducció

En C#, els mètodes són blocs de codi que realitzen una tasca específica. Entendre com definir i utilitzar paràmetres de mètodes i tipus de retorn és crucial per escriure codi reutilitzable i mantenible. Aquesta secció cobrirà els conceptes bàsics dels paràmetres de mètodes i tipus de retorn, progressant cap a conceptes més avançats.

Paràmetres de Mètodes

Què són els Paràmetres de Mètodes?

Els paràmetres de mètodes són variables que es passen als mètodes per proporcionar dades d'entrada. Permeten que els mètodes operin sobre dades diferents sense canviar el codi del mètode.

Tipus de Paràmetres

  • Paràmetres de Valor: Passa una còpia de les dades al mètode.
  • Paràmetres de Referència: Passa una referència a les dades reals, permetent que el mètode modifiqui les dades originals.
  • Paràmetres de Sortida: S'utilitzen per retornar múltiples valors d'un mètode.

Sintaxi i Exemples

Paràmetres de Valor

public void PrintMessage(string message)
{
    Console.WriteLine(message);
}

// Ús
PrintMessage("Hello, World!");

Explicació: El mètode PrintMessage pren un únic paràmetre de valor message de tipus string.

Paràmetres de Referència

public void Increment(ref int number)
{
    number++;
}

// Ús
int myNumber = 5;
Increment(ref myNumber);
Console.WriteLine(myNumber); // Sortida: 6

Explicació: El mètode Increment pren un paràmetre de referència number utilitzant la paraula clau ref, permetent que modifiqui la variable original.

Paràmetres de Sortida

public void GetValues(out int x, out int y)
{
    x = 10;
    y = 20;
}

// Ús
int a, b;
GetValues(out a, out b);
Console.WriteLine($"a: {a}, b: {b}"); // Sortida: a: 10, b: 20

Explicació: El mètode GetValues utilitza la paraula clau out per retornar múltiples valors.

Tipus de Retorn

Què són els Tipus de Retorn?

Els tipus de retorn especifiquen el tipus de valor que un mètode retorna després de l'execució. Si un mètode no retorna cap valor, utilitza la paraula clau void.

Tipus de Retorn Comuns

  • Tipus Primitus: int, float, double, char, etc.
  • Tipus Complexos: string, arrays, objectes, etc.
  • Void: Indica que no hi ha cap valor de retorn.

Sintaxi i Exemples

Retornant un Valor

public int Add(int a, int b)
{
    return a + b;
}

// Ús
int result = Add(5, 3);
Console.WriteLine(result); // Sortida: 8

Explicació: El mètode Add retorna un valor int, que és la suma dels dos paràmetres.

Retornant un Objecte

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

public Person CreatePerson(string name, int age)
{
    return new Person { Name = name, Age = age };
}

// Ús
Person person = CreatePerson("Alice", 30);
Console.WriteLine($"Name: {person.Name}, Age: {person.Age}"); // Sortida: Name: Alice, Age: 30

Explicació: El mètode CreatePerson retorna un objecte Person inicialitzat amb els paràmetres proporcionats.

Tipus de Retorn Void

public void DisplayMessage(string message)
{
    Console.WriteLine(message);
}

// Ús
DisplayMessage("Hello, World!");

Explicació: El mètode DisplayMessage no retorna cap valor, indicat per la paraula clau void.

Conceptes Avançats

Sobrecàrrega de Mètodes

La sobrecàrrega de mètodes permet que múltiples mètodes amb el mateix nom però diferents paràmetres coexisteixin.

public void Display(int number)
{
    Console.WriteLine(number);
}

public void Display(string message)
{
    Console.WriteLine(message);
}

// Ús
Display(10); // Sortida: 10
Display("Hello"); // Sortida: Hello

Explicació: El mètode Display està sobrecarregat per manejar tant paràmetres int com string.

Paràmetres Opcional

Els paràmetres opcionals permeten ometre arguments quan es crida un mètode.

public void Greet(string name, string greeting = "Hello")
{
    Console.WriteLine($"{greeting}, {name}!");
}

// Ús
Greet("Alice"); // Sortida: Hello, Alice!
Greet("Bob", "Hi"); // Sortida: Hi, Bob!

Explicació: El mètode Greet té un paràmetre opcional greeting amb un valor per defecte de "Hello".

Arguments Nominals

Els arguments nominals milloren la llegibilitat del codi especificant explícitament els noms dels paràmetres.

public void DisplayInfo(string name, int age)
{
    Console.WriteLine($"Name: {name}, Age: {age}");
}

// Ús
DisplayInfo(age: 25, name: "Alice"); // Sortida: Name: Alice, Age: 25

Explicació: El mètode DisplayInfo es crida utilitzant arguments nominals per a més claredat.

Conclusió

Entendre els paràmetres de mètodes i els tipus de retorn és fonamental per escriure codi efectiu en C#. En dominar els paràmetres de valor, referència i sortida, així com diversos tipus de retorn, pots crear mètodes versàtils i reutilitzables. Els conceptes avançats com la sobrecàrrega de mètodes, els paràmetres opcionals i els arguments nominals milloren encara més la teva capacitat per escriure codi net i mantenible.
