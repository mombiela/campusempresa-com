Introducció

Els operadors són símbols especials en C# que realitzen operacions sobre variables i valors. Són fonamentals per a la programació ja que permeten manipular dades i variables de diverses maneres. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en C#, des de l'aritmètica bàsica fins a operadors més avançats.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul (resta d'una divisió)

Exemple

int a = 10;
int b = 3;

int sum = a + b;        // 13
int difference = a - b; // 7
int product = a * b;    // 30
int quotient = a / b;   // 3
int remainder = a % b;  // 1

Console.WriteLine($"Suma: {sum}, Diferència: {difference}, Producte: {product}, Quocient: {quotient}, Resta: {remainder}");

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors.

  • == : Igual a
  • != : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual que
  • <= : Menor o igual que

Exemple

int a = 10;
int b = 3;

bool isEqual = (a == b);       // false
bool isNotEqual = (a != b);    // true
bool isGreater = (a > b);      // true
bool isLess = (a < b);         // false
bool isGreaterOrEqual = (a >= b); // true
bool isLessOrEqual = (a <= b);    // false

Console.WriteLine($"Igual: {isEqual}, Diferent: {isNotEqual}, Major: {isGreater}, Menor: {isLess}, Major o Igual: {isGreaterOrEqual}, Menor o Igual: {isLessOrEqual}");

Operadors Lògics

Els operadors lògics s'utilitzen per combinar declaracions condicionals.

  • && : Lògic AND
  • || : Lògic OR
  • ! : Lògic NOT

Exemple

bool a = true;
bool b = false;

bool andResult = a && b; // false
bool orResult = a || b;  // true
bool notResult = !a;     // false

Console.WriteLine($"AND: {andResult}, OR: {orResult}, NOT: {notResult}");

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • = : Assignació simple
  • += : Sumar i assignar
  • -= : Restar i assignar
  • *= : Multiplicar i assignar
  • /= : Dividir i assignar
  • %= : Mòdul i assignar

Exemple

int a = 10;

a += 5; // a = a + 5;  => 15
a -= 3; // a = a - 3;  => 12
a *= 2; // a = a * 2;  => 24
a /= 4; // a = a / 4;  => 6
a %= 4; // a = a % 4;  => 2

Console.WriteLine($"Valor final de a: {a}");

Operadors a Nivell de Bit

Els operadors a nivell de bit s'utilitzen per realitzar operacions sobre representacions binàries de nombres.

  • & : AND a nivell de bit
  • | : OR a nivell de bit
  • ^ : XOR a nivell de bit
  • ~ : NOT a nivell de bit
  • << : Desplaçament a l'esquerra
  • >> : Desplaçament a la dreta

Exemple

int a = 5;  // 0101 en binari
int b = 3;  // 0011 en binari

int andResult = a & b; // 0001 => 1
int orResult = a | b;  // 0111 => 7
int xorResult = a ^ b; // 0110 => 6
int notResult = ~a;    // 1010 => -6 (en forma de complement a 2)
int leftShift = a << 1; // 1010 => 10
int rightShift = a >> 1; // 0010 => 2

Console.WriteLine($"AND: {andResult}, OR: {orResult}, XOR: {xorResult}, NOT: {notResult}, Desplaçament a l'Esquerra: {leftShift}, Desplaçament a la Dreta: {rightShift}");

Operador Condicional (Ternari)

L'operador condicional és una forma abreujada d'una declaració if-else.

  • ?: : Condicional

Exemple

int a = 10;
int b = 20;

string result = (a > b) ? "a és més gran" : "b és més gran";

Console.WriteLine(result); // Sortida: b és més gran

Operadors de Coalescència Nul·la

Els operadors de coalescència nul·la s'utilitzen per proporcionar un valor per defecte per a tipus nul·lables.

  • ?? : Coalescència nul·la
  • ??= : Assignació de coalescència nul·la

Exemple

int? a = null;
int b = a ?? 5; // Si a és null, b serà 5

int? c = null;
c ??= 10; // Si c és null, assignar 10 a c

Console.WriteLine($"b: {b}, c: {c}"); // Sortida: b: 5, c: 10

Resum

Els operadors en C# són essencials per realitzar diverses operacions sobre dades. Des de l'aritmètica bàsica fins a operacions més complexes a nivell de bit, entendre com utilitzar aquests operadors de manera efectiva és crucial per a qualsevol programador de C#. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en la manipulació de dades i l'escriptura de codi eficient.
