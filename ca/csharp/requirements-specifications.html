Introducció

Entendre els requisits i les especificacions és crucial per a qualsevol projecte de desenvolupament de programari. En aquesta secció, cobrirem els conceptes bàsics de la recollida de requisits, l'escriptura d'especificacions i com aquests conceptes s'apliquen a la programació en C#.

Conceptes Clau

  • Requisits: Aquestes són les necessitats o condicions que un programari ha de satisfer o posseir. Els requisits es recullen típicament dels interessats.
  • Especificacions: Aquestes són descripcions detallades de la funcionalitat, el comportament i les restriccions del programari. Les especificacions es deriven dels requisits i serveixen com a plànol per al desenvolupament.

Recollida de Requisits

Tipus de Requisits

  • Requisits Funcionals: Defineixen què ha de fer el sistema.
    • Exemple: El sistema ha de permetre als usuaris iniciar sessió.
  • Requisits No Funcionals: Defineixen com el sistema realitza una funció.
    • Exemple: El sistema ha de respondre a les entrades dels usuaris en menys de 2 segons.

Tècniques per Recollir Requisits

  • Entrevistes: Parlar amb els interessats per entendre les seves necessitats.
  • Enquestes/Questionaris: Recollir informació d'un grup més gran.
  • Tallers: Sessions col·laboratives amb els interessats.
  • Observació: Veure com els usuaris interactuen amb el sistema actual.

Escriptura d'Especificacions

Components d'una Bona Especificació

  • Claredat: Les especificacions han de ser clares i no ambigües.
  • Completesa: Tots els detalls necessaris han d'estar inclosos.
  • Consistència: No hi ha d'haver informació conflictiva.
  • Verificabilitat: Cada requisit ha de ser verificable mitjançant proves.

Exemple d'un Document d'Especificació

# Especificació de la Funcionalitat d'Inici de Sessió d'Usuari

## Requisits Funcionals
1. El sistema ha de permetre als usuaris iniciar sessió utilitzant el seu correu electrònic i contrasenya.
2. El sistema ha de validar les credencials de l'usuari contra la base de dades.

## Requisits No Funcionals
1. El procés d'inici de sessió ha de completar-se en menys de 2 segons.
2. El sistema ha de bloquejar el compte de l'usuari després de 5 intents fallits d'inici de sessió.

## Cas d'Ús
### Cas d'Ús: Inici de Sessió d'Usuari
- **Actor**: Usuari
- **Precondició**: L'usuari ha de tenir un compte registrat.
- **Flux Principal**:
  1. L'usuari navega a la pàgina d'inici de sessió.
  2. L'usuari introdueix el correu electrònic i la contrasenya.
  3. El sistema valida les credencials.
  4. El sistema redirigeix l'usuari al tauler de control.
- **Postcondició**: L'usuari ha iniciat sessió i és redirigit al tauler de control.
- **Flux Alternatiu**:
  - Si les credencials són invàlides, el sistema mostra un missatge d'error.
  - Si el compte està bloquejat, el sistema mostra un missatge de bloqueig.

Aplicació de Requisits i Especificacions en C#

Exemple: Implementació de l'Inici de Sessió d'Usuari en C#

Implementem una funcionalitat simple d'inici de sessió d'usuari basada en l'especificació anterior.

Pas 1: Definir la Classe Usuari

public class User
{
    public string Email { get; set; }
    public string Password { get; set; }
}

Pas 2: Crear una Base de Dades Simulada

public static class MockDatabase
{
    private static List<User> users = new List<User>
    {
        new User { Email = "user@example.com", Password = "password123" }
    };

    public static User GetUserByEmail(string email)
    {
        return users.FirstOrDefault(u => u.Email == email);
    }
}

Pas 3: Implementar la Funció d'Inici de Sessió

public class AuthService
{
    public string Login(string email, string password)
    {
        User user = MockDatabase.GetUserByEmail(email);
        if (user == null || user.Password != password)
        {
            return "Credencials invàlides";
        }
        return "Inici de sessió reeixit";
    }
}

Pas 4: Provar la Funció d'Inici de Sessió

public class Program
{
    public static void Main(string[] args)
    {
        AuthService authService = new AuthService();
        string result = authService.Login("user@example.com", "password123");
        Console.WriteLine(result); // Sortida: Inici de sessió reeixit

        result = authService.Login("user@example.com", "wrongpassword");
        Console.WriteLine(result); // Sortida: Credencials invàlides
    }
}
