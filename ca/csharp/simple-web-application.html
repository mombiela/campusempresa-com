Introducció

En aquest tema, et guiarem a través de la construcció d'una aplicació web simple utilitzant C#. Això cobrirà els conceptes bàsics per a principiants i gradualment es mourà cap a conceptes més avançats. Al final d'aquest tema, hauríeu de tenir una comprensió sòlida de com crear, gestionar i desplegar una aplicació web utilitzant C#.

Configurant el Teu Entorn

Abans de començar a codificar, configurem les eines i l'entorn necessaris.

  • Instal·lar Visual Studio: Descarrega i instal·la Visual Studio, que és un entorn de desenvolupament integrat (IDE) per a C#.
  • Instal·lar .NET SDK: Assegura't de tenir instal·lat el .NET SDK. Això és necessari per construir i executar aplicacions .NET.

Creant un Nou Projecte

  1. Obre Visual Studio.
  2. Selecciona "Crear un nou projecte".
  3. Tria "Aplicació Web ASP.NET Core" i fes clic a "Següent".
  4. Configura el nom i la ubicació del teu projecte, després fes clic a "Crear".
  5. Selecciona "Aplicació Web (Model-View-Controller)" i fes clic a "Crear".

Entenent l'Estructura del Projecte

Un cop creat el projecte, veuràs una varietat d'arxius i carpetes. Aquests són els components clau:

  • Controllers: Gestionen les sol·licituds HTTP entrants i envien respostes al client.
  • Views: Plantilles HTML que mostren dades a l'usuari.
  • Models: Representen les dades i la lògica de negoci.

Construint l'Aplicació

Creant un Model

Primer, creem un model simple. Els models representen les dades en la teva aplicació.

public class Product
{
    public int Id { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
}

Creant un Controller

A continuació, crea un controlador per gestionar les sol·licituds HTTP.

using Microsoft.AspNetCore.Mvc;
using System.Collections.Generic;

public class ProductController : Controller
{
    public IActionResult Index()
    {
        var products = new List<Product>
        {
            new Product { Id = 1, Name = "Laptop", Price = 999.99M },
            new Product { Id = 2, Name = "Smartphone", Price = 499.99M }
        };
        return View(products);
    }
}

Creant una Vista

Ara, crea una vista per mostrar els productes.

  1. Fes clic dret a la carpeta "Views", selecciona "Afegir" > "Nova Carpeta", i anomena-la "Product".
  2. Fes clic dret a la carpeta "Product", selecciona "Afegir" > "Nou Element", i tria "Vista Razor". Anomena-la "Index.cshtml".
@model IEnumerable<Product>

<!DOCTYPE html>
<html>
<head>
    <title>Products</title>
</head>
<body>
    <h1>Product List</h1>
    <table>
        <thead>
            <tr>
                <th>Id</th>
                <th>Name</th>
                <th>Price</th>
            </tr>
        </thead>
        <tbody>
            @foreach (var product in Model)
            {
                <tr>
                    <td>@product.Id</td>
                    <td>@product.Name</td>
                    <td>@product.Price</td>
                </tr>
            }
        </tbody>
    </table>
</body>
</html>

Executant l'Aplicació

Per executar l'aplicació, prem F5 o fes clic al botó "Iniciar Depuració" a Visual Studio. Això construirà i llançarà la teva aplicació web en un navegador.

Temes Avançats

Afegint una Base de Dades

Per a usuaris més avançats, integrar una base de dades és essencial. Aquí tens una breu visió general:

  • Entity Framework Core: Utilitza EF Core per interactuar amb la base de dades.
  • Migracions: Gestiona els canvis d'esquema de la base de dades.

Autenticació i Autorització

Implementar la seguretat és crucial per a qualsevol aplicació web.

  • Identity Framework: Utilitza ASP.NET Core Identity per a l'autenticació i autorització d'usuaris.
  • Control d'Accés Basat en Rols: Defineix rols i restringeix l'accés a certes parts de l'aplicació.

Desplegament

Desplegar la teva aplicació a un servidor o servei al núvol és el pas final.

  • Azure: Desplega la teva aplicació a Microsoft Azure.
  • Docker: Contenitza la teva aplicació utilitzant Docker.

Conclusió

Construir una aplicació web simple en C# implica entendre el patró MVC, crear models, controladors i vistes, i executar l'aplicació. A mesura que avances, pots integrar bases de dades, implementar seguretat i desplegar la teva aplicació. Aquest tema proporciona una base, i s'encoratja a explorar més i construir aplicacions més complexes.
