Introducció als Bucles

Els bucles són construccions fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En C#, hi ha diversos tipus de bucles, cadascun adequat per a diferents escenaris. Entendre com utilitzar els bucles de manera efectiva és crucial per escriure codi eficient i concís.

Tipus de Bucles

Bucle `for`

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi

for (inicialització; condició; increment)
{
    // Codi a executar
}

Exemple

for (int i = 0; i < 5; i++)
{
    Console.WriteLine("Iteració: " + i);
}
  • Inicialització: Estableix el punt de partida del bucle.
  • Condició: El bucle s'executa mentre aquesta condició sigui certa.
  • Increment: Actualitza la variable de control del bucle.

Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix per endavant. Continua executant-se mentre la condició especificada sigui certa.

Sintaxi

while (condició)
{
    // Codi a executar
}

Exemple

int i = 0;
while (i < 5)
{
    Console.WriteLine("Iteració: " + i);
    i++;
}
  • Condició: El bucle s'executa mentre aquesta condició sigui certa.

Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el cos del bucle s'executarà almenys una vegada.

Sintaxi

do
{
    // Codi a executar
} while (condició);

Exemple

int i = 0;
do
{
    Console.WriteLine("Iteració: " + i);
    i++;
} while (i < 5);
  • Condició: El bucle s'executa mentre aquesta condició sigui certa, però el bloc de codi s'executa almenys una vegada abans que es provi la condició.

Bucle `foreach`

El bucle foreach s'utilitza per iterar sobre una col·lecció o una matriu. És particularment útil quan necessites accedir a cada element d'una col·lecció sense modificar-lo.

Sintaxi

foreach (tipus variable en col·lecció)
{
    // Codi a executar
}

Exemple

string[] fruits = { "Poma", "Plàtan", "Cirera" };
foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}
  • Col·lecció: La matriu o col·lecció a iterar.
  • Variable: Representa l'element actual de la col·lecció.

Bucles Imbricats

Els bucles es poden imbricar dins d'altres bucles. Això és útil per treballar amb matrius multidimensionals o per realitzar iteracions complexes.

Exemple

for (int i = 0; i < 3; i++)
{
    for (int j = 0; j < 3; j++)
    {
        Console.WriteLine("i: " + i + ", j: " + j);
    }
}
  • Bucle Extern: S'executa tres vegades.
  • Bucle Intern: S'executa tres vegades per a cada iteració del bucle extern.

Instruccions de Control de Bucles

`break`

La instrucció break finalitza el bucle immediatament.

Exemple

for (int i = 0; i < 10; i++)
{
    if (i == 5)
    {
        break;
    }
    Console.WriteLine("Iteració: " + i);
}

`continue`

La instrucció continue salta l'iteració actual i procedeix a la següent iteració del bucle.

Exemple

for (int i = 0; i < 10; i++)
{
    if (i % 2 == 0)
    {
        continue;
    }
    Console.WriteLine("Nombre senar: " + i);
}

Resum

Els bucles són construccions essencials en C# que et permeten executar codi repetidament basant-se en una condició. Els bucles for, while, do-while i foreach serveixen per a diferents propòsits i són útils en diversos escenaris. Entendre com utilitzar aquests bucles i controlar el seu flux amb instruccions com break i continue et permetrà escriure codi més eficient i efectiu. Practica utilitzant aquests bucles en diferents contextos per esdevenir proficient en la seva aplicació.
