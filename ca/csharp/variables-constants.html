Variables i Constants en C#
Tipus de Dades ►

Introducció

En C#, les variables i constants són conceptes fonamentals que et permeten emmagatzemar i manipular dades. Entendre com declarar, inicialitzar i utilitzar variables i constants és essencial per a qualsevol programador de C#. Aquesta secció et guiarà a través d'aquests conceptes des d'un nivell principiant fins a un nivell avançat.

Variables

Què és una Variable?

  • Una variable és una ubicació d'emmagatzematge en memòria amb un tipus específic que determina quin tipus de dades pot contenir.
  • Les variables tenen noms (identificadors) que s'utilitzen per referenciar les dades emmagatzemades.

Declarar i Inicialitzar Variables

  • Declaració: Especificar el tipus i el nom de la variable.
  • Inicialització: Assignar un valor inicial a la variable.

Sintaxi

type variableName = initialValue;

Exemple

int age = 25; // Declara una variable enter anomenada 'age' i la inicialitza amb el valor 25.
string name = "John"; // Declara una variable cadena anomenada 'name' i la inicialitza amb el valor "John".

Tipus de Variables

  • Tipus Primitius: int, float, double, char, bool, etc.
  • Tipus de Referència: string, arrays, classes, etc.

Exemple

int number = 10; // Tipus enter
float pi = 3.14f; // Tipus de punt flotant
char letter = 'A'; // Tipus caràcter
bool isTrue = true; // Tipus booleà
string greeting = "Hello, World!"; // Tipus cadena

Àmbit de les Variables

  • Variables Locals: Declarades dins d'un mètode i accessibles només dins d'aquest mètode.
  • Variables Globals: Declarades fora dels mètodes i accessibles a tota la classe.

Exemple

class Program
{
    int globalVar = 10; // Variable global

    void Method()
    {
        int localVar = 5; // Variable local
        Console.WriteLine(localVar);
    }
}

Constants

Què és una Constant?

  • Una constant és una variable el valor de la qual no pot ser canviat un cop assignat.
  • Les constants es declaren utilitzant la paraula clau const.

Declarar Constants

  • Sintaxi: const type constantName = value;

Exemple

const double PI = 3.14159; // Declara una constant doble anomenada 'PI' amb el valor 3.14159.
const int DAYS_IN_WEEK = 7; // Declara una constant enter anomenada 'DAYS_IN_WEEK' amb el valor 7.

Per què Utilitzar Constants?

  • Les constants proporcionen una manera d'utilitzar noms significatius per a valors fixos, fent el codi més llegible i mantenible.
  • Eviten la modificació accidental de valors que haurien de romandre constants al llarg del programa.

Comparació: Variables vs. Constants

| Característica | Variables | Constants | |----------------|------------------------------------|------------------------------------| | Mutabilitat | Es poden canviar després de la inicialització | No es poden canviar després de la inicialització | | Declaració | type variableName = value; | const type constantName = value; | | Ús | Utilitzades per a valors que poden canviar | Utilitzades per a valors fixos | | Exemple | int age = 25; | const int DAYS_IN_WEEK = 7; |

Temes Avançats

Camps Readonly

  • Similar a les constants però poden assignar-se un valor tant en el moment de la declaració com en el constructor.
  • Declarats utilitzant la paraula clau readonly.

Exemple

class Circle
{
    public readonly double Radius;

    public Circle(double radius)
    {
        Radius = radius; // Pot assignar-se en el constructor
    }
}

Variables i Constants Estàtiques

  • Variables Estàtiques: Compartides entre totes les instàncies d'una classe.
  • Constants Estàtiques: Constants que es comparteixen entre totes les instàncies d'una classe.

Exemple

class MathConstants
{
    public const double PI = 3.14159; // Constant estàtica
    public static int counter = 0; // Variable estàtica
}

Conclusió

Entendre les variables i constants és crucial per a una programació efectiva en C#. Les variables et permeten emmagatzemar i manipular dades, mentre que les constants proporcionen una manera d'utilitzar valors fixos que milloren la llegibilitat i mantenibilitat del codi. En dominar aquests conceptes, podràs escriure programes en C# més eficients i fiables.
