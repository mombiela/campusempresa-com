Introducció

Benvingut al teu viatge en la programació en C#! En aquesta secció, et guiarem a través de l'escriptura del teu primer programa en C#. Al final d'aquesta lliçó, entendràs l'estructura bàsica d'un programa en C# i com executar-lo.

Configurant el teu entorn

Abans de començar a codificar, necessites configurar el teu entorn de desenvolupament. Segueix aquests passos:

  1. Instal·la Visual Studio: Descarrega i instal·la Visual Studio Community Edition des del lloc web oficial.
  2. Crea un nou projecte:
    • Obre Visual Studio.
    • Selecciona "Crear un nou projecte".
    • Tria "Aplicació de consola (.NET Core)".
    • Fes clic a "Següent", dona un nom al teu projecte i fes clic a "Crear".

Entenent l'estructura bàsica

Un programa en C# consta de diversos components clau. Aquí tens un exemple senzill d'un programa en C#:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hola, món!");
        }
    }
}

Components clau explicats

  • using System;: Aquesta línia importa l'espai de noms System, que conté classes fonamentals per a operacions bàsiques.
  • namespace HelloWorld: Els espais de noms s'utilitzen per organitzar el codi i prevenir conflictes de noms.
  • class Program: Una classe és un pla per crear objectes. La classe Program conté el codi que s'executarà.
  • static void Main(string[] args): El mètode Main és el punt d'entrada d'un programa en C#. És on el programa comença l'execució.
  • Console.WriteLine("Hola, món!");: Aquesta línia imprimeix "Hola, món!" a la consola.

Escriure i executar el teu primer programa

Anem a escriure i executar el teu primer programa en C# pas a pas:

  1. Obre Visual Studio i crea un nou projecte d'aplicació de consola com s'ha descrit anteriorment.
  2. Reemplaça el codi per defecte en Program.cs amb el següent:
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hola, món!");
        }
    }
}
  1. Executa el programa:
    • Fes clic al botó "Iniciar" (o prem F5).
    • Hauries de veure una finestra de consola mostrant "Hola, món!".

Exercicis

Per reforçar la teva comprensió, prova els següents exercicis:

  1. Modifica el missatge: Canvia el missatge en Console.WriteLine per imprimir el teu nom en lloc de "Hola, món!".
  2. Afegeix una altra línia: Afegeix una altra instrucció Console.WriteLine per imprimir un segon missatge.
  3. Experimenta amb comentaris: Afegeix comentaris al teu codi per descriure què fa cada part. Utilitza // per a comentaris d'una sola línia i /* */ per a comentaris de múltiples línies.

Exemple:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            // Això imprimeix un missatge de benvinguda a la consola
            Console.WriteLine("Hola, [El teu nom]!");
            
            // Això imprimeix un altre missatge
            Console.WriteLine("Benvingut a la programació en C#!");
        }
    }
}

Conclusió

Felicitats! Has escrit i executat el teu primer programa en C#. Ara entens l'estructura bàsica d'una aplicació en C# i com executar-la. Aquest coneixement fonamental serà crucial mentre ens endinsem en temes més avançats. Continua experimentant amb el codi i prova els exercicis per consolidar la teva comprensió. Feliç codificació!
