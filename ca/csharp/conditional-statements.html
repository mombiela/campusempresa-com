Les declaracions condicionals són fonamentals en la programació, ja que et permeten prendre decisions basades en certes condicions. En C#, les declaracions condicionals inclouen if, else if, else, switch i operadors ternaris. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús més avançat d'aquestes declaracions.

Declaracions If

La declaració if és la forma més bàsica de declaració condicional. Avalua una expressió booleana i executa un bloc de codi si l'expressió és certa.

Sintaxi

if (condition)
{
    // Codi a executar si la condició és certa
}

Exemple

int number = 10;
if (number > 5)
{
    Console.WriteLine("El número és més gran que 5.");
}

Explicació

  • condition: Una expressió booleana que avalua a true o false.
  • Si number > 5 és cert, es mostrarà el missatge "El número és més gran que 5.".

Declaracions If-Else

La declaració if-else proporciona un bloc de codi alternatiu a executar si la condició és falsa.

Sintaxi

if (condition)
{
    // Codi a executar si la condició és certa
}
else
{
    // Codi a executar si la condició és falsa
}

Exemple

int number = 3;
if (number > 5)
{
    Console.WriteLine("El número és més gran que 5.");
}
else
{
    Console.WriteLine("El número no és més gran que 5.");
}

Explicació

  • Si number > 5 és fals, es mostrarà el missatge "El número no és més gran que 5.".

Declaracions Else If

La declaració else if et permet comprovar múltiples condicions.

Sintaxi

if (condition1)
{
    // Codi a executar si la condició1 és certa
}
else if (condition2)
{
    // Codi a executar si la condició2 és certa
}
else
{
    // Codi a executar si cap de les condicions anteriors és certa
}

Exemple

int number = 7;
if (number > 10)
{
    Console.WriteLine("El número és més gran que 10.");
}
else if (number > 5)
{
    Console.WriteLine("El número és més gran que 5 però menor o igual a 10.");
}
else
{
    Console.WriteLine("El número és 5 o menys.");
}

Explicació

  • Les condicions es comproven en ordre. El bloc de codi de la primera condició certa s'executa.
  • Si number > 10 és fals però number > 5 és cert, es mostrarà el missatge "El número és més gran que 5 però menor o igual a 10.".

Declaracions Switch

La declaració switch s'utilitza per seleccionar un dels molts blocs de codi a executar.

Sintaxi

switch (expression)
{
    case value1:
        // Codi a executar si expression == value1
        break;
    case value2:
        // Codi a executar si expression == value2
        break;
    // Més casos...
    default:
        // Codi a executar si cap dels casos coincideix
        break;
}

Exemple

int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Dilluns");
        break;
    case 2:
        Console.WriteLine("Dimarts");
        break;
    case 3:
        Console.WriteLine("Dimecres");
        break;
    default:
        Console.WriteLine("Un altre dia");
        break;
}

Explicació

  • La declaració switch avalua l'expression i executa el bloc de codi que coincideix amb el case.
  • Si day és 3, es mostrarà el missatge "Dimecres".

Operador Ternari

L'operador ternari és una forma abreujada de les declaracions if-else.

Sintaxi

condition ? expression1 : expression2;

Exemple

int number = 8;
string result = (number > 5) ? "Més gran que 5" : "5 o menys";
Console.WriteLine(result);

Explicació

  • Si number > 5 és cert, result serà "Més gran que 5".
  • En cas contrari, result serà "5 o menys".

Conclusió

Les declaracions condicionals són essencials per controlar el flux d'un programa basat en diferents condicions. Entendre i utilitzar if, else if, else, switch i l'operador ternari et permetrà escriure codi més dinàmic i sensible. Practica aquests conceptes amb diversos exemples per esdevenir proficient en la presa de decisions dins dels teus programes en C#.
