La reflexió és una característica poderosa en C# que et permet inspeccionar i interactuar amb les metadades dels tipus en temps d'execució. Això pot ser particularment útil per a tasques com la creació dinàmica de tipus, la invocació de mètodes i l'accés a atributs.

Conceptes Clau de la Reflexió

  • Metadades: Informació sobre els tipus definits en un programa, com ara classes, interfícies i mètodes.
  • Assembly: Una biblioteca de codi compilada utilitzada per aplicacions .NET. La reflexió es pot utilitzar per inspeccionar el contingut d'un assembly.
  • Tipus: Representa declaracions de tipus: tipus de classe, tipus d'interfície, tipus de matriu, tipus de valor, tipus d'enumeració, paràmetres de tipus i tipus de punter.
  • MemberInfo: Proporciona informació sobre els membres d'una classe, com ara mètodes, propietats, camps i esdeveniments.

Ús de la Reflexió

Carregar un Assembly

Per utilitzar la reflexió, sovint comences carregant un assembly. Això es pot fer utilitzant la classe Assembly.

using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        // Carregar l'assembly actual
        Assembly assembly = Assembly.GetExecutingAssembly();
        Console.WriteLine("Nom complet de l'assembly: " + assembly.FullName);
    }
}

Obtenir Tipus d'un Assembly

Un cop tens un assembly, pots obtenir els tipus definits en ell.

using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Assembly assembly = Assembly.GetExecutingAssembly();
        Type[] types = assembly.GetTypes();

        foreach (Type type in types)
        {
            Console.WriteLine("Tipus: " + type.Name);
        }
    }
}

Inspeccionar Membres del Tipus

Pots inspeccionar els membres d'un tipus, com ara els seus mètodes, propietats i camps.

using System;
using System.Reflection;

class SampleClass
{
    public int SampleProperty { get; set; }
    public void SampleMethod() { }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);

        // Obtenir propietats
        PropertyInfo[] properties = type.GetProperties();
        foreach (PropertyInfo property in properties)
        {
            Console.WriteLine("Propietat: " + property.Name);
        }

        // Obtenir mètodes
        MethodInfo[] methods = type.GetMethods();
        foreach (MethodInfo method in methods)
        {
            Console.WriteLine("Mètode: " + method.Name);
        }
    }
}

Invocar Mètodes Dinàmicament

La reflexió et permet invocar mètodes en temps d'execució sense conèixer els seus noms en temps de compilació.

using System;
using System.Reflection;

class SampleClass
{
    public void SampleMethod()
    {
        Console.WriteLine("SampleMethod invocat!");
    }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);
        MethodInfo method = type.GetMethod("SampleMethod");

        if (method != null)
        {
            object instance = Activator.CreateInstance(type);
            method.Invoke(instance, null);
        }
    }
}

Treballar amb Atributs

La reflexió es pot utilitzar per llegir atributs personalitzats aplicats a tipus i membres.

using System;
using System.Reflection;

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class SampleAttribute : Attribute
{
    public string Description { get; set; }
}

[Sample(Description = "Aquesta és una classe de mostra.")]
class SampleClass
{
    [Sample(Description = "Aquest és un mètode de mostra.")]
    public void SampleMethod() { }
}

class Program
{
    static void Main()
    {
        Type type = typeof(SampleClass);
        object[] classAttributes = type.GetCustomAttributes(typeof(SampleAttribute), false);

        foreach (SampleAttribute attr in classAttributes)
        {
            Console.WriteLine("Descripció de l'atribut de la classe: " + attr.Description);
        }

        MethodInfo method = type.GetMethod("SampleMethod");
        object[] methodAttributes = method.GetCustomAttributes(typeof(SampleAttribute), false);

        foreach (SampleAttribute attr in methodAttributes)
        {
            Console.WriteLine("Descripció de l'atribut del mètode: " + attr.Description);
        }
    }
}

Conclusió

La reflexió en C# és una eina versàtil que et permet inspeccionar i manipular les metadades dels tipus en temps d'execució. Es pot utilitzar per a una varietat de tasques, incloent la creació dinàmica de tipus, la invocació de mètodes i la inspecció d'atributs. Tot i que és poderosa, la reflexió s'ha d'utilitzar amb prudència a causa del seu potencial impacte en el rendiment i la seguretat. Entendre els conceptes bàsics de la reflexió pot millorar enormement la teva capacitat per escriure aplicacions C# flexibles i dinàmiques.
