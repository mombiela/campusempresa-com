Què és LINQ?

Language Integrated Query (LINQ) és una característica poderosa en C# que permet als desenvolupadors escriure consultes directament dins del llenguatge C#. LINQ proporciona una manera consistent de consultar diverses fonts de dades, com ara col·leccions, bases de dades, XML i més.

  • Sintaxi Consistent: LINQ utilitza una sintaxi uniforme per consultar diferents fonts de dades.
  • Seguretat de Tipus: Les consultes es comproven en temps de compilació, reduint errors en temps d'execució.
  • Suport IntelliSense: Visual Studio proporciona suport IntelliSense per a consultes LINQ, facilitant l'escriptura i depuració.

Sintaxi Bàsica de LINQ

Les consultes LINQ s'escriuen utilitzant sintaxi de consulta o sintaxi de mètode. Ambdues aconsegueixen el mateix resultat però tenen enfocaments sintàctics diferents.

Sintaxi de Consulta

La sintaxi de consulta s'assembla a SQL i és més fàcil d'entendre per a aquells familiaritzats amb SQL.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = from num in numbers
                  where num % 2 == 0
                  select num;

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

Sintaxi de Mètode

La sintaxi de mètode utilitza mètodes d'extensió i expressions lambda.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = numbers.Where(num => num % 2 == 0);

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

Operadors LINQ

LINQ proporciona una varietat d'operadors per realitzar diferents operacions sobre dades. Aquí hi ha alguns operadors clau:

  • Where: Filtra elements basats en una condició.
  • Select: Projecta cada element en una nova forma.
  • OrderBy: Ordena elements en ordre ascendent.
  • GroupBy: Agrupa elements que comparteixen un atribut comú.
  • Join: Uneix dues seqüències basades en una clau.

Exemple: Utilitzant Múltiples Operadors

string[] names = { "John", "Jane", "Jake", "Jessie", "James" };

var filteredNames = names
    .Where(name => name.StartsWith("J"))
    .OrderBy(name => name)
    .Select(name => name.ToUpper());

foreach (var name in filteredNames)
{
    Console.WriteLine(name);
}

LINQ amb Diferents Fonts de Dades

LINQ a Objectes

LINQ es pot utilitzar per consultar col·leccions en memòria com arrays, llistes, etc.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };

var oddNumbers = from num in numbers
                 where num % 2 != 0
                 select num;

foreach (var num in oddNumbers)
{
    Console.WriteLine(num);
}

LINQ a SQL

LINQ es pot utilitzar per consultar bases de dades. Això requereix configurar un context de dades.

using (var context = new DataContext())
{
    var customers = from customer in context.Customers
                    where customer.City == "Seattle"
                    select customer;

    foreach (var customer in customers)
    {
        Console.WriteLine(customer.Name);
    }
}

LINQ a XML

LINQ es pot utilitzar per consultar documents XML.

XDocument doc = XDocument.Load("books.xml");

var books = from book in doc.Descendants("book")
            where (int)book.Element("price") > 20
            select book.Element("title").Value;

foreach (var title in books)
{
    Console.WriteLine(title);
}

Exercicis

  1. Filtratge Bàsic: Escriu una consulta LINQ per filtrar els números parells d'un array d'enters.
  2. Projecció: Utilitza LINQ per seleccionar els noms dels estudiants que han obtingut més de 80 d'una llista d'objectes estudiants.
  3. Ordenació: Escriu una consulta LINQ per ordenar una llista de productes pel seu preu en ordre descendent.
  4. Agrupació: Agrupa una llista d'empleats pel seu departament i mostra els noms dels departaments juntament amb el recompte d'empleats en cada departament.
  5. Unió: Utilitza LINQ per unir dues llistes: una que conté identificadors de clients i una altra que conté comandes, i mostra els identificadors de clients juntament amb els detalls de les seves comandes.

Resum

LINQ és una característica versàtil i poderosa en C# que simplifica la consulta de dades a través de diferents fonts de dades. Proporcionant una sintaxi consistent i aprofitant la potència del llenguatge C#, LINQ millora la productivitat i la llegibilitat del codi. Tant si treballes amb col·leccions en memòria, bases de dades o documents XML, LINQ ofereix un enfocament unificat per a la manipulació de dades.
