Introducció

Les piles i les cues són estructures de dades fonamentals en la informàtica. S'utilitzen per emmagatzemar i gestionar dades en un ordre específic, i entendre-les és crucial per resoldre diversos problemes de programació. En aquesta secció, explorarem com implementar i utilitzar piles i cues en C#.

Piles

Què és una pila?

Una pila és una col·lecció d'elements que segueix el principi Last In, First Out (LIFO). Això significa que l'últim element afegit a la pila serà el primer a ser eliminat.

Conceptes clau

  • Push: Afegir un element a la part superior de la pila.
  • Pop: Eliminar l'element superior de la pila.
  • Peek: Veure l'element superior sense eliminar-lo.
  • IsEmpty: Comprovar si la pila està buida.

Implementació d'una pila en C#

C# proporciona una classe Stack<T> integrada en l'espai de noms System.Collections.Generic. Aquí teniu com utilitzar-la:

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Stack<int> stack = new Stack<int>();

        // Afegir elements a la pila
        stack.Push(1);
        stack.Push(2);
        stack.Push(3);

        // Veure l'element superior
        Console.WriteLine("Element superior: " + stack.Peek()); // Sortida: 3

        // Eliminar elements de la pila
        Console.WriteLine("Element eliminat: " + stack.Pop()); // Sortida: 3
        Console.WriteLine("Element eliminat: " + stack.Pop()); // Sortida: 2

        // Comprovar si la pila està buida
        Console.WriteLine("La pila està buida? " + stack.Count == 0); // Sortida: Fals
    }
}

Implementació personalitzada d'una pila

Per a una comprensió més profunda, implementem una pila des de zero:

public class CustomStack<T>
{
    private List<T> elements = new List<T>();

    public void Push(T item)
    {
        elements.Add(item);
    }

    public T Pop()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La pila està buida");

        T item = elements[elements.Count - 1];
        elements.RemoveAt(elements.Count - 1);
        return item;
    }

    public T Peek()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La pila està buida");

        return elements[elements.Count - 1];
    }

    public bool IsEmpty()
    {
        return elements.Count == 0;
    }
}

Cues

Què és una cua?

Una cua és una col·lecció d'elements que segueix el principi First In, First Out (FIFO). Això significa que el primer element afegit a la cua serà el primer a ser eliminat.

Conceptes clau

  • Enqueue: Afegir un element al final de la cua.
  • Dequeue: Eliminar l'element frontal de la cua.
  • Peek: Veure l'element frontal sense eliminar-lo.
  • IsEmpty: Comprovar si la cua està buida.

Implementació d'una cua en C#

C# proporciona una classe Queue<T> integrada en l'espai de noms System.Collections.Generic. Aquí teniu com utilitzar-la:

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Queue<int> queue = new Queue<int>();

        // Afegir elements a la cua
        queue.Enqueue(1);
        queue.Enqueue(2);
        queue.Enqueue(3);

        // Veure l'element frontal
        Console.WriteLine("Element frontal: " + queue.Peek()); // Sortida: 1

        // Eliminar elements de la cua
        Console.WriteLine("Element eliminat: " + queue.Dequeue()); // Sortida: 1
        Console.WriteLine("Element eliminat: " + queue.Dequeue()); // Sortida: 2

        // Comprovar si la cua està buida
        Console.WriteLine("La cua està buida? " + queue.Count == 0); // Sortida: Fals
    }
}

Implementació personalitzada d'una cua

Per a una comprensió més profunda, implementem una cua des de zero:

public class CustomQueue<T>
{
    private List<T> elements = new List<T>();

    public void Enqueue(T item)
    {
        elements.Add(item);
    }

    public T Dequeue()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La cua està buida");

        T item = elements[0];
        elements.RemoveAt(0);
        return item;
    }

    public T Peek()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La cua està buida");

        return elements[0];
    }

    public bool IsEmpty()
    {
        return elements.Count == 0;
    }
}

Resum

En aquesta secció, hem cobert els conceptes bàsics de les piles i les cues, incloent-hi les seves operacions clau i com implementar-les en C#. Hem explorat tant les classes integrades Stack<T> i Queue<T> com les implementacions personalitzades per aprofundir en la nostra comprensió. Dominar aquestes estructures de dades és essencial per a una gestió eficient de dades i la resolució de problemes en la programació.
