Introducció a ASP.NET
Construcció d'una Aplicació Web Simple ►

ASP.NET és un potent framework per construir aplicacions web utilitzant C# i l'ecosistema .NET. Aquesta secció t'introduirà als conceptes bàsics d'ASP.NET, els seus conceptes clau i com començar a construir la teva primera aplicació web.

Què és ASP.NET?

ASP.NET és un framework de codi obert, del costat del servidor, dissenyat per al desenvolupament web per produir pàgines web dinàmiques. Va ser desenvolupat per Microsoft per permetre als programadors construir llocs web, aplicacions i serveis dinàmics.

Característiques Clau d'ASP.NET

  • Multiplataforma: ASP.NET Core és multiplataforma, el que significa que funciona en Windows, macOS i Linux.
  • Alt Rendiment: Optimitzat per al rendiment, fent-lo adequat per a aplicacions amb molt trànsit.
  • MVC i Web API Unificats: Combina les característiques de MVC (Model-View-Controller) i Web API en un sol framework.
  • Injecció de Dependències: Suport integrat per a la injecció de dependències, facilitant la gestió de dependències.
  • Razor Pages: Simplifica el desenvolupament d'escenaris centrats en pàgines utilitzant un model de codificació basat en pàgines.

Configuració del Teu Entorn de Desenvolupament

Abans de començar a desenvolupar amb ASP.NET, necessites configurar el teu entorn de desenvolupament.

Prerequisits

  • Visual Studio: Descarrega i instal·la Visual Studio (Community, Professional o Enterprise).
  • .NET SDK: Assegura't de tenir instal·lat el .NET SDK.

Creant el Teu Primer Projecte ASP.NET

  1. Obre Visual Studio.
  2. Crea un nou projecte:
    • Selecciona "Create a new project".
    • Tria "ASP.NET Core Web Application" i fes clic a "Next".
  3. Configura el teu nou projecte:
    • Posa un nom al teu projecte (per exemple, "MyFirstAspNetApp").
    • Tria una ubicació per desar el teu projecte.
    • Fes clic a "Create".
  4. Selecciona una plantilla:
    • Tria "Web Application (Model-View-Controller)".
    • Fes clic a "Create".

Entenent l'Estructura del Projecte

Quan crees un nou projecte ASP.NET, Visual Studio genera una estructura de projecte per defecte. Aquí tens els components clau:

  • Controllers: Gestionen les sol·licituds HTTP entrants i envien respostes al client.
  • Views: Defineixen la interfície d'usuari (UI) utilitzant la sintaxi Razor.
  • Models: Representen les dades i la lògica de negoci.
  • wwwroot: Conté fitxers estàtics com CSS, JavaScript i imatges.
  • appsettings.json: Fitxer de configuració per a les configuracions de l'aplicació.

Exemple: Controlador Bàsic

using Microsoft.AspNetCore.Mvc;

namespace MyFirstAspNetApp.Controllers
{
    public class HomeController : Controller
    {
        public IActionResult Index()
        {
            return View();
        }
    }
}

En aquest exemple, la classe HomeController hereta de Controller i defineix una acció Index que retorna una vista.

Exemple: Vista Bàsica

Crea un nou fitxer anomenat Index.cshtml a la carpeta Views/Home:

@{
    ViewData["Title"] = "Home Page";
}

<h1>Benvingut a ASP.NET!</h1>
<p>Aquesta és la teva primera aplicació ASP.NET.</p>

Aquesta vista Razor mostra un missatge de benvinguda senzill.

Executant la Teva Aplicació

Per executar la teva aplicació:

  1. Prem F5 o fes clic al botó "Run" a Visual Studio.
  2. El teu navegador web s'obrirà i hauràs de veure el missatge de benvinguda definit a la teva vista Index.cshtml.

Resum

En aquesta introducció a ASP.NET, has après:

  • Què és ASP.NET i les seves característiques clau.
  • Com configurar el teu entorn de desenvolupament.
  • Com crear un nou projecte ASP.NET a Visual Studio.
  • L'estructura bàsica d'un projecte ASP.NET.
  • Com crear un controlador i una vista senzills.

Entenent aquests conceptes fonamentals, ara estàs preparat per aprofundir en ASP.NET i construir aplicacions web més complexes.
