Introducció a les Llistes Enllaçades

Les llistes enllaçades són una estructura de dades fonamental en la informàtica, proporcionant una manera d'emmagatzemar una col·lecció d'elements en un ordre lineal. A diferència dels arrays, les llistes enllaçades no requereixen una assignació de memòria contigua, fent-les més flexibles per a certs tipus d'operacions.

Conceptes Clau

  • Node: La unitat bàsica d'una llista enllaçada, que conté dades i una referència (o punter) al següent node.
  • Cap: El primer node d'una llista enllaçada.
  • Cua: L'últim node d'una llista enllaçada, que apunta a null.
  • Llista Enllaçada Simple: Una llista enllaçada on cada node apunta al següent node.
  • Llista Enllaçada Doble: Una llista enllaçada on cada node apunta tant al següent com al node anterior.

Llista Enllaçada Simple

Estructura d'un Node

En C#, un node en una llista enllaçada simple es pot representar com una classe amb dues propietats: una per a les dades i una per a la referència al següent node.

public class Node
{
    public int Data { get; set; }
    public Node Next { get; set; }

    public Node(int data)
    {
        Data = data;
        Next = null;
    }
}

Creació d'una Llista Enllaçada Simple

Per crear una llista enllaçada simple, necessitem una classe que gestioni els nodes i proporcioni mètodes per manipular la llista.

public class SinglyLinkedList
{
    private Node head;

    public SinglyLinkedList()
    {
        head = null;
    }

    // Mètode per afegir un node al final
    public void AddLast(int data)
    {
        Node newNode = new Node(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            Node current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
        }
    }

    // Mètode per mostrar la llista
    public void Display()
    {
        Node current = head;
        while (current != null)
        {
            Console.Write(current.Data + " -> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Exemple d'Ús

class Program
{
    static void Main(string[] args)
    {
        SinglyLinkedList list = new SinglyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Sortida: 1 -> 2 -> 3 -> null
    }
}

Llista Enllaçada Doble

Estructura d'un Node

Un node en una llista enllaçada doble conté referències tant al següent com al node anterior.

public class DoublyNode
{
    public int Data { get; set; }
    public DoublyNode Next { get; set; }
    public DoublyNode Previous { get; set; }

    public DoublyNode(int data)
    {
        Data = data;
        Next = null;
        Previous = null;
    }
}

Creació d'una Llista Enllaçada Doble

Una classe de llista enllaçada doble gestiona els nodes i proporciona mètodes per manipular la llista.

public class DoublyLinkedList
{
    private DoublyNode head;

    public DoublyLinkedList()
    {
        head = null;
    }

    // Mètode per afegir un node al final
    public void AddLast(int data)
    {
        DoublyNode newNode = new DoublyNode(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            DoublyNode current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
            newNode.Previous = current;
        }
    }

    // Mètode per mostrar la llista
    public void Display()
    {
        DoublyNode current = head;
        while (current != null)
        {
            Console.Write(current.Data + " <-> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Exemple d'Ús

class Program
{
    static void Main(string[] args)
    {
        DoublyLinkedList list = new DoublyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Sortida: 1 <-> 2 <-> 3 <-> null
    }
}

Operacions Avançades

Inserció al Principi

Afegir un node al principi de la llista requereix actualitzar el cap i la referència següent del nou node.

public void AddFirst(int data)
{
    Node newNode = new Node(data);
    newNode.Next = head;
    head = newNode;
}

Eliminació d'un Node

Eliminar un node implica actualitzar les referències dels nodes anterior i següent per saltar-se el node a eliminar.

public void Delete(int data)
{
    if (head == null) return;

    if (head.Data == data)
    {
        head = head.Next;
        return;
    }

    Node current = head;
    while (current.Next != null && current.Next.Data != data)
    {
        current = current.Next;
    }

    if (current.Next != null)
    {
        current.Next = current.Next.Next;
    }
}

Reversió d'una Llista Enllaçada

Revertir una llista enllaçada implica canviar la direcció dels punters.

public void Reverse()
{
    Node previous = null;
    Node current = head;
    Node next = null;

    while (current != null)
    {
        next = current.Next;
        current.Next = previous;
        previous = current;
        current = next;
    }

    head = previous;
}

Conclusió

Les llistes enllaçades són una estructura de dades versàtil que es pot utilitzar per gestionar col·leccions de dades de manera eficient. Entendre com implementar i manipular llistes enllaçades en C# és una habilitat valuosa per a qualsevol programador. Dominant tant les llistes enllaçades simples com les dobles, així com operacions avançades com la inserció, eliminació i reversió, pots afrontar una àmplia gamma de desafiaments de programació.
