Entity Framework
Introducció a ASP.NET ►

Entity Framework (EF) és un mapador objecte-relacional (ORM) de codi obert per a ADO.NET, que permet als desenvolupadors treballar amb una base de dades utilitzant objectes .NET. Elimina la necessitat de la major part del codi d'accés a dades que els desenvolupadors normalment necessiten escriure.

Introducció a Entity Framework

  • Què és Entity Framework?

    • Un framework ORM per a aplicacions .NET.
    • Simplifica l'accés a dades permetent als desenvolupadors treballar amb dades com a objectes.
    • Suporta múltiples proveïdors de bases de dades (SQL Server, SQLite, MySQL, etc.).

  • Per què utilitzar Entity Framework?

    • Redueix la quantitat de codi repetitiu.
    • Proporciona un nivell més alt d'abstracció per a les operacions de bases de dades.
    • Suporta consultes LINQ, fent la manipulació de dades més intuïtiva.

Configuració d'Entity Framework

  • Instal·lació d'Entity Framework

    • Utilitza el Gestor de Paquets NuGet a Visual Studio.
    • Comanda: Install-Package EntityFramework

  • Creació d'un Model

    • Defineix el teu model de dades utilitzant classes.
    • Exemple: 
      public class Product
{
    public int ProductId { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
}

  • Classe DbContext

    • Actua com un pont entre el teu model i la base de dades.
    • Exemple: 
      public class ApplicationDbContext : DbContext
{
    public DbSet<Product> Products { get; set; }
}

Operacions bàsiques CRUD

  • Crear

    • Afegir una nova entitat a la base de dades.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var product = new Product { Name = "Laptop", Price = 999.99M };
    context.Products.Add(product);
    context.SaveChanges();
}

  • Llegir

    • Recuperar entitats de la base de dades.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var products = context.Products.ToList();
    foreach (var product in products)
    {
        Console.WriteLine($"Product: {product.Name}, Price: {product.Price}");
    }
}

  • Actualitzar

    • Modificar una entitat existent.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var product = context.Products.First(p => p.ProductId == 1);
    product.Price = 899.99M;
    context.SaveChanges();
}

  • Eliminar

    • Eliminar una entitat de la base de dades.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var product = context.Products.First(p => p.ProductId == 1);
    context.Products.Remove(product);
    context.SaveChanges();
}

Funcionalitats avançades

  • Migracions

    • Gestionar els canvis d'esquema de la base de dades.
    • Comandes:
      • Afegir Migració: Add-Migration MigrationName
      • Actualitzar Base de Dades: Update-Database

  • Relacions

    • Definir relacions entre entitats (un-a-molts, molts-a-molts).
    • Exemple: 
      public class Category
{
    public int CategoryId { get; set; }
    public string Name { get; set; }
    public ICollection<Product> Products { get; set; }
}

public class Product
{
    public int ProductId { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
    public int CategoryId { get; set; }
    public Category Category { get; set; }
}

  • Consultes LINQ

    • Utilitza LINQ per consultar la base de dades.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var expensiveProducts = context.Products
                                   .Where(p => p.Price > 500)
                                   .ToList();
}

  • Operacions Asíncrones

    • Realitzar operacions de base de dades de manera asíncrona.
    • Exemple: 
      using (var context = new ApplicationDbContext())
{
    var products = await context.Products.ToListAsync();
}

Consideracions de rendiment

  • Càrrega Peresa vs. Càrrega Eager

    • Càrrega Peresa: Carrega dades relacionades sota demanda.
    • Càrrega Eager: Carrega dades relacionades com a part de la consulta inicial.
    • Exemple de Càrrega Eager: 
      using (var context = new ApplicationDbContext())
{
    var products = context.Products.Include(p => p.Category).ToList();
}

  • Consultes amb Seguiment vs. Sense Seguiment

    • Seguiment: Manté un seguiment dels canvis a les entitats.
    • Sense Seguiment: Millor rendiment per a operacions de només lectura.
    • Exemple de Consulta Sense Seguiment: 
      using (var context = new ApplicationDbContext())
{
    var products = context.Products.AsNoTracking().ToList();
}

Conclusió

Entity Framework és un potent ORM que simplifica l'accés a dades en aplicacions .NET. Comprenent les seves funcionalitats bàsiques i avançades, pots gestionar eficientment les teves operacions de base de dades amb un mínim de codi. Tant si estàs realitzant operacions bàsiques CRUD com si estàs optimitzant el rendiment amb tècniques avançades, Entity Framework proporciona les eines que necessites per construir aplicacions robustes i escalables.
