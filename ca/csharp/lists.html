Introducció a les Llistes

Les llistes en C# formen part del namespace System.Collections.Generic i proporcionen una manera d'emmagatzemar col·leccions d'objectes. A diferència dels arrays, les llistes poden créixer i reduir-se dinàmicament en mida.

  • Namespace: System.Collections.Generic
  • Classe: List<T>

Conceptes Clau

  • Mida Dinàmica: Les llistes poden redimensionar-se automàticament a mesura que s'afegeixen o s'eliminen elements.
  • Seguretat de Tipus: Les llistes són fortament tipades, el que significa que només pots emmagatzemar elements d'un tipus especificat.
  • Indexació Basada en Zero: Com els arrays, les llistes estan indexades a partir de zero.

Crear i Inicialitzar Llistes

Per crear una llista, cal especificar el tipus d'elements que emmagatzemarà.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Crear una llista d'enters
        List<int> numbers = new List<int>();

        // Inicialitzar una llista amb valors
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Mostrar la llista
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}

Explicació

  • List<int> numbers = new List<int>();: Crea una llista buida d'enters.
  • List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };: Crea i inicialitza una llista de cadenes amb tres elements.

Afegir i Eliminar Elements

Pots afegir elements a una llista utilitzant el mètode Add i eliminar-los utilitzant els mètodes Remove o RemoveAt.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 1, 2, 3 };

        // Afegir elements
        numbers.Add(4);
        numbers.Add(5);

        // Eliminar elements
        numbers.Remove(2); // Elimina la primera ocurrència de 2
        numbers.RemoveAt(0); // Elimina l'element a l'índex 0

        // Mostrar la llista
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explicació

  • numbers.Add(4);: Afegeix l'element 4 al final de la llista.
  • numbers.Remove(2);: Elimina la primera ocurrència de l'element 2.
  • numbers.RemoveAt(0);: Elimina l'element a l'índex 0.

Accedir als Elements

Pots accedir als elements d'una llista utilitzant l'indexador.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Accedir als elements
        string firstFruit = fruits[0];
        string secondFruit = fruits[1];

        Console.WriteLine(firstFruit); // Sortida: Apple
        Console.WriteLine(secondFruit); // Sortida: Banana
    }
}

Explicació

  • fruits[0]: Accedeix al primer element de la llista.
  • fruits[1]: Accedeix al segon element de la llista.

Iterar sobre una Llista

Pots iterar sobre una llista utilitzant un bucle foreach o un bucle for.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Utilitzant bucle foreach
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }

        // Utilitzant bucle for
        for (int i = 0; i < fruits.Count; i++)
        {
            Console.WriteLine(fruits[i]);
        }
    }
}

Explicació

  • foreach (string fruit in fruits): Itera sobre cada element de la llista.
  • for (int i = 0; i < fruits.Count; i++): Itera sobre la llista utilitzant un índex.

Operacions Avançades

Ordenar una Llista

Pots ordenar una llista utilitzant el mètode Sort.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 3, 1, 4, 1, 5, 9 };

        // Ordenar la llista
        numbers.Sort();

        // Mostrar la llista ordenada
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explicació

  • numbers.Sort();: Ordena la llista en ordre ascendent.

Trobar Elements

Pots trobar elements en una llista utilitzant mètodes com Contains, Find, i FindAll.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Comprovar si la llista conté un element específic
        bool hasApple = fruits.Contains("Apple");

        // Trobar el primer element que compleix una condició
        string fruitWithA = fruits.Find(f => f.Contains("a"));

        // Trobar tots els elements que compleixen una condició
        List<string> fruitsWithA = fruits.FindAll(f => f.Contains("a"));

        Console.WriteLine(hasApple); // Sortida: True
        Console.WriteLine(fruitWithA); // Sortida: Banana
        foreach (string fruit in fruitsWithA)
        {
            Console.WriteLine(fruit); // Sortida: Banana, Cherry
        }
    }
}

Explicació

  • fruits.Contains("Apple");: Comprova si "Apple" està a la llista.
  • fruits.Find(f => f.Contains("a"));: Troba el primer element que conté la lletra "a".
  • fruits.FindAll(f => f.Contains("a"));: Troba tots els elements que contenen la lletra "a".

Conclusió

Les llistes en C# són una manera versàtil i potent de gestionar col·leccions d'objectes. Ofereixen mida dinàmica, seguretat de tipus i una varietat de mètodes per afegir, eliminar, accedir i manipular elements. Entenent i utilitzant les llistes, pots escriure programes en C# més eficients i flexibles.
