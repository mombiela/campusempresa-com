Introducció als fluxos de fitxers

Els fluxos de fitxers en C# s'utilitzen per llegir i escriure fitxers. Proporcionen una manera de gestionar les operacions de fitxers de manera eficient i segura. Entendre els fluxos de fitxers és crucial per a tasques com l'emmagatzematge de dades, la gestió de configuracions i el registre de logs.

Conceptes bàsics

  • Flux: Una seqüència de bytes que es poden llegir o escriure.
  • FileStream: Una classe al namespace System.IO utilitzada per llegir i escriure fitxers.
  • StreamReader: Una classe per llegir caràcters d'un flux de bytes en una codificació particular.
  • StreamWriter: Una classe per escriure caràcters a un flux en una codificació particular.

Treballant amb FileStream

Llegint d'un fitxer

Per llegir d'un fitxer, pots utilitzar la classe FileStream juntament amb StreamReader.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read))
        using (StreamReader sr = new StreamReader(fs))
        {
            string content = sr.ReadToEnd();
            Console.WriteLine(content);
        }
    }
}
  • FileMode.Open: Obre el fitxer si existeix.
  • FileAccess.Read: Concedeix accés de lectura al fitxer.
  • ReadToEnd(): Llegeix tots els caràcters des de la posició actual fins al final del flux.

Escriure a un fitxer

Per escriure a un fitxer, pots utilitzar la classe FileStream juntament amb StreamWriter.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write))
        using (StreamWriter sw = new StreamWriter(fs))
        {
            sw.Write(content);
        }
    }
}
  • FileMode.Create: Crea un nou fitxer, o sobreescriu un fitxer existent.
  • FileAccess.Write: Concedeix accés d'escriptura al fitxer.
  • Write(): Escriu una cadena al flux.

Operacions avançades amb fluxos de fitxers

Operacions de fitxers asíncrones

Les operacions asíncrones poden millorar el rendiment evitant bloquejar el fil principal.

Lectura asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read, FileShare.Read, 4096, true))
        using (StreamReader sr = new StreamReader(fs))
        {
            string content = await sr.ReadToEndAsync();
            Console.WriteLine(content);
        }
    }
}
  • ReadToEndAsync(): Llegeix asíncronament tots els caràcters des de la posició actual fins al final del flux.

Escriptura asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = "Hello, Async World!";
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write, FileShare.None, 4096, true))
        using (StreamWriter sw = new StreamWriter(fs))
        {
            await sw.WriteAsync(content);
        }
    }
}
  • WriteAsync(): Escriu asíncronament una cadena al flux.

Gestió de dades binàries

Per a dades binàries, utilitza BinaryReader i BinaryWriter.

Llegint dades binàries

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.bin";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read))
        using (BinaryReader br = new BinaryReader(fs))
        {
            int number = br.ReadInt32();
            Console.WriteLine(number);
        }
    }
}
  • ReadInt32(): Llegeix un enter de 4 bytes amb signe del flux.

Escriure dades binàries

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.bin";
        int number = 12345;
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write))
        using (BinaryWriter bw = new BinaryWriter(fs))
        {
            bw.Write(number);
        }
    }
}
  • Write(int): Escriu un enter de 4 bytes amb signe al flux.

Resum

Els fluxos de fitxers en C# proporcionen una manera poderosa de gestionar operacions de fitxers, des de la lectura i escriptura bàsiques fins a la gestió més avançada de dades asíncrones i binàries. Dominant els fluxos de fitxers, pots gestionar eficientment l'emmagatzematge i la recuperació de dades en les teves aplicacions.
