Introducció

En C#, les classes i els objectes són conceptes fonamentals de la programació orientada a objectes (OOP). Entendre aquests conceptes és crucial per escriure codi eficient i organitzat. Aquesta secció et guiarà des dels conceptes bàsics fins als temes avançats relacionats amb les classes i els objectes en C#.

Què és una Classe?

Una classe és un plànol per crear objectes. Defineix un tipus de dada agrupant dades i mètodes que treballen sobre les dades en una sola unitat.

Conceptes Clau

  • Definició de Classe: Una classe es defineix utilitzant la paraula clau class.
  • Membres: Les classes poden contenir camps, propietats, mètodes i esdeveniments.
  • Modificadors d'Accés: Controlen la visibilitat dels membres de la classe (per exemple, public, private, protected).

Exemple

public class Person
{
    // Camps
    private string name;
    private int age;

    // Propietats
    public string Name
    {
        get { return name; }
        set { name = value; }
    }

    public int Age
    {
        get { return age; }
        set { age = value; }
    }

    // Mètode
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}

Què és un Objecte?

Un objecte és una instància d'una classe. Quan es defineix una classe, no es reserva memòria fins que es crea un objecte d'aquesta classe.

Conceptes Clau

  • Instanciació: Crear un objecte utilitzant la paraula clau new.
  • Accés als Membres: Utilitzar l'operador punt (.) per accedir a camps, propietats i mètodes.

Exemple

// Creant un objecte de la classe Person
Person person1 = new Person();
person1.Name = "John";
person1.Age = 30;
person1.DisplayInfo(); // Sortida: Name: John, Age: 30

Constructors

Els constructors són mètodes especials que es criden quan es crea un objecte. S'utilitzen per inicialitzar objectes.

Conceptes Clau

  • Constructor per Defecte: Un constructor sense paràmetres.
  • Constructor Parametritzat: Un constructor que pren paràmetres.
  • Sobrecàrrega de Constructors: Definir múltiples constructors amb diferents paràmetres.

Exemple

public class Person
{
    // Camps
    private string name;
    private int age;

    // Constructor per Defecte
    public Person()
    {
        name = "Unknown";
        age = 0;
    }

    // Constructor Parametritzat
    public Person(string name, int age)
    {
        this.name = name;
        this.age = age;
    }

    // Mètode
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {name}, Age: {age}");
    }
}

// Utilitzant Constructors
Person person1 = new Person(); // Constructor per defecte
Person person2 = new Person("Alice", 25); // Constructor parametritzat
person1.DisplayInfo(); // Sortida: Name: Unknown, Age: 0
person2.DisplayInfo(); // Sortida: Name: Alice, Age: 25

Herència

L'herència permet a una classe heretar camps i mètodes d'una altra classe. La classe que hereta s'anomena classe derivada, i la classe de la qual s'hereta s'anomena classe base.

Conceptes Clau

  • Classe Base: La classe de la qual s'hereta.
  • Classe Derivada: La classe que hereta de la classe base.
  • Paraula clau base: S'utilitza per accedir als membres de la classe base.

Exemple

// Classe Base
public class Animal
{
    public void Eat()
    {
        Console.WriteLine("Eating...");
    }
}

// Classe Derivada
public class Dog : Animal
{
    public void Bark()
    {
        Console.WriteLine("Barking...");
    }
}

// Utilitzant Herència
Dog dog = new Dog();
dog.Eat(); // Sortida: Eating...
dog.Bark(); // Sortida: Barking...

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen. Es pot aconseguir mitjançant la sobrescriptura de mètodes i la sobrecàrrega de mètodes.

Conceptes Clau

  • Sobrescriptura de Mètodes: Redefinir un mètode en una classe derivada.
  • Sobrecàrrega de Mètodes: Definir múltiples mètodes amb el mateix nom però diferents paràmetres.

Exemple

// Classe Base
public class Animal
{
    public virtual void MakeSound()
    {
        Console.WriteLine("Animal sound");
    }
}

// Classe Derivada
public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Bark");
    }
}

// Utilitzant Polimorfisme
Animal myAnimal = new Dog();
myAnimal.MakeSound(); // Sortida: Bark

Encapsulació

L'encapsulació és el concepte d'embolcallar dades i mètodes que operen sobre les dades dins d'una sola unitat (classe). Restringeix l'accés directe a alguns dels components de l'objecte.

Conceptes Clau

  • Membres Privats: Membres que no es poden accedir directament des de fora de la classe.
  • Mètodes Públics: Mètodes que proporcionen accés controlat als membres privats.

Exemple

public class Account
{
    private double balance;

    public void Deposit(double amount)
    {
        if (amount > 0)
        {
            balance += amount;
        }
    }

    public double GetBalance()
    {
        return balance;
    }
}

// Utilitzant Encapsulació
Account myAccount = new Account();
myAccount.Deposit(100);
Console.WriteLine(myAccount.GetBalance()); // Sortida: 100

Conclusió

Les classes i els objectes són els blocs de construcció de la programació orientada a objectes en C#. Entendre aquests conceptes et permet crear codi modular, reutilitzable i mantenible. En dominar les classes, els objectes, l'herència, el polimorfisme i l'encapsulació, pots aprofitar tot el potencial de C# per construir aplicacions robustes.
