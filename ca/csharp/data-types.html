Introducció als Tipus de Dades

Els tipus de dades són fonamentals per a qualsevol llenguatge de programació, i C# no és una excepció. Defineixen el tipus de dades que es poden emmagatzemar i manipular dins d'un programa. Entendre els tipus de dades és crucial per escriure codi eficient i lliure d'errors.

Tipus de Dades Primitives

C# proporciona diversos tipus de dades integrats, també coneguts com a tipus de dades primitives. Aquests són els tipus més bàsics i s'utilitzen per emmagatzemar valors simples.

Tipus Enter

  • int: Representa un enter de 32 bits amb signe.
  • long: Representa un enter de 64 bits amb signe.
  • short: Representa un enter de 16 bits amb signe.
  • byte: Representa un enter de 8 bits sense signe.
int age = 25;
long distance = 9876543210L;
short temperature = -5;
byte level = 255;

Tipus de Punt Flotant

  • float: Representa un nombre de punt flotant de precisió simple.
  • double: Representa un nombre de punt flotant de doble precisió.
  • decimal: Representa un valor decimal precís de 128 bits, adequat per a càlculs financers.
float pi = 3.14F;
double e = 2.7182818284;
decimal price = 19.99M;

Tipus Booleà

  • bool: Representa un valor booleà que pot ser true o false.
bool isActive = true;
bool isCompleted = false;

Tipus Caràcter

  • char: Representa un únic caràcter Unicode de 16 bits.
char grade = 'A';
char symbol = '#';

Tipus Cadena

  • string: Representa una seqüència de caràcters.
string greeting = "Hello, World!";
string name = "Alice";

Tipus de Dades Complexes

C# també admet tipus de dades més complexos que poden emmagatzemar múltiples valors o estructures més complexes.

Arrays

Els arrays són col·leccions d'elements del mateix tipus.

int[] numbers = { 1, 2, 3, 4, 5 };
string[] names = { "Alice", "Bob", "Charlie" };

Llistes

Les llistes són similars als arrays però són més flexibles i formen part del namespace System.Collections.Generic.

using System.Collections.Generic;

List<int> scores = new List<int> { 90, 85, 100 };
scores.Add(95);

Diccionaris

Els diccionaris emmagatzemen parelles clau-valor i formen part del namespace System.Collections.Generic.

using System.Collections.Generic;

Dictionary<string, int> ages = new Dictionary<string, int>();
ages["Alice"] = 30;
ages["Bob"] = 25;

Tipus Nullable

Els tipus nullable permeten que els tipus de valor es puguin assignar a null.

int? nullableInt = null;
nullableInt = 5;

Conversió de Tipus

La conversió de tipus és el procés de convertir un tipus de dada en un altre. C# admet tant conversions implícites com explícites.

Conversió Implícita

Les conversions implícites són gestionades automàticament pel compilador.

int num = 123;
double doubleNum = num; // Conversió implícita de int a double

Conversió Explícita

Les conversions explícites requereixen un operador de cast.

double doubleNum = 123.45;
int num = (int)doubleNum; // Conversió explícita de double a int

Resum

Entendre els tipus de dades és essencial per a qualsevol programador de C#. Aquest tema ha cobert:

  • Tipus de dades primitives com int, float, bool, char i string.
  • Tipus de dades complexes com arrays, llistes i diccionaris.
  • Tipus nullable i conversió de tipus.

En dominar aquests conceptes, estaràs ben preparat per gestionar dades de manera efectiva en els teus programes C#.
