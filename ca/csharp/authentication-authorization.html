L'autenticació i l'autorització són components crítics de qualsevol aplicació segura. En aquesta secció, explorarem aquests conceptes en el context de la programació en C#, des dels principis bàsics fins a les implementacions avançades.

Introducció a l'Autenticació i l'Autorització

L'autenticació és el procés de verificar la identitat d'un usuari o sistema. L'autorització, en canvi, determina què pot fer un usuari autenticat. Ambdues són essencials per assegurar aplicacions.

Conceptes Clau

  • Autenticació: Verificar la identitat d'un usuari.
  • Autorització: Determinar què pot fer un usuari autenticat.
  • Identitat: Informació sobre l'usuari (per exemple, nom d'usuari, rols).
  • Afirmacions: Declaracions sobre l'usuari (per exemple, correu electrònic, rol).
  • Tokens: Peces d'informació codificades utilitzades per a l'autenticació i l'autorització.

Autenticació Bàsica en C#

Utilitzant Autenticació Bàsica

L'autenticació bàsica implica enviar un nom d'usuari i una contrasenya amb cada sol·licitud. Això es fa típicament utilitzant capçaleres HTTP.

using System;
using System.Net.Http;
using System.Net.Http.Headers;
using System.Text;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        var client = new HttpClient();
        var authToken = Encoding.ASCII.GetBytes("username:password");
        client.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Basic", Convert.ToBase64String(authToken));

        var response = await client.GetAsync("https://example.com/protected-resource");
        if (response.IsSuccessStatusCode)
        {
            Console.WriteLine("Accés concedit");
        }
        else
        {
            Console.WriteLine("Accés denegat");
        }
    }
}

Explicació

  • HttpClient: Utilitzat per enviar sol·licituds HTTP.
  • Capçalera d'Autorització: Conté el nom d'usuari i la contrasenya codificats.
  • Codificació Base64: Converteix el nom d'usuari i la contrasenya a una cadena base64.

Autenticació Basada en Tokens

L'autenticació basada en tokens implica emetre un token a l'usuari després d'un inici de sessió exitós. El token s'utilitza per a sol·licituds posteriors.

Utilitzant JWT (JSON Web Tokens)

JWT és un format de token popular utilitzat per a l'autenticació.

using System;
using System.IdentityModel.Tokens.Jwt;
using System.Security.Claims;
using System.Text;
using Microsoft.IdentityModel.Tokens;

class Program
{
    static void Main(string[] args)
    {
        var tokenHandler = new JwtSecurityTokenHandler();
        var key = Encoding.ASCII.GetBytes("your-256-bit-secret");
        var tokenDescriptor = new SecurityTokenDescriptor
        {
            Subject = new ClaimsIdentity(new Claim[]
            {
                new Claim(ClaimTypes.Name, "username"),
                new Claim(ClaimTypes.Role, "Admin")
            }),
            Expires = DateTime.UtcNow.AddHours(1),
            SigningCredentials = new SigningCredentials(new SymmetricSecurityKey(key), SecurityAlgorithms.HmacSha256Signature)
        };
        var token = tokenHandler.CreateToken(tokenDescriptor);
        var tokenString = tokenHandler.WriteToken(token);

        Console.WriteLine(tokenString);
    }
}

Explicació

  • JwtSecurityTokenHandler: Crea i valida tokens JWT.
  • ClaimsIdentity: Representa la identitat i les afirmacions de l'usuari.
  • SecurityTokenDescriptor: Descriu les propietats del token.
  • SigningCredentials: Utilitzat per signar el token.

Autorització en C#

Autorització Basada en Rols

L'autorització basada en rols restringeix l'accés basat en els rols dels usuaris.

using System;
using System.Collections.Generic;

class Program
{
    static void Main(string[] args)
    {
        var userRoles = new List<string> { "Admin", "User" };
        if (IsAuthorized(userRoles, "Admin"))
        {
            Console.WriteLine("Accés concedit a recursos d'Admin");
        }
        else
        {
            Console.WriteLine("Accés denegat");
        }
    }

    static bool IsAuthorized(List<string> userRoles, string requiredRole)
    {
        return userRoles.Contains(requiredRole);
    }
}

Explicació

  • List userRoles: Representa els rols assignats a l'usuari.
  • IsAuthorized: Comprova si l'usuari té el rol requerit.

Autorització Basada en Polítiques

L'autorització basada en polítiques permet regles més complexes.

using System;
using System.Collections.Generic;

class Program
{
    static void Main(string[] args)
    {
        var userClaims = new Dictionary<string, string>
        {
            { "Department", "HR" },
            { "Level", "Senior" }
        };

        if (IsAuthorized(userClaims, "Department", "HR") && IsAuthorized(userClaims, "Level", "Senior"))
        {
            Console.WriteLine("Accés concedit a recursos de HR Senior");
        }
        else
        {
            Console.WriteLine("Accés denegat");
        }
    }

    static bool IsAuthorized(Dictionary<string, string> userClaims, string claimType, string requiredValue)
    {
        return userClaims.ContainsKey(claimType) && userClaims[claimType] == requiredValue;
    }
}

Explicació

  • Dictionary<string, string> userClaims: Representa les afirmacions associades amb l'usuari.
  • IsAuthorized: Comprova si l'usuari té l'afirmació requerida.

Resum

En aquesta secció, hem cobert els conceptes bàsics de l'autenticació i l'autorització en C#. Hem explorat:

  • Autenticació bàsica utilitzant capçaleres HTTP.
  • Autenticació basada en tokens amb JWT.
  • Autorització basada en rols i basada en polítiques.

Entendre aquests conceptes és crucial per construir aplicacions segures. A mesura que avancis, considera integrar aquestes tècniques amb frameworks com ASP.NET Core per a solucions de seguretat més robustes.
