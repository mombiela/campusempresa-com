Configurar l'entorn de desenvolupament és el primer pas per començar a programar en C#. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar.

Instal·lant Visual Studio

Visual Studio és l'Entorn de Desenvolupament Integrat (IDE) més popular per al desenvolupament en C#. Proporciona una suite completa d'eines per construir, depurar i desplegar aplicacions.

Passos per Instal·lar Visual Studio

  1. Descarregar Visual Studio:

    • Ves a la pàgina de descàrrega de Visual Studio.
    • Tria la versió que s'adapti a les teves necessitats (Community, Professional o Enterprise).

  2. Executar l'Instal·lador:

    • Obre l'instal·lador descarregat.
    • Tria les càrregues de treball que necessites. Per al desenvolupament en C#, selecciona "Desenvolupament d'escriptori .NET" i "Desenvolupament web ASP.NET" si tens previst treballar en aplicacions web.

  3. Instal·lar els Components Necessaris:

    • L'instal·lador descarregarà i instal·larà els components necessaris. Això pot trigar una estona depenent de la velocitat d'internet i el rendiment del sistema.

  4. Llançar Visual Studio:

    • Un cop completada la instal·lació, llança Visual Studio.
    • Inicia sessió amb el teu compte de Microsoft si se't demana.

Configurant el Teu Primer Projecte en C#

Després d'instal·lar Visual Studio, el següent pas és crear el teu primer projecte en C#.

Creant un Nou Projecte

  1. Obrir Visual Studio:

    • Inicia Visual Studio des del menú d'inici o la drecera d'escriptori.

  2. Crear un Nou Projecte:

    • Fes clic a "Crear un nou projecte" des de la finestra d'inici.
    • Selecciona "Aplicació de consola (.NET Core)" per a una aplicació de consola simple.
    • Fes clic a "Següent."

  3. Configurar el Teu Projecte:

    • Introdueix un nom per al teu projecte.
    • Tria una ubicació per desar el teu projecte.
    • Fes clic a "Crear."

Entenent l'Estructura del Projecte

Quan crees un nou projecte en C#, Visual Studio genera diversos fitxers i carpetes. Aquí tens una breu visió general:

  • Program.cs: Aquest és el punt d'entrada principal de la teva aplicació.
  • Carpetes obj i bin: Aquestes carpetes contenen binaris compilats i altres fitxers temporals.
  • Fitxer .csproj: Aquest és el fitxer del projecte que conté la configuració del projecte.

Escriure el Teu Primer Programa en C#

Escrivim un simple programa "Hola, Món!" per assegurar-nos que tot està configurat correctament.

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hola, Món!");
        }
    }
}
  • Explicació:
    • using System; importa l'espai de noms System, que conté classes fonamentals.
    • namespace HelloWorld defineix un espai de noms anomenat HelloWorld.
    • class Program defineix una classe anomenada Program.
    • static void Main(string[] args) és el punt d'entrada de l'aplicació.
    • Console.WriteLine("Hola, Món!"); imprimeix "Hola, Món!" a la consola.

Configurant Visual Studio Code (Opcional)

Si prefereixes un editor lleuger, Visual Studio Code (VS Code) és una gran alternativa.

Passos per Instal·lar Visual Studio Code

  1. Descarregar VS Code:

    • Ves a la pàgina de descàrrega de Visual Studio Code.
    • Descarrega l'instal·lador per al teu sistema operatiu.

  2. Instal·lar VS Code:

    • Executa l'instal·lador i segueix les instruccions en pantalla.

  3. Instal·lar l'Extensió de C#:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca "C#".
    • Fes clic a "Instal·lar" a l'extensió de C# de Microsoft.

Creant un Nou Projecte en C# a VS Code

  1. Obrir una Nova Carpeta:

    • Fes clic a "Fitxer" > "Obrir Carpeta" i selecciona o crea una nova carpeta per al teu projecte.

  2. Inicialitzar un Nou Projecte:

    • Obre el terminal a VS Code (Vista > Terminal).
    • Executa la següent comanda per crear una nova aplicació de consola: 
      dotnet new console

  3. Executar la Teva Aplicació:

    • Al terminal, executa: 
      dotnet run

Resum

Configurar l'entorn de desenvolupament és un pas crucial per començar el teu viatge amb C#. Tant si tries Visual Studio com Visual Studio Code, ambdós proporcionen eines potents per ajudar-te a escriure, depurar i executar les teves aplicacions. Seguint els passos descrits anteriorment, ara hauries de tenir un entorn de treball llest per al desenvolupament en C#.
