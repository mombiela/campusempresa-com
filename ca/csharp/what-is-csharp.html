Introducció a C#

C# (pronunciat "C-sharp") és un llenguatge de programació modern i orientat a objectes desenvolupat per Microsoft com a part de la seva iniciativa .NET. Està dissenyat per ser senzill, potent, segur en termes de tipus i orientat a objectes. C# s'utilitza àmpliament per desenvolupar aplicacions per a Windows, aplicacions web i jocs utilitzant el motor de jocs Unity.

Característiques Clau de C#

  • Orientat a Objectes: C# suporta els principis de la programació orientada a objectes (OOP) com l'encapsulació, l'herència i el polimorfisme.
  • Segur en termes de Tipus: C# aplica una comprovació estricta de tipus, la qual cosa ajuda a prevenir errors de tipus i millora la fiabilitat del codi.
  • Orientat a Components: C# suporta el desenvolupament de components de programari, que es poden reutilitzar en diferents aplicacions.
  • Interoperable: C# pot interoperar amb altres llenguatges i tecnologies, especialment aquelles dins de l'ecosistema .NET.
  • Escalable i Actualitzable: Les aplicacions en C# es poden escalar i actualitzar fàcilment per satisfer els requisits canviants.

Sintaxi i Estructura Bàsica

Un programa senzill en C# típicament consisteix en un mètode Main, que serveix com a punt d'entrada per a l'aplicació. Aquí teniu un exemple d'un programa bàsic en C#:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}

Explicació

  • using System;: Aquesta línia inclou l'espai de noms System, que conté classes fonamentals i classes base que defineixen tipus de dades comunament utilitzats, esdeveniments i gestors d'esdeveniments, interfícies, atributs i excepcions de processament.
  • namespace HelloWorld: Els espais de noms s'utilitzen per organitzar el codi i prevenir conflictes de noms.
  • class Program: Una classe és un plànol per crear objectes. En aquest cas, Program és el nom de la classe.
  • static void Main(string[] args): El mètode Main és el punt d'entrada d'una aplicació en C#. La paraula clau static significa que el mètode pertany a la classe en si mateixa en lloc d'una instància de la classe. void indica que el mètode no retorna cap valor. string[] args és una matriu de cadenes que pot emmagatzemar arguments de línia de comandes.
  • Console.WriteLine("Hello, World!");: Aquesta línia imprimeix "Hello, World!" a la consola.

Conclusió

C# és un llenguatge de programació versàtil i potent que és adequat per a una àmplia gamma d'aplicacions. El seu fort tipatge, característiques orientades a objectes i interoperabilitat el fan una elecció popular entre els desenvolupadors. Tant si esteu construint aplicacions d'escriptori, serveis web o jocs, C# proporciona les eines i característiques necessàries per crear programari robust i eficient.

En les següents seccions d'aquest curs, aprofundirem en els diversos aspectes de la programació en C#, començant per la sintaxi bàsica i avançant cap a temes més avançats. Feliç programació!
