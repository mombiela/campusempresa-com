Introducció als Arrays

Els arrays són una estructura de dades fonamental en C# que et permeten emmagatzemar múltiples valors del mateix tipus en una sola variable. Són útils per gestionar col·leccions de dades de manera eficient.

Conceptes Clau

  • Definició: Un array és una col·lecció de variables que s'accedeixen amb un número d'índex.
  • Mida Fixa: Els arrays tenen una mida fixa que es defineix quan es creen.
  • Indexació Basada en Zero: El primer element d'un array està a l'índex 0.

Creació d'Arrays

Per crear un array en C#, necessites especificar el tipus d'elements que contindrà i la seva mida.

// Declaració i inicialització d'un array d'enters amb 5 elements
int[] numbers = new int[5];

// Declaració i inicialització d'un array amb valors específics
int[] predefinedNumbers = { 1, 2, 3, 4, 5 };

Accés als Elements d'un Array

Pots accedir als elements d'un array utilitzant el seu índex.

int firstNumber = numbers[0]; // Accés al primer element
numbers[1] = 10; // Assignació del valor 10 al segon element

Iteració sobre Arrays

Pots utilitzar bucles per iterar sobre arrays.

// Utilitzant un bucle for
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}

// Utilitzant un bucle foreach
foreach (int number in predefinedNumbers)
{
    Console.WriteLine(number);
}

Conceptes Avançats d'Arrays

Arrays Multidimensionals

C# suporta arrays multidimensionals, com ara arrays 2D.

// Declaració d'un array 2D
int[,] matrix = new int[3, 3];

// Inicialització d'un array 2D
int[,] predefinedMatrix = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

// Accés als elements d'un array 2D
int element = matrix[1, 2]; // Accés a l'element de la fila 1, columna 2

Arrays Dentats

Els arrays dentats són arrays d'arrays, on cada "array intern" pot tenir una longitud diferent.

// Declaració d'un array dentat
int[][] jaggedArray = new int[3][];

// Inicialització de l'array dentat
jaggedArray[0] = new int[] { 1, 2 };
jaggedArray[1] = new int[] { 3, 4, 5 };
jaggedArray[2] = new int[] { 6, 7, 8, 9 };

// Accés als elements d'un array dentat
int jaggedElement = jaggedArray[1][2]; // Accés al tercer element del segon array

Mètodes d'Array

C# proporciona diversos mètodes integrats per treballar amb arrays, com Array.Sort, Array.Reverse i Array.Copy.

int[] numbers = { 5, 3, 8, 1, 2 };

// Ordenació de l'array
Array.Sort(numbers);

// Inversió de l'array
Array.Reverse(numbers);

// Còpia de l'array
int[] copiedNumbers = new int[numbers.Length];
Array.Copy(numbers, copiedNumbers, numbers.Length);

Conclusió

Els arrays són una part poderosa i essencial de la programació en C#. Et permeten emmagatzemar i gestionar col·leccions de dades de manera eficient. Entendre com crear, accedir i manipular arrays és crucial per a qualsevol programador de C#. A mesura que avancis, trobaràs estructures d'array més complexes com arrays multidimensionals i dentats, que ofereixen encara més flexibilitat per gestionar dades.
