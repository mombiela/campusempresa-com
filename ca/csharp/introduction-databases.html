Visió General

Les bases de dades són essencials per emmagatzemar, recuperar i gestionar dades en aplicacions. Aquesta secció t'introduirà a les bases de dades, com interactuar amb elles utilitzant C#, i els conceptes bàsics de SQL (Structured Query Language).

Conceptes Clau

  • Base de Dades: Una col·lecció organitzada de dades, generalment emmagatzemades i accedides electrònicament des d'un sistema informàtic.
  • SQL: Un llenguatge estàndard per accedir i manipular bases de dades.
  • RDBMS: Sistema de Gestió de Bases de Dades Relacionals, un tipus de sistema de gestió de bases de dades que emmagatzema dades en taules i permet relacions entre taules.
  • Operacions CRUD: Les quatre operacions bàsiques per gestionar dades - Crear, Llegir, Actualitzar, Esborrar.

Configuració

Abans d'entrar en el codi, assegura't de tenir el següent:

  • SQL Server: Instal·la SQL Server o utilitza un servei basat en el núvol com Azure SQL Database.
  • SQL Server Management Studio (SSMS): Una eina per gestionar bases de dades SQL Server.
  • Visual Studio: IDE per escriure codi C#.

Connexió a una Base de Dades

Per interactuar amb una base de dades en C#, necessites establir una connexió utilitzant la classe SqlConnection.

Exemple

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            try
            {
                connection.Open();
                Console.WriteLine("Connexió exitosa!");
            }
            catch (Exception ex)
            {
                Console.WriteLine("Connexió fallida: " + ex.Message);
            }
        }
    }
}

Explicació:

  • connectionString: Conté els detalls necessaris per connectar-se a la base de dades.
  • SqlConnection: Representa una connexió a una base de dades SQL Server.
  • Open(): Obre la connexió a la base de dades.
  • try-catch: Gestiona qualsevol excepció que es produeixi durant l'intent de connexió.

Operacions CRUD

Crear

Per inserir dades en una base de dades, utilitza la instrucció SQL INSERT INTO.

Exemple

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "INSERT INTO Users (Name, Age) VALUES (@Name, @Age)";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 30);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Files afectades: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicació:

  • SqlCommand: Representa una instrucció SQL per executar contra una base de dades SQL Server.
  • Paràmetres: Utilitzats per prevenir la injecció SQL mitjançant la parametrització de la consulta.

Llegir

Per recuperar dades, utilitza la instrucció SQL SELECT.

Exemple

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "SELECT Name, Age FROM Users";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);

            try
            {
                connection.Open();
                SqlDataReader reader = command.ExecuteReader();
                while (reader.Read())
                {
                    Console.WriteLine($"Nom: {reader["Name"]}, Edat: {reader["Age"]}");
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicació:

  • SqlDataReader: Proporciona una manera de llegir un flux només endavant de files des d'una base de dades SQL Server.

Actualitzar

Per modificar dades existents, utilitza la instrucció SQL UPDATE.

Exemple

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "UPDATE Users SET Age = @Age WHERE Name = @Name";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 31);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Files afectades: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicació:

  • UPDATE: Modifica registres existents en una taula.

Esborrar

Per eliminar dades, utilitza la instrucció SQL DELETE.

Exemple

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "DELETE FROM Users WHERE Name = @Name";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Files afectades: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explicació:

  • DELETE: Elimina registres d'una taula.
