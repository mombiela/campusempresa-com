Connectar-se a una base de dades és una habilitat fonamental per a qualsevol desenvolupador de C#. Aquest tema et guiarà a través del procés des de nivells principiants fins a avançats, cobrint els conceptes bàsics de les connexions a bases de dades, l'execució de consultes i la gestió eficient de les dades.

Introducció a les Bases de Dades

Abans d'entrar en el codi, és essencial entendre què és una base de dades i per què s'utilitza.

  • Base de dades: Una col·lecció estructurada de dades que es poden accedir, gestionar i actualitzar fàcilment.
  • SQL (Structured Query Language): Un llenguatge estàndard per gestionar i manipular bases de dades.

Configuració del Teu Entorn

Per connectar-te a una base de dades en C#, necessitaràs:

  1. Un Sistema de Gestió de Bases de Dades (DBMS): Exemples inclouen SQL Server, MySQL, PostgreSQL, etc.
  2. .NET Framework o .NET Core: Assegura't de tenir la versió més recent instal·lada.
  3. Llibreria de Connexió a Bases de Dades: Per a SQL Server, pots utilitzar System.Data.SqlClient.

Connexió Bàsica a una Base de Dades SQL Server

Pas 1: Instal·lar els Paquets Necessaris

Per a .NET Core, pots instal·lar el paquet System.Data.SqlClient via NuGet:

dotnet add package System.Data.SqlClient

Pas 2: Establir una Connexió

Aquí tens un exemple senzill de com connectar-te a una base de dades SQL Server:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            try
            {
                connection.Open();
                Console.WriteLine("Connexió exitosa!");
            }
            catch (Exception ex)
            {
                Console.WriteLine($"S'ha produït un error: {ex.Message}");
            }
        }
    }
}
  • Explicació:
    • SqlConnection: Representa una connexió a una base de dades SQL Server.
    • connectionString: Conté la informació necessària per connectar-se a la base de dades.
    • connection.Open(): Obre la connexió a la base de dades.
    • using statement: Assegura que la connexió es tanqui i es destrueixi correctament.

Execució de Comandes SQL

Pas 1: Executar una Consulta Senzilla

Per executar una consulta, pots utilitzar la classe SqlCommand:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT COUNT(*) FROM your_table";
            SqlCommand command = new SqlCommand(query, connection);
            
            int count = (int)command.ExecuteScalar();
            Console.WriteLine($"Nombre de files: {count}");
        }
    }
}
  • Explicació:
    • SqlCommand: Representa una instrucció SQL per executar contra una base de dades SQL Server.
    • ExecuteScalar(): Executa la consulta i retorna la primera columna de la primera fila en el conjunt de resultats.

Pas 2: Llegir Dades amb `SqlDataReader`

Per a consultes més complexes, pots utilitzar SqlDataReader per llegir dades fila per fila:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT * FROM your_table";
            SqlCommand command = new SqlCommand(query, connection);
            
            using (SqlDataReader reader = command.ExecuteReader())
            {
                while (reader.Read())
                {
                    Console.WriteLine($"ID: {reader["ID"]}, Nom: {reader["Name"]}");
                }
            }
        }
    }
}
  • Explicació:
    • ExecuteReader(): Executa la consulta i retorna un SqlDataReader per llegir els resultats.
    • reader.Read(): Avança al següent registre en el conjunt de resultats.

Temes Avançats

Utilitzar Paràmetres per Prevenir la Injecció SQL

Per prevenir atacs de injecció SQL, sempre utilitza paràmetres en les teves consultes:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT * FROM your_table WHERE Name = @name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@name", "John Doe");
            
            using (SqlDataReader reader = command.ExecuteReader())
            {
                while (reader.Read())
                {
                    Console.WriteLine($"ID: {reader["ID"]}, Nom: {reader["Name"]}");
                }
            }
        }
    }
}
  • Explicació:
    • command.Parameters.AddWithValue("@name", "John Doe"): Afegeix un paràmetre a la consulta SQL per prevenir la injecció SQL.

Gestió de Transaccions

Les transaccions asseguren que una sèrie d'operacions o bé totes tinguin èxit o bé totes fallin, mantenint la integritat de les dades:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            SqlTransaction transaction = connection.BeginTransaction();
            
            try
            {
                SqlCommand command1 = new SqlCommand("INSERT INTO your_table (Name) VALUES ('John Doe')", connection, transaction);
                command1.ExecuteNonQuery();
                
                SqlCommand command2 = new SqlCommand("INSERT INTO your_table (Name) VALUES ('Jane Doe')", connection, transaction);
                command2.ExecuteNonQuery();
                
                transaction.Commit();
                Console.WriteLine("Transacció confirmada.");
            }
            catch (Exception ex)
            {
                transaction.Rollback();
                Console.WriteLine($"Transacció revertida: {ex.Message}");
            }
        }
    }
}
  • Explicació:
    • BeginTransaction(): Comença una nova transacció.
    • transaction.Commit(): Confirma la transacció.
    • transaction.Rollback(): Reverteix la transacció en cas d'error.

Conclusió

Connectar-se a una base de dades en C# implica entendre els conceptes bàsics dels sistemes de gestió de bases de dades, configurar el teu entorn i utilitzar les classes i mètodes apropiats per executar consultes i gestionar dades. Seguint les millors pràctiques com utilitzar paràmetres per prevenir la injecció SQL i gestionar transaccions correctament, pots assegurar-te que les teves aplicacions siguin segures i fiables.
