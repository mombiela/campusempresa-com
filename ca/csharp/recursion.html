La recursió és un concepte fonamental en la programació on una funció es crida a si mateixa per resoldre instàncies més petites del mateix problema. Aquest tema cobrirà els conceptes bàsics de la recursió, les seves aplicacions i tècniques avançades en C#.

Introducció a la Recursió

  • Definició: La recursió és un mètode on la solució a un problema depèn de les solucions a instàncies més petites del mateix problema.
  • Cas Base: La condició sota la qual la recursió acaba.
  • Cas Recursiu: La part de la funció on la funció es crida a si mateixa.

Exemple: Càlcul del Factorial

El factorial d'un nombre n (denotat com n!) és el producte de tots els enters positius menors o iguals a n.

Funció Factorial en C#

using System;

class Program
{
    static void Main()
    {
        int number = 5;
        Console.WriteLine($"Factorial de {number} és {Factorial(number)}");
    }

    static int Factorial(int n)
    {
        if (n <= 1) // Cas base
            return 1;
        else
            return n * Factorial(n - 1); // Cas recursiu
    }
}
  • Explicació:
    • La funció Factorial es crida a si mateixa amb n-1 fins que n és 1.
    • El cas base assegura que la recursió s'atura quan n és 1.

Comprensió de les Funcions Recursives

Conceptes Clau

  • Desbordament de Pila: Es produeix quan la recursió és massa profunda, superant la mida de la pila.
  • Recursió de Cua: Una forma especial de recursió on la crida recursiva és l'última operació de la funció.

Exemple: Seqüència de Fibonacci

La seqüència de Fibonacci és una sèrie de nombres on cada nombre és la suma dels dos precedents.

Funció Fibonacci en C#

using System;

class Program
{
    static void Main()
    {
        int number = 10;
        for (int i = 0; i < number; i++)
        {
            Console.Write(Fibonacci(i) + " ");
        }
    }

    static int Fibonacci(int n)
    {
        if (n <= 1) // Cas base
            return n;
        else
            return Fibonacci(n - 1) + Fibonacci(n - 2); // Cas recursiu
    }
}
  • Explicació:
    • La funció Fibonacci es crida a si mateixa dues vegades amb n-1 i n-2.
    • El cas base assegura que la recursió s'atura quan n és 0 o 1.

Tècniques Avançades de Recursió

Recursió de Cua

La recursió de cua pot ser optimitzada pel compilador per evitar el desbordament de pila.

Factorial Recursiu de Cua en C#

using System;

class Program
{
    static void Main()
    {
        int number = 5;
        Console.WriteLine($"Factorial de {number} és {TailRecursiveFactorial(number, 1)}");
    }

    static int TailRecursiveFactorial(int n, int accumulator)
    {
        if (n <= 1) // Cas base
            return accumulator;
        else
            return TailRecursiveFactorial(n - 1, n * accumulator); // Cas recursiu de cua
    }
}
  • Explicació:
    • La funció TailRecursiveFactorial utilitza un acumulador per emmagatzemar els resultats intermedis.
    • La crida recursiva és l'última operació, fent-la recursiva de cua.

Recursió vs. Iteració

| Característica | Recursió | Iteració | |-----------------|------------------------------------|------------------------------------| | Definició | La funció es crida a si mateixa | Construccions de bucle (for, while)| | Ús de Memòria | Utilitza memòria de pila | Utilitza memòria de munt | | Rendiment | Pot ser més lenta per la sobrecàrrega | Generalment més ràpida | | Llegibilitat | Sovint més llegible i concisa | Pot ser més complexa | | Cas Base | Requereix un cas base per acabar | Acaba amb la condició del bucle |

Aplicacions Pràctiques de la Recursió

  • Recorreguts d'Arbres: La recursió s'utilitza comunament en el recorregut d'estructures d'arbres (per exemple, arbres binaris).
  • Algoritmes de Dividir i Vèncer: Algoritmes com QuickSort i MergeSort utilitzen recursió.
  • Programació Dinàmica: Alguns problemes de programació dinàmica es resolen utilitzant recursió amb memoització.

Exemple: Recorregut d'Arbre Binari

Recorregut In-Order en C#

using System;

class Node
{
    public int Data;
    public Node Left, Right;

    public Node(int item)
    {
        Data = item;
        Left = Right = null;
    }
}

class BinaryTree
{
    Node root;

    void InOrderTraversal(Node node)
    {
        if (node == null)
            return;

        InOrderTraversal(node.Left); // Recorre el subarbre esquerre
        Console.Write(node.Data + " "); // Visita el node
        InOrderTraversal(node.Right); // Recorre el subarbre dret
    }

    public static void Main()
    {
        BinaryTree tree = new BinaryTree();
        tree.root = new Node(1);
        tree.root.Left = new Node(2);
        tree.root.Right = new Node(3);
        tree.root.Left.Left = new Node(4);
        tree.root.Left.Right = new Node(5);

        Console.WriteLine("El recorregut in-order de l'arbre binari és:");
        tree.InOrderTraversal(tree.root);
    }
}
  • Explicació:
    • La funció InOrderTraversal recorre recursivament el subarbre esquerre, visita el node, i després recorre el subarbre dret.

Conclusió

La recursió és una eina poderosa en la programació en C# que permet solucions elegants a problemes complexos. Comprenent els conceptes bàsics, reconeixent la importància dels casos base, i explorant tècniques avançades com la recursió de cua, els programadors poden utilitzar efectivament la recursió en diverses aplicacions. Recorda sempre considerar el potencial de desbordament de pila i comparar la recursió amb la iteració per escollir l'enfocament més adequat per al teu problema específic.
