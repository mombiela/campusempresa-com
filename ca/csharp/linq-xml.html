Introducció a LINQ to XML

LINQ to XML és una manera potent i flexible de treballar amb dades XML en C#. Et permet consultar, crear, modificar i eliminar dades XML utilitzant la sintaxi LINQ (Language Integrated Query). Aquesta secció et guiarà des dels conceptes bàsics fins als avançats de LINQ to XML.

Començant amb LINQ to XML

Què és XML?

  • XML significa eXtensible Markup Language.
  • És un llenguatge de marques molt semblant a HTML però està dissenyat per emmagatzemar i transportar dades.
  • XML és llegible tant per humans com per màquines.

Estructura Bàsica d'XML

<bookstore>
  <book>
    <title>Programming C#</title>
    <author>John Doe</author>
    <price>29.99</price>
  </book>
</bookstore>

Configurant el Teu Entorn

  • Assegura't de tenir .NET instal·lat.
  • Utilitza Visual Studio o qualsevol altre IDE de C#.

Creant Documents XML

Utilitzant XElement i XDocument

  • XElement representa un element XML.
  • XDocument representa un document XML sencer.

Exemple: Creant un Document XML Simple

using System.Xml.Linq;

XElement book = new XElement("book",
    new XElement("title", "Programming C#"),
    new XElement("author", "John Doe"),
    new XElement("price", "29.99")
);

XDocument bookstore = new XDocument(
    new XElement("bookstore", book)
);

Console.WriteLine(bookstore);
  • XElement s'utilitza per crear elements individuals.
  • XDocument s'utilitza per crear el document arrel.

Consultar Dades XML

Consultes Bàsiques LINQ

  • Utilitza consultes LINQ per recuperar dades de documents XML.

Exemple: Consultar Dades XML

XDocument doc = XDocument.Load("books.xml");

var books = from book in doc.Descendants("book")
            where (double)book.Element("price") > 20.00
            select book;

foreach (var book in books)
{
    Console.WriteLine(book);
}
  • XDocument.Load carrega un document XML des d'un fitxer.
  • El mètode Descendants recupera tots els elements descendents amb el nom especificat.
  • La consulta LINQ filtra llibres amb un preu superior a 20.00.

Modificant Dades XML

Afegir, Actualitzar i Eliminar Elements

  • Modifica dades XML afegint, actualitzant o eliminant elements.

Exemple: Afegir un Element

XDocument doc = XDocument.Load("books.xml");

XElement newBook = new XElement("book",
    new XElement("title", "Learning LINQ"),
    new XElement("author", "Jane Doe"),
    new XElement("price", "39.99")
);

doc.Element("bookstore").Add(newBook);
doc.Save("books.xml");
  • El mètode Add afegeix un nou element al document XML.
  • El mètode Save guarda els canvis al fitxer XML.

Exemple: Actualitzar un Element

XDocument doc = XDocument.Load("books.xml");

var book = doc.Descendants("book")
              .Where(b => b.Element("title").Value == "Programming C#")
              .FirstOrDefault();

if (book != null)
{
    book.Element("price").Value = "24.99";
    doc.Save("books.xml");
}
  • El mètode Where filtra elements basats en una condició.
  • FirstOrDefault recupera el primer element que compleix la condició o null si no hi ha elements que compleixin.

Exemple: Eliminar un Element

XDocument doc = XDocument.Load("books.xml");

var book = doc.Descendants("book")
              .Where(b => b.Element("title").Value == "Programming C#")
              .FirstOrDefault();

if (book != null)
{
    book.Remove();
    doc.Save("books.xml");
}
  • El mètode Remove elimina l'element especificat del document XML.

Tècniques Avançades de LINQ to XML

Espais de Noms en XML

  • Els espais de noms XML s'utilitzen per evitar conflictes de noms d'elements.

Exemple: Treballant amb Espais de Noms

XNamespace ns = "http://www.example.com/books";
XDocument doc = new XDocument(
    new XElement(ns + "bookstore",
        new XElement(ns + "book",
            new XElement(ns + "title", "Programming C#"),
            new XElement(ns + "author", "John Doe"),
            new XElement(ns + "price", "29.99")
        )
    )
);

Console.WriteLine(doc);
  • XNamespace representa un espai de noms XML.
  • Els elements es creen amb el prefix de l'espai de noms.

Transformant Dades XML

  • Utilitza LINQ to XML per transformar dades XML en diferents formats.

Exemple: Transformant XML a un Altre Format XML

XDocument doc = XDocument.Load("books.xml");

XDocument transformedDoc = new XDocument(
    new XElement("library",
        from book in doc.Descendants("book")
        select new XElement("item",
            new XElement("name", book.Element("title").Value),
            new XElement("writer", book.Element("author").Value),
            new XElement("cost", book.Element("price").Value)
        )
    )
);

Console.WriteLine(transformedDoc);
  • Transforma dades XML projectant-les en un nou format utilitzant consultes LINQ.

Conclusió

LINQ to XML proporciona una manera robusta i eficient de treballar amb dades XML en C#. Des de crear i consultar documents XML fins a modificar-los i transformar-los, LINQ to XML aprofita la potència de LINQ per fer la manipulació d'XML senzilla i intuïtiva. Dominant aquestes tècniques, podràs manejar estructures de dades XML complexes amb facilitat i integrar el processament XML sense problemes en les teves aplicacions C#.
