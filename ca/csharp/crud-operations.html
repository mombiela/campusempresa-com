Operacions CRUD en C#
Entity Framework ►

CRUD significa Crear, Llegir, Actualitzar i Esborrar. Aquestes són les quatre operacions bàsiques per gestionar dades en una base de dades. En aquesta secció, explorarem com realitzar operacions CRUD en C# utilitzant diversos enfocaments, incloent ADO.NET i Entity Framework.

Introducció a les Operacions CRUD

  • Crear: Afegir nous registres a la base de dades.
  • Llegir: Recuperar registres existents de la base de dades.
  • Actualitzar: Modificar registres existents a la base de dades.
  • Esborrar: Eliminar registres de la base de dades.

Configuració de l'Entorn

Abans de submergir-nos en les operacions CRUD, assegura't de tenir la següent configuració:

  • Visual Studio instal·lat.
  • SQL Server o qualsevol altre sistema de gestió de bases de dades.
  • Una comprensió bàsica de la sintaxi de C#.

Utilitzant ADO.NET per a les Operacions CRUD

Operació de Creació

Per inserir un nou registre a la base de dades:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "INSERT INTO Students (Name, Age) VALUES (@Name, @Age)";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 21);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Comprovar si la inserció ha estat exitosa
            if (result > 0)
                Console.WriteLine("Registre inserit amb èxit.");
            else
                Console.WriteLine("Error en inserir el registre.");
        }
    }
}

Operació de Lectura

Per recuperar registres de la base de dades:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "SELECT * FROM Students";
            SqlCommand command = new SqlCommand(query, connection);

            connection.Open();
            SqlDataReader reader = command.ExecuteReader();

            while (reader.Read())
            {
                Console.WriteLine($"ID: {reader["ID"]}, Name: {reader["Name"]}, Age: {reader["Age"]}");
            }
        }
    }
}

Operació d'Actualització

Per actualitzar un registre existent a la base de dades:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "UPDATE Students SET Age = @Age WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 22);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Comprovar si l'actualització ha estat exitosa
            if (result > 0)
                Console.WriteLine("Registre actualitzat amb èxit.");
            else
                Console.WriteLine("Error en actualitzar el registre.");
        }
    }
}

Operació d'Esborrat

Per esborrar un registre de la base de dades:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "DELETE FROM Students WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Comprovar si l'esborrat ha estat exitós
            if (result > 0)
                Console.WriteLine("Registre esborrat amb èxit.");
            else
                Console.WriteLine("Error en esborrar el registre.");
        }
    }
}

Utilitzant Entity Framework per a les Operacions CRUD

Entity Framework (EF) és un ORM (Object-Relational Mapper) que simplifica l'accés a dades en aplicacions .NET.

Configuració d'Entity Framework

  1. Instal·la Entity Framework a través del Gestor de Paquets NuGet:

    Install-Package EntityFramework

  2. Crea una classe model i una classe de context:

using System.Data.Entity;

public class Student
{
    public int ID { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
}

public class SchoolContext : DbContext
{
    public DbSet<Student> Students { get; set; }
}

Operació de Creació

Per inserir un nou registre utilitzant Entity Framework:

using System;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = new Student { Name = "Jane Doe", Age = 20 };
            context.Students.Add(student);
            context.SaveChanges();
            Console.WriteLine("Registre inserit amb èxit.");
        }
    }
}

Operació de Lectura

Per recuperar registres utilitzant Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var students = context.Students.ToList();
            foreach (var student in students)
            {
                Console.WriteLine($"ID: {student.ID}, Name: {student.Name}, Age: {student.Age}");
            }
        }
    }
}

Operació d'Actualització

Per actualitzar un registre existent utilitzant Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                student.Age = 21;
                context.SaveChanges();
                Console.WriteLine("Registre actualitzat amb èxit.");
            }
        }
    }
}

Operació d'Esborrat

Per esborrar un registre utilitzant Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                context.Students.Remove(student);
                context.SaveChanges();
                Console.WriteLine("Registre esborrat amb èxit.");
            }
        }
    }
}

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les operacions CRUD en C# utilitzant tant ADO.NET com Entity Framework. Hem après com crear, llegir, actualitzar i esborrar registres en una base de dades. Entendre aquestes operacions és fonamental per a qualsevol aplicació que interactuï amb una base de dades. A mesura que progressis, pots explorar temes més avançats com la gestió de transaccions, procediments emmagatzemats i l'optimització del rendiment.
