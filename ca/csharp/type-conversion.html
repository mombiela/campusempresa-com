La conversió de tipus és un concepte essencial en la programació en C#, permetent als desenvolupadors convertir dades d'un tipus a un altre. Aquest tema cobreix els conceptes bàsics de la conversió de tipus, incloent-hi conversions implícites i explícites, i aprofundeix en conceptes més avançats com les conversions personalitzades i l'ús de mètodes de conversió.

Conversions Implícites

Les conversions implícites són gestionades automàticament pel compilador de C#. Aquestes conversions són segures i no resulten en pèrdua de dades.

  • Conversions Numèriques: Convertir tipus numèrics més petits a tipus numèrics més grans.
  • Conversions de Referència: Convertir classes derivades a classes base.

Exemples de Conversions Implícites

int intValue = 123;
double doubleValue = intValue; // Conversió implícita de int a double

DerivedClass derived = new DerivedClass();
BaseClass baseClass = derived; // Conversió implícita de classe derivada a classe base

Conversions Explícites

Les conversions explícites, o càstings, requereixen que el programador especifiqui la conversió. Aquestes conversions poden potencialment resultar en pèrdua de dades o excepcions en temps d'execució.

  • Conversions Numèriques: Convertir tipus numèrics més grans a tipus numèrics més petits.
  • Conversions de Referència: Convertir classes base a classes derivades (requereix un càsting).

Exemples de Conversions Explícites

double doubleValue = 123.45;
int intValue = (int)doubleValue; // Conversió explícita de double a int

BaseClass baseClass = new DerivedClass();
DerivedClass derived = (DerivedClass)baseClass; // Conversió explícita de classe base a classe derivada

Mètodes de Conversió

C# proporciona diversos mètodes per convertir entre tipus, que formen part de la classe System.Convert.

  • ToInt32(): Converteix un valor a un enter de 32 bits.
  • ToDouble(): Converteix un valor a un nombre de punt flotant de doble precisió.
  • ToString(): Converteix un valor a la seva representació en cadena.

Exemples de Mètodes de Conversió

string stringValue = "123";
int intValue = Convert.ToInt32(stringValue); // Converteix cadena a int

double doubleValue = Convert.ToDouble(stringValue); // Converteix cadena a double

int number = 456;
string numberString = Convert.ToString(number); // Converteix int a cadena

Conversions Personalitzades

Les conversions personalitzades es poden definir sobrecarregant els operadors implicit i explicit en una classe. Això permet un major control sobre com es converteixen els objectes d'aquesta classe cap a i des d'altres tipus.

Exemple de Conversions Personalitzades

public class CustomType
{
    public int Value { get; set; }

    // Conversió implícita de CustomType a int
    public static implicit operator int(CustomType custom)
    {
        return custom.Value;
    }

    // Conversió explícita de int a CustomType
    public static explicit operator CustomType(int value)
    {
        return new CustomType { Value = value };
    }
}

// Ús
CustomType custom = new CustomType { Value = 123 };
int intValue = custom; // Conversió implícita
CustomType newCustom = (CustomType)intValue; // Conversió explícita

Resum

La conversió de tipus en C# és una característica poderosa que permet una manipulació flexible de dades. Entendre les diferències entre conversions implícites i explícites, així com com utilitzar mètodes de conversió i definir conversions personalitzades, és crucial per escriure codi robust i eficient en C#. Sempre cal tenir en compte la possible pèrdua de dades i les excepcions en temps d'execució quan es realitzen conversions explícites.
