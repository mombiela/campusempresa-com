El Projecte Sobre nosaltres Contribuir Donacions Llicència

Els diccionaris són una estructura de dades fonamental en C# que et permeten emmagatzemar parells clau-valor. Són molt eficients per a cerques, insercions i eliminacions. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat dels diccionaris en C#.

Introducció als Diccionaris

  • Definició: Un diccionari és una col·lecció de parells clau-valor on cada clau és única.
  • Espai de noms: System.Collections.Generic
  • Classe: Dictionary<TKey, TValue>

Crear un Diccionari

Per crear un diccionari, has d'especificar els tipus per a les claus i els valors.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Crear un diccionari amb claus de tipus string i valors de tipus int
        Dictionary<string, int> ages = new Dictionary<string, int>();
    }
}

Afegir Elements

Pots afegir elements a un diccionari utilitzant el mètode Add o l'indexador.

ages.Add("Alice", 30);
ages["Bob"] = 25;

Accedir als Elements

Accedeix als elements per la seva clau utilitzant l'indexador.

int aliceAge = ages["Alice"];
Console.WriteLine($"L'edat de l'Alice és {aliceAge}");

Eliminar Elements

Elimina elements utilitzant el mètode Remove.

ages.Remove("Alice");

Comprovar Claus i Valors

Utilitza ContainsKey i ContainsValue per comprovar l'existència de claus i valors.

bool hasAlice = ages.ContainsKey("Alice");
bool hasAge30 = ages.ContainsValue(30);

Iterar sobre un Diccionari

Pots iterar sobre un diccionari utilitzant un bucle foreach.

foreach (KeyValuePair<string, int> kvp in ages)
{
    Console.WriteLine($"Clau: {kvp.Key}, Valor: {kvp.Value}");
}

Operacions Avançades amb Diccionaris

Inicialitzar amb Valors

Pots inicialitzar un diccionari amb valors utilitzant la sintaxi d'inicialitzador de col·leccions.

Dictionary<string, int> ages = new Dictionary<string, int>
{
    { "Alice", 30 },
    { "Bob", 25 },
    { "Charlie", 35 }
};

Actualitzar Valors

Pots actualitzar valors assignant un nou valor a una clau existent.

ages["Bob"] = 26;

Fusionar Diccionaris

Pots fusionar dos diccionaris utilitzant el mètode Union de LINQ.

using System.Linq;

Dictionary<string, int> moreAges = new Dictionary<string, int>
{
    { "David", 40 },
    { "Eve", 22 }
};

var mergedAges = ages.Union(moreAges).ToDictionary(kvp => kvp.Key, kvp => kvp.Value);

Gestionar Claus Inexistents

Utilitza TryGetValue per gestionar de manera segura les claus inexistents.

if (ages.TryGetValue("Alice", out int aliceAge))
{
    Console.WriteLine($"L'edat de l'Alice és {aliceAge}");
}
else
{
    Console.WriteLine("L'Alice no és al diccionari.");
}

Consideracions de Rendiment

  • Complexitat Temporal: Mitjana O(1) per a cerques, insercions i eliminacions.
  • Ús de Memòria: Pot ser més alt en comparació amb altres col·leccions a causa de la necessitat d'emmagatzemar claus i valors.

Ús Comú

  • Configuració de Paràmetres: Emmagatzemar configuracions d'aplicacions on les claus són noms de paràmetres i els valors són els valors dels paràmetres.
  • Memòria Cau: Emmagatzemar dades accedides freqüentment per a una recuperació ràpida.
  • Comptar Ocorrències: Comptar el nombre d'ocurrències d'elements en una col·lecció.

Conclusió

Els diccionaris són una estructura de dades versàtil i eficient en C#. Proporcionen cerques, insercions i eliminacions ràpides, fent-los ideals per a una àmplia gamma d'aplicacions. Entenent com crear, manipular i iterar sobre diccionaris, pots aprofitar el seu poder en els teus programes en C#.
