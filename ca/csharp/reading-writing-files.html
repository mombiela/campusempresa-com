Introducció

Llegir i escriure fitxers és una habilitat fonamental en la programació. En C#, l'espai de noms System.IO proporciona classes i mètodes per gestionar operacions amb fitxers. Aquest tema cobrirà els conceptes bàsics de la lectura i escriptura de fitxers, així com tècniques més avançades.

Conceptes Bàsics de la Gestió de Fitxers

Conceptes Clau

  • Fitxer: Representa un fitxer en el sistema de fitxers.
  • Flux: Una seqüència de bytes que es poden llegir o escriure.
  • FileStream: Proporciona un flux per a operacions amb fitxers.
  • StreamReader: Llegeix caràcters d'un flux de bytes.
  • StreamWriter: Escriu caràcters en un flux de bytes.

Llegir Fitxers

Utilitzant File.ReadAllText

La manera més senzilla de llegir un fitxer és utilitzant el mètode File.ReadAllText, que llegeix tot el text d'un fitxer en una sola cadena.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = File.ReadAllText(path);
        Console.WriteLine(content);
    }
}

Utilitzant StreamReader

Per a més control, pots utilitzar la classe StreamReader.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamReader sr = new StreamReader(path))
        {
            string line;
            while ((line = sr.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
    }
}

Escriure Fitxers

Utilitzant File.WriteAllText

La manera més senzilla d'escriure en un fitxer és utilitzant el mètode File.WriteAllText, que escriu una cadena en un fitxer, creant el fitxer si no existeix.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        File.WriteAllText(path, content);
    }
}

Utilitzant StreamWriter

Per a més control, pots utilitzar la classe StreamWriter.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamWriter sw = new StreamWriter(path))
        {
            sw.WriteLine("Hello, World!");
            sw.WriteLine("This is a new line.");
        }
    }
}

Gestió Avançada de Fitxers

Conceptes Clau

  • FileMode: Especifica com el sistema operatiu ha d'obrir un fitxer.
  • FileAccess: Defineix el nivell d'accés permès a un fitxer.
  • FileShare: Determina com el fitxer serà compartit per processos.

Utilitzant `FileStream`

Per a escenaris avançats, pots utilitzar la classe FileStream, que proporciona més control sobre les operacions amb fitxers.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (FileStream fs = new FileStream(path, FileMode.OpenOrCreate, FileAccess.ReadWrite))
        {
            byte[] info = new System.Text.UTF8Encoding(true).GetBytes("Hello, World!");
            fs.Write(info, 0, info.Length);
        }
    }
}

Operacions Asíncrones amb Fitxers

C# també suporta operacions asíncrones amb fitxers, que poden millorar el rendiment en aplicacions amb moltes operacions d'E/S.

Lectura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = await File.ReadAllTextAsync(path);
        Console.WriteLine(content);
    }
}

Escriptura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        await File.WriteAllTextAsync(path, content);
    }
}

Resum

En aquest tema, hem cobert els conceptes bàsics i tècniques avançades per llegir i escriure fitxers en C#. Hem explorat mètodes senzills com File.ReadAllText i File.WriteAllText, així com classes més avançades com StreamReader, StreamWriter i FileStream. A més, hem tocat les operacions asíncrones amb fitxers per gestionar tasques d'E/S de manera eficient. Dominar aquestes tècniques et permetrà gestionar operacions amb fitxers de manera efectiva en les teves aplicacions C#.
