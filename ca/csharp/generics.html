Els genèrics són una característica poderosa en C# que et permet definir classes, mètodes i estructures de dades amb un marcador de posició per al tipus de dades que emmagatzemen o utilitzen. Això permet la reutilització de codi, la seguretat de tipus i l'optimització del rendiment.

Introducció als Genèrics

Els genèrics et permeten escriure una classe o mètode que pot treballar amb qualsevol tipus de dades. En lloc d'especificar un tipus de dades específic, utilitzes un marcador de posició, que es reemplaça amb el tipus de dades real quan la classe o el mètode s'instancia o es crida.

Conceptes Clau

  • Paràmetre de Tipus: Un marcador de posició per a un tipus de dades, especificat utilitzant claudàtors angulars (<T>).
  • Classe Genèrica: Una classe que pot operar amb qualsevol tipus de dades.
  • Mètode Genèric: Un mètode que pot operar amb qualsevol tipus de dades.
  • Seguretat de Tipus: Assegura que el tipus de dades sigui consistent, reduint errors en temps d'execució.
  • Reutilització de Codi: Permet que el mateix codi funcioni amb diferents tipus de dades.

Classes Genèriques

Una classe genèrica es defineix amb un paràmetre de tipus. Aquí tens un exemple d'una classe genèrica simple:

public class GenericClass<T>
{
    private T data;

    public GenericClass(T value)
    {
        data = value;
    }

    public T GetData()
    {
        return data;
    }
}

Exemple d'Ús

GenericClass<int> intInstance = new GenericClass<int>(10);
Console.WriteLine(intInstance.GetData()); // Sortida: 10

GenericClass<string> stringInstance = new GenericClass<string>("Hola");
Console.WriteLine(stringInstance.GetData()); // Sortida: Hola

Mètodes Genèrics

Un mètode genèric es defineix amb un paràmetre de tipus. Aquí tens un exemple:

public class GenericMethods
{
    public void PrintData<T>(T data)
    {
        Console.WriteLine(data);
    }
}

Exemple d'Ús

GenericMethods gm = new GenericMethods();
gm.PrintData<int>(100); // Sortida: 100
gm.PrintData<string>("Hola Món"); // Sortida: Hola Món

Restriccions en els Paràmetres de Tipus

Pots imposar restriccions en els paràmetres de tipus per restringir els tipus que es poden utilitzar. Aquí tens algunes restriccions comunes:

  • where T : struct: T ha de ser un tipus de valor.
  • where T : class: T ha de ser un tipus de referència.
  • where T : new(): T ha de tenir un constructor sense paràmetres.
  • where T : BaseClass: T ha d'heretar de BaseClass.
  • where T : Interface: T ha d'implementar la interfície especificada.

Exemple amb Restriccions

public class GenericClassWithConstraints<T> where T : class, new()
{
    private T data;

    public GenericClassWithConstraints()
    {
        data = new T();
    }

    public T GetData()
    {
        return data;
    }
}

Col·leccions Genèriques

C# proporciona diverses col·leccions genèriques en l'espai de noms System.Collections.Generic, com ara List<T>, Dictionary<TKey, TValue>, i Queue<T>.

Exemple amb List

List<int> numbers = new List<int>();
numbers.Add(1);
numbers.Add(2);
numbers.Add(3);

foreach (int number in numbers)
{
    Console.WriteLine(number); // Sortida: 1 2 3
}

Exemple amb Dictionary

Dictionary<int, string> dict = new Dictionary<int, string>();
dict.Add(1, "Un");
dict.Add(2, "Dos");

foreach (var kvp in dict)
{
    Console.WriteLine($"Clau: {kvp.Key}, Valor: {kvp.Value}");
    // Sortida: Clau: 1, Valor: Un
    //         Clau: 2, Valor: Dos
}

Genèrics Avançats

Covariància i Contravariància

La covariància i la contravariància et permeten utilitzar un tipus més derivat del que s'especifica originalment (covariància) o un tipus més genèric (contravariància).

  • Covariància: Et permet utilitzar un tipus derivat on s'espera un tipus base.
  • Contravariància: Et permet utilitzar un tipus base on s'espera un tipus derivat.

Exemple de Covariància

IEnumerable<string> strings = new List<string>();
IEnumerable<object> objects = strings; // Covariància

Exemple de Contravariància

Action<object> actObject = (obj) => Console.WriteLine(obj);
Action<string> actString = actObject; // Contravariància

Resum

Els genèrics en C# proporcionen una manera de crear components de codi flexibles i reutilitzables que poden treballar amb qualsevol tipus de dades mentre mantenen la seguretat de tipus. Comprenent i utilitzant classes, mètodes i col·leccions genèriques, pots escriure codi més eficient i fàcil de mantenir. Les restriccions en els paràmetres de tipus i conceptes avançats com la covariància i la contravariància milloren encara més el poder dels genèrics, convertint-los en una eina essencial en la caixa d'eines d'un programador de C#.
