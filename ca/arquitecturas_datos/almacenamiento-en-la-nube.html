Introducció

L'emmagatzematge al núvol és una tecnologia que permet als usuaris guardar dades i arxius en servidors remots, accessibles a través d'internet. Aquesta tecnologia és fonamental en les arquitectures de dades modernes, ja que ofereix escalabilitat, flexibilitat i accessibilitat.

Conceptes Clau

  • Escalabilitat: Capacitat d'augmentar o disminuir els recursos d'emmagatzematge segons les necessitats.
  • Accessibilitat: Possibilitat d'accedir a les dades des de qualsevol lloc amb connexió a internet.
  • Seguretat: Mesures per protegir les dades emmagatzemades al núvol, incloent xifratge i autenticació.
  • Cost: Models de pagament per ús que permeten a les empreses pagar només per l'emmagatzematge que utilitzen.
  • Redundància: Emmagatzematge de dades en múltiples ubicacions per assegurar la disponibilitat i recuperació davant fallades.

Tipus d'Emmagatzematge al Núvol

  • Emmagatzematge d'Objectes: Ideal per emmagatzemar grans quantitats de dades no estructurades, com arxius multimèdia. Exemples: Amazon S3, Google Cloud Storage.
  • Emmagatzematge de Blocs: Utilitzat per aplicacions que requereixen accés ràpid i de baixa latència a les dades. Exemples: Amazon EBS, Google Persistent Disk.
  • Emmagatzematge d'Arxius: Similar als sistemes d'arxius tradicionals, però accessible a través del núvol. Exemples: Amazon EFS, Google Filestore.

Comparació de Tipus d'Emmagatzematge

| Tipus d'Emmagatzematge | Ús Principal | Exemples | Avantatges | Desavantatges | |------------------------|--------------|----------|------------|---------------| | Emmagatzematge d'Objectes | Dades no estructurades | Amazon S3, Google Cloud Storage | Escalabilitat, cost-efectiu | Latència més alta | | Emmagatzematge de Blocs | Aplicacions d'alta performance | Amazon EBS, Google Persistent Disk | Baixa latència, alta performance | Cost més alt | | Emmagatzematge d'Arxius | Sistemes d'arxius tradicionals | Amazon EFS, Google Filestore | Fàcil integració, accessibilitat | Menys escalable |

Exemples d'Ús

Emmagatzematge d'Objectes amb Amazon S3

import boto3

# Crear una sessió de S3
s3 = boto3.client('s3')

# Pujar un arxiu a un bucket de S3
s3.upload_file('local_file.txt', 'my_bucket', 's3_file.txt')

# Descarregar un arxiu d'un bucket de S3
s3.download_file('my_bucket', 's3_file.txt', 'local_file.txt')

Emmagatzematge de Blocs amb Amazon EBS

# Crear un volum EBS
aws ec2 create-volume --size 10 --region us-west-2 --availability-zone us-west-2a --volume-type gp2

# Adjuntar el volum a una instància EC2
aws ec2 attach-volume --volume-id vol-1234567890abcdef0 --instance-id i-1234567890abcdef0 --device /dev/sdf

Emmagatzematge d'Arxius amb Amazon EFS

# Crear un sistema d'arxius EFS
aws efs create-file-system --creation-token my-efs

# Muntar el sistema d'arxius en una instància EC2
sudo mount -t nfs4 -o nfsvers=4.1 fs-12345678.efs.us-west-2.amazonaws.com:/ efs

Conclusió

L'emmagatzematge al núvol és una peça clau en les arquitectures de dades modernes, oferint una varietat d'opcions que s'adapten a diferents necessitats i casos d'ús. Comprendre els diferents tipus d'emmagatzematge i les seves característiques permet als professionals de dades dissenyar solucions eficients i escalables.
