Introducció als Genèrics
Funcions genèriques ►

Els genèrics són una característica poderosa en TypeScript que et permeten crear components reutilitzables i flexibles. Et permeten escriure funcions, classes i interfícies que funcionen amb una varietat de tipus de dades mentre es manté la seguretat de tipus. Aquesta secció et guiarà a través dels conceptes bàsics dels genèrics, la seva sintaxi i casos pràctics d'ús.

Què són els Genèrics?

Els genèrics proporcionen una manera de crear components que poden treballar amb qualsevol tipus de dades. Ajuden a fer el codi més flexible i reutilitzable sense sacrificar la seguretat de tipus.

  • Seguretat de Tipus: Assegura que els tipus de les variables es coneixen en temps de compilació.
  • Reutilitzabilitat: Permet que la mateixa funció, classe o interfície funcioni amb diferents tipus de dades.
  • Flexibilitat: Permet la creació de components més generals i abstractes.

Sintaxi Bàsica

La sintaxi bàsica per als genèrics implica l'ús de claudàtors angulars (< >) per definir un tipus genèric.

Funcions Genèriques

Una funció genèrica pot acceptar qualsevol tipus de dades com a argument i retornar un valor del mateix tipus.

function identity<T>(arg: T): T {
    return arg;
}

let output1 = identity<string>("Hola, TypeScript!");
let output2 = identity<number>(42);
  • T és una variable de tipus que actua com a marcador de posició per al tipus que es passarà a la funció.
  • identity<string>("Hola, TypeScript!") especifica que T és string.
  • identity<number>(42) especifica que T és number.

Classes Genèriques

Les classes genèriques funcionen de manera similar a les funcions genèriques. Et permeten crear una classe que pot manejar diferents tipus.

class GenericNumber<T> {
    zeroValue: T;
    add: (x: T, y: T) => T;
}

let myGenericNumber = new GenericNumber<number>();
myGenericNumber.zeroValue = 0;
myGenericNumber.add = (x, y) => x + y;
  • GenericNumber<T> és una classe genèrica on T és una variable de tipus.
  • myGenericNumber és una instància de GenericNumber amb T establert a number.

Interfícies Genèriques

Les interfícies genèriques et permeten definir la forma d'un objecte que pot treballar amb diversos tipus.

interface GenericIdentityFn<T> {
    (arg: T): T;
}

function identity<T>(arg: T): T {
    return arg;
}

let myIdentity: GenericIdentityFn<number> = identity;
  • GenericIdentityFn<T> és una interfície genèrica.
  • myIdentity és una funció que s'adhereix a la interfície GenericIdentityFn<number>.

Restriccions

De vegades, necessites imposar restriccions als tipus que es poden utilitzar amb genèrics. Això es pot fer utilitzant la paraula clau extends.

interface Lengthwise {
    length: number;
}

function loggingIdentity<T extends Lengthwise>(arg: T): T {
    console.log(arg.length);
    return arg;
}

loggingIdentity({ length: 10, value: "Hola" });
  • T extends Lengthwise assegura que T té una propietat length.
  • loggingIdentity ara només pot acceptar arguments que tinguin una propietat length.

Ús de Múltiples Variables de Tipus

Pots utilitzar múltiples variables de tipus en una sola funció genèrica, classe o interfície.

function swap<T, U>(tuple: [T, U]): [U, T] {
    return [tuple[1], tuple[0]];
}

let swappedTuple = swap([7, "set"]); // [ "set", 7 ]
  • T i U són variables de tipus.
  • La funció swap intercanvia els elements de la tupla.

Casos Pràctics d'Ús

Els genèrics s'utilitzen àmpliament en TypeScript per a diversos propòsits:

  • Col·leccions: Arrays, llistes enllaçades, piles, cues.
  • Funcions d'Utilitat: Ordenació, cerca, filtratge.
  • Estructures de Dades: Arbres, gràfics, mapes.

Exemple: Pila Genèrica

class Stack<T> {
    private items: T[] = [];

    push(item: T): void {
        this.items.push(item);
    }

    pop(): T | undefined {
        return this.items.pop();
    }
}

let numberStack = new Stack<number>();
numberStack.push(10);
numberStack.push(20);
console.log(numberStack.pop()); // 20

let stringStack = new Stack<string>();
stringStack.push("Hola");
stringStack.push("Món");
console.log(stringStack.pop()); // "Món"
  • Stack<T> és una classe genèrica.
  • numberStack i stringStack són instàncies de Stack amb diferents tipus.

Conclusió

Els genèrics en TypeScript proporcionen una manera poderosa de crear components flexibles, reutilitzables i segurs en termes de tipus. Entenent i utilitzant els genèrics, pots escriure codi més abstracte i generalitzat que funcioni amb una varietat de tipus de dades. Això no només millora la reutilitzabilitat del codi, sinó que també assegura la seguretat de tipus, fent que les teves aplicacions TypeScript siguin més robustes i mantenibles.
