Introducció

TypeScript proporciona un compilador potent (tsc) que inclou una àmplia gamma d'opcions per personalitzar el procés de compilació. Entendre aquestes opcions és crucial per configurar els teus projectes TypeScript de manera efectiva.

Opcions Bàsiques del Compilador

`target`

  • Especifica la versió objectiu d'ECMAScript.
  • Valors comuns: ES5, ES6/ES2015, ES2016, ES2017, ES2018, ES2019, ES2020, ES2021, ES2022, ESNext.
{
  "compilerOptions": {
    "target": "ES6"
  }
}

`module`

  • Determina el sistema de mòduls a utilitzar.
  • Valors comuns: CommonJS, AMD, System, UMD, ES6/ES2015, ESNext.
{
  "compilerOptions": {
    "module": "CommonJS"
  }
}

`lib`

  • Especifica els fitxers de biblioteca que s'inclouran en la compilació.
  • Valors comuns: ES5, ES6, DOM, DOM.Iterable, ScriptHost, WebWorker.
{
  "compilerOptions": {
    "lib": ["ES6", "DOM"]
  }
}

Opcions de Verificació Estricta de Tipus

`strict`

  • Habilita totes les opcions de verificació estricta de tipus.
  • Inclou: strictNullChecks, strictFunctionTypes, strictPropertyInitialization, noImplicitAny, noImplicitThis, alwaysStrict.
{
  "compilerOptions": {
    "strict": true
  }
}

`noImplicitAny`

  • Genera un error en expressions i declaracions amb un tipus any implícit.
{
  "compilerOptions": {
    "noImplicitAny": true
  }
}

`strictNullChecks`

  • Assegura que null i undefined només siguin assignables a ells mateixos i any.
{
  "compilerOptions": {
    "strictNullChecks": true
  }
}

Opcions de Resolució de Mòduls

`baseUrl`

  • Especifica el directori base per resoldre noms de mòduls no relatius.
{
  "compilerOptions": {
    "baseUrl": "./src"
  }
}

`paths`

  • Una sèrie d'entrades que re-mapen les importacions a ubicacions de cerca relatives al baseUrl.
{
  "compilerOptions": {
    "baseUrl": "./",
    "paths": {
      "@app/*": ["src/app/*"],
      "@components/*": ["src/components/*"]
    }
  }
}

Opcions de Sortida

`outDir`

  • Redirigeix l'estructura de sortida al directori.
{
  "compilerOptions": {
    "outDir": "./dist"
  }
}

`rootDir`

  • Especifica el directori arrel dels fitxers d'entrada.
{
  "compilerOptions": {
    "rootDir": "./src"
  }
}

`sourceMap`

  • Genera els fitxers .map corresponents.
{
  "compilerOptions": {
    "sourceMap": true
  }
}

Opcions Avançades

`incremental`

  • Habilita la compilació incremental.
{
  "compilerOptions": {
    "incremental": true
  }
}

`composite`

  • Habilita la compilació de projectes.
{
  "compilerOptions": {
    "composite": true
  }
}

`declaration`

  • Genera els fitxers .d.ts corresponents.
{
  "compilerOptions": {
    "declaration": true
  }
}

Exemple de Configuració

Aquí tens un exemple de fitxer tsconfig.json que combina diverses de les opcions discutides:

{
  "compilerOptions": {
    "target": "ES6",
    "module": "CommonJS",
    "lib": ["ES6", "DOM"],
    "strict": true,
    "baseUrl": "./",
    "paths": {
      "@app/*": ["src/app/*"],
      "@components/*": ["src/components/*"]
    },
    "outDir": "./dist",
    "rootDir": "./src",
    "sourceMap": true,
    "incremental": true,
    "composite": true,
    "declaration": true
  }
}

Conclusió

Entendre i configurar les opcions del compilador TypeScript pot millorar significativament el teu flux de treball de desenvolupament. Aprofitant aquestes opcions, pots adaptar el procés de compilació per satisfer les necessitats específiques del teu projecte, assegurant una millor seguretat de tipus, una sortida optimitzada i una millor resolució de mòduls.
