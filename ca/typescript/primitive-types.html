Tipus Primitivos en TypeScript
Arrays i tuples ►

Introducció

Els tipus primitius són els tipus de dades més bàsics disponibles en TypeScript. Entendre aquests tipus és crucial ja que formen la base per a estructures de dades i operacions més complexes. En aquesta secció, explorarem els diferents tipus primitius disponibles en TypeScript, el seu ús i com treballar amb ells de manera efectiva.

Tipus Primitivos Clau

Number

  • Representa tant nombres enters com de punt flotant.
  • Suporta literals hexadecimals, decimals, binaris i octals.
let decimal: number = 6;
let hex: number = 0xf00d;
let binary: number = 0b1010;
let octal: number = 0o744;

console.log(decimal); // Output: 6
console.log(hex); // Output: 61453
console.log(binary); // Output: 10
console.log(octal); // Output: 484

String

  • Representa dades textuals.
  • Pot estar tancat en cometes simples ('), cometes dobles (") o backticks (`) per a literals de plantilla.
let color: string = "blue";
let sentence: string = `The color is ${color}`;

console.log(color); // Output: blue
console.log(sentence); // Output: The color is blue

Boolean

  • Representa un valor lògic: true o false.
let isDone: boolean = false;

console.log(isDone); // Output: false

Null i Undefined

  • null i undefined són tipus que només tenen un valor cadascun: null i undefined respectivament.
  • Per defecte, null i undefined són subtipus de tots els altres tipus.
let u: undefined = undefined;
let n: null = null;

console.log(u); // Output: undefined
console.log(n); // Output: null

Symbol

  • Representa un identificador únic.
  • Creat utilitzant la funció Symbol.
let sym1 = Symbol();
let sym2 = Symbol("key");

console.log(sym1 === sym2); // Output: false

BigInt

  • Representa nombres enters més grans que el nombre més gran que JavaScript pot representar de manera fiable amb el primitiu number.
  • Creat afegint n al final d'un enter o utilitzant la funció BigInt.
let big: bigint = 100n;

console.log(big); // Output: 100n

Comparació de Tipus Primitivos

| Tipus | Exemple | Descripció | |-------------|-------------------|-------------------------------------| | number | let num = 42; | Representa valors numèrics | | string | let str = "Hi"; | Representa dades textuals | | boolean | let isTrue = true; | Representa valors lògics | | null | let n = null; | Representa l'absència intencional de qualsevol valor d'objecte | | undefined | let u; | Representa una variable no inicialitzada | | symbol | let sym = Symbol(); | Representa un identificador únic | | bigint | let big = 100n; | Representa enters grans |

Assertions de Tipus

TypeScript et permet anul·lar el tipus inferit utilitzant assertions de tipus. Això és útil quan saps més sobre el tipus d'un valor que TypeScript.

let someValue: any = "this is a string";
let strLength: number = (someValue as string).length;

console.log(strLength); // Output: 16

Conclusió

Entendre els tipus primitius en TypeScript és essencial per a qualsevol programador. Aquests tipus formen els blocs de construcció de les teves aplicacions, i saber com utilitzar-los de manera efectiva t'ajudarà a escriure codi més robust i lliure d'errors. En aquesta secció, hem cobert els tipus primitius bàsics, el seu ús i com realitzar assertions de tipus. A mesura que progressis, veuràs com aquests conceptes fonamentals s'integren amb funcions més avançades de TypeScript.
