Modificadors d'accés a TypeScript
Classes abstractes ►

Els modificadors d'accés a TypeScript són paraules clau que estableixen l'accessibilitat de propietats i mètodes en classes. Ajuden a encapsular les dades i controlar-ne la visibilitat des de fora de la classe.

Principals modificadors d'accés

TypeScript proporciona tres principals modificadors d'accés:

  • public
  • private
  • protected

Públic

  • El modificador public és el nivell d'accés per defecte.
  • Els membres marcats com a public poden ser accessibles des de qualsevol lloc, tant dins com fora de la classe.
class Animal {
    public name: string;

    constructor(name: string) {
        this.name = name;
    }

    public makeSound() {
        console.log(`${this.name} makes a sound.`);
    }
}

const dog = new Animal('Dog');
console.log(dog.name); // Accessible
dog.makeSound(); // Accessible

Privat

  • El modificador private restringeix l'accés als membres dins de la mateixa classe.
  • Els membres marcats com a private no poden ser accessibles des de fora de la classe.
class Animal {
    private name: string;

    constructor(name: string) {
        this.name = name;
    }

    private makeSound() {
        console.log(`${this.name} makes a sound.`);
    }

    public revealName() {
        console.log(`The animal's name is ${this.name}`);
    }
}

const cat = new Animal('Cat');
// console.log(cat.name); // Error: Property 'name' is private and only accessible within class 'Animal'.
// cat.makeSound(); // Error: Method 'makeSound' is private and only accessible within class 'Animal'.
cat.revealName(); // Accessible

Protegit

  • El modificador protected permet l'accés als membres dins de la classe i les seves subclasses.
  • Els membres marcats com a protected no poden ser accessibles des de fora de la classe o les seves subclasses.
class Animal {
    protected name: string;

    constructor(name: string) {
        this.name = name;
    }

    protected makeSound() {
        console.log(`${this.name} makes a sound.`);
    }
}

class Dog extends Animal {
    constructor(name: string) {
        super(name);
    }

    public bark() {
        console.log(`${this.name} barks.`);
        this.makeSound(); // Accessible within subclass
    }
}

const dog = new Dog('Dog');
// console.log(dog.name); // Error: Property 'name' is protected and only accessible within class 'Animal' and its subclasses.
// dog.makeSound(); // Error: Method 'makeSound' is protected and only accessible within class 'Animal' and its subclasses.
dog.bark(); // Accessible

Taula comparativa

| Modificador | Accessible dins de la classe | Accessible dins de la subclasse | Accessible fora de la classe | |-------------|------------------------------|---------------------------------|------------------------------| | public | Sí | Sí | Sí | | private | Sí | No | No | | protected | Sí | Sí | No |

Conclusió

Els modificadors d'accés a TypeScript són essencials per controlar la visibilitat i accessibilitat dels membres de la classe. Ajuden a encapsular les dades i assegurar-se que només les parts previstes del teu codi poden interactuar amb certes propietats i mètodes. Entendre i utilitzar els modificadors public, private i protected de manera efectiva pot conduir a un codi més robust i mantenible.
