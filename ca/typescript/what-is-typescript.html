Què és TypeScript?
Configuració de l'entorn de TypeScript ►

TypeScript és un potent superconjunt de JavaScript amb tipatge estàtic que es compila a JavaScript pla. Va ser desenvolupat per Microsoft per abordar algunes de les mancances de JavaScript, especialment per al desenvolupament d'aplicacions a gran escala. TypeScript afegeix tipus estàtics opcionals, classes i interfícies a JavaScript, facilitant l'escriptura i manteniment de bases de codi complexes.

Característiques Clau de TypeScript

  • Tipatge Estàtic: TypeScript introdueix tipus estàtics a JavaScript, permetent als desenvolupadors detectar errors en temps de compilació en lloc de temps d'execució.
  • Inferència de Tipus: TypeScript pot inferir tipus, reduint la necessitat d'anotacions de tipus explícites.
  • Compatibilitat: El codi TypeScript es compila a JavaScript pla, assegurant la compatibilitat amb el codi i les biblioteques existents de JavaScript.
  • Suport Millorat per IDE: TypeScript proporciona millors eines i autocompleció en IDEs moderns, millorant la productivitat dels desenvolupadors.
  • Programació Orientada a Objectes: TypeScript suporta característiques com classes, interfícies i herència, facilitant la implementació de dissenys orientats a objectes.

Per què Utilitzar TypeScript?

  • Millora de la Qualitat del Codi: El tipatge estàtic ajuda a detectar errors a l'inici del procés de desenvolupament.
  • Millors Eines: Suport millorat per IDE amb característiques com autocompleció, refactorització i navegació.
  • Escalabilitat: Més fàcil de gestionar i mantenir grans bases de codi.
  • Interoperabilitat: S'integra sense problemes amb les biblioteques i marcs de treball existents de JavaScript.

Sintaxi Bàsica i Exemples

Anotacions de Tipus

TypeScript permet especificar tipus per a variables, paràmetres de funcions i valors de retorn.

let message: string = "Hola, TypeScript!";
let count: number = 42;
let isActive: boolean = true;

Funcions

Pots definir tipus per als paràmetres de funcions i els valors de retorn.

function add(a: number, b: number): number {
    return a + b;
}

let result: number = add(5, 3); // result és 8

Interfícies

Les interfícies defineixen la forma d'un objecte, facilitant l'aplicació d'una estructura.

interface User {
    name: string;
    age: number;
}

let user: User = {
    name: "Alice",
    age: 30
};

Classes

TypeScript suporta classes i herència, permetent la programació orientada a objectes.

class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet(): string {
        return `Hola, em dic ${this.name}`;
    }
}

let person = new Person("Bob", 25);
console.log(person.greet()); // "Hola, em dic Bob"

TypeScript vs. JavaScript

| Característica | JavaScript | TypeScript | |------------------------|-------------------------------------|-------------------------------------| | Tipatge | Dinàmic | Estàtic (amb inferència de tipus) | | Detecció d'Errors | Temps d'execució | Temps de compilació | | Classes i Interfícies | Classes ES6 (sense interfícies) | Classes i Interfícies | | Eines | Suport bàsic per IDE | Suport millorat per IDE | | Compatibilitat | JavaScript natiu | Es compila a JavaScript |

Conclusió

TypeScript és una eina robusta que millora JavaScript afegint tipus estàtics, classes i interfícies. Ajuda els desenvolupadors a detectar errors a l'inici, millora la qualitat del codi i fa que el desenvolupament d'aplicacions a gran escala sigui més manejable. Entenent els conceptes bàsics de TypeScript, pots aprofitar les seves característiques per escriure codi més net i mantenible. A mesura que avancis en aquest curs, adquiriràs coneixements més profunds sobre les característiques avançades de TypeScript i les millors pràctiques.
