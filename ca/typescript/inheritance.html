Herència en TypeScript
Modificadors d'accés ►

L'herència és un concepte fonamental en la programació orientada a objectes que permet a una classe heretar propietats i mètodes d'una altra classe. En TypeScript, l'herència ajuda a crear una jerarquia de classes que comparteixen una estructura i comportament comuns, promovent la reutilització i mantenibilitat del codi.

Conceptes Clau de l'Herència

  • Classe Base (Classe Pare): La classe les propietats i mètodes de la qual són heretats.
  • Classe Derivada (Classe Filla): La classe que hereta propietats i mètodes de la classe base.
  • Paraula Clau extends: Utilitzada per crear una classe derivada a partir d'una classe base.
  • Sobreescriptura de Mètodes: La capacitat d'una classe derivada per proporcionar una implementació específica d'un mètode que ja està definit en la seva classe base.
  • Paraula Clau super: Utilitzada per cridar el constructor o mètodes de la classe base des de la classe derivada.

Creació d'una Classe Base

Comencem definint una classe base simple anomenada Animal.

class Animal {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    makeSound(): void {
        console.log(`${this.name} fa un so.`);
    }
}

En aquest exemple:

  • La classe Animal té una propietat name i un mètode makeSound.
  • El constructor inicialitza la propietat name.

Creació d'una Classe Derivada

Ara, creem una classe derivada Dog que hereta d'Animal.

class Dog extends Animal {
    breed: string;

    constructor(name: string, breed: string) {
        super(name); // Crida el constructor de la classe base
        this.breed = breed;
    }

    makeSound(): void {
        console.log(`${this.name} borda.`);
    }
}

En aquest exemple:

  • La classe Dog extén la classe Animal utilitzant la paraula clau extends.
  • La classe Dog té una propietat addicional breed.
  • El constructor de Dog crida el constructor d'Animal utilitzant super(name).
  • El mètode makeSound és sobreescrit per proporcionar una implementació específica per a Dog.

Ús de la Classe Derivada

Creem una instància de la classe Dog i utilitzem els seus mètodes.

const myDog = new Dog('Buddy', 'Golden Retriever');
myDog.makeSound(); // Sortida: Buddy borda.
console.log(`El meu gos és un ${myDog.breed}.`); // Sortida: El meu gos és un Golden Retriever.

Sobreescriptura de Mètodes

La sobreescriptura de mètodes permet a una classe derivada proporcionar una implementació específica d'un mètode que ja està definit en la seva classe base. Això és útil quan la classe derivada necessita canviar o estendre el comportament del mètode de la classe base.

class Cat extends Animal {
    constructor(name: string) {
        super(name);
    }

    makeSound(): void {
        console.log(`${this.name} miola.`);
    }
}

const myCat = new Cat('Whiskers');
myCat.makeSound(); // Sortida: Whiskers miola.

Resum

  • L'herència permet a una classe heretar propietats i mètodes d'una altra classe, promovent la reutilització del codi.
  • La paraula clau extends s'utilitza per crear una classe derivada a partir d'una classe base.
  • La paraula clau super s'utilitza per cridar el constructor o mètodes de la classe base.
  • La sobreescriptura de mètodes permet a una classe derivada proporcionar una implementació específica d'un mètode definit en la classe base.

Entenent i utilitzant l'herència en TypeScript, pots crear codi més estructurat i mantenible, aprofitant el poder dels principis de la programació orientada a objectes.
