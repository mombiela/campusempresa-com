Classes Genèriques
Restriccions genèriques ►

En aquesta secció, explorarem el concepte de Classes Genèriques en TypeScript. Les genèriques permeten crear components flexibles i reutilitzables que poden treballar amb una varietat de tipus de dades. Això és particularment útil per crear estructures de dades com llistes, piles i cues.

Què són les Genèriques?

Les genèriques proporcionen una manera de crear classes, funcions i interfícies que poden treballar amb diferents tipus de dades sense sacrificar la seguretat de tipus. Permeten definir un marcador de posició per a un tipus que es pot especificar més tard.

Conceptes Clau

  • Paràmetre de Tipus: Un marcador de posició per a un tipus que es especifica quan la classe genèrica és instanciada.
  • Seguretat de Tipus: Assegura que els tipus de variables es coneixen en temps de compilació, reduint els errors en temps d'execució.
  • Reutilitzabilitat: Les classes genèriques es poden reutilitzar amb diferents tipus de dades, fent el teu codi més flexible i mantenible.

Definint una Classe Genèrica

Per definir una classe genèrica, utilitzes claudàtors angulars (<>) per especificar el paràmetre de tipus. Aquí tens un exemple senzill d'una classe genèrica en TypeScript:

class Box<T> {
    private content: T;

    constructor(content: T) {
        this.content = content;
    }

    getContent(): T {
        return this.content;
    }

    setContent(content: T): void {
        this.content = content;
    }
}

En aquest exemple:

  • T és el paràmetre de tipus.
  • La classe Box ara pot contenir qualsevol tipus de contingut, especificat quan la classe és instanciada.

Utilitzant una Classe Genèrica

Pots crear instàncies de la classe genèrica amb diferents tipus:

let numberBox = new Box<number>(123);
console.log(numberBox.getContent()); // Output: 123

let stringBox = new Box<string>("Hello, TypeScript!");
console.log(stringBox.getContent()); // Output: Hello, TypeScript!

Múltiples Paràmetres de Tipus

També pots definir una classe genèrica amb múltiples paràmetres de tipus:

class Pair<K, V> {
    private key: K;
    private value: V;

    constructor(key: K, value: V) {
        this.key = key;
        this.value = value;
    }

    getKey(): K {
        return this.key;
    }

    getValue(): V {
        return this.value;
    }
}

Utilitzant la classe Pair:

let pair = new Pair<string, number>("age", 30);
console.log(pair.getKey()); // Output: age
console.log(pair.getValue()); // Output: 30

Restriccions en els Paràmetres de Tipus

De vegades potser voldràs restringir els tipus que es poden utilitzar com a paràmetres de tipus. Pots aconseguir això utilitzant restriccions:

class Container<T extends { length: number }> {
    private content: T;

    constructor(content: T) {
        this.content = content;
    }

    getContentLength(): number {
        return this.content.length;
    }
}

En aquest exemple:

  • El paràmetre de tipus T està restringit a tipus que tenen una propietat length.

Utilitzant la classe Container:

let stringContainer = new Container<string>("Hello");
console.log(stringContainer.getContentLength()); // Output: 5

let arrayContainer = new Container<number[]>([1, 2, 3, 4]);
console.log(arrayContainer.getContentLength()); // Output: 4

Resum

Les classes genèriques en TypeScript proporcionen una manera poderosa de crear components flexibles i reutilitzables. Utilitzant paràmetres de tipus, pots escriure classes que treballen amb qualsevol tipus de dades mentre mantens la seguretat de tipus. Això fa que el teu codi sigui més robust i fàcil de mantenir.

Punts Clau

  • Les genèriques permeten crear classes que poden treballar amb qualsevol tipus de dades.
  • Els paràmetres de tipus són marcadors de posició per a tipus que es especifiquen quan la classe és instanciada.
  • Pots definir múltiples paràmetres de tipus i afegir restriccions a ells.
  • Utilitzar genèriques millora la reutilitzabilitat del codi i la seguretat de tipus.

Dominant les genèriques, pots escriure codi TypeScript més flexible i mantenible, fent que les teves aplicacions siguin més robustes i fàcils d'extendre.
