Introducció a GraphQL

  • Què és GraphQL?

    • Un llenguatge de consulta per a APIs.
    • Permet als clients sol·licitar només les dades que necessiten.
    • Desenvolupat per Facebook.

  • Conceptes Clau:

    • Esquema: Defineix l'estructura de les dades.
    • Consultes: Sol·licituds per llegir dades.
    • Mutacions: Sol·licituds per escriure dades.
    • Resolvers: Funcions que gestionen la lògica per obtenir dades.

Configuració de TypeScript amb GraphQL

  • Prerequisits:

    • Node.js i npm instal·lats.
    • Comprensió bàsica de TypeScript.

  • Instal·lació de Paquets Necessaris:

    npm install graphql express express-graphql @types/express @types/graphql typescript ts-node

  • Configuració de TypeScript:

    // tsconfig.json
{
  "compilerOptions": {
    "target": "ES6",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}

Creació d'un Servidor Bàsic de GraphQL amb TypeScript

  • Configuració del Servidor Express: 
    import express from 'express';
import { graphqlHTTP } from 'express-graphql';
import { buildSchema } from 'graphql';

const app = express();

// Defineix l'esquema de GraphQL
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// Defineix els resolvers
const root = {
  hello: () => {
    return 'Hola món!';
  },
};

// Configura el punt final de GraphQL
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Servidor en funcionament a http://localhost:4000/graphql'));

GraphQL Avançat amb TypeScript

  • Seguretat de Tipus amb GraphQL:

    • Utilitza graphql-code-generator per generar tipus de TypeScript a partir del teu esquema de GraphQL.
    npm install @graphql-codegen/cli @graphql-codegen/typescript @graphql-codegen/typescript-resolvers

    • Configuració de Codegen:

      # codegen.yml
schema: "http://localhost:4000/graphql"
generates:
  src/generated/graphql.ts:
    plugins:
      - "typescript"
      - "typescript-resolvers"

    • Execució de Codegen:

      npx graphql-codegen

  • Ús dels Tipus Generats:

    import { Resolvers } from './generated/graphql';

const resolvers: Resolvers = {
  Query: {
    hello: () => 'Hola món!',
  },
};

Integració de GraphQL amb una Base de Dades

  • Connexió a una Base de Dades:

    • Utilitza TypeORM per a les interaccions amb la base de dades.
    npm install typeorm reflect-metadata sqlite3

    • Configuració de TypeORM:

      import "reflect-metadata";
import { createConnection } from "typeorm";
import { User } from "./entity/User";

createConnection().then(async connection => {
  console.log("Connectat a la base de dades");
}).catch(error => console.log(error));

    • Definició d'Entitats:

      import { Entity, PrimaryGeneratedColumn, Column } from "typeorm";

@Entity()
export class User {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  name: string;

  @Column()
  email: string;
}

    • Actualització dels Resolvers per Utilitzar la Base de Dades:

      import { getRepository } from "typeorm";
import { User } from "./entity/User";

const resolvers: Resolvers = {
  Query: {
    users: async () => {
      const userRepository = getRepository(User);
      return await userRepository.find();
    },
  },
  Mutation: {
    addUser: async (_, { name, email }) => {
      const userRepository = getRepository(User);
      const user = userRepository.create({ name, email });
      await userRepository.save(user);
      return user;
    },
  },
};
