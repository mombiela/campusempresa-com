Introducció

En aquesta secció, explorarem com utilitzar TypeScript amb MongoDB, una base de dades NoSQL popular. Cobrirem els conceptes bàsics de la connexió a MongoDB, la realització d'operacions CRUD (Crear, Llegir, Actualitzar, Eliminar) i l'ús de TypeScript per garantir la seguretat de tipus i millorar la qualitat del codi.

Configuració de l'entorn

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

Prerequisits

  • Node.js instal·lat
  • TypeScript instal·lat
  • MongoDB instal·lat i en funcionament

Instal·lació de paquets necessaris

Per interactuar amb MongoDB en un projecte TypeScript, necessitem instal·lar els següents paquets:

  • mongodb: Driver de MongoDB per a Node.js
  • @types/mongodb: Definicions de tipus per al driver de MongoDB
npm install mongodb @types/mongodb

Connexió a MongoDB

Per connectar-nos a MongoDB, necessitem crear una instància de MongoClient i connectar-la al nostre servidor MongoDB.

Exemple de codi

import { MongoClient } from 'mongodb';

const uri = 'mongodb://localhost:27017';
const client = new MongoClient(uri);

async function connectToDatabase() {
    try {
        await client.connect();
        console.log('Connected to MongoDB');
    } catch (error) {
        console.error('Error connecting to MongoDB:', error);
    }
}

connectToDatabase();

Explicació

  • Importem MongoClient del paquet mongodb.
  • Definim l'URI per al nostre servidor MongoDB.
  • Creem una nova instància de MongoClient.
  • Definim una funció asíncrona connectToDatabase per connectar-nos al servidor MongoDB.
  • Utilitzem await client.connect() per establir la connexió.
  • Gestionem qualsevol error de connexió utilitzant un bloc try-catch.

Operacions CRUD

Les operacions CRUD són fonamentals quan es treballa amb bases de dades. Vegem com realitzar aquestes operacions utilitzant TypeScript i MongoDB.

Crear

Per inserir documents en una col·lecció de MongoDB, utilitzem els mètodes insertOne o insertMany.

async function createDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const newDocument = { name: 'John Doe', age: 30 };

    try {
        const result = await collection.insertOne(newDocument);
        console.log('Document inserted with _id:', result.insertedId);
    } catch (error) {
        console.error('Error inserting document:', error);
    }
}

createDocument();

Llegir

Per llegir documents d'una col·lecció de MongoDB, utilitzem el mètode find.

async function readDocuments() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    try {
        const documents = await collection.find({}).toArray();
        console.log('Documents:', documents);
    } catch (error) {
        console.error('Error reading documents:', error);
    }
}

readDocuments();

Actualitzar

Per actualitzar documents en una col·lecció de MongoDB, utilitzem els mètodes updateOne o updateMany.

async function updateDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const filter = { name: 'John Doe' };
    const update = { $set: { age: 31 } };

    try {
        const result = await collection.updateOne(filter, update);
        console.log('Documents matched:', result.matchedCount);
        console.log('Documents modified:', result.modifiedCount);
    } catch (error) {
        console.error('Error updating document:', error);
    }
}

updateDocument();

Eliminar

Per eliminar documents d'una col·lecció de MongoDB, utilitzem els mètodes deleteOne o deleteMany.

async function deleteDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const filter = { name: 'John Doe' };

    try {
        const result = await collection.deleteOne(filter);
        console.log('Documents deleted:', result.deletedCount);
    } catch (error) {
        console.error('Error deleting document:', error);
    }
}

deleteDocument();

Ús d'interfícies TypeScript

Per garantir la seguretat de tipus, podem definir interfícies TypeScript per als nostres documents.

Exemple d'interfície

interface User {
    name: string;
    age: number;
}

async function createTypedDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection<User>('users');

    const newUser: User = { name: 'Jane Doe', age: 25 };

    try {
        const result = await collection.insertOne(newUser);
        console.log('Document inserted with _id:', result.insertedId);
    } catch (error) {
        console.error('Error inserting document:', error);
    }
}

createTypedDocument();

Explicació

  • Definim una interfície User amb les propietats name i age.
  • Utilitzem la interfície quan definim el tipus de col·lecció: db.collection<User>('users').
  • Creem un nou document d'usuari que s'adhereix a la interfície User.

Conclusió

En aquesta secció, hem cobert com utilitzar TypeScript amb MongoDB. Hem après a configurar l'entorn, connectar-nos a MongoDB i realitzar operacions CRUD. A més, hem vist com utilitzar interfícies TypeScript per garantir la seguretat de tipus. Combinant TypeScript amb MongoDB, podem escriure codi més robust i mantenible.
