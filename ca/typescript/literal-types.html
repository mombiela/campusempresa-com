Introducció als Tipus Literals

Els tipus literals a TypeScript et permeten especificar valors exactes que una variable pot tenir. Aquesta característica millora la seguretat de tipus assegurant que les variables només acceptin valors predefinits.

Conceptes Clau

  • Tipus Literals: Aquests són tipus que representen valors específics en lloc de tipus generals.
  • Literals de Cadenes: Tipus que només poden ser un conjunt de valors de cadena.
  • Literals Numèrics: Tipus que només poden ser un conjunt de valors numèrics.
  • Literals Booleans: Tipus que només poden ser true o false.
  • Tipus Unió: Combinació de múltiples tipus literals per permetre que una variable tingui qualsevol d'uns quants valors.

Tipus Literals de Cadenes

Els tipus literals de cadenes et permeten especificar que una variable només pot ser un conjunt de valors de cadena.

type Direction = 'North' | 'South' | 'East' | 'West';

let direction: Direction;

direction = 'North'; // Vàlid
direction = 'South'; // Vàlid
direction = 'Up';    // Error: El tipus '"Up"' no es pot assignar al tipus 'Direction'.

Exemple Explicat

  • type Direction = 'North' | 'South' | 'East' | 'West';: Defineix un tipus Direction que només pot ser una de les quatre cadenes especificades.
  • let direction: Direction;: Declara una variable direction del tipus Direction.
  • Assignar direction a 'North' o 'South' és vàlid, però assignar-lo a 'Up' resulta en un error de tipus.

Tipus Literals Numèrics

Els tipus literals numèrics restringeixen una variable a valors numèrics específics.

type DiceRoll = 1 | 2 | 3 | 4 | 5 | 6;

let roll: DiceRoll;

roll = 1; // Vàlid
roll = 7; // Error: El tipus '7' no es pot assignar al tipus 'DiceRoll'.

Exemple Explicat

  • type DiceRoll = 1 | 2 | 3 | 4 | 5 | 6;: Defineix un tipus DiceRoll que només pot ser un dels sis números especificats.
  • let roll: DiceRoll;: Declara una variable roll del tipus DiceRoll.
  • Assignar roll a 1 és vàlid, però assignar-lo a 7 resulta en un error de tipus.

Tipus Literals Booleans

Els tipus literals booleans restringeixen una variable a true o false.

type IsEnabled = true | false;

let enabled: IsEnabled;

enabled = true;  // Vàlid
enabled = false; // Vàlid
enabled = 'yes'; // Error: El tipus '"yes"' no es pot assignar al tipus 'IsEnabled'.

Exemple Explicat

  • type IsEnabled = true | false;: Defineix un tipus IsEnabled que només pot ser true o false.
  • let enabled: IsEnabled;: Declara una variable enabled del tipus IsEnabled.
  • Assignar enabled a true o false és vàlid, però assignar-lo a 'yes' resulta en un error de tipus.

Tipus Unió

Els tipus unió combinen múltiples tipus literals, permetent que una variable tingui qualsevol d'uns quants valors.

type Status = 'success' | 'error' | 'loading';

let currentStatus: Status;

currentStatus = 'success'; // Vàlid
currentStatus = 'error';   // Vàlid
currentStatus = 'pending'; // Error: El tipus '"pending"' no es pot assignar al tipus 'Status'.

Exemple Explicat

  • type Status = 'success' | 'error' | 'loading';: Defineix un tipus Status que només pot ser una de les tres cadenes especificades.
  • let currentStatus: Status;: Declara una variable currentStatus del tipus Status.
  • Assignar currentStatus a 'success' o 'error' és vàlid, però assignar-lo a 'pending' resulta en un error de tipus.

Conclusió

Els tipus literals a TypeScript proporcionen una manera d'especificar valors exactes que una variable pot tenir, millorant la seguretat de tipus i reduint errors. Utilitzant literals de cadenes, numèrics i booleans, pots crear codi més precís i previsible. Els tipus unió amplien aquesta capacitat permetent que les variables acceptin múltiples valors literals, fent que el teu codi TypeScript sigui tant flexible com robust.

Resum

  • Els tipus literals restringeixen les variables a valors específics.
  • Els literals de cadenes, numèrics i booleans són tipus comuns.
  • Els tipus unió combinen múltiples literals per a una major flexibilitat.
  • Utilitzar tipus literals millora la seguretat de tipus i redueix errors en temps d'execució.
