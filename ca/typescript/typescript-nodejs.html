Introducció a TypeScript amb Node.js

  • Visió general de TypeScript i els seus beneficis.
  • Per què utilitzar TypeScript amb Node.js?
  • Configuració d'un projecte TypeScript amb Node.js.

Configuració de l'entorn

  • Instal·lació de Node.js i npm.
  • Instal·lació de TypeScript globalment: 
    npm install -g typescript
  • Inicialització d'un nou projecte Node.js: 
    mkdir my-typescript-node-project
cd my-typescript-node-project
npm init -y
  • Afegint TypeScript al projecte: 
    npm install typescript --save-dev
  • Creació d'un fitxer tsconfig.json: 
    npx tsc --init

Conceptes bàsics de TypeScript

  • Anotacions de tipus: 
    let message: string = "Hello, TypeScript with Node.js!";
console.log(message);
  • Interfícies: 
    interface User {
  name: string;
  age: number;
}

const user: User = {
  name: "John Doe",
  age: 30
};
console.log(user);
  • Classes: 
    class Person {
  constructor(public name: string, public age: number) {}

  greet() {
    console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
  }
}

const person = new Person("Jane Doe", 25);
person.greet();

Integració de TypeScript amb Node.js

  • Creació d'un servidor simple amb Express: 
    npm install express @types/express --save

    import express, { Request, Response } from 'express';

const app = express();
const port = 3000;

app.get('/', (req: Request, res: Response) => {
  res.send('Hello, TypeScript with Node.js!');
});

app.listen(port, () => {
  console.log(`Server is running at http://localhost:${port}`);
});
  • Compilació de TypeScript a JavaScript: 
    npx tsc
  • Execució del JavaScript compilat: 
    node dist/index.js

Funcionalitats avançades de TypeScript

  • Genèrics: 
    function identity<T>(arg: T): T {
  return arg;
}

let output = identity<string>("myString");
console.log(output);
  • Decoradors: 
    function log(target: any, key: string) {
  let value = target[key];

  const getter = () => {
    console.log(`Get: ${key} => ${value}`);
    return value;
  };

  const setter = (newVal: any) => {
    console.log(`Set: ${key} => ${newVal}`);
    value = newVal;
  };

  Object.defineProperty(target, key, {
    get: getter,
    set: setter,
    enumerable: true,
    configurable: true
  });
}

class Example {
  @log
  public name: string;

  constructor(name: string) {
    this.name = name;
  }
}

const example = new Example("TypeScript");
example.name = "Node.js";
console.log(example.name);

Testejant TypeScript amb Node.js

  • Configuració de Jest per a tests: 
    npm install jest ts-jest @types/jest --save-dev
  • Configuració de Jest: 
    npx ts-jest config:init
  • Escriure un test simple: 
    // sum.ts
export const sum = (a: number, b: number): number => {
  return a + b;
};

// sum.test.ts
import { sum } from './sum';

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3);
});
  • Executar els tests: 
    npx jest
