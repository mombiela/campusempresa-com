Introducció al JSX

JSX (JavaScript XML) és una extensió de sintaxi per a JavaScript, comunament utilitzada amb React per descriure com hauria de ser la interfície d'usuari. TypeScript suporta JSX i proporciona comprovació de tipus i autocompletat per a JSX.

  • Conceptes bàsics de JSX: JSX et permet escriure sintaxi semblant a HTML dins de JavaScript.
  • Integració amb React: JSX s'utilitza principalment amb React per crear components.

Configuració de TypeScript amb JSX

Per utilitzar TypeScript amb JSX, cal configurar correctament el teu projecte TypeScript.

  1. Instal·la TypeScript i React:

    npm install typescript @types/react @types/react-dom

  2. Configura tsconfig.json:

    {
  "compilerOptions": {
    "target": "ES6",
    "module": "commonjs",
    "jsx": "react",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  }
}

Escriure JSX en TypeScript

Exemple bàsic

Aquí tens un exemple senzill d'un component React escrit en TypeScript amb JSX:

import React from 'react';

type GreetingProps = {
  name: string;
};

const Greeting: React.FC<GreetingProps> = ({ name }) => {
  return <h1>Hello, {name}!</h1>;
};

export default Greeting;
  • Interfície de Props: Defineix la forma de les props utilitzant interfícies o tipus de TypeScript.
  • Component Funcional: Utilitza React.FC per tipar el component funcional.

Exemple avançat

Creem un component més complex que inclou estat i maneig d'esdeveniments:

import React, { useState } from 'react';

type CounterProps = {
  initialCount: number;
};

const Counter: React.FC<CounterProps> = ({ initialCount }) => {
  const [count, setCount] = useState(initialCount);

  const increment = () => setCount(count + 1);
  const decrement = () => setCount(count - 1);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={increment}>Increment</button>
      <button onClick={decrement}>Decrement</button>
    </div>
  );
};

export default Counter;
  • Gestió de l'Estat: Utilitza el hook useState per gestionar l'estat.
  • Maneig d'Esdeveniments: TypeScript infereix els tipus per als manejadors d'esdeveniments, assegurant la seguretat de tipus.

Comprovació de Tipus i Autocompletat

TypeScript millora l'experiència de desenvolupament proporcionant comprovació de tipus i autocompletat per a JSX.

  • Seguretat de Tipus: Assegura que els components rebin les props correctes.
  • Autocompletat: Ajuda a escriure JSX proporcionant suggeriments per a les propietats i mètodes dels components.

Exemple amb Comprovació de Tipus

Considera un component que espera una llista d'elements:

import React from 'react';

type ItemListProps = {
  items: string[];
};

const ItemList: React.FC<ItemListProps> = ({ items }) => {
  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
};

export default ItemList;
  • Validació de Props: TypeScript assegura que items és una matriu de cadenes.

Errors Comuns i Resolució de Problemes

Errors Comuns

  • Desajust de Tipus: Passar props de tipus incorrectes.
  • Props Faltants: No proporcionar les props requerides.

Consells per a la Resolució de Problemes

  • Comprova els Tipus: Assegura't que totes les props i variables d'estat estan correctament tipades.
  • Utilitza Paquets @types: Instal·la i utilitza paquets @types per a biblioteques de tercers.

Conclusió

TypeScript amb JSX proporciona una manera robusta de construir aplicacions React amb seguretat de tipus i una experiència de desenvolupament millorada. Aprofitant les característiques de comprovació de tipus i autocompletat de TypeScript, pots detectar errors aviat i escriure codi més mantenible.

  • Punts Clau:
    • TypeScript millora JSX amb seguretat de tipus.
    • Una configuració adequada és essencial per utilitzar TypeScript amb JSX.
    • TypeScript proporciona autocompletat i comprovació de tipus per a JSX, millorant l'experiència de desenvolupament.

Seguint les directrius i exemples proporcionats, pots utilitzar efectivament TypeScript amb JSX per construir aplicacions React escalables i mantenibles.
