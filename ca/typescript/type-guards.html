Els Type Guards a TypeScript
Àlies de tipus ►

TypeScript és un llenguatge potent que es basa en JavaScript afegint tipus estàtics. Una de les característiques clau que fan que TypeScript sigui robust és la seva capacitat per realitzar comprovacions de tipus en temps de compilació. Els Type Guards són una manera d'assegurar que el teu codi es comporti correctament restringint els tipus dins d'un bloc condicional.

Què són els Type Guards?

Els Type Guards són expressions que realitzen comprovacions en temps d'execució per assegurar que una variable és d'un tipus específic. Ajuden TypeScript a entendre el tipus d'una variable dins d'un bloc, permetent una comprovació de tipus més precisa i reduint el risc d'errors en temps d'execució.

Conceptes Clau

  • Restricció de Tipus: El procés de refinar el tipus d'una variable dins d'un bloc condicional.
  • Predicats de Tipus: Funcions que retornen un booleà i s'utilitzen per afirmar el tipus d'una variable.
  • Operador typeof: Utilitzat per comprovar el tipus d'una variable.
  • Operador instanceof: Utilitzat per comprovar si un objecte és una instància d'una classe específica.
  • Type Guards Definits per l'Usuari: Funcions personalitzades que ajuden en la comprovació de tipus.

Utilitzant `typeof` per als Type Guards

L'operador typeof s'utilitza comunament per comprovar tipus primitius com string, number, boolean, etc.

function printId(id: number | string) {
    if (typeof id === "string") {
        console.log(`ID és una cadena: ${id.toUpperCase()}`);
    } else {
        console.log(`ID és un número: ${id.toFixed(2)}`);
    }
}

En aquest exemple, typeof id === "string" restringeix el tipus de id a string dins del bloc, permetent utilitzar mètodes específics de cadena com toUpperCase().

Utilitzant `instanceof` per als Type Guards

L'operador instanceof s'utilitza per comprovar si un objecte és una instància d'una classe específica.

class Dog {
    bark() {
        console.log("Woof!");
    }
}

class Cat {
    meow() {
        console.log("Meow!");
    }
}

function makeSound(animal: Dog | Cat) {
    if (animal instanceof Dog) {
        animal.bark();
    } else {
        animal.meow();
    }
}

Aquí, animal instanceof Dog restringeix el tipus de animal a Dog, permetent cridar el mètode bark() de manera segura.

Type Guards Definits per l'Usuari

Els Type Guards definits per l'usuari són funcions que retornen un predicat de tipus. Són útils per a comprovacions de tipus complexes.

interface Fish {
    swim: () => void;
}

interface Bird {
    fly: () => void;
}

function isFish(pet: Fish | Bird): pet is Fish {
    return (pet as Fish).swim !== undefined;
}

function move(pet: Fish | Bird) {
    if (isFish(pet)) {
        pet.swim();
    } else {
        pet.fly();
    }
}

En aquest exemple, isFish és un Type Guard definit per l'usuari que comprova si pet és un Fish. El tipus de retorn pet is Fish és un predicat de tipus que diu a TypeScript el tipus de pet dins del bloc if.

Conclusió

Els Type Guards són una característica potent a TypeScript que ajuden a assegurar que el teu codi sigui segur en termes de tipus i lliure d'errors en temps d'execució. Utilitzant typeof, instanceof i Type Guards definits per l'usuari, pots restringir els tipus dins de blocs condicionals, fent que el teu codi sigui més robust i fàcil de mantenir.

Entendre i utilitzar efectivament els Type Guards millorarà significativament les teves habilitats de programació en TypeScript, permetent-te escriure aplicacions més segures i fiables.
