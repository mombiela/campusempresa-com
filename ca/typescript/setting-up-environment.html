Introducció

Configurar l'entorn de TypeScript és el primer pas per començar a desenvolupar aplicacions utilitzant TypeScript. Aquesta secció et guiarà a través del procés d'instal·lació de TypeScript, la configuració del teu entorn de desenvolupament i l'escriptura del teu primer programa en TypeScript.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb un sistema operatiu (Windows, macOS o Linux).
  • Node.js instal·lat a la teva màquina. Pots descarregar-lo des de nodejs.org.

Instal·lant TypeScript

Per instal·lar TypeScript, necessites utilitzar npm (Node Package Manager), que ve amb Node.js. Obre el teu terminal o símbol del sistema i executa la següent comanda:

npm install -g typescript

Aquesta comanda instal·la TypeScript globalment al teu sistema, fent que la comanda tsc (TypeScript Compiler) estigui disponible des de qualsevol directori.

Verificant la Instal·lació

Per verificar que TypeScript s'ha instal·lat correctament, executa la següent comanda al teu terminal:

tsc --version

Hauries de veure el número de versió de TypeScript imprès a la pantalla, indicant que la instal·lació ha estat exitosa.

Configurant un Projecte TypeScript

  1. Crear un Directori de Projecte: Comença creant un nou directori per al teu projecte TypeScript.

    mkdir my-typescript-project
cd my-typescript-project

  2. Inicialitzar un Projecte Node.js: Inicialitza un nou projecte Node.js executant:

    npm init -y

    Aquesta comanda crea un fitxer package.json amb configuracions per defecte.

  3. Instal·lar TypeScript Localment: Tot i que hem instal·lat TypeScript globalment, és una bona pràctica instal·lar-lo localment al teu projecte també.

    npm install typescript --save-dev

  4. Crear un Fitxer tsconfig.json: El fitxer tsconfig.json s'utilitza per configurar les opcions del compilador de TypeScript. Crea aquest fitxer a l'arrel del teu directori de projecte:

    tsc --init

    Aquesta comanda genera un fitxer tsconfig.json amb configuracions per defecte.

Configurant tsconfig.json

El fitxer tsconfig.json conté diverses opcions per personalitzar el compilador de TypeScript. Aquí tens algunes opcions clau:

  • target: Especifica la versió de JavaScript a la qual es compilarà el codi TypeScript.
  • module: Defineix el sistema de mòduls a utilitzar (per exemple, CommonJS, ES6).
  • outDir: Especifica el directori de sortida per als fitxers JavaScript compilats.
  • rootDir: Especifica el directori arrel dels fitxers font de TypeScript.

Exemple de tsconfig.json:

{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true
  }
}

Escriure el teu Primer Programa en TypeScript

  1. Crear un Directori de Fonts: Crea un directori anomenat src per emmagatzemar els teus fitxers TypeScript.

    mkdir src

  2. Crear un Fitxer TypeScript: Dins del directori src, crea un fitxer anomenat index.ts.

    // src/index.ts
let message: string = "Hello, TypeScript!";
console.log(message);

  3. Compilar el Fitxer TypeScript: Utilitza el compilador de TypeScript per compilar el teu codi TypeScript a JavaScript.

    tsc

    Aquesta comanda llegeix el fitxer tsconfig.json i compila els fitxers TypeScript al directori src, generant els fitxers JavaScript al directori dist.

  4. Executar el JavaScript Compilat: Utilitza Node.js per executar el fitxer JavaScript compilat.

    node dist/index.js

    Hauries de veure el missatge "Hello, TypeScript!" imprès a la pantalla.

Conclusió

Configurar l'entorn de TypeScript implica instal·lar TypeScript, configurar les opcions del compilador i escriure el teu primer programa en TypeScript. Seguint aquests passos, has establert les bases per desenvolupar aplicacions utilitzant TypeScript. A les següents seccions, aprofundirem en les característiques i capacitats de TypeScript, construint sobre aquesta configuració inicial.
