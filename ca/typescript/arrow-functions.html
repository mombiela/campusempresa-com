Funcions Fletxa
Classes i interfícies ►

Les funcions fletxa són una manera concisa d'escriure funcions en TypeScript. Són sintàcticament més curtes que les expressions de funció tradicionals i no vinculen el seu propi this, arguments, super o new.target. Això les fa particularment útils en certs contextos, com dins de mètodes d'array o quan necessites preservar el context de this.

Conceptes Clau

  • Sintaxi: Les funcions fletxa utilitzen la sintaxi =>.
  • this lèxic: Les funcions fletxa no tenen el seu propi context this; hereten this de l'àmbit pare.
  • Retorn implícit: Si el cos de la funció conté només una única expressió, pots ometre les claus i la paraula clau return.

Sintaxi Bàsica

La sintaxi bàsica d'una funció fletxa és:

(param1, param2, ..., paramN) => expression

Si la funció té només un paràmetre, pots ometre els parèntesis:

param => expression

Per a funcions sense paràmetres, has d'utilitzar parèntesis buits:

() => expression

Exemples

Funció Fletxa Simple

Una funció fletxa simple que suma dos números:

const add = (a: number, b: number): number => a + b;

console.log(add(2, 3)); // Sortida: 5

Funció Fletxa amb Cos de Bloc

Si necessites més d'una única expressió, pots utilitzar un cos de bloc:

const multiply = (a: number, b: number): number => {
    return a * b;
};

console.log(multiply(2, 3)); // Sortida: 6

Funció Fletxa sense Paràmetres

Una funció fletxa que retorna un valor constant:

const getFive = (): number => 5;

console.log(getFive()); // Sortida: 5

`this` Lèxic

Les funcions fletxa no tenen el seu propi context this. En canvi, hereten this de l'àmbit pare. Això és particularment útil en situacions on vols preservar el context de this.

Exemple amb `this` Lèxic

Considera el següent exemple:

class Counter {
    count: number = 0;

    increment() {
        setTimeout(() => {
            this.count++;
            console.log(this.count);
        }, 1000);
    }
}

const counter = new Counter();
counter.increment(); // Sortida després d'1 segon: 1

En aquest exemple, la funció fletxa dins de setTimeout hereta this del mètode increment, que es refereix a la instància de Counter.

Comparació amb Funcions Tradicionals

| Característica | Funció Fletxa | Funció Tradicional | |----------------------|---------------------------------|---------------------------------| | Sintaxi | param => expression | function(param) { return expression; } | | Context this | Lèxic (heretat de l'àmbit pare) | Dinàmic (depèn de com es crida) | | Objecte arguments | No disponible | Disponible | | Constructor | No es pot utilitzar com a constructor | Es pot utilitzar com a constructor |

Exercicis

  1. Convertir a Funció Fletxa: Converteix la següent funció tradicional a una funció fletxa.

    function greet(name: string): string {
    return `Hello, ${name}!`;
}

  2. this Lèxic: Crea una classe amb un mètode que utilitzi una funció fletxa per registrar una propietat de la classe després d'un retard.

  3. Mètodes d'Array: Utilitza una funció fletxa amb el mètode map per crear un nou array on cada element estigui duplicat.

    const numbers = [1, 2, 3, 4];
// El teu codi aquí

Conclusió

Les funcions fletxa proporcionen una manera concisa i poderosa d'escriure funcions en TypeScript. Són particularment útils per preservar el context de this i per escriure funcions curtes d'una sola expressió. Entendre com i quan utilitzar funcions fletxa pot ajudar-te a escriure codi més net i mantenible.
