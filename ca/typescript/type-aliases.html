Introducció als Àlies de Tipus

Els àlies de tipus a TypeScript et permeten crear un nou nom per a un tipus existent. Això pot fer que el teu codi sigui més llegible i fàcil de gestionar, especialment quan treballes amb tipus complexos.

  • Propòsit: Simplificar definicions de tipus complexes i millorar la llegibilitat del codi.
  • Sintaxi: Utilitza la paraula clau type seguida del nom de l'àlies i la definició del tipus.

Ús Bàsic dels Àlies de Tipus

Els àlies de tipus es poden utilitzar per canviar el nom de tipus primitius, tipus d'objectes i fins i tot tipus d'unió.

Tipus Primitius

Pots crear àlies de tipus per a tipus primitius com string, number, etc.

type Name = string;
type Age = number;

let userName: Name = "Alice";
let userAge: Age = 30;

Tipus d'Objectes

Els àlies de tipus són particularment útils per definir tipus d'objectes complexos.

type User = {
  name: string;
  age: number;
  email: string;
};

let user: User = {
  name: "Alice",
  age: 30,
  email: "alice@example.com"
};

Tipus d'Unió

Els àlies de tipus també es poden utilitzar per simplificar tipus d'unió.

type ID = string | number;

let userId: ID = "abc123";
let orderId: ID = 987654;

Ús Avançat dels Àlies de Tipus

Els àlies de tipus es poden combinar amb altres característiques de TypeScript per crear definicions de tipus més potents i flexibles.

Tipus d'Intersecció

Pots utilitzar àlies de tipus per definir tipus d'intersecció, que combinen múltiples tipus en un.

type Person = {
  name: string;
  age: number;
};

type Employee = {
  employeeId: number;
  department: string;
};

type EmployeeDetails = Person & Employee;

let employee: EmployeeDetails = {
  name: "Alice",
  age: 30,
  employeeId: 12345,
  department: "Engineering"
};

Genèrics amb Àlies de Tipus

Els àlies de tipus també poden ser genèrics, permetent-te crear definicions de tipus reutilitzables.

type Response<T> = {
  data: T;
  error: string | null;
};

let successResponse: Response<string> = {
  data: "Operation successful",
  error: null
};

let errorResponse: Response<null> = {
  data: null,
  error: "Something went wrong"
};

Àlies de Tipus vs Interfícies

Tant els àlies de tipus com les interfícies es poden utilitzar per definir tipus d'objectes, però hi ha algunes diferències.

| Característica | Àlies de Tipus | Interfície | |------------------|----------------|------------| | Sintaxi | type | interface| | Extensió de Tipus| Sí, utilitzant intersecció (&) | Sí, utilitzant extends | | Fusió | No | Sí | | Usos | Tipus d'unió, primitius, genèrics | Tipus d'objectes, implementacions de classes |

Comparació d'Exemples

// Utilitzant Àlies de Tipus
type Point = {
  x: number;
  y: number;
};

// Utilitzant Interfície
interface IPoint {
  x: number;
  y: number;
}

let point1: Point = { x: 10, y: 20 };
let point2: IPoint = { x: 15, y: 25 };

Conclusió

Els àlies de tipus a TypeScript proporcionen una manera potent de crear codi més llegible i mantenible permetent-te definir noms personalitzats per a tipus. Són versàtils i es poden utilitzar per a tipus primitius, tipus d'objectes, tipus d'unió, tipus d'intersecció i genèrics. Entendre quan i com utilitzar els àlies de tipus, així com les diferències entre àlies de tipus i interfícies, t'ajudarà a escriure codi TypeScript més net i eficient.
