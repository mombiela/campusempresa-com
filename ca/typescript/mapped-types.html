Tipus Mapejats en TypeScript
Estil de codi i formatació ►

Els tipus mapejats són una característica poderosa en TypeScript que et permet crear nous tipus transformant-ne d'existents. Són particularment útils per crear tipus que són variacions d'altres tipus, com ara fer que totes les propietats siguin opcionals o només de lectura.

Conceptes Clau

  • Tipus Mapejats: Tipus que transformen propietats d'un tipus existent.
  • Tipus d'Utilitat de TypeScript: Tipus mapejats predefinits proporcionats per TypeScript, com ara Partial, Readonly, Record i Pick.

Tipus Mapejats Bàsics

Els tipus mapejats utilitzen l'operador keyof i claudàtors per iterar sobre les propietats d'un tipus existent i transformar-les.

Exemple: Fer que Totes les Propietats Siguin Opcionals

type Person = {
  name: string;
  age: number;
};

type OptionalPerson = {
  [P in keyof Person]?: Person[P];
};

// Equivalent a:
// type OptionalPerson = {
//   name?: string;
//   age?: number;
// };

En aquest exemple, OptionalPerson és un tipus mapejat que fa que totes les propietats de Person siguin opcionals.

Tipus d'Utilitat de TypeScript

TypeScript proporciona diversos tipus d'utilitat que es construeixen utilitzant tipus mapejats. Aquests tipus d'utilitat simplifiquen les transformacions de tipus comunes.

Partial

El tipus d'utilitat Partial fa que totes les propietats d'un tipus siguin opcionals.

type Person = {
  name: string;
  age: number;
};

type PartialPerson = Partial<Person>;

// Equivalent a:
// type PartialPerson = {
//   name?: string;
//   age?: number;
// };

Readonly

El tipus d'utilitat Readonly fa que totes les propietats d'un tipus siguin només de lectura.

type Person = {
  name: string;
  age: number;
};

type ReadonlyPerson = Readonly<Person>;

// Equivalent a:
// type ReadonlyPerson = {
//   readonly name: string;
//   readonly age: number;
// };

Record

El tipus d'utilitat Record construeix un tipus amb un conjunt de propietats K de tipus T.

type PageInfo = {
  title: string;
};

type Page = "home" | "about" | "contact";

type Pages = Record<Page, PageInfo>;

// Equivalent a:
// type Pages = {
//   home: PageInfo;
//   about: PageInfo;
//   contact: PageInfo;
// };

Pick

El tipus d'utilitat Pick construeix un tipus seleccionant un conjunt de propietats K del tipus T.

type Person = {
  name: string;
  age: number;
  address: string;
};

type PersonNameAndAge = Pick<Person, "name" | "age">;

// Equivalent a:
// type PersonNameAndAge = {
//   name: string;
//   age: number;
// };

Tipus Mapejats Personalitzats

Pots crear els teus propis tipus mapejats personalitzats per adaptar-los a necessitats específiques.

Exemple: Fer que Totes les Propietats Siguin Nullable

type Person = {
  name: string;
  age: number;
};

type Nullable<T> = {
  [P in keyof T]: T[P] | null;
};

type NullablePerson = Nullable<Person>;

// Equivalent a:
// type NullablePerson = {
//   name: string | null;
//   age: number | null;
// };

Conclusió

Els tipus mapejats en TypeScript proporcionen una manera flexible de transformar tipus existents en nous. Entenent i utilitzant tipus mapejats, pots crear definicions de tipus més dinàmiques i reutilitzables. Els tipus d'utilitat integrats de TypeScript com Partial, Readonly, Record i Pick són exemples pràctics de com els tipus mapejats poden simplificar les transformacions de tipus. Els tipus mapejats personalitzats amplien encara més aquesta capacitat, permetent-te adaptar les transformacions de tipus a les teves necessitats específiques.
