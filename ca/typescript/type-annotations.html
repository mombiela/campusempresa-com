Anotacions de Tipus en TypeScript
Compilació de TypeScript ►

Les anotacions de tipus són una característica poderosa en TypeScript que et permet especificar els tipus de variables, paràmetres de funció i valors de retorn. Això ajuda a detectar errors d'hora i fa que el teu codi sigui més llegible i mantenible.

Què són les Anotacions de Tipus?

Les anotacions de tipus són una manera de declarar explícitament el tipus d'una variable, paràmetre de funció o valor de retorn. Això ajuda a TypeScript a entendre quin tipus de dades s'està utilitzant i pot prevenir molts errors comuns de programació.

Conceptes Clau

  • Inferència de Tipus: TypeScript sovint pot inferir el tipus d'una variable basant-se en el seu valor.
  • Anotacions Explícites: Pots declarar explícitament el tipus d'una variable, paràmetre de funció o valor de retorn.
  • Tipus Bàsics: Els tipus comuns inclouen string, number, boolean, array, tuple, enum, any, void, null i undefined.

Anotacions de Tipus Bàsiques

Variables

Pots anotar variables amb un tipus específic utilitzant un dos punts seguit del nom del tipus.

let username: string = "JohnDoe";
let age: number = 30;
let isAdmin: boolean = true;

Funcions

Anota els paràmetres de funció i els tipus de retorn per assegurar-te que s'utilitzen correctament.

function add(a: number, b: number): number {
    return a + b;
}

let result: number = add(5, 3);

Arrays

Anota arrays especificant el tipus d'elements que contenen.

let numbers: number[] = [1, 2, 3, 4, 5];
let names: string[] = ["Alice", "Bob", "Charlie"];

Anotacions de Tipus Avançades

Tuples

Les tuples són arrays amb un nombre fix d'elements de tipus específics.

let person: [string, number] = ["Alice", 25];

Enums

Els enums et permeten definir un conjunt de constants amb nom.

enum Color {
    Red,
    Green,
    Blue
}

let favoriteColor: Color = Color.Green;

Any

El tipus any pot contenir qualsevol valor, però s'ha d'utilitzar amb moderació ja que evita la comprovació de tipus.

let randomValue: any = 42;
randomValue = "Hello";
randomValue = true;

Void

El tipus void s'utilitza per a funcions que no retornen cap valor.

function logMessage(message: string): void {
    console.log(message);
}

Null i Undefined

Pots anotar explícitament variables com null o undefined.

let emptyValue: null = null;
let notAssigned: undefined = undefined;

Anotacions de Tipus en la Pràctica

Exemple: Perfil d'Usuari

Creem un perfil d'usuari simple amb anotacions de tipus.

type UserProfile = {
    username: string;
    age: number;
    isAdmin: boolean;
};

let user: UserProfile = {
    username: "JohnDoe",
    age: 30,
    isAdmin: true
};

function printUserProfile(user: UserProfile): void {
    console.log(`Username: ${user.username}, Age: ${user.age}, Admin: ${user.isAdmin}`);
}

printUserProfile(user);

Exemple: Carret de la Compra

Un exemple més complex que implica arrays i funcions.

type Product = {
    id: number;
    name: string;
    price: number;
};

let cart: Product[] = [];

function addToCart(product: Product): void {
    cart.push(product);
}

function calculateTotal(cart: Product[]): number {
    return cart.reduce((total, product) => total + product.price, 0);
}

addToCart({ id: 1, name: "Laptop", price: 999.99 });
addToCart({ id: 2, name: "Mouse", price: 19.99 });

console.log(`Total: $${calculateTotal(cart)}`);

Conclusió

Les anotacions de tipus en TypeScript proporcionen una manera robusta d'assegurar que el teu codi sigui segur en termes de tipus i més fàcil d'entendre. Declarant explícitament els tipus de variables, paràmetres de funció i valors de retorn, pots detectar errors d'hora i fer que el teu codi sigui més mantenible. A mesura que et familiaritzis amb TypeScript, trobaràs que les anotacions de tipus són una eina inestimable en la teva caixa d'eines de programació.
