Introducció als Hooks

Els hooks són funcions que et permeten "enganxar-te" a les característiques d'estat i cicle de vida de React des dels components funcionals. Van ser introduïts a React 16.8 i s'han convertit en una part fonamental del desenvolupament modern de React. Quan utilitzes TypeScript, pots aprofitar el seu sistema de tipus per fer els teus hooks més robustos i mantenibles.

Hooks bàsics amb TypeScript

useState

El hook useState et permet afegir estat als teus components funcionals. Aquí tens com pots utilitzar-lo amb TypeScript:

import React, { useState } from 'react';

const Counter: React.FC = () => {
  const [count, setCount] = useState<number>(0);

  return (
    <div>
      <p>Has fet clic {count} vegades</p>
      <button onClick={() => setCount(count + 1)}>Fes clic aquí</button>
    </div>
  );
};

export default Counter;
  • Conceptes clau:
    • useState pren un estat inicial i retorna una matriu amb l'estat actual i una funció per actualitzar-lo.
    • TypeScript pot inferir el tipus a partir de l'estat inicial, però també pots definir-lo explícitament.

useEffect

El hook useEffect et permet realitzar efectes secundaris en components funcionals. Aquí tens un exemple amb TypeScript:

import React, { useState, useEffect } from 'react';

const Timer: React.FC = () => {
  const [seconds, setSeconds] = useState<number>(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setSeconds(prevSeconds => prevSeconds + 1);
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return <div>Segons: {seconds}</div>;
};

export default Timer;
  • Conceptes clau:
    • useEffect s'executa després del renderitzat i pot opcionalment netejar-se després d'ell mateix.
    • El segon argument de useEffect és una matriu de dependències que controla quan s'executa l'efecte.

Hooks avançats amb TypeScript

useReducer

El hook useReducer és útil per gestionar lògica d'estat complexa. Aquí tens un exemple:

import React, { useReducer } from 'react';

interface State {
  count: number;
}

type Action = { type: 'increment' } | { type: 'decrement' };

const initialState: State = { count: 0 };

function reducer(state: State, action: Action): State {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}

const Counter: React.FC = () => {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Compte: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
    </div>
  );
};

export default Counter;
  • Conceptes clau:
    • useReducer és similar a useState però és més adequat per a lògica d'estat complexa.
    • La funció reductora pren l'estat actual i una acció, i retorna un nou estat.

Hooks personalitzats

Els hooks personalitzats et permeten extreure i reutilitzar lògica a través de múltiples components. Aquí tens un exemple:

import { useState, useEffect } from 'react';

function useFetch<T>(url: string): { data: T | null, loading: boolean, error: Error | null } {
  const [data, setData] = useState<T | null>(null);
  const [loading, setLoading] = useState<boolean>(true);
  const [error, setError] = useState<Error | null>(null);

  useEffect(() => {
    fetch(url)
      .then(response => response.json())
      .then(data => {
        setData(data);
        setLoading(false);
      })
      .catch(error => {
        setError(error);
        setLoading(false);
      });
  }, [url]);

  return { data, loading, error };
}

export default useFetch;
  • Conceptes clau:
    • Els hooks personalitzats són funcions JavaScript els noms de les quals comencen amb "use".
    • Poden cridar altres hooks i encapsular lògica reutilitzable.

Conclusió

Els hooks són una característica poderosa de React que et permeten utilitzar l'estat i altres característiques de React sense escriure una classe. Quan es combinen amb TypeScript, esdevenen encara més poderosos proporcionant seguretat de tipus i millor suport d'eines. Entenent i utilitzant hooks com useState, useEffect, useReducer i hooks personalitzats, pots escriure aplicacions React més mantenibles i robustes.
