Arrays i tuples a TypeScript
Enumeracions ►

Introducció

En aquesta secció, explorarem els arrays i les tuples a TypeScript. Els arrays i les tuples són estructures de dades fonamentals que et permeten emmagatzemar col·leccions de valors. Entendre com utilitzar-les de manera efectiva és crucial per a qualsevol programador de TypeScript.

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors en una sola variable. A TypeScript, els arrays poden ser d'un tipus específic o una unió de tipus.

Declarar arrays

Pots declarar arrays a TypeScript utilitzant dues sintaxis:

  1. Utilitzant claudàtors ([]):

    let numbers: number[] = [1, 2, 3, 4, 5];
let strings: string[] = ["apple", "banana", "cherry"];

  2. Utilitzant el tipus genèric Array:

    let numbers: Array<number> = [1, 2, 3, 4, 5];
let strings: Array<string> = ["apple", "banana", "cherry"];

Accedir als elements d'un array

Pots accedir als elements d'un array utilitzant el seu índex, començant per 0:

let fruits: string[] = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Sortida: apple
console.log(fruits[1]); // Sortida: banana

Mètodes d'array

Els arrays de TypeScript venen amb una varietat de mètodes integrats. Aquí hi ha alguns dels més utilitzats:

  • push(): Afegeix un o més elements al final d'un array.

    let numbers: number[] = [1, 2, 3];
numbers.push(4);
console.log(numbers); // Sortida: [1, 2, 3, 4]

  • pop(): Elimina l'últim element d'un array.

    let numbers: number[] = [1, 2, 3];
numbers.pop();
console.log(numbers); // Sortida: [1, 2]

  • map(): Crea un nou array amb els resultats de cridar una funció proporcionada a cada element.

    let numbers: number[] = [1, 2, 3];
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Sortida: [2, 4, 6]

  • filter(): Crea un nou array amb tots els elements que passen la prova implementada per la funció proporcionada.

    let numbers: number[] = [1, 2, 3, 4, 5];
let even = numbers.filter(num => num % 2 === 0);
console.log(even); // Sortida: [2, 4]

Tuples

Les tuples són un tipus especial d'array on el tipus de cada element és conegut. Et permeten expressar un array amb un nombre fix d'elements els tipus dels quals són coneguts però poden ser diferents.

Declarar tuples

Pots declarar tuples especificant el tipus de cada element a l'array:

let person: [string, number] = ["Alice", 30];

Accedir als elements d'una tuple

Pots accedir als elements d'una tuple igual que als arrays, utilitzant el seu índex:

let person: [string, number] = ["Alice", 30];
console.log(person[0]); // Sortida: Alice
console.log(person[1]); // Sortida: 30

Mètodes de tuple

Les tuples suporten els mateixos mètodes que els arrays, però has de tenir cura amb els tipus:

let person: [string, number] = ["Alice", 30];
person.push("Engineer");
console.log(person); // Sortida: ["Alice", 30, "Engineer"]

Nota: Afegir elements més enllà dels tipus declarats pot conduir a comportaments inesperats.

Comparació: Arrays vs Tuples

| Característica | Arrays | Tuples | |------------------|---------------------------------|---------------------------------| | Longitud | Pot variar | Fixa | | Tipus d'elements | Homogeni o Unió de Tipus | Heterogeni | | Cas d'ús | Llistes d'elements similars | Estructura fixa amb diferents tipus |

Conclusió

Els arrays i les tuples són estructures de dades essencials a TypeScript. Els arrays són flexibles i poden contenir múltiples valors del mateix tipus, mentre que les tuples tenen una longitud fixa i poden contenir valors de diferents tipus. Entendre com utilitzar aquestes estructures de manera efectiva t'ajudarà a escriure codi TypeScript més robust i segur en termes de tipus.
