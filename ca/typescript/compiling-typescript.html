Introducció

TypeScript és un superconjunt de JavaScript amb tipatge estàtic que es compila a JavaScript pla. Entendre com compilar TypeScript és crucial per aprofitar les seves potents funcions de verificació de tipus i assegurar que el teu codi s'executi en qualsevol entorn JavaScript.

Conceptes Clau

  • Compilador TypeScript (tsc): El compilador TypeScript (tsc) és una eina de línia de comandes que compila fitxers TypeScript (.ts) en fitxers JavaScript (.js).
  • Fitxer de Configuració (tsconfig.json): Un fitxer de configuració que especifica els fitxers arrel i les opcions del compilador necessàries per compilar el projecte.

Instal·lant TypeScript

Abans de poder compilar TypeScript, cal instal·lar-lo. Pots instal·lar TypeScript globalment utilitzant npm (Node Package Manager).

npm install -g typescript

Verifica la instal·lació comprovant la versió:

tsc --version

Compilant un Sol Fitxer

Per compilar un sol fitxer TypeScript, utilitza la comanda tsc seguida del nom del fitxer.

tsc filename.ts

Aquesta comanda generarà un fitxer JavaScript corresponent (filename.js) en el mateix directori.

Exemple

Considera el següent fitxer TypeScript example.ts:

let message: string = "Hello, TypeScript!";
console.log(message);

Compila'l utilitzant:

tsc example.ts

Això produirà example.js:

var message = "Hello, TypeScript!";
console.log(message);

Utilitzant tsconfig.json

Per a projectes més grans, gestionar les opcions de compilació mitjançant el fitxer tsconfig.json és més eficient. Aquest fitxer et permet especificar diverses opcions del compilador i els fitxers que s'han d'incloure en la compilació.

Creant tsconfig.json

Pots crear un fitxer tsconfig.json utilitzant la següent comanda:

tsc --init

Aquesta comanda genera un fitxer tsconfig.json per defecte amb diverses opcions comentades.

Exemple de tsconfig.json

Aquí tens un exemple senzill d'un fitxer tsconfig.json:

{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "outDir": "./dist",
    "strict": true
  },
  "include": ["src/**/*"]
}
  • target: Especifica la versió de JavaScript a la qual es compilarà.
  • module: Especifica el sistema de mòduls a utilitzar.
  • outDir: Especifica el directori de sortida per als fitxers JavaScript compilats.
  • strict: Habilita totes les opcions de verificació estricta de tipus.
  • include: Especifica els fitxers o directoris a incloure en la compilació.

Compilant amb tsconfig.json

Un cop tinguis un fitxer tsconfig.json, simplement executa:

tsc

Aquesta comanda compilarà tots els fitxers especificats a la secció include de tsconfig.json utilitzant les opcions definides a compilerOptions.

Opcions Comunes del Compilador

Aquí tens algunes opcions del compilador comunament utilitzades:

| Opció | Descripció | |-----------------|-----------------------------------------------------------------------------| | target | Especifica la versió ECMAScript objectiu (p. ex., es5, es6, es2015). | | module | Especifica el sistema de mòduls (p. ex., commonjs, amd, es6). | | outDir | Especifica el directori de sortida per als fitxers compilats. | | rootDir | Especifica el directori arrel dels fitxers d'entrada. | | strict | Habilita totes les opcions de verificació estricta de tipus. | | noImplicitAny | Genera un error en expressions i declaracions amb un tipus any implícit. | | sourceMap | Genera fitxers .map corresponents per a depuració. |

Conclusió

Compilar TypeScript és un procés senzill que es pot gestionar utilitzant el compilador TypeScript (tsc). Per a projectes més grans, utilitzar un fitxer tsconfig.json simplifica la gestió de les opcions del compilador i els fitxers inclosos. Entendre aquests conceptes bàsics t'ajudarà a compilar i gestionar eficientment els teus projectes TypeScript, assegurant un codi JavaScript robust i amb tipatge segur.
