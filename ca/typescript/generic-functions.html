Funcions Genèriques en TypeScript
Classes genèriques ►

Les genèriques en TypeScript et permeten crear components reutilitzables que funcionen amb una varietat de tipus en lloc d'un de sol. Aquest tema cobrirà els conceptes bàsics de les funcions genèriques, la seva sintaxi i exemples pràctics per ajudar-te a entendre com utilitzar-les de manera efectiva.

Introducció a les Genèriques

Les genèriques proporcionen una manera de crear components flexibles i reutilitzables. Et permeten definir funcions, classes i interfícies que funcionen amb una varietat de tipus sense sacrificar la seguretat de tipus.

Conceptes Clau

  • Variables de Tipus: Marcadors de posició per a tipus que pots especificar quan utilitzes la funció genèrica.
  • Inferència de Tipus: TypeScript sovint pot inferir els arguments de tipus que passes a una funció genèrica.
  • Restriccions: Pots restringir els tipus que es poden utilitzar amb les genèriques.

Sintaxi Bàsica de les Funcions Genèriques

Una funció genèrica utilitza variables de tipus per definir els tipus amb els quals pot treballar. Aquestes variables de tipus es especifiquen dins de claudàtors (<>) després del nom de la funció.

function identity<T>(arg: T): T {
    return arg;
}

En aquest exemple:

  • T és una variable de tipus.
  • identity és una funció genèrica que pren un argument arg de tipus T i retorna un valor de tipus T.

Utilitzant Funcions Genèriques

Pots cridar una funció genèrica amb diferents tipus:

let output1 = identity<string>("Hola, TypeScript!");
let output2 = identity<number>(42);

TypeScript també pot inferir l'argument de tipus:

let output3 = identity("Hola, TypeScript!"); // inferit com a string
let output4 = identity(42); // inferit com a number

Funcions Genèriques amb Múltiples Variables de Tipus

Pots definir funcions genèriques amb múltiples variables de tipus:

function pair<T, U>(first: T, second: U): [T, U] {
    return [first, second];
}

let pair1 = pair<string, number>("Edat", 30);
let pair2 = pair<boolean, string>(true, "Èxit");

Restriccions en les Genèriques

De vegades, necessites restringir els tipus que es poden utilitzar amb una funció genèrica. Pots aconseguir això utilitzant restriccions.

interface Lengthwise {
    length: number;
}

function logLength<T extends Lengthwise>(arg: T): T {
    console.log(arg.length);
    return arg;
}

logLength("Hola, TypeScript!"); // Funciona, string té una propietat length
logLength([1, 2, 3]); // Funciona, array té una propietat length
// logLength(42); // Error: number no té una propietat length

Exemples Pràctics

Funció Genèrica per a Operacions amb Arrays

Un cas d'ús comú per a les genèriques és crear funcions d'utilitat per a arrays.

function getFirstElement<T>(arr: T[]): T | undefined {
    return arr[0];
}

let firstString = getFirstElement<string>(["poma", "plàtan", "cirera"]);
let firstNumber = getFirstElement<number>([1, 2, 3, 4, 5]);

Funció Genèrica amb Tipus per Defecte

Pots proporcionar un tipus per defecte per a una funció genèrica.

function createArray<T = string>(length: number, value: T): T[] {
    return Array(length).fill(value);
}

let stringArray = createArray(3, "per defecte"); // Inferit com a string[]
let numberArray = createArray<number>(3, 42); // Especificat explícitament com a number[]

Conclusió

Les funcions genèriques en TypeScript són una característica poderosa que et permet escriure codi flexible i reutilitzable. Entenent la sintaxi bàsica, com utilitzar múltiples variables de tipus i com aplicar restriccions, pots crear funcions robustes que funcionen amb una varietat de tipus. Practica utilitzant genèriques en diferents escenaris per sentir-te més còmode amb aquest concepte i millorar les teves habilitats de programació en TypeScript.
