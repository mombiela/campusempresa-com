Exportar i Importar Mòduls
Espais de noms ►

En aquesta secció, explorarem com exportar i importar mòduls en TypeScript. Els mòduls són una manera d'organitzar i encapsular codi, fent-lo més fàcil de gestionar i reutilitzar. Entendre com treballar amb mòduls és crucial per construir aplicacions escalables i mantenibles.

Conceptes Clau

  • Mòduls: Un mòdul és un fitxer que conté codi TypeScript. Els mòduls poden exportar classes, funcions, variables i interfícies per ser utilitzades en altres mòduls.
  • Exportar: Fer que el codi estigui disponible per ser importat en altres mòduls.
  • Importar: Portar codi d'altres mòduls per utilitzar-lo en el mòdul actual.
  • Exportacions per Defecte: Una única exportació d'un mòdul que es pot importar sense claus.
  • Exportacions Nomenades: Múltiples exportacions d'un mòdul que s'han d'importar utilitzant claus.

Exportar Mòduls

Exportacions Nomenades

Les exportacions nomenades permeten exportar múltiples elements d'un mòdul. Cada element s'ha d'importar utilitzant el seu nom exacte.

// mathUtils.ts
export const add = (a: number, b: number): number => {
    return a + b;
};

export const subtract = (a: number, b: number): number => {
    return a - b;
};

Exportacions per Defecte

Les exportacions per defecte permeten exportar un únic element d'un mòdul. Aquest element es pot importar sense utilitzar claus.

// logger.ts
const log = (message: string): void => {
    console.log(message);
};

export default log;

Importar Mòduls

Importar Exportacions Nomenades

Per importar exportacions nomenades, utilitzes claus i els noms exactes de les exportacions.

// app.ts
import { add, subtract } from './mathUtils';

console.log(add(5, 3)); // Sortida: 8
console.log(subtract(5, 3)); // Sortida: 2

Importar Exportacions per Defecte

Per importar exportacions per defecte, simplement utilitzes el nom que vols assignar al mòdul importat.

// app.ts
import log from './logger';

log('Aquest és un missatge de registre'); // Sortida: Aquest és un missatge de registre

Importar Totes les Exportacions

Pots importar totes les exportacions d'un mòdul utilitzant la sintaxi * as. Això és útil quan vols agrupar totes les exportacions sota un únic espai de noms.

// app.ts
import * as MathUtils from './mathUtils';

console.log(MathUtils.add(5, 3)); // Sortida: 8
console.log(MathUtils.subtract(5, 3)); // Sortida: 2

Combinació d'Exportacions per Defecte i Nomenades

Un mòdul pot tenir tant exportacions per defecte com nomenades. Pots importar-les juntes com es mostra a continuació:

// utils.ts
export const multiply = (a: number, b: number): number => {
    return a * b;
};

const divide = (a: number, b: number): number => {
    return a / b;
};

export default divide;
// app.ts
import divide, { multiply } from './utils';

console.log(multiply(6, 3)); // Sortida: 18
console.log(divide(6, 3)); // Sortida: 2

Conclusió

Entendre com exportar i importar mòduls en TypeScript és essencial per escriure codi modular, mantenible i reutilitzable. Utilitzant exportacions nomenades, exportacions per defecte i tècniques d'importació, pots gestionar efectivament les dependències i organitzar el teu codi. Practica aquests conceptes amb els exemples i exercicis proporcionats per consolidar la teva comprensió.
