Introducció

Configurar un projecte React amb TypeScript pot semblar intimidatori al principi, però amb les eines i passos adequats, esdevé senzill. Aquesta secció et guiarà a través del procés de crear un nou projecte React amb TypeScript des de zero, assegurant que entenguis cada pas del camí.

Prerequisits

Abans de començar, assegura't de tenir instal·lat al teu ordinador:

  • Node.js (es recomana la versió LTS)
  • npm (ve amb Node.js) o Yarn

Guia Pas a Pas

1. Crear un Nou Projecte React

Primer, necessitem crear un nou projecte React. Podem utilitzar Create React App (CRA) amb la plantilla TypeScript per a aquest propòsit.

npx create-react-app my-typescript-app --template typescript

Aquesta comanda crearà un nou directori anomenat my-typescript-app amb un projecte React configurat per utilitzar TypeScript.

2. Navegar al Directori del Projecte

Mou-te al nou directori del projecte:

cd my-typescript-app

3. Entendre l'Estructura del Projecte

El teu directori del projecte hauria de semblar-se a això:

my-typescript-app/
├── node_modules/
├── public/
├── src/
│   ├── App.css
│   ├── App.test.tsx
│   ├── App.tsx
│   ├── index.css
│   ├── index.tsx
│   ├── logo.svg
│   ├── react-app-env.d.ts
│   └── setupTests.ts
├── package.json
├── tsconfig.json
└── README.md
  • src/: Conté els teus components de TypeScript i React.
  • tsconfig.json: Fitxer de configuració de TypeScript.
  • package.json: Llista les dependències i scripts.

4. Configuració de TypeScript

El fitxer tsconfig.json és crucial per als projectes de TypeScript. Aquí tens una visió general bàsica del seu contingut:

{
  "compilerOptions": {
    "target": "es5",
    "lib": ["dom", "dom.iterable", "esnext"],
    "allowJs": true,
    "skipLibCheck": true,
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true,
    "strict": true,
    "forceConsistentCasingInFileNames": true,
    "noFallthroughCasesInSwitch": true,
    "module": "esnext",
    "moduleResolution": "node",
    "resolveJsonModule": true,
    "isolatedModules": true,
    "noEmit": true,
    "jsx": "react-jsx"
  },
  "include": ["src"]
}
  • target: Especifica la versió objectiu d'ECMAScript.
  • lib: Especifica els fitxers de biblioteca a incloure.
  • strict: Habilita totes les opcions de comprovació estricta de tipus.

5. Escriure el Teu Primer Component TypeScript

Creem un component TypeScript senzill.

Crea un nou fitxer src/Hello.tsx:

import React from 'react';

interface HelloProps {
  name: string;
}

const Hello: React.FC<HelloProps> = ({ name }) => {
  return <h1>Hello, {name}!</h1>;
};

export default Hello;
  • HelloProps: Defineix les propietats per al component.
  • React.FC<HelloProps>: Especifica que Hello és un component funcional amb propietats del tipus HelloProps.

6. Utilitzar el Component

Ara, utilitzem el nostre component Hello a App.tsx.

import React from 'react';
import Hello from './Hello';

function App() {
  return (
    <div className="App">
      <Hello name="World" />
    </div>
  );
}

export default App;

7. Executar el Projecte

Per veure el teu projecte en acció, executa:

npm start

Això iniciarà el servidor de desenvolupament i obrirà el teu projecte en el navegador web per defecte.

Conclusió

Configurar un projecte React amb TypeScript implica crear un nou projecte amb Create React App, entendre l'estructura del projecte, configurar TypeScript i escriure el teu primer component TypeScript. Amb aquests passos, ara tens una base sòlida per construir aplicacions més complexes utilitzant TypeScript i React.
