Introducció als Fitxers de Declaració

Els fitxers de declaració en TypeScript s'utilitzen per descriure la forma del codi JavaScript al compilador de TypeScript. Permeten que TypeScript entengui els tipus d'objectes, funcions i altres entitats en biblioteques JavaScript, habilitant la comprovació de tipus i el suport d'IntelliSense.

  • Propòsit: Proporcionar informació de tipus sobre el codi JavaScript.
  • Extensió de Fitxer: .d.ts

Quan Utilitzar Fitxers de Declaració

  • Utilitzant Biblioteques JavaScript: Quan s'utilitzen biblioteques JavaScript de tercers que no tenen suport integrat per TypeScript.
  • Interoperabilitat: Per assegurar una interoperabilitat suau entre bases de codi TypeScript i JavaScript.
  • Seguretat de Tipus: Per millorar la seguretat de tipus i l'experiència del desenvolupador amb un millor suport d'eines.

Creant Fitxers de Declaració

Estructura Bàsica

Un fitxer de declaració típicament conté declaracions de tipus per a variables, funcions, classes i mòduls. Aquí teniu un exemple bàsic:

// myLibrary.d.ts
declare module 'myLibrary' {
  export function greet(name: string): void;
  export const version: string;
}

Declarant Variables i Funcions

  • Variables: Utilitzeu la paraula clau declare per declarar variables.
  • Funcions: Utilitzeu la paraula clau declare per declarar funcions amb els seus paràmetres i tipus de retorn.
// variables.d.ts
declare const API_URL: string;
declare let isProduction: boolean;

// functions.d.ts
declare function add(a: number, b: number): number;
declare function log(message: string): void;

Declarant Classes

Les classes es poden declarar amb les seves propietats i mètodes:

// classes.d.ts
declare class Person {
  name: string;
  age: number;
  constructor(name: string, age: number);
  greet(): void;
}

Declarant Mòduls

Els mòduls es poden declarar per encapsular tipus i valors:

// modules.d.ts
declare module 'myModule' {
  export interface User {
    id: number;
    name: string;
  }
  export function getUser(id: number): User;
}

Fitxers de Declaració Avançats

Fusionant Declaracions

TypeScript permet fusionar múltiples declaracions per estendre tipus o mòduls existents:

// merging.d.ts
interface Window {
  customProperty: string;
}

declare module 'myLibrary' {
  export function newFunction(): void;
}

Declaracions Globals

Les declaracions globals s'utilitzen per definir tipus que estan disponibles globalment a tot el projecte:

// globals.d.ts
declare global {
  interface String {
    toTitleCase(): string;
  }
}

Utilitzant Paquets `@types`

Moltes biblioteques JavaScript populars tenen definicions de tipus disponibles a través de l'àmbit @types a npm:

npm install @types/lodash

Aquests es poden utilitzar directament sense necessitat d'escriure els vostres propis fitxers de declaració.

Exemples Pràctics

Exemple: Utilitzant un Fitxer de Declaració per a una Biblioteca JavaScript

Suposem que esteu utilitzant una biblioteca JavaScript anomenada awesomeLib:

// awesomeLib.d.ts
declare module 'awesomeLib' {
  export function awesomeFunction(param: string): number;
  export interface AwesomeInterface {
    id: number;
    name: string;
  }
}

En el vostre codi TypeScript, ara podeu utilitzar awesomeLib amb seguretat de tipus:

import { awesomeFunction, AwesomeInterface } from 'awesomeLib';

const result: number = awesomeFunction('test');
const obj: AwesomeInterface = { id: 1, name: 'Awesome' };

Exemple: Estenent un Objecte Global

Estenent l'objecte Window per incloure una propietat personalitzada:

// globalExtensions.d.ts
declare global {
  interface Window {
    customProperty: string;
  }
}

// usage.ts
window.customProperty = 'Hello, World!';
console.log(window.customProperty);

Resum

Els fitxers de declaració són una característica poderosa en TypeScript que proporcionen informació de tipus per al codi JavaScript, millorant la seguretat de tipus i l'experiència del desenvolupador. Són essencials quan es treballa amb biblioteques JavaScript i per assegurar una interoperabilitat suau entre TypeScript i JavaScript. Entenent com crear i utilitzar fitxers de declaració, podeu aprofitar tot el potencial de TypeScript en els vostres projectes.
