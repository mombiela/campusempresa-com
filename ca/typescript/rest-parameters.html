Introducció als Paràmetres Rest

Els paràmetres rest són una característica de TypeScript que permet a una funció acceptar un nombre indefinit d'arguments com a array. Això és particularment útil quan no saps quants arguments es passaran a una funció. Els paràmetres rest es denoten amb tres punts ... seguits del nom de l'array que contindrà els arguments.

Conceptes Clau

  • Sintaxi: Els paràmetres rest es declaren prefixant el nom del paràmetre amb tres punts (...).
  • Anotació de Tipus: El tipus dels paràmetres rest és sempre un tipus d'array.
  • Ús: Els paràmetres rest han de ser l'últim paràmetre en la signatura de la funció.

Sintaxi i Ús

Sintaxi Bàsica

La sintaxi bàsica per als paràmetres rest és la següent:

function functionName(...restParameter: Type[]): ReturnType {
    // Cos de la funció
}

Exemple: Sumar Nombres

Vegem un exemple senzill on sumem un nombre indefinit de nombres:

function sum(...numbers: number[]): number {
    return numbers.reduce((acc, curr) => acc + curr, 0);
}

console.log(sum(1, 2, 3)); // Sortida: 6
console.log(sum(4, 5, 6, 7)); // Sortida: 22
  • Explicació:
    • ...numbers: number[] declara un paràmetre rest anomenat numbers que serà un array de nombres.
    • El mètode reduce s'utilitza per sumar tots els nombres de l'array.

Exemple: Concatenar Cadenes

Un altre exemple és concatenar múltiples cadenes:

function concatenateStrings(...strings: string[]): string {
    return strings.join(' ');
}

console.log(concatenateStrings('Hello', 'World')); // Sortida: "Hello World"
console.log(concatenateStrings('TypeScript', 'is', 'fun')); // Sortida: "TypeScript is fun"
  • Explicació:
    • ...strings: string[] declara un paràmetre rest anomenat strings que serà un array de cadenes.
    • El mètode join s'utilitza per concatenar totes les cadenes de l'array amb un espai com a separador.

Ús Avançat

Combinació de Paràmetres Rest amb Altres Paràmetres

Els paràmetres rest es poden combinar amb altres paràmetres, però han de ser l'últim paràmetre:

function greet(greeting: string, ...names: string[]): string {
    return `${greeting}, ${names.join(' and ')}!`;
}

console.log(greet('Hello', 'Alice', 'Bob')); // Sortida: "Hello, Alice and Bob!"
console.log(greet('Hi', 'Charlie', 'Dave', 'Eve')); // Sortida: "Hi, Charlie and Dave and Eve!"
  • Explicació:
    • greeting: string és un paràmetre regular.
    • ...names: string[] és un paràmetre rest que recull tots els arguments addicionals en un array.

Errors Comuns

  • Ordre dels Paràmetres: Els paràmetres rest han de ser sempre l'últim paràmetre en la signatura de la funció.
  • Consistència de Tipus: Tots els elements de l'array de paràmetres rest han de ser del mateix tipus.

Conclusió

Els paràmetres rest a TypeScript proporcionen una manera flexible de gestionar funcions que accepten un nombre variable d'arguments. Utilitzant paràmetres rest, pots escriure funcions més genèriques i reutilitzables. Recorda que els paràmetres rest han de ser l'últim paràmetre en la signatura de la funció i han de tenir un tipus consistent.

Dominant els paràmetres rest, pots millorar les teves funcions de TypeScript per ser més adaptables i potents, fent el teu codi més robust i mantenible.
