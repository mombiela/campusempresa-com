Introducció als Genèrics

Els genèrics proporcionen una manera de crear components reutilitzables. Permeten definir un component que pot treballar amb una varietat de tipus de dades en lloc d'un sol. Això és especialment útil en TypeScript per crear codi flexible i reutilitzable.

Per què Utilitzar Genèrics amb Interfícies?

  • Reutilització: Crear interfícies que poden manejar múltiples tipus de dades.
  • Seguretat de Tipus: Assegurar que els tipus siguin consistents a tot el codi.
  • Flexibilitat: Permetre que funcions i classes operin amb diferents tipus de dades sense perdre informació de tipus.

Definició d'una Interfície Genèrica

Una interfície genèrica es pot definir utilitzant la sintaxi <T>, on T és un marcador de posició per al tipus que es especificarà més tard.

Exemple: Interfície Genèrica Bàsica

interface Box<T> {
    contents: T;
}

let stringBox: Box<string> = { contents: "Hola, Món!" };
let numberBox: Box<number> = { contents: 42 };

console.log(stringBox.contents); // Sortida: Hola, Món!
console.log(numberBox.contents); // Sortida: 42
  • Explicació:
    • Box<T> és una interfície genèrica on T pot ser qualsevol tipus.
    • stringBox és una instància de Box amb T com string.
    • numberBox és una instància de Box amb T com number.

Ús de Múltiples Paràmetres de Tipus

Es poden definir interfícies amb múltiples paràmetres de tipus per manejar escenaris més complexos.

Exemple: Interfície Genèrica amb Múltiples Paràmetres de Tipus

interface Pair<K, V> {
    key: K;
    value: V;
}

let numberStringPair: Pair<number, string> = { key: 1, value: "Un" };
let stringBooleanPair: Pair<string, boolean> = { key: "isActive", value: true };

console.log(numberStringPair.key, numberStringPair.value); // Sortida: 1 Un
console.log(stringBooleanPair.key, stringBooleanPair.value); // Sortida: isActive true
  • Explicació:
    • Pair<K, V> és una interfície genèrica amb dos paràmetres de tipus K i V.
    • numberStringPair és una instància de Pair amb K com number i V com string.
    • stringBooleanPair és una instància de Pair amb K com string i V com boolean.

Restriccions en els Genèrics

De vegades, pot ser necessari restringir els tipus que es poden utilitzar amb la interfície genèrica. Això es pot fer utilitzant restriccions.

Exemple: Interfície Genèrica amb Restriccions

interface Lengthwise {
    length: number;
}

interface Collection<T extends Lengthwise> {
    items: T[];
    getLength(): number;
}

let stringCollection: Collection<string> = {
    items: ["Hola", "Món"],
    getLength: function() {
        return this.items.length;
    }
};

console.log(stringCollection.getLength()); // Sortida: 2
  • Explicació:
    • Lengthwise és una interfície que requereix una propietat length.
    • Collection<T extends Lengthwise> és una interfície genèrica on T ha de tenir una propietat length.
    • stringCollection és una instància de Collection amb T com string.

Usos Pràctics

  • Estructures de Dades: Implementació de piles, cues i llistes enllaçades.
  • Respostes d'API: Maneig de diferents tipus de respostes d'API de manera segura.
  • Funcions d'Utilitat: Creació de funcions que poden operar amb diversos tipus de dades.

Exemple: Interfície de Pila Genèrica

interface Stack<T> {
    push(item: T): void;
    pop(): T | undefined;
    peek(): T | undefined;
    size(): number;
}

class ArrayStack<T> implements Stack<T> {
    private storage: T[] = [];

    push(item: T): void {
        this.storage.push(item);
    }

    pop(): T | undefined {
        return this.storage.pop();
    }

    peek(): T | undefined {
        return this.storage[this.storage.length - 1];
    }

    size(): number {
        return this.storage.length;
    }
}

let numberStack = new ArrayStack<number>();
numberStack.push(10);
numberStack.push(20);
console.log(numberStack.pop()); // Sortida: 20
console.log(numberStack.size()); // Sortida: 1
  • Explicació:
    • Stack<T> és una interfície genèrica per a una estructura de dades de pila.
    • ArrayStack<T> és una classe que implementa la interfície Stack<T> utilitzant un array.
    • numberStack és una instància de ArrayStack amb T com number.

Conclusió

L'ús de genèrics amb interfícies en TypeScript permet crear codi flexible, reutilitzable i segur. Entenent com definir i utilitzar interfícies genèriques, es poden construir aplicacions més robustes que poden manejar una varietat de tipus de dades sense sacrificar la seguretat de tipus.
