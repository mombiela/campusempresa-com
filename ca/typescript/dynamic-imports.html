Imports dinàmics
Introducció als decoradors ►

Els imports dinàmics en TypeScript et permeten carregar mòduls de manera asíncrona, cosa que pot ser especialment útil per optimitzar el rendiment i gestionar dependències en aplicacions grans. Aquest tema et guiarà des dels conceptes bàsics fins als avançats dels imports dinàmics en TypeScript.

Què són els imports dinàmics?

Els imports dinàmics et permeten carregar mòduls de JavaScript a demanda, en lloc de tenir tots els mòduls carregats al començament de la teva aplicació. Això pot ajudar a millorar el temps de càrrega inicial i el rendiment general de la teva aplicació.

  • Imports estàtics: Carregats al començament de l'aplicació.
  • Imports dinàmics: Carregats quan es necessiten, utilitzant la funció import().

Sintaxi dels imports dinàmics

Els imports dinàmics utilitzen la funció import(), que retorna una promesa que es resol amb l'objecte del mòdul.

// Import estàtic
import { myFunction } from './myModule';

// Import dinàmic
const modulePromise = import('./myModule');
modulePromise.then(module => {
  const myFunction = module.myFunction;
  myFunction();
});

Beneficis dels imports dinàmics

  • Optimització del rendiment: Carrega només els mòduls necessaris quan es necessiten.
  • Divisió de codi: Divideix la teva aplicació en trossos més petits.
  • Càrrega mandrosa: Carrega components o mòduls només quan són necessaris.

Exemple bàsic

Comencem amb un exemple bàsic de com utilitzar imports dinàmics en TypeScript.

// myModule.ts
export function greet(name: string) {
  console.log(`Hello, ${name}!`);
}

// main.ts
async function loadModule() {
  const module = await import('./myModule');
  module.greet('World');
}

loadModule();

En aquest exemple:

  1. myModule.ts exporta una funció greet.
  2. main.ts importa dinàmicament myModule i crida la funció greet.

Gestió d'errors

Quan utilitzes imports dinàmics, és essencial gestionar els errors que poden ocórrer durant el procés de càrrega.

async function loadModule() {
  try {
    const module = await import('./myModule');
    module.greet('World');
  } catch (error) {
    console.error('Error carregant el mòdul:', error);
  }
}

loadModule();

Ús avançat

Imports condicionals

Pots importar mòduls de manera condicional basant-te en certes condicions.

async function loadModule(condition: boolean) {
  if (condition) {
    const module = await import('./moduleA');
    module.doSomething();
  } else {
    const module = await import('./moduleB');
    module.doSomethingElse();
  }
}

loadModule(true);

Imports de camins dinàmics

Pots construir dinàmicament el camí al mòdul que vols importar.

async function loadModule(moduleName: string) {
  const module = await import(`./${moduleName}`);
  module.default();
}

loadModule('myModule');

Ús dels imports dinàmics amb React

Els imports dinàmics poden ser especialment útils en React per a la càrrega mandrosa de components.

import React, { Suspense, lazy } from 'react';

const LazyComponent = lazy(() => import('./LazyComponent'));

function App() {
  return (
    <div>
      <Suspense fallback={<div>Loading...</div>}>
        <LazyComponent />
      </Suspense>
    </div>
  );
}

export default App;

Resum

Els imports dinàmics en TypeScript proporcionen una manera poderosa d'optimitzar el rendiment de la teva aplicació carregant mòduls només quan són necessaris. Ofereixen diversos beneficis, incloent l'optimització del rendiment, la divisió de codi i la càrrega mandrosa. Entenent i utilitzant els imports dinàmics, pots crear aplicacions més eficients i responsives.

  • Conceptes clau:

    • Els imports dinàmics utilitzen la funció import().
    • Retornen una promesa que es resol amb l'objecte del mòdul.
    • Es poden utilitzar per a l'optimització del rendiment, la divisió de codi i la càrrega mandrosa.

  • Exemples:

    • Import dinàmic bàsic.
    • Gestió d'errors en imports dinàmics.
    • Imports condicionals i de camins dinàmics.
    • Ús dels imports dinàmics amb React per a la càrrega mandrosa de components.

Dominant els imports dinàmics, pots millorar significativament el rendiment i la mantenibilitat de les teves aplicacions TypeScript.
