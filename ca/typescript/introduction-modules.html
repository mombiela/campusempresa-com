Introducció als Mòduls
Exportació i importació de mòduls ►

Els mòduls són una part fonamental de TypeScript i del JavaScript modern, permetent-te organitzar i gestionar el teu codi de manera més efectiva. Aquesta secció t'introduirà al concepte de mòduls, com crear-los i com utilitzar-los en els teus projectes de TypeScript.

Què són els Mòduls?

Els mòduls són peces de codi reutilitzables que es poden importar i exportar entre diferents parts de la teva aplicació. Ajuden a organitzar el codi, evitar la contaminació de l'àmbit global i gestionar les dependències.

Conceptes Clau

  • Exportar: Fer variables, funcions, classes o interfícies disponibles per a altres mòduls.
  • Importar: Portar variables, funcions, classes o interfícies d'altres mòduls al mòdul actual.
  • Exportació Per Defecte: Una única exportació d'un mòdul que es pot importar sense especificar un nom.
  • Exportació Nominal: Múltiples exportacions d'un mòdul que s'han d'importar utilitzant els seus noms específics.

Exportar i Importar

Exportar

Pots exportar variables, funcions, classes o interfícies d'un mòdul utilitzant la paraula clau export.

Exemple: Exportació Nominal

// mathUtils.ts
export function add(a: number, b: number): number {
    return a + b;
}

export function subtract(a: number, b: number): number {
    return a - b;
}

Exemple: Exportació Per Defecte

// logger.ts
export default function log(message: string): void {
    console.log(message);
}

Importar

Pots importar variables, funcions, classes o interfícies d'altres mòduls utilitzant la paraula clau import.

Exemple: Importar Exportacions Nominals

// main.ts
import { add, subtract } from './mathUtils';

console.log(add(5, 3)); // Sortida: 8
console.log(subtract(5, 3)); // Sortida: 2

Exemple: Importar Exportació Per Defecte

// main.ts
import log from './logger';

log('Aquest és un missatge de registre'); // Sortida: Aquest és un missatge de registre

Combinació d'Exportacions Nominals i Per Defecte

Pots combinar exportacions nominals i per defecte en un únic mòdul.

// utils.ts
export function multiply(a: number, b: number): number {
    return a * b;
}

export default function divide(a: number, b: number): number {
    return a / b;
}
// main.ts
import divide, { multiply } from './utils';

console.log(multiply(6, 3)); // Sortida: 18
console.log(divide(6, 3)); // Sortida: 2

Re-exportar

Pots re-exportar mòduls per crear un únic punt d'entrada per a múltiples mòduls.

// mathOperations.ts
export { add, subtract } from './mathUtils';
export { multiply, default as divide } from './utils';
// main.ts
import { add, subtract, multiply, divide } from './mathOperations';

console.log(add(5, 3)); // Sortida: 8
console.log(subtract(5, 3)); // Sortida: 2
console.log(multiply(6, 3)); // Sortida: 18
console.log(divide(6, 3)); // Sortida: 2

Conclusió

Els mòduls són una característica poderosa en TypeScript que t'ajuden a organitzar i gestionar el teu codi de manera més efectiva. Entenent com exportar i importar mòduls, pots crear aplicacions més modulars, mantenibles i escalables. Practica creant i utilitzant mòduls en els teus projectes per sentir-te més còmode amb aquest concepte essencial.
