Introducció

En TypeScript, les funcions són un bloc de construcció fonamental de qualsevol aplicació. Entendre com definir i utilitzar tipus de funcions és crucial per escriure codi robust i segur en termes de tipus. Aquesta secció cobrirà els conceptes bàsics dels tipus de funcions, incloent com definir-los, utilitzar-los i alguns conceptes avançats.

Definició de Tipus de Funcions

En TypeScript, pots definir el tipus d'una funció especificant els tipus dels seus paràmetres i el tipus del seu valor de retorn.

Sintaxi Bàsica

Aquí tens la sintaxi bàsica per definir un tipus de funció:

type GreetFunction = (name: string) => string;

En aquest exemple:

  • GreetFunction és un àlies de tipus per a una funció que pren un paràmetre de tipus string i retorna un string.

Exemple

Creem una funció que coincideixi amb aquest tipus:

const greet: GreetFunction = (name: string) => {
    return `Hola, ${name}!`;
};

console.log(greet("Alice")); // Output: Hola, Alice!

Paràmetres Opcional i Per Defecte

TypeScript et permet definir paràmetres opcionals i per defecte en funcions.

Paràmetres Opcional

Els paràmetres opcionals es denoten amb un ? després del nom del paràmetre.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age) => {
    if (age) {
        return `Hola, ${name}. Tens ${age} anys.`;
    }
    return `Hola, ${name}!`;
};

console.log(greet("Alice")); // Output: Hola, Alice!
console.log(greet("Bob", 30)); // Output: Hola, Bob. Tens 30 anys.

Paràmetres Per Defecte

Els paràmetres per defecte s'assignen un valor per defecte en la signatura de la funció.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age = 25) => {
    return `Hola, ${name}. Tens ${age} anys.`;
};

console.log(greet("Alice")); // Output: Hola, Alice. Tens 25 anys.
console.log(greet("Bob", 30)); // Output: Hola, Bob. Tens 30 anys.

Tipus de Funcions en Interfícies

També pots definir tipus de funcions dins d'interfícies.

Exemple

interface MathOperations {
    add: (a: number, b: number) => number;
    subtract: (a: number, b: number) => number;
}

const operations: MathOperations = {
    add: (a, b) => a + b,
    subtract: (a, b) => a - b,
};

console.log(operations.add(5, 3)); // Output: 8
console.log(operations.subtract(5, 3)); // Output: 2

Sobrecàrrega de Funcions

La sobrecàrrega de funcions et permet definir múltiples signatures per a una sola funció.

Exemple

function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
    return a + b;
}

console.log(add(5, 3)); // Output: 8
console.log(add("Hola, ", "món!")); // Output: Hola, món!

Conclusió

Entendre els tipus de funcions en TypeScript és essencial per escriure codi segur en termes de tipus i mantenible. Definint tipus de funcions, utilitzant paràmetres opcionals i per defecte, aprofitant les interfícies i utilitzant sobrecàrrega de funcions, pots crear funcions més robustes i flexibles. Practica aquests conceptes amb diferents exemples per solidificar la teva comprensió i convertir-te en un expert en tipus de funcions de TypeScript.
