Introducció als Enums

Els enums (abreviatura d'Enumeracions) són una característica de TypeScript que et permet definir un conjunt de constants amb nom. Faciliten el treball amb conjunts de valors relacionats i milloren la llegibilitat i mantenibilitat del codi.

Conceptes Clau

  • Enumeració: Una manera de definir una col·lecció de valors relacionats.
  • Constants amb Nom: Cada valor en un enum és una constant amb un nom.
  • Seguretat de Tipus: Els enums proporcionen seguretat de tipus assegurant que només es poden utilitzar valors predefinits.

Definició d'Enums

En TypeScript, pots definir un enum utilitzant la paraula clau enum.

Exemple

enum Direction {
    Up,
    Down,
    Left,
    Right
}

En aquest exemple, Direction és un enum amb quatre valors possibles: Up, Down, Left i Right.

Explicació

  • Per defecte, els valors dels membres de l'enum s'incrementen automàticament començant des de 0.
  • Up és 0, Down és 1, Left és 2, i Right és 3.

Ús dels Enums

Pots utilitzar enums per declarar variables i assignar-los valors d'enum.

Exemple

let move: Direction = Direction.Up;
console.log(move); // Sortida: 0

Explicació

  • La variable move és de tipus Direction.
  • Se li assigna el valor Direction.Up, que correspon a 0.

Personalització dels Valors d'Enum

Pots assignar valors personalitzats als membres de l'enum.

Exemple

enum Status {
    Success = 1,
    Failure = -1,
    Pending = 0
}

Explicació

  • Success té assignat el valor 1.
  • Failure té assignat el valor -1.
  • Pending té assignat el valor 0.

Enums de Cadena

Els enums també poden tenir valors de cadena.

Exemple

enum Color {
    Red = "RED",
    Green = "GREEN",
    Blue = "BLUE"
}

Explicació

  • Cada membre de l'enum Color té assignat un valor de cadena.

Enums Heterogenis

Els enums poden tenir tant valors numèrics com de cadena, tot i que això és menys comú.

Exemple

enum Mixed {
    No = 0,
    Yes = "YES"
}

Explicació

  • No té assignat el valor numèric 0.
  • Yes té assignat el valor de cadena "YES".

Tipus de Membres d'Enum

Els membres d'enum poden ser constants o calculats.

Membres Constants

Aquests són membres amb valors que es poden calcular en temps de compilació.

Exemple

enum ConstantEnum {
    A = 1,
    B = A + 1
}

Explicació

  • A és 1.
  • B és A + 1, que és 2.

Membres Calculats

Aquests són membres amb valors que es calculen en temps d'execució.

Exemple

enum ComputedEnum {
    X = "X".length,
    Y = 10 * 2
}

Explicació

  • X és la longitud de la cadena "X", que és 1.
  • Y és el resultat de l'expressió 10 * 2, que és 20.

Mapeig Invers d'Enum

TypeScript proporciona mapeig invers per als enums numèrics.

Exemple

enum Days {
    Monday,
    Tuesday,
    Wednesday
}

console.log(Days.Monday); // Sortida: 0
console.log(Days[0]); // Sortida: "Monday"

Explicació

  • Days.Monday dóna el valor numèric 0.
  • Days[0] dóna la cadena "Monday".

Enum com a Tipus

Els enums es poden utilitzar com a tipus per assegurar que les variables només poden prendre els valors definits en l'enum.

Exemple

enum Size {
    Small,
    Medium,
    Large
}

function selectSize(size: Size): void {
    console.log(`Selected size: ${Size[size]}`);
}

selectSize(Size.Medium); // Sortida: Selected size: Medium

Explicació

  • La funció selectSize accepta un paràmetre de tipus Size.
  • Això assegura que només es poden passar valors vàlids de Size a la funció.

Conclusió

Els enums en TypeScript proporcionen una manera poderosa de definir i utilitzar conjunts de constants relacionades. Milloren la llegibilitat, mantenibilitat i seguretat de tipus del codi. Entenent com definir, utilitzar i personalitzar els enums, pots aprofitar tot el seu potencial en els teus projectes de TypeScript.
