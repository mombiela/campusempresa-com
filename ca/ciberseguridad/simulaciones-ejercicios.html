Simulacions i Exercicis de Seguretat

Introducció

En aquesta secció del curs, ens enfocarem en la pràctica de conceptes de ciberseguretat a través de simulacions i exercicis. Aquestes activitats estan dissenyades per reforçar el coneixement teòric i proporcionar experiència pràctica en la identificació i mitigació d'amenaces de seguretat.

Objectius

  • Aplicar conceptes teòrics de ciberseguretat en situacions pràctiques.
  • Desenvolupar habilitats per identificar i mitigar amenaces de seguretat.
  • Millorar la capacitat de resposta davant incidents de seguretat.

Simulacions de Seguretat

Simulació d'un Atac de Phishing

El phishing és una tècnica comuna utilitzada pels atacants per obtenir informació confidencial. En aquesta simulació, els estudiants aprendran a identificar correus electrònics de phishing i a prendre les mesures adequades per protegir-se.

Passos de la Simulació

  1. Recepció del Correu Electrònic: Els estudiants rebran un correu electrònic simulat que conté un enllaç sospitós.
  2. Anàlisi del Correu: Se'ls demanarà que analitzin el correu en cerca de senyals de phishing, com errors gramaticals, URLs sospitoses i remitents desconeguts.
  3. Resposta al Correu: Els estudiants hauran de decidir si reporten el correu, l'eliminen o prenen alguna altra acció.

Exemple de Codi per a Anàlisi de URLs

import re

def is_suspicious_url(url):
    # Expressió regular per detectar URLs sospitoses
    pattern = re.compile(r'https?://(?!www\.)')
    return bool(pattern.search(url))

# Exemple d'ús
url = "http://malicious-site.com"
print("URL sospitosa" si is_suspicious_url(url) else "URL segura")

Simulació d'un Atac de Força Bruta

En aquesta simulació, els estudiants aprendran com els atacants utilitzen atacs de força bruta per obtenir accés no autoritzat a sistemes i com protegir-se contra aquests atacs.

Passos de la Simulació

  1. Configuració de l'Entorn: Es proporcionarà un entorn segur on els estudiants puguin realitzar atacs de força bruta sense causar dany.
  2. Execució de l'Atac: Els estudiants utilitzaran eines com Hydra o John the Ripper per intentar trencar contrasenyes.
  3. Implementació de Mesures de Seguretat: Després de l'atac, els estudiants implementaran mesures de seguretat com el bloqueig de comptes després de múltiples intents fallits.

Exemple de Codi per a Bloqueig de Comptes

class UserAccount:
    def __init__(self, username):
        self.username = username
        self.failed_attempts = 0
        self.is_locked = False

    def login(self, password):
        if self.is_locked:
            return "Compte bloquejada"
        if password == "correct_password":
            self.failed_attempts = 0
            return "Inici de sessió exitós"
        else:
            self.failed_attempts += 1
            if self.failed_attempts >= 3:
                self.is_locked = True
                return "Compte bloquejada"
            return "Contrasenya incorrecta"

# Exemple d'ús
user = UserAccount("usuari1")
print(user.login("wrong_password"))
print(user.login("wrong_password"))
print(user.login("wrong_password"))
print(user.login("correct_password"))

Exercicis de Seguretat

Exercici d'Anàlisi de Vulnerabilitats

Els estudiants realitzaran una anàlisi de vulnerabilitats en un sistema simulat utilitzant eines com Nmap i OpenVAS.

Passos de l'Exercici

  1. Escaneig de Ports: Utilitzar Nmap per identificar ports oberts i serveis en execució.
  2. Detecció de Vulnerabilitats: Utilitzar OpenVAS per detectar vulnerabilitats en els serveis identificats.
  3. Informe de Vulnerabilitats: Redactar un informe detallat de les vulnerabilitats trobades i les possibles solucions.

Exemple de Comanda Nmap

nmap -sV -p 1-65535 target_ip

Exercici de Configuració de Tallafocs

Els estudiants configuraran regles de tallafocs per protegir un sistema d'atacs externs.

Passos de l'Exercici

  1. Identificació de Serveis Crítics: Determinar quins serveis han d'estar accessibles des de l'exterior.
  2. Configuració de Regles: Configurar regles de tallafocs per permetre només el trànsit necessari.
  3. Prova de Configuració: Provar la configuració del tallafocs per assegurar-se que només el trànsit permès pot passar.

Exemple de Configuració de Tallafocs amb iptables

# Permetre trànsit SSH
iptables -A INPUT -p tcp --dport 22 -j ACCEPT

# Permetre trànsit HTTP
iptables -A INPUT -p tcp --dport 80 -j ACCEPT

# Bloquejar tot el trànsit restant
iptables -A INPUT -j DROP

Conclusió

Les simulacions i exercicis de seguretat són essencials per desenvolupar habilitats pràctiques en ciberseguretat. A través d'aquestes activitats, els estudiants poden aplicar conceptes teòrics en situacions reals, millorant la seva capacitat per identificar i mitigar amenaces de seguretat. La pràctica constant i l'actualització de coneixements són clau per mantenir-se al dia en el camp de la ciberseguretat.
