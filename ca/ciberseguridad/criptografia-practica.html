Introducció a la Criptografia

La criptografia és una disciplina fonamental en la ciberseguretat que s'encarrega de protegir la informació mitjançant tècniques de codificació. En aquest tema, explorarem com s'aplica la criptografia en la pràctica, els seus principals algorismes i com implementar-los en el desenvolupament de programari.

Conceptes Clau

  • Xifrat i Desxifrat: Procés de convertir informació llegible en un format codificat i viceversa.
  • Clau Criptogràfica: Informació utilitzada per xifrar i desxifrar dades.
  • Algorismes Simètrics: Utilitzen la mateixa clau per xifrar i desxifrar.
  • Algorismes Asimètrics: Utilitzen un parell de claus (pública i privada) per xifrar i desxifrar.
  • Hashing: Transformació de dades en una cadena de longitud fixa que representa les dades originals.

Algorismes de Xifrat Simètric

Els algorismes simètrics són ràpids i eficients per xifrar grans quantitats de dades. Alguns dels més utilitzats són:

  • AES (Advanced Encryption Standard): Algorisme de xifrat per blocs que utilitza claus de 128, 192 o 256 bits.
  • DES (Data Encryption Standard): Algorisme més antic i menys segur que AES, utilitza una clau de 56 bits.
  • 3DES (Triple DES): Millora de DES que aplica el xifrat tres vegades per a major seguretat.

Exemple de Xifrat Simètric amb AES en Python

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clau de 16 bytes (128 bits)
key = get_random_bytes(16)

# Crear un objecte de xifrat AES
cipher = AES.new(key, AES.MODE_EAX)

# Dades a xifrar
data = b'Aquest és un missatge secret'

# Xifrar les dades
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(data)

print(f'Xifrat: {ciphertext}')

En aquest exemple, utilitzem la biblioteca pycryptodome per xifrar un missatge amb AES.

Algorismes de Xifrat Asimètric

Els algorismes asimètrics són més lents però permeten la distribució segura de claus. Els més comuns són:

  • RSA (Rivest-Shamir-Adleman): Utilitza claus de longitud variable, comunament 2048 o 4096 bits.
  • ECC (Elliptic Curve Cryptography): Ofereix la mateixa seguretat que RSA però amb claus més curtes.

Exemple de Xifrat Asimètric amb RSA en Python

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP
from Crypto.Random import get_random_bytes

# Generar un parell de claus RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objecte de xifrat RSA
cipher = PKCS1_OAEP.new(public_key)

# Dades a xifrar
data = b'Aquest és un missatge secret'

# Xifrar les dades
ciphertext = cipher.encrypt(data)

print(f'Xifrat: {ciphertext}')

En aquest exemple, utilitzem la biblioteca pycryptodome per xifrar un missatge amb RSA.

Hashing

El hashing és una tècnica utilitzada per verificar la integritat de les dades. Els algorismes de hashing més comuns són:

  • SHA-256 (Secure Hash Algorithm 256 bits): Produeix un hash de 256 bits.
  • MD5 (Message Digest Algorithm 5): Produeix un hash de 128 bits, però és menys segur que SHA-256.

Exemple de Hashing amb SHA-256 en Python

import hashlib

# Dades a hashear
data = b'Aquest és un missatge secret'

# Crear un objecte de hashing SHA-256
hash_object = hashlib.sha256(data)

# Obtenir el hash en format hexadecimal
hash_hex = hash_object.hexdigest()

print(f'Hash: {hash_hex}')

En aquest exemple, utilitzem la biblioteca hashlib per generar un hash SHA-256 d'un missatge.

Comparació d'Algorismes

| Algorisme | Tipus | Longitud de Clau | Seguretat | |-----------|------|-------------------|-----------| | AES | Simètric | 128, 192, 256 bits | Alta | | DES | Simètric | 56 bits | Baixa | | 3DES | Simètric | 168 bits | Mitjana | | RSA | Asimètric | 1024, 2048, 4096 bits | Alta | | ECC | Asimètric | 160-521 bits | Alta | | SHA-256 | Hashing | 256 bits | Alta | | MD5 | Hashing | 128 bits | Baixa |

Conclusió

La criptografia és essencial per protegir la informació en l'àmbit de la ciberseguretat. Conèixer i aplicar correctament els diferents algorismes de xifrat i hashing permet assegurar la confidencialitat, integritat i autenticitat de les dades. En aquest tema, hem explorat els conceptes clau, els algorismes més utilitzats i com implementar-los en Python, proporcionant una base sòlida per a la seva aplicació pràctica en el desenvolupament de programari segur.
