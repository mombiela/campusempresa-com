Introducció a les classes i objectes

En Kotlin, les classes i els objectes són blocs de construcció fonamentals de la programació orientada a objectes. Permeten crear estructures de codi reutilitzables i modelar entitats del món real.

Conceptes clau

  • Classe: Un pla per crear objectes. Defineix propietats i comportaments.
  • Objecte: Una instància d'una classe. Representa una implementació específica del pla de la classe.
  • Constructor: Una funció especial que inicialitza un objecte.
  • Propietats: Variables dins d'una classe que contenen dades.
  • Mètodes: Funcions dins d'una classe que defineixen comportaments.

Definició d'una classe

Una classe en Kotlin es defineix utilitzant la paraula clau class seguida del nom de la classe.

class Person {
    var name: String = ""
    var age: Int = 0

    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicació

  • class Person { ... }: Defineix una classe anomenada Person.
  • var name: String = "": Una propietat name de tipus String amb un valor per defecte.
  • var age: Int = 0: Una propietat age de tipus Int amb un valor per defecte.
  • fun introduce() { ... }: Un mètode introduce que imprimeix un missatge.

Creació d'objectes

Per crear un objecte d'una classe, utilitza la paraula clau new seguida del nom de la classe i parèntesis.

fun main() {
    val person = Person()
    person.name = "John"
    person.age = 30
    person.introduce()
}

Explicació

  • val person = Person(): Crea un objecte de la classe Person.
  • person.name = "John": Estableix la propietat name de l'objecte person.
  • person.age = 30: Estableix la propietat age de l'objecte person.
  • person.introduce(): Crida el mètode introduce de l'objecte person.

Constructors

Kotlin proporciona un constructor primari i constructors secundaris per inicialitzar objectes.

Constructor primari

El constructor primari forma part de la capçalera de la classe.

class Person(val name: String, var age: Int) {
    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicació

  • class Person(val name: String, var age: Int) { ... }: Defineix un constructor primari amb els paràmetres name i age.

Utilitzant el constructor primari

fun main() {
    val person = Person("John", 30)
    person.introduce()
}

Constructor secundari

Una classe també pot tenir constructors secundaris.

class Person {
    var name: String
    var age: Int

    constructor(name: String, age: Int) {
        this.name = name
        this.age = age
    }

    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicació

  • constructor(name: String, age: Int) { ... }: Defineix un constructor secundari.

Utilitzant el constructor secundari

fun main() {
    val person = Person("John", 30)
    person.introduce()
}

Herència

Kotlin suporta la herència, permetent que una classe hereti propietats i mètodes d'una altra classe.

Classe base

open class Animal {
    open fun sound() {
        println("Animal makes a sound")
    }
}

Classe derivada

class Dog : Animal() {
    override fun sound() {
        println("Dog barks")
    }
}

Explicació

  • open class Animal { ... }: Defineix una classe base Animal que pot ser heretada.
  • class Dog : Animal() { ... }: Defineix una classe derivada Dog que hereta d'Animal.
  • override fun sound() { ... }: Sobreescriu el mètode sound a la classe Dog.

Utilitzant la herència

fun main() {
    val dog = Dog()
    dog.sound()  // Output: Dog barks
}

Classes de dades

Kotlin proporciona un tipus especial de classe anomenada classe de dades, que s'utilitza per contenir dades.

Definició d'una classe de dades

data class User(val name: String, val age: Int)

Explicació

  • data class User(val name: String, val age: Int): Defineix una classe de dades User amb les propietats name i age.

Utilitzant una classe de dades

fun main() {
    val user = User("Alice", 25)
    println(user)  // Output: User(name=Alice, age=25)
}

Resum

En aquest tema, hem cobert els conceptes bàsics de les classes i els objectes en Kotlin, incloent:

  • Definició de classes i creació d'objectes.
  • Utilització de constructors primaris i secundaris.
  • Implementació de la herència.
  • Utilització de classes de dades per contenir dades.

Entendre aquests conceptes és crucial per construir codi robust i reutilitzable en Kotlin. Practica creant les teves pròpies classes i objectes per consolidar la teva comprensió.
