Introducció

Els operadors són símbols especials o paraules clau utilitzades per realitzar operacions sobre variables i valors. En Kotlin, els operadors es classifiquen en diversos tipus, com ara aritmètics, relacionals, lògics i més. Entendre com utilitzar aquests operadors és fonamental per escriure codi Kotlin efectiu.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul (resta de la divisió)

Exemple

fun main() {
    val a = 10
    val b = 5

    println("Suma: ${a + b}")        // Sortida: 15
    println("Resta: ${a - b}")       // Sortida: 5
    println("Multiplicació: ${a * b}")  // Sortida: 50
    println("Divisió: ${a / b}")        // Sortida: 2
    println("Mòdul: ${a % b}")         // Sortida: 0
}

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors. Retornen un resultat Boolean.

  • == : Igual a
  • != : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual a
  • <= : Menor o igual a

Exemple

fun main() {
    val a = 10
    val b = 5

    println("Igual a: ${a == b}")       // Sortida: false
    println("Diferent de: ${a != b}")   // Sortida: true
    println("Major que: ${a > b}")    // Sortida: true
    println("Menor que: ${a < b}")       // Sortida: false
    println("Major o igual: ${a >= b}") // Sortida: true
    println("Menor o igual: ${a <= b}")  // Sortida: false
}

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions Booleanes.

  • && : Lògic AND
  • || : Lògic OR
  • ! : Lògic NOT

Exemple

fun main() {
    val a = true
    val b = false

    println("Lògic AND: ${a && b}")    // Sortida: false
    println("Lògic OR: ${a || b}")     // Sortida: true
    println("Lògic NOT: ${!a}")        // Sortida: false
}

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. També es poden combinar amb operadors aritmètics.

  • = : Assignació simple
  • += : Sumar i assignar
  • -= : Restar i assignar
  • *= : Multiplicar i assignar
  • /= : Dividir i assignar
  • %= : Mòdul i assignar

Exemple

fun main() {
    var a = 10

    a += 5
    println("Sumar i assignar: $a")        // Sortida: 15

    a -= 3
    println("Restar i assignar: $a")   // Sortida: 12

    a *= 2
    println("Multiplicar i assignar: $a")   // Sortida: 24

    a /= 4
    println("Dividir i assignar: $a")     // Sortida: 6

    a %= 5
    println("Mòdul i assignar: $a")    // Sortida: 1
}

Operadors Unaris

Els operadors unaris s'utilitzen amb només un operand.

  • + : Unari més
  • - : Unari menys
  • ++ : Increment
  • -- : Decrement
  • ! : Lògic NOT

Exemple

fun main() {
    var a = 5

    println("Unari més: ${+a}")         // Sortida: 5
    println("Unari menys: ${-a}")        // Sortida: -5

    a++
    println("Increment: $a")             // Sortida: 6

    a--
    println("Decrement: $a")             // Sortida: 5

    val b = false
    println("Lògic NOT: ${!b}")        // Sortida: true
}

Operadors a Nivell de Bit

Els operadors a nivell de bit s'utilitzen per realitzar operacions a nivell de bit.

  • shl : Desplaçament a l'esquerra amb signe
  • shr : Desplaçament a la dreta amb signe
  • ushr : Desplaçament a la dreta sense signe
  • and : AND a nivell de bit
  • or : OR a nivell de bit
  • xor : XOR a nivell de bit
  • inv : Inversió a nivell de bit

Exemple

fun main() {
    val a = 5  // 0101 en binari
    val b = 3  // 0011 en binari

    println("AND a nivell de bit: ${a and b}")   // Sortida: 1 (0001 en binari)
    println("OR a nivell de bit: ${a or b}")     // Sortida: 7 (0111 en binari)
    println("XOR a nivell de bit: ${a xor b}")   // Sortida: 6 (0110 en binari)
    println("Inversió a nivell de bit: ${a.inv()}") // Sortida: -6 (bits invertits de 5)
    println("Desplaçament a l'esquerra amb signe: ${a shl 1}") // Sortida: 10 (1010 en binari)
    println("Desplaçament a la dreta amb signe: ${a shr 1}") // Sortida: 2 (0010 en binari)
    println("Desplaçament a la dreta sense signe: ${a ushr 1}") // Sortida: 2 (0010 en binari)
}

Conclusió

Els operadors són una part fonamental de la programació en Kotlin, permetent-te realitzar una àmplia gamma d'operacions sobre variables i valors. Des d'operacions aritmètiques bàsiques fins a operacions complexes a nivell de bit, entendre com utilitzar aquests operadors de manera efectiva millorarà la teva capacitat per escriure codi Kotlin eficient i llegible. Practica utilitzant aquests operadors en diferents escenaris per sentir-te més còmode amb la seva funcionalitat.
