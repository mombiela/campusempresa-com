Introducció

En aquesta secció, explorarem com implementar diverses funcions en Kotlin. Això inclourà entendre els conceptes bàsics de la implementació de funcions, explorar conceptes intermedis i endinsar-nos en tècniques avançades. Al final d'aquesta secció, hauríeu de poder implementar funcions amb confiança en les vostres aplicacions Kotlin.

Implementació Bàsica de Funcions

Funcions

Les funcions són els blocs de construcció de qualsevol programa Kotlin. Permeten encapsular codi en blocs reutilitzables.

  • Definició d'una Funció: Utilitzeu la paraula clau fun.
  • Paràmetres de la Funció: Especifiqueu els paràmetres dins dels parèntesis.
  • Tipus de Retorn: Especifiqueu el tipus de retorn després d'un dos punts.
fun greet(name: String): String {
    return "Hola, $name!"
}

fun main() {
    println(greet("Alice")) // Sortida: Hola, Alice!
}

Variables

Les variables emmagatzemen dades que es poden utilitzar i manipular dins del vostre programa.

  • Variables Immutables: Utilitzeu val per a variables de només lectura.
  • Variables Mutables: Utilitzeu var per a variables que es poden canviar.
val immutableValue = 10
var mutableValue = 20

mutableValue = 30 // Això està permès
// immutableValue = 40 // Això causarà un error

Flux de Control

Les instruccions de flux de control permeten dictar el flux del vostre programa.

  • Instruccions If-Else: Lògica condicional bàsica.
  • Expressió When: La versió de Kotlin de l'instrucció switch.
val number = 10

// If-Else
if (number > 5) {
    println("El número és més gran que 5")
} else {
    println("El número és 5 o menys")
}

// Expressió When
when (number) {
    1 -> println("Un")
    2 -> println("Dos")
    10 -> println("Deu")
    else -> println("Número desconegut")
}

Implementació Intermèdia de Funcions

Classes i Objectes

Les classes són plànols per crear objectes. Els objectes són instàncies de classes.

  • Definició d'una Classe: Utilitzeu la paraula clau class.
  • Constructor Primari: Definiu-lo directament a l'encapçalament de la classe.
  • Propietats: Variables dins d'una classe.
class Person(val name: String, var age: Int)

fun main() {
    val person = Person("Alice", 30)
    println(person.name) // Sortida: Alice
    println(person.age)  // Sortida: 30
}

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe.

  • Classes Obertes: Utilitzeu la paraula clau open per permetre l'herència.
  • Subclasse: Utilitzeu : per heretar d'una superclasse.
open class Animal(val name: String) {
    fun makeSound() {
        println("$name fa un so")
    }
}

class Dog(name: String) : Animal(name) {
    fun bark() {
        println("$name borda")
    }
}

fun main() {
    val dog = Dog("Buddy")
    dog.makeSound() // Sortida: Buddy fa un so
    dog.bark()      // Sortida: Buddy borda
}

Interfícies

Les interfícies defineixen un contracte que les classes poden implementar.

  • Definició d'una Interfície: Utilitzeu la paraula clau interface.
  • Implementació d'una Interfície: Utilitzeu : seguit del nom de la interfície.
interface Drivable {
    fun drive()
}

class Car : Drivable {
    override fun drive() {
        println("El cotxe està conduint")
    }
}

fun main() {
    val car = Car()
    car.drive() // Sortida: El cotxe està conduint
}

Implementació Avançada de Funcions

Funcions d'Ordre Superior

Funcions que prenen altres funcions com a paràmetres o que retornen funcions.

  • Tipus de Funció: Especifiqueu el tipus de funcions.
  • Expressions Lambda: Funcions anònimes.
fun calculate(x: Int, y: Int, operation: (Int, Int) -> Int): Int {
    return operation(x, y)
}

fun main() {
    val sum = calculate(5, 3) { a, b -> a + b }
    println(sum) // Sortida: 8
}

Coroutines

Les coroutines són una manera de gestionar la programació asíncrona en Kotlin.

  • Llançament de Coroutines: Utilitzeu launch o async de kotlinx.coroutines.
  • Funcions de Suspensió: Funcions que es poden pausar i reprendre.
import kotlinx.coroutines.*

fun main() = runBlocking {
    launch {
        delay(1000L)
        println("Món!")
    }
    println("Hola,")
}

Funcions d'Extensió

Afegiu noves funcions a classes existents sense modificar el seu codi font.

  • Definició d'una Funció d'Extensió: Utilitzeu el nom de la classe com a tipus receptor.
fun String.reverse(): String {
    return this.reversed()
}

fun main() {
    val original = "Kotlin"
    val reversed = original.reverse()
    println(reversed) // Sortida: niltoK
}

Conclusió

Implementar funcions en Kotlin implica entendre i utilitzar diversos constructes del llenguatge com ara funcions, classes, herència, interfícies, funcions d'ordre superior, coroutines i funcions d'extensió. En dominar aquests conceptes, podeu construir aplicacions Kotlin robustes i eficients. Continueu practicant aquestes funcions per aprofundir en la vostra comprensió i millorar les vostres habilitats de programació.
