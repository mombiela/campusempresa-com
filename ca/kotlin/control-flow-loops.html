Introducció

Les instruccions de control de flux són essencials en la programació, ja que et permeten dictar l'ordre en què s'executen les instruccions. Els bucles són una part fonamental del control de flux, permetent-te executar un bloc de codi múltiples vegades. En Kotlin, hi ha diversos tipus de bucles, cadascun adequat per a diferents escenaris.

Tipus de Bucles en Kotlin

Bucle `for`

El bucle for en Kotlin s'utilitza per iterar sobre un rang, col·lecció o qualsevol cosa que proporcioni un iterador.

Sintaxi

for (item in collection) {
    // Codi a executar per a cada element
}

Exemple

val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number)
}
  • Explicació: Aquest bucle itera sobre cada element de la llista numbers i l'imprimeix.

Bucle `while`

El bucle while executa un bloc de codi mentre una condició especificada sigui certa.

Sintaxi

while (condition) {
    // Codi a executar mentre la condició sigui certa
}

Exemple

var i = 0
while (i < 5) {
    println(i)
    i++
}
  • Explicació: Aquest bucle imprimeix números del 0 al 4. El bucle continua mentre i sigui menor que 5.

Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

Sintaxi

do {
    // Codi a executar
} while (condition)

Exemple

var i = 0
do {
    println(i)
    i++
} while (i < 5)
  • Explicació: Aquest bucle també imprimeix números del 0 al 4. La diferència clau és que la condició es comprova després d'executar el bloc de codi.

Conceptes Avançats de Bucles

Instruccions de Control de Bucles

Kotlin proporciona diverses instruccions de control per gestionar l'execució dels bucles:

  • break: Termina el bucle immediatament.
  • continue: Salta la iteració actual i procedeix a la següent.
  • return: Surt de la funció en què es troba el bucle.

Exemple amb break i continue

for (i in 1..10) {
    if (i == 5) break
    if (i % 2 == 0) continue
    println(i)
}
  • Explicació: Aquest bucle imprimeix números senars de l'1 al 9. S'atura completament quan i és igual a 5 a causa de la instrucció break. La instrucció continue salta els números parells.

Bucles Imbricats

Pots col·locar un bucle dins d'un altre bucle. Això és útil per treballar amb estructures de dades multidimensionals.

Exemple

for (i in 1..3) {
    for (j in 1..3) {
        println("i = $i, j = $j")
    }
}
  • Explicació: Aquest bucle imbricat imprimeix totes les combinacions de i i j de l'1 al 3.

Etiquetes de Bucles

Kotlin et permet etiquetar bucles i controlar el seu flux utilitzant aquestes etiquetes.

Exemple

outer@ for (i in 1..3) {
    for (j in 1..3) {
        if (i == 2 && j == 2) break@outer
        println("i = $i, j = $j")
    }
}
  • Explicació: La instrucció break@outer termina el bucle extern quan i és igual a 2 i j és igual a 2.

Conclusió

Els bucles són una eina poderosa en Kotlin que et permeten executar codi múltiples vegades de manera eficient. Entendre els diferents tipus de bucles i com controlar el seu flux és crucial per escriure programes efectius en Kotlin. Practica utilitzant els bucles for, while i do-while, juntament amb instruccions de control com break i continue, per convertir-te en un expert en la gestió del flux de control en les teves aplicacions.
