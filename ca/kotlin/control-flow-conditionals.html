Introducció als Condicionals

Els condicionals són fonamentals en la programació ja que et permeten executar diferents codis basats en certes condicions. En Kotlin, els condicionals es gestionen principalment utilitzant les instruccions if, else if i else, així com l'expressió when.

Instruccions If-Else

La instrucció if és la forma més bàsica de condicional. Et permet executar un bloc de codi només si una condició especificada és certa.

Sintaxi

if (condition) {
    // codi a executar si la condició és certa
} else {
    // codi a executar si la condició és falsa
}

Exemple

val number = 10

if (number > 0) {
    println("El número és positiu")
} else {
    println("El número no és positiu")
}

Explicació

  • if (number > 0): Comprova si number és més gran que 0.
  • println("El número és positiu"): S'executa si la condició és certa.
  • else: S'executa si la condició és falsa.
  • println("El número no és positiu"): S'executa si la condició és falsa.

Instruccions Else If

Quan tens múltiples condicions a comprovar, pots utilitzar else if.

Sintaxi

if (condition1) {
    // codi a executar si condition1 és certa
} else if (condition2) {
    // codi a executar si condition2 és certa
} else {
    // codi a executar si cap de les condicions anteriors és certa
}

Exemple

val number = 0

if (number > 0) {
    println("El número és positiu")
} else if (number < 0) {
    println("El número és negatiu")
} else {
    println("El número és zero")
}

Explicació

  • else if (number < 0): Comprova si number és menor que 0.
  • println("El número és negatiu"): S'executa si la segona condició és certa.
  • else: S'executa si cap de les condicions és certa.
  • println("El número és zero"): S'executa si cap de les condicions és certa.

Expressió When

L'expressió when en Kotlin és una alternativa poderosa a la cadena if-else if-else. És similar a la instrucció switch en altres llenguatges.

Sintaxi

when (variable) {
    value1 -> {
        // codi a executar si variable == value1
    }
    value2 -> {
        // codi a executar si variable == value2
    }
    else -> {
        // codi a executar si cap de les condicions anteriors és certa
    }
}

Exemple

val dayOfWeek = 3

val dayName = when (dayOfWeek) {
    1 -> "Dilluns"
    2 -> "Dimarts"
    3 -> "Dimecres"
    4 -> "Dijous"
    5 -> "Divendres"
    6 -> "Dissabte"
    7 -> "Diumenge"
    else -> "Dia invàlid"
}

println(dayName)

Explicació

  • when (dayOfWeek): Avalua el valor de dayOfWeek.
  • 1 -> "Dilluns": S'executa si dayOfWeek és 1.
  • else -> "Dia invàlid": S'executa si cap de les condicions coincideix.

Ús Avançat de When

L'expressió when també es pot utilitzar sense un argument, actuant com una cadena avançada de if-else.

Exemple

val number = -10

val result = when {
    number > 0 -> "Positiu"
    number < 0 -> "Negatiu"
    else -> "Zero"
}

println(result)

Explicació

  • when {}: No es proporciona cap argument.
  • number > 0 -> "Positiu": S'executa si number és més gran que 0.
  • number < 0 -> "Negatiu": S'executa si number és menor que 0.
  • else -> "Zero": S'executa si cap de les condicions coincideix.

Conclusió

Els condicionals són essencials per controlar el flux dels teus programes en Kotlin. Les instruccions if-else són senzilles i útils per a condicions simples, mentre que l'expressió when ofereix una manera més llegible i flexible de gestionar múltiples condicions. Entendre i utilitzar aquests constructes de manera efectiva et permetrà escriure codi més dinàmic i sensible.
