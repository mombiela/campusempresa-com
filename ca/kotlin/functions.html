Introducció a les Funcions

Les funcions són blocs fonamentals en Kotlin, permetent encapsular fragments de codi reutilitzables. Ajuden a organitzar el codi, fer-lo més llegible i reduir la redundància.

Conceptes Clau

  • Definició de Funció: Com definir una funció.
  • Crida de Funció: Com cridar una funció.
  • Paràmetres i Arguments: Passar dades a les funcions.
  • Tipus de Retorn: Especificar què retorna una funció.
  • Arguments per Defecte i Nombrats: Millorar les crides de funció.
  • Funcions d'Ordre Superior: Funcions que prenen altres funcions com a paràmetres o les retornen.
  • Expressions Lambda: Funcions anònimes utilitzades per a codi concís.

Definició i Crida de Funcions

En Kotlin, les funcions es defineixen utilitzant la paraula clau fun.

Exemple

fun greet(name: String): String {
    return "Hello, $name!"
}

fun main() {
    val greeting = greet("Alice")
    println(greeting)  // Output: Hello, Alice!
}
  • fun greet(name: String): String defineix una funció anomenada greet que pren un paràmetre String i retorna un String.
  • return "Hello, $name!" especifica el valor a retornar de la funció.
  • greet("Alice") crida la funció amb "Alice" com a argument.

Paràmetres i Arguments

Les funcions poden prendre múltiples paràmetres, i els arguments són els valors reals passats a aquests paràmetres.

Exemple

fun add(a: Int, b: Int): Int {
    return a + b
}

fun main() {
    val sum = add(5, 3)
    println(sum)  // Output: 8
}
  • fun add(a: Int, b: Int): Int defineix una funció amb dos paràmetres Int.
  • add(5, 3) crida la funció amb 5 i 3 com a arguments.

Tipus de Retorn

Les funcions poden retornar valors de qualsevol tipus. Si una funció no retorna cap valor, el seu tipus de retorn és Unit.

Exemple

fun printMessage(message: String): Unit {
    println(message)
}

fun main() {
    printMessage("Hello, Kotlin!")  // Output: Hello, Kotlin!
}
  • : Unit indica que la funció no retorna cap valor.

Arguments per Defecte i Nombrats

Kotlin permet especificar valors per defecte per als paràmetres i utilitzar arguments nombrats per millorar la llegibilitat.

Exemple

fun greet(name: String = "Guest"): String {
    return "Hello, $name!"
}

fun main() {
    println(greet())  // Output: Hello, Guest!
    println(greet("Alice"))  // Output: Hello, Alice!
}
  • name: String = "Guest" estableix un valor per defecte per al paràmetre name.
  • greet() crida la funció sense arguments, utilitzant el valor per defecte.

Arguments Nombrats

fun formatMessage(message: String, prefix: String = "Info", suffix: String = "!"): String {
    return "$prefix: $message$suffix"
}

fun main() {
    println(formatMessage("System started"))  // Output: Info: System started!
    println(formatMessage("System started", suffix = "!!!"))  // Output: Info: System started!!!
}
  • Els arguments nombrats permeten especificar arguments pel seu nom, millorant la claredat.

Funcions d'Ordre Superior

Les funcions d'ordre superior són funcions que prenen altres funcions com a paràmetres o les retornen.

Exemple

fun calculate(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
    return operation(a, b)
}

fun main() {
    val sum = calculate(5, 3, ::add)
    println(sum)  // Output: 8
}
  • operation: (Int, Int) -> Int és un paràmetre de tipus funció.
  • ::add passa la funció add com a argument.

Expressions Lambda

Les lambdes són funcions anònimes que poden ser utilitzades per simplificar les funcions d'ordre superior.

Exemple

fun main() {
    val sum = calculate(5, 3) { x, y -> x + y }
    println(sum)  // Output: 8
}
  • { x, y -> x + y } és una expressió lambda que suma dos números.

Conclusió

Les funcions en Kotlin són versàtils i poderoses, permetent escriure codi net, modular i reutilitzable. Des de definicions bàsiques de funcions fins a conceptes avançats com funcions d'ordre superior i expressions lambda, dominar les funcions és essencial per a una programació efectiva en Kotlin. Practica definint, cridant i utilitzant funcions de diverses maneres per aprofundir la teva comprensió i millorar les teves habilitats de programació.
