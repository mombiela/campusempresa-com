Introducció a Kotlin

  • Visió general de Kotlin
  • Història i evolució de Kotlin
  • Per què Kotlin? Beneficis i casos d'ús

Començant amb Kotlin

  • Configuració de l'entorn de desenvolupament
    • Instal·lant IntelliJ IDEA
    • Configurant Kotlin a IntelliJ IDEA
  • Escriure el teu primer programa en Kotlin 
    fun main() {
    println("Hello, Kotlin!")
}
  • Entenent la sintaxi i l'estructura de Kotlin

Sintaxi bàsica i característiques

  • Variables i tipus de dades
    • Immutable (val) vs Mutable (var)
    val immutableValue: Int = 10
var mutableValue: Int = 20
  • Tipus de dades bàsics: Int, Double, String, Boolean
  • Inferència de tipus
  • Seguretat de nul·litat 
    var nullableString: String? = "Hello"
nullableString = null

Flux de control

  • Declaracions condicionals: if, else, when 
    val number = 10
val result = if (number > 0) "Positive" else "Negative"

val day = 3
val dayName = when(day) {
    1 -> "Monday"
    2 -> "Tuesday"
    3 -> "Wednesday"
    else -> "Unknown"
}
  • Bucles: for, while, do-while 
    for (i in 1..5) {
    println(i)
}

var x = 5
while (x > 0) {
    println(x)
    x--
}

Funcions

  • Definint i cridant funcions 
    fun greet(name: String): String {
    return "Hello, $name"
}

println(greet("Kotlin"))
  • Arguments per defecte i amb nom 
    fun printMessage(message: String = "Default Message") {
    println(message)
}

printMessage()
printMessage("Custom Message")
  • Funcions d'expressió única 
    fun add(a: Int, b: Int) = a + b

Programació orientada a objectes

  • Classes i objectes 
    class Person(val name: String, var age: Int)

val person = Person("John", 30)
println(person.name)
  • Herència 
    open class Animal(val name: String)

class Dog(name: String) : Animal(name)

val dog = Dog("Buddy")
println(dog.name)
  • Interfícies 
    interface Drivable {
    fun drive()
}

class Car : Drivable {
    override fun drive() {
        println("Driving a car")
    }
}

val car = Car()
car.drive()

Característiques avançades de Kotlin

  • Classes de dades 
    data class User(val name: String, val age: Int)

val user = User("Alice", 25)
println(user)
  • Classes segellades 
    sealed class Result
data class Success(val data: String) : Result()
data class Error(val message: String) : Result()

fun handleResult(result: Result) {
    when (result) {
        is Success -> println("Success: ${result.data}")
        is Error -> println("Error: ${result.message}")
    }
}
  • Funcions d'extensió 
    fun String.isPalindrome(): Boolean {
    return this == this.reversed()
}

println("madam".isPalindrome())
  • Coroutines per a programació asíncrona 
    import kotlinx.coroutines.*

fun main() = runBlocking {
    launch {
        delay(1000L)
        println("World!")
    }
    println("Hello,")
}

Programació funcional en Kotlin

  • Lambdas i funcions d'ordre superior 
    val sum: (Int, Int) -> Int = { a, b -> a + b }
println(sum(1, 2))

fun operate(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
    return operation(a, b)
}

println(operate(3, 4, sum))
  • Col·leccions i operacions funcionals 
    val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 }
println(doubled)

val filtered = numbers.filter { it % 2 == 0 }
println(filtered)

Millors pràctiques i consells

  • Escriure codi net i llegible
  • Utilitzar Kotlin idiomàtic
  • Ús efectiu de la biblioteca estàndard de Kotlin
  • Gestió d'errors i excepcions 
    try {
    val result = riskyOperation()
    println(result)
} catch (e: Exception) {
    println("An error occurred: ${e.message}")
}

Conclusió

  • Resum dels conceptes clau
  • Recursos addicionals per a més aprenentatge
  • Encoratjament per practicar i construir projectes

Al final d'aquest curs, hauríeu de tenir una comprensió sòlida de Kotlin, des de la sintaxi bàsica fins a les característiques avançades, i estar ben equipats per escriure codi efectiu i idiomàtic en Kotlin.
