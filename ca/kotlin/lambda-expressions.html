Introducció a les Expressions Lambda

Les expressions lambda són una manera concisa de representar funcions anònimes. Són una característica poderosa en Kotlin que et permet escriure codi més net i llegible. En aquesta secció, cobrirem els conceptes bàsics de les expressions lambda i com utilitzar-les de manera efectiva.

Què és una Expressió Lambda?

  • Una expressió lambda és essencialment una funció sense nom.
  • Es pot passar com a paràmetre a altres funcions o emmagatzemar en variables.
  • Sintaxi: { parameters -> body }

Sintaxi Bàsica

val sum = { a: Int, b: Int -> a + b }
println(sum(3, 4)) // Output: 7
  • val sum declara una expressió lambda que pren dos enters a i b i retorna la seva suma.
  • println(sum(3, 4)) crida l'expressió lambda amb els arguments 3 i 4.

Utilitzant Expressions Lambda

Assignant Lambda a Variables

Pots assignar una expressió lambda a una variable i utilitzar-la més tard.

val greet = { name: String -> "Hola, $name!" }
println(greet("Alice")) // Output: Hola, Alice!

Lambda com a Paràmetres de Funció

Les lambdas es poden passar com a paràmetres a funcions d'ordre superior.

fun operateOnNumbers(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
    return operation(a, b)
}

val result = operateOnNumbers(5, 3, { x, y -> x * y })
println(result) // Output: 15
  • operateOnNumbers és una funció que pren dos enters i una expressió lambda com a paràmetres.
  • L'expressió lambda { x, y -> x * y } es passa a operateOnNumbers.

Simplificant la Sintaxi Lambda

Kotlin et permet simplificar les expressions lambda en certs contextos.

  • Si l'últim paràmetre d'una funció és una lambda, es pot col·locar fora dels parèntesis.
  • Si una lambda té només un paràmetre, pots utilitzar it en lloc de nomenar explícitament el paràmetre.
val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 }
println(doubled) // Output: [2, 4, 6, 8, 10]

Característiques Avançades de les Lambdas

Capturant Variables

Les lambdas poden capturar variables del seu context circumdant.

var counter = 0
val incrementCounter = { counter += 1 }
incrementCounter()
println(counter) // Output: 1

Retornant Valors des de Lambdas

Les lambdas poden retornar valors igual que les funcions regulars.

val isEven = { number: Int -> number % 2 == 0 }
println(isEven(4)) // Output: true
println(isEven(7)) // Output: false

Lambda amb Receptor

Kotlin suporta lambdas amb receptors, que et permeten cridar mètodes sobre l'objecte receptor dins de la lambda.

val buildString = buildString {
    append("Hola, ")
    append("Món!")
}
println(buildString) // Output: Hola, Món!
  • buildString és una funció de la biblioteca estàndard que pren una lambda amb un receptor StringBuilder.

Conclusió

Les expressions lambda són una característica versàtil i poderosa en Kotlin que poden ajudar-te a escriure codi més concís i llegible. Entenent els conceptes bàsics i les característiques avançades de les expressions lambda, pots aprofitar-les per crear programes en Kotlin més eficients i expressius. Practica utilitzant lambdas en diferents contextos per sentir-te més còmode amb la seva sintaxi i capacitats.
