Introducció a les Col·leccions

Kotlin proporciona un conjunt ric de tipus de col·leccions que són tant mutables com immutables. Les col·leccions s'utilitzen per emmagatzemar grups d'objectes relacionats.

  • List: Una col·lecció ordenada d'elements.
  • Set: Una col·lecció d'elements únics.
  • Map: Una col·lecció de parells clau-valor.

List

Una List és una col·lecció ordenada que permet elements duplicats.

val fruits = listOf("Poma", "Plàtan", "Cirera")
println(fruits) // Output: [Poma, Plàtan, Cirera]

MutableList

Una MutableList és una List amb mètodes per afegir, eliminar i actualitzar elements.

val fruits = mutableListOf("Poma", "Plàtan")
fruits.add("Cirera")
println(fruits) // Output: [Poma, Plàtan, Cirera]

Set

Un Set és una col·lecció que no conté elements duplicats.

val numbers = setOf(1, 2, 3, 3)
println(numbers) // Output: [1, 2, 3]

MutableSet

Un MutableSet és un Set amb mètodes per afegir, eliminar i actualitzar elements.

val numbers = mutableSetOf(1, 2, 3)
numbers.add(4)
println(numbers) // Output: [1, 2, 3, 4]

Map

Un Map és una col·lecció de parells clau-valor.

val countryCodes = mapOf("US" to "Estats Units", "CA" to "Canadà")
println(countryCodes) // Output: {US=Estats Units, CA=Canadà}

MutableMap

Un MutableMap és un Map amb mètodes per afegir, eliminar i actualitzar parells clau-valor.

val countryCodes = mutableMapOf("US" to "Estats Units")
countryCodes["CA"] = "Canadà"
println(countryCodes) // Output: {US=Estats Units, CA=Canadà}

Operacions Funcionals en Col·leccions

Les col·leccions de Kotlin venen amb una varietat d'operacions funcionals que faciliten la manipulació de dades.

  • map: Transforma cada element de la col·lecció.
  • filter: Filtra elements basant-se en una condició.
  • reduce: Redueix la col·lecció a un sol valor.
  • forEach: Realitza una acció per a cada element.
  • sortedBy: Ordena la col·lecció basant-se en una funció selectora.

map

Transforma cada element de la col·lecció.

val numbers = listOf(1, 2, 3)
val doubled = numbers.map { it * 2 }
println(doubled) // Output: [2, 4, 6]

filter

Filtra elements basant-se en una condició.

val numbers = listOf(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter { it % 2 == 0 }
println(evenNumbers) // Output: [2, 4]

reduce

Redueix la col·lecció a un sol valor.

val numbers = listOf(1, 2, 3, 4)
val sum = numbers.reduce { acc, num -> acc + num }
println(sum) // Output: 10

forEach

Realitza una acció per a cada element.

val fruits = listOf("Poma", "Plàtan", "Cirera")
fruits.forEach { println(it) }
// Output:
// Poma
// Plàtan
// Cirera

sortedBy

Ordena la col·lecció basant-se en una funció selectora.

val fruits = listOf("Plàtan", "Poma", "Cirera")
val sortedFruits = fruits.sortedBy { it }
println(sortedFruits) // Output: [Poma, Plàtan, Cirera]

Conclusió

El marc de col·leccions de Kotlin proporciona una manera poderosa i flexible de treballar amb grups de dades. Entendre els diferents tipus de col·leccions i les seves operacions funcionals et permetrà escriure codi més concís i llegible. En dominar aquests conceptes, podràs manipular dades de manera eficient i realitzar transformacions complexes amb facilitat.
