Funcions d'ordre superior a Kotlin
Col·leccions i Operacions Funcionals ►

Les funcions d'ordre superior són una característica poderosa a Kotlin que et permeten escriure codi més flexible i reutilitzable. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de les funcions d'ordre superior a Kotlin.

Què són les Funcions d'Ordre Superior?

Les funcions d'ordre superior són funcions que prenen altres funcions com a paràmetres, retornen una funció, o fan ambdues coses. Són una característica clau de la programació funcional i poden ajudar a fer que el teu codi sigui més modular i expressiu.

Conceptes Clau

  • Tipus de Funció: El tipus d'una funció que es pot passar com a paràmetre o retornar.
  • Expressió Lambda: Una manera concisa de representar una funció.
  • Funció Anònima: Una funció sense nom, sovint utilitzada com a paràmetre.
  • Funcions Inline: Funcions que es poden inlinar per reduir la sobrecàrrega.

Ús Bàsic

Tipus de Funcions

A Kotlin, pots definir un tipus de funció utilitzant la sintaxi (Tipus) -> TipusDeRetorn.

// Una funció que pren un Int i retorna un String
val intToString: (Int) -> String = { it.toString() }

Passar Funcions com a Paràmetres

Pots passar funcions com a paràmetres a altres funcions.

fun performOperation(x: Int, operation: (Int) -> Int): Int {
    return operation(x)
}

val square: (Int) -> Int = { it * it }

fun main() {
    println(performOperation(4, square)) // Sortida: 16
}

Retornar Funcions

Les funcions també poden retornar altres funcions.

fun createMultiplier(factor: Int): (Int) -> Int {
    return { number -> number * factor }
}

fun main() {
    val multiplyBy2 = createMultiplier(2)
    println(multiplyBy2(5)) // Sortida: 10
}

Expressions Lambda

Les lambdes són una manera concisa de definir funcions. Es poden utilitzar allà on s'esperen tipus de funció.

val add: (Int, Int) -> Int = { a, b -> a + b }

fun main() {
    println(add(3, 4)) // Sortida: 7
}

Sintaxi

  • Sintaxi Completa: { x: Int, y: Int -> x + y }
  • Inferència de Tipus: { x, y -> x + y }
  • Un Sol Paràmetre: { it * 2 }

Funcions Anònimes

Les funcions anònimes són similars a les lambdes però amb una sintaxi més explícita.

val subtract = fun(x: Int, y: Int): Int {
    return x - y
}

fun main() {
    println(subtract(10, 3)) // Sortida: 7
}

Funcions Inline

Les funcions inline es poden utilitzar per reduir la sobrecàrrega de les funcions d'ordre superior inlinant el cos de la funció al lloc de la crida.

inline fun <T> measureTime(block: () -> T): T {
    val start = System.currentTimeMillis()
    val result = block()
    val end = System.currentTimeMillis()
    println("Temps emprat: ${end - start}ms")
    return result
}

fun main() {
    measureTime {
        // Alguna operació de llarga durada
        Thread.sleep(1000)
    }
}

Ús Avançat

Composició de Funcions

Pots composar funcions per crear operacions més complexes.

fun <A, B, C> compose(f: (B) -> C, g: (A) -> B): (A) -> C {
    return { x -> f(g(x)) }
}

val multiplyBy2 = { x: Int -> x * 2 }
val add3 = { x: Int -> x + 3 }

fun main() {
    val multiplyAndAdd = compose(add3, multiplyBy2)
    println(multiplyAndAdd(4)) // Sortida: 11
}

Currificació

La currificació transforma una funció que pren múltiples arguments en una sèrie de funcions que cadascuna pren un sol argument.

fun add(x: Int) = { y: Int -> x + y }

fun main() {
    val add5 = add(5)
    println(add5(3)) // Sortida: 8
}

Conclusió

Les funcions d'ordre superior són una eina versàtil a Kotlin que poden ajudar-te a escriure codi més expressiu i reutilitzable. Entenent com utilitzar els tipus de funció, les expressions lambda, les funcions anònimes i les funcions inline, pots aprofitar tot el potencial de les capacitats de programació funcional de Kotlin. Practica aquests conceptes amb els exemples proporcionats i intenta crear les teves pròpies funcions d'ordre superior per solidificar la teva comprensió.
