Introducció

En aquesta secció, cobrirem els conceptes bàsics de les variables i els tipus de dades en Kotlin. Entendre aquests conceptes és fonamental per a qualsevol programador de Kotlin, ja siguis principiant o estiguis avançant en les teves habilitats.

Variables

Les variables són contenidors per emmagatzemar valors de dades. En Kotlin, pots declarar variables utilitzant val o var.

  • val (valor): Variable immutable, el que significa que un cop se li assigna un valor, no es pot canviar.
  • var (variable): Variable mutable, el que significa que el seu valor es pot canviar.

Declaració de Variables

Així és com pots declarar variables en Kotlin:

val immutableVariable: String = "Hola, Kotlin!"
var mutableVariable: Int = 42
  • immutableVariable és un val, així que el seu valor no es pot canviar després de la inicialització.
  • mutableVariable és un var, així que el seu valor es pot canviar.

Inferència de Tipus

Kotlin pot inferir el tipus de la variable a partir del valor assignat, així que pots ometre la declaració del tipus:

val inferredString = "Hola, Kotlin!"
var inferredInt = 42

Tipus de Dades

Kotlin té un conjunt ric de tipus de dades. Aquests són els principals tipus de dades que trobaràs:

  • Tipus Numèrics: Byte, Short, Int, Long, Float, Double
  • Caràcter: Char
  • Booleà: Boolean
  • Cadena: String
  • Arrays: Array

Tipus Numèrics

Kotlin proporciona una varietat de tipus numèrics per manejar diferents rangs i precisions.

| Tipus | Mida (bits) | Valor Mínim | Valor Màxim | |-------|-------------|-----------------------------|-----------------------------| | Byte | 8 | -128 | 127 | | Short | 16 | -32,768 | 32,767 | | Int | 32 | -2,147,483,648 | 2,147,483,647 | | Long | 64 | -9,223,372,036,854,775,808 | 9,223,372,036,854,775,807 | | Float | 32 | 1.4E-45 | 3.4028235E38 | | Double| 64 | 4.9E-324 | 1.7976931348623157E308 |

Tipus de Caràcter i Booleà

  • Char: Representa un sol caràcter.
  • Boolean: Representa un valor que pot ser true o false.
val charExample: Char = 'A'
val booleanExample: Boolean = true

Tipus de Cadena

Les cadenes en Kotlin són seqüències de caràcters. Són immutables.

val stringExample: String = "Hola, Kotlin!"

Arrays

Els arrays en Kotlin estan representats per la classe Array. Pots crear un array utilitzant la funció arrayOf.

val intArray: Array<Int> = arrayOf(1, 2, 3, 4, 5)
val stringArray: Array<String> = arrayOf("Kotlin", "Java", "Swift")

Exemples i Exercicis

Exemple 1: Declaració Bàsica de Variables

fun main() {
    val name: String = "John"
    var age: Int = 25

    println("Nom: $name")
    println("Edat: $age")

    // Intentar canviar el valor d'un val causarà un error de compilació
    // name = "Doe" // Descomentar aquesta línia causarà un error

    // Canviar el valor d'un var està permès
    age = 26
    println("Edat Actualitzada: $age")
}

Exercici 1: Declaració de Variables

  1. Declara una variable val per emmagatzemar el teu color favorit.
  2. Declara una variable var per emmagatzemar la teva edat actual.
  3. Imprimeix ambdues variables.

Exemple 2: Utilitzant Diferents Tipus de Dades

fun main() {
    val byteValue: Byte = 127
    val shortValue: Short = 32767
    val intValue: Int = 2147483647
    val longValue: Long = 9223372036854775807L
    val floatValue: Float = 3.14F
    val doubleValue: Double = 3.141592653589793

    println("Valor Byte: $byteValue")
    println("Valor Short: $shortValue")
    println("Valor Int: $intValue")
    println("Valor Long: $longValue")
    println("Valor Float: $floatValue")
    println("Valor Double: $doubleValue")
}

Exercici 2: Tipus de Dades

  1. Declara variables de cada tipus numèric i assigna'ls valors apropiats.
  2. Imprimeix cada variable.

Conclusió

Entendre les variables i els tipus de dades és crucial per programar en Kotlin. En dominar aquests conceptes bàsics, estableixes una base sòlida per a temes més avançats. Recorda practicar la declaració i l'ús de diferents tipus de variables per consolidar la teva comprensió.
