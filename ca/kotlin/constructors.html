Introducció als Constructors

Els constructors són funcions especials a Kotlin utilitzades per inicialitzar objectes. Juguen un paper crucial en la configuració de l'estat inicial d'un objecte. Kotlin proporciona dos tipus de constructors:

  • Constructors Primaris
  • Constructors Secundaris

Constructors Primaris

El constructor primari és part de la capçalera de la classe i s'utilitza per inicialitzar la classe. És concís i està directament associat amb la declaració de la classe.

Sintaxi i Exemple

class Person(val name: String, var age: Int)
  • Person és el nom de la classe.
  • val name: String i var age: Int són propietats inicialitzades pel constructor primari.

Explicació

  • Les paraules clau val i var defineixen propietats que s'inicialitzen a través del constructor.
  • name és una propietat de només lectura (immutable).
  • age és una propietat mutable.

Exemple amb Bloc d'Inicialització

Si necessites realitzar una inicialització addicional, pots utilitzar el bloc init.

class Person(val name: String, var age: Int) {
    init {
        println("El nom de la persona és $name i l'edat és $age")
    }
}
  • El bloc init s'executa immediatament després del constructor primari.

Constructors Secundaris

Els constructors secundaris s'utilitzen quan necessites més flexibilitat en la inicialització de la classe. Es defineixen utilitzant la paraula clau constructor.

Sintaxi i Exemple

class Person {
    var name: String
    var age: Int

    constructor(name: String, age: Int) {
        this.name = name
        this.age = age
    }
}
  • constructor(name: String, age: Int) és el constructor secundari.
  • this.name i this.age es refereixen a les propietats de la classe.

Explicació

  • Els constructors secundaris permeten una lògica d'inicialització addicional.
  • Poden cridar el constructor primari utilitzant this() si la classe en té un.

Exemple amb Múltiples Constructors

class Person {
    var name: String
    var age: Int

    constructor(name: String) {
        this.name = name
        this.age = 0 // Edat per defecte
    }

    constructor(name: String, age: Int) {
        this.name = name
        this.age = age
    }
}
  • La classe Person té dos constructors secundaris.
  • El primer constructor inicialitza name i estableix una age per defecte.
  • El segon constructor inicialitza tant name com age.

Combinació de Constructors Primaris i Secundaris

Pots combinar constructors primaris i secundaris per a una inicialització més complexa.

Exemple

class Person(val name: String) {
    var age: Int = 0

    constructor(name: String, age: Int) : this(name) {
        this.age = age
    }
}
  • El constructor primari inicialitza name.
  • El constructor secundari inicialitza age i crida el constructor primari utilitzant : this(name).

Paràmetres del Constructor i Valors per Defecte

Kotlin et permet proporcionar valors per defecte per als paràmetres del constructor, fent els constructors més flexibles.

Exemple

class Person(val name: String, var age: Int = 0)
  • age té un valor per defecte de 0.
  • Pots crear un objecte Person amb o sense especificar l'age.

Ús

val person1 = Person("Alice")
val person2 = Person("Bob", 25)
  • person1 té name "Alice" i age per defecte 0.
  • person2 té name "Bob" i age 25.

Resum

  • Constructors Primaris: Part de la capçalera de la classe, concisos, i poden utilitzar blocs init per a una inicialització addicional.
  • Constructors Secundaris: Proporcionen més flexibilitat, utilitzen la paraula clau constructor, i poden cridar constructors primaris.
  • Valors per Defecte: Simplifiquen la creació d'objectes proporcionant valors per defecte per als paràmetres del constructor.

Entendre els constructors a Kotlin és essencial per crear objectes ben inicialitzats i robustos. Dominant tant els constructors primaris com els secundaris, pots gestionar diversos escenaris d'inicialització de manera efectiva.
