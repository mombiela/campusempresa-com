Introducció a Kotlin

Kotlin és un llenguatge de programació modern, tipat estàticament, que s'executa a la Java Virtual Machine (JVM) i també es pot compilar a JavaScript o codi natiu. Està dissenyat per ser totalment interoperable amb Java i ofereix una sintaxi més concisa, cosa que el fa una elecció popular per al desenvolupament d'Android i altres aplicacions.

Conceptes Clau

  • Variables i Constants
  • Tipus de Dades
  • Flux de Control
  • Funcions
  • Classes i Objectes

Variables i Constants

A Kotlin, pots declarar variables usant var i constants usant val.

Variables

var mutableVariable: Int = 10
mutableVariable = 20 // Això està permès
  • var s'utilitza per declarar una variable mutable, és a dir, el seu valor pot canviar.

Constants

val immutableVariable: Int = 10
immutableVariable = 20 // Això causarà un error de compilació
  • val s'utilitza per declarar una variable de només lectura, és a dir, el seu valor no pot canviar un cop assignat.

Tipus de Dades

Kotlin té un conjunt ric de tipus de dades integrats.

Tipus de Dades Bàsics

  • Nombres: Int, Long, Float, Double
  • Caràcters: Char
  • Booleans: Boolean
  • Cadenes: String

Exemple

val anInt: Int = 42
val aDouble: Double = 3.14
val aChar: Char = 'A'
val aBoolean: Boolean = true
val aString: String = "Hola, Kotlin!"

Flux de Control

Kotlin proporciona diverses maneres de controlar el flux del teu programa.

If-Else

val max = if (a > b) a else b
  • La instrucció if es pot utilitzar com una expressió.

When

val result = when (x) {
    1 -> "Un"
    2 -> "Dos"
    else -> "Desconegut"
}
  • when és una versió més potent de la instrucció switch en altres llenguatges.

Bucles

Bucle For

for (i in 1..10) {
    println(i)
}
  • El bucle for itera sobre un rang.

Bucle While

var i = 0
while (i < 10) {
    println(i)
    i++
}
  • El bucle while continua mentre la condició sigui certa.

Funcions

Les funcions a Kotlin són ciutadans de primera classe i es poden declarar de diverses maneres.

Funció Bàsica

fun add(a: Int, b: Int): Int {
    return a + b
}
  • Les funcions es declaren usant la paraula clau fun.

Funció d'Una Sola Expressió

fun add(a: Int, b: Int): Int = a + b
  • Les funcions amb una sola expressió es poden escriure de manera més concisa.

Arguments per Defecte

fun greet(name: String = "Món") {
    println("Hola, $name!")
}
  • Les funcions poden tenir valors de paràmetre per defecte.

Classes i Objectes

Kotlin suporta la programació orientada a objectes amb classes i objectes.

Classe Bàsica

class Person(val name: String, var age: Int)
  • Una classe pot tenir propietats declarades en el constructor principal.

Creació d'un Objecte

val person = Person("Joan", 30)
println(person.name) // Joan
println(person.age)  // 30
  • Els objectes es creen usant la paraula clau new.

Mètodes en Classes

class Person(val name: String, var age: Int) {
    fun greet() {
        println("Hola, em dic $name i tinc $age anys.")
    }
}
  • Els mètodes es poden definir dins de les classes.

Conclusió

Entendre la sintaxi i l'estructura bàsica de Kotlin és crucial per escriure codi efectiu i eficient. Això inclou saber com declarar variables i constants, entendre els tipus de dades, controlar el flux del teu programa, definir funcions i crear classes i objectes. Dominar aquests conceptes fonamentals proporcionarà una base sòlida per a temes de programació Kotlin més avançats.
