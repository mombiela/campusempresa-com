Els genèrics en Kotlin et permeten escriure codi flexible i reutilitzable. Et permeten crear classes, interfícies i funcions que funcionen amb qualsevol tipus de dada mentre proporcionen seguretat de tipus. Aquest tema et guiarà des dels conceptes bàsics dels genèrics fins a conceptes més avançats.

Introducció als Genèrics

Els genèrics són una característica poderosa en Kotlin que et permet definir classes, interfícies i funcions amb paràmetres de tipus. Això significa que pots escriure codi que funcioni amb diferents tipus sense sacrificar la seguretat de tipus.

Per què utilitzar genèrics?

  • Seguretat de tipus: Els genèrics asseguren que detectis errors de tipus en temps de compilació en lloc de temps d'execució.
  • Reutilitzabilitat: Pots escriure codi més flexible i reutilitzable.
  • Rendiment: Els genèrics poden ajudar a evitar la necessitat de fer conversions de tipus, cosa que pot millorar el rendiment.

Sintaxi bàsica

Per definir una classe o funció genèrica, utilitzes claudàtors angulars (<>) per especificar el paràmetre de tipus.

// Classe genèrica
class Box<T>(val value: T)

// Funció genèrica
fun <T> printItem(item: T) {
    println(item)
}

En els exemples anteriors, T és un paràmetre de tipus que es pot substituir per qualsevol tipus.

Classes Genèriques

Les classes genèriques et permeten crear classes que poden manejar diferents tipus de dades.

Exemple: Classe genèrica Box

class Box<T>(val value: T)

fun main() {
    val intBox = Box(1)
    val stringBox = Box("Hola")

    println(intBox.value)  // Sortida: 1
    println(stringBox.value)  // Sortida: Hola
}

En aquest exemple, Box pot contenir qualsevol tipus de valor, fent-lo un contenidor versàtil.

Múltiples paràmetres de tipus

Pots definir una classe amb múltiples paràmetres de tipus.

class Pair<A, B>(val first: A, val second: B)

fun main() {
    val pair = Pair(1, "Un")
    println(pair.first)  // Sortida: 1
    println(pair.second)  // Sortida: Un
}

Funcions Genèriques

Les funcions genèriques et permeten escriure funcions que poden operar amb diferents tipus.

Exemple: Funció genèrica Print

fun <T> printItem(item: T) {
    println(item)
}

fun main() {
    printItem(123)  // Sortida: 123
    printItem("Hola")  // Sortida: Hola
}

En aquest exemple, printItem pot imprimir qualsevol tipus d'element.

Restriccions en els Paràmetres de Tipus

Pots restringir els tipus que es poden utilitzar com a paràmetres de tipus utilitzant restriccions.

Límits Superiors

Pots especificar un límit superior per a un paràmetre de tipus utilitzant el símbol :.

fun <T : Number> sum(a: T, b: T): Double {
    return a.toDouble() + b.toDouble()
}

fun main() {
    println(sum(1, 2))  // Sortida: 3.0
    println(sum(1.5, 2.5))  // Sortida: 4.0
    // println(sum("1", "2"))  // Error: Incompatibilitat de tipus
}

En aquest exemple, T està restringit a ser un subtipo de Number.

Variància

La variància et permet definir com la subtipificació entre tipus més complexos es relaciona amb la subtipificació entre els seus components.

Covariància

La covariància permet que un tipus sigui un subtipo d'un altre tipus. Utilitzes la paraula clau out per declarar un paràmetre de tipus com a covariant.

class Box<out T>(val value: T)

fun main() {
    val intBox: Box<Int> = Box(1)
    val numberBox: Box<Number> = intBox  // La covariància permet això
    println(numberBox.value)  // Sortida: 1
}

Contravariància

La contravariància permet que un tipus sigui un supertipus d'un altre tipus. Utilitzes la paraula clau in per declarar un paràmetre de tipus com a contravariant.

class Box<in T> {
    fun put(value: T) { /*...*/ }
}

fun main() {
    val numberBox: Box<Number> = Box()
    val intBox: Box<Int> = numberBox  // La contravariància permet això
    intBox.put(1)  // Això està permès
}

Projeccions de Tipus

Les projeccions de tipus et permeten utilitzar tipus genèrics d'una manera més flexible.

Projeccions Estrella

Les projeccions estrella (*) et permeten utilitzar un tipus genèric sense especificar el paràmetre de tipus exacte.

fun printBox(box: Box<*>) {
    println(box.value)
}

fun main() {
    val intBox = Box(1)
    val stringBox = Box("Hola")

    printBox(intBox)  // Sortida: 1
    printBox(stringBox)  // Sortida: Hola
}

Resum

Els genèrics en Kotlin proporcionen una manera poderosa d'escriure codi flexible, reutilitzable i segur de tipus. Entenent els conceptes bàsics de les classes i funcions genèriques, les restriccions, la variància i les projeccions de tipus, pots crear aplicacions més robustes i versàtils. Practica utilitzant genèrics en els teus projectes per comprendre plenament el seu potencial i beneficis.
