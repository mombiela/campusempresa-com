El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar l'entorn de desenvolupament és el primer pas en el teu viatge per aprendre Kotlin. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per escriure i executar programes Kotlin de manera eficient.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb connexió a internet.
  • Comprensió bàsica de com navegar pel teu sistema operatiu (Windows, macOS o Linux).

Instal·lació del JDK (Java Development Kit)

Kotlin s'executa a la Java Virtual Machine (JVM), així que necessites instal·lar el JDK.

  1. Descarrega el JDK:

    • Visita la pàgina de descàrrega de l'Oracle JDK o AdoptOpenJDK.
    • Tria la versió compatible amb el teu sistema operatiu.

  2. Instal·la el JDK:

    • Segueix les instruccions d'instal·lació específiques per al teu sistema operatiu.
    • Assegura't que el JDK s'afegeixi a la variable d'entorn PATH del teu sistema.

  3. Verifica la instal·lació:

    • Obre un terminal o símbol del sistema.
    • Escriu java -version i prem Enter. Hauries de veure la versió del JDK que has instal·lat.

Instal·lació d'un Entorn de Desenvolupament Integrat (IDE)

Un IDE proporciona eines per escriure, depurar i executar el teu codi Kotlin de manera eficient. L'IDE més popular per a Kotlin és IntelliJ IDEA.

  1. Descarrega IntelliJ IDEA:

    • Visita el lloc web de JetBrains.
    • Tria l'edició Community (gratuïta) o l'edició Ultimate (de pagament).

  2. Instal·la IntelliJ IDEA:

    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.

  3. Configura IntelliJ IDEA per a Kotlin:

    • Obre IntelliJ IDEA.
    • Ves a File > New > Project.
    • Selecciona Kotlin de la llista de tipus de projectes.
    • Tria JVM | IDEA i fes clic a Next.
    • Segueix les instruccions per crear un nou projecte Kotlin.

Configuració de les Eines de Línia de Comandes de Kotlin

Si prefereixes treballar des de la línia de comandes, pots configurar les eines de línia de comandes de Kotlin.

  1. Descarrega el Compilador de Kotlin:

    • Visita el lloc web de Kotlin.

  2. Instal·la el Compilador de Kotlin:

    • Extreu l'arxiu descarregat a un directori de la teva elecció.
    • Afegeix el directori bin de l'arxiu extret a la variable d'entorn PATH del teu sistema.

  3. Verifica la instal·lació:

    • Obre un terminal o símbol del sistema.
    • Escriu kotlinc -version i prem Enter. Hauries de veure la versió del compilador de Kotlin que has instal·lat.

Escriure el Teu Primer Programa en Kotlin

Escrivim un programa senzill en Kotlin per assegurar-nos que tot està configurat correctament.

  1. Utilitzant IntelliJ IDEA:

    • Obre IntelliJ IDEA i crea un nou projecte Kotlin.

    • Crea un nou fitxer Kotlin: File > New > Kotlin File/Class.

    • Anomena el fitxer HelloWorld.kt.

    • Afegeix el següent codi a HelloWorld.kt:

      fun main() {
    println("Hola, Món!")
}

    • Executa el programa fent clic al botó verd d'execució o fent clic dret al fitxer i seleccionant Run 'HelloWorldKt'.

  2. Utilitzant la Línia de Comandes:

    • Obre un editor de text i crea un nou fitxer anomenat HelloWorld.kt.

    • Afegeix el següent codi a HelloWorld.kt:

      fun main() {
    println("Hola, Món!")
}

    • Desa el fitxer i obre un terminal o símbol del sistema.

    • Navega al directori que conté HelloWorld.kt.

    • Compila el programa utilitzant la comanda: kotlinc HelloWorld.kt -include-runtime -d HelloWorld.jar.

    • Executa el programa utilitzant la comanda: java -jar HelloWorld.jar.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu viatge de programació en Kotlin. Instal·lant el JDK, configurant IntelliJ IDEA i configurant les eines de línia de comandes, ara estàs preparat per començar a escriure i executar programes Kotlin. En les següents seccions, aprofundirem en la sintaxi de Kotlin, les seves característiques i les millors pràctiques. Feliç programació!
