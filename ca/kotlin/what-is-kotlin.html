Què és Kotlin?
Configuració de l'Entorn de Desenvolupament ►

Kotlin és un llenguatge de programació modern, tipat estàticament, que s'executa a la Java Virtual Machine (JVM) i també es pot compilar a JavaScript o codi natiu. Va ser desenvolupat per JetBrains i està dissenyat per ser totalment interoperable amb Java. Kotlin és conegut per la seva sintaxi concisa, característiques de seguretat i paradigmes de programació moderns.

Característiques Clau de Kotlin

  • Interoperabilitat: Kotlin és totalment interoperable amb Java, el que significa que pots utilitzar biblioteques i frameworks de Java dins de codi Kotlin i viceversa.
  • Concisença: Kotlin redueix el codi boilerplate, fent que la teva base de codi sigui més llegible i mantenible.
  • Seguretat de Nulls: El sistema de tipus de Kotlin està dissenyat per eliminar excepcions de punters nuls fent que tots els tipus siguin no-nullables per defecte.
  • Funcions d'Extensió: Kotlin et permet estendre la funcionalitat de classes existents sense modificar el seu codi font.
  • Coroutines: Kotlin proporciona suport integrat per a coroutines, que simplifiquen la programació asíncrona.
  • Casts Intel·ligents: Kotlin automàticament fa casts de tipus quan és segur fer-ho, reduint la necessitat de casts explícits.

Per què Escollir Kotlin?

  • Característiques Modernes del Llenguatge: Kotlin inclou moltes característiques modernes del llenguatge que milloren la productivitat i la seguretat del codi.
  • Llenguatge Oficial de Google per a Android: Kotlin està oficialment suportat per Google per al desenvolupament d'Android, fent-lo una elecció preferida per als desenvolupadors mòbils.
  • Comunitat i Ecosistema: Kotlin té una comunitat creixent i un ric ecosistema de biblioteques i eines.

Sintaxi Bàsica i Exemples

Vegem una mica de sintaxi bàsica de Kotlin per entendre com funciona.

Exemple de Hello World

fun main() {
    println("Hello, World!")
}
  • La paraula clau fun s'utilitza per declarar una funció.
  • main és el punt d'entrada del programa.
  • println és una funció de la biblioteca estàndard per imprimir text a la consola.

Variables

Kotlin té dos tipus de variables: val per a variables de només lectura i var per a variables mutables.

val name: String = "Kotlin" // Variable immutable
var age: Int = 10           // Variable mutable
  • val s'utilitza per a variables que no es poden reasignar.
  • var s'utilitza per a variables que es poden reasignar.

Seguretat de Nulls

El sistema de tipus de Kotlin diferencia entre tipus nullable i no-nullable.

var nullableName: String? = null // Tipus nullable
var nonNullableName: String = "Kotlin" // Tipus no-nullable
  • String? denota un tipus nullable.
  • String denota un tipus no-nullable.

Funcions

Les funcions en Kotlin són ciutadans de primera classe i es poden definir de diverses maneres.

fun add(a: Int, b: Int): Int {
    return a + b
}

// Funció d'expressió única
fun multiply(a: Int, b: Int) = a * b
  • Les funcions poden tenir un tipus de retorn especificat després de la llista de paràmetres.
  • Les funcions d'expressió única poden ometre el tipus de retorn si es pot inferir.

Conclusió

Kotlin és un llenguatge de programació potent i versàtil que ofereix moltes característiques modernes dissenyades per millorar la productivitat i la seguretat del codi. La seva interoperabilitat amb Java, sintaxi concisa i conjunt robust de característiques el fan una excel·lent elecció per a una àmplia gamma d'aplicacions, des del desenvolupament mòbil fins a la programació del costat del servidor. A mesura que avancis en aquest curs, adquiriràs una comprensió més profunda de les capacitats de Kotlin i com aprofitar-les per escriure codi eficient i mantenible.
