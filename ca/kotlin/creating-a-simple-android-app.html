Introducció

En aquesta secció, et guiarem a través del procés de creació d'una aplicació Android simple utilitzant Kotlin. Aquest curs està dissenyat per a principiants i et portarà gradualment a un nivell avançat. Al final d'aquest tema, tindràs una comprensió sòlida de com desenvolupar una aplicació bàsica d'Android.

Configuració del Teu Entorn de Desenvolupament

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·la Android Studio: Descarrega i instal·la l'última versió d'Android Studio des del lloc web oficial.
  • Configura Android Studio: Segueix l'assistent de configuració per configurar Android Studio i instal·lar els SDKs necessaris.
  • Crea un Nou Projecte: Obre Android Studio i crea un nou projecte amb una activitat buida.

Comprensió de l'Estructura del Projecte

Un projecte Android consta de diversos components clau:

  • Fitxer Manifest: AndroidManifest.xml conté informació essencial sobre la teva aplicació.
  • Fitxers Java/Kotlin: Conté el codi font de la teva aplicació.
  • Recursos: Inclou dissenys, cadenes de text, imatges i altres recursos.

Creació de la Teva Primera Activitat

Una activitat representa una sola pantalla amb una interfície d'usuari.

Pas 1: Defineix el Disseny

Crea un disseny simple per a la teva activitat a activity_main.xml.

<!-- res/layout/activity_main.xml -->
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <TextView
        android:id="@+id/helloTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:layout_centerInParent="true"/>
</RelativeLayout>

Pas 2: Implementa l'Activitat

Escriu el codi Kotlin per a la teva activitat a MainActivity.kt.

// MainActivity.kt
package com.example.myapplication

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}

Executar la Teva Aplicació

  • Connecta un Dispositiu: Connecta un dispositiu Android via USB o utilitza un emulador.
  • Executa l'Aplicació: Fes clic al botó "Run" a Android Studio per construir i desplegar la teva aplicació.

Afegir Interactivitat

Afegim un botó per canviar el text del TextView quan es faci clic.

Pas 1: Actualitza el Disseny

Afegeix un botó a activity_main.xml.

<!-- res/layout/activity_main.xml -->
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <TextView
        android:id="@+id/helloTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:layout_centerInParent="true"/>

    <Button
        android:id="@+id/changeTextButton"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Change Text"
        android:layout_below="@id/helloTextView"
        android:layout_centerHorizontal="true"
        android:layout_marginTop="20dp"/>
</RelativeLayout>

Pas 2: Actualitza l'Activitat

Modifica MainActivity.kt per gestionar els clics del botó.

// MainActivity.kt
package com.example.myapplication

import android.os.Bundle
import android.widget.Button
import android.widget.TextView
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        val helloTextView: TextView = findViewById(R.id.helloTextView)
        val changeTextButton: Button = findViewById(R.id.changeTextButton)

        changeTextButton.setOnClickListener {
            helloTextView.text = "Text Changed!"
        }
    }
}

Conclusió

En aquest tema, hem cobert els conceptes bàsics de la creació d'una aplicació Android simple utilitzant Kotlin. Hem configurat l'entorn de desenvolupament, hem entès l'estructura del projecte, hem creat una activitat i hem afegit interactivitat. Aquest coneixement fonamental t'ajudarà a mesura que avancis cap a temes més avançats en el desenvolupament d'Android. Continua experimentant i construint aplicacions més complexes per millorar les teves habilitats!
