El scripting en Kotlin és una característica potent que et permet escriure scripts en Kotlin, permetent un prototipat ràpid, automatització i configuració sense la necessitat d'una configuració completa del projecte. Aquesta secció et guiarà a través dels conceptes bàsics del scripting en Kotlin, els seus casos d'ús i com escriure i executar scripts en Kotlin.

Introducció al Scripting en Kotlin

  • Què és el Scripting en Kotlin?

    • El scripting en Kotlin et permet escriure i executar codi Kotlin sense la necessitat d'una configuració completa del projecte.
    • És útil per a tasques com l'automatització, el prototipat ràpid i la configuració.

  • Casos d'Ús del Scripting en Kotlin

    • Automatització de tasques repetitives.
    • Escriure scripts de construcció.
    • Crear petites utilitats.
    • Prototipat i proves de fragments de codi.

Configuració del Scripting en Kotlin

  • Prerequisits

    • Assegura't de tenir el compilador de Kotlin instal·lat. Pots descarregar-lo des del lloc web oficial de Kotlin.

  • Creació d'un Fitxer de Script en Kotlin

    • Els fitxers de script en Kotlin tenen l'extensió .kts.
    • Pots crear un fitxer de script en Kotlin utilitzant qualsevol editor de text o un IDE com IntelliJ IDEA.

Exemple: Script de Hello World

// hello.kts
println("Hello, Kotlin Scripting!")
  • Executar el Script
    • Pots executar el script utilitzant el compilador de Kotlin des de la línia de comandes: 
      kotlinc -script hello.kts

Escriure Scripts en Kotlin

  • Sintaxi Bàsica
    • Els scripts en Kotlin utilitzen la mateixa sintaxi que el codi Kotlin regular.
    • Pots definir variables, funcions i classes dins d'un script.

Exemple: Variables i Funcions

// script.kts
val greeting = "Hello"
val name = "World"

fun greet(greeting: String, name: String) {
    println("$greeting, $name!")
}

greet(greeting, name)
  • Imports i Dependències
    • Pots importar funcions i classes de la llibreria estàndard de Kotlin.
    • Les dependències externes es poden afegir utilitzant l'anotació @file:DependsOn.

Exemple: Utilitzar una Llibreria Externa

// script.kts
@file:DependsOn("com.squareup.okhttp3:okhttp:4.9.0")

import okhttp3.OkHttpClient
import okhttp3.Request

val client = OkHttpClient()
val request = Request.Builder().url("https://api.github.com").build()
val response = client.newCall(request).execute()

println(response.body?.string())

Característiques Avançades del Scripting

  • Configuració del Script

    • Els scripts es poden configurar utilitzant anotacions com @file:DependsOn i @file:Repository.

  • Utilitzar el Scripting en Kotlin amb Gradle

    • Els scripts en Kotlin es poden utilitzar en fitxers de construcció de Gradle (build.gradle.kts).

Exemple: Script de Construcció de Gradle

// build.gradle.kts
plugins {
    kotlin("jvm") version "1.5.21"
}

repositories {
    mavenCentral()
}

dependencies {
    implementation(kotlin("stdlib"))
}
  • Scripting Interactiu amb el REPL de Kotlin
    • Kotlin proporciona un bucle de lectura-avaluació-impressió (REPL) per al scripting interactiu.
    • Pots iniciar el REPL executant kotlinc a la línia de comandes.

Conclusió

El scripting en Kotlin és una eina versàtil que millora la productivitat permetent-te escriure i executar codi Kotlin de manera ràpida i eficient. Ja sigui que estiguis automatitzant tasques, escrivint scripts de construcció o prototipant, el scripting en Kotlin proporciona una manera fluida i potent de fer ús de les capacitats de Kotlin sense la càrrega d'una configuració completa del projecte. Entenent els conceptes bàsics i explorant les característiques avançades, pots aprofitar tot el potencial del scripting en Kotlin en el teu flux de treball de desenvolupament.
