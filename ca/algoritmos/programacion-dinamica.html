Introducció a la Programació Dinàmica

La programació dinàmica és una tècnica d'optimització utilitzada per resoldre problemes complexos dividint-los en subproblemes més simples. És especialment útil en problemes d'optimització on les solucions dels subproblemes es reutilitzen per construir la solució del problema original.

Conceptes Clau

  • Subproblemes Superposats: Els problemes poden descompondre's en subproblemes més petits que es resolen repetidament.
  • Estructura Òptima: La solució òptima del problema pot construir-se a partir de les solucions òptimes dels seus subproblemes.
  • Memoització: Tècnica d'emmagatzematge en memòria cau dels resultats dels subproblemes per evitar càlculs redundants.
  • Tabulació: Tècnica de construir una taula de baix cap a dalt per emmagatzemar els resultats dels subproblemes.

Exemples Clàssics

Fibonacci

El problema de calcular el n-èsim nombre de Fibonacci és un exemple clàssic de programació dinàmica.

Solució Recursiva (Ineficient)

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

Solució amb Memoització

def fibonacci_memo(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fibonacci_memo(n-1, memo) + fibonacci_memo(n-2, memo)
    return memo[n]

Solució amb Tabulació

def fibonacci_tab(n):
    if n <= 1:
        return n
    table = [0] * (n + 1)
    table[1] = 1
    for i in range(2, n + 1):
        table[i] = table[i-1] + table[i-2]
    return table[n]

Problema de la Motxilla (0/1 Knapsack)

El problema de la motxilla és un altre exemple clàssic on es busca maximitzar el valor total dels ítems en una motxilla amb una capacitat limitada.

Descripció del Problema

  • Entrada: Una llista de pesos i valors d'ítems, i una capacitat màxima de la motxilla.
  • Sortida: El valor màxim que es pot obtenir sense excedir la capacitat de la motxilla.

Solució amb Programació Dinàmica

def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0 for _ in range(capacity + 1)] for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        for w in range(1, capacity + 1):
            if weights[i-1] <= w:
                dp[i][w] = max(dp[i-1][w], dp[i-1][w-weights[i-1]] + values[i-1])
            else:
                dp[i][w] = dp[i-1][w]
    
    return dp[n][capacity]

Comparació de Tècniques

| Tècnica | Avantatges | Desavantatges | |---------------|---------------------------------------------|-------------------------------------------| | Recursiva | Fàcil d'implementar i entendre | Ineficient, alta complexitat temporal | | Memoització | Redueix la complexitat temporal significativament | Pot consumir molta memòria | | Tabulació | Eficient en temps i espai | Requereix una comprensió més profunda |

Exercicis Pràctics

  1. Exercici de Fibonacci: Implementa la solució de Fibonacci utilitzant memoització i tabulació. Compara el temps d'execució amb la solució recursiva.
  2. Exercici de la Motxilla: Donat un conjunt d'ítems amb pesos i valors, i una capacitat de motxilla, implementa la solució utilitzant programació dinàmica.
  3. Exercici de Camins en una Matriu: Troba el nombre de camins possibles des de la cantonada superior esquerra fins a la cantonada inferior dreta d'una matriu, només movent-te cap a la dreta o cap avall.

Conclusió

La programació dinàmica és una tècnica poderosa per resoldre problemes d'optimització que involucren subproblemes superposats i estructures òptimes. Comprendre i aplicar memoització i tabulació pot millorar significativament l'eficiència dels algorismes. Practicar amb problemes clàssics com Fibonacci i la motxilla ajudarà a consolidar aquests conceptes i tècniques.
