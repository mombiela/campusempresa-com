El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquesta secció, proporcionarem una llista de recursos addicionals i lectures recomanades que t'ajudaran a aprofundir en l'estudi d'algoritmes. Aquests recursos inclouen llibres, articles, llocs web i cursos en línia que són altament valorats en la comunitat de TI.

Llibres Recomanats

  • "Introduction to Algorithms" per Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest i Clifford Stein

    • Aquest llibre és una referència clàssica en el camp dels algoritmes. Cobreix una àmplia gamma de temes i és adequat tant per a principiants com per a professionals avançats.

  • "Algorithms" per Robert Sedgewick i Kevin Wayne

    • Aquest llibre ofereix una introducció clara i concisa als algoritmes, amb un enfocament en la implementació pràctica en Java.

  • "The Algorithm Design Manual" per Steven S. Skiena

    • Aquest llibre és conegut pel seu enfocament pràctic i els seus estudis de cas, cosa que el fa ideal per a aquells que busquen aplicar algoritmes a problemes del món real.

Articles i Publicacions

  • "A Survey of Data Structures and Algorithms" per Robert Endre Tarjan

    • Un article fonamental que proporciona una visió general de les estructures de dades i algoritmes més importants.

  • "The Art of Computer Programming" per Donald E. Knuth

    • Encara que és una sèrie de llibres, molts capítols estan disponibles com a articles independents. És una obra de referència essencial per a qualsevol professional de TI.

Llocs Web i Plataformes d'Aprenentatge

  • GeeksforGeeks

    • Un lloc web que ofereix tutorials detallats sobre una àmplia varietat d'algoritmes i estructures de dades, amb exemples de codi en múltiples llenguatges de programació.

  • HackerRank

    • Una plataforma de codificació que ofereix desafiaments i concursos per practicar algoritmes i estructures de dades.

  • Coursera

    • Cursos en línia sobre algoritmes oferts per universitats de renom, com el curs "Algorithms" de la Universitat de Princeton.

Exemples de Codi

A continuació, es presenten alguns exemples de codi per il·lustrar com implementar algoritmes bàsics en Python.

Exemple de Cerca Binària

def binary_search(arr, x):
    low = 0
    high = len(arr) - 1
    mid = 0

    while low <= high:
        mid = (high + low) // 2

        # Si x està al mig
        if arr[mid] < x:
            low = mid + 1
        elif arr[mid] > x:
            high = mid - 1
        else:
            return mid

    return -1

# Exemple d'ús
arr = [2, 3, 4, 10, 40]
x = 10
result = binary_search(arr, x)

if result != -1:
    print(f"Element trobat a l'índex {result}")
else:
    print("Element no trobat a l'array")

Exemple d'Ordenació per Inserció

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

# Exemple d'ús
arr = [12, 11, 13, 5, 6]
insertion_sort(arr)
print("Array ordenat:", arr)
