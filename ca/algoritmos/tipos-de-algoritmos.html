Introducció

En el món de la informàtica i la programació, els algoritmes són fonamentals per resoldre problemes i realitzar tasques de manera eficient. Existeixen diversos tipus d'algoritmes, cadascun amb les seves pròpies característiques i aplicacions. En aquesta secció, explorarem els tipus més comuns d'algoritmes, proporcionant exemples i explicacions clares per ajudar els professionals de TI a comprendre els seus usos i beneficis.

Algoritmes de Cerca

Els algoritmes de cerca s'utilitzen per trobar un element específic dins d'una estructura de dades. Els dos tipus més comuns són:

  • Cerca Lineal: Recorre cada element de l'estructura de dades fins a trobar l'element buscat.
  • Cerca Binària: Requereix que l'estructura de dades estigui ordenada. Divideix repetidament el rang de cerca a la meitat fins a trobar l'element.

Exemple de Cerca Lineal

def cerca_lineal(llista, objectiu):
    for i in range(len(llista)):
        if llista[i] == objectiu:
            return i
    return -1

# Ús de l'algoritme
llista = [2, 4, 6, 8, 10]
objectiu = 6
resultat = cerca_lineal(llista, objectiu)
print(f'Element trobat a l\'índex: {resultat}')

Exemple de Cerca Binària

def cerca_binaria(llista, objectiu):
    inici = 0
    fi = len(llista) - 1

    while inici <= fi:
        mig = (inici + fi) // 2
        if llista[mig] == objectiu:
            return mig
        elif llista[mig] < objectiu:
            inici = mig + 1
        else:
            fi = mig - 1
    return -1

# Ús de l'algoritme
llista = [2, 4, 6, 8, 10]
objectiu = 6
resultat = cerca_binaria(llista, objectiu)
print(f'Element trobat a l\'índex: {resultat}')

Algoritmes d'Ordenació

Els algoritmes d'ordenació organitzen els elements d'una estructura de dades en un ordre específic. Alguns dels algoritmes més coneguts són:

  • Ordenació per Bombolla: Compara i canvia elements adjacents repetidament fins que la llista estigui ordenada.
  • Ordenació per Selecció: Troba l'element més petit i el col·loca al principi, repetint el procés per a la resta de la llista.
  • Ordenació Ràpida (Quick Sort): Divideix i conquista, seleccionant un pivot i ordenant els elements menors i majors al voltant del pivot.

Exemple d'Ordenació per Bombolla

def ordenacio_bombolla(llista):
    n = len(llista)
    for i in range(n):
        for j in range(0, n-i-1):
            if llista[j] > llista[j+1]:
                llista[j], llista[j+1] = llista[j+1], llista[j]

# Ús de l'algoritme
llista = [64, 34, 25, 12, 22, 11, 90]
ordenacio_bombolla(llista)
print(f'Llista ordenada: {llista}')

Exemple d'Ordenació Ràpida

def quick_sort(llista):
    if len(llista) <= 1:
        return llista
    else:
        pivot = llista[0]
        menors = [x for x in llista[1:] if x <= pivot]
        majors = [x for x in llista[1:] si x > pivot]
        return quick_sort(menors) + [pivot] + quick_sort(majors)

# Ús de l'algoritme
llista = [64, 34, 25, 12, 22, 11, 90]
llista_ordenada = quick_sort(llista)
print(f'Llista ordenada: {llista_ordenada}')

Algoritmes de Recursió

La recursió és una tècnica en la qual una funció es crida a si mateixa per resoldre un problema. És especialment útil per a problemes que poden dividir-se en subproblemes més petits del mateix tipus.

  • Factorial d'un número: Calcula el producte de tots els enters positius fins a un número donat.
  • Fibonacci: Genera una seqüència en la qual cada número és la suma dels dos anteriors.

Exemple de Factorial

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

# Ús de l'algoritme
numero = 5
resultat = factorial(numero)
print(f'Factorial de {numero} és {resultat}')

Exemple de Fibonacci

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Ús de l'algoritme
numero = 6
resultat = fibonacci(numero)
print(f'Fibonacci de {numero} és {resultat}')
