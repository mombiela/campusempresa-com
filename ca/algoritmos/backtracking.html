Introducció

Backtracking és una tècnica d'algoritmes utilitzada per resoldre problemes de manera recursiva, construint solucions pas a pas i eliminant aquelles que no compleixen amb les restriccions del problema. És especialment útil en problemes de cerca i optimització on la solució pot ser representada com una seqüència de decisions.

Conceptes Clau

  • Recursió: Backtracking es basa en la recursió per explorar totes les possibles solucions.
  • Espai de Solucions: El conjunt de totes les possibles solucions que l'algoritme pot explorar.
  • Poda: El procés d'eliminar solucions parcials que no poden portar a una solució completa vàlida.
  • Estat: Representa una configuració parcial de la solució.

Exemples de Problemes Resolts amb Backtracking

  • Problema de les N-Reines
  • Sudoku
  • Generació de Permutacions
  • Problemes de Laberints

Algoritme General de Backtracking

  1. Definir l'espai de solucions: Identificar totes les possibles configuracions del problema.
  2. Triar una opció: Seleccionar una opció vàlida per avançar en la solució.
  3. Recursió: Cridar recursivament l'algoritme amb la nova configuració.
  4. Desfer l'elecció: Si l'opció no porta a una solució vàlida, desfer l'elecció i provar una altra.

Exemple: Problema de les N-Reines

El problema de les N-Reines consisteix a col·locar N reines en un tauler d'escacs de NxN de manera que cap reina pugui atacar una altra.

Codi d'Exemple en Python

def is_safe(board, row, col):
    # Verificar la columna
    for i in range(row):
        if board[i][col] == 1:
            return False

    # Verificar la diagonal superior esquerra
    for i, j in zip(range(row, -1, -1), range(col, -1, -1)):
        if board[i][j] == 1:
            return False

    # Verificar la diagonal superior dreta
    for i, j in zip(range(row, -1, -1), range(col, len(board))):
        if board[i][j] == 1:
            return False

    return True

def solve_n_queens(board, row):
    if row >= len(board):
        return True

    for col in range(len(board)):
        if is_safe(board, row, col):
            board[row][col] = 1
            if solve_n_queens(board, row + 1):
                return True
            board[row][col] = 0  # Backtrack

    return False

def print_board(board):
    for row in board:
        print(" ".join(str(cell) for cell in row))

n = 4
board = [[0] * n for _ in range(n)]
if solve_n_queens(board, 0):
    print_board(board)
else:
    print("No solution found")

Explicació del Codi

  1. Funció is_safe: Verifica si és segur col·locar una reina a la posició (row, col).
  2. Funció solve_n_queens: Intenta resoldre el problema de les N-Reines utilitzant backtracking.
  3. Funció print_board: Imprimeix el tauler d'escacs amb les reines col·locades.

Avantatges i Desavantatges del Backtracking

Avantatges

  • Simplicitat: Fàcil d'implementar i entendre.
  • Flexibilitat: Pot adaptar-se a una àmplia varietat de problemes.

Desavantatges

  • Eficiència: Pot ser ineficient per a problemes grans a causa de l'exploració exhaustiva de l'espai de solucions.
  • Complexitat: La complexitat temporal pot ser exponencial en el pitjor dels casos.

Conclusió

Backtracking és una tècnica poderosa per resoldre problemes de cerca i optimització. Encara que pot ser ineficient per a problemes grans, la seva simplicitat i flexibilitat el fan una eina valuosa en l'arsenal de qualsevol programador. Practicar amb problemes clàssics com el de les N-Reines o Sudoku pot ajudar a entendre millor com i quan utilitzar aquesta tècnica.
