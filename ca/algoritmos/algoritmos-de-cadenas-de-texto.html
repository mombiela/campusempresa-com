Introducció

Els algorismes de cadenes de text són fonamentals en la informàtica i tenen aplicacions en diverses àrees com la cerca d'informació, el processament de llenguatge natural i la bioinformàtica. Aquest tema cobreix els conceptes clau i els algorismes més utilitzats per manipular i analitzar cadenes de text.

Conceptes Clau

  • Cadenes de Text: Seqüències de caràcters que poden representar paraules, frases o qualsevol altre tipus de dades textuals.
  • Subcadena: Una porció d'una cadena de text.
  • Patró: Una cadena de text que es busca dins d'una altra cadena.
  • Prefix i Sufix: Subcadenes que comencen des de l'inici o acaben al final d'una cadena, respectivament.

Algorismes Comuns

Cerca de Subcadenes

Algorisme de Força Bruta

L'algorisme de força bruta compara el patró amb totes les possibles subcadenes de la cadena principal.

def força_bruta(text, patró):
    n = len(text)
    m = len(patró)
    for i in range(n - m + 1):
        j = 0
        while j < m and text[i + j] == patró[j]:
            j += 1
        if j == m:
            return i
    return -1

text = "hola món"
patró = "món"
print(força_bruta(text, patró))  # Sortida: 5

Algorisme de Knuth-Morris-Pratt (KMP)

L'algorisme KMP millora l'eficiència evitant comparacions redundants mitjançant l'ús d'una taula de prefixos.

def construir_taula_kmp(patró):
    m = len(patró)
    taula = [0] * m
    j = 0
    for i in range(1, m):
        if patró[i] == patró[j]:
            j += 1
            taula[i] = j
        else:
            if j != 0:
                j = taula[j - 1]
                i -= 1
            else:
                taula[i] = 0
    return taula

def kmp(text, patró):
    n = len(text)
    m = len(patró)
    taula = construir_taula_kmp(patró)
    i = j = 0
    while i < n:
        if patró[j] == text[i]:
            i += 1
            j += 1
        if j == m:
            return i - j
        elif i < n and patró[j] != text[i]:
            if j != 0:
                j = taula[j - 1]
            else:
                i += 1
    return -1

text = "hola món"
patró = "món"
print(kmp(text, patró))  # Sortida: 5

Comparació d'Algorismes

| Algorisme | Complexitat Temporal | Avantatges | Desavantatges | |-------------------|----------------------|---------------------------------|--------------------------------| | Força Bruta | O(n * m) | Fàcil d'implementar | Ineficient per a cadenes llargues| | Knuth-Morris-Pratt| O(n + m) | Eficient per a patrons llargs | Més complex d'implementar |

Transformacions de Cadenes

Algorisme de Levenshtein

L'algorisme de Levenshtein calcula la distància mínima d'edició entre dues cadenes, és a dir, el nombre mínim d'operacions necessàries per transformar una cadena en una altra.

def distància_levenshtein(s1, s2):
    si len(s1) < len(s2):
        return distància_levenshtein(s2, s1)

    si len(s2) == 0:
        return len(s1)

    anterior_fila = range(len(s2) + 1)
    per i, c1 in enumerate(s1):
        actual_fila = [i + 1]
        per j, c2 in enumerate(s2):
            inserir = anterior_fila[j + 1] + 1
            eliminar = actual_fila[j] + 1
            substituir = anterior_fila[j] + (c1 != c2)
            actual_fila.append(min(inserir, eliminar, substituir))
        anterior_fila = actual_fila

    return anterior_fila[-1]

s1 = "kitten"
s2 = "sitting"
print(distància_levenshtein(s1, s2))  # Sortida: 3

Conclusió

Els algorismes de cadenes de text són eines poderoses per a la manipulació i anàlisi de dades textuals. Des de la cerca de subcadenes fins a la transformació de cadenes, aquests algorismes tenen aplicacions pràctiques en múltiples camps. Comprendre i saber implementar aquests algorismes és essencial per a qualsevol professional de TI que treballi amb dades textuals.
