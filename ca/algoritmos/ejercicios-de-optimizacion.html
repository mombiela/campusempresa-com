Introducció

L'optimització d'algoritmes és una habilitat crucial per als professionals de TI, ja que permet millorar l'eficiència i el rendiment de les aplicacions. En aquesta secció, explorarem diferents tècniques i exercicis pràctics per optimitzar algoritmes, amb l'objectiu de reduir el temps d'execució i l'ús de recursos.

Conceptes Clau

  • Complexitat Temporal i Espacial: Mesura del temps i espai que un algoritme requereix.
  • Algoritmes de Cerca i Ordenació: Tècniques per trobar i organitzar dades de manera eficient.
  • Programació Dinàmica: Mètode per resoldre problemes complexos dividint-los en subproblemes més simples.
  • Algoritmes Greedy: Estratègia que pren la millor decisió local en cada pas amb l'esperança de trobar una solució global òptima.
  • Optimització de Codi: Tècniques per millorar el rendiment del codi a nivell d'implementació.

Exercicis Pràctics

Exercici 1: Optimització de Cerca Lineal

Descripció: Donat un array de números enters, optimitza una funció que busca un número específic en l'array.

Codi Inicial:

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

Optimització:

  • Utilitza un bucle while en lloc de for per reduir el nombre de comparacions.
  • Implementa una cerca binària si l'array està ordenat.

Codi Optimitzat:

def linear_search_optimized(arr, target):
    i = 0
    while i < len(arr):
        if arr[i] == target:
            return i
        i += 1
    return -1

Exercici 2: Optimització d'Ordenació per Bombolla

Descripció: Millora l'algoritme d'ordenació per bombolla per reduir el nombre de comparacions innecessàries.

Codi Inicial:

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

Optimització:

  • Introdueix una bandera per detectar si l'array ja està ordenat i evitar iteracions addicionals.

Codi Optimitzat:

def bubble_sort_optimized(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

Exercici 3: Programació Dinàmica per a la Motxilla

Descripció: Utilitza programació dinàmica per resoldre el problema de la motxilla, maximitzant el valor total sense excedir la capacitat.

Codi Inicial:

def knapsack(weights, values, capacity):
    n = len(values)
    dp = [[0 for x in range(capacity + 1)] for x in range(n + 1)]
    
    for i in range(n + 1):
        for w in range(capacity + 1):
            if i == 0 o w == 0:
                dp[i][w] = 0
            elif weights[i-1] <= w:
                dp[i][w] = max(values[i-1] + dp[i-1][w-weights[i-1]], dp[i-1][w])
            else:
                dp[i][w] = dp[i-1][w]
    
    return dp[n][capacity]

Optimització:

  • Redueix l'espai utilitzat optimitzant la matriu dp a una sola dimensió.

Codi Optimitzat:

def knapsack_optimized(weights, values, capacity):
    n = len(values)
    dp = [0] * (capacity + 1)
    
    per i in range(n):
        per w in range(capacity, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[capacity]
