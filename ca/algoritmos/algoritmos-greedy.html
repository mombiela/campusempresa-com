Introducció als Algorismes Greedy

Els algorismes Greedy (o voraces) són una classe d'algorismes que construeixen solucions a problemes mitjançant una sèrie d'eleccions locals òptimes. La idea principal és prendre la millor decisió possible en cada pas amb l'esperança de trobar una solució global òptima.

Característiques dels Algorismes Greedy

  • Elecció Greedy: En cada pas, se selecciona l'opció que sembla ser la millor en aquell moment.
  • Solució Parcial: La solució es construeix pas a pas, afegint elements a una solució parcial.
  • Irrevocabilitat: Una vegada que es pren una decisió, no es revisa ni es canvia.

Exemples de Problemes Resolts amb Algorismes Greedy

Problema del Canvi de Monedes

L'objectiu és donar canvi utilitzant la menor quantitat de monedes possible.

Exemple de Codi en Python

def canvi_monedes(quantitat, monedes):
    resultat = []
    for moneda in sorted(monedes, reverse=True):
        while quantitat >= moneda:
            quantitat -= moneda
            resultat.append(moneda)
    return resultat

# Exemple d'ús
quantitat = 63
monedes = [1, 5, 10, 25]
print(canvi_monedes(quantitat, monedes))  # Sortida: [25, 25, 10, 1, 1, 1]

Problema de la Motxilla Fraccionària

L'objectiu és maximitzar el valor total dels objectes a la motxilla sense excedir la seva capacitat.

Exemple de Codi en Python

class Objecte:
    def __init__(self, pes, valor):
        self.pes = pes
        self.valor = valor
        self.valor_per_pes = valor / pes

def motxilla_fraccionaria(capacitat, objectes):
    objectes.sort(key=lambda x: x.valor_per_pes, reverse=True)
    valor_total = 0
    for obj in objectes:
        si capacitat >= obj.pes:
            capacitat -= obj.pes
            valor_total += obj.valor
        else:
            valor_total += obj.valor_per_pes * capacitat
            break
    return valor_total

# Exemple d'ús
objectes = [Objecte(10, 60), Objecte(20, 100), Objecte(30, 120)]
capacitat = 50
print(motxilla_fraccionaria(capacitat, objectes))  # Sortida: 240.0

Avantatges i Desavantatges dels Algorismes Greedy

Avantatges

  • Simplicitat: Són fàcils d'implementar i entendre.
  • Eficiència: Sovint són ràpids i tenen una complexitat computacional baixa.

Desavantatges

  • No sempre òptims: No garanteixen trobar la solució òptima per a tots els problemes.
  • Irrevocabilitat: Les decisions no es poden canviar una vegada preses, cosa que pot portar a solucions subòptimes.

Comparació amb Altres Enfocaments

| Característica | Algorismes Greedy | Programació Dinàmica | Força Bruta | |----------------------|-------------------|-----------------------|--------------| | Complexitat | Baixa | Mitjana | Alta | | Garantia d'Òptim | No sempre | Sí | Sí | | Facilitat d'Implementació | Alta | Mitjana | Baixa |

Conclusió

Els algorismes Greedy són una eina poderosa i eficient per resoldre certs tipus de problemes. Encara que no sempre garanteixen la solució òptima, la seva simplicitat i rapidesa els fan útils en moltes situacions pràctiques. És important entendre quan i com aplicar-los adequadament per aprofitar els seus avantatges i mitigar els seus desavantatges.
