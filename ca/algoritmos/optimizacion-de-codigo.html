Optimització de Codi

L'optimització de codi és una pràctica essencial en el desenvolupament de programari que busca millorar l'eficiència i el rendiment d'un programa. Aquest tema és crucial per a qualsevol professional de TI, ja que un codi optimitzat pot resultar en aplicacions més ràpides, eficients i escalables.

Conceptes Clau

  • Eficiència de l'Algorisme: Mesura dels recursos (temps i espai) que un algorisme consumeix.
  • Complexitat Temporal: Temps que triga un algorisme a executar-se, generalment expressat en notació Big O.
  • Complexitat Espacial: Quantitat de memòria que un algorisme necessita durant la seva execució.
  • Optimització Prematura: Intentar optimitzar el codi abans de tenir una versió funcional i correcta, la qual cosa pot ser contraproduent.
  • Perfilat de Codi: Tècnica per mesurar el rendiment de diferents parts del codi i detectar colls d'ampolla.

Estratègies d'Optimització

1. Selecció d'Algorismes Adequats

Escollir l'algorisme correcte pot tenir un impacte significatiu en el rendiment d'una aplicació.

  • Exemple: Per buscar un element en una llista, utilitzar una cerca binària (O(log n)) en lloc d'una cerca lineal (O(n)) si la llista està ordenada.
# Cerca Lineal
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Cerca Binària
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

2. Reducció de la Complexitat

Simplificar el codi per reduir la seva complexitat temporal i espacial.

  • Exemple: Evitar bucles niats innecessaris.
# Codi no optimitzat
def find_duplicates(arr):
    duplicates = []
    for i in range(len(arr)):
        for j in range(i + 1, len(arr)):
            if arr[i] == arr[j]:
                duplicates.append(arr[i])
    return duplicates

# Codi optimitzat
def find_duplicates(arr):
    seen = set()
    duplicates = set()
    for num in arr:
        if num in seen:
            duplicates.add(num)
        else:
            seen.add(num)
    return list(duplicates)

3. Ús Eficient d'Estructures de Dades

Escollir l'estructura de dades adequada pot millorar significativament el rendiment.

  • Exemple: Utilitzar un diccionari per comptar la freqüència d'elements en lloc d'una llista.
# Ús de llista
def count_frequencies(arr):
    freq = [0] * 100  # Assumint que els elements estan en el rang 0-99
    for num in arr:
        freq[num] += 1
    return freq

# Ús de diccionari
def count_frequencies(arr):
    freq = {}
    for num in arr:
        if num in freq:
            freq[num] += 1
        else:
            freq[num] = 1
    return freq

4. Eliminació de Codi Redundant

Eliminar codi innecessari o redundant per millorar la llegibilitat i el rendiment.

  • Exemple: Evitar càlculs repetits dins de bucles.
# Codi no optimitzat
def calculate_squares(arr):
    squares = []
    for num in arr:
        squares.append(num * num)
    return squares

# Codi optimitzat
def calculate_squares(arr):
    return [num * num for num in arr]

Eines de Perfilat

Utilitzar eines de perfilat per identificar i solucionar colls d'ampolla en el codi.

  • cProfile: Eina de perfilat per a Python.
  • Valgrind: Eina d'anàlisi de rendiment per a C/C++.
  • JProfiler: Eina de perfilat per a aplicacions Java.

Exemple Complet

A continuació, es mostra un exemple complet de com optimitzar un codi que busca duplicats en una llista i compta la seva freqüència.

# Codi no optimitzat
def find_and_count_duplicates(arr):
    duplicates = []
    for i in range(len(arr)):
        for j in range(i + 1, len(arr)):
            if arr[i] == arr[j]:
                duplicates.append(arr[i])
    
    freq = {}
    for num in duplicates:
        if num in freq:
            freq[num] += 1
        else:
            freq[num] = 1
    return freq

# Codi optimitzat
def find_and_count_duplicates(arr):
    seen = set()
    duplicates = set()
    for num in arr:
        if num in seen:
            duplicates.add(num)
        else:
            seen.add(num)
    
    freq = {}
    for num in duplicates:
        freq[num] = arr.count(num)
    return freq

Conclusió

L'optimització de codi és una habilitat crucial per a qualsevol professional de TI. En seleccionar algorismes adequats, reduir la complexitat, utilitzar eficientment les estructures de dades i eliminar codi redundant, es pot millorar significativament el rendiment d'una aplicació. A més, l'ús d'eines de perfilat permet identificar i solucionar colls d'ampolla de manera efectiva. Practicar aquestes tècniques ajudarà a desenvolupar aplicacions més ràpides, eficients i escalables.
