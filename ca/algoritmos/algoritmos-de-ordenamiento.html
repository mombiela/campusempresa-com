Introducció

Els algorismes d'ordenació són fonamentals en la informàtica i s'utilitzen per organitzar dades en un ordre específic, ja sigui ascendent o descendent. Aquests algorismes són essencials per millorar l'eficiència d'altres operacions com cerques i fusions de dades.

Tipus d'Algorismes d'Ordenació

Hi ha diversos tipus d'algorismes d'ordenació, cadascun amb les seves pròpies característiques i casos d'ús. A continuació, es descriuen alguns dels més comuns:

Ordenació per Bombolla (Bubble Sort)

  • Descripció: És un dels algorismes més simples. Compara parells adjacents i els intercanvia si estan en l'ordre incorrecte.
  • Complexitat: O(n^2) en el pitjor i promig dels casos.
  • Avantatges: Fàcil d'entendre i implementar.
  • Desavantatges: Ineficient per a grans conjunts de dades.
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# Exemple d'ús
arr = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(arr))

Ordenació per Selecció (Selection Sort)

  • Descripció: Troba l'element més petit i el col·loca en la primera posició, després repeteix el procés per a la resta de l'array.
  • Complexitat: O(n^2) en el pitjor i promig dels casos.
  • Avantatges: Menor nombre d'intercanvis comparat amb l'ordenació per bombolla.
  • Desavantatges: Encara ineficient per a grans conjunts de dades.
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

# Exemple d'ús
arr = [64, 25, 12, 22, 11]
print(selection_sort(arr))

Ordenació per Inserció (Insertion Sort)

  • Descripció: Construeix l'array ordenat d'un en un, prenent un element i col·locant-lo en la seva posició correcta.
  • Complexitat: O(n^2) en el pitjor i promig dels casos.
  • Avantatges: Eficient per a conjunts de dades petits o gairebé ordenats.
  • Desavantatges: Ineficient per a grans conjunts de dades.
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i-1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

# Exemple d'ús
arr = [12, 11, 13, 5, 6]
print(insertion_sort(arr))

Ordenació Ràpida (Quick Sort)

  • Descripció: Utilitza l'enfocament de "divideix i venceràs". Selecciona un 'pivot' i particiona l'array en dos sub-arrays, després ordena els sub-arrays recursivament.
  • Complexitat: O(n log n) en el millor i promig dels casos, O(n^2) en el pitjor cas.
  • Avantatges: Molt eficient en la pràctica per a grans conjunts de dades.
  • Desavantatges: Pot ser ineficient en el pitjor cas si no es tria bé el pivot.
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[len(arr) // 2]
        left = [x for x in arr if x < pivot]
        middle = [x for x in arr if x == pivot]
        right = [x for x in arr if x > pivot]
        return quick_sort(left) + middle + quick_sort(right)

# Exemple d'ús
arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))

Ordenació per Fusió (Merge Sort)

  • Descripció: També utilitza l'enfocament de "divideix i venceràs". Divideix l'array en dues meitats, les ordena recursivament i després les fusiona.
  • Complexitat: O(n log n) en tots els casos.
  • Avantatges: Molt eficient i estable.
  • Desavantatges: Requereix espai addicional per a la fusió.
def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        L = arr[:mid]
        R = arr[mid:]

        merge_sort(L)
        merge_sort(R)

        i = j = k = 0

        while i < len(L) and j < len(R):
            if L[i] < R[j]:
                arr[k] = L[i]
                i += 1
            else:
                arr[k] = R[j]
                j += 1
            k += 1

        while i < len(L):
            arr[k] = L[i]
            i += 1
            k += 1

        while j < len(R):
            arr[k] = R[j]
            j += 1
            k += 1
    return arr

# Exemple d'ús
arr = [12, 11, 13, 5, 6, 7]
print(merge_sort(arr))

Comparació d'Algorismes d'Ordenació

Conclusió

Els algorismes d'ordenació són eines essencials en la programació i la informàtica. Cada algorisme té els seus propis avantatges i desavantatges, i l'elecció de l'algorisme adequat depèn del context i les característiques del conjunt de dades a ordenar. És important comprendre les diferències en complexitat i eficiència per seleccionar l'algorisme més adequat per a cada situació.
