Què és Docker?

Docker és una plataforma de contenidorització que permet als desenvolupadors empaquetar aplicacions i les seves dependències en un contenidor, assegurant que l'aplicació s'executi de manera uniforme en qualsevol entorn.

Conceptes Clau

  • Contenidor: Un contenidor és una unitat estàndard de programari que empaqueta el codi i totes les seves dependències perquè l'aplicació s'executi de manera ràpida i fiable d'un entorn informàtic a un altre.
  • Imatge: Una imatge de Docker és una plantilla de només lectura que conté un conjunt d'instruccions per crear un contenidor que pot executar-se en la plataforma Docker.
  • Dockerfile: Un fitxer de text que conté totes les instruccions (comandes) que un usuari podria cridar a la línia de comandes per assemblar una imatge de Docker.
  • Docker Hub: Un servei de registre d'imatges de Docker que permet emmagatzemar i compartir imatges de contenidors.

Instal·lació de Docker

Per començar a utilitzar Docker, primer necessites instal·lar-lo al teu sistema. Aquí et mostrem com fer-ho en diferents sistemes operatius:

Instal·lació a Windows

  1. Descarrega Docker Desktop des de Docker Hub.
  2. Executa l'instal·lador i segueix les instruccions en pantalla.
  3. Un cop instal·lat, obre Docker Desktop i assegura't que Docker estigui corrent.

Instal·lació a macOS

  1. Descarrega Docker Desktop des de Docker Hub.
  2. Obre el fitxer .dmg i arrossega Docker a la carpeta d'Aplicacions.
  3. Inicia Docker des de la carpeta d'Aplicacions.

Instal·lació a Linux

Per a Ubuntu:

sudo apt-get update
sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    software-properties-common

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

sudo add-apt-repository \
   "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
   $(lsb_release -cs) \
   stable"

sudo apt-get update
sudo apt-get install docker-ce

Primers Passos amb Docker

Executar el teu Primer Contenidor

Un cop Docker estigui instal·lat, pots executar el teu primer contenidor. Farem servir una imatge de prova anomenada hello-world.

docker run hello-world

Aquesta comanda descarrega una imatge de prova de Docker i executa un contenidor que imprimeix "Hello from Docker!".

Crear una Imatge de Docker

Per crear la teva pròpia imatge de Docker, necessites un Dockerfile. Aquí tens un exemple simple:

# Usa una imatge base de Python
FROM python:3.8-slim

# Estableix el directori de treball
WORKDIR /app

# Copia el fitxer de requeriments i l'aplicació
COPY requirements.txt requirements.txt
COPY app.py app.py

# Instal·la les dependències
RUN pip install -r requirements.txt

# Defineix la comanda per defecte
CMD ["python", "app.py"]

Per construir aquesta imatge, usa la següent comanda:

docker build -t my-python-app .

Executar un Contenidor des de la teva Imatge

Un cop hagis creat la teva imatge, pots executar un contenidor basat en ella:

docker run my-python-app

Comandes Bàsiques de Docker

Aquí hi ha una taula amb algunes comandes bàsiques de Docker i la seva descripció:

| Comanda | Descripció | |--------------------------|-------------------------------------------------| | docker run | Executa un contenidor | | docker ps | Llista els contenidors en execució | | docker images | Llista les imatges disponibles | | docker build | Construeix una imatge des d'un Dockerfile | | docker pull | Descarrega una imatge des d'un repositori | | docker push | Puja una imatge a un repositori | | docker stop | Atura un contenidor en execució | | docker rm | Elimina un contenidor | | docker rmi | Elimina una imatge |

Conclusió

Docker és una eina poderosa que facilita la creació, desplegament i execució d'aplicacions en contenidors. En aprendre els conceptes bàsics i comandes essencials, pots començar a aprofitar els avantatges de la contenidorització en els teus projectes de desenvolupament i operacions. En les següents lliçons, aprofundirem en temes més avançats com l'orquestració de contenidors i la integració contínua amb Docker.
