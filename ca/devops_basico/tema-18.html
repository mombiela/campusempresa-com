Introducció als contenidors

Els contenidors són una tecnologia que permet empaquetar aplicacions i les seves dependències en un entorn aïllat i reproduïble. Això facilita la portabilitat i la consistència entre diferents entorns de desenvolupament, prova i producció.

  • Aïllament: Cada contenidor opera de manera independent, cosa que redueix els conflictes entre aplicacions.
  • Portabilitat: Els contenidors poden executar-se en qualsevol sistema que suporti la tecnologia de contenidors, com Docker.
  • Eficiència: Els contenidors comparteixen el mateix sistema operatiu, cosa que els fa més lleugers que les màquines virtuals.

Docker: La plataforma de contenidors més popular

Docker és una plataforma que permet als desenvolupadors crear, desplegar i executar aplicacions en contenidors.

  • Docker Engine: El motor que permet la creació i execució de contenidors.
  • Docker Hub: Un repositori en línia per compartir i emmagatzemar imatges de contenidors.

Instal·lació de Docker

Per començar a treballar amb Docker, primer necessites instal·lar-lo al teu sistema.

Instal·lació a Linux

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

Instal·lació a Windows i macOS

Visita Docker Desktop i segueix les instruccions per al teu sistema operatiu.

Creació de contenidors

Per crear un contenidor, primer necessites una imatge de Docker. Les imatges són plantilles de només lectura que contenen el sistema operatiu i les aplicacions necessàries.

Exemple: Crear un contenidor de Nginx

  1. Descarregar la imatge de Nginx: 
    docker pull nginx
  2. Executar un contenidor de Nginx: 
    docker run --name mynginx -d -p 8080:80 nginx
    • --name mynginx: Assigna un nom al contenidor.
    • -d: Executa el contenidor en segon pla.
    • -p 8080:80: Mapa el port 80 del contenidor al port 8080 de l'amfitrió.

Verificar el contenidor en execució

docker ps

Gestió de contenidors

Un cop tens contenidors en execució, necessites saber com gestionar-los.

Llistar contenidors

  • Contenidors en execució: 
    docker ps
  • Tots els contenidors (inclosos els aturats): 
    docker ps -a

Aturar i eliminar contenidors

  • Aturar un contenidor: 
    docker stop mynginx
  • Eliminar un contenidor: 
    docker rm mynginx

Inspeccionar contenidors

Per obtenir detalls sobre un contenidor específic:

docker inspect mynginx

Volums i xarxes a Docker

Els volums i les xarxes són components essencials per a la gestió avançada de contenidors.

Volums

Els volums permeten persistir dades més enllà del cicle de vida d'un contenidor.

  • Crear un volum: 
    docker volume create myvolume
  • Muntar un volum en un contenidor: 
    docker run -d -v myvolume:/data --name mynginx nginx

Xarxes

Les xarxes permeten la comunicació entre contenidors.

  • Crear una xarxa: 
    docker network create mynetwork
  • Connectar un contenidor a una xarxa: 
    docker run -d --network mynetwork --name mynginx nginx

Conclusió

La creació i gestió de contenidors és una habilitat fonamental en DevOps. Docker facilita aquest procés en proporcionar eines robustes per al desenvolupament, desplegament i gestió d'aplicacions en contenidors. Amb el coneixement bàsic de Docker, pots començar a construir entorns de desenvolupament més eficients i portables.

Recorda practicar aquests comandaments i conceptes per familiaritzar-te amb el flux de treball de Docker. La pràctica constant t'ajudarà a dominar la gestió de contenidors!
