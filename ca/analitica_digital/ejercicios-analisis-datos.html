Introducció

En aquesta secció del curs d'Analítica Digital, ens enfocarem en exercicis pràctics d'anàlisi de dades. Aquests exercicis estan dissenyats per ajudar-te a aplicar els conceptes teòrics apresos en situacions del món real. Treballarem amb diferents tipus de dades i utilitzarem diverses eines i tècniques per analitzar-les.

Objectius

  • Aplicar tècniques d'anàlisi de dades en situacions pràctiques.
  • Utilitzar eines de programari per a l'anàlisi de dades.
  • Interpretar i comunicar els resultats de l'anàlisi de dades.

Eines Necessàries

  • Python: Llenguatge de programació utilitzat per a la manipulació i anàlisi de dades.
  • Pandas: Biblioteca de Python per a la manipulació de dades.
  • Matplotlib/Seaborn: Biblioteques de Python per a la visualització de dades.
  • Jupyter Notebook: Entorn interactiu per a la programació en Python.

Exercici 1: Anàlisi Exploratori de Dades (EDA)

Descripció

Realitzarem una anàlisi exploratòria d'un conjunt de dades per entendre la seva estructura, identificar patrons i detectar anomalies.

Passos

  1. Carregar el conjunt de dades:

    • Utilitzarem un conjunt de dades d'exemple, com el conjunt de dades d'iris.

  2. Inspeccionar les dades:

    • Verificar les primeres files del conjunt de dades.
    • Obtenir un resum estadístic de les variables.

  3. Visualitzar les dades:

    • Crear gràfics de dispersió per visualitzar les relacions entre les variables.
    • Crear histogrames per entendre la distribució de les variables.

Codi d'Exemple

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Carregar el conjunt de dades
df = sns.load_dataset('iris')

# Inspeccionar les primeres files
print(df.head())

# Resum estadístic
print(df.describe())

# Gràfic de dispersió
sns.pairplot(df, hue='species')
plt.show()

# Histogrames
df.hist(bins=20, figsize=(10, 8))
plt.show()

Explicació

  • Carregar el conjunt de dades: Utilitzem la funció load_dataset de Seaborn per carregar el conjunt de dades d'iris.
  • Inspeccionar les primeres files: Utilitzem head() per veure les primeres files del DataFrame.
  • Resum estadístic: Utilitzem describe() per obtenir un resum estadístic de les variables.
  • Gràfic de dispersió: Utilitzem pairplot de Seaborn per crear gràfics de dispersió entre totes les variables.
  • Histogrames: Utilitzem hist de Pandas per crear histogrames de les variables.

Exercici 2: Neteja de Dades

Descripció

Netejarem un conjunt de dades eliminant valors nuls i duplicats, i transformant dades categòriques.

Passos

  1. Carregar el conjunt de dades:

    • Utilitzarem un conjunt de dades amb valors nuls i duplicats.

  2. Eliminar valors nuls:

    • Identificar i eliminar files amb valors nuls.

  3. Eliminar duplicats:

    • Identificar i eliminar files duplicades.

  4. Transformar dades categòriques:

    • Convertir variables categòriques en variables numèriques.

Codi d'Exemple

# Carregar el conjunt de dades
df = pd.read_csv('data.csv')

# Eliminar valors nuls
df = df.dropna()

# Eliminar duplicats
df = df.drop_duplicates()

# Transformar dades categòriques
df['category'] = df['category'].astype('category').cat.codes

print(df.head())

Explicació

  • Carregar el conjunt de dades: Utilitzem read_csv de Pandas per carregar el conjunt de dades.
  • Eliminar valors nuls: Utilitzem dropna() per eliminar files amb valors nuls.
  • Eliminar duplicats: Utilitzem drop_duplicates() per eliminar files duplicades.
  • Transformar dades categòriques: Convertim la columna categòrica category en codis numèrics utilitzant astype('category').cat.codes.

Exercici 3: Anàlisi de Correlació

Descripció

Analitzarem la correlació entre variables en un conjunt de dades per identificar relacions significatives.

Passos

  1. Carregar el conjunt de dades:

    • Utilitzarem un conjunt de dades d'exemple.

  2. Calcular la matriu de correlació:

    • Utilitzar la funció corr() de Pandas.

  3. Visualitzar la matriu de correlació:

    • Crear un mapa de calor per visualitzar les correlacions.

Codi d'Exemple

# Carregar el conjunt de dades
df = pd.read_csv('data.csv')

# Calcular la matriu de correlació
correlation_matrix = df.corr()

# Visualitzar la matriu de correlació
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm')
plt.show()

Explicació

  • Carregar el conjunt de dades: Utilitzem read_csv de Pandas per carregar el conjunt de dades.
  • Calcular la matriu de correlació: Utilitzem corr() per calcular la matriu de correlació.
  • Visualitzar la matriu de correlació: Utilitzem heatmap de Seaborn per crear un mapa de calor de la matriu de correlació.
