Introducció

Excel és una eina poderosa per a l'anàlisi i la gestió de dades, i dominar les dreceres de teclat pot millorar significativament la teva productivitat. Aquesta secció et guiarà a través de dreceres i consells essencials, començant des de nivells bàsics fins a avançats.

Dreceres per a principiants

Navegació bàsica

  • Tecles de fletxa: Mou-te una cel·la amunt, avall, a l'esquerra o a la dreta.
  • Ctrl + Tecles de fletxa: Salta fins a la vora de les regions de dades.
  • Tab: Mou-te una cel·la a la dreta.
  • Shift + Tab: Mou-te una cel·la a l'esquerra.

Edició bàsica

  • Enter: Completa una entrada de cel·la i mou-te una cel·la avall.
  • Esc: Cancel·la una entrada de cel·la.
  • Ctrl + C: Copia les cel·les seleccionades.
  • Ctrl + V: Enganxa les cel·les copiades.
  • Ctrl + X: Retalla les cel·les seleccionades.
  • Ctrl + Z: Desfés l'última acció.
  • Ctrl + Y: Refés l'última acció desfeta.

Format bàsic

  • Ctrl + B: Posa en negreta el text seleccionat.
  • Ctrl + I: Posa en cursiva el text seleccionat.
  • Ctrl + U: Subratlla el text seleccionat.

Dreceres intermèdies

Navegació avançada

  • Ctrl + Inici: Mou-te al començament del full de càlcul.
  • Ctrl + Final: Mou-te a l'última cel·la amb dades.
  • Pàgina amunt/Pàgina avall: Mou-te una pantalla amunt/avall.
  • Alt + Pàgina amunt/Pàgina avall: Mou-te una pantalla a l'esquerra/dreta.

Edició avançada

  • Ctrl + D: Omple cap avall.
  • Ctrl + R: Omple cap a la dreta.
  • Ctrl + ;: Insereix la data actual.
  • Ctrl + Shift + :: Insereix l'hora actual.

Format avançat

  • Ctrl + 1: Obre el quadre de diàleg Format de cel·les.
  • Ctrl + Shift + $: Aplica el format de moneda.
  • Ctrl + Shift + %: Aplica el format de percentatge.
  • Ctrl + Shift + ^: Aplica el format exponencial.

Dreceres avançades

Navegació experta

  • Ctrl + Shift + Tecles de fletxa: Selecciona un rang de cel·les.
  • Ctrl + Espai: Selecciona tota la columna.
  • Shift + Espai: Selecciona tota la fila.
  • Ctrl + Shift + Espai: Selecciona tot el full de càlcul.

Edició experta

  • Alt + Enter: Comença una nova línia a la mateixa cel·la.
  • Ctrl + Shift + L: Activa o desactiva els filtres.
  • Ctrl + T: Crea una taula.
  • Ctrl + Alt + V: Obre el quadre de diàleg Enganxar especial.

Format expert

  • Ctrl + Shift + F: Obre el quadre de diàleg Format de cel·les amb la pestanya Font seleccionada.
  • Ctrl + Shift + P: Obre el quadre de diàleg Format de cel·les amb la pestanya Font seleccionada.
  • Ctrl + Shift + &: Aplica el contorn de la cel·la.
  • Ctrl + Shift + _: Elimina el contorn de la cel·la.

Consells de productivitat

Personalització de la cinta i la barra d'eines d'accés ràpid

  • Afegeix comandes d'ús freqüent: Personalitza la barra d'eines d'accés ràpid per incloure comandes que utilitzes sovint.
  • Crea pestanyes personalitzades: Organitza les teves comandes més utilitzades en pestanyes personalitzades a la cinta.

Ús de rangs amb nom

  • Defineix noms per a rangs: Utilitza el gestor de noms (Ctrl + F3) per crear i gestionar rangs amb nom per a una referència més fàcil en les fórmules.

Aprofitament dels complements d'Excel

  • Instal·la complements útils: Explora i instal·la complements des de la botiga d'Office per ampliar la funcionalitat d'Excel.

Automatització de tasques amb macros

  • Grava macros: Utilitza el gravador de macros per automatitzar tasques repetitives.
  • Assigna macros a botons: Crea botons al teu full de càlcul o a la barra d'eines d'accés ràpid per executar macros amb un sol clic.

Conclusió

Dominar les dreceres de teclat i els consells de productivitat a Excel pot estalviar-te molt de temps i esforç. Començant amb dreceres bàsiques i incorporant gradualment tècniques més avançades, pots treballar de manera més eficient i efectiva. Practica aquestes dreceres regularment per convertir-te en un expert i explora consells i trucs addicionals per millorar encara més les teves habilitats en Excel.
