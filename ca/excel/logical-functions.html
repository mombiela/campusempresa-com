Introducció

Les funcions lògiques a Excel són eines essencials per realitzar operacions condicionals. Et permeten crear fórmules que poden prendre decisions basades en criteris específics. Les funcions lògiques més utilitzades són IF, AND i OR. Aquesta secció et guiarà a través d'aquestes funcions des d'un nivell principiant fins a un nivell avançat.

Funció IF

La funció IF s'utilitza per realitzar una prova lògica i retornar un valor si la prova és certa i un altre valor si la prova és falsa.

Sintaxi

IF(logical_test, value_if_true, value_if_false)

Exemple

Suposem que tens una llista de puntuacions d'estudiants i vols determinar si cada estudiant ha aprovat o suspès. La puntuació mínima per aprovar és 50.

| Estudiant | Puntuació | Resultat | |-----------|------------|----------| | John | 45 | | | Alice | 78 | | | Bob | 52 | |

Pots utilitzar la següent fórmula a la columna "Resultat":

=IF(B2 >= 50, "Aprovat", "Suspès")

Explicació

  • logical_test: B2 >= 50 (comprova si la puntuació és major o igual a 50)
  • value_if_true: "Aprovat" (retornat si la prova és certa)
  • value_if_false: "Suspès" (retornat si la prova és falsa)

Funció AND

La funció AND s'utilitza per comprovar múltiples condicions al mateix temps. Retorna TRUE si totes les condicions són certes i FALSE si alguna condició és falsa.

Sintaxi

AND(logical1, [logical2], ...)

Exemple

Suposem que vols determinar si un estudiant ha aprovat tant l'examen de Matemàtiques com el d'Anglès. La puntuació mínima per aprovar ambdues assignatures és 50.

| Estudiant | Puntuació Matemàtiques | Puntuació Anglès | Resultat | |-----------|------------------------|------------------|----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Pots utilitzar la següent fórmula a la columna "Resultat":

=IF(AND(B2 >= 50, C2 >= 50), "Aprovat", "Suspès")

Explicació

  • logical1: B2 >= 50 (comprova si la puntuació de Matemàtiques és major o igual a 50)
  • logical2: C2 >= 50 (comprova si la puntuació d'Anglès és major o igual a 50)

Funció OR

La funció OR s'utilitza per comprovar múltiples condicions al mateix temps. Retorna TRUE si alguna de les condicions és certa i FALSE si totes les condicions són falses.

Sintaxi

OR(logical1, [logical2], ...)

Exemple

Suposem que vols determinar si un estudiant ha aprovat l'examen de Matemàtiques o el d'Anglès. La puntuació mínima per aprovar ambdues assignatures és 50.

| Estudiant | Puntuació Matemàtiques | Puntuació Anglès | Resultat | |-----------|------------------------|------------------|----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Pots utilitzar la següent fórmula a la columna "Resultat":

=IF(OR(B2 >= 50, C2 >= 50), "Aprovat", "Suspès")

Explicació

  • logical1: B2 >= 50 (comprova si la puntuació de Matemàtiques és major o igual a 50)
  • logical2: C2 >= 50 (comprova si la puntuació d'Anglès és major o igual a 50)

Combinació d'IF amb AND/OR

Pots combinar la funció IF amb AND i OR per crear proves lògiques més complexes.

Exemple

Suposem que vols determinar si un estudiant ha aprovat tant l'examen de Matemàtiques com el d'Anglès, però si ha aprovat almenys un, se li hauria de donar l'opció de "Recuperar".

| Estudiant | Puntuació Matemàtiques | Puntuació Anglès | Resultat | |-----------|------------------------|------------------|----------| | John | 45 | 55 | | | Alice | 78 | 80 | | | Bob | 52 | 49 | |

Pots utilitzar la següent fórmula a la columna "Resultat":

=IF(AND(B2 >= 50, C2 >= 50), "Aprovat", IF(OR(B2 >= 50, C2 >= 50), "Recuperar", "Suspès"))

Explicació

  • El primer IF comprova si ambdues puntuacions són >= 50.
  • Si és cert, retorna "Aprovat".
  • Si és fals, comprova si alguna de les puntuacions és >= 50.
  • Si és cert, retorna "Recuperar".
  • Si és fals, retorna "Suspès".

Conclusió

Les funcions lògiques com IF, AND i OR són eines poderoses a Excel que et permeten realitzar operacions condicionals complexes. Entenent i combinant aquestes funcions, pots crear fórmules dinàmiques i flexibles per satisfer una àmplia gamma de necessitats. Practica utilitzant aquestes funcions amb diferents conjunts de dades per convertir-te en un expert en la seva aplicació.
