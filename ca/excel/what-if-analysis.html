Introducció a l'anàlisi de supòsits

L'anàlisi de supòsits a Excel et permet experimentar amb dades per veure com els canvis afecten els resultats. Això és particularment útil per a la presa de decisions i la previsió. Les tres eines principals per a l'anàlisi de supòsits a Excel són:

  • Cerca d'objectius
  • Taules de dades
  • Gestor d'escenaris

Cerca d'objectius

La Cerca d'objectius és una eina senzilla que et permet trobar el valor d'entrada necessari per assolir un objectiu específic. Funciona ajustant una variable per arribar a un valor objectiu.

Com utilitzar la Cerca d'objectius

  1. Configura les teves dades: Assegura't de tenir una fórmula que calculi el resultat que t'interessa.
  2. Obre la Cerca d'objectius: Ves a la pestanya Dades, fes clic a Anàlisi de supòsits i selecciona Cerca d'objectius.
  3. Configura la Cerca d'objectius:
    • Establir cel·la: La cel·la que conté la fórmula.
    • A valor: El resultat desitjat que vols assolir.
    • Canviant cel·la: La cel·la que Excel canviarà per assolir el resultat desitjat.

Exemple

Suposem que vols saber quant has de vendre per assolir uns ingressos de 10.000 $.

| A | B | |-----------|---------| | Unitats venudes| 100 | | Preu | 50 $ | | Ingressos | =A2*B2 |

Per trobar les unitats venudes necessàries per assolir 10.000 $ en ingressos:

  1. Establir cel·la: B3 (Ingressos)
  2. A valor: 10000
  3. Canviant cel·la: A2 (Unitats venudes)

Taules de dades

Les Taules de dades et permeten veure com els canvis en una o dues variables afecten els resultats d'una fórmula. Hi ha dos tipus de Taules de dades:

  • Taula de dades d'una variable
  • Taula de dades de dues variables

Taula de dades d'una variable

Una Taula de dades d'una variable mostra com canviar una variable impacta el resultat.

Com crear una Taula de dades d'una variable

  1. Configura les teves dades: Crea una fórmula que depengui d'una variable.
  2. Crea una estructura de taula: Llista els valors de la variable en una columna o fila.
  3. Insereix la Taula de dades:
    • Selecciona el rang que inclou la fórmula i els valors de la variable.
    • Ves a la pestanya Dades, fes clic a Anàlisi de supòsits i selecciona Taula de dades.
    • Per a una columna de valors de variable, introdueix la referència de la cel·la per a la variable a la Cel·la d'entrada de columna.

Exemple

Calcula els ingressos per diferents unitats venudes.

| A | B | C | |-----------|---------|---------| | Unitats venudes| Ingressos | | | 50 | | | | 100 | | | | 150 | | | | 200 | | | | Fórmula | =A2*50 | |

  1. Selecciona el rang A1:B5.
  2. Ves a Dades > Anàlisi de supòsits > Taula de dades.
  3. Introdueix A2 com la Cel·la d'entrada de columna.

Taula de dades de dues variables

Una Taula de dades de dues variables mostra com canviar dues variables simultàniament afecta el resultat.

Com crear una Taula de dades de dues variables

  1. Configura les teves dades: Crea una fórmula que depengui de dues variables.
  2. Crea una estructura de taula: Llista els valors d'una variable en una fila i els de l'altra en una columna.
  3. Insereix la Taula de dades:
    • Selecciona el rang que inclou la fórmula i els valors de les variables.
    • Ves a la pestanya Dades, fes clic a Anàlisi de supòsits i selecciona Taula de dades.
    • Introdueix les referències de les cel·les per a les cel·les d'entrada de fila i columna.

Exemple

Calcula els ingressos per diferents unitats venudes i preus.

| A | B | C | D | E | |-----------|---------|---------|---------|---------| | Unitats venudes| 40 | 50 | 60 | 70 | | 100 | | | | | | 150 | | | | | | 200 | | | | | | Fórmula | =A2*B1 | | | |

  1. Selecciona el rang A1:E4.
  2. Ves a Dades > Anàlisi de supòsits > Taula de dades.
  3. Introdueix A2 com la Cel·la d'entrada de fila i B1 com la Cel·la d'entrada de columna.

Gestor d'escenaris

El Gestor d'escenaris et permet crear i desar diferents conjunts de valors d'entrada, anomenats escenaris, i canviar entre ells per veure com afecten els teus resultats.

Com utilitzar el Gestor d'escenaris

  1. Configura les teves dades: Identifica les cel·les que canviaran en cada escenari.
  2. Obre el Gestor d'escenaris: Ves a la pestanya Dades, fes clic a Anàlisi de supòsits i selecciona Gestor d'escenaris.
  3. Crea escenaris:
    • Fes clic a Afegeix per crear un nou escenari.
    • Nomena l'escenari i selecciona les cel·les canviants.
    • Introdueix els valors per a cada escenari.
  4. Visualitza escenaris: Selecciona un escenari i fes clic a Mostra per veure els resultats.

Exemple

Suposem que tens diferents estratègies de vendes amb unitats venudes i preus variables.

| A | B | C | |-----------|---------|---------| | Unitats venudes| 100 | | | Preu | 50 $ | | | Ingressos | =A2*B2 | |

  1. Ves a Dades > Anàlisi de supòsits > Gestor d'escenaris.
  2. Fes clic a Afegeix per crear un escenari per a l'Estratègia 1:
    • Nom: Estratègia 1
    • Cel·les canviants: A2, B2
    • Valors: 100, 50
  3. Afegeix un altre escenari per a l'Estratègia 2:
    • Nom: Estratègia 2
    • Cel·les canviants: A2, B2
    • Valors: 150, 45
  4. Selecciona un escenari i fes clic a Mostra per veure l'impacte en els ingressos.

Conclusió

L'anàlisi de supòsits a Excel és una eina poderosa per explorar diferents escenaris i prendre decisions informades. Utilitzant la Cerca d'objectius, les Taules de dades i el Gestor d'escenaris, pots analitzar com els canvis en les variables afecten els teus resultats i planificar en conseqüència. Practica utilitzant aquestes eines amb les teves pròpies dades per convertir-te en un expert en anàlisi de supòsits.
