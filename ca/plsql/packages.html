Introducció als Paquets

Els paquets en PL/SQL són objectes d'esquema que agrupen tipus, variables i subprogrames (procediments i funcions) PL/SQL relacionats lògicament. Ajuden a organitzar i gestionar grans programes PL/SQL proporcionant modularitat, encapsulació i reutilització.

Conceptes Clau

  • Modularitat: Els paquets permeten agrupar procediments, funcions, variables i altres construccions PL/SQL relacionades.
  • Encapsulació: Els paquets poden amagar els detalls d'implementació dels subprogrames i exposar només les parts necessàries.
  • Reutilització: Un cop creat un paquet, els seus components es poden reutilitzar en diverses aplicacions.

Estructura d'un Paquet

Un paquet en PL/SQL consta de dues parts:

  1. Especificació del Paquet: Declara els elements públics que poden ser accessibles per altres blocs PL/SQL.
  2. Cos del Paquet: Defineix la implementació dels subprogrames declarats en l'especificació del paquet.

Especificació del Paquet

L'especificació del paquet és la interfície del paquet. Declara els tipus, variables, constants, excepcions, cursos i subprogrames que són accessibles des de fora del paquet.

CREATE OR REPLACE PACKAGE package_name IS
  -- Declaracions de tipus públics
  TYPE type_name IS RECORD (...);

  -- Declaracions de variables públiques
  variable_name datatype;

  -- Declaracions de constants públiques
  constant_name CONSTANT datatype := value;

  -- Declaracions d'excepcions públiques
  exception_name EXCEPTION;

  -- Declaracions de cursos públics
  CURSOR cursor_name IS select_statement;

  -- Declaracions de procediments i funcions públiques
  PROCEDURE procedure_name(parameter_list);
  FUNCTION function_name(parameter_list) RETURN return_datatype;
END package_name;

Cos del Paquet

El cos del paquet conté la implementació dels subprogrames declarats en l'especificació del paquet. També pot contenir declaracions privades que no són accessibles fora del paquet.

CREATE OR REPLACE PACKAGE BODY package_name IS
  -- Declaracions de tipus privats
  TYPE private_type_name IS RECORD (...);

  -- Declaracions de variables privades
  private_variable_name datatype;

  -- Declaracions de constants privades
  private_constant_name CONSTANT datatype := value;

  -- Declaracions d'excepcions privades
  private_exception_name EXCEPTION;

  -- Declaracions de cursos privats
  CURSOR private_cursor_name IS select_statement;

  -- Implementació de procediments i funcions públiques
  PROCEDURE procedure_name(parameter_list) IS
  BEGIN
    -- Implementació del procediment
  END procedure_name;

  FUNCTION function_name(parameter_list) RETURN return_datatype IS
  BEGIN
    -- Implementació de la funció
    RETURN value;
  END function_name;

  -- Procediments i funcions privades
  PROCEDURE private_procedure_name IS
  BEGIN
    -- Implementació del procediment
  END private_procedure_name;

  FUNCTION private_function_name RETURN return_datatype IS
  BEGIN
    -- Implementació de la funció
    RETURN value;
  END private_function_name;
END package_name;

Exemple: Creació i Ús d'un Paquet

Creem un paquet senzill que gestiona dades d'empleats.

Especificació del Paquet

CREATE OR REPLACE PACKAGE emp_pkg IS
  -- Declaració de tipus públic
  TYPE emp_rec IS RECORD (
    emp_id NUMBER,
    emp_name VARCHAR2(100),
    emp_salary NUMBER
  );

  -- Declaració de procediment públic
  PROCEDURE add_employee(p_emp_id NUMBER, p_emp_name VARCHAR2, p_emp_salary NUMBER);

  -- Declaració de funció pública
  FUNCTION get_employee(p_emp_id NUMBER) RETURN emp_rec;
END emp_pkg;

Cos del Paquet

CREATE OR REPLACE PACKAGE BODY emp_pkg IS
  -- Variable privada per emmagatzemar dades d'empleats
  emp_data emp_rec;

  -- Implementació del procediment add_employee
  PROCEDURE add_employee(p_emp_id NUMBER, p_emp_name VARCHAR2, p_emp_salary NUMBER) IS
  BEGIN
    emp_data.emp_id := p_emp_id;
    emp_data.emp_name := p_emp_name;
    emp_data.emp_salary := p_emp_salary;
  END add_employee;

  -- Implementació de la funció get_employee
  FUNCTION get_employee(p_emp_id NUMBER) RETURN emp_rec IS
  BEGIN
    IF emp_data.emp_id = p_emp_id THEN
      RETURN emp_data;
    ELSE
      RETURN NULL;
    END IF;
  END get_employee;
END emp_pkg;

Ús del Paquet

BEGIN
  -- Afegir un empleat
  emp_pkg.add_employee(1, 'John Doe', 50000);

  -- Recuperar dades d'empleats
  DECLARE
    emp emp_pkg.emp_rec;
  BEGIN
    emp := emp_pkg.get_employee(1);
    DBMS_OUTPUT.PUT_LINE('Nom de l'empleat: ' || emp.emp_name);
  END;
END;

Conclusió

Els paquets en PL/SQL proporcionen una manera poderosa d'organitzar i gestionar el teu codi PL/SQL. Agrupant procediments, funcions i altres elements relacionats, els paquets milloren la modularitat, l'encapsulació i la reutilització. Entendre com crear i utilitzar paquets és essencial per desenvolupar aplicacions PL/SQL robustes i mantenibles.
