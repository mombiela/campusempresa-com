Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en PL/SQL. Entendre aquests conceptes és crucial per escriure codi PL/SQL eficient i efectiu.

Variables en PL/SQL

Les variables en PL/SQL s'utilitzen per emmagatzemar dades que es poden manipular durant l'execució d'un bloc de codi. Són essencials per realitzar càlculs, emmagatzemar resultats intermedis i gestionar el flux de dades.

Declaració de Variables

Per declarar una variable en PL/SQL, cal especificar el seu nom i tipus de dades. Opcionalment, també es pot inicialitzar amb un valor.

DECLARE
    variable_name data_type [:= initial_value];
BEGIN
    -- Bloc PL/SQL
END;
  • variable_name: El nom de la variable.
  • data_type: El tipus de dades que la variable contindrà.
  • initial_value: Un valor inicial opcional per a la variable.

Exemple

DECLARE
    employee_id NUMBER := 1001;
    employee_name VARCHAR2(50) := 'John Doe';
    hire_date DATE := SYSDATE;
BEGIN
    DBMS_OUTPUT.PUT_LINE('Employee ID: ' || employee_id);
    DBMS_OUTPUT.PUT_LINE('Employee Name: ' || employee_name);
    DBMS_OUTPUT.PUT_LINE('Hire Date: ' || hire_date);
END;

En aquest exemple:

  • employee_id és una variable numèrica inicialitzada a 1001.
  • employee_name és una variable de cadena inicialitzada a 'John Doe'.
  • hire_date és una variable de data inicialitzada a la data actual.

Tipus de Dades en PL/SQL

PL/SQL admet diversos tipus de dades per gestionar diferents tipus de dades. Aquests tipus de dades es poden classificar àmpliament en tipus de dades escalars, compostos, de referència i LOB (Objecte Gran).

Tipus de Dades Escalars

Els tipus de dades escalars contenen un sol valor. Inclouen:

  • NUMBER: Per a valors numèrics.
  • VARCHAR2: Per a cadenes de caràcters de longitud variable.
  • CHAR: Per a cadenes de caràcters de longitud fixa.
  • DATE: Per a valors de data i hora.
  • BOOLEAN: Per a valors lògics (TRUE, FALSE, NULL).

Exemple

DECLARE
    age NUMBER := 30;
    first_name VARCHAR2(20) := 'Alice';
    is_active BOOLEAN := TRUE;
    birth_date DATE := TO_DATE('1990-01-01', 'YYYY-MM-DD');
BEGIN
    DBMS_OUTPUT.PUT_LINE('Age: ' || age);
    DBMS_OUTPUT.PUT_LINE('First Name: ' || first_name);
    DBMS_OUTPUT.PUT_LINE('Is Active: ' || is_active);
    DBMS_OUTPUT.PUT_LINE('Birth Date: ' || birth_date);
END;

Tipus de Dades Compostos

Els tipus de dades compostos poden contenir múltiples valors. Inclouen:

  • RECORD: Un grup d'elements de dades relacionats.
  • TABLE: Una col·lecció d'elements de dades del mateix tipus.

Exemple

DECLARE
    TYPE EmployeeRec IS RECORD (
        id NUMBER,
        name VARCHAR2(50),
        salary NUMBER
    );
    employee EmployeeRec;
BEGIN
    employee.id := 1001;
    employee.name := 'John Doe';
    employee.salary := 50000;
    DBMS_OUTPUT.PUT_LINE('Employee ID: ' || employee.id);
    DBMS_OUTPUT.PUT_LINE('Employee Name: ' || employee.name);
    DBMS_OUTPUT.PUT_LINE('Employee Salary: ' || employee.salary);
END;

Tipus de Dades de Referència

Els tipus de dades de referència contenen punters a altres elements de dades. Inclouen:

  • REF CURSOR: Un punter a un cursor que es pot utilitzar per obtenir files d'un conjunt de resultats de consulta.

Exemple

DECLARE
    TYPE EmpCurTyp IS REF CURSOR;
    emp_cursor EmpCurTyp;
    emp_record employees%ROWTYPE;
BEGIN
    OPEN emp_cursor FOR SELECT * FROM employees;
    LOOP
        FETCH emp_cursor INTO emp_record;
        EXIT WHEN emp_cursor%NOTFOUND;
        DBMS_OUTPUT.PUT_LINE('Employee ID: ' || emp_record.employee_id);
    END LOOP;
    CLOSE emp_cursor;
END;

Tipus de Dades LOB

Els tipus de dades LOB s'utilitzen per emmagatzemar objectes grans com text, imatges i dades binàries. Inclouen:

  • BLOB: Objecte Gran Binari.
  • CLOB: Objecte Gran de Caràcters.
  • NCLOB: Objecte Gran de Caràcters Nacionals.
  • BFILE: Fitxer Binari.

Exemple

DECLARE
    image BLOB;
    text CLOB;
BEGIN
    -- Suposant que image i text ja estan poblats
    DBMS_OUTPUT.PUT_LINE('Image size: ' || DBMS_LOB.GETLENGTH(image));
    DBMS_OUTPUT.PUT_LINE('Text size: ' || DBMS_LOB.GETLENGTH(text));
END;

Resum

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en PL/SQL. Hem après com declarar i inicialitzar variables, i hem explorat diversos tipus de dades incloent escalars, compostos, de referència i tipus de dades LOB. Entendre aquests conceptes és fonamental per escriure programes PL/SQL robustos.
