Introducció

La instrucció SELECT és una de les comandes més fonamentals i freqüentment utilitzades en SQL i PL/SQL. Et permet recuperar dades d'una o més taules en una base de dades. Aquesta secció cobrirà els conceptes bàsics de la instrucció SELECT i gradualment avançarà cap a un ús més avançat.

Instrucció SELECT bàsica

La forma més senzilla d'una instrucció SELECT recupera totes les columnes d'una sola taula.

SELECT * FROM employees;
  • SELECT: Especifica les columnes a recuperar.
  • *: Un caràcter comodí que significa "totes les columnes."
  • FROM: Especifica la taula de la qual recuperar les dades.
  • employees: El nom de la taula.

Seleccionant columnes específiques

També pots especificar quines columnes vols recuperar.

SELECT first_name, last_name, salary FROM employees;
  • first_name, last_name, salary: Les columnes a recuperar.

Filtrant dades amb la clàusula WHERE

La clàusula WHERE et permet filtrar les files que es retornen.

SELECT first_name, last_name, salary 
FROM employees 
WHERE department_id = 10;
  • WHERE: Especifica la condició per filtrar files.
  • department_id = 10: La condició que s'ha de complir perquè una fila s'inclogui en el resultat.

Utilitzant operadors de comparació

Pots utilitzar diversos operadors de comparació en la clàusula WHERE.

  • =: Igual a
  • <>: Diferent de
  • >: Major que
  • <: Menor que
  • >=: Major o igual que
  • <=: Menor o igual que
SELECT first_name, last_name, salary 
FROM employees 
WHERE salary > 50000;

Ordenant dades amb ORDER BY

La clàusula ORDER BY et permet ordenar el conjunt de resultats.

SELECT first_name, last_name, salary 
FROM employees 
ORDER BY salary DESC;
  • ORDER BY: Especifica la columna o columnes per ordenar.
  • salary: La columna per ordenar.
  • DESC: Ordena el resultat en ordre descendent. Utilitza ASC per ordre ascendent (per defecte).

Agregant dades amb GROUP BY

La clàusula GROUP BY s'utilitza per agrupar files que tenen els mateixos valors en columnes especificades en files de resum.

SELECT department_id, AVG(salary) AS average_salary 
FROM employees 
GROUP BY department_id;
  • GROUP BY: Especifica la columna o columnes per agrupar.
  • AVG(salary): Una funció d'agregació que calcula el salari mitjà.
  • AS average_salary: Un àlies per a la columna calculada.

Funcions d'agregació comunes

  • AVG(): Calcula el valor mitjà.
  • COUNT(): Compta el nombre de files.
  • SUM(): Calcula la suma de valors.
  • MAX(): Troba el valor màxim.
  • MIN(): Troba el valor mínim.

Filtrant grups amb HAVING

La clàusula HAVING s'utilitza per filtrar grups basats en una condició.

SELECT department_id, AVG(salary) AS average_salary 
FROM employees 
GROUP BY department_id 
HAVING AVG(salary) > 60000;
  • HAVING: Especifica la condició per filtrar grups.
  • AVG(salary) > 60000: La condició que s'ha de complir perquè un grup s'inclogui en el resultat.

Unint taules

Pots recuperar dades de múltiples taules utilitzant unions.

INNER JOIN

Un INNER JOIN retorna files quan hi ha una coincidència en ambdues taules.

SELECT e.first_name, e.last_name, d.department_name 
FROM employees e 
INNER JOIN departments d ON e.department_id = d.department_id;
  • INNER JOIN: Especifica el tipus d'unió.
  • ON: Especifica la condició per a la unió.

LEFT JOIN

Un LEFT JOIN retorna totes les files de la taula esquerra i les files coincidents de la taula dreta.

SELECT e.first_name, e.last_name, d.department_name 
FROM employees e 
LEFT JOIN departments d ON e.department_id = d.department_id;

RIGHT JOIN

Un RIGHT JOIN retorna totes les files de la taula dreta i les files coincidents de la taula esquerra.

SELECT e.first_name, e.last_name, d.department_name 
FROM employees e 
RIGHT JOIN departments d ON e.department_id = d.department_id;

Subconsultes

Una subconsulta és una consulta inserida dins d'una altra consulta.

Subconsulta simple

Una subconsulta simple retorna un valor únic.

SELECT first_name, last_name 
FROM employees 
WHERE department_id = (SELECT department_id FROM departments WHERE department_name = 'Sales');

Subconsulta correlacionada

Una subconsulta correlacionada fa referència a columnes de la consulta externa.

SELECT first_name, last_name 
FROM employees e 
WHERE salary > (SELECT AVG(salary) FROM employees WHERE department_id = e.department_id);

Conclusió

La instrucció SELECT és una eina poderosa per consultar dades en PL/SQL. Dominant les seves diverses clàusules i opcions, pots recuperar i manipular dades de manera eficient. Practica utilitzant diferents tipus d'instruccions SELECT per convertir-te en un expert en la recuperació i anàlisi de dades.
