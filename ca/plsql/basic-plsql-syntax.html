Introducció a PL/SQL

PL/SQL (Procedural Language/Structured Query Language) és l'extensió procedimental de SQL i la base de dades relacional d'Oracle Corporation. Combina el poder de manipulació de dades de SQL amb el poder de processament dels llenguatges procedimentals.

Conceptes clau

  • Estructura de bloc: PL/SQL es basa en una estructura de bloc. Cada bloc consta de tres seccions: Declaració, Execució i Gestió d'Excepcions.
  • Variables: S'utilitzen per emmagatzemar dades que es poden manipular durant l'execució del programa.
  • Estructures de control: Inclouen declaracions condicionals i bucles per controlar el flux d'execució.
  • Cursors: S'utilitzen per recuperar múltiples files d'una consulta.
  • Gestió d'Excepcions: Mecanismes per gestionar errors en temps d'execució.

Estructura de bloc de PL/SQL

Un bloc de PL/SQL té la següent estructura:

DECLARE
   -- Secció de declaració: variables, constants, cursors, etc.
BEGIN
   -- Secció d'execució: declaracions SQL i PL/SQL
EXCEPTION
   -- Secció de gestió d'excepcions: codi de gestió d'errors
END;
/

Exemple d'un bloc simple de PL/SQL

DECLARE
   v_message VARCHAR2(50);
BEGIN
   v_message := 'Hello, PL/SQL!';
   DBMS_OUTPUT.PUT_LINE(v_message);
EXCEPTION
   WHEN OTHERS THEN
      DBMS_OUTPUT.PUT_LINE('An error occurred.');
END;
/
  • DECLARE: Aquesta secció és opcional i s'utilitza per declarar variables, constants i cursors.
  • BEGIN: Aquesta secció conté les declaracions executables.
  • EXCEPTION: Aquesta secció és opcional i s'utilitza per gestionar excepcions.
  • END: Marca el final del bloc de PL/SQL.
  • /: S'utilitza per executar el bloc en SQL*Plus o SQL Developer.

Variables

Les variables en PL/SQL s'utilitzen per emmagatzemar dades temporalment. Han de ser declarades a la secció de declaració.

Declaració de variables

DECLARE
   v_name VARCHAR2(30);
   v_age NUMBER;
BEGIN
   v_name := 'John Doe';
   v_age := 25;
   DBMS_OUTPUT.PUT_LINE('Name: ' || v_name);
   DBMS_OUTPUT.PUT_LINE('Age: ' || v_age);
END;
/
  • v_name: Una variable de tipus VARCHAR2.
  • v_age: Una variable de tipus NUMBER.

Estructures de control

PL/SQL admet diverses estructures de control per gestionar el flux d'execució.

Declaracions condicionals

IF-THEN-ELSE

DECLARE
   v_score NUMBER := 85;
BEGIN
   IF v_score >= 90 THEN
      DBMS_OUTPUT.PUT_LINE('Grade: A');
   ELSIF v_score >= 80 THEN
      DBMS_OUTPUT.PUT_LINE('Grade: B');
   ELSE
      DBMS_OUTPUT.PUT_LINE('Grade: C');
   END IF;
END;
/

Bucles

Bucle FOR

BEGIN
   FOR i IN 1..5 LOOP
      DBMS_OUTPUT.PUT_LINE('Iteration: ' || i);
   END LOOP;
END;
/

Bucle WHILE

DECLARE
   v_counter NUMBER := 1;
BEGIN
   WHILE v_counter <= 5 LOOP
      DBMS_OUTPUT.PUT_LINE('Counter: ' || v_counter);
      v_counter := v_counter + 1;
   END LOOP;
END;
/

Cursors

Els cursors s'utilitzen per gestionar múltiples files retornades per una consulta.

Cursor implícit

BEGIN
   FOR rec IN (SELECT first_name, last_name FROM employees WHERE department_id = 10) LOOP
      DBMS_OUTPUT.PUT_LINE('Name: ' || rec.first_name || ' ' || rec.last_name);
   END LOOP;
END;
/

Cursor explícit

DECLARE
   CURSOR emp_cursor IS SELECT first_name, last_name FROM employees WHERE department_id = 10;
   v_first_name employees.first_name%TYPE;
   v_last_name employees.last_name%TYPE;
BEGIN
   OPEN emp_cursor;
   LOOP
      FETCH emp_cursor INTO v_first_name, v_last_name;
      EXIT WHEN emp_cursor%NOTFOUND;
      DBMS_OUTPUT.PUT_LINE('Name: ' || v_first_name || ' ' || v_last_name);
   END LOOP;
   CLOSE emp_cursor;
END;
/

Gestió d'Excepcions

Les excepcions s'utilitzen per gestionar errors que es produeixen durant l'execució d'un bloc de PL/SQL.

Exemple de gestió d'excepcions

DECLARE
   v_num NUMBER;
BEGIN
   v_num := 10 / 0; -- Això causarà un error de divisió per zero
EXCEPTION
   WHEN ZERO_DIVIDE THEN
      DBMS_OUTPUT.PUT_LINE('Error: Division by zero');
   WHEN OTHERS THEN
      DBMS_OUTPUT.PUT_LINE('An unexpected error occurred');
END;
/

Conclusió

Comprendre la sintaxi bàsica de PL/SQL és crucial per escriure codi eficient i lliure d'errors. Aquest tema ha cobert els blocs fonamentals de PL/SQL, incloent l'estructura de bloc, la declaració de variables, les estructures de control, els cursors i la gestió d'excepcions. Dominar aquests conceptes bàsics proporcionarà una base sòlida per a una programació més avançada en PL/SQL.
