Introducció

Preparar-se per a una certificació PL/SQL pot ser una tasca gratificant que valida les teves habilitats i coneixements en el Llenguatge Procedural de SQL d'Oracle. Aquesta secció et guiarà a través de les àrees clau en les quals has de centrar-te, proporcionarà exemples i oferirà exercicis per assegurar-te que estàs ben preparat per a l'examen de certificació.

Conceptes Bàsics de PL/SQL

Comprensió de la Sintaxi i Estructura de PL/SQL

PL/SQL és un llenguatge estructurat en blocs. Cada bloc consta de tres seccions: declaració, execució i gestió d'excepcions.

DECLARE
  -- Secció de declaració
  v_example VARCHAR2(100);
BEGIN
  -- Secció d'execució
  v_example := 'Hello, PL/SQL!';
  DBMS_OUTPUT.PUT_LINE(v_example);
EXCEPTION
  -- Secció de gestió d'excepcions
  WHEN OTHERS THEN
    DBMS_OUTPUT.PUT_LINE('S'ha produït un error.');
END;

Declaració de Variables i Constants

Les variables i constants es declaren a la secció de declaració.

DECLARE
  v_name VARCHAR2(50);
  c_max CONSTANT NUMBER := 100;
BEGIN
  v_name := 'John Doe';
  DBMS_OUTPUT.PUT_LINE('Nom: ' || v_name);
  DBMS_OUTPUT.PUT_LINE('Valor màxim: ' || c_max);
END;

Estructures de Control

Sentències Condicionals

Les sentències condicionals et permeten executar codi basat en certes condicions.

DECLARE
  v_score NUMBER := 85;
BEGIN
  IF v_score >= 90 THEN
    DBMS_OUTPUT.PUT_LINE('Nota: A');
  ELSIF v_score >= 80 THEN
    DBMS_OUTPUT.PUT_LINE('Nota: B');
  ELSE
    DBMS_OUTPUT.PUT_LINE('Nota: C');
  END IF;
END;

Bucles

Els bucles s'utilitzen per executar un bloc de codi múltiples vegades.

BEGIN
  FOR i IN 1..5 LOOP
    DBMS_OUTPUT.PUT_LINE('Iteració: ' || i);
  END LOOP;
END;

Cursors

Cursors Implícits i Explícits

Els cursors s'utilitzen per gestionar els resultats de les consultes fila per fila.

DECLARE
  CURSOR c_employees IS
    SELECT first_name, last_name FROM employees;
  v_first_name employees.first_name%TYPE;
  v_last_name employees.last_name%TYPE;
BEGIN
  OPEN c_employees;
  LOOP
    FETCH c_employees INTO v_first_name, v_last_name;
    EXIT WHEN c_employees%NOTFOUND;
    DBMS_OUTPUT.PUT_LINE(v_first_name || ' ' || v_last_name);
  END LOOP;
  CLOSE c_employees;
END;

Gestió d'Excepcions

Excepcions Predefinides i Definides per l'Usuari

Les excepcions gestionen errors d'execució en PL/SQL.

DECLARE
  v_number NUMBER;
BEGIN
  v_number := 10 / 0; -- Això causarà una excepció predefinida
EXCEPTION
  WHEN ZERO_DIVIDE THEN
    DBMS_OUTPUT.PUT_LINE('La divisió per zero no està permesa.');
  WHEN OTHERS THEN
    DBMS_OUTPUT.PUT_LINE('S'ha produït un error inesperat.');
END;

Procediments i Funcions Emmagatzemats

Creació i Invocació de Procediments i Funcions

Els procediments i funcions encapsulen codi reutilitzable.

CREATE OR REPLACE PROCEDURE greet_user(p_name IN VARCHAR2) IS
BEGIN
  DBMS_OUTPUT.PUT_LINE('Hola, ' || p_name || '!');
END;
/

BEGIN
  greet_user('Alice');
END;

Paquets

Creació de Paquets

Els paquets agrupen procediments, funcions i altres elements relacionats.

CREATE OR REPLACE PACKAGE my_package IS
  PROCEDURE greet(p_name IN VARCHAR2);
END my_package;
/

CREATE OR REPLACE PACKAGE BODY my_package IS
  PROCEDURE greet(p_name IN VARCHAR2) IS
  BEGIN
    DBMS_OUTPUT.PUT_LINE('Hola, ' || p_name || '!');
  END greet;
END my_package;
/

BEGIN
  my_package.greet('Bob');
END;

Triggers

Creació de Triggers

Els triggers s'executen automàticament en resposta a certs esdeveniments.

CREATE OR REPLACE TRIGGER trg_before_insert
BEFORE INSERT ON employees
FOR EACH ROW
BEGIN
  :NEW.created_at := SYSDATE;
END;

Col·leccions PL/SQL

Arrays Associatius

Les col·leccions emmagatzemen múltiples elements en una sola variable.

DECLARE
  TYPE t_numbers IS TABLE OF NUMBER INDEX BY PLS_INTEGER;
  v_numbers t_numbers;
BEGIN
  v_numbers(1) := 10;
  v_numbers(2) := 20;
  DBMS_OUTPUT.PUT_LINE('Primer número: ' || v_numbers(1));
  DBMS_OUTPUT.PUT_LINE('Segon número: ' || v_numbers(2));
END;

SQL Dinàmic

Ús de EXECUTE IMMEDIATE

El SQL dinàmic et permet construir i executar sentències SQL en temps d'execució.

DECLARE
  v_table_name VARCHAR2(30) := 'employees';
  v_sql VARCHAR2(100);
BEGIN
  v_sql := 'SELECT COUNT(*) FROM ' || v_table_name;
  EXECUTE IMMEDIATE v_sql INTO v_count;
  DBMS_OUTPUT.PUT_LINE('Nombre d'empleats: ' || v_count);
END;
