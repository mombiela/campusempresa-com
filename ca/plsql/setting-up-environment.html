Configurar el teu entorn és el primer pas crucial per aprendre PL/SQL. Aquesta secció et guiarà a través de les eines i configuracions necessàries per començar amb el desenvolupament de PL/SQL.

Prerequisits

Abans d'endinsar-nos en la configuració del teu entorn, assegura't de tenir els següents prerequisits:

  • Coneixement bàsic de SQL.
  • Un ordinador amb connexió a internet.
  • Privilegis administratius per instal·lar programari.

Eines necessàries

Per desenvolupar i executar codi PL/SQL, necessitaràs les següents eines:

  1. Oracle Database: La base de dades on s'executarà el teu codi PL/SQL.
  2. SQL*Plus: Una eina de línia de comandes per executar ordres SQL i PL/SQL.
  3. Oracle SQL Developer: Una eina gràfica per al desenvolupament de bases de dades.

Instal·lació d'Oracle Database

Oracle Database és la columna vertebral del desenvolupament de PL/SQL. Segueix aquests passos per instal·lar-lo:

  1. Descarregar Oracle Database:

    • Visita la pàgina de descàrrega d'Oracle Database.
    • Tria la versió adequada per al teu sistema operatiu.

  2. Instal·lar Oracle Database:

    • Segueix les instruccions d'instal·lació proporcionades a la pàgina de descàrrega.
    • Durant la instal·lació, anota el SID (Identificador del Sistema) i la contrasenya per als usuaris SYS i SYSTEM.

Configuració de SQL*Plus

SQL*Plus és una eina de línia de comandes que et permet interactuar amb l'Oracle Database. Normalment s'inclou amb la instal·lació d'Oracle Database.

  • Llançament de SQL*Plus:
    • Obre el teu terminal o símbol del sistema.
    • Escriu sqlplus i prem Enter.
    • Inicia sessió utilitzant les credencials que vas anotar durant la instal·lació d'Oracle Database.
sqlplus SYSTEM/contrasenya@localhost:1521/SID

Substitueix contrasenya per la teva contrasenya real i SID pel teu Identificador del Sistema.

Instal·lació d'Oracle SQL Developer

Oracle SQL Developer és una eina gràfica gratuïta que simplifica les tasques de desenvolupament de bases de dades.

  1. Descarregar Oracle SQL Developer:

    • Visita la pàgina de descàrrega d'Oracle SQL Developer.
    • Tria la versió adequada per al teu sistema operatiu.

  2. Instal·lar Oracle SQL Developer:

    • Segueix les instruccions d'instal·lació proporcionades a la pàgina de descàrrega.
    • Llança SQL Developer després de la instal·lació.

  3. Configurar una connexió:

    • Obre SQL Developer.
    • Fes clic a la icona verda + per crear una nova connexió.
    • Introdueix els detalls de la connexió:
      • Nom de la connexió: Qualsevol nom que prefereixis.
      • Nom d'usuari: SYSTEM
      • Contrasenya: La teva contrasenya.
      • Nom de l'amfitrió: localhost
      • Port: 1521
      • SID: El teu SID

  4. Provar i desar la connexió:

    • Fes clic al botó Test per assegurar-te que la connexió funciona.
    • Si és correcte, fes clic a Save.

Ordres bàsiques de SQL*Plus

Aquí tens algunes ordres bàsiques de SQL*Plus per començar:

  • Connectar-se a la base de dades:

    CONNECT SYSTEM/contrasenya@localhost:1521/SID;

  • Executar una consulta SQL:

    SELECT * FROM dual;

  • Sortir de SQL*Plus:

    EXIT;

Operacions bàsiques amb SQL Developer

Aquí tens algunes operacions bàsiques que pots realitzar amb SQL Developer:

  • Executar una consulta SQL:

    • Obre un nou SQL Worksheet fent clic dret sobre la teva connexió i seleccionant SQL Worksheet.
    • Escriu la teva consulta SQL i prem Ctrl+Enter per executar.

  • Crear una nova taula:

    CREATE TABLE employees (
    employee_id NUMBER PRIMARY KEY,
    first_name VARCHAR2(50),
    last_name VARCHAR2(50),
    hire_date DATE
);

  • Inserir dades en una taula:

    INSERT INTO employees (employee_id, first_name, last_name, hire_date)
VALUES (1, 'John', 'Doe', SYSDATE);

Conclusió

Configurar el teu entorn és un primer pas crític en el desenvolupament de PL/SQL. Instal·lant Oracle Database, SQLPlus i Oracle SQL Developer, tens totes les eines necessàries per començar a escriure i executar codi PL/SQL. Assegura't de familiaritzar-te amb les ordres i operacions bàsiques tant en SQLPlus com en SQL Developer per agilitzar el teu procés de desenvolupament.

En la següent secció, ens endinsarem en els conceptes bàsics de la sintaxi i estructura de PL/SQL. Bona programació!
