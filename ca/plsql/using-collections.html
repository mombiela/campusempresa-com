Introducció a les Col·leccions

Les col·leccions en PL/SQL són tipus de dades compostes que et permeten emmagatzemar i manipular conjunts d'elements. Són similars als arrays en altres llenguatges de programació, però ofereixen més flexibilitat i funcionalitat.

Tipus de Col·leccions

PL/SQL suporta tres tipus principals de col·leccions:

  • Arrays Associatius (Taules indexades)
  • Taules Imbricades
  • VARRAYs (Arrays de mida variable)

Arrays Associatius (Taules indexades)

Els arrays associatius, també coneguts com a taules indexades, són col·leccions que utilitzen números arbitraris i cadenes com a claus.

Sintaxi i Exemple

DECLARE
  TYPE assoc_array IS TABLE OF VARCHAR2(100) INDEX BY PLS_INTEGER;
  my_array assoc_array;
BEGIN
  my_array(1) := 'Primer Element';
  my_array(2) := 'Segon Element';
  DBMS_OUTPUT.PUT_LINE(my_array(1)); -- Outputs: Primer Element
  DBMS_OUTPUT.PUT_LINE(my_array(2)); -- Outputs: Segon Element
END;

Conceptes Clau

  • Mida Dinàmica: Els arrays associatius poden créixer dinàmicament.
  • Indexació: Poden ser indexats per enters o cadenes.
  • Escàs: Els elements no necessiten ser contigus.

Taules Imbricades

Les taules imbricades són col·leccions que es poden emmagatzemar en taules de bases de dades i es poden manipular com a conjunts complets.

Sintaxi i Exemple

DECLARE
  TYPE nested_table IS TABLE OF VARCHAR2(100);
  my_table nested_table := nested_table('Element 1', 'Element 2', 'Element 3');
BEGIN
  FOR i IN 1..my_table.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_table(i));
  END LOOP;
END;

Conceptes Clau

  • Mida Dinàmica: Les taules imbricades poden créixer dinàmicament.
  • Densa: Inicialment densa però pot esdevenir escassa després de les eliminacions.
  • Emmagatzematge: Es poden emmagatzemar en columnes de bases de dades.

VARRAYs (Arrays de mida variable)

Les VARRAYs són col·leccions amb una mida màxima fixa, cosa que les fa més eficients per a certes operacions.

Sintaxi i Exemple

DECLARE
  TYPE varray_type IS VARRAY(5) OF VARCHAR2(100);
  my_varray varray_type := varray_type('Element 1', 'Element 2');
BEGIN
  FOR i IN 1..my_varray.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_varray(i));
  END LOOP;
END;

Conceptes Clau

  • Mida Fixa: La mida màxima es defineix en el moment de la declaració.
  • Densa: Sempre densa, sense buits entre elements.
  • Emmagatzematge: Es poden emmagatzemar en columnes de bases de dades.

Comparació de Tipus de Col·leccions

| Característica | Arrays Associatius | Taules Imbricades | VARRAYs | |-------------------|--------------------------|------------------------|-----------------------| | Indexació | Arbitrària (enter, cadena) | Seqüencial (enter) | Seqüencial (enter) | | Mida Dinàmica | Sí | Sí | No (mida màxima fixa) | | Emmagatzematge en BD | No | Sí | Sí | | Escassetat | Sí | Sí (després d'eliminacions) | No |

Ús Avançat

Operacions Massives

Les col·leccions PL/SQL suporten operacions massives que poden millorar significativament el rendiment.

Exemple: Bulk Collect

DECLARE
  TYPE number_table IS TABLE OF NUMBER;
  my_numbers number_table;
BEGIN
  SELECT employee_id BULK COLLECT INTO my_numbers FROM employees;
  FOR i IN 1..my_numbers.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_numbers(i));
  END LOOP;
END;

Exemple: FORALL

DECLARE
  TYPE number_table IS TABLE OF NUMBER;
  my_numbers number_table := number_table(1, 2, 3, 4, 5);
BEGIN
  FORALL i IN 1..my_numbers.COUNT
    INSERT INTO my_table (column_name) VALUES (my_numbers(i));
END;

Mètodes de Col·lecció

PL/SQL proporciona diversos mètodes integrats per manipular col·leccions:

  • COUNT: Retorna el nombre d'elements.
  • EXISTS(n): Comprova si l'element n existeix.
  • FIRST: Retorna el primer índex.
  • LAST: Retorna l'últim índex.
  • PRIOR(n): Retorna l'índex anterior a n.
  • NEXT(n): Retorna l'índex següent a n.
  • EXTEND: Afegeix elements a la col·lecció.
  • TRIM: Elimina elements de la col·lecció.
  • DELETE: Elimina tots o elements específics.

Exemple: Ús de Mètodes de Col·lecció

DECLARE
  TYPE nested_table IS TABLE OF VARCHAR2(100);
  my_table nested_table := nested_table('Element 1', 'Element 2', 'Element 3');
BEGIN
  DBMS_OUTPUT.PUT_LINE('Count: ' || my_table.COUNT); -- Outputs: Count: 3
  my_table.DELETE(2); -- Elimina el segon element
  DBMS_OUTPUT.PUT_LINE('Count after delete: ' || my_table.COUNT); -- Outputs: Count after delete: 2
  my_table.EXTEND(2); -- Afegeix dos elements més
  DBMS_OUTPUT.PUT_LINE('Count after extend: ' || my_table.COUNT); -- Outputs: Count after extend: 4
END;

Conclusió

Les col·leccions en PL/SQL són eines poderoses que et permeten gestionar conjunts de dades de manera eficient. Entendre els diferents tipus de col·leccions—arrays associatius, taules imbricades i VARRAYs—juntament amb les seves característiques i mètodes únics, et permetrà escriure codi PL/SQL més flexible i amb millor rendiment. Utilitza operacions massives i mètodes de col·lecció per optimitzar les teves tasques de processament de dades.
