Exemples de Codi

Aquí tens alguns exemples de codi senzills per començar amb PL/SQL:

Exemple de Hello World

BEGIN
  DBMS_OUTPUT.PUT_LINE('Hello, World!');
END;
/

Explicació:

  • BEGIN i END s'utilitzen per definir un bloc PL/SQL.
  • DBMS_OUTPUT.PUT_LINE és una procedura que mostra text a la consola.

Exemple de Funció Simple

CREATE OR REPLACE FUNCTION get_greeting(name IN VARCHAR2) RETURN VARCHAR2 IS
BEGIN
  RETURN 'Hello, ' || name || '!';
END;
/

Explicació:

  • CREATE OR REPLACE FUNCTION s'utilitza per definir una nova funció.
  • name IN VARCHAR2 especifica un paràmetre d'entrada.
  • RETURN VARCHAR2 especifica el tipus de retorn.
  • La funció concatena 'Hello, ' amb el nom d'entrada i retorna el resultat.
