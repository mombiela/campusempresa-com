Introducció

PL/SQL (Procedural Language/Structured Query Language) és l'extensió procedimental de SQL i la base de dades relacional d'Oracle Corporation. Combina el poder de manipulació de dades de SQL amb el poder de processament dels llenguatges procedimentals. En aquesta secció, aprendrem com escriure un programa senzill de "Hola món" en PL/SQL.

Conceptes clau

  • Estructura de bloc PL/SQL: Els programes PL/SQL estan compostos per blocs, que poden ser anònims o amb nom.
  • Bloc anònim: Un bloc PL/SQL que no està emmagatzemat a la base de dades i s'executa immediatament.
  • Paquet DBMS_OUTPUT: Un paquet que et permet mostrar la sortida dels blocs PL/SQL, subprogrames, paquets i triggers.

Estructura de bloc PL/SQL

Un bloc PL/SQL consta de tres seccions:

  1. Secció de declaració: Opcional. S'utilitza per declarar variables, constants, cursos, etc.
  2. Secció d'execució: Obligatòria. Conté el codi que realitza les operacions desitjades.
  3. Secció de gestió d'excepcions: Opcional. S'utilitza per gestionar errors en temps d'execució.

Estructura bàsica d'un bloc PL/SQL

DECLARE
   -- Secció de declaració (opcional)
BEGIN
   -- Secció d'execució (obligatòria)
EXCEPTION
   -- Secció de gestió d'excepcions (opcional)
END;
/

Escriure "Hola món" en PL/SQL

Per escriure un programa de "Hola món" en PL/SQL, utilitzarem el procediment DBMS_OUTPUT.PUT_LINE per imprimir el missatge.

Codi d'exemple

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hola, món!');
END;
/

Explicació

  • BEGIN: Marca el començament de la secció d'execució.
  • DBMS_OUTPUT.PUT_LINE: Un procediment que mostra una línia de text.
  • 'Hola, món!': La cadena que es vol imprimir.
  • END;: Marca el final del bloc PL/SQL.
  • /: Executa el bloc PL/SQL en SQL*Plus o SQL Developer.

Activar la sortida en SQL*Plus o SQL Developer

Abans d'executar el bloc PL/SQL, cal activar la sortida del servidor per veure el missatge imprès.

Activar la sortida del servidor

SET SERVEROUTPUT ON;

Exemple complet amb sortida del servidor

SET SERVEROUTPUT ON;

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hola, món!');
END;
/

Conclusió

En aquesta secció, hem après com escriure un programa senzill de "Hola món" en PL/SQL. Hem cobert l'estructura bàsica d'un bloc PL/SQL i com utilitzar el procediment DBMS_OUTPUT.PUT_LINE per imprimir la sortida. Aquest coneixement fonamental t'ajudarà a mesura que progressis cap a tasques de programació PL/SQL més complexes.
