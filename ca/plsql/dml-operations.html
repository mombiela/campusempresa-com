Introducció

En PL/SQL, les operacions del Llenguatge de Manipulació de Dades (DML) són essencials per gestionar i manipular dades dins de les taules de la base de dades. Les operacions DML principals inclouen INSERT, UPDATE i DELETE. Aquesta secció cobrirà aquestes operacions en detall, proporcionant explicacions clares, exemples i exercicis per ajudar-te a entendre i aplicar aquests conceptes de manera efectiva.

Operació INSERT

La instrucció INSERT s'utilitza per afegir noves files a una taula.

Sintaxi

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Exemple

INSERT INTO employees (employee_id, first_name, last_name, email, hire_date, job_id)
VALUES (101, 'John', 'Doe', 'john.doe@example.com', '2023-01-15', 'IT_PROG');

Explicació

  • employees: El nom de la taula on es col·locaran les dades.
  • (employee_id, first_name, last_name, email, hire_date, job_id): Les columnes de la taula on s'inseriran els valors.
  • VALUES (101, 'John', 'Doe', 'john.doe@example.com', '2023-01-15', 'IT_PROG'): Els valors que s'inseriran a les columnes especificades.

Exercici

  1. Insereix una nova fila a la taula departments amb els següents valors:
    • department_id: 10
    • department_name: 'Finance'
    • manager_id: 205
    • location_id: 1700

Operació UPDATE

La instrucció UPDATE s'utilitza per modificar files existents en una taula.

Sintaxi

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Exemple

UPDATE employees
SET email = 'john.doe@newexample.com', job_id = 'SA_REP'
WHERE employee_id = 101;

Explicació

  • employees: El nom de la taula on es modificaran les dades.
  • SET email = 'john.doe@newexample.com', job_id = 'SA_REP': Les columnes i els seus nous valors.
  • WHERE employee_id = 101: La condició per identificar quina(es) fila(es) s'actualitzaran.

Exercici

  1. Actualitza la taula departments per canviar el manager_id a 206 per al departament amb department_id 10.

Operació DELETE

La instrucció DELETE s'utilitza per eliminar files existents d'una taula.

Sintaxi

DELETE FROM table_name
WHERE condition;

Exemple

DELETE FROM employees
WHERE employee_id = 101;

Explicació

  • employees: El nom de la taula de la qual s'eliminaran les dades.
  • WHERE employee_id = 101: La condició per identificar quina(es) fila(es) s'eliminaran.

Exercici

  1. Elimina la fila de la taula departments on el department_id és 10.

Resum

En aquesta secció, hem cobert les operacions bàsiques DML en PL/SQL: INSERT, UPDATE i DELETE. Aquestes operacions et permeten afegir, modificar i eliminar dades dins de les teves taules de la base de dades. Entendre i dominar aquestes operacions és fonamental per gestionar dades de manera efectiva en qualsevol aplicació de base de dades.

  • INSERT: Afegeix noves files a una taula.
  • UPDATE: Modifica files existents en una taula.
  • DELETE: Elimina files existents d'una taula.

Practica aquestes operacions amb els exercicis proporcionats per reforçar la teva comprensió i guanyar confiança en l'ús de les instruccions DML en PL/SQL.
