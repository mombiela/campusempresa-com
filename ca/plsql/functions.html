Introducció a les Funcions

Les funcions en PL/SQL són subprogrames que retornen un únic valor. S'utilitzen per encapsular lògica reutilitzable, fent el codi modular i més fàcil de mantenir.

Conceptes Clau

  • Declaració de Funció: L'estructura d'una funció en PL/SQL.
  • Cos de la Funció: Conté la part executable de la funció.
  • Tipus de Retorn: Especifica el tipus de dades del valor retornat per la funció.
  • Paràmetres: Valors d'entrada que la funció pot acceptar.

Creació de Funcions

Per crear una funció en PL/SQL, s'utilitza la sentència CREATE FUNCTION.

Sintaxi

CREATE [OR REPLACE] FUNCTION nom_funcio 
   (parametre_1 tipus_dada, parametre_2 tipus_dada, ...)
RETURN tipus_retorn 
IS
   -- Secció de declaració
BEGIN
   -- Secció executable
   RETURN valor;
EXCEPTION
   -- Secció de gestió d'excepcions
END nom_funcio;

Exemple

Creem una funció simple que calcula el quadrat d'un nombre.

CREATE OR REPLACE FUNCTION calcula_quadrat (p_nombre IN NUMBER)
RETURN NUMBER
IS
   v_resultat NUMBER;
BEGIN
   v_resultat := p_nombre * p_nombre;
   RETURN v_resultat;
EXCEPTION
   WHEN OTHERS THEN
      RETURN NULL;
END calcula_quadrat;

Explicació

  • Nom de la Funció: calcula_quadrat
  • Paràmetre: p_nombre de tipus NUMBER
  • Tipus de Retorn: NUMBER
  • Variable: v_resultat per emmagatzemar el resultat
  • Lògica: Multiplica p_nombre per ell mateix i retorna el resultat
  • Gestió d'Excepcions: Retorna NULL si es produeix algun error

Crida de Funcions

Les funcions es poden cridar des de blocs PL/SQL, sentències SQL o altres subprogrames PL/SQL.

Exemple

DECLARE
   v_quadrat NUMBER;
BEGIN
   v_quadrat := calcula_quadrat(5);
   DBMS_OUTPUT.PUT_LINE('El quadrat de 5 és: ' || v_quadrat);
END;

Explicació

  • Declaració: Declara una variable v_quadrat per emmagatzemar el resultat.
  • Crida de Funció: Crida calcula_quadrat amb 5 com a argument.
  • Sortida: Mostra el resultat utilitzant DBMS_OUTPUT.PUT_LINE.

Característiques Avançades de les Funcions

Funcions amb Múltiples Paràmetres

Les funcions poden acceptar múltiples paràmetres, tant d'entrada (IN) com de sortida (OUT).

Exemple

CREATE OR REPLACE FUNCTION calcula_area (p_longitud IN NUMBER, p_amplada IN NUMBER)
RETURN NUMBER
IS
   v_area NUMBER;
BEGIN
   v_area := p_longitud * p_amplada;
   RETURN v_area;
EXCEPTION
   WHEN OTHERS THEN
      RETURN NULL;
END calcula_area;

Explicació

  • Paràmetres: p_longitud i p_amplada de tipus NUMBER
  • Lògica: Multiplica p_longitud per p_amplada per calcular l'àrea

Ús de Funcions en Sentències SQL

Les funcions es poden utilitzar directament en sentències SQL.

Exemple

SELECT calcula_area(10, 20) AS area FROM dual;

Explicació

  • Sentència SQL: Crida calcula_area amb 10 i 20 com a arguments i selecciona el resultat.

Funcions Recursives

Les funcions es poden cridar a elles mateixes recursivament per resoldre problemes que es poden descompondre en tasques més petites i repetitives.

Exemple

CREATE OR REPLACE FUNCTION factorial (p_nombre IN NUMBER)
RETURN NUMBER
IS
BEGIN
   IF p_nombre = 0 THEN
      RETURN 1;
   ELSE
      RETURN p_nombre * factorial(p_nombre - 1);
   END IF;
EXCEPTION
   WHEN OTHERS THEN
      RETURN NULL;
END factorial;

Explicació

  • Cas Base: Retorna 1 si p_nombre és 0.
  • Cas Recursiu: Es crida a ella mateixa amb p_nombre - 1.

Conclusió

Les funcions en PL/SQL són eines poderoses per crear codi reutilitzable i modular. Poden acceptar múltiples paràmetres, gestionar excepcions i fins i tot cridar-se a elles mateixes recursivament. Entendre com crear i utilitzar funcions de manera efectiva és essencial per escriure programes PL/SQL eficients. Practica creant i cridant funcions per convertir-te en un expert en el seu ús.
