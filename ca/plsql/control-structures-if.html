Introducció

En PL/SQL, les estructures de control s'utilitzen per dictar el flux d'execució dins d'un programa. Una de les estructures de control més fonamentals és la sentència IF, que et permet executar certes seccions de codi basant-se en condicions específiques.

Conceptes clau

  • Sentència IF: Executa un bloc de codi si una condició especificada és certa.
  • Sentència ELSIF: Proporciona condicions addicionals si la condició inicial de l'IF és falsa.
  • Sentència ELSE: Executa un bloc de codi si cap de les condicions anteriors és certa.
  • Expressions booleanes: Condicions utilitzades dins de les sentències IF que avaluen a TRUE, FALSE o NULL.

Sentència IF bàsica

La forma més simple d'una sentència IF comprova una sola condició i executa un bloc de codi si la condició és certa.

Sintaxi

IF condició THEN
    -- sentències a executar si la condició és certa
END IF;

Exemple

DECLARE
    v_salary NUMBER := 5000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salari és superior a 3000');
    END IF;
END;

En aquest exemple, es mostrarà el missatge "El salari és superior a 3000" perquè la condició v_salary > 3000 és certa.

Sentència IF-ELSE

Una sentència IF-ELSE et permet executar un bloc de codi si la condició és certa i un altre bloc si la condició és falsa.

Sintaxi

IF condició THEN
    -- sentències a executar si la condició és certa
ELSE
    -- sentències a executar si la condició és falsa
END IF;

Exemple

DECLARE
    v_salary NUMBER := 2000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salari és superior a 3000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salari és 3000 o menys');
    END IF;
END;

En aquest exemple, es mostrarà el missatge "El salari és 3000 o menys" perquè la condició v_salary > 3000 és falsa.

Sentència IF-ELSIF-ELSE

La sentència IF-ELSIF-ELSE et permet comprovar múltiples condicions seqüencialment. Si una condició és certa, s'executa el seu bloc de codi corresponent i la resta es salten.

Sintaxi

IF condició1 THEN
    -- sentències a executar si la condició1 és certa
ELSIF condició2 THEN
    -- sentències a executar si la condició2 és certa
ELSE
    -- sentències a executar si cap de les condicions anteriors és certa
END IF;

Exemple

DECLARE
    v_salary NUMBER := 4000;
BEGIN
    IF v_salary > 5000 THEN
        DBMS_OUTPUT.PUT_LINE('El salari és superior a 5000');
    ELSIF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salari és superior a 3000 però igual o inferior a 5000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salari és 3000 o menys');
    END IF;
END;

En aquest exemple, es mostrarà el missatge "El salari és superior a 3000 però igual o inferior a 5000" perquè la condició v_salary > 3000 és certa.

Sentències IF niades

Pots niar sentències IF dins d'altres per crear una lògica condicional més complexa.

Exemple

DECLARE
    v_salary NUMBER := 4500;
    v_bonus NUMBER := 500;
BEGIN
    IF v_salary > 3000 THEN
        IF v_bonus > 0 THEN
            DBMS_OUTPUT.PUT_LINE('El salari és superior a 3000 i el bonus és positiu');
        ELSE
            DBMS_OUTPUT.PUT_LINE('El salari és superior a 3000 però no hi ha bonus');
        END IF;
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salari és 3000 o menys');
    END IF;
END;

En aquest exemple, es mostrarà el missatge "El salari és superior a 3000 i el bonus és positiu" perquè ambdues condicions v_salary > 3000 i v_bonus > 0 són certes.

Conclusió

Les sentències IF són una part fonamental de la programació en PL/SQL, permetent-te controlar el flux d'execució basant-se en condicions. Entenent i utilitzant les sentències IF, IF-ELSE, IF-ELSIF-ELSE i les sentències IF niades, pots crear programes PL/SQL més dinàmics i responsius. Practica escrivint aquestes sentències amb diferents condicions per convertir-te en un expert en controlar la lògica del teu codi PL/SQL.
