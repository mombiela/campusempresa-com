Introducció

En aquesta secció, ens enfocarem en aplicar els conceptes de seguretat informàtica a través d'exercicis pràctics. Aquests exercicis estan dissenyats per reforçar el coneixement teòric i proporcionar experiència pràctica en la identificació i mitigació de vulnerabilitats.

Exercici 1: Anàlisi de Vulnerabilitats

Objectiu

Identificar i analitzar vulnerabilitats en una aplicació web utilitzant eines d'escaneig de seguretat.

Eines Recomanades

  • OWASP ZAP: Una eina d'escaneig de seguretat gratuïta i de codi obert.
  • Nmap: Utilitzat per a l'escaneig de xarxes i ports.

Passos a Seguir

  1. Instal·lació de OWASP ZAP:

    • Descarrega i instal·la OWASP ZAP des de OWASP ZAP.

  2. Escaneig d'una Aplicació Web:

    • Obre OWASP ZAP.
    • Introdueix la URL de l'aplicació web que desitges escanejar.
    • Inicia l'escaneig i espera que finalitzi.

  3. Anàlisi de Resultats:

    • Revisa els resultats de l'escaneig.
    • Identifica les vulnerabilitats trobades.
    • Classifica les vulnerabilitats segons la seva severitat (alta, mitjana, baixa).

Exemple de Codi

# Instal·lació de Nmap en una distribució basada en Debian
sudo apt-get install nmap

# Escaneig de ports oberts en una adreça IP
nmap -sS 192.168.1.1

Tasques

  • Realitza un escaneig complet d'una aplicació web de prova.
  • Documenta les vulnerabilitats trobades i la seva classificació.
  • Proposa possibles solucions per mitigar les vulnerabilitats identificades.

Exercici 2: Implementació de Polítiques de Seguretat

Objectiu

Desenvolupar i implementar polítiques de seguretat en un entorn de xarxa.

Eines Recomanades

  • Firewall: Configuració de regles de firewall per controlar el trànsit de xarxa.
  • IDS/IPS: Sistemes de detecció i prevenció d'intrusions.

Passos a Seguir

  1. Configuració d'un Firewall:

    • Defineix les regles de firewall necessàries per protegir la xarxa.
    • Implementa les regles utilitzant iptables en Linux.

  2. Implementació de IDS/IPS:

    • Instal·la i configura Snort com a IDS/IPS.
    • Defineix les regles de detecció i prevenció.

Exemple de Codi

# Configuració bàsica d'iptables per permetre només trànsit HTTP i HTTPS
sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 443 -j ACCEPT
sudo iptables -A INPUT -j DROP

# Instal·lació de Snort en una distribució basada en Debian
sudo apt-get install snort

# Configuració bàsica de Snort
sudo snort -c /etc/snort/snort.conf -i eth0

Tasques

  • Configura un firewall en un servidor Linux per permetre només trànsit HTTP i HTTPS.
  • Instal·la i configura Snort per monitoritzar el trànsit de xarxa.
  • Documenta les regles implementades i el seu propòsit.

Exercici 3: Xifrat de Dades

Objectiu

Implementar tècniques de xifrat per protegir dades sensibles.

Eines Recomanades

  • OpenSSL: Eina de línia de comandes per a operacions de xifrat.
  • GPG: Eina per a xifrat i signatura de dades.

Passos a Seguir

  1. Xifrat d'Arxius amb OpenSSL:

    • Utilitza OpenSSL per xifrar i desxifrar arxius.

  2. Xifrat de Missatges amb GPG:

    • Genera un parell de claus GPG.
    • Xifra i desxifra un missatge utilitzant GPG.

Exemple de Codi

# Xifrat d'un arxiu amb OpenSSL
openssl enc -aes-256-cbc -salt -in arxiu.txt -out arxiu.enc

# Desxifrat d'un arxiu amb OpenSSL
openssl enc -d -aes-256-cbc -in arxiu.enc -out arxiu.txt

# Generació d'un parell de claus GPG
gpg --gen-key

# Xifrat d'un missatge amb GPG
echo "Missatge secret" | gpg --encrypt --recipient 'nom@example.com' > missatge.gpg

# Desxifrat d'un missatge amb GPG
gpg --decrypt missatge.gpg

Tasques

  • Xifra i desxifra un arxiu de text utilitzant OpenSSL.
  • Genera un parell de claus GPG i xifra un missatge.
  • Documenta el procés i les comandes utilitzades.

Conclusió

Aquests exercicis pràctics estan dissenyats per proporcionar experiència pràctica en la identificació i mitigació de vulnerabilitats, la implementació de polítiques de seguretat i el xifrat de dades. En completar aquests exercicis, els estudiants estaran millor preparats per enfrontar desafiaments de seguretat en entorns reals.
