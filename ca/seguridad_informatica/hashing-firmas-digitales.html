Introducció

En aquesta secció del curs sobre Fonaments de Seguretat Informàtica, explorarem dos conceptes fonamentals: el hashing i les firmes digitals. Ambdós són essencials per garantir la integritat i autenticitat de la informació en sistemes informàtics.

Hashing

El hashing és un procés que transforma una entrada de dades (de qualsevol mida) en una sortida de mida fixa, generalment una cadena de caràcters alfanumèrics. Aquesta sortida es coneix com a hash o resum criptogràfic.

Característiques del Hashing

  • Determinístic: La mateixa entrada sempre produeix el mateix hash.
  • Ràpid de calcular: El hash de qualsevol entrada ha de ser calculable en un temps raonable.
  • Resistència a col·lisions: És difícil trobar dues entrades diferents que produeixin el mateix hash.
  • Resistència a preimatges: És difícil revertir el hash per obtenir l'entrada original.
  • Resistència a la segona preimatge: És difícil trobar una segona entrada que produeixi el mateix hash que una entrada donada.

Algoritmes de Hashing Comuns

  • MD5: Produeix un hash de 128 bits. No és segur per a aplicacions criptogràfiques a causa de vulnerabilitats conegudes.
  • SHA-1: Produeix un hash de 160 bits. També considerat insegur per a aplicacions criptogràfiques.
  • SHA-256: Part de la família SHA-2, produeix un hash de 256 bits i és àmpliament utilitzat en aplicacions de seguretat.

Exemple de Codi en Python

import hashlib

# Text d'entrada
data = "Seguretat Informàtica"

# Crear un objecte hash SHA-256
hash_object = hashlib.sha256(data.encode())

# Obtenir el hash en format hexadecimal
hex_dig = hash_object.hexdigest()

print("Hash SHA-256:", hex_dig)

Aplicacions del Hashing

  • Verificació d'integritat de fitxers: Comparar el hash d'un fitxer descarregat amb el hash proporcionat pel proveïdor.
  • Emmagatzematge de contrasenyes: Guardar el hash de les contrasenyes en lloc de les contrasenyes en si.
  • Firmes digitals: Utilitzar el hash d'un missatge com a part del procés de firma digital.

Firmes Digitals

Les firmes digitals són un mecanisme criptogràfic que permet verificar l'autenticitat i la integritat d'un missatge, programari o document digital. Utilitzen criptografia asimètrica, que involucra un parell de claus: una clau privada i una clau pública.

Procés de Firma Digital

  1. Generació del Hash: Es calcula el hash del missatge o document.
  2. Firma del Hash: El hash es xifra amb la clau privada del remitent, creant la firma digital.
  3. Enviament del Missatge i la Firma: El missatge i la firma digital s'envien al destinatari.
  4. Verificació de la Firma: El destinatari desxifra la firma digital usant la clau pública del remitent i compara el hash resultant amb el hash del missatge rebut.

Exemple de Codi en Python

from Crypto.PublicKey import RSA
from Crypto.Signature import pkcs1_15
from Crypto.Hash import SHA256

# Generar un parell de claus RSA
key = RSA.generate(2048)
private_key = key.export_key()
public_key = key.publickey().export_key()

# Missatge a signar
message = b"Seguretat Informàtica"

# Crear el hash del missatge
hash_object = SHA256.new(message)

# Signar el hash amb la clau privada
signature = pkcs1_15.new(key).sign(hash_object)

# Verificar la firma amb la clau pública
try:
    pkcs1_15.new(key.publickey()).verify(hash_object, signature)
    print("La firma és vàlida.")
except (ValueError, TypeError):
    print("La firma no és vàlida.")

Aplicacions de les Firmes Digitals

  • Autenticació de missatges: Garantir que el missatge prové del remitent legítim.
  • Integritat de documents: Assegurar que el document no ha estat alterat.
  • Certificats digitals: Verificar la identitat d'entitats en comunicacions segures.

Conclusió

El hashing i les firmes digitals són components crucials en la seguretat informàtica. El hashing permet verificar la integritat de les dades, mentre que les firmes digitals asseguren l'autenticitat i la integritat dels missatges i documents. Comprendre aquests conceptes i la seva implementació és essencial per a qualsevol professional de TI que treballi en el camp de la seguretat informàtica.
