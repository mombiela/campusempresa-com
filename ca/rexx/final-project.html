El Projecte Sobre nosaltres Contribuir Donacions Llicència

Visió General

El projecte final d'aquest curs de REXX està dissenyat per consolidar el teu aprenentatge des de nivells principiants fins a avançats. Aplicaràs els conceptes, tècniques i millors pràctiques que has après al llarg del curs per crear un script REXX complet. Aquest projecte posarà a prova la teva comprensió de la programació en REXX i la teva capacitat per resoldre problemes del món real utilitzant REXX.

Requisits del Projecte

El teu projecte final hauria de complir els següents requisits:

  • Longitud de l'Script: L'script hauria de tenir almenys 100 línies de codi.
  • Funcionalitat: L'script hauria de realitzar una tasca significativa, com ara processament de dades, manipulació d'arxius o automatització d'una tasca repetitiva.
  • Ús de Funcions de REXX: L'script hauria de demostrar l'ús de diverses funcions de REXX, incloent:
    • Variables i tipus de dades
    • Estructures de control (bucles, condicionals)
    • Funcions i subrutines
    • Operacions d'entrada/sortida d'arxius
    • Gestió d'errors
  • Documentació: L'script hauria d'estar ben documentat amb comentaris que expliquin la finalitat de cada secció de codi.
  • Interacció amb l'Usuari: L'script hauria d'incloure alguna forma d'interacció amb l'usuari, com ara sol·licituds d'entrada o arguments de línia de comandes.

Idees de Projectes Exemple

Aquí tens algunes idees de projectes exemple per inspirar-te:

  • Analitzador de Fitxers de Registre: Crea un script que llegeixi un fitxer de registre, extregui informació rellevant i generi un informe resum.
  • Automatització de Tasques: Desenvolupa un script que automatitzi una tasca repetitiva, com ara canviar el nom dels fitxers en un directori segons un patró específic.
  • Processament de Dades: Escriu un script que processi un fitxer CSV, realitzi transformacions de dades i generi els resultats en un nou fitxer.
  • Quiz Interactiu: Crea una aplicació de quiz interactiu que faci una sèrie de preguntes a l'usuari i proporcioni retroalimentació segons les seves respostes.

Codi Exemple

A continuació es mostra un exemple d'un simple script REXX que llegeix un fitxer de text i compta el nombre de línies:

/* Simple REXX script to count lines in a file */
parse arg filename

if filename = '' then do
  say 'Please provide a filename as an argument.'
  exit 1
end

lines = 0

/* Open the file for reading */
call stream filename, 'c', 'open read'

if stream(filename, 's') = 'READY:' then do
  do while lines = stream(filename, 'c', 'read')
    lines = lines + 1
  end
  call stream filename, 'c', 'close'
  say 'Number of lines in the file:' lines
end
else do
  say 'Failed to open the file.'
  exit 1
end

Explicació

  • parse arg filename: Aquesta línia recupera el nom del fitxer dels arguments de la línia de comandes.
  • if filename = '' then do: Aquest bloc verifica si s'ha proporcionat un nom de fitxer i surt si no és així.
  • lines = 0: Inicialitza el comptador de línies.
  • call stream filename, 'c', 'open read': Obre el fitxer per llegir-lo.
  • if stream(filename, 's') = 'READY:' then do: Verifica si el fitxer s'ha obert correctament.
  • do while lines = stream(filename, 'c', 'read'): Llegeix cada línia del fitxer i incrementa el comptador de línies.
  • call stream filename, 'c', 'close': Tanca el fitxer després de llegir-lo.
  • say 'Number of lines in the file:' lines: Mostra el nombre total de línies.

Directrius de Presentació

  • Presentació del Codi: Presenta el teu fitxer d'script REXX (.rexx o .rex) a través de la plataforma del curs.
  • Documentació: Inclou un fitxer README que expliqui la finalitat del teu script, com executar-lo i qualsevol dependència o requisit.
  • Data Límit: Assegura't de presentar el teu projecte abans de la data límit especificada.

Conclusió

El projecte final és una oportunitat per mostrar les teves habilitats de programació en REXX i la teva creativitat. En completar aquest projecte, demostraràs la teva capacitat per aplicar REXX per resoldre problemes pràctics i crear scripts útils. Bona sort i diverteix-te programant!
