La manipulació de fitxers és una habilitat essencial per a qualsevol programador. En aquesta secció, explorarem com gestionar fitxers utilitzant REXX, des d'operacions bàsiques fins a tècniques més avançades.

Introducció a la Manipulació de Fitxers

La manipulació de fitxers implica diverses operacions com ara llegir fitxers, escriure fitxers i gestionar atributs de fitxers. En REXX, aquestes operacions es realitzen utilitzant funcions i comandes integrades.

Conceptes Clau

  • Gestió de Fitxers: Obrir, llegir, escriure i tancar fitxers.
  • Funcions d'I/O de Fitxers: Funcions proporcionades per REXX per a l'entrada i sortida de fitxers.
  • Gestió d'Errors: Gestionar errors que es produeixen durant les operacions de fitxers.

Operacions Bàsiques de Fitxers

Obrir i Tancar Fitxers

Per realitzar qualsevol operació en un fitxer, primer cal obrir-lo. En REXX, els fitxers es solen obrir utilitzant la funció OPEN i es tanquen utilitzant la funció CLOSE.

Exemple: Obrir i Tancar un Fitxer

/* Obrir un fitxer per llegir */
file = 'example.txt'
fileHandle = open(file, 'r')

/* Realitzar operacions en el fitxer */

/* Tancar el fitxer */
call close(fileHandle)

Llegir Fitxers

Llegir un fitxer es pot fer línia per línia o en blocs. La funció LINEIN s'utilitza comunament per llegir línies d'un fitxer.

Exemple: Llegir un Fitxer Línia per Línia

file = 'example.txt'
fileHandle = open(file, 'r')

do while lines(fileHandle) > 0
    line = linein(fileHandle)
    say line
end

call close(fileHandle)

Escriure Fitxers

Escriure en un fitxer es fa utilitzant la funció LINEOUT. Es poden escriure línies en un fitxer una a una.

Exemple: Escriure en un Fitxer

file = 'output.txt'
fileHandle = open(file, 'w')

call lineout fileHandle, 'Aquesta és la primera línia.'
call lineout fileHandle, 'Aquesta és la segona línia.'

call close(fileHandle)

Operacions Avançades de Fitxers

Afegir a Fitxers

Afegir dades a un fitxer existent es pot fer utilitzant la funció OPEN amb el mode d'afegir (a).

Exemple: Afegir a un Fitxer

file = 'output.txt'
fileHandle = open(file, 'a')

call lineout fileHandle, 'Aquesta línia s'ha afegit.'

call close(fileHandle)

Gestió d'Errors en Operacions de Fitxers

Una bona gestió d'errors assegura que el teu programa pugui gestionar de manera elegant problemes com fitxers inexistents o errors de permisos.

Exemple: Gestió d'Errors

file = 'nonexistent.txt'
fileHandle = open(file, 'r')

if fileHandle = '' then do
    say 'Error: No es pot obrir el fitxer.'
    exit
end

/* Realitzar operacions en el fitxer */

call close(fileHandle)

Funcions de Gestió de Fitxers

REXX proporciona diverses funcions per gestionar fitxers, com DELETE per eliminar un fitxer i EXISTS per comprovar si un fitxer existeix.

Exemple: Eliminar un Fitxer

file = 'output.txt'

if exists(file) then do
    call delete file
    say 'Fitxer eliminat amb èxit.'
end
else do
    say 'El fitxer no existeix.'
end

Resum

En aquesta secció, hem cobert les bases i algunes tècniques avançades per a la manipulació de fitxers en REXX. Has après a:

  • Obrir i tancar fitxers.
  • Llegir i escriure fitxers.
  • Afegir dades a fitxers.
  • Gestionar errors durant les operacions de fitxers.
  • Gestionar fitxers utilitzant funcions integrades.

Entendre aquests conceptes et permetrà gestionar fitxers de manera efectiva en els teus programes REXX, fent els teus scripts més robustos i versàtils.
