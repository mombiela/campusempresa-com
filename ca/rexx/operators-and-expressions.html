Introducció

En aquesta secció, explorarem els diferents operadors i expressions en REXX. Els operadors són símbols especials que realitzen operacions sobre variables i valors. Les expressions són combinacions de variables, valors i operadors que REXX avalua per produir un altre valor. Entendre els operadors i les expressions és fonamental per escriure programes REXX efectius.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • Suma (+): Suma dos números.
  • Resta (-): Resta el segon número del primer.
  • Multiplicació (*): Multiplica dos números.
  • Divisió (/): Divideix el primer número pel segon.
  • Divisió Entera (%): Divideix el primer número pel segon i retorna la part entera del quocient.
  • Mòdul (//): Divideix el primer número pel segon i retorna el residu.

Exemples

/* Suma */
result = 5 + 3
say result  /* Sortida: 8 */

/* Resta */
result = 10 - 4
say result  /* Sortida: 6 */

/* Multiplicació */
result = 7 * 2
say result  /* Sortida: 14 */

/* Divisió */
result = 20 / 4
say result  /* Sortida: 5 */

/* Divisió Entera */
result = 20 % 3
say result  /* Sortida: 6 */

/* Mòdul */
result = 20 // 3
say result  /* Sortida: 2 */

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors. Retornen un valor booleà (1 per cert, 0 per fals).

  • Igual (= o ==): Comprova si dos valors són iguals.
  • No Igual (\= o <>): Comprova si dos valors no són iguals.
  • Menor Que (<): Comprova si el primer valor és menor que el segon.
  • Major Que (>): Comprova si el primer valor és major que el segon.
  • Menor o Igual Que (<=): Comprova si el primer valor és menor o igual que el segon.
  • Major o Igual Que (>=): Comprova si el primer valor és major o igual que el segon.

Exemples

/* Igual */
result = 5 = 5
say result  /* Sortida: 1 */

/* No Igual */
result = 5 \= 3
say result  /* Sortida: 1 */

/* Menor Que */
result = 3 < 5
say result  /* Sortida: 1 */

/* Major Que */
result = 7 > 4
say result  /* Sortida: 1 */

/* Menor o Igual Que */
result = 5 <= 5
say result  /* Sortida: 1 */

/* Major o Igual Que */
result = 6 >= 3
say result  /* Sortida: 1 */

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions.

  • I (&): Retorna cert si ambdues condicions són certes.
  • O (|): Retorna cert si almenys una condició és certa.
  • NO (\): Retorna el contrari de la condició.

Exemples

/* I */
result = (5 = 5) & (3 < 4)
say result  /* Sortida: 1 */

/* O */
result = (5 = 5) | (3 > 4)
say result  /* Sortida: 1 */

/* NO */
result = \ (5 = 3)
say result  /* Sortida: 1 */

Operadors de Cadena

Els operadors de cadena s'utilitzen per manipular cadenes.

  • Concatenació (||): Uneix dues cadenes.
  • Subcadena (substr): Extreu una subcadena d'una cadena.

Exemples

/* Concatenació */
result = "Hello" || " " || "World"
say result  /* Sortida: Hello World */

/* Subcadena */
result = substr("Hello World", 1, 5)
say result  /* Sortida: Hello */

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació Simple (=): Assigna el valor de la dreta a la variable de l'esquerra.
  • Assignació Composta (+=, -=, *=, /=, %=, //=): Combina una operació aritmètica amb l'assignació.

Exemples

/* Assignació Simple */
a = 10
say a  /* Sortida: 10 */

/* Assignació Composta */
a += 5
say a  /* Sortida: 15 */

a -= 3
say a  /* Sortida: 12 */

a *= 2
say a  /* Sortida: 24 */

a /= 4
say a  /* Sortida: 6 */

a %= 2
say a  /* Sortida: 3 */

a //= 2
say a  /* Sortida: 1 */

Resum

En aquesta secció, hem cobert els diferents operadors i expressions en REXX, incloent operadors aritmètics, de comparació, lògics, de cadena i d'assignació. Entendre aquests operadors i com utilitzar-los en expressions és crucial per escriure programes REXX efectius. Practica utilitzant aquests operadors en diferents escenaris per sentir-te més còmode amb el seu ús.
