Introducció a les estructures de dades

Les estructures de dades són essencials per organitzar i gestionar dades de manera eficient. En REXX, entendre com utilitzar i manipular diverses estructures de dades t'ajudarà a escriure programes més efectius i eficients.

Tipus de dades bàsics

Abans d'endinsar-nos en estructures de dades complexes, és important entendre els tipus de dades bàsics en REXX:

  • Cadenes: El tipus de dades més comú en REXX, utilitzat per a text i números.
  • Números: REXX tracta els números com a cadenes però permet operacions aritmètiques.

Exemple: Tipus de dades bàsics

/* Exemple de cadena */
name = "John Doe"
say "Nom: " name

/* Exemple de número */
age = 30
say "Edat: " age

Matrius

Les matrius en REXX són matrius associatives, també conegudes com a variables de tija. Et permeten emmagatzemar i recuperar dades utilitzant claus.

Crear i utilitzar matrius

  • Definir una matriu: Utilitza una variable de tija seguida d'un punt.
  • Assignar valors: Utilitza la variable de tija amb un índex.
  • Accedir als valors: Utilitza l'índex per recuperar el valor.

Exemple: Matrius

/* Definir una matriu */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Accedir als elements de la matriu */
say "Estudiant 1: " students.1
say "Estudiant 2: " students.2
say "Estudiant 3: " students.3

Llistes

REXX no té estructures de dades de llista integrades, però pots simular llistes utilitzant matrius o cadenes.

Exemple: Llistes utilitzant matrius

/* Definir una llista utilitzant una matriu */
list.1 = "apple"
list.2 = "banana"
list.3 = "cherry"

/* Iterar sobre la llista */
do i = 1 to 3
    say "Element " i ": " list.i
end

Exemple: Llistes utilitzant cadenes

/* Definir una llista utilitzant una cadena */
list = "apple banana cherry"

/* Dividir la cadena en paraules */
parse var list item1 item2 item3

say "Element 1: " item1
say "Element 2: " item2
say "Element 3: " item3

Piles i cues

Les piles i cues es poden implementar utilitzant matrius o llistes. Una pila segueix l'ordre Últim en Entrar Primer en Sortir (LIFO), mentre que una cua segueix l'ordre Primer en Entrar Primer en Sortir (FIFO).

Exemple: Implementació de pila

/* Definir una pila */
stack.1 = "first"
stack.2 = "second"
stack.3 = "third"

/* Treure de la pila */
do i = 3 to 1 by -1
    say "Tret: " stack.i
end

Exemple: Implementació de cua

/* Definir una cua */
queue.1 = "first"
queue.2 = "second"
queue.3 = "third"

/* Desencuar */
do i = 1 to 3
    say "Desencuat: " queue.i
end

Taules de hash

Les matrius associatives de REXX es poden utilitzar per crear taules de hash, on cada clau s'associa a un valor.

Exemple: Taula de hash

/* Definir una taula de hash */
hashTable.["name"] = "John Doe"
hashTable.["age"] = 30
hashTable.["city"] = "New York"

/* Accedir als elements de la taula de hash */
say "Nom: " hashTable.["name"]
say "Edat: " hashTable.["age"]
say "Ciutat: " hashTable.["city"]

Estructures de dades avançades

Per a estructures de dades més complexes, potser necessitaràs combinar estructures bàsiques o utilitzar biblioteques externes.

Exemple: Matrius niades

/* Definir una matriu niada */
students.1.name = "Alice"
students.1.age = 20
students.2.name = "Bob"
students.2.age = 22

/* Accedir als elements de la matriu niada */
say "Estudiant 1: " students.1.name ", Edat: " students.1.age
say "Estudiant 2: " students.2.name ", Edat: " students.2.age

Conclusió

Entendre i utilitzar estructures de dades en REXX és crucial per a una programació eficient. Des de matrius bàsiques fins a estructures niades més complexes, dominar aquests conceptes millorarà la teva capacitat per gestionar i manipular dades de manera efectiva. Practica amb els exemples proporcionats i explora més per aprofundir en els teus coneixements.
