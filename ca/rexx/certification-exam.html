Visió General

L'examen de certificació per a REXX (Restructured Extended Executor) està dissenyat per provar els teus coneixements i habilitats en diversos nivells de competència, des de principiant fins a avançat. Aquesta secció et guiarà a través de les àrees clau en què has de centrar-te, proporcionarà exemples i oferirà consells per a la preparació.

Àrees Clau de Focalització

1. Sintaxi i Estructura Bàsica

  • Comprensió de la Sintaxi de REXX: Aprèn les regles bàsiques de la sintaxi, incloent com escriure i executar programes simples en REXX.
  • Variables i Tipus de Dades: Entén com declarar i utilitzar variables, i els diferents tipus de dades disponibles en REXX.
  • Estructures de Control: Familiaritza't amb bucles (DO, DO WHILE, DO UNTIL) i declaracions condicionals (IF, THEN, ELSE).
/* Programa Simple en REXX */
say "Hola, Món!"

/* Variables */
nom = "Joan"
say "Hola, " nom

/* Declaració Condicional */
if nom = "Joan" then
  say "Benvingut, Joan!"
else
  say "Qui ets?"

2. Funcions i Procediments

  • Funcions Incorporades: Aprèn les funcions incorporades més utilitzades com LENGTH, SUBSTR, DATE i TIME.
  • Funcions Definides per l'Usuari: Entén com crear i utilitzar les teves pròpies funcions i procediments.
/* Exemple de Funció Incorporada */
nom = "Joan"
say "La longitud del nom és: " LENGTH(nom)

/* Funció Definida per l'Usuari */
say saluda("Joan")

saluda: procedure
  parse arg nom
  return "Hola, " nom

3. Manipulació de Cadenes

  • Concatenació: Aprèn com concatenar cadenes utilitzant l'operador ||.
  • Subcadenes: Extreu parts de cadenes utilitzant la funció SUBSTR.
  • Coincidència de Patrons: Utilitza la funció POS per trobar patrons dins de cadenes.
/* Concatenació de Cadenes */
primerNom = "Joan"
cognom = "Doe"
nomComplet = primerNom || " " || cognom
say nomComplet

/* Extracció de Subcadenes */
text = "Hola, Món!"
say SUBSTR(text, 1, 5)  /* Sortida: Hola */

/* Coincidència de Patrons */
text = "Hola, Món!"
posició = POS("Món", text)
say "Posició de 'Món': " posició

4. Operacions d'E/S de Fitxers

  • Lectura de Fitxers: Aprèn com obrir, llegir i tancar fitxers.
  • Escriptura de Fitxers: Entén com escriure dades a fitxers.
/* Lectura d'un Fitxer */
nomFitxer = "exemple.txt"
call stream nomFitxer, "c", "open read"
do while lines(nomFitxer) > 0
  línia = linein(nomFitxer)
  say línia
end
call stream nomFitxer, "c", "close"

/* Escriptura a un Fitxer */
nomFitxer = "sortida.txt"
call stream nomFitxer, "c", "open write"
call lineout nomFitxer, "Això és una prova."
call stream nomFitxer, "c", "close"

5. Gestió d'Errors i Depuració

  • Captura d'Errors: Utilitza la instrucció SIGNAL per gestionar errors.
  • Tècniques de Depuració: Aprèn com utilitzar TRACE per depurar els teus programes en REXX.
/* Captura d'Errors */
signal on syntax name gestorErrors
say "Això causarà un error: " 1 / 0
exit

gestorErrors:
  say "S'ha produït un error!"
  exit

/* Depuració amb TRACE */
trace ?r
say "Aquest és un missatge de depuració."

Consells per a la Preparació

  • Practica Regularment: Escriu i executa programes en REXX per familiaritzar-te amb la sintaxi i les característiques.
  • Revisa la Documentació: Repassa la documentació oficial de REXX i qualsevol recurs addicional proporcionat en el curs.
  • Exàmens de Prova: Fes exàmens de prova per familiaritzar-te amb el format i els tipus de preguntes que podries trobar.

Conclusió

Preparar-se per a l'examen de certificació de REXX requereix una comprensió exhaustiva de la sintaxi, funcions i característiques del llenguatge. Centrant-te en les àrees clau esmentades anteriorment i practicant regularment, pots construir la confiança i les habilitats necessàries per tenir èxit. Bona sort!
