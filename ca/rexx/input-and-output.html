Introducció

Les operacions d'entrada i sortida (I/O) són fonamentals en qualsevol llenguatge de programació, ja que permeten que els programes interactuïn amb els usuaris i altres sistemes. En REXX, les operacions d'I/O són senzilles i fàcils d'aprendre, cosa que el converteix en un llenguatge excel·lent tant per a principiants com per a usuaris avançats.

Entrada Bàsica

Utilitzant la instrucció `PULL`

La instrucció PULL s'utilitza per llegir l'entrada de l'usuari. Espera que l'usuari escrigui alguna cosa i després emmagatzema l'entrada en una variable.

/* Exemple d'ús de PULL per obtenir l'entrada de l'usuari */
say "Si us plau, introdueix el teu nom:"
pull name
say "Hola, " name "!"
  • Explicació:
    • say "Si us plau, introdueix el teu nom:" mostra una sol·licitud a l'usuari.
    • pull name espera que l'usuari escrigui alguna cosa i ho emmagatzema a la variable name.
    • say "Hola, " name "!" mostra una salutació utilitzant l'entrada.

Sortida Bàsica

Utilitzant la instrucció `SAY`

La instrucció SAY s'utilitza per mostrar la sortida a l'usuari.

/* Exemple d'ús de SAY per mostrar la sortida */
say "Benvingut al curs de programació REXX!"
  • Explicació:
    • say "Benvingut al curs de programació REXX!" imprimeix el missatge a la pantalla.

Entrada i Sortida de Fitxers

Llegint d'un Fitxer

Per llegir d'un fitxer, s'utilitza la funció LINEIN. Aquesta funció llegeix una línia d'un fitxer.

/* Exemple de lectura d'un fitxer */
fileName = "example.txt"
file = open(fileName, "r")
if file = "" then do
  say "Error: No s'ha pogut obrir el fitxer."
  exit
end

line = linein(file)
do while line \= ""
  say line
  line = linein(file)
end

call close file
  • Explicació:
    • fileName = "example.txt" especifica el fitxer a llegir.
    • file = open(fileName, "r") obre el fitxer per a lectura.
    • if file = "" then do comprova si el fitxer s'ha obert correctament.
    • line = linein(file) llegeix una línia del fitxer.
    • do while line \= "" recorre el fitxer fins al final.
    • say line imprimeix cada línia a la pantalla.
    • call close file tanca el fitxer.

Escriure a un Fitxer

Per escriure a un fitxer, s'utilitza la funció LINEOUT. Aquesta funció escriu una línia a un fitxer.

/* Exemple d'escriptura a un fitxer */
fileName = "output.txt"
file = open(fileName, "w")
if file = "" then do
  say "Error: No s'ha pogut obrir el fitxer."
  exit
end

line = "Aquesta és una línia de text."
call lineout file, line

call close file
  • Explicació:
    • fileName = "output.txt" especifica el fitxer on escriure.
    • file = open(fileName, "w") obre el fitxer per a escriptura.
    • if file = "" then do comprova si el fitxer s'ha obert correctament.
    • line = "Aquesta és una línia de text." especifica la línia a escriure.
    • call lineout file, line escriu la línia al fitxer.
    • call close file tanca el fitxer.

Operacions d'I/O Avançades

Utilitzant Fluxos

REXX proporciona funcions de flux per a operacions de fitxer més avançades. La funció STREAM es pot utilitzar per consultar i manipular fluxos de fitxer.

/* Exemple d'ús de la funció STREAM */
fileName = "example.txt"
file = open(fileName, "r")
if file = "" then do
  say "Error: No s'ha pogut obrir el fitxer."
  exit
end

position = stream(file, "seek", "end")
say "La mida del fitxer és:" position "bytes."

call close file
  • Explicació:
    • position = stream(file, "seek", "end") mou el punter del fitxer al final i retorna la mida del fitxer.
    • say "La mida del fitxer és:" position "bytes." imprimeix la mida del fitxer.

Conclusió

Entendre les operacions d'entrada i sortida en REXX és crucial per crear programes interactius i útils. Dominant l'I/O bàsic amb PULL i SAY, l'I/O de fitxer amb LINEIN i LINEOUT, i les operacions avançades amb STREAM, podràs gestionar una àmplia gamma de tasques en els teus programes REXX. Practica aquests conceptes amb diversos exemples per convertir-te en un expert en les operacions d'I/O en REXX.
