Subrutines i Funcions en REXX
Lectura de Fitxers ►

Introducció

Les subrutines i funcions són conceptes fonamentals en la programació REXX que et permeten modularitzar el teu codi, fent-lo més llegible, mantenible i reutilitzable. Aquesta secció et guiarà a través dels conceptes bàsics de subrutines i funcions, i introduirà gradualment conceptes més avançats.

Conceptes Clau

  • Subrutines: Blocs de codi que realitzen una tasca específica i es poden cridar des de diferents parts d'un programa.
  • Funcions: Similars a les subrutines però dissenyades per retornar un valor.
  • Instrucció CALL: Utilitzada per invocar una subrutina.
  • Instrucció RETURN: Utilitzada per retornar el control al codi que crida, opcionalment amb un valor.

Subrutines

Definició d'una Subrutina

Una subrutina en REXX es defineix utilitzant una etiqueta seguida d'un dos punts. La subrutina es pot cridar utilitzant la instrucció CALL.

/* Programa Principal */
say "Iniciant el programa"
call mySubroutine
say "De tornada al programa principal"

/* Definició de la Subrutina */
mySubroutine:
    say "Dins de la subrutina"
    return

Cridar una Subrutina

Pots cridar una subrutina des de qualsevol lloc del teu programa utilitzant la instrucció CALL.

call mySubroutine

Passar Arguments a Subrutines

Es poden passar arguments a les subrutines i accedir-hi utilitzant la instrucció ARG.

/* Programa Principal */
call mySubroutine "Hola, Món!"

/* Definició de la Subrutina */
mySubroutine:
    parse arg message
    say message
    return

Funcions

Definició d'una Funció

Les funcions són similars a les subrutines però estan dissenyades per retornar un valor. Es defineixen utilitzant la mateixa sintaxi d'etiqueta i dos punts.

/* Programa Principal */
result = myFunction(5, 3)
say "El resultat és" result

/* Definició de la Funció */
myFunction:
    parse arg a, b
    return a + b

Cridar una Funció

Les funcions es criden utilitzant el seu nom seguit de parèntesis que contenen qualsevol argument.

result = myFunction(5, 3)

Retornar Valors de Funcions

La instrucció RETURN s'utilitza per retornar un valor d'una funció.

return a + b

Conceptes Avançats

Variables Locals

Les variables en subrutines i funcions són globals per defecte. Per crear variables locals, utilitza la instrucció PROCEDURE.

/* Programa Principal */
x = 10
call mySubroutine
say "Valor de x en el programa principal:" x

/* Definició de la Subrutina */
mySubroutine:
    procedure
    x = 5
    say "Valor de x en la subrutina:" x
    return

Funcions Recursives

REXX suporta funcions recursives, on una funció es crida a si mateixa.

/* Programa Principal */
result = factorial(5)
say "El factorial de 5 és" result

/* Definició de la Funció Recursiva */
factorial:
    parse arg n
    if n = 1 then return 1
    return n * factorial(n - 1)

Resum

Les subrutines i funcions són eines essencials en REXX per crear codi modular i reutilitzable. Les subrutines realitzen tasques sense retornar valors, mentre que les funcions estan dissenyades per retornar valors. Entendre com definir, cridar i gestionar aquests constructes, incloent-hi la gestió d'arguments i variables locals, és crucial per a una programació efectiva en REXX. En dominar aquests conceptes, podràs escriure codi més organitzat i eficient.
