Configurar l'entorn REXX és el primer pas per dominar aquest potent llenguatge de script. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de REXX al teu sistema, assegurant que tinguis tot el que necessites per començar a escriure i executar scripts REXX.

Instal·lar REXX

Abans de poder començar a escriure scripts REXX, necessites instal·lar un intèrpret REXX. Hi ha diverses opcions disponibles, depenent del teu sistema operatiu.

Windows

Per als usuaris de Windows, Regina REXX és una opció popular. Segueix aquests passos per instal·lar-lo:

  1. Descarregar Regina REXX:

    • Visita el lloc web de Regina REXX.
    • Descarrega l'instal·lador apropiat per a la teva versió de Windows.

  2. Instal·lar Regina REXX:

    • Executa l'instal·lador descarregat.
    • Segueix les instruccions en pantalla per completar la instal·lació.

  3. Verificar la Instal·lació:

    • Obre el Símbol del sistema.
    • Escriu rexx i prem Enter. Si està instal·lat correctament, hauries de veure el prompt de REXX.

macOS

Per als usuaris de macOS, pots utilitzar el gestor de paquets Homebrew per instal·lar Regina REXX:

  1. Instal·lar Homebrew (si no està ja instal·lat):

    • Obre Terminal.
    • Executa la comanda: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·lar Regina REXX:

    • Executa la comanda: 
      brew install regina-rexx

  3. Verificar la Instal·lació:

    • Escriu rexx a Terminal i prem Enter. Hauries de veure el prompt de REXX.

Linux

Per als usuaris de Linux, Regina REXX es pot instal·lar utilitzant el gestor de paquets de la teva distribució:

  1. Debian/Ubuntu:

    • Obre Terminal.
    • Executa la comanda: 
      sudo apt-get install regina-rexx

  2. Fedora:

    • Obre Terminal.
    • Executa la comanda: 
      sudo dnf install regina-rexx

  3. Verificar la Instal·lació:

    • Escriu rexx a Terminal i prem Enter. Hauries de veure el prompt de REXX.

Configurar l'Entorn

Després d'instal·lar REXX, potser necessitis configurar el teu entorn per facilitar l'execució de scripts REXX.

Configurar Variables d'Entorn

Configurar variables d'entorn pot ajudar-te a executar scripts REXX des de qualsevol directori sense especificar el camí complet a l'intèrpret.

Windows

  1. Obrir Variables d'Entorn:

    • Fes clic amb el botó dret a 'Aquest PC' o 'Ordinador' a l'escriptori o a l'Explorador de fitxers.
    • Selecciona 'Propietats'.
    • Fes clic a 'Configuració avançada del sistema'.
    • Fes clic a 'Variables d'entorn'.

  2. Afegir REXX a PATH:

    • A la secció 'Variables del sistema', troba i selecciona la variable 'Path', després fes clic a 'Edita'.
    • Fes clic a 'Nou' i afegeix el camí al directori on està instal·lat Regina REXX (per exemple, C:\Program Files\Regina).

  3. Aplicar Canvis:

    • Fes clic a 'D'acord' per tancar tots els quadres de diàleg.

macOS i Linux

  1. Obrir Terminal.

  2. Editar el Fitxer de Configuració de la Shell:

    • Per a Bash, obre .bashrc o .bash_profile: 
      nano ~/.bashrc
    • Per a Zsh, obre .zshrc: 
      nano ~/.zshrc

  3. Afegir REXX a PATH:

    • Afegeix la següent línia al fitxer: 
      export PATH=$PATH:/usr/local/bin

  4. Aplicar Canvis:

    • Desa el fitxer i executa: 
      source ~/.bashrc
    • O per a Zsh: 
      source ~/.zshrc

Escriure el Teu Primer Script REXX

Ara que el teu entorn està configurat, escrivim un simple script REXX per assegurar-nos que tot funciona correctament.

  1. Crear un Nou Fitxer:

    • Obre el teu editor de text preferit.
    • Crea un nou fitxer anomenat hello.rexx.

  2. Escriure l'Script:

    • Afegeix el següent codi al fitxer: 
      /* Hello World in REXX */
say "Hello, World!"

  3. Desar el Fitxer.

  4. Executar l'Script:

    • Obre el Símbol del sistema (Windows) o Terminal (macOS/Linux).
    • Navega al directori on has desat hello.rexx.
    • Executa l'script escrivint: 
      rexx hello.rexx
    • Hauries de veure la sortida: 
      Hello, World!

Conclusió

Configurar l'entorn REXX implica instal·lar l'intèrpret i configurar el teu sistema per executar fàcilment scripts REXX. Seguint els passos descrits en aquesta secció, ara hauries de tenir un entorn REXX completament funcional. També has escrit i executat el teu primer script REXX, confirmant que tot està configurat correctament. A les següents seccions, aprofundirem en la sintaxi i les característiques del llenguatge REXX.
