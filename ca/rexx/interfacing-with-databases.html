Interfície amb bases de dades és una habilitat crucial per a qualsevol programador. Aquest tema et guiarà a través dels conceptes bàsics de la connexió i interacció amb bases de dades utilitzant REXX, progressant cap a tècniques més avançades i millors pràctiques.

Introducció a les Bases de Dades

Abans d'endinsar-nos en les interaccions específiques de REXX amb bases de dades, fem una breu introducció sobre què són les bases de dades i per què són importants.

  • Base de Dades: Una col·lecció estructurada de dades que es poden accedir, gestionar i actualitzar fàcilment.
  • DBMS (Sistema de Gestió de Bases de Dades): Programari que interactua amb els usuaris finals, aplicacions i la pròpia base de dades per capturar i analitzar dades.

Exemples comuns de DBMS inclouen:

  • MySQL
  • PostgreSQL
  • SQLite
  • IBM Db2

Configuració de l'Entorn

Per interfície amb una base de dades utilitzant REXX, necessites:

  1. Un intèrpret de REXX.
  2. Una base de dades a la qual connectar-se.
  3. Els controladors o API adequats per a la base de dades.

Exemple: Configuració d'una Base de Dades MySQL

  1. Instal·lar MySQL: Segueix les instruccions al lloc web de MySQL per descarregar i instal·lar MySQL.
  2. Crear una Base de Dades: Utilitza la línia de comandes de MySQL o una eina GUI com MySQL Workbench per crear una base de dades.
CREATE DATABASE example_db;
USE example_db;
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

Connexió a una Base de Dades

Per connectar-se a una base de dades des de REXX, normalment s'utilitza una API o una biblioteca que proporciona les funcions necessàries.

Exemple: Utilitzant ODBC amb REXX

ODBC (Open Database Connectivity) és una API estàndard per accedir a sistemes de gestió de bases de dades.

  1. Instal·lar el Controlador ODBC: Assegura't de tenir instal·lat el controlador ODBC per a la teva base de dades.
  2. Configurar la Font de Dades ODBC: Configura un DSN (Nom de la Font de Dades) per a la teva base de dades.

Exemple de Codi REXX

/* Script REXX per connectar-se a una base de dades MySQL utilitzant ODBC */
dsn = "example_db"
user = "your_username"
password = "your_password"

call RxFuncAdd 'SQLLoadFuncs', 'rexxsql', 'SQLLoadFuncs'
call SQLLoadFuncs

call SQLConnect dsn, user, password
if SQLCODE <> 0 then do
    say "Connexió fallida: " SQLERRMSG
    exit
end

say "Connectat a la base de dades amb èxit!"
call SQLDisconnect
  • RxFuncAdd: Registra una funció externa.
  • SQLLoadFuncs: Carrega les funcions SQL.
  • SQLConnect: Connecta a la base de dades utilitzant el DSN, nom d'usuari i contrasenya.
  • SQLDisconnect: Desconnecta de la base de dades.

Execució de Consultes SQL

Un cop connectat, pots executar consultes SQL per interactuar amb la base de dades.

Exemple: Inserir Dades

query = "INSERT INTO users (name, email) VALUES ('John Doe', 'john.doe@example.com')"
call SQLExecDirect query
if SQLCODE <> 0 then do
    say "Inserció fallida: " SQLERRMSG
    exit
end

say "Dades inserides amb èxit!"

Exemple: Recuperar Dades

query = "SELECT * FROM users"
call SQLExecDirect query
if SQLCODE <> 0 then do
    say "Consulta fallida: " SQLERRMSG
    exit
end

do while SQLFetch() = 0
    say "ID:" SQLData(1) "Nom:" SQLData(2) "Correu electrònic:" SQLData(3)
end
  • SQLExecDirect: Executa una instrucció SQL directament.
  • SQLFetch: Recupera la següent fila d'un resultat de consulta.
  • SQLData: Recupera dades de la fila actual.

Gestió d'Errors

La gestió adequada d'errors és essencial per a interaccions robustes amb bases de dades.

Exemple: Gestió d'Errors en REXX

query = "SELECT * FROM non_existing_table"
call SQLExecDirect query
if SQLCODE <> 0 then do
    say "Error: " SQLERRMSG
    exit
end
  • SQLCODE: Conté el codi de retorn de l'última operació SQL.
  • SQLERRMSG: Conté el missatge d'error si es produeix un error.

Tècniques Avançades

Utilitzant Sentències Preparades

Les sentències preparades poden millorar el rendiment i la seguretat.

query = "INSERT INTO users (name, email) VALUES (?, ?)"
call SQLPrepare query
call SQLBindParameter 1, 'John Doe'
call SQLBindParameter 2, 'john.doe@example.com'
call SQLExecute
if SQLCODE <> 0 then do
    say "Inserció fallida: " SQLERRMSG
    exit
end

say "Dades inserides amb èxit amb sentència preparada!"

Transaccions

Les transaccions asseguren la integritat de les dades agrupant múltiples operacions en una sola unitat de treball.

call SQLBeginTransaction
query1 = "UPDATE users SET email = 'new.email@example.com' WHERE id = 1"
query2 = "DELETE FROM users WHERE id = 2"

call SQLExecDirect query1
if SQLCODE <> 0 then do
    call SQLRollback
    say "Transacció fallida: " SQLERRMSG
    exit
end

call SQLExecDirect query2
if SQLCODE <> 0 then do
    call SQLRollback
    say "Transacció fallida: " SQLERRMSG
    exit
end

call SQLCommit
say "Transacció completada amb èxit!"
  • SQLBeginTransaction: Comença una nova transacció.
  • SQLCommit: Confirma la transacció actual.
  • SQLRollback: Desfà la transacció actual.

Resum

Interfície amb bases de dades en REXX implica:

  • Configurar l'entorn.
  • Connectar-se a la base de dades.
  • Executar consultes SQL.
  • Gestionar errors.
  • Utilitzar tècniques avançades com sentències preparades i transaccions.

Dominant aquestes habilitats, pots gestionar i manipular dades de manera eficient dins dels teus programes REXX, fent les teves aplicacions més potents i versàtils.
