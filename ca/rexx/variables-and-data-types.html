Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en REXX. Entendre aquests conceptes és crucial per escriure programes REXX efectius i eficients. Començarem amb els conceptes bàsics i gradualment passarem a temes més avançats.

Variables en REXX

Què és una Variable?

  • Una variable en REXX és un nom simbòlic associat amb un valor.
  • Les variables s'utilitzen per emmagatzemar dades que es poden manipular i recuperar al llarg del programa.

Declaració de Variables

  • En REXX, les variables no necessiten ser declarades explícitament abans d'usar-les.
  • Una variable es crea la primera vegada que se li assigna un valor.
/* Exemple d'assignació de variables */
name = "John Doe"
age = 30

Regles de Nomenclatura de Variables

  • Els noms de les variables han de començar amb una lletra (A-Z, a-z) o un caràcter especial (com . o _).
  • Els caràcters següents poden ser lletres, dígits (0-9) o caràcters especials.
  • Els noms de les variables no distingeixen entre majúscules i minúscules.
/* Noms de variables vàlids */
firstName = "Alice"
_lastName = "Smith"
age2 = 25

/* Noms de variables invàlids */
// 2ndName = "Bob"  // Comença amb un dígit
// @age = 30        // Comença amb un caràcter invàlid

Assignació de Valors a Variables

  • Els valors s'assignen a les variables utilitzant l'operador =.
  • REXX determina automàticament el tipus del valor assignat.
/* Assignació de diferents tipus de valors */
name = "Jane Doe"
age = 28
height = 5.7
isStudent = "Yes"

Tipus de Dades en REXX

Visió General dels Tipus de Dades

  • REXX és un llenguatge de tipus dinàmic, el que significa que el tipus d'una variable es determina en temps d'execució.
  • Els principals tipus de dades en REXX són:
    • Cadenes de text
    • Nombres (enters i de punt flotant)
    • Valors booleans (representats com cadenes "0" o "1")

Cadenes de Text

  • Les cadenes de text són seqüències de caràcters tancades entre cometes dobles (").
/* Exemple de variables de cadena de text */
greeting = "Hello, World!"
name = "Alice"

Nombres

  • Els nombres en REXX poden ser enters o de punt flotant.
  • REXX gestiona automàticament les operacions numèriques.
/* Exemple de variables numèriques */
integerNumber = 42
floatingNumber = 3.14

/* Realització d'operacions aritmètiques */
sum = integerNumber + floatingNumber

Valors Booleans

  • REXX no té un tipus booleà distint. En canvi, utilitza cadenes "0" (fals) i "1" (cert).
/* Exemple de valors booleans */
isTrue = "1"
isFalse = "0"

Conversió de Tipus

  • REXX converteix automàticament entre cadenes i nombres segons sigui necessari.
  • La conversió explícita es pot fer utilitzant funcions integrades.
/* Conversió automàtica de tipus */
numString = "123"
number = numString + 1  /* Converteix "123" a 123 i suma 1 */

/* Conversió explícita de tipus */
stringNumber = "456"
convertedNumber = VALUE(stringNumber)

Temes Avançats

Variables Compostes

  • Les variables compostes (també conegudes com arrays o arrays associatius) permeten estructures de dades més complexes.
  • S'indexen per cadenes o nombres.
/* Exemple de variables compostes */
person.1 = "Alice"
person.2 = "Bob"
person.3 = "Charlie"

say person.1  /* Mostra: Alice */

Abast de les Variables

  • Les variables en REXX poden ser locals o globals.
  • Les variables locals es defineixen dins d'un procediment i no són accessibles fora d'aquest.
  • Les variables globals són accessibles a tot el programa.
/* Exemple d'abast de les variables */
globalVar = "I am global"

procedure
  localVar = "I am local"
  say localVar
return

say globalVar  /* Mostra: I am global */
// say localVar  /* Error: localVar no està definida fora del procediment */

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades en REXX. Hem après com declarar i assignar valors a les variables, els diferents tipus de dades disponibles i com realitzar conversions de tipus. També hem tocat temes més avançats com les variables compostes i l'abast de les variables. Entendre aquests conceptes és essencial per escriure programes REXX efectius.
