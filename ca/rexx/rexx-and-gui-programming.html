Introducció a REXX i la Programació GUI

REXX (Restructured Extended Executor) és un llenguatge de script versàtil que es pot utilitzar tant per a tasques de script senzilles com per al desenvolupament d'aplicacions complexes. Una de les aplicacions avançades de REXX és la programació GUI (Interfície Gràfica d'Usuari), que permet als desenvolupadors crear interfícies fàcils d'usar per a les seves aplicacions.

En aquesta secció, explorarem com utilitzar REXX per a la programació GUI, cobrint els conceptes bàsics i avançant cap a conceptes més avançats.

Conceptes Clau en la Programació GUI amb REXX

  • Biblioteques GUI: Comprendre les biblioteques disponibles per al desenvolupament GUI en REXX.
  • Programació Basada en Esdeveniments: Gestionar les interaccions dels usuaris a través d'esdeveniments.
  • Widgets: Utilitzar components GUI com botons, camps de text i etiquetes.
  • Disposicions: Organitzar widgets en una finestra.
  • Callbacks: Enllaçar les accions dels usuaris a funcions.

Configuració del Teu Entorn

Abans d'endinsar-nos en la programació GUI amb REXX, assegura't de tenir les eines i biblioteques necessàries instal·lades. Una biblioteca popular per a la programació GUI en REXX és ooRexx amb el pont BSF4ooRexx, que permet a REXX interactuar amb biblioteques Java.

Instal·lació de ooRexx i BSF4ooRexx

  1. Descarrega i instal·la ooRexx des del lloc web oficial.
  2. Descarrega i instal·la BSF4ooRexx des del lloc web oficial.
  3. Assegura't que Java estigui instal·lat al teu sistema, ja que BSF4ooRexx depèn de les biblioteques Java.

Aplicació GUI Bàsica

Comencem amb una aplicació GUI senzilla que crea una finestra amb un botó.

Exemple: GUI Hola Món

::requires BSF.CLS

bsf.loadLibrary("java.awt.*")
bsf.loadLibrary("javax.swing.*")

-- Crear un JFrame (finestra)
frame = .bsf~new("javax.swing.JFrame", "Hello World GUI")

-- Crear un JButton (botó)
button = .bsf~new("javax.swing.JButton", "Click Me")

-- Afegir un action listener al botó
button~addActionListener(.bsf~new("java.awt.event.ActionListener", "actionPerformed", "say 'Button Clicked!'"))

-- Afegir el botó al frame
frame~getContentPane()~add(button)

-- Establir l'operació de tancament per defecte
frame~setDefaultCloseOperation(javax.swing.JFrame~EXIT_ON_CLOSE)

-- Establir la mida de la finestra
frame~setSize(300, 200)

-- Fer visible la finestra
frame~setVisible(.true)

Explicació

  • Carregar Biblioteques: Carreguem les biblioteques Java necessàries per als components GUI.
  • Crear JFrame: Es crea una finestra utilitzant javax.swing.JFrame.
  • Crear JButton: Es crea un botó utilitzant javax.swing.JButton.
  • Afegir Action Listener: S'afegeix un action listener al botó per gestionar els esdeveniments de clic.
  • Afegir Botó al Frame: El botó s'afegeix a la finestra.
  • Establir Operació de Tancament: S'estableix l'operació de tancament per defecte per sortir de l'aplicació quan es tanqui la finestra.
  • Establir Mida i Visibilitat: S'estableix la mida de la finestra i es fa visible.

Conceptes Avançats de GUI

Un cop et sentis còmode amb les aplicacions GUI bàsiques, pots explorar conceptes més avançats.

Gestió d'Esdeveniments

La gestió d'esdeveniments és crucial per a aplicacions interactives. Pots gestionar diversos esdeveniments com clics de ratolí, premses de tecles i esdeveniments de finestra.

Exemple: Gestionar Esdeveniments de Clic de Ratolí

::requires BSF.CLS

bsf.loadLibrary("java.awt.*")
bsf.loadLibrary("javax.swing.*")

frame = .bsf~new("javax.swing.JFrame", "Mouse Event Example")
label = .bsf~new("javax.swing.JLabel", "Click anywhere inside the window")

frame~getContentPane()~add(label)
frame~setSize(300, 200)
frame~setVisible(.true)

frame~addMouseListener(.bsf~new("java.awt.event.MouseAdapter", "mouseClicked", "say 'Mouse Clicked at (' || .bsf~this~getX() || ', ' || .bsf~this~getY() || ')'"))

frame~setDefaultCloseOperation(javax.swing.JFrame~EXIT_ON_CLOSE)

Explicació

  • Mouse Listener: S'afegeix un mouse listener al frame per gestionar els esdeveniments de clic de ratolí.
  • Mostrar Coordenades: Les coordenades del clic de ratolí es mostren a la consola.
