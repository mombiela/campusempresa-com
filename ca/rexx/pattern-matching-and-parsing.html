Coincidència de Patrons i Anàlisi en REXX
Construccions de Bucle ►

La coincidència de patrons i l'anàlisi són habilitats essencials per a qualsevol programador que treballi amb dades de text. En REXX, aquestes tasques són senzilles i potents, gràcies a les seves funcions integrades i la seva sintaxi flexible. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades en la coincidència de patrons i l'anàlisi utilitzant REXX.

Introducció a la Coincidència de Patrons

La coincidència de patrons implica comprovar si una cadena compleix un patró especificat. En REXX, això sovint es fa utilitzant la funció POS i la instrucció PARSE.

Conceptes Clau

  • Funció POS: Troba la posició d'una subcadena dins d'una cadena.
  • Instrucció PARSE: Extreu parts d'una cadena basant-se en patrons.

Funció POS

La funció POS retorna la posició de la primera ocurrència d'una subcadena dins d'una cadena. Si la subcadena no es troba, retorna 0.

Sintaxi

position = POS(substring, string [, start])

Exemple

string = "Hello, World!"
substring = "World"
position = POS(substring, string)
SAY "Position of 'World' in the string is:" position

Sortida:

Position of 'World' in the string is: 8

Anàlisi Bàsica amb la Instrucció PARSE

La instrucció PARSE s'utilitza per dividir cadenes en variables basant-se en patrons o delimitadors.

Conceptes Clau

  • PARSE VAR: Analitza una variable.
  • PARSE VALUE: Analitza un valor o expressió.
  • PARSE ARG: Analitza arguments passats a un script REXX.

PARSE VAR

La instrucció PARSE VAR divideix una variable en múltiples parts.

Sintaxi

PARSE VAR variable part1 part2 part3 ...

Exemple

fullName = "John Doe"
PARSE VAR fullName firstName lastName
SAY "First Name:" firstName
SAY "Last Name:" lastName

Sortida:

First Name: John
Last Name: Doe

Tècniques Avançades d'Anàlisi

L'anàlisi avançada implica utilitzar patrons i plantilles per extreure estructures de dades més complexes.

Conceptes Clau

  • Plantilles: Defineixen l'estructura de les dades a analitzar.
  • Coincidència de Patrons: Utilitza patrons per extreure parts específiques de les dades.

Utilitzant Plantilles

Les plantilles es poden utilitzar amb la instrucció PARSE per especificar posicions exactes o delimitadors.

Exemple

address = "123 Main St, Springfield, IL 62704"
PARSE VAR address street ',' city ',' state zip
SAY "Street:" street
SAY "City:" city
SAY "State:" state
SAY "ZIP Code:" zip

Sortida:

Street: 123 Main St
City: Springfield
State: IL
ZIP Code: 62704

Coincidència de Patrons amb PARSE

Pots utilitzar patrons per coincidir parts específiques d'una cadena.

Exemple

data = "Name: John Doe, Age: 30, Occupation: Developer"
PARSE VAR data "Name: " name ", Age: " age ", Occupation: " occupation
SAY "Name:" name
SAY "Age:" age
SAY "Occupation:" occupation

Sortida:

Name: John Doe
Age: 30
Occupation: Developer

Conclusió

La coincidència de patrons i l'anàlisi són eines potents en REXX que et permeten manipular i extreure informació de cadenes de manera eficient. Dominant la funció POS i la instrucció PARSE, pots gestionar una àmplia gamma de tasques de processament de text, des de cerques simples de subcadenes fins a extraccions complexes de dades.

Resum

  • Funció POS: Utilitzada per trobar la posició d'una subcadena.
  • Instrucció PARSE: Utilitzada per dividir cadenes en variables basant-se en patrons o delimitadors.
  • Plantilles i Patrons: Permeten tècniques avançades d'anàlisi per extreure estructures de dades complexes.

Amb aquestes habilitats, estàs ben equipat per gestionar dades de text en REXX, fent els teus scripts més potents i versàtils.
