Introducció

Les estructures de control són fonamentals en qualsevol llenguatge de programació, ja que permeten controlar el flux d'execució basat en certes condicions. En REXX, les estructures de control principals inclouen IF, DO i SELECT. Aquesta secció et guiarà a través d'aquestes estructures des d'un nivell principiant fins a un nivell avançat.

Instruccions IF

La instrucció IF en REXX s'utilitza per executar un bloc de codi condicionalment.

Instrucció IF bàsica

La forma més simple d'una instrucció IF comprova una sola condició.

/* Instrucció IF bàsica */
age = 20
if age >= 18 then
  say "You are an adult."

Instrucció IF-ELSE

Pots estendre la instrucció IF amb una clàusula ELSE per manejar condicions alternatives.

/* Instrucció IF-ELSE */
age = 16
if age >= 18 then
  say "You are an adult."
else
  say "You are a minor."

Instruccions IF imbricades

Les instruccions IF es poden imbricar per comprovar múltiples condicions.

/* Instruccions IF imbricades */
age = 16
if age >= 18 then
  say "You are an adult."
else
  if age >= 13 then
    say "You are a teenager."
  else
    say "You are a child."

Bucles DO

El bucle DO en REXX s'utilitza per repetir un bloc de codi múltiples vegades.

Bucle DO bàsic

Un simple bucle DO s'executa un nombre fix de vegades.

/* Bucle DO bàsic */
do i = 1 to 5
  say "Iteration" i
end

Bucle DO WHILE

Un bucle DO WHILE s'executa mentre una condició sigui certa.

/* Bucle DO WHILE */
i = 1
do while i <= 5
  say "Iteration" i
  i = i + 1
end

Bucle DO UNTIL

Un bucle DO UNTIL s'executa fins que una condició esdevingui certa.

/* Bucle DO UNTIL */
i = 1
do until i > 5
  say "Iteration" i
  i = i + 1
end

Bucle DO FOREVER

Un bucle DO FOREVER s'executa indefinidament fins que es terminin explícitament.

/* Bucle DO FOREVER */
i = 1
do forever
  say "Iteration" i
  i = i + 1
  if i > 5 then leave
end

SELECT Case

La instrucció SELECT en REXX s'utilitza per manejar múltiples condicions de manera més eficient que les instruccions IF imbricades.

Instrucció SELECT bàsica

Una instrucció SELECT avalua múltiples condicions i executa el bloc de codi corresponent.

/* Instrucció SELECT bàsica */
day = "Monday"
select
  when day = "Monday" then
    say "Start of the work week."
  when day = "Friday" then
    say "End of the work week."
  otherwise
    say "Midweek day."
end

Instruccions SELECT imbricades

Les instruccions SELECT també es poden imbricar per a una presa de decisions més complexa.

/* Instruccions SELECT imbricades */
day = "Monday"
time = "Morning"
select
  when day = "Monday" then
    select
      when time = "Morning" then
        say "Start of the work week in the morning."
      otherwise
        say "Start of the work week in the afternoon."
    end
  when day = "Friday" then
    say "End of the work week."
  otherwise
    say "Midweek day."
end

Conclusió

Les estructures de control com IF, DO i SELECT són essencials per gestionar el flux dels teus programes REXX. Entendre aquests constructes et permetrà escriure codi més eficient i llegible. Practica aquests exemples i intenta crear els teus propis escenaris per aprofundir en la teva comprensió.
