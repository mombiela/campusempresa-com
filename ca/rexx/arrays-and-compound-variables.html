Introducció

En aquesta secció, explorarem les arrays i les variables compostes en REXX. Les arrays i les variables compostes són essencials per gestionar col·leccions de dades de manera eficient. Cobrirem els conceptes bàsics de les arrays, com crear-les i manipular-les, i aprofundirem en temes més avançats com les arrays multidimensionals i les arrays dinàmiques.

Conceptes clau

  • Arrays: Una col·lecció d'elements que es poden accedir utilitzant un índex.
  • Variables compostes: Variables que poden contenir múltiples valors, sovint utilitzades per simular arrays en REXX.
  • Indexació: El mètode d'accedir a elements individuals dins d'una array o variable composta.

Arrays simples en REXX

REXX no té suport integrat per arrays tradicionals. En canvi, utilitza variables compostes per simular arrays.

Creació i ús de variables compostes

Les variables compostes en REXX es creen utilitzant un nom base seguit d'un o més subíndexs.

/* Array simple utilitzant variables compostes */
array.1 = "Poma"
array.2 = "Plàtan"
array.3 = "Cirera"

say array.1  /* Sortida: Poma */
say array.2  /* Sortida: Plàtan */
say array.3  /* Sortida: Cirera */

Iteració a través de les arrays

Pots iterar a través de les arrays utilitzant un bucle DO.

/* Iterar a través de l'array */
do i = 1 to 3
    say array.i
end

Arrays multidimensionals

REXX et permet crear arrays multidimensionals utilitzant múltiples subíndexs.

Exemple d'una array bidimensional

/* Array bidimensional */
matrix.1.1 = "A1"
matrix.1.2 = "A2"
matrix.2.1 = "B1"
matrix.2.2 = "B2"

say matrix.1.1  /* Sortida: A1 */
say matrix.2.2  /* Sortida: B2 */

Iteració a través d'arrays multidimensionals

/* Iterar a través d'una array bidimensional */
do i = 1 to 2
    do j = 1 to 2
        say matrix.i.j
    end
end

Arrays dinàmiques

REXX et permet afegir elements dinàmicament a les arrays.

Afegir elements dinàmicament

/* Afegir elements dinàmicament */
array.4 = "Dàtil"
array.5 = "Saüc"

do i = 1 to 5
    say array.i
end

Exemple pràctic: Emmagatzemar i recuperar dades

Creem un programa senzill que emmagatzemi i recuperi noms d'estudiants i les seves notes.

/* Programa per emmagatzemar i recuperar notes d'estudiants */
students.1 = "Alice"
grades.1 = 85
students.2 = "Bob"
grades.2 = 90
students.3 = "Charlie"
grades.3 = 78

/* Recuperar i mostrar noms d'estudiants i notes */
do i = 1 to 3
    say students.i "ha obtingut" grades.i
end

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les arrays i les variables compostes en REXX. Hem après com crear i manipular arrays simples i multidimensionals utilitzant variables compostes. També hem explorat com afegir elements dinàmicament a les arrays i iterar a través d'elles. Entendre aquests conceptes és crucial per gestionar col·leccions de dades de manera eficient en programes REXX. Practica aquests exemples i intenta crear les teves pròpies arrays per sentir-te més còmode amb aquests conceptes.
