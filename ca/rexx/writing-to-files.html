Escriure a fitxers és una habilitat fonamental en la programació REXX, permetent-te guardar dades, registres i altra informació per a un ús posterior. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades per escriure a fitxers en REXX.

Introducció a l'escriptura de fitxers

Abans de submergir-nos en els detalls, és essencial entendre els conceptes bàsics de la gestió de fitxers en REXX.

  • Gestió de fitxers: El procés de crear, escriure, llegir i tancar fitxers.
  • Fluxos: En REXX, els fitxers es tracten com a fluxos de dades que es poden llegir o escriure.

Escriptura bàsica de fitxers

Obrir un fitxer per escriure

Per escriure a un fitxer, primer cal obrir-lo. En REXX, això es fa utilitzant la comanda OPEN.

/* Obrir un fitxer per escriure */
file = "example.txt"
call stream file, "w", "open"
  • file: El nom del fitxer al qual vols escriure.
  • "w": El mode en què s'ha d'obrir el fitxer. "w" significa mode d'escriptura.
  • "open": L'acció a realitzar sobre el fitxer.

Escriure dades a un fitxer

Un cop el fitxer està obert, pots escriure-hi dades utilitzant la funció LINEOUT.

/* Escriure una línia al fitxer */
call lineout file, "Hello, World!"
  • file: El gestor del fitxer.
  • "Hello, World!": La cadena a escriure al fitxer.

Tancar el fitxer

Després d'escriure al fitxer, és crucial tancar-lo per assegurar-se que totes les dades es guarden correctament.

/* Tancar el fitxer */
call stream file, "c", "close"
  • "c": El mode per tancar el fitxer.
  • "close": L'acció a realitzar.

Exemple complet

Aquí tens un exemple complet que obre un fitxer, hi escriu una línia i després el tanca.

/* Exemple complet d'escriptura a un fitxer */
file = "example.txt"
call stream file, "w", "open"
call lineout file, "Hello, World!"
call stream file, "c", "close"

Escriptura avançada de fitxers

Afegir dades a un fitxer

Per afegir dades a un fitxer existent sense sobreescriure'l, utilitza el mode "a".

/* Obrir un fitxer per afegir */
file = "example.txt"
call stream file, "a", "open"
call lineout file, "Appending this line."
call stream file, "c", "close"

Escriure múltiples línies

Pots escriure múltiples línies cridant LINEOUT diverses vegades.

/* Escriure múltiples línies a un fitxer */
file = "example.txt"
call stream file, "w", "open"
call lineout file, "First line."
call lineout file, "Second line."
call lineout file, "Third line."
call stream file, "c", "close"

Gestió d'errors

És important gestionar els errors que poden ocórrer durant les operacions amb fitxers. Pots comprovar el codi de retorn de la funció STREAM per assegurar-te que l'operació ha estat exitosa.

/* Exemple de gestió d'errors */
file = "example.txt"
rc = stream(file, "w", "open")
if rc <> 0 then do
    say "Error opening file"
    exit(1)
end

rc = lineout(file, "Hello, World!")
if rc <> 0 then do
    say "Error writing to file"
    call stream file, "c", "close"
    exit(1)
end

call stream file, "c", "close"

Resum

Escriure a fitxers en REXX implica tres passos principals: obrir el fitxer, escriure-hi dades i tancar el fitxer. Dominant aquests passos, pots gestionar eficientment les operacions amb fitxers en els teus programes REXX. Recorda gestionar els errors de manera adequada per assegurar que els teus programes siguin robustos i fiables.

  • Passos bàsics: Obrir, escriure, tancar.
  • Modes: "w" per escriure, "a" per afegir.
  • Gestió d'errors: Comprova els codis de retorn per gestionar els errors.

Amb aquestes habilitats, ara pots escriure dades a fitxers en REXX amb confiança, des de línies de text simples fins a estructures de dades més complexes.
