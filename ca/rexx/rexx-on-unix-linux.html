Introducció a REXX

  • Què és REXX?
    • REXX (REstructured eXtended eXecutor) és un llenguatge de script desenvolupat per IBM.
    • Conegut per la seva facilitat d'ús i llegibilitat.
  • Història i Evolució
    • Va originar-se als anys 70 per a mainframes d'IBM.
    • Ha evolucionat per donar suport a múltiples plataformes, incloent Unix/Linux.

Configuració de REXX a Unix/Linux

  • Instal·lació
    • La majoria de distribucions Unix/Linux inclouen intèrprets REXX com Regina REXX.
    • Exemple de comanda d'instal·lació per a Regina REXX: 
      sudo apt-get install regina-rexx
  • Verificació de la Instal·lació
    • Comprova la instal·lació executant: 
      rexx -v

Sintaxi i Estructura Bàsica

  • Programa Hello World

    • Exemple: 
      /* Hello World en REXX */
say "Hello, World!"
    • Explicació:
      • /* ... */ s'utilitza per a comentaris.
      • say és una funció incorporada per imprimir text a la pantalla.

  • Variables i Tipus de Dades

    • Les variables són de tipus dinàmic.
    • Exemple: 
      name = "Alice"
age = 30
say "Nom:" name
say "Edat:" age

Estructures de Control

  • Declaracions Condicionals

    • Declaracions if, else i select.
    • Exemple: 
      age = 20
if age >= 18 then
  say "Ets un adult."
else
  say "Ets un menor."

  • Bucles

    • Bucles do.
    • Exemple: 
      do i = 1 to 5
  say "Número:" i
end

Funcions i Procediments

  • Funcions Incorporades

    • Funcions comunes com length, substr, left, right.
    • Exemple: 
      text = "Hello, World!"
say "Longitud:" length(text)
say "Subcadena:" substr(text, 1, 5)

  • Funcions Definides per l'Usuari

    • Exemple: 
      say "La suma de 5 i 3 és:" sum(5, 3)

sum: procedure
  parse arg a, b
  return a + b

Gestió de Fitxers

  • Llegir d'un Fitxer

    • Exemple: 
      file = "example.txt"
call linein file
do while lines(file) > 0
  line = linein(file)
  say line
end
call lineout file

  • Escriure a un Fitxer

    • Exemple: 
      file = "output.txt"
call lineout file, "Aquesta és una línia de text."
call lineout file

Temes Avançats

  • Interfície amb Comandes Unix/Linux

    • Utilitzant address per executar comandes de shell.
    • Exemple: 
      address system "ls -l"

  • Gestió d'Errors

    • Utilitzant signal per gestionar errors.
    • Exemple: 
      signal on error
call nonexistent_function
exit

error:
  say "S'ha produït un error."

  • Expressions Regulars

    • Utilitzant expressions regulars per a la coincidència de patrons.
    • Exemple: 
      text = "The quick brown fox"
if text \= ".*quick.*" then
  say "Coincidència trobada!"

Conclusió

  • Resum
    • REXX és un llenguatge de script versàtil adequat per a diverses tasques a Unix/Linux.
    • Els conceptes clau inclouen sintaxi bàsica, estructures de control, gestió de fitxers i interfície avançada amb comandes del sistema.
  • Pròxims Passos
