Introducció

La manipulació de cadenes és una habilitat fonamental en la programació REXX. Aquest tema cobrirà diverses tècniques i funcions per gestionar cadenes, des d'operacions bàsiques fins a manipulacions avançades.

Operacions Bàsiques amb Cadenes

Concatenació

  • Concepte: Combinar dues o més cadenes en una sola.
  • Exemple: 
    string1 = "Hello"
string2 = "World"
combinedString = string1 || " " || string2
say combinedString  /* Output: Hello World */

Extracció de Subcadena

  • Concepte: Extreure una part d'una cadena.
  • Exemple: 
    fullString = "Hello World"
subString = substr(fullString, 1, 5)
say subString  /* Output: Hello */

Longitud de la Cadena

  • Concepte: Trobar la longitud d'una cadena.
  • Exemple: 
    myString = "Hello"
length = length(myString)
say length  /* Output: 5 */

Operacions Intermèdies amb Cadenes

Retallar Cadenes

  • Concepte: Eliminar espais inicials i/o finals.
  • Exemple: 
    myString = "  Hello World  "
trimmedString = strip(myString)
say trimmedString  /* Output: Hello World */

Canviar Majúscules i Minúscules

  • Concepte: Convertir cadenes a majúscules o minúscules.
  • Exemple: 
    myString = "Hello World"
upperCaseString = translate(myString)
lowerCaseString = translate(myString, "abcdefghijklmnopqrstuvwxyz", "ABCDEFGHIJKLMNOPQRSTUVWXYZ")
say upperCaseString  /* Output: HELLO WORLD */
say lowerCaseString  /* Output: hello world */

Reemplaçar Subcadenes

  • Concepte: Reemplaçar parts d'una cadena amb una altra cadena.
  • Exemple: 
    myString = "Hello World"
newString = overlay("REXX", myString, 7)
say newString  /* Output: Hello REXX */

Operacions Avançades amb Cadenes

Coincidència de Patrons i Substitució

  • Concepte: Trobar i reemplaçar patrons dins de cadenes.
  • Exemple: 
    myString = "The quick brown fox jumps over the lazy dog"
newString = changestr("fox", "cat", myString)
say newString  /* Output: The quick brown cat jumps over the lazy dog */

Dividir Cadenes

  • Concepte: Dividir una cadena en múltiples parts basant-se en un delimitador.
  • Exemple: 
    myString = "apple,banana,cherry"
parse var myString fruit1 "," fruit2 "," fruit3
say fruit1  /* Output: apple */
say fruit2  /* Output: banana */
say fruit3  /* Output: cherry */

Formatació de Cadenes

  • Concepte: Donar format a cadenes segons una disposició o estructura específica.
  • Exemple: 
    name = "John"
age = 30
formattedString = "Name: " || name || ", Age: " || age
say formattedString  /* Output: Name: John, Age: 30 */

Resum

La manipulació de cadenes en REXX implica una varietat d'operacions, des de la concatenació bàsica i l'extracció de subcadenes fins a la coincidència de patrons avançada i la formatació de cadenes. Dominar aquestes tècniques és essencial per a una programació efectiva en REXX. Practiqueu aquests conceptes amb diferents exemples per esdevenir proficient en la gestió de cadenes en REXX.
