Els arrays associatius, també coneguts com a variables de tija en REXX, són una característica poderosa que et permet crear i manipular col·leccions de dades indexades per cadenes. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat dels arrays associatius en REXX.

Introducció als Arrays Associatius

Els arrays associatius en REXX es implementen utilitzant variables de tija. Una variable de tija és una col·lecció de variables que comparteixen un prefix comú, conegut com el nom de la tija. Cada element de la col·lecció s'accedeix utilitzant un índex únic.

Conceptes Clau

  • Variable de tija: Una variable que actua com a prefix per a una col·lecció de variables relacionades.
  • Índex: Una cadena o número utilitzat per accedir a un element específic en la variable de tija.
  • Element: Un element individual en l'array associatiu, accedit utilitzant el nom de la tija i un índex.

Sintaxi Bàsica

Per definir i utilitzar una variable de tija en REXX, segueixes aquesta sintaxi:

stem.index = value

Aquí, stem és el nom de la tija, index és l'identificador únic per a l'element, i value és la dada que vols emmagatzemar.

Exemple

/* Defineix una variable de tija */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Accedeix als elements */
say students.1  /* Sortida: Alice */
say students.2  /* Sortida: Bob */
say students.3  /* Sortida: Charlie */

Ús Avançat dels Arrays Associatius

A mesura que et familiaritzes amb les variables de tija, pots començar a utilitzar característiques i tècniques més avançades.

Indexació Dinàmica

Pots utilitzar valors dinàmics per a l'índex, permetent estructures de dades més flexibles i poderoses.

/* Utilitzant indexació dinàmica */
do i = 1 to 3
    students.i = "Student" || i
end

/* Accedeix als elements dinàmicament */
do i = 1 to 3
    say students.i  /* Sortida: Student1, Student2, Student3 */
end

Arrays Associatius Multidimensionals

REXX et permet crear arrays associatius multidimensionals utilitzant múltiples índexs.

/* Defineix un array associatiu multidimensional */
grades.Alice.Math = 95
grades.Alice.English = 88
grades.Bob.Math = 78
grades.Bob.English = 85

/* Accedeix als elements */
say grades.Alice.Math     /* Sortida: 95 */
say grades.Bob.English    /* Sortida: 85 */

Iterar Sobre Arrays Associatius

Pots iterar sobre tots els elements en una variable de tija utilitzant el bucle do i el prefix stem..

/* Defineix una variable de tija */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Itera sobre els elements */
i = 1
do while students.i \= ""
    say students.i
    i = i + 1
end

Comprovar l'Existència

Pots comprovar si un element existeix en una variable de tija utilitzant la funció symbol.

/* Defineix una variable de tija */
students.1 = "Alice"

/* Comprova l'existència */
if symbol("students.2") = "VAR" then
    say "Student 2 exists"
else
    say "Student 2 does not exist"

Conclusió

Els arrays associatius, o variables de tija, en REXX proporcionen una manera flexible i poderosa de gestionar col·leccions de dades. Des de l'ús bàsic fins a tècniques avançades com la indexació dinàmica i els arrays multidimensionals, les variables de tija poden millorar enormement les teves capacitats de programació en REXX. Practica utilitzant aquestes característiques per convertir-te en un expert en la gestió d'estructures de dades complexes en REXX.
