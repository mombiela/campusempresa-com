En aquesta secció, explorarem com cridar programes externs des d'un script REXX. Aquesta és una característica poderosa que permet als scripts REXX interactuar amb altres programaris i realitzar una àmplia gamma de tasques més enllà de les capacitats del mateix REXX.

Introducció

Cridar programes externs pot ser útil per a:

  • Automatitzar tasques del sistema.
  • Integrar scripts REXX amb altres programaris.
  • Estendre la funcionalitat de REXX aprofitant altres eines.

Sintaxi Bàsica

En REXX, pots cridar un programa extern utilitzant la instrucció ADDRESS. La instrucció ADDRESS et permet especificar l'entorn en el qual s'ha d'executar la comanda.

Exemple: Cridar una Comanda de Shell

Aquí tens un exemple senzill de com cridar una comanda de shell des d'un script REXX:

/* Script REXX per cridar una comanda de shell externa */
ADDRESS SYSTEM 'echo Hello, World!'

En aquest exemple:

  • ADDRESS SYSTEM especifica que la comanda s'ha d'executar en el shell del sistema.
  • 'echo Hello, World!' és la comanda que s'està executant.

Capturar la Sortida

Pots capturar la sortida d'una comanda externa utilitzant la funció LINEIN. Això és útil quan necessites processar la sortida de la comanda dins del teu script REXX.

Exemple: Capturar la Sortida de la Comanda

/* Script REXX per capturar la sortida d'una comanda de shell */
ADDRESS SYSTEM 'dir > temp.txt'
output = LINEIN('temp.txt')
SAY output

En aquest exemple:

  • ADDRESS SYSTEM 'dir > temp.txt' executa la comanda dir i redirigeix la sortida a un fitxer anomenat temp.txt.
  • output = LINEIN('temp.txt') llegeix la primera línia de la sortida del fitxer.
  • SAY output imprimeix la sortida capturada.

Gestió d'Errors

Quan crides programes externs, és important gestionar els errors de manera adequada. Pots comprovar el codi de retorn de la comanda per determinar si s'ha executat correctament.

Exemple: Comprovar el Codi de Retorn

/* Script REXX per comprovar el codi de retorn d'una comanda de shell */
ADDRESS SYSTEM 'nonexistent_command'
if RC \= 0 then
  SAY 'La comanda ha fallat amb el codi de retorn:' RC
else
  SAY 'La comanda s'ha executat correctament'

En aquest exemple:

  • ADDRESS SYSTEM 'nonexistent_command' intenta executar una comanda que no existeix.
  • if RC \= 0 then comprova si el codi de retorn (RC) no és igual a 0, indicant un error.
  • SAY 'La comanda ha fallat amb el codi de retorn:' RC imprimeix un missatge d'error si la comanda ha fallat.

Tècniques Avançades

Per a casos d'ús més avançats, pot ser necessari interactuar amb el programa extern d'una manera més complexa, com passar arguments o gestionar múltiples línies de sortida.

Exemple: Passar Arguments

/* Script REXX per passar arguments a un programa extern */
arg1 = 'Hello'
arg2 = 'World'
ADDRESS SYSTEM 'echo' arg1 arg2

En aquest exemple:

  • arg1 i arg2 són variables que contenen els arguments a passar.
  • ADDRESS SYSTEM 'echo' arg1 arg2 executa la comanda echo amb els arguments.

Exemple: Gestionar Múltiples Línies de Sortida

/* Script REXX per gestionar múltiples línies de sortida */
ADDRESS SYSTEM 'dir > temp.txt'
do while LINES('temp.txt') > 0
  line = LINEIN('temp.txt')
  SAY line
end

En aquest exemple:

  • do while LINES('temp.txt') > 0 fa un bucle mentre hi hagi línies per llegir en el fitxer.
  • line = LINEIN('temp.txt') llegeix cada línia de la sortida.
  • SAY line imprimeix cada línia.

Conclusió

Cridar programes externs des de scripts REXX amplia les capacitats dels teus scripts permetent-los interactuar amb el sistema i altres programaris. Entenent la sintaxi bàsica, capturant la sortida, gestionant errors i utilitzant tècniques avançades, pots integrar efectivament comandes externes als teus scripts REXX.

Practica aquests conceptes amb diversos programes i comandes externes per esdevenir proficient en aprofitar aquesta poderosa característica de REXX.
