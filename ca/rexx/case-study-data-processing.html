Introducció

En aquest estudi de cas, explorarem com processar dades utilitzant REXX, un potent llenguatge de script. Cobrirem els següents conceptes clau:

  • Llegir dades de fitxers
  • Processar i manipular dades
  • Escriure dades a fitxers
  • Gestió d'errors

Al final d'aquest estudi de cas, tindràs una comprensió sòlida de com gestionar tasques de processament de dades en REXX.

Llegir Dades de Fitxers

Llegir dades de fitxers és una tasca fonamental en el processament de dades. En REXX, això es pot fer utilitzant la funció LINEIN.

Exemple

/* Llegir dades d'un fitxer */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: No es pot obrir el fitxer'
else
  do while lines(file) > 0
    line = linein(file)
    say 'Línia llegida:' line
  end
call close file
  • open(filename, 'r'): Obre el fitxer en mode de lectura.
  • lines(file): Retorna el nombre de línies que queden per llegir en el fitxer.
  • linein(file): Llegeix una línia del fitxer.
  • call close file: Tanca el fitxer.

Processar i Manipular Dades

Un cop les dades es llegeixen d'un fitxer, sovint cal processar-les o manipular-les. Això pot incloure tasques com filtrar, transformar o agregar dades.

Exemple

/* Processar dades: Convertir a majúscules i filtrar línies que continguin 'ERROR' */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: No es pot obrir el fitxer'
else
  do while lines(file) > 0
    line = linein(file)
    line = translate(line) /* Convertir a majúscules */
    if pos('ERROR', line) > 0 then
      say 'Línia filtrada:' line
  end
call close file
  • translate(line): Converteix la línia a majúscules.
  • pos('ERROR', line): Comprova si la cadena 'ERROR' és present a la línia.

Escriure Dades a Fitxers

Després de processar les dades, sovint cal escriure els resultats a un fitxer. Això es pot fer utilitzant la funció LINEOUT.

Exemple

/* Escriure dades processades a un nou fitxer */
inputFile = 'data.txt'
outputFile = 'processed_data.txt'
inFile = open(inputFile, 'r')
outFile = open(outputFile, 'w')
if inFile = '' | outFile = '' then
  say 'Error: No es poden obrir els fitxers'
else
  do while lines(inFile) > 0
    line = linein(inFile)
    line = translate(line) /* Convertir a majúscules */
    if pos('ERROR', line) > 0 then
      call lineout outFile, line
  end
call close inFile
call close outFile
  • open(outputFile, 'w'): Obre el fitxer en mode d'escriptura.
  • call lineout outFile, line: Escriu una línia al fitxer de sortida.

Gestió d'Errors

La gestió d'errors és crucial en el processament de dades per assegurar la robustesa del script. REXX proporciona mecanismes per gestionar errors de manera elegant.

Exemple

/* Gestió d'errors en el processament de dades */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: No es pot obrir el fitxer'
else
  do while lines(file) > 0
    line = linein(file)
    if rc \= 0 then
      do
        say 'Error en llegir la línia'
        leave
      end
    line = translate(line) /* Convertir a majúscules */
    if pos('ERROR', line) > 0 then
      say 'Línia filtrada:' line
  end
call close file
  • if rc \= 0: Comprova si el codi de retorn (rc) indica un error.

Conclusió

En aquest estudi de cas, hem cobert els aspectes essencials del processament de dades en REXX:

  • Llegir dades de fitxers
  • Processar i manipular dades
  • Escriure dades a fitxers
  • Implementar la gestió d'errors

En dominar aquests conceptes, estaràs ben equipat per gestionar diverses tasques de processament de dades utilitzant REXX. Practica aquests exemples i intenta aplicar-los a les teves pròpies necessitats de processament de dades per reforçar la teva comprensió.
