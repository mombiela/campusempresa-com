Introducció a REXX

REXX (REstructured eXtended eXecutor) és un llenguatge de programació dissenyat per a ser fàcil d'aprendre i llegir. Sovint s'utilitza per a scripting, manipulació de dades i prototipat ràpid. Aquesta secció cobrirà els conceptes bàsics de REXX i com començar a utilitzar-lo a Windows.

Conceptes Clau

  • Simplicitat: REXX està dissenyat per ser fàcil de llegir i escriure.
  • Llenguatge Interpretat: Els scripts REXX són interpretats, el que significa que s'executen directament sense necessitat de compilació prèvia.
  • Portabilitat: REXX es pot utilitzar en diversos sistemes operatius, incloent Windows.

Configuració de REXX a Windows

Per començar a programar en REXX a Windows, necessites instal·lar un intèrpret de REXX. Un intèrpret popular és Regina REXX.

Passos per Instal·lar Regina REXX

  1. Descarregar Regina REXX: Visita el lloc web de Regina REXX i descarrega l'instal·lador per a Windows.
  2. Executar l'Instal·lador: Segueix les instruccions en pantalla per instal·lar Regina REXX.
  3. Configurar Variables d'Entorn: Afegeix el camí a l'executable de Regina REXX a la variable d'entorn PATH del teu sistema.

Verificant la Instal·lació

Per verificar que Regina REXX està instal·lat correctament, obre un Símbol del Sistema i escriu:

rexx -v

Hauries de veure la informació de la versió de Regina REXX.

Escriure el Teu Primer Script REXX

Escrivim un simple script de "Hola, Món!" en REXX.

Exemple: Hola, Món!

Crea un nou fitxer anomenat hello.rexx i afegeix el següent codi:

/* Hola Món en REXX */
say "Hola, Món!"

Per executar l'script, obre un Símbol del Sistema, navega al directori que conté hello.rexx, i escriu:

rexx hello.rexx

Hauries de veure la sortida:

Hola, Món!

Sintaxi Bàsica i Comandes

Entendre la sintaxi bàsica i les comandes és crucial per escriure scripts REXX efectius.

Variables i Tipus de Dades

  • Variables: Les variables REXX no tenen tipus i poden contenir qualsevol tipus de dada.
  • Assignació: Utilitza l'operador = per assignar valors a les variables.

Exemple: Variables

/* Exemple de Variable */
name = "John Doe"
age = 30
say "Nom:" name
say "Edat:" age

Estructures de Control

  • IF-THEN-ELSE: Execució condicional.
  • DO-END: Construcció de bucle.

Exemple: Estructures de Control

/* Exemple d'Estructures de Control */
age = 20
if age >= 18 then
    say "Ets un adult."
else
    say "Ets un menor."

/* Exemple de Bucle */
do i = 1 to 5
    say "Iteració:" i
end

Temes Avançats

A mesura que et sentis més còmode amb REXX, pots explorar temes més avançats com l'entrada/sortida de fitxers, la gestió d'errors i la interfície amb altres programes.

Entrada/Sortida de Fitxers

  • Llegir Fitxers: Utilitza la funció linein.
  • Escriure Fitxers: Utilitza la funció lineout.

Exemple: Entrada/Sortida de Fitxers

/* Exemple d'Entrada/Sortida de Fitxers */
filename = "example.txt"

/* Escriure al fitxer */
call lineout filename, "Això és una prova."

/* Llegir del fitxer */
line = linein(filename)
say "Llegit del fitxer:" line

Gestió d'Errors

  • Signal: Utilitza la instrucció signal per gestionar errors.
  • Trap: Utilitza la instrucció trap per capturar errors específics.

Exemple: Gestió d'Errors

/* Exemple de Gestió d'Errors */
signal on syntax name errorHandler

/* Error intencionat */
x = 1 / 0

errorHandler:
    say "S'ha produït un error."
    exit

Conclusió

REXX és un llenguatge de programació versàtil i fàcil d'aprendre que és molt adequat per a tasques de scripting i automatització a Windows. Entenent la sintaxi bàsica, les estructures de control i les característiques avançades com l'entrada/sortida de fitxers i la gestió d'errors, pots escriure scripts REXX potents per automatitzar diverses tasques.
