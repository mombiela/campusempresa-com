Introducció

Llegir des de fitxers és una habilitat fonamental en programació, permetent-te processar i analitzar dades emmagatzemades en fitxers externs. En aquesta secció, cobrirem com llegir des de fitxers utilitzant REXX, començant pels conceptes bàsics i avançant cap a tècniques més avançades.

Lectura Bàsica de Fitxers

Obrir un Fitxer

Per llegir des d'un fitxer en REXX, primer has d'obrir-lo. Això es fa utilitzant la funció OPEN.

/* Obrir un fitxer per llegir */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error obrint el fitxer'
ELSE
  SAY 'Fitxer obert amb èxit'
  • file: El nom del fitxer que vols obrir.
  • file_handle: Un identificador que representa el fitxer obert. Si el fitxer no es pot obrir, file_handle serà 0.

Llegir Línies d'un Fitxer

Un cop el fitxer està obert, pots llegir-ne el contingut línia per línia utilitzant la funció LINEIN.

/* Llegir línies del fitxer */
DO WHILE LINES(file_handle) > 0
  line = LINEIN(file_handle)
  SAY line
END
  • LINES(file_handle): Retorna el nombre de línies restants en el fitxer.
  • LINEIN(file_handle): Llegeix la següent línia del fitxer.

Tancar un Fitxer

Després d'acabar de llegir del fitxer, és important tancar-lo utilitzant la funció CLOSE.

/* Tancar el fitxer */
CALL CLOSE file_handle

Lectura Avançada de Fitxers

Llegir Línies Específiques

Pots llegir línies específiques d'un fitxer especificant el número de línia en la funció LINEIN.

/* Llegir la cinquena línia del fitxer */
line = LINEIN(file_handle, 5)
SAY line

Llegir amb Gestió d'Errors

És una bona pràctica incloure la gestió d'errors quan es treballa amb fitxers per gestionar problemes inesperats.

/* Obrir un fitxer amb gestió d'errors */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error obrint el fitxer'
ELSE DO
  /* Llegir línies amb gestió d'errors */
  DO WHILE LINES(file_handle) > 0
    line = LINEIN(file_handle)
    IF RC \= 0 THEN
      SAY 'Error llegint la línia'
    ELSE
      SAY line
  END
  /* Tancar el fitxer */
  CALL CLOSE file_handle
END

Llegir Fitxers Grans

Per a fitxers grans, potser voldràs llegir blocs de dades en lloc de línia per línia per millorar el rendiment.

/* Llegir blocs de dades del fitxer */
DO WHILE LINES(file_handle) > 0
  chunk = CHARIN(file_handle, 1, 1024) /* Llegir 1024 caràcters */
  SAY chunk
END
  • CHARIN(file_handle, start, length): Llegeix un nombre especificat de caràcters començant des d'una posició donada.

Resum

En aquesta secció, hem cobert els conceptes bàsics de llegir des de fitxers en REXX, incloent obrir, llegir i tancar fitxers. També hem explorat tècniques avançades com llegir línies específiques, gestió d'errors i llegir fitxers grans en blocs. Dominar aquestes habilitats et permetrà processar i analitzar dades emmagatzemades en fitxers externs de manera eficient utilitzant REXX.
