Introducció als Macros REXX

REXX (Restructured Extended Executor) és un llenguatge de script versàtil que s'utilitza àmpliament tant per a la programació de scripts com de macros. En aquesta secció, explorarem el concepte dels macros REXX, els seus usos i com crear-los i executar-los.

Què és un Macro REXX?

  • Un macro REXX és un script escrit en el llenguatge REXX que automatitza tasques repetitives.
  • Els macros es poden utilitzar per ampliar la funcionalitat de les aplicacions, automatitzar fluxos de treball i simplificar operacions complexes.

Beneficis d'Utilitzar Macros REXX

  • Automatització: Automatitza tasques repetitives per estalviar temps i reduir errors.
  • Personalització: Adapta les aplicacions per satisfer necessitats específiques.
  • Eficiència: Augmenta la productivitat racionalitzant els fluxos de treball.

Creació d'un Macro REXX Simple

Per crear un macro REXX, cal escriure un script REXX i desar-lo amb una extensió .rexx o .cmd. A continuació es mostra un exemple simple d'un macro REXX que imprimeix "Hola, Món!".

/* Hello World REXX Macro */
say "Hello, World!"

Explicació

  • /* ... */: Això és un comentari en REXX. Els comentaris són ignorats per l'intèrpret.
  • say "Hello, World!": La instrucció say mostra la cadena "Hello, World!" a la consola.

Executar un Macro REXX

Per executar un macro REXX, cal tenir un intèrpret REXX instal·lat al sistema. Pots executar el macro des de la línia de comandes o dins d'una aplicació que suporti REXX.

Execució des de la Línia de Comandes

  1. Obre la teva interfície de línia de comandes (CLI).
  2. Navega al directori on està desat el teu script REXX.
  3. Executa l'script utilitzant l'intèrpret REXX.
rexx hello.rexx

Macros REXX Avançats

A mesura que et familiaritzis més amb REXX, podràs crear macros més complexos que incloguin variables, bucles i declaracions condicionals.

Variables

Les variables en REXX són de tipus dinàmic i no necessiten declaració explícita.

/* Variable Example */
name = "John Doe"
say "Hello, " name

Bucles

REXX suporta diversos tipus de bucles, incloent bucles do.

/* Loop Example */
do i = 1 to 5
    say "Iteration" i
end

Declaracions condicionals

REXX utilitza declaracions if per a l'execució condicional.

/* Conditional Example */
age = 20
if age >= 18 then
    say "You are an adult."
else
    say "You are a minor."

Exemple Pràctic: Macro de Processament de Fitxers

Creem un macro més pràctic que llegeix un fitxer i imprimeix el seu contingut línia per línia.

/* File Processing Macro */
fileName = "example.txt"
call stream fileName, "c", "open"
if stream(fileName, "s") = "READY:" then do
    do while lines(fileName) > 0
        line = linein(fileName)
        say line
    end
    call stream fileName, "c", "close"
end
else
    say "Error: Cannot open file" fileName

Explicació

  • call stream fileName, "c", "open": Obre el fitxer per a lectura.
  • if stream(fileName, "s") = "READY:" then do: Comprova si el fitxer està llest per ser llegit.
  • do while lines(fileName) > 0: Itera a través de cada línia del fitxer.
  • line = linein(fileName): Llegeix una línia del fitxer.
  • say line: Imprimeix la línia a la consola.
  • call stream fileName, "c", "close": Tanca el fitxer després de la lectura.

Conclusió

Els macros REXX són eines poderoses per automatitzar tasques i ampliar la funcionalitat de les aplicacions. Entenent els conceptes bàsics de la programació en REXX, pots crear macros simples i, gradualment, avançar cap a macros més complexos que involucrin variables, bucles i declaracions condicionals. Practica escrivint els teus propis macros per automatitzar tasques i millorar la teva productivitat.
