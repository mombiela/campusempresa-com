Introducció

REXX (Restructured Extended Executor) és un llenguatge de programació que és particularment fort en la manipulació de cadenes. Aquesta secció cobrirà diverses funcions de cadena en REXX, des de les bàsiques fins a les avançades, proporcionant explicacions clares, exemples i exercicis.

Funcions bàsiques de cadena

LENGTH

  • Propòsit: Retorna la longitud d'una cadena.
  • Sintaxi: LENGTH(string)
/* Exemple */
string = "Hello, World!"
say LENGTH(string)  /* Sortida: 13 */

SUBSTR

  • Propòsit: Extreu una subcadena d'una cadena.
  • Sintaxi: SUBSTR(string, start, length)
/* Exemple */
string = "Hello, World!"
say SUBSTR(string, 8, 5)  /* Sortida: World */

POS

  • Propòsit: Troba la posició d'una subcadena dins d'una cadena.
  • Sintaxi: POS(needle, haystack, start)
/* Exemple */
string = "Hello, World!"
say POS("World", string)  /* Sortida: 8 */

LEFT i RIGHT

  • Propòsit: Extreu un nombre especificat de caràcters del costat esquerre o dret d'una cadena.
  • Sintaxi: LEFT(string, length) i RIGHT(string, length)
/* Exemple */
string = "Hello, World!"
say LEFT(string, 5)   /* Sortida: Hello */
say RIGHT(string, 6)  /* Sortida: World! */

Funcions intermèdies de cadena

STRIP

  • Propòsit: Elimina espais inicials i/o finals d'una cadena.
  • Sintaxi: STRIP(string, option, char)
/* Exemple */
string = "   Hello, World!   "
say STRIP(string)  /* Sortida: "Hello, World!" */

OVERLAY

  • Propòsit: Superposa una subcadena sobre una cadena.
  • Sintaxi: OVERLAY(new, old, start, length)
/* Exemple */
string = "Hello, World!"
say OVERLAY("REXX", string, 8, 4)  /* Sortida: Hello, REXX! */

TRANSLATE

  • Propòsit: Traduïx caràcters en una cadena.
  • Sintaxi: TRANSLATE(string, to, from)
/* Exemple */
string = "Hello, World!"
say TRANSLATE(string, "123", "Hlo")  /* Sortida: 1e22, W2r2d! */

VERIFY

  • Propòsit: Verifica caràcters en una cadena contra un conjunt de caràcters permesos.
  • Sintaxi: VERIFY(string, reference, option)
/* Exemple */
string = "Hello, World!"
say VERIFY(string, "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz, !")  /* Sortida: 0 */

Funcions avançades de cadena

WORD

  • Propòsit: Extreu una paraula específica d'una cadena.
  • Sintaxi: WORD(string, n)
/* Exemple */
string = "Hello, World! Welcome to REXX."
say WORD(string, 3)  /* Sortida: Welcome */

WORDPOS

  • Propòsit: Troba la posició d'una paraula dins d'una cadena.
  • Sintaxi: WORDPOS(word, string)
/* Exemple */
string = "Hello, World! Welcome to REXX."
say WORDPOS("Welcome", string)  /* Sortida: 3 */

WORDS

  • Propòsit: Compta el nombre de paraules en una cadena.
  • Sintaxi: WORDS(string)
/* Exemple */
string = "Hello, World! Welcome to REXX."
say WORDS(string)  /* Sortida: 5 */

DELSTR

  • Propòsit: Elimina una subcadena d'una cadena.
  • Sintaxi: DELSTR(string, start, length)
/* Exemple */
string = "Hello, World!"
say DELSTR(string, 8, 6)  /* Sortida: Hello, */

INSERT

  • Propòsit: Insereix una subcadena en una cadena.
  • Sintaxi: INSERT(new, old, start, length, pad)
/* Exemple */
string = "Hello, World!"
say INSERT("REXX", string, 8)  /* Sortida: Hello, REXXXWorld! */

Resum

En aquesta secció, hem cobert una gamma de funcions de cadena en REXX, des de les bàsiques fins a les avançades. Aquestes funcions són essencials per a una manipulació efectiva de cadenes, que és una tasca comuna en molts escenaris de programació. Entendre i dominar aquestes funcions millorarà significativament la teva capacitat per treballar amb dades de text en REXX.

Exercicis

  1. Escriu un script en REXX per extreure el nom de domini d'una adreça de correu electrònic.
  2. Crea una funció que capitalitzi la primera lletra de cada paraula en una cadena.
  3. Escriu un script per comptar el nombre de vocals en una cadena donada.
  4. Desenvolupa una funció que inverteixi les paraules en una frase mantenint les paraules en l'ordre original.

Practicant aquests exercicis, reforçaràs la teva comprensió de les funcions de cadena en REXX i milloraràs les teves habilitats de resolució de problemes.
