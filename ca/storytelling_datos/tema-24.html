Llibres

  • "Storytelling with Data: A Data Visualization Guide for Business Professionals" per Cole Nussbaumer Knaflic
    Aquest llibre és una guia essencial per aprendre a explicar històries efectives amb dades. Ofereix principis pràctics i exemples clars per millorar les teves habilitats de visualització de dades.

  • "The Data Warehouse Toolkit: The Definitive Guide to Dimensional Modeling" per Ralph Kimball i Margy Ross
    Encara que se centra en la modelització dimensional, aquest llibre proporciona una base sòlida per entendre com estructurar i presentar dades de manera efectiva.

  • "Data Science for Business: What You Need to Know about Data Mining and Data-Analytic Thinking" per Foster Provost i Tom Fawcett
    Aquest llibre és ideal per comprendre com aplicar tècniques de ciència de dades en un context empresarial, la qual cosa és crucial per explicar històries convincents amb dades.

Articles i Blogs

  • "The Art of Storytelling in Data Science" per Bernard Marr
    Aquest article explora com les dades poden ser utilitzades per explicar històries impactants i ofereix consells pràctics per millorar les teves habilitats de storytelling.

  • "Data Storytelling: The Essential Data Science Skill Everyone Needs" en el blog de Towards Data Science
    Un article que destaca la importància del storytelling en la ciència de dades i proporciona exemples i estratègies per millorar aquesta habilitat.

  • "How to Tell a Story with Data" en el blog de Tableau
    Aquest article de Tableau ofereix una guia pas a pas per crear narratives efectives amb dades utilitzant les seves eines de visualització.

Cursos en Línia

  • "Storytelling with Data" en Coursera
    Un curs ofert per Cole Nussbaumer Knaflic que aprofundeix en els principis del seu llibre i ofereix exercicis pràctics per millorar les teves habilitats de visualització de dades.

    Un curs que cobreix els fonaments del storytelling amb dades i ofereix exemples pràctics i exercicis per aplicar el que s'ha après en un context empresarial.

Eines i Programari

  • Tableau
    Una eina de visualització de dades que permet crear gràfics interactius i dashboards. Ideal per explicar històries complexes amb dades.

  • Power BI
    Una eina de Microsoft que facilita la creació d'informes i dashboards interactius. És especialment útil per integrar dades de diverses fonts i presentar-les de manera coherent.

  • D3.js
    Una biblioteca de JavaScript per produir visualitzacions de dades dinàmiques i interactives en navegadors web. Requereix coneixements de programació, però ofereix una gran flexibilitat.

Exemples de Codi

Exemple de Visualització amb D3.js

// Selecciona l'element SVG
var svg = d3.select("svg");

// Defineix les dades
var data = [10, 20, 30, 40, 50];

// Crea un escalador lineal
var x = d3.scaleLinear()
          .domain([0, d3.max(data)])
          .range([0, 500]);

// Afegeix rectangles al SVG
svg.selectAll("rect")
   .data(data)
   .enter()
   .append("rect")
   .attr("width", function(d) { return x(d); })
   .attr("height", 20)
   .attr("y", function(d, i) { return i * 25; });

Aquest exemple mostra com crear una simple visualització de barres utilitzant D3.js. Les dades s'escala i es representen com a rectangles en un element SVG.
