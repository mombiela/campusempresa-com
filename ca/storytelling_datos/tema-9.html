Introducció

En aquesta secció del curs sobre Storytelling amb Dades, explorarem diverses eines de visualització de dades. Aquestes eines són essencials per transformar dades complexes en gràfics i diagrames comprensibles, facilitant la comunicació d'informació clau a diferents audiències.

Eines de Programació

  1. Matplotlib (Python)

    • Descripció: Matplotlib és una biblioteca de Python per crear gràfics estàtics, animats i interactius.
    • Característiques Clau:
      • Àmplia personalització de gràfics.
      • Compatible amb altres biblioteques de Python com NumPy i Pandas.
      • Suport per a gràfics 2D i 3D.
    • Exemple: 
      import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 20, 25, 30, 35]

plt.plot(x, y)
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Simple Line Plot')
plt.show()

  2. ggplot2 (R)

    • Descripció: ggplot2 és una biblioteca de R per crear gràfics complexos i personalitzats.
    • Característiques Clau:
      • Basat en la gramàtica de gràfics.
      • Alta personalització i flexibilitat.
      • Integració amb altres biblioteques de R.
    • Exemple: 
      library(ggplot2)

data <- data.frame(
  x = c(1, 2, 3, 4, 5),
  y = c(10, 20, 25, 30, 35)
)

ggplot(data, aes(x=x, y=y)) +
  geom_line() +
  labs(title="Simple Line Plot", x="X-axis", y="Y-axis")

  3. D3.js (JavaScript)

    • Descripció: D3.js és una biblioteca de JavaScript per produir visualitzacions dinàmiques i interactives en navegadors web.
    • Característiques Clau:
      • Manipulació directa del DOM.
      • Suport per a animacions i transicions.
      • Alta flexibilitat i personalització.
    • Exemple: 
      <!DOCTYPE html>
<html>
<head>
  <script src="https://d3js.org/d3.v6.min.js"></script>
</head>
<body>
  <script>
    var data = [10, 20, 25, 30, 35];
    var width = 500;
    var height = 300;

    var svg = d3.select("body")
                .append("svg")
                .attr("width", width)
                .attr("height", height);

    svg.selectAll("rect")
       .data(data)
       .enter()
       .append("rect")
       .attr("x", (d, i) => i * 30)
       .attr("y", d => height - d)
       .attr("width", 25)
       .attr("height", d => d)
       .attr("fill", "blue");
  </script>
</body>
</html>

Conclusió

Les eines de visualització de dades són fonamentals per al storytelling amb dades. L'elecció de l'eina adequada depèn de diversos factors, incloent-hi el nivell d'interactivitat requerit, la facilitat d'ús i la integració amb altres eines. Ja sigui que prefereixis una solució de programari com Tableau o Power BI, o una biblioteca de programació com Matplotlib, ggplot2 o D3.js, cada eina té els seus propis avantatges i desavantatges. Familiaritzar-se amb aquestes eines et permetrà comunicar les teves dades de manera més efectiva i persuasiva.
