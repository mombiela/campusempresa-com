◄ Principis de la Visualització de Dades
Eines de Visualització de Dades ►

En el món del storytelling amb dades, triar el gràfic adequat és crucial per comunicar de manera efectiva la informació. A continuació, explorarem els tipus de gràfics més comuns i quan és apropiat utilitzar-los.

Gràfics de Barres

Els gràfics de barres són ideals per comparar quantitats entre diferents categories. Són fàcils d'entendre i molt versàtils.

  • Ús Principal: Comparar valors entre diferents categories.
  • Exemple: Comparar les vendes de diferents productes en un trimestre.
import matplotlib.pyplot as plt

# Dades d'exemple
categories = ['Producte A', 'Producte B', 'Producte C']
vendes = [100, 150, 80]

plt.bar(categories, vendes)
plt.xlabel('Productes')
plt.ylabel('Vendes')
plt.title('Vendes per Producte')
plt.show()

Gràfics de Línies

Els gràfics de línies són perfectes per mostrar tendències al llarg del temps. Són útils per visualitzar canvis i patrons en les dades.

  • Ús Principal: Mostrar tendències al llarg del temps.
  • Exemple: Evolució de les vendes mensuals durant un any.
import matplotlib.pyplot as plt

# Dades d'exemple
mesos = ['Gener', 'Febrer', 'Març', 'Abril']
vendes = [100, 120, 130, 150]

plt.plot(mesos, vendes, marker='o')
plt.xlabel('Mesos')
plt.ylabel('Vendes')
plt.title('Evolució de Vendes Mensuals')
plt.show()

Gràfics de Dispersió

Els gràfics de dispersió són útils per mostrar la relació entre dues variables. Poden ajudar a identificar correlacions i patrons.

  • Ús Principal: Mostrar la relació entre dues variables.
  • Exemple: Relació entre la inversió en màrqueting i les vendes.
import matplotlib.pyplot as plt

# Dades d'exemple
inversió = [10, 20, 30, 40, 50]
vendes = [15, 25, 35, 45, 55]

plt.scatter(inversió, vendes)
plt.xlabel('Inversió en Màrqueting (en milers)')
plt.ylabel('Vendes (en milers)')
plt.title('Relació entre Inversió en Màrqueting i Vendes')
plt.show()

Gràfics de Pastís

Els gràfics de pastís són útils per mostrar la proporció de diferents parts d'un tot. No obstant això, s'han d'utilitzar amb precaució ja que poden ser difícils d'interpretar amb moltes categories.

  • Ús Principal: Mostrar la proporció de parts d'un tot.
  • Exemple: Distribució del mercat entre diferents competidors.
import matplotlib.pyplot as plt

# Dades d'exemple
categories = ['Competidor A', 'Competidor B', 'Competidor C']
participació = [40, 35, 25]

plt.pie(participació, labels=categories, autopct='%1.1f%%')
plt.title('Distribució del Mercat')
plt.show()

Gràfics d'Àrea

Els gràfics d'àrea són similars als gràfics de línies, però les àrees sota les línies estan omplertes. Són útils per mostrar la magnitud de les tendències al llarg del temps.

  • Ús Principal: Mostrar la magnitud de les tendències al llarg del temps.
  • Exemple: Evolució del trànsit web mensual.
import matplotlib.pyplot as plt

# Dades d'exemple
mesos = ['Gener', 'Febrer', 'Març', 'Abril']
trànsit = [1000, 1500, 2000, 2500]

plt.fill_between(mesos, trànsit, color="skyblue", alpha=0.4)
plt.plot(mesos, trànsit, marker='o', color="Slateblue", alpha=0.6)
plt.xlabel('Mesos')
plt.ylabel('Trànsit Web')
plt.title('Evolució del Trànsit Web Mensual')
plt.show()

Conclusió

Triar el gràfic adequat és essencial per comunicar les teves dades de manera efectiva. Aquí hi ha un resum ràpid de quan utilitzar cada tipus de gràfic:

| Tipus de Gràfic | Ús Principal | |---------------------|----------------------------------------------------| | Gràfic de Barres | Comparar valors entre diferents categories | | Gràfic de Línies | Mostrar tendències al llarg del temps | | Gràfic de Dispersió | Mostrar la relació entre dues variables | | Gràfic de Pastís | Mostrar la proporció de parts d'un tot | | Gràfic d'Àrea | Mostrar la magnitud de les tendències al llarg del temps |

Recorda sempre considerar la teva audiència i el missatge que desitges transmetre en triar el tipus de gràfic.
