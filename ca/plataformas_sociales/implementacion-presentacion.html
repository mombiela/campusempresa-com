Introducció

En aquesta secció, aprendràs com implementar i presentar el teu projecte de plataformes de xarxes socials. Aquest procés inclou la integració de funcionalitats, proves, i la preparació d'una presentació efectiva per a stakeholders.

Implementació del Projecte

Integració de Funcionalitats

  • Autenticació d'Usuaris: Implementa un sistema d'autenticació segur.

    from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy
from werkzeug.security import generate_password_hash, check_password_hash

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///users.db'
db = SQLAlchemy(app)

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

@app.route('/register', methods=['POST'])
def register():
    data = request.get_json()
    hashed_password = generate_password_hash(data['password'], method='sha256')
    new_user = User(username=data['username'], password=hashed_password)
    db.session.add(new_user)
    db.session.commit()
    return jsonify({'message': 'User registered successfully'})

if __name__ == '__main__':
    app.run(debug=True)
    • Explicació: Aquest codi crea una API simple per registrar usuaris utilitzant Flask i SQLAlchemy. S'assegura que les contrasenyes s'emmagatzemin de manera segura mitjançant hashing.

  • Publicació de Continguts: Permet als usuaris publicar contingut.

    @app.route('/post', methods=['POST'])
def create_post():
    data = request.get_json()
    new_post = Post(content=data['content'], user_id=data['user_id'])
    db.session.add(new_post)
    db.session.commit()
    return jsonify({'message': 'Post created successfully'})
    • Explicació: Aquest endpoint permet als usuaris crear publicacions. S'assumeix que existeix un model Post prèviament definit.

Proves del Projecte

  • Proves Unitàries: Assegura't que cada component funcioni correctament.

    import unittest

class TestUserModel(unittest.TestCase):
    def test_password_hashing(self):
        user = User(username='testuser', password='testpassword')
        self.assertTrue(user.password != 'testpassword')

if __name__ == '__main__':
    unittest.main()
    • Explicació: Aquest exemple de prova unitaria verifica que les contrasenyes no s'emmagatzemin en text pla.

  • Proves d'Integració: Verifica que els components funcionin junts.

    def test_create_post(self):
    response = self.client.post('/post', json={'content': 'Hello World', 'user_id': 1})

Presentació del Projecte

Preparació de la Presentació

  • Estructura de la Presentació:

    • Introducció: Breu descripció del projecte.
    • Objectius: Què s'espera aconseguir amb el projecte.
    • Funcionalitats Clau: Descripció de les principals característiques.
    • Demostració en Viu: Mostra el projecte en funcionament.
    • Conclusió: Resum i pròxims passos.

  • Eines de Presentació:

    • Diapositives: Utilitza eines com PowerPoint o Google Slides.
    • Demostració en Viu: Prepara un entorn de prova per mostrar el projecte en acció.

Exemple de Diapositiva

| Secció | Contingut | |------------------|--------------------------------------------------------------------------| | Introducció | Descripció breu del projecte i el seu propòsit. | | Objectius | Llistat d'objectius específics que el projecte busca assolir. | | Funcionalitats | Descripció de les funcionalitats clau amb captures de pantalla. | | Demostració | Vídeo o demostració en viu del projecte en funcionament. | | Conclusió | Resum dels èxits i pròxims passos o millores futures. |

Conclusió

La implementació i presentació d'un projecte de plataformes de xarxes socials requereix una planificació acurada i una execució meticulosa. Assegura't d'integrar totes les funcionalitats necessàries, realitzar proves exhaustives i preparar una presentació clara i concisa per comunicar efectivament la teva feina als stakeholders.
