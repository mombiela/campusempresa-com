Introducció a Redis

Redis és un magatzem de dades de codi obert i en memòria utilitzat com a base de dades, memòria cau i intermediari de missatges. Admet diverses estructures de dades com cadenes, hash, llistes, conjunts i més. Aquest curs et guiarà des dels conceptes bàsics de Redis fins a temes avançats, assegurant que tinguis una comprensió completa de les seves capacitats i aplicacions.

Conceptes Clau

  • Emmagatzematge en Memòria: Redis emmagatzema dades en memòria, proporcionant operacions de lectura i escriptura extremadament ràpides.
  • Estructures de Dades: Redis admet múltiples estructures de dades com cadenes, llistes, conjunts, conjunts ordenats, hash, bitmaps, hyperloglogs i índexs geoespacials.
  • Persistència: Redis pot persistir dades al disc, permetent la recuperació de dades després d'un reinici.
  • Replicació: Redis admet la replicació mestre-esclau, permetent redundància de dades i alta disponibilitat.
  • Transaccions: Redis admet transaccions, permetent que múltiples comandes es puguin executar en una única operació atòmica.
  • Missatgeria Pub/Sub: Redis proporciona un paradigma de missatgeria de publicació/subscripció, útil per a aplicacions de missatgeria en temps real.
  • Scripting Lua: Redis permet la creació de scripts al servidor utilitzant Lua, proporcionant capacitats poderoses i flexibles de manipulació de dades.

Estructura del Curs

Nivell Principiant

  1. Introducció a Redis

    • Què és Redis?
    • Casos d'ús i aplicacions
    • Instal·lació de Redis

  2. Comandes Bàsiques

    • Connexió a Redis
    • Operacions CRUD amb cadenes
    • Exemple: 
      SET key "value"
GET key
DEL key

  3. Treballant amb Estructures de Dades

    • Llistes
    • Conjunts
    • Hash
    • Exemple: 
      LPUSH mylist "item1"
SADD myset "member1"
HSET myhash field1 "value1"

  4. Persistència

    • Instantànies RDB
    • AOF (Fitxer Només Afegeix)
    • Configuració de la persistència

Nivell Intermedi

  1. Estructures de Dades Avançades

    • Conjunts ordenats
    • Bitmaps
    • HyperLogLogs
    • Exemple: 
      ZADD myzset 1 "one"
SETBIT mybitmap 7 1
PFADD myhyperloglog "element"

  2. Replicació i Alta Disponibilitat

    • Configuració de la replicació mestre-esclau
    • Configuració de Redis Sentinel per a alta disponibilitat

  3. Transaccions i Scripting

    • Utilització de MULTI/EXEC per a transaccions
    • Introducció a l'scripting Lua
    • Exemple: 
      MULTI
SET key1 "value1"
SET key2 "value2"
EXEC

Nivell Avançat

  1. Escalabilitat de Redis

    • Partició de dades amb Redis Cluster
    • Configuració i gestió d'un Redis Cluster

  2. Optimització del Rendiment

    • Tècniques d'optimització de memòria
    • Configuració de Redis per a alt rendiment

  3. Seguretat

    • Assegurant les instàncies de Redis
    • Autenticació i autorització
    • Exemple: 
      CONFIG SET requirepass "yourpassword"

  4. Monitorització i Manteniment

    • Monitorització del rendiment de Redis
    • Estratègies de còpia de seguretat i restauració

Resum

Aquest curs proporcionarà una comprensió exhaustiva de Redis, des de les operacions bàsiques fins a configuracions i optimitzacions avançades. Al final del curs, estaràs equipat amb el coneixement per utilitzar Redis de manera efectiva en diverses situacions, assegurant un alt rendiment i fiabilitat per a les teves aplicacions.
