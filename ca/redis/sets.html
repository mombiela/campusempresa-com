Introducció als Conjunts

Els conjunts a Redis són una col·lecció desordenada de cadenes úniques. Són útils per emmagatzemar col·leccions d'elements únics, com ara identificadors d'usuari, etiquetes o qualsevol altre tipus de dades úniques.

Conceptes Clau

  • Col·lecció Desordenada: A diferència de les llistes, els conjunts no mantenen cap ordre dels elements.
  • Elements Únics: Els conjunts asseguren automàticament que tots els elements siguin únics.
  • Operacions Eficients: Redis proporciona operacions eficients per afegir, eliminar i comprovar l'existència d'elements en un conjunt.

Operacions Bàsiques amb Conjunts

Afegir Elements a un Conjunt

Per afegir elements a un conjunt, utilitza la comanda SADD.

SADD myset "element1"
SADD myset "element2" "element3"

Comprovar la Pertinença

Per comprovar si un element és membre d'un conjunt, utilitza la comanda SISMEMBER.

SISMEMBER myset "element1"  # Retorna 1 (cert) si element1 és al conjunt, 0 (fals) en cas contrari

Eliminar Elements d'un Conjunt

Per eliminar elements d'un conjunt, utilitza la comanda SREM.

SREM myset "element1"

Recuperar Tots els Elements

Per recuperar tots els elements d'un conjunt, utilitza la comanda SMEMBERS.

SMEMBERS myset  # Retorna tots els elements del conjunt

Operacions Avançades amb Conjunts

Cardinalitat del Conjunt

Per obtenir el nombre d'elements d'un conjunt, utilitza la comanda SCARD.

SCARD myset  # Retorna el nombre d'elements del conjunt

Recuperació d'Element Aleatori

Per obtenir un element aleatori d'un conjunt, utilitza la comanda SRANDMEMBER.

SRANDMEMBER myset  # Retorna un element aleatori del conjunt

Operacions amb Conjunts: Unió, Intersecció i Diferència

Redis proporciona comandes per realitzar operacions amb conjunts com ara unió, intersecció i diferència.

Unió

Per obtenir la unió de múltiples conjunts, utilitza la comanda SUNION.

SUNION set1 set2  # Retorna la unió de set1 i set2

Intersecció

Per obtenir la intersecció de múltiples conjunts, utilitza la comanda SINTER.

SINTER set1 set2  # Retorna la intersecció de set1 i set2

Diferència

Per obtenir la diferència entre conjunts, utilitza la comanda SDIFF.

SDIFF set1 set2  # Retorna els elements de set1 que no són a set2

Emmagatzemar Resultats d'Operacions amb Conjunts

Pots emmagatzemar els resultats d'operacions amb conjunts en un nou conjunt utilitzant comandes com SUNIONSTORE, SINTERSTORE i SDIFFSTORE.

SUNIONSTORE result_set set1 set2  # Emmagatzema la unió de set1 i set2 a result_set
SINTERSTORE result_set set1 set2  # Emmagatzema la intersecció de set1 i set2 a result_set
SDIFFSTORE result_set set1 set2   # Emmagatzema la diferència de set1 i set2 a result_set

Exemples Pràctics

Exemple 1: Gestió d'Etiquetes d'Usuari

Imagina que tens un sistema d'etiquetatge on els usuaris poden etiquetar els seus articles favorits. Pots utilitzar conjunts per gestionar aquestes etiquetes.

SADD user:1:tags "redis" "database" "nosql"
SADD user:2:tags "redis" "caching"

# Trobar etiquetes comunes entre dos usuaris
SINTER user:1:tags user:2:tags  # Retorna ["redis"]

Exemple 2: Visitants Únics en Temps Real

Pots utilitzar conjunts per fer un seguiment dels visitants únics d'un lloc web en temps real.

SADD visitors:2023-10-01 "user1"
SADD visitors:2023-10-01 "user2"
SADD visitors:2023-10-01 "user1"  # Duplicat, no s'afegirà

SCARD visitors:2023-10-01  # Retorna 2, el nombre de visitants únics

Conclusió

Els conjunts a Redis són una manera potent i eficient de gestionar col·leccions d'elements únics. Proporcionen una varietat d'operacions que es poden utilitzar per realitzar manipulacions de dades complexes. Entenent i utilitzant aquestes operacions, pots gestionar i consultar les teves dades a Redis de manera efectiva.

Recorda practicar aquestes comandes i explorar els seus casos d'ús per obtenir una millor comprensió de com es poden utilitzar els conjunts en les teves aplicacions.
