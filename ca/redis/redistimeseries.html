RedisTimeSeries és un mòdul de Redis dissenyat per gestionar dades de sèries temporals de manera eficient. Amplia Redis amb un conjunt de comandes i estructures de dades específicament optimitzades per a dades de sèries temporals, facilitant l'emmagatzematge, la consulta i l'anàlisi de dades de sèries temporals.

Introducció a RedisTimeSeries

RedisTimeSeries proporciona una estructura de dades especialitzada per a dades de sèries temporals, que són dades indexades per temps. Aquest mòdul és particularment útil per a aplicacions que requereixen una ingesta i consulta d'alt rendiment de dades de sèries temporals, com ara sistemes de monitorització, aplicacions IoT i anàlisi de dades de mercats financers.

Característiques Clau

  • Emmagatzematge Eficient de Dades: Optimitzat per emmagatzemar grans volums de dades de sèries temporals.
  • Alt Rendiment: Capacitat per gestionar un alt rendiment d'escriptura i lectura.
  • Agregació i Reducció de Mostres: Suport integrat per a l'agregació i la reducció de mostres de dades.
  • Compactació: Compactació automàtica de dades per reduir els requisits d'emmagatzematge.
  • Consulta: Capacitats avançades de consulta, incloent-hi consultes de rang i filtratge.

Instal·lació de RedisTimeSeries

Per utilitzar RedisTimeSeries, cal instal·lar el mòdul. Pots fer-ho utilitzant el CLI del mòdul Redis o incloent-lo en la teva configuració de Redis.

Exemple d'Instal·lació

# Descarregar i instal·lar RedisTimeSeries
git clone https://github.com/RedisTimeSeries/RedisTimeSeries.git
cd RedisTimeSeries
make

# Carregar el mòdul a Redis
redis-server --loadmodule ./redistimeseries.so

Comandes Bàsiques

Crear una Sèrie Temporal

Per crear una sèrie temporal, utilitza la comanda TS.CREATE. Aquesta comanda inicialitza una nova clau de sèrie temporal.

TS.CREATE temperature:room1

Afegir Punts de Dades

Per afegir punts de dades a una sèrie temporal, utilitza la comanda TS.ADD. Aquesta comanda requereix una marca de temps i un valor.

TS.ADD temperature:room1 1633024800 22.5
TS.ADD temperature:room1 1633028400 23.0

Consultar Dades

Per consultar dades d'una sèrie temporal, utilitza la comanda TS.RANGE. Aquesta comanda recupera punts de dades dins d'un rang de temps especificat.

TS.RANGE temperature:room1 1633024800 1633028400

Agregació i Reducció de Mostres

RedisTimeSeries suporta l'agregació i la reducció de mostres per resumir dades en intervals especificats.

TS.RANGE temperature:room1 1633024800 1633028400 AGGREGATION avg 3600

Característiques Avançades

Regles de Compactació

Les regles de compactació permeten reduir automàticament les mostres de dades a mesura que s'ingereixen. Això és útil per reduir els requisits d'emmagatzematge mentre es manté la informació important.

TS.CREATERULE temperature:room1 avg_temperature:room1 AVG 3600

Consultes de Múltiples Rangs

Les consultes de múltiples rangs permeten consultar diverses sèries temporals alhora. Això és útil per comparar dades entre diferents sèries.

TS.MRANGE 1633024800 1633028400 FILTER room=room1

Etiquetatge i Filtratge

Pots afegir etiquetes a les sèries temporals per categoritzar-les i filtrar-les més fàcilment.

TS.CREATE temperature:room1 LABELS room room1
TS.CREATE temperature:room2 LABELS room room2

# Consultar totes les sèries temporals amb l'etiqueta 'room=room1'
TS.MRANGE 1633024800 1633028400 FILTER room=room1

Exemple Pràctic

Vegem un exemple pràctic d'ús de RedisTimeSeries per monitoritzar dades de temperatura de diverses habitacions.

Exemple Pas a Pas

  1. Crear Sèries Temporals per a Cada Habitació

    TS.CREATE temperature:room1 LABELS room room1
TS.CREATE temperature:room2 LABELS room room2

  2. Afegir Punts de Dades

    TS.ADD temperature:room1 1633024800 22.5
TS.ADD temperature:room1 1633028400 23.0
TS.ADD temperature:room2 1633024800 21.0
TS.ADD temperature:room2 1633028400 21.5

  3. Consultar Dades per a una Habitació Específica

    TS.RANGE temperature:room1 1633024800 1633028400

  4. Agregació de Dades

    TS.RANGE temperature:room1 1633024800 1633028400 AGGREGATION avg 3600

  5. Filtrar Dades per Etiquetes

    TS.MRANGE 1633024800 1633028400 FILTER room=room1

Conclusió

RedisTimeSeries és un mòdul potent per gestionar dades de sèries temporals a Redis. Proporciona emmagatzematge eficient, alt rendiment i capacitats avançades de consulta, fent-lo ideal per a aplicacions que requereixen anàlisi en temps real de dades de sèries temporals. Aprofitant característiques com l'agregació, la compactació i l'etiquetatge, pots construir solucions robustes i escalables de dades de sèries temporals amb RedisTimeSeries.
