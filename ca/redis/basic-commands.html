Introducció a les comandes de Redis

Redis és un magatzem de dades en memòria, utilitzat com a base de dades, memòria cau i intermediari de missatges. Per interactuar amb Redis, utilitzes un conjunt de comandes que realitzen diverses operacions sobre les dades emmagatzemades a Redis. Aquesta secció cobrirà les comandes bàsiques que necessites per començar amb Redis.

Connexió a Redis

Abans de poder executar qualsevol comanda, has de connectar-te al teu servidor Redis. Pots fer-ho utilitzant el Redis CLI (Interfície de Línia de Comandes).

redis-cli

Un cop connectat, pots començar a emetre comandes.

Tipus de dades bàsics i comandes

Cadenes

Les cadenes són el tipus de valor més bàsic que pots emmagatzemar a Redis. Són segures per a binaris, el que significa que poden contenir qualsevol tipus de dades, com una imatge JPEG o un objecte Ruby serialitzat.

  • SET: Estableix el valor d'una clau.

    SET key "value"

    Exemple:

    SET name "John"

  • GET: Obté el valor d'una clau.

    GET key

    Exemple:

    GET name

  • DEL: Elimina una clau.

    DEL key

    Exemple:

    DEL name

Hashos

Els hashos són mapes entre camps de cadena i valors de cadena, el que els fa perfectes per representar objectes.

  • HSET: Estableix el valor d'un camp en un hash.

    HSET key field value

    Exemple:

    HSET user:1000 name "John"

  • HGET: Obté el valor d'un camp en un hash.

    HGET key field

    Exemple:

    HGET user:1000 name

  • HDEL: Elimina un camp en un hash.

    HDEL key field

    Exemple:

    HDEL user:1000 name

Llistes

Les llistes són col·leccions ordenades de cadenes. Pots afegir elements al cap o a la cua de la llista.

  • LPUSH: Afegeix un o més elements al cap d'una llista.

    LPUSH key value [value ...]

    Exemple:

    LPUSH mylist "world"
LPUSH mylist "hello"

  • LRANGE: Obté un rang d'elements d'una llista.

    LRANGE key start stop

    Exemple:

    LRANGE mylist 0 -1

  • LPOP: Elimina i obté el primer element d'una llista.

    LPOP key

    Exemple:

    LPOP mylist

Conjunts

Els conjunts són col·leccions desordenades de cadenes úniques.

  • SADD: Afegeix un o més membres a un conjunt.

    SADD key member [member ...]

    Exemple:

    SADD myset "hello"
SADD myset "world"

  • SMEMBERS: Obté tots els membres d'un conjunt.

    SMEMBERS key

    Exemple:

    SMEMBERS myset

  • SREM: Elimina un o més membres d'un conjunt.

    SREM key member [member ...]

    Exemple:

    SREM myset "hello"

Conjunts ordenats

Els conjunts ordenats són similars als conjunts però on cada membre està associat amb una puntuació. Els elements estan ordenats per les seves puntuacions.

  • ZADD: Afegeix un o més membres a un conjunt ordenat, o actualitza la seva puntuació si ja existeix.

    ZADD key score member [score member ...]

    Exemple:

    ZADD myzset 1 "one"
ZADD myzset 2 "two"

  • ZRANGE: Retorna un rang de membres en un conjunt ordenat, per índex.

    ZRANGE key start stop [WITHSCORES]

    Exemple:

    ZRANGE myzset 0 -1

  • ZREM: Elimina un o més membres d'un conjunt ordenat.

    ZREM key member [member ...]

    Exemple:

    ZREM myzset "one"

Conclusió

En aquesta secció, hem cobert les comandes bàsiques per interactuar amb Redis. Hem explorat com connectar-se a Redis i realitzar operacions sobre diferents tipus de dades com cadenes, hashos, llistes, conjunts i conjunts ordenats. Entendre aquestes comandes bàsiques és essencial per utilitzar Redis de manera efectiva en les teves aplicacions. A mesura que progressis, aprendràs comandes i tècniques més avançades per aprofitar tot el potencial de Redis.
