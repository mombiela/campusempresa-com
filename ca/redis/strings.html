Introducció a les cadenes de Redis

Les cadenes de Redis són el tipus de valor més bàsic que pots emmagatzemar a Redis. Són segures per a dades binàries, cosa que significa que poden contenir qualsevol tipus de dada, incloent text, números o objectes serialitzats.

  • Segures per a dades binàries: Les cadenes poden contenir qualsevol dada, com ara text, números o fins i tot dades binàries com imatges o objectes serialitzats.
  • Longitud màxima: Una cadena de Redis pot tenir fins a 512 MB de longitud.

Operacions bàsiques amb cadenes

Redis proporciona una varietat de comandes per treballar amb cadenes. Aquí tens algunes de les comandes més utilitzades:

Establir i obtenir cadenes

  • SET: Estableix el valor d'una clau.
  • GET: Obté el valor d'una clau.
SET mykey "Hello, Redis!"
GET mykey

Explicació:

  • SET mykey "Hello, Redis!" emmagatzema la cadena "Hello, Redis!" a la clau mykey.
  • GET mykey recupera el valor emmagatzemat a mykey.

Modificar cadenes

  • APPEND: Afegeix un valor a una clau.
  • STRLEN: Retorna la longitud de la cadena emmagatzemada en una clau.
APPEND mykey " How are you?"
GET mykey
STRLEN mykey

Explicació:

  • APPEND mykey " How are you?" afegeix " How are you?" al valor existent de mykey.
  • GET mykey recupera el valor actualitzat.
  • STRLEN mykey retorna la longitud de la cadena emmagatzemada a mykey.

Incrementar i decrementar

  • INCR: Incrementa el valor enter d'una clau en un.
  • DECR: Decrementa el valor enter d'una clau en un.
  • INCRBY: Incrementa el valor enter d'una clau per una quantitat especificada.
  • DECRBY: Decrementa el valor enter d'una clau per una quantitat especificada.
SET counter 10
INCR counter
GET counter
INCRBY counter 5
GET counter
DECR counter
GET counter
DECRBY counter 3
GET counter

Explicació:

  • SET counter 10 inicialitza la clau counter amb el valor 10.
  • INCR counter incrementa el valor de counter en 1.
  • INCRBY counter 5 incrementa el valor de counter en 5.
  • DECR counter decrementa el valor de counter en 1.
  • DECRBY counter 3 decrementa el valor de counter en 3.

Operacions avançades amb cadenes

Redis també suporta operacions més avançades amb cadenes, que poden ser útils en diverses situacions.

Establir amb caducitat

  • SETEX: Estableix el valor d'una clau amb un temps de caducitat.
  • PSETEX: Estableix el valor d'una clau amb un temps de caducitat en mil·lisegons.
SETEX tempkey 10 "This will expire in 10 seconds"
GET tempkey

Explicació:

  • SETEX tempkey 10 "This will expire in 10 seconds" estableix el valor de tempkey i fa que caduqui després de 10 segons.

Establir condicionalment

  • SETNX: Estableix el valor d'una clau només si la clau no existeix.
SETNX newkey "This will only be set if newkey does not exist"
GET newkey

Explicació:

  • SETNX newkey "This will only be set if newkey does not exist" estableix el valor de newkey només si newkey no existeix.

Obtenir i establir en una sola comanda

  • GETSET: Estableix el valor d'una clau i retorna el seu valor antic.
SET mykey "Initial value"
GETSET mykey "New value"
GET mykey

Explicació:

  • GETSET mykey "New value" estableix mykey a "New value" i retorna el valor antic "Initial value".

Resum

Les cadenes de Redis són versàtils i poden emmagatzemar una àmplia gamma de tipus de dades. Suporten diverses operacions, des de l'establiment i obtenció bàsics fins a funcions més avançades com l'establiment condicional i la caducitat. Entendre aquestes operacions és crucial per utilitzar Redis de manera efectiva en les teves aplicacions.

En dominar aquestes comandes, pots gestionar i manipular dades de cadena de manera eficient a Redis, fent-lo una eina poderosa per a les teves necessitats de desenvolupament.
