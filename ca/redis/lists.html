Introducció a les Llistes de Redis

Les Llistes de Redis són una de les estructures de dades més simples i poderoses disponibles a Redis. Són essencialment llistes enllaçades de cadenes, ordenades per ordre d'inserció. Aquesta secció cobrirà els conceptes bàsics de les Llistes de Redis, incloent com crear, manipular i recuperar dades de les llistes.

Conceptes Clau

  • Llista: Una col·lecció ordenada de cadenes.
  • Ordre d'Inserció: Els elements es guarden en l'ordre en què són inserits.
  • Operacions: Les operacions comunes inclouen afegir, eliminar i recuperar elements.

Creació i Gestió de Llistes

Afegir Elements a una Llista

Pots afegir elements a una llista de Redis utilitzant les comandes LPUSH i RPUSH. LPUSH afegeix elements al cap (esquerra) de la llista, mentre que RPUSH afegeix elements a la cua (dreta).

# Afegir elements al cap de la llista
LPUSH mylist "world"
LPUSH mylist "hello"

# Afegir elements a la cua de la llista
RPUSH mylist "!"

Recuperar Elements d'una Llista

Per recuperar elements d'una llista, pots utilitzar la comanda LRANGE, que retorna un rang especificat d'elements.

# Recuperar tots els elements de la llista
LRANGE mylist 0 -1
# Sortida: 1) "hello" 2) "world" 3) "!"

Eliminar Elements d'una Llista

Pots eliminar elements d'una llista utilitzant les comandes LPOP i RPOP. LPOP elimina el primer element, mentre que RPOP elimina l'últim element.

# Eliminar el primer element
LPOP mylist
# Sortida: "hello"

# Eliminar l'últim element
RPOP mylist
# Sortida: "!"

Operacions Avançades amb Llistes

Retallar una Llista

La comanda LTRIM et permet mantenir només un rang especificat d'elements en una llista, efectivament retallant la llista.

# Mantenir només els dos primers elements
LTRIM mylist 0 1

Inserir Elements en Posicions Específiques

La comanda LINSERT et permet inserir elements abans o després d'un element pivot especificat.

# Inserir "there" després de "hello"
LINSERT mylist AFTER "hello" "there"

Operacions de Bloqueig

Redis proporciona versions de bloqueig de les operacions de pop, BLPOP i BRPOP, que bloquegen la connexió fins que un altre client afegeix a la llista o s'arriba a un temps d'espera.

# Bloquejar pop del cap de la llista amb un temps d'espera de 5 segons
BLPOP mylist 5

Usos de les Llistes de Redis

Les Llistes de Redis són versàtils i poden ser utilitzades en diverses situacions, com ara:

  • Cues de Missatges: Utilitzant LPUSH per encolar i RPOP per desencolar.
  • Programació de Tasques: Emmagatzemant tasques en una llista i processant-les en ordre.
  • Registres: Afegint entrades de registre a una llista per a la seva posterior recuperació i anàlisi.

Resum

Les Llistes de Redis són una estructura de dades poderosa i flexible que pot ser utilitzada per a una àmplia gamma d'aplicacions. Suporten una varietat d'operacions per afegir, eliminar i recuperar elements, fent-les adequades per a tasques com la cua de missatges, la programació de tasques i el registre. Entendre com utilitzar efectivament les Llistes de Redis pot millorar enormement la teva capacitat per gestionar col·leccions ordenades de dades a Redis.
