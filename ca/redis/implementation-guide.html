Introducció a Redis

Redis és un magatzem de dades de codi obert, en memòria, utilitzat com a base de dades, memòria cau i intermediari de missatges. Admet diverses estructures de dades com cadenes, hashes, llistes, conjunts i més. Aquesta guia et portarà des dels conceptes bàsics de Redis fins a tècniques d'implementació avançades.

Configuració de Redis

Instal·lació

  • Linux: 
    sudo apt update
sudo apt install redis-server
  • macOS: 
    brew install redis
  • Windows: Utilitza el Subsistema de Windows per a Linux (WSL) o Docker.

Iniciant Redis

  • Linux/macOS: 
    redis-server
  • Windows (utilitzant WSL): 
    redis-server

Connectant-se a Redis

  • Utilitzant Redis CLI: 
    redis-cli

Comandes bàsiques de Redis

Cadenes

  • Establir un valor de cadena: 
    SET key "value"
  • Obtenir un valor de cadena: 
    GET key

Llistes

  • Afegir a una llista: 
    LPUSH mylist "value1"
LPUSH mylist "value2"
  • Recuperar d'una llista: 
    LRANGE mylist 0 -1

Hashes

  • Establir un camp de hash: 
    HSET myhash field1 "value1"
  • Obtenir un camp de hash: 
    HGET myhash field1

Conjunts

  • Afegir a un conjunt: 
    SADD myset "value1"
SADD myset "value2"
  • Obtenir tots els membres d'un conjunt: 
    SMEMBERS myset

Conceptes intermedis de Redis

Expiració i TTL

  • Establir una clau amb una expiració: 
    SETEX key 10 "value"  # La clau expira en 10 segons
  • Comprovar el TTL d'una clau: 
    TTL key

Transaccions

  • Iniciar una transacció: 
    MULTI
  • Executar una transacció: 
    EXEC

Pub/Sub

  • Publicar un missatge: 
    PUBLISH channel "message"
  • Subscriure's a un canal: 
    SUBSCRIBE channel

Tècniques avançades de Redis

Persistència de Redis

  • RDB (Còpia de seguretat de la base de dades Redis): Instantànies periòdiques del teu conjunt de dades.
  • AOF (Fitxer només d'append): Registra cada operació d'escriptura rebuda pel servidor.

Clúster de Redis

  • Configurar un clúster de Redis: 
    redis-cli --cluster create 127.0.0.1:7000 127.0.0.1:7001 127.0.0.1:7002 --cluster-replicas 1

Scripting en Lua

  • Executar un script en Lua: 
    EVAL "return redis.call('SET', KEYS[1], ARGV[1])" 1 key value

Conclusió

Redis és una eina poderosa per gestionar estructures de dades en memòria, oferint alt rendiment i flexibilitat. Des de comandes bàsiques fins a clústers i scripting avançats, dominar Redis pot millorar significativament el rendiment i l'escalabilitat de la teva aplicació. Practica les comandes i conceptes discutits en aquesta guia per convertir-te en un expert en Redis.
