RedisAI és un mòdul de Redis dissenyat per executar models d'aprenentatge profund i gestionar tot el cicle de vida dels models d'IA dins de Redis. Permet executar models dels marcs d'aprenentatge profund més populars, com TensorFlow, PyTorch i ONNX, directament dins de Redis, permetent inferències d'IA amb baixa latència i alt rendiment.

Introducció a RedisAI

  • Què és RedisAI?

    • RedisAI és un mòdul de Redis que permet executar i gestionar models d'IA dins de Redis.
    • Admet múltiples marcs d'aprenentatge profund, incloent TensorFlow, PyTorch i ONNX.
    • RedisAI proporciona una manera senzilla d'integrar capacitats d'IA a les teves aplicacions basades en Redis.

  • Per què utilitzar RedisAI?

    • Baixa Latència: RedisAI aprofita l'estructura de dades en memòria de Redis per proporcionar inferències d'IA amb baixa latència.
    • Escalabilitat: RedisAI pot escalar horitzontalment amb Redis, fent-lo adequat per a aplicacions d'alt rendiment.
    • Flexibilitat: Admet múltiples marcs d'IA, permetent-te triar l'eina que millor s'adapti a les teves necessitats.

Configuració de RedisAI

  • Prerequisits:

    • Una instal·lació funcional de Redis.
    • Docker (opcional però recomanat per facilitar la configuració).

  • Passos d'instal·lació:

    1. Utilitzant Docker: 
      docker run -p 6379:6379 --name redisai redislabs/redisai
    2. Construint des del codi font:
      • Clona el repositori de RedisAI des de GitHub.
      • Segueix les instruccions de construcció proporcionades al repositori.

  • Carregant el mòdul RedisAI:

    redis-server --loadmodule /path/to/redisai.so

Operacions bàsiques amb RedisAI

  • Carregant un model:

    • RedisAI et permet carregar models a Redis i executar-los.
    • Exemple: Carregant un model de TensorFlow. 
      import redis

r = redis.Redis()

with open('model.pb', 'rb') as f:
    model = f.read()

r.execute_command('AI.MODELSET', 'mymodel', 'TF', 'CPU', 'INPUTS', 'input', 'OUTPUTS', 'output', model)

  • Executant inferència:

    • Exemple: Executant inferència utilitzant el model carregat. 
      import numpy as np

input_data = np.random.rand(1, 3, 224, 224).astype(np.float32).tobytes()
r.execute_command('AI.TENSORSET', 'input_tensor', 'FLOAT', 1, 3, 224, 224, 'BLOB', input_data)
r.execute_command('AI.MODELRUN', 'mymodel', 'INPUTS', 'input_tensor', 'OUTPUTS', 'output_tensor')
output = r.execute_command('AI.TENSORGET', 'output_tensor', 'BLOB')

Funcions avançades de RedisAI

  • Gestió de scripts:

    • RedisAI admet l'execució de scripts personalitzats utilitzant el runtime de TorchScript o ONNX.
    • Exemple: Carregant i executant un TorchScript. 
      script = """
def forward(input):
    return input + 1
"""
r.execute_command('AI.SCRIPTSET', 'myscript', 'CPU', script)
r.execute_command('AI.SCRIPTRUN', 'myscript', 'forward', 'INPUTS', 'input_tensor', 'OUTPUTS', 'output_tensor')

  • Versionat de models:

    • RedisAI admet el versionat de models, permetent-te gestionar múltiples versions del mateix model.
    • Exemple: Carregant una nova versió d'un model. 
      r.execute_command('AI.MODELSET', 'mymodel', 'TF', 'CPU', 'INPUTS', 'input', 'OUTPUTS', 'output', model, 'TAG', 'v2')

  • Monitorització i depuració:

    • RedisAI proporciona comandes per monitoritzar i depurar l'execució de models.
    • Exemple: Comprovant la informació del model. 
      redis-cli AI.MODELGET mymodel META

Casos d'ús i millors pràctiques

  • Casos d'ús:

    • Sistemes de recomanació en temps real.
    • Detecció de fraus.
    • Processament d'imatges i vídeos.
    • Processament del llenguatge natural.

  • Millors pràctiques:

    • Optimitzar Models: Assegura't que els models estan optimitzats per a la inferència per reduir la latència.
    • Gestió de Recursos: Monitoritza i gestiona els recursos per evitar colls d'ampolla.
    • Control de Versions: Utilitza el versionat de models per gestionar actualitzacions i reversions.

Conclusió

RedisAI amplia les capacitats de Redis integrant l'execució de models d'IA directament dins de la base de dades. Això permet aplicacions d'IA amb baixa latència i alt rendiment que poden escalar amb Redis. Entenent els conceptes bàsics de RedisAI, configurant-lo i aprofitant les seves funcions avançades, pots construir aplicacions potents impulsades per IA que es beneficiïn del rendiment i l'escalabilitat de Redis.

RedisAI és una eina potent per a desenvolupadors que busquen integrar IA a les seves aplicacions basades en Redis. Amb el seu suport per a múltiples marcs i funcions avançades, RedisAI proporciona una manera flexible i eficient de desplegar i gestionar models d'IA.
