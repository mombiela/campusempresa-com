En aquesta secció, cobrirem els passos necessaris per instal·lar Redis en diferents sistemes operatius. Redis és un magatzem de dades en memòria, utilitzat com a base de dades, memòria cau i intermediari de missatges. Una instal·lació adequada és el primer pas per utilitzar les seves potents característiques.

Instal·lant Redis a Linux

Utilitzant el Gestor de Paquets (APT per a Ubuntu/Debian)

  1. Actualitza la teva llista de paquets: 
    sudo apt update
  2. Instal·la Redis: 
    sudo apt install redis-server
  3. Inicia el servei Redis: 
    sudo systemctl start redis
  4. Habilita Redis per iniciar-se en arrencar: 
    sudo systemctl enable redis

Utilitzant el Gestor de Paquets (YUM per a CentOS/RHEL)

  1. Afegeix el repositori EPEL: 
    sudo yum install epel-release
  2. Instal·la Redis: 
    sudo yum install redis
  3. Inicia el servei Redis: 
    sudo systemctl start redis
  4. Habilita Redis per iniciar-se en arrencar: 
    sudo systemctl enable redis

Compilant des del Codi Font

  1. Instal·la les dependències de compilació: 
    sudo apt update
sudo apt install build-essential tcl
  2. Descarrega l'última versió estable de Redis: 
    curl -O http://download.redis.io/redis-stable.tar.gz
  3. Extreu l'arxiu tarball: 
    tar xzvf redis-stable.tar.gz
  4. Compila Redis: 
    cd redis-stable
make
  5. Executa el conjunt de proves de Redis: 
    make test
  6. Instal·la Redis: 
    sudo make install

Instal·lant Redis a macOS

Utilitzant Homebrew

  1. Instal·la Homebrew si encara no ho has fet: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instal·la Redis: 
    brew install redis
  3. Inicia el servei Redis: 
    brew services start redis

Instal·lant Redis a Windows

Utilitzant WSL (Subsistema Windows per a Linux)

  1. Habilita WSL i instal·la una distribució de Linux (per exemple, Ubuntu) des de la Microsoft Store.
  2. Obre el terminal WSL i segueix els passos d'instal·lació de Linux esmentats anteriorment.

Utilitzant Redis per a Windows

  1. Descarrega l'últim arxiu zip de Redis des de l'arxiu de Microsoft.
  2. Extreu l'arxiu zip a un directori de la teva elecció.
  3. Executa el servidor Redis: 
    redis-server.exe

Verificant la Instal·lació

Després d'instal·lar Redis, pots verificar la instal·lació executant el CLI de Redis i executant una comanda simple.

  1. Obre el teu terminal o símbol del sistema.
  2. Executa el CLI de Redis: 
    redis-cli
  3. Executa una comanda simple: 
    ping
    • Hauries de veure la resposta: 
      PONG

Conclusió

En aquesta secció, hem cobert els passos per instal·lar Redis en diversos sistemes operatius, incloent Linux, macOS i Windows. Ja sigui que utilitzis un gestor de paquets, compilis des del codi font o utilitzis un mètode específic per a Windows, el procés és senzill i assegura que tinguis una instància de Redis en funcionament, llesta per a una exploració i desenvolupament posteriors. En les següents seccions, aprofundirem en la configuració i ús efectiu de Redis.
