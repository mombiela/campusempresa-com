Redis és un magatzem de estructures de dades en memòria utilitzat com a base de dades, memòria cau i intermediari de missatges. Entendre els tipus de dades bàsics en Redis és crucial per aprofitar tot el seu potencial. Aquesta secció cobrirà els tipus de dades fonamentals en Redis, incloent Cadenes, Llistes, Conjunts, Hashes i Conjunts Ordenats.

Cadenes

Les cadenes són el tipus de valor més bàsic que pots emmagatzemar en Redis. Són segures per a binaris, el que significa que poden contenir qualsevol dada, com una imatge JPEG o un objecte Ruby serialitzat.

  • Conceptes clau:

    • Les cadenes poden emmagatzemar fins a 512 MB de dades.
    • Les operacions comunes inclouen establir, obtenir i afegir valors.

  • Exemples:

SET mykey "Hello, Redis!"
GET mykey

Explicació:

  • SET mykey "Hello, Redis!" emmagatzema la cadena "Hello, Redis!" sota la clau mykey.
  • GET mykey recupera el valor emmagatzemat a mykey.

Llistes

Les llistes són col·leccions de valors ordenats. S'implementen com a llistes enllaçades, el que significa que inserir un element al cap o a la cua és molt eficient.

  • Conceptes clau:

    • Les llistes poden contenir fins a 2^32 - 1 elements (més de 4 mil milions).
    • Les operacions comunes inclouen empènyer i treure elements de tots dos extrems.

  • Exemples:

LPUSH mylist "World"
LPUSH mylist "Hello"
LRANGE mylist 0 -1

Explicació:

  • LPUSH mylist "World" empènyer "World" a la llista mylist.
  • LPUSH mylist "Hello" empènyer "Hello" a la llista mylist.
  • LRANGE mylist 0 -1 recupera tots els elements de la llista mylist.

Conjunts

Els conjunts són col·leccions de valors únics i desordenats. Són útils per emmagatzemar elements únics i realitzar operacions de conjunt com unions i interseccions.

  • Conceptes clau:

    • Els conjunts no permeten valors duplicats.
    • Les operacions comunes inclouen afegir, eliminar i comprovar la pertinença.

  • Exemples:

SADD myset "Hello"
SADD myset "World"
SADD myset "Hello"  # Això no afegirà "Hello" de nou
SMEMBERS myset

Explicació:

  • SADD myset "Hello" afegeix "Hello" al conjunt myset.
  • SADD myset "World" afegeix "World" al conjunt myset.
  • SMEMBERS myset recupera tots els membres del conjunt myset.

Hashes

Els hashes són mapes entre camps de cadena i valors de cadena. Són perfectes per representar objectes.

  • Conceptes clau:

    • Els hashes són ideals per emmagatzemar petites peces de dades.
    • Les operacions comunes inclouen establir, obtenir i eliminar camps.

  • Exemples:

HSET myhash field1 "Hello"
HSET myhash field2 "World"
HGET myhash field1
HGETALL myhash

Explicació:

  • HSET myhash field1 "Hello" estableix el camp field1 en el hash myhash a "Hello".
  • HSET myhash field2 "World" estableix el camp field2 en el hash myhash a "World".
  • HGET myhash field1 recupera el valor de field1 en el hash myhash.
  • HGETALL myhash recupera tots els camps i valors en el hash myhash.

Conjunts Ordenats

Els conjunts ordenats són similars als conjunts però amb una puntuació associada per a cada membre. Els membres estan ordenats per les seves puntuacions.

  • Conceptes clau:

    • Els conjunts ordenats són útils per a sistemes de classificació.
    • Les operacions comunes inclouen afegir elements amb puntuacions, recuperar elements per puntuació i eliminar elements.

  • Exemples:

ZADD myzset 1 "one"
ZADD myzset 2 "two"
ZRANGE myzset 0 -1 WITHSCORES

Explicació:

  • ZADD myzset 1 "one" afegeix el membre "one" amb una puntuació d'1 al conjunt ordenat myzset.
  • ZADD myzset 2 "two" afegeix el membre "two" amb una puntuació de 2 al conjunt ordenat myzset.
  • ZRANGE myzset 0 -1 WITHSCORES recupera tots els membres en el conjunt ordenat myzset juntament amb les seves puntuacions.

Conclusió

Entendre els tipus de dades bàsics en Redis és essencial per utilitzar eficaçment aquest potent magatzem de estructures de dades en memòria. Cada tipus de dada serveix per a diferents casos d'ús, des de l'emmagatzematge simple de clau-valor amb Cadenes fins a estructures de dades més complexes com Llistes, Conjunts, Hashes i Conjunts Ordenats. Dominar aquests conceptes bàsics proporcionarà una base sòlida per a funcions i operacions més avançades de Redis.
