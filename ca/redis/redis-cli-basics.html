Introducció

Redis CLI (Interfície de Línia de Comandes) és una eina potent que et permet interactuar amb el teu servidor Redis directament des de la línia de comandes. Aquesta secció cobrirà els conceptes bàsics d'ús de Redis CLI, incloent la connexió a un servidor Redis, l'execució de comandes i la comprensió de la sortida.

Connexió al servidor Redis

Per començar a utilitzar Redis CLI, primer has de connectar-te al teu servidor Redis. Això es fa típicament utilitzant la comanda redis-cli seguida de l'adreça i el port del servidor.

Exemple

redis-cli -h 127.0.0.1 -p 6379
  • -h: Especifica l'amfitrió (per defecte és 127.0.0.1).
  • -p: Especifica el port (per defecte és 6379).

Comandes bàsiques

Un cop connectat, pots començar a executar comandes Redis. Aquí tens algunes de les comandes més utilitzades:

PING

La comanda PING comprova si el servidor està en funcionament.

PING
  • Sortida esperada: PONG

SET i GET

La comanda SET emmagatzema un parell clau-valor, i la comanda GET recupera el valor d'una clau.

SET mykey "Hello, Redis!"
GET mykey
  • Sortida esperada per GET mykey: "Hello, Redis!"

DEL

La comanda DEL elimina una clau.

DEL mykey
  • Sortida esperada: (integer) 1 (si la clau existia i ha estat eliminada)

Treballant amb tipus de dades

Redis suporta diversos tipus de dades, incloent cadenes, llistes, conjunts, hashes i conjunts ordenats. Aquí tens algunes comandes bàsiques per a cada tipus:

Cadenes

SET key1 "value1"
GET key1

Llistes

LPUSH mylist "item1"
LPUSH mylist "item2"
LRANGE mylist 0 -1
  • LPUSH: Afegeix un element a la llista.
  • LRANGE: Recupera elements de la llista.

Conjunts

SADD myset "member1"
SADD myset "member2"
SMEMBERS myset
  • SADD: Afegeix un membre al conjunt.
  • SMEMBERS: Recupera tots els membres del conjunt.

Hashes

HSET myhash field1 "value1"
HGET myhash field1
  • HSET: Estableix el valor d'un camp en un hash.
  • HGET: Recupera el valor d'un camp en un hash.

Conjunts ordenats

ZADD myzset 1 "member1"
ZADD myzset 2 "member2"
ZRANGE myzset 0 -1 WITHSCORES
  • ZADD: Afegeix un membre amb una puntuació al conjunt ordenat.
  • ZRANGE: Recupera membres en un rang especificat.

Opcions útils de CLI

Redis CLI ofereix diverses opcions per millorar la teva experiència:

Mode interactiu

Inicia Redis CLI en mode interactiu simplement executant:

redis-cli
  • Aquest mode et permet introduir comandes una a una i veure els resultats immediatament.

Mode no interactiu

També pots executar comandes directament des de la shell sense entrar en mode interactiu:

redis-cli SET key "value"
  • Això és útil per a la creació de scripts i l'automatització.

Pipelining

Per executar múltiples comandes en una sola sol·licitud, utilitza pipelining:

redis-cli --pipe
  • Això pot millorar significativament el rendiment per a operacions en lot.

Conclusió

Redis CLI és una eina essencial per gestionar i interactuar amb el teu servidor Redis. Comprenent les comandes bàsiques i les opcions, pots realitzar diverses operacions de manera eficient i resoldre problemes. Practica utilitzant aquestes comandes per sentir-te més còmode amb Redis CLI, i estaràs ben preparat per abordar temes més avançats en Redis.
