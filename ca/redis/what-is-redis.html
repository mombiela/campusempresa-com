Introducció a Redis

Redis, que significa Remote Dictionary Server (Servidor de Diccionari Remot), és un magatzem de dades de codi obert i en memòria. S'utilitza com a base de dades, memòria cau i intermediari de missatges. Redis admet diverses estructures de dades com ara cadenes, hash, llistes, conjunts i més.

Característiques Clau de Redis

  • Emmagatzematge en Memòria: Redis emmagatzema dades en memòria, cosa que el fa extremadament ràpid per a operacions de lectura i escriptura.
  • Persistència: Tot i ser una base de dades en memòria, Redis ofereix opcions de persistència, permetent que les dades es guardin al disc.
  • Estructures de Dades: Redis admet una varietat d'estructures de dades, incloent cadenes, hash, llistes, conjunts i conjunts ordenats.
  • Operacions Atòmiques: Totes les operacions de Redis són atòmiques, assegurant la consistència de les dades.
  • Replicació: Redis admet la replicació mestre-esclau, que es pot utilitzar per a la redundància de dades i alta disponibilitat.
  • Missatgeria Pub/Sub: Redis inclou un paradigma de missatgeria de publicació/subscripció, útil per a aplicacions de missatgeria en temps real.
  • Scripting Lua: Redis permet executar scripts Lua per a operacions complexes.
  • Transaccions: Redis admet transaccions a través dels comandaments MULTI, EXEC, DISCARD i WATCH.

Per què Utilitzar Redis?

Redis és àmpliament utilitzat per a diversos propòsits gràcies a la seva velocitat i versatilitat. Aquí hi ha alguns casos d'ús comuns:

  • Memòria Cau: Redis s'utilitza sovint per emmagatzemar en memòria cau dades accedides freqüentment per reduir la latència i millorar el rendiment.
  • Emmagatzematge de Sessions: Les aplicacions web utilitzen Redis per emmagatzemar dades de sessió, proporcionant accés ràpid i persistència.
  • Anàlisi en Temps Real: Redis pot gestionar la ingesta de dades a alta velocitat i l'anàlisi en temps real.
  • Cues de Missatges: Les característiques de llistes i pub/sub de Redis el fan adequat per implementar cues de missatges.
  • Classificacions/Comptatge: Els conjunts ordenats de Redis són ideals per crear classificacions i comptadors.

Conceptes i Comandaments Bàsics

Tipus de Dades

Redis admet diversos tipus de dades, cadascun amb comandaments específics:

  • Cadenes: El tipus més senzill, utilitzat per emmagatzemar text o dades binàries.
  • Hash: Mapes entre camps de cadena i valors de cadena, útils per representar objectes.
  • Llistes: Col·leccions ordenades de cadenes, permetent la inserció i eliminació des de tots dos extrems.
  • Conjunts: Col·leccions no ordenades de cadenes úniques.
  • Conjunts Ordenats: Similar als conjunts però amb una puntuació associada a cada membre, permetent l'ordenació.

Comandaments Bàsics

Aquí hi ha alguns comandaments bàsics de Redis amb exemples:

Cadenes

SET key "value"   # Estableix el valor d'una clau
GET key           # Obté el valor d'una clau
DEL key           # Elimina una clau

Hash

HSET user:1 name "Alice"   # Estableix un camp en un hash
HGET user:1 name           # Obté un camp d'un hash
HDEL user:1 name           # Elimina un camp d'un hash

Llistes

LPUSH mylist "item1"   # Empeny un element a la llista (esquerra)
RPUSH mylist "item2"   # Empeny un element a la llista (dreta)
LPOP mylist            # Extreu un element de la llista (esquerra)
RPOP mylist            # Extreu un element de la llista (dreta)

Conjunts

SADD myset "item1"   # Afegeix un element al conjunt
SREM myset "item1"   # Elimina un element del conjunt
SMEMBERS myset       # Obté tots els membres del conjunt

Conjunts Ordenats

ZADD myzset 1 "item1"   # Afegeix un element amb una puntuació al conjunt ordenat
ZREM myzset "item1"     # Elimina un element del conjunt ordenat
ZRANGE myzset 0 -1      # Obté tots els elements del conjunt ordenat

Conclusió

Redis és un magatzem de dades en memòria potent i versàtil que es pot utilitzar per a una varietat de propòsits, des de la memòria cau i l'emmagatzematge de sessions fins a l'anàlisi en temps real i les cues de missatges. El seu suport per a múltiples estructures de dades i operacions atòmiques el converteix en una eina valuosa per als desenvolupadors que busquen construir aplicacions d'alt rendiment. Entendre els conceptes i comandaments bàsics de Redis és el primer pas per aprofitar tot el seu potencial en els teus projectes.
