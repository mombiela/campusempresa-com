Introducció als Conjunts Ordenats

Els conjunts ordenats a Redis són una estructura de dades potent que combina les característiques d'un conjunt i una llista ordenada. Cada element en un conjunt ordenat està associat amb una puntuació, que s'utilitza per ordenar els elements en ordre ascendent. Això fa que els conjunts ordenats siguin ideals per a escenaris on necessites mantenir una col·lecció ordenada d'elements únics.

Conceptes Clau

  • Unicitat: Cada membre en un conjunt ordenat és únic.
  • Puntuació: Cada membre està associat amb una puntuació, que és un nombre de punt flotant.
  • Ordenació: Els membres estan ordenats per les seves puntuacions en ordre ascendent.
  • Consultes per Rang: Recupera eficientment un rang d'elements basat en les seves puntuacions o rangs.

Operacions Bàsiques

Afegir Elements

Per afegir elements a un conjunt ordenat, utilitzes la comanda ZADD. Aquesta comanda requereix el nom del conjunt ordenat, la puntuació i el membre.

ZADD mysortedset 1 "element1"
ZADD mysortedset 2 "element2"
ZADD mysortedset 3 "element3"

Recuperar Elements

Pots recuperar elements d'un conjunt ordenat utilitzant diverses comandes:

  • ZRANGE: Obtenir elements pel seu rang.
ZRANGE mysortedset 0 -1
  • ZRANGEBYSCORE: Obtenir elements per la seva puntuació.
ZRANGEBYSCORE mysortedset 1 2
  • ZREVRANGE: Obtenir elements pel seu rang en ordre invers.
ZREVRANGE mysortedset 0 -1

Eliminar Elements

Per eliminar elements d'un conjunt ordenat, utilitzes la comanda ZREM.

ZREM mysortedset "element1"

Operacions Avançades

Incrementar Puntuacions

Pots incrementar la puntuació d'un element utilitzant la comanda ZINCRBY.

ZINCRBY mysortedset 2 "element2"

Comptar Elements

Per comptar el nombre d'elements dins d'un rang de puntuació específic, utilitza la comanda ZCOUNT.

ZCOUNT mysortedset 1 3

Eliminar Elements per Puntuació o Rang

Pots eliminar elements basant-te en la seva puntuació o rang utilitzant les comandes ZREMRANGEBYSCORE i ZREMRANGEBYRANK.

ZREMRANGEBYSCORE mysortedset 1 2
ZREMRANGEBYRANK mysortedset 0 1

Casos d'Ús

Classificacions

Els conjunts ordenats són ideals per implementar classificacions on necessites fer un seguiment de puntuacions i rangs.

ZADD leaderboard 100 "user1"
ZADD leaderboard 200 "user2"
ZADD leaderboard 150 "user3"

ZRANGE leaderboard 0 -1 WITHSCORES

Programació de Tasques

Pots utilitzar conjunts ordenats per programar tasques associant cada tasca amb una marca de temps.

ZADD tasks 1633036800 "task1"
ZADD tasks 1633123200 "task2"

ZRANGEBYSCORE tasks 1633036800 1633123200

Resum

Els conjunts ordenats a Redis són una estructura de dades versàtil que et permet mantenir una col·lecció d'elements únics ordenats per puntuacions. Admeten una àmplia gamma d'operacions, des d'afegir i eliminar elements fins a realitzar consultes complexes per rang. Els casos d'ús comuns inclouen classificacions i programació de tasques, fent dels conjunts ordenats una eina valuosa per a moltes aplicacions.

En dominar els conjunts ordenats, pots aprofitar Redis per gestionar col·leccions ordenades de dades de manera eficient, millorant el rendiment i les capacitats de les teves aplicacions.
