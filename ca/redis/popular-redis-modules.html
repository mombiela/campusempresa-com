Redis és un potent magatzem de dades en memòria, sovint utilitzat com a base de dades, memòria cau i intermediari de missatges. Per ampliar les seves capacitats, Redis ofereix diversos mòduls que afegeixen nous tipus de dades i funcionalitats. En aquesta secció, explorarem alguns dels mòduls més populars de Redis, les seves característiques i com utilitzar-los.

RedisJSON

RedisJSON és un mòdul que et permet emmagatzemar, actualitzar i recuperar valors JSON a Redis. Proporciona emmagatzematge i recuperació eficients de documents JSON.

Característiques Clau

  • Emmagatzematge Eficient: Emmagatzema documents JSON en un format binari per a un emmagatzematge i recuperació eficients.
  • Consulta Rica: Suporta capacitats de consulta complexes, incloent-hi consultes nidificades.
  • Operacions Atòmiques: Permet actualitzacions atòmiques de documents JSON.

Exemple d'Ús

# Suposant que tens els paquets redis-py i redisjson instal·lats
import redis
from redis.commands.json.path import Path

# Connectar-se a Redis
r = redis.Redis(host='localhost', port=6379)

# Establir un document JSON
r.json().set('user:1000', Path.root_path(), {
    'name': 'John Doe',
    'age': 30,
    'address': {
        'city': 'New York',
        'state': 'NY'
    }
})

# Obtenir un document JSON
user = r.json().get('user:1000')
print(user)

# Actualitzar un document JSON
r.json().set('user:1000', Path('.age'), 31)

RedisGraph

RedisGraph és un mòdul de base de dades de gràfics per a Redis, que et permet emmagatzemar i consultar estructures de dades de gràfics.

Característiques Clau

  • Llenguatge de Consulta Cypher: Utilitza el llenguatge de consulta Cypher, que també és utilitzat per Neo4j.
  • Alta Rendiment: Optimitzat per a rendiment amb processament de gràfics en memòria.
  • Model de Dades Ric: Suporta nodes, arestes i propietats.

Exemple d'Ús

# Suposant que tens els paquets redis-py i redisgraph instal·lats
import redis
from redisgraph import Graph, Node, Edge

# Connectar-se a Redis
r = redis.Redis(host='localhost', port=6379)
graph = Graph('social', r)

# Crear nodes
john = Node(label='person', properties={'name': 'John Doe', 'age': 30})
jane = Node(label='person', properties={'name': 'Jane Doe', 'age': 25})
graph.add_node(john)
graph.add_node(jane)

# Crear una aresta
edge = Edge(john, 'knows', jane)
graph.add_edge(edge)

# Cometre el gràfic
graph.commit()

# Consultar el gràfic
query = "MATCH (p:person) WHERE p.age > 20 RETURN p.name, p.age"
result = graph.query(query)
for record in result.result_set:
    print(record)

RedisTimeSeries

RedisTimeSeries és un mòdul dissenyat per gestionar dades de sèries temporals amb alt rendiment i facilitat d'ús.

Característiques Clau

  • Emmagatzematge Eficient: Optimitzat per emmagatzemar dades de sèries temporals.
  • Agregació: Suporta diverses funcions d'agregació com mitjana, suma, mínim, màxim, etc.
  • Mostreig: Permet el mostreig de dades per a un emmagatzematge eficient.

Exemple d'Ús

# Suposant que tens els paquets redis-py i redistimeseries instal·lats
import redis
from redistimeseries.client import Client

# Connectar-se a Redis
r = redis.Redis(host='localhost', port=6379)
ts = Client(r)

# Crear una sèrie temporal
ts.create('temperature:room1')

# Afegir punts de dades
ts.add('temperature:room1', 1609459200000, 22.5)  # Temps en mil·lisegons
ts.add('temperature:room1', 1609459260000, 23.0)

# Consultar la sèrie temporal
data = ts.range('temperature:room1', 1609459200000, 1609459260000)
print(data)

RedisBloom

RedisBloom és un mòdul que proporciona estructures de dades probabilístiques com filtres de Bloom, filtres de Cuckoo i Count-Min Sketch.

Característiques Clau

  • Estructures de Dades Probabilístiques: Gestionen eficientment grans conjunts de dades amb garanties probabilístiques.
  • Baix Ús de Memòria: Utilitza menys memòria en comparació amb les estructures de dades tradicionals.
  • Alt Rendiment: Optimitzat per a alt rendiment.

Exemple d'Ús

# Suposant que tens els paquets redis-py i redisbloom instal·lats
import redis
from redisbloom.client import Client

# Connectar-se a Redis
r = redis.Redis(host='localhost', port=6379)
bloom = Client(r)

# Crear un filtre de Bloom
bloom.bfCreate('usernames', 0.01, 1000)

# Afegir elements al filtre de Bloom
bloom.bfAdd('usernames', 'user1')
bloom.bfAdd('usernames', 'user2')

# Comprovar l'existència
print(bloom.bfExists('usernames', 'user1'))  # True
print(bloom.bfExists('usernames', 'user3'))  # False

Conclusió

Els mòduls de Redis amplien les funcionalitats bàsiques de Redis, convertint-lo en una eina versàtil per a diversos casos d'ús. RedisJSON, RedisGraph, RedisTimeSeries i RedisBloom són alguns dels mòduls més populars que afegeixen característiques potents per gestionar documents JSON, dades de gràfics, dades de sèries temporals i estructures de dades probabilístiques, respectivament. Aprofitant aquests mòduls, pots construir aplicacions més complexes i eficients amb Redis.
