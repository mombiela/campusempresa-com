Introducció

Els sistemes distribuïts són una arquitectura en la qual múltiples components ubicats en diferents màquines treballen junts per assolir un objectiu comú. Aquesta arquitectura ofereix diverses avantatges, però també presenta desafiaments significatius. En aquesta secció, explorarem ambdós aspectes per proporcionar una comprensió completa dels sistemes distribuïts.

Avantatges dels Sistemes Distribuïts

Escalabilitat

  • Horitzontal i Vertical: Els sistemes distribuïts permeten escalar tant horitzontalment (afegint més màquines) com verticalment (millorant les capacitats de les màquines existents).
  • Exemple: Un sistema de microserveis pot escalar horitzontalment afegint més instàncies d'un servei específic per manejar una càrrega creixent.

Tolerància a Fallades

  • Redundància: La replicació de dades i serveis en múltiples nodes assegura que el sistema pugui continuar funcionant fins i tot si alguns nodes fallen.
  • Exemple: En un sistema d'emmagatzematge distribuït com HDFS, les dades es repliquen en diversos nodes per garantir la disponibilitat.

Flexibilitat i Manteniment

  • Desplegament Independent: Els components poden ser actualitzats, reemplaçats o escalats de manera independent.
  • Exemple: En una arquitectura de microserveis, cada servei pot ser desenvolupat i desplegat de manera independent, facilitant el manteniment i l'actualització.

Rendiment

  • Distribució de Càrrega: La càrrega de treball es distribueix entre múltiples nodes, millorant el rendiment general del sistema.
  • Exemple: Un clúster de servidors web pot distribuir les sol·licituds entrants entre diversos servidors per millorar el temps de resposta.

Desafiaments dels Sistemes Distribuïts

Complexitat

  • Coordinació i Sincronització: La coordinació entre múltiples nodes pot ser complexa i propensa a errors.
  • Exemple: Implementar un algorisme de consens com Paxos o Raft per assegurar la consistència de dades en un sistema distribuït.

Latència i Rendiment

  • Latència de Xarxa: La comunicació entre nodes pot introduir latència, afectant el rendiment del sistema.
  • Exemple: En un sistema de bases de dades distribuïdes, les consultes que requereixen dades de múltiples nodes poden ser més lentes a causa de la latència de xarxa.

Seguretat

  • Superfície d'Atac Ampliada: Més nodes i punts de comunicació signifiquen més vectors d'atac potencials.
  • Exemple: Assegurar la comunicació entre nodes mitjançant xifrat i autenticació per prevenir atacs d'intermediaris.

Consistència

  • Model CAP: Els sistemes distribuïts han d'equilibrar entre Consistència, Disponibilitat i Tolerància a Particions (CAP), cosa que pot ser un desafiament.
  • Exemple: En una base de dades distribuïda, triar entre consistència forta i disponibilitat pot dependre dels requisits específics de l'aplicació.

Comparació d'Avantatges i Desafiaments

Exemple de Codi

A continuació, es mostra un exemple simple d'un sistema distribuït utilitzant microserveis en Python amb Flask:

# service_a.py
from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/service_a')
def service_a():
    return jsonify({"message": "Hello from Service A!"})

if __name__ == '__main__':
    app.run(port=5000)

# service_b.py
from flask import Flask, jsonify
import requests

app = Flask(__name__)

@app.route('/service_b')
def service_b():
    response = requests.get('http://localhost:5000/service_a')
    return jsonify({"message": "Hello from Service B!", "service_a_response": response.json()})

if __name__ == '__main__':
    app.run(port=5001)

En aquest exemple:

  • Service A: Un microservei que respon amb un missatge simple.
  • Service B: Un altre microservei que crida a Service A i combina la seva resposta amb el seu propi missatge.

Conclusió

Els sistemes distribuïts ofereixen nombroses avantatges, com escalabilitat, tolerància a fallades, flexibilitat i millor rendiment. No obstant això, també presenten desafiaments significatius, incloent complexitat, latència, seguretat i consistència. Comprendre aquests aspectes és crucial per dissenyar i mantenir sistemes distribuïts efectius i eficients.
