Introducció

Llegir i escriure fitxers és una habilitat fonamental per a qualsevol programador Java. Aquest tema cobrirà els conceptes bàsics de la gestió de fitxers en Java, incloent la lectura de fitxers, l'escriptura de fitxers i la gestió d'excepcions. Al final d'aquesta secció, seràs capaç de llegir dades de fitxers i escriure dades a fitxers utilitzant Java.

Conceptes Clau

  • Classe File: Representa un fitxer o un camí de directori.
  • FileReader i FileWriter: Classes per llegir i escriure fitxers de text.
  • BufferedReader i BufferedWriter: Classes per a una lectura i escriptura eficient.
  • FileInputStream i FileOutputStream: Classes per llegir i escriure fitxers binaris.
  • Gestió d'Excepcions: Gestionar errors que es produeixen durant les operacions de fitxers.

Treballant amb la Classe File

La classe File en Java s'utilitza per representar noms de camins de fitxers i directoris.

import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");

        // Comprovar si el fitxer existeix
        if (file.exists()) {
            System.out.println("El fitxer existeix.");
        } else {
            System.out.println("El fitxer no existeix.");
        }
    }
}

Llegir de Fitxers

Utilitzant FileReader i BufferedReader

FileReader i BufferedReader s'utilitzen per llegir fitxers de text de manera eficient.

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Utilitzant FileInputStream

FileInputStream s'utilitza per llegir fitxers binaris.

import java.io.FileInputStream;
import java.io.IOException;

public class ReadBinaryFileExample {
    public static void main(String[] args) {
        try (FileInputStream fis = new FileInputStream("example.bin")) {
            int content;
            while ((content = fis.read()) != -1) {
                System.out.print((char) content);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Escriure a Fitxers

Utilitzant FileWriter i BufferedWriter

FileWriter i BufferedWriter s'utilitzen per escriure fitxers de text de manera eficient.

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("example.txt"))) {
            bw.write("Hola, Món!");
            bw.newLine();
            bw.write("Això és un fitxer de text.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Utilitzant FileOutputStream

FileOutputStream s'utilitza per escriure fitxers binaris.

import java.io.FileOutputStream;
import java.io.IOException;

public class WriteBinaryFileExample {
    public static void main(String[] args) {
        try (FileOutputStream fos = new FileOutputStream("example.bin")) {
            String content = "Hola, Món Binari!";
            fos.write(content.getBytes());
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Gestió d'Excepcions

Gestionar excepcions és crucial quan es treballa amb fitxers per assegurar que el programa pugui gestionar errors de manera elegant.

import java.io.File;
import java.io.FileNotFoundException;
import java.io.FileReader;
import java.io.IOException;

public class ExceptionHandlingExample {
    public static void main(String[] args) {
        File file = new File("example.txt");
        try (FileReader fr = new FileReader(file)) {
            int content;
            while ((content = fr.read()) != -1) {
                System.out.print((char) content);
            }
        } catch (FileNotFoundException e) {
            System.out.println("Fitxer no trobat: " + e.getMessage());
        } catch (IOException e) {
            System.out.println("Error d'E/S: " + e.getMessage());
        }
    }
}

Resum

En aquesta secció, hem cobert:

  • La classe File per representar camins de fitxers.
  • Llegir fitxers de text utilitzant FileReader i BufferedReader.
  • Llegir fitxers binaris utilitzant FileInputStream.
  • Escriure fitxers de text utilitzant FileWriter i BufferedWriter.
  • Escriure fitxers binaris utilitzant FileOutputStream.
  • Gestionar excepcions per gestionar errors durant les operacions de fitxers.

Entendre aquests conceptes et permetrà gestionar de manera efectiva l'entrada i sortida de fitxers en les teves aplicacions Java. Practica aquests exemples i intenta crear les teves pròpies operacions de gestió de fitxers per consolidar la teva comprensió.
