Què és la Programació Orientada a Objectes (POO)?

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Utilitza diversos conceptes clau per crear codi reutilitzable i modular.

  • Objectes: Instàncies de classes que representen entitats amb atributs i comportaments.
  • Classes: Plànols per crear objectes, definint les seves propietats i mètodes.
  • Herència: Mecanisme per crear una nova classe utilitzant les propietats i mètodes d'una classe existent.
  • Polimorfisme: Capacitat de processar objectes de manera diferent segons el seu tipus de dada o classe.
  • Encapsulació: Agrupació de les dades (atributs) i mètodes (funcions) que operen sobre les dades en una sola unitat o classe, i restringint l'accés a alguns dels components de l'objecte.
  • Abstracció: Amagar els detalls complexos de la implementació i mostrar només les característiques necessàries d'un objecte.

Conceptes Clau de la POO en Java

Classes i Objectes

Una classe en Java és un plànol per crear objectes. Un objecte és una instància d'una classe.

// Definir una classe
public class Car {
    // Atributs
    String color;
    String model;
    int year;

    // Mètode
    void displayInfo() {
        System.out.println("Model: " + model + ", Color: " + color + ", Year: " + year);
    }
}

// Crear un objecte
public class Main {
    public static void main(String[] args) {
        Car myCar = new Car(); // Creant un objecte de Car
        myCar.color = "Red";
        myCar.model = "Toyota";
        myCar.year = 2020;
        myCar.displayInfo(); // Cridant el mètode
    }
}

Herència

L'herència permet que una nova classe hereti les propietats i mètodes d'una classe existent.

// Classe base
class Animal {
    void eat() {
        System.out.println("Aquest animal menja menjar.");
    }
}

// Classe derivada
class Dog extends Animal {
    void bark() {
        System.out.println("El gos borda.");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.eat(); // Mètode heretat
        myDog.bark(); // Mètode propi
    }
}

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen.

class Animal {
    void sound() {
        System.out.println("Aquest animal fa un so.");
    }
}

class Cat extends Animal {
    void sound() {
        System.out.println("El gat miola.");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("El gos borda.");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myAnimal;
        
        myAnimal = new Cat();
        myAnimal.sound(); // Resultat: El gat miola.
        
        myAnimal = new Dog();
        myAnimal.sound(); // Resultat: El gos borda.
    }
}

Encapsulació

L'encapsulació restringeix l'accés directe a alguns dels components d'un objecte, cosa que es pot aconseguir utilitzant modificadors d'accés.

public class Person {
    // Variables privades
    private String name;
    private int age;

    // Mètodes públics getter i setter
    public String getName() {
        return name;
    }

    public void setName(String newName) {
        name = newName;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int newAge) {
        age = newAge;
    }
}

public class Main {
    public static void main(String[] args) {
        Person person = new Person();
        person.setName("John");
        person.setAge(30);
        System.out.println("Nom: " + person.getName());
        System.out.println("Edat: " + person.getAge());
    }
}

Abstracció

L'abstracció és el concepte d'amagar els detalls complexos de la implementació i mostrar només les característiques necessàries d'un objecte.

abstract class Animal {
    // Mètode abstracte (no té cos)
    public abstract void animalSound();
    // Mètode regular
    public void sleep() {
        System.out.println("Aquest animal dorm.");
    }
}

class Pig extends Animal {
    public void animalSound() {
        // El cos de animalSound() es proporciona aquí
        System.out.println("El porc diu: oinc oinc");
    }
}

public class Main {
    public static void main(String[] args) {
        Pig myPig = new Pig();
        myPig.animalSound(); // Resultat: El porc diu: oinc oinc
        myPig.sleep(); // Resultat: Aquest animal dorm.
    }
}

Conclusió

La Programació Orientada a Objectes (POO) en Java és un paradigma potent que ajuda a crear codi modular, reutilitzable i mantenible. Entenent i aplicant els conceptes bàsics de classes, objectes, herència, polimorfisme, encapsulació i abstracció, pots dissenyar aplicacions robustes i escalables. A mesura que progressis de nivell principiant a avançat, aquests principis es convertiran en la base per a pràctiques de programació més complexes i eficients.
