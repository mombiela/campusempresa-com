Introducció

Benvingut al teu viatge en la programació en Java! En aquesta secció, et guiarem a través de l'escriptura del teu primer programa en Java. Al final d'aquesta lliçó, entendràs l'estructura bàsica d'un programa en Java i com compilar-lo i executar-lo.

Configuració del teu entorn

Abans de començar a codificar, necessites configurar el teu entorn de desenvolupament.

  • Instal·la el Java Development Kit (JDK): Descarrega i instal·la l'última versió del JDK des del lloc web oficial d'Oracle.
  • Instal·la un Entorn de Desenvolupament Integrat (IDE): Et recomanem utilitzar IntelliJ IDEA, Eclipse o NetBeans. Aquests eines proporcionen una interfície amigable per escriure, compilar i executar programes en Java.

Escriure el teu primer programa en Java

Escrivim un programa senzill en Java que imprimeixi "Hello, World!" a la consola.

Guia pas a pas

  1. Crea un nou fitxer Java: Obre el teu IDE i crea un nou projecte en Java. Dins del projecte, crea un nou fitxer Java anomenat HelloWorld.java.

  2. Escriu el codi: Introdueix el següent codi al teu fitxer HelloWorld.java:

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

  3. Explicació del codi:

    • public class HelloWorld: Aquesta línia declara una classe pública anomenada HelloWorld. En Java, cada aplicació ha de tenir almenys una definició de classe.
    • public static void main(String[] args): Aquest és el mètode principal. És el punt d'entrada de qualsevol aplicació en Java. La Màquina Virtual de Java (JVM) crida aquest mètode per iniciar el programa.
    • System.out.println("Hello, World!");: Aquesta línia imprimeix la cadena "Hello, World!" a la consola. System.out és un flux de sortida estàndard, i println és un mètode que imprimeix una línia de text.

Compilar i executar el teu programa

Per veure el teu programa en acció, necessites compilar-lo i executar-lo.

Compilar el programa

  • Utilitzant la línia de comandes:
    1. Obre la teva interfície de línia de comandes (CLI).
    2. Navega al directori on es troba el teu fitxer HelloWorld.java.
    3. Compila el programa escrivint: 
      javac HelloWorld.java
    4. Si no hi ha errors, aquesta comanda produirà un fitxer anomenat HelloWorld.class.

Executar el programa

  • Utilitzant la línia de comandes:
    1. Al mateix directori, executa el programa compilat escrivint: 
      java HelloWorld
    2. Hauries de veure la sortida: 
      Hello, World!

Errors comuns i resolució de problemes

Aquí tens alguns errors comuns que podries trobar i com solucionar-los:

  • Classe no trobada: Assegura't que estàs en el directori correcte i que el nom de la classe coincideix amb el nom del fitxer.
  • Errors de sintaxi: Revisa el teu codi per trobar errors tipogràfics o punts i comes que faltin.
  • JDK no instal·lat: Assegura't que el JDK està instal·lat correctament i que les comandes javac i java estan disponibles en el PATH del teu sistema.

Conclusió

Felicitats! Has escrit, compilat i executat el teu primer programa en Java. Aquest senzill exemple de "Hello, World!" t'introdueix a l'estructura bàsica d'una aplicació en Java i al procés de compilar i executar codi en Java. A mesura que avancis en aquest curs, construiràs sobre aquests fonaments per crear aplicacions en Java més complexes i potents. Continua practicant, i feliç codificació!
