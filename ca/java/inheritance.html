L'herència és un concepte fonamental en la programació orientada a objectes (OOP) que permet que una nova classe hereti propietats i comportaments (mètodes) d'una classe existent. Això promou la reutilització del codi i estableix una jerarquia natural entre classes.

Conceptes Clau

  • Superclasse (Classe Pare): La classe les propietats i mètodes de la qual són heretats.
  • Subclasse (Classe Filla): La classe que hereta de la superclasse.
  • Paraula clau extends: Utilitzada per indicar que una classe està heretant d'una altra classe.
  • Sobreescriptura de Mètodes: Una subclasse pot proporcionar una implementació específica d'un mètode que ja està definit en la seva superclasse.
  • Paraula clau super: Utilitzada per referir-se a l'objecte de la classe pare immediata.

Exemple Bàsic d'Herència

Comencem amb un exemple senzill per entendre com funciona l'herència en Java.

// Superclasse
class Animal {
    void eat() {
        System.out.println("Aquest animal menja menjar.");
    }
}

// Subclasse
class Dog extends Animal {
    void bark() {
        System.out.println("El gos borda.");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.eat();  // Mètode heretat
        myDog.bark(); // Mètode específic de la subclasse
    }
}

Explicació

  • Superclasse (Animal): Conté un mètode eat().
  • Subclasse (Dog): Hereta el mètode eat() d'Animal i també té el seu propi mètode bark().
  • Classe Principal: Crea una instància de Dog i crida tant els mètodes eat() com bark().

Sobreescriptura de Mètodes

La sobreescriptura de mètodes permet que una subclasse proporcioni una implementació específica per a un mètode que ja està definit en la seva superclasse.

class Animal {
    void sound() {
        System.out.println("Aquest animal fa un so.");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("El gos borda.");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myAnimal = new Animal();
        myAnimal.sound(); // Sortida: Aquest animal fa un so.

        Dog myDog = new Dog();
        myDog.sound(); // Sortida: El gos borda.
    }
}

Explicació

  • Sobreescriptura de Mètodes: El mètode sound() en Dog sobreescriu el mètode sound() en Animal.
  • Anotació @Override: Indica que un mètode està sobreescrivint un mètode en la seva superclasse.

Ús de la Paraula Clau `super`

La paraula clau super s'utilitza per accedir als mètodes i constructors de la superclasse.

class Animal {
    void sound() {
        System.out.println("Aquest animal fa un so.");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        super.sound(); // Crida el mètode de la superclasse
        System.out.println("El gos borda.");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.sound(); // Sortida: Aquest animal fa un so. El gos borda.
    }
}

Explicació

  • super.sound(): Crida el mètode sound() de la classe Animal abans d'executar el mètode sound() en la classe Dog.

Herència i Constructors

Els constructors no són heretats, però un constructor de subclasse pot cridar un constructor de superclasse utilitzant la paraula clau super.

class Animal {
    Animal() {
        System.out.println("Animal creat.");
    }
}

class Dog extends Animal {
    Dog() {
        super(); // Crida el constructor de la superclasse
        System.out.println("Gos creat.");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog(); // Sortida: Animal creat. Gos creat.
    }
}

Explicació

  • Constructor de la Superclasse: El constructor d'Animal es crida primer.
  • Constructor de la Subclasse: El constructor de Dog es crida després del constructor de la superclasse.

Resum

  • Herència permet que una classe hereti propietats i mètodes d'una altra classe, promovent la reutilització del codi.
  • Sobreescriptura de Mètodes permet que una subclasse proporcioni una implementació específica d'un mètode ja definit en la seva superclasse.
  • La paraula clau super s'utilitza per referir-se a l'objecte de la classe pare immediata, permetent l'accés als mètodes i constructors de la superclasse.
  • Els constructors no són heretats però poden ser cridats des d'una subclasse utilitzant la paraula clau super.

Entenent i utilitzant l'herència, pots crear un codi més organitzat i modular, fent que els teus programes en Java siguin més eficients i fàcils de mantenir.
