Introducció

Java és un llenguatge de programació d'alt nivell, basat en classes i orientat a objectes, dissenyat per tenir el mínim de dependències d'implementació possibles. És un llenguatge de programació de propòsit general destinat a permetre als desenvolupadors d'aplicacions escriure una vegada i executar en qualsevol lloc (WORA), el que significa que el codi Java compilat pot executar-se en totes les plataformes que suporten Java sense necessitat de recompilació.

Característiques clau de Java

  • Independència de plataforma: El codi Java es compila en bytecode, que pot executar-se en qualsevol dispositiu equipat amb una Màquina Virtual Java (JVM).
  • Orientat a objectes: Java es basa en els principis d'objectes i classes, promovent codi modular i reutilitzable.
  • Robust i segur: Java proporciona una gestió de memòria forta, maneig d'excepcions i un gestor de seguretat per definir regles d'accés per a les classes.
  • Multifil: Java suporta la multiexecució, permetent l'execució concurrent de dos o més fils per a una màxima utilització de la CPU.
  • Alt rendiment: El compilador Just-In-Time (JIT) de Java permet un alt rendiment convertint el bytecode en codi màquina natiu en temps d'execució.
  • Distribuït: Java té un conjunt ric d'APIs per a xarxes, facilitant el treball amb sistemes distribuïts.

Kit de desenvolupament Java (JDK) i Entorn d'execució Java (JRE)

  • JDK: El Kit de Desenvolupament Java inclou eines per desenvolupar, depurar i monitoritzar aplicacions Java. Conté el JRE, un intèrpret/carregador (Java), un compilador (javac), un arxivador (jar), un generador de documentació (javadoc) i altres eines necessàries per al desenvolupament en Java.
  • JRE: L'Entorn d'Execució Java proporciona biblioteques, la Màquina Virtual Java (JVM) i altres components per executar aplicacions escrites en Java. No inclou eines de desenvolupament com compiladors o depuradors.

| Component | Descripció | |-----------|-------------| | JDK | Kit de desenvolupament per a aplicacions Java, inclou JRE i eines de desenvolupament. | | JRE | Entorn d'execució per executar aplicacions Java, inclou JVM i biblioteques estàndard. | | JVM | Màquina Virtual Java, interpreta el bytecode i l'executa en la màquina host. |

Sintaxi i estructura bàsiques

Els programes Java es componen de classes i mètodes. Aquí hi ha un exemple senzill d'un programa Java:

// HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explicació

  • public class HelloWorld: Defineix una classe pública anomenada HelloWorld.
  • public static void main(String[] args): El mètode principal és el punt d'entrada de qualsevol aplicació Java. public significa que és accessible des de qualsevol lloc, static significa que pertany a la classe en lloc de les instàncies de la classe, void significa que no retorna cap valor, i String[] args és una matriu d'arguments de línia de comandes.
  • System.out.println("Hello, World!"): Imprimeix la cadena "Hello, World!" a la consola.

Programació Orientada a Objectes (OOP) en Java

Java és fonamentalment orientat a objectes, el que significa que utilitza objectes i classes per estructurar programes. Els conceptes clau inclouen:

  • Classe: Un pla per crear objectes. Defineix un tipus de dades agrupant dades i mètodes que treballen amb les dades.
  • Objecte: Una instància d'una classe. Conté estat (camps) i comportament (mètodes).
  • Herència: Mecanisme on una classe hereta els camps i mètodes d'una altra classe.
  • Polimorfisme: Capacitat de processar objectes de manera diferent segons el seu tipus de dades o classe.
  • Encapsulació: Agrupació de les dades (camps) i el codi (mètodes) que manipula les dades en una sola unitat o classe.
  • Abstracció: Amagar detalls d'implementació complexos i mostrar només les característiques necessàries d'un objecte.

Conclusió

Java és un llenguatge de programació versàtil i potent que s'utilitza àmpliament en diversos dominis, des d'aplicacions web fins a sistemes empresarials i aplicacions mòbils. La seva independència de plataforma, característiques de seguretat robustes i principis orientats a objectes el fan una excel·lent elecció tant per a principiants com per a desenvolupadors experimentats. Entendre els conceptes bàsics de Java, incloent la seva sintaxi, estructura i característiques clau, és el primer pas per dominar aquest llenguatge i aprofitar tot el seu potencial en el desenvolupament de programari.
