Introducció a Swing

Swing és una part de les Java Foundation Classes (JFC) utilitzada per crear aplicacions basades en finestres. Està construït sobre l'Abstract Window Toolkit (AWT) i proporciona un conjunt més ric de components GUI.

  • Components lleugers: A diferència de l'AWT, els components de Swing són lleugers, és a dir, estan escrits completament en Java i no depenen del codi natiu.
  • Aspecte i sensació endollables: Swing permet canviar l'aparença de la teva aplicació sense alterar el codi subjacent.
  • Arquitectura MVC: Swing segueix l'arquitectura Model-Vista-Controlador, que separa les dades (model), la interfície d'usuari (vista) i la lògica de control (controlador).

Configuració d'una aplicació Swing

Per crear una aplicació bàsica de Swing, has de seguir aquests passos:

  1. Importar paquets de Swing: Importa els paquets necessaris de Swing i AWT.
  2. Crear un JFrame: La finestra principal de la teva aplicació.
  3. Afegir components: Afegeix diversos components de Swing com botons, etiquetes, camps de text, etc.
  4. Establir el disseny: Defineix el gestor de disseny per organitzar els components.
  5. Mostrar el marc: Fes visible el marc.

Exemple: Una aplicació simple de Swing

import javax.swing.JFrame;
import javax.swing.JLabel;

public class HelloWorldSwing {
    public static void main(String[] args) {
        // Crear un nou JFrame
        JFrame frame = new JFrame("Hello World Swing");
        
        // Establir l'operació de tancament per defecte
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // Afegir un JLabel al marc
        JLabel label = new JLabel("Hello, World!");
        frame.getContentPane().add(label);
        
        // Establir la mida del marc
        frame.setSize(300, 100);
        
        // Fer visible el marc
        frame.setVisible(true);
    }
}
  • Explicació:
    • JFrame: La finestra principal.
    • JLabel: Una etiqueta de text simple.
    • setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE): Assegura que l'aplicació es tanqui quan es tanca la finestra.
    • setSize(300, 100): Estableix la mida de la finestra.
    • setVisible(true): Fa visible la finestra.

Components comuns de Swing

Swing proporciona una varietat de components per construir una interfície d'usuari.

  • JButton: Un botó que pot desencadenar una acció.
  • JLabel: Una àrea de visualització per a una cadena de text curta o una imatge.
  • JTextField: Un camp de text d'una sola línia per a l'entrada de l'usuari.
  • JTextArea: Una àrea de múltiples línies per mostrar/editar text.
  • JCheckBox: Una caixa que es pot marcar o desmarcar.
  • JRadioButton: Un botó en un grup on només un pot ser seleccionat.
  • JComboBox: Una llista desplegable d'elements.

Exemple: Afegint més components

import javax.swing.*;

public class SwingComponentsExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Swing Components Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 200);
        
        // Crear components
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Label:");
        JTextField textField = new JTextField(20);
        JCheckBox checkBox = new JCheckBox("Check Me");
        JRadioButton radioButton = new JRadioButton("Select Me");
        JComboBox<String> comboBox = new JComboBox<>(new String[]{"Option 1", "Option 2", "Option 3"});
        
        // Afegir components al marc
        JPanel panel = new JPanel();
        panel.add(label);
        panel.add(textField);
        panel.add(button);
        panel.add(checkBox);
        panel.add(radioButton);
        panel.add(comboBox);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explicació:
    • JPanel: Un contenidor per agrupar components.
    • JButton, JLabel, JTextField, JCheckBox, JRadioButton, JComboBox: Diversos components de Swing afegits al panell.

Gestió d'esdeveniments a Swing

Swing utilitza el model d'esdeveniments de delegació per gestionar esdeveniments. Això implica tres components:

  • Font de l'esdeveniment: El component que genera l'esdeveniment (per exemple, un botó).
  • Objecte de l'esdeveniment: Encapsula la informació de l'esdeveniment.
  • Receptor de l'esdeveniment: Una interfície que rep i processa l'esdeveniment.

Exemple: Gestionant clics de botó

import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonClickExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Click Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(300, 200);
        
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Button not clicked");
        
        // Afegir ActionListener al botó
        button.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                label.setText("Button clicked!");
            }
        });
        
        JPanel panel = new JPanel();
        panel.add(button);
        panel.add(label);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explicació:
    • ActionListener: Una interfície per gestionar esdeveniments d'acció.
    • actionPerformed(ActionEvent e): Mètode cridat quan es fa clic al botó.
    • label.setText("Button clicked!"): Actualitza el text de l'etiqueta quan es fa clic al botó.

Gestors de disseny

Els gestors de disseny s'utilitzen per organitzar components en un contenidor. Alguns gestors de disseny comuns inclouen:

  • FlowLayout: Organitza els components en un flux d'esquerra a dreta.
  • BorderLayout: Organitza els components en cinc regions: Nord, Sud, Est, Oest i Centre.
  • GridLayout: Organitza els components en una graella de cel·les.
  • BoxLayout: Organitza els components verticalment o horitzontalment.

Exemple: Utilitzant BorderLayout

import javax.swing.*;
import java.awt.*;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);
        
        JButton northButton = new JButton("North");
        JButton southButton = new JButton("South");
        JButton eastButton = new JButton("East");
        JButton westButton = new JButton("West");
        JButton centerButton = new JButton("Center");
        
        frame.setLayout(new BorderLayout());
        frame.add(northButton, BorderLayout.NORTH);
        frame.add(southButton, BorderLayout.SOUTH);
        frame.add(eastButton, BorderLayout.EAST);
        frame.add(westButton, BorderLayout.WEST);
        frame.add(centerButton, BorderLayout.CENTER);
        
        frame.setVisible(true);
    }
}
  • Explicació:
    • BorderLayout: Divideix el contenidor en cinc regions.
    • frame.add(component, BorderLayout.REGION): Afegeix components a regions específiques.

Conclusió

Swing és una eina potent per crear aplicacions GUI en Java. Entenent els conceptes bàsics dels components de Swing, la gestió d'esdeveniments i els gestors de disseny, pots construir aplicacions interactives i fàcils d'usar. A mesura que progressis, podràs explorar funcions més avançades i personalitzacions per millorar encara més les teves aplicacions.
