Introducció

Els operadors són símbols especials en Java que realitzen operacions específiques sobre un, dos o tres operands i després retornen un resultat. Són fonamentals per escriure codi efectiu i eficient. Aquesta secció cobrirà els diversos tipus d'operadors disponibles en Java, des de l'aritmètica bàsica fins als operadors bit a bit i lògics més avançats.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul (resta)

Exemple

int a = 10;
int b = 5;

int sum = a + b; // 15
int difference = a - b; // 5
int product = a * b; // 50
int quotient = a / b; // 2
int remainder = a % b; // 0

System.out.println("Sum: " + sum);
System.out.println("Difference: " + difference);
System.out.println("Product: " + product);
System.out.println("Quotient: " + quotient);
System.out.println("Remainder: " + remainder);

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors. Retornen un resultat booleà.

  • == : Igual a
  • != : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual que
  • <= : Menor o igual que

Exemple

int a = 10;
int b = 5;

boolean isEqual = (a == b); // false
boolean isNotEqual = (a != b); // true
boolean isGreater = (a > b); // true
boolean isLess = (a < b); // false
boolean isGreaterOrEqual = (a >= b); // true
boolean isLessOrEqual = (a <= b); // false

System.out.println("Is Equal: " + isEqual);
System.out.println("Is Not Equal: " + isNotEqual);
System.out.println("Is Greater: " + isGreater);
System.out.println("Is Less: " + isLess);
System.out.println("Is Greater or Equal: " + isGreaterOrEqual);
System.out.println("Is Less or Equal: " + isLessOrEqual);

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples expressions booleanes.

  • && : AND lògic
  • || : OR lògic
  • ! : NOT lògic

Exemple

boolean a = true;
boolean b = false;

boolean andResult = a && b; // false
boolean orResult = a || b; // true
boolean notResult = !a; // false

System.out.println("AND Result: " + andResult);
System.out.println("OR Result: " + orResult);
System.out.println("NOT Result: " + notResult);

Operadors Bit a Bit

Els operadors bit a bit realitzen operacions sobre bits individuals de tipus enters.

  • & : AND bit a bit
  • | : OR bit a bit
  • ^ : XOR bit a bit
  • ~ : Complement bit a bit
  • << : Desplaçament a l'esquerra
  • >> : Desplaçament a la dreta
  • >>> : Desplaçament a la dreta sense signe

Exemple

int a = 5; // 0101 en binari
int b = 3; // 0011 en binari

int andResult = a & b; // 0001 en binari, que és 1
int orResult = a | b; // 0111 en binari, que és 7
int xorResult = a ^ b; // 0110 en binari, que és 6
int complementResult = ~a; // 1010 en binari, que és -6 (complement a 2)
int leftShiftResult = a << 1; // 1010 en binari, que és 10
int rightShiftResult = a >> 1; // 0010 en binari, que és 2
int unsignedRightShiftResult = a >>> 1; // 0010 en binari, que és 2

System.out.println("AND Result: " + andResult);
System.out.println("OR Result: " + orResult);
System.out.println("XOR Result: " + xorResult);
System.out.println("Complement Result: " + complementResult);
System.out.println("Left Shift Result: " + leftShiftResult);
System.out.println("Right Shift Result: " + rightShiftResult);
System.out.println("Unsigned Right Shift Result: " + unsignedRightShiftResult);

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • = : Assignació simple
  • += : Sumar i assignar
  • -= : Restar i assignar
  • *= : Multiplicar i assignar
  • /= : Dividir i assignar
  • %= : Mòdul i assignar

Exemple

int a = 10;

a += 5; // a = a + 5, així que a és ara 15
a -= 3; // a = a - 3, així que a és ara 12
a *= 2; // a = a * 2, així que a és ara 24
a /= 4; // a = a / 4, així que a és ara 6
a %= 5; // a = a % 5, així que a és ara 1

System.out.println("Final value of a: " + a);

Operadors Unaris

Els operadors unaris operen sobre un sol operand.

  • + : Més unari (promou l'operand)
  • - : Menys unari (nega l'operand)
  • ++ : Operador d'increment (augmenta el valor en 1)
  • -- : Operador de decrement (disminueix el valor en 1)
  • ! : Operador de complement lògic

Exemple

int a = 10;
int b = -a; // b és ara -10

a++; // a és ara 11
a--; // a és ara 10 de nou

boolean c = true;
boolean d = !c; // d és ara false

System.out.println("Value of b: " + b);
System.out.println("Value of a after increment and decrement: " + a);
System.out.println("Value of d: " + d);

Operador Ternari

L'operador ternari és una forma abreujada de l'instrucció if-else i s'utilitza per avaluar expressions booleanes.

  • ? : : Operador ternari

Exemple

int a = 10;
int b = 5;

int max = (a > b) ? a : b; // max és 10 perquè a és major que b

System.out.println("Maximum value: " + max);

Conclusió

Els operadors són eines essencials en la programació en Java, permetent als desenvolupadors realitzar una àmplia gamma d'operacions sobre les dades. Entendre com utilitzar aquests operadors de manera efectiva pot millorar enormement la teva capacitat per escriure codi eficient i llegible. Des de l'aritmètica bàsica fins a les operacions bit a bit avançades, dominar aquests operadors és un pas crucial per convertir-se en un programador Java proficient.
