Introducció

En aquesta secció, explorarem com treballar amb ResultSet en Java per manipular dades recuperades d'una base de dades. Cobrirà els conceptes bàsics de ResultSet, com navegar-hi i com actualitzar dades utilitzant-lo. Aquest tema és crucial per a qualsevol persona que vulgui construir aplicacions robustes en Java que interactuïn amb bases de dades.

Entenent ResultSet

Un ResultSet és una taula de dades que representa el resultat d'una consulta a una base de dades. Es genera executant una instrucció que consulta la base de dades.

Conceptes Clau

  • Interfície ResultSet: La interfície ResultSet proporciona mètodes per recuperar i manipular les dades en el conjunt de resultats.
  • Cursor: El ResultSet manté un cursor que apunta a la seva fila actual de dades.
  • Tipus de ResultSet:
    • TYPE_FORWARD_ONLY: El cursor només pot avançar.
    • TYPE_SCROLL_INSENSITIVE: El cursor pot moure's tant cap endavant com cap enrere, però el conjunt de resultats no és sensible als canvis fets per altres.
    • TYPE_SCROLL_SENSITIVE: El cursor pot moure's tant cap endavant com cap enrere, i el conjunt de resultats és sensible als canvis fets per altres.

Creant un ResultSet

Per crear un ResultSet, necessites executar una consulta utilitzant un Statement o PreparedStatement.

Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydatabase", "user", "password");
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

Navegant ResultSet

Pots navegar per les files d'un ResultSet utilitzant diversos mètodes.

Mètodes Clau

  • next(): Mou el cursor a la següent fila.
  • previous(): Mou el cursor a la fila anterior (només per a ResultSet desplaçables).
  • first(): Mou el cursor a la primera fila.
  • last(): Mou el cursor a l'última fila.
  • absolute(int row): Mou el cursor a la fila especificada.

Exemple

while (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    System.out.println("ID: " + id + ", Nom: " + name);
}

Recuperant Dades de ResultSet

Pots recuperar dades de la fila actual d'un ResultSet utilitzant mètodes getter.

Mètodes Clau

  • getInt(String columnLabel): Recupera un valor enter.
  • getString(String columnLabel): Recupera un valor de cadena.
  • getDouble(String columnLabel): Recupera un valor doble.
  • getDate(String columnLabel): Recupera un valor de data.

Exemple

if (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    double salary = rs.getDouble("salary");
    Date hireDate = rs.getDate("hire_date");
    System.out.println("ID: " + id + ", Nom: " + name + ", Salari: " + salary + ", Data de Contractació: " + hireDate);
}

Actualitzant Dades amb ResultSet

Un ResultSet també es pot utilitzar per actualitzar dades a la base de dades.

Conceptes Clau

  • Tipus de Concurrència:
    • CONCUR_READ_ONLY: El ResultSet no es pot actualitzar.
    • CONCUR_UPDATABLE: El ResultSet es pot actualitzar.

Exemple

Per crear un ResultSet actualitzable, necessites especificar el tipus de concurrència quan crees el Statement.

Statement stmt = conn.createStatement(ResultSet.TYPE_SCROLL_INSENSITIVE, ResultSet.CONCUR_UPDATABLE);
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

// Mou-te a la fila que vols actualitzar
if (rs.next()) {
    rs.updateString("name", "John Doe");
    rs.updateRow();
}

Inserint Dades

També pots inserir noves files a la base de dades utilitzant ResultSet.

rs.moveToInsertRow();
rs.updateInt("id", 101);
rs.updateString("name", "Jane Doe");
rs.updateDouble("salary", 75000);
rs.updateDate("hire_date", java.sql.Date.valueOf("2023-01-01"));
rs.insertRow();

Eliminant Dades

Per eliminar una fila de la base de dades utilitzant ResultSet:

if (rs.next()) {
    rs.deleteRow();
}

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de treballar amb ResultSet en Java. Hem explorat com crear, navegar, recuperar, actualitzar, inserir i eliminar dades utilitzant ResultSet. Entendre aquests conceptes és essencial per a una manipulació efectiva de dades en aplicacions Java que interactuen amb bases de dades. Practica aquestes tècniques per convertir-te en un expert en la gestió d'operacions de bases de dades en Java.
