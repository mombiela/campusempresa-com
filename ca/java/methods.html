Introducció als Mètodes

Els mètodes són blocs de codi que realitzen una tasca específica i es poden cridar sempre que sigui necessari. Ajuden a organitzar el codi, fer-lo reutilitzable i millorar la llegibilitat.

Conceptes Clau

  • Definició: Un mètode és una col·lecció d'instruccions agrupades per realitzar una operació.
  • Sintaxi: La sintaxi bàsica per definir un mètode en Java és: 
    returnType methodName(parameters) {
    // cos del mètode
}
  • Tipus de Retorn: Especifica el tipus de valor que retorna el mètode. Utilitza void si el mètode no retorna cap valor.
  • Nom del Mètode: Un identificador únic per al mètode.
  • Paràmetres: Entrades per al mètode, tancades entre parèntesis. Si no es necessiten entrades, deixa els parèntesis buits.
  • Cos del Mètode: El bloc de codi que defineix què fa el mètode.

Exemple

public class Main {
    // Mètode per sumar dos números
    public int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        Main obj = new Main();
        int result = obj.add(5, 3);
        System.out.println("Sum: " + result); // Output: Sum: 8
    }
}

En aquest exemple:

  • add és un mètode que pren dos enters com a paràmetres i retorna la seva suma.
  • El mètode main és el punt d'entrada del programa on es crida el mètode add.

Tipus de Mètodes

Mètodes d'Instància

Els mètodes d'instància pertanyen a una instància d'una classe. Requereixen que es creï un objecte de la classe abans que es puguin cridar.

public class Calculator {
    public int multiply(int a, int b) {
        return a * b;
    }

    public static void main(String[] args) {
        Calculator calc = new Calculator();
        int product = calc.multiply(4, 5);
        System.out.println("Product: " + product); // Output: Product: 20
    }
}

Mètodes Estàtics

Els mètodes estàtics pertanyen a la classe en lloc de qualsevol instància. Es poden cridar sense crear un objecte de la classe.

public class MathUtils {
    public static int square(int num) {
        return num * num;
    }

    public static void main(String[] args) {
        int result = MathUtils.square(6);
        System.out.println("Square: " + result); // Output: Square: 36
    }
}

Sobrecàrrega de Mètodes

La sobrecàrrega de mètodes permet que una classe tingui més d'un mètode amb el mateix nom, sempre que les seves llistes de paràmetres siguin diferents.

public class Printer {
    public void print(String s) {
        System.out.println(s);
    }

    public void print(int i) {
        System.out.println(i);
    }

    public static void main(String[] args) {
        Printer printer = new Printer();
        printer.print("Hello, World!"); // Output: Hello, World!
        printer.print(100);             // Output: 100
    }
}

Conceptes Avançats de Mètodes

Recursivitat

La recursivitat és una tècnica on un mètode es crida a si mateix per resoldre un problema.

public class Factorial {
    public int factorial(int n) {
        if (n == 0) {
            return 1;
        } else {
            return n * factorial(n - 1);
        }
    }

    public static void main(String[] args) {
        Factorial fact = new Factorial();
        int result = fact.factorial(5);
        System.out.println("Factorial: " + result); // Output: Factorial: 120
    }
}

Pas per Valor

Java utilitza el pas per valor per passar arguments als mètodes. Això significa que es passa una còpia de l'argument al mètode.

public class PassByValue {
    public void modifyValue(int num) {
        num = 10;
    }

    public static void main(String[] args) {
        int x = 5;
        PassByValue obj = new PassByValue();
        obj.modifyValue(x);
        System.out.println("Value of x: " + x); // Output: Value of x: 5
    }
}

Conclusió

Els mètodes són blocs fonamentals en la programació en Java. Ajuden a organitzar el codi, fer-lo reutilitzable i millorar la llegibilitat. Entendre els diferents tipus de mètodes, la sobrecàrrega de mètodes, la recursivitat i el concepte de pas per valor és crucial per escriure programes Java eficients i mantenibles. Practica escriure i cridar mètodes per convertir-te en un expert en utilitzar-los de manera efectiva.
