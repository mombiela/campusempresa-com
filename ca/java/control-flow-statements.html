Sentències de control de flux
Bucles ►

Les sentències de control de flux en Java et permeten dictar l'ordre en què s'executen les sentències. Aquesta secció cobrirà les sentències de control de flux essencials, incloent sentències condicionals, bucles i sentències de ramificació.

Sentències condicionals

Les sentències condicionals permeten que el teu programa prengui decisions basades en certes condicions.

Sentència if

La sentència if executa un bloc de codi si una condició especificada és certa.

int number = 10;
if (number > 0) {
    System.out.println("El número és positiu.");
}

Sentència if-else

La sentència if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

int number = -10;
if (number > 0) {
    System.out.println("El número és positiu.");
} else {
    System.out.println("El número no és positiu.");
}

Escala if-else-if

L'escala if-else-if et permet provar múltiples condicions seqüencialment.

int number = 0;
if (number > 0) {
    System.out.println("El número és positiu.");
} else if (number < 0) {
    System.out.println("El número és negatiu.");
} else {
    System.out.println("El número és zero.");
}

Sentència switch

La sentència switch et permet executar un bloc de codi d'entre molts basat en el valor d'una expressió.

int day = 3;
switch (day) {
    case 1:
        System.out.println("Dilluns");
        break;
    case 2:
        System.out.println("Dimarts");
        break;
    case 3:
        System.out.println("Dimecres");
        break;
    default:
        System.out.println("Dia invàlid");
        break;
}

Sentències de bucle

Les sentències de bucle et permeten executar un bloc de codi múltiples vegades.

Bucle for

El bucle for s'utilitza quan es coneix el nombre d'iteracions.

for (int i = 0; i < 5; i++) {
    System.out.println("Iteració: " + i);
}

Bucle while

El bucle while s'utilitza quan el nombre d'iteracions no es coneix i depèn d'una condició.

int i = 0;
while (i < 5) {
    System.out.println("Iteració: " + i);
    i++;
}

Bucle do-while

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

int i = 0;
do {
    System.out.println("Iteració: " + i);
    i++;
} while (i < 5);

Sentències de ramificació

Les sentències de ramificació et permeten alterar el flux de control en el teu programa.

Sentència break

La sentència break s'utilitza per sortir d'un bucle o d'una sentència switch prematurament.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println("Iteració: " + i);
}

Sentència continue

La sentència continue salta l'iteració actual d'un bucle i procedeix a la següent iteració.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue;
    }
    System.out.println("Iteració: " + i);
}

Sentència return

La sentència return surt del mètode actual i opcionalment retorna un valor.

public int add(int a, int b) {
    return a + b;
}

Conclusió

Les sentències de control de flux són fonamentals per programar en Java. Et permeten controlar el flux d'execució del teu programa basat en condicions i bucles. Entendre i dominar aquestes sentències et permetrà escriure programes Java més complexos i eficients. Practica utilitzant aquestes sentències en diferents escenaris per obtenir una sòlida comprensió de la seva funcionalitat.
