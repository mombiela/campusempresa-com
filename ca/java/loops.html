Introducció als Bucles

Els bucles són constructes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. Java admet diversos tipus de bucles, cadascun amb el seu propi cas d'ús.

Conceptes Clau

  • Iteració: Execució repetida d'un conjunt d'instruccions.
  • Instruccions de Control de Bucle: Paraules clau que gestionen el flux dels bucles (break, continue).

Tipus de Bucles

1. Bucle `for`

El bucle for s'utilitza quan el nombre d'iteracions es coneix per endavant.

Sintaxi

for (initialization; condition; update) {
    // Codi a executar
}

Exemple

for (int i = 0; i < 5; i++) {
    System.out.println("Iteració: " + i);
}
  • Inicialització: int i = 0 estableix el punt de partida.
  • Condició: i < 5 comprova si el bucle ha de continuar.
  • Actualització: i++ incrementa el comptador del bucle.

2. Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix i depèn d'una condició.

Sintaxi

while (condition) {
    // Codi a executar
}

Exemple

int i = 0;
while (i < 5) {
    System.out.println("Iteració: " + i);
    i++;
}
  • Condició: i < 5 comprova si el bucle ha de continuar.

3. Bucle `do-while`

El bucle do-while és similar al bucle while però garanteix almenys una iteració.

Sintaxi

do {
    // Codi a executar
} while (condition);

Exemple

int i = 0;
do {
    System.out.println("Iteració: " + i);
    i++;
} while (i < 5);
  • Condició: i < 5 comprova si el bucle ha de continuar després de la primera iteració.

4. Bucle `for` Millorat (for-each)

El bucle for millorat s'utilitza per iterar sobre arrays o col·leccions.

Sintaxi

for (type element : array) {
    // Codi a executar
}

Exemple

int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println("Número: " + number);
}
  • Tipus: int especifica el tipus d'elements en l'array.
  • Element: number és la variable que conté cada element durant la iteració.

Instruccions de Control de Bucle

Instrucció `break`

La instrucció break surt del bucle immediatament.

Exemple

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println("Iteració: " + i);
}
  • Sortida: Iteracions de 0 a 4.

Instrucció `continue`

La instrucció continue salta l'iteració actual i procedeix a la següent.

Exemple

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;
    }
    System.out.println("Número senar: " + i);
}
  • Sortida: Números senars de l'1 al 9.

Resum

Els bucles són essencials per realitzar tasques repetitives de manera eficient. Java proporciona diversos tipus de bucles (for, while, do-while i bucle for millorat) per gestionar diferents escenaris. Entendre com utilitzar les instruccions de control de bucle com break i continue pot ajudar a gestionar el flux dels bucles de manera efectiva. Practicar amb aquests constructes construirà una base sòlida per a conceptes de programació més avançats.
