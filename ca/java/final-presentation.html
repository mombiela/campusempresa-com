Introducció

La presentació final és una oportunitat perquè els estudiants mostrin la seva comprensió i habilitats adquirides al llarg del curs de Java. Aquesta presentació implicarà la creació d'un projecte complet que demostri l'aplicació de diversos conceptes de Java, des de la sintaxi bàsica fins a característiques avançades.

Objectius

  • Demostrar competència en programació Java.
  • Aplicar principis de programació orientada a objectes.
  • Utilitzar biblioteques i frameworks de Java.
  • Implementar les millors pràctiques de codificació.
  • Presentar el projecte de manera efectiva a una audiència.

Directrius del Projecte

Selecció del Projecte

  • Trieu un projecte que us interessi i que sigui factible dins del temps donat.
  • Assegureu-vos que el projecte incorpori múltiples conceptes de Java apresos durant el curs.

Requisits del Projecte

  • Requisits Bàsics:
    • Ús de la sintaxi bàsica de Java (variables, tipus de dades, operadors).
    • Estructures de control (if-else, bucles).
    • Funcions i mètodes.
  • Requisits Intermedis:
    • Programació orientada a objectes (classes, objectes, herència, polimorfisme).
    • Gestió d'excepcions.
    • Framework de col·leccions (ArrayList, HashMap, etc.).
  • Requisits Avançats:
    • Multithreading.
    • Operacions d'E/S de fitxers.
    • Ús de biblioteques o frameworks externs (per exemple, JavaFX per a GUI, JDBC per a la connectivitat de bases de dades).

Idees de Projectes Exemple

  • Sistema de Gestió de Biblioteques:
    • Característiques: Afegir, eliminar, actualitzar i cercar llibres.
    • Ús de classes per a llibres, membres i transaccions.
    • Gestió de fitxers per a l'emmagatzematge de dades.
  • Aplicació de Xat:
    • Característiques: Missatgeria en temps real entre usuaris.
    • Ús de multithreading per gestionar múltiples clients.
    • GUI per a la interacció amb l'usuari.
  • Plataforma de Comerç Electrònic:
    • Característiques: Llistat de productes, carret de la compra, processament de comandes.
    • Ús de col·leccions per gestionar productes i comandes.
    • Connectivitat de bases de dades per emmagatzemar informació d'usuaris i comandes.

Estructura de la Presentació

Introducció

  • Presenta't breument i presenta el teu projecte.
  • Explica la motivació darrere de la selecció del projecte.

Visió General del Projecte

  • Descriu el propòsit i la funcionalitat del projecte.
  • Destaca les característiques i components clau.

Implementació Tècnica

  • Discuteix l'arquitectura i el disseny del projecte.
  • Explica els conceptes clau de Java utilitzats en el projecte.
  • Proporciona exemples de codi per il·lustrar parts importants de la implementació.
// Exemple: Classe que representa un Llibre en un Sistema de Gestió de Biblioteques
public class Book {
    private String title;
    private String author;
    private String isbn;

    public Book(String title, String author, String isbn) {
        this.title = title;
        this.author = author;
        this.isbn = isbn;
    }

    // Getters i setters
    public String getTitle() {
        return title;
    }

    public void setTitle(String title) {
        this.title = title;
    }

    public String getAuthor() {
        return author;
    }

    public void setAuthor(String author) {
        this.author = author;
    }

    public String getIsbn() {
        return isbn;
    }

    public void setIsbn(String isbn) {
        this.isbn = isbn;
    }

    @Override
    public String toString() {
        return "Book [title=" + title + ", author=" + author + ", isbn=" + isbn + "]";
    }
}

Reptes i Solucions

  • Discuteix qualsevol repte enfrontat durant el desenvolupament del projecte.
  • Explica com vas abordar aquests reptes.

Demostració

  • Proporciona una demostració en viu del projecte.
  • Mostra com funciona el projecte i les seves característiques clau.

Conclusió

  • Resumeix el projecte i la seva importància.
  • Reflexiona sobre el que has après del projecte.
  • Discuteix possibles millores o extensions futures.

Resum

La presentació final és una part crucial del curs de Java, permetent als estudiants demostrar la seva comprensió completa de la programació en Java. En seleccionar un projecte significatiu, implementar-lo de manera efectiva i presentar-lo clarament, els estudiants poden mostrar les seves habilitats i preparació per al desenvolupament de Java en el món real. Aquest exercici no només reforça l'aprenentatge, sinó que també construeix confiança en l'aplicació dels conceptes de Java per resoldre problemes pràctics.
