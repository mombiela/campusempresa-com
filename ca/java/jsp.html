JavaServer Pages (JSP) és una tecnologia utilitzada per desenvolupar pàgines web que suporten contingut dinàmic. Permet incrustar codi Java en pàgines HTML, facilitant la creació d'aplicacions web que són tant interactives com funcionals.

Introducció a JSP

  • Definició: JSP és una tecnologia del costat del servidor que permet la creació d'aplicacions web dinàmiques i independents de la plataforma.
  • Propòsit: Separar la capa de presentació de la lògica de negoci en aplicacions web.
  • Sintaxi bàsica: Les pàgines JSP són essencialment pàgines HTML amb codi Java incrustat.

Configuració de JSP

  • Requisits:
    • Java Development Kit (JDK)
    • Apache Tomcat o qualsevol altre contenidor de servlets
    • Entorn de Desenvolupament Integrat (IDE) com Eclipse o IntelliJ IDEA

Exemple: Configuració d'una Pàgina JSP Simple

  1. Instal·lar Apache Tomcat: Descarrega i instal·la Apache Tomcat.
  2. Crear una Aplicació Web: En el teu IDE, crea un nou projecte d'aplicació web.
  3. Afegir un Fitxer JSP: Crea un nou fitxer anomenat index.jsp al directori webapp.
<!DOCTYPE html>
<html>
<head>
    <title>Welcome to JSP</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Current Date and Time: <%= new java.util.Date() %></p>
</body>
</html>

Sintaxi i Directives de JSP

  • Scriptlets: Incrusta codi Java en HTML utilitzant <% %>.
  • Expressions: Mostra valors al client utilitzant <%= %>.
  • Declaracions: Declara variables i mètodes utilitzant <%! %>.
  • Directives: Proporciona informació global sobre la pàgina JSP utilitzant <%@ %>.

Exemple: Utilitzant Scriptlets, Expressions i Declaracions

<!DOCTYPE html>
<html>
<head>
    <title>JSP Syntax Example</title>
</head>
<body>
    <h1>JSP Syntax</h1>
    <% 
        // Scriptlet
        String message = "Hello, JSP!";
    %>
    <p>Message: <%= message %></p> <!-- Expression -->
    <%
        // Declaration
        int add(int a, int b) {
            return a + b;
        }
    %>
    <p>Sum of 5 and 3: <%= add(5, 3) %></p>
</body>
</html>

Objectes Implícits de JSP

  • request: Representa l'objecte HttpServletRequest.
  • response: Representa l'objecte HttpServletResponse.
  • out: Utilitzat per enviar sortida al client.
  • session: Representa l'objecte HttpSession.
  • application: Representa l'objecte ServletContext.
  • config: Representa l'objecte ServletConfig.
  • pageContext: Proporciona accés a diversos espais de noms.
  • page: Es refereix a la pàgina JSP actual.
  • exception: Disponible només en pàgines d'error.

Exemple: Utilitzant Objectes Implícits

<!DOCTYPE html>
<html>
<head>
    <title>Implicit Objects Example</title>
</head>
<body>
    <h1>Implicit Objects in JSP</h1>
    <p>Client IP Address: <%= request.getRemoteAddr() %></p>
    <p>Session ID: <%= session.getId() %></p>
    <p>Server Name: <%= application.getServerInfo() %></p>
</body>
</html>

Accions i Etiquetes de JSP

  • Accions Estàndard: Accions predefinides com <jsp:include>, <jsp:forward>, i <jsp:param>.
  • Etiquetes Personalitzades: Etiquetes definides per l'usuari per encapsular contingut reutilitzable.

Exemple: Utilitzant Accions Estàndard

<!DOCTYPE html>
<html>
<head>
    <title>Standard Actions Example</title>
</head>
<body>
    <h1>Including Another JSP</h1>
    <jsp:include page="header.jsp" />
    <p>This is the main content of the page.</p>
    <jsp:include page="footer.jsp" />
</body>
</html>

Gestió d'Errors en JSP

  • Pàgines d'Error: Defineix pàgines d'error per gestionar excepcions.
  • isErrorPage: Atribut per especificar si una pàgina JSP és una pàgina d'error.
  • errorPage: Atribut per especificar la pàgina d'error per a una JSP.

Exemple: Definint una Pàgina d'Error

  1. Pàgina JSP Principal:
<!DOCTYPE html>
<html>
<head>
    <title>Error Handling Example</title>
</head>
<body>
    <h1>Error Handling in JSP</h1>
    <%
        // This will throw an exception
        int result = 10 / 0;
    %>
</body>
</html>
  1. Pàgina d'Error (error.jsp):
<%@ page isErrorPage="true" %>
<!DOCTYPE html>
<html>
<head>
    <title>Error Page</title>
</head>
<body>
    <h1>An Error Occurred</h1>
    <p>Error Message: <%= exception.getMessage() %></p>
</body>
</html>

Conclusió

JavaServer Pages (JSP) és una tecnologia potent per crear contingut web dinàmic. Entenent els conceptes bàsics de la sintaxi JSP, els objectes implícits, les accions i la gestió d'errors, pots construir aplicacions web robustes que separen la capa de presentació de la lògica de negoci. Practica creant les teves pròpies pàgines JSP i experimenta amb diferents característiques per aprofundir en la teva comprensió.
