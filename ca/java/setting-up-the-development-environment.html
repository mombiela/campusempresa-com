El teu Primer Programa en Java ►

Configurar l'entorn de desenvolupament és el primer pas en el teu viatge per convertir-te en un programador Java competent. Aquesta secció et guiarà a través dels passos necessaris per preparar el teu sistema per al desenvolupament en Java, des de la instal·lació del Java Development Kit (JDK) fins a la configuració d'un Entorn de Desenvolupament Integrat (IDE).

Instal·lació del Java Development Kit (JDK)

El Java Development Kit (JDK) és un entorn de desenvolupament de programari utilitzat per desenvolupar aplicacions Java. Inclou el Java Runtime Environment (JRE), un intèrpret/carregador (Java), un compilador (javac), un arxivador (jar), un generador de documentació (Javadoc) i altres eines necessàries per al desenvolupament en Java.

Passos per Instal·lar el JDK

  1. Descarregar el JDK:

    • Visita la pàgina de descàrrega de l'Oracle JDK.
    • Tria la versió adequada per al teu sistema operatiu (Windows, macOS o Linux).
    • Descarrega l'instal·lador.

  2. Instal·lar el JDK:

    • Executa l'instal·lador descarregat.
    • Segueix les instruccions en pantalla per completar la instal·lació.
    • Nota el directori d'instal·lació, ja que el necessitaràs més tard.

  3. Configurar les Variables d'Entorn:

    • Windows:

      • Obre el Menú d'Inici i cerca "Variables d'Entorn."
      • Fes clic a "Edita les variables d'entorn del sistema."
      • A la finestra de Propietats del Sistema, fes clic a "Variables d'Entorn."
      • Sota Variables del Sistema, fes clic a "Nou" i afegeix JAVA_HOME amb el camí al directori d'instal·lació del JDK.
      • Troba la variable Path, fes clic a "Edita" i afegeix %JAVA_HOME%\bin.

    • macOS/Linux:

      • Obre una terminal.
      • Edita el teu fitxer de perfil de shell (per exemple, ~/.bash_profile o ~/.zshrc) i afegeix les següents línies: 
        export JAVA_HOME=/path/to/your/jdk
export PATH=$JAVA_HOME/bin:$PATH
      • Desa el fitxer i executa source ~/.bash_profile o source ~/.zshrc per aplicar els canvis.

  4. Verificar la Instal·lació:

    • Obre una terminal o símbol del sistema.
    • Executa la comanda java -version per comprovar si Java està instal·lat correctament.
    • Hauries de veure una sortida similar a: 
      java version "17.0.1" 2021-10-19 LTS
Java(TM) SE Runtime Environment (build 17.0.1+12-LTS-39)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.1+12-LTS-39, mixed mode, sharing)

Configuració d'un Entorn de Desenvolupament Integrat (IDE)

Un Entorn de Desenvolupament Integrat (IDE) proporciona instal·lacions completes als programadors per al desenvolupament de programari. Els IDEs populars per a Java inclouen IntelliJ IDEA, Eclipse i NetBeans.

IntelliJ IDEA

  1. Descarregar i Instal·lar IntelliJ IDEA:

    • Visita la pàgina de descàrrega d'IntelliJ IDEA.
    • Tria l'edició Community (gratuïta) o l'edició Ultimate (de pagament).
    • Descarrega i executa l'instal·lador.

  2. Configurar IntelliJ IDEA:

    • Obre IntelliJ IDEA després de la instal·lació.
    • Segueix l'assistent de configuració inicial per configurar l'IDE.
    • Configura el camí del JDK si se't demana.

  3. Crear un Nou Projecte:

    • Fes clic a "Create New Project."
    • Selecciona "Java" de la llista de tipus de projectes.
    • Tria el JDK que vas instal·lar anteriorment.
    • Fes clic a "Next" i segueix les indicacions per crear el teu projecte.

Eclipse

  1. Descarregar i Instal·lar Eclipse:

    • Visita la pàgina de descàrrega d'Eclipse.
    • Descarrega el paquet "Eclipse IDE for Java Developers."
    • Extreu el fitxer descarregat i executa l'instal·lador.

  2. Configurar Eclipse:

    • Obre Eclipse després de la instal·lació.
    • Selecciona un directori de workspace on es guardaran els teus projectes.
    • Ves a Window > Preferences > Java > Installed JREs i afegeix el camí del JDK si no està detectat.

  3. Crear un Nou Projecte:

    • Ves a File > New > Java Project.
    • Introdueix el nom del projecte i configura el JDK.
    • Fes clic a "Finish" per crear el projecte.

NetBeans

  1. Descarregar i Instal·lar NetBeans:

    • Visita la pàgina de descàrrega de NetBeans.
    • Descarrega l'instal·lador per al teu sistema operatiu.
    • Executa l'instal·lador i segueix les instruccions en pantalla.

  2. Configurar NetBeans:

    • Obre NetBeans després de la instal·lació.
    • Ves a Tools > Java Platforms i afegeix el camí del JDK si no està detectat.

  3. Crear un Nou Projecte:

    • Ves a File > New Project.
    • Selecciona "Java" de les categories de projectes i "Java Application" dels tipus de projectes.
    • Segueix les indicacions per crear el teu projecte.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu viatge de programació en Java. Instal·lant el JDK i configurant un IDE com IntelliJ IDEA, Eclipse o NetBeans, crees una base sòlida per desenvolupar aplicacions Java. Amb el teu entorn preparat, ara pots centrar-te en escriure i executar codi Java de manera eficient. En les següents seccions, ens endinsarem en els conceptes bàsics de la programació en Java, començant per entendre la sintaxi i l'estructura d'un programa Java.
