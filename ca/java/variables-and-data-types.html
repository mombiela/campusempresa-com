Introducció

A Java, les variables i els tipus de dades són conceptes fonamentals que formen la base de tota la programació. Entendre com declarar, inicialitzar i utilitzar variables, així com conèixer els diferents tipus de dades disponibles, és crucial per escriure programes Java efectius.

Variables

Les variables són contenidors per emmagatzemar valors de dades. A Java, cada variable s'ha de declarar amb un tipus de dada específic, que determina el tipus de dades que pot contenir.

Declaració de Variables

Per declarar una variable a Java, cal especificar el tipus de dada seguit del nom de la variable.

int myNumber;

Inicialització de Variables

Pots inicialitzar una variable en el moment de la declaració o més tard en el codi.

int myNumber = 10; // Declaració i inicialització
myNumber = 20;     // Reassignació

Regles de Nomenclatura de Variables

  • Els noms de les variables són sensibles a majúscules i minúscules.
  • Han de començar amb una lletra, guió baix (_) o signe de dòlar ($).
  • Els caràcters posteriors poden ser lletres, dígits, guions baixos o signes de dòlar.
  • Les paraules reservades (com int, class, etc.) no es poden utilitzar com a noms de variables.

Tipus de Dades

Java és un llenguatge fortament tipat, el que significa que cada variable ha de tenir un tipus declarat. Els tipus de dades de Java es divideixen en dues categories: tipus de dades primitius i tipus de dades de referència.

Tipus de Dades Primitius

Java té 8 tipus de dades primitius:

| Tipus de Dada | Mida | Descripció | Valors d'Exemple | |---------------|---------|-----------------------------------|---------------------------| | byte | 8-bit | Emmagatzema nombres enters de -128 a 127 | -128, 0, 127 | | short | 16-bit | Emmagatzema nombres enters de -32,768 a 32,767 | -32768, 0, 32767 | | int | 32-bit | Emmagatzema nombres enters de -2^31 a 2^31-1 | -2147483648, 0, 2147483647 | | long | 64-bit | Emmagatzema nombres enters de -2^63 a 2^63-1 | -9223372036854775808, 0, 9223372036854775807 | | float | 32-bit | Emmagatzema nombres fraccionaris, suficient per a 6 a 7 dígits decimals | 3.14f, -0.001f | | double | 64-bit | Emmagatzema nombres fraccionaris, suficient per a 15 dígits decimals | 3.141592653589793, -0.0000001 | | boolean | 1-bit | Emmagatzema valors true o false | true, false | | char | 16-bit | Emmagatzema un sol caràcter/lletra o valors ASCII | 'a', 'Z', '1' |

Tipus de Dades de Referència

Els tipus de dades de referència són objectes i arrays. Es creen utilitzant constructors de classes.

String myString = "Hello, World!";
int[] myArray = {1, 2, 3, 4, 5};

Conversió de Tipus

Java admet tant la conversió de tipus implícita com explícita.

Conversió de Tipus Implícita

També coneguda com a conversió d'ampliació, es produeix automàticament quan un tipus de dada més petit s'assigna a un tipus de dada més gran.

int myInt = 9;
double myDouble = myInt; // Conversió automàtica: int a double

Conversió de Tipus Explícita

També coneguda com a conversió de reducció, requereix un càsting explícit.

double myDouble = 9.78;
int myInt = (int) myDouble; // Càsting manual: double a int

Exemples i Exercicis

Exemple 1: Declaració i Inicialització Bàsica de Variables

public class Main {
    public static void main(String[] args) {
        int age = 25;
        double salary = 50000.50;
        boolean isEmployed = true;
        char grade = 'A';
        
        System.out.println("Age: " + age);
        System.out.println("Salary: " + salary);
        System.out.println("Employed: " + isEmployed);
        System.out.println("Grade: " + grade);
    }
}

Exercici 1: Declarar i Inicialitzar Variables

  1. Declara una variable de tipus int anomenada year i inicialitza-la amb l'any actual.
  2. Declara una variable de tipus float anomenada temperature i inicialitza-la amb un valor de la teva elecció.
  3. Declara una variable de tipus boolean anomenada isJavaFun i inicialitza-la a true.

Exemple 2: Conversió de Tipus

public class Main {
    public static void main(String[] args) {
        int myInt = 9;
        double myDouble = myInt; // Conversió automàtica: int a double
        System.out.println("myInt: " + myInt);
        System.out.println("myDouble: " + myDouble);
        
        double anotherDouble = 9.78;
        int anotherInt = (int) anotherDouble; // Càsting manual: double a int
        System.out.println("anotherDouble: " + anotherDouble);
        System.out.println("anotherInt: " + anotherInt);
    }
}

Exercici 2: Conversió de Tipus

  1. Declara una variable de tipus double anomenada pi i inicialitza-la a 3.14159.
  2. Converteix pi a un int i emmagatzema-ho en una variable anomenada piInt.
  3. Imprimeix tant pi com piInt.

Conclusió

Entendre les variables i els tipus de dades és essencial per a qualsevol programador Java. Les variables emmagatzemen dades, i els tipus de dades defineixen el tipus de dades que una variable pot contenir. El sistema de tipatge fort de Java assegura que les variables s'utilitzin correctament, reduint errors i millorant la fiabilitat del codi. Practica declarant, inicialitzant i convertint variables per convertir-te en un expert en la gestió de dades a Java.
