Connectar-se a una base de dades és una habilitat fonamental per a qualsevol desenvolupador Java. Aquest tema et guiarà a través del procés d'establir una connexió a una base de dades, executar consultes i gestionar els resultats. Cobrirem tant conceptes bàsics com avançats, assegurant-nos que tinguis una comprensió sòlida de la connectivitat de bases de dades en Java.

Introducció a JDBC

Java Database Connectivity (JDBC) és una API que permet a les aplicacions Java interactuar amb una àmplia gamma de bases de dades. JDBC proporciona mètodes per consultar i actualitzar dades en una base de dades, i és una part crucial de la biblioteca estàndard de Java.

Conceptes Clau

  • Driver: Un component de programari que permet a les aplicacions Java interactuar amb una base de dades.
  • Connexió: Una sessió entre l'aplicació Java i la base de dades.
  • Statement: Una interfície utilitzada per executar consultes SQL.
  • ResultSet: Una taula de dades que representa el resultat d'una consulta.

Configuració del Teu Entorn

Abans de poder connectar-te a una base de dades, necessites configurar el teu entorn de desenvolupament.

Passos

  1. Instal·la una Base de Dades: Instal·la una base de dades com MySQL, PostgreSQL o SQLite.
  2. Afegeix el Driver JDBC al Teu Projecte: Descarrega el driver JDBC per a la teva base de dades i afegeix-lo al classpath del teu projecte.

Exemple: Afegir el Driver JDBC de MySQL

Si estàs utilitzant Maven, afegeix la següent dependència al teu pom.xml:

<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.26</version>
</dependency>

Establir una Connexió

Per connectar-te a una base de dades, necessites carregar el driver JDBC i establir una connexió utilitzant la classe DriverManager.

Exemple de Codi

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseConnection {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String user = "root";
        String password = "password";

        try {
            // Carregar el driver JDBC
            Class.forName("com.mysql.cj.jdbc.Driver");

            // Establir la connexió
            Connection connection = DriverManager.getConnection(url, user, password);
            System.out.println("Connexió establerta amb èxit!");

            // Tancar la connexió
            connection.close();
        } catch (ClassNotFoundException e) {
            System.out.println("Driver JDBC no trobat.");
            e.printStackTrace();
        } catch (SQLException e) {
            System.out.println("No s'ha pogut establir la connexió.");
            e.printStackTrace();
        }
    }
}

Explicació

  • Class.forName("com.mysql.cj.jdbc.Driver"): Carrega el driver JDBC de MySQL.
  • DriverManager.getConnection(url, user, password): Estableix una connexió a la base de dades.
  • connection.close(): Tanca la connexió per alliberar recursos.

Executar Consultes SQL

Un cop hagis establert una connexió, pots executar consultes SQL utilitzant les interfícies Statement o PreparedStatement.

Exemple de Codi: Executar una Consulta

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.sql.Statement;

public class ExecuteQuery {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String user = "root";
        String password = "password";

        try {
            Connection connection = DriverManager.getConnection(url, user, password);
            Statement statement = connection.createStatement();

            // Executar una consulta
            String query = "SELECT * FROM users";
            ResultSet resultSet = statement.executeQuery(query);

            // Processar el result set
            while (resultSet.next()) {
                System.out.println("ID d'Usuari: " + resultSet.getInt("id"));
                System.out.println("Nom d'Usuari: " + resultSet.getString("username"));
            }

            // Tancar recursos
            resultSet.close();
            statement.close();
            connection.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

Explicació

  • statement.executeQuery(query): Executa la consulta SQL i retorna un ResultSet.
  • resultSet.next(): Mou el cursor a la següent fila del result set.
  • resultSet.getInt("id"): Recupera el valor de la columna "id" com a enter.
  • resultSet.getString("username"): Recupera el valor de la columna "username" com a cadena.

Utilitzar Sentències Preparades

Les sentències preparades s'utilitzen per executar consultes amb paràmetres, cosa que ajuda a prevenir atacs d'injecció SQL i millorar el rendiment.

Exemple de Codi: Utilitzar PreparedStatement

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.sql.SQLException;

public class PreparedStatementExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String user = "root";
        String password = "password";

        try {
            Connection connection = DriverManager.getConnection(url, user, password);

            // Crear una sentència preparada
            String query = "SELECT * FROM users WHERE username = ?";
            PreparedStatement preparedStatement = connection.prepareStatement(query);
            preparedStatement.setString(1, "john_doe");

            // Executar la consulta
            ResultSet resultSet = preparedStatement.executeQuery();

            // Processar el result set
            while (resultSet.next()) {
                System.out.println("ID d'Usuari: " + resultSet.getInt("id"));
                System.out.println("Nom d'Usuari: " + resultSet.getString("username"));
            }

            // Tancar recursos
            resultSet.close();
            preparedStatement.close();
            connection.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

Explicació

  • connection.prepareStatement(query): Crea un objecte PreparedStatement per enviar sentències SQL amb paràmetres.
  • preparedStatement.setString(1, "john_doe"): Estableix el primer paràmetre de la consulta a "john_doe".
  • preparedStatement.executeQuery(): Executa la consulta i retorna un ResultSet.

Gestionar Excepcions

La gestió adequada d'excepcions és crucial quan es treballa amb bases de dades per assegurar-se que els recursos es tanquin i els errors es registrin.

Exemple de Codi: Gestionar Excepcions

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import java.sql.Statement;

public class ExceptionHandlingExample {
    public static void main(String[] args) {
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String user = "root";
        String password = "password";

        Connection connection = null;
        Statement statement = null;

        try {
            connection = DriverManager.getConnection(url, user, password);
            statement = connection.createStatement();

            // Executar una actualització
            String update = "UPDATE users SET username = 'new_username' WHERE id = 1";
            statement.executeUpdate(update);

        } catch (SQLException e) {
            e.printStackTrace();
        } finally {
            // Tancar recursos al bloc finally
            try {
                if (statement != null) statement.close();
                if (connection != null) connection.close();
            } catch (SQLException e) {
                e.printStackTrace();
            }
        }
    }
}

Explicació

  • try-catch-finally: Assegura que els recursos es tanquin fins i tot si es produeix una excepció.
  • statement.close() i connection.close(): Tanquen la sentència i la connexió al bloc finally per assegurar-se que sempre s'executin.

Conclusió

Connectar-se a una base de dades en Java implica diversos passos, des de configurar el teu entorn fins a executar consultes i gestionar els resultats. Entenent i practicant aquests conceptes, pots interactuar eficientment amb bases de dades en les teves aplicacions Java. Recorda sempre gestionar les excepcions adequadament i tancar els recursos per mantenir l'estabilitat i el rendiment de la teva aplicació.
