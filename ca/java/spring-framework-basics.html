Introducció al Framework Spring

El Framework Spring és un framework potent i ric en funcionalitats per construir aplicacions Java. Proporciona suport d'infraestructura integral per al desenvolupament d'aplicacions Java, facilitant la gestió i configuració de les aplicacions.

  • Conceptes Clau:
    • Inversió de Control (IoC): Un principi de disseny on el control dels objectes o parts d'un programa es transfereix a un contenidor o framework.
    • Injecció de Dependències (DI): Una tècnica per aconseguir IoC, on les dependències són injectades pel framework en lloc de ser creades pels objectes mateixos.
    • Programació Orientada a Aspectes (AOP): Un paradigma de programació que té com a objectiu augmentar la modularitat permetent la separació de preocupacions transversals.

Configuració del Framework Spring

Abans d'endinsar-se en el Framework Spring, cal configurar l'entorn de desenvolupament.

  • Passos per Configurar:
    1. Instal·lar el Java Development Kit (JDK): Assegura't de tenir el JDK instal·lat a la teva màquina.
    2. Configurar Maven o Gradle: Aquests eines de construcció ajuden a gestionar les dependències i construir el projecte.
    3. Afegir Dependències de Spring: Afegeix les dependències de Spring al teu pom.xml (per Maven) o build.gradle (per Gradle).

Exemple: Afegir Dependències de Spring amb Maven

<dependencies>
    <dependency>
        <groupId>org.springframework</groupId>
        <artifactId>spring-context</artifactId>
        <version>5.3.10</version>
    </dependency>
</dependencies>

Conceptes Bàsics del Framework Spring

Inversió de Control (IoC) i Injecció de Dependències (DI)

IoC i DI són els conceptes bàsics del Framework Spring. Ajuden a crear aplicacions poc acoblades.

  • Contenidor IoC: El contenidor que gestiona el cicle de vida i la configuració dels objectes de l'aplicació.
  • Bean: Un objecte que és instanciat, assemblat i gestionat pel contenidor IoC de Spring.
  • Tipus d'Injecció de Dependències:
    • Injecció per Constructor
    • Injecció per Setter

Exemple: Injecció per Constructor

public class TextEditor {
    private SpellChecker spellChecker;

    // Injecció per Constructor
    public TextEditor(SpellChecker spellChecker) {
        this.spellChecker = spellChecker;
    }

    public void spellCheck() {
        spellChecker.checkSpelling();
    }
}

Exemple: Injecció per Setter

public class TextEditor {
    private SpellChecker spellChecker;

    // Injecció per Setter
    public void setSpellChecker(SpellChecker spellChecker) {
        this.spellChecker = spellChecker;
    }

    public void spellCheck() {
        spellChecker.checkSpelling();
    }
}

Configuració de Spring

Spring es pot configurar utilitzant XML, anotacions o configuració basada en Java.

Configuració Basada en XML

<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans
       http://www.springframework.org/schema/beans/spring-beans.xsd">

    <bean id="spellChecker" class="com.example.SpellChecker"/>
    <bean id="textEditor" class="com.example.TextEditor">
        <constructor-arg ref="spellChecker"/>
    </bean>
</beans>

Configuració Basada en Anotacions

@Component
public class SpellChecker {
    public void checkSpelling() {
        System.out.println("Checking spelling...");
    }
}

@Component
public class TextEditor {
    private final SpellChecker spellChecker;

    @Autowired
    public TextEditor(SpellChecker spellChecker) {
        this.spellChecker = spellChecker;
    }

    public void spellCheck() {
        spellChecker.checkSpelling();
    }
}

Configuració Basada en Java

@Configuration
public class AppConfig {

    @Bean
    public SpellChecker spellChecker() {
        return new SpellChecker();
    }

    @Bean
    public TextEditor textEditor() {
        return new TextEditor(spellChecker());
    }
}

Spring Boot

Spring Boot simplifica el desenvolupament d'aplicacions Spring proporcionant un conjunt de convencions i eines.

  • Característiques Clau:
    • Auto-Configuració: Configura automàticament la teva aplicació Spring basant-se en les dependències que has afegit.
    • Aplicacions Independents: Et permet crear aplicacions independents amb servidors incrustats.
    • Característiques Preparades per a Producció: Inclou mètriques, comprovacions de salut i configuració externalitzada.

Exemple: Crear una Aplicació Spring Boot

@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

Conclusió

El Framework Spring és un framework complet i versàtil per construir aplicacions Java. Entenent els seus conceptes bàsics com IoC, DI i AOP, i aprenent a configurar-lo utilitzant XML, anotacions o configuració basada en Java, pots construir aplicacions robustes i mantenibles. Spring Boot simplifica encara més el procés de desenvolupament proporcionant auto-configuració i característiques preparades per a producció, facilitant la creació i desplegament d'aplicacions ràpidament.
