Introducció

File I/O (Input/Output) és un concepte fonamental en la programació en Java que permet als programes llegir i escriure fitxers. Aquesta secció cobrirà els conceptes bàsics de la gestió de fitxers en Java, incloent la lectura de fitxers, l'escriptura de fitxers i la gestió d'excepcions.

Conceptes clau

  • Classe File: Representa un fitxer o un camí de directori.
  • FileReader i FileWriter: Classes per llegir i escriure fitxers de caràcters.
  • BufferedReader i BufferedWriter: Classes per a una lectura i escriptura eficients.
  • FileInputStream i FileOutputStream: Classes per llegir i escriure fitxers binaris.
  • Gestió d'excepcions: Gestionar errors que es produeixen durant les operacions de fitxers.

Treballant amb la classe File

La classe File en Java s'utilitza per representar noms de camins de fitxers i directoris. Proporciona mètodes per crear, eliminar i comprovar propietats de fitxers i directoris.

Exemple: Creant un objecte File

import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");
        
        if (file.exists()) {
            System.out.println("El fitxer existeix.");
        } else {
            System.out.println("El fitxer no existeix.");
        }
    }
}

En aquest exemple, creem un objecte File que representa un fitxer anomenat "example.txt" i comprovem si existeix.

Llegint d'un fitxer

Per llegir d'un fitxer, pots utilitzar FileReader i BufferedReader per a fitxers de caràcters, o FileInputStream per a fitxers binaris.

Exemple: Llegint un fitxer de text

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

En aquest exemple, s'utilitza BufferedReader per llegir el fitxer línia per línia. La declaració try-with-resources assegura que el fitxer es tanqui automàticament.

Escriure en un fitxer

Per escriure en un fitxer, pots utilitzar FileWriter i BufferedWriter per a fitxers de caràcters, o FileOutputStream per a fitxers binaris.

Exemple: Escriure en un fitxer de text

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("example.txt"))) {
            bw.write("Hola, món!");
            bw.newLine();
            bw.write("Això és una prova.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

En aquest exemple, s'utilitza BufferedWriter per escriure text al fitxer. La declaració try-with-resources assegura que el fitxer es tanqui automàticament.

Gestió d'excepcions

Les operacions de File I/O poden llançar diverses excepcions, com ara FileNotFoundException i IOException. Una gestió adequada d'excepcions és crucial per assegurar que el teu programa pugui gestionar errors de manera elegant.

Exemple: Gestió d'excepcions

import java.io.File;
import java.io.FileNotFoundException;
import java.io.FileReader;
import java.io.IOException;

public class ExceptionHandlingExample {
    public static void main(String[] args) {
        File file = new File("nonexistent.txt");
        try (FileReader fr = new FileReader(file)) {
            // Llegir el contingut del fitxer
        } catch (FileNotFoundException e) {
            System.out.println("Fitxer no trobat: " + file.getName());
        } catch (IOException e) {
            System.out.println("S'ha produït un error d'I/O: " + e.getMessage());
        }
    }
}

En aquest exemple, gestionem FileNotFoundException i IOException per separat per proporcionar missatges d'error específics.

Resum

En aquesta secció, hem cobert els conceptes bàsics de File I/O en Java, incloent:

  • Crear i utilitzar la classe File.
  • Llegir i escriure fitxers de text utilitzant FileReader, FileWriter, BufferedReader i BufferedWriter.
  • Gestionar excepcions que poden ocórrer durant les operacions de fitxers.

Entendre aquests conceptes és essencial per treballar amb fitxers en Java, i formen la base per a tècniques més avançades de gestió de fitxers.
