Introducció als Constructors

Els constructors són mètodes especials a Java que s'utilitzen per inicialitzar objectes. Es criden quan es crea una instància d'una classe. A diferència dels mètodes regulars, els constructors no tenen un tipus de retorn, ni tan sols void.

Conceptes Clau

  • Propòsit: Inicialitzar nous objectes.
  • Nom: Ha de coincidir amb el nom de la classe.
  • Sense Tipus de Retorn: Els constructors no tenen un tipus de retorn.
  • Constructor per Defecte: Si no es defineix cap constructor, Java proporciona un constructor per defecte.
  • Sobrecàrrega: Els constructors es poden sobrecarregar per proporcionar diferents maneres d'inicialitzar objectes.

Ús Bàsic del Constructor

Constructor per Defecte

Si no defineixes cap constructor, Java proporciona automàticament un constructor per defecte que inicialitza l'objecte amb valors per defecte.

public class Car {
    // Camps
    String model;
    int year;

    // Constructor per defecte proporcionat per Java
    public Car() {
        // Sense codi d'inicialització
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car(); // Crida al constructor per defecte
        System.out.println(myCar.model); // null
        System.out.println(myCar.year);  // 0
    }
}

Constructor Parametritzat

Pots definir els teus propis constructors per inicialitzar objectes amb valors específics.

public class Car {
    // Camps
    String model;
    int year;

    // Constructor parametritzat
    public Car(String model, int year) {
        this.model = model;
        this.year = year;
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Toyota", 2021); // Crida al constructor parametritzat
        System.out.println(myCar.model); // Toyota
        System.out.println(myCar.year);  // 2021
    }
}

Sobrecàrrega de Constructors

Pots tenir múltiples constructors en una classe amb diferents paràmetres. Això es coneix com a sobrecàrrega de constructors.

public class Car {
    // Camps
    String model;
    int year;

    // Constructor per defecte
    public Car() {
        this.model = "Unknown";
        this.year = 0;
    }

    // Constructor parametritzat
    public Car(String model, int year) {
        this.model = model;
        this.year = year;
    }

    // Un altre constructor parametritzat
    public Car(String model) {
        this.model = model;
        this.year = 2022; // Any per defecte
    }
}

public class Main {
    public static void main(String[] args) {
        Car car1 = new Car(); // Crida al constructor per defecte
        Car car2 = new Car("Honda", 2020); // Crida al constructor parametritzat
        Car car3 = new Car("Ford"); // Crida a un altre constructor parametritzat

        System.out.println(car1.model + " " + car1.year); // Unknown 0
        System.out.println(car2.model + " " + car2.year); // Honda 2020
        System.out.println(car3.model + " " + car3.year); // Ford 2022
    }
}

Temes Avançats

Enllaç de Constructors

L'enllaç de constructors és el procés de cridar un constructor des d'un altre constructor dins de la mateixa classe o des de la classe pare.

Utilitzant this()

Pots cridar un altre constructor en la mateixa classe utilitzant this().

public class Car {
    String model;
    int year;

    // Constructor per defecte
    public Car() {
        this("Unknown", 0); // Crida al constructor parametritzat
    }

    // Constructor parametritzat
    public Car(String model, int year) {
        this.model = model;
        this.year = year;
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car(); // Crida al constructor per defecte, que crida al constructor parametritzat
        System.out.println(car.model + " " + car.year); // Unknown 0
    }
}

Utilitzant super()

Pots cridar un constructor des de la classe pare utilitzant super().

class Vehicle {
    String type;

    // Constructor de la classe pare
    public Vehicle(String type) {
        this.type = type;
    }
}

public class Car extends Vehicle {
    String model;
    int year;

    // Constructor de la classe fill
    public Car(String type, String model, int year) {
        super(type); // Crida al constructor de la classe pare
        this.model = model;
        this.year = year;
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car("Sedan", "Toyota", 2021);
        System.out.println(car.type + " " + car.model + " " + car.year); // Sedan Toyota 2021
    }
}

Resum

Els constructors són essencials per inicialitzar objectes a Java. Poden ser per defecte o parametritzats, i es poden sobrecarregar per proporcionar múltiples maneres de creació d'objectes. Conceptes avançats com l'enllaç de constructors permeten un codi més flexible i mantenible. Entendre els constructors és fonamental per a una programació efectiva en Java.
