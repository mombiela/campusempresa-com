Visió General

El Desenvolupament Web amb Java és una àrea potent i versàtil de la programació en Java que permet als desenvolupadors crear aplicacions web dinàmiques i interactives. Aquesta secció t'introduirà als conceptes fonamentals, eines i frameworks utilitzats en el Desenvolupament Web amb Java.

Conceptes Clau

  • Servidors Web i Servidors d'Aplicacions: Entén la diferència entre servidors web (per exemple, Apache HTTP Server) i servidors d'aplicacions (per exemple, Apache Tomcat, JBoss).
  • Servlets: Aprèn sobre els Java Servlets, que són programes Java que s'executen en un servidor i gestionen les sol·licituds dels clients.
  • JavaServer Pages (JSP): Descobreix com JSP et permet crear pàgines web generades dinàmicament basades en HTML, XML o altres tipus de documents.
  • Frameworks: Introdueix-te als frameworks web populars de Java com Spring i JavaServer Faces (JSF).

Configuració del Teu Entorn

Abans de submergir-te en la codificació, necessites configurar el teu entorn de desenvolupament.

  1. Instal·la el Java Development Kit (JDK): Assegura't de tenir instal·lada la versió més recent del JDK.
  2. Entorn de Desenvolupament Integrat (IDE): Utilitza un IDE com IntelliJ IDEA, Eclipse o NetBeans.
  3. Servidor Web: Instal·la un servidor web com Apache Tomcat.

Creant el Teu Primer Servlet

Un Servlet és una classe Java que amplia les capacitats dels servidors que allotgen aplicacions accedides mitjançant un model de programació de sol·licitud-resposta.

Exemple: HelloWorldServlet

import java.io.IOException;
import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;

@WebServlet("/hello")
public class HelloWorldServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        response.setContentType("text/html");
        response.getWriter().println("<h1>Hello, World!</h1>");
    }
}
  • Explicació:
    • @WebServlet("/hello"): Aquesta anotació mapatge el servlet al patró d'URL /hello.
    • doGet(HttpServletRequest request, HttpServletResponse response): Aquest mètode gestiona les sol·licituds GET.
    • response.setContentType("text/html"): Estableix el tipus de contingut de la resposta a HTML.
    • response.getWriter().println("<h1>Hello, World!</h1>"): Escriu la resposta HTML.

JavaServer Pages (JSP)

JSP et permet incrustar codi Java en pàgines HTML per crear contingut dinàmic.

Exemple: HelloWorld.jsp

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>Hello World JSP</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
  • Explicació:
    • <%@ page contentType="text/html;charset=UTF-8" language="java" %>: Directiva de pàgina que estableix el tipus de contingut i la codificació de caràcters.
    • La resta del fitxer és HTML estàndard amb codi Java incrustat.
