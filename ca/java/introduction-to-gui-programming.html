Visió General

La programació d'interfícies gràfiques d'usuari (GUI) permet als desenvolupadors crear aplicacions visualment interactives. En Java, la programació GUI es fa principalment utilitzant les biblioteques Swing i JavaFX. Aquesta secció t'introduirà als conceptes bàsics de la programació GUI en Java, cobrint conceptes essencials, components i exemples pràctics.

Conceptes Clau

  • Programació Basada en Esdeveniments: Les aplicacions GUI són basades en esdeveniments, és a dir, responen a accions de l'usuari com clics, premses de tecles, etc.
  • Components: Blocs de construcció bàsics d'una GUI, com botons, etiquetes, camps de text, etc.
  • Contenidors: Contenen i organitzen components, com panells, marcs i finestres.
  • Gestors de Disseny: Defineixen com es disposen els components dins dels contenidors.

Començant amb Swing

Swing és una part de les Classes de Fonaments de Java (JFC) i proporciona un conjunt ric de components GUI.

Creant una Finestra Simple

Per crear una finestra bàsica en Swing, has d'utilitzar la classe JFrame.

import javax.swing.JFrame;

public class SimpleWindow {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Simple Window");
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació:
    • JFrame: Un contenidor de nivell superior que representa una finestra.
    • setSize(): Estableix la mida de la finestra.
    • setDefaultCloseOperation(): Defineix l'operació per defecte quan es tanca la finestra.
    • setVisible(): Fa que la finestra sigui visible.

Afegint Components

Afegim un botó a la finestra.

import javax.swing.JButton;
import javax.swing.JFrame;

public class ButtonExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Example");
        JButton button = new JButton("Click Me");
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(button); // Afegint el botó al marc
        frame.setVisible(true);
    }
}
  • Explicació:
    • JButton: Un component que representa un botó.
    • add(): Afegeix el botó al marc.

Gestionant Esdeveniments

Per gestionar clics de botó, has d'implementar un ActionListener.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonClickExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Click Example");
        JButton button = new JButton("Click Me");
        
        button.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                System.out.println("Button was clicked!");
            }
        });
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.add(button);
        frame.setVisible(true);
    }
}
  • Explicació:
    • ActionListener: Una interfície per rebre esdeveniments d'acció.
    • actionPerformed(): Mètode que es crida quan es fa clic al botó.

Gestors de Disseny

Els gestors de disseny controlen la posició i la mida dels components dins d'un contenidor.

BorderLayout

BorderLayout divideix el contenidor en cinc regions: Nord, Sud, Est, Oest i Centre.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.BorderLayout;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setLayout(new BorderLayout());
        
        frame.add(new JButton("North"), BorderLayout.NORTH);
        frame.add(new JButton("South"), BorderLayout.SOUTH);
        frame.add(new JButton("East"), BorderLayout.EAST);
        frame.add(new JButton("West"), BorderLayout.WEST);
        frame.add(new JButton("Center"), BorderLayout.CENTER);
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació:
    • BorderLayout: Un gestor de disseny que disposa els components en cinc regions.

GridLayout

GridLayout disposa els components en una graella de cel·les, cadascuna de mida igual.

import javax.swing.JButton;
import javax.swing.JFrame;
import java.awt.GridLayout;

public class GridLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("GridLayout Example");
        frame.setLayout(new GridLayout(2, 2)); // 2 files, 2 columnes
        
        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));
        frame.add(new JButton("Button 4"));
        
        frame.setSize(400, 300);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació:
    • GridLayout: Un gestor de disseny que disposa els components en una graella.

Introducció a JavaFX

JavaFX és un conjunt d'eines GUI modern per a Java, que proporciona API riques per desenvolupar aplicacions d'escriptori.

Creant una Aplicació JavaFX Simple

Per crear una aplicació JavaFX bàsica, has d'estendre la classe Application i sobreescriure el mètode start().

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.stage.Stage;

public class SimpleJavaFXApp extends Application {
    @Override
    public void start(Stage primaryStage) {
        Button button = new Button("Click Me");
        button.setOnAction(e -> System.out.println("Button was clicked!"));
        
        Scene scene = new Scene(button, 400, 300);
        primaryStage.setTitle("Simple JavaFX Application");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
    
    public static void main(String[] args) {
        launch(args);
    }
}
  • Explicació:
    • Application: La classe base per a aplicacions JavaFX.
    • start(): El punt d'entrada principal per a aplicacions JavaFX.
    • Stage: La finestra principal.
    • Scene: El contenidor per a tot el contingut en un gràfic d'escena.
    • Button: Un component de botó JavaFX.
