Introducció a l'Estructura de Col·leccions

L'Estructura de Col·leccions de Java proporciona un conjunt d'interfícies i classes per gestionar un grup d'objectes com una unitat única. És part del paquet java.util i inclou diverses estructures de dades com llistes, conjunts i mapes.

  • Conceptes Clau:
    • Interfície Collection: La interfície arrel per a totes les classes de col·leccions.
    • Interfície List: Una col·lecció ordenada (també coneguda com a seqüència).
    • Interfície Set: Una col·lecció que no permet elements duplicats.
    • Interfície Map: Un objecte que associa claus a valors, sense permetre claus duplicades.

Interfície Collection

La interfície Collection és la interfície arrel de l'Estructura de Col·leccions de Java. Proporciona mètodes bàsics que totes les col·leccions han d'implementar.

  • Mètodes Comuns:
    • add(E e): Afegeix un element a la col·lecció.
    • remove(Object o): Elimina un element especificat de la col·lecció.
    • size(): Retorna el nombre d'elements a la col·lecció.
    • clear(): Elimina tots els elements de la col·lecció.
    • isEmpty(): Comprova si la col·lecció està buida.
Collection<String> collection = new ArrayList<>();
collection.add("Hello");
collection.add("World");
System.out.println(collection.size()); // Output: 2

Interfície List

La interfície List amplia la interfície Collection i representa una col·lecció ordenada. Els elements es poden accedir pel seu índex enter.

  • Implementacions:

    • ArrayList: Una implementació de matriu redimensionable.
    • LinkedList: Una implementació de llista doblement enllaçada.

  • Mètodes Comuns:

    • get(int index): Retorna l'element a la posició especificada.
    • set(int index, E element): Reemplaça l'element a la posició especificada.
    • add(int index, E element): Insereix l'element a la posició especificada.
    • remove(int index): Elimina l'element a la posició especificada.
List<String> list = new ArrayList<>();
list.add("Apple");
list.add("Banana");
list.add(1, "Cherry");
System.out.println(list.get(1)); // Output: Cherry

Interfície Set

La interfície Set amplia la interfície Collection i representa una col·lecció que no permet elements duplicats.

  • Implementacions:

    • HashSet: Una implementació de taula de hash.
    • LinkedHashSet: Una implementació de taula de hash i llista enllaçada.
    • TreeSet: Una implementació d'arbre Red-Black.

  • Mètodes Comuns:

    • add(E e): Afegeix l'element especificat si no està ja present.
    • remove(Object o): Elimina l'element especificat si està present.
Set<String> set = new HashSet<>();
set.add("One");
set.add("Two");
set.add("One"); // Element duplicat, no s'afegirà
System.out.println(set.size()); // Output: 2

Interfície Map

La interfície Map no amplia la interfície Collection. Representa una col·lecció de parells clau-valor.

  • Implementacions:

    • HashMap: Una implementació de taula de hash.
    • LinkedHashMap: Una implementació de taula de hash i llista enllaçada.
    • TreeMap: Una implementació d'arbre Red-Black.

  • Mètodes Comuns:

    • put(K key, V value): Associa el valor especificat amb la clau especificada.
    • get(Object key): Retorna el valor associat amb la clau especificada.
    • remove(Object key): Elimina l'associació per a la clau especificada.
    • keySet(): Retorna una vista de conjunt de les claus.
    • values(): Retorna una vista de col·lecció dels valors.
Map<String, Integer> map = new HashMap<>();
map.put("A", 1);
map.put("B", 2);
System.out.println(map.get("A")); // Output: 1
map.remove("A");
System.out.println(map.size()); // Output: 1

Comparació de List, Set i Map

| Característica | List | Set | Map | |------------------|--------------------|--------------------|--------------------| | Permet Duplicats | Sí | No | Claus: No, Valors: Sí | | Ordenat | Sí | No (excepte LinkedHashSet) | No (excepte LinkedHashMap) | | Accés per Índex | Sí | No | No | | Parell Clau-Valor| No | No | Sí |

Conclusió

L'Estructura de Col·leccions de Java és una eina potent i flexible per gestionar grups d'objectes. Entendre les diferents interfícies i les seves implementacions et permetrà triar l'estructura de dades adequada per a les teves necessitats. Practica utilitzant aquestes col·leccions per convertir-te en un expert en la gestió de dades en Java.
