Introducció als Arrays

Els arrays són una estructura de dades fonamental a Java que et permeten emmagatzemar múltiples valors del mateix tipus en una sola variable. Són útils per gestionar col·leccions de dades de manera eficient.

Conceptes Clau

  • Definició: Un array és un objecte contenidor que manté un nombre fix de valors d'un sol tipus.
  • Indexació: Els arrays estan indexats a partir de zero, és a dir, el primer element està a l'índex 0.
  • Mida Fixa: Un cop creat un array, la seva mida no es pot canviar.

Creació i Inicialització d'Arrays

Per crear un array a Java, cal especificar el tipus d'elements que contindrà i la seva mida.

// Declaració i instanciació
int[] numbers = new int[5];

// Inicialització
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

També pots declarar, instanciar i inicialitzar un array en una sola línia:

int[] numbers = {10, 20, 30, 40, 50};

Accés als Elements de l'Array

Pots accedir als elements d'un array utilitzant el seu índex:

int firstNumber = numbers[0]; // 10
int secondNumber = numbers[1]; // 20

Operacions Comunes amb Arrays

  • Longitud: Pots obtenir la longitud d'un array utilitzant la propietat .length.
  • Iteració a través dels Arrays: Pots utilitzar un bucle for o un bucle for millorat per iterar a través dels elements de l'array.
// Utilitzant un bucle for
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// Utilitzant un bucle for millorat
for (int number : numbers) {
    System.out.println(number);
}

Conceptes Avançats d'Arrays

Arrays Multidimensionals

Java suporta arrays multidimensionals, que són arrays d'arrays.

// Declaració i instanciació d'un array 2D
int[][] matrix = new int[3][3];

// Inicialització
matrix[0][0] = 1;
matrix[0][1] = 2;
matrix[0][2] = 3;
matrix[1][0] = 4;
matrix[1][1] = 5;
matrix[1][2] = 6;
matrix[2][0] = 7;
matrix[2][1] = 8;
matrix[2][2] = 9;

També pots inicialitzar un array 2D en una sola línia:

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Mètodes i Utilitats d'Arrays

Java proporciona diversos mètodes d'utilitat per als arrays a la classe java.util.Arrays.

  • Ordenació: Pots ordenar un array utilitzant Arrays.sort().
import java.util.Arrays;

int[] numbers = {5, 3, 8, 1, 2};
Arrays.sort(numbers); // {1, 2, 3, 5, 8}
  • Cerca: Pots cercar un element utilitzant Arrays.binarySearch().
int index = Arrays.binarySearch(numbers, 3); // Retorna l'índex de l'element 3
  • Còpia: Pots copiar un array utilitzant Arrays.copyOf().
int[] copy = Arrays.copyOf(numbers, numbers.length);

Errors Comuns i Millors Pràctiques

  • Fora de Límits: Accedir a un índex fora dels límits de l'array llançarà una ArrayIndexOutOfBoundsException.
  • Arrays Nuls: Sempre comprova si un array és nul abans d'accedir als seus elements per evitar una NullPointerException.
  • Mida Fixa: Recorda que els arrays tenen una mida fixa. Si necessites un array redimensionable, considera utilitzar ArrayList.

Exercicis

  1. Operacions Bàsiques amb Arrays: Crea un array d'enters, inicialitza'l amb valors i imprimeix cada valor utilitzant un bucle.
  2. Array Multidimensional: Crea un array 2D i inicialitza'l amb valors. Escriu un programa per imprimir l'array 2D en forma de matriu.
  3. Utilitats d'Arrays: Utilitza Arrays.sort() per ordenar un array i Arrays.binarySearch() per trobar un element en l'array ordenat.

Conclusió

Els arrays són una part poderosa i essencial de la programació en Java. Et permeten emmagatzemar i gestionar col·leccions de dades de manera eficient. Entendre com crear, manipular i utilitzar arrays és crucial per a qualsevol programador de Java. A mesura que avancis, trobaràs que els arrays són un concepte fonamental que suporta estructures de dades i algorismes més complexos.
