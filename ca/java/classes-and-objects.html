Introducció a les Classes i Objectes

En Java, les classes i els objectes són conceptes fonamentals que formen la base de la programació orientada a objectes (OOP). Entendre aquests conceptes és crucial per escriure codi eficient i organitzat.

Conceptes Clau

  • Classe: Un pla per crear objectes. Defineix un tipus de dada agrupant dades i mètodes que treballen sobre les dades en una sola unitat.
  • Objecte: Una instància d'una classe. Es crea a partir d'una classe i pot utilitzar els mètodes definits a la classe.

Definició d'una Classe

Una classe en Java es defineix utilitzant la paraula clau class seguida del nom de la classe i un parell de claus {}.

public class Car {
    // Camps (o variables d'instància)
    String color;
    String model;
    int year;

    // Mètodes
    void displayDetails() {
        System.out.println("Model: " + model);
        System.out.println("Color: " + color);
        System.out.println("Year: " + year);
    }
}

Explicació

  • Camps: Variables que mantenen l'estat de l'objecte.
  • Mètodes: Funcions que defineixen el comportament de l'objecte.

Creació d'Objectes

Els objectes es creen utilitzant la paraula clau new seguida del constructor de la classe.

public class Main {
    public static void main(String[] args) {
        // Creació d'un objecte de la classe Car
        Car myCar = new Car();
        
        // Assignació de valors als camps
        myCar.color = "Red";
        myCar.model = "Toyota";
        myCar.year = 2021;
        
        // Crida a un mètode
        myCar.displayDetails();
    }
}

Explicació

  • Instanciació: Car myCar = new Car(); crea un nou objecte de la classe Car.
  • Accés a Camps i Mètodes: Utilitza l'operador punt . per accedir als camps i mètodes de l'objecte.

Constructors

Els constructors són mètodes especials que es criden quan un objecte és instanciat. Inicialitzen l'objecte.

public class Car {
    String color;
    String model;
    int year;

    // Constructor
    public Car(String color, String model, int year) {
        this.color = color;
        this.model = model;
        this.year = year;
    }

    void displayDetails() {
        System.out.println("Model: " + model);
        System.out.println("Color: " + color);
        System.out.println("Year: " + year);
    }
}

Ús de Constructors

public class Main {
    public static void main(String[] args) {
        // Creació d'un objecte utilitzant el constructor
        Car myCar = new Car("Red", "Toyota", 2021);
        
        // Crida a un mètode
        myCar.displayDetails();
    }
}

Explicació

  • Definició del Constructor: public Car(String color, String model, int year) inicialitza els camps de l'objecte Car.
  • Creació d'Objecte amb Constructor: Car myCar = new Car("Red", "Toyota", 2021); inicialitza l'objecte amb valors especificats.

Encapsulació

L'encapsulació és el concepte d'embolcallar dades (camps) i mètodes en una sola unitat (classe) i restringir l'accés a alguns components de l'objecte.

Exemple amb Encapsulació

public class Car {
    private String color;
    private String model;
    private int year;

    // Constructor
    public Car(String color, String model, int year) {
        this.color = color;
        this.model = model;
        this.year = year;
    }

    // Mètodes Getter i Setter
    public String getColor() {
        return color;
    }

    public void setColor(String color) {
        this.color = color;
    }

    public String getModel() {
        return model;
    }

    public void setModel(String model) {
        this.model = model;
    }

    public int getYear() {
        return year;
    }

    public void setYear(int year) {
        this.year = year;
    }

    void displayDetails() {
        System.out.println("Model: " + model);
        System.out.println("Color: " + color);
        System.out.println("Year: " + year);
    }
}

Explicació

  • Camps Privats: Els camps es declaren com private per restringir l'accés directe.
  • Mètodes Getter i Setter: Mètodes públics per accedir i actualitzar els camps privats.

Herència

L'herència és un mecanisme on una classe adquireix les propietats (camps) i comportaments (mètodes) d'una altra classe.

Exemple d'Herència

// Classe Base
public class Vehicle {
    String brand;

    public void honk() {
        System.out.println("Beep! Beep!");
    }
}

// Classe Derivada
public class Car extends Vehicle {
    String model;

    public void displayDetails() {
        System.out.println("Brand: " + brand);
        System.out.println("Model: " + model);
    }
}

Ús de l'Herència

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car();
        myCar.brand = "Toyota";
        myCar.model = "Corolla";
        
        myCar.honk(); // Mètode de la classe Vehicle
        myCar.displayDetails(); // Mètode de la classe Car
    }
}

Explicació

  • Classe Base (Vehicle): Conté propietats i mètodes comuns.
  • Classe Derivada (Car): Hereta propietats i mètodes de la classe base i pot tenir propietats i mètodes addicionals.

Polimorfisme

El polimorfisme permet que els mètodes facin coses diferents segons l'objecte sobre el qual actuen, tot i que comparteixin el mateix nom.

Exemple de Polimorfisme

public class Animal {
    public void sound() {
        System.out.println("Animal makes a sound");
    }
}

public class Dog extends Animal {
    public void sound() {
        System.out.println("Dog barks");
    }
}

public class Cat extends Animal {
    public void sound() {
        System.out.println("Cat meows");
    }
}

Ús del Polimorfisme

public class Main {
    public static void main(String[] args) {
        Animal myAnimal = new Animal();
        Animal myDog = new Dog();
        Animal myCat = new Cat();
        
        myAnimal.sound();
        myDog.sound();
        myCat.sound();
    }
}

Explicació

  • Sobreescriptura de Mètodes: Les classes derivades (Dog i Cat) proporcionen implementacions específiques del mètode sound.
  • Comportament Polimòrfic: El mètode sound es comporta de manera diferent segons el tipus d'objecte.

Resum

  • Classes i Objectes: Conceptes bàsics de l'OOP en Java.
  • Definició de Classes: Utilitza la paraula clau class.
  • Creació d'Objectes: Utilitza la paraula clau new.
  • Constructors: Mètodes especials per inicialitzar objectes.
  • Encapsulació: Restringeix l'accés als components de l'objecte utilitzant camps privats i mètodes getter/setter públics.
  • Herència: Mecanisme per adquirir propietats i comportaments d'una altra classe.
  • Polimorfisme: Permet que els mètodes realitzin tasques diferents segons l'objecte sobre el qual actuen.

Entendre aquests conceptes t'ajudarà a escriure codi més organitzat, reutilitzable i mantenible en Java.
