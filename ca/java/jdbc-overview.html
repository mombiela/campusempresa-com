Introducció a JDBC

Java Database Connectivity (JDBC) és una API que permet a les aplicacions Java interactuar amb bases de dades relacionals. Proporciona mètodes per consultar i actualitzar dades en una base de dades, convertint-se en una eina crucial per als desenvolupadors Java que treballen amb bases de dades.

Conceptes clau de JDBC

  • API JDBC: Un conjunt d'interfícies i classes en el paquet java.sql.
  • Driver Manager: Gestiona una llista de controladors de bases de dades.
  • Driver: Una interfície que gestiona les comunicacions amb el servidor de bases de dades.
  • Connexió: Una interfície que proporciona mètodes per connectar-se a una base de dades.
  • Statement: Una interfície que representa una instrucció SQL.
  • ResultSet: Una interfície que representa el conjunt de resultats d'una consulta.
  • SQLException: Una excepció que proporciona informació sobre un error d'accés a la base de dades.

Configuració de JDBC

Per utilitzar JDBC, necessites tenir el controlador JDBC per a la teva base de dades. Aquest controlador és típicament proporcionat pel venedor de la base de dades.

Passos per configurar JDBC

  1. Incloure el controlador JDBC en el teu projecte: Afegeix la biblioteca del controlador JDBC al classpath del teu projecte.
  2. Carregar el controlador JDBC: Utilitza Class.forName() per carregar la classe del controlador.
  3. Establir una connexió: Utilitza DriverManager.getConnection() per connectar-te a la base de dades.
  4. Crear una instrucció: Utilitza el mètode Connection.createStatement().
  5. Executar una consulta: Utilitza el mètode Statement.executeQuery() per a consultes SELECT o Statement.executeUpdate() per a consultes INSERT, UPDATE, DELETE.
  6. Processar el ResultSet: Itera a través del ResultSet per recuperar dades.
  7. Tancar la connexió: Sempre tanca els objectes ResultSet, Statement i Connection per alliberar recursos.

Exemple bàsic de JDBC

Aquí tens un exemple senzill que demostra com connectar-se a una base de dades, executar una consulta i processar els resultats.

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.Statement;
import java.sql.SQLException;

public class JDBCBasicExample {
    public static void main(String[] args) {
        // URL de la base de dades
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        // Credencials de la base de dades
        String user = "root";
        String password = "password";

        Connection connection = null;
        Statement statement = null;
        ResultSet resultSet = null;

        try {
            // Carregar el controlador JDBC
            Class.forName("com.mysql.cj.jdbc.Driver");

            // Establir una connexió
            connection = DriverManager.getConnection(url, user, password);

            // Crear una instrucció
            statement = connection.createStatement();

            // Executar una consulta
            String sql = "SELECT id, name, age FROM users";
            resultSet = statement.executeQuery(sql);

            // Processar el conjunt de resultats
            while (resultSet.next()) {
                int id = resultSet.getInt("id");
                String name = resultSet.getString("name");
                int age = resultSet.getInt("age");

                System.out.println("ID: " + id + ", Name: " + name + ", Age: " + age);
            }
        } catch (SQLException | ClassNotFoundException e) {
            e.printStackTrace();
        } finally {
            // Tancar els recursos
            try {
                if (resultSet != null) resultSet.close();
                if (statement != null) statement.close();
                if (connection != null) connection.close();
            } catch (SQLException e) {
                e.printStackTrace();
            }
        }
    }
}

Operacions comunes de JDBC

Inserir dades

String insertSQL = "INSERT INTO users (name, age) VALUES ('John Doe', 30)";
int rowsAffected = statement.executeUpdate(insertSQL);
System.out.println("Files inserides: " + rowsAffected);

Actualitzar dades

String updateSQL = "UPDATE users SET age = 31 WHERE name = 'John Doe'";
int rowsAffected = statement.executeUpdate(updateSQL);
System.out.println("Files actualitzades: " + rowsAffected);

Esborrar dades

String deleteSQL = "DELETE FROM users WHERE name = 'John Doe'";
int rowsAffected = statement.executeUpdate(deleteSQL);
System.out.println("Files esborrades: " + rowsAffected);

Gestió d'excepcions SQL

Sempre gestiona SQLException per manejar errors de bases de dades de manera efectiva. Utilitza els mètodes proporcionats per SQLException per obtenir informació detallada sobre l'error.

catch (SQLException e) {
    System.err.println("SQLState: " + e.getSQLState());
    System.err.println("Codi d'error: " + e.getErrorCode());
    System.err.println("Missatge: " + e.getMessage());
}

Conclusió

JDBC és una API potent que permet a les aplicacions Java interactuar amb bases de dades. Entendre els conceptes bàsics de JDBC, com establir una connexió, executar consultes i gestionar resultats, és essencial per a qualsevol desenvolupador Java que treballi amb bases de dades. Seguint els passos i exemples proporcionats, pots començar a integrar operacions de bases de dades en les teves aplicacions Java de manera eficient.
