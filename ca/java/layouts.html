Introducció a les Disposicions

En Java, les disposicions s'utilitzen per organitzar components GUI dins d'un contenidor. Entendre les disposicions és crucial per crear interfícies d'usuari ben organitzades i visualment atractives.

Conceptes Clau

  • Contenidor: Un component que pot contenir altres components.
  • Component: Un element GUI individual com un botó, etiqueta o camp de text.
  • Gestor de Disposició: Un objecte que controla la mida i la posició dels components dins d'un contenidor.

Tipus de Gestors de Disposició

FlowLayout

FlowLayout organitza els components en un flux d'esquerra a dreta, molt semblant a les línies de text en un paràgraf. Quan una fila està plena, es mou a la següent fila.

Exemple:

import javax.swing.*;
import java.awt.*;

public class FlowLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("FlowLayout Example");
        frame.setLayout(new FlowLayout());

        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));

        frame.setSize(300, 100);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació: Aquest exemple crea un JFrame amb un FlowLayout. S'afegeixen tres botons, i s'organitzaran en una fila fins que s'excedeixi l'amplada del marc.

BorderLayout

BorderLayout divideix el contenidor en cinc regions: Nord, Sud, Est, Oest i Centre. Cada regió pot contenir només un component.

Exemple:

import javax.swing.*;
import java.awt.*;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setLayout(new BorderLayout());

        frame.add(new JButton("North"), BorderLayout.NORTH);
        frame.add(new JButton("South"), BorderLayout.SOUTH);
        frame.add(new JButton("East"), BorderLayout.EAST);
        frame.add(new JButton("West"), BorderLayout.WEST);
        frame.add(new JButton("Center"), BorderLayout.CENTER);

        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació: Aquest exemple demostra l'ús de BorderLayout. Cada botó es col·loca en una de les cinc regions.

GridLayout

GridLayout organitza els components en una quadrícula de cel·les, amb cada cel·la de la mateixa mida.

Exemple:

import javax.swing.*;
import java.awt.*;

public class GridLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("GridLayout Example");
        frame.setLayout(new GridLayout(2, 2)); // 2 files, 2 columnes

        frame.add(new JButton("Button 1"));
        frame.add(new JButton("Button 2"));
        frame.add(new JButton("Button 3"));
        frame.add(new JButton("Button 4"));

        frame.setSize(200, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació: Aquest exemple utilitza GridLayout per organitzar quatre botons en una quadrícula de 2x2.

CardLayout

CardLayout permet canviar entre diferents panells (targetes) en el mateix contenidor, un a la vegada.

Exemple:

import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class CardLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("CardLayout Example");
        CardLayout cardLayout = new CardLayout();
        JPanel panel = new JPanel(cardLayout);

        JPanel card1 = new JPanel();
        card1.add(new JLabel("Card 1"));
        JPanel card2 = new JPanel();
        card2.add(new JLabel("Card 2"));

        panel.add(card1, "Card 1");
        panel.add(card2, "Card 2");

        JButton switchButton = new JButton("Switch Card");
        switchButton.addActionListener(new ActionListener() {
            public void actionPerformed(ActionEvent e) {
                cardLayout.next(panel);
            }
        });

        frame.add(panel, BorderLayout.CENTER);
        frame.add(switchButton, BorderLayout.SOUTH);

        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació: Aquest exemple utilitza CardLayout per canviar entre dos panells. En fer clic al botó, es canvia el panell visible.

Disposicions Avançades

GridBagLayout

GridBagLayout és un gestor de disposició flexible i complex que alinea components verticalment, horitzontalment o al llarg de la seva línia base sense requerir que els components siguin de la mateixa mida.

Exemple:

import javax.swing.*;
import java.awt.*;

public class GridBagLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("GridBagLayout Example");
        frame.setLayout(new GridBagLayout());
        GridBagConstraints gbc = new GridBagConstraints();

        gbc.gridx = 0;
        gbc.gridy = 0;
        frame.add(new JButton("Button 1"), gbc);

        gbc.gridx = 1;
        gbc.gridy = 0;
        frame.add(new JButton("Button 2"), gbc);

        gbc.gridx = 0;
        gbc.gridy = 1;
        gbc.gridwidth = 2;
        frame.add(new JButton("Button 3"), gbc);

        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
  • Explicació: Aquest exemple demostra l'ús de GridBagLayout. L'objecte GridBagConstraints s'utilitza per especificar la posició i la mida de cada component.

Resum

  • FlowLayout: Organitza els components en una fila, movent-se a la següent fila quan sigui necessari.
  • BorderLayout: Divideix el contenidor en cinc regions (Nord, Sud, Est, Oest, Centre).
  • GridLayout: Organitza els components en una quadrícula de cel·les de la mateixa mida.
  • CardLayout: Permet canviar entre diferents panells (targetes).
  • GridBagLayout: Un gestor de disposició flexible que permet disposicions complexes de components.

Entendre aquests gestors de disposició és essencial per crear aplicacions GUI en Java efectives i fàcils d'utilitzar. Practica utilitzant cada gestor de disposició per convertir-te en un expert en el disseny d'interfícies en Java.
