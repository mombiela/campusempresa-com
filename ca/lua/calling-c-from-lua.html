Cridar C des de Lua
Utilitzar Lua amb Altres Llenguatges ►

En aquesta secció, explorarem com estendre Lua cridant funcions C des dels scripts Lua. Aquesta és una característica potent que et permet aprofitar les biblioteques C existents o optimitzar parts crítiques de rendiment de la teva aplicació Lua.

Per què cridar C des de Lua?

  • Rendiment: C és un llenguatge compilat i pot executar certes tasques més ràpidament que Lua.
  • Biblioteques Existents: Hi ha moltes biblioteques disponibles en C que podries voler utilitzar en la teva aplicació Lua.
  • Accés a Nivell de Sistema: C proporciona un accés més directe als recursos del sistema i al maquinari.

Prerequisits

Abans de submergir-te en cridar C des de Lua, assegura't de tenir:

  • Comprensió bàsica de l'scripting Lua.
  • Coneixements bàsics de programació en C.
  • Un entorn de desenvolupament funcional tant per a Lua com per a C.

Configurant l'Entorn

Per cridar C des de Lua, necessites:

  1. Instal·lar Lua: Assegura't que Lua està instal·lat al teu sistema.
  2. Instal·lar un Compilador C: GCC o Clang són comunament utilitzats.
  3. API C de Lua: Familiaritza't amb l'API C de Lua, que permet que el codi C interactuï amb Lua.

Escriure una Funció C Simple

Comencem escrivint una funció C simple que cridarem des de Lua.

// mylib.c
#include <lua.h>
#include <lauxlib.h>
#include <lualib.h>

// Una funció C simple que suma dos números
static int add(lua_State *L) {
    double a = luaL_checknumber(L, 1);
    double b = luaL_checknumber(L, 2);
    lua_pushnumber(L, a + b);
    return 1; // Nombre de valors de retorn
}

// Registrant la funció
static const struct luaL_Reg mylib [] = {
    {"add", add},
    {NULL, NULL} // Sentinella
};

int luaopen_mylib(lua_State *L) {
    luaL_newlib(L, mylib);
    return 1;
}

Explicació

  • luaL_checknumber: Comprova i recupera els arguments de la funció de la pila de Lua.
  • lua_pushnumber: Empeny el resultat de nou a la pila de Lua.
  • luaL_Reg: Estructura utilitzada per registrar les funcions C.
  • luaopen_mylib: Funció cridada per Lua per carregar la biblioteca.

Compilant el Codi C

Compila el codi C en una biblioteca compartida que Lua pugui carregar.

gcc -shared -o mylib.so -fPIC mylib.c -I/usr/include/lua5.3
  • -shared: Crea una biblioteca compartida.
  • -fPIC: Genera codi independent de la posició.
  • -I/usr/include/lua5.3: Inclou les capçaleres de Lua.

Cridant la Funció C des de Lua

Ara, escrivim un script Lua per cridar la nostra funció C.

-- test.lua
local mylib = require("mylib")

local result = mylib.add(10, 20)
print("El resultat és: " .. result)

Explicació

  • require("mylib"): Carrega la biblioteca compartida.
  • mylib.add(10, 20): Crida la funció C.

Executant l'Script Lua

Executa l'script Lua per veure el resultat.

lua test.lua

Hauries de veure la sortida:

El resultat és: 30

Gestió d'Errors

Quan crides funcions C des de Lua, és important gestionar els errors de manera adequada.

  • luaL_error: Genera un error de Lua des de C.
  • luaL_checktype: Assegura que els arguments són del tipus esperat.

Exemple:

static int safe_add(lua_State *L) {
    if (!lua_isnumber(L, 1) || !lua_isnumber(L, 2)) {
        return luaL_error(L, "S'esperaven dos números");
    }
    double a = lua_tonumber(L, 1);
    double b = lua_tonumber(L, 2);
    lua_pushnumber(L, a + b);
    return 1;
}

Conclusió

Cridar C des de Lua et permet estendre les capacitats de Lua aprofitant la potència i el rendiment de C. Seguint els passos descrits en aquesta secció, pots integrar funcions C en els teus scripts Lua, proporcionant una manera fluida d'enriquir les teves aplicacions.

  • Impuls de Rendiment: Optimitza parts crítiques de la teva aplicació.
  • Integració de Biblioteques: Utilitza biblioteques C existents.
  • Accés al Sistema: Aconsegueix més control sobre els recursos del sistema.

Experimenta amb funcions C més complexes i explora la documentació de l'API C de Lua per aprofitar completament el poder de cridar C des de Lua.
