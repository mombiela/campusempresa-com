Introducció als Diccionaris

En Lua, els diccionaris es coneixen com a taules. Les taules són una estructura de dades versàtil que es pot utilitzar per representar arrays, llistes, conjunts i diccionaris (també coneguts com a arrays associatius o mapes). Et permeten emmagatzemar parelles clau-valor, on cada clau és única i està associada amb un valor.

Creació d'un Diccionari

Per crear un diccionari en Lua, utilitzes el constructor de taules {} i defineixes les parelles clau-valor dins d'aquest.

Exemple:

-- Creant un diccionari amb algunes parelles clau-valor
local person = {
    name = "John",
    age = 30,
    occupation = "Engineer"
}

En aquest exemple:

  • name, age i occupation són claus.
  • "John", 30 i "Engineer" són els valors corresponents.

Accés als Valors

Pots accedir als valors d'un diccionari utilitzant les claus.

Exemple:

print(person.name)       -- Sortida: John
print(person["age"])     -- Sortida: 30
print(person.occupation) -- Sortida: Engineer
  • person.name i person["name"] accedeixen al valor associat amb la clau name.
  • De la mateixa manera, person.age i person["age"] accedeixen al valor associat amb la clau age.

Modificació dels Valors

Pots modificar els valors d'un diccionari assignant nous valors a les claus.

Exemple:

person.age = 31
person["occupation"] = "Senior Engineer"

print(person.age)          -- Sortida: 31
print(person.occupation)   -- Sortida: Senior Engineer

Afegir i Eliminar Parelles Clau-Valor

Pots afegir noves parelles clau-valor a un diccionari o eliminar les existents.

Afegir:

person.gender = "Male"
person["country"] = "USA"

print(person.gender)   -- Sortida: Male
print(person.country)  -- Sortida: USA

Eliminar:

Per eliminar una parella clau-valor, assignes la clau a nil.

person.gender = nil
person["country"] = nil

print(person.gender)   -- Sortida: nil
print(person.country)  -- Sortida: nil

Iteració sobre un Diccionari

Pots iterar sobre totes les parelles clau-valor d'un diccionari utilitzant la funció pairs.

Exemple:

for key, value in pairs(person) do
    print(key, value)
end

Això mostrarà:

name    John
age     31
occupation  Senior Engineer

Diccionaris Imbricats

Les taules de Lua poden contenir altres taules, permetent-te crear diccionaris imbricats.

Exemple:

local company = {
    name = "TechCorp",
    employees = {
        {name = "Alice", position = "Developer"},
        {name = "Bob", position = "Designer"}
    }
}

print(company.employees[1].name)  -- Sortida: Alice
print(company.employees[2].position)  -- Sortida: Designer

Ús Avançat

Metataules i Metamètodes

Les taules de Lua poden tenir metataules, que et permeten canviar el comportament de les taules. Els metamètodes són funcions especials que es poden definir en una metataula per personalitzar les operacions sobre les taules.

Exemple:

local mt = {
    __index = function(table, key)
        return "Key not found"
    end
}

local t = setmetatable({}, mt)
print(t.someKey)  -- Sortida: Key not found

En aquest exemple, si no es troba una clau a la taula t, es crida el metamètode __index, retornant "Key not found".

Resum

  • Les taules de Lua es poden utilitzar com a diccionaris per emmagatzemar parelles clau-valor.
  • Pots crear, accedir, modificar, afegir i eliminar parelles clau-valor en un diccionari.
  • La iteració sobre diccionaris es fa utilitzant la funció pairs.
  • Les taules es poden imbricar per crear estructures de dades complexes.
  • Les metataules i els metamètodes proporcionen una personalització avançada del comportament de les taules.

Entendre els diccionaris en Lua és crucial per gestionar i manipular dades de manera eficient. Amb aquests conceptes bàsics i avançats, pots aprofitar les taules de Lua per construir programes robustos i flexibles.
