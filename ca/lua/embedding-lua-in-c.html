Inserir Lua en C
Cridar C des de Lua ►

Inserir Lua en C et permet aprofitar la simplicitat i flexibilitat de Lua dins d'una aplicació en C. Aquesta secció et guiarà a través del procés d'inserir Lua en C, des de la integració bàsica fins a interaccions més avançades.

Introducció a Inserir Lua en C

  • Per què inserir Lua en C?

    • Lua és lleuger i fàcil d'integrar.
    • Permet la creació de scripts dinàmics dins d'una aplicació en C.
    • Lua es pot utilitzar per ampliar la funcionalitat d'un programa en C sense recompilar.

  • Passos bàsics per inserir Lua:

    1. Inicialitzar l'entorn Lua
    2. Carregar i executar scripts Lua
    3. Accedir a variables i funcions Lua des de C
    4. Cridar funcions C des de Lua

Configuració de l'Entorn

Per inserir Lua en un programa en C, necessites incloure les capçaleres de Lua i enllaçar contra la biblioteca Lua.

Incloure les Capçaleres de Lua

#include <lua.h>
#include <lualib.h>
#include <lauxlib.h>

Enllaçar contra la Biblioteca Lua

Quan compiles el teu programa en C, enllaça contra la biblioteca Lua. Per exemple, utilitzant GCC:

gcc -o myprogram myprogram.c -llua

Integració Bàsica

Inicialitzar l'Entorn Lua

Abans de poder executar codi Lua, necessites crear un estat Lua.

lua_State *L = luaL_newstate();
luaL_openlibs(L); // Carregar biblioteques Lua

Carregar i Executar Scripts Lua

Pots carregar i executar un script Lua des d'un fitxer o una cadena.

Des d'un Fitxer

if (luaL_dofile(L, "script.lua") != LUA_OK) {
    fprintf(stderr, "Error: %s\n", lua_tostring(L, -1));
}

Des d'una Cadena

const char *lua_code = "print('Hello from Lua')";
if (luaL_dostring(L, lua_code) != LUA_OK) {
    fprintf(stderr, "Error: %s\n", lua_tostring(L, -1));
}

Accedir a Variables i Funcions Lua des de C

Accedir a Variables Lua

Pots empènyer i obtenir variables des de la pila Lua.

Exemple: Obtenir una Variable Lua

Script Lua (script.lua):

number = 42

Codi en C:

lua_getglobal(L, "number");
if (lua_isnumber(L, -1)) {
    int number = lua_tointeger(L, -1);
    printf("Number from Lua: %d\n", number);
}
lua_pop(L, 1); // Eliminar la variable de la pila

Cridar Funcions Lua des de C

Pots cridar funcions Lua empènyent-les a la pila i després utilitzant lua_pcall.

Exemple: Cridar una Funció Lua

Script Lua (script.lua):

function add(a, b)
    return a + b
end

Codi en C:

lua_getglobal(L, "add");
if (lua_isfunction(L, -1)) {
    lua_pushnumber(L, 10); // Primer argument
    lua_pushnumber(L, 20); // Segon argument

    if (lua_pcall(L, 2, 1, 0) != LUA_OK) {
        fprintf(stderr, "Error: %s\n", lua_tostring(L, -1));
    } else {
        int result = lua_tointeger(L, -1);
        printf("Result from Lua: %d\n", result);
    }
}
lua_pop(L, 1); // Eliminar el resultat de la funció de la pila

Cridar Funcions C des de Lua

Pots registrar funcions C per ser cridades des de Lua.

Registrar una Funció C

Exemple: Registrar i Cridar una Funció C

Codi en C:

int my_c_function(lua_State *L) {
    int arg = lua_tointeger(L, -1);
    lua_pushnumber(L, arg * 2);
    return 1; // Nombre de valors de retorn
}

lua_register(L, "my_c_function", my_c_function);

Script Lua:

result = my_c_function(21)
print(result) -- Output: 42

Conclusió

Inserir Lua en C pot millorar significativament la flexibilitat i funcionalitat de les teves aplicacions en C. Seguint els passos descrits en aquesta secció, pots:

  • Inicialitzar i gestionar un entorn Lua dins del teu programa en C.
  • Carregar i executar scripts Lua.
  • Accedir a variables i funcions Lua des de C.
  • Registrar i cridar funcions C des de Lua.

Amb aquestes habilitats, pots crear aplicacions potents i dinàmiques que aprofitin els punts forts tant de Lua com de C.
