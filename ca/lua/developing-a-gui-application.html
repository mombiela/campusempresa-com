Introducció al Desenvolupament de GUI en Lua

  • Què és una GUI?
    • GUI significa Interfície Gràfica d'Usuari. Permet als usuaris interactuar amb dispositius electrònics utilitzant icones gràfiques i indicadors visuals.
  • Per què utilitzar Lua per al Desenvolupament de GUI?
    • Lua és lleuger, fàcil d'integrar, i té diverses biblioteques que suporten el desenvolupament de GUI.

Configurant l'Entorn

  • Instal·lant Lua
    • Descarrega Lua des del lloc web oficial: lua.org
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.
  • Triant una Biblioteca GUI
    • Biblioteques GUI populars per Lua inclouen:
      • LÖVE: Principalment per al desenvolupament de jocs però es pot utilitzar per a aplicacions GUI generals.
      • IUP: Un conjunt d'eines multi-plataforma per construir interfícies gràfiques d'usuari.
      • LÖVE2D: Dissenyat específicament per al desenvolupament de jocs 2D però es pot adaptar per a aplicacions GUI.

GUI Bàsica amb LÖVE

  • Instal·lant LÖVE
    • Descarrega LÖVE des del lloc web oficial: love2d.org
    • Segueix les instruccions d'instal·lació per al teu sistema operatiu.
  • Creant una Finestra Simple 
    function love.load()
    love.window.setTitle("La Meva Primera Aplicació GUI")
    love.window.setMode(800, 600)
end

function love.draw()
    love.graphics.print("Hola, Món!", 400, 300)
end
    • Explicació:
      • love.load(): Inicialitza la finestra amb un títol i mida.
      • love.draw(): Renderitza el text "Hola, Món!" a les coordenades (400, 300).

Gestionant l'Entrada de l'Usuari

  • Entrada del Ratolí

    function love.mousepressed(x, y, button, istouch, presses)
    if button == 1 then -- 1 és el botó esquerre del ratolí
        print("Botó esquerre del ratolí premut a: ", x, y)
    end
end
    • Explicació:
      • love.mousepressed(): Detecta les pressions dels botons del ratolí i imprimeix les coordenades.

  • Entrada del Teclat

    function love.keypressed(key, scancode, isrepeat)
    if key == "space" then
        print("Tecla espai premuda")
    end
end
    • Explicació:
      • love.keypressed(): Detecta les pressions de les tecles i imprimeix un missatge quan es prem la tecla espai.

Creant Elements Interactius

  • Botons 
    local button = {x = 350, y = 250, width = 100, height = 50, text = "Clica'm"}

function love.draw()
    love.graphics.rectangle("line", button.x, button.y, button.width, button.height)
    love.graphics.printf(button.text, button.x, button.y + 15, button.width, "center")
end

function love.mousepressed(x, y, button, istouch, presses)
    if button == 1 i x > button.x i x < button.x + button.width i y > button.y i y < button.y + button.height then
        print("Botó clicat!")
    end
end
    • Explicació:
      • Dibuixa un rectangle i centra el text dins d'ell.
      • Detecta si el botó és clicat basant-se en les coordenades del ratolí.

Funcionalitats Avançades de GUI

  • Creant un Menú 
    local menu = {
    {text = "Iniciar Joc", action = function() print("Iniciant joc...") end},
    {text = "Opcions", action = function() print("Obrint opcions...") end},
    {text = "Sortir", action = function() love.event.quit() end}
}

function love.draw()
    for i, item in ipairs(menu) do
        love.graphics.printf(item.text, 0, 100 + (i - 1) * 30, love.graphics.getWidth(), "center")
    end
end

function love.keypressed(key)
    if key == "return" then
        menu[1].action()
    end
end
    • Explicació:
      • Mostra una llista d'elements del menú.
      • Executa accions basades en l'entrada de l'usuari.

Conclusió

Desenvolupar una aplicació GUI en Lua pot ser senzill amb les eines i biblioteques adequades. Començant amb la creació bàsica de finestres i la gestió de l'entrada de l'usuari, pots construir elements interactius més complexos i funcionalitats. Biblioteques com LÖVE proporcionen un marc robust per crear aplicacions GUI atractives i funcionals. Amb pràctica i exploració, pots desenvolupar aplicacions sofisticades adaptades a les teves necessitats.
