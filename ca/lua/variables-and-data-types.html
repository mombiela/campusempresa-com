Variables i tipus de dades en Lua

Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en Lua. Entendre aquests conceptes bàsics és crucial per escriure programes Lua efectius i eficients.

Variables

Les variables s'utilitzen per emmagatzemar dades que es poden referenciar i manipular en un programa. En Lua, les variables són de tipus dinàmic, és a dir, no cal declarar explícitament el seu tipus.

Declaració de variables

Per declarar una variable en Lua, simplement assignes un valor a aquesta utilitzant l'operador =.

-- Declaració d'una variable
local myVariable = 10
  • Paraula clau local: Limita l'abast de la variable al bloc on es declara.
  • myVariable: El nom de la variable.
  • 10: El valor assignat a la variable.

Variables globals vs locals

  • Variables globals: Accessibles des de qualsevol lloc del programa.
  • Variables locals: Accessibles només dins del bloc on es declaren.
-- Variable global
globalVar = "I am global"

-- Variable local
local localVar = "I am local"

Tipus de dades

Lua admet diversos tipus de dades bàsics, cadascun amb diferents propòsits. Els tipus de dades principals en Lua són:

  • nil
  • boolean
  • number
  • string
  • table
  • function
  • userdata
  • thread

nil

El tipus nil representa l'absència d'un valor.

local noValue = nil

boolean

El tipus boolean té dos valors: true i false.

local isLuaFun = true
local isTired = false

number

El tipus number representa nombres reals (de coma flotant).

local age = 25
local pi = 3.14

string

El tipus string representa seqüències de caràcters.

local greeting = "Hello, World!"

table

El tipus table és una estructura de dades potent i flexible que es pot utilitzar com a arrays, diccionaris i més.

-- Taula tipus array
local fruits = {"apple", "banana", "cherry"}

-- Taula tipus diccionari
local person = {
    name = "John",
    age = 30
}

function

El tipus function representa blocs de codi executables.

local function greet(name)
    return "Hello, " .. name
end

userdata

El tipus userdata permet emmagatzemar dades arbitràries de C en variables Lua.

thread

El tipus thread representa fils d'execució independents per a coroutines.

Exemples i exercicis

Exemple 1: Treballant amb variables

local name = "Alice"
local age = 28
local isStudent = true

print("Name: " .. name)
print("Age: " .. age)
print("Is Student: " .. tostring(isStudent))

Exercici 1: Crear i imprimir variables

  1. Crea una variable per emmagatzemar el teu color preferit.
  2. Crea una variable per emmagatzemar la teva alçada en centímetres.
  3. Imprimeix ambdues variables.

Exemple 2: Utilitzant taules

local car = {
    make = "Toyota",
    model = "Corolla",
    year = 2020
}

print("Car Make: " .. car.make)
print("Car Model: " .. car.model)
print("Car Year: " .. car.year)

Exercici 2: Crear i manipular una taula

  1. Crea una taula per emmagatzemar informació sobre un llibre (títol, autor, any).
  2. Imprimeix cada peça d'informació de la taula.

Conclusió

Entendre les variables i els tipus de dades és essencial per a qualsevol programador de Lua. Les variables et permeten emmagatzemar i manipular dades, mentre que els tipus de dades defineixen el tipus de dades amb què estàs treballant. En dominar aquests conceptes bàsics, estaràs ben preparat per afrontar tasques de programació més complexes en Lua.
