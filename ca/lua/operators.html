Operadors en Lua
Estructures de Control ►

Introducció

Els operadors són fonamentals en qualsevol llenguatge de programació, i Lua no és una excepció. Permeten realitzar diverses operacions sobre les dades, com ara càlculs aritmètics, comparacions i operacions lògiques. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en Lua, des de l'aritmètica bàsica fins als operadors lògics i relacionals més avançats.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques.

  • + : Suma
  • - : Resta
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul (resta de la divisió)
  • ^ : Exponenciació
  • - : Negació unària

Exemples

local a = 10
local b = 3

print(a + b)  -- Sortida: 13
print(a - b)  -- Sortida: 7
print(a * b)  -- Sortida: 30
print(a / b)  -- Sortida: 3.3333333333333
print(a % b)  -- Sortida: 1
print(a ^ b)  -- Sortida: 1000
print(-a)     -- Sortida: -10

Operadors Relacionals

Els operadors relacionals comparen dos valors i retornen un resultat booleà (true o false).

  • == : Igual a
  • ~= : Diferent de
  • > : Major que
  • < : Menor que
  • >= : Major o igual que
  • <= : Menor o igual que

Exemples

local x = 5
local y = 10

print(x == y)  -- Sortida: false
print(x ~= y)  -- Sortida: true
print(x > y)   -- Sortida: false
print(x < y)   -- Sortida: true
print(x >= y)  -- Sortida: false
print(x <= y)  -- Sortida: true

Operadors Lògics

Els operadors lògics s'utilitzen per combinar o invertir valors booleans.

  • and : I lògic
  • or : O lògic
  • not : No lògic

Exemples

local a = true
local b = false

print(a and b)  -- Sortida: false
print(a or b)   -- Sortida: true
print(not a)    -- Sortida: false

Operador de Concatenació

L'operador de concatenació (..) s'utilitza per unir dues cadenes de text.

Exemples

local str1 = "Hola"
local str2 = "Món"

print(str1 .. " " .. str2)  -- Sortida: Hola Món

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. Lua suporta l'operador d'assignació bàsic =.

Exemples

local a = 10
local b = 20

a = a + b  -- a ara és 30
print(a)   -- Sortida: 30

Operadors d'Assignació Composada

Lua no té operadors d'assignació composada com += o -=, però es pot aconseguir el mateix efecte utilitzant l'operador d'assignació bàsic.

Exemples

local a = 10

a = a + 5  -- Equivalent a a += 5 en altres llenguatges
print(a)   -- Sortida: 15

Precedència d'Operadors

La precedència d'operadors determina l'ordre en què es realitzen les operacions. Lua segueix un ordre específic de precedència, que es resumeix en la taula següent:

| Operador | Descripció | |----------------|----------------------------| | ^ | Exponenciació | | not | No lògic | | *, /, % | Multiplicació, Divisió, Mòdul | | +, - | Suma, Resta | | .. | Concatenació | | <, >, <=, >=, ==, ~= | Operadors relacionals | | and | I lògic | | or | O lògic |

Exemples

local result = 5 + 2 * 3  -- La multiplicació es realitza primer
print(result)  -- Sortida: 11

local result2 = (5 + 2) * 3  -- Els parèntesis canvien l'ordre de les operacions
print(result2)  -- Sortida: 21

Conclusió

Els operadors són eines essencials en Lua que permeten realitzar una àmplia gamma d'operacions sobre les dades. Entendre com utilitzar els operadors aritmètics, relacionals, lògics i de concatenació, així com conèixer l'ordre de precedència, et permetrà escriure codi Lua més eficient i efectiu. Practica utilitzant aquests operadors en diferents escenaris per sentir-te més còmode amb la seva funcionalitat.
