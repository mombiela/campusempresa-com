Llegir i Escriure Fitxers en Lua
Manipulació de Fitxers ►

Introducció

Llegir i escriure fitxers és una habilitat fonamental en la programació. En Lua, les operacions amb fitxers són senzilles i potents, permetent-te gestionar fitxers amb facilitat. Aquesta secció et guiarà a través dels conceptes bàsics de la gestió de fitxers en Lua, des d'obrir i tancar fitxers fins a llegir i escriure dades.

Obrir i Tancar Fitxers

Per treballar amb fitxers en Lua, primer has d'obrir-los. Lua proporciona la funció io.open per a aquest propòsit.

Obrir un Fitxer

La funció io.open requereix dos arguments: el camí del fitxer i el mode en què vols obrir el fitxer.

  • Modes:
    • "r": Mode de lectura (per defecte). Obre el fitxer per a lectura.
    • "w": Mode d'escriptura. Obre el fitxer per a escriptura (i crea el fitxer si no existeix).
    • "a": Mode d'addició. Obre el fitxer per a afegir dades.
    • "r+": Mode de lectura/actualització. Obre el fitxer tant per a lectura com per a escriptura.
    • "w+": Mode d'escriptura/actualització. Obre el fitxer tant per a lectura com per a escriptura (i crea el fitxer si no existeix).
    • "a+": Mode d'addició/actualització. Obre el fitxer tant per a lectura com per a afegir dades.
local file = io.open("example.txt", "r") -- Obre 'example.txt' per a lectura
if not file then
    print("No s'ha pogut obrir el fitxer")
    return
end

Tancar un Fitxer

Sempre tanca el fitxer després d'haver acabat amb ell per alliberar recursos del sistema.

file:close()

Llegir de Fitxers

Lua proporciona diverses maneres de llegir fitxers. Pots llegir tot el fitxer, un nombre específic de bytes, o línia per línia.

Llegir Tot el Fitxer

Per llegir tot el contingut d'un fitxer, utilitza el mètode :read("*all").

local file = io.open("example.txt", "r")
local content = file:read("*all")
print(content)
file:close()

Llegir Línies

Per llegir un fitxer línia per línia, utilitza el mètode :read("*line").

local file = io.open("example.txt", "r")
for line in file:lines() do
    print(line)
end
file:close()

Llegir un Nombre Específic de Bytes

Per llegir un nombre específic de bytes, passa el nombre de bytes com a argument al mètode :read.

local file = io.open("example.txt", "r")
local content = file:read(10) -- Llegeix els primers 10 bytes
print(content)
file:close()

Escriure a Fitxers

Escriure a fitxers en Lua és tan senzill com llegir-los. Pots escriure cadenes, nombres, o qualsevol altre tipus de dada que es pugui convertir a una cadena.

Escriure Cadenes

Per escriure una cadena a un fitxer, utilitza el mètode :write.

local file = io.open("example.txt", "w")
file:write("Hola, Món!")
file:close()

Escriure Múltiples Línies

Per escriure múltiples línies, pots utilitzar múltiples crides a :write o una sola crida amb caràcters de nova línia.

local file = io.open("example.txt", "w")
file:write("Primera línia\n")
file:write("Segona línia\n")
file:close()

Afegir a Fitxers

Afegir dades a un fitxer significa afegir noves dades al final del fitxer sense sobreescriure el contingut existent.

local file = io.open("example.txt", "a")
file:write("Aquesta és una línia afegida.\n")
file:close()

Gestió d'Errors

Quan treballes amb fitxers, és essencial gestionar els possibles errors, com intentar obrir un fitxer inexistent o no tenir permisos.

Comprovar Errors

Pots comprovar si un fitxer s'ha obert correctament verificant si el manejador del fitxer és nil.

local file, err = io.open("nonexistent.txt", "r")
if not file then
    print("Error en obrir el fitxer: " .. err)
else
    -- Procedir amb les operacions del fitxer
    file:close()
end

Resum

En aquesta secció, has après a:

  • Obrir i tancar fitxers en diversos modes.
  • Llegir fitxers utilitzant diferents mètodes.
  • Escriure i afegir dades a fitxers.
  • Gestionar errors quan treballes amb fitxers.

La gestió de fitxers és una part crucial de moltes aplicacions, i dominar aquests conceptes bàsics t'ajudarà a gestionar dades de manera eficient en els teus programes Lua. Practica aquests conceptes amb diferents operacions de fitxers per sentir-te més còmode amb la gestió de fitxers en Lua.
