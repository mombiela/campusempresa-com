Biblioteca estàndard d'E/S
Tècniques de Depuració ►

La Biblioteca estàndard d'E/S en Lua proporciona funcions per a operacions d'entrada i sortida, permetent-te llegir i escriure fitxers, així com interactuar amb la consola. Aquest tema cobrirà els conceptes bàsics d'ús de la Biblioteca estàndard d'E/S, incloent la lectura i escriptura de fitxers, i la gestió de l'entrada i sortida estàndard.

Conceptes clau

  • Entrada estàndard (stdin): Llegir entrada de l'usuari o d'una altra font d'entrada.
  • Sortida estàndard (stdout): Escriure sortida a la consola o a una altra destinació de sortida.
  • Gestió de fitxers: Obrir, llegir, escriure i tancar fitxers.

Llegir de l'entrada estàndard

Lua proporciona diverses funcions per llegir l'entrada de l'usuari. La funció més utilitzada és io.read.

Exemple: Llegir una línia d'entrada

print("Entra el teu nom:")
local name = io.read()
print("Hola, " .. name .. "!")

En aquest exemple:

  • io.read() llegeix una línia d'entrada de l'usuari.
  • L'entrada es guarda a la variable name.
  • El programa després imprimeix una salutació utilitzant l'entrada.

Escriure a la sortida estàndard

Per escriure sortida a la consola, Lua utilitza les funcions print i io.write.

Exemple: Escriure sortida

print("Hola, món!")
io.write("Aquesta és una línia sense un caràcter de nova línia.")
io.write("Això continua a la mateixa línia.")

En aquest exemple:

  • print mostra text seguit d'un caràcter de nova línia.
  • io.write mostra text sense afegir un caràcter de nova línia.

Gestió de fitxers

La Biblioteca estàndard d'E/S de Lua proporciona funcions per obrir, llegir, escriure i tancar fitxers.

Obrir un fitxer

Per obrir un fitxer, utilitza la funció io.open. Aquesta funció retorna un manejador de fitxer que pots utilitzar per realitzar operacions sobre el fitxer.

local file = io.open("example.txt", "r")

En aquest exemple:

  • "example.txt" és el nom del fitxer.
  • "r" és el mode en què s'obre el fitxer. Els modes comuns inclouen:
    • "r": Mode de lectura (per defecte).
    • "w": Mode d'escriptura (crea un fitxer nou o trunca un fitxer existent).
    • "a": Mode d'addició (escriu dades al final del fitxer).

Llegir d'un fitxer

Pots llegir d'un fitxer utilitzant el mètode read del manejador de fitxer.

local file = io.open("example.txt", "r")
local content = file:read("*all")
print(content)
file:close()

En aquest exemple:

  • file:read("*all") llegeix tot el contingut del fitxer.
  • file:close() tanca el fitxer.

Escriure a un fitxer

Per escriure a un fitxer, utilitza el mètode write del manejador de fitxer.

local file = io.open("example.txt", "w")
file:write("Hola, fitxer!")
file:close()

En aquest exemple:

  • file:write("Hola, fitxer!") escriu la cadena al fitxer.
  • file:close() tanca el fitxer.

Gestió d'errors

És important gestionar els errors quan es treballa amb fitxers. Lua proporciona la funció pcall per gestionar errors de manera elegant.

Exemple: Gestió d'errors de fitxers

local function readFile(filename)
    local file, err = io.open(filename, "r")
    if not file then
        print("Error obrint el fitxer: " .. err)
        return
    end
    local content = file:read("*all")
    file:close()
    return content
end

local content = readFile("nonexistent.txt")
if content then
    print(content)
end

En aquest exemple:

  • io.open retorna nil i un missatge d'error si el fitxer no es pot obrir.
  • El missatge d'error es mostra si el fitxer no es pot obrir.

Resum

La Biblioteca estàndard d'E/S en Lua és una eina poderosa per gestionar operacions d'entrada i sortida. Les funcions clau inclouen:

  • io.read per llegir l'entrada.
  • print i io.write per escriure la sortida.
  • io.open, file:read, file:write i file:close per a la gestió de fitxers.

Entenent aquestes funcions, pots gestionar efectivament l'entrada i sortida en els teus programes Lua.
