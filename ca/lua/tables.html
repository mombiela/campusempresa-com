Taules en Lua
Arrays ►

Les taules són una part fonamental de Lua, servint com a estructura de dades principal. Poden ser utilitzades com a arrays, diccionaris o fins i tot objectes. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat de les taules en Lua.

Introducció a les Taules

Les taules en Lua són versàtils i poden emmagatzemar diferents tipus de valors, incloent números, cadenes, funcions i fins i tot altres taules.

Crear una Taula

Per crear una taula, utilitzes la sintaxi {}.

local myTable = {}

Afegir Elements a una Taula

Pots afegir elements a una taula utilitzant claus numèriques o de cadena.

-- Utilitzant claus numèriques
myTable[1] = "Lua"
myTable[2] = "Programació"

-- Utilitzant claus de cadena
myTable["language"] = "Lua"
myTable["type"] = "Scripting"

Accedir als Elements de la Taula

Accedeix als elements utilitzant les seves claus.

print(myTable[1])          -- Sortida: Lua
print(myTable["language"]) -- Sortida: Lua

Taules com a Arrays

En Lua, les taules poden ser utilitzades com a arrays, on s'utilitzen índexs numèrics per emmagatzemar i accedir als elements.

Exemple: Array Simple

local fruits = {"Poma", "Plàtan", "Cirera"}

for i = 1, #fruits do
    print(fruits[i])
end

Explicació

  • #fruits dóna la longitud de l'array.
  • El bucle itera a través de cada element, imprimint-lo.

Taules com a Diccionaris

Les taules també poden funcionar com a diccionaris (o mapes), on s'utilitzen claus de cadena.

Exemple: Diccionari Simple

local capitals = {
    França = "París",
    Alemanya = "Berlín",
    Itàlia = "Roma"
}

print(capitals["França"]) -- Sortida: París

Explicació

  • Les claus són noms de països, i els valors són les seves capitals.
  • Accedir al valor utilitzant capitals["França"] retorna "París".

Taules Imbricades

Les taules poden contenir altres taules, permetent estructures de dades complexes.

Exemple: Taula Imbricada

local person = {
    name = "Joan",
    age = 30,
    address = {
        street = "123 Lua St",
        city = "Lua City"
    }
}

print(person.name)          -- Sortida: Joan
print(person.address.city)  -- Sortida: Lua City

Explicació

  • La taula person conté una altra taula address.
  • Accedeix als elements imbricats utilitzant la notació de punt.

Funcions de Taula

Lua proporciona diverses funcions integrades per manipular taules.

Funcions de Taula Comunes

  • table.insert(table, [pos,] value): Insereix un valor a la taula en la posició especificada.
  • table.remove(table, [pos]): Elimina l'element en la posició especificada.
  • table.sort(table, [comp]): Ordena els elements de la taula.

Exemple: Utilitzant Funcions de Taula

local numbers = {5, 3, 8, 1}

table.insert(numbers, 2, 7) -- Insereix 7 a la posició 2
table.sort(numbers)         -- Ordena la taula

for i, v in ipairs(numbers) do
    print(v)
end

Explicació

  • table.insert(numbers, 2, 7) insereix 7 a la segona posició.
  • table.sort(numbers) ordena la taula en ordre ascendent.
  • ipairs s'utilitza per iterar sobre la taula ordenada.

Metataules i Metamètodes

Les metataules et permeten canviar el comportament de les taules. Els metamètodes són funcions especials que es poden definir en una metataula per sobreescriure comportaments per defecte.

Exemple: Metataula amb __index

local default = {x = 0, y = 0}
local point = setmetatable({}, {__index = default})

print(point.x) -- Sortida: 0

Explicació

  • La taula default proporciona valors per defecte.
  • setmetatable estableix la metataula per a point amb __index apuntant a default.
  • Accedir a point.x retorna 0 perquè point hereta de default.

Conclusió

Les taules en Lua són poderoses i flexibles, servint com a arrays, diccionaris i fins i tot objectes. Entendre com crear, manipular i estendre taules és crucial per a una programació efectiva en Lua. Practica utilitzant taules en diferents escenaris per convertir-te en un expert en el seu ús.
