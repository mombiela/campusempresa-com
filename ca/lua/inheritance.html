Herència a Lua
Polimorfisme ►

L'herència és un concepte fonamental en la programació orientada a objectes (OOP) que permet que una classe (o objecte) hereti propietats i mètodes d'una altra classe (o objecte). A Lua, l'herència s'aconsegueix utilitzant metataules i el metamètode __index.

Conceptes Clau

  • Classe: Un pla per crear objectes (instàncies).
  • Objecte: Una instància d'una classe.
  • Metataula: Una taula que pot canviar el comportament d'una altra taula.
  • Metamètode __index: S'utilitza per buscar valors en una taula quan no es troben a la taula original.

Creació d'una Classe Bàsica

A Lua, les classes es representen típicament utilitzant taules. Aquí tens com pots crear una classe simple:

-- Definir una classe bàsica
Person = {}
Person.__index = Person

-- Constructor
function Person:new(name, age)
    local instance = setmetatable({}, Person)
    instance.name = name
    instance.age = age
    return instance
end

-- Mètode per mostrar informació
function Person:display()
    print("Nom: " .. self.name .. ", Edat: " .. self.age)
end

-- Crear una instància de Person
local person1 = Person:new("Alice", 30)
person1:display()  -- Sortida: Nom: Alice, Edat: 30

Explicació

  • Person = {}: Defineix una nova taula per representar la classe.
  • Person.__index = Person: Estableix el metamètode __index a la classe mateixa, permetent l'herència.
  • Person:new(name, age): Funció constructora per crear una nova instància.
  • setmetatable({}, Person): Estableix la metataula de la nova instància a Person.
  • self.name i self.age: Variables d'instància.

Implementació de l'Herència

Per implementar l'herència, creem una nova classe que hereti d'una classe existent utilitzant el metamètode __index.

-- Definir una subclasse
Student = setmetatable({}, {__index = Person})
Student.__index = Student

-- Constructor
function Student:new(name, age, studentID)
    local instance = Person.new(self, name, age)
    setmetatable(instance, Student)
    instance.studentID = studentID
    return instance
end

-- Mètode per mostrar informació de l'estudiant
function Student:display()
    Person.display(self)
    print("ID de l'Estudiant: " .. self.studentID)
end

-- Crear una instància de Student
local student1 = Student:new("Bob", 22, "S12345")
student1:display()
-- Sortida:
-- Nom: Bob, Edat: 22
-- ID de l'Estudiant: S12345

Explicació

  • Student = setmetatable({}, {__index = Person}): Crea una nova taula per a la subclasse i estableix el seu __index a Person.
  • Student:new(name, age, studentID): Funció constructora per a la subclasse.
  • Person.new(self, name, age): Crida al constructor de la classe pare.
  • setmetatable(instance, Student): Estableix la metataula de la nova instància a Student.
  • Person.display(self): Crida al mètode de la classe pare.

Resum

L'herència a Lua s'aconsegueix utilitzant metataules i el metamètode __index. Establint el __index d'una subclasse a la seva classe pare, podem heretar propietats i mètodes. Això permet la reutilització de codi i una estructura jeràrquica clara.

Punts Clau

  • Utilitza taules per representar classes.
  • Utilitza el metamètode __index per permetre l'herència.
  • Crida als constructors i mètodes de la classe pare utilitzant el paràmetre self.

Entenent i utilitzant l'herència, pots crear programes Lua més complexos i organitzats, fent el teu codi més modular i fàcil de mantenir.
