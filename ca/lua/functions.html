Introducció a les Funcions

Les funcions són un bloc de construcció fonamental en Lua, permetent encapsular codi en blocs reutilitzables. Ajuden a organitzar el codi, evitar la repetició i fer els programes més modulars.

Definició d'una Funció

En Lua, les funcions es defineixen utilitzant la paraula clau function seguida del nom de la funció i un parell de parèntesis.

function sayHello()
    print("Hello, World!")
end
  • function: Paraula clau per definir una funció.
  • sayHello: Nom de la funció.
  • print("Hello, World!"): Cos de la funció, que imprimeix "Hello, World!" a la consola.

Crida a una Funció

Per executar el codi dins d'una funció, cal cridar-la pel seu nom seguit de parèntesis.

sayHello()  -- Output: Hello, World!

Paràmetres i Arguments de Funció

Les funcions poden prendre paràmetres, que són variables passades a la funció per personalitzar el seu comportament.

Paràmetre Únic

function greet(name)
    print("Hello, " .. name .. "!")
end

greet("Alice")  -- Output: Hello, Alice!

Múltiples Paràmetres

function add(a, b)
    return a + b
end

print(add(3, 4))  -- Output: 7
  • return: Paraula clau utilitzada per retornar un valor de la funció.

Àmbit de Variables

Les variables definides dins d'una funció són locals a aquesta funció i no es poden accedir fora d'ella.

function example()
    local x = 10  -- Variable local
    print(x)
end

example()  -- Output: 10
print(x)  -- Error: x no està definida

Funcions Anònimes

Les funcions en Lua poden ser anònimes, és a dir, no tenen un nom. Sovint s'utilitzen com a arguments per a altres funcions o s'assignen a variables.

local square = function(x)
    return x * x
end

print(square(5))  -- Output: 25

Funcions d'Ordre Superior

Lua suporta funcions d'ordre superior, que són funcions que poden prendre altres funcions com a arguments o retornar-les com a resultats.

Exemple: Utilitzar una Funció com a Argument

function applyFunction(func, value)
    return func(value)
end

local double = function(x)
    return x * 2
end

print(applyFunction(double, 5))  -- Output: 10

Clausures

Una clausura és una funció que captura les variables locals del seu àmbit envoltant.

function makeCounter()
    local count = 0
    return function()
        count = count + 1
        return count
    end
end

local counter = makeCounter()
print(counter())  -- Output: 1
print(counter())  -- Output: 2

Funcions Recursives

Una funció recursiva és aquella que es crida a si mateixa. Això és útil per a tasques que es poden descompondre en subtasques similars.

Exemple: Funció Factorial

function factorial(n)
    if n == 0 then
        return 1
    else
        return n * factorial(n - 1)
    end
end

print(factorial(5))  -- Output: 120

Optimització de Crides en Cua

Lua suporta l'optimització de crides en cua, que permet que certs tipus de funcions recursives s'executin sense augmentar la pila de crides.

Exemple: Funció Recursiva en Cua

function tailFactorial(n, acc)
    acc = acc or 1
    if n == 0 then
        return acc
    else
        return tailFactorial(n - 1, n * acc)
    end
end

print(tailFactorial(5))  -- Output: 120

Resum

  • Les funcions en Lua es defineixen utilitzant la paraula clau function.
  • Les funcions poden prendre paràmetres i retornar valors.
  • Les variables dins de les funcions són locals a aquesta funció.
  • Lua suporta funcions anònimes, funcions d'ordre superior i clausures.
  • Les funcions recursives es poden cridar a si mateixes, i Lua suporta l'optimització de crides en cua per a una recursió eficient.

Entendre les funcions és crucial per escriure codi modular i reutilitzable en Lua. Practica definint, cridant i manipulant funcions per convertir-te en un expert en el seu ús.
