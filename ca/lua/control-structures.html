Estructures de control en Lua
Bucles ►

Les estructures de control són fonamentals en la programació ja que et permeten controlar el flux del teu codi. En Lua, les estructures de control inclouen condicionals, bucles i altres construccions que ajuden a gestionar l'execució del teu codi. Aquesta secció cobrirà els aspectes bàsics i avançats de les estructures de control en Lua.

Condicionals

Els condicionals s'utilitzen per executar codi basat en certes condicions. Lua suporta les instruccions if, else i elseif.

Instrucció If

La instrucció if et permet executar un bloc de codi si una condició especificada és certa.

local age = 20

if age >= 18 then
    print("Ets un adult.")
end

Instrucció If-Else

La instrucció if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

local age = 16

if age >= 18 then
    print("Ets un adult.")
else
    print("Ets un menor.")
end

Instrucció If-Elseif-Else

La instrucció if-elseif-else et permet comprovar múltiples condicions.

local score = 85

if score >= 90 then
    print("Nota: A")
elseif score >= 80 then
    print("Nota: B")
elseif score >= 70 then
    print("Nota: C")
else
    print("Nota: F")
end

Bucles

Els bucles s'utilitzen per executar un bloc de codi múltiples vegades. Lua suporta els bucles while, for i repeat-until.

Bucle While

El bucle while executa un bloc de codi mentre una condició especificada sigui certa.

local count = 1

while count <= 5 do
    print("El compte és: " .. count)
    count = count + 1
end

Bucle For

El bucle for s'utilitza per iterar sobre un rang de valors.

for i = 1, 5 do
    print("Iteració: " .. i)
end

Bucle Repeat-Until

El bucle repeat-until executa un bloc de codi fins que una condició especificada esdevingui certa.

local count = 1

repeat
    print("El compte és: " .. count)
    count = count + 1
until count > 5

Estructures de control avançades

Bucles niats

Els bucles es poden niar dins d'altres per manejar escenaris més complexos.

for i = 1, 3 do
    for j = 1, 2 do
        print("i: " .. i .. ", j: " .. j)
    end
end

Instrucció Break

La instrucció break s'utilitza per sortir d'un bucle prematurament.

for i = 1, 10 do
    if i == 5 then
        break
    end
    print("i: " .. i)
end

Comportament similar a Continue

Lua no té una instrucció continue incorporada, però pots aconseguir un comportament similar utilitzant goto.

for i = 1, 10 do
    if i % 2 == 0 then
        goto continue
    end
    print("i: " .. i)
    ::continue::
end

Resum

Les estructures de control són essencials per gestionar el flux dels teus programes en Lua. Dominant els condicionals, els bucles i les construccions de control avançades, podràs escriure codi més eficient i llegible. Practica utilitzant aquestes estructures en diversos escenaris per convertir-te en un programador proficient en Lua.
