Configuració de l'Entorn Lua
El Teu Primer Programa en Lua ►

Configurar l'entorn Lua és el primer pas per començar a programar en Lua. Aquesta secció et guiarà a través del procés d'instal·lació de Lua al teu sistema, la configuració d'un Entorn de Desenvolupament Integrat (IDE) i l'execució del teu primer script Lua.

Instal·lació de Lua

Windows

  1. Descarregar Binaris de Lua:

    • Visita el lloc web oficial de Lua: lua.org
    • Descarrega l'última versió de Lua per a Windows.

  2. Extreure i Instal·lar:

    • Extreu el fitxer ZIP descarregat.
    • Mou la carpeta extreta a una ubicació desitjada (per exemple, C:\Lua).

  3. Configurar Variables d'Entorn:

    • Obre el Menú d'Inici, cerca "Variables d'Entorn" i selecciona "Edita les variables d'entorn del sistema."
    • Fes clic a "Variables d'Entorn."
    • Sota "Variables del sistema," troba la variable Path i fes clic a "Edita."
    • Afegeix el camí a l'executable de Lua (per exemple, C:\Lua\bin).

  4. Verificar la Instal·lació:

    • Obre el Símbol del sistema i escriu lua -v.
    • Hauries de veure la informació de la versió de Lua.

macOS

  1. Utilitzant Homebrew:

    • Obre Terminal.
    • Instal·la Homebrew si encara no ho has fet: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la Lua: brew install lua

  2. Verificar la Instal·lació:

    • A Terminal, escriu lua -v.
    • Hauries de veure la informació de la versió de Lua.

Linux

  1. Utilitzant el Gestor de Paquets:

    • Obre Terminal.
    • Per a sistemes basats en Debian (per exemple, Ubuntu): sudo apt-get install lua5.3
    • Per a sistemes basats en Red Hat (per exemple, Fedora): sudo dnf install lua

  2. Verificar la Instal·lació:

    • A Terminal, escriu lua -v.
    • Hauries de veure la informació de la versió de Lua.

Configuració d'un IDE

Tot i que pots escriure codi Lua en qualsevol editor de text, utilitzar un IDE pot millorar significativament la teva productivitat. Aquí tens algunes opcions populars:

  • ZeroBrane Studio:

    • Un IDE Lua lleuger amb suport de depuració.
    • Descarrega des de ZeroBrane Studio.
    • Segueix les instruccions d'instal·lació al lloc web.

  • Visual Studio Code:

    • Un editor de codi versàtil amb suport per a Lua mitjançant extensions.
    • Descarrega des de Visual Studio Code.
    • Instal·la l'extensió "Lua" des del Marketplace d'Extensions.

Executar el Teu Primer Script Lua

Escriure un Script Simple

  1. Crear un Nou Fitxer:

    • Obre el teu IDE o editor de text.
    • Crea un nou fitxer i desa'l com a hello.lua.

  2. Escriure Codi Lua:

    • Afegeix el següent codi a hello.lua: 
      print("Hola, Món!")

  3. Executar el Script:

    • Obre un terminal o símbol del sistema.
    • Navega al directori on està desat hello.lua.
    • Executa el script escrivint: lua hello.lua
    • Hauries de veure la sortida: Hola, Món!

Explicació del Codi

  • print("Hola, Món!"): Aquesta línia de codi imprimeix la cadena "Hola, Món!" a la consola. La funció print és una funció incorporada de Lua utilitzada per a la sortida.

Conclusió

Configurar l'entorn Lua implica instal·lar Lua al teu sistema operatiu, triar i configurar un IDE, i executar el teu primer script Lua. Seguint aquests passos, assegures que el teu entorn de desenvolupament està llest per a una programació Lua més avançada. En les següents seccions, aprofundirem en la sintaxi de Lua, estructures de dades i funcions més avançades.
