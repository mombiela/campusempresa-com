Introducció

La Biblioteca de Depuració en Lua és una eina poderosa que permet als desenvolupadors inspeccionar i manipular l'execució de programes Lua. Proporciona funcions per accedir i modificar variables, establir punts de ruptura i traçar l'execució del programa. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat de la Biblioteca de Depuració.

Conceptes Clau

  • Depuració: El procés d'identificar i eliminar errors dels programes informàtics.
  • Punts de Ruptura: Punts en el programa on l'execució es pausa, permetent la inspecció de l'estat del programa.
  • Traça de Pila: Un informe dels marcs de pila actius en un cert punt del temps durant l'execució d'un programa.

Ús Bàsic

Carregar la Biblioteca de Depuració

Per utilitzar la Biblioteca de Depuració, necessites requerir-la en el teu script Lua:

local debug = require("debug")

Inspeccionar Variables

Pots utilitzar la funció debug.getlocal per inspeccionar variables locals dins d'una funció.

Exemple:

function exampleFunction()
    local a = 10
    local b = 20
    debug.debug()  -- Entrar en mode interactiu
end

exampleFunction()

Quan es crida la funció debug.debug(), Lua entra en un mode interactiu on pots inspeccionar i modificar variables.

Establir Punts de Ruptura

Els punts de ruptura es poden establir utilitzant la funció debug.sethook.

Exemple:

local function hook()
    print("Punt de ruptura assolit!")
    debug.debug()  -- Entrar en mode interactiu
end

debug.sethook(hook, "l")  -- Establir un hook de línia

function exampleFunction()
    local a = 10
    local b = 20
    print(a + b)
end

exampleFunction()

En aquest exemple, la funció hook es crida abans de cada línia de codi en exampleFunction, permetent-te inspeccionar l'estat del programa.

Ús Avançat

Traça de Pila

Pots obtenir una traça de pila utilitzant la funció debug.traceback.

Exemple:

function exampleFunction()
    local a = 10
    local b = 20
    print(debug.traceback("Traça de pila:"))
end

exampleFunction()

Això imprimirà la traça de pila en el punt on es crida debug.traceback.

Modificar Variables

Pots modificar variables locals utilitzant la funció debug.setlocal.

Exemple:

function exampleFunction()
    local a = 10
    local b = 20
    print("Abans de la modificació: a = " .. a)
    debug.setlocal(1, 1, 30)  -- Modificar la primera variable local en el primer nivell de pila
    print("Després de la modificació: a = " .. a)
end

exampleFunction()

En aquest exemple, el valor de a es modifica de 10 a 30.

Perfilat

Pots utilitzar la Biblioteca de Depuració per perfilar el teu codi establint hooks per comptar les crides de funció i el temps d'execució.

Exemple:

local call_count = 0

local function hook(event)
    if event == "call" then
        call_count = call_count + 1
    end
end

debug.sethook(hook, "cr")  -- Establir hooks per a esdeveniments de crida i retorn

function exampleFunction()
    local a = 10
    local b = 20
    return a + b
end

exampleFunction()
exampleFunction()

print("Comptador de crides de funció: " .. call_count)

Aquest exemple compta el nombre de crides de funció fetes durant l'execució del script.

Conclusió

La Biblioteca de Depuració en Lua és una eina versàtil que pot ajudar-te a inspeccionar, modificar i perfilar els teus programes Lua. Entenent com utilitzar les seves funcions, pots depurar i optimitzar efectivament el teu codi. Recorda utilitzar aquestes eines amb prudència, ja que l'ús excessiu de les característiques de depuració pot afectar el rendiment del teu programa.
