Introducció a les Clausures

Les clausures són un concepte fonamental en Lua i moltes altres llengües de programació. Permeten que les funcions capturin i mantinguin l'accés a variables del seu àmbit definitori, fins i tot després que aquest àmbit hagi acabat d'executar-se.

Conceptes Clau

  • Àmbit de la Funció: La regió del codi on es defineix una funció i on les seves variables locals són accessibles.
  • Àmbit Lexical: La capacitat d'una funció d'accedir a variables del seu àmbit definitori.
  • Clausura: Una funció que manté l'accés al seu àmbit lexical, fins i tot quan la funció s'executa fora d'aquest àmbit.

Entendre les Clausures amb Exemples

Exemple Bàsic

Comencem amb un exemple senzill per il·lustrar el concepte de clausures en Lua.

function outerFunction()
    local outerVariable = "I am from the outer function"
    
    function innerFunction()
        print(outerVariable)
    end
    
    return innerFunction
end

local myClosure = outerFunction()
myClosure()  -- Output: I am from the outer function
  • Explicació:
    • outerFunction defineix una variable local outerVariable.
    • innerFunction es defineix dins de outerFunction i accedeix a outerVariable.
    • outerFunction retorna innerFunction, que després s'assigna a myClosure.
    • Quan es crida myClosure, encara té accés a outerVariable de outerFunction.

Cas Pràctic

Les clausures sovint s'utilitzen per crear funcions amb estat privat.

function createCounter()
    local count = 0
    
    return function()
        count = count + 1
        return count
    end
end

local counter1 = createCounter()
print(counter1())  -- Output: 1
print(counter1())  -- Output: 2

local counter2 = createCounter()
print(counter2())  -- Output: 1
  • Explicació:
    • createCounter inicialitza una variable local count.
    • Retorna una funció anònima que incrementa i retorna count.
    • Cada crida a createCounter crea una nova clausura amb la seva pròpia variable count.

Ús Avançat de les Clausures

Clausures en Iteradors

Les clausures es poden utilitzar per crear iteradors personalitzats.

function rangeIterator(start, stop)
    local current = start - 1
    
    return function()
        current = current + 1
        if current <= stop then
            return current
        end
    end
end

for number in rangeIterator(1, 5) do
    print(number)  -- Output: 1 2 3 4 5
end
  • Explicació:
    • rangeIterator pren els valors start i stop.
    • Retorna una clausura que incrementa current i el retorna fins que current superi stop.
    • El bucle for utilitza aquest iterador per imprimir números de l'1 al 5.

Clausures per a l'Encapsulació

Les clausures poden encapsular l'estat i proporcionar-hi accés controlat.

function createAccount(initialBalance)
    local balance = initialBalance
    
    return {
        deposit = function(amount)
            balance = balance + amount
            return balance
        end,
        withdraw = function(amount)
            if amount <= balance then
                balance = balance - amount
                return balance
            else
                return "Insufficient funds"
            end
        end,
        getBalance = function()
            return balance
        end
    }
end

local account = createAccount(100)
print(account.deposit(50))    -- Output: 150
print(account.withdraw(30))   -- Output: 120
print(account.getBalance())   -- Output: 120
  • Explicació:
    • createAccount inicialitza una variable local balance.
    • Retorna una taula amb funcions per deposit, withdraw i getBalance.
    • Aquestes funcions formen clausures sobre la variable balance, permetent-hi accés controlat.

Conclusió

Les clausures són una característica poderosa en Lua que permeten a les funcions capturar i mantenir l'accés a variables del seu àmbit definitori. Permeten una sèrie de tècniques de programació, des de l'encapsulació simple fins a patrons d'iteradors complexos. Entendre les clausures millorarà significativament la teva capacitat per escriure codi Lua flexible i mantenible.
