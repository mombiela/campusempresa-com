Què és la Programació Orientada a Objectes (POO)?

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Els objectes es poden considerar com entitats del món real que tenen atributs (dades) i comportaments (mètodes).

Conceptes Clau de la POO

  • Classe: Un plànol per crear objectes. Defineix un tipus d'objecte segons els atributs i mètodes que tindrà l'objecte.
  • Objecte: Una instància d'una classe. Conté valors reals en lloc de variables.
  • Herència: Un mecanisme on una nova classe hereta propietats i comportaments d'una classe existent.
  • Encapsulació: L'agrupació de dades i mètodes que operen sobre les dades dins d'una unitat, per exemple, una classe.
  • Polimorfisme: La capacitat de presentar la mateixa interfície per a diferents formes subjacents (tipus de dades).

POO en Lua

Lua és un llenguatge de programació lleuger, d'alt nivell i multi-paradigma dissenyat principalment per a ús incrustat en aplicacions. Tot i que Lua no té suport integrat per a la POO, proporciona mecanismes per implementar conceptes de POO.

Creació de Classes i Objectes

En Lua, les taules i les metataules s'utilitzen per crear classes i objectes.

Exemple: Definint una Classe Simple

-- Definir una classe utilitzant una taula
Person = {}
Person.__index = Person

-- Constructor per a la classe Person
function Person:new(name, age)
    local self = setmetatable({}, Person)
    self.name = name
    self.age = age
    return self
end

-- Mètode per mostrar els detalls de la persona
function Person:display()
    print("Nom: " .. self.name .. ", Edat: " .. self.age)
end

-- Creant una instància de Person
local person1 = Person:new("Alice", 30)
person1:display()  -- Sortida: Nom: Alice, Edat: 30

Herència en Lua

L'herència es pot implementar establint la metataula d'una classe derivada a la classe base.

Exemple: Implementant Herència

-- Definir una classe base
Animal = {}
Animal.__index = Animal

function Animal:new(name)
    local self = setmetatable({}, Animal)
    self.name = name
    return self
end

function Animal:speak()
    print(self.name .. " fa un so.")
end

-- Definir una classe derivada
Dog = setmetatable({}, {__index = Animal})
Dog.__index = Dog

function Dog:new(name, breed)
    local self = Animal.new(self, name)
    setmetatable(self, Dog)
    self.breed = breed
    return self
end

function Dog:speak()
    print(self.name .. " borda.")
end

-- Creant una instància de Dog
local dog1 = Dog:new("Buddy", "Golden Retriever")
dog1:speak()  -- Sortida: Buddy borda.

Encapsulació en Lua

L'encapsulació es pot aconseguir utilitzant variables locals i proporcionant mètodes getter i setter.

Exemple: Encapsulació

-- Definir una classe amb encapsulació
Account = {}
Account.__index = Account

function Account:new(balance)
    local self = setmetatable({}, Account)
    local balance = balance  -- variable privada
    function self:getBalance()
        return balance
    end
    function self:deposit(amount)
        balance = balance + amount
    end
    return self
end

-- Creant una instància de Account
local account1 = Account:new(1000)
print(account1:getBalance())  -- Sortida: 1000
account1:deposit(500)
print(account1:getBalance())  -- Sortida: 1500

Resum

  • POO és un paradigma poderós que ajuda a organitzar el codi agrupant dades i mètodes.
  • Lua suporta la POO a través de taules i metataules.
  • Conceptes clau de la POO com classes, objectes, herència i encapsulació es poden implementar en Lua.
  • Practica creant les teves pròpies classes i experimentant amb herència i encapsulació per aprofundir en la teva comprensió.

En dominar aquests conceptes, podràs escriure codi més modular, reutilitzable i mantenible en Lua.
