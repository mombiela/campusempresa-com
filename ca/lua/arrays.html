Introducció als Arrays

Els arrays són una estructura de dades fonamental en la programació, utilitzada per emmagatzemar múltiples valors en una sola variable. A Lua, els arrays es implementen utilitzant taules, que són estructures de dades versàtils que poden contenir diversos tipus de valors.

Conceptes Clau

  • Taules: La principal estructura de dades de Lua, utilitzada per crear arrays.
  • Indexació: Els arrays de Lua són indexats a partir de 1, és a dir, el primer element s'accedeix amb l'índex 1.
  • Inicialització: Els arrays es poden inicialitzar amb valors o crear-se buits i omplir-se més tard.

Creació d'Arrays

Els arrays a Lua es creen utilitzant taules. Aquí hi ha algunes maneres de crear i inicialitzar arrays:

Exemple 1: Inicialització d'un Array amb Valors

-- Creant un array amb valors inicials
local fruits = {"apple", "banana", "cherry"}
print(fruits[1])  -- Sortida: apple
print(fruits[2])  -- Sortida: banana
print(fruits[3])  -- Sortida: cherry

Exemple 2: Creació d'un Array Buit i Afegint Valors

-- Creant un array buit
local numbers = {}

-- Afegint valors a l'array
numbers[1] = 10
numbers[2] = 20
numbers[3] = 30

print(numbers[1])  -- Sortida: 10
print(numbers[2])  -- Sortida: 20
print(numbers[3])  -- Sortida: 30

Accés i Modificació d'Elements d'un Array

Pots accedir i modificar elements en un array utilitzant els seus índexs.

Exemple 3: Accés i Modificació d'Elements

local colors = {"red", "green", "blue"}

-- Accés a elements
print(colors[1])  -- Sortida: red

-- Modificació d'elements
colors[2] = "yellow"
print(colors[2])  -- Sortida: yellow

Iteració sobre Arrays

Lua proporciona diverses maneres d'iterar sobre arrays, incloent bucles for i la funció ipairs.

Exemple 4: Utilitzant un Bucle `for`

local animals = {"cat", "dog", "bird"}

for i = 1, #animals do
    print(animals[i])
end
-- Sortida:
-- cat
-- dog
-- bird

Exemple 5: Utilitzant `ipairs`

local cars = {"Toyota", "Ford", "BMW"}

for index, value in ipairs(cars) do
    print(index, value)
end
-- Sortida:
-- 1   Toyota
-- 2   Ford
-- 3   BMW

Tècniques Avançades d'Arrays

A mesura que et familiaritzis amb els arrays, pots explorar tècniques més avançades com ara arrays multidimensionals i funcions de manipulació d'arrays.

Arrays Multidimensionals

Els arrays multidimensionals són arrays d'arrays. Es poden utilitzar per representar matrius o graelles.

Exemple 6: Creació d'un Array 2D

local matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
}

print(matrix[1][1])  -- Sortida: 1
print(matrix[2][3])  -- Sortida: 6

Funcions de Manipulació d'Arrays

Lua proporciona diverses funcions integrades per manipular arrays, com ara table.insert, table.remove, i table.sort.

Exemple 7: Utilitzant `table.insert` i `table.remove`

local list = {"a", "b", "c"}

-- Inserint un element
table.insert(list, 2, "x")
print(table.concat(list, ", "))  -- Sortida: a, x, b, c

-- Eliminant un element
table.remove(list, 3)
print(table.concat(list, ", "))  -- Sortida: a, x, c

Exemple 8: Ordenant un Array

local numbers = {5, 3, 8, 1, 2}

table.sort(numbers)
for i, v in ipairs(numbers) do
    print(v)
end
-- Sortida:
-- 1
-- 2
-- 3
-- 5
-- 8

Conclusió

Els arrays a Lua són potents i flexibles, gràcies a la versatilitat de les taules. Entenent com crear, accedir, modificar i manipular arrays, pots gestionar eficaçment col·leccions de dades en els teus programes Lua. Practica amb els exemples proporcionats i explora tècniques més avançades per convertir-te en un expert en l'ús d'arrays.
