Introducció a les Metataules

Les metataules a Lua proporcionen una manera poderosa de canviar el comportament de les taules. Et permeten definir comportaments personalitzats per a operacions estàndard com la suma, la resta i fins i tot la indexació de taules.

  • Què és una Metataula? Una metataula és una taula que pot canviar el comportament d'una altra taula. Actua com un pla per a com una taula ha de comportar-se sota certes condicions.

  • Establir una Metataula Pots establir una metataula per a una taula utilitzant la funció setmetatable.

Exemple

local myTable = {}
local myMetatable = {}

setmetatable(myTable, myMetatable)

En aquest exemple, myTable ara té myMetatable com la seva metataula.

Metamètodes

Els metamètodes són funcions especials que es poden definir en una metataula per canviar el comportament d'una taula. S'identifiquen per claus específiques en la metataula.

  • Metamètodes Comuns
    • __index: Personalitza la indexació de taules.
    • __newindex: Personalitza l'assignació de taules.
    • __add: Personalitza l'operació de suma.
    • __sub: Personalitza l'operació de resta.
    • __mul: Personalitza l'operació de multiplicació.
    • __div: Personalitza l'operació de divisió.
    • __tostring: Personalitza la representació en cadena d'una taula.

Exemple: Indexació Personalitzada

local myTable = {a = 1, b = 2}
local myMetatable = {
    __index = function(table, key)
        if key == "c" then
            return 3
        else
            return nil
        end
    end
}

setmetatable(myTable, myMetatable)

print(myTable.a)  -- Sortida: 1
print(myTable.c)  -- Sortida: 3

En aquest exemple, accedir a myTable.c retorna 3 a causa del metamètode __index personalitzat.

Metamètodes Avançats

Per a casos d'ús més avançats, pots definir metamètodes per a operacions aritmètiques, concatenació i més.

Exemple: Suma Personalitzada

local vector1 = {x = 1, y = 2}
local vector2 = {x = 3, y = 4}

local vectorMetatable = {
    __add = function(v1, v2)
        return {x = v1.x + v2.x, y = v1.y + v2.y}
    end
}

setmetatable(vector1, vectorMetatable)
setmetatable(vector2, vectorMetatable)

local result = vector1 + vector2
print(result.x, result.y)  -- Sortida: 4 6

En aquest exemple, sumar vector1 i vector2 resulta en una nova taula amb la suma dels seus valors x i y.

Casos Pràctics

Les metataules i els metamètodes es poden utilitzar per a:

  • Implementar programació orientada a objectes.
  • Crear taules de només lectura.
  • Definir comportaments personalitzats per a operacions matemàtiques.
  • Controlar l'accés als elements de la taula.

Exemple: Taula de Només Lectura

local function readonly(table)
    local proxy = {}
    local mt = {
        __index = table,
        __newindex = function(t, key, value)
            error("Intent de modificar una taula de només lectura")
        end
    }
    setmetatable(proxy, mt)
    return proxy
end

local originalTable = {a = 1, b = 2}
local roTable = readonly(originalTable)

print(roTable.a)  -- Sortida: 1
roTable.a = 10    -- Error: Intent de modificar una taula de només lectura

En aquest exemple, roTable és una versió de només lectura de originalTable.

Conclusió

Les metataules i els metamètodes a Lua proporcionen una manera flexible de personalitzar el comportament de les taules. Comprenent i utilitzant aquestes característiques, pots crear programes Lua més potents i eficients. Tant si estàs implementant comportaments personalitzats per a operacions aritmètiques com si estàs creant taules de només lectura, les metataules i els metamètodes són eines essencials en la teva caixa d'eines de programació Lua.
