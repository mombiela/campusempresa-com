Introducció

Lua és un llenguatge de script lleuger i d'alt nivell dissenyat per a ús incrustat en aplicacions. Una de les seves característiques més poderoses és la seva capacitat per integrar-se perfectament amb altres llenguatges de programació. Aquesta secció cobrirà com utilitzar Lua amb altres llenguatges, centrant-se en C/C++, Python i Java.

Lua amb C/C++

Lua va ser dissenyat per ser incrustat en aplicacions C/C++. Això el converteix en una elecció natural per estendre la funcionalitat d'aquests llenguatges.

Conceptes Clau

  • Incrustar Lua en C/C++: Integrar Lua en una aplicació C/C++ per executar scripts Lua.
  • Cridar C/C++ des de Lua: Estendre Lua cridant funcions C/C++ des de scripts Lua.
  • API C de Lua: El conjunt de funcions proporcionades per Lua per interactuar amb C/C++.

Exemple: Incrustar Lua en C

#include <lua.h>
#include <lauxlib.h>
#include <lualib.h>

int main(void) {
    lua_State *L = luaL_newstate();  // Crear un nou estat Lua
    luaL_openlibs(L);                // Obrir biblioteques estàndard

    if (luaL_loadfile(L, "script.lua") || lua_pcall(L, 0, 0, 0)) {
        fprintf(stderr, "Error: %s\n", lua_tostring(L, -1));
        return 1;
    }

    lua_close(L);  // Tancar l'estat Lua
    return 0;
}

En aquest exemple:

  • Creem un nou estat Lua amb luaL_newstate().
  • Carreguem i executem un script Lua (script.lua) utilitzant luaL_loadfile() i lua_pcall().
  • Gestionem errors comprovant els valors de retorn de luaL_loadfile() i lua_pcall().
  • Finalment, tanquem l'estat Lua amb lua_close().

Exemple: Cridar C des de Lua

#include <lua.h>
#include <lauxlib.h>
#include <lualib.h>

static int c_add(lua_State *L) {
    int a = luaL_checkinteger(L, 1);
    int b = luaL_checkinteger(L, 2);
    lua_pushinteger(L, a + b);
    return 1;  // Nombre de valors de retorn
}

int main(void) {
    lua_State *L = luaL_newstate();
    luaL_openlibs(L);

    lua_register(L, "c_add", c_add);  // Registrar la funció C

    luaL_dostring(L, "print(c_add(2, 3))");  // Cridar la funció C des de Lua

    lua_close(L);
    return 0;
}

En aquest exemple:

  • Definim una funció C c_add que suma dos enters.
  • Registrem aquesta funció amb Lua utilitzant lua_register().
  • Cridem la funció C des de Lua utilitzant luaL_dostring().

Lua amb Python

Tot i que Lua i Python són tots dos llenguatges de script d'alt nivell, es poden utilitzar junts de diverses maneres.

Conceptes Clau

  • Interfície Lua amb Python: Utilitzar biblioteques per cridar Lua des de Python i viceversa.
  • Biblioteques: lupa és una biblioteca popular per integrar Lua amb Python.

Exemple: Utilitzar Lupa

from lupa import LuaRuntime

lua = LuaRuntime(unpack_returned_tuples=True)
lua_func = lua.eval('''
    function(a, b)
        return a + b
    end
''')

result = lua_func(2, 3)
print(result)  # Sortida: 5

En aquest exemple:

  • Creem un entorn Lua utilitzant LuaRuntime.
  • Definim una funció Lua que suma dos números utilitzant lua.eval().
  • Cridem la funció Lua des de Python i imprimim el resultat.

Lua amb Java

Integrar Lua amb Java es pot aconseguir utilitzant biblioteques com luaj.

Conceptes Clau

  • Interfície Lua amb Java: Utilitzar biblioteques per cridar Lua des de Java i viceversa.
  • Biblioteques: luaj és una biblioteca popular per integrar Lua amb Java.

Exemple: Utilitzar Luaj

import org.luaj.vm2.Globals;
import org.luaj.vm2.LuaValue;
import org.luaj.vm2.lib.jse.JsePlatform;

public class Main {
    public static void main(String[] args) {
        Globals globals = JsePlatform.standardGlobals();
        LuaValue chunk = globals.load("return 2 + 3");
        LuaValue result = chunk.call();
        System.out.println(result.toint());  // Sortida: 5
    }
}

En aquest exemple:

  • Creem un entorn Lua utilitzant JsePlatform.standardGlobals().
  • Carreguem i executem un script Lua utilitzant globals.load() i chunk.call().
  • Imprimim el resultat de l'script Lua.

Conclusió

Integrar Lua amb altres llenguatges com C/C++, Python i Java pot millorar significativament la funcionalitat i flexibilitat de les teves aplicacions. Incrustant Lua, cridant funcions Lua des d'altres llenguatges i viceversa, pots aprofitar els punts forts tant de Lua com del llenguatge amfitrió. Aquesta secció ha proporcionat una introducció bàsica i exemples per començar a utilitzar Lua al costat d'altres llenguatges de programació.
