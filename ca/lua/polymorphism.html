Introducció al Polimorfisme

El polimorfisme és un concepte fonamental en la programació orientada a objectes que permet que els objectes siguin tractats com a instàncies de la seva classe pare en lloc de la seva classe real. Això permet que una única funció pugui gestionar diferents tipus d'objectes, fent que el codi sigui més flexible i reutilitzable.

En Lua, el polimorfisme s'aconsegueix mitjançant l'ús de taules i metataules. El tipatge dinàmic de Lua i les estructures de taules flexibles fan que sigui un llenguatge únic i potent per implementar el polimorfisme.

Conceptes Clau

  • Polimorfisme: La capacitat de diferents objectes per respondre a la mateixa crida de funció de maneres diferents.
  • Taules: La principal estructura de dades de Lua, que es pot utilitzar per representar objectes.
  • Metataules: Taules especials que poden canviar el comportament d'altres taules, permetent el polimorfisme.

Implementació del Polimorfisme en Lua

Ús de Taules i Metataules

En Lua, les taules s'utilitzen per crear objectes, i les metataules s'utilitzen per definir com es comporten aquests objectes. En establir metataules, podem sobreescriure comportaments per defecte i implementar el polimorfisme.

Exemple: Polimorfisme Bàsic amb Taules i Metataules

Comencem amb un exemple senzill per il·lustrar el polimorfisme en Lua.

Pas 1: Definir una Classe Base

-- Definir una classe base 'Shape'
Shape = {}
Shape.__index = Shape

function Shape:new()
    local instance = setmetatable({}, self)
    return instance
end

function Shape:draw()
    print("Dibuixant una forma")
end

Pas 2: Definir Classes Derivades

-- Definir una classe derivada 'Circle'
Circle = setmetatable({}, {__index = Shape})

function Circle:draw()
    print("Dibuixant un cercle")
end

-- Definir una classe derivada 'Rectangle'
Rectangle = setmetatable({}, {__index = Shape})

function Rectangle:draw()
    print("Dibuixant un rectangle")
end

Pas 3: Demostrar el Polimorfisme

-- Crear instàncies de Circle i Rectangle
local shapes = {Circle:new(), Rectangle:new()}

-- Cridar el mètode draw en cada forma
for _, shape in ipairs(shapes) do
    shape:draw()
end

Explicació

  • Classe Base (Shape): Definim una classe base Shape amb un mètode draw.
  • Classes Derivades (Circle i Rectangle): Creem classes derivades Circle i Rectangle que hereten de Shape i sobreescriuen el mètode draw.
  • Comportament Polimòrfic: Quan cridem el mètode draw en instàncies de Circle i Rectangle, s'executa el mètode adequat segons la classe real de l'objecte.

Polimorfisme Avançat

Ús de Sobrecàrrega de Funcions

Lua no suporta la sobrecàrrega de funcions tradicional, però podem aconseguir un comportament similar utilitzant taules i metataules.

Exemple: Sobrecàrrega de Funcions

-- Definir una funció que es comporta de manera diferent segons el tipus d'entrada
function process(input)
    if type(input) == "number" then
        print("Processant un número: " .. input)
    elseif type(input) == "string" then
        print("Processant una cadena: " .. input)
    else
        print("Tipus no suportat")
    end
end

-- Provar la funció amb diferents tipus d'entrada
process(42)
process("Hola, Lua!")

Explicació

  • Sobrecàrrega de Funcions: La funció process es comporta de manera diferent segons el tipus de la seva entrada, demostrant una forma de polimorfisme.

Conclusió

El polimorfisme en Lua és un concepte potent que permet un codi flexible i reutilitzable. Mitjançant l'ús de taules i metataules, podem crear objectes que responen a la mateixa crida de funció de maneres diferents. Això fa que el nostre codi sigui més modular i fàcil de mantenir.

En aquest tema, hem cobert:

  • El concepte bàsic de polimorfisme.
  • Com implementar el polimorfisme utilitzant taules i metataules.
  • Un exemple de sobrecàrrega de funcions per aconseguir un comportament polimòrfic.

Entendre i aplicar el polimorfisme en Lua millorarà la teva capacitat per escriure codi eficient i escalable.
