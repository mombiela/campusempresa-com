Introducció a Lua

Lua és un llenguatge de programació lleuger, d'alt nivell i multi-paradigma dissenyat principalment per a ús incrustat en aplicacions. És conegut per la seva simplicitat i flexibilitat, cosa que el fa una elecció popular per al desenvolupament de jocs, scripting i prototipatge ràpid.

Conceptes clau

  • Sensibilitat a majúscules i minúscules: Lua distingeix entre majúscules i minúscules, és a dir, Variable i variable es consideren identificadors diferents.
  • Comentaris: Lua admet comentaris d'una sola línia i de múltiples línies.
  • Espais en blanc: Els espais en blanc generalment s'ignoren en Lua, però s'utilitzen per separar tokens.

Comentaris

  • Comentaris d'una sola línia: Utilitza -- per iniciar un comentari d'una sola línia.
  • Comentaris de múltiples línies: Utilitza --[[ per iniciar i ]] per acabar un comentari de múltiples línies.
-- Aquest és un comentari d'una sola línia

--[[
Aquest és un
comentari de múltiples línies
]]

Variables i tipus de dades

  • Declaració de variables: Les variables en Lua són de tipus dinàmic i es poden declarar sense especificar un tipus.
  • Tipus de dades: Lua admet diversos tipus de dades bàsics, incloent nil, boolean, number, string, function, userdata, thread i table.
local myNumber = 10       -- number
local myString = "Hola"   -- string
local myBoolean = true    -- boolean
local myNil = nil         -- nil

Operadors

Lua admet una varietat d'operadors per a operacions aritmètiques, relacionals, lògiques i altres.

Operadors aritmètics

  • + (suma)
  • - (resta)
  • * (multiplicació)
  • / (divisió)
  • % (mòdul)
  • ^ (exponenciació)
local a = 5
local b = 2
print(a + b)  -- Sortida: 7
print(a - b)  -- Sortida: 3
print(a * b)  -- Sortida: 10
print(a / b)  -- Sortida: 2.5
print(a % b)  -- Sortida: 1
print(a ^ b)  -- Sortida: 25

Operadors relacionals

  • == (igual a)
  • ~= (diferent de)
  • < (menor que)
  • > (més gran que)
  • <= (menor o igual a)
  • >= (més gran o igual a)
local x = 10
local y = 20
print(x == y)  -- Sortida: false
print(x ~= y)  -- Sortida: true
print(x < y)   -- Sortida: true
print(x > y)   -- Sortida: false
print(x <= y)  -- Sortida: true
print(x >= y)  -- Sortida: false

Operadors lògics

  • and
  • or
  • not
local a = true
local b = false
print(a and b)  -- Sortida: false
print(a or b)   -- Sortida: true
print(not a)    -- Sortida: false

Estructures de control

Lua proporciona diverses estructures de control per gestionar el flux del programa.

Sentències If

local age = 18
if age >= 18 then
    print("Ets un adult.")
else
    print("Ets un menor.")
end

Bucles While

local count = 1
while count <= 5 do
    print(count)
    count = count + 1
end

Bucles For

for i = 1, 5 do
    print(i)
end

Bucles Repeat-Until

local count = 1
repeat
    print(count)
    count = count + 1
until count > 5

Funcions

Les funcions en Lua són valors de primera classe i es poden emmagatzemar en variables, passar com a arguments i retornar d'altres funcions.

Definició d'una funció

function greet(name)
    return "Hola, " .. name
end

print(greet("Món"))  -- Sortida: Hola, Món

Funcions anònimes

local add = function(a, b)
    return a + b
end

print(add(2, 3))  -- Sortida: 5

Taules

Les taules són l'única estructura de dades en Lua i es poden utilitzar com a arrays, diccionaris o fins i tot objectes.

Creació d'una taula

local person = {
    name = "Joan",
    age = 30
}

print(person.name)  -- Sortida: Joan
print(person.age)   -- Sortida: 30

Iteració sobre una taula

local fruits = {"poma", "plàtan", "cirera"}

for index, value in ipairs(fruits) do
    print(index, value)
end

Conclusió

Entendre la sintaxi i l'estructura bàsiques de Lua és crucial per escriure codi efectiu i eficient. Això inclou saber com declarar variables, utilitzar operadors, controlar el flux del programa, definir funcions i treballar amb taules. Amb aquests fonaments, estàs ben preparat per explorar característiques i aplicacions més avançades de Lua.
