Manipulació de fitxers en Lua
Biblioteca Estàndard d'I/O ►

La manipulació de fitxers és una habilitat essencial per a qualsevol programador. En Lua, les operacions amb fitxers són senzilles i potents, permetent-te llegir, escriure i manipular fitxers amb facilitat. Aquesta secció et guiarà des dels conceptes bàsics fins a tècniques més avançades de manipulació de fitxers en Lua.

Introducció a la Gestió de Fitxers

Abans d'endinsar-nos en la manipulació de fitxers, és important entendre els conceptes bàsics i les funcions que proporciona Lua per gestionar fitxers.

Conceptes Clau

  • Modes de Fitxer: Diferents modes per obrir fitxers (r, w, a, etc.).
  • Gestors de Fitxer: Objectes que representen fitxers oberts.
  • Biblioteca d'E/S Estàndard: La biblioteca integrada de Lua per a operacions amb fitxers.

Modes de Fitxer

| Mode | Descripció | |------|-------------| | r | Mode de lectura. Obre el fitxer per llegir. | | w | Mode d'escriptura. Obre el fitxer per escriure (crea un fitxer nou o trunca un fitxer existent). | | a | Mode d'addició. Obre el fitxer per afegir. | | r+ | Mode de lectura/escriptura. Obre el fitxer tant per llegir com per escriure. | | w+ | Mode de lectura/escriptura. Obre el fitxer tant per llegir com per escriure (crea un fitxer nou o trunca un fitxer existent). | | a+ | Mode de lectura/afegir. Obre el fitxer tant per llegir com per afegir. |

Operacions Bàsiques amb Fitxers

Obrir i Tancar Fitxers

Per realitzar qualsevol operació en un fitxer, primer cal obrir-lo. Lua proporciona la funció io.open per a aquest propòsit.

-- Obrir un fitxer en mode de lectura
local file = io.open("example.txt", "r")

-- Sempre comprova si el fitxer s'ha obert correctament
if not file then
    print("No s'ha pogut obrir el fitxer")
    return
end

-- Realitzar operacions amb el fitxer...

-- Tancar el fitxer
file:close()

Llegir de Fitxers

Hi ha diverses maneres de llegir d'un fitxer en Lua. Pots llegir tot el fitxer, un nombre específic de bytes o línia per línia.

-- Llegir tot el fitxer
local file = io.open("example.txt", "r")
local content = file:read("*all")
print(content)
file:close()

-- Llegir línia per línia
local file = io.open("example.txt", "r")
for line in file:lines() do
    print(line)
end
file:close()

Escriure a Fitxers

Escriure a un fitxer és igual de senzill. Pots escriure cadenes o dades formatades.

-- Escriure a un fitxer
local file = io.open("example.txt", "w")
file:write("Hola, Món!\n")
file:write("Aquesta és una nova línia.")
file:close()

Operacions Avançades amb Fitxers

Afegir a Fitxers

Afegir permet afegir dades al final d'un fitxer existent sense sobreescriure el seu contingut.

-- Afegir a un fitxer
local file = io.open("example.txt", "a")
file:write("\nAfegint aquesta línia.")
file:close()

Llegir i Escriure Fitxers Binari

Per als fitxers binaris, cal gestionar les dades de manera diferent. Lua proporciona funcions per llegir i escriure dades binàries.

-- Escriure dades binàries
local file = io.open("example.bin", "wb")
file:write(string.char(0x42, 0x43, 0x44))
file:close()

-- Llegir dades binàries
local file = io.open("example.bin", "rb")
local data = file:read("*all")
for i = 1, #data do
    print(string.byte(data, i))
end
file:close()

Gestió d'Errors

La gestió adequada d'errors és crucial quan es treballa amb fitxers per evitar bloquejos i pèrdues de dades.

-- Exemple de gestió d'errors
local file, err = io.open("nonexistent.txt", "r")
if not file then
    print("Error en obrir el fitxer: " .. err)
else
    -- Realitzar operacions amb el fitxer...
    file:close()
end

Conclusió

La manipulació de fitxers en Lua és una característica potent que et permet gestionar diverses operacions amb fitxers amb facilitat. Entenent les tècniques bàsiques i avançades, pots llegir, escriure i gestionar fitxers de manera eficient en els teus programes Lua. Recorda sempre gestionar els errors amb cura i tancar els fitxers correctament per assegurar la integritat de les dades i l'estabilitat del programa.
