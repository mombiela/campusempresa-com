Introducció a les dades estructurades

Les dades estructurades són una forma d'etiquetar contingut al teu lloc web perquè els motors de cerca puguin entendre'l millor. Utilitzant un format estàndard, com JSON-LD, pots proporcionar informació addicional sobre el teu contingut, cosa que pot millorar la visibilitat del teu lloc als resultats de cerca.

  • Definició: Les dades estructurades són un format estandarditzat per proporcionar informació sobre una pàgina i classificar el contingut de la pàgina.
  • Beneficis: Millora la visibilitat als motors de cerca, augmenta la taxa de clics (CTR), i facilita la indexació del contingut.

Tipus de dades estructurades

Hi ha diversos tipus de dades estructurades que pots implementar depenent del contingut del teu lloc web:

  • Articles: Per a notícies, blogs i articles.
  • Productes: Per a botigues en línia que venen productes.
  • Esdeveniments: Per a promocionar esdeveniments.
  • Persones: Per a perfils de persones.
  • Organitzacions: Per a informació sobre empreses i organitzacions.
  • Receptes: Per a llocs de cuina i receptes.

Formats de dades estructurades

El format més comunament utilitzat per implementar dades estructurades és JSON-LD (JavaScript Object Notation for Linked Data).

Exemple de JSON-LD per a un article

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Com implementar dades estructurades per a SEO",
  "author": {
    "@type": "Person",
    "name": "Joan Pérez"
  },
  "datePublished": "2023-10-01",
  "image": "https://example.com/images/article.jpg",
  "publisher": {
    "@type": "Organization",
    "name": "Exemple de Publicació",
    "logo": {
      "@type": "ImageObject",
      "url": "https://example.com/logo.jpg"
    }
  },
  "description": "Una guia completa sobre com implementar dades estructurades per millorar el SEO del teu lloc web."
}
</script>

Explicació del codi

  • @context: Defineix el context de la informació, en aquest cas, https://schema.org.
  • @type: Especifica el tipus de contingut, en aquest cas, Article.
  • headline: El títol de l'article.
  • author: Informació sobre l'autor de l'article.
  • datePublished: Data de publicació de l'article.
  • image: URL de la imatge associada amb l'article.
  • publisher: Informació sobre l'organització que publica l'article.
  • description: Una breu descripció del contingut de l'article.

Implementació en HTML

Per implementar dades estructurades al teu lloc web, has d'inserir el codi JSON-LD a la secció <head> del teu HTML o just abans del tancament de l'etiqueta </body>.

Exemple d'implementació en HTML

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Com implementar dades estructurades per a SEO</title>
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "Article",
    "headline": "Com implementar dades estructurades per a SEO",
    "author": {
      "@type": "Person",
      "name": "Joan Pérez"
    },
    "datePublished": "2023-10-01",
    "image": "https://example.com/images/article.jpg",
    "publisher": {
      "@type": "Organization",
      "name": "Exemple de Publicació",
      "logo": {
        "@type": "ImageObject",
        "url": "https://example.com/logo.jpg"
      }
    },
    "description": "Una guia completa sobre com implementar dades estructurades per millorar el SEO del teu lloc web."
  }
  </script>
</head>
<body>
  <h1>Com implementar dades estructurades per a SEO</h1>
  <p>Una guia completa sobre com implementar dades estructurades per millorar el SEO del teu lloc web.</p>
</body>
</html>

Eines per validar dades estructurades

És important validar les teves dades estructurades per assegurar-te que estan correctament implementades. Algunes eines útils inclouen:

  • Google's Structured Data Testing Tool: Permet provar i validar les teves dades estructurades.
  • Rich Results Test: Eina de Google per verificar si les teves dades estructurades poden generar resultats enriquits.

Conclusió

Implementar dades estructurades és una tècnica poderosa per millorar la visibilitat i el rendiment del teu lloc web als motors de cerca. Utilitzant formats com JSON-LD i eines de validació, pots assegurar-te que el teu contingut està ben etiquetat i optimitzat per a SEO. A mesura que avancis en la implementació de dades estructurades, podràs explorar tipus més complexos i específics per maximitzar els beneficis per al teu lloc web.
