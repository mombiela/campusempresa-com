En aquesta secció, tractarem com inserir dades a les taules de PostgreSQL. Aquesta és una habilitat fonamental per a qualsevol usuari de bases de dades, ja que et permet omplir les teves taules amb les dades necessàries per a les teves aplicacions. Començarem amb insercions bàsiques i avançarem cap a tècniques més avançades.

Insercions Bàsiques

La manera més senzilla d'inserir dades en una taula de PostgreSQL és utilitzant la instrucció INSERT INTO.

Sintaxi

INSERT INTO nom_taula (columna1, columna2, columna3, ...)
VALUES (valor1, valor2, valor3, ...);

Exemple

Suposem que tenim una taula anomenada students amb la següent estructura:

CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    age INT,
    grade VARCHAR(10)
);

Per inserir un nou estudiant en aquesta taula, utilitzaries la següent instrucció SQL:

INSERT INTO students (name, age, grade)
VALUES ('John Doe', 20, 'A');

Explicació

  • INSERT INTO students: Especifica la taula on s'inseriran les dades.
  • (name, age, grade): Llista les columnes que rebran les dades.
  • VALUES ('John Doe', 20, 'A'): Proporciona les dades reals que s'inseriran a les columnes especificades.

Inserir Múltiples Files

Pots inserir múltiples files en una sola instrucció INSERT INTO separant els valors de cada fila amb una coma.

Exemple

INSERT INTO students (name, age, grade)
VALUES 
    ('Jane Smith', 22, 'B'),
    ('Emily Johnson', 21, 'A'),
    ('Michael Brown', 23, 'C');

Explicació

  • Cada conjunt de valors dins de la clàusula VALUES correspon a una nova fila a la taula students.

Inserir Dades d'una Altra Taula

També pots inserir dades en una taula seleccionant dades d'una altra taula. Això és útil per copiar dades o transformar dades d'una taula a una altra.

Sintaxi

INSERT INTO taula_destí (columna1, columna2, ...)
SELECT columna1, columna2, ...
FROM taula_origen
WHERE condició;

Exemple

Suposem que tenim una altra taula new_students amb la mateixa estructura que students. Per copiar totes les dades de new_students a students, utilitzaries:

INSERT INTO students (name, age, grade)
SELECT name, age, grade
FROM new_students;

Explicació

  • SELECT name, age, grade FROM new_students: Selecciona les dades de la taula new_students.
  • INSERT INTO students (name, age, grade): Insereix les dades seleccionades a la taula students.

Utilitzar Valors per Defecte

Si una columna té un valor per defecte, pots ometre-la en la instrucció INSERT INTO. PostgreSQL utilitzarà automàticament el valor per defecte per a aquesta columna.

Exemple

Suposem que la columna grade a la taula students té un valor per defecte de 'B'. Pots inserir un nou estudiant sense especificar la nota:

INSERT INTO students (name, age)
VALUES ('Alice Green', 19);

Explicació

  • La columna grade s'establirà automàticament a 'B' per a la nova fila.

Inserir Dades JSON

PostgreSQL admet tipus de dades JSON, permetent-te inserir dades JSON directament en una taula.

Exemple

Suposem que tenim una taula student_info amb una columna JSON:

CREATE TABLE student_info (
    id SERIAL PRIMARY KEY,
    info JSON
);

Per inserir dades JSON en aquesta taula:

INSERT INTO student_info (info)
VALUES ('{"name": "Tom White", "age": 24, "grade": "A"}');

Explicació

  • La columna info emmagatzema l'objecte JSON com un valor únic.

Conclusió

Inserir dades a les taules de PostgreSQL és una operació fonamental que es pot realitzar de diverses maneres segons les teves necessitats. Tant si estàs inserint files individuals, múltiples files, copiant dades d'una altra taula o treballant amb dades JSON, PostgreSQL proporciona eines flexibles i potents per gestionar les teves dades de manera eficient. Entendre aquestes tècniques t'ajudarà a omplir i gestionar les teves bases de dades de manera efectiva.
