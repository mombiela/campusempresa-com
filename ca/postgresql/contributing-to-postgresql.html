El Projecte Sobre nosaltres Contribuir Donacions Llicència

Contribuir a PostgreSQL pot ser una experiència gratificant, ja sigui arreglant errors, afegint noves funcionalitats o millorant la documentació. Aquesta secció et guiarà a través del procés de contribuir a PostgreSQL, des de la configuració del teu entorn de desenvolupament fins a enviar el teu primer pedaç.

Configuració del Teu Entorn de Desenvolupament

Abans de començar a contribuir, necessites configurar el teu entorn de desenvolupament.

  • Instal·la Git: PostgreSQL utilitza Git per al control de versions.
  • Clona el Repositori de PostgreSQL: Necessites clonar el repositori de PostgreSQL a la teva màquina local.
  • Instal·la les Dependències de Construcció: Assegura't de tenir totes les eines i biblioteques necessàries per construir PostgreSQL des del codi font.

Instal·lant Git

Si encara no has instal·lat Git, pots descarregar-lo des de git-scm.com. Segueix les instruccions d'instal·lació per al teu sistema operatiu.

Clonant el Repositori de PostgreSQL

Obre el teu terminal i executa la següent comanda per clonar el repositori de PostgreSQL:

git clone https://github.com/postgres/postgres.git
cd postgres

Instal·lant les Dependències de Construcció

Les dependències necessàries per construir PostgreSQL poden variar segons el teu sistema operatiu. Aquí tens les comandes per a alguns sistemes operatius comuns:

  • Ubuntu/Debian:

    sudo apt-get update
sudo apt-get install build-essential libreadline-dev zlib1g-dev flex bison

  • Fedora/Red Hat:

    sudo dnf groupinstall "Development Tools"
sudo dnf install readline-devel zlib-devel flex bison

  • macOS:

    xcode-select --install
brew install readline

Construint PostgreSQL des del Codi Font

Un cop tinguis totes les dependències instal·lades, pots construir PostgreSQL des del codi font.

  1. Configura la Construcció: Executa el script configure per configurar l'entorn de construcció.

    ./configure

  2. Compila el Codi Font: Utilitza la comanda make per compilar el codi font.

    make

  3. Executa les Proves: Assegura't que tot funciona correctament executant les proves.

    make check

  4. Instal·la PostgreSQL: Finalment, instal·la PostgreSQL al teu sistema.

    sudo make install

Entenent la Base de Codi

Abans de fer cap canvi, és essencial entendre l'estructura de la base de codi de PostgreSQL.

  • src/backend: Conté el codi del servidor principal.
  • src/include: Conté els fitxers de capçalera.
  • src/bin: Conté aplicacions client com psql.
  • src/test: Conté scripts de prova i dades de prova.

Fent Canvis

Quan estiguis llest per fer canvis, segueix aquests passos:

  1. Crea una Nova Branca: Sempre crea una nova branca per als teus canvis.

    git checkout -b my-feature-branch

  2. Fes els Teus Canvis: Edita el codi i fes els teus canvis.

  3. Comet els Teus Canvis: Comet els teus canvis amb un missatge descriptiu.

    git add .
git commit -m "Descripció dels meus canvis"

  4. Executa les Proves: Assegura't que els teus canvis no trenquen res executant les proves.

    make check

Enviant un Pedaç

Un cop hagis fet i provat els teus canvis, pots enviar un pedaç.

  1. Genera un Pedaç: Utilitza la comanda git format-patch per generar un fitxer de pedaç.

    git format-patch origin/master

  2. Envia el Pedaç: Envia el fitxer de pedaç a la llista de correu de PostgreSQL per a revisió.

Conclusió

Contribuir a PostgreSQL implica configurar el teu entorn de desenvolupament, entendre la base de codi, fer canvis i enviar pedaços. Seguint aquests passos, pots contribuir de manera efectiva i ajudar a millorar PostgreSQL. Recorda interactuar amb la comunitat, buscar feedback i millorar contínuament les teves contribucions. Feliç programació!
