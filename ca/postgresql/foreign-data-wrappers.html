Wrappers de Dades Externes
Tipus de dades personalitzats ►

Els Wrappers de Dades Externes (FDWs) són una característica poderosa de PostgreSQL que et permet accedir i manipular dades emmagatzemades en bases de dades externes com si fossin part de la teva base de dades PostgreSQL. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat dels FDWs.

Introducció als Wrappers de Dades Externes

  • Definició: Els Wrappers de Dades Externes són una característica de PostgreSQL que et permet consultar i interactuar amb fonts de dades externes.
  • Casos d'ús:
    • Integrar dades de diferents bases de dades.
    • Accedir a dades de fonts no relacionals.
    • Realitzar consultes federades a través de múltiples fonts de dades.

Configuració dels Wrappers de Dades Externes

Instal·lació de l'Extensió Necessària

Per utilitzar els FDWs, necessites instal·lar l'extensió adequada. Per exemple, per connectar-te a una altra base de dades PostgreSQL, utilitzaries l'extensió postgres_fdw.

CREATE EXTENSION IF NOT EXISTS postgres_fdw;

Creació d'un Servidor Extern

Un servidor extern defineix els detalls de connexió a la font de dades externa.

CREATE SERVER foreign_server
FOREIGN DATA WRAPPER postgres_fdw
OPTIONS (host 'remote_host', dbname 'remote_db', port '5432');

Creació de Mapes d'Usuari

Els mapes d'usuari defineixen els detalls d'autenticació per accedir al servidor extern.

CREATE USER MAPPING FOR local_user
SERVER foreign_server
OPTIONS (user 'remote_user', password 'remote_password');

Creació de Taules Externes

Les taules externes representen les dades externes dins de la teva base de dades PostgreSQL.

CREATE FOREIGN TABLE foreign_table (
    id integer,
    name text,
    value numeric
)
SERVER foreign_server
OPTIONS (schema_name 'public', table_name 'remote_table');

Consultar Taules Externes

Un cop la taula externa està configurada, pots consultar-la igual que qualsevol altra taula a la teva base de dades PostgreSQL.

SELECT * FROM foreign_table;

Ús Avançat

Importar un Esquema Extern

Pots importar un esquema sencer des del servidor extern.

IMPORT FOREIGN SCHEMA public
FROM SERVER foreign_server
INTO local_schema;

Unir Taules Locals i Externes

Pots realitzar unions entre taules locals i externes.

SELECT local_table.id, local_table.name, foreign_table.value
FROM local_table
JOIN foreign_table ON local_table.id = foreign_table.id;

Consideracions de Rendiment

  • Capacitats de Pushdown: Els FDWs poden empènyer certes operacions (com filtres i unions) al servidor extern per millorar el rendiment.
  • Latència de Xarxa: Tingues en compte la latència de xarxa quan consultis grans conjunts de dades des de servidors remots.

Wrappers de Dades Externes Suportats

PostgreSQL suporta una varietat de FDWs per a diferents fonts de dades. Aquí tens una comparació d'alguns FDWs populars:

| FDW | Font de Dades | Cas d'Ús | |-----------------|-------------------|-------------------------------------------| | postgres_fdw | PostgreSQL | Accedir a altres bases de dades PostgreSQL| | mysql_fdw | MySQL | Integrar bases de dades MySQL | | oracle_fdw | Oracle | Accedir a bases de dades Oracle | | file_fdw | CSV/Flat Files | Llegir dades de fitxers CSV o plans | | mongodb_fdw | MongoDB | Integrar col·leccions MongoDB |

Conclusió

Els Wrappers de Dades Externes són una eina versàtil a PostgreSQL que permet una integració perfecta amb diverses fonts de dades externes. Entenent com configurar i utilitzar els FDWs, pots ampliar les capacitats de la teva base de dades PostgreSQL i realitzar consultes complexes a través de diferents sistemes de dades. Ja sigui unint dades de múltiples bases de dades o accedint a dades no relacionals, els FDWs proporcionen una solució robusta per a les teves necessitats d'integració de dades.
