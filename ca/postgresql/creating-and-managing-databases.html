Creació de Bases de Dades

  • Crear una Base de Dades:

    CREATE DATABASE mydatabase;

    Explicació: Aquesta comanda crea una nova base de dades anomenada mydatabase.

  • Llistar Bases de Dades:

    \l

    Explicació: Aquesta comanda llista totes les bases de dades en la instància de PostgreSQL.

  • Connectar-se a una Base de Dades:

    \c mydatabase

    Explicació: Aquesta comanda connecta a la base de dades mydatabase.

Gestió de Bases de Dades

  • Canviar el Nom d'una Base de Dades:

    ALTER DATABASE mydatabase RENAME TO newdatabase;

    Explicació: Aquesta comanda canvia el nom de mydatabase a newdatabase.

  • Eliminar una Base de Dades:

    DROP DATABASE mydatabase;

    Explicació: Aquesta comanda elimina la base de dades mydatabase. Utilitzeu amb precaució, ja que aquesta acció és irreversible.

Usuaris i Permisos

  • Crear un Usuari:

    CREATE USER myuser WITH PASSWORD 'mypassword';

    Explicació: Aquesta comanda crea un nou usuari myuser amb la contrasenya mypassword.

  • Concedir Permisos:

    GRANT ALL PRIVILEGES ON DATABASE mydatabase TO myuser;

    Explicació: Aquesta comanda concedeix tots els privilegis sobre mydatabase a myuser.

  • Revocar Permisos:

    REVOKE ALL PRIVILEGES ON DATABASE mydatabase FROM myuser;

    Explicació: Aquesta comanda revoca tots els privilegis sobre mydatabase de myuser.

Taules i Esquemes

  • Crear una Taula:

    CREATE TABLE employees (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    position VARCHAR(50),
    salary NUMERIC
);

    Explicació: Aquesta comanda crea una taula anomenada employees amb columnes per a id, name, position i salary.

  • Llistar Taules:

    \dt

    Explicació: Aquesta comanda llista totes les taules en la base de dades actual.

  • Crear un Esquema:

    CREATE SCHEMA myschema;

    Explicació: Aquesta comanda crea un nou esquema anomenat myschema.

  • Utilitzar un Esquema:

    SET search_path TO myschema;

    Explicació: Aquesta comanda estableix el camí de cerca a myschema, fent-lo l'esquema per defecte per a les comandes següents.

Gestió Avançada de Bases de Dades

  • Còpia de Seguretat i Restauració:

    • Còpia de Seguretat d'una Base de Dades:

      pg_dump mydatabase > mydatabase_backup.sql

      Explicació: Aquesta comanda crea una còpia de seguretat de mydatabase i la guarda a mydatabase_backup.sql.

    • Restaurar una Base de Dades:

      psql mydatabase < mydatabase_backup.sql

      Explicació: Aquesta comanda restaura mydatabase des del fitxer mydatabase_backup.sql.

Optimització del Rendiment

  • Indexació:

    CREATE INDEX idx_name ON employees (name);

    Explicació: Aquesta comanda crea un índex a la columna name de la taula employees per millorar el rendiment de les consultes.

  • Optimització de Consultes:

    • Utilització de EXPLAIN per analitzar plans de consultes: 
      EXPLAIN SELECT * FROM employees WHERE name = 'John Doe';
      Explicació: Aquesta comanda mostra el pla d'execució per a la consulta, ajudant a identificar colls d'ampolla de rendiment.

  • Vacuuming:

    VACUUM FULL;

    Explicació: Aquesta comanda recupera emmagatzematge i optimitza la base de dades eliminant tuples mortes.

Conclusió

En aquest tema, hem cobert els aspectes essencials de la creació i gestió de bases de dades en PostgreSQL, des de la creació bàsica de bases de dades fins a tècniques avançades d'optimització del rendiment. Entenent aquests conceptes, podeu gestionar efectivament les vostres bases de dades PostgreSQL, assegurant que siguin segures, eficients i escalables.
