Introducció

Les funcions de data i hora són essencials per gestionar i manipular dades temporals a PostgreSQL. Aquesta secció cobrirà diverses funcions i operacions que pots realitzar sobre tipus de dades de data i hora, des de nivells bàsics fins a avançats.

Funcions Bàsiques de Data i Hora

Data i Hora Actuals

  • CURRENT_DATE: Retorna la data actual.
  • CURRENT_TIME: Retorna l'hora actual.
  • CURRENT_TIMESTAMP: Retorna la data i hora actuals (timestamp).
SELECT CURRENT_DATE;       -- Retorna la data actual
SELECT CURRENT_TIME;       -- Retorna l'hora actual
SELECT CURRENT_TIMESTAMP;  -- Retorna la data i hora actuals

Extracció de Parts d'una Data/Hora

  • EXTRACT(field FROM source): Extreu una part específica d'un valor de data/hora.
SELECT EXTRACT(YEAR FROM CURRENT_DATE);  -- Extreu l'any de la data actual
SELECT EXTRACT(MONTH FROM CURRENT_DATE); -- Extreu el mes de la data actual
SELECT EXTRACT(DAY FROM CURRENT_DATE);   -- Extreu el dia de la data actual

Aritmètica de Data i Hora

  • Suma/Restar: Pots sumar o restar intervals a/de dates.
SELECT CURRENT_DATE + INTERVAL '1 day';  -- Afegeix un dia a la data actual
SELECT CURRENT_DATE - INTERVAL '1 month'; -- Resta un mes de la data actual

Funcions Intermèdies de Data i Hora

Formatació de Data i Hora

  • TO_CHAR: Converteix un valor de data/hora a una cadena amb un format especificat.
SELECT TO_CHAR(CURRENT_TIMESTAMP, 'YYYY-MM-DD HH24:MI:SS'); -- Format de l'actual timestamp

Anàlisi de Data i Hora

  • TO_DATE: Converteix una cadena a una data.
  • TO_TIMESTAMP: Converteix una cadena a un timestamp.
SELECT TO_DATE('2023-10-01', 'YYYY-MM-DD'); -- Converteix una cadena a una data
SELECT TO_TIMESTAMP('2023-10-01 12:30:00', 'YYYY-MM-DD HH24:MI:SS'); -- Converteix una cadena a un timestamp

Càlcul d'Edat

  • AGE: Calcula l'edat entre dues dates.
SELECT AGE('2023-10-01', '2000-01-01'); -- Calcula l'edat entre dues dates

Funcions Avançades de Data i Hora

Fusos Horaris

  • AT TIME ZONE: Converteix un timestamp a un fuso horari diferent.
SELECT '2023-10-01 12:00:00'::timestamp AT TIME ZONE 'UTC'; -- Converteix a fuso horari UTC
SELECT '2023-10-01 12:00:00'::timestamp AT TIME ZONE 'America/New_York'; -- Converteix a fuso horari de Nova York

Aritmètica d'Interval

  • JUSTIFY_DAYS: Ajusta intervals per ser més llegibles.
  • JUSTIFY_HOURS: Ajusta intervals per ser més llegibles.
SELECT JUSTIFY_DAYS(INTERVAL '30 days'); -- Ajusta l'interval per ser més llegible
SELECT JUSTIFY_HOURS(INTERVAL '25 hours'); -- Ajusta l'interval per ser més llegible

Agregació de Data i Hora

  • Agrupació per Data/Hora: Útil per resumir dades en períodes de temps.
SELECT DATE_TRUNC('month', order_date) AS month, COUNT(*) 
FROM orders 
GROUP BY DATE_TRUNC('month', order_date); -- Agrupa comandes per mes

Resum

En aquesta secció, hem cobert una àmplia gamma de funcions de data i hora a PostgreSQL, des de la recuperació bàsica de la data i hora actuals fins a la manipulació i formatació avançada. Entendre aquestes funcions et permetrà gestionar dades temporals de manera més efectiva, ja sigui realitzant aritmètica de dates simple o conversions complexes de fusos horaris. Practica aquestes funcions amb dades del món real per consolidar la teva comprensió i millorar les teves habilitats en PostgreSQL.
