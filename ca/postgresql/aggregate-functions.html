Les funcions agregades són essencials en SQL per realitzar càlculs sobre múltiples files d'una columna d'una taula i retornar un valor únic. PostgreSQL ofereix una varietat de funcions agregades per ajudar-te a resumir i analitzar les teves dades de manera efectiva.

Funcions agregades clau

COUNT

La funció COUNT retorna el nombre de files d'entrada que coincideixen amb una condició específica.

SELECT COUNT(*) FROM employees;

Aquesta consulta retorna el nombre total de files a la taula employees.

SUM

La funció SUM calcula la suma total d'una columna numèrica.

SELECT SUM(salary) FROM employees;

Aquesta consulta retorna la suma total de tots els salaris a la taula employees.

AVG

La funció AVG calcula el valor mitjà d'una columna numèrica.

SELECT AVG(salary) FROM employees;

Aquesta consulta retorna el salari mitjà de tots els empleats.

MAX

La funció MAX retorna el valor màxim d'una columna.

SELECT MAX(salary) FROM employees;

Aquesta consulta retorna el salari més alt a la taula employees.

MIN

La funció MIN retorna el valor mínim d'una columna.

SELECT MIN(salary) FROM employees;

Aquesta consulta retorna el salari més baix a la taula employees.

Ús de funcions agregades amb GROUP BY

La clàusula GROUP BY s'utilitza amb funcions agregades per agrupar el conjunt de resultats per una o més columnes.

SELECT department, AVG(salary) 
FROM employees 
GROUP BY department;

Aquesta consulta retorna el salari mitjà per a cada departament.

Clàusula HAVING

La clàusula HAVING s'utilitza per filtrar grups basats en una condició, similar a la clàusula WHERE però per a grups.

SELECT department, AVG(salary) 
FROM employees 
GROUP BY department 
HAVING AVG(salary) > 50000;

Aquesta consulta retorna els departaments on el salari mitjà és superior a 50.000.

Combinació de funcions agregades

Pots utilitzar múltiples funcions agregades en una sola consulta per obtenir un resum complet.

SELECT department, COUNT(*), AVG(salary), MAX(salary), MIN(salary) 
FROM employees 
GROUP BY department;

Aquesta consulta retorna el recompte d'empleats, el salari mitjà, el salari màxim i el salari mínim per a cada departament.

Funcions agregades amb DISTINCT

La paraula clau DISTINCT es pot utilitzar dins de funcions agregades per considerar només valors únics.

SELECT COUNT(DISTINCT department) 
FROM employees;

Aquesta consulta retorna el nombre de departaments únics a la taula employees.

Funcions de finestra

Les funcions de finestra et permeten realitzar càlculs agregats a través d'un conjunt de files de la taula que estan d'alguna manera relacionades amb la fila actual.

SELECT employee_id, department, salary, 
       AVG(salary) OVER (PARTITION BY department) AS avg_department_salary 
FROM employees;

Aquesta consulta retorna el salari de cada empleat juntament amb el salari mitjà del seu departament.

Conclusió

Les funcions agregades a PostgreSQL són eines poderoses per resumir i analitzar dades. Dominant aquestes funcions, pots realitzar càlculs complexos i obtenir informació valuosa dels teus conjunts de dades. Recorda utilitzar les clàusules GROUP BY i HAVING per refinar les teves consultes i extreure informació significativa. Practica utilitzant aquestes funcions amb diferents conjunts de dades per convertir-te en un expert en anàlisi de dades amb PostgreSQL.
