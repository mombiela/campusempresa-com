Introducció a SQL i PostgreSQL

  • SQL (Structured Query Language): Un llenguatge estàndard per gestionar i manipular bases de dades.
  • PostgreSQL: Un sistema avançat de gestió de bases de dades relacionals (RDBMS) de codi obert conegut per la seva robustesa i compliment dels estàndards.

Configuració de PostgreSQL

  • Instal·lació: Instruccions per instal·lar PostgreSQL en diversos sistemes operatius.
  • Connexió a PostgreSQL: Utilitzant l'eina de línia de comandes psql o eines gràfiques com pgAdmin.

Creació de bases de dades i taules

  • Crear una base de dades:

    CREATE DATABASE mydatabase;
    • Aquesta comanda crea una nova base de dades anomenada mydatabase.

  • Crear una taula:

    CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
    • id SERIAL PRIMARY KEY: Clau primària amb auto-increment.
    • name VARCHAR(100): Un camp de caràcter variable amb una longitud màxima de 100.
    • email VARCHAR(100) UNIQUE: Un camp únic per emmagatzemar adreces de correu electrònic.
    • created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP: Un camp de marca de temps amb un valor per defecte de l'hora actual.

Manipulació bàsica de dades

  • Inserir dades:

    INSERT INTO users (name, email) VALUES ('John Doe', 'john.doe@example.com');
    • Aquesta comanda insereix una nova fila a la taula users.

  • Actualitzar dades:

    UPDATE users SET email = 'john.new@example.com' WHERE name = 'John Doe';
    • Aquesta comanda actualitza el correu electrònic de l'usuari anomenat 'John Doe'.

  • Eliminar dades:

    DELETE FROM users WHERE name = 'John Doe';
    • Aquesta comanda elimina l'usuari anomenat 'John Doe' de la taula users.

Consultar dades

  • Instrucció SELECT bàsica:

    SELECT * FROM users;
    • Aquesta comanda recupera totes les columnes de la taula users.

  • Filtrar resultats amb WHERE:

    SELECT * FROM users WHERE email = 'john.doe@example.com';
    • Aquesta comanda recupera les files on el correu electrònic és 'john.doe@example.com'.

  • Ordenar resultats amb ORDER BY:

    SELECT * FROM users ORDER BY created_at DESC;
    • Aquesta comanda recupera tots els usuaris ordenats pel camp created_at en ordre descendent.

Agregar dades

  • COUNT, AVG, MAX, MIN, SUM: 
    SELECT COUNT(*) FROM users;
SELECT AVG(salary) FROM employees;
SELECT MAX(age) FROM users;
SELECT MIN(age) FROM users;
SELECT SUM(salary) FROM employees;
    • Aquestes comandes realitzen diverses funcions agregades a les columnes especificades.

Unir taules

  • INNER JOIN:

    SELECT users.name, orders.amount 
FROM users 
INNER JOIN orders ON users.id = orders.user_id;
    • Aquesta comanda recupera dades de les taules users i orders on hi ha una coincidència de user_id.

  • LEFT JOIN:

    SELECT users.name, orders.amount 
FROM users 
LEFT JOIN orders ON users.id = orders.user_id;
    • Aquesta comanda recupera tots els usuaris i les seves comandes, incloent-hi els usuaris sense comandes.

Conclusió

En aquest tema, hem cobert els conceptes fonamentals de SQL i PostgreSQL, incloent la creació de bases de dades i taules, manipulació bàsica de dades, consultes, agregació de dades i unir taules. Aquests conceptes bàsics formen la base per a operacions SQL més avançades i tasques de gestió de bases de dades. Practica aquestes comandes per familiaritzar-te amb la sintaxi de SQL i les capacitats de PostgreSQL.
