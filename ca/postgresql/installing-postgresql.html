Introducció

PostgreSQL és un sistema de bases de dades objecte-relacional potent i de codi obert. És conegut per la seva robustesa, extensibilitat i compliment dels estàndards. Aquesta secció et guiarà a través del procés d'instal·lació de PostgreSQL en diversos sistemes operatius.

Instal·lació en Diferents Sistemes Operatius

Windows

  1. Descarregar l'Instal·lador:

    • Visita el lloc web oficial de PostgreSQL: PostgreSQL Downloads.
    • Descarrega l'instal·lador per a l'última versió.

  2. Executar l'Instal·lador:

    • Fes doble clic al fitxer .exe descarregat per començar el procés d'instal·lació.
    • Segueix les instruccions de l'assistent d'instal·lació:
      • Tria el directori d'instal·lació.
      • Selecciona els components que vols instal·lar (la selecció per defecte és recomanada per a principiants).
      • Defineix la contrasenya per al superusuari de PostgreSQL (l'usuari per defecte és postgres).
      • Tria el número de port (el per defecte és 5432).
      • Selecciona la configuració de la localització.

  3. Completar la Instal·lació:

    • Fes clic a "Next" en els passos restants i després a "Finish" per completar la instal·lació.
    • PostgreSQL s'instal·larà i el servei de PostgreSQL s'iniciarà automàticament.

macOS

  1. Utilitzant Homebrew:

    • Obre l'aplicació Terminal.
    • Instal·la Homebrew si no ho has fet ja: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la PostgreSQL utilitzant Homebrew: 
      brew install postgresql

  2. Inicialitzar i Iniciar PostgreSQL:

    • Inicialitza el clúster de bases de dades: 
      initdb /usr/local/var/postgres
    • Inicia el servei de PostgreSQL: 
      brew services start postgresql

  3. Verificar la Instal·lació:

    • Comprova la versió de PostgreSQL per verificar la instal·lació: 
      psql --version

Linux

  1. Utilitzant el Gestor de Paquets (Debian/Ubuntu):

    • Actualitza la llista de paquets: 
      sudo apt update
    • Instal·la PostgreSQL: 
      sudo apt install postgresql postgresql-contrib

  2. Utilitzant el Gestor de Paquets (Red Hat/CentOS):

    • Instal·la el repositori de PostgreSQL: 
      sudo yum install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-$(rpm -E %rhel)-x86_64/pgdg-redhat-repo-latest.noarch.rpm
    • Instal·la PostgreSQL: 
      sudo yum install -y postgresql13-server postgresql13

  3. Inicialitzar i Iniciar PostgreSQL:

    • Inicialitza el clúster de bases de dades (per Red Hat/CentOS): 
      sudo /usr/pgsql-13/bin/postgresql-13-setup initdb
    • Inicia el servei de PostgreSQL: 
      sudo systemctl start postgresql
    • Habilita PostgreSQL per iniciar-se en arrencar: 
      sudo systemctl enable postgresql

Passos Posteriors a la Instal·lació

  1. Accedir a PostgreSQL:

    • Obre el terminal interactiu de PostgreSQL (psql): 
      sudo -i -u postgres
psql

  2. Crear una Nova Base de Dades i Usuari:

    • Crea una nova base de dades: 
      CREATE DATABASE mydatabase;
    • Crea un nou usuari amb una contrasenya: 
      CREATE USER myuser WITH PASSWORD 'mypassword';
    • Concedeix privilegis al nou usuari: 
      GRANT ALL PRIVILEGES ON DATABASE mydatabase TO myuser;

Conclusió

Instal·lar PostgreSQL és un procés senzill que varia lleugerament segons el teu sistema operatiu. Seguint els passos descrits en aquesta secció, hauràs d'aconseguir tenir PostgreSQL funcionant al teu ordinador. Aquesta configuració servirà com a base per a la resta del curs, on aprendràs a gestionar bases de dades, realitzar consultes i utilitzar les funcions avançades de PostgreSQL.
