Introducció

En aquesta secció, cobrirem els passos necessaris per implementar una base de dades utilitzant PostgreSQL. Això inclou crear bases de dades, definir esquemes i omplir taules amb dades. Al final d'aquesta secció, tindràs una comprensió sòlida de com configurar i gestionar una base de dades PostgreSQL.

Creació d'una Base de Dades

Per començar a utilitzar PostgreSQL, primer has de crear una base de dades. Aquí tens com fer-ho:

Utilitzant el Comandament `createdb`

El comandament createdb és una manera convenient de crear una nova base de dades des de la línia de comandaments.

createdb mydatabase

Utilitzant Comandaments SQL

També pots crear una base de dades utilitzant comandaments SQL dins de la interfície psql.

CREATE DATABASE mydatabase;
  • Nota: Assegura't de tenir els permisos necessaris per crear una base de dades.

Connexió a una Base de Dades

Un cop la base de dades està creada, has de connectar-t'hi per començar a treballar-hi.

Utilitzant el Comandament `psql`

Pots connectar-te a la teva base de dades utilitzant l'eina de línia de comandaments psql.

psql mydatabase

Utilitzant Comandaments SQL

Dins de psql, pots connectar-te a una base de dades utilitzant el comandament \c.

\c mydatabase

Definició d'Esquemes

Els esquemes són una manera d'organitzar i gestionar els objectes de la teva base de dades. Per defecte, PostgreSQL crea un esquema anomenat public.

Creació d'un Esquema

Pots crear un nou esquema utilitzant el comandament CREATE SCHEMA.

CREATE SCHEMA myschema;

Configuració del Camí de Cerca

Pots configurar el camí de cerca per especificar en quins esquemes PostgreSQL ha de buscar primer.

SET search_path TO myschema, public;

Creació de Taules

Les taules són els blocs fonamentals d'una base de dades. Emmagatzemen dades en files i columnes.

Creació Bàsica de Taules

Aquí tens com pots crear una taula simple:

CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
  • Explicació:
    • id SERIAL PRIMARY KEY: Crea una clau primària auto-incremental.
    • username VARCHAR(50) NOT NULL: Defineix una columna de nom d'usuari que no pot ser nul·la.
    • email VARCHAR(100) NOT NULL: Defineix una columna de correu electrònic que no pot ser nul·la.
    • created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP: Estableix un segell de temps per defecte per quan es crea la fila.

Omplir Taules amb Dades

Un cop les teves taules estan creades, pots començar a inserir-hi dades.

Inserció de Dades

Utilitza el comandament INSERT INTO per afegir dades a la teva taula.

INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com');

Inserció Massiva

Per inserir múltiples files alhora, pots utilitzar múltiples claus VALUES.

INSERT INTO users (username, email) VALUES 
('jane_doe', 'jane@example.com'),
('alice', 'alice@example.com'),
('bob', 'bob@example.com');

Consultar Dades

Recuperar dades de les teves taules es fa utilitzant la sentència SELECT.

Consulta Bàsica

Aquí tens una consulta simple per recuperar tots els usuaris:

SELECT * FROM users;

Filtrar Dades

Utilitza la clàusula WHERE per filtrar resultats.

SELECT * FROM users WHERE username = 'john_doe';

Ordenar Dades

Utilitza la clàusula ORDER BY per ordenar resultats.

SELECT * FROM users ORDER BY created_at DESC;

Actualitzar Dades

Per modificar dades existents, utilitza la sentència UPDATE.

Actualització Bàsica

Aquí tens com pots actualitzar el correu electrònic d'un usuari:

UPDATE users SET email = 'new_email@example.com' WHERE username = 'john_doe';

Esborrar Dades

Per eliminar dades de la teva taula, utilitza la sentència DELETE.

Esborrat Bàsic

Aquí tens com pots esborrar un usuari:

DELETE FROM users WHERE username = 'john_doe';

Conclusió

En aquesta secció, hem cobert els passos essencials per implementar una base de dades en PostgreSQL. Has après com crear i connectar-te a una base de dades, definir esquemes, crear i omplir taules, i realitzar operacions bàsiques de manipulació de dades. Amb aquestes habilitats, ara estàs equipat per configurar i gestionar una base de dades PostgreSQL de manera efectiva.
