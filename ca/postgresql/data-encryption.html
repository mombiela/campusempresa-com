Introducció al Xifrat de Dades

El xifrat de dades és un aspecte crític de la seguretat de bases de dades, assegurant que la informació sensible estigui protegida contra l'accés no autoritzat. PostgreSQL ofereix diverses metodologies per xifrar dades, tant en repòs com en trànsit.

Conceptes Clau

  • Xifrat: El procés de convertir text pla en text xifrat per evitar l'accés no autoritzat.
  • Desxifrat: El procés de convertir text xifrat de nou en text pla.
  • Xifrat Simètric: Utilitza la mateixa clau tant per al xifrat com per al desxifrat.
  • Xifrat Asimètric: Utilitza un parell de claus (pública i privada) per al xifrat i desxifrat.
  • SSL/TLS: Protocols per xifrar dades en trànsit entre el client i el servidor.

Xifrat de Dades en Trànsit

El xifrat de dades en trànsit assegura que les dades transferides entre el servidor PostgreSQL i els clients siguin segures.

Configuració de SSL/TLS

  1. Generar Certificats SSL:

    openssl req -new -text -out server.req
openssl rsa -in privkey.pem -out server.key
openssl req -x509 -in server.req -text -key server.key -out server.crt

  2. Configurar PostgreSQL per Utilitzar SSL: Edita el fitxer postgresql.conf:

    ssl = on
ssl_cert_file = 'server.crt'
ssl_key_file = 'server.key'

  3. Reiniciar PostgreSQL:

    sudo systemctl restart postgresql

  4. Connectar Utilitzant SSL:

    psql "host=your_host dbname=your_db user=your_user sslmode=require"

Xifrat de Dades en Repòs

El xifrat de dades en repòs assegura que les dades emmagatzemades estiguin protegides contra l'accés no autoritzat.

Xifrat Transparent de Dades (TDE)

PostgreSQL no suporta nativament TDE, però pots utilitzar eines de tercers o xifrat a nivell de sistema de fitxers.

Utilitzant pgcrypto per al Xifrat a Nivell de Columna

L'extensió pgcrypto proporciona funcions per xifrar i desxifrar dades a nivell de columna.

  1. Instal·lar pgcrypto:

    CREATE EXTENSION pgcrypto;

  2. Xifrant Dades:

    INSERT INTO users (username, password)
VALUES ('user1', crypt('my_password', gen_salt('bf')));

  3. Desxifrant Dades:

    SELECT username
FROM users
WHERE password = crypt('my_password', password);

Exemple: Xifrant Dades Sensibles d'Usuaris

-- Crear una taula amb columnes xifrades
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username TEXT NOT NULL,
    password TEXT NOT NULL
);

-- Inserir dades amb xifrat
INSERT INTO users (username, password)
VALUES ('user1', crypt('my_password', gen_salt('bf')));

-- Verificar la contrasenya
SELECT username
FROM users
WHERE username = 'user1' AND password = crypt('my_password', password);

Conclusió

El xifrat de dades és essencial per assegurar la informació sensible en bases de dades PostgreSQL. Xifrant dades en trànsit utilitzant SSL/TLS i xifrant dades en repòs utilitzant eines com pgcrypto, pots millorar significativament la seguretat de les teves bases de dades PostgreSQL. Comprendre i implementar aquestes tècniques de xifrat ajudarà a protegir les teves dades contra l'accés no autoritzat i possibles violacions.
