Introducció a la cerca de text complet

La cerca de text complet (FTS) a PostgreSQL et permet cercar de manera eficient documents o text dins d'una base de dades. A diferència del simple patró de coincidència amb LIKE, FTS pot gestionar consultes complexes i classificar els resultats segons la rellevància.

Conceptes clau

  • Tokenització: Dividir el text en tokens (paraules o termes).
  • Normalització: Convertir els tokens a una forma estàndard (per exemple, minúscules, arrelament).
  • Diccionaris: S'utilitzen per definir com es processen els tokens.
  • TSVector: Un tipus de dades que emmagatzema text preprocessat per a una cerca eficient.
  • TSQuery: Un tipus de dades que representa una consulta de cerca.

Configuració de la cerca de text complet

Creació d'una columna TSVector

Per utilitzar FTS, cal crear una columna que emmagatzemi el text preprocessat.

ALTER TABLE articles ADD COLUMN tsv tsvector;
UPDATE articles SET tsv = to_tsvector('english', title || ' ' || body);

Indexació de la columna TSVector

Crear un índex a la columna tsvector millora el rendiment de la cerca.

CREATE INDEX idx_fts ON articles USING gin(tsv);

Realització de cerques de text complet

Cerca bàsica

Pots realitzar una cerca bàsica de text complet utilitzant l'operador @@.

SELECT title, body
FROM articles
WHERE tsv @@ to_tsquery('english', 'postgresql & search');

Classificació dels resultats de la cerca

Per classificar els resultats de la cerca segons la rellevància, utilitza la funció ts_rank.

SELECT title, body, ts_rank(tsv, to_tsquery('english', 'postgresql & search')) AS rank
FROM articles
WHERE tsv @@ to_tsquery('english', 'postgresql & search')
ORDER BY rank DESC;

Tècniques avançades de cerca de text complet

Ús de múltiples diccionaris

Pots utilitzar múltiples diccionaris per gestionar diferents idiomes o vocabularis especialitzats.

CREATE TEXT SEARCH CONFIGURATION my_config (COPY = pg_catalog.english);
ALTER TEXT SEARCH CONFIGURATION my_config
    ALTER MAPPING FOR hword, hword_part, word
    WITH simple, english_stem;

Destacar termes de cerca

Destacar els termes de cerca en els resultats pot facilitar veure per què s'ha coincidit un document.

SELECT title, ts_headline('english', body, to_tsquery('english', 'postgresql & search')) AS highlighted_body
FROM articles
WHERE tsv @@ to_tsquery('english', 'postgresql & search');

Conclusió

La cerca de text complet a PostgreSQL és una característica potent que permet cerques de text eficients i complexes. Comprenent i utilitzant la tokenització, la normalització i els tipus de dades tsvector i tsquery, pots crear funcionalitats de cerca robustes a les teves aplicacions. Recorda indexar les teves columnes tsvector i considera utilitzar tècniques avançades com múltiples diccionaris i destacat de termes per millorar les teves capacitats de cerca.
