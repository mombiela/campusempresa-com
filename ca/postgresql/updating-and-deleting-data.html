En aquesta secció, cobrirem com actualitzar i eliminar dades a PostgreSQL. Aquestes operacions són fonamentals per mantenir i gestionar la teva base de dades de manera efectiva.

Actualització de Dades

L'actualització de dades a PostgreSQL es fa mitjançant la instrucció UPDATE. Aquesta instrucció et permet modificar registres existents en una taula.

Sintaxi Bàsica

La sintaxi bàsica per a la instrucció UPDATE és la següent:

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
  • table_name: El nom de la taula on vols actualitzar dades.
  • SET column1 = value1, column2 = value2, ...: Especifica les columnes i els seus nous valors.
  • WHERE condition: Especifica quines files s'han d'actualitzar. Si s'ometeix, totes les files de la taula s'actualitzaran.

Exemple

Considerem una taula employees amb la següent estructura:

| id | name | salary | |-----|----------|--------| | 1 | Alice | 50000 | | 2 | Bob | 60000 | | 3 | Charlie | 55000 |

Per actualitzar el salari d'en Bob a 65000, utilitzaries la següent consulta:

UPDATE employees
SET salary = 65000
WHERE name = 'Bob';

Actualització de Múltiples Columnes

Pots actualitzar múltiples columnes en una sola instrucció UPDATE. Per exemple, per actualitzar tant el nom com el salari de l'empleat amb id 3:

UPDATE employees
SET name = 'Charles', salary = 58000
WHERE id = 3;

Ús de Subconsultes en Actualitzacions

També pots utilitzar subconsultes per actualitzar dades basant-te en valors d'altres taules. Per exemple, si tens una altra taula bonuses:

| employee_id | bonus | |-------------|-------| | 1 | 5000 | | 2 | 7000 | | 3 | 6000 |

Per actualitzar el salary a la taula employees afegint el corresponent bonus de la taula bonuses:

UPDATE employees
SET salary = salary + (SELECT bonus FROM bonuses WHERE bonuses.employee_id = employees.id);

Eliminació de Dades

L'eliminació de dades a PostgreSQL es fa mitjançant la instrucció DELETE. Aquesta instrucció et permet eliminar registres existents d'una taula.

Sintaxi Bàsica

La sintaxi bàsica per a la instrucció DELETE és la següent:

DELETE FROM table_name
WHERE condition;
  • table_name: El nom de la taula de la qual vols eliminar dades.
  • WHERE condition: Especifica quines files s'han d'eliminar. Si s'ometeix, totes les files de la taula s'eliminaran.

Exemple

Per eliminar l'empleat anomenat 'Charlie' de la taula employees:

DELETE FROM employees
WHERE name = 'Charlie';

Eliminació de Totes les Files

Per eliminar totes les files d'una taula, pots ometre la clàusula WHERE:

DELETE FROM employees;

Ús de Subconsultes en Eliminacions

També pots utilitzar subconsultes per eliminar dades basant-te en valors d'altres taules. Per exemple, per eliminar empleats que tenen un bonus superior a 6000:

DELETE FROM employees
WHERE id IN (SELECT employee_id FROM bonuses WHERE bonus > 6000);

Conclusió

L'actualització i eliminació de dades són operacions essencials per gestionar una base de dades PostgreSQL. La instrucció UPDATE et permet modificar registres existents, mentre que la instrucció DELETE et permet eliminar registres. Ambdues instruccions es poden utilitzar amb condicions i subconsultes per realitzar operacions més complexes. Entendre aquestes comandes t'ajudarà a mantenir la integritat i l'exactitud de la teva base de dades.
