Les funcions de cadena són eines essencials en PostgreSQL per manipular i consultar dades de text. Aquesta secció cobrirà diverses funcions de cadena des de nivells bàsics fins a avançats, proporcionant explicacions clares, exemples i exercicis.

Funcions bàsiques de cadena

`LENGTH()`

  • Descripció: Retorna el nombre de caràcters en una cadena.
  • Sintaxi: LENGTH(string)
SELECT LENGTH('PostgreSQL'); -- Retorna 10

`UPPER()` i `LOWER()`

  • Descripció: Converteix una cadena a majúscules o minúscules.
  • Sintaxi: UPPER(string), LOWER(string)
SELECT UPPER('postgresql'); -- Retorna 'POSTGRESQL'
SELECT LOWER('PostgreSQL'); -- Retorna 'postgresql'

`TRIM()`

  • Descripció: Elimina espais en blanc al principi i al final d'una cadena.
  • Sintaxi: TRIM(string)
SELECT TRIM('  PostgreSQL  '); -- Retorna 'PostgreSQL'

`SUBSTRING()`

  • Descripció: Extreu una subcadena d'una cadena.
  • Sintaxi: SUBSTRING(string FROM start FOR length)
SELECT SUBSTRING('PostgreSQL' FROM 1 FOR 4); -- Retorna 'Post'

Funcions intermèdies de cadena

`POSITION()`

  • Descripció: Troba la posició d'una subcadena dins d'una cadena.
  • Sintaxi: POSITION(substring IN string)
SELECT POSITION('SQL' IN 'PostgreSQL'); -- Retorna 8

`CONCAT()`

  • Descripció: Concatena dues o més cadenes.
  • Sintaxi: CONCAT(string1, string2, ...)
SELECT CONCAT('Post', 'gre', 'SQL'); -- Retorna 'PostgreSQL'

`REPLACE()`

  • Descripció: Reemplaça totes les ocurrències d'una subcadena dins d'una cadena amb una altra subcadena.
  • Sintaxi: REPLACE(string, old_substring, new_substring)
SELECT REPLACE('PostgreSQL', 'Post', 'My'); -- Retorna 'MygreSQL'

`LPAD()` i `RPAD()`

  • Descripció: Omple una cadena a l'esquerra o a la dreta fins a una longitud especificada.
  • Sintaxi: LPAD(string, length, pad_string), RPAD(string, length, pad_string)
SELECT LPAD('SQL', 5, '0'); -- Retorna '00SQL'
SELECT RPAD('SQL', 5, '0'); -- Retorna 'SQL00'

Funcions avançades de cadena

`REGEXP_MATCHES()`

  • Descripció: Retorna totes les coincidències d'un patró d'expressió regular en una cadena.
  • Sintaxi: REGEXP_MATCHES(string, pattern)
SELECT REGEXP_MATCHES('abc123xyz', '[0-9]+'); -- Retorna {123}

`SPLIT_PART()`

  • Descripció: Divideix una cadena en un delimitador especificat i retorna la n-èssima subcadena.
  • Sintaxi: SPLIT_PART(string, delimiter, field)
SELECT SPLIT_PART('PostgreSQL is awesome', ' ', 2); -- Retorna 'is'

`TO_CHAR()`

  • Descripció: Converteix diversos tipus de dades a una cadena formatada segons un format especificat.
  • Sintaxi: TO_CHAR(value, format)
SELECT TO_CHAR(12345, '99999'); -- Retorna '12345'
SELECT TO_CHAR(current_timestamp, 'YYYY-MM-DD'); -- Retorna la data actual en format 'YYYY-MM-DD'

Exercicis

  1. Exercici bàsic: Escriu una consulta per convertir la cadena 'hello world' a majúscules.

    SELECT UPPER('hello world');

  2. Exercici intermig: Escriu una consulta per reemplaçar 'SQL' amb 'Database' en la cadena 'PostgreSQL'.

    SELECT REPLACE('PostgreSQL', 'SQL', 'Database');

  3. Exercici avançat: Escriu una consulta per extreure l'any de la data actual utilitzant TO_CHAR().

    SELECT TO_CHAR(current_date, 'YYYY');

Conclusió

Les funcions de cadena en PostgreSQL són eines poderoses per a la manipulació i consulta de text. Des de funcions bàsiques com LENGTH() i UPPER() fins a funcions avançades com REGEXP_MATCHES() i TO_CHAR(), dominar aquestes funcions millorarà la teva capacitat per gestionar i analitzar dades de text de manera efectiva. Practica els exercicis proporcionats per consolidar la teva comprensió i aplicar aquestes funcions en escenaris del món real.
