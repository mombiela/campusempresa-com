Introducció

L'Entrada i Sortida (I/O) estàndard són conceptes fonamentals en la programació en C. Permeten que els programes interactuïn amb els usuaris i altres programes llegint entrades i escrivint sortides. Aquesta secció cobrirà els conceptes bàsics de l'I/O estàndard, incloent funcions com printf, scanf, getchar i putchar.

Sortida Bàsica amb `printf`

La funció printf s'utilitza per a mostrar text a la consola. Forma part de la biblioteca d'I/O estàndard, que s'inclou afegint #include <stdio.h> al començament del teu programa.

Sintaxi

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
  • #include <stdio.h>: Inclou la biblioteca d'I/O estàndard.
  • printf("Hello, World!\n");: Imprimeix "Hello, World!" seguit d'un caràcter de nova línia.

Especificadors de Format

printf utilitza especificadors de format per a imprimir diferents tipus de dades:

  • %d: Enter
  • %f: Número de punt flotant
  • %c: Caràcter
  • %s: Cadena

Exemple

#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char initial = 'A';
    char name[] = "Alice";

    printf("Name: %s\n", name);
    printf("Initial: %c\n", initial);
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);

    return 0;
}

Entrada Bàsica amb `scanf`

La funció scanf s'utilitza per a llegir entrada formatada des de l'entrada estàndard (normalment el teclat).

Sintaxi

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Your age is %d\n", age);
    return 0;
}
  • scanf("%d", &age);: Llegeix un enter de l'usuari i l'emmagatzema a la variable age.

Especificadors de Format

scanf utilitza els mateixos especificadors de format que printf per a llegir diferents tipus de dades:

  • %d: Enter
  • %f: Número de punt flotant
  • %c: Caràcter
  • %s: Cadena

Exemple

#include <stdio.h>

int main() {
    int age;
    float height;
    char initial;
    char name[50];

    printf("Enter your name: ");
    scanf("%s", name);
    printf("Enter your initial: ");
    scanf(" %c", &initial); // Nota l'espai abans de %c per consumir qualsevol caràcter de nova línia restant
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Enter your height: ");
    scanf("%f", &height);

    printf("Name: %s\n", name);
    printf("Initial: %c\n", initial);
    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);

    return 0;
}

Entrada i Sortida de Caràcters

Per a l'entrada i sortida de caràcters individuals, s'utilitzen les funcions getchar i putchar.

Exemple de `getchar`

#include <stdio.h>

int main() {
    char ch;
    printf("Enter a character: ");
    ch = getchar();
    printf("You entered: %c\n", ch);
    return 0;
}

Exemple de `putchar`

#include <stdio.h>

int main() {
    char ch = 'A';
    printf("The character is: ");
    putchar(ch);
    putchar('\n');
    return 0;
}

Funcions Avançades d'I/O

Per a operacions d'I/O més avançades, C proporciona funcions com fgets i fputs per a l'entrada i sortida de cadenes.

Exemple de `fgets`

#include <stdio.h>

int main() {
    char str[50];
    printf("Enter a string: ");
    fgets(str, sizeof(str), stdin);
    printf("You entered: %s", str);
    return 0;
}

Exemple de `fputs`

#include <stdio.h>

int main() {
    char str[] = "Hello, World!";
    printf("The string is: ");
    fputs(str, stdout);
    putchar('\n');
    return 0;
}

Resum

  • printf: S'utilitza per a la sortida formatada.
  • scanf: S'utilitza per a l'entrada formatada.
  • getchar: Llegeix un caràcter individual de l'entrada estàndard.
  • putchar: Escriu un caràcter individual a la sortida estàndard.
  • fgets: Llegeix una cadena de l'entrada estàndard.
  • fputs: Escriu una cadena a la sortida estàndard.

Entendre aquestes funcions bàsiques d'I/O és crucial per a interactuar amb usuaris i altres programes. Practica utilitzant aquestes funcions per a convertir-te en un expert en la gestió de l'entrada i sortida en C.
