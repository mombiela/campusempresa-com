Què són els Punter?

Els punters són un concepte fonamental en la programació en C que et permeten accedir i manipular directament la memòria. Entendre els punters és crucial per a una programació en C eficient i efectiva.

Conceptes Clau

  • Definició de Punter: Un punter és una variable que emmagatzema l'adreça de memòria d'una altra variable.
  • Declaració de Punter: Declarar un punter implica especificar el tipus de dades al qual apunta, seguit d'un asterisc (*) i el nom del punter.
  • Inicialització de Punter: Els punters es poden inicialitzar amb l'adreça d'una variable utilitzant l'operador d'adreça (&).
  • Desreferenciació: Accedir al valor emmagatzemat a l'adreça de memòria a la qual apunta un punter utilitzant l'operador de desreferenciació (*).

Declarar i Inicialitzar Punter

Per declarar un punter, especifiques el tipus de dades al qual apuntarà, seguit d'un asterisc (*) i el nom del punter.

int *ptr; // Declara un punter a un enter

Per inicialitzar un punter, li assignes l'adreça d'una variable utilitzant l'operador d'adreça (&).

int var = 10;
int *ptr = &var; // ptr ara conté l'adreça de var

Desreferenciació de Punter

Desreferenciar un punter significa accedir al valor emmagatzemat a l'adreça de memòria a la qual apunta el punter. Això es fa utilitzant l'operador de desreferenciació (*).

int var = 10;
int *ptr = &var;
printf("Valor de var: %d\n", *ptr); // Mostra: Valor de var: 10

Aritmètica de Punter

Els punters es poden incrementar o decrementar per apuntar a la següent o anterior ubicació de memòria del mateix tipus de dades.

int arr[3] = {10, 20, 30};
int *ptr = arr; // Apunta al primer element de l'array

ptr++; // Ara apunta al segon element
printf("Segon element: %d\n", *ptr); // Mostra: Segon element: 20

Operacions Comunes amb Punter

  • Assignar Adreces: Utilitzant l'operador d'adreça (&).
  • Desreferenciació: Utilitzant l'operador de desreferenciació (*).
  • Aritmètica de Punter: Incrementar o decrementar punters.
  • Punter Null: Punter que s'assigna un valor NULL per indicar que no apunten a cap ubicació de memòria vàlida.

Exemple de Codi

Aquí tens un exemple complet que demostra la declaració, inicialització i desreferenciació de punter:

#include <stdio.h>

int main() {
    int var = 10;
    int *ptr = &var; // Punter a var

    printf("Adreça de var: %p\n", (void*)&var);
    printf("Adreça emmagatzemada a ptr: %p\n", (void*)ptr);
    printf("Valor apuntat per ptr: %d\n", *ptr);

    return 0;
}

Explicació

  • int var = 10; declara una variable enter var i la inicialitza a 10.
  • int *ptr = &var; declara un punter ptr i l'inicialitza amb l'adreça de var.
  • printf("Adreça de var: %p\n", (void*)&var); mostra l'adreça de var.
  • printf("Adreça emmagatzemada a ptr: %p\n", (void*)ptr); mostra l'adreça emmagatzemada a ptr.
  • printf("Valor apuntat per ptr: %d\n", *ptr); mostra el valor emmagatzemat a l'adreça a la qual apunta ptr, que és el valor de var.

Conclusió

Els punters són una característica poderosa en C que proporcionen accés directe a la memòria i permeten una manipulació eficient de les dades. Entenent com declarar, inicialitzar i utilitzar punters, pots escriure programes en C més eficients i flexibles. La pràctica amb punters és essencial, ja que s'utilitzen àmpliament en diversos escenaris de programació, incloent l'assignació dinàmica de memòria, estructures de dades i arguments de funcions.
