Introducció a les Llistes Enllaçades

Les llistes enllaçades són una estructura de dades fonamental en la informàtica, que proporcionen una manera d'emmagatzemar una col·lecció d'elements. A diferència dels arrays, les llistes enllaçades no requereixen ubicacions de memòria contigües, cosa que les fa flexibles i eficients per a certes operacions.

Conceptes Clau

  • Node: La unitat bàsica d'una llista enllaçada, que conté dades i una referència (o punter) al següent node.
  • Cap: El primer node d'una llista enllaçada.
  • Cua: L'últim node d'una llista enllaçada, que apunta a NULL.
  • NULL: Un valor especial que indica el final de la llista.

Tipus de Llistes Enllaçades

Llista Enllaçada Simple

Una llista enllaçada simple és el tipus més senzill de llista enllaçada, on cada node apunta al següent node en la seqüència.

Llista Enllaçada Doble

Una llista enllaçada doble permet la travessia en ambdues direccions, ja que cada node conté punters tant al següent com al node anterior.

Llista Enllaçada Circular

En una llista enllaçada circular, l'últim node apunta de nou al primer node, formant una estructura circular.

Implementació d'una Llista Enllaçada Simple en C

Definició de l'Estructura del Node

#include <stdio.h>
#include <stdlib.h>

// Defineix l'estructura del node
struct Node {
    int data;
    struct Node* next;
};
  • data: Emmagatzema el valor del node.
  • next: Punter al següent node de la llista.

Creació d'un Nou Node

struct Node* createNode(int data) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = data;
    newNode->next = NULL;
    return newNode;
}
  • malloc: Assigna memòria per al nou node.
  • data: Inicialitza les dades del node.
  • next: Estableix el punter següent a NULL.

Inserció d'un Node al Començament

void insertAtBeginning(struct Node** head, int data) {
    struct Node* newNode = createNode(data);
    newNode->next = *head;
    *head = newNode;
}
  • head: Doble punter al node cap per permetre la modificació.
  • newNode->next: Apunta al cap actual.
  • *head: Actualitza el cap al nou node.

Travessia de la Llista

void traverseList(struct Node* head) {
    struct Node* current = head;
    while (current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
}
  • current: Punter utilitzat per travessar la llista.
  • printf: Imprimeix les dades de cada node seguides d'una fletxa.

Eliminació d'un Node

void deleteNode(struct Node** head, int key) {
    struct Node* temp = *head;
    struct Node* prev = NULL;

    // Si el node cap mateix conté la clau
    if (temp != NULL && temp->data == key) {
        *head = temp->next;
        free(temp);
        return;
    }

    // Cerca la clau
    while (temp != NULL && temp->data != key) {
        prev = temp;
        temp = temp->next;
    }

    // Si la clau no estava present a la llista
    if (temp == NULL) return;

    // Desenllaça el node i allibera memòria
    prev->next = temp->next;
    free(temp);
}
  • temp: Punter temporal per travessar la llista.
  • prev: Punter al node anterior per actualitzar l'enllaç.
  • free: Desassigna la memòria del node eliminat.

Operacions Avançades en Llistes Enllaçades

Inversió d'una Llista Enllaçada

void reverseList(struct Node** head) {
    struct Node* prev = NULL;
    struct Node* current = *head;
    struct Node* next = NULL;

    while (current != NULL) {
        next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    *head = prev;
}
  • prev: Fa el seguiment del node anterior.
  • current: Fa el seguiment del node actual.
  • next: Emmagatzema temporalment el següent node.

Detecció d'un Cicle en una Llista Enllaçada

int detectCycle(struct Node* head) {
    struct Node* slow = head;
    struct Node* fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast) {
            return 1; // Cicle detectat
        }
    }
    return 0; // No hi ha cicle
}
  • slow: Es mou un pas a la vegada.
  • fast: Es mou dos passos a la vegada.
  • Detecció de cicle: Si slow i fast es troben, existeix un cicle.

Conclusió

Les llistes enllaçades són una estructura de dades versàtil que es pot utilitzar per gestionar col·leccions de dades de manera eficient. Entendre les operacions bàsiques, com la inserció, eliminació, travessia, i operacions avançades com la inversió i la detecció de cicles, és crucial per aprofitar les llistes enllaçades en diverses aplicacions. En dominar aquests conceptes, els programadors poden millorar les seves habilitats de resolució de problemes i optimitzar el seu codi per obtenir un millor rendiment.
