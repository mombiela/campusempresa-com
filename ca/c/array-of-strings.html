Array de Cadenes en C
Introducció als Punteres ►

Introducció

En la programació en C, una matriu de cadenes és essencialment una matriu bidimensional de caràcters. Cada fila en aquesta matriu representa una cadena, i cada columna representa un caràcter en aquesta cadena. Aquest concepte és fonamental per gestionar múltiples cadenes de manera eficient en C.

Conceptes Clau

  • Cadena: Una seqüència de caràcters terminada per un caràcter nul (\0).
  • Matriu de Cadenes: Una matriu bidimensional on cada element és una cadena.

Declarar una Matriu de Cadenes

Per declarar una matriu de cadenes en C, cal especificar el nombre de cadenes i la longitud màxima de cada cadena.

Exemple

char fruits[3][10] = {
    "Apple",
    "Banana",
    "Cherry"
};

En aquest exemple:

  • fruits és una matriu de 3 cadenes.
  • Cada cadena pot tenir fins a 9 caràcters (més el terminador nul).

Accedir als Elements

Es pot accedir a cadenes i caràcters individuals utilitzant la indexació de matrius.

Exemple

#include <stdio.h>

int main() {
    char fruits[3][10] = {
        "Apple",
        "Banana",
        "Cherry"
    };

    // Accedint a la primera cadena
    printf("Primer fruita: %s\n", fruits[0]);

    // Accedint al primer caràcter de la segona cadena
    printf("Primer caràcter de la segona fruita: %c\n", fruits[1][0]);

    return 0;
}

Modificar Cadenes

Es poden modificar les cadenes en la matriu accedint directament als elements.

Exemple

#include <stdio.h>
#include <string.h>

int main() {
    char fruits[3][10] = {
        "Apple",
        "Banana",
        "Cherry"
    };

    // Modificant la primera cadena
    strcpy(fruits[0], "Apricot");

    printf("Primera fruita modificada: %s\n", fruits[0]);

    return 0;
}

Operacions Comunes

Inicialitzar una Matriu de Cadenes

Es pot inicialitzar una matriu de cadenes en el moment de la declaració.

Exemple

char colors[4][10] = {
    "Red",
    "Green",
    "Blue",
    "Yellow"
};

Iterar a Través d'una Matriu de Cadenes

Es poden utilitzar bucles per iterar a través de cada cadena en la matriu.

Exemple

#include <stdio.h>

int main() {
    char colors[4][10] = {
        "Red",
        "Green",
        "Blue",
        "Yellow"
    };

    for (int i = 0; i < 4; i++) {
        printf("Color %d: %s\n", i + 1, colors[i]);
    }

    return 0;
}

Temes Avançats

Assignació Dinàmica

Per a més flexibilitat, es poden utilitzar punters i assignació dinàmica de memòria per gestionar una matriu de cadenes.

Exemple

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main() {
    int n = 3;
    char **fruits = (char **)malloc(n * sizeof(char *));
    
    for (int i = 0; i < n; i++) {
        fruits[i] = (char *)malloc(10 * sizeof(char));
    }

    strcpy(fruits[0], "Apple");
    strcpy(fruits[1], "Banana");
    strcpy(fruits[2], "Cherry");

    for (int i = 0; i < n; i++) {
        printf("Fruita %d: %s\n", i + 1, fruits[i]);
        free(fruits[i]);
    }

    free(fruits);

    return 0;
}

Resum

  • Una matriu de cadenes en C és una matriu bidimensional de caràcters.
  • Cada fila en la matriu representa una cadena, i cada columna representa un caràcter en aquesta cadena.
  • Es poden accedir i modificar les cadenes utilitzant la indexació de matrius.
  • L'assignació dinàmica proporciona més flexibilitat per gestionar una matriu de cadenes.

Entendre les matrius de cadenes és crucial per gestionar dades de text de manera eficient en C. Aquest coneixement és fonamental per a temes més avançats com la gestió de fitxers, l'anàlisi de dades i el processament d'entrada d'usuari.
