Introducció

Els punters a funcions són una característica poderosa en C que permet emmagatzemar l'adreça d'una funció en una variable punter. Això permet crides de funcions dinàmiques, passar funcions com a arguments i crear arrays de funcions. Entendre els punters a funcions és crucial per a la programació avançada en C, especialment en escenaris com funcions de callback i implementació de taules de funcions.

Conceptes Clau

  • Declaració de Punter a Funció: Com declarar un punter a una funció.
  • Assignació de Funcions a Punters: Com assignar una funció a un punter a funció.
  • Cridar Funcions a través de Punters: Com cridar una funció utilitzant un punter a funció.
  • Passar Punters a Funció com a Arguments: Com passar punters a funció a altres funcions.
  • Arrays de Punters a Funció: Com crear i utilitzar arrays de punters a funció.

Declaració de Punter a Funció

Per declarar un punter a una funció, cal especificar el tipus de retorn de la funció i els tipus de paràmetres. La sintaxi és la següent:

return_type (*pointer_name)(parameter_types);

Exemple

// Declarar un punter a una funció que pren dos enters i retorna un enter
int (*operation)(int, int);

Assignació de Funcions a Punters

Pots assignar l'adreça d'una funció a un punter a funció utilitzant el nom de la funció sense parèntesis.

Exemple

// Definir una funció que coincideixi amb la signatura del punter
int add(int a, int b) {
    return a + b;
}

// Assignar la funció al punter
operation = add;

Cridar Funcions a través de Punters

Un cop un punter a funció està assignat, pots cridar la funció utilitzant el punter.

Exemple

// Cridar la funció utilitzant el punter
int result = operation(5, 3); // result serà 8

Passar Punters a Funció com a Arguments

Pots passar punters a funció a altres funcions, permetent codi flexible i reutilitzable.

Exemple

// Funció que pren un punter a funció com a argument
void executeOperation(int (*op)(int, int), int x, int y) {
    printf("Result: %d\n", op(x, y));
}

// Ús
executeOperation(add, 5, 3); // Sortida: Result: 8

Arrays de Punters a Funció

Els arrays de punters a funció es poden utilitzar per emmagatzemar múltiples funcions i cridar-les dinàmicament.

Exemple

// Definir múltiples funcions
int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

// Crear un array de punters a funció
int (*operations[])(int, int) = {add, subtract, multiply};

// Cridar funcions des de l'array
int sum = operations[0](5, 3);       // sum serà 8
int difference = operations[1](5, 3); // difference serà 2
int product = operations[2](5, 3);    // product serà 15

Conclusió

Els punters a funcions són una característica versàtil en C que permet crides de funcions dinàmiques i flexibles. Entenent com declarar, assignar i utilitzar punters a funció, pots escriure codi més modular i reutilitzable. Tant si passes funcions com a arguments, crees arrays de funcions o implementes mecanismes de callback, els punters a funcions són una eina essencial en la programació avançada en C.
