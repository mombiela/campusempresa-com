Introducció als Bucles

Els bucles són constructes fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. En C, hi ha tres tipus principals de bucles:

  • for loop
  • while loop
  • do-while loop

Cada tipus de bucle serveix per a diferents propòsits i es pot utilitzar segons els requisits específics del problema que estàs resolent.

El Bucle `for`

El bucle for s'utilitza comunament quan el nombre d'iteracions es coneix per endavant. Consta de tres parts: inicialització, condició i increment/decrement.

Sintaxi

for (initialization; condition; increment/decrement) {
    // Codi a executar
}

Exemple

#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {
        printf("Iteració %d\n", i);
    }
    return 0;
}

En aquest exemple:

  • Inicialització: int i = 0 estableix el punt de partida.
  • Condició: i < 5 assegura que el bucle s'executi mentre i sigui menor que 5.
  • Increment: i++ augmenta i en 1 després de cada iteració.

Conceptes Clau

  • Inicialització: Estableix el punt de partida del bucle.
  • Condició: Determina quant de temps s'executarà el bucle.
  • Increment/Decrement: Actualitza la variable del bucle després de cada iteració.

El Bucle `while`

El bucle while s'utilitza quan el nombre d'iteracions no es coneix per endavant. Continua executant-se mentre la condició especificada sigui certa.

Sintaxi

while (condition) {
    // Codi a executar
}

Exemple

#include <stdio.h>

int main() {
    int i = 0;
    while (i < 5) {
        printf("Iteració %d\n", i);
        i++;
    }
    return 0;
}

En aquest exemple:

  • El bucle comença amb i = 0.
  • La condició i < 5 es comprova abans de cada iteració.
  • i++ incrementa i després de cada iteració.

Conceptes Clau

  • Condició: El bucle s'executa mentre aquesta condició sigui certa.
  • Inicialització i Increment/Decrement: Aquests s'han de gestionar dins del cos del bucle.

El Bucle `do-while`

El bucle do-while és similar al bucle while, però garanteix que el cos del bucle s'executi almenys una vegada.

Sintaxi

do {
    // Codi a executar
} while (condition);

Exemple

#include <stdio.h>

int main() {
    int i = 0;
    do {
        printf("Iteració %d\n", i);
        i++;
    } while (i < 5);
    return 0;
}

En aquest exemple:

  • El cos del bucle s'executa primer.
  • La condició i < 5 es comprova després de cada iteració.

Conceptes Clau

  • Execució Garantida: El cos del bucle s'executa almenys una vegada.
  • Condició: Es comprova després que el cos del bucle s'hagi executat.

Comparació de les Estructures de Bucle

| Tipus de Bucle | Inicialització | Comprovació de la Condició | Increment/Decrement | Execució Garantida | |----------------|----------------|---------------------------|---------------------|--------------------| | for | Al començament | Abans de cada iteració | Al final de cada iteració | No | | while | Abans del bucle | Abans de cada iteració | Dins del cos del bucle | No | | do-while | Abans del bucle | Després de cada iteració | Dins del cos del bucle | Sí |

Bucles Aniuats

Els bucles es poden aniuar dins d'altres bucles. Això és útil per a estructures de dades multidimensionals com matrius.

Exemple

#include <stdio.h>

int main() {
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            printf("i = %d, j = %d\n", i, j);
        }
    }
    return 0;
}

En aquest exemple:

  • El bucle exterior s'executa 3 vegades.
  • Per a cada iteració del bucle exterior, el bucle interior s'executa 3 vegades.

Errors Comuns

  • Bucles Infinits: Assegura't que la condició del bucle eventualment esdevingui falsa.
  • Errors de Fora per Un: Tingues cura amb els límits del bucle per evitar executar el bucle una vegada de més o de menys.
  • Inicialització i Increment/Decrement Correctes: Assegura't que aquests estiguin correctament establerts per evitar errors lògics.

Conclusió

Els bucles són constructes poderosos que et permeten executar codi múltiples vegades de manera eficient. Entendre les diferències entre els bucles for, while i do-while, així com com utilitzar bucles aniuats, és crucial per a una programació efectiva en C. Practica escrivint diferents tipus de bucles per esdevenir proficient en el seu ús.
