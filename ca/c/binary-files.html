Els fitxers binari són un aspecte crucial de la gestió de fitxers en la programació en C. A diferència dels fitxers de text, que emmagatzemen dades en un format llegible per a humans, els fitxers binari emmagatzemen dades en un format que és directament llegible per l'ordinador. Això permet un emmagatzematge més eficient i temps d'accés més ràpids.

Conceptes Clau

  • Fitxers Binari vs Text: Entendre les diferències entre fitxers binari i de text.
  • Operacions amb Fitxers: Aprendre a obrir, llegir, escriure i tancar fitxers binari.
  • Punter de Fitxer: Entendre el paper dels punters de fitxer en les operacions amb fitxers binari.
  • Estructures i Fitxers Binari: Aprendre a llegir i escriure tipus de dades complexos utilitzant estructures.

Fitxers Binari vs Text

Els fitxers binari emmagatzemen dades en el mateix format en què es representen a la memòria, mentre que els fitxers de text emmagatzemen dades com una seqüència de caràcters.

| Característica | Fitxers de Text | Fitxers Binari | |------------------|--------------------------------|---------------------------------| | Format d'emmagatzematge | Caràcters llegibles per a humans | Format binari llegible per màquina | | Mida | Més gran degut a la codificació de caràcters | Més petit, més compacte | | Llegibilitat | Fàcilment llegible per a humans | No llegible per a humans | | Rendiment | Més lent degut a la codificació/descodificació | Més ràpid degut a la representació directa de la memòria |

Operacions amb Fitxers

Obrir un Fitxer Binari

Per obrir un fitxer binari, utilitza la funció fopen amb el mode "rb" per llegir i "wb" per escriure.

FILE *file;
file = fopen("example.bin", "wb");
if (file == NULL) {
    printf("Error obrint el fitxer.\n");
    return 1;
}

Escriure en un Fitxer Binari

Utilitza la funció fwrite per escriure dades en un fitxer binari. La funció fwrite pren quatre arguments: un punter a les dades que s'han d'escriure, la mida de cada element, el nombre d'elements, i el punter de fitxer.

int data = 12345;
fwrite(&data, sizeof(int), 1, file);

Llegir d'un Fitxer Binari

Utilitza la funció fread per llegir dades d'un fitxer binari. La funció fread també pren quatre arguments: un punter al buffer on es guardaran les dades, la mida de cada element, el nombre d'elements, i el punter de fitxer.

int buffer;
fread(&buffer, sizeof(int), 1, file);
printf("Dades llegides del fitxer: %d\n", buffer);

Tancar un Fitxer Binari

Sempre tanca un fitxer després d'haver acabat amb ell utilitzant la funció fclose.

fclose(file);

Punter de Fitxer

Els punters de fitxer s'utilitzen per mantenir el seguiment de la posició actual en el fitxer. Funcions com fseek i ftell s'utilitzen per manipular i consultar el punter de fitxer.

  • fseek: Mou el punter de fitxer a una ubicació específica.
  • ftell: Obté la posició actual del punter de fitxer.
fseek(file, 0, SEEK_SET); // Mou al començament del fitxer
long position = ftell(file); // Obté la posició actual
printf("Posició actual: %ld\n", position);

Estructures i Fitxers Binari

Els fitxers binari són particularment útils per emmagatzemar tipus de dades complexos com les estructures.

Escriure Estructures en un Fitxer Binari

struct Person {
    char name[50];
    int age;
};

struct Person person = {"John Doe", 30};
fwrite(&person, sizeof(struct Person), 1, file);

Llegir Estructures d'un Fitxer Binari

struct Person person;
fread(&person, sizeof(struct Person), 1, file);
printf("Nom: %s, Edat: %d\n", person.name, person.age);

Conclusió

Els fitxers binari ofereixen una manera més eficient d'emmagatzemar i recuperar dades en comparació amb els fitxers de text. Entenent com obrir, llegir, escriure i tancar fitxers binari, així com treballar amb punters de fitxer i estructures, pots gestionar eficaçment les dades binàries en els teus programes en C. Aquest coneixement és essencial per a tasques que requereixen alt rendiment i emmagatzematge eficient de dades.
