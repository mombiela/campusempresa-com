Què és la Programació?

La programació és el procés de crear un conjunt d'instruccions que indiquen a un ordinador com realitzar una tasca. Aquestes instruccions estan escrites en un llenguatge de programació. C és un dels llenguatges de programació més utilitzats a causa de la seva eficiència i control sobre els recursos del sistema.

Per què Aprendre C?

  • Base per a Altres Llenguatges: Molts llenguatges moderns com C++, Java i Python tenen les seves arrels en C.
  • Programació a Nivell de Sistema: C s'utilitza per a la programació de sistemes, incloent sistemes operatius i sistemes encastats.
  • Rendiment: C proporciona accés de baix nivell a la memòria i als processos del sistema, fent-lo altament eficient.

Conceptes Bàsics en C

Variables i Tipus de Dades

Les variables s'utilitzen per emmagatzemar dades que poden ser manipulades pel programa. Cada variable en C ha de ser declarada amb un tipus de dada específic.

  • int: Nombres enters
  • float: Nombres de punt flotant
  • char: Caràcters simples
#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char initial = 'A';

    printf("Edat: %d\n", age);
    printf("Alçada: %.1f\n", height);
    printf("Inicial: %c\n", initial);

    return 0;
}

Operadors

Els operadors són símbols que realitzen operacions sobre variables i valors.

  • Operadors Aritmètics: +, -, *, /, %
  • Operadors Relacionals: ==, !=, >, <, >=, <=
  • Operadors Lògics: &&, ||, !
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int sum = a + b;
    int isEqual = (a == b);

    printf("Suma: %d\n", sum);
    printf("Són iguals? %d\n", isEqual);

    return 0;
}

Estructures de Control

Les estructures de control s'utilitzen per controlar el flux d'execució en un programa.

  • if-else: Declaracions condicionals
  • for: Bucle amb un nombre conegut d'iteracions
  • while: Bucle amb un nombre desconegut d'iteracions
  • do-while: Bucle que s'executa almenys una vegada
#include <stdio.h>

int main() {
    int number = 10;

    // declaració if-else
    if (number > 0) {
        printf("El número és positiu\n");
    } else {
        printf("El número no és positiu\n");
    }

    // bucle for
    for (int i = 0; i < 5; i++) {
        printf("i: %d\n", i);
    }

    // bucle while
    int count = 0;
    while (count < 3) {
        printf("Compte: %d\n", count);
        count++;
    }

    // bucle do-while
    int j = 0;
    do {
        printf("j: %d\n", j);
        j++;
    } while (j < 2);

    return 0;
}

Funcions

Les funcions són blocs de codi que realitzen una tasca específica i poden ser reutilitzades.

  • Declaració: Especifica el nom de la funció, el tipus de retorn i els paràmetres.
  • Definició: Conté el codi real que s'executarà.
  • Crida: Executa la funció.
#include <stdio.h>

// Declaració de la funció
void greet();

// Funció principal
int main() {
    greet(); // Crida a la funció
    return 0;
}

// Definició de la funció
void greet() {
    printf("Hola, Món!\n");
}

Conclusió

En aquest tema introductori, hem cobert els conceptes bàsics de la programació en C, incloent variables, tipus de dades, operadors, estructures de control i funcions. Aquests conceptes fonamentals són els blocs de construcció per a temes més avançats en la programació en C. Entendre aquests conceptes bàsics proporcionarà una base sòlida per al teu viatge cap a tècniques de programació més complexes i poderoses.
