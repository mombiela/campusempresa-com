Introducció

Les funcions de temps i data són essencials en moltes tasques de programació, com ara registrar esdeveniments, mesurar intervals de temps i programar tasques. En C, la biblioteca <time.h> proporciona una varietat de funcions per treballar amb dates i temps.

Conceptes Clau

  • Temps Epoch: El nombre de segons que han transcorregut des de les 00:00:00 UTC de l'1 de gener de 1970.
  • time_t: Un tipus de dades utilitzat per representar el temps.
  • struct tm: Una estructura utilitzada per contenir una data i hora.

Funcions Bàsiques de Temps

Obtenir l'Hora Actual

Per obtenir l'hora actual, pots utilitzar la funció time(), que retorna l'hora actual com un valor time_t.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    current_time = time(NULL);
    printf("Current time: %ld\n", current_time);
    return 0;
}
  • time(NULL) retorna l'hora actual.
  • current_time conté el nombre de segons des de l'Epoch.

Convertir time_t a Format Llegible per Humans

Per convertir time_t a un format llegible per humans, utilitza la funció ctime().

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    current_time = time(NULL);
    printf("Current time: %s", ctime(&current_time));
    return 0;
}
  • ctime(&current_time) converteix time_t a una cadena que representa l'hora local.

Treballant amb struct tm

Descomposar time_t en Components

La funció localtime() converteix time_t a una struct tm que representa l'hora local.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    struct tm *local_time;
    
    current_time = time(NULL);
    local_time = localtime(&current_time);
    
    printf("Year: %d\n", local_time->tm_year + 1900);
    printf("Month: %d\n", local_time->tm_mon + 1);
    printf("Day: %d\n", local_time->tm_mday);
    printf("Hour: %d\n", local_time->tm_hour);
    printf("Minute: %d\n", local_time->tm_min);
    printf("Second: %d\n", local_time->tm_sec);
    
    return 0;
}
  • localtime(&current_time) retorna un punter a una struct tm.
  • tm_year és el nombre d'anys des de 1900.
  • tm_mon és el nombre de mesos des de gener (0-11).

Formatant Temps i Data

La funció strftime() formata el temps i la data segons un format especificat.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    struct tm *local_time;
    char buffer[80];
    
    current_time = time(NULL);
    local_time = localtime(&current_time);
    
    strftime(buffer, 80, "Today is %A, %B %d, %Y", local_time);
    printf("%s\n", buffer);
    
    return 0;
}
  • strftime(buffer, 80, "format", local_time) formata la data i l'hora.
  • Especificadors de format comuns:
    • %A: Nom complet del dia de la setmana.
    • %B: Nom complet del mes.
    • %d: Dia del mes.
    • %Y: Any amb segle.

Funcions Avançades de Temps

Mesurant Intervals de Temps

Per mesurar intervals de temps, utilitza la funció clock(), que retorna el temps de processador consumit pel programa.

#include <stdio.h>
#include <time.h>

int main() {
    clock_t start, end;
    double cpu_time_used;
    
    start = clock();
    // Codi a mesurar
    for (int i = 0; i < 1000000; i++);
    end = clock();
    
    cpu_time_used = ((double) (end - start)) / CLOCKS_PER_SEC;
    printf("Time used: %f seconds\n", cpu_time_used);
    
    return 0;
}
  • clock() retorna el nombre de ticks de rellotge des que el programa va començar.
  • CLOCKS_PER_SEC és el nombre de ticks de rellotge per segon.

Manipulant el Temps

La funció mktime() converteix una struct tm de nou a time_t.

#include <stdio.h>
#include <time.h>

int main() {
    struct tm time_info = {0};
    time_t future_time;
    
    time_info.tm_year = 2023 - 1900; // Any des de 1900
    time_info.tm_mon = 11 - 1; // Mes (0-11)
    time_info.tm_mday = 25; // Dia del mes
    
    future_time = mktime(&time_info);
    printf("Future time: %s", ctime(&future_time));
    
    return 0;
}
  • mktime(&time_info) converteix struct tm a time_t.

Resum

  • La biblioteca <time.h> proporciona funcions essencials per treballar amb temps i data en C.
  • time_t i struct tm són tipus de dades clau per manejar el temps.
  • Funcions com time(), ctime(), localtime(), i strftime() ajuden a obtenir i formatar l'hora actual.
  • Funcions avançades com clock() i mktime() són útils per mesurar intervals de temps i manipular el temps.

Entendre aquestes funcions et permetrà manejar el temps i la data de manera efectiva en els teus programes en C.
