Introducció

Les operacions amb fitxers són una part essencial de la programació en C. Permeten llegir i escriure fitxers, permetent la persistència i manipulació de dades. Aquesta secció cobrirà els conceptes bàsics de les operacions amb fitxers, incloent l'obertura, lectura, escriptura i tancament de fitxers.

Funcions de gestió de fitxers

C proporciona un conjunt de funcions de biblioteca estàndard per a la gestió de fitxers. Aquí tens les funcions clau que utilitzaràs:

  • fopen(): Obre un fitxer.
  • fclose(): Tanca un fitxer.
  • fread(): Llegeix dades d'un fitxer.
  • fwrite(): Escriu dades a un fitxer.
  • fgets(): Llegeix una cadena d'un fitxer.
  • fputs(): Escriu una cadena a un fitxer.
  • fprintf(): Escriu dades amb format a un fitxer.
  • fscanf(): Llegeix dades amb format d'un fitxer.

Obrir i tancar fitxers

Per realitzar qualsevol operació en un fitxer, primer has d'obrir-lo utilitzant la funció fopen(). Després de completar les operacions, hauries de tancar el fitxer utilitzant la funció fclose().

Exemple: Obrir i tancar un fitxer

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "r"); // Obre el fitxer per a lectura

    if (file == NULL) {
        printf("Error obrint el fitxer.\n");
        return 1;
    }

    // Realitza operacions amb el fitxer aquí

    fclose(file); // Tanca el fitxer
    return 0;
}

Llegir d'un fitxer

Hi ha múltiples maneres de llegir dades d'un fitxer en C. Les funcions més comunes són fread(), fgets() i fscanf().

Exemple: Llegir un fitxer utilitzant `fgets()`

#include <stdio.h>

int main() {
    FILE *file;
    char buffer[100];

    file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error obrint el fitxer.\n");
        return 1;
    }

    while (fgets(buffer, 100, file) != NULL) {
        printf("%s", buffer);
    }

    fclose(file);
    return 0;
}

Escriure a un fitxer

Per escriure dades a un fitxer, pots utilitzar funcions com fwrite(), fputs() i fprintf().

Exemple: Escriure a un fitxer utilitzant `fprintf()`

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w"); // Obre el fitxer per a escriptura

    if (file == NULL) {
        printf("Error obrint el fitxer.\n");
        return 1;
    }

    fprintf(file, "Hola, Món!\n");
    fprintf(file, "Escrivint a un fitxer en C.\n");

    fclose(file);
    return 0;
}

Modes de fitxer

Quan obres un fitxer, has d'especificar el mode en què vols obrir-lo. Aquí tens alguns modes de fitxer comuns:

| Mode | Descripció | |------|----------------------------------| | "r" | Obre per a lectura. | | "w" | Obre per a escriptura (crea el fitxer si no existeix, trunca si existeix). | | "a" | Obre per a afegir (crea el fitxer si no existeix). | | "r+" | Obre per a lectura i escriptura. | | "w+" | Obre per a lectura i escriptura (crea el fitxer si no existeix, trunca si existeix). | | "a+" | Obre per a lectura i afegir (crea el fitxer si no existeix). |

Gestió d'errors

És crucial gestionar els errors quan es realitzen operacions amb fitxers per assegurar que el teu programa pugui manejar situacions inesperades de manera elegant.

Exemple: Gestió d'errors en operacions amb fitxers

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "r");

    if (file == NULL) {
        perror("Error obrint el fitxer");
        return 1;
    }

    // Realitza operacions amb el fitxer aquí

    if (fclose(file) != 0) {
        perror("Error tancant el fitxer");
        return 1;
    }

    return 0;
}

Conclusió

Les operacions amb fitxers en C són fonamentals per gestionar la persistència i manipulació de dades. Entenent com obrir, llegir, escriure i tancar fitxers, pots gestionar efectivament les dades en els teus programes. Recorda sempre gestionar els errors de manera elegant per fer els teus programes robustos i fiables.
