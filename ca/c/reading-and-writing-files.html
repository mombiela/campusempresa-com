Introducció

En la programació en C, gestionar fitxers és una habilitat crucial per llegir dades de fitxers i escriure dades en fitxers. Aquest tema cobrirà els conceptes bàsics de les operacions amb fitxers, incloent-hi obrir, llegir, escriure i tancar fitxers. Al final d'aquesta secció, seràs capaç de realitzar operacions bàsiques d'entrada/sortida de fitxers en C.

Operacions amb Fitxers en C

Conceptes Clau

  • Punter de Fitxer: Un punter de tipus FILE utilitzat per fer el seguiment del fitxer que s'està accedint.
  • Obrir un Fitxer: Utilitzar fopen() per obrir un fitxer.
  • Tancar un Fitxer: Utilitzar fclose() per tancar un fitxer.
  • Llegir d'un Fitxer: Utilitzar funcions com fgetc(), fgets() i fread().
  • Escriure en un Fitxer: Utilitzar funcions com fputc(), fputs() i fwrite().

Obrir i Tancar Fitxers

Per treballar amb fitxers en C, primer has d'obrir-los. Això es fa utilitzant la funció fopen(), que retorna un punter de fitxer. Després de realitzar les operacions necessàries, has de tancar el fitxer utilitzant fclose().

Sintaxi

FILE *fopen(const char *filename, const char *mode);
int fclose(FILE *stream);

Modes per fopen()

| Mode | Descripció | |------|-------------| | "r" | Obrir per llegir. El fitxer ha d'existir. | | "w" | Obrir per escriure. Crea un fitxer nou si no existeix o trunca el fitxer si existeix. | | "a" | Obrir per afegir. Les dades s'afegeixen al final del fitxer. | | "r+" | Obrir per llegir i escriure. El fitxer ha d'existir. | | "w+" | Obrir per llegir i escriure. Crea un fitxer nou si no existeix o trunca el fitxer si existeix. | | "a+" | Obrir per llegir i afegir. Crea un fitxer nou si no existeix. |

Exemple

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error obrint el fitxer!\n");
        return 1;
    }
    // Realitzar operacions amb el fitxer
    fclose(file);
    return 0;
}

Llegir d'un Fitxer

Hi ha diverses funcions per llegir dades d'un fitxer, cadascuna adequada per a diferents tipus de dades i casos d'ús.

fgetc()

Llegeix un sol caràcter del fitxer.

int fgetc(FILE *stream);

fgets()

Llegeix una cadena del fitxer.

char *fgets(char *str, int n, FILE *stream);

fread()

Llegeix un bloc de dades del fitxer.

size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);

Exemple

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error obrint el fitxer!\n");
        return 1;
    }
    
    char ch;
    while ((ch = fgetc(file)) != EOF) {
        putchar(ch);
    }
    
    fclose(file);
    return 0;
}

Escriure en un Fitxer

Similar a la lectura, hi ha diverses funcions per escriure dades en un fitxer.

fputc()

Escriu un sol caràcter al fitxer.

int fputc(int char, FILE *stream);

fputs()

Escriu una cadena al fitxer.

int fputs(const char *str, FILE *stream);

fwrite()

Escriu un bloc de dades al fitxer.

size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);

Exemple

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error obrint el fitxer!\n");
        return 1;
    }
    
    fputs("Hola, Món!\n", file);
    
    fclose(file);
    return 0;
}

Conclusió

Entendre com llegir i escriure fitxers en C és essencial per a moltes tasques de programació. Aquest tema ha cobert els conceptes bàsics de les operacions amb fitxers, incloent-hi obrir, llegir, escriure i tancar fitxers. Practicant aquests conceptes, seràs capaç de gestionar operacions d'entrada/sortida de fitxers de manera efectiva en els teus programes en C.
