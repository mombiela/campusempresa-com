Prototips i Definicions de Funcions ►

Què és una Funció?

En programació en C, una funció és un bloc de codi que realitza una tasca específica. Les funcions ajuden a organitzar el codi, fent-lo més llegible, reutilitzable i més fàcil de depurar.

Conceptes Clau

  • Declaració de Funció (Prototip): Especifica el nom de la funció, el tipus de retorn i els paràmetres.
  • Definició de Funció: Conté el cos real de la funció.
  • Crida de Funció: Executa la funció.

Per què Utilitzar Funcions?

  • Modularitat: Divideix problemes complexos en tasques més petites i manejables.
  • Reutilitzabilitat: Les funcions es poden reutilitzar en diferents parts del programa.
  • Mantenibilitat: Més fàcil d'actualitzar i mantenir el codi.

Sintaxi de Funció

Declaració de Funció

Una funció s'ha de declarar abans de ser utilitzada. Això també es coneix com a prototip de funció.

return_type function_name(parameter_list);

Definició de Funció

La definició de la funció inclou el codi real que s'executarà quan es cridi la funció.

return_type function_name(parameter_list) {
    // cos de la funció
}

Crida de Funció

Per executar una funció, simplement es crida pel seu nom seguit de parèntesis.

function_name(arguments);

Exemple: Funció Bàsica

Creem una funció simple que suma dos enters.

Declaració de Funció

int add(int a, int b);

Definició de Funció

int add(int a, int b) {
    return a + b;
}

Crida de Funció

#include <stdio.h>

// Declaració de funció
int add(int a, int b);

int main() {
    int result;
    result = add(5, 3); // Crida de funció
    printf("La suma és: %d\n", result);
    return 0;
}

// Definició de funció
int add(int a, int b) {
    return a + b;
}

Tipus de Funcions

Funcions Incorporades

C proporciona moltes funcions incorporades com printf(), scanf(), strlen(), etc.

Funcions Definides per l'Usuari

Aquestes són funcions que crees per realitzar tasques específiques.

Tipus de Retorn i Paràmetres

Tipus de Retorn

Una funció pot retornar un valor de qualsevol tipus de dada. Si una funció no retorna cap valor, el seu tipus de retorn és void.

Paràmetres

Les funcions poden tenir zero o més paràmetres. Els paràmetres permeten passar dades a la funció.

Exemple: Funció sense Paràmetres i sense Valor de Retorn

#include <stdio.h>

void greet() {
    printf("Hola, Món!\n");
}

int main() {
    greet(); // Crida de funció
    return 0;
}

Abast i Durada de les Variables

Variables Locals

Les variables declarades dins d'una funció són locals a aquesta funció i no es poden accedir fora d'ella.

Variables Globals

Les variables declarades fora de totes les funcions són globals i poden ser accedides per qualsevol funció del programa.

Exemple: Variables Locals vs Globals

#include <stdio.h>

int globalVar = 10; // Variable global

void display() {
    int localVar = 20; // Variable local
    printf("Variable global: %d\n", globalVar);
    printf("Variable local: %d\n", localVar);
}

int main() {
    display();
    return 0;
}

Recursivitat

Una funció que es crida a si mateixa es coneix com a funció recursiva. La recursivitat pot ser una eina poderosa per resoldre problemes que es poden dividir en problemes més petits i similars.

Exemple: Factorial Usant Recursivitat

#include <stdio.h>

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int number = 5;
    printf("El factorial de %d és %d\n", number, factorial(number));
    return 0;
}

Resum

  • Les funcions són essencials per escriure codi modular, reutilitzable i mantenible.
  • Una funció consisteix en una declaració, definició i crida.
  • Les funcions poden tenir diferents tipus de retorn i paràmetres.
  • Entendre l'abast i la durada de les variables és crucial quan es treballa amb funcions.
  • La recursivitat és una tècnica on una funció es crida a si mateixa per resoldre un problema.

En dominar les funcions, seràs capaç d'escriure programes en C més eficients i organitzats.
