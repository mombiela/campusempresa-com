Introducció

El programa "Hola Món" és tradicionalment el primer programa escrit pels principiants que aprenen un nou llenguatge de programació. Serveix com una introducció simple a la sintaxi i l'estructura del llenguatge. En aquesta secció, cobrirem com escriure, compilar i executar un programa "Hola Món" en C.

Escriure el Programa

Per escriure un programa "Hola Món" en C, cal entendre l'estructura bàsica d'un programa en C. Aquí teniu el codi per a un simple programa "Hola Món":

#include <stdio.h>

int main() {
    printf("Hola, Món!\n");
    return 0;
}

Explicació

  • #include <stdio.h>: Aquesta línia inclou la biblioteca d'Entrada i Sortida Estàndard, que és necessària per utilitzar la funció printf.
  • int main() { ... }: Aquesta és la funció principal on comença l'execució del programa.
  • printf("Hola, Món!\n");: Aquesta línia imprimeix la cadena "Hola, Món!" a la consola. El \n és un caràcter de nova línia que mou el cursor a la següent línia.
  • return 0;: Aquesta línia finalitza la funció principal i retorna 0 al sistema operatiu, indicant que el programa ha acabat amb èxit.

Compilar el Programa

Per executar un programa en C, primer cal compilar-lo. Aquí teniu els passos per compilar i executar el programa "Hola Món":

  1. Guardar el Programa: Guardeu el codi en un fitxer amb una extensió .c, per exemple, hola.c.

  2. Obrir Terminal/Command Prompt: Navegueu al directori on heu guardat el fitxer.

  3. Compilar el Programa: Utilitzeu un compilador de C com gcc per compilar el programa. Executeu la següent comanda:

    gcc hola.c -o hola

    Aquesta comanda compila hola.c i crea un fitxer executable anomenat hola.

  4. Executar el Programa: Executeu el programa compilat executant:

    ./hola

Exemple

$ gcc hola.c -o hola
$ ./hola
Hola, Món!

Conceptes Clau

  • Directives del Preprocessador: Les línies que comencen amb # són directives del preprocessador. #include <stdio.h> diu al compilador que inclogui la biblioteca d'Entrada i Sortida Estàndard.
  • Funció Principal: El punt d'entrada d'un programa en C. Tots els programes en C han de tenir una funció main.
  • Sortida Estàndard: printf s'utilitza per imprimir text a la consola.
  • Instrucció de Retorn: return 0; indica que el programa s'ha executat amb èxit.

Errors Comuns

  • Falta de Punt i Coma: Tota instrucció en C ha d'acabar amb un punt i coma (;). Oblidar-se'n resultarà en un error de compilació.
  • Sensibilitat a Majúscules i Minúscules: C distingeix entre majúscules i minúscules. main és diferent de Main.
  • Extensió del Fitxer: Assegureu-vos que el vostre fitxer estigui guardat amb una extensió .c.

Conclusió

El programa "Hola Món" és un punt de partida fonamental per aprendre programació en C. Introduïu l'estructura bàsica d'un programa en C, incloent directives del preprocessador, la funció principal i la sortida estàndard. Entenent i practicant aquest simple programa, establiu les bases per a conceptes més complexos de programació en C.
