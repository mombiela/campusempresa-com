Introducció al Switch Case

La instrucció switch en C és una instrucció de control que permet provar una variable per igualtat contra una llista de valors. Cada valor s'anomena cas, i la variable que es canvia es comprova per a cada cas.

Conceptes Clau

  • Instrucció Switch: Una instrucció de control que prova una variable contra múltiples casos.
  • Cas: Un valor possible per a la variable que es prova.
  • Instrucció Break: S'utilitza per acabar un cas en la instrucció switch.
  • Cas per Defecte: Un cas opcional que s'executa si cap dels casos especificats coincideix amb la variable.

Sintaxi del Switch Case

La sintaxi bàsica d'una instrucció switch case en C és la següent:

switch (expressió) {
    case constant1:
        // codi a executar si l'expressió és igual a constant1
        break;
    case constant2:
        // codi a executar si l'expressió és igual a constant2
        break;
    // pots tenir qualsevol nombre d'instruccions case
    default:
        // codi a executar si l'expressió no coincideix amb cap cas
}

Exemple

Aquí tens un exemple senzill per demostrar l'ús de la instrucció switch case:

#include <stdio.h>

int main() {
    int dia = 3;

    switch (dia) {
        case 1:
            printf("Dilluns\n");
            break;
        case 2:
            printf("Dimarts\n");
            break;
        case 3:
            printf("Dimecres\n");
            break;
        case 4:
            printf("Dijous\n");
            break;
        case 5:
            printf("Divendres\n");
            break;
        case 6:
            printf("Dissabte\n");
            break;
        case 7:
            printf("Diumenge\n");
            break;
        default:
            printf("Dia invàlid\n");
    }

    return 0;
}

Explicació

  • La instrucció switch avalua l'expressió dia.
  • Després compara dia amb cada cas.
  • Quan dia coincideix amb un cas (en aquest exemple, dia és 3, així que coincideix amb case 3), el bloc de codi corresponent s'executa.
  • La instrucció break impedeix que l'execució continuï al següent cas.
  • Si cap cas coincideix, el cas default s'executa.

Ús Avançat

Switch Case Imbricat

Pots imbricar instruccions switch dins d'altres. Això és útil per a processos de presa de decisions més complexos.

#include <stdio.h>

int main() {
    char nota = 'B';

    switch (nota) {
        case 'A':
        case 'B':
            printf("Bon treball!\n");
            switch (nota) {
                case 'A':
                    printf("Excel·lent!\n");
                    break;
                case 'B':
                    printf("Molt bé!\n");
                    break;
            }
            break;
        case 'C':
            printf("Has aprovat.\n");
            break;
        default:
            printf("Millor sort la propera vegada.\n");
    }

    return 0;
}

Comportament de Caiguda

En C, si omets la instrucció break, el programa continua executant el següent cas. Això es coneix com a comportament de caiguda.

#include <stdio.h>

int main() {
    int num = 2;

    switch (num) {
        case 1:
            printf("Un\n");
        case 2:
            printf("Dos\n");
        case 3:
            printf("Tres\n");
        default:
            printf("Per defecte\n");
    }

    return 0;
}

Explicació

  • En aquest exemple, com que no hi ha instrucció break després de case 2, el programa continuarà executant case 3 i el cas default, resultant en la sortida: 
    Dos
Tres
Per defecte

Resum

  • La instrucció switch és una eina poderosa per gestionar múltiples condicions basades en el valor d'una sola variable.
  • Cada case representa un valor possible per a la variable.
  • La instrucció break és crucial per evitar el comportament de caiguda.
  • El cas default és opcional però útil per gestionar valors inesperats.
  • L'ús avançat inclou instruccions switch imbricades i caiguda intencionada per a lògica específica.

Entenent i utilitzant la instrucció switch case, pots escriure codi més llegible i mantenible per a escenaris de presa de decisions complexos.
