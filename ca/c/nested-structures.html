Introducció a les Estructures Imbricades

Les estructures imbricades en C et permeten crear tipus de dades complexos incrustant una estructura dins d'una altra. Això és particularment útil quan necessites representar dades jeràrquiques o compostes.

Conceptes Clau

  • Estructura: Un tipus de dada definit per l'usuari que agrupa variables relacionades.
  • Estructura Imbricada: Una estructura que conté una altra estructura com a membre.

Definició d'Estructures Imbricades

Per definir una estructura imbricada, simplement declares una estructura dins d'una altra. Aquí tens un exemple bàsic:

#include <stdio.h>

// Defineix l'estructura interna
struct Date {
    int day;
    int month;
    int year;
};

// Defineix l'estructura externa
struct Student {
    char name[50];
    int age;
    struct Date birthdate; // Estructura imbricada
};

int main() {
    // Declara i inicialitza una variable Student
    struct Student student1 = {"John Doe", 20, {15, 8, 2000}};
    
    // Accés als membres de l'estructura imbricada
    printf("Nom: %s\n", student1.name);
    printf("Edat: %d\n", student1.age);
    printf("Data de naixement: %02d/%02d/%d\n", student1.birthdate.day, student1.birthdate.month, student1.birthdate.year);
    
    return 0;
}

Explicació

  • Estructura Interna (Date): Conté day, month i year.
  • Estructura Externa (Student): Conté name, age i una estructura Date imbricada anomenada birthdate.
  • Inicialització: L'estructura Student s'inicialitza amb valors, incloent-hi l'estructura Date imbricada.
  • Accés als Membres: Utilitza l'operador punt (.) per accedir als membres tant de l'estructura externa com de la imbricada.

Usos Pràctics

Les estructures imbricades són útils en diversos escenaris, com ara:

  • Registres de Base de Dades: Representar un registre amb múltiples camps, alguns dels quals són en si mateixos tipus complexos.
  • Dades Jeràrquiques: Modelar dades jeràrquiques com ara organigrames o sistemes de fitxers.

Exemple: Registres d'Empleats

Considera un exemple on necessites emmagatzemar informació d'empleats, incloent-hi la seva adreça.

#include <stdio.h>

// Defineix l'estructura Address
struct Address {
    char street[100];
    char city[50];
    char state[50];
    int zip;
};

// Defineix l'estructura Employee
struct Employee {
    char name[50];
    int id;
    struct Address address; // Estructura imbricada
};

int main() {
    // Declara i inicialitza una variable Employee
    struct Employee emp1 = {"Alice Smith", 101, {"123 Main St", "Springfield", "IL", 62701}};
    
    // Accés als membres de l'estructura imbricada
    printf("Nom: %s\n", emp1.name);
    printf("ID: %d\n", emp1.id);
    printf("Adreça: %s, %s, %s, %d\n", emp1.address.street, emp1.address.city, emp1.address.state, emp1.address.zip);
    
    return 0;
}

Explicació

  • Estructura Interna (Address): Conté street, city, state i zip.
  • Estructura Externa (Employee): Conté name, id i una estructura Address imbricada anomenada address.
  • Inicialització i Accés: Similar a l'exemple anterior, però amb dades més complexes.

Exercicis

  1. Crea una estructura imbricada per a un Book que inclogui una estructura Author.

    • Author hauria de tenir name i birthYear.
    • Book hauria de tenir title, price i un Author.

  2. Escriu un programa per inicialitzar i imprimir els detalls d'un Book.

#include <stdio.h>

// Defineix l'estructura Author
struct Author {
    char name[50];
    int birthYear;
};

// Defineix l'estructura Book
struct Book {
    char title[100];
    float price;
    struct Author author; // Estructura imbricada
};

int main() {
    // Declara i inicialitza una variable Book
    struct Book book1 = {"The Great Gatsby", 10.99, {"F. Scott Fitzgerald", 1896}};
    
    // Accés als membres de l'estructura imbricada
    printf("Títol: %s\n", book1.title);
    printf("Preu: $%.2f\n", book1.price);
    printf("Autor: %s\n", book1.author.name);
    printf("Any de naixement de l'autor: %d\n", book1.author.birthYear);
    
    return 0;
}

Resum

Les estructures imbricades en C proporcionen una manera poderosa de modelar dades complexes incrustant una estructura dins d'una altra. Això permet un codi més organitzat i llegible, especialment quan es tracta de dades jeràrquiques o compostes. Entenent i utilitzant estructures imbricades, pots crear programes més sofisticats i mantenibles.
