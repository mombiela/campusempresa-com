Introducció

En la programació en C, les variables i les constants són conceptes fonamentals que cal entendre per escriure codi efectiu i eficient. Aquesta secció cobrirà què són les variables i les constants, com declarar-les i utilitzar-les, i les diferències entre elles.

Variables

Què és una Variable?

  • Una variable és una ubicació d'emmagatzematge en memòria amb un tipus específic que determina quin tipus de dades pot contenir.
  • Les variables tenen noms (identificadors) que s'utilitzen per accedir als seus valors emmagatzemats.

Declarar Variables

  • Per declarar una variable, cal especificar el seu tipus seguit del seu nom.
  • Sintaxi: type variable_name;
int age;
float salary;
char initial;

Inicialitzar Variables

  • Pots inicialitzar una variable en el moment de la declaració.
  • Sintaxi: type variable_name = value;
int age = 25;
float salary = 50000.50;
char initial = 'A';

Utilitzar Variables

  • Les variables es poden utilitzar en expressions i instruccions.
  • Exemple:
#include <stdio.h>

int main() {
    int age = 25;
    float salary = 50000.50;
    char initial = 'A';

    printf("Age: %d\n", age);
    printf("Salary: %.2f\n", salary);
    printf("Initial: %c\n", initial);

    return 0;
}

Constants

Què és una Constant?

  • Una constant és un valor que no pot ser alterat pel programa durant la seva execució.
  • Les constants s'utilitzen per definir valors fixos que no canvien.

Declarar Constants

  • En C, pots declarar constants utilitzant la paraula clau const o la directiva de preprocessador #define.

Utilitzant la Paraula Clau const

  • Sintaxi: const type constant_name = value;
const int DAYS_IN_WEEK = 7;
const float PI = 3.14159;
const char NEWLINE = '\n';

Utilitzant la Directiva de Preprocessador #define

  • Sintaxi: #define CONSTANT_NAME value
#define DAYS_IN_WEEK 7
#define PI 3.14159
#define NEWLINE '\n'

Utilitzar Constants

  • Les constants es poden utilitzar en expressions i instruccions igual que les variables.
  • Exemple:
#include <stdio.h>

#define PI 3.14159

int main() {
    const int DAYS_IN_WEEK = 7;

    printf("Days in a week: %d\n", DAYS_IN_WEEK);
    printf("Value of PI: %.5f\n", PI);

    return 0;
}

Diferències Entre Variables i Constants

| Característica | Variables | Constants | |-----------------|---------------------------------|-----------------------------------| | Mutabilitat | Pot canviar durant l'execució del programa | No pot canviar un cop definida | | Declaració | type variable_name; | const type constant_name = value; o #define CONSTANT_NAME value | | Ús | Utilitzat per emmagatzemar dades que poden canviar | Utilitzat per emmagatzemar valors fixos |

Conclusió

Entendre les variables i les constants és crucial per gestionar dades en els teus programes en C. Les variables et permeten emmagatzemar i manipular dades que poden canviar, mentre que les constants proporcionen una manera de definir valors que romanen iguals al llarg del programa. Dominant aquests conceptes, podràs escriure codi més robust i mantenible.
