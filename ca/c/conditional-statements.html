Declaracions condicionals en C
Bucles en C ►

Les declaracions condicionals són fonamentals en la programació, ja que permeten l'execució de codi basat en certes condicions. En C, les declaracions condicionals ajuden a controlar el flux del programa, fent possible executar diferents blocs de codi basats en criteris específics.

Conceptes clau

  • if statement: Executa un bloc de codi si una condició especificada és certa.
  • else statement: Executa un bloc de codi si la mateixa condició és falsa.
  • else if statement: Especifica una nova condició per provar si la primera condició és falsa.
  • switch statement: Selecciona un dels molts blocs de codi per executar basat en el valor d'una expressió.

if Statement

La declaració if és la forma més simple de declaració condicional. Avalua una condició i executa el bloc de codi següent si la condició és certa.

Sintaxi

if (condició) {
    // Codi a executar si la condició és certa
}

Exemple

#include <stdio.h>

int main() {
    int number = 10;

    if (number > 0) {
        printf("El número és positiu.\n");
    }

    return 0;
}

En aquest exemple, el programa comprova si number és més gran que 0. Com que number és 10, la condició és certa, i es mostra el missatge "El número és positiu.".

else Statement

La declaració else segueix una declaració if i executa un bloc de codi si la condició en la declaració if és falsa.

Sintaxi

if (condició) {
    // Codi a executar si la condició és certa
} else {
    // Codi a executar si la condició és falsa
}

Exemple

#include <stdio.h>

int main() {
    int number = -5;

    if (number > 0) {
        printf("El número és positiu.\n");
    } else {
        printf("El número no és positiu.\n");
    }

    return 0;
}

Aquí, el programa comprova si number és més gran que 0. Com que number és -5, la condició és falsa, i es mostra el missatge "El número no és positiu.".

else if Statement

La declaració else if permet comprovar múltiples condicions. Si la primera condició és falsa, comprova la següent condició, i així successivament.

Sintaxi

if (condició1) {
    // Codi a executar si la condició1 és certa
} else if (condició2) {
    // Codi a executar si la condició1 és falsa i la condició2 és certa
} else {
    // Codi a executar si tant la condició1 com la condició2 són falses
}

Exemple

#include <stdio.h>

int main() {
    int number = 0;

    if (number > 0) {
        printf("El número és positiu.\n");
    } else if (number < 0) {
        printf("El número és negatiu.\n");
    } else {
        printf("El número és zero.\n");
    }

    return 0;
}

En aquest exemple, el programa comprova si number és més gran que 0, menor que 0, o igual a 0, i mostra el missatge corresponent.

switch Statement

La declaració switch s'utilitza per seleccionar un dels molts blocs de codi per executar. Sovint s'utilitza quan tens múltiples condicions que depenen del valor d'una sola variable.

Sintaxi

switch (expressió) {
    case valor1:
        // Codi a executar si l'expressió és igual a valor1
        break;
    case valor2:
        // Codi a executar si l'expressió és igual a valor2
        break;
    // Pots tenir qualsevol nombre de declaracions case
    default:
        // Codi a executar si l'expressió no coincideix amb cap case
}

Exemple

#include <stdio.h>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            printf("Dilluns\n");
            break;
        case 2:
            printf("Dimarts\n");
            break;
        case 3:
            printf("Dimecres\n");
            break;
        case 4:
            printf("Dijous\n");
            break;
        case 5:
            printf("Divendres\n");
            break;
        case 6:
            printf("Dissabte\n");
            break;
        case 7:
            printf("Diumenge\n");
            break;
        default:
            printf("Dia invàlid\n");
    }

    return 0;
}

En aquest exemple, el programa utilitza una declaració switch per imprimir el nom del dia basat en el valor de day. Com que day és 3, imprimeix "Dimecres".

Resum

Les declaracions condicionals en C són essencials per controlar el flux d'un programa. Les declaracions if, else, i else if permeten comprovar condicions simples i complexes, mentre que la declaració switch proporciona una manera neta de manejar múltiples condicions basades en una sola variable. Entendre i utilitzar aquestes declaracions de manera efectiva et permetrà escriure programes més dinàmics i responsius.
