Introducció

En la programació en C, els punters i els arrays són conceptes fonamentals que estan estretament relacionats. Entendre com funcionen i com utilitzar-los de manera efectiva és crucial per escriure programes en C eficients i potents. Aquesta secció cobrirà els conceptes bàsics dels punters i els arrays, la seva relació i l'ús avançat.

Punteros

Els punters són variables que emmagatzemen l'adreça de memòria d'una altra variable. Són eines poderoses que permeten la gestió dinàmica de la memòria, la manipulació eficient d'arrays i la creació d'estructures de dades complexes.

Conceptes Clau

  • Declaració de Punter: Declarar un punter implica especificar el tipus de dades al qual apuntarà.
  • Inicialització de Punter: Assignar una adreça de memòria a un punter.
  • Desreferenciació: Accedir al valor emmagatzemat a l'adreça de memòria que conté el punter.
  • Aritmètica de Punter: Realitzar operacions aritmètiques sobre punters per navegar per la memòria.

Exemple de Codi: Ús Bàsic de Punter

#include <stdio.h>

int main() {
    int var = 10;
    int *ptr = &var; // Declaració i inicialització de punter

    printf("Valor de var: %d\n", var);
    printf("Adreça de var: %p\n", &var);
    printf("Valor emmagatzemat en ptr: %p\n", ptr);
    printf("Valor apuntat per ptr: %d\n", *ptr); // Desreferenciació

    return 0;
}

Explicació:

  • int *ptr = &var; declara un punter ptr i l'inicialitza amb l'adreça de var.
  • *ptr s'utilitza per accedir al valor emmagatzemat a l'adreça a la qual apunta ptr.

Arrays

Els arrays són col·leccions d'elements del mateix tipus emmagatzemats en ubicacions de memòria contigües. Proporcionen una manera d'emmagatzemar i gestionar múltiples valors de manera eficient.

Conceptes Clau

  • Declaració d'Array: Definir un array amb una mida especificada.
  • Inicialització d'Array: Assignar valors als elements d'un array.
  • Indexació d'Array: Accedir als elements d'un array utilitzant índexs.
  • Arrays Multidimensionals: Arrays amb més d'una dimensió, com ara matrius.

Exemple de Codi: Ús Bàsic d'Array

#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5}; // Declaració i inicialització d'array

    for (int i = 0; i < 5; i++) {
        printf("Element a l'índex %d: %d\n", i, arr[i]); // Indexació d'array
    }

    return 0;
}

Explicació:

  • int arr[5] = {1, 2, 3, 4, 5}; declara un array arr de mida 5 i l'inicialitza amb valors.
  • arr[i] s'utilitza per accedir als elements de l'array.

Relació Entre Punteros i Arrays

Els punters i els arrays estan estretament relacionats en C. El nom d'un array actua com un punter constant al primer element de l'array. Aquesta relació permet utilitzar punters per navegar i manipular arrays de manera eficient.

Conceptes Clau

  • Nom d'Array com a Punter: El nom d'un array és un punter al seu primer element.
  • Punter a Array: Utilitzar punters per iterar i manipular els elements d'un array.
  • Aritmètica de Punter amb Arrays: Utilitzar l'aritmètica de punter per accedir als elements de l'array.

Exemple de Codi: Punteros i Arrays

#include <stdio.h>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int *ptr = arr; // Nom d'array com a punter

    for (int i = 0; i < 5; i++) {
        printf("Element a l'índex %d: %d\n", i, *(ptr + i)); // Aritmètica de punter
    }

    return 0;
}

Explicació:

  • int *ptr = arr; assigna l'adreça del primer element de arr a ptr.
  • *(ptr + i) utilitza l'aritmètica de punter per accedir als elements de l'array.

Ús Avançat

Assignació Dinàmica de Memòria

Els punters són essencials per a l'assignació dinàmica de memòria, que permet la creació d'arrays la mida dels quals es pot determinar en temps d'execució.

Exemple de Codi: Assignació Dinàmica de Memòria

#include <stdio.h>
#include <stdlib.h>

int main() {
    int n;
    printf("Introdueix el nombre d'elements: ");
    scanf("%d", &n);

    int *arr = (int *)malloc(n * sizeof(int)); // Assignació dinàmica de memòria

    for (int i = 0; i < n; i++) {
        arr[i] = i + 1; // Inicialització d'array
    }

    for (int i = 0; i < n; i++) {
        printf("Element a l'índex %d: %d\n", i, arr[i]);
    }

    free(arr); // Alliberament de memòria assignada

    return 0;
}

Explicació:

  • malloc s'utilitza per assignar memòria dinàmicament.
  • free s'utilitza per alliberar la memòria un cop ja no es necessita.

Conclusió

Els punters i els arrays són conceptes fonamentals en la programació en C que proporcionen eines poderoses per a la gestió de memòria i la manipulació eficient de dades. Entendre la seva relació i com utilitzar-los de manera efectiva és crucial per escriure programes en C robustos. En dominar els punters i els arrays, pots crear aplicacions dinàmiques i eficients que aprofitin tot el poder del llenguatge C.
