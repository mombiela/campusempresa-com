Introducció a la Implementació en C

La implementació en C implica escriure el codi real que realitza les tasques definides en la fase de disseny. Aquesta secció et guiarà a través del procés d'implementar programes en C, des de la sintaxi bàsica fins a les característiques avançades.

Conceptes Clau

  • Sintaxi i Estructura: Comprendre la sintaxi bàsica i l'estructura d'un programa en C.
  • Variables i Tipus de Dades: Declarar i utilitzar variables de diferents tipus de dades.
  • Estructures de Control: Utilitzar sentències if-else, bucles i casos switch.
  • Funcions: Definir i cridar funcions, passar arguments i retornar valors.
  • Punteres: Comprendre i utilitzar punteres per a la gestió de memòria.
  • Arrays i Cadenes: Treballar amb arrays i cadenes.
  • Estructures: Definir i utilitzar estructures per agrupar dades relacionades.
  • Entrada/Sortida de Fitxers: Llegir i escriure fitxers.

Sintaxi i Estructura Bàsiques

Un programa en C típicament consisteix en funcions i sentències. La funció main és el punt d'entrada de qualsevol programa en C.

#include <stdio.h>

int main() {
    printf("Hola, Món!\n");
    return 0;
}
  • #include <stdio.h>: Directiva del preprocessador per incloure la biblioteca estàndard d'entrada-sortida.
  • int main(): Funció principal on comença l'execució.
  • printf("Hola, Món!\n");: Imprimeix "Hola, Món!" a la consola.
  • return 0;: Retorna 0 per indicar una execució exitosa.

Variables i Tipus de Dades

Les variables emmagatzemen dades que poden ser utilitzades i manipulades en un programa. C suporta diversos tipus de dades com int, float, char, etc.

#include <stdio.h>

int main() {
    int edat = 25;
    float alçada = 5.9;
    char inicial = 'A';

    printf("Edat: %d\n", edat);
    printf("Alçada: %.1f\n", alçada);
    printf("Inicial: %c\n", inicial);

    return 0;
}
  • int edat = 25;: Declara una variable enter edat i la inicialitza a 25.
  • float alçada = 5.9;: Declara una variable de punt flotant alçada i la inicialitza a 5.9.
  • char inicial = 'A';: Declara una variable de caràcter inicial i la inicialitza a 'A'.

Estructures de Control

Les estructures de control dirigeixen el flux d'execució en un programa.

Sentències If-Else

#include <stdio.h>

int main() {
    int numero = 10;

    if (numero > 0) {
        printf("El número és positiu.\n");
    } else {
        printf("El número no és positiu.\n");
    }

    return 0;
}

Bucles

Bucle For

#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {
        printf("i = %d\n", i);
    }

    return 0;
}

Bucle While

#include <stdio.h>

int main() {
    int i = 0;

    while (i < 5) {
        printf("i = %d\n", i);
        i++;
    }

    return 0;
}

Cas Switch

#include <stdio.h>

int main() {
    int dia = 3;

    switch (dia) {
        case 1:
            printf("Dilluns\n");
            break;
        case 2:
            printf("Dimarts\n");
            break;
        case 3:
            printf("Dimecres\n");
            break;
        default:
            printf("Altres dies\n");
    }

    return 0;
}

Funcions

Les funcions permeten encapsular codi en blocs reutilitzables.

#include <stdio.h>

void saluda() {
    printf("Hola, Món!\n");
}

int main() {
    saluda();
    return 0;
}
  • void saluda(): Defineix una funció anomenada saluda que no retorna cap valor.
  • saluda();: Crida la funció saluda.

Punteres

Les punteres emmagatzemen l'adreça de memòria d'una altra variable.

#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = &num;

    printf("Valor de num: %d\n", num);
    printf("Adreça de num: %p\n", &num);
    printf("Valor de ptr: %p\n", ptr);
    printf("Valor apuntat per ptr: %d\n", *ptr);

    return 0;
}
  • int *ptr = &num;: Declara una puntera ptr que emmagatzema l'adreça de num.
  • *ptr: Desreferencia la puntera per obtenir el valor emmagatzemat a l'adreça.

Arrays i Cadenes

Els arrays emmagatzemen múltiples valors del mateix tipus.

#include <stdio.h>

int main() {
    int numeros[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        printf("numeros[%d] = %d\n", i, numeros[i]);
    }

    return 0;
}

Les cadenes són arrays de caràcters.

#include <stdio.h>

int main() {
    char salutacio[] = "Hola, Món!";

    printf("%s\n", salutacio);

    return 0;
}

Estructures

Les estructures agrupen dades relacionades de diferents tipus.

#include <stdio.h>
#include <string.h>

struct Persona {
    char nom[50];
    int edat;
    float alçada;
};

int main() {
    struct Persona persona1;

    // Assignar valors a persona1
    strcpy(persona1.nom, "Joan Doe");
    persona1.edat = 30;
    persona1.alçada = 5.9;

    printf("Nom: %s\n", persona1.nom);
    printf("Edat: %d\n", persona1.edat);
    printf("Alçada: %.1f\n", persona1.alçada);

    return 0;
}

Entrada/Sortida de Fitxers

Llegir i escriure fitxers.

Escriure a un Fitxer

#include <stdio.h>

int main() {
    FILE *fptr;
    fptr = fopen("fitxer.txt", "w");

    if (fptr == NULL) {
        printf("Error obrint el fitxer!\n");
        return 1;
    }

    fprintf(fptr, "Hola, Món!\n");
    fclose(fptr);

    return 0;
}

Llegir d'un Fitxer

#include <stdio.h>

int main() {
    FILE *fptr;
    char buffer[255];

    fptr = fopen("fitxer.txt", "r");

    if (fptr == NULL) {
        printf("Error obrint el fitxer!\n");
        return 1;
    }

    while (fgets(buffer, 255, fptr)) {
        printf("%s", buffer);
    }

    fclose(fptr);

    return 0;
}

Comprenent i practicant aquests conceptes, estaràs ben equipat per escriure programes en C eficients i efectius.
