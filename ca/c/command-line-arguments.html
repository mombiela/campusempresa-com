Introducció

Els arguments de la línia de comandes són una manera de proporcionar entrada a un programa des de la línia de comandes. Permeten als usuaris especificar paràmetres o opcions quan executen un programa, fent-lo més flexible i dinàmic.

Conceptes Clau

  • Interfície de Línia de Comandes (CLI): Una interfície basada en text utilitzada per interactuar amb programari i sistemes operatius.
  • Arguments: Valors proporcionats al programa en temps d'execució a través de la línia de comandes.
  • argc (Compte d'Arguments): Un enter que representa el nombre d'arguments de la línia de comandes.
  • argv (Vector d'Arguments): Una matriu de punters de caràcters que llista tots els arguments.

Entenent els Arguments de la Línia de Comandes

En C, els arguments de la línia de comandes es gestionen utilitzant dos paràmetres de la funció main:

int main(int argc, char *argv[]) {
    // El teu codi aquí
}
  • argc: Aquest paràmetre (compte d'arguments) conté el nombre d'arguments passats al programa, incloent el nom del programa.
  • argv: Aquest paràmetre (vector d'arguments) és una matriu de cadenes (punters de caràcters) que representen els arguments.

Exemple: Ús Bàsic

Vegem un exemple senzill per entendre com funcionen els arguments de la línia de comandes:

#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Nombre d'arguments: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    return 0;
}

Explicació

  • El programa imprimeix el nombre d'arguments (argc).
  • Després, itera a través de la matriu argv i imprimeix cada argument.

Executant el Programa

Per executar el programa amb arguments de la línia de comandes, compila'l i executa'l des del terminal:

gcc main.c -o main
./main arg1 arg2 arg3

Sortida

Nombre d'arguments: 4
Argument 0: ./main
Argument 1: arg1
Argument 2: arg2
Argument 3: arg3

Ús Avançat

Analitzant Arguments

En programes més complexos, potser necessitaràs analitzar i validar els arguments de la línia de comandes. Aquí tens un exemple que espera dos arguments numèrics i calcula la seva suma:

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    if (argc != 3) {
        printf("Ús: %s <num1> <num2>\n", argv[0]);
        return 1;
    }

    int num1 = atoi(argv[1]);
    int num2 = atoi(argv[2]);
    int suma = num1 + num2;

    printf("Suma: %d\n", suma);
    return 0;
}

Explicació

  • El programa comprova si el nombre d'arguments és exactament 3 (incloent el nom del programa).
  • Converteix els arguments de cadenes a enters utilitzant atoi.
  • Calcula la suma i la imprimeix.

Executant el Programa

gcc main.c -o main
./main 5 10

Sortida

Suma: 15

Errors Comuns

  • Compte d'Arguments Incorrecte: Sempre valida el nombre d'arguments per evitar comportaments inesperats.
  • Tipus de Dades Invàlids: Assegura't que els arguments són del tipus esperat (per exemple, convertint cadenes a enters).

Conclusió

Els arguments de la línia de comandes són una característica poderosa en C que permeten fer els teus programes més flexibles i interactius. Entenent com utilitzar argc i argv, pots crear programes que acceptin entrada de l'usuari directament des de la línia de comandes, millorant la seva usabilitat i funcionalitat. Practica escrivint programes que prenguin diferents tipus d'entrada i realitzin diverses operacions basades en aquestes entrades.
