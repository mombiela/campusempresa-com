Passatge de Paràmetres en C
Recursivitat ►

Introducció

El passatge de paràmetres és un concepte fonamental en la programació en C que permet que les funcions rebin valors d'entrada i retornin resultats. Entendre com es passen els paràmetres a les funcions és crucial per escriure codi C eficient i efectiu.

Tipus de Passatge de Paràmetres

En C, hi ha dues maneres principals de passar paràmetres a les funcions:

  1. Passatge per Valor
  2. Passatge per Referència

Passatge per Valor

Quan els paràmetres es passen per valor, es fa una còpia del valor real del paràmetre en memòria. Això significa que els canvis fets al paràmetre dins de la funció no afecten el valor original.

Exemple:

#include <stdio.h>

void addTen(int num) {
    num = num + 10;
    printf("Valor dins de la funció: %d\n", num);
}

int main() {
    int number = 5;
    addTen(number);
    printf("Valor fora de la funció: %d\n", number);
    return 0;
}

Explicació:

  • number es passa a addTen per valor.
  • Dins de addTen, num és una còpia de number.
  • Els canvis a num no afecten number.

Passatge per Referència

Quan els paràmetres es passen per referència, la funció rep una referència (o punter) al paràmetre real. Això permet que la funció modifiqui el valor original.

Exemple:

#include <stdio.h>

void addTen(int *num) {
    *num = *num + 10;
    printf("Valor dins de la funció: %d\n", *num);
}

int main() {
    int number = 5;
    addTen(&number);
    printf("Valor fora de la funció: %d\n", number);
    return 0;
}

Explicació:

  • number es passa a addTen per referència utilitzant la seva adreça (&number).
  • Dins de addTen, num és un punter a number.
  • Els canvis a *num afecten number.

Comparació del Passatge per Valor i per Referència

| Característica | Passatge per Valor | Passatge per Referència | |------------------------|-----------------------------------|-------------------------------------| | Tipus de Paràmetre | Còpia del valor real | Adreça del valor real | | Ús de Memòria | Més (a causa de la còpia) | Menys (sense còpia, només referència)| | Efecte sobre l'Original | Cap efecte | Pot modificar el valor original | | Cas d'Ús | Quan el valor original ha de romandre inalterat | Quan el valor original necessita ser modificat |

Exemples Pràctics

Intercanvi de Valors

Passatge per Valor (Incorrecte):

#include <stdio.h>

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(x, y);
    printf("x: %d, y: %d\n", x, y); // La sortida serà x: 10, y: 20
    return 0;
}

Passatge per Referència (Correcte):

#include <stdio.h>

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);
    printf("x: %d, y: %d\n", x, y); // La sortida serà x: 20, y: 10
    return 0;
}

Conclusió

Entendre el passatge de paràmetres és essencial per a una programació efectiva en C. Dominant tant el passatge per valor com per referència, pots controlar com es manipulen les dades dins de les teves funcions, portant a un codi més eficient i predictible. Recorda:

  • Utilitza el passatge per valor quan no vulguis que la funció modifiqui les dades originals.
  • Utilitza el passatge per referència quan necessitis que la funció modifiqui les dades originals.
