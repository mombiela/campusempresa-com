Introducció

Els operadors són símbols especials en C que realitzen operacions sobre variables i valors. Són fonamentals per manipular dades i realitzar càlculs en un programa. Aquesta secció cobrirà els diversos tipus d'operadors disponibles en C, el seu ús i exemples per il·lustrar la seva funcionalitat.

Tipus d'Operadors

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • Suma (+): Suma dos operands.
  • Resta (-): Resta el segon operand del primer.
  • Multiplicació (*): Multiplica dos operands.
  • Divisió (/): Divideix el primer operand pel segon.
  • Mòdul (%): Retorna el residu d'una operació de divisió.
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("Suma: %d\n", a + b);       // Sortida: 13
    printf("Resta: %d\n", a - b);    // Sortida: 7
    printf("Multiplicació: %d\n", a * b); // Sortida: 30
    printf("Divisió: %d\n", a / b);       // Sortida: 3
    printf("Mòdul: %d\n", a % b);        // Sortida: 1
    return 0;
}

Operadors Relacionals

Els operadors relacionals comparen dos valors i retornen un resultat booleà (cert o fals).

  • Igual a (==): Comprova si dos operands són iguals.
  • No igual a (!=): Comprova si dos operands no són iguals.
  • Major que (>): Comprova si el primer operand és major que el segon.
  • Menor que (<): Comprova si el primer operand és menor que el segon.
  • Major o igual a (>=): Comprova si el primer operand és major o igual que el segon.
  • Menor o igual a (<=): Comprova si el primer operand és menor o igual que el segon.
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    printf("a == b: %d\n", a == b); // Sortida: 0 (fals)
    printf("a != b: %d\n", a != b); // Sortida: 1 (cert)
    printf("a > b: %d\n", a > b);   // Sortida: 0 (fals)
    printf("a < b: %d\n", a < b);   // Sortida: 1 (cert)
    printf("a >= b: %d\n", a >= b); // Sortida: 0 (fals)
    printf("a <= b: %d\n", a <= b); // Sortida: 1 (cert)
    return 0;
}

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions o per invertir la lògica d'una condició.

  • AND lògic (&&): Retorna cert si ambdós operands són certs.
  • OR lògic (||): Retorna cert si almenys un dels operands és cert.
  • NOT lògic (!): Inverteix el valor booleà de l'operand.
#include <stdio.h>

int main() {
    int a = 1, b = 0;
    printf("a && b: %d\n", a && b); // Sortida: 0 (fals)
    printf("a || b: %d\n", a || b); // Sortida: 1 (cert)
    printf("!a: %d\n", !a);         // Sortida: 0 (fals)
    printf("!b: %d\n", !b);         // Sortida: 1 (cert)
    return 0;
}

Operadors a Nivell de Bit

Els operadors a nivell de bit realitzen operacions sobre les representacions binàries dels enters.

  • AND (&): Realitza una operació AND a nivell de bit.
  • OR (|): Realitza una operació OR a nivell de bit.
  • XOR (^): Realitza una operació XOR a nivell de bit.
  • NOT (~): Realitza una operació NOT a nivell de bit.
  • Desplaçament a l'esquerra (<<): Desplaça bits a l'esquerra.
  • Desplaçament a la dreta (>>): Desplaça bits a la dreta.
#include <stdio.h>

int main() {
    int a = 5;  // Binari: 0101
    int b = 3;  // Binari: 0011
    printf("a & b: %d\n", a & b); // Sortida: 1 (0001)
    printf("a | b: %d\n", a | b); // Sortida: 7 (0111)
    printf("a ^ b: %d\n", a ^ b); // Sortida: 6 (0110)
    printf("~a: %d\n", ~a);       // Sortida: -6 (inverteix tots els bits)
    printf("a << 1: %d\n", a << 1); // Sortida: 10 (1010)
    printf("a >> 1: %d\n", a >> 1); // Sortida: 2 (0010)
    return 0;
}

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables. També poden realitzar operacions i assignar el resultat a la variable.

  • Assignació (=): Assigna l'operand dret a l'operand esquerre.
  • Suma i assigna (+=): Suma l'operand dret a l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Resta i assigna (-=): Resta l'operand dret de l'operand esquerre i assigna el resultat a l'operand esquerre.
  • Multiplica i assigna (*=): Multiplica l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.
  • Divideix i assigna (/=): Divideix l'operand esquerre per l'operand dret i assigna el resultat a l'operand esquerre.
  • Mòdul i assigna (%=): Pren el mòdul utilitzant l'operand esquerre i l'operand dret i assigna el resultat a l'operand esquerre.
#include <stdio.h>

int main() {
    int a = 10;
    a += 5; // Equivalent a a = a + 5
    printf("a += 5: %d\n", a); // Sortida: 15
    a -= 3; // Equivalent a a = a - 3
    printf("a -= 3: %d\n", a); // Sortida: 12
    a *= 2; // Equivalent a a = a * 2
    printf("a *= 2: %d\n", a); // Sortida: 24
    a /= 4; // Equivalent a a = a / 4
    printf("a /= 4: %d\n", a); // Sortida: 6
    a %= 5; // Equivalent a a = a % 5
    printf("a %%= 5: %d\n", a); // Sortida: 1
    return 0;
}

Operadors d'Increment i Decrement

Aquests operadors s'utilitzen per augmentar o disminuir el valor d'una variable en un.

  • Increment (++): Augmenta el valor en un.
  • Decrement (--): Disminueix el valor en un.
#include <stdio.h>

int main() {
    int a = 10;
    a++; // Equivalent a a = a + 1
    printf("a++: %d\n", a); // Sortida: 11
    a--; // Equivalent a a = a - 1
    printf("a--: %d\n", a); // Sortida: 10
    return 0;
}

Operador Condicional (Ternari)

L'operador condicional és una forma abreujada d'una instrucció if-else. Pren tres operands.

  • Sintaxi: condició ? expressió_si_cert : expressió_si_fals
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int max = (a > b) ? a : b; // Si a > b, max = a; en cas contrari, max = b
    printf("Valor màxim: %d\n", max); // Sortida: 20
    return 0;
}

Conclusió

Els operadors en C són eines essencials per realitzar una àmplia gamma d'operacions, des de l'aritmètica bàsica fins a manipulacions lògiques i a nivell de bit complexes. Entendre com utilitzar aquests operadors de manera efectiva és crucial per escriure programes en C eficients i funcionals. Practiqueu utilitzant aquests operadors en diferents escenaris per esdevenir proficient en la seva aplicació.
