L'aritmètica de punter és una característica poderosa en C que et permet realitzar operacions en punters per navegar a través de la memòria. Entendre l'aritmètica de punter és crucial per treballar amb arrays, memòria dinàmica i diverses estructures de dades.

Conceptes Clau

  • Punters i Adreces de Memòria: Els punters emmagatzemen adreces de memòria de variables. L'aritmètica de punter implica manipular aquestes adreces.
  • Tipus de Punter: El tipus d'un punter determina la mida dels passos fets durant les operacions aritmètiques.
  • Increment i Decrement: Pots incrementar o decrementar punters per moure't a la següent o anterior ubicació de memòria.
  • Addició i Subtracció: Pots sumar o restar enters a/de punters per navegar a través de la memòria.

Increment i Decrement

Quan incrementes o decrementes un punter, es mou a la següent o anterior ubicació de memòria basant-se en el tipus al qual apunta.

Exemple: Incrementant un Punter

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr = arr; // ptr apunta al primer element de arr

    printf("Adreça inicial del punter: %p\n", ptr);
    printf("Valor inicial: %d\n", *ptr);

    ptr++; // Mou-te al següent enter en l'array

    printf("Adreça del punter després de l'increment: %p\n", ptr);
    printf("Valor després de l'increment: %d\n", *ptr);

    return 0;
}

Explicació:

  • ptr inicialment apunta a arr[0].
  • Després de ptr++, apunta a arr[1].

Addició i Subtracció

Pots sumar o restar enters a/de punters per moure't diversos passos alhora.

Exemple: Addició de Punter

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr = arr; // ptr apunta al primer element de arr

    printf("Adreça inicial del punter: %p\n", ptr);
    printf("Valor inicial: %d\n", *ptr);

    ptr += 2; // Mou-te dos enters endavant

    printf("Adreça del punter després de l'addició: %p\n", ptr);
    printf("Valor després de l'addició: %d\n", *ptr);

    return 0;
}

Explicació:

  • ptr inicialment apunta a arr[0].
  • Després de ptr += 2, apunta a arr[2].

Subtracció de Punter

La subtracció de punter es pot utilitzar per trobar la distància entre dos punters.

Exemple: Subtracció de Punter

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr1 = &arr[0];
    int *ptr2 = &arr[2];

    printf("Adreça del Punter 1: %p\n", ptr1);
    printf("Adreça del Punter 2: %p\n", ptr2);

    int distance = ptr2 - ptr1; // Calcula la distància entre ptr2 i ptr1

    printf("Distància entre punters: %d\n", distance);

    return 0;
}

Explicació:

  • ptr1 apunta a arr[0].
  • ptr2 apunta a arr[2].
  • La distància entre ptr2 i ptr1 és 2.

Aritmètica de Punter amb Diferents Tipus

L'aritmètica de punter es comporta de manera diferent amb diferents tipus de dades perquè la mida del tipus afecta la mida del pas.

Exemple: Aritmètica de Punter amb `char` i `int`

#include <stdio.h>

int main() {
    char charArr[3] = {'a', 'b', 'c'};
    int intArr[3] = {10, 20, 30};

    char *charPtr = charArr;
    int *intPtr = intArr;

    printf("Adreça inicial del punter char: %p\n", charPtr);
    printf("Adreça inicial del punter int: %p\n", intPtr);

    charPtr++;
    intPtr++;

    printf("Adreça del punter char després de l'increment: %p\n", charPtr);
    printf("Adreça del punter int després de l'increment: %p\n", intPtr);

    return 0;
}

Explicació:

  • charPtr++ es mou per 1 byte (mida de char).
  • intPtr++ es mou per 4 bytes (mida de int).

Resum

L'aritmètica de punter és un concepte fonamental en C que et permet navegar a través de la memòria de manera eficient. Entenent com incrementar, decrementar, sumar i restar punters, pots manipular arrays i altres estructures de dades més efectivament. Recorda que el tipus del punter determina la mida dels passos fets durant les operacions aritmètiques. Practica aquests conceptes amb diferents tipus de dades per obtenir una comprensió més profunda.
