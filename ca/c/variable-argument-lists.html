Les llistes d'arguments variables permeten que les funcions acceptin un nombre indefinit d'arguments. Això és particularment útil per a funcions com printf i scanf que necessiten gestionar un nombre variable de paràmetres.

Conceptes Clau

  • Funcions Variàdiques: Funcions que poden prendre un nombre variable d'arguments.
  • Biblioteca stdarg.h: Proporciona macros per gestionar arguments variables.
  • Macros: va_list, va_start, va_arg i va_end.

Ús de Llistes d'Arguments Variables

Per crear una funció variàdica, cal incloure la biblioteca stdarg.h i utilitzar macros específiques per gestionar els arguments.

Passos per Crear una Funció Variàdica

  1. Incloure la Biblioteca stdarg.h:

    #include <stdarg.h>

  2. Definir la Funció amb Punts Suspensius:

    • Utilitza ... a la llista de paràmetres de la funció per indicar que la funció pot acceptar un nombre variable d'arguments.
    void exampleFunction(int fixedArg, ...);

  3. Inicialitzar el Tipus va_list:

    • Utilitza va_list per declarar una variable que contindrà els arguments.
    va_list args;

  4. Començar a Processar Arguments:

    • Utilitza va_start per inicialitzar la variable va_list.
    va_start(args, fixedArg);

  5. Accedir a Cada Argument:

    • Utilitza va_arg per recuperar cada argument.
    int nextArg = va_arg(args, int);

  6. Netejar:

    • Utilitza va_end per netejar la variable va_list.
    va_end(args);

Exemple: Suma d'un Nombre Variable d'Enters

Aquí tens un exemple senzill d'una funció que calcula la suma d'un nombre variable d'enters:

#include <stdarg.h>
#include <stdio.h>

int sum(int count, ...) {
    va_list args;
    int total = 0;

    // Inicialitzar la llista d'arguments
    va_start(args, count);

    // Bucle a través de tots els arguments
    for (int i = 0; i < count; i++) {
        total += va_arg(args, int);
    }

    // Netejar la llista d'arguments
    va_end(args);

    return total;
}

int main() {
    printf("Suma de 1, 2, 3: %d\n", sum(3, 1, 2, 3));
    printf("Suma de 5, 10, 15, 20: %d\n", sum(4, 5, 10, 15, 20));
    return 0;
}

Explicació

  • Definició de la Funció:
    • int sum(int count, ...): La funció sum pren un argument fix count i un nombre variable d'arguments addicionals.
  • Inicialització:
    • va_list args;: Declara una variable per contenir la llista d'arguments.
    • va_start(args, count);: Inicialitza la llista d'arguments, començant després del paràmetre count.
  • Accés als Arguments:
    • total += va_arg(args, int);: Recupera el següent argument de la llista, assumint que és de tipus int.
  • Neteja:
    • va_end(args);: Neteja la llista d'arguments.

Resum

  • Funcions Variàdiques: Útils per a funcions que necessiten gestionar un nombre variable de paràmetres.
  • Biblioteca stdarg.h: Proporciona les macros necessàries per gestionar arguments variables.
  • Macros:
    • va_list: Tipus per contenir els arguments variables.
    • va_start: Inicialitza la llista d'arguments.
    • va_arg: Accedeix a cada argument.
    • va_end: Neteja la llista d'arguments.

Entenent i utilitzant aquests conceptes, pots crear funcions flexibles i potents en C que gestionin un nombre variable d'arguments de manera eficient.
