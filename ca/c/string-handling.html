Introducció

La manipulació de cadenes és un aspecte fonamental de la programació en C. Les cadenes en C són arrays de caràcters terminats per un caràcter nul ('\0'). Aquest tema cobrirà els conceptes bàsics de la manipulació de cadenes, incloent la declaració, inicialització, funcions de la biblioteca estàndard i tècniques avançades de manipulació de cadenes.

1. Conceptes Bàsics de les Cadenes en C

1.1 Declaració i Inicialització

  • Les cadenes es declaren com arrays de caràcters.
  • Han de ser terminades amb un caràcter nul ('\0').
char str1[20]; // Declaració d'una cadena amb una mida màxima de 20 caràcters
char str2[] = "Hello, World!"; // Inicialització amb un literal de cadena

1.2 Entrada i Sortida de Cadenes

  • Utilitza scanf i printf per a l'entrada i sortida bàsiques.
  • gets i puts també es poden utilitzar, però gets és insegur i s'ha d'evitar.
char name[50];
printf("Enter your name: ");
scanf("%s", name); // Nota: scanf deixa de llegir al primer espai en blanc
printf("Hello, %s!\n", name);

2. Funcions de la Biblioteca Estàndard per a Cadenes

2.1 `strlen` - Longitud de la Cadena

  • Retorna la longitud de la cadena (excloent el caràcter nul).
#include <stdio.h>
#include <string.h>

char str[] = "Hello";
int length = strlen(str); // length = 5

2.2 `strcpy` - Còpia de la Cadena

  • Copia una cadena a una altra.
#include <stdio.h>
#include <string.h>

char src[] = "Hello";
char dest[20];
strcpy(dest, src); // dest ara conté "Hello"

2.3 `strcat` - Concatenació de Cadenes

  • Afegeix una cadena al final d'una altra.
#include <stdio.h>
#include <string.h>

char str1[20] = "Hello";
char str2[] = " World";
strcat(str1, str2); // str1 ara conté "Hello World"

2.4 `strcmp` - Comparació de Cadenes

  • Compara dues cadenes lexicogràficament.
#include <stdio.h>
#include <string.h>

char str1[] = "Hello";
char str2[] = "World";
int result = strcmp(str1, str2); // result < 0 perquè "Hello" és menor que "World"

2.5 `strchr` i `strrchr` - Cerca de Caràcters

  • strchr troba la primera ocurrència d'un caràcter.
  • strrchr troba l'última ocurrència d'un caràcter.
#include <stdio.h>
#include <string.h>

char str[] = "Hello, World!";
char *first = strchr(str, 'o'); // first apunta al primer 'o' en "Hello, World!"
char *last = strrchr(str, 'o'); // last apunta a l'últim 'o' en "Hello, World!"

3. Manipulació Avançada de Cadenes

3.1 Cadenes Dinàmiques

  • Utilitza punters i assignació dinàmica de memòria per a cadenes la mida de les quals no es coneix en temps de compilació.
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

char *dynamicStr = (char *)malloc(50 * sizeof(char));
if (dynamicStr != NULL) {
    strcpy(dynamicStr, "Dynamic String");
    printf("%s\n", dynamicStr);
    free(dynamicStr); // Allibera sempre la memòria assignada dinàmicament
}

3.2 Tokenització de Cadenes

  • Utilitza strtok per dividir una cadena en tokens basats en delimitadors.
#include <stdio.h>
#include <string.h>

char str[] = "Hello, World! Welcome to C programming.";
char *token = strtok(str, " ,.!");

while (token != NULL) {
    printf("%s\n", token);
    token = strtok(NULL, " ,.!"); // Continua tokenitzant la resta de la cadena
}

Conclusió

La manipulació de cadenes en C implica entendre com es representen les cadenes com arrays de caràcters i utilitzar diverses funcions de la biblioteca estàndard per manipular-les. Des d'operacions bàsiques com copiar i concatenar fins a tècniques més avançades com l'assignació dinàmica i la tokenització, dominar la manipulació de cadenes és essencial per a una programació efectiva en C. Recorda sempre manipular les cadenes amb cura per evitar errors comuns com desbordaments de memòria intermèdia i fuites de memòria.
