Introducció a les cadenes en C

En C, les cadenes són arrays de caràcters terminats per un caràcter nul ('\0'). Entendre com manipular cadenes és crucial per a moltes tasques de programació, des del processament de text simple fins a l'anàlisi de dades complexa.

Conceptes clau

  • Declaració de cadena: Les cadenes es declaren com arrays de caràcters.
  • Terminador nul: El final d'una cadena està marcat per un caràcter nul ('\0').
  • Inicialització de cadena: Les cadenes es poden inicialitzar en el moment de la declaració.

Exemple

char str1[] = "Hello, World!";
char str2[20] = "Hello, C!";
char str3[20]; // Cadena no inicialitzada

Funcions estàndard de cadena

La Biblioteca Estàndard de C proporciona un conjunt de funcions per a la manipulació de cadenes, declarades al fitxer de capçalera <string.h>.

Funcions de cadena comunament utilitzades

  • strlen(): Retorna la longitud d'una cadena.
  • strcpy(): Copia una cadena a una altra.
  • strncpy(): Copia un nombre especificat de caràcters d'una cadena a una altra.
  • strcat(): Concatena dues cadenes.
  • strncat(): Concatena un nombre especificat de caràcters d'una cadena a una altra.
  • strcmp(): Compara dues cadenes.
  • strncmp(): Compara un nombre especificat de caràcters de dues cadenes.
  • strchr(): Troba la primera ocurrència d'un caràcter en una cadena.
  • strstr(): Troba la primera ocurrència d'una subcadena en una cadena.

Exemple d'ús

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    char str3[40];

    // Longitud de la cadena
    printf("Longitud de str1: %lu\n", strlen(str1));

    // Còpia de cadena
    strcpy(str3, str1);
    printf("str3 després de strcpy: %s\n", str3);

    // Concatenació de cadena
    strcat(str3, " ");
    strcat(str3, str2);
    printf("str3 després de strcat: %s\n", str3);

    // Comparació de cadena
    if (strcmp(str1, str2) == 0) {
        printf("str1 i str2 són iguals\n");
    } else {
        printf("str1 i str2 no són iguals\n");
    }

    return 0;
}

Funcions avançades de cadena

Per a manipulacions de cadena més complexes, C proporciona funcions addicionals.

Funcions de gestió de memòria

  • strdup(): Duplica una cadena mitjançant l'assignació dinàmica de memòria.
  • strtok(): Tokenitza una cadena basada en delimitadors.

Exemple d'ús

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main() {
    char str[] = "Hello, World!";
    char *token;
    char *dup_str;

    // Duplicació de cadena
    dup_str = strdup(str);
    if (dup_str != NULL) {
        printf("Cadena duplicada: %s\n", dup_str);
        free(dup_str); // Alliberar la memòria assignada
    }

    // Tokenització de cadena
    token = strtok(str, ", ");
    while (token != NULL) {
        printf("Token: %s\n", token);
        token = strtok(NULL, ", ");
    }

    return 0;
}

Resum

Les funcions de cadena en C són eines essencials per a la manipulació i gestió de text. Des d'operacions bàsiques com la còpia i la concatenació fins a tasques més avançades com la tokenització i la gestió dinàmica de memòria, dominar aquestes funcions millorarà significativament les teves capacitats de programació. Recorda sempre incloure el fitxer de capçalera <string.h> i gestionar la memòria amb cura quan treballis amb cadenes dinàmiques.
