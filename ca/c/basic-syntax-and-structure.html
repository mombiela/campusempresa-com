Introducció

Entendre la sintaxi i l'estructura bàsiques del C és crucial per escriure programes eficients i sense errors. Aquesta secció cobrirà els elements fonamentals de la programació en C, incloent l'estructura d'un programa en C, tipus de dades, variables i operacions bàsiques d'entrada/sortida.

Estructura d'un Programa en C

Un programa en C típicament consisteix en les següents parts:

  • Comandes del Preprocessador
  • Funcions
  • Variables
  • Declaracions i Expressions
  • Comentaris

Exemple d'un Programa Simple en C

#include <stdio.h> // Comanda del Preprocessador

// Funció Principal
int main() {
    // Declaració de Variable
    int number;

    // Declaració de Sortida
    printf("Enter a number: ");

    // Declaració d'Entrada
    scanf("%d", &number);

    // Declaració de Sortida
    printf("You entered: %d\n", number);

    return 0; // Declaració de Retorn
}

Explicació:

  • #include <stdio.h>: Aquesta és una comanda del preprocessador que indica al compilador que inclogui la biblioteca estàndard d'entrada/sortida abans de compilar el programa.
  • int main() { ... }: Aquesta és la funció principal on comença l'execució del programa.
  • int number;: Això declara una variable enter anomenada number.
  • printf("Enter a number: ");: Això imprimeix el missatge "Enter a number:" a la consola.
  • scanf("%d", &number);: Això llegeix una entrada enter de l'usuari i l'emmagatzema a la variable number.
  • printf("You entered: %d\n", number);: Això imprimeix el missatge "You entered:" seguit del valor de number.
  • return 0;: Aquesta declaració finalitza la funció principal i retorna el valor 0 al procés que la va cridar.

Tipus de Dades i Variables

En C, els tipus de dades especifiquen el tipus de dades que una variable pot contenir. Els tipus de dades comuns inclouen:

  • int: Tipus enter
  • float: Tipus de punt flotant
  • char: Tipus de caràcter
  • double: Tipus de punt flotant de doble precisió

Declaració de Variables

Les variables s'han de declarar abans de ser utilitzades. La sintaxi per declarar una variable és:

data_type variable_name;

Exemple:

int age;
float salary;
char grade;
double distance;

Operacions Bàsiques d'Entrada/Sortida

C proporciona funcions estàndard per a operacions d'entrada i sortida, principalment a través de la biblioteca stdio.h.

Entrada utilitzant `scanf`

La funció scanf s'utilitza per llegir entrada formatada des de l'entrada estàndard (teclat).

int number;
scanf("%d", &number); // Llegeix un enter i l'emmagatzema a la variable 'number'

Sortida utilitzant `printf`

La funció printf s'utilitza per imprimir sortida formatada a la sortida estàndard (consola).

int number = 10;
printf("The number is: %d\n", number); // Imprimeix "The number is: 10"

Comentaris

Els comentaris s'utilitzen per explicar el codi i són ignorats pel compilador. Hi ha dos tipus de comentaris en C:

  • Comentaris d'una sola línia: Comencen amb //
  • Comentaris de múltiples línies: Envolten dins de /* ... */

Exemple:

// Aquest és un comentari d'una sola línia

/*
Aquest és un
comentari de múltiples línies
*/

Resum

En aquesta secció, hem cobert la sintaxi i l'estructura bàsiques d'un programa en C. Hem après sobre els components essencials com les comandes del preprocessador, funcions, variables i declaracions. També hem explorat operacions bàsiques d'entrada/sortida utilitzant scanf i printf, i l'ús de comentaris per fer el codi més llegible. Entendre aquests fonaments és el primer pas per convertir-se en un programador competent en C.
