Introducció

Les funcions matemàtiques són essencials en la programació per realitzar diversos càlculs. En C, la biblioteca estàndard proporciona una àmplia gamma de funcions matemàtiques que es poden utilitzar per realitzar operacions com trigonometria, exponenciació i logaritmes.

Conceptes Clau

  • Biblioteca Estàndard: El fitxer de capçalera math.h conté les declaracions per a les funcions matemàtiques.
  • Prototips de Funcions: Entendre els prototips d'aquestes funcions és crucial per al seu ús correcte.
  • Tipus de Retorn: La majoria de funcions matemàtiques retornen valors de tipus double.
  • Gestió d'Errors: Algunes funcions poden retornar valors especials com NaN (Not a Number) o INF (Infinity) en cas d'errors.

Funcions Matemàtiques Comunes

Funcions Aritmètiques Bàsiques

  • fabs(double x): Retorna el valor absolut de x.
  • fmod(double x, double y): Retorna el residu de x dividit per y.
#include <stdio.h>
#include <math.h>

int main() {
    double x = -5.3;
    double y = 2.0;

    printf("El valor absolut de %.2f és %.2f\n", x, fabs(x));
    printf("El residu de %.2f dividit per %.2f és %.2f\n", x, y, fmod(x, y));

    return 0;
}

Funcions Exponencials i Logarítmiques

  • exp(double x): Retorna e elevat a la potència de x.
  • log(double x): Retorna el logaritme natural (base e) de x.
  • log10(double x): Retorna el logaritme en base 10 de x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 2.0;

    printf("e elevat a la potència de %.2f és %.2f\n", x, exp(x));
    printf("El logaritme natural de %.2f és %.2f\n", x, log(x));
    printf("El logaritme en base 10 de %.2f és %.2f\n", x, log10(x));

    return 0;
}

Funcions de Potència

  • pow(double base, double exponent): Retorna base elevat a la potència de exponent.
  • sqrt(double x): Retorna l'arrel quadrada de x.
#include <stdio.h>
#include <math.h>

int main() {
    double base = 3.0;
    double exponent = 4.0;

    printf("%.2f elevat a la potència de %.2f és %.2f\n", base, exponent, pow(base, exponent));
    printf("L'arrel quadrada de %.2f és %.2f\n", base, sqrt(base));

    return 0;
}

Funcions Trigonomètriques

  • sin(double x): Retorna el sinus de x (x en radians).
  • cos(double x): Retorna el cosinus de x (x en radians).
  • tan(double x): Retorna la tangent de x (x en radians).
#include <stdio.h>
#include <math.h>

int main() {
    double angle = M_PI / 4; // 45 graus en radians

    printf("El sinus de %.2f radians és %.2f\n", angle, sin(angle));
    printf("El cosinus de %.2f radians és %.2f\n", angle, cos(angle));
    printf("La tangent de %.2f radians és %.2f\n", angle, tan(angle));

    return 0;
}

Funcions Hiperbòliques

  • sinh(double x): Retorna el sinus hiperbòlic de x.
  • cosh(double x): Retorna el cosinus hiperbòlic de x.
  • tanh(double x): Retorna la tangent hiperbòlica de x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 1.0;

    printf("El sinus hiperbòlic de %.2f és %.2f\n", x, sinh(x));
    printf("El cosinus hiperbòlic de %.2f és %.2f\n", x, cosh(x));
    printf("La tangent hiperbòlica de %.2f és %.2f\n", x, tanh(x));

    return 0;
}

Gestió d'Errors

  • Errors de Domini: Es produeixen quan els valors d'entrada estan fora del domini sobre el qual la funció està definida.
  • Errors de Rang: Es produeixen quan el resultat d'una funció no es pot representar dins del rang del tipus de retorn.

Exemple de Gestió d'Errors

#include <stdio.h>
#include <math.h>
#include <errno.h>

int main() {
    errno = 0; // Reiniciar l'indicador global d'errors
    double result = sqrt(-1.0);

    si (errno == EDOM) {
        printf("S'ha produït un error de domini\n");
    } else {
        printf("L'arrel quadrada és %.2f\n", result);
    }

    return 0;
}

Conclusió

Les funcions matemàtiques en C són eines poderoses per realitzar una àmplia gamma de càlculs. Entenent les funcions de la biblioteca estàndard, els seus prototips i com gestionar els errors, pots utilitzar aquestes funcions de manera efectiva en els teus programes. Practica utilitzant aquestes funcions amb diferents entrades per obtenir una comprensió més profunda i convertir-te en un expert en el seu ús.
