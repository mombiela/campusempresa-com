Què són les Estructures en C?

Les estructures en C són tipus de dades definits per l'usuari que permeten combinar elements de dades de diferents tipus. Les estructures s'utilitzen per representar un registre, per exemple, un registre d'estudiant podria contenir un nom, edat i notes.

Conceptes Clau

  • Definició: Una estructura es defineix utilitzant la paraula clau struct.
  • Membres: Les variables declarades dins d'una estructura s'anomenen membres.
  • Accés als Membres: Els membres d'una estructura s'accedeixen utilitzant l'operador punt (.).
  • Inicialització: Les estructures es poden inicialitzar en el moment de la declaració.

Definint una Estructura

Per definir una estructura, utilitzeu la paraula clau struct seguida del nom de l'estructura i els membres de l'estructura tancats entre claus {}.

struct Student {
    char name[50];
    int age;
    float marks;
};

En aquest exemple:

  • Student és el nom de l'estructura.
  • name, age i marks són membres de l'estructura.

Declarant Variables d'Estructura

Després de definir una estructura, podeu declarar variables d'aquest tipus.

struct Student student1, student2;

Aquí, student1 i student2 són variables de tipus struct Student.

Accedint als Membres de l'Estructura

Podeu accedir als membres d'una estructura utilitzant l'operador punt (.).

student1.age = 20;
student1.marks = 85.5;
strcpy(student1.name, "John Doe");

En aquest exemple:

  • student1.age estableix l'edat de student1 a 20.
  • student1.marks estableix les notes de student1 a 85.5.
  • strcpy(student1.name, "John Doe") copia la cadena "John Doe" al membre name de student1.

Inicialitzant Estructures

Les estructures es poden inicialitzar en el moment de la declaració.

struct Student student1 = {"John Doe", 20, 85.5};

Això inicialitza student1 amb el nom "John Doe", edat 20, i notes 85.5.

Programa Exemple

Aquí teniu un programa complet que demostra l'ús de les estructures.

#include <stdio.h>
#include <string.h>

// Defineix l'estructura
struct Student {
    char name[50];
    int age;
    float marks;
};

int main() {
    // Declara i inicialitza variables d'estructura
    struct Student student1 = {"John Doe", 20, 85.5};
    struct Student student2;

    // Assigna valors a student2
    strcpy(student2.name, "Jane Smith");
    student2.age = 22;
    student2.marks = 90.0;

    // Imprimeix els detalls de student1
    printf("Student 1:\n");
    printf("Name: %s\n", student1.name);
    printf("Age: %d\n", student1.age);
    printf("Marks: %.2f\n", student1.marks);

    // Imprimeix els detalls de student2
    printf("Student 2:\n");
    printf("Name: %s\n", student2.name);
    printf("Age: %d\n", student2.age);
    printf("Marks: %.2f\n", student2.marks);

    return 0;
}

Conclusió

Les estructures en C proporcionen una manera d'agrupar elements de dades relacionats de diferents tipus en una sola unitat. Això és particularment útil per representar estructures de dades complexes com registres. Entenent com definir, declarar, inicialitzar i accedir a les estructures, podeu gestionar i manipular dades relacionades de manera més efectiva en els vostres programes.
