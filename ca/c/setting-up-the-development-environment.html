Configuració de l'Entorn de Desenvolupament

Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre programació en C. Aquesta secció et guiarà a través del procés d'instal·lació de les eines necessàries i la configuració del teu sistema per escriure, compilar i executar programes en C.

Triar un IDE o Editor de Text

Abans de començar a escriure codi en C, necessites un lloc per escriure'l. Pots triar entre un Entorn de Desenvolupament Integrat (IDE) o un simple editor de text.

  • IDEs: Proporcionen un entorn complet amb funcions com la completació de codi, depuració i gestió de projectes.
    • Exemples: Visual Studio, Code::Blocks, Eclipse CDT.
  • Editors de Text: Lleugers i flexibles, sovint utilitzats amb eines de línia de comandes.
    • Exemples: Visual Studio Code, Sublime Text, Atom.

Instal·lar Visual Studio Code (VS Code)

VS Code és un editor de text popular que admet la programació en C a través d'extensions.

  1. Descarregar i Instal·lar VS Code:

    • Visita el lloc web de VS Code.
    • Descarrega l'instal·lador per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació.

  2. Instal·lar l'Extensió C/C++:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitats al costat de la finestra.
    • Cerca "C/C++" i instal·la l'extensió proporcionada per Microsoft.

Instal·lar Code::Blocks

Code::Blocks és un IDE gratuït i de codi obert dissenyat específicament per al desenvolupament en C i C++.

  1. Descarregar i Instal·lar Code::Blocks:
    • Visita el lloc web de Code::Blocks.
    • Descarrega l'instal·lador que inclou el compilador MinGW.
    • Segueix les instruccions d'instal·lació.

Instal·lar un Compilador de C

Un compilador és necessari per convertir el teu codi en C en un programa executable. Els compiladors comuns inclouen GCC (GNU Compiler Collection) i Clang.

Instal·lar GCC a Windows

  1. Descarregar MinGW:

    • Visita el lloc web de MinGW.
    • Descarrega el Gestor d'Instal·lació de MinGW.

  2. Instal·lar MinGW:

    • Executa l'instal·lador i selecciona els paquets gcc i g++.
    • Segueix les instruccions d'instal·lació.

  3. Configurar Variables d'Entorn:

    • Afegeix el camí al directori bin de MinGW a la variable d'entorn PATH del teu sistema.

Instal·lar GCC a macOS

  1. Instal·lar les Eines de Línia de Comandes de Xcode:
    • Obre Terminal.
    • Executa la comanda: xcode-select --install.

Instal·lar GCC a Linux

  1. Utilitzar el Gestor de Paquets:
    • Per a sistemes basats en Debian (com Ubuntu): sudo apt-get install build-essential.
    • Per a sistemes basats en Red Hat (com Fedora): sudo dnf install gcc.

Verificar la Instal·lació

Després d'instal·lar el teu compilador, és important verificar que tot estigui configurat correctament.

Escriure un Programa Simple en C

Crea un nou fitxer anomenat hello.c i afegeix el següent codi:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Compilar i Executar el Programa

  1. Obre un Terminal o Símbol del Sistema.
  2. Navega al directori que conté hello.c.
  3. Compila el Programa:
    • Per a GCC: gcc hello.c -o hello
    • Per a Clang: clang hello.c -o hello
  4. Executa el Programa:
    • A Windows: hello
    • A macOS/Linux: ./hello

Hauries de veure la sortida:

Hello, World!

Conclusió

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un programador proficient en C. Triant les eines adequades i assegurant-te que el teu compilador està correctament instal·lat, crees una base sòlida per al teu viatge d'aprenentatge. Tant si prefereixes un IDE complet com Code::Blocks com un editor de text flexible com VS Code, la clau és assegurar-te que pots escriure, compilar i executar els teus programes en C de manera eficient.
