Introducció

Els tipus de dades en C són fonamentals per entendre com emmagatzemar, manipular i utilitzar dades en els teus programes. Aquesta secció cobrirà els tipus de dades bàsics i avançats en C, el seu ús i com triar el tipus de dada adequat per a les teves necessitats.

Tipus de Dades Bàsics

C proporciona diversos tipus de dades bàsics que s'utilitzen per definir variables i el tipus de dades que poden contenir.

Tipus Enter

  • int: S'utilitza per emmagatzemar enters (nombres sencers).
  • short: Una versió més curta d'int, típicament utilitzant menys memòria.
  • long: Una versió més llarga d'int, típicament utilitzant més memòria.
  • long long: Una versió encara més llarga d'int per a nombres molt grans.
int a = 10;
short b = 20;
long c = 30000;
long long d = 4000000000;

Tipus de Punt Flotant

  • float: S'utilitza per emmagatzemar nombres de punt flotant de precisió simple.
  • double: S'utilitza per emmagatzemar nombres de punt flotant de doble precisió.
  • long double: S'utilitza per a nombres de punt flotant de precisió estesa.
float e = 3.14f;
double f = 3.141592653589793;
long double g = 3.14159265358979323846L;

Tipus de Caràcter

  • char: S'utilitza per emmagatzemar caràcters simples. També es pot utilitzar per emmagatzemar enters petits.
char h = 'A';
char i = 65; // Valor ASCII de 'A'

Tipus Booleà

  • _Bool: S'utilitza per emmagatzemar valors booleans (cert o fals).
_Bool j = 1; // cert
_Bool k = 0; // fals

Tipus de Dades Derivats

Els tipus de dades derivats es construeixen a partir dels tipus de dades bàsics i inclouen arrays, punters, estructures i unions.

Arrays

Els arrays són col·leccions d'elements del mateix tipus.

int arr[5] = {1, 2, 3, 4, 5};
char str[] = "Hola";

Punters

Els punters emmagatzemen l'adreça de memòria d'una altra variable.

int l = 10;
int *ptr = &l;

Estructures

Les estructures s'utilitzen per agrupar diferents tipus de variables sota un mateix nom.

struct Person {
    char name[50];
    int age;
};

struct Person person1;
person1.age = 30;

Unions

Les unions són similars a les estructures però utilitzen la mateixa ubicació de memòria per a tots els seus membres.

union Data {
    int intVal;
    float floatVal;
    char charVal;
};

union Data data;
data.intVal = 10;

Tipus Enumerats

Els tipus enumerats (enums) són tipus definits per l'usuari que consisteixen en constants enters amb nom.

enum Weekday {Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday};
enum Weekday today = Wednesday;

Qualificadors de Tipus

Els qualificadors de tipus afegeixen propietats especials a les variables.

  • const: Indica que el valor de la variable no es pot canviar.
  • volatile: Indica que el valor de la variable pot canviar en qualsevol moment sense cap acció per part del codi.
  • restrict: Un qualificador de punter que indica que el punter és l'única referència a aquesta ubicació de memòria.
const int m = 100;
volatile int n = 200;
int * restrict p = &n;

Resum

Entendre els tipus de dades en C és crucial per a una programació eficient. Els tipus de dades bàsics inclouen enters, nombres de punt flotant, caràcters i booleans. Els tipus de dades derivats inclouen arrays, punters, estructures i unions, mentre que els tipus enumerats permeten constants amb nom definides per l'usuari. Els qualificadors de tipus afegeixen propietats addicionals a les variables, millorant el control sobre com es fan servir i s'emmagatzemen les dades. En dominar aquests conceptes, podràs escriure programes en C més eficients i efectius.
