Introducció

En aquest tema, explorarem dues estructures de dades fonamentals en programació: els arrays i les llistes. Aquestes estructures són essencials per emmagatzemar i manipular col·leccions de dades de manera eficient. Aprendrem les seves característiques, com s'utilitzen i les diferències clau entre elles.

Arrays

Definició

Un array és una estructura de dades que conté un nombre fix d'elements del mateix tipus. Els elements s'emmagatzemen en ubicacions de memòria contigües i s'accedeix a ells mitjançant índexs.

Característiques

  • Mida fixa: La mida de l'array es defineix al moment de la seva creació i no pot canviar.
  • Accés ràpid: Es pot accedir a qualsevol element en temps constant O(1) utilitzant el seu índex.
  • Tipus homogeni: Tots els elements de l'array han de ser del mateix tipus de dades.

Exemple de Codi

# Declaració d'un array en Python
array = [1, 2, 3, 4, 5]

# Accés a elements de l'array
print(array[0])  # Imprimeix: 1
print(array[3])  # Imprimeix: 4

# Modificació d'un element de l'array
array[2] = 10
print(array)  # Imprimeix: [1, 2, 10, 4, 5]

Avantatges i Desavantatges

| Avantatges | Desavantatges | |-------------------------------|------------------------------------| | Accés ràpid a elements | Mida fixa | | Eficiència en l'ús de memòria | No permet insercions o eliminacions dinàmiques |

Llistes

Definició

Una llista és una estructura de dades que pot contenir un nombre variable d'elements, i aquests elements poden ser de diferents tipus. Les llistes són més flexibles que els arrays i permeten operacions dinàmiques com insercions i eliminacions.

Característiques

  • Mida dinàmica: Les llistes poden créixer o reduir-se segons sigui necessari.
  • Accés seqüencial: Encara que es pot accedir a qualsevol element per índex, les operacions d'inserció i eliminació poden ser més costoses en termes de temps.
  • Tipus heterogeni: Les llistes poden contenir elements de diferents tipus de dades.

Exemple de Codi

# Declaració d'una llista en Python
llista = [1, "dos", 3.0, True]

# Accés a elements de la llista
print(llista[1])  # Imprimeix: dos
print(llista[3])  # Imprimeix: True

# Modificació d'un element de la llista
llista[2] = "tres"
print(llista)  # Imprimeix: [1, "dos", "tres", True]

# Inserció d'un nou element
llista.append(4)
print(llista)  # Imprimeix: [1, "dos", "tres", True, 4]

# Eliminació d'un element
llista.remove("dos")
print(llista)  # Imprimeix: [1, "tres", True, 4]

Avantatges i Desavantatges

| Avantatges | Desavantatges | |-------------------------------|------------------------------------| | Mida dinàmica | Accés més lent a elements | | Permet insercions i eliminacions dinàmiques | Major ús de memòria en comparació amb arrays |

Comparació entre Arrays i Llistes

| Característica | Arrays | Llistes | |-------------------------------|------------------------------------|------------------------------------| | Mida | Fixa | Dinàmica | | Tipus d'elements | Homogeni | Heterogeni | | Accés a elements | Ràpid (O(1)) | Més lent (O(n) en el pitjor cas) | | Insercions i eliminacions | No permeses | Permeïdes |

Conclusió

Els arrays i les llistes són estructures de dades fonamentals que tot programador ha de conèixer. Els arrays són ideals per a situacions on es necessita un accés ràpid i la mida de la col·lecció és coneguda d'antuvi. Les llistes, d'altra banda, ofereixen més flexibilitat i són útils quan es requereix una mida dinàmica i la capacitat de realitzar insercions i eliminacions.

Comprendre quan i com utilitzar cadascuna d'aquestes estructures és crucial per escriure codi eficient i efectiu. Practica amb els exemples proporcionats i experimenta amb les teves pròpies implementacions per enfortir la teva comprensió.
