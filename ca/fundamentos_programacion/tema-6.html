Introducció

En programació, els operadors i les expressions són fonamentals per realitzar càlculs, prendre decisions i manipular dades. Aquest tema cobreix els diferents tipus d'operadors disponibles en els llenguatges de programació i com s'utilitzen per formar expressions.

Tipus d'Operadors

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • Suma (+): Suma dos operandes.
  • Resta (-): Resta el segon operant del primer.
  • Multiplicació (*): Multiplica dos operandes.
  • Divisió (/): Divideix el primer operant pel segon.
  • Mòdul (%): Retorna el restant de la divisió del primer operant pel segon.
# Exemple d'operadors aritmètics en Python
a = 10
b = 3

print(a + b)  # Suma: 13
print(a - b)  # Resta: 7
print(a * b)  # Multiplicació: 30
print(a / b)  # Divisió: 3.3333...
print(a % b)  # Mòdul: 1

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a les variables.

  • Assignació simple (=): Assigna el valor de l'operant dret a l'operant esquerre.
  • Assignació amb suma (+=): Suma l'operant dret a l'operant esquerre i assigna el resultat a l'operant esquerre.
  • Assignació amb resta (-=): Resta l'operant dret de l'operant esquerre i assigna el resultat a l'operant esquerre.
  • Assignació amb multiplicació (*=): Multiplica l'operant dret per l'operant esquerre i assigna el resultat a l'operant esquerre.
  • Assignació amb divisió (/=): Divideix l'operant esquerre per l'operant dret i assigna el resultat a l'operant esquerre.
# Exemple d'operadors d'assignació en Python
x = 5
x += 3  # Equivalent a x = x + 3
print(x)  # Resultat: 8

x -= 2  # Equivalent a x = x - 2
print(x)  # Resultat: 6

x *= 4  # Equivalent a x = x * 4
print(x)  # Resultat: 24

x /= 3  # Equivalent a x = x / 3
print(x)  # Resultat: 8.0

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors i retornar un valor booleà (True o False).

  • Igual a (==): Retorna True si ambdós operants són iguals.
  • Diferent de (!=): Retorna True si els operants no són iguals.
  • Major que (>): Retorna True si l'operant esquerre és major que el dret.
  • Menor que (<): Retorna True si l'operant esquerre és menor que el dret.
  • Major o igual que (>=): Retorna True si l'operant esquerre és major o igual que el dret.
  • Menor o igual que (<=): Retorna True si l'operant esquerre és menor o igual que el dret.
# Exemple d'operadors de comparació en Python
a = 10
b = 20

print(a == b)  # Igual a: False
print(a != b)  # Diferent de: True
print(a > b)   # Major que: False
print(a < b)   # Menor que: True
print(a >= b)  # Major o igual que: False
print(a <= b)  # Menor o igual que: True

Operadors Lògics

Els operadors lògics s'utilitzen per combinar expressions booleanes.

  • AND (and): Retorna True si ambdues expressions són True.
  • OR (or): Retorna True si almenys una de les expressions és True.
  • NOT (not): Inverteix el valor de l'expressió booleana.
# Exemple d'operadors lògics en Python
a = True
b = False

print(a and b)  # AND: False
print(a or b)   # OR: True
print(not a)    # NOT: False

Expressions

Una expressió és una combinació de valors, variables, operadors i funcions que s'avaluen per produir un altre valor.

Exemples d'Expressions

  • Expressió aritmètica: 3 + 5 * 2 s'avalua com 13.
  • Expressió de comparació: 10 > 5 s'avalua com True.
  • Expressió lògica: (10 > 5) and (3 < 8) s'avalua com True.
# Exemple d'expressions en Python
result = (10 + 5) * 2  # Expressió aritmètica
print(result)  # Resultat: 30

is_greater = 10 > 5  # Expressió de comparació
print(is_greater)  # Resultat: True

logical_expression = (10 > 5) and (3 < 8)  # Expressió lògica
print(logical_expression)  # Resultat: True

Conclusió

Els operadors i les expressions són components essencials en qualsevol llenguatge de programació. Comprendre com utilitzar-los correctament és crucial per realitzar càlculs, prendre decisions i manipular dades de manera efectiva. Practicar amb diferents tipus d'operadors i expressions ajudarà a enfortir les teves habilitats de programació i a resoldre problemes de manera més eficient.
