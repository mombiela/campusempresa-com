Introducció

L'àmbit de variables és un concepte fonamental en programació que defineix l'accessibilitat i la vida útil de les variables dins d'un programa. Comprendre l'àmbit de les variables és crucial per evitar errors, com l'ús de variables no inicialitzades o la sobrescriptura accidental de dades.

Tipus d'Àmbit

Hi ha diversos tipus d'àmbit de variables, cadascun amb les seves pròpies regles i usos. Els principals tipus són:

  • Àmbit Local
  • Àmbit Global
  • Àmbit de Bloc
  • Àmbit de Paràmetre

Àmbit Local

Les variables locals són aquelles que es declaren dins d'una funció o un mètode. Només són accessibles dins d'aquesta funció o mètode i deixen d'existir quan la funció acaba la seva execució.

def funcio_exemple():
    variable_local = 10
    print(variable_local)  # Sortida: 10

funcio_exemple()
# print(variable_local)  # Error: variable_local no està definida fora de la funció

Àmbit Global

Les variables globals es declaren fora de qualsevol funció o mètode i són accessibles des de qualsevol part del programa. No obstant això, el seu ús ha de ser curós per evitar conflictes i errors difícils de depurar.

variable_global = 20

def funcio_exemple():
    print(variable_global)  # Sortida: 20

funcio_exemple()
print(variable_global)  # Sortida: 20

Àmbit de Bloc

L'àmbit de bloc es refereix a variables que es declaren dins de blocs de codi, com bucles o condicionals. En alguns llenguatges, aquestes variables només són accessibles dins del bloc on es van declarar.

if (true) {
    let variable_bloc = 30;
    console.log(variable_bloc);  // Sortida: 30
}
// console.log(variable_bloc);  // Error: variable_bloc no està definida fora del bloc

Àmbit de Paràmetre

Els paràmetres d'una funció tenen un àmbit local dins d'aquesta funció. Són accessibles només dins de la funció i s'utilitzen per rebre valors quan la funció és cridada.

def funcio_exemple(parametre):
    print(parametre)  # Sortida: el valor passat com a argument

funcio_exemple(40)  # Sortida: 40

Comparació d'Àmbits

| Tipus d'Àmbit | Declaració | Accessibilitat | Vida Útil | |------------------|-------------------------------|----------------------------------------|-------------------------------------| | Local | Dins d'una funció/mètode | Només dins de la funció/mètode | Fins que la funció/mètode acaba | | Global | Fora de qualsevol funció | En qualsevol part del programa | Durant tota l'execució del programa | | Bloc | Dins d'un bloc de codi | Només dins del bloc | Fins que el bloc acaba | | Paràmetre | En la definició de la funció | Només dins de la funció | Fins que la funció acaba |

Exemples Pràctics

Exemple 1: Àmbit Local i Global en Python

variable_global = 50

def funcio_exemple():
    variable_local = 60
    print(variable_local)  # Sortida: 60
    print(variable_global)  # Sortida: 50

funcio_exemple()
print(variable_global)  # Sortida: 50
# print(variable_local)  # Error: variable_local no està definida fora de la funció

Exemple 2: Àmbit de Bloc en JavaScript

let variable_global = 70;

if (true) {
    let variable_bloc = 80;
    console.log(variable_bloc);  // Sortida: 80
}

console.log(variable_global);  // Sortida: 70
// console.log(variable_bloc);  // Error: variable_bloc no està definida fora del bloc

Conclusió

L'àmbit de variables és un concepte essencial en programació que ajuda a gestionar l'accessibilitat i la vida útil de les variables. Comprendre els diferents tipus d'àmbit i les seves regles permet escriure codi més net, eficient i lliure d'errors. Assegura't de practicar amb exemples i exercicis per consolidar la teva comprensió d'aquest tema fonamental.
