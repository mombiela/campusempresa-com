Introducció

Les estructures de control de selecció permeten que un programa prengui decisions i executi diferents blocs de codi en funció de certes condicions. Aquestes estructures són fonamentals per crear programes dinàmics i adaptables. En aquest tema, explorarem les estructures de control de selecció més comunes: if, else if, else i switch.

Estructura `if`

L'estructura if avalua una condició i executa un bloc de codi si la condició és verdadera.

Sintaxi

if condició:
    # Bloc de codi a executar si la condició és verdadera

Exemple

edat = 18
if edat >= 18:
    print("Ets major d'edat.")

En aquest exemple, el missatge "Ets major d'edat." s'imprimirà perquè la condició edat >= 18 és verdadera.

Estructura `else`

L'estructura else s'utilitza juntament amb if per executar un bloc de codi alternatiu si la condició és falsa.

Sintaxi

if condició:
    # Bloc de codi a executar si la condició és verdadera
else:
    # Bloc de codi a executar si la condició és falsa

Exemple

edat = 16
if edat >= 18:
    print("Ets major d'edat.")
else:
    print("Ets menor d'edat.")

En aquest cas, el missatge "Ets menor d'edat." s'imprimirà perquè la condició edat >= 18 és falsa.

Estructura `else if` (`elif` en Python)

L'estructura else if (o elif en Python) permet avaluar múltiples condicions seqüencialment.

Sintaxi

if condició1:
    # Bloc de codi a executar si condició1 és verdadera
elif condició2:
    # Bloc de codi a executar si condició2 és verdadera
else:
    # Bloc de codi a executar si cap condició anterior és verdadera

Exemple

nota = 85
if nota >= 90:
    print("Excel·lent")
elif nota >= 80:
    print("Molt bé")
elif nota >= 70:
    print("Bé")
else:
    print("Necessites millorar")

En aquest exemple, el missatge "Molt bé" s'imprimirà perquè la condició nota >= 80 és verdadera.

Estructura `switch` (disponible en alguns llenguatges)

L'estructura switch permet seleccionar entre múltiples blocs de codi basats en el valor d'una expressió. No tots els llenguatges de programació suporten switch.

Sintaxi (Exemple en JavaScript)

switch (expressió) {
    case valor1:
        // Bloc de codi a executar si expressió === valor1
        break;
    case valor2:
        // Bloc de codi a executar si expressió === valor2
        break;
    default:
        // Bloc de codi a executar si cap condició anterior és verdadera
}

Exemple

let dia = 3;
switch (dia) {
    case 1:
        console.log("Dilluns");
        break;
    case 2:
        console.log("Dimarts");
        break;
    case 3:
        console.log("Dimecres");
        break;
    default:
        console.log("Dia no vàlid");
}

En aquest exemple, el missatge "Dimecres" s'imprimirà perquè dia és igual a 3.

Comparació d'estructures de selecció

| Estructura | Ús principal | Avantatges | Desavantatges | |------------|--------------|------------|---------------| | if | Avaluar una sola condició | Simple i directe | Pot tornar-se complex amb múltiples condicions | | else | Proveir una alternativa si if és fals | Complementa if | No es pot usar sol | | else if | Avaluar múltiples condicions seqüencials | Més clar que múltiples if | Pot tornar-se llarg | | switch | Seleccionar entre múltiples valors d'una expressió | Més llegible per múltiples valors | No disponible en tots els llenguatges |

Conclusió

Les estructures de control de selecció són essencials per a la presa de decisions en la programació. Comprendre com i quan usar if, else, else if i switch et permetrà escriure codi més flexible i eficient. Practica amb diferents exemples i escenaris per dominar aquestes estructures i aplicar-les adequadament en els teus projectes.
