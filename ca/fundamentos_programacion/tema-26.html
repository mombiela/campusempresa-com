Introducció

En aquesta secció, presentarem el projecte que desenvoluparem al llarg del curs de Fonaments de la Programació. Aquest projecte servirà com una aplicació pràctica dels conceptes que aprendrem, permetent-te consolidar els teus coneixements i habilitats en programació.

Objectius del Projecte

  • Aplicar conceptes bàsics de programació: Variables, estructures de control, funcions, etc.
  • Desenvolupar habilitats de resolució de problemes: Analitzar i descompondre problemes complexos en parts manejables.
  • Fomentar bones pràctiques de codificació: Escriure codi net, comentat i ben estructurat.
  • Introduir eines i entorns de desenvolupament: Familiaritzar-se amb IDEs, sistemes de control de versions, etc.

Descripció del Projecte

El projecte consistirà a desenvolupar una aplicació de gestió de tasques. Aquesta aplicació permetrà als usuaris:

  • Crear tasques: Afegir noves tasques amb detalls com títol, descripció i data de venciment.
  • Llistar tasques: Veure totes les tasques creades, amb opcions per filtrar i ordenar.
  • Actualitzar tasques: Modificar els detalls d'una tasca existent.
  • Eliminar tasques: Esborrar tasques que ja no siguin necessàries.

Requisits del Projecte

Funcionalitats Bàsiques

  • Interfície d'Usuari: Una interfície simple que permeti la interacció amb les funcionalitats esmentades.
  • Persistència de Dades: Emmagatzemar les tasques en un fitxer o base de dades perquè persisteixin entre sessions.
  • Validació de Dades: Assegurar-se que les dades ingressades per l'usuari siguin vàlides i coherents.

Tecnologies i Eines

  • Llenguatge de Programació: Python
  • Entorn de Desenvolupament: Visual Studio Code (VSCode)
  • Control de Versions: Git i GitHub

Estructura del Projecte

Mòduls i Components

  • main.py: Punt d'entrada de l'aplicació.
  • task_manager.py: Mòdul que conté la lògica per gestionar les tasques.
  • ui.py: Mòdul que maneja la interfície d'usuari.
  • storage.py: Mòdul responsable de la persistència de dades.

Exemple de Codi

A continuació, es mostra un exemple de com podria semblar l'estructura bàsica de task_manager.py:

# task_manager.py

tasks = []

def add_task(title, description, due_date):
    task = {
        'title': title,
        'description': description,
        'due_date': due_date,
        'completed': False
    }
    tasks.append(task)

def list_tasks():
    for task in tasks:
        print(f"Title: {task['title']}, Due Date: {task['due_date']}, Completed: {task['completed']}")

def update_task(index, title=None, description=None, due_date=None, completed=None):
    if index < len(tasks):
        if title:
            tasks[index]['title'] = title
        if description:
            tasks[index]['description'] = description
        if due_date:
            tasks[index]['due_date'] = due_date
        if completed is not None:
            tasks[index]['completed'] = completed

def delete_task(index):
    if index < len(tasks):
        tasks.pop(index)

Exercici Pràctic

  1. Crear una Tasca: Implementa la funció add_task a task_manager.py.
  2. Llistar Tasques: Implementa la funció list_tasks i prova llistar les tasques creades.
  3. Actualitzar una Tasca: Implementa la funció update_task i prova actualitzar una tasca existent.
  4. Eliminar una Tasca: Implementa la funció delete_task i prova eliminar una tasca.
