Introducció a les funcions

Les funcions són blocs de codi reutilitzables que realitzen una tasca específica. En programació, les funcions permeten organitzar i estructurar el codi de manera eficient, facilitant el seu manteniment i comprensió.

Conceptes clau

  • Definició de funció: Especifica el nom de la funció, els paràmetres que accepta i el bloc de codi que executa.
  • Crida a funció: És el procés d'invocar una funció perquè executi el seu bloc de codi.
  • Paràmetres i arguments: Els paràmetres són variables que es defineixen en la declaració de la funció. Els arguments són els valors que es passen a la funció quan es crida.
  • Valor de retorn: És el valor que una funció pot retornar després de la seva execució.

Definició de funcions

Per definir una funció en un llenguatge de programació, generalment es segueix una sintaxi específica. A continuació, es mostra com definir una funció en Python:

def nom_de_la_funcio(parametre1, parametre2):
    # Bloc de codi
    resultat = parametre1 + parametre2
    return resultat

Exemple explicat

def sumar(a, b):
    resultat = a + b
    return resultat
  • def: Paraula clau que indica la definició d'una funció.
  • sumar: Nom de la funció.
  • (a, b): Paràmetres de la funció.
  • resultat = a + b: Bloc de codi que realitza la suma de a i b.
  • return resultat: Retorna el resultat de la suma.

Ús de funcions

Un cop una funció ha estat definida, es pot cridar en qualsevol part del codi. A continuació, es mostra com cridar la funció sumar definida anteriorment:

resultat_suma = sumar(5, 3)
print(resultat_suma)  # Sortida: 8

Exemple explicat

  • sumar(5, 3): Crida a la funció sumar amb els arguments 5 i 3.
  • resultat_suma: Variable que emmagatzema el valor retornat per la funció sumar.
  • print(resultat_suma): Imprimeix el resultat de la suma, que és 8.

Paràmetres i arguments

Les funcions poden acceptar múltiples paràmetres i es poden passar diferents tipus d'arguments:

Exemple amb múltiples paràmetres

def multiplicar(x, y, z):
    return x * y * z

resultat_multiplicacio = multiplicar(2, 3, 4)
print(resultat_multiplicacio)  # Sortida: 24

Exemple amb diferents tipus d'arguments

def saludar(nom, edat):
    return f"Hola, {nom}. Tens {edat} anys."

missatge = saludar("Ana", 30)
print(missatge)  # Sortida: Hola, Ana. Tens 30 anys.

Valor de retorn

Les funcions poden retornar valors que després poden ser utilitzats en altres parts del programa. Si una funció no té una declaració return, retornarà None per defecte en Python.

Exemple sense valor de retorn

def imprimir_missatge():
    print("Aquest és un missatge.")

imprimir_missatge()  # Sortida: Aquest és un missatge.

Exemple amb valor de retorn

def quadrat(numero):
    return numero * numero

resultat_quadrat = quadrat(5)
print(resultat_quadrat)  # Sortida: 25

Conclusió

Les funcions són una eina fonamental en la programació que permeten modularitzar i reutilitzar el codi. Comprendre com definir i utilitzar funcions és essencial per escriure codi eficient i mantenible. Al llarg d'aquest tema, hem vist com definir funcions, passar paràmetres, retornar valors i cridar funcions en Python. Practicar aquests conceptes t'ajudarà a desenvolupar habilitats sòlides en programació.
