Introducció

Els llenguatges de programació són eines fonamentals en el desenvolupament de programari. Permeten als programadors comunicar-se amb els ordinadors per realitzar tasques específiques. En aquesta secció, explorarem els conceptes bàsics dels llenguatges de programació, els seus tipus i característiques, així com alguns exemples pràctics.

Tipus de llenguatges de programació

Hi ha diversos tipus de llenguatges de programació, cadascun amb les seves pròpies característiques i usos específics. A continuació, es presenten els tipus més comuns:

  • Llenguatges de baix nivell: Són llenguatges que estan més a prop del llenguatge màquina. Exemples inclouen el llenguatge assemblador.
  • Llenguatges d'alt nivell: Són més abstractes i fàcils d'entendre per als humans. Exemples inclouen Python, Java, i C++.
  • Llenguatges de scripting: S'utilitzen principalment per automatitzar tasques. Exemples inclouen JavaScript, Perl, i Ruby.
  • Llenguatges de programació funcional: Es centren en l'ús de funcions matemàtiques. Exemples inclouen Haskell i Lisp.
  • Llenguatges de programació orientada a objectes: Es centren en la creació d'objectes que contenen dades i mètodes. Exemples inclouen Java i C#.

Característiques dels llenguatges de programació

Cada llenguatge de programació té característiques que el fan adequat per a certs tipus de tasques. Algunes de les característiques clau inclouen:

  • Sintaxi: L'estructura i regles que defineixen com escriure el codi.
  • Semàntica: El significat de les construccions del llenguatge.
  • Paradigmes de programació: L'enfocament o estil de programació que suporta el llenguatge (e.g., orientat a objectes, funcional, procedimental).
  • Portabilitat: La capacitat d'executar el codi en diferents plataformes sense modificacions.
  • Eficiència: Què tan ràpid i eficient és el codi en termes d'ús de recursos.

Exemples de llenguatges de programació

Python

Python és un llenguatge d'alt nivell conegut per la seva simplicitat i llegibilitat. És àmpliament utilitzat en desenvolupament web, anàlisi de dades, intel·ligència artificial, i més.

# Exemple d'un programa simple en Python
def salutació(nom):
    return f"Hola, {nom}!"

print(salutació("Món"))

Java

Java és un llenguatge de programació orientat a objectes que és àmpliament utilitzat en aplicacions empresarials, desenvolupament d'Android, i sistemes distribuïts.

// Exemple d'un programa simple en Java
public class Salutació {
    public static void main(String[] args) {
        System.out.println(saludar("Món"));
    }

    public static String saludar(String nom) {
        return "Hola, " + nom + "!";
    }
}

JavaScript

JavaScript és un llenguatge de scripting que s'utilitza principalment per al desenvolupament web, permetent la creació de pàgines web interactives.

// Exemple d'un programa simple en JavaScript
function salutació(nom) {
    return "Hola, " + nom + "!";
}

console.log(salutació("Món"));

Comparació de llenguatges de programació

A continuació, es presenta una taula comparativa d'alguns llenguatges de programació populars:

| Llenguatge | Paradigma | Ús principal | Avantatges | |------------|---------------------|----------------------------|-------------------------------------| | Python | Multiparadigma | Desenvolupament web, IA, scripts | Sintaxi simple, gran comunitat | | Java | Orientat a objectes | Aplicacions empresarials | Portabilitat, robustesa | | JavaScript | Scripting | Desenvolupament web | Interactivitat, integració web | | C++ | Multiparadigma | Sistemes embeguts, jocs | Eficiència, control de maquinari | | Haskell | Funcional | Investigació, educació | Expressivitat, maneig de funcions |

Conclusió

Els llenguatges de programació són eines essencials en el desenvolupament de programari, cadascun amb les seves pròpies característiques i usos específics. Comprendre els diferents tipus de llenguatges i les seves aplicacions pot ajudar els programadors a triar l'eina adequada per a cada tasca. A mesura que avancem en el curs, explorarem més a fons alguns d'aquests llenguatges i les seves aplicacions pràctiques.
