Introducció

En aquesta secció del curs sobre Fonaments de la Programació, aprendrem sobre els paràmetres i el retorn de valors en funcions. Aquests conceptes són fonamentals per escriure codi modular i reutilitzable. Entendre com passar informació a les funcions i com rebre resultats d'elles és crucial per a qualsevol programador.

Paràmetres

Els paràmetres són variables que s'utilitzen per rebre valors quan es crida a una funció. Aquests valors es coneixen com a arguments. Els paràmetres permeten que les funcions siguin més flexibles i reutilitzables.

Tipus de Paràmetres

  • Paràmetres Posicionals: Els valors es passen a la funció en el mateix ordre en què es defineixen els paràmetres.
  • Paràmetres Nomenats: Els valors es passen a la funció utilitzant el nom del paràmetre, cosa que permet especificar els arguments en qualsevol ordre.
  • Paràmetres per Defecte: Els paràmetres poden tenir valors per defecte, que s'utilitzen si no es proporciona un argument.

Exemple de Paràmetres

def saludar(nom, missatge="Hola"):
    print(f"{missatge}, {nom}!")

# Crida amb paràmetres posicionals
saludar("Ana")

# Crida amb paràmetres nomenats
saludar(missatge="Bon dia", nom="Carlos")

# Crida amb paràmetre per defecte
saludar("Luis")

En aquest exemple, la funció saludar té dos paràmetres: nom i missatge. El paràmetre missatge té un valor per defecte de "Hola".

Retorn de Valors

El retorn de valors permet que una funció torni un resultat al lloc on va ser cridada. Això és útil per realitzar càlculs i tornar el resultat, o per a qualsevol operació que necessiti tornar informació.

Ús del Retorn de Valors

  • Retorn Simple: Una funció pot tornar un sol valor utilitzant la paraula clau return.
  • Retorn Múltiple: Una funció pot tornar múltiples valors com una tupla.

Exemple de Retorn de Valors

def sumar(a, b):
    return a + b

resultat = sumar(3, 4)
print(resultat)  # Sortida: 7

def operacions(a, b):
    suma = a + b
    resta = a - b
    return suma, resta

suma, resta = operacions(5, 2)
print(f"Suma: {suma}, Resta: {resta}")  # Sortida: Suma: 7, Resta: 3

En aquest exemple, la funció sumar torna la suma de dos números. La funció operacions torna tant la suma com la resta de dos números.

Comparació de Paràmetres i Retorn de Valors

| Concepte | Descripció | Exemple | |---------------------|-----------------------------------------------------------------------------|------------------------------------------| | Paràmetres | Variables que reben valors quan es crida a una funció | def saludar(nom, missatge="Hola") | | Retorn de Valors | Valors que una funció torna al lloc on va ser cridada | return a + b | | Paràmetres Posicionals | Els valors es passen en el mateix ordre en què es defineixen els paràmetres | saludar("Ana") | | Paràmetres Nomenats | Els valors es passen utilitzant el nom del paràmetre | saludar(missatge="Bon dia", nom="Carlos") | | Paràmetres per Defecte | Paràmetres que tenen valors per defecte | def saludar(nom, missatge="Hola") | | Retorn Simple | Una funció torna un sol valor | return a + b | | Retorn Múltiple | Una funció torna múltiples valors com una tupla | return suma, resta |

Conclusió

Entendre com funcionen els paràmetres i el retorn de valors és essencial per escriure funcions efectives i reutilitzables. Els paràmetres permeten que les funcions rebin informació, mentre que el retorn de valors permet que les funcions tornin resultats. Practicar aquests conceptes t'ajudarà a escriure codi més modular i eficient.
