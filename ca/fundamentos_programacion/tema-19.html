Introducció

En aquest tema, explorarem dues estructures de dades fonamentals en la programació: els diccionaris i els conjunts. Aquestes estructures són essencials per manejar col·leccions de dades de manera eficient i són àmpliament utilitzades en diversos llenguatges de programació.

Diccionaris

Els diccionaris són col·leccions de parells clau-valor. Cada clau en un diccionari és única i s'utilitza per accedir al seu valor corresponent. Els diccionaris són molt útils quan necessites associar dades relacionades entre si.

Característiques dels Diccionaris

  • Clau-Valor: Cada element en un diccionari té una clau i un valor.
  • Accés Ràpid: Els diccionaris permeten un accés ràpid als valors mitjançant les seves claus.
  • Mutables: Els diccionaris poden ser modificats després de la seva creació.
  • No Ordenats: En versions anteriors a Python 3.7, els diccionaris no mantenien l'ordre dels elements. A partir de Python 3.7, els diccionaris mantenen l'ordre d'inserció.

Exemple de Diccionari en Python

# Creació d'un diccionari
estudiant = {
    "nom": "Joan",
    "edat": 21,
    "carrera": "Enginyeria Informàtica"
}

# Accés a un valor
print(estudiant["nom"])  # Sortida: Joan

# Modificació d'un valor
estudiant["edat"] = 22

# Adició d'un nou parell clau-valor
estudiant["promig"] = 8.5

# Eliminació d'un parell clau-valor
del estudiant["carrera"]

# Iteració sobre un diccionari
for clau, valor in estudiant.items():
    print(f"{clau}: {valor}")

Operacions Comunes amb Diccionaris

  • Accedir a un valor: diccionari[clau]
  • Modificar un valor: diccionari[clau] = nou_valor
  • Afegir un nou parell clau-valor: diccionari[nova_clau] = nou_valor
  • Eliminar un parell clau-valor: del diccionari[clau]
  • Iterar sobre un diccionari: for clau, valor in diccionari.items()

Conjunts

Els conjunts són col·leccions desordenades d'elements únics. Són útils per realitzar operacions matemàtiques com la unió, intersecció i diferència.

Característiques dels Conjunts

  • Elements Únics: Cada element en un conjunt és únic.
  • Desordenats: Els conjunts no mantenen l'ordre dels elements.
  • Mutables: Els conjunts poden ser modificats després de la seva creació.
  • Operacions Matemàtiques: Els conjunts suporten operacions com unió, intersecció i diferència.

Exemple de Conjunt en Python

# Creació d'un conjunt
fruites = {"poma", "plàtan", "cirera"}

# Adició d'un element
fruites.add("taronja")

# Eliminació d'un element
fruites.remove("plàtan")

# Verificació de pertinença
print("poma" in fruites)  # Sortida: True

# Operacions de conjunts
conjunt_a = {1, 2, 3, 4}
conjunt_b = {3, 4, 5, 6}

# Unió
unio = conjunt_a | conjunt_b  # {1, 2, 3, 4, 5, 6}

# Intersecció
interseccio = conjunt_a & conjunt_b  # {3, 4}

# Diferència
diferencia = conjunt_a - conjunt_b  # {1, 2}

Operacions Comunes amb Conjunts

  • Afegir un element: conjunt.add(element)
  • Eliminar un element: conjunt.remove(element)
  • Verificar pertinença: element in conjunt
  • Unió: conjunt_a | conjunt_b
  • Intersecció: conjunt_a & conjunt_b
  • Diferència: conjunt_a - conjunt_b

Comparació entre Diccionaris i Conjunts

| Característica | Diccionaris | Conjunts | |----------------------|--------------------------------------|----------------------------------| | Estructura | Parells clau-valor | Elements únics | | Ordre | No ordenats (fins a Python 3.6) | No ordenats | | Mutabilitat | Mutables | Mutables | | Accés | Mitjançant claus | Mitjançant pertinença | | Operacions | Accés, modificació, eliminació | Unió, intersecció, diferència |

Conclusió

Els diccionaris i conjunts són estructures de dades poderoses i flexibles que permeten manejar col·leccions de dades de manera eficient. Els diccionaris són ideals per associar parells clau-valor, mentre que els conjunts són perfectes per manejar col·leccions d'elements únics i realitzar operacions matemàtiques. Comprendre aquestes estructures i les seves operacions bàsiques és fonamental per a qualsevol programador.
