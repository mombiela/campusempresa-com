Què és un algorisme?

Un algorisme és una seqüència finita de passos ben definits que s'utilitzen per resoldre un problema o realitzar una tasca específica. Els algorismes són fonamentals en la programació, ja que proporcionen una guia clara sobre com ha d'executar-se una tasca.

Característiques d'un algorisme

  • Finitud: Un algorisme ha de tenir un nombre finit de passos.
  • Definició: Cada pas de l'algorisme ha d'estar clarament definit.
  • Entrada: Un algorisme pot tenir zero o més entrades.
  • Sortida: Un algorisme ha de tenir almenys una sortida.
  • Efectivitat: Els passos de l'algorisme han de ser prou bàsics per ser duts a terme, en principi, per una persona amb llapis i paper.

Exemple d'un algorisme simple

Per il·lustrar com funciona un algorisme, considerem un exemple senzill: un algorisme per sumar dos números.

Descripció de l'algorisme

  1. Llegir el primer número, \( A \).
  2. Llegir el segon número, \( B \).
  3. Sumar \( A \) i \( B \) per obtenir \( C \).
  4. Mostrar el resultat \( C \).

Representació en pseudocodi

El pseudocodi és una forma d'escriure algorismes que s'assembla al codi de programació, però és més fàcil d'entendre. Aquí està el pseudocodi per al nostre exemple:

Inici
    Llegir A
    Llegir B
    C = A + B
    Mostrar C
Fi

Implementació en Python

A continuació, es mostra com es pot implementar aquest algorisme en Python:

# Llegir el primer número
A = float(input("Ingrese el primer número: "))

# Llegir el segon número
B = float(input("Ingrese el segundo número: "))

# Sumar els dos números
C = A + B

# Mostrar el resultat
print("La suma de", A, "i", B, "és", C)

Tipus d'algorismes

Hi ha diversos tipus d'algorismes, cadascun adequat per a diferents tipus de problemes. Alguns dels més comuns inclouen:

  • Algorismes de cerca: Utilitzats per trobar un element específic dins d'una estructura de dades.
  • Algorismes d'ordenació: Utilitzats per ordenar elements en una estructura de dades.
  • Algorismes de grafs: Utilitzats per resoldre problemes relacionats amb grafs, com trobar el camí més curt entre dos nodes.
  • Algorismes d'optimització: Utilitzats per trobar la millor solució a un problema donat un conjunt de restriccions.

Comparació d'algorismes de cerca

| Algorisme | Complexitat Temporal | Millor Cas | Pitjor Cas | |--------------------|----------------------|------------|------------| | Cerca Lineal | O(n) | O(1) | O(n) | | Cerca Binària | O(log n) | O(1) | O(log n) |

Exercici pràctic

Per consolidar el que hem après, aquí tens un exercici pràctic:

Exercici: Algorisme per trobar el més gran de tres números

  1. Llegir tres números: \( A \), \( B \) i \( C \).
  2. Comparar els números per trobar el més gran.
  3. Mostrar el número més gran.

Pseudocodi

Inici
    Llegir A
    Llegir B
    Llegir C
    Si A >= B i A >= C llavors
        Major = A
    Si no si B >= A i B >= C llavors
        Major = B
    Si no
        Major = C
    FiSi
    Mostrar Major
Fi

Implementació en Python

# Llegir els tres números
A = float(input("Ingrese el primer número: "))
B = float(input("Ingrese el segundo número: "))
C = float(input("Ingrese el tercer número: "))

# Trobar el número més gran
if A >= B and A >= C:
    Major = A
elif B >= A and B >= C:
    Major = B
else:
    Major = C

# Mostrar el resultat
print("El més gran dels tres números és:", Major)

Conclusió

Els algorismes són essencials en la programació, ja que proporcionen una seqüència clara de passos per resoldre problemes. Comprendre com dissenyar i analitzar algorismes és una habilitat fonamental per a qualsevol programador. En aquest tema, hem cobert els conceptes bàsics dels algorismes, les seves característiques, i hem vist exemples pràctics de com es poden implementar en Python. Amb aquesta base, estaràs millor preparat per abordar problemes més complexos i dissenyar els teus propis algorismes.
