Introducció

Les estructures de control condicionals són fonamentals en la programació, ja que permeten que el flux d'execució d'un programa variï en funció de certes condicions. Aquestes estructures permeten als programadors prendre decisions dins del codi, executant diferents blocs de codi segons es compleixin o no certes condicions.

Tipus d'Estructures Condicionals

En la majoria dels llenguatges de programació, les estructures de control condicionals més comunes són:

  • if
  • else
  • else if
  • switch (o case en alguns llenguatges)

`if`

L'estructura if avalua una condició i, si aquesta és verdadera, executa un bloc de codi.

# Exemple en Python
x = 10
if x > 5:
    print("x és més gran que 5")

`else`

L'estructura else s'utilitza juntament amb if per executar un bloc de codi alternatiu si la condició del if és falsa.

# Exemple en Python
x = 3
if x > 5:
    print("x és més gran que 5")
else:
    print("x no és més gran que 5")

`else if` (o `elif` en Python)

L'estructura else if permet avaluar múltiples condicions seqüencialment. Si la primera condició és falsa, s'avalua la següent, i així successivament.

# Exemple en Python
x = 7
if x > 10:
    print("x és més gran que 10")
elif x > 5:
    print("x és més gran que 5 però menor o igual a 10")
else:
    print("x és 5 o menor")

`switch` (o `case`)

L'estructura switch permet seleccionar entre múltiples blocs de codi a executar, basant-se en el valor d'una expressió.

// Exemple en Java
int day = 3;
switch (day) {
    case 1:
        System.out.println("Dilluns");
        break;
    case 2:
        System.out.println("Dimarts");
        break;
    case 3:
        System.out.println("Dimecres");
        break;
    default:
        System.out.println("Un altre dia");
        break;
}

Comparació d'Estructures Condicionals

| Estructura | Ús Principal | Avantatges | Desavantatges | |------------|---------------|----------|-------------| | if | Avaluar una sola condició | Simple i directe | Pot tornar-se complex amb múltiples condicions | | else | Proveir una alternativa si if és fals | Fàcil de llegir | Depèn de if | | else if | Avaluar múltiples condicions seqüencials | Més control sobre el flux | Pot ser difícil de mantenir amb moltes condicions | | switch | Seleccionar entre múltiples opcions basades en un valor | Clar per a múltiples opcions | Menys flexible que if-else per a condicions complexes |

Exercicis Pràctics

  1. Exercici 1: Condicional simple

    • Escriu un programa que verifiqui si un número és positiu, negatiu o zero.
    # Exercici 1
num = int(input("Introdueix un número: "))
if num > 0:
    print("El número és positiu")
elif num < 0:
    print("El número és negatiu")
else:
    print("El número és zero")

  2. Exercici 2: Ús de switch

    • Escriu un programa que imprimeixi el nom del dia de la setmana basat en un número de l'1 al 7.
    // Exercici 2
int day = 4; // Canvia aquest valor per provar diferents dies
switch (day) {
    case 1:
        System.out.println("Dilluns");
        break;
    case 2:
        System.out.println("Dimarts");
        break;
    case 3:
        System.out.println("Dimecres");
        break;
    case 4:
        System.out.println("Dijous");
        break;
    case 5:
        System.out.println("Divendres");
        break;
    case 6:
        System.out.println("Dissabte");
        break;
    case 7:
        System.out.println("Diumenge");
        break;
    default:
        System.out.println("Número invàlid");
        break;
}

Conclusió

Les estructures de control condicionals són essencials per a la presa de decisions en la programació. Comprendre com i quan utilitzar-les és crucial per escriure codi eficient i clar. A través de if, else, else if i switch, els programadors poden dirigir el flux d'execució dels seus programes de manera lògica i organitzada. Practicar amb exemples i exercicis ajudarà a consolidar aquests conceptes i a aplicar-los correctament en situacions reals.
