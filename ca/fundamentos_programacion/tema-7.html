Introducció

L'entrada i sortida de dades (I/O) és un concepte fonamental en la programació que permet als programes interactuar amb el món exterior. L'entrada es refereix a l'obtenció de dades des d'una font externa, com el teclat o un fitxer, mentre que la sortida es refereix a la presentació de dades a l'usuari o a un altre sistema, com la pantalla o un fitxer.

Entrada de Dades

L'entrada de dades permet als programes rebre informació de l'usuari o d'altres fonts. En la majoria dels llenguatges de programació, existeixen funcions específiques per manejar l'entrada de dades.

Entrada des del Teclat

L'entrada des del teclat és una de les formes més comunes d'obtenir dades de l'usuari.

Exemple en Python:

# Sol·licitar a l'usuari que ingressi el seu nom
nom = input("Si us plau, ingressa el teu nom: ")
print("Hola, " + nom + "!")

Exemple en Java:

import java.util.Scanner;

public class EntradaTeclat {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Si us plau, ingressa el teu nom: ");
        String nom = scanner.nextLine();
        System.out.println("Hola, " + nom + "!");
    }
}

Entrada des de Fitxers

Llegir dades des de fitxers és una altra forma comuna d'entrada de dades.

Exemple en Python:

# Llegir dades des d'un fitxer
with open('dades.txt', 'r') as fitxer:
    contingut = fitxer.read()
    print(contingut)

Exemple en Java:

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class EntradaFitxer {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("dades.txt"))) {
            String linia;
            while ((linia = br.readLine()) != null) {
                System.out.println(linia);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Sortida de Dades

La sortida de dades permet als programes mostrar informació a l'usuari o a altres sistemes.

Sortida a la Pantalla

La sortida a la pantalla és la forma més directa de mostrar informació a l'usuari.

Exemple en Python:

# Mostrar un missatge a la pantalla
print("Hola, Món!")

Exemple en Java:

public class SortidaPantalla {
    public static void main(String[] args) {
        System.out.println("Hola, Món!");
    }
}

Sortida a Fitxers

Escriure dades en fitxers és una forma comuna de sortida de dades.

Exemple en Python:

# Escriure dades en un fitxer
with open('sortida.txt', 'w') as fitxer:
    fitxer.write("Hola, Món!")

Exemple en Java:

import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class SortidaFitxer {
    public static void main(String[] args) {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter("sortida.txt"))) {
            bw.write("Hola, Món!");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Comparació de Mètodes d'Entrada i Sortida

| Mètode | Python | Java | |--------------|---------------------------------|----------------------------------| | Entrada Teclat | input() | Scanner.nextLine() | | Entrada Fitxer | open('fitxer.txt', 'r') | BufferedReader + FileReader | | Sortida Pantalla | print() | System.out.println() | | Sortida Fitxer | open('fitxer.txt', 'w') | BufferedWriter + FileWriter |

Conclusió

L'entrada i sortida de dades són components essencials en qualsevol programa. Comprendre com manejar la I/O en diferents llenguatges de programació és crucial per desenvolupar aplicacions que interactuïn eficaçment amb els usuaris i altres sistemes. Practicar amb exemples i exercicis ajudarà a consolidar aquests conceptes i a aplicar-los en projectes reals.
