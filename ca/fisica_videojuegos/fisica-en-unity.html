Introducció a la Física en Unity

Unity és una de les plataformes més populars per al desenvolupament de videojocs, i el seu motor de física és una eina poderosa per crear interaccions realistes entre objectes. En aquesta secció, explorarem els conceptes bàsics de la física en Unity, com configurar l'entorn i com utilitzar els components de física per crear efectes realistes en els teus jocs.

Conceptes Clau

  • Rigidbody: Component que permet que un objecte sigui afectat per la física.
  • Collider: Component que defineix la forma de l'objecte per a les col·lisions.
  • Forces: Aplicació de forces per moure objectes.
  • Gravetat: Configuració i manipulació de la gravetat en el joc.
  • Detecció de Col·lisions: Mètodes per detectar i respondre a col·lisions entre objectes.
  • Joints: Connexions físiques entre objectes.

Configuració Inicial

Per començar a treballar amb física en Unity, és necessari configurar alguns components bàsics en els teus objectes.

Afegir un Rigidbody

El component Rigidbody és essencial perquè un objecte sigui afectat per la física. Per afegir un Rigidbody a un objecte:

  1. Selecciona l'objecte en la jerarquia.
  2. Ves al panell d'Inspector.
  3. Fes clic a "Add Component".
  4. Busca i selecciona "Rigidbody".
// Exemple de codi per afegir un Rigidbody mitjançant script
void Start() {
    Rigidbody rb = gameObject.AddComponent<Rigidbody>();
}

Afegir un Collider

El Collider defineix la forma de l'objecte per a les col·lisions. Unity ofereix diversos tipus de colliders, com BoxCollider, SphereCollider, i MeshCollider.

// Exemple de codi per afegir un BoxCollider mitjançant script
void Start() {
    BoxCollider collider = gameObject.AddComponent<BoxCollider>();
}

Aplicació de Forces

Per moure un objecte utilitzant la física, pots aplicar forces al seu Rigidbody.

Mètodes per Aplicar Forces

  • AddForce: Aplica una força contínua.
  • AddTorque: Aplica una força de rotació.
// Exemple de codi per aplicar una força
void Update() {
    if (Input.GetKey(KeyCode.Space)) {
        Rigidbody rb = GetComponent<Rigidbody>();
        rb.AddForce(Vector3.up * 10f);
    }
}

Gravetat

La gravetat en Unity pot ser configurada globalment o individualment per a cada Rigidbody.

Configuració Global de la Gravetat

Pots ajustar la gravetat global des de Edit > Project Settings > Physics.

Configuració Individual de la Gravetat

Per desactivar la gravetat en un Rigidbody específic:

void Start() {
    Rigidbody rb = GetComponent<Rigidbody>();
    rb.useGravity = false;
}

Detecció de Col·lisions

Unity ofereix diversos mètodes per detectar col·lisions entre objectes.

Mètodes de Detecció

  • OnCollisionEnter: Cridat quan comença una col·lisió.
  • OnCollisionStay: Cridat mentre la col·lisió continua.
  • OnCollisionExit: Cridat quan acaba una col·lisió.
// Exemple de codi per detectar col·lisions
void OnCollisionEnter(Collision collision) {
    Debug.Log("Col·lisió detectada amb: " + collision.gameObject.name);
}

Joints

Els Joints permeten connectar dos objectes i definir com interactuen físicament.

Tipus de Joints

  • FixedJoint: Connecta dos objectes de manera fixa.
  • HingeJoint: Permet la rotació al voltant d'un punt.
  • SpringJoint: Connecta dos objectes amb una força de moll.
// Exemple de codi per afegir un HingeJoint
void Start() {
    HingeJoint hinge = gameObject.AddComponent<HingeJoint>();
    hinge.connectedBody = anotherGameObject.GetComponent<Rigidbody>();
}

Conclusió

La física en Unity és una eina poderosa que permet crear interaccions realistes en els teus videojocs. En entendre i utilitzar components com Rigidbody, Collider, i Joints, pots simular moviments i col·lisions de manera efectiva. Practica aquests conceptes i experimenta amb diferents configuracions per dominar la física en Unity i portar els teus jocs al següent nivell.
