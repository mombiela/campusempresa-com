Introducció

La resolució de col·lisions és un aspecte fonamental en la física de videojocs. Una vegada detectades les col·lisions entre objectes, és crucial resoldre-les de manera que els objectes reaccionin de forma realista. Aquest tema cobreix els conceptes bàsics i avançats de la resolució de col·lisions, incloent-hi mètodes matemàtics i exemples de codi.

Conceptes Clau

  • Col·lisió: Ocorre quan dos o més objectes en un videojoc es troben en el mateix espai al mateix temps.
  • Normal de Col·lisió: Vector perpendicular a la superfície de contacte en el punt de col·lisió.
  • Penetració: La distància que un objecte ha penetrat en un altre durant una col·lisió.
  • Impuls: Canvi en la quantitat de moviment d'un objecte degut a una força aplicada durant un curt període de temps.

Mètodes de Resolució de Col·lisions

Mètode de Separació d'Eixos (SAT)

El Mètode de Separació d'Eixos (SAT) és una tècnica utilitzada per detectar i resoldre col·lisions entre polígons convexos.

  • Pas 1: Projectar els vèrtexs dels polígons en un eix.
  • Pas 2: Verificar si les projeccions se solapen.
  • Pas 3: Si hi ha solapament en tots els eixos, s'ha detectat una col·lisió.
  • Pas 4: Resoldre la col·lisió movent els objectes al llarg de l'eix de menor penetració.

Exemple de Codi: SAT en 2D

def project_polygon(vertices, axis):
    min_proj = float('inf')
    max_proj = float('-inf')
    for vertex in vertices:
        projection = vertex.dot(axis)
        min_proj = min(min_proj, projection)
        max_proj = max(max_proj, projection)
    return min_proj, max_proj

def overlap(minA, maxA, minB, maxB):
    return minB <= maxA and minA <= maxB

def sat_collision(polygonA, polygonB):
    axes = get_axes(polygonA) + get_axes(polygonB)
    for axis in axes:
        minA, maxA = project_polygon(polygonA, axis)
        minB, maxB = project_polygon(polygonB, axis)
        if not overlap(minA, maxA, minB, maxB):
            return False
    return True

# Example usage
polygonA = [Vector2(0, 0), Vector2(1, 0), Vector2(1, 1), Vector2(0, 1)]
polygonB = [Vector2(0.5, 0.5), Vector2(1.5, 0.5), Vector2(1.5, 1.5), Vector2(0.5, 1.5)]
print(sat_collision(polygonA, polygonB))  # Output: True

Impuls i Conservació del Moment

Per resoldre col·lisions de manera més realista, s'utilitza el concepte d'impuls i conservació del moment.

  • Impuls: \( J = \frac{-(1 + e) \cdot (v \cdot n)}{(1/m1 + 1/m2)} \)
    • \( e \) és el coeficient de restitució.
    • \( v \) és la velocitat relativa.
    • \( n \) és la normal de col·lisió.
    • \( m1 \) i \( m2 \) són les masses dels objectes.

Exemple de Codi: Resolució d'Impuls

def resolve_collision(obj1, obj2, normal, restitution):
    relative_velocity = obj2.velocity - obj1.velocity
    velocity_along_normal = relative_velocity.dot(normal)
    
    if velocity_along_normal > 0:
        return
    
    impulse_magnitude = -(1 + restitution) * velocity_along_normal
    impulse_magnitude /= (1 / obj1.mass + 1 / obj2.mass)
    
    impulse = impulse_magnitude * normal
    obj1.velocity -= (1 / obj1.mass) * impulse
    obj2.velocity += (1 / obj2.mass) * impulse

# Example usage
class GameObject:
    def __init__(self, mass, velocity):
        self.mass = mass
        self.velocity = velocity

obj1 = GameObject(1, Vector2(1, 0))
obj2 = GameObject(1, Vector2(-1, 0))
normal = Vector2(1, 0)
restitution = 0.5

resolve_collision(obj1, obj2, normal, restitution)
print(obj1.velocity, obj2.velocity)  # Output: Vector2(-0.5, 0) Vector2(0.5, 0)

Conclusió

La resolució de col·lisions és essencial per crear experiències de joc realistes i fluides. Des de mètodes geomètrics com el SAT fins a tècniques físiques com la conservació del moment, cada enfocament té les seves aplicacions i avantatges. Practicar amb exemples de codi i comprendre els fonaments matemàtics darrere d'aquests mètodes permetrà als desenvolupadors implementar sistemes de col·lisió efectius en els seus videojocs.
