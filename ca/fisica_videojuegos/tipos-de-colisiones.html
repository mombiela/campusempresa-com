En la física de videojocs, les col·lisions són fonamentals per crear interaccions realistes entre objectes. Aquest tema cobreix els diferents tipus de col·lisions que es poden implementar en un motor de joc, les seves característiques i com es poden gestionar programàticament.

Col·lisions AABB (Axis-Aligned Bounding Box)

Les col·lisions AABB són un dels mètodes més simples i eficients per detectar col·lisions entre objectes en un videojoc. Aquestes col·lisions utilitzen caixes delimitadores alineades amb els eixos per simplificar els càlculs.

  • Característiques:

    • Les caixes estan alineades amb els eixos del sistema de coordenades.
    • Fàcil d'implementar i computacionalment eficient.
    • Ideal per a objectes rectangulars o cúbics.

  • Exemple de Codi:

class AABB:
    def __init__(self, x, y, width, height):
        self.x = x
        self.y = y
        self.width = width
        self.height = height

def check_collision(a, b):
    return (a.x < b.x + b.width and
            a.x + a.width > b.x and
            a.y < b.y + b.height and
            a.y + a.height > b.y)

# Exemple d'ús
box1 = AABB(0, 0, 10, 10)
box2 = AABB(5, 5, 10, 10)
print(check_collision(box1, box2))  # Hauria d'imprimir True

Col·lisions Circulars

Les col·lisions circulars són útils per a objectes que tenen una forma rodona o esfèrica. Aquest mètode utilitza la distància entre els centres dels cercles per determinar si hi ha una col·lisió.

  • Característiques:

    • Adequat per a objectes rodons.
    • Requereix el càlcul de la distància entre dos punts.
    • Pot ser més costós computacionalment que AABB per a objectes no circulars.

  • Exemple de Codi:

import math

class Circle:
    def __init__(self, x, y, radius):
        self.x = x
        self.y = y
        self.radius = radius

def check_collision(circle1, circle2):
    distance = math.sqrt((circle1.x - circle2.x) ** 2 + (circle1.y - circle2.y) ** 2)
    return distance < (circle1.radius + circle2.radius)

# Exemple d'ús
circle1 = Circle(0, 0, 5)
circle2 = Circle(7, 0, 5)
print(check_collision(circle1, circle2))  # Hauria d'imprimir True

Les col·lisions OBB són més complexes que les AABB ja que les caixes poden estar orientades en qualsevol direcció. Això permet una detecció de col·lisions més precisa per a objectes que no estan alineats amb els eixos.

  • Característiques:

    • Les caixes poden estar orientades en qualsevol direcció.

Conclusió

L'elecció del tipus de col·lisió depèn de la forma i el comportament dels objectes en el teu videojoc. Les col·lisions AABB i circulars són fàcils d'implementar i eficients, mentre que les col·lisions OBB i de polígons ofereixen major precisió a costa d'una major complexitat computacional. Entendre aquests conceptes et permetrà crear interaccions més realistes i optimitzades en els teus videojocs.
