Introducció

La simulació de partícules és una tècnica fonamental en la física de videojocs que permet crear efectes visuals realistes com foc, fum, explosions i aigua. En aquest tema, explorarem els conceptes bàsics de la simulació de partícules, com implementar-les en un motor de videojocs i alguns exemples pràctics.

Conceptes Clau

  • Partícula: La unitat bàsica en una simulació de partícules. Pot representar qualsevol cosa, des d'una gota d'aigua fins a una espurna de foc.
  • Emissor de partícules: L'origen de les partícules. Pot ser un punt, una línia, una àrea o un volum.
  • Forces: Factors que afecten el moviment de les partícules, com la gravetat, el vent i la fricció.
  • Vida útil: El temps durant el qual una partícula existeix abans de desaparèixer.
  • Renderitzat: El procés de dibuixar les partícules a la pantalla.

Implementació Bàsica

Creació de Partícules

Per crear una partícula, necessitem definir les seves propietats bàsiques com posició, velocitat i vida útil. Aquí hi ha un exemple en pseudocodi:

class Particle:
    def __init__(self, position, velocity, life_span):
        self.position = position
        self.velocity = velocity
        self.life_span = life_span
        self.age = 0

    def update(self, delta_time):
        self.position += self.velocity * delta_time
        self.age += delta_time
        if self.age > self.life_span:
            self.is_alive = False

Emissor de Partícules

L'emissor de partícules és responsable de generar noves partícules. Aquí hi ha un exemple en pseudocodi:

class ParticleEmitter:
    def __init__(self, position, rate):
        self.position = position
        self.rate = rate
        self.particles = []

    def emit(self, delta_time):
        num_new_particles = int(self.rate * delta_time)
        for _ in range(num_new_particles):
            new_particle = Particle(self.position, random_velocity(), random_life_span())
            self.particles.append(new_particle)

    def update(self, delta_time):
        for particle in self.particles:
            particle.update(delta_time)
        self.particles = [p for p in self.particles if p.is_alive]

Aplicació de Forces

Les forces afecten el moviment de les partícules. Aquí hi ha un exemple de com aplicar la gravetat:

def apply_gravity(particle, delta_time):
    gravity = Vector(0, -9.81)
    particle.velocity += gravity * delta_time

Renderitzat de Partícules

Finalment, necessitem renderitzar les partícules a la pantalla. Aquí hi ha un exemple en pseudocodi:

def render_particles(particles, screen):
    for particle in particles:
        if particle.is_alive:
            draw_particle(screen, particle.position)

Exemple Complet

Aquí hi ha un exemple complet que combina tots els elements anteriors:

class ParticleSystem:
    def __init__(self, emitter):
        self.emitter = emitter

    def update(self, delta_time):
        self.emitter.emit(delta_time)
        self.emitter.update(delta_time)
        for particle in self.emitter.particles:
            apply_gravity(particle, delta_time)

    def render(self, screen):
        render_particles(self.emitter.particles, screen)

# Ús del sistema de partícules
emitter = ParticleEmitter(Vector(0, 0), 10)  # Emet 10 partícules per segon
particle_system = ParticleSystem(emitter)

while game_is_running:
    delta_time = get_delta_time()
    particle_system.update(delta_time)
    particle_system.render(screen)

Conclusió

La simulació de partícules és una tècnica poderosa per afegir efectes visuals realistes als videojocs. En entendre els conceptes bàsics i com implementar-los, pots crear una àmplia varietat d'efectes visuals. Practica creant diferents tipus d'emissors i aplicant diverses forces per veure com afecten el comportament de les partícules.
