Introducció a Sass

Sass (Syntactically Awesome Stylesheets) és un preprocesador de CSS que afegeix potència i elegància al llenguatge bàsic. Et permet utilitzar variables, regles imbricades, mixins, funcions i més, tot amb una sintaxi totalment compatible amb CSS.

Conceptes clau de Sass

  • Variables: Emmagatzema valors que pots reutilitzar al llarg de la teva fulla d'estils.
  • Imbricació: Imbrica els teus selectors CSS d'una manera que segueixi la mateixa jerarquia visual del teu HTML.
  • Partials i Import: Divideix el teu CSS en peces més petites i reutilitzables.
  • Mixins: Crea trossos de codi reutilitzables.
  • Herència: Comparteix un conjunt de propietats CSS d'un selector a un altre.

Configuració de Sass en un projecte Ionic

Instal·lació de Sass

Per utilitzar Sass amb Ionic, necessites instal·lar les eines necessàries. Ionic CLI ve amb suport integrat per a Sass.

  1. Crea un nou projecte Ionic:

    ionic start myApp tabs
cd myApp

  2. Instal·la Sass:

    npm install sass --save-dev

Configuració de Sass

Ionic utilitza una carpeta scss per als fitxers Sass per defecte. Pots configurar el teu projecte per utilitzar Sass creant fitxers .scss.

  1. Crea una carpeta scss:

    mkdir src/theme/scss

  2. Crea un fitxer principal de Sass:

    touch src/theme/scss/variables.scss

  3. Importa el fitxer principal de Sass en els teus estils globals:

    // src/global.scss
@import 'theme/scss/variables';

Utilitzant les característiques de Sass en Ionic

Variables

Les variables en Sass et permeten emmagatzemar valors que pots reutilitzar al llarg de la teva fulla d'estils.

// src/theme/scss/variables.scss
$primary-color: #3880ff;
$secondary-color: #3dc2ff;

body {
  background-color: $primary-color;
  color: $secondary-color;
}

Imbricació

L'imbricació et permet imbricar els teus selectors CSS d'una manera que segueixi la mateixa jerarquia visual del teu HTML.

// src/theme/scss/variables.scss
nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;

    li {
      display: inline-block;
      margin-right: 5px;

      a {
        text-decoration: none;
        color: $primary-color;

        &:hover {
          color: $secondary-color;
        }
      }
    }
  }
}

Partials i Import

Els partials et permeten dividir el teu CSS en peces més petites i reutilitzables. Pots importar aquests partials a la teva fulla d'estils principal.

// src/theme/scss/_buttons.scss
.button {
  background-color: $primary-color;
  border: none;
  color: white;
  padding: 10px 20px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
}

// src/theme/scss/variables.scss
@import 'buttons';

Mixins

Els mixins et permeten crear trossos de codi reutilitzables.

// src/theme/scss/_mixins.scss
@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
  -moz-border-radius: $radius;
  -ms-border-radius: $radius;
  -o-border-radius: $radius;
  border-radius: $radius;
}

// src/theme/scss/variables.scss
@import 'mixins';

.box { 
  @include border-radius(10px); 
}

Herència

L'herència et permet compartir un conjunt de propietats CSS d'un selector a un altre.

// src/theme/scss/_base.scss
.message {
  padding: 10px;
  border: 1px solid #ccc;
  color: #333;
}

// src/theme/scss/variables.scss
@import 'base';

.success {
  @extend .message;
  border-color: green;
}

.error {
  @extend .message;
  border-color: red;
}

Conclusió

Utilitzar Sass amb Ionic millora la teva capacitat per gestionar i mantenir les teves fulles d'estil de manera eficient. Aprofitant les característiques de Sass com variables, imbricació, partials, mixins i herència, pots crear CSS més organitzat i reutilitzable. Això no només millora el teu flux de treball de desenvolupament, sinó que també fa que la teva base de codi sigui més escalable i fàcil de mantenir.
