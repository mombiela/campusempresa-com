Visió General

La tematització en Ionic et permet personalitzar l'aspecte i la sensació de la teva aplicació per adaptar-la a la teva marca o preferències de disseny. Aquesta secció et guiarà a través dels conceptes bàsics de la tematització en Ionic, des de la comprensió dels conceptes bàsics fins a la implementació de temes personalitzats.

Conceptes Clau

  • Variables CSS: Ionic utilitza variables CSS (propietats personalitzades) per definir els colors del tema i altres estils.
  • Estils Globals: Estils que s'apliquen a tota l'aplicació.
  • Estils de Components: Estils que s'apliquen a components específics.
  • Mode Fosc: Un tema que utilitza colors més foscos per a una millor llegibilitat en entorns de poca llum.

Variables CSS

Les variables CSS són una manera poderosa de gestionar i aplicar estils a tota la teva aplicació Ionic. Et permeten definir valors reutilitzables que es poden aplicar a diverses propietats CSS.

Exemple

:root {
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

En aquest exemple, --ion-color-primary, --ion-color-secondary i --ion-color-tertiary són variables CSS que defineixen els colors primari, secundari i terciari del teu tema.

Estils Globals

Els estils globals es defineixen al fitxer src/theme/variables.scss. Aquí és on pots establir els estils per defecte per a tota la teva aplicació.

Exemple

// src/theme/variables.scss
:root {
  --ion-font-family: 'Open Sans', sans-serif;
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

En aquest exemple, establim la família de fonts per defecte i els colors per a tota l'aplicació.

Estils de Components

Els estils dels components estan delimitats a components individuals. Pots definir aquests estils a la fulla d'estils del component.

Exemple

// src/pages/home/home.page.scss
.page-home {
  ion-content {
    --background: #f0f0f0;
  }

  .custom-button {
    --background: var(--ion-color-primary);
    --color: white;
  }
}

En aquest exemple, establim un color de fons personalitzat per al ion-content i un estil personalitzat per a un botó dins de la pàgina home.

Mode Fosc

Ionic admet el mode fosc, que es pot habilitar establint una classe CSS a l'element body.

Exemple

<!-- index.html -->
<body class="dark-theme">
  <app-root></app-root>
</body>

Variables del Mode Fosc

Pots definir variables específiques per al mode fosc al teu fitxer variables.scss.

// src/theme/variables.scss
body.dark-theme {
  --ion-background-color: #222428;
  --ion-text-color: #ffffff;
}

En aquest exemple, establim els colors de fons i de text per al mode fosc.

Resum

La tematització en Ionic és una manera poderosa de personalitzar l'aparença de la teva aplicació. Utilitzant variables CSS, estils globals, estils de components i el mode fosc, pots crear una experiència d'usuari cohesiva i visualment atractiva. Recorda:

  • Utilitzar variables CSS per a estils reutilitzables i mantenibles.
  • Definir estils globals a variables.scss per a una consistència a tota l'aplicació.
  • Delimitar estils a components individuals quan sigui necessari.
  • Implementar el mode fosc per millorar la llegibilitat en entorns de poca llum.

Amb aquestes eines i tècniques, pots crear un aspecte únic i polit per a les teves aplicacions Ionic.
