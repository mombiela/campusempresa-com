Capacitor és un entorn natiu multiplataforma que facilita la creació d'aplicacions web que s'executen nativament en iOS, Android i la web. Proporciona una API consistent per accedir a funcionalitats natives, com ara la càmera, la geolocalització i el sistema de fitxers.

Introducció a Capacitor

  • Capacitor és una alternativa moderna a Cordova.
  • Permet construir aplicacions web que s'executen nativament en múltiples plataformes.
  • Proporciona una API consistent per accedir a les característiques natives del dispositiu.

Conceptes Clau

  • Plugins: Capacitor utilitza plugins per accedir a les característiques natives del dispositiu.
  • Bridge: El bridge connecta l'aplicació web amb la plataforma nativa.
  • WebView: L'aplicació web s'executa dins d'un WebView en la plataforma nativa.

Configurant Capacitor

Per començar a utilitzar Capacitor en el teu projecte Ionic, cal instal·lar-lo i inicialitzar-lo en el teu projecte.

Instal·lació

Primer, instal·la Capacitor i la seva CLI:

npm install @capacitor/core @capacitor/cli

Inicialitzant Capacitor

Inicialitza Capacitor en el teu projecte Ionic:

npx cap init [appName] [appId]
  • appName: El nom de la teva aplicació.
  • appId: L'identificador únic per a la teva aplicació (per exemple, com.example.app).

Afegint Plataformes

Afegeix les plataformes que vols suportar:

npx cap add android
npx cap add ios

Utilitzant Plugins de Capacitor

Capacitor ve amb un conjunt de plugins bàsics que proporcionen accés a les característiques natives del dispositiu.

Accedint a la Informació del Dispositiu

Per accedir a la informació del dispositiu, utilitza el plugin Device:

npm install @capacitor/device

Exemple: Obtenint Informació del Dispositiu

import { Device } from '@capacitor/device';

const logDeviceInfo = async () => {
  const info = await Device.getInfo();
  console.log(info);
};

logDeviceInfo();
  • Aquest codi importa el plugin Device i registra la informació del dispositiu a la consola.

Utilitzant el Plugin de la Càmera

Per utilitzar la càmera, instal·la el plugin Camera:

npm install @capacitor/camera

Exemple: Fent una Foto

import { Camera, CameraResultType } from '@capacitor/camera';

const takePhoto = async () => {
  const photo = await Camera.getPhoto({
    resultType: CameraResultType.Uri,
    source: CameraSource.Camera,
    quality: 90
  });
  console.log(photo);
};

takePhoto();
  • Aquest codi importa el plugin Camera i fa una foto utilitzant la càmera del dispositiu.

Ús Avançat de Capacitor

Per a un ús més avançat, pots crear plugins personalitzats o utilitzar plugins de la comunitat.

Creant Plugins Personalitzats

Els plugins personalitzats permeten estendre la funcionalitat de Capacitor per adaptar-se a les teves necessitats específiques.

Exemple: Estructura d'un Plugin Personalitzat

my-plugin/
├── android/
├── ios/
├── src/
│   └── index.ts
├── package.json
└── capacitor.config.json
  • Aquesta estructura mostra la disposició bàsica d'un plugin personalitzat de Capacitor.

Utilitzant Plugins de la Comunitat

Els plugins de la comunitat són creats per la comunitat de Capacitor i es poden utilitzar per afegir funcionalitats addicionals a la teva aplicació.

Exemple: Instal·lant un Plugin de la Comunitat

npm install @capacitor-community/barcode-scanner
  • Aquesta comanda instal·la un plugin de lector de codis de barres de la comunitat de Capacitor.

Conclusió

Capacitor és una eina potent que permet construir aplicacions web multiplataforma amb facilitat. Entenent com configurar Capacitor, utilitzar els seus plugins bàsics i estendre la seva funcionalitat amb plugins personalitzats i de la comunitat, pots crear aplicacions robustes i riques en funcionalitats que s'executen nativament en múltiples plataformes.
