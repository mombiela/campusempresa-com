Introducció al Router d'Ionic

El Router d'Ionic és una eina poderosa que et permet gestionar la navegació dins de les teves aplicacions Ionic. Està construït sobre el Router d'Angular, proporcionant una manera fluida de gestionar la navegació i el routing a la teva aplicació.

Conceptes Clau

  • Rutes: Defineixen els camins i components que s'han de mostrar.
  • Router Outlet: Actua com a marcador de posició per als components rutejats.
  • Navegació: Navegació programàtica entre rutes.
  • Guàrdies: Protegeixen les rutes d'accés no autoritzat.

Configurant el Router d'Ionic

Per utilitzar el Router d'Ionic, has d'assegurar-te que el teu projecte està configurat correctament.

Instal·lant Dependències

Assegura't de tenir les dependències necessàries instal·lades:

npm install @ionic/angular @angular/router

Configurant Rutes

Defineix les teves rutes al fitxer app-routing.module.ts:

import { NgModule } from '@angular/core';
import { PreloadAllModules, RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: '',
    redirectTo: 'home',
    pathMatch: 'full'
  },
  {
    path: 'home',
    loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
  },
  {
    path: 'details',
    loadChildren: () => import('./details/details.module').then(m => m.DetailsPageModule)
  }
];

@NgModule({
  imports: [
    RouterModule.forRoot(routes, { preloadingStrategy: PreloadAllModules })
  ],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Utilitzant Router Outlet

La directiva RouterOutlet s'utilitza per mostrar els components rutejats:

<ion-app>
  <ion-router-outlet></ion-router-outlet>
</ion-app>

Navegant Entre Rutes

Pots navegar entre rutes utilitzant el servei Router o la directiva routerLink.

Utilitzant el Servei Router

Injecta el servei Router i utilitza el mètode navigate:

import { Component } from '@angular/core';
import { Router } from '@angular/router';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  constructor(private router: Router) {}

  goToDetails() {
    this.router.navigate(['/details']);
  }
}

Utilitzant la Directiva routerLink

Utilitza la directiva routerLink en les teves plantilles:

<ion-button [routerLink]="['/details']">Anar a Detalls</ion-button>

Guàrdies de Ruta

Les guàrdies de ruta es poden utilitzar per controlar l'accés a certes rutes.

Creant una Guàrdia

Genera una guàrdia utilitzant Angular CLI:

ng generate guard auth

Implementant la Guàrdia

Implementa la interfície CanActivate a la teva guàrdia:

import { Injectable } from '@angular/core';
import { CanActivate, ActivatedRouteSnapshot, RouterStateSnapshot, UrlTree } from '@angular/router';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class AuthGuard implements CanActivate {
  canActivate(
    next: ActivatedRouteSnapshot,
    state: RouterStateSnapshot): Observable<boolean | UrlTree> | Promise<boolean | UrlTree> | boolean | UrlTree {
    // Afegeix la teva lògica d'autenticació aquí
    return true; // o false basat en l'estat d'autenticació
  }
}

Aplicant la Guàrdia a les Rutes

Aplica la guàrdia a les teves rutes a app-routing.module.ts:

const routes: Routes = [
  {
    path: 'details',
    loadChildren: () => import('./details/details.module').then(m => m.DetailsPageModule),
    canActivate: [AuthGuard]
  }
];

Conclusió

El Router d'Ionic és una part essencial de la construcció d'aplicacions Ionic, proporcionant una manera robusta de gestionar la navegació i el routing. Entenent les rutes, els router outlets, els mètodes de navegació i les guàrdies de ruta, pots crear aplicacions dinàmiques i segures. Practica implementant aquests conceptes en els teus projectes per obtenir una comprensió més profunda i habilitat en l'ús del Router d'Ionic.
