Introducció a les Aplicacions Web Progressives (PWAs)

  • Definició de les Aplicacions Web Progressives
  • Beneficis de les PWAs
    • Capacitats fora de línia
    • Millora del rendiment
    • Experiència semblant a una aplicació
    • Compatibilitat multiplataforma

Configuració del teu Entorn Ionic

  • Instal·lació de Node.js i npm
  • Instal·lació de l'Ionic CLI 
    npm install -g @ionic/cli
  • Creació d'un nou projecte Ionic 
    ionic start myPWA blank --type=angular

Construint la teva Primera PWA amb Ionic

  • Visió general de l'estructura del projecte
  • Executant l'aplicació Ionic 
    cd myPWA
ionic serve

Afegint Funcionalitats PWA

  • Activant els Service Workers
    • Explicació dels Service Workers
    • Afegint Service Worker al projecte 
      ng add @angular/pwa --project app
  • Creació d'un fitxer manifest
    • Propòsit del fitxer manifest
    • Exemple d'un fitxer manifest 
      {
  "name": "My PWA",
  "short_name": "PWA",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#4A90E2",
  "icons": [
    {
      "src": "assets/icons/icon-192x192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "assets/icons/icon-512x512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}

Millorant l'Experiència de l'Usuari

  • Afegint una pantalla de benvinguda
  • Personalitzant el tema de l'aplicació
    • Utilitzant les capacitats de temàtica d'Ionic
    • Exemple de personalització del tema en variables.scss 
      $colors: (
  primary: #4A90E2,
  secondary: #32db64,

Capacitats Fora de Línia

  • Estratègies de caché
    • Explicació de les estratègies de caché (per exemple, Cache First, Network First)
    • Implementant una estratègia Cache First 
      self.addEventListener('fetch', function(event) {
  event.respondWith(
    caches.match(event.request).then(function(response) {
      return response || fetch(event.request);
    })
  );
});

Funcionalitats Avançades de PWA

  • Sincronització en segon pla
    • Explicació i casos d'ús
    • Exemple d'implementació 
      self.addEventListener('sync', function(event) {
  if (event.tag == 'myFirstSync') {
    event.waitUntil(doSomeStuff());
  }
});
  • Notificacions push
    • Configurant les notificacions push
    • Exemple d'enviament d'una notificació push 
      self.registration.showNotification('Hello World!', {
  body: 'This is a push notification.',
  icon: 'assets/icons/icon-192x192.png'
});
