Introducció

Els formularis i les entrades són components fonamentals en qualsevol aplicació, permetent als usuaris interactuar amb l'app proporcionant dades. A Ionic, crear formularis i gestionar l'entrada de l'usuari és senzill, gràcies al seu ric conjunt de components d'interfície d'usuari i les potents capacitats de gestió de formularis d'Angular.

Elements Bàsics del Formulari

Ionic proporciona una varietat d'elements de formulari que es poden utilitzar per capturar l'entrada de l'usuari. Aquests elements estan estilitzats per coincidir amb l'aspecte i la sensació de la teva aplicació Ionic.

Entrades de Text

Les entrades de text s'utilitzen per capturar dades de text d'una sola línia de l'usuari.

<ion-item>
  <ion-label position="floating">Nom d'usuari</ion-label>
  <ion-input type="text"></ion-input>
</ion-item>
  • <ion-item>: Un contenidor per als elements del formulari.
  • <ion-label>: Una etiqueta per al camp d'entrada.
  • <ion-input>: El camp d'entrada en si.

Entrades de Contrasenya

Les entrades de contrasenya s'utilitzen per capturar informació sensible com les contrasenyes.

<ion-item>
  <ion-label position="floating">Contrasenya</ion-label>
  <ion-input type="password"></ion-input>
</ion-item>

Entrades de Correu Electrònic

Les entrades de correu electrònic són específicament per capturar adreces de correu electrònic.

<ion-item>
  <ion-label position="floating">Correu Electrònic</ion-label>
  <ion-input type="email"></ion-input>
</ion-item>

Entrades de Número

Les entrades de número s'utilitzen per capturar dades numèriques.

<ion-item>
  <ion-label position="floating">Edat</ion-label>
  <ion-input type="number"></ion-input>
</ion-item>

Elements Avançats del Formulari

Ionic també proporciona elements de formulari més avançats per capturar diferents tipus de dades.

Entrades de Selecció

Les entrades de selecció permeten als usuaris triar d'una llista d'opcions.

<ion-item>
  <ion-label>Fruita Preferida</ion-label>
  <ion-select>
    <ion-select-option value="apple">Poma</ion-select-option>
    <ion-select-option value="banana">Plàtan</ion-select-option>
    <ion-select-option value="cherry">Cirera</ion-select-option>
  </ion-select>
</ion-item>

Entrades de Data

Les entrades de data permeten als usuaris seleccionar una data.

<ion-item>
  <ion-label>Data de Naixement</ion-label>
  <ion-datetime display-format="MM/DD/YYYY"></ion-datetime>
</ion-item>

Entrades de Commutador

Les entrades de commutador s'utilitzen per a eleccions binàries, com ara interruptors d'activació/desactivació.

<ion-item>
  <ion-label>Activar Notificacions</ion-label>
  <ion-toggle></ion-toggle>
</ion-item>

Validació del Formulari

La validació del formulari és crucial per assegurar-se que les dades introduïdes per l'usuari són correctes i completes. Angular proporciona eines potents per a la validació de formularis, que es poden integrar fàcilment amb els formularis d'Ionic.

Formularis Basats en Plantilles

Els formularis basats en plantilles són senzills i adequats per a casos d'ús bàsics.

<form #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Nom d'usuari</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <div *ngIf="myForm.controls.username?.invalid && myForm.controls.username?.touched">
    El nom d'usuari és obligatori.
  </div>
  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>

Formularis Reactius

Els formularis reactius proporcionen més control i són adequats per a escenaris complexos.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      username: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.myForm.valid) {
      console.log(this.myForm.value);
    }
  }
}
<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <ion-item>
    <ion-label position="floating">Nom d'usuari</ion-label>
    <ion-input formControlName="username"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('username').invalid && myForm.get('username').touched">
    El nom d'usuari és obligatori.
  </div>

  <ion-item>
    <ion-label position="floating">Correu Electrònic</ion-label>
    <ion-input formControlName="email"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('email').invalid && myForm.get('email').touched">
    Es requereix un correu electrònic vàlid.
  </div>

  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>

Gestió de l'Enviament del Formulari

La gestió de l'enviament del formulari implica capturar les dades introduïdes per l'usuari i processar-les, com ara enviar-les a un servidor.

<form (ngSubmit)="onSubmit(myForm)" #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Nom d'usuari</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>
import { Component } from '@angular/core';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  onSubmit(form) {
    if (form.valid) {
      console.log(form.value);
    }
  }
}

Conclusió

Els formularis i les entrades són essencials per capturar dades dels usuaris en qualsevol aplicació. Ionic proporciona un conjunt complet d'elements de formulari que són fàcils d'utilitzar i integrar amb les capacitats de gestió de formularis d'Angular. Tant si estàs construint formularis senzills amb enfocaments basats en plantilles com formularis complexos amb formularis reactius, Ionic té les eines que necessites per crear formularis robustos i fàcils d'utilitzar.
