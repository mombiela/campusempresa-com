Introducció

En aquest tema, et guiarem a través de la construcció d'una aplicació de tasques pendents utilitzant Ionic. Aquest projecte t'ajudarà a entendre els conceptes bàsics d'Ionic, incloent-hi components d'interfície d'usuari, navegació, gestió d'estat i emmagatzematge de dades. Al final d'aquest tema, tindràs una aplicació de tasques pendents funcional i una sòlida comprensió de com construir aplicacions mòbils amb Ionic.

Configuració de l'entorn

Abans de començar a construir l'aplicació, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar Node.js i npm: Assegura't de tenir Node.js i npm instal·lats al teu sistema.
  • Instal·lar Ionic CLI: Executa la següent comanda per instal·lar Ionic CLI globalment: 
    npm install -g @ionic/cli
  • Crear un nou projecte Ionic: Utilitza Ionic CLI per crear un nou projecte: 
    ionic start todo-app blank
  • Navegar al directori del projecte: Mou-te al directori del projecte: 
    cd todo-app

Construir la interfície d'usuari

Començarem construint la interfície d'usuari per a la nostra aplicació de tasques pendents.

Pàgina principal

La pàgina principal mostrarà la llista de tasques pendents i un botó per afegir noves tasques.

  • Crear pàgina principal: Genera una nova pàgina utilitzant Ionic CLI:

    ionic generate page home

  • Actualitzar app-routing.module.ts: Configura la pàgina principal com a ruta per defecte.

    const routes: Routes = [
  { path: '', redirectTo: 'home', pathMatch: 'full' },
  { path: 'home', loadChildren: () => import('./home/home.module').then(m => m.HomePageModule) }
];

  • Dissenyar la pàgina principal: Modifica home.page.html per incloure una llista i un botó.

    <ion-header>
  <ion-toolbar>
    <ion-title>Llista de tasques</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-list>
    <ion-item *ngFor="let item of todoItems">
      {{ item }}
    </ion-item>
  </ion-list>
  <ion-fab vertical="bottom" horizontal="end" slot="fixed">
    <ion-fab-button (click)="addItem()">
      <ion-icon name="add"></ion-icon>
    </ion-fab-button>
  </ion-fab>
</ion-content>

Afegir tasques

Necessitem implementar la funcionalitat per afegir noves tasques a la llista.

  • Actualitzar home.page.ts: Afegeix un mètode per gestionar l'afegiment de noves tasques. 
    import { Component } from '@angular/core';
import { AlertController } from '@ionic/angular';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  todoItems: string[] = [];

  constructor(private alertController: AlertController) {}

  async addItem() {
    const alert = await this.alertController.create({
      header: 'Nova tasca',
      inputs: [
        {
          name: 'item',
          type: 'text',
          placeholder: 'Introdueix la teva tasca'
        }
      ],
      buttons: [
        {
          text: 'Cancel·lar',
          role: 'cancel',
          cssClass: 'secondary'
        },
        {
          text: 'Afegir',
          handler: (data) => {
            if (data.item) {
              this.todoItems.push(data.item);
            }
          }
        }
      ]
    });

    await alert.present();
  }
}

Gestió d'estat

Per gestionar l'estat de la nostra aplicació, utilitzarem un servei.

  • Crear un servei de tasques pendents: Genera un nou servei utilitzant Ionic CLI:

    ionic generate service todo

  • Implementar el servei: Actualitza todo.service.ts per gestionar la llista de tasques pendents.

    import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class TodoService {
  private todoItems: string[] = [];

  constructor() {}

  getItems(): string[] {
    return this.todoItems;
  }

  addItem(item: string) {
    this.todoItems.push(item);
  }
}

  • Actualitzar la pàgina principal per utilitzar el servei: Modifica home.page.ts per utilitzar el TodoService.

    import { Component } from '@angular/core';
import { AlertController } from '@ionic/angular';
import { TodoService } from '../services/todo.service';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  todoItems: string[];

  constructor(private alertController: AlertController, private todoService: TodoService) {
    this.todoItems = this.todoService.getItems();
  }

  async addItem() {
    const alert = await this.alertController.create({
      header: 'Nova tasca',
      inputs: [
        {
          name: 'item',
          type: 'text',
          placeholder: 'Introdueix la teva tasca'
        }
      ],
      buttons: [
        {
          text: 'Cancel·lar',
          role: 'cancel',
          cssClass: 'secondary'
        },
        {
          text: 'Afegir',
          handler: (data) => {
            if (data.item) {
              this.todoService.addItem(data.item);
              this.todoItems = this.todoService.getItems();
            }
          }
        }
      ]
    });

    await alert.present();
  }
}

Emmagatzematge de dades

Per persistir les tasques pendents, utilitzarem Ionic Storage.

  • Instal·lar Ionic Storage: Afegeix els paquets necessaris.

    npm install @ionic/storage-angular
npm install @ionic/storage

  • Configurar l'emmagatzematge: Actualitza app.module.ts per configurar Ionic Storage.

    import { IonicStorageModule } from '@ionic/storage-angular';

@NgModule({
  ...
  imports: [
    ...
    IonicStorageModule.forRoot()
  ],
  ...
})
export class AppModule {}

  • Actualitzar el servei de tasques pendents: Modifica todo.service.ts per utilitzar Ionic Storage.

    import { Injectable } from '@angular/core';
import { Storage } from '@ionic/storage-angular';

@Injectable({
  providedIn: 'root'
})
export class TodoService {
  private todoItems: string[] = [];

  constructor(private storage: Storage) {
    this.init();
  }

  async init() {
    await this.storage.create();
    this.todoItems = (await this.storage.get('todoItems')) || [];
  }

  getItems(): string[] {
    return this.todoItems;
  }

  async addItem(item: string) {
    this.todoItems.push(item);
    await this.storage.set('todoItems', this.todoItems);
  }
}
