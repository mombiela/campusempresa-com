Visió General

En aquesta secció, explorarem els fonaments de la navegació en aplicacions Ionic. La navegació és un aspecte crucial de qualsevol aplicació mòbil, ja que defineix com els usuaris es mouen entre diferents pàgines o vistes dins de l'aplicació. Cobrirem els conceptes bàsics, els tipus de navegació i proporcionarem exemples pràctics per ajudar-te a entendre com implementar la navegació en els teus projectes Ionic.

Conceptes Clau

  • Controlador de Navegació: Gestiona la pila de navegació i l'historial.
  • Pàgines: Vistes o pantalles individuals en l'aplicació.
  • Enrutament: Defineix els camins i components associats a cada ruta.
  • NavController: Un servei que proporciona mètodes per navegar entre pàgines.

Tipus de Navegació

Ionic admet diversos tipus de navegació, incloent:

  1. Navegació Push: Afegir una nova pàgina a la pila de navegació.
  2. Navegació Pop: Eliminar la pàgina actual de la pila de navegació.
  3. Navegació Root: Establir una nova pàgina root, netejant la pila de navegació.
  4. Navegació per Pestanyes: Utilitzar pestanyes per canviar entre diferents vistes.

Configurant la Navegació

Instal·lant Ionic CLI

Abans de començar, assegura't de tenir instal·lat el Ionic CLI. Si no, pots instal·lar-lo utilitzant npm:

npm install -g @ionic/cli

Creant un Nou Projecte Ionic

Crea un nou projecte Ionic utilitzant la següent comanda:

ionic start myApp blank

Navega al directori del projecte:

cd myApp

Afegint Pàgines

Per afegir una nova pàgina, utilitza el Ionic CLI:

ionic generate page Home
ionic generate page About

Configurant Rutes

Obre src/app/app-routing.module.ts i configura les rutes:

import { NgModule } from '@angular/core';
import { PreloadAllModules, RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: '',
    redirectTo: 'home',
    pathMatch: 'full'
  },
  {
    path: 'home',
    loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
  },
  {
    path: 'about',
    loadChildren: () => import('./about/about.module').then(m => m.AboutPageModule)
  }
];

@NgModule({
  imports: [
    RouterModule.forRoot(routes, { preloadingStrategy: PreloadAllModules })
  ],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Utilitzant NavController per a la Navegació

Per navegar entre pàgines, pots utilitzar el servei NavController. Aquí tens un exemple de com navegar des de la pàgina Home a la pàgina About:

Pàgina Home (home.page.ts)

import { Component } from '@angular/core';
import { NavController } from '@ionic/angular';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  constructor(private navCtrl: NavController) {}

  navigateToAbout() {
    this.navCtrl.navigateForward('/about');
  }
}

Plantilla de la Pàgina Home (home.page.html)

<ion-header>
  <ion-toolbar>
    <ion-title>
      Home
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-button (click)="navigateToAbout()">Go to About</ion-button>
</ion-content>

Resum

En aquesta secció, hem cobert els conceptes bàsics de la navegació en Ionic, incloent els diferents tipus de navegació i com configurar i configurar rutes en una aplicació Ionic. També hem demostrat com utilitzar el servei NavController per navegar entre pàgines. Entendre aquests conceptes és essencial per construir aplicacions mòbils intuïtives i fàcils d'usar amb Ionic. En les següents seccions, aprofundirem en tècniques avançades de navegació i millors pràctiques.
