Introducció

En aquest mòdul, et guiarem a través del procés de desenvolupament d'una aplicació de xat utilitzant Ionic. Això cobrirà tot, des de la configuració del teu entorn de desenvolupament fins al desplegament de la teva aplicació. Al final d'aquest mòdul, tindràs una aplicació de xat completament funcional i una sòlida comprensió d'Ionic.

Configuració de l'Entorn de Desenvolupament

Abans de començar a codificar, necessitem configurar el nostre entorn de desenvolupament.

  • Instal·lar Node.js i npm: Ionic requereix Node.js i npm (Node Package Manager).
  • Instal·lar Ionic CLI: La Interfície de Línia de Comandes d'Ionic (CLI) és una eina poderosa per desenvolupar aplicacions Ionic.
# Instal·lar Node.js i npm des de https://nodejs.org/
# Instal·lar Ionic CLI
npm install -g @ionic/cli
  • Crear un Nou Projecte Ionic: Utilitza l'Ionic CLI per crear un nou projecte.
ionic start chatApp blank
cd chatApp

Implementació de l'Autenticació

L'autenticació és essencial per a qualsevol aplicació de xat. Utilitzarem Firebase Authentication per a aquest propòsit.

  • Configurar Firebase: Crea un projecte Firebase i afegeix la teva aplicació.
# Instal·lar Firebase
npm install firebase @angular/fire
  • Configurar Firebase a la Teva Aplicació: Afegeix la configuració de Firebase a la teva aplicació.
// src/environments/environment.ts
export const environment = {
  firebaseConfig: {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
  }
};
  • Implementar Inici de Sessió i Registre: Utilitza els mètodes de Firebase Authentication.
// src/app/services/auth.service.ts
import { Injectable } from '@angular/core';
import { AngularFireAuth } from '@angular/fire/auth';

@Injectable({
  providedIn: 'root'
})
export class AuthService {
  constructor(private afAuth: AngularFireAuth) {}

  login(email: string, password: string) {
    return this.afAuth.signInWithEmailAndPassword(email, password);
  }

  register(email: string, password: string) {
    return this.afAuth.createUserWithEmailAndPassword(email, password);
  }

  logout() {
    return this.afAuth.signOut();
  }
}

Missatgeria en Temps Real

Per a la missatgeria en temps real, utilitzarem Firebase Firestore.

  • Configurar Firestore: Inicialitza Firestore a la teva aplicació.
// src/app/services/chat.service.ts
import { Injectable } from '@angular/core';
import { AngularFirestore } from '@angular/fire/firestore';

@Injectable({
  providedIn: 'root'
})
export class ChatService {
  constructor(private firestore: AngularFirestore) {}

  getMessages() {
    return this.firestore.collection('messages').valueChanges();
  }

  sendMessage(message: string) {
    return this.firestore.collection('messages').add({
      text: message,
      timestamp: new Date()
    });
  }
}
  • Mostrar Missatges a la Sala de Xat: Utilitza el ChatService per obtenir i mostrar missatges.
// src/app/chat-room/chat-room.page.ts
import { Component, OnInit } from '@angular/core';
import { ChatService } from '../services/chat.service';

@Component({
  selector: 'app-chat-room',
  templateUrl: './chat-room.page.html',
  styleUrls: ['./chat-room.page.scss'],
})
export class ChatRoomPage implements OnInit {
  messages = [];
  newMessage = '';

  constructor(private chatService: ChatService) {}

  ngOnInit() {
    this.chatService.getMessages().subscribe(messages => {
      this.messages = messages;
    });
  }

  sendMessage() {
    if (this.newMessage.trim()) {
      this.chatService.sendMessage(this.newMessage);
      this.newMessage = '';
    }
  }
}

Funcions Avançades

Per fer la nostra aplicació de xat més robusta, podem afegir funcions avançades com ara:

Perfils d'Usuari

  • Crear Pàgina de Perfil: Permet als usuaris actualitzar la seva informació de perfil.
ionic generate page Profile
  • Actualitzar la Informació del Perfil: Utilitza Firestore per emmagatzemar i recuperar perfils d'usuari.
// src/app/services/profile.service.ts
import { Injectable } from '@angular/core';
import { AngularFirestore } from '@angular/fire/firestore';

@Injectable({
  providedIn: 'root'
})
export class ProfileService {
  constructor(private firestore: AngularFirestore) {}

  getUserProfile(userId: string) {
    return this.firestore.collection('profiles').doc(userId).valueChanges();
  }

  updateUserProfile(userId: string, profileData: any) {
    return this.firestore.collection('profiles').doc(userId).set(profileData);
  }
}

Compartició d'Arxius

  • Permetre la Pujada d'Arxius: Utilitza Firebase Storage per pujar i compartir arxius.
// src/app/services/file.service.ts
import { Injectable } from '@angular/core';
import { AngularFireStorage } from '@angular/fire/storage';

@Injectable({
  providedIn: 'root'
})
export class FileService {
  constructor(private storage: AngularFireStorage) {}

  uploadFile(file: File) {
    const filePath = `uploads/${file.name}`;
    return this.storage.upload(filePath, file);
  }

  getFileUrl(filePath: string) {
    return this.storage.ref(filePath).getDownloadURL();
  }
}

Notificacions Push

  • Configurar Notificacions Push: Utilitza Firebase Cloud Messaging (FCM) per a notificacions push.
# Instal·lar FCM
npm install @ionic-native/fcm
  • Configurar FCM: Afegeix la configuració de FCM a la teva aplicació.
// src/app/app.component.ts
import { FCM } from '@ionic-native/fcm/ngx';

constructor(private fcm: FCM) {
  this.fcm.getToken().then(token => {
    console.log(token);
  });

  this.fcm.onNotification().subscribe(data => {
    if (data.wasTapped) {
      console.log('Rebut en segon pla');
    } else {
      console.log('Rebut en primer pla');
    }
  });
}
