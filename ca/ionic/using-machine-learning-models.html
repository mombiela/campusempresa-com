Introducció

En aquesta secció, explorarem com integrar models d'aprenentatge automàtic en una aplicació Ionic. Això inclourà entendre els conceptes bàsics de l'aprenentatge automàtic, com utilitzar models preentrenats i com desplegar aquests models dins d'una aplicació Ionic.

Conceptes Clau

  • Aprenentatge Automàtic (ML): Un subconjunt de la intel·ligència artificial que implica entrenar algoritmes per fer prediccions o decisions basades en dades.
  • Models Preentrenats: Models que ja han estat entrenats en un gran conjunt de dades i es poden utilitzar per a diverses tasques sense necessitat d'entrenar des de zero.
  • TensorFlow.js: Una biblioteca de JavaScript per entrenar i desplegar models de ML al navegador i a Node.js.

Configuració de l'Entorn

Abans de començar a integrar models d'aprenentatge automàtic, necessitem configurar el nostre entorn de desenvolupament.

  1. Instal·lar Ionic CLI:

    npm install -g @ionic/cli

  2. Crear un Nou Projecte Ionic:

    ionic start ml-app blank
cd ml-app

  3. Instal·lar TensorFlow.js:

    npm install @tensorflow/tfjs

Utilitzant un Model Preentrenat

Utilitzarem un model preentrenat per a la classificació d'imatges en aquest exemple. TensorFlow.js proporciona diversos models preentrenats que es poden integrar fàcilment.

Carregant el Model

Primer, necessitem carregar el model preentrenat. Utilitzarem el model MobileNet per a la classificació d'imatges.

import * as tf from '@tensorflow/tfjs';
import * as mobilenet from '@tensorflow-models/mobilenet';

async function loadModel() {
    const model = await mobilenet.load();
    console.log('Model carregat amb èxit');
    return model;
}

loadModel();

Fent Prediccions

Un cop el model està carregat, podem utilitzar-lo per fer prediccions sobre imatges.

async function classifyImage(imageElement) {
    const model = await loadModel();
    const predictions = await model.classify(imageElement);
    console.log('Prediccions: ', predictions);
}

// Exemple d'ús amb un element d'imatge
const img = document.getElementById('image');
classifyImage(img);

Mostrant les Prediccions a Ionic

Per mostrar les prediccions en una aplicació Ionic, podem vincular els resultats a la interfície d'usuari.

<ion-header>
  <ion-toolbar>
    <ion-title>Classificació d'Imatges</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-img id="image" src="path_to_image.jpg"></ion-img>
  <ion-button (click)="classifyImage()">Classificar Imatge</ion-button>
  <ion-list>
    <ion-item *ngFor="let prediction of predictions">
      {{ prediction.className }}: {{ prediction.probability | percent }}
    </ion-item>
  </ion-list>
</ion-content>
import { Component } from '@angular/core';
import * as tf from '@tensorflow/tfjs';
import * as mobilenet from '@tensorflow-models/mobilenet';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  predictions: any[] = [];

  async classifyImage() {
    const img = document.getElementById('image') as HTMLImageElement;
    const model = await mobilenet.load();
    this.predictions = await model.classify(img);
  }
}

Temes Avançats

Per a usuaris avançats, podem explorar l'entrenament i el desplegament de models personalitzats.

Entrenament de Models Personalitzats

Entrenar un model personalitzat implica diversos passos, incloent la recopilació de dades, el preprocessament, la definició del model, l'entrenament i l'avaluació. TensorFlow.js permet realitzar tots aquests passos directament en JavaScript.

Desplegament de Models

Desplegar un model en una aplicació Ionic es pot fer utilitzant TensorFlow.js, però per a models més complexos, podríeu considerar utilitzar un servei de backend com TensorFlow Serving o una solució basada en el núvol.

Conclusió

Integrar models d'aprenentatge automàtic en una aplicació Ionic pot millorar significativament les seves capacitats. Aprofitant models preentrenats, podeu afegir ràpidament funcions avançades com la classificació d'imatges. Per a casos d'ús més avançats, TensorFlow.js proporciona les eines necessàries per entrenar i desplegar models personalitzats. Amb aquest coneixement, podeu començar a construir aplicacions intel·ligents que aprofitin el poder de l'aprenentatge automàtic.
