Les notificacions push són una eina poderosa per involucrar els usuaris proporcionant informació oportuna i rellevant directament als seus dispositius. En aquest tema, cobrirem els aspectes essencials per implementar notificacions push en una aplicació Ionic, des de nivells principiants fins a avançats.

Introducció a les Notificacions Push

  • Definició: Les notificacions push són missatges enviats des d'un servidor al dispositiu d'un usuari, fins i tot quan l'aplicació no està funcionant activament.
  • Usos: Alertes, recordatoris, promocions, actualitzacions, etc.
  • Plataformes: Android, iOS i Web.

Configuració de les Notificacions Push a Ionic

Prerequisits

  • Ionic CLI: Assegureu-vos de tenir la versió més recent de l'Ionic CLI instal·lada.
  • Compte de Firebase: Firebase Cloud Messaging (FCM) s'utilitzarà per enviar notificacions.
  • Capacitor o Cordova: Ionic admet tant Capacitor com Cordova per a la funcionalitat nativa.

Instal·lació dels Plugins Necessaris

  1. Capacitor:

    npm install @capacitor/push-notifications
npx cap sync

  2. Cordova:

    ionic cordova plugin add phonegap-plugin-push
npm install @ionic-native/push

Configuració de Firebase

  1. Crear un Projecte a Firebase:

    • Aneu a la Consola de Firebase.
    • Creeu un nou projecte o utilitzeu-ne un d'existent.

  2. Afegir la Vostra Aplicació a Firebase:

    • Registreu la vostra aplicació (iOS i Android).
    • Descarregueu els fitxers google-services.json (per a Android) i GoogleService-Info.plist (per a iOS) i col·loqueu-los a les directoris adequades.

Inicialització de les Notificacions Push

Exemple amb Capacitor

  1. Importar i Inicialitzar:

    import { PushNotifications } from '@capacitor/push-notifications';

PushNotifications.requestPermissions().then(result => {
  if (result.receive === 'granted') {
    PushNotifications.register();
  } else {
    // Mostra algun error
  }
});

  2. Gestió de les Notificacions:

    PushNotifications.addListener('pushNotificationReceived', (notification) => {
  console.log('Notificació rebuda: ', notification);
});

PushNotifications.addListener('pushNotificationActionPerformed', (notification) => {
  console.log('Acció de notificació realitzada: ', notification);
});

Exemple amb Cordova

  1. Importar i Inicialitzar: 
    import { Push, PushObject, PushOptions } from '@ionic-native/push/ngx';

constructor(private push: Push) {}

initializePushNotifications() {
  const options: PushOptions = {
    android: {},
    ios: {
      alert: 'true',
      badge: true,
      sound: 'false'
    }
  };

  const pushObject: PushObject = this.push.init(options);

  pushObject.on('notification').subscribe((notification: any) => {
    console.log('Notificació rebuda', notification);
  });

  pushObject.on('registration').subscribe((registration: any) => {
    console.log('Dispositiu registrat', registration);
  });

  pushObject.on('error').subscribe(error => console.error('Error amb el plugin Push', error));
}

Temes Avançats

Personalització de les Notificacions

  • Estructura del Payload: Personalitzeu el payload per incloure dades addicionals. 
    {
  "to": "device_token",
  "notification": {
    "title": "Nou Missatge",
    "body": "Heu rebut un nou missatge.",
    "icon": "myicon"
  },
  "data": {
    "additionalData": "value"
  }
}

Gestió de les Notificacions en Segon Pla

  • Capacitor:

    PushNotifications.addListener('pushNotificationReceived', (notification) => {
  if (notification.foreground) {
    // Gestionar notificació en primer pla
  } else {
    // Gestionar notificació en segon pla
  }
});

  • Cordova:

    pushObject.on('notification').subscribe((notification: any) => {
  if (notification.additionalData.foreground) {
    // Gestionar notificació en primer pla
  } else {
    // Gestionar notificació en segon pla
  }
});

Programació de les Notificacions

  • Notificacions Locals: Utilitzeu el plugin Local Notifications per programar notificacions. 
    import { LocalNotifications } from '@capacitor/local-notifications';

LocalNotifications.schedule({
  notifications: [
    {
      title: "Notificació Programada",
      body: "Aquesta és una notificació programada.",
      id: 1,
      schedule: { at: new Date(Date.now() + 1000 * 60 * 5) }, // 5 minuts a partir d'ara
      sound: null,
      attachments: null,
      actionTypeId: "",
      extra: null
    }
  ]
});

Conclusió

Les notificacions push són una característica crucial per a les aplicacions mòbils modernes, permetent la comunicació en temps real amb els usuaris. Seguint aquesta guia, hauríeu de poder configurar i personalitzar les notificacions push a la vostra aplicació Ionic, gestionar-les tant en estats de primer pla com de segon pla, i fins i tot programar notificacions locals. A mesura que avanceu, considereu explorar casos d'ús més complexos i integrar analítiques per mesurar l'efectivitat de les vostres notificacions.
