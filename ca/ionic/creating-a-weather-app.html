Introducció

En aquest curs, et guiarem a través del procés de creació d'una aplicació del temps utilitzant Ionic. Aquest curs està dissenyat per portar-te des d'un nivell principiant fins a un nivell avançat, cobrint tots els conceptes i tècniques essencials necessaris per construir una aplicació del temps completament funcional.

Prerequisits

Abans de començar aquest curs, hauríeu de tenir:

  • Coneixements bàsics d'HTML, CSS i JavaScript.
  • Node.js i npm instal·lats a la vostra màquina.
  • Comprensió bàsica d'Angular (ja que Ionic està construït sobre Angular).

Configurant l'Entorn

Per començar amb Ionic, necessiteu configurar el vostre entorn de desenvolupament.

Instal·lant Ionic CLI

Primer, instal·leu Ionic CLI globalment utilitzant npm:

npm install -g @ionic/cli

Creant un Nou Projecte Ionic

Creeu un nou projecte Ionic utilitzant la següent comanda:

ionic start weather-app blank --type=angular

Navegueu al directori del vostre projecte:

cd weather-app

Construint la Interfície d'Usuari

En aquesta secció, construirem la interfície d'usuari bàsica per a la nostra aplicació del temps.

Creant Pàgines

Ionic utilitza el sistema de routing d'Angular per navegar entre pàgines. Creem una pàgina d'inici i una pàgina de detalls.

ionic generate page home
ionic generate page details

Dissenyant la Pàgina d'Inici

Obriu src/app/home/home.page.html i afegiu el següent codi per crear un disseny bàsic:

<ion-header>
  <ion-toolbar>
    <ion-title>Weather App</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-item>
    <ion-label position="floating">Enter City</ion-label>
    <ion-input [(ngModel)]="city"></ion-input>
  </ion-item>
  <ion-button expand="full" (click)="getWeather()">Get Weather</ion-button>
  <ion-card *ngIf="weather">
    <ion-card-header>
      <ion-card-title>{{ weather.name }}</ion-card-title>
    </ion-card-header>
    <ion-card-content>
      <p>Temperature: {{ weather.main.temp }}°C</p>
      <p>Condition: {{ weather.weather[0].description }}</p>
    </ion-card-content>
  </ion-card>
</ion-content>

Afegint Lògica a la Pàgina d'Inici

Obriu src/app/home/home.page.ts i afegiu el següent codi per gestionar l'entrada de l'usuari i obtenir dades del temps:

import { Component } from '@angular/core';
import { HttpClient } from '@angular/common/http';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  city: string;
  weather: any;

  constructor(private http: HttpClient) {}

  getWeather() {
    const apiKey = 'YOUR_API_KEY';
    const url = `https://api.openweathermap.org/data/2.5/weather?q=${this.city}&appid=${apiKey}&units=metric`;

    this.http.get(url).subscribe(data => {
      this.weather = data;
    });
  }
}

Integrant amb una API del Temps

Per obtenir dades del temps en temps real, utilitzarem l'API d'OpenWeatherMap.

Obtenint una Clau API

  • Registreu-vos a OpenWeatherMap.
  • Navegueu a la secció d'API i genereu una clau API.

Utilitzant la Clau API

Substituïu 'YOUR_API_KEY' en el mètode getWeather amb la vostra clau API real.

Funcionalitats Avançades

Un cop la funcionalitat bàsica estigui funcionant, podem afegir funcionalitats més avançades.

Afegint un Indicador de Càrrega

Per millorar l'experiència de l'usuari, mostreu un indicador de càrrega mentre es recuperen les dades.

import { LoadingController } from '@ionic/angular';

constructor(private http: HttpClient, private loadingController: LoadingController) {}

async getWeather() {
  const loading = await this.loadingController.create({
    message: 'Fetching weather...',
  });
  await loading.present();

  const apiKey = 'YOUR_API_KEY';
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${this.city}&appid=${apiKey}&units=metric`;

  this.http.get(url).subscribe(data => {
    this.weather = data;
    loading.dismiss();
  }, error => {
    loading.dismiss();
    console.error('Error fetching weather data', error);
  });
}

Navegant a la Pàgina de Detalls

Passeu dades a la pàgina de detalls i mostreu informació meteorològica més detallada.

Modificant la Pàgina d'Inici

<ion-button expand="full" (click)="goToDetails()">View Details</ion-button>
import { NavController } from '@ionic/angular';

constructor(private http: HttpClient, private loadingController: LoadingController, private navCtrl: NavController) {}

goToDetails() {
  this.navCtrl.navigateForward('/details', {
    queryParams: {
      weather: JSON.stringify(this.weather)
    }
  });
}

Afegint Lògica a la Pàgina de Detalls

Obriu src/app/details/details.page.ts i afegiu el següent codi:

import { Component, OnInit } from '@angular/core';
import { ActivatedRoute } from '@angular/router';

@Component({
  selector: 'app-details',
  templateUrl: './details.page.html',
  styleUrls: ['./details.page.scss'],
})
export class DetailsPage implements OnInit {
  weather: any;

  constructor(private route: ActivatedRoute) {}

  ngOnInit() {
    this.route.queryParams.subscribe(params => {
      if (params && params.weather) {
        this.weather = JSON.parse(params.weather);
      }
    });
  }
}
