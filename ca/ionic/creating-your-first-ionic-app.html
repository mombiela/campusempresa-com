Introducció

En aquesta secció, et guiarem a través del procés de creació de la teva primera aplicació Ionic. Ionic és un marc popular per construir aplicacions mòbils multiplataforma utilitzant tecnologies web com HTML, CSS i JavaScript. Al final d'aquesta secció, tindràs una aplicació bàsica d'Ionic funcionant a la teva màquina local.

Prerequisits

Abans de començar, assegura't de tenir instal·lat al teu ordinador el següent:

  • Node.js i npm (Node Package Manager)
  • Ionic CLI (Command Line Interface)
  • Un editor de codi (per exemple, Visual Studio Code)

Configurant el teu entorn

  1. Instal·la Node.js i npm: Descarrega i instal·la Node.js des de nodejs.org. Això també instal·larà npm.
  2. Instal·la Ionic CLI: Obre el teu terminal o símbol del sistema i executa la següent comanda: 
    npm install -g @ionic/cli
    Aquesta comanda instal·la Ionic CLI globalment al teu sistema.

Creant un nou projecte Ionic

  1. Obre el teu terminal i navega fins al directori on vols crear el teu projecte.

  2. Executa la següent comanda per crear un nou projecte Ionic:

    ionic start myFirstApp blank
    • myFirstApp és el nom del teu projecte.
    • blank és la plantilla inicial. Pots triar altres plantilles com tabs o sidemenu.

  3. Navega fins al directori del teu projecte:

    cd myFirstApp

Executant la teva aplicació Ionic

  1. Serveix l'aplicació localment executant: 
    ionic serve
    Aquesta comanda compilarà la teva aplicació i l'obrirà al teu navegador web per defecte. Hauries de veure una aplicació Ionic en blanc funcionant.

Entenent l'estructura del projecte

Aquí tens una breu visió general dels fitxers i directoris clau en el teu nou projecte Ionic:

  • src/: Conté el codi font de la teva aplicació.
    • app/: Conté el mòdul principal de l'aplicació i el component arrel.
    • assets/: Conté actius estàtics com imatges i fonts.
    • environments/: Conté fitxers de configuració específics per a cada entorn.
    • theme/: Conté estils globals i variables.
    • index.html: El fitxer HTML principal.
    • main.ts: El punt d'entrada principal de l'aplicació.
  • www/: Conté la sortida compilada de la teva aplicació.
  • ionic.config.json: Fitxer de configuració per a Ionic CLI.

Afegint un component simple

Anem a afegir un component simple a la teva aplicació per mostrar un missatge de benvinguda.

  1. Genera un nou component:

    ionic generate component Welcome

  2. Modifica el component:

    • Obre src/app/welcome/welcome.component.html i afegeix el següent codi: 
      <div>
  <h1>Benvingut a la meva primera aplicació Ionic!</h1>
</div>
    • Obre src/app/welcome/welcome.component.ts i assegura't que es vegi així: 
      import { Component } from '@angular/core';

@Component({
  selector: 'app-welcome',
  templateUrl: './welcome.component.html',
  styleUrls: ['./welcome.component.scss']
})
export class WelcomeComponent {}

  3. Utilitza el component a la teva aplicació:

    • Obre src/app/app.component.html i afegeix la següent línia: 
      <app-welcome></app-welcome>

  4. Serveix l'aplicació de nou:

    ionic serve

    Ara hauries de veure el missatge de benvinguda mostrat a la teva aplicació.

Conclusió

Felicitats! Has creat amb èxit la teva primera aplicació Ionic i has afegit un component simple. En aquesta secció, has après a configurar el teu entorn de desenvolupament, crear un nou projecte Ionic, entendre l'estructura del projecte i afegir un component bàsic. Aquest coneixement fonamental t'ajudarà a mesura que avancis en temes més avançats en el desenvolupament amb Ionic.
