Introducció

En aquesta secció, explorarem com utilitzar el mòdul HttpClient d'Angular dins d'una aplicació Ionic. HttpClient és una manera potent i flexible d'interactuar amb serveis de backend a través d'HTTP. Al final d'aquesta secció, seràs capaç de realitzar operacions CRUD (Crear, Llegir, Actualitzar, Eliminar) utilitzant HttpClient.

Configurant HttpClient a Ionic

Instal·lant HttpClient

Primer, assegura't que tens instal·lat el mòdul HttpClient d'Angular. Normalment està inclòs per defecte en els projectes Angular, però el pots afegir si falta.

npm install @angular/common@latest

Important HttpClientModule

A continuació, importa el HttpClientModule al fitxer del mòdul principal de la teva aplicació Ionic (app.module.ts).

import { HttpClientModule } from '@angular/common/http';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    IonicModule.forRoot(),
    AppRoutingModule,
    HttpClientModule // Afegeix HttpClientModule aquí
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule {}

Realitzant sol·licituds HTTP

Sol·licitud GET

Una sol·licitud GET s'utilitza per recuperar dades d'un servidor. Aquí tens com pots realitzar una sol·licitud GET utilitzant HttpClient.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  getData() {
    return this.http.get(this.apiUrl);
  }
}

Sol·licitud POST

Una sol·licitud POST s'utilitza per enviar dades a un servidor. A continuació tens un exemple de com realitzar una sol·licitud POST.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  postData(data: any) {
    return this.http.post(this.apiUrl, data);
  }
}

Sol·licitud PUT

Una sol·licitud PUT s'utilitza per actualitzar dades existents al servidor.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  updateData(id: string, data: any) {
    return this.http.put(`${this.apiUrl}/${id}`, data);
  }
}

Sol·licitud DELETE

Una sol·licitud DELETE s'utilitza per eliminar dades del servidor.

import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  deleteData(id: string) {
    return this.http.delete(`${this.apiUrl}/${id}`);
  }
}

Gestionant respostes i errors

Gestionant respostes

Pots gestionar les respostes subscrivint-te a l'observable retornat pels mètodes HttpClient.

this.dataService.getData().subscribe(
  data => {
    console.log('Dades rebudes:', data);
  },
  error => {
    console.error('Error:', error);
  }
);

Gestió d'Errors

Per gestionar errors de manera més elegant, pots utilitzar operadors de RxJS com catchError.

import { catchError } from 'rxjs/operators';
import { throwError } from 'rxjs';

this.dataService.getData().pipe(
  catchError(error => {
    console.error('Error:', error);
    return throwError(error);
  })
).subscribe(
  data => {
    console.log('Dades rebudes:', data);
  }
);

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'utilitzar el mòdul HttpClient d'Angular dins d'una aplicació Ionic. Hem après com configurar HttpClient, realitzar diversos tipus de sol·licituds HTTP (GET, POST, PUT, DELETE), i gestionar respostes i errors. Dominar aquests conceptes et permetrà interactuar amb serveis de backend de manera efectiva, fent que les teves aplicacions Ionic siguin més dinàmiques i funcionals.
