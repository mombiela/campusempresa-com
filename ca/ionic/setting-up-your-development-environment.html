El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar el teu entorn de desenvolupament és el primer pas per convertir-te en un expert amb Ionic. Aquesta secció et guiarà a través dels passos necessaris per preparar el teu entorn per al desenvolupament amb Ionic, des d'instal·lar el programari necessari fins a configurar el teu primer projecte.

Prerequisits

Abans de submergir-te en la configuració, assegura't de tenir els següents prerequisits:

  • Coneixements bàsics d'HTML, CSS i JavaScript.
  • Node.js i npm instal·lats a la teva màquina.

Instal·lació de Node.js i npm

Ionic requereix Node.js i npm (Node Package Manager) per gestionar dependències i executar scripts. Segueix aquests passos per instal·lar-los:

  1. Descarrega Node.js:

    • Visita el lloc web de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.

  2. Instal·la Node.js:

    • Executa l'instal·lador descarregat i segueix les instruccions en pantalla.
    • Verifica la instal·lació executant les següents comandes al teu terminal: 
      node -v
npm -v
    • Hauries de veure els números de versió tant de Node.js com de npm.

Instal·lació de l'Ionic CLI

La Interfície de Línia de Comandes d'Ionic (CLI) és una eina poderosa que t'ajuda a crear, construir i executar aplicacions Ionic. Per instal·lar l'Ionic CLI, utilitza npm:

npm install -g @ionic/cli

Verifica la instal·lació executant:

ionic --version

Hauries de veure el número de versió de l'Ionic CLI.

Configuració del teu primer projecte Ionic

Amb l'Ionic CLI instal·lat, ara pots crear el teu primer projecte Ionic. Segueix aquests passos:

  1. Crea un nou projecte:

    ionic start myFirstApp blank
    • myFirstApp és el nom del teu projecte.
    • blank és la plantilla inicial. Pots triar altres plantilles com tabs o sidemenu.

  2. Navega al directori del teu projecte:

    cd myFirstApp

  3. Serveix l'aplicació:

    ionic serve
    • Aquesta comanda iniciarà un servidor de desenvolupament local i obrirà la teva aplicació en un navegador web.
    • Hauries de veure l'aplicació predeterminada d'Ionic executant-se al teu navegador.

Configuració del teu entorn de desenvolupament

Per millorar la teva experiència de desenvolupament, considera configurar les següents eines:

  • Visual Studio Code (VS Code):

    • Un editor de codi popular amb un excel·lent suport per a JavaScript i TypeScript.
    • Descarrega i instal·la des del lloc web de VS Code.

  • Extensions d'Ionic per a VS Code:

    • Instal·la extensions com "Ionic" i "Angular Language Service" per a una millor completació de codi i depuració.

  • Git:

    • Sistema de control de versions per gestionar el teu codi.
    • Descarrega i instal·la des del lloc web de Git.

Exemple: Crear un component Ionic simple

Creem un component simple per entendre els conceptes bàsics del desenvolupament amb Ionic.

  1. Genera un nou component:

    ionic generate component my-component

  2. Edita el component:

    • Obre src/app/my-component/my-component.component.html i afegeix el següent codi: 
      <ion-header>
  <ion-toolbar>
    <ion-title>
      My Component
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <p>Hello, this is my first Ionic component!</p>
</ion-content>

  3. Utilitza el component en una pàgina:

    • Obre src/app/home/home.page.html i afegeix el següent codi: 
      <app-my-component></app-my-component>

  4. Serveix l'aplicació:

    ionic serve
    • Hauries de veure el nou component renderitzat a la pàgina d'inici.

Conclusió

Configurar el teu entorn de desenvolupament és un pas crucial en el teu viatge de desenvolupament amb Ionic. Seguint els passos descrits en aquesta secció, ara hauries de tenir un entorn de desenvolupament Ionic completament funcional. També has creat el teu primer projecte Ionic i un component simple. En les properes seccions, aprofundirem en les característiques i capacitats d'Ionic, construint sobre aquesta base.
