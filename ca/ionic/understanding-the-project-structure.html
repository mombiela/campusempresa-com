En aquesta secció, explorarem l'estructura d'un projecte Ionic. Comprendre l'estructura del projecte és crucial per navegar i gestionar eficientment la teva base de codi, especialment a mesura que la teva aplicació creix en complexitat.

Conceptes Clau

  • Ionic Framework: Un framework popular per construir aplicacions mòbils multiplataforma utilitzant tecnologies web com HTML, CSS i JavaScript.
  • Estructura del Projecte: L'organització dels fitxers i directoris dins d'un projecte Ionic, que ajuda a mantenir una base de codi neta i escalable.

Estructura Bàsica d'un Projecte Ionic

Quan crees un nou projecte Ionic, el framework configura una estructura per defecte per a tu. Aquí tens una visió general dels principals directoris i fitxers que trobaràs:

Directori Arrel

  • node_modules/: Conté tots els paquets npm instal·lats per al teu projecte.
  • platforms/: Conté codi específic per a plataformes (per exemple, Android, iOS).
  • resources/: Emmagatzema icones i pantalles d'inici per a diferents plataformes.
  • src/: El directori principal de codi font on resideix el teu codi d'aplicació.
  • www/: El directori de sortida on es guarda l'aplicació compilada.
  • config.xml: Fitxer de configuració per a Cordova.
  • ionic.config.json: Fitxer de configuració per a l'Ionic CLI.
  • package.json: Llista les dependències i scripts del projecte.
  • tsconfig.json: Fitxer de configuració de TypeScript.

Directori `src/`

El directori src/ és on passaràs la major part del teu temps. Aquí tens una descripció del seu contingut:

  • app/: Conté el mòdul principal de l'aplicació i el component principal.
  • assets/: Emmagatzema recursos estàtics com imatges, fonts, etc.
  • environments/: Fitxers de configuració per a diferents entorns (per exemple, desenvolupament, producció).
  • pages/: Conté pàgines individuals de la teva aplicació.
  • services/: Conté serveis per gestionar la lògica de negoci i les dades.
  • theme/: Conté estils globals i variables de tema.

Exemple d'Estructura de Projecte

Aquí tens un exemple de com podria ser l'estructura típica d'un projecte Ionic:

my-ionic-app/
├── node_modules/
├── platforms/
├── resources/
├── src/
│   ├── app/
│   │   ├── app.component.ts
│   │   ├── app.module.ts
│   │   └── app-routing.module.ts
│   ├── assets/
│   │   └── icon.png
│   ├── environments/
│   │   ├── environment.ts
│   │   └── environment.prod.ts
│   ├── pages/
│   │   ├── home/
│   │   │   ├── home.page.html
│   │   │   ├── home.page.scss
│   │   │   └── home.page.ts
│   ├── services/
│   │   └── data.service.ts
│   ├── theme/
│   │   ├── variables.scss
│   │   └── global.scss
│   ├── index.html
│   ├── main.ts
│   └── polyfills.ts
├── www/
├── config.xml
├── ionic.config.json
├── package.json
└── tsconfig.json

Explicació Detallada dels Fitxers Clau

`app.module.ts`

Aquest fitxer és el mòdul arrel de la teva aplicació. Importa els mòduls necessaris i declara components.

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { RouteReuseStrategy } from '@angular/router';

import { IonicModule, IonicRouteStrategy } from '@ionic/angular';
import { AppComponent } from './app.component';
import { AppRoutingModule } from './app-routing.module';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, IonicModule.forRoot(), AppRoutingModule],
  providers: [{ provide: RouteReuseStrategy, useClass: IonicRouteStrategy }],
  bootstrap: [AppComponent],
})
export class AppModule {}

`home.page.ts`

Aquest fitxer defineix la lògica per al component de la pàgina d'inici.

import { Component } from '@angular/core';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  constructor() {}
}

`home.page.html`

Aquest fitxer conté la plantilla HTML per a la pàgina d'inici.

<ion-header>
  <ion-toolbar>
    <ion-title>
      Home
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <p>Welcome to the Home Page!</p>
</ion-content>

Conclusió

Comprendre l'estructura d'un projecte Ionic és fonamental per a un desenvolupament efectiu. Familiaritzant-te amb l'organització dels fitxers i directoris, podràs navegar pel teu projecte més eficientment i mantenir una base de codi neta. A mesura que avancis en aquest curs, aquest coneixement fonamental t'ajudarà a construir aplicacions més complexes i escalables.
