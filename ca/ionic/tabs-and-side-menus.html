Introducció

En aquesta secció, explorarem com crear i gestionar pestanyes i menús laterals en aplicacions Ionic. Les pestanyes i els menús laterals són components essencials per construir aplicacions mòbils intuïtives i navegables. Al final d'aquesta secció, seràs capaç d'implementar aquestes característiques en els teus projectes Ionic.

Pestanyes en Ionic

Què són les Pestanyes?

Les pestanyes són un patró de navegació comú que permet als usuaris canviar entre diferents vistes o pàgines dins d'una aplicació. Cada pestanya representa una secció diferent de l'aplicació.

Creant un Disseny Bàsic de Pestanyes

Per crear un disseny bàsic de pestanyes en Ionic, pots utilitzar el component ion-tabs juntament amb els components ion-tab-bar i ion-tab-button.

Exemple:

<ion-tabs>
  <ion-tab-bar slot="bottom">
    <ion-tab-button tab="home">
      <ion-icon name="home"></ion-icon>
      <ion-label>Home</ion-label>
    </ion-tab-button>

    <ion-tab-button tab="settings">
      <ion-icon name="settings"></ion-icon>
      <ion-label>Settings</ion-label>
    </ion-tab-button>
  </ion-tab-bar>
</ion-tabs>
  • ion-tabs: El contenidor per a la navegació de pestanyes.
  • ion-tab-bar: La barra que conté els botons de les pestanyes.
  • ion-tab-button: Cada botó representa una pestanya.

Enllaçant Pestanyes a Pàgines

Per enllaçar pestanyes a pàgines específiques, has de definir rutes en el teu fitxer app-routing.module.ts.

Exemple:

const routes: Routes = [
  {
    path: 'tabs',
    component: TabsPage,
    children: [
      {
        path: 'home',
        loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
      },
      {
        path: 'settings',
        loadChildren: () => import('./settings/settings.module').then(m => m.SettingsPageModule)
      }
    ]
  },
  {
    path: '',
    redirectTo: '/tabs/home',
    pathMatch: 'full'
  }
];
  • Defineix rutes filles sota el camí tabs.
  • Utilitza loadChildren per carregar de manera mandrosa els mòduls per a cada pestanya.

Menús Laterals en Ionic

Què és un Menú Lateral?

Un menú lateral (també conegut com a calaix o calaix de navegació) és un panell que es desplaça des del costat de la pantalla, proporcionant opcions de navegació.

Creant un Menú Lateral Bàsic

Per crear un menú lateral, pots utilitzar el component ion-menu juntament amb ion-menu-toggle i ion-content.

Exemple:

<ion-menu side="start" contentId="main-content">
  <ion-header>
    <ion-toolbar>
      <ion-title>Menu</ion-title>
    </ion-toolbar>
  </ion-header>
  <ion-content>
    <ion-list>
      <ion-menu-toggle auto-hide="false">
        <ion-item routerLink="/home">
          <ion-icon slot="start" name="home"></ion-icon>
          <ion-label>Home</ion-label>
        </ion-item>
        <ion-item routerLink="/settings">
          <ion-icon slot="start" name="settings"></ion-icon>
          <ion-label>Settings</ion-label>
        </ion-item>
      </ion-menu-toggle>
    </ion-list>
  </ion-content>
</ion-menu>

<ion-router-outlet id="main-content"></ion-router-outlet>
  • ion-menu: El contenidor per al menú lateral.
  • side="start": Especifica que el menú ha d'aparèixer des del costat esquerre.
  • contentId: Enllaça el menú a l'àrea de contingut principal.
  • ion-menu-toggle: Alterna el menú obert i tancat.
  • ion-router-outlet: L'àrea de contingut principal on es mostraran les pàgines.

Combinant Pestanyes i Menús Laterals

Pots combinar pestanyes i menús laterals per crear una estructura de navegació més complexa.

Exemple:

<ion-menu side="start" contentId="main-content">
  <ion-header>
    <ion-toolbar>
      <ion-title>Menu</ion-title>
    </ion-toolbar>
  </ion-header>
  <ion-content>
    <ion-list>
      <ion-menu-toggle auto-hide="false">
        <ion-item routerLink="/tabs/home">
          <ion-icon slot="start" name="home"></ion-icon>
          <ion-label>Home</ion-label>
        </ion-item>
        <ion-item routerLink="/tabs/settings">
          <ion-icon slot="start" name="settings"></ion-icon>
          <ion-label>Settings</ion-label>
        </ion-item>
      </ion-menu-toggle>
    </ion-list>
  </ion-content>
</ion-menu>

<ion-router-outlet id="main-content"></ion-router-outlet>

<ion-tabs>
  <ion-tab-bar slot="bottom">
    <ion-tab-button tab="home">
      <ion-icon name="home"></ion-icon>
      <ion-label>Home</ion-label>
    </ion-tab-button>
    <ion-tab-button tab="settings">
      <ion-icon name="settings"></ion-icon>
      <ion-label>Settings</ion-label>
    </ion-tab-button>
  </ion-tab-bar>
</ion-tabs>
  • Aquest exemple combina tant pestanyes com un menú lateral, proporcionant múltiples opcions de navegació.

Conclusió

Les pestanyes i els menús laterals són components de navegació potents en Ionic que ajuden a crear una interfície d'usuari amigable. Entenent com implementar i combinar aquests components, pots construir aplicacions mòbils complexes i intuïtives. Practica creant diferents dissenys i explora opcions de personalització addicionals per millorar els teus projectes Ionic.
