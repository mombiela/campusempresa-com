Introducció

En aquesta secció, explorarem com utilitzar la càmera i gestionar fitxers en una aplicació Ionic. Això inclou capturar imatges, accedir a la biblioteca de fotos del dispositiu i gestionar fitxers dins de l'aplicació. Al final d'aquesta secció, seràs capaç d'integrar la funcionalitat de la càmera i les capacitats de gestió de fitxers a les teves aplicacions Ionic.

Configuració de l'Entorn

Abans de submergir-nos en el codi, assegura't de tenir els següents requisits previs:

  • Node.js instal·lat
  • Ionic CLI instal·lat (npm install -g @ionic/cli)
  • Un projecte Ionic nou o existent

Instal·lació dels Plugins Necessaris

Per treballar amb la càmera i el sistema de fitxers, necessitem instal·lar els següents plugins de Cordova:

  • @ionic-native/camera
  • @ionic-native/file

Executa les següents comandes per instal·lar aquests plugins:

ionic cordova plugin add cordova-plugin-camera
npm install @ionic-native/camera

ionic cordova plugin add cordova-plugin-file
npm install @ionic-native/file

Integració de la Càmera

Captura d'Imatges

Per capturar imatges utilitzant la càmera del dispositiu, segueix aquests passos:

  1. Importa el Mòdul de la Càmera:

    import { Camera, CameraOptions } from '@ionic-native/camera/ngx';

  2. Injecció del Servei de la Càmera:

    constructor(private camera: Camera) { }

  3. Defineix les Opcions de la Càmera:

    const options: CameraOptions = {
  quality: 100,
  destinationType: this.camera.DestinationType.FILE_URI,
  encodingType: this.camera.EncodingType.JPEG,
  mediaType: this.camera.MediaType.PICTURE
};

  4. Captura d'Imatge:

    this.camera.getPicture(options).then((imageData) => {
  let capturedImage = imageData;
  // Fes alguna cosa amb la imatge capturada
}, (err) => {
  // Gestiona l'error
  console.log("Problema amb la càmera:" + err);
});

Accés a la Biblioteca de Fotos

Per accedir a les imatges de la biblioteca de fotos del dispositiu, modifica el sourceType a les opcions de la càmera:

const options: CameraOptions = {
  quality: 100,
  destinationType: this.camera.DestinationType.FILE_URI,
  sourceType: this.camera.PictureSourceType.PHOTOLIBRARY,
  encodingType: this.camera.EncodingType.JPEG,
  mediaType: this.camera.MediaType.PICTURE
};

Gestió de Fitxers

Desar Fitxers

Per desar fitxers dins de l'aplicació, segueix aquests passos:

  1. Importa el Mòdul de Fitxers:

    import { File } from '@ionic-native/file/ngx';

  2. Injecció del Servei de Fitxers:

    constructor(private file: File) { }

  3. Escriure en un Fitxer:

    this.file.writeFile(this.file.dataDirectory, 'myFile.txt', 'Hello World', { replace: true }).then(() => {
  console.log('Fitxer escrit correctament');
}).catch((err) => {
  console.log('Error en escriure el fitxer', err);
});

Llegir Fitxers

Per llegir fitxers, utilitza el següent codi:

this.file.readAsText(this.file.dataDirectory, 'myFile.txt').then((text) => {
  console.log('Contingut del fitxer:', text);
}).catch((err) => {
  console.log('Error en llegir el fitxer', err);
});

Esborrar Fitxers

Per esborrar fitxers, utilitza el següent codi:

this.file.removeFile(this.file.dataDirectory, 'myFile.txt').then(() => {
  console.log('Fitxer eliminat correctament');
}).catch((err) => {
  console.log('Error en eliminar el fitxer', err);
});

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'integrar la funcionalitat de la càmera i gestionar fitxers en una aplicació Ionic. Has après com capturar imatges, accedir a la biblioteca de fotos i realitzar operacions bàsiques de fitxers com desar, llegir i esborrar fitxers. Aquestes habilitats són essencials per construir aplicacions mòbils robustes que interactuen amb el maquinari del dispositiu i el sistema de fitxers. Continua practicant aquests conceptes per convertir-te en un expert en la gestió de mitjans i fitxers en els teus projectes Ionic.
