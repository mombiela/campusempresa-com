Introducció

En aquesta secció, aprofundirem en les animacions i transicions avançades en Ionic. Les animacions i transicions poden millorar molt l'experiència de l'usuari proporcionant retroalimentació visual i fent que l'aplicació sembli més dinàmica i responsiva. Cobrirem conceptes clau, exemples pràctics i exercicis per ajudar-te a dominar aquestes tècniques.

Conceptes Clau

  • Animacions i Transicions CSS: Entendre els fonaments de les animacions i transicions CSS.
  • API d'Animacions d'Ionic: Aprofitar l'API d'animacions integrada d'Ionic per a animacions més complexes.
  • Animacions Basades en Gestos: Crear animacions que responguin als gestos dels usuaris.
  • Optimització del Rendiment: Assegurar que les animacions funcionin de manera fluida en diversos dispositius.

Animacions i Transicions CSS

Les animacions i transicions CSS són la base de qualsevol animació basada en web. Són senzilles d'utilitzar i poden ser molt potents.

Transicions CSS

Les transicions permeten canviar els valors de les propietats de manera suau (durant una durada determinada).

/* Exemple d'una transició senzilla */
.button {
  background-color: blue;
  transition: background-color 0.5s ease;
}

.button:hover {
  background-color: green;
}

Animacions CSS

Les animacions permeten canviar els valors de les propietats al llarg del temps, utilitzant fotogrames clau.

/* Exemple d'una animació senzilla */
@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

.element {
  animation: slideIn 0.5s forwards;
}

API d'Animacions d'Ionic

Ionic proporciona una potent API d'animacions que permet animacions més complexes més enllà del CSS bàsic.

Crear una Animació Bàsica

A continuació, es mostra com crear una animació senzilla utilitzant l'API d'animacions d'Ionic:

import { createAnimation } from '@ionic/core';

const animation = createAnimation()
  .addElement(document.querySelector('.my-element'))
  .duration(1000)
  .iterations(1)
  .keyframes([
    { offset: 0, transform: 'scale(1)', opacity: '1' },
    { offset: 1, transform: 'scale(1.5)', opacity: '0.5' }
  ]);

animation.play();

Enllaçar Animacions

Pots enllaçar múltiples animacions per a seqüències més complexes.

const animation1 = createAnimation()
  .addElement(document.querySelector('.element1'))
  .duration(1000)
  .fromTo('opacity', '1', '0');

const animation2 = createAnimation()
  .addElement(document.querySelector('.element2'))
  .duration(1000)
  .fromTo('opacity', '0', '1');

const parentAnimation = createAnimation()
  .addAnimation([animation1, animation2]);

parentAnimation.play();

Animacions Basades en Gestos

Les animacions basades en gestos responen a interaccions dels usuaris com ara lliscaments, pessics i tocs.

Exemple: Lliscament per Esborrar

A continuació, es mostra un exemple d'una animació de lliscament per esborrar:

import { Gesture, GestureController } from '@ionic/angular';

const gesture = new Gesture({
  el: document.querySelector('.swipe-item'),
  gestureName: 'swipe',
  onMove: ev => {
    const opacity = 1 - Math.abs(ev.deltaX) / 100;
    ev.target.style.transform = `translateX(${ev.deltaX}px)`;
    ev.target.style.opacity = opacity.toString();
  },
  onEnd: ev => {
    if (Math.abs(ev.deltaX) > 100) {
      ev.target.style.transition = 'transform 0.5s, opacity 0.5s';
      ev.target.style.transform = `translateX(${ev.deltaX > 0 ? 100 : -100}%)`;
      ev.target.style.opacity = '0';
    } else {
      ev.target.style.transition = 'transform 0.5s, opacity 0.5s';
      ev.target.style.transform = 'translateX(0)';
      ev.target.style.opacity = '1';
    }
  }
});

gesture.enable();

Optimització del Rendiment

Les animacions poden ser intensives en recursos, per tant, és crucial optimitzar-les per al rendiment.

  • Utilitza l'Aceleració per Maquinari: Utilitza les propietats transform i opacity per aprofitar l'acceleració per GPU.
  • Limita els Repaints i Reflows: Evita propietats que desencadenin recalculacions de disseny.
  • Debounce Gestures: Throttle o debounce els esdeveniments de gestos per reduir el nombre de vegades que s'activen les animacions.
