Visió General

En aquesta secció, explorarem les peticions HTTP en el context de les aplicacions Ionic. Entendre com fer peticions HTTP és crucial per interactuar amb APIs, obtenir dades i enviar informació als servidors. Al final d'aquesta secció, seràs capaç de realitzar operacions bàsiques HTTP utilitzant Ionic.

Conceptes Clau

  • HTTP (HyperText Transfer Protocol): La base de qualsevol intercanvi de dades a la Web i un protocol utilitzat per transmetre peticions i informació d'hipertext.
  • API (Application Programming Interface): Un conjunt de regles que permeten que diferents entitats de programari es comuniquin entre elles.
  • Mètodes HTTP: Les accions que es poden realitzar en un recurs, com GET, POST, PUT, DELETE.
  • Endpoints: URLs on les APIs poden ser accedides pels clients.

Configurant Ionic per a Peticions HTTP

Abans de fer peticions HTTP, assegura't de tenir un projecte Ionic configurat. Si no, pots crear-ne un utilitzant la següent comanda:

ionic start myApp blank

A continuació, instal·la el mòdul necessari del client HTTP:

npm install @angular/common@latest

Realitzant Peticions HTTP en Ionic

Ionic utilitza el mòdul HttpClient d'Angular per gestionar les peticions HTTP. Anem a veure els passos per fer una petició GET per obtenir dades d'una API.

Important HttpClientModule

Primer, importa el HttpClientModule en el teu app.module.ts:

import { HttpClientModule } from '@angular/common/http';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, HttpClientModule],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Creant un Servei per a Peticions HTTP

Crea un servei per gestionar les peticions HTTP. Executa la següent comanda per generar un servei:

ionic generate service api

En el api.service.ts generat, importa HttpClient i injecta'l en el constructor:

import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class ApiService {

  private apiUrl = 'https://jsonplaceholder.typicode.com';

  constructor(private http: HttpClient) { }

  getPosts(): Observable<any> {
    return this.http.get(`${this.apiUrl}/posts`);
  }
}

Utilitzant el Servei en un Component

Ara, utilitzem aquest servei en un component per obtenir i mostrar dades. Obre home.page.ts i modifica'l de la següent manera:

import { Component, OnInit } from '@angular/core';
import { ApiService } from '../services/api.service';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage implements OnInit {
  posts: any[] = [];

  constructor(private apiService: ApiService) {}

  ngOnInit() {
    this.apiService.getPosts().subscribe(data => {
      this.posts = data;
    });
  }
}

Mostrant Dades en la Plantilla

Finalment, actualitza home.page.html per mostrar les dades obtingudes:

<ion-header>
  <ion-toolbar>
    <ion-title>
      Exemple de Peticions HTTP
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-list>
    <ion-item *ngFor="let post of posts">
      <ion-label>
        <h2>{{ post.title }}</h2>
        <p>{{ post.body }}</p>
      </ion-label>
    </ion-item>
  </ion-list>
</ion-content>

Resum

En aquesta secció, hem cobert els conceptes bàsics de fer peticions HTTP en una aplicació Ionic. Hem discutit conceptes clau com HTTP, API i mètodes HTTP. També hem vist com configurar un projecte Ionic, crear un servei per gestionar peticions HTTP i mostrar les dades obtingudes en un component. Entendre aquests fonaments et permetrà interactuar amb diverses APIs i construir aplicacions dinàmiques i basades en dades.
