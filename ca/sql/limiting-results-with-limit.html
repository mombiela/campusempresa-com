Introducció

En SQL, la clàusula LIMIT s'utilitza per especificar el nombre de registres a retornar d'una consulta. Això és particularment útil quan es treballa amb grans conjunts de dades on només es necessita un subconjunt de les dades per a l'anàlisi o per a la seva visualització.

Ús Bàsic de LIMIT

La clàusula LIMIT s'afegeix al final d'una sentència SELECT per restringir el nombre de files retornades.

Sintaxi

SELECT column1, column2, ...
FROM table_name
LIMIT number_of_rows;

Exemple

Considera una taula anomenada employees amb les següents dades:

| id | nom | departament | salari | |----|------------|-------------|--------| | 1 | John Doe | HR | 50000 | | 2 | Jane Smith | IT | 60000 | | 3 | Sam Brown | Finances | 55000 | | 4 | Lisa White | IT | 62000 | | 5 | Tom Clark | HR | 48000 |

Per recuperar només els primers 3 empleats, pots utilitzar la següent consulta:

SELECT * FROM employees
LIMIT 3;

Això retornarà:

| id | nom | departament | salari | |----|------------|-------------|--------| | 1 | John Doe | HR | 50000 | | 2 | Jane Smith | IT | 60000 | | 3 | Sam Brown | Finances | 55000 |

Ús de LIMIT amb OFFSET

La clàusula OFFSET s'utilitza conjuntament amb LIMIT per saltar un nombre específic de files abans de començar a retornar les files.

Sintaxi

SELECT column1, column2, ...
FROM table_name
LIMIT number_of_rows OFFSET offset_value;

Exemple

Per recuperar els següents 2 empleats després de saltar els primers 2, pots utilitzar:

SELECT * FROM employees
LIMIT 2 OFFSET 2;

Això retornarà:

| id | nom | departament | salari | |----|------------|-------------|--------| | 3 | Sam Brown | Finances | 55000 | | 4 | Lisa White | IT | 62000 |

Usos Pràctics

  • Paginació: Quan es mostren dades en una pàgina web, sovint cal mostrar un nombre limitat de registres per pàgina.
  • Optimització del Rendiment: Limitar el nombre de files retornades pot millorar significativament el rendiment de la consulta, especialment amb grans conjunts de dades.

Exemple: Paginació

Suposa que vols mostrar 2 empleats per pàgina. Per a la primera pàgina:

SELECT * FROM employees
LIMIT 2 OFFSET 0;

Per a la segona pàgina:

SELECT * FROM employees
LIMIT 2 OFFSET 2;

Per a la tercera pàgina:

SELECT * FROM employees
LIMIT 2 OFFSET 4;

Conclusió

La clàusula LIMIT és una eina poderosa en SQL per controlar el nombre de files retornades per una consulta. És especialment útil per a la paginació i l'optimització del rendiment. Combinant LIMIT amb OFFSET, pots gestionar i navegar fàcilment a través de grans conjunts de dades. Entendre com utilitzar efectivament aquestes clàusules millorarà la teva capacitat per escriure consultes SQL eficients i fàcils d'usar.
