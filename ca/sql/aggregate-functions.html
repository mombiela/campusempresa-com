Funcions agregades (COUNT, SUM, AVG, MAX, MIN)
Clàusula GROUP BY ►

Introducció

Les funcions agregades en SQL s'utilitzen per realitzar càlculs sobre múltiples files d'una columna d'una taula i retornar un únic valor. Són essencials per a l'anàlisi de dades i la generació d'informes. Les funcions agregades més utilitzades són COUNT, SUM, AVG, MAX i MIN.

COUNT

La funció COUNT retorna el nombre de files que coincideixen amb una condició especificada.

Sintaxi

SELECT COUNT(column_name)
FROM table_name
WHERE condition;

Exemple

SELECT COUNT(employee_id)
FROM employees
WHERE department = 'Sales';

Aquesta consulta compta el nombre d'empleats en el departament de 'Vendes'.

SUM

La funció SUM retorna la suma total d'una columna numèrica.

Sintaxi

SELECT SUM(column_name)
FROM table_name
WHERE condition;

Exemple

SELECT SUM(salary)
FROM employees
WHERE department = 'Sales';

Aquesta consulta calcula el salari total de tots els empleats en el departament de 'Vendes'.

AVG

La funció AVG retorna el valor mitjà d'una columna numèrica.

Sintaxi

SELECT AVG(column_name)
FROM table_name
WHERE condition;

Exemple

SELECT AVG(salary)
FROM employees
WHERE department = 'Sales';

Aquesta consulta calcula el salari mitjà dels empleats en el departament de 'Vendes'.

MAX

La funció MAX retorna el valor màxim en un conjunt de valors.

Sintaxi

SELECT MAX(column_name)
FROM table_name
WHERE condition;

Exemple

SELECT MAX(salary)
FROM employees
WHERE department = 'Sales';

Aquesta consulta troba el salari més alt en el departament de 'Vendes'.

MIN

La funció MIN retorna el valor mínim en un conjunt de valors.

Sintaxi

SELECT MIN(column_name)
FROM table_name
WHERE condition;

Exemple

SELECT MIN(salary)
FROM employees
WHERE department = 'Sales';

Aquesta consulta troba el salari més baix en el departament de 'Vendes'.

Combinació de funcions agregades

Pots utilitzar múltiples funcions agregades en una sola consulta per obtenir un resum complet de les teves dades.

Exemple

SELECT 
    COUNT(employee_id) AS total_employees,
    SUM(salary) AS total_salary,
    AVG(salary) AS average_salary,
    MAX(salary) AS highest_salary,
    MIN(salary) AS lowest_salary
FROM employees
WHERE department = 'Sales';

Aquesta consulta proporciona un resum del nombre d'empleats, salari total, salari mitjà, salari més alt i salari més baix en el departament de 'Vendes'.

Agrupació de dades amb funcions agregades

Per realitzar càlculs agregats sobre subconjunts de dades, pots utilitzar la clàusula GROUP BY.

Exemple

SELECT 
    department,
    COUNT(employee_id) AS total_employees,
    SUM(salary) AS total_salary,
    AVG(salary) AS average_salary,
    MAX(salary) AS highest_salary,
    MIN(salary) AS lowest_salary
FROM employees
GROUP BY department;

Aquesta consulta proporciona un resum de les estadístiques dels empleats per a cada departament.

Conclusió

Les funcions agregades són eines poderoses per resumir i analitzar dades en SQL. En dominar COUNT, SUM, AVG, MAX i MIN, pots realitzar una àmplia gamma de tasques d'anàlisi de dades. Recorda utilitzar la clàusula GROUP BY quan necessitis agregar dades en diferents categories. Practica aquestes funcions amb diversos conjunts de dades per esdevenir proficient en l'anàlisi de dades utilitzant SQL.
