La instrucció SELECT és una de les ordres més fonamentals i àmpliament utilitzades en SQL. Et permet recuperar dades d'una o més taules en una base de dades. Aquest tema cobrirà la instrucció SELECT des d'un nivell principiant fins a un nivell avançat, assegurant que entenguis la seva sintaxi, ús i diverses característiques avançades.

Introducció a SELECT

La instrucció SELECT s'utilitza per fer consultes a la base de dades i recuperar dades. La sintaxi bàsica d'una instrucció SELECT és:

SELECT column1, column2, ...
FROM table_name;

Ús Bàsic

  • Seleccionar Totes les Columnes: Per seleccionar totes les columnes d'una taula, utilitza l'asterisc (*).
SELECT * FROM employees;
  • Seleccionar Columnes Específiques: Per seleccionar columnes específiques, llista els noms de les columnes separats per comes.
SELECT first_name, last_name FROM employees;

Filtrar Dades amb WHERE

La clàusula WHERE s'utilitza per filtrar registres que compleixin certes condicions.

SELECT first_name, last_name
FROM employees
WHERE department = 'Sales';

Ordenar Dades amb ORDER BY

La clàusula ORDER BY s'utilitza per ordenar el conjunt de resultats en ordre ascendent o descendent.

SELECT first_name, last_name
FROM employees
ORDER BY last_name ASC;
  • Ordre Ascendent: ASC (per defecte)
  • Ordre Descendent: DESC

Limitar Resultats amb LIMIT

La clàusula LIMIT s'utilitza per especificar el nombre de registres a retornar.

SELECT first_name, last_name
FROM employees
ORDER BY last_name ASC
LIMIT 10;

Característiques Intermèdies de SELECT

Àlies

Els àlies s'utilitzen per donar un nom temporal a una taula o una columna.

  • Àlies de Columna:
SELECT first_name AS fname, last_name AS lname
FROM employees;
  • Àlies de Taula:
SELECT e.first_name, e.last_name
FROM employees AS e;

Funcions Agregades

Les funcions agregades realitzen un càlcul sobre un conjunt de valors i retornen un valor únic.

  • COUNT: Compta el nombre de files.
SELECT COUNT(*) FROM employees;
  • SUM: Calcula la suma d'una columna numèrica.
SELECT SUM(salary) FROM employees;
  • AVG: Calcula el valor mitjà d'una columna numèrica.
SELECT AVG(salary) FROM employees;
  • MAX i MIN: Troben els valors màxims i mínims.
SELECT MAX(salary) FROM employees;
SELECT MIN(salary) FROM employees;

GROUP BY

La instrucció GROUP BY agrupa files que tenen els mateixos valors en files de resum.

SELECT department, COUNT(*)
FROM employees
GROUP BY department;

HAVING

La clàusula HAVING s'utilitza per filtrar grups basats en una condició.

SELECT department, COUNT(*)
FROM employees
GROUP BY department
HAVING COUNT(*) > 5;

Característiques Avançades de SELECT

Subconsultes

Una subconsulta és una consulta dins d'una altra consulta.

  • Subconsulta en SELECT:
SELECT first_name, last_name, (SELECT department FROM departments WHERE departments.id = employees.department_id) AS department
FROM employees;
  • Subconsulta en WHERE:
SELECT first_name, last_name
FROM employees
WHERE department_id = (SELECT id FROM departments WHERE name = 'Sales');

Unions

Les unions s'utilitzen per combinar files de dues o més taules basades en una columna relacionada.

  • INNER JOIN: Retorna registres que tenen valors coincidents en ambdues taules.
SELECT employees.first_name, employees.last_name, departments.name
FROM employees
INNER JOIN departments ON employees.department_id = departments.id;
  • LEFT JOIN: Retorna tots els registres de la taula esquerra, i els registres coincidents de la taula dreta.
SELECT employees.first_name, employees.last_name, departments.name
FROM employees
LEFT JOIN departments ON employees.department_id = departments.id;
  • RIGHT JOIN: Retorna tots els registres de la taula dreta, i els registres coincidents de la taula esquerra.
SELECT employees.first_name, employees.last_name, departments.name
FROM employees
RIGHT JOIN departments ON employees.department_id = departments.id;
  • FULL OUTER JOIN: Retorna tots els registres quan hi ha una coincidència en qualsevol de les taules.
SELECT employees.first_name, employees.last_name, departments.name
FROM employees
FULL OUTER JOIN departments ON employees.department_id = departments.id;

UNION

L'operador UNION s'utilitza per combinar els conjunts de resultats de dues o més instruccions SELECT.

SELECT first_name, last_name FROM employees
UNION
SELECT first_name, last_name FROM managers;

Funcions de Finestra

Les funcions de finestra realitzen un càlcul sobre un conjunt de files de la taula que estan relacionades d'alguna manera amb la fila actual.

SELECT first_name, last_name, salary, 
       RANK() OVER (PARTITION BY department_id ORDER BY salary DESC) AS rank
FROM employees;

Conclusió

La instrucció SELECT és una eina poderosa en SQL que et permet recuperar i manipular dades de diverses maneres. Des de la recuperació bàsica de dades fins a consultes complexes que involucren unions, subconsultes i funcions de finestra, dominar la instrucció SELECT és essencial per a qualsevol programador SQL. Practica utilitzant aquestes característiques per convertir-te en un expert en la consulta de bases de dades de manera efectiva.
