Introducció a INNER JOIN

En SQL, la clàusula INNER JOIN s'utilitza per combinar files de dues o més taules basant-se en una columna relacionada entre elles. Només retorna les files que tenen valors coincidents en ambdues taules.

Conceptes Clau

  • Condició de Join: La condició que especifica com es combinen les files d'una taula amb les files d'una altra taula.
  • Conjunt de Resultats: El conjunt de files retornades per l'operació INNER JOIN.
  • Clau Primària: Un identificador únic per a una fila en una taula.
  • Clau Forana: Una columna o un conjunt de columnes en una taula que fa referència a la clau primària en una altra taula.

Sintaxi Bàsica

La sintaxi bàsica per a un INNER JOIN és la següent:

SELECT columns
FROM table1
INNER JOIN table2
ON table1.column = table2.column;

Exemple

Considerem dues taules, Employees i Departments:

Taula Employees

| EmployeeID | Nom | DepartmentID | |------------|----------|--------------| | 1 | Alice | 101 | | 2 | Bob | 102 | | 3 | Charlie | 103 | | 4 | David | 101 |

Taula Departments

| DepartmentID | Nom del Departament | |--------------|---------------------| | 101 | HR | | 102 | IT | | 103 | Finance | | 104 | Marketing |

Consulta INNER JOIN

Per obtenir una llista d'empleats juntament amb els noms dels seus departaments, pots utilitzar la següent consulta INNER JOIN:

SELECT Employees.Name, Departments.DepartmentName
FROM Employees
INNER JOIN Departments
ON Employees.DepartmentID = Departments.DepartmentID;

Conjunt de Resultats

| Nom | Nom del Departament | |---------|---------------------| | Alice | HR | | Bob | IT | | Charlie | Finance | | David | HR |

Ús Avançat

Unir Múltiples Taules

Pots unir més de dues taules utilitzant múltiples clàusules INNER JOIN. Per exemple:

SELECT Employees.Name, Departments.DepartmentName, Projects.ProjectName
FROM Employees
INNER JOIN Departments ON Employees.DepartmentID = Departments.DepartmentID
INNER JOIN EmployeeProjects ON Employees.EmployeeID = EmployeeProjects.EmployeeID
INNER JOIN Projects ON EmployeeProjects.ProjectID = Projects.ProjectID;

Utilitzant Àlies

Els àlies poden fer que les teves consultes SQL siguin més llegibles:

SELECT e.Name, d.DepartmentName
FROM Employees AS e
INNER JOIN Departments AS d
ON e.DepartmentID = d.DepartmentID;

Filtrant Resultats

Pots afegir una clàusula WHERE per filtrar els resultats:

SELECT e.Name, d.DepartmentName
FROM Employees AS e
INNER JOIN Departments AS d
ON e.DepartmentID = d.DepartmentID
WHERE d.DepartmentName = 'IT';

Exercicis

  1. INNER JOIN Bàsic: Escriu una consulta per unir les taules Students i Courses per llistar tots els estudiants juntament amb els cursos en què estan inscrits.
  2. Múltiples INNER JOINs: Escriu una consulta per unir les taules Orders, Customers i Products per llistar totes les comandes juntament amb els noms dels clients i els detalls dels productes.
  3. Filtrant amb INNER JOIN: Escriu una consulta per unir les taules Employees i Departments i filtrar els resultats per mostrar només els empleats del departament de 'Finance'.

Conclusió

La clàusula INNER JOIN és una eina poderosa en SQL per combinar files de múltiples taules basant-se en una columna relacionada. Entenent i utilitzant INNER JOIN, pots recuperar i analitzar dades de bases de dades relacionals de manera eficient. Practica amb diferents escenaris i conjunts de dades per dominar l'ús de INNER JOIN en les teves consultes SQL.
