Introducció

En SQL, la clàusula WHERE s'utilitza per filtrar registres que compleixen certs criteris. Aquesta clàusula és essencial per consultar dades específiques d'una base de dades, fent-la un concepte fonamental tant per a principiants com per a usuaris avançats.

Ús Bàsic de WHERE

La clàusula WHERE s'utilitza en una instrucció SELECT per especificar condicions que els registres han de complir per ser inclosos en el conjunt de resultats.

Sintaxi

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Exemple

Considera una taula anomenada employees amb les següents columnes: id, name, age, i department.

SELECT name, age
FROM employees
WHERE department = 'Sales';

Aquesta consulta recupera el name i l'age dels empleats que treballen al departament de Vendes.

Operadors Comuns en la Clàusula WHERE

La clàusula WHERE admet diversos operadors per definir condicions.

  • Operadors de Comparació: =, !=, >, <, >=, <=
  • Operadors Lògics: AND, OR, NOT
  • Operadors de Rang: BETWEEN
  • Operadors de Conjunt: IN
  • Coincidència de Patrons: LIKE
  • Comprovació de Null: IS NULL, IS NOT NULL

Exemples

Operadors de Comparació

SELECT name, age
FROM employees
WHERE age > 30;

Aquesta consulta recupera el name i l'age dels empleats que tenen més de 30 anys.

Operadors Lògics

SELECT name, department
FROM employees
WHERE department = 'Sales' AND age < 40;

Aquesta consulta recupera el name i el department dels empleats que treballen a Vendes i tenen menys de 40 anys.

Operadors de Rang

SELECT name, age
FROM employees
WHERE age BETWEEN 25 AND 35;

Aquesta consulta recupera el name i l'age dels empleats que tenen entre 25 i 35 anys.

Operadors de Conjunt

SELECT name, department
FROM employees
WHERE department IN ('Sales', 'Marketing');

Aquesta consulta recupera el name i el department dels empleats que treballen a Vendes o Màrqueting.

Coincidència de Patrons

SELECT name
FROM employees
WHERE name LIKE 'J%';

Aquesta consulta recupera el name dels empleats el nom dels quals comença amb 'J'.

Comprovació de Null

SELECT name, department
FROM employees
WHERE department IS NULL;

Aquesta consulta recupera el name i el department dels empleats que no tenen un departament assignat.

Tècniques Avançades de Filtratge

Combinació de Múltiples Condicions

Pots combinar múltiples condicions utilitzant operadors lògics per crear consultes complexes.

SELECT name, age, department
FROM employees
WHERE (department = 'Sales' OR department = 'Marketing') AND age > 30;

Aquesta consulta recupera el name, l'age, i el department dels empleats que treballen a Vendes o Màrqueting i tenen més de 30 anys.

Subconsultes en la Clàusula WHERE

Una subconsulta es pot utilitzar dins d'una clàusula WHERE per filtrar dades basant-se en el resultat d'una altra consulta.

SELECT name, age
FROM employees
WHERE department = (SELECT department FROM departments WHERE manager_id = 5);

Aquesta consulta recupera el name i l'age dels empleats que treballen al departament gestionat pel manager amb id 5.

Conclusió

La clàusula WHERE és una eina poderosa en SQL per filtrar dades basant-se en condicions específiques. Dominant l'ús de diversos operadors i combinant múltiples condicions, pots crear consultes precises i eficients per extreure les dades que necessites. Practica l'ús de la clàusula WHERE amb diferents operadors i condicions per convertir-te en un expert en el filtratge de dades en SQL.
