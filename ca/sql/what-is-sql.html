Introducció a SQL

SQL, o Structured Query Language, és un llenguatge de programació estàndard dissenyat específicament per gestionar i manipular bases de dades relacionals. S'utilitza per realitzar diverses operacions sobre les dades emmagatzemades en una base de dades, com ara consultes, actualitzacions, insercions i eliminacions de dades.

Conceptes clau de SQL

  • Base de dades: Una col·lecció estructurada de dades emmagatzemades electrònicament en un sistema informàtic.
  • Taula: Una col·lecció d'entrades de dades relacionades que consisteix en columnes i files.
  • Consulta: Una sol·licitud de dades o informació d'una taula de base de dades o una combinació de taules.
  • Esquema: L'estructura que defineix l'organització de les dades en una base de dades, incloent taules, columnes i relacions.
  • Clau primària: Un identificador únic per a un registre en una taula.
  • Clau forana: Un camp en una taula que identifica de manera única una fila d'una altra taula.

Comandes bàsiques de SQL

Les comandes SQL es categoritzen en diversos tipus, incloent Llenguatge de Consulta de Dades (DQL), Llenguatge de Definició de Dades (DDL), Llenguatge de Manipulació de Dades (DML) i Llenguatge de Control de Dades (DCL).

Llenguatge de Consulta de Dades (DQL)

  • SELECT: S'utilitza per recuperar dades d'una base de dades.
SELECT column1, column2
FROM table_name;

Exemple:

SELECT first_name, last_name
FROM employees;

Aquesta consulta recupera els noms i cognoms de tots els empleats de la taula employees.

Llenguatge de Definició de Dades (DDL)

  • CREATE: S'utilitza per crear una nova taula o base de dades.
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
);

Exemple:

CREATE TABLE employees (
    employee_id INT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    hire_date DATE
);

Aquesta comanda crea una nova taula anomenada employees amb les columnes especificades.

  • ALTER: S'utilitza per modificar un objecte existent de la base de dades, com ara una taula.
ALTER TABLE table_name
ADD column_name datatype;

Exemple:

ALTER TABLE employees
ADD email VARCHAR(100);

Aquesta comanda afegeix una nova columna email a la taula employees.

  • DROP: S'utilitza per eliminar una taula o base de dades.
DROP TABLE table_name;

Exemple:

DROP TABLE employees;

Aquesta comanda elimina la taula employees de la base de dades.

Llenguatge de Manipulació de Dades (DML)

  • INSERT: S'utilitza per inserir noves dades en una taula.
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

Exemple:

INSERT INTO employees (employee_id, first_name, last_name, hire_date)
VALUES (1, 'John', 'Doe', '2023-01-01');

Aquesta comanda insereix un nou registre d'empleat a la taula employees.

  • UPDATE: S'utilitza per actualitzar dades existents dins d'una taula.
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Exemple:

UPDATE employees
SET email = 'john.doe@example.com'
WHERE employee_id = 1;

Aquesta comanda actualitza l'adreça de correu electrònic de l'empleat amb employee_id 1.

  • DELETE: S'utilitza per eliminar dades d'una taula.
DELETE FROM table_name
WHERE condition;

Exemple:

DELETE FROM employees
WHERE employee_id = 1;

Aquesta comanda elimina el registre de l'empleat amb employee_id 1 de la taula employees.

Llenguatge de Control de Dades (DCL)

  • GRANT: S'utilitza per donar privilegis d'accés als usuaris a la base de dades.
GRANT SELECT, INSERT ON table_name TO 'username';

Exemple:

GRANT SELECT, INSERT ON employees TO 'john_doe';

Aquesta comanda concedeix privilegis de SELECT i INSERT a la taula employees a l'usuari john_doe.

  • REVOKE: S'utilitza per retirar els permisos concedits als usuaris.
REVOKE SELECT, INSERT ON table_name FROM 'username';

Exemple:

REVOKE SELECT, INSERT ON employees FROM 'john_doe';

Aquesta comanda revoca els privilegis de SELECT i INSERT a la taula employees de l'usuari john_doe.

Conclusió

SQL és una eina essencial per a qualsevol persona que treballi amb bases de dades relacionals. Permet als usuaris gestionar i manipular dades de manera eficient a través d'una varietat de comandes. Entendre els conceptes bàsics i les comandes de SQL és el primer pas per dominar la gestió de bases de dades i convertir-se en un expert en tasques de manipulació de dades. A mesura que avancis en aquest curs, aprendràs tècniques SQL més avançades i millors pràctiques per millorar les teves habilitats en la gestió de bases de dades.
