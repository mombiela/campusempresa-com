Introducció

El llenguatge de consulta estructurat (SQL) és el llenguatge estàndard per gestionar i manipular bases de dades relacionals. Entendre la sintaxi bàsica de SQL és crucial per realitzar operacions com consultar dades, actualitzar registres i gestionar estructures de bases de dades.

Conceptes clau

  • Sentències SQL: Comandes utilitzades per comunicar-se amb la base de dades.
  • Clàusules: Components de les sentències SQL que especifiquen condicions o accions.
  • Paraules clau: Paraules reservades amb un significat especial en SQL.
  • Identificadors: Noms donats a objectes de la base de dades com taules i columnes.
  • Operadors: Símbols utilitzats per especificar condicions en les sentències SQL.

Sentències SQL

Les sentències SQL són els blocs de construcció de SQL. Aquí hi ha algunes de les sentències SQL més comunes:

Sentència SELECT

La sentència SELECT s'utilitza per consultar dades d'una base de dades.

SELECT column1, column2
FROM table_name;
  • SELECT: Especifica les columnes a recuperar.
  • FROM: Especifica la taula a consultar.

Clàusula WHERE

La clàusula WHERE s'utilitza per filtrar registres.

SELECT column1, column2
FROM table_name
WHERE condition;
  • WHERE: Especifica la condició per filtrar registres.

Sentència INSERT INTO

La sentència INSERT INTO s'utilitza per afegir nous registres a una taula.

INSERT INTO table_name (column1, column2)
VALUES (value1, value2);
  • INSERT INTO: Especifica la taula i les columnes on inserir dades.
  • VALUES: Especifica els valors a inserir.

Sentència UPDATE

La sentència UPDATE s'utilitza per modificar registres existents.

UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;
  • UPDATE: Especifica la taula a actualitzar.
  • SET: Especifica les columnes i els nous valors.
  • WHERE: Especifica la condició per identificar els registres a actualitzar.

Sentència DELETE

La sentència DELETE s'utilitza per eliminar registres d'una taula.

DELETE FROM table_name
WHERE condition;
  • DELETE FROM: Especifica la taula d'on eliminar registres.
  • WHERE: Especifica la condició per identificar els registres a eliminar.

Exemples i exercicis

Exemple 1: Seleccionar dades

Recuperar els noms i edats de tots els empleats de la taula employees.

SELECT name, age
FROM employees;

Exercici 1: Seleccionar dades

Escriu una consulta SQL per recuperar el title i l'author de la taula books.

Exemple 2: Filtrar dades

Recuperar els noms dels empleats que tenen més de 30 anys.

SELECT name
FROM employees
WHERE age > 30;

Exercici 2: Filtrar dades

Escriu una consulta SQL per recuperar el title dels llibres publicats després de l'any 2000 de la taula books.

Exemple 3: Inserir dades

Afegir un nou empleat a la taula employees.

INSERT INTO employees (name, age, department)
VALUES ('John Doe', 28, 'Marketing');

Exercici 3: Inserir dades

Escriu una consulta SQL per afegir un nou llibre a la taula books amb les columnes title, author i year.

Exemple 4: Actualitzar dades

Actualitzar l'edat d'un empleat anomenat 'John Doe' a 29 anys.

UPDATE employees
SET age = 29
WHERE name = 'John Doe';

Exercici 4: Actualitzar dades

Escriu una consulta SQL per canviar l'year d'un llibre titulat 'SQL Basics' a 2021 en la taula books.

Exemple 5: Eliminar dades

Eliminar un empleat anomenat 'John Doe' de la taula employees.

DELETE FROM employees
WHERE name = 'John Doe';

Exercici 5: Eliminar dades

Escriu una consulta SQL per eliminar un llibre titulat 'Old Book' de la taula books.

Conclusió

Entendre la sintaxi bàsica de SQL és essencial per interactuar amb bases de dades relacionals. En dominar les sentències SELECT, INSERT INTO, UPDATE i DELETE, pots realitzar una àmplia gamma de tasques de manipulació de dades. Practica aquestes sentències i les seves clàusules per construir una base sòlida en SQL.
