Construir una Aplicació CRUD Simple

Introducció

En aquesta secció, aprendrem a construir una aplicació CRUD (Crear, Llegir, Actualitzar, Esborrar) simple utilitzant SQL. Les operacions CRUD són fonamentals per interactuar amb bases de dades i són essencials per a qualsevol aplicació basada en bases de dades.

Configuració de l'Entorn

Abans de començar, assegura't de tenir el següent:

  • Un sistema de gestió de bases de dades SQL (DBMS) com MySQL, PostgreSQL o SQLite instal·lat.
  • Una eina de client de base de dades com MySQL Workbench, pgAdmin o DBeaver.

Crear la Base de Dades i les Taules

Primer, necessitem crear una base de dades i les taules necessàries.

Crear la Base de Dades

CREATE DATABASE SimpleCRUD;
USE SimpleCRUD;

Explicació:

  • CREATE DATABASE SimpleCRUD; crea una nova base de dades anomenada SimpleCRUD.
  • USE SimpleCRUD; canvia a la base de dades SimpleCRUD.

Crear la Taula

CREATE TABLE Users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Explicació:

  • id INT AUTO_INCREMENT PRIMARY KEY crea una clau primària autoincremental.
  • username VARCHAR(50) NOT NULL defineix una columna de nom d'usuari que no pot ser nul·la.
  • email VARCHAR(100) NOT NULL defineix una columna de correu electrònic que no pot ser nul·la.
  • created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP estableix una marca de temps per defecte per quan es crea el registre.

Operacions CRUD

Crear

Per afegir nous registres a la taula Users, utilitzem la instrucció INSERT.

INSERT INTO Users (username, email) VALUES ('john_doe', 'john@example.com');

Explicació:

  • INSERT INTO Users (username, email) especifica la taula i les columnes on inserir dades.
  • VALUES ('john_doe', 'john@example.com') proporciona els valors per a les columnes especificades.

Llegir

Per recuperar dades de la taula Users, utilitzem la instrucció SELECT.

SELECT * FROM Users;

Explicació:

  • SELECT * FROM Users; recupera totes les columnes de la taula Users.

Actualitzar

Per modificar registres existents, utilitzem la instrucció UPDATE.

UPDATE Users SET email = 'john_new@example.com' WHERE username = 'john_doe';

Explicació:

  • UPDATE Users SET email = 'john_new@example.com' canvia el correu electrònic de l'usuari.
  • WHERE username = 'john_doe'; especifica quin registre actualitzar.

Esborrar

Per eliminar registres de la taula Users, utilitzem la instrucció DELETE.

DELETE FROM Users WHERE username = 'john_doe';

Explicació:

  • DELETE FROM Users especifica la taula d'on esborrar.
  • WHERE username = 'john_doe'; especifica quin registre esborrar.

Operacions CRUD Avançades

Utilitzar Transaccions

Les transaccions asseguren que una sèrie d'operacions SQL s'executin com una única unitat de treball.

START TRANSACTION;

INSERT INTO Users (username, email) VALUES ('jane_doe', 'jane@example.com');
UPDATE Users SET email = 'jane_new@example.com' WHERE username = 'jane_doe';

COMMIT;

Explicació:

  • START TRANSACTION; inicia una nova transacció.
  • COMMIT; guarda tots els canvis fets durant la transacció.

Gestionar Errors amb Transaccions

Per gestionar errors, podem utilitzar ROLLBACK per desfer els canvis si alguna cosa va malament.

START TRANSACTION;

INSERT INTO Users (username, email) VALUES ('jane_doe', 'jane@example.com');

-- Simular un error
UPDATE Users SET email = 'invalid_email' WHERE username = 'jane_doe';

ROLLBACK;

Explicació:

  • ROLLBACK; desfà tots els canvis fets durant la transacció si es produeix un error.

Conclusió

En aquest tema, hem cobert els conceptes bàsics per construir una aplicació CRUD simple utilitzant SQL. Hem après a:

  • Crear una base de dades i una taula.
  • Realitzar operacions CRUD bàsiques: Crear, Llegir, Actualitzar i Esborrar.
  • Utilitzar transaccions per assegurar la integritat de les dades.

En dominar aquestes operacions fonamentals, pots gestionar i manipular dades de manera efectiva en qualsevol base de dades basada en SQL.
