Introducció

En aquesta secció, explorarem els conceptes fonamentals de les bases de dades i les taules en SQL. Comprendre aquests conceptes és crucial per a qualsevol persona que vulgui treballar amb SQL, ja que formen l'esquelet de l'emmagatzematge i la manipulació de dades.

Què és una Base de Dades?

Una base de dades és una col·lecció organitzada de dades, generalment emmagatzemada i accedida electrònicament des d'un sistema informàtic. Les bases de dades s'utilitzen per emmagatzemar, recuperar i gestionar dades de manera eficient.

Conceptes Clau:

  • Sistema de Gestió de Bases de Dades (DBMS): Programari que interactua amb l'usuari, les aplicacions i la pròpia base de dades per capturar i analitzar dades.
  • SQL (Structured Query Language): Un llenguatge de programació estàndard dissenyat específicament per gestionar i manipular bases de dades.

Què és una Taula?

Una taula és una col·lecció de dades relacionades mantingudes en un format estructurat dins d'una base de dades. Consisteix en files i columnes.

Conceptes Clau:

  • Files (Registres): Cada fila en una taula representa un únic element de dades estructurat implícitament.
  • Columnes (Camps): Cada columna representa un atribut específic de les dades.

Exemple d'Estructura de Taula:

| ID | Nom | Edat | Correu Electrònic | |-----|-----------|------|----------------------| | 1 | John Doe | 30 | john.doe@example.com | | 2 | Jane Smith| 25 | jane.smith@example.com|

Crear una Base de Dades

Per crear una base de dades, utilitzes la instrucció CREATE DATABASE.

Exemple:

CREATE DATABASE my_database;

Aquesta comanda crea una nova base de dades anomenada my_database.

Crear una Taula

Per crear una taula, utilitzes la instrucció CREATE TABLE.

Exemple:

CREATE TABLE users (
    ID INT PRIMARY KEY,
    Name VARCHAR(100),
    Age INT,
    Email VARCHAR(100)
);

Aquesta comanda crea una taula anomenada users amb quatre columnes: ID, Name, Age i Email.

Inserir Dades en una Taula

Per inserir dades en una taula, utilitzes la instrucció INSERT INTO.

Exemple:

INSERT INTO users (ID, Name, Age, Email) VALUES (1, 'John Doe', 30, 'john.doe@example.com');
INSERT INTO users (ID, Name, Age, Email) VALUES (2, 'Jane Smith', 25, 'jane.smith@example.com');

Aquestes comandes insereixen dos registres a la taula users.

Consultar Dades d'una Taula

Per recuperar dades d'una taula, utilitzes la instrucció SELECT.

Exemple:

SELECT * FROM users;

Aquesta comanda recupera tots els registres de la taula users.

Filtrar Dades:

Pots filtrar les dades utilitzant la clàusula WHERE.

Exemple:

SELECT * FROM users WHERE Age > 25;

Aquesta comanda recupera registres de la taula users on l'Edat és superior a 25.

Actualitzar Dades en una Taula

Per actualitzar dades existents en una taula, utilitzes la instrucció UPDATE.

Exemple:

UPDATE users SET Email = 'john.newemail@example.com' WHERE ID = 1;

Aquesta comanda actualitza el correu electrònic de l'usuari amb ID 1.

Esborrar Dades d'una Taula

Per esborrar dades d'una taula, utilitzes la instrucció DELETE.

Exemple:

DELETE FROM users WHERE ID = 2;

Aquesta comanda esborra el registre de l'usuari amb ID 2.

Conclusió

Comprendre les bases de dades i les taules és fonamental per treballar amb SQL. Les bases de dades emmagatzemen col·leccions de dades, i les taules organitzen aquestes dades en files i columnes. Dominar les operacions bàsiques com crear bases de dades i taules, inserir, consultar, actualitzar i esborrar dades proporcionarà una base sòlida per a temes més avançats de SQL.
