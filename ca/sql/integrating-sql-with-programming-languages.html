Introducció

Integrar SQL amb llenguatges de programació és una habilitat crucial per als desenvolupadors, ja que els permet interactuar amb bases de dades directament des del seu codi. Aquesta secció cobrirà com connectar-se a bases de dades, executar consultes SQL i gestionar els resultats utilitzant llenguatges de programació populars com Python, Java i JavaScript.

Connectar-se a Bases de Dades

Python

Python ofereix diverses biblioteques per a la integració de bases de dades, amb sqlite3 i SQLAlchemy sent entre les més populars.

Utilitzant sqlite3

import sqlite3

# Connectar-se a la base de dades
conn = sqlite3.connect('example.db')

# Crear un objecte cursor
cursor = conn.cursor()

# Executar una consulta simple
cursor.execute("SELECT * FROM users")

# Obtenir tots els resultats
results = cursor.fetchall()

# Tancar la connexió
conn.close()

# Imprimir resultats
for row in results:
    print(row)
  • Explicació:
    • sqlite3.connect('example.db'): Connecta a la base de dades SQLite.
    • cursor.execute("SELECT * FROM users"): Executa una consulta SQL.
    • cursor.fetchall(): Recupera totes les files de la consulta executada.
    • conn.close(): Tanca la connexió a la base de dades.

Java

Java utilitza JDBC (Java Database Connectivity) per a les interaccions amb bases de dades.

Utilitzant JDBC

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.Statement;

public class DatabaseExample {
    public static void main(String[] args) {
        try {
            // Connectar-se a la base de dades
            Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/exampledb", "user", "password");

            // Crear un objecte statement
            Statement stmt = conn.createStatement();

            // Executar una consulta
            ResultSet rs = stmt.executeQuery("SELECT * FROM users");

            // Processar els resultats
            while (rs.next()) {
                System.out.println(rs.getString("username"));
            }

            // Tancar la connexió
            conn.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
  • Explicació:
    • DriverManager.getConnection(...): Estableix una connexió amb la base de dades MySQL.
    • stmt.executeQuery("SELECT * FROM users"): Executa una consulta SQL.
    • rs.next(): Itera a través del conjunt de resultats.
    • conn.close(): Tanca la connexió a la base de dades.

JavaScript

JavaScript, particularment en Node.js, utilitza biblioteques com mysql i pg per a les interaccions amb bases de dades.

Utilitzant mysql en Node.js

const mysql = require('mysql');

// Crear una connexió
const connection = mysql.createConnection({
  host: 'localhost',
  user: 'user',
  password: 'password',
  database: 'exampledb'
});

// Connectar-se a la base de dades
connection.connect();

// Executar una consulta
connection.query('SELECT * FROM users', (error, results, fields) => {
  if (error) throw error;
  
  // Processar els resultats
  results.forEach(row => {
    console.log(row.username);
  });
});

// Tancar la connexió
connection.end();
  • Explicació:
    • mysql.createConnection(...): Crea una connexió amb la base de dades MySQL.
    • connection.query('SELECT * FROM users', ...): Executa una consulta SQL.
    • results.forEach(...): Itera a través del conjunt de resultats.
    • connection.end(): Tanca la connexió a la base de dades.

Executar Consultes SQL

Python

Utilitzant sqlite3 per executar diferents tipus de consultes:

# Inserir un nou usuari
cursor.execute("INSERT INTO users (username, email) VALUES (?, ?)", ('john_doe', 'john@example.com'))

# Actualitzar el correu electrònic d'un usuari
cursor.execute("UPDATE users SET email = ? WHERE username = ?", ('new_email@example.com', 'john_doe'))

# Eliminar un usuari
cursor.execute("DELETE FROM users WHERE username = ?", ('john_doe',))

# Confirmar els canvis
conn.commit()
  • Explicació:
    • cursor.execute(...): Executa comandes SQL amb espais reservats per als paràmetres.
    • conn.commit(): Confirma la transacció a la base de dades.

Java

Utilitzant JDBC per executar diferents tipus de consultes:

// Inserir un nou usuari
stmt.executeUpdate("INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com')");

// Actualitzar el correu electrònic d'un usuari
stmt.executeUpdate("UPDATE users SET email = 'new_email@example.com' WHERE username = 'john_doe'");

// Eliminar un usuari
stmt.executeUpdate("DELETE FROM users WHERE username = 'john_doe'");
  • Explicació:
    • stmt.executeUpdate(...): Executa comandes SQL per a operacions d'inserció, actualització i eliminació.

JavaScript

Utilitzant mysql en Node.js per executar diferents tipus de consultes:

// Inserir un nou usuari
connection.query('INSERT INTO users (username, email) VALUES (?, ?)', ['john_doe', 'john@example.com'], (error, results, fields) => {
  if (error) throw error;
});

// Actualitzar el correu electrònic d'un usuari
connection.query('UPDATE users SET email = ? WHERE username = ?', ['new_email@example.com', 'john_doe'], (error, results, fields) => {
  if (error) throw error;
});

// Eliminar un usuari
connection.query('DELETE FROM users WHERE username = ?', ['john_doe'], (error, results, fields) => {
  if (error) throw error;
});
  • Explicació:
    • connection.query(...): Executa comandes SQL amb espais reservats per als paràmetres.

Gestionar Resultats

Python

Processar resultats de consultes en Python:

# Obtenir un resultat
result = cursor.fetchone()
print(result)

# Obtenir múltiples resultats
results = cursor.fetchmany(5)
for row in results:
    print(row)
  • Explicació:
    • cursor.fetchone(): Obté una sola fila.
    • cursor.fetchmany(5): Obté un nombre especificat de files.

Java

Processar resultats de consultes en Java:

// Obtenir i processar resultats
ResultSet rs = stmt.executeQuery("SELECT * FROM users");
while (rs.next()) {
    System.out.println("Username: " + rs.getString("username"));
    System.out.println("Email: " + rs.getString("email"));
}
  • Explicació:
    • rs.getString("column_name"): Recupera el valor d'una columna especificada.

JavaScript

Processar resultats de consultes en JavaScript:

connection.query('SELECT * FROM users', (error, results, fields) => {
  if (error) throw error;
  
  results.forEach(row => {
    console.log(`Username: ${row.username}`);
    console.log(`Email: ${row.email}`);
  });
});
  • Explicació:
    • row.column_name: Accedeix al valor d'una columna especificada.

Conclusió

Integrar SQL amb llenguatges de programació és essencial per construir aplicacions robustes que interactuen amb bases de dades. Entenent com connectar-se a bases de dades, executar consultes i gestionar resultats en Python, Java i JavaScript, els desenvolupadors poden crear aplicacions potents i eficients basades en dades. Practiqueu aquests conceptes amb projectes del món real per consolidar la vostra comprensió i millorar les vostres habilitats de programació.
