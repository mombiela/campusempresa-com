Vectors a Rust
Strings ►

Els vectors són una de les estructures de dades més utilitzades a Rust. Proporcionen una matriu dinàmica que pot créixer o disminuir de mida. Aquest tema cobrirà tot, des dels conceptes bàsics de la creació i ús de vectors fins a operacions més avançades i consideracions de rendiment.

Introducció als Vectors

Els vectors a Rust són proporcionats pel tipus Vec<T>, on T representa el tipus d'elements emmagatzemats en el vector.

Creació d'un Vector

  • Vector Buit: Pots crear un vector buit utilitzant Vec::new().
  • Vector amb Valors Inicials: Pots crear un vector amb valors inicials utilitzant la macro vec!.
// Creació d'un vector buit
let mut v: Vec<i32> = Vec::new();

// Creació d'un vector amb valors inicials
let v = vec![1, 2, 3, 4, 5];

Afegir Elements a un Vector

  • Utilitzant el Mètode push: Pots afegir elements al final del vector utilitzant el mètode push.
let mut v = Vec::new();
v.push(1);
v.push(2);
v.push(3);

Accedir als Elements d'un Vector

  • Indexació: Pots accedir als elements utilitzant la indexació, però vés amb compte ja que pot provocar un pànic si l'índex està fora de límits.
  • Utilitzant el Mètode get: El mètode get retorna un Option<&T> que és més segur.
let v = vec![1, 2, 3, 4, 5];

// Accedir als elements utilitzant la indexació
let third = &v[2];
println!("El tercer element és {}", third);

// Accedir als elements utilitzant el mètode `get`
match v.get(2) {
    Some(third) => println!("El tercer element és {}", third),
    None => println!("No hi ha tercer element."),
}

Iterar sobre un Vector

  • Utilitzant un Bucle for: Pots iterar sobre els elements d'un vector utilitzant un bucle for.
  • Utilitzant Iteradors: Rust proporciona mètodes d'iterador potents per a operacions més complexes.
let v = vec![1, 2, 3, 4, 5];

// Iterar utilitzant un bucle `for`
for i in &v {
    println!("{}", i);
}

// Iterar utilitzant un iterador
for i in v.iter() {
    println!("{}", i);
}

Operacions Avançades amb Vectors

Eliminar Elements

  • Utilitzant el Mètode pop: Elimina l'últim element i el retorna.
  • Utilitzant el Mètode remove: Elimina l'element en un índex especificat.
let mut v = vec![1, 2, 3, 4, 5];

// Eliminar l'últim element
let last = v.pop();
println!("S'ha eliminat l'últim element: {:?}", last);

// Eliminar un element a l'índex 2
let third = v.remove(2);
println!("S'ha eliminat el tercer element: {}", third);

Tallar Vectors

  • Slices: Pots crear slices de vectors per treballar amb un subconjunt d'elements.
let v = vec![1, 2, 3, 4, 5];
let slice = &v[1..4]; // Això crea un slice que conté els elements 2, 3 i 4

for i in slice {
    println!("{}", i);
}

Capacitat del Vector i Rendiment

  • Capacitat: Els vectors tenen una capacitat, que és la quantitat d'espai assignat per als elements. La capacitat pot ser diferent de la longitud.
  • Consideracions de Rendiment: Entendre com creixen els vectors pot ajudar-te a escriure codi més eficient.
let mut v = Vec::with_capacity(10);
println!("Capacitat: {}", v.capacity());

for i in 0..10 {
    v.push(i);
}
println!("Capacitat després d'afegir 10 elements: {}", v.capacity());

v.push(11);
println!("Capacitat després d'afegir un element més: {}", v.capacity());

Conclusió

Els vectors són una estructura de dades versàtil i potent a Rust. Permeten redimensionament dinàmic i proporcionen una sèrie de mètodes per manipular dades de manera eficient. Entenent com crear, accedir i modificar vectors, així com com manejar operacions més avançades, pots aprofitar els vectors per escriure programes en Rust més efectius. Recorda sempre considerar les implicacions de rendiment, especialment quan treballis amb grans conjunts de dades o aplicacions crítiques pel rendiment.
