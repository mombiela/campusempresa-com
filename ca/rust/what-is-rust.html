Introducció a Rust

Rust és un llenguatge de programació de sistemes que se centra en la velocitat, la seguretat de la memòria i el paral·lelisme. Va ser creat per Mozilla Research i va aparèixer per primera vegada el 2010. Rust està dissenyat per ser un llenguatge segur, concurrent i pràctic, cosa que el converteix en una excel·lent opció per a aplicacions crítiques en termes de rendiment.

Característiques clau de Rust

  • Seguretat de la memòria: Rust garanteix la seguretat de la memòria sense necessitat d'un col·lector d'escombraries.
  • Concurrència: Rust proporciona potents primitives de concurrència.
  • Rendiment: El rendiment de Rust és comparable al de C i C++.
  • Sistema de propietat: El sistema de propietat únic de Rust ajuda a gestionar la memòria i evitar carreres de dades.
  • Comunitat i ecosistema: Rust té una comunitat creixent i un ric ecosistema de biblioteques i eines.

El sistema de propietat de Rust

Una de les característiques més distintives de Rust és el seu sistema de propietat, que garanteix la seguretat de la memòria i evita les carreres de dades. El sistema de propietat es basa en tres conceptes principals:

  • Propietat: Cada valor a Rust té un únic propietari.
  • Préstec: Es poden prestar referències a un valor, ja sigui mutablement o immutablement.
  • Durades de vida: Les durades de vida asseguren que les referències siguin vàlides mentre s'utilitzin.

Exemple: Propietat i préstec

fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 es mou a s2, s1 ja no és vàlid

    // println!("{}", s1); // Això causaria un error en temps de compilació

    let s3 = String::from("world");
    let s4 = &s3; // s4 pren prestat s3 de manera immutable

    println!("{}", s3); // Això és vàlid
    println!("{}", s4); // Això també és vàlid
}

En aquest exemple:

  • s1 es mou a s2, fent que s1 sigui invàlid.
  • s3 es pren prestat de manera immutable per s4, permetent que tant s3 com s4 es puguin utilitzar.

El model de concurrència de Rust

El model de concurrència de Rust es basa en el concepte de propietat i préstec, que ajuda a prevenir carreres de dades en temps de compilació. Rust proporciona diverses primitives de concurrència:

  • Fils: La biblioteca estàndard de Rust proporciona suport per a fils.
  • Canals: Utilitzats per a la transmissió de missatges entre fils.
  • Mutexes i RwLocks: Per a la concurrència d'estat compartit.

Exemple: Utilitzant fils

use std::thread;

fn main() {
    let handle = thread::spawn(|| {
        for i in 1..10 {
            println!("hola número {} des del fil creat!", i);
        }
    });

    for i in 1..5 {
        println!("hola número {} des del fil principal!", i);
    }

    handle.join().unwrap();
}

En aquest exemple:

  • Es crea un nou fil per imprimir números.
  • El fil principal també imprimeix números.
  • handle.join().unwrap() assegura que el fil principal esperi que el fil creat acabi.

Rendiment a Rust

El rendiment de Rust és comparable al de C i C++ gràcies al seu control de baix nivell sobre els recursos del sistema i les abstraccions de cost zero. Rust aconsegueix això sense sacrificar la seguretat, cosa que el converteix en una opció ideal per a aplicacions crítiques en termes de rendiment.

Exemple: Rendiment amb abstraccions de cost zero

fn main() {
    let numbers: Vec<i32> = (1..100).collect();
    let sum: i32 = numbers.iter().sum();
    println!("Sum: {}", sum);
}

En aquest exemple:

  • Es crea un vector de números.
  • La suma dels números es calcula utilitzant un iterador, que és una abstracció de cost zero a Rust.

Comunitat i ecosistema

Rust té una comunitat vibrant i creixent. L'ecosistema de Rust inclou:

  • Cargo: El gestor de paquets i sistema de construcció de Rust.
  • Crates.io: El repositori central per a les biblioteques de Rust.
  • Rustfmt: Una eina per formatar codi Rust.
  • Clippy: Un linter per a codi Rust.

Conclusió

Rust és un potent llenguatge de programació de sistemes que ofereix seguretat de la memòria, concurrència i rendiment sense sacrificar la facilitat d'ús. El seu sistema de propietat únic, el robust model de concurrència i l'ecosistema creixent el converteixen en una excel·lent opció per a una àmplia gamma d'aplicacions, des del desenvolupament web fins als sistemes encastats. A mesura que avancis en aquest curs, obtindràs una comprensió més profunda de les característiques de Rust i com aprofitar-les per construir programari eficient i segur.
