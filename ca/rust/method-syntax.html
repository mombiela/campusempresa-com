Introducció a la Sintaxi dels Mètodes

A Rust, els mètodes són similars a les funcions però es defineixen en el context d'una struct (o un enum o trait). Els mètodes s'utilitzen per operar sobre les dades contingudes dins de la struct. Es defineixen utilitzant la paraula clau impl i tenen un primer paràmetre especial, self, que representa la instància de la struct.

  • Mètodes vs Funcions:
    • Els mètodes es defineixen en el context d'una struct, enum o trait.
    • Els mètodes tenen accés a la instància de la struct sobre la qual es criden.
    • Les funcions es defineixen fora de les structs i no tenen accés a una instància a menys que es passi explícitament.

Definició de Mètodes

Per definir un mètode, utilitzes el bloc impl per a la struct. Dins d'aquest bloc, defineixes els mètodes utilitzant la paraula clau fn.

Exemple: Definició d'un Mètode Simple

struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    // Mètode per calcular l'àrea del rectangle
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

fn main() {
    let rect = Rectangle { width: 30, height: 50 };
    println!("L'àrea del rectangle és {} píxels quadrats.", rect.area());
}
  • Explicació:
    • Rectangle és una struct amb camps width i height.
    • El bloc impl Rectangle conté mètodes per a la struct Rectangle.
    • El mètode area pren self prestat de manera immutable (&self) i calcula l'àrea del rectangle.
    • A main, creem una instància de Rectangle i cridem el mètode area sobre ella.

Paràmetres del Mètode

Els mètodes poden prendre paràmetres addicionals a part de self.

Exemple: Mètode amb Paràmetres Addicionals

impl Rectangle {
    // Mètode per comprovar si el rectangle pot contenir un altre rectangle
    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}

fn main() {
    let rect1 = Rectangle { width: 30, height: 50 };
    let rect2 = Rectangle { width: 10, height: 40 };
    let rect3 = Rectangle { width: 40, height: 60 };

    println!("Pot rect1 contenir rect2? {}", rect1.can_hold(&rect2));
    println!("Pot rect1 contenir rect3? {}", rect1.can_hold(&rect3));
}
  • Explicació:
    • El mètode can_hold pren un paràmetre addicional other que és una referència a un altre Rectangle.
    • Comprova si self pot contenir completament other.

Funcions Associades

Les funcions associades són funcions que estan associades amb una struct però no prenen self com a paràmetre. Sovint s'utilitzen per a constructors.

Exemple: Funció Associada

impl Rectangle {
    // Funció associada per crear un quadrat
    fn square(size: u32) -> Rectangle {
        Rectangle { width: size, height: size }
    }
}

fn main() {
    let square = Rectangle::square(20);
    println!("El quadrat té una amplada de {} i una alçada de {}.", square.width, square.height);
}
  • Explicació:
    • La funció square és una funció associada perquè no pren self.
    • Es crida utilitzant la sintaxi :: i retorna un Rectangle on l'amplada i l'alçada són iguals.

Enllaç de Mètodes

Els mètodes es poden enllaçar si retornen self o una referència a self.

Exemple: Enllaç de Mètodes

struct Circle {
    radius: f64,
}

impl Circle {
    // Mètode per establir el radi
    fn set_radius(&mut self, radius: f64) -> &mut Self {
        self.radius = radius;
        self
    }

    // Mètode per calcular l'àrea
    fn area(&self) -> f64 {
        std::f64::consts::PI * self.radius * self.radius
    }
}

fn main() {
    let mut circle = Circle { radius: 1.0 };
    circle.set_radius(5.0).set_radius(10.0);
    println!("L'àrea del cercle és {} unitats quadrades.", circle.area());
}
  • Explicació:
    • El mètode set_radius pren una referència mutable a self i retorna una referència mutable a self.
    • Això permet l'enllaç de mètodes, on es poden enllaçar múltiples crides de mètodes.

Conclusió

La sintaxi dels mètodes a Rust et permet definir comportaments per a les teves structs, fent el teu codi més organitzat i modular. Utilitzant mètodes, pots encapsular funcionalitat i proporcionar una interfície clara per interactuar amb les teves estructures de dades. Recorda:

  • Els mètodes es defineixen dins de blocs impl i tenen accés a la instància de la struct.
  • Els mètodes poden prendre paràmetres addicionals i es poden enllaçar.
  • Les funcions associades són útils per crear constructors i altres funcions d'utilitat.

Entendre i utilitzar la sintaxi dels mètodes és crucial per escriure codi idiomàtic i eficient a Rust.
