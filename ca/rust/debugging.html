Depuració en Rust
Construir un Projecte ►

La depuració és una habilitat essencial per a qualsevol programador, i Rust proporciona diverses eines i tècniques per ajudar-te a identificar i solucionar problemes en el teu codi. Aquest tema et guiarà a través dels conceptes bàsics de la depuració en Rust, des de nivells per a principiants fins a nivells avançats.

Introducció a la Depuració

La depuració és el procés d'identificar, analitzar i eliminar errors o bugs del teu codi. En Rust, la depuració es pot realitzar utilitzant diverses eines i tècniques, incloent:

  • Depuració amb print
  • Utilitzar els missatges d'error del compilador de Rust
  • Depuració amb un Entorn de Desenvolupament Integrat (IDE)
  • Utilitzar la macro dbg!
  • Depuració avançada amb GDB i LLDB

Depuració amb Print

La depuració amb print és una de les tècniques de depuració més simples i comunes. Consisteix a inserir instruccions de print en el teu codi per mostrar els valors de les variables i el flux d'execució.

Exemple

fn main() {
    let x = 10;
    let y = 20;
    let result = add(x, y);
    println!("Resultat: {}", result);
}

fn add(a: i32, b: i32) -> i32 {
    println!("Sumant {} i {}", a, b);
    a + b
}

En aquest exemple, les instruccions println! s'utilitzen per imprimir els valors de les variables i el resultat de la funció add.

Utilitzar els Missatges d'Error del Compilador de Rust

El compilador de Rust proporciona missatges d'error detallats que poden ajudar-te a identificar i solucionar problemes en el teu codi. Presta molta atenció a aquests missatges, ja que sovint inclouen suggeriments per resoldre els errors.

Exemple

fn main() {
    let x = 10;
    let y = "20"; // Això causarà un error de desajust de tipus
    let result = add(x, y);
    println!("Resultat: {}", result);
}

fn add(a: i32, b: i32) -> i32 {
    a + b
}

Quan compiles aquest codi, el compilador de Rust produirà un missatge d'error indicant un desajust de tipus entre i32 i &str.

Depuració amb un Entorn de Desenvolupament Integrat (IDE)

Molts IDEs, com Visual Studio Code i IntelliJ IDEA, proporcionen eines de depuració integrades que et permeten establir punts de ruptura, avançar pas a pas pel codi i inspeccionar variables.

Exemple

  1. Obre el teu projecte de Rust en Visual Studio Code.
  2. Instal·la l'extensió de Rust si no ho has fet ja.
  3. Estableix punts de ruptura fent clic a la vora al costat dels números de línia.
  4. Comença a depurar prement F5 o seleccionant "Executar > Començar a depurar" des del menú.
  5. Utilitza els controls de depuració per avançar pas a pas pel teu codi i inspeccionar variables.

Utilitzar la Macro `dbg!`

La macro dbg! és una manera convenient d'imprimir el valor d'una expressió i la seva ubicació en el codi font. És especialment útil per a una depuració ràpida.

Exemple

fn main() {
    let x = 10;
    let y = 20;
    let result = dbg!(add(x, y));
    println!("Resultat: {}", result);
}

fn add(a: i32, b: i32) -> i32 {
    dbg!(a + b)
}

En aquest exemple, la macro dbg! imprimeix el valor de l'expressió i el fitxer i número de línia on es crida.

Depuració Avançada amb GDB i LLDB

Per a una depuració més avançada, pots utilitzar GDB (GNU Debugger) o LLDB (LLVM Debugger). Aquestes eines et permeten realitzar una depuració de baix nivell, com inspeccionar memòria i registres.

Exemple

  1. Compila el teu codi de Rust amb informació de depuració: 
    rustc -g main.rs
  2. Inicia GDB o LLDB amb el teu binari compilat: 
    gdb ./main
    o 
    lldb ./main
  3. Estableix punts de ruptura, executa el teu programa i utilitza les ordres del depurador per inspeccionar variables i avançar pas a pas pel codi.

Resum

La depuració és una part crucial del procés de desenvolupament, i Rust proporciona diverses eines i tècniques per ajudar-te a identificar i solucionar problemes en el teu codi. Des de la simple depuració amb print i l'ús dels missatges d'error del compilador de Rust fins a la depuració avançada amb IDEs i depuradors de baix nivell com GDB i LLDB, tens una àmplia gamma d'opcions per triar. En dominar aquestes tècniques de depuració, pots convertir-te en un programador de Rust més efectiu i eficient.
