Introducció als HashMaps

Els HashMaps són una de les estructures de dades més utilitzades en programació. Et permeten emmagatzemar dades en parelles clau-valor, proporcionant una recuperació eficient de dades basada en claus. A Rust, el HashMap forma part de la biblioteca estàndard i ofereix una manera robusta de gestionar col·leccions d'elements.

Conceptes Clau

  • Parelles Clau-Valor: Cada entrada en un HashMap consisteix en una clau i un valor.
  • Hashing: Les claus es codifiquen per determinar on emmagatzemar el valor a la memòria.
  • Propietat: Les regles de propietat de Rust s'apliquen a les claus i valors en un HashMap.
  • Rendiment: Els HashMaps proporcionen una complexitat temporal mitjana de O(1) per a insercions, eliminacions i cerques.

Creació i Ús dels HashMaps

Creació d'un HashMap

Per crear un HashMap, cal importar el tipus HashMap del mòdul std::collections.

use std::collections::HashMap;

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);
    
    println!("{:?}", scores);
}

Accés als Valors

Pots accedir als valors en un HashMap utilitzant el mètode get, que retorna una Option<&V>.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    let team_name = String::from("Blue");
    match scores.get(&team_name) {
        Some(score) => println!("El puntatge per a {} és {}", team_name, score),
        None => println!("No s'ha trobat cap puntatge per a {}", team_name),
    }
}

Iteració sobre un HashMap

Pots iterar sobre les parelles clau-valor en un HashMap utilitzant un bucle for.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    for (key, value) in &scores {
        println!("{}: {}", key, value);
    }
}

Actualització de Valors

Pots actualitzar valors en un HashMap inserint un nou valor per a una clau existent.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    scores.insert(String::from("Blue"), 25); // Actualitza el valor per a "Blue"

    println!("{:?}", scores);
}

Eliminació de Valors

Per eliminar un valor d'un HashMap, utilitza el mètode remove.

fn main() {
    let mut scores = HashMap::new();
    scores.insert(String::from("Blue"), 10);
    scores.insert(String::from("Yellow"), 50);

    scores.remove(&String::from("Blue"));

    println!("{:?}", scores);
}

Ús Avançat dels HashMaps

Gestió de Propietat i Préstec

Les regles de propietat de Rust s'apliquen als HashMaps. Quan insereixes una parella clau-valor, el HashMap adquireix la propietat de la clau i el valor.

fn main() {
    let field_name = String::from("Favorite color");
    let field_value = String::from("Blue");

    let mut map = HashMap::new();
    map.insert(field_name, field_value);

    // field_name i field_value ja no són vàlids aquí
}

Ús de HashMap amb Tipus Personalitzats

Pots utilitzar tipus personalitzats com a claus o valors en un HashMap, però el tipus de clau ha d'implementar els trets Eq i Hash.

use std::collections::HashMap;

#[derive(Hash, Eq, PartialEq, Debug)]
struct Person {
    name: String,
    age: u8,
}

fn main() {
    let mut people = HashMap::new();
    let person = Person { name: String::from("Alice"), age: 30 };

    people.insert(person, "Engineer");

    for (key, value) in &people {
        println!("{:?} és un/a {}", key, value);
    }
}

Mètodes de HashMap

Aquí tens alguns mètodes útils proporcionats pel tipus HashMap:

| Mètode | Descripció | |-----------------|-----------------------------------------------------| | new | Crea un HashMap buit. | | insert | Insereix una parella clau-valor al HashMap. | | get | Retorna una referència al valor corresponent a la clau. | | remove | Elimina una parella clau-valor del HashMap. | | contains_key | Comprova si el HashMap conté una clau. | | entry | Proporciona accés mutable in situ a una entrada. |

Conclusió

Els HashMaps són una estructura de dades potent i flexible a Rust, adequada per a diversos casos d'ús on necessites emmagatzemar i recuperar dades de manera eficient. Entendre com crear, accedir, actualitzar i gestionar HashMaps millorarà significativament la teva capacitat per escriure programes efectius en Rust. A mesura que avancis, descobriràs que dominar els HashMaps és crucial per gestionar estructures de dades i algorismes més complexos.
