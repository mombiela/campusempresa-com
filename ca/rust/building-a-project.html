Construir un projecte en Rust implica diversos passos, des de configurar el teu entorn fins a escriure, provar i desplegar el teu codi. Aquest tema et guiarà a través del procés, proporcionant explicacions clares, exemples i exercicis per ajudar-te a entendre cada pas.

Configurar el Teu Entorn

Abans de començar a construir un projecte, necessites configurar el teu entorn de desenvolupament Rust.

  • Instal·lar Rust: Utilitza rustup, l'instal·lador de la cadena d'eines de Rust. 
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Actualitzar Rust: Assegura't de tenir la versió més recent. 
    rustup update
  • Instal·lar un IDE o Editor de Text: Les opcions populars inclouen Visual Studio Code amb l'extensió de Rust, IntelliJ IDEA amb el plugin de Rust, o Sublime Text.

Crear un Nou Projecte

Els projectes de Rust es gestionen utilitzant Cargo, el gestor de paquets i sistema de construcció de Rust.

  • Crear un nou projecte:

    cargo new my_project
cd my_project

    Aquesta comanda crea un nou directori anomenat my_project amb l'estructura següent:

    my_project
├── Cargo.toml
└── src
    └── main.rs

  • Cargo.toml: Aquest fitxer conté metadades sobre el teu projecte, com ara el seu nom, versió i dependències.

  • src/main.rs: Aquest és el fitxer de codi font principal on residirà el teu codi Rust.

Escriure el Teu Primer Programa

Escrivim un simple programa de "Hola, món!" per començar.

  • Edita src/main.rs:

    fn main() {
    println!("Hello, world!");
}

  • Construeix i executa el projecte:

    cargo run

    Aquesta comanda compila el teu codi i executa l'executable resultant.

Afegir Dependències

A mesura que el teu projecte creixi, probablement necessitaràs afegir biblioteques externes (crates).

  • Afegeix una dependència: Obre Cargo.toml i afegeix la dependència sota [dependencies].

    [dependencies]
rand = "0.8"

  • Utilitza la dependència en el teu codi:

    use rand::Rng;

fn main() {
    let mut rng = rand::thread_rng();
    let n: u32 = rng.gen_range(0..10);
    println!("Random number: {}", n);
}

Estructura del Projecte

A mesura que el teu projecte es torni més complex, necessitaràs organitzar el teu codi en mòduls i fitxers separats.

  • Crear un mòdul: Crea un nou fitxer al directori src, per exemple, src/lib.rs.

    // src/lib.rs
pub fn greet() {
    println!("Hello from the library!");
}

  • Utilitza el mòdul a main.rs:

    mod lib;

fn main() {
    lib::greet();
}

Provar el Teu Codi

Rust té suport integrat per escriure i executar proves.

  • Escriu una prova: Afegeix una funció de prova a src/lib.rs.

    #[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_greet() {
        greet();
    }
}

  • Executa les proves:

    cargo test

Construir per a la Publicació

Quan el teu projecte estigui llest per al desplegament, voldràs construir-lo en mode de publicació per obtenir un rendiment optimitzat.

  • Construeix en mode de publicació: 
    cargo build --release
    El binari optimitzat es trobarà al directori target/release.

Conclusió

Construir un projecte en Rust implica configurar el teu entorn, crear un nou projecte, escriure i organitzar el teu codi, afegir dependències, provar i finalment construir per a la publicació. Seguint aquests passos, pots gestionar i desenvolupar eficientment els teus projectes Rust de principi a fi. Practica aquests passos amb projectes petits per sentir-te més còmode amb l'ecosistema de Rust i les seves eines.
