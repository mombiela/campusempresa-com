Introducció a les Funcions

Les funcions són blocs de construcció fonamentals en Rust, permetent encapsular codi en unitats reutilitzables. Ajuden a organitzar el codi, millorar la llegibilitat i reduir la redundància.

Sintaxi Bàsica

Una funció en Rust es defineix utilitzant la paraula clau fn, seguida del nom de la funció, els paràmetres i el cos de la funció.

fn main() {
    println!("Hola, món!");
}

Definició de Funcions

Per definir una funció, especifiques el nom, els paràmetres (si n'hi ha) i el tipus de retorn (si n'hi ha).

fn greet(name: &str) {
    println!("Hola, {}!", name);
}

fn main() {
    greet("Alice");
}
  • fn: Paraula clau per definir una funció.
  • greet: Nom de la funció.
  • name: &str: Paràmetre amb tipus &str (slice de cadena).
  • println!: Macro per imprimir a la consola.

Paràmetres i Tipus de Retorn

Les funcions poden tenir múltiples paràmetres i retornar un valor. El tipus de retorn es especifica després d'una fletxa (->).

fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    let sum = add(5, 3);
    println!("Suma: {}", sum);
}
  • a: i32, b: i32: Paràmetres de tipus i32.
  • -> i32: Tipus de retorn i32.
  • a + b: Expressió per retornar la suma de a i b.

Conceptes Avançats de Funcions

Sobrecàrrega de Funcions (No Suportada)

Rust no suporta la sobrecàrrega de funcions directament. En canvi, pots utilitzar traits o diferents noms de funció per aconseguir una funcionalitat similar.

Closures

Les closures són funcions anònimes que poden capturar variables del seu entorn.

fn main() {
    let x = 10;
    let add_x = |y: i32| x + y;
    println!("Resultat: {}", add_x(5));
}
  • |y: i32| x + y: Closure capturant x de l'entorn.
  • add_x(5): Cridant la closure amb l'argument 5.

Funcions d'Ordre Superior

Les funcions que prenen altres funcions com a paràmetres o que retornen funcions es diuen funcions d'ordre superior.

fn apply<F>(f: F, x: i32) -> i32
where
    F: Fn(i32) -> i32,
{
    f(x)
}

fn main() {
    let double = |x: i32| x * 2;
    println!("Resultat: {}", apply(double, 5));
}
  • F: Fn(i32) -> i32: Paràmetre genèric F restringit a ser una funció que pren i32 i retorna i32.
  • apply(double, 5): Aplicant la closure double a 5.

Punteres de Funció

Les punteres de funció permeten passar funcions com a arguments a altres funcions.

fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn apply_function(f: fn(i32, i32) -> i32, a: i32, b: i32) -> i32 {
    f(a, b)
}

fn main() {
    println!("Resultat: {}", apply_function(add, 2, 3));
}
  • fn(i32, i32) -> i32: Tipus de la puntera de funció.
  • apply_function(add, 2, 3): Passant la funció add com a argument.

Resum

Les funcions en Rust són eines versàtils que ajuden a organitzar i reutilitzar el codi. Des de definicions bàsiques de funcions fins a conceptes avançats com closures i funcions d'ordre superior, Rust proporciona característiques poderoses per treballar amb funcions de manera efectiva. Entendre aquests conceptes és crucial per escriure codi en Rust que sigui net, eficient i mantenible.
