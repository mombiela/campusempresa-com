Flux de control a Rust
Entendre la Propietat ►

El flux de control és un concepte fonamental en la programació que et permet dictar l'ordre en què s'executen les instruccions i expressions. A Rust, el flux de control es gestiona mitjançant declaracions condicionals, bucles i coincidència de patrons. Aquesta secció et guiarà des de construccions bàsiques fins a avançades de flux de control a Rust.

Declaracions condicionals

Les declaracions condicionals et permeten executar codi basat en certes condicions.

Declaracions `if`

La declaració if s'utilitza per executar codi només si una condició és certa.

fn main() {
    let number = 7;

    if number < 10 {
        println!("El número és menor que 10");
    }
}

`else` i `else if`

Pots estendre la declaració if amb else i else if per gestionar múltiples condicions.

fn main() {
    let number = 7;

    if number < 5 {
        println!("El número és menor que 5");
    } else if number == 5 {
        println!("El número és exactament 5");
    } else {
        println!("El número és més gran que 5");
    }
}

Ús de `if` en una declaració `let`

A Rust, pots utilitzar if en una declaració let per assignar un valor basat en una condició.

fn main() {
    let condition = true;
    let number = if condition { 5 } else { 10 };

    println!("El valor del número és: {}", number);
}

Bucles

Els bucles et permeten executar un bloc de codi repetidament.

`loop`

La paraula clau loop crea un bucle infinit.

fn main() {
    let mut counter = 0;

    loop {
        counter += 1;
        println!("Comptador: {}", counter);

        if counter == 5 {
            break;
        }
    }
}

`while`

El bucle while s'executa mentre una condició és certa.

fn main() {
    let mut number = 3;

    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }

    println!("DESPEGAMENT!!!");
}

`for`

El bucle for s'utilitza per iterar sobre una col·lecció.

fn main() {
    let a = [10, 20, 30, 40, 50];

    for element in a.iter() {
        println!("El valor és: {}", element);
    }
}

Coincidència de patrons

La coincidència de patrons a Rust és una característica poderosa que et permet comparar un valor amb una sèrie de patrons.

`match`

La declaració match s'utilitza per a la coincidència de patrons.

fn main() {
    let number = 6;

    match number {
        1 => println!("Un!"),
        2 => println!("Dos!"),
        3 => println!("Tres!"),
        4 | 5 | 6 => println!("Quatre, cinc o sis!"),
        _ => println!("Una altra cosa!"),
    }
}

Desestructuració amb `match`

Pots desestructurar tipus de dades complexos amb match.

fn main() {
    let pair = (2, -2);

    match pair {
        (x, y) if x == y => println!("Els números són iguals"),
        (x, y) if x + y == 0 => println!("Els números són oposats"),
        (x, _) => println!("El primer número és {}", x),
    }
}

Conclusió

El flux de control a Rust es gestiona mitjançant declaracions condicionals, bucles i coincidència de patrons. Entendre aquestes construccions és essencial per escriure codi eficient i llegible a Rust. En dominar if, else, loop, while, for i match, pots gestionar una àmplia gamma d'escenaris de programació de manera efectiva.
