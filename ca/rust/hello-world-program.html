Introducció

El programa "Hola, món!" és un programa senzill però fonamental que sovint s'utilitza per introduir un nou llenguatge de programació. En aquesta secció, veurem com escriure i executar un programa "Hola, món!" en Rust, explicant cada part del codi en detall.

Escriure el programa "Hola, món!"

Configurar el teu entorn

Abans d'escriure el nostre primer programa en Rust, assegura't que tens Rust instal·lat a la teva màquina. Pots instal·lar Rust seguint les instruccions al lloc web oficial de Rust.

Crear un nou projecte

Rust utilitza un sistema de construcció i gestor de paquets anomenat Cargo. Per crear un nou projecte, obre el teu terminal i executa:

cargo new hello_world
cd hello_world

Això crearà un nou directori anomenat hello_world amb l'estructura següent:

hello_world
├── Cargo.toml
└── src
    └── main.rs

Escriure el codi

Obre el fitxer src/main.rs en el teu editor de text preferit i reemplaça el seu contingut amb el codi següent:

fn main() {
    println!("Hola, món!");
}

Explicació

  • fn main() { ... }: Això defineix la funció principal, que és el punt d'entrada d'un programa en Rust.
  • println!("Hola, món!");: Aquesta línia imprimeix "Hola, món!" a la consola. println! és una macro en Rust utilitzada per imprimir text.

Executar el programa

Per executar el teu programa, utilitza la comanda següent en el teu terminal:

cargo run

Hauries de veure la sortida:

Hola, món!

Conceptes clau

  • Funcions: En Rust, les funcions es defineixen utilitzant la paraula clau fn. La funció main és especial perquè és el punt d'entrada del programa.
  • Macros: La macro println! s'utilitza per imprimir text a la consola. Les macros en Rust s'invoquen amb un ! i poden realitzar generació de codi en temps de compilació.
  • Cargo: Cargo és el sistema de construcció i gestor de paquets de Rust, que simplifica la gestió de dependències i la construcció de projectes.

Conclusió

En aquesta secció, hem vist com escriure i executar un senzill programa "Hola, món!" en Rust. Hem introduït l'estructura bàsica d'un programa en Rust, incloent la funció principal i l'ús de la macro println!. Entendre aquests conceptes bàsics és crucial ja que formen la base per a programes més complexos en Rust.
