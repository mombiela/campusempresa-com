Introducció als Enums

Els Enums, abreviatura de "enumeracions", són una característica poderosa a Rust que et permeten definir un tipus enumerant els seus possibles valors. Els Enums són particularment útils per representar un valor que podria ser un de diversos tipus diferents.

Definició d'Enums

Per definir un enum, utilitza la paraula clau enum seguida del nom de l'enum i les seves variants possibles.

enum Direction {
    North,
    South,
    East,
    West,
}
  • Direction és el nom de l'enum.
  • North, South, East, West són les variants de l'enum.

Ús dels Enums

Pots crear instàncies d'un enum especificant el nom de l'enum seguit de :: i el nom de la variant.

let direction = Direction::North;

Variants d'Enum amb Dades

Els Enums també poden contenir dades, cosa que els fa encara més versàtils.

enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
  • Quit: No té dades associades.
  • Move: Conté una estructura anònima amb els camps x i y.
  • Write: Conté un String.
  • ChangeColor: Conté una tupla amb tres valors i32.

Coincidència de Patrons

La coincidència de patrons és un mecanisme per comprovar un valor contra un patró. És una construcció de control de flux poderosa que et permet desestructurar i coincidir amb tipus de dades complexos com els enums.

L'expressió `match`

L'expressió match s'utilitza per comparar un valor contra una sèrie de patrons i executar codi basat en quin patró coincideix.

fn main() {
    let direction = Direction::North;

    match direction {
        Direction::North => println!("Heading North!"),
        Direction::South => println!("Heading South!"),
        Direction::East => println!("Heading East!"),
        Direction::West => println!("Heading West!"),
    }
}
  • L'expressió match pren un valor i el compara contra cada patró per torn.
  • Quan un patró coincideix, el bloc de codi corresponent s'executa.

Coincidència amb Enums que Contenen Dades

Quan coincideixes contra variants d'enum que contenen dades, pots vincular les dades a variables dins del patró.

fn process_message(msg: Message) {
    match msg {
        Message::Quit => println!("Quit message received."),
        Message::Move { x, y } => println!("Move to coordinates: ({}, {})", x, y),
        Message::Write(text) => println!("Message: {}", text),
        Message::ChangeColor(r, g, b) => println!("Change color to RGB: ({}, {}, {})", r, g, b),
    }
}
  • Message::Move { x, y }: Desestructura l'estructura anònima i vincula x i y als valors respectius.
  • Message::Write(text): Vincula text al String contingut en la variant.
  • Message::ChangeColor(r, g, b): Vincula r, g i b als valors i32 respectius en la tupla.

Coincidència de Patrons Avançada

La coincidència de patrons a Rust pot ser molt sofisticada, permetent condicions complexes i desestructuració.

Coincidència de Múltiples Patrons

Pots coincidir amb múltiples patrons utilitzant l'operador |.

fn main() {
    let direction = Direction::North;

    match direction {
        Direction::North | Direction::South => println!("Vertical direction"),
        Direction::East | Direction::West => println!("Horizontal direction"),
    }
}

Coincidència de Rangs

Pots coincidir amb rangs de valors utilitzant ..=.

fn main() {
    let number = 5;

    match number {
        1..=5 => println!("Number is between 1 and 5"),
        _ => println!("Number is out of range"),
    }
}

Desestructuració d'Enums Complexos

Pots desestructurar estructures de dades niades dins dels enums.

enum ComplexMessage {
    Move { x: i32, y: i32, z: i32 },
    ChangeColor { red: i32, green: i32, blue: i32 },
}

fn process_complex_message(msg: ComplexMessage) {
    match msg {
        ComplexMessage::Move { x, y, z } => println!("Move to coordinates: ({}, {}, {})", x, y, z),
        ComplexMessage::ChangeColor { red, green, blue } => println!("Change color to RGB: ({}, {}, {})", red, green, blue),
    }
}

Conclusió

Els Enums i la coincidència de patrons són característiques essencials a Rust que proporcionen maneres poderoses de representar i manipular dades. Els Enums et permeten definir un tipus amb múltiples variants, opcionalment contenint dades. La coincidència de patrons et permet desestructurar i manejar aquestes variants d'una manera neta i llegible.

  • Enums: Defineixen tipus amb múltiples variants.
  • Coincidència de Patrons: Desestructuren i manegen tipus de dades complexos.

En dominar aquests conceptes, podràs escriure codi Rust més expressiu i robust.
