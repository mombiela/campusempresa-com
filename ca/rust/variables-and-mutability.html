Variables i mutabilitat a Rust
Tipus de Dades ►

Introducció

A Rust, les variables són blocs de construcció fonamentals que emmagatzemen dades. Entendre com declarar, utilitzar i modificar variables és crucial per escriure programes efectius en Rust. Aquesta secció cobrirà els conceptes bàsics de les variables, el concepte de mutabilitat i com treballar amb variables mutables i immutables.

Declaració de variables

A Rust, les variables es declaren utilitzant la paraula clau let. Per defecte, les variables són immutables, és a dir, els seus valors no es poden canviar un cop establerts.

fn main() {
    let x = 5;
    println!("El valor de x és: {}", x);
}
  • let: Paraula clau utilitzada per declarar una variable.
  • x: Nom de la variable.
  • 5: Valor inicial assignat a la variable x.

Mutabilitat

Per permetre que el valor d'una variable es pugui canviar, cal declarar-la com a mutable utilitzant la paraula clau mut.

fn main() {
    let mut y = 10;
    println!("El valor inicial de y és: {}", y);
    y = 15;
    println!("El nou valor de y és: {}", y);
}
  • mut: Paraula clau que indica que la variable és mutable.
  • y = 15;: Canvi del valor de y després de la seva declaració inicial.

Ombrejament

Rust permet declarar una nova variable amb el mateix nom que una variable anterior. Això es coneix com a ombrejament i permet reutilitzar noms de variables mentre es transformen els seus valors.

fn main() {
    let z = 6;
    let z = z + 1;
    println!("El valor de z després de l'ombrejament és: {}", z);
}
  • L'ombrejament permet transformacions sense necessitat de declarar nous noms de variables.
  • El nou z ombreja l'antic z, però són variables diferents.

Constants

Les constants són similars a les variables immutables però es declaren utilitzant la paraula clau const i han de tenir el seu tipus especificat. Les constants són sempre immutables i es poden declarar en qualsevol àmbit, inclòs l'àmbit global.

const MAX_POINTS: u32 = 100_000;
  • const: Paraula clau utilitzada per declarar una constant.
  • MAX_POINTS: Nom de la constant, convencionalment escrit en majúscules.
  • u32: Tipus de la constant.
  • 100_000: Valor assignat a la constant.

Tipus de variables

Rust és un llenguatge tipat estàticament, el que significa que el tipus d'una variable ha de ser conegut en temps de compilació. Rust sovint pot inferir el tipus basant-se en el valor, però també es pot especificar explícitament el tipus.

fn main() {
    let a: i32 = 42; // Anotació de tipus explícita
    let b = 3.14;    // Tipus inferit com a f64
}
  • i32: Enter signat de 32 bits.
  • f64: Número de coma flotant de 64 bits.

Exemples i exercicis

Exemple 1: Variable immutable

fn main() {
    let name = "Alice";
    println!("Hola, {}!", name);
}
  • Aquest exemple demostra una variable immutable name que conté una cadena.

Exemple 2: Variable mutable

fn main() {
    let mut counter = 1;
    println!("Counter: {}", counter);
    counter += 1;
    println!("Counter: {}", counter);
}
  • Aquest exemple mostra una variable mutable counter el valor de la qual s'incrementa.

Exercici 1: Declarar i modificar una variable mutable

  1. Declara una variable mutable score i estableix el seu valor inicial a 50.
  2. Imprimeix el valor de score.
  3. Canvia el valor de score a 75.
  4. Imprimeix el nou valor de score.

Exercici 2: Utilitzar l'ombrejament

  1. Declara una variable immutable temperature i estableix el seu valor a 20.
  2. Ombreja temperature afegint 5 al seu valor.
  3. Imprimeix el nou valor de temperature.

Resum

  • Les variables a Rust són immutables per defecte.
  • Utilitza la paraula clau mut per fer una variable mutable.
  • L'ombrejament permet reutilitzar noms de variables per a transformacions.
  • Les constants són sempre immutables i han de tenir el seu tipus especificat.
  • Rust és un llenguatge tipat estàticament, i els tipus es poden inferir o especificar explícitament.

Entendre les variables i la mutabilitat és essencial per gestionar dades i estat en els teus programes Rust. Practica declarant, modificant i ombrejant variables per convertir-te en un expert en la gestió de dades a Rust.
