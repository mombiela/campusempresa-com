Introducció als Structs

Els structs a Rust són una manera de crear tipus de dades personalitzats que agrupen valors relacionats. Són similars a les classes en altres llenguatges de programació però sense mètodes. Els structs són essencials per organitzar dades complexes i fer que el teu codi sigui més llegible i mantenible.

Definició d'un Struct

Per definir un struct a Rust, utilitzes la paraula clau struct seguida del nom del struct i els seus camps.

Definició Bàsica d'un Struct

Aquí tens un exemple senzill de definició d'un struct:

struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}

En aquest exemple:

  • User és el nom del struct.
  • username, email, sign_in_count i active són els camps del struct.
  • Cada camp té un nom i un tipus.

Creació d'Instàncies de Structs

Un cop has definit un struct, pots crear instàncies d'aquest.

Creació d'una Instància

Aquí tens com crear una instància del struct User:

fn main() {
    let user1 = User {
        username: String::from("johndoe"),
        email: String::from("johndoe@example.com"),
        sign_in_count: 1,
        active: true,
    };

    println!("Username: {}", user1.username);
}

En aquest exemple:

  • user1 és una instància del struct User.
  • La funció String::from s'utilitza per crear valors String per als camps username i email.
  • La macro println! s'utilitza per imprimir el camp username de user1.

Actualització de Structs

Pots crear una nova instància d'un struct basant-te en una instància existent, utilitzant la sintaxi d'actualització.

Sintaxi d'Actualització

Aquí tens un exemple d'actualització d'un struct:

fn main() {
    let user1 = User {
        username: String::from("johndoe"),
        email: String::from("johndoe@example.com"),
        sign_in_count: 1,
        active: true,
    };

    let user2 = User {
        email: String::from("janedoe@example.com"),
        ..user1
    };

    println!("Username: {}", user2.username);
    println!("Email: {}", user2.email);
}

En aquest exemple:

  • user2 es crea basant-se en user1.
  • La sintaxi ..user1 copia els camps restants de user1 a user2.

Tuple Structs

Rust també suporta tuple structs, que són similars als tuples regulars però tenen tipus amb nom.

Definició i Ús de Tuple Structs

Aquí tens un exemple d'un tuple struct:

struct Color(i32, i32, i32);

fn main() {
    let black = Color(0, 0, 0);

    println!("Black color: ({}, {}, {})", black.0, black.1, black.2);
}

En aquest exemple:

  • Color és un tuple struct amb tres camps i32.
  • black és una instància del tuple struct Color.
  • Els camps de black s'accedeixen utilitzant la notació de punt amb índexs.

Structs de Tipus Unitari

Els structs de tipus unitari són structs sense cap camp. Són útils per implementar traits sense emmagatzemar dades.

Definició i Ús de Structs de Tipus Unitari

Aquí tens un exemple d'un struct de tipus unitari:

struct AlwaysEqual;

fn main() {
    let subject = AlwaysEqual;

    println!("This struct has no fields!");
}

En aquest exemple:

  • AlwaysEqual és un struct de tipus unitari.
  • subject és una instància de AlwaysEqual.

Resum

Els structs a Rust són una manera poderosa d'agrupar dades relacionades. Venen en tres variants: structs regulars, tuple structs i structs de tipus unitari. Entendre com definir i utilitzar structs és fonamental per organitzar i gestionar dades complexes en programes Rust.

  • Els structs regulars tenen camps amb nom.
  • Els tuple structs tenen camps sense nom.
  • Els structs de tipus unitari no tenen camps.

En dominar els structs, pots escriure codi Rust més organitzat i mantenible.
