Introducció als Fils

Els fils permeten que un programa realitzi múltiples tasques simultàniament, fent un ús eficient dels recursos del sistema. Rust proporciona un suport robust per als fils, garantint seguretat i rendiment.

Conceptes Clau

  • Concurrència: Executar múltiples tasques al mateix temps.
  • Paral·lelisme: Executar múltiples tasques simultàniament en múltiples processadors.
  • Seguretat de Fils: Assegurar que les dades s'accedeixen correctament quan hi ha múltiples fils implicats.

Creació de Fils

La biblioteca estàndard de Rust proporciona el mòdul std::thread per a la creació i gestió de fils.

Creació Bàsica de Fils

Per crear un nou fil, utilitzes la funció thread::spawn.

use std::thread;

fn main() {
    let handle = thread::spawn(|| {
        for i in 1..10 {
            println!("Hola des del fil creat: {}", i);
        }
    });

    for i in 1..5 {
        println!("Hola des del fil principal: {}", i);
    }

    handle.join().unwrap();
}
  • Explicació:
    • thread::spawn crea un nou fil.
    • El tancament dins de spawn és el codi que s'executa en el nou fil.
    • handle.join() assegura que el fil principal espera que el fil creat acabi.

Seguretat de Fils amb `Mutex` i `Arc`

Quan múltiples fils necessiten accedir a dades compartides, Rust proporciona Mutex (Exclusió Mútua) i Arc (Comptatge de Referències Atòmic) per garantir la seguretat dels fils.

Ús de `Mutex`

Un Mutex permet que només un fil accedeixi a les dades a la vegada.

use std::sync::{Mutex, Arc};
use std::thread;

fn main() {
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];

    for _ in 0..10 {
        let counter = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let mut num = counter.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }

    for handle in handles {
        handle.join().unwrap();
    }

    println!("Resultat: {}", *counter.lock().unwrap());
}
  • Explicació:
    • Arc permet la propietat múltiple del Mutex.
    • Mutex::lock bloqueja les dades per al fil actual.
    • Arc::clone crea una referència al Mutex embolicat en Arc.

Gestió Avançada de Fils

Rust proporciona funcions més avançades per a la gestió de fils, com ara canals per a la comunicació entre fils.

Ús de Canals

Els canals permeten que els fils es comuniquin enviant missatges.

use std::sync::mpsc;
use std::thread;
use std::time::Duration;

fn main() {
    let (tx, rx) = mpsc::channel();

    thread::spawn(move || {
        let val = String::from("hola");
        tx.send(val).unwrap();
    });

    let rebut = rx.recv().unwrap();
    println!("Rebut: {}", rebut);
}
  • Explicació:
    • mpsc::channel crea un transmissor (tx) i un receptor (rx).
    • tx.send envia dades al canal.
    • rx.recv rep dades del canal.

Conclusió

Els fils en Rust proporcionen eines poderoses per a la programació concurrent mentre garanteixen seguretat i rendiment. Utilitzant thread::spawn, Mutex, Arc i canals, pots crear aplicacions multithread eficients i segures. Entendre aquests conceptes és crucial per desenvolupar programes robustos en Rust que aprofitin tot el potencial dels processadors multicore moderns.
