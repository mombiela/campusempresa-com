Introducció a Rust

Rust és un llenguatge de programació de sistemes que se centra en la velocitat, la seguretat de la memòria i el paral·lelisme. És sintàcticament similar a C++, però garanteix la seguretat de la memòria mitjançant un comprovador de préstecs per validar referències.

Conceptes clau

  • Variables i mutabilitat
  • Tipus de dades
  • Funcions
  • Flux de control
  • Propietat i préstec

Variables i mutabilitat

A Rust, les variables són immutables per defecte. Per fer una variable mutable, cal utilitzar la paraula clau mut.

fn main() {
    let x = 5; // variable immutable
    println!("El valor de x és: {}", x);

    let mut y = 10; // variable mutable
    println!("El valor de y és: {}", y);
    y = 15;
    println!("El nou valor de y és: {}", y);
}

Explicació

  • let x = 5; declara una variable immutable x amb el valor 5.
  • let mut y = 10; declara una variable mutable y amb el valor 10.
  • y = 15; canvia el valor de y a 15.

Tipus de dades

Rust és un llenguatge de tipus estàtic, el que significa que ha de conèixer els tipus de totes les variables en temps de compilació. Els principals tipus de dades inclouen:

  • Tipus escalars: enters, números de punt flotant, booleans i caràcters.
  • Tipus compostos: tuples i arrays.

Exemple de tipus escalars

fn main() {
    let integer: i32 = 42;
    let float: f64 = 3.14;
    let boolean: bool = true;
    let character: char = 'R';

    println!("Enter: {}", integer);
    println!("Flotant: {}", float);
    println!("Booleà: {}", boolean);
    println!("Caràcter: {}", character);
}

Exemple de tipus compostos

fn main() {
    let tuple: (i32, f64, u8) = (500, 6.4, 1);
    let array: [i32; 3] = [1, 2, 3];

    println!("Tuple: ({}, {}, {})", tuple.0, tuple.1, tuple.2);
    println!("Array: [{}, {}, {}]", array[0], array[1], array[2]);
}

Funcions

Les funcions a Rust es defineixen utilitzant la paraula clau fn. Poden prendre paràmetres i retornar valors.

fn main() {
    let result = add(5, 3);
    println!("La suma és: {}", result);
}

fn add(x: i32, y: i32) -> i32 {
    x + y
}

Explicació

  • fn main() és el punt d'entrada del programa.
  • fn add(x: i32, y: i32) -> i32 defineix una funció anomenada add que pren dos paràmetres i32 i retorna un i32.

Flux de control

Rust proporciona construccions de flux de control comunes com if, else, loop, while i for.

Exemple d'If-Else

fn main() {
    let number = 7;

    if number < 5 {
        println!("El número és menor que 5");
    } else {
        println!("El número és major o igual a 5");
    }
}

Exemple de Loop

fn main() {
    let mut count = 0;

    loop {
        count += 1;
        if count == 5 {
            break;
        }
    }

    println!("El bucle s'ha aturat en el compte: {}", count);
}

Propietat i préstec

La propietat és la característica més única de Rust, que permet la seguretat de la memòria sense un recollidor d'escombraries. Cada valor a Rust té una variable que s'anomena el seu propietari. Només pot haver-hi un propietari alhora, i quan el propietari surt de l'abast, el valor es descarta.

Exemple de propietat

fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 es mou a s2

    // println!("{}", s1); // Aquesta línia causaria un error en temps de compilació
    println!("{}", s2);
}

Exemple de préstec

fn main() {
    let s1 = String::from("hello");
    let len = calculate_length(&s1); // s1 es presta

    println!("La longitud de '{}' és {}.", s1, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

Explicació

  • let s2 = s1; mou s1 a s2, fent que s1 ja no sigui vàlid.
  • &s1 crea una referència a s1 que es pot utilitzar sense prendre'n la propietat.

Conclusió

Entendre la sintaxi bàsica i l'estructura de Rust és crucial per escriure codi segur i eficient. Dominant les variables, els tipus de dades, les funcions, el flux de control i la propietat, es posa una base sòlida per a conceptes més avançats de programació en Rust. Practica aquests conceptes bàsics a fons per convertir-te en un programador proficient en Rust.
