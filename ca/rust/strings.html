Introducció a les cadenes

Les cadenes són una part fonamental de la programació, permetent-te treballar amb text. A Rust, les cadenes són més complexes que en alguns altres llenguatges a causa del focus de Rust en la seguretat i el rendiment.

Conceptes clau

  • Tipus de cadenes: Rust té dos tipus principals de cadenes: String i &str.
  • Propietat i préstec: Entendre com Rust maneja la memòria amb cadenes.
  • Codificació UTF-8: Les cadenes de Rust estan codificades en UTF-8, cosa que afecta com treballes amb elles.

Tipus de cadenes

`String`

El tipus String és una estructura de dades ampliable i allotjada al munt. S'utilitza quan necessites una cadena pròpia que puguis modificar.

Exemple

fn main() {
    let mut s = String::from("Hello");
    s.push_str(", world!");
    println!("{}", s); // Outputs: Hello, world!
}
  • String::from("Hello"): Crea una nova String a partir d'un literal de cadena.
  • s.push_str(", world!"): Afegeix una porció de cadena a la String.

`&str`

El tipus &str, també conegut com a porció de cadena, és una referència a una cadena. S'utilitza quan necessites una vista immutable d'una cadena.

Exemple

fn main() {
    let s = "Hello, world!";
    println!("{}", s); // Outputs: Hello, world!
}
  • "Hello, world!": Un literal de cadena, que és de tipus &str.

Propietat i préstec amb cadenes

Propietat

A Rust, String posseeix les seves dades, cosa que significa que és responsable de gestionar la memòria on s'emmagatzema la cadena.

Exemple

fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 es mou a s2
    // println!("{}", s1); // Això causaria un error en temps de compilació
}
  • s1 es mou a s2, i s1 ja no és vàlid.

Préstec

Pots prestar una cadena utilitzant una referència, permetent-te utilitzar la cadena sense prendre'n la propietat.

Exemple

fn main() {
    let s1 = String::from("hello");
    let len = calculate_length(&s1);
    println!("The length of '{}' is {}.", s1, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}
  • &s1: Es passa una referència a s1 a la funció.
  • s.len(): Retorna la longitud de la cadena.

Codificació UTF-8

Les cadenes de Rust estan codificades en UTF-8, cosa que significa que poden contenir qualsevol seqüència UTF-8 vàlida. Això afecta com pots indexar i dividir les cadenes.

Exemple

fn main() {
    let hello = "Здравствуйте";
    let s = &hello[0..4];
    println!("{}", s); // Outputs: Зд
}
  • &hello[0..4]: Divideix la cadena, però has de tenir cura de dividir en límits UTF-8 vàlids.

Operacions comunes amb cadenes

Concatenació

Pots concatenar cadenes utilitzant l'operador + o la macro format!.

Exemple

fn main() {
    let s1 = String::from("Hello, ");
    let s2 = String::from("world!");
    let s3 = s1 + &s2; // s1 es mou aquí i ja no es pot utilitzar
    println!("{}", s3); // Outputs: Hello, world!
}

Iteració

Pots iterar sobre les cadenes per caràcters o bytes.

Exemple

fn main() {
    let s = String::from("hello");
    for c in s.chars() {
        println!("{}", c);
    }
}
  • s.chars(): Retorna un iterador sobre els caràcters de la cadena.

Conclusió

Les cadenes a Rust són una manera poderosa i flexible de manejar text, però venen amb complexitats a causa de les regles de propietat i préstec de Rust i la codificació UTF-8. Entendre les diferències entre String i &str, com gestionar la propietat i com realitzar operacions comunes t'ajudarà a treballar eficaçment amb cadenes a Rust.
