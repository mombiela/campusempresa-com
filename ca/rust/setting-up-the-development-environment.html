Configuració de l'Entorn de Desenvolupament
Programa Hola, Món! ►

Configurar l'entorn de desenvolupament és un primer pas crucial per aprendre Rust. Aquesta secció et guiarà a través del procés d'instal·lació de Rust, configuració d'un editor de codi i configuració del teu entorn per a un desenvolupament eficient.

Instal·lant Rust

Per començar a programar en Rust, necessites instal·lar el llenguatge de programació Rust al teu ordinador. Rust proporciona una eina anomenada rustup per gestionar versions de Rust i eines associades.

Passos per Instal·lar Rust

  1. Instal·lar Rustup:

    • Obre el teu terminal.

    • Executa la següent comanda per descarregar i instal·lar rustup:

      curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

  2. Segueix les Instruccions en Pantalla:

    • L'instal·lador et guiarà a través del procés d'instal·lació.
    • Per defecte, instal·larà l'última versió estable de Rust.

  3. Verifica la Instal·lació:

    • Després de la instal·lació, pots verificar que Rust està instal·lat correctament executant:

      rustc --version

      Aquesta comanda hauria de mostrar la versió de Rust que has instal·lat.

Configurant un Editor de Codi

Escollir l'editor de codi adequat pot millorar significativament la teva productivitat. Aquí, configurarem Visual Studio Code (VS Code), un editor popular i potent per al desenvolupament en Rust.

Instal·lant Visual Studio Code

  1. Descarrega VS Code:

    • Ves al lloc web de Visual Studio Code.
    • Descarrega l'instal·lador per al teu sistema operatiu (Windows, macOS o Linux).

  2. Instal·la VS Code:

    • Executa l'instal·lador descarregat i segueix les instruccions d'instal·lació.

Configurant VS Code per a Rust

  1. Instal·la l'Extensió de Rust:

    • Obre VS Code.
    • Ves a la vista d'Extensions fent clic a la icona d'Extensions a la Barra d'Activitat al costat de la finestra o prement Ctrl+Shift+X.
    • Cerca "Rust" i instal·la l'extensió "Rust (rls)" de l'equip de Rust.

  2. Instal·la Extensions Addicionals (Opcional però Recomanat):

    • CodeLLDB: Proporciona suport per a depuració.
    • Crates: Ajuda a gestionar dependències en el teu fitxer Cargo.toml.
    • Rust Analyzer: Una alternativa a RLS, que ofereix més funcions i millor rendiment.

  3. Configura l'Extensió de Rust:

    • Obre el Command Palette (Ctrl+Shift+P).

    • Escriu "Preferences: Open Settings (JSON)" i selecciona-ho.

    • Afegeix la següent configuració als teus paràmetres:

      {
  "rust-analyzer.cargo.runBuildScripts": true,
  "rust-analyzer.procMacro.enable": true
}

Creant el Teu Primer Projecte en Rust

Ara que tens Rust i VS Code configurats, creem el teu primer projecte en Rust utilitzant Cargo, el gestor de paquets i sistema de construcció de Rust.

Passos per Crear un Nou Projecte

  1. Obre el Teu Terminal:

    • Navega al directori on vols crear el teu projecte.

  2. Crea un Nou Projecte de Cargo:

    • Executa la següent comanda:

      cargo new hello_world

    • Aquesta comanda crea un nou directori anomenat hello_world amb l'estructura següent:

      hello_world
├── Cargo.toml
└── src
    └── main.rs

  3. Navega al Directori del Projecte:

    • Canvia al directori del projecte:

      cd hello_world

  4. Construeix i Executa el Teu Projecte:

    • Construeix el projecte executant:

      cargo build

    • Executa el projecte executant:

      cargo run

    • Hauries de veure la sortida:

      Hello, world!

Conclusió

Configurar l'entorn de desenvolupament per a Rust implica instal·lar Rust utilitzant rustup, configurar un editor de codi com Visual Studio Code i configurar-lo amb les extensions necessàries. Amb el teu entorn llest, pots crear i executar el teu primer projecte en Rust utilitzant Cargo. Aquesta configuració proporcionarà una base sòlida per al teu viatge de programació en Rust, permetent-te escriure, construir i depurar aplicacions en Rust de manera eficient.
