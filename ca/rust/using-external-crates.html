Ús de Crates Externs
Lifetimes ►

En Rust, els crates externs són biblioteques de codi que pots utilitzar en els teus projectes per evitar reinventar la roda. Aquesta secció et guiarà a través del procés d'utilitzar crates externs, des de trobar-los i afegir-los al teu projecte, fins a entendre com utilitzar-los de manera efectiva.

Què són els Crates?

  • Crates: Paquets de codi Rust. Poden ser biblioteques o executables.
  • Crates Externs: Crates que no formen part de la biblioteca estàndard i que estan allotjats en repositoris com crates.io.

Trobar Crates

Per trobar crates externs, pots utilitzar el lloc web crates.io. És el registre oficial de paquets de Rust on pots cercar crates per nom, categoria o paraules clau.

Afegir Crates Externs al Teu Projecte

Per utilitzar un crate extern, has d'afegir-lo al teu fitxer Cargo.toml. Aquest fitxer és on especifiques les dependències del teu projecte.

Exemple

Suposem que vols utilitzar el crate rand per generar nombres aleatoris. Aquí tens com afegir-lo al teu projecte:

  1. Obre el teu fitxer Cargo.toml.
  2. Afegeix el crate sota la secció [dependencies].
[dependencies]
rand = "0.8.4"

Això indica a Cargo (el gestor de paquets de Rust) que descarregui i inclogui la versió 0.8.4 del crate rand en el teu projecte.

Utilitzar Crates en el Teu Codi

Un cop hagis afegit el crate al teu Cargo.toml, pots començar a utilitzar-lo en el teu codi Rust important-lo amb la paraula clau use.

Exemple

Aquí tens un exemple senzill d'utilitzar el crate rand per generar un nombre aleatori:

// Importa el crate rand
use rand::Rng;

fn main() {
    // Crea un generador de nombres aleatoris
    let mut rng = rand::thread_rng();
    
    // Genera un nombre aleatori entre 1 i 10
    let n: u32 = rng.gen_range(1..11);
    
    println!("Nombre aleatori: {}", n);
}

Explicació

  • use rand::Rng;: Aquesta línia importa el trait Rng del crate rand, que proporciona mètodes per generar nombres aleatoris.
  • let mut rng = rand::thread_rng();: Això crea un generador de nombres aleatoris.
  • let n: u32 = rng.gen_range(1..11);: Això genera un nombre aleatori entre 1 i 10.
  • println!("Nombre aleatori: {}", n);: Això imprimeix el nombre aleatori generat.

Gestionar Versions de Crates

Quan especifiques una versió de crate en Cargo.toml, pots utilitzar diferents requisits de versió:

  • Versió Exacta: rand = "0.8.4" - Utilitza exactament la versió 0.8.4.
  • Requisits de Caret: rand = "^0.8.4" - Utilitza la versió compatible més recent (per exemple, 0.8.5, 0.8.6, però no 0.9.0).
  • Requisits de Tilde: rand = "~0.8.4" - Utilitza la versió de correcció més recent (per exemple, 0.8.4, 0.8.5, però no 0.9.0).

Exercicis

  1. Afegeix i Utilitza el Crate regex:

    • Afegeix el crate regex al teu Cargo.toml.
    • Escriu un programa que utilitzi el crate regex per trobar i imprimir totes les adreces de correu electrònic en una cadena donada.

  2. Petició HTTP amb reqwest:

    • Afegeix el crate reqwest al teu Cargo.toml.
    • Escriu un programa que faci una petició GET a https://api.github.com i imprimeixi la resposta.

Conclusió

Utilitzar crates externs en Rust et permet aprofitar el treball de la comunitat i evitar reinventar la roda. Entenent com trobar, afegir i utilitzar crates externs, pots millorar significativament la teva productivitat i la funcionalitat dels teus projectes Rust. Recorda sempre consultar la documentació dels crates que utilitzes per entendre les seves capacitats i limitacions.
