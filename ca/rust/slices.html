Introducció als Fragments

Els fragments són una vista d'una seqüència contigua d'elements en una col·lecció en lloc de la col·lecció mateixa. Són una referència a una porció d'un array o un vector, permetent treballar amb un subconjunt de la col·lecció sense copiar dades.

Conceptes Clau

  • Fragments: Una referència a una seqüència contigua d'elements en una col·lecció.
  • Fragments d'Array: Fragments que fan referència a una porció d'un array.
  • Fragments de Vector: Fragments que fan referència a una porció d'un vector.
  • Immutabilitat: Els fragments són immutables per defecte, però es poden crear fragments mutables.

Creació i Ús de Fragments

Fragments d'Array

Per crear un fragment d'un array, utilitzes un rang dins de claudàtors.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[1..4]; // Fragmentant des de l'índex 1 fins al 3 (4 és exclusiu)
    
    println!("Fragment: {:?}", slice); // Sortida: Fragment: [2, 3, 4]
}

Fragments de Vector

De manera similar, pots crear un fragment d'un vector.

fn main() {
    let vector = vec![10, 20, 30, 40, 50];
    let slice = &vector[2..]; // Fragmentant des de l'índex 2 fins al final
    
    println!("Fragment: {:?}", slice); // Sortida: Fragment: [30, 40, 50]
}

Fragments Mutables

Els fragments mutables permeten modificar els elements del fragment.

fn main() {
    let mut array = [1, 2, 3, 4, 5];
    let slice = &mut array[1..4];
    
    slice[0] = 10; // Modificant el primer element del fragment
    println!("Array modificat: {:?}", array); // Sortida: Array modificat: [1, 10, 3, 4, 5]
}

Fragments vs. Arrays i Vectors

| Característica | Array | Vector | Fragment | |----------------|----------------|----------------|----------------| | Mida Fixa | Sí | No | No | | Mida Dinàmica | No | Sí | No | | Propietat | Sí | Sí | No | | Mutabilitat | Sí | Sí | Sí (si és fragment mutable) | | Sintaxi | [T; N] | Vec<T> | &[T] o &mut [T] |

Operacions Comunes amb Fragments

Iterar sobre Fragments

Pots iterar sobre fragments utilitzant un bucle for.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[..];
    
    for element in slice {
        println!("{}", element);
    }
}

Fragmentar Cadenes

Els fragments també es poden utilitzar amb cadenes, però cal anar amb compte amb els límits de caràcters UTF-8.

fn main() {
    let s = String::from("hello");
    let slice = &s[0..2]; // Fragmentant els dos primers bytes
    
    println!("Fragment: {}", slice); // Sortida: Fragment: he
}

Conclusió

Els fragments en Rust proporcionen una manera poderosa de referenciar i treballar amb una porció d'una col·lecció sense copiar dades. Són versàtils, permetent referències tant immutables com mutables, i es poden utilitzar amb arrays, vectors i cadenes. Entendre els fragments és crucial per a la manipulació eficient de dades i la gestió de memòria en Rust.
