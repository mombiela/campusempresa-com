Introducció a les Cues

Les cues són estructures de dades lineals que segueixen el principi FIFO (First In, First Out), és a dir, el primer element en entrar és el primer en sortir. Són útils en situacions on es necessita processar elements en l'ordre en què van ser rebuts.

Conceptes Clau

  • FIFO (First In, First Out): El primer element agregat a la cua serà el primer en ser remogut.
  • Enqueue: Operació per agregar un element al final de la cua.
  • Dequeue: Operació per remoure el primer element de la cua.
  • Front: Operació per obtenir el primer element de la cua sense remoure'l.
  • Rear: Operació per obtenir l'últim element de la cua.

Tipus de Cues

Cua Simple

Una cua simple permet operacions d'inserció en un extrem (rear) i operacions d'eliminació en l'altre extrem (front).

Cua Circular

Una cua circular és una millora de la cua simple que connecta l'últim element de la cua amb el primer, formant un cercle. Això permet un ús eficient de l'espai.

Cua de Prioritat

Una cua de prioritat és una estructura on cada element té una prioritat associada. Els elements amb major prioritat són processats abans que els de menor prioritat.

Implementació de Cues en Python

Cua Simple

A continuació, es mostra una implementació bàsica d'una cua simple usant una llista en Python:

class Queue:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

    def enqueue(self, item):
        self.items.append(item)

    def dequeue(self):
        if not self.is_empty():
            return self.items.pop(0)
        else:
            raise IndexError("Dequeue from empty queue")

    def front(self):
        if not self.is_empty():
            return self.items[0]
        else:
            raise IndexError("Front from empty queue")

    def rear(self):
        if not self.is_empty():
            return self.items[-1]
        else:
            raise IndexError("Rear from empty queue")

    def size(self):
        return len(self.items)

# Exemple d'ús
q = Queue()
q.enqueue(1)
q.enqueue(2)
q.enqueue(3)
print(q.dequeue())  # Sortida: 1
print(q.front())    # Sortida: 2
print(q.rear())     # Sortida: 3
print(q.size())     # Sortida: 2

Cua Circular

La implementació d'una cua circular pot ser més eficient en termes d'ús de memòria:

class CircularQueue:
    def __init__(self, max_size):
        self.max_size = max_size
        self.queue = [None] * max_size
        self.front = -1
        self.rear = -1

    def is_empty(self):
        return self.front == -1

    def is_full(self):
        return (self.rear + 1) % self.max_size == self.front

    def enqueue(self, item):
        if self.is_full():
            raise OverflowError("Enqueue on full queue")
        elif self.is_empty():
            self.front = self.rear = 0
        else:
            self.rear = (self.rear + 1) % self.max_size
        self.queue[self.rear] = item

    def dequeue(self):
        if self.is_empty():
            raise IndexError("Dequeue from empty queue")
        elif self.front == self.rear:
            item = self.queue[self.front]
            self.front = self.rear = -1
        else:
            item = self.queue[self.front]
            self.front = (self.front + 1) % self.max_size
        return item

    def front_item(self):
        if self.is_empty():
            raise IndexError("Front from empty queue")
        return self.queue[self.front]

    def rear_item(self):
        if self.is_empty():
            raise IndexError("Rear from empty queue")
        return self.queue[self.rear]

# Exemple d'ús
cq = CircularQueue(3)
cq.enqueue(1)
cq.enqueue(2)
cq.enqueue(3)
print(cq.dequeue())  # Sortida: 1
cq.enqueue(4)
print(cq.front_item())  # Sortida: 2
print(cq.rear_item())   # Sortida: 4

Conclusió

Les cues són una estructura de dades fonamental que s'utilitza en diverses aplicacions, des de la gestió de tasques en sistemes operatius fins a la implementació d'algoritmes de cerca en grafs. Comprendre com implementar i utilitzar cues és essencial per a qualsevol professional de TI. En aquest tema, hem explorat les cues simples i circulars, proporcionant exemples clars i concisos per a la seva implementació en Python.
