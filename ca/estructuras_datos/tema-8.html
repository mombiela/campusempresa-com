Introducció

En aquesta secció del curs sobre Estructures de Dades, ens enfocarem en les llistes, una de les estructures de dades més fonamentals i versàtils en la programació. Les llistes permeten emmagatzemar col·leccions d'elements, que poden ser de qualsevol tipus de dada, i ofereixen una varietat d'operacions útils per manipular aquests elements.

Conceptes Clau

  • Definició de Llista: Una llista és una col·lecció ordenada d'elements que poden ser accedits pel seu índex.
  • Operacions Bàsiques: Inserció, eliminació, accés i cerca d'elements.
  • Tipus de Llistes: Llistes enllaçades, llistes dobles, llistes circulars.

Exercicis Pràctics

Exercici 1: Creació i Accés a Elements

Objectiu: Crear una llista i accedir als seus elements.

Instruccions:

  1. Crea una llista amb els números de l'1 al 5.
  2. Accedeix al primer i últim element de la llista.
  3. Imprimeix tots els elements de la llista.

Codi d'Exemple:

# Creació de la llista
meva_llista = [1, 2, 3, 4, 5]

# Accés al primer i últim element
primer_element = meva_llista[0]
últim_element = meva_llista[-1]

# Imprimir els elements
print("Primer element:", primer_element)
print("Últim element:", últim_element)
print("Tots els elements:", meva_llista)

Exercici 2: Modificació d'Elements

Objectiu: Modificar elements dins d'una llista.

Instruccions:

  1. Canvia el segon element de la llista a 10.
  2. Afegeix el número 6 al final de la llista.
  3. Elimina el tercer element de la llista.

Codi d'Exemple:

# Llista inicial
meva_llista = [1, 2, 3, 4, 5]

# Modificació del segon element
meva_llista[1] = 10

# Afegir un nou element al final
meva_llista.append(6)

# Eliminar el tercer element
del meva_llista[2]

# Imprimir la llista modificada
print("Llista modificada:", meva_llista)

Exercici 3: Iteració i Suma d'Elements

Objectiu: Iterar sobre una llista i calcular la suma dels seus elements.

Instruccions:

  1. Itera sobre cada element de la llista.
  2. Calcula la suma de tots els elements.

Codi d'Exemple:

# Llista inicial
meva_llista = [1, 2, 3, 4, 5]

# Inicialitzar suma
suma = 0

# Iterar i sumar elements
for element in meva_llista:
    suma += element

# Imprimir la suma
print("Suma dels elements:", suma)

Exercici 4: Llistes Anidades

Objectiu: Treballar amb llistes dins de llistes (llistes anidades).

Instruccions:

  1. Crea una llista que contingui tres llistes, cadascuna amb tres números.
  2. Accedeix al segon element de la tercera llista.
  3. Imprimeix tots els elements de la llista anidada.

Codi d'Exemple:

# Creació de la llista anidada
llista_anidada = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Accés al segon element de la tercera llista
element = llista_anidada[2][1]

# Imprimir l'element
print("Segon element de la tercera llista:", element)

# Imprimir tots els elements de la llista anidada
for subllista in llista_anidada:
    for item in subllista:
        print(item, end=' ')
    print()

Conclusió

Les llistes són una estructura de dades essencial en la programació, oferint flexibilitat i eficiència per emmagatzemar i manipular col·leccions d'elements. A través d'aquests exercicis pràctics, hem explorat com crear, modificar, iterar i treballar amb llistes anidades. Comprendre i dominar l'ús de llistes és fonamental per a qualsevol professional de TI, ja que són àmpliament utilitzades en diverses aplicacions i algorismes.
