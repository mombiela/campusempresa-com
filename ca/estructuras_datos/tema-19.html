Introducció a les Cues

Les cues són estructures de dades fonamentals que segueixen el principi FIFO (First In, First Out), on el primer element en entrar és el primer en sortir. Són àmpliament utilitzades en escenaris on es necessita gestionar tasques en ordre seqüencial, com en sistemes d'impressió, gestió de processos en sistemes operatius, i en algorismes de cerca en grafs.

Conceptes Clau

  • FIFO (First In, First Out): El primer element que s'insereix a la cua és el primer a ser remogut.
  • Operacions Bàsiques:
    • enqueue: Afegir un element al final de la cua.
    • dequeue: Remoure el primer element de la cua.
    • peek o front: Obtenir el primer element de la cua sense remoure'l.
    • isEmpty: Verificar si la cua està buida.
  • Implementacions Comunes:
    • Cues usant llistes enllaçades.
    • Cues usant arrays.

Exemples de Codi

Implementació d'una Cua usant Arrays

class Queue:
    def __init__(self):
        self.queue = []

    def enqueue(self, item):
        self.queue.append(item)
        print(f"Enqueued: {item}")

    def dequeue(self):
        if not self.isEmpty():
            item = self.queue.pop(0)
            print(f"Dequeued: {item}")
            return item
        else:
            print("Queue is empty")
            return None

    def peek(self):
        if not self.isEmpty():
            return self.queue[0]
        else:
            print("Queue is empty")
            return None

    def isEmpty(self):
        return len(self.queue) == 0

# Exemple d'ús
q = Queue()
q.enqueue(1)
q.enqueue(2)
q.enqueue(3)
print(f"Front element: {q.peek()}")
q.dequeue()
q.dequeue()
q.dequeue()
q.dequeue()  # Intentar dequeuing quan la cua està buida

Implementació d'una Cua usant Llistes Enllaçades

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class LinkedListQueue:
    def __init__(self):
        self.front = None
        self.rear = None

    def enqueue(self, item):
        new_node = Node(item)
        if self.rear is None:
            self.front = self.rear = new_node
        else:
            self.rear.next = new_node
            self.rear = new_node
        print(f"Enqueued: {item}")

    def dequeue(self):
        if self.isEmpty():
            print("Queue is empty")
            return None
        temp = self.front
        self.front = temp.next
        si la front és None:
            self.rear = None
        print(f"Dequeued: {temp.value}")
        return temp.value

    def peek(self):
        if not self.isEmpty():
            return self.front.value
        else:
            print("Queue is empty")
            return None

    def isEmpty(self):
        return self.front is None

# Exemple d'ús
llq = LinkedListQueue()
llq.enqueue(1)
llq.enqueue(2)
llq.enqueue(3)
print(f"Front element: {llq.peek()}")
llq.dequeue()
llq.dequeue()
llq.dequeue()
llq.dequeue()  # Intentar dequeuing quan la cua està buida

Exercicis Pràctics

  1. Implementar una Cua Circular:

    • Descripció: Implementa una cua circular usant arrays. Assegura't de manejar correctament els casos en què la cua estigui plena o buida.
    • Punts clau:
      • Ús de punters front i rear.
      • Maneig del wrap-around en l'array.

  2. Simulació d'un Sistema d'Impressió:

    • Descripció: Simula un sistema d'impressió on les tasques d'impressió s'encolen i es processen en ordre.
    • Punts clau:
      • Cada tasca d'impressió té un identificador únic i una durada.
      • Implementa una funció per afegir tasques i una altra per processar-les.

  3. Cua de Prioritat Bàsica:

    • Descripció: Implementa una cua de prioritat bàsica on cada element té una prioritat associada i els elements amb major prioritat es processen primer.
    • Punts clau:
      • Ús d'una estructura de dades adequada per manejar les prioritats (pot ser un heap o una llista ordenada).
      • Implementació de les operacions enqueue i dequeue amb prioritats.

Conclusió

Les cues són una estructura de dades essencial en la informàtica, amb aplicacions pràctiques en diversos camps. Comprendre com implementar i utilitzar cues és fonamental per a qualsevol professional de TI. A través dels exemples i exercicis proporcionats, els estudiants poden enfortir la seva comprensió i habilitats en el maneig de cues, preparant-se millor per afrontar problemes del món real que requereixin aquesta estructura de dades.
