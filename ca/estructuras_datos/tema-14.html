Introducció

En aquest tema, explorarem el concepte de cua, una estructura de dades fonamental en informàtica. Les cues s'utilitzen en diverses aplicacions, des de la gestió de tasques en sistemes operatius fins a la implementació d'algoritmes de cerca en intel·ligència artificial.

Definició de Cua

Una cua és una estructura de dades lineal que segueix el principi FIFO (First In, First Out), és a dir, el primer element en entrar és el primer en sortir. Això és similar a una fila en la vida real, on les persones s'atenen en l'ordre en què van arribar.

Característiques Principals

  • FIFO (First In, First Out): El primer element en ser afegit a la cua serà el primer en ser remogut.
  • Operacions Bàsiques:
    • Enqueue: Afegir un element al final de la cua.
    • Dequeue: Remoure el primer element de la cua.
    • Front: Obtenir el primer element de la cua sense remoure'l.
    • IsEmpty: Verificar si la cua està buida.
    • Size: Obtenir el nombre d'elements en la cua.

Implementació d'una Cua

Les cues poden implementar-se de diverses maneres, sent les més comunes mitjançant arrays o llistes enllaçades. A continuació, es presenta una implementació bàsica utilitzant una llista enllaçada en Python.

Exemple de Codi en Python

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class Queue:
    def __init__(self):
        self.front = None
        self.rear = None
        self.size = 0

    def is_empty(self):
        return self.size == 0

    def enqueue(self, value):
        new_node = Node(value)
        if self.rear:
            self.rear.next = new_node
        self.rear = new_node
        if not self.front:
            self.front = new_node
        self.size += 1

    def dequeue(self):
        if self.is_empty():
            raise IndexError("Dequeue from empty queue")
        value = self.front.value
        self.front = self.front.next
        if not self.front:
            self.rear = None
        self.size -= 1
        return value

    def get_front(self):
        if self.is_empty():
            raise IndexError("Front from empty queue")
        return self.front.value

    def get_size(self):
        return self.size

# Exemple d'ús
queue = Queue()
queue.enqueue(10)
queue.enqueue(20)
print(queue.dequeue())  # Output: 10
print(queue.get_front())  # Output: 20
print(queue.get_size())  # Output: 1

Explicació del Codi

  • Classe Node: Representa un node en la llista enllaçada, amb un valor i un punter al següent node.
  • Classe Queue: Implementa la cua amb mètodes per a les operacions bàsiques.
    • enqueue: Afegeix un nou node al final de la cua.
    • dequeue: Remou el node del front de la cua i en retorna el valor.
    • get_front: Retorna el valor del node al front de la cua sense remoure'l.
    • is_empty: Verifica si la cua està buida.
    • get_size: Retorna el nombre d'elements en la cua.

Comparació entre Implementacions

| Característica | Array | Llista Enllaçada | |---------------------|-----------------|------------------------------| | Complexitat Enqueue | O(1) | O(1) | | Complexitat Dequeue | O(n) | O(1) | | Ús de Memòria | Fix | Dinàmic | | Flexibilitat | Menor | Major |

Conclusió

Les cues són estructures de dades essencials que permeten gestionar elements en un ordre específic, seguint el principi FIFO. La seva implementació pot variar segons les necessitats específiques del problema, sent les llistes enllaçades una opció flexible i eficient per a moltes aplicacions. Comprendre com funcionen i com implementar-les és crucial per a qualsevol professional en l'àrea de TI.
