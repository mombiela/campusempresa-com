Introducció a les Llistes Circulars

Les llistes circulars són una variació de les llistes enllaçades en les quals l'últim node apunta de nou al primer node, formant un cicle. Aquesta estructura permet recórrer la llista de manera contínua sense necessitat d'arribar a un final explícit.

Característiques de les Llistes Circulars

  • Cicle Continu: L'últim node s'enllaça al primer node, creant un cicle.
  • Accés Seqüencial: Es pot recórrer la llista de manera seqüencial des de qualsevol node.
  • No hi ha Node Nul: A diferència de les llistes enllaçades simples, no hi ha un node final que apunti a null.

Tipus de Llistes Circulars

  • Llista Circular Simple: Cada node té un enllaç al següent node i l'últim node s'enllaça al primer.
  • Llista Circular Doble: Cada node té enllaços tant al següent node com al node anterior, i l'últim node s'enllaça al primer i viceversa.

Operacions Bàsiques

Inserció

La inserció en una llista circular pot realitzar-se en diverses posicions: al començament, al final o en una posició específica.

Exemple d'Inserció al Començament

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class CircularLinkedList:
    def __init__(self):
        self.head = None

    def insert_at_beginning(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            new_node.next = self.head
        else:
            temp = self.head
            while temp.next != self.head:
                temp = temp.next
            new_node.next = self.head
            temp.next = new_node
            self.head = new_node

# Exemple d'ús
cll = CircularLinkedList()
cll.insert_at_beginning(10)
cll.insert_at_beginning(20)

Eliminació

L'eliminació en una llista circular també pot realitzar-se en diverses posicions: al començament, al final o en una posició específica.

Exemple d'Eliminació al Començament

class CircularLinkedList:
    # ... (mètodes anteriors)

    def delete_at_beginning(self):
        if self.head:
            if self.head.next == self.head:
                self.head = None
            else:
                temp = self.head
                while temp.next != self.head:
                    temp = temp.next
                temp.next = self.head.next
                self.head = self.head.next

# Exemple d'ús
cll.delete_at_beginning()

Cerca

La cerca en una llista circular implica recórrer la llista des del node inicial fins a trobar el node desitjat o tornar al node inicial.

Exemple de Cerca

class CircularLinkedList:
    # ... (mètodes anteriors)

    def search(self, key):
        temp = self.head
        if self.head:
            while True:
                if temp.data == key:
                    return True
                temp = temp.next
                if temp == self.head:
                    break
        return False

# Exemple d'ús
print(cll.search(10))  # Output: True o False

Avantatges i Desavantatges

Avantatges

  • Recorregut Continu: Ideal per a aplicacions que requereixen un recorregut continu, com en sistemes operatius per a la gestió de processos.
  • No hi ha Node Nul: No hi ha necessitat de verificar si s'ha arribat al final de la llista.

Desavantatges

  • Complexitat en la Implementació: Més complexa d'implementar i mantenir que una llista enllaçada simple.
  • Possibles Cicles Infinit: Risc de cicles infinits si no es maneja adequadament.

Comparació amb Altres Llistes

| Característica | Llista Enllaçada Simple | Llista Enllaçada Doble | Llista Circular Simple | Llista Circular Doble | |------------------------|-------------------------|------------------------|------------------------|-----------------------| | Enllaç al Següent | Sí | Sí | Sí | Sí | | Enllaç a l'Anterior | No | Sí | No | Sí | | Últim Node a Primer | No | No | Sí | Sí | | Complexitat d'Inserció | Mitjana | Alta | Mitjana | Alta |

Conclusió

Les llistes circulars són una estructura de dades poderosa i flexible que permet un recorregut continu dels elements. Encara que la seva implementació pot ser més complexa que altres llistes enllaçades, ofereixen avantatges significatives en aplicacions específiques. És crucial entendre tant els seus beneficis com les seves limitacions per utilitzar-les de manera efectiva en el desenvolupament de programari.
