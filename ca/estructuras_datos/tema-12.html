Introducció

Les piles són una estructura de dades fonamental en informàtica, caracteritzades pel seu comportament LIFO (Last In, First Out). En aquest tema, explorarem diverses aplicacions pràctiques de les piles en el desenvolupament de programari i altres àrees de la informàtica.

Aplicacions Comunes de les Piles

1. Avaluació d'Expressions

Les piles s'utilitzen àmpliament en l'avaluació d'expressions aritmètiques i lògiques, especialment en notacions postfija (RPN) i prefija.

  • Notació Postfija (RPN): En aquesta notació, els operadors segueixen els seus operands. Les piles faciliten l'avaluació d'aquestes expressions.

    Exemple de codi en Python:

    def evaluate_postfix(expression):
    stack = []
    for char in expression:
        if char.isdigit():
            stack.append(int(char))
        else:
            b = stack.pop()
            a = stack.pop()
            si char == '+':
                stack.append(a + b)
            elif char == '-':
                stack.append(a - b)
            elif char == '*':
                stack.append(a * b)
            elif char == '/':
                stack.append(a / b)
    return stack.pop()

expression = "231*+9-"
print(evaluate_postfix(expression))  # Output: -4

2. Gestió de la Recursió

Les piles són fonamentals per implementar la recursió. El sistema de crides de funcions utilitza una pila per gestionar les crides i retorns de funcions.

  • Exemple de Recursió: La funció factorial és un exemple clàssic de recursió.

    Exemple de codi en Python:

    def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

3. Desfer i Refer en Aplicacions

Les piles s'utilitzen en aplicacions de programari per implementar funcionalitats de desfer (undo) i refer (redo).

  • Desfer i Refer: En realitzar una acció, es guarda en una pila de desfer. En desfer una acció, es mou a una pila de refer.

    Exemple de codi en Python:

    class UndoRedo:
    def __init__(self):
        self.undo_stack = []
        self.redo_stack = []

    def do_action(self, action):
        self.undo_stack.append(action)
        self.redo_stack.clear()

    def undo(self):
        if self.undo_stack:
            action = self.undo_stack.pop()
            self.redo_stack.append(action)
            return action
        return None

    def redo(self):
        if self.redo_stack:
            action = self.redo_stack.pop()
            self.undo_stack.append(action)
            return action
        return None

ur = UndoRedo()
ur.do_action("Acció 1")
ur.do_action("Acció 2")
print(ur.undo())  # Output: Acció 2
print(ur.redo())  # Output: Acció 2

4. Navegació en Pàgines Web

Els navegadors web utilitzen piles per gestionar la navegació cap endavant i cap enrere entre pàgines.

  • Navegació de Pàgines: En visitar una nova pàgina, es guarda en una pila d'historial. En retrocedir, es mou a una pila de pàgines futures.

    Exemple de codi en Python:

    class BrowserHistory:
    def __init__(self):
        self.history_stack = []
        self.future_stack = []

    def visit(self, url):
        self.history_stack.append(url)
        self.future_stack.clear()

    def back(self):
        if self.history_stack:
            url = self.history_stack.pop()
            self.future_stack.append(url)
            return url
        return None

    def forward(self):
        if self.future_stack:
            url = self.future_stack.pop()
            self.history_stack.append(url)
            return url
        return None

browser = BrowserHistory()
browser.visit("page1.com")
browser.visit("page2.com")
print(browser.back())  # Output: page2.com
print(browser.forward())  # Output: page2.com

Conclusió

Les piles són una estructura de dades versàtil i poderosa amb nombroses aplicacions en el desenvolupament de programari i altres àrees de la informàtica. Des de l'avaluació d'expressions fins a la implementació de funcionalitats de desfer i refer, les piles juguen un paper crucial en l'eficiència i funcionalitat de moltes aplicacions. Comprendre i saber implementar piles és essencial per a qualsevol professional de TI.
