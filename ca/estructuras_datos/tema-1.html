Introducció

Les estructures de dades són una forma fonamental d'organitzar, gestionar i emmagatzemar dades de manera eficient perquè puguin ser utilitzades de manera efectiva en diverses aplicacions. En l'àmbit de la informàtica i la programació, les estructures de dades són essencials per al disseny d'algorismes eficients i l'optimització del rendiment del programari.

Conceptes Clau

  • Definició: Una estructura de dades és una col·lecció de valors, la relació entre ells i les operacions que es poden realitzar sobre ells.
  • Importància: Permeten l'organització i manipulació eficient de grans quantitats de dades.
  • Tipus d'Estructures de Dades: Es poden classificar en diverses categories, com estructures de dades lineals i no lineals.

Tipus d'Estructures de Dades

Estructures de Dades Lineals

  • Arrays (Arranjaments): Una col·lecció d'elements, tots del mateix tipus, emmagatzemats en ubicacions de memòria contigües. 
    # Exemple d'un array en Python
array = [1, 2, 3, 4, 5]
print(array[2])  # Sortida: 3
  • Llistes Enllaçades: Una seqüència d'elements on cada element apunta al següent. 
    # Exemple bàsic d'una llista enllaçada en Python
class Node:
    def __init__(self, dada):
        self.dada = dada
        self.seguent = None

node1 = Node(1)
node2 = Node(2)
node1.seguent = node2
print(node1.seguent.dada)  # Sortida: 2
  • Piles (Stacks): Una col·lecció d'elements que segueix el principi LIFO (Last In, First Out). 
    # Exemple d'una pila en Python usant una llista
pila = []
pila.append(1)
pila.append(2)
print(pila.pop())  # Sortida: 2
  • Cues (Queues): Una col·lecció d'elements que segueix el principi FIFO (First In, First Out). 
    # Exemple d'una cua en Python usant una llista
from collections import deque
cua = deque()
cua.append(1)
cua.append(2)
print(cua.popleft())  # Sortida: 1

Estructures de Dades No Lineals

  • Arbres: Una estructura jeràrquica on cada node té un valor i una llista de referències a altres nodes (els seus fills). 
    # Exemple bàsic d'un node d'arbre en Python
class NodeArbre:
    def __init__(self, dada):
        self.dada = dada
        self.fills = []

arrel = NodeArbre(1)
fill1 = NodeArbre(2)
arrel.fills.append(fill1)
print(arrel.fills[0].dada)  # Sortida: 2
  • Grafs: Una col·lecció de nodes i arestes on els nodes poden estar connectats de manera arbitrària. 
    # Exemple bàsic d'un graf en Python usant un diccionari
graf = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}
print(graf['A'])  # Sortida: ['B', 'C']

Comparació d'Estructures de Dades

| Estructura de Dades | Accés | Inserció | Eliminació | Cerca | |---------------------|-------|----------|------------|-------| | Array | O(1) | O(n) | O(n) | O(n) | | Llista Enllaçada | O(n) | O(1) | O(1) | O(n) | | Pila | O(n) | O(1) | O(1) | O(n) | | Cua | O(n) | O(1) | O(1) | O(n) | | Arbre Binari | O(log n)| O(log n)| O(log n) | O(log n)| | Graf | O(n) | O(1) | O(1) | O(n) |

Conclusió

Les estructures de dades són fonamentals per al desenvolupament de programari eficient i eficaç. Comprendre les diferents estructures de dades i les seves característiques permet als professionals de TI seleccionar l'estructura adequada per a cada problema, optimitzant així el rendiment i l'eficiència de les seves aplicacions. En els propers temes, aprofundirem en cadascuna d'aquestes estructures, explorant les seves implementacions, avantatges i desavantatges en detall.
