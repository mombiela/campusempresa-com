Introducció

Un graf és una estructura de dades fonamental en la informàtica i les matemàtiques, utilitzada per modelar relacions entre objectes. Els grafs són àmpliament utilitzats en diverses aplicacions, des de xarxes socials fins a algorismes de cerca i optimització.

Definició de Graf

Un graf \( G \) es defineix com un parell \( (V, E) \), on:

  • \( V \) és un conjunt de vèrtexs (o nodes).
  • \( E \) és un conjunt d'arestes (o enllaços) que connecten parells de vèrtexs.

Tipus de Grafs

  • Graf no dirigit: Les arestes no tenen una direcció específica. Si hi ha una aresta entre els vèrtexs \( u \) i \( v \), es pot recórrer en ambdós sentits.
  • Graf dirigit (dígraf): Les arestes tenen una direcció específica. Una aresta del vèrtex \( u \) al vèrtex \( v \) es denota com \( (u, v) \).

Representació de Grafs

Els grafs poden representar-se de diverses maneres, les més comunes són:

  • Llista d'Adjacència: Cada vèrtex té una llista dels vèrtexs als quals està connectat.
  • Matriu d'Adjacència: Una matriu \( n \times n \) on \( n \) és el nombre de vèrtexs. L'entrada \( (i, j) \) és 1 si hi ha una aresta del vèrtex \( i \) al vèrtex \( j \), i 0 en cas contrari.

Exemple de Llista d'Adjacència

Vèrtexs: {A, B, C, D}
Arestes: {(A, B), (A, C), (B, D), (C, D)}

Llista d'Adjacència:
A: B, C
B: D
C: D
D:

Exemple de Matriu d'Adjacència

Vèrtexs: {A, B, C, D}
Arestes: {(A, B), (A, C), (B, D), (C, D)}

Matriu d'Adjacència:
    A B C D
A [ 0 1 1 0 ]
B [ 0 0 0 1 ]
C [ 0 0 0 1 ]
D [ 0 0 0 0 ]

Propietats dels Grafs

  • Grau d'un vèrtex: Nombre d'arestes incidents a un vèrtex. En grafs dirigits, es distingeix entre grau d'entrada i grau de sortida.
  • Camí: Seqüència de vèrtexs on cada parell consecutiu està connectat per una aresta.
  • Cicle: Camí que comença i acaba en el mateix vèrtex.
  • Connectivitat: Un graf és connex si hi ha un camí entre qualsevol parell de vèrtexs.

Exemples de Codi

Representació d'un Graf en Python usant Llista d'Adjacència

class Graf:
    def __init__(self):
        self.graf = {}

    def afegir_vèrtex(self, vèrtex):
        if vèrtex not in self.graf:
            self.graf[vèrtex] = []

    def afegir_aresta(self, vèrtex1, vèrtex2):
        if vèrtex1 in self.graf and vèrtex2 in self.graf:
            self.graf[vèrtex1].append(vèrtex2)
            self.graf[vèrtex2].append(vèrtex1)  # Per a grafs no dirigits

    def mostrar_graf(self):
        for vèrtex in self.graf:
            print(f"{vèrtex}: {', '.join(self.graf[vèrtex])}")

# Exemple d'ús
g = Graf()
g.afegir_vèrtex('A')
g.afegir_vèrtex('B')
g.afegir_vèrtex('C')
g.afegir_vèrtex('D')
g.afegir_aresta('A', 'B')
g.afegir_aresta('A', 'C')
g.afegir_aresta('B', 'D')
g.afegir_aresta('C', 'D')
g.mostrar_graf()

Representació d'un Graf en Python usant Matriu d'Adjacència

class GrafMatriu:
    def __init__(self, num_vèrtexs):
        self.num_vèrtexs = num_vèrtexs
        self.matriu = [[0] * num_vèrtexs for _ in range(num_vèrtexs)]

    def afegir_aresta(self, vèrtex1, vèrtex2):
        self.matriu[vèrtex1][vèrtex2] = 1
        self.matriu[vèrtex2][vèrtex1] = 1  # Per a grafs no dirigits

    def mostrar_matriu(self):
        for fila in self.matriu:
            print(fila)

# Exemple d'ús
g = GrafMatriu(4)
g.afegir_aresta(0, 1)
g.afegir_aresta(0, 2)
g.afegir_aresta(1, 3)
g.afegir_aresta(2, 3)
g.mostrar_matriu()

Conclusió

Els grafs són una eina poderosa per modelar i resoldre problemes complexos en informàtica. Comprendre la seva estructura i representació és crucial per a qualsevol professional de TI. En aquest tema, hem cobert les definicions bàsiques, tipus, representacions i propietats dels grafs, així com exemples pràctics d'implementació en Python.
