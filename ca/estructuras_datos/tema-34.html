Introducció

En aquesta secció del curs sobre Estructures de Dades, ens enfocarem en els grafs, una estructura fonamental en la informàtica i les ciències de la computació. Els grafs s'utilitzen per modelar relacions i connexions entre diferents entitats. A través d'exercicis pràctics, aprendrem a implementar i manipular grafs en diferents contextos.

Conceptes Clau

  • Graf: Conjunt de nodes (o vèrtexs) i arestes (o enllaços) que connecten parells de nodes.
  • Tipus de Grafs:
    • Dirigit: Les arestes tenen una direcció.
    • No dirigit: Les arestes no tenen direcció.
    • Ponderat: Les arestes tenen un pes associat.
    • No ponderat: Les arestes no tenen pes.
  • Representacions de Grafs:
    • Llista d'Adjacència: Utilitza llistes per emmagatzemar els nodes adjacents.
    • Matriu d'Adjacència: Utilitza una matriu per representar les connexions entre nodes.

Exercici 1: Implementació d'un Graf No Dirigit

Objectiu

Implementar un graf no dirigit utilitzant una llista d'adjacència.

Instruccions

  1. Crear una classe Graf.
  2. Implementar mètodes per afegir nodes i arestes.
  3. Implementar un mètode per imprimir el graf.

Exemple de Codi

class Graf:
    def __init__(self):
        self.graf = {}

    def afegir_node(self, node):
        si node no està en self.graf:
            self.graf[node] = []

    def afegir_aresta(self, node1, node2):
        si node1 està en self.graf i node2 està en self.graf:
            self.graf[node1].append(node2)
            self.graf[node2].append(node1)

    def imprimir_graf(self):
        per node en self.graf:
            print(f"{node}: {self.graf[node]}")

# Exemple d'ús
g = Graf()
g.afegir_node("A")
g.afegir_node("B")
g.afegir_node("C")
g.afegir_aresta("A", "B")
g.afegir_aresta("A", "C")
g.imprimir_graf()

Explicació

  • Classe Graf: Defineix l'estructura del graf.
  • Mètode afegir_node: Afegeix un node al graf.
  • Mètode afegir_aresta: Afegeix una aresta entre dos nodes.
  • Mètode imprimir_graf: Imprimeix la llista d'adjacència del graf.

Exercici 2: Implementació d'un Graf Dirigit Ponderat

Objectiu

Implementar un graf dirigit i ponderat utilitzant una llista d'adjacència.

Instruccions

  1. Crear una classe GrafDirigitPonderat.
  2. Implementar mètodes per afegir nodes i arestes amb pesos.
  3. Implementar un mètode per imprimir el graf.

Exemple de Codi

class GrafDirigitPonderat:
    def __init__(self):
        self.graf = {}

    def afegir_node(self, node):
        si node no està en self.graf:
            self.graf[node] = []

    def afegir_aresta(self, node1, node2, pes):
        si node1 està en self.graf i node2 està en self.graf:
            self.graf[node1].append((node2, pes))

    def imprimir_graf(self):
        per node en self.graf:
            print(f"{node}: {self.graf[node]}")

# Exemple d'ús
g = GrafDirigitPonderat()
g.afegir_node("A")
g.afegir_node("B")
g.afegir_node("C")
g.afegir_aresta("A", "B", 5)
g.afegir_aresta("A", "C", 10)
g.imprimir_graf()

Explicació

  • Classe GrafDirigitPonderat: Defineix l'estructura del graf dirigit i ponderat.
  • Mètode afegir_node: Afegeix un node al graf.
  • Mètode afegir_aresta: Afegeix una aresta amb pes entre dos nodes.
  • Mètode imprimir_graf: Imprimeix la llista d'adjacència del graf amb pesos.

Exercici 3: Cerca en Grafs

Objectiu

Implementar algorismes de cerca en grafs: BFS (Breadth-First Search) i DFS (Depth-First Search).

Instruccions

  1. Crear mètodes per BFS i DFS en la classe Graf.
  2. Implementar la lògica dels algorismes de cerca.

Exemple de Codi

class Graf:
    def __init__(self):
        self.graf = {}

    def afegir_node(self, node):
        si node no està en self.graf:
            self.graf[node] = []

    def afegir_aresta(self, node1, node2):
        si node1 està en self.graf i node2 està en self.graf:
            self.graf[node1].append(node2)
            self.graf[node2].append(node1)

    def bfs(self, inici):
        visitats = set()
        cua = [inici]

        mentre cua:
            node = cua.pop(0)
            si node no està en visitats:
                print(node, end=" ")
                visitats.add(node)
                cua.extend(self.graf[node])

    def dfs(self, inici, visitats=None):
        si visitats és None:
            visitats = set()
        visitats.add(inici)
        print(inici, end=" ")

        per veí en self.graf[inici]:
            si veí no està en visitats:
                self.dfs(veí, visitats)

# Exemple d'ús
g = Graf()
g.afegir_node("A")
g.afegir_node("B")
g.afegir_node("C")
g.afegir_node("D")
g.afegir_aresta("A", "B")
g.afegir_aresta("A", "C")
g.afegir_aresta("B", "D")
print("BFS:")
g.bfs("A")
print("\nDFS:")
g.dfs("A")

Explicació

  • Mètode bfs: Implementa la cerca en amplada.
  • Mètode dfs: Implementa la cerca en profunditat.
