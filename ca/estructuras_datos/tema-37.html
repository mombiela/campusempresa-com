Introducció

Les estructures de dades són fonamentals en la informàtica i s'utilitzen per organitzar i emmagatzemar dades de manera eficient. En aquest tema, explorarem com les estructures de dades s'apliquen en situacions del món real, proporcionant exemples clars i exercicis pràctics per ajudar els professionals de TI a comprendre la seva importància i aplicació.

Conceptes Clau

  • Estructures de Dades: Formes d'organitzar i emmagatzemar dades per facilitar-ne l'accés i modificació.
  • Tipus Comuns: Arrays, llistes enllaçades, piles, cues, arbres, grafs, taules hash.
  • Aplicacions Pràctiques: Ús d'estructures de dades en aplicacions del món real com bases de dades, sistemes operatius, xarxes i més.

Exemples d'Aplicació en la Vida Real

Arrays

Els arrays són una estructura de dades simple però poderosa. S'utilitzen per emmagatzemar elements del mateix tipus en una seqüència contigua de memòria.

Exemple: Sistema de Reserves de Vols

En un sistema de reserves de vols, els arrays poden utilitzar-se per emmagatzemar la llista de seients disponibles en un avió.

# Array que representa els seients en un avió
seients = [True, True, False, True, False]  # True indica que el seient està disponible

# Reservar un seient
def reservar_seient(seients, numero_seient):
    if seients[numero_seient]:
        seients[numero_seient] = False
        return "Seient reservat amb èxit"
    else:
        return "Seient ja està ocupat"

print(reservar_seient(seients, 2))  # Sortida: Seient ja està ocupat
print(reservar_seient(seients, 1))  # Sortida: Seient reservat amb èxit

Llistes Enllaçades

Les llistes enllaçades són una estructura de dades on cada element (node) conté un valor i una referència al següent node en la seqüència.

Exemple: Sistema de Gestió de Tasques

En un sistema de gestió de tasques, les llistes enllaçades poden utilitzar-se per mantenir una llista de tasques pendents.

class Node:
    def __init__(self, dada):
        self.dada = dada
        self.seguent = None

class LlistaEnllaçada:
    def __init__(self):
        self.cap = None

    def afegir_tasca(self, tasca):
        nou_node = Node(tasca)
        nou_node.seguent = self.cap
        self.cap = nou_node

    def mostrar_tasques(self):
        actual = self.cap
        while actual:
            print(actual.dada)
            actual = actual.seguent

# Crear una llista de tasques
llista_tasques = LlistaEnllaçada()
llista_tasques.afegir_tasca("Comprar llet")
llista_tasques.afegir_tasca("Enviar correu")
llista_tasques.afegir_tasca("Trucar al client")

llista_tasques.mostrar_tasques()
# Sortida:
# Trucar al client
# Enviar correu
# Comprar llet

Piles

Les piles són estructures de dades que segueixen el principi LIFO (Last In, First Out). S'utilitzen comunament en situacions on l'últim element afegit és el primer en ser remogut.

Exemple: Navegador Web

En un navegador web, les piles s'utilitzen per implementar la funcionalitat de "enrere" i "endavant".

class Pila:
    def __init__(self):
        self.items = []

    def esta_buida(self):
        return self.items == []

    def apilar(self, item):
        self.items.append(item)

    def desapilar(self):
        return self.items.pop()

# Crear una pila d'historial de navegació
historial = Pila()
historial.apilar("pagina1.com")
historial.apilar("pagina2.com")
historial.apilar("pagina3.com")

print(historial.desapilar())  # Sortida: pagina3.com
print(historial.desapilar())  # Sortida: pagina2.com

Cues

Les cues són estructures de dades que segueixen el principi FIFO (First In, First Out). S'utilitzen en situacions on el primer element afegit és el primer en ser remogut.

Exemple: Sistema d'Impressió

En un sistema d'impressió, les cues s'utilitzen per gestionar les tasques d'impressió en l'ordre en què van ser rebudes.

class Cua:
    def __init__(self):
        self.items = []

    def esta_buida(self):
        return self.items == []

    def encolar(self, item):
        self.items.insert(0, item)

    def desencolar(self):
        return self.items.pop()

# Crear una cua de tasques d'impressió
cua_impressio = Cua()
cua_impressio.encolar("Document1")
cua_impressio.encolar("Document2")
cua_impressio.encolar("Document3")

print(cua_impressio.desencolar())  # Sortida: Document1
print(cua_impressio.desencolar())  # Sortida: Document2

Conclusió

Les estructures de dades són essencials per al desenvolupament d'aplicacions eficients i efectives. En comprendre com s'apliquen en situacions del món real, els professionals de TI poden dissenyar i optimitzar sistemes que gestionin dades de manera més eficient. A través d'exemples pràctics i exercicis, hem explorat com les estructures de dades com arrays, llistes enllaçades, piles i cues s'utilitzen en aplicacions quotidianes.
