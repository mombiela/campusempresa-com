Introducció

En l'àmbit de les estructures de dades, un arbre és una estructura jeràrquica que s'utilitza per representar relacions entre elements de manera eficient. Els arbres són fonamentals en la informàtica i s'utilitzen en una varietat d'aplicacions, des de bases de dades fins a algorismes de cerca i classificació.

Definició d'Arbre

Un arbre és una estructura de dades no lineal composta per nodes, on cada node té un valor i una llista de referències a altres nodes (els seus fills). El node superior es diu arrel, i cada node pot tenir zero o més nodes fills. Els nodes sense fills es diuen fulles.

Característiques d'un Arbre

  • Arrel: El node superior de l'arbre.
  • Node: Cada element de l'arbre.
  • Fill: Un node que descendeix d'un altre node.
  • Pare: Un node que té un o més nodes fills.
  • Fulla: Un node que no té fills.
  • Alçada de l'arbre: La longitud del camí més llarg des de l'arrel fins a una fulla.
  • Profunditat d'un node: La longitud del camí des de l'arrel fins a aquest node.

Tipus d'Arbres

Existeixen diversos tipus d'arbres, cadascun amb característiques i usos específics:

  • Arbre Binari: Cada node té com a màxim dos fills.
  • Arbre Binari de Cerca (BST): Un arbre binari on el valor de cada node és més gran que els valors de tots els nodes en el seu subarbre esquerre i menor que els valors de tots els nodes en el seu subarbre dret.
  • Arbre AVL: Un arbre binari de cerca auto-balancejat.
  • Arbre B: Un arbre auto-balancejat que manté dades ordenades i permet cerques, insercions i eliminacions en temps logarítmic.

Exemple d'Arbre Binari

A continuació, es mostra un exemple d'un arbre binari i la seva representació en codi Python:

class Node:
    def __init__(self, valor):
        self.valor = valor
        self.esquerra = None
        self.dreta = None

class ArbreBinari:
    def __init__(self):
        self.arrel = None

    def inserir(self, valor):
        if self.arrel is None:
            self.arrel = Node(valor)
        else:
            self._inserir_recursiu(valor, self.arrel)

    def _inserir_recursiu(self, valor, node_actual):
        if valor < node_actual.valor:
            if node_actual.esquerra is None:
                node_actual.esquerra = Node(valor)
            else:
                self._inserir_recursiu(valor, node_actual.esquerra)
        else:
            if node_actual.dreta is None:
                node_actual.dreta = Node(valor)
            else:
                self._inserir_recursiu(valor, node_actual.dreta)

# Exemple d'ús
arbre = ArbreBinari()
arbre.inserir(10)
arbre.inserir(5)
arbre.inserir(15)

En aquest exemple, es defineix una classe Node per representar cada node de l'arbre i una classe ArbreBinari per manejar la inserció de nous nodes.

Comparació de Tipus d'Arbres

| Tipus d'Arbre | Característiques | Usos Comuns | |--------------------------|----------------------------------------------------------------------------------|----------------------------------------------| | Arbre Binari | Cada node té com a màxim dos fills. | Expressions aritmètiques, estructures de dades.| | Arbre Binari de Cerca | Els nodes estan ordenats de manera que cada node és més gran que el seu subarbre esquerre i menor que el seu subarbre dret. | Cerca i ordenació de dades. | | Arbre AVL | Un arbre binari de cerca auto-balancejat. | Aplicacions on es requereix balanceig dinàmic. | | Arbre B | Un arbre auto-balancejat que manté dades ordenades. | Bases de dades i sistemes de fitxers. |

Conclusió

Els arbres són una estructura de dades essencial en la informàtica, proporcionant una manera eficient d'organitzar i gestionar dades jeràrquiques. Comprendre els diferents tipus d'arbres i les seves aplicacions és crucial per a qualsevol professional de TI, ja que aquests conceptes s'utilitzen en una àmplia varietat de contextos, des d'algorismes de cerca fins a la gestió de bases de dades.
