Introducció

Les llistes doblement enllaçades són una estructura de dades fonamental en la programació i la informàtica. A diferència de les llistes simplement enllaçades, en les quals cada node apunta només al següent node, en una llista doblement enllaçada, cada node conté dos punters: un al node següent i un altre al node anterior. Això permet una navegació bidireccional a través de la llista, la qual cosa pot ser útil en diverses aplicacions.

Conceptes Clau

  • Node: La unitat bàsica d'una llista doblement enllaçada. Cada node conté un valor i dos punters: un al següent node i un altre al node anterior.
  • Cap (Head): El primer node de la llista.
  • Cua (Tail): L'últim node de la llista.
  • Punter Següent (Next): Un punter que apunta al següent node en la llista.
  • Punter Anterior (Prev): Un punter que apunta al node anterior en la llista.

Estructura d'un Node

En una llista doblement enllaçada, cada node té la següent estructura:

class Node:
    def __init__(self, valor):
        self.valor = valor
        self.seguent = None  # Punter al següent node
        self.anterior = None   # Punter al node anterior

Operacions Bàsiques

Inserció

La inserció en una llista doblement enllaçada pot realitzar-se en diverses posicions: al començament, al final o en una posició específica.

Inserció al Començament

def inserir_al_comencament(self, valor):
    nou_node = Node(valor)
    if self.cap is None:
        self.cap = self.cua = nou_node
    else:
        nou_node.seguent = self.cap
        self.cap.anterior = nou_node
        self.cap = nou_node

Inserció al Final

def inserir_al_final(self, valor):
    nou_node = Node(valor)
    if self.cua is None:
        self.cap = self.cua = nou_node
    else:
        nou_node.anterior = self.cua
        self.cua.seguent = nou_node
        self.cua = nou_node

Eliminació

L'eliminació també pot realitzar-se en diverses posicions: al començament, al final o en una posició específica.

Eliminació al Començament

def eliminar_al_comencament(self):
    if self.cap is None:
        return
    if self.cap == self.cua:
        self.cap = self.cua = None
    else:
        self.cap = self.cap.seguent
        self.cap.anterior = None

Eliminació al Final

def eliminar_al_final(self):
    if self.cua is None:
        return
    if self.cap == self.cua:
        self.cap = self.cua = None
    else:
        self.cua = self.cua.anterior
        self.cua.seguent = None

| Característica | Llista Simplement Enllaçada | Llista Doblemen Enllaçada | |-------------------------------|-----------------------------|---------------------------| | Punters per node | 1 | 2 | | Navegació | Unidireccional | Bidireccional | | Complexitat d'inserció/eliminació | O(n) en el pitjor cas | O(1) en el millor cas | | Ús de memòria | Menor | Major |

Conclusió

Les llistes doblement enllaçades són una estructura de dades versàtil que permet una navegació bidireccional i operacions eficients d'inserció i eliminació. Encara que consumeixen més memòria que les llistes simplement enllaçades a causa dels punters addicionals, la seva flexibilitat i eficiència en certes operacions les fan una opció valuosa en moltes aplicacions.
