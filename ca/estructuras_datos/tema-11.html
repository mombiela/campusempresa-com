Introducció a les Piles

Una pila és una estructura de dades lineal que segueix el principi LIFO (Last In, First Out), la qual cosa significa que l'últim element en entrar és el primer en sortir. Les piles són fonamentals en la informàtica i s'utilitzen en diverses aplicacions, com la gestió de memòria, l'avaluació d'expressions i la implementació d'algoritmes de retrocés.

Operacions Fonamentals

Les operacions bàsiques que es poden realitzar en una pila són:

  • Push: Afegir un element a la pila.
  • Pop: Eliminar l'element superior de la pila.
  • Peek (o Top): Obtenir l'element superior de la pila sense eliminar-lo.
  • isEmpty: Verificar si la pila està buida.
  • Size: Obtenir el nombre d'elements en la pila.

Push

L'operació push afegeix un nou element a la part superior de la pila.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

# Exemple d'ús
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.items)  # Sortida: [1, 2]

Pop

L'operació pop elimina i retorna l'element superior de la pila.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "La pila està buida"

    def is_empty(self):
        return len(self.items) == 0

# Exemple d'ús
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.pop())  # Sortida: 2
print(pila.items)  # Sortida: [1]

Peek (o Top)

L'operació peek retorna l'element superior de la pila sense eliminar-lo.

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "La pila està buida"

    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        else:
            return "La pila està buida"

    def is_empty(self):
        return len(self.items) == 0

# Exemple d'ús
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.peek())  # Sortida: 2
print(pila.items)   # Sortida: [1, 2]

isEmpty

L'operació isEmpty verifica si la pila està buida.

class Pila:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

# Exemple d'ús
pila = Pila()
print(pila.is_empty())  # Sortida: True
pila.push(1)
print(pila.is_empty())  # Sortida: False

Size

L'operació size retorna el nombre d'elements en la pila.

class Pila:
    def __init__(self):
        self.items = []

    def size(self):
        return len(self.items)

# Exemple d'ús
pila = Pila()
pila.push(1)
pila.push(2)
print(pila.size())  # Sortida: 2

Comparació d'Operacions

| Operació | Descripció | Complexitat Temporal | |----------|-------------------------------------|----------------------| | Push | Afegir un element a la pila | O(1) | | Pop | Eliminar l'element superior | O(1) | | Peek | Obtenir l'element superior | O(1) | | isEmpty | Verificar si la pila està buida | O(1) | | Size | Obtenir el nombre d'elements | O(1) |

Conclusió

Les piles són una estructura de dades fonamental en la informàtica, i comprendre les seves operacions bàsiques és crucial per a qualsevol professional de TI. Les operacions push, pop, peek, isEmpty i size són essencials per manipular piles de manera eficient. Practicar amb exemples de codi i exercicis ajudarà a solidificar aquests conceptes i preparar-se per a aplicacions més avançades.
