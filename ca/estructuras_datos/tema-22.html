Introducció

Un Arbre Binari de Cerca (ABC) és una estructura de dades jeràrquica que facilita la cerca, inserció i eliminació d'elements. Cada node en un ABC té com a màxim dos fills, i s'organitza de manera que per a qualsevol node donat, tots els valors en el subarbre esquerre són menors i tots els valors en el subarbre dret són majors.

Conceptes Clau

  • Node: Element de l'arbre que conté un valor i referències als seus nodes fills.
  • Arrel: Node superior de l'arbre.
  • Fill Esquerre: Node el valor del qual és menor que el node pare.
  • Fill Dret: Node el valor del qual és major que el node pare.
  • Fulla: Node que no té fills.
  • Alçada de l'Arbre: Longitud del camí més llarg des de l'arrel fins a una fulla.

Operacions Bàsiques

Cerca

La cerca en un ABC es realitza comparant el valor cercat amb el valor del node actual i movent-se a l'esquerra o dreta segons correspongui.

class Node:
    def __init__(self, valor):
        self.valor = valor
        self.esquerre = None
        self.dret = None

def cercar(node, valor):
    si node és None o node.valor == valor:
        return node
    si valor < node.valor:
        return cercar(node.esquerre, valor)
    else:
        return cercar(node.dret, valor)

Inserció

Per inserir un nou valor, se segueix el mateix principi de comparació utilitzat en la cerca fins a trobar una posició buida.

def inserir(node, valor):
    si node és None:
        return Node(valor)
    si valor < node.valor:
        node.esquerre = inserir(node.esquerre, valor)
    else:
        node.dret = inserir(node.dret, valor)
    return node

Eliminació

Eliminar un node en un ABC pot ser més complex, ja que hi ha tres casos a considerar:

  1. El node és una fulla.
  2. El node té un sol fill.
  3. El node té dos fills.
def eliminar(node, valor):
    si node és None:
        return node
    si valor < node.valor:
        node.esquerre = eliminar(node.esquerre, valor)
    elif valor > node.valor:
        node.dret = eliminar(node.dret, valor)
    else:
        si node.esquerre és None:
            return node.dret
        elif node.dret és None:
            return node.esquerre
        temp = trobar_minim(node.dret)
        node.valor = temp.valor
        node.dret = eliminar(node.dret, temp.valor)
    return node

def trobar_minim(node):
    actual = node
    mentre actual.esquerre no és None:
        actual = actual.esquerre
    return actual

Exemple Pràctic

A continuació, es mostra un exemple de com construir un ABC i realitzar operacions bàsiques.

# Crear l'arbre
arrel = Node(50)
inserir(arrel, 30)
inserir(arrel, 20)
inserir(arrel, 40)
inserir(arrel, 70)
inserir(arrel, 60)
inserir(arrel, 80)

# Cercar un valor
resultat = cercar(arrel, 40)
print("Valor trobat:", resultat.valor si resultat else "No trobat")

# Eliminar un valor
arrel = eliminar(arrel, 20)

Conclusió

Els Arbres Binàris de Cerca són una estructura fonamental en la informàtica que permet realitzar operacions eficients de cerca, inserció i eliminació. Comprendre el seu funcionament i com implementar-los és essencial per a qualsevol professional de TI que treballi amb estructures de dades i algorismes.
