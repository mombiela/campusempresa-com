Introducció

En aquest tema, explorarem dos algorismes fonamentals per al recorregut de grafs: BFS (Breadth-First Search) i DFS (Depth-First Search). Aquests algorismes són essencials per resoldre problemes en diverses àrees de la informàtica, com la cerca de camins, la detecció de cicles i la connectivitat de grafs.

Breadth-First Search (BFS)

BFS és un algorisme de cerca en grafs que explora tots els nodes a la mateixa distància del node inicial abans de passar als nodes a major distància. És especialment útil per trobar el camí més curt en grafs no ponderats.

Conceptes Clau

  • Cua (Queue): Estructura de dades utilitzada per mantenir l'ordre dels nodes a visitar.
  • Nivells: BFS explora el graf nivell per nivell.

Algorisme

  1. Inicialitzar una cua i afegir el node inicial.
  2. Marcar el node inicial com a visitat.
  3. Mentre la cua no estigui buida:
    • Treure un node de la cua.
    • Processar el node.
    • Afegir tots els nodes adjacents no visitats a la cua i marcar-los com a visitats.

Exemple de Codi

from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)
    
    while queue:
        node = queue.popleft()
        print(node, end=" ")
        
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

# Exemple d'ús
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

bfs(graph, 'A')

Depth-First Search (DFS)

DFS és un algorisme de cerca en grafs que explora tan lluny com sigui possible al llarg de cada branca abans de retrocedir. És útil per a tasques com la detecció de cicles i la generació d'arbres d'expansió.

Conceptes Clau

  • Pila (Stack): Estructura de dades utilitzada per mantenir l'ordre dels nodes a visitar.
  • Recursió: DFS pot implementar-se de manera recursiva.

Algorisme

  1. Inicialitzar una pila i afegir el node inicial.
  2. Marcar el node inicial com a visitat.
  3. Mentre la pila no estigui buida:
    • Treure un node de la pila.
    • Processar el node.
    • Afegir tots els nodes adjacents no visitats a la pila i marcar-los com a visitats.

Exemple de Codi

def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    visited.add(start)
    print(start, end=" ")
    
    for neighbor in graph[start]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

# Exemple d'ús
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

dfs(graph, 'A')

Comparació entre BFS i DFS

| Característica | BFS | DFS | |----------------|-----|-----| | Estructura de Dades | Cua (Queue) | Pila (Stack) | | Ús Principal | Trobar el camí més curt en grafs no ponderats | Exploració completa de grafs, detecció de cicles | | Complexitat Temporal | O(V + E) | O(V + E) | | Complexitat Espacial | O(V) | O(V) |

Conclusió

BFS i DFS són algorismes fonamentals per al recorregut de grafs, cadascun amb els seus propis avantatges i aplicacions. BFS és ideal per trobar camins més curts en grafs no ponderats, mentre que DFS és útil per explorar completament un graf i detectar cicles. Comprendre aquests algorismes i les seves implementacions és crucial per resoldre una àmplia varietat de problemes en informàtica.
