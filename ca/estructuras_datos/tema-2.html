Introducció

Les estructures de dades són fonamentals en la programació i el desenvolupament de programari. Permeten organitzar i emmagatzemar dades de manera eficient, cosa que és crucial per al rendiment i l'escalabilitat de les aplicacions. En aquesta secció, explorarem per què les estructures de dades són tan importants i com impacten en la programació.

Conceptes Clau

  • Eficiència: Les estructures de dades adequades poden millorar significativament l'eficiència dels algorismes.
  • Organització: Permeten organitzar les dades de manera lògica i accessible.
  • Escalabilitat: Faciliten la gestió de grans volums de dades.
  • Manteniment: Fan que el codi sigui més fàcil de mantenir i entendre.

Tipus d'Estructures de Dades

Estructures de Dades Lineals

  • Arrays: Col·leccions d'elements del mateix tipus emmagatzemats en ubicacions contigües de memòria.
  • Llistes Enllaçades: Col·leccions de nodes on cada node conté una dada i una referència al següent node.
  • Piles (Stacks): Estructures LIFO (Last In, First Out) on l'últim element afegit és el primer a ser eliminat.
  • Cues (Queues): Estructures FIFO (First In, First Out) on el primer element afegit és el primer a ser eliminat.

Estructures de Dades No Lineals

  • Arbres: Estructures jeràrquiques amb un node arrel i subnodes fills.
  • Grafs: Conjunts de nodes connectats per arestes, utilitzats per representar relacions complexes.

Comparació d'Estructures de Dades

| Estructura de Dades | Accés | Inserció | Eliminació | Cerca | |---------------------|-------|----------|------------|-------| | Array | O(1) | O(n) | O(n) | O(n) | | Llista Enllaçada | O(n) | O(1) | O(1) | O(n) | | Pila | O(n) | O(1) | O(1) | O(n) | | Cua | O(n) | O(1) | O(1) | O(n) | | Arbre Binari | O(log n)| O(log n)| O(log n) | O(log n)| | Graf | O(n) | O(1) | O(1) | O(n) |

Exemples de Codi

Array

# Definició d'un array en Python
array = [1, 2, 3, 4, 5]

# Accés a un element
print(array[2])  # Output: 3

# Inserció d'un element
array.append(6)

# Eliminació d'un element
array.remove(3)

Llista Enllaçada

# Definició d'un node en una llista enllaçada
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

# Creació de la llista enllaçada
node1 = Node(1)
node2 = Node(2)
node1.next = node2

# Accés a un element
print(node1.next.data)  # Output: 2

Pila

# Definició d'una pila en Python usant una llista
stack = []

# Inserció d'un element
stack.append(1)
stack.append(2)

# Eliminació d'un element
stack.pop()  # Output: 2

Cua

# Definició d'una cua en Python usant una llista
from collections import deque
queue = deque()

# Inserció d'un element
queue.append(1)
queue.append(2)

# Eliminació d'un element
queue.popleft()  # Output: 1

Conclusió

Les estructures de dades són essencials per a la programació eficient i efectiva. Triar l'estructura de dades adequada pot millorar significativament el rendiment d'una aplicació i facilitar-ne el manteniment. Al llarg d'aquest curs, aprofundirem en cadascuna d'aquestes estructures, explorant-ne els avantatges, desavantatges i casos d'ús específics.
