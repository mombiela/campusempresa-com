Introducció

En l'àmbit de les estructures de dades i algoritmes, els algoritmes de camins mínims són fonamentals per resoldre problemes d'optimització en grafs. Dos dels algoritmes més coneguts per a aquest propòsit són l'algoritme de Dijkstra i l'algoritme de Floyd-Warshall. Aquest tema abordarà ambdós algoritmes, explicant els seus fonaments, aplicacions i proporcionant exemples de codi per a una comprensió pràctica.

Algoritme de Dijkstra

Descripció

L'algoritme de Dijkstra és un algoritme de cerca de camins mínims que troba el camí més curt des d'un node origen a tots els altres nodes en un graf ponderat amb pesos no negatius.

Passos de l'Algoritme

  1. Inicialitzar la distància al node origen com 0 i a tots els altres nodes com infinit.
  2. Marcar tots els nodes com no visitats.
  3. Seleccionar el node no visitat amb la distància més petita.
  4. Actualitzar la distància dels nodes adjacents al node seleccionat.
  5. Marcar el node seleccionat com visitat.
  6. Repetir els passos 3-5 fins que tots els nodes hagin estat visitats.

Exemple de Codi

import heapq

def dijkstra(graph, start):
    # Inicialització
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0
    priority_queue = [(0, start)]
    
    while priority_queue:
        current_distance, current_node = heapq.heappop(priority_queue)
        
        if current_distance > distances[current_node]:
            continue
        
        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight
            
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(priority_queue, (distance, neighbor))
    
    return distances

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

print(dijkstra(graph, 'A'))

Aplicacions

  • Xarxes de telecomunicacions
  • Sistemes de navegació GPS
  • Planificació de rutes en logística

Algoritme de Floyd-Warshall

Descripció

L'algoritme de Floyd-Warshall és un algoritme de programació dinàmica que troba els camins més curts entre tots els parells de nodes en un graf ponderat, permetent pesos negatius però sense cicles negatius.

Passos de l'Algoritme

  1. Crear una matriu de distàncies, on l'entrada (i, j) representa la distància del node i al node j.
  2. Inicialitzar la matriu de distàncies amb els pesos de les arestes del graf.
  3. Per a cada node k, actualitzar la matriu de distàncies considerant si passar per k redueix la distància entre qualsevol parell de nodes (i, j).

Exemple de Codi

def floyd_warshall(graph):
    # Inicialització
    nodes = list(graph.keys())
    dist = {node: {node: float('infinity') for node in nodes} for node in nodes}
    
    for node in nodes:
        dist[node][node] = 0
        for neighbor, weight in graph[node].items():
            dist[node][neighbor] = weight
    
    # Actualització de distàncies
    for k in nodes:
        for i in nodes:
            for j in nodes:
                if dist[i][j] > dist[i][k] + dist[k][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
    
    return dist

# Exemple d'ús
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}

distances = floyd_warshall(graph)
for node, dist in distances.items():
    print(f"Distàncies des de {node}: {dist}")

Aplicacions

  • Anàlisi de xarxes socials
  • Sistemes de recomanació
  • Optimització de rutes en xarxes de transport

Comparació de Dijkstra i Floyd-Warshall

| Característica | Dijkstra | Floyd-Warshall | |-------------------------|-----------------------------------|-----------------------------------| | Tipus de Graf | Ponderat, sense pesos negatius | Ponderat, permet pesos negatius | | Complexitat Temporal | O(V^2) amb matriu d'adjacència, O(E + V log V) amb heap | O(V^3) | | Ús Principal | Camí més curt des d'un node | Camins més curts entre tots els parells de nodes | | Implementació | Més simple | Més complexa |

Conclusió

Els algoritmes de Dijkstra i Floyd-Warshall són eines poderoses per resoldre problemes de camins mínims en grafs. Dijkstra és ideal per trobar el camí més curt des d'un sol node a tots els altres, mentre que Floyd-Warshall és més adequat per trobar camins mínims entre tots els parells de nodes. Comprendre aquests algoritmes i les seves aplicacions és essencial per a qualsevol professional de TI que treballi amb estructures de dades i algoritmes.
