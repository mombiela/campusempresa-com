Introducció a les Cues

Les cues són estructures de dades lineals que segueixen el principi FIFO (First In, First Out), és a dir, el primer element en entrar és el primer en sortir. Són àmpliament utilitzades en escenaris on es requereix un processament seqüencial d'elements, com en la gestió de tasques, sistemes d'impressió i maneig de sol·licituds en servidors.

Conceptes Clau

  • Enqueue (Encolar): Operació per afegir un element al final de la cua.
  • Dequeue (Desencolar): Operació per eliminar l'element del davant de la cua.
  • Front (Davant): Operació per obtenir l'element al davant de la cua sense eliminar-lo.
  • Rear (Final): Operació per obtenir l'element al final de la cua sense eliminar-lo.
  • IsEmpty (Està Buida): Operació per verificar si la cua està buida.
  • IsFull (Està Plena): Operació per verificar si la cua està plena (en el cas de cues amb capacitat limitada).

Implementació de Cues

A continuació, es presenta una implementació bàsica d'una cua utilitzant una llista en Python.

class Queue:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

    def enqueue(self, item):
        self.items.append(item)
        print(f"Enqueued: {item}")

    def dequeue(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items.pop(0)

    def front(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items[0]

    def rear(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items[-1]

    def size(self):
        return len(self.items)

# Exemple d'ús
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
queue.enqueue(3)
print("Front element:", queue.front())
print("Rear element:", queue.rear())
print("Dequeued:", queue.dequeue())
print("Queue size:", queue.size())

Exercicis Pràctics

  1. Implementar una Cua Circular: Modifica la implementació anterior perquè funcioni com una cua circular amb capacitat limitada.
  2. Simulació d'un Sistema d'Impressió: Utilitza la cua per simular un sistema d'impressió on les tasques es processen en l'ordre en què arriben.
  3. Verificació de Palíndroms: Usa una cua per verificar si una paraula és un palíndrom (es llegeix igual d'esquerra a dreta i de dreta a esquerra).

Comparació de Cues i Piles

| Característica | Cua | Pila | |----------------|-----|------| | Ordre de Processament | FIFO (First In, First Out) | LIFO (Last In, First Out) | | Operacions Principals | Enqueue, Dequeue | Push, Pop | | Exemples d'Ús | Gestió de tasques, sistemes d'impressió | Desfer/refer en editors, avaluació d'expressions |

Conclusió

Les cues són una estructura de dades fonamental en la informàtica, especialment útil en situacions on l'ordre de processament és crucial. Comprendre les operacions bàsiques i la seva implementació és essencial per a qualsevol professional de TI. Practicar amb exemples i exercicis pràctics ajudarà a consolidar aquests conceptes i a preparar-se per a aplicacions més avançades.
