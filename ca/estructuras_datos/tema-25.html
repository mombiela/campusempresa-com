Introducció

En aquesta secció del curs sobre Estructures de Dades, aprendrem sobre les operacions bàsiques amb arbres. Els arbres són estructures de dades jeràrquiques que consisteixen en nodes connectats per arestes. Cada node conté un valor i pot tenir fills, formant una estructura en forma d'arbre.

Conceptes Clau

  • Node: Element bàsic d'un arbre que conté un valor o dada.
  • Arrel: El node superior d'un arbre, que no té pares.
  • Fill: Node que descendeix d'un altre node.
  • Pare: Node que té un o més fills.
  • Fulla: Node que no té fills.
  • Subarbre: Arbre format per un node i tots els seus descendents.
  • Alçada: Longitud del camí més llarg des de l'arrel fins a una fulla.
  • Profunditat: Longitud del camí des de l'arrel fins a un node específic.

Operacions Bàsiques

1. Inserció

La inserció és el procés d'afegir un nou node a l'arbre. La ubicació del nou node depèn del tipus d'arbre (per exemple, arbre binari de cerca).

Exemple de Codi: Inserció en un Arbre Binari de Cerca (BST)

class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def insert(root, key):
    if root is None:
        return Node(key)
    else:
        if root.val < key:
            root.right = insert(root.right, key)
        else:
            root.left = insert(root.left, key)
    return root

# Exemple d'ús
root = Node(50)
root = insert(root, 30)
root = insert(root, 20)
root = insert(root, 40)
root = insert(root, 70)
root = insert(root, 60)
root = insert(root, 80)

2. Cerca

La cerca és el procés de trobar un node amb un valor específic a l'arbre.

Exemple de Codi: Cerca en un Arbre Binari de Cerca (BST)

def search(root, key):
    if root is None or root.val == key:
        return root
    if root.val < key:
        return search(root.right, key)
    return search(root.left, key)

# Exemple d'ús
result = search(root, 40)
if result:
    print("Node trobat:", result.val)
else:
    print("Node no trobat")

3. Eliminació

L'eliminació és el procés d'eliminar un node de l'arbre. Aquest procés pot ser més complex, ja que s'han de considerar diversos casos (node sense fills, amb un fill o amb dos fills).

Exemple de Codi: Eliminació en un Arbre Binari de Cerca (BST)

def minValueNode(node):
    current = node
    while current.left is not None:
        current = current.left
    return current

def deleteNode(root, key):
    if root is None:
        return root
    if key < root.val:
        root.left = deleteNode(root.left, key)
    elif key > root.val:
        root.right = deleteNode(root.right, key)
    else:
        if root.left is None:
            return root.right
        elif root.right is None:
            return root.left
        temp = minValueNode(root.right)
        root.val = temp.val
        root.right = deleteNode(root.right, temp.val)
    return root

# Exemple d'ús
root = deleteNode(root, 20)

4. Recorreguts

Els recorreguts són mètodes per visitar tots els nodes d'un arbre en un ordre específic. Els tipus més comuns són:

  • Inorden (In-order): Visita el subarbre esquerre, el node arrel i després el subarbre dret.
  • Preorden (Pre-order): Visita el node arrel, el subarbre esquerre i després el subarbre dret.
  • Postorden (Post-order): Visita el subarbre esquerre, el subarbre dret i després el node arrel.

Exemple de Codi: Recorreguts en un Arbre Binari

def inorder(root):
    if root:
        inorder(root.left)
        print(root.val, end=' ')
        inorder(root.right)

def preorder(root):
    if root:
        print(root.val, end=' ')
        preorder(root.left)
        preorder(root.right)

def postorder(root):
    if root:
        postorder(root.left)
        postorder(root.right)
        print(root.val, end=' ')

# Exemple d'ús
print("Inorden:")
inorder(root)
print("\nPreorden:")
preorder(root)
print("\nPostorden:")
postorder(root)

Conclusió

En aquesta secció, hem cobert les operacions bàsiques amb arbres, incloent la inserció, cerca, eliminació i recorreguts. Aquestes operacions són fonamentals per treballar amb arbres i són aplicables en diverses estructures de dades i algorismes. Assegura't de practicar aquests conceptes i exemples de codi per consolidar la teva comprensió.
