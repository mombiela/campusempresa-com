Introducció a les Piles

Les piles són una estructura de dades fonamental en la informàtica, utilitzada per emmagatzemar i gestionar dades de manera ordenada. Es caracteritzen per seguir el principi LIFO (Last In, First Out), el que significa que l'últim element en entrar és el primer en sortir.

Conceptes Clau

  • LIFO (Last In, First Out): L'últim element que s'afegeix a la pila és el primer en ser remogut.
  • Operacions Bàsiques:
    • Push: Afegir un element a la pila.
    • Pop: Remoure l'element superior de la pila.
    • Peek/Top: Obtenir l'element superior sense remoure'l.
    • isEmpty: Verificar si la pila està buida.
  • Aplicacions Comunes:
    • Maneig d'expressions aritmètiques.
    • Implementació d'algoritmes de retrocés (backtracking).
    • Gestió de trucades a funcions (pila de trucades).

Implementació de Piles en Python

A continuació, es presenta una implementació bàsica d'una pila utilitzant una llista en Python.

class Pila:
    def __init__(self):
        self.items = []

    def esta_vacia(self):
        return len(self.items) == 0

    def apilar(self, item):
        self.items.append(item)

    def desapilar(self):
        if not self.esta_vacia():
            return self.items.pop()
        else:
            raise IndexError("La pila està buida")

    def ver_tope(self):
        if not self.esta_vacia():
            return self.items[-1]
        else:
            raise IndexError("La pila està buida")

Explicació del Codi

  • __init__: Inicialitza una llista buida per emmagatzemar els elements de la pila.
  • esta_vacia: Torna True si la pila està buida, en cas contrari False.
  • apilar: Afegeix un element al final de la llista (part superior de la pila).
  • desapilar: Remou i torna l'últim element de la llista. Llença una excepció si la pila està buida.
  • ver_tope: Torna l'últim element de la llista sense remoure'l. Llença una excepció si la pila està buida.

Exemple d'Ús

A continuació, es mostra un exemple de com utilitzar la classe Pila.

mi_pila = Pila()
mi_pila.apilar(1)
mi_pila.apilar(2)
mi_pila.apilar(3)

print(mi_pila.ver_tope())  # Sortida: 3
print(mi_pila.desapilar())  # Sortida: 3
print(mi_pila.ver_tope())  # Sortida: 2

Comparació amb Altres Estructures de Dades

Conclusió

Les piles són una estructura de dades essencial en la programació, especialment útil en situacions on es requereix un accés LIFO. La implementació en Python és senzilla i pot ser estesa per aplicacions més complexes. Comprendre com funcionen les piles i com implementar-les és fonamental per a qualsevol professional de TI, ja que són la base de molts algoritmes i estructures més avançades.
