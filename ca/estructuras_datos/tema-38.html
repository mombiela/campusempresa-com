Projectes Finals

Introducció

En aquesta secció, els estudiants aplicaran els coneixements adquirits sobre estructures de dades en projectes pràctics. Aquests projectes estan dissenyats per consolidar l'aprenentatge i proporcionar experiència pràctica en la implementació i manipulació de diverses estructures de dades.

Objectius dels Projectes Finals

  • Aplicar conceptes teòrics en escenaris pràctics.
  • Desenvolupar habilitats de resolució de problemes utilitzant estructures de dades.
  • Millorar la capacitat d'escriure codi eficient i ben estructurat.
  • Fomentar el treball en equip i la col·laboració (si els projectes són grupals).

Descripció dels Projectes

Projecte 1: Sistema de Gestió de Biblioteca

Descripció: Desenvolupar un sistema de gestió de biblioteca que permeti realitzar operacions bàsiques com afegir, eliminar, buscar i llistar llibres. Utilitzar una estructura de dades adequada per a cada operació.

Requisits:

  • Implementar una llista enllaçada per emmagatzemar els llibres.
  • Cada llibre ha de tenir atributs com títol, autor, ISBN i any de publicació.
  • Funcionalitats:
    • Afegir un nou llibre.
    • Eliminar un llibre per ISBN.
    • Buscar un llibre per títol o autor.
    • Llistar tots els llibres a la biblioteca.

Exemple de Codi:

class Llibre:
    def __init__(self, titol, autor, isbn, any):
        self.titol = titol
        self.autor = autor
        self.isbn = isbn
        self.any = any
        self.seguent = None

class Biblioteca:
    def __init__(self):
        self.cap = None

    def afegir_llibre(self, llibre):
        if not self.cap:
            self.cap = llibre
        else:
            actual = self.cap
            mentre actual.seguent:
                actual = actual.seguent
            actual.seguent = llibre

    def eliminar_llibre(self, isbn):
        actual = self.cap
        anterior = None
        mentre actual i actual.isbn != isbn:
            anterior = actual
            actual = actual.seguent
        if anterior is None:
            self.cap = actual.seguent
        elif actual:
            anterior.seguent = actual.seguent

    def buscar_llibre(self, titol):
        actual = self.cap
        mentre actual:
            if actual.titol == titol:
                return actual
            actual = actual.seguent
        return None

    def llistar_llibres(self):
        actual = self.cap
        mentre actual:
            print(f'Títol: {actual.titol}, Autor: {actual.autor}, ISBN: {actual.isbn}, Any: {actual.any}')
            actual = actual.seguent

Projecte 2: Sistema de Gestió de Tasques

Descripció: Crear un sistema de gestió de tasques que permeti als usuaris afegir, eliminar, buscar i llistar tasques. Utilitzar una estructura de dades adequada per gestionar les tasques.

Requisits:

  • Implementar una pila (stack) per gestionar les tasques.
  • Cada tasca ha de tenir atributs com descripció, data de venciment i prioritat.
  • Funcionalitats:
    • Afegir una nova tasca.
    • Eliminar la tasca més recent.
    • Buscar una tasca per descripció.
    • Llistar totes les tasques pendents.

Exemple de Codi:

class Tasca:
    def __init__(self, descripcio, data_venciment, prioritat):
        self.descripcio = descripcio
        self.data_venciment = data_venciment
        self.prioritat = prioritat

class GestioTasques:
    def __init__(self):
        self.pila = []

    def afegir_tasca(self, tasca):
        self.pila.append(tasca)

    def eliminar_tasca(self):
        if self.pila:
            return self.pila.pop()
        return None

    def buscar_tasca(self, descripcio):
        for tasca in self.pila:
            if tasca.descripcio == descripcio:
                return tasca
        return None

    def llistar_tasques(self):
        for tasca in reversed(self.pila):
            print(f'Descripció: {tasca.descripcio}, Data de Venciment: {tasca.data_venciment}, Prioritat: {tasca.prioritat}')

Projecte 3: Sistema de Gestió de Contactes

Descripció: Desenvolupar un sistema de gestió de contactes que permeti afegir, eliminar, buscar i llistar contactes. Utilitzar una estructura de dades adequada per a cada operació.

Requisits:

  • Implementar un arbre binari de cerca (BST) per emmagatzemar els contactes.
  • Cada contacte ha de tenir atributs com nom, número de telèfon i correu electrònic.
  • Funcionalitats:
    • Afegir un nou contacte.
    • Eliminar un contacte per nom.
    • Buscar un contacte per nom.
    • Llistar tots els contactes en ordre alfabètic.

Exemple de Codi:

class Contacte:
    def __init__(self, nom, telefon, correu):
        self.nom = nom
        self.telefon = telefon
        self.correu = correu
        self.esquerra = None
        self.dreta = None

class GestioContactes:
    def __init__(self):
        self.arrel = None

    def afegir_contacte(self, contacte):
        if not self.arrel:
            self.arrel = contacte
        else:
            self._afegir(self.arrel, contacte)

    def _afegir(self, node, contacte):
        if contacte.nom < node.nom:
            if node.esquerra is None:
                node.esquerra = contacte
            else:
                self._afegir(node.esquerra, contacte)
        else:
            if node.dreta is None:
                node.dreta = contacte
            else:
                self._afegir(node.dreta, contacte)

    def eliminar_contacte(self, nom):
        self.arrel = self._eliminar(self.arrel, nom)

    def _eliminar(self, node, nom):
        if not node:
            return node
        if nom < node.nom:
            node.esquerra = self._eliminar(node.esquerra, nom)
        elif nom > node.nom:
            node.dreta = self._eliminar(node.dreta, nom)
        else:
            if not node.esquerra:
                return node.dreta
            elif not node.dreta:
                return node.esquerra
            temp = self._min_valor_node(node.dreta)
            node.nom = temp.nom
            node.telefon = temp.telefon
            node.correu = temp.correu
            node.dreta = self._eliminar(node.dreta, temp.nom)
        return node

    def _min_valor_node(self, node):
        actual = node
        mentre actual.esquerra:
            actual = actual.esquerra
        return actual

    def buscar_contacte(self, nom):
        return self._buscar(self.arrel, nom)

    def _buscar(self, node, nom):
        if not node o node.nom == nom:
            return node
        if nom < node.nom:
            return self._buscar(node.esquerra, nom)
        return self._buscar(node.dreta, nom)

    def llistar_contactes(self):
        self._inorden(self.arrel)

    def _inorden(self, node):
        if node:
            self._inorden(node.esquerra)
            print(f'Nom: {node.nom}, Telèfon: {node.telefon}, Correu: {node.correu}')
            self._inorden(node.dreta)
