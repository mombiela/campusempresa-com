Representació de Grafs
Tipus de Grafs: Dirigits i No Dirigits ►

Introducció

En aquesta secció del curs sobre Estructures de Dades, explorarem la representació de grafs. Els grafs són estructures fonamentals en la informàtica que s'utilitzen per modelar relacions i connexions entre diferents entitats. Comprendre com representar grafs de manera eficient és crucial per resoldre problemes complexos en àrees com xarxes, algorismes de cerca, i anàlisi de dades.

Conceptes Clau

  • Graf: Una col·lecció de nodes (o vèrtexs) i arestes (o enllaços) que connecten parells de nodes.
  • Node (Vèrtex): Un punt o entitat en el graf.
  • Aresta (Enllaç): Una connexió entre dos nodes en el graf.
  • Graf Dirigit: Un graf on les arestes tenen una direcció.
  • Graf No Dirigit: Un graf on les arestes no tenen direcció.
  • Graf Ponderat: Un graf on les arestes tenen un pes associat.
  • Graf No Ponderat: Un graf on les arestes no tenen pes.

Representacions de Grafs

Hi ha diverses formes de representar grafs en la memòria d'un ordinador. Les dues representacions més comunes són la matriu d'adjacència i la llista d'adjacència.

Matriu d'Adjacència

Una matriu d'adjacència és una matriu 2D on cada cel·la (i, j) indica si existeix una aresta entre el node i i el node j.

Avantatges

  • Fàcil d'implementar i entendre.
  • Accés ràpid per verificar si existeix una aresta entre dos nodes.

Desavantatges

  • Consumeix molt espai, especialment per a grafs dispersos (sparse graphs).

Exemple

Considerem un graf no dirigit amb 4 nodes:

0 -- 1
|  / |
| /  |
2 -- 3

La matriu d'adjacència seria:

| | 0 | 1 | 2 | 3 | |---|---|---|---|---| | 0 | 0 | 1 | 1 | 0 | | 1 | 1 | 0 | 1 | 1 | | 2 | 1 | 1 | 0 | 1 | | 3 | 0 | 1 | 1 | 0 |

Codi en Python

# Crear una matriu d'adjacència per a un graf amb 4 nodes
n = 4
matriu_adjacencia = [[0] * n for _ in range(n)]

# Afegir arestes
matriu_adjacencia[0][1] = 1
matriu_adjacencia[0][2] = 1
matriu_adjacencia[1][0] = 1
matriu_adjacencia[1][2] = 1
matriu_adjacencia[1][3] = 1
matriu_adjacencia[2][0] = 1
matriu_adjacencia[2][1] = 1
matriu_adjacencia[2][3] = 1
matriu_adjacencia[3][1] = 1
matriu_adjacencia[3][2] = 1

# Imprimir la matriu d'adjacència
for fila in matriu_adjacencia:
    print(fila)

Llista d'Adjacència

Una llista d'adjacència és una col·lecció de llistes, on cada llista i conté els nodes adjacents al node i.

Avantatges

  • Més eficient en termes d'espai per a grafs dispersos.
  • Fàcil d'iterar sobre els nodes adjacents.

Desavantatges

  • Accés més lent per verificar si existeix una aresta entre dos nodes.

Exemple

Considerem el mateix graf no dirigit amb 4 nodes:

0 -- 1
|  / |
| /  |
2 -- 3

La llista d'adjacència seria:

0: [1, 2]
1: [0, 2, 3]
2: [0, 1, 3]
3: [1, 2]

Codi en Python

# Crear una llista d'adjacència per a un graf amb 4 nodes
llista_adjacencia = [[] for _ in range(4)]

# Afegir arestes
llista_adjacencia[0].append(1)
llista_adjacencia[0].append(2)
llista_adjacencia[1].append(0)
llista_adjacencia[1].append(2)
llista_adjacencia[1].append(3)
llista_adjacencia[2].append(0)
llista_adjacencia[2].append(1)
llista_adjacencia[2].append(3)
llista_adjacencia[3].append(1)
llista_adjacencia[3].append(2)

# Imprimir la llista d'adjacència
for i in range(len(llista_adjacencia)):
    print(f"{i}: {llista_adjacencia[i]}")

Conclusió

En aquesta secció, hem explorat les dues representacions més comunes de grafs: la matriu d'adjacència i la llista d'adjacència. Cada una té els seus propis avantatges i desavantatges en termes d'espai i eficiència d'accés. L'elecció de la representació adequada depèn del tipus de graf i de les operacions que es necessitin realitzar amb més freqüència. Comprendre aquestes representacions és fonamental per treballar amb grafs de manera eficient en diverses aplicacions de la informàtica.
