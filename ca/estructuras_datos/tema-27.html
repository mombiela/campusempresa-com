Exercicis Pràctics amb Arbres

Introducció

En aquesta secció, ens enfocarem en exercicis pràctics que t'ajudaran a comprendre i aplicar els conceptes d'arbres en estructures de dades. Els arbres són una estructura jeràrquica que s'utilitza en diverses aplicacions, des de la representació de jerarquies fins a l'optimització de cerques i l'organització de dades.

Conceptes Clau

  • Node: Element bàsic d'un arbre que conté un valor o dada.
  • Arrel: El node superior d'un arbre.
  • Fill: Node descendent d'un altre node.
  • Pare: Node que té un o més nodes fills.
  • Fulla: Node que no té fills.
  • Alçada: Longitud del camí més llarg des de l'arrel fins a una fulla.
  • Profunditat: Longitud del camí des de l'arrel fins a un node específic.
  • Arbre Binari: Arbre en el qual cada node té com a màxim dos fills.
  • Arbre de Cerca Binària (BST): Arbre binari en el qual per a cada node, els valors del subarbre esquerre són menors i els del subarbre dret són majors.

Exercici 1: Creació d'un Arbre Binari

Descripció

En aquest exercici, crearàs un arbre binari simple i realitzaràs operacions bàsiques com inserció i recorregut.

Codi d'Exemple

class Node:
    def __init__(self, valor):
        self.valor = valor
        self.esquerre = None
        self.dret = None

class ArbreBinari:
    def __init__(self):
        self.arrel = None

    def inserir(self, valor):
        if self.arrel is None:
            self.arrel = Node(valor)
        else:
            self._inserir_recursiu(valor, self.arrel)

    def _inserir_recursiu(self, valor, node_actual):
        si valor < node_actual.valor:
            si node_actual.esquerre és None:
                node_actual.esquerre = Node(valor)
            else:
                self._inserir_recursiu(valor, node_actual.esquerre)
        else:
            si node_actual.dret és None:
                node_actual.dret = Node(valor)
            else:
                self._inserir_recursiu(valor, node_actual.dret)

    def recorregut_inorden(self, node_actual):
        si node_actual no és None:
            self.recorregut_inorden(node_actual.esquerre)
            print(node_actual.valor, end=' ')
            self.recorregut_inorden(node_actual.dret)

# Ús de l'arbre
arbre = ArbreBinari()
valors = [7, 3, 9, 1, 5, 8, 10]
per valor en valors:
    arbre.inserir(valor)

print("Recorregut Inorden:")
arbre.recorregut_inorden(arbre.arrel)

Explicació

  • Classe Node: Defineix un node amb un valor i punters als seus fills esquerre i dret.
  • Classe ArbreBinari: Defineix l'arbre binari amb mètodes per inserir valors i recórrer l'arbre en ordre.
  • Mètode inserir: Insereix un valor en l'arbre, ubicant-lo en la posició correcta.
  • Mètode recorregut_inorden: Realitza un recorregut en ordre de l'arbre, imprimint els valors en ordre ascendent.

Exercici 2: Cerca en un Arbre de Cerca Binària (BST)

Descripció

En aquest exercici, implementaràs un mètode per cercar un valor en un BST.

Codi d'Exemple

class ArbreBinariCerca(ArbreBinari):
    def cercar(self, valor):
        return self._cercar_recursiu(valor, self.arrel)

    def _cercar_recursiu(self, valor, node_actual):
        si node_actual és None:
            return False
        elif valor == node_actual.valor:
            return True
        elif valor < node_actual.valor:
            return self._cercar_recursiu(valor, node_actual.esquerre)
        else:
            return self._cercar_recursiu(valor, node_actual.dret)

# Ús de l'arbre de cerca
arbre_bst = ArbreBinariCerca()
valors = [7, 3, 9, 1, 5, 8, 10]
per valor en valors:
    arbre_bst.inserir(valor)

print("Cercar 5:", arbre_bst.cercar(5))  # Hauria d'imprimir True
print("Cercar 6:", arbre_bst.cercar(6))  # Hauria d'imprimir False

Explicació

  • Classe ArbreBinariCerca: Hereta d'ArbreBinari i afegeix un mètode per cercar valors.
  • Mètode cercar: Crida a un mètode recursiu per cercar un valor en l'arbre.
  • Mètode _cercar_recursiu: Realitza la cerca recursiva, comparant el valor cercat amb el valor del node actual i movent-se a l'esquerra o dreta segons correspongui.

Conclusió

En aquesta secció, hem cobert exercicis pràctics per treballar amb arbres binaris i arbres de cerca binària. Aquests exercicis t'ajudaran a comprendre com es estructuren i manipulen els arbres, així com a realitzar operacions bàsiques com inserció, recorregut i cerca. Practicar aquests conceptes és fonamental per dominar les estructures de dades i millorar les teves habilitats en programació i anàlisi d'algoritmes.
