Introducció a les Piles

Les piles són una estructura de dades fonamental en informàtica que segueix el principi de LIFO (Last In, First Out). Això significa que l'últim element en ser afegit a la pila serà el primer en ser remogut. Les operacions bàsiques d'una pila són:

  • push: Afegir un element al capdamunt de la pila.
  • pop: Remoure l'element del capdamunt de la pila.
  • peek: Veure l'element al capdamunt de la pila sense remoure'l.
  • isEmpty: Verificar si la pila està buida.

Exercici 1: Implementació Bàsica d'una Pila

Objectiu

Implementar una pila utilitzant una llista en Python i realitzar les operacions bàsiques.

Instruccions

  1. Crear una classe Pila.
  2. Implementar els mètodes push, pop, peek i isEmpty.
  3. Provar la funcionalitat de la pila amb alguns exemples.

Exemple de Codi

class Pila:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.isEmpty():
            return self.items.pop()
        else:
            return "La pila està buida"

    def peek(self):
        if not self.isEmpty():
            return self.items[-1]
        else:
            return "La pila està buida"

    def isEmpty(self):
        return len(self.items) == 0

# Proves
pila = Pila()
pila.push(1)
pila.push(2)
pila.push(3)
print(pila.peek())  # Sortida: 3
print(pila.pop())   # Sortida: 3
print(pila.pop())   # Sortida: 2
print(pila.isEmpty())  # Sortida: False
print(pila.pop())   # Sortida: 1
print(pila.isEmpty())  # Sortida: True

Explicació

  • push: Afegeix un element al final de la llista.
  • pop: Remou i retorna l'últim element de la llista.
  • peek: Retorna l'últim element de la llista sense remoure'l.
  • isEmpty: Verifica si la llista està buida.

Exercici 2: Verificació de Parèntesis Balancejats

Objectiu

Utilitzar una pila per verificar si els parèntesis en una expressió estan balancejats.

Instruccions

  1. Crear una funció verificar_parentesis que prengui una cadena com a entrada.
  2. Utilitzar una pila per verificar si els parèntesis estan balancejats.

Exemple de Codi

def verificar_parentesis(expressio):
    pila = Pila()
    for char in expressio:
        if char == '(':
            pila.push(char)
        elif char == ')':
            if pila.isEmpty():
                return False
            pila.pop()
    return pila.isEmpty()

# Proves
print(verificar_parentesis("(())"))  # Sortida: True
print(verificar_parentesis("(()"))   # Sortida: False
print(verificar_parentesis(")("))    # Sortida: False

Explicació

  • Recorre cada caràcter de l'expressió.
  • Si el caràcter és un '(', s'afegeix a la pila.
  • Si el caràcter és un ')', es remou un element de la pila.
  • Al final, la pila ha d'estar buida perquè els parèntesis estiguin balancejats.

Exercici 3: Conversió de Notació Infix a Postfix

Objectiu

Convertir una expressió en notació infix (convencional) a notació postfix (notació polonesa inversa) utilitzant una pila.

Instruccions

  1. Crear una funció infix_a_postfix que prengui una expressió infix com a entrada.
  2. Utilitzar una pila per manejar els operadors i parèntesis.

Exemple de Codi

def infix_a_postfix(expressio):
    precedencia = {'+': 1, '-': 1, '*': 2, '/': 2, '^': 3}
    pila = Pila()
    sortida = []

    for char in expressio:
        if char.isalnum():
            sortida.append(char)
        elif char == '(':
            pila.push(char)
        elif char == ')':
            while not pila.isEmpty() and pila.peek() != '(':
                sortida.append(pila.pop())
            pila.pop()
        else:
            while (not pila.isEmpty() and pila.peek() != '(' and
                   precedencia[char] <= precedencia[pila.peek()]):
                sortida.append(pila.pop())
            pila.push(char)

    while not pila.isEmpty():
        sortida.append(pila.pop())

    return ''.join(sortida)

# Proves
print(infix_a_postfix("a+b*(c^d-e)^(f+g*h)-i"))  # Sortida: abcd^e-fgh*+^*+i-

Explicació

  • Utilitza una pila per emmagatzemar operadors i parèntesis.
  • Els operandes s'afegeixen directament a la sortida.
  • Els operadors es manegen segons la seva precedència i s'afegeixen a la sortida en l'ordre correcte.

Conclusió

Les piles són una estructura de dades versàtil i essencial en la informàtica. A través d'aquests exercicis, hem après a implementar una pila bàsica, verificar parèntesis balancejats i convertir expressions infix a postfix. Aquests exercicis no només reforcen l'enteniment de les piles, sinó que també mostren la seva aplicabilitat en problemes del món real.
