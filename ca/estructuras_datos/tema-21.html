Introducció als Arbres Binàris

Un arbre binari és una estructura de dades jeràrquica en la qual cada node té com a màxim dos fills, denominats fill esquerre i fill dret. Els arbres binaris són fonamentals en la informàtica i s'utilitzen en diverses aplicacions, com la representació d'expressions aritmètiques, l'organització de dades jeràrquiques i la implementació d'estructures de dades eficients com els arbres de cerca binària.

Conceptes Clau

  • Node: La unitat bàsica d'un arbre binari que conté un valor o dada i referències als seus fills.
  • Arrel: El node superior de l'arbre. No té pares.
  • Fill: Un node descendent d'un altre node.
  • Fulla: Un node que no té fills.
  • Subarbre: Un arbre format per un node i tots els seus descendents.
  • Alçada: La longitud del camí més llarg des de l'arrel fins a una fulla.
  • Profunditat: La longitud del camí des de l'arrel fins a un node específic.

Tipus d'Arbres Binàris

  • Arbre Binari Complet: Tots els nivells, excepte possiblement l'últim, estan completament plens, i tots els nodes estan el més a l'esquerra possible.
  • Arbre Binari Perfecte: Tots els nivells estan completament plens.
  • Arbre Binari de Cerca (BST): Un arbre binari en el qual per a cada node, tots els nodes en el seu subarbre esquerre tenen valors menors i tots els nodes en el seu subarbre dret tenen valors majors.

Operacions Bàsiques

Inserció

La inserció en un arbre binari de cerca (BST) implica col·locar un nou node en la posició correcta per mantenir la propietat d'ordre del BST.

class Node:
    def __init__(self, valor):
        self.valor = valor
        self.esquerre = None
        self.dret = None

class ArbreBinariCerca:
    def __init__(self):
        self.arrel = None

    def inserir(self, valor):
        if self.arrel is None:
            self.arrel = Node(valor)
        else:
            self._inserir_recursiu(self.arrel, valor)

    def _inserir_recursiu(self, node, valor):
        if valor < node.valor:
            if node.esquerre is None:
                node.esquerre = Node(valor)
            else:
                self._inserir_recursiu(node.esquerre, valor)
        else:
            if node.dret is None:
                node.dret = Node(valor)
            else:
                self._inserir_recursiu(node.dret, valor)

Cerca

La cerca en un BST implica recórrer l'arbre des de l'arrel, triant el subarbre esquerre o dret en funció del valor que es busca.

def cercar(self, valor):
    return self._cercar_recursiu(self.arrel, valor)

def _cercar_recursiu(self, node, valor):
    if node is None o node.valor == valor:
        return node
    if valor < node.valor:
        return self._cercar_recursiu(node.esquerre, valor)
    else:
        return self._cercar_recursiu(node.dret, valor)

Eliminació

Eliminar un node en un BST és més complex i depèn de si el node té zero, un o dos fills.

def eliminar(self, valor):
    self.arrel = self._eliminar_recursiu(self.arrel, valor)

def _eliminar_recursiu(self, node, valor):
    if node is None:
        return node
    if valor < node.valor:
        node.esquerre = self._eliminar_recursiu(node.esquerre, valor)
    elif valor > node.valor:
        node.dret = self._eliminar_recursiu(node.dret, valor)
    else:
        if node.esquerre is None:
            return node.dret
        elif node.dret is None:
            return node.esquerre
        temp = self._min_valor_node(node.dret)
        node.valor = temp.valor
        node.dret = self._eliminar_recursiu(node.dret, temp.valor)
    return node

def _min_valor_node(self, node):
    actual = node
    while actual.esquerre is not None:
        actual = actual.esquerre
    return actual

Exemples i Aplicacions

Exemple d'Inserció i Cerca

arbre = ArbreBinariCerca()
arbre.inserir(50)
arbre.inserir(30)
arbre.inserir(70)
arbre.inserir(20)
arbre.inserir(40)
arbre.inserir(60)
arbre.inserir(80)

# Cercar un valor
node = arbre.cercar(60)
if node:
    print(f"Node trobat: {node.valor}")
else:
    print("Node no trobat")

Aplicacions d'Arbres Binàris

  • Arbres d'Expressió: Utilitzats per avaluar expressions matemàtiques.
  • Arbres de Decisió: Utilitzats en algorismes d'aprenentatge automàtic.
  • Arbres de Segment: Utilitzats en problemes de rang i consultes.

Conclusió

Els arbres binaris són una estructura de dades versàtil i eficient que s'utilitza en una àmplia varietat d'aplicacions. Comprendre els conceptes bàsics i les operacions fonamentals dels arbres binaris és essencial per a qualsevol professional de TI. A través de la pràctica i la implementació d'aquestes operacions, els estudiants poden desenvolupar una comprensió profunda de com funcionen els arbres binaris i com poden aplicar-se en problemes del món real.
