Introducció

Les llistes són una de les estructures de dades més fonamentals i versàtils en la programació. Permeten emmagatzemar col·leccions ordenades d'elements i proporcionen una varietat d'operacions per manipular aquests elements de manera eficient. En aquest tema, explorarem les operacions bàsiques amb llistes, incloent la creació, accés, modificació i eliminació d'elements.

Creació de Llistes

Per crear una llista en Python, simplement s'utilitzen claudàtors [] i es separen els elements per comes.

# Crear una llista buida
la_meva_llista = []

# Crear una llista amb elements
la_meva_llista = [1, 2, 3, 4, 5]

Accés a Elements

Els elements d'una llista es poden accedir mitjançant índexs, on el primer element té un índex de 0.

la_meva_llista = [10, 20, 30, 40, 50]

# Accedir al primer element
print(la_meva_llista[0])  # Sortida: 10

# Accedir al tercer element
print(la_meva_llista[2])  # Sortida: 30

Modificació d'Elements

Pots modificar els elements d'una llista assignant un nou valor a l'índex corresponent.

la_meva_llista = [10, 20, 30, 40, 50]

# Modificar el segon element
la_meva_llista[1] = 25
print(la_meva_llista)  # Sortida: [10, 25, 30, 40, 50]

Afegir Elements

Hi ha diverses formes d'afegir elements a una llista:

  • append(): Afegeix un element al final de la llista.
  • insert(): Insereix un element en una posició específica.
  • extend(): Afegeix múltiples elements al final de la llista.
la_meva_llista = [1, 2, 3]

# Afegir un element al final
la_meva_llista.append(4)
print(la_meva_llista)  # Sortida: [1, 2, 3, 4]

# Inserir un element en la segona posició
la_meva_llista.insert(1, 1.5)
print(la_meva_llista)  # Sortida: [1, 1.5, 2, 3, 4]

# Afegir múltiples elements al final
la_meva_llista.extend([5, 6])
print(la_meva_llista)  # Sortida: [1, 1.5, 2, 3, 4, 5, 6]

Eliminació d'Elements

Pots eliminar elements d'una llista utilitzant diverses tècniques:

  • remove(): Elimina la primera aparició d'un valor específic.
  • pop(): Elimina i retorna l'element en una posició específica.
  • del: Elimina un element en una posició específica o un rang d'elements.
la_meva_llista = [1, 2, 3, 4, 5]

# Eliminar el primer element amb valor 3
la_meva_llista.remove(3)
print(la_meva_llista)  # Sortida: [1, 2, 4, 5]

# Eliminar i retornar l'últim element
últim_element = la_meva_llista.pop()
print(últim_element)  # Sortida: 5
print(la_meva_llista)  # Sortida: [1, 2, 4]

# Eliminar el primer element
del la_meva_llista[0]
print(la_meva_llista)  # Sortida: [2, 4]

Iteració sobre Llistes

Pots iterar sobre els elements d'una llista utilitzant un bucle for.

la_meva_llista = [10, 20, 30, 40, 50]

# Iterar sobre els elements de la llista
for element in la_meva_llista:
    print(element)

Conclusió

Les llistes són una estructura de dades essencial en la programació, oferint una forma flexible i eficient d'emmagatzemar i manipular col·leccions d'elements. En aquest tema, hem cobert les operacions bàsiques amb llistes, incloent la creació, accés, modificació, addició i eliminació d'elements. Comprendre aquestes operacions és fonamental per treballar amb llistes de manera efectiva en els teus programes.
