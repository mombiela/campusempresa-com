Introducció

Les cues són estructures de dades fonamentals que segueixen el principi FIFO (First In, First Out). Això significa que el primer element en entrar és el primer en sortir. Les cues tenen diverses aplicacions en el camp de la informàtica i són essencials per a la gestió de tasques en sistemes operatius, processament de dades en xarxes i molt més.

Aplicacions Comunes de les Cues

1. Gestió de Tasques en Sistemes Operatius

  • Cues d'impressió: Els treballs d'impressió s'emmagatzemen en una cua i es processen en l'ordre en què es reben.
  • Planificació de processos: Els sistemes operatius utilitzen cues per gestionar l'execució de processos. Els processos es col·loquen en una cua d'espera i s'executen en ordre.
from queue import Queue

# Exemple de cua d'impressió
cua_impressio = Queue()

# Afegir treballs d'impressió a la cua
cua_impressio.put("Document1.pdf")
cua_impressio.put("Document2.pdf")
cua_impressio.put("Document3.pdf")

# Processar treballs d'impressió
while not cua_impressio.empty():
    treball = cua_impressio.get()
    print(f"Imprimint {treball}")

2. Gestió de Sol·licituds en Xarxes

  • Cues de paquets: En xarxes, els paquets de dades s'emmagatzemen en cues abans de ser enviats o processats.
  • Balanceig de càrrega: Els servidors utilitzen cues per gestionar les sol·licituds dels clients i distribuir la càrrega de manera equitativa.
from queue import Queue

# Exemple de cua de paquets de xarxa
cua_paquets = Queue()

# Afegir paquets a la cua
cua_paquets.put("Paquet1")
cua_paquets.put("Paquet2")
cua_paquets.put("Paquet3")

# Processar paquets
while not cua_paquets.empty():
    paquet = cua_paquets.get()
    print(f"Processant {paquet}")

3. Algoritmes de Cerca en Grafs

  • Cerca en amplada (BFS): Utilitza una cua per explorar els nodes d'un graf nivell per nivell.
  • Detecció de cicles: Les cues poden ajudar a detectar cicles en grafs dirigits.
from collections import deque

# Exemple de BFS en un graf
def bfs(graf, inici):
    visitats = set()
    cua = deque([inici])
    
    while cua:
        node = cua.popleft()
        si node no està en visitats:
            print(f"Visitant {node}")
            visitats.add(node)
            cua.extend(graf[node] - visitats)

# Graf representat com un diccionari
graf = {
    'A': {'B', 'C'},
    'B': {'A', 'D', 'E'},
    'C': {'A', 'F'},
    'D': {'B'},
    'E': {'B', 'F'},
    'F': {'C', 'E'}
}

bfs(graf, 'A')

4. Simulació de Sistemes

  • Simulació de cues en bancs: Modelar l'espera de clients en un banc.
  • Simulació de trànsit: Modelar el flux de vehicles en una intersecció.
from queue import Queue
import random

# Exemple de simulació de cua en un banc
class Client:
    def __init__(self, id):
        self.id = id

cua_banc = Queue()
clients = [Client(i) for i in range(1, 6)]

# Afegir clients a la cua
for client in clients:
    cua_banc.put(client)

# Processar clients
while not cua_banc.empty():
    client = cua_banc.get()
    print(f"Atentent el client {client.id}")

Conclusió

Les cues són una estructura de dades versàtil i essencial en diverses aplicacions informàtiques. Des de la gestió de tasques en sistemes operatius fins a la simulació de sistemes complexos, les cues faciliten la gestió de processos i dades de manera eficient. Comprendre i saber implementar cues és fonamental per a qualsevol professional de TI, ja que el seu ús s'estén a múltiples àrees de la informàtica.
