Introducció

Els arbres són una estructura de dades fonamental en informàtica, utilitzada per representar jerarquies i relacions entre elements. En aquest tema, explorarem diverses aplicacions dels arbres en el camp de la informàtica i com s'utilitzen per resoldre problemes comuns.

Aplicacions Comunes dels Arbres

1. Arbres Binàries de Cerca (BST)

Els Arbres Binàries de Cerca són una estructura de dades que permet la cerca, inserció i eliminació d'elements de manera eficient.

  • Propietats:

    • Cada node té com a màxim dos fills.
    • El fill esquerre conté un valor menor que el node pare.
    • El fill dret conté un valor més gran que el node pare.

  • Exemple de Codi:

    class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def insert(root, key):
    if root is None:
        return Node(key)
    else:
        if root.val < key:
            root.right = insert(root.right, key)
        else:
            root.left = insert(root.left, key)
    return root

# Crear l'arbre
root = Node(50)
root = insert(root, 30)
root = insert(root, 20)
root = insert(root, 40)
root = insert(root, 70)
root = insert(root, 60)
root = insert(root, 80)

2. Arbres AVL

Els Arbres AVL són una variant dels Arbres Binàries de Cerca que es mantenen balancejats per assegurar que les operacions de cerca, inserció i eliminació es realitzin en temps logarítmic.

  • Propietats:

    • És un arbre binari de cerca.
    • La diferència d'altura entre els subarbres esquerre i dret de qualsevol node és com a màxim 1.

  • Exemple de Codi:

    class Node:
    def __init__(self, key):
        self.key = key
        self.left = None
        self.right = None
        self.height = 1

def insert(root, key):
    if not root:
        return Node(key)
    elif key < root.key:
        root.left = insert(root.left, key)
    else:
        root.right = insert(root.right, key)

    root.height = 1 + max(get_height(root.left), get_height(root.right))

    balance = get_balance(root)

    # Rotacions per balancejar l'arbre
    if balance > 1 and key < root.left.key:
        return right_rotate(root)
    if balance < -1 and key > root.right.key:
        return left_rotate(root)
    if balance > 1 and key > root.left.key:
        root.left = left_rotate(root.left)
        return right_rotate(root)
    if balance < -1 and key < root.right.key:
        root.right = right_rotate(root.right)
        return left_rotate(root)

    return root

def get_height(root):
    if not root:
        return 0
    return root.height

def get_balance(root):
    if not root:
        return 0
    return get_height(root.left) - get_height(root.right)

def right_rotate(z):
    y = z.left
    T3 = y.right
    y.right = z
    z.left = T3
    z.height = 1 + max(get_height(z.left), get_height(z.right))
    y.height = 1 + max(get_height(y.left), get_height(y.right))
    return y

def left_rotate(z):
    y = z.right
    T2 = y.left
    y.left = z
    z.right = T2
    z.height = 1 + max(get_height(z.left), get_height(z.right))
    y.height = 1 + max(get_height(y.left), get_height(y.right))
    return y

3. Arbres B

Els Arbres B són una estructura de dades auto-balancejada que manté dades ordenades i permet cerques, insercions i eliminacions en temps logarítmic. Són àmpliament utilitzats en sistemes de bases de dades i sistemes de fitxers.

  • Propietats:

    • Cada node pot tenir més de dos fills.
    • Tots els nodes fulla estan al mateix nivell.
    • Els nodes interns poden tenir un nombre variable de fills dins d'un rang predefinit.

  • Exemple de Codi:

    class BTreeNode:
    def __init__(self, t, leaf=False):
        self.t = t
        self.leaf = leaf
        self.keys = []
        self.children = []

class BTree:
    def __init__(self, t):
        self.root = BTreeNode(t, True)
        self.t = t

    def insert(self, k):
        root = self.root
        si len(root.keys) == (2 * self.t) - 1:
            temp = BTreeNode(self.t)
            self.root = temp
            temp.children.insert(0, root)
            self.split_child(temp, 0)
            self.insert_non_full(temp, k)
        else:
            self.insert_non_full(root, k)

    def insert_non_full(self, x, k):
        i = len(x.keys) - 1
        if x.leaf:
            x.keys.append(None)
            while i >= 0 and k < x.keys[i]:
                x.keys[i + 1] = x.keys[i]
                i -= 1
            x.keys[i + 1] = k
        else:
            while i >= 0 and k < x.keys[i]:
                i -= 1
            i += 1
            if len(x.children[i].keys) == (2 * self.t) - 1:
                self.split_child(x, i)
                if k > x.keys[i]:
                    i += 1
            self.insert_non_full(x.children[i], k)

    def split_child(self, x, i):
        t = self.t
        y = x.children[i]
        z = BTreeNode(t, y.leaf)
        x.children.insert(i + 1, z)
        x.keys.insert(i, y.keys[t - 1])
        z.keys = y.keys[t:(2 * t) - 1]
        y.keys = y.keys[0:t - 1]
        if not y.leaf:
            z.children = y.children[t:(2 * t)]
            y.children = y.children[0:t - 1]

Conclusió

Els arbres són una estructura de dades versàtil i poderosa amb aplicacions en diverses àrees de la informàtica. Des de la cerca eficient en Arbres Binàries de Cerca i Arbres AVL fins a la gestió de grans volums de dades en Arbres B, la seva comprensió i ús adequat són essencials per a qualsevol professional de TI. A través d'exemples pràctics i exercicis, els estudiants poden desenvolupar una sòlida comprensió de com i quan utilitzar aquestes estructures de dades en els seus projectes.
