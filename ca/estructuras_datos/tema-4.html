Introducció a les Llistes Enllaçades

Les llistes enllaçades són una estructura de dades fonamental que consisteix en una seqüència d'elements, on cada element apunta al següent. A diferència dels arrays, les llistes enllaçades no requereixen una mida fixa i permeten una inserció i eliminació d'elements més eficient.

  • Node: La unitat bàsica d'una llista enllaçada que conté un valor i una referència (o enllaç) al següent node.
  • Cap (Head): El primer node de la llista enllaçada.
  • Cua (Tail): L'últim node de la llista enllaçada, que apunta a null o None.

Tipus de Llistes Enllaçades

Llista Enllaçada Simple

En una llista enllaçada simple, cada node apunta al següent node en la seqüència.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last_node = self.head
        while last_node.next:
            last_node = last_node.next
        last_node.next = new_node

    def print_list(self):
        current_node = self.head
        while current_node:
            print(current_node.data, end=" -> ")
            current_node = current_node.next
        print("None")

# Exemple d'ús
ll = LinkedList()
ll.append(1)
ll.append(2)
ll.append(3)
ll.print_list()  # Output: 1 -> 2 -> 3 -> None

Llista Enllaçada Doble

En una llista enllaçada doble, cada node té dues referències: una al següent node i una altra al node anterior.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last_node = self.head
        while last_node.next:
            last_node = last_node.next
        last_node.next = new_node
        new_node.prev = last_node

    def print_list(self):
        current_node = self.head
        while current_node:
            print(current_node.data, end=" <-> ")
            current_node = current_node.next
        print("None")

# Exemple d'ús
dll = DoublyLinkedList()
dll.append(1)
dll.append(2)
dll.append(3)
dll.print_list()  # Output: 1 <-> 2 <-> 3 <-> None

Llista Enllaçada Circular

En una llista enllaçada circular, l'últim node apunta de nou al primer node, formant un cicle.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class CircularLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            new_node.next = self.head
            return
        last_node = self.head
        while last_node.next != self.head:
            last_node = last_node.next
        last_node.next = new_node
        new_node.next = self.head

    def print_list(self):
        current_node = self.head
        if not self.head:
            return
        while True:
            print(current_node.data, end=" -> ")
            current_node = current_node.next
            if current_node == self.head:
                break
        print("Head")

# Exemple d'ús
cll = CircularLinkedList()
cll.append(1)
cll.append(2)
cll.append(3)
cll.print_list()  # Output: 1 -> 2 -> 3 -> Head

Operacions Comunes

  • Inserció: Afegir un nou node a la llista.
  • Eliminació: Treure un node existent de la llista.
  • Cerca: Trobar un node amb un valor específic.
  • Recorregut: Visitar cada node de la llista per realitzar alguna operació.

Comparació de Llistes Enllaçades

| Tipus de Llista | Avantatges | Desavantatges | |---------------------|---------------------------------------------|----------------------------------------------| | Simple | Fàcil d'implementar, menys memòria usada | No es pot recórrer cap enrere | | Doble | Recorregut bidireccional | Més memòria usada degut a referències extra | | Circular | Recorregut continu sense fi | Complexitat addicional en maneig d'enllaços |

Conclusió

Les llistes enllaçades són una estructura de dades versàtil i eficient per a moltes aplicacions. Comprendre els diferents tipus de llistes enllaçades i les seves operacions bàsiques és crucial per a qualsevol professional de TI. Practicar amb exemples de codi i exercicis ajudarà a consolidar aquest coneixement i a aplicar-lo en situacions del món real.
