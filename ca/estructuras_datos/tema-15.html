Introducció a les Cues

Les cues són estructures de dades lineals que segueixen el principi FIFO (First In, First Out), on el primer element en entrar és el primer en sortir. Són àmpliament utilitzades en programació i sistemes informàtics per gestionar tasques i recursos.

Cua Simple

Una cua simple és la forma més bàsica de cua. En ella, els elements s'afegeixen al final (enqueuing) i s'eliminen del front (dequeuing).

Característiques d'una Cua Simple

  • FIFO: El primer element en entrar és el primer en sortir.
  • Operacions bàsiques:
    • enqueue: Afegir un element al final de la cua.
    • dequeue: Eliminar un element del front de la cua.
    • front: Obtenir l'element del front sense eliminar-lo.
    • isEmpty: Verificar si la cua està buida.

Exemple de Codi en Python

class CuaSimple:
    def __init__(self):
        self.items = []

    def enqueue(self, item):
        self.items.append(item)

    def dequeue(self):
        if not self.isEmpty():
            return self.items.pop(0)
        else:
            return None

    def front(self):
        if not self.isEmpty():
            return self.items[0]
        else:
            return None

    def isEmpty(self):
        return len(self.items) == 0

# Exemple d'ús
cua = CuaSimple()
cua.enqueue(1)
cua.enqueue(2)
print(cua.dequeue())  # Sortida: 1
print(cua.front())    # Sortida: 2

Cua Doble (Deque)

Una cua doble, o deque (Double-Ended Queue), permet la inserció i eliminació d'elements tant al front com al final de la cua.

Característiques d'una Cua Doble

  • Operacions en ambdós extrems:
    • enqueueFront: Afegir un element al front de la cua.
    • enqueueRear: Afegir un element al final de la cua.
    • dequeueFront: Eliminar un element del front de la cua.
    • dequeueRear: Eliminar un element del final de la cua.
    • isEmpty: Verificar si la cua està buida.

Exemple de Codi en Python

class Deque:
    def __init__(self):
        self.items = []

    def enqueueFront(self, item):
        self.items.insert(0, item)

    def enqueueRear(self, item):
        self.items.append(item)

    def dequeueFront(self):
        if not self.isEmpty():
            return self.items.pop(0)
        else:
            return None

    def dequeueRear(self):
        if not self.isEmpty():
            return self.items.pop()
        else:
            return None

    def isEmpty(self):
        return len(self.items) == 0

# Exemple d'ús
deque = Deque()
deque.enqueueFront(1)
deque.enqueueRear(2)
print(deque.dequeueFront())  # Sortida: 1
print(deque.dequeueRear())   # Sortida: 2

Cua de Prioritat

Una cua de prioritat és una estructura de dades en la qual cada element té una prioritat associada. Els elements amb major prioritat s'eliminen abans que els de menor prioritat.

Característiques d'una Cua de Prioritat

  • Prioritat: Cada element té una prioritat.
  • Operacions bàsiques:
    • enqueue: Afegir un element amb una prioritat.
    • dequeue: Eliminar l'element amb la major prioritat.
    • isEmpty: Verificar si la cua està buida.

Exemple de Codi en Python

import heapq

class CuaDePrioritat:
    def __init__(self):
        self.heap = []

    def enqueue(self, item, priority):
        heapq.heappush(self.heap, (priority, item))

    def dequeue(self):
        if not self.isEmpty():
            return heapq.heappop(self.heap)[1]
        else:
            return None

    def isEmpty(self):
        return len(self.heap) == 0

# Exemple d'ús
cua_prioritat = CuaDePrioritat()
cua_prioritat.enqueue("tasca1", 2)
cua_prioritat.enqueue("tasca2", 1)
print(cua_prioritat.dequeue())  # Sortida: "tasca2"
print(cua_prioritat.dequeue())  # Sortida: "tasca1"

Comparació de Tipus de Cues

| Tipus de Cua | Inserció al Front | Inserció al Final | Eliminació del Front | Eliminació del Final | Prioritat | |-------------------|-------------------|-------------------|----------------------|----------------------|-----------| | Cua Simple | No | Sí | Sí | No | No | | Cua Doble (Deque) | Sí | Sí | Sí | Sí | No | | Cua de Prioritat | No | Sí | Sí (per prioritat) | No | Sí |

Conclusió

Les cues són estructures de dades fonamentals en la informàtica, i comprendre els seus diferents tipus i usos és essencial per a qualsevol professional de TI. Les cues simples són ideals per a tasques bàsiques de gestió de dades, les cues dobles ofereixen major flexibilitat en permetre operacions en ambdós extrems, i les cues de prioritat són crucials per a sistemes on la prioritat de les tasques és important.
