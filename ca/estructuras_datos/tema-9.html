Introducció

Una pila és una estructura de dades lineal que segueix el principi LIFO (Last In, First Out), la qual cosa significa que l'últim element en entrar és el primer en sortir. Aquest concepte és fonamental en la informàtica i s'utilitza en diverses aplicacions com la gestió de memòria, l'avaluació d'expressions i la implementació d'algoritmes.

Característiques d'una Pila

  • LIFO (Last In, First Out): L'últim element inserit és el primer a ser remogut.
  • Operacions bàsiques:
    • Push: Afegir un element a la pila.
    • Pop: Remoure l'element superior de la pila.
    • Peek/Top: Obtenir l'element superior sense remoure'l.
    • isEmpty: Verificar si la pila està buida.
    • Size: Obtenir el nombre d'elements a la pila.

Implementació d'una Pila

Les piles poden ser implementades de diverses maneres, sent les més comunes:

  • Usant Arrays: Una pila de mida fixa.
  • Usant Llistes Enllaçades: Una pila de mida dinàmica.

Implementació usant Arrays

class StackArray:
    def __init__(self, capacity):
        self.capacity = capacity
        self.stack = [None] * capacity
        self.top = -1

    def push(self, item):
        if self.top == self.capacity - 1:
            raise Exception("Stack Overflow")
        self.top += 1
        self.stack[self.top] = item

    def pop(self):
        if self.top == -1:
            raise Exception("Stack Underflow")
        item = self.stack[self.top]
        self.stack[self.top] = None
        self.top -= 1
        return item

    def peek(self):
        if self.top == -1:
            raise Exception("Stack is empty")
        return self.stack[self.top]

    def isEmpty(self):
        return self.top == -1

    def size(self):
        return self.top + 1

Implementació usant Llistes Enllaçades

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class StackLinkedList:
    def __init__(self):
        self.head = None
        self.count = 0

    def push(self, item):
        new_node = Node(item)
        new_node.next = self.head
        self.head = new_node
        self.count += 1

    def pop(self):
        if self.head is None:
            raise Exception("Stack Underflow")
        item = self.head.value
        self.head = self.head.next
        self.count -= 1
        return item

    def peek(self):
        if self.head is None:
            raise Exception("Stack is empty")
        return self.head.value

    def isEmpty(self):
        return self.head is None

    def size(self):
        return self.count

Comparació entre Implementacions

| Característica | Array | Llista Enllaçada | |----------------------|--------------------------------|---------------------------------| | Mida | Fixa | Dinàmica | | Espai | Pre-alocat | Alocat segons necessitat | | Complexitat Push | O(1) | O(1) | | Complexitat Pop | O(1) | O(1) | | Ús de Memòria | Més eficient si la mida és coneguda | Menys eficient degut als punters |

Exemple Pràctic

A continuació, es mostra un exemple pràctic de com usar una pila per verificar si una seqüència de parèntesis està balancejada.

def is_balanced(expression):
    stack = StackArray(len(expression))
    for char in expression:
        if char in "({[":
            stack.push(char)
        elif char in ")}]":
            if stack.isEmpty():
                return False
            top_char = stack.pop()
            if not matches(top_char, char):
                return False
    return stack.isEmpty()

def matches(opening, closing):
    opens = "({["
    closes = ")}]"
    return opens.index(opening) == closes.index(closing)

# Exemple d'ús
expression = "{[()]}"
print(is_balanced(expression))  # Output: True

Conclusió

Les piles són una estructura de dades fonamental en la informàtica, amb aplicacions en diverses àrees com la gestió de memòria i l'avaluació d'expressions. Comprendre com implementar i utilitzar piles és essencial per a qualsevol professional de TI. Les implementacions poden variar segons les necessitats específiques, ja sigui usant arrays per a una mida fixa o llistes enllaçades per a una mida dinàmica.
