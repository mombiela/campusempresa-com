Introducció

El Patró Memento és un patró de disseny de comportament que permet capturar i externalitzar l'estat intern d'un objecte sense violar la encapsulació, de manera que l'objecte pugui ser restaurat a aquest estat més tard. Aquest patró és útil per implementar funcionalitats de desfer i refer en aplicacions.

Conceptes Clau

  • Originador: L'objecte el qual estat intern es desitja guardar.
  • Memento: Un objecte que emmagatzema l'estat intern de l'Originador.
  • Cuidador (Caretaker): L'encarregat de sol·licitar i emmagatzemar Mementos, però no opera ni inspecciona el seu contingut.

Estructura del Patró

| Component | Descripció | |------------|-------------| | Originador | L'objecte que té un estat intern que necessita ser guardat i restaurat. | | Memento | Emmagatzema l'estat intern de l'Originador. | | Cuidador | Gestiona els Mementos i sol·licita a l'Originador que restauri el seu estat quan sigui necessari. |

Implementació en Codi

A continuació, es presenta un exemple en Python per il·lustrar el Patró Memento:

class Memento:
    def __init__(self, state):
        self._state = state

    def get_state(self):
        return self._state

class Originator:
    def __init__(self):
        self._state = None

    def set_state(self, state):
        print(f"Setting state to {state}")
        self._state = state

    def save_to_memento(self):
        print("Saving to Memento.")
        return Memento(self._state)

    def restore_from_memento(self, memento):
        self._state = memento.get_state()
        print(f"State restored to {self._state}")

class Caretaker:
    def __init__(self):
        self._mementos = []

    def add_memento(self, memento):
        self._mementos.append(memento)

    def get_memento(self, index):
        return self._mementos[index]

# Ús del Patró Memento
originator = Originator()
caretaker = Caretaker()

# Canviar estat i guardar en Memento
originator.set_state("State1")
caretaker.add_memento(originator.save_to_memento())

originator.set_state("State2")
caretaker.add_memento(originator.save_to_memento())

# Restaurar estat des de Memento
originator.restore_from_memento(caretaker.get_memento(0))

Explicació del Codi

  1. Memento: La classe Memento emmagatzema l'estat de l'Originator. Té un mètode get_state per recuperar l'estat emmagatzemat.
  2. Originator: La classe Originator té mètodes per establir (set_state), guardar (save_to_memento) i restaurar (restore_from_memento) el seu estat.
  3. Caretaker: La classe Caretaker gestiona els Mementos. Pot agregar (add_memento) i recuperar (get_memento) Mementos.

Exercici Pràctic

  1. Exercici 1: Implementa el Patró Memento en una aplicació de text que permeti desfer i refer canvis.
  2. Exercici 2: Extén l'exemple anterior per manejar múltiples estats complexos, com l'estat d'un joc.

Conclusió

El Patró Memento és una solució elegant per a problemes on es necessita guardar i restaurar l'estat d'un objecte sense violar la seva encapsulació. Aquest patró és especialment útil en aplicacions que requereixen funcionalitats de desfer i refer, proporcionant una forma estructurada i segura de manejar l'estat intern dels objectes.
