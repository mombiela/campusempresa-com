Introducció

El patró Facade és un dels patrons de disseny estructurals més utilitzats en el desenvolupament de programari. El seu principal objectiu és proporcionar una interfície simplificada a un conjunt d'interfícies en un subsistema, fent que el subsistema sigui més fàcil d'utilitzar.

Objectius del Patró Facade

  • Simplificació: Proveir una interfície simple i clara per a un sistema complex.
  • Desacoblament: Reduir la dependència entre el client i el subsistema.
  • Facilitat d'ús: Fer que el sistema sigui més fàcil d'entendre i utilitzar.

Conceptes Clau

  • Facade: La classe que proporciona una interfície simplificada a un conjunt d'interfícies en un subsistema.
  • Subsistema: Conjunt de classes i components que realitzen les funcionalitats complexes.
  • Client: La classe o component que utilitza la interfície simplificada proporcionada pel Facade.

Exemple de Codi

A continuació, es presenta un exemple en Python per il·lustrar com s'implementa el patró Facade.

# Subsistema Complex
class SistemaDeAudio:
    def encendre(self):
        print("Sistema de audio encès")

    def ajustarVolum(self, nivell):
        print(f"Volum ajustat a {nivell}")

class SistemaDeVideo:
    def encendre(self):
        print("Sistema de vídeo encès")

    def ajustarResolucio(self, resolucio):
        print(f"Resolució ajustada a {resolucio}")

class SistemaDeIl·luminacio:
    def encendre(self):
        print("Sistema d'il·luminació encès")

    def ajustarBrillantor(self, nivell):
        print(f"Brillantor ajustada a {nivell}")

# Facade
class HomeTheaterFacade:
    def __init__(self):
        self.audio = SistemaDeAudio()
        self.video = SistemaDeVideo()
        self.il·luminacio = SistemaDeIl·luminacio()

    def iniciarCinemaACasa(self):
        self.audio.encendre()
        self.audio.ajustarVolum(5)
        self.video.encendre()
        self.video.ajustarResolucio("1080p")
        self.il·luminacio.encendre()
        self.il·luminacio.ajustarBrillantor(3)
        print("Cinema a casa iniciat")

# Client
def main():
    cinemaACasa = HomeTheaterFacade()
    cinemaACasa.iniciarCinemaACasa()

if __name__ == "__main__":
    main()

Explicació del Codi

  • Subsistema Complex: SistemaDeAudio, SistemaDeVideo i SistemaDeIl·luminacio són classes que representen diferents components del sistema.
  • Facade: HomeTheaterFacade és la classe que proporciona una interfície simplificada per interactuar amb el subsistema complex.
  • Client: La funció main actua com el client que utilitza la interfície simplificada proporcionada per HomeTheaterFacade.

Comparació amb Altres Patrons

| Patró | Propòsit | Exemple d'Ús | |-----------------|--------------------------------------------------------------------------|-------------------------------------------------------------------------------| | Facade | Proveir una interfície simplificada a un conjunt d'interfícies en un subsistema | Simplificar la interacció amb un sistema de cinema a casa | | Adapter | Permetre que classes amb interfícies incompatibles treballin juntes | Adaptar una interfície d'un servei de tercers perquè funcioni amb el teu sistema | | Proxy | Proveir un substitut o marcador per controlar l'accés a un objecte | Controlar l'accés a un objecte remot o costós de crear |

Conclusió

El patró Facade és una eina poderosa per simplificar la interacció amb sistemes complexos, proporcionant una interfície clara i fàcil d'utilitzar. En reduir l'acoblament entre el client i el subsistema, facilita el manteniment i l'escalabilitat del programari. Aquest patró és especialment útil en sistemes grans i complexos on la simplicitat i la claredat són essencials per a l'eficiència i la facilitat d'ús.
