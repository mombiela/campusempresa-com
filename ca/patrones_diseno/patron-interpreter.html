Introducció

El Patró Interpreter és un dels patrons de disseny de comportament que s'utilitza per definir una representació gramatical per a un llenguatge i un intèrpret que utilitza la representació per interpretar sentències en el llenguatge. Aquest patró és útil quan es necessita avaluar o interpretar sentències en un llenguatge específic de domini (DSL).

Conceptes Clau

  • Gramàtica: Conjunt de regles que defineix l'estructura d'un llenguatge.
  • Expressió: Una instància de la gramàtica que pot ser avaluada.
  • Context: Informació global que pot ser utilitzada durant la interpretació.
  • Intèrpret: Component que processa les expressions basades en la gramàtica definida.

Estructura del Patró Interpreter

El patró Interpreter generalment es compon de les següents classes:

  1. AbstractExpression: Defineix la interfície per interpretar el context.
  2. TerminalExpression: Implementa una operació interpretativa associada amb els elements terminals de la gramàtica.
  3. NonTerminalExpression: Implementa una operació interpretativa associada amb els elements no terminals de la gramàtica.
  4. Context: Conté informació global que pot ser utilitzada durant la interpretació.

Diagrama de Classes

| Classe | Descripció | |-----------------------|----------------------------------------------------------------------------| | AbstractExpression | Defineix la interfície per interpretar el context. | | TerminalExpression | Implementa l'operació interpretativa per a elements terminals. | | NonTerminalExpression | Implementa l'operació interpretativa per a elements no terminals. | | Context | Conté informació global que pot ser utilitzada durant la interpretació. |

Exemple de Codi

A continuació, es mostra un exemple simple del Patró Interpreter en Python. Aquest exemple interpreta una expressió matemàtica simple.

class Context:
    def __init__(self, expression):
        self.expression = expression
        self.index = 0

class AbstractExpression:
    def interpret(self, context):
        pass

class NumberExpression(AbstractExpression):
    def interpret(self, context):
        start = context.index
        while context.index < len(context.expression) and context.expression[context.index].isdigit():
            context.index += 1
        return int(context.expression[start:context.index])

class AddExpression(AbstractExpression):
    def __init__(self, left, right):
        self.left = left
        self.right = right

    def interpret(self, context):
        return self.left.interpret(context) + self.right.interpret(context)

class SubtractExpression(AbstractExpression):
    def __init__(self, left, right):
        self.left = left
        self.right = right

    def interpret(self, context):
        return self.left.interpret(context) - self.right.interpret(context)

# Ús del patró Interpreter
context = Context("3+5-2")
expression = AddExpression(NumberExpression(), SubtractExpression(NumberExpression(), NumberExpression()))
result = expression.interpret(context)
print(f"Resultat: {result}")  # Sortida: Resultat: 6

Explicació del Codi

  • Context: Manté l'expressió i l'índex actual.
  • AbstractExpression: Defineix la interfície per a les expressions.
  • NumberExpression: Interpreta números en l'expressió.
  • AddExpression: Interpreta l'operació de suma.
  • SubtractExpression: Interpreta l'operació de resta.

En l'exemple, es crea un context amb l'expressió "3+5-2". Després, es construeix una expressió composta utilitzant AddExpression i SubtractExpression. Finalment, s'interpreta l'expressió completa i s'obté el resultat.

Conclusió

El Patró Interpreter és útil per definir i avaluar llenguatges específics de domini. Encara que pot no ser adequat per a llenguatges complexos a causa de la dificultat de mantenir una gramàtica extensa, és molt efectiu per a llenguatges simples i ben definits. Aquest patró permet una fàcil extensió i modificació de la gramàtica i les regles d'interpretació.
