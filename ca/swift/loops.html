Els bucles són construccions fonamentals en la programació que et permeten executar un bloc de codi múltiples vegades. Swift proporciona diversos tipus de bucles per manejar diferents escenaris. Aquesta secció et guiarà des dels conceptes bàsics fins a l'ús avançat dels bucles a Swift.

Tipus de Bucles

Swift admet tres tipus principals de bucles:

  • Bucles for-in
  • Bucles while
  • Bucles repeat-while

Bucles `for-in`

El bucle for-in s'utilitza per iterar sobre una seqüència, com ara elements en una matriu, rangs de números o caràcters en una cadena.

Sintaxi

for item in collection {
    // Codi a executar
}

Exemple

let fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits {
    print(fruit)
}

En aquest exemple, el bucle itera sobre cada element de la matriu fruits i el mostra.

Bucles `while`

El bucle while continua executant un bloc de codi mentre una condició especificada sigui certa.

Sintaxi

while condition {
    // Codi a executar
}

Exemple

var count = 5

while count > 0 {
    print("Count is \(count)")
    count -= 1
}

Aquest bucle mostrarà el valor de count i el decrementarà fins que count ja no sigui més gran que 0.

Bucles `repeat-while`

El bucle repeat-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada abans que es valori la condició.

Sintaxi

repeat {
    // Codi a executar
} while condition

Exemple

var count = 5

repeat {
    print("Count is \(count)")
    count -= 1
} while count > 0

Aquest bucle es comporta de manera similar a l'exemple del bucle while però assegura que el bloc de codi s'executi almenys una vegada.

Declaracions de Control de Bucles

Swift proporciona declaracions de control per alterar el flux dels bucles:

  • break
  • continue

Declaració `break`

La declaració break finalitza el bucle immediatament.

Exemple

for number in 1...5 {
    if number == 3 {
        break
    }
    print(number)
}

Aquest bucle mostrarà 1 i 2, i després es finalitzarà quan number sigui igual a 3.

Declaració `continue`

La declaració continue salta la iteració actual i procedeix a la següent iteració del bucle.

Exemple

for number in 1...5 {
    if number == 3 {
        continue
    }
    print(number)
}

Aquest bucle mostrarà 1, 2, 4 i 5, saltant la iteració on number és igual a 3.

Bucles Imbricats

Els bucles es poden imbricar dins d'altres bucles per manejar escenaris més complexos.

Exemple

for i in 1...3 {
    for j in 1...3 {
        print("i = \(i), j = \(j)")
    }
}

Aquest exemple demostra un bucle imbricat on el bucle extern s'executa tres vegades, i per a cada iteració del bucle extern, el bucle intern també s'executa tres vegades.

Iterar a Través de Col·leccions

El bucle for-in de Swift és particularment útil per iterar a través de col·leccions com matrius, diccionaris i conjunts.

Iterar a Través d'una Matriu

let numbers = [1, 2, 3, 4, 5]

for number in numbers {
    print(number)
}

Iterar a Través d'un Diccionari

let studentGrades = ["John": "A", "Jane": "B", "Jim": "C"]

for (student, grade) in studentGrades {
    print("\(student) received a grade of \(grade)")
}

Iterar a Través d'un Conjunt

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Tècniques Avançades de Bucles

Utilitzar `stride` per a Increments Personalitzats

La funció stride et permet crear rangs amb increments personalitzats.

Exemple

for number in stride(from: 0, to: 10, by: 2) {
    print(number)
}

Aquest bucle mostrarà 0, 2, 4, 6 i 8.

Iterar amb Seqüències Enumerades

El mètode enumerated() retorna una seqüència de parells (índex, element), útil quan necessites tant l'índex com el valor.

Exemple

let animals = ["Dog", "Cat", "Bird"]

for (index, animal) in animals.enumerated() {
    print("Animal \(index + 1): \(animal)")
}

Aquest bucle mostrarà:

Animal 1: Dog
Animal 2: Cat
Animal 3: Bird

Conclusió

Els bucles són una eina poderosa a Swift que et permeten realitzar tasques repetitives de manera eficient. Entendre els diferents tipus de bucles i les seves declaracions de control et permetrà escriure codi més efectiu i concís. Practica utilitzant bucles amb diverses col·leccions i condicions per convertir-te en un expert en el seu ús.
