En aquesta secció, explorarem com iterar sobre col·leccions en Swift. Les col·leccions en Swift inclouen arrays, diccionaris, conjunts i rangs. Entendre com iterar eficientment sobre aquestes col·leccions és fonamental per a qualsevol programador de Swift.

Conceptes clau

  • Col·leccions en Swift: Arrays, diccionaris, conjunts i rangs.
  • Bucle For-in: El bucle principal utilitzat per iterar sobre col·leccions.
  • Mètode Enumerated: Utilitzat per obtenir tant l'índex com el valor.
  • Funcions d'ordre superior: forEach, map, filter, reduce.

Iterant sobre arrays

Els arrays són col·leccions ordenades d'elements. Comencem amb els conceptes bàsics d'iterar sobre arrays.

Utilitzant el bucle For-in

El bucle for-in és la manera més comuna d'iterar sobre un array.

let fruits = ["Poma", "Plàtan", "Cirera"]

for fruit in fruits {
    print(fruit)
}

Explicació:

  • fruits és un array de cadenes.
  • El bucle for-in itera sobre cada element de l'array i el mostra.

Utilitzant el mètode Enumerated

El mètode enumerated proporciona tant l'índex com el valor de cada element de l'array.

let fruits = ["Poma", "Plàtan", "Cirera"]

for (index, fruit) in fruits.enumerated() {
    print("Fruit \(index + 1): \(fruit)")
}

Explicació:

  • enumerated retorna una seqüència de parelles (índex, element).
  • El bucle mostra tant l'índex com el valor.

Iterant sobre diccionaris

Els diccionaris són col·leccions de parelles clau-valor. Iterar sobre diccionaris és lleugerament diferent dels arrays.

Utilitzant el bucle For-in

let capitals = ["França": "París", "Japó": "Tòquio", "EUA": "Washington D.C."]

for (country, capital) in capitals {
    print("La capital de \(country) és \(capital).")
}

Explicació:

  • capitals és un diccionari amb noms de països com a claus i ciutats capitals com a valors.
  • El bucle for-in itera sobre cada parella clau-valor.

Iterant sobre conjunts

Els conjunts són col·leccions no ordenades d'elements únics. Iterar sobre conjunts és similar als arrays però sense ordre garantit.

Utilitzant el bucle For-in

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Explicació:

  • uniqueNumbers és un conjunt d'enters.
  • El bucle for-in itera sobre cada element del conjunt.

Iterant sobre rangs

Els rangs representen una seqüència de valors. Sovint s'utilitzen en bucles.

Utilitzant el bucle For-in

for number in 1...5 {
    print(number)
}

Explicació:

  • 1...5 és un rang tancat que inclou nombres de l'1 al 5.
  • El bucle for-in itera sobre cada nombre del rang.

Funcions d'ordre superior

Swift proporciona diverses funcions d'ordre superior que es poden utilitzar per iterar sobre col·leccions d'una manera més estil funcional.

forEach

El mètode forEach és una alternativa al bucle for-in.

let fruits = ["Poma", "Plàtan", "Cirera"]

fruits.forEach { fruit in
    print(fruit)
}

Explicació:

  • forEach pren una clàusula que s'executa per a cada element de l'array.

map

El mètode map transforma cada element de la col·lecció.

let numbers = [1, 2, 3, 4, 5]
let squaredNumbers = numbers.map { $0 * $0 }

print(squaredNumbers)

Explicació:

  • map aplica la clàusula a cada element i retorna un nou array amb els elements transformats.

filter

El mètode filter retorna una nova col·lecció que conté elements que compleixen una condició donada.

let numbers = [1, 2, 3, 4, 5]
let evenNumbers = numbers.filter { $0 % 2 == 0 }

print(evenNumbers)

Explicació:

  • filter aplica la clàusula a cada element i retorna un nou array amb elements que compleixen la condició.

reduce

El mètode reduce combina tots els elements de la col·lecció en un sol valor.

let numbers = [1, 2, 3, 4, 5]
let sum = numbers.reduce(0, +)

print(sum)

Explicació:

  • reduce pren un valor inicial i una clàusula que combina el valor inicial amb cada element de l'array.

Conclusió

Iterar sobre col·leccions és una habilitat fonamental en la programació en Swift. Tant si treballes amb arrays, diccionaris, conjunts o rangs, entendre els diversos mètodes i tècniques per a la iteració t'ajudarà a escriure codi més eficient i llegible. A més, aprofitar funcions d'ordre superior com forEach, map, filter i reduce pot conduir a un codi més expressiu i concís. Practica aquests conceptes per convertir-te en un expert en la gestió de col·leccions en Swift.
