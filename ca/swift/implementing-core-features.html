Introducció

En aquesta secció, explorarem com implementar funcionalitats bàsiques en Swift. Això inclou entendre els blocs de construcció fonamentals del llenguatge, com ara variables, constants, funcions i flux de control. Al final d'aquesta secció, seràs capaç de crear programes bàsics en Swift i entendre com estructurar el teu codi de manera efectiva.

Variables i Constants

Les variables i constants són les unitats bàsiques d'emmagatzematge en Swift. Et permeten emmagatzemar i manipular dades.

  • Variables: Utilitza var per declarar una variable que pot canviar.
  • Constants: Utilitza let per declarar una constant que no pot canviar.

Exemple:

var myVariable = 42
myVariable = 50
print(myVariable) // Sortida: 50

let myConstant = 42
// myConstant = 50 // Això causarà un error perquè myConstant és una constant
print(myConstant) // Sortida: 42

Tipus de Dades

Swift és un llenguatge segur de tipus, el que significa que t'encoratja a ser clar sobre els tipus de valors amb els quals el teu codi pot treballar.

  • Tipus de Dades Comuns:
    • Int: Nombres enters
    • Double i Float: Nombres de punt flotant
    • Bool: Valors booleans
    • String: Dades textuals

Exemple:

let integer: Int = 10
let floatingPoint: Double = 10.5
let boolean: Bool = true
let text: String = "Hola, Swift!"

Funcions

Les funcions són blocs de codi autònoms que realitzen una tasca específica.

  • Definició d'una Funció: Utilitza la paraula clau func.
  • Crida d'una Funció: Utilitza el nom de la funció seguit de parèntesis.

Exemple:

func greet(name: String) -> String {
    return "Hola, \(name)!"
}

let greeting = greet(name: "Alice")
print(greeting) // Sortida: Hola, Alice!

Flux de Control

Les instruccions de flux de control et permeten controlar l'execució del teu codi basant-se en certes condicions.

  • Instruccions If: Executa codi basat en una condició.
  • Instruccions Switch: Tria entre múltiples valors possibles.
  • Bucles: Repeteix un bloc de codi múltiples vegades.

Exemple:

// Instrucció If
let condition = true
if condition {
    print("La condició és certa")
} else {
    print("La condició és falsa")
}

// Instrucció Switch
let number = 3
switch number {
case 1:
    print("Un")
case 2:
    print("Dos")
case 3:
    print("Tres")
default:
    print("Altres números")
}

// Bucle For
for i in 1...5 {
    print(i)
}

// Bucle While
var count = 5
while count > 0 {
    print(count)
    count -= 1
}

Col·leccions

Swift proporciona tipus de col·leccions potents com ara arrays, conjunts i diccionaris per emmagatzemar grups de valors.

  • Arrays: Col·leccions ordenades de valors.
  • Conjunts: Col·leccions desordenades de valors únics.
  • Diccionaris: Col·leccions desordenades de parells clau-valor.

Exemple:

// Array
var fruits: [String] = ["Poma", "Plàtan", "Cirera"]
fruits.append("Dàtil")
print(fruits) // Sortida: ["Poma", "Plàtan", "Cirera", "Dàtil"]

// Conjunt
var uniqueNumbers: Set<Int> = [1, 2, 3, 3, 4]
print(uniqueNumbers) // Sortida: [2, 3, 1, 4]

// Diccionari
var capitals: [String: String] = ["França": "París", "Japó": "Tòquio"]
capitals["Itàlia"] = "Roma"
print(capitals) // Sortida: ["França": "París", "Japó": "Tòquio", "Itàlia": "Roma"]

Conclusió

En aquesta secció, hem cobert les funcionalitats bàsiques de Swift, incloent variables, constants, tipus de dades, funcions, flux de control i col·leccions. Aquests blocs de construcció són essencials per escriure qualsevol programa en Swift. Entendre i dominar aquests conceptes proporcionarà una base sòlida per a temes més avançats en la programació amb Swift.
