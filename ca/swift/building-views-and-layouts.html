Introducció

Construir vistes i dissenys és una habilitat fonamental per a qualsevol desenvolupador d'iOS. En aquesta secció, cobrirem els conceptes bàsics de la creació d'interfícies d'usuari en Swift utilitzant UIKit i SwiftUI. Al final d'aquest tema, seràs capaç de crear dissenys complexos i entendre els principis darrere del disseny responsiu en aplicacions iOS.

Conceptes Bàsics de UIKit

Vistes i Jerarquies de Vistes

  • UIView: El bloc de construcció bàsic per a tots els elements de la UI.
  • UIViewController: Gestiona una vista i el seu cicle de vida.
  • Jerarquia de Vistes: Com les vistes estan niades entre si.
import UIKit

class MyViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let redView = UIView()
        redView.backgroundColor = .red
        redView.frame = CGRect(x: 50, y: 50, width: 100, height: 100)
        
        let blueView = UIView()
        blueView.backgroundColor = .blue
        blueView.frame = CGRect(x: 20, y: 20, width: 50, height: 50)
        
        redView.addSubview(blueView)
        self.view.addSubview(redView)
    }
}
  • Explicació: En aquest exemple, creem una vista vermella i una vista blava. La vista blava s'afegeix com a subvista de la vista vermella, i la vista vermella s'afegeix a la vista principal del controlador de vista.

Auto Layout

  • Restriccions: Defineixen la mida i la posició de les vistes.
  • NSLayoutConstraint: Crear restriccions programàticament.
  • Llenguatge de Format Visual: Una manera abreujada de definir restriccions.
import UIKit

class MyViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        let redView = UIView()
        redView.backgroundColor = .red
        redView.translatesAutoresizingMaskIntoConstraints = false
        self.view.addSubview(redView)
        
        NSLayoutConstraint.activate([
            redView.centerXAnchor.constraint(equalTo: self.view.centerXAnchor),
            redView.centerYAnchor.constraint(equalTo: self.view.centerYAnchor),
            redView.widthAnchor.constraint(equalToConstant: 100),
            redView.heightAnchor.constraint(equalToConstant: 100)
        ])
    }
}
  • Explicació: Aquí, utilitzem Auto Layout per centrar una vista vermella al mig de la pantalla i establir la seva mida a 100x100 punts.

Conceptes Bàsics de SwiftUI

Vistes i Modificadors

  • View: El bloc de construcció bàsic en SwiftUI.
  • Modificadors: Mètodes que retornen una nova vista amb els canvis aplicats.
import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hola, món!")
            .font(.largeTitle)
            .foregroundColor(.blue)
            .padding()
    }
}
  • Explicació: Aquest exemple de SwiftUI crea una vista de text amb una font de títol gran, color blau i una mica de padding al voltant.

Contenidors de Disseny

  • HStack: Apila vistes horitzontalment.
  • VStack: Apila vistes verticalment.
  • ZStack: Superposa vistes una sobre l'altra.
import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Text("Hola, món!")
                .font(.largeTitle)
                .foregroundColor(.blue)
            HStack {
                Text("Esquerra")
                Spacer()
                Text("Dreta")
            }
            .padding()
        }
    }
}
  • Explicació: Aquest exemple utilitza un VStack per apilar un títol i un HStack. L'HStack conté dues vistes de text amb un espaiador al mig per empènyer-les cap als marges.

Tècniques Avançades de Disseny

Vistes Personalitzades

  • Subclassificació de UIView: Crear vistes personalitzades subclassificant UIView.
  • Vistes Personalitzades en SwiftUI: Crear components reutilitzables en SwiftUI.
import UIKit

class CustomView: UIView {
    override init(frame: CGRect) {
        super.init(frame: frame)
        self.backgroundColor = .green
    }
    
    required init?(coder: NSCoder) {
        fatalError("init(coder:) has not been implemented")
    }
}
  • Explicació: Aquest exemple de UIKit mostra com crear una vista personalitzada subclassificant UIView.
import SwiftUI

struct CustomView: View {
    var body: some View {
        Rectangle()
            .fill(Color.green)
            .frame(width: 100, height: 100)
    }
}
  • Explicació: Aquest exemple de SwiftUI crea una vista personalitzada que mostra un rectangle verd.

Disseny Responsiu

  • Classes de Mida: Adaptar dissenys per a diferents mides de pantalla i orientacions.
  • GeometryReader: Accedir a la informació de mida i posició en SwiftUI.
import SwiftUI

struct ResponsiveView: View {
    var body: some View {
        GeometryReader { geometry in
            if geometry.size.width > 600 {
                Text("Pantalla Gran")
                    .font(.largeTitle)
            } else {
                Text("Pantalla Petita")
                    .font(.body)
            }
        }
    }
}
  • Explicació: Aquest exemple de SwiftUI utilitza GeometryReader per canviar la mida del text en funció de l'amplada de la pantalla.

Conclusió

Construir vistes i dissenys en Swift implica entendre tant UIKit com SwiftUI. UIKit proporciona un enfocament més tradicional basat en codi, mentre que SwiftUI ofereix una sintaxi declarativa que pot simplificar moltes tasques. Dominar ambdós et donarà la flexibilitat per crear interfícies d'usuari complexes i responsives per a qualsevol aplicació iOS.
