Introducció a UIKit

UIKit és un potent framework proporcionat per Apple per construir interfícies d'usuari en aplicacions iOS. Ofereix una àmplia gamma d'eines i components per crear interfícies visualment atractives i responsives.

Conceptes clau

  • UIView: El bloc de construcció bàsic per a tots els elements de la UI.
  • UIViewController: Gestiona una jerarquia de vistes per a la teva aplicació.
  • Auto Layout: Un sistema per definir la posició i la mida de les vistes.
  • Storyboards i XIBs: Eines per dissenyar interfícies visualment.
  • Gestió d'esdeveniments: Gestionar les interaccions de l'usuari.

Començant amb UIView i UIViewController

Creant una vista simple

Per crear una vista simple, pots utilitzar la classe UIView. Aquí tens un exemple bàsic:

import UIKit

class SimpleViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()

        let simpleView = UIView()
        simpleView.backgroundColor = .blue
        simpleView.frame = CGRect(x: 50, y: 50, width: 200, height: 200)
        self.view.addSubview(simpleView)
    }
}
  • Explicació:
    • UIViewController: Gestiona el cicle de vida de la vista.
    • viewDidLoad(): Es crida després que la vista del controlador hagi estat carregada a la memòria.
    • UIView(): Crea una nova vista.
    • backgroundColor: Estableix el color de fons de la vista.
    • frame: Defineix la posició i la mida de la vista.
    • addSubview(): Afegeix la vista a la jerarquia de vistes.

Utilitzant Auto Layout

Auto Layout et permet crear interfícies responsives que s'adapten a diferents mides de pantalla i orientacions.

Exemple bàsic d'Auto Layout

Aquí tens com crear una vista amb Auto Layout:

import UIKit

class AutoLayoutViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()

        let autoLayoutView = UIView()
        autoLayoutView.backgroundColor = .red
        autoLayoutView.translatesAutoresizingMaskIntoConstraints = false
        self.view.addSubview(autoLayoutView)

        NSLayoutConstraint.activate([
            autoLayoutView.centerXAnchor.constraint(equalTo: self.view.centerXAnchor),
            autoLayoutView.centerYAnchor.constraint(equalTo: self.view.centerYAnchor),
            autoLayoutView.widthAnchor.constraint(equalToConstant: 100),
            autoLayoutView.heightAnchor.constraint(equalToConstant: 100)
        ])
    }
}
  • Explicació:
    • translatesAutoresizingMaskIntoConstraints: Desactiva les restriccions automàtiques.
    • NSLayoutConstraint.activate(): Activa les restriccions.
    • centerXAnchor, centerYAnchor: Centra la vista.
    • widthAnchor, heightAnchor: Estableix l'amplada i l'alçada de la vista.

Storyboards i XIBs

Els Storyboards i XIBs són eines visuals per dissenyar interfícies d'usuari.

Utilitzant Storyboards

  • Creant un controlador de vista:
    1. Obre el fitxer Main.storyboard.
    2. Arrossega un controlador de vista des de la Biblioteca d'Objectes fins al llenç.
    3. Afegeix elements de la UI com etiquetes, botons i imatges.
    4. Estableix les restriccions utilitzant Auto Layout.

Utilitzant XIBs

  • Creant un fitxer XIB:
    1. Crea un nou fitxer XIB.
    2. Dissenya la interfície utilitzant l'Interface Builder.
    3. Enllaça el fitxer XIB a una classe UIView personalitzada.

Gestió d'esdeveniments

Gestionar les interaccions de l'usuari és crucial per crear aplicacions interactives.

Gestionant tocs de botons

Aquí tens com gestionar el toc d'un botó:

import UIKit

class ButtonViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()

        let button = UIButton(type: .system)
        button.setTitle("Tap Me", for: .normal)
        button.addTarget(self, action: #selector(buttonTapped), for: .touchUpInside)
        button.frame = CGRect(x: 100, y: 100, width: 100, height: 50)
        self.view.addSubview(button)
    }

    @objc func buttonTapped() {
        print("Button was tapped!")
    }
}
  • Explicació:
    • UIButton(type: .system): Crea un botó amb estil de sistema.
    • setTitle(): Estableix el títol del botó.
    • addTarget(): Afegeix una parella objectiu-acció per al botó.
    • @objc func buttonTapped(): Defineix el mètode d'acció.
