Introducció a SwiftUI
Construcció de Vistes i Dissenys ►

SwiftUI és el framework d'Apple per construir interfícies d'usuari a totes les plataformes d'Apple utilitzant Swift. Proporciona una sintaxi declarativa en Swift que permet als desenvolupadors construir interfícies d'usuari d'una manera més intuïtiva i menys propensa a errors.

Conceptes Clau de SwiftUI

  • Sintaxi Declarativa: Descriure què ha de fer la UI.
  • Vistes i Modificadors: Blocs de construcció de SwiftUI.
  • Gestió de l'Estat: Gestionar els canvis de dades en la UI.
  • Dissenys: Disposar les vistes a la pantalla.
  • Interactivitat: Gestionar l'entrada de l'usuari i els gestos.

Sintaxi Declarativa

SwiftUI utilitza una sintaxi declarativa, la qual cosa significa que descrius com ha de ser la UI i com ha de comportar-se. Això és diferent de l'enfocament imperatiu utilitzat a UIKit.

Exemple

import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hola, SwiftUI!")
            .font(.largeTitle)
            .padding()
    }
}

En aquest exemple:

  • Text("Hola, SwiftUI!") crea una vista de text.
  • .font(.largeTitle) i .padding() són modificadors que canvien l'aparença de la vista de text.

Vistes i Modificadors

Les vistes són els blocs de construcció de SwiftUI. Els modificadors són mètodes que es poden encadenar a una vista per canviar-ne l'aparença o el comportament.

Exemple

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Text("Hola, SwiftUI!")
                .font(.largeTitle)
                .padding()
            Text("Benvingut al món de la UI declarativa.")
                .foregroundColor(.gray)
        }
    }
}

En aquest exemple:

  • VStack és una pila vertical que disposa les seves vistes filles verticalment.
  • Les vistes Text es modifiquen amb .font, .padding i .foregroundColor.

Gestió de l'Estat

La gestió de l'estat és crucial a SwiftUI per gestionar dades dinàmiques i actualitzar la UI en conseqüència.

Exemple

import SwiftUI

struct ContentView: View {
    @State private var counter = 0

    var body: some View {
        VStack {
            Text("Comptador: \(counter)")
                .font(.largeTitle)
            Button(action: {
                counter += 1
            }) {
                Text("Incrementar")
            }
            .padding()
        }
    }
}

En aquest exemple:

  • @State s'utilitza per declarar una variable d'estat counter.
  • El Button incrementa el comptador quan es toca, i la vista Text s'actualitza automàticament.

Dissenys

SwiftUI proporciona diversos contenidors de disseny com VStack, HStack i ZStack per disposar les vistes.

Exemple

import SwiftUI

struct ContentView: View {
    var body: some View {
        HStack {
            Text("Esquerra")
            Spacer()
            Text("Dreta")
        }
        .padding()
    }
}

En aquest exemple:

  • HStack disposa les seves vistes filles horitzontalment.
  • Spacer crea un espai flexible entre les dues vistes Text.

Interactivitat

SwiftUI facilita la gestió de l'entrada de l'usuari i els gestos.

Exemple

import SwiftUI

struct ContentView: View {
    @State private var isOn = false

    var body: some View {
        Toggle(isOn: $isOn) {
            Text("Interruptor")
        }
        .padding()
    }
}

En aquest exemple:

  • Toggle és un interruptor que es vincula a la variable d'estat isOn.
  • La UI s'actualitza automàticament quan es canvia l'interruptor.

Conclusió

SwiftUI simplifica el procés de construir interfícies d'usuari amb la seva sintaxi declarativa i la seva potent gestió de l'estat. Entenent els conceptes clau com les vistes, els modificadors, la gestió de l'estat, els dissenys i la interactivitat, pots crear UI dinàmiques i responsives per a totes les plataformes d'Apple. A mesura que avancis en el curs, aprofundiràs en cadascun d'aquests conceptes i aprendràs a construir aplicacions complexes utilitzant SwiftUI.
