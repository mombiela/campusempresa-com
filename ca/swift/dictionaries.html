Introducció als Diccionaris

Els diccionaris a Swift són col·leccions que emmagatzemen associacions entre claus del mateix tipus i valors del mateix tipus. Cada valor està associat amb una clau única, que actua com a identificador per a aquest valor dins del diccionari.

  • Parell Clau-Valor: Cada entrada en un diccionari és un parell clau-valor.
  • Claus Úniques: Les claus han de ser úniques dins d'un diccionari.
  • Col·lecció Desordenada: Els diccionaris són col·leccions desordenades, el que significa que l'ordre dels parells clau-valor no està garantit.

Creació i Inicialització de Diccionaris

Pots crear diccionaris utilitzant literals de diccionari o l'inicialitzador del tipus Dictionary.

Utilitzant Literals de Diccionari

var studentGrades: [String: String] = ["John": "A", "Jane": "B", "Jim": "C"]

En aquest exemple:

  • El diccionari studentGrades té claus String i valors String.
  • Les claus són noms d'estudiants, i els valors són les seves notes.

Utilitzant l'Inicialitzador del Diccionari

var emptyDictionary = [Int: String]()

Això crea un diccionari buit amb claus Int i valors String.

Accés i Modificació de Diccionaris

Pots accedir i modificar diccionaris utilitzant subscripció i diversos mètodes.

Accés als Valors

let johnGrade = studentGrades["John"]
print(johnGrade) // Optional("A")
  • Accedir a un valor retorna un opcional perquè la clau podria no existir.

Modificació dels Valors

studentGrades["Jane"] = "A+"
  • Això actualitza el valor per a la clau "Jane" a "A+".

Afegir Nous Parells Clau-Valor

studentGrades["Jake"] = "B"
  • Això afegeix un nou parell clau-valor al diccionari.

Eliminar Parells Clau-Valor

studentGrades["Jim"] = nil
  • Això elimina el parell clau-valor per a "Jim".

Iteració sobre Diccionaris

Pots iterar sobre els parells clau-valor en un diccionari utilitzant un bucle for-in.

for (student, grade) in studentGrades {
    print("\(student): \(grade)")
}
  • Això imprimeix cada parell clau-valor en el diccionari.

Mètodes Comuns de Diccionaris

Swift proporciona diversos mètodes útils per treballar amb diccionaris.

`count`

Retorna el nombre de parells clau-valor en el diccionari.

print(studentGrades.count) // 3

`isEmpty`

Comprova si el diccionari està buit.

print(studentGrades.isEmpty) // false

`removeValue(forKey:)`

Elimina el valor per a la clau especificada i el retorna.

if let removedGrade = studentGrades.removeValue(forKey: "Jane") {
    print("Removed grade: \(removedGrade)")
}

`keys` i `values`

Propietats que retornen col·leccions de les claus i valors del diccionari.

let students = studentGrades.keys
let grades = studentGrades.values

Funcions Avançades de Diccionaris

Els diccionaris a Swift ofereixen funcions més avançades per a desenvolupadors experimentats.

Fusió de Diccionaris

Pots fusionar dos diccionaris utilitzant el mètode merge(_:uniquingKeysWith:).

var additionalGrades = ["Alice": "A", "Bob": "B"]
studentGrades.merge(additionalGrades) { (current, _) in current }
  • Això fusiona additionalGrades en studentGrades, mantenint els valors actuals per a claus duplicades.

Filtratge de Diccionaris

Pots filtrar diccionaris utilitzant el mètode filter.

let filteredGrades = studentGrades.filter { $0.value == "A" }
print(filteredGrades) // ["John": "A", "Alice": "A"]
  • Això crea un nou diccionari amb només els parells clau-valor on el valor és "A".

Transformació de Diccionaris

Pots transformar diccionaris utilitzant el mètode mapValues.

let uppercasedGrades = studentGrades.mapValues { $0.uppercased() }
print(uppercasedGrades) // ["John": "A", "Jane": "A+", "Jake": "B", "Alice": "A", "Bob": "B"]
  • Això crea un nou diccionari amb els valors transformats a majúscules.

Conclusió

Els diccionaris a Swift són col·leccions potents i flexibles que et permeten emmagatzemar i gestionar parells clau-valor de manera eficient. Tant si ets un principiant com un desenvolupador avançat, entendre com crear, accedir, modificar i utilitzar diccionaris és essencial per a una programació efectiva en Swift. En dominar aquests conceptes, podràs manejar estructures de dades complexes i realitzar diverses operacions amb facilitat.
