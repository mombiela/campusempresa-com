Sentències de Transferència de Control a Swift
Optionals i Desempaquetament ►

Les sentències de transferència de control són essencials en la programació, ja que permeten canviar el flux d'execució del teu codi. A Swift, aquestes sentències t'ajuden a gestionar bucles, condicionals i altres estructures de control de manera efectiva. Aquesta secció cobrirà les diverses sentències de transferència de control disponibles a Swift, des de l'ús bàsic fins a l'avançat.

Sentència Break

La sentència break s'utilitza per sortir immediatament d'un bucle o d'una sentència switch.

Ús en Bucle

for number in 1...10 {
    if number == 5 {
        break
    }
    print(number)
}
  • Explicació: Aquest bucle imprimeix números del 1 al 4. Quan number és igual a 5, la sentència break surt del bucle.

Ús en Sentències Switch

let fruit = "Apple"

switch fruit {
case "Apple":
    print("It's an apple.")
    break
case "Banana":
    print("It's a banana.")
default:
    print("Unknown fruit.")
}
  • Explicació: La sentència break aquí és redundant perquè Swift surt automàticament del cas del switch després d'executar el cas coincident. No obstant això, pot ser útil per a la llegibilitat o modificacions futures.

Sentència Continue

La sentència continue salta la iteració actual d'un bucle i procedeix a la següent iteració.

Ús en Bucle

for number in 1...10 {
    if number % 2 == 0 {
        continue
    }
    print(number)
}
  • Explicació: Aquest bucle imprimeix només números senars del 1 al 10. Quan number és parell, la sentència continue salta la iteració actual.

Sentència Fallthrough

La sentència fallthrough s'utilitza en casos switch per continuar l'execució al següent cas.

Ús en Sentències Switch

let number = 3

switch number {
case 1:
    print("One")
case 2:
    print("Two")
case 3:
    print("Three")
    fallthrough
case 4:
    print("Four")
default:
    print("Other number")
}
  • Explicació: Quan number és 3, el cas del switch imprimeix "Three" i després "Four" a causa de la sentència fallthrough.

Sentència Return

La sentència return surt d'una funció i opcionalment retorna un valor.

Ús en Funcions

func greet(name: String) -> String {
    return "Hello, \(name)!"
}

let greeting = greet(name: "Alice")
print(greeting)
  • Explicació: La sentència return surt de la funció greet i retorna la cadena de salutació.

Sentència Throw

La sentència throw s'utilitza per llençar un error a Swift.

Ús en Funcions

enum CustomError: Error {
    case invalidInput
}

func validate(input: Int) throws {
    if input < 0 {
        throw CustomError.invalidInput
    }
}

do {
    try validate(input: -1)
} catch {
    print("Caught an error: \(error)")
}
  • Explicació: La sentència throw llença un error CustomError.invalidInput si l'entrada és negativa. El bloc do-catch gestiona l'error.

Sentència Guard

La sentència guard s'utilitza per transferir el control fora d'un àmbit si una condició no es compleix.

Ús en Funcions

func process(input: Int?) {
    guard let validInput = input else {
        print("Invalid input")
        return
    }
    print("Processing \(validInput)")
}

process(input: nil)
process(input: 10)
  • Explicació: La sentència guard comprova si input no és nil. Si és nil, imprimeix "Invalid input" i surt de la funció. En cas contrari, continua processant l'entrada vàlida.

Resum

Les sentències de transferència de control a Swift són eines poderoses per gestionar el flux del teu codi. Et permeten:

  • Sortir de bucles i sentències switch (break).
  • Saltar iteracions en bucles (continue).
  • Continuar l'execució al següent cas del switch (fallthrough).
  • Sortir de funcions i retornar valors (return).
  • Llençar i gestionar errors (throw).
  • Assegurar que es compleixin condicions abans de continuar (guard).

Entendre i utilitzar aquestes sentències de manera efectiva t'ajudarà a escriure codi Swift més robust i llegible.
