Introducció

A Swift, els operadors són símbols o frases especials que utilitzes per comprovar, canviar o combinar valors. Swift admet la majoria dels operadors estàndard de C i millora diverses capacitats per eliminar errors comuns de codificació. Aquesta secció cobrirà els operadors bàsics a Swift, incloent-hi els operadors aritmètics, de comparació, lògics i d'assignació.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar operacions matemàtiques bàsiques.

  • Suma (+): Suma dos valors.
  • Resta (-): Resta un valor d'un altre.
  • Multiplicació (*): Multiplica dos valors.
  • Divisió (/): Divideix un valor per un altre.
  • Resta (%): Retorna el residu de dividir un valor per un altre.

Exemples

let a = 10
let b = 3

let sum = a + b        // sum és 13
let difference = a - b // difference és 7
let product = a * b    // product és 30
let quotient = a / b   // quotient és 3
let remainder = a % b  // remainder és 1

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar dos valors.

  • Igual a (==): Comprova si dos valors són iguals.
  • No igual a (!=): Comprova si dos valors no són iguals.
  • Major que (>): Comprova si el valor de l'esquerra és més gran que el valor de la dreta.
  • Menor que (<): Comprova si el valor de l'esquerra és menor que el valor de la dreta.
  • Major o igual a (>=): Comprova si el valor de l'esquerra és més gran o igual que el valor de la dreta.
  • Menor o igual a (<=): Comprova si el valor de l'esquerra és menor o igual que el valor de la dreta.

Exemples

let x = 5
let y = 10

let isEqual = x == y       // isEqual és fals
let isNotEqual = x != y    // isNotEqual és cert
let isGreater = x > y      // isGreater és fals
let isLess = x < y         // isLess és cert
let isGreaterOrEqual = x >= y // isGreaterOrEqual és fals
let isLessOrEqual = x <= y // isLessOrEqual és cert

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions booleans.

  • AND Lògic (&&): Retorna cert si ambdues condicions són certes.
  • OR Lògic (||): Retorna cert si almenys una condició és certa.
  • NOT Lògic (!): Inverteix el valor booleà.

Exemples

let isTrue = true
let isFalse = false

let andResult = isTrue && isFalse  // andResult és fals
let orResult = isTrue || isFalse   // orResult és cert
let notResult = !isTrue            // notResult és fals

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació (=): Assigna el valor de la dreta a la variable de l'esquerra.
  • Operadors d'Assignació Composada: Combinen una operació aritmètica amb l'assignació.

Operadors d'Assignació Composada

  • Assignació de Suma (+=): Suma i assigna el valor.
  • Assignació de Resta (-=): Resta i assigna el valor.
  • Assignació de Multiplicació (*=): Multiplica i assigna el valor.
  • Assignació de Divisió (/=): Divideix i assigna el valor.
  • Assignació de Resta (%=): Calcula el residu i assigna el valor.

Exemples

var number = 10

number += 5  // number és ara 15
number -= 3  // number és ara 12
number *= 2  // number és ara 24
number /= 4  // number és ara 6
number %= 5  // number és ara 1

Conclusió

Entendre els operadors bàsics és fonamental per a qualsevol programador de Swift. Aquests operadors et permeten realitzar tasques essencials com càlculs aritmètics, comparacions, operacions lògiques i assignacions de valors. Dominar aquests operadors proporcionarà una base sòlida per a conceptes de programació més avançats. Practica utilitzant aquests operadors en diferents escenaris per sentir-te còmode amb la seva sintaxi i funcionalitat.
