Introducció

En Swift, les variables i les constants són blocs fonamentals per emmagatzemar i manipular dades. Entendre com declarar-les i utilitzar-les és essencial per a qualsevol programador de Swift. Aquesta secció cobrirà els conceptes bàsics de les variables i les constants, incloent la seva declaració, ús i diferències.

Variables

Les variables s'utilitzen per emmagatzemar valors que poden canviar amb el temps. En Swift, declares una variable utilitzant la paraula clau var.

Declaració de Variables

Per declarar una variable, utilitzes la paraula clau var seguida del nom de la variable i una anotació de tipus opcional.

var nomVariable: Tipus = valorInicial

Exemple

var edat: Int = 25
var nom: String = "John"

Conceptes Clau

  • Inferència de Tipus: Swift sovint pot inferir el tipus d'una variable a partir del valor inicial, de manera que pots ometre l'anotació de tipus.
  • Mutabilitat: Les variables poden ser canviades després de ser declarades.
var edat = 25 // Swift infereix que edat és de tipus Int
edat = 26 // Canviant el valor de edat

Constants

Les constants s'utilitzen per emmagatzemar valors que no canvien un cop establerts. En Swift, declares una constant utilitzant la paraula clau let.

Declaració de Constants

Per declarar una constant, utilitzes la paraula clau let seguida del nom de la constant i una anotació de tipus opcional.

let nomConstant: Tipus = valorInicial

Exemple

let anyNaixement: Int = 1995
let pi: Double = 3.14159

Conceptes Clau

  • Immutabilitat: Les constants no poden ser canviades després de ser declarades.
  • Inferència de Tipus: Igual que les variables, Swift pot inferir el tipus d'una constant a partir del valor inicial.
let anyNaixement = 1995 // Swift infereix que anyNaixement és de tipus Int
// anyNaixement = 1996 // Això causarà un error perquè anyNaixement és una constant

Comparant Variables i Constants

| Característica | Variables (var) | Constants (let) | |----------------|-------------------|-------------------| | Mutabilitat | Pot ser canviada | No pot ser canviada | | Declaració | Paraula clau var| Paraula clau let | | Cas d'ús | Valors que canvien| Valors que romanen constants |

Exemples de Codi

Utilitzant Variables

var salutacio = "Hola, Món!"
print(salutacio) // Sortida: Hola, Món!
salutacio = "Hola, Swift!"
print(salutacio) // Sortida: Hola, Swift!

Utilitzant Constants

let nombreMàximD'IntentsDeSessió = 3
var intentActualDeSessió = 0

// Descomentar la següent línia causarà un error
// nombreMàximD'IntentsDeSessió = 4

Conclusió

Entendre la diferència entre variables i constants és crucial per escriure codi efectiu en Swift. Les variables (var) s'utilitzen per a valors que poden canviar, mentre que les constants (let) s'utilitzen per a valors que romanen iguals. Utilitzant-les adequadament, pots fer que el teu codi sigui més predictible i fàcil d'entendre. Recorda aprofitar la inferència de tipus de Swift per escriure codi més net i concís.
