Les declaracions condicionals són fonamentals en la programació ja que permeten executar diferents codis basats en certes condicions. A Swift, les principals declaracions condicionals són if, else if, else i switch.

Conceptes clau

  • If Statement: Executa un bloc de codi si una condició especificada és certa.
  • Else If Statement: Proporciona una condició addicional per comprovar si la condició inicial if és falsa.
  • Else Statement: Executa un bloc de codi si cap de les condicions anteriors és certa.
  • Switch Statement: Avalua un valor i executa blocs de codi corresponents basats en casos coincidents.

If Statement

La declaració if s'utilitza per executar un bloc de codi si una condició és certa.

let temperature = 30

if temperature > 25 {
    print("És un dia calorós!")
}

En aquest exemple, es mostrarà el missatge "És un dia calorós!" perquè la condició temperature > 25 és certa.

Else If Statement

La declaració else if permet comprovar múltiples condicions.

let temperature = 20

if temperature > 25 {
    print("És un dia calorós!")
} else if temperature > 15 {
    print("És un dia càlid!")
}

Aquí, es mostrarà el missatge "És un dia càlid!" perquè la primera condició és falsa, però la segona condició temperature > 15 és certa.

Else Statement

La declaració else s'utilitza per executar un bloc de codi si cap de les condicions anteriors és certa.

let temperature = 10

if temperature > 25 {
    print("És un dia calorós!")
} else if temperature > 15 {
    print("És un dia càlid!")
} else {
    print("És un dia fred!")
}

En aquest cas, es mostrarà el missatge "És un dia fred!" perquè ambdues condicions anteriors són falses.

Switch Statement

La declaració switch avalua un valor i executa el bloc de cas corresponent.

let dayOfWeek = 3

switch dayOfWeek {
case 1:
    print("Dilluns")
case 2:
    print("Dimarts")
case 3:
    print("Dimecres")
case 4:
    print("Dijous")
case 5:
    print("Divendres")
case 6:
    print("Dissabte")
case 7:
    print("Diumenge")
default:
    print("Dia invàlid")
}

En aquest exemple, es mostrarà el missatge "Dimecres" perquè dayOfWeek és 3.

Switch Statement amb Rangs i Tuples

La declaració switch de Swift és potent i suporta rangs i tuples.

let score = 85

switch score {
case 0..<50:
    print("Suspès")
case 50..<75:
    print("Aprovat")
case 75...100:
    print("Excel·lent")
default:
    print("Puntuació invàlida")
}

Aquí, es mostrarà el missatge "Excel·lent" perquè score es troba dins del rang de 75 a 100.

Resum

  • If Statement: Executa codi si una condició és certa.
  • Else If Statement: Comprova condicions addicionals si les anteriors són falses.
  • Else Statement: Executa codi si cap de les condicions és certa.
  • Switch Statement: Avalua un valor i el compara amb múltiples casos.

Entendre i utilitzar les declaracions condicionals de manera efectiva et permet controlar el flux dels teus programes en Swift, fent-los més dinàmics i sensibles a diferents entrades i condicions.
