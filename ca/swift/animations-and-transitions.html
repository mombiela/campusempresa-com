Introducció a les Animacions i Transicions

Les animacions i transicions són essencials per crear interfícies d'usuari atractives i interactives en aplicacions iOS. Swift proporciona eines i marcs de treball poderosos per implementar animacions suaus i visualment atractives.

Conceptes Clau

  • Animacions UIView: Animacions bàsiques que es poden aplicar a les propietats de UIView.
  • Core Animation: Un marc de treball més avançat per crear animacions complexes.
  • Transicions: Animacions que ocorren quan es canvia d'una vista a una altra.

Animacions UIView

Les animacions UIView són la manera més senzilla d'animar canvis a les vistes. Són fàcils d'implementar i perfectes per a principiants.

Animacions Bàsiques UIView

Pots animar propietats com la posició, mida, opacitat i més utilitzant UIView.animate.

UIView.animate(withDuration: 1.0) {
    myView.alpha = 0.0
}
  • withDuration: Especifica la durada de l'animació.
  • animations: Un tancament que conté els canvis a animar.

Enllaçar Animacions

Pots enllaçar múltiples animacions per obtenir efectes més complexos.

UIView.animate(withDuration: 1.0, animations: {
    myView.alpha = 0.0
}) { _ in
    UIView.animate(withDuration: 1.0) {
        myView.alpha = 1.0
    }
}

Core Animation

Core Animation proporciona més control i flexibilitat per crear animacions avançades.

Animació Bàsica Core Animation

Core Animation utilitza CABasicAnimation per animar propietats de capa.

let animation = CABasicAnimation(keyPath: "position")
animation.fromValue = CGPoint(x: 0, y: 0)
animation.toValue = CGPoint(x: 100, y: 100)
animation.duration = 1.0
myView.layer.add(animation, forKey: "positionAnimation")
  • keyPath: La propietat a animar.
  • fromValue: El valor inicial de la propietat.
  • toValue: El valor final de la propietat.
  • duration: La durada de l'animació.

Agrupar Animacions

Pots agrupar múltiples animacions utilitzant CAAnimationGroup.

let positionAnimation = CABasicAnimation(keyPath: "position")
positionAnimation.fromValue = CGPoint(x: 0, y: 0)
positionAnimation.toValue = CGPoint(x: 100, y: 100)

let opacityAnimation = CABasicAnimation(keyPath: "opacity")
opacityAnimation.fromValue = 1.0
opacityAnimation.toValue = 0.0

let group = CAAnimationGroup()
group.animations = [positionAnimation, opacityAnimation]
group.duration = 1.0

myView.layer.add(group, forKey: "groupAnimation")

Transicions

Les transicions són animacions que ocorren quan es canvia entre vistes o controladors de vista.

Transicions de Vista

Pots animar transicions entre vistes utilitzant UIView.transition.

UIView.transition(with: containerView, duration: 1.0, options: .transitionFlipFromLeft, animations: {
    containerView.addSubview(newView)
}, completion: nil)
  • with: La vista a transitar.
  • duration: La durada de la transició.
  • options: El tipus de transició (per exemple, .transitionFlipFromLeft).

Transicions de Controlador de Vista

Les transicions personalitzades entre controladors de vista es poden aconseguir utilitzant UIViewControllerAnimatedTransitioning.

class CustomTransition: NSObject, UIViewControllerAnimatedTransitioning {
    func transitionDuration(using transitionContext: UIViewControllerContextTransitioning?) -> TimeInterval {
        return 1.0
    }

    func animateTransition(using transitionContext: UIViewControllerContextTransitioning) {
        guard let fromView = transitionContext.view(forKey: .from),
              let toView = transitionContext.view(forKey: .to) else { return }

        let containerView = transitionContext.containerView
        containerView.addSubview(toView)
        toView.alpha = 0.0

        UIView.animate(withDuration: 1.0, animations: {
            toView.alpha = 1.0
        }) { _ in
            transitionContext.completeTransition(!transitionContext.transitionWasCancelled)
        }
    }
}
  • transitionDuration: Especifica la durada de la transició.
  • animateTransition: Conté la lògica de l'animació.

Conclusió

Les animacions i transicions són eines poderoses per millorar l'experiència de l'usuari en aplicacions iOS. Començant amb animacions bàsiques UIView i progressant cap a animacions més avançades amb Core Animation i transicions personalitzades, Swift proporciona un conjunt complet d'eines per crear animacions suaus i atractives. Practica implementant aquestes tècniques per crear aplicacions visualment atractives i interactives.
