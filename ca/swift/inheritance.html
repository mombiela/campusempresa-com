El Projecte Sobre nosaltres Contribuir Donacions Llicència

L'herència és un concepte fonamental en la programació orientada a objectes que permet a una classe heretar propietats i mètodes d'una altra classe. Això ajuda a reutilitzar codi i crear una relació jeràrquica entre classes.

Conceptes Clau

  • Superclasse: La classe les propietats i mètodes de la qual són heretats.
  • Subclasse: La classe que hereta propietats i mètodes de la superclasse.
  • Sobreescriptura: El procés de proporcionar una nova implementació per a un mètode o propietat heretada.
  • Classes Finals: Classes que no poden ser heretades.

Creació d'una Superclasse

Per crear una superclasse en Swift, defineixes una classe amb propietats i mètodes que poden ser heretats per altres classes.

class Vehicle {
    var currentSpeed = 0.0
    
    func description() -> String {
        return "Traveling at \(currentSpeed) miles per hour"
    }
}

En aquest exemple, Vehicle és una superclasse amb una propietat currentSpeed i un mètode description().

Creació d'una Subclasse

Una subclasse hereta propietats i mètodes de la seva superclasse. Crees una subclasse especificant la superclasse després del nom de la subclasse, separada per dos punts.

class Bicycle: Vehicle {
    var hasBasket = false
}

Aquí, Bicycle és una subclasse de Vehicle. Hereta la propietat currentSpeed i el mètode description() de Vehicle.

Sobreescriptura de Mètodes

Una subclasse pot sobreescriure mètodes i propietats de la seva superclasse per proporcionar una implementació específica.

class Bicycle: Vehicle {
    var hasBasket = false
    
    override func description() -> String {
        return "Traveling at \(currentSpeed) miles per hour on a bicycle"
    }
}

En aquest exemple, el mètode description() és sobreescrit a la classe Bicycle per proporcionar una descripció més específica.

Ús de Mètodes i Propietats de la Superclasse

Una subclasse pot utilitzar la paraula clau super per accedir als mètodes i propietats de la seva superclasse.

class Tandem: Bicycle {
    var currentNumberOfPassengers = 0
    
    override func description() -> String {
        return super.description() + " with \(currentNumberOfPassengers) passengers"
    }
}

Aquí, la classe Tandem, que és una subclasse de Bicycle, utilitza super.description() per cridar el mètode description() de la seva superclasse i després afegeix informació addicional.

Classes Finals

Si vols evitar que una classe sigui heretada, pots marcar-la com a final.

final class Train: Vehicle {
    var numberOfCarriages = 0
}

La classe Train no pot ser subclassificada perquè està marcada com a final.

Exercicis

  1. Crea una Superclasse i Subclasse:

    • Defineix una superclasse Animal amb una propietat name i un mètode makeSound().
    • Crea una subclasse Dog que sobreescrigui el mètode makeSound() per imprimir "Woof!".

  2. Sobreescriu Mètodes:

    • Crea una superclasse Shape amb un mètode area().
    • Crea subclasses Circle i Rectangle que sobreescriguin el mètode area() per calcular l'àrea de la forma.

  3. Utilitza Mètodes de la Superclasse:

    • Crea una superclasse Employee amb un mètode work().
    • Crea una subclasse Manager que sobreescrigui el mètode work() i utilitzi super.work() per cridar el mètode de la superclasse.

Resum

L'herència et permet crear una nova classe basada en una classe existent, promovent la reutilització de codi i establint una relació jeràrquica entre classes. Els conceptes clau inclouen superclasses, subclasses, sobreescriptura de mètodes i classes finals. Entenent i utilitzant l'herència, pots crear codi més organitzat i mantenible en Swift.
