Introducció

En aquesta secció, ens endinsarem en tècniques avançades d'interfície d'usuari en Swift, centrant-nos en la creació d'interfícies d'usuari dinàmiques, responsives i visualment atractives. Això inclourà treballar amb components avançats de UIKit, animacions, vistes personalitzades i aprofitar SwiftUI per al desenvolupament modern d'interfícies d'usuari.

Components Avançats de UIKit

UIKit proporciona un marc robust per construir interfícies d'usuari a iOS. Aquí, explorarem alguns dels components més avançats i com utilitzar-los de manera efectiva.

UICollectionView

  • Visió General: UICollectionView és una manera poderosa i flexible de presentar una col·lecció de dades en un disseny personalitzable.
  • Conceptes Clau:
    • Font de Dades i Delegat: Implementació dels protocols UICollectionViewDataSource i UICollectionViewDelegate.
    • Dissenys Personalitzats: Utilització de UICollectionViewFlowLayout o creació de dissenys personalitzats subclassejant UICollectionViewLayout.
import UIKit

class MyCollectionViewController: UICollectionViewController, UICollectionViewDelegateFlowLayout {
    override func viewDidLoad() {
        super.viewDidLoad()
        collectionView.register(UICollectionViewCell.self, forCellWithReuseIdentifier: "cell")
    }

    override func collectionView(_ collectionView: UICollectionView, numberOfItemsInSection section: Int) -> Int {
        return 20
    }

    override func collectionView(_ collectionView: UICollectionView, cellForItemAt indexPath: IndexPath) -> UICollectionViewCell {
        let cell = collectionView.dequeueReusableCell(withReuseIdentifier: "cell", for: indexPath)
        cell.backgroundColor = .blue
        return cell
    }

    func collectionView(_ collectionView: UICollectionView, layout collectionViewLayout: UICollectionViewLayout, sizeForItemAt indexPath: IndexPath) -> CGSize {
        return CGSize(width: 100, height: 100)
    }
}

UITableView amb Cel·les Personalitzades

  • Visió General: UITableView és un element bàsic en el desenvolupament d'iOS, i personalitzar les cel·les pot millorar molt l'experiència de l'usuari.
  • Conceptes Clau:
    • UITableViewCell Personalitzada: Creació i registre de cel·les personalitzades.
    • Alçades Dinàmiques: Ajust de les alçades de les cel·les dinàmicament segons el contingut.
import UIKit

class CustomTableViewCell: UITableViewCell {
    let customLabel = UILabel()

    override init(style: UITableViewCell.CellStyle, reuseIdentifier: String?) {
        super.init(style: style, reuseIdentifier: reuseIdentifier)
        customLabel.translatesAutoresizingMaskIntoConstraints = false
        contentView.addSubview(customLabel)
        NSLayoutConstraint.activate([
            customLabel.centerYAnchor.constraint(equalTo: contentView.centerYAnchor),
            customLabel.leadingAnchor.constraint(equalTo: contentView.leadingAnchor, constant: 16)
        ])
    }

    required init?(coder: NSCoder) {
        fatalError("init(coder:) has not been implemented")
    }
}

class MyTableViewController: UITableViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        tableView.register(CustomTableViewCell.self, forCellReuseIdentifier: "customCell")
    }

    override func tableView(_ tableView: UITableView, numberOfRowsInSection section: Int) -> Int {
        return 10
    }

    override func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell {
        let cell = tableView.dequeueReusableCell(withIdentifier: "customCell", for: indexPath) as! CustomTableViewCell
        cell.customLabel.text = "Fila \(indexPath.row)"
        return cell
    }
}

Animacions

Les animacions poden fer que la teva aplicació sembli més dinàmica i responsiva. Swift proporciona diverses maneres de crear animacions.

Animacions de UIView

  • Visió General: Manera simple i efectiva d'animar vistes.
  • Conceptes Clau:
    • Animacions Bàsiques: Utilització de UIView.animate per a animacions simples.
    • Animacions de Molla: Afegir efectes de molla per a animacions més naturals.
UIView.animate(withDuration: 1.0, animations: {
    self.myView.alpha = 0.0
}) { _ in
    UIView.animate(withDuration: 1.0) {
        self.myView.alpha = 1.0
    }
}

Animació de Nucli

  • Visió General: Més control sobre les animacions utilitzant Core Animation.
  • Conceptes Clau:
    • CABasicAnimation: Animacions bàsiques de propietats.
    • CAKeyframeAnimation: Animacions complexes al llarg d'un camí.
let animation = CABasicAnimation(keyPath: "position")
animation.fromValue = CGPoint(x: 0, y: 0)
animation.toValue = CGPoint(x: 100, y: 100)
animation.duration = 1.0
myView.layer.add(animation, forKey: "position")

Vistes Personalitzades

Crear vistes personalitzades permet encapsular elements d'interfície d'usuari complexos i reutilitzar-los a tota la teva aplicació.

Dibuix amb Core Graphics

  • Visió General: Dibuix personalitzat utilitzant Core Graphics.
  • Conceptes Clau:
    • Dibuix de Formes: Utilització de UIBezierPath per dibuixar formes.
    • Dibuix Personalitzat: Sobreescriure el mètode draw(_:).
class CustomView: UIView {
    override func draw(_ rect: CGRect) {
        guard let context = UIGraphicsGetCurrentContext() else { return }
        context.setFillColor(UIColor.red.cgColor)
        context.fillEllipse(in: rect)
    }
}

Controls Personalitzats

  • Visió General: Creació de controls personalitzats reutilitzables.
  • Conceptes Clau:
    • Encapsulació: Encapsulació de funcionalitat dins d'un control personalitzat.
    • Integració amb Interface Builder: Fer controls personalitzats IBDesignable i IBInspectable.
@IBDesignable
class CustomButton: UIButton {
    @IBInspectable var borderColor: UIColor = .black {
        didSet {
            layer.borderColor = borderColor.cgColor
        }
    }

    override init(frame: CGRect) {
        super.init(frame: frame)
        setup()
    }

    required init?(coder: NSCoder) {
        super.init(coder: coder)
        setup()
    }

    private func setup() {
        layer.borderWidth = 2
        layer.borderColor = borderColor.cgColor
    }
}

SwiftUI

SwiftUI és el marc modern d'Apple per construir interfícies d'usuari a totes les plataformes d'Apple utilitzant Swift.

Introducció a SwiftUI

  • Visió General: Sintaxi declarativa per construir interfícies d'usuari.
  • Conceptes Clau:
    • Vistes i Modificadors: Blocs de construcció de SwiftUI.
    • Gestió d'Estat: Utilització de @State, @Binding i altres wrappers de propietat.
import SwiftUI

struct ContentView: View {
    @State private var isOn = false

    var body: some View {
        VStack {
            Toggle(isOn: $isOn) {
                Text("Interruptor")
            }
            .padding()

            if isOn {
                Text("L'interruptor està ENCÈS")
            } else {
                Text("L'interruptor està APAGAT")
            }
        }
    }
}

Tècniques Avançades de SwiftUI

  • Visió General: Aprofitament de les funcions avançades de SwiftUI.
  • Conceptes Clau:
    • Vistes Personalitzades: Creació de vistes personalitzades reutilitzables.
    • Animacions: Utilització de les capacitats d'animació integrades de SwiftUI.
struct CustomCircleView: View {
    @State private var scale: CGFloat = 1.0

    var body: some View {
        Circle()
            .fill(Color.blue)
            .frame(width: 100, height: 100)
            .scaleEffect(scale)
            .onTapGesture {
                withAnimation(.spring()) {
                    scale = scale == 1.0 ? 1.5 : 1.0
                }
            }
    }
}
