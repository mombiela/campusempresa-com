Introducció

Les funcions són blocs fonamentals en la programació en Swift. Permeten encapsular un bloc de codi que realitza una tasca específica, que després es pot reutilitzar al llarg del teu programa. Aquesta secció cobrirà com definir i cridar funcions, des de conceptes bàsics fins a avançats.

Definició de Funcions

Definició Bàsica de Funció

Una funció en Swift es defineix utilitzant la paraula clau func, seguida del nom de la funció, un parell de parèntesis i un conjunt de claus que contenen el codi de la funció.

func greet() {
    print("Hola, Món!")
}
  • func: Paraula clau per definir una funció.
  • greet: Nom de la funció.
  • (): Parèntesis que poden contenir paràmetres.
  • { ... }: Claus que contenen el codi de la funció.

Funció amb Paràmetres

Les funcions poden prendre paràmetres per fer-les més flexibles i reutilitzables.

func greet(name: String) {
    print("Hola, \(name)!")
}
  • name: String: Paràmetre anomenat name de tipus String.

Funció amb Valor de Retorn

Les funcions també poden retornar valors. El tipus de retorn es especifica després de la llista de paràmetres.

func add(a: Int, b: Int) -> Int {
    return a + b
}
  • -> Int: Indica que la funció retorna un Int.

Crida de Funcions

Crida d'una Funció sense Paràmetres

Per cridar una funció, simplement utilitza el seu nom seguit de parèntesis.

greet()

Crida d'una Funció amb Paràmetres

Quan crides una funció amb paràmetres, proporciona els arguments dins dels parèntesis.

greet(name: "Alice")

Ús del Valor de Retorn d'una Funció

Pots capturar el valor de retorn d'una funció en una variable o utilitzar-lo directament.

let sum = add(a: 5, b: 3)
print(sum) // Sortida: 8

Conceptes Avançats de Funcions

Valors de Paràmetres per Defecte

Pots proporcionar valors per defecte per als paràmetres, fent-los opcionals quan es crida la funció.

func greet(name: String = "Món") {
    print("Hola, \(name)!")
}

greet() // Sortida: Hola, Món!
greet(name: "Alice") // Sortida: Hola, Alice!

Paràmetres Variàdics

Un paràmetre variàdic accepta zero o més valors d'un tipus especificat.

func sum(numbers: Int...) -> Int {
    var total = 0
    for number in numbers {
        total += number
    }
    return total
}

let totalSum = sum(numbers: 1, 2, 3, 4, 5)
print(totalSum) // Sortida: 15

Paràmetres In-Out

Els paràmetres in-out permeten que una funció modifiqui el valor d'un paràmetre.

func increment(value: inout Int) {
    value += 1
}

var number = 10
increment(value: &number)
print(number) // Sortida: 11

Tipus de Funció

Les funcions tenen tipus, que es poden utilitzar com a paràmetres o tipus de retorn per a altres funcions.

func multiply(a: Int, b: Int) -> Int {
    return a * b
}

let operation: (Int, Int) -> Int = multiply
let result = operation(3, 4)
print(result) // Sortida: 12

Funcions Imbricades

Les funcions es poden definir dins d'altres funcions, proporcionant una manera d'encapsular funcions auxiliars.

func outerFunction() {
    func innerFunction() {
        print("Funció interna")
    }
    innerFunction()
}

outerFunction() // Sortida: Funció interna

Conclusió

Les funcions en Swift són eines versàtils i poderoses que t'ajuden a escriure codi net, reutilitzable i mantenible. Des de definicions i crides bàsiques fins a conceptes avançats com paràmetres in-out i funcions imbricades, dominar les funcions millorarà significativament les teves habilitats de programació en Swift. Practica definint i cridant funcions amb diferents paràmetres i tipus de retorn per convertir-te en un expert en el seu ús efectiu.
